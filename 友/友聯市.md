'''联合市'''（'''{{lang-en|Union city}}'''）又称'''盟市'''、'''友联市'''、'''尤宁城'''为[[新泽西州|紐澤西州]][[哈德孙县（郡）|哈德遜郡]]（縣）的一座城市，面积1.283平方英里(3.22平方公里)，人口 66,455（2010年），其人口密度为每平方英里51,797人（每平方公里20,004.5人）。该市是[[:en:List_of_United_States_cities_by_population_density|美国人口密度第二高的城市]]。

友联市设立于1925年6月1日，为[[:en:Union_Hill,_New_Jersey|友联山镇]]和[[:en:West_Hoboken,_New_Jersey|西霍博肯镇]]合并而成。该市经历了两拨移民潮，第一次主要为德国裔移民，第二次则主要为说[[西班牙语|西班牙语]]的拉丁美洲裔。两次移民潮很大的影响了该市的风格和发展方向。友联市有两个绰号，分别是“美国刺绣之都”和“[[哈德孙河|哈德孙河]]上的小[[哈瓦那|哈瓦那]]”<ref>{{cite web | title = 小哈瓦那和哈德孙河上的小哈瓦那| url = http://www.barrypopik.com/index.php/florida/entry/little_havana_miami_little_havana_on_the_hudson_union_city_new_jersey/}} </ref>，这体现了该城在历史上重要的两个方面。该市每年三月举行全美最长的[[:en:_Passion_play|耶稣受难表演]]<ref>{{cite web| title = Park Performing Art Center| url = http://www.parkpac.org/pp_pas.html| deadurl = yes| archiveurl = https://web.archive.org/web/20110727164656/http://www.parkpac.org/pp_pas.html| archivedate = 2011-07-27}} </ref>，以及每年六月举行的盛大的古巴日游行，都能吸引了数千人观看。<ref>{{cite web | title = 第12届北哈德孙郡古巴日游行| url = http://www.nj.com/hudson/index.ssf/2011/06/photos_12th_annual_north_hudso.html}}</ref>
