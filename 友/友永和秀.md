'''友永和秀'''{{BD|1952年|4月28日||}}，[[日本|日本]]資深男性[[動畫師|動畫師]]。{{link-ja|Telecom Animation Film|テレコム・アニメーションフィルム}}代表董事。{{link-ja|日本動畫師·演出協會|日本アニメーター・演出協会}}（JAniCA）會員。

== 簡介 ==
高中畢業前往東京發展之前，在大阪過著當上班族的生活有半年的時間。1971年，加入[[白土武|白土武]]經營的作畫工作室{{link-ja|Tiger Production|タイガープロダクション}}，1975年經歷過{{link-ja|OH！Production|OH!プロダクション}}所屬，1980年加入現在所屬的{{link-ja|Telecom Animation Film|テレコム・アニメーションフィルム}}<ref>「友永和秀專訪」《{{link-ja|別冊寶島|別冊宝島}} 魯邦三世PERFECT BOOK TV版Part 2 故事解析篇》，[[寶島社|寶島社]]，2003年。</ref>。

== 來歷、人物 ==
1972年以電視動畫《[[惡魔人|惡魔人]]》負責動畫一職出道<ref>「魯邦三世動畫工作人員隨筆 友永和秀與魯邦三世」《魯邦三世 LUPIN the 40！ ～The animation～》，不知火Pro編著，[[雙葉社|雙葉社]]，2007年，第127頁。</ref>。Tiger Production所屬時期，曾參加製作《[[無敵鐵金剛|無敵鐵金剛]]》和《[[宇宙戰艦大和號|宇宙戰艦大和號]]》。之後在OH！Production所屬時期，以[[原畫|原畫]]工作參加{{link-ja|小松原一男|小松原一男}}擔任人物設定作品《[[金剛戰神|金剛戰神]]》、《[[無敵龍捲風|無敵龍捲風]]》、《[[無敵巴拉達|無敵巴拉達]]》等由[[東映動畫|東映動畫]]統包製作的[[機器人動畫|機器人動畫]]<ref name="komatsubara">《小松原一男動畫畫集》，{{link-ja|小松原一男|小松原一男}}著、{{link-ja|並木孝|なみきたかし}}編輯，{{link-ja|東急AGENCY|東急エージェンシー}}，2002年，第153頁。</ref>。1979加入Telecom Animation Film<ref>[[大塚康生|大塚康生]] 《小尼莫的野心》，[[德間書店|德間書店]]，2004年，第80頁。</ref>之後，次年1980年以《{{link-ja|姿三四郎|姿三四郎}}》擔任[[作畫監督|作畫監督]]正式出道<ref group="注">但稍早在1977年電視動畫《[[鐵臂馬魯斯|鐵臂馬魯斯]]》第23話擔當過作畫監督。</ref><ref name="fumetsu">[[赤星政尚|赤星政尚]]等「CHAPTER.3 製作機器人動畫的男人們 機器人動畫 走入殿堂工作人員紳士錄」《不滅的超級機器人大全 無敵鐵金剛、變形金剛、新機動戰記鋼彈W徹底大研究》，{{link-ja|二見書房|二見書房}}，1998年9月25日，第132頁 ISBN 4-576-98138-2</ref>。從此以後，友永以Telecom所屬動畫師持續參加國內製作或跨海合作作品，以及關係企業[[TMS娛樂|TMS Entertainment]]的作品。

值得注目的和受到高評價的作品如下。

1974年電視動畫《宇宙戰艦大和號》中[[大和號戰艦|戰艦大和]]的回想和決戰七色星團的場面<ref name="fumetsu" /><ref>{{link-ja|冰川龍介|氷川竜介}}「動畫特殊技法的變遷」《圖説電視動畫全書》，御園{{lang|ja|まこと}}編著，原書房，1999年，第201－203頁。</ref><ref name="sf-anime">EYECOM Files編「第5章 技術 『動畫表現之歷史』 冰川龍介」《科幻動畫原來這麼有趣 從機動戰士鋼彈到新世紀福音戰士》，{{link-ja|Aspect (企業)|アスペクト (企業)|Aspect}}，1997年1月7日，第169頁 ISBN 4-89366-643-6</ref><ref name="dogaoh">冰川龍介取材、構成「證言 巨大機器人動畫的時代 金田伊功專訪」《動畫王 Vol.1 決戰！巨大機器人動畫》，Kinema旬報社，1997年，第188頁。</ref>，1979年由Telecom承包製作的[[動畫電影|動畫電影]]《[[魯邦三世卡里奧斯特羅之城|魯邦三世：卡里奧斯特羅之城]]》中的車輛追逐場面<ref name="komatsubara" /><ref>堀田純司、[[GAINAX|GAINAX]]《採訪GAINAX》，[[講談社|講談社]]，2005年，第110頁。</ref>及《[[銀河鐵道999|劇場版 銀河鐵道999]]》中{{link-ja|阿爾卡迪亞號Z|アルカディア号}}進入戰鬥高潮的場面<ref name="sf-anime" /><ref name="dogaoh" /><ref>Fusion Product編《世界與日本動畫的150選》，{{link-ja|Fusion Product|ふゅーじょんぷろだくと}}，2003年，第10、172－173頁。</ref>。

1984年，與[[近藤喜文|近藤喜文]]共同製作完成3分鐘半試映動畫《[[小尼莫|小尼莫]]》在美國上映後受到動畫業界的熱烈歡迎<ref>《小尼莫的野心》，第129－130頁。</ref>，同時成為友永動畫生涯的代表作之一<ref>小黑祐一郎{{lang|ja|[http://www.style.fm/as/05_column/animesama22.shtml 「アニメ様の七転八倒 第22回 日本アニメ史 空前の大プロジェクト」]}}. WEB {{link-ja|Anime Style|アニメスタイル}} [2005-11-21] {{ja icon}}.</ref>。

根據同事[[大塚康生|大塚康生]]的說法，友永讓機械動起來的感覺和動作技術之表現，在觀眾的評價中受到高度肯定<ref>大塚康生、森遊機《大塚康生專訪 動畫無拘無束》，{{link-ja|實業之日本社|実業之日本社}}，2006年，第38頁。</ref>。而且，跟友永曾經一起參與《宇宙戰艦大和號》與《銀河鐵道999》等作品製作的[[金田伊功|金田伊功]]表示，一直將他視為競爭對手、也是業界好友<ref name="dogaoh" />。

1982年開始花了2年時間完成播映電視動畫《[[名偵探福爾摩斯|名偵探福爾摩斯]]》的登場角色中，莫里亞蒂的部下陶德是以擔任作畫監督的友永他為原型所設計的<ref>{{lang|ja|[http://www.telecom-anime.com/telecom/zadankai/holmes/zadankai1_2i.html 座談会テーマ1：「名探偵ホームズ」（其の2）]}}. [http://www.telecom-anime.com/ Telecom Animation Film公式官網] [2018-09-22] {{ja icon}}.</ref>。

2016年，友永榮獲第25回[[日本電影影評人大獎|日本電影影評人大獎]]動畫部門功勞獎<ref>{{cite news|url=http://natalie.mu/eiga/news/188463|script-title=ja:批評家大賞アニメ部門で永井豪が最高賞に、功労賞·渡辺宙明を串田アキラが祝福|newspaper=[[Natalie_(網站)|電影Natalie]]|date=2016-05-25|accessdate=2016-05-25|language=ja}}</ref>。
