'''雷蒙·阿隆'''（{{lang|fr|Raymond Aron}}，{{bd|1905年|3月14日|1983年|10月17日|catIdx=R}}），<ref name=rfi01>{{Cite web |url=http://cn.rfi.fr/%E6%94%BF%E6%B2%BB/20181211-%E4%BB%8B%E5%85%A5%E7%9A%84%E6%97%81%E8%A7%82%E8%80%85%E9%9B%B7%E8%92%99%C2%B7%E9%98%BF%E9%9A%86%E7%AC%AC%E4%B8%80%E9%83%A8%E5%88%86-%E9%9B%B7%E8%92%99%C2%B7%E9%98%BF%E9%9A%86%E7%9A%84%E6%97%B6%E4%BB%A3%E8%83%8C%E6%99%AF |title=介入的旁观者雷蒙·阿隆第一部分： 雷蒙·阿隆的时代背景 |date=2018-12-11 |publisher=法廣}}</ref>[[法國|法國]]社會學家、哲學家、政治學家，「Aron」也有人翻譯成'''阿宏'''或'''亞宏'''。以批判左派法國思想家[[薩特|薩特]]聞名。

== 生平 ==
阿隆為一名猶太律師之子，1930年獲[[巴黎高等師範學院|巴黎高等師範學院]]哲學史博士，1940年為逃離[[納粹|納粹]]而流亡[[倫敦|倫敦]]加入[[自由法國|自由法國]]，戰爭結束後任教於[[巴黎政治学院|巴黎政治学院]]和[[国家行政学院_(法国)|國家行政學院]]，1955年至1968年任教於[[索邦大學|索邦大學]]，1970年起任教法蘭西學院。

== 思想 ==
在[[社會學|社會學]]上主張[[自由主義|自由主義]]，主要以研究人和組織之關係以及[[公民社會|公民社會]]、[[工業社會|工業社會]]之諸多問題，但也以[[現實主義|現實主義]]國際關係大師聞名於世。在報紙上的投書大大影響了法國和德國建立共同體，但毫不客氣批評薩特和[[西蒙波娃|西蒙波娃]]的左派傾共、無神思想，說他是「知識圈一霸，在圈內大行恐怖主義，但只在某一層次」。1983年，獲得荷蘭[[伊拉斯謨斯獎|伊拉斯謨斯獎]]。
