'''雷振邦'''（{{bd|1916年|5月1日|1997年|10月20日|catIdx=Lei雷}}），[[满族|满族]]，是[[中国|中国]]著名的[[电影|电影]][[音乐|音乐]][[作曲家|作曲家]]、[[一级作曲|一级作曲]]、[[中国音乐家协会|中国音乐家协会]]理事、[[中国电影音乐协会|中国电影音乐协会]]副会长、[[中国电影协会|中国电影协会]]常务理事、第六届[[全国政协|全国政协]]委员、[[中国民主同盟|中国民主同盟]]盟员。

== 生平 ==
雷振邦出生于[[北京|北京]]，从小就接触[[京戏|京戏]]。七八岁时，能唱京剧小段，还能用[[胡琴|胡琴]]演奏。  

1928年，在[[沈阳|沈阳]][[奉天公立学校|奉天公立学校]]就读，后来升入[[南满中学|南满中学]]。学会吹[[口琴|口琴]]，并加入学校口琴队，成为口琴队的指挥。  

1939年，赴[[日本|日本]]求学，毕业于[[东京|东京]]日本高等音乐学校作曲系。  

1943年，回国，先后在[[北平女子中学|北平女子中学]]和[[惠中女子中学|惠中女子中学]]做音乐教员。[[抗日战争|抗日战争]]胜利后，他组织了一个业余交响乐团。  

1949年，到[[中国电影乐团|中国电影乐团]]工作，1955年，被调到[[长春电影制片厂|长春电影制片厂]]任作曲。

== 作品风格 ==

雷振邦善于吸收中国各少数民族的[[音乐|音乐]]元素作曲，创作了许多具有浓厚少数民族风格、脍炙人口的电影插曲，其中最著名的如为电影《[[刘三姐_(电影)|刘三姐]]》创作的[[壮族|壮族]]风格的[[民歌|民歌]]，为电影《[[冰山上的来客|冰山上的来客]]》创作的[[塔吉克族|塔吉克族]]风格歌曲《[[花儿为什么这样红|花儿为什么这样红]]》，为电影《[[五朵金花|五朵金花]]》创作的[[白族|白族]]风格的歌曲《[[蝴蝶泉边|蝴蝶泉边]]》以及为电影《[[芦笙恋歌|芦笙恋歌]]》创作的[[拉祜族|拉祜族]]歌曲《婚誓》等，至今仍然活跃在中国的音乐舞台上。此外尚有[[朝鲜族|朝鲜族]]、[[景颇族|景颇族]]、[[彝族|彝族]]等风格的电影歌曲。

== 家庭 ==

雷振邦的女儿[[雷蕾|雷蕾]]也是一位作曲家。
