'''雷奥妮·奥索夫斯基'''（{{Lang-de|'''Leonie Ossowski'''}}，{{bd|1925年|8月15日|2019年|2月4日|catIdx=Ossowski}}），原名'''约兰特·冯·布兰登施泰因'''（{{lang|de|'''Jolanthe von Brandenstein'''}}），<ref name="RN" />  [[德国|德国]][[作家|作家]]。她也曾经使用[[笔名|笔名]]“乔·蒂德曼（{{lang|de|Jo Tiedemann}}）”创作作品。<ref name="Munzinger" /> 雷奥妮·奥索夫斯基的作品包括给年轻人创作的小说《大颤栗（{{lang|de|Die große Flatter}}）》，该作品曾多次被翻拍成影视作品；电影剧本《{{Link-en|两个母亲|Zwei Mütter}}》；以及故事和[[非虚构类|非虚构类]]作品。雷奥妮·奥索夫斯基被授予的知名奖项包括由{{Link-en|德国笔会|PEN Centre Germany}}所颁发的[[赫尔曼·凯斯滕奖|赫尔曼·凯斯滕奖]]和{{Link-en|格里姆奖|Grimme-Preis|阿道夫·格里姆奖}}。

== 生平 ==
雷奥妮·奥索夫斯基原名“约兰特·冯·布兰登施泰因（{{lang|de|Jolanthe von Brandenstein}}）”，于1925年8月15日出生在魏玛共和国{{Link-en|波森－西普鲁士|Posen-West Prussia}}的勒尔斯多夫（{{lang|de|Röhrsdorf}}），该地现属于[[波兰|波兰]][[卢布斯卡省|卢布斯卡省]][[弗斯霍瓦县|弗斯霍瓦县]]的{{Link-en|奥索瓦斯恩|Osowa Sień}}。雷奥妮·奥索夫斯基的父亲是{{Link-de|布兰登施泰因 (贵族)|Brandenstein (Adelsgeschlecht)|贵族}}庄园主洛萨·冯·布兰登施泰因（{{lang|de|Lothar von Brandenstein}}）；而她的母亲则是作家露丝·冯·奥斯陶（{{lang|de|Ruth von Ostau}}）。雷奥妮·奥索夫斯基还有一名姐姐{{Link-de|伊芙·梅林|Yvonne Merin}}，她后来成为了一名[[演员|演员]]。[[第二次世界大战|第二次世界大战]]结束的时候，雷奥妮·奥索夫斯基逃亡至[[图林根州|图林根州]]的[[巴特萨尔聪根|巴特萨尔聪根]]；随后又搬迁至[[黑森州|黑森州]]；最后定居在{{Link-en|上施瓦本|Upper Swabia}}。<ref name="Budeus-Budde" /> 期间，雷奥妮·奥索夫斯基尝试过许多工作，例如销售员、工厂工人和照片实验室助理等。

从1950年代开始，雷奥妮·奥索夫斯基就开始用笔名创作[[短篇故事|短篇故事]]。当雷奥妮·奥索夫斯基于1953年访问[[东德|东德]]的时候，她接到了国有电影制片厂[[德国电影股份公司|德国电影股份公司]]的[[剧本|剧本]]创作委托。于是她创作了《{{Link-en|两个母亲|Zwei Mütter}}》的电影剧本，该片由{{Link-en|弗兰克·拜耳|Frank Beyer}}执导并于1957年6月28日首映。<ref>{{cite web |title=Filmdetails: Zwei Mütter (1957) |url=https://www.defa-stiftung.de/filme/filmsuche/zwei-muetter/ |publisher=[[德国电影股份公司|德国电影股份公司]]基金会 |accessdate=2019-02-06}}</ref> 一年后，她又创作了小说《{{Link-de|没有天空的星星|Stern ohne Himmel (Roman)}}》，<ref name="Munzinger" /><ref name="Sandford" /> 随后这部作品也被改编成了电影。雷奥妮·奥索夫斯基在1958年跟随她的家庭搬迁至[[曼海姆|曼海姆]]。1968年，她首次在[[西德|西德]]出版了第一部小说。与此同时，她还出版了故事集、非虚构类书籍、影视剧本及舞台剧本。<ref name="BR" /> 雷奥妮·奥索夫斯基是{{Link-en|德国笔会|PEN Centre Germany}}的成员。

从1970年代开始，雷奥妮·奥索夫斯基开始成为一名[[社会工作者|社会工作者]]。她关切监狱里的在押的年轻人，并为那些被释放的年轻人兴建{{Link-de|集体公寓|Wohngemeinschaft}}。<ref name="Budeus-Budde" />

雷奥妮·奥索夫斯基在1974年访问了她的出生地，并写了一部有关战争及战后人们生活的三部曲小说，以此来表达她对波兰观点的同情。<ref name="BR" /> 1977年，奥索夫斯基创作了《大颤栗（{{lang|de|Die große Flatter}}）》，这是一部写给年轻人的小说。《大颤栗》讲述了两个在曼海姆的无家可归的青年人的故事。这部作品后来被改编成一部三部曲的电视迷你系列剧，并由[[里奇·穆勒|里奇·穆勒]]执导。该剧于1979年首播之后，曾多次获奖。<ref name="Sandford" /><ref name="BR" />

雷奥妮·奥索夫斯基在1980年搬家至[[柏林市|柏林市]]，并在这里居住到她去世为止。<ref name="Piper" /> 雷奥妮·奥索夫斯基于2019年2月4日去世，享壽93岁。<ref name="RN" /> 她与丈夫[[神学家|神学家]]{{Link-de|路易-费迪南·冯·佐贝尔蒂茨|Louis-Ferdinand von Zobeltitz}}拥有7个孩子。<ref name="Budeus-Budde" />

== 所获奖项 ==
雷奥妮·奥索夫斯基于2006年获得{{Link-en|德国笔会|PEN Centre Germany}}的[[赫尔曼·凯斯滕奖|赫尔曼·凯斯滕奖]]。<ref name="BR" /><ref name="Kesten" /> 2004年，奥索夫斯基又荣获{{Link-en|安德雷亚斯·格吕菲乌斯奖|Andreas Gryphius Prize}}。<ref name="Gryphius" /> 同时，她也曾获得{{Link-en|格里姆奖|Grimme-Preis|阿道夫·格里姆奖}}和{{Link-de|曼海姆市席勒奖|Schillerpreis der Stadt Mannheim}}。<ref name="Piper" />
