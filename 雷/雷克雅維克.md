'''雷克雅維克'''（{{lang-is|'''Reykjavík'''}}，{{IPA-is|ˈreiːkjaˌviːk|Icelandic pronunciation:|Is-Reykjavík.oga}}）位於[[法赫萨湾|法赫萨湾]]南部，是斯堪地那維亞[[北歐國家|北歐島國]][[冰島|冰島]]的[[首都|首都]]，亦是冰島最大的城市。雷克雅維克地理上非常接近[[北極圈|北極圈]]，其[[纬度|纬度]]64°08'N，是全世界最北的首都。是全世界（北歐）市容最乾淨無煙城、全靠[[地熱能|地熱能]]的首都圈。在2012年，雷克雅維克人口大概为120,000人（[[大雷克雅未克区|大雷克雅未克区]]人口则超过200,000人），是全冰岛的文化、经济和政务活动中心。

雷克雅維克曾被[[丹麥|丹麥]]及[[挪威|挪威]]管治過，亦在其管治期間發展了本地的經濟及產業，故此，於冰島獨立成主權國家時，該城便被立為首都。現時，冰島議會大廳、政府大樓、國家博物館等設施都設立於雷克雅維克。

== 歷史 ==
雷克雅維克由870年起就成為居民點。根據歷史記載，該地第一批常住居民來自[[斯堪的納維亞|斯堪的納維亞]]。由於地熱能源豐富，[[溫泉|溫泉]]眾多，因此城市上空經常濔漫著如霧的水氣，當地人以為水氣是煙，所以便將此地命名為雷克雅維克，[[冰島語|冰島語]]意指「冒煙之灣岸」。

歷史上，雷克雅維克曾分別隸屬於[[挪威|挪威]]與[[丹麥|丹麥]]。1752年，丹麥國王決定於該地發展工業以推動經濟。此後，製棉、捕魚、農業及造船業都逐漸發展起來。1798年，當地議會開始運作。1801年，成為主教駐地。

[[二次大戰|二次大戰]]初期，[[英国|英]][[美國|美]]兩國多次向身為[[中立國|中立國]]的丹麥要求在雷克雅維克建立基地以應付歐洲戰事的需要，但都被拒。直至1940年4月丹麥被[[納粹德國|納粹德國]]佔領後，冰島同丹麥的聯繫被切斷，冰島宣佈行使外交權和軍事防禦權。5月10日英國出兵佔領了冰島，四艘戰艦到達了雷克雅維克並於當地的碼頭停泊，數小時後，由於當地居民並不抗拒[[同盟國_(第二次世界大戰)|同盟國]]。故此雷克雅維克於二戰後期便成為了英、美兩國的其中一個駐紥點。1941年7月8日，冰島與美國簽訂《戰時防務條約》，美軍代替英軍進駐冰島。1944年6月17日，冰島自丹麥獨立，雷克雅維克成為首都。

== 交通及能源 ==
冰島全國並無設置[[鐵路|鐵路]]，故此對外交通需以[[公路|公路]]為主。另外，全國最大機場[[凯夫拉维克国际机场|凯夫拉维克国际机场]]及第二大機場[[雷克雅未克機場|雷克雅未克機場]]亦位於此城及附近區域，但航班主要為本地航線以及飛往[[格陵蘭|格陵蘭]]、[[法羅群島|法羅群島]]等較鄰近地區。

雷克雅維克的能源主要以[[溫泉|溫泉]]的[[地熱能|地熱能]]為主，溫泉水即使經過管道輸送後，水溫仍高達攝氏90度以上，故此，雷克雅維克極少使用[[石油|石油]]、[[煤|煤]]等資源產生能量，亦因此而令該市受污染程度極低，故亦有「無煙城」的美譽。

==教育==
雷克雅維克有三所大學，分別是[[冰島大學|冰島大學]]、[[雷克雅未克大學|雷克雅未克大學]]以及[[冰島藝術學院|冰島藝術學院]]。
== 地理及氣候 ==
雖然雷克雅維克位處北極圈附近，但由於在冬季受到[[墨西哥湾暖流|墨西哥湾暖流]]以及[[北大西洋暖流|北大西洋暖流]]影響，其氣候比起[[莫斯科|莫斯科]]以及[[渥太華|渥太華]]等同樣位處北極圈附近的城市暖和得多，属于近极性[[海洋性气候|海洋性气候]]（科本法：Cfc）。雷克雅維克位處沿海，東北部有高山圍繞，年降雨量約達850毫米。1月平均氣溫約為[[攝氏|攝氏]]0度，7月則為攝氏11度左右，并自1971年1月30日起，最低气温从未低于零下20摄氏度。<ref>{{cite web |url=http://andvari.vedur.is/vedurfar/yfirlit/medaltalstoflur/Stod_001_Reykjavik.ManMedal.txt |title=Mánaðargildi fyrir stöð 001 - Reykjavík |format=TXT |publisher=Veðurstofa Íslands |accessdate=2010-03-29 }}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>
