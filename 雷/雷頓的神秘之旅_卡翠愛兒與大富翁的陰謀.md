== 登場角色 ==

<div class="notice metadata" id="spoiler"><small>'''[[聲優|聲優]]：遊戲／電視動畫'''</small></div>

=== 主要角色： ===

; {{nihongo|卡翠愛兒・雷頓|カトリーエイル・レイトン||聲優：[[有村架純|有村架純]]／[[花澤香菜|花澤香菜]]}}

本作的女主角，艾尔夏尔·雷顿的女儿，昵称为“卡特莉”。雷頓神秘偵探社的老闆娘，有著開朗活潑的性格，自身熱愛甜點與打扮。

於第20話得知自己並非雷頓教授的親生女兒，而是考古學家卡米拉·艾薩爾的孩子。

; {{nihongo|夏洛|シャーロ||聲優：[[役所廣司|役所廣司]]／[[小杉十郎太|小杉十郎太]]}}

会講人类的语言的狗。人类中只有卡特莉和诺亚能与他对话。作品中的吐槽擔當，同時負責狗界的偵探事件，好好得蒐集情報也是可以像卡特莉一樣靈光一閃解開謎題的。

; {{nihongo|諾亞|ノア・モントール||聲優：[[坂口健太郎|坂口健太郎]]／[[池田恭祐|池田恭祐]]}}

卡特莉的助手兼偵探社員工，性格溫柔，在回憶篇自述在大學時被卡特莉拯救後開始暗戀卡特莉而成為他的助手。

; {{nihongo|阿斯波洛警部|ダージリン・アスポワロ||聲優：[[井上肇|井上肇]]／[[多田野曜平|多田野曜平]]、[[黑澤剛史|黑澤剛史]]（回想）}}

蘇格蘭場警官，常委託偵探探社調查案件。

'''{{Nihongo|傑拉爾丁·羅亞|ジェラルディン・ロイヤー||聲優：[[山村響|山村響]]}}'''

蘇格蘭場的犯罪分析官，視卡翠愛兒為對手。
