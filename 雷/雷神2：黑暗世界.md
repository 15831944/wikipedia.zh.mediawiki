== 劇情 ==
很久以前，[[奧丁_(漫威漫畫)|奧丁]]之父{{link-en|博爾 (漫威漫畫)|Bor (comics)|博爾}}在{{link-en|薩法塔夫漢|Svartálfar|黑暗世界}}領軍擊退{{link-en|魔兵 (漫威漫畫)|Elves (Marvel Comics)|黑暗魔兵}}領袖[[咒神魔雷基|魔雷基]]，其妄圖利用一種具有黑化能力的武器「[[無限寶石#現實寶石_Reality_Stone|乙太]]」來統治宇宙。魔雷基犧牲他的族民而與少許軍隊逃離至宇宙深處沉寂，而波爾便將乙太封印至宇宙最遙遠的地方。在今日，[[索爾_(驚奇漫畫)|索爾]]繼續守護著宇宙九界，屢次百戰百勝而即將繼承王位，但他慢慢感覺到王者不再是他所追求的東西。而在[[地球|地球]]等待他兩年的[[珍·佛斯特|珍·佛斯特]]決定放棄尋找他，回去[[英國|英國]][[倫敦|倫敦]]繼續天體研究。一日，珍的實習生黛絲發現一個出現多數隔空移物傳送門的貨櫃場，珍追蹤信號源時被一股力量強迫拖進入乙太封印地，靠近封印乙太的石柱時被其視為宿主而被強行附身。在宇宙深處沉睡的魔雷基而與其他部下們感覺到乙太呼喚而相繼甦醒，發現地球剛好是五千年才會發生一次的宇宙九界串連成直線之現象「聚合」（{{lang|en|Convergence}}）的中心。

珍甦醒後發現她回到貨櫃場，出去時從黛絲發現她擔心其失蹤而叫來警察。索爾從[[海姆達爾_(漫威漫畫)|海姆達爾]]得知珍失蹤的事情後靠[[彩虹橋_(北歐神話)|彩虹橋]]回到珍面前，但珍有點沒辦法接受索爾當初沒有照約定回來。當警察打算將珍帶回去偵訊時，珍體內的乙太開始釋放能量進行自防，使得索爾在別無選擇下將珍帶到{{le|阿斯嘉 (漫威漫畫)|Asgard (comics)|阿斯嘉}}治癒和看護。不久，魔雷基領軍對阿斯嘉進行大規模進攻，派遣心腹戰士{{link-en|咒士 (漫威漫畫)|Kurse|阿格姆}}進入地牢臥底時裏應外合，阿格姆也釋放所有罪犯造成慌亂；但他唯獨沒放出因為[[復仇者聯盟_(電影)|在地球的深重罪行]]而被關在地牢的[[洛基_(漫威漫畫)|洛基]]。魔雷基隨著一架戰機抵達皇宮，炸毀奧丁的王位來到皇后寢宮，殺死守護珍的芙瑞嘉。沒及時趕來的索爾電焦魔雷基的右臉，而剩餘敵軍則集體撤退。當晚，阿斯嘉德舉行芙瑞嘉與其他死難者的葬禮，所有人含著悲傷看著芙瑞嘉火化離去，而地牢中的洛基得知芙瑞嘉的死訊後也承受打擊。

索爾預計他們不會撐過下一次進攻，發現連奧丁因為妻子之死而喪失理智，便決定聯合戰友們孤軍奮戰。由於彩虹橋被奧丁關閉，索爾只好找唯一知道國度之間秘密通道的洛基來聯手，兄弟倆帶著珍在海姆達爾、{{link-en|沃斯塔格|Volstagg}}、{{link-en|凡達爾|Fandral}}與[[希芙_(漫威漫畫)|希芙]]的帮助下，從秘密通道到達黑暗世界。為了成功擊潰乙太而讓魔雷基無法使用，兄弟倆便上演一場[[苦肉計|苦肉計]]後全力反擊，但卻錯估形勢，乙太在電擊之下還是完好無損。魔雷基和乙太合體、獲得所有力量後前往地球，索爾盡全力與阿格姆決鬥卻遭到擊敗，洛基為了救索爾而身受重傷，但他也趁機啟動阿格姆身上的黑洞炸彈而讓他被炸死。索爾本來想對洛基搶救並對他正名，但洛基在道完歉後還是死去。僅剩兩人的索爾和珍原本被困在山洞裏，但珍的手機突然響起而使得他們倆通過一個傳送門而成功回到倫敦貨櫃場。

兩人回去後和黛絲和艾瑞克·賽維格討論先前的重力異常現象，艾瑞克將他近來所調查的聚合地點列出後，發現所有地點連接中心在倫敦[[格林威治|格林威治]]。隔天，魔雷基搭乘指揮塔登陸[[格林威治大學|格林威治大學]]廣場上，索爾則全力和他對戰，但珍靠一些空間轉換儀器在廣場上製造多數傳送門，讓兩者之間的戰鬥掉入好幾個國度。人類的空軍戰鬥機試著發射導彈攻擊魔雷基但是因為重力異常現象而失效。之後，魔雷基發現九界已連成一條直線，便開始釋放乙太吞噬各個國度。索爾獨自接過珍的轉換儀器走進乙太產生的風暴中，用錘子和一支儀器棒靠錘釘子方式砸中魔雷基，讓珍將他轉移回黑暗世界。戰爭平息後，索爾在力盡下昏迷不醒，指揮塔因為嚴重受損而崩塌。珍也在拖不動他的情況下決定抱著他一起殉情時，所幸艾瑞克在關鍵時刻將塔轉移。而當時被轉回黑暗世界的魔雷基，直接被轉移回來的指揮塔砸死。

索爾恢復清醒後和珍相擁，之後返回阿斯嘉與奧丁解釋違規一事，表明他決定放棄繼承王位、回去地球守護他深愛的珍與地球。奧丁准許他的請求，索爾高兴地回到地球與珍再度團聚。但當他離開後，坐在王位上的奧丁現出他實為洛基的原型，其靠假造死亡而[[雷神索爾3：諸神黃昏|成功霸佔王位]]。<!--彩蛋一-->後來，沃斯塔格與希芙將剩餘的乙太封存至盒中後，將其交給收藏着全宇宙最神聖物品的[[收藏者_(漫威漫畫)|坦納里·帝凡]]，表明此物放在收藏着「[[無限寶石#空間寶石_Space_Stone|宇宙魔方]]」的阿斯嘉寶庫並不安全。當兩人離開後，坦納里握著乙太盒說「[[銀河護衛隊_(電影)|到手一個，還差五個]]」。<!--彩蛋二

== 評價 ==
[[烂番茄|烂番茄]]网站据262个专业评论显示，有66%的新鲜度，平均得分6.2；[[Metacritic|Metacritic]]上38家媒体综评54分，評價兩極。一般认为本片充满幽默与高风险的动作场面，较符合影迷的口味，但主要批評於缺乏吸引人的劇情。

''[[明報|明報]]''的[[石琪|石琪]]給予3星半（最高5星），他說：「《雷神奇俠2》最生動有趣是現代倫敦戲，科學家[[妮妲莉·寶雯|妮妲莉·寶雯]]與女門徒及男實習生都妙趣橫生，在石柱陣裸跑的瘋狂教授亦有奇趣。神國惡鬥也增添了奇情，演雷神的[[克里斯·海姆斯沃斯|基斯·咸士禾夫]]愈來愈有型，演反叛弟弟的[[汤姆·希德勒斯顿|湯·希杜斯頓]]好戲，到底是忠是奸相當懸疑。」<ref>{{cite newspaper|url=http://ol.mingpao.com/cfm/star20.cfm?File=20131103/sae03/myb1.txt|title=石琪說戲﹕中國式情迷藍茉莉|date=2013年11月3日|newspaper=''[[明報|明報]]''}}</ref>

== 聯繫 ==
電視劇《[[神盾局特工|神盾局特工]]》中，第八集「水井（The Well）」與第十五集「魅惑女神（Yes Men）」與此部電影有聯繫。
*第八集的發生時間位於此部電影結束後，神盾局的小組開始來到格林威治大學中收拾殘局，其中包括損毀的黑暗之塔。
*第十五集中，一名來自於阿斯嘉的女囚犯在電影中黑暗精靈攻打時暗中逃離，並利用洛基使用的秘密通道逃到地球，而迫使希芙利用彩虹橋來到地球而與神盾局並肩作戰。
