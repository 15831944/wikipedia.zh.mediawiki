'''雷諾酒廠'''（{{lang-en|'''Renault Winery'''}}，發音 {{IPAc-en|ɹ|ɪ|ˈ|n|ɔː|l|t}}，{{respell|re|NALT|'}}<ref name="Toms">{{Cite web |url = http://www.americanwineryguide.com/winery_reviews/renault-winery/516/ |title = Renault Winery Review |author = Charlie Toms |date = 2013-06-21 |publisher = American Winery Guide |language = en}}</ref>）是一間{{link-en|釀酒廠|Winery}}，位於[[新澤西州|新澤西州]][[大西洋縣|大西洋縣]]的{{link-en|蛋港市 (新澤西州)|Egg Harbor City, New Jersey|蛋港市}}和{{link-en|加洛韋鎮區 (新澤西州)|Galloway Township, New Jersey|加洛韋鎮區}}。<ref>{{Cite web |url = http://www.pressofatlanticcity.com/business/renault-winery-uses-solar-power-to-kill-insects/article_4353b9fc-f530-11e0-9f0c-001cc4c002e0.html |title = Renault Winery uses solar power to kill insects |date = 2011-10-12 |publisher = The Press of Atlantic City |language = en}}</ref>這是新澤西州最古老活躍的釀酒廠。<ref>{{Cite web |url = http://www.nytimes.com/2005/07/17/dining/so-crisp-so-complex-so-unexpected.html |title = So Crisp, So Complex, So Unexpected |author = David Corcoran |date = 2005-07-17 |publisher = The New York Times |language = en}}</ref><ref name="Jackson">[https://books.google.com/books/about/Garden_State_Wineries_Guide.html?id=3hyfSgAACAAJ Jackson, Bart. ''Garden State Wineries Guide.'' (South San Francisco, CA: Wine Appreciation Guild, 2011).] {{ISBN|9781934259573}}.</ref>雷諾酒廠是新澤西州較大的葡萄種植者之一，擁有48英畝種植葡萄，每年生產2萬箱葡萄酒。<ref name="Toms" /><ref>{{Cite web |url = http://www.pressofatlanticcity.com/communities/hammonton_egg-harbor-city/article_d0d5dada-23e9-54b2-a6e9-31899f0e34f4.html |title = Hot summer has area wineries beginning grape harvest early |author = Michelle Lee |date = 2010-09-08 |publisher = The Press of Atlantic City |language = en}}</ref>釀酒廠以其創始人命名。<ref name="Schmidt">[https://books.google.com/books?id=RQIgAAAACAAJ&dq=wines+and+wineries+of+new+jersey+schmidt&hl=en&sa=X&ei=jKVMUZWAOtDy0QGdn4GoAg&ved=0CDkQ6AEwAA Schmidt, R. Marilyn. ''Wines and Wineries of New Jersey.'' (Chatsworth, NJ: Pine Barrens Press, 1999).] {{ISBN|9780937996386}}. Renault Winery is named after Louis Nicolas Renault.</ref>

== 歷史 ==
[[File:Renault_Winery_Champagne_Bottle_color_crop.tif|thumb]]
[[葡萄園|葡萄園]]是在1864年由一名來自[[艾河畔馬勒伊_(馬恩省)|艾河畔馬勒伊]]的法國移民Louis Nicolas Renault首次種植的，他從[[法國|法國]]帶來了原始的[[釀酒葡萄|釀酒葡萄]]。<ref name="Schmidt" /><ref name="Rignani">[https://books.google.com/books?id=99G4B0Acbl4C&pg=PA52&dq=Mareuil-sur-Ay+renault&hl=en&sa=X&ei=g5ezUYPeI4beqgHwsoHIDw&ved=0CC0Q6AEwAA#v=onepage&q=renault&f=false Rignani, Jennifer Papale. ''Images of America: New Jersey Wineries.'' (Charleston, SC: Arcadia, 2008).] {{ISBN|9780738557229}}.</ref>雷諾酒廠在1870年向公眾開放，多年來是美國最大的[[香檳酒|香檳酒]]生產商。<ref name="Rignani" />雷諾酒廠在[[禁酒時期|禁酒時期]]透過獲得許可證來製作{{link-en|聖禮酒|Sacramental wine}}而生存下來，還有製作各種藥用酒，包括由遍及美國的[[藥劑師|藥劑師]]所出售的「Renault Wine Tonic」。<ref>{{Cite web |url = http://abcnews.go.com/Travel/atlantic-city-setting-hbo-boardwalk-empire-series/story?id=11656770 |title = Experience 'Boardwalk Empire' in Atlantic City |author = Scott Mayerowitz |date = 2010-09-20 |publisher = ABC News |language = en}}</ref>Mildred Norman，在她作為{{link-en|Peace Pilgrim|Peace Pilgrim}}旅行橫過大陸前的幾十年，曾在釀酒廠擔任秘書。

[[第二次世界大戰|第二次世界大戰]]後，[[大西洋城|大西洋城]]的{{link-en|政治老大|Political boss}}和詐騙販[[伊努·L·強森|伊努·「努基」·強森]]被雷諾酒廠僱用。釀酒廠一度種植了600英畝葡萄，每年出售50萬件葡萄酒，但近幾十年來的產量大幅下降。<ref name="Rignani" />釀酒廠發展成為[[度假村|度假村]]，於1983年開設[[餐館|餐廳]]，2000年開設[[酒店|酒店]]和第二間餐廳，以及2004年開設[[高爾夫球場|高爾夫球場]]。<ref>Bull, John V.R. "A Delightful Place Grows on the Vine" in ''The Philadelphia Inquirer'' (2 October 1983). Retrieved 8 June 2013.</ref>

== 葡萄酒 ==
雷諾酒廠位於{{link-en|Outer Coastal Plain AVA|Outer Coastal Plain AVA}}，並且以{{link-en|黑巴科|Baco noir}}、[[品麗珠|品麗珠]]、[[赤霞珠|赤霞珠]]、{{link-en|卡尤佳|Cayuga White}}、[[莎當妮|莎當妮]]、[[梅洛葡萄|梅洛葡萄]]、{{link-en|諾亞葡萄|Noah (grape)}}、{{link-en|諾頓葡萄|Norton (grape)}}、{{link-en|小維多|Petit Verdot}}、{{link-en|灰比諾|Pinot gris}}、[[黑比諾|黑比諾]]、[[雷司令|雷司令]]、{{link-en|桑嬌維賽|Sangiovese}}、[[白蘇維翁|白蘇維翁]]、{{link-en|威代爾|Vidal blanc}}來釀製葡萄酒。雷諾酒廠還以[[藍莓|藍莓]]釀製[[果酒|果酒]]，而且是美國唯一一間製造藍莓香檳的釀酒廠。<ref>{{Cite web |url = http://www.washingtonpost.com/wp-dyn/content/article/2002/07/17/AR2005033116179.html |title = New Jersey Uncorked |author = David A. Taylor |date = 2002-07-17 |publisher = The Washington Post |language = en}}</ref>
