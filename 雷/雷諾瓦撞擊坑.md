'''雷諾瓦撞擊坑'''（'''Renoir'''）是一個位於[[水星|水星]]的撞擊坑，直徑219.88公里，座標橫跨15.71-20.87°S, 49.18-54.61°W。1976年[[國際天文學聯合會|國際天文學聯合會]]以法國畫家[[皮耶-奧古斯特·雷諾瓦|皮耶-奧古斯特·雷諾瓦]]命名該撞擊坑<ref name="gazetteer">{{cite web |url = http://planetarynames.wr.usgs.gov/Feature/4995?__fsk=290462071 |title = Renoir |work = Gazetteer of Planetary Nomenclature |publisher = International Astronomical Union and Working Group for Planetary System Nomenclature |date = 7 March 2011 |accessdate = 17 August 2012}}</ref>。

== 地質 ==
雷諾瓦撞擊坑是一個[[複雜撞擊坑|環峰盆地]]，在水星上還有數個類似的撞擊盆地，包含[[拉德特拉迪盆地|拉德特拉迪盆地]]和[[拉赫曼尼諾夫撞擊坑|拉赫曼尼諾夫撞擊坑]]。雖然這些盆地在水星上相對年輕，但雷諾瓦撞擊坑是水星上這類撞擊坑中最老的。因為雷諾瓦撞擊坑較年老，因此在坑內可以看到較多[[地质构造|地质构造]]的痕跡，以及內部比其他環峰盆地更多的撞擊坑<ref>{{cite web |url = http://messenger.jhuapl.edu/gallery/sciencePhotos/image.php?gallery_id=2&image_id=939 |title = Peak-Ringed Renoir |date = 17 August 2012 |publisher = NASA/Johns Hopkins University Applied Physics Laboratory/Carnegie Institution of Washington |accessdate = 17 August 2012 |deadurl = yes |archiveurl = https://web.archive.org/web/20150908031200/http://messenger.jhuapl.edu/gallery/sciencePhotos/image.php?gallery_id=2&image_id=939 |archivedate = 8 九月 2015 }}</ref>。因此雷諾瓦撞擊坑被認為形成於水星歷史中隕石撞擊最高峰期的末期，在水星年代屬於科伊伯紀。在分類上雷諾瓦撞擊坑屬於多環盆地<ref>{{cite web |url = http://astrogeology.usgs.gov/PlanetaryMapping/DIGGEOL/mercury/h6/h6.pdf |title = Geologic Map of the Kuiper (H-6) Quadrangle of Mercury |year = 1981 |publisher = NASA, US Geologic Survey |last1 = De Hon |first1 = R.A. |last2 = Scott |first2 = D.H. |last3 = Underwood, J.R., Jr. |accessdate = 18 August 2012}}</ref>。雷諾瓦撞擊坑內有一個高反照率的區域，被認為是火山活動的遺跡<ref>{{cite web |url = http://www.planetary.org/blogs/emily-lakdawalla/2010/2387.html |title = LPSC: Wrapping up Tuesday: The Moon, Mars, Mercury, Vesta, and back to Mars |last = Lakdawalla |first = Emily |date = 12 March 2010 |publisher = The Planetary Society |accessdate = 18 August 2012}}</ref>。該撞擊坑和拉赫曼尼諾夫撞擊坑都是水星上環峰內有高反照率區域的撞擊盆地<ref>{{cite journal |url = http://www.sciencemag.org/content/329/5992/668.full |title = Evidence for Young Volcanism on Mercury from the Third MESSENGER Flyby |date = 6 August 2010 |journal = Science |volume = 329 |issue = 5992 |pages = 668–671 |doi = 10.1126/science.1188186 |last1 = Prockter |first1 = Louise M. |last2 = Ernst |first2 = Carolyn M. |last3 = Denevi |first3 = Brett W. |last4 = Chapman |first4 = Clark R. |last5 = Head |first5 = James W., III|others = Sean C. Solomon, Thomas R. Watters, Robert G. Strom, Gabriele Cremonese, Simone Marchi, Matteo Massironi |accessdate = 18 August 2012|bibcode = 2010Sci...329..668P }}</ref>。

雷諾瓦撞擊坑內有一個同心圓環結構，因此它屬於「同心環撞擊盆地」<ref name="Rodionova"/>。它內部的環結構相當明顯，不過類似的撞擊盆地通常其外環會比內環更加明顯。類似雷諾瓦撞擊坑的撞擊盆地在其內部和周圍區域都有深谷存在。水星和其他類地行星相較之下較小的半徑和較低的質量代表了它的撞擊盆地（例如雷諾瓦撞擊坑和[[羅丹撞擊坑|羅丹撞擊坑]]）比其他類地行星大；因此，在月球等質量和體積比水星更小天體上的多環撞擊盆地，例如月球的[[赫茨普龍環型山|赫茨普龍環型山]]和[[東方海|東方海]]的直徑都較水星上的更大<ref name="Rodionova">{{cite journal |url = http://adsabs.harvard.edu/full/1980LPI....11..949R |title = Multi-Ring Basins on the Moon, Mars and Mercury |last1 = Rodionova |first1 = Z.F. |last2 = Skobeleva |first2 = T.P. |journal = Lunar and Planetary Science |pages = 949–951 |bibcode = 1980LPI....11..949R |year = 1980 |accessdate = 18 August 2012}}</ref>。
