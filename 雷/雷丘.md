'''雷丘'''（{{lang-ja|'''ライチュウ'''}}，{{lang-en|'''Raichu'''}}，香港舊譯：'''雷超'''）是[[精靈寶可夢|精靈寶可夢]]系列登場的{{NUMBEROFPOKEMON}}種虛構[[角色|角色]]（[[怪物|怪獸]]）中的一種。

== 特徵 ==
雷丘屬於電屬性的鼠類[[寶可夢|寶可夢]]，身體的條紋比[[皮卡丘|皮卡丘]]還深，臉頰為黃色，尾巴細長的閃電圖形暗喻著其強大的電力。
阿罗拉样子的雷丘為電屬性與超能力屬性，毛色为深褐色，双手双脚变为白色，体态也更加圆润。黄色耳朵相对于其他地区的雷丘的耳朵更加圆滑，并且在内部有卷曲，很像超级差不多娃娃的耳朵；眼睛的颜色从黑色变为由蓝至黄的渐变色；鼻子也变成了浅色；电力袋和尾巴末端的黄色更加明亮；尾巴的形状变得很圆滑，雷丘可以踩上去作為衝浪板。

== [[神奇寶貝遊戲列表|遊戲]]中的雷丘 ==
由[[皮卡丘|皮卡丘]]使用「雷之石」後進化而成。豐緣圖鑑編號为157，全國圖鑑編號為26，最早出現於精靈寶可夢紅／綠版，擅長招式與皮卡丘相同。不過在遊戲中，進化後就無法利用等級的提升學習新招式。

特性為「靜電」。「靜電」的效果是受到近身攻擊時對手可能陷入麻痺狀態。

== 動畫中的雷丘 ==
枯葉市道館館主馬志士擁有一隻雷丘，曾在與小智的第一次比賽中打敗小智的皮卡丘，後來在第二次比賽中被小智的皮卡丘成功反敗為勝。
