{{印尼历史}}
'''荷屬東印度'''是指1800年至1949年[[荷蘭|荷蘭]]人所統治的[[印度尼西亞|印度尼西亞]]。1596年，荷兰人[[霍特曼|霍特曼]]率领的一支荷兰船队到达[[爪哇岛|爪哇岛]]的[[万丹_(印尼)|万丹]]。1603年，荷兰在爪哇建商站；1605年，征服盛产香料的[[马鲁古群岛|马鲁古群岛]]中的[[安汶岛|安汶岛]]、[[帝利岛|帝利岛]]。1619年，攻占爪哇岛上的[[雅加达|雅加达]]，命名[[巴达维亚|巴达维亚]]，作为[[荷兰东印度公司|荷兰东印度公司]]在东方的总部。在荷兰人到来之前，[[葡萄牙|葡萄牙]]人已经在印尼地区建立了一些商站。之后，[[英国|英国]]人也登陆印尼，建立[[殖民地|殖民地]]。而[[荷兰|荷兰]]变成了这些欧洲国家裡面占领最多领土的国家。荷兰最初通过[[荷兰东印度公司|荷兰东印度公司]]对这些地区实行殖民统治。1799年东印度公司解散后殖民地被[[荷兰政府|荷兰政府]]接管，史称荷属东印度。

[[第一次世界大戰|第一次世界大戰]]結束後，當地人開始爭取自治。到了[[第二次世界大戰|第二次世界大戰]]，當荷蘭本土被[[德國|德國]]完全佔領後，荷蘭政府宣佈終止與當時最大貿易伙伴[[日本|日本]]的貿易，觸發日本對印尼的侵略行動。這可以說是印尼得以獨立的原因。[[日本二戰投降|日本二戰投降]]的消息傳到印尼之後，[[蘇加諾|蘇加諾]]立即在之後一日發表印尼獨立宣言。之後在短短五日間，印尼全國都宣告脫離荷蘭政府的管治，荷兰欲夺回殖民地，双方展开三年多的[[印度尼西亚独立战争|印荷战争]]。有[[日軍|日軍]]兩千多人改名換姓，偽裝為當地人，他們幫助印尼訓練士兵與荷蘭作戰，又[[联合国|联合国]]向荷蘭施壓的緣故，印尼在战争中占了上风。最終荷蘭當局在1949年与印尼签署{{le|圆桌会议协定|Dutch–Indonesian Round Table Conference}}，宣佈放棄對印尼的管治權，使印尼得以正式獨立，[[苏加诺|苏加诺]]當選为印尼第一任总统。

== 名称 ==
印度（Indies）一词源自[[拉丁语|拉丁语]]Indus，意义相同。印尼群岛被称为[[东印度|东印度]]，与[[加勒比海|加勒比海]]的[[荷属西印度|荷属领地]]相区分。联合东印度公司在1620年的文献中已经开始使用该词。

[[File:Myristica_fragrans_-_Köhler–s_Medizinal-Pflanzen-097.jpg|250px]]的[[肉豆蔻|肉豆蔻]]，曾為世界上最具價值的商品。歐洲因而將殖民勢力帶入印度尼西亞。]]

== 背景 ==
今日的印尼群岛本分为诸个邻海与内陆城邦。从1595年起，[[荷兰人|荷兰人]]开始到东南亚收购香料，利润可达四倍以上，而在市场竞争日趋激烈的情况下，荷兰王国政府将这些商人统一纳入旗下[[荷兰|荷兰]][[联合东印度公司|联合东印度公司]]内。东印度公司被授权对外宣布战争、修筑城垒并签订條约。在爪哇岛开埠贸易不久，[[东印度公司|东印度公司]]就将[[巴达维亚|巴达维亚]]设为首府，成为公司在[[亚洲贸易|亚洲贸易]]的枢纽。在维护对豆蔻、丁香等本地香料的市场垄断时，公司还引进了[[可可豆|可可豆]]、[[茶叶|茶叶]]、[[烟草|烟草]]、[[橡胶|橡胶]]、[[鸦片|鸦片]]等非原生[[经济作物|经济作物]]。在日益庞大的战争经费和严重的贪污情况的影响下，联合东印度公司的资金大幅萎缩，最后在1800年1月1日宣布破产，其产业有荷兰政府直接接收。当时领土疆域包括爪哇岛周围、苏门答腊岛部分、马鲁古岛及万鸦老、望加锡等海港。
