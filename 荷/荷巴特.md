'''霍巴特'''（{{lang-en|'''Hobart'''}}）是[[澳洲|澳洲]][[塔斯馬尼亞州|塔斯馬尼亞州]][[首府|首府]]，是该州人口最多的城市；為僅次於[[悉尼|悉尼]]的澳洲第二古老城市，第十一大城市，城市人口超過廿二萬，面積1357.3平方公里。

作為塔斯馬尼亞州的金融和行政中心，荷巴特也是澳大利亚、[[法國|法國]]兩國的南極科研母港。

== 歷史 ==
當地的原住居民為半遊牧的[[摩鴻尼亞部族|摩鴻尼亞部族]]的成員。1803年，儘管當時已有法國探險者存在，[[英國|英國]]人在[[萊斯頓灣|萊斯頓灣]]成為了首批刑事定居者。1804年，居地移至荷巴特的[[朔羅芬灣|朔羅芬灣]]現址。後來一系列的血腥衝突在歐洲人和原住居民之間發生。一如在美洲發生的，歐洲人所帶來的疾病灭绝了土著居民。遺下的地域則迅速地由外來自由定居者和囚犯所佔領。

由於[[德文特河|德文特河]]口是全澳最好的深水港之一，世界第二深的天然港（僅次於巴西[[里約熱內盧|里約熱內盧]]），亦是南大洋的[[鯨魚|鯨魚]]和[[海豹|海豹]]貿易中心，因此它迅速地發展成一個主要港口，並有相關的工業，比如造船業。

1836年2月，[[查爾斯·羅伯特·達爾文|查爾斯·羅伯特·達爾文]]，作為科学考察船[[猎犬号|猎犬号]]的一员，到訪了荷巴特鎮。在《[[獵犬之行|獵犬之行]]》一書中，他叙述自己「對少數的大房屋尤其深刻，不论是已建或興建中。」據荷巴特鎮1835年人口普查，當地有13,826居民，而全塔斯馬尼亞州則有36,505人口。

1836到1843年，塔斯馬尼亞省/荷巴特總督是赫赫有名的[[約翰·富蘭克林|約翰·富蘭克林]]爵士。

1842年，荷巴特鎮成為一個城市，並於1875年改名荷巴特。

== 地理 ==
[[File:LGA_Tas_SE_Hobart.png|thumb]]
荷巴特處於[[塔斯马尼亚岛|塔斯马尼亚岛]]東南的[[德文特河|德文特河]](Derwent Water)口。市中心位於西岸，鄰接[[朔羅芬灣|朔羅芬灣]]。荷巴特市郊區沿河分散及至[[威靈頓山|威靈頓山]](Mount Wellington)腳下，覆蓋河東、西兩岸的相當大地域。

=== 氣候 ===

荷巴特属[[温带海洋性气候|温带海洋性气候]]，四季分明。作為南半球的一個城市，夏季是十二月至二月，冬季六月至八月。

下列為當地其中一些平均氣溫：
* 平均夏季最高氣溫21.1 °C (70 °F) 
* 平均夏季最低氣溫11.9 °C (53 °F) 
* 平均冬季最高氣溫12.5 °C (54.5 °F) 
* 平均冬季最低氣溫5.4 °C (41.5 °F) 
* 平均年度雨量628mm (25 inches) 
* 平均雨量最多月份－十月，63mm (2.5 inches) 
* 平均雨量最少月份－二月，40mm (1.8 inches)

秋冬時會看見[[威靈頓山|威靈頓山]]山頂下雪。近來夏季也時有山頂積雪及冰雹。

== 康樂 ==
荷巴特以[[帆船|帆船]]比賽而世界聞名。它是[[悉尼-霍巴特帆船大賽|悉尼—荷巴特帆船比賽]]的終點。帆船比賽在[[耶誕節|耶誕節]]後的一天在悉尼開始。因此慶祝帆船的抵達是[[荷巴特夏季節慶|荷巴特夏季節慶]]的一部份。

荷巴特夏季節慶是耶誕節後開始，直至一月中旬完結的飲食節目。在[[薩拉曼卡_(塔斯马尼亚)|薩拉曼卡]]（Salamanca）舉行的[[塔州之味|塔州之味]]是其主要節目之一，當地遊客能品嘗到美味的食物和酒。

每年十月，[[皇家荷巴特表演|皇家荷巴特表演]]都會在格蘭洛奇的[[皇家表演場|皇家表演場]]舉行。該節目聚焦於塔州人的農業開發，節目包括畜牧評選及砍木比賽。表演為期四日，星期三始至星期六（十月第四個星期六）結束。當中星期四是塔州南部的公眾假期。

自1991年起，荷巴特是[[泰加塔州|泰加塔州]]賽車節目的終點，於每年四月舉辦。

[[塔斯马尼亚皇家植物園|塔斯马尼亚皇家植物園]]（Royal Botanical Garden）每年一度的[[鬱金香節慶|鬱金香節慶]]，亦是當地受歡迎的喜慶之一。

== 娛樂 ==
荷巴特娛樂并不豐富，因为該城市只有兩家戲院及小量商場食市。但逢週末可到比較熱鬧的薩拉曼卡集市（Salamanca）购買新鮮蔬果。

荷巴特是塔州管絃樂團的基地，駐於市中心河邊的聯邦音樂演奏廳，提供全年的演奏節目。荷巴特的[[皇家劇場|皇家劇場]]是澳洲最古老的劇場。荷巴特的仙蒂湾（Sandy Bay）还是皇家遊艇俱樂部的基地。

== 交通 ==
荷巴特有通往市郊區的巴士，以及到各地的长途巴士。从荷巴特到北部重镇[[朗塞斯顿|朗瑟士敦]]开车约需2小时车程。从塔斯马尼亚南部通往西部、北部和东部各镇的公路都在荷巴特汇集，交通便利。荷巴特机场则位于市区东北部，距离市中心约半小时车程，是一座小型的机场，可供降落类似[[空中巴士|空中巴士]][[A320|A320]]这样的支线客机，每天该机场有好几次班机往返荷巴特与[[墨尔本|墨尔本]]。
