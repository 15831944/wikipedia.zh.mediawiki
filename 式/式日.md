《'''式日'''》（{{lang-ja|式日}}）是日本動畫導演[[庵野秀明|庵野秀明]]在2000年12月7日在[[東京寫真美術館|東京寫真美術館]]公開的真人電影，並由動畫工作室[[吉卜力工作室|吉卜力]]旗下的{{nihongo|「Studio Kajino」|スタジオカジノ}}工作室擔任本作的製片商。

作品名稱「式日」在劇中是代表「執行儀式的日子」意思。<ref name="story">{{Cite web |url=http://shikijitsu.ube.ac/story.html |title=《式日：Story》 |work=Studio Kajino | accessdate=2012-08-31}}</ref>

== 概要 ==
本作是庵野秀明繼1998年翻拍[[村上龍|村上龍]]的小說《[[狂戀高校生|狂戀高校生]]》後，個人的第二部真人電影。<ref>{{Cite web |url=http://shikijitsu.ube.ac/staff_cast.html |title=《式日：STAFF&CAST》 |work=Studio Kajino | accessdate=2012-08-31}}</ref><ref name="story" />方式是以[[DV|DV]]等攝影器材，所完成的35mm格式的影片。作品場景則挑選在庵野秀明的故鄉[[山口縣|山口縣]]。

電影的內容改編自演員兼小說家[[藤谷文子|藤谷文子]]在2000年發表的作品{{nihongo|《逃避夢・烤犬》|逃避夢・焼け犬}}，而藤谷文子本人也在劇中飾演女主角。劇中男主角則由先前執導過《[[情書_(電影)|情書]]》、《[[燕尾蝶_(電影)|燕尾蝶]]》電影的導演[[岩井俊二|岩井俊二]]擔當，同時也是岩井俊二個人首次擔任劇中演員的作品。<ref name="story" />

== 劇情 ==
故事的開頭是一位回到故鄉的男性電影導演，他先前雖然在東京工作時獲得事業上的成功，但因為一直無法做出心中想拍攝的風格作品、之後對自己有所不滿返回老家。

當他回到家鄉時看到在數條鐵路交會的地方，出現了一名行為奇異的女子，而那名女子口中唸著「明天是我的生日」。

之後男性導演對那名女子產生了好奇心，便想以她為題材來拍攝作品，並慢慢與那名女子倆人每日一同生活。
