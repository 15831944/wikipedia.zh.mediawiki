'''垫江县'''位于[[重庆市|重庆市]]中部，属于[[重庆五大功能区#渝东北生态涵养发展区|渝东北生态涵养发展区]]。幅员面积一千五百一十八平方公里，总人口九十五万，下辖两个[[乡|乡]]两个[[街道|街道]]二十二个[[镇|镇]]。

垫江是国花[[牡丹|牡丹]]的发源地，每年当[[垫江牡丹|垫江牡丹]]竞相开放时， 当地都会在[[垫江体育馆|垫江体育馆]]举办演唱会，以此开启[[垫江牡丹文化节|牡丹文化节]]。

垫江的陆路交通十分发达。[[沪渝高速|沪渝高速]]、[[沪蓉高速|沪蓉高速]]在境内[[太平镇|太平]]互通，[[垫江站|垫江站]]位于[[渝万城际铁路|渝万城际铁路]]中点，通过[[长垫梁货运铁路|长垫梁货运铁路]]可利用长寿港，而[[垫丰武高速|垫丰武高速]]则连接了同处于[[重庆五大功能区#渝东北生态涵养发展区|渝东北生态涵养发展区]]的[[丰都县|丰都县]]与深处[[重庆五大功能区#渝东南生态保护发展区|渝东南生态保护发展区]]的[[武隆区|武隆区]]。

== 历史沿革 ==
垫江建县于[[西魏恭帝|西魏恭帝]]二年(555)年，迄今已有1400多年的历史。[[巴国|巴国]]时期，垫江地区属巴国临江县地。巴国灭亡后，垫江先后隶属于秦巴郡及[[汉朝|汉朝]]至[[晋朝|晋朝]]时期的[[巴郡|巴郡]]临江县地。

[[唐朝|唐朝]]初年，朝廷改临江县地为清水县，后又更其名为桂溪县。[[宋朝|宋朝]]时期，桂溪县并入垫江县。[[元朝|元朝]]初年，垫江县并入丰都县，同时设置临江巡检司管理原垫江县地区。元朝末年，又复置垫江县。

[[清朝|清朝]]时期，垫江县先后隶属于[[重庆府|重庆府]][[忠州_(唐朝)|忠州 (唐朝)]]和[[忠州直隶州|忠州直隶州]]。[[民国|民国]]时，垫江又先后隶属于东川道和第十行政督察区。[[中华人民共和国成立|中华人民共和国成立]]后，垫江仍先后隶属于[[大竹县|大竹]]、[[涪陵|涪陵]]专区。直至重庆升格为[[直辖市|直辖市]]，垫江才改为由重庆市直接管辖的县级行政区<ref>{{cite web |url=http://dj.cq.gov.cn/aboutus/Index_3.html |title=历史沿革 |accessdate=2017-01-11 |publisher=垫江县人民政府 |deadurl=no |archiveurl=https://web.archive.org/web/20151208231639/http://dj.cq.gov.cn/aboutus/Index_3.html |archivedate=2015-12-08 |df= }}</ref>。

== 行政区划 ==
垫江下辖两乡两街道和二十二个镇。在二十二个镇中，又划设了包括[[澄溪镇|澄溪镇]]、[[高安镇_(垫江县)|高安镇]]在内的两个县级经济中心镇，及分别作为南部片区中心的[[坪山镇|坪山镇]]、中南片区中心的[[高峰镇_(垫江县)|高峰镇]]和北部片区中心的[[周嘉镇|周嘉镇]]等三个片区中心镇<ref>{{Cite web |url= http://jgh.cqsdj.gov.cn/Html/19/ghfc/2013-5/77664.html |title= 垫江县城乡总体规划 |date= 2013-05-13 |publisher= 垫江县规划局 |language= zh-hans |accessdate= 2017-02-20 |deadurl= no |archiveurl= https://web.archive.org/web/20151210211442/http://jgh.cqsdj.gov.cn/Html/19/ghfc/2013-5/77664.html |archivedate= 2015-12-10 |df=  }} </ref>。

== 地理 ==
垫江位于重庆市中部，地处[[华蓥山|华蓥山脉]]东部。幅员面积1518平方公里，地理坐标为东经107°13′至107°40′，北纬29°38′至30°31′。西临[[四川省|四川省]][[邻水县|邻水县]]、[[大竹县|大竹县]]，北靠[[梁平县|梁平县]]，东与[[忠县|忠县]]、[[丰都县|丰都县]]相连，南部地区则与[[长寿区|长寿区]]、[[涪陵区|涪陵区]]相接。

垫江地貌以[[丘陵|丘陵]]为主，境内东西部山岭耸峙，切割成谷，千米山峰屹立，山间槽地交错；中部高滩河纵惯全境，一系列溪河、小沟冲、平坝镶嵌其中，形成绵亘起伏的丘陵。最高海拔1183m，最抵海拔320m，全县地势北高南低，按高程可分为边缘局部中低山区、西南局部及边缘低山区和中部丘陵河谷三部分。

地震因素方面，垫江地震基本烈度为小于6度区，地震动峰值加速度为0.05g，特征周期为0.25s<ref>{{cite web |url=http://www.djgtfg.gov.cn/Html/12/dzhj/gyxdzzl/2009-11/82056.html |title=垫江县自然地理及地质条件 |date=2009-11-30 |accessdate=2017-04-28 |publisher=重庆市垫江县国土房管局 |deadurl=no |archiveurl=https://web.archive.org/web/20180611110815/http://www.djgtfg.gov.cn/Html/12/dzhj/gyxdzzl/2009-11/82056.html |archivedate=2018-06-11 |df= }}</ref>。

=== 气候 ===
垫江地处[[亚热带|亚热带]]，气候温和湿润，四季分明；属[[季风气候|季风气候]]，春早冷暖多变，夏热常有干旱发生，秋凉多连绵阴雨，冬冷无严寒。年平均气温17.0摄氏度，历年平均降水量为1183.1毫米<ref>{{Cite web|url=http://www.cq.xinhuanet.com/2008/2008-04/07/content_12895819.htm|title=垫江的基本情况|date=2008-04-07|publisher=新华网重庆频道|accessdate=2017-04-28}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }} </ref>。

=== 特产 ===
垫江的特产包括可食用的[[垫江石磨豆花|垫江石磨豆花]]、垫江砸酒、垫江牛头、黄沙白柚以及垫江丹皮、酱瓜，和文化工艺产品如垫江交雕、棕竹手杖以及牛角烟具等<ref>{{cite web |url=http://shop.bytravel.cn/produce/index711.html |title=垫江特产介绍 |accessdate=2017-09-25 |publisher=博雅特产网 |deadurl=no |archiveurl=https://web.archive.org/web/20171007121508/http://shop.bytravel.cn/produce/index711.html |archivedate=2017-10-07 |df= }}</ref>。

== 经济 ==
2015年，垫江县实现[[地区生产总值|地区生产总值]]2398394万元，同比增长11.1% 。其中：[[第一产业|第一产业]]增加值为352883万元，[[第二产业|第二产业]]增加值为1196810万元，[[第三产业|第三产业]]增加值为848701万元，分别增长5.1%、12.6%、11.5%。产业结构由2014年的14.5：57.0：28.5调整为14.7：49.9：35.4<ref>{{Cite web |url= http://jtj.cqsdj.gov.cn/Html/25/zwgk/tjgb/2016-4/128497.html |title= 2015年垫江县国民经济和社会发展统计公报 |date= 2016-04-15 |publisher= 垫江县统计局 |accessdate= 2017-04-28 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }} </ref>。

2015年，垫江县的[[人均GDP|人均GDP]]为35442.59元，位列重庆38区县第23位，处于中游水平，但低于全市平均值<ref>{{Cite web |url= http://www.phbang.cn/finance/data/152421.html |title= 2015年重庆各区县GDP和人均GDP排名 |date= 2016-03-15 |publisher= PHBang |accessdate= 2017-04-28 |deadurl= no |archiveurl= https://web.archive.org/web/20161130011400/http://www.phbang.cn/finance/data/152421.html |archivedate= 2016-11-30 |df=  }} </ref>。
