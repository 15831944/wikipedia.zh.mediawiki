'''垫江实验中学'''全称为重庆市垫江实验中学校，是重庆市[[重点中学|重点中学]]之一，坐落于美丽的[[牡丹|牡丹]]之乡重庆市[[垫江县|垫江县]]。学校建立于1930年。前身为垫江师范学校，在2000年转型为普通高完中，并于2011年升格为[[重庆市|重庆市]]重点中学。<ref>{{Cite web |url= http://djsyzx.com/jianjie/rongyu/87.html |title= 《重庆市人民政府关于同意重庆市垫江实验中学校等10所学校为重庆市重点中学的批复文件{渝府【2011】6号》文件实物 |language= zh-hans |accessdate= 2015-12-10 |publisher=重庆市人民政府}}} </ref>

==历史沿革==
*1930年，成立垫江乡村女子师范学校。
**1931年，更名为乡村师范科。
**1932年，再次更名为垫江女子乡村师范学校。
**1994年，经[[国家教委|国家教委]]评估，成为标准化师范学校。
**1998年，垫江教师进修校被并入，更名为垫江[[师范|师范]]进修学校<ref>{{cite web |url=http://djsyzx.com/jianjie/jieshao/102.html |title=垫江实验中学历史沿革 |accessdate=2015-12-10 |publisher=重庆市垫江实验中学校  }}</ref>。

*2000年，学校[[转型|转型]]为普通高完中。
**2002年，垫江县桂溪中学被并入。
**2009年，再次更名为重庆市垫江实验中学校。
**2011年，升格为重庆市重点中学。

== 教育情况 ==
*学生：约5200人
*教职工：有教职工298人，其中182名[[教师|教师]]曾获得重庆市市级、垫江县[[县级|县级]]优秀教师。
