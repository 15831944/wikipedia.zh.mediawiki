{{作品名稱|name=''The Bling Ring''|hk=閃閃靚賊|tw=星光大盜|cn=珠光宝气}}是由[[蘇菲亞·柯波拉|蘇菲亞·柯波拉]]自編自導的電影，改編自一宗於2008年至2009年間發生的真實事件。2012年3月於[[洛杉磯|洛杉磯]]開始拍攝，上映日期為2013年6月。

電影講述一群青少年一起偷偷潛入好萊塢貝弗利山莊的明星家中，偷竊貴重物品。影片改編自真實案件《'''[[:en:Bling_Ring|好萊塢竊盜案]]'''》:2008年10月－2009年8月，一個以青少年為主體的盜竊團伙屢次成功摸入好萊塢貝弗利山莊多位名流明星的家中，共盜取了超過300萬美元價值的貴重物品。其中遭遇洗劫的包括[[芭莉絲·希爾頓|芭莉絲·希爾頓]]、[[琳賽·蘿涵|琳賽·蘿涵]]、[[奧蘭多·布魯|奧蘭多·布魯]]、[[梅根·福克斯|梅根·福克斯]]、[[瑞秋·比爾森|瑞秋·比爾森]]等人。導演科波拉表示，她希望能藉此片以冷靜的視角向所有人展現，如今的名流文化和現代文明。<ref>{{cite web |url=http://gb.cri.cn/27564/2012/05/23/3465s3694901.htm |title='《炫富幫》首曝宣傳照 艾瑪·華森演繹科波拉新作'May 23, 2012}}</ref>

== 製作 ==
=== 演出陣容 ===
'''炫富幫'''的演員陣容除了有《[[哈利波特|哈利波特系列]]》女星[[艾瑪·華森|艾瑪·華森]],還有因《[[美國恐怖故事|美國恐怖故事]]》而迅速成名的[[泰莎·法蜜嘉|泰莎·法蜜嘉]]及多位新人－{{link-en|張凱蒂|Katie Chang}}、 {{link-en|伊瑟爾·布魯薩爾|Israel Broussard}} 、{{link-en|卡洛斯·米蘭達|Carlos Miranda}}演出偷竊同黨，此外，[[萊絲莉·曼恩|萊絲莉·曼恩]]亦加盟這部電影演出艾瑪·華森所飾角色〔Nicki〕的母親。而於「'''{{link-en|好萊塢竊盜案|Bling Ring}}'''」中受害的名媛[[帕麗斯·希爾頓|帕麗斯·希爾頓]]亦於電影中客串演出。亦有傳媒於炫富幫片場拍到好萊塢巨星[[克斯汀·鄧斯特|克斯汀·鄧斯特]]參加演出。<ref>{{cite web |url=http://news.china.com.cn/rollnews/2012-03/28/content_13528743.htm |title='《炫富帮》再曝片场 克尔斯滕 罗斯戴尔现身'March 28, 2012}}</ref>

此前曾有消息傳出，{{link-en|好萊塢竊盜案|Bling Ring}}中成員之一的{{link-en|亞歷克·娜絲|Alexis Neiers}}的妹妹{{link-en|苔絲·泰勒|Tess Taylor}}有望出演該片。另外，劇組還向另外一位成員{{link-en|尼克·保高|Nick Prugo}}發出邀請，希望其可以擔任《炫富幫》的顧問，但遭到了拒絕。 <ref>{{cite web |url=http://gb.cri.cn/27564/2012/03/01/3465s3580896.htm |title='艾瑪·華森或将出演《炫富帮》 变“好莱坞财宝大盗”'March 1, 2012}}</ref>
