'''捷克斯洛伐克'''（[[捷克文|捷克文]]／[[斯洛伐克文|斯洛伐克文]]：Československo），當時台灣譯作'''捷克斯拉夫'''<ref>{{cite web |author1=[[中國文化大學|中國文化大學]]、[[國科會|國科會]] |title=捷克斯拉夫 |url=http://ap6.pccu.edu.tw/Encyclopedia_media/main-oh.asp?id=10160&lpage=1&cpage=1 |website=中華百科全書 |accessdate=2018-09-09}}</ref>，是一个存在於1918年至1992年的[[欧洲国家|欧洲国家]]，解体後演變為两个独立國家──[[捷克共和国|捷克共和国]]和[[斯洛伐克共和国|斯洛伐克共和国]]。

==简介 ==
成立於1918年，位于[[中歐|中歐]]，由[[捷克|捷克]]及[[斯洛伐克|斯洛伐克]]所組成，當時的國名是[[捷克斯洛伐克共和國|捷克斯洛伐克共和國]]，是世界第六大工業國（前五名為美、蘇、德、英、法）。在[[第二次世界大戰|第二次世界大戰]]期間，捷克斯洛伐克部分地區（主要是[[蘇台德區|蘇台德區]]）被[[納粹德國|納粹德國]]占領，並成立了[[波希米亞和摩拉維亞保護國|波希米亞和摩拉維亞保護國]]及[[斯洛伐克共和國_(1939-1945)|斯洛伐克共和國]]兩個[[傀儡國|傀儡國]]，其他地區被波蘭及匈牙利所瓜分。1960年，捷克斯洛伐克改名為[[捷克斯洛伐克社會主義共和國|捷克斯洛伐克社會主義共和國]]。在1989年發生的[[天鵝絨革命|天鵝絨革命]]後，又更名為[[捷克和斯洛伐克聯邦共和國|捷克和斯洛伐克聯邦共和國]]。1992年，捷克和斯洛伐克聯邦共和國解体，並於1993年1月1日起成為[[捷克共和国|捷克共和國]]及[[斯洛伐克共和国|斯洛伐克共和国]]兩個獨立的國家。

== 歷史 ==
捷克斯洛伐克地区位於[[欧洲|欧洲]]的中心，在其独立之前受[[哈布斯堡王朝|哈布斯堡王朝]]建立的[[奧地利帝國|奧地利帝國]]统治达300年之久，1867年奧地利帝國改組成[[奧匈帝國|奧匈帝國]]，捷克和斯洛伐克地区分別劃歸奧地利和匈牙利。1918年[[第一次世界大戰|第一次世界大戰]]結束後，[[奧匈帝國|奧匈帝國]]兵敗解體，[[波希米亞|波希米亞]]、[[摩拉維亞|摩拉維亞]]及[[斯洛伐克|斯洛伐克]]等地區组成一个獨立的國家，名為[[捷克斯洛伐克共和國|捷克斯洛伐克共和國]]。国家独立后，捷克斯洛伐克的经济发展迅速，到[[第二次世界大战|第二次世界大战]]前，曾在世界上十大工业化国家中排名第六（僅次於[[美国|美]]、[[苏联|蘇]]、[[德国|德]]、[[英国|英]]、[[法国|法]]，高於[[意大利|-{zh-hans:意; zh-hant:意; zh-tw:義;}-]]、[[日本|日]]、[[波蘭|波]]、[[土耳其|土]]、[[西班牙|西]]）。

1938年，[[英国|英]][[法国|法]]等国与[[德国|德国]]和[[意大利|意大利]]签定[[慕尼黑协定|慕尼黑协定]]，在没有捷克斯洛伐克代表参加的情况下决定将其西部领土[[苏台德区|苏台德区]]割让给德国。1939年3月，納粹德國以接收蘇台德區為由，出兵將整個捷克斯洛伐克吞併，德国在捷克成立[[波希米亚和摩拉维亚保护国|波希米亚和摩拉维亚保护国]]，斯洛伐克则成立受德国保护的[[斯洛伐克共和國_(1939-1945)|斯洛伐克共和國]]，斯洛伐克的部分地区割让给德国的盟友[[匈牙利|匈牙利]]。

1945年，[[苏联|苏联]]向東歐反攻，依[[波茨坦協定|波茨坦協定]]捷克斯洛伐克除了布拉格以西被美軍佔領外，包含首都[[布拉格|布拉格]]皆被[[苏联红军|苏联红军]][[布拉格攻勢|攻佔]]，捷克和斯洛伐克再度合併，割让德国與匈牙利的领土重新划归捷克斯洛伐克，但[[外喀爾巴阡州|外喀爾巴阡州]]则割让给苏联的[[乌克兰苏维埃社会主义共和国|乌克兰苏维埃社会主义共和国]]。因為一部分由美國占領之緣故，捷克斯洛伐克一開始恢復成民主國家，但在蘇聯暗中支援下1948年發生[[二月事件|二月事件]]，成為傾向苏联之[[社會主義|社會主義]]國家。并加入苏联提倡建立的[[华沙条约组织|华沙条约组织]]和[[经济互助委员会|经济互助委员会]]。1960年，捷克斯洛伐克共和国改名为[[捷克斯洛伐克社会主义共和国|捷克斯洛伐克社会主义共和国]]。

1968年，当时[[捷克斯洛伐克共产党|捷克斯洛伐克共产党]]第一书记[[亞歷山大·杜布切克|杜布切克]]发动了“[[布拉格之春|布拉格之春]]”的经济和政治改革运动，为苏联所不容，苏联入侵捷克斯洛伐克平亂，布拉格之春改革运动戛然而止，捷共第一书记杜布切克被挟持至莫斯科，苏联的行为受到世界各国的广泛批评。

1989年11月，捷克斯洛伐克成功發起[[天鵝絨革命|天鵝絨革命]]後，結束了共產黨的[[一党专政|一党专政]]，1990年，国名改为[[捷克和斯洛伐克联邦共和国|捷克和斯洛伐克联邦共和国]]。不過，由於民主化革命後的經濟改革中，捷、斯兩方的經濟差距日益拉大，原本隐藏的民族矛盾也因為民主化而开始浮出水面。1992年，捷克和斯洛伐克分道扬镳，经过议会表决，於1993年1月1日宣布正式解体为[[捷克|捷克]]和[[斯洛伐克|斯洛伐克]]两个国家，史稱[[天鵝絨分離|天鵝絨分離]]。

== 共產黨領導人 ==
從1921年至1945年[[捷克斯洛伐克共產黨|捷克斯洛伐克共產黨]]最高領導人的稱號為[[總書記|總書記]]（Generální Tajemník），從1945年至1953年稱主席（Předseda），從1953年至1971年稱[[第一書記|第一書記]]（První Tajemník），從1971年至1989年又稱總書記。
