'''捷勝镇'''（漢語拼音：'''Jiesheng'''）是[[中華人民共和國|中華人民共和國]][[廣東省|廣東省]][[汕尾市|汕尾市]][[城区_(汕尾市)|城區]]南端的一個[[建制鎮|建制鎮]]，古稱'''捷琅埔'''，因城所狀如四方，且周高中低，極似[[漁罾|漁罾]]，故別稱'''罾城'''。

==位置==
捷勝镇位於汕尾市城區東南部沿海，離[[汕尾市|汕尾市]]中心20公里。東與[[紅海灣經濟開發試驗區|紅海灣經濟開發試驗區]]毗鄰，西與汕尾市區相連，北與城區[[東涌鎮|東涌鎮]]接壤，南臨[[南海|南海]]。全鎮總面積有52平方公里，鎮區面積8.9平方公里，建成面積1.79平方公里，[[海岸線|海岸線]]長12公里。

捷勝枕山面海，山水奇秀；氣候溫暖，雨量充足；冬暖夏涼，四季常青。地勢西北高、東南低，屬丘陵地帶；三面環山，一面向海。山多巨石重疊，海爲優良漁場。西北部有蜿蜒盤錯的白雲山、雞啼嶺、霞仔山等山脈。南部距陸地五公里的海域，有龜齡島、菜嶼、撈投嶼、青嶼、鷹嶼、牛皮洲、三角洲、赤臘等島嶼，明石暗礁盤踞，是鮑魚、海膽、龍蝦、膏蟹、角螺、蚌蚝、石斑及紫菜、果菜等海珍生物的盛産之地。外面那浩瀚無際的太平洋海域，是廈門、汕頭至香港、廣州及東南亞航線的必經之地。沿海沙灘自東而西又有石狗湖、雙石湖、沙坑湖及東坑湖相距並列，是出水與容水所在，也是天然養殖的場所。陸地中部，九伯嶺（原稱“九敗嶺”）山脈分隔爲東西兩處農區，旱地具多，水田多分布于東、西窪地，全鎮共有耕地11858畝，其中旱田6975畝，水田4883畝，耕地多爲砂質土和壤質土，只有少量的淤泥土壤。適耕性較強，屬四季常綠産區，盛産著稻谷、蕃薯、花生、西瓜、冬瓜、荷蘭豆、薯菇、荔枝、香蕉、楊桃、番茄等糧食及經濟作物。全區有九伯嶺、東坑、前進三宗蓄水量三百萬立方米以上的水庫和錫坑等三宗小型水庫。總蓄水量達800多萬立方米；沿海修築了防潮堤，營造了防風林帶，出海口還建有水閘。鎮（區）辦企業主要有建築、漁農具、造船、制鹽、塑料、貝雕等工廠。原貝雕工藝廠的産品，曾連續十多年參加廣交會的展銷，埔尾建築隊更爲出名。區內還設有衛生院、影劇院、電影院等。

==人口==
2005年人口統計顯示，全鎮人口約53220人，合計10056戶，從業人員20484人，農業人口10219。主要民族為[[漢族|漢族]]，通用[[海丰话|海丰话]]方言。{鶴佬話}

==歷史==
原址為古戰場[[捷琅埔|捷琅埔]]．[[明|明]][[洪武|洪武]]二十八年（西元1395），千戶[[候良|候良]]率吏民英勇抗擊並大敗[[倭寇|倭寇]]，故改稱捷勝。[[明|明]]、[[清代|清代]]設[[所署|所署]]、鎮[[撫司|撫司]]、[[吏目廳|吏目廳]]等官方機構，軍事上屬媽石衛，行政上屬[[海豐縣|海豐縣]]多錫都。[[民國|民國]]時期屆[[海豐縣|海豐縣]]第七區署。1958年成立捷勝[[人民公社|人民公社]]。1987年改設[[鎮|鎮]]，1988年[[汕尾市|汕尾市]]成立後歸市城區管轄。

早在六千年前的新石器時代中晚期，便有先民在捷勝的沙坑、東坑、牛肚和沙角尾一帶漁耕狩獵、聚居生息。他們胼手胝足，開辟草萊，創造著捷勝的遠古文化。歷夏、商、周三代至春秋戰國時期爲南蠻之地，是百越族的一部分。

公元前221年，秦始皇統一嶺南，置南海、桂林、象郡三郡，海豐始被納入中國版圖，捷勝亦隸屬南海郡博羅縣地。

西漢元鼎六年（公元前111年），漢武帝滅南越，開鑿楊桃嶺（今稱羊蹄嶺）官道，其後分諸郡置海豐縣（一說置縣始于東晉鹹和六年，備考），仍隸南海郡（杜氏《通典》、宋樂史《太平廣記》、《海豐縣志》），捷勝乃屬南海郡海豐縣統轄。經魏晉、隋唐，捷勝仍屬海豐縣地。

北宋熙甯二年（公元1069年，一說“熙甯元年”），王安石推行“募兵制”新法：“民十家爲一保，五十家爲一大保，十大家爲都保。”因而海豐全縣改“里”爲“都保”，分設興賢、石塘、金錫、楊安、坊廓、石帆、吉康、龍溪八都，捷勝則隸屬海豐縣金錫都。

迨至明洪武二十八年（1395年）八月十五日，捷勝始置千戶城所，隸屬惠州府海豐縣碣石衛金錫都。清朝沿用明朝舊制，捷勝仍隸惠州府海豐縣碣石衛金錫都。其時，捷勝所統轄區域較廣，除了現在的格塘、馬鞍山、軍船頭、茔頭、大塘、沙角尾、牛肚、沙坑、東坑、雙湖、石厝、南門崗、大淋、石頭、南門外、城內、新建鄉、大水溝、烏土、李厝鄉、程厝鄉、貓空、屈籠、內坑、格坑、大布、埔尾、芴仔、公頂坑、龍頭鄉、大流、水閣、聯星、金德鄉等34個自然村之外，還曾轄有花樹、大華、苦厝鄉、後沃、沙海、鲎山羅、鲎山魏、卡吳、船尾地、馬巷、牌仔、龍溪、石龜、郭厝寮、豬母寮、黃竹坑、空內、遼口、青龍頭、町前等鄉村。隨著歷史政壇的變化，捷勝爲區爲鎮，或合或分，旋而複始。

辛亥革命（1911年）以後，廢除清朝舊制，建立民國。民國二年（1913年），劃捷勝、寮口、東洲坑、田墘、湖內及遮浪六鄉鎮設立捷勝區公所，駐地捷勝城內，爲海豐縣第八區。民國十六年（1927年）11月大革命取得勝利，在捷勝衙門和北門街紅樓分別建立中共海豐縣第七區委員會和海豐縣第七區蘇維埃政府。至民國十七年（1928年）3月大革命失敗後，仍爲民國海豐縣第七區。捷勝原已分設的東、西、南、北四鎮，後並設爲東南和西北兩鎮。民國二十五年（1936），第七區公所改爲第四區署。民國三十年（1941年）農曆二月廿七日，日本侵略汕尾，捷勝處于淪陷狀態。民國三十一年（1942年）六月，捷勝並入汕尾第四區署（署址設汕尾）；同年農曆十月初九日夜，日軍撤離汕尾，恢複第四區署（署址設捷勝）。民國三十三年（1944年），東南鎮和西北鎮合並爲捷勝鎮。民國三十四年（1945年）1月，日本侵略者第二次登陸汕尾和海城，捷勝設立僞村公所。同年八月十五日，日本投降，恢複鎮公所建置，隸屬汕尾第四區署。

民國三十八年（1949年）7月1日，捷勝被中共佔領，建立鎮人民政府，劃歸海豐第七區所轄（所址設田墘）。1952年9月，捷勝分設第八區公所，駐地捷勝，下轄捷勝鎮（指城內）、新聯、新安、西郊、石崗、沙塘、大流、東沙、洪流、民安、安華等11個鄉鎮。1953年，第八區改名爲捷勝區，下轄捷勝、沙塘、大流、石崗、西郊、安華等6個鄉鎮，至1956年下轄鄉鎮除安華劃歸流安鄉外，其余五個鄉鎮合並爲捷勝和大流兩大鄉。1957年合並爲捷勝鄉。1958年9月，撤區組建人民公社，設管理委員會，下轄東村、南村、西村、北村、居民、聯和、聯明、聯星、大流、埔尾、前進、東坑、沙坑、牛肚、沙角尾、五愛、石崗、漁民等18個大隊，共224個生産隊。1968年，公社、大隊管委會改名爲革命委員會。1984年3月又改建爲區公所，下轄大流、埔尾、前進、東坑、沙坑、沙角尾、五愛、石崗等8個鄉政府及捷勝人民政府。1987年4月撤區公所建立鎮建制，捷勝仍屬海豐縣地。1988年3月7日，汕尾設立地級市，劃沿海的汕尾、紅草、馬宮、東湧、田墘、捷勝、遮浪七鎮爲城區，捷勝從海豐縣析出，改屬汕尾市城區統轄。自古以來，捷勝均屬政權要地，是全鎮政治、經濟、文化發展的中心。

從以上歷史變遷情況看，捷勝這個古鎮的滄桑史，基本上可以劃分爲“辟荒聚落”、“建置發展”和“農運解放”三個時期。當然，這不是唯一的劃分方法，比如用時代、朝代或年號順序來劃分亦無不可。

==大事记==
新石器時代中晚期（前6000年——3000多年），古越族人在捷勝沿海的沙坑、沙角尾、牛肚及東坑村一帶漁耕狩獵，開辟草萊，繁衍生息。

===宋===
宋熙甯年間（1068—1077年），海豐設八都：興賢、石塘、金錫、楊安、石帆、坊廓、吉康、龍溪。捷勝隸金錫都。

宋景炎二年（1277年）八月，文天祥在江西空坑（粵贛交界處，今江西省吉水縣附近）抗元失敗，遂與杜浒、鄒風至循州，集散兵屯于南嶺，複惠州。奉母同行，收兵海豐，屯麗江浦。後士族範良臣與鄭複翁來附，遂略定甲子、碣石、捷勝、船澳等沿海港嶼。
===明===
明洪武十三年（1380年），福建船隊在捷勝鶴嶺建造凝波寺。
同年，建城隍廟及戲台。

明洪武二十七年（1394年）海豐沿海倭患頻仍，都指揮使花茂奏立衛所。
是年，千戶侯良在鶴嶺建文武官署（一說“捷勝所署”在南門）。

洪武二十八年（1395年），千戶侯良建造捷勝所城。茅埔墟鎮和大石牯村居民移居城內。

明弘治十七年（1504年），金錫都（今田墘、捷勝、遮浪、東湧、汕尾、紅草、陶河、赤坑、大湖一帶）大海潮。平地水深一二丈，死者無數。

明嘉靖十七年（1538年），大海潮，金錫都死幾百人，多絕戶。捷勝沿海漁民死傷慘重。

明萬曆元年（1573年），抗倭名將劉松遊黎明洞題刻“山海奇觀”詩聯。

明天啓乙丑年（1626年）冬，廖天佐作《黎明洞石刻詩》。

明崇祯三年（1630年），裁革捷勝倉大使職額。

明崇祯九年（1636年）七月，台風，牛吹入海，船吹上岸，捷勝居民損失慘重。
===清===
清順治四年（1647年）春，蘇成占據捷勝所城。夏，與舉人林呈祥率船隊攻海豐縣城，揚言反清複明。

清順治七年（1650年），捷勝所城守備汪古，害怕蘇成、蘇利抗清義軍攻城，以漁船載巨石沈塞石狗湖口。

清順治十八年（1661年），朝廷頒下“遷海令”，敕令江、浙、閩、粵、魯五省沿海居民內遷30至50-{里}-。

清康熙元年（1662年）正月初五日，吏部侍郎科爾坤、兵部侍郎介山同平藩等率兵萬余人抵海豐，迫遷猶甚。
: 二年（1663年）八月，吏部侍郎伊里布、兵部侍郎碩圖來令再遷時，以河內鹿境山爲界離海六十里、離城十里，迫令農、漁、鹽等民，不許在界外居住，不許出海捕魚，違者逮捕。
: 三年（1664年）四月遷界，盡燒民屋船只，凡越界者無論遠近均立斬。捷勝所城居民遷走一空。

清康熙七年（1668年），清廷開始放寬海禁，下令展界，界外百姓陸續回城，但還不准百姓出海捕撈。

清康熙八年（1669年）春，捷勝所城設遊擊一員。
是年正月十三日，廣東巡撫王來任上疏奏請複界；冬，兩廣總督周有德上《奏複沿海安插地方發兵防守疏》。同年複界。

清康熙十七年（1678年）至十九年（1680年）期間，廖天佐在黎明洞作《遷移圖說》。

清康熙二十二年（1683年），江、浙、閩、粵、魯五省始全面展土複界，准許百姓出海捕撈。捷勝所城人口陸續增加。

清康熙五十六年（1717年），添設捷徑（捷勝）、遮浪營房二十一間，撥右營把總一名，兵六十三名防守。

清雍正十年（1732年），捷勝建內文閣和外文閣。

清道光二十六年（1846年），汕美炮台移捷勝角。

清道光三十年（1850年），洋盜充斥，捷勝所城目兵張時高隨標出洋捕賊陣亡。

清-{咸}-豐三年（1853年）秋，有活東千百于楓林塘爭鬥數日，第二年又鬥于舊圩及捷勝所城，且有抓落葉以捍身如作彭排之狀者。

清光緒三十二年（1906年），住持僧遇緣禅師擴建白雲寺，並易寺名爲：“得道庵”。適逢碣石鎮中衡遊府正任水師提標、前營都阃府吳祥光（捷勝鎮群衆稱吳大佬）巡察捷勝城，遇緣禅師請其題寫大門匾額“得道庵”三字。

清宣統三年（1911年），辛亥革命爆發，捷勝所城數十名老將和文武官員棄城逃走，城所廢置。
===民國===
民國二年（1913年），劃捷勝、寮口、東洲坑、田墘、湖內及遮浪六鄉鎮，設立捷勝區公所，駐地捷勝後山頭，隸屬海豐縣第八區。

民國三年（1914年），西秦“華天樂” 戲班招收“合同腳”30多人，在捷勝城外油車間培訓。

民國八年（1919年）六月，在海豐縣學生聯合總會的領導下，捷勝的中小學生組織檢查隊，四處查封日貨。

民國十二年（1923年）正月，大旱，至四月初五日才下大雨。六月至七月連續七次台風，海豐千余艘魚船沈沒，數千人淹死。捷勝亦損失頗重。

民國十三年（1924年）二月，彭湃、李勞工在捷勝約參加農會召開的恢複大會，與會代表及農民共有二千多人，彭湃在會上發表演說，會後還舉行了示威遊行。

民國十四年（1925年）六月十三日，捷勝舉行援助上海“五卅”慘案大會，會場有五千多人，由大會主席報告“五卅”慘案的情況，各代表亦相繼發言。會後舉行示威遊行，散發傳單，高呼“打倒帝國主義”等口號，各學校還組織宣傳隊向群衆演講。
: 同年夏，中共黨員楊江、葉娘庇、林躍俞、陳慶廣、蔡舉德、莫捷光、李生等，接受中共海豐特委指示，在東坑成立第七區支部。

民國十六年（1927年）十一月十九日（農曆十月廿六）上午八時許，紅二師四團一個連和工農革命軍聯合攻城，城所告破。
: 是年，在捷勝衙門和北門街紅樓，分別建立中共海豐縣第七區委員會和海豐縣第七區蘇維埃政府。

民國十七年（1928年）三月，捷勝分爲東南和西北兩鎮。

民國十九年（1930年），軍委軍令部陸測總局勘測捷勝板塊，繪制地圖。

民國二十五年（1936年），第七區公所改爲第四區署。
: 是年，意大利神甫麥兆漢在捷勝牛肚、沙坑、東坑、沙角尾等文化遺址考古。

民國二十七年（1938年）三月，海豐縣第四區青抗會（青年抗敵同志會）在捷勝成立，幹事會成員有何世漢、蔡烈、何宗漢、李民、何志雄、賴志、劉漢章、梁良梯、何鼎元、許昌熾等。

民國二十九年（1940年）一月一日，海豐縣長姚芝榮巡察捷勝，于得道庵山門外西側岩壁題刻“同歸佛海”四字。

民國三十年（1941年），以陳鐵、淩炳權（自稱先遣軍司令部）爲首的龜齡島海匪，在捷勝海域大肆搶劫來往船只（叫“擒羊牯”）。

民國三十一年（1942年）六月，捷勝並入汕尾第四區署，署址設汕尾；十一月，恢複第四區署，署址設捷勝。
: 是年，著名民俗學家楊成志在捷勝考古。

民國三十二年（1943年）春，大旱，至五月十六日（農曆四月十八日）才降雨，旱期約三個半月，河流枯竭，田地龜裂，作物失收，造成嚴重大饑荒。時海豐餓死者46309人，有2998戶絕戶。捷勝漁民餓死者亦數以千計。入夏，霍亂又流行。

民國三十三年（1944年），東南鎮和西北鎮合並爲捷勝鎮。

民國三十四年（1945年）一月，日本侵略者第二次登陸汕尾、海城，捷勝設立僞村公所。五月，東縱六支隊挺進海陸豐，全面出擊日本侵略者，建立敵後根據地及民主政權，在捷勝成立抗日民主政府。八月十五日，日本投降，恢複鎮公所建制，隸屬汕尾第四區署。

民國三十五年（1946年），民族學家顧鐵符到捷勝考古。

民國三十六年（1947年）七月十九日，美國華僑輪船公司“豐慶輪”由南洋經香港往汕頭航行，至汕尾港時突遭風暴襲擊，當晚在龜齡島海域附近的菜嶼觸礁船毀，500多名乘客及員工獲救，後由港英兵艦駁運香港。

民國三十七年（1948年）二月，建立田墘捷勝聯防隊。八月二十日，汕尾地區群衆因不滿當局通貨惡性膨脹，掀起搶米鬥爭，捷勝人民亦卷入鬥爭之中，當局驚惶失措。加上龜齡島海匪擾亂，社會秩序極其紛亂。
: 是年秋，海導基組在西門第六街何植東家秘密成立。

民國三十八年（1949年）二月，設立捷勝情報站。七月一日，捷勝解放，建立捷勝鎮人民政府，劃歸海豐第七區所轄；並于該日成立第一個中國新民主主義青年團支部（後改稱“共青團”）。八月二十二日上午，龜齡島海匪吳炯烙部入侵捷勝，搜捕革命幹部、勒索迫害革命家屬。十一月二十八日，吳炯烙部在龜齡島被紅軍圍殲，吳炯烙被捕。十二月下旬，捷勝鎮成立青年聯合會。

1950年8月，捷勝成立漁協分會，屬海豐縣漁民協會管轄。

1951年，汕尾廣東省立高級水産技術學校在捷勝龜齡島采集自然海區的紫菜苗，移到汕尾港栽培，開始了華南紫菜人工養殖。是年12月15日，捷勝成立漁民供銷合作社。

1952年9月，捷勝設立第八區公所，駐地捷勝。

1953年，第八區改名捷勝區，下轄捷勝、沙塘、大流、石崗、西郊、安華六個鄉鎮。

1958年9月，捷勝撤區組建人民公社，設管理委員會。

1958年1月，捷勝九伯嶺水庫動土興工，萬民歡騰。

1959年，禁止對魚類資源破壞嚴重的20多艚敲罟作業，並禁止使用魚炮炸魚。

1971年8月16日下午3時30分，汕尾鎮29艘風帆蝦船和遮浪公社駁運站1艘機動漁船，共355人，在龜齡島東北附近遭突發性大海潮（俗稱“杜猴塞洞”）襲擊，9艘船被擊碎，18艘嚴重損壞，320人遇難，其中被救回295人，25人死亡。

1979年，沿海漁區出現偷渡外流、走私風，海豐全縣偷渡漁船有414艘，外流香港9295人，其中九個沿海公社（鎮）5669人。捷勝偷渡、走私風氣亦盛極一時。

1980年，捷勝建立台風警報站。

1983年3月22日淩晨，汕尾鎮新豐大隊14042號機船在捷勝龜齡島發現屍體六首。經查實是：遮浪公社紅坎大隊四石柱村24056號機船于3月21日夜間，因霧在龜齡島觸礁沈沒，船上17人全部死亡。

1983年——1984年，海豐縣文物普查隊在捷勝采集文物資料，進行考古研究。

1984年，捷勝鎮人民政府辦公大樓建成。
: 是年9月，捷勝鎮重興土木，耗資40萬元重修得道庵風景風。

1985年，海豐縣財政局撥給貸款4萬元，幫助捷勝鎮發展鮑魚生産，購買福建東山的人工培育鮑苗5.7萬個，投放于龜齡島區，進行養殖增殖。

1986年10月，捷勝鎮成立財政所，業務隸屬海豐縣財政局鄉鎮財政管理股管理。

1987年農曆正月廿日，捷勝舉行遊神賽會，舉城同慶。10月，得道庵風景區修建工程完成。

1988年3月7日，全省實行市管縣體制，惠陽地區撤銷，海豐縣劃歸新成立的汕尾市管轄，捷勝析出海豐縣歸屬汕尾市城區。

1994年1月，捷勝成立詩詞社。

1995年10月，得道庵理事會聃請廣州光孝寺主持方丈釋新成大師爲刻庵主持。

1996年12月，廣東中華詩詞學會領導楊奎章、張作斌、李汝倫、許可青、楊應彬、熊鑒、陳哲、老烈、張春芳、朱帆等莅捷考察，對捷勝古城深厚的人文歷史亦予以肯定和贊賞，並各自題辭紀勝。

1998年，廣東省省長盧瑞華、副省長歐廣元視察捷勝。
: 是年5月，捷勝成立書畫社。

1999年2月10日，捷勝首屆書畫作品展開幕。

2003年初，捷勝成立民樂社。6月27日，汕尾市民間文藝家協會組織30多位文藝家到捷勝采風，對捷勝鎮的人文歷史及名勝古迹作了大力的推介。7月，廣東省文物研究所副所長邱立誠莅捷考察。12月31日，捷勝詩詞社、書畫社聯合舉辦“古城風”詩書畫作品巡回展。
: 是年，捷勝被上級文物部門定爲“文化藝術之鄉”；並經農業部批准，捷勝石狗湖成爲汕尾市新漁港。

2004年1月1日，軍船頭村從城區東湧鎮析出，改隸捷勝鎮管轄。

==行政區劃==
捷勝镇分為14個[[行政村|行政村]]、4個[[社區|社區]]，共35個[[自然村|自然村]]。

==自然與地理==
捷勝地處[[廣東省|廣東省]][[汕尾市|汕尾市]]城區東南部，即[[東經|東經]]115º22′30″——115º26′45″、[[北緯|北緯]]22º45′00″——22º42′30″，東、北、南與汕尾、[[東涌|東涌]]、[[紅海灣|紅海灣]]（田乾、遮浪）接壤，南臨[[太平洋|太平洋]]海域。

==氣候==
捷勝屬[[亞熱帶|亞熱帶]]海洋性區域，年平均氣溫21.9℃，年降雨量1700毫米。春夏季多為西南、東南風，冬季為北至東北風，夏秋季受[[臺風|臺風]]影響較明顯。
