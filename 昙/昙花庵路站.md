'''昙花庵路站'''是建设中的[[杭州地铁6号线|杭州地铁6号线]]的一个车站，位于杭州市[[江干区|江干区]]云峰街道昙花庵路筑塘路，沿昙花庵路东西向布置。<ref>{{cite web |title=地铁6号线二期站点位置图出炉 周边最近的楼盘都在这了！ - 导购 -杭州乐居网 |url=http://hz.leju.com/news/2017-12-29/11146352345724369559842.shtml |website=新浪乐居 |language=zh}}</ref>

==结构==
昙花庵路站为地下三层双柱三跨钢筋混凝土箱形结构，车站总长279.6米。标准段基坑宽22.1米，深21.61米；端头井基坑宽26米，南端头井深23.21米，北端头井深23.85米。<ref>{{cite web |title=杭州地铁6号线昙花庵路车站地连墙全线最快完工 |url=http://www.crfeb.com.cn/cg/1510/1893385/index.html |website=中铁一局}}</ref>

==历史==
*2018年9月13日，车站开工<ref>{{cite web |title=关于地铁6号线二期昙花庵路站主体工程  施工期间调整交通组织措施的通告 |url=http://www.hzpolice.gov.cn/Html/201809/06/d6d7ab82-0af8-4314-a5c4-a62d528dba41.html |website=杭州市公安局}}</ref>
