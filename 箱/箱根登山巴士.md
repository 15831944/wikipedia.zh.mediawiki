'''箱根登山巴士株式会社'''（{{lang-ja|箱根登山バス株式会社}}，{{lang-en|Hakone Tozan Bus Co., Ltd.}}）是[[小田急電鐵|小田急集團]]旗下的[[小田急箱根控股|小田急箱根控股]]所屬的公路運輸業者，主要經營[[神奈川縣|神奈川縣]][[小田原市|小田原]]及[[箱根町|箱根]]週邊地區交通運輸，總部設於神奈川県小田原市。

箱根登山巴士的前身為1913年3月1日開業的'''小田原電氣鐵道'''包車業務（{{lang|ja|貸自動車}}）<ref name="kato">{{cite book|script-title=ja:箱根山の近代交通|publisher=神奈川新聞社|author=加藤利之
|language=ja|year=1995|isbn=978-4-87645-189-0}}</ref>，以及1914年8月15日創業的'''富士屋自働車'''的包車業務<ref name="toyota-museum">{{cite web|url=https://www.toyota.co.jp/Museum/kandayori/backnumber/magazine93/magazine93_4.pdf|script-title=ja:富士屋ホテルと自動車 その３：富士屋自働車の営業展開|publisher=豐田汽車博物館|language=ja|accessdate=2018-11-30}}</ref>，兩者於1932年合併為'''富士箱根自動車'''<ref name="toyota-museum"/>，二戰時隨著1921年設立的'''足柄自動車'''與[[箱根登山鐵道|箱根登山鐵道]]整合，成為同社的汽車部門<ref name="toyota-museum"/>。2002年10月，小田急集團進行業務重組，箱根登山鐵道的汽車業務分拆成立[[箱根登山巴士|箱根登山巴士]]<ref name="tozan-history">{{Cite web|url=http://www.hakone-tozan.co.jp/company/history.html|language=ja|script-title=ja:箱根登山鉄道の歴史　年表|publisher=箱根登山鐵道|accessdate=2018-11-08|date=2007-01-06}}</ref>。

== 事業内容 ==
; 路線巴士
: 路線巴士主要以[[小田原市|小田原市]]和[[箱根町|箱根町]]為中心<ref name="br110"/><ref>{{cite web|url=https://www.hakone-tozanbus.co.jp/info/bus/rosen/|script-title=ja:路線図|publisher=箱根登山巴士|language=ja|accessdate=2018-11-30}}</ref>，營業地區遍及足柄地域（[[南足柄市|南足柄市]]、[[開成町|開成町]]、[[松田町|松田町]]）與[[真鶴町|真鶴町]]和[[湯河原町|湯河原町]]<ref name="br110"/>，部份路線會駛入[[静岡縣|静岡縣]][[御殿場市|御殿場市]]。
; 包租巴士
: 箱根登山巴士保有小田原養護学校5輛[[校車|校車]]<ref name="br110">{{cite book|script-title=ja:バスラマ・インターナショナル|volume=第110号|script-chapter=ja:「バス事業者訪問123 箱根登山バス」|language=ja|publisher=ぽると出版|date=2008-11|page=p38-52|isbn=978-4-89980-110-8}} </ref>，1輛企業接送用車<ref name="br110"/>，和一般大型包租巴士1輛<ref name="br110"/> 。
; 特定巴士
: 除一般的巴士業務外，箱根登山巴士也受託負責營運箱根町立箱根之森小学3輛校車<ref name="br110"/><ref>{{cite web|url=http://hmori-es.hakone-edu.jp/modules/hp_jpage17/blog_detail.php?page_parent=730|script-title=ja:学校日記：レタンPG60|page=p4|publisher=箱根町立箱根之森小学|language=ja|date=2018-07-30|accessdate=2018-11-30}}</ref>，以及[[富士菲林|富士菲林]]的接駁巴士<ref name="br110"/>。
; 箱根行李托運服務（箱根キャリーサービス）
: 提供往返箱根湯本站及合作住宿業者的行李運送服務<ref>{{cite web|url=https://www.hakone-tozanbus.co.jp/carry/file/carry.pdf|script-title=ja:箱根の旅は荷物を預けてスマート散策|publisher=箱根登山巴士|accessdate=2018-11-30}}{{ja icon}}{{zh icon}}{{en icon}}{{ko icon}}</ref>，於2005年3月開始營業<ref>{{cite web|url=http://www.mlit.go.jp/common/001140910.pdf|script-title=ja:小田急グループにおける『手ぶら観光カウンター』設置の取り組みについて|page=p4|publisher=小田急電鐵|language=ja|date=2016-07-08|accessdate=2018-11-30}}</ref>。
