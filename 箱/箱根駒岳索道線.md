'''箱根駒岳索道'''（{{jpn|j=箱根 駒ヶ岳ロープウェイ|hg=はこね こまがたけろうぷうぇい|rm=hakone komgatake rōpuwei}}，{{lang-en|Hakone Komgatake Ropeway}}）是[[日本|日本]][[神奈川縣|神奈川縣]][[足柄下郡|足柄下郡]][[箱根町|箱根町]]境內一條連接蘆之湖與駒之岳的登山觀光纜車索道，由西武集團旗下王子飯店營運。駒之岳山頂可飽覽富士山、蘆之湖以及駿河湾一帶的景色，因此吸引許多遊客利用前往參觀。另同集團的{{link-ja|箱根駒岳纜車|駒ヶ岳ケーブルカー}}在2005年起癈線，故此目前是前往駒之岳山頂的唯一交通手段。

== 路線及車站 ==
索道全長1,783米，全線位於[[神奈川縣|神奈川縣]][[箱根町|箱根町]]境內，由箱根園站出發往駒岳山頂站全程需時約7分。<ref name="mania">{{cite web|language=ja|script-title=ja:知れば旅がもっと楽しくなるマニアックな情報|url=http://m.princehotels.co.jp/amuse/hakone-en/ropeway/fun/maniac.html|publisher=箱根駒岳索道|accessdate=2018-11-05}}</ref>

索道從蘆之湖畔的箱根園出發，連接箱根火山中央口的駒之岳（海拔1,327米）。山頂還設有祭祀靈峰神山的[[箱根神社|箱根神社元宮]]，於1964年由西武集團的創辦人堤康次郎捐獻重建。

== 歷史 ==
*[[1963年|1963年]]4月27日，箱根駒岳索道開業。<ref name="mania"/>
*[[1986年|1986年]]10月，車廂更新。<ref name="mania"/>
*[[1990年|1990年]]7月，自動列車運行系統啓用<ref name="mania"/>
*[[2016年|2016年]]2月1日，西武集團事業群重組，駒岳索道業務從伊豆箱根鐵道移交王子飯店管理。<ref name="izuhakone-his">{{cite web|language=ja|script-title=ja:伊豆箱根鉄道　会社の沿革：新しい西武グループの一員として（2005年以降～）報|url=http://www.izuhakone.co.jp/corporate/history/zenshi_2005-/index.html/|publisher=伊豆箱根鐵道|accessdate=2018-11-05}}</ref>

== 周遊券 ==
;箱根旅助通票
索道屬於西武系設施，可透過伊豆箱根鐵道發行的「箱根旅助通票」（{{lang|ja|箱根旅助け}}）套票任意搭乘，購買同集團「箱根巴士周遊券」（{{lang|ja|箱根バスフリー}}）的旅客也可憑票券以優惠價購票。<ref name="tabidasuke">{{cite web|language=zh-hant|title=箱根旅助通票|url=http://www.izuhakone.co.jp/bus/files/bus_coupon_chinese_han_20180123.pdf|publisher=伊豆箱根鐵道|date=2018-01-23|accessdate=2018-11-05}}</ref>

;箱根周遊券
「箱根周遊券」（{{lang|ja|箱根フリーパス}}）為小田急集團發行的套票，可以自由利用小田急系設施{{efn|包括[[箱根登山鐵道|箱根登山鐵道]]、[[箱根空中纜車|箱根空中纜車]]、[[箱根海賊觀光船|箱根海賊觀光船]]、箱根登山巴士}}，於西武鐵道各站{{efn|池袋站、西武新宿站、小竹向原站、新櫻台站、多摩川線各站除外。}}亦有販售。雖該票券無法任意搭乘西武集團的駒岳索道，但可憑票券以優惠價乘車。<ref name="navi">{{cite web|language=zh-hant|title=箱根周遊券：優惠折扣|url=https://www.hakonenavi.jp/chinese/freepass/|publisher=箱根導航|accessdate=2018-11-05}}</ref>
