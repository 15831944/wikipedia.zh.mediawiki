'''箱崎九大前站'''（{{jpn|j=箱崎九大前駅|hg=はこざききゅうだいまええき|rm=Hakozaki-Kyūdai-mae eki|f=y}}）是一個位於[[日本|日本]][[福岡縣|福岡縣]][[福岡市|福岡市]][[東區_(福岡市)|東區]]箱崎三丁目，屬於[[福岡市地下鐵|福岡市地下鐵]][[箱崎線|箱崎線]]的[[鐵路車站|鐵路車站]]。此站位於舊[[福岡市內線|福岡市內線]]（正式屬[[貝塚線|宮地岳線]]）的箱崎松原站正下方。[[車站編號|車站編號]]是'''H06'''。

== 歷史 ==
*1983年（[[昭和|昭和]]58年）9月12日 - 確定站名。
*1986年（昭和61年）
**1月31日 - 伴隨馬出九大醫院前－此站間延伸開業<ref name="RJ241"/>。
**11月12日 - 伴隨延伸至貝塚站而成為中途站<ref name="RJ241"/>。

== 車站結構 ==
地下2層[[島式月台|島式月台]]1面2線的[[地底車站|地底車站]]。
