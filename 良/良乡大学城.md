
'''良乡大学城'''位于[[北京市|北京市]][[房山区|房山区]][[拱辰街道_(北京市)|拱辰街道]]，是北京市的两大[[大学城|高教园区]]之一（另外一个是[[沙河高教园区|沙河高教园区]]）。园区规划总占地10111亩，总建筑面积496.94万平方米，预计总投资300亿元，入驻规模达到10-12万人，按照功能定位，划分为学校的教学及生活区、西部配套区、中央设施区、中央绿化景观区。<ref>{{cite web |title=园区简介_房山高教园区公共信息网 |url=http://www.univercity.gov.cn/jgxx/yqjj/index.htm |accessdate=2018-06-10}}</ref>

== 入驻高校 ==
=== 北京工商大学 ===
[[北京工商大学|北京工商大学]]良乡校区位于良乡大学南部，占地面积61万平方米，建设面积25.8万平方米。良乡校区一期于2004年10月投入使用，二期工程于2017年12月开工。<ref>{{cite web |title=学校简介_北京工商大学 |url=http://www.btbu.edu.cn/xxgk/xxgk2/index.htm |accessdate=2018-06-10}}</ref><ref>{{cite news |title=学校良乡校区二期新建工程奠基仪式隆重举行_北京工商大学 |url=http://www.btbu.edu.cn/news/zhxw/132331.htm |accessdate=2018-06-10}}</ref>

=== 北京理工大学 ===
{{main|北京理工大学良乡校区}}
[[北京理工大学|北京理工大学]]良乡校区位于良乡大学城东部。2002年10月，学校与[[房山区|房山区]]人民政府签订《合作共建北京理工大学协议》，2005年11月正式启动良乡校区建设，2007年9月迎来第一批本科生入驻，2015年11月开始良乡校区东区建设。<ref>{{cite web |title=北京理工大学良乡校区概况 |url=http://liangxiang.bit.edu.cn/xqzl/xqgk/index.htm |accessdate=2018-06-10}}</ref>2017年12月，良乡校区教学楼组团项目开建。<ref>{{cite news |title=奠基 8栋大楼一个组团，北理工良乡校区迎来体量最大工程 |url=http://www.sohu.com/a/210266642_154262 |accessdate=2016-06-10 |work=搜狐教育}}</ref>

截止2017年3月，良乡校区总建设面积33.44万平方米，投入建设资金近15亿元，有本科生近9000人，研究生610人，留学生300余人，4个学院整体迁入。预计在“十三五”末，将有12个学院主体与学校机关主体入驻良乡校区。

=== 中国社会科学院大学 ===
{{main|中国社会科学院大学}}
[[中国社会科学院大学|中国社会科学院大学]]主校区设在良乡大学城，占地面积49.4万平方米。<ref>{{cite web |title=学校简介_中国社会科学院大学 |url=http://ucass.gscass.cn/html/list/201/list1.html |accessdate=2018-06-10}}</ref>该校区为中国社会科学院大学前身之一的[[中国社会科学院|中国社会科学院]][[研究生院|中国社会科学院研究生院]]良乡校区，于2008年4月开工，2011年4月正式落成。<ref>{{cite web |title=院史回顾_中国社会科学院研究生院 |url=http://www.gscass.cn/html/list/205/list1.html |accessdate=2018-06-10}}</ref>

=== 北京中医药大学 ===
[[北京中医药大学|北京中医药大学]]良乡校区位于良乡大学城南部，占地面积1433余亩，总建筑规模52万余平方米，于2015年9月投入使用。<ref>{{cite web |title=我校良乡校区正式启用_北京中医药大学 |url=http://www.bucm.edu.cn/xxxw/29646.htm |accessdate=2018-06-10}}</ref>计划于2020年完成科研与教学主体搬迁。<ref>{{cite news |title=http://edu.qianlong.com/2015/1012/28416.shtml |url=http://edu.qianlong.com/2015/1012/28416.shtml |accessdate=2018-06-10}}</ref>

=== 首都师范大学 ===
[[首都师范大学|首都师范大学]]于良乡大学城设有良乡校区基础学部，总占地面积为1064亩,总建筑面积为15-20万平方米。

== 交通 ==
[[北京地铁|北京地铁]][[北京地铁房山线|房山线]]于2010年12月30日开通运营，自东北-西南方向穿过良乡大学城，设有[[良乡大学城西站|良乡大学城西站]]、[[良乡大学城站|良乡大学城站]]、[[良乡大学城北站|良乡大学城北站]]。
