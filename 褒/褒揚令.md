'''褒揚令'''，是[[中華民國政府|中華民國政府]]依據法律表揚對國家有巨大貢獻之[[中華民國國民|國民]]所發之正式文告；「褒揚」是[[法律行為|法律行為]]，需依法定程序核定施行。

== 歷史 ==
[[古中國|古中國]]歷代，對有功之人都制訂獎恤政策。

[[中华民国|中华民国]][[國父|國父]][[孫中山|孫中山]]受舉為[[中華民國臨時大總統|中華民國臨時大總統]]時，即提請[[中华民国初年国会|參議院]]迅速通過《獎恤條例》、成立[[稽勳局|稽勳局]]，著手處理開國前先烈獎恤事宜。[[北京政府|北京政府]]時，仍循例頒布悼恤令（優恤令）。

== 褒揚條件 ==
=== 1914年 ===
褒揚令之[[法源|法源]]，先是[[北洋政府|北洋政府]]於民國3年（1914年）3月11日[[中華民國臨時大總統|大總統]]令制定公布的《褒揚條例》，其第一條規定，有下列行誼之一者得以褒揚<ref>[http://gaz.ncl.edu.tw/detail.jsp?sysid=D1400089 政府公報資訊網--制定「褒揚條例」]</ref>：
# [[孝|孝]]行卓絕著聞鄉里者
# 婦女節烈[[貞操|貞操]]可以風世者
# 特著義行可稱揚者
# 耆年碩德爲鄉里矜式者
# 賑卹鄉族救濟貧困事狀昭著者
# 創興公益事業或捐助財產千元以上爲公益事業者
# 著述書籍製造器用於學術技藝有發明或改良之功者
# 提倡節儉及其他善良風俗化行鄉邑有事狀可稱舉者
# 年逾百歲者

=== 1917年 ===
民國6年（1917年）11月20日修正公布的《褒揚條例》，第一條規定，合於下列之一者由[[內務部|內務部]]呈請褒揚之<ref>[http://gaz.ncl.edu.tw/detail.jsp?sysid=D1700084 政府公報資訊網--修正「褒揚條例」]</ref>：
# 孝行純篤
# 特著義行
# 盡心公益
# 有功藝術
# 碩德淑行
# 睦婣任卹
# 節烈婦女
# 年登百歲

=== 1931年 ===
{{Wikisource|褒揚條例 (民國20年)}}
[[File:方幼祥褒揚令.jpg|thumb]]褒揚令]]
[[立法院|立法院]]於民國20年（1931年）7月4日制定、[[國民政府|國民政府]]7月11日公布的《褒揚條例》，允許褒揚德行優異（凡忠孝仁愛信義和平，足以保存固有之道德者）或熱心公益（凡創辦教育慈善及其他公益之事業，或因辦理此等事業而捐助款項者）者。<ref>[http://lis.ly.gov.tw/lghtml/lawstat/version2/01111/0111120070400.htm 立法院法律系統：褒揚條例（民國20年）]第一條、第二條</ref>[[中華民國內政部|內政部]]審核受褒揚者之事實，擬具褒揚方法，呈請[[行政院|行政院]]轉呈國民政府行之。褒揚方法分為[[匾額|匾額]]與褒章。

=== 1986年 ===
{{Wikisource|褒揚條例 (民國75年)}}
[[File:Yu_Da-wei's_Presidential_Citation_at_Yu_Da-wei_Museum_20110824.jpg|thumb]]褒揚令]]
目前的法條是民國75年（1986年）11月18日修正全文10條、11月28日總統修正公布的《褒揚條例》，<ref>[http://lis.ly.gov.tw/lghtml/lawstat/version2/01111/0111175111800.htm 立法院法律系統：褒揚條例（民國75年）]</ref>其第一條載明褒揚宗旨是：「為褒揚國民立德、立功、立言，貢獻國家，激勵當世，垂之史冊，昭示來茲，特制定本條例。」<ref>[http://law.moj.gov.tw/LawClass/LawSingle.aspx?Pcode=A0030004&FLNO=1 修正《褒揚條例》第一條]</ref>

修正《褒揚條例》第二條規定：有下列各款情事之一者，依本條例褒揚之：<ref>[http://law.moj.gov.tw/LawClass/LawSingle.aspx?Pcode=A0030004&FLNO=2 修正《褒揚條例》第二條]</ref>

# 致力[[國民革命|國民革命]]大業，對國家民族有特殊貢獻者。
# 參預[[戡亂|戡亂]]建國大計，應變有方，臨難不苟，卓著忠勤，具有勳績者。
# 執行國策，折衝壇坫，在外交或國際事務上有重大成就者。
# 興辦教育文化事業，發揚[[中華文化|中華文化]]，具有特殊貢獻者。
# 冒險犯難，忠貞不拔，壯烈成仁者。
# 有重要學術貢獻及著述，為當世所推重者。
# 有重要[[發明|發明]]，確屬有裨國計民生者。
# 德行崇劭，流風廣被，足以轉移習尚，為世楷模者。
# 團結僑胞，激勵愛國情操，有特殊事蹟者。
# 捐獻財物，熱心公益，績效昭著者。
# 其他對國家社會有特殊貢獻，足堪褒揚者。

褒揚方式有明令褒揚以及題頒匾額，但明令褒揚對象限亡故者。<ref>[http://law.moj.gov.tw/LawClass/LawSingle.aspx?Pcode=A0030004&FLNO=3 修正《褒揚條例》第三條]</ref>受褒揚者之事蹟，必須經由[[行政院會議|行政院會議]]通過。<ref>[http://law.moj.gov.tw/LawClass/LawSingle.aspx?Pcode=A0030004&FLNO=4 修正《褒揚條例》第四條]</ref>[[法人|法人]]團體受褒揚，僅能給予頒匾。<ref>[http://law.moj.gov.tw/LawClass/LawSingle.aspx?Pcode=A0030005&FLNO=12 《褒揚條例施行細則》第十二條]</ref>外國人合於條件，亦可褒揚。<ref>[http://law.moj.gov.tw/LawClass/LawSingle.aspx?Pcode=A0030004&FLNO=8 修正《褒揚條例》第八條]</ref>

== 文例 ==
如民國初年歷任總統、元首之悼恤令如下，皆書古文言文體：

* [[孫文|孫文]]（1925年3月12日）：
前臨時大總統孫文，倡導共和，肇我華夏。辛亥之役，成功不居，仍於國計民生殫心擘畫。宏謨毅力，薄海同欽。本執政夙慕耆勳，亟資匡濟，就職伊始，敦勸入都，方期克享遐齡，共籌國是。天胡不憖，遽奪元功。軫念艱虞，彌深愴悼。所有飾終典禮，著內務部詳加擬議，務極優隆，用符崇德報功之至意。

* [[袁世凱|袁世凱]]（1916年6月7日）：
民國肇興，由於辛亥之役。前大總統贊成共和，奠定大局，苦心擘畫，聽夕勤勞、天不假年，遘疾長逝。追懷首績，薄海同悲。本大總統患難周旋，尤深愴痛。所有喪葬典禮，應由國務院轉飭辦理人員參酌中外典章，詳加擬議，務極優隆，用符國家崇德報功之至意。

* [[黎元洪|黎元洪]]（1928年6月8日）：
前大總統黎元洪，辛亥之役，武昌起義，翊贊共和，功在民國。及袁氏僭號，利誘威脅，義不為屈，凜然大節，薄海同欽。茲聞遘疾彌留，猶廑國計，追懷遺烈，愴悼尤深，所有喪葬典禮，著內政部詳加擬議，務示優隆，以彰崇報元勳之典。此令

* [[馮國璋|馮國璋]]（1919年12月30日）:
前代理大總統馮國璋，久掌戎韜，勳隆望重，辛亥之役，贊助共和，疆寄迭膺，又安大局，迨以副總統代理大總統職務，適値南北糾紛之際，苦心規畫，听夕賢勞，退任以來，仍資匡贊，方冀修齡克享，同覩平成，天不憗遺，悲深薄海，彌留之頃，猶以時局危迫，促進統一爲言，愛國之忱，溢於詞表，本大總統睠懷疇昔，愴悼尤殷，所有飾終典禮，著由國務院從優擬議，用符崇德報功之至意，此令

* [[段祺瑞|段祺瑞]]（1936年11月5日）:
前臨時執政段祺瑞，持躬廉介，謀國公忠。辛亥倡率各軍贊助共和，功在民國。及袁氏僭號，潔身引退，刀維正義，節概凜然。嗣值復辟變作，誓師馬廠，迅遏逆氛，卒能重奠邦基，鞏固政體，殊功碩望，薄海同欽。茲聞在滬溘逝，老成凋謝，惋悼實深，應即予以國葬，並發給治喪費一萬元。生平事蹟，存備宣付史館。用示國家篤念耆勳之至意。此令！

*[[徐世昌|徐世昌]]（1939年6月8日）:
徐世昌，國之耆宿，望重群倫。比年息影津門，優遊道素。寇陷華北，屢思威脅利誘，逞厥陰謀，獨能不屈不撓，凜然自守，高風亮節，有識同欽。方幸國有老成，共資矜式。遽聞溘逝，震悼良深。溯其生平，學識閎通，風度沖穆。秉政之日，對內以和平息爭為念，對外以維護主權為心，功雖未竟，志業自有可傳。近聞臥病彌留，攖懷國難，尤見忠誠團結，終始不渝，應予明令褒揚，特給治喪費一萬元，生平事蹟存備宣付國史館，一俟寇患靖平，再議飾終令典，用示國家眷念耆賢，激勵忠貞之至意。此令。

*[[曹錕|曹錕]]（1938年6月14日）:
故陸軍上將曹錕，息影津沽，抱道自重。比歲以來，值寇勢之方張，遭奸佞之叵測，逼迫紛乘，而該上將正氣凜然，始終峻拒，不屈不撓，通國具瞻。且於疾革彌留之際，惓惓以抗戰勝利為念，忠誠純篤，志節昭然，尤見軍人之風範，足垂奕禩之淸芬。今老成永逝，軫悼殊深，允宜明令褒揚，式資當世模楷，待先頒贈「華胄忠良」匾額一方，一俟寇氛靖平，再議飾終令典。凡其舊日僚屬，能斷志勵操矢志報國者，並當一體宏獎，優予登庸。此令。
