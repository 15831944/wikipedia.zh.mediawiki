'''县辖区'''，简称“区”，为[[中国大陆|中国大陆地区]]1990年代前后[[撤区并乡|撤区并乡]]之前普遍存在的一种[[行政区划|行政区划单位]]，区划层次介于[[县|县]]和[[乡级行政区|乡]]之间。县辖区的行政事务机构称为“区公所”。现行《[[中华人民共和国地方各级人民代表大会和地方各级人民政府组织法|中华人民共和国地方各级人民代表大会和地方各级人民政府组织法]]》规定：
<blockquote>
第六十八条 省、自治区的人民政府在必要的时候，经国务院批准，可以设立若干派出机关。<br>
　　县、自治县的人民政府在必要的时候，经省、自治区、直辖市的人民政府批准，可以设立若干'''区公所'''，作为它的派出机关。<br>
　　市辖区、不设区的市的人民政府，经上一级人民政府批准，可以设立若干[[街道办事处|街道办事处]]，作为它的派出机关。
</blockquote>

到2018年为止，中国大陆全境仅存一个县辖区，即[[河北省|河北省]][[张家口市|张家口市]][[涿鹿县|涿鹿县]][[南山区_(涿鹿县)|南山区]]。区划统计上，“县辖区”归入乡镇一级区划层级，由于管辖乡镇，实际上的行政地位高于乡。

== 历史 ==
=== 早期沿革 ===
撤区并乡以前，绝大部分县份的区划单位为区，区的数量从5个到10个，大县多的超过10个。同一时期，县治、少数具有独特经济地位的建制镇为与区处同一行政层次的地位，此类建制镇因由县直管，称为“直管镇”或直辖镇。县辖区的行政机构称为“区公所”，区公所为县政府派出机构，其行政官员称为“区长”。县辖区管辖的乡镇数量平均管辖4～5个，少则2～3个，多则5～8个。

=== 撤区设乡 ===
改革开放后，实行撤区并乡，基本为原来的二个乡（镇）合并为现行一个乡（镇），较小的乡镇则由三四个乡镇合并为一个乡镇，少数面积小、人口少的县辖区，区划调整后仅合并为一个乡镇。

2003年2月20日，山东省[[长岛县|长岛县]]撤销8个区公所，改设为乡镇，至此山东省内不再有区公所设置。

2005年3月，新疆[[若羌县|若羌县]]撤销祁曼塔格区，设立祁曼塔格管理委员会。<ref>{{Cite web |url=http://www.xjbz.gov.cn/html/jrbz2010/2010-12/29/10_58_28_239.html |title=我州撤区建镇工作稳步推进　三县撤区使“小政府”实现大服务 |access-date=2012-02-17 |archive-url=https://web.archive.org/web/20140519093027/http://www.xjbz.gov.cn/html/jrbz2010/2010-12/29/10_58_28_239.html |archive-date=2014-05-19 |dead-url=yes }}</ref>

2005年6月，湖南[[沅陵县|沅陵县]]撤销8个区公所，至此湖南省内区公所全部撤销。<ref>[http://news.sina.com.cn/o/2005-06-22/17096243167s.shtml 湖南乡镇行政区划调整：400个区公所被全部撤销]</ref>

2006年3月，新疆[[且末县|且末县]]撤销阿羌区，所辖乡由且末县直接管辖。<ref>{{Cite web |url=http://www.xjbz.gov.cn/html/jrbz2010/2010-12/29/10_58_28_239.html |title=我州撤区建镇工作稳步推进　三县撤区使“小政府”实现大服务 |access-date=2012-02-17 |archive-url=https://web.archive.org/web/20140519093027/http://www.xjbz.gov.cn/html/jrbz2010/2010-12/29/10_58_28_239.html |archive-date=2014-05-19 |dead-url=yes }}</ref>

2008年6月30日，新疆[[莎车县|莎车县]]撤销所辖的7个区公所。

2009年2月6日，新疆[[和静县|和静县]]撤销[[巴音布鲁克区|巴音布鲁克区公所]]。<ref>[http://zwgk.xjbz.gov.cn/JJ001/html/JJ001-2300-2009-00002.htm 转发自治区人民政府关于同意和静县巴音乌鲁乡区划调整批复的通知] {{webarchive|url=https://web.archive.org/web/20150512004413/http://zwgk.xjbz.gov.cn/JJ001/html/JJ001-2300-2009-00002.htm |date=2015-05-12 }}</ref>

2014年，[[新疆维吾尔自治区|新疆维吾尔自治区]][[喀什地区|喀什地区]][[泽普县|泽普县]]撤销[[奎依巴格区|奎依巴格区]]。

== 努图克 ==
