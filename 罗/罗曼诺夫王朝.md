'''罗曼诺夫王朝'''（{{lang-ru|Романовы}}）是1613年至1917年统治[[俄罗斯帝國|俄罗斯]]的[[王朝|王朝]]。罗曼诺夫王朝是[[俄罗斯|俄罗斯]]的一個王朝，它也是[[俄罗斯历史|俄国历史]]上最强盛的王朝，使俄国由东欧一个闭塞的小国扩展为世界范围的强国。

== [[彼得大帝改革|彼得大帝的革新]] ==
罗曼诺夫王朝的第一代[[沙皇|沙皇]]是[[米哈伊尔·罗曼诺夫|米哈伊尔一世]]，但包括他在内的几位俄罗斯沙皇都不为人们所熟知。王朝最著名的沙皇是[[彼得大帝|彼得一世]]，即彼得·阿列克谢耶维奇·罗曼诺夫，人们一般称他为彼得大帝，後來的君主都稱“皇帝”。由于他积极鼓励俄国西方化，并鼓励废除包括蓄胡须和穿着冗杂的长袍在内的诸多礼节，彼得一世一般被看作是俄罗斯帝国的最初奠基人和开化的代名词。俄国的第一家兵工厂、第一家报馆都是在他的倡导下成立的。

由于彼得大帝的革新，俄国迅速发展成为[[东欧|东欧]]的强国。[[恩格斯|恩格斯]]曾这样指出：“惊慌的西欧，在伊凡统治之初还不知道北方存在着一个[[莫斯科公国|莫斯科公国]]，现在由于看到一个硕大的帝国出现在东部边境而目瞪口呆。”俄国迅速发展各种产业，并在[[大北方战争|大北方战争]]中一举打败俄国的夙敌和北方强国[[瑞典|瑞典]]，夺取了[[芬兰大公国|芬兰大公国]]和[[波罗的海|波罗的海]]的出海口。彼得一世还打败[[奥斯曼土耳其帝国|奥斯曼土耳其帝国]]，夺取了[[黑海|黑海]]的出海口。彼得一世于1721年放弃“全俄罗斯的君主、上帝加冕的神圣沙皇”这一从伊凡四世开始世袭的称号，而选择了“皇帝”这一西欧君主国（如[[神圣罗马帝国|神圣罗马帝国]]）最高统治者的称号，借此象征[[俄罗斯帝国|俄罗斯帝国]]与[[西欧|西欧]]的各个国家有相同的政治地位。

== 荷尔斯泰因-戈托普-罗曼诺夫王朝 ==
彼得二世（彼得·阿列克谢耶维奇，1727年-1730年在位）去世后，罗曼诺夫王朝断绝男嗣，至未婚无嗣的[[伊莉莎白·彼得羅芙娜|伊莉莎白·彼得羅芙娜]]女皇（1741-1762年在位）去世之后，由彼得大帝的外孙[[德国|德国]]人荷尔斯泰因-戈托普王室的彼得·烏爾里希大公继承皇位，稱[[彼得三世_(俄國)|彼得三世]]，荷尔斯泰因-戈托普-罗曼诺夫王朝开始。

王朝另一位著名的皇帝是[[叶卡捷琳娜二世|叶卡捷琳娜二世]]，她在位期间（1762年-1796年），俄国已经从昏睡中觉醒，并且在西欧和国际事务中具有很高的威望。她在位34年，发动了6次大规模的战争：包括三次瓜分[[波兰|波兰]]（1772年、1793年和1795年）、两次[[俄土战争|俄土战争]]和一次[[俄瑞战争|俄瑞战争]]，通过这些战争和强权统治，叶卡捷琳娜二世建立了一个跨欧、亚、美三大洲、领土面积居世界第一位的超级帝国。在这一时期，俄国的国土面积扩大了63万平方公里（其中有46万平方公里来自波兰）。

== 中衰 ==
[[File:Flag_of_the_Russian_Empire_(black-yellow-white).svg|thumb]]
[[亞歷山大一世_(俄國)|亚历山大一世]]统治时期（1801年-1825年），帝国的经济和政治达到顶峰。亚历山大一世最大的功绩在于打败了[[法国|法国]]的侵略者拿破仑·波拿巴，即法蘭西皇帝[[拿破仑一世|拿破仑一世]]。借此绞杀了法国大资产阶级专权的[[法兰西第一帝国|法兰西第一帝国]]。但在亚历山大统治的后期，由于人民的反抗和[[神秘主义|神秘主义]]的影响，俄罗斯帝国开始衰败。1825年至1855年暴君[[尼古拉一世_(俄国)|尼古拉一世]]统治期间，俄国的军事行动频繁，物价飞涨，开始出现崩溃的征兆。尤其是[[克里米亚战争|克里米亚战争]]，使被压迫的农奴们苦不堪言。1856年战争结束之后，俄国的[[农奴制|农奴制]]已经危在旦夕。1861年至1862年皇帝[[亞歷山大二世_(俄國)|亚历山大二世]]决定采取措施，于是有了自上而下的「亚历山大二世改革」，通过改革，俄国开始真正走上了[[资本主义|资本主义]]路线。

== 衰亡 ==
第一次世界大战爆发前夕，俄、英、法三国结成[[协约国|协约国]]集团。1917年，战争即将结束时，由于[[资产阶级|资产阶级]]和[[无产阶级|无产阶级]]运动，罗曼诺夫王朝的统治被挤垮，末代皇帝[[尼古拉二世_(俄罗斯)|尼古拉二世]]被迫在[[奥拉宁堡|奥拉宁堡]]宣布退位，并于1918年7月17日在[[叶卡捷琳堡|叶卡捷琳堡]]与皇室家族一起被[[布尔什维克|布尔什维克]]枪决，至此，罗曼诺夫王朝结束。

=== 后裔 ===

[[亚历山大三世_(俄国)|亚历山大三世]]的两个儿子，末代沙皇[[尼古拉二世_(俄罗斯)|尼古拉二世]]和家人，及弟弟[[米哈伊尔·亚历山德罗维奇_(亚历山大三世之子)|米哈伊尔大公]]在[[十月革命|十月革命]]期间相继被处决。米哈伊尔大公唯一的儿子[[格奥尔基·米哈伊洛维奇，布拉索夫伯爵|布拉索夫伯爵格奥尔基·米哈伊洛维奇]]于1931年在法国逝世。至此，亚历山大三世的男系血统断绝。亚历山大三世的侄子[[基里尔·弗拉基米洛维奇|基里尔·弗拉基米洛维奇]]成为俄罗斯[[王位覬覦者|皇位覬覦者]]。基里尔·弗拉基米洛维奇之子[[弗拉基米尔·基里洛维奇|弗拉基米尔·基里洛维奇]]仅有一女[[玛利亚·弗拉基米洛夫娜|玛利亚·弗拉基米洛夫娜]]。至此，因对[[薩利克法|女性继承权]]的分歧，俄罗斯皇位觊觎者分为两人，即玛利亚·弗拉基米洛夫娜，另为[[季米特里·羅曼諾維奇|季米特里·羅曼諾維奇]]。
