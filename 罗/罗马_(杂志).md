'''罗马'''（{{lang-ar|رومية|Rome}}, {{lang-en|Rumiyah}}）是[[伊斯兰国|伊斯兰国]]用于宣传和招募的[[在线杂志|在线杂志]]。该网站于2016年9月首次出版，并以几种语言发行，包括英语、法语、德语、俄语、印尼语和维吾尔语。<ref name="adnani">{{Cite web|url=https://news.siteintelgroup.com/Jihadist-News/in-new-english-magazine-rumiyah-is-calls-for-lone-wolf-attacks-in-australia-west.html|title=In New Magazine 'Rumiyah,' IS Calls for Lone-Wolf Attacks in Australia, West|last=SITE|website=news.siteintelgroup.com|access-date=2016-09-10}}</ref><ref>{{Cite web |url= https://www.independent.co.uk/news/world/middle-east/isis-propaganda-terror-group-losses-syria-iraq-a7228286.html |author=Bethan McKernan|title=Isis' new magazine Rumiyah shows the terror group is 'struggling to adjust to losses' |publisher=独立报|date=2016-09-06|access-date=2016-09-23}}</ref>

该杂志2016年中发行时取代了《[[達比克_(雜誌)|达比克]]》、《[[伊斯兰之家_(杂志)|伊斯兰之家]]》和其他杂志。分析人士认为，名称的改变，部分原因是由于2016年10月土耳其领导的[[達比克攻勢|对达比克镇的军事进攻]]，致使伊斯兰国即将失去对其控制。<ref>{{Cite web |url= http://www.newyorker.com/magazine/2016/12/12/after-the-islamic-state|author=Robin Wright|title=After the Islamic State|publisher=《纽约客》|date=2016-12-02|access-date=2017-02-07}}</ref><ref name="dabiqfall">{{Cite web |url=http://www.longwarjournal.org/archives/2016/10/town-of-dabiq-falls-to-turkish-backed-forces.php|title=Town of Dabiq falls to Turkish-backed forces|publisher=''The long War Journal''|date=2016-10-17|access-date=2017-02-07}}</ref><ref>{{Cite web |url= https://www.independent.co.uk/news/world/middle-east/isis-islamic-state-children-iraq-syria-western-targets-a7485316.html |author=Kim Sengupta|title=Isis indoctrinating children to plan attacks on Big Ben, Eiffel Tower and Statue of Liberty
|publisher=《独立报》|access-date=2017-02-07}}</ref><ref>{{Cite web |url= http://www.understandingwar.org/sites/default/files/ISW%20The%20Virtual%20Caliphate%20Gambhir%202016.pdf|title=The Virtual Caliphate: ISIS'S Information Warfare
|publisher=''Institute for the Study of War''|access-date=2017-02-07}}</ref>

名称''Rumiyah''（罗马）是引用了[[聖訓|聖訓]]，[[穆罕默德|穆罕默德]]说，穆斯林会依次征服[[君士坦丁堡|君士坦丁堡]]和[[罗马|罗马]]。<ref>{{Cite web |url=http://www.thedailybeast.com/articles/2016/09/08/an-isis-plot-to-blow-up-notre-dame-cathedral-and-rule-the-world.html|title=An ISIS Plot to Blow Up Notre Dame Cathedral—and Rule the World?
|access-date=2016-02-07}}</ref><ref name="WrightState"/> 

就像达比克一样，每期开始都以引用[[阿布·艾尤卜·穆哈吉尔|阿布·艾尤卜·穆哈吉尔]]的话作为开场:“奥穆瓦哈丁，欢呼吧，因为真主阿拉，我们不会从圣战中休息，除非在Rumiyah(罗马)的橄榄树下。”<ref name="dabiqfall"/>

第一期发行于伊斯兰国发言人[[阿布·穆罕默德·阿德纳尼|阿布·穆罕默德·阿德纳尼]]死后。<ref name="adnani"/>2016年10月，[[伊斯兰国|伊斯兰国]]发布了第二期的杂志，其中包括对非穆斯林的攻击，包括详细描述如何对一小群人进行持刀攻击。{{Fact|time=2017-12-06T12:54:00+00:00}}

2016年10月，《罗马》建议追随者进行刺杀攻击，并认为穆斯林历史上的圣战者们以[[安拉|安拉]]的名义“将那些[[卡菲勒|卡菲勒]]用刀剑切割四肢，刺穿他们的身体。”该杂志告诉读者，刀具容易获取和藏匿，并且穆斯林制造好的致命武器，可能会被怀疑。<ref name="WrightState">{{cite news|last1=Wright|first1=Robin|title=The Hand of ISIS at Ohio State|url=http://www.newyorker.com/news/news-desk/the-hand-of-isis-at-ohio-state|accessdate=2017-04-04|publisher=纽约客|date=2016-11-26}}</ref>
