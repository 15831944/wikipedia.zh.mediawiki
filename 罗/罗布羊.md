'''罗布羊'''，产于[[中华人民共和国|中华人民共和国]][[新疆维吾尔族自治区|新疆维吾尔族自治区]][[巴音郭楞蒙古自治州|巴音郭楞蒙古自治州]][[尉犁县|尉犁县]]的一个肉脂兼用型{{le|绵羊品种列表|List of sheep breeds|绵羊品种}}，是当地[[绵羊|绵羊]]中的主体优势畜种<ref name="mutong">{{cite web |title=新疆地方绵羊种质资源及研究利用 |url=http://www.xumurc.com/main/ShowNews_34966.html |website=牧通人才网 |accessdate=2018-08-03 |archive-url=https://web.archive.org/web/20180803123945/http://www.xumurc.com/main/ShowNews_34966.html |archive-date=2018-08-03 |dead-url=no }}</ref>。2009年，罗布羊被[[中国农业部|中国农业部]]列入《[[国家级畜禽遗传资源保护名录|国家级畜禽遗传资源保护名录]]》，属于国家级畜禽遗传资源保护品种<ref>{{cite web |title=中华人民共和国农业部公告 第1325号 |url=http://www.moa.gov.cn/nybgb/2010/deq/201805/t20180529_6145958.htm |website=中华人民共和国农业农村部}}</ref>。2015年，“尉犁罗布羊肉”经中国农业部批准，获得[[国家农产品地理标志认证|国家农产品地理标志认证]]<ref>{{cite web |title=尉犁罗布羊肉获“天下羊肉尉犁香”美誉 |url=http://news.ifeng.com/a/20150824/44502235_0.shtml |website=凤凰网 |accessdate=2018-08-03 |archive-url=https://web.archive.org/web/20180803123945/http://news.ifeng.com/a/20150824/44502235_0.shtml |archive-date=2018-08-03 |dead-url=no }}</ref>。

罗布羊对恶劣自然条件适应能力强，适宜平原区的荒漠、半荒漠草场，实际饲养以大群放牧为主，多以300至500只为一群，全年放牧地均位于[[塔里木河|塔里木河]]及[[孔雀河|孔雀河]]流域的草场。原产地自尉犁县塔里木河大坝沿塔河下行至[[若羌县|若羌县]]境内，包括尉犁县的[[塔里木乡_(尉犁县)|塔里木乡]]、[[墩阔坦乡|墩阔坦乡]]、[[兴平乡_(尉犁县)|兴平乡]]、[[古勒巴格乡_(尉犁县)|古勒巴格乡]]，以及若羌县的[[吾塔木乡|吾塔木乡]]。<ref name="turxun" />

== 品种形成 ==
据尉犁县境内[[塔里木河|塔里木河]]流域的本地老人称，罗布羊是封建时期塔里木河流域一些牧主（[[巴依|巴依]]）所饲养的羊群<ref name="turxun" />。

据[[新疆建设兵团|新疆建设兵团]]塔里木畜牧科技重点实验室利用7个[[微卫星|微卫星]]位点及[[线粒体DNA|线粒体DNA]] [[D環|D-loop环]]序列分析，初步认为罗布羊是由[[中东|中东]]经[[蒙古高原|蒙古高原]]到达[[新疆|新疆]]，后来与新疆南部地方羊种混杂而形成的品种<ref>{{cite journal |author1=郑永富等 |title=新疆罗布羊群体遗传多样性及系统进化分析 |journal=生物技术 |date=2016 |volume=26 |issue=6 |pages=574-580}}</ref>。

== 体形外貌 ==
罗布羊体型紧凑，体质结实，头颈躯干及四肢结构匀称。成年罗布羊体高在70cm以上，体重在50kg左右。羊头比较小，鼻梁为拱起状，两眼突出，耳朵大且下垂。公羊一般有螺旋形的大[[犄角|角]]；母羊一般无角，个别有小角。头部毛多，除鼻梁外面部有比较长的盖毛，额部的鬓毛也比较长。颈部粗短，无肉垂。躯干和四肢均比较粗短，躯干略呈圆筒形，[[羊蹄|羊蹄]]较小。羊尾一般为[[坎土曼|坎土曼]]形状，尾尖向上弯曲。<ref name="turxun">{{cite journal |author1=吐尔逊·司马义 |author2=热西旦·帕塔尔 |author3=包尔汗 |author4=西仁古丽 |title=罗布羊遗传资源调查报告 |journal=草食家畜 |date=2010 |issue=2 |pages=31-32}}</ref>

== 繁殖性能 ==
罗布羊一般12到18个月[[性成熟|性成熟]]，6月至8月[[发情|发情]]，发情期为17天，可自然[[交配|交配]]，也可[[人工授精|人工授精]]<ref name="cnr">{{cite web |title=新疆尉犁罗布羊人工授精取得成效 产羔95% |url=http://news.163.com/11/0223/13/6TJ3REA500014JB5.html# |website=网易-中国广播网 |access-date=2018-08-03 |archive-url=https://web.archive.org/web/20180803123945/http://news.163.com/11/0223/13/6TJ3REA500014JB5.html |archive-date=2018-08-03 |dead-url=no }}</ref>，怀孕期为150天。[[羊羔|羊羔]]的成活率可达90%。<ref name="turxun" />

== 人类利用 ==
=== 羊肉 ===
罗布羊肉的肉质精、[[脂肪|脂肪]]少、膻味小，肉鲜嫩<ref>{{cite web |title=新疆尉犁县“罗布羊”外形普通 肉价却不普通 |url=http://www.chinanews.com/df/2011/12-20/3543600.shtml |website=中国新闻网 |access-date=2018-08-03 |archive-url=https://web.archive.org/web/20111221181951/http://www.chinanews.com/df/2011/12-20/3543600.shtml |archive-date=2011-12-21 |dead-url=no }}</ref>。利用罗布羊制作的羊肉串和[[烤全羊|烤全羊]]曾在美食纪录片《[[人生一串|人生一串]]》第一集中介绍<ref>{{cite web |title=人生一串：第一集 无肉不欢 |url=https://www.bilibili.com/bangumi/play/ep216794 |website=bilibili |access-date=2018-08-03 |archive-url=https://web.archive.org/web/20180803123945/https://www.bilibili.com/bangumi/play/ep216794 |archive-date=2018-08-03 |dead-url=no }}</ref>。

=== 羊毛 ===
罗布羊毛属于异质半粗毛<ref>{{cite journal |author1=王志义 |author2=张茂疆 |title=和田羊毛品质研究及其标准的制订 |journal=中国纤检 |date=1988 |issue=10 |pages=18-20}}</ref>，这种毛型主要由绒毛、两型毛{{efn|两型毛是指羊毛中有不连续的毛髓，一根纤维上同时有细毛和粗毛特性的羊毛}}和少量发毛组成。成年公羊的年产毛量为1.2千克左右，母羊为0.98千克左右<ref name="turxun" />。其羊毛可用于加工[[地毯|地毯]]<ref>{{cite journal |author1=阎风英 |author2=李耿民 |title=新疆地毯羊毛物理性能品质分析报告 |journal=新疆畜牧业 |date=1988 |issue=1 |pages=35-38}}</ref>。

== 商品开发 ==
尉犁罗布羊自2009年获得“国家农产品地理标志认证”，地理标志的发放和管理由社会团体<ref>{{cite web |title=行政许可事项公示 |url=http://www.yuli.gov.cn/Government/PublicInfoShow.aspx?ID=26139 |website=尉犁县政府信息公开平台}}</ref>尉犁县金秋胡杨旅游协会负责<ref>{{cite web |author1=汪涛 |title=“尉犁罗布羊”羊肉“身份证”启用 |url=http://news.ifeng.com/a/20160125/47219637_0.shtml |website=凤凰网-亚心网}}</ref>。为加大该品种的产业发展，尉犁县畜牧兽医部门采取了为母羊人工授精，给养殖户发放种公羊等措施<ref name="cnr" />。

据媒体报道，尉犁县有利用[[电子商务|电子商务]]销售罗布羊肉和制品的现象<ref>{{cite web |title=农民办电商 达西再腾飞 |url=http://paper.ce.cn/jjrb/html/2015-09/10/content_256110.htm |website=经济日报 |access-date=2018-08-03 |archive-url=https://web.archive.org/web/20180803123945/http://paper.ce.cn/jjrb/html/2015-09/10/content_256110.htm |archive-date=2018-08-03 |dead-url=no }}</ref><ref>{{cite web |title=新疆尉犁县“小巴郎”烤全羊将搭上电商快车销往全国 |url=http://news.ifeng.com/a/20150212/43163843_0.shtml |website=凤凰网-亚心网 |access-date=2018-08-03 |archive-url=https://web.archive.org/web/20150501145129/http://news.ifeng.com/a/20150212/43163843_0.shtml |archive-date=2015-05-01 |dead-url=no }}</ref>。
