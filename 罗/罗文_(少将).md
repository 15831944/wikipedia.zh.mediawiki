'''罗文'''（{{bd|1913年||1996年}}），[[辽宁|辽宁]][[凌源|凌源]]人，[[中国人民解放军少将|中国人民解放军少将]]。

== 简历 ==
早年留学[[日本|日本]]，参加“东北青年解放社”，从[[东京大学|东京大学]]经济系肄业。1935年参加[[东北军|东北军]]，[[西安事变|西安事变]]后投向[[延安|延安]]。1938年参加[[八路军|八路军]]进入[[抗大|抗大]]，同年加入[[中国共产党|中国共产党]]。[[第二次国共内战|第二次国共内战]]时期曾任[[第四野战军|第四野战军]]后勤部参谋长。[[中华人民共和国|中华人民共和国]]成立后任[[中国人民解放军|中国人民解放军]][[总后勤部|总后勤部]]计划处处长，[[高级后勤学校|高级后勤学校]]教育长。[[朝鲜战争|朝鲜战争]]时期任[[中国人民志愿军|中国人民志愿军]]后勤司令部参谋处处长，副参谋长。从朝鲜战场回国后任中国人民解放军总后勤部检查局局长，总后勤部车管部副部长，总后勤部运输部部长，总后勤部装备部副部长。1964年晋升少将军衔。<ref>[http://news.ifeng.com/history/special/dbjzhjj/200910/1014_8351_1387702.shtml 凤凰网：罗文少将]</ref>

== 荣誉 ==
朝鲜二级[[国旗勋章|国旗勋章]]、二级[[自由独立勋章|自由独立勋章]]

中国人民解放军二级[[独立自由勋章|独立自由勋章]]、二级[[解放勋章|解放勋章]]、[[独立功勋荣誉章|独立功勋荣誉章]]

== 家庭 ==
