'''罗学瓒'''（1893年1月31日—1930年8月27日）[[湖南|湖南]][[湘潭|湘潭]]马家河南岸（今[[株洲市|株洲市]]天元区）人，早期[[中共|中共]]党员和领导人之一。<ref name="新民学会首批会员罗学瓒 ">{{cite web|url=http://news.xinhuanet.com/politics/2011-09/15/c_122036451.htm|title=《新民学会首批会员罗学瓒》|accessdate=2011年9月15日|work=|publisher=新华网|language=zh-hans}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
== 简介 ==
1894年1月31日，罗学瓒出生于[[湘潭县|湘潭县]]马家河一个农民家庭。<ref name="新民学会首批会员罗学瓒 " />

1906年，罗学瓒在本地青龙桥罗氏[[明德小学|明德小学]]读书。<ref name="罗学瓒  ">{{cite web |url=http://politics.people.com.cn/GB/8198/68298/68302/4638854.html|title=《罗学瓒》|accessdate=2006年7月27日|work=|publisher=人民网|language=zh-hans}}</ref>

1912年，罗学瓒考入[[湖南省立第四师范|湖南省立第四师范]]学校，后来此校并入[[湖南省立第一师范|湖南省立第一师范]]学校，与[[毛泽东|毛泽东]]、[[蔡和森|蔡和森]]等为同班同学。<ref name="新民学会首批会员罗学瓒 "/><ref name="罗学瓒  "/>

1918年4月，罗学瓒成为毛泽东、蔡和森组织[[新民学会|新民学会]]成立后的第一批会员，并成为骨干。。同年8月，入[[北京大学|北京大学]]预备班学习[[法文|法文]]。<ref name="新民学会首批会员罗学瓒 "/><ref name="罗学瓒  "/>

1919年，罗学瓒赴[[法国|法国]]勤工俭学，<ref name="新民学会首批会员罗学瓒 "/>罗学瓒曾先后在[[蒙达尼公学|蒙达尼公学]]学习和在[[施乃德钢铁厂|施乃德钢铁厂]]做工。<ref name="罗学瓒  "/>和[[李维汉|李维汉]]等组织了“工学励进会”，学习[[马克思主义|马克思主义]]，坚定了走[[俄国|俄国]][[十月革命|十月革命]]的道路。在法国期间，罗学瓒因为参加革命活动，1921年10月，他被法国政府强行遣送回国，同年底回到[[上海|上海]]后，即加入[[中国共产党|中国共产党]]。<ref name="新民学会首批会员罗学瓒 "/><ref name="罗学瓒  "/>

1922年秋，罗学瓒回到[[长沙|长沙]]，先后在人力车夫工会、湖南省工团联合会从事工人运动。并胜利领导了人力车工人罢工斗争。<ref name=" 党的早期农民运动领导人、革命烈士——罗学瓒">{{cite web |url=http://hn.rednet.cn/c/2005/05/03/690656.htm|title=《党的早期农民运动领导人、革命烈士——罗学瓒》|accessdate= 2005-05-03 |work=|publisher=湖南频道|language=zh-hans}}</ref>

1923年，罗学瓒任湖南外交后援会文书主任、青年救国团主席，积极参加了反帝爱国运动。在此期间，还在毛泽东创办的自修大学附设的补习学校以及湘江中学任教。[[中共湘区委员会|中共湘区委员会]]改为湖南区委后任区委宣传部长，并以省特派员的身份视察农民运动。

1925年，罗学瓒任中共[[湖南省|湖南省]][[醴陵县|醴陵县]]县委书记。他发展党团员，在各区建立党支部，开展同[[国民党|国民党]]左派的统一战线工作。领导醴陵农民运动，并组织了平民救国团。<ref name=" 党的早期农民运动领导人、革命烈士——罗学瓒"/>

1927年春，毛泽东到[[醴陵|醴陵]]考察农民运动，罗学瓒陪同毛泽东对全县各地农民运动进行了实地考察。<ref name="新民学会首批会员罗学瓒 "/>

大革命失败后坚持斗争，罗学瓒受中央委派，与[[夏明翰|夏明翰]]共同负责湖南省委组织部的工作。<ref name="新民学会首批会员罗学瓒 "/>

1927年9月，罗学瓒任中共湖南省委委员兼[[湘潭|湘潭]]工委书记。<ref name="罗学瓒  "/>

1928年5月，罗学瓒被调到[[上海|上海]]工作，后被派往[[山东|山东]]做党的工作并在山东[[齐鲁大学|齐鲁大学]]任教。<ref name="罗学瓒  "/><ref name="新民学会首批会员罗学瓒 "/>

1929年，罗学瓒被派往[[杭州|杭州]]，参加中共[[浙江省|浙江省]]委的领导工作，先任省委宣传部部长，随后任省委书记。不久被叛徒出卖而被逮捕，在狱中坚强不屈。<ref name=" 党的早期农民运动领导人、革命烈士——罗学瓒"/><ref name="罗学瓒  "/>

1930年8月27日，罗学瓒在[[杭州|杭州]]被秘密杀害，年仅36岁。在刑场慷慨激昂陈词：“死有什么可怕！革命者，只要死得其所。大家不要为我难过，希望继续努力为[[共产主义|共产主义]]奋斗到底！”<ref name=" 党的早期农民运动领导人、革命烈士——罗学瓒"/><ref name="新民学会首批会员罗学瓒 "/><ref name="罗学瓒  "/>
== 著作 ==

《[[不平等条约|不平等条约]]的概述》

== 评价 ==
1961年，[[谢觉哉|谢觉哉]]作诗称赞他和[[陈佑魁|陈佑魁]]：“浑身是劲陈佑魁，鞭辟入里罗学瓒。远瞩高瞻天下小，出生入死一边站。忠义之血不白流，阶级之仇彻底算。三十年前恶战场，巍然两个英雄汉。”<ref name=" 党的早期农民运动领导人、革命烈士——罗学瓒"/>
