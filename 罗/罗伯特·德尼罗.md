'''小罗伯特·安东尼·德尼罗'''（{{Lang-en|'''Robert Anthony De Niro Jr. '''}}，{{bd|1943年|8月17日|||catIdx=D}}）生於[[美國|美國]][[紐約|紐約]][[曼哈頓|曼哈頓]]，知名的[[美国|美国]][[电影|电影]][[演员|演员]]和[[制片人|制片人]]，擁有美國和[[意大利|意大利]]公民身份。<ref>{{Cite web |url = http://www.femalefirst.co.uk/celebrity/Robert+De+Niro-703.html |title = De Niro Will Get Italian Citizenship |publisher = femalefirst.co.uk |language = en}}</ref><ref>{{Cite web |url = http://www.romacinemafest.it/ecm/web/fcr/en/home/international-rome-film-festival/past-editions/2006/news/content/de-niro-i-have-an-italian-passport-i-have-finally-come-home.0000.FCR-352 |title = De Niro: "I have an Italian passport, I have finally come home" |publisher = romacinemafest.it |language = en |deadurl = yes |archiveurl = https://web.archive.org/web/20131230233200/http://www.romacinemafest.it/ecm/web/fcr/en/home/international-rome-film-festival/past-editions/2006/news/content/de-niro-i-have-an-italian-passport-i-have-finally-come-home.0000.FCR-352 |archivedate = 2013年12月30日 |df =  }}</ref>

迪尼路在1974年的電影《[[教父2|教父2]]》中飾演年輕的[[維托·柯里昂|維托·柯里昂]]，為此他贏得[[奧斯卡最佳男配角獎|奧斯卡最佳男配角獎]]。他與導演[[馬田·史高西斯|馬田·史高西斯]]的[[馬田·史高西斯與羅拔·迪尼路|長期合作]]讓他在1980年電影《[[狂牛|狂牛]]》中飾演{{link-en|杰克·拉莫塔|Jake LaMotta}}而獲得[[奧斯卡最佳男主角獎|奧斯卡最佳男主角獎]]。他於2003年獲得[[AFI終身成就獎|AFI終身成就獎]]、2010年獲得[[西席·地密爾獎|金球獎西席·地密爾獎]]，以及2016年獲得由總統[[巴拉克·奧巴馬|巴拉克·奧巴馬]]頒發的[[總統自由勳章|總統自由勳章]]。

迪尼路的第一部主要電影角色是運動電影《{{link-en|战鼓轻敲|Bang the Drum Slowly (film)}}》（1973年）和史高西斯的犯罪電影《[[窮街陋巷|窮街陋巷]]》（1973年）。他因史高西斯執導的心理驚悚電影《[[的士司機_(電影)|的士司機]]》（1976年）和《[[海角驚魂|海角驚魂]]》（1991年）而被提名奧斯卡獎。迪尼路也因[[麥可·西米諾|麥可·西米諾]]的越戰電影《[[獵鹿者|獵鹿者]]》（1978年）、{{link-en|佩尼·馬歇爾|Penny Marshall}}的《[[無語問蒼天|無語問蒼天]]》（1990年）和[[大衛·歐·羅素|大衛·歐·羅素]]的浪漫喜劇電影《[[失戀自作業|失戀自作業]]》（2012年）而得到提名。他在史高西斯的犯罪電影《[[盜亦有道|盜亦有道]]》（1990年）中飾演黑幫{{link-en|James Burke (gangster)|James Burke (gangster)|Jimmy Conway}}和黑色幽默電影《[[喜劇之王_(1982年電影)|喜劇之王]]》（1983年）中飾演Rupert Pupkin，為他取得提名[[英國電影學院獎|英國電影學院獎]]。

迪尼路憑歌舞電影《[[紐約紐約_(電影)|紐約紐約]]》（1977年）、動作喜劇電影《[[午夜狂奔|午夜狂奔]]》（1988年）、黑幫喜劇電影《[[教父咪搞|教父咪搞]]》（1999年）和喜劇電影《[[非常外父揀女婿|非常外父揀女婿]]》（2000年）而獲得四項提名[[金球獎最佳音樂及喜劇類電影男主角|金球獎最佳音樂及喜劇類電影男主角]]。其他著名的演出包括《{{link-en|1900|1900 (film)}}》（1976年）、《[[義薄雲天_(1984年電影)|義薄雲天]]》（1984年）、《[[妙想天開|妙想天開]]》（1985年）、《[[戰火浮生|戰火浮生]]》（1986年）、《[[義膽雄心_(1987年電影)|義膽雄心]]》（1987年）、《[[盜火線|盜火線]]》（1995年）和《[[賭城風雲_(1995年電影)|賭城風雲]]》（1995年）。他曾導演和主演電影，例如犯罪電影《[[不一樣的童年|不一樣的童年]]》（1993年）和間諜電影《[[CIA驚天殺局|CIA驚天殺局]]》（2006年）。

== 生平 ==
=== 早年 ===
他出生于[[美国|美国]][[纽约|纽约]]，[[意大利裔美国人|意大利裔美国人]]，父母均是[[画家|画家]]，而父亲更兼[[雕塑家|雕塑家]]和[[诗人|诗人]]，[[抽象表现主义|抽象表现主义]]画家的父亲于1993年逝世，享年71岁<ref >{{cite web|url=http://ent.163.com/14/0125/11/9JE9R79F000300B1.html | language= | title=德尼罗拍片纪念画家父亲 圣丹斯首映6月份播出 |publisher= m1905电影网 |date=2014-01-25 |accessdate= }}</ref>。在[[曼哈顿|曼哈顿]]的[[小意大利|小意大利]]长大。

16岁时他跟随一个循环演出的剧团第一次以演戏賺钱<ref>{{Harvnb|Dougan|2003| p=17}}</ref>。他第一次上电影是1963年拍摄的《{{link-en|结婚派对|The Wedding Party (film)}}》，不过这部电影一直到1969年才放映。

=== 電影事業 ===
==== 成名之前 ====
[[File:RobertdeNiro26.JPG|thumb]]]]
德尼罗於1972年曾參與經典電影《[[教父_(電影)|教父]]》的試鏡，可惜落選，但亦因而給導演[[法蘭西斯·柯波拉|法蘭西斯·柯波拉]]留下很深印象。翌年德尼罗於《{{link-en|战鼓轻敲|Bang the Drum Slowly (film)}}》中他扮演一位将死的[[棒球|棒球]]运动员廣受好評，更获得了纽约批评家最佳演员奖。同年他接拍日後的長期合作伙伴[[马丁·斯科塞斯|马丁·斯科塞斯]]导演的《[[穷街陋巷|穷街陋巷]]》，將街頭小混混Johnny Boy一角演得淋漓盡致，贏得國家影評人協會最佳男配角獎，二人更開始於荷里活備受關注。<br>

==== 憑教父續集及的士司機成名 ====

由於首集取得空前成功，[[法蘭西斯·柯波拉|法蘭西斯·柯波拉]]決定於1974年開拍《[[教父第二集|教父續集]]》，更邀請德尼罗扮演年輕時期的教父。为演活這角色，德尼罗赴意大利西西里島居住三個多月，更學習了一口流利的西西里語，更模仿其偶像[[馬龍·白蘭度|馬龍·白蘭度]]於《[[教父_(電影)|教父]]》演出時的肢體語言和聲線，出色演技为他贏得[[奥斯卡最佳男配角奖|奥斯卡最佳男配角奖]]，電影亦獲得了[[奥斯卡最佳電影|奥斯卡最佳電影]]，德尼罗迅即成了國際知名影星。

1976年德尼罗第二次與[[马丁·斯科塞斯|马丁·斯科塞斯]]合作，於《[[計程車司機_(電影)|計程車司機]]》扮演從越戰回歸城市的退伍軍人{{link-en|崔維斯·拜寇|Travis Bickle}}（Travis Bickle），並廣泛被認定是其演員生涯中最經典的角色，同時更讓他獲得首個[[奧斯卡最佳男主角獎|奧斯卡最佳男主角獎]]的提名，電影亦獲得[[康城影展|康城影展]]最佳電影[[金棕櫚獎|金棕櫚獎]]，至今亦於多個選舉和投票中被選为影史最佳電影之一。同年德尼罗亦於美國殿堂級導演[[伊力·卡山|伊力·卡山]]最後一部作品《銀色大亨》中擔任主角。

1977年德尼罗於《[[越戰獵鹿人|越戰獵鹿人]]》中再次扮演越戰軍人，這影片獲得[[奧斯卡最佳電影|奧斯卡最佳電影]]，德尼罗個人亦第二度獲得[[奧斯卡最佳男主角獎|奧斯卡最佳男主角獎]]提名，而他在影片中挑戰[[俄羅斯輪盤|俄羅斯輪盤]]中的情節亦廣為人所熟悉。

1980年德尼罗看過[[拳击|拳击]]手杰克·拉·玛的自傳後有意將其拍攝成電影，於是力勸[[马丁·斯科塞斯|马丁·斯科塞斯]]執導，最後成就了《[[愤怒的公牛|愤怒的公牛]]》。德尼罗除了參與真實的拳擊訓練外，为演活年老退休後變得肥胖的杰克·拉·玛，更加重27公斤的體重演出。德尼罗憑這部電影的演出終獲得[[奧斯卡最佳男主角獎|奧斯卡最佳男主角獎]]，至今其演出更被廣泛認定为影史最佳演出。

==== 勇奪影帝後 ====
[[File:Robert_De_Niro_-_GianAngelo_Pistoia.jpg|缩略图]]

80年代德尼罗接拍電影的數量比70年代明顯更多，但獎項和代表作卻較少。1983年德尼罗參演[[马丁·斯科塞斯|马丁·斯科塞斯]]執導的《喜劇之王》，是其首次參與喜劇的演出。1984年德尼罗主演了經典西部片導演[[塞吉歐·李昂尼|塞吉歐·李昂尼]]的史詩式遺作《[[四海兄弟|四海兄弟]]》，是德尼罗於80年代的代表作。1986年德尼罗主演的《[[教會|教會]]》贏得了[[康城影展|康城影展]]最佳電影[[金棕櫚獎|金棕櫚獎]]，其演出卻沒有得到任何獎項。翌年他在《[[鐵面無私|鐵面無私]]》再次增磅演出，扮演著名黑幫頭目[[艾爾·卡彭|艾爾·卡彭]]，角色的戲份雖少卻相當觸目，當中其以棒球棍殺人的情節更成为了經典場面。

==== 再度回勇及首執導筒 ====

1990年德尼罗繼7年後再次與[[马丁·斯科塞斯|马丁·斯科塞斯]]合作，參與《[[盜亦有道|盜亦有道]]》的演出，電影獲得空前的成功，被廣泛評为與《[[教父_(電影)|教父]]》同为影史上最佳的黑幫電影。同年德尼罗於《[[無語問蒼天|無語問蒼天]]》中作出突破性的嘗試，成功演活長年中風及後因藥物過敏而間歇性抽搐的病人，讓他獲得闊別十年的[[奧斯卡最佳男主角獎|奧斯卡最佳男主角獎]]提名。翌年德尼罗於《[[海角驚魂|海角驚魂]]》扮演強姦及殺人犯亦讓他連續兩年獲得[[奧斯卡最佳男主角獎|奧斯卡最佳男主角獎]]提名。

1993年德尼罗首執導筒，執導電影《[[布朗克斯的故事|布朗克斯的故事]]》獲得好評，獲得著名影評人[[羅傑·伊伯特|羅傑·伊伯特]]給予四星滿分的評價。1995年德尼罗與長期合作伙伴[[马丁·斯科塞斯|马丁·斯科塞斯]]最後一次於電影長片合作，與[[莎朗·史東|莎朗·史東]]於《[[賭城風雲_(1995年電影)|賭城風雲]]》飾演夫婦。同年德尼罗與長久被認定作競爭對手的[[艾尔·帕西诺|艾尔·帕西诺]]合演警匪片《[[烈火悍將|烈火悍將]]》，德尼罗扮演匪徒而帕西诺則演探員，二人這次合作中只有三場對手戲，有趣的是二人在整部影片中沒有同時被拍攝在同一個鏡頭內。《[[烈火悍將|烈火悍將]]》亦被廣泛認为是警匪電影的經典作之一。

==== 轉攻喜劇 ====

德尼罗在90年代中後期開始，已較少接拍荷里活一線導演的作品，接拍電影的質數有下降趨勢，獎項亦寥寥可數。當中主要出演的更是喜劇，不少作品如《[[門當父不對|門當父不對]]》系列、《老大靠邊閃》系列在票房上取得相當不俗的成績。值得一提的是，德尼罗於2001年於《The Score》與[[馬龍·白蘭度|馬龍·白蘭度]]合作演出，這是後者臨終前最後一部演出作品，亦是兩位影帝唯一一次合作。

==== 闊別二十年再獲奧斯卡肯定 ====

2010年代德尼罗依然主力接拍喜劇，但在曾合作的新世代演員[[布萊德利·古柏|布萊德利·古柏]]推薦下，接拍了由[[大衛·歐·羅素|大衛·歐·羅素]]執導的《[[派特的幸福劇本|派特的幸福劇本]]》，這有別於十多年內德尼罗所接拍的喜劇，它獲得[[奧斯卡最佳電影|奧斯卡最佳電影]]等多項大獲的提名，擔任男配角的德尼罗更獲得了繼21年前以來、久違的[[奧斯卡金像獎|奧斯卡金像獎]]提名，德尼罗於[[大衛·歐·羅素|大衛·歐·羅素]]期後的兩部作品《[[瞞天大佈局|瞞天大佈局]]》及《[[翻轉幸福|翻轉幸福]]》均有參演，兩部均獲得多項重要獎項的提名。德尼罗於輕喜劇《[[高年級實習生|高年級實習生]]》的演出，更讓他獲得[[評論家選擇獎|評論家選擇獎]]的「喜劇組最佳男主角」提名，這亦是德尼罗闊別多年的重要男主角獎項提名。

== 個人生活 ==
[[File:Robert_De_Niro_and_Grace_Hightower_in_2008.jpg|thumb]]
德尼罗曾結過兩次婚；第一任妻子為美籍非裔演員Diahnne Abbott，兩人1976年結婚，1988年離婚，育有一子Raphael，而迪尼路亦[[領養|領養]]了Abbott與她前夫所生的女兒Drena；第二任妻子為Grace Hightower，兩人1997年結婚，育有一子（1998年出生）。

2006年10月21日獲[[意大利|意大利]]國籍。

2011年12月，诞下一女兒。

== 榮譽 ==

2010年11月9日，[[第68届金球獎|第68届金球獎]][[塞西尔·B·德米尔奖|終身成就獎]]（Cecil B. DeMille award）授予罗伯特·德尼罗（Robert De Niro）<ref>[http://ent.qq.com/a/20101111/000205.htm 罗伯特·德尼罗获第68届美国金球奖终身成就奖]</ref>。德尼罗曾获得过八次金球奖提名，并凭借《愤怒的公牛》问鼎最佳男主角。颁奖典礼于2011年1月16日（周日）在贝弗利山的希尔顿酒店举行<ref>[http://v.youku.com/v_show/id_XMjM3NjkzMjY4.html 罗伯特-德尼罗获金球奖终身成就奖视频]</ref>。

2011年5月，他担任[[第64届戛纳电影节|第64届戛纳电影节]]主竞赛评审团主席。
