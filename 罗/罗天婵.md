'''罗天婵'''（{{bd|1934年|||}}）女，祖籍[[广东|广东]][[梅县|梅县]]，[[客家人|客家人]]，生于[[广东|广东]][[韶关|韶关]]，[[中华人民共和国|中华人民共和国]]抒情女中音歌唱家<ref name=qq/><ref name=nfw>{{Cite web |url = http://www.southcn.com/ent/zhuanti2/2005guangdong/kaocha/ |title =“永恒的旋律”演唱会将在中山纪念堂举行  |author =  |date = 2005 |publisher =  南方网|language =  |accessdate =  |quote =  }}</ref>。

==生平==
1934年罗天婵生于广东韶关，自幼喜爱音乐。1950年考入[[广州|广州]]师范幼儿示范科，学习钢琴演奏和声乐演唱，在校期间经常登台演唱。1953年考入[[中央歌舞团|中央歌舞团]]合唱队声乐专修班，先后师从[[杨金岚|杨金岚]]、[[孙家馨|孙家馨]]、[[魏启贤|魏启贤]]、[[张利娟|张利娟]]等歌唱家和声乐教授，后取得[[文学学士|文学学士]]学位。毕业后留中央歌舞团合唱队及[[中央乐团|中央乐团]]独唱组任首席女中音独唱演员<ref name=qq>{{Cite web |url =http://v.qq.com/x/star/169723#!view=detail?tabid=2  |title = 罗天婵 |author =  |date =  |publisher = 腾讯 |language =  |accessdate = 2018-02-13 |quote =  }}</ref><ref>{{Cite web |url = http://www.cnr.cn/zgzb/zgdwt76/wzsl/201403/t20140316_515086157.shtml |title = 罗天婵 |author =  |date =2014-03-16  |publisher = 央广网 |language =  |accessdate =  |quote =  }}</ref>。

1956年8月，在全国音乐周中，担任大型合唱《红军根据地》的主要领唱和留学生作品专场音乐会的女中音独唱<ref name=qq/>。[[文革|文革]]结束后，她演唱了作曲家[[施光南|施光南]]创作的抒情歌曲《打起手鼓唱起歌》，成为经典名曲。1978年，罗天婵随团到[[海南岛|海南岛]]慰问部队，同行的施光南又带去一首刚完成的作品《吐鲁番的葡萄熟了》，向罗天婵征求意见。回到[[北京|北京]]后，施光南根据罗天婵的建议修改了此歌，邀罗天婵首唱。施光南的另一首经典歌曲《祝酒歌》是在[[粉碎四人帮|粉碎“四人帮”]]后不久写成，并分别给罗天婵和[[中央歌剧院|中央歌剧院]]女中音歌唱家[[苏凤娟|苏凤娟]]试唱，[[李光羲|李光羲]]无意中在同事苏凤娟处见到此歌，很快把它唱红，成为李光羲的保留曲目<ref name=xinmin>{{Cite web |url = http://xmwb.xinmin.cn/html/2013-01/20/content_19_7.htm |title =  博采众长成一格|author =  |date =2013-01-20  |publisher =新民晚报  |language =  |accessdate =  |quote =  }}</ref>。

1986年，罗天婵参加“世界名曲大汇唱”。同年，在第二届[[全国青年歌手电视大奖赛|全国青年歌手电视大奖赛]]中首次担任美声唱法的女中音评委。1999年，参加大型电视音乐会《光荣与梦想——共和国的歌》的录制。2000年出演大型交响清唱剧《江姐》中的[[双枪老太婆|双枪老太婆]]。2003年在[[中国中央电视台|中国中央电视台]]《名家教歌》栏目中为其首唱曲目《打起手鼓唱起歌》、《吐鲁番的葡萄熟了》录制声乐教育片。2005年应邀参加《永恒的旋律——名家名歌广东演唱会》演出<ref name=qq/>。她曾多次随中国艺术团访问[[加拿大|加拿大]]、[[美国|美国]]、[[印度|印度]]、[[日本|日本]]、[[泰国|泰国]]、[[墨西哥|墨西哥]]、[[菲律宾|菲律宾]]、[[马来西亚|马来西亚]]、[[新加坡|新加坡]]及[[香港|香港]]、[[澳门|澳门]]地区<ref name=nfw/>。

罗天婵是第七至十届[[全国政协|全国政协]]委员。

==作品==
罗天婵的代表曲目有《打起手鼓唱起歌》、《吐鲁番的葡萄熟了》、《阿佤人民唱新歌》、《一个黑人姑娘在歌唱》、《歌唱农村新面貌》（广东[[南音|南音]]）、《补衣歌》、《加米拉之歌》、《向北京致敬》、《母爱之歌》、《克拉玛依之歌》、《胡茄十八拍》、《摇篮曲》、《孤独的牧羊人》、《五木催眠歌》、《五彩缤纷》、《哈巴涅拉舞曲》等。她还为许多电影制片厂配录过影视插曲，如《思乡曲》、《湘累》、《春光好》等<ref name=qq/><ref name=nfw/>。
