罗马书写体是一款流通于古罗马的一种书法流派，在一定程度上影响了[[中世纪|中世纪]]的书法。分为旧式罗马书写体与新式罗马书写体。

==  旧式罗马书写体 ==

旧式罗马书写体，又称书信体，是一种一般用于书信使用的日常书写流派。其通常被商人用于商务的书写，也被尚在学习拉丁文的学生用于书写[[拉丁文|拉丁文]], 亦被皇室用于书写皇室文件。当时，已有一种书写速度更快的书写方法在罗马通用, 但旧式罗马书写体一般用于对书写速度要求较高的非正式写作。 虽然这类字体的创造时间一般认为早于一世纪，但1世纪到二世纪是旧式罗马书写体传播的高峰期。

[[File:I_littera_in_manuscripto.jpg|right]]

对当时古罗马适应拉丁文的普通民众来说，旧式罗马书写体非常难以阅读。因为这种字体采用了许多[[连字|连字]], 使许多信件看起来难以辨认——“a”看起来像安色尔体中的a，但左侧的比划依旧是直的，b和d则非常难以区分，p和t也非常类似，v则被写在了基线以上.<ref>Oxford, [http://vindolanda.csad.ox.ac.uk/exhibition/paleo-1.shtml Scripts at Vindolanda] [http://vindolanda.csad.ox.ac.uk/exhibition/paleo-2.shtml page 2] [http://vindolanda.csad.ox.ac.uk/exhibition/paleo-3.shtml page 3]</ref>

==  新式罗马书写体 ==

新式罗马书写体, 由旧罗马书写体改进而成。一般在三世纪到七世纪被广泛使用，比起旧式罗马书写体，新式罗马书写体更容易理解; "a", "b", "d"和 "e" 使用了更容易理解的字形, 使这几个字母与其他的字母更为相称。有学者认为安色尔体中的字母极有可能由新式罗马书写体改进而成，因两种字体的“A”、“G”、“R”和“S”特别相似。<ref>Oxford, [http://vindolanda.csad.ox.ac.uk/exhibition/paleo-1.shtml Scripts at Vindolanda: Historical context].</ref>
