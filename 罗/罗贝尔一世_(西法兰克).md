'''罗贝尔一世'''（{{lang-fr|Robert I<sup>er</sup>}}，{{bd|865年|8月15日|923年|6月15日|catIdx=R}}）是出身于[[卡佩家族|卡佩家族]]的[[法國君主列表|西法蘭克國王]]（922年—923年在位）。

== 生平 ==

罗贝尔一世是卡佩家族的领军人物[[强者罗贝尔|强者罗贝尔]]的儿子、西法蘭克國王[[厄德一世_(西法蘭克)|厄德一世]]的弟弟、大于格的父親、卡佩王朝的開創人[[于格·卡佩|于格·卡佩]]的祖父。898年，其兄長死後，羅貝爾並沒有要求繼承王位，只繼承兄長的王國。

922年，以羅貝爾及其女婿勃艮第公爵拉烏爾一世為首的一班西法蘭克貴族叛變，反對當時的加洛林王朝西法蘭克國王[[查理三世_(法兰西)|查理三世]]。貴族們推舉羅貝爾為西法蘭克國王，實際上成為與查理三世對立的“敵對國王”，查理三世逃亡到洛泰林吉亞（現今的[[洛林|洛林]]），西法蘭克因而出现了两个國王并立的局面。

== 死亡 ==

923年6月15日，罗贝尔一世在与查理三世战斗的[[苏瓦松战役|苏瓦松战役]]中阵亡。而羅貝爾率領的軍隊打敗了查理三世的軍隊，並將查理三世俘虜及囚禁。其後貴族們再推舉勃艮第公爵拉烏爾為西法蘭克國王。

== 配偶与子女 ==

罗贝尔先后跟[[曼恩的阿黛尔|曼恩的阿黛尔]]和[[韦尔芒杜瓦的比阿特丽斯|韦尔芒杜瓦的比阿特丽斯]]结婚，有一子三女：

* [[法兰西的艾玛|法兰西的艾玛]]（894年—934年），於921年嫁給[[勃艮第公爵|勃艮第公爵]][[拉烏爾_(西法蘭克國王)|拉烏爾]]。
* 法兰西的阿黛拉（895年—931年）
* [[伟大的于格|伟大的于格]]（898年—956年）
* 法兰西的里希尔德（生卒年不详）
