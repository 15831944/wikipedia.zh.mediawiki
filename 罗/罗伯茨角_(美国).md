'''罗伯茨角'''（{{lang-en|Point Roberts}}，在当地称为'''Point Bob'''或 '''The Point'''）是美国[[华盛顿州|华盛顿州]][[霍特科姆县_(华盛顿州)|霍特科姆县]]的一个[[普查规定居民点|普查规定居民点]]（CDP），[[美國郵區編號|邮政编码为]]98281<ref>[http://www.zipinfo.com/cgi-local/zipsrch.exe?cnty=cnty&zip=98281 Zip Code Lookup]</ref>，在[[2010年美國人口普查|2010人口普查]]中的人口为1,314人。
罗伯茨角是美国本土的一部分，但是它并不与其地理上相连。罗伯茨角因此是美国的一块飞地。它位于加拿大卑诗省温哥华市市郊的三角洲南面的措瓦森半岛（{{lang-en|Tsawwassen Peninsula}}）的最南端。要想从美国其它地方陆路前往罗伯茨角，只能先入境加拿大。要想直接前往华盛顿州及美国的其它地方，可以通过海路或空路穿越边界湾（{{lang-en|Boundary Bay}}）。

==交通==
连接加拿大不列颠哥伦比亚省三角洲的56街是罗伯茨角唯一的对外道路。另外罗伯茨角还有一个小型机场对公众开放。

== 经济 ==
罗伯茨角有很多邮件代收商店以及[[加油站|加油站]]，以其便宜的油价和接近大温地区的地理位置吸引加拿大居民南下取货和加油。
