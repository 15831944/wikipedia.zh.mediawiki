'''罗曼·尤里耶维奇·罗曼年科'''（{{lang-ru|Рома́н Ю́рьевич Романе́нко}}，{{bd|1971年|8月9日|}}）是[[俄罗斯|俄罗斯]][[政治人物|政治人物]]，已退役[[宇航员|宇航员]]，世界第495位宇航员，俄罗斯第103位宇航员<ref name=nasabio>{{cite web |title=Roman Yurievich Romanenko |url=https://web.archive.org/web/20170414181808/https://www.jsc.nasa.gov/Bios/htmlbios/romanenko.html |publisher=NASA |accessdate=2019-04-20}}</ref>。
==早年==
出生于宇航员世家，他的父亲[[尤里·罗曼年科|尤里·罗曼年科]]曾经三次飞往太空。1988年毕业于[[列宁格勒|列宁格勒]][[亚历山大·瓦西里耶维奇·苏沃洛夫|苏沃洛夫]]军事学校。1988年进入[[切尔尼戈夫|切尔尼戈夫]]空军飞行员学校，1992年毕业，获得飞行员工程师资格<ref name=nasabio/>。
==宇航员生涯==
1997年12月被选为[[加加林宇航员培训中心|加加林宇航员培训中心]]测试宇航员候选人。1997年12月至1999年11月，完成了所有航天基础训练。1999年11月，在成功通过所有考验后获得了测试宇航员资格<ref name=nasabio/>。
===远征20/21===
2009年5月27日10时34分<ref>{{cite web |title=Expedition 20 Crew Launches from Baikonur |url=https://www.nasa.gov/mission_pages/station/expeditions/expedition20/launch.html |publisher=NASA |accessdate=2019-04-20}}</ref>，[[联盟TMA-15|联盟TMA-15]]携带罗曼年科、[[弗兰克·德温内|弗兰克·德温内]]和[[罗伯特·瑟斯克|罗伯特·瑟斯克]]从哈萨克斯坦[[拜科努尔发射场|拜科努尔发射场]]成功升空<ref>{{cite news |author=聶雲鵬 |title=俄羅斯“聯盟TMA-15”載人飛船奔赴國際空間站|url=http://big5.china.com.cn/international/txt/2009-05/28/content_17847859.htm |work=[[中國網|中國網]] |agency=[[新华网|新华网]] |date=2009-05-28 |archiveurl=https://web.archive.org/web/20190420083015/http://big5.china.com.cn/international/txt/2009-05/28/content_17847859.htm |archivedate=2019-04-20}}</ref>。

2009年12月1日7时15分，[[联盟TMA-15|联盟TMA-15]]成功降落在哈萨克斯坦大草原上<ref>{{cite web |title=Expedition 21 Crew Lands in Kazakhstan |url=https://www.nasa.gov/mission_pages/station/expeditions/expedition21/exp21_land.html |publisher=NASA |accessdate=2019-04-20}}</ref>。

2012年12月19日12时12分，俄[[联盟-FG|联盟-FG]]运载火箭携[[联盟TMA-07M|联盟TMA-07M]]号载人飞船在[[哈萨克斯坦|哈萨克斯坦]]境内的[[拜科努尔发射场|拜科努尔发射场]]起飞，随后载人飞船与火箭第三级成功实现分离，并进入预定轨道飞行。其中罗曼年科是飞船的指令长<ref>{{cite web |author=李晶晶 |title=俄罗斯联盟号飞船升空 3宇航员飞往国际空间站 |url=http://world.huanqiu.com/regions/2012-12/3399990.html?agt=77_2.undefined |publisher=[[环球网|环球网]] |archiveurl=https://web.archive.org/web/20190420080934/http://world.huanqiu.com/regions/2012-12/3399990.html?agt=77_2.undefined |archivedate=2019-04-20 |date=2012-12-20}}</ref>。

2013年4月19日下午14时03分，他和[[帕维尔·维诺格拉多夫|帕维尔·维诺格拉多夫]]打开舱门，进行了6小时38分钟的[[舱外活动|舱外活动]]<ref>{{cite web |author=Pete Harding |title=Cosmonauts successfully conclude Russian spacewalk |url=https://www.nasaspaceflight.com/2013/04/cosmonauts-russian-spacewalk-iss-rs32/ |publisher=nasaspaceflight |accessdate=2019-04-18 |date=2013-04-19}}</ref>。这次太空行走的主要任务是安装并启动太空实验装置“情况”，以便研究“超级航天器”与[[电离层|电离层]]在国际空间站近表层区域的相互作用。另一项任务是更换[[星辰号服务舱|星辰号服务舱]]大气[[透射|透射]]计的[[照準器|照準器]]<ref>{{cite web |author=贺颖骏 |title=俄罗斯宇航员实施今年首次太空行走 |url=http://tech.ifeng.com/discovery/detail_2013_04/20/24426262_0.shtml |publisher=[[新华网|新华网]] |archiveurl=https://web.archive.org/web/20190420081341/http://tech.ifeng.com/discovery/detail_2013_04/20/24426262_0.shtml |archivedate=2019-04-20}}</ref>。

2013年5月14日2时31分，他和队员乘坐联盟TMA-07M在哈萨克斯坦东南部安全着陆<ref>{{cite web |author=王琦琛 |title=三名国际空间站宇航员返回地球 曾录制首个太空MV |url=http://www.chinadaily.com.cn/hqzx/2013-05/14/content_16497348.htm |publisher=[[中国日报|中国日报]] |accessdate=2019-04-20 |archiveurl=https://web.archive.org/web/20141018082500/http://www.chinadaily.com.cn/hqzx/2013-05/14/content_16497348.htm |archivedate=2014-10-18 |date=2013-05-14}}</ref>。

==退役==
2015年10月12日，他因成功当选第七届[[国家杜马|国家杜马]]议员而从宇航员队退役。从10月14日起，他就任[[统一俄罗斯|统一俄罗斯]]党代表[[阿穆尔州|阿穆尔州]]的国家杜马议员<ref>{{cite web |title=Космонавт-единоросс Роман Романенко стал депутатом Госдумы |url=http://www.vesti.ru/doc.html?id=2675149&cid=5 |publisher=Vesti |accessdate=2019-04-20 |language=ru |date=2015-10-14}}</ref>。
