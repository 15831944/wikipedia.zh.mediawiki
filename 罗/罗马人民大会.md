'''罗马人民大会'''是[[古罗马|古罗马]]时期[[羅馬共和國|羅馬共和國]]的[[立法機構|立法機構]]。会议推行[[直接民主制|直接民主制]]，权力建立在普通公民的基础之上，而经选举产生的代表掌握[[权力|权力]]，罗马人民大会与[[罗马元老院|罗马元老院]]的组成是当今议会[[兩院制|兩院制]]的雛形。大會授與正式的政府權力，設給四個獨立分開的人民會議—[[區會議|區會議]]、[[百人會議|百人會議]]，[[部族會議|部族會議]]，以及[[平民會議|平民會議]]。与近代[[議會|議會]]不同，大会兼有[[立法|立法]]、[[司法|司法]]與[[選舉|選舉]]的功能，就像近代[[議會制|議會制]]的下議院一樣，雖然沒有[[權力分立|權力分立]]，但體制存在制衡，在羅馬共和國中是非凡出眾地精心結構的。他們擁有權力去制定[[追溯法令|追溯法令]]，溯及既往地，使既成法律不合法。值得注意的是，[[羅馬元老院|羅馬元老院]]是審議法律的最高權力機構，但並未擁有立法或司法的權力。

== 區會議 ==
區會議，又称“库里亚大会”，是最為古老的羅馬會議，在"卡拉塔會議"("{{link-en|Comitia Calata|Comitia Calata}}")之後，代表原始的三個羅馬部族，組織在三十個“區”(库里亚)，投票單位，即各自投出一張集體的票。這項機制，原始地選出所有較高階的長官，但是後來只確認與舉行就職典禮，那些被百人會議所選出的人。

== 百人會議 ==
百人會議，又称“森都利亚大会”，包括[[貴族|貴族]]與[[平民|平民]]兩者，組織成五個經濟階級([[騎士經濟階級|騎士經濟階級]]與元老階級是第一階級)，而且分配，在內部的分派當中，稱為百人隊。身份資格，在百人會議中，需要確定的經濟地位，而且權力已重重地授與第一與第二階級。百人會議，開會每年地，去選出次年度的[[執政官|執政官]]與[[副執政|副執政]]，以及每五年去選出[[監察官|監察官]]。它也開會去審理''叛國重罪''(''perduellio'')的案子，儘管這後來的功能廢而不用，[[路奇烏斯·阿普萊烏斯·薩圖爾尼努斯|路奇烏斯·阿普萊烏斯·薩圖爾尼努斯]]引入較為行得通的型態(''maiestas'')之後。

公民的投票，並未計算在百人會議中。說得更恰當一點，個人的投票，已計算在他的[[百人隊|百人隊]]之內，而且決定[[百人隊|百人隊]]的投票結果。因為只有第一階級的十八個(而且最富裕的)百人隊，被維持至一百名成員的名義上的量額，那些百人隊成員，行使不成比例的影響力，對投票的結果。百人會議，原始地武士的軍事會議，必須開會在''[[羅馬城神聖疆域|羅馬城神聖疆域]]''之外，在[[戰神廣場_(羅馬)|戰神廣場]]上，而且是，就效果來說，極端地笨拙去召開與管理。它並非常態地使用，除非去選舉次年度的長官。百人會議被慣常地引用，以極少的當代證據，作為古典的先例，替[[美國|美國]]的[[選舉團|選舉團]]。

== 部族會議 ==
''部族會議''(''Comitia Tributa'')包括[[貴族|貴族]]與[[平民|平民]]兩者，分配在三十五個[[部族|部族]]當中，所有的[[羅馬|羅馬]]公民被安置在其中，為了管理與選舉的目的。羅馬城市人口的絕大多數，被分配在四個城市的部族之中，它意味著，他們的票數，就個人而言，是不重要的。就像百人會議，投票是間接的，以一票分配給各自的部族。投票已因此嚴重地傾斜，有利於三十一個[[農村|農村]]部族。部族會議開會，在民會會場的凹處，在[[羅馬中樞區|羅馬中樞區]]，而且選出[[市政官|市政官]](只是''座椅資格'')，[[財務官|財務官]]，以及士兵的[[中隊長|中隊長]](''tribuni militum'')。它處理大多數的審判，直到[[羅馬獨裁官|獨裁官]][[蘇拉|蘇拉]]建立常設的法庭(''quaestiones'')。

== 平民會議 ==
[[平民會議|平民會議]]也是部族會議，但是它排除所有的貴族，他們被禁止去參加它的開會。只有平民[[保民官|保民官]](''tribuni plebis'')，能夠召開平民大會，而且它慣例地開會在民會會場的凹處(貴族的元老常常觀望從[[霍斯提里烏斯會所|霍斯提里烏斯會所]]的階梯，以及問難[[保民官|保民官]]，在開會的時候。羅馬的政治，是相當地更為粗暴蠻橫，比起甚至近代的[[英國下議院|英國下議院]])。平民大會是共和之受惠的立法機構，儘管技術上來說，它的法律被稱為“[[公民投票|公民投票]]”。它選舉[[市政官|市政官]](只是''平民'')，以及平民[[保民官|保民官]]，而且進行審判；後者的功能，廢而不用，[[蘇拉|蘇拉]]建立常設法庭之後。

== 元老院 ==
雖然常設的[[羅馬元老院|羅馬元老院]]通過元老院的勸告(senatus consulta)，建議法律與措施，這些是適合相比的，與近代[[聯合國|聯合國]]的決議，而且並未實際上地貫徹任何法律的力量—除了在元老院議決為捍衛共和(senatus consultum de republica defendenda)的案件中，所謂的終極命令，建立袖珍的獨裁官(pocket dictator)，透過指示[[執政官|執政官]]去留心著共和不應受到傷害，以及在[[元老院終極議決|元老院終極議決]]的案件中，一種深思熟慮，以動亂(tumultus)的命令為依據，而且那常常導致iustitium，意思是停止或法律之停止的公佈。
 
前者能夠被看到，在[[朱古達戰爭|朱古達戰爭]]的指導中，當元老院通過''[[元老院終極議決|元老院議決]]''，延展[[克溫圖斯·凱奇利烏斯·美提路斯·努米狄庫斯|克溫圖斯·凱奇利烏斯·美提路斯·努米狄庫斯]]的保有期，身為最高指揮官，但是平民大會通過[[公民投票|公民投票]]，指派[[蓋烏斯·馬利烏斯|蓋烏斯·馬利烏斯]]替換[[美提路斯·努米狄庫斯|美提路斯·努米狄庫斯]]的。雖然[[凱撒|凱撒]]被''元老院議決''提名為[[山南高盧人|山南高盧人]]與[[伊里利庫姆|伊里利庫姆]]的[[資深長官|資深執政官]]，他卻被[[公民投票|公民投票]]給予[[阿爾卑斯山那邊的高盧|阿爾卑斯山那邊的高盧]]({{link-en|Gallia Transalpina|Gallia Transalpina}})。第二個例子能夠被看到，在漢尼拔入侵意大利的前後關係中。[[元老院終極議決|元老院終極議決]]呼籲所有的公民，去採取不論什麼必要的措施，對國家的拯救。''動亂''(''Tumultus'')是社會與政治本質的事件，需要非常的措施。''動亂''的宣告常常導致''停止''(''iustitium'')的公佈，根據它，公民被呼籲去捍衛國家，''而且留神著無使傷害到達它''。

元老院也有能力去恢復法律的治理，在緊急期間的結束時，以及歸還共和長官的[[統治大權|統治大權]]。這就是案例，漢尼拔戰敗之後。

在政治上的術語，雖然，尊崇的表示'''元老院及羅馬人民'''(Senatus Populusque Romanum)(縮寫為SPQR)，常常用來作為一種表示， 替羅馬的國家，清楚地證實全面的認知，即羅馬合法地被人民的意志(在會議中)，與元老院，以及在他們的威權之下，被長官所治理。只有當[[元首制|元首制]]已建立之時 - 合法地在共和之內，它從未被撤銷 - 的是權力的移轉至皇帝，不再是一位'臨時的'強人，好像與[[羅馬獨裁官|獨裁官]]相反，開始去具體表現國家政治上地，雖然從未依照法律，而且從此使羅馬的''莊嚴''(maiestas)具體化。

== 蘇拉的改變 ==
在他的[[執政官|執政官]]任期，在公元前88年，[[蘇拉|蘇拉]]通過一系列的三條法律，阻礙部族會議與平民會議考慮任何的法律，除非它被''[[元老院終極議決|元老院議決]]''，以贊同的"請通過"("do pass")的建議送至會議。他的第四條法律，再建構百人會議，以致於第一階級 — 元老階級與最有勢力的[[騎士經濟階級|騎士經濟階級]] — 具有接近百分之五十的投票權力。他的第五條法律，剝奪兩個人民大會 — 部族會議與平民會議 — 它們的立法功能，把所有的立法權，交給再建構的百人會議手中。部族會議已保留著確定長官的選舉，與審判的指導 — 但是沒有審判能夠被舉行，除非被''[[元老院終極議決|元老院議決]]''授權。 

這些改革被[[馬利烏斯|馬利烏斯]]與[[秦納|秦納]]所領導的[[平民派|平民派]]推翻，然而又被[[蘇拉|蘇拉]]在他的[[羅馬獨裁官|獨裁官]]任期（''共和體制的''）時恢復，而且再度被推翻，在他的去世之後。它們說明一種極為廣泛的昇降浮沉與直接的轉換，在羅馬國家的體制中，在共和與[[羅馬帝國|帝國]]的期間。
