'''罗瑞卿'''（{{bd|1906年|5月31日|1978年|8月3日|罗瑞卿}}），原名'''其荣'''，[[四川|四川]][[南充|南充]]人。[[中国工农红军|中国工农红军]]、[[中国人民解放军|中国人民解放军]]及[[中华人民共和国|中华人民共和国]][[党和国家领导人|领导人]]之一，曾任[[中共中央书记处书记|中共中央书记处书记]]、[[国务院副总理|国务院副总理]]、[[解放军总参谋长|解放军总参谋长]]等党、政、军要职，授[[中国人民解放军大将|中国人民解放军大将]]军衔。

罗瑞卿早年考入[[黄埔军校|黄埔军校]]，随后参加[[國民革命軍北伐|國民革命軍北伐]]，之后加入[[中国共产党|中国共产党]]。随后参加[[红四军_(红一方面军)|红四军]]，担任红四军政治部宣传部长等职位，之后改为[[红一军团|红一军团]]保卫局局长，随[[中央红军|中央红军]]进行[[长征|战略转移]]。抗日战争期间，担任[[抗日红军大学|抗日红军大学]]校长、[[十八集团军|十八集团军]]野战政治部主任等职位。第二次国共内战期间，罗担任晋察冀中央局、华北军区政治部主任兼第二兵团第一政治委员，率领第二兵团参与石家庄、[[太原战役|太原战役]]等。中华人民共和国成立后，其[[中央人民政府公安部|中央人民政府公安部]]、[[中华人民共和国公安部|中华人民共和国公安部]]首任部长、第2、3届[[中华人民共和国国务院副总理|中华人民共和国国务院副总理]]、[[中国人民解放军总参谋部|中國人民解放軍總參謀長]]等职。[[文化大革命|文化大革命]]期间，因[[罗瑞卿事件|罗瑞卿事件]]而被批判，后跳楼负重伤。[[文革|文革]]结束后[[平反|平反]]。1978年赴[[西德|西德]]治疗腿疾时病逝。

==生平==
===早年革命生涯===
1906年5月31日，罗瑞卿出生于[[四川|四川]][[南充|南充]][[舞凤乡|舞凤乡]]清泉坝马家坡，此后在村中读私塾。1920年，考入南充北区大林寺高等小学。1923年，受其[[外祖父|外祖父]]资助考入南充县立中学。1926年7月离家出走，到考入成都高等蚕桑学校，但因凑不足学费未能入学。1926年10月在重庆结识了共青团四川省委工作的[[任伯芳|任伯芳]]，与[[任伯芳|任伯芳]]一起考入国共合作创办的武汉[[中央军事政治学校|中央军事政治学校]]（即[[黄埔军校|黄埔军校]]武汉分校），12月乘船东下武汉。1927年2月1日编在入伍生总队（黄埔军校第六期）政治第一大队第二队，与[[陈伯钧|陈伯钧]]同学。1927年[[四一二事件|四一二事件]]，国共合作破裂。1927年5月中旬至6月下旬，黄埔军校武汉分校被编入[[叶挺|叶挺]]指挥的[[叶挺独立团|中央独立师第一团]]，参加讨伐[[夏斗寅|夏斗寅]]和西征[[杨森|杨森]]的战斗<ref>{{cite book |title = 将帅从这里起步 |authors=张植信 等 编著 |year=1988 |page =167 |location=沈阳 |publisher=辽宁人民出版社 |ISBN = 7-205-00393-8}}</ref>。后因作战表现出色，团长蓝腾蛟调到团部当传令兵。1927年7月中下旬，黄埔武汉分校被改编为第二方面军军官教导团（团长叶剑英）东征讨蒋，罗瑞卿编入共产党员[[叶镛|叶镛]]为连长的第二连任副班长。1927年8月2日教导团由武昌沿江东下，准备参加[[南昌起义|南昌起义]]，船到[[黄石市|黄石]]，传来了南昌起义的消息，[[张发奎|张发奎]]担心教导团也发生叛乱，8月4日在九江把教导团缴枪，愿走愿留自便。当天晚上中共在教导团的负责人[[陈毅|陈毅]]召集共产党员开会决定已暴露的党员分散，或去南昌找起义军，或回乡从事农民运动。罗瑞卿领了十几块钱路费回到了武汉，8月中旬至9月下旬伤寒病入[[武汉市仁济医院|仁济医院]]住院一个多月。出院后住在汉口四川会馆。由于武汉的[[白色恐怖|白色恐怖]]越来越厉害，1928年1月罗瑞卿离开武汉到湖南[[常德|常德]]。1928年2月去[[澧州|澧州]]，经[[任伯芳|任伯芳]]介绍在[[鲁涤平|鲁涤平]]所属第二军教导师政治部宣传科任上尉宣传员。1928年6月离开鲁部队到长沙。1928年8月罗瑞卿和任伯芳一起到上海。1928年10月，“经和党中央接上关系的[[任伯芳|任伯芳]]介绍，同[[中共中央军委|中央军委]]的工作人员[[欧阳钦|欧阳钦]]接上了头，在上海转为中国共产党党员。”。<ref>[http://cpc.people.com.cn/GB/64162/82819/93664/97833/6067152.html 罗瑞卿生平大事年表（1926-1935）]</ref>

1929年，受中共中央军委派遣，离开上海赴[[湘鄂西苏区|湘鄂西苏区]]红军，后因交通堵塞折返上海。次月，赴正在闽西的[[红四军_(红一方面军)|红四军]]工作，后到闽西上杭蛟洋游击队任教官。5月，[[朱德|朱德]]、[[毛泽东|毛泽东]]率部再次入闽。罗瑞卿与[[傅柏翠|傅柏翠]]、[[曾省吾|曾省吾]]率部协同红四军作战，掩护红四军向龙岩前进。同月，上杭蛟洋游击队改编为红59团，傅柏翠任团长，罗瑞卿任参谋长。次月，部队红四军第四纵队，傅伯翠任司令员，张鼎丞任党代表，罗瑞卿任参谋处主任。9月，红4纵队参加攻打[[上杭|上杭]]的战斗，被调任第二纵队五支队任党代表。10月，率部随红四军主力出击[[东江|东江]]、[[长汀|长汀]]，随后调往二纵队政治部任宣传科长。12月，参加[[古田会议|古田会议]]，次年改为第二纵队政治部主任<ref name="bio2">{{cite web | language = zh-hans | publisher = 网易 | title = 罗瑞卿 | url = http://war.news.163.com/07/0724/19/3K6KP57000011232.html | author = 国际在线 | date = 2007-07-24 | accessdate = 2010-06-23 }}</ref>。

1930年，部队进行整编，部队改为红四军第十一师，罗瑞卿任[[党代表|党代表]]，后改任[[政治委员|政治委员]]<ref>{{cite book |author =《中国人民解放军军史》编写组|title=中国人民解放军军史：第一卷 |year=2010 |publisher=军事科学出版社 |page = 130 |ISBN = 978-7-80237-381-5}}</ref>。之后指挥参加攻占[[吉安市|吉安]]、[[峡江|峡江]]的战斗。12月，参加[[中国共产党第一次反围剿战争|中共第一次反围剿战争]]，并率部在[[龙冈战斗|龙冈战斗]]中与其他部队诱敌深入，一举全歼[[张辉瓒|张辉瓒]]两个旅和师部，并活捉国军十八师师长张辉瓒。次年，罗瑞卿与[[曾士峨|曾士峨]]率部向东追击[[谭道源|谭道源]]师，歼灭三千余人。3月，随[[红一方面军|红一方面军]]主力向南转移，参与[[中国共产党第二次反围剿战争|中共第二次反围剿战争]]；5月16日，与曾士峨奉命率红11师抢占观音崖隘口，协助红军主力围歼[[王金钰|王金钰]]部第28师大部和第47师，在该战中身负重伤，直至年底才康复<ref>{{cite book |language = zh-hans |author=《罗瑞卿传》编写组 |title=罗瑞卿传 |pages=27-31 |year=2007 |location=北京 |publisher=当代中国出版社 |ISBN = 978-7-800-92501-6}}</ref>。12月，任红四军政治部宣传部长兼四军随营学校政治委员。

[[File:Yang_Shangkun1936-2.jpg|thumb]]、[[杨尚昆|杨尚昆]]、[[聂荣臻|聂荣臻]]、[[徐海东|徐海东]]。后排左起：罗瑞卿、[[程子华|程子华]]、[[陈光|陈光]]、[[邓小平|邓小平]]。]]

1932年3月，红一军团进行整编，罗瑞卿任红四军政治委员。次月，与[[王良_(紅軍)|王良]]率部参加[[漳州战役|漳州战役]]，全歼[[张贞|张贞]]国军第49师。5月，回师赣南中央苏区。7月10日，指挥红四军参加[[南雄|南雄]]、[[水口|水口]]战役，先后击溃粤敌15个团。8月，罗指挥红四军主攻乐安、宜黄，全歼国军第27师，俘获五千余人。10月，率部参加建黎泰战役，次月占领[[建宁|建宁]]、[[黎川|黎川]]、[[泰宁|泰宁]]广大地区。11月，部队北进，占领[[资溪|资溪]]、[[金溪|金溪]]。1933年，罗改任[[红一军团|红一军团]]保卫局局长，参与指挥[[中国共产党第四次反围剿战争|中共第四次反围剿战争]]。2月，罗与[[徐彦刚|徐彦刚]]率红七师、九师参加[[黄陂伏击战|黄陂伏击战]]，全歼国军第52师，活捉师长李明。3月21日，参加草台冈战斗，国军第11师大部被歼。同年9月，参加[[中国共产党第五次反围剿战争|中共第五次反围剿战争]]。

1934年1月15日，罗瑞卿赴[[瑞金|瑞金]]，出席[[中华苏维埃共和国|中华苏维埃共和国]]第二次全国苏维埃代表大会。此后，随红一军团参加三溪圩、广昌、温坊战斗等。10月，率领一军团保卫局随[[中央红军|中央红军]]开始[[长征|战略转移]]。1935年，参加强渡[[乌江|乌江]]。并为[[中共中央政治局|中共中央政治局]][[遵义会议|遵义会议]]作外围警戒工作<ref name="欧阳青">{{cite book |author=欧阳青 |title=大授衔：1955共和国将帅授衔档案|year=2011 |publisher=长城出版社 |location=北京|pages=297|isbn=978-7-5483-0058-8 |language = zh-hans}}</ref>。1月，率部六渡赤水，5月19日，罗瑞卿改任中央红军先遣队参谋长，并率先遣队通过大凉山彝族区，参与指挥强渡大渡河等。此后翻越夹金山、雪山，抵达毛儿盖，后到达班佑。9月20日，红一方面军和军委纵队整编为陕甘支队，罗瑞卿任第二纵队政治部主任，次月与[[彭雪枫|彭雪枫]]、[[李富春|李富春]]率第二纵队到达陕北[[吴起镇|吴起镇]]。不久调任红一方面军保卫局局长，并参加[[直罗镇战役|直罗镇战役]]。

=== 抗日战争时期 ===
[[File:1940nierongzhen.jpg|thumb]]、罗瑞卿、[[刘伯承|刘伯承]]、[[朱德|朱德]]、[[蔡树藩|蔡树藩]]、[[邓小平|邓小平]]]]

1936年2月至5月，罗随红一方面军参加[[东征战役|东征战役]]，之后返回陕北，担任[[抗日红军大学|抗日红军大学]]教育长，随后作为中共代表团成员随周恩来赴西安解决[[西安事变|西安事变]]。1937年，担任[[抗日红军大学|抗日红军大学]]校长。1938年，任[[抗日军事政治大学|抗日军事政治大学]]副校长<ref name="bio2" />。2月，赴武汉，代表八路军总部参加国民政府军事委员会政治部召开的政治工作会议，挂上校军衔<ref>{{cite web|url =http://www.sn.xinhuanet.com/2008-07/30/content_13975439.htm|title =红军改编为八路军初期部队作战序列|date =2008-07-30|language =zh-hans|publisher =新华网|deadurl =yes|archiveurl =https://web.archive.org/web/20150924103019/http://www.sn.xinhuanet.com/2008-07/30/content_13975439.htm|archivedate =2015-09-24}}</ref>。此后数年一直担任[[抗大|抗大]]校长，培养中共干部<ref name="bio">{{cite web | language = zh-hans | publisher = 新华网 | title = 罗瑞卿 | url = http://news.xinhuanet.com/ziliao/2004-07/03/content_1567621.htm | author =  | date = |accessdate = 2010-06-23 }}</ref>。

1940年2月，[[抗大|抗大]]总校从陕北迁移到山西，此后罗瑞卿调任[[国民革命军第十八集团军|国民革命军第十八集团军]]野战政治部主任。8月，协助彭德怀指挥安排[[百团大战|百团大战]]中的政治工作，并组织战地工作巡视团赴晋冀鲁豫地区帮助部队整顿。同年10月，参加指挥[[关家垴战斗|关家垴战斗]]。1941年，罗瑞卿在辽县桐峪镇与[[郝治平|郝治平]]结婚。同年4月，被任命为华北军委分会委员。1943年，罗瑞卿奉命返回延安。1945年春，罗瑞卿出席[[中国共产党第七次全国代表大会|中国共产党第七次全国代表大会]]，被选为候补中央委员。

=== 第二次国共内战时期 ===
1945年8月18日，中共中央书记处会议决定，组织晋察冀中央局。罗被任命为晋察冀中央局副书记、晋察冀军区副政治委员兼政治部主任，此后赶赴[[张家口|张家口]]，与[[聂荣臻|聂荣臻]]、[[萧克|萧克]]等组织[[绥东战役|绥东战役]]。10月，担任新组建的晋察冀第二野战军政治委员。次月，担任冀热辽中央分局第二书记、第二政治委员兼政治部主任、政治委员<ref name="军区">{{cite book |authors=张明金、刘立勤 |title=中国人民解放军历史上的200个军区 |year=2010|publisher=解放军文艺出版社|ISBN = 978-7-5033-2252-5|pages=420}}</ref>，随即率部赴[[承德|承德]]，指挥[[古北口保卫战|古北口保卫战]]<ref>{{cite book |author=北京军区《华北野战部队战史》编写组 |title=中国人民解放军华北野战部队战史|year=2011|publisher=解放军出版社 |page = 54 |ISBN = 978-7-5065-6162-4}}</ref>。1946年1月，国共双方达成停止军事冲突的协议，并组成由国共美三方军事调处执行部，罗瑞卿被任命为中共方面参谋长<ref name="bio2" />。

同年秋，第二次国共内战爆发，国民党部队进攻中原解放区。同年7月，罗瑞卿协助聂荣臻进攻[[山西|山西]][[大同市|大同]]，但是并未攻下<ref>{{cite book|author=洪兵, 汪徐和 |title = 中国雄师: 第一野战军名将谱, 雄师录, 征战记 |year=2006 |publisher=中共党史出版社 |ISBN = 9787801993267 |pages=88-89}}</ref><ref name="zzx">{{cite book |author=张宗逊 |title=张宗逊回忆录|year=1990年|publisher=解放军出版社|location=北京市西城区地安门西大街40号 |ISBN = 7-5065-0673-4 |pages=300-303}}</ref>。1947年，参预指挥[[保南战役|保南战役]]，其次组织晋察冀军区部队整训与补充。6月2日，罗瑞卿任晋察冀野战军第一政治委员，[[杨得志|杨得志]]任司令员。随后参与指挥[[青沧战役|青沧战役]]，攻克[[青县|青县]]、[[沧县|沧县]]、[[永清|永清]]。随后发起保北战役，攻克[[徐水|徐水]]、[[固城|固城]]。9月，率部发动大清河北战役，重创国军第16军。11月，与杨得志指挥部队发起[[石家庄战役|石家庄战役]]，攻占[[石家庄|石家庄]]<ref>{{cite book |author=《中国人民解放军军史》编写组 |title=中国人民解放军军史：第三卷|year=2010 |publisher=军事科学出版社 |page = 141 |ISBN = 978-7-80237-381-5}}</ref>。1948年1月，与杨得志指挥晋察冀野战军发起[[涞水战役|涞水战役]]，歼灭傅作义部32师，35军军长[[鲁英麟|鲁英麟]]兵败自杀<ref name="郭廷以">{{cite book |author =[[郭廷以|郭廷以]]|title =近代中國史綱| year = 2012| pages=517| publisher = 上海人民出版社、格致出版社|isbn =978-7-5432-2049-2}}</ref>。1948年5月，罗瑞卿被任命为华北军区政治部主任兼第二兵团第一政治委员。此后，率领其部连克[[古北口|古北口]]、[[昌黎|昌黎]]、[[石门镇|石门镇]]、[[涞水|涞水]]等地，歼灭国军三万余人。12月，第二兵团参加[[平津战役|平津战役]]，再出平绥线，全歼国军第35军<ref name="bio" />。

1949年1月，率第二兵团进至[[北平|北平]]北部，进行围城。此后部队整编，其改任华北军区政治部主任兼第十九兵团政治委员<ref name="bio" />。同年3月，出席在[[西柏坡|西柏坡]]召开的[[中共七届二中全会|中共七届二中全会]]。之后，任太原前线司令部副政治委员和总前委第一副书记及太原军管会副主任，协助[[徐向前|徐向前]]组织[[太原战役|太原战役]]<ref>{{cite book|author=山西省史志硏究院 |title=山西通志 军事志 |url = http://books.google.com/books?id=2IHmAAAAMAAJ |year=1997 |publisher=中华书局|pages=88–90}}</ref>。4月下旬，协助[[周士第|周士第]]指挥最后攻城，并占领[[太原|太原]]。6月，离开太原调往北平，出任中央军委公安部部长。并于9月，出席[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]<ref name="bio" />。

===中华人民共和国成立后===
{{Main|罗瑞卿事件}}
[[File:MaoZedong_and_LuoRuiqing.jpg|thumb]]在雪中视察。]]

1949年10月1日，[[中华人民共和国|中华人民共和国]]成立，罗瑞卿出席[[中华人民共和国开国大典|中华人民共和国开国大典]]。随后，其主持召开第一次全国公安工作会议，讨论公安机关的组织工作及方针政策。之后，被任命为[[中央人民政府政务院|中央人民政府政务院]]政务委员、[[中央人民政府人民革命军事委员会|中央人民政府人民革命军事委员会]]委员、[[政务院政治法律委员会|政务院政治法律委员会]]副主任、政法办公室主任、[[中央人民政府公安部|中央人民政府公安部]]部长；次月，兼任[[北京市公安局|北京市公安局]]局长。1951年7月，罗瑞卿以公安部长名义公布《[[城市户口管理暂行条例|城市户口管理暂行条例]]》，并在该年起展开全国性的[[三反|三反]]、[[五反|五反]]运动<ref>{{cite web | language = en | publisher = TIME | title = CHINA: High Tide of Terror | url = http://www.time.com/time/magazine/article/0,9171,808241-1,00.html | author =  | date = 1956-05-05 | accessdate = 2010-06-15 }}</ref>。1953年，兼任[[中央人民公安学院|中央人民公安学院]]院长。1955年，被授予[[中国人民解放军大将|中国人民解放军大将]]军衔及一级[[八一勋章_(1955年)|八一勋章]]、一级[[独立自由勋章|独立自由勋章]]和一级[[解放勋章|解放勋章]]<ref name="bio2" />。

1956年，主持召开第一次全国公安政治工作会议等，主持[[肃反|肃反]]及经济保卫工作；次年，起草《[[中华人民共和国治安管理处罚条例|中华人民共和国治安管理处罚条例]]》、户口登记条例等法律法规等。1959年以后，任[[国务院副总理|国务院副总理]]、[[国防部|国防部]]副部长、[[中央军委|中央军委]]秘书长、[[中国人民解放军总参谋部|中国人民解放军总参谋长]]、国防工业办公室主任、[[中央军委|中央军委]]常委、[[中央书记处|中央书记处]]书记<ref name="bio2" />。1961年，与[[贺龙|贺龙]]、[[聂荣臻|聂荣臻]]主持召开[[国防科工委|国防科工委]]北戴河工作会议，研究贯彻“[[调整、巩固、充实、提高|调整、巩固、充实、提高]]”的八字方针，缩短国防工业生产建设战线问题，对国防工业的科研、生产、基本建设进行调整<ref name="bio" />。

{{Wikisource|关于罗瑞卿错误问题的报告}}
[[File:1965-11_1965_罗瑞卿.jpg|thumb]]
1965年12月11日，罗瑞卿由昆明临时调往上海，参加中共中央政治局常委扩大会议。在会上，[[林彪|林彪]]、[[叶群|叶群]]、[[吴法宪|吴法宪]]、[[李作鹏|李作鹏]]等人在毛泽东首肯下<ref name="文革Ch1">{{Cite book|author=罗平汉|title=文革前夜的中国|location=北京|publisher=人民出版社 |date=2007|ISBN=978-7-50-731605-6|pages=284-291|url=http://history.sina.com.cn/bk/ds/2014-01-03/141659427.shtml}}</ref>，对罗瑞卿进行突然袭击，被指责为“反对突出政治”、“反对林彪”。随后，被调离军事工作领导岗位<ref>{{cite web |language = zh-hans | publisher = [[炎黄春秋|炎黄春秋]] | title = 杨成武谈揭批罗瑞卿实情 | url = http://www.yhcqw.com/html/qsp/2008/417/08417102917C8HJH1499AG1497JIJJEI84I.html | author = 陈虹 |issue = 2005年第10期 | accessdate = 2010-06-23 }}</ref>。1966年3月，[[1966年京西宾馆会议|京西宾馆会议]]上，罗瑞卿被挂上“篡军反党”、“反对毛主席”、“反对毛泽东思想”而受到批判。3月18日，罗瑞卿从三楼跳下导致双脚跟骨受伤，被送入[[北京医院|北京医院]]治疗<ref name="文革时罗瑞卿因何从三楼跳下要以死明志 " />。同年5月，中共中央政治局扩大会议召开，[[刘少奇|刘少奇]]主持批判“[[彭真|彭真]]、罗瑞卿、[[陆定一|陆定一]]、[[杨尚昆|杨尚昆]]的‘反党错误’”，并决定停止和撤销该四人全部职务<ref>{{cite web |url =http://www.singtaonet.com:82/global/wg/wg10/t20060515_213078.html |title =劉少奇批彭羅陸楊（1966） |date =2006-05-15 |accessdate =2015-05-28 |language =zh-hant |publisher =星岛国际 |deadurl =yes |archiveurl =https://web.archive.org/web/20121026023254/http://www.singtaonet.com:82/global/wg/wg10/t20060515_213078.html |archivedate =2012-10-26 }}</ref>。罗瑞卿因此被停止中央书记处书记、国务院副总理等职务。同年10月，罗瑞卿逐渐康复；但12月，被[[红卫兵|红卫兵]]劫出医院，押到[[海淀区|海淀区]][[罗道庄|罗道庄]]的[[北京卫戍区|北京卫戍区]]部队驻地。1967年4月3日，因伤口长期未愈，被送入[[解放军总医院|解放军总医院]]治疗，治疗期间经常被提讯或批斗。1969年初，做左小腿截肢和左[[股骨头|股骨头]]切除手术。1973年1月，因[[冠心病|冠心病]]、[[高血压|高血压]]等病住院治疗。之后复发再次住院。同年11月，被解除监护。12月21日，毛泽东在参加中央军委会议时，表示其是“听了林彪一面之辞，错整了罗瑞卿”<ref>{{cite book |language = zh-hans | author = 胡华 | coauthors = 中共党史人物研究会 |title = 中共党史人物传 |url =  |year = 1991年 |location = |publisher = 陕西人民出版社 |ISBN = 9787224015782 |page = 67 }}</ref>。

{{Wikisource|中共中央关于为罗瑞卿同志平反的通知}}

“[[文化大革命|文化大革命]]”结束后，政治上得到[[平反冤假错案|平反]]。1977年，参加[[中共第十一次全国代表大会|中共第十一次全国代表大会]]，当选为中央委员，并被任命为中央军委常委、军委秘书长。1978年，在《[[实践是检验真理的唯一标准|实践是检验真理的唯一标准]]》发表后，促成《马克思主义的一个最基本的原则》在《[[解放军报|解放军报]]》上发表，推动了“[[真理标准大讨论|真理标准大讨论]]”的开展<ref>{{cite book |author=叶永烈 |title = 邓小平改变中国 |year=2014 |publisher=四川人民出版社、华夏出版社 |location= |pages=338-344 |ISBN = 978-7-220-08819-3}}</ref>。7月赴[[西德|西德]][[海德堡大学|海德堡大学]]骨科医院治疗腿病，于8月3日突发心肌梗塞病逝。两日后，中央派专机将罗瑞卿的灵柩接回北京，[[邓小平|邓小平]]、[[叶剑英|叶剑英]]等前往机场迎灵。1978年8月12日，中共中央召开[[追悼会|追悼会]]，[[天安门广场|天安门广场]]、[[新华门|新华门]]、[[中华人民共和国外交部|外交部]]下半旗致哀，[[邓小平|邓小平]]致悼词<ref>{{cite web |language = zh-hans |publisher = 华夏经纬网（转自《文史博览》） |title = 罗瑞卿夫人郝治平：他耿直、拼命、秉性难移 |url = http://www.huaxia.com/zt/js/07-048/542923.html |author = 吴东峰 |date = 2007-06-20 | accessdate = 2010-06-22 }}</ref><ref>{{cite web |language = zh-hans |publisher = 粤成一百门户网（转载自《往事亲历》） |title = 革命40年未入党：哪位大将文革被揭是“假党员” |url = http://www.yc100.com/news.asp?action=newsinfo&id=688955&page=6 |author = 董雅丽 |date = 2011-12-14 |accessdate = 2011-12-17 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。罗瑞卿的骨灰安放在[[北京市八宝山革命公墓|北京市八宝山革命公墓]]。

== 评价 ==
[[党和国家最高领导人|党和国家最高领导人]][[毛泽东|毛泽东]]评价罗瑞卿时说：“天塌下来有罗长子顶着”<ref name="文革时罗瑞卿因何从三楼跳下要以死明志 ">{{cite web |url = http://history.huanqiu.com/people/2012-01/2329316_2.html |title = 《文革时罗瑞卿因何从三楼跳下要以死明志》 |accessdate = 2012-01-05 |publisher = 环球时报 |language = zh-cn |deadurl = yes |archiveurl = https://web.archive.org/web/20120114022806/http://history.huanqiu.com/people/2012-01/2329316_2.html |archivedate = 2012年1月14日 |df =  }}</ref>。

1967年，[[叶剑英|叶剑英]]就罗瑞卿的跳楼作了一首诗，其中“将军一跳身名裂，故人回首成永诀”套改自[[南宋|南宋]]爱国词人[[辛弃疾|辛弃疾]]《贺新郎·别茂嘉十二弟》追叹[[西汉|西汉]]名将[[李陵|李陵]]的名句“將軍百戰身名裂。向河梁、回头万里，故人长绝。”；对此，周恩来严厉批判其“完全站到罗瑞卿一边”<ref>{{cite web |url = http://www.takungpao.com/history/content/2012-05/28/content_293830_2.htm |title = 1967年周恩来批叶剑英：大闹怀仁堂竟拍十多次桌 |publisher = [[南方都市报|南方都市报]] |website = [[大公网|大公网]] |date = 2012-05-28 |deadurl = yes |archiveurl = https://web.archive.org/web/20160916073552/http://www.takungpao.com/history/content/2012-05/28/content_293830_2.htm |archivedate = 2016-09-16 }}</ref>。
