'''罗子国城遗址'''位于[[湖南省|湖南省]][[汨罗市|汨罗市]][[城关镇|城关镇]]西北的罗城，现属于[[屈原管理区|屈原管理区]][[河市镇|河市镇]]古罗城村一组，为[[汨罗江|汨罗江]]下游南岸的一块土洲。1956年成为湖南省级文物保护单位，2013年列入第七批[[全国重点文物保护单位|全国重点文物保护单位]]名单<ref>{{cite web |title=汨罗江畔罗子国城遗址列入全国重点文物保护单位 |url=http://hunan.sina.com.cn/news/m/2013-05-26/140554018.html |accessdate=2018-07-09 |date=2013-05-26}}</ref>。

==布局==
城址东南长约590米，西北宽约400米，整体略呈长方形，南面略内凹，总面积约23.6万平方米。三面[[护城河|护城河]]保存较好，现宽约5-10米，深超过3米。
部分城墙保存较好，墙基宽14米，高3米，用黄土分层夯筑而成<ref>{{cite web |title=岳阳市省级文物保护单位简介——罗子国城遗址 |url=http://wgxj.yueyang.gov.cn/wgxj/8520/8528/8529/content_381925.html |publisher=岳阳市文广新局 |accessdate=2018-07-09 |date=2012-06-29}}</ref><ref>{{cite web |title=全国重点文物保护单位—罗子国城遗址 |url=http://www.yueyang.gov.cn/yywh/whyc/wh/content_482268.html |accessdate=2018-07-09 |date=2014-12-24}}</ref> 。

==历史==
[[罗子国|罗子国]]是[[商|商]]代中叶芈部落的一个分枝，[[春秋|春秋]]时为[[楚国|楚国]]所灭，其遗民迁于[[湖北|湖北]][[枝江|枝江]]，后又迁至湖南汨罗附近。罗国子民迁此筑城<ref>{{cite web |author1=喻立新 |title=长沙先秦历史初探 |url=http://szb.changsha.gov.cn/csfangzhi/fzqikan/fzqk2013_/fzqk2013_2/201501/t20150129_691613.html |accessdate=2018-07-09 |date=2015-01-29}}</ref>。
