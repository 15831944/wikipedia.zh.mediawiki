''' 罗石贤 '''{{BD|1940年|12月}}，[[湖南|湖南]][[长沙|长沙]][[宁乡|宁乡]]人，中国[[作家|作家]]。<ref name="罗石贤">{{cite web |url=http://www.chinawriter.com.cn/zxhy/member/3540.shtml|title=《罗石贤》|accessdate=|work=|publisher=中国作家网|language=zh-hans}}</ref>

== 简介 ==
1958年，罗石贤毕业于[[湘潭师范学院|湘潭师范学院]]，<ref name="罗石贤"/> 毕业后在宁乡[[云山|云山]]师范教书。

1972年，罗石贤开始发表文学作品。

1979年，罗石贤加入[[中国作协|中国作协]]。<ref name="罗石贤"/>

== 履历 ==

*《洞庭文艺》主编
*岳阳地区文联副主席
*《[[洞庭湖|洞庭湖]]》文学主编
*[[岳阳市|岳阳市]]文联主席
*[[湖南省文联|湖南省文联]]第三、四、五届委员
*湖南省作协第三、四、五届理事
