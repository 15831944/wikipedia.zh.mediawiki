'''罗旭'''（{{bd|1934年|12月9日|2001年|11月20日}}）女，祖籍[[福建|福建]][[福州|福州]]，生于[[上海|上海]]，[[中华人民共和国外交部|中华人民共和国外交部]]官员、翻译家<ref name=luo>{{Cite web |url =http://www.luos.org/list.asp?Unid=2556  |title =外交世家——福州罗氏新梅公系  |author =  |date =  |publisher =罗氏家园  |language =  |accessdate =2017-09-14  |quote =  }}</ref>。

==生平==
罗旭是福州罗氏新梅公支系第九世孙女。叔曾祖是晚清大臣[[罗丰禄|罗丰禄]]。祖父[[罗忠尧|罗忠尧]]曾任[[清朝|清朝]]驻[[新加坡|新加坡]]总领事。父亲[[罗孝章|罗孝章]]留学[[美国|美国]]获硕士学位，任[[美国电报电话公司|美国电报电话公司]]驻上海办事处总工程师<ref name=luo/>。

1958年，罗旭自[[北京大学|北京大学]]西语系[[法语|法语]]专业毕业，被分配到[[中华人民共和国外交部|中华人民共和国外交部]]工作。在翻译室工作的最初几年，罗旭曾先后充任[[毛泽东|毛泽东]]、[[周恩来|周恩来]]、[[宋庆龄|宋庆龄]]、[[陈毅|陈毅]]等中央领导人会见法语外宾时的译员。1963年至1964年，罗旭被借调到[[中共中央编译局|中共中央编译局]]，参加《关于国际共产主义运动总路线的论战》等文章的翻译工作<ref name=luo/>。

1960年代末至1970年代初，[[王海容|王海容]]、[[唐闻生|唐闻生]]、[[齐宗华|齐宗华]]、罗旭、[[章含之|章含之]]被称为中国外交界的“五朵金花”<ref name=zgqnw>{{Cite web |url = http://agzy.youth.cn/qsnag/sdxw0/201208/t20120822_2376011.htm |title =  中国外交界的“五朵金花”|author =  |date = 2012-08-22 |publisher =中国青年网  |language =  |accessdate =  |quote =  }}</ref>。罗旭是法语专业毕业，又从小学过[[英语|英语]]，因此可进行法语、英语的双语口译。1972年5月[[索马里|索马里]]最高革命委员会主席[[穆罕默德·西亚德·巴雷|西亚德]]访华，正逢[[非洲|非洲]]国家驻华使团在[[北京饭店|北京饭店]]举办活动庆祝“[[非洲日|非洲日]]”，周恩来与西亚德应邀参加，罗旭任译员。西亚德不懂法语，非洲驻华使团长、[[几内亚|几内亚]]大使不懂英语，罗旭从中翻译，获得了非洲使团礼宾官的称赞，周恩来也表示满意。1972年，一次罗旭和[[王海容|王海容]]、[[唐闻生|唐闻生]]到毛泽东处参加接待某非洲国家元首。会见外宾前，毛泽东笑着说：“你们这些人的名字好厉害，唐闻生是闻生不闻死，罗旭和[[后羿|后羿]]一样会射下九个太阳。”据说这指[[江青|江青]]1971年冬在[[钓鱼台国宾馆|钓鱼台国宾馆]]宴请几位译员，罗旭认为这和工作无关而未去。后来，周恩来曾称赞罗旭是“很有个性的人”<ref name=luo/>。

1972年到1975年间，罗旭多次出席重大国际会议。1972年秋，罗旭随中华人民共和国外交部副部长[[乔冠华|乔冠华]]出席[[联合国大会|联合国大会]]第二十七届会议。1973年春，罗旭随中华人民共和国外交部部长[[姬鹏飞|姬鹏飞]]到[[法国|法国]][[巴黎|巴黎]]参加关于[[越南|越南]]问题的国际会议，这次会议最终促成了[[越南战争|越南战争]]的结束。1974年春，罗旭作为副代表，随[[邓小平|邓小平]]参加联合国大会第六届特别会议。1975年秋，罗旭作为副代表出席联合国大会第三十届会议<ref name=luo/>。

1975年后，罗旭从事法文、英文的翻译工作，先后参与了《[[戴高乐|戴高乐]]回忆录》、《[[巴尔扎克|巴尔扎克]]全集》等名著的翻译校订，并且翻译了不少有关中国经济改革的理论文章。1985年，罗旭调到外交部新闻司工作，以[[参赞|参赞]]衔主管新闻发布工作。1988年，罗旭到[[纽约|纽约]][[联合国|联合国]]总部任[[中国常驻联合国代表团|中国常驻联合国代表团]]经济和新闻[[参赞|参赞]]，1992年冬卸任回国，任外交部西欧司参赞<ref name=luo/>。

罗旭喜爱法语文学，特别是[[乔治·桑|乔治·桑]]和[[司汤达|司汤达]]的作品《[[红与黑|红与黑]]》、《巴马修道院》等。罗旭还很喜欢[[贝多芬|贝多芬]]、[[莫扎特|莫扎特]]等人的[[古典音乐|古典音乐]]作品<ref name=luo/>。

罗旭的丈夫[[赵稷华|赵稷华]]历任外交部美大司副处长，[[中国常驻联合国代表团|中国常驻联合国代表团]]{{Link-de|公使銜參贊|Gesandter-Botschaftsrat}}，外交部港澳事务办公室主任。在中英关于香港问题的谈判中，赵稷华任[[中英联合联络小组|中英联合联络小组]]中方首席代表。[[香港回归|香港回归]]后，赵稷华任[[外交部驻香港特派员公署|外交部驻香港特派员公署]]副特派员。当时罗旭在[[香港|香港]]任[[联合出版集团|联合出版集团]]高级研究员<ref name=luo/>。

2001年11月20日，罗旭病逝，享年68岁。遗体捐给[[北京|北京]][[协和医科大学|协和医科大学]]供科研教学用<ref name=luo/>。
