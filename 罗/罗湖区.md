'''罗湖区'''是[[中国|中国]][[广东省|广东省]][[深圳市|深圳市]]下辖的一个[[市辖区|市辖区]]，是[[深圳经济特区|深圳经济特区]]内的四区之一。南面隔[[深圳河|深圳河]]与[[香港|香港]]的[[新界|新界]][[北區|北區]]相望，其他三面与[[福田区|福田区]]、[[龙华区_(深圳市)|龙华区]]、[[龙岗区|龙岗区]]、[[盐田区|盐田区]]相邻。往香港的一条重要的鐵路过境通道[[罗湖口岸|罗湖口岸]]、陸路過境通道[[文錦渡口岸|文錦渡口岸]]及未來的東部[[蓮塘/香園圍口岸|蓮塘/香園圍口岸]]均位於本区。

== 地理 ==
罗湖区属于[[丘陵|丘陵]]地带，被较多低矮的山坡包围。现在罗湖区内田贝、太安、布心、东湖等地有几条路的因丘陵影响道路的两侧高度有明显不同，其实有一些小山在建市之初为了建设需要挖平了。有指罗湖一名并非指区内有一名叫“罗湖”的湖泊，而是古南越话中，“罗湖”指低矮的小山，意指罗湖区附近一带较多低矮的丘陵地貌。深圳市最高的山峰[[梧桐山|梧桐山]]（944米）位於羅湖區。羅湖區內有東湖、洪湖、仙湖等湖泊，現在周圍都設了[[公園|公園]]，供人遊覽。[[深圳水庫|深圳水庫]]是供應深港兩地用水的重要水庫。羅湖區面積78.75平方公里，設10個[[街道辦事處|街道辦事處]]。西側以紅嶺路與[[福田區|福田區]]分界，東側以梧桐山與[[鹽田區|鹽田區]]交界，南侧以[[深圳河|深圳河]]为界与香港[[北區_(香港)|北区]]隔河相望，北側界限為[[龙岗区|龙岗区]]邊界。

== 历史 ==
罗湖区旧称[[宝安县|宝安县]]“[[深圳镇|深圳镇]]”，得名于現今東門一帶旧称“[[深圳圩|深圳圩]]”的范围。深圳镇大约管辖现今罗湖区范围内。[[廣九鐵路|廣九鐵路]]開通後，於此地設立深圳站。1953年，寶安縣政府考慮到深圳墟連接廣九鐵路，交通便利，便將寶安縣城從[[南頭|南頭]]搬到深圳墟，逐漸成為彼鄰[[香港|香港]]的商業中心。1979年3月[[改革开放|改革开放]]，寶安縣改制取名[[深圳市|深圳市]]，而深圳镇于1979年10月改制成立罗湖区，取其附近[[罗湖口岸|罗湖口岸]]附近的罗湖村的“罗湖”为名，管辖当时深圳镇、沙头角镇、梅沙镇、盐田镇等地，是当时深圳最早成立的區。1997年11月，沙头角、盐田和梅沙的镇区從羅湖區分出，建立[[鹽田區|鹽田區]]，確立了現在的罗湖区管轄範圍。

== 人口 ==
根据2010年（11月1日零时）第六次人口普查结果，罗湖区常住人口923,423人，其中男性465,533人，女性人口为457,890人，男女性别比例为102比100。按年龄段分类，0—14岁人口为109,078人，占11.81%；15—64岁人口为786,800人，占85.21%；65岁及以上人口为27545人，占2.98%。按民族划分，汉族人口907,754人，占98.30%；各少数民族人口156,69人，占1.70%。按受教育程度划分，具有大学程度的人口为212,323人，占22.99%；具有高中（含中专）程度的人口为267,142人，占28.93%；具有初中程度的人口为291,676人，占31.59%；具有小学程度的人口为93,407人，占10.12%。文盲人口（15岁及以上不识字的人）为5,258人，文盲率0.57%。这次普查，接受普查登记的境外人员有33,227人，以香港居民为主，达到29,830人，占境外人员的89.78%<ref name="第六次人口普查数据">2010年第六次人口普查数据来源于 [http://www.szlh.gov.cn/tjj/a/2011/h02/a176392_572158.html 《深圳市罗湖区2010年第六次全国人口普查主要数据公报》] {{webarchive|url=https://web.archive.org/web/20140201164100/http://www.szlh.gov.cn/tjj/a/2011/h02/a176392_572158.html |date=2014-02-01 }}</ref>。

=== 住宅区 ===
罗湖区是深圳市最早开发的区，住宅区的分布没有统一的规划。区内各个年代建成的楼群互相交错，这是因为新开发的小区往往都不顾及周围的环境。1980年代的住宅楼一般只有4-6层高，有一点商业功能就可以叫“大厦”了；而随着土地的逐渐被开发，近几年建成的小区一般都会在15-30层，而且无一例外的，住宅楼的低层作为商铺出租。2003年，罗湖区开始解决城市发展过程中的一些遗留问题，进行了改造[[城中村|城中村]]、全面接管二线关插花地等行动。罗湖区的大型住宅区有[[錦上花家園|錦上花家園]]、[[百仕达花园|百仕达花园]]等。

=== 商业区 ===
罗湖区有三个主要的商圈：东门商圈、人民南商圈、地王商圈。
==== 东门商圈 ====
東門一带是原来深圳镇的所在，有大约300年的历史，因此这一带又被称为“老街”(地鐵取其名)。长期以来，东门一直就是深圳（及其前身[[宝安县|宝安县]]）最主要的商业区之一。1999年，商业区经过复古风格的改造之后，定为步行街。中心广场有铜浮雕“老东门墟市图”，反映[[清朝|清朝]]末年至[[民國|民國]]初期的繁華。东门商圈主要销售中低档商品，有各具特色的小店铺云集。市内几家大型连锁百货商店（[[茂业|茂业]]、[[天虹|天虹]]等）也进驻此地；但这里有不少饭店(酒店)，吃喝購物的地方也可不少，尤其集中在[[東門商業區|東門商業步行街]]。[[深圳地铁|深圳地铁]]的[[老街站|老街站]]亦在這裡。

==== 人民南商圈 ====
因地理上以[[人民南路|人民南路]]為中轴线而得名。又因创造了“[[深圳速度|深圳速度]]”的深圳[[国贸大厦|国贸大厦]]在此范围内，所以又叫“国贸商圈”。处于罗湖中心地带，鄰近[[羅湖火车站|羅湖火车站]]以北，及[[東門商業區|東門商業區]]以南。[[深圳地铁|深圳地铁]]的[[國貿站|國貿站]]亦在此商圈內。由于临近[[罗湖|罗湖]]口岸，很多到深的香港人在此消费。这里消费水平偏高，高级商场[[金光華廣場|金光華廣場]]和大饭店均建於此。比较著名的五星级酒店有[[彭年酒店|彭年酒店]]和[[阳光酒店|阳光酒店]]，二者只有一路相隔。 另外還有已經在[[深圳证券交易所|深圳證券交易所]]上市的[[新都酒店|新都酒店]]。2006年，深圳市对人民南商圈进行整治，美化了街景和灯光，设立了一处文化广场，并建起一条从南到北贯穿商圈、直达东门步行街的二层商业连廊，将沿线的各个商城连接起来。這裡也有多家百貨公司如[[天虹百貨|天虹百貨]]、[[茂業百貨|茂業百貨]]、[[吉之島|吉之島]]超市等。

==== 地王商圈 ====
此商圈的主要部分是[[地王大厦|地王大厦]]（信兴广场），此楼高70层，384米，曾是深圳第一高楼。到了2011年，[[京基金融中心|京基金融中心]]落成後，以441.28米成為深圳的最高建筑（現被[[平安國際金融中心|平安國際金融中心]]取代）。該處亦設多個大型商場，包括2004年落成的大型购物中心[[深圳万象城|深圳万象城]]及2010年落成的[[KK_Mall|KK Mall]]。

实际上现在随着深圳市中心的西移，这个商圈的影响力已开始减弱。但因为[[深圳书城|深圳书城]]和[[深圳大剧院|深圳大剧院]]就坐落在地王大厦附近，来往的人流没有减少，但其作為商圈的功能則開始漸漸轉變。深圳地铁[[大劇院站_(深圳市)|大劇院站]]亦在本商圈。此外，[[深圳证券交易所|深圳证券交易所]]和[[深圳发展银行|深圳发展银行]]总部也在此商圈内。

=== 工业区 ===
罗湖的工业区主要集中在远离中心区的边界地带，以制造业为主，模式与周围其它地区的民营工厂并无差别。既远离商业区又靠近铁道的地方会建起物流园区（就是仓库区），如以笋岗站（旧称深圳北站）为依托建立起的笋岗-清水河物流园区。随着城市的扩张，这些工业区逐渐被住宅区所包围。
