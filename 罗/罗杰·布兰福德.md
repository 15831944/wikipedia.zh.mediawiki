'''罗杰·戴维·布兰福德'''，[[皇家学会|FRS]] [[英國皇家天文學會|FRAS]]（{{lang-en|'''Roger David Blandford'''}}，{{bd|1949年|8月28日}}），[[英国|英国]]理论[[天体物理学|天体物理学家]]，因其对[[黑洞|黑洞]]的研究而知名。

== 早年生活 ==
布兰福德出生于英国的[[格兰瑟姆|格兰瑟姆]]，成长于[[伯明翰|伯明翰]]。

== 研究生涯 ==
布兰福德是著名的天体物理学家，他提出了[[布蘭德福–日納傑過程|布兰福德-日纳杰过程]]，一个从[[黑洞|黑洞]]中提取能量的模型。2005年，他给天文学社区写了一封公开信，表达了他对乔治·W·布什政府的[[空间科学|空间科学]]政策的关注。<ref>[http://www.physicstoday.org/vol-58/iss-4/p10.html Exploring the Universe - Physics Today April 2005] {{webarchive|url=https://web.archive.org/web/20070417152053/http://www.physicstoday.org/vol-58/iss-4/p10.html |date=2007-04-17 }}</ref>

他是Astro2010的主席，Astro10旨在通过十年的调研明确和建议下一个十年中[[美国|美国]][[天文学|天文学]]研究基金的拨款优先级。Astro2010报告于2010年8月13日发布。<ref>{{Cite book|title=New Worlds, New Horizons in Astronomy and Astrophysics|publisher=[[National_Academies_Press|National Academies Press]]|year=2010|isbn=978-0-309-15799-5|location=Washington, DC|doi=10.17226/12951}}</ref>

== 职位 ==
布兰福德是[[皇家学会会士|皇家学会会士]]、美国[[美国国家科学院|国家科学院]]会员、[[英國皇家天文學會|皇家天文学会会员]]和[[美国文理科学院|美国文理科学院]]会员。<ref>[http://www.slac.stanford.edu/slac/faculty/hepfaculty/blandford.html SLAC HEP Faculty: Roger Blandford] {{webarchive|url=https://web.archive.org/web/20070711061942/http://www.slac.stanford.edu/slac/faculty/hepfaculty/blandford.html |date=2007-07-11 }}</ref> 目前，他是[[史丹佛大學|斯坦福大学]]人文与科学学院的卢克·布洛瑟姆教授，斯坦福大学和[[SLAC國家加速器實驗室|斯坦福线性加速器中心]]（SLAC）[[SLAC國家加速器實驗室|国家加速实验室]]的物理学教授<ref name="stanford_faculty_blandford">{{cite web |url=http://physics.stanford.edu/people/faculty/roger-blandford |title=存档副本 |accessdate=2013-08-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20130616001641/https://physics.stanford.edu/people/faculty/roger-blandford |archivedate=2013-06-16 }}</ref>。他在2003年到2013年间担任科维理粒子天体物理和宇宙学研究所的Pehong 和 Adele Chen主任。<ref>{{cite web |url=http://kipac.stanford.edu/kipac/Org-Chart |title=存档副本 |accessdate=2013-08-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20130927185418/http://kipac.stanford.edu/kipac/Org-Chart |archivedate=2013-09-27 }}</ref>
