'''罗马尼亚社会主义共和国'''是[[第二次世界大战|二战]]后在[[罗马尼亚|罗马尼亚]]建立的社会主义政权，[[罗马尼亚共产党|罗马尼亚共产党]]为[[执政党|执政党]]。在1947至1965年间的正式名称为'''罗马尼亚人民共和国'''（{{lang-ro|Republica Populară Romînă}}，RPR），1965至1989年的正式名称为'''罗马尼亚社会主义共和国'''（{{lang-ro|Republica Socialistă România}}，RSR）。

==历史==
1944年8月23日，[[罗马尼亚共产党|罗马尼亚共产党]]领导武装起义，推翻[[安东尼斯库|安东尼斯库]]的亲[[纳粹德國|纳粹德國]]政府并加入[[盟军|盟军]]一方。战后，[[苏联|苏联]]成为了罗马尼亚盟军的唯一代表，并在1945年3月6日协助成立了一个以[[彼得鲁·格罗查|格罗查]]總理为首的亲苏的新政府，成立初期新政府成员中也包括一些曾在战前取缔共产党的领导人。在若干次共产党人的示威后，共产党人渐渐掌握了政府的控制权。1947年罗马尼亚国王[[米哈伊一世|米哈伊一世]]迫于压力退位并流亡海外，随后[[罗马尼亚共产党中央委员会总书记|罗马尼亚共产党中央委员会总书记]][[格奧爾基·喬治烏-德治|格奧爾基·喬治烏-德治]]宣布成立罗马尼亚人民共和国。

20世纪50年代罗马尼亚政府开始寻求摆脱[[苏联|苏联]]的控制。1958年罗马尼亚说服[[苏联政府|苏联政府]]撤走了所有[[苏联军队|苏联军队]]。1965年[[尼古拉·齐奥塞斯库|-{zh-hans:尼古拉·齐奥塞斯库; zh-hant:尼古拉·西奧塞古; zh-hk:尼古拉·壽西斯古; zh-tw:尼古拉·希奧塞古;}-]] 成为了[[罗马尼亚共产党中央委员会总书记|罗马尼亚共产党中央委员会总书记]]并在1967年兼任了[[罗马尼亚总统|国家元首]]，并通过新宪法改国名为罗马尼亚社会主义共和国。1968年[[布拉格之春|苏联入侵捷克斯洛伐克]]时，齐奥塞斯库对苏联的谴责使他在国内和西方获得了一些认可。在1974年他成为了新设的罗马尼亚总统。通过外债拉动的经济增长和大量的政治压迫，最终导致1989年12月爆发了[[1989年羅馬尼亞革命|1989年羅馬尼亞革命]]，人民推翻了齐奥塞斯库政权，齐奥塞斯库与其夫人被枪决，罗马尼亚共产党和其[[共產主義政權|共產主義政權]]亦在齐奥塞斯库夫婦處決後即时消亡。羅馬尼亞救國陣線政府其後实现了[[民主化|民主化]]、[[多党制|多党制]]及有[[公民提名|公民提名]]的选举，同时新的[[罗马尼亚国徽|罗马尼亚国徽]]于1992年啟用，取代了[[共产主义|共产主义]]的红星标誌。自此罗马尼亚已逐渐擺脫共产主义时期的[[独裁|独裁]]、经济落後及不民主的社会。

== 媒体 ==
===电视===
自1985年起罗马尼亚的电视频道被减少到两个，并且大多数节目为对党总书记[[尼古拉·齐奥塞斯库|-{zh-hans:尼古拉·齐奥塞斯库; zh-hant:尼古拉·西奧塞古; zh-hk:尼古拉·壽西斯古; zh-tw:尼古拉·希奧塞古;}-]] 和其夫人[[埃列娜·齐奥塞斯库|埃列娜·齐奥塞斯库]]的[[个人崇拜|个人崇拜]]赞美。<ref>{{cite web |url=http://www.realitatea.net/dezbatere-realitatea-net-era-sau-nu-mai-bine-pe-vremea-comunismului_894651.html |title=存档副本 |accessdate=2012-07-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20120425172248/http://www.realitatea.net/dezbatere-realitatea-net-era-sau-nu-mai-bine-pe-vremea-comunismului_894651.html |archivedate=2012-04-25 }}</ref><ref>http://www.evz.ro/detalii/stiri/prima-transmisie-color-din-romania-fara-stirea-ceausestilor-video-933453.html</ref>在这种情况下大多数罗马尼亚人安装电视天线并收看[[保加利亚|保加利亚]]的电视节目。<ref>{{Cite web |url=http://www.romlit.ro/revista_revistelor16 |title=存档副本 |accessdate=2012-07-06 |archiveurl=https://web.archive.org/web/20150415075110/http://www.romlit.ro/revista_revistelor16 |archivedate=2015-04-15 |deadurl=yes }}</ref>有一些人也尝试接受[[南斯拉夫社会主义联邦共和国|南斯拉夫社会主义联邦共和国]]的电视信号但通常需要较好的天线并接近边境。在罗马尼亚拥有彩色电视的人很少，在1981年罗马尼亚成为了在欧洲唯一使用黑白电视讯号的国家（1981年[[阿尔巴尼亚|阿尔巴尼亚]]停止黑白讯号发射）。在1983年8月23日罗马尼亚开始了第一次彩色电视讯号发射，与1984年的20%相比1989年彩色节目所占比例上升到80%。<ref>http://www.evz.ro/detalii/stiri/prima-transmisie-color-din-romania-fara-stirea-ceausestilor-video-933453.html</ref>
