'''罗益昌'''（{{bd|1964年|2月||}}），[[江苏|江苏]][[连云港|连云港]]人，[[中国人民解放军中将|中国人民解放军中将]]。

==生平==
罗益昌长期在[[中国人民解放军空军|中国人民解放军空军]]从事政工工作，曾任[[中国人民解放军空军雷达学院|中国人民解放军空军雷达学院]]政治委员、[[中国人民解放军空降兵第十五军|中国人民解放军空降兵第十五军]]政治部主任等职。2011年7月任[[中国人民解放军沈阳军区空军|中国人民解放军沈阳军区空军]]政治部主任。2015年升任[[中国人民解放军沈阳军区|中国人民解放军沈阳军区]]副政治委员，并任沈阳军区党委常委。2016年初，任沈阳军区善后工作办公室政治委员、党委书记<ref name=zhuren>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1367408 |title =  罗益昌少将升任沈阳军区副政委，曾任沈空政治部主任|author =  |date = 2015-08-23 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.sohu.com/20160203/n436776316.shtml |title = 徐经年任沈阳军区善后办主任 罗益昌任善后办政委 |author =  |date = 2016-02-03 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

2010年7月晋升为空军少将军衔<ref name=zhuren/>。2016年晋升为中将军衔<ref>{{Cite web |url = http://www.81.cn/lj/2016-02/12/content_6904860.htm |title = 某善后办罗益昌政委蹲点过年强调抓教育保安全 |author =  |date = 2016-02-12  |publisher =中国军网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.mod.gov.cn/jmsd/2016-10/25/content_4752992.htm |title = 论突围 |author =   罗益昌|date = 2016-10-25 |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。
