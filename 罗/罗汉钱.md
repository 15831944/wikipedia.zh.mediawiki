'''罗汉钱'''，清朝[[康熙|康熙]]年间所铸[[制钱|制钱]]“[[康熙通宝|康熙通宝]]”的一个版别，外观通常为黄色，面文的“熙”字有其他版别所用的“-{熈}-”字不同，“通”字下方的[[偏旁|偏旁]]为单点“辶”而非其他版别的双点“⻍”。

== 传说 ==
传说为1713年[[康熙帝|康熙帝]]六十大寿时由[[宝泉局|宝泉局]]所铸<ref>{{cite journal |author1=国胜连 |title=从康熙“罗汉钱”到苏联“红军票” |journal=海燕 |date=2017 |issue=5 |pages=154-156}}</ref><ref name="YKLeungKangHsiTungPaoE">{{cite web|url= http://ykleungn.tripod.com/kanghsie.htm|title= K'ang Hsi T'ung Pao ( 1661 ~ 1722 A.D. ) - Part E.|date= 2018|accessdate= 22 September 2018|author= Y.K. Leung|publisher= Tripod.com|language= en|archive-url= https://web.archive.org/web/20180922211830/http://ykleungn.tripod.com/kanghsie.htm|archive-date= 2018-09-22|dead-url= no}}</ref>，时称“万寿钱”，因其特殊的颜色，一开始发行便被认为具有“[[压胜|压胜]]”之力，后收藏界称之为“罗汉钱”。另外也有传说是1718年清军入藏作战，当地喇嘛出金和[[罗汉|罗汉]]数尊熔铸铜钱，以赍军需，故称“罗汉钱”<ref name="刘征图谱">{{cite book |author1=刘征 |title=顺治康熙图谱 |date=2006-05 |location=北京 }}</ref>{{rp|162}}。

传说罗汉钱中含金。一说康熙皇帝在其六十大寿时赐给宝泉局金罗汉一尊，用于铸钱赏赐给功臣，因此所铸钱币中含金；一说康熙五十六年（1717年）[[年羹尧|年羹尧]]在西藏用佛寺所献鎏金的佛像、罗汉像铸钱用于发饷，因而罗汉钱含有少量的金<ref name="传说">{{cite journal |author1=叶春茂 |author2=叶傲霜 |author3=叶向东 |title=罗汉钱的传说(钱币史话之七) |journal=人民论坛 |date=1996 |issue=10 |page=52}}</ref>。曾有人对一枚罗汉钱进行光谱定性分析，测定结果并不含金<ref>{{cite journal |author1=王连洲 |title=罗汉钱不含金 |journal=中国钱币 |date=1985 |issue=1 |page=71}}</ref>。

据余榴梁《罗汉钱探源》中称[[马定祥|马定祥]]曾从[[杭州|杭州]]的朗悟方丈那里获知“罗汉钱”名字的来源。据朗悟称，清朝[[道光|道光]]年间，杭州[[净慈寺|净慈寺]]准备重修，有僧人在一尊[[罗汉|罗汉]]像的佛腔中发现一串“万寿钱”，该寺借机宣传将这些钱佩戴在身上会有罗汉保佑，可延年益寿、驱邪避祸，以此向信徒募款，从而“万寿钱”的名字逐渐被“罗汉钱”取代<ref name="余榴梁1985">{{cite journal |author1=余榴梁 |title=罗汉钱探源 |journal=中国钱币 |date=1985 |issue=1 |pages=70-71}}</ref>。

大卫·哈蒂尔（David Hartill）称，罗汉钱的传说最早出现于1851年前后，并认为其与当时日本流传的{{le|宽永通宝|Kan'ei Tsūhō}}的类似故事有关<ref name="Hartill-2005">{{cite book |author1=David Hartill |title=Cast Chinese Coins |date=2005-09-22 |publisher=Trafford Publishing |location=Trafford |isbn=9781412054669}}</ref>{{rp|286}}<ref>Chinese Coinage Web Site (Charm.ru) - [http://www.charm.ru/library/charm5.htm K'ang Hsi Coin Charm] {{Wayback|url=http://www.charm.ru/library/charm5.htm |date=20151217233635 }} by Vladimir Belyaev, Y.K. Leung, and John O. Dell. Retrieved: 22 September 2018.</ref>。

== 版别 ==
传世的清朝罗汉钱包括头炉钱和后铸钱两类。头炉钱为康熙年间宝泉局铸造，其特征为阔边大样。后铸钱为道光之后，特别是[[同治|同治]]、[[光绪|光绪]]年间民间仿铸，铸造粗陋，铜色不纯<ref name="余榴梁1985" />。
