'''罗贯中'''（{{bd|1320年||1400年|catIdx=L}}）<ref>陈辽，《江苏社会科学》 ，2007年第4期179-182</ref>，名'''本'''，字'''彥直'''，另一字'''貫中'''，[[以字行|以字行]]，号'''湖海散人'''，[[元朝|元]]末[[明朝|明]]初通俗[[小说家|小说家]]，四大名著之一《[[三国演义|三国演义]]》的作者。但他的[[籍贯|籍贯]]在文学界和学术界一直存在争论，一说是东原（今[[山东|山东]][[东平|东平]]）<ref>明刻版“嘉靖本”《三国志通俗演义》序</ref><ref>杜贵晨, 罗贯中籍贯“东原说”辨论,《齐鲁学刊》1995年第5期</ref>，一说是[[太原|太原]]（今[[山西|山西]]太原）<ref>[[賈仲名|賈仲名]]：《[[录鬼簿续编|录鬼簿续编]]》</ref> ，一说是庐陵（江西吉安）<ref>{{Cite book|title = 义务教育课程标准实验教科书《語文》九年级上册.|last = |first = |publisher = 人民教育出版社出版，北京出版集团重印，北京新华书店发行，人民教育出版社印刷厂印刷|year = 2009年3月第1版，2015年6月第7次印刷|isbn = 978-7-107-21515-5|location = |pages = Page 156}}</ref>，一说是[[钱塘|钱塘]]（今[[浙江|浙江]][[杭州|杭州]]）<ref>田汝成（明）《西湖游览志余》卷二十五</ref>，后两者不可确考。近年雖由於在山西省[[祁县|祁县]]河湾村发现了罗贯中之[[家谱|家谱]]，以及个人使用的[[印章|印章]]，但並未說明其人為三國演義的作者<ref>焦琛、郭维忠撰《評楊立仁發現的“羅貫中家譜”》</ref><ref>《中華文化論壇》陳〈三國演義〉作者羅貫中》</ref><ref>北京大学学报：哲社版，杜贵晨《近百年《三国演义》研究学术失范的一个显例——论《录鬼簿续编》“罗贯中”条资料当先悬置或存疑》</ref>。据最新研究，山西太原确有罗贯中，却是另一个杂剧家[[罗贯中_(杂剧家)|罗贯中]]（约1323年－约1397年）<ref name="art1">[http://www.literature.org.cn/Article.aspx?ID=31926 两个罗贯中] {{webarchive|url=https://web.archive.org/web/20151208235837/http://www.literature.org.cn/Article.aspx?ID=31926 |date=2015-12-08 }}</ref>，而《三国演义》的作者罗贯中最为可能是[[山东|山东]][[东平|东平]]人<ref name="art1"/><ref>陈辽，《江苏社会科学》 ，2007年第4期179-182</ref>。

关於罗贯中太原说，主要见于明朝[[贾仲明|贾仲明]]的《[[录鬼簿续编|录鬼簿续编]]》，其他史书不见记载。有研究说《录鬼簿续编》不是贾仲明所写；也有研究说《录鬼簿续编》存在抄写错误，将"东原"抄成"太原"，而且历史上有一段时间今济南长清为中心毗连东平的一带正叫做「东太原」。有关他的许多事蹟多为民间传说，因而一直以来众说纷纭<ref name="施耐庵新证" />。对于太原说实有一个致命缺陷，如果罗贯中是山西人，在《三国演义》中为何对山东境内的琐屑历史人物如此熟悉和感兴趣。有研究认为作者对“东平府”和“太原府”熟悉、认知乃至热情的程度是很不一样的，《三国演义》作者罗贯中对“东原”有某种“故土”情结<ref>杜贵晨(2002),"《三国演义》作者罗贯中是山东东平人",《南都学坛》2002年第6期]</ref>。

一般认为，《[[水浒传|水浒传]]》也是由罗贯中帮助整理修改的<ref>[[李贽|李贽]]《〈忠义水浒传〉序》说：“施罗二公，身在元，心在宋；虽生元日，实愤宋事。是故愤二帝之北狩，则称破大辽以泄其愤；愤南渡之苟安，则称灭方腊以泄其愤。敢问泄愤者谁乎？则前日啸聚水浒之强人也，欲不谓之忠义不可也。是故施罗二公传《水浒》，而复以忠义名其传焉。”</ref>。另外一种说法是，《水浒传》的后半部是他写的<ref name="施耐庵新证">曹晋杰，朱步楼，施耐庵新证，学林出版社，1987年4月</ref>。《水浒传》的后半部，在一百回版本是三十回，包括了[[招安|招安]]，征[[契丹|契丹]]和征[[方腊|方腊]]，在一百二十回版本还包括征[[田虎|田虎]]、征[[王庆|王庆]]。《水浒传》的后半部有时也称之为《[[征四寇|征四寇]]》，比较吻合罗贯中的“忠君”思想。

== 生平 ==

罗贯中8岁开始在[[私塾|私塾]]学[[四書五經|四書五經]]。14岁时母亲病故，于是辍学随父亲去[[苏州|苏州]]、[[杭州|杭州]]一带做生意。但是罗贯中对[[商業|商業]]不感兴趣，在父亲的同意下，他到[[慈溪|慈溪]]随当时的著名学者[[赵宝丰|赵宝丰]]学习。

[[元惠宗|元惠宗]][[至正|至正]]十六年（1356年），罗贯中辞别赵宝丰，到[[张士诚|张士诚]][[幕府|幕府]]作[[幕賓|賓]]。第二年在罗贯中的建议下，张士诚打败了[[朱元璋|朱元璋]]的部下[[康茂才|康茂才]]的进攻。同年，张士诚的弟弟兵败被[[元朝|元朝]]俘虏，张士诚只好投降。降元後，张士诚贪图享乐。到至正二十三年，张士诚看到[[蒙古人|蒙古人]]没落，又再次称王。包括罗贯中在内的许多[[幕僚|幕僚]]都建议暂缓称王，但是不被采纳。罗贯中自此对张士诚失去了信心，返回老家。

路上他遇到同乡[[贾仲明|贾仲明]]，得知父亲已经逝世，继母改嫁，便不再打算回老家。在河阳山（今日[[江蘇省|江蘇省]][[蘇州市|蘇州市]][[張家港|張家港]]）他遇到了正在撰写《[[水浒传|水浒传]]》的[[施耐庵|施耐庵]]。罗贯中非常赞同他写书劝世的主张，于是拜施耐庵为师<ref>[[胡应麟|胡应麟]]《少室山房笔丛》云：“元人武林施某所编《水浒传》特为盛行……其门人罗本亦效为《三国志演义》。”</ref>。此後，罗贯中一直陪伴在施耐庵旁边，帮助抄写书稿。随後自己也开始撰写《三国演义》，并得到了施耐庵的许多指点。

另說：羅貫中也在[[張士誠|張士誠]]處任職<ref>[[顾苓|顾苓]]《塔影园集》说“客霸府张士诚”</ref>，士誠敗歿，奉[[朱元璋|朱元璋]]令，遂專心著作出三國演義，感化風俗人心。

[[元惠宗|元惠宗]][[至正|至正]]二十六年春（1366年），罗贯中的老师赵宝丰去世。罗贯中赶往[[慈溪|慈溪]]祭奠。几个月後，施耐庵为躲避兵乱全家迁往了[[兴化|兴化]]。罗贯中找不到施耐庵，就在河阳山继续写《三国演义》。

当罗贯中打听到施耐庵下落，准备投奔时，正巧赶上施耐庵因《水浒传》而被朱元璋[[禁錮|禁錮]]。于是罗贯中赶往[[金陵|金陵]]，找到了施耐庵的好友[[刘伯温|刘伯温]]帮忙营救。经过一年多後，施耐庵终于出狱。罗贯中雇船送他回[[兴化|兴化]]。但是由於施耐庵途中染病，罗贯中陪伴他在[[淮安|淮安]]养病。不久施耐庵去世，罗贯中帮忙料理了[[喪事|喪事]]，然後带着《水浒传》书稿去[[福建|福建]][[建阳|建阳]]找人刻印。

但是在建阳无人敢承印《水浒传》。他没有办法，只好到[[杭州|杭州]]暂时住下。在这裡他整理修改了《水浒传》，也有人说《水浒传》後三十回就是这时完成的。同时他还改定了自己的《三国演义》，又写了《[[三遂平妖传|三遂平妖传]]》二十回，病逝在[[杭州|杭州]]。
