'''罗哲文'''{{BD|1924年|6月15日|2012年|5月14日|catIdx=Luo罗}}，中国[[四川|四川]][[宜宾|宜宾]]人，中国古建筑学家。

==生平==
1940年考入[[中国营造学社|中国营造学社]]，师从[[梁思成|梁思成]]、[[刘敦桢|刘敦桢]]等。1946年随中国营造学社到北平，作为梁思成（时任[[清华大学|清华大学]]建筑系主任）的助理，并在清华大学与中国营造学社合办的[[中国建筑研究所|中国建筑研究所]]工作。1950年到[[文化部|文化部]]文物局（后改为[[国家文物局|国家文物局]]）任职，担任文物局局长[[郑振铎|郑振铎]]的业务秘书。后任文物档案资料研究室、[[中国文物研究所|中国文物研究所]]所长、国家文物局古建筑专家组组长、[[中国文物学会|中国文物学会]]名誉会长、[[全国历史文化名城|全国历史文化名城]]保护专家委员会副主任、[[中国长城学会|中国长城学会]]副会长等职，并担任[[中国人民政治协商会议|中国人民政治协商会议]]第六、七、八届全国委员会委员。<ref name="cp">{{Cite web |url=http://www.cp.com.cn/newsdetail.cfm?iCntno=1738 |title=罗哲文 古建筑学家 |accessdate=2010-12-17 |publisher=商务印书馆 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url=http://news.163.com/10/1211/02/6NJE4PVP00014AED.html |title=罗哲文：大运河是最后的牵挂 |accessdate=2010-12-17 |publisher=网易. 原载宁波日报. }}</ref><ref>{{Cite web |url=http://www.chinaculture.org/gb/2009-01/22/content_320634.htm |title=罗哲文：一辈子的建筑情缘 |accessdate=2010-12-17 |publisher=中国文化网 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2012年5月14日23时52分在[[北京|北京]]病逝<ref>{{Cite web |url=http://www.chinanews.com/cul/2012/05-15/3890477.shtml |title=古建筑专家罗哲文遗体告别仪式定于5月20日举行 |publisher=中国新闻网 }}</ref>，是[[中国营造学社|中国营造学社]]最后一位去世的成员。<ref>{{Cite web |url=http://cul.sohu.com/s2012/diyixianchang38/ |title=梁思成学生、著名古建专家罗哲文逝世 营造社风流云散  |publisher=搜狐网 }}</ref>

主要著作有《中国古塔》、《中国古代建筑简史》、《长城》、《长城史话》和《中国帝王陵》等。<ref name="cp"/>
