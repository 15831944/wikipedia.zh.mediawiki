'''羅利'''（{{lang-en|Raleigh}}）是[[北卡羅來納州|北卡羅來納州]]的首府及[[韋克縣_(北卡羅來納州)|韋克郡]]的郡政府所在地。因為種植了許多[[橡樹|橡樹]]，所以也被稱為"橡樹之城"。羅利是北卡州中繼[[夏洛特_(北卡羅萊那州)|夏洛特]]後人口第二多的城市。<ref name="autogenerated1">[http://www.raleighnc.gov/portal/server.pt/gateway/PTARGS_0_2_306_202_0_43/http%3B/pt03/DIG_Web_Content/category/Resident/Raleigh_At_A_Glance/Cat-1C-20051006-152447-Raleigh_Demographics.html City of Raleigh | Raleigh Demographics<!-- Bot generated title -->]</ref>2009年1月1日所統計的人口數是385,507。<ref name="autogenerated1" /><ref>[http://money.cnn.com/2007/06/27/real_estate/258_fastest_growing_cities/index.htm The 258 fastest growing U.S. cities - Jun. 27, 2007<!-- Bot generated title -->]</ref>根據美國人口普查局的資料，羅利是美國人口成長率最高的都會區。<ref>[http://www.census.gov/Press-Release/www/releases/archives/population/013426.html<!-- Bot generated title -->] {{webarchive|url=https://web.archive.org/web/20090322104554/http://www.census.gov/Press-Release/www/releases/archives/population/013426.html |date=2009-03-22 }}</ref>

羅利、[[德罕_(北卡羅來納州)|德罕]]和[[教堂山_(北卡羅來納州)|教堂山]]是{{link-en|三角 (北卡羅萊納州)|The Triangle (North Carolina)|三角}}都會區最主要的組成城市。三角都會區源自位於羅利和德罕間成立在1959年的[[三角研究園|三角研究園]]。羅利城市大部份都位在[[韋克縣_(北卡羅來納州)|韋克郡]]內，另外有一小部份位於[[德罕郡_(北卡羅來納州)|德罕郡]]內。联想集团的美国总部就坐落在此地。<ref name="map1">[http://www.raleighnc.gov/publications/Planning/Maps/Raleigh_Durham_Annexation_Agreement_Lines.pdf Raleigh Durham Annexation Agreement Lines]</ref>

== 地理与气候 ==
罗利属典型的[[副热带湿润气候|副热带湿润气候]]，四季分明，气温变化相对和缓，全年降水分配均匀。冬季温和，微潮，时而偏向温暖，日最高气温低于{{convert|0|C}}的平均日数为2.7天，日最低气温低于{{convert|0|C}}的平均日数为69天，低于{{convert|-5|C}}的平均日数为16天；夏季相对炎热潮湿，日最高气温超过{{convert|30|C}}的日数年均有86天，超过{{convert|35|C}}的有14天。<ref name = "NWS Raleigh, NC"/>最冷月（1月）均温{{convert|41.0|F|1|disp=flip}}，极端最低气温{{convert|−9|F|0|disp=flip}}（1985年1月21日）。<ref name = "NWS Raleigh, NC"/>最热月（7月）均温{{convert|80.0|F|1|disp=flip}}，极端最高气温{{convert|105|F|0|disp=flip}}（上次出现于2012年7月8日）。<ref name = "NWS Raleigh, NC"/>无霜期平均为207天（4月7日至10月30日）。<ref name = "NWS Raleigh, NC"/>年均降水量约{{convert|43.3|in|sigfig=3|disp=flip}}，年极端最少降水量为{{convert|29.93|in|disp=flip}}（1933年），最多为{{convert|64.22|in|disp=flip}}（1936年）。<ref name = "NWS Raleigh, NC"/>年均降雪量为{{convert|6.1|in|cm|disp=flip}}，<ref name = "NWS Raleigh, NC"/>但许多冬季的积累降雪量微少以致难以测量，1892–93年的降雪量最多，积累降雪量为{{convert|31.6|in|cm|disp=flip}}。<ref name = "NWS Raleigh, NC"/>
