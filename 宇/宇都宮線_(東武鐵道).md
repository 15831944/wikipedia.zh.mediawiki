'''宇都宮線'''（{{jpn|j=宇都宮線|hg=うつのみやせん|rm=Utsunomiya sen|f=y}}）是一條連結[[栃木縣|栃木縣]][[栃木市|栃木市]][[新栃木站|新栃木站]]和同縣[[宇都宮市|宇都宮市]]{{link-ja|東武宇都宮站|東武宇都宮駅}}，屬於[[東武鐵道|東武鐵道]]的鐵路線。車站編號的路線記號為'''TN'''。

== 路線資料 ==
* 路線距離：24.3公里
* [[軌距|軌距]]：1067毫米
* 站數：11個（包括起終點站）
* 複線路段：沒有（全線[[單線鐵路|單線]]）
* 電氣化路段：全線（[[直流電|直流電]]1500V）
* [[閉塞_(鐵路)|閉塞方式]]：自動閉塞式
* 最高速度：90公里/小時
* 最長編組：4節編組
* 保安裝置：[[自動列車停止裝置#東武鉄道TSP式（多変周式・パターン照査型）・東京都交通局T形ATS|東武型ATS]]

== 概要 ==
[[栃木縣|栃木縣]]縣廳所在地[[宇都宮市|宇都宮市]]與[[壬生町|壬生町]]和[[栃木市|栃木市]]間連結的地域生活路線。[[東武宇都宮站|東武宇都宮站]]至[[江曾島站|江曾島站]]、[[西川田站|西川田站]]、[[壬生站|壬生站]]附近的區間很多人使用。經[[東武日光線|東武日光線]]・[[東武伊勢崎線|伊勢崎線]]可與東京都心連接。

== 历史 ==
以前於淺草（伊勢崎線） - 東武宇都宮間為直通準急（現在的區間急行）運行、2006年3月18日時刻表改正之後取消直通運行（所要時間：2時間30分 - 3時間程度）。

* 1931年（昭和6年）8月11日 新栃木 - 東武宇都宮間全線開業。
* 1931年（昭和6年）11月1日 野州大塚站開業。
* 1932年（昭和7年）4月1日 花房町站開業。
* 1932年（昭和7年）4月17日 （臨）野球場前站開業。
* 1933年（昭和8年）12月15日 野球場前站成為常設站，改稱南宇都宮站。
* 1944年（昭和19年）6月30日 花房町站休止。
* 1944年（昭和19年）7月1日 江曾島站開業。
* 1944年（昭和19年）10月1日 野州平川站開業。
* 1953年（昭和28年） 淺草 - 東武宇都宮間的無料急行・準急（毎時1-2班運行）增加。
* 1954年（昭和29年）9月24日 花房町站廢止。
* 1956年（昭和31年） 淺草 - 東武宇都宮間的有料急行給予愛稱「{{lang|ja|しもつけ}}」。
* 1959年（昭和34年） 淺草 - 東武宇都宮間的急行「{{lang|ja|しもつけ}}」廢止。淺草-東武宇都宮間直通列車再次變回準急・快速（毎時1-2班）。
* 1959年（昭和34年）11月17日 東武宇都宮站移轉，縮短0.1 km。
* 1965年（昭和40年）6月7日 玩具之町（{{lang|ja|おもちゃのまち}}）站開業。
* 1970年（昭和45年）2月5日 柳原信號所開設。
* 1988年（昭和63年）8月9日 淺草 - 東武宇都宮間的快速急行「{{lang|ja|しもつけ}}」運行開始。
* 1989年（平成元年）11月28日 柳原信號所休止。
* 1991年（平成3年）7月21日 快速急行「{{lang|ja|しもつけ}}」昇格為急行。
* 2006年（平成18年）3月18日 急行「{{lang|ja|しもつけ}}」昇格為特急。
* 2007年（平成19年）8月 部份時間帯的普通列車開始單人運行。
* 2007年（平成19年）10月31日 全普通列車開始單人運行。

== 車站列表 ==
* 所有車站均位於[[栃木縣|栃木縣]]。

* 所有車站的業務委託{{link-ja|車武車站服務|東武ステーションサービス}}（新栃木站除外）。
