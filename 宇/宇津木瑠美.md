'''宇津木瑠美'''（{{bd|1988年|12月5日|catIdx=}}）是一位[[日本|日本]]女子[[足球|足球]][[運動員|運動員]]，現效力於[[法國甲組女子足球聯賽|法國甲組女子足球聯賽]][[蒙彼利埃足球俱樂部_(女子足球隊)|蒙彼利埃女子隊]]，位置為[[後衛_(足球)|後衛]]或[[中場#防守中場|防守中場]]。同時她也代表[[日本國家女子足球隊|日本]]參加兩屆[[女子世界盃足球賽|女子世界盃]]（[[2007年女子世界盃足球賽|2007年]]、[[2011年女子世界盃足球賽|2011年]]）、[[2008年夏季奧林匹克運動會女子足球比賽|2008年奧運會]]、[[2008年世界女子青年錦標賽|2008年U20世界盃]]及兩屆的[[女子亞洲盃足球賽|女子亞洲盃足球賽]]（[[2008年女子亞洲盃足球賽|2008年]]、[[2010年女子亞洲盃足球賽|2010年]]）。

== 經歷 ==
因為家中哥哥的影響之下，兩歲就開始接觸足球<ref name=referenceA>{{Cite journal|language=ja|author=砂坂美紀|title= PICK UP NEW HERO 女子編 File 008 宇津木瑠美|journal=週刊足球雜誌|issue=1216|pages=p. 51|publisher=棒球雜誌社|date=2008年12月}}</ref>。六歲開始參加[[神奈川縣|神奈川縣]][[川崎市|川崎市]]的小球隊川崎翅翼練習，小學四年級通過[[日本職業足球甲級聯賽|J1]]球隊[[川崎前鋒|川崎前鋒]]的測試，進入所屬少年隊<ref name=referenceA/>。

2001年，加入[[日視美人|日視美人]]的青少年隊日視姑娘，同年也得到參與[[全日本女子足球選手權大會|全日本女子足球選手權大會]]的機會，並進入日本U-16、U-18選拔名單，可惜最後因被認為太過年輕而未入選。隔年，以14歲的年齡被登錄為日視美人正式球員，也成功地成為國家代表隊成員，參加[[2002年亞足聯U19女子青年錦標賽|2002年亞足聯U19女子青年錦標賽]]<ref name=referenceA/>，最終日本也獲得冠軍，獲得參加首屆[[U19女子世界盃|U19女子世界盃]]的資格。2004年6月27日她首次在正式比賽替補出場，面對對手[[東電海風|YKK東北男人婆]]。9月19日，首次先發上陣，對上[[AC長野合作者女子隊|大原學園JaSRA女子足球隊]]。同年，再次代表日本參加[[2004年亞足聯U19女子青年錦標賽|亞足聯U19女子青年錦標賽]]，結果在半準決賽0比1負於[[中國國家女子青年足球隊|中國]]。

2005年，新選拔日本U-17代表，參加在[[韓國|韓國]]舉辦的[[2005年亞足聯U17女子少年錦標賽|2005年亞足聯U17女子少年錦標賽]]，決賽中戰至[[互射十二碼|互射十二碼]]，她罰球命中，日本也淘汰[[中國國家女子少年足球隊|中國]]，取得冠軍。5月21日，在[[日本國家女子足球隊|日本]]與[[紐西蘭國家女子足球隊|紐西蘭]]的比賽中首次亮相。10月16日，聯賽面對[[伊賀女忍者|伊賀女忍者]]的比賽攻入生涯首個聯賽進球。

2007年及2008年，先後代表日本參加[[2007年女子世界盃足球賽|2007年女子世界盃足球賽]]、[[2007年亞足聯U17女子少年錦標賽|2007年亞足聯U17女子少年錦標賽]]、[[2008年夏季奧林匹克運動會女子足球比賽|2008年北京奧運]]、[[2008年世界女子青年錦標賽|2008年U20世界盃]]，其中2008年U20世界盃擔任主將，最後球隊止步於半準決賽。2010年7月24日，公布加入法國的[[蒙彼利埃足球俱樂部_(女子足球隊)|蒙彼利埃女子隊]]消息<ref>[http://www.mhscfoot.com/liste-saisons/saison-2010-2011/articles/rumy-utsugi-une-nippone-chez-les-feminines Rumy Utsugi, une nippone chez les féminines] {{webarchive|url=https://web.archive.org/web/20120731055053/http://www.mhscfoot.com/liste-saisons/saison-2010-2011/articles/rumy-utsugi-une-nippone-chez-les-feminines |date=2012-07-31 }}{{fr icon}}[[蒙彼利埃足球俱樂部_(女子足球隊)|蒙彼利埃女子隊]]網站 2010.7.24</ref><ref>[http://www.verdy.co.jp/beleza/belezatop/bnews/b2010/100728b.html 宇津木瑠美選手移籍のお知らせ] {{Wayback|url=http://www.verdy.co.jp/beleza/belezatop/bnews/b2010/100728b.html |date=20150831204719 }}{{ja icon}} [[日視美人|日視美人]]網站 2010.7.28</ref>，是第一位到該聯賽踢球的日本選手。之後參加[[2011年女子世界盃足球賽|2011年女子世界盃足球賽]]，與隊友一齊幫助日本獲得首座冠軍。
