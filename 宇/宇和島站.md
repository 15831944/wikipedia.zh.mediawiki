'''宇和島站'''（{{jpn|j=宇和島駅|hg=うわじまえき|rm=Uwajima eki|f=y}}）是一位於[[日本|日本]][[愛媛縣|愛媛縣]][[宇和島市|宇和島市]]錦町、隸屬於[[四國旅客鐵道|四國旅客鐵道]]（JR四國）的[[鐵路車站|鐵路車站]]。車站編號為'''U27/G47'''。

本站是既[[予讚線|予讚線]]的終點站，亦是[[予土線|予土線]]的發車站－雖然正式起點在[[北宇和島站|北宇和島站]]。

==車站構造==
建築物為結合[[酒店|酒店]]及車站的多功能建築物。地下為車站大堂，設有候車大堂、售票窗口、自動售票機、小賣店（包括子公司的麵包店）、酒店餐廳、洗手間等。<ref>[http://ku-gyou.net/JR-S/uwajima.html 非官方車站介紹。]</ref>樓上則為JR四國酒店集團旗下的“宇和島克萊曼酒店”，共設有82間客房。<ref>[http://www.shikoku.ne.jp/clement-uwajima/outline/outline.html 酒店概覽（日文）。]</ref>

現時共設有[[側式月台|側式月台]]、[[島式月台|島式月台]]各1個，共提供2線3面的月台配置，由於是終點站，所以採用了港灣式月台設計。近站舍的側式月台為1號，有效長度達8輛，是所有上行的特急列車的發車月台。島式月台為2、3號月台，有效長度較短，只有6輛，通常用作普通列車發車月台，亦有一部份予讚線普通列車是由1號月台開出的。

1號月台過往曾為部份島式月台，在上行方向盡頭，可見到在另一邊有白邊界痕跡，這裏曾經設有一個短小月台供短編成的客車靠泊，現在已廢除，並整理為停車場。<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/yosan/st-uwajima_h0.jpg |title=存档副本 |accessdate=2012-09-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304065928/http://www.shikoku.org.uk/JR/LINE/yosan/st-uwajima_h0.jpg |archivedate=2016-03-04 }}</ref>
3號月台近港灣盡頭位的月台被欄杆所封閉，同時路軌裝有駛繼續駛往下行的側線道，這是為了讓機關車頭能容易的離開列車並能快速地移位另一端作回程時用。<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/yosan/st-uwajima_h4.jpg |title=存档副本 |accessdate=2012-09-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304091548/http://www.shikoku.org.uk/JR/LINE/yosan/st-uwajima_h4.jpg |archivedate=2016-03-04 }}</ref>

另外，由於本站屬非電化區間，由月台盡頭起，每間一段距離均都設有[[加油站|給油站]]，供柴油列車補充燃料。<ref>[http://www.shikoku.org.uk/JR/LINE/yosan/st-uwajima_oil3.jpg 路軌旁的加油槍。] {{webarchive|url=https://web.archive.org/web/20160304082350/http://www.shikoku.org.uk/JR/LINE/yosan/st-uwajima_oil3.jpg |date=2016-03-04 }}</ref><ref>[http://hacchi-no-he.net/line/yosan/station/0870_uwajima.htm 非官方車站介紹。]</ref>
