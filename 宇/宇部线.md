'''宇部线'''（{{jpn|j=宇部線|hg=うべせん|rm=Ube sen|f=y}}）是一條連結[[山口縣|山口縣]][[山口市|山口市]]的[[新山口站|新山口站]]至山口縣[[宇部市|宇部市]]的{{link-ja|宇部站|宇部駅}}，屬於[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）的鐵路線（[[幹線|幹線]]）。

== 概要 ==
宇部线的线路沿[[濑户内海|濑户内海]]西南端的{{link-ja|周防滩|周防灘}}敷设并贯穿[[宇部市|宇部市]]的中心区域。宇部线线路中宇部市的{{link-ja|宇部站|宇部駅}}以及[[山口市|山口市]]南部的[[新山口站|新山口站]]因为与[[山阳本线|山阳本线]]交汇而成为旅客中心站点。山阳本线与宇部线于宇部站东侧的[[新山口站|新山口站]]交汇，而西侧则延伸至宇部向西的[[厚狭站|厚狭站]]（山口县[[山阳小野田市|山阳小野田市]]）和[[下关站|下关站]]（山口县[[下关市|下关市]]）。<br>

宇部线曾经以运输[[煤|煤]]等的运输线路而兴盛，拥有众多向沿线工厂方向的支线。其最主要的运输作业为[[宇部兴产|宇部兴产]]向{{link-ja|中央玻璃株式会社|セントラル硝子}}提供的[[石灰岩|石灰岩]]运输。该运输职能于2009年10月18日起废止，转为由{{link-ja|宇部兴产专用道路|宇部興産専用道路}}的卡车运输替代。

全線由{{link-ja|西日本旅客鐵道廣島支社|西日本旅客鉄道広島支社|廣島支社}}的{{link-ja|山口地域鐵道部|山口地域鉄道部}}管轄<ref>[http://www.westjr.co.jp/company/issue/data/ データで見るJR西日本] - 西日本旅客鉄道</ref>。

== 运营状况 ==
=== 普通列车 ===
由于车辆的[[一人控制|一人控制]]化，在[[新山口站|新山口站]]至{{link-ja|宇部站|宇部駅}}间以及{{link-ja|宇部岬站|宇部岬駅}}及{{link-ja|宇部新川站|宇部新川駅}}至{{link-ja|宇部站|宇部駅}}区间（周一至周六车辆往返至宇部岬站）的路段，每1小时有1班车运行。同时，部分自{{link-ja|宇部站|宇部駅}}出发的列车与[[山阳本线|山阳本线]]交汇并行驶至山阳本线的[[厚狭站|厚狭站]]及[[下关站|下关站]]（全程由列车长管理）。此外，[[小野田线|小野田线]]的全部列车都会经过{{link-ja|居能站|居能駅}}至{{link-ja|宇部新川站|宇部新川駅}}路段。
=== 快速列车 ===
自2015年之后，不再有快速列车行驶于宇部线路段。以下为曾经于宇部线路段运营的快速列车：
; 快速列车「龙王」
: {{link-ja|小野田站|小野田駅}}至小郡站（现在的[[新山口站|新山口站]]）之间的经过[[小野田线|小野田线]]及[[宇部线|宇部线]]运行的快速列车，于1959年2月15日开始运营，1961年4月30日结束运营。
:
; 快速列车「常磐(ときわ)」
: {{link-ja|宇部新川站|宇部新川駅}}至小郡站（现在的[[新山口站|新山口站]]）之间运行的快速列车，于1959年2月15日开始运营，1965年9月30日结束运营。
:
; 无名快速列车
: 从1975年3月开始，于小郡站（现在的[[新山口站|新山口站]]）至{{link-ja|宇部站|宇部駅}}之间的经过[[宇部线|宇部线]]运行的快速列车。部分列车直通至[[鹿儿岛本线|鹿儿岛本线]]的{{link-ja|南福冈站|南福岡駅}}并作为鹿儿岛本线的快速列车存在。
:
; 快速列车「云母号(きらら号)」
: 为运输{{link-ja|山口21世纪未来博览会|山口きらら博}}的观光客而设立的于小郡站（现在的[[新山口站|新山口站]]）至[[阿知须站|阿知须站]]之间的{{link-ja|国铁117系电车|国鉄117系電車}}运行的临时快速列车，于2001年7月14日开始运营，同年9月30日结束运营。
:
; 快速「希望传递号(のぞみリレー号)」
: 2003年10月1日开始作为[[山阳新干线|山阳新干线]]的[[希望号列车|希望号列车]]接驳的快速列车。主线由[[新山口站|新山口站]]至{{link-ja|宇部新川站|宇部新川駅}}之间运行。最初在新山口站至宇部新川站之间在[[阿知须站|阿知须站]]和{{link-ja|床波站|床波駅}}停车，2008年3月15日的改正后改为在{{link-ja|宇部岬站|宇部岬駅}}，{{link-ja|東新川站 (山口縣)|東新川駅 (山口県)|東新川}}和{{link-ja|琴芝站|琴芝駅|琴芝}}停车。于2009年3月14日改正废除，并由每日下行2班，上行3班变更为下行2班，上行1班的普通列车。
===货运列车===
宇部线的[[货运列车|货运列车]]运营是宇部站至宇部岬站之间，连接[[美祢线|美祢线]]的{{link-ja|重安站|重安駅}}与宇部岬站路段的专用货物列车，每日2班往返。（星期三1班往返，在厚狭站分离编组至的列车班次美祢线的车辆每日一班）货运车辆车头为{{link-ja|国铁DE10系柴油机车|国鉄DE10形ディーゼル機関車}}，货车为{{link-ja|国铁9500型漏斗货车|国鉄ホキ9500形貨車}}（车籍为太平洋水泥株式会社）。2009年10月18日，宇部线全部货运列车停止运营。

=== 廢除路段 ===
站名、停留場名後的（）內的起點起計的營業距離。編入宇部線前（小野田線時代）的旅客站省略。

; 貨物支線（2006年廢除）
: 居能站（0.0）－{{link-ja|宇部港站|宇部港駅}}（2.2）
:
; 貨物支線（1961年廢除）
: 宇部港站（0.0）－沖之山新礦站（1.8）
:* 作為國鐵線廢除後賣予宇部興產，作為該社的{{link-ja|專用側線|専用側線}}使用。
:
; 本線（1952年廃止）
: 宇部新川站（0.0）－助田停留場（0.9）－藤曲站（1.5）－岩鼻站（3.3）
:* 助田停留場在廢線前的1941年廢除。現在線的宇部新川站－居能站間。
:* 藤曲站廢除後，電留線用作整備和作為宇部電車區使用，1996年電車區也廢除。

=== 廢站 ===
（）內是新山口站起計的營業距離。廢除路段的車站除外。

* 周防江崎停留場：1943年廢除，上嘉川站－深溝站間（4.3）
* 岩倉停留場：1943年廢除、岩倉站附近（8.7）
* 白土停留場：1943年廢除、丸尾站－床波站間（17.2）
* 長生炭礦停留場：1943年廢除、床波站－常盤站間（19.8）
* 開作停留場：1926年廢除、岩鼻站－宇部站間（約31.1）
