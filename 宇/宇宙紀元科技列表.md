== 氘核-{zh-hant:鐳射;zh-hans:激光;}-送電系統  ==
氘核鐳射送電系統是由ZAFT研發用來替[[機動戰士|機動戰士]]充電的系統。主要是為了配合禁止在軍事上使用[[反中子干擾器|反中子干擾器]]和[[核能|核能]]的條約。利用此系統、一艘有裝備Deuterion光束發射器的母艦就能夠隨時供能給機動戰士、快速替其補給不足的電力。原理是透過艦上發射氘核鐳射光線，通過機體特定接收器，進行無線充電。

第一部有此系統的母艦為智慧女神（Minerva）級戰艦一號艦[[LHM-BB01_Minerva|智慧女神號]]（Minerva），以及透過額角接收光線的衝擊高達（[[ZGMF-X56S_Impulse|Impulse Gundam]]）

== 幻像化粒子 ==
'''幻像化粒子'''是一種由太平洋聯合（大洋洲聯合）所研發出來的人工粒子。此種粒子能夠使用在多種軍事用途上面。在血腥情人節戰爭之後地球聯合軍和ZAFT兩軍簽訂了禁止使用此種粒子的條約。但是雙方都有再繼續秘密開發這種粒子的技術。

=== 幻像化粒子隱身系統 ===
又稱為「海市蜃樓系統」，由奧布與地球聯合共同開發、並裝置於G計劃的新型機體「[[GAT-X207_Blitz|電擊鋼彈]]」之上。

實戰應用上，幻像粒子可產生如隱身的效果，透過將幻像粒子覆蓋全體、令電磁波（包括可見光及雷達波等）繞過機體前進。使得敵人在視覺和電戰設備上都無法偵測到機體存在。聯合軍其後將當中扭曲電磁波的技術改良，成為防御用裝備「能源偏向裝甲」 ，並應用於第二代高達「[[GAT-X252_Forbidden|禁斷高達]]」上。

ZAFT進攻希奈波利斯、奪取高達事件，幻想化系統技術，亦隨著電擊鋼彈流入ZAFT，並應用在超巨大粒子炮「[[創世_(鋼彈)|創世紀]]」，克魯澤隊更以此特點，利用電擊鋼彈發動海市蜃樓系統，攻陷號稱鐵壁的要塞阿爾提密斯。

另外，在ZAFT向奧布本土進行突攻時，電擊鋼彈被[[GAT-X105_突擊高達|-{zh-cn:突击敢达; zh-tw:攻擊鋼彈; zh-hk:突擊高達;}-]]先斬去右臂，再被擊破。奧布五大家族、擁有「天之御柱」的薩克哈家族，暗中將右臂從海底打撈，並運返天之御柱，安裝予失去機體右臂的異端鋼彈金色機，成為「異端鋼彈金色機．天」。

後來雖然幻像粒子被禁止、但是在CE 73時、地球聯合軍的戰艦Girty Lue使用它來潛入ZAFT的軍用庫一號進行偷取GUNDAM的計劃、以及其後幻痛小隊強奪D.S.S.D開發的新型MS（[[GSX-401FW_Stargazer|Stargazer Gundam]]）作戰中。

== 納米折射裝甲「八咫鏡」==

[[ORB-01_Akatsuki|ORB-01 -{zh-hans:曉; zh-hant:曉;}-]]裝備的裝甲，原理是以納米比例的光束曲折格子層，與超細微等離子臨界制御層所組成的鏡面裝甲，能夠將敵人的光束攻擊直接反射回去，或者是抵擋戰艦的陽電子光束。由於以鏡面裝甲組成，外觀上便形成類似黃金的顏色。

雖然曉的裝甲能帶來很高的防禦力，但同時裝甲的製造、維修成本自然變得相當的龐大，本機裝甲的建造成本，已可生產20架以上的[[MBF-M1_M1_Astray|M1-{zh-hans:歧途高达;zh-hk:迷惘高達;zh-tw:異端鋼彈;}-]]。

== 陽電子干擾 ==
'''陽電子干擾'''是一種在大氣圈中發射陽電子炮時產生的副作用。主要是利用發射陽電子炮之後，戰艦前方粒子被清空使空氣阻力大幅減少所產生的真空期間來加速升空。在現實中此做法並非不可行，但是能與前方空氣對消的陽電子量若拿來產生能源，其能源大小等同於能產生幾百顆氫彈爆炸的能量，實際上只是浪費能源。

== 反中子干擾器 ==
尤利烏斯條約中，規定禁止核武，為貫徹條例內容，遂出現中子干擾器，抵銷核聚變過程，使核子兵器無效化。然而當ZAFT與地球聯合開戰後，雙方為求戰勝，乃暗中開發「反中子干擾器」，使核子兵器得以再次應用戰場上。

== Gungnir系統 ==
'''Gungnir系統'''使用[[北歐神話|北歐神話]]中主神[[奧丁|奧丁]]的長槍來命名、是一種非常強力的[[電磁脈衝|電磁脈衝]]兵器。此系統發出的電磁波甚至足夠在金屬中產生電力、使[[質量投射器|質量投射器]]（見[[#質量投射器|下文]]）的金屬軌道被破壞。

{{lang|ja|[[w:ja:グングニール_(機動戦士ガンダムSEED)|w:ja:グングニール (機動戦士ガンダムSEED)]]}}

== 抗光束炸彈 ==
'''抗光束炸彈'''是一種圓筒型的炸彈、會在放出後爆炸並釋出一團團能吸收光束能量的粒子。這些粒子雖然很快就會擴散開去而失去防禦能力、但是足夠在短時間內防禦到光束武器。

== 光波防禦盾 ==
'''光波防禦盾'''是一種能夠防禦光束和實體彈的能量護盾。是由歐亞聯邦開發並使用來保護他們的宇宙要塞亞提密斯的。此防盾因此亦被稱為亞提密斯之盾。此技術後來亦有被使用在機動戰士上面、裝備在Hyperion GUNDAM系列上面。光波防盾並不能防禦有[[機動戰士技術|抗光束塗層]]的實體武器上面、因此裝備此護盾的機體能發射出穿過護盾的子彈、敵方的實體劍亦可以斬過此護盾。另外、光波防盾亦能在特殊配合下穿透過另一個光波防盾。

== 陽電子偏折器 ==
'''陽電子偏折器'''是一種利用陽電子來防禦實彈和光束兵器的護盾。此護盾甚至能夠有效防禦到陽電子炮的攻擊。

== 魚鱗系統 ==
'''魚鱗系統'''是一種ZAFT開發來推進地面戰艦的推進器。它使用數千塊的機械鱗片的震動來在泥或沙中推進，除了戰艦，此投術亦應用於MS，在土質鬆軟的地層進行鑽地偷襲。最有名者為「Geo-GOOhN」，曾在CE.73 ZAFT以搜捕加百列為名，對奧布發動的戰爭中，突亡古拿托·埃馬·塞蘭身處的地下基地，令其受波及而死。

此系統亦能夠在水上使用。在較小的載具上面、甚至能夠作為水底下的推進。不過因為機械鱗片不能承受深海的高壓、所以裝備此系統的載具只能在近水面的地方活動。因為此系統能夠往各方向都有同樣力量的推進力、因此裝備此系統能非常有效的提升機體的機動性和運。

應用機體：
Geo-GOOhN

== [[質量投射器|質量投射器]] ==
質量投射器是現實中存在的一種[[推進裝置|推進裝置]]。

在CE的世界中，質量投射器主要用於地球表面將太空飛行載具加速到一定速度以減少前往太空所消耗的能量、增加[[載酬|載酬]]。而在載運貨物的貢獻上比載人更大、因為人體可以承受的重力有限。如果只運送貨物來說、可以直接加速到可以脫離地球[[重力|重力]]的速度。大多數質量加速器在位於赤道附近，因為這裡有利達到7.9千公尺／秒的第一[[宇宙速度|宇宙速度]]。質量加速器是重要的戰略目標，因為它是CE中唯一可以低成本和快速地將大量物資送進宇宙的方法。而且加速器需要承受極大的作用力，對材料選用和工序有很高的要求。故此質量加速器是難於修建的。而面對如Gungnir系統的強力的電磁脈衝兵器更因電磁波在金屬中產生電力而被破壞。在SEED中，[[ZAFT|ZAFT]]攻陷所有在地球的質量投射器（[[O.R.B.|-{zh-hans:歐普;zh-hk:歐普;zh-tw:歐普聯合首長國;}-]]輝夜宇宙港除外），把[[地球聯合|地球聯合軍]]封鎖在地球內。

其中已知的設置地點為：東亞共和國高雄宇宙港（Yggdrasil Kaohsiung。台灣境內，被ZAFT攻陷，在雅金‧杜威戰役後收回。）、南美合眾國巴拿馬宇宙港（Panama。巴拿馬境內，在SEED時被Gungnir系統破壞）、南非統一機構巧手宇宙港（Habilis。東非維多利亞湖畔，被ZAFT佔領後被[[地球聯合|地球聯合軍]]奪回）、歐普輝夜宇宙港（Kaguya。[[O.R.B.|-{zh-hans:歐普;zh-hk:歐普;zh-tw:歐普聯合首長國;}-]]，在SEED時自爆破壞，SEED DESTINY時重建完成）

== [[天梯|天梯]] ==
軌道電梯是現實中存在著基礎的一種運輸方案。

在CE的世界中，'''天之御柱'''，原本是剛完成的奧布軌道電梯頂部，戰爭開始後軌道電梯被廢。頂部的設施轉為軍備製造，由薩哈克家負責。是足以與本國的淤能碁呂島地下工廠匹敵的大規模工廠。地區位置在地球36000千米的高空，位於同步衛星軌道上。PS裝甲、以及ASTRAY用的發泡金屬只能在無重力（或接近無重力的微小重力）的環境下精製，所以要在天之御柱製造。被製造出來的PS裝甲，是要作為後備零件用來安裝在[[GAT-X105_Strike|Strike Gundam]]。

根據[[機動戰士GUNDAM_SEED_ASTRAY|機動戰士GUNDAM SEED ASTRAY]]所指，CE 58年，這一年[[烏茲米·納拉·阿斯哈|烏茲米·納拉·阿斯哈]]就任奧布聯合首長國的代表首長。他在就任的同時計劃作爲國家的事業的軌道電梯的建設。如果使用它的話，比起使用[[火箭|火箭]]和[[航天飞机|太空梭]]的宇宙往返，實現了更安全和更低的成本。其優點帶來的經濟效果是無法計量。那裏已經是宇宙，在那個位置建設的最頂部，本身就是宇宙太空站。而建設首先從最頂部的宇宙太空站開始。在宇宙太空站內，設置了大規模的工場，從月球和小行星帶運來的資源，被製造成電梯的支柱部。但是，正好在最頂部的宇宙太空站完成的時候，[[P.L.A.N.T|PLANT]]和[[地球|地球]]的關係惡化，終于變成了開戰。結果，天之御柱被活用作擁有生産兵器的工場的宇宙太空站。

天之御柱對兵器生産來說變得占重要的分量的時候，其管理就變成由薩哈克家負責。薩哈克家，是擔當奧布的政治的五大氏族之中，尤其與軍事關係特別深厚的氏族。諷刺的是，原本預定是和平利用而製造的軌道電梯「天之御柱」，脫胎換骨地變成製造死亡（軍火）的地方，其後隆德·蜜納·薩哈克更將從希奈波利斯奪取的金色異端高達，裝上從奧布海底打撈的[[GAT-X207_Blitz|Blitz Gundam]]右臂，改造成金色機．天。

奧布被聯合軍佔領後，薩哈克家族領袖隆德·蜜納·薩哈克，在天之御柱發表「天空的宣言」。同時亦協助廢物商公會成員羅．裴爾（LOWE）復修勇者高達（Dreadnought Gundam）的龍騎兵系統，即是X背包。
