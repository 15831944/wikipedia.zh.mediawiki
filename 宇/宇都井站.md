'''宇都井站'''（{{jpn|j=宇都井駅|hg=うづいえき|rm=Uzui eki|f=y}}）位於[[島根縣|島根縣]][[邑智郡|邑智郡]][[邑南町|邑南町]]宇都井下鄉，為[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）[[三江線|三江線]]的一個鐵路站。

== 歷史 ==
* 1975年（昭和50年）8月31日正式開業。
* 1987年（昭和62年）4月1日因[[國鐵分割民營化|國鐵分割民營化]]交由[[西日本旅客鐵道|西日本旅客鐵道]]營運。
* 2009年（平成21年）10月3日同年11月29日只限周末及公眾假期停車15分鐘。
* 2018年（平成30年）
**1月1日 [[NHK|NHK]]的跨年特別節目《{{lang|ja|ゆく年くる年}}》在此站現場直播車站夜景及跨年特別列車。
**4月1日 與三江線廢線同時關閉。<ref>{{cite web|url = https://topics-jp.com/2824.html | title = 宇都井駅【廃止まであと1年未満に迫った、絶景無人駅】JR西日本・三江線［駅ノートはどうなる？］|publisher = Topics-Jp |deadurl = yes |archiveurl =  https://web.archive.org/web/20170801075751/https://topics-jp.com/2824.html |archivedate = 2017年8月1日 |language = ja}}</ref>

該站是一個只有單式1面1線的月臺[[高架車站|高架車站]]（[[停車場_(鐵路)#日本JR定義|停留所]]），為{{link-ja|濱田鐵路部|浜田鉄道部}}管理的一個[[招呼站|無人站]]。因前後皆有隧道，月臺及等候室位處離地二十米，為日本最高的架空月臺<ref name="雑学">{{Cite book|language=ja|author=鉄道ものしり王国(編)|authorlink=|year=2013|title=メッチャおもしろくて、タメになる！最強の鉄道雑学王|pages=182-183|publisher=[[河出書房|河出書房]]|isbn=978-4-309-02175-1}}</ref>，此站亦因此被稱爲「天空之站」<ref>{{Cite web|url = http://www.pref.shimane.lg.jp/admin/seisaku/koho/esque/2014/92/92/01.html |title = 特集しまね発見鉄道旅 |publisher = 島根県広聴広報課 |accessdate = 2015年5月4日|language = ja}}</ref>。該站沒有設置升降機或扶手電梯等升降設備，從地面到月臺需走上共116級的樓梯<ref name="雑学"/>。

車站的樓梯上設有等候室，由當地居民定期打掃，並在座椅上放置座墊。可是，站内並沒有自動販售機等設備。樓梯下設有男女共用的廁所。

== 周邊 ==
車站周邊為被山谷包圍的民居。

== 使用情況 ==
該站2014年度一日平均乘車人次不足一人。另外於2004年度平均人次為四人，1994年度為九人，而1984年度則為八人。
