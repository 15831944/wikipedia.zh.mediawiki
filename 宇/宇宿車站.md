'''宇宿車站'''（{{jpn|j=宇宿駅|hg=うすきえき|rm=Usuki eki|f=y}}）是一位於[[日本|日本]][[鹿兒島縣|鹿兒島縣]][[鹿兒島市|鹿兒島市]]宇宿3丁目34番地1，[[九州旅客鐵道|九州旅客鐵道]][[指宿枕崎線|指宿枕崎線]]沿線的[[鐵路車站|鐵路車站]]。
babah

== 車站構造 ==
[[側式月台|側式月台]]1面1線的[[地面車站|地面車站]]。有[[自動售票機|自動售票機]]。
== 利用狀況 ==
*九州旅客鉄道
**2006年度平均每日乘客量有943人次。
==历史==
*1986年（昭和61年）12月1日 - [[日本國有鐵道|日本國有鐵道]]開設。
*1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]，變成[[九州旅客鐵道|九州旅客鐵道]]、指宿枕崎線的車站。
*2006年（平成18年）3月18日 - 由無人站變為有人站。
