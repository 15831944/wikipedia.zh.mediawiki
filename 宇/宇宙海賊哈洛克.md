《'''宇宙海賊哈洛克'''》（{{jp|j=宇宙海賊キャプテンハーロック|rm=Uchū Kaizoku Kyaputen Hārokku|lead=yes}}）是一部由導演[[荒牧伸志|荒牧伸志]]製作的2013年日本3D CG動畫科幻電影。<ref name = "ann1">{{cite web | url = http://www.animenewsnetwork.com/news/2013-01-31/space-pirate-captain-harlock-remake-english-teaser-posted |title= Space Pirate Captain Harlock Remake's English Teaser Posted|date= 2013-01-31 | accessdate = 2013-01-03| work = [[Anime_News_Network|Anime News Network]]}}</ref><ref name ="ann2">{{cite web |url= http://www.animenewsnetwork.com/interest/2013-01-31/captain-harlock-sci-fi-anime-remake-to-open-this-fall |title = Captain Harlock Sci-Fi Anime's Remake to Open This Fall | date=2013-01-31|accessdate= 2013-01-03|work= [[Anime_News_Network|Anime News Network]]}}</ref><ref>{{cite web |url = http://www.filmbiz.asia/news/toeis-harlock-captured-by-gfm |title = Toei's Harlock captured by GFM |first = Patrick |last = Frater |date = 2013-05-10 |accessdate = 2013-05-10 |work = {{link-en|Film Business Asia}} |deadurl = yes |archiveurl = https://web.archive.org/web/20131028175147/http://www.filmbiz.asia/news/toeis-harlock-captured-by-gfm |archivedate = 2013-10-28 }}</ref>

2010年，東映動畫宣佈他們已經試驗以[[電腦圖學|電腦圖學]]重製[[宇宙海賊哈洛克船長|早期漫畫風格的電視劇]]，並將在當年的[[東京國際動畫博覽會|東京國際動畫博覽會]]中展示。隔年，他們在[[安錫國際動畫影展|安錫國際動畫影展]]展示了宇宙海盜哈洛克的預覽版本。這是東映動畫第二部製作預算超過3000萬美元的作品。福井晴敏重建了故事情節以反映現代社會並由東映動畫提供最新的電影製作技術<ref name="ann2"/>。在[[第70屆威尼斯國際電影節|第70屆威尼斯國際電影節]]，一個英語旁白敘述的國際預告片被釋出<ref>{{cite web | url = http://www.labiennale.org/en/cinema/70th-festival/line-up/off-sel/out-of-competition/ | title = Out of Competition | work = La biennale | accessdate = 26 July 2013 | deadurl = yes | archiveurl = https://web.archive.org/web/20130728070801/http://www.labiennale.org/en/cinema/70th-festival/line-up/off-sel/out-of-competition | archivedate = 2013年7月28日 }}</ref><ref>{{cite news|url= https://www.theguardian.com/film/2013/jul/25/venice-film-festival-2013-line-up/| title=Venice film festival 2013: the full line-up| work= The Guardian |accessdate=25 July 2013|location=London|date = 2013-07-25}}</ref>，並在第33屆{{link-en|夏威夷國際電影節|Hawaii International Film Festival}}上放映<ref>{{cite web | url = http://program.hiff.org/films/detail/harlock_space_pirate_2013 | title = Harlock: space pirate | year = 2013 | publisher = HIFF Fall Festival | work = Program | accessdate = 2013-10-20 | archive-url = https://web.archive.org/web/20131021010224/http://program.hiff.org/films/detail/harlock_space_pirate_2013 | archive-date = 2013-10-21 | dead-url = yes }}</ref>。它在[[日本電影金像獎最佳動畫片獎|日本電影金像獎最佳動畫片獎]]中被提名，而後在2014年的3D創意藝術獎於 最佳國際3D - 動畫類 獲得了 Lumière 獎。<ref name=":0">{{Cite news |url = http://ajw.asahi.com/article/cool_japan/anime_news/AJ201402170017 |title = 'Captain Harlock' named best international animated feature at 3D Creative Arts Awards |date = February 17, 2014 |work = Anime news |accessdate = 30 October 2014 |deadurl = yes |archiveurl = https://web.archive.org/web/20141030194447/http://ajw.asahi.com/article/cool_japan/anime_news/AJ201402170017 |archivedate = 2014年10月30日 }}</ref>

== 故事情節 ==

這是個外星殖民發展殆盡的時代，人類勢力遍佈銀河各個角落，總人口衝破五千億。地球聯邦政府內的蓋亞議會刻意封印地球，不讓人類接近。為爭奪母星地球的居住權，各方勢力之間爆發一場星際大戰。戰爭英雄哈洛克戰後卻在宇宙戰艦艦首刻上巨大骷髏，以宇宙海盜之姿高舉反叛大旗，公然與地球聯邦政府對立。青年馬雅接獲政府密令，臥底潛入宇宙戰艦阿爾卡迪亞號，伺機暗殺哈洛克。然而，馬雅在擔任船員的過程中，逐漸被哈洛克所吸引，原本的殺意也開始動搖。阿爾卡迪亞號堪稱是窮途末路者的最後去處，在哈洛克的引領下，40位船員與聯邦政府的戰爭一觸即發...。

== 評價 ==

電影獲得的評價褒貶不一<ref>{{cite news|url=http://www.hollywoodreporter.com/review/harlock-space-pirate-venice-review-622484 |title=Harlock: Space Pirate: Venice Review |publisher=The Hollywood Reporter |date= 2013-09-05|accessdate=2013-10-20 |first=David |last=Rooney}}</ref><ref>{{cite web|author= |url=http://kotaku.com/harlock-space-pirate-is-a-big-pretty-action-filled-d-1281027164 |title=Harlock: Space Pirate is a Big, Pretty, Action-filled Disappointment |publisher=Kotaku.com |date=2013-09-07 |accessdate=2013-10-20}}</ref><ref>{{cite news|author=Leslie Felperin |url=http://variety.com/2013/film/reviews/space-pirate-captain-harlock-review-venice-1200601251/ |title=‘Space Pirate Captain Harlock’ Review: Shinji Aramaki's Epic a Nerd's Delight |publisher=Variety |date=2013-09-06 |accessdate=2013-10-20}}</ref>。至2013年9月29日為止，該電影已在日本獲得 ¥437,326,416 [[日圓|日圓]]的票房收入<ref name="mojo">{{cite web | url= http://www.boxofficemojo.com/movies/intl/?page=&country=JP&id=_fSPACEPIRATECAPTA01|title= Japanese Box Office, September 28–29|date=2013-10-06|accessdate= 2014-01-16|work= [[Box_Office_Mojo|Box Office Mojo]]}}</ref><ref>{{cite web |url= http://www.animenewsnetwork.com/news/2013-10-06/japanese-box-office-september-28-29|title= Japanese Box Office, September 28–29|date=2013-10-06|accessdate= 2014-01-16|work= [[Anime_News_Network|Anime News Network]]}}</ref>。該電影亦為義大利有史以來最成功的日本電影之一，截至2014年1月為止賺入大約 $6,800,000 美元<ref name=ANN1>{{cite web|title=Captain Harlock CG Film Now Available on Netflix in U.S., U.K.|url=https://www.animenewsnetwork.com/news/2014-08-03/captain-harlock-cg-film-now-available-on-netflix-in-u.s-u.k/.77231|publisher=[[Anime_News_Network|Anime News Network]]|accessdate=January 11, 2015|date=August 3, 2014}}</ref>。電影製作人[[詹姆斯·卡梅隆|詹姆斯{{·}}卡梅隆]]也曾讚美這部電影的3D使用技巧。

這部電影在2014年1月18日由[[華納兄弟|華納兄弟]]在[[洛杉磯|洛杉磯]]舉辦的第五屆3D創意藝術獎中被公認為是最佳的國際動畫片<ref name=":0" />，亦在[[第37回日本電影金像獎|第37回日本電影金像獎]]被提名為[[日本電影金像獎最佳動畫片獎|日本電影金像獎最佳動畫片獎]]候選。<ref>{{cite web |url= http://www.animenewsnetwork.com/news/2014-01-16/wind-rises-madoka-lupin-vs-conan-harlock-kaguya-earn-japan-academy-prize-nods|title= Wind Rises, Madoka, Lupin vs. Conan, Harlock, Kaguya Earn Japan Academy Prize Nods|date=2014-01-16|accessdate= 2014-01-16|work= [[Anime_News_Network|Anime News Network]]}}</ref>
