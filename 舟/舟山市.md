'''舟山市'''（[[吴语|吴语]]拼音: Cieusae Zy）是[[中华人民共和国|中华人民共和国]][[浙江省|浙江省]]下辖的[[地级市|地级市]]，其行政区域范围为整个[[舟山群岛|舟山群岛]]。群岛之中，以[[舟山岛|舟山岛]]最大，其“形如舟楫”，故名舟山。

== 歷史 ==
===古代===
春秋时，舟山属[[越國|越]]，称“甬东”（[[甬江|甬江]]之东），又喻称“海中洲”。

公元738年，唐[[开元|开元]]二十六年，置县，以境内有翁山而命名为“翁山县”，史上群岛置县之始。至公元771年，[[大历|大历]]六年，翁山县因被[[袁晁|袁晁]]率起义军占领而被撤废县治。

公元1073年，北宋熙宁六年，重新置縣，名昌國縣。「意其东控日本，北接登莱，南连瓯闽，西通吴会，实海中之巨障，足以昌壮国势焉。」

[[元朝|元朝]]初年升格为昌國州。

[[明朝|明朝]][[洪武|洪武]]二年(1369)又降为昌國縣，二十年更罢县。洪武二十年将舟山岛城区和镇外的居民和其他46山(岛)的居民徙迁内陆。其後，浙江海商及葡萄牙人，建立港口。公元1548年，明[[嘉靖|嘉靖]]二十七年，浙江巡抚[[朱纨|朱纨]]派都指挥卢镗率兵进攻舟山[[佛渡岛|佛渡岛]][[双屿港|双屿港]]，港口以木石淤塞，城市被大火烧光。数百葡萄牙人被杀，侥幸活命的乘船逃走。

[[清朝|清朝]][[康熙|康熙]]取得[[台湾|台湾]]后，沿海解禁，舟山群岛人口日增。公元1687年，清康熙二十六年，再次设县，因认为“舟”字动而不稳，定名为定海县。「海定则波宁」, 以祈「海波永定」。定海之名称实际移自浙江大陆，为此原定海县同时更名为鎮海縣（即今宁波市[[镇海区_(宁波市)|镇海区]]的前身）。

===近代===
[[第一次鸦片战争|第一次鸦片战争]]期间，卽1840年至1842年，英军兩度占领宁波的[[定海区|定海]]。第一次英軍不攻廣州，轉攻沿海各城，及攻陷舟山。後清庭打發回廣州海域等候撤軍。第二次因雙方不滿[[穿鼻草約|穿鼻草約]]，而重新開戰。北上攻陷沿岸各城，再佔舟山，攻到江寧城外，清廷議和。兩次英軍以清制，[[定海军政府|自行設衙門管治]]。按[[南京条约|江寧條約]]所定，道光二十六年（1846年）英军撤军。

鸦片战争之後，[[道光|道光]]二十一年（1841年），海防紧張，定海县升格为定海直隸厅，直属浙江省。

[[中華民國|中華民國]]成立後，废厅改县，復改定海縣。归属于宁波市管辖。

1937年日本全面侵華，舟山落入日軍之手。1942年10月1日，[[日本|日本]]軍船[[里斯本丸|里斯本丸]]运[[英国|英国]]战俘往日本，中途在舟山被[[美国|美国]][[美國太平洋艦隊|太平洋舰队]][[潜艇|潜艇]]部队第81分队的“鲈鱼”号发射[[鱼雷|鱼雷]]击中，<ref name=":0">{{Cite web|url=http://www.people.com.cn/GB/junshi/2640759.html|title=人民网—运俘船沉入大海 中国渔民勇救盟军战俘|accessdate=2017-04-01|work=www.people.com.cn}}</ref> 日軍棄船，千多名英军战俘遇害。 <ref name=":0" /> 當地[[中国|中国]]渔民義救出300多名英国军人。

1945年，日本戰敗投降，舟山復歸中華民國。1949年，民国三十八年，因战时需要，定海县分为定海、滃州（岱山县前身）两县，[[马迹山|马迹山]]以北苏属各岛设立[[江蘇省_(中華民國)|江苏省]][[嵊泗县|嵊泗县]]。1950年5月13日夜至16日拂晓，驻舟山的[[中华民国国军|中华民国国军]]秘密自舟山撤退，滃州县实际解散，是爲[[舟山撤退|舟山撤退]]。

[[中華人民共和國|中華人民共和國]]成立后，[[中國共產黨|中國共產黨]]進占舟山群岛，成立定海县人民政府，属宁波专区管辖。1953年3月经政务院批准，定海县辖区分为定海县、普陀县、[[岱山县|岱山县]]3县，并从江苏省划入嵊泗县，成立舟山专区（为今日舟山地级市的前身）。1954年[[象山县|象山县]]自[[宁波市|宁波专区]]划入，但在1958年转划归[[台州专区|台州专区]]。1959年撤定海、普陀、岱山、嵊泗县，合并成立舟山县，舟山专区仅辖舟山1县。1960年1月4日，舟山专区撤消，舟山县改属宁波专区。同年11月，所属嵊泗人民公社划归[[上海市|上海市]]。1962年5月重新设立[[舟山专区|舟山专区]]，并撤销舟山县，改分为定海、普陀、岱山、[[大衢县|大衢]]、嵊泗（自上海归还后设县）5县。1964年大衢县撤销，其辖区分别划归岱山、嵊泗2县。1967年3月起舟山专区改称[[舟山地区|舟山地区]]。

1987年1月，经国务院批准，撤销舟山地区和定海、普陀2县，成立舟山市，辖2区（[[定海区|定海区]]、[[普陀区_(舟山市)|普陀区]]）2县（岱山县、嵊泗县），实行以市领导区、县新体制。<ref>[http://www.zhoushan.gov.cn/renshizhoushan/zsgl.jsp?iInfoClass=926 舟山概况] {{webarchive|url=https://web.archive.org/web/20111006071012/http://zhoushan.gov.cn/renshizhoushan/zsgl.jsp?iInfoClass=926 |date=2011-10-06 }}，舟山市人民政府官方网站，2011年7月16日查询</ref>

2017年4月，[[浙江自贸区|浙江自贸区]]成立，涵盖舟山离岛片区78.98平方公里，舟山岛北部片区15.62平方公里，舟山岛南部片区25.35平方公里共三个片区119.95平方公里<ref>{{cite news|title=中国(浙江)自由贸易试验区今天挂牌成立|url=http://news.sina.com.cn/c/2017-04-01/doc-ifycwunr8491753.shtml|accessdate=2017-04-02|date=2017-04-01}}</ref>。

== 地理 ==
*位置：位于[[浙江省|浙江省]][[舟山群岛|舟山群岛]]，地处中国长江口南侧，[[杭州湾|杭州湾]]外的东海海域，是中国第一个以群岛设市的地级行政区划。行政范围为舟山群岛全部，共有大小岛屿1391个，其中有人居住的岛屿有103个。
*面积：区域总面积2.22万平方公里，其中陆地面积1440.12平方公里
*行政中心：临城新区行政大楼
===气候===
舟山气候夏季高温冬季温和，四季分明，海洋性较强，季风显著，属于[[副热带季风气候|副热带季风气候]]。最冷（1月）平均气温5.8℃，最热（8月）平均气温27.1℃，年平均气温16.4℃。极端最高气温42.3℃（2013年8月8日），极端最低气温-6.1℃（1955年1月16日、1958年1月16日）。年降水量1442.5毫米。

=== 行政区划 ===
舟山群岛史上曾三度建县，两度被废。三次设县时期，均隶属浙江省大陆的明州（或宁波府）管辖。

目前，舟山市辖2个[[市辖区|市辖区]]、2个[[县_(中华人民共和国)|县]]。2005年舟山市建立[[渔农村新型社区|渔农村新型社区]]，在一定程度上行使村民委员会的公共管理和服务功能。

* 市辖区：[[定海区|定海区]]、[[普陀区_(舟山市)|普陀区]]
* 县：[[岱山县|岱山县]]、[[嵊泗县|嵊泗县]]

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为112.1万人<ref>{{cite book |author=舟山市统计局|title=《舟山市2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加11.98万人，增长11.95%，年平均增长率为1.14%。其中，男性人口为58.84万人，占52.48%；女性人口为53.29万人，占47.52%。总人口性别比（以女性为100）为110.43。0－14岁人口为11.43万人，占10.19%；15－59岁人口为83.07万人，占74.08%；60岁及以上人口为17.63万人，占15.73%，其中65岁及以上人口为11.78万人，占10.50%。居住在城镇的人口为71.31万人，占63.59%；居住在乡村的人口为40.82万人，占36.41%。
根据2015年舟山市统计年鉴，全市[[常住人口|常住人口]]为115.2万人。

== 交通 ==
{{主条目|舟山交通}}
[[File:20090606_Putuoshan_8807.jpg|thumb]]
[[File:Jintang_Bridge-3.jpg|thumb]]]]
*海运 是主要的对外交通渠道，有[[定海_(城区)|定海]]、[[沈家门|沈家门]]、[[高亭|高亭]]、[[菜园|泗礁]]等客运港口，有班轮通[[上海|上海]]、[[宁波|宁波]]、[[温州|温州]]、[[福州|福州]]等外埠。
*公路 公路通过车渡联运可以通浙江省内大部分地区。岛上交通四通八达，主要岛屿上有班车。向外有舟山跨海大桥（岑港大桥、响礁门大桥、桃夭门大桥、西堠门大桥、金塘大桥）与大陆连接。
*航空 群岛现有民用机场1个，即[[舟山普陀山机场|舟山普陀山机场]]，位于[[朱家尖岛|朱家尖岛]]。在岱山岛上有军用机场，是国民政府退往台湾前修建。目前舟山正在计划开辟岛际直升飞机业务。
*高速公路 舟山通过[[甬舟高速公路|甬舟高速公路]](G9211)与大陆相连。高速公路的主体部分为[[舟山跨海大桥|舟山跨海大桥]]，由岑港大桥、响礁门大桥、桃夭门大桥、西堠门大桥和金塘大桥五座大桥组成，于2009年12月25日正式试通车。

== 宗教 ==
[[普陀山|普陀山]]是[[观音菩萨|观音菩萨]]的道场，所以舟山民众信奉[[佛教|佛教]]的尤多。同时[[道教|道教]]也有一定基础，各岛各村神庙林立。民众在佛事活动中，不称自己为“某某乡某某村人”，而称“某某地方某某庙界下弟子”。另外[[基督教|基督教]]也有一定的传播，尤其是在[[衢山岛|衢山岛]]上，基督教有上百年传教历史，该岛不少民众成为[[基督徒|基督徒]]。
