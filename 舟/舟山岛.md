'''舟山岛'''位于[[中国|中国]][[杭州湾|杭州湾]]东南方、[[浙江|浙江]]省东北部海域。第一次海岛调查时[[面积|面积]]为476.1653[[平方公里|平方公里]]，为[[舟山群岛|舟山群岛]]的主岛，浙江省第一大岛，中国沿海第三大岛，南与[[宁波|宁波]]相望。由于东港海涂开发，以及最近的钓浜、钓梁等[[围垦|围垦]]工程使得[[梁横山|梁横山]]等小岛与舟山岛连陆，所以该岛的面积无疑有所增加，估计目前的实际面积应接近500平方公里。

== 地理 ==
该岛除四周局部狭窄的[[冲积平原|冲积平原]]外，其余为[[山地|山地]]、[[丘陵|丘陵]]，高度一般为[[海拔|海拔]]100～400[[米|米]]。最高点黄杨尖，[[海拔|海拔]]503.6米。

舟山岛氣候是屬於[[亞熱帶季風氣候|亞熱帶季風氣候]]，由於舟山乃一島嶼，易受[[東海|東海]]帶來的海洋氣流影響，故舟山在冬天的時候的氣溫，比起[[上海|上海]]和[[杭州|杭州]]等大陸城市為高。

== 歷史 ==
2004年，[[舟山市|舟山市]]开始建造舟山本岛与中國大陆的跨海大桥，称为[[舟山大陆连岛工程|舟山大陆连岛工程]]，2008年竣工，2009年12月25日通车。 

== 行政區劃 ==
舟山岛东部属[[舟山市|舟山市]][[普陀区_(舟山)|普陀区]]，西部属舟山市[[定海区|定海区]]。中部为临城新区，是舟山市的行政中心。
