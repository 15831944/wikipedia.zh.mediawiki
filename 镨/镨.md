'''镨'''（{{标音|注音=ㄆㄨˇ|拼音=pǔ}}）是一种化学[[元素|元素]]，它的[[化学符号|化学符号]]是'''Pr'''，它的[[原子序数|原子序数]]是59，属于[[镧系元素|镧系元素]]，也是[[稀土元素|稀土元素]]之一。

== 主要性质 ==
镨是一种银白色的，中等柔软的[[金属|金属]]元素，是[[镧系元素|镧系元素]]，在空气中抗腐蚀能力比[[镧|镧]]、[[铈|铈]]、[[钕|钕]]和[[铕|铕]]都要强，但暴露在空气中会产生一层易碎的绿色氧化物，所以纯镨必须在矿物油或充[[氬|氬]]玻璃管中保存。

== 应用 ==
镨主要应用于：

* 和[[镁|镁]]一起用于制造飞机引擎的[[合金|合金]]中；
* 用于碳弧光照明的碳芯中；
* 镨的氧化物用于为玻璃或珐琅添加黄色；
* 镨的化合物也用作催化剂；
* 镨钕混合物可以用于制造电焊和玻璃制造使用的护目镜。

== 历史 ==
镨的名称 Praseodymium 来源于[[希腊|希腊]]语词''prasios''（绿色）和''didymos''（成对的）。

1841年, 发现[[镧|镧]]的[[瑞典|瑞典]][[化学|化学]]家卡尔·古斯塔夫·莫散德从含镧的矿物中分离出一种稀有的[[didymium|didymium]]土， 1874年, 瑞典[[地质|地质]]学家波·提奥多·克莱夫证实了didymium土实际是两种元素的混合物。1879年, 勒考·德·布瓦包得兰从铌钇矿(samarskite) 中取得的didymium 土中分离出一种新元素[[钐|钐]]，因此取名Samarium。 直到1885年, [[奥地利|奥地利]] [[化学家|化学家]] C·F·奥尔·冯·韦耳斯拔男爵才成功地将didymium土分离为镨和钕。两种元素的金属[[盐|盐]]的[[颜色|颜色]]不同。

== 自然界存在状态 ==
镨在自然界中存在于独居石和氟碳铈矿中，可以用离子交换法提取，在混合稀土金属中，大约有5%的镨。

== 同位素 ==
{{main|镨的同位素}}
自然界中镨只有一种稳定同位素， 141-Pr。 但有38种放射性同位素，其中比较稳定的有143-Pr，半衰期为13.57 天； 142-Pr，半衰期为19.12小时。 其他的放射性同位素的半衰期都超不过5.985 小时，大部分的半衰期少于33秒。镨还有6个亚稳态，比较稳定的是138m-Pr (t<sub>½</sub> 2.12 小时)， 142m-Pr (t<sub>½</sub> 14.6 分) 和134m-Pr (t<sub>½</sub> 11 分)。

镨的同位素原子量从120.955 [[原子量单位|u]] (121-Pr)到 158.955 u (159-Pr)。稳定同位素 141-Pr如果放出β射线，会俘获[[电子|电子]]。主要放射产物为[[铈|铈]]的同位素铈58和[[钕|钕]]的同位素钕60。

== 注意事项 ==
像其他稀土元素一样，镨具有慢性低毒，不是生物必须元素。
