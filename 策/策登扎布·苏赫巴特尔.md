'''策登扎布·苏赫巴特尔'''（{{lang-en|Tsedenjav Sukhbaatar}}，{{bd|1962年|||}}）[[蒙古族|蒙古族]]，[[蒙古国|蒙古国]][[色楞格省|色楞格省]]人，[[蒙古国|蒙古国]]外交官。<ref name=ce>[http://bbs1.people.com.cn/postDetail.do?id=99899547 策登扎布·苏赫巴特尔，人民网，2010-05-24]</ref>

==生平==
1962年，苏赫巴特尔生于蒙古色楞格省。1986年，毕业于[[莫斯科国际关系学院|莫斯科国际关系学院]]国际法律专业。1986年至1991年，任[[蒙古人民共和国|蒙古人民共和国]]外交部条法局随员、三秘、二秘。1991年至1992年，为[[伦敦大学|伦敦大学]]进修生。1992年至1995年，任[[蒙古国|蒙古国]]外交部条法局参赞。1995年至1997年，出任[[蒙古国|蒙古国]]常驻[[联合国日内瓦办事处|联合国日内瓦办事处]]参赞。1997年至2001年，任蒙古国驻[[大不列颠及北爱尔兰联合王国|大不列颠及北爱尔兰联合王国]]大使<ref name=ce/>。

2001年至2004年，任[[蒙古民主党|蒙古民主党]]负责外事的秘书长。2004年至2006年，任蒙古国法律、内务部副部长。2006年至2009年，重新担任蒙古民主党负责外事的秘书长。2009年6月至9月，任[[蒙古国总统|蒙古国总统]]外事顾问。2009年11月，任蒙古国驻[[中华人民共和国|中华人民共和国]]大使<ref name=ce/>。2016年8月离任<ref>{{Cite web |url =http://aalco-beijing.mfa.gov.cn/web/wjbxw_673019/t1387033.shtml  |title = 外交部部长助理孔铉佑会见蒙古离任驻华大使苏赫巴特尔 |author =  |date = 2016-08-04 |publisher =  中华人民共和国外交部|language =  |accessdate =  |quote =  }}</ref>。

==家庭==
苏赫巴特尔已婚，育有三子。
