'''猩猩属'''（[[学名|学名]]：{{lang|la|''Pongo''}}），也叫'''人猿'''、'''红猩猩'''、'''紅毛猩猩'''，[[靈長目|靈長目]][[人科|人科]]的一属，與[[猴子|猴子]]最大不同的地方就是沒有[[尾巴|尾巴]]，能用手或脚拿东西。[[馬來語|馬來語]]和[[印尼語|印尼語]]叫做''Orang utan''，意思
“森林中的人”。與[[人類|人類]]十分相近，與人類[[基因|基因]]相似度達96.4％。通常不用聲音溝通，通常有好幾個個體會在同一個區域活動，但彼此不干擾，平均[[壽命|壽命]]大概40年，平均[[身高|身高]]大概171—180[[公分|公分]]。

== 分类 ==
红猩猩属是[[猩猩亚科|猩猩亚科]]中仅存的一属，共有三[[种|种]]，即[[婆羅洲猩猩|婆羅洲猩猩]]与[[蘇門達臘猩猩|蘇門達臘猩猩]]，以及最新發現的塔巴努里猩猩。

早前曾被分类为猩猩亚科的[[大猩猩|大猩猩]]（''Gorilla''）、[[黑猩猩|黑猩猩]]（''Pan troglodytes''）以及[[倭黑猩猩|倭黑猩猩]]（''Pan paniscus''）等因为在亲缘上与[[人类|人类]]更接近在后来而被重新分类到[[人亚科|人亚科]]之中。

== 分布 ==
[[婆羅洲猩猩|婆羅洲猩猩]]与[[蘇門達臘猩猩|蘇門達臘猩猩]]分别分布于[[馬來西亞|馬來西亞]]和[[印尼|印尼]]的[[婆羅洲|婆羅洲]]以及[[印尼|印尼]]的[[蘇門達臘|蘇門達臘]]。猩猩体被红色长毛。四肢末端皆有可对握的手，适应于树栖生活。无尾。食物为[[果实|果实]]、[[花|花]]、[[树叶|树叶]]和树皮。部分猩猩也吃[[昆虫|昆虫]]和[[鸟|鸟]]类。

== 数量 ==
猩猩的生育策略都是採取子代數目少，但撫養期間很長的策略：怀孕期约233～263天，大概要六到七歲左右才能漸漸獨立生活。通常是雌性負責照顧後代，雄性大多獨立行動。因為生育期如此漫長，整體的生育率顯得很低，再加上棲息地被破壞，獵人的捕捉，許多猩猩面臨瀕臨絕種的危險。猩猩的数量在过去一百年明显减少了91％，现今生存在[[婆羅洲|婆羅洲]]的猩猩已剩大约五万五千只。

== 古籍中的描述 ==
《[[禮記|禮記]]·曲禮上》：“鸚鵡能言，不離飛鳥。猩猩能言，不離禽獸。今人而無禮，雖能言，不亦禽獸之心乎。”《[[爾雅|爾雅]]‧釋獸》將猩猩列於蒙頌、猱蝯、玃父、威夷、麔麚、贙、豦、蜼之後，又說：“猩猩小而好啼。”《[[山海經|山海經]]·南山經》說：「招搖之山有獸焉，其狀如禺而白耳，伏行人走，其名曰狌狌。食之善走。」《[[呂氏春秋|呂氏春秋]]．本味》則說：“肉之美者，猩猩之唇、獾獾之炙、雋觾之翠、述蕩之掔、髦象之約。”<ref>《[[呂氏春秋|呂氏春秋]]‧孝行覽第二》，174 頁。</ref>《[[水經注|水經注]]》說[[交趾|交趾]]平道縣：“縣有猩猩獸，形若黃狗，又似狟豚，人面頭顏端正。善與人言，音聲麗妙如婦人好女，對語交言，聞之無不酸楚。其肉甘美，可以斷榖，窮年不厭。”<ref>酈道元《水經注》卷三七葉榆河，第六冊，63 頁。</ref>
