'''穿山疏港高速公路'''是[[浙江省|浙江省]][[宁波市|宁波市]]的一条已开通的高速公路，属于浙江省级高速公路，编号S20，是规划中[[浙江沿海高速公路|浙江沿海高速公路]]的组成部分。起于[[宁波港|宁波港]]穿山四期港区，终于[[宁波绕城高速公路|宁波绕城高速公路]]东段好思房枢纽，总长33.113公里。于2009年12月30日开建<ref>{{Cite web |url=http://news.sina.com.cn/s/2009-12-30/130216855806s.shtml |title=穿山至好思房公路上午开工2012年底通车 |accessdate=2018-02-20 |author=中国宁波网 |date=2009-12-30 |publisher=新浪网}}</ref>，2013年6月28日建成通车<ref>{{Cite web |url=http://news.cnnb.com.cn/system/2013/06/28/007765889.shtml |title=穿山疏港高速今天23:58通车 双向六车道 |accessdate=2018-02-20 |author=中国宁波网 |date=2013-06-28}}</ref>。高速公路设置6处互通立交，1处服务区。

==互通枢纽和服务设施列表==
穿山疏港高速公路共设6个互通出入口和枢纽，1个服务区，以及1个主线收费站。
