{{Vgname|穿山甲之戀|en=Pangolin Love}}是一款由[[Google塗鴉|Google塗鴉]]製作的[[網頁遊戲|網頁遊戲]]，於2017年2月11日、12日、13日及14日之間分4關發布。該遊戲是特別為[[西洋情人節|西洋情人節]]而設計的，可在[[Google|Google]]網站遊玩<ref name="Gadgetmatch" />。

== 劇情與玩法 ==

[[File:Pangolin_Love-1.png|thumb]]、音符、絲帶及花。]]

該遊戲是一款2D卷軸遊戲<ref name="大人物"/>，圍繞著一隻穿山甲尋找完美的情人節禮物，送給他的伴侶<ref name="Time"/><ref name="Pocket-lint.com"/>。第1關的背景設在[[迦納|迦納]]，需要收集[[可可豆|可可豆]]<ref name="Time"/><ref name="Google-1"/>。第2關的背景設定在[[印度|印度]]，需收集的物品為音符<ref name="Time"/><ref name="Google-2"/>。第3關與第4關分別設定在[[中國|中國]]及[[菲律賓|菲律賓]]，要收集絲帶及花<ref name="Google-3"/><ref name="Google-4"/>。前3關需收集150件物品，計時3分鐘；而最後1關則需收集300件物品，計時4分鐘<ref name="Pocket-lint.com"/>。

《穿山甲之戀》有分[[個人電腦|個人電腦]]（PC）版及手機版。PC版的玩家須透過鍵盤控制角色移動。手機版需晃動手機來移動角色，手指點擊則為跳躍<ref name="大人物"/>。此外，玩家亦須避開鋸齒狀的障礙物，若碰觸到了的話，角色會有一段時間動彈不得<ref name="Pocket-lint.com"/>。關卡中會有些能讓角色進入、藏有物品的隱藏空間，但當角色接近時便看不到了<ref name="Pocket-lint.com"/>。此外，遊戲中還有一些道具，如能讓角色擺盪的燈籠<ref name="Pocket-lint.com"/>。

== 開發 ==

該遊戲由喬丹·湯普森（{{lang|en|Jordan Thompson}}）、布萊恩·莫瑞（{{lang|en|Brian Murray}}）、班·麥克馬恩（{{lang|en|Ben McMahan}}）、克里斯·霍姆（{{lang|en|Kris Hom}}）及雅各布·萊蒂（{{lang|en|Jacob Lettie}}）策畫，並由切爾西·桑德斯（{{lang|en|Chelsea Saunders}}）負責設計<ref name="Google-4"/>。製作團隊花費了1年的時間來開發《穿山甲之戀》<ref name="Techcityng.com"/>。湯普森表示，「我們希望能透過這款遊戲，讓你更加了解這些奇妙的生物」<ref name="Quartz"/>。遊戲的配樂及音效由賽勒斯·海特（{{lang|en|Silas Hite}}）負責<ref name="Google-4"/>。

== 發布 ==

《穿山甲之戀》的第1關、第2關、第3關及第4關分別於2017年2月11日<ref name="Time"/><ref name="Google-1"/>、12日<ref name="Time"/><ref name="Google-2"/>、13日<ref name="Time"/><ref name="Google-3"/>及14日<ref name="Google-4"/>發布。在該遊戲的第1關、第2關、第3關及第4關中分別出現了[[大穿山甲|大穿山甲]]<ref name="Time"/><ref name="Google-1"/>、[[印度穿山甲|印度穿山甲]]<ref name="Time"/><ref name="Google-2"/>、[[中華穿山甲|中華穿山甲]]<ref name="Time"/><ref name="Google-3"/>及[[菲律賓穿山甲|菲律賓穿山甲]]<ref name="Google-4"/>。Google與[[世界自然基金會|世界自然基金會]]（WWF）合作，建立了一個專門的網頁，推廣保護鱗狀哺乳動物的重要性<ref name="Time"/>。《{{link-en|Quartz (刊物)|Quartz (publication)|Quartz}}》指出，該遊戲讓「穿山甲」這個字串在Google的查詢量達到過去5年內的新高<ref name="Quartz"/>。

== 評價 ==

Pocket-lint.com認為《穿山甲之戀》很像《[[音速小子系列|音速小子]]》<ref name="Pocket-lint.com"/>。《Gadgetmatch》的卡特·德羅薩里奧（{{lang|en|Kat Del Rosario}}）稱讚了遊戲的動畫及視覺效果，並認為其配樂可愛且活潑，使之成為一款完美的遊戲。德羅薩里奧亦提到，該遊戲能讓人在短暫的空閒時間裡放鬆心情<ref name="Gadgetmatch"/>。
