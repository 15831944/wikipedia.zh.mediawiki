{{標點符號|?}}

'''問號'''（'''？'''），用於[[疑問句|疑問句]]結尾。疑問句末尾的停頓，用問號，反問句的末尾，也用問號。<ref>中华人民共和国国家标准，《标点符号用法》，1995年12月13日发布，1996年6月1日实施。</ref><ref>中華民國教育部国语推行委员会，《重订标点符号手册》，中华民国八十六年三月台湾学术网络三版。</ref>

== 起源 ==
拉丁語{{lang|la|''quæstio''}}，意即[[問題|問題]]，簡寫成Qo。將大楷的Q寫在小楷的o上的符號經轉變後便成為現代的問號。

另一說是認為問號起於9世紀，最初是一點並有一條向右上傾的曲線（好像.～，但曲線的右部分較高）。該點表示句子的終結，好像[[句號|句號]]一般。曲線表示聲調的提昇，可能和早期的音樂記號有所結合。

== 變體 ==
[[西班牙語|西班牙語]]自18世紀後便有這個用法：問句除了結尾要有問號之外，起首時要有倒問號「'''¿'''」。不過，這種格式在追求便捷的網絡聊天中通常略去。

1580年代，反問問號出現，用於[[反問|反問]]句尾。它是一般問號的反轉，其開口不是朝向前面的句子。這個用法於1600年代消失。

[[疑問驚嘆號|疑問驚嘆號]]是[[感歎號|感歎號]]和問號的結合。

[[阿拉伯語|阿拉伯語]]由於文字是由右寫至左；因此問號方向與一般的問號顛倒，寫作「'''{{rtl-lang|ar|؟}}'''」；虽然[[希伯来语|希伯来语]]也是从右往左书写，但问号仍旧用开口朝左的“?”。

[[希腊语|希腊语]]中的问号是类似于分号的“{{lang|el|;}}”（U+037E），但和英语的分号“;”（U+003B）是不同的。

== 電腦 ==
=== 字元 ===
問號（?）是[[Unicode|Unicode]]和[[ASCII|ASCII]]字元[[63|63]]或[[十六进制|0x003F]] / U+003F。

[[全形|全形]]問號（？）在 U+FF1F：
*[[台港澳|台港澳]]和[[日本|日本]]標準為置中，如「<span style="border: 1px dotted red;">{{lang|zh-tw|？}}</span>」，[[直排|直排]]時亦然
*[[中國大陸|中國大陸]]標準為靠左下，如「<span style="border: 1px dotted red;">{{lang|zh-cn|<span style{{=}}"position:relative;left:-5px;">？</span>}}</span>」；直排時則靠右上，Unicode 4.1增加配合中國大陸寫法之「垂直問號（PRESENTATION FORM FOR VERTICAL QUESTION MARK），U+FE16」「<span style="border: 1px dotted red;">︖</span>」<ref>http://www.unicode.org/charts/PDF/UFE10.pdf</ref>，以兼容 [[GB_18030|GB 18030]]。
*另有「小問號（SMALL QUESTION MARK）」「<span style="border: 1px dotted red;">{{lang|zh-tw|﹖}}</span>」U+FE56，源自 [[CNS_11643|CNS 11643]]／[[Big5|Big5]]。

而倒問號（¿）則為Unicode字元191（U+00BF），在微軟視窗內透過按Alt鍵不放在九宮數字鍵盤鍵入0191打出，或將語系設成[[西班牙文|西班牙文]]，打「=」。在[[X11|X11]]，連續打兩個?會得出倒問號。在[[Mac_OS_X|Mac OS X]]，按住Shift鍵與Option鍵後，再按/可打出。

=== 編程語言 ===
* ?是大多数[[程序设计语言|程序设计语言]]的[[条件运算符|条件运算符]][[?:|?:]]的部分。
* 在[[正規表示式|正規表示式]]，?表示之前的字元可以出現1或0次。
* 在[[Ruby|Ruby]]，字元前加「?」會傳回該字元的ASCII值，例如?a會傳回97

=== 其他 ===
在很多[[軟體|軟體]]中，會以「?」表示該字元不在程式的[[字集|字集]]當中。

== 國際象棋 ==
在[[國際象棋|國際象棋]]的棋譜，「?」表示劣着，「??」表示不應犯的大誤着。
