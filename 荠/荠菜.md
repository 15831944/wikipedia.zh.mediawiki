'''荠菜'''（[[學名|學名]]：''Capsella bursa-pastoris''），又名'''护生草'''、'''地菜'''、'''地米菜'''、'''菱闸菜'''等，[[十字花科|十字花科]][[荠菜属|荠菜属]]植物。起源于[[东欧|东欧]]和[[小亚细亚|小亚细亚]]，目前在世界各地都很常见。其[[种加词|种加词]]来自[[拉丁语|拉丁语]]，意思是“小盒子”、“牧人的钱包”，是形容它的[[角果|角果]]形状像牧人的钱包。

== 形态 ==
*一、二年生[[草本植物|草本植物]]。
*白色的主根瘦长。主[[茎|茎]]直立，分出细茎。
*幼苗时，根出叶辐射状的平铺地面，有羽状深裂；长圆形或线状披针形的茎生叶，顶部的叶片大都接近线形，叶基成耳状抱茎，边缘或有缺刻或锯齿，叶有细柔毛。
*春天开白色细长的小[[花|花]]，集成总状[[花序|花序]]，顶生或腋生，绿色的花萼和白色的花冠，[[花瓣|花瓣]]为4。
*短角果呈倒三角形，扁平，内含多数[[种子|种子]]。

== 生態 ==
种子受潮后分泌出粘稠的分泌物，可以吸引並粘住虫子，並可能吸收蟲子的養分。荠菜性喜温和，只要有足够的阳光，[[土壤|土壤]]不太干燥，荠菜都可以生长。

== 分布 ==
分布在全世界的[[温带|温带]]地区，性喜温暖但耐寒力强。野生于田野，也可人工栽培。

== 利用 ==
荠菜的种子、叶和根都可以食用。可作涼拌或煮、炒食。在中華民國經常用來炒年糕或作為[[餛飩|餛飩]]或[[餃子|餃子]]餡的一部分；在日本薺菜做為正月初七[[人日|人日]][[七菜|七菜]]的一部分，海峽兩岸泉州裔的七寶羹亦有包含薺菜，目前僅部分人保留正月初七吃素菜的習慣。

荠菜还可以入药，有明目、利水、和脾、止血等功效。不過懷有身孕或哺乳中的婦女忌食，因薺菜含有醇化合物，有類似催產素的功效，可促進子宮收縮；孕婦食用後，可能導致胎動不安或出血，甚至流產。有心肺疾病的患者在服用時亦應小心。
