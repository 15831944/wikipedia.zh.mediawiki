'''-{zh-hans:豹猫;zh-hk:豹貓;zh-hant:石虎}-'''（學名：''{{lang|la|Prionailurus bengalensis}}''），又名'''山貓'''、'''錢貓'''、'''-{zh-hans:石虎;zh-hk:石虎;zh-hant:豹貓}-'''，是產於[[亚洲|亞洲]]的[[猫科|猫科]][[豹貓屬|石虎屬]]野生動物。

石虎的體型與[[家猫|家猫]]大致相仿，以外表來看容易誤認為家貓，因此偶有野生豹貓被當家貓收養，但各亚种的差别比较大，例如[[印度尼西亚|印度尼西亚]]的亚种平均体长45厘米，尾长20厘米，而[[西伯利亚|西伯利亚]]的亚种体长则达到60厘米，尾长40厘米。

豹貓的毛皮也有很多种颜色：南方的豹貓为黄色，北方的则为银灰色（推測與適應各種生活環境有關）。胸部及腹部是白色。豹貓的斑点一般为黑色。

豹貓乃[[肉食性动物|肉食性]][[夜行動物|夜行動物]]，多在晚上或清晨於[[郊外|郊外]]出沒，通常以[[啮齿类|啮齿类]]、[[鸟|鸟]]类、[[鱼|鱼]]类、[[爬行类|爬行类]]及小型[[哺乳动物|哺乳动物]]为食。除了交配季节外，它们一般为独处。豹貓的妊娠期为65-70天，每胎2-4个後代。

根據國際自然保護聯盟2015年版資料顯示，石虎生長於海拔高度0公尺至3240公尺，在大部分範圍，石虎是穩定且易於使用退化的森林，甚至大量修改過的棲息地，如油棕和甘蔗園。<ref>[https://www.iucnredlist.org/species/18146/50661611#  Leopard Cat Prionailurus bengalensis]</ref>

== 生物特徵 ==
[[File:MSU_V2P2_-_Felis_euptilura_skull.png|thumb]]
豹貓是種小型貓科動物，原產地位於[[東南亞|東南亞]]和[[印度次大陸|印度次大陸]]。豹貓共有11個亞種。-{豹貓}-的名字源于它們所有亞種外表的斑點都類似豹紋（家貓則很少見豹紋）。與家貓的主要差異是：長而粗的尾巴，額頭有兩條白色縱帶，及兩耳後方是黑底白斑。

一般而言豹貓的大小和家貓相仿，但分佈地區對豹貓的大小有很大區別。印尼的豹貓平均約45公分和20公分的尾巴。到了俄罗斯[[阿穆爾州|阿穆爾州]]南部，豹貓的尺寸就有40公分到60公分。豹貓肩高41公分，體重4.5公斤～6.8公斤與家貓類似。它們皮毛的顏色也有很大差異，南方的豹貓是黃色，而北方是銀灰色，胸和頭部下方是白色的。豹貓有黑色斑紋，可能是點狀或其它形狀，依亞種而定。

== 栖所 ==
[[File:Tsushima_Cat_001.jpg|thumb]]
豹貓是貓科動物中分佈最廣泛的物種，其分布地包括[[印尼|印尼]]、[[马来西亚|马来西亚]]、[[菲律賓|菲律賓]]、[[婆羅洲|婆羅洲]]、[[中南半島|中南半島]]（緬甸到越南）、中國大陸、[[朝鮮半島|朝鮮半島]]、[[台灣|台灣]]、[[印度|印度]]、[[巴基斯坦|巴基斯坦]]等。它們的棲息地類型非常多，從[[热带雨林|热带雨林]]到[[針葉林|針葉林]]、從半沙漠到農業特別是接近水源的地區，甚至海拔三千米高的山地。豹貓主要栖息在[[森林|森林]]与[[雨林|雨林]]地带，通常为距河道较近的潮湿区域。

== 棲息地與行為 ==
豹貓擅長爬樹，也能游泳。它們是[[夜行性動物|夜行性動物]]，白天它們在洞穴中，它們的洞穴包括中空的樹幹、地洞、或是山洞。除了繁殖期外，豹貓是獨居動物，在南方的豹貓沒有固定的繁殖期；在較冷的北方，它們的繁殖期約在三、四月，讓氣候溫暖得足以讓新生的豹貓存活。豹貓通常養育小豹貓7至10個月。豹貓通常在18個月時長成，人類豢養的雄豹貓長成期可能縮到7個月，雌豹貓可能縮短到10個月就長成。

== 繁殖與成長 ==
豹貓交配期約5-9日，在60-70日的懷孕期後，每胎通常生下2-4隻幼豹貓。新生豹貓體重在75-130公克之間，在兩週時體重就會加倍。五週時體重成長到出生時的四倍。幼豹貓的眼睛在10日時張開，在23日開始吃固體食物，到四週大時，開始出現永久的犬齒，與開始吃固體食物的時間一致。如果幼貓不幸夭折，母豹貓可以再度進入發情期，同一年再產一胎。
2018年1月23日，香港漁農自然護理署連同愛護動物協會及香港警察於錦田救出豹貓、珍稀靈長類動物及龜類。

== 食物 ==
豹貓是[[肉食性動物|肉食性動物]]，捕獵很多小型生物，包括[[哺乳動物|哺乳動物]]、[[兩棲動物|兩棲動物]]、[[蜥蜴|蜥蜴]]、[[鳥類|鳥類]]、[[昆蟲|昆蟲]]、[[蛇|蛇]]。北方的豹貓也吃[[野兔|野兔]]，也会吃[[草|草]]、[[蛋|蛋]]和[[水棲動物|水棲動物]]作為補充食物。

== [[亚种|亚种]] ==
* ''Prionailurus bengalensis alleni''：[[海南岛|海南岛]] ；
* ''Prionailurus bengalensis bengalensis''：[[印度|印度]]、[[孟加拉国|孟加拉国]]、[[马来西亚|马来西亚]]、[[中印半島|中印半岛]]、[[云南|云南]]；
* ''Prionailurus bengalensis borneoensis''：[[婆罗洲|婆罗洲]]；
* ''Prionailurus bengalensis chinensis''：[[臺灣|臺灣]]、[[中國大陸|中國大陸]]（雲南除外）；
* ''Prionailurus bengalensis euptailura''：[[西伯利亚|西伯利亚]]东部、[[蒙古|蒙古]]、中国[[中国东北地区|东北]]；
* ''Prionailurus bengalensis heaneyi''： [[巴拉望島|巴拉望島]]；
* ''Prionailurus bengalensis horsfieldi''：[[喜马拉雅山|喜马拉雅山]]；
* ''Prionailurus bengalensis javaensis''：[[爪哇岛|爪哇岛]]；
* ''Prionailurus bengalensis rabori''：[[內革羅島|內革羅島]]、[[宿霧|宿霧]]、[[班乃島|班乃島]]； 
* ''Prionailurus bengalensis sumatranus''：[[苏门答腊|苏门答腊]]；
* ''Prionailurus bengalensis trevelyani''：[[巴基斯坦|巴基斯坦]]东部。

[[File:Leopard_Cat_Tennoji.jpg|right]]

*''Prionailurus bengalensis iriomotensis''，即[[西表山貓|西表山貓]]，1967年，科學家經過基因檢測，認定為豹貓亞種。
*''Prionailurus bengalensis euptailurus''，即[[對馬山貓|對馬山貓]]。對馬山貓最初被認為是另一个種，後來被歸為豹貓的亞種，現在認為是中國[[中国东北地区|东北]]地區豹貓的一个亞種。

== 与人类的关系 ==
考古学和形态学研究表明，在[[新石器时代|新石器时代]]的中国第一种被驯化的猫是豹猫，这种驯化至少在5000年前就开始了。然而，随着时间的推移，驯养的豹猫逐渐被[[非洲野猫|非洲野猫]]的后代代替。<ref name="Vigne2016">{{cite journal|last1=Vigne|first1=J.-D.|last2=Evin|first2=A.|last3=Cucchi|first3=T.|last4=Dai|first4=L.|last5=Yu|first5=C.|last6=Hu|first6=S.|last7=Soulages|first7=N.|last8=Wang|first8=W.|last9=Sun|first9=Z.|last10=Gao|first10=J.|last11=Dobney|first11=K.|last12=Yuan|first12=J.|title=Earliest "Domestic" Cats in China Identified as Leopard Cat (''Prionailurus bengalensis'')|journal=PLOS ONE|volume=11|issue=1|date=2016-01-22|page=e0147295|doi=10.1371/journal.pone.0147295|pmid=26799955|pmc=4723238}}</ref>

== 保護 ==
[[瀕臨絕種野生動植物國際貿易公約|華盛頓公約]]將[[孟加拉国|孟加拉国]]、[[印度|印度]]以及[[泰國|泰國]]的族群列入附錄一禁止進行國際貿易，而其他族群亦列入華盛頓公約附錄二。[[孟加拉国|孟加拉国]]、[[柬埔寨|柬埔寨]]、[[印度|印度]]、[[印尼|印尼]]、[[日本|日本]]、[[馬來西亞|馬來西亞]]、[[緬甸|緬甸]]、[[尼泊爾|尼泊爾]]、[[巴基斯坦|巴基斯坦]]、[[菲律賓|菲律賓]]、[[俄罗斯|俄罗斯]]、[[泰國|泰國]]等国也制定了禁止[[狩獵|狩獵]]豹猫的[[法律|法律]]。

=== 中國大陸 ===
豹猫是中国大陆[[国家保护的有益的或者有重要经济、科学研究价值的陆生野生动物名录|国家三有保护动物]]<ref>[http://www.forestry.gov.cn/portal/bhxh/s/709/content-466654.html 中国林业局 - 《“三有”保护动物名录》]</ref>，有5个亚种，包括分布于[[云南|云南]]、[[贵州|贵州]]西部及[[广西|广西]]西部的指名亚种；分布于[[中国东北地区|东北]]、[[华北|华北]]及[[中国西北地区|西北]]地区的为北方亚种；分布于[[华东|华东]]、[[华中|华中]]及[[华南|华南]]地区的为华东亚种；分布于[[云南|云南]]北部、[[四川|四川]]西部、[[西藏|西藏]]东南部及[[甘肃|甘肃]]南部的川西亚种；分布于[[海南岛|海南岛]]的海南亚种。<ref>[http://www.bjkp.gov.cn/art/2008/5/9/art_2273_141928.html 北京科普之窗 -中国珍稀动物]</ref>中国大陆總面積9,596,960平方公里，石虎廣泛分布，族群穩定，總數數百萬隻。

=== 香港 === 
[[File:Prionailurus_bengalensis_in_Hong_Kong.webm|thumb]]洞梓山莊地下露台發現的野生豹貓]] 
豹貓受到[[香港法例|香港法例]]第170章《保護野生動物條例》和第586章《保護瀕危動植物物種條例》保護<ref name="afcd">{{cite web|title=香港生物多樣性數據庫|url=/tc_chi/conservation/hkbiodiversity/database/popup_record.asp?id=3535&lang=tc|website=漁農自然護理署}}</ref>，在[[香港郊野公園|香港郊野公園]]也有出沒記錄<ref>{{cite web|url=/mobile/tc/lifestyle/article.php?id=52388|title=誤認豹貓是虎 行山夫婦虛驚 專家：野生動物不宜走近|first=|last=ETNet|publisher=}}</ref>，並有野生族群。據[[漁護署|漁護署]]資料，豹貓廣泛分佈於除[[大嶼山|大嶼山]]外之香港各郊區<ref name="afcd"/>。

=== 臺灣 ===
台灣目前普遍分佈在[[新竹|新竹]]地區<ref>[https://www.mirrormedia.mg/story/20181220soc010 野生石虎現身新竹芎林山溝　後肢遭夾嚴重變形]</ref>、[[苗栗縣|苗栗縣]]、[[台中市|台中市]]、[[南投縣|南投縣]]、[[彰化縣|彰化縣]]、[[嘉義縣|嘉義縣]]<ref>[https://udn.com/news/story/7326/3210101 瀕臨絕種石虎嘉義現蹤]</ref>。台灣的石虎和中國大陸的屬於同一亞種，又稱作華南豹貓，不是台灣特有種。<ref>[https://tvmsa.tw/tag/%E7%9F%B3%E8%99%8E/ 2015台大獸醫系朝陽講堂]</ref>台灣島面積有35,887平方公里，石虎數量研究不出來。

=== 日本 ===
日本的兩種亞種，[[西表山貓|西表山貓]]與[[對馬山貓|對馬山貓]]。因西表島面積僅289平方公里，石虎數量百隻。
