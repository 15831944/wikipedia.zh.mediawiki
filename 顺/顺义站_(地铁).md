'''顺义站'''位于[[北京市|北京市]][[顺义区|顺义区]]城区中心附近，是[[北京地铁|北京地铁]][[北京地铁15号线|15号线]]（一期二段）的一座车站。

== 位置 ==
顺义站位于[[府前中街|府前中街]]－[[府前东街_(顺义)|府前东街]]和[[光明南街_(顺义)|光明南街]]－[[光明北街_(顺义)|光明北街]]交汇处。15号线站台成东西走向。<ref>{{cite web|url=http://www.bjghw.gov.cn/web/static/articles/catalog_14300/article_ff808081262b02de01262c30a5a2000a/ff808081262b02de01262c30a5a2000a.html|title=轨道交通15号线一期工程规划方案公告|date=2010-01-14|publisher=[[北京市规划委员会|北京市规划委员会]]|accessdate=2010-12-18|archive-url=https://web.archive.org/web/20100728091626/http://www.bjghw.gov.cn/web/static/articles/catalog_14300/article_ff808081262b02de01262c30a5a2000a/ff808081262b02de01262c30a5a2000a.html|archive-date=2010-07-28|dead-url=yes}}</ref>

== 结构 ==
该站为地下车站。地下一层为站厅层，地下二层为15号线站台，地下三层预留有南北向线路的站台。站厅层为圆形，直径53米。<ref>{{cite web|url=http://www.ynet.com/view.jsp?oid=76290155|title=15号线将再开通4站 乘客出地铁或直达地库|date=2011-01-19|accessdate=2011-01-20|deadurl=yes|archiveurl=https://web.archive.org/web/20110120201159/http://ynet.com/view.jsp?oid=76290155|archivedate=2011-01-20}}</ref>

== 事故 ==
[[北京时间|当地时间]]2010年7月14日下午4时30分左右，顺义站工地发生渗水事故，造成2名工人死亡，8人受伤。直接经济损失146万元。这起事故中的8名相关责任人被移送司法机关处理。<ref>{{cite web|url=http://news.china.com.cn/rollnews/2010-07/15/content_3229793.htm|title=关注：北京地铁M15号线顺义站事故现场出现渗水|date=2010-07-15|accessdate=2011-01-20}}</ref><ref>{{Cite web|url=http://beijing.qianlong.com/3825/2011/01/13/2160@6541584.htm|title=地铁顺义站事故8人被追究刑责|date=2011-01-13|accessdate=2011-01-20}}</ref>
