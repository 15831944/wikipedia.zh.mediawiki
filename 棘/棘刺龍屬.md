'''棘刺龍屬'''（[[學名|學名]]：''Spinophorosaurus''）是[[蜥腳下目|蜥腳下目]][[恐龍|恐龍]]的一[[屬_(生物)|屬]]，化石發現於[[尼日|尼日]]的Irhazer組地層的下層，目前難以確定該地層的地質年代，可能屬於[[侏儸紀|侏儸紀]]中期的[[巴柔階|巴柔階]]或[[巴通階|巴通階]]，或者更早期。棘刺龍的唯一化石相當完整，是目前[[非洲|非洲]]所出土的最完整原始蜥腳類恐龍化石，只有[[中國|中國]]、[[南美洲|南美洲]]的數個原始蜥腳類化石比棘刺龍還完整<ref name = Remesetal2009a/>。

==發現與命名==
[[File:Spinophorosaurus.png|thumb]]
[[File:Spinophorosaurus_NT.jpg|thumb]]
[[正模標本|正模標本]]（編號GCP-CV-4229）被發現於[[尼日|尼日]]的[[阿加德茲|阿加德茲]]，由一群[[西班牙|西班牙]]挖掘團隊進行名為「DAGGERS」的挖掘工作時所發現。正模標本包含：頭顱骨的腦殼、[[眶後骨|眶後骨]]、[[鱗狀骨|鱗狀骨]]、[[方骨|方骨]]、[[翼骨|翼骨]]、[[上隅骨|上隅骨]]，以及接近完整的身體骨骼，只缺少[[胸骨|胸骨]]、前臂、手掌、腳掌。正模標本目前存放於尼日自然歷史博物館<ref name = Remesetal2009a/>。

[[副模標本|副模標本]]（編號NMB-1698-R）則是由[[德國|德國]]挖掘團隊所發現，是一個部分頭顱骨、不完整的身體骨骼。副模標本的骨頭部位裡，正模標本所缺少的部位包含：[[前上頜骨|前上頜骨]]、[[上頜骨|上頜骨]]、[[淚骨|淚骨]]、[[齒骨|齒骨]]、[[隅骨|隅骨]]、右側的所有[[肋骨|肋骨]]、[[肱骨|肱骨]]、以及零散的腳部骨頭。正模標本、副模標本所重複的部位包含：眶後骨、鱗狀骨、翼骨、上隅骨、牙齒、脊柱、[[肩胛骨|肩胛骨]]。由於兩個化石發現於同一地層帶，有高度類似處，因此被歸類於相同動物<ref name = Remesetal2009a/>。

[[模式種|模式種]]是'''尼日棘刺龍'''（''S. nigerensis''），是在2009年由古生物學家克里斯提·拉莫斯（Kristian Remes）等人所敘述、命名。屬名在[[古希臘文|古希臘文]]意為「帶有長刺的[[蜥蜴|蜥蜴]]」；種名則是以尼日為名<ref name = Remesetal2009a>{{en}}Remes, K., Ortega, F., Fierro, I., Joger, U., Kosma, R. and Ferrer, J.M.M. (2009). "A new basal sauropod dinosaur from the Middle Jurassic of Niger and the early evolution of sauropoda." ''PLoS ONE'', '''4'''(9): e6924. {{doi|10.1371/journal.pone.0006924}} [http://www.plosone.org/article/info:doi/10.1371/journal.pone.0006924 full text online].</ref>。同年的稍晚研究，根據棘刺龍的腦殼結構、神經解剖學，推論棘刺龍的演化位置介於原始[[蜥腳形亞目|蜥腳形亞目]]、[[新蜥腳類|新蜥腳類]]恐龍之間<ref name = Remesetal2009b>{{en}}Remes, K., Ortega, F., Fierro, I., Joger, U., Kosma, R. and Ferrer, J.M.M. (2009). "A new basal sauropod dinosaur from the Middle Jurassic of Niger and the early evolution of sauropoda." PLoS ONE, 4(9): e6924.doi:10.1371/journal.pone.0006924  {{doi|10.1371/journal.pone.0030060}}</ref>。

==體徵==
棘刺龍的身長估計為約13公尺。棘刺龍是非常特殊的蜥腳類恐龍，正模標本被發現具有兩根長刺狀的[[皮內成骨|皮內成骨]]（Osteoderms），皮內成骨的基部接近圓形、末端呈長刺狀，左側皮內成骨較大、外形比率略有不同。雖然這兩根長刺被發現於骨盆部位，位於脫落、位置移動的肩胛骨的下方，但研究人員根據長刺、尾巴末端的骨頭外形，而推論其生前可能位於尾巴末端，類似[[劍龍類|劍龍類]]恐龍的尾刺。除此之外，生存於亞洲的[[蜀龍|蜀龍]]則具有尾巴尾鎚；蜀龍是種原始真蜥腳類恐龍<ref name = Remesetal2009a/>。

==演化位置與分類==
研究人員根據棘刺龍的[[方骨|方骨]]與後肢特徵，提出棘刺龍是[[塔鄒達龍|塔鄒達龍]]的近親，兩者都生存於非洲（當時為[[岡瓦納大陸|岡瓦納大陸]]的一部分）、都是典型的原始蜥腳類恐龍。棘刺龍也被列為[[真蜥腳類|真蜥腳類]][[演化支|演化支]]的姊妹[[分類單元|分類單元]]。但是，棘刺龍具有某些進階型特徵，類似時代相近、生存於[[勞亞大陸|勞亞大陸]]的真蜥腳類恐龍，例如：[[蜀龍|蜀龍]]、[[馬門溪龍科|馬門溪龍科]]，兩者的類似處在於[[頸椎|頸椎]]、[[尾椎|尾椎]]、[[肩胛骨|肩胛骨]]、[[肱骨|肱骨]]；另一方面，棘刺龍則較不類似[[南美洲|南美洲]]、[[印度|印度]]的原始蜥腳類恐龍，例如：頸椎的長度與側腔形狀、頸椎與[[背椎|背椎]]的[[神經棘|神經棘]]、肱骨與肩胛骨的形狀<ref name = Remesetal2009a/>。

早期理論普遍認為，[[侏羅紀|侏羅紀]]早中期的蜥腳類恐龍廣泛分布於[[盤古大陸|盤古大陸]]，而沒有達到高密度的個體數量。當盤古大陸在侏羅紀中晚期開始分裂後，各大陸的蜥腳類恐龍開始獨自發展。近年研究則質疑這個地理分隔理論。棘刺龍的發現，有助於古生物學家推論侏羅紀早中期的蜥腳類恐龍演化歷史、地理分佈<ref name = Remesetal2009a/>。

研究人員提出兩種可能原因。這些非洲原始蜥腳類恐龍可能[[平行演化|平行演化]]出進階型特徵。另一種可能則是，棘刺龍的某些類似真蜥腳類特徵，演化自更原始的蜥腳類恐龍，因此非洲的棘刺龍才具有類似亞洲真蜥腳類的進階型特徵。研究人員推論，侏羅紀早中期的蜥腳類恐龍演化過程關鍵可能發生於勞亞大陸的[[北美洲|北美洲]]地區，當時位於氣候潮濕的[[赤道|赤道]]地區<ref name = Remesetal2009a/>。
