'''綜合用途建築物'''（{{lang-en|Composite Building}}），是[[香港|香港]][[1950年代香港|1950年代]]至[[1960年代香港|1960年代]]一種盛行的樓宇種類，通常會商住工業並用。

== 歷史 ==
[[File:HK_Wan_Chai_Road_north.jpg|缩略图]]商住樓宇的地舖]]
「綜合用途建築物」這名稱可追溯到香港《[[建築物條例|建築物條例]]》1950年代用來形容結合工業、商業和住宅用途的樓宇。

綜合用途建築物在香港的1950至1960年代特別蓬勃，有三個主要原因：

# 1949年[[中华人民共和国|中華人民共和國]]成立，香港的[[资本主义|資本主義]]提供了平台資本家在香港發展工業；
# 1950年代初中華人民共和國受到貿易裁制，[[英屬香港|港英政府]]決定把香港的經濟模式由[[轉口港|轉口港]]轉型[[制造业|製造業]]<ref>{{Cite web|url=http://www1.hkej.com/dailynews/articlePrint/id/543903|title=何謂街角樓？|accessdate=2018-08-19|author=卡夫卡|date=2010-12-23|publisher=信報}}</ref>；
# 綜合用途建築物能滿足突然急增的[[住宅|住宅]]和[[第二產業|工業]]的物業需求，而且還在初期階段，所以當時數量有增無減<ref>{{Cite web|url=https://hk.lifestyle.appledaily.com/lifestyle/special/daily/article/20150414/19110994|title=轉角有間老賓館|accessdate=2018-08-24|author=黃碧珊|date=2015-04-14|publisher=蘋果日報}}</ref>。

== 建築規格 ==
[[File:Ma_Tou_Kok_Road_No91-113_201108.jpg|缩略图]]的舊樓群，因為鄰近[[啟德機場|啟德機場]]而只有約7至8層高]]
* 香港《[[建築物條例|建築物條例]]》定義「綜合用途建築物」為「部分是供居住而建」及「部分是為居住以外用途而建」的建築物<ref>{{Cite web|url=https://www.elegislation.gov.hk/hk/cap123|title=第123章 《建築物條例》|accessdate=2018-08-20|author=屋宇署|date=1983|publisher=電子版香港法例}}</ref>；
* 高度等於10層或以上的的建築物需同時加裝[[升降机|升降机]]，因此當時興建的綜合用途建築物通常會以此為限以減低成本<ref>{{Cite web|url=http://hongwrong.com/michael-wolf-cornerhouses/|title=PHOTOGRAPHY – Michael Wolf's Hong Kong Cornerhouses|accessdate=2018-08-20|author=Tom Grundy|date=2012-06-28|publisher=Hong Wrong}}</ref>；
* 因為當時[[啟德機場|啟德機場]]飛機升降的緣故，[[九龍半島|九龍半島]]的各種建築物在高度上亦有限制<ref>{{Cite web|url=https://www.indesignlive.hk/articles/a-piece-of-hong-kongs-history-composite-buildings|title=A Piece of Hong Kong's History: Composite Buildings|accessdate=2018-08-20|author=Janice Seow|date=2016-06-29|publisher=Indesignlive.hk}}</ref>。

== 建築特色 ==
[[File:HK_Tai_Kok_Tsui_Fir_Street_Bedford_Road_corner_building_tong_lau_太安樓_Tai_On_House_29-Dec-2012_(6).jpg|缩略图]]]]
*位於街道交界之間的建築會以圓角設計
*一樓以上各層加建懸臂式[[露臺|露臺]]至行人路邊緣；
* 窗戶中或上下層之間寫有店舖的名稱或掛上招牌；
* 「推鋼窗」，即用窗戶把加建的露臺圍封，然後用作自住或租借<ref>{{Cite book|title=Hong Kong Corner Houses 街頭街尾|last=Wolf|first=Michael|publisher=Hong Kong University Press|year=2010|isbn=9789888028726|location=Hong Kong|pages=18}}</ref>。

== 現況 ==
[[香港製造業|香港製造業]]已經式微，因此現今綜合用途建築物除了有住宅外，通常的商業用途包括：賓館、醫館、美容院、按摩院、宗親會、健身中心等。

現在很多舊區的商住樓宇都成為[[地產發展商|發展商]]收購的目標，以層數更多的純住宅或商業高樓取代，例如[[石硤尾|石硤尾]][[香港深水埗區街道列表#巴域街|巴域街]]及[[市區重建局|市區重建局]]的[[九龍城|九龍城]][[馬頭圍道|馬頭圍道]]/[[春田街|春田街]]重建項目等；亦因此類綜合用途建築物都建於20世紀中期，大部份會出現老化現象，業主會出資進行復修工程，以改善樓宇安全及甚至提升外部形象。

== 流行文化 ==
[[File:Yick_Cheong_Building_and_Yick_Fat_Building._Aug._2015.C.JPG|缩略图]]的益發和[[益昌大廈|益昌大廈]]是高密度的例子]]
不少以香港為題材的電影丶電視及動漫遊戲，尤其是20世紀末期的作品，都會加入這類建築物作背景，而且街道兩旁掛滿招牌及[[霓虹燈|霓虹燈]]，來突顯香港高密度和多元化的城市元素。受影響或啟發的電影例子有《[[银翼杀手|銀翼殺手]]》（1982年）丶《[[攻殼機動隊_(電影)|攻殼機動隊]]》（1995年電影）；遊戲例子有《[[杀出重围|駭客入侵]]》（2000年）丶《[[莎木II|莎木 II]]》（2001年）丶《[[香港秘密警察|香港秘密警察]]》（2012年）；電視動漫例子有《[[金田一少年之事件簿#各媒體改編|金田一少年之事件簿　香港九龍財寶殺人事件]]》（2014年），部份作品更是以[[赛博朋克|數碼龐克]]（Cyberpunk）為主題的科幻故事。
