'''太平军二破江南大营'''是1860年（清[[咸丰_(年号)|咸丰]]十年）2月至5月，[[太平军|太平军]]再次击破围困[[天京|天京]]（今[[南京|南京]]）的清军[[江南大营|江南大营]]的战役。

1858年初，清廷钦差大臣[[和春|和春]]、[[提督|提督]][[張國樑|張國樑]]、署理[[提督|提督]][[周天受|周天受]]率兵围困天京，再次营建[[江南大营|江南大营]]，并设大营于[[天京城|天京城]]东。1860年1月底至2月，清军连续攻占了天京城西北的下关、九湫洲，天京危在旦夕。咸丰十年二月，金陵迟迟未克<ref>《东华续录·咸丰朝》，卷九有咸丰十年二月严旨斥责：“上年和春亲往常州与何桂清商议筹饷，许乃钊又赴浙劝捐。原为速克金陵起见，乃仅克九洲一处，殊不满意。若不及早图维，因循日久，以后饷需难继，和春身为统帅，其能当此重咎耶？”</ref>。在此紧要关头，总理朝政的干王[[洪仁玕|洪仁玕]]与忠王[[李秀成|李秀成]]商议后决定采用“围魏救赵”的策略。

== 兵圍杭州 ==
2月10日，李秀成率大军2万余人抵达南陵，再绕过[[宁国府|宁国府]]（今[[宣州市|宣州市]]），于24日攻占[[广德|广德]]，留部分军队驻守后，率主力轻装奔袭[[杭州|杭州]]。侍王[[李世贤|李世贤]]则率兵由虹星桥东攻[[湖州|湖州]]，掩护李秀成攻打杭州。3月中旬李秀成率精兵偷袭杭州未遂，转而采强攻，仍然未果。李秀成于是在[[清波门|清波门]]外挖掘地道至城墙下并埋上地雷，3月19日晨，城墙被炸塌数丈，太平军攻入了[[杭州城|杭州城]]<ref>《平定粤寇纪略》，卷九，页１２云：“贼之扑杭州，只图分我兵力，以解金陵之围，不意全未设防，致为所陷。是以大营兵到，即踉跄而逃。”</ref>，浙江[[巡撫|巡撫]][[羅遵殿|羅遵殿]]自盡。

江南大营粮饷多來自浙江，浙江也是朝廷财赋重地，杭州危急，清军极其惊慌。清廷闻迅后，急令[[和春|和春]]分兵增援杭州。和春迫于[[咸丰帝|咸丰帝]]严旨屡催，不得不从江南大营分兵1.3万增援杭州，总兵[[刘季三|刘季三]]、游击[[梁成桂|梁成桂]]各带两千人驻扎高淳、东坝，副将[[向奎|向奎]]率兵二千援湖州，总兵[[张玉良|张玉良]]统兵二千由苏州、常州援浙。张玉良部星夜兼程，3月23日，全军赶到杭州。

== 擊破江南大營 ==
4月4日，李秀成见调离江南大营的清军目的达成，遂在城上多树旗帜，虚设疑兵，便悄悄撤出杭州率部挥师北上。4月18日在[[安徽|安徽]]建平（今郎溪）聚齐各路将领，兵分两路回救天京:一路由辅王[[杨辅清|杨辅清]]率领，攻取高淳、东坝、溧水，直扺天京南郊的秣陵关；另一路由[[李世贤|李世贤]]率领，攻占[[溧阳|溧阳]]、[[句容|句容]]后南攻占天京东南的淳化镇。4月27日英王[[陈玉成|陈玉成]]奉天王[[洪秀全|洪秀全]]天京令，则率部由江北来援，太平军号称10万，形成了对江南大营的反包围之势；4月29日夜英王見對岸砲響紅光，10万雄師自[[浦口|浦口]]渡江返回援天京。

5月2日，太平军兵分五路，总攻开始，[[李世贤|李世贤]]部自北门洪山、燕子矶，[[李秀成|李秀成]]部自尧化门，[[刘官芳|刘官芳]]、[[陈坤书|陈坤书]]部自高桥门，[[杨辅清|杨辅清]]部自雨花台，[[陈玉成|陈玉成]]部自善桥方向。五路太平军连日冒雨出击，与清军大战。5月15日，陈玉成部首先突破大营，清军苦心经营多年的江南大营再次被摧毁，[[和春|和春]]負傷，江南提督[[張國樑|張國樑]]潰走[[丹陽|丹陽]]，廣西[[提督|提督]][[張玉良_(清朝)|張玉良]]也敗退，署湖南提督[[周天受|周天受]]敗往[[寧國|寧國]]。天京之围基本解除，並且攻陷丹阳、常州、无锡、苏州、江阴、宜兴、长兴、吴江、平望、嘉兴、昆山、太仓、嘉定、青浦、松江等城<ref>《东南纪略》，页５載：“所有守（常州）城兵勇，已于三十日晚散去，城外各营亦大半溃，丹阳大营溃兵继之。奔无锡者十之七，奔江阴者十之三，所过村庄，辄肆淫掠。惟江阴之申江民团，齐心协力，拒杀甚众。”</ref>。

== 結果 ==
江南大营雖号称五六万众，但虚伍缺额<ref>《陶风楼藏名贤手札》，第五册，邵懿辰致曾国藩函。 </ref>，[[阎敬铭|阎敬铭]]曾批评说：“吾闻江南大营未败时，诸将锦衣玉食，倡优歌舞，其厮养皆吸洋烟，莫不有桑中之喜，志溺气惰，贼氛一动，如以菌受斧。”<ref>[[陈继聪|陈继聪]]：　《忠义纪闻录》，卷三，页１４。</ref>江南大营的再溃，使清廷对于其以往所依恃的绿营军，完全失去了信赖。曾國藩、左宗棠與湘軍的崛起開啟[[同治中興|同治中興]]的序曲。

1860年2月，湘、楚军攻克[[太湖|太湖]]、[[潜山|潜山]]等地，太平军[[陈玉成|陈玉成]]及捻军[[张洛行|张洛行]]等部战败，“军兴数年以来，仅见之大战”，<ref>胡林翼：《克复太湖县城疏》</ref>安庆险象渐现。但天京统帅部下令陈玉成主力前往解天京之圍，陈玉成只好率主力吴如孝、刘昌林部东下。曾国藩遂趁机调湘军曾国荃部圍困安庆，1861年安庆淪陷，1864年天京相繼淪陷。
