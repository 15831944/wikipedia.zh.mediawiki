{{字节单位}}
-{zh-cn:'''太[[字节|字节]]'''或'''太拉字节'''; zh-tw:'''兆[[位元組|位元組]]''';}-（{{lang-en|Terabyte}}，縮寫為'''TB'''）是一种信息計量[[单位|单位]]，现今通常在标示[[硬盘|硬盘]]總容量、或具有大容量的储存介质之储存容量时使用。

== 与其他储存单位的换算 ==
* 1TB = 1,000 (10<sup>3</sup>) [[GB|GB]]
* 1TB = 1,000,000 (10<sup>6</sup>) [[MB|MB]]
* 1TB = 1,000,000,000 (10<sup>9</sup>) [[KB|KB]]
* 1TB = 1,000,000,000,000 (10<sup>12</sup>) [[字节|字节]]

不过，Terabyte往往可以指[[Tebibyte|Tebibyte]]，其换算是：
* 1TiB = 1,024 (2<sup>10</sup>) [[GiB|GiB]] 
* 1TiB = 1,048,576 (2<sup>20</sup>) [[MiB|MiB]]
* 1TiB = 1,073,741,824 (2<sup>30</sup>) [[KiB|KiB]]
* 1TiB = 1,099,511,627,776 (2<sup>40</sup>) [[字节|字节]]

由于换算方法的不同往往造成混淆。例如，一般的硬盘生产厂商采用1,000,000,000,000（10<sup>12</sup>）字节 = 1TB的进制，而 [[Microsoft_Windows|Microsoft Windows]] 等[[操作系统|操作系统]]则采用[[Tebibyte|Tebibyte]]换算，故有其报告的容量比硬盘标示的容量小的情况发生。
