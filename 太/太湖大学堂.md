'''太湖大学堂'''，位于[[江苏省|江苏省]][[吴江市|吴江市]][[七都镇|七都镇]][[庙港|庙港]]，创立者是著名国学大师[[南怀瑾|南怀瑾]]，创建于2006年，为私立教育机构，分为大学和小学（吴江太湖国际实验学校）。<ref name=" souhu" >{{cite web |url= http://news.sohu.com/20090629/n264845801.shtml|title=南怀瑾的太湖大学堂 |accessdate= 2009年6月29日|publisher=搜狐网 |work= |language =zh-cn}}</ref>

== 历史 ==
2006年，南怀瑾在[[江苏省|江苏省]][[吴江市|吴江市]][[七都镇|七都镇]][[庙港|庙港]]的[[太浦河|太浦河]]旁边开创太湖大学堂，[[台湾|台湾]][[薇阁集团|薇阁集团]]董事长[[李传洪|李传洪]]为学堂出资人。<ref name=" souhu" />

== 院系 ==
太湖大学堂分为大学和小学。<ref name="mo " >{{cite web |url= http://finance.ifeng.com/roll/20090926/1285351.shtml|title=太湖大学堂：风声雨声读书声 |accessdate= 2009年9月26日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>

2007年，小学部开学，小学采用全日制寄宿制，且有[[武术|武术]]、[[中草药|中草药]]、[[书法|书法]]等传统文化的课程，以诵读和释义为主。<ref name="mo " />

=== 科目 ===
*[[史学|史学]]，《[[二十四史|二十四史]]》等
*[[经学|经学]]，[[儒家|儒家]]经典
*[[禅学|禅学]]，[[佛教|佛教]]经典
*[[道学|道学]]，[[道教|道教]]经典
*[[经济学|经济学]]
*[[中医学|中医学]]，《[[黄帝内经|黄帝内经]]》等
*[[生命科学|生命科学]]

== 宗旨 ==
传播中国传统文化，同时与现代[[自然科学|自然科学]]、[[人文科学|人文科学]]相结合，发展[[认知科学|认知科学]]与[[生命科学|生命科学]]研究。<ref name="mo " />

== 对外交流 ==
太湖大学堂和[[中国人民大学|中国人民大学]]、[[法国国立东方语言与文化学院|法国国立东方语言与文化学院]]、[[中国科技大学|中国科技大学]]、[[复旦大学|复旦大学]]（儒学文化研究中心）、[[美国管理协会|美国管理协会]]（中国）等大学为合作机构。<ref name=" souhu" />
