'''太田氏'''是[[日本|日本]][[氏族|氏族]]。主要有下列諸多流派：
#[[清和源氏|清和源氏]][[源賴光|賴光]]流太田氏。
#[[清和源氏|清和源氏]][[源賴親|賴親]]流太田氏。
#[[鐮倉幕府|鐮倉]]・[[室町幕府|室町幕府]]中，世襲[[問注所|問注所]]執事的一族。鎌倉幕府問注所執事[[三善康信|三善康信]]之子[[三善康連|康連]]的後代。
#[[藤原氏|藤原氏]][[藤原秀鄉|秀鄉]]流太田氏。

== 清和源氏、賴光流太田氏==
[[File:Japanese_crest_Oota_Kikyou.svg|thumb]]]]
[[攝津源氏|攝津源氏]]的後代，祖先是源賴光的玄孫[[源賴政|賴政]]（源三位賴政）的末子[[源廣綱|源廣綱]]。廣綱的子孫[[源資國|源資國]]住在[[丹波國|丹波國]][[桑田郡|桑田郡]][[太田|太田]]並稱為太田氏。

[[太田資清|太田資清]]仕於[[扇谷上杉氏|扇谷上杉氏]]，資清之子資長（[[太田道灌|太田道灌]]）在[[武藏國|武藏國]][[豐島郡|豐島郡]]築起[[江戶城|江戶城]]（[[東京都|東京都]][[千代田區|千代田區]]），成為[[上杉定正|上杉定正]]的執事而活躍著，但是被定正所殺，因此道灌之子[[太田資康|資康]]仕於[[山内上杉氏|山内上杉氏]]。還有資清・道灌父子在武藏國[[入間郡|入間郡]]築起[[河越城|河越城]]（[[埼玉縣|埼玉縣]][[川越市|川越市]]）。另外亦有他們在武藏國[[埼玉郡|埼玉郡]]築起[[岩槻城|岩槻城]]（埼玉縣[[埼玉市|埼玉市]][[岩槻區|岩槻區]]）的說法（築城者是[[成田正等|成田正等]]的説法比較有力）。道灌的養子[[太田資家|資家]]進入岩槻城並成為岩槻太田家。

子孫則分別仕於[[後北條氏|後北條氏]]、[[里見氏|里見氏]]、[[佐竹氏|佐竹氏]]。服從[[德川家康|德川家康]]而成為大名的太田氏（[[太田資宗|太田資宗]]）支流則不能判斷是否屬於此流。

資高之孫[[太田資元|資元]]跟隨[[佐賀|佐賀]]的[[龍造寺家兼|龍造寺家兼]]，後來成為[[佐賀藩|佐賀藩]]藩主[[鍋島氏|鍋島氏]]的親戚並擔任[[家老|家老]]，變成[[太田鍋島氏|太田鍋島氏]]直到[[幕末|幕末]]為止。

== 清和源氏、賴親流太田氏==
[[大和源氏|大和源氏]]之祖[[源賴親|源賴親]]的後裔，本據在[[攝津國|攝津國]][[嶋下郡|嶋下郡]]太田。[[平安時代|平安時代]]末期登場的有[[太田賴基|太田賴基]]，世世代代以[[太田城|太田城]]為居城。

== 三善流太田氏 ==
[[鐮倉幕府|鐮倉幕府]][[問注所|問注所]]執事[[三善康信|三善康信]]之子[[三善康連|康連]]在鐮倉時代中期得到[[下野國|下野國]][[太田庄|太田庄]]，於是改姓為太田。

後來跟從[[足利尊氏|足利尊氏]]而活躍，移住[[周防國|周防國]][[玖珂郡|玖珂郡]][[椙杜郷|椙杜郷]]的一族改姓[[椙杜氏|椙杜氏]]，並仕於[[大内氏|大内氏]]、[[毛利氏|毛利氏]]，後來成為[[長府藩|長府藩]]筆頭[[家老|家老]]。

== 藤原秀鄉流太田氏 ==
[[藤原秀鄉|藤原秀鄉]]的後裔，本據是在武藏國太田莊（現在的[[久喜市|久喜市]]、[[加須市|加須市]]、[[羽生市|羽生市]]、[[埼玉市|埼玉市]][[岩槻區|岩槻區]]）的一族。出了[[小山氏|小山氏]]、[[下河邊氏|下河邊氏]]、[[大河戶氏|大河戶氏]]等族人。

祖先是被稱為太田大夫・太田別當的[[太田行尊|太田行尊]]（行隆）。開發太田莊並一直傳承至[[太田行政|行政]]━[[太田行光|行光]]━[[太田行廣|行廣]]━[[太田行朝|行朝]]。行朝跟隨[[源賴朝|源賴朝]]參加[[野木宮合戰|野木宮合戰]]，因此被封到太田莊。但是在[[建久|建久]]5年（1194年）6月，行朝殺傷在[[久伊豆神社|久伊豆神社]]的神人，神人一側因此抗議，導致賴朝派遣[[二階堂行光|二階堂行光]]來仲裁的事態，行朝在太田莊的莊司職被沒收。以後[[小山氏|小山氏]]等庶家開始活躍，但是一族之長的秀鄉流太田氏則沒落（《[[久喜市史|久喜市史]]》、《[[鷺宮町史|鷺宮町史]]》）。
