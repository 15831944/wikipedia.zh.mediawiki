'''太田強戶停車區'''（{{lang|ja|おおたごうどパーキングエリア}}）是[[群馬縣|群馬縣]][[太田市|太田市]][[北關東自動車道|北關東自動車道]]預定設置的[[公路服務區|停車區]]。

== 道路 ==
* [[北關東自動車道|北關東自動車道]]

== 設施 ==
* 建設中。預定以上下線集約型方式修建，一併設置[[智慧型交流道|智慧型交流道]]。

== 設置經緯 ==
本停車區與智慧型交流道當初預計與北關東自動車道通車時同時啟用，但是因為建設費壓縮的關係，停車區的建設計劃在2003年被凍結<ref name="jomo20110712">[http://www.raijin.com/news/a/2011/07/12/news01.htm 北關東道・太田藪塚～太田桐生間的智慧型交流道，凍結計畫再度啟動] {{webarchive|url=https://web.archive.org/web/20150924084431/http://www.raijin.com/news/a/2011/07/12/news01.htm |date=2015-09-24 }} 上毛新聞 2011年7月12日</ref>，智慧型交流道的計畫也在2009年政權交替時遭到凍結<ref name="jomo20110712" />。NEXCO東日本以北關東自動車道的交通量和[[波志江停車區|波志江停車區]]、佐野停車區（暫稱，現在的[[出流原停車區|出流原停車區]]）的混雜狀況作為是否設置的判斷標準<ref>[https://archive.is/20120906140024/http://www.pref.gunma.jp/07/n04810026.html はばたけ群馬・縣土整備地域計畫（太田地域、桐生・綠地域、館林・邑樂地域））（案）相關意見募集結果（Archive.isによるアーカイブ）] 群馬縣東部縣民局 2009年3月27日發表</ref>。之後北關東道的交通量大大地超過預期，因此NEXCO東日本針對周邊停車區的混雜狀況開始調查，而群馬縣與太田市也提出了再度建設的要求<ref>[http://www.raijin.com/news/a/2011/07/12/news01.htm 北關東道・太田藪塚～太田桐生間的智慧型交流道，凍結計畫再度啟動] {{webarchive|url=https://web.archive.org/web/20150924084431/http://www.raijin.com/news/a/2011/07/12/news01.htm |date=2015-09-24 }} 上毛新聞 2011年7月12日</ref>。2013年發下智慧型交流道的連結許可<ref>{{Cite web |url=http://www.mlit.go.jp/report/press/road01_hh_000360.html |title= 關於交流道之追加設置 |date=2013-06-11 |publisher=國土交通省道路局 |accessdate=2013-06-11}}</ref><ref>[http://www.city.ota.gunma.jp/005gyosei/0100-001tosi-plan/01news/otasmart-ic.html （暫稱）決定太田智慧型交流道事業化！] {{webarchive|url=https://web.archive.org/web/20140102192718/http://www.city.ota.gunma.jp/005gyosei/0100-001tosi-plan/01news/otasmart-ic.html |date=2014-01-02 }} - 太田市、2013年6月12日。</ref>，同時間進行停車區的整備<ref name="info">{{PDFlink|[http://www.jehdra.go.jp/pdf/1162.pdf 高速道路利便施設の連結に関する情報提供（東日本高速道路株式會社管理分）]}}（日本高速公路保有・債務返還機構、2013年12月17日現在、2014年1月2日閲覧）</ref>。
