'''太平區'''是位於[[中華民國|中華民國]][[臺中市_(直轄市)|臺中市]]偏東南的[[市轄區|市轄區]]。人口為臺中市第四高，擁有19萬人口，次於[[北屯區|北屯區]]、[[西屯區|西屯區]]和[[大里區|大里區]]；面積為該市第二大轄區（最大的為[[和平區_(台灣)|和平區]]），約120.7[[平方公里|平方公里]]。

該區大部分為[[丘陵|丘陵地]]，人口集中於西部的[[臺中盆地|臺中盆地]]邊緣，又以西北方[[中山路_(太平區)|中山路]]沿線最為密集。主要農作物有[[枇杷|枇杷]]、[[龍眼|龍眼]]、[[荔枝|荔枝]]、[[香蕉|香蕉]]等，其中以枇杷最為有名，並於每年三、四月舉辦[[太平枇杷節|太平枇杷節]]。

太平區東邊以[[雪山山脈|雪山山脈]]支脈[[大橫屏山|大橫屏山]]系（本區最高峰，海拔1206公尺）和[[臺灣省|臺灣省]][[南投縣|南投縣]][[國姓鄉|國姓鄉]]為鄰，東北部以[[中央山脈|中央山脈]]支脈[[頭嵙山|頭嵙山]]系和[[新社區|新社區]]接壤，西部以[[旱溪|旱溪]]、[[大里溪|大里溪]]與[[東區_(臺中市)|東區]]相隔，北以[[廍仔坑溪|廍仔坑溪]]與[[北屯區|北屯區]]為界，西南部以[[三汀山|三汀山]][[丘陵|丘陵]]（部分為[[九九峰|九九峰]]自然保留區域）和[[大里區|大里區]]、[[霧峰區|霧峰區]]緊鄰。全區4/5面積為山地區域（其中60%為三汀山丘陵區），95%人口集中在該區西部的[[台中盆地|台中盆地]]上。

境內的河川主要有[[旱溪|旱溪]]、[[大里溪|大里溪]]（支流:廍仔坑溪）、[[頭汴坑溪|頭汴坑溪]]（支流:[[北坑溪|北坑溪]]、[[東汴坑溪|東汴坑溪]]、[[濁水坑溪|濁水坑溪]]、[[清水坑溪|清水坑溪]]）以及[[草湖溪|草湖溪]]等<ref>[http://www.taiping.gov.tw/taiping-03.asp 太平區公所-地理位置] {{webarchive|url=https://web.archive.org/web/20081228125955/http://www.taiping.gov.tw/taiping-03.asp |date=2008-12-28 }}</ref>，均為[[大肚溪|大肚溪]]支系[[大里溪|大里溪]]上游；位於一江橋東北方500公尺山上的大潭仔，為本區唯一的[[湖泊|湖泊]]。氣候為[[亞熱帶|亞熱帶]][[季風氣候|季風氣候]]，年平均溫度約為攝氏20-22.3度，年平均[[相對溼度|相對溼度]]為77%，年平均降雨量為1,642（西部平原區）至2,333（東部山區）公釐。


[[清朝|清朝]]之前，太平古名傳聞很多，有「鳥松頭」、「鳥榕頭」、「鳥銃頭」和「蔗種頭」等。其中「鳥榕頭」與「鳥松頭」皆源自當時墾首[[太平林家|太平林家]]大宅院（[[九二一地震|九二一地震]]震毀）門前兩側所種的鳥[[松樹|松樹]]（雀[[榕樹|榕樹]]）而得名<ref>白棟樑，《鳥榕頭與它的根－太平市誌》，1998年，頁92</ref>。而「太平」的由來有兩種說法：
* 清領時期太平屬於原住民的勢力範圍，清廷將之劃入土牛界外（包含今日臺中市九期重劃區範圍），為禁止漢人進入開墾之化外之地，故有番仔溝、番仔路（今北太平）…等地名，太平林家（[[霧峰林家|霧峰林家]]支系）在[[三汀山|三汀山]]搭設望高寮，抵禦原住民，後因雙方和睦相處，而改名太平。
* 另有一說，[[甲午戰爭|甲午]]戰敗，清廷割讓臺灣，日軍抵台時，在中部各地都受到民兵頑強抵抗，開達太平時，並未遭受任何抗拒，故改名太平。<ref>太平市公所編印:《太平市志》（上冊）開墾篇，2006年出版</ref>。

清朝時代，與現在[[台中市|台中市]]的中、東、南區全部，西、北二區的部份以及[[大里區|大里區]]的全部和[[烏日區|烏日區]]的一部份合屬「藍興堡」。西元1895年（清光緒21年）前，太平屬[[臺灣府|臺灣府]][[臺灣縣_(1887年)|臺灣縣]][[藍興堡|藍興堡]]。1896年[[日本|日本]]統治台灣後，改屬[[台中縣|台中縣]][[台中辦務署|台中辦務署]]藍興堡。1901年變更為[[台中廳|台中廳]]廳直隸下的藍興堡部份區域<ref>廖瑞銘等，《太平市志》－政事篇，太平市公所發行</ref>。1920年實施五州制，改太平為大平，'''-{[[大平庄|大平庄]]}-'''隸屬[[台中州|台中州]][[大屯郡|大屯郡]]管轄，下轄大平、番仔路、三汴、車籠埔、頭汴坑等五個[[大字_(行政區劃)|大字]]<ref name=歷史/>。

1945年[[中華民國政府|中華民國政府]]接收[[日本帝國|日本帝國]][[台灣總督府|台灣總督府]]所轄區域後，改為[[臺灣省|臺灣省]]臺中縣大屯[[縣轄區_(中華民國)|區]]'''太平鄉'''（此時臺中縣包含今臺中市、[[彰化縣|彰化縣]]、[[南投縣|南投縣]]轄區，但不包括當時的省轄臺中市及彰化市，縣治設於[[員林市|員林]]）。1950年廢區署，再改為臺中縣太平鄉。1996年因人口達15萬，依法改制為'''太平市'''，成為當時全台灣面積最大的縣轄市<ref>[http://nrch.cca.gov.tw/twpedia.php?id=4017 台灣大百科全書-太平市]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2005年，依法將原來里別由19[[里_(消歧義)|里]]增加為39里。2010年12月25日臺中縣市合併為[[直轄市_(中華民國)|直轄市]]，同時改制為臺中市'''太平區'''<ref name=歷史>{{Cite web|url=http://www.taiping.taichung.gov.tw/ct.asp?xItem=224970&ctNode=9400&mp=138010|title=歷史由來|accessdate=2017-06-16|author=|date=2017-04-05|publisher=臺中市太平區公所|language=zh-tw|archiveurl=https://web.archive.org/web/20150403011212/http://www.taiping.taichung.gov.tw/ct.asp?xItem=224970&ctNode=9400&mp=138010|archivedate=2015-04-03|dead-url=yes}}</ref>。

== 舊地名 ==
*'''鳥榕頭、太平、大平'''：本區最初之舊稱，聚落位置大約在現今中興路周遭，靠近永豐路為下太平，太平國小處則為上太平。詳細請見上方沿革。
*'''公館崎'''：太平里一帶，靠近[[東區_(臺中市)|東區]]。
*'''下廍子'''：太平路與東區接壤之處，目前還存有下廍巷此路名，當時設立[[糖廍|糖廍]]於該處（太平早期產甘蔗），與其北方一些的廍子相對而來。
*'''三叉'''：位於當今東平里、永平里一帶，為早期輕便車從山區運送農特產品至集中市場的行經路線。
*'''坪林'''：為眷村聚集處，有國軍坪林營區、[[國軍臺中總醫院|國軍臺中總醫院]]等軍方設施，後坪林營區單位大量撤出，現在主要為[[國立勤益科技大學|國立勤益科技大學]]的教職員生活動的地方。
*'''廍子坑口'''：於[[臺中市私立華盛頓高級中學|華盛頓中學]]之廍子溪出山處的坑口。
*'''溪洲'''：現今宜欣社區之處。早期該地為大坑溪與廍子坑溪沖積的[[河洲|河洲]]，土質砂礫居多，肥力與保水性差，不利種植人口稀少，為太平發展最差的地方，多處地方為墓園，後工商業興起，中小企業進駐與宜欣社區建設，形成北太平發展甚早的重要聚落。
*'''番仔路'''：現今的[[臺中市立新光國民中學|新光國中]]一帶，為清廷土牛溝經過之處。周邊亦存有名為番仔溝的小溪。
*'''新坪'''：太平區有兩處新坪，一為現今新坪里、田園新村一帶，另一處則在車籠埔北方、冬瓜山附近之地區。
*'''車籠埔'''：
*'''七星'''：光隆里一帶、車籠埔的南方，早期該地門牌以七星為名，後來才改為路名標示。
*'''黃竹坑'''：位於[[草湖溪|草湖溪]]上游，大里竹子坑再入山之處。現今人口不足千人，係因原省府欲興造建民水庫，準遷出不准遷入，但發生[[九二一大地震|九二一大地震]]後，計畫完全停止。
*'''茶寮'''：一江橋過頭汴坑溪入山之處，為古時作為平地與山區的中繼站，擺放茶水供旅人歇息，再繼續入山。
*'''頭汴、二汴、三汴、東汴'''：「汴」之字義為擋水之匝門，因頭汴坑溪出山後轉向南方，故舊時自太平山區築圳道引水至位於溪邊北部坪林一帶灌溉，依序經過三處擋水的地方（即古時聚落之處）。除今日較耳熟能詳的頭汴外，二汴的位置約在中山路和平橋洪厝，三汴則再往前推進至中山路一段坪林一帶。而東汴則為頭汴再入山更東邊的地區。

== 行政 ==
[[File:太平區公所.jpg|thumb]]
=== 區公所 ===
太平區公所直屬於[[臺中市政府|臺中市政府]]。首長為官派[[區長|區長]]，並由主任秘書襄理公所相關事務，下設民政課（設有調解委員會、管理區里民活動中心）、社會課（設有長青學苑）、農業及建設課、公用課、人文課以及秘書室、人事室、會計室、政風室等，共計5課4室。

目前的辦公大樓興建於1980年，因建築體老舊，且該處街巷路幅過小行車停車困難，洽公不便，因此有遷建的打算，預計撥用位於祥順路一段與宜昌東路160巷口的文中三用地重建，距離太平戶政、地政事務所僅1公里多，未來落成後民眾到各機關洽公將更方便<ref>[http://news.ltn.com.tw/news/local/paper/1136286 〈中部〉太平區公所遷建 教局允供地]</ref><ref>[http://news.ltn.com.tw/news/life/breakingnews/2505220 台中太平新聯合行政中心 確定蓋在這裡！]</ref>。

== 經濟發展 ==
太平區可概分為北、南、東三區域，一般簡單概述約用[[東平路_(太平區)|東平路]]與為界，實際上由自然分界－廍子坑溪與坪林排水支線，分出南北太平，而[[大里溪|大里溪]]支流[[頭汴坑溪|頭汴坑溪]]再劃出東太平，為山地與小部分的山麓地帶。

;商業
本區西側之台中盆地邊緣為主要人口活動區，是主要商業活動與住宅區，尤以北太平占本區近50%的人口。北太平商業活動集中於[[中山路_(太平區)|中山路]]、[[樹孝路|樹孝路]]地區；南太平則以[[中興路_(太平區)|中興路]]、[[太平路_(太平區)|太平路]]為主。

;工業
工業產業佔本區總就業人口43%，主要分佈在東太平南方與[[大里區|大里區]]相連的[http://www.moeaidb.gov.tw/iphw/dali/ 大里工業區]、南太平東北方的太平工業區、以及南太平西南方的工業地區，以生產機械設備、鋁製品及塑膠製品為主。目前[[臺中市政府|臺中市政府]]於太平區西南側[[台糖|台糖]]土地正著手規劃[[太平產業園區|太平產業園區]]的設置。

;農業
東部山區為主要農業區，設有[[頭汴坑休閒農業區|頭汴坑休閒農業區]]，主要農作物特產有[[枇杷|枇杷]] 、[[龍眼|龍眼]]、[[荔枝|荔枝]]、[[香蕉|香蕉]]等，尤其以每年三、四月的太平枇杷節最為盛大，並設有「大湖桶枇杷專業區」。

== 景點 ==
[[File:太平買菸場.jpg|thumb]] 
[[File:護國清涼寺.JPG|thumb]]
[[File:太平蝙蝠洞.JPG|thumb]]

太平區全區4/5面積為山地區域，觀光旅遊資源幾乎集中在東部山區，區公所將其劃成頭汴坑、酒桶山、車籠埔、草嶺及竹仔坑山系。[[太平枇杷節|太平枇杷節]]為太平區每年會舉辦的農特產文化祭。

;文化資產
*[[吳鸞旂墓園|吳鸞旂墓園]]（吳家花園）：直轄市定古蹟
*[[太平買菸場|太平買菸場]]：臺中市產業與民生設施類歷史建築
*[[九九峰|九九峰]]：臺中市自然地景（範圍及太平、[[霧峰|霧峰]]、[[草屯|草屯]]、[[國姓|國姓]]）

;山區景點
*頭汴坑山系：[[頭汴坑|頭汴坑]]休閒農業區、草嶺步道、咬人狗坑步道、望高寮
*酒桶山系：[[暗影山|酒桶山]]風景區、[[仙女瀑布|仙女瀑布]]、赤崁頂
*[[車籠埔|車籠埔]]山系：[[一江橋|一江橋]]、南國巷小桂林、大潭仔、太平廣興宮、太平古農莊文物館
*草嶺山系：蝙蝠洞、[[護國清涼寺|護國清涼寺]]
*竹仔坑山系：黃竹坑、[[草湖溪|草湖溪]]、[[林文察|林文察]]衣冠冢
