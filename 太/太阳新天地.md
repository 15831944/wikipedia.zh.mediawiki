'''太阳新天地''' 是[[广州|广州]]一家购物中心，位于天河区<ref>[http://news.dichan.sina.com.cn/2012/01/11/431581.html 太阳新天地——天河CBD最具创新时尚购物中心璀璨亮相. 新浪地产. 2012-01-11发布. 2013-12-25 检索]</ref>的珠江新城<ref>[http://www.winshang.com/zt/taiyangxintiandi/ 广州太阳新天地——百大标杆商业项目. 赢商网. 2011年发布. 2013-12-25 检索]</ref>中央商务区。

== 简介 ==
于2012年完成建设，太阳新天地总楼面面積約15万平方米，是广州珠江新城一带最大的单体购物中心<ref>[http://www.ycwb.com/ePaper/xkb/html/2013-12/20/content_329467.htm?div=-1 最具影响力商业品牌. 金羊网. 2013-12-20发布. 2013－12－25 检索] {{webarchive|url=https://web.archive.org/web/20131221005933/http://www.ycwb.com/ePaper/xkb/html/2013-12/20/content_329467.htm?div=-1 |date=2013-12-21 }}</ref>。
太阳新天地是由美国建筑商Altoon Partners以及英国室内设计商{{link|貝諾|Benoy}}共同设计的。目前由广州百嘉信集团负责管理。

太阳新天地有80多家品牌包括CLASS CAVALLI<ref>[http://www.pinterest.com/pin/514606694892617478/ 太阳新天地购物中心 Class Cavalli 店. Pinterest. 2013-12-25 检索]</ref><ref>[http://www.dianping.com/shop/15104545/map Class Cavalli的地图. 大众点评网. 2014-01-25 检索]</ref>、{{link-en|德克·毕盖帕克|DIRK BIKKEMBERGS}}、H&M<ref>[http://www2.hm.com/zh_cn/customer-service/shopping-at-hm/store-locator.html#CN-CN0414 H&M 广州太阳新天地店. H&M门店信息. 2015-10-15 检索] {{Wayback|url=http://www2.hm.com/zh_cn/customer-service/shopping-at-hm/store-locator.html |date=20150903214132 }}</ref>、NEW LOOK、[[無印良品|無印良品]]、[[优衣库|优衣库]]<ref>[http://www.uniqlo.com/cn/shop/shop322.html UNIQLO 广州太阳新天地店. 优衣库官方网站. 2013-12-25 检索]</ref>，及{{link-ja|美思佰乐|マックスバリュ東海}}超市<ref>[http://news.dichan.sina.com.cn/2012/07/25/534502.html 日本高端精品超市美思佰乐Maxvalu 中国一号店棋落广州CBD太阳新天地. 新浪房产. 2012-07-25发布. 2013-12-25 检索]</ref>。[[中影国际影城|中影国际影城]]的广州旗舰店也位于太阳新天地<ref>[http://www.zyhyth.com/about.jsp?classid=7 中影国际影城（太阳新天地店）. 中影国际影城. 2013-12-25 检索] {{webarchive|url=https://web.archive.org/web/20131225131243/http://www.zyhyth.com/about.jsp?classid=7 |date=2013-12-25 }}</ref>。

== 商场位置 ==

太阳新天地位于广州市珠江新城马场路36号<ref>[http://map.baidu.com/?newmap=1&s=inf%26uid%3D73a39ae4037835fb4d776ed1%26wd%3D天河区马场路36号太阳新天地+%26all%3D1%26c%3D257from=alamap&tpl=mapdots. 太阳新天地购物中心西南门. 百度地图. 2013-12-25 检索]</ref>。附近有珠江公园、广州马会与暨南大学<ref>[http://www.jnu.edu.cn/xxgk.html 暨南大学学校概况. 2013-12-25 检索] {{webarchive|url=https://web.archive.org/web/20131221183231/http://www.jnu.edu.cn/xxgk.html |date=2013-12-21 }}</ref>。 商场的公共交通便利。购物中心门外有公交车站、广州地铁5号线的潭村站。此外，太阳新天地的停车场有1200多个车位。

太阳新天地的周边有广粤天地、 广州易通大厦、誉峰公寓等住宅楼盘。太阳新天地的周边也有甲级办公楼、国际大酒店以及酒店试公寓包括联通广场、广东农信大厦、富力科讯大厦、云来斯堡酒店、马会酒店<ref>[http://www.daodao.com/HotelsNear-g298555-d3363635-Happy_Valley-Guangzhou_Guangdong.html#02,1387953087888,bc_restaurants:-1,bc_attractions:-1,th_name:S太阳新天地购物中心,th_type:SATTRACTION,th_value:S3363635,th_coords:S23.12314|113.34315,th_url:S/Attraction_Review-g298555-d3363635-Reviews-Happy_Valley-Guangzhou_Guangdong.html,poi:S23.12314|113.34315,newPoi:S1,l1price:S800|999999,distance:S,distanceSelect:S,cat:S-1  太阳新天地购物中心附近酒店. 到到网. 2013-12-25 检索]</ref>、[[W酒店|W酒店]]、广州丽思卡尔顿、广州国际金融中心雅诗阁公寓等。
