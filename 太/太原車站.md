'''太原車站'''，曾暫名'''北屯車站'''，位於[[臺中市|臺中市]][[北屯區|北屯區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺中線|臺中線]]（山線）的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
原車站位於太原路北側。為配合[[臺中都會區鐵路高架捷運化計畫|臺鐵高架捷運化]]計畫，於太原路與南京東路、東光路口南側興建的高架車站，2016年10月16日啟用，設置兩座島式月台，唯初通車時僅啟用中間兩條正線，仍未設置自動驗票閘門，舊站於2017年3月拆除，側線直到2018年10月28日才啟用。

== 歷史 ==
*2002年11月22日啟用，設站時為[[簡易站|簡易站]]，興建時以所在行政區[[北屯區|北屯區]]暫名為「'''北屯車站'''」，通車前因鄰近太原路更名'''太原車站'''。
*2015年6月30日：啟用多卡通刷卡機。
*2016年10月16日：高架站體第一階段完工並先行通車，各級列車改行駛於高架軌道，本站變更為乙種簡易站。
*2017年3月：舊車站站體拆除。
*2018年10月28日，本站西第二主正線啟用。

== 營運狀況 ==
*目前為乙種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]，主要停靠[[臺鐵區間車|區間車]]。
*指定由台中站管理。
*區間快車：僅2042次。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。
*台鐵所有簡易站之中，進出站人數年年維持在第三名（第一、二名分別為[[汐科車站|汐科站]]、[[大橋車站|大橋站]]）。
