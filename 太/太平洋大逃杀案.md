'''太平洋大逃杀案'''，又称'''鲁荣渔2682号惨案'''、'''“鲁荣渔2682”号案'''，指2011年“鲁荣渔2682”号渔船出海时，33名船员内部争斗，最终11人存活、16人死亡、6人失踪的案件。存活的11人均被判刑，其中5人判处死刑。

== 背景 ==
“鲁荣渔2682号”是大洋鱿钓船，船长三四十米，隶属山东[[荣成市鑫发水产食品有限公司|荣成市鑫发水产食品有限公司]]，自2000年开始出外海作业。船员们多数是亲戚、熟人。{{r|南都}} 船员中存在刘贵夺为首的“东北帮”、包德格吉日胡（以下简称“包德”）为首的“内蒙帮”、来自大连的管理层。船员多为临时招募，他们在工作中发现工作强度大、时间长，讨论说收入可能达不到公司的所谓“保底工资”。<ref name=申欣旺>{{Cite web|url=http://news.ifeng.com/shendu/zgxwzk/detail_2013_08/01/28158391_0.shtml|title=海上生死劫|accessdate=|author=申欣旺|date=2013-08-01|publisher=中国新闻周刊|archive-url=https://web.archive.org/web/20170718114610/http://news.ifeng.com/shendu/zgxwzk/detail_2013_08/01/28158391_0.shtml|archive-date=2017-07-18|dead-url=no}}</ref>

== 案情 ==
2010年12月27日，“鲁荣渔2682号”载33名船员出海驶向[[秘鲁|秘鲁]]。2011年6月16日，位于智利附近海域的渔船在向所属公司现场指挥船报告作业位置后失踪。7月9日晨，北京海事卫星地面站电话联系到渔船船长李承权，获悉渔船安全，在夏威夷附近海域，正驶回国内。此后渔船再度失联。7月25日晨，中国渔业管理部门接到渔船求救信号。{{r|南都}} 经日方、中方先后救援，8月12日，渔船抵达山东[[荣成市|荣成市]][[石岛港|石岛港]]。{{r|新京}}

根据起诉书、判决书、裁定书，案情如下：{{r|南都|新京|新京2}}

2011年6月16日23时许，刘贵夺指使黄金波、王鹏破坏船上通讯设备，随即同包德、双喜等人持刀、棍闯入船长室，威逼船长李承权返航，李承权被迫用卫星导航设定回国返航路线。伙食长夏琦勇持刀要求见船长，被姜晓龙、刘贵夺、刘成建捅刺和击打，抛入海中。

7月20日晚至21日晨，刘贵夺召集姜晓龙等人，先杀害疑有反抗迹象的六人，再杀害另外三人。九人先后被杀害后抛海。期间一人失踪。

7月23日至24日，刘贵夺等人为筹集偷渡到日本的经费并制造被劫假象，指使剩余船员通过卫星电话，以生病、受伤为由，让各自家人向其银行卡上打款。

7月24日，黄金波向刘贵夺告密称包德等人意欲谋反。此前李承权、崔勇、段志芳为求自保，主动要求加入刘贵夺团体。当晚，刘贵夺指使黄金波、王鹏、李承权、崔勇持刀捅刺包德，包德被迫跳海。随后双喜、戴福顺、包宝成、单国喜、邱荣华五人也被逼跳海。

7月25日凌晨4时许，渔船机舱进水失去动力（这时有一人失踪。刘贵夺认为进水原因是失踪者打开海底总阀），船体倾斜。四位船员跳上自制木筏试图逃走，但木筏漂回渔船附近。刘贵夺、李承权等人朝木筏扔铁坠，四人被迫跳海，其中一人求救后被拉上渔船，李承权提出段志芳、项立山没“沾血”，刘贵夺遂指使段、项二人将他沉海杀害。

由于渔船漏水，他们重新打开通讯设备求救。

== 审判 ==
2012年11月14日，[[山东省|山东省]][[文登市人民法院|文登市人民法院]]开庭审理本案。<ref name=南都>{{Cite news|url=http://news.sina.com.cn/c/2012-11-30/040025695521.shtml|title=“鲁荣渔2682号”的秘密|author=张国栋|date=2012-11-30|work=南方都市报|accessdate=|archive-url=https://web.archive.org/web/20130715005044/http://news.sina.com.cn/c/2012-11-30/040025695521.shtml|archive-date=2013-07-15|dead-url=no}}</ref>2013年7月19日宣判，11名船员均被判有罪，刘贵夺、姜晓龙、刘成建、黄金波、李承权5人死刑，王鹏[[死刑缓期执行|死缓]]，冯兴艳无期徒刑，梅林盛、崔勇2人有期徒刑15年，项立山5年，段志芳4年。<ref name=新京>{{Cite news|url=http://epaper.bjnews.com.cn/html/2013-07/20/content_451456.htm?div=0|title=“鲁荣渔2682”号案宣判 5死1死缓|author=朱柳笛|date=2013-07-20|work=新京报|accessdate=|archive-url=https://web.archive.org/web/20190108025647/http://epaper.bjnews.com.cn/html/2013-07/20/content_451456.htm?div=0|archive-date=2019-01-08}}</ref>

== 评论 ==
腾讯今日话题认为，第一个死者的出现会让犯罪者产生“一不做二不休”的想法；封闭空间内犯罪者高度紧张；犯罪团伙讲究“你我”，非同乡者互不信任；目标（保命）无法实现时，人们会寻求替代方案。今日话题认为，远洋渔船矛盾丛生的原因是随意的船员选拔、与辛苦远洋捕捞极不相称的待遇。应当加强立法、执法规范，改变远洋渔业生态。<ref>{{Cite web|url=https://view.news.qq.com/zt2013/ycca/index.htm|title=远洋渔船大逃杀：11船员为何杀害22名同伴|accessdate=|author=|date=2013|publisher=腾讯今日话题|archive-url=https://web.archive.org/web/20181027190055/http://view.news.qq.com/zt2013/ycca/index.htm|archive-date=2018-10-27|dead-url=no}}</ref>
