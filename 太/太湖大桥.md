'''太湖大桥'''是中国[[江苏省|江苏省]][[苏州市|苏州市]][[吴中区|吴中区]]的一座公路跨湖大桥，自[[胥口镇_(苏州市)|胥口镇]][[渔洋山|渔洋山]]南侧起，途经[[长沙岛|长沙岛]]、[[叶山岛|叶山岛]]，最后到达到[[西山_(苏州)|西山岛]]。太湖大桥分为1号桥（胥口至长沙岛段）、2号桥（长沙岛至叶山岛段）和3号桥（叶山岛至西山岛段）。线路全长7.18公里，其中三段桥梁长4308米。大桥建成后极大方便了进出西山岛的人员和物资，为地区快速发展提供了重要的作用。太湖大桥还是中国内湖最长的桥梁<ref>{{Cite web|url=http://www.china.com.cn/aboutchina/txt/2009-10/23/content_18751809.htm|title=1994年10月25日 我国内湖第一长桥太湖大桥通车|accessdate=2018-03-30|author=|date=2009-10-23|work=中国网|publisher=|language=zh-CN}}</ref>。

== 建设 ==
西山岛为[[太湖|太湖]]中最大的一个[[島嶼|岛屿]]，也是中国内湖最大的岛屿<ref>{{Cite web|url=http://news.163.com/16/0929/07/C2480FBM00014AED.html|title=醉美太湖西山岛  健康生态休闲小镇|accessdate=2018-03-30|author=|date=2016-09-29|last=|work=网易|publisher=|language=zh-CN}}</ref>，总面积79.82平方公里。因四面环水，自古以来只能以[[船运|船运]]与外界交通。直至1983年，西山岛与[[东山镇_(苏州市)|东山镇]]间开通[[轮渡|轮渡]]，开始有[[長途客運|长途客车]]通过轮渡开行于西山岛与苏州市区之间。

1987年，[[吴县|吴县]]政府开始研讨太湖大桥的建设[[可行性研究|可行性]]。1989年开始进行地质勘探等前期准备活动<ref name=":1">{{Cite book|url=http://www.dfzb.suzhou.gov.cn/database_books_detail.aspx?bid=4544|title=《西山镇志》|last=|first=|publisher=苏州大学出版社|year=2001.06|isbn=9787810378239|location=|pages=|chapter=第十一章 第二节}}</ref>。1991年11月，[[江苏省|江苏省]]计划经济委员会批复太湖大桥项目立项<ref name=":1" />。1992年敲定由[[同济大学|同济大学]]建筑设计院提供的施工方案<ref>{{Cite web|url=http://www.xinhuanet.com/science/2015-10/25/c_134744213.htm|title=1994年10月25日 太湖大桥通车|accessdate=2018-03-30|author=科普中国|date=2015-10-25|work=|publisher=中国科协|language=zh-CN}}</ref>。1992年10月27日正式动工，1994年6月30日，太湖大桥3号桥率先合拢，1994年10月25日全线建成通车。

太湖大桥由[[吴县|吴县]]政府主导，项目总投资1.15亿元，其中社会各界[[捐款|捐款]]共5984万元<ref name=":0">{{Cite book|title=苏州市志（1986-2005）第一册|last=|first=|publisher=江苏凤凰科学技术出版社|year=2014.12|isbn=9787553738857|location=|pages=P.537|chapter=第11卷}}</ref>，西山岛当地群众和在岛各单位共筹资1000万元<ref name=":1" /><ref>{{Cite web|url=http://js.ifeng.com/a/20171023/6087582_0.shtml|title=中国内湖第一长桥 太湖大桥建桥记|accessdate=2018-03-30|author=|date=2017-10-23|work=凤凰网江苏|publisher=|language=zh-CN}}</ref>。

随着社会经济发展，原桥日渐不能满足进出西山岛的客流需求，且需要进行较大规模的整修，2013年苏州市决定开展西山岛进出通道扩建工程。工程包括沿原太湖大桥旁新建一座同等规模同等桥型的复线桥，新桥宽17米，与老桥并排相距仅在1米以内，增设管廊系统和雨水收集装置。复线桥建设完成后，整个进出西山岛通行能力将提升接近3倍<ref>{{Cite web|url=http://news.ifeng.com/gundong/detail_2013_07/08/27237767_0.shtml|title=太湖大桥“姊妹桥”年内开建|accessdate=2018-03-30|author=|date=2013-07-08|work=凤凰网|publisher=|language=zh-CN}}</ref>。同时配套建设长3.3公里的渔洋山隧道，以提升太湖大桥出入口的吞吐能力。这一工程项目总长11.99公里，总投资12.1亿元<ref>{{Cite web|url=http://roll.sohu.com/20130707/n380911398.shtml|title=太湖大桥姊妹桥马上开建了|accessdate=2018-03-30|author=|date=2013-07-07|work=搜狐|publisher=}}</ref>。2013年10月14日，新桥开工<ref>{{Cite web|url=http://news.163.com/13/1015/10/9B7IM7S000014AED.html#from=relevant#xwwzy_35_bottomnewskwd|title=苏州开建太湖大桥“姊妹桥”|accessdate=2018-03-30|author=|date=2013-10-15|last=|work=网易|publisher=|language=zh-CN}}</ref>。2016年12月20日，新桥正式通车，同时老桥进行全封闭维护大修<ref>{{Cite web|url=http://sz.sina.com.cn/news/2016-12-21/detail-ifxytqaw0196295.shtml|title=苏州太湖大桥新桥启用 预计明年双线通车|accessdate=2018-03-30|author=|date=2016-12-21|work=新浪苏州|publisher=|language=zh-CN}}</ref>。2017年9月30日，随着老桥修缮完毕，太湖大桥复线工程及渔洋山隧道正式全线通车<ref>{{Cite web|url=http://www.szdushi.com.cn/news/201709/150673419244326.shtml|title=太湖大桥复线工程和渔洋山隧道正式运行通车|accessdate=2018-03-30|author=|date=2017-09-30|work=苏州都市网|publisher=|language=zh-CN}}</ref>，以新桥为进岛方向，老桥为出岛方向。

== 大事记 ==
1992年5月，时任[[中共中央政治局常委|中共中央政治局常委]]、[[中华人民共和国国务院总理|国务院总理]][[李鹏|李鹏]]来苏考察时题写桥名<ref>{{Cite book|url=http://www.dfzb.suzhou.gov.cn/database_books_detail.aspx?bid=613|title=《吴县志》|last=|first=|publisher=上海古籍出版社|year=1994.02|isbn=9787532516209|location=|pages=|chapter=大事记}}</ref>。

太湖大桥建成后至1996年，时任[[中国共产党中央委员会总书记|中共中央总书记]]兼[[中华人民共和国主席|国家主席]][[江泽民|江泽民]]，[[中华人民共和国国务院总理|国务院总理]][[李鹏|李鹏]]，[[中华人民共和国国务院副总理|国务院副总理]][[朱镕基|朱镕基]]、[[吴邦国|吴邦国]]，[[中华人民共和国副主席|国家副主席]][[荣毅仁|荣毅仁]]等领导人分别前来视察。1996年9月，[[新加坡|新加坡]][[新加坡内阁|内阁]][[内阁资政|资政]][[李光耀|李光耀]]和夫人游览太湖大桥及西山岛<ref>{{Cite book|url=http://www.dfzb.suzhou.gov.cn/database_books_detail.aspx?bid=4544|title=《西山镇志》|last=|first=|publisher=苏州大学出版社|year=2001.06|isbn=9787810378239|location=|pages=|chapter=大事记}}</ref>。

2000年10月6日，中国吴县环太湖世界[[特技飞行|特技飞行]]大奖赛举行驾机穿越太湖大桥1号桥桥孔的特技表演，共有7名[[飛行員|飞行员]]参与并完成了该表演，包括5位来自中国的飞行员和来自[[匈牙利|匈牙利]]的{{Link-en|Péter Besenyei|Péter Besenyei|贝森叶}}及来自[[立陶宛|立陶宛]]的{{Link-en|Jurgis Kairys|Jurgis Kairys|凯里斯}}<ref>{{Cite web|url=http://mil.news.sina.com.cn/2000-10-05/5883.html?domain=mil.news.sina.com.cn&amp;vt=4&from=wap|title=6日穿越太湖大桥桥洞的中外飞行员已经产生|accessdate=2018-03-30|author=|date=2000-10-05|work=新浪|publisher=|language=zh-CN}}</ref><ref>{{Cite web|url=http://news.sohu.com/20091006/n267177489.shtml|title=2000年10月6日 中国飞行员穿越太湖大桥桥洞-搜狐新闻|accessdate=2018-03-30|author=|date=2009-10-06|work=搜狐|publisher=|language=zh-CN}}</ref>。

2017年4月19日，太湖大桥新桥1号桥发生公交车与大巴车相撞事故，造成1死39伤<ref>{{Cite web|url=https://news.qq.com/a/20170419/045195.htm|title=苏州太湖大桥一公交车与大客车相撞 已致1死39伤|accessdate=2018-03-30|author=|date=2017-04-19|work=腾讯新闻|publisher=|language=zh-CN}}</ref>。

== 桥梁参数 ==
[[File:太湖大桥_Taihu_Bridge_-_panoramio_(1).jpg|thumb]]

太湖大桥初建时，桥面宽度12米，双向2车道，按[[中华人民共和国公路等级|二级公路]]标准建设，设计时速60公里/小时。主通航孔为两个6米宽的预应力[[混凝土|混凝土]]单[[箱梁|箱梁]]连续组成，其余引桥部分均为跨径22米的预应力混凝土简支空心板梁。全桥采用钻孔灌注桩技术<ref name=":0" />。桥面为细石混凝土结构。
* 1号桥全长1768米，总计73孔，有两个主通航孔，标高10.5米。
* 2号桥全长1621米，总计68孔，有一个主通航孔，标高22.5米。
* 3号桥全长919米，总计40孔，有一个主通航孔，标高10.5米。
