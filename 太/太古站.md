'''太古站'''（{{lang-en|Tai Koo Station}}）是[[港鐵|港鐵]][[港島綫|港島綫]]的一個[[鐵路車站|鐵路車站]]，位於[[香港|香港]][[東區_(香港)|東區]][[鰂魚涌|鰂魚涌]][[康山|康山]]地底，由[[寶嘉建築|寶嘉建築]]負責興建，於1985年5月31日啟用。車站所在的位置，鄰近香港島鰂魚涌往昔的一座大型船塢「[[太古船塢|太古船塢]]」，該船塢於1980年代被[[太古地產|太古地產]]改建為大型住宅屋苑[[太古城|太古城]]<ref>[http://news.stheadline.com/figure/?id=494 太古船塢 今日的太古城]，頭條日報</ref>，車站因此而被命名為'''太古站'''。而該站設有兩個主要車站上蓋物業，分別為[[康怡花園|康怡花園]]及[[康山花園|康山花園]]。

===大堂===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = HK MTR Tai Koo Station Concourse.jpg | caption1 = 車站售票大堂，前方為一塊鳴謝有份參與港島線工程的公司不鏽鋼牌匾（將軍澳站也有同類牌匾）
| image2 = Tai_Koo_opening.jpg | caption2 = [[香港總督|港督]][[尤德|尤德]]主持港島綫的啟用典禮紀念牌
}}
由於太古站是在堅固的[[花崗岩|花崗岩]]地底的[[東南亞|東南亞]]最大的人工石洞內興建，以拱形結構支撐，站體直接在石洞內，以簡單的混凝土樑柱結構方式建造，所以大堂廣闊無柱，而月台則只有兩排柱陣結構，而通往出入口的行人隧道則以鑽挖方式興建。

大堂內設有[[便利商店|便利店]]及[[麵包糕餅店|麵包糕餅店]]等商店，以及不少自助服務設施供乘客享用，包括[[自動櫃員機|自動櫃員機]]、[[自動售賣機|自動售賣機]]、自動照相機等<ref name="shops"/>。大堂近A出入口亦設有[[香港郵政|香港郵政]][[郵箱|郵箱]]<ref>{{Cite web|language=zh-hk|publisher=港鐵公司|title=鐵路四處遊－郵箱服務|url=http://www.mtr.com.hk/chi/getting_around/post.html|author=港鐵公司|accessdate=2014-04-19}}</ref><ref>{{Cite web|language=zh-hk|publisher=香港郵政|title=郵箱位置表－東區|url=http://app1.hongkongpost.hk/CGI/location/box_content.jsp?region=%27Hong%20Kong%27&lang=zh_TW#eastern|author=香港郵政|accessdate=2009-07-27|deadurl=yes|archiveurl=https://web.archive.org/web/20131008213422/http://app1.hongkongpost.hk/CGI/location/box_content.jsp?region=%27Hong%20Kong%27&lang=zh_TW#eastern|archivedate=2013-10-08}}</ref>，方便乘客在車站內投寄[[郵件|郵件]]。另外，大堂近D出入口設有1部「[[會員服務站|會員服務站]]」，供乘客享用港鐵友禮會的會員優惠<ref>{{Cite web|language=zh-hk|publisher=港鐵公司|title=鐵路四處遊－「會員服務站」|url=http://www.mtr.com.hk/ch/customer/services/csp.html|author=港鐵公司|accessdate=2016-09-26}}</ref>。

在[[互聯網|互聯網]]服務上，大堂內設有「iCentre」免費上網服務設施<ref name="icentre">{{Cite web|language=zh-hk|publisher=港鐵公司|title=鐵路四處遊－iCentre互聯網中心|url=http://www.mtr.com.hk/chi/getting_around/icentre.html|author=港鐵公司|accessdate=2014-04-19}}</ref>，以及免費由[[電訊盈科|電訊盈科]]提供的[[Wi-Fi|Wi-Fi]]熱點<ref>{{Cite web|language=zh-hk|publisher=港鐵公司|title=鐵路四處遊－Wi-Fi無線寬頻|url=http://www.mtr.com.hk/chi/getting_around/free_wifi.html|author=港鐵公司|accessdate=2014-04-19}}</ref>，乘客可在候車的時候透過任何支援Wi-Fi的電子產品存取互聯網。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Tai Koo Station 2018 01 part4.jpg | caption1 = 2號月台（往堅尼地城方向）
}}
太古站是一個地底車站，設有一個[[島式月台|島式月台]]，並設有[[月台幕門|月台幕門]]。由於車站是在堅固的花崗岩地底興建，花崗岩裂隙及毛細管作用滲出地下水至車站內，加上對商戶污水排放估計錯誤，因此月台以前經常出現異味，直至調整排污系統及完成月台幕門後氣味才有改善。

太古站東面設有一條儲車軌，稱為[[太古側綫|太古側綫]]<ref>[http://mtr.hk365day.com/Railway/Special/track/MTR_track_diagram_full.png 港鐵軌道配置圖]</ref>，是在港島綫興建時，由於地鐵柴灣車廠尚未完成，為方便把車輛運送至港島綫路軌，側綫曾延伸至海邊臨時車廠，後因柴灣車廠落成而封閉地底至臨時車廠路段。現時如果當[[西灣河站|西灣河站]]至[[柴灣站|柴灣站]]出現故障而需要暫停服務時，往柴灣方向的列車可以該站作為臨時的終點站。在這種情況下，列車首先會在該站1號月台卸客，之後會駛入太古側綫用作調頭之用。其後，列車將會駛入該站2號月台，接載往[[堅尼地城|堅尼地城]]沿途各站的乘客。太古站於月台兩端（隧道內）各設有兩台大型抽煙扇，在一般情況下並不會運作，只會在緊急時使用。這四台抽煙扇約兩米高，乘客在列車進出太古站月台時會較易發現。

==利用狀況==
太古站位於[[香港|香港]][[東區_(香港)|東區]]人口最稠密的地區，加上鄰近設有大型商場[[太古城中心|太古城中心]]和[[康怡廣場|康怡廣場]]，以及[[太古城|太古城]]、[[康怡花園|康怡花園]]、[[康山花園|康山花園]]及[[南豐新邨|南豐新邨]]等大型屋苑和[[港島東中心|港島東中心]]等商廈，本站人流一直穩定。而在週六、週日及公眾假期期間，由於有大量的乘客利用車站前往太古城和[[康怡廣場|康怡廣場]]購物和觀看電影，而平日則有不少上學和上班之乘客進出該站，因此車站是港島綫沿綫最繁忙的港鐵站之一。

==歷史==
===命名及建造===
在1970年代港島綫定線的初稿中，並沒有太古站。只有一個名為'''芬尼站'''的地下車站設於現時鰂魚涌街附近。1983年，由於政府再次修改[[鰂魚涌|鰂魚涌]]發展藍圖為商業區，因此在發表港島綫計劃時，於[[康山|康山]]地底加設'''康山站'''，成為太古站雛型。政府其後批准[[太古地產|太古地產]]把位於[[鰂魚涌|鰂魚涌]]的[[太古船塢|太古船塢]]原址，發展大型住宅項目「太古城」，並將這個鄰近前「太古船塢」的車站，命名為'''太古站'''，太古站於1985年5月31日啟用。

===短程特別班次===
在港島綫通車初期，由於車隊不足，地鐵曾一度設金鐘站側綫至太古站側綫的短程特別班次，由1986年10月1日起，所有港島綫列車皆行走上環至柴灣段。不過，近年早上繁忙時間仍會有空車由側綫開出，不停太古及鰂魚涌兩站，作為由北角至上環的短程特別班次。

===新建D出入口===
在太古站通車初期，通往太古城的出入口則只有一個E1出入口，居住在該處的居民只能靠此出入口出入而造成不便。港鐵當局決定在太古城道及太古城中心一座的地下加建2個新出入口，即現時的D1及D2出入口，並為太古城的居民帶來更大的方便。有關工程在1993年動工，在1995年完成，而新出入口亦能應付港島東中心落成後額外使用率。

===第二條過海鐵路===
1984年初，政府有意發展東區海底隧道，作為行車和鐵路兩用隧道，其中鐵路部份以地鐵[[觀塘綫|觀塘綫]]由[[觀塘站|觀塘站]]經[[藍田站|藍田站]]延伸至香港島。原計劃觀塘綫在太古站和港島綫交匯，但因為要遷就行車隧道的走線，最後改以[[鰂魚涌站|鰂魚涌站]]作為兩線交匯的車站。
