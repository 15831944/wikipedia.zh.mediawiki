'''太原钢铁（集团）有限公司'''，簡稱'''太钢集团'''、'''太原钢铁'''、'''太钢'''，是位于[[山西省|山西省]][[太原市|太原市]]的一家大型[[钢铁|钢铁]]企业，太钢集团的最終控制人是[[山西省人民政府国有资产监督管理委员会|山西省人民政府国有资产监督管理委员会]]，但透過[[山西省国有资本投资运营有限公司|山西省国有资本投资运营有限公司]]持股。太钢是中国特大型的钢铁企业，也是山西省最大的企业，太钢的厂房和家属区等附属设施在今天的太原市占据了城区北部的相当大的部分，有'''十里钢城'''之称。

== 沿革 ==
其前身是由[[阎锡山|阎锡山]]的[[西北实业公司|西北实业公司]]于1932年筹建成立的'''西北炼钢厂'''。

1949年太原解放后更名为'''西北钢铁公司'''。

1958年，更名为'''太原钢铁公司'''。

1996年1月改组为太原钢铁（集团）有限公司。

1998年10月子公司[[太钢不锈钢|太钢不锈钢]]上市<ref>{{cite web |url=http://www.tisco.com.cn/show.jsp?id=223 |title=存档副本 |accessdate=2009-07-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20110609175949/http://www.tisco.com.cn/show.jsp?id=223 |archivedate=2011-06-09 }}</ref>。

2011年9月1日由太鋼集團、寶鋼集團、 [[中信|中信]]金屬公司、[[鞍鋼|鞍鋼]]和[[首鋼|首鋼]]共同出資成立的中國鈮業投資控股有限公司成功以19.5億美元收購世界最大鈮公司巴西礦冶（CBMM）15%的股權。CBMM由巴西Moreira Salles家族控制，五大企業完成入股權後，該家族仍持有七成控股權{{Fact|time=2018-08-14T19:40:35+00:00}}。

==子公司==
===太钢医疗===
太钢集团透過子公司太钢医疗，擁有太钢总医院<ref name=太钢总医院>{{cite web|url=http://www.tisco.com.cn/yiliaoweisheng/20120927134204811474.html |title=医疗卫生|publisher=太钢集团|publication-place=太原|accessdate=2018年8月16日|language=zh-cn}}</ref>。太钢总医院是一間[[三级甲等医院|三级甲等]]综合医院，擁有[[山西医科大学|山西医科大学]]附属医院、检验学系临床教学基地等資格<ref name=太钢总医院/>。[[中国人寿保险|中国人寿保险]]旗下私募股权投资基金「國壽大健康」是太钢医疗的小股東<ref>{{cite news|url=http://finance.sina.com.cn/roll/2018-01-15/doc-ifyqqciz7092997.shtml|title=中国人寿探索“保险+健康”模式|date=2018年1月15日|accessdate=2018年8月17日|newspaper=经济参考报|language=zh-cn|via=新浪财经}}</ref>。
===太钢房地产开发===
太钢房地产开发是太原市[[尖草坪区|尖草坪区]]、[[杏花岭区|杏花岭区]]的太钢翠馨苑、太钢锦绣苑、太钢锦华苑、太钢新钢苑、太钢如意苑、太钢滨河苑、房苑等小区的开发商<ref>{{cite web|url=http://www.tisco.com.cn/fangdichankaifa/20120927133251733469.html|title=房地产开发|publisher=太钢集团|publication-place=太原|accessdate=2018年8月17日|language=zh-cn}}</ref><ref>{{cite news|url=https://news.qq.com/a/20120703/000396.htm|title=太钢2418户职工将迁新居|date=2012年7月3日|accessdate=2018年8月17日|language=zh-cn|website=山西新闻网|via=腾讯}}</ref>。
