'''太和庄天主堂'''，位于[[北京市|北京市]][[房山区|房山区]][[长沟镇_(北京市)|长沟镇]]太和庄村。现为[[北京市房山区普查登记文物|北京市房山区普查登记文物]]。

==简介==
太和庄天主堂位于太和庄村中心，在[[中华民国|中华民国]]时期便已存在。教堂周边都是教堂的产业，占地规模很大。

[[抗日战争|抗日战争]]初期，在太和庄天主堂周边发生了“太和庄惨案”。[[七七事变|七七事变]]后，1937年9月，侵华[[日军|日军]]占领了房山、[[良乡|良乡]]地区。房山各地人民纷纷组织地方武装，抗击日军。当时，一路（时称华北抗日同盟军，俗称“老便”）司令周文龙率3000多人在乡沟、窦店一带活动。其部下朱龙带领全队人员驻太和庄天主堂内。他们袭击日军，逮捕日军的探子，并将探子捆在太和庄天主堂里审讯。1937年12月23日拂晓，日军突然从太和庄村东北的砖窑沟侵入太和庄村。约凌晨四五点钟，太和庄村东北突响枪声，村中数百名村民被从睡梦中惊醒。日军冲进村里，见人就杀，见草垛和房子就放火。孙秀一家14口全被堵在家中，不得不扒着墙头往外跳，每个跳过墙头的家人都被日军挑死，共被挑死七人。孙士元的祖母、父亲、叔叔等人也往外跑。日军看见孙奎，端着刺刀便要冲上去刺死。孙奎的老母亲见此便使劲攥住刺刀。日军士兵向后一拽，孙奎母亲的手指当场被挑断。这个日军士兵又追上孙奎将其用刺刀刺死。日军来到于福家，于福的老母亲说：“我们都是好人，别打死我们。”日军见这老人跑不动，便示意她出去，老人背着孙女向外走，没走多远，日本兵就朝她开枪，孙女当场死亡。王以寿钻进草垛里躲藏，被日军点燃草垛活活烧死。大批太和庄村民们顺大道向西逃。日军将被追上的村民都用刺刀挑死，追不上的就开枪。跑得快的村民刚到村边，就被早架好的机枪扫射身亡。大人被打死后，有的婴儿还活着。刚出生几个月的姜家女婴哭着在死人堆里爬。自12月23日拂晓至正午，日军行凶整整半天。在太和庄惨案中，太和庄全村共78人被杀，数十间房屋及不少柴垛、草垛被烧毁。<ref>{{Cite web |url =http://mt.sohu.com/20160706/n458106731.shtml  |title =  “七七”勿忘国耻，记发生在房山地区的侵华惨案！|author =  |date =  2016-07-06|publisher =  搜狐|language =  |accessdate =  |quote =  }}</ref>

[[中华人民共和国|中华人民共和国]]成立后，太和庄天主堂被关闭。后来被弃置不用。2009年6月23日，[[北京市政协|北京市政协]]副主席[[赵文芝|赵文芝]]来到长沟镇太和庄天主堂、琉璃河镇[[立教天主堂|立教天主堂]]旧址调研。[[北京市民族事务委员会|北京市民族事务委员会]]主任、北京市宗教局局长[[申建军|申建军]]，北京市政协民族宗教委员会办公室主任聂胜利，北京市天主教爱国会副主席[[石洪喜|石洪喜]]，房山区政协主席范文彦、副主席李惠英陪同调研。<ref>{{Cite web|url =http://zxta.bjfsh.gov.cn/dsyl/201501/20150121041108613.html|title =2009年政协工作大事记|author =|date =2015-01-21|publisher =中国人民政治协商会议北京市房山区委员会|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20161228031737/http://zxta.bjfsh.gov.cn/dsyl/201501/20150121041108613.html|archivedate =2016-12-28}}</ref>

2012年，“太和庄天主堂”被列为[[北京市房山区普查登记文物|北京市房山区普查登记文物]]。<ref>{{Cite web|url = http://whw.bjfsh.gov.cn/whycbh/wwbh/19078.htm|title = 北京市房山区第三次全国文物普查不可移动文物名录|author = |date = 2015-08-19|publisher = 房山区文化委员会|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20161228032506/http://whw.bjfsh.gov.cn/whycbh/wwbh/19078.htm|archive-date = 2016-12-28|dead-url = yes}}</ref>
