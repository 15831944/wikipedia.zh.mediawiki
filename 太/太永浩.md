'''太永浩'''<ref>前译为太勇浩</ref>（{{lang-ko|태영호}}，{{lang-en|Thae Yong-Ho}}，{{bd|1962年|7月25日|||}}），前[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]駐英國大使館[[公使|公使]]。他與其家人在2016年7月[[脫北者|脫北流亡]]到[[大韓民國|大韓民國]]，是繼[[張承吉|張承吉]]後另一名投奔韓國的高級別外交官。

==生平==
太永浩生於[[出身成份|出身成份]]良好的家庭<ref name="BBC">{{cite web|url=http://www.bbc.com/zhongwen/trad/world/2016/08/160817_nkorea_defectorl|title=分析：接二連三的「菁英階層」脫北潮|work=BBC|accessdate=2016-08-17}}</ref>。在中學時期，他已獲准到中國留學，學習英語和漢語<ref name="cnnews">{{cite web|url=http://cnnews.chosun.com/client/news/viw.asp?cate=C01&mcate=M1001&nNewsNumb=20160845744&nidx=45745|title=流亡的朝鲜驻英国外交官是大使馆公使|work=朝鮮日報|accessdate=2016-08-17}}</ref>。太永浩中学在[[北京五十五中|北京五十五中]]和北京外语学院附属学校就读，大学时回到平壤，其后又来到北外英语系进修，前后总共有八年的时间在北京生活。<ref>{{cite news|author1=刘粤瑛|title=专访太永浩:朝鲜——不能不说的秘密|url=http://news.takungpao.com/world/exclusive/2017-03/3431582.html|accessdate=2018-05-28|work=大公网|date=2017-03-17}}</ref>回国后毕业于平壤国际关系大学，委派到[[朝鮮民主主義人民共和國外務省|外務省]]工作，獲选为丹麦语1号培养翻译（金正日专属翻译候补）<ref name="cnnews"/>，因應安排到丹麥留學。1993年开始在大使馆担任秘書<ref name="cnnews"/>。大使館撤銷後派到[[瑞典|瑞典]]一段時間。不久後，他應召回國出任外務省歐洲事務課長<ref name="cnnews"/>。因此，他被視為朝鮮最優秀的對歐外交官<ref name="cnnews"/>。

2012年，太永浩奉派到朝鮮駐英大使館當公使，負責對英宣傳朝鮮、監視在[[倫敦|倫敦]]的[[脫北者|脫北者]]，以及在英國記者們到訪[[平壤|平壤]]採訪時的聯繫負責人<ref>{{cite web|url=http://www.bbc.com/zhongwen/simp/uk/2016/08/160817_nkorea_diplomat_defection|title=韩国证实朝鲜驻英公使携家人叛逃首尔|work=BBC|accessdate=2016-08-17}}</ref><ref>{{cite web|url=http://cnnews.chosun.com/client/news/viw.asp?cate=C01&mcate=M1001&nNewsNumb=20160845748&nidx=45749|title=朝鲜驻英国大使馆公使太永浩流亡到首尔|work=朝鮮日報|accessdate=2016-08-17}}</ref>。他主張[[朝鮮勞動黨|朝鮮勞動黨]]和[[金正恩|金正恩]]的統治未得到外國傳媒的正確報導而令世人對朝鮮產生誤解。在2014年的一次公開演講中，他以英語指出外國媒體就朝鮮禁止人民自由出境的報導是「完全扭曲的消息」，又認為「若英國人或美國人知道世上有一個提供免費教育、居住和醫療的國家，就會重新審視朝鮮」<ref name="BBC"/><ref name="cnnews"/>。此外，他還在同年負責接待金正恩的哥哥[[金正哲|金正哲]]<ref name="cnnews"/>。

===流亡韩国===
太永浩及其家人忽然於2016年7月中旬失去聯絡。英国《星期日快报》称太永浩在叛逃前两个月在高尔夫球场与英国特工进行接触，一个月前携家人乘坐[[英國皇家空军|英國皇家空军]]飞机飞往[[美军|美军]]駐[[德国|德国]]的[[拉姆施泰因空军基地|拉姆施泰因空军基地]]，而后换乘另一架飞机，抵达韩国首爾<ref>{{cite web|url=http://www.bbc.com/zhongwen/simp/world/2016/08/160821_uk_press_nkorea_defector|title=英媒：朝驻英外交官叛逃酷似间谍小说|work=BBC|accessdate=2016-08-25}}</ref>。同年8月17日，[[統一部|大韓民國統一部]]宣稱太永浩與和家人皆已身在韓國，又解釋其出逃是基於「對金正恩政府感到失望」及「渴望韓國的自由和民主」<ref>{{cite web|url=http://cn.nytimes.com/world/20160818/north-korea-defector-thae-yong-ho-britain/zh-hant/?utm_source=fb-nytimeschinese&utm_medium=social&utm_campaign=curl|title=朝鮮駐英高級外交官叛逃韓國|work=New York Times|accessdate=2016-08-17}}</ref>。
太永浩和夫人以及两个儿子在2016年投奔韩国后，自2017年1月任[[韩国国家情报院|韩国国家情报院]]下属安保战略研究院顾问一职。2018年5月，太永浩辞去顾问职务，表示将自由地工作活动，并认真推广自己的部落格。<ref>{{cite news|author1=尹喜勋|title=批判金正恩的太永浩辞韩政府研究所工作|url=http://cnnews.chosun.com/client/news/viw.asp?cate=C01&mcate=M1001&nNewsNumb=20180549830&nidx=49831|accessdate=2018-05-25|work=朝鲜日报中文网|date=2018-05-24}}</ref>

2018年5月，太永浩出版了长达542页的回忆录《三楼书记室的暗号》（3층 서기실의 암호），详细记述了他作为朝鲜外交官的生活，揭露了金氏家族的统治内幕。<ref>{{cite news|author1=CHOE SANG-HUN|title=叛逃朝鲜高官新书揭露金正恩洗清家族成员内情|url=https://cn.nytimes.com/asia-pacific/20180524/north-korea-kim-jong-un-book/|accessdate=2018-05-25|work=纽约时报中文网|date=2018年5月24日}}</ref>该书一出版就在韩国成为畅销书。<ref>{{cite news |author1=Ji-hye Choi |title=2018年5月第5周统计结果 |url=http://kbook-chn.or.kr/bbs/board.php?bo_table=201806ch&wr_id=11 |accessdate=2018-08-31 |work=韩书趋势 |date=2018-06-28}}</ref>

== 朝方定罪 ==
2016年8月20日，[[朝中社|朝中社]]报道他私吞大笔国家资金、出卖国家机密、[[强奸|强奸]]未成年人，于2016年6月下令將之召回调查。[[朝鲜中央检察院|朝鲜中央检察院]]了解犯罪资料后，7月12日正式决定对其故意泄秘罪、国家财产贪污罪、未成年性交罪展开侦查。朝中社形容太永浩怕法律处分抛弃祖国和父母兄弟叛逃，体现了没有做人起码的道义、毫无良心和道德可言的败类嘴脸<ref>"동족대결의 새로운 모략극" 《[[朝鮮中央通訊社|朝鮮中央通訊社]]》 20160820</ref><ref>[http://www.kcna.kp/kcna.user.article.retrieveNewsViewInfoList.kcmsf 朝中社评论：新一出反朝对抗阴谋剧 朝中社平壤8月20日电] {{webarchive|url=https://web.archive.org/web/20170304112449/http://www.kcna.kp/kcna.user.article.retrieveNewsViewInfoList.kcmsf |date=2017-03-04 }}</ref>。
8月22日，[[韩国统一部|韩国统一部]]部長洪容杓表示，朝方为弃朝投韩的驻英公使太永浩扣上罪犯帽子意在减少流亡消息传入国内后产生的影响。<ref>{{cite web|url=http://chinese.yonhapnews.co.kr/newpgm/9908000000.html?cid=ACK20160822004100881|title=韩统一部长:朝为维稳给流亡公使乱扣罪名|work=韩联社|accessdate=2016-08-26}}</ref>

== 家庭 ==
先前报道说太永浩的父亲是朝鲜前中央委员会委员、抗日游击队“红一代”[[太炳烈|太炳烈]]，做过金日成的传令兵，哥哥是朝鲜[[金日成综合大学|金日成综合大学]]校长[[太炯哲|太炯哲]]<ref>{{Cite web |url=http://news.163.com/16/0818/15/BUOU8EE40001121M.html |title=韩媒：朝鲜驻英外交官携家属抵韩 系朝高官之子 |access-date=2016-08-18 |archive-url=https://web.archive.org/web/20160819063134/http://news.163.com/16/0818/15/BUOU8EE40001121M.html |archive-date=2016-08-19 |dead-url=yes }}</ref>。24日，[[大韓民國国家情报院|大韓民國国家情报院]]确认他并非太炳烈之子<ref>[http://chinese.joins.com/gb/article.do?method=detail&art_id=156090 国家情报院：投诚的太永浩夫人正是游击队家庭出身]</ref>。妻子吳惠善（1966年生），其父是[[朝鮮人民軍总参谋部|朝鮮人民軍总参谋部]]副总参谋长{{Link-ko|吳琴鐵|오금철}}，祖父[[吳白龍|吳白龍]]是[[金日成|金日成]]的游击队战友、劳动党军事部部长<ref>[http://chinese.joins.com/gb/article.do?method=detail&art_id=155974 聚焦朝鲜高层外交官脱北]</ref>。 

长子（1990年生）获得英国海默斯密斯医院公共保健领域的经济学学士学位，曾写过一篇主旨为“若想把平壤打造成世界化都市，必须扩充残疾人停车空间”的论文。次子太金赫（1997年生）则刚刚在伦敦西部地区的公立阿克顿高中毕业，取得Ａ+成绩、正准备就读英国[[帝国理工学院|帝国理工学院]]。学习数学和电脑工学。太永浩生活费困难，无力承担学费，韩国媒体认为这是他叛逃的原因<ref>[http://cnnews.chosun.com/client/news/viw.asp?cate=C01&mcate=M1001&nNewsNumb=20160845762&nidx=45763 朝鲜驻英大使馆公使流亡和次子学业有关]</ref>。先前报道称其可能还有一个女儿留在朝鲜<ref>[http://chinese.joins.com/gb/article.do?method=detail&art_id=155959 朝鲜驻英国大使馆公使逃亡属于“移民型脱北”]</ref>，国家情报院确认他没有女儿。<ref>[http://chinese.joins.com/gb/article.do?method=detail&art_id=156090 国家情报院：投诚的太永浩夫人正是游击队家庭出身]</ref>
