'''太子城站'''位于[[河北省|河北省]][[张家口市|张家口市]][[崇礼区|崇礼区]][[四台嘴乡|四台嘴乡]]太子城村，是在建的[[京张城际铁路|京张城际铁路]]支线——崇礼铁路的一座车站。<ref>{{cite web |author1=张家口房天下 |title=2018年张家口崇礼太子城高铁站新进程揭秘! |url=http://zhangjiakou.news.fang.com/2018-01-22/27639109.htm}}</ref>站址位于崇礼奥运村南侧1公里处，步行几分钟即可抵达奥运村。可以满足官员、运动员、记者等乘坐火车在1小时内在北京和崇礼赛区内转场的需求。<ref>{{cite web |author1=国家体育总局 |title=申办冬奥会场馆规划张家口赛区亮点丰富 |url=http://www.sport.gov.cn/n16/n2061573/n2760843/6119477.html}}</ref>

太子城站总建筑面积11988平方米，其中主站房5990平方米，地下换乘中心（奥运临时候车室）5998平方米。地下一层，地上一层（局部夹层）。车站中心里程K52+450，地面层绝对标高1582.9m。<ref>{{cite web |author1=中铁六局采购网 |title=中铁六局集团建筑安装分公司新建崇礼铁路太子城站站房及相关工程集装箱采购竞争性谈判公告 |url=http://wm.crsg.com.cn/9-1.asp?ID=1821}}</ref>该车站建设单位是北京局站房工程项目管理部，设计单位是中铁工程设计咨询集团有限公司，监理单位是西安铁一院咨询监理有限公司，施工单位是中铁六局集团公司，车站预计于2019年12月31日投入使用。

太子城站共设有南北走向的4条到发线（含正线），2座450×12×1.25米的岛式站台；为更好满足奥运期间人流疏散需求，主站房一侧还设有一座450×8×1.25米的基本站台，因此最东侧股道形成了在铁路系统内少见的西班牙式站台。

目前太子城站是最高时速250km/h的崇礼铁路的终点，未来铁路将延伸至[[崇礼|崇礼]]、黑城子，并通过改建既有线至[[正蓝旗_(内蒙古)|正蓝旗]]，最终至[[锡林浩特|锡林浩特]]。<ref>{{cite web |author1=锡林浩特市人民政府 |title=太子城至锡林浩特快速铁路可研编制完成 |url=http://www.xilinhaote.gov.cn/xwzx/xsyaowen/201709/t20170928_1866615.html}}</ref>
