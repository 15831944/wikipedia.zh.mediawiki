'''太平洋雙色鰻鱺'''（[[学名|学名]]：{{lang|la|'''''Anguilla bicolor pacifica'''''}}），又名'''二色鰻'''、'''短鰭鰻'''，為[[輻鰭魚綱|輻鰭魚綱]][[鰻鱺目|鰻鱺目]][[鰻鱺亞目|鰻鱺亞目]][[鰻鱺科|鰻鱺科]]的其中一[[種|種]]。

== 分布 ==
本魚分布於[[印度|印度]][[太平洋|太平洋]]區，包括[[帛琉|帛琉]]、[[臺灣|臺灣]]、[[菲律賓|菲律賓]]及[[新幾內亞|新幾內亞]]等河川溪流皆有其蹤跡。

== 特徵 ==
本魚體延長呈圓柱形，但肛門後的尾部則稍側扁。頭部中大，吻略長而圓鈍，鱗片細小，隱埋於皮下，體表光滑多黏液。體背灰色而腹面灰棕色至紅棕色，胸鰭極為短小，無腹鰭。體長可達123公分。

== 生態 ==
本魚為洄游性魚類，成魚棲息在河川當中，夜行性，屬肉食性，以底棲動物、小型魚類、[[甲殼類|甲殼類]]等為食。

== 經濟利用 ==
可食用，經濟價值不大。
