'''太和站'''（{{lang-en|Tai Wo Station}}）是[[港鐵|港鐵]][[東鐵綫|東鐵綫]]的一座鐵路車站，位於[[新界|新界]][[大埔區|大埔區]][[太和邨|太和邨]]，亦是前[[九廣鐵路|九廣鐵路]]少數和[[東鐵線|東鐵線]]唯一建於[[公共屋邨|公共屋邨]]內及以公共屋邨命名的車站。

==結構==
太和站是一個建築在堤上的車站。為免太和邨居民受嘈音滋擾，該車站被一道[[隔音屏障|隔音屏障]]覆蓋。

===大堂===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Tai Wo Station 2017 08 part2.jpg | caption1 = 大堂
| image2 = Tai Wo Station 2018 05 part1.jpg | caption2 = 商店
}}
太和站大堂位於1號月台旁。太和站興建時，大堂原屬太和邨商場（今[[太和廣場|太和廣場]]）的一部份，但在商場在2009年翻新時，大堂與商場由商場的管理公司──[[領展|領展]]所安裝的玻璃門分隔。

太和站設有4間商店，包括[[便利店|便利店]]、[[麵包糕餅店|麵包糕餅店]]、零食專賣店等。站內亦設有[[自動櫃員機|自動櫃員機]]<ref name="shops"/>。
[[File:Tai_Wo_Station_2017_08_part3.jpg|thumb]]

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Tai Wo Station 2019 03 part1.jpg | caption1 = 1號月台﹙往羅湖／落馬洲方向﹚
}}
太和站設有兩個[[側式月台|側式月台]]，並無安裝[[月台幕門|月台幕門]]。乘客往返兩個月台及大堂需使用月台上方的行人天橋。另外，月台西面（往羅湖方向車頭及往紅磡方向車尾）是一個彎路段，因此該處空隙較大。

===出入口===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Tai Wo Station 2017 08 part1.jpg | caption1 = A出入口
| image2 = Tai Wo Station 2018 05 part2.jpg | caption2 = B出入口

太和站設有2個出入口，A出入口位於太和廣場二期的大堂，而B出入口則位於往返大堂及2號月台的行人天橋上。由於B出入口位於狹小的行人天橋上，因此出入口只設有出入閘機及客務中心，並無大型的大堂及商店。

==利用狀況==
本站與大埔墟站是大埔區僅有的兩個車站，本站位於[[太和邨|太和邨]]內，亦鄰近[[大埔舊墟|大埔舊墟]]，[[太和市|太和市]]和林村河南岸的村屋，以及附近一帶私人屋苑。本站也是服務大埔北部和西部，不少[[康樂園|康樂園]]、[[泰亨|泰亨]]及[[林村谷|林村谷]]一帶鄉村居民多前往本站轉乘鐵路前往各區。

由於太和邨內對外巴士不多，太和邨乘客会步行至[[廣福道|廣福道]]乘搭前往市區的巴士路線，加上本站设于太和邨內，出入口较少且必须途经太和广场前往，因此本站客量一般。

==歷史==
===興建車站===
[[File:Tai_Wo_Station.jpg|220px]]
[[File:Tai_Wo_Station_Concourse_2008.jpg|220px]]
九廣鐵路—英段（今東鐵綫）全面電氣化計劃時已經有計劃興建太和站，但是九鐵沒有在1983年九廣鐵路—英段全面電氣化計劃完成前興建。隨著[[太和邨|太和邨]]的落成，附近人口於1985至87年間不斷增加，九鐵在既有的路軌上加建車站。太和站於1986年3月4日動工興建，1989年5月9日完工並投入使用。

1995年2月，B出口完工並投入使用。

===減少列車聲浪工程===
[[File:Tai_Wo_Station_Outside_View_2012.jpg|220px]]
[[九廣鐵路公司|九廣鐵路公司]]在1993年開展一項為期十年的「減少列車聲浪工程」計劃，為受到列車聲浪影響市民的路段加裝隔音屏障，太和站為東鐵綫第一個加裝隔音屏障的路段。工程將該站改為密封式設計及在外圍加裝隔音屏障，可減少15分貝。工程於1995年5月完成。

===翻新月台===
兩鐵合併後，港鐵為東鐵綫各月台進行較低成本的翻新工程，以遮掩東鐵綫車站月台殘舊的觀感。工程包括重新為各東鐵綫車站配主題色、代表植物，為月台橫樑及柱子翻上月台配色的油漆，貼上印有車站主題植物的圖案、附近建築物、車站名書法字的牆紙，加設直立式站牌、路線圖、指示牌、告示等等。此站配色為黃色，主題植物為牡丹花，街景為香港鐵路博物館。
