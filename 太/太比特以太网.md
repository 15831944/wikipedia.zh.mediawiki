'''太比特以太网'''（'''Terabit Ethernet'''，缩写'''TbE'''）是指速度超过[[100吉比特以太网|100吉比特以太网]]（100 Gbit/s）的[[以太网|以太网]]。'''400吉比特以太网'''（'''400G'''、'''400GbE'''）和'''200吉比特以太网'''（'''200G'''、'''200GbE'''）<ref name="NGOATH">{{cite web |url=http://www.ieee802.org/3/bs/NGOATH_3bs_Objectives_16_0122.pdf|title=IEEE 802.3 NGOATH SG Adopted Changes to 802.3bs Project Objectives}}</ref>标准由[[电气电子工程师学会|电气电子工程师学会]]P802.3bs工作组制定，采用类似100吉比特以太网<ref name="register2012">{{cite web |url=https://www.theregister.co.uk/2012/09/28/terabit_ethernet_too_fast/ |title=Network boffins say Terabit Ethernet is TOO FAST: Sticking to 400Gb for now}}</ref><ref>[http://www.fibre-systems.com/feature/board-optics-beyond-pluggables On-board optics: beyond pluggables]</ref>的广泛技术，已于2017年12月6日获得批准。<ref name="approval">{{Cite web|title=[STDS-802-3-400G] IEEE P802.3bs Approved! |publisher=IEEE 802.3bs Task Force |url=http://www.ieee802.org/3/400GSG/email/msg01519.html |accessdate=2017-12-14}}</ref><ref name="LHUFF2016">{{cite web |url=http://www.connectorsupplier.com/high-speed-transmission-update-200g400g/|title=High-Speed Transmission Update: 200G/400G
}}</ref>在2016年，多家网络设备供应商已经为200G和400G以太网提供专有解决方案。<ref name="LHUFF2016" />

{{as of|2018|1}}，尚无超过400 Gbit/s的IEEE[[技术路线|技术路线]]——这可能在2020年以后才会出现<ref>{{cite web |url=http://ieeexplore.ieee.org/document/7980557/|title=Recent trends in next generation terabit Ethernet and gigabit wireless local area network|publisher=IEEE}}</ref>，但当单信道100 GbE链路可用时，预计会出现800 GbE。此种技术可能会允许更大的信道捆绑，可能通过10或16个信道提供1 TbE或1.6 TbE的线路。<ref name="800gbe">{{cite web |url=http://www.networkcomputing.com/networking/ethernet-roadmap-raft-new-speeds/339990210|title=Ethernet Roadmap: A Raft Of New Speeds |author=Jim O'Reilly|publisher=NETWORKComputing.com}}</ref><ref name="roadmap">{{cite web |url=http://www.ethernetalliance.org/wp-content/uploads/2015/03/Ethernet-Roadmap-2sides-29Feb.pdf|title=The 2016 Ethernet Roadmap|publisher=ethernetalliance.org}}</ref>

==历史==
[[Facebook|Facebook]]和[[Google|Google]]以及其他公司已表示需要太比特以太网。<ref>{{cite web
 | first = Michael | last = Feldman
 | url = http://www.hpcwire.com/2010/02/03/facebook_dreams_of_terabit_ethernet/
 | title =  Facebook Dreams of Terabit Ethernet | date = February 3, 2010
 | work = HPCwire | publisher =  Tabor Communications, Inc.
}}</ref>现有技术可以实现400 Gbit/秒的速度，而1 Tbit/秒（1000 Gbit/秒）则可能需要不同的技术。<ref name="register2012"/><ref>{{cite web
 | url = http://www.lightreading.com/document.asp?doc_id=188442&
 | title = Dare We Aim for Terabit Ethernet?
 | work = Light Reading | publisher = UBM TechWeb,
 | first = Craig | last = Matsumoto
 | date = March 5, 2010
}}</ref>于是，在2012年9月举行的IEEE工业连接高速以太网共识小组会议上，400 GbE被选中作为下一代目标。<ref name="register2012"/>2016年1月还增加了200 GbE作为目标。

[[加利福尼亞大學聖塔芭芭拉分校|加利福尼亞大學聖塔芭芭拉分校]]（UCSB）吸引到了[[安捷倫科技|安捷倫科技]]、[[Google|Google]]、[[英特尔|英特尔]]、[[罗克韦尔柯林斯|罗克韦尔柯林斯]]以及[[威訊|威訊]]的帮助，来帮助研发下一代以太网。<ref>
 {{cite web
 |title = The Terabit Ethernet Chase Begins
 |author = Craig Matsumoto
 |publisher = Light Reading
 |date = October 26, 2010
 |url = http://www.lightreading.com/document.asp?doc_id=199046&
 |accessdate = December 15, 2011
 }}</ref>

截至2016年早期，来自思科、Juniper和其他主要制造商的基于机箱/模块化的核心路由器平台支持每个插槽400 Gbit/秒的全双工数据速率。目前已有一个、两个和四个端口的100 GbE和一个端口400 GbE的线路卡，预计在802.3cd标准被批准后将有200 GbE线路卡出现。<ref>{{Cite web|url=http://www.cisco.com/c/en/us/products/collateral/routers/carrier-routing-system/datasheet-c78-730788.html|title=Cisco 4 x 100 Gbit/s interface|last=|first=|date=|website=|publisher=|access-date=}}</ref><ref>{{Cite web|url=https://networks.nokia.com/press/2015/alcatel-lucent-boosts-operator-capacity-deliver-big-data-and-video-over-existing-networks-launch|title=Alcatel-Lucent boosts operator capacity to deliver big data and video over existing networks with launch of 400G IP router interface|last=|first=|date=|website=|publisher=|access-date=}}</ref>

IEEE 802.3的“400 Gb/s以太网研究组”于2013年3月开始制定400 Gbit/s标准。<ref>{{Cite web |title= 400 Gb/s Ethernet Study Group |url= http://www.ieee802.org/3/400GSG/ |work= Group web site |publisher= IEEE 802.3 |accessdate = May 23, 2013 }}</ref>研究组成果于2014年3月27日公布并被批准。随后，IEEE 802.3bs工作组<ref>[http://www.ieee802.org/3/bs/ IEEE 802.3bs Task Force]</ref>开始努力为数种链路距离提供物理层规范。<ref name="obj">{{Cite web |title=Objectives |publisher=IEEE 802.3bs Task Force |date=Mar 2014 |url=http://www.ieee802.org/3/bs/Objectives_14_0320.pdf |accessdate=2015-03-01}}</ref>

IEEE P802.3bs标准于2017年12月6日获得批准<ref name="approval"/>并在线发布。<ref>{{Cite web|title=P802.3bs - IEEE Standard for Ethernet Amendment: Media Access Control Parameters, Physical Layers and Management Parameters for 200 Gb/s and 400 Gb/s Operation |publisher=IEEE 802.3bs Task Force |url=http://standards.ieee.org/findstds/standard/802.3bs-2017.html |accessdate=2017-12-14}}</ref>
