'''太麻里車站'''位於[[台灣|台灣]][[臺東縣|臺東縣]][[太麻里鄉|太麻里鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[南迴線|南迴線]]的[[鐵路車站|鐵路車站]]；在「迎接[[千禧年|千禧年]]第一道曙光」活動中聲名大噪，號稱台灣最早看見日出的車站。

== 車站構造 ==
*站內設有[[島式月台|島式月台]]兩座，目前第一月台已停用。

== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]。
*2018/10/12 年度改點之後 實施週日班表
*停靠狀態：
**[[自強號列車|自強號]]：順行316、324次，逆行303次不停靠。
**[[莒光號列車|莒光號]]：[[觀光列車|觀光列車]]及順行710次( 週五行駛 )、6710（週日行駛）不停靠。
**[[臺鐵區間車|區間車]]：皆停靠（[[復興號列車|復興號]]車廂編組）。
**[[普快車|普快車]]：皆停靠
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1988年1月1日：卑南站（今[[臺東車站|臺東站]]）至太麻里站間開始臨時營業。
*1992年12月16日：於[[枋寮車站|枋寮站]]舉行[[南迴線|南迴鐵路]]通車典禮，開始正式營業。
*1999年12月31日至2000年1月1日：行駛「迎接[[千禧年|千禧年]]第一道曙光」專開列車。
*2016年6月28日：啟用多卡通刷卡機。
