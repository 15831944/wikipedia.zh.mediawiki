'''太上皇后'''，是[[太上皇|太上皇]][[正妻|正妻]]的称号，因夫君在世，為有所區別而不稱[[皇太后|皇太后]]。也有部分太上皇的正妻被加封为皇太后，或在“[[皇后|皇后]]”前加上徽号。详见'''[[太后#太上皇后|太后]]'''。

==中國歷史上的太上皇后==
# [[太上皇后_(漢太上皇)|某氏]]，西汉太上皇[[劉太公|刘太公]]的继妻，汉高祖[[刘邦|刘邦]]尊父亲为太上皇，继母为太上皇后。
# [[胡皇后_(北齐武成帝)|胡氏]]，北齐武成帝[[齐武成帝|高湛]]的皇后。565年，高湛禅位给太子[[齐后主|高纬]]，自任太上皇，胡皇后为太上皇后。
# [[穆邪利|穆邪利]]，小字黃花，北齊後主[[齐后主|高緯]]第三任皇后。577年，北周攻陷晋阳，高纬传位幼主[[齊幼主|高恒]]，高恒尊母亲穆邪利为太上皇后。
# [[莊憲皇后|庄宪皇后王氏]]，唐顺宗[[唐顺宗|李诵]]的良娣。805年，顺宗即位不久便被迫传位于太子[[唐宪宗|李纯]]，退位为太上皇，王良娣被儿子尊为太上皇后。
# [[何皇后_(唐昭宗)|宣穆皇后何氏]]，唐昭宗[[唐昭宗|李晔]]的皇后，史稱積善太后。900年，刘季述迎立其子[[李𥙿|李𥙿]]为皇帝，尊昭宗为太上皇，何皇后为太上皇后。
# [[顯肅皇后|显肃皇后郑氏]]。宋徽宗[[宋徽宗|赵佶]]的第二任皇后。1126年，宋徽宗禅位于子[[宋钦宗|趙桓]]，欽宗即位後尊鄭皇后為太上皇后。
# [[宪圣慈烈皇后|宪圣慈烈皇后吴氏]]，宋高宗[[宋高宗|赵构]]的第二任皇后。1162年，高宗禅位于[[宋孝宗|赵昚]]，吴皇后为太上皇后。
# [[李鳳娘|慈懿皇后李鳳娘]]，宋光宗[[宋光宗|赵惇]]的皇后。1200年，光宗讓位于其子[[宋宁宗|趙擴]]后，李鳳娘成為太上皇后。

===未得太上皇后稱號的太上皇正妻===
# [[成肅皇后|成肅皇后謝氏]]，宋孝宗[[宋孝宗|赵眘]]的第二任皇后、第三位正妻。1187年，高宗崩。1189年，孝宗禅位于[[宋光宗|赵惇]]，孝宗自稱太上皇，謝皇后为壽成皇后，為實質的太上皇后。
# [[孝莊睿皇后|孝莊睿皇后錢氏]]，明英宗[[明英宗|朱祁鎮]]的皇后。1449年，英宗親征瓦剌，被俘。弟郕王[[明景帝|朱祁鈺]]監國，不久登基。遙尊英宗為太上皇，後被迎回。[[土木之變|土木之變]]後直至到[[奪門之變|奪門之變]]，錢皇后未得到太上皇后的尊封。

==日本的太上皇后==
日本的天皇於退位後稱[[太上天皇|太上天皇]]，而出家的太上天皇則稱[[太上法皇|太上法皇]]。然而不論是太上天皇或太上法皇，其正妻（[[皇后|皇后]]或[[中宮|中宮]]）皆未有如中國一般擁有「太上皇后」一號。日本歷史上太上天皇、太上法皇的正妻，有的被加封為[[皇太夫人|皇太夫人]]（如[[藤原溫子|藤原溫子]]）、或是被加封為[[皇太后|皇太后]]（如[[昌子內親王|昌子內親王]]），又或得到[[女院|女院]]的封號及地位。

日本[[今上天皇|今上天皇]]（[[明仁|明仁]]）將於2019年4月30日[[禪讓|內禪]]其長子－－[[皇太子|皇太子]][[德仁親王|德仁親王]]。日本政府為此制定新稱號「上皇」，用以稱呼退位後的明仁天皇；而明仁天皇之妻-[[美智子皇后|美智子皇后]]亦將隨其夫一同退位，由德仁親王的正妻－－[[雅子妃|雅子太子妃]]繼位為新任皇后。而有關美智子皇后在退位後的稱號，由於皇太夫人一號自[[平安時代|平安時代]]晚期後已不使用，女院的封號及相關制度也在[[明治維新|明治維新]]後被廢除，雖然[[皇室典範|皇室典範]]明確規範前任天皇的皇后稱為皇太后，但近代的幾位皇太后，都是在丈夫過世後因新皇登基而從先帝皇后升格為皇太后（如[[香淳皇后|香淳皇后]]、[[貞明皇后|貞明皇后]]），或是做為[[國母|國母]]逝世後的諡號（如[[昭憲皇太后|昭憲皇太后]]、[[英照皇太后|英照皇太后]]），因此若美智子皇后在退位後改號為皇太后，難免予人「先皇駕崩」的觀感，因而日本政府亦制定新封號「上皇后」，用以在將來稱呼隨明仁天皇一同退位的美智子皇后<ref>[http://news.ltn.com.tw/news/world/breakingnews/2031800 天皇退位條款 擬限當今天皇]</ref>。
