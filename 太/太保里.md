'''太保里'''隸屬於[[嘉義縣|嘉義縣]][[太保市|太保市]]，東鄰[[後潭里|後潭里]]，南接[[春珠里|春珠里]]，西鄰[[崙頂里|崙頂里]]，北隔[[新埤里|新埤里]]<ref name=":0">{{Cite web|url=http://taibao.cyhg.gov.tw/cp.aspx?n=68AF3B64ECE864A9|title=太保市市公所里辦公室|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。

  太保里，其前身為太保鄉，有『太保村』、『太保莊』等舊名，而其最早原名為『溝尾莊』。

  [[明鄭時期|鄭氏時期]]時，已由潮州普寧人張瑞來台開拓。康熙六十年(1721年)根據[[藍鼎元|藍鼎元]]計載之文獻，可知於康熙末時溝尾莊以分為溝尾前莊、溝尾後庄二個庄頭，今日之太保里即為當時之溝尾前莊，又名為前溝尾莊<ref>{{Cite book|title=太保市志|last=|first=|publisher=|year=|isbn=|location=|pages=}}</ref>。

 [[臺灣清治時期|清領時期]]時為紀念曾住在前溝尾庄的太子太保—閩浙水路提督[[王得祿|王得祿]]將軍逝世，除追封他為伯爵，也將前溝農莊更名為『太保』<ref name=":1">{{Cite web|url=http://gn.moi.gov.tw/geonames/index.aspx|title=地名資訊服務網|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。

== 地理 ==
太保里有二條交通主要幹道，分別為南北、東西走向，嘉四九號的麻太路和一六八線的嘉朴公路。而其中的嘉朴公路既連接中山高速公路和東西快速道路，同時亦為高速鐵路嘉義站之所在<ref name=":0" />。

太保里內居民宗教信仰多元，而以道教為大宗，後依信仰比例多寡依序為佛教，基督教和天主教<ref name=":0" />。

寺廟奉祀主神有七星娘娘、三山國王、玄天上帝......等。

== 聚落歷史 ==
鄭氏時期，潮州普寧人張瑞來台開拓。

  居民以傅姓、張姓為大宗，約有上百戶住民，同時於溝尾有一聚落加走庄，在其勢力逐漸消弭後，太保才漸漸富庶<ref name=":1" />。

== 教育 ==
太保里內建有太保國小、[[太保國中|太保國中]]兩教育機構。
太保國小創立於民國前十二年(1899年)，現任校長為連國欽<ref name=":2">{{Cite web|url=http://schbase.cyc.edu.tw/schbase106/|title=嘉義縣學校資料庫|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。

太保國中於民國五十七年(1968年)八月一日創立，為順應同年延長九年國民教育。馬痕吉為創校校長，現任校長為邱獻萱<ref name=":2" />。

== 文化資產 ==
太保里內設有寺廟，為里內之信仰中心 

  [[福濟宮|福濟宮]]

  張氏於清出康熙元年自福建泉州府渡海來台謀生，以七星娘娘為信仰於此地。當初興建之神宇較為簡陋，以木料草茸搭建。

  嘉慶十七年(1812年)，王得祿為擴建提督衙，遷原廟址於太保段三零七號，至民國六十五(1976)年承七星娘娘輿旨，擇建於舊址。後向外募捐，眾信眾於民國六十七(1978)年十月協力完成廟殿<ref name=":3">{{Cite web|url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/taibau/1001018-XBG/view|title=文化資訊地理資訊系統|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。

[[玄保宮|玄保宮]]

  玄保宮於民國七十七年(1988年)建立，以管理人制為其組織型態。以選舉方式產生管理人。張恰為目前主任委員，廟有信徒27人<ref name=":3" />。

== 景點 ==
太保里內有王氏家廟、市立兒童公園、石獅等<ref name=":0" />。

王氏家廟

太保里中的王氏家廟，供奉出生於太保市，當時官位最高的台灣人—王得祿<ref name=":1" />。

== 農業 ==
太保里位屬[[嘉南平原|嘉南平原]]，給水由於灌溉設施的便利而無虞，提高了生產量。其主要農作物有稻米、香瓜、玉米和番茄......等<ref name=":0" />。
