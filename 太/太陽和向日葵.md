《'''太陽和向日葵'''》（{{lang-ja|太陽と向日葵}}）是日本女子組合[[Flower_(組合)|Flower]]的第5張單曲，於2013年8月7日由[[日本索尼音樂娛樂|Sony Music Associated Records]]發行。

== 概要 ==
與前作《[[恋人是聖誕老人#FLOWER的單曲|戀人是聖誕老人]]》相隔約9個月後推出的新作品。本作與同所屬[[E-girls|E-girls]]的姊妹團體[[Happiness_(組合)|Happiness]]之單曲《{{link-ja|Sunshine Dream 〜只得一次的夏天〜|Sunshine Dream 〜一度きりの夏〜}}》同日發行，象徵對決的意味。<ref>{{Cite web |title=HappinessとFlowerが、“武者修行”対決シングル同時発売|url=http://natalie.mu/music/news/96665 |publisher=[[Natalie_(網站)|Natalie]] |date=2013-08-07 |accessdate=2017-05-06 |language=ja}}</ref>〈太陽與向日葵〉由[[鷲尾伶菜|鷲尾伶菜]]、{{link-ja|武藤千春|武藤千春}}及{{link-ja|市來杏香|市來杏香}}擔任主唱，是Flower首次嘗試三主唱的主題曲<ref group="注">過去收錄於《[[Still_(Flower單曲)|Still]]》的B面曲《Fadeless Love》也以三主唱形式演唱。</ref>

本作分為「初回生產限定盤」、「通常盤」、「期間生產限定盤」三種版本發售。「初回生產限定盤」收錄其他兩版本未有的歌曲〈what i want...〉，並附有音樂錄影帶的DVD和相片本。此外，「期間生產限定盤」至同年9月底為止限定發行，僅收錄〈太陽和向日葵〉一首歌曲。

== 銷售成績 ==
本作於[[Oricon公信榜|Oricon公信榜]]統計中，週榜最高排名五位，在榜時間共七週<ref name="oricon" />。[[Billboard_JAPAN|Japan Hot 100]]方面，最高排行第七名<ref name="Billboard" />。另外在開放音樂下載的首日，獲得[[RecoChoku|RecoChoku]]日榜首位<ref>{{Cite web |title=
Flower、対決シングル「太陽と向日葵」が配信初日に着うた(R)1位獲得＆MVショートVer.視聴スタート |url=https://www.barks.jp/news/?id=1000092187 |publisher={{link-ja|BARKS|BARKS}} |date=2013-07-10 |accessdate=2017-05-06 |language=ja}}</ref>。
