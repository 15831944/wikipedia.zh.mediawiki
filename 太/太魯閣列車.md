'''太魯閣列車'''（英語譯名：Taroko Express），全稱：'''太魯閣自強號列車'''，通稱：'''太魯閣號'''，為[[臺灣鐵路管理局|臺灣鐵路管理局]]以2006年引進之[[擺式列車|傾斜式]][[電聯車|電聯車]][[台鐵TEMU1000型電聯車|TEMU1000型]]開行的特快車，初期行駛[[宜蘭線|宜蘭線]]、[[北迴線|北迴線]]，2008年2月25日首次行駛[[花蓮車站|花蓮]]－[[彰化車站|彰化]]，原[[台北車站|臺北]]－[[花蓮車站|花蓮]]間的[[自強號列車|自強號]]最快行駛時間需約2小時20分鐘，以太魯閣列車行駛將可縮短至2小時，[[花蓮車站|花蓮]]－[[彰化車站|彰化]]間為4小時12分，由於台中鐵路高架化，太魯閣號大舉退出山線，改行駛海線，西部幹線僅剩278次及285次。

== 命名 ==
列車名稱不再採用傳統[[自強號列車|自強]]、[[莒光號列車|莒光]]、[[復興號列車|復興]]等以[[中華民族|中華民族]][[愛國教育|愛國教育]]口號命名之方式，而改向民間徵求命名，由[[中華民國鐵道文化協會|中華民國鐵道文化協會]]整理出「曙光／AURORA」、「飛魚／FLYING FISH EXPRESS」、「太魯閣／TAROKO」等名稱，再由臺灣鐵路管理局內部開會選出<ref>[http://www.epochtimes.com/b5/6/1/17/n1193173.htm 台鐵新購傾斜式列車 命名太魯閣].大紀元時報.2006-01-18 </ref>。

[[臺灣鐵路管理局|臺灣鐵路管理局]]於2006年1月17日評定由「太魯閣」獲選。命名理由為：[[太魯閣國家公園|太魯閣國家公園]]之自然風光聞名國內外，其[[英語|英語]]名稱轉音自太魯閣語「''Taroko''」（原語拼音作'''Truku'''），無論[[漢語|漢語]]、[[閩南語|閩南語]]、[[客家語|客家語]]、[[日語|日語]]、[[英語|英語]]等皆易於記憶。結果出爐後，2006年1月20日、3月15日、6月1日及2007年1月21日、5月8日出版之免費[[時刻表|時刻表]]皆以該型車為封面。其中3月15日、6月1日之版本更於內頁介紹列車基本資料及命名沿革，透過新聞媒體，民眾也熟知「太魯閣號」這個名稱。

雖然通稱的「太魯閣號」這個名稱已廣為使用，且顯示於部分車站的最近發車時刻表，但臺鐵為避免票種過度複雜，亦配合其簡化車種的政策，故其票價比照普通[[自強號列車|自強號]]，稱為「太魯閣自強號列車」。台鐵發行的時刻表將太魯閣列車的班次標為「太魯閣」；車票上除標示自強外，並特別標示「太魯閣」、「Taroko Express」以供識別。

== 營運概況 ==
*太魯閣號因過彎傾斜易使站立乘客發生危險，故實施全車強制對號入座，不發售無座票，無票或不是持有該班次太魯閣號車票的旅客（含持悠遊卡、一卡通等電子票證、定期票、其他日期車次車種的車票者）不得搭乘，否則將依規定重新補票並加收50%票價。

== 使用列車 ==
*[[台鐵TEMU1000型電聯車|台鐵TEMU1000型電聯車]]是台鐵首次引進的[[傾斜式列車|傾斜式列車]]<ref>邑采客服.[http://www.inn88.com.tw/blog/rewrite.php/read-89.html 台鐵傾斜式列車~太魯閣號 (花蓮V.S台北真便利)].花蓮邑采精緻旅店的blog .2009-11-25 </ref>，使得它在彎道可以較非傾斜式列車快約25km／h的速度行駛，在多彎的[[宜蘭線|宜蘭線]]可大幅縮短[[八堵車站|八堵]]－[[頭城車站|頭城]]路段的行車時間。

== 歷史事件 ==
*2007年5月8日，開始營運[[台北車站|臺北]]－[[花蓮車站|花蓮]]間的路線，並創下該區間最快行車時間紀錄。
*2008年2月2日，第二批進口的[[台鐵TEMU1000型電聯車|TEMU1000型]]傾斜式電車將於2008年春節以加班車的方式行駛臺北－員林間的直達車，該加班車創下臺鐵有史以來於此區間最短行車時間的紀錄。
*2008年2月25日，於2008年春節期間以加班車方式行駛的[[花蓮車站|花蓮]]－[[員林車站|員林]]（後因[[員林車站|員林]]進行高架化工程無法提供車輛長時間停靠）間的太魯閣號改為定期班次，每天各來回一班。
*2008年3月1日，臺鐵宣布日後將提供一列太魯閣號作為包車使用，供機關團體、公司行號搭配旅遊及商務行程使用，稱為「太魯閣專車」；但是逢週五六日或連續假日時，太魯閣號將會全數上線營運，此時是無法包車的。
*2008年5月15日，大量增發定期班車。
*2008年9月20日，臺北鐵路地下化前最後一段平面鐵道由1074次太魯閣號為最後一班南下火車，亦為最後一列停靠平面松山火車站的火車。
*2008年10月24日，因東部幹線假日的運能嚴重不足，逢週五到週日時一日增開4班次。
*2009年10月27日，春節加班車延駛至[[田中車站|田中]]。
*2010年12月22日起，新時刻表標示變更為'''太魯閣號'''。
*2012年1月17日，第278次（編組TED1009-1010）由[[田中車站|田中]]→[[花蓮車站|花蓮]]，在[[埔心車站|埔心車站]]南方'''幸福水泥'''平交道撞上闖越平交道的砂石車，火車頭全毀、司機殉職，並導致假日班次只能以其他車種代替。詳見「'''[[臺鐵埔心平交道事故|臺鐵埔心平交道事故]]'''」
*2012年4月21日，第275次（已停駛）下午5:35經[[竹北市|竹北市]]光明地下道上方路段，電車線遭扯斷約2公里，事件導致[[新竹車站|新竹]]－[[竹北車站|竹北]]間鐵路僅能單線通行，約63班列車、近3萬旅客受影響。
*2014年4月29日，由於[[台鐵TEMU1000型電聯車|TEMU1000]]進行維修，206、282、271、247次自4月29日起由太魯閣號改為[[普悠瑪號|普悠瑪號]]替駛。
*2014年7月16日，路線大改點，首度行駛海線及山線，西部延伸至[[斗六車站|斗六]]、東部延伸至[[壽豐車站|壽豐]]。
*2014年8月22日，因149次PP自強號的機車故障，請283次'''太魯閣號'''到斗六站後繼續延駛到高雄站，以8708次回送到樹林，這是'''太魯閣號'''首次行駛到南部。
*2015年2月春節假期首度於[[樹林車站|樹林]]－[[花蓮車站|花蓮]]／[[臺東車站|台東]]間試辦'''太魯閣號'''、[[普悠瑪號|普悠瑪號]]加班車部分或全部車廂自由座，全車自由座座位372人、立位186人，這是'''太魯閣號'''以加班車的名義首次營運至[[臺東車站|台東]]；並限定於乘車當日上午6時至車站窗口購票，不適用網路、電話語音及[[超商|超商]]訂票。日後每逢連續假日也很有可能比照辦理。
*2015年10月15日，台鐵大改點後本型列車替駛[[樹林車站|樹林]]－[[花蓮車站|花蓮]]－[[臺東車站|台東]]（[[知本車站|知本]]）間原普悠瑪的班次 ( 402、426、438、401、417、441次 )，至於原本的太魯閣班次（232、278、280、232、288、203、273、283、285次），則是全改[[普悠瑪列車|普悠瑪號]]行駛。
*2016年3月21日，台鐵改點後，台鐵推出太魯閣Hello Kitty彩繪列車<ref>黃立翔.[http://news.ltn.com.tw/news/life/breakingnews/1628663 千呼萬喚！台鐵Hello kitty太魯閣3月21日首航].自由時報.2016-03-11</ref>，並推出周邊商品，深受民眾喜愛每逢五至日行駛426-441車次，周一到周四僅提供包車使用。
*2016年4月21日，推出太魯閣國家公園彩繪列車 (馬白水彩繪列車)<ref>陳裕謀.[http://www.railway.gov.tw/upload/indexnews/upload/10986/1050412%E6%96%B0%E8%81%9E%E7%A8%BF-%E9%A6%AC%E7%99%BD%E6%B0%B4%E5%BD%A9%E7%B9%AA%E5%88%97%E8%BB%8A.odt 搭太魯閣列車去太魯閣　台鐵推出「馬白水太魯閣之美彩繪列車」新聞稿].交通部臺灣鐵路管理局.2016-04-12</ref>，首度行駛到花蓮，第一班行駛到花蓮的班次為5934次。並將原本由本型列車行駛的438、401次再度改回由[[普悠瑪號|普悠瑪號]]行駛，202、236、252、207、217、237次則改由本型列車行駛及增開固定逢周六使用馬白水彩繪太魯閣列車 行駛 291、292車次行駛彰化到花蓮到2016年10月15日最後行駛山線也意外復活。
*2016年6月27日，一列由[[樹林車站|樹林]]開往[[知本車站|知本]]的426次太魯閣號，在行經宜蘭[[礁溪車站|礁溪]]時發生故障，台鐵緊急派出一列[[普悠瑪號|普悠瑪號]]繼續南下，造成旅客誤點76分<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20160627/895432/ 太魯閣號故障延遲76分鐘，370名旅客受影響].蘋果日報.2016-06-27 </ref><ref>江志雄.[http://www.newsjs.com/url.php?p=http://news.ltn.com.tw/news/life/breakingnews/1743535 太魯閣號故障暫停礁溪 300名乘客受影響].自由時報.2016-06-27 </ref>。
*2016年8月8日，配合「Hello Kitty上阿里山『森旅行』」活動，由太魯閣Hello Kitty彩繪列車行駛，以加班車的名義行駛於[[南港車站|南港]]－[[屏東車站|屏東]]間<ref>汪淑芬.[http://www.cna.com.tw/news/ahel/201608020195-1.aspx 台鐵Kitty彩繪車 父親節首到南部].中央通訊社.2016-08-02 </ref>。
*2016年10月20日，台鐵大改點後原來的278車次、285車次普悠瑪號改由太魯閣號行駛，將增停[[南港車站|南港]]，減停[[松山車站|松山]]並延駛至[[員林車站|員林]]，由[[樹林車站|樹林]]－[[花蓮車站|花蓮]]－[[臺東車站|台東]]本型的（207車次、236車次、237車次、445車次）改為普悠瑪號行駛。另外，為了配合臺中高架化的月臺，將太魯閣列車暫時退出山線（取消291及292車次 花蓮彰化間往返）待階梯改造完成後才會繼續行駛，太魯閣Hello Kitty彩繪列車增班 固定逢週五402-417-436 逢週六405-426-441 逢週日、一 206-217-238-239。
*2017年1月24日，因台中鐵路高架化月台原因首次開進台中高架鐵路試運轉。
*2017年2月1日，285次(花蓮開往員林)21:38於新竹=香山間東線農會前平交道(108公里+261公尺)處，撞到闖越摩托車，騎士無受傷，經路警處理285次22:25(晚50分)開車，路線恢復正常行駛，共影響9列次、258分、旅客約2076人。
*2017年4月27日，台鐵改點，228、229次普悠瑪號改為本車行駛；278、285車次減停[[南港車站|南港]]，增停[[松山車站|松山]]，回歸過去的停站方式。
*2018年2月24日，因海線苑裡=新埔=白沙屯間電車線異常，285次馬白水太魯閣號列車 改行經山線，竹南後僅停靠苗栗站後直駛至彰化站('''台中高架段不停靠''')，是首次有正班車開進台中高架路段之記錄。
*2018年5月3日，因556次莒光號電力機車頭故障(於苑裡更換機車)，緊急調派本型車代打樹林=花蓮段(原車狀況排除後繼續行駛且不載客，僅辦理行包業務)，為首次代打其他車種之記錄。
*2018年10月29日，[[中華民國交通部|交通部]][[中華民國交通部長|部長]][[吳宏謀|吳宏謀]]宣佈10月30日開始，[[太魯閣列車|太魯閣]]和[[普悠瑪|普悠瑪]]單人駕駛先安排雙駕駛機制<ref>{{cite news|author= |title=補救台鐵安全 3年內補足3千人力!|url=https://news.cts.com.tw/cts/politics/201810/201810291941361.html|accessdate=2018-10-29|newspaper=《[[華視新聞|華視新聞]]》 |date=2018-10-30}}</ref>。
