《'''太陽的哀悼歌'''》（{{lang-ja|たいようの哀悼歌}}）是日本女子組合[[Flower_(組合)|Flower]]的第15張單曲，於2017年8月23日由[[日本索尼音樂娛樂|日本索尼音樂娛樂]]發行。

== 概要 ==
與前作《[[MOON_JELLYFISH|MOON JELLYFISH]]》相隔約四個月後的單曲，2017年的第三張單曲。以「初回生產限定盤A(CD + DVD)」、「初回生產限定盤B(CD + DVD)」、「通常盤」、「期間生產限定盤」四種版本發售。<ref>{{Cite web |title=15thシングル「たいようの哀悼歌」アートワーク＆収録内容詳細が解禁！ |url=http://www.flower-ldh.jp/info/detail.html?id=484482&paged=3 |publisher=Flower Official Website |date=2017-07-29 |accessdate=2017-09-17|language=ja}}</ref>「初回生產限定盤」附有32頁相冊的袖套規格，「期間生產限定盤」是電視動漫《未來國家的Altair》作封面<ref name="anime">{{Cite web |title=最新シングル「たいようの哀悼歌」、テレビアニメ「将国のアルタイル」のエンディングテーマに決定！ |url=http://www.sonymusic.co.jp/artist/flower/info/482507?_ga=2.93877208.47540597.1497163293-252159711.1497163293 |publisher=Flower Sony Offical Website |date=2017-06-06 |accessdate=2017-09-17|language=ja}}</ref>。

標題歌〈太陽的哀悼歌〉是2017年7月播放的電視動漫《未來國家的Altair》的片尾曲。<ref name="anime"/>自第11張單曲《瞳孔深處的銀河》再次成為電視動畫片尾曲。
「初回生產限定盤」收錄的〈熱帶魚的淚 Asiatic version〉與2014年6月發售的《熱帶魚的淚》是不同版本。

「初回生產限定盤A」附屬的DVD，收錄了在2016年10月至12月進行的單獨巡演並在2017年1月份舉行的「Flower Theater 2016 〜THIS IS Flower〜 THE FINAL」片段，約110分鐘全22首歌曲收錄。

== 銷售成績 ==
在8月22日的[[Oricon|Oricon]]日間排名中，以40,000張銷售量排名1位。之後還有其他作品的追擊，保持首日的銷售量直到結束，在9月4日的Oricon週間排名中，以賣出58,000張排名1位。是組合出道7年來首次獲得Oricon週間1位。另外，在動漫週間排名中，繼《瞳孔深處的銀河》後連續兩次連續獲得1位。
