'''太平輪沉沒事件'''發生於1949年（[[民國38年|民國38年]]）1月27日，中華民國中聯企業公司<ref>{{Cite web |url = http://blog.chinatimes.com/ways1234/archive/2008/04/24/272178.html |title = 尋找太平輪記事 前言 |author = 張典婉 |language = zh |accessdate = November 15, 2014 |6 =  |deadurl = yes |archiveurl = https://web.archive.org/web/20080425153915/http://blog.chinatimes.com/ways1234/archive/2008/04/24/272178.html |archivedate = 2008年4月25日 }}</ref>向太平洋船務公司租用的[[客輪|客輪]]「太平輪」因[[超載|超載]]（2,093噸）和於夜間航行时沒有開啟[[航行燈|航行燈]]，在[[上海|上海]]開往[[基隆|基隆]]的途中于在[[舟山群島|舟山群島]]海域的[[白節山|白節山]]附近與一艘載有2,700噸煤礦及木材的[[貨輪|貨輪]]「建元輪」相撞，兩船相继沉沒。太平輪上共證實932人喪生<ref name="蘋果日報20150603"/><ref name="東方日報20150603"/>，而建元輪則有72人罹難<ref name="鳳凰大視野"/>。[[澳洲|澳洲]][[軍艦|軍艦]]{{link-en|HMAS Warramunga|HMAS Warramunga (I44)}}救起34人，加上由舟山群島漁民所救的未記名人員，共只有50多人生還。<ref>[http://focus.news.163.com/10/0525/15/67HRB1HM00011SM9.html 中国的泰坦尼克号：太平轮沉没记]，网易</ref>

== 船隻 ==
太平輪是中聯企業公司的豪華客輪，[[排水量|排水量]]2,489總噸，由大股東[[周曹裔|周曹裔]]在上海所經營<ref>《太平輪1949》2009年 張典婉著 商周出版</ref>（中聯公司是由一群寧波同鄉集資興辦的輪船航運公司，總經理周曹裔，其餘還有四位股東：龔聖治、[[蔡天鐸|蔡天鐸]]、馬世燧、周慶雲）<ref>[http://dangshi.people.com.cn/BIG5/n/2013/1008/c85037-23119074-4.html 太平輪悲歌——64年前震驚海峽兩岸的沉船事件【4】]</ref>，租用自[[东莱银行|东莱银行]]开办的太平船務公司。[[國共內戰|國共內戰]]後期，大批的撤退國軍退出[[中國大陸|中國大陸]]。這些人用[[金條|金條]]或靠[[人際關係|人際關係]]換取艙位，擠上早已客滿的太平輪。1949年1月27日（[[農曆|農曆]][[除夕|除夕]]前一天），太平輪原預計上午10時出發，因等待裝運[[中華民國中央銀行|中華民國中央銀行]]的一批[[銀元|銀元]]，延至下午4時18分啟航。太平輪搭載的「最後一批乘客」，總近1000人（有票乘客508人，船員124人，無票者約300人）；另載有沉重貨物，包括600噸鋼條、《[[東南日報|東南日報]]》印刷器材與[[白報紙|白報紙]]100多噸、中華民國中央銀行重要文件1317箱、臺北[[迪化街|迪化街]]商家訂購的[[南北貨|南北貨]]等。

== 事件經過 ==
1949年1月27日，太平輪從[[上海市_(中华民国)|上海市]]開往[[基隆市|基隆]]的途中，於夜間航行时為逃避[[宵禁|宵禁]]，未開航行燈，晚間23時45分在[[舟山群島|舟山群島]]海域的[[白節山|白節山]]附近（北緯30°25'，東經122°<ref>[http://maps.google.com/maps?f=q&source=s_q&hl=en&geocode=%20&q=N30%C2%B025',+E122%C2%B0&sll=30.145127,122.34375&sspn=5.793664,11.601562&ie=UTF8&ll=30.415519,122.000427&spn=1.444814,2.900391&z=9&iwloc=A 太平轮沉没地点在Google地图上的位置]</ref>）與一艘由基隆出發的載有2700噸煤礦及木材的「建元輪」{{NoteTag|[[無錫|無錫]]麵粉大王[[榮鴻元|榮鴻元]]所有。}}相撞。攔腰被撞的建元輪立刻沉沒，船上72人溺斃，有3人被救至太平輪；而太平輪在45分鐘後（1月28日0時30分）沉沒，此時船上的人幾乎都在慘叫，落水者因當時天寒而凍死或[[溺斃|溺斃]]，船上達932人罹難。

1月28日凌晨，[[澳大利亞皇家海軍|澳大利亞皇家海軍]]軍艦經過罹難地點附近海域，救獲35位（30男5女，30名男性中包括2名建元輪船員）乘客，但因在冬天海上漂流多時，其中一位女性终因體力不支而死亡。在澳洲軍艦的記錄中，確認救起的存活人員共計34人（30男4女）。另加舟山群島漁民陸續救起的未記名人員，總計生還者有50人左右。死者中不乏有名望、富商級的人物，包括赴台為[[國立音樂院|國立音樂院]]尋覓校地的音樂家[[吳伯超|吳伯超]]、前[[遼寧省政府|遼寧省政府]]主席[[徐箴|徐箴]]、著名球評[[張昭雄_(球評)|張昭雄]]之父張生、和刑事鑑定專家[[李昌鈺|李昌鈺]]之父李浩民等<ref>[http://gcontent.nddaily.com/e/8d/e8dfff4676a47048/Blog/9a4/7cbbd6.html 今日海祭太平轮 大陆参与者居多] {{webarchive|url=https://web.archive.org/web/20100527095803/http://gcontent.nddaily.com/e/8d/e8dfff4676a47048/Blog/9a4/7cbbd6.html |date=2010-05-27 }}，南都網</ref>。而據生還者回憶，事發後曾有多艘船舶從旁經過，但皆未施救<ref>丁雯靜、陳郁婷，《太平輪：亂世傳奇真相》，臺北：八旗文化，2015年，頁51</ref>。

現時[[上海市档案馆|上海市档案馆]]尚留有当年的记录：{{cquote|太平（轮）十一点过白节灯山，十二时半又回白节灯山边沉没。<br>太平（轮）十一点四十五分发求救信号……两船相撞处约在白节山东南二海里。}}
而檔案館檔案旁边的手绘示意图显示，由西北向东南方向行驶的太平轮在白节山南侧与建元轮相撞，太平轮企圖向白节山北岸的途中，在距離岸邊僅僅500公尺處，鍋爐爆炸，側翻沈沒。更讓人難以理解的是，撞船地點是白節山南方海域，而白節山的南岸是平坦的沙岸，適合擱淺，至少可保船上大部份人員安全，但船長卻繞西岸企圖在北岸漁港靠岸，結果在只差三分鐘船程的外海約 500公尺處沉沒。

太平轮沉沒後，海上漂流著許多[[珠寶|珠寶]][[首飾|首飾]]、[[佛像|佛像]][[牌位|牌位]]、木箱文牘及[[棉花|棉花]]等，令附近的漁民大吃一驚。

太平轮失事後，媒體歸納了超載、超速、抄捷徑三條主因，認為这是導致海難的致命因素，但根據近年來的科學分析，这些因素似乎都難以成立<ref>丁雯靜、陳郁婷，《太平輪：亂世傳奇真相》，臺北：八旗文化，2015年，頁154</ref>。不過，太平輪預定的起航時間為當日上午10點，卻延宕至下午4點後方才起航，起航前曾有600公噸鋼條凌亂加載在艙面，發貨人與收貨人都難以查找，恐加劇了海難的發生。

另外，亦有傳聞稱太平輪的失事與國共內戰中的兩黨鬥爭有關。確有生還者表示，在碰撞發生後向岸邊搶灘的過程中，在距離岸邊500時前船體發生了爆炸，導致最終的傾覆。而據調查，在當時中聯公司的五位船東中，有四位是共產黨的地下黨員<ref>丁雯靜、陳郁婷，《太平輪：亂世傳奇真相》，臺北：八旗文化，2015年，頁134</ref>。且三等艙旅客張漢也是中共地下黨員，其船票正是從地下黨同志手中獲得<ref>丁雯靜、陳郁婷，《太平輪：亂世傳奇真相》，臺北：八旗文化，2015年，頁188</ref>。

== 事件之後 ==
*[[中華民國|中華民國]][[上海臨時法院|上海臨時法院]]判決中聯企業公司賠償，但當時上海在事件後[[上海戰役|短短四個月]]即被[[中国人民解放軍|中国人民解放軍]]佔領，賠償金額又屬[[天文數字|天文數字]]。雪上加霜的是，太平輪的保險公司「[[上海華泰保險公司|上海華泰保險公司]]」在出事後立刻宣布[[倒閉|倒閉]]，因此中聯企業公司必須自負盈虧賠償。中聯企業公司最後結束營運。中聯企業公司旗下兩艘輪船被鐵鍊鎖在[[高雄港|高雄港]]，最後全部鏽爛，另外兩艘貨輪被[[中國共產黨|中國共產黨]]接收。另外一艘客貨輪「華聯輪」於1950年過戶給中聯，於1951年1月13日在[[東北季風|東北季風]]中觸礁沉沒，打撈[[拆船|拆船]]後船殼於基隆港當做[[躉船|躉船]]使用。
*1951年，[[基隆港|基隆港]]東10碼頭[[海軍基隆後勤支援指揮部|海軍基隆後勤支援指揮部]]營區立碑「太平輪遇難旅客紀念碑」，由[[于右任|于右任]]題字。
*2004年11月，[[民主進步黨|民主進步黨]]族群事務部與[[鳳凰衛視|鳳凰衛視]]合作拍攝[[紀錄片|紀錄片]]《尋找太平輪》。
*2012年4月2至4日，[[旺旺中時媒體集團|旺旺中時媒體集團]][[長天傳播|長天傳播]]製作紀錄片《驚濤太平輪：1949東方鐵達尼號沉船之謎》在[[中天新聞台|中天新聞台]]首映，[[丁雯靜|丁雯靜]]與[[洪慧真|洪慧真]]擔任製作人，[[陳文茜|陳文茜]]擔任主持人，[[金士傑|金士傑]]負責[[說書|說書]]。

== 61週年紀念海祭 ==
首次海峽两岸的太平輪罹難者家屬一同進行的海祭儀式，由長天傳播代表洪慧真（《驚濤太平輪》製作人）與《太平輪一九四九》作者張典婉一同發起。

海祭行程計畫半年多，因天氣因素影響船隻出海狀況，並且生還者與罹難者家屬多數為年逾七十的老人家，為安全上的第一考量，海祭日期延後原定日期兩天後舉行。

2010年5月25日上午七時，由舟山群島的嵊泗港口出海，海祭船隻先抵達太平輪沉沒地點─白節山海域，由洪慧真主持儀式，帶領罹難家屬與親友們，一同在沉船地點獻上一千朵白菊花，以及罹難者家屬之一黃似蘭女士所準備的一千隻紙鶴，表示對太平輪千餘位罹難人員的哀悼與紀念。

之後轉向生還者被澳洲軍艦救起的地點停留，罹難者家屬與親友們以50朵紅玫瑰獻給一同參與海祭的太平輪生還者─葉倫明先生與王兆蘭女士，表示對太平輪事件獲救的人員祝福與重生。儀式結束後，原船人馬一同返回嵊泗港口，當日約中午時間全員平安抵達，為海祭畫下完美句點。<ref>[http://gcontent.nddaily.com/c/f5/cf50b28ef624912f/Blog/e9d/2fe26f.html 中国的泰坦尼克号太平轮离散记忆] {{webarchive|url=https://web.archive.org/web/20100529154719/http://gcontent.nddaily.com/c/f5/cf50b28ef624912f/Blog/e9d/2fe26f.html |date=2010-05-29 }}，南都网</ref>
