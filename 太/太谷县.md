'''太谷'''是山西省晋中市下辖县，面积1034平方公里，人口30万。县人民政府驻明星镇。

== 历史 ==
[[春秋|春秋]]时称'''阳邑'''，[[隋|隋]]时更名太谷。

太谷与邻近的[[平遥县|平遥]]、[[祁县|祁县]]一样，都是晚清时期中国的金融中心之一，晋商云集之所，历来有“金太谷、银祁县、吃不完米麵的榆次县”之赞誉。

== 地理与气候 ==
太谷东南为山地，西北地势平坦。

气候有华北高地特征。

==交通==
[[File:Taigu_Railway_Station_2013.08.27_12-04-13.jpg|300px]]]]
[[南同蒲铁路|南同蒲铁路]]，[[太焦铁路|太焦铁路]]、[[大运公路|大运公路]]、太长高速、龙城高速、108国道、102省道、大西高铁途经县境。

== 经济 ==
太谷是农业县，主要经济作物有小麦、玉米、高粱、各种杂粮。经济果林有壶瓶枣、苹果、梨子、桃等。

==物产==
特产[[太谷饼|太谷饼]]、[[龟龄集酒|龟龄集酒]]、定坤丹、壶瓶枣。

== 方言 ==
[[太谷话|太谷话]]属[[晋语|晋语]][[并州片|并州片]]方言，有五个声调，其中两个入声调，保留有喉塞音的入声韵尾。

== 教育 ==
境内的[[太谷中学|太谷中学]]、太谷二中在附近较有影响。
[[山西农业大学|山西农业大学]]、山西农业大学信息学院、[[山西省交通技师学院|山西省交通技师学院]]等也都在太谷境内。原[[太谷师范学校|太谷师范学校]]经与其他师范学校合并，成立[[晋中师范高等专科学校|晋中师范高等专科学校]]，已迁往榆次区办学。
