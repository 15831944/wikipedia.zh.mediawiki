[[File:Taikoo_Shing_Road_and_Tai_Fung_Avenue,_intersection_(Hong_Kong).jpg|thumb]]'''太古城'''（'''Taikoo Shing'''）是一個位於[[香港島|香港島]][[東區_(香港)|東區]][[鰂魚涌|鰂魚涌]]的大型住宅屋苑，由[[太古地產|太古地產]]於1975年4月開始興建，後來成為[[港島東_(地產物業)|港島東]]的其中一部分。

太古城是香港島其中一個[[中產|中產]]居民為主的地區，居民收入與其餘中產區如[[西半山|西半山]]的中環半山相若，根據2011年香港人口普查，太古城東的收入中位數為62,370<ref>http://www.census2011.gov.hk/tc/district-profiles/ca/eastern/c02.html</ref>，太古城西的收入中位數為60,040<ref>http://www.census2011.gov.hk/tc/district-profiles/ca/eastern/c01.html</ref>，為東半山區（即70,000）的90%<ref>http://www.census2011.gov.hk/tc/district-profiles/ca/central-and-western/a02.html</ref>。

== 歷史 ==
太古城的前身為[[太古船塢|太古船塢]]（於1907年建成），它的紀念碑仍在[[太古城中心|太古城中心]]前展示。1970年代太古船塢遷出後，原址大片土地被太古地產分階段重新發展，第一期「翠湖台」先於1977年落成。其中洞庭閣以呎價$220開售，220個單位於一日內全數售罄。<ref>{{cite news|author1=邵超|title=一樓一古：神話不滅　太古城|url=http://hk.apple.nextmedia.com/supplement/culture/art/20130403/18214837|accessdate=2016-12-10|publisher=蘋果日報|date=2013-04-03}}</ref>

其後第二期至最新的第十一期「海天花園」於1979年至1987年分階段完成<ref>{{cite news|title=太古城兩房1,110萬元成交，實用呎價半年貴12%！|url=http://businessfocus.presslogic.online/2018/08/13/article/72541/amp/|accessdate=2018-10-02|publisher=蘋果日報|date=2018-08-13}}</ref>。在太古城中心第二期完成後，區內再沒有新住宅興建。而在太古城中心第三期及第四期兩幢商廈於1992年入伙後，太古集團在1993年亦把太古城中心第一期重建成商廈。而「好運街市」及栢嘉商業大廈舊址，則改建名為「[[東隅|東隅]]」的四星級酒店，於2010年1月開業<ref>「東隅」新聞稿：[http://www.east-hongkong.com/2009-05-13_Swire_Hotels_to_open_EAST_tc.pdf 充滿時尚生活品味的休閒式商務酒店「東隅」定於2010年1月開業]（2009年5月13日）</ref>。

== 概況 ==
太古城佔地53畝，於1977年至1987年分階段入伙，內建有61幢住宅大廈，共提供12,698個單位，單位面積由585至1,237平方呎，人口超過3萬，儼如一個小城市。雖然樓齡較高，但由於鄰近太古坊和太古城中心等商業中心，因此受到一班於附近上班人士的歡迎。它亦是在港[[日本人|日本人]]及[[韓國人|韓國人]]的聚居地之一。

太古城為香港首個設有園藝花園及綠化平台設計的私人屋苑。平台花園亦擺放當年最頂尖香港藝術家的作品，如[[文樓_(雕塑家)|文樓]]和[[李福華|李福華]]等。屋苑共有43件雕塑作品，其中文樓作品最多。<ref>{{cite news |author1=邵超 |title=一樓一古：神話不滅　太古城 |url=https://hk.lifestyle.appledaily.com/lifestyle/culture/daily/article/20130403/18214837 |accessdate=2018-06-09 |publisher=蘋果日報 |date=2018-06-09}}</ref>

除平台花園外，部分樓宇底層亦設有商舖及停車場，商舖包括[[超級市場|超級市場]]、[[便利店|便利店]]、[[銀行|銀行]]、[[診所|診所]]及食肆等，方便居民。

太古城向海邊的北面是[[鰂魚涌公園|鰂魚涌公園]]，南面向山一面則鄰接另一大型屋苑[[康怡花園|康怡花園]]與[[康山花園|康山花園]]。太古城西面建有有蓋[[行人天橋|行人天橋]]連接[[太古坊|太古坊]]。

== 太古城中心 ==
{{main|太古城中心}}
位於太古城中部的[[太古城中心|太古城中心]]，是香港島其中一個規模最大型的[[商場|購物中心]]，商場內設有多間[[百貨公司|百貨公司]]、各式[[店舖|店舖]]、[[酒樓|酒樓]][[食肆|食肆]]、美食廣場、[[戲院|戲院]]等，亦設有真雪[[溜冰|溜冰]]場。並設有3座[[寫字樓#辦公大樓|辦公大廈]]及1座[[酒店|酒店]]。現時太古城中心的辦公大樓與鄰近的[[太古坊|太古坊]]已成為了港島區的另一個商業中心圈。

== 交通 ==
太古城交通方便，在太古城中心地下（[[太古城道|太古城道]]路面）及五樓（[[英皇道|英皇道]]路面）設有[[巴士|巴士]]站，另外太古城中心地下亦設有的士站。區內有多條行人通道連接[[港鐵|港鐵]][[太古站|太古站]]。太古城亦設有道路出入口連接[[港島東區走廊|港島東區走廊]]，以及[[東區海底隧道|東區海底隧道]]（經[[西灣河|西灣河]]及東區走廊）。

== 政治與行政 ==
由於太古城人口眾多密集，它在[[東區區議會|東區區議會]]的選區分界被劃成東（C02）、西（C01）兩區，各佔一個議席。位於[[太古城道|太古城道]]以南、[[船塢里|船塢里]]以東的部份，嚴格來說並不是太古城的一部份，因為這部份其實在原太古船塢以外的地方。不過由於這部份只有單棟式大廈的匯豪峰有常住人口，而在區議會分界裡亦被劃進太古城西，所以那裡的工業大廈亦是太古城的一部份，而[[太古地產|太古地產]]亦對部分大廈進行收購。<ref>{{cite news |title=太古鰂魚涌兩工廈申強拍 估值逾57億創紀錄 |url=https://www.mpfinance.com/php/dailyp2.php?node=1517597568090&issue=20180203 |accessdate=2019-02-20 |publisher=明報 |date=2018-02-03}}</ref>

目前代表東、西兩區的區議員，分別是[[獨立人士|獨立人士]][[王振星|王振星]]及[[民主黨_(香港)|民主黨]][[趙家賢|趙家賢]]。

== 屋苑問題 ==
*發展商太古地產正計劃將第五期（即安盛臺）平台的休憩處和商舖改為食肆，有太古城居民擔心進一步將平台發展為食肆區，會吸引屋苑旁的港島東中心及外來的人士光顧，因而可能對屋苑的衛生環境和居民健康及社區產生負面影響。<ref>東方日報：太古城設平台食肆鬧風波（2009年1月3日）</ref> 政府於2009年共收到三宗有關太古城第五期發展食肆的投訴，其中一宗指一間食肆有蟑螂滋生，另一宗投訴涉及兩間食肆，問題包括油煙排放、員工在梯間丟棄煙蒂及便溺；以及太古城第五期發展的食肆沒有妥善處理垃圾而引致附近地方老鼠及蚊蟲滋生。<ref>東方日報：太古城平台食肆審批受質疑（2009年1月8日）</ref>
*毗鄰太古城的鰂魚涌公園治安情況令人關注，單單2009年已經發生了數次-{扑}-頭劫案和非禮案。2018年間更有[[鰂魚涌公園槍擊案|槍擊案]]發生。另外，公園晚間的童黨問題也引起居民的普遍關注。<ref>生活區報（香港經濟日報） - 港島東：鰂魚涌公園治安有待改善 兩月3宗搶劫傷人案 居民夜歸人心惶惶（2009年3月6日）</ref>
