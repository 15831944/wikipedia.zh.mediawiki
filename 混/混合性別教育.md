
'''混合性別教育'''（mixed-sex education，co-education），也称'''混合性別教學'''、'''男女校'''，是指男女同在一个[[学校|学校]]一起接受[[教育|教育]]的教学安排。在混合教育成为主流之前，早年都是[[男女分校|男女分校]]，绝大多数重要的[[高等教育机构|高等教育机构]]只招收[[男生|男生]]，[[女生|女生]]在[[女子学校|女子学校]]接受教育。

== 中国 ==

中国古代学校，以私人兴办的[[私塾|私塾]]、[[书院|书院]]为主，政府主导下的教育体系则是[[官学|官学]]。学生在[[科举|科举]]中考取名次，入朝为官，是旧式教育最高目标。[[男尊女卑|男尊女卑]]之下，女子不能为官，不从事社会职业，被束缚于家庭之中，仅限成为[[家庭主妇|家庭主妇]]，或从事农业生产。因此，女子的教育机会远少于男子。而在[[男女大防|男女大防]]之下，仅接受[[家庭教育|家庭教育]]。家庭、宗族所办的私塾有些也同时招收女生。[[明朝|明朝]][[李贽|李贽]]、清代的[[袁枚|袁枚]]等人招收女弟子，为当时世俗不容。袁枚为女弟子出版诗集之举，更被批评。[[章学诚|章学诚]]评价“近有无耻妄人，以风流自命，蛊惑士女”，“大江以南，名门大家闺阁，多为所诱。征刻诗稿，标榜声名，无复男女之嫌<ref>{{Cite web |url=http://news.zgkqw.com/zhuanti/system/2012/05/07/010184775.shtml |title=章学诚于袁枚之偏见 |accessdate=2018-06-23 |author=那秋生 |date=2012-05-07 |work= |publisher=中国柯桥网|language=Zh-hans }}</ref>。”

[[鸦片战争|鸦片战争]]后，中国门户开放。[[中国天主教|天主教]]合法化，外国基督教传教团体亦在此时进入中国。为了更好的的传教，教会创办学校，自1844年起，陆续开设多所[[女校|女校]]。这是中国女性走出家庭，接受现代教育的肇始<ref name="a1"/>。1905年，广州的教会大学——[[岭南大学_(广州)|岭南大学]]允许教职工的女儿和少数女子插入该校。至1918年，公开招收女生，男女同班授课。是中国最早实行男女同校的高等学校<ref name="a2">{{Cite web |url=https://www.sohu.com/a/119941016_528605 |title=在“男女授受不亲”的中国，男女同校如何实现？ |accessdate=2018-06-23 |author= |date=2016-11-26 |work= |publisher=搜狐网|language=Zh-hans }}</ref>。

1907年，[[學部_(清朝)|学部]]颁布《女子小学堂章程》、《女子师范学堂章程》，标志女子教育获得政府认可<ref name="a1">{{Cite web |url=http://edu.ifeng.com/a/20150309/41002406_0.shtml |title=中国近代女子教育：女校曾是贫苦女孩的收容所 |accessdate=2018-06-22 |author= |date=2015-03-09 |work= |publisher=凤凰网，来源：东方早报 |language=Zh-hans |archive-url=https://web.archive.org/web/20180622220335/http://edu.ifeng.com/a/20150309/41002406_0.shtml |archive-date=2018-06-22 |dead-url=yes }}</ref>，但《女子小学堂章程》明确规定[[男女分校|男女分校]]。1911年4月，《请变更初等教育方法案》规定：“初等小学儿童年龄在10岁以内，准男女同学。”，政府首次允许男女同校<ref name="a2"/>。

[[中华民国|中华民国]]建立之初，1912年1月，[[中華民國臨時政府_(南京)|中华民国临时政府]]颁布《普通教育暂行办法》，明确男女相同的受教育权<ref name="a2"/>。此后开始实行的[[壬子癸丑学制|壬子癸丑学制]]，同样规定女子与男子享有同等受教育权，不再局限于初等教育。1915年，湖南省教育会建议设立女子高等学校，“许女子有入大学之机会”。1917年，[[北京女子师范学校|北京女子师范学校]]高等部成立，是中国内地最早的女子大学。1919年，[[金陵女子大学|金陵女子大学]]畢業了中國第一批獲得學士學位的女大學生，但是没有混合教育的公立高等学校。在此前一年，[[上海大同大学|上海大同大学]]成为最早实行男女同校的私立大学。[[香港大学|香港大学]]则是中国内地之外，最早实行男女同校的公立大学<ref name="a2"/>。

中国内地第一所提供混合教育的公立高等学校是[[南京高等师范学校|南京高等师范学校]]。当时，[[陶行知|陶行知]]——中国混合教育的倡导人，在1919年12月7日南京高等师范学校校务会议上正式提出了《规定女子旁听法案》，并建议次年正式招收女生。这个提议受到许多知名人士的反对的同时，也得到了校长[[郭秉文|郭秉文]]、学监主任[[刘伯明_(哲学家)|劉伯明]]以及[[陆志韦|陆志韦]]、[[杨杏佛|杨杏佛]]等教授的支持。这个法案最终获得通过。1920年，南京高等师范学校招生了中国最早从公立混合学府毕业的女大学生，共8位女生，由此才于1924年走出了中國第一届于混合教育高等学府的女學士。同时，任北京大学校长的[[蔡元培|蔡元培]]在1919年底、1920年1月两次表示，大学没有被禁止招收女生，女生报考北大，成绩合格即可录取。于是，1920年2月，北京大学出现第一个女旁听生，并在1921年开始招收正式的女生。1920年，允许女生旁听之初，[[北洋政府|北洋政府]]教育部曾告诫北京大学：“惟国立学校为社会视听所系，所有女生旁听办法，务须格外慎重，以免发生弊端，致于女学前途转兹障碍，斯为重要。”教育部在暗中查访，发现北京大学并无男女同舍寄宿之事，方才没有禁止男女同校。1922年，北洋政府颁布[[壬戌学制|壬戌学制]]，采取“不分性别的单轨学制”，标志男女同校获得法律认可<ref name="a2"/>。此后，越来越多的中国大学实行了男女同校的教学安排。

在中国大陆，民国时期有很多的女校（包括中小学、大学），从1949年中华人民共和国成立后，几乎所有的中小学和大学都变成了混合型学府。近几年来又有少数新的女校出现。

== 香港 ==
香港的混合學校建立於英國殖民統治時期。[[聖保羅男女中學|聖保羅男女中學]]（St. Paul's Co-educational College）是香港第一所混合教育中學。此校於1915年成立是名為聖保羅女書院（St. Paul's Girls' College）。[[第二次世界大戰|第二次世界大戰]]完結後，與本為男校的[[聖保羅書院|聖保羅書院]]（St. Paul's College）暫時合併。後來當聖保羅書院遷回原址授課後，聖保羅女書院維持混合教学，並更名為聖保羅男女中學。香港其他著名的男女校還包括[[民生書院|民生書院]]、[[香港培正中學|香港培正中學]]、[[伊利沙伯中學|伊利沙伯中學]]和[[荃灣官立中學|荃灣官立中學]]等等。

== 新加坡 ==
新加坡的教育机构大多属混合教育学府，唯有部分的传统名校为[[單性別教育|男校]]或[[女子学校|女校]]（例如新加坡女子中学，南洋女中，[[海星中学_(新加坡)|海星中学]]，[[公教中学|公教中学]]等）。<ref>[http://sis.moe.gov.sg/ School Information Service(SIS)]，新加坡教育部学校信息咨询系统（SIS）</ref>

== 英國 ==
在英國，一些学校实行混合教育，而其他的則是純粹的[[單性別教育|男校]]或是[[單性別教育|女校]]，以前許多的[[單性別教育|單性學校]]在數十年之前曾經接受過相同性别的學生，比如說克利夫頓大學於1987年開始接受女生。

== 美国 ==
美国第一个混合教育高等教育机构是俄亥俄州的[[奥伯林学院|奥伯林学院]]（Oberlin College）。[[女权主义|女权主义]]者在[[美国内战|美国内战]]时代大力推动男女同校，到1872年共有97所美国大学招收女生。一些学校拒绝男女教育完全合为一体，但是愿意在亲近的关联学校提供女生教育，随后类似种族隔离学校“分离但平等”原则的变体形式在美国部分地区发展起来。这种配对的例子包括位于[[马萨诸塞|马萨诸塞]]的[[哈佛大学|哈佛大学]]的[[拉德克利夫学院|拉德克利夫学院]]，位于[[纽约|纽约]]的[[哥伦比亚大学|哥伦比亚大学]]的[[巴纳德学院|巴纳德学院]]，以及位于[[罗德岛州|罗德岛]]的[[布朗大学|布朗大学]]的[[布朗大学彭布罗克学院|彭布罗克学院]]。许多性别分开的女子学校建立起来，著名的有[[七姊妹學院|七姊妹學院]]。其中的部分学校现在实行混合教育，如[[瓦萨学院|瓦萨学院]]，另一些则没有，如[[维斯理学院|维斯理学院]]。

应当提及的是，许多或者多数“普通学校”，邻里、乡、村学校，在19世纪末的成立之初就实行混合教育，部分原因是小的学区没有财力为男孩和女孩提供分开教育的设施。

值得一提的是，在一个世纪多一点的时间之后的美国主流高等教育系统，美国女子获得了多数的学士学位，女生占大学本科招生人数的60%。
