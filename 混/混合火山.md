'''混合火山'''（又稱'''複合火山'''）通常是由火山周圍的地形是否混合著[[熔岩流|熔岩流]]與[[火山碎屑岩|火山碎屑岩]]來判定。.<ref>{{Cite web|url = http://volcano.oregonstate.edu/dieng-volcanic-complex|title = Dieng Volcanic Complex – Volcano World – Oregon State University|website = volcano.oregonstate.edu|publisher = [[Oregon_State_University|Oregon State University]]|access-date = 2016-03-21}}</ref>這些火山可能由於[[火山噴發類型|噴發類型]]的改變，或是特定火山上主要通道口位置而形成。<ref name=":0">{{Cite book|url = https://books.google.com/books?id=OhKWHJqZy8cC&printsec=frontcover#v=onepage&q&f=false|title = Plate Tectonics, Volcanoes, and Earthquakes|publisher = Rosen Publishing |year = 2010|isbn = 9781615301065|editor-last = Rafferty|editor-first = John P.|location = New York|pages = 174}}</ref>[[複式火山|複式火山]]的火山口若充滿熔岩流也可能形成大型[[破火山口|破火山口]]，或是在主火山口的邊緣形成許多小型的[[火山渣錐|火山渣錐]]、[[火山穹丘|火山穹丘]]、[[火山口|火山口]]。<ref name=":0" /><ref>{{Cite book|url = https://books.google.com/books?id=BilBFXIW5c4C&printsec=frontcover#v=onepage&q&f=false|title = Volcanoes|last = Decker|first = Robert|last2 = Decker|first2 = Barbara|publisher = W. H. Freeman and Company |year = 2006|isbn = 9780716789291|edition = 4|location = New York|pages = 170|author-link = }}</ref>
