'''混合泳'''，是四种不同[[游泳|游泳]]姿势的组合，比赛中可以由一名运动员完成（即个人混合泳）或由四名运动员完成（即混合泳接力）。

== 个人混合泳 ==
个人混合泳是指一名游泳运动员在一次比赛中用四种不同姿势游完相同的距离。

=== 泳姿次序 ===
个人混合泳包括四种姿势。一般每种泳姿游总距离的1/4，个人混合泳以下面的顺序：
# [[蝶泳|蝶泳]]
# [[仰泳|仰泳]]
# [[蛙泳|蛙泳]]
# [[自由泳|自由泳]]，只能使用蝶泳、仰泳和蛙泳之外的任何泳姿。多数游泳运动员使用[[捷泳|捷泳]]。

== 混合泳接力 ==

混合泳接力由四名不同的运动员完成接力，每一名运动员游一种姿势。

=== 泳姿次序 ===
仰泳是第一个入水姿势，否则将造成冲撞。其他泳姿以快慢的顺序（最慢的蛙泳在前，最快的自由泳在后）。混合泳接力以下面的顺序：
# [[仰泳|仰泳]]
# [[蛙泳|蛙泳]]
# [[蝶泳|蝶泳]]
# [[自由泳|自由泳]]，限制使用蝶泳、仰泳和蛙泳之外的任何泳姿。多数游泳运动员使用[[捷泳|捷泳]]。
