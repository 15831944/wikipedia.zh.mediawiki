'''混合式火箭''' ，是一種[[火箭推進器|火箭推進器]]，其推進劑包含[[固態|固態]]的[[燃料|燃料]]以及[[液態|液態]]的[[氧化劑|氧化劑]]。和固態火箭、液態火箭同樣屬於化學火箭，具備能夠控制推力大小、成本相對低等特性，同時具備更高的安全性。許多公司及團隊都相繼投入相關研究，不過其比衝值仍待突破。

==歷史==
===早期發展===
1953年，太平洋火箭協會開發出長約180公分、直徑約10公分的混和式火箭，命名為XDF-23。該協會曾嘗試以木材、棉花、石蠟作為燃料，不過最後的設計採用[[液態氧|液態氧]]作為氧化劑，並使用一種橡膠作為燃料。<ref>[https://books.google.com/books?id=Nd8DAAAAMBAJ&pg=PA81&dq=1954+Popular+Mechanics+January&hl=en&sa=X&ei=Q3YzT6TaKu2o0AHgjvW_Ag&ved=0CDMQ6AEwATgK#v=onepage&q&f=true April 1954 Popular Mechanics magazine pp. 81-85.] 2016-10-23</ref>
===近年發展===
2004年，[[維珍銀河|維珍銀河]]公司開發的[[太空船一號|太空船一號]]使用兩階段推進系統。第一階段先以飛行母船載上高空後，第二階段採用混合式火箭引擎推進，完成最高飛行高度112公里的[[次軌道太空飛行|次軌道太空飛行]]。
2016年臺灣晉陞公司成功發展混和式火箭

==基本組成==
混合式火箭主要由氧化劑供應系統、燃燒室及噴嘴組成。
; 氧化劑供應系統
: 液態的氧化劑經由閥件及管路注入至燃燒室中，提供氧化反應所需的氧化劑。與[[液態火箭|液態火箭]]相似的是，混合式火箭也能夠藉由調整氧化劑的流量來控制推力。特別的是，即使關閉推進器，在燃燒室內部的燃料冷卻之前，只要重新注入氧化劑就可以再次啟動推進器，不必重新啟動點火裝置。<ref>[https://www.youtube.com/watch?v=YjPFu2PEVlo&list=PLg0F1u3vZaexksRQxGdLtInNQSZJJDV18&index=1 20160929 ARRC Hybrid Rocket Engine Thrust Throttling Test (400N)] 2016-10-24</ref>
; 燃燒室
: 由於燃料是固態的，因此只有燃料的表面能發生燃燒反應。與[[固態火箭|固態火箭]]相似的是，燃燒室內的燃料可以製造成不同形狀以獲得不同燃燒表面積。
; 噴嘴
: 燃燒室內部產生的大量高溫氣體經過噴嘴，壓縮再膨脹的過程使熱能轉換為動能，推進器因而獲得推力。相較於液態火箭，混合式火箭的氧化劑可以不必經過預熱升溫，因此噴嘴上不用纏繞複雜的管路。然而這也象徵噴嘴沒有低溫氧化劑的冷卻，因此較要求耐熱及隔熱。

==推進劑==
混合式火箭推進劑，透過液態的氧化劑和固態的燃料在燃燒室內發生[[氧化反應|氧化反應]]產生熱量及高溫氣體，然後通過噴嘴噴出並提供[[反作用力|反作用力]]。
; 氧化劑
: 常見的氧化劑包括[[液態氧|液態氧]]、[[笑氣|笑氣]]。
; 燃料
: 常見的燃料包括橡膠、[[石蠟|石蠟]]、[[高分子化合物|高分子化合物]]，也有團隊曾經嘗試使用木材、棉花等物質作為燃料。
:*臺灣[[前瞻火箭研究中心|前瞻火箭研究中心]]使用[[壓克力|壓克力]]作為燃料及燃燒室本體，並使用液態氧作為氧化劑展示混合式火箭的運作，透明的燃燒室可以看見火焰在其中燃燒的情形。<ref>[https://www.youtube.com/watch?v=U--Ht3oL-ys&list=PLg0F1u3vZaexksRQxGdLtInNQSZJJDV18&index=6 Demonstration of hybrid rocket motor with acrylic and gaseous oxygen] 2016-10-24</ref>
:*日本的一間糖果公司使用棉花糖作為燃料，並成功將火箭推進248公尺高。<ref>[https://www.youtube.com/watch?v=80gwq7vNtUw CANDY ROCKET PROJECT キャンディでハイブリッドロケットを飛ばせ！] 2016-10-24</ref>
