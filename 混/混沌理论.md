'''混沌理论'''（'''Chaos theory'''）是关于[[非线性系统|非线性系统]]在一定参数条件下展现[[分岔理論|分岔]]（bifurcation）、周期运动与非周期运动相互纠缠，以至于通向某种非周期有序运动的理论。在[[耗散系统|耗散系统]]和[[保守系统|保守系统]]中，混沌运动有不同表现，前者有[[吸引子|吸引子]]，后者无（也称含[[混吸引子|混吸引子]]）。

从20世纪80年代中期到20世纪末，混沌理论迅速吸引了数学、物理、工程、生态学、经济学、气象学、情报学等诸多领域学者有关注，引发了全球混沌热。混沌，也写作浑沌（比如《[[庄子|庄子]]》）。自然科学中讲的混沌运动指确定性系统中展示的一种類似[[随机|随机]]的行为或性态。确定性（deterministic）是指方程不含随机项的系统，也称[[动力系统|动力系统]]（dynamical system）。典型的模型有[[單峰映象|單峰映象]]（logistic map）迭代系统，[[愛德華·諾頓·勞侖次|洛伦兹]]微分方程系统，[[若斯叻吸引子|若斯叻吸引子]]，[[杜芬方程|杜芬方程]]，[[蔡氏电路|蔡氏电路]]，[[陳氏吸引子|陳氏吸引子]]等。为浑沌理论做出重要贡献的学者有[[庞加莱|庞加莱]]、[[洛伦兹|洛伦兹]]、{{link-ja|上田睆亮|上田睆亮}}（Y. Ueda）、[[米切爾·費根鮑姆|费根堡姆]]、[[詹姆士·约克|约克]]、[[李天岩|李天岩]]、[[斯蒂芬·斯梅尔|斯美尔]]、[[本華·曼德博|芒德勃罗]]和[[郝柏林|郝柏林]]等。混沌理论向前可追溯到19世纪[[庞加莱|庞加莱]]等人对天体力学的研究，他提出了{{le|同宿轨道|Homoclinic orbit}}、{{le|异宿轨道|Heteroclinic orbit}}的概念，他也被称为浑沌学之父。

混沌行为可以在许多自然系统中被观测到，例如天气和气候。<ref>{{cite book|last=Ivancevic|first=Vladimir G.|title=Complex nonlinearity: chaos, phase transitions, topology change, and path integrals|year=2008|publisher=Springer|isbn=978-3-540-79356-4|author2=Tijana T. Ivancevic}}</ref>对于这个行为的研究，可以通过分析混沌[[数学模型|数学模型]]，或者通过诸如{{link-en|递归图|Recurrence plot}}和[[庞加莱映射|庞加莱映射]]等分析技术。

== 定义 ==
混沌理论是一种兼具质性思考与量化分析的方法，用以探讨动态系统中无法用单一的数据关系，而必须用整体，连续的数据关系才能加以解释及预测之行为。

{{Cquote|一切事物的原始状态，都是一堆看似毫不关联的碎片，但是这种混沌状态结束后，这些无机的碎片会有机地汇集成一个整体。}}

混沌一词原指发现宇宙混乱状态的描述，古希腊哲学家对于宇宙之源起即持混沌论，主张宇宙是由混沌之初逐渐形成现今有条不紊的世界。在井然有序的宇宙中，科学家经过长期的探讨，逐一发现众多自然界中的规律，如大家熟知的[[万有引力|万有引力]]、[[杠杆原理|杠杆原理]]、[[相对论|相对论]]等。这些自然规律都能用单一的数学公式加以描述，并可以依据此公式准确预测物体的行径。

近半世纪以来，科学家发现许多自然现象即使可以化为单纯的数学公式，但是其行径却无法加以预测。如气象学家[[愛德華·諾頓·勞侖次|愛德華·諾頓·勞侖次]]（[[愛德華·諾頓·勞侖次|Edward Lorenz]]）发现简单的热[[对流|对流]]现象居然能引起令人无法想象的气象变化，产生所谓的“[[蝴蝶效应|蝴蝶效应]]”。60年代，美国数学家[[史蒂芬·斯梅爾|史蒂芬·斯梅爾]]（[[斯蒂芬·斯梅爾|Stephen Smale]]）发现某些物体的行径经过某种规则性变化之后，随后的发展并无一定的轨迹可循，呈现失序的混沌状态。

== 背景 ==
1963年[[美国|美国]]气象学家[[愛德華·諾頓·勞侖次|愛德華·勞侖次]]提出混沌理论（Chaos），非线性系统具有的多样性和多尺度性。混沌理论解释了[[決定系统|決定系统]]可能产生[[隨機|隨機]]结果。理论的最大的贡献是用简单的模型获得明确的非周期结果。在[[气象|气象]]、[[航空|航空]]及[[航天|航天]]等领域的研究裡有重大的作用。

== 應用 ==
混沌理論在許多[[科學|科學]]學科中得到廣泛應用，包括：[[數學|數學]]、[[生物學|生物學]]、[[信息技術|信息技術]]、[[經濟學|經濟學]]、[[工程學|工程學]]、[[金融學|金融學]]、[[哲學|哲學]]、[[物理學|物理學]]、[[政治學|政治學]]、[[人口學|人口學]]、[[心理學|心理學]]和[[機器人學|機器人學]]。

多種系統的渾沌狀態在實驗室中得到觀察，包括[[電路|電路]]、[[激光|激光]]、[[流體|流體]]的動態，以及機械和電磁裝置。在自然中進行的有對[[天氣|天氣]]、[[衛星|衛星]]運動、[[天體磁場|天體磁場]]、[[生態學|生態學]]中的種群增長、[[神經元|神經元]]中的動作電位和[[分子振動|分子振動]]的觀察。

渾沌理論最成功的應用之一在于[[生態學|生態學]]中的[[雷克動態綜合模型|雷克動態綜合模型]]，在其中顯示了受密度制約之下的種群增長如何引致混沌狀態。

===混沌控制===
'''混沌控制'''由''狄透''（William Ditto）、''賈芬卡''（Alan Garfinkel）、''約克''（Jim Yorke），將此想法化為實用技術，用微小的變化開始，造成希望所想的巨大改變。

== 混沌動力學 ==
混沌系統有三種性質：
# 受初始狀態影響的敏感性，初始条件非常微小的变动也可以导致最终状态的巨大差别。
# 具有拓撲混合性；不严格地来说，就是系统会将初始空间的拓扑性质彻底打乱，使得任何初始状态变换到其他任何位置。
# 周期軌道稠密，即在任何初始值附近都可以找到具有周期轨道的值。
