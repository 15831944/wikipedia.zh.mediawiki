'''寿春第三叛'''，又称'''诸葛诞之乱'''，是250年代[[寿春三叛|寿春三叛]]的第三叛同時也是最后一叛。和前两叛一样，此叛是[[三国|三国]]时期对[[曹魏|曹魏]]司马氏的讨伐。

== 背景 ==
在[[高平陵之变|高平陵之变]]和寿春前两叛后，司马氏摄曹魏国政，几乎完全掌控了曹魏。[[寿春第二叛|寿春第二叛]]后，[[司马师|司马师]]卒，将权柄移交弟弟司马昭。当时魏帝为[[曹髦|曹髦]]。因[[文钦|文钦]]逃往东吴，寿春事务被魏将[[诸葛诞|诸葛诞]]接管。诸葛诞目睹[[王凌|王凌]]、[[毌丘俭|毌丘俭]]崛起和败亡，在第二叛中也起到了巨大的镇压叛乱作用。但他和被司马氏所灭的[[夏侯玄|夏侯玄]]、[[邓飏|邓飏]]都是密友，又目睹王凌和毌丘俭被夷三族，心里不安，就在[[淮南|淮南]]地区逐漸累積實力。

== 导火索 ==
[[甘露|甘露]]元年冬，东吴攻徐堨，诸葛诞的兵马足以应付，却还向朝廷要十万军队，还想筑城保守淮南。司马昭得知他怀有疑心，又因为他是旧臣，派长史[[贾充|贾充]]去征他入朝为[[司空|司空]]。诸葛诞得到诏令，感到害怕，杀[[扬州|扬州]]刺史[[乐綝|乐綝]]，夺取当地的四五万驻军，宣布反叛司马昭。主簿宣隆、部曲督秦絜因反对诸葛诞举事，被诸葛诞所杀。后来二人之子被用为骑都尉，加以赠赐。平寇将军临渭亭侯[[庞会|庞会]]、骑督偏将军[[路蕃|路蕃]]率左右斩门而出，庞会因而进爵乡侯，路蕃封亭侯。诸葛诞派长史[[吴纲|吴纲]]携子[[诸葛靓|诸葛靓]]入东吴为人质以求吴援。因此东吴派出了寿春第二叛时更多的军队，大量支援诸葛诞。司马昭也立即进讨诸葛诞。曹髦派散骑常侍[[裴秀|裴秀]]、给事黄门侍郎钟会与司马昭一同前去。司马昭挟持曹髦和[[明元郭皇后|郭太后]]从征。

== 叛乱 ==
=== 东吴的援助 ===
司马昭的第一步行动是在寿春附近的要塞[[丘头|丘头]]驻扎了26万大军。他命王基、陈骞率部包围寿春以期使叛军在叛乱初期就大量伤亡。司马昭知道东吴正在派大军赶往寿春以南，也从本部分出大量士兵给石苞、州泰、胡质统领，试图阻止吴军来援。之前叛魏的文钦及吴将[[全怿|全怿]]、[[唐咨|唐咨]]预见到司马昭的行动，抢先在魏军围城前进入寿春。魏军不能制止，将军李广临敌不进，[[泰山|泰山]]太守[[常时|常时]]称疾不出，都被司马昭处斩以正军法。文钦率部数次试图反攻破围，皆败。朱异率领的吴军后队从寿春西南的驻扎地[[安丰|安丰]]出发，很快被魏军击退，未能重创魏军。朱异逃回见孙綝。孙綝将军队分给丁奉、[[黎斐|黎斐]]和朱异统领，让他们各自率军沿[[巢湖|巢湖]]方向行进，意图冲击魏军，阻止围城。魏军迅速移动，在吴军抵达寿春前予以截击，成功击溃孙綝大军。诸葛诞军和吴军士气大受打击。因屡次战斗不是无功就是战败，孙綝处决朱异。讽刺的是，吴军溃败的同时，粮饷也被[[胡烈|胡烈]]奇袭所毁。很多军队投降司马昭或开小差。孙綝绝望，认定诸葛诞无望抗魏，下令离开寿春，返回[[建业|建业]]。

=== 倒戈 ===
诸葛诞军被围困在寿春，虽然对魏军造成了一些伤亡，使得参战者之一、司马昭的弟弟东中郎将广阳乡侯[[司马亮|司马亮]]遭到免官，但士气低落，伤亡惨重。有人建议立刻进攻寿春城，司马昭认为「寿春城池坚固，敌人守城士兵众多，强攻可能会损失重大，如果敌人有外援，更会表里受敌，这不是一个稳妥的办法。现在诸葛诞、文钦、唐咨都在寿春城内，正是天赐良机，我要用万全之策，轻松击败敌人」。<ref>《三国志》卷28：初围寿春，议者多欲急攻之，大将军以为：“城固而众多，攻之必力屈，若有外寇，表里受敌，此危道也。今三叛相聚於孤城之中，天其或者将使同就戮，吾当以全策縻之，可坐而制也。”</ref>诸葛诞的副手[[蒋班|蒋班]]、[[焦彝|焦彝]]建议诸葛诞集中从一侧突围，扭转战局。所部吴军和主力分离留在寿春战斗的文钦强烈反对，称其余吴军很快就会和诸葛诞合兵，但他并没意识到孙綝已率军返回都城建业。诸葛诞嘲笑蒋班、焦彝的建议，同意文钦，更称想要杀了蒋、焦。两人害怕，率大量诸葛诞军倒戈，诸葛诞餘部残存的希望几乎破灭。在钟会建议下，司马昭伪造全怿侄子[[全祎|全祎]]、[[全仪|全仪]]写给全怿、[[全端|全端]]的劝降信，以期制造更多倒戈。全怿得信，信以为真，立即投降了司马昭。诸葛诞军遭奇袭，伤亡更加惨重。

=== 镇压 ===
258年正月，诸葛诞、唐咨、文钦、文钦子文鸯率餘部决定试图破围。他们很快再次战败并蒙受重大伤亡。此时，诸葛诞军已经很少，城内粮食也耗尽，投降司马昭的军队更多了。

文钦明智地建议放弃部下的北方士兵，只留残余吴军在城内，以保障粮食。诸葛诞没有听从，斥文钦为懦夫、试图骗自己帮他脱困。随后他杀了文钦。文钦子文鸯、[[文虎_(三國)|文虎]]得知父亲被杀，很快逃离寿春投降司马昭，司马昭也不计其父先前所为，封二人为将军。文钦二子倒戈使叛军士气完全瓦解。

二月，魏军攻入诸葛诞的寿春总部。诸葛诞绝望逃离寿春，逃命途中被胡奋部所杀。吴将[[于诠|于诠]]在和魏军交战时阵亡，吴将唐咨、[[王祚|王祚]]和诸葛诞为数不多的餘部一同投降司马昭。

孙綝原本派弟弟[[孙恩_(东吴)|孙恩]]前来救援，取代朱异的角色，但因为诸葛诞败亡，孙恩并未成行。

诸葛诞麾下数百人因不投降被斩，拱手为列，魏军每斩其中一人前，都说降，但这些人至死没有一个投降，都说：“为诸葛公死，不恨。”诸葛诞得人心如此，被时人比作[[田横|田横]]。

== 后果 ==
平叛后，司马昭被曹魏视为英雄，重新取得了曹魏要人的信任和友谊。虽然曹髦不久试图谋杀司马昭，却反而在攻打[[洛阳|洛阳]]司马昭府邸时被杀。

更重要的是，寿春系列叛乱平息，使得其余魏军集中力量灭蜀。

== 时下引用 ==
此战和淮南前两叛一样，在[[光荣|光荣]]的《[[真·三國無雙6|真·三國無雙6]]》晋传中表现为可玩阶段。玩家操控司马昭。此战也被分为两部分。在[[真·三國無雙7|真·三國無雙7]]则作为史实路线关卡，除司马昭可用之外，王元姬、钟会和贾充也可使用，而当中诸葛诞杀文钦的原因则是文钦对诸葛诞接受不明援军而表示异议，因而遭诸葛诞斩杀。
