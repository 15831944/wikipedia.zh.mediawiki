'''朴埇佑'''（{{lang-ko|'''박용우'''}}，{{bd|1971年|3月16日|catIdx=H韓}}），[[韓國|韓國]][[男演員|男演員]]，名字常被音譯為'''朴勇宇'''。

== 個人生活 ==
2009年公開了與女友[[趙安_(韓國藝人)|趙安]]初次相識時對她一見鍾情的往事，但最終因兩人離多聚少，很自然的感情轉淡，在相愛兩年後分手。<ref>{{Cite web |url= http://www.koreastardaily.com/tc/news/1394 |title= 昔日情侶終成陌路 趙安朴勇宇分手|date=2010-02-23 |publisher=KoreaStarDaily}}</ref>

2012年出席Channel A《Shocking》節目中，自曝曾因自閉症而苦惱，不過現在已經完全克服。<ref>{{Cite web |url= http://ent.sina.com.cn/v/j/2012-01-04/17093524737.shtml |title= 韓國藝人樸勇宇自曝病史曾因自閉症而苦惱|date=2012-01-04 |publisher=sina/bntnews}}</ref>

== 經歷 ==
MBC演員訓練班第24期學員（1994年）。1993年在連續劇《[[吳博士家的人們|吳博士家的人們]]》中從兼職現場工作人員（大學時期打工）成為演員而出道。

朴埇佑出道時在熱門劇集《[[公寓_(電視劇)|公寓]]》中扮演的追求[[崔真實|崔真實]]的富家公子吸引了人們的目光，通過《[[陷阱_(韓影)|陷阱]]》、《[[鋤暴特警3|鋤暴特警3]]》以及《[[生死諜變|生死諜變]]》中的配角登上大螢幕。2005年，和[[車勝元|車勝元]]合演的大型古裝驚悚恐怖片《[[血之淚|血之淚]]》讓觀眾見識到了朴埇佑的個性演技。2006年，他在《[[甜蜜的恐怖戀人|甜蜜的恐怖戀人]]》中飾演了一位沒有成功戀愛過一次的大學英國文學講師，將男人的雙重性刻畫得入骨入髓。

2007年，同[[南宮民|南宮民]]合演了犯罪驚悚片《[[美麗的星期天|美麗的星期天]]》及與[[韓彩英|韓彩英]]、[[嚴正花|嚴正花]]和[[李東健|李東健]]合作回響不錯的愛情片《[[現在是和相愛的人一起生活嗎？|現在是和相愛的人一起生活嗎？]]》，成為真正的主演級大明星。之後他又陸續演出了《孩子們》、《[[爸爸_(韓影)|爸爸]]》、《[[濟眾院|濟眾院]]》、《[[我的愛蝴蝶夫人|我的愛蝴蝶夫人]]》等作品。2013年，在新作《[[華頤：吞噬怪物的孩子|華頤：吞噬怪物的孩子]]》中首次挑戰大反派演技。
