'''朴新星'''（{{kor|k='''박새별'''|rm='''Park Sae-byul'''}}，{{bd|1985年|10月24日}}）是[[韩国|韩国]]的一位[[创作歌手|创作歌手]]。

== 学历 ==
* [[延世大學|延世大學]]心理学学士<ref>{{Cite news|url=http://tenasia.hankyung.com/archives/4119|title=박새별│My name is…|language=ko}}</ref>

== 活动 ==
* 2006年在[[延世大學|延世大学]][[心理学|心理学专业]]就读时，参与Lucid fall的公演，和[[Epik_High|Epik High]]、Toy等歌手开始表演活动。
* 2008年10月15日，发布了第一张迷你专辑《日记》。
* 2010年3月，有她个人作词作曲、编曲、演奏、制作的正规一辑《启明星》发售。<ref>{{Cite news|url=http://www.nocutnews.co.kr/show.asp?idx=1441368|title=유희열 수제자 박새별, 첫 정규앨범 ''새벽별'' 발매|language=ko}}</ref>
* 2009年参加了[[光州文化广播|光州MBC]]的节目《文化演唱会 NANJANG》。
* 2011年3月6日-2012年2月28日，担任EBS（Educational Broadcasting System）FM节目《美妙的童谣》的电台主播。
* 2010年，于延世大学心理学系毕业。
* 2012年，进入[[韩国科学技术院|韩国科学技术院（Kaist）]]深造。
* 2013年，发布了正规二辑《高跟鞋》。
