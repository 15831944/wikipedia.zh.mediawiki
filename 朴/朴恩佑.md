'''朴恩佑'''（{{lang-ko|'''박은우'''}}，{{bd|1990年|3月30日||}}），[[韩国|韩国]][[创作歌手|创作歌手]]。

== 生平 ==
朴恩佑于2007年以“17岁大田少女”的名义出道，艺名“Xeno”。2008年凭借现场歌曲“Who are you? sexy My Boy”中男性化的舞蹈在韩国网络意外走红。不少韩国网友在网络上将她的照片恶搞<ref>{{cite web |url=https://uncyclopedia.kr/wiki/%EC%A0%9C%EB%85%B8 |title= 제노사이드 커터 |publisher=uncyclopedia.kr |language=ko |deadurl=no}}</ref>。但是在仅仅十几天的时间后，她再次登台献唱，清纯可爱的淑女装彻底颠覆了之前的形象。这当时在韩国网络上被称为“Xeno的逆袭”<ref>{{cite web |url=http://weekly.khan.co.kr/khnm.html?mode=view&code=115&artid=18064&pt=nv |title= [언더그라운드 넷]제노의 역습과 디제이 쿠 |publisher=주간경향 |language=ko |deadurl=no}}</ref>。成名后她专注作词和作曲，在2012年为[[韩剧|韩剧]][[绅士的品格|绅士的品格]]录制了[[原声带|原声带]]“Everyday”再次被大众熟知<ref group="注"><small>在绅士的品格OST中，其中文名被错译为朴恩宇</small></ref><ref>{{cite web |url=http://www.newdaily.co.kr/site/data/html/2012/10/19/2012101900109.html|title= '팝핀현준' 구사일생 스토리..."춤이 나를 구했다" |publisher=newdaily.co.kr |language=ko |deadurl=no}}</ref><ref>{{cite web |url=http://news.kstyle.com/article.ksn?articleNo=1945444|title= 「紳士の品格」ドラマの人気と共にOSTの人気もUP |publisher=newdaily.co.kr |language=ja |deadurl=no}}</ref>。
