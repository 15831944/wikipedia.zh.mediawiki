[2014-01-10]{{zh}}</ref>（{{lang-ko|'''박진영'''}}，{{lang-en|'''Park Jin Young'''}}，{{bd|1994年|9月22日|catIdx=Jr.}}），藝名：'''珍榮'''（{{lang-ko|'''진영'''}}，{{lang-en|'''Jin Young'''}}），[[韓國|韓國]][[慶尚南道|慶尚南道]][[昌原市|昌原市]][[鎮海區_(昌原市)|鎮海區]]出身<ref>2014年7月19日節目《King Of Ratings》GOT7專訪. KBS. [2014-07-19]</ref>，[[韓國|韓國]][[男歌手|男歌手]]、[[演員|演員]]。2012年1月出演[[韓國放送公社|KBS]]電視劇《[[Dream_High_2|Dream High 2]]》，以演員身份出道；2012年以男子團體[[JJ_Project|JJ Project]]成員出道，擔任rapper；2014年1月16日以[[嘻哈|嘻哈]]男子團體[[GOT7|GOT7]]成員出道，擔任領唱。

其韓文名字與所屬[[JYP娛樂|JYP娛樂]]公司創立人、製作人[[朴軫永|朴軫永]]相同，故在出道初期使用藝名'''Jr.'''（讀作“Junior”），取Park Jin Young '''Jr.'''（小JYP）之意，後來為了方便粉絲和大眾認讀，曾將藝名改寫為'''Junior''' <ref>[http://got7.jype.com/notice.asp?idx=61&page=%20&vnum=6 Jr.改名為「Junior」之官方聲明] GOT7官方網站. [2015-05-20]{{ko}}{{en}}</ref>，2016年8月16日透過個人推特表示將以本名“'''珍榮'''”（Jinyoung）開始活動。<ref>[https://twitter.com/jrjyp/status/765512335783972865 Junior改名為「珍榮」之聲明] 朴珍榮個人推特. [2016-08-16]{{ko}}{{en}}</ref>

== 生涯 ==
=== 出道前 ===
家中成員有父母和兩個姊姊<ref>[http://thestar.chosun.com/site/data/html_dir/2014/02/25/2014022502729.html 【더★프로필】 갓세븐(GOT7) JB-마크-잭슨-JR, 형라인 4인방의 수집품은?] The Star. [2014-02-27]{{ko}}</ref>，2009年參加[[JYP娛樂|JYP娛樂]]第五期公開選秀，與[[林在範|JB]]組成團體表演並雙雙成為冠軍。經歷約4年的練習生時期，二人於2012年1月出演[[KBS|KBS]]《[[Dream_High_2|Dream High 2]]》，成為歌手前率先以演員身份出道。

=== JJ Project活動 ===
2012年5月24日與[[林在範|JB]]以雙人團體[[JJ_Project|JJ Project]]身分出道，發行單曲《[[Bounce_(JJ_Project歌曲)|Bounce]]》。2017年7月31日，JJ Project時隔5年多以迷你專輯《[[Verse_2|Verse 2]]》再度回歸，除了舞台表演以外，還首度舉辦攝影展。

=== GOT7活動 ===
2014年1月7日，JYP娛樂公開了7人新男團[[GOT7|GOT7]]的最後三名成員，珍榮名列其中，成為GOT7成員之一。1月16日正式出道，隊內擔任副唱、領舞及形象。2015年3月19日起，與隊友[[BamBam|BamBam]]、[[SHINee|SHINee]]的[[Key_(歌手)|Key]]和[[CNBLUE|CNBLUE]]的[[李正信|李正信]]一起擔任 [[Mnet|Mnet]]《[[M!_Countdown|M! Countdown]]》的固定MC<ref>{{cite news| url = http://tenasia.hankyung.com/archives/472364|title = 키·이정신·주니어·뱀뱀, ‘엠카’ MC로 첫 인사.. ‘눈호강 라인업’| publisher = 텐아시아 10asia| date = 2015-03-19 | language = ko}}</ref>，2016年3月3日下車<ref>{{cite news| url = http://www.xportsnews.com/jenter/?ac=article_view&entry_id=699201&_REFERER=https%3A%2F%2Fwww.google.com.tw%2F|title = 
'엠카' 주니어·뱀뱀 MC 하차…"갓세븐 새 앨범 나온다"| publisher = 엑스포츠뉴스 xportsnews| date = 2016-03-03 | language = ko}}</ref>。2017年2月5日起，與[[BLACKPINK|BLACKPINK]]的[[金智秀_(歌手)|Jisoo]]及[[NCT|NCT]]的[[道英|道英]]一同擔任[[SBS|SBS]]《[[人氣歌謠|人氣歌謠]]》的固定MC<ref>{{cite news| url = http://entertain.naver.com/read?oid=213&aid=0000943154|title = ‘인가’ PD “갓세븐 진영·블랙핑크 지수·NCT 도영, 새 MC 확정”[공식]| publisher = TV리포트| date = 2017-02-05 | language = ko}}</ref>，2018年2月4日下車。

=== 演員生涯 ===
2012年1月，珍榮參與演出[[KBS|KBS]]月火迷你連續劇《[[Dream_High_2|Dream High 2]]》，在以歌手身份出道之前，先以演員身份出道。2013年4月，演出[[MBC|MBC]]水木迷你連續劇《[[當男人戀愛時|當男人戀愛時]]》。2015年5月29日在[[JTBC|JTBC]]金土迷你連續劇《[[親愛的恩東啊|親愛的恩東啊]]》中扮演男主角的10代時期，精湛的演技受到韓國觀眾的讚賞<ref>{{cite news| url = http://entertain.naver.com/read?oid=311&aid=0000491363|title = '은동아' 갓세븐 주니어 "주진모 아역, 부끄럽기 싫었다" (인터뷰)| publisher = 엑스포츠뉴스| date = 2015-07-05 | language = ko}}</ref><ref>{{cite news| url = http://www.soompi.com/2015/05/31/got7s-jr-impresses-with-acting-skills-in-my-love-eun-dong/|title = GOT7′s Jr. Impresses with Acting Skills in “My Love Eun Dong”| publisher = soompi| date = 2015-05-31 | language = en}}</ref>，韓國歌手[[Star_(韓國歌手)|星]]亦特地透過SNS稱讚珍榮，甚至表達希望兒子能長大成為像他一樣的人<ref>{{cite news| url = http://www.koreaboo.com/photo/byul-reveals-hopes-son-dream-will-grow-like-got7s-jr/|title = Byul reveals that she hopes her son Dream will grow up to be like GOT7’s Junior| publisher = koreaboo| date = 2015-06-18 | language = en}}</ref>。

2015年12月21日，媒體報導珍榮將主演以校園暴力為主題的電影《[[雪花_(電影)|雪花]]》（台灣翻譯片名為《迷途羔羊》）。拍攝工作於2016年2月3日結束，在完成編輯與後期製作後，4月底於第17屆[[全州國際電影節|全州國際電影節]]（Jeonju International Film Festival，簡稱JIFF）「全州影院計劃2016」首次公開<ref>{{cite news| url = https://www.soompi.com/2016/04/26/posters-for-got7s-juniors-first-movie-reveal-warmth-amidst-the-snow/|title = Posters for GOT7’s Junior’s First Movie Reveal Warmth Amidst the Snow| publisher = soompi| date = 2016-04-26 | language = en}}</ref>。2016年4月28日，出席全州國際電影節開幕紅毯儀式，進行電影宣傳活動<ref>{{cite news| url = http://kpopherald.koreaherald.com/view.php?ud=201604271528326858434_2|title = GOT7 Junior invited to JIFF| publisher = K-POP Herald| date = 2016-04-27 | language = en}}</ref>。

2016年11月，演出[[SBS|SBS]]水木迷你連續劇《[[藍色海洋的傳說|藍色海洋的傳說]]》，飾演男主角[[李敏鎬|李敏鎬]]的青年時期<ref>{{cite news| url = https://www.soompi.com/2016/09/30/got7s-jinyoung-talks-join-lee-min-ho-jun-ji-hyuns-legend-blue-sea/|title = GOT7’s Jinyoung In Talks To Join Lee Min Ho and Jun Ji Hyun’s “Legend Of The Blue Sea”| publisher = soompi| date = 2016-09-30 | language = en}}</ref>。該劇由《[[來自星星的你|來自星星的你]]》編劇[[朴智恩_(編劇)|朴智恩]]作家執筆，《[[主君的太陽|主君的太陽]]》[[陳赫_(導演)|陳赫]]導演執導，集結李敏鎬、[[全智賢|全智賢]]主演，陣容堅強，珍榮在劇中的戲份雖然不多但表現出色，成功吸引觀眾與媒體的視線<ref>{{cite news| url = http://stoo.asiae.co.kr/news/view.htm?idxno=2017010417302289937|title = [신년기획]박보검·류준열 2016년 발견한 男神★… 정유년엔 뭐할까| publisher = 스포츠투데이| date = 2017-01-06 | language = ko}}</ref>。

2017年3月1日，首部主演電影《[[雪花_(電影)|雪花]]》正式於韓國院線上映<ref>{{cite news| url = http://www.koreastardaily.com/tc/news/91079|title = GOT7珍榮、智友主演電影《雪花》預告片公開　3/1日上映| publisher = 韓星網| date = 2017-02-15 | language = zh}}</ref>。

2017年9月11日，由[[JYP娛樂|JYP Pictures]]製作、珍榮擔綱男主角的[[JTBC|JTBC]]網路劇《[[魔術學校|魔術學校]]》首播，以每集15分鐘、全劇共16集的篇幅，於9月29日終映。2018年2月13日起，《魔術學校》正式於JTBC電視台每周一二深夜時段播出。

2019年3月11日，珍榮首次擔任電視劇男主角的[[tvN月火連續劇|tvN月火連續劇]]《[[會讀心術的那小子|會讀心術的那小子]]》正式開播。該片由《[[花遊記|花遊記]]》金炳秀導演與《[[吸血鬼檢察官|吸血鬼檢察官]]》楊珍雅作家合作打造，描述只要有肢體接觸就能知道對方秘密的少年，和拼命想要隱藏自己內心傷痛的少女，兩人之間的故事，是一齣超能力浪漫驚悚電視劇<ref>{{cite news| url = https://www.koreastardaily.com/tc/news/113239|title =《會讀心術的那小子》公開劇本閱讀現場！朴珍榮、辛睿恩、金多順、金權等演員出席| publisher = 韓星網| date = 2019-01-19 | language = zh}}</ref>。珍榮不僅展現多樣面貌，並隨著劇情發展，以細膩深沉的演技普遍獲得好評<ref>{{cite news| url = https://entertain.naver.com/read?oid=011&aid=0003539768|title ='그녀석' 박진영(GOT7), 빗속 열연 호평에 “많은 응원 감사..끝까지 힘낼 것”| publisher = 서울경제| date = 2019-04-17 | language = ko}}</ref>。
