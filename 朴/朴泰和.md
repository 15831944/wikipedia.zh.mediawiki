'''朴泰和'''（{{kor|名=北|諺=박태화}}，{{bd|1919年|10月15日|2005年|11月}}），曾用名'''朴泰浩'''（{{朝|諺=박태호|link=no}}），法名'''鹤林'''，[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]僧人，出生于[[咸镜北道|咸镜北道]]，曾任[[朝鲜佛教徒联盟|朝鲜佛教徒联盟]]中央委员会委员长、朝鲜[[最高人民会议|最高人民会议]]常务委员会委员。

== 生平 ==
1919年10月15日，朴泰浩出生于[[咸镜北道|咸镜北道]][[吉州郡|吉州郡]]<ref name="bulkyo21">{{cite web |title=조불련의 설계자, 최고의 교학승 |url=http://www.bulkyo21.com/news/articlePrint.html?idxno=37784 |website=불교닷컴 |accessdate=2019-03-18 |date=2017-09-11}}</ref>。1934年至1936年，朴泰浩在[[普贤寺_(朝鲜)|普贤寺]]佛学院进修，1937年在普贤寺出家为僧。1946年，任[[朝鲜佛教徒联盟|朝鲜佛教徒联盟]]咸镜北道委员会部长。1955年至1958年，在[[人民经济大学|人民经济大学]]学习。1959年，任朝佛联咸镜北道委员会委员长，1970年任朝佛联中央委员会副委员长，1979年9月任委员长。1986年11月，当选第8届[[最高人民会议|最高人民会议]]代议员。1989年5月，当选朝鲜宗教人协议会（{{lang|ko|조선종교인협의회}}）副委员长。1990年4月，当选第9届最高人民会议代议员。1991年1月，当选{{tsl|ko|조국통일범민족연합|祖国统一泛民族联合}}北方本部中央委员。1991年7月，当选朝日友好亲善协会（{{lang|ko|북-일 우호친선협회}}）副会长。1993年8月，当选祖国统一泛民族联合北方本部副议长。1996年6月，获得“[[禅师|大禅师]]”头衔。1998年9月，当选第10届最高人民会议代议员、最高人民会议常务委员会委员。1999年9月，改名为朴泰和（{{lang|ko|박태화}}）。2003年9月，当选第11届最高人民会议代议员、最高人民会议常务委员会委员。2004年12月，“6.15南北共同宣言实践连带”（{{lang|ko|6.15공동선언실천}}）北方筹备委员会副委员长。2005年11月去世<ref name="统一部">{{cite web |title=박태화(남성) |url=http://nkinfo.unikorea.go.kr/nkp/theme/viewPeople.do?nkpmno=1026 |website=统一部 |accessdate=2019-03-20}}</ref>。
