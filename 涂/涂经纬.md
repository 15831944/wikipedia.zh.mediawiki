== 简介 ==
1980年，-{涂}-经纬出生于[[贵州省|贵州省]][[贵阳市|贵阳市]]，后考入[[中国传媒大学|中国传媒大学]]。<ref name="zcr " />

2003年，-{涂}-经纬担任[[中国中央电视台电影频道|中国中央电视台电影频道]]《[[中国电影报道|中国电影报道]]》主持人。<ref name="zcr " />

== 个人生活 ==
2006年，-{涂}-经纬和[[撒贝宁|撒贝宁]]一起主持[[长春电影节|长春电影节]]，两人互相留下手机号码，经常发短信交流沟通。<ref name=" xhw" >{{cite web |url= http://news.xinhuanet.com/ent/2010-09/17/c_12579163.htm |title=金童玉女劳燕分飞 -{涂}-经纬与撒贝宁分手内幕曝光 |publisher=新华网 |accessdate=2010-09-17 |work= |language =zh-cn}}</ref><ref name="xwwa " >{{cite web |url=http://www.chinanews.com/yl/2010/09-17/2539185.shtml |title=-{涂}-经纬恋上"金主"享奢华 撒贝宁新女友曝光 |publisher=中国新闻网 |accessdate=2010-09-17 |work= |language =zh-cn}}</ref><ref name="banana " >{{cite web |url=http://ent.163.com/10/0916/10/6GMQAA2K00031H2L.html |title=美女主播经纬“金主”男友曝光 |publisher=网易 |accessdate=2010-09-16 |work= |language =zh-cn}}</ref>

2006年年底，[[中国中央电视台|中国中央电视台]]梦想剧场拍摄方言版《[[上海滩|上海滩]]》，[[撒贝宁|撒贝宁]]扮演男主角，-{涂}-经纬扮演女主角，两人合作之后，擦出[[爱情|爱情]]的火花。<ref name=" xhw" /><ref name="xwwa " /><ref name="banana " />

2008年，-{涂}-经纬和撒贝宁经常传出[[分手|分手]]的消息。<ref name=" xhw" /><ref name="xwwa " /><ref name="banana " />

2009年，-{涂}-经纬和撒贝宁正式分手。<ref name=" xhw" /><ref name="xwwa " /><ref name="banana " />
