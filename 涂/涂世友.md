== 简介 ==
1974年，涂世友出生于[[湖北省|湖北省]][[鄂州市|鄂州市]]。<ref name="wya " /> 
她的[[小学|小学]]和[[初中|初中]]都在鄂州城区的学校里度过。<ref name="wya " /> 
初中毕业后，考取了中等师范学校，[[中专|中专]]毕业后，她被分配到[[鄂州|鄂州]]一所小学教书。<ref name="wya " /> 
两年后，她辞职去了一家企业，当起了宣传员，在第四年中，她报考了[[武汉大学|武汉大学]]新闻系，2001年毕业，并且获得[[硕士|硕士]]学位。<ref name="wya " /> 
涂世友在[[武汉大学|武汉大学]]毕业后，成为[[自由撰稿人|自由撰稿人]]，收入微薄，此时她已经38岁了。<ref name="wya " /> 
2011年12月，涂世友在[[武汉同济医院|武汉同济医院]]做了处女检查证明。随后，她和一个杭州男生建立了“雅品贞操网”。<ref name="wya " /> 
2012年3月20日，涂世友在微博上贴出“征婚告示”：{{quote|“本人38岁，硕士学位，处女。现面向全国征婚：1、作风正派、党员优先；2、有正当职业、无婚史、公务员优先；3、四十岁以下、硕士以上学历；4、婚前不能发生性接触；5、婚后三年考察期不能过性生活、考察期结束后方可行夫妻之事；6、处男优先。1至5条缺一不可，有意者请留言。”<ref>{{cite web |url=http://news.163.com/12/0321/00/7T3468AC00011229.html |title=“贞操女神”微博征婚：党员公务员优先 |accessdate= 2012年|work= |publisher=网易 |language =zh-cn}}</ref> }}
2012年3月21日，涂世友“征婚告示”的策划人被曝光。<ref>{{cite web |url=http://news.163.com/12/0322/05/7T66K9F300011229.html |title=“贞操女神”终止征婚 策划人现身 |accessdate= 2012年|work= |publisher=网易 |language =zh-cn}}</ref><ref>{{cite web |url=http://society.huanqiu.com/roll/2012-03/2544995.html |title=“贞操女神”幕后策划人浮出水面 网友不解 |accessdate= 2012年|work= |publisher=环球时报 |language =zh-cn}}</ref>

== 思想 ==
涂世友认为，当今社会已经严重扭曲了[[青少年|青少年]]的[[性观念|性观念]]，于是她决心拯救迷途中的青少年，遂建立了“雅品贞操网”，她向媒体表示说：“青少年[[性意识|性意识]]的觉醒和情感的萌动，也需要家长和爱心人士施以健康的[[价值观|价值观]]和[[社交|社交]]能力来引导。青少年轻率混乱的性经历造成的悲剧越来越多。”<ref name="wya " /> 

== 评价 ==
[[黄石中心医院|黄石中心医院]]妇科主治医生潘伟：“涂世友的行为也只能代表个人的主观意识，但从[[生理学|生理学]]角度讲，不论年龄大小，婚前守贞现象的存在是合理的，也不会对生理造成危害，社会各界也不必过多的去评判这种行为。”<ref name="wya " /> 

[[武汉大学|武汉大学]]政治公共管理学院教授[[尚重生|尚重生]]：“创办如此网站号召守贞，从[[道德|道德]]、[[法律|法律]]上讲，是行得通的，而且，出发点和宗旨非常好，她在竭力倡导一种[[传统美德|传统美德]]，倡导当代社会人类的[[性安全|性安全]]、[[性健康|性健康]]。”<ref name="wya " />
