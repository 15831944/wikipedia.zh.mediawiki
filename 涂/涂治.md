'''涂治'''（{{bd|1901年|7月1日<ref>{{Cite web|url=http://casad.cas.cn/aca/316/ygysmd-200906-t20090624_1809734.html|title=涂治----中国科学院学部与院士|accessdate=2018-10-12|work=casad.cas.cn}}</ref>|1976年|3月30日<ref>{{Cite web|url=http://www.xjau.edu.cn/2005/1125/c145a33970/pagem.htm|title=忆第一任院长涂治教授|accessdate=2018-10-12|work=www.xjau.edu.cn}}</ref>|catIdx=T}}）<ref group="注">中华人民共和国农业部涂治同志事迹中为1901年8月出生，卒于1976年3月</ref>，字'''策三'''<ref>{{Cite web|url=http://www.moa.gov.cn/ztzl/xzgcllsnsnmfrwbz/snmfrwfc/200909/t20090927_1358550.htm|title=涂治同志事迹|accessdate=2018-10-12|work=www.moa.gov.cn}}</ref>，出生于[[湖北省|湖北省]][[黄陂县|黄陂县]]东乡涂家湾<ref name="a1"/>，中国农学家，一级教授。<ref name=dhw>{{Cite web |url=http://news.dahe.cn/2012/03-29/101202786.html |title=涂治 |accessdate=2017-06-14 |author=刘卫东 |date=2012-03-29 |work=责任编辑：王晓云 |publisher=大河网，河南大学供稿|language=Zh-hans }}</ref><ref name=zky>[http://sourcedb.cas.cn/sourcedb_ad_cas/zw2/ysxx/ygysmd/ 中国科学院学部与院士·院士信息·已故院士名单，中国科学院，于2014-05-05查阅] {{webarchive|url=https://web.archive.org/web/20120427015056/http://sourcedb.cas.cn/sourcedb_ad_cas/zw2/ysxx/ygysmd/ |date=2012-04-27 }}</ref>[[第一届全国政协委员|第一]]至[[第四届全国政协委员|四届全国政协委员]]、[[第四届全国人大代表|第四届全国人大代表]]。

在担任河南大学教授的十年中，他发表学术论文35篇，出版著作2部，开创了现代农业技术的推广应用，较早提出在河南省大面积推广农业机械。1934年任院长后，他专设河南大学现代农业技术推广处，并且在河南大学农学院的培养方案中强调本科生必须掌握一两种现代农业技术。他支持学生的抗日爱国行动，并投身到斗争中。河南大学提供的文章称<ref name=dhw/>，期间，涂治秘密参加了[[中国共产党|中国共产党]]，成为河南大学教授中极少的中国共产党地下党员之一。他常年将自己一半月薪交给中共河南大学党支部。<ref name=zky/>

1940年，涂治被[[中共中央|中共中央]]派到[[新疆农学院|新疆农学院]]担任院长，负责中国共产党交给的新疆上层统一战线工作。1948年，因叛徒告密，他遭政府迫害一年，被关押后受到各种酷刑，始终不屈服。后来在中国共产党的营救下获释，继续任新疆农学院院长，为促进[[新疆和平解放|新疆和平解放]]发挥了很大作用，因此后来获得[[中国人民解放军|中国人民解放军]][[西北野战军|西北野战军]]政治部颁发的“[[毛泽东奖章|毛泽东奖章]]”和“[[人民功臣章|人民功臣章]]<ref name="a2"/>”（有称“西北解放纪念章<ref name=dhw/><ref name=zky/>”）。

1949年8月，[[扎巴依喀勒山空难|扎巴依喀勒山空难]]发生，赴[[北平|北平]]参加[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]的新疆代表团全体人员遇难。消息传到新疆之后，[[三区革命|新疆三区]]最后决定再派出三位代表：[[赛福鼎|赛福鼎]]（[[阿合买提江·哈斯木|阿合买提江·哈斯木]]离开后新疆三区的实际负责人），[[阿里木江·哈肯巴也夫|阿里木江·哈肯巴也夫]]（[[塔城|塔城]]副专员），涂治（迪化[[新疆大学|新疆学院]]副院长）。9月15日，三人顺利抵达北平，赶上了[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]召开<ref>{{cite web|title=邓力群：初到新疆的历程(2)|url=http://www.21ccom.net/articles/rwcq/article_2012051860020_2.html|date=2012-05-18|publisher=[[共识网|共识网]]|work=当代中国史研究，2012年第2期|archiveurl=https://web.archive.org/web/20160305215306/http://www.21ccom.net/articles/rwcq/article_2012051860020_2.html|archivedate=2016-03-05|deadurl=yes}}</ref>。[[新疆省人民政府|新疆省人民政府]]成立后，涂治任[[新疆省人民政府委员会|新疆省人民政府委员会]]委员兼农林厅厅长<ref name="a2">{{Cite web |url=http://news.xjau.edu.cn/plus/view.php?aid=192 |title=忆第一任院长涂治教授 |accessdate=2017-06-14 |author=朱洪柱 |date=2005-11-25 |work= |publisher=新疆农业大学新闻网 |language=Zh-hans }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1950年1月15日<ref>{{Cite web |url=http://dangshi.people.com.cn/n/2013/0220/c85037-20540552-4.html |title=我所知道的饶正锡将军（四）|accessdate=2017-06-14 |author=杨根林 |date=2013-02-20 |work=责任编辑：王新玲、孙琳 |publisher=人民网，来源：《新疆党史》授权中国共产党新闻网发布|language=Zh-hans }}</ref>，中共迪化市委发展涂治、[[禹占林|禹占林]]等人加入中国共产党，成为[[迪化市|迪化市]]发展的第一批中国共产党党员。参与组建<ref name="a2"/>[[新疆农业大学|中国人民解放军新疆八一农学院]]（今新疆农业大学）。在1952年到1970年担任首任院长期间，他撰写并翻译出版了数十万字的农业科技专著及丛书，撰写了数十篇推广现代农业技术的学术论文，为[[新疆生产建设兵团|新疆生产建设兵团]]培养了许多农业科技人才，使[[新疆维吾尔自治区|新疆维吾尔自治区]]的农牧业及农业机械化水平都大幅提高。截至1974年，新疆生产建设兵团已经开发农田1100万亩，建起160多个机械化程度比较高的国营农场，新疆的农业机械化水平在全中国处于领先地位。<ref name=dhw/><ref name=zky/>

1955年，涂治当选[[中国科学院学部委员|中国科学院学部委员]]，兼任[[中国农业科学院|中国农业科学院]]学术委员会委员。1956年，当选[[中国共产党新疆维吾尔自治区委员会|中国共产党新疆维吾尔自治区委员会]]委员。他担任中国人民解放军新疆八一农学院教授、院长，[[中国科学院新疆分院|中国科学院新疆分院]]第一副院长，[[新疆农业科学院|新疆农业科学院]]院长。1976年秋，涂治逝世，享年75岁。<ref name=dhw/><ref name=zky/>
