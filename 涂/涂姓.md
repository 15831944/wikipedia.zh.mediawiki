'''涂姓'''<ref group="读音">（ 汉语拼音：tú 注音：ㄊㄨˊ广东话：tou4 潮州話：tou5(thôu) 西南官話：tou2）</ref>是[[中文姓氏|中文姓氏]]之一，在明朝《[[百家姓|百家姓]]续编》中排第464位。根据中国公安部身份证数据库，2008年涂姓人口约85万，列全部汉族姓氏的第153位<ref>{{cite web | title=中国最新300大姓排名 | url=http://www.chinataiwan.org/zppd/XSDG/200901/t20090106_811068.htm }}</ref>。

== 起源 ==
1.[[凃姓|凃姓]]是以[[河流|河流]]名為姓氏。古代該河常結冰因而有凃水說（江蘇南京市六合區內的長江支流滁河），[[唐朝|唐朝]]時改為滁水，現在則名為滁河。根據《名賢氏族言行類稿》中的記載：凃氏為[[洪州|洪州]]人（即[[豫章郡|豫章郡]]，今[[江西|江西]][[九江|九江]]一帶），以冰字首凃為姓。後來他房祖先因對朝廷有功而受[[明朝|明朝]]皇帝御賜封姓：原本“凃姓”字形“二點冰”，多一點是以[[硃砂|硃砂]]紅點，變化成字形“三點水”的“涂姓”，示明為有功之後。

2.-{塗}-姓源于-{塗}-山氏，出自夏朝[[大禹|大禹]]之妻-{塗}-山氏，属于[[复姓|复姓]]省文简化为氏。据史籍《姓氏族谱笺释》、《风俗通》等记载，夏王朝时期有复姓-{塗}-山氏，其族人后省文去山字，称-{塗}-氏。[[中华人民共和国|中华人民共和国]]成立后，因[[汉字简化|汉字简化]]，-{塗}-姓改为-{涂}-姓。

== 郡望 ==
*豫章郡：漢朝時改原九江郡為豫章郡。今江西南昌、九江一帶。
*南昌郡：春秋战国时期属吴楚。秦朝时期属九江郡。今江西南昌。
*宜黄郡：宜黄境域在春秋时期属吴国。战国初期属越国，楚灭越后属楚国。秦始皇统一中国后属九江郡。今江西宜黄县。

== 堂號 ==
*豫章堂、南昌堂、宜黄堂、五桂堂

==名人==
*[[涂惲|-{涂}-惲]]：漢朝官員、學者，曾任諫議大夫，《尚書》研究者
*[[涂欽|-{涂}-欽]]：晉朝官員，封新吳侯，為東南塗氏之祖
*[[涂瑞|-{涂}-瑞]]：字邦祥，明朝官員、學者、書法家，成化年間進士，曾任翰林院編修、修撰
*[[涂天相|涂天相]]：清朝康熙四十二年进士，官至刑部、兵部、工部尚书
*[[涂长望|涂长望]]：中国近代气象科学的奠基人之一，九三学社组织者之一，中国科学院院士
*[[涂光炽|-{涂}-光炽]]：地质学家、矿床学家、地球化学家，中国地球化学研究的奠基人，中国科学院院士
*[[涂永强|-{涂}-永强]]：兰州大学功能有机分子化学国家重点实验室主任，中国科学院院士
*[[涂羽卿|涂羽卿]]：中国物理学家，曾任圣约翰大学校长
*[[涂謹申|涂謹申]]：香港立法會議員
*[[涂醒哲|涂醒哲]]：[[嘉義市|嘉義市]]第9任市長，中華民國第8任衛生署(衛福部)署長
*[[涂敏恆|涂敏恆]]：台灣作曲家，著名作品《送你一把泥土》
*[[涂又仁|-{涂}-又仁]]：台灣藝人，劇場演員出身，演員、主持人。
