'''蔣勳'''（{{bd|1947年|1月8日|}}），生於[[西安|西安]]、成長於[[臺北|臺北]]，著名[[畫家|畫家]]、[[詩人|詩人]]與[[作家|作家]]。

== 生平 ==
蔣勳父親為[[福建|福建]][[長樂|長樂]]人，服務於糧食局，母親[[陝西|陝西]][[西安|西安]]駐防[[正白旗滿洲|正白旗滿洲]]人。蔣勳出生於[[西安|西安]]，戰後舉家移居台灣。自小成長於[[台北|台北]][[大龍峒|大龍峒]]，<ref>[http://news.chinatimes.com/Chinatimes/Philology/Philology-artnews/0,3409,112009090200332+110513+20090902+news,00.html 吳錦勳著，《台灣，請聽我說》，天下文化出版] {{webarchive|url=https://web.archive.org/web/20131203022117/http://news.chinatimes.com/Chinatimes/Philology/Philology-artnews/0,3409,112009090200332+110513+20090902+news,00.html |date=2013-12-03 }}</ref>他認為自己的母語是[[西安話|西安的地方方言]]。

在[[法國|法國]]留學時他曾參與當地華人[[無政府主義|無政府主義]]組織，並辦了刊物《歐洲通訊》，後來因故退出。他在大學擔任教職時，支持[[黨外|黨外]]的[[民主化|民主化]]改革運動，也曾撰文聲援[[美麗島事件|美麗島事件]]的[[王拓|王拓]]，因而遭到革職。

2016年1月10日現身台北「[[綠黨社會民主黨聯盟|綠黨社會民主黨聯盟]]造勢大會」，表示[[綠黨|綠黨]]、[[社會民主黨|社會民主黨]]長久以來在台灣環保、食安、居住正義等奉獻，讓其從政治的冷感中走出來，所以「我把我的第一次獻給他們」。<ref>{{cite news|author=林良昇|title=為綠社盟站台 蔣勳：讓我從政治的冷感中走出來|url=http://news.ltn.com.tw/news/politics/breakingnews/1567663|accessdate=2016-01-10|newspaper=[[自由時報|自由時報]]|date=2016-01-15}}</ref>

== 作品 ==
蔣勳以花卉、水景繪畫受台灣人歡迎、以[[美感|美感]]的教學和省思受到學子喜愛。他畢業於[[中國文化大學|中國文化大學]]史學系<ref name="craahs">{{Cite web|url=http://craahs.pccu.edu.tw/bin/home.php|title=中國文化大學史學系|accessdate=2017-03-07|work=craahs.pccu.edu.tw|language=zh-tw}}</ref>和藝術研究所。於1972年到[[法國|法國]]留學，1976年返台。在繪畫創作之餘，蔣勳也在文學界勤勞耕耘，他出版過多本詩集、並曾擔任台灣早期美術刊物[[雄獅美術|雄獅美術]]的主編（總編輯）。

蔣勳曾得過全台灣[[小說|小說]]比賽第一名、[[中國時報|中國時報]][[新詩|新詩]]推薦獎以及[[吳魯芹|吳魯芹]]文學獎，他從青年時代就是個文藝青年。1966年參加[[耕莘青年寫作會|耕莘青年寫作會]]舉辦的寫作班，後來進而成為授課教師。他還有另一個特色就是曾經做過[[廣播|廣播]]節目《文化廣場》、此節目由台灣[[警察廣播電台|警察廣播電台]]播出，相當受到好評，獲得了1988年的[[金鐘獎|金鐘獎]]。自2004年起，於[[IC之音電台|IC之音電台]]．FM97.5（[[竹科廣播股份有限公司|竹科廣播股份有限公司]]）主持每週五晚間八點「美的沉思」節目，並以此節目再度獲得2005年[[廣播金鐘獎|廣播金鐘獎]]最佳藝術文化主持人獎。
