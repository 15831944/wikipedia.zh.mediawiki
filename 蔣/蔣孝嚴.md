'''蔣孝嚴'''（{{bd|1942年|3月1日<ref>[http://book.jrj.com.cn/book/book/detail_8751.shtml 蒋家门外的孩子]
根据[[蒋孝严|蒋孝严]]著述《蒋家门外的孩子》一文中所叙述：{{Cquote 2|我於二○○二年五月十日，向台北市大安区户政事务所正式提出更正身份证父母姓氏不实及生父认领事宜之申请。七月十二日与桂林市第二人民医院（原广西省立医院）联系，由该院於八月一日出具出生证明：“章亚若於公元一九四二年叁月一日在我院分娩一对双胞胎（当时乳名大毛、小毛，又称丽儿、狮儿），即章孝严、章孝慈。特此证明。”}} 蒋孝严先生与桂林市第二人民医院联系，由该院所出具出生证明“章亚若於公元一九四二年叁月一日在我院分娩一对双胞胎”可认为蒋孝严先生本人亦认可此一证明内容。</ref>
<ref name="birthdate">[http://www.scu.edu.tw/planning/Historyweb/new/new_page_10.htm#%B3%B9%A7%B5%B7O 東吳大學歷任校長] 上註明章孝慈是1942年5月出生，由於蔣孝嚴、蔣孝慈兩人是雙胞胎，所以原先所知1941年5月2日生可能有誤。蔣孝嚴也曾說過身分證上的生日是舅舅代報的，不是正確日期。</ref>|||}}）原名'''章孝嚴'''，[[中華民國|中華民國]]政治人物，籍贯[[浙江|浙江]][[奉化|奉化]]，生於[[廣西|廣西]][[桂林|桂林]]，[[蔣經國|蔣經國]][[雙胞胎|雙胞胎]]兒子中的兄長。曾任[[中國國民黨副主席|國民黨副主席]]、[[立法委員|立法委員]]、[[總統府秘書長|總統府秘書長]]、[[中國國民黨|國民黨秘書長]]、[[行政院副院長|行政院副院長]]、[[外交部|外交部]]部長等職。

== 早年生活 ==
章孝嚴與[[章孝慈|章孝慈]]是[[蔣經國|蔣經國]]與部屬[[章亞若|章亞若]][[外遇|婚外]]所生的[[雙胞胎|雙胞胎]]，他們於[[中國抗日戰爭|中國抗日戰爭]]期間在[[桂林|桂林]]出生。由於蔣經國已經與[[蔣方良|蔣方良]]結婚，因此這兩名非婚生子僅能隨母親的[[章姓|章姓]]，名字由[[蔣中正|蔣中正]]定為「孝嚴」、「孝慈」<ref>{{Cite web |url=http://www.books.com.tw/exep/prod/booksfile.php?item=0010330483 |title=蔣家門外的孩子—蔣孝嚴逆流而上 天下文化 2006年05月05日 |accessdate=2012年1月16日 |archiveurl=https://web.archive.org/web/20070824080959/http://www.books.com.tw/exep/prod/booksfile.php?item=0010330483 |archivedate=2007年8月24日 |deadurl=yes }}</ref>，與蔣經國的其他兒女一樣是「孝」[[字輩|字輩]]。

章亚若於1942年8月被[[謀殺|謀殺]]，兩兄弟由章亞若的弟弟章浩若及他的妻子紀琛扶養；舅舅及舅母在官方文件上登記為兩兄弟的生父母，直到2005年才改回真正的生父母。

[[國共內戰|國共內戰]]其間，兩兄弟隨外婆周錦華移居台灣[[新竹|新竹]]，由外婆、舅舅（章渙若）、舅母（章向鳳妹）繼續撫養。1960年章孝嚴畢業於臺灣省立新竹中學（今[[國立新竹高級中學|國立新竹高級中學]]），與1959年[[許信良|許信良]]同校但為下一期校友。

後來兩兄弟同時在[[東吳大學|東吳大學]]就讀。1960年底，在他們18歲時才知道祖父是蔣中正、蔣經國為他們父親。<ref>《[[聯合報|聯合報]]》，台北，1990年1月2日</ref>

== 從政資歷 ==
=== 外交工作 ===
1974年到1977年間，章孝嚴在[[中華民國駐美國大使館|中華民國駐美國大使館]]工作。這期間在[[喬治城大學|喬治城大學]]進修，獲得碩士學位，[[公家|公家]]高層逐步拔擢，升到[[中華民國外交部|中華民國外交部]]北美司司長。

1996年6月10日至1997年10月20日期間，擔任[[中華民國外交部|中華民國外交部]]部長。

2008年11月，兼任[[中國國民黨副主席|中國國民黨副主席]]，2014年4月底卸任。

2011年4月，[[中國國民黨|中國國民黨]]同黨的競爭者[[羅淑蕾|羅淑蕾]]在國民黨初選民調中以不到百分之一的差距險勝蔣孝嚴，雖然北市黨部聲明不是最終的結果，但蔣孝嚴仍決定退出臺北市第三選區立法委員的選舉並投入輔選，並展現君子風度主動擔任羅淑蕾[[立法委員|立法委員]]競選團隊主任委員。

== 改為蔣姓 ==
原稱「章孝嚴」，一直等到蔣經國配偶[[蒋方良|蒋方良]]去世，正式改稱「蔣孝嚴」，並稱其子女將會跟隨。章孝嚴身份證父親欄寫的是舅父章浩若，母親欄是舅母紀琛。章孝嚴提出經由[[王昇|王昇]]將軍證實在[[章孝嚴|章孝嚴]]、[[章孝慈|章孝慈]]小的時候，蔣經國提供了兩個孩子的生活費，有撫養的事實<ref>[http://www.chinareviewnews.com/doc/1010/3/0/2/101030241_3.html?coluid=7&kindid=0&docid=101030241&mdate=0724120319 蔣孝嚴：身處“蔣家門外”，留下終身遺憾 中國評論 2009-07-24]</ref>。

2002年7月，章孝嚴到[[江西|江西]][[南昌|南昌]]，找到章浩若兩個兒子章修純、章修維，寫下了聲明書，證明章浩若是他們的父親，非章孝嚴生父，章亞若是他們的三姑，也是章家雙胞胎的生母。此後章孝嚴從桂林醫院取得出生証明，母親欄寫的是舅媽紀琛女士，但她已經遷居美國多年。2003年章孝嚴到美國訪問舅媽，在[[中華民國|中華民國]]駐洛杉磯辦事處兩位人員的見證下，取得幾根頭髮拿去做DNA化驗，在駐外辦事處見證下，紀琛和章孝嚴這對「法律母子」的關係劃斷。<ref>[http://www.businessweekly.com.tw/article.php?id=15757 章孝嚴跋涉萬里追蹤DNA]</ref>2005年3月經由[[臺北市政府民政局|臺北市政府民政局]]、[[內政部|內政部]]、[[法務部|法務部]]等部門一系列複雜的法律程序，將身份証父親欄改為蔣經國。<ref>[http://news.sina.com.cn/c/2005-07-23/07536507175s.shtml 蔣孝嚴：認祖歸宗背後的秘密 瀟湘晨報 2005年07月23日]</ref>

其弟[[章孝慈|章孝慈]]曾於《那段剝花生充饑的日子》一文中提過，兩人的外婆周錦華女士曾對他們說過，兩人的父親[[蔣經國|蔣經國]]是個正直勇敢能幹的人。

==外遇誹聞事件==
1999年12月21日，時為總統府秘書長章孝嚴透過總統府發言人丁遠超發表聲明，坦承11月10日於國民黨秘書長任內「在不正常的情況下」簽下一紙「我承諾在明年六月底以前完成和她（黃美倫）辦妥離婚的手續」的台北晶華酒店便箋。<ref>[http://forums.chinatimes.com/special/king/88122202.htm　章孝嚴婚外情 王筱嬋傳為女主角]</ref><ref>[http://forums.chinatimes.com/special/king/scandal.htm　章孝嚴緋聞案全收錄]</ref><ref>[http://old.ltn.com.tw/2002/new/nov/14/today-p12.htm　王筱嬋︰曾打章孝嚴一巴掌], [[自由時報|自由時報]], 2002-11-14</ref>辭職後的章孝嚴成為迄今史上任期最短的[[總統府秘書長|總統府秘書長]]。
