
[[蔣經國|蔣經國]]在[[台灣|台灣]]廣泛營造出平易近人形象。其中有關其溫和形象經常被[[中國國民黨|中國國民黨]]提及，許多[[泛藍|泛藍]]支持者承認蔣經國努力發展台灣經濟和政治開放，對台灣經濟發展作出巨大貢獻。[[黨外運動|黨外運動]]及之後[[泛綠|泛綠]]陣營對蔣經國意見較含蓄。一些長期支持政治自由者認為，雖然蔣經國晚年解除[[台灣省戒嚴令|台灣省戒嚴令]]，但是蔣經國大部分統治時期仍是對臺灣實施威權統治。在[[陳水扁|陳水扁]]當選總統後，則下令開始[[去蔣化|去蔣化]]，移除過去擺設蔣經國的畫像。

而「蔣經國」的名稱也常在[[臺灣|臺灣]]道路等相關命名中出現，例如[[桃園市|桃園市]][[桃園區|桃園區]][[經國路_(桃園市)|經國路]]、[[新竹市|新竹市]][[經國路_(新竹市)|經國路]]、[[臺中市|臺中市]][[大甲區|大甲區]][[經國路_(大甲區)|經國路]]，但數量遠不及以其父所命名之中正路。此外尚有[[中華民國空軍|中華民國空軍]][[F-CK-1經國號戰鬥機|F-CK-1經國號戰鬥機]]、[[嘉義市|嘉義市]][[東區|東區]][[經國新城團結愛國眷村社區|經國新城團結愛國眷村社區]]、[[經國管理暨健康學院|經國管理暨健康學院]]、[[桃園市立經國國民中學|桃園市立經國國民中學]]、東南科技大學經國[[圖書館|圖書館]]等。

== 古蹟建築 ==
# 杭州蔣經國故居：始建於1931年，占地約400坪，由主副兩棟樓組成，是西式磚木別墅，緊鄰杭州西湖。據媒體報導稱，蔣經國曾於1948年10月至11月期間，攜妻兒來杭州在此暫住一個月。
# 贛南蔣經國舊居
#[[大溪陵寢|大溪陵寢]]：為奉厝中華民國第六、第七任總統蔣經國遺體靈櫬之處，位於台灣桃園市大溪區福安里。
#[[桃園市|桃園市]]的'''經國紀念館'''：原先是大溪遊客中心，經[[桃園縣政府|桃園縣政府]]於2012年整頓後重新設立，並結合[[大溪陵寢|大溪陵寢]]、復興賓館、[[慈湖陵寢|慈湖陵寢]]、慈湖紀念雕塑公園等地標規劃為兩蔣文化園區。
#[[金門縣|金門縣]]的'''蔣經國先生紀念館'''：1989年，為紀念蔣經國對於國家貢獻及長期關懷金門縣發展，在金門防衛司令部司令官[[黃幸強|黃幸強]]和[[程邦治|程邦治]]倡議，於福建省金門縣成立蔣經國先生紀念館，並於2008年重新整修後再度啟用。地址於金門縣金寧鄉的金門國家公園乳山遊客中心之中山紀念林內，同[[大溪陵寢|大溪陵寢]]一樣為[[四合院|四合院]]建築<ref>
[http://sixstar.moc.gov.tw/blog/w321461/knowHowFronListAction.do?method=doReadKnowHow&source=2&knowhowId=292 金門縣蔣經國先生紀念館]</ref>。
#[[連江縣|連江縣]]的'''經國先生紀念堂'''：經過軍政首長會議討論後，於福建省連江縣馬祖列島勝利水庫西側山丘上成立經國先生紀念堂。仿[[中山陵|中山陵]]、[[中正紀念堂|中正紀念堂]]為藍瓦白牆建築。紀念館1樓為經國先生坐姿銅像與遺囑，2樓則為文物史料，展示經國先生歷年視察馬祖防務的老照片，以及各類文書往來的手稿，1994年6月完工開幕<ref>[http://www.matsu-nsa.gov.tw/user/Attraction-Content.aspx?a=31&l=1 連江縣經國先生紀念堂]</ref>。
# [[七海寓所|七海寓所]]：[[臺北市市定古蹟|臺北市市定古蹟]]，位於臺北市大直北安路，是故總統蔣經國的總統官邸。因官邸警衛室代號為「七海」，所以稱為「七海寓所」。

== 重要文物 ==	
# 蔣經國紀念酒：主要由[[金門酒廠|金門酒廠]]發行。例如：民國67年蔣經國謝東閔總統副總統就職紀念酒。
# 組曲《[[蔣經國紀念歌|蔣經國紀念歌]]》：為了緬懷故去的中華民國總統蔣經國而創作的紀念組曲，創作於1988年，作者[[黃友棣|黃友棣]]，亦即1975年蔣中正逝世後《蔣公紀念歌》的譜寫者。	
# 蔣經國紀念郵票：由[[中華郵政|中華郵政]]發行。
# 蔣經國紀念金幣：主要由[[中華民國中央銀行|中華民國中央銀行]]委託[[中央造幣廠|中央造幣廠]]發行。
# 蔣經國紀念流通幣：2010年4月27日，中華民國中央銀行委託中央造幣廠發行「蔣故總統經國先生百年誕辰紀念流通幣」，面額新台幣10元。
