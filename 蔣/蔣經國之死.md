1988年1月13日下午3時50分，[[中華民國總統|中華民國總統]]、[[中國國民黨主席|中國國民黨主席]][[蔣經國|蔣經國]]因[[心衰竭|心衰竭]]在[[臺北市|臺北市]][[七海寓所|七海寓所]]逝世，享年78歲。蔣去世後，副總統[[李登輝|李登輝]]依《[[中華民國憲法|中華民國憲法]]》繼任總統，此後[[中國國民黨|中國國民黨]]內引發一連串的權力鬥爭；最終李登輝順利接掌[[中國國民黨主席|中國國民黨主席]]，全面接掌黨政軍權力。

== 背景 ==
蔣經國1972年成為[[行政院院長|行政院院長]]后開始體弱多病，主要是[[糖尿病|糖尿病]]日漸惡化引起[[視網膜|視網膜]]模糊、[[腎臟|腎臟]]發炎和雙腿肌肉壞死。這使得其後來必須藉由[[輪椅|輪椅]]代步。<ref name="演變">{{cite book|author=茅家琦|url=http://books.google.com.tw/books?id=nNuAWMnHvyYC&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false|title=《蔣經國的一生和他的思想演變》|location=[[臺北|臺北]]|publisher=[[臺灣商務印書館|臺灣商務印書館]]|year=2003|isbn=9789570517965}}</ref>{{rp|437}}[[臺北榮民總醫院|臺北榮民總醫院]]診治，認為其[[血糖|血糖]]含量很難恢復正常。<ref name="演變"/>{{rp|437}}

1976年10月2日，[[宋美齡|宋美齡]]函電蔣經國：「……唯一方法祗有多休息與不為已甚彥-{云}-養病如養虎盼毋可勿視……母九月卅日」<ref>{{cite book|author=蔣經國|title=《蔣經國書信集：與宋美齡往來函電》（上）|location=臺北|publisher=[[國史館|國史館]]|year=2009|pages=|isbn=9789860195903|accessdate=2014-02-21}}</ref>

1982年2月3日，蔣經國總統在進行左眼[[視網膜|視網膜]]手術後，意識到健康問題嚴重，開始將軍權完全交給[[郝柏村|郝柏村]]負責（出任[[參謀總長|參謀總長]]）。蔣經國視力出現問題，直至逝世前一段時間，曾多次入院做眼部手術，病情嚴重。<ref name="演變"/>{{rp|416}}

1987年10月10日，由於長期糖尿病引起之足部末梢神經炎，手術後雙足無力，首次公開以輪椅代步，主持[[國慶日_(中華民國)|國慶]]大典。<ref name="列傳(4)">{{cite book|editors=[[王成斌|王成斌]]等主編|title=《民國高級將領列傳》(4)|location=北京|publisher=[[解放軍出版社|解放軍出版社]]|year=1998}}</ref>{{rp|526}}

1988年1月2日，[[鄭南榕|鄭南榕]]創辦的《人權時代週刊》205號以封面照片報導蔣經國左眼已經瞎掉，<ref>{{cite book|author=|title=《人權時代週刊》205號|location=|publisher=|year=1988|pages=|isbn=|accessdate=}}</ref>顯示其糖尿病的惡化。並在1月9日《人權時代週刊》206號中報導〈蔣經國大限已近 國民黨內鬪加劇〉，追蹤蔣經國健康情形。<ref>{{cite book|author=|title=《人權時代週刊》206號|location=|publisher=|year=1988|pages=|isbn=|accessdate=}}</ref>

== 驟逝 ==
1988年1月1日，蔣經國坐在輪椅上主持中樞會議。<ref name="列傳(4)"/>{{rp|527}}1月13日早上7時30分，在七海寓所起床後突感身體不適。<ref name="列傳(4)"/>{{rp|527}}有輕度噁心嘔吐現象，經[[醫生|醫生]]檢查後，其[[血壓|血壓]]、[[脈搏|脈搏]]和[[體溫|體溫]]皆正常，由於未進食[[早餐|早餐]]，隨即以[[靜脈注射|靜脈注射]]方式補充營養。下午1時55分，大量[[出血|吐血]]。<ref name="列傳(4)"/>{{rp|527}}蔣經國迅速[[休克|休克]]及心臟呼吸衰竭。雖經醫療小組立即以[[心肺復甦|心肺復甦]]搶救，延至下午3時55分[[心臟|心臟]]停止跳動，[[瞳孔|瞳孔]]散大，因[[心衰竭|心衰竭]]病逝於七海寓所，享壽77歲。

副總統李登輝則是在總統府會客室接見外客時，由秘書[[蘇志誠|蘇志誠]]報告蔣經國的機要秘書王家驊來電請李登輝到七海官邸，李登輝到時蔣經國已經逝世<ref name="李登輝">{{cite book |author=鄒景雯採訪 |title=《李登輝執政告白實錄》 |location=臺北 |publisher=印刻出版 |year=2001 |isbn=957-2044-30-3 }}</ref>。日後李登輝與[[司馬遼太郎|司馬遼太郎]]對談時曾表示「他就算罹患了那樣的重病，也沒想到自己會就此結束生命。所以，臨終的父親沒有對兒子留下任何像是遺言的訊息。」<ref>{{cite book |author=司馬遼太郎 |title=《台湾紀行》 |location= |publisher=[[朝日新聞|朝日新聞]]社 |year=1994 |pages=500 |accessdate=2015-10-29 }}</ref>

晚上8時8分，在[[司法院|司法院]]院長[[林洋港|林洋港]]監誓下，中華民國副總統李登輝依《[[中華民國憲法|中華民國憲法]]》宣誓繼任中華民國總統。晚上8時45分，[[電視台|電視台]]中斷正常節目廣播，公佈蔣經國逝世及李登輝宣誓繼任總統之消息<ref>{{cite news |title=〈早晨心臟病發 下午吐血告不治 蔣經國昨猝然去世 台島局勢益受矚目 國民黨三軍戒備 宣佈禁止聚眾集會遊行〉 |publisher=《[[大公報|大公報]]》 |page=1 |date=1988-01-14 |accessdate=2014-02-09 }}</ref><ref>{{cite news |title=〈蔣總統逝世李登輝即繼任 決遵守憲法增進全民福祉〉 |publisher=《[[華僑日報|華僑日報]]》 |page=1 |date=1988-01-14 |accessdate=2014-02-09 }}</ref>。

蔣經國逝世後，官方公布宣稱是由蔣經國的機要秘書[[王家驊|王家驊]]記下的蔣經國遺囑，全文為：

{{Cquote|
經國受全國國民之付託，相與努力於以[[三民主義統一中國|三民主義統一中國]]大業，為共同奮鬥之目標。萬一，-{余}-為天年所限，務望我政府與民衆堅守反共復國決策，並望始終一貫積極推行[[民主|民主]]憲政建設。全國軍民，在[[孫文|國父]]三民主義與[[總統蔣公遺囑|先總統遺訓]]指引之下，務須團結一致，奮鬥到底，加速光復大陸，完成以三民主義統一中國之大業，是所切囑。

中華民國七十七年元月五日王家驊敬謹記述

李登輝 [[俞國華|俞國華]] [[倪文亞|倪文亞]] [[林洋港|林洋港]] [[孔德成|孔德成]] [[黃尊秋|黃尊秋]] [[蔣孝勇|蔣孝勇]]
}}

1988年1月23日，作家[[李敖|李敖]]以「遺囑上的見證人簽名是到靈堂才簽的」為由，向台北市地檢處以偽造文書罪名告發時任總統李登輝，與五院院長俞國華、倪文亞、林洋港、孔德成、黃尊秋等人偽造蔣經國遺囑。同年3月，台北地檢處偵查終結，認為沒有犯罪的刑責，予以結案<ref>[https://www.ettoday.net/news/20180318/1132773.htm 告李登輝偽造蔣經國遺囑　李敖選總統支持兩岸統一和談].ETtoday.2018-03-18</ref>。根據繼任總統的李登輝回憶，事實上他與在場的僚屬及家屬[[俞國華|俞國華]]、[[郝柏村|郝柏村]]、[[沈昌煥|沈昌煥]]、蔣孝勇等人確認，證實蔣經國並未留下遺囑。因此李登輝與眾官員、家屬，依據蔣經國平常的發言、思想，在官邸討論遺囑內容後，由王家驊繕寫在十行紙上，以便數小時後公布<ref name="李登輝"/>{{rp|49}}。由於當時遺囑並未有蔣經國本人簽名、而且是由王家驊寫在市面文具店常見的[[金山牌|金山牌]]信紙上，而非公文用紙，因此公布時曾一度遭到民間雜誌質疑「偽造遺囑」。然而在李登輝2001年的回憶錄出版前，國民黨一直宣稱該份遺囑為真。

== 身後 ==
1月14日，設靈堂於[[台北榮民總醫院|台北榮民總醫院]]「懷遠堂」，遺像前擺放著宋美齡致送之十字架花圈。<ref name="列傳(4)"/>{{rp|527}}俄籍妻[[蔣方良|蔣方良]]坐輪椅攜子女多次往靈堂行禮拜祭。<ref name="列傳(4)"/>{{rp|527}}1月22日，移靈台北市[[圓山|圓山]][[國民革命忠烈祠|忠烈祠]]，供民眾瞻仰遺容，一週間前往者逾百萬之眾，一百零一歲之[[張群|張群]]亦坐輪椅前往，臥病十八年之長子[[蔣孝文|蔣孝文]]也由人攙扶靈前跪祭。<ref name="列傳(4)"/>{{rp|527}}1月30日，舉行大殮，奉厝[[桃園市|桃園縣]][[大溪區|大溪鎮]]「頭寮賓館」。<ref name="列傳(4)"/>{{rp|527}}

[[李登輝|李登輝]]追思蔣經國<ref>{{cite book|author=蔣經國|others=漆高儒|title=《蔣經國的一生》|date=1991-03-31|publisher=傳記文學出版社|location=台北|isbn=9578506074|quote=以李登輝來說，在「百日追思感言中」，就有這樣一段：民國七十一年七月，經國先生蒞臨[[中興新村|中興新村]]，是他生前最後一次視察台灣省政，在召見各廳處長、省府委員和中部各縣市長、議長談話後，他帶著那一貫的笑容說：「李主席，你知道為什麼請你擔任省主席嗎？」登輝未及回答，他緊接著說：「為民服務最重要的就是忠誠，我知道你做人處事都很有誠意，所以希望你用真心誠意來民眾服務。」推心置腹的信任與期許，登輝永生難忘。}}</ref>{{rp|238}}，並致贈輓聯哀悼：「厚澤豈能忘，四十年汗盡血枯，注斯土斯民始有今日；遺言猶在耳，億萬人水深火熱，誓一心一德早復中原。」<ref>{{cite web|url=http://hkmag.crntt.com/crn-webapp/mag/docDetail.jsp?coluid=29&docid=100939282&page=9|title=〈永懷經國先生〉|author=[[馬英九|馬英九]]|publisher=《[[中國評論|中國評論]]月刊》|date=2009-04-12}}</ref>[[黃少谷|黃少谷]]對蔣經國輓聯云：「憂患久相隨，論事陳情，許我言無不盡；治平時在念，拯民救國，痛公志有未申。」<ref>{{cite book|author=蔣經國|others=漆高儒|title=《蔣經國的一生》|date=1991-03-31|publisher=[[傳記文學_(雜誌)|傳記文學]]出版社|location=台北|isbn=9578506074}}</ref>{{rp|206}}

蔣經國逝世後，如同[[蔣中正|蔣中正]]決定靈柩暫厝在桃園縣大溪鎮[[慈湖陵寢|慈湖陵寢]]，蔣經國靈柩也暫厝於距離不遠的[[大溪陵寢|大溪陵寢]]。不過兩人都希望待來日光復大陸，再奉安於南京[[紫金山|紫金山]]。<ref>{{cite book|author=[[陳布雷|陳布雷]]等編著|title=《蔣介石先生年表》|date=1978-06-01|publisher=傳記文學出版社|location=台北}}</ref>{{rp|134}}<ref>《蔣總統經國先生言論著述彙編》第九輯</ref>{{rp|62}}1988年，為紀念蔣經國，中華民國作曲家[[黃友棣|黃友棣]]特別編寫組曲《[[蔣經國紀念歌|蔣經國紀念歌]]》。2004年1月，[[蔣方良|蔣方良]]曾經提議將蔣中正與蔣經國陵墓改移到位於[[新北市|臺北縣]][[汐止區|汐止市]]的[[國軍示範公墓|國軍示範公墓]]。遷移儀式最初訂在2005年春季，由於蔣方良於2004年逝世，負責整起儀式的蔣經國長女[[蔣孝章|蔣孝章]]直到2005年冬季才正式進行。

== 影響 ==
蔣經國逝世，宣告統治台灣43年的蔣家時代正式告終，蔣家走入歷史，李登輝其後全面接掌黨政軍權力，自此台灣民主化、本土化，1989年正式解除黨禁，1991年首次[[1991年中華民國國民大會代表選舉|全面改選國大代表]]，1992年首次全面[[1992年中華民國立法委員選舉|改選立法委員]]，1994年全面恢復[[1994年中華民國省市長暨省市議員選舉|民選地方自治]]，1996年首次舉行[[1996年中華民國總統選舉|總統直選]]，李登輝順利連任，台灣在1987年蔣經國解嚴後10年內完成民主化。
