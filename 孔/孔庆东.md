'''孔庆东'''（{{bd|1964年|9月22日|catIdx=Kong孔庆东}}），自称'''孔和尚'''（和尚原意是老师），汉族人，是一名中共党员。出生于[[黑龙江|黑龙江]][[哈尔滨|哈尔滨]]，祖籍[[山东|山东]][[费县|费县]]。[[哈尔滨市第三中学|哈尔滨市第三中学]]校友，本科毕业后师从于北大中文系的两位名师：是[[钱理群|钱理群]]先生的硕士、[[严家炎|严家炎]]先生的博士。[[北京大学|北京大学]][[文学|文学]]博士，现任[[北京大学|北京大学]]中文系教授<ref>{{cite web |url = http://www.sinoss.net/2011/0606/33623.php |title = 孔庆东 |publisher=中國高校人文社會科學信息網 |accessdate=2012-02-09}}</ref>，自称[[孔子和平獎|孔子和平獎]]發起人及評委<ref>{{cite web|url=http://china.dwnews.com/news/2011-12-09/58390595.html|title=孔庆东宣布普京获孔子和平奖|publisher=多維新聞|date=2011年12月9日|accessdate=2012-02-09|deadurl=yes|archiveurl=https://web.archive.org/web/20120207211221/http://china.dwnews.com/news/2011-12-09/58390595.html|archivedate=2012年2月7日|df=}}</ref>，常以具爭議性的言論，如极端仇视“西方资本主义自由化”思想而被受到網民注意。他曾经在韩国梨花女子大学任教2年。

== 早年 ==
孔庆东是孔子七十三代孙，属大宗户三支，十八世祖是袭封[[衍圣公|衍圣公]][[孔讷|孔讷]]第四子[[孔公镗|孔公镗]]<ref>孔徳墉主编 《孔子世家谱》 文化艺术出版社 ISBN 978-7-5039-3789-7</ref>，他对[[武侠小说|武侠小说]]较有研究，在一定程度上引起了学术界对武侠小说的重视。孔曾在中央电视台《[[百家讲坛|百家讲坛]]》节目主讲[[金庸|金庸]]和[[鲁迅|鲁迅]]。其《正说鲁迅》节目被牵强地指出存在多处错误。<ref>{{cite web |url = http://ent.sina.com.cn/x/2008-03-10/02301941734.shtml |title=孔庆东讲鲁迅穿帮多 |publisher=华龙网-重庆晚报|date=2008年3月10日|accessdate=2010-06-02}}</ref>。

孔庆东曾中途参加了[[六四事件|六四事件]]，并當選[[北大學運籌委會|北大學運籌委會]]第四任召集人，数日后因其他学生领袖不满其对校党委唯命是从而被改选下台<ref>{{cite book|title=《六四日记》|author=封从德|publisher=溯源书社|year=2009|location=香港|pages=646|isbn=978-988-16442-6-8}}</ref>。「六四」後被分配到北京一所中學任教，幾年後重返北大繼續學業<ref name="狡辯">{{cite news |url = http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20120122&sec_id=4104&subsec_id=11866&art_id=16006673 |title=孔慶東狡辯再罵港人冇腦|publisher=香港蘋果日報|date=2012-01-22}}</ref><ref>{{cite journal |url = http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20120206&sec_id=4104&subsec_id=15337&art_id=16043909&coln_id=6970284 |title=與孔慶東絕交書|author=中國獨立作家 余杰|publisher=香港蘋果日報|date=2012-02-06}}</ref><ref>{{cite news|url=http://www.libertytimes.com.tw/2012/new/feb/3/today-fo1.htm|author=林翠儀|title=狗論激化對立，孔慶東強硬大駡香港人|publisher=自由時報電子報|date=2012-02-03|deadurl=yes|archiveurl=https://web.archive.org/web/20120205201237/http://www.libertytimes.com.tw/2012/new/feb/3/today-fo1.htm|archivedate=2012-02-05}}</ref>。
一些评论认为，孔庆东之后的思想變得[[極左派|激进左倾]]（但孔庆东本人多次强调现代青年应避免非左即右的简单化思想）。其文章经常被[[乌有之乡|乌有之乡]]所刊登。他经常指出对手的“汉奸”性，其体现为不站在民众的立场、中华民族的立场，有意或无意地损害中华民族利益。

== 争议 ==

=== 鲁迅的地位不可动摇 ===
他最推崇的作家是鲁迅, 但同时又认为《狂人日记》、《阿Q正传》对现在的中学生来讲的确太深了。而《从百草园到三味书屋》这一类散文则更容易为中学生所理解。

=== 批《伶人往事》 ===
2007年，孔庆东在演讲中公开批驳[[章诒和|章诒和]]的《[[伶人往事|伶人往事]]》一书，指“她那个阶级是我们政权的敌人”，“共产党对他们是极其宽大的，但他们仍梦想变天，说当年反右反错了”，孔庆东说：“你（指右派）既然认为是堂堂正正的英雄，为什么要求共产党平反？”“改革开放后都平反昭雪了，但大右派还百倍疯狂地向人民索取，比当年凶恶10倍”。“我们平民百姓的血泪谁去写？矿井砸死60多人，谁给每人写一部《往事并不冒烟》他们一人死了赔多少钱？生命都是有价钱的。上层人的生命价格和下层人的生命价格不一样吗？革命本来就要改变这东西。”

=== 助手伤人 ===
2008年11月14日，当时南方报业旗下的《[[新京报|新京报]]》文化版编辑徐来在其博客“钱烈宪要发言”上发表文章《传孔庆东被拘审查》，称孔庆东因组织“北大主体思想学习小组，为朝鲜提供情报被警方带走”。2009年2月14日，当徐来在朝阳区万达广场举办新书《想象中的动物》读者见面会时，被孔庆东的助理杨春持刀刺成重伤。杨春以故意伤害罪判处有期徒刑4年6个月。南方报网同时称孔庆东对于杨春刺伤徐来一事并不知情。<ref>{{cite web |url=http://ent.nfdaily.cn/content/2009-08/07/content_5509329.htm |title=孔庆东被曝泄露情报 助理刺伤媒体人获刑四年半 |publisher=京华时报 |date=2009-08-07 |accessdate=2010-05-15 |archive-url=https://web.archive.org/web/20100701112717/http://ent.nfdaily.cn/content/2009-08/07/content_5509329.htm |archive-date=2010-07-01 |dead-url=yes }}</ref>

=== 推崇朝鲜 ===
2010年，孔庆东批评[[韩国|韩国]]，推崇[[朝鲜|朝鲜]]的制度和[[金正日|金正日]]的「[[主体思想|主体思想]]」，称“没有伟大的领袖和劳动党，没有强大的[[朝鲜人民军|人民军]]，没有宁死不屈的民族骨气和奋发忘我的劳动热情，早都死绝了。”他的《独立韩秋》一书在韩国政府要求下遭禁，随后改名《匹马西风》重新出版。在书中孔庆东指朝鲜人民沒有捱餓，其說法遭炮轟不盡不實。於2012年中期，孔慶東於其[http://t.qq.com/beidakongqingdong 微博]中讚揚北韓是個學習型國家。重視教育，反指「日本人雖然也滿大街看書，但看的多是漫畫，而且八成是色情漫畫」。<ref>{{cite web|url=http://news.cn.yahoo.com/kongqingdong/|title=文化人物系列之孔庆东|publisher=雅虎资讯－人物声音 第22期|quote=他，冷嘲热讽、愤世嫉俗，极力推崇北朝鲜的制度和金正日的主体思想；他赞美朝鲜专制|accessdate=2012-03-21|deadurl=yes|archiveurl=https://web.archive.org/web/20120314214708/http://news.cn.yahoo.com/kongqingdong/|archivedate=2012-03-14}}</ref>。

2010年2月，孔庆东在第一视频（v1.cn）节目中对新闻“英媒曝朝鲜为金正恩建造豪宅办公室”的评论中称“刚从朝鲜回来，朝鲜人民没有挨饿”，并且中国人也“从60年代到现在吃得好好的，红光满面的”，该新闻是“帝国主义”和“狼”对北朝鲜领导人的妖魔化和丑化，并指责当代中国人“不要脸”，没有资格去“跟着帝国主义批评自己的一个兄弟国家”。孔在节目中还当着电视观众的面批评该栏目主持人石菲犯了错误，要求其做“深刻的反思”。

=== 设计“辱骂”南方系记者诱其上钩 ===
2011年11月7日下午，孔在其[[新浪|新浪]]官方认证[[微博|微博]]上声称，之前在拒绝《[[南方人物周刊|南方人物周刊]]》采访时爆粗口:
{{cquote|一分钟前，汉奸刊物《南方人物周刊》电话骚扰要采访我，态度很和气，语言很阴险。孔和尚斩钉截铁答复了一个排比句：去你妈的！滚你妈的！操你妈的！}}随后，遭辱骂的记者曹林华接受《[[济南时报|济南时报]]》的采访，指出孔庆东并没有在电话中说出上述排比句，只是在微博中夸大了用語，並认为孔庆东的性格具有理性和表演性的两面性。此事引发了媒体和一些公共知识分子的批评。但是相当多网民表达了对孔庆东的支持<ref>{{cite web |url = http://www.bbc.co.uk/zhongwen/simp/indepth/2011/11/111115_analysis_kongqingdong_3mothers.shtml|publisher=BBC中文网 |accessdate=2011-11-17 |title = 评论：中国媒体右倾政治左转？}}</ref>，比如[[凤凰网|凤凰网]]举行的网络调查中目前得到了多数投票者的支持<ref>{{cite web |url = http://news.ifeng.com/mainland/special/kongqingdong/|publisher=凤凰网|accessdate=2011-11-11 |title=粗口教授孔庆东}}</ref><ref>{{cite web |url = http://view.news.qq.com/zt2011/kqd/index.htm |publisher=腾讯网 |date=2011-11-11 |accessdate=2011-11-11 |title=为何6成网友支持孔庆东骂记者}}</ref>。孔庆东事后说，他的“三骂”不过“是个火力侦察”，有意让“汉奸媒体”对他“ 进行‘反革命围剿’”，让他们“自我暴露”。

=== 辱骂部分甘当走狗、歧视同胞的香港人 ===
{{Main|孔慶東辱罵港人事件}}

2012年1月19日，孔庆东於中国第一視頻網絡電視台《孔和尚有話說》節目中，對香港網上一段內地兒童於香港[[港鐵|港鐵]]車廂進食的短片作出評論。孔庆东認為港人過去受到[[殖民主義|殖民主義]]影響，自我身分優越，在約6分半鐘長的節目中，說了3次「香港很多人是狗」。隨即引起许多香港人强烈不满，同时也引发了一些内地网友的指责，质疑其作为[[北京大学|北京大学]]教授的素质。<ref>{{cite news|url=http://news.163.com/12/0121/10/7O9KLPHG00014JB6.html|title=北大教授孔庆东骂“部分香港人是狗”|author=中国新闻网|publisher=网易新闻|date=2012-01-21|accessdate=2012-01-21|deadurl=yes|archiveurl=https://web.archive.org/web/20120124023621/http://news.163.com/12/0121/10/7O9KLPHG00014JB6.html|archivedate=2012-01-24}}</ref>

另一方面，孔庆东亦談及到本港的殖民地歷史及法治制度，他表示:
{{cquote|很多香港人不认为自己是中国人，总是开口就说我们香港啦，你们中国啦……这种人給人家英國殖民者當走狗當慣了，到現在都是狗，你們不是人。我知道香港人有很多是好人，但也有很多香港人，至今都是狗。}}
另外，他又認為香港的法治制度是英國殖民時代的遺留物，並把以上提到的一类香港人比作[[朝鮮籍日本兵|朝鮮籍日本兵]]、[[台灣籍日本兵|台灣籍日本兵]]。他說：「凡是用法治維持起來的秩序，說明你們的人沒有素質、沒有自覺……一個字：賤。」<ref>[http://news.mingpao.com/20120121/gaa1.htm 北大惹火教授罵港人是狗 時事評論員：中港矛盾深化 促政策介入] - 香港明報</ref>

1月21日，面对南方报系等媒体的质疑，孔庆东接受第一視頻電話訪問時，曾經解释道：
{{cquote|我說過（所有）香港人是狗嗎？沒有！}}
{{cquote|我甚麼時候說過不說普通話的就是王八蛋？......你用對方不懂的話去侮辱對方就是王八蛋！}}
此外，孔庆东指出是这些媒體歪曲其言論，斷章取義，故意炒作、抹黑他<ref>[http://www.dzwww.com/xinwen/guoneixinwen/201201/t20120122_6875881.htm 北大教授孔慶東稱遭斷章取義 沒說「香港人是狗」] - 大眾網</ref>，
{{cquote|特別是過年的時候，就有一些言論，破壞香港跟內地的關係}}
他對媒體的報道手法感到遺憾及要求道歉，又強調自己只是罵部分香港人是狗，而他也曾罵北京人、上海人是狗。{{cquote|我說哪裏都有一部份人是狗，北京也有一部份是狗}}
他稱不少內地人到香港旅遊時遭到不禮貌的對待，認為自己只是批判事實，是為了香港人好。

孔慶東又在微博中寫到﹕「假如真有一個人說『香港人是狗』，那這個人應該道歉。而這句話恰好是南方報系說的，所以我鄭重要求南方報系向我並同時向香港人民道歉。支援南方報系認罪的舉手！」<ref>[http://news.mingpao.com/20120122/gaa1.htm 東鐵罵戰主角籲勿分化 孔慶東堅稱「部分港人是狗」] - 香港明報</ref>

=== 批评台湾民主选举 ===
2012年1月下旬，孔庆东在电视评论节目上中嘲讽台湾选举是假民主，说[[马英九|马英九]]在[[2012年中華民國總統選舉|总统选举]]中得票只不过600万是“微弱胜出”，“连半个[[北京|北京]]都不如”；他表示没看见马英九执政四年来台湾有什么进步，他还嘲讽台湾总统大选是人们只顾着在电视上看热闹的“选举[[电视剧|电视剧]]”。[[民主进步党|民主进步党]]稱孔只会讨好[[共产党|共产极权]]，而没有一点学术良心；[[中国国民党|中国国民党]]認為台湾选举获得了世界肯定。一些台湾网友认为“孔狭隘的认知程度令人惊讶，其偏激言论只显示出他一点也不了解台湾民主。”<ref>{{cite web |url = http://www.zaobao.com/zg/zg120128_009.shtml |language= |title=蓝绿同声挞伐孔庆东 |publisher=联合早报 |date=2012-01-28 |accessdate= }}</ref>

=== 力挺薄熙来 ===
2012年3月15日，中共重庆市委书记[[薄熙来|薄熙来]]被免职，孔庆东在电视评论节目上说这是“[[反革命|反革命]][[政变|政变]]”，是“[[汉奸|汉奸]][[卖国贼|卖国贼]]”的做法，倡议为薄鸣不平，质问“你为薄熙来做了什么？”<ref>{{cite web |url = http://www.wenxuecity.com/news/2012/03/16/1682094.html |title = 孔庆东含泪力挺薄熙来：大家都起来和黑暗作斗争 |publisher=文学城 |date=2012-03-16 |accessdate = 2012-03-16}}</ref>

2012年3月20日，第一視頻網的《孔慶東有話說》评论节目突然停播，相关视频被删除。而第一視頻在網站刊出通告稱﹕「近期各地網友反映無法收看直播節目，為滿足廣大網友需求，第一視頻進行直播技術與設備升級，在此期間部分直播節目暫停播出，播出時間另行通知。」該網工作人員向记者證實，暫停播出的只有孔慶東這一個節目。被外界认为与3月15日在该节目中力挺薄熙来煽动大家挺薄的说话所导致<ref>[http://www.chinatown.com.au/news_33913.html 孔慶東挺薄熙來 節目突然停播]</ref>。

=== 辱骂“汉奸”、“被汉奸忽悠的人”===
一次节目中，孔庆东支持地方政府花上百亿修建红色主题公园，对反对修建公园、以纳税人身份质疑的人破口大骂：“我们的老百姓大部分都是被汉奸忽悠的人！你是纳税人，我也是纳税人！你纳税是应该，不纳税给我滚蛋！纳税人怎么了？纳税人就能指挥政府了吗？你他妈是什么玩意，我滚你妈的蛋！”<ref>[http://tv.sohu.com/20120813/n350542608.shtml 孔庆东爆粗口大骂纳税人“滚你妈的蛋”]</ref><ref>[http://www.weibo.com/2230913455/ywYLXmUnG 搜狐视频]</ref>

=== 骂网友“狗汉奸”被判赔200元 ===
2012年5月6日，孔庆东在微博上发表博文《立春过后是立夏》，中国劳动关系学院学生关凯元称文中包括的一首七律“格律不对……好歹孤仄孤平不该犯”，孔庆东回复“你就是个狗汉奸……我看你妈又孤又平”。关凯元遂向法院起诉孔庆东。2013年，海淀法院一审判决孔庆东在一家全国发行的报刊公开道歉，并赔偿精神抚慰金200元、给付公证费1000元。<ref>{{cite web |url = http://news.sina.com.cn/s/2013-05-09/023927066469.shtml |title = 孔庆东骂网友“狗汉奸”被判赔200元 |publisher = 新京报 |date = 2013年5月9日|accessdate = 2013-05-09}}</ref>

=== 講述“六四真相”微博账号被封 ===
2014年5月23日，一名自称是当年参与[[六四事件|六四事件]]镇压的“中央警卫队警卫”在孔庆东的[[新浪微博|新浪微博]]上留言，表示当时政府的做法是为了维护社会稳定而采取的正当举措。孔庆东随即还击：<ref>[https://freeweibo.com/weibo/3713026882029857 孔庆东 六四]，自由微博.</ref><ref>[https://freeweibo.com/weibo/3713028845493988 孔庆东 六四（2）]，自由微博.</ref>

{{cquote|胡说，根本没有暴乱。是你们血腥开枪扫射人民群众后，反过来栽赃陷害！你能举出一个实施暴乱的大学生名字吗？}}
{{cquote|你不了解情况就胡说，你对不起惨死的数百爱国民众，也对不起冤死的那几个解放军战士。}}

孔庆东还即兴透露了当时的其他情况，如“烧军车是不知道身份的人干的，学生还扭送了一些打砸抢分子交给警察”、“4月底美国势力就已渗透进来”、“有学生领袖已被美国收买，一心希望屠杀的发生”、“有领导谎报军情，夸大事态，事实上5月底学运已成强弩之末”、“特务们用墨水玷污毛主席像想嫁祸学生，结果被学生现场抓获扭送派出所”、“5月20日北大学生和北京市民英勇出击，将人民解放军主力野战部队，阻滞于三环四环以外”等。<ref>[https://freeweibo.com/weibo/3712972998278792 孔庆东]，自由微博.</ref>

随后孔庆东的[[新浪微博|新浪微博]]账号被注销关闭，两个月后才恢复功能。

=== 投诉南京电视台《听我韶韶》一审被判败诉 ===
“到底是教授还是野兽？”[[南京电视台|南京电视台]]主持人、名嘴吴晓平在《[[听我韶韶|听我韶韶]]》栏目中，就涉及北京大学教授孔庆东的一起案件进行了评论，之后被孔庆东诉至法院，索赔20万元。2014年12月17日，北京市海淀区人民法院官方微博通报了这起案件，一审驳回孔庆东的诉讼请求。《中青报》12月19日发表文章：“不愿容忍舆论批评的孔庆东败诉了”<ref>[http://media.people.com.cn/n/2014/1219/c40606-26236988.html  中国青年报：不愿容忍舆论批评的孔庆东败诉了]</ref>。
