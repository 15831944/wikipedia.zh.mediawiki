《'''孔雀王'''》是[[荻野真|荻野真]]創作的[[青年漫畫|青年漫畫]]與改編作品，於1985年獲得《第十二屆Jump青年漫畫大賞》。孔雀王本義為[[佛教|佛教]][[二十八部眾|二十八部眾]]的一員金色孔雀王──[[孔雀明王|孔雀明王]]。

== 故事簡介 ==
=== 孔雀王 ===
1985年至1989年在《[[週刊Young_Jump|週刊Young Jump]]》（{{lang|ja|週刊ヤングジャンプ}}）上連載。
一開始主角退魔師'''孔雀'''以清除魔物為主，從嬲樂尼祭祀凶皇佛開始與本作最終敵人六道眾戰鬥。
大聖[[歡喜天|歡喜天]]策劃使孔雀王復活的計畫，黃幡星操控死人倶摩羅，魔神軍荼利策劃的歐魯加與歐康姐弟化成黑暗大日如來的悲劇，墮落黑暗的裏高野退魔師鳳凰，最後孔雀尋找聖杯與終極軍團展開最終的決鬥。以主角出生的秘密為主軸，以全世界的神話為動機，最後成為一部氣勢磅礡的故事。

=== 孔雀王 退魔聖傳 ===
1990年至1992年在《[[週刊Young_Jump|週刊Young Jump]]》上連載，是前作孔雀王的續篇。
畫風稍微有點變化，但是暗示主角是[[素戔嗚尊|素戔]]轉生這點與前作設定產生矛盾（像撒旦是被敵人招喚出的，跟前作鳳凰就是撒旦的化身這點有所矛盾）。可以想作是平行空間的概念。

跟前作一樣開始是退魔小故事，中盤經歷暗黑神特斯加托利帕加的戰鬥，遇到眼球怪物才開始轉移到本作的主軸與[[天神_(日本神話)|天津神]]戰鬥。真相漸漸明朗，以及朋友一個接一個失去的殘酷衝擊；故事到了這邊就突然停止連載，之後只出了3卷的'''怨靈侍'''當作交代。

因為突然停止連載，當時出現各式各樣的猜測。認為是因為將[[日本神話|日本神話]]的神祇當作敵人，描寫神跟人的戰爭引起了宗教團體的憤怒與抗議才停止的，將[[天照大神|天照大神]]與[[月讀尊|月讀尊]]當作是日本古代神祇的'''外國侵略者'''（單行本第9卷），描寫[[古事記|古事記]]與[[日本書紀|日本書紀]]等沒有記載的事情，皆與真相相差太遠，遭到[[右翼|右翼]]團體的抗議才終止的。

=== 孔雀王 曲神紀 ===
在2006年3月28日發售的《[[週刊Young_Jump|週刊Young Jump]]》上開始連載。是前作孔雀王退魔聖傳的續作，為了終止[[天神_(日本神話)|天津神]]與[[國神|國津神]]的鬥爭，尋找「[[素戔嗚尊|素戔]]之牙」。敘述退魔聖傳後半段故事，開始於四年後的世界。

=== 孔雀王起義 ===
《孔雀王RISING》是在[[小學館|小學館]]的《[[月刊Big_Comic_Spirits|月刊Big Comic Spirits]]》2012年5月號上開始連載。描寫少年時期的孔雀，但劇中的設定為了配合連載改以平成年代為背景。

=== 孔雀王 戰國轉生 ===
在{{link-ja|リイド社|リイド社}}的《{{link-ja|漫畫亂雙月刊 戰國武將列傳|コミック乱ツインズ 戦国武将列伝}}》2012年12月號上開始連載。

== 登場人物 ==
=== 孔雀跟朋友 ===
本篇裡的主要人物，記載跟孔雀一起行動的朋友。

; 孔雀
: 全篇的主角，裏高野的退魔師。師傅是慈空阿闍梨，是裏高野第九階的『中僧都』。曾說過本名叫做「明」<ref name="孔雀王，第六集，黃幡星">孔雀王，第六集，黃幡星</ref>，父親曾經是裏高野的退魔師「慈覺」，母親是為了救出在黑暗的人而不斷旅行的[[地藏菩薩|地藏菩薩]]化身<ref name="孔雀王，第十一集，上人鳴動">孔雀王，第十一集，上人鳴動</ref>，雙胞胎姐姐「朋子」（之後的[[天蛇王|天蛇王]]）跟孔雀一樣從出生就有黃幡星宿命的祕密。也是墮天使[[路西法|魯西菲爾]]的投胎轉世（作品一開始就設定[[孔雀明王|孔雀明王]]與[[路西法|魯西菲爾]]是同一位神）。

: 守護神是[[孔雀明王|孔雀明王]]，擁有自在使用其他神佛的法力的才能，非常喜歡吃東西，可以毫不在乎的喝酒，有時也會玩柏青哥不守戒律的和尚。表現很像傻瓜，修行跟退魔經常失敗。對於交通工具沒轍，很容易就暈<ref>孔雀王，第一集，黃泉路</ref>。
: 孔雀王是[[雅茲迪|雅茲迪教派]]的主神。主神是為了救助人類的[[墮天使|墮天使]]Malak Taus（孔雀形象的大天使，文義通「孔雀王」）他们认为墮天使已为神所原谅，大天使地位恢复。因此，雅兹迪教派被视为魔鬼崇拜者。因此孔雀大天使其實是教派對他的尊稱，基督徒及回教徒则认为那是魔鬼。
: 無印結束時約22、3歲（慈空將2、3歲的孔雀帶回後「花20年的時間傳授密法」）。退魔聖傳後半約25歲，曲神紀時應該是20歲後半段，臉卻變得很年輕。

; [[阿修羅|阿修羅]]
: 孔雀想要幫助的流著黑暗之血的少女。守護神是[[阿修羅王|阿修羅王]]，有自由自在操作火炎的能力。
: 在故鄉因為流著魔神[[阿修羅|阿修羅]]的血所以被禁止靠近，生下來時就被幽禁在佛塔中，是解開封印孔雀王的地獄門關鍵，所以被六道眾的[[大聖歓喜天|大聖歓喜天]]帶走。<ref name="孔雀王，第四集，阿修羅">孔雀王，第四集，阿修羅</ref>
: 被大聖洗腦，與孔雀成為敵人，但是被孔雀的善良喚回心智。<ref>孔雀王，第四集，地獄門</ref>
: 從大聖手中救出後，與孔雀跟慈空一起生活，也開始上學，也成為在退魔時的幫手。
: 故事沒有交代阿修羅的身世，不過第十二卷中被吉克弗力德指她「擁有純種亞利安人的眼睛」而遭俘虜。
: 本來是個開朗很有精神的少女，在退魔聖傳中為了賺零用錢而幫人退魔，有現代小孩追隨流行的一面。有著一頭漂亮金髮跟碧眼的美少女，是無印的最終女英雄。

: 初期在連載雜誌設定「登場時冒犯了大聖歡喜天，之後就以性調教方式持續洗腦，直到不敢反抗命令。」不過，單行本收錄時去掉了這個設定，甚至連載雜誌也去掉了。

: 在退魔聖傳結束時被囚禁在異世界裡。救她出來是曲神紀中，孔雀的目標之一。<ref>孔雀王曲神紀，第一集，再相遇…!阿修羅!!</ref>
: 在第一次出現時12歲，無印結束時是15歲。在退魔聖傳結束時大約是17歲。
: 在登場人物中是外表變化最顯著的，每系列都像是不同的人。

; 慈空[[阿闍梨|阿闍梨]]
: 孔雀的師傅。裏高野指導層[[阿闍梨|阿闍梨]]的地位，也是裏高野的退魔師。
: 孔雀父親「慈覺」的師傅，在[[西藏|西藏]]遇到了年幼的孔雀，但這段不祥的記憶被封印了。
: 因為是師傅，所以知識跟靈力都在孔雀之上。
: 平常會不顧阿闍梨的身分在山下的荒廢寺廟過生活，喝酒跟上酒店的壞習慣比孔雀還要嚴重。<ref>孔雀王，第四集，兇皇佛</ref>

: 在退魔聖傳後半遭遇襲擊，現在生死不明。<ref name="孔雀王退魔聖傳，第九集，獵鬼"/>

; 王仁丸
: 收錢幫人呪殺的呪禁道士，能使用式鬼。在事件中與孔雀發生戰鬥，認為孔雀太天真了，但也認同他。<ref>孔雀王，第二集，王仁丸</ref>
: 看起來很冷漠，其實有濃厚人情味，好幾次救出陷入危機的孔雀。守護神是[[大黑天|大黑天]]。
: 擁有強韌的身體跟靈力，是孔雀的好對手。實際是[[鬼|鬼]]與人類女子所生的半人半鬼，因為流著黑暗的血使得故鄉的小村莊被軍荼利消滅。<ref>孔雀王，第九集，軍荼利</ref>

; 黃海峰
: 中國古代流傳的呪術集團『黃家仙道』的仙道士。父親是黃家仙道的領導人，武器是靈劍『獅咬劍』。
: 屍解仙事件與孔雀一同戰鬥，<ref>孔雀王，第七集，黃海峰</ref>之後的軍荼利篇接受父親的命令，為了阻止變成人工孔雀王的歐康而與孔雀作戰，最後為了救出歐康與歐魯加而與孔雀一起向六道眾戰鬥。<ref>孔雀王，第八集，歐康</ref>

: 是個無法隱瞞情感的善良性格，與孔雀的姊姊朋子相愛。<ref>孔雀王，第十三集，黑暗之心</ref>
: 但是沒辦法救出已經被六道眾喚醒天蛇王的朋子，身心都墮入黑暗<ref>孔雀王，第十四集，光明墜落</ref>，取代八葉老師裡[[文殊菩薩|文殊菩薩]]的位置<ref>孔雀王，第十五集，孔雀城崩潰</ref>。
: 無印的最終決戰時阻擋在孔雀面前<ref>孔雀王，第十六集，最後的敵人</ref>，之後為了救出朋子即使自己被消滅也不在乎，成為金剛界曼荼羅四印會的最後一人<ref>孔雀王，第十六集，壽終正寢</ref>。
: 最後救了朋子，並一起生活。<ref name="孔雀王，第十六集，孔雀">孔雀王，第十六集，孔雀</ref>

: 在退魔聖傳沒有登場，卻在「曲神紀」登場。
: 以農夫身分生活著，為了取回被[[伊奘諾尊|伊奘諾尊]]奪走的兒子「明星」，而與孔雀一起行動。
: 老婆是天蛇王，兒子是[[素戔嗚尊|素戔嗚尊]]，令人感到不可思議的家庭。

=== 孔雀有血緣關係的人物 ===
記載與孔雀有血緣關係的人物。

; 朋子
: 與孔雀同樣是黃幡星轉世的雙胞胎姐姐，背負了在站在闇曼荼羅中心的天蛇王宿命的女性。
: 小時候與父親慈覺跟孔雀一起每天都被裏高野的刺客追殺而逃亡，在六道眾首領八葉老師的詭計下，將與父親跟弟弟分離變成孤兒到處流浪時，被吉克弗力德撿回扶養長大。

: 小時候受到驚嚇喪失記憶，過著如同玩偶般的日子，在與阿修羅和黃的相遇之後，溫暖了她的心，並與黃墜入愛河。但是在看到父親慈覺的[[木乃伊|木乃伊]]後，過去不祥的記憶都回來了，真正的天蛇王也覺醒了。<ref name="孔雀王，第十四集，天蛇覺醒">孔雀王，第十四集，天蛇覺醒</ref>

: 最終決戰後與黃一起在[[中國|中國]]生活，還生下了兒子「明星」；卻因為孔雀捲入天津神和國津神的鬥爭中，為保護明星而遭[[伊邪那岐命|伊邪那岐命]]咀咒。黃海峰為保其命，把她送往中國的神界，交託[[伏羲|伏羲]]神照顧。

; 慈覺
: 孔雀與朋子的父親，曾經是慈空的弟子，人稱裏高野最強的退魔師。
: 對日漸強大的黑暗力量感到憂心，進入冥界想讓孔雀王復活，但因孔雀王的魂魄已經消滅而絕望。就在這個時候，愛上救了自己的魔族女人，並生下了黃幡星之子，孔雀與朋子。<ref name="孔雀王，第十一集，阿明與朋子">孔雀王，第十一集，阿明與朋子</ref>

: 跟妻子離別後，一個人持續跟裏高野的刺客戰鬥，就在躲避的旅程中，筋疲力盡時受到八葉老師的挑撥，親手將朋子的心給殺死了。<ref name="孔雀王，第十四集，天蛇覺醒"/>
: 明（孔雀）與重傷的慈覺被尼泊爾山僧所救後，為挽回傷害朋子的罪孽，決定絕食成[[即身佛|即身佛]]，把生命和密法留給明（孔雀）以對抗六道眾和拯救朋子。

; 孔雀與朋子的母親
: 慈覺在冥界相遇的魔族女性，真正身分是想要救出在地獄徘徊的全部亡靈的[[地藏菩薩|地藏菩薩]]化身。<ref name="孔雀王，第十一集，上人鳴動"/>
: 將因為鳳凰的謀略而掉落冥界的孔雀送回人世。

=== 裏高野 ===
以[[弘法大師|弘法大師]][[空海|空海]]為祖師的真言宗（真言密宗）總本山的[[高野山|高野山]]的裏，一般人是不知道他們存在的。
他們以消滅世間一切的魔物為使命。
但是，實際上裏高野是否存在，高野山沒有做承認。
這個問題讓作者跟相關人士被禁止出入高野山。
※裏高野的內容設定是參考真言立川流。

; 日光
: 裏高野的[[座主|座主]]「藥師大醫王」的兒子，擁有第二位權位[[權大僧正|權大僧正]]，是裏高野的年輕領導者。
: 實戰部隊「五輪坊」的首領，小時候跟著父親藥師大醫王消滅黑暗時，被任命為裏高野的最高指揮者。<ref name="孔雀王，第十一集，阿明與朋子"/>
: 因為守護神是最高神[[大日如来|大日如来]]，有著超凡的靈力，在還是小孩子的時候就擁有能將當時裏高野最強的退魔師慈覺打倒的力量。<ref name="孔雀王，第十一集，阿明與朋子"/>
: 為了消滅黑暗而生，對敵人絕不妥協的性格，為了阻止鳳凰的野心即使裏高野整個消失，也要使用被禁止的兵器阿修斗羅。<ref>孔雀王，第十一集，阿修斗羅</ref>
: 天蛇王決戰後成為裏高野的[[座主|座主]]，並重建御山。<ref name="孔雀王，第十六集，孔雀"/>

: 退魔聖傳中裏高野崩壞時，託付給孔雀之後便消失於天之岩戶。<ref name="孔雀王退魔聖傳，第十集，神之座"/>

: 守護神是[[大日如來|大日如來]]是確定的，妹妹月讀是[[月光菩薩|月光菩薩]]，父親是藥師大醫王[[藥師佛|藥師如來]]，應該是參考[[藥師三尊|藥師三尊]]，因此也有認為應該是[[日光菩薩|日光菩薩]]的說法。

; 月讀
: 裏高野女人堂的堂主。裏高野的[[座主|座主]]「藥師大醫王」的女兒，日光的妹妹。
: 一出生就以雙眼盲目，能看到一般人看不到的事情，守護神是[[月光菩薩|月光菩薩]]。<ref>孔雀王，第十集，第六天魔王</ref>
: 與哥哥日光不同，父親藥師大醫王給予他救助黑暗的孩子的使命。<ref name="孔雀王，第十一集，阿明與朋子"/>

: 後面提到鳳凰的母親也叫這個名字，是前代裏高野女人堂的主人。<ref>孔雀王，第十六集，阿修羅神</ref>

: 應該是雙眼盲目，但退魔聖傳在打鬥時，眼睛似乎能看得見。

; 藥師大醫王
: 統治全部裏高野的[[座主|座主]]，是日光跟月讀的父親。
: 黃幡星復活之際，無情的下達要將當時出生可能是「黑暗之子」的小孩全部殺掉的命令。、另一方面覺得拯救世界光是依靠光明的力量是不夠的，給予月讀救出黑暗之子的使命。<ref name="孔雀王，第十一集，阿明與朋子"/>

: 裏高野因為鳳凰的關係崩壞時，將未來託付給日光後，為了封印冥界之門而變成[[木乃伊|木乃伊]]。

: 守護神在作品中並沒有明白說出，但以大醫王的名稱與兩個小孩分別代表太陽與月亮，認為以[[藥師三尊|藥師三尊]]的中心[[藥師佛|藥師如來]]的說法比較有力（大醫王是藥師佛的別名）。

: 之後，在退魔聖傳也有登場，雖然是無印後已經過了幾年但依然還活著，個性卻是只重視自己生命跟成規的[[利己主義|利己主義]]者，最後被孔雀化成灰。<ref name="孔雀王退魔聖傳，第九集，黑暗的後高野">孔雀王退魔聖傳，第九集，黑暗的後高野</ref>

; 嵐
: 裏高野的實戰部隊「五輪坊」的「風」部隊頭領，是個女扮男裝的[[比丘尼|比丘尼]]。守護神是風天神。
: 以稱作『紫雲絲』的鋼線當做武器，有將軍荼利的軍團全滅的能力。<ref name="孔雀王，第八集，崑崙">孔雀王，第八集，崑崙</ref>
: 與同樣是五輪坊頭領的雷火一同追殺把部下殺死的歐康。

: 實際上是軍荼利所製造的黑暗之子的一員，背上有六道眾的紋章。<ref name="孔雀王，第八集，崑崙"/>

; 雷火
: 裏高野的實戰部隊「五輪坊」的「火」部隊頭領。
: 與嵐一起追殺把部下殺死的歐康，在登上[[香巴拉|香巴拉]]時被歐魯加殺掉。<ref>孔雀王，第九集，天蛇王</ref>

; 慈海
: 慈空的師傅，超過140歲，是裏高野唯一的[[阿闍梨|大阿闍梨]]。擁有使天蛇王復活的關鍵[[聖杯|聖杯]]。<ref name="孔雀王，第十二集，死靈呪術師">孔雀王，第十二集，死靈呪術師</ref>
: 在出羽龍山寺進行斷絕五穀的千日修行，遭受終極軍團的襲擊而將[[聖杯|聖杯]]託付給孔雀。<ref>孔雀王，第十二集，聖杯與聖槍</ref>

: 在頻死時使出驚人的靈力，以九字發勁將終極軍團的大軍消滅，但被八葉化身的卡爾·豪斯柯華打敗。<ref>孔雀王，第十二集，襲擊聖杯</ref>

; [[鳳凰|鳳凰]]
:父母是被逐出裏高野的退魔師和前代月讀，後來弒父出走被認是[[撒旦|撒旦]]的化身而憤世妒俗，並召喚出織田信長。被孔雀擊進地獄後獲納粹第四帝國所救，並加盟六道眾；最終之戰中曾奪取孔雀之肉身及力量，卻因不能承受眾佛之力而爆亡，臨終前體會母親的關愛而化解對自身不幸命運的仇恨，大徹大悟後[[涅槃|涅槃]]往生。孔雀從中覺悟世間沒有化解不了的仇恨而重生。

=== 六道眾 ===
想在世界建立六道魔界，持有蛇的紋章的黑暗密法集團。是裏高野的宿敵，瞄準流著孔雀王的血的孔雀。

; 大聖歡喜天
: 六道眾裡的象頭魔人，原型是印度教的[[象頭神|象頭神甘涅沙]]，利用阿修羅將封印的地獄門打開，企圖使沉睡的孔雀王甦醒。
: 使用能操作凍氣的密法，能將裏高野最強的實戰部隊藥師十二神將消滅，敗給了被孔雀打開心靈的阿修羅。<ref name="孔雀王，第四集，阿修羅"/>

; 俱摩羅
: 崇拜孔雀王的教團「光翼教團」始祖，自稱是孔雀王的化身。俱摩羅天在佛經中是指初禪天的梵王，座騎亦是孔雀。<ref>孔雀王，第五集，光翼教會</ref>
: 黃幡星之子，十幾年前差點被裏高野殺害，幸好被夜叉王救出<ref name="孔雀王，第五集，俱摩羅">孔雀王，第五集，俱摩羅</ref>，被六道眾認為是孔雀王轉世。
: 他在光翼除了使用操控死人之術光翼，跟孔雀一樣可以使用孔雀明王呪，還曾經一度打敗孔雀。<ref>孔雀王，第五集，在黃幡星之下</ref>

: 實際上是個死人，他的命在十幾年前就被慈空奪走，因為小時候被裏高野殺害的怨念，成為被黃幡星操控的人偶。<ref name="孔雀王，第六集，黃幡星"/>

; 夜叉王
: 俱摩羅的僕人，原是裏高野五大明王之一的密法者。守護神是[[金剛夜叉明王|金剛夜叉明王]]。<ref>孔雀王，第五集，金剛夜叉王</ref>
: 反對裏高野將可能是黃幡星轉世之子集中殺掉而帶著年幼的俱摩羅逃走，死在慈空手上，被復活以後就跟隨著俱摩羅身邊。<ref>孔雀王，第六集，夜叉之心</ref>

: 雖然是個死人，但是卻沒有失去人的心智，救了被俱摩羅打敗的孔雀，最後將一切託付給孔雀後死去。

; 羅我
: 俱摩羅隨側的女子，真正身分是六道眾的[[愛染明王|愛染明王]]能以密法叫出「天弓」，有消滅五輪坊「地」部隊的能力。<ref>孔雀王，第六集，羅我</ref>
: 深信俱摩羅就是孔雀王轉世，認為當俱摩羅成為孔雀王時自己將成為八葉的一人，被王仁丸與慈空打敗。<ref>孔雀王，第六集，夜叉王</ref>

; 軍荼利
: 六道眾八大明王的一員，以[[俄羅斯帝國|俄羅斯帝國]]末期的[[格里高利·葉菲莫維奇·拉斯普京|古瑞可利·拉斯普欽]]在人間活動，自稱神人。收集全世界的黑暗之血到[[俄羅斯|俄羅斯]]冰凍的土地上，想以人工的方式使孔雀王誕生。<ref>孔雀王，第七集，神人</ref>
: 崑崙篇裡使出坤達利尼的[[查克拉|恰庫拉]]威脅到嵐與王仁丸，但不知道自己是人造魔神而承受不住強大的[[查克拉|恰庫拉]]因此自我毀滅。<ref>孔雀王。第九集，最後的恰庫拉</ref>

; 吉克弗力德‧米特加爾多
: 納粹第四帝國之統帥，真身為配上機械身體與希特拉腦袋的人造人，統轄一班早由納粹時代以東西方魔法製造，配備黨衛軍裝備的亡靈軍團，企圖以藏於裏高野的基督聖杯（頭骨）之神力復興納粹，因而聯合死靈咒術師卡爾‧豪斯柯華追捕孔雀和朋子，以奪其血。詎料被覺醒為天蛇王的朋子轟散。
: 手上的P38手鎗由日耳曼神話英雄吉克孟多以北歐神王奧丁的神劍打造，鎗身刻上曼荼羅圖案，配上施了魔法的子彈，便可射滅神魔咒法。

; 卡爾‧豪斯柯華
: 六道眾幹部，真身同為機械。曾以死靈咒術師的身分唆使希特拉沉迷黑魔法，並指使日本關東軍及綠龍會召集全日本宗派，到處找尋基督聖杯（頭骨）。

; 黑暗曼荼羅的魔神們
: 因為朋子的覺醒，跟隨著黑暗的盟主天蛇王的數百個魔神<ref>孔雀王，第十三集，黑暗曼荼羅</ref>。對黑暗世界極渴望。每一個都是強大的神，有毀滅中國好幾個軍團跟戰車的實力<ref>孔雀王，第十五集，666之獸</ref>。

; 八葉老師
: 有'''「黑暗的管理者」'''之稱，掌管六道眾謎一樣的八位魔神<ref>孔雀王，第十集，八葉老師</ref>。以密宗胎藏界[[曼荼羅|曼荼羅]]裡中台八葉院的八佛（寶幢、開敷華王、無量壽、天鼓雷音等四如來、以及普賢、文殊、觀自在、慈氏等四菩薩）的姿態出現，企圖從朋子和孔雀中挑選一人為魔界大日如來。
: 跟宇宙一起誕生的不死神祇，即使是孔雀王的力量也無法打倒，為了建立六道魔界向孔雀派出各種刺客。
: 其中一員的文殊菩薩被[[聖槍|聖槍]]打倒<ref>孔雀王，第十四集，決裂</ref>，空位由黃海峰補上，最後因為黃海峰背叛，令八葉之胎藏界[[曼荼羅|曼荼羅]]陣崩壞，被孔雀、阿修羅、王仁丸、慈空等組成的密宗金剛界[[曼荼羅|曼荼羅]]四印會之陣消滅。
: 在黑暗的大日如來篇將六道眾說成是源自遙遠的太古時代飛來地球的高度機械文明外星人，之後演變成神話與宗教。<ref>孔雀王，第九集，西王母</ref>

=== 「退魔聖傳」、「曲神紀」主要登場人物 ===
; 服部遮那
: 退魔聖傳登場。父親擁有[[能樂|能樂]]宗家，是活著就可以來往冥界的幽玄之子。<ref name="孔雀王退魔聖傳，第一集，幽玄之子">孔雀王退魔聖傳，第一集，幽玄之子</ref>
: 是唯一能拿操控鬼的鬼一法眼刀的人類，被想要利用鬼一法眼刀的鬼人鬼道會「蛾王」誘拐走。<ref name="孔雀王退魔聖傳，第一集，幽玄之子"/>
: 因為被[[源義經|源義經]]的惡靈附身<ref>孔雀王退魔聖傳，第二集，邪道</ref>，義經邪惡的意念將京都的鬼門打開，孔雀們阻止魔界怪物將地上變成地獄。因退魔而救出。
: 因為這次相遇，被暫時託付在慈空的寺廟。
: 在寺廟時被阿修羅當做雜役使喚，幫缺少零用錢的阿修羅偽裝成孔雀進行退魔的工作。<ref>孔雀王退魔聖傳，第三集，福瘤觀音</ref>
: 之後，為了能樂師的修行而道別，6年後在曲神紀與孔雀再相遇，現在跟孔雀一行人尋找牙齒。
: 名字在退魔聖傳一開始時就知道了。

; 蓮華
: 退魔聖傳登場。使用[[戰輪|戰輪]]的裏高野女人堂的女退魔師。<ref name="孔雀王退魔聖傳，第四集，黑皮貴婦">孔雀王退魔聖傳，第四集，黑皮貴婦</ref>
: 守護神是[[大勢至菩薩|勢至菩薩]]，使用護符就可以使用神的力量。
: 自大又頑強的性格受到裏高野反彈，被裏高野趕出去。<ref name="孔雀王退魔聖傳，第四集，黑皮貴婦"/>
: 但是，在吸血鬼[[吉爾·德·萊斯|吉爾·德·雷]]事件，學會相信朋友跟戰士的心。<ref>孔雀王退魔聖傳，第六集，戰士</ref>
: 返回裏高野繼續修行，月讀大人奪取裏高野的時候被殺害。<ref name="孔雀王退魔聖傳，第九集，天之岩戶(前篇)">孔雀王退魔聖傳，第九集，天之岩戶(前篇)</ref>
: 心臟被抽出，成為隨著[[天神_(日本神話)|天津神]]意思行動的木偶，神的舞蹈巫女[[天鈿女命|天宇受賣命]]。<ref name="孔雀王退魔聖傳，第十集，開啟岩戶">孔雀王退魔聖傳，第十集，開啟岩戶</ref>

; 月讀大人（月蛾）
: 退魔聖傳登場。天津最高神的一員，實質上的[[天神_(日本神話)|天津神]]首領。與月讀相同名稱與髮型，性別為男性。
: 為了讓長眠於裏高野的[[天照大神|天照大神]]復活而奪取裏高野。
: 收集三種神器，在快要成功打開岩戶叫醒沉睡的天照大神時，因為日光的策略使得天照大神復活的計畫失敗了。<ref name="孔雀王退魔聖傳，第十集，魔鏡">孔雀王退魔聖傳，第十集，魔鏡</ref>
: 而且，臉因為中了孔雀的術法而受傷，之後都戴著面具。<ref name="孔雀王退魔聖傳，第十集，神之座">孔雀王退魔聖傳，第十集，神之座</ref>
: 天照大神復活失敗後，、帶著成為[[天鈿女命|天宇受賣命]]的蓮華將各地的天津神叫醒。<ref name="孔雀王退魔聖傳，第十集，出羽奧岳山">孔雀王退魔聖傳，第十集，出羽奧岳山</ref>
: 到了曲神紀，月讀的念法從「ツ'''ク'''ヨミ」（tsukuyomi）變成「ツ'''キ'''ヨミ」（tsukiyomi），另外知道了名子叫做'''月蛾'''，能夠控制[[蛾|蛾]]。
: 在退魔聖傳能唱出[[月光菩薩|月光菩薩]]真言，使用冰與冷氣之術，曲神紀裡被加上了蛾的外貌，以及能使用蛾的能力。

; [[天鈿女命|天宇受賣命]]
: 上述的蓮華被月讀大人賜與了新生命後的新身分，專門跳[[神樂|神樂]]舞叫醒沉睡的[[天神_(日本神話)|天津神]]們。<ref name="孔雀王退魔聖傳，第十集，出羽奧岳山"/>
: 蓮華的記憶與人格都消失了，是月讀大人忠實的僕人。

; [[思兼神|思兼神]]
: 退魔聖傳登場。在世界一開始就存在的[[別天神|別天津神]]，[[天神_(日本神話)|天津神]]裡的智慧之神。<ref name="孔雀王退魔聖傳，第八集，思兼神">孔雀王退魔聖傳，第八集，思兼神</ref>
: 目魂事件的幕後主使，也是最早登場的天津神。<ref name="孔雀王退魔聖傳，第八集，思兼神"/>
: 精通所有法術被稱為智慧之神，能使出比孔雀更強的密法。
: 為了守護[[天照大神|天照大神]]，騙[[空海|空海]]建造裏高野。<ref name="孔雀王退魔聖傳，第九集，黑暗的後高野"/>
: 真面目是個只有[[腦|腦]]跟[[眼|眼珠]]的怪物，幾千年間要持續生存就必須一直更換肉體。<ref name="孔雀王退魔聖傳，第八集，神的真相">孔雀王退魔聖傳，第八集，神的真相</ref>
: 奪取裏高野時附在月讀的身體上，最後被日光把腦給打個粉碎。<ref name="孔雀王退魔聖傳，第十集，神之座"/>

; [[少彥名|少彥名]]神
: 退魔聖傳登場。在世界一開始就存在的[[別天神|別天津神]]，[[國神|國津神]]裡的智慧之神。
: 有像蛾一樣的翅膀，個子非常小隻，頭卻非常大。身體可以隱形。<ref name="孔雀王退魔聖傳，第十一集，踏上封神之路">孔雀王退魔聖傳，第十一集，踏上封神之路</ref>
: 因為接受其他妖魔的請託，得知他想要製造可以前往他界的神馬，卻不斷製造出妖魔。<ref>孔雀王退魔聖傳，第十一集，少彥名神</ref>
: 知道孔雀是被[[素戔嗚尊|素戔]]之牙選擇的事情後，了解孔雀的目的是將世界上所有的神消滅，因此想要親眼看到神最後的下場如何，跟著孔雀一起尋找[[素戔嗚尊|素戔]]之牙。<ref name="孔雀王退魔聖傳，第十一集，踏上封神之路"/>
: 愛喝酒到無可救藥的程度，會為了買酒錢在別人家裡翻箱倒櫃找錢。

; 宣導涼子
: 退魔聖傳登場。因為向孔雀請求幫助，開啟了「天津神篇」漫長的故事。<ref>孔雀王退魔聖傳，第六集，黑靈</ref>
: 被人犬襲擊咬死<ref name="孔雀王退魔聖傳，第七集，人犬">孔雀王退魔聖傳，第七集，人犬</ref>，被國津神的建御名方帶回鬼部村。<ref name="孔雀王退魔聖傳，第七集，鬼戰">孔雀王退魔聖傳，第七集，鬼戰</ref>
: 擁有使[[素戔嗚尊|素戔]]復活的關鍵<ref name="孔雀王退魔聖傳，第七集，鬼戰"/>，現在生死行蹤都不明。

; 建御名方
: 退魔聖傳登場。
: 留著[[素戔嗚尊|素戔]]血液的直系子孫，現在國津神之主。打算從天津神手上奪回國家。<ref name="孔雀王退魔聖傳，第八集，神的真相"/>
: 國津神現在最強的戰士，能變成巨大的鬼怪。<ref name="孔雀王退魔聖傳，第七集，鬼戰"/>
: 手腳能自由分離做出攻擊<ref name="孔雀王退魔聖傳，第七集，鬼戰"/>，即使是人的狀態也能使出這種能力<ref name="孔雀王退魔聖傳，第八集，神的真相"/>。

; 下照比命
: 建御名方同母異父的妹妹。退魔聖傳就就登場了，但名稱跟設定到曲神紀才比較明朗。
: 能變成貓型的獸人，善用她優秀的敏捷度做攻擊。
: 在月讀大人的手下，曲神手中被孔雀救出，之後就跟孔雀一起行動。
: 名子為下照比命，但是卻讓孔雀們叫她「織姬」。
: 蠻不在乎的個性習慣以「～喵」為口頭禪，喜歡看愛情電視劇跟亂買手機。

; 大咬太夫
: 退魔聖傳登場。目魂一黨其中一名暗殺者，現在犬神統的下一任統主。
: 固定穿黑色西裝打扮的男子，真面目是稱為人犬的獸人。<ref name="孔雀王退魔聖傳，第七集，人犬"/>
: 人犬是在太古時代天津神侵略的時候背叛國津神的鬼之一族，雖然被稱作是'''眷屬''，但實際上比較接近天津神的奴隸。<ref>孔雀王，第十集，人犬的野心</ref>
: 在滿月之夜因為月的靈力變身成不死身的[[大口真神|大口真神]]。<ref name="孔雀王退魔聖傳，第七集，人犬"/>

; 十六夜太夫
: 退魔聖傳登場。大咬太夫的妹妹。
: 曲神紀再次登場，樣子跟性格都與退魔聖傳時有很大的改變。

; 荒川
: 曲神紀登場。年邁的游泳教師。

; 伊邪那岐神
: 創造日本列島的神。

; [[須佐之男|須佐之男]]
: 古代最強神，國津神與鬼的祖先。
: 詳細情況還不清楚，但是鬼的樣子連天照大神都會害怕到想要躲起來<ref name="孔雀王退魔聖傳，第十集，魔鏡"/>。但是，最後還是被天津神殺掉，牙齒卻繼承了能力，當牙齒全都收集到時就會獲得素戔的力量，能夠將所有的神都消滅<ref name="孔雀王退魔聖傳，第十一集，踏上封神之路"/>。

; 黃明星
: 曲神紀登場。黃與朋子的兒子。神仙轉世，被當作是[[須佐之男|須佐之男]]的化身。
: 年齡大約6、7歲左右。

; 王仁丸六角
: 惡名昭彰的呪禁道家王仁丸一家的么弟（王仁丸的弟弟）
: 聽從哥哥王仁丸的指示，跟孔雀通知危險並一起共同行動。
: 相對於長男王仁丸能使用「式神=妖怪・俗神」，六角能使役「死鬼神=死靈」。
: 曲神紀第六集提到王仁丸兄弟一共有6人，「怨靈侍」的六角跟王仁丸六角是同一個人。
