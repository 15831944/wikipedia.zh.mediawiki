'''孔劉'''（{{lang-ko|'''공유'''}}，{{bd|1979年|7月10日|catIdx=Gong Yoo}}），本名'''孔地哲'''（{{lang-ko|공지철}}），[[韓國|韓國]][[男演員|男演員]]、[[模特兒|模特兒]]，[[本貫|本貫]][[曲阜孔氏_(朝鮮半島)|曲阜孔氏]]（訪問中曾透露自己為中國古代著名教育家和思想家、[[儒家|儒家]]创始人[[孔子|孔子]]的後代<ref>{{cite news|url=http://ent.ltn.com.tw/news/breakingnews/2054966|title=（專訪／影音）孔劉親口認證是孔子後代！與她的關係是…|work=自由時報|date=2017-05-03|language=zh-tw}}</ref>），藝名「孔劉」取自父母姓氏，常被音譯為「'''孔侑'''」。因頭形較小，常有「대가리（小豆）」的別號，自2001年以《學校4》出道後，影迷們亦以「꽁（GGong）」作稱呼。 於2007年出演MBC電視劇《[[咖啡王子1號店|咖啡王子1號店]]》開始走红，2016年底，以作品《[[孤單又燦爛的神－鬼怪|孤單又燦爛的神－鬼怪]]》再創事業巔峰。2017年，獲得[[第53屆百想藝術大賞|第53屆百想藝術大賞]]電視部門男子最優秀演技賞。
[[File:LG전자,_스마트폰_‘옵티머스’_모델로_‘공유’_영입.jpg|thumb]]宣傳畫報]]

== 個人簡介 ==
孔劉早期以優秀的外型出任服裝模特兒身高184公分，在大學初年考入[[Mnet_Media|Mnet]]，任職VJ。2001年以[[韓國放送公社|KBS]]電視劇《{{link-ko|學校4|학교 4 (드라마)}}》出道，2003年憑藉《[[二十歲_(電視劇)|二十歲]]》獲[[SBS演技大賞|SBS演技大賞]]新人獎，2005年因在《[[餅乾老師星星糖|餅乾老師星星糖]]》中的演出備受歡迎。2007年出演《[[咖啡王子1號店|咖啡王子1號店]]》獲[[MBC演技大賞|MBC演技大賞]]男子優秀演技賞。 

2008年1月14日入伍，期間曾於[[大韓民國|大韓民國]]國軍廣播主持《孔劉等待的20時》，以網路向全球廣播，創下不亞於同時段商業節目的收聽率。晉升兵長時，孔劉獲上級致贈小說《熔爐》作為禮物，孔劉閱讀後深受震撼，退伍後即與經紀公司奔走籌資，竭力促成電影《[[熔爐_(電影)|熔爐]]》的誕生。2011年，電影《熔爐》上映，孔劉演繹姜仁浩一角。電影內容揭發兒童性侵問題，韓國光州特教學校性暴力事件受到大眾重視，產生「熔爐效應」，並促使政府修定「熔爐法」法例。雖為限制級電影，但票房成績超過467萬觀影人次，使孔劉的演藝生涯再創高峰<ref>{{Cite web |url=https://www.chinatimes.com/realtimenews/20161110006395-260404 |title=熔爐劇情真實殘酷 孔劉入戲痛苦4個月 - 娛樂 |accessdate=2019-04-21 |last=中時電子報 |work=中時電子報 |language=zh-TW |archiveurl=https://archive.is/3khED |archivedate=2019-04-21}}</ref>。

2016年，主演《[[男與女_(電影)|關不住的誘惑]]》、《[[屍速列車|屍速列車]]》、《[[密探|密探]]》等多部電影作品，其中《密探》於韓國達到750萬觀影人次的優異票房，《屍速列車》更是創下該年度唯一突破千萬觀影人次票房紀錄，同時亦於亞洲多國取得佳績，孔劉並因此晉升「千萬演員」的行列<ref>[https://udn.com/news/story/8/1939170 「屍速列車」3天狂掃6千萬 制霸韓影在台記錄！] 聯合報{{zh}}</ref>。同年，主演韓國有線電視台[[TVN|tvN]]電視劇《[[孤單又燦爛的神－鬼怪|孤單又燦爛的神－鬼怪]]》，飾演男主角鬼怪金信。該劇播出後旋即獲得熱烈迴響，孔劉連續七週奪得電視劇演員話題性冠軍。根據韓國企業評判所公布的廣告模特評價，孔劉2016年（6月第13名、7月第8名、8月第1名、10月第5名、12月第1名），2017年（1月第1名、2月第1名、3月第1名、4月第1名、5月第2名、6月第2名、7月第3名），連續五個月奪得冠軍。孔劉也憑藉金信一角，獲得[[第53屆百想藝術大賞|第53屆百想藝術大賞]]電視部門男子最優秀演技賞<ref>{{Cite web|url=http://www.nownews.com/n/2017/05/03/2507320|title=53屆百想藝術大賞完整獲獎名單　孔劉、朴寶劍不負眾望|accessdate=2017-05-05|author=NOW今日新聞|date=2017-05-03|publisher=}}</ref>。

== 社會活動 ==
'''[[聯合國兒童基金會|聯合國兒童基金會]]兒童權利特別代表'''

2013年11月20日，孔劉被聯合國兒童基金會韓國委員會委任為兒童權利特別代表；期望孔劉積極參與並竭力促進兒童權利。

'''[[大韓民國國稅廳|大韓民國國稅廳]]宣傳大使'''

2014年3月3日，孔劉于韓國第48屆納稅人日紀念儀式上，被選定為模範納稅人，獲得總統表彰；2014年7月7日，被正式委任為韓國國稅廳宣傳大使，致力傳播誠實納稅文化。
