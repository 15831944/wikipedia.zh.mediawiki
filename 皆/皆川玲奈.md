'''皆川 玲奈'''（{{jpn|kn=みながわ れいな|rm=Minagawa Reina|f=h}}，{{bd|1991年|6月30日|||minagawa}}），日本[[東京都|東京都]]出身的[[TBS電視台|TBS電視台]]女性[[播报员|播报员]]。原[[奧斯卡傳播|奧斯卡傳播]]经纪公司所属艺人，旧艺名为'''星川 玲奈'''（{{jpn|kn=ほしかわ れいな|rm=Hoshikawa Reina}}）。身高162cm，血型为[[ABO式血液型|A型]]<ref name="tbs2014">{{Cite news|url = http://www.sponichi.co.jp/entertainment/news/2014/10/03/kiji/K20141003009037990.html|title = TBS初! 新人アナウンサー3人全員がレギュラーデビュー|newspaper = [[体育日本|スポーツニッポン]]|date = 2014-10-03|accessdate = 2015-08-04}}</ref>。

== 经历 ==
在小学生时代的[[2003年|2003年]]获得“第9回[[全日本國民美少女比賽|全日本國民美少女比賽]]”审查员特别奖<ref name="news23" />，因而被[[奧斯卡傳播|奧斯卡傳播]]发掘，以「星川玲奈」作为艺名出道。[[2005年|2005年]]被[[Seventeen_(雜誌)|Seventeen]]杂志选为专属模特<ref name="tbs2014" /><ref name="news23">{{Cite news|title = 皆川玲奈アナ 国民的美少女、レーサー、そしてNEWS23抜てき! |newspaper = スポーツニッポン|date = 2016-03-26|url = http://www.sponichi.co.jp/entertainment/news/2016/03/26/kiji/K20160326012284830.html|accessdate = 2016-03-26}}</ref>，2008年毕业。之后，在[[青山学院大学|青山学院大学]]学习期间以本名「皆川玲奈」移籍到{{tsl|ja|キューブ (芸能プロダクション)|CUBE (经纪公司)|CUBE}}经纪公司，出演过一些[[电视剧|电视剧]]<ref name="cube">{{Cite web|url = http://cubeinc.co.jp/members/prf/106.html|title = 皆川 玲奈 （MINAGAWA REINA）|publisher =キューブ|accessdate = 2016-06-15|archiveurl =https://web.archive.org/web/20111010011932/http://cubeinc.co.jp/members/prf/106.html |archivedate = 2011-10-10}}</ref>。

在青山学院大学就学期间，她还参加了体育会汽车部社团，考取了国内A级[[驾照|驾照]]，出场过多次赛车比赛<ref name="news23" />，曾获得日本女子团体赛冠军<ref>{{cite web |title=クルマは相棒、大学自動車部では優勝も　皆川玲奈アナ |url=https://web.archive.org/web/20180326124047/https://www.asahi.com/articles/ASKDT5WB8KDTUPQJ009.html |publisher=[[朝日新聞社|朝日新聞デジタル]] |date=2018-01-12}}</ref>。

[[2014年|2014年]]4月加入[[TBS電視台|TBS電視台]]实习。在[[7月19日|7月19日]]到[[8月31日|8月31日]]间与同期入社的同事进行了街头宣传活动<ref>{{Cite news|title = 枡田絵理奈アナ、「夏サカス街コン」に「社員も参加していいの?」|newspaper = 映画.com ニュース|date = 2014-06-11|url = http://eiga.com/news/20140611/9/|accessdate = 2014-12-27}}</ref>。

2016年3月，前辈播报员[[小林悠_(主播)|小林悠]]因身体原因离职，她便接替其位置，担任『[[NEWS23|NEWS23]]』等节目主播<ref name="news23" />。
