'''创维集团有限公司'''（英文'''Skyworth'''，简称'''创维'''）创立于1988年，总部设立于[[中国|中国]][[广东省|广东省]][[深圳市|深圳市]]高新科技园区，是一家跨越粤港两地，生产消费类电子、网络及通讯产品的大型高科技上市公司。'''香港创维数码控股有限公司'''于2000年在香港主板上市{{sehk|0751}}，目前是创维集团的控股公司。目前创维是中国三大彩电龙头企业之一<ref name="三大">{{Cite web |url = http://www.gd.xinhuanet.com/newscenter/ztbd/2008-12/04/content_14969198.htm |title = 创维集团：中国三大彩电龙头企业之一 |date = 2008年12月4日 |language = zh-hans |accessdate = 2015-02-08 |deadurl = yes |archiveurl = https://web.archive.org/web/20140811024223/http://www.gd.xinhuanet.com/newscenter/ztbd/2008-12/04/content_14969198.htm |archivedate = 2014年8月11日 }}</ref>
，同时也是世界十大彩电品牌，特別是室外大型顯示屏，是中国显示行业领导品牌和中国电子百强第十三位<ref name="十大">{{Cite web |url = http://www.chinabgao.com/stat/stats/34732.html |title = 2014年液晶彩电十大品牌排行榜-中国报告大厅 |date = 2014年7月11日 |language = zh-hans |accessdate = 2015-02-08 }}</ref>。此外還有照明燈具事業。

== 历史 ==
1988年，创维在香港成立，初期名称为“创维实业有限公司”。

1990年，创维实业有限公司正式进入制造业，主要生产电视遥控器。

1992年，“创维集团”于[[香港|香港]]成立，主要生产彩电、VCD、DVD，家庭影院和卫星接收机等产品。

1994年，创维与深圳RGB电子公司成功合资，成立“创维-RGB电子公司”。

1996年，“创维电子城”在深圳宝安区建成投产

2000年4月，“创维数码控股有限公司”在香港主板上市，集资数十亿港元。

2002年，创维投入使用位于深圳市高新科技园区的“创维（深圳）数字研究中心”。

2003年，“SKYWORTH创维”荣获广东省著名商标称号，董事局主席[[黄宏生|黄宏生]]当选中华人民共和国[[全国政协|全国政协]]委员及[[深圳民营企业家商会|深圳民营企业家商会]]会长，同年，创维集团总部迁入位于广东深圳高新科技园新落成的创维大厦。

2006年，启用全新企业标识。

2009年，创维与第26届世界大学生运动会组委会执行局签约，成为中国2011年深圳第26届世界大学生夏季运动会的赞助商。<ref name="事记">{{Cite web |url = http://info.homea.hc360.com/2012/09/200949912406-all.shtml |title = 回顾中国家电企业：创维走过的24年历史 |date = 2012年9月20日 |language = zh-hans |accessdate = 2015-02-08 }}</ref>

2014年9月，创维集团下属的创维数字股份有限公司在[[深交所|深交所]][[借壳|借壳]]“华润锦华”上市，并更改股票简称为“创维数字”。<ref>[http://news.hea.cn/2014/0926/219620.shtml 机顶盒方兴未艾 创维数字借壳今日上市]</ref>

截至目前，创维集团旗下设深圳创维-RGB电子公司、海外发展公司、数字技术公司、电器科技公司、群欣安防科技公司、液晶器件公司等十多家产业公司。拥有成都、南京、宜春三大物流中心；深圳石岩、深圳龙岗、广州、南京、内蒙古呼和浩特等生产基地；香港、深圳、北京、广州、南京等科研机构。<ref name="成员">{{Cite web |url = http://www.skyworth.com/cn/aboutus-default-20.html |title = 集团成员-创维集团 |date = 2014年7月11日 |language = zh-hans |accessdate = 2015-02-08 }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 前董事长黄宏生入狱事件 ==
創維创始人兼前主席是身兼[[全國政協|全國政協]]委員的[[黃宏生|黃宏生]]，其胞弟是[[黃培昇|黃培昇]]。 <ref>{{Cite web|url=http://hk.news.yahoo.com/060713/12/1q2x2.html |title=創維兄弟偷公款重囚6年 蔣震鄭慕智政協副主席求情不果 |date=2006年7月14日 |language=zh-hk |accessdate=2015-02-08 |deadurl=bot: unknown |archiveurl=https://web.archive.org/web/20060717195454/http://hk.news.yahoo.com/060713/12/1q2x2.html |archivedate=2006年7月17日 }}</ref>2005年，黄宏生被控于2001年1月至6月间，串谋其母罗玉英，从创维数码全资子公司创维集团有限公司（现更名创维电视控股有限公司）账户中盗窃公司50万港元。另外，黄宏生兄弟被控于2000年11月至2004年10月间，与罗玉英及王鹏串谋，以“佣金”名义分别窃取上市公司资金221万港元和4838万港元。2006年7月13日，黃宏生及其胞弟、創維前執行董事黃培昇因串謀盜竊及串謀詐騙創維數碼5000多萬港元被判[[監禁|監禁]]6年。其出狱后蜕变为懂得授权、善于分享、热衷激励的“隐形船长”，在2010年出狱首年，在全球彩电产量萎缩的背景下，创维营收却大涨34%；他更是通过分拆上市的方式（拟将一家上市公司拆为三家），给团队内部创业的机会。2013年8月6日，外界猜测良久的创维机顶盒业务分拆上市方案出炉 <ref>{{Cite web | url = http://finance.qq.com/a/20131114/009344.htm#p=20 | title=大败局的复活密码 出狱企业家能否东山再起_财经_腾讯网 |date = 2013年11月14日 |language = zh-hans |accessdate = 2015-02-08}}</ref>。
