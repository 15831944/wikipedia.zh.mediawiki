'''應仁'''是[[日本|日本]][[年號|年號]]之一，在[[文正|文正]]之後、[[文明_(年號)|文明]]之前，指的是1467年到1468年這段期間，此時期的天皇是[[後土御門天皇|後土御門天皇]]，[[室町幕府|室町幕府]]的將軍是[[足利義政|足利義政]]。

== 改元 ==
*文正二年三月五日（1467年4月9日）因為戰亂開始而改元。
*應仁三年四月二十八日（1469年6月8日）改元為文明。

== 出典 ==
出自『[[維城典訓|維城典訓]]』的「仁之感物，物之應仁，若影隨形、猶聲致響」之句。

== 應仁年間要事 ==
*應仁元年一月十七日（1467年2月21日）：[[畠山義就|畠山義就]]襲擊在[[上御靈社|上御靈社]]的[[畠山政長|畠山政長]]，為[[應仁之亂|應仁之亂]]開始的發端，此亂事一直到[[文明（日本年號）|文明]]九年十一月二十日（1477年12月25日）才宣告結束。
*應仁元年五月二十日（1467年6月21日）：應仁之亂，[[山名持豐|山名持豐]]與畠山義就舉兵。
