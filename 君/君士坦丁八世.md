'''君士坦丁八世'''（{{lang-el|Κωνσταντίνος Η΄}}，960年－1028年11月15日），為[[東羅馬帝國|東羅馬帝國]][[馬其頓王朝|馬其頓王朝]]的[[拜占庭皇帝列表|皇帝]]，在位期間為西元1025年－1028年。
[[File:1025AD_LA.svg|thumb]]

== 早年事迹 ==
君士坦丁八世是拜占廷皇帝[[羅曼努斯二世|羅曼努斯二世]]之子，其母[[狄奥法諾_(10世纪)|狄奥法諾]]有[[亞美尼亞|亞美尼亞]]血统、前任皇帝[[巴西爾二世|巴西爾二世]]之弟。962年，羅曼努斯二世將君士坦丁及其兄巴西爾二世加冕成為共治皇帝。當羅曼努斯二世在963年去世時，君士坦丁年僅3歲，因此他與其兄[[巴西爾二世|巴西爾]]都因年龄太小而被暫時排除了繼承皇位的可能性 （不過君士坦丁曾被他的父親加冕爲[[共治|共治]]皇帝，所以從理論上說他的統治應始於是962年）。狄奧法諾皇后與帝國的一名主要將領[[尼基福魯斯·福卡斯|尼基福魯斯·福卡斯]]  （即尼基福鲁斯二世）结婚，尼基福魯斯因而獲得皇帝的地位，之後[[約翰一世·齊米斯基斯|約翰一世·齊米斯基斯]]及其兄巴西爾二世相繼登上了皇位。  

君士坦丁年青時與保加利亞國王女兒定婚，但是最後與海倫娜結婚，兩人生了三個女兒歐多齊婭、[[佐伊女皇|佐伊]]及[[狄奧多拉二世|狄奧多拉]]，亦即後來的佐伊女皇及狄奧多拉二世女皇。

其兄巴西爾二世在位期間，君士坦丁樂於享受皇族的權利而不涉及政事，只在巴西爾二世鎮壓貴族叛亂時參與征討。

== 登上帝位 ==
1025年，[[巴西爾二世|巴西爾二世]]去世，君士坦丁才成了唯一的皇帝。君士坦丁當時已年邁垂暮，巴西爾二世在位期間從貴族手上奪得的控制權很快便失去，國政又由君士坦丁堡的官僚貴族取得控制權。君士坦丁終日在酒席宴會和大競技場的娛樂中消磨時光，將國家大事交給官僚貴族，很快便將巴西爾二世辛苦積下的帝國財產揮霍一空，拜占廷帝國由此走向衰落。

== 晚年 ==
君士坦丁八世也像其兄一樣，沒有留下男嗣。君士坦丁因為所生的都是女兒，所以很擔心他未來的女婿會過多地干預皇室內部事務，於是他把染上[[天花|天花]]毁容的大女兒歐多齊婭送入修道院，並禁止他的兩個小女兒結婚。直到彌留之際，他才讓次女佐伊與他選中的皇位繼承人[[羅曼努斯三世|羅曼諾斯]]於1028年11月12日結婚。此時，佐伊已經50歲了，並且喪失了生育能力。結婚三天之後，羅曼諾斯和佐伊共同登上皇位，稱 「羅曼努斯三世皇帝和佐伊一世女皇」。
