'''坐摩巫祭神'''（{{lang-ja|イカスリノミカンナギノマツルカミ}}）為[[日本|日本]][[神道|神道]]的[[神祇|神祇]]，《[[延喜式神名帳|延喜式神名帳]]》指出共五位神明（詳情見次段說明），亦可寫作'''座摩巫祭神'''。

== 解說 ==
據《[[古語拾遺|古語拾遺]]》記述，原本宮中[[神祇官|神祇官]]在西院祭祀坐摩巫（{{ja}}イカスリノミカンナギ），而《延喜式神名帳》則提到坐摩巫祭神共有五位神明，其說明如下：
# 生井神（{{ja}}イクヰノカミ）：具有生命力的井水之神。
# 福井神（{{ja}}サクヰノカミ）：幸福與繁榮的井水之神。
# 綱長井神（{{ja}}ツナカヰノカミ）：深且清澈的井水之神。
# [[矢乃波波木神|波比祇神]]（{{ja}}ハヒキノカミ）：房屋庭院之神、帚神、竈神<ref>參看{{ja}}[http://www.genbu.net/data/settu/ikasuri_title.htm?print=on 玄松子の記憶：坐摩神社]。</ref><ref>詳見{{ja}}[http://21coe.kokugakuin.ac.jp/db/jinja/kindex2.php?J_ID=1104 國學院大學：波比祇神]。</ref>。
# 阿須波神（{{ja}}アスハノカミ）：竈神、旅人之神<ref>參看{{ja}}[http://www.genbu.net/data/settu/ikasuri_title.htm?print=on 玄松子の記憶：坐摩神社]。</ref><ref>詳見{{ja}}[http://21coe.kokugakuin.ac.jp/db/jinja/kindex2.php?J_ID=1105 國學院大學：阿須波神]。</ref>。

== 信仰 ==
在日本主祭坐摩巫祭神的[[神社|神社]]計有下述：
* 福長神社（[[京都府|京都府]][[京都市|京都市]][[上京區|上京區]]）：奉祀福井神、綱長井神。
* 坐摩神社（[[大阪府|大阪府]][[大阪市|大阪市]][[中央區_(大阪市)|中央區]]）
* 足羽神社（[[福井縣|福井縣]][[福井市|福井市]]）：將五尊神明合稱為「大宮地之靈」。
