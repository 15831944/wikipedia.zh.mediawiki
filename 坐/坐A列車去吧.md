'''坐A列車去吧'''（{{jpn|j=A列車で行こう|hg=エーれっしゃでいこう|rm=Ē ressha de ikō|f=y}}）是[[九州旅客鐵道|九州旅客鐵道]]（JR九州）主要運行在[[熊本站|熊本站]] - [[三角站|三角站]]之间，途經[[鹿兒島本線|鹿兒島本線]]和[[三角線_(九州旅客鐵道)|三角線]]的[[特別急行列車|特急列車]] 。

==概要==
特急「坐A列車去吧」於2011年10月8日開始作為[[熊本市|熊本市]] - [[天草群島|天草群島]]路線中的觀光列車運營。列車於[[三角站|三角站]]接續來往鄰接的三角港，和下島的本渡港的「天草宝島線」定期渡輪。隨着本列車投入運營，其運行区間的熊本站 - 三角站間在2011年9月1日起被賦予「天草三角線」（{{lang|ja|あまくさみすみ線}}）的愛稱。同時三角站也配合進行了翻新工程<ref>{{Cite web |title=JR九州、三角駅を南蛮風にリニューアル |date=2011年9月17日 |publisher=観光経済新聞 |url=https://www.kankokeizai.com/jr%E4%B9%9D%E5%B7%9E%E3%80%81%E4%B8%89%E8%A7%92%E9%A7%85%E3%82%92%E5%8D%97%E8%9B%AE%E9%A2%A8%E3%81%AB%E3%83%AA%E3%83%8B%E3%83%A5%E3%83%BC%E3%82%A2%E3%83%AB/ }}</ref><ref>{{Cite web |title=れとろ駅舎 三角駅 |accessdate=2019年2月12日 |publisher=れとろ駅舎 |url=http://www.retro-station.jp/09_kyushu/misumi.html }}</ref><ref>{{Cite web |title=【まったり駅探訪】三角線（あまくさみすみ線）三角駅に行ってきました。 |date=2017年3月29日 |publisher=アメーバ |url=https://ameblo.jp/aru-king/entry-12260514335.html }}</ref>。是1986年11月1日急行列車「火之山」（現在的特急列車「[[九州横断特急|九州横断特急]]」的前身）[[直通運行|直通運行]]至三角線結束後近30年來，三角線再次有優等列車運行，也是首次有特急列車運行。

=== 列車名的由来 ===
列車名的頭文字「'''A'''」是，「以[[西方世界|南蛮文化]]来的天草為動機、演繹以[[歐洲|歐洲]]的大人的旅程」<ref>{{Cite news |title=天草へは「A列車で行こう」 JR三角線に特急 |newspaper=熊本日日新聞 |date=2011年05月18日 |author=井上直樹 |url=http://kumanichi.com/news/local/main/20110518002.shtml |agency=熊本日日新聞社 |archiveurl=https://web.archive.org/web/20110518212011/http://kumanichi.com/news/local/main/20110518002.shtml |archivedate=2011年5月18日 |accessdate=2017-02-10 |dead-url=yes }}</ref>為概念，并以大人 ('''A'''dult) 和天草 ('''A'''makusa) 的首字母命名<ref>{{Cite web|title=「A列車で行こう」 大人向けに観光列車 熊本-三角|date=2011年5月18日|publisher=西日本新聞|url=http://www.nishinippon.co.jp/nnp/item/242553|access-date=2019年2月12日|archive-url=https://web.archive.org/web/20110920030340/http://www.nishinippon.co.jp/nnp/item/242553|archive-date=2011年9月20日|dead-url=yes}}</ref>。

==運行概況==
主要在周末、長假期期間的每日，在熊本站 - 三角站之間運行3個往返班次。由於列車只有兩節車廂，因此和「九州横断特急」以及「[[隼人之風號列車|隼人之風]]」一樣，沒有列車長進行乘務工作，而是由客室乗務員進行的[[一人控制|一人控制]]方式。

使用經過改造之2輛編成的{{link-ja|日本國鐵Kiha 185系柴油客車|国鉄キハ185系気動車|Kiha 185系}}，屬{{link-ja|熊本車輛中心|熊本鉄道事業部#熊本車両センター}}。車輛設計由[[水戶岡銳治|水戶岡銳治]]擔當<ref>{{Cite web |author=峯浦憲光 |date=2011年03月20日 |url=http://railf.jp/news/2011/03/20/143900.html |title=特急“阿蘇ゆるっと博号”にラッピング車 |work=鉄道ファン railf.jp 鉄道ニュース |publisher=交友社 |accessdate=2017年02月10日}}</ref>。

車廂以「16世紀[[地理大发现|地理大发现]]時代的歐洲文化」和「美好過去的天草」為主題設計<ref name="concept">{{Cite press release |title=特急「A列車で行こう」の魅力 |publisher=九州旅客铁道 |date=2011年08月16日 |url=http://www13.jrkyushu.co.jp/newsreleaseweb.nsf/9dd28b8cb8f46cee49256a7d0030d2e6/78fb5e999a02c1b9492578ee0036cfce/$FILE/%E7%89%B9%E6%80%A5%E3%80%8C%EF%BC%A1%E5%88%97%E8%BB%8A%E3%81%A7%E8%A1%8C%E3%81%93%E3%81%86%E3%80%8D%E3%81%AE%E9%AD%85%E5%8A%9B.pdf |format=PDF |archiveurl=https://web.archive.org/web/20120330212749/http://www13.jrkyushu.co.jp/newsreleaseweb.nsf/9dd28b8cb8f46cee49256a7d0030d2e6/78fb5e999a02c1b9492578ee0036cfce/$FILE/%E7%89%B9%E6%80%A5%E3%80%8C%EF%BC%A1%E5%88%97%E8%BB%8A%E3%81%A7%E8%A1%8C%E3%81%93%E3%81%86%E3%80%8D%E3%81%AE%E9%AD%85%E5%8A%9B.pdf |archivedate=2012年3月30日 |accessdate=2017年02月10日 |dead-url=yes }}</ref>，主要配以黑色和金色雙色，營造出教堂的風格，設有酒吧台和[[彩繪玻璃|彩繪玻璃]]。

1号車定員28人，2号車定員56人，共84人。全車為普通車指定席。1号車除了沙發長椅，還設有吧枱「A-TRAIN BAR」。2号車設有四組4人用的半卡位廂座<ref>{{Cite web |date=2011年05月17日 |url=http://rail.hobidas.com/news/info/article/127695.html |title=JR九州 三角線に観光特急列車 |work=鉄道ホビダス 最新鉄道情報 |publisher=ネコ・パブリッシング |accessdate=2017年02月10日}}</ref>。酒吧除了提供[[啤酒|啤酒]]和[[高球_(雞尾酒)|高球酒]]外，還供應本列車限定的原創[[鸡尾酒|鸡尾酒]]等酒類。車内的背景音樂采用由向谷实編曲的[[爵士經典曲|爵士經典曲]]「[[Take_the_'A'_Train|Take the 'A' Train]]」<ref name="concept"/>。
