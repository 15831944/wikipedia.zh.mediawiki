'''坐月子'''，是一些[[東亞|東亞]][[民族|民族]]的婦女在[[分娩|生產]]過後休息調養身心的習俗，時間約一個月，故稱坐月子。[[粵語|粵語]]稱'''-{坐月}-'''，[[閩南語|閩南語]]稱'''-{做月內}-'''，[[客家話|客家話]]稱'''-{做月}-'''，[[中原官話|中原官話]]稱'''-{坐月子}-'''或稱'''作月子'''，[[日語|日語]]稱'''產後之肥立'''（{{lang|ja|産後の肥立ち}}、{{lang|ja|さんごのひだち}}）[[韓語|韓語]]稱為'''三七'''（{{lang|ko|삼칠}}、{{lang|ko|三七}}，即21日）。依各地民情風俗不同，也有四十天、兩個月和一百天的差别。坐月子實際上是對產婦的行為和照顧方式的限制。

==習俗==
===華人===
在廣東及附近的一些地區，產婦坐月子到了第12天之後（由嬰兒出生當天起計12天），親友才可以探望產婦及嬰兒，該天稱為「[[十二朝|十二朝]]」。廣東人也在該天開始派發[[薑醋|薑醋]]。而有些家庭會根據其家鄉的習慣舉行儀式，如拜祭[[祖先|祖先]]。

某些地區流行"頭三胎靠婆家"的習俗, 意指頭三胎要由婆家準備所有坐月子需要的物品及開銷。

===越南===
在越南只有華人，以及一些跟中國跨界的種族，需要坐月子，此習俗為各國華裔獨有，本土越族[[京族|京族]]以及南島民族，無此習俗。

===其他===
中國古代少数民族婦女，生完孩子便起來做事。由男性代替坐月子的习俗。<ref>《[[太平廣記|太平廣記]]》卷四八三引[[尉遲樞|尉遲樞]]的《[[南楚新聞|南楚新聞]]》記載：“南方有[[仡佬族|獠婦]]，生子便起。其夫臥床褥，飲食皆如乳婦。”《[[马可·波罗游记|马可·波罗游记]]》：“[[傣族|傣族]]妇女产子，洗后裹以襁褓，产妇立起工作，产妇之夫则抱子卧床四十日”；《黔记》卷二：郎慈苗在咸宁州，其俗更异，妇人产子，必夫守房，不逾门户，弥月乃出。产妇则出入耕作，措饭食，以供夫乳儿，日无暇刻。”</ref>

== 注意事項 ==
产后6周内应保持房间通风、适当行走、保持身体清洁（尤其为阴部与乳房，剖腹产需注意伤口）以免感染、注意牙龈卫生、不可性交，哺乳期应注意避孕。切勿长期卧床、关闭门窗与空调、禁足以及停止清洁身体。

== 飲食 ==
产后应保持正常较清淡饮食，多饮水，提高肉类、蛋类以及脱脂牛奶的摄入，切勿食用骨头汤、酒精或高脂肪食品。

== 批评 ==
现代医学认为，中国传统观念中的坐月子并非全部合理，如“不吃盐”实际上会加重身体脱水。因為鹽分會隨著排尿而自然代謝，不吃鹽容易導致體內電解質與滲透壓不平衡，而引發脫水現象。另酒精会通过母乳影响婴儿的正常發育<ref>[http://baby.sina.com.cn/health/2005-03-08/47_69775.shtml 坐月子调养二十个错误]</ref><ref>[http://www.rayli.com.cn/0010/2010-12-12/L0010001003_797248.html 产后进补别被老方法“害”了]</ref><ref>[http://songshuhui.net/archives/53887#comment-118504 科学松鼠会：破除坐月子的流言和陋习]</ref>
