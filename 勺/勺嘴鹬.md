'''勺嘴鹬'''（[[學名|學名]]：''{{lang|la|Calidris pygmaea}}''）又名'''-{zh-cn:琵嘴鹬;zh-tw:勺嘴鷸;}-'''、'''匙嘴鷸'''，是一種小型的[[涉禽|涉禽]]，僅在極少數的[[凍土層|凍土層]]地帶上繁殖，並在[[東南亞|東南亞]]的[[濕地|濕地]]過冬。在2007年的一次由[[國際鳥盟|國際鳥盟]]進行的統計中，勺嘴鹬的數目可能少於100對，[[IUCN紅色名錄|IUCN紅色名錄]]將其保護現狀由[[瀕危|瀕危]]提升到極危，是今年內8種被提升到極危程度的鳥類的其中一種。

== 分類學 ==
這物種由[[卡爾·林奈|卡爾·林奈]]在1758年，在其《[[自然系統|自然系統]]》（Systema Naturae）一書中首次介紹，並命名為 ''Platalea pygmea''。<ref>{{la icon}} {{cite book | last=Linnaeus | first=C | authorlink=Carolus Linnaeus | title = Systema naturae per regna tria naturae, secundum classes, ordines, genera, species, cum characteribus, differentiis, synonymis, locis. Tomus I. Editio decima, reformata. | publisher=Holmiae. (Laurentii Salvii). | date=1758| quote = P. corpore supra fusco, subtus albo. |pages=140}}</ref>。但在1821年由[[斯雲·尼爾遜|斯雲·尼爾遜]]（Sven Nilsson）分類到目前的一個屬並沿用至今<ref> {{cite book | last=Nilsson | first=S |  title= Ornithologica Suecica volume II (1) | date=1758|  |pages=29}}</ref> 。現時勺嘴鹬屬於鴴形目鷸科內的一種涉禽，但將涉禽歸到鴴形目本身就是[[鳥類DNA分類系統|鳥類DNA分類系統]]的一大缺陷。此外目前將勺嘴鹬看成是勺嘴鷸屬內的[[單屬種|單屬種]]雖然合理，但不管是從它奇特的[[形態適應特徵|形態適應特徵]]或表現出歧異的[[DNA序列|DNA序列]]中去判斷，勺嘴鷸都應該從其相近的種族中進一步的分別出來<ref name=Thomas>Thomas, Gavin H.; Wills, Matthew A. & Székely, Tamás (2004): A supertree approach to shorebird phylogeny. ''BMC Evol. Biol.'' '''4''': 28. {{doi|10.1186/1471-2148-4-28}} [http://www.pubmedcentral.org/picrender.fcgi?artid=515296&blobtype=pdf PDF fulltext]{{dead link|date=六月 2017 |bot=InternetArchiveBot }} [http://www.pubmedcentral.org/articlerender.fcgi?artid=515296#supplementary-material-sec Supplementary Material] {{Webarchive|url=https://archive.is/20130802041856/http://www.pubmedcentral.org/articlerender.fcgi?artid=515296%23supplementary-material-sec#supplementary-material-sec |date=2013年8月2日 }}</ref>。

== 外觀描述 ==
[[File:SpoonbilledSandpiper.jpg|thumb]]
勺嘴鹬外表上最獨特的地方在牠那湯匙形狀的嘴，但在野外不易辨認，體長一般達14至16cm。[[繁殖|繁殖]]中的成鳥頭、頸及胸均為紅棕色並帶暗褐色縱紋。上身主要為黑色，有皮黃色至淡赤褐色的羽緣。非繁殖季中的成鳥體色沒有紅色，反之然上身換上淡褐灰色及在覆羽上有白色羽緣。下半身為全白，腳黑色，[[虹膜|虹膜]]為褐色<ref name = speciessheet>{{cite web|title= Spoon-billed Sandpiper  ''Eurynorhynchus pygmeus''|work=Species factsheet |url= http://www.birdlife.org/datazone/species/index.html?action=SpcHTMDetails.asp&sid=3060&m=0|publisher= BirdLife International|accessdate=2008-03-06}}</ref>。

勺嘴鹬在與同伴溝通時會發出從低聲的皮聲到顫動的灰聲。求偶鳴唱時，則會間歇地發出嗡嗡聲及漸沉的皮－比－比啼囀聲。雄性也會透過演示短距離的盤旋、打轉及急速邊鳴唱邊下沉的飛行技藝以達致求偶目的。每年六到七月間於凍土層的海岸地區築巢，喜在長有[[苔蘚|苔蘚]]、矮柳及草的沙地，或在湖泊及草澤邊等築巢。進食時把嘴深入泥中，一邊前進一邊左右移動頸部。翅長98-106mm，嘴19-24mm，嘴尖左右寬10-12mm，[[跗節|跗節]]長19-22mm及尾長37-39mm<ref name=shorebirds>Hayman, P., J. Marchant and T. Prater (1986) Shorebirds. Croom Helm.</ref>。

== 棲地及活躍範圍 ==
這種鳥僅在俄羅斯東北地帶的[[楚科奇半島|楚科奇半島]]上及沿著[[堪察加半島|堪察加半島]]的地峽附近繁殖後代<ref name = speciessheet/>。每年冬天來臨前，牠們就會於[[太平洋|太平洋]]的西岸進行[[鸟类迁徙|遷徙]]。經過[[日本|日本]]、[[北韓|北韓]]、[[南韓|南韓]]、[[中國|中國]]、[[臺灣|臺灣]]，最後來到牠們過冬的[[東南亞|東南亞]]地區。多個東南亞國家均曾錄得勺嘴鹬來訪的紀錄，包括[[印度|印度]]、[[孟加拉|孟加拉]]、[[斯里蘭卡|斯里蘭卡]]、[[緬甸|緬甸]]、[[泰國|泰國]]、[[越南|越南]]、[[菲律賓|菲律賓]]、[[馬來西亞|馬來西亞]]及[[新加坡|新加坡]]等<ref name = speciessheet/>。

== 現狀 ==
最近一次由國際鳥盟對其重要繁殖地──俄羅斯的楚科奇半島的調查顯示，現時全球僅餘下少於100對<ref>[http://www.birdlife.info/wbdbwebstaging/SpcHarnessDetails.asp?sid=3060&m=0 Spoon-billed Sandpiper - BirdLife Species Factsheet]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}，載國際鳥盟，2008年5月20日查閱</ref>。對比起1970年，仍有近2000到2800對<ref name = Birdlifenews>{{cite web|title= Unique wader faces extinction |work= |url= http://www.birdlife.org/news/news/2007/01/sbs_workshop.html|publisher= BirdLife International|accessdate=2008-03-06}}</ref>，2000年數目下降至1000對，2005年僅餘下少於400對，每年的個體數目均急速減少。如整體情況沒有改善，勺嘴鹬將可能在未來數年踏上滅絕之路。研究認為牠們數目急降的主因包括繁育生境及遷飛過程時中轉站的破壞。其中位於南韓重要的一站──[[新萬金防潮堤|新萬金防潮堤]]近40000[[公頃|公頃]]的大型[[填海|填海]]計劃所破壞<ref>[http://www.restoresaemangeum.com/int/home.html RESTORE Saemangeum] {{webarchive|url=https://web.archive.org/web/20071111025447/http://www.restoresaemangeum.com/int/home.html |date=2007-11-11 }}，載RestoreSaemangeum網站，於2008年5月20日查閱</ref><ref name = speciessheet/>。而每年亦有為數不少的勺嘴鹬在中轉站當地被捕獵<ref name =Birdwatch >{{cite web|title= New hope for Spoon-billed Sandpipers |work= |url= http://www.birdwatch.co.uk/website/index.php?option=com_content&task=view&id=933&Itemid=32|publisher=Birdwatch |accessdate=2008-03-06}}</ref>。此外，[[全球暖化|全球暖化]]導致牠們特有的繁殖地──凍土層的減少也影響了牠們可繁殖後代的機會。

現時其中轉站當中受保護的地區包括中國的[[盐城|盐城]]、香港的[[米埔|米埔濕地]]及位於印度的[[卡利米爾角|卡利米爾角]]和[[奇爾瓦湖|奇爾瓦湖]]<ref name = speciessheet/>。一隊專家研究團隊在緬甸的其中兩個海岸找到至少84頭勺嘴鹬，表示當地是其重要的度冬區。國際鳥盟除密切注視該兩片濕地外，並希望研究出勺嘴鹬過冬中轉站的所有地點，以保護這種鳥類<ref name =Birdwatch >{{cite web|title= New hope for Spoon-billed Sandpipers |work= |url= http://www.birdwatch.co.uk/website/index.php?option=com_content&task=view&id=933&Itemid=32|publisher=Birdwatch |accessdate=2008-03-06}}</ref>。

自2011年起，[[英國|英國]]的[[野鳥和溼地信託|野鳥和溼地信託]]（The Wildfowl & Wetlands Trust, WWT）開始針對性地進行人工繁殖的工作。為防止它們進一步滅絕，它們分別在2011年11月及2012年7月期間，從俄羅斯的極北地區將鳥蛋帶到英國位於[[告羅士打郡|告羅士打郡]]的保肓中心進行繁殖。兩年間共27尾幼鳥已從這中心中孵化。這項活動將一宜持續，直至在其南遷路程上的各個站點的保育工作均順利開展為止。<ref>{{cite web|author=WWT|title=First ever spoon-billed sandpiper chicks hatch in the UK|url=http://www.wwt.org.uk/news/all-news/2012/07/wwt-news/first-ever-spoon-billed-sandpiper-chicks-hatch-in-the-uk/|accessdate=2012-07-18}}</ref>
