'''環帶綱'''（{{lang|la|Clitellata}}），是[[环节动物门|环节动物门]]下的一个纲，在[[系統演化|系統演化]]上爲一個[[單系群|單系群]]，包括[[蛭|蛭]]（[[螞蟥|螞蟥]]，含[[水蛭|水蛭]]、[[旱蛭|旱蛭]]）和[[寡毛綱|寡毛類]]（如[[蚯蚓|蚯蚓]]）等。牠們的特色是身體上有一節像是「衣領」一樣的環帶，在生育時會產生[[卵繭|卵繭]]。環帶綱大約有8000多個品種。

與[[多毛綱|多毛綱]]（{{lang|la|Polychaeta}}）不同的是：牠們的頭部（[[圍口節|圍口節]]）沒有[[疣足|疣足]]（{{lang|la|parapodia}}），而且發展比較簡單。

== 習性 ==
大多數環帶綱的生物可在陸地上及水裡生活。

== 繁殖 ==
所有環帶綱動物都是雌雄同體的。在繁殖期間，環帶綱動物會分泌一種會硬化的表層。牠會從這表層向後爬走，並將[[受精卵|受精卵]]或同時間將[[卵子|卵子]]及[[精子|精子]]留在這表層內，此表層會形成一個[[卵繭|卵繭]]。受精卵會在沒有幼蟲的階段進而生長。<ref>{{cite book en|last = Reichardt|first = Anna Katharina|title = Systematische Zoologie|date = 2006|pages = 63, 67-68 }}</ref>

== 分類 ==
* [[原環蟲綱|原環蟲綱]]（{{lang|la|Acanthobdellida}}）
* [[蛭蚓纲|蛭蚓纲]]（{{lang|la|Branchiobdellae}}）
* [[蛭亞綱|蛭亞綱]]（{{lang|la|Hirudinea}}）
* [[寡毛亞綱|寡毛亞綱]]（{{lang|la|Oligochaeta}}）：例如：[[蚯蚓|蚯蚓]]（不論是水生還是陸生的）。
