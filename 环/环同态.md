在[[环论|环论]]或[[抽象代数|抽象代数]]中，'''环[[同态|同态]]'''是指两个环R與S之间的映射f保持两个环的加法与乘法运算。

更加精确地，如果''R''和''S''是环，则环同态是一个函数''f'' : ''R'' → ''S''，使得：
* ''f''(''a'' + ''b'') = ''f''(''a'') + ''f''(''b'')，对于''R''内的所有''a''和''b''；
* ''f''(''ab'') = ''f''(''a'') ''f''(''b'')，对于''R''内的所有''a''和''b''；
* ''f''(1) = 1。

如果我们不要求环具有乘法单位元，则最后一个条件不需要。

==性质==

直接从这些定义，我们可以推出：
* ''f''(0) = 0
* ''f''(−''a'') = −''f''(''a'')
* 如果''a''在''R''内具有乘法逆元，则''f''(''a'')在''S''内具有乘法逆元，且有''f''(''a''<sup>−1</sup>) = (''f''(''a''))<sup>−1</sup>。
* ''f''的[[核_(代数)|核]]，定义为ker(''f'') = {''a'' in ''R'' : ''f''(''a'') = 0}，是''R''内的一个[[环理想|理想]]。每一个交换环''R''内的理想都可以从某个环同态用这种方法得出。对于具有单位元的环，环同态的核是一个没有单位元的子环。
* 环同态''f''是单射，当且仅当ker(''f'') = {0}。
* ''f''的[[像|像]]，im(''f'')，是''S''的一个子环。
* 如果''f''是[[双射|双射]]，那么它的逆映射''f''<sup>−1</sup>也是环同态。在这种情况下，''f''称为'''同构'''。在环论的立场下，同构的环不能被区分。
* 如果存在一个环同态''f'' : ''R'' → ''S''，那么''S''的[[特征_(代数)|特征]][[整除|整除]]''R''的特征。这有时候可以用来证明在一定的环''R''和''S''之间，不存在环同态''R'' → ''S''。
* 如果''R''是一个[[域|域]]，则''f''要么是单射，要么是零函数。（但是，如果''f''保持乘法单位元，则它不能是零函数）。
* 如果''R''和''S''都是[[域|域]]，则im(''f'')是''S''的一个子域（如果''f''不是零函数）。
* 如果''R''和''S''是交换环，''S''没有[[零因子|零因子]]，则ker(''f'')是''R''的一个[[素理想|素理想]]。
* 如果''R''和''S''是交换环，''S''是一个域，且''f''是满射，则ker(''f'')是''R''的一个[[最大理想|最大理想]]。
* 对于每一个环''R''，都存在一个唯一的环同态'''Z''' → ''R''。这就是说，整数环是环[[范畴|范畴]]中的[[始对象|始对象]]。

==例子==
* 函数''f'' : '''Z''' → '''Z'''<sub>''n''</sub>，由''f''(''a'') = [''a'']<sub>''n''</sub> = ''a'' '''mod''' ''n''定义，是一个[[满射|满射]]的环同态，它的核为''n'''''Z'''。
* 当''n'' > 1时，不存在环同态'''Z'''<sub>''n''</sub> → '''Z'''。
* 如果'''R'''[''X'']表示变量为''X''的所有实系数[[多项式|多项式]]的环，'''C'''表示[[复数|复数]]，则函数''f'' : '''R'''[''X''] → '''C'''，由''f''(''p'') = ''p''(''i'')定义（在多项式''p''中用虚数单位''i''来代替变量''X''），是一个满射的环同态。''f''的核由'''R'''[''X'']内所有能被''X''<sup>2</sup> + 1整除的多项式组成。.

==环同态的种类==
* [[双射|双射]]的环同态称为''环同构''。
* 定义域与值域相同的环同态称为''环自同态''。

在环范畴中，单射的环同态与[[单同态|单同态]]是相等的：如果''f'':''R''→''S''是单同态而不是单射，则它把某个''r<sub>1</sub>''和''r<sub>2</sub>''映射到''S''的同一个元素。考虑从'''Z'''[''x'']到''R''的两个映射''g<sub>1</sub>''和''g<sub>2</sub>''，分别把''x''映射到''r<sub>1</sub>''和''r<sub>2</sub>''；''f'' <small>o</small> ''g<sub>1</sub>''和''f'' <small>o</small> ''g<sub>2</sub>''是相等的，但由于''f''是单同态，这是不可能的。

然而，在环范畴中，满射的环同态与[[满同态|满同态]]是非常不同的。例如，'''Z''' ⊆ '''Q'''是满同态，但不是满射。
