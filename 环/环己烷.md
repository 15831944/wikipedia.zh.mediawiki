'''环己烷'''，[[环烷烃|环烷烃]]的一种，也称'''六氢化[[苯|苯]]'''。分子量：84.160。沸点：80.74℃。凝固点：6.5℃。无色，易燃，微溶于水，具有挥发性，微有刺激性气味的液体，无腐蚀性，分子式：C<sub>6</sub>H<sub>12</sub>。
作为一种重要的工业原料，环己烷主要用于合成尼龙的原料己二酸和己内酰胺。
环己烷是非极性溶剂，在涂料和清漆中有较广泛的应用。相对与苯来说环己烷毒性要小，因此在医药上用环己烷作为苯的替代[[溶剂|溶剂]]。

== 合成 ==

工业上一般是通过苯的氢化来合成，也可以通过石油的催化重整再经分馏得到。苯催化加氢一般选择镍，钯，铂等作为催化剂，加氢反应为放热反应。通过在石油中蒸馏得到环己烷只有荷兰飞利浦公司生产。

1894年，[[阿道夫·冯·拜尔|阿道夫·冯·拜尔]]以庚二酸的[[狄克曼缩合反应|狄克曼缩合反应]]开始并後接一系列还原反应合成环己烷：

: [[File:Cyclohexane_Synthesis.svg|550px]]

同年，E. Haworth和W.H. Perkin Jr.（1860年－1929年）以1,6-二溴己烷的[[武兹反应|武兹反应]]开始并以相同方式合成环己烷：

: [[File:Cyclohexane_Synthesis_Perkin.svg|300px]]

令人惊奇的是，他们的合成路线最终得到的产物比环己烷沸点高10℃，这一难题於1895年由Markovnikov、N.M. Kishner和Nikolay Zelinsky解决，他们将产物烷重新确定为[[甲基环戊烷|甲基环戊烷]]，是由环己烷发生意想不到的[[重排反应|重排反应]]生成的。

: [[File:CyclohexaneBerthelot.svg|300px]]

== 用途 ==
环己烷与[[亚硝酰氯|亚硝酰氯]]发生光化学反应得到环己烷[[肟|肟]]，后者经[[贝克曼重排|贝克曼重排]]生成内[[酰胺|酰胺]]，再开环聚合得到[[尼龙|尼龙]]-6。这个工艺使得环己烷产生了非常重要的工业应用价值。

可以[[萃取|萃取]][[海水|海水]]中微量的[[碘|碘]]。

{{Fact|另外，由于近年发现[[己烷|己烷]]有潜在的致[[癌|癌]]可能，所以在[[西欧|西欧]]国家的[[实验室|实验室]]已经禁用己烷来过[[色谱|管柱層析]]，而用环己烷替代。|time=2015-04-20T23:30:59+00:00}}
