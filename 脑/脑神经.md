'''脑神经'''（Cranial nerves）属于[[周围神经系统|周围神经系统]]，区别于由[[脊髓|脊髓]]发出的[[脊神经|脊神经]]。它们是直接由[[脑|脑]]发出。在人类中，传统上认为一共有 12 对脑神经，其中有 10 对分布于头面部。除第 1-2 对外，其余 10 对脑神经都是从[[脑幹|脑幹]]发出。

==非人脊椎动物的脑神经==
许多其他[[脊椎动物|脊椎动物]]的脑神经分布与人相似，在其他的[[羊膜动物|羊膜动物]]（非两栖类四足动物）中就已经[[演化|演化]]出了第 XI 和第 XII 对脑神经。在一些原始的软骨鱼中，比如[[白斑角鲨|白斑角鲨]]或者[[海绿角鲨|海绿角鲨]]中，还存在另外一对脑神经，被命名为 [[第_0_对脑神经|第 0 对脑神经]]，在演化上被认为是经典的脑神经出现之前的结构。

== 功能 ==
脑神经有着重要的功能：[[视觉|视觉]]、[[嗅觉|嗅觉]]、[[听觉|听觉]]、[[味觉|味觉]]均由脑神经传递，人类丰富的面部表情也由脑神经控制。脑神经中的迷走神经参与[[自主神经系统|自主神经系统]]的构成。

== 成分 ==
脑神经划分为7种[[纤维|纤维]]成分：
# 一般躯体感觉纤维(GSA)：分布于[[皮肤|皮肤]]、[[肌|肌]]、[[肌腱|肌腱]]和[[眶|眶]]内、[[口|口]]、[[鼻|鼻]]大部分[[粘膜|粘膜]]
# 特殊躯体感觉纤维(SSA)：分布于[[外胚层|外胚层]]衍化来的特殊[[感觉器官|感觉器官]]([[视器|视器]]、[[前庭器官|前庭器官]])
# 一般内脏感觉纤维(GVA)：分布于[[头|头]]、[[颈|颈]]、[[胸|胸]]、[[腹|腹]]的[[脏器|脏器]]
# 特殊内脏感觉纤维(SVA)：分布于[[味蕾|味蕾]]和[[嗅器|嗅器]]
# 一般躯体运动纤维(GSE)：分布于[[中胚层|中胚层]]衍化来的[[眼球外肌|眼球外肌]]、[[舌肌|舌肌]]等[[横纹肌|横纹肌]]
# 一般内脏运动纤维(GVE)：分布于[[平滑肌|平滑肌]]、[[心肌|心肌]]和[[腺体|腺体]]
# 特殊内脏运动纤维(SVE)：分布于[[咀嚼肌|咀嚼肌]]、[[表情肌|表情肌]]和[[咽喉肌|咽喉肌]]等


此為背誦口訣：
一嗅二視三動眼，
四滑五叉六外旋，
七顏八聽九舌咽，
十迷走啊十一副，
十二舌下神經柱。
