'''玉熙宫'''，原址位于今[[北京市|北京市]][[西城区|西城区]][[文津街|文津街]][[中国国家图书馆古籍馆|中国国家图书馆古籍馆]]处，是[[明朝|明朝]]皇家宫殿，现已无存。

==简介==
明朝时，玉熙宫是帝王游乐之所。此处距[[万寿宫_(明朝皇家宫殿)|万寿宫]]很近。《[[明史|明史]]》记载：“[[嘉靖|嘉靖]]四十年十一月辛亥（1561年12月31日）夜，万寿宫灾”。万寿宫又叫永寿宫，是[[嘉靖帝|嘉靖帝]]的别宫。《[[万历野获编|万历野获编]]》载，这次火灾是因嘉靖帝喝醉了酒，与新宠幸的宫姬萧美人，在貂皮帐内燃放小焰火取乐，引燃貂皮帐酿成火灾，“殿内乘舆一切服御，及先朝异宝，尽付之一炬”。深夜万寿宫中起火，内侍多有死者，嘉靖帝被[[锦衣卫|锦衣卫]]救出，护驾到玉熙宫避难。《金鳌退食笔记》载：“玉熙宫在[[西安门|西安门]]里街北，[[金鳌玉蝀桥|金鳌玉蝀桥]]之西。[[明世宗|明世宗]]嘉靖四十年十一月辛亥，万寿宫灾，暂御玉熙宫。神宗时，选近侍三百余名，于玉熙宫学习官戏，岁时陛座，则承应之。”[[吴伟业|吴伟业]]《琵琶行》曰：“先皇驾幸玉熙宫、凤纸佥名唤乐工。苑内水嬉金傀儡，殿头过锦玉玲珑。一自中原盛豺虎，暖阁才人罢歌舞。插柳停搊素手筝，烧灯罢击花奴鼓。”<ref name=youlai>{{Cite web|url =http://www.bjxch.gov.cn/BJXCHHuDongJiaoLiu/BJXCHXCBAXiangQing.ycs?GUID=561847|title =文津街的由来|author =|date =2013-11-22|publisher =北京西城报|language =|accessdate =|quote =}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

玉熙宫西有羊房、延寿庵、安乐堂。明朝[[成化|成化]]年间，[[明宪宗|明宪宗]]朱见深专宠[[万贵妃|万贵妃]]。[[孝穆纪皇后|孝穆纪皇后]]托病居住安乐堂，并在此生子[[朱祐樘|朱祐樘]]，后继位当皇帝成为[[明孝宗|明孝宗]]<ref name=youlai/><ref>{{Cite web |url =http://history.sohu.com/20160908/n467958114.shtml  |title = 明朝弘治皇帝出生地被张冠李戴 两地同名安乐堂 |author =  |date =2016-09-08  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

[[清朝|清朝]]，玉熙宫、羊房、延寿庵、安乐堂皆废。玉熙宫改成[[御马监|御马监]]西马场，羊房谐音改为养蜂夹道。[[康熙|康熙]]十三年五月，曾在原玉熙宫设席殿，停放[[仁孝皇后|仁孝皇后]]梓宫，集百官举哀。后来玉熙宫改为内厩，豢养御马，称御马监西马场，俗称御马场。养蜂夹道南口一带，[[中华民国|中华民国]]初年改为军营，公府操场<ref name=youlai/>。
