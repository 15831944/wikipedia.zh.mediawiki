'''玉佛寺'''（{{lang-th|วัดพระศรีรัตนศาสดาราม}}）是[[泰國|泰國]]境內的[[佛教|佛教]]聖地，位於首都[[曼谷|曼谷]]市內，屬於[[曼谷大皇宫|國家大皇宮]]的範圍。

== 歷史 ==
玉佛寺建於1782年，經歷上百年餘年的悠悠歲月。玉佛寺屬於泰國[[曼谷王朝|曼谷王朝]]開朝時建築，自從郑昭麾下部将昭披耶節基推翻[[吞武里王朝|吞武里王朝]]之後，加冕為王世稱[[拉玛一世|拉玛一世]]，並下達命令将湄南河西岸舊都所在地迁到湄南河对岸的曼谷，寺内供奉着由拉玛一世王迎请入寺的泰国国宝“玉佛像”，且在此地為皇家舉行重要的儀式，例如：每年按照熱季、雨季、涼季三季變換的時間，泰國國王或王子親自為玉佛更衣以保国泰民安的隆重儀式，从拉玛一世王迄今沿袭不变，今天玉佛寺被視為國家守護的宗教聖地。

玉佛寺目前占大皇宮的面積約四分之一，位於宮內的東北方向，寺內供奉的[[帕佛陀大摩尼寶玉佛|帕佛陀大摩尼寶玉佛]]（{{lang|th|พระพุทธมหามณีรัตนปฏิมากร}}）簡稱玉佛（{{lang|th|พระแก้วมรกต}}），由一整塊翠綠壁[[玉部|玉]]（[[翡翠|翡翠]]）雕成，高約66公分，寬約48公分，由玻璃保護，上有多層華蓋，基座高昂。据说这尊玉佛是在许多地方停留过，初期是镶在清莱府玉佛寺的佛塔中，后来佛塔被雷劈毁之後才发现这尊玉佛在内，当时这尊玉佛是被灰泥（stucco)盖住而贴滿金箔，后来佛像眼部所盖上的灰泥部分開始裂开，才发现佛像原来是整块翡翠而雕制的，后来這尊玉佛就到[[南邦府|南邦府]]、[[清邁府|清邁府]]、[[老撾|老撾]][[萬象|萬象]]玉佛寺入寺供奉。[[吞武里王朝|吞武里王朝]][[鄭昭|鄭昭]]出兵征討歸附緬甸的老撾三個小王國，同程迎请玉佛到吞武里[[郑王寺|郑王寺]]（{{lang|th|วัดอรุณราชวราราม}}）入寺供奉。后来[[拉玛一世|拉玛一世]]將泰國首都遷至曼谷之后，即请玉佛到玉佛寺皇宫内供奉。

玉佛每年按照熱季、雨季、涼季三季變換的時間，泰國國王或王子親自為玉佛更衣換服，以示尊敬保佑國泰民安的皇家重要儀式，从拉玛一世王迄今沿袭不变。

== 特色 ==
玉佛寺內的所有建築都有加高的白色大理石基座和繽紛奪目的[[鑲嵌畫|鑲嵌畫]]裝飾，在主殿北方，有三座佛塔，分別是純泰式藏經閣（{{lang|th|พระมณฑป}}，Phra Mondop），內有拉瑪一世寫得[[藏經|藏經]]、錫蘭式金色舍利佛塔（{{lang|th|พระศรีรัตนเจดีย์}}，Phra Sri Ratana Chedi），由[[拉瑪四世|拉瑪四世]]所建、父祖皇室宗廟（{{lang|th|ปราสาทพระเทพบิดร}}，Prasat Phra Debidorn）。

最外圍的圍牆則是白色的，相當簡單，內牆繪有泰國版本[[罗摩衍那|羅摩耶那史詩]]（{{lang|th|รามายณะ}}，Ramayana）神話的場景。參觀時可由玉佛寺北門開始，以順時針的方向觀賞178幅壁畫，多數壁畫都有[[泰文|泰文]]的說明。除此之外，在玉佛寺中出现五米高的守门神雕像，就有幾個主要在[[羅摩衍那|羅摩衍那]]史詩神話中的大型神像，如猴王[[哈奴曼|哈奴曼]]（{{lang|th|หนุมาน}}，Hanuman）反派十面巨人[[罗波那|罗波那]]（{{lang|th|ทศกัณฑ์}}，Ravana）。

東側則有一排8座，呈一条直線的佛塔，外貼中國風味的磁磚，一共有8種顏色，代表佛教的不同元素。

此外，玉佛寺區裡，有[[吳哥窟|吳哥窟]]的模型位於藏經閣北方中，是泰國國王[[拉瑪三世|拉瑪三世]]所增加的收藏。在[[柬埔寨|柬埔寨]]的[[高棉帝國|高棉帝國]]時期，兩地的文化經常有交流且有共通之處。

== 參觀 ==

玉佛寺被視為泰國的三大佛寺之一，遊客入內必須衣著整齊，不得穿露出腳踝和肩膀的服飾，不得穿[[拖鞋|拖鞋]]，並依據參觀方向參觀。如進到主寺中，必須脫鞋而且席地而坐，保持安靜，腳板嚴禁朝向佛像方向，且禁止拍照。

玉佛寺位於曼谷 Na Phralan 路上，每日開放時間是8:30—15:30，泰國人參觀免費，外國遊客則收取500[[泰銖|泰銖]]（2013年1月），如衣著不合規定，在買票處有[[紗籠|紗籠]]可以租借。
