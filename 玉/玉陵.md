'''玉陵'''（{{ryu|r=霊御殿|kn=タマウドゥン|rm=Tamaudun}}），又稱'''玉御殿'''或'''靈御殿'''，是[[琉球國|琉球國]][[第二尚氏王朝|第二尚氏王朝]]歷代國王的[[陵墓|陵墓]]。位于現[[沖繩縣|沖繩縣]][[那霸市|那霸市]][[首里|首里]][[金城町|金城町]]。1501年（[[明|明]][[弘治|弘治]]十四年），第三代國王[[尚真王|尚真王]]（1477年至1526年在位）為改葬其父[[尚圓王|尚圓王]]而修建。目前玉陵已經被列為[[世界遺產|世界遺產]]，是琉球最大的[[破風墓|破風墓]]。

== 概要 ==
{{wide image|Tamaudun 02.jpg|1200px|玉陵全景}}

整個玉陵可分為中室、東室和西室3部分。中室為放置葬禮後洗骨前遺骨的房間，東室安放洗骨後的國王、王妃的遺骨，西室安葬其它王室成員。建造物外是被石壁隔開的外庭和中庭，中庭是用珊瑚礁碎片铺成。

在[[第二次世界大戰|第二次世界大戰]]中，玉陵的東室、西室被嚴重破壞。現在能看見的大部分建筑都是第二次世界大戰之后重新建設的。為紀念在第二次世界大戰中死亡的沖繩縣立一中的學生而建立的「一中健兒之塔」就在玉陵附近。

2000年，以「[[琉球王國的城堡以及相關遺產群|琉球王國的城堡以及相關遺產群]]」的一部分而登入[[世界遺產|世界遺產]]名錄<ref>{{cite web|title=世界遗产官网|url=http://whc.unesco.org/en/list/972/accessdate=2016-4-26}}</ref>。除了世界遺產之外，玉陵也是指定史跡，玉陵墓室石牆為[[重要文化財|重要文化財]]。石雕獅子和玉陵碑為縣定有形文化財。
== 玉陵中的被葬者 ==
[[玉陵碑文|玉陵碑文]]記載，根據[[尚真王|尚真王]]在世時的約定，有資格埋葬於玉陵中的人物為尚真王、王太后[[宇喜也嘉|宇喜也嘉]]（{{lang|ryu|おぎやか}}，尚真王之母）、初代[[聞得大君|聞得大君]]音智殿茂金（尚真王妹）、佐司笠按司真鍋樽（尚真王長女）、中城王子[[尚清|尚清]]（後繼位）、今歸仁王子[[尚韶威|尚韶威]]（尚真王第三子）、越來王子[[尚龍德|尚龍德]]（尚真王第四子）、金武王子[[尚享仁|尚享仁]]（尚真王第六子）、豐見城王子[[尚源道|尚源道]]（尚真王第七子）以及上述人員的子孫。然而尚清王繼位之後違反了這個約定，將廢太子[[尚維衡|尚維衡]]（浦添王子朝滿）的遺骨遷葬於玉陵。此後各王子子孫也不能埋葬於此，而是葬於各家族的墳地中。因此玉陵中的埋葬者多為歷代國王和王妃。

1931年，琉球末代王世子[[尚典|尚典]]的夫人祥子（野嵩按司加那志）是最後一位被埋葬於玉陵的人物。

== 玉陵以外的尚家墓所 ==
[[第二尚氏王朝|第二尚氏王朝]]諸王中，第2代[[尚宣威王|尚宣威王]]和第7代[[尚寧王|尚寧王]]沒有被葬在玉陵。尚寧王由於統治期內國家被[[薩摩藩|薩摩藩]]侵佔，認為無顏面對祖宗，遺言不入玉陵，被葬在了[[浦添極樂陵|浦添極樂陵]]（今[[沖繩縣|沖繩縣]][[浦添市|浦添市]]仲間），而[[尚宣威王|尚宣威王]]的墳墓，相傳在今[[沖繩市|沖繩市]]的[[八重島|八重島]]。

此外，[[伊是名玉陵|伊是名玉陵]]位於[[伊是名島|伊是名島]][[伊是名城|伊是名城]]山麓的旁邊，是[[尚稷|尚稷]]夫妻（[[尚圓王|尚圓王]]父母）以及歷代[[祝女|祝女]]的墓所。2005年，尚家21代當主[[尚昌|尚昌]]的長女[[井伊文子|井伊文子]]、22代當主[[尚裕|尚裕]]被葬於此。[[山川之玉陵|山川之玉陵]]則是琉球王室另外的墓所，葬有[[尚泰王|尚泰王]]第七子[[尚時|尚時]]。

尚家第21代當主[[尚昌|尚昌]]的墳墓不在玉陵，而是在[[日本|日本]][[東京|東京]][[東台區|東台區]]上野。
