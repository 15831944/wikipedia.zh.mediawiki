'''-{玉里}-車站'''位於[[台灣|台灣]][[花蓮縣|花蓮縣]][[玉里鎮|玉里鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。本站站房已配合「花東新車站運動」計畫改建完工。

== 車站構造 ==
*[[岸式月台|岸式月台]]、[[島式月台|島式月台]]各一座。
*車站商店：車站設有[[統一超商|7-ELEVEN]]（現因配合站房改建而停止營業中）。
*本站在2014年起所推動的花東新車站改造計畫中，僅屬翻修改造，並非將整個站體拆除重建，由楊炳國建築師事務所與英商奧雅納工程顧問有限公司所規劃設計<ref>[http://visionf2012.pixnet.net/blog/post/29218904-%E7%8E%89%E9%87%8C%E8%BB%8A%E7%AB%99%E4%BB%8B%E7%B4%B9 引領未來—花東新車站成果設計展(玉里車站介紹)]</ref>。。
*本站於2015年開始翻修後，增建電梯，修改月台高度，並在新月台的遮雨棚上方加裝太陽能發電板。

== 營運狀況 ==
*日據時期，窄軌(762mm)花東線於明治43年(1910年)從[[花蓮港|花蓮港]]開始建築，大正8年(1919年)首先完工至玉里(璞石閣)的87.1公里輕便鐵路，並開始營運。
*為銜接[[北迴線|北迴線]]鐵路，自民國71年(1982年)以後，沿線拓軌(1067mm)工程陸續完工，舊花東線廢除。
*目前為[[臺灣鐵路管理局車站等級#一等站|一等站]]，各級列車都有停靠。
*部分臺東=玉里、玉里=花蓮間的[[臺鐵區間車|區間車]]以此站為終點站及始發站。
*除了[[臺東車站|臺東站]]之外，本站為[[臺東線|臺東線]]全線唯一的一等站。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1917年9月22日：「璞石閣驛」開業，本站為花東輕便鐵道第一期計畫的終點站。
*1917年11月1日：改名為「玉里」。
*1926年3月25日：臺東線全線通車，在此舉辦通車儀式。
*1968年，改建完成的鋼筋混泥土站房正式啟用<ref>[http://blog.xuite.net/akao_chen/mntnsea/113472411 《xuite日誌》隨意窩-玉里車站]</ref>。
*2007年3月29日：配合玉里＝東里間改線工程，三民＝東里間自動號誌停用約8小時。
*2016年2月，配合「花東新車站運動」計畫進行改建的車站改造工程，初步完工，被批評單調無趣<ref>[http://news.ltn.com.tw/news/local/paper/939059 花3億拉皮 玉里車站挨批像監獄《自由時報》]</ref>，鎮長力邀藝術家前來彩繪<REF>[https://news.housefun.com.tw/news/article/212217122072.html 優席夫揮筆　玉里車站將變美《好房網》]</REF>。
*2016年6月28日：啟用多卡通刷卡機。
*2016年8月，新站體逢雨漏水，仍未完成驗收<ref>[http://news.ltn.com.tw/news/local/paper/1017210 改建剛完工 玉里車站逢大雨漏水《自由時報》]</ref>。
