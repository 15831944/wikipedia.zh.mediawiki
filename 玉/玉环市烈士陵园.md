'''玉环市烈士陵园'''，又称'''玉环市革命烈士陵园'''，位于中国[[浙江省|浙江省]][[玉环市|玉环市]][[玉城街道_(玉环市)|玉城街道]]的中青山麓上，是一所主要安葬[[第二次国共内战|第二次国共内战]]中玉环阵亡战士的陵园。陵园建于1959年，占地面积2.2万平方米，共有365级台阶。1981年7月以“'''革命烈士纪念塔及墓'''”名义公布为[[玉环市文物保护单位|玉环市文物保护单位]]。

== 历史 ==
[[File:Yuhuan_Martyrs_Memorial_Park_5.jpg|缩略图]]
玉环市烈士陵园由玉环县人民政府（后为[[玉环市|玉环市]]人民政府）决定兴建，经浙江省人民政府批准，旨在纪念[[革命烈士|革命烈士]]，1958年4月20日开始动工，1959年末落成<ref name="玉环县志（民政）"/><ref name ="中国城镇: 浙江卷"/>。陵园内安葬了最少361名烈士，大部分为[[第二次国共内战|第二次国共内战]]期间在玉环阵亡的[[中国共产党|中国共产党]]军人和民兵，其中又以阵亡于玉环沿海岛屿[[披山岛|披山岛]]、[[鸡山岛|鸡山岛]]、[[洋屿|洋屿]]、[[大鹿岛|大鹿岛]]和[[小鹿岛|小鹿岛]]者为主<ref name="玉环胜迹图志"/><ref name="玉环新闻网"/><ref name="玉环县志（文化）"/>。烈士当中，120人的姓名已经无从考究，最少202人并非出身于玉环而在玉环身亡<ref name="玉环新闻网"/><ref name="玉环县志（人物）"/>。

陵园在1981年以“革命烈士纪念塔及墓”名义列入玉环县文物保护单位（后随着玉环成为[[县级市|县级市]]，改为[[玉环市文物保护单位|玉环市文物保护单位]]），并在1995年列入[[台州市爱国主义教育基地|台州市爱国主义教育基地]]<ref name="玉环胜迹图志"/><ref name="玉环文物概览"/>。玉环市民政局在2010年代投入资金，为陵园进行修缮工程，包括增加环境绿化和照明，并先后对陵园不同设施予以拆建和维修<ref name="玉环新闻网2010"/>。

近年来，包括[[章秀成|章秀成]]、[[颜曰春|颜曰春]]在内的多名不幸牺牲的[[人民警察|人民警察]]先后安葬于此。

== 结构 ==
[[File:Yuhuan_Martyrs_Memorial_Park_3.jpg|缩略图]]
玉环市烈士陵园占地2.2万平方米，位于[[玉城街道_(玉环市)|玉城街道]]中青山南麓，依山坡地势由下至上而建，由毕业于杭州美术学院（现[[中国美术学院|中国美术学院]]）雕塑系的玉环人[[郭世桢|郭世桢]]负责设计和施工<ref name="今日玉环"/>。陵园设有一道石制楼梯，由大门位置通往陵园最高处，以便游人上山参观；楼梯台阶共有365级，特意设计成与一年的天数相合，象征人们全年无休地缅怀烈士<ref name ="中国城镇: 浙江卷"/><ref name="玉环胜迹图志"/><ref name="今日玉环"/>。

陵园大门为[[牌坊|牌坊]]，四柱三门，正面有[[谢觉哉|谢觉哉]]题写的“烈士陵园”字样，背面有[[郭沫若|郭沫若]]题写的“永垂不朽”字样<ref name="今日玉环"/>。沿阶梯拾级而上，可到达墓顶平台，正中央处为一座塑像，刻画[[中国人民解放军|中国人民解放军]]军人与民兵的形象，由郭世桢聘请两个玉环百姓雕塑而成<ref name="今日玉环"/>。塑像东侧建有一座仿古休闲凉亭，为重檐五角结构<ref name="今日玉环"/>。

[[File:Yuhuan_Martyrs_Memorial_Park_4.jpg|缩略图]]
山坡上方建有烈士纪念馆，又称为革命文物陈列馆，馆内悬挂[[杨炎宾|杨炎宾]]、[[陈能孝|陈能孝]]、[[陈增佳|陈增佳]]、[[阮禾秀|阮禾秀]]、[[林存邦|林存邦]]等三十多名烈士遗像，并展出着烈士遗物等文物；纪念馆共有五座，连成一排<ref name ="中国城镇: 浙江卷"/><ref name="玉环新闻网"/><ref name="玉环县志（文化）"/><ref name="今日玉环"/>。纪念馆右方有一座纪念碑亭，亭内石碑以大青石制成，碑文赞颂烈士功绩<ref name="玉环新闻网"/><ref name="今日玉环"/>。从纪念馆位置顺坡拾级而上，可到达占地约300平方米的纪念广场（又称墓前平台）；广场用作举行烈士纪念活动，两旁有凉亭建筑，中央设有讲台，台前有[[毛泽东|毛泽东]]话语“成千成万的先烈，为着人民的利益，在我们的前头英勇地牺牲了，让我们高举起他们的旗帜，踏着他们的血迹，前进吧！”的浮雕<ref name="玉环胜迹图志"/><ref name="今日玉环"/>。广场右下方建有一座半圆形坟墓，单独安葬少年烈士[[林森火|林森火]]一人<ref name="今日玉环"/>。主体墓区位于广场上方，依山而建，坟墓排列成弧形，共有三层，每层两列，各层坟墓前方均设有墓道，墓道相通；墓碑以青石制造，镌刻烈士姓名<ref name="玉环新闻网"/>。

陵园最高处称为墓顶平台，建有各种纪念建筑<ref name="玉环胜迹图志"/>。中央位置建有一座高11.5米、边宽2米的纪念塔，塔身呈六角椎形，刻有[[朱德|朱德]]手迹“革命烈士永垂不朽”[[鎏金|鎏金]]字样，塔顶饰有[[紅色五角星|紅色五角星]]<ref name="玉环县志（文化）"/>。纪念塔西边有1989年落成的杨炎宾烈士纪念亭，塔后方有“渡海解放玉环巨幅青石浮雕”以及碑刻<ref name="玉环胜迹图志"/><ref name="今日玉环"/>。
