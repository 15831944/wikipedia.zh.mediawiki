'''玉里野生動物保護區'''及'''玉里野生動物重要棲息環境'''位於臺灣[[花蓮縣|花蓮縣]][[卓溪鄉|卓溪鄉]]，位在[[中央山脈|中央山脈]]主脊東側，西南方毗臨[[玉山國家公園|玉山國家公園]]，西北方連接[[丹大野生動物重要棲息環境|丹大野生動物重要棲息環境]]，為依據《野生動物保育法》公告之野生動物保護區及重要棲息環境。

==歷史沿革==
保護區為[[行政院農業委員會林務局|林務局]]花蓮林管處玉里事業區管轄的國有林地<ref>{{Cite web |url=http://conservation.forest.gov.tw/ct.asp?xItem=59919&ctNode=178&mp=10 |title=玉里野生動物保護區 |accessdate=2013-01-09 |publisher=農委會林務局 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304115246/http://conservation.forest.gov.tw/ct.asp?xItem=59919&ctNode=178&mp=10 |archivedate=2016-03-04 }}</ref><ref>{{Cite web |url=http://conservation.forest.gov.tw/ct.asp?xItem=3359&ctNode=176&mp=10 |title=玉里野生動物重要棲息環境 |accessdate=2013-01-09 |publisher=農委會林務局 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304111316/http://conservation.forest.gov.tw/ct.asp?xItem=3359&ctNode=176&mp=10 |archivedate=2016-03-04 }}</ref>。1973年，林務局為保存[[紅檜|紅檜]]及[[台灣杉|台灣杉]]母樹林，指定面積約132公頃之區域為保護區。1981年保護對象擴及野生動物，至1992年保護區範圍擴大至[[豐坪溪|豐坪溪]]上游。2000年，依據《野生動物保育法》公告為野生動物保護區及野生動物重要棲息環境。

2011年1月8日，保護區管理站在花蓮[[瑞穗鄉|瑞穗鄉]]的瑞穗生態教育館掛牌成立<ref>{{Cite news |url=http://www.libertytimes.com.tw/2011/new/jan/9/today-north11.htm |title=管理站掛牌 守護玉里動保區 |author=花孟璟 |publisher=自由時報 |date=2011-01-09 |accessdate=2013-01-09}}</ref>。館方也對外提供保護區的生態教育資源<ref>{{Cite web |url=http://subtpg.tpg.gov.tw/web-life/taiwan/9808/9808-06.html |title=玉里野生動物保護區生態教育館 |author=林少雯 |work=臺灣月刊 |publisher=臺灣省政府 |date=2009-08 |accessdate=2013-01-09}}</ref><ref>{{Cite web |url=http://rsecology77.blogspot.tw/ |title=瑞穗生態教育館 |accessdate=2013-01-09}}</ref>。
