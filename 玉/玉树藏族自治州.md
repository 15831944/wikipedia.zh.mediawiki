'''玉树藏族自治州'''（{{bo|z=ཡུལ་ཤུལ་བོད་རིགས་རང་སྐྱོང་ཁུལ།|w=yul shul bod-rigs rang-skyong-khul}}），简称'''玉树州'''，是[[中华人民共和国|中华人民共和国]][[青海省|青海省]]下辖的[[自治州|自治州]]，位于青海省西南部，总面积为20.49万平方公里（不含由[[格尔木市|格尔木市]]代管的[[唐古拉山镇|唐古拉山镇]]），总人口为40.17万人，其中[[藏族|藏族]]占97％，农牧业人口21.7万人，自治州首府驻[[玉树市|玉树市]][[结古镇|结古镇]]。2010年4月14日上午7时起，玉树县（现玉树市）发生[[里氏|里氏]]7.1级地震（參見：[[2010年玉树地震|2010年玉树地震]]）。

2010年10月17日15時14分，青海玉樹發生芮氏規模 6.2 地震。<ref>{{cite web|url=http://news.rthk.hk/rthk/ch/component/k2/1291154-20161017.htm |title=青海玉樹州雜多縣6.2級地震 |publisher=香港電臺網站}}</ref>

== 历史 ==

古为[[吐蕃|吐蕃]]孙波如属地。[[元朝|元朝]]归[[吐蕃等路宣慰司|吐蕃等路宣慰司]]管辖，[[明|明]]末[[清|清]]初蒙古[[和硕特部|和硕特部]]入[[青海|青海]]，控制玉树藏族各部头人，赠爵为诸[[台吉|台吉]]。[[晚清|晚清]]朝時由[[西寧辦事大臣|西寧辦事大臣]]直接管辖。[[民国|民国]]时期设置玉树、囊谦、称多3县，县之下千百户制度因袭如故。 

1914年，[[北洋政府|北洋政府]]為勘查甘川邊界，派遣以[[周務學|周務學]]為首，[[周希武|周希武]]、[[牛載坤|牛載坤]]二人為特派員抵達玉樹結古鎮。周希武在玉樹地區，考察九個月，寫出《[[玉樹土司調查記|玉樹土司調查記]]》。1916年，北洋政府根據周務學團隊調查，玉樹被置於青海管轄範圍之內<ref name="一位献身青海的陇南名士">[http://www.qh.xinhuanet.com/2009-11/13/content_18226962.htm 一位献身青海的陇南名士] {{webarchive|url=https://web.archive.org/web/20160304201517/http://www.qh.xinhuanet.com/2009-11/13/content_18226962.htm |date=2016-03-04 }}</ref>。

1949年10月，[[人民解放军|人民解放军]][[青海省军政委员会|青海省军政委员会]]驻玉树特派员办公处成立，正式接管玉树军政大权，辖玉树、称多、囊谦3县，办公处驻玉树县结古。1950年6月，撤销青海省军政委员会驻玉树特派员办公处，成立青海省玉树分区人民行政督察专员公署。

1951年1月，玉树分区人民行政督察专员公署改称玉树专区专员公署。

1951年7月，玉树藏族自治区筹备委员会成立。1951年12月25日玉树藏族自治区召开第一届各族各界人民代表大会，会议决议：撤销玉树专区，成立玉树藏族自治区（专区级）。自治区辖称多、囊谦2县。玉树县地域由自治区直辖

1955年6月，玉树藏族自治区改称玉树藏族自治州，辖玉树（未成立县级机构）、称多、囊谦、治多、杂多、曲麻莱6县。

== 地理 ==
北与[[海西蒙古族藏族自治州|海西蒙古族藏族自治州]]为邻，东与[[果洛藏族自治州|果洛藏族自治州]]相通，东南与[[四川省|四川省]][[甘孜藏族自治州|甘孜藏族自治州]]毗连，南及西南同[[西藏自治区|西藏自治区]]的[[昌都市|昌都市]]和[[那曲地区|那曲地区]]交界，西北角与[[新疆维吾尔自治区|新疆维吾尔自治区]]的[[巴音郭楞蒙古自治州|巴音郭楞蒙古自治州]]接壤。东西长738公里，南北宽406公里。

平均海拨4000米以上，最高点6621米，[[唐古拉山|唐古拉山]]绵延南境，[[可可西里山|可可西里山]]、[[巴颜喀拉山|巴颜喀拉山]]横贯北境，多雪峰冰川。黄河、[[澜沧江|澜沧江]]导源境内，长江上游[[通天河|通天河]]斜贯。气候高寒。

== 经济 ==

1999年全州实现[[国民生产总值|国民生产总值]]5.96亿元，其中，第一产业完成4.0亿元。玉树州是青海省主要畜产品生产基地，畜产品产量约占全省总产量的1/4。

==自然災害==
{{seealso|2010年玉树地震}}
2010年4月14日上午7时起，玉树县发生[[里氏|里氏]]7.1级地震，玉樹州政府、玉樹市政府所在地結古鎮有9成以上民房倒塌。
