'''玉环书城'''是位于[[中国|中国]][[浙江省|浙江省]][[玉环市|玉环市]]的一座图书售卖场，以玉环市[[新华书店|新华书店]]旗舰门市部为基础，共有四层，营业面积达7000平方米，售书30万册，并发展成为一个集购书、阅读、休闲等功能的大型文化综合体。该书城位于玉环市[[玉城街道_(玉环市)|玉城街道]]桃花岭村榴岛文化广场，于2013年12月31日（一说2014年10月15日）与[[玉环影城|玉环影城]]一同开工。2017年1月18日，位于该书城底楼旁的新华书店临时卖场开业，仅一层；2019年1月11日，玉环书城四层全部装修完成，对外试营业。

== 前身 ==
[[File:Yuhuan_Bookstore_10.jpg|缩略图]]
玉环书城的前身是位于玉环市[[玉城街道_(玉环市)|玉城街道]]城中路25号的玉环县新华书店大楼，该大楼内是原来的玉环县新华书店旗舰门市部。玉环在[[第二次国共内战|解放]]前，由于处于[[岛|海岛]]带来的交通闭塞，暂未有书店开设。在此背景下，1950年6月，新华书店温州中心支店派员工，在现在的城中路1号筹建新华书店门市部，但同年因[[中國國民黨|国民党]]残余部队对玉环的袭击而暂停营业。1952年3月，新华书店[[乐清市|乐清]]支店玉环门市部建立。在坎门、楚门等地设立图书代销网点后，1956年4月，新华书店玉环支店正式建立，店址位于今西城路16号。1977年10月1日，位于现城中路25号的九间三层、共850[[平方米|平方米]]的营业大楼正式开张营业。<ref name=":1">{{Cite news|url=http://yhnews.zjol.com.cn/yuhuan/system/2017/01/10/020990213.shtml|title=【视角·记忆】40年书香味浓 搬迁前的玉环县新华书店|author=郑钘予|date=2017-01-16|work=中国玉环新闻网|accessdate=|archiveurl=https://web.archive.org/web/20190421080802/http://yhnews.zjol.com.cn/yuhuan/system/2017/01/10/020990213.shtml|archivedate=2019-04-21}}</ref>

== 建设 ==
[[File:Yuhuan_Bookstore_13.jpg|缩略图]]
2010年以来，玉环政府开始对文化建设工作的重视。为了满足人民群众的精神文化需求，决定在桃花岭村新建一座较大规模的书城及影城。2013年12月31日（一说2014年10月15日<ref name=":0">{{Cite news|url=http://yhnews.zjol.com.cn/yuhuan/system/2015/09/15/019739096.shtml|title=玉环书城、影城年底前竣工验收|author=王一方|date=2015-09-15|work=中国玉环新闻网|accessdate=|archiveurl=https://web.archive.org/web/20190421080814/http://yhnews.zjol.com.cn/yuhuan/system/2015/09/15/019739096.shtml|archivedate=2019-04-21}}</ref>）上午，玉环书城、[[玉环影城|玉环影城]]的开工典礼在桃花岭休闲文化广场（现[[榴岛文化广场|榴岛文化广场]]）工程工地上举行。<ref>{{Cite news|url=http://yhnews.zjol.com.cn/zhuanti/lc/dtbd/201312/t20131231_492072.htm|title=玉环影城、书城项目举行开工典礼|author=董伯志|date=2013-12-31|work=中国玉环新闻网|accessdate=|archiveurl=https://web.archive.org/web/20190421080828/http://yhnews.zjol.com.cn/zhuanti/lc/dtbd/201312/t20131231_492072.htm|archivedate=2019-04-21}}</ref>这一地区原是玉环良种繁育场的破旧厂房，项目建设完成后将成为玉环的标志性文化设施之一<ref>{{Cite news|url=http://yhnews.zjol.com.cn/yuhuan/system/2016/12/17/020944303.shtml|title=跨越发展中的玉环：腾飞！以项目夯实基石|author=朱小兵 詹晓霞|date=2016-12-17|work=中国台州网-台州日报|accessdate=|archiveurl=https://web.archive.org/web/20190421080838/http://yhnews.zjol.com.cn/yuhuan/system/2016/12/17/020944303.shtml|archivedate=2019-04-21}}</ref>。其中，玉环书城这一工程由浙江[[新华书店|新华书店]]集团有限公司投资兴建，项目总投资6550万元，主体建筑为四层，建筑面积19030平方米<ref name=":0" />。

经过三至四年的建设，玉环书城的建设日渐完成。而率先完工的榴岛文化广场吸引市民来此休闲，为了给市民创造一个良好的休闲阅读环境，玉环市新华书店旗舰门市部决定在玉环书城设立一个临时卖场。2017年1月18日，位于城中路的旧店关闭，并整体迁入玉环书城大门左侧的“新华书店临时卖场”。这一临时卖场的面积达到1200平方米，可容纳5万多类书籍。但临时卖场仅供过渡使用。<ref>{{Cite news|url=http://yhnews.zjol.com.cn/yuhuan/system/2017/01/09/020985424.shtml|title=新华书店下周搬“新家”啦 场所更大 环境更优 休闲服务更多|author=路伟凤|date=2017-01-09|work=[[今日玉环|今日玉环]]|accessdate=|archiveurl=https://web.archive.org/web/20190421080854/http://yhnews.zjol.com.cn/yuhuan/system/2017/01/09/020985424.shtml|archivedate=2019-04-21}}</ref>两年后的2019年1月11日，位于玉环市长兴路262号的榴岛文化广场内<ref name=":2">{{Cite news|url=http://dy.163.com/v2/article/detail/E6HNSVGV0514JR9B.html|title=2019，偶遇玉环书城：宇宙是流变，生活是意见|author=|date=2019-01-27|work=玉环旅游<small>（玉环市风景旅游管理局官方账号）</small>|accessdate=|archiveurl=https://web.archive.org/web/20190421080913/http://dy.163.com/v2/article/detail/E6HNSVGV0514JR9B.html|archivedate=2019-04-21}}</ref>的玉环书城正式试营业。

== 现状 ==
[[File:Yuhuan_Bookstore_2.jpg|缩略图]]
玉环书城是一幢较大的四层建筑<ref name=":1" />，主体分为两层及[[夹层|夹层]]。主展厅位于一层与夹层，陈列[[社会科学|社科]]、文艺和生活类书籍，并开设日常杂货、陶艺制品销售区，划分出20个主题展台；二层设有为儿童开设的小柚子儿童馆<ref name=":2" />和柚狮学堂等区块，以及同玉环市青年企业家协会合办的“青商书院”<ref>{{Cite news|url=http://yhnews.zjol.com.cn/yuhuan/system/2019/01/22/031416133.shtml|title=阮聪颖寄语：下好“先手棋” 打好主动仗|author=路伟凤|date=2019-01-22|work=中国玉环新闻网|accessdate=|archivedate=2019-04-21|archiveurl=https://web.archive.org/web/20190421082518/http://yhnews.zjol.com.cn/yuhuan/system/2019/01/22/031416133.shtml}}</ref>；三到四层多是一些课外辅导班、兴趣班等。另外，将[[咖啡|咖啡]]馆、手工制作区、亲子活动区、摄影美术馆等其他内容融入。<ref name=":3" />书架直通天花板，并采用原木色<ref name=":2" />，在将图书按书架号陈列的同时，通过特别策划，对书目种类进行扩充细化，并设置每月更新的阅读榜单和与市民互动的“阅读主张”板块。整体布局宽敞明亮，采用灯光和音乐使得环境自然温和。营业面积较老店增加到7000平方米，售书量也增加到30万册。<ref name=":3">{{Cite news|url=http://yhnews.zjol.com.cn/yuhuan/system/2019/01/14/031398181.shtml|title=大型文化综合体——玉环书城试营业|author=郑茜予 龚松 王凌轩|date=2019-01-14|work=中国玉环新闻网|accessdate=|archiveurl=https://web.archive.org/web/20190421080923/http://yhnews.zjol.com.cn/yuhuan/system/2019/01/14/031398181.shtml|archivedate=2019-04-21}}</ref>

此外，玉环书城采用“文化+休闲+体检+商业”的新模式<ref name=":1" />，与其他多个单位合作，举行一系列平面展览、讲座、读书会等文化交流活动<ref name=":3" />。
