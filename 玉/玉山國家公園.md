'''玉山國家公園'''是[[中華民國|中華民國]]第二座[[中華民國國家公園|國家公園]]，前身為[[台灣日治時期|日治時期]]成立之[[新高阿里山國立公園|新高阿里山國立公園]]（1937–1945）。1985年4月6日公告計畫，1985年4月10日成立管理處，總面積103,121.4公頃，涵蓋台灣本島[[行政|行政]]區域包括[[南投縣|南投縣]]、[[嘉義縣|嘉義縣]]、[[高雄市|高雄市]]以及[[花蓮縣|花蓮縣]]，為台灣陸域面積最大之國家公園。園區位居台灣本島[[中央|中央]]地帶，地理位置獨特，奇峰兀立，雄偉壯麗，為台灣高山少數仍保存原始風貌之地區。其間包括有台灣第一高峰，海拔3952公尺之[[玉山主峰|玉山主峰]]，主峰鄰近地區崇山峻嶺，溪谷深邃，天然植被隨海拔之變化而異，由亞熱帶、溫帶以至寒帶林相次變化，野生動物遍布，並具有清朝所建歷史遺跡[[八通關古道|八通關古道]]；因此，全區蘊藏許多珍貴之生態資源及人文史蹟。

==歷史==
[[File:119k_boundary_tablet.JPG|thumb]]
[[玉山|玉山]]地區早於[[台灣日治時期|日治時期]]，即由台灣總督府設有國立公園委員會，指定為「新高、阿里山國立公園」預定地，1937年12月27日「[[新高阿里山國立公園|新高阿里山國立公園]]」與「[[大屯國立公園|大屯國立公園]]」、「[[次高太魯閣國立公園|次高太魯閣國立公園]]」同時成立，但是三座國立公園因日本戰敗而廢止。

[[臺灣戰後時期|戰後]]不久，[[中華民國政府|中華民國政府]]遷臺，開始著手研擬國家公園法規與設立。《國家公園法》經立法院與內政、司法等二個委員會進行聯席會議12次審查後，最後於1972年5月宣佈三讀通過，並提交至[[中華民國總統|總統]]於同年6月13日公布實施<ref>{{cite web |url=https://npda.cpami.gov.tw/tab1/web1_main.php?mod=C |title=國家公園法公布實施與修法歷程 |work=臺灣國家公園數位典藏主題網 |author= |publisher=內政部營建署 |accessdate=2014-12-07 |language=zh-tw |date= }}</ref>。1979年3月，行政院頒布《[[台灣地區|台灣地區]]綜合開發計畫》，將玉山、[[南橫|南橫]][[檜谷|檜谷]]等地區指定為國家公園預定地<ref name="cpami">{{cite web |url=https://npda.cpami.gov.tw/tab1/web1_main.php?mod=D2 |title=臺灣國家公園發展歷史 玉山國家公園 |work=臺灣國家公園數位典藏主題網 |author= |publisher=內政部營建署 |accessdate=2014-12-07 |language=zh-tw |date= }}</ref>。1983年1月1日，玉山國家公園的區域範圍奉行政院於第1806次院會中核定通過後，內政部開始著手邀請專家學者乃就區域內生態及人文資源之[[實地考察|野外調查]]，分別對地理地形、動物、植物、遊憩及人文史蹟資源委託國立[[國立臺灣大學|台灣大學]]及內政部國家公園計畫委員會審議，並積極展開區域內資源與土地使用之規劃。

1985年2月7日，玉山國家公園計畫遂經行政院於第1921次院會中核定通過，交由內政部公告實施<ref name="cpami"/><ref name="三通1-2"/>。同年4月10日，成立管理處，原計畫設於[[塔塔加|塔塔加]]，因土地問題與台灣大學協商不斷，最後才改設於水-{里}-，並將辦公處所暫時借用於民宅<ref name="園史154頁">{{cite book |title=《臺灣國家公園史：一九〇〇－二〇〇〇》|chapter=〈玉山國家公園新建國家公園訓練中心啟用典禮〉|page=第154頁 |author=游登良 |isbn=9570115149 |publisher=內政部營建署 |language=zh-tw |year=2002 }}</ref><ref>{{cite book |title=《臺灣國家公園史：一九〇〇－二〇〇〇》|chapter=〈葉世文訪談紀錄〉|page=第251-254頁 |author=游登良 |isbn=9570115149 |publisher=內政部營建署 |language=zh-tw |year=2002 }}</ref><ref>{{cite book |title=《東海大學景觀學系88學年度碩士論文：塔塔加地區土地經營管理衝突之研究》|chapter=〈第一節 土地利用衝突背景概述〉|page=第49-60頁 |author=葉文玉 |publisher=東海大學 |language=zh-tw |year=2000 }}</ref>。1992年2月13日，玉山國家公園管理處暨訓練中心新建工程正式動工<ref name="園史154頁"/>。1995年4月11日，玉山國家公園管理處暨訓練中心落成啟用，並展開為期二天的典禮活動，結束10年來一直借用民宅做為辦公處所之歷史<ref name="園史154頁"/>。玉山國家公園管理處暨訓練中心位於現今水里鄉中山路一棟外觀呈白色的大樓，依臨[[水里溪|水-{里}-溪]]，背倚[[濁水溪|濁水溪]]，門前佇立一塊岩石刻題「玉山國家公園管理處 Yushan National Park」。2012年，在玉山國家公園第三次通盤檢討中公告，將梅山地區之原住民保留地給予劃出，園區範圍由105,490公頃減至103,121.4公頃<ref name="三通1-2">玉山國家公園第三次通盤檢討計畫書，1-2頁</ref>。

==地理==
[[File:Kuan-Shan-Lanscape-View.jpg|thumb]]
===範圍===
玉山國家公園東起[[馬利加南山|馬利加南山]]、[[喀西帕南山|喀西帕南山]]、[[玉里山|玉里山]]主稜線，南沿[[新康山|新康山]]、[[三叉山|三叉山]]後沿[[中央山脈|中央山脈]]至[[塔關山|塔關山]]、[[關山|關山]]，西自梅山村西側溪谷順楠溪林道西側稜線至[[鹿林山|鹿林山]]、[[同富山|同富山]]，北沿東埔村1 鄰北側溪谷至[[郡大山|郡大山]]稜線，再順[[哈伊拉漏溪|哈伊拉漏溪]]至[[馬利加南山|馬利加南山]]北峰；計畫範圍內包括[[玉山|玉山]]群峰、[[秀姑巒山|秀姑巒山]]、[[大水窟山|大水窟山]]、[[塔芬山|塔芬山]]、[[雲峰|雲峰]]、[[關山|關山]]及[[南橫公路|南橫公路]]梅山至埡口段<ref name="三通1-2"/>。

國家公園面積共103,121.4公頃，行政區包括[[南投縣|南投縣]][[信義鄉_(臺灣)|信義鄉]]24,163.8公頃、[[嘉義縣|嘉義縣]][[阿里山鄉|阿里山鄉]]1,404公頃、[[高雄市|高雄市]][[桃源區|桃源區]]34,936.9公頃、[[花蓮縣|花蓮縣]][[卓溪鄉|卓溪鄉]]42,616.7公頃<ref name="三通2">玉山國家公園第三次通盤檢討計畫書，第二章</ref>。

===區劃===
[[File:Entrance_to_the_Yushan_National_Park_Headquarters.JPG|thumb]]

[[行政院農業委員會林務局|農委會林務局]]南投、嘉義、花蓮、屏東四個林區管理處所有之土地占國家公園面積逾91%，剩餘面積分別為[[國立臺灣大學|臺大]]實驗林8,328.3公頃及信義鄉東埔村1鄰的原住民保留地574.8公頃，兩者位於園區西北角，共占園區面積8.64%<ref name="三通2"/>。

玉山國家公園區域內共有兩處列為生態保護區域，面積為73,622.34公頃，占國家公園總面積71％，含有豐富的動、植物及地形景觀資源<ref>玉山國家公園第三次通盤檢討計畫書，8-8頁</ref>。特殊天然景緻劃設為特別景觀區者共有九處，包括[[玉山|玉山]]群峰、[[秀姑巒山|秀姑巒山]]、[[達芬尖山|達芬尖山]]、[[塔芬山|塔芬山]]、[[雲峰|雲峰]]、[[新康山|新康山]]、[[三叉山|三叉山]]、[[向陽山|向陽山]]、[[關山|關山]]斷崖、[[南橫公路|南橫公路]]沿線及[[日治八通關越嶺道|日治八通關越嶺道]]等地。成立之初，玉山國家公園管理處將清代修築之[[八通關古道|八通關古道]]及其兩側遺址劃為史蹟保存區，並於園區北部的[[觀高|觀高]]、西北部的[[塔塔加|塔塔加]]、中部的[[梅蘭鞍部|梅蘭]]、西南部的[[天池_(南橫公路)|天池]]、南部的[[梅山村|梅山]]以及東部的[[大分_(台灣地名)|大分]]則分別設置遊憩區，遂經多次調整後，梅蘭、大分等地皆已改劃為生態保護區，梅山則自園區劃出為原住民保留地，其餘仍保持不變。全區依環境資源分為[[玉山國家公園#西北園區|西北園區]]、[[玉山國家公園#南部園區|南部園區]]、[[玉山國家公園#東部園區|東部園區]]、資源核心區等四個區塊<ref name="三通4">玉山國家公園第三次通盤檢討計畫書，第四章</ref>。

====西北園區====
位於西北側，郡大林道以西及[[楠梓仙溪|楠梓仙溪]]以北之地區。包括東埔、觀高、玉山主峰及楠溪林道等<ref name="三通4"/>。可藉[[台21線|台21線]]、[[台18線|台18線]]以及投60線進入，有觀高、塔塔加兩處遊憩區，並設有塔塔加遊客中心及排雲登山服務中心。新中橫公路沿線的[[塔塔加夫妻神木|夫妻樹]]、[[塔塔加|塔塔加]]為該區景點，[[東埔溫泉|東埔溫泉]]便在園區旁。此區另有[[國立臺灣大學|臺灣大學]]實驗林以及[[國立中央大學|中央大學]][[鹿林天文台|鹿林天文台]]等學術研究設施。

====南部園區====
[[File:Meishan_Visitor_Center,_Yushan_National_Park_Headquarters_20031205.jpg|thumb]]
[[File:Paiyun_Lodge.jpg|thumb]]]]
位於西南部，包括[[三叉山|三叉山]]、[[向陽山|向陽山]]、南橫三山、[[梅蘭鞍部|梅蘭]]林道及[[南橫公路|南橫公路]]沿線至關山[[埡口_(南橫公路)|埡口]]隧道等地區<ref name="三通4"/>。園區西側外有[[梅山村|梅山]]遊客服務中心，藉[[台20線|台20線]]南橫公路進入，有[[天池_(南橫公路)|天池]]遊憩區，[[長青祠|長青祠]]、[[檜谷|檜谷]]皆在南橫沿線。

====東部園區====
位於東南側，[[米亞桑溪|米亞桑溪]]與[[樂樂溪|樂樂溪]]連線以東地區，包括[[大分_(台灣地名)|大分]]、[[瓦拉米|瓦拉米]]及[[新康山|新康山]]地區<ref name="三通4"/>。[[台30線|台30線]]僅可到達園區邊界的山風，園區外有[[南安_(台灣地名)|南安]]遊客中心。南安瀑布、山風瀑布、佳心、喀西帕南紀念碑、黃麻溪谷、瓦拉米等多處景點位於此處。

====資源核心區====
位於計畫區東北部及中央地帶，包括郡大林道以東、[[郡大溪|郡大溪]]、中央山脈[[秀姑巒山|秀姑巒山]]至[[三叉山|三叉山]]之間，西以[[荖濃溪|荖濃溪]]為界、東至[[米亞桑溪|米亞桑溪]]與[[樂樂溪|樂樂溪]]連接線<ref name="三通4"/>。本區利用八通關日治越道及登山步道聯絡各地，出入以登山者為主。

===地形===
全區由海拔300餘公尺的[[樂樂溪|樂樂溪]]河谷到3,952公尺的臺灣最高峰[[玉山|玉山]]，地形高而陡峻，3,000公尺以上地區約占全區面積12.74%，2,000公尺至3,000公尺之地區約占54.57%，有39.73%的山坡坡度達46度以上<ref name="三通2"/>。區域範圍橫跨台灣兩大山脈：[[玉山山脈|玉山山脈]]主要山地與[[中央山脈|中央山脈]]南段山地，轄區內屬於[[台灣百岳|台灣百岳]]的名峰林立，總計共有三十座。其中，屬[[玉山山脈|玉山山脈]]者有十一座，屬[[中央山脈|中央山脈]]者有十九座。

===水文===
河流以[[八通關|八通關]]及[[塔塔加|塔塔加]]鞍部為分水嶺，南有[[荖濃溪|荖濃溪]]、[[楠梓仙溪|楠梓仙溪]]、[[拉庫音溪|拉庫音溪]]等[[高屏溪|高屏溪]]上游之主要支流，高屏溪流域的集水區占國家公園面積39%；北為[[陳有蘭溪|陳有蘭溪]]、[[郡大溪|郡大溪]]及[[沙里仙溪|沙里仙溪]]等[[濁水溪|濁水溪]]上游主要支流，占全區20%。發源於[[中央山脈|中央山脈]]東側之[[樂樂溪|樂樂溪]]及其支流則向東流匯入[[秀姑巒溪|秀姑巒溪]]，占全區41%<ref name="三通2"/>。

===氣候===
本區年平均降雨量約3,600毫米左右，全年降雨日數約140日，集中於5月至8月間<ref name="三通2"/>。本園區除5到8月外，其餘各月均下霜；但因風大，除山谷外，其餘各地降霜量並不多。雪期始於10月，止於5月，在海拔2,000公尺以上地區均會下雪，冬季乾旱不明顯時，3,000公尺以上地區的積雪可達四個月。

===交通===
[[File:Tataka.jpg|thumb]]遊憩區]]
[[File:YushanSunrise3.jpg|thumb]]日出]]
[[新中橫公路|新中橫公路]][[水里玉山線|水-{里}-玉山<math>\rightarrow</math>線]]及[[嘉義玉山線|嘉義玉山線]]由西進入國家公園後，於[[塔塔加|塔塔加]]交會<ref name="三通4"/><ref name="三通5"/>。另有[[台20線|台20線]]（[[南橫公路|南橫公路]]）梅山段至埡口貫穿南部園區，此段長約30公里，另外通往東埔一鄰主要聯外道路為投60線鄉道，[[台30線|台30線]]則僅由[[玉里鎮|玉里鎮]]通達東部園區之山風。除上述主要道路外，尚有七條林道。林道主要分布於園區之北、西及南側，供林務單位作森林防火、造林撫育使用。

步道方面，以八通關日治越道及南二段為東西及南北骨幹，有多條路線可供登山健行者選擇，部分路線需先申請才可進入<ref name="三通5"/><ref>玉山國家公園第三次通盤檢討計畫書，9-45頁</ref>。

==生態==
===動物===
據初步調查，本園區至少有58種哺乳類，占臺灣哺乳類物種數之74％，幾乎包含所有臺灣中大型哺乳動物，尤其是[[臺灣野山羊|臺灣野山羊]]、[[臺灣水鹿|臺灣水鹿]]、[[臺灣黑熊|臺灣黑熊]]、[[臺灣野豬|臺灣野豬]]、[[山羌|山羌]]、[[臺灣獼猴|臺灣獼猴]]等為本區常見，[[樂樂溪|樂樂溪]]流域為目前臺灣黑熊族群數量最多之區域<ref name="三通3">玉山國家公園第三次通盤檢討計畫書，第三章</ref><ref>{{Cite web |url=http://www.ysnp.gov.tw/upload/documents/20120204_164348.81440.pdf |title=玉山國家公園區域內野生動物種數 |date=2011 |publisher=玉山國家公園管理處 |format=PDF}}</ref>。本園區鳥類約有191種，其中23種臺灣特有種鳥類皆可在園區發現。本區的爬蟲類與兩棲類數量較少，爬蟲類有蛇類14種，蜥蜴4種，以[[阿里山龜殼花|阿里山龜殼花]]、[[梭德氏遊蛇|梭德氏遊蛇]]及[[斯文豪氏攀木蜥蜴|斯文豪氏攀木蜥蜴]]等數量較多。兩棲類共發現有13種，其中有尾類有[[阿里山山椒魚|阿里山山椒魚]]及[[楚南氏山椒魚|楚南氏山椒魚]]2種，均為臺灣特有種，為世界山椒魚分布帶之最南界；無尾類有11種，以[[盤古蟾蜍|盤古蟾蜍]]、[[莫氏樹蛙|莫氏樹蛙]]及[[褐樹蛙|褐樹蛙]]為臺灣特有種。園區內共紀錄有魚類4科12種，其中有[[臺東間爬岩鰍|臺東間爬岩鰍]]、[[高身鏟頜魚|高身鏟頜魚]]、[[何氏棘魞|何氏棘魞]]、[[臺灣石賓|臺灣石賓]]、[[粗首鱲|粗首鱲]]為臺灣特有種。園區內昆蟲826種，蝴蝶種類達282種，約占全臺蝴蝶種類400種的70%，在東埔溫泉及塔塔加一帶有相當數量。

===植物===
玉山國家公園約87%地區為天然林，在3,600公尺以上為高山植群，內單子葉植物總計426種、雙子葉植物1,469種、裸子植物28種、蕨類植物437種、苔蘚植物177種、菌類植物147種，如:玉山杜鵑、玉山圓柏、玉山薄雪草、台灣冷杉等。

==人文==
===原住民===
園區範圍早期為[[鄒族|鄒族]]與[[布農族|布農族]]的生活領域，於[[陳有蘭溪|陳有蘭溪]]流域與[[樂樂溪|樂樂溪]]流域之瓦拉米、黃麻等遺址附近發現的石器與陶器等史前遺蹟與遺物，證明本地的人類活動至少有一千年的歷史<ref name="三通3"/>。

目前園區內只有西北角的南投縣信義鄉東埔村1鄰有人口集中的聚落，主要為布農族聚居處，設籍約400人<ref name="三通5">玉山國家公園第三次通盤檢討計畫書，第五章</ref>。園區東部的拉庫拉庫溪流域有布農族舊聚落遺址，為[[台灣日治時期|日治後期]]推行大規模集團移住而遭廢置的舊有部落，大致仍維持百年前的規模，是研究東部布農族實質環境的重要地區<ref name="三通3"/>。此處已發現布農族舊建築群51處，建築構造物241處，遺跡成散狀分佈於海拔600至1,900公尺間的山稜上。

===古道===
