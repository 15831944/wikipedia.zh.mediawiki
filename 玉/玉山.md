'''玉山'''，泛指'''玉山主峰'''，[[海拔|海拔]]3,952.43±0.045公尺<ref name="height"/>，為台灣最高峰，百岳之首，位於[[玉山國家公園|玉山國家公園]]。<ref name=ysnp>[http://www.ysnp.gov.tw/resourcesoverview.aspx?id=1 玉山國家公園-地形資源概況] {{webarchive|url=https://web.archive.org/web/20150415225618/http://www.ysnp.gov.tw/resourcesoverview.aspx?id=1 |date=2015-04-15 }}</ref><ref>{{cite web|url=http://www.ysnp.gov.tw/faq_messages.aspx?id=28|title=常見問答玉山的高度為何？|publisher=[[玉山國家公園|玉山國家公園]]|language=zh-tw|deadurl=yes|archiveurl=https://web.archive.org/web/20150620065654/http://www.ysnp.gov.tw/faq_messages.aspx?id=28|archivedate=2015-06-20}}</ref>布農族稱其為 Saviah/Savih、卡那卡那富族稱其為Tanungu'incu、鄒族稱其為Patungkuonʉ（'''[[八通關|八通關]]'''）、排灣族稱其為kanasi，位於[[南投縣|南投縣]][[信義鄉_(臺灣)|信義鄉]]、[[高雄市|高雄市]][[桃源區|桃源區]]及[[嘉義縣|嘉義縣]][[阿里山鄉|阿里山鄉]]交界處。早期西方人稱'''摩里遜山'''（Mt. Morrison）。[[台灣日治時期|日治時期]]時，因日本人發現玉山比[[日本|日本]]本土的[[富士山|富士山]]還要高，於是稱為'''新高山'''。在[[臺灣百岳|臺灣百岳]]中，代表布農的玉山與泰雅的[[雪山_(臺灣)|雪山]]、阿美的[[秀姑巒山|秀姑巒山]]、[[南湖大山|南湖大山]]、排灣的[[北大武山|北大武山]]合稱「五-{岳}-」，為臺灣最具代表性的五座高山。

玉山山容氣勢磅礡、雄踞一方，四季景緻分明而優美；春天[[玉山杜鵑|玉山杜鵑]]盛開、夏季雲海繚繞、秋季則遍布[[法國菊|法國菊]]，冬季則雪白如玉；「玉山積雪」成為[[臺灣八景|臺灣八景]]之一<ref name="勝蹟篇">《重修臺灣省通志》，卷二土地志勝蹟篇，第九頁</ref>。1985年4月（民國74年），玉山群峰周邊劃入玉山國家公園中。玉山是[[布農族|布農族]]與[[鄒族|鄒族]]共同的聖山之一，也是當代臺灣的象徵之一。

== 玉山群峰 ==
[[File:玉山主峰_01.jpg|thumb]]

玉山主峰周圍的群峰包括：
* [[玉山東峰|玉山東峰]]，標高3,869公尺，百岳之一，為台灣第四高峰（百岳排名第三），亦為百岳[[台灣百岳#十峻|十峻]]之首。
* [[玉山北峰|玉山北峰]]，標高3,858公尺，百岳之一，山頂設有玉山氣象站。
* [[玉山南峰|玉山南峰]]，標高3,844公尺，百岳[[台灣百岳#十峻|十峻]]之一。
* [[玉山西峰|玉山西峰]]，標高3,518公尺，百岳之一，山頂留存有[[日治時期|日治時期]]所設置的西山神祠。
* [[玉山前峰|玉山前峰]]，標高3,239公尺。百岳之一。
* [[東小南山|東小南山]]，標高3,744公尺。百岳之一。
* [[南玉山|南玉山]]，標高3,383公尺。百岳之一。
* [[鹿山|鹿山]]，標高2,981公尺。百岳之一。
* [[玉山北北峰|玉山北北峰]]，位於玉山北峰北側，標高約3,833公尺。
* [[玉山圓峰|玉山圓峰]]，或稱圓峰，位於主峰與南峰之間，設有圓峰山屋。標高約3,752公尺。
* [[小南山|小南山]]，標高約3,582公尺。
* [[北玉山|北玉山]]，位於玉山山脈的對關山西南、玉山北北峰西北處，標高約1,849公尺。

此外台灣雪山山脈有座名為「[[阿玉山|阿玉山]]」的山岳，標高約1,419公尺，位於[[宜蘭縣|宜蘭縣]]、[[新北市|新北市]]交界處，鄰近另有座稱作「下阿玉山」的山，此兩山與玉山在地理位置上沒有相關。

玉山群峰周邊目前已劃入1985年4月10日成立的[[中華民國|中華民國]]第二座國家公園—[[玉山國家公園|玉山國家公園]]的特別景觀區之一，為玉山國家公園八個特別景觀區中面積最大的一個，[[新中橫公路|新中部橫貫公路]]經過[[塔塔加遊憩區|塔塔加遊憩區]]，有供過路民眾休憩外，由於該休息區地點適中可以遙望玉山主峰，在近年成為熱門的觀光地點之一。

攀登玉山一般有二條路線，西側的塔塔加鞍部線最多人利用，東側的八通關線也有人利用，但多用為從塔塔加上山後，不想原路下山的另一個選擇，登山界尚曾循南稜的廣東丸山、南玉山，或北稜的沙裡仙溪林道、北玉山路線上登，玉山冬季多會積雪，亦有山友循積雪深厚或結冰堅硬的劍溝以冰攀上登。

玉山主峰、西峰間的鞍部山坳有建一棟[[排雲山莊|排雲山莊]]，是此山區唯一有人管理的山中旅館，惟因攀登玉山人數眾多，山莊容量與腹地有限，使攀登玉山在自然條件上並不困難，反而是取得床位難上加難。

== 山巔 ==
[[File:150._Distant_View_of_Mt._Niitaka_from_Mt._Ari._Formosa..JPG|thumb]]遠望新高山（今玉山）的明信片]]
[[鳥居龍藏|鳥居龍藏]]於1900年首次登頂。玉山主峰於1917年由日本陸軍參謀本部[[陸地測量部|陸地測量部]]設立了一等三角點，1924年[[海拔|海拔]]高度測量為3,950公尺。日治結束後，曾長期採用3,997公尺的標高，現今仍有部分地圖沿用，係起源於1957年美國遠東陸軍製圖局的[[數據|數據]]。1966年時，政府曾在玉山山頂最高點處設置[[于右任|于右任]]的銅像，以補足4,000公尺高度；然而此銅像於1995年11月1日[[玉山于右任銅像破壞事件|遭鋸斷]]，後經黏合，卻又於翌年5月發現整個銅像遭丟入山谷裡，不知所終<ref>[http://www.libertytimes.com.tw/2007/new/feb/13/today-fo5.htm 玉山于右任像斷頭元凶說始末 (自由時報)] {{webarchive|url=https://web.archive.org/web/20120707081835/http://www.libertytimes.com.tw/2007/new/feb/13/today-fo5.htm |date=2012-07-07 }}</ref>。近年來在新技術的測量下，重新測得海拔為3,952公尺，於是玉山國家公園管理處在銅像原址改立巨石，上刻「玉山主峰」及「標高3952公尺」字樣，作為台灣最高點的象徵。

關於切除銅像的人士，前[[台北二二八紀念館|台北二二八紀念館]]館長[[葉博文|葉博文]]於2007年2月出面表示是第一次破壞的參與者之一（另二位參與者為前[[綠色和平電台|綠色和平電台]]主持人[[林長安|林長安]]與[[綠黨_(臺灣)|綠黨]]2002年[[台北市議員|台北市議員]]候選人[[陳朝順|陳朝順]]），並說明旨在還玉山原有風貌，因移除銅像為當時政治氛圍不允之事<ref>[http://www.ettoday.com/2007/02/13/301-2055211.htm 12年前鋸斷玉山于右任銅像]</ref>。至於第二次破壞則尚未有人出面承認。

[[File:TheTopOfJadeMountain.jpg|thumb]]。]]
而-{于}-右任銅像被毀後，《新觀念雜誌》於1998年6月21日改立「願心清如玉，義重如山」碑文重新定義玉山精神。此一碑文被玉山國家公園管理處以有為企業（[[玉山銀行|玉山銀行]]）行銷之嫌為由拆除擱置於旁，被遊客踩破；又以新階段任務邁國際為由，改立「Mt. Jade Main Peak」<ref>[http://www.libertytimes.com.tw/2007/new/jan/16/today-life3.htm 玉山精神毀？雜誌「碑」情請命 (自由時報)]</ref>。

此外，在玉山群峰中的第三高峰[[玉山北峰|玉山北峰]]（海拔3,858公尺）上建有一隸屬於[[中央氣象局|中央氣象局]]的[[玉山氣象站|玉山氣象站]]，是台灣境內海拔位置最高的一棟建築物。

2013年4月4日，登山人士抵達玉山主峰頂，發現大理石製的玉山石碑與英文告示牌嚴重裂開，而且鋼筋外露，連一旁的登山鋼索也被破壞，玉山國家公園管理處獲得通報後，該年4月7日天晴，排雲管理站才派人上山勘察，目視研判可能是雷擊造成，因為玉山氣象站也於該年4月4日這天遭到雷擊，導致觀測電腦的資料處理器故障，只能改採人工紀錄觀測<ref>[http://www.tvbs.com.tw/news/news_list.asp?no=carlito1120130408125603&&dd=2013/4/8%20%A4U%A4%C8%2001:03:02 轟！雷劈玉山　主峰石碑歪斜、基座裂（TVBS新聞）]，黃茂松、廖佩倫，2013年4月8日刊載，2013年4月9日查閱</ref><ref>[http://udn.com/NEWS/NATIONAL/NATS2/7815226.shtml 玉山主峰石碑被雷劈壞了 (聯合新聞網)]，李承穎，2013年4月8日刊載，2013年4月9日查閱</ref>，現已修復<ref>[http://www.ysnp.gov.tw/news_messages.aspx?id=1671 玉山主峰「精神石碑」已修復原貌！（玉山國家公園）] {{webarchive|url=https://web.archive.org/web/20141030030453/http://www.ysnp.gov.tw/news_messages.aspx?id=1671 |date=2014-10-30 }}，2013年4月26日刊載</ref>。

== 名稱演變 ==
[[File:Nitaka_shi.jpg|200px]]

玉山是[[台灣原住民|台灣原住民]]中[[布農族|布農族]]與[[鄒族|鄒族]]共同的聖山。布農族稱其為'''Saviah / Savih'''、'''Tongku Saveq'''（意為高大的山）、[[卡那卡那富族|卡那卡那富族]]稱其為'''Tanungu'incu'''、鄒族稱其為'''Patungkuonʉ'''（意為發亮的山或[[石英|石英]]之山；漢語音譯為「[[八通關|八通關]]」）<ref>[http://523.org.tw/old/article/94/day_05WI01.htm 畫說名山－－玉山 - 山林札記]</ref>。'''玉山'''一詞最早見於[[郁永河|郁永河]]的《[[裨海紀遊|裨海紀遊]]》，而在清康熙年間編篆的《[[台灣府志|台灣府志]]》有記述：「玉山，在鳳山縣。山甚高，皆雲霧罩于其上，時或天氣光霽，遙望皆白石，因名為玉山。」，可見玉山作為漢名，已開始流通。《[[雲林縣採訪冊|雲林縣採訪冊]]》中記載：「八通關山又名玉山。......終日皆履雪地。即彰化志所稱雪山也。」，可見玉山又稱雪山，玉山似乎並非統一的山名。

對於西方而言，早期認為玉山最早在文獻上的記載發生於1857年，一艘美國商船[[亞歷山大號|亞歷山大號]]（USS Alexander）由[[安平港|安平]]離港時，船長摩-{里}-遜（W.Morrison）見到高聳的玉山之後將其記載在航海日誌中，因此有摩-{里}-遜山（'''Mt. Morrison'''）這稱呼，並廣為通行於西方諸國，但此說並不合理，因於1845年由英國海軍部海道測量局出版之航海地圖（'''Formosa Island'''）中已標注Mt. Morrison之名，即使是因摩-{里}-遜而命名也必定早於1845年，因此另有一說，認為Mt. Morrison的名稱由來，是為了紀念英國第一位於1807年來華傳教的牧師[[馬禮遜|馬禮遜]]（Robert Morrison）<ref>{{cite book |last=傳傑|first=陸 |year=2014 |title=被誤解的台灣老地名：從古地圖洞悉台灣地名的前世今生|publisher=遠足文化 | isbn=9789865787486 |pages=176-183}}</ref><ref>{{cite book |publisher = J. Murray |publication-place = London |ol=25302448M |title = A Handbook for Travellers in Japan |publication-date = 1903 |last1=Chamberlain |first1=B. |authorlink1=Basil Hall Chamberlain |author2=Mason, W.B. |edition=7th |page=554}}</ref>。

1895年，台灣因[[清朝|清朝]]敗戰簽訂[[馬關條約|馬關條約]]割讓給[[日本|日本]]，使得玉山也成為日本領土。[[日本陸軍|日本陸軍]]參謀本部陸地測量部1895年7月開始進行臺灣的測量（1896年9月完成），發現玉山的高度超越了日本的第一高峰——海拔3,776公尺的[[富士山|富士山]]。因此在1897年6月28日，由[[拓務省|拓殖務省]]發布告示，將摩里遜山更名'''新高山'''（{{jpn|j=新高山|hg=にいたかやま|rm=Niitakayama|f=y}}）<ref name="陳">{{cite journal|author=陳文添 |title=〈「新高山」名稱探源〉|journal=《臺灣文獻別冊》10 |date=2004-09-30 |publisher=國史館臺灣文獻館 |pages=37─42頁}}</ref>，意指日本領土新的最高峰，周圍群峰一併更名，主峰又稱「新高主山」，也稱作「嘉義新高」，並且在山頂上設立[[神社|神社]]「新高祠」。稍後，1923年時亦將[[雪山_(臺灣)|雪山]]命名為「[[次高山|次高山]]」，意指第二高峰。當時的新高山與[[次高山|次高山]]（[[雪山_(臺灣)|雪山]]）、[[能高山|能高山]]並稱「台灣三高」。

根據[[日本海軍|日本海軍]]的傳統，[[重型巡洋艦|重型巡洋艦]]向來以山峰為名，因此日本海軍聯合艦隊中也有命名為「[[新高號防護巡洋艦|新高]]」的戰艦存在。傳言中1941年12月2日下令[[珍珠港事件|攻擊珍珠港]]的密碼「<span lang="ja">ニイタカヤマノボレ一二○八</span>」（攀登新高山一二○八）中的新高山就是指玉山<ref>山川新作，《空母艦爆隊―艦爆塔乗員死闘の記録》</ref>。

[[臺灣戰後時期|中華民國接管臺灣之後]]，在1947年12月1日宣佈山名訂為玉山。

== 地理特色 ==
[[File:YushanSunrise3.jpg|thumb]]
玉山主峰山貌高峻，四面皆是陡壁危崖，南北兩側是千仞峭壁，西側絕壑深溝，東側則是碎石陡坡。玉山無論山容或山勢皆在台灣為最具規模，除了是台灣五-{岳}-之首、百岳之王外，更重要的是玉山群峰地區蘊含著珍貴的生命寶藏。這裡有[[亞熱帶|亞熱帶]]、[[暖溫帶|暖溫帶]]、[[冷溫帶|冷溫帶]]及[[高山寒原帶|高山寒原帶]]的不同氣候型態，衍生出多樣化的動物群種及植物林相，生態資源相當豐富。

除此，根據日治時期學者[[鹿野忠雄|鹿野忠雄]]、佐佐保雄及近代[[齊士崢|齊士崢]]等人的研究，發現玉山保存有冰河地形遺跡。根據統計，包含「U形谷」、「冰斗」、「側冰磧」等，其中以玉山至八通關的冰河地形最為豐富。<ref>《科學人》，2008年第77期7月號</ref><ref>[http://club.ntu.edu.tw/~mtclub/database/%A5x%C6W%B0%AA%A4s%A6B%AAe%A6a%A7%CE%AA%BA%B7s%B5o%B2%7B.doc http://club.ntu.edu.tw/~mtclub/database/%A5x%C6W%B0%AA%A4s%A6B%AAe%A6a%A7%CE%AA%BA%B7s%B5o%B2%7B.doc]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==題詠==
<poem>
*望玉山（淸•[[周鍾瑄|周鍾瑄]]）
浮嵐高捲日初生，一片晴光照眼明。
積雪不消三伏後，層冰常訝四時成。
疑他匹練非吳市，遮莫胥濤向越城。
大璞已教天地鑿，山靈穩臥不須驚。
*玉山歌 （淸•[[陳夢林|陳夢林]]）
須彌山北水晶宮，天開圖畫自璁瓏。
不知何年飛海東，幻成三箇玉芙蓉。
莊嚴色相儼三公，皓白鬚眉冰雪容。
夾輔日月拄穹窿，俯視眾山皆群工。
帝天不許俗塵通，四時長遣白雲封。
偶然一見杳難逢，惟有霜寒月在冬。
靈光片刻曜虛空，萬象清明曠發蒙。
須臾雲起碧紗籠，依舊虛無縹緲中。
山下螞蟥如蟻叢，蝮蛇如斗捷如風。
婆娑大樹老飛蟲，攢肌吮血斷人蹤。
自古未有登其峰，於戲！雖欲從之將焉從。
*望玉山歌（淸·章甫）<ref>志云：大武巒山後障三峰並列，終歲雪封如紗籠香篆，冬日晴明乃得見，頃則雲霧復合。又陳夢林記云：山北與水沙連內山錯。
</ref>
天蒼蒼，海茫茫。
武巒後，沙連旁。
半空浮白，萬島開張。
非冰非水，非雪非霜。
老翁認得眞面目，云是玉山發異光。
山上寶光山下照，萬丈清高萬丈長。
晴雲展拓三峰立，一峰獨聳鎮中央。
須臾變幻千萬狀，晶瑩摩盪異尋常。
四時多隱三冬見，如練如瀑如截肪。
駭目驚人不一足，莫辨璧圓與圭方。
我聞輝山知韞玉，又聞採玉出崑岡。
可求猶是人間寶，爭似此山空瞻望。
當時有客癡山鑿，自恃雄心豪力強。
豈知愈入愈深處，歸於無何有之鄉。
嗟乎玉山願望幾曾見，我今何幸願為償。
償來願望亦造化，多謝山靈不可忘。
山靈歸去將誰說，依舊囊紗而篆香。
大璞自然天地秘，未知韞匵何處藏。
且將一片餘光好，袖來寶貴入詩囊。
</poem>
