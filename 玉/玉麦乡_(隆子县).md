'''玉麦乡'''（{{bo|z=ཡུལ་སྨད་|w=yul smad}}），是{{PRC admin/fullname|54/22/31/208/000|3}}下辖的一个[[乡_(中华人民共和国)|乡]]。<ref>{{PRC admin/ref sgc|{{PRC admin/parent|54/22/31/208/000}}}}</ref>[[麦克马洪线|麦克马洪线]]穿越玉麦乡南部，线以东区域是[[藏南地区|藏南]]的一部分，为印度所占领。

玉麦乡全乡境域面积3644平方公里，中国实际控制面积1987平方公里，群众实际活动范围300平方公里。全乡仅有9户居民、32人，为中华人民共和国人口最少的乡。<ref>{{cite news|url=http://news.sina.com.cn/c/nd/2016-11-01/doc-ifxxfuff7417908.shtml|title=中国人口最少的乡仅32人 为何引来西藏书记调研|date=2016-11-01|accessdate=2016-11-01}}</ref>

首任乡长桑杰曲巴与其女儿卓嘎、央宗姐妹三人在玉麦放牧守边直到1996年有外人迁入<ref>{{cite news|url=http://xz.people.com.cn/n2/2017/1128/c138901-30971410.html|title=卓嘎、央宗姐妹一家几代人在玉麦放牧守边的故事|accessdate=2017-11-28|work=人民网}}</ref>，其事迹被大量报道。2018年3月1日，卓嘎、央宗姐妹被选为2017年[[感动中国年度人物|感动中国年度人物]]。

== 历史沿革 ==
*1951年以前，属隆子宗；1951年，[[中国人民解放军|中国人民解放军]]入藏后，设玉门乡，属扎日区。
*1960年，玉门乡改名玉麦乡。
*1969年，玉麦乡改为玉麦公社。
*1979年，随扎日区划归隆子县。
*1984年，再次改为玉麦乡，属隆子县扎日区。
*1999年，扎日区撤销，玉麦乡直属隆子县。
*2009年，玉麦乡人民代表大会挂牌成立。
*2011年，玉麦公安边防派出所成立。

== 人口 ==
 
*截止2017年，全乡共32人，在校就读学生12人（大学4名、毕业1名、高中6名、小学2名）、残疾儿童1名（未上学）、学龄前儿童2名、成年人16名（男8人、女8人）。

== 经济 ==
2016年全乡农牧民人均纯收入达55752元，比2015年的26202元增长112.7%，位列隆子县第一。<ref>{{cite news|title=习近平牵挂玉麦乡|url=http://politics.people.com.cn/n1/2017/1102/c1001-29621712.html|accessdate=2017-11-02|work=人民日报海外版}}</ref>
