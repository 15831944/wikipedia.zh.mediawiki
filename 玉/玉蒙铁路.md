'''玉蒙铁路'''纵跨[[中国|中国]][[云南省|云南省]][[玉溪市|玉溪市]]和[[红河州|红河州]]，是规划建设的[[泛亚铁路_(中国)|泛亚铁路]]东线的重要组成部分。线路自玉溪南站引出，经[[通海|通海]]、[[建水|建水]]到达[[蒙自市|蒙自市]]，全长约141公里，设计时速120 km，批复投资44.69亿元，是国家I级电气化铁路，于2005年12月15日开工。全线隧道35个、桥梁61座，桥隧共计77．84公里，占线路总长度54．95%。控制性工程为秀山隧道、柿花树隧道和汉邑村隧道。<ref>{{Cite web |url=http://www.yn.xinhuanet.com/topic/2011-01/18/content_21894826.htm |title=云南在建铁路项目简介 |date=2011年1月18日 |accessdate=2012年12月23日 |publisher=昆明铁路局宣传部 |deadurl=yes |archiveurl=https://www.webcitation.org/68qrM645d?url=http://www.yn.xinhuanet.com/topic/2011-01/18/content_21894826.htm |archivedate=2012年7月2日 }}</ref>2012年8月14日上午10时16分，铁路全线铺通。<ref>{{Cite web|url=http://news.xinhuanet.com/politics/2012-08/15/c_123585655.htm |title=玉蒙铁路全线铺通 |date=2012年8月15日 |accessdate=2012年12月23日 |publisher=新华网 }}</ref>2013年2月23日，玉蒙铁路正式开通运营。先期开通货运<ref>{{Cite web |url=http://yn.yunnan.cn/html/2013-02/24/content_2628447.htm |title=玉蒙铁路昨日通车运营 今年5月开通客运 |date=2013年2月24日 |accessdate=2013年2月26日 |publisher=新华网 |archiveurl=https://web.archive.org/web/20160306114237/http://yn.yunnan.cn/html/2013-02/24/content_2628447.htm |archivedate=2016年3月6日 |deadurl=yes }}</ref>，并于4月28日开行[[昆明站|昆明站]]至[[蒙自北站|蒙自北站]]的客运列车。<ref>{{Cite web |url=http://yn.yunnan.cn/html/2013-04/27/content_2710363.htm |title=昆明至蒙自火车票49.5元 火车汽车出行优缺点PK |date=2013年4月27日 |accessdate=2013年4月27日 |publisher=新华网 |archive-url=https://web.archive.org/web/20130526223310/http://yn.yunnan.cn/html/2013-04/27/content_2710363.htm |archive-date=2013-05-26 |dead-url=yes }}</ref>
