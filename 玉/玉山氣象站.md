'''玉山氣象站'''，全銜'''交通部中央氣象局玉山氣象站'''，為[[中華民國|中華民國]][[交通部中央氣象局|交通部中央氣象局]]所屬的一個[[氣象站|氣象站]]，位於[[玉山北峰|玉山北峰]]峰頂，[[海拔|海拔]][[標高|標高]]3844.784公尺，不但是[[臺灣|臺灣]]海拔最高的氣象站，也是臺灣海拔最高的建築物。該站的通信地址設於[[嘉義縣|嘉義縣]][[阿里山鄉|阿里山鄉]]東阿里山113號，實際[[門牌號碼|門牌號碼]]為[[南投縣|南投縣]][[信義鄉_(臺灣)|信義鄉]][[東埔村|東埔村]]一鄰玉山北峰1號<ref>台灣地理百科叢書系列 《台灣的國家公園》 2002年 衛宏晉著 遠足文化 ISBN 9868015413</ref><ref>《台灣氣象傳奇》 2007年 洪致文著 玉山社 ISBN 9789866789106</ref>。該站主要功能為觀測及記錄各種[[天氣預報|天氣預報]]、特報資料及突變天氣之查詢供應，並主動即時將各種氣象紀錄回傳給位在[[臺北市|臺北市]]的中央氣象局本部，同時也提供[[登山客|登山客]]預約參訪及急難救助。

== 歷史 ==
[[File:玉山主峰_01.jpg|thumb]]

*1943年10月1日，[[臺灣日治時期|日治時期]]末期，[[臺灣總督府|臺灣總督府]]將[[鹿林山|鹿林山]]山麓的氣象觀測站遷至玉山北峰，設立「'''臺灣總督府新高山氣象台'''」。
*1945年，[[臺灣戰後時期|臺灣光復]]，[[中華民國國民政府|中華民國國民政府]]正式接收台灣後，於該年11月改稱為「'''臺灣省氣象局玉山測候所'''」，隸屬於[[臺灣省行政長官公署教育處|臺灣省行政長官公署教育處]]。
*1947年9月，改稱為「'''臺灣省氣象所玉山測候所'''」，隸屬於[[臺灣省政府交通處|臺灣省政府交通處]]。
*1971年7月，改稱為「'''中央氣象局玉山氣象測站'''」，並核定為四等氣象測報機構（即四等站）。
*1989年8月1日，全銜定為「'''交通部中央氣象局玉山氣象站'''」，仍屬四等站。
*1998年，由於房舍老舊，玉山氣象站開始分兩期進行改建工程，至2000年完工。<ref>{{Cite web |url=http://www.newtaiwan.com.tw/bulletinview.jsp?bulletinid=18321 |title=黃木壽 玉山築夢的建築商 - 新台灣新聞週刊 2004.06.29 |accessdate=2013-10-16 |archiveurl=https://web.archive.org/web/20131017031119/http://www.newtaiwan.com.tw/bulletinview.jsp?bulletinid=18321 |archivedate=2013-10-17 |deadurl=yes }}</ref>

== 編制 ==
玉山氣象站在交通部中央氣象局附屬測報機構的編制中屬於四等氣象站，現有員額計4員（置主任1員、職員3員）。[[李臺軍|李臺軍]]曾在該站任職。
