'''玉皇塔'''，位于[[北京市|北京市]][[房山区|房山区]][[大石窝镇|大石窝镇]]高庄村北，[[云居寺|云居寺]]东南，是[[北京市文物保护单位|北京市文物保护单位]]。

==简介==
玉皇塔始建于[[辽朝|辽朝]]。坐北朝南。为八角形七级密檐式砖结构塔。高约15米。塔座是八角[[须弥座|须弥座]]式，上面装饰着砖雕人物故事、动物图象等等；塔的中心设有八角柱，塔身正面设有券门可进入塔内。门内曾供[[汉白玉|汉白玉]][[玉皇大帝|玉皇大帝]]雕像。塔身东、西、北面设有砖雕假门，其他四个斜面设有砖雕假窗。再向上是密檐七层塔檐，各角角梁都悬挂着铜铃。塔顶是八角攒尖式，八条脊从封顶的铁球向八个方向延伸，尽头设有垂兽套兽<ref>{{Cite web |url =  http://www.bj.chinanews.com/news/2005/2005-12-29/1/8339.html|title = 北京古塔 |author =  |date = 2005-12-29 |publisher = 北京网 |language =  |accessdate =  |quote =  }}</ref>。

1995年，玉皇塔被公布为第五批[[北京市文物保护单位|北京市文物保护单位]]<ref name="北京市第五批文物保护单位">{{cite web|title=第五批文物保护单位|url=http://govfile.beijing.gov.cn/Govfile/front/content/11995037_0.html|accessdate=2017-08-30|deadurl=yes|archiveurl=https://web.archive.org/web/20170831001438/http://govfile.beijing.gov.cn/Govfile/front/content/11995037_0.html|archivedate=2017-08-31}}</ref>。2004年4月到7月，房山区进行了[[周吉祥塔|周吉祥塔]]、玉皇塔、[[照塔|照塔]]和[[应公长老寿塔|应公长老寿塔]]四座古塔与[[万佛堂、孔水洞石刻及塔|万佛堂]]修缮工程<ref>{{cite web|title=房山四塔、万佛堂等修缮工程正在进行|url=http://www.huaxia.com/zhwh/whxx/2004/06/84363.html|publisher=华夏经纬网|date= 2004-06-17|accessdate=2017-08-21}}</ref>。
