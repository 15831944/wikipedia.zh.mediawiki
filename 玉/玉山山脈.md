[[File:Taiwan-Yushan_Range.jpg|thumb]]
'''玉山山脈'''為[[台灣五大山脈|台灣五大山脈]]之一。位於[[南台灣|南台灣]]，北起南投縣[[水里鄉|水里鄉]][[濁水溪|濁水溪]]南岸，南抵[[高雄市|高雄市]]的[[六龜區|六龜區]][[十八羅漢山|十八羅漢山]]附近，長約180[[公里|公里]]，是[[台灣五大山脈|台灣五大山脈]]中最短者。玉山山脈的東側為[[中央山脈|中央山脈]]南端，兩者隔荖濃溪為界；西側則隔著[[旗山溪|旗山溪]]，與[[阿里山山脈|阿里山山脈]]為鄰。由於[[台灣日治時期|日治時代]][[日本人|日本人]]發現此山比[[富士山|富士山]]還高，為當時[[日本國|日本國]]境內最高峰，被稱為'''[[新高山脈|新高山脈]]'''。玉山山脈對[[台灣|台灣]]的代表性可從[[中華民國|中華民國]][[新台幣|新台幣]][[1000元|1000元]]大鈔上發現

==特色==
[[File:Mount_Yu_Shan_-_Taiwan.jpg|thumb]]
玉山山脈的走向先是東西向，再轉為北北東—南南西方向，最高峰[[玉山|玉山]]（又稱[[玉山主峰|玉山主峰]]）海拔達3952公尺，是台灣第一高峰，也使[[臺灣島|臺灣島]]成為世界地勢高度第四高的島嶼。著名的[[台灣百岳|台灣百岳]]高峰中，玉山山脈名列前十名的就有五座，如[[玉山主峰|玉山主峰]]（3952公尺）為第一高峰、[[玉山東峰|玉山東峰]]（3869公尺）為台灣百岳第三高峰、[[玉山北峰|玉山北峰]]（3858公尺）為台灣百岳第四高峰、[[玉山南峰|玉山南峰]]（3844公尺）為台灣百岳第五高峰、[[東小南山|東小南山]]（3711公尺）為台灣百岳第九高峰。這些高山皆位於[[玉山主峰|玉山主峰]]周邊，合稱[[玉山#玉山群峰|玉山群峰]]，隸屬[[玉山國家公園|玉山國家公園]]管理局轄區。

==形成== 
[[File:Yushan_satellite_photo.jpg|thumb]]
玉山山塊因[[歐亞板塊|歐亞]]、[[菲律賓海板塊|菲律賓海板塊]]相擠撞而高隆，主稜脈略呈十字形，南北長而東西短，十字之交點即為[[玉山主峰|玉山主峰]]，區域範圍位居台灣中央地帶，地跨[[花蓮|花蓮]]、[[高雄|高雄]]、[[南投|南投]]、[[嘉義|嘉義]]4縣市4鄉10村，地形可概分為[[東埔山塊|東埔山塊]]、[[玉山山塊|玉山山塊]]及[[中央山塊|中央山塊]]三大區，是[[濁水溪|濁水溪]]、[[高屏溪|高屏溪]]及[[秀姑巒溪|秀姑巒溪]]三大河川最重要的上游[[集水區|集水區]]<ref>{{Cite web |url =http://www.ysnp.gov.tw/aboutus.aspx?id=1 |title =關於玉山國家公園 |deadurl =yes |archiveurl =https://web.archive.org/web/20110611163932/http://www.ysnp.gov.tw/aboutus.aspx?id=1 |archivedate =2011-06-11 }}</ref>。

由於組成玉山山脈組成岩層的地質年代較老<ref>{{Cite web |url =http://web2.nmns.edu.tw/PubLib/NewsLetter/101/298/a-6.pdf |title = 閱讀山脈—臺灣的山脈與地質摘要}}</ref>；而隔鄰的[[阿里山山脈|阿里山山脈]]的岩層則大多是地質年代較年輕的[[砂岩|砂岩]]、[[頁岩|頁岩]]和[[砂頁岩|砂頁岩]]互層<ref>{{Cite web |url = http://www.ali-nsa.gov.tw/Article.aspx?sno=04006390 |title = 肆、大地資源}}</ref>，從地質年代上連續的特性以及兩條山脈間多條組條大斷層的地質證據推論，阿里山山脈原本位於玉山山脈之上<ref>{{Cite web |url =http://big5.taiwan.cn/twzlk/twgk/dl/201012/t20101223_1661345.htm |title = 阿里山山脈}}</ref>，因東方板塊強烈擠壓而斷裂，上方較年輕的岩層往西滑動，形成了阿里山山脈，下方較老的岩層則形成玉山山脈。
