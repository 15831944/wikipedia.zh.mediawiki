==形態==
玉葉金花是一種常綠攀援狀[[灌木|灌木]]植物，嫩枝披棕褐色貼伏狀短柔毛，成長後柔毛脫落，常高約1－2[[米|米]]，高可達5米。[[葉|葉]]卵狀披針形或卵狀長圓形，薄紙質或膜質，單葉對生或近輪生，先端漸尖，基部稍偏斜或平截，全緣，表面深綠色，近無毛或披疏毛，背面淡綠色，密披白色短柔毛，長約5－9厘米，寬約2－4厘米；[[葉脈|葉脈]]披毛，側脈約7對，中脈及側脈隆起；
[[葉柄|葉柄]]密披灰褐色粗柔毛，長約3－8毫米；[[托葉|托葉]]三角形，2深裂，長約5－7毫米，頂端2深裂，裂片條形，披柔毛，長約4－6毫米。[[花|花]]為聚傘[[花序|花序]]，頂生，密生多花，無總[[花梗|花梗]]或具極短的梗；苞片線形，披硬毛，長約4毫米；花5枚，花無梗或具極短的梗；[[花萼|花萼]]管陀螺形，外披柔毛，長約2－4毫米，萼簷5深裂，裂片線形，通常比花萼管長2倍以上，基部密披白色柔毛，向上漸稀疏，長約3－4毫米；有些花其中的1－2枚萼裂片會增大變成葉狀裂片，有時缺失，闊卵形至長圓形，白色，頂端短尖或鈍，基部狹窄，有縱脈5－7條，長約2.5－4厘米，寬與長近相等，柄兩面披柔毛，長約1－2.8厘米；[[花冠|花冠]]黃色，冠管柔弱，外披貼伏短柔毛，內面喉部密披棒狀毛，長約2厘米；花冠裂片5枚，長圓狀披針形，頂端漸尖，鑷合狀排列，內面密生金黃色粉末狀小凸點，長約4毫米；[[雄蕊|雄蕊]]5枚，著生於花冠喉部，[[花絲|花絲]]極短，[[雌蕊|雌蕊]]1枚，[[柱頭|柱頭]]2歧；[[子房|子房]]2室。[[果|果]]為肉質[[漿果|漿果]]，近球形，頂部有[[萼檐|萼檐]]脫落後的環狀疤痕，疏披柔毛，成熟時轉為黑紫色，長約8－10毫米，寬約6－7.5毫米；[[果柄|果柄]]疏披毛，長約4－5毫米<ref name = "浸會大學"/><ref name = "曾文水庫旅遊網"/><ref name = "台北植物園"/><ref name = "南方藥用植物圖鑒"/><ref name = "海南植物誌 "/><ref>[http://www.cvh.org.cn/difangzhi/guizhou/page/6/254.pdf 貴州植物誌]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}，第254－256頁</ref><ref>[http://www.cvh.org.cn/difangzhi/zhejiang/page/6/100.pdf 浙江植物誌]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}，第100－101頁</ref><ref name = "華南植物園">[http://www.scib.cas.cn/xwzx/kply/201004/t20100407_2815393.html 小野花也有春天：玉葉金花]中國科學院華南植物園</ref>。

== 分佈 ==
分佈於[[中國|中國]]、[[台灣|台灣]]中北部、[[日本|日本]]等地，原產於[[亞熱帶地區|亞熱帶地區]]。中國國內分佈於[[福建|福建]]、[[廣東|廣東]]、[[香港|香港]]、[[廣西|廣西]]、[[海南|海南]]、[[四川|四川]]、[[江西|江西]]、[[湖南|湖南]]及[[浙江|浙江]]等省份。多生長於山麓至低海拔山區、溪谷、灌叢或村旁等地<ref name = "浸會大學"/><ref name = "台北植物園"/>。

==醫藥用途==
{{medical}}
玉葉金花的枝葉入藥，味甘、微苦，性凉，有毒，歸膀胱、肺及大腸經，[[藥材|藥材]]主產於[[福建|福建]]、[[廣東|廣東]]、[[台灣|台灣]]等地，[[中醫|中醫]]歸類為[[解表藥|解表藥]]、[[利濕藥|利濕藥]]。中藥名為玉葉金花或山甘草，具[[解表|解表]]，清熱利濕、解毒消腫、清涼消暑、活血化瘀等功效，主治[[感冒|感冒]]、[[發熱|發熱]]、[[咳嗽|咳嗽]]、[[咽喉|咽喉]]腫痛、咽喉炎、[[支氣管|支氣管]]炎、[[中暑|中暑]]、[[痢疾|痢疾]]、洩瀉、[[腎炎|腎炎]]水腫、瘡瘍膿腫、[[子宮|子宮]]出血、濕熱小便不利及毒蛇咬傷等。根與藤能治腸胃疾病及預防[[流感|流感]]等功效。[[福建|福建]]民間利用玉葉金花的枝葉煎劑作為[[墮胎|墮胎]]或[[避孕|避孕]]之用。[[廣西|廣西]]民間於夏季時熬製玉葉金花作解毒[[涼茶|涼茶]]及防暑之用。<ref name = "浸會大學"/><ref name = "台灣教育部"/><ref name = "南方藥用植物圖鑒"/><ref name = "華南植物園"/><ref>[http://www.tcmlib.com/zy/html23/showdetail-323232362ce5b1b1e79498e88d892c7a79.html 山甘草]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}意古中醫</ref><ref name = "中科院鼎湖山站">[http://www.efloras.org/florataxon.aspx?flora_id=620&taxon_id=200022166 玉叶金花]中科院鼎湖山站植物名錄</ref>

==相似物種==
*[[散玉葉金花|散玉葉金花]]（''Mussaenda divaricata''），同為茜草科玉葉金花屬植物，分別在於玉葉金花的葉及白色花瓣狀萼裂片較散玉葉金花的為小<ref name = "中科院鼎湖山站"/>。

*[[楠藤|楠藤]]（''Mussaenda erosa''），同為茜草科玉葉金花屬植物，分別在於玉葉金花的花序較楠藤的為小及玉葉金花沒有總花梗<ref name = "樹木谷"/><ref name = "葉脈網"/><ref>[http://www.producegreen.org.hk/hkwildflower/100flower/flower_80.htm  楠藤]香港野花網</ref>。
