'''開放美工圖庫'''（Openclipart Library）是一個作為[[自由內容|自由內容]]的向量[[美工圖片|美工圖片]]的收藏集及社群。其口號是「Openclipart是一個最大的創作最佳的自由原創美工圖片而絕無任何理由為你所用的藝術家社群。」<ref>{{cite web|url=http://openclipart.org/about|title=About Openclipart|date=2016|accessdate=2016-02-10|archive-url=https://web.archive.org/web/20160115020642/https://openclipart.org/about|archive-date=2016-01-15|dead-url=yes}}</ref>

==計劃現況==
截至2017年1月末，Openclipart展示了來自5,800多位藝術家超過118,000件[[SVG|SVG]]圖形的作品。<ref>{{cite web|url=https://openclipart.org/statistics|title=Openclipart Statistics|author=Openclipart|date=2016|accessdate=2016-08-27}}</ref>所有藏品皆可免費下載。全部圖像由貢獻者宣告進入[[公有領域|公有領域]]，並以[[SVG|SVG]]格式存儲及提供[[PNG|PNG]]格式的縮圖。

==應用==
* Inkpad for iPad，內含Openclipart Integration。<ref>{{cite web|url=http://inkpad.org|title=Inkpad iOS App|date=2016|accessdate=2016-02-10}}</ref>
* 向量圖形編輯器[[Inkscape|Inkscape]]可從Openclipart線上匯入向量圖形至您的工作區。（注意Windows版的版本號不得低於0.49）<ref>[http://www.oss-marketplace.com/index.php/downloads-mainmenu-63/Inkscape/Entwicklerversionen/ Inkscape latest build with Import OCAL support]</ref>
* Microsoft Office應用程式<ref>[https://store.office.com/openclipart-WA104199826.aspx?assetid=WA104199826 Microsoft Office app]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
* Google Docs Add-on<ref>[https://chrome.google.com/webstore/detail/openclipart/jhlifdiflonnipfdneahhngjdcaloajn?hl=en Google Docs Add-on]</ref>
* LibreOffice擴充套件<ref>[http://extensions.libreoffice.org/extensions/openclipart-org-integration LibreOffice Openclipart.org integration]</ref>
* iOS Clipart<ref>[https://itunes.apple.com/be/app/clipart/id588303554?mt=8 Clipart]</ref>、iOS PosterMaker<ref>[https://itunes.apple.com/ag/app/postermaker-real-printable/id423574589?mt=8 PosterMaker]</ref>
* [https://clipartxtras.com Clipart]搜尋引擎，Openclipart for Android<ref>[https://play.google.com/store/apps/details?id=apps101.co.za.openclipart2 Free Clip Art - Android Apps on Google Play]</ref>
* 用於WordPress的Clipart外掛模組<ref>[http://wordpress.org/extend/plugins/clipart/ Clipart plugin]</ref>
* 用於[[Moodle|Moodle]]的Clipart外掛模組<ref>[https://moodle.org/plugins/view.php?plugin=repository_openclipart  Moodle plugin that allows the user to search and import clipart]</ref>
* Linux發行版
一些[[Linux發行版|Linux發行版]]，包括[[Mandriva|Mandriva]]和[[Ubuntu|Ubuntu]]都包含了許多Openclipart收藏集發行套件，裝有SVG、PNG或[[OpenDocument|OpenDocument]]格式檔案。<ref>[http://packages.ubuntu.com/hardy/openclipart Ubuntu openclipart Packages]</ref>這些發行板基於2005年ccHost之前釋出的版本，自從標準的釋出版在轉換到ccHost軟體之後被終止。Openclipart 0.19，第一個在此次轉換之後釋出的版本，於2009年3月釋出。包括最近釋出的2.0版及更新的套件，這些發行版包含了在相應的bug追蹤器歸檔的bug以多次打包Openclipart。

Openclipart包含在[[Linux_Format|Linux Format]] issues 123<ref>[http://www.linuxformat.com/archives?issue=123 Linux Format Issue 123]</ref>及132<ref>[http://www.linuxformat.com/archives?issue=132 Linux Format Issue 132]</ref>中的光碟封面上作為一個來自Openclipart收藏集的可瀏覽的SVG套件。
