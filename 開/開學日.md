'''開學日'''是在某個學年[[上學|上學]]的第一日。不同地區的開學日都在不同的[[季節|季節]]，而開學日所做的事，各地亦有所不同。

== 澳洲 ==
[[南半球|南半球]]四季與北半球相反，所以開學日也不相同。一般而言，在澳洲，大學的第一級的開學日為每年二月底左右（接近秋季）；第二級的開學日則為每年九月头左右（接近春季）。

== 香港 ==
香港的開學日一般都在[[秋季|秋季]]的9月第一個上學日，而每年9月全月亦定為開課月。往常即使第一個上學日在星期六，亦會照常[[上課|上課]]，現今則延到將來的星期一才正在開課。

在開學日的[[早上|早上]]，[[公共運輸|公共運輸]]一般會加強[[服務|服務]]，把[[繁忙時段|繁忙時段]]提早，以加密班次，讓學生能夠在未適應新的上課時間前仍能盡量準時回校。

在[[2007年|2007年]]，由於全香港大多數[[中學|中學]]、[[小學|小學]]皆已轉為[[全日制學校|全日制學校]]，當地首次把開學日訂為星期六以外的第一個上學日，即當年的[[9月3日|9月3日]]。開學日變更，引起新入職[[教師|教師]]的資歷計算問題。一直以來，香港教育部門對新入職教師的資歷計算，都由[[新曆|新曆]]月的第一個[[工作日|工作日]]開始計算。但由於2007年的首個[[工作天|工作天]]並非開學日，若新入職教師在9月3日才正式[[上班|上班]]，當月的所有工作日都不會獲計算到資歷裡。所以，有不少學校都安排在[[9月1日|9月1日]]當日舉行校務[[會議|會議]]，使新入職教師亦能在曆月的開始上班，從而得以保存這個曆月的資歷。

== 臺灣 ==
*國民小學、[[台灣國中教育|國民中學]]、高級中學、職業學校：上學年於[[9月1日|9月1日]]左右開學，下學期配合[[春節|春節]]於2月10至2月17日左右開學。因各縣市教育局有各自規定，所以每個學期的開學日不見得相同，但各縣市的調整日期差距也只差個一天左右。
*大學、五專：上學期於9月20日前後開學，下學期於2月20日前後開學。各校開學日差距也約一天至兩天左右，相差日期並不大。

== 中國大陸 ==
每個學年有兩個開學日：一個是秋季學期（第一個學期）的開學日，固定於公曆的9月1日（如果9月1日為周末，則為之後第一個星期一）；另一個是春季學期（第二個學期）開學日因[[春節|春節]]時間變化而每年不同，一般在[[元宵節|元宵節]]左右。

== 日本及韓國 ==
[[日本|日本]]及[[韓國|韓國]]的開學日一般都在[[春季|春季]]，開學日當日會有「開學式」。

== 美國 ==
[[美國|美國]]的[[公立學校|公立學校]]開學[[地球日|日]]通常為[[年|每年]][[勞動日|勞動日]]（Labor Day）的下[[地球日|一日]]，一般是於[[9月|9月]]的第[[一|一]]個或第[[二|二]]個[[星期|星期]]裡。
