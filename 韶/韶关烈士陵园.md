'''韶关烈士陵园'''，又称'''韶关市烈士陵园'''，是一座安葬[[烈士_(中华人民共和国)|中华人民共和国烈士]]的纪念性公园，位于中国[[广东省|广东省]][[韶关市|韶关市]][[浈江区|浈江区]]陵园西路石门架右侧山。陵园占地30万平方米，建于1950年代下旬。

== 历史 ==
韶关烈士陵园于1957年12月7日动工兴建，翌年4月5日完工落成，后来进行了数次装修工程<ref name="广东省革命遗址通览"/><ref name="广东革命史迹通览"/>。陵园在1988年4月18日列入[[广东省重点烈士纪念建筑物保护单位|广东省重点烈士纪念建筑物保护单位]]，1995年5月列入韶关市爱国主义教育基地<ref name="广东革命史迹通览"/>。

== 设施 ==
韶关烈士陵园占地30万平方米<ref name="韶关市志 上"/>。陵园入口处为一座坐东朝西的[[门楼|门楼]]，高10米、宽15米，屋顶以绿色瓦片铺砌而成，墙身为红色，题有“韶关烈士陵园”字样；大门为园拱门，两旁各自摆设一头[[石狮子|石狮子]]<ref name="广东省革命遗址通览"/><ref name="广东革命史迹通览"/>。门楼前方设有一个小广场，里面种满了松树、柏树等树木<ref name="韶关文明网"/>。陵道位于小广场前方，是一道红色石梯，通往革命烈士纪念碑<ref name="广东革命史迹通览"/>。纪念碑高16米，正面刻有[[红色五角星|红色五角星]]与“革命烈士纪念碑”浮雕字样；碑底座高3米，正面刻有“永垂不朽”字样，三个侧面刻上碑文和诸位烈士姓名<ref name="广东省革命遗址通览"/>。纪念碑四方设有祭台，侧边为烈士墓地<ref name="广东省革命遗址通览"/><ref name="韶关文明网"/>。除了这些设施以外，陵园内还有八角亭、人工湖等建筑，以及原本位于[[韶关中山公园|韶关中山公园]]、后在1978年3月22日迁至陵园内的北江人民烈士纪念碑<ref name="广东省革命遗址通览"/>。

== 纪念人群 ==
最初兴建韶关烈士陵园的目的，是为了纪念革命战争时期（[[国民革命军北伐|国民革命军北伐]]至[[第二次国共内战|第二次国共内战]]）牺牲的[[烈士_(中华人民共和国)|中华人民共和国烈士]]<ref name="广东省革命遗址通览"/>。不过，随着时间推移，陵园后来亦安葬了殉职公安人员等其他烈士<ref name="韶关市公安局"/>。

== 祭扫 ==
韶关烈士陵园的祭扫高峰为清明节前后，韶关市委宣传部机关、政府部门、学校、幼儿园、企业、社会团体等很多单位都会在这个时期组织集体扫墓活动，甚至每年举办一次，有些单位会在陵园内进行重温[[中国共产党|中国共产党]][[中国共产党入党誓词|入党誓词]]的教育活动；亦有不少市民单独或举家前往祭扫，尤以当地民众为甚<ref name="韶关文明网"/>。[[韶关市公安局|韶关市公安局]]会组织已加入中国共产党的民警人员到陵园集体祭扫，重温[[中华人民共和国人民警察入警誓词|人民警察入警誓词]]，并向[[何勇_(公安人员)|何勇]]、[[郑华武|郑华武]]、[[付少云|付少-{云}-]]三位安葬于此陵园的殉职公安人员致意<ref name="韶关市公安局"/>。
