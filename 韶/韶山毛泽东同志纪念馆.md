'''韶山毛泽东同志纪念馆'''，简称'''毛泽东同志纪念馆'''，原名'''韶山毛泽东同志旧居陈列馆'''，位于[[湖南省|湖南省]][[韶山市|韶山市]][[韶山冲|韶山冲]]引凤山，纪念馆始建于1963年，以展览[[毛泽东|毛泽东]]生前的衣食住行生活物品为主的国家一级博物馆。

== 沿革 ==
1963年，韶山毛泽东同志旧居陈列馆开始建筑。

1964年10月1日，韶山毛泽东同志旧居陈列馆正式宣布对外开放。

1983年4月20日，[[中国共产党中央顾问委员会|中国共产党中央顾问委员会]]主任[[邓小平|邓小平]]亲笔题名。<ref name="fish " />

== 建筑 ==
韶山毛泽东同志纪念馆的主建筑是仿照[[苏州园林|苏州园林]]而建，建筑面积6千多平方米，陈列展览面积1完4千4百平方米，总建筑面积2万9千7百平方米。<ref name="fish " />附屬建築有遺物館與紀念園。

== 馆藏 ==
韶山毛泽东同志纪念馆现藏文物、文献、资料4万余件，其中毛泽东晚年生活遗物6千余件。<ref name="fish ">{{cite web |url=http://www.ssmzd.com/sitehtml/jlg/bggk/bgjj.htm |title=本馆简介 |accessdate=2012年10月28日 |publisher=韶山毛泽东同志纪念馆 |work= |language=zh-cn |archive-url=https://web.archive.org/web/20120807191438/http://www.ssmzd.com/sitehtml/jlg/bggk/bgjj.htm |archive-date=2012-08-07 |dead-url=yes }}</ref>
