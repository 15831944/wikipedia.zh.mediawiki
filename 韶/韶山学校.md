'''韶山学校'''，位于[[湖南省|湖南省]][[韶山市|韶山市]][[韶山冲|韶山冲]][[张家山|张家山]]，始建于1921年，为湖南省重点中学之一。

== 沿革 ==
[[File:ShaoShan_High_School_4.jpg|thumb]]
1921年，韶山冲建立“毛氏族校”，创始人是毛简臣、毛麓钟、毛泽东等人，它是学校的前身。<ref name=" fish">{{cite web |url=http://www.ssxw.net/Info.aspx?ModelId=1&Id=41722 |title=韶山学校 |accessdate=2006年11月15日 |publisher=韶山新闻网 |work= |language=zh-cn }}{{Dead link|date=2019年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="man ">{{cite web |url=http://edu.ifeng.com/news/200909/0930_6978_1371522.shtml |title=韶山学校：每间教室里都挂着毛主席画像 |accessdate=2009年9月30日 |publisher=凤凰网 |work= |language=zh-cn |archive-url=https://web.archive.org/web/20160304192326/http://edu.ifeng.com/news/200909/0930_6978_1371522.shtml |archive-date=2016-03-04 |dead-url=yes }}</ref>

1922年，毛氏族校分出2个学校，毛氏族校改为“毛氏一校”，另外在毛震公祠建立“毛氏二校”。 

1943年，毛宇居、毛国翘倡导，毛氏一校更名为“兴华小学”，毛氏二校更名为“震东小学”。

1951年，兴华小学更名为“韶山乡第三村校”。

1952年，韶山乡第三村校更名为“湘潭县韶山小学”。毛宇居请毛泽东给学校题名，毛泽东亲笔题名“韶山学校”，故而在1953年，学校更名为“韶山学校”。<ref name=" fish" /><ref name="man " />

1956年，韶山学校增加初中。<ref name=" fish" />

1960年，韶山学校增加高中。<ref name=" fish" />

== 教育 ==
学校包括小学、初中、高中。
