'''{{ruby-py|泌|-{zh-cn:bì;zh-tw:ㄅㄧˋ;zh-hk:bat<sup>6</sup>}-}}阳县'''位于[[中华人民共和国|中华人民共和国]][[河南省|河南省]]南部，是[[驻马店市|驻马店市]]下辖的一个[[县_(中华人民共和国)|县]]。

== 行政区划 ==
泌阳县目前下辖3个街道办事处、8个镇、11个乡：
*街道办事处：花园街道办事处、泌水街道办事处、古城街道办事处；
*镇：[[羊册镇|羊册镇]]、[[马谷田镇|马谷田镇]]、[[春水镇|春水镇]]、[[官庄镇|官庄镇]]、[[赊湾镇|赊湾镇]]、[[郭集镇|郭集镇]]、[[泰山庙镇|泰山庙镇]]和[[王店镇|王店镇]]；
*乡：高店乡、盘古乡、高邑乡、铜山乡、下碑寺乡、象河乡、付庄乡、贾楼乡、黄山口乡、杨家集乡和双庙街乡。<ref name="stats">{{cite web|url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2015/41/17/411726.html|title=中华人民共和国统计局|accessdate=2017-04-08}}</ref>

== 地理 ==
泌阳县位于[[南阳盆地|南阳盆地]]的东部，处在[[秦岭-淮河线|秦岭-淮河线]]上，属于暖温带大陆性季风气候，四季较为分明。县治在泌水之北，得名泌阳。
东与确山县、遂平县、驿城区接壤，南部与南阳市的桐柏县为邻、西部与南阳市的唐河县及社旗县为邻，北接南阳市的方城县、平顶山市的舞钢市、漯河市的舞阳县。

虽然泌阳县是[[河南省|河南省]]的贫困县，但当地仍保有比省内其他地区更佳的自然环境，并以“青山绿水环保县”作为县的口号<ref name="countywebsite">{{cite web|url=http://202.102.233.134/xqzx/by/bymain.htm|title=中国驻马店泌阳|accessdate=2009-04-07|deadurl=yes|archiveurl=https://web.archive.org/web/20070928220856/http://202.102.233.134/xqzx/by/bymain.htm|archivedate=2007-09-28}}</ref>。

== 历史 ==
泌阳早在[[旧石器时代|旧石器时代]]就已有人类定居。从当地已发掘的8处古文化遗址当中，有3处属于[[旧石器时代|旧石器时代]]，5处属于[[新石器时代|新石器时代]]。泌阳当地的历史文化属于[[龙山文化|龙山文化]]，是河南省的省级文物保护重点。

在[[春秋战国时期|春秋战国时期]]，当地曾为[[吴国|吴国]]及[[楚国|楚国]]的疆域。除了有楚国的城门遗址和楚长城遗址以外，还有吴王的陵墓。
