'''酏劑'''{{notetag|{{標音|字=酏|注音=ㄧˊ|拼音=yí|同音字=怡}}}}（{{lang-en|elixir}}、{{Lang-ar|الإكسير}}、{{Lang-gr|ξήριον}}<ref>http://www.etymonline.com/index.php?term=elixir</ref><ref>{{cite web |url=http://www.wordnik.com/words/elixir/etymologies |title=存档副本 |accessdate=2012-05-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20100302200917/http://www.wordnik.com/words/elixir/etymologies |archivedate=2010-03-02 }}</ref>）是一类口服的、澄清的、具有甜味的[[液體|液體]]，用於[[醫療|醫療]]用途，治療不适症狀。當成製劑使用時，至少含有一种有效成分內服。

== 形式 ==
=== 非醫療用途===
當成是溶媒或者是載體，用來運輸藥物。如[[芳香酏|芳香酏]]、[[異酒精酏|異酒精酏]]、[[複方苯甲醛酏劑|複方苯甲醛酏劑]]。

活性成分溶在含有15-50%酒精的溶液中。

=== 醫療用途 ===
# 抗組織胺酏劑(Antihistaminic elixirs)： chlorampheniramine maleate elixirs (USP), diphenhydramine HCl elixirs.
# 鎮靜安眠酏劑(Sedative and hypnotic elixirs)： pediatric chloral hydrate elixirs.
# 化痰劑(Expectorant)：terpin hydrate elixirs.
# 其他類別(Miscellaneous)：acetaminophen (paracetamol) elixirs, which are used as analgesics.

== 組成 ==
酏劑是個水-酒精溶液，至少含有一個以上的有效成分。 酒精的作用是：
# 溶解活性成分和輔料。
# 減少糖的結晶。
# 當做保存劑。
# 掩蓋活性成分不好的味道。
# 提高風味。

通常會選擇可以溶解活性成分的酒精量，並且保持溶液的澄清。高濃度的酒精可能會使的產品在使用時有燒灼感。

酏劑可能含有以下[[輔料|輔料]]：
# 糖或者是糖替代品，如[[山梨醇|山梨醇]]（sorbitol）
# [[保存劑|保存劑]]，如[[對羥基苯甲酸酯|對羥基苯甲酸酯]]、[[苯甲酸鈉|苯甲酸鈉]]
# [[緩衝溶液|緩衝溶液]]（Buffering solution）
# [[調味劑|調味劑]]
# [[螯合劑|螯合劑]]
