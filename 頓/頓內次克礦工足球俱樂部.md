'''薩克達足球會'''（{{lang-uk|Футбольний клуб «Шахтар» Донецьк}}），是[[烏克蘭足球超級聯賽|烏克蘭足球超級聯賽]]球隊，该队汇集了东欧国家的许多国脚，并在2000年后的乌克兰联赛中占据统治地位。

== 球會榮譽 ==
* '''[[烏克蘭足球超級聯賽|烏克蘭足球超級聯賽]]：11'''
:: 2002, 2005, 2006, 2008, 2010, 2011, 2012, 2013, 2014, 2017, 2018

* '''[[烏克蘭盃|烏克蘭盃]]：12'''
:: 1995, 1997, 2001, 2002, 2004, 2008, 2011, 2012, 2013, 2016, 2017, 2018

* '''[[烏克蘭超級盃|烏克蘭超級盃]]：4'''
:: 2005, 2008, 2010, 2012

* '''[[歐洲足協盃|歐洲足協盃]]：1'''
:: [[2008–09年歐洲足協盃|2009]]

* '''[[蘇聯盃|蘇聯盃]]：4'''
:: 1961, 1962, 1980, 1983

* '''[[:en:Soviet_Super_Cup|蘇聯超級盃]]：1'''
:: 1983
