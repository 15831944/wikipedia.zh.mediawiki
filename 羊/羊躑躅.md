'''羊躑躅'''（[[學名|學名]]：''{{lang|la|Rhododendron molle}}''），別稱'''黃杜鵑'''（本草蒙荃）、'''黃花杜鵑'''、'''黃花女'''、'''黃牯牛花'''、'''黃喇叭花'''、'''黃色映心紅'''、'''黃躑躅'''、'''鬧羊花'''（本草綱目）、'''鬧頭花'''、'''驚羊花'''、'''驚羊草'''、'''隔羊花'''、'''羊不食'''、'''羊不食草'''（本草拾遺）、'''牛不食'''、'''老虎花'''、'''毛老虎'''、'''搜山虎'''、'''山茶花'''、'''映山黃'''、'''玉枝'''（名醫別錄）、'''悶頭花'''、'''雷公花'''、'''一杯倒'''、'''三錢三'''、'''六軸子'''、'''八釐麻'''及'''八里麻'''等<ref name="自然標本館"/><ref name="中國">{{cite encyclopedia| title=羊躑躅| encyclopedia=中國植物誌| volume=57(2)| pages=367-369| url=http://v2.cvh.org.cn/zhiwuzhi/page/57(2)/367.PDF}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，為[[杜鵑花科|杜鵑花科]][[杜鵑花屬|杜鵑花屬]][[植物|植物]] <ref name="自然標本館"/>。全株有毒，誤服過量可致[[嘔吐|嘔吐]]、[[腹瀉|腹瀉]]、腹痛、[[痙攣|痙攣]]、[[心跳|心跳]]減慢、[[血壓|血壓]]下降及呼吸困難等症狀<ref name="中國"/><ref name="神農本草經">{{cite book | title=圖解神農本草經 | publisher=山東美術出版社 | author=陶隱夕 | page=443-445 | isbn=978-7-5330-2529-8|date=2008年8月}}</ref>，嚴重可致呼吸停止而死亡<ref name = "神農本草經"/>。花果可供制作[[鎮痛劑|鎮痛劑]]、[[麻醉劑|麻醉劑]] <ref name="中國"/><ref name="安徽">{{cite encyclopedia| title=羊躑躅| encyclopedia=安徽植物誌| volume=4| pages=10-11| url=http://v2.cvh.org.cn/difangzhi/anhui/page/4/010.pdf}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，全株可供作[[農藥|農藥]] <ref name="安徽"/>。本種因羊隻誤食其葉引致躑躅而亡，故而得名<ref name="中國"/><ref name="本草綱目">{{cite book | title=精編本草綱目圖文本 | publisher=華夏出版社 | author=沈連生 | page=142 | isbn=978-7-5080-2966-5|date=2007年7月}}</ref>。

==分佈==
本種現分佈於[[中國|中國]]及[[日本|日本]]等地，中國國內分佈於[[廣西|廣西]]、[[廣東|廣東]]、[[福建|福建]]、[[雲南|雲南]]、[[四川|四川]]、[[貴州|貴州]]、[[安徽|安徽]]、[[江西|江西]]、[[貴州|貴州]]、[[河南|河南]]、[[湖北|湖北]]、[[湖南|湖南]]、[[江蘇|江蘇]]、[[浙江|浙江]]等省區<ref name="ars"/>，本種喜生於山坡草地及丘陵地帶上的雜木林或灌叢之下<ref name="中國"/>。 
 
==形態特徵==
羊躑躅是一種落葉[[灌木|灌木]]植物，枝條圓柱形<ref name="福建">{{cite encyclopedia| title=羊躑躅| encyclopedia=福建植物誌| volume=4| pages=231| url=http://v2.cvh.org.cn/difangzhi/fujian/page/4/231.pdf}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，直立，分枝稀疏，幼時密披灰白色或棕色<ref name="山東">{{cite encyclopedia| title=羊躑躅| encyclopedia=山東植物誌| volume=| pages=860| url=http://v2.cvh.org.cn/difangzhi/shandong/page/%E4%B8%8B%E5%8D%B7/860.pdf}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>柔毛及柔毛狀剛毛<ref name="浙江">{{cite encyclopedia| title=羊躑躅| encyclopedia=浙江植物誌| volume=5| pages=13-14| url=http://v2.cvh.org.cn/difangzhi/zhejiang/page/5/013a.pdf}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，老枝稍帶褐色<ref name="山東"/>，無毛<ref name="浙江"/>，近光滑<ref name="山東"/>；冬芽時常披灰柔毛<ref name="山東"/>；植株高約0.3<ref name="山東"/>－2[[米|米]] <ref name="中國"/>。

===葉===

[[葉|葉]]為長圓形、矩圓形<ref name="河南">{{cite encyclopedia| title=羊躑躅| encyclopedia=河南植物誌| volume=3| pages=195-196| url=http://v2.cvh.org.cn/difangzhi/henan/page/3/195a.pdf}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、長圓狀披針形、橢圓狀倒披針形，紙質，互生<ref name="安徽"/>，頂端[[wikt:急尖|急尖]]或鈍<ref name="浙江"/>，具短尖頭，基部寬楔形<ref name="貴州">{{cite encyclopedia| title=羊躑躅| encyclopedia=貴州植物誌| volume=3| pages=234-235| url=http://v2.cvh.org.cn/difangzhi/guizhou/page/3/234.pdf}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，表面深綠色<ref name="貴州"/>，表面幼時披微柔毛及銀白色的細剛毛<ref name="安徽"/>，中脈凸出，背面蒼白色<ref name="浙江"/>，背面密披灰白色柔毛<ref name="福建"/>，邊緣披刺毛狀睫毛<ref name="浙江"/>，沿中脈披黃褐色剛毛，側脈10<ref name="安徽"/>－22<ref name="福建"/>對，凸出，表面不明顯，背面較顯著，延伸至邊緣處時常呈網狀分叉<ref name="安徽，披微柔毛及稀疏剛毛<ref name="安徽"/>，長約5<ref name="中國"/>－15<ref name="山東"/>厘米，寬約1.5<ref name="中國"/>－5<ref name="安徽"/>厘米；[[葉柄|葉柄]]稍帶褐色<ref name="安徽"/>，披柔毛及少量剛毛，長約3－4厘米<ref name="福建"/>
 
===花=== 
[[File:Chinese_Azalea_-_Flickr_-_treegrow.jpg|thumb]]

[[File:Rhododendron_molle.jpg|thumb]]
[[花|花]]5－12朵<ref name="安徽"/>，可多達13朵，排列成傘形式總狀[[花序|花序]]<ref name="浙江"/>，頂生，花葉同時盛開<ref name="浙江"/>或先於[[葉|葉]]開放<ref name="福建"/>；小花梗綠色<ref name="安徽"/>，細長<ref name="安徽"/>，披淺黃色微柔毛<ref name="福建"/>及剛毛，長約1<ref name="中國"/>－2.5<ref name="福建"/>厘米；[[花萼|花萼]]裂片5枚<ref name="浙江"/>，圓齒狀，細小，披淺黃色微柔毛<ref name="福建"/>，邊緣處披剛毛狀長睫毛<ref name="安徽"/><ref name="河南"/>，長約2<ref name="安徽"/>－6<ref name="貴州"/>毫米；[[花冠|花冠]]闊漏斗形，黃色或金黃色，亞種''R. molle subsp. japonicum''開紅色花，長約3<ref name="貴州"/>－5.2<ref name="浙江"/>厘米，直徑約4.5<ref name="福建"/>－6.5<ref name="安徽"/>厘米；花冠管圓筒形，向基部漸狹，表面披微柔毛，長約2.6厘米 <ref name="中國"/>；花冠裂片5枚，卵狀長圓形或橢圓形，上側一枚較大<ref name="安徽"/>，內部上側有紅色<ref name="中國"/>或淡綠色<ref name="安徽"/><ref name="河南"/>斑點，表面披微柔毛，長約2.8厘米 <ref name="中國"/>；[[雄蕊|雄蕊]]5枚，不等長，長不超過花冠，花絲扁平，中部以下披微柔毛<ref name="中國"/>；[[花藥|花藥]]金黃色<ref name="安徽"/>；[[子房|子房]]圓錐狀，密披灰白色或淺黃色<ref name="福建"/>柔毛及疏細剛毛，長約4毫米<ref name="中國"/>；[[花柱|花柱]]無毛，長於雄蕊<ref name="貴州"/>，突出雄蕊群之外<ref name="山東"/>，長約6厘米 <ref name="中國"/>；柱頭頭狀，綠色<ref name="安徽"/>。

===果實===

[[果實|果實]]為[[蒴果|蒴果]]，圓柱狀長圓形<ref name="浙江"/>，具6條縱肋<ref name="安徽"/>，密披柔毛及疏剛毛，成熟時深褐色<ref name="福建"/>，長約2<ref name="福建"/>－3.5<ref name="山東"/>厘米，[[直徑|直徑]]約8－10[[毫米|毫米]]<ref name="浙江"/>。

==醫藥用途==
{{medical}}
===根===
本種根部可治肺寒[[咳嗽|咳嗽]]、[[痰|痰]]多、[[類風濕性關節炎|類風濕性關節炎]]及跌打損傷等<ref name = "神農本草經"/><ref name="浙江"/>。

===花===

本種以羊躑躅的乾燥[[花朵|花朵]]入藥，[[中藥|中藥]]名為鬧羊花，[[藥材|藥材]]主產於[[江蘇|江蘇]]、[[浙江|浙江]]、[[江西|江西]] <ref name="香港中藥材">{{cite book | title=香港中藥材圖鑑 | publisher=香港浸會大學中醫藥學院 | author=趙中振 | page=391 | isbn=988-97448-1-3|date=2003年10月}}</ref>、[[河南|河南]]、[[湖南|湖南]]、[[湖北|湖北]]等地，始載於《[[神農本草經|神農本草經]]》，列為下品<ref name="當代藥用">{{cite book | title=當代藥用植物典 4 | publisher=香港賽馬會中藥研究院 | author=蕭培根 | page=416-418 | isbn=978-988-99226-3-4|date=2007年8月}}</ref>。《[[神農本草經|神農本草經]]》中原文記載為「羊躑躅，味辛，溫。主賊風在皮膚中淫淫痛；溫瘧；惡毒；諸痹。生川谷。」<ref name = "神農本草經"/><ref name = "本草綱目"/>。味辛，性溫，有大毒，歸肝經<ref name="意古中醫">{{cite web| url=http://www.tcmlib.com/zy/html71/showdetail-373037342ce997b9e7be8ae88ab12c7a79.html| title=鬧羊花| publisher=意古中醫| deadurl=yes| archiveurl=https://web.archive.org/web/20080517160620/http://www.tcmlib.com/zy/html71/showdetail-373037342ce997b9e7be8ae88ab12c7a79.html| archivedate=2008-05-17}}</ref>，具驅風除濕、止痛、消腫、散瘀、殺蟲解毒、麻醉用藥等功效<ref name = "神農本草經"/><ref name = "當代藥用"/><ref name = "本草綱目"/>，主治[[皮膚|皮膚]]頑[[癬|癬]]、[[疥瘡|疥瘡]]、[[風濕|風濕]]痹痛、傷折疼痛、牙痛、偏正[[頭痛|頭痛]]等 <ref name = "神農本草經"/><ref name = "當代藥用"/><ref name = "本草綱目"/>。現代藥理研究表明羊躑躅具鎮痛、降[[血壓|血壓]]、抗[[心律|心律]]失常、抗[[腎小球|腎小球]][[腎炎|腎炎]]、抗菌、殺蟲等作用，臨床應用於[[風濕|風濕]]、跌打疼痛、[[偏頭痛|偏頭痛]]、神經性[[頭痛|頭痛]]、牙痛、[[高血壓|高血壓]]、[[心跳|心跳]]過速、手術[[麻醉|麻醉]]等治療上<ref name = "神農本草經"/><ref name = "當代藥用"/>。[[中醫|中醫]]歸類為袪風藥<ref name="意古中醫"/>。因本藥含有毒性，不宜久服，體虛及氣血虛者忌服<ref name="意古中醫"/>。

===果實===

本種以乾燥[[果實|果實]]入藥，中藥名為六軸子<ref name="浙江"/>，味辛，性溫，有毒，具袪風除濕、散瘀止痛、定喘及止淀等功效，主治風寒濕痹、關節腫痛、癰疽腫毒、跌打損傷、瀉痢、喘咳等。中醫歸類為袪風、止痛藥<ref name="意古">{{cite web| url=http://www.tcmlib.com/zy/html10/showdetail-3933362ce585ade8bdb4e5ad902c7a79.html| title=六軸子| publisher=意古中醫}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。因本藥含有毒性，不宜久服，體虛及無瘀者忌服<ref name="意古"/>。

==藥材鑑定==
===花===

中藥鬧羊花以乾燥[[花朵|花朵]]入藥，於每年四至五月間花初開時採收，清除雜質後曬乾或陰乾<ref name="香港中藥材"/>；本品由5－12朵花所組成，簇生於一總花梗上，為已開放的花朵，表面黃色至黃褐色<ref name="常用中藥材">{{cite book | title=常用中藥材鑑別圖典 | publisher=萬里機構 | author=趙中振, 陳虎彪 | page=289 | isbn=978-962-14-4150-8|date=2010年1月}}</ref>，多呈皺縮狀，有時具花梗 <ref name="香港中藥材"/>；[[花梗|花梗]]棕褐色，披短茸毛，長約1－7厘米 <ref name="香港中藥材"/>；[[花萼|花萼]]裂片5枚，細小，邊緣處披剛毛狀長睫毛<ref name="香港中藥材"/>；[[花冠|花冠]]鐘狀<ref name = "常用中藥材"/>，冠筒黃灰至黃褐色，表面披柔毛，先端卷折，長約3厘米 <ref name="香港中藥材"/>；花冠裂片5枚，寬卵形<ref name = "常用中藥材"/>，與冠筒等長<ref name="香港中藥材"/>；雄蕊5枚，[[花絲|花絲]]捲曲，略長或等長於花冠<ref name = "常用中藥材"/>，[[花藥|花藥]]紅棕色，先端孔裂<ref name="香港中藥材"/>；[[雌蕊|雌蕊]][[花柱|花柱]]長於雄蕊，[[柱頭|柱頭]]頭狀<ref name="香港中藥材"/>；氣微香，味微麻，有毒<ref name="香港中藥材"/>；傳統經驗則認為以棕黃色者為佳<ref name = "常用中藥材"/>。本品種載錄於《[[中國藥典|中國藥典]]》2005年版，定為中藥鬧羊花的法定原植物來源種，主要透過[[薄層色譜法|薄層色譜法]]等鑒別資料，以控制藥材的質量<ref name = "當代藥用"/>。本品種亦載錄於香港第一類別中藥材（毒劇）名單之中<ref name="香港中藥材"/>。

===果實===
中藥六軸子以羊躑躅的乾燥果實入藥，於每年九至十月果實未成熟開裂前採收，清除雜質後曬乾<ref name="意古"/>。
