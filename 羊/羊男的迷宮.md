《'''羊男的迷宮'''》（{{lang-es|''El laberinto del fauno''}}），[[2006年电影|2006年]]由[[墨西哥|墨西哥]]跟[[西班牙|西班牙]]聯合製作的影片，導演為[[墨西哥|墨西哥]]籍的[[吉勒摩·戴托羅|吉勒摩·戴托羅]]，影片首先在[[2006年电影|2006年]]的[[戛纳电影节|戛纳电影节]]正式競賽單元面世，頗獲好評。

电影的西班牙名字指的是[[罗马神话|罗马神话]]中半人半羊的农牧神，英语名字则指的是[[希腊神话|希腊神话]]中的半人半羊神。然而，吉勒摩·戴托羅指出电影中的像羊一样的人物形象并非是希腊神话中的潘神。电影是由墨西哥EsperantoFilms公司制作并发行的。

《羊男的迷宮》故事背景為1944年西班牙<ref>{{Cite book|title=有生之年非看不可的1001部電影|last=史蒂文·傑伊·施奈德|publisher=全國百佳出版社|year=2011年1月|isbn=978-7-5117-0619-5|location=北京|pages=921}}</ref>，此時[[西班牙內戰|西班牙內戰]]結束不久，[[佛朗哥|佛朗哥]]實行[[法西斯主義|法西斯主義]]獨裁統治；同時，[[佛朗哥|佛朗哥]]部隊受到殘餘的共和軍及共產黨游擊隊的攻擊。

== 劇情 ==
故事看似是由奧菲莉亞創造出的一個充滿幻想、奇幻的虛構世界，然而從劇情、角色的樣態、色調等卻都顯露出殘酷又陰森的氣息。

11歲的奧菲莉亞跟著母親投靠繼父--上尉維多，上尉與游擊隊間進行的戰火林立讓奧菲莉亞恐懼的只能躲進自己的幻想世界--被精靈帶進迷宮，並在那裡遇到[[牧神|牧神]]（[[潘_(希臘神話)|潘]]），他告訴奧菲莉亞她原是地底王國的公主，而必須完成三個任務才能再次回去。

== 本片相關資訊 ==
[[File:Pan's_Labyrinth_premiere.jpg|缩略图]]
[[墨西哥|墨西哥]]籍的導演葛雷摩·戴托羅在2001年的時候已循墨西哥跟[[西班牙|西班牙]]聯合製片的模式拍了一部《[[鬼童院|鬼童院]]》（製片是西班牙大導演[[艾慕杜華|艾慕杜華]]，時代定在1939年的西班牙，正是[[西班牙內戰|西班牙內戰]]（1936-1939年）的末期。《鬼童院》跟《羊男的迷宮》都屬於奇幻類型，但《鬼童院》偏向[[寫實主義|寫實主義]]的鬼故事，《羊男的迷宮》則接近被二分為{{le|奇幻世界|Fantasy world}}和[[法西斯主義|法西斯主義]]的現實世界。

== 得獎紀錄 ==
===[[第79届奥斯卡金像奖|第79届奥斯卡金像奖]]===
本片在[[2007年电影|2007年]]1月被提名入選角逐該獎的六個獎項：「[[奧斯卡最佳原創劇本|最佳原著劇本]]」、「[[奧斯卡最佳攝影|最佳攝影]]」、「[[奧斯卡最佳化妝|最佳化妝]]」、「[[奧斯卡最佳藝術指導|最佳藝術指導]]」、「[[奧斯卡最佳原創音樂|最佳配樂]]」、「[[奧斯卡最佳外語片|最佳外語片]]」，2月25日公佈獲得該獎之「最佳攝影」、「最佳化妝」、「最佳藝術指導」三個獎項<ref>{{Cite web|title=第79届奥斯卡金像奖获奖名单|url=http://ent.sina.com.cn/f/79oscar/hjmd.html|accessdate=2019-04-11|date=2007-02-25|publisher=新浪网}}</ref>。

此外亦榮獲當年[[星雲獎|星雲獎]]「最佳劇本」、[[雨果獎|雨果獎]]「最佳戲劇表現：長劇」。
