'''羊亚科'''（[[学名|学名]]：{{lang|la|Caprinae}}），是[[牛科|牛科]]下的一个亚科。因为这个亚科内的动物之间的亲属关系还不很清楚，为了显示这个不清楚性，在生物分类学上亚科以下又分族。

高鼻羚羊族和斑羚屬俗称羚羊。

== 主要特征 ==
除了[[湖羊|湖羊]]两性都无角外，两性都有角。雄性角大于雌性角。角多扁平，断面略成三角形，表面有许多环行横棱隆起。

== 分布 ==
除[[澳洲|澳洲]]无自然分布外遍布世界各地。

== 主要食物 ==
[[草本植物|草本植物]]、[[灌木|灌木]]的叶子。

== 主要天敌 ==
[[虎|虎]]、[[豹|豹]]、[[狮|狮]]、[[狼|狼]]、[[猞猁|猞猁]]等。

== 保护现状 ==
[[山羊|山羊]]、[[绵羊|绵羊]]安全，其余均为一、二级保护动物。
