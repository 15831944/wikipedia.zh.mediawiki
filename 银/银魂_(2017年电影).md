《'''银魂'''》（{{lang-ja|'''銀魂'''}}）是一部[[2017年電影|2017年]]上映的由[[福田雄一|福田雄一]]导演的日本动作喜剧片，改编自[[空知英秋|空知英秋]]所创作的著名漫画《[[银魂|银魂]]》，主角[[坂田銀時|坂田銀時]]由[[小栗旬|小栗旬]]主演，电影于2017年7月14日在日本上映。續集《[[銀魂2：規矩是為了被打破而存在的|銀魂2：規矩是為了被打破而存在的]]》将于2018年8月在日本上映。

== 劇情 ==
{{expand section}}
《銀魂》真人版電影的劇情主要改編自原作漫畫中的“[[銀魂劇場版_新譯紅櫻篇|紅櫻篇]]”。講述萬事屋三人組坂田銀時、志村新八與神樂追尋神秘妖刀“紅櫻”，結果卻被捲入了一場陰謀之中。

== 制作 ==
《银魂》在2016年8月开拍，同年10月杀青。<ref>[http://ent.sina.com.cn/m/f/2016-10-21/doc-ifxwztrt0034274.shtml 真人版《银魂》杀青 小栗旬菅田将晖合影].新浪娱乐.2016-10-21.[2017-08-21].</ref>电影改编自在日本连载超过13年的著名同名漫画，由曾执导过《勇者闯魔城》系列的[[福田雄一|福田雄一]]担任导演和编剧，漫画原作者[[空知英秋|空知英秋]]也是看过该系列后确定由福田来担任《银魂》真人电影的导演。<ref>[http://ent.sina.com.cn/m/f/2017-06-16/doc-ifyhfnqa4347885.shtml 空知英秋钦点导演 真人版《银魂》被赞神还原].新浪娱乐.2017-06-16.[2017-08-21].</ref>

电影的主题曲是由日本乐队[[UVERworld|UVERworld]]所创作的单曲《DECIDED》。<ref>[http://www.oricon.co.jp/news/2089408/full/ UVERworld、実写『銀魂』主題歌 TAKUYA∞「まさか、関われるとは」].Oricon News.2017-04-19.[2017-08-21].</ref>

== 票房 ==
《银魂》在日本上映后获得了票房上的成功，首周获得5.4亿日元的票房，但不及《精灵宝可梦》的20周年纪念作《[[精靈寶可夢劇場版：就決定是你了！|精靈寶可夢劇場版：就決定是你了！]]》，位列当周票房榜第二名。<ref>[http://www.oricon.co.jp/news/2094323/full/ 夏休み興行本番突入 ヒットした映画は? 『ポケモン』『銀魂』『パイレーツ』].Oricon News.2017-07-18.[2017-08-21].</ref>电影上映30天后日本国内票房达到30亿日元。<ref name="JPNBoxOffice"/>

电影在香港上映首周获得122万港币，位列当周票房的第八位。在中国大陆上映首周3天票房达到6100万人民币，位列当周票房第五位；上映三周总票房超过8000万人民币。<ref>[http://www.cbooo.cn/m/664926 银魂 中国票房].艺恩电影.[2017-09-26].</ref>
