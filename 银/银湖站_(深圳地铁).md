'''银湖站'''是[[深圳地铁|深圳地铁]][[深圳地铁9号线|9号线]]中的一个车站，位于北环大道银湖汽车站南侧，沿北环大道呈东西方向布置。车站北端设一条联络线与[[笔架山停车场|笔架山停车场]]接轨。车站装修以白色为基准色，墙柱用黄色陶瓷钢板装饰，天花由白色铝合金吊挂横向挂置而成，整体效果简洁大方。<ref>[http://sz.bendibao.com/news/2015215/671775.htm 深圳地铁9号线银湖站提前封顶] 2015-02-15</ref>

== 車站設施與藝術牆 ==
本站設有洗手間，位於B出口旁。

本站藝術牆名為《凝固的綠色》，以水泥模擬各種樹葉標本，代表深圳孕育了豐富多樣的植物物種，觀眾可以通過介紹鏈接到每種植物詳細的說明。

* 2014年9月5日，银湖站全面开建<ref>[http://www.sznews.com/news/content/2014-09/06/content_10131305.htm 深圳地铁9号线银湖站全面开建 明年3月有望车站封顶]深圳新闻网 2014-09-06</ref>
* 2014年10月18日，工地发生塌陷，未有人员伤亡。<ref>[http://sz.house.163.com/14/1018/09/A8R25GT700073T3P.html 地铁9号线银湖站工地塌陷 工人差点被埋井下]2014-10-18</ref>
* 2015年2月14日，银湖站提前封顶。

== 未來發展 ==
