'''广东银禧科技股份有限公司'''（{{lang-en|Silverage}}，{{szse|300221}}），简称'''银禧科技'''，是一家1997年成立于广东省[[东莞市|东莞市]]的高分子新材料科技公司。2011年，登录[[深圳证券交易所创业板|深圳证券交易所创业板]]上市<ref>{{Cite book|title=珠江三角洲城市群年鉴 2012 总第3卷|last=王林生|first=|publisher=广州：广东人民出版社|year=2012年11月|isbn=978-7-218-08443-5|location=|pages=306}}</ref>。公司位于广东省[[东莞市|东莞市]][[虎门镇|虎门镇]][[居岐工业区|居岐工业区]]，在东莞、中山、[[苏州|苏州]]、[[深圳|深圳]]设有生产研发基地，产品覆盖改性工程塑料、[[LED|LED]]光电材料及配件、[[智能手机|智能手机]]金属构件、[[3D打印|3D打印]]及互联网平台等业务领域。

== 概述 ==
1997年8月，银禧科技创立，前身是'''东莞银禧塑胶有限公司'''。之后公司不断发展，相继建立广东（虎门）银禧、银禧工程塑料（道滘）和苏州银禧高分子材料有限公司三家子公司。2011年5月25日，在[[东莞证券|东莞证券]]的保荐推动下<ref>{{Cite web|url=http://economy.enorth.com.cn/system/2011/05/16/006555142.shtml|title=新股申购：银禧科技(300221)发行价18.00元/股|accessdate=2017-06-18|work=economy.enorth.com.cn}}</ref>，银禧科技成功登陆[[深圳证券交易所|深圳证券交易所]]。2011年，银禧科技通过成为非电力电缆用改性PVC的龙头，也是中国大陆改性塑料行业最具竞争力和成长性的领先企业<ref>{{Cite web|url=http://finance.qq.com/a/20121126/005205.htm|title=银禧科技深度报告:改性PVC支撑现有估值,LED灯用塑料爆发潜力巨大_财经_腾讯网|accessdate=2017-06-18|work=finance.qq.com|language=zh-CN}}</ref><ref>{{Cite web|url=http://info.plas.hc360.com/2012/12/200933381031.shtml|title=改性PVC龙头 银禧科技介入改性工程塑料领域-改性PVC,龙头,银禧科技-塑料行业-hc360慧聪网|accessdate=2017-06-18|last=慧聪网|work=info.plas.hc360.com}}</ref>。

2015年8月，公司成立全资子公司深圳三维魔坊网络有限公司，通过3D打印创意礼品商城、3D打印服务、设计师资源整合等。2015年，公司对旗下子公司银禧光电启动股份制改制。并在同年12月，将子公司银禧光电通过[[全国中小企业股份转让系统|全国中小企业股份转让系统]]批准，登录新三板<ref>{{Cite web|url=http://www.cs.com.cn/ssgs/gsxw/201508/t20150805_4771270.html|title=银禧科技子公司改制拟登陆新三板_公司新闻_中证网|accessdate=2017-06-18|last=张红瑜|work=www.cs.com.cn}}</ref><ref>{{Cite web|url=http://www.silverage.cn/licheng.aspx|title=广东银禧科技股份有限公司|accessdate=2017-06-18|work=www.silverage.cn|archive-url=https://web.archive.org/web/20170618085538/http://www.silverage.cn/licheng.aspx|archive-date=2017-06-18|dead-url=yes}}</ref>。2016年4月，银禧科技启动并购兴科电子科技有限公司；同年10月25日获得证监会通过。2017年3月，完成对兴科电子科技有限公司的并购案，成为[[乐视|乐视]]、[[小米|小米]]、[[锤子|锤子]]、[[OPPO|OPPO]]、[[美图|美图]]的供应商<ref>{{Cite web|url=http://www.shoujibao.cn/news/show-htm-itemid-18100.html|title=银禧科技肥收乐视CNC供应商兴科电子，一年多赚4亿_手机报_手机报在线|accessdate=2017-06-18|work=www.shoujibao.cn}}</ref>。

== 产品 ==
银禧科技的产品主要包括五大类型：阻燃材料、耐候材料、增强增韧材料、塑料合金材料以及环保耐用材料等。产品广泛应用于电线电缆、电子电气、家用电器和商用设备<ref>{{Cite book|title=聚合物抗车辙沥青路面研究与应用|last=付丽琴|first=|publisher=北京：中国建材工业出版社|year=2014年5月|isbn=978-7-5160-0798-3|location=|pages=14}}</ref>，以及汽车、交通、建筑、玩具、灯饰和医疗等领域。银禧科技具有年产30万吨改性高分子材料的生产能力<ref>{{Cite web|url=http://mt.sohu.com/it/d20170601/145183221_488166.shtml|title=2017年改性塑料行业将如何发展？|accessdate=2017-06-18|work=mt.sohu.com}}</ref>。
