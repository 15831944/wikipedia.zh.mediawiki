'''银氰酸钾'''是无机化合物，为无色晶体，易溶于水。

== 制备 ==
* 在[[氧化剂|氧化剂]]的存在下，用[[氰化钾|氰化钾]]溶液将银溶解：
:<math>\mathsf{4Ag + 8KCN + O_2 + 2H_2O \ \xrightarrow{}\ 4K[Ag(CN)_2] + 4KOH }</math>
:<math>\mathsf{2Ag + 4KCN + H_2O_2 \ \xrightarrow{}\ 2K[Ag(CN)_2] + 2KOH }</math>

* [[氰化钾|氰化钾]]溶液溶解[[氯化银|氯化银]]：
:<math>\mathsf{AgCl + 2KCN \ \xrightarrow{}\ K[Ag(CN)_2] + KCl }</math>

== 化学性质 ==
* 加热分解：
:<math>\mathsf{2K[Ag(CN)_2] \ \xrightarrow{250-420^oC}\ 2KCN + 2Ag + C_2N_2\uparrow }</math>

* 与强酸反应：
:<math>\mathsf{2K[Ag(CN)_2] + 2H_2SO_4 \ \xrightarrow{80^oC}\ Ag_2SO_4\downarrow + K_2SO_4 + 4HCN\uparrow }</math>

* 复分解：
:<math>\mathsf{2K[Ag(CN)_2] + K_2S \ \xrightarrow{}\ Ag_2S\downarrow + 4KCN }</math>

* 置换出金属银：
:<math>\mathsf{2K[Ag(CN)_2] + Zn \ \xrightarrow{}\ K_2[Zn(CN)_4] + 2Ag\downarrow }</math>

== 用途 ==
* [[镀银|镀银]]时的电解质。
* 作为杀菌剂，生产中的防腐剂。<ref>{{cite web | url = http://cameochemicals.noaa.gov/chemical/5151 | work = CAMEO Chemicals | publisher = [[美国国家海洋和大气管理局|NOAA]] | title = Potassium Silver Cyanide}}</ref>
* 化学合成。
