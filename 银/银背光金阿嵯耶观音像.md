'''银背光金阿嵯耶观音像'''，[[宋代|宋代]][[大理国|大理国]]文物，1978年出土于[[云南|云南]][[大理市|大理]][[崇圣寺_(大理)|崇圣寺]]主塔千寻塔。观音像约高26厘米，总重1135克，總高29.5厘米<ref name="国宝档案">{{cite web |title=《国宝档案》 20121212 宋 银背光金阿嵯耶观音立像 |url=http://tv.cctv.com/2012/12/12/VIDE1355315855878899.shtml |website=央视网 |accessdate=2019-01-31 |date=2012-12-12}}</ref>，是目前已知以纯金铸造最大的宋代佛像，现藏于[[云南省博物馆|云南省博物馆]]<ref name="海外版">{{cite news |title=馆藏精品 |url=http://paper.people.com.cn/rmrbhwb/html/2016-01/02/content_1644547.htm |accessdate=2019-01-31 |work=人民日报海外版 |date=2016-01-02}}</ref>。

== 出土 ==
[[File:银背光金阿嵯耶观音立像_上身.jpg|thumb]]
云南省大理市点翠山应乐峰下的[[崇圣寺_(大理)|崇圣寺]]兴建于南诏[[保和_(年號)|保和]]十三年（[[唐文宗|唐文宗]][[开成|开成]]元年，公元836年），毁于清穆宗[[同治|同治]]十一年（1872年），仅三座古塔留存。1978年，中国政府拨专款对年久失修的三塔进行维修加固，文物工作者在维修三塔时，在主塔千寻塔塔顶发现南诏、大理国时期文物文物680多件<ref name="杜瑜丽2008">{{cite journal |author1=杜瑜丽 |title=大鹏金翅鸟与白族的金鸡崇拜 |journal=云南农业大学学报(社会科学) |date=2008 |issue=2 |pages=115-117}}</ref>，同时出土的还有[[大理国银鎏金镶珠金翅鸟|银鎏金镶珠金翅鸟]]等<ref name="雅昌网">{{cite web |title=银鎏金镶珠金翅鸟 |url=https://news.artron.net/20131129/n540344.html |website=雅昌网-新华日报 |accessdate=2019-01-07 |date=2013-11-29 |archive-url=https://web.archive.org/web/20190107233345/https://news.artron.net/20131129/n540344.html |archive-date=2019-01-07 |dead-url=no }}</ref>。

== 文化背景 ==
[[唐代|唐代]]时，[[南诏国|南诏国]]在今[[中国西南|中国西南]]一带建立，[[佛教|佛教]]在该政权建立早期传入云南地区<ref name="国宝档案" />。“[[阿嵯耶|阿嵯耶]]”一名最早见于《[[南诏图传|南诏图传]]》，也被称为“真身[[观音|观音]]”，为[[南诏|南诏]]、大理国时期独有，是佛教传入大理地区后[[世俗化|世俗化]]的结果<ref name="云南省博物馆">{{cite web |title=宋银背光金阿嵯耶观音像 |url=http://www.ynmuseum.org/detail/174.html |website=云南省博物馆 |accessdate=2019-01-31}}</ref>。

== 外观描述 ==
该尊阿嵯耶观音像由纯[[金|金]]打造，观音束有高发髻，髻中藏[[佛|佛]]，戴化佛天冠，发髻两边有多股发辫垂到肩部，左边十一条，右边十条；观音面作女相，面相恬静安详，双目微睁前视，略带笑容；观音像身躯纤长，线条优美，脖子上戴着有珠宝的项圈，上身袒露，双臂上戴着三角形臂钏，右手手腕上戴着联珠手镯；左手掌心向外，手腕微微抬起，右手放置在胸前，食指和拇指相拈，两手结合组成了代表阿嵯耶观音像的“妙音天印”独特手印；下身着轻薄的长裙，长裙装饰如湿衣出水，束花形腰带，两侧伸出飘带，环绕在长裙上，裙饰阴刻U形纹，裙尾作鱼尾状；双脚赤足，足下有用来固定在底座上的方柱形[[榫卯|榫头]]。观音像身后饰银质背光，背光呈长葫芦形，或称舟形，镂空雕花，内作[[莲花|莲花]]纹，向外分别为优罗钵花、联珠纹和火焰纹<ref name="国宝档案" /><ref name="海外版" /><ref name="云南省博物馆" />。
