'''银氨溶液'''（{{lang-en|Tollens' reagent}}），也称'''多伦试剂'''、'''吐伦试剂'''、'''土伦试剂'''，指含有二[[氨|氨]]合[[银|银]][[氧化态|(I)]][[离子|离子]]（[Ag(NH<sub>3</sub>)<sub>2</sub>]<sup>+</sup>）的水溶液，一般由[[硝酸银|硝酸银]]或其他[[:Category:银化合物|银化合物]]与[[氨水|氨水]]反应制取，用作[[银镜反应|银镜反应]]的试剂。

==制备==
# 向[[硝酸银|硝酸银]]溶液中加入几滴[[氢氧化钠|氢氧化钠]]稀溶液，产生棕色的[[氧化银|氧化银]]沉淀。
# 向溶液中滴加浓[[氨水|氨水]]，直至棕色沉淀刚好溶解。所得的溶液即为银氨溶液，含[Ag(NH<sub>3</sub>)<sub>2</sub>]NO<sub>3</sub>(aq)。

反应方程式为：
:AgNO<sub>3</sub> + NaOH → [[氢氧化银|AgOH]]↓ + NaNO<sub>3</sub>
:AgOH + 2 NH<sub>3</sub> → [Ag(NH<sub>3</sub>)<sub>2</sub>]OH (aq)

滴加氢氧化钠时首先产生白色的[[氢氧化银|氢氧化银]]沉淀，非常不稳定，立即分解为棕色的[[氧化银|氧化银]]和水。硝酸银也可由一些不溶的银盐代替，但制备时不需加入氢氧化钠，直接加入氨水将不溶物溶解即可。

==用途==
银氨溶液与[[斐林试剂|斐林试剂]]类似，具有微弱的[[氧化性|氧化性]]，在自身被还原后，会在容器内壁附着上一层光亮的[[银|银]]层，即是所谓的[[银镜反应|银镜反应]]。一般用银镜反应来鉴定[[醛|醛]]的存在，相应的氧化产物是[[羧酸根|羧酸根]]离子，酸化得到[[羧酸|羧酸]]。[[甲醛|甲醛]]或[[甲酸|甲酸]]可被彻底氧化为[[二氧化碳|二氧化碳]]。

[[File:Redox_Tollens_Oxidationszahlen_C.svg|thumb]]发生银镜反应的方程式。]]

银氨溶液与[[2,4-二硝基苯肼|2,4-二硝基苯肼]]（Brady试剂）连用可以检出样品中醛基[[官能团|官能团]]的存在。若样品与2,4-二硝基苯肼作用呈阳性，产生黄色或红色的二硝基苯[[腙|腙]]沉淀，则可证明样品为[[羰基化合物|羰基化合物]]，可能是[[醛|醛]]或[[酮|酮]]。由于醛具还原性，而酮不具有还原性，此时再用银氨溶液测试，便可区分这二者官能团，醛将银氨溶液还原而产生银镜。此方法对[[醛糖|醛糖]]和[[酮糖|酮糖]]也基本有效。

银氨溶液也可用于鉴定末端[[炔烃|炔烃]]，现象是产生了白色的[[炔化银|炔化银]]（[[乙炔银|乙炔银]]、[[碳化银|碳化银]]）沉淀。

==安全==
