'''银河级星舰'''（Galaxy Class ）是《[[星际迷航|星际迷航]]》中，隶属于[[星际联邦|星际联邦]][[星际舰队|星际舰队]]的一个[[星际旅行舰级列表|星舰级别]]。在《[[星际迷航|星际迷航]]》的世界中，它是[[星际舰队|星际舰队]]最庞大的星舰。于2357年在[[火星|火星]][[乌托邦造船厂|乌托邦造船厂]]组装完成。其中最著名的是[[星际迷航：下一代|星际迷航：下一代]][[主角|主角]]舰[[联邦星舰进取号（NCC-1701-D)|联邦星舰进取号（NCC-1701-D)]]。

== 特点 ==

== 历史 ==
银河级花了长达14年来设计、研发，融合了当时[[星际联邦|星际联邦]]最先进的科技。

银河级星舰除完成为期7年的标准任务等探索任务，还曾在2367年反击[[博格人|博格人]]等战斗中发挥重要作用。
