'''得梅因紀事報'''<ref name="nytimes"/>（{{lang-en|The Des Moines Register}}）是[[艾奧瓦州|艾奧瓦州]]的每日[[早報|早報]]。紀事報在艾奧瓦州的大部分地區出售。

== 歷史 ==
[[愛荷華州|愛荷華州]][[得梅因|得梅因]]的第一份報紙的前身是《愛荷華星報》（{{lang|en|The Iowa Star}}）。1849年7月，巴洛·格蘭杰（Barlow Granger）在[[得梅因河|得梅因河]]和{{link-en|浣熊河|Raccoon River}}的交匯處的廢棄小木屋創刊。1854年，星報成為《愛荷華政論報》（{{lang|en|the Iowa Statesman}}），也是替[[民主黨|民主黨]]傳聲的報紙。1857年，政論報轉為一周三刊在愛荷華的州報。1870年，政論報成為《愛荷華州領導者》（{{lang|en|the Iowa State Leader}}），做為民主黨的報紙，並在接下來的32年與[[共和黨|共和黨]]的《愛荷華州紀事報》（{{lang|en|the Iowa Daily State Register}}）競爭<ref name="DMR-2015"/>。

1902年，[[道奇堡|道奇堡]]的喬治·羅伯茨（George Roberts ）買下紀事報和領導者，並將兩份報紙合併為早報。1903年11月7日，[[阿爾戈納_(愛荷華州)|阿爾戈納]]的銀行家{{link-en|老加德納·考爾斯|Gardner Cowles Sr.}}收購該日報，並命名為《紀事兼領導者報》（the Register and Leader）。1915年，成為《得梅因紀事報》<ref name="DMR-2015"/>。

在考爾斯家族掌握報社的所有權以後，紀事報成為愛荷華最具影響力的大報，並採用了「愛荷華眾之所望」（The Newspaper Iowa Depends Upon）的口號。隨著公路系統的改進，報紙通過火車分發到愛荷華州的所有角落，再以卡車運送。

2000年，紀事報在[[得梅因|得梅因]]南方開設一處新的印刷廠和分銷設施，新聞和廣告部門仍留在得梅因市中心。在原辦公地運營95年後，紀事報在2012年宣布他們將遷移到新的地點，首都廣場以東再過三個街區<ref>{{cite web |title=Des Moines Register signs lease for new space in Capital Square |url=http://blogs.desmoinesregister.com/dmr/index.php/2012/09/17/des-moines-register-signs-lease-for-new-space-in-capital-square |first=Donnelle |last=Eller |website=Des Moines Register |date=September 17, 2012 |archiveurl=https://archive.today/20130616121717/http://blogs.desmoinesregister.com/dmr/index.php/2012/09/17/des-moines-register-signs-lease-for-new-space-in-capital-square |archivedate=2013-06-16 |access-date=2018-10-31 |dead-url=no }}</ref>。2013年6月15日，紀事報從洛卡斯特街（Locust Street）715號搬到洛卡斯特街400號<ref name="DMR-2015"/>。 2014年，得梅因紀事報所屬的媒體大樓以160萬美元的價格出售，舊建築將重新開發為公寓和零售空間的混合型態<ref>{{Cite web |title = Des Moines Register building sold |url = http://www.desmoinesregister.com/story/money/business/development/2014/11/26/register-building-sold-lofts/19564271/ |last = Aschbrenner |first = Joel |website = Des Moines Register |date = November 27, 2014 |archiveurl = https://archive.today/20150520211216/http://www.desmoinesregister.com/story/money/business/development/2014/11/26/register-building-sold-lofts/19564271/ |archivedate = 2015-05-20 |access-date = 2018-10-31 |dead-url = no }}</ref>。

=== 2013年至今 ===
2018年，[[中國|中國]]在得梅因紀事報的增刊中，嚴厲批評了美國要對貿易戰的反擊。川普總統則抨擊中國干預美國選舉<ref name="nytimes">{{cite web|title=《中國日報》在美國報紙上出版增刊引發風波 |url=https://cn.nytimes.com/asia-pacific/20180928/china-trump-election-meddling/zh-hant/ |publisher=[[紐約時報|紐約時報]] |date=2018-09-28 |accessdate=2018-10-28}}</ref>。後來的10月，副總統[[彭斯|彭斯]]也以這份愛荷華的報紙當作同樣的事例<ref>{{cite web|title= Pence accuses China of trying to undermine Trump  |url=https://www.politico.com/story/2018/10/04/pence-china-undermining-trump-868240 |publisher=[[Politico|Politico]] |date=2018-10-04}}</ref>。

== 獎項 ==
紀事報獲得16個普立茲獎項<ref>{{cite web |title=Des Moines Register Pulitzer Prizes and awards |url=http://desmoinesregister.com/apps/pbcs.dll/article?AID=/99999999/HELP/40517008 |publisher=Des Moines Register |date=2012-05-18 |archiveurl=https://archive.today/20130122130415/http://www.desmoinesregister.com/viewart/99999999/HELP/40517008/Des-Moines-Register-Pulitzer-Prizes-awards |archivedate=2013-01-22 |access-date=2019-04-17 |dead-url=no }}</ref>。
* 6個國內報導獎（Pulitzer Prize for National Reporting）：1954, 1958, 1968, 1976, 1979, 1985
* 4個社論寫作獎（Pulitzer Prize for Editorial Writing）：1938, 1943, 1956, 2018
* 3個漫畫創作獎（Pulitzer Prize for Editorial Cartooning）：1924, 1943, 1963
* 1個攝影獎（Pulitzer Prize for Photography）：1952
* 1個特寫攝影獎（Pulitzer Prize for Feature Photography）：1987
* 1個突發新聞攝影獎（Pulitzer Prize for Breaking News Photography）：2010
* 1個[[普立茲公共服務獎|公共服務獎]]：1991
