'''德克萨斯州'''下辖254个县，辖县数量居美国各州之首<ref>{{cite web|title=How Many Counties are in Your State? |publisher=Click and Learn |url=http://www.clickandlearn.cc/FreeBlacklineMaps/Counties.htm |accessdate=2009-08-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20090422163024/http://www.clickandlearn.cc/FreeBlacklineMaps/Counties.htm |archivedate=2009-04-22 }}</ref>。最初德州当地政府在西班牙和墨西哥统治下以市级单位划分，在[[德克萨斯共和国|德克萨斯共和国]]1836年独立时其设有23个市，这些市组成了德州最初的县。最后一个设立的县是于1921年设立的[[凱內迪縣_(德克薩斯州)|凱內迪縣]]，但[[洛文縣_(德克薩斯州)|洛文縣]]是最新的县。其最早在1893年创建，但在1897年被撤销，然后在1931年重新设立。大部分新设立尤其是靠近西北的县均为在19世纪70年代从[[比爾縣_(德克薩斯州)|比爾縣]]中新创建。<ref name="TSHA">{{cite web |title = TSHA: County organization |publisher = The Texas State Historical Association |url = http://www.tshaonline.org/handbook/online/articles/muc10 |accessdate = 2011-08-19 }}</ref><ref>{{cite web |title = TSHA: Kenedy County |publisher = The Texas State Historical Association |url = http://www.tshaonline.org/handbook/online/articles/hcl13 |accessdate=2011-08-19 }}</ref><ref>{{cite web |title = TSHA: Loving County |publisher = The Texas State Historical Association |url = http://www.tshaonline.org/handbook/online/articles/hck04 |accessdate=2011-08-19 }}</ref>
