'''議會制'''又稱'''內閣制'''、'''議會民主制'''，是一種[[政治制度|政治制度]]，特點是「{{link-en|議會無上|Parliamentary sovereignty}}」，[[政府首脑|政府首脑]]（[[總理|總理]]或[[首相|首相]]）權力來自議會，授權有兩種途徑：第一是議會改選後的多數議席支持，第二是政府首腦贏得議會的[[不信任动议|信任投票]]。因此，所屬[[政党|政党]]未能贏得議會大選的政府首腦連同其內閣必須提出辭職，而未能通過議會[[不信任动议|信任投票]]的政府首腦，連同其內閣也必須辭職，由議會席位居多數的黨派中協商選舉產生新的首腦與內閣。議會制下政府的[[行政權|行政權]]與[[立法權|立法權]]不完全分離，確保其所領導的政府具民意基礎。

== 概述 ==
在採用這種政治體制的國家中，[[立法机构|立法机构]]與[[行政机构|行政機關]]並不完全分立。议会是国家权力的中心。議會民主制一般實行於民主國家，因此存在著多個參加議會[[選舉|選舉]]的[[政黨|政黨]]。議會內閣制的[[政府首脑|政府首脑]]（行政首長）與[[國家元首|國家元首]]（head of state）分開，其國家元首通常是儀式性職務，不享有實際的[[行政權|行政權]]，其权力一般仅限于任命议会中的多数党领袖或者多党政治联盟领袖担任政府[[总理|总理]]。少數議會制國家有民選總統的產生（如[[愛爾蘭|愛爾蘭]]、[[冰島|冰島]]），總統有部分職責如任命總理（需議會同意）與處理[[國防|國防]]與[[外交|外交]]事務。

議會制下的行政中樞為政府的[[內閣|內閣]]，由[[政府首腦|政府首腦]]（一般為[[总理|总理]]或[[首相|首相]]、[[日本內閣總理大臣|内閣總理大臣]]）領導。[[閣揆|閣揆]]及全體[[閣員|閣員]]，一般也是從議會[[議員|議員]]中產生，並在任職後保留議會中的議席，并向议会负责。[[政府首腦|政府首腦]]往往由議會多數黨領袖擔任。但是，當多數黨的席次不過半數時，幾個少數黨也有可能通過建立同盟積聚過半議席從而獲得任命[[政府首腦|政府首腦]]的權力，此時的政府稱為[[聯合政府|聯合政府]]，內閣稱為'''聯合內閣'''。如果内阁的提案遭到议会拒绝或否决，必须集体辞职或解散议会，重新选举。

在很多國家，內閣或內閣內的任何成員都可以通過議會對其的[[不信任投票|不信任投票]]而被免職。此外，[[政府首腦|政府首腦]]也可以決定解散議會，重新舉行議會選舉，來換取議會的配合。有些國家的制度是經過向[[國家元首|元首]]禮貌性報告後，由元首簽署命令。

有少數議會共和制國家（如[[南非|南非]]、[[波札那|波札那]]、[[馬紹爾群島|馬紹爾群島]]、[[諾魯|諾魯]]）行政首長就是國家元首（一般即是[[總統|總統]]），多半與其他議會制國家一樣，是由議會多數黨領袖擔任。不過任期固定，不會受議會影響。

議會民主制甚至可以被用於地方政府中：例如[[挪威|挪威]]首都[[奧斯陸|奧斯陸]]的行政體系就是立法體系的一部分，[[美国|美国]]、[[英国|英国]]、[[德国|德国]]、[[澳大利亚|澳大利亚]]等國家的一些[[州|州]]、[[郡|郡]]或[[邦|邦]]政府也是採行議會制。

由於[[政體|政體]]的不同，又可分為「[[议会制君主立憲制|议会制君主立憲制]]」及「[[议会共和制|议会共和制]]」。

== 歷史起源 ==
現代意義上的議會民主制可追溯到18世紀的[[英国|英国]]，雖然同一時期的[[瑞典|瑞典]]也已經擁有類似的政治制度，但其影響力較小。

議會民主制的開始可被認為是在1714年，當時的[[英國|英國]]理論上還是應由[[英国君主|英国君主]]主持[[內閣|內閣]]，挑選閣員。但因當時的國王[[乔治一世_(大不列颠)|喬治一世]]來自[[神聖羅馬帝國|神聖羅馬帝國]]的[[漢諾威選侯國|漢諾威選侯國]]，屬於[[德意志|德意志]]，母語為[[德語|德語]]，無法流利地使用[[英國內閣|英國內閣]]中慣用的[[英語|英語]]，於是[[敕命|敕命]][[財政部第一大臣|財政部第一大臣]]，[[輝格黨|輝格黨]][[黨魁|黨魁]][[罗伯特·沃波尔|沃波爵士]]，代替自己主持內閣會議，逐漸就發展出[[英國首相|英國首相]]一職。隨著[[英國國會|英國國會]]的逐漸民主化和議會權力的增大，國會開始控制[[英國政府|英國政府]]，並最終能夠決定君主必須任命誰來組建內閣。

議會民主制在[[歐洲大陸|歐洲大陸]]的傳播主要是在[[第一次世界大战|第一次世界大战]]之後，戰勝的國家[[英国|英国]]和[[法国|法国]]向戰敗國輸出自己的民主政治體系，例如[[德国|德国]]的[[威瑪共和|威瑪共和]]時期和[[奥地利|奥地利]]就是在此時採用議會民主制至今。早在19世紀歐洲政治力量的活躍已經為這些國家的民主化和議會民主制做好準備。一次大戰後，這種民主化運動被視為是抵抗具有廣泛群眾基礎的極端政治勢力的手段之一，也因此議會民主制遭到[[极右|极右]]政治勢力和許多群眾的抵制。

===二次大戰後===
歐洲的議會民主制真正成熟是在[[第二次世界大战|第二次世界大战]]之後，目前除了[[法国|法国]]採用[[半总统制|半总统制]]的政治體制（學理上稱為[[半总统制|雙首長制]]）外，大多數[[西欧|西欧]]國家都採用議會民主制。1990年[[东欧|东欧]][[社会主义国家|社会主义国家]]改制後，則多傾向使用直選元首下的雙首長制，但總統的權力受到限制，總理的權力較大。

反觀[[总统制|总统制]]，由於總統的行政權力非常大，易形成獨裁統治。如[[中亚|中亚]]、[[非洲|非洲]]及[[拉丁美洲|拉丁美洲]]許多採取總統制的國家，長期出現強人專政的局面。

== 优缺点 ==
=== 優點 ===
# 政府首長（首相／總理）由[[立法机构|立法机构]]最大政黨的黨魁出任，以確保政府的政策在立法機構得到支持。
# 政府全體[[內閣|內閣]]成員也是從議會[[民意代表|議員]]中產生，內閣執政可避免議員逐漸與實際社會脫節。
# 行政和立法機構有聯繫，可提高政府效率和避免政變發生。
# 稱職的政府首長可以不限任期。
# 雖然任期不限，但立法機構可隨時[[罢免选举|罷免]]不稱職的政府首長，不像[[总统制|总统制]]有不能在任期屆滿前使其下臺的缺點。
# 单次選舉消耗之經費比总统制少。
# 實行議會制的國家大部份成功。
# 是解決[[權力分立|三權分立]]行政及立法機關之間矛盾的其中一種方法（在議會制之下，[[行政机构|行政机构]]的領導來自立法機關的多數派。[[行政机构|行政]]、[[立法机构|立法]]並不完全分離。現代一般認為，成功和穩定的自由民主政制不一定需要徹底的三權分立。事實上，除了政黨沒有嚴重衝突的美國以外，所有開始實行民主便使用總統制的國家，它們的首次民主嘗試都以失敗告終。相反議會制的成功率反而較高）

=== 缺點 ===
# 二次大戰前，議會民主制在歐洲大陸遭遇到的挑戰是[[少數派政府|少數派政府]]所帶來的政治不穩定和極端政黨的上台。各政黨間的不合作導致「少數議會民主制」的產生，[[少數派政府|少數派政府]]往往導致政府遭到議會頻繁的不信任投票，引起政治危機。在一戰後的歐洲，此議會民主制初次嘗試失敗，極端勢力通過民主體制取得政權，卻可逃避「[[不信任投票|不信任投票]]」之民主制度制約，典型例子包括歐洲的獨裁者，德國的[[阿道夫·希特勒|阿道夫·希特勒]]、意大利的[[贝尼托·墨索里尼|贝尼托·墨索里尼]]、西班牙的[[弗朗西斯科·佛朗哥|弗朗西斯科·佛朗哥]]與葡萄牙的[[安東尼奧·德·奧利維拉·薩拉查|安東尼奧·德·奧利維拉·薩拉查]]也是议会制产生。
# 在議會中反對黨佔多數時，必然导致政府改组、[[不信任动议|不信任动议]]倒台或议会提前解散的结果，不像總統制，總統任期固定，不會被轻易撤換，总理不一定完成任期。
# 在實行兩院制的國家，若反對黨在其中一院控制多數（即[[較勁國會|較勁國會]]），容易引發憲政危機及政治僵局，如[[日本|日本]]及[[澳大利亚|澳大利亚]]。
# 在議會中執政黨佔絕對多數時，反對黨很難制衡執政黨，並且反對黨或會因爲席次過少而被邊緣化，如[[新加坡|新加坡]]。
# 最大政黨的黨魁出任總理，長期執政下容易形成某政黨長期[[一党优势制|一黨獨大]]的局面。如[[马来西亚|马来西亚]]及[[日本|日本]]。
# 多黨組成联合政府，会出现由非最大政黨的黨魁出任总理，在多黨联合执政下，总理权力及影響力較弱。
# 行政上很難趨向專業化，因爲議員的行政專業不如[[技術官僚|技術官僚]]。
# 容易形成政黨衝突。在多黨聯合政府中，只要其中一個主要政黨退出，政府便失去議會多数，導致被迫提前举行選舉，或改与另一政党组成联合政府。
# 政治權力容易集中，造成行政權和立法權重叠。
# 內閣是由議會多數黨組成，有可能造成同黨包庇，缺乏議會的監督功能。
# 在多黨制的國家中，如果沒有政黨組成政黨聯盟的話，極有可能變成[[懸空國會|懸空國會]]。

== 與投票制度的關係 ==
議會民主制本身並不一定會產生過多的小黨派和極端政黨，當議會民主制與[[比例代表制|比例代表制]]（Proportionate representation）的選舉計票方法結合時，容易產生較多小黨，導致[[少數派政府|小黨執政]]。在搭配[[比例代表制|比例代表制]]的制度下，一個黨派只要在全國獲得超過一定百分比（在很多國家是4-5%）的政黨選票，就可以進入議會，無論在個人選區中是否有本黨當選人。 而在[[小選區制|單一選區制]]（Plurality rule or Winner-Take-All）下，一個候選人必須在一個地方選區獲得比其他候選人多的票才能進入議會。也就是說，一個有極端主張的政黨，在[[比例代表制|比例代表制]]制度下只要有一定的支持者就有可能進行議會，但是在純粹的[[小選區制|單一選區制]]（不搭配比例代表）選舉下，由於極端黨派的選票被分散在各個地方選區（而且在任何一個地方選區極端政黨都很難拿到過半的選票獲得當選），極端政黨很難進入議會。例如實行純粹單一選區制的英國，極端民族主義黨派「[[英國國家黨|英國國家黨]]」全國得票超過百分之五，但在國會卻毫無席位。而日本實行單一選區制加政黨比例代表制的兩票，所以沒有在全國任何地方選區贏得席次的[[日本共產黨|日本共產黨]]，因為仍佔有一部份全國政黨得票而能分配席次。所以，選舉制度使用[[比例代表制|比例代表制]]還是純粹的[[小選區制|單一選區制]]，是影響政府組成與表現的重要的因素。

議會制政府，實行[[小選區制|單一選區制]]，容易導致國內[[政党制度|政黨體系]]的[[兩黨制|兩黨制]]，造成內閣與政府首腦的兩黨輪流執政，以英國為其典型。實行[[比例代表制|比例代表制]]，容易造成[[政党制度|政黨體系]]的[[多黨制|多黨制]]，導致聯合內閣與[[聯合政府|聯合政府]]。

== 相似體制 ==
=== 西敏制 ===
[[西敏制|西敏制]]是一種特殊的議會民主制，源於[[英国|英国]]，名字就取自英國議會的所在地[[伦敦|伦敦]][[威斯敏斯特宫|威斯敏斯特宫]]。這種體系被廣泛用於前英國殖民地國家，並已經成為大多數[[英联邦|英联邦]]國家採用的政治體制。在西敏制中，國家元首擁有較大的緊急權力，但受到慣例（而非法律）的制約而較少執行。其他形式的'''議會制'''，如[[德国|德国]]與[[意大利|意大利]]的體制，跟西敏制有很大不同。
