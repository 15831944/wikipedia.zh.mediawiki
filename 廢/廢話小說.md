{{engname|《廢話小說》|'' Out of the Blur ''}}是1996年出品的一部香港電影，由[[林海峰|林海峰]]執導<ref>{{Cite web |url= http://hkmdb.com/db/movies/view.mhtml?id=8207&display_set=big5|title =真心英雄（1996）| publisher =香港影庫|language = 中文}}</ref>，以都市人的狂想故事為電影題材，[[黎明|黎明]]、[[張學友|張學友]]、[[林海峰|林海峰]]、[[周慧敏|周慧敏]]、[[黃秋生|黃秋生]]、[[劉以達|劉以達]]等演出<ref>{{Cite web |url= https://moviecool.asia/film/qG3cbTCdBSc|title =廢話小說| publisher =影史庫|language = 中文}}</ref>，電影不設對白，僅以配樂和畫面來表達出劇情，在嘉禾[[利舞臺戲院|利舞臺戲院]]上映<ref>{{cite web|url=http://www.filmcritics.org.hk/film-review/node/2015/06/23/%E5%BB%A2%E8%A9%B1%E5%B0%8F%E8%AA%AA-%E5%BE%9E%E8%A7%B8%E8%A6%BA%E5%87%BA%E7%99%BC|title=廢話小說－從觸覺出發|author =登徒|date=1996年|publisher=香港影評庫| language = 中文}}</ref><ref>{{cite web|url=http://www.hkfilmblog.com/archives/11764|title=只有一句對白的電影《廢話小說》|date=2009年9月12日|publisher=講鏟片|language = 中文}}</ref>。

==情節介紹==
電影由十四個短篇故事組合而成，包括：死囚坐在電椅上等待行刑；白領職員趕上班；公司老闆與下屬和妻子的相處情況；互不認識的乘客在巴士上並肩熟睡；正在減肥的女孩邊吃邊看健康舞減肥的錄影帶；男女偷情；由人手進行交易操作的汽水機；小丑和魔術師在兔女郎的喪禮上表演；看守兇案現場的警察以無聊的玩意解悶；利用二手煙向人逼供；生日壽星許願；劫匪到打劫錶行；尋找遺失的泊車證；圖書館老師的寧靜生活，整部電影僅以音樂和畫面來呈現故事內容<ref>{{Cite web |url= https://movie.douban.com/review/1991020/|title =當年的官方介紹（廢話小說）| publisher =豆瓣電影|language = 中文}}</ref>。
