'''汐見車站'''（{{jpn|j=汐見駅|hg=しおみえき|rm=Shiomi eki|f=y}}）位於[[北海道|北海道]][[勇拂郡|勇拂郡]][[鵡川町|鵡川町]]汐見一區，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[日高本線|日高本線]]上的站。電報略號是'''{{lang|ja|オミ}}'''。

== 車站構造 ==
[[側式月台|側式月台]]1面1線的地面站。沒有站房，只有一座小型的候車室。

== 車站周邊 ==
* 汐見漁港
* 鵡川盛土墳墓群

== 歷史 ==
* 1959年（[[昭和|昭和]]34年）12月18日 - 設立。僅辦理旅客業務。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為北海道旅客鐵道（JR北海道）轄下的車站。
