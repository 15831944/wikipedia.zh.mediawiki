'''冬山幫'''（{{lang-en|'''Winter Hill Gang'''}}）是一個在[[麻省|麻省]][[波士頓|波士頓]]的[[黑社會|有組織犯罪]]聯盟，成員主要是[[愛爾蘭裔美國人|愛爾蘭裔美國人]]和[[意大利裔美國人|意大利裔美國人]]。它的名字來自波士頓以北的[[薩默維爾_(馬薩諸塞州)|薩默維爾]]的{{link-en|冬山 (薩默維爾，馬薩諸塞州)|Winter Hill, Somerville, Massachusetts|冬山}}附近。其成員包括波士頓黑幫{{link-en|詹姆斯·麥連|James McLean (mobster)|巴迪·麥連}}、[[白毛·巴爾傑|白毛·巴爾傑]]、{{link-en|霍伊·溫特|Howie Winter}}、{{link-en|喬·麥當勞|Joe McDonald (mobster)}}、{{link-en|強尼·馬圖納諾|Johnny Martorano}}和{{link-en|史蒂芬·佛萊明|Stephen Flemmi}}。他們在1965年由麥連和溫特統治下最具影響力，直到1979年由巴爾傑領導為止。

冬山幫這個名是由1970年代《{{link-en|波士頓先驅報|Boston Herald}}》的記者命名，雖然名稱幾乎沒有被公開用來作為參考。據說冬山幫成員參與了最典型的組織犯罪活動，他們或許最為人知的是在美國東北部操縱[[賽馬|賽馬]]比賽，以及向[[臨時派愛爾蘭共和軍|愛爾蘭共和軍]]（IRA）運送武器。<ref>{{Cite web |url = http://www.espn.com/horse-racing/story/_/id/9481509/the-jockey-crossed-whitey-bulger |title = The reverend's deal with the devil |author = Bill Finley |date = 2013-07-17 |publisher = ESPN.com |language = en}}</ref>二十一名成員和合夥人，包括溫特，在1979年被聯邦檢察官起訴。<ref>{{Cite web |url = http://archive.boston.com/news/packages/whitey/globe_stories/1998/whitey_and_the_fbi_part_4_sidebar_b.htm |title = The search for 'Whitey' Bulger |date = 1998-07-22 |publisher = Boston.com |language = en}}</ref>
