== 故事簡介 ==
因為身高太高感到自卑的大空遙，與因為身高長不高而煩惱的想要放棄沙灘排球的表妹、比嘉彼方在沖繩組成了沙灘排球組合，兩人一高一矮，一外行一經驗者，一個積極向上一個畏縮不前，完全相反的兩位少女在沙灘上展開青春動感的故事。

== 登場人物 ==
=== 沙灘排球社 ===
==== 「遙彼」組 ====
作品的中心人物。高中2年級<ref name="第1話">《[[Manga_Time_Kirara_Forward|Manga Time Kirara Forward]]》2015年10月號刊載的第1話內容。</ref>。作品中以「遙彼（{{lang-ja|はるかな}}）」組作稱呼。

; {{nihongo|大空遙|大空 遥（おおぞら はるか）|聲：[[優木加奈|優木加奈]]<ref name="character voice">{{cite web|url=http://www.harukana-receive.jp/staffcast/|title=製作人員&配音員|publisher=電視動畫「遙的接球」官方網站|accessdate=2018-06-23|language=ja}}</ref>}}
: 身高：172公分<ref name="第8話">《Manga Time Kirara Forward》2016年5月號刊載的第8話內容。</ref>
: 有著豐滿的胸部和臀部，令彼方看她的臀部不自覺的著迷。

; {{nihongo|比嘉彼方|比嘉 かなた（ひが かなた）|聲：[[宮下早紀|宮下早紀]]<ref name="character voice"/>}}
: 身高：151公分<ref name="第8話" />。
: 大空遥同为高二生的表妹，住在冲绳。性格文静内向，曾经打过沙排，但因身高不够而放弃了，这次将因为遥的关系再次开始打沙排。

第6話初登場。作品中以「惠紅愛（{{lang-ja|エクレア}}）」組作稱呼。

; {{nihongo|湯瑪斯·紅愛|トーマス·紅愛（トーマス·クレア）|Claire Thomas|聲：[[種崎敦美|種崎敦美]]<ref name="character voice"/>}}
: 身高：171公分<ref name="第8話" />。惠美理的雙胞胎姊姊。

; {{nihongo|湯瑪斯·惠美理|トーマス·恵美理（トーマス·エミリ）|Emily Thomas|聲：[[末柄里惠|末柄里惠]]<ref name="character voice"/>}}
: 身高：168公分<ref name="第8話" />。紅愛的雙胞胎妹妹。戴著眼鏡。與遙等人同樣是高中二年級。

==== 沙灘排球社的相關者 ====
; {{nihongo|大城明|大城 あかり（おおしろ あかり）|聲：[[木村千咲|木村千咲]]<ref name="character voice"/>}}
: 第13話初登場。

; {{nihongo|瑪莉莎|マリッサ（Marissa）|聲：[[庄司宇芽香|庄司宇芽香]]}}
: 第16話初登場，沙灘排球社教練<ref name="第16話">《Manga Time Kirara Forward》2017年1月號刊載的第16話內容。</ref>。紅愛與惠美理的母親，擔任沙灘排球社教練之前是美國西海岸的職業沙灘排球教練<ref name="第16話" />。身高比遙還要高的身材<ref name="第16話" />。雖然教育方式嚴格，被定評為[[斯巴達教育|斯巴達]]教練<ref name="第16話" /><ref name="第17話">《Manga Time Kirara Forward》2017年2月號刊載的第17話內容。</ref>，但在社員的眼中是恩師。

=== 其它學校的勁敵 ===
==== 彩紗、成美組 ====
作品一開始登場。居住在[[沖繩縣|沖繩]]。與大空遙和比嘉彼方同屆。高中1年級曾獲高中沙灘排球冠軍<ref name="第2話">《Manga Time Kirara Forward》2015年11月號刊載的第2話內容。</ref>，其實力和外表被外界稱為「沙灘排球界的偶像」<ref name="第14話">《Manga Time Kirara Forward》2016年11月號刊載的第14話內容。</ref>。

; {{nihongo|遠井成美|遠井 成美（とおい なるみ）|聲：[[島袋美由利|島袋美由利]]<ref name="character voice"/>}}
: 彼方的小學沙灘排球隊友。從小學4年級開始與彼方認識<ref name="第17話">《Manga Time Kirara Forward》2017年2月號刊載的第17話內容。</ref>。

; {{nihongo|立花彩紗|立花 彩紗（たちばな あやさ）|聲：[[伊藤加奈惠|伊藤加奈惠]]<ref name="character voice"/>}}

==== 「愛衣舞」組 ====
第9話初登場。作品中以「愛衣舞（{{lang-ja|あいまい}}）」組作稱呼。

; {{nihongo|砂川舞|砂川 舞（すながわ まい）|聲：[[小池理子|小池理子]]<ref name="character voice"/>}}
: 和遙一開始是在買用作制服的泳裝時，爭奪起同一件泳裝而認識。之後在原作第11話（電視動畫版第5話）與學姊愛衣搭檔參加沙灘排球大賽第1回對戰的對手是遙和彼方時再次見面。

; {{nihongo|棚原愛衣|棚原 愛衣（たなはら あい）|聲：[[鬼頭明里|鬼頭明里]]<ref name="character voice"/>}}
: 第22話再登場時目前就讀大學，並擔任全國高中沙灘排球大會沖繩預賽的裁判。更早之前邀請主動加入排球社的學妹舞說了一句「排球是打倒個頭大的人的運動」，但自從在室內排球大賽中敗退以來自覺對舞撒了謊，想著試著不撒謊要為舞做點什麼而參加了沙灘排球大賽。

==== 「陽柑」組 ====
第20話初登場。{{lang|ja|やんばる}}南高中2年級，平常日曬出一身黑的姊妹。作品中以「陽柑（{{lang-ja|ようかん}}）」組作稱呼，或者是「新垣、新垣」組稱呼。

; {{nihongo|新垣陽菜|新垣 陽菜（あらがき ような）|聲：[[明坂聰美|明坂聰美]]}}
: 高中2年級<ref name="第20話">《Manga Time Kirara Forward》2017年5月號刊載的第20話內容。</ref>。柑菜的姊姊。戴著眼鏡。

; {{nihongo|新垣柑菜|新垣 柑菜（あらがき かんな）|聲：[[高橋未奈美|高橋未奈美]]}}
: 高中1年級<ref name="第24話">《Manga Time Kirara Forward》2017年9月號刊載的第24話內容。</ref>。陽菜的妹妹。

=== 遙與彼方的親屬 ===
; {{nihongo|比嘉空|比嘉 ソラ（ひが そら）|聲：[[山田榮子|山田榮子]]}}
: 遙與彼方的外婆。與丈夫（遙與彼方的外公）同居。
: 漫畫版只在對話中登場。在動畫版則追加了她的造型設定，以及遙與彼方寄居的生活和享用[[沖繩料理|沖繩料理]]的場面。

'''遙的母親'''（聲：[[藤田奈央|藤田奈央]]）
: 電視動畫版第1話登場。原作裡只在對話中登場。
: 原作第1話提到她認為讓遙在東京獨自生活會添麻煩，遙因此搬到沖繩與彼方生活，但兩人分居的緣由則未有提及。動畫第1話裡則是因獨自到英語圈國家出差而與遙分居。

'''彼方的母親'''（聲：[[金子有希|金子有希]]）
: 第17話初登場。在彼方讀小學的時候，與丈夫2人於同一天死去<ref name="第19話">《Manga Time Kirara Forward》2017年4月號刊載的第19話內容。</ref>。生前曾經建議她跟轉學生成美一起打沙灘排球<ref name="第17話">《Manga Time Kirara Forward》2017年2月號刊載的第17話內容。</ref>。之後彼方遵從母親的建議，開始跟成美一起打沙灘排球以培養感情。
: 電視動畫版第1話設定在彼方家的佛堂遺照中登場。

; {{nihongo|龜吉|亀吉（かめきち）}}
: 電視動畫版第1話登場。彼方飼養的烏龜。

== 影響作者的作品 ==
原作者如意自在在Twitter表示，是因為看了[[濱弓場雙|濱弓場雙]]的漫畫《[[花舞少女|花舞少女]]》而開始畫遙的接球<ref>{{Cite tweet|author=如意自在|user=nyoi_jizai|number=718090587241472004|script-title=ja:私がきららフォワードではるかなレシーブを連載するきっかけになったのがハナヤマタです。...|date=2016-04-07|accessdate=2018-08-18|language=ja}}</ref>。連載時間是從2015年10月號開始，至2018年4月號迎接《花舞少女》完結為止，兩部漫畫都同樣在《Manga Time Kirara Forward》進行連載。
