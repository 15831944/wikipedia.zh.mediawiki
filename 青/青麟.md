'''青麟'''（{{bd|？||1854年|catIdx=Q青}}），[[表字|字]]'''墨卿'''，图们氏，[[满洲|满洲]][[正白旗|正白旗]]人。[[道光|道光]]二十一年（1841年）进士，选[[庶吉士|庶吉士]]，授[[编修|编修]]，迁[[中允|中允]]擢[[侍讲|侍讲]]。五迁至[[内阁学士|内阁学士]]。督[[江苏|江苏]][[學政|學政]]有官譽。[[咸丰|咸丰]]二年，擢[[户部侍郎|户部侍郎]]。督催丰北塞决工程。三年出督[[湖北|湖北]][[学政|学政]]（相當于省教育廳長），调[[礼部侍郎|礼部侍郎]]。

== [[德安|德安]]守功 ==
*时[[太平軍|太平軍]]由[[江西|江西]]回窜[[湖北|湖北]]，青麟按试[[德安|德安]]，闻戰報停试，督率[[知府|知府]]易容之募[[乡勇|乡勇]]筹防守，府城获全。疏陈军事，请[[湖北|湖北]]、[[江西|江西]]、[[安徽|安徽]]三省合剿，以期得力。
*四年，授[[湖北巡抚|湖北巡抚]]。城中兵仅千人，[[荆州|荆州]]将军台涌署[[湖廣总督|湖廣总督]]，未到任；而[[太平軍|太平軍]]由[[黄州|黄州]]进至[[汉阳|汉阳]]、[[汉口|汉口]]，渡江欲第二次扑攻[[武昌|武昌]]。青麟督[[总兵|总兵]]杨昌泗、[[游击|游击]][[侯凤岐|侯凤岐]]与[[副都统|副都统]]魁玉水陆合击，擊退[[太平軍|太平軍]]；复败之豹子海、鲁家港，毁贼垒五。

== [[武昌|武昌]]省城失守 ==
*已而[[太平軍|太平軍]]扑塘角、鲇鱼套，逼攻省城，青麟在[[武昌|武昌]]武胜门督战，城中忽火起，土匪内应，兵尽溃，[[武昌|武昌]]第二次失守。青麟将自殺，众阻止自殺擁之逃到[[长沙|长沙]]，折赴[[荆州|荆州]]。

== [[咸豐|咸豐]]怒斬 ==
*初，[[清文宗|清文宗]]闻青麟出家財犒军，甚嘉之，至是愤武昌屡失，弃城越境，罪尤重，诏曰：“青麟简任封圻，正当贼匪充斥，武昌兵单饷匮。朕以其任学政时保守德安，念其勤劳，畀以重任。省垣布置，屡次击贼获胜。八十馀日之中，困苦艰难，所奏原无虚假，朕方严催援兵接应。六月初间，魁玉、杨昌泗等连破贼营，但能激厉力战，何致遽陷？'''婴城固守，解围有日，犹将宥过论功。纵力尽捐躯，褒忠有典，岂不心迹光明？'''乃仓皇远避，径赴长沙，直是弃城而逃。'''长沙非所辖之地，越境偷生，何词以解？若再加宽典，是疆臣守土之责，几成具文，何以对死事诸臣耶！'''朕赏罚一秉大公，岂能以前此微劳，稍从末减？俟到荆州时，交[[官文|官文]]传旨正法。”遂令[[官文|官文]]斬'''青麟'''。
*逝後，[[曾國藩|曾國藩]]為青麟抱不平。
