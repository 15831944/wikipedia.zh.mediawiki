'''青海省铁卜加草原改良试验站'''是[[青海省人民政府|青海省人民政府]][[青海省农牧厅|农牧厅]]下辖的[[正处级|正处级]]单位<ref name="a1">{{Cite web |url=http://news.ifeng.com/a/20180522/58397729_0.shtml |title=省铁卜加草原改良试验站移交省农牧厅管理 |accessdate=2018-06-12 |author= |date=2018-05-22 |work= |publisher=凤凰网，来源：青海日报|language=Zh-hans }}</ref>，位于[[海南藏族自治州|海南藏族自治州]][[共和县|共和县]][[石乃亥乡|石乃亥乡]]境内，成立于1959年8月<ref name="a2">{{Cite web |url=http://www.qhgh.gov.cn/html/5574/262536.html |title=铁卜加草原改良试验站 |accessdate=2018-06-12 |author= |date=2012-03-07 |work= |publisher=共和县人民政府网站|language=Zh-hans }}</ref>，其辖区以'''铁卜加草改站'''之名列为共和县的一个[[类似乡级单位|类似乡级单位]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|63/25/21/404/000}}}}</ref>。

2012年时，铁卜加草改站隶属青海省三江源集团有限责任公司，是企业化管理的[[事业单位|事业单位]]。内设4个科室、2个下属公司。主营业务：草地资源动态调查与监测、[[牧草|牧草]]良种繁育与推广、退化草地综合治理、草原鼠虫害监测与预报、牧草种子生产与销售等<ref name="a2"/>。是[[甘肃农业大学|甘肃农业大学]]、[[青海大学|青海大学]]、[[湟源牧校|湟源牧校]]的定点实习基地<ref>{{Cite web |url=http://news.swun.cn/info/1045/37618.htm |title=青藏高原研究院科研人员赴青海省铁卜加草原改良试验站考察  |accessdate=2018-06-12 |author=泽丹卓玛 |date=2017-07-25 |work=审核：文勇立，责编：米治鸿 |publisher=西南民族大学新闻网|language=Zh-hans }}</ref>。2018年5月16日<ref name="a1"/>，调整为青海省农牧厅管理的正处级单位。

== 试验站机构 ==

*科室：党政综合办公室、生产经营办公室、科研项目办公室、财务室<ref name="a2"/>。
*下属公司：青海省铁卜加草地生态科技开发有限公司、青海省铁卜加畜牧业科技开发有限公司<ref name="a2"/>。
