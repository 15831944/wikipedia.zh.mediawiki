'''青岛康有为故居'''（'''胶澳总督副官官邸旧址'''），位于中国[[山东省|山东省]][[青岛市|青岛市]][[市南区|市南区]]福山支路5号，始建于1899年，建筑具有德式和中式双重风格。现为[[山东省文物保护单位|山东省文物保护单位]]。

==概况==
该建筑位于在[[汇泉湾|汇泉湾]]畔、[[小鱼山_(青岛)|小鱼山]]麓，原为1899至1902年任德国胶澳总督副官的海军上尉利林克隆男爵（{{lang|de|Hauptmann Freiherr von Liliencron}}）的住宅<ref>{{cite web |url=http://www.qdda.gov.cn/front/laoqingdao/preview.jsp?subjectid=12259376569844725001&rootID=12259393205621563001&ID=26453|title=《总督副官住宅》|accessdate=2010-05-31|work=|publisher=青岛档案信息网}}</ref>。1924年[[康有为|康有为]]购入此房作为寓所，并在此居住直至1927年3月病逝。1949年后青岛市文物事业管理局曾在此办公。2000年由青岛市政府投资拆除重建，迁出其中的住户，并作为纪念馆正式对外开放。

{{Wikisourcehas|甲子六月領得德國舊提督樓}}
在[[戊戌变法|戊戌变法]]失败后，作为维新派领袖的康有为长期流亡海外，直至辛亥革命后才得以回国。1917年，康有为参与[[张勋复辟|张勋复辟]]失败后来青岛避难，这也是他第一次来青岛，在给友人的信中赞誉青岛“红瓦绿树，碧海蓝天，不寒不暑，可车可舟，中国第一”<ref name="multiple1">{{cite web|url=http://qpinet.qingdao.gov.cn/nianjianku.nsf/42c84fdc7fa67b40482570d60007f00b/13e6aba394ba047a482570f90023e7fa?OpenDocument|title=《康有为故居》|accessdate=2010-04-12|work=张树枫|publisher=青岛政务信息公众网}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，并萌生了在此居住的意愿。此处原为1923年他第二次来青岛时市政当局安排的下榻之处，康有为对此居所十分满意，称“青岛此屋之佳，吾生所未有”、“此屋卑小，而园甚大，望海碧波，仅距百步”<ref name="multiple1" />，次年出资购入后题名为“'''天游园'''”，悬[[宣统帝|宣统帝]]所题的“'''天游堂'''”于正厅<ref>{{cite web |url=http://www.qdda.gov.cn/front/qingdaojingguan/preview.jsp?subjectid=12259376569844725001&ID=1806374|title=《福山支路》|accessdate=2010-04-12|work=鲁海|publisher=青岛档案信息网}}</ref>，并对各居室作了精心布置，其中内部的家具陈设多为[[溥偉|溥伟]]移居时所赠。在此居住期间，康有为曾多次会见文化名人，并潜心研究书艺<ref>{{cite web |url=http://travel.sina.com.cn/spot/qdkywgg.html|title=《青岛康有为故居》|accessdate=2010-04-12|work=|publisher=新浪旅游频道}}</ref>。

建筑为砖木结构，包括地下室共三层，修复后的建筑面积为1128平方米，周边为庭院，南侧设宽大的石阶，由此进入一楼正厅，其中安放有一座康有为塑像，门上悬挂的“康有为故居”匾额为[[刘海粟|刘海粟]]题写。陈列展室设于一楼和二楼，其中客厅、书房和卧室为复原陈列，其他房间设“康有为生平史迹展”、“康有为的世界之路” 图片展及墨宝展、藏品展等<ref>{{cite web|url=http://www.qdwhw.com/newsdetails.aspx?news=710|title=《青岛康有为故居纪念馆》|accessdate=2010-04-12|work=|publisher=青岛文化网}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
