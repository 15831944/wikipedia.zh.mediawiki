《'''青草迴旋詩'''》（{{lang-th|ลำนำหญ้า}}，{{lang-fr|Rondeau de l'herbe}}<ref>[https://library.stou.ac.th/odi/princess/detail1.html แนะนำหนังสือ]</ref>）是[[泰國|泰國]][[詩琳通公主|詩琳通公主]]于1971年所作的[[現代詩|現代詩]]<ref>{{cite web |url=http://www.chinanews.com/2002-10-21/26/234605.html |title=图文特写：泰王国诗琳通公主海口行 |last= |first= |date=2002-10-21 |website= |publisher= |access-date= |quote=}}</ref>，格式为法国16世纪流行的{{link-en|迴旋詩|Rondeau (forme fixe)}}，全文共13行<ref>{{cite web |url=http://www.readers365.com/Shijie/233/mydoc000.htm |title=青草回旋诗① |last= |first= |date= |website=欧美新诗精选 |publisher= |access-date= |quote=}}</ref>。《青草回旋诗》以[[北京外国语大学|北京外国语大学]]教授季難生和張青的译本通行<ref>{{cite web |url=https://m.hujiang.com/th/p171894/ |title=(泰国)诗琳通 |last= |first= |date=2010-07-30 |website=沪江 |publisher= |access-date= |quote=}}</ref>；另有王曄和邢慧如的译本，名为《小草的歌》<ref>{{cite web |url= http://www.chinawriter.com.cn/n1/2016/0708/c405176-28538436.html|title=诗琳通公主 |last= |first= |date=2016-07-08 |website=中国作家网 |publisher=川妮 |access-date= |quote=}}</ref>，收录在1985年11月[[少年儿童出版社_(上海)|少年儿童出版社]]所出版的[[小草的歌|同名诗集]]中<ref>{{cite news |author=XPO |last= |first= |date=1986-01-05 |title=泰国公主诗琳通诗集 《小草的歌》中文版在中国出版 |url= http://eresources.nlb.gov.sg/newspapers/Digitised/Article/lhzb19860105-1.2.35.6|newspaper=[[联合早报|联合早报]] |location=第18页 |access-date= }}</ref><ref>{{cite web |url=http://www.afinance.cn/new/Print.asp?ArticleID=558550&Page=4 |title=泰国公主素颜游武汉：哎，来碗热干面 |last= |first= |date=2013-04-13 |website= |publisher= 中国时刻网 |access-date= |quote=}}</ref>。

2019年4月4日，诗琳通公主在[[北京|北京]][[淮扬府_(餐厅)|淮扬府]]用餐时，洛扬送公主一把写有诗歌的[[扇子|团扇]]作为生日礼物。扇子由[[扬州市|扬州市]]青年书法家协会副会长刘进题写，正面用[[隶书|隶书]][[竖排|竖排]]书写“诗琳通公主阁下<sup>雅赏</sup>”；背面为诗琳通公主早年诗作《青草回旋诗》<ref>{{cite web |url=http://news.yznews.com.cn/2019-04/08/content_7034034.htm |title=泰国榴莲做青团淮扬菜配绿杨春 这份公主菜单上都有啥 |last= |first= |date=2019-04-08 |website=扬州网 |publisher=扬州晚报|access-date= |quote=}}</ref>，其中“融合在风儿的歌唱里”被误抄成“融合在风儿的歌声里”；后记“诗琳通公主一九七一年所作《青草回旋诗》，以季难生、张青译本录之。乙亥桃花开时，拐上静波恭录。”
