'''南宋官窯青瓷簋'''是一件現存於[[臺灣|臺灣]][[國立故宮博物院|國立故宮博物院]]的文物，原典藏於清宮[[壽皇殿|壽皇殿]]<ref name = "貴似晨星"/>，其全高12.7公分、口徑17.5×19.5公分、底徑16.0×16.0公分、足徑15.8公分。<ref name = "貴似晨星">{{Cite book|title=《貴似晨星：清官傳世12致14世紀青瓷特展》|last=|first=|publisher=國立故宮博物院|year=2016|isbn=978-957-562-766-9|location=|page=69}}</ref>

==年代==
[[南宋|南宋]]初創基即嚴重感覺代表正統的禮樂用器之闕如。[[靖康之變|靖康之變]]，金人在[[汴京|汴京]]大肆搜刮，裝載二千零五十車的[[宋朝|宋朝]]府庫積蓄，北運燕京，主要的禮儀祭器幾乎皆不免。<ref>{{cite journal en |author1 = 蔡玫芬 |authorlink = |coauthors = |year =1997|title = 〈論「定州白瓷器，有芒不堪用」句的真確性及十二世紀官方瓷器之諸問題〉|journal =《故宮學術季刊》 |volume = 15 |issue = 2|page =88 |id = |url = |accessdate = |quote = }}</ref>[[南宋|南宋]]紹興年間，朝野上下醞釀而出復興[[北宋|北宋]]文化的思潮，讓 《宣和博古圖》成為重建禮制器用的典範。影響所及，也讓[[南宋|南宋]][[官窯|官窯]]器中出現一類仿銅器，此品即為其中的一個例證。[[浙江省|浙江省]]浙江市郊壇下[[官窯|官窯]]窯址及[[南宋太廟遺址|南宋太廟遺址]]也曾出土相似的器類。<ref name = "貴似晨星"/>

==外觀==
器形模仿青銅[[簋|簋]]，歛口略呈橢圓形，碩腹，平底，圈足略高。口足磨損而加鑲銅邊，口下飾弦紋兩道，腹兩側各按飾一螭形耳，耳下削平，露出黑灰色的胎體。胎骨較厚，通體施濃潤青釉，釉色粉青，釉面滿佈冰裂紋，邊稜釉薄處微現粉青光澤。<ref name = "貴似晨星"/>

==器形淵源==
[[簋|簋]]是指有圓形器口、圓腹、圈足和一定大小等基本器形，用以盛裝黍稷的[[青銅器|青銅器]]。這些基本器形隨足型、腹壁曲線和耳型的差異，又可再細分成若干類。[[簋|簋]]型器自[[商朝|商朝]]早期開始萌芽發展，到了[[西周|西周]]中、晚期和[[春秋時期|春秋]]早期，[[簋|簋]]型器臻於鼎盛，並由盛而衰。<ref>{{Cite book|title=《商周青銅粢盛器特展圖錄》|last=|first=|publisher=國立故宮博物院|year=1985|isbn=957-562-180-8|location=|pages=25、56-64}}</ref>[[戰國|戰國]]時期，喪葬制度發生了變化。[[三晉|三晉]]兩周地區的貴族大墓自[[戰國|戰國]]早、中期起逐漸用陶禮器代替青銅禮器隨葬，仿照銅器形式的[[鼎|鼎]]、豆、壺、[[簋|簋]]、簠、甗等等成套成組的生產。到了[[漢武帝|漢武帝]]時期，用於喪葬的[[鼎|鼎]]、敦、盤、匜等仿銅禮器雖還有所生產，但倉、[[灶|灶]]、[[井|井]]、爐等模型[[明器|明器]]的燒造正在急遽增長，到了[[東漢|東漢]]時期，舊式陶禮器顯著減少以至絕跡。<ref>{{Cite book|title=《中國陶瓷史》|last=|first=|publisher=文物出版社|year=1982|isbn=978-7-5010-0953-4|location=|pages=97-106}}</ref>直到宋代因為金石學的發展，再加上[[宋徽宗|宋徽宗]]試圖再現三代制禮作樂，進而編成《重修宣和博古圖》一書，而成為日後仿古器的製作藍本。
