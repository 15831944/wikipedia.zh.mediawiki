'''青岛饭店'''是[[中华人民共和国|中国]][[青岛市|青岛市]]的一家饭店，现位于[[市南区|市南区]][[香港中路|香港中路]]66号<ref Name=qing>{{cite web |url=http://www.sohu.com/a/133764992_349516|title=青岛饭店：忘不了的儿时味道|accessdate=2017-04-13|work=|publisher=}}</ref><ref Name=zhong>{{cite book |author=李明 |title=《中山路:一条街道和一座城市的历史》|year=2009 |publisher=中国海洋大学出版社 |location=青岛 |isbn=9787811252101}}</ref>。该饭店前身是位于今[[中山路_(青岛)|中山路]][[曲阜路_(青岛)|曲阜路]]路口东北角的青岛咖啡店，建于1930年代<ref Name=zhong/><ref Name=pan>{{cite web |url=http://arc.dailyqd.com/2014-05/07/content_57681.htm|title=盘点青岛开埠之初西餐业 希腊人开咖啡店|accessdate=2014-05-07|work=|publisher=青报网}}</ref>。1956年改为[[公私合营|公私合营]]青岛咖啡饭店。1965年饭店扩建，原建筑被拆除，建成一栋五层大楼，并改名为青岛饭店<ref Name=pan/>，1971年10月开业<ref Name=qing/><ref Name=zhong/>。1977年，饭店向东扩建九层新楼。青岛饭店在[[计划经济|计划经济]]时代的青岛曾占有重要地位<ref Name=qing/>。2004年，因饭店所在街区面临改造，搬迁至东部新区的[[香港中路|香港中路]]66号，9月28日重新开业<ref Name=qing/><ref Name=zhong/>。
