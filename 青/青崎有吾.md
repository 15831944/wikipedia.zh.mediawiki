'''青崎有吾'''（{{bd|1991年|6月28日}}<ref name="cy">{{cite web |title=青崎有吾 （アオサキユウゴ ） |url=http://www.tsogen.co.jp/np/author/1792 |website=東京創元社 |accessdate=2018-11-18 |archive-url=https://web.archive.org/web/20160515013221/http://www.tsogen.co.jp/np/author/1792# |archive-date=2016-05-15 |dead-url=no }}</ref><ref>{{cite web |author1=青崎有吾 |title=誕生日なのでお母さんが金平糖買ってきてくれた |url=https://twitter.com/AosakiYugo/status/615117441681653760 |website=Twitter |accessdate=2018-11-18 |archive-url=https://web.archive.org/web/20160403224535/https://twitter.com/AosakiYugo/status/615117441681653760# |archive-date=2016-04-03 |dead-url=no }}</ref>），是[[日本|日本]][[推理小說|推理小說]]家，生於[[神奈川縣|神奈川縣]][[横滨市|横滨市]]<ref name="shinbun">{{cite web |title=体育館で殺人事件が起きるからこのタイトル以外ありえない──一つの物から謎が解かれる国名シリーズへのオマージュ |url=http://toshoshimbun.jp/books_newspaper/week_description.php?shinbunno=3088&syosekino=5518 |website=図書新聞 |accessdate=2018-11-18 |archive-url=https://web.archive.org/web/20160304185554/http://toshoshimbun.jp/books_newspaper/week_description.php?shinbunno=3088&syosekino=5518# |archive-date=2016-03-04 |dead-url=no }}</ref>，畢業於[[明治大学|明治大学]]文學系，在校期间是明治大学推理研究会的成员。<ref name="cy"/>青崎有吾作品的特色是将校园生活的青春气息与[[艾勒里·奎恩|艾勒里·奎恩]]式的逻辑推理相结合。<ref>{{cite web |title=日系推理重磅佳作《敲响密室之门》中文版引进 |url=https://ibook.ifeng.com/48235/news.shtml?srctag=pc2m&back |website=凤凰网 |accessdate=2018-11-20}}</ref>

進入明治大學後，青崎有吾向有關[[輕小說|輕小說]]的獎項遞交了兩份作品，但沒有獲獎。2012年《體育館之謎》獲得第22屆{{link-ja|鮎川哲也獎|鮎川哲也賞}}<ref>{{cite web |title=第22回鮎川哲也賞決定 |url=http://www.tsogen.co.jp/award/ayukawa/22nd/ |website=東京創元社 |accessdate=2018-11-18 |archive-url=https://web.archive.org/web/20180216205049/http://www.tsogen.co.jp/award/ayukawa/22nd/# |archive-date=2018-02-16 |dead-url=no }}</ref>，青崎有吾成為史上首個獲得此獎的[[平成|平成]]時代出生的作家。<ref name="shinbun"/>2014年，《水族館之謎》入圍第14屆[[本格推理大獎|本格推理大獎]]小說部候补作品。<ref>{{cite web |title=本格ミステリ大賞受賞作・候補作一覧1-18回 |url=http://prizesworld.com/prizes/mystery/hmys.htm |website=文学賞の世界 |accessdate=2018-11-18 |archive-url=https://web.archive.org/web/20180708074223/http://prizesworld.com/prizes/mystery/hmys.htm# |archive-date=2018-07-08 |dead-url=no }}</ref>
