'''青馬大橋'''（{{lang-en|Tsing Ma Bridge}}）是全球最長的行車[[鐵路|鐵路]]雙用[[懸索吊橋|懸索吊橋]]，以及全球第11長的懸索吊橋。大橋主跨長1,377米，連引道全長為2,160米。大橋屬於[[香港8號幹線|香港8號幹線]][[青嶼幹線|青嶼幹線]]的一部份，跨越[[馬灣海峽|馬灣海峽]]，將[[青衣島|青衣]]和[[馬灣|馬灣]]連接起來。青馬大橋為[[香港道路|香港道路]]重要的一部份，因為它連同[[汲水門大橋|汲水門大橋]]，共同擔當著連接[[大嶼山|大嶼山]]、[[赤鱲角|赤鱲角]][[香港國際機場|香港國際機場]]與市區的唯一行車通道。車速限制每小時80公里。青馬大橋屬青馬管制區範圍，[[隧道交通督導員|管制區人員]]獲授權執行《[[青馬管制區條例|青馬管制區條例]]》。

== 簡介 ==
青馬大橋由{{tsl|en|Mott MacDonald|莫特·麦克唐纳公司}}設計。「青」是指[[青衣|青衣]]，而「馬」是指[[馬灣|馬灣]]，它離海面高度62米，而橋塔高度則為206米。吊起大橋橋身的主纜直徑為1.1米，而主纜主跨是由33,400條直徑5.38毫米的鋼絲，以空中絞織方法組成。每條主纜會產生5萬[[公噸|公噸]]的[[拉力|拉力]]，總長度達16萬[[公里|公里]]，足以環繞[[地球|地球]]4次。大橋的結構鋼材主要在[[英國|英國]]及[[日本|日本]]製造，並在[[中國|中國]][[廣東省|廣東省]][[東莞市|東莞市]][[珠江|珠江]]附近組合成50件組件，再以躉船運往施工地點。重量方面，結構鋼材重量5萬公噸，橋塔和錨碇混凝土重量則為50萬公噸<ref name=r1>香港機場核心計劃工程簡訊，1997年4月出版</ref>。

青馬大橋採用雙層設計，橋的露天上層為3線雙程分隔快速公路，有蓋的下層則為2條[[港鐵|港鐵]][[東涌綫|東涌綫]]及[[機場快綫|機場快綫]]的共用路軌，和2條供緊急時（例如[[颱風|颱風]]吹襲和嚴重交通意外時）使用的單線行車道路。青馬大橋於1992年5月25日開始動工興建，耗資71.44億港元。大橋連同[[青嶼幹線|青嶼幹線]]其餘部份，於1997年4月27日開幕，並由前[[英國首相|英國首相]][[戴卓爾夫人|戴卓爾夫人]]、[[港督|港督]][[彭定康|彭定康]]及[[布政司|布政司]][[陳方安生|陳方安生]]主持開幕儀式。而大橋則於同年5月22日正式通車。

青馬大橋屬青馬管制區範圍內，現由交通基建管理合約有限公司負責管理。

== 旅遊景點 ==
青馬大橋是香港的一個重要地標和景點，所以吸引了世界各地的遊客參觀。為應付遊客，政府在青衣西北部設立[[青嶼幹線訪客中心|青嶼幹線訪客中心]]及觀景台。訪客中心設有大橋模型、相片和互動遊戲；從螺旋式的小路繞著圓柱形的觀景台而上，遊人便可以較近距離或高角度，遠眺青馬大橋、汲水門大橋及汀九橋，觀景台設有行人天橋連接訪客中心與停車場。訪客中心及觀景台於1997年隨青嶼幹線通車對外開放，最初由[[新機場工程統籌署|新機場工程統籌署]]負責管理及運作。部門於1998年解散後，展覽中心交由民政事務總署葵青民政事務處負責管理。

由於青馬大橋不設人行道，遊客無法於橋上步行。現時政府只在[[渣打香港馬拉松|國際馬拉松賽事]]舉行期間開放橋面，作為比賽場地。另外亦曾於2008年5月2日將部分橋面給予[[2008年夏季奧林匹克運動會|2008年夏季奧林匹克運動會]]的火炬手[[2008年夏季奧林匹克運動會香港區火炬接力|傳遞聖火]]。

== 殊榮 ==
1999年，青馬大橋榮獲由[[美國|美國]][[建築|建築]]界選出的「二十世紀十大建築成就獎」得主之一，與[[英法海峽隧道|英法海峽隧道]]及[[三藩市|三藩市]][[金門大橋|金門大橋]]等其他九項世界級工程共同享有這項殊榮。

== 前往方法 ==
遊人可以從[[港鐵|港鐵]][[青衣站|青衣站]]A1出口的小巴站，搭乘[[新界區專線小巴308M線|新界區專線小巴308M線]]至青嶼幹線觀景台站下車（部分小巴不在訪客中心停站，遊人須於上車時向司機查詢）；或者從港鐵青衣站A1出口，搭乘計程車前往青嶼幹線觀景台。
