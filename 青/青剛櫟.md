'''青剛櫟'''（[[學名|學名]]：{{lang|la|''Quercus macrocarpa'' or ''Cyclobalanopsis glauca''}}，{{lang|en|Ring-cupped Oak}}、{{lang|en|Japanese Blue Oak}}、{{lang|en|Glaucous-leaf Oak}}、櫟/{{lang|en|lì}}(力)/）為[[殼斗科|殼斗科]][[櫟属|櫟属]][[青剛櫟亞屬|青剛櫟亞屬]]<ref>{{Cite web |url=http://tesri.coa.gov.tw/show_nature_detail.php?id=852&f_title=&f_title2=&f_author=&f_serial=&f_beg_yy=1993&f_beg_mm=3&f_end_yy=2010&f_end_mm=3&f_pno=show_nature |title=台灣原生殼斗科植物各屬之分類初探 |accessdate=2011-08-11 |work= |publisher=李權裕 |date=2005-03-01 }}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>的常綠喬木，別稱有白校欑、校欑、鐵椆、青栲

== 特徵 ==
常綠中型[[喬木|喬木]]，樹高可達20公尺，[[樹皮|樹皮]]灰褐色、略光滑，有不明顯縱向細紋裂，[[葉|葉]]形為長卵或橢圓形，單葉互生，長8至14[[公分|公分]]，寬2.5至6公分，葉前端尾狀漸尖，且具鋸齒狀葉緣，基部楔形，葉面深綠至淡綠色，表面平滑，葉背為粉白色，有短毛疏生，花期於每年3~4月，結果期長達3個月，於每年10月持續直到12月<ref>{{Cite web|url=http://ecotech.tesri.gov.tw/html/content04_1_3.htm#|title=原生景觀植物各論|accessdate=2011-08-13|work=|publisher=行政院農委會特生中心|date=|deadurl=yes|archiveurl=https://web.archive.org/web/20080527153932/http://ecotech.tesri.gov.tw/html/content04_1_3.htm|archivedate=2008-05-27}}</ref>；[[果|果]]為[[堅果|堅果]]，橢圓形，面光滑，高1至1.6公分，徑0.9至1.4公分，頂端有一圓錐形突尖，基部圓形，其堅果與另一台灣特有種[[森氏櫟|森氏櫟]]果實相較，其型態較修長。
新鮮邊材為乳白色，木質線及導管均甚顯著。<ref>劉業經：《台灣重要樹木彩色圖誌》，1970年4月出版，第254頁，台灣：林務局、國立中興大學。</ref>

== 分佈 ==
主要分布於[[中國|中國]]、[[朝鮮半島|朝鮮半島]]、[[印度|印度]]及[[日本|日本]]；在中國，喜生於[[海拔|海拔]]60至2600[[公尺|公尺]]的山坡地或溪谷，常成闊葉林或針闊葉混交林，分佈省份從華北、華中到華南，甚至雲南、西藏等省區均可見，是中國遍佈面積最廣的闊葉樹種之一。

在[[台灣|台灣]]，主要分佈海拔2000公尺以下之闊葉樹林內，在平地亦可常見作為都市綠化樹種。

== 用途 ==
木材質地堅硬，是許多建築用材及器具把柄的重要材料，以前也常做為鐵路枕木使用<ref>{{Cite web|url=http://www.taipei-expopark.tw/ct.asp?xItem=39300&ctNode=7367&mp=3|title=花博期間園藝植栽資料庫|accessdate=2011-08-13|work=|publisher=台北市政府|date=}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。<br />另外，青剛櫟所結的堅果內含豐富[[澱粉|澱粉]]、[[蛋白質|蛋白質]]等營養素，是許多哺乳類動物──如[[台灣黑熊|台灣黑熊]]的主要食物來源<ref>{{Cite web|url=http://np.cpami.gov.tw/chinese/index.php?option=com_content&view=article&id=4396&Itemid=26|title=青剛櫟果實豐收－台灣黑熊痕跡頻率創歷年紀錄第2高|accessdate=2011-08-13|work=|publisher=內政部營建署台灣國家公園管理處|date=2011-07-19|deadurl=yes|archiveurl=https://web.archive.org/web/20131212065205/http://np.cpami.gov.tw/chinese/index.php?option=com_content&view=article&id=4396&Itemid=26|archivedate=2013-12-12}}</ref>，雖因味道苦澀不被人所食用，但因其造型奇特，常被人們拿來做童玩等手工藝品。
木屑可以用於香菇的培養基底，並且保留幹，為永續樹種。
