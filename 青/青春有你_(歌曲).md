《'''青春有你'''》是中國[[爱奇艺|爱奇艺]][[青春有你|同名真人秀節目]]的主題曲，由100位青春有你訓練生共同演唱，於2019年1月14日發行<ref>{{Cite news|url = http://ent.huanqiu.com/movie/yingshi-neidi/2019-01/14062661.html|title =《青春有你》主题曲首发 百位训练生亮相|author = 张玮|publisher = 环球网|date = 2019-1-14|accessdate=2019-1-14|language = zh-hans}}</ref>。

== 發行 ==
歌曲於2019年1月14日發行，在发行前一天公布，<ref>{{Cite web|url = https://weibo.com/6870452244/HbJrinMkD|title =#青春有你# 是所有人期待已久的舞台，是所... 来自爱奇艺青春有你 - 微博 |author = 青春有你|date = 2019-1-13|accessdate=2019-1-14|language = zh-hans}}</ref>同时放出10秒的片段预告<ref>{{Cite web|url = https://weibo.com/6870452244/HbLY14FYq|title = #青春有你主题曲#春卷抢先带来10s试听~ 1月... 来自爱奇艺青春有你 - 微博 |author = 青春有你|date = 2019-1-13|accessdate=2019-1-14|language = zh-hans}}</ref>。同日演出视频播出，由[[李汶翰|李汶翰]]担任中心位置，并與何昶希、[[儲曉祥|卡斯柏]]、嘉羿、連淮偉、夏翰宇、車慧軒、馮俊杰、[[管櫟|管櫟]]、葉子銘、邱薄翰、[[姜聖民|姜聖民]]和徐方舟处于舞台最高处。<ref>{{Cite web|title=《青春有你》主题曲李汶翰C位，张艺兴解说|publisher=新京报|url=http://www.bjnews.com.cn/ent/2019/01/14/539309.html|date=2019-01-14|author=张赫|accessdate=2019-01-14|language=zh-hans}}</ref><ref>{{Cite web|title=《青春有你》超帥C位是「韓國出道的他」！ 主題曲MV曝光了|publisher=東森新聞|url=https://star.ettoday.net/news/1356042|date=2019-01-14|author=李湘文|accessdate=2019-01-14|language=zh-hant}}</ref>
