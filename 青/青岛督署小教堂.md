'''青岛督署小教堂'''（Evangelische Gouvernementskapelle）是[[胶州湾租借地|青岛德租时期]]的一座[[信义会|信义会]]教堂，原址位于中国[[山东省|山东省]][[青岛市|青岛市]][[市南区|市南区]][[江苏路_(青岛)|江苏路]][[湖南路_(青岛)|湖南路]]路口附近，建于1899年，1998年拆除<ref Name=christ>{{cite web |url=http://www.tsingtau.org/christuskirche-tsingtau-qingdao/|title=Christuskirche in Tsingtau (Qingdao)}}</ref><ref Name=ming>{{cite book|author=王栋|title=《青岛影像 1898-1928：明信片中的城市记忆》|year=2017|publisher=中国海洋大学出版社|location=青岛|isbn=978-7-5670-1113-7}}</ref>。

==位置==
德租时期，教堂建在青石山（[[江苏路基督教堂_(青岛)|德国信义会教堂]]所在的小山）以南，因此青石山也被德国人称为“小教堂山”（Kapellenhügel）。教堂的大门朝向今天的江苏路。由于湖南路在江苏路以东的路段直到1909年才建成，因此教堂可被看作是江苏路旁的建筑。1901年，[[青岛督署学校|青岛督署学校]]在教堂北侧建成。1914年，教堂西侧与北侧各建有一座住宅楼，即今天的[[江苏路_(青岛)#俾斯麦街施蒂洛住宅旧址|江苏路5号、湖南路7号]]与[[张店路#哈赫迈斯特住宅旧址|张店路13号]]，湖南路5号后来又在教堂与湖南路7号之间建成。因此，教堂的位置可以确定在今湖南路5号东侧、张店路13号与江苏路小学老楼南侧、湖南路张店路路口西北角，紧靠湖南路。该位置现在是[[青岛江苏路小学|江苏路小学]]的一个体育场。

==历史==
1897年11月14日，德军占领胶州湾，此后随着越来越多的德军的入驻，建立一座教堂的需求愈加紧迫。1899年胶澳总督府出资开始修建一座临时的教堂，即为督署[[小圣堂|小教堂]]，由1898年2月到达青岛的城市规划师兼青岛第一位专业建筑师[[马克斯·克诺普夫|马克斯·克诺普夫]]（Max Knopff）设计。总督府原本希望该教堂由德国新教教会和[[天主教|天主教]][[圣言会|圣言会]]共同使用，但圣言会拒绝了。1899年平安夜，教堂竣工，同善会牧师[[尉礼贤|卫礼贤]]进行了献堂布道，信义会牧师[[和士谦|和士谦]]主持了[[平安夜|平安夜]]礼拜<ref Name=christ/>。自建成后该教堂一直作为军民两用教堂使用，因此建设经费由总督府承担。1910年10月23日，[[江苏路基督教堂_(青岛)|新的信义会教堂]]在青石山上竣工，青岛外籍新教教徒的礼拜改为在新教堂举行，原小教堂改为[[青岛督署学校|督署学校]]的体育馆（Turnhalle）<ref Name=christ/><ref>{{cite web |url=http://www.tsingtau.org/kataster-karten-tsingtau-1914/|title=Katasterkarten Tsingtau}}</ref>。[[青岛江苏路小学|青岛江苏路小学]]成立后，该建筑又成了小学的一部分。1998年江苏路小学校舍改造时这座近百年的教堂被拆除。<ref Name=christ/><ref Name=ming/>

==建筑特色==
作为一座临时教堂，这座教堂相对小的规模、长条状平面布局、单堂设计与屋顶敷设的[[仰合瓦|中式灰瓦]]都体现出其应急性。但是几乎占满两个立面的大[[三角楣饰|山花]]、侧面的拱窗与[[扶壁|扶壁]]及各种细节线脚装饰又为该堂增添了德租早期建筑所共有的复古风格。另外该堂的平面布局也遵循了欧洲本土教堂的习惯，将[[教堂建筑#祭坛|祭坛]]设置在东端，大门在西端。
