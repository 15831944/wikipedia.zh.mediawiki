《'''[[青春有你|青春有你]]'''》是2019年[[中華人民共和國|中國]][[愛奇藝|愛奇藝]]推出的偶像男團競演養成類真人秀節目，此節目本為《[[偶像練習生|偶像練習生]]》第二季<ref>{{Cite web|title=《偶练2》更名《青春有你》，张艺兴或将留任|publisher=新京报|url=http://www.bjnews.com.cn/ent/2018/12/05/527778.html|date=2018-12-05|author=张赫|accessdate=2019-01-13|language=zh-hans}}</ref><ref>{{Cite web|title=《偶像練習生2》百人名單 驚見已出道「台灣男星」參賽|publisher=中國時報|url=https://www.chinatimes.com/realtimenews/20181206003362-260404|date=2018-12-06|author=李家穎|accessdate=2018-12-10|language=zh-hant}}</ref>。

== 第一集 ==
* 訓練生在出場前需根據自己的目標選擇想入座的座位號碼，並戴著相應的號碼名牌入場。
* 出場後，訓練生並非依自選的號碼入座，而是根據眼前小屏幕所顯示的實力初判斷結果排名入座。排名前60的訓練生們入座場內第一等候區，而排在後39位的訓練生們則必須離場，入座場外第二等候區。
* 依照各自經紀公司分組並進行等級評價表演，之後由在場的導師們根據訓練生們的綜合能力進行A、B、C、D或F的初次等級評價。最佳的A代表「優秀」，而最差的F則代表該訓練生「還沒有準備好」。

== 第二集 ==
* 延續上一期繼續進行訓練生的初次等級評價。99位訓練生表演完畢後，進入到B班的有11人、C班24人、D班44人、F班20人。本季無人在初次等級評價被分配到A班，首開節目先例。
* 發布第一次任務考核：節目同名主題曲《青春有你》表演。訓練生需在三天內在分配到的班級內與其他同等級的訓練生們一起學習主題曲的演繹和舞蹈。
* 訓練生根據初次等級分配宿舍。

== 第三集 ==
* 延續上一期繼續進行第一次任務考核的聲樂和舞蹈訓練。為期三天的訓練後，訓練生們以單獨錄製主題曲表演視頻的方式進行考核，且錄製機會只有一次。
* 導師在此次的考核中將進行訓練生們的再次等級評價。再次等級評價結果為A班10人、B班14人、C班20人、D班31人、F班24人。
* 重新分班後，訓練生根據所獲得的再次等級分組進行主題曲舞台表演彩排。根據現場彩排情況，青春製作人代表張藝興在徵得總導演的同意後決定提升12位進步顯著的訓練生們的等級（新等級以斜體註明，之前所在等級以刪體注明）。最終等級評價結果為A班13人、B班15人、C班20人、D班31人、F班20人，而此最終等級評價決定了訓練生們在主題曲表演舞台上所站的位置。
* 進行A班訓練生主題曲表演中心位選拔：獲得A等級的訓練生依序在主題曲原編舞基礎下加入一個亮點動作，並由其他班級的訓練生們和青春製作人代表張藝興共同選出表現最佳的訓練生並給予主題曲表演中心位置。此次選拔中獲得主題曲表演中心位置為樂華娛樂訓練生李汶翰。
* 每位訓練生在春節前獲得5分鐘與家人通話的時間。
* 發布第二次任務考核主題：小組對決（第一場公演）。99位訓練生分為16組，每2組將會被分配到同首歌曲，共8首。
* 每位訓練生早前已以投票的方式選擇出最想組隊的隊員，而獲得最高票數的訓練生（李汶翰）將擁有優先選擇歌曲和隊友的權利。其餘的組隊將由抽球的方式決定。首先由青春製作人代表張藝興以抽球的方式隨機選出一名訓練生，而該訓練生在選擇隊友組隊後抽取另一位訓練生為對手，並由該對手選擇隊友組隊對抗，以此類推。
* 選定歌曲後，訓練生們進行組內分工，包括選擇出組內的隊長和中心位置人選。

* 延續上一期繼續進行小組對決任務考核。個別導師在公演前給予每組訓練與指導後，根據現場彩排情況，在分配到同首歌曲的兩個隊伍中選擇出團隊表現較好的一組，並給予該組登上舞台進行公演的權利。落敗的另一組則不能進行舞台公演，只能完成練習室版本的表演。

== 第五集 ==
*本期進行第一次排名公布，排名結果綜合青春製作人在2019年1月21日至2月9日的線上助力以及小組對決公演舞台一百倍的現場助力值，總數為77,919,015點助力值，而排在前9名的訓練生佔了當中的40,472,233點助力值。本次排名在前60的訓練生將進入下一階段的訓練和表演。此外，藝術指導團也根據訓練生在第一階段的訓練情況以及舞台表演考核視頻作出舞台實力和潛力評判，從而推薦10位訓練生進入下一階段的訓練和表演。因此，無法進入排名前60但獲得藝術指導團認可的訓練生也有資格進入下一階段的訓練和表演。
*排名結果分批公布，首先公布排名第59名至第51名的訓練生，之後是第50名至第41名、第40名至第31名、第30名至第21名、第20名至第10名、然後是出道位第9名至第1名。關鍵性的第60名則在最後公布，公布前揭曉了排在最靠近這個位置的4名訓練生（姚博嵐、圳南、齊浩然、吳飛）。藝術指導團所推薦的10位訓練生當中，8位已排在前60內（以粉色背景註明），因此僅有2位訓練生獲得推薦進入下一階段的訓練和表演（以黃色背景註明）。其餘沒有獲得推薦的34名訓練生則確定被淘汰{{notetag|節目無完整公布排在第60名後的訓練生排名與助力值結果，只在節目尾聲通過字幕揭曉部分訓練生的排名（詳情可參見「[[#總排名|分集總排名]]」第五期欄目）。排在最靠近第60名的4名訓練生中，姚博嵐獲得關鍵的第60名名額，圳南的排名在字幕中揭曉為第61名，而沒有被揭曉排名的齊浩然和吳飛則順位為第62或63名。}}。
*排名公布前，訓練生與自家經紀公司的訓練生組隊進行入場式並入座。青春製作人代表張藝興在入場後給每位訓練生分派新年紅包，收穫空紅包的其中一名訓練生（嘉羿）獲得張藝興在春節晚會上表演的道具「糖福祿」為禮物。
*訓練生拔河遊戲：訓練生與節目組工作人員進行拔河對抗，以爭奪的方式換取火鍋葷菜、鍋底以及飯後甜品。每輪拔河派出5位訓練生參與遊戲，而參與遊戲的訓練生們將根據想爭奪的食材從10位身穿相應食材上衣的節目組工作人員中挑選5位進行對抗。訓練生如在該輪遊戲中獲勝，將獲得相對應的獎勵。遊戲結束後，訓練生獲得所有食材並啟程至火鍋店用餐。
*最青春TOP榜：每位訓練生從所有訓練生中選擇一位心目中的「最青春擔當」（即為顏值擔當）。

== 第六集 ==
*李榮浩創作興趣課：李榮浩導師給部分訓練生上一堂創作興趣課。李榮浩導師先讓訓練生們展示個人原創作品，之後以編曲為主題給訓練生佈置了任務作業。根據李榮浩導師的判斷，每位訓練生被定下了他們各自最不擅長的曲風，而訓練生需根據所分配到的曲風在三天內完成新的創作和編曲，並在三天後的第二堂課展示作品。
*發布第三次任務考核主題：位置測評（第二場公演）。本次任務考核提供訓練生三大不同位置的曲目（聲樂、舞蹈、說唱）供他們選擇，而訓練生將根據各自心儀的團體表演位置來進行選擇相應的曲目進行公演。62位訓練生分成12個小組進行對決，聲樂位置類別為5組、舞蹈位置類別3組、說唱位置類別4組。訓練生們在進行選歌前在身上佩戴了代表自己位置的胸牌，皆為訓練生們最初選擇能力的位置。
*訓練生歌曲選擇順序根據第一次排名公布的結果來決定，排名靠前的訓練生擁有優先選擇權。除了兩首舞蹈位置類別歌曲的人數被定為6人之外，每首歌曲的人數固定為5人。歌曲人數滿了後，還沒有進行選擇歌曲的訓練生將不能選擇該歌曲。本次組隊過程中選擇同一首歌曲的訓練生自動組成一隊，而訓練生在選擇歌曲前不會知道排在前面的訓練生所選擇的曲目，因此訓練生們未必能與各自心儀的訓練生組成一隊。選定歌曲後，訓練生們進行組內分工，包括選擇出組內的隊長和中心位置人選。
*每組在登台演出前由各大類別的負責導師與教練介紹出場與串場，李榮浩導師負責聲樂組、蔡依林導師和徐明浩教練舞蹈組、歐陽靖導師和艾福傑尼教練說唱組。每組公演結束後，現場的每位青春製作人根據訓練生個人表現進行現場助力。現場的青春製作人可為多位訓練生助力，但只能給每位訓練生助力一次。如果沒有想要助力的訓練生，青春製作人也可選擇棄權。現場助力結束後，每組前往結果公布區領取現場助力結果。本期將由4個小組率先登台進行公演。

== 第八集 ==
*發布第四次任務考核主題：主題考核（第三場公演）。本次任務考核將協助訓練生找到屬於自己的表演風格，為他們準備了五首不同表演風格、曲風與主題的歌曲供選擇。本場公演的五首表演歌曲是由國內外頂尖音樂製作人與製作團隊為訓練生量身打造的新歌曲目，訓練生即是它們的原唱。在第二次排名公布前，在線的62位訓練生將臨時分配到五首歌曲表演組的其中一組進行訓練，而此分組安排已在2019年2月22日至月26日由青春製作人通過愛奇藝線上投票決定。
*本次任務考核歌曲分配結果發布形式為訓練生依序向青春製作人代表張藝興領取存放著訓練生個人歌曲卡的信封，而訓練生需根據歌曲卡上的數字移動至相應的訓練室與其他獲得同樣數字的訓練生一起等待分配結果（獲得同樣數字的訓練生皆分配到同首歌曲並形成一組）。五組訓練生集結完畢後，由張藝興在個別訓練室揭曉被分配到該組的表演歌曲。
*在本次考核任務訓練期間將進行第二次排名公布。因此，僅排名在前35的訓練生有資格登上舞台進行本場的公演，而其餘的訓練生則在訓練中途離開節目，不能參與舞台表演。在第二次排名公布前，每組暫時分為兩個小組進行訓練，並進行組內分工，選擇出隊長和每個小組的中心位置人選。
*本期進行第二次排名公布，排名結果綜合青春製作人在2019年2月15日至3月9日的線上助力以及位置測評公演舞台所獲得的獎勵助力值數，總數為263,358,683點助力值。本次排名在前35的訓練生將進入下一階段的訓練和表演。此外，藝術指導團也根據訓練生在第二階段的訓練情況以及舞台表演考核視頻作出舞台實力和潛力評判，從而推薦6位訓練生進入下一階段的訓練和表演。因此，無法進入排名前35但獲得藝術指導團認可的訓練生也有資格進入下一階段的訓練和表演。
*排名結果分批公布，首先公布排名第34名至第19名的訓練生，之後是第18名至第10名，然後是出道位第9名至第5名、第4名至第3名、以及壓軸的第2名至第1名。在公布第4名之前也揭曉了第1名的4位候選人（即為排名前4的訓練生）。關鍵性的第35名則在最後公布，公布前也揭曉了排在最靠近這個位置的4名訓練生。藝術指導團所推薦的6位訓練生當中，3位已排在前35內（以粉色背景註明），因此僅有3位訓練生獲得推薦進入下一階段的訓練和表演（以黃色背景註明）。其餘沒有獲得推薦的的24名訓練生則確定被淘汰。
*藝術指導團黃豆豆公開課：藝術指導團成員黃豆豆給全體訓練生上一堂公開課，分享他從藝數十年來對舞蹈的感悟。黃豆豆老師也在公開課的尾聲發布了節目第二波名為「星星的孩子」的正能量公益行動，即為訓練生關注與幫助患有孤獨症孩童的公益任務項目。全體訓練生在本次公益任務中前往了北京市通州區珍愛孤獨症兒童康復中心接觸了患有孤獨症的孩童們並幫助他們與外界進行交流。

== 第九集 ==
*藝術指導團滕矢初公開課：藝術指導團成員滕矢初給全體訓練生上一堂公開課，分享他從藝數十年來對古典音樂的感悟。滕矢初老師也在公開課的尾聲發布了節目第三波名為「青春有你愛心林」的正能量公益行動，為訓練生配合全國第41個植樹節而踐行的植樹公益任務項目。全體訓練生在本次公益任務中前往了太陽村參與植樹並打造了「青春有你愛心林」，為綠化環境做出一份貢獻。
*延續上一期繼續進行主題考核任務。經過第二次排名公布後，在線的38名訓練生根據主題考核任務小組的最終限定人數進行重新分組。在五個小組當中，僅一組達到限定人數而無需移動組內成員，其餘的四組則是因人數超員或缺員而必須進行人員調整。人數超員的小組根據組內最適合詮釋主題曲目表演的人員進行投票，按照小組的限定人數給組內每位成員打分（第一名滿分，第二名減一分，以此類推）。根據此次票選個人的總分高低，確定最終離開各超員組的訓練生（以深灰色背景註明）。因僅有一組人數缺員，所有從超員組離開的訓練生將移動到該組進行主題考核任務的訓練和表演（以斜體註明，之前所在組合以深灰色背景與刪體注明）。
*公演結束後，現場的每位青春製作人根據訓練生在本階段的舞台專業表現進行現場助力。每位青春製作人共擁有兩次助力機會，首先從五組主題考核任務小組中選出「最為推薦」的一組，再從「最為推薦」的小組中選出一名「最為推薦」的訓練生。最終獲選為「最為推薦」的小組獲得50萬點助力值的獎勵，而此獎勵助力值的組內分配規則為組內「最為推薦」的訓練生（即為個人助力值最高訓練生）獲得當中的20萬點助力值（以粉色背景註明），而另外30萬點助力值則由組內其他的訓練生平分（以黃色背景註明）。
*本輪公演現場助力排名發布僅公布部分結果，排在前三名的訓練生的具體排名和最終獲得50萬點助力值的小組將在第三次排名公布時揭曉。
*「為家鄉代言，我可以！」正能量分享：訓練生在本期第三場公演後以小組的形式向青春製作人推薦他們個別家鄉的特色景點與美食。

== 第十集 ==
*藝術指導團王潔實公開課：藝術指導團成員王潔實給全體訓練生上一堂公開課，分享他從藝數十年來對歌唱的感悟。王潔實老師也在公開課的尾聲發布了節目第四波名為「青春敬老，陪伴有你」的正能量公益行動。王潔實老師與全體訓練生在本次公益任務中前往了康語軒俱樂部北京孫河老年公寓為這所敬老院裡的老人舉辦一場溫馨快樂的生日會。訓練生們在此次的活動中分為兩組，一組負責陪伴老人，而另一組負責為生日會準備才藝表演。
*本期進行第三次排名公布，排名結果綜合青春製作人在2019年3月15日至3月23日的線上助力以及主題考核公演舞台所獲得的獎勵助力值數，總數為118,430,860點助力值。本次排名在前20的訓練生將進入最終舞台任務的訓練和表演。此外，藝術指導團也根據訓練生在第三階段的訓練情況以及舞台表演考核視頻作出舞台實力和潛力評判，從而推薦3位訓練生進入下一階段的訓練和表演。因此，無法進入排名前20但獲得藝術指導團認可的訓練生也有資格進入最終舞台任務的訓練和表演。
*上一期未揭曉主題考核公演排在前三的訓練生的具體排名在本期第三次排名公布前發布，也確定了最終獲得50萬點獎勵助力值的小組。
*排名結果分批公布，首先公布排名第19名至第10名的訓練生，然後是出道位第9名至第5名、第4名至第1名。在公布第4名之前也揭曉了第1名的4位候選人（即為排名前4的訓練生）。關鍵性的第20名則在最後公布，公布前先揭曉了排名第38名至第24名的訓練生，之後揭曉第23名至第20名的訓練生。藝術指導團所推薦的3位訓練生當中，1位已排在前20內（以粉色背景註明），因此僅有2位訓練生獲得推薦進入最終舞台任務的訓練和表演（以黃色背景註明）。其餘沒有獲得推薦的的16名訓練生則確定被淘汰。
*青春輔導員井柏然公開課：訓練生先前通過不記名的形式寫下對其他訓練生想說的話並投入道具小井裡，隔天由青春輔導員井柏然作為主持邀請收穫信件的訓練生現場分享信件中的青春故事。
*青春有禮：訓練生根據第三場公演的所在組合組成五大隊為支持他們的青春製作人贏得禮物。訓練生以團隊形式齊心協力越過海洋球障礙到另一端領取禮物盒，並通過同一個海洋球障礙把禮物盒帶回遊戲起點。每組基礎時長為100秒，而根據此前青春製作人在微博上為各組選擇的遊戲「法寶」，每組獲得了不同的加時進行遊戲的權利（《Time》組加時20秒、《火種》組加時25秒、《I'm Sorry》組加時15秒、《迷宮》組加時5秒、《重塑 Rebuild》組加時15秒）。最終各組訓練生為青春製作人贏得的禮物數量分別是《Time》組19件、《火種》組16件、《I'm Sorry》組14件、《迷宮》組19件、《重塑 Rebuild》組14件。
*發布最後一次任務考核主題：最終舞台（決賽公演）。本次任務考核為最終九人組合成團前的最後一場舞台公演。在線的22名訓練生分為兩組詮釋兩首最終舞台評價曲，而曲目選擇以及位置分配根據訓練生在第三次排名公布的成績進行選擇。訓練生在自己想選擇歌曲的位置上貼上自己的頭像，排名低的訓練生給予優先選擇。排名高的訓練生如有想要選擇的歌曲位置被排名低的訓練生先選擇了，排名高的訓練生有權選擇該位置並把之前選擇該位置的訓練生更換到歌曲裡其他尚未被其他訓練生選擇的位置（也可把該訓練生換到另一首歌曲的位置），但每位訓練生只有一次移動其他訓練生的權利。每首歌曲的中心位置將根據訓練生消化歌曲亮點部分的能力選出，而每位訓練生都有機會競選該位置。

== 第十一集 ==
*導師的一封信：每位導師給全體訓練生寫了一封鼓勵信件，並在節目開場時以朗讀的方式向大家分享。
*藝術指導團蔣大為公開課：藝術指導團成員蔣大為給全體訓練生上第二堂公開課，繼續向大家分享他數十年來的從藝經驗和領悟。
*「騰格里沙漠鎖邊生態林項目」公益行動：在第三場排名公布被淘汰的訓練生當中，排在前五名的訓練生（谷藍帝、胡文煊、師銘澤、徐炳超、丁飛俊）前往阿拉善沙漠進行節目第五波的正能量公益行動。訓練生們參觀了當地的一顆古樹，並且為此次的百萬森林計畫在沙漠種下他們的「青春之樹」。
*發布導師合作舞台任務：本次任務為非考核性質的特別舞台任務，聚齊了節目所有導師與教練和訓練生共同完成一次合作舞台。此次的組隊規則為訓練生在15分鐘內發送信息給想組隊的導師，而每位訓練生只能向一位導師發送信息。該導師如果在限時內向訓練生回覆「我可以」，且代表兩人組隊成功。蔡依林導師和徐明浩教練因工作原因缺席了此次的組隊活動，兩人的組隊工作分別交由李榮浩導師和艾福傑尼教練負責以決定他們的最終訓練生組合。每個表演組合的中心位置均由個別的導師擔任。
*有問必答電話亭：訓練生與通過微博選出的青春製作人進行通話。
