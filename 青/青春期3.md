《'''青春期3'''》（又名《'''青春期3：游戏青春'''》；{{lang-en|'''Pubescence 3'''}}）2012年5月15日上映于[[中国大陆|中国大陆]]的[[校园|校园]][[爱情片|爱情片]]，为[[管晓杰|管晓-{杰}-]]《青春期系列》的收尾之作。<ref>{{cite web |url= http://games.qq.com/a/20120504/000155.htm|title=《青春期3》15号上映 描绘网游人群 |accessdate=2012年5月4日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

== 簡介 ==
青春期3承接第一部青春期,主要描述程小雨和王小菲的爱情故事。本篇內容不同於第二集[[青春失樂園|青春失樂園]],本篇以較悲傷殘酷的方式銜接第一集[[青春期_(2011年电影)|青春期]]描繪程小雨和王小菲接下來的人生和愛情發展.

== 花絮 ==
2012年4月11日，电影正式开机拍摄。<ref>{{cite web |url=http://www.ha.chinanews.com.cn/lanmu/news/1844/2012-04-11/news-1844-158055.shtml |title=芭乐《青春期3》今日开机 未来发力手机视频 |accessdate=2012-04-11 |publisher=中新网 |work= |language=zh-cn }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

赵奕欢在此集中首度出演[[床戏|床戏]]。<ref>{{cite web |url=http://games.qq.com/a/20120506/000067.htm#p=12 |title=大尺度《青春期3》曝网游人群众生相 |accessdate=2012年5月5日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

谢瑾因出演此片而人气飙升，受到追捧。<ref>{{cite web |url=http://yule.sohu.com/20120515/n343242602.shtml |title=谢瑾受邀录制《天天向上》 与张扬飙泪备受关注 |accessdate=2012年5月15日 |publisher= 搜狐网|work= |language =zh-cn}}</ref>

该片与[[网络游戏|网络游戏]]《[[桃園_(遊戲)|桃園]]》建立了合作关系。<ref>{{cite web |url=http://game.21cn.com/online/news/2012/05/10/11785960.shtml |title=比基尼照放送！青春期3赵奕欢玩《桃园》 |accessdate=2012-05-10 |publisher=21cn |work= |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20120531043657/http://game.21cn.com/online/news/2012/05/10/11785960.shtml |archivedate=2012-05-31 }}</ref>

该片反映的主要社会问题是青少年尤其是学生群体的网游上瘾，通宵达旦玩游戏、逃课玩游戏、集体通宵等的学校学生问题。<ref>{{cite web |url= http://games.qq.com/a/20120503/000280.htm#p=4|title=《青春期3》：那些年 为游戏逃课的日子 |accessdate= 2012年5月3日|publisher= 腾讯网|work= |language =zh-cn}}</ref>
