'''青铜'''是纯[[銅|銅]]（紫銅）加入[[鋅|鋅]]與[[鎳|鎳]]以外的金屬所產生的[[合金|合金]]，如加入[[錫|錫]]、[[鉛|鉛]]或[[鋁|鋁]]的[[銅合金|銅合金]]，古时青铜器埋在土里后颜色因[[氧化|氧化]]而青灰，故命名为青铜。与纯铜（红铜）相比，青铜强度高且[[熔点|熔点]]低（25％的锡[[冶炼|冶炼]]青铜，熔点就会降低到800℃。纯铜（红铜）的熔点为1083℃）。青铜的[[铸造|铸造]]性好，耐磨且[[化学|化学]]性质稳定。

== 不含錫的「青铜」 ==
有許多名稱中有青銅但不含錫的銅合金，如[[铝青铜|铝青铜]]、[[铍青铜|铍青铜]]、[[锰青铜|锰青铜]]、[[硅青铜|硅青铜]]。在銅合金的分類中，[[黃銅|黃銅]]和[[白銅|白銅]]（铜镍合金）以外的都稱為青銅。

== 各類特性 ==
*錫青銅的鑄造性能﹑減摩性能和機械性能較佳，適合於製造[[齒輪|齒輪]]、[[軸承|軸承]]、[[蝸輪|蝸輪]]等。
*鉛青銅適合於製造[[發動機|發動機]]的[[軸承|軸承]]材料。
*鋁青銅強度高，耐磨性和耐蝕性較佳，常用於鑄造高載荷的[[軸套|軸套]]、[[齒輪|齒輪]]、船用[[螺旋槳|螺旋槳]]等。
*鈹青銅適合於製造煤礦和油庫的[[無火花工具|無火花工具]]。
*鈹青銅和磷青銅的導電性較佳，彈性極限高，適合於製造[[電接觸元件|電接觸元件]]和精密[[彈簧|彈簧]]。

== 反常膨脹 ==
青銅器遇冷會反常地膨脹。這一特性使得澆鑄後其紋飾或刻字之棱角外凸而美觀。
