'''青洲總站'''（[[葡萄牙文|葡萄牙文]]：'''EST. ILHA VERDE/ TERMINAL'''、{{lang-en|'''Ilha Verde Terminal'''}}）位於[[澳門|澳門]][[花地瑪堂區|花地瑪堂區]]的一個終點站，位於青洲上街。[[澳巴|澳巴]]'''[[澳巴23路線|23]]'''、'''[[澳門巴士27路線|27]]'''、'''[[新福利28B路線|28B]]'''和'''[[新福利28BX路線|29]]'''路線以該站為終點站。[[澳巴|澳巴]]'''[[新福利8路線|8]]'''、'''[[新福利8A路線|8A]]'''路線中途停靠該站。

== 沿革 ==

== 歷史 ==
2017年7月29日，[[澳門巴士8路線|澳門巴士8路線]]和[[澳門巴士8A路線|澳門巴士8A路線]]遷往「工業園街」站為總站。

== 設施 ==
*[[澳巴|澳巴]]站長室。
