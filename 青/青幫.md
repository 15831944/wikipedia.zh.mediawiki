'''青幫'''原稱「'''清幫'''」，是中國古老的[[秘密结社|秘密结社]]，[[清|清]][[雍正|雍正]]四年（1726年）[[翁岩|翁岩]]、[[錢堅|錢堅]]及[[潘清|潘清]]所創，三人崇信[[羅教|羅教]]，遠奉[[明朝|明朝]][[山東|山東]][[羅清教|羅清教]]創始人[[羅清|羅清]]為始祖，一說清幫之「清」即為紀念羅清之意，後因[[避諱|避諱]]，漸改為「青幫」，或說避免[[朝廷|朝廷]]注意，改為「'''安清幫'''」。徒眾昔多以運糧的[[漕運|漕運]]為業，故稱「'''漕幫'''」、「'''糧船幫'''」。大江南北入幫者頗眾。青幫是[[清朝|清]]初以来流行最廣、影響最深遠的民間秘密結社之一，俗說「瓶中太滿水須走，青葉紅花白蓮藕」，即是意指「反對[[滿清|滿清]]的幫會，有'''青幫'''、'''[[洪門|洪門]]'''、'''[[白蓮教|白蓮教]]'''三大主力」。

==源起==
{{unreferenced-section}}
===背景===
清[[雍正|雍正]]年間，因[[清朝|清朝]]立國之初屢屢征戰，與[[康熙|康熙]]末年百官[[貪腐|貪腐]]嚴重，待傳位[[雍正|雍正]]皇朝時國庫便已空虛，南北貧富差距日增，[[雍正|雍正]]皇帝將問題綜整入理，並召來眾國策大臣相商對應之道。[[雍正|雍正]]以農務為先，大興農務以降，為暢通糧運之道，避免旱路行走不易之弊，於是大出皇榜，由[[欽差大臣|欽差大臣]][[田文鏡|田文鏡]]招民興辦水路糧運。

===創立===
据传说，當時，[[杭州|杭州]]三位[[羅清教|羅清教]][[弟子|弟子]]揭皇榜願受此任，並[[結誼|結誼]]為異姓兄弟。這三位依年歲長幼而分，長者[[翁岩|翁岩]]，字福明，道號德慧，[[江蘇|江蘇]][[常熟|常熟]]人，原籍[[山東|山東]][[東昌|東昌]]府[[聊城|聊城]]縣鼓樓街，[[秀才|秀才]]出身，後棄文習武，在[[河南|河南]]少林寺習藝，[[雍正|雍正]]二年加入[[天地會|天地會]]；次者[[錢堅|錢堅]]，字福齋，道號德正，原籍[[山東|山東]][[東昌府|東昌府]][[聊城縣|聊城縣]]，遷居[[河南|河南]]，經商，[[雍正|雍正]]二年隨翁祖加入[[天地會|天地會]]，他与翁岩同是[[天地會|天地會]]首领[[张岳|张岳]]的部下；最年幼者[[潘清|潘清]]，字宣亭，道號德林，[[浙江|浙江]][[杭州|杭州]]武林門外啞叭橋人（有的野史將三人寫為翁麟、錢保、潘安）。

此三位清幫的祖師爺，均因受教於[[羅清教|羅清教]]下，此外，長房翁祖出自[[少林|少林]]，且[[羅祖|羅祖]]本源自[[禪門|禪門]]，為追本溯源，因此清幫同時也供奉中國[[禪宗|禪宗]]老祖[[達摩祖師|達摩]]、二祖[[慧可|慧可]]、三祖[[鑒智|鑒智]]、四祖[[道信|道信]]、五祖[[弘忍|弘忍]]、六祖[[惠能|惠能]]禪師，常常設有「禪門祖師[[神位|蓮座]]」，將各祖師名諱列上，[[傳道|傳道]]亦傳禪法，以[[宗教|宗教]]立幫。

三位清幫祖師於[[雍正|雍正]]四年揭下皇榜，創立糧運之道，水路運糧主要是以[[通州|通州]]至[[杭州|杭州]]這段航程，建設了七十二個半碼頭，設立一百二十八幫半，而其中所謂的半碼頭，指的是專做南北雜貨小買賣用的碼頭。十七世紀時，訓民必先教民，古時候的通識教育並不普及，宗教的力量及勸人為善的理念，對於約束幫眾行為，則是最好規律法條，而這半幫指的即是承傳祖訓，修行修身傳教傳道的香火船，此為蘭山半幫，幫主為潘房三十六弟子中的頂山門（眾弟子中排行中間者）弟子姚玉字清，[[杭州|杭州]]人士，道號文銓，後代稱為姚祖。

清幫雖為翁、錢、潘三位祖師所創，然草創之初，翁、錢二位祖師先後仙逝，潘祖獨撐大志，與門下弟子共同訂定家規法則，勸戒幫眾修德論道，將一幫市井船夫，治理有條有序，強調師帶徒的體制，幫中大小以[[字輩|字輩]]論之，儼然是個大家族，並設立家廟，凡入幫者，不論何姓，一旦入幫，均為「潘家子孫」，因此不僅僅是入幫會，而是入家族，且不論何字班輩，一師皆為師，一徒皆為徒，受四方長輩恩下，也使得清幫有別於其他[[秘密會社|秘密會社]]，師徒兄弟間感情特別親切。

[[乾隆|乾隆]]三十一年，潘祖於運糧時在鳳林閘下仙逝，由潘祖開山門弟子[[王降|王降]]，字'''相陽'''，[[浙江|浙江]][[杭州|杭州]]西門外人，及關山門弟子[[蕭玉德|蕭玉德]]，字'''明新'''，[[蘇州府|蘇州府]][[吳門|吳門]]外人，扶柩回[[杭州|杭州]]武林門外，是為[[王祖|王祖]]及[[蕭祖|蕭祖]]，因此後，清幫對於各師父領下的開山門、關山門弟子均特別敬重。此外，潘祖仙逝之時，有第五代文字輩弟子[[王培玉|王培玉]]，字藍田，道號文功，直隸通州壩人，當時僅進過小香，清幫稱為一腳門內一腳門外，由眾師兄幫他上大香開法後（清門規矩，師父若「過舫」，即過世，其門下弟子若未上大香或開法者，可由頂山門弟子主香代師執行，唯不可代師再收徒），其守墓終日，純誠哀慟不已，飲食不思，數日後隨潘祖仙逝，清幫為念其護主忠誠，又因其為糟船舵工，在職上他繫全船安危於一身，故封其為護法小爺，於香堂門傍供奉，永做記念。清門中為記念這些祖師，則稱為三老四少，三老分為前後三老，前三老指的是金、羅、陸祖，後三老為翁、錢、潘祖。四少即為：朱、劉、黃、石，四個講道庵室所出學生，分別為翁祖講道的朱寺庵、錢祖劉寺庵、潘祖黃寺庵、護法小爺石寺庵。四庵代表為：翁祖弟子黃泉，錢祖弟子石士寶，潘祖弟子蕭玉德，最特別的則為石寺庵，誠上所述，護法小爺開法不久便已仙逝，哪來的弟子？所以石寺庵弟子其實指的是領幫行運，護祐全幫的總領幫主，或是護祐清幫有功之人，意為培養忠義護主之風，在潘祖之後，為文字班宿慶祥領幫，故石室庵弟子以宿慶祥做為代表。

繼潘祖之後，眾推[[宿慶祥|宿慶祥]]領幫行運，幫隔數代後，乃至清[[咸豐|咸豐]]四年，[[太平天國|太平天國]]之亂，皇糧遭毁，無法承運糧務，至此後，清幫主要經濟命脈斷絕，後至清光緒十二年，一百二十八幫半，僅餘六大幫頭，而最後這六幫，接續傳承家運，至光緒二十七年民運改為官運，不再行運江河，昔日千船萬旗大江飄的景象，從此之後正式走出歷史，指為糧船不開，雀桿不點頭，因此後期的清幫弟子，均在陸上行走四方，又稱為旱碼頭入會。

有人認為，清幫是[[洪門|洪門]]的一個支派，其實不然，但確有傳說翁、錢二位祖師均乃[[洪門|洪門]]（[[天地會|天地會]]）會員，且創幫之始，很多成員出身[[洪門|洪門]]也甚有可能。

學者認為，漕运是供应京城粮米的命脉，清朝政府对漕运异常重视，不可能将这一任务外包给没有官方背景的民间人士，更不可能允许有反清復明思想的天地会人士成立会党组织，以上揭皇榜的说法极有可能是后来的清幫人士穿鑿附会。

== 青幫與洪門 ==
{{unreferenced-section}}
雖然青幫與[[洪門|洪門]]有相當之淵源，但[[洪門|洪門]]講究[[結拜|結拜]]兄弟[[義氣|義氣]]，信奉[[道教|道教]]與[[民間信仰|民間信仰]]，指天地立誓，祭[[關帝君|關帝君]]，崇奉已逝的先烈如[[九皇大帝|九皇]]、[[少林五祖|五祖]]、[[陳近南|仙鶴]]、[[萬雲龍|雲龍]]等。青幫信仰[[羅教|羅教]]，[[羅教|羅教]]源於[[禪|禪]]學，故實行[[禪宗|禪宗]]制度，所以與[[洪門|洪門]]中互稱[[兄弟|兄弟]]不同，青幫組織禮拜[[釋迦如來|釋迦如來]]、[[達摩祖師|達摩祖師]]、[[慧能|六祖大師]]、[[羅祖|羅祖]]等，深具[[佛教|佛教]]色彩，拜師入幫，會眾以師徒相稱，崇尚「師徒如父子」。青幫組織比[[洪門|洪門]]更嚴密，規矩更複雜，於是也更加秘密。因此有言道：「青幫一條線，洪門一大片」。

傾向於行會性質的青幫提倡「隱身市井」，並未如[[洪門|洪門]]般強調[[反清復明|反清復明]]，由於青幫並未積極從事反清活動，所以[[洪門|洪門]]一度將其當作叛徒，嚴禁[[洪門|洪門]]會員轉投青幫，稱「由青轉洪，掛綵披紅；由洪轉青，剝皮抽筋。」。但為了避免衝突，兩派仍多稱「青洪一家」，所謂「紅花綠葉白蓮藕（指[[洪門|洪門]]、青幫、[[白蓮教|白蓮教]]），三教原來是一家」。尤其清末[[革命黨|革命黨]]思想勃興，不少青幫弟子也開始反清，兩派逐漸減少敵對。

依中華民國憲法第 14條：「人民有集會及結社之自由」，其實目前是民主社會，有法律規範，各地安青協會都已登記有案公開合法化，所有成員都須依法公平對待！時代在進步，安青以開放胸懷結合志同道合，願意奉獻一己，成就安青大業的仁人志士，厥為首要目標，而「剝皮抽筋」係清朝時代因時因地制宜之說，不符現代法律規範，在現今時代稍有不慎這種話會被解讀為黑幫暴力組織，據此，團結一心、發揚中華文化、家門興旺、義氣千秋是首要安清精神，有識之士宜為體認。

== 行政組織 ==
{{unreferenced-section}}
青幫主要以四庵六部做為行政管理中樞。

===四庵===
所謂的庵，是依祖師而立的講學堂與中央行政部門，相當於現在的府院，分別為朱寺庵、劉寺庵、黃寺庵、石室庵，這四個庵室，共通的部份便是[[羅教|羅教]]教義傳授，其餘的則依祖師專長不同，而有所不同，朱寺庵的主講師為翁祖，講授的內容為禪學（[[臨濟宗|臨濟宗]]派），以及武學（[[少林|少林]][[武術|武術]]），此外下司轄兵部與刑部。劉寺庵主講師為錢祖，講授的則為經商交易之道與財務管理，並為工部之直轄主管機關。黃寺庵主講師為潘祖，傳授行運管理學論，教授古聖賢之道，且直接司轄吏部與戶部。石寺庵則為小爺仙逝後創立，主講者為宿領幫，講授幫規，授予新進人員義氣千秋，及江湖禮節，可以說是職前訓練所，另為禮部之直轄主管機關。

===六部===
六部指的是[[吏部|吏部]]、[[禮部|禮部]]、[[戶部|戶部]]、[[工部|工部]]、[[兵部|兵部]]、[[刑部|刑部]]。這六部最初的主管，現在已不可考。

====吏部====
吏部專門編纂幫中大小事務，編寫清幫史書，並將史書置於杭州家廟蔵經閣樓上，此外除了史書的工作之外，亦負責抄經管理羅祖書籍與家譜，然不幸的是[[咸豐|咸豐]]四年[[太平天國|太平天國]]之亂，[[杭州|杭州]]家廟遭[[太平軍|太平軍]]焚毀，以至於文件史料無存。

====户部====
戶部則負責管理人員，是青幫相當重要的人事主管部門，舉凡拜帖進家，人員晉升，各方師父，均需投書予戶部，由戶部詳列人員清冊之中，最後年終綜整後交予吏部，謄入家譜。

====礼部====
禮部則主管規矩儀注之訂定與考核人員禮儀，並於每次開香堂時，主持規劃與邀請各方老大到場，同時亦負責審核人員進家之三幫九代是否合乎規矩禮法，並於香堂設立時擔任執堂師一職。

====工部====
工部則責於各項幫中工程，如造船，維修，後勤補給，香堂搭設等工作，是青幫效率挺高的工兵小組，所有各幫維修人員，均為工部管理，並接受一般民眾委任修船及其它建設工作。此外工部亦為清幫的基本會計部門，全幫所得計算與管理均為工部之責，清算後呈報劉寺庵。

====兵部====
兵部則為遇外敵時，統籌規劃作戰之部門；青幫弟子在外，難免與其他會社起些衝突，當衝突無法解決時，會先派出談判小組，這各幫的談判小組，歸兵部管理，當事情仍無法解決有可能發生戰鬥時，談判小組會將問題回報與兵部，兵部則開始調動可用人力，實施戰鬥準備。

====刑部====
刑部則為清幫的執法單位，當門下弟子犯有幫規時，各方師父會依情節輕重，回報予刑部，最後由刑部設立刑部香堂，對不肖弟子實行懲處。另外，不論開立何種香堂，門外與門內的巡堂師（內外刑堂），則由刑部派立。

*第一代「清」字輩指的是[[金純|金純]]，號碧峰，[[明朝|明朝]][[南京|南京]][[麒麟街道_(南京市)|麒麟門]]外金家村人氏，為[[永樂帝|永樂帝]][[大臣|大臣]]，[[永樂遷都|永樂遷都]]時營造[[帝都|帝都]]有功，後封為[[伯爵|伯爵]]，好[[飲酒|飲酒]]，政治上寬仁[[無為|無為]]，在[[刑部|刑部]]任內嚴禁[[酷刑|酷刑]]。金純據說頗喜[[佛法|佛法]]，找出了[[少林寺|少林寺]][[達摩祖師|達摩祖師]]定下的前後四十八[[字輩|字輩]]作為派系，於是他的法名，便叫做「清源」，據說羅清慕其為人，並推崇他為第一代祖師。
*第二代「靜」字輩有[[羅清|羅清]]，[[山東|山東]][[即墨|即墨]]人，[[北直|北直]][[密雲|密雲]]衛[[軍人|軍人]]，[[羅教|羅教]]教主，法名「靜清」，號稱「羅祖」，是青幫與羅教實際上的祖師。陳瀛，字易水，道號靜海，[[廣西|廣西]][[貴縣|貴縣]]人，羅祖的同修。
*第三代「道」字輩有陸飛，字羽飛，道號為道行，[[江蘇|江蘇]][[丹徒|丹徒]]人，曾任江西總兵，[[甲申之變|甲申之變]]後隱居[[茅山|茅山]]、[[廣東|廣東]]娥素山。趙大官，字悟本，道號為道元，[[金陵|金陵]]霞邑人，[[明朝|明朝]]末進士，[[崇禎|崇禎]]年曾任總兵之職。
*第四代翁岩、錢堅、潘清三人乃「德」字輩，為[[避諱|避諱]]祖師，前四字「清、靜、道、德」清幫後代，一般不再使用於[[譜名|譜名]]。

== 輩分稱呼 ==
{{unreferenced-section}}
===長輩的稱呼===
不論何幫，大一輩者，本命師與眾已開法之師父，均以某師父稱之；未開法大一輩之長者，則不論男女，均稱為師叔；大二輩者不論為哪一幫頭及是否開法，均統稱為某師爺；大三輩者則不論何幫及是否開法，均稱為師太；大四輩則不論何幫及是否開法，統稱為太師太；大五輩本幫幾乎沒有，若真有那便稱為祖節。

===平輩的稱呼===
同參一師者稱為親同參，同一師爺之分系同輩，則稱為同參，引進或傳道幫頭之同輩弟子則稱為同山，其他非三幫九代幫頭之同輩，稱為平香。

===晚輩的稱呼===
某師父的兒子，一律統稱為小老大；徒弟或是同輩的次子或兒女，則稱為少君，弟佬的兒女或徒弟，則稱為法孫；法孫的徒弟或兒女，則稱為少法孫；若還有以下，則稱為法將。

== 盤道認親 ==
{{unreferenced-section}}
青幫人士來自[[五湖|五湖]][[四海|四海]]，要認清是否是自家兄弟，須經由盤道條口確認。這與兵士站衛兵喊討口令辭的道理一樣。會有這樣的規矩，主要是為避免讓外人得知太多幫門中的祕密。清門於[[雍正|雍正]]四年，由三祖揭皇榜並受師命立幫，及潘祖執法行運五十一載後傳世至今，這期間雖受命於朝廷，然實際是為實現仗義江湖濟弱扶貧為目的。而青幫祖訓要記取根源不忘本，這從清幫三教合一，尊前提後的作法可以明瞭。此外，領幫祖師們，本就為[[天地會|天地會]]成員，因此每每設香堂，會先要求弟子們整襟，不忘前朝，溯本追源。

這樣做法難免為有心人士利用，若向公家進讒言，勢必對於幫會命脈造成莫大傷害。因此青幫要弟子們謹守祕密，並嚴守三談三不談原則（同道能談、香堂能談、告幫能談；茶館不談、酒肆不談、澡堂不談）。然而青幫自祖師創幫，由[[通州|通州]]至[[杭州|杭州]]這段運河，共立幫頭一百二十八幫半，七十二個半碼頭，糧船亦有九千九百九十九隻半，幫眾數以萬計，要認清是否為自家人實屬不易，要確認是否真為自家人，勢必得有一套明確繁雜的辨識之方，這個方法便稱為盤道條口。經盤道後可確認身份，至於盤道條口，均需師父親傳，不可外洩，例為幫中機密。

除了能確實嚴守幫中祕密不外洩於人，在亂世當道局勢不穩時，亦可做為[[情報|情報]]交換前的確認工作。在近代史中，第二十三代的[[杜月笙|杜月笙]]運用幫中力量，為[[國民政府|國民政府]]於[[抗日|抗日]]期間，做了不少的[[情報|情報]]貢獻。幫中成員謹遵三露三不露（遇急、遇難、遇盤查露，外人、熟人、親人不露）原則。所謂「三分安青七分交情」，只三分安青，便已足夠讓青幫弟兄渡過難關，這是青幫義氣相照共持共扶的原則。

== 近代 ==
青幫起於[[漕運|漕運]]，因此多流傳于[[江南|江南]]一帶。後因海運興起，漕運沒落，於是大批青幫弟子進入[[上海|上海]]，從事各種行業。由於上海的特殊性，租界和[[革命黨|革命黨]]經常需要借助青幫的力量。[[抗日戰爭|抗日戰爭]]時代，[[日軍|日軍]]佔領上海時期，[[戴笠|戴笠]]也需要利用青幫做[[情報|情報]]工作。因此青幫的勢力也愈加強大，很多[[革命黨|革命黨]]和[[洪門|洪門]]會員也投入青幫。上海名人[[黃金榮|黃金榮]]、[[杜月笙|杜月笙]]、[[張嘯林|張嘯林]]，[[同盟會|同盟會]]的[[陳其美|陳其美]]、[[蔣中正|蔣中正]]，[[中共|中共]]的[[汪壽華|汪壽華]]皆是青幫人士。蒋介石也曾向黃金榮拜师。

==現狀==
目前青幫主要分佈在[[台灣|台灣]]，並成立有合法[[社團|社團]]「[[中華安清會|中華安清會]]」，不少軍界、警界及演藝界人士屬於青幫弟子。在美國部份城市，青幫也有一定影響，惟有另類看法係當年杜月笙本於所為以行刑模式殺了六名共產黨員，而後受迫遷徙至香港，香港三合會模式亦見其與此有所關連。

目前已知清門全國性社會團有：中華正義社、中華安清會、中華安和清平協會（現改名中華安清協會），以及中華安清公益事業協會， 中華安清巖社，台灣大中華家庭關懷協會等，至於地方性社團有，台北市安清協會，桃園一心安親會，台中安清協會，台南安清協會，杭三文化協會，台灣安清協會，高雄安清協會，屏東安清協會等；因彼此皆系出同門實有密切聯繫與相互支應更在家人般的親情互為支撐下有所作為。<ref>以上資料，出自於民族精神道義結社清門（青幫）官方網站www.allfamily168.org</ref>

== 青幫[[野史|野史]] ==
# 青幫本稱“[[安慶|安慶]]幫”，查明太祖[[朱元璋|朱元璋]]龍興之地，原為[[安徽|安徽]][[鳳陽|鳳陽]][[朱家崗|朱家崗]]，崗內有朱安社，朱慶社兩社，青幫陸祖取以名幫。清[[乾隆下江南|乾隆下江南]]，混入漕運庵堂，得入安慶幫。取得三幫九代後，得悉安慶幫之真意，原為[[明朝|明朝]]反清復明。喻令改名為“安清幫”，否則以反動治罪論。彼時領幫前人不得不虛以委蛇，但又恐傳之後世，失卻本旨，表面忠於滿清，但一再告誡幫規要尊儀注莫忘，隱含[[反清復明|反清復明]]民族革命之意識，表面視之毫無破綻。[[乾隆皇|乾隆皇]]並欽賜“蟠龍棍”，以為家法，上頭刻有“打死無罪”四字，故青幫家法有二，一為[[警策棒|警策棒]]（俗稱佛板，源於禪宗[[體罰|體罰]]弟子的用具），一為蟠龍棍，直至今日青幫開香堂必有此二家法供於堂上。（節錄自《杭三幫·根派宗代》）
# [[抗日|抗日]]時期國民政府軍事委員長[[蔣中正|蔣中正]]原為[[上海|上海]]青幫老大[[黃金榮|黃金榮]]門下，屬二十三代悟字輩，自其升任[[國民政府軍事委员会|國民政府軍事委员会]]委員長起，為避免閒言閒語，並尊重其為中央軍事統領，遂退回門生請帖。其後也借重清幫[[杜月笙|杜月笙]]之力對日抗戰，並在[[杜月笙|杜月笙]]之墓題有“義節聿昭”四字可資供證。
# [[香港|香港]]現任港區[[全國人大代表|全國人大代表]]及常務委員會成員[[范徐麗泰|范徐麗泰]]的父親與青幫有關，其父[[徐大統|徐大統]]是[[杜月笙|杜月笙]]的左右手。<ref>[http://hk.next.nextmedia.com/template/next/art_main.php?iss_id=734&sec_id=1000853&art_id=3952270 揭范徐麗泰青幫父親身世] - 壹週刊 - 734期 - 時事， 2004年04月1日</ref>
# 與[[尹清楓命案|尹清楓命案]]有關的[[單亦誠|單亦誠]]在青幫中輩分極高，認識他的多稱他為「單將軍」或「單老爺子」，雖然他喜歡與影藝界來往，也收過很多乾女兒，但真正了解他的人並不多。<ref>單亦誠在青幫屬「通」字輩，在早年的軍中幫會中，地位崇高，[[單亦誠|單亦誠]]是以第二十二代「通」字輩弟子領幫，已故[[周宗泉|周宗泉]]少將及在尹案中被調查的前海軍上校祝本立，屬「悟」字輩，師承第二十二代「通」字輩的受業門生，而單亦誠為「通」字輩的老爺子，可見他在軍中的地位。 更新日期:[http://news.chinatimes.com/focus/50108988/112011062100086.html 2011/06/21] {{webarchive|url=https://web.archive.org/web/20110712132007/http://news.chinatimes.com/focus/50108988/112011062100086.html |date=2011-07-12 }} 03:06 呂昭隆／特稿 中國時報</ref>
