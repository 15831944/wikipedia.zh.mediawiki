'''青森县'''（{{jpn|j=青森県|hg=あおもりけん|hgt=あをもりけん|rm=Aomori ken|f=y}}）是[[日本|日本]][[本州|本州]]最北部的一个縣，縣廳位於[[青森市|青森市]]。在[[令制國|令制國]]的時代屬於[[陸奧國|陸奧國]]。青森縣面積約有9645平方公里，約佔日本國土面積的約2.5%。人口約有128萬，約佔日本總人口的1%。青森縣內轄有10市、22町、8村，共有40個市町村<ref>{{cite web |url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/enkaku/hensen/post-107.html　 |script-title=ja:市町村合併（昭和の大合併及び平成の大合併） |accessdate=2017-05-11 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。青森縣歷史悠久，發現了{{link-ja|三內丸山遺跡|三内丸山遺跡}}等大型[[繩文時代|繩文時代]]的遺跡。青森縣擁有[[白神山地|白神山地]]，[[十和田湖|十和田湖]]、[[八甲田山|八甲田山]]、[[岩木山|岩木山]]、{{link-ja|佛浦|仏ヶ浦}}等自然景觀。[[奧羽山脈|奧羽山脈]]縱貫青森縣中央，山脈西側的[[津輕地方|津輕地方]]和東側的[[南部地方|南部地方]]各有不同的歷史、氣候、文化、風土特徵。

青森縣是日本前列的農業縣，糧食自給率達123%<ref>{{cite web |url=http://www.maff.go.jp/tohoku/syokuryou/zikyuritu/　 |script-title=ja:東北地域食料自給率 |accessdate=2017-05-11 |publisher={{lang|ja|東北農政局}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，[[蘋果|蘋果]]、[[大蒜|大蒜]]、[[牛蒡|牛蒡]]的產量高居日本第一<ref>{{cite web |url=http://www.pref.aomori.lg.jp/kids/cld-sangyou.html　 |script-title=ja:県の産業 |accessdate=2017-05-11 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。八戶港是日本前列的漁港，[[管魷目|魷魚]]捕撈量位居日本第一<ref>{{cite web |url=http://www.city.hachinohe.aomori.jp/index.cfm/14,185,113,html　|script-title=ja:水産業の概要 |accessdate=2017-05-12 |publisher={{lang|ja|八戸市}} |language=ja}}</ref>。位於青森縣東南部的八戶臨海工業地帶是該縣主要的工業區，有火力發電廠、造紙廠、電機工廠、造船廠等工廠。下北半島的陸奧小川原湖地區有眾多核能關聯設施。此外青森縣內還有日本最大規模的風力發電廠。

青森縣有[[青森機場|青森機場]]和[[三澤機場|三澤機場]]兩個機場。[[東北新幹線|東北新幹線]]連接了青森和[[東京|東京]]兩地，[[北海道新幹線|北海道新幹線]]則通過[[青函隧道|青函隧道]]連接青森和函館。除新幹線外還有[[青森鐵道線|青森鐵道線]]、[[弘南鐵道|弘南鐵道]]各線路以及[[東日本旅客鐵道|JR東日本]]的[[奧羽本線|奧羽本線]]等鐵道線。縣內的主要公路有[[東北縱貫自動車道|東北縱貫自動車道]]、[[國道4號_(日本)|國道4號]]和[[國道7號_(日本)|國道7號]]。青森主要的民俗慶典活動則有[[青森睡魔祭|青森睡魔祭]]、{{link-ja|弘前睡魔祭|弘前ねぷたまつり}}、{{link-ja|五所川原立佞武多|五所川原立佞武多}}、{{link-ja|八戶三社大祭|八戸三社大祭}}等。

== 歷史 ==
=== 史前時期至平安時代 ===
[[File:140913_Sannai-Maruyama_site_Aomori_Japan01bs6bs6.jpg|left]]
青森縣內的{{link-ja|大平山元I遺跡|大平山元I遺跡}}發現了[[舊石器時代|舊石器時代]]的土器，顯示青森至少在距今12,000年之前就已經有人類活動<ref>{{cite web |url=http://jomon-japan.jp/jomon-sites/odai-yamamoto/　 |script-title=ja:大平山元遺跡 |accessdate=2017-05-12 |publisher={{lang|ja|北海道・北東北の縄文遺跡群}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。青森縣內有眾多[[繩文時代|繩文時代]]晚期的大型遺跡。位於[[青森市|青森市]]，距今約5,500至4,000年前的{{link-ja|三內丸山遺跡|三内丸山遺跡}}是日本最大規模的繩文時代遺跡<ref>{{cite web |url=http://sannaimaruyama.pref.aomori.jp/about/　 |script-title=ja:三内丸山遺跡とは？ |accessdate=2017-05-12 |publisher={{lang|ja|三内丸山遺跡}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[津輕市|津輕市]]的{{link-ja|龜岡石器時代遺跡|亀ヶ岡石器時代遺跡}}則以[[遮光器土偶|遮光器土偶]]著稱<ref>{{cite web |url=http://jomon-japan.jp/jomon-sites/kamegaoka/　 |script-title=ja:亀ヶ岡石器時代遺跡 |accessdate=2017-05-12 |publisher={{lang|ja|北海道・北東北の縄文遺跡群}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[八戶市|八戶市]]的{{link-ja|是川遺跡|是川遺跡}}歷史幅度橫跨繩文時代前期至晚期，發現了漆器等文物<ref>{{cite web |url=http://www.city.hachinohe.aomori.jp/index.cfm/12,69657,129,63,html　|script-title=ja:是川遺跡について  |accessdate=2017-05-12 |publisher={{lang|ja|八戸市}} |language=ja}}</ref>。

進入[[彌生時代|彌生時代]]之後，日本開始出現大規模稻作文化。[[弘前市|弘前市]]的{{link-ja|砂澤遺跡|砂沢遺跡}}是日本最北的[[彌生時代|彌生時代]]前期水田遺跡<ref>{{cite web |url=http://www.city.hirosaki.aomori.jp/gaiyou/bunkazai/kuni/kuni25.html　|script-title=ja:砂沢遺跡出土品  |accessdate=2017-05-12 |publisher={{lang|ja|弘前市}} |language=ja}}</ref>。然而青森縣在[[古墳時代|古墳時代]]的歷史幾乎空白，未發現任何古墳時代前期的{{link-ja|前方後圓墳|前方後円墳}}遺跡<ref name="Name">{{cite book |author={{lang|ja|竹内理三}}|script-title=ja:《角川日本地名大辞典 2 青森県》 |year=1985 |publisher={{lang|ja|角川書店}} |location=東京都 |isbn=4-04-001020-5|language=ja}}</ref>{{rp|26}}。7世紀時[[律令制|律令制]]國家開始出現後，青森縣內才發現{{link-ja|阿光坊古墳群|阿光坊古墳群}}<ref>{{cite web |url=http://bunka.nii.ac.jp/heritages/detail/212170　|script-title=ja:阿光坊古墳群  |accessdate=2017-05-12 |publisher={{lang|ja|文化遺産オンライン}} |language=ja}}</ref>等遺跡。

《[[日本書紀|日本書紀]]》記載655年時，[[齊明天皇|齊明天皇]]在難波宮授予津輕[[蝦夷|蝦夷]]冠位，這是「津輕」地名首次出現在文獻<ref name="Name"></ref>{{rp|26}}。包括青森縣在內的[[北東北|北東北]]地區在古墳時代末期之前並不是[[大和朝廷|大和朝廷]]的支配地區。但在7世紀中期之後，隨著齊明天皇遣[[阿倍比羅夫|阿倍比羅夫]]攻打北東北地區，青森縣和大和朝廷之間的關係也日益密切。9世紀之後，日本中央政府又先後派[[坂上田村麻呂|坂上田村麻呂]]、[[文室綿麻呂|文室綿麻呂]]等人進攻北東北地區，青森亦逐漸置於律令制國家的體制內<ref name="Name"></ref>{{rp|27}}。

=== 鎌倉時代至江戶時代 ===
[[File:Hirosaki-castle_Aomori_JAPAN.jpg|left]]
12世紀後期{{link-ja|奧州合戰|奥州合戦}}之後，津輕已進入[[陸奧國|陸奧國]]管轄範圍之內<ref name="Name"></ref>{{rp|28}}。{{link-ja|安東氏|安東氏|安藤氏}}是[[鎌倉時代|鎌倉時代]]時期青森的統治者。安藤氏注重發展海上貿易和水軍。{{link-ja|十三湊|十三湊}}是當時安藤氏的主要貿易港口，發現了日本其他地區和中國的文物<ref name="Name"></ref>{{rp|29}}。然而在1322年（[[元亨_(日本)|元亨]]2年），安藤氏發生內訌，勢力從此開始減弱<ref name="Name"></ref>{{rp|30}}。[[南北朝_(日本)|南北朝時代]]之後，青森出現安藤氏和據有岩手縣南部至青森縣東部內陸地區的[[南部氏|南部氏]]對峙的局面。15世紀中期，安藤氏被南部氏攻滅，戰爭以南部氏的勝利告終<ref name="Name"></ref>{{rp|33}}。

南部氏在15世紀繼續擴大勢力。但在16世紀後期，南部氏發生分裂。南部氏一族的大浦為信（後改名[[津輕為信|津輕為信]]）佔領了津輕地方並興建[[弘前城|弘前城]]<ref>{{cite web|url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/enkaku/chusei/post-115.html　|script-title=ja:為信の津軽統一|accessdate=2017-05-12|publisher={{lang|ja|青森県}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1590年（[[天正_(日本)|天正]]18年），豐臣秀吉承認津輕為信對津輕地區的支配權。此後青森的大部分地區分屬南部氏和津輕氏<ref name="History">{{cite web|url=http://www.pref.aomori.lg.jp/k-kensei/history.html　|script-title=ja:年表で見る青森県の歴史|accessdate=2017-05-12|publisher={{lang|ja|青森県}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[江戶時代|江戶時代]]的青森仍然大部分地區分屬南部氏的[[盛岡藩|盛岡藩]]和津輕氏的[[弘前藩|弘前藩]]。但東南部的部分地區由和盛岡藩同屬南部氏的八戶藩支配，中部是弘前藩的支藩黑石藩的領地<ref name="History"></ref>。南部氏和津輕氏的領地在气候上有很大差異。由於冷害的侵襲，青森東南部地區在江戶時代曾經多次發生飢荒<ref name="Name"></ref>{{rp|49}}。困窘的財政狀況使得盛岡藩、八戶藩、弘前藩在18及19世紀進行過大規模改革，並取得了一定的成效<ref name="Name"></ref>{{rp|51}}。同時飢荒還促使青森的人口開始向[[北海道|北海道]]的[[道南|道南]]地區移民<ref name="Name"></ref>{{rp|49}}。[[幕末|幕末]][[戊辰戰爭|戊辰戰爭]]時期，弘前藩和盛岡藩最初都加入了[[奥羽越列藩同盟|奥羽越列藩同盟]]，但弘前藩在中途叛逃加入新政府軍陣營<ref>{{cite web|url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/enkaku/tanjo/post-111.html　|script-title=ja:東北戊辰戦争と斗南藩成立|accessdate=2017-05-12|publisher={{lang|ja|青森県}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。戰後盛岡藩的領地被大幅削減，幕府在被沒收的盛岡藩領地中新設斗南藩和七戶藩。廢藩置縣前夕時，青森縣內有弘前、黑石、八戶、七戶、斗南五藩<ref name="Name"></ref>{{rp|55}}。

=== 青森縣的成立 ===
[[File:Toyosu_hachinohe.JPG|right]]
1871年，日本實施[[廢藩置縣|廢藩置縣]]。在經過統合之後，青森縣內五藩和北海道[[渡島半島|渡島半島]]地區的館縣合併，成立弘前縣，縣廳位於弘前。然而由於弘前縣面積遼闊，縣廳弘前偏居一隅，不便管理。因此弘前縣在誕生19天之後縣廳就遷移到[[青森市|青森]]並改名青森縣<ref name="History"></ref>。在1872年舊館縣地區移交[[開拓使|開拓使]]和1876年二戶郡劃給[[岩手縣|岩手縣]]之後，青森縣確立了現在的行政區域<ref>{{cite web|url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/enkaku/tanjo/post-110.html　|script-title=ja:青森県の誕生|accessdate=2017-05-12|publisher={{lang|ja|青森県}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。19至20世紀之交，[[東北本線|東北本線]]和[[青函連絡船|青函連絡船]]的開通使得青森進入鐵路時代，實現交通近代化<ref name="History"></ref>。但同一時期的青森縣也發生了199人遇難的[[八甲田雪中行軍遭難事件|八甲田雪中行軍遭難事件]]慘劇<ref name="History"></ref>。自明治時期開始，[[蘋果|蘋果]]種植和軍馬生產開始成為青森縣的重要產業<ref name="Name"></ref>{{rp|57}}。進入[[大正|大正]]時期之後，八戶完成了八戶漁港，弘前市和青森市的經濟也有明顯發展。青森縣的城市地區經濟取得了明顯增長<ref name="Name"></ref>{{rp|58}}。但另一方面，農村地區則因冷害而多次出現飢荒，迫使眾多農民被迫移民北海道、[[樺太廳|樺太]]等地<ref name="Name"></ref>{{rp|59}}。[[第二次世界大戰|第二次世界大戰]]爆發之後，青森縣亦進入戰時體制。蘋果被迫減產以生產米麥，工業也以軍需物資為生產重點<ref name="Name"></ref>{{rp|59}}。1945年7月，美軍對青森市進行{{link-ja|青森大空襲|青森大空襲|大規模空襲}}，超過1,000人在空襲中遇難<ref>{{cite web |url=https://www.city.aomori.aomori.jp/somu/shiseijouhou/aomorishi-matidukuri/heiwa/aomorikusyu.html |script-title=ja:青森空襲（昭和20年7月28日） |accessdate=2017-05-12 |publisher={{lang|ja|青森市}} |language=ja}}</ref>。

戰後青森縣一度被盟軍佔領，現在美軍在青森縣內仍有[[三澤飛行場|三澤基地]]。青森縣在戰後開始進行土地改革等大規模改革，並且在1949年成立了縣內首座大學[[弘前大學|弘前大學]]<ref name="Name"></ref>{{rp|59}}。青森縣內的市町村在昭和初期進行了大規模合併，市町村數從1953年的3市33町127村減少到1955年的6市30町38村<ref name="Name"></ref>{{rp|59}}。1950年代時，青森縣曾以重工業化為發展目標，提出了核能發電廠、東北自動車道、東北新幹線、[[青函隧道|青函隧道]]等大型工程計劃。但這並未改變[[日本戰後經濟奇蹟|經濟高速成長]]時期勞動力提供縣的角色<ref name="Name"></ref>{{rp|60}}。在[[第一次石油危機|第一次石油危機]]之後，隨著日本經濟增長速度減速，青森縣的開發計劃也被迫大幅變更<ref name="Name"></ref>{{rp|60}}。1980年代，青森縣先後完成了東北縱貫自動車道、新[[青森機場|青森機場]]、[[青函隧道|青函隧道]]等大型工程。但青森縣的人口也自這一時期開始減少<ref name="History"></ref>。2002年，隨著[[東北新幹線|東北新幹線]]延伸至[[八戶站|八戶站]]，青森縣迎來了新幹線時代。此後新幹線又在2010年進一步延伸至[[新青森站|新青森站]]<ref>{{cite web |url=http://www.pref.aomori.lg.jp/soshiki/kikaku/kikaku/files/h21yokuwakaru-2-4.pdf　 |script-title=ja:東北新幹線全線開業への道のり |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>。青森縣雖然在戰後取得顯著發展，但平均壽命、人均所得等指標均處於後段，顯示青森仍然是日本國內的後進地區<ref name="Chishi">{{cite book |author={{lang|ja|田村俊和·石井英也・日野正輝}}|script-title=ja:《日本の地誌4　東北》 |year=2008 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=4-254-16764-4|language=ja}}</ref>{{rp|172}}。

== 地理 ==
[[File:Satellite_image_of_Aomori,_Japan_in_May_2001.png|left]]
青森縣南側和[[岩手縣|岩手縣]]、[[秋田縣|秋田縣]]接壤，北側隔[[津輕海峽|津輕海峽]]和[[北海道|北海道]]相望，東側是[[太平洋|太平洋]]，西側是[[日本海|日本海]]<ref>{{cite web |url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/ichi/post-88.html　 |script-title=ja:青森県に接している県 |accessdate=2017-05-11 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。青森縣主要由東側的[[下北半島|下北半島]]和西側的[[津輕半島|津輕半島]]兩個半島構成，兩個半島中間是[[陸奧灣|陸奧灣]]及[[夏泊半島|夏泊半島]]。這一地形也構成了[[青森縣旗|青森縣旗]]的圖案。青森縣的面積有9645.56平方公里，在日本的都道府縣中排名第8位<ref>{{cite web |url=http://www.gsi.go.jp/KOKUJYOHO/MENCHO/201610/area_todofuken.pdf　 |script-title=ja:都道府県別面積 |accessdate=2017-05-12 |publisher={{lang|ja|国土地理院}} |language=ja }}{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>。青森縣內標高100米以下的地區佔近半成，而標高1000米以上的地區只有1%，地形較為平坦<ref>{{cite web |url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/chikei/post-99.html　 |script-title=ja:地形の概況 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。青森縣的地形以中部的[[奧羽山脈|奧羽山脈]]為界，東西截然不同。西部主要是由{{link-ja|岩木川|岩木川}}形成的沖積低地，東部則是由火山灰覆蓋的洪積台地<ref name="Name"></ref>{{rp|19}}。

[[File:OiraseValley2.JPG|right]]
青森縣西北部[[津輕半島|津輕半島]]的東部地區是山地地形，西部則是平原<ref>{{cite web |url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/chikei/post-97.html　 |script-title=ja:主な山脈、山地 |accessdate=2017-05-12 |url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/chikei/post-95.html　 |script-title=ja:主な平野、台地 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，平原的西南側有屬於鳥海火山帶的青森縣最高峰[[岩木山|岩木山]]，標高1625米<ref>{{cite web |url=http://www.aptinet.jp/Detail_display_00000018.html　 |script-title=ja:岩木山 |accessdate=2017-05-12 |publisher={{lang|ja|アプティネット}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。岩木山南側的白神山地則構成了青森縣和秋田縣的縣界。青森縣中部青森中部的十和田、八甲田火山屬於那須火山帶，是青森縣的東西分水嶺。十和田湖就是十和田火山的[[破火山口|破火山口]]，最深處達326.8米<ref>{{cite web |url=http://www.aptinet.jp/Detail_display_00000335.html　 |script-title=ja:十和田湖 |accessdate=2017-05-12 |publisher={{lang|ja|アプティネット}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。十和田、八甲田火山的北端是青森平原。青森縣東部地區則是以台地地形為主，沿海地區有青森縣面積第一大湖[[小川原湖|小川原湖]]等海跡湖<ref>{{cite web |url=http://www.town.tohoku.lg.jp/kankou/course/kankou_course03.html　 |script-title=ja:小川原湖 |accessdate=2017-05-12 |publisher={{lang|ja|東北町}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。青森縣西部的最大水系是{{link-ja|岩木川|岩木川}}，東部地區的主要河川則有{{link-ja|高瀨川|高瀬川}}、{{link-ja|奧入瀨川|奥入瀬川}}<ref>{{cite web |url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/chikei/post-94.html　 |script-title=ja:主な河川 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。下北半島的[[大間崎|大間崎]]是[[本州|本州]]最北端的地點<ref>{{cite web |url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/chikei/post-92.html　 |script-title=ja:主な岬、崎 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

青森縣豐富的自然景觀使得青森縣擁有[[十和田八幡平國立公園|十和田八幡平國立公園]]和[[三陸復興國立公園|三陸復興國立公園]]兩個國立公園，以及[[下北半島國定公園|下北半島國定公園]]和[[津輕國定公園|津輕國定公園]]兩個國定公園，並且還擁有七個縣立自然公園<ref>{{cite web |url=http://www.pref.aomori.lg.jp/nature/nature/s-kouen_kouen.html　 |script-title=ja:青森県内の自然公園 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。位處青森和秋田兩縣交界處的[[白神山地|白神山地]]更是世界自然遺產<ref>{{cite web |url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/shizen/post-103.html　 |script-title=ja:自然環境 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[File:Aomori_bay.jpg|left]]
青森縣不僅在自然地理上可以分為東西兩部分，在人文地理上亦分為[[津輕地方|津輕地方]]和[[南部地方|南部地方]]。津輕地方包括了青森市、東津輕郡、弘前市、黑石市、平川市、中津輕郡、南津輕郡、五所川原市、津輕市、西津輕郡、北津輕郡。南部地方包括了陸奧市、下北郡、十和田市、三澤市、上北郡、八戶市、三戶郡<ref name="Kenminkyoku">{{cite web |url=http://reiki.pref.aomori.lg.jp/reiki_honbun/ac00100101.html　 |script-title=ja:青森県地域県民局及び行政機関設置条例 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。縣廳所在地[[青森市|青森市]]和江戶時代的兩大城下町[[八戶市|八戶市]]（八戶藩）、[[弘前市|弘前市]]（津輕藩）是青森縣的三大城市。青森市由津輕藩開港於1624年，最初是弘前的外港。在成為青森縣廳之後，青森市憑藉居中的地理位置和交通節點的優勢而發展為青森縣的經濟、政治、交通中心<ref name="Chishi"></ref>{{rp|182}}。八戶市位於青森縣東南部，是青森縣最大的工業城市和漁港<ref name="Chishi"></ref>{{rp|204}}。弘前市則是青森最早設[[市制_(日本制度)|市]]的城市，是青森縣的教育、文化中心，也是縣內主要蘋果產區<ref name="Chishi"></ref>{{rp|189}}。
=== 氣候 ===
青森縣氣候四季分明，夏季短暫涼爽，冬季漫長寒冷。以那須火山帯為界，青森縣西部和東部的氣候明顯不同<ref>{{cite web |url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/kisyo/post-102.html　 |script-title=ja:気候の概況 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。青森縣西部屬於[[日本海側氣候|日本海側氣候]]，在冬季受到西北季風的強烈影響，降雪量極多。青森市是都道府縣廳所在地城市中唯一一個全市轄區均是特別豪雪地帶的城市<ref>{{cite web |url=http://www.mlit.go.jp/crd/chisei/aomori.html　 |script-title=ja:青森県（豪雪地帯 －全県指定） |accessdate=2017-05-12 |publisher={{lang|ja|国土交通省}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。但在夏季的溫度和日照時間多於東部的太平洋沿岸地區<ref>{{cite web |url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/kisyo/post-101.html　 |script-title=ja:四季の特徴 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。青森縣東部則屬於[[太平洋側氣候|太平洋側氣候]]。在春夏之交時受到寒流的影響，有時會有冷夏的現象出現，亦是古代時期飢荒的主要原因。冬季的太平洋沿岸地區則降雪較少多晴天。青森縣南部的內陸地區由於距離海洋較遠，因此溫差較大，冬季比北部更加寒冷，夏季則高溫乾燥<ref name="Name"></ref>{{rp|19}}。
=== 行政區劃 ===
青森縣現在管轄有10市22町8村，町村分屬八個郡。[[青森市|青森市]]和[[東津輕郡|東津輕郡]]（[[平內町|平內町]]、[[今別町|今別町]]、[[蓬田村|蓬田村]]、[[外濱町|外濱町]]）屬於東青地域縣民局。[[弘前市|弘前市]]、[[黑石市|黑石市]]、[[平川市|平川市]]、[[中津輕郡|中津輕郡]]（[[西目屋村|西目屋村]]）、[[南津輕郡|南津輕郡]]（[[藤崎町|藤崎町]]、[[大鰐町|大鰐町]]、[[田舍館村|田舍館村]]）屬於中南地域縣民局。[[五所川原市|五所川原市]]、[[津輕市|津輕市]]、[[西津輕郡|西津輕郡]]（[[鰺澤町|鰺澤町]]、[[深浦町|深浦町]]）、北津輕郡（[[板柳町|板柳町]]、[[鶴田町|鶴田町]]、[[中泊町|中泊町]]）屬於西北地域縣民局。[[陸奧市|陸奧市]]和[[下北郡|下北郡]]（[[大間町|大間町]]、[[東通村|東通村]]、[[風間浦村|風間浦村]]、[[佐井村|佐井村]]）屬於下北地域縣民局。[[十和田市|十和田市]]、[[三澤市|三澤市]]、[[上北郡|上北郡]]（[[野邊地町|野邊地町]]、[[七戶町|七戶町]]、[[六戶町|六戶町]]、[[橫濱町|橫濱町]]、[[東北町|東北町]]、[[六所村|六所村]]、[[奧入瀨町|奧入瀨町]]）屬於上北地域縣民局。八戶市和[[三戶郡|三戶郡]]（[[三戶町|三戶町]]、[[五戶町|五戶町]]、[[田子町|田子町]]、[[南部町_(青森縣)|南部町]]、[[階上町|階上町]]、[[新鄉村|新鄉村]]）屬於三八地域縣民局<ref name="Kenminkyoku"></ref>。

== 人口 ==
[[File:09年青森県の人口の推移.png|right]]
青森縣在1920年時有人口756,454人，是[[東北地方|東北地方]]人口最少的縣<ref name="Population1">{{cite web |url=http://www6.pref.aomori.lg.jp/tokei/data/0000002768/0000002768_2_1.pdf　 |script-title=ja:青森県の人口と世帯数 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1940年人口普查時，青森縣人口突破百萬大關<ref name="Population1"></ref>。此後青森縣人口數又接連超過秋田縣、[[山形縣|山形縣]]、岩手縣，在1965年成為東北地方僅次於[[宮城縣|宮城縣]]、[[福島縣|福島縣]]的人口第三大縣，達1,416,591人<ref name="Population1"></ref>。1983年，青森縣人口達到1,529,269人的峰值<ref name="Population2">{{cite web |url=http://www.pref.aomori.lg.jp/soshiki/kikaku/seisaku/files/sousei_pref_aomori_vision.pdf　 |script-title=ja:青森県長期人口ビジョン |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。此後青森縣人口持續減少。在2004年至2014年期間，青森縣人口減少了9%，是日本人口減少幅度第二大的縣，僅次於秋田縣<ref name="Population3">{{cite web |url=http://www.pref.aomori.lg.jp/soshiki/kikaku/tokei/files/H27_hakusho_1-5.pdf　 |script-title=ja:青森県の人口 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。青森縣人口佔日本總人口的比重也從1950年代的1.6%下降到現在的約1%<ref name="Population3"></ref>。2017年4月1日時，青森縣有人口1,282,545人<ref>{{cite web |url=http://www6.pref.aomori.lg.jp/tokei/document_view.php?sheet_no=3979　|script-title=ja:平成29年4月1日推計人口 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja}}</ref>。

人口外流和老齡化、少子化都是青森縣人口減少的原因。青森縣長期是人口外流縣，人口遷出率是日本首位。眾多青年人口因升學和就業而在20歲前後時離開青森。就人口遷出的趨勢變化來看，青森的人口外流量和縣內的就業情勢有密切的關係<ref>{{cite web |url=http://www.pref.aomori.lg.jp/soshiki/kikaku/tokei/files/tokei_topics3.pdf　 |script-title=ja:青森県の人口移動者数の特徴からわかること |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。青森自1999年開始人口的自然增長率就是負增長<ref name="Population3"></ref>。現在青森縣15至64歲人口約佔總人口的六成，而65歲以上人口佔總人口的約四分之一，是14歲以下人口的兩倍<ref name="Population2"></ref>。青森縣的人口分佈較為分散，並沒有出現一極集中的情況。青森縣人口最多的城市是[[青森市|青森市]]、[[八戶市|八戶市]]和[[弘前市|弘前市]]，三市人口合計佔青森縣總人口的約53%<ref name="Population1"></ref>。青森縣的[[總和生育率|總和生育率]]在2000年代中期之前曾高於日本平均，但現在只有1.4，略低於日本平均值<ref name="Population2"></ref>。青森縣已經將人口減少問題作為縣政的最重要課題<ref>{{cite web |url=http://www.pref.aomori.lg.jp/kensei/seisaku/sousei.html　 |script-title=ja:青森県の人口減少対策 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

青森縣的男性[[预期寿命|平均壽命]]是77.28歲，女性平均壽命是85.34歲，均在日本所有都道府縣中敬陪末座，但男女差距卻是全國最大<ref>{{cite web|url=http://www.mhlw.go.jp/toukei/saikin/hw/life/tdfk10/dl/02.pdf|title=|accessdate=2017-05-15|author=|date=|publisher={{lang|ja|厚生労働省}}|language=ja|script-title=ja:都道府県別にみた平均余命}}</ref>。值得一提的是，青森縣有超過三成的新生兒都是未婚先孕，這一比例在日本高居第三<ref>{{cite web |url=http://www.mhlw.go.jp/toukei/saikin/hw/jinkou/tokusyu/syussyo06/dl/01.pdf　 |script-title=ja:平成 22 年度「出生に関する統計」の概況 |accessdate=2017-05-12 |publisher={{lang|ja|厚生労働省}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 政治 ==

1996年日本導入[[小選舉區制|小選舉區制]]之後，在[[眾議院_(日本)|眾議院]]選舉中，青森被劃分為4個選舉區。這四個選舉區均是[[自由民主黨_(日本)|自民黨]]佔據壓倒性優勢的選區。2000年以來，這四個選舉區中，除了[[青森縣第1區|青森縣第1區]]在2009年曾由[[民主黨_(1998年)|民主黨]]候選人當選之外，其他三個選舉區自民黨保持全勝<ref>{{cite web |url=http://www.pref.aomori.lg.jp/soshiki/senkan/files/47shu_tokuhyo.pdf　 |script-title=ja:衆議院議員選挙　候補者別得票数 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2017年選舉區重劃後，青森縣被劃分為3個選舉區<ref>{{cite web|url=http://www.soumu.go.jp/main_content/000240040.pdf|title=|accessdate=2017-10-22|author=|date=|publisher={{lang|ja|総務省}}|language=ja|script-title=ja:青森県}}</ref>。在選區重劃後的首次眾議院選舉中，自民黨依然壟斷了縣內三個議席<ref>{{cite web|url=http://www.yomiuri.co.jp/election/shugiin/2017/kaihyou/ya02.html|title=|accessdate=2017-10-24|author=|date=|publisher={{lang|ja|読売新聞}}|language=ja|script-title=ja:【小選挙区】青森県}}</ref>。在[[參議院_(日本)|參議院]]選舉中，青森縣屬於[[青森縣選舉區|青森縣選舉區]]，共有2名議員。目前自民黨和[[民進黨_(日本)|民進黨]]在該選區各有一名議員<ref>{{cite web |url=http://www.pref.aomori.lg.jp/soshiki/senkan/files2/san_tokuhyo.pdf　 |script-title=ja:参議院議員選挙　候補者別得票数 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[青森縣知事列表|青森縣知事]]亦多由保守派人士擔任。現任青森縣知事是{{link-ja|三村申吾|三村申吾}}。他在2003年首次當選，並在2007年、2011年、2015年順利連任<ref>{{cite web|url=http://www.pref.aomori.lg.jp/soshiki/senkan/files2/tiji_tokuhyou.pdf　|script-title=ja:青森県知事選挙　候補者別得票数|accessdate=2017-05-12|publisher={{lang|ja|青森県}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。{{link-ja|青森縣議會|青森県議会}}共有48名議員，其中自民黨是最大黨派，有31名議員<ref>{{cite web|url=http://www.pref.aomori.lg.jp/soshiki/gikai/giin-kaiha.html　|script-title=ja:議員の紹介　（会派別）|accessdate=2017-05-12|publisher={{lang|ja|青森県}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。青森縣財政狀況十分嚴峻，財政力指數只有0.332，低於日本平均值0.473。但近年青森縣已實現財政盈餘，財政狀況略有好轉<ref>{{cite web |url=http://www.pref.aomori.lg.jp/soshiki/soumu/zaisei/files/zaiseijijou_201612_all.pdf.pdf　 |script-title=ja:青森県財政事情 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
=== 軍事 ===
由於[[津輕海峽|津輕海峽]]是[[公海|公海]]，允許外國的潛水艇及軍艦航行。加上外國軍用機亦可通過津輕海峽上空飛行。使得青森縣在日本的國防中有重要地位。[[自衛隊|自衛隊]]陸、海、空部隊均在青森縣有駐軍。{{link-ja|大湊基地|大湊基地}}是[[海上自衛隊|海上自衛隊]]規模最大的基地之一<ref>{{cite web |url=http://www.mod.go.jp/msdf/oominato/about/index.html　 |script-title=ja:大湊地方隊について |accessdate=2017-05-12 |publisher={{lang|ja|大湊地方隊}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[美國空軍|美國空軍]]也在青森縣設有[[三澤飛行場|三澤基地]]<ref>{{cite web |url=http://www.mod.go.jp/asdf/misawa/kiti/index.html　 |script-title=ja:基地紹介 |accessdate=2017-05-12 |publisher={{lang|ja|防衛省}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。青森縣也是[[東京都|東京都]]以外唯一一個陸海空自衛隊將領均有常駐的縣。

== 經濟 ==
2013年度，青森縣的縣內生產總值有44,115億日元，位居日本第30位。人均縣民所得是242.6萬日元，位居日本第40位，大約是日本平均值的八成<ref>{{cite web |url=http://www.pref.aomori.lg.jp/soshiki/kikaku/tokei/files/H28.7.topics.pdf　 |script-title=ja:平成25年度県民経済計算（全都道府県）における青森県の位置づけ |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
===第一產業===
[[File:Malus_domestica_Fuji_Apple_Hirosaki_Aomori_Japan_20161016b.jpg|left]]
2015年，青森縣有農家44,781戶，農業就業人口64,746人，耕地面積15.33萬公頃<ref>{{cite web |url=http://www.pref.aomori.lg.jp/soshiki/nourin/nosui/files/H26to_1-2.pdf　 |script-title=ja:主要指標 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在青森的縣內生產總值中，農業佔比4.1%，遠高於日本平均值0.9%<ref name="Agriculture">{{cite web |url=http://www.pref.aomori.lg.jp/soshiki/nourin/nosui/files/H26zuall.pdf　 |script-title=ja:図説　農林水産業の動向 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。青森縣的農業生產額達2,879億日元，位居日本第八，並且在近年實現穩定增長<ref name="Agriculture"></ref>。青森的農業及漁業就業者比例也遠高於日本平均值<ref>{{cite web |url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/sangyo/tokucho/　 |script-title=ja:産業の特徴と経済構造 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

青森的農業生產額中以畜牧產品比重最大，其次是水果、蔬菜、稻米<ref name="Agriculture"></ref>。由於青森縣地形變化多樣，不同地區氣候風土差異很大，因此農業生產也各具地區特色。西部的津輕地區在春季至秋季的氣候比較溫和，土壤亦較肥沃，以蘋果、水稻等作物的種植為主。東部的南部地區則以旱田和畜產業為主<ref>{{cite web |url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/sangyo/nogyo/post-278.html　 |script-title=ja:農業の概要 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。青森的[[大蒜|大蒜]]、[[蘋果|蘋果]]生產量不僅是日本首位，且單獨佔據日本國內過半市場份額<ref>{{cite web |url=http://www.pref.aomori.lg.jp/kensei/tokei/graph-aomori_nousui.html　 |script-title=ja:こどもデータランド－グラフでみる青森県－農水産業 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。青森的蘋果種植史開始於明治時期，在戰前就已經是日本主要蘋果產地。1960年代，青森的蘋果產業曾面臨香蕉等廉價外國產水果的衝擊。此後當地積極研發新品種，使得青森的蘋果產業得以度過危機<ref>{{cite web |url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/sangyo/nogyo/post-275.html　 |script-title=ja:果樹（りんご栽培の歴史、りんごの生産・販売・加工 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。著名蘋果品種[[富士蘋果|富士]]就是發祥於青森，現已佔青森蘋果總產量的近半數，並普及至世界多地<ref>{{cite web |url=http://www.town.fujisaki.lg.jp/index.cfm/8,0,13,226,html　 |script-title=ja:りんご「ふじ」発祥の地 |accessdate=2017-05-12 |publisher={{lang|ja|藤崎町}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

|bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2013年度，青森縣的木材生產量是日本第七位<ref>{{cite web |url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/sangyo/ringyo/　 |script-title=ja:林業 ===第三產業===

== 文化 ==
=== 方言 ===
青森縣是當地方言和日語標準語差距最大的地區之一。青森縣的方言可以分為日本海沿岸地區的{{link-ja|津輕方言|津軽弁}}、太平洋沿岸地區的{{link-ja|南部方言|南部弁}}和[[下北半島|下北半島]]地區的{{link-ja|下北方言|下北弁}}。津輕方言和南部方言之間大致以奧羽山脈為界<ref name="Dialect">{{cite book |author={{lang|ja|佐藤和之}}|script-title=ja:《青森県のことば》 |year=2003 |publisher={{lang|ja|明治書院}} |location=東京都 |isbn=4-625-62303-0|language=ja}}</ref>{{rp|3}}。一般認為津輕方言較具抑揚頓挫，而南部方言則較為柔和<ref name="Dialect"></ref>{{rp|5}}。在津輕方言中，又以城下町和文化中心的弘前方言地位最高，被視為津輕方言的標準語<ref name="Dialect"></ref>{{rp|28}}。南部方言在音調、助詞文法、敬語上和津輕方言略有不同<ref name="Dialect"></ref>{{rp|28}}。下北方言雖然音調和南部方言近似，但在詞彙等方面則受到津輕方言、北海道方言一定程度的影響<ref>{{cite web|url=http://www6.pref.aomori.lg.jp/p-view/information/konnatoko/topics/kotoba/post-269.html　|script-title=ja:同じ意味でも言葉が違う！|accessdate=2017-05-12|publisher={{lang|ja|青森県}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
=== 文學 ===
[[太宰治|太宰治]]是青森縣出身的作家中最著名的一人。太宰治出生在青森縣北津輕郡金木村（現[[五所川原市|五所川原市]]），是[[無賴派|無賴派]]的代表作家。他的[[跑吧！美樂斯|跑吧！美樂斯]]被選入日本國語教科書，知名度極高。[[斜陽_(小說)|斜陽]]、[[人間失格|人間失格]]等作品也十分著名。太宰治的小說{{link-ja|津輕 (小説)|津軽 (小説)|津輕}}是以他的故鄉青森為舞台<ref>{{cite web |url=http://dazai.or.jp/modules/know/know_annals.html　|script-title=ja:太宰治の生涯 略年譜|accessdate=2017-05-12 |publisher={{lang|ja|大宰ミュージアム}} |language=ja}}</ref>。除了太宰治之外，青森縣還有{{link-ja|今官一|今官一}}、{{link-ja|三浦哲郎|三浦哲郎}}等獲得[[直木獎|直木獎]]、[[芥川獎|芥川獎]]的作家<ref>{{cite web|url=http://www6.pref.aomori.lg.jp/p-view/information/sugoihito/bungaku/　|script-title=ja:文学|accessdate=2017-05-12|publisher={{lang|ja|青森県}}|language=ja}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 觀光 ==
[[File:Sirakami_santi.JPG|right]]
青森縣擁有眾多自然風光觀光資源。青森、秋田兩縣交界處的[[白神山地|白神山地]]和[[屋久島|屋久島]]並列為日本最初的世界自然遺產，擁有貴重的日本山毛欅原生林<ref>{{cite web |url=http://www.pref.aomori.lg.jp/nature/nature/shirakami.html　 |script-title=ja:世界遺産白神山地 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在日本經濟高速成長時期，眾多闊葉林被經濟價值高的人工針葉林取代，但白神山地卻幸運地未被採伐。1970年代時，政府曾計劃修築縱貫白神山地的公路，也在當地人士的反對之下撤回，使得白神山地得以保留珍貴的自然生態<ref name="Chishi"></ref>{{rp|182}}。白神山地內的[[十二湖|十二湖]]以高透明度和蔚藍的湖水而著稱<ref>{{cite web |url=http://www.daiichikanko.jp/about06.html　 |script-title=ja:オススメの白神「十二湖」の楽しみ方 |accessdate=2017-05-12 |publisher={{lang|ja|白神なび}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。青森南部的[[十和田湖|十和田湖]]和[[奧入瀨溪流|奧入瀨溪流]]是特別名勝。十和田湖是一個火口湖，也是日本面積第十二大湖泊和深度第三深湖泊，以紅葉景觀而著稱<ref>{{cite web |url=http://www.towada-kankou.jp/towadako-area.html　 |script-title=ja:十和田湖周辺の見どころ |accessdate=2017-05-12 |publisher={{lang|ja|十和田市}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。奧入瀨溪流則是日本著名的清流，流域保存有原始的天然林<ref>{{cite web |url=http://towadako.or.jp/sansaku-map/oirase-nenokuchi/　 |script-title=ja:奥入瀬渓流～子ノ口 － 十和田湖散策マップ |accessdate=2017-05-12 |publisher={{lang|ja|十和田湖国立公園协会}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。兩者都被劃入[[十和田八幡平國立公園|十和田八幡平國立公園]]加以保護<ref>{{cite web |url=http://towadako.or.jp/towadako-oirase/　 |script-title=ja:十和田湖・奥入瀬渓流の魅力 ～深呼吸したくなる十和田湖・奥入瀬～ |accessdate=2017-05-12 |publisher={{lang|ja|十和田湖国立公園协会}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[下北半島國定公園|下北半島國定公園]]內則有[[恐山|恐山]]、{{link-ja|佛浦|仏ヶ浦}}、[[大間崎|大間崎]]等景點<ref>{{cite web |url=http://www.pref.aomori.lg.jp/nature/nature/kouen_shimokita.html　 |script-title=ja:下北半島国定公園 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[津輕國定公園|津輕國定公園]]內地形變化多端，內有{{link-ja|龍飛崎|竜飛崎}}、[[岩木山|岩木山]]等景點<ref>{{cite web |url=http://www.pref.aomori.lg.jp/nature/nature/kouen_tsugaru.html　 |script-title=ja:津軽国定公園 |accessdate=2017-05-12 |publisher={{lang|ja|青森県}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

青森縣內溫泉度假地眾多，著名的有[[古牧溫泉|古牧溫泉]]、{{link-ja|酸湯|酸ヶ湯|酸湯溫泉}}、{{link-ja|淺蟲溫泉|浅虫温泉}}等地。酸湯溫泉是日本第一個獲得國民保養溫泉地稱號的溫泉<ref name="Chishi"></ref>{{rp|179}}。位於青森市郊區的淺蟲溫泉因交通便利而遊客較多，有「東北的[[熱海市|熱海]]」之稱<ref>{{cite web |url=http://www.mutusinpou.co.jp/%E6%B4%A5%E8%BB%BD%E3%81%AE%E8%A1%97%E3%81%A8%E9%A2%A8%E6%99%AF/2015/03/35640.html　 |script-title=ja:遊興地となる浅虫温泉＝２２ |accessdate=2017-05-12 |publisher={{lang|ja|陸奥新報}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[弘前城|弘前城]]是日本十二[[現存天守|現存天守]]之一，也是東北地方唯一的現存天守<ref>{{cite web |url=http://www.hirosakipark.jp/hirosakijo.html　 |script-title=ja:東北で唯一の現存天守 弘前城 |accessdate=2017-05-12 |publisher={{lang|ja|弘前公園}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[弘前公園|弘前公園]]擁有日本最古的[[染井吉野櫻|染井吉野櫻]]，擁有超過2600株櫻花<ref>{{cite web |url=http://www.hirosakipark.jp/viewspot/sakura.html　 |script-title=ja:桜 （サクラ） |accessdate=2017-05-12 |publisher={{lang|ja|弘前公園}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。每年春季的{{link-ja|弘前櫻花祭|弘前さくらまつり}}是青森縣最大的觀光活動之一<ref>{{cite web |url=http://www.hirosakipark.jp/sakura/cherryblossomfestival.html　 |script-title=ja:弘前さくらまつりについて |accessdate=2017-05-12 |publisher={{lang|ja|弘前さくらまつり}} |language=ja }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。秋季的弘前公園舉辦有{{link-ja|弘前城菊花和紅葉節|弘前城菊と紅葉まつり}}，呈現出和春季截然不同的風貌<ref>{{cite web |url=http://www.hirosaki-kanko.or.jp/web/edit.html?id=cat02_autumn_momiji　|script-title=ja:弘前城菊と紅葉まつり |accessdate=2017-05-12 |publisher={{lang|ja|弘前観光コンベンション協会}} |language=ja}}</ref>
