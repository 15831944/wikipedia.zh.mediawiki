'''西昌街224巷'''，常稱作'''青草巷'''，是[[臺北市|臺北市]][[萬華區|萬華區]]的一條巷道，位於[[艋舺龍山寺|艋舺龍山寺]]、[[艋舺地藏王廟|艋舺地藏王廟]]附近。青草巷因有許多販賣青草、中藥的集中地而聞名，最早的草藥店可追溯至超過100年前，如今仍保存[[臺灣清治時期|清代]]的路街分布<ref name="臺北市地名與路街沿革史"> {{Cite book | author = 湯熙勇 | title = 臺北市地名與路街沿革史 | location = 臺北市 | publisher = [[臺北市文獻委員會|臺北市文獻委員會]] | date = 2002 | pages = 365 | ISBN = 957-01-3019-9 | accessdate = 2018-06-29 | url = | language = zh-tw | quote = }} </ref>。2015年，[[臺北市政府|臺北市政府]]將'''青草巷歷史建築群'''列為[[臺灣歷史建築|歷史建築]]，成為[[文化資產|文化資產]]。<ref name="文化資產">{{Cite web |url = https://nchdb.boch.gov.tw/assets/overview/historicalBuilding/20150824000001 |title = 青草巷歷史建築群 |author = |date = |publisher = [[文化部|文化部]][[文化資產局|文化資產局]] |language = zh-tw |accessdate = 2018-06-29}}</ref>

==歷史==
關於青草巷的起源有許多說法。一說[[艋舺|艋舺]]早期開墾時疾病眾多，居民通常賴口傳偏方或龍山寺的藥籤，民眾因而到附近青草店買青草治病<ref> {{Cite book | author = 黃沼元 | title = 台灣的老街 | location = 臺北縣新店市 | publisher = [[遠足文化|遠足文化]] | date = 2002 | pages = 40 | ISBN = 957-28031-1-5 | accessdate = 2018-06-29 | url = | language = zh-tw | quote = }} </ref><ref>{{cite news |language = zh-cn |author = 龍敏 |coauthors = 劉舒凌 |url = http://www.xinhuanet.com/tw/2017-04/04/c_129524500.htm |title = 台北青草巷：隽永清新 百年传承 |publisher = [[中國新聞社|中國新聞社]] |pages = |date = 2017-04-04 |accessdate = 2018-06-29}}</ref>，因此又被稱為「救命街」<ref>{{cite news |language = zh-tw |author = 朱純慧 |coauthors = 謝祝芬 |url =http://www.nextmag.com.tw/realtimenews/news/19510208 |title = 《老字號》百年家業婿傳承　德安青草店 |publisher = [[壹週刊|壹週刊]] |pages = |date = 2015-05-21 |accessdate = 2018-06-29}}</ref>，也有「龍山寺旁的草藥特別有效」的傳言；一說萬華附近是[[遊廓|遊廓]]，許多人至此尋求草藥治療性病<ref name="TheTaipeiPictorial">{{Cite journal|author= 徐亞媛 |url= |title= 走尋台北人系列（五）|journal= 台北畫刊|year= 1996 |issue= 346 |pages= 40 }}</ref>。

龍山寺的周圍很早有青草藥店聚集。但直到[[臺灣戰後時期|戰後]]，周圍的道路拓寬、房屋改建，青草店鋪才聚集至青草巷內<ref name="臺北歷史散步"> {{Cite book | author = 莊展鵬 | title = 臺北歷史散步 | location = 臺北市 | publisher = [[遠流出版公司|遠流出版公司]] | date = 1991 | pages = 42 | ISBN = 957-32-1114-9 | accessdate = 參考日期2018-06-29| url = | language = zh-tw | quote = }} </ref>。

2001年開始，[[臺北市政府|臺北市政府]]與當地住戶合作，將其整修為臺北市文化生活空間改善的示範地，並設立解說版描述青草巷的演變過程<ref name="臺北市地名與路街沿革史" />。2015年，臺北市政府以基地原屬螺陽公會所有，為清代[[祭祀公業|祭祀公業]]重要歷史見證、青草巷為龍山寺旁重要歷史街巷空間，為臺北市傳統信仰、青草產業、[[泉州三邑|泉州三邑]]移民聚集之艋舺城市中心，具有城市社會史見證意義等原因，登錄青草巷歷史建築群為臺北市歷史建築<ref name="文化資產" />。

== 產業特色 ==
青草巷附近的青草店傳統以販賣[[藥草|藥草]]、[[青草茶|青草茶]]為主。[[台灣日治時期|日治時期]]時，草藥由店家自行採摘<ref name="TheTaipeiPictorial" />。現在的青草來源來自台灣各地，來自北部的用作新鮮青草，而來自南部的多曬乾使用。民眾常在廟中求完藥籤後，在青草店抓藥草內服、外敷。為了消暑，也有許多人會到青草店購買青草茶<ref>{{cite news |language = zh-tw |author = 林嘉玫 |coauthors = 薛學華 |url = https://www.youtube.com/watch?v=9zVUDG1-UfU |title = 天熱喝涼茶 萬華青草巷生意好 |publisher = [[民視|民視]] |pages = |date = 2011-07-28|accessdate = 2018-06-29}}</ref>。

1980年代立法禁止寺廟開藥方、與[[全民健康保險|全民健康保險]]制度實施，青草巷買氣逐漸減少。手搖飲料的普及，也使青草茶市場萎縮。部分青草業者謀求轉型，以手沖茶等方式推廣青草<ref>{{cite news |language = zh-tw |author = 蔡碧月 |coauthors = |url =http://www.nextmag.com.tw/realtimenews/news/371084 |title = 生存空間被壓縮　青草老店改造自救 |publisher = [[壹週刊|壹週刊]] |pages = |date = 2017-12-14 |accessdate = 2018-06-29}}</ref>。青草巷店家目前也已推出藥草[[宅配|宅配]]服務，只要一通電話，店家就會將青草送至民眾家中，還有宅配到[[金門縣|金門縣]]的紀錄。<ref>{{Cite web |url = https://whdo.gov.taipei/News_Content.aspx?n=BA44C2AB122C71F4&s=088D6742A9341CC9 |title = 地方特色─青草巷 |author = 臺北市萬華區公所 |date = 2009-11-02 |publisher =臺北市萬華區公所|language = zh-tw |accessdate = 2018-06-29 |quote =  }}</ref>
