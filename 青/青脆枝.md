'''青脆枝'''（[[学名|学名]]：{{lang|la|'''''Nothapodytes nimmoniana'''''}}）为[[茶茱萸科|茶茱萸科]][[鹰紫花树属|鹰紫花树属]]下的一个种。
因其莖部分枝甚多，容易折斷故名為青脆枝<ref>{{cite book |author1=黃鵬 |title=東部地區常見藥用植物圖鑑-1 |publisher=行政院農業委員會花蓮區農業改良場 |isbn=9789860343199 |page=120-121 |url=https://www.books.com.tw/products/0010578055}}</ref>。

== 名稱 ==
又可稱為臭味假柴龙树或臭馬比木<ref>{{cite book |author1=章仁香 |title=台灣原住民族藥用植物圖鑑 |date=2012 |isbn=9789860182408 |page=149 |url=https://www.apc.gov.tw/portal/docDetail.html?CID=D6DC92246BCF08A8&DID=0C3331F0EBD318C2965DEAE61BD4C0FD}}</ref>。
臺灣不同地區的原住民對其有不同之稱呼。布農族稱之為Masanglav Mazaubum Tu Ismud，西拉雅族稱之為Sasabo, 鄒族稱之為Cuum Patungana，達悟(雅美)族稱之為 Kamanvoaga<ref>{{cite book |author1=章仁香 |title=台灣原住民族藥用植物圖鑑 |date=2012 |isbn=9789860182408 |page=149 |url=https://www.apc.gov.tw/portal/docDetail.html?CID=D6DC92246BCF08A8&DID=0C3331F0EBD318C2965DEAE61BD4C0FD}}</ref>。
早期學名為''Nothapodytes foetida'' (Wight) Sleum<ref>{{cite book |author1=林則桐 |author2=呂勝由 |title=蘭嶼植物 |publisher=臺灣省政府教育廳 |page=120}}</ref>。

== 形態 == 
可為喬木或灌木，自然環境中，高度可達3至3.5公尺，主幹雖直立，但容易產生分枝。
常綠喬木葉序互生或對生。葉形橢圓卵形或近橢圓披針形，長8-25公分左右，寬6-13公分左右。葉基漸狹至鈍形，葉尖銳尖或漸尖。毛被物存在於近軸面與遠軸面。具有葉炳<ref>{{cite book |author1=鐘詩文 |title=台灣原生植物圖鑑第六卷 |publisher=貓頭鷹出版 |isbn=9789862623527 |page=158}}</ref>。
花序為繖房花序型或聚繖花序型，花序頂生。花瓣內有絨毛。花兩性。花色雪白。花萼筒狀且五裂。花數五數。子房上位。花式為
H<sup>w(H)</sup> L<sup>a</sup> I<sup>R,Co,Cy</sup> C a<sup>4-5</sup> C o<sup>4-5</sup> S<sup>4-5</sup>，G<sup>[3-5]</sup> O<sup>1</sup> P a F<sup>D</sup><ref>{{cite book |author1=黃增泉 |title=植物分類學 : 臺灣維管束植物科誌 |publisher=南天書局有限公司 |isbn=9576383951 |page=288}}</ref>。
果實為核果，橢圓形，成熟時轉紅<ref>{{cite book |author1=林則桐 |author2=呂勝由 |title=蘭嶼植物 |publisher=臺灣省政府教育廳 |page=120}}</ref>。

== 生態 == 
在臺灣，可見於蘭嶼和綠島之灌木叢或者次生林之中，亦常為海濱植物<ref>{{cite book |author1=黃增泉 |title=植物分類學 : 臺灣維管束植物科誌 |publisher=南天書局有限公司 |isbn=9576383951 |page=288}}</ref>。
同時也分布於印度半島、錫蘭、中南半島、蘇門答臘、呂宋島、琉球<ref>{{cite book |author1=林則桐 |author2=呂勝由 |title=蘭嶼植物 |publisher=臺灣省政府教育廳 |page=120}}</ref>。
現在多為人工栽培，以日本業者在臺灣東部進行契作為最大宗<ref>{{cite book |author1=黃鵬 |title=東部地區常見藥用植物圖鑑-1 |publisher=行政院農業委員會花蓮區農業改良場 |isbn=9789860343199 |page=120-121 |url=https://www.books.com.tw/products/0010578055}}</ref>。

== 民族植物學 ==
在臺灣，布農族、西拉雅族、鄒族和達悟(雅美)族使用該植物<ref>{{cite book |author1=章仁香 |title=台灣原住民族藥用植物圖鑑 |date=2012 |isbn=9789860182408 |page=149 |url=https://www.apc.gov.tw/portal/docDetail.html?CID=D6DC92246BCF08A8&DID=0C3331F0EBD318C2965DEAE61BD4C0FD}}</ref>。

== 藥用 ==
全株可為藥用部位。味辛性溫，微量毒性。抗癌潛力，尤其針對直腸癌，除去濕氣，退風寒，可消浮腫，緩解關節疼痛<ref>{{cite book |author1=章仁香 |title=台灣原住民族藥用植物圖鑑 |date=2012 |isbn=9789860182408 |page=149 |url=https://www.apc.gov.tw/portal/docDetail.html?CID=D6DC92246BCF08A8&DID=0C3331F0EBD318C2965DEAE61BD4C0FD}}</ref>。藥效化學成分為喜樹鹼(Camptothecin)<ref>{{cite book |author1=黃鵬 |title=東部地區常見藥用植物圖鑑-1 |publisher=行政院農業委員會花蓮區農業改良場 |isbn=9789860343199 |page=120-121 |url=https://www.books.com.tw/products/0010578055}}</ref>。

== 繁殖方式 ==
採種後以種子進行栽培<ref>{{cite book |author1=黃鵬 |title=東部地區常見藥用植物圖鑑-1 |publisher=行政院農業委員會花蓮區農業改良場 |isbn=9789860343199 |page=120-121 |url=https://www.books.com.tw/products/0010578055}}</ref>。
