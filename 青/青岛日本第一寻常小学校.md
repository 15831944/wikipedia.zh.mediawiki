'''青岛日本第一寻常高等小学校旧址'''位于中国[[山东省|山东省]][[青岛市|青岛市]][[市北区|市北区]][[武定路|武定路]]29号，建于1917年，是青岛老城区最有代表性的日本建筑之一。

==历史==
1915年，青岛守备军司令部在原德国[[青岛督署学校|督署学校]]的校址上建立了青岛小学校。1917年3月26日，位于{{lang|ja|花咲町}}（今武定路）、由建筑师小山良树、田原正义和长冈平藏联合设计、华胜公司施工的新校舍建成，原青岛小学校迁入该校舍并改名为青岛第一[[寻常高等小学校|寻常高等小学校]]，原校址设立第二寻常高等小学校。同年5月10日，[[文部省|文部省]]指定该校为“在外指定学校<ref Name=ben>{{cite web |url=http://www.qdda.gov.cn/expertLibrary.do?method=viewingArticle&articleId=13559019741192144001&subject_id=12259376220001761001|title=日本第一次占领青岛时期的移民潮|accessdate=2012-12-19|work=孙保锋|publisher=青岛档案信息网}}</ref><ref>{{cite web |url=http://arc.dailyqd.com/2015-04/14/content_241136_all.htm|title=盘点日本第一次占领青岛时所建小学校|accessdate=2015-04-14|publisher=青岛日报}}</ref>。[[五三惨案|1928年日本出兵山东]]时曾为日本海军陆战队驻地。1941年根据[[国民学校令|国民学校令]]改称青岛日本第一国民学校。1945年学校解散。1949年后由军队接管，后成为青岛警备司令部驻地。警备司令部迁往东部后仍为军事管理区。2010年后改为酒店。该建筑自2005年以后即为青岛市文物保护单位<ref Name=qing>{{cite web |url=http://club.qingdaonews.com/showAnnounce_57_3838583_1_0.htm|title=青岛第一日本寻常小学旧址}}</ref>。

==建筑特色==
[[File:青岛日本第一寻常小学校后侧及体育场_03.jpg|thumb]]
建筑群由主楼、体育场及礼堂等附属建筑组成。主楼正立面朝西临武定路，并向东延伸出两翼，共两层，附设坡屋顶阁楼，上覆牛舌瓦，花岗岩剁斧石砌墙基，浅色拉毛墙面。正立面中央檐口处起三角形大山墙，正立面两端、南北立面两端及建筑两翼的东端檐口处起大小不一的三角形山墙。正立面屋顶中央有一木制尖顶塔楼。墙面以花岗岩石雕装饰，具有[[哥特复兴式|哥特复兴式]]风格。内部为水磨石地面，石制楼梯及楼梯柱，铁制扶手<ref Name=qing/>。主楼东北方有一礼堂及两层附属建筑，外立面设计相仿。体育场位于建筑群东侧，东临[[热河路_(青岛)|热河路]]和[[德平路|德平路]]，面积广大，附有看台。该体育场在[[青岛市体育场|青岛市体育场]]建成前是青岛市区最大的专用体育场。

学校体育场于1950年代改为青岛市第二体育场，其后屡经变迁。校舍主楼的木制塔楼已不存在，建筑内部在改造为酒店时进行了部分改造<ref Name=qing/>。礼堂约于1960年代被拆除，其东侧的附属建筑与新建的礼堂相连接，后来大规模改造为酒店餐厅。临武定路一面保存着一段与主楼同时建成、花岗岩砌成的外墙与铁制栅栏。
