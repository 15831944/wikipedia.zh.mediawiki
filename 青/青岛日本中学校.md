'''青岛日本中学校旧址'''位于中国[[山东省|山东省]][[青岛市|青岛市]][[市南区|市南区]][[鱼山路|鱼山路]]5号，即今[[中国海洋大学|中国海洋大学]]鱼山路校区南门及附近。建于1920-1921年，是青岛老城区最有代表性的日本建筑之一。

==历史==
===原清军炮队营===
日本中学校舍的原址，是[[章高元|章高元]]驻防时期（1892-1897）所建炮队营（又称炮兵营）营房，位于[[总兵衙门_(青岛)|总兵衙门]]以东，[[小鱼山_(青岛)|小鱼山]]以北，平面呈矩形，正门由石材砌筑，朝向西南，拱门上方有“营门”二字牌匾。周围筑有夯土围墙，墙内营房均为中式平房。德军占领胶澳后，将其沿用为兵营，仍称为炮兵营（Artillerielager），德租初期部分总督府机构亦曾设于此。夯土围墙不久被拆除改为栅栏墙。德军各新营房建成后，营内部分房屋拆除，改为军官网球场。该营房是唯一留存到第一次日占时期的原清军营房。1920年3月，为建设日本中学校舍，原营房被拆除<ref name=ta>{{cite book |author=李明 |title=《塔楼上的青岛》|year=2008 |publisher=青岛出版社 |location=青岛 |isbn=9787543649705}}</ref><ref>{{cite web |url=http://www.weibo.com/u/1404199805?profile_ftype=1&is_all=1&is_search=1&key_word=炮兵营#_0|title=炮兵营}}</ref>。

===日本中学校===
[[File:青岛日本中学校.jpg|thumb]]
1917年2月21日青岛军政署发布军政告示第11号，决定在在青岛旭兵营（原德军[[伊尔蒂斯兵营|伊尔蒂斯兵营]]，今北海舰队司令部）设立青岛中学校。3月末，在II号营房举行入学考试，4月4日开学，暂借II号营房作为校舍。7月5日，日本中学校成为日本[[文部省|文部省]]在外指定学校<ref name=ri>{{cite web |url=http://www.weibo.com/u/1404199805?profile_ftype=1&is_all=1&is_search=1&key_word=日本中学#1491049133646|title=日本中学}}</ref>。当时，有学生110人，日籍教职员工28人，学制6年，设修身、日语、数学、物理、化学、矿物、历史、地理、第一外国语、第二外国语、图画、唱歌、体操等学科。1920年3月5日，当局斥资45万日元在原炮兵营建造日本中学新校舍，由青岛民政署土木课建筑师[[三上贞|三上贞]]设计，华商[[公和兴工程局|公和兴工程局]]承建，1921年6月21日建成<ref name=ta/>，同月末完成了学校的搬迁，7月，寄宿公寓也完成并进行了搬迁<ref Name=ben>{{cite web |url=http://www.qdda.gov.cn/expertLibrary.do?method=viewingArticle&articleId=13559019741192144001&subject_id=12259376220001761001|title=日本第一次占领青岛时期的移民潮|accessdate=2012-12-19|work=孙保锋|publisher=青岛档案信息网}}</ref>。1922年[[北洋政府|北洋政府]]收回青岛后，该校改由日本居留民团承办<ref Name=yi>{{cite web |url=http://arc.dailyqd.com/2015-03/30/content_231670_all.htm|title=1920年代的青岛日本学校|accessdate=2015-03-30|publisher=青岛日报}}</ref>。

===抗战之后===
[[File:国立山东大学复校.jpg|thumb]]
1945年青岛日本中学校解散，校舍改为驻青美军兵营。在[[司徒雷登|司徒雷登]]的斡旋下，[[中国海洋大学#国立山东（青岛）大学时期（1930年－1958年）|国立山东大学]]收回了位于原德军俾斯麦兵营的原有校舍，并将原日本中学校舍一并接收<ref name=ri/>。1946年10月，国立山东大学在青岛复校。自此原日本中学校舍随国立山大与后来的山东海洋学院的变迁而屡经变迁，现为[[中国海洋大学|中国海洋大学]]鱼山路校区的一部分。

==建筑特色==
[[File:青岛日本中学校旧址卫星地图图解.jpg|thumb]]
日本中学校舍由两栋主楼及其他附属建筑组成。大门朝向西南，临有明町（今鱼山路），由两个巨大的花岗岩石柱组成。

南侧主楼平面呈“E”形，共两层，附带坡屋顶阁楼，花岗岩剁斧石砌成墙基，灰白色墙面上有波浪纹，屋顶铺牛舌瓦。主入口设置在西南面（正立面）中央，附有由大理石柱撑起的拱形门廊，檐口起山墙，山墙后侧有一巨大的塔楼，塔楼上覆红瓦四坡屋顶。墙面窗户为长条形，二至四扇为一组，饰以大块花岗岩，形成遍布外立面的竖直线条。屋顶开有波浪形老虎窗。楼内空间宽敞，水磨石地面，楼梯以条石砌成，附设石砌楼梯柱及铁制扶手<ref name=ta/>。

北侧主楼平面呈一字型，两层附带坡屋顶阁楼，建筑材料与外立面装饰与南侧主楼基本相同。主入口设在偏西侧，西端起山花，有一尖顶塔楼。楼内部楼梯为木质。主楼西北侧的洼地设有操场。

这座砖木结构的大楼，设有室内暖气设备和完善的水流式下水道，在当时的日本国内也很少有这样宏伟的学校建筑。1935年被任命为日本中学校校长的岸辰雄教曾这样描述校舍：“看到青岛中学的第一眼就让我吃了一大惊。富丽堂皇的石造校舍。完善的配套设施，校内各角落甚至都用上了蒸汽供暖设施和让人羡慕的温室。就是在国内都没有设施如此完备的中学……” <ref Name=yi/><ref name=ri/>《青岛概要》称其“外观之宏伟，为我国中学中极为罕见，各种设备和内容也非常完善”<ref Name=ben/>。

约1940年代，南侧主楼西北翼向北扩建一大段。约相同年代，南侧主楼的塔楼顶部四坡屋顶被拆除。1949年后，南侧主楼被命名为“六二楼”，以纪念[[青岛解放|青岛解放]]，北侧主楼被命名为“胜利楼”，名称沿用至今。两栋教学楼除室内小部分改造外，基本保存完好。原操场处现为[[中国海洋大学|海大]]篮球场。大门处的两个石砌门柱已成为海大的标志，在海大的新建校区的大门处均仿建有两个石柱。各校区的许多新教学楼都模仿了原日本中学校教学楼的外立面装饰。
