'''青山靈安尊王'''，簡稱'''青山王'''、'''靈安尊王'''，是[[福建|福建]][[泉州三邑|泉州三邑]][[惠安縣|惠安縣]]青山的守護神，除了原有的[[山神|山神]]、行政神的神格，也頗具司法神的職能，相當於[[泉州三邑|三邑]]的[[城隍爺|城隍爺]]<ref>[http://www.ktw02.org/~ktw/religion/40.htm  蔡百銓 《定居衝突期與商業發展期族群守護神與關公崇拜》]{{dead link|date=十一月 2017 |bot=InternetArchiveBot }}</ref>，故青山王配祀有[[判官_(神祇)|判官]]、[[陰陽司|陰陽司]]等諸司[[幕僚|幕僚]]、[[謝范將軍|謝范將軍]]等神，也常常有出巡、[[暗訪_(民俗)|暗訪]]等活動，主要目的就是明察轄區善惡，緝捕惡鬼凶神。

[[日本|日本]][[學者|學者]][[鈴木清一郎|鈴木清一郎]]則在《臺灣舊慣冠婚葬祭與年中行事》一書中指出青山王有[[代天巡狩|代天巡狩]]的職能，與[[王爺信仰|王爺信仰]]有關。隨著[[泉州三邑|泉州三邑]]移民在[[臺灣|臺灣]]的開墾，臺灣的青山王信仰也日益擴大，其最有名的廟宇是[[臺北市|臺北市]]的[[艋舺青山宮|艋舺青山宮]]。

== 源由 ==
此神源由，有兩種說法。

其一：[[東漢|東漢]]、[[三國|三國]]時代[[孫吳|孫吳]]將領[[張滾|張滾]]。[[建安_(東漢)|建安]]十七年（西元212年），張滾奉派駐守[[泉州|泉州]][[惠安|惠安]]地區，頗有治績，故鄉民奉之為神稱「青山王」。[[明代|明代]][[何喬遠|何喬遠]]的《[[閩書|閩書]]‧惠安縣觀應篇》主此說，認為「青山王」是孫吳的將軍「張滾」。

其二：[[五代十國|五代十國]]時，「[[閩國|閩國]]」的[[將軍_(東亞古代)|將軍]]張悃，奉命鎮守惠安一帶，防備[[海盜|海盜]]。軍紀嚴明、愛民如子，逝世之後屢屢顯靈，因在青山附近建廟祭祀，故尊稱其為「青山王」。[[明世宗|明世宗]][[嘉靖|嘉靖]]時編纂之《惠安縣志》主此說：「青山在縣南，閩時將軍張悃，嘗立寨於此以御海寇。既歿，鄉人廟而祀之，至今不廢。」

== 傳奇 ==

傳說張將軍原葬在惠安縣[[衙門|衙門]]附近，惠安縣每有新官上任，皆前往祭之。[[宋太宗|宋太宗]][[太平興國|太平興國]]年間，[[縣令|縣令]]崔知節上任，原不去參拜，後因民意，不得已而前往祭拜時，墓碑突然傾倒，才看到墓碑竟刻有四句詩：「太平興國間，古縣本惠安；今逢崔知節，送我上青山。」崔大驚，並將其墓遷往青山山下（今福建省惠安縣[[山霞鎮|山霞鎮]]青山東麓），建「[[惠安青山宮|青山宮]]」供奉之，從此被鄉民稱為「青山王」，被視為青山的[[山神|山神]]與地方的[[境主|境主]]<ref>[http://www.rhythmsmonthly.com/?p=35585 【歲時風土紀】迎青山王 看顧艋舺百餘載]</ref>。

在[[臺北|臺北]][[艋舺|艋舺]]（今[[臺北市|臺北市]][[萬華區|萬華區]]），又傳說：[[臺灣日治時期|臺灣日治時期]]艋舺青山宮附近有些[[妓院|聲色場所]]，每次華燈初上，正是青樓開市良辰，有些風塵女郎時常出外招攬客人，而青山宮配祀的[[謝范將軍|謝范將軍]]就在此時顯靈，於附近頻頻現身，嚇得那些風塵女郎紛紛避走。

== 敕封 ==

*[[宋高宗|宋高宗]][[建炎|建炎]]年間，金兵軍隊入侵，是為[[采石之戰_(1161年)|釆石磯之役]]。張將軍顯靈助戰，[[江淮|江淮]][[都督|都督]][[虞允文|虞允文]]奏稟朝廷，因此被加封為「'''靈惠侯'''」。夫人華氏也加封為「'''昌順夫人'''」。

*[[宋端宗|宋端宗]][[景炎|景炎]]年間，加封張悃為「'''靈安王'''」，其夫人封「'''顯慶妃'''」。
