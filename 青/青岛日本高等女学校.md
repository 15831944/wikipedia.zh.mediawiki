'''青岛日本高等女学校旧址'''位于中国[[山东省|山东省]][[青岛市|青岛市]][[市北区|市北区]][[黄台路|黄台路]]10号，始建于1916年，是青岛第一次日据时期（1914-1922）所建设的几座日本学校建筑之一。主教学楼已不存在，宿舍楼保存至今，为青岛市文物保护单位。

==历史==
[[File:青岛日本高等女校_06.jpg|thumb]]
1916年4月15日，青岛高等女学校开学，为日本[[文部省|文部省]]在外指定学校。该校专门招收日本子女，学制3年。校址最初设在原[[德华大学|德华大学]]校舍内，原德军野战炮兵营的长营房内，位于三笠町（今黄台路）的新校舍于次年9月22日开始建设，建设耗资144970银元。1918年5月31日，新教学楼建成，学校于11月19日搬迁至新址。1920年，宿舍楼建成。建校初的课程为修身（伦理）、国语（日本语）、汉文、习字、数学、物理、化学、矿物、第一外国语（英语）、第二外国语（法语）、图画、唱歌、体操、家事等；后增加实习与缝纫讲座等课，学制也由之前的三年改为五年。[[北洋政府|北洋政府]]收回青岛后不久的1923年10月4日，学校改称青岛日本高等女学校。1928年时学校有学生368人<ref Name=ben>{{cite web |url=http://www.qdda.gov.cn/expertLibrary.do?method=viewingArticle&articleId=13559019741192144001&subject_id=12259376220001761001|title=日本第一次占领青岛时期的移民潮|accessdate=2012-12-19|work=孙保锋|publisher=青岛档案信息网}}</ref><ref>{{cite web |url=http://arc.dailyqd.com/2015-04/14/content_241136_all.htm|title=盘点日本第一次占领青岛时所建小学校|accessdate=2015-04-14|publisher=青岛日报}}</ref>。有照片资料显示教学楼曾为“临时派遣第十师团司令部”。

1945年日本投降，日侨回国，学校停办。原教学楼于同年失火导致屋顶烧毁，据说可能是日本人纵火所致<ref Name=qing>{{cite web |url=http://club.qingdaonews.com/touch/show_57_3838139_1_0.htm|title=黄台路百年老建筑——日本高等女校旧址}}</ref>。1946年，国民党[[绥靖区#第二绥靖区|第二绥靖区]]司令长官兼山东省政府主席[[王耀武|王耀武]]占用该校校舍，并于1948年建立了以[[蒋中正|蒋介石]]的字命名的中正中学，自任为学校董事长。

解放军于1949年6月2日接管青岛后，原在[[潍县|潍县]]解放区的华东大学于1950年进驻该校校舍。同年，华东大学为参与筹建[[山东师范大学|山东师范学院]]而迁往[[济南|济南]]，艺术系仍留在青岛。1952年，原艺术系迁至上海并入华东戏剧专科学校（今[[上海戏剧学院|上海戏剧学院]]）。同年，由多所大学的土木工程等科系组建而成的[[青岛工学院_(1952年)|青岛工学院]]将日本女校旧址作为校址使用。1956年，青岛工学院重新整合迁往武汉。同年8月，山东大学医学院从[[山东大学|山东大学]]脱离，改为[[青岛医学院|青岛医学院]]，黄台路原日本女校校舍成为青岛医学院黄台路本部。1993年，原青岛医学院并入[[青岛大学|青岛大学]]，改称青岛大学医学院。1996年，原教学楼拆除用于商业开发，原址于2003年建设都市华庭高层住宅小区。几年后，青岛大学医学院黄台路校区取消并迁往其他校区，原日本女校宿舍楼在闲置一段时间后被改为商业用途，现同时为一家旅馆及一些私立艺术辅导班所使用<ref Name=qing/>。

==建筑特色==
[[File:青岛日本高等女校_04.jpg|thumb]]
女校校舍位于三笠町（今[[黄台路|黄台路]]）北侧靠近大和町（今[[辽宁路_(青岛)|辽宁路]]）的位置，邻近若鹤山（今[[贮水山|贮水山]]），由西侧的教学楼和东侧的宿舍楼组成。西侧教学楼正立面呈“横三纵五”的对称布局，上下两层附设半地下室和坡屋顶阁楼，花岗岩剁斧石砌墙基。墙面窗户与其他日本建筑类似，形状细长，二至三扇为一组，一楼窗户为矩形，以花岗岩装饰，二楼窗户顶端为拱券式。屋顶设有三角形老虎窗，中央有一座尖顶塔楼。立面中央与两端的檐口处起三角形山墙。外立面整体风格偏向于[[哥特复兴式|哥特复兴式]]。

较前者晚一点建成的宿舍楼则被认为模仿了德国建筑的风格。建筑两层加坡屋顶阁楼，正立面布局不对称，中央起山墙和塔楼，细节装饰模仿了西式桁架结构，墙面以瓷砖和花岗岩装饰，窗户样式仍为细长形，以花岗岩条石装饰。内部地面为水磨石，主楼梯以大块石料砌成，楼梯柱亦为石制，扶手为铁制，带有花纹。副楼梯为木质结构<ref Name=qing/><ref name=ta>{{cite book |author=李明 |title=《塔楼上的青岛》|year=2008 |publisher=青岛出版社 |location=青岛 |isbn=9787543649705}}</ref>。

1945年后，原日本女校教学楼屋顶因失火被烧毁，未能恢复原貌。该楼于约1996年被拆除。原宿舍楼内部因商业用途被部分改造。

校舍的设计者不详，图纸现藏于[[早稻田大学|早稻田大学]]图书馆。校舍中原存有[[王耀武|王耀武]]所书“国民政府主席蒋公六秩诞辰献校纪念”石碑，青岛大学医学院取消黄台路校区后，将该碑移至[[青岛日本第三国民学校|松山路校区]]存放。
