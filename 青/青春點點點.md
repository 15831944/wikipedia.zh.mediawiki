《'''青春點點點'''》是一檔[[台灣|台灣]]{{link-ja|電台節目|ラジオ番組|廣播節目}}，於[[台北|台北]][[飛碟電台|飛碟電台]]（FM 92.1）播放，2004年10月17日首播，2007年1月23日在原本塊狀節目之外新增帶狀節目，自開播至今，深受[[學生|學生族群]]的歡迎，也有不少[[上班族|上班族]]和[[夜貓子|夜貓]][[聽眾|聽眾]]。

2017年2月19日為週日《青春點點點》最後一集播出。3月7日為平日《青春點點點》最後一集播出。節目結束後，將「馬克信箱」的傳統延續至《MARC ORANGE》YOUTUBE平台，每週更新一次。<ref>{{cite web |author1=MARC ORANGE |title=馬克信箱 |url=https://www.youtube.com/user/UFOMarc1}}</ref>節目結束後飛碟電台曾經請馬克、瑪麗分開主持平日下午兩點到四點《哥哥妹妹有意思》節目，並於2018年3月5日起正式合體主持。

*2001年夏天參加飛碟DJ徵選，以18歲之姿成爲當時該電台最年輕的DJ。
*跟中天主播劉涵竹為大學同學。
*在尚未露臉前以38歲大叔身分走跳，並曾經被形容長得像[[曾文鼎|曾文鼎]]、[[幽遊白書|幽遊白書]]的桑原，近期則是台版[[小籔千豐|小籔千豐]]。
*2012年7月7日，廣播電台創舉，同一節目主持人吳瑪麗因整天只吃一個麵包，主持人歐馬克於現場節目，離開所屬大樓步行至鄰近區域購買飯糰和溫豆漿給瑪麗，自己點了份蛋餅夾油條；同時間現場手機連線現場，還被聽眾美男子開車接回電台，並且沒有跳錶。
*2013年5月14日，創造新的英文縮寫BEY，其意同不意外。
*2014年1月3日，透露「小時候」喜歡大膝蓋水蜜桃姊姊(朱安禹)，以及蒐集很多關於李心潔的周邊商品。
*2015年2月，上網路廣播[[青春愛消遣|青春愛消遣]]當特別來賓，表明自己想在雞排妹身上做科學七分鐘，並表示女友和瑪麗同時落水不會救瑪麗，讓瑪麗相當心寒。<ref>{{cite web |author1=青春愛消遣 |title=青春愛消遣第45集「青春大來賓」(飛碟電台青春點點點DJ歐馬克) |url=http://youngloveplay.blogspot.com/search?q=%E6%AD%90%E9%A6%AC%E5%85%8B&max-results=20&by-date=true}}</ref>
*2017青春點點點節目停播後，轉型為YouTuber，曾與[[阿滴|阿滴]]、[[小熱唱|小熱唱]]、[[黃阿瑪|黃阿瑪]]、[[嘎老師|嘎老師]]等YouTuber一起錄製短片。
*2018年3月21日，在老婆Katia生日當天前往大安戶政事務所登記結婚。

== 青春點點點大事記 ==

*2013年12月14日，於西門河岸留言舉行「精神損害演唱會」，是第一次與聽眾同樂的售票活動，也是馬克第一次半露臉出現於聽眾面前。<ref>{{cite web |author1=MARC ORANGE |title=青春點點點 精神損害 演唱會 |url=https://www.youtube.com/watch?v=L1YiO6HQnUs}}</ref>
*2014年8月2日，於新北市政府舉辦「馬克瑪麗脫口秀」，門票一掃而空，並現場邀聽眾示範科學七分鐘。<ref>{{cite web |author1=楊立範 |title=青春點點點-馬克瑪麗脫口秀---科學七分鐘 |url=https://www.youtube.com/watch?v=aToZ1phoI7c}}</ref>
*2015年8月10日，發行新書《為青春出發：馬克瑪麗的歐遊點點點》，紀錄二人的歐洲行，並在站前金石堂舉行簽書會。<ref>{{cite book |author1=馬克, 瑪麗 |title=為青春出發：馬克瑪麗的歐遊點點點 |date=2015-08-10 |publisher=啟動文化 |isbn=9789869166072}}</ref>
*2016年於台北車站舉行聯誼活動，以16型人格理論出發，沒有配對成功。
*2016年11月20日，舉辦電影「腸腸搞轟趴」版聚，卻沒意料到電影會這麼難看。
*2017年3月7日，節目停播，當夜凌晨聚集數百聽眾於飛碟電台樓下，進行現場「畢業典禮」，氣氛感人。<ref>{{cite news |author1=三立新聞娛樂中心 |title=《青春點點點》走入歷史！點友聚集電台　舉布條大合唱淚送 |url=https://www.setn.com/News.aspx?NewsID=230997 |date=2017-03-07}}</ref>
*2017年11月25日，舉行三場「瑪麗電影院」脫口秀，由兩位DJ現場描述爛電影情節。<ref>{{cite web |author1=ACCUPASS |title=瑪麗電影院 |url=https://www.accupass.com/event/1711030725442762521770}}</ref>
*2018年3月18日，於花博園區舉行青春點點點逝世一週年追思紀念會，並呈現登基大典、未成年主張以及現場版馬克信箱。<ref>{{cite web |author1=MARC ORANGE |title=青春點點點追思紀念會 - 馬克信箱 |url=https://www.youtube.com/watch?v=v_TrL0tQ0jg}}</ref>
*2018年11月25日，舉行預立遺囑工作坊，邀請專業律師教導如何預立遺囑，活動盈餘全數捐獻。<ref>{{cite web |author1=ACCUPASS |title=預立遺囑工作坊 |url=https://www.accupass.com/event/1811051937164400419600}}</ref>
*2019年3月31日，於花博流行館舉辦青春點點點逝世二週年追思紀念會，邀請點友攜帶不要的舊衣配件參與Fashion Day。現場貴族決定以星座方式分組對決，但由於最後時間不足，無法選出第一名。<ref>{{cite web |author1=MARC ORANGE |title=青春點點點追思紀念會 - Fashion Day|url=https://www.accupass.com/event/1903050306581863039755}}</ref>

==爭議==
*瑪麗曾在節目中說出「想自殺的人應該是經過深思熟慮，不用阻止他們」、「不需讓座給老年人，也許他們不覺得自己老」等言論，遭到聽眾抨擊。<ref name="聯合報">{{cite news|author =江祥綾|url =http://i.imgur.com/S4NLuwi.jpg|title =飛碟主持言論挨轟|publisher =聯合報|pages =D4 星聞e把罩|date =2007-05-30|archiveurl =https://archive.is/20161213130220/http://i.imgur.com/S4NLuwi.jpg|archivedate =2016-12-13|deadurl =no}}</ref>

*馬克曾在節目中批評一位發片男歌手的作品「難聽」，引起該唱片公司不滿，後經飛碟電台總經理向唱片公司及歌手道歉，並播放該歌手的作品，才平息紛爭。<ref name="聯合報"></ref>
