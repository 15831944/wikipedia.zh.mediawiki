《'''青春失乐园'''》（又名《'''青春期Ⅱ'''》；{{lang-en|'''''Paradise Lost'''''}}）2011年电影《[[青春期_(2011年电影)|青春期]]》的续作。该片于2011年11月11日在[[中国大陆|中国大陆]]上映。<ref>{{cite web |url=http://ent.ifeng.com/zz/detail_2011_12/26/11576712_0.shtml|title=《青春失乐园 影迷狂欢夜 歌友会唱响圣诞》|accessdate=2011年12月26日|work=|publisher=凤凰网|language= zh-hans}}</ref>

== 剧情 ==
本篇銜接第一集[[青春期|青春期]]主要描述程小雨和王小菲及[[90后|90后]]的5个[[大学生|大学生]]和他们的两个大学[[老师|老师]]之间的[[爱情|爱情]]故事。<ref name="ab " />

== 主题 ==
《青春失乐园》承接第一部《[[青春期_(2011年電影)|青春期]]》，继续对年轻人的性态度进行描绘和探究，该片中年轻人对性的表现更加成熟。<ref name="ab ">{{cite web |url=http://ent.ifeng.com/zz/detail_2011_11/09/10529856_0.shtml|title=电影《青春失乐园》首映 打造中国版美国派|accessdate=2011年11月9日|work=|publisher=凤凰网|language=zh-hans}}</ref>
