== 历史 ==
青年奥林匹克公园是为了建立培养更强烈的社区感并连接[[濱海灣_(新加坡)|滨海湾]]而建造的。公园在首届[[青年奥林匹克运动会|青年奥林匹克运动会]]，也就是[[2010年夏季青年奧林匹克運動會|2010年夏季青年奥运会]]举办后被命名。2010年7月，为了深入改善环境，公园安装了以奥运会为主题的艺术设施。

== 今天 ==
该公园提供可供休息的阴凉角落，并提供了一个良好的摄影眺望点。有一个木板行人道连接公园和[[滨海宝龙坊|滨海宝龙坊]]，并且有螺旋桥连接公园和[[濱海灣_(新加坡)|滨海湾]][[金沙|金沙综合度假胜地]]。

== 旅游景点 ==
著名旅游景点之一是通向螺旋桥入口的大石阶，在这里游客可以读到阿曼达·崇所作的名为《狮心》的诗篇。

另一个游客可以参观的景点是风景优美的迷宫，这是由安尼科特·恰范的作品中发展而来的。在迷宫的中央，参观者可以看到一个4.5米高的“山”，是受到乔伊斯·谢的画作启发的雕塑。还有一个石墙，上面由青少年画上例如[[鱼尾狮|鱼尾狮]]、海湾滨海艺术中心剧院和[[组屋|组屋]]等[[新加坡|新加坡]]最著名的[[地標|地标]]。

“情绪光景”是由肯尼斯·谭先生提出的概念。这是一个互动式的“跳房子”式的板砖，当踩上去时它就会发光。

还有一个有趣的设施是由Lui Honfay设计的“色带椅”。
