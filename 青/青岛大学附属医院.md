'''青岛大学附属医院'''是位于[[中华人民共和国|中国]][[青岛市|青岛市]]的一所[[三级甲等|三级甲等]]综合性医院，前身为1946年建立的[[国立山东大学|国立山东大学]]附属医院<ref>{{Cite news|url=http://www.dzwww.com/shandong/sdnews/201504/t20150414_12217951.htm|title=青医附院前身督署医院变迁史：日德战争期间成炮轰的目标|author=|date=2015-04-14|work=大众网|accessdate=2017-05-04}}</ref>，后另经多次名称和隶属更迭于2013年变更至现名<ref>{{Cite news|url=http://news.qingdaonews.com/qingdao/2013-10/31/content_10069606.htm|title=青医附院更名青岛大学附属医院 启用科教大楼|author=|date=2013-10-31|work=青岛日报|accessdate=2017-05-04}}</ref>。青岛大学附属医院医资力量雄厚<ref>{{Cite news|url=http://qingdao.iqilu.com/qdyaowen/2017/0309/3433980.shtml|title=中国大医院门诊量百强榜出炉 青大附院列第13位|author=|date=2017-03-09|work=青岛日报|accessdate=2017-05-04}}</ref><ref>{{Cite news|url=http://dongying.dzwww.com/dubao/201506/t20150616_12560824.html|title=我省科研团队发现3个新痛风易感基因|author=李振|date=2015-06-16|work=大众日报|accessdate=2017-05-04}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，重视信息化建设，是山东省省内首家支持[[支付宝|支付宝]]支付的医院<ref>{{Cite news|url=http://www.dailyqd.com/2014-12/30/content_192042.htm|title=取号缴费不用排长队 青大附院用上支付宝|author=王娉|date=2014-12-30|work=青岛日报|accessdate=2017-05-04}}</ref>。医院现有黄岛、市南、市北、崂山四个院区。
== 市南院区 ==
青岛大学附属医院市南院区（本部）位于青岛市[[市南区|市南区]][[江苏路_(青岛)|江苏路]]16号，是医院总部所在地，负责各个院区的统筹管理工作。该院址最初为[[青岛督署医院|督署医院]]的所在地。
== 崂山院区 ==
青岛大学附属医院崂山院区（东院区）位于青岛市[[崂山区|崂山区]]海尔路59号，是青岛大学附属医院于2006年12月全资收购的青岛万杰医院改制而成，于2007年正式开业<ref>{{Cite news|url=http://epaper.qingdaonews.com/html/qdrb/20150812/qdrb897155.html|title=青大附院东区继续扩建|author=钱卓|date=2015-08-12|work=青岛日报|accessdate=2017-05-04}}</ref>。
== 黄岛院区 ==
青岛大学附属医院黄岛院区位于青岛市[[黄岛区|黄岛区]]五台山路1677号，也称“西海岸医院”或“西海岸医疗中心”，于2011年正式建成投入使用<ref>{{Cite news|url=http://news.163.com/11/0711/12/78MBC6N300014AED.html|title=青医附院黄岛分院开业|author=杨林|date=2011-07-11|work=齐鲁晚报|accessdate=2017-05-05}}</ref>。

== 市北院区 ==
青岛大学附属医院市北院区位于青岛市[[市北区|市北区]]嘉兴路7号，于2014年7月正式开业，原址是四方机厂医院<ref>{{Cite news|url=http://news.qd.fang.com/2014-02-20/12140336.htm|title=青大附院北院区7月开放 位于四方机厂医院旧址|author=杨林|date=2014-02-20|work=齐鲁晚报|accessdate=2017-05-05}}</ref>。
