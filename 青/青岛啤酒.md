'''青岛啤酒'''（{{hkex|168}}、{{sse|600600}}、{{OTC|TSGTY}}）是[[中国|中国]]历史最悠久的[[啤酒|啤酒]][[品牌|品牌]]之一，创始于1903年。

== 历史 ==
1903年，由来自[[德国|德国]]的商人联合投资40万[[德國馬克|德國馬克]]在[[青島|青島]]成立了日耳曼啤酒公司青岛股份公司，采用德国的酿造技术以及原料，號稱使用[[嶗山|嶗山]]（位于青島東部）[[礦泉水|礦泉水]]釀造，在中国生产德国风味的啤酒，主要供应一些在中国的德国人和另外一些西方人。

1914年，[[第一次世界大战|第一次世界大战]]爆发以后，[[日本|日本]]侵占青岛。1916年，日本人侵占了德国商人的股份，买下英国商人的股份，将公司更名为“大日本麦酒株式会社青岛工场”。生产品牌除原青岛啤酒外，也生产日本大麦酒旗下品牌[[朝日啤酒|朝日啤酒]]。
[[File:Chinese_New_Year.jpg|缩略图]]
1945年随着[[日本|日本]]的战败，[[中華民國政府|中華民國政府]]成立[[行政院|行政院]]收復區全國性事業接收委員會，沒收[[日本|日本]]在[[中華民國|中國]]之資產，於是工厂由[[青岛市政府|青岛市政府]]当局派员接管，更名为“青岛啤酒公司”，隸屬於[[行政院|行政院]]山東青島區產業處理局。1947年6月14日，由[[中國國民黨|中國國民黨]]黨營企業“[[齐鲁企业|齐鲁企业]]股份有限公司”购买，成為“齊魯企業青島啤酒廠”。1949年，中华人民共和国的成立，青岛啤酒被[[中國共产党|中國共产党]]没收，更名为“国营青岛啤酒厂”。

从1972年进入美国市场，不久就成为在美国市场上最畅销的中国啤酒，打破了美国380种进口啤酒销售量增长速度的纪录{{Fact|time=2018-09-23T15:22:27+00:00}}，并3次在美国华盛顿国际评酒会荣获冠军{{Fact|time=2018-09-23T15:22:27+00:00}}。现在青岛啤酒销往全世界70多个国家，一直占据着全国啤酒出口量的50％以上。

90年代初，通过与另外3个工厂的联合，成立了股份制公司，青岛啤酒厂也更名为青岛啤酒股份有限公司，现在27%的股份被美国的[[安海斯-布希|安海斯-布希]]公司持有，2008年[[英博集團|英博集團]]購併安海斯-布希，中國商務部反壟斷局於11月18日决定附条件批准購併[http://www.mofcom.gov.cn/aarticle/ae/ai/200811/20081105899250.html 商务部公告英博公司收购AB公司案反垄断审查的决定]。

1993年7月，青岛啤酒H股在香港上市，是中国第一家内地在港发行H股上市的企业，同批赴港上市的还有[[上海石化|上海石化]]等共9家。

1998年，青岛啤酒实施“大名牌发展战略”，率先在全国开展大规模兼并扩张，截至2007年9月,已在全国18个省市拥有或者绝对控股71家分公司 [http://www.sse.com.cn/pv_obj_cache/pv_obj_id_C3F87DDCF52BA89BEBD3C7EDF61E4A8ABEC43A00/filename/600600_20080331_3.pdf]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}。

2002年，海峽兩岸加入[[世界貿易組織|世界貿易組織]]（WTO）之後，台灣開放大陸產酒類進口，台灣的飲料大廠[[三洋維士比|三洋維士比]]集團轉投資之台灣青啤股份有限公司取得銷售代理權，青岛啤酒在台灣上市<ref name="ttbc">[http://www.ttbc.com.tw/about/ 關於青啤]</ref>。

2002年10月21日，美国A-B公司和青啤公司在美国纽约正式签署《战略性投资协议》，A-B公司成为青啤公司战略投资者。

2005年，台灣青啤股份有限公司取得商標授權後，在[[屏東縣|屏東縣]][[內埔鄉|內埔鄉]]設立觀光酒廠，青岛啤酒在台灣生產<ref name="ttbc" />。

2005年8月11日，青岛啤酒与北京奥组委签约，成为[[北京奧運|北京奧運]]国内啤酒赞助商。

2009年4月30日百威英博（AB InBev），以每股19.78港元，向日本朝日啤酒出售青岛啤酒19.9%權益，套現6.665億美元（折合港幣51.69億元），朝日成為青啤第二大股東，這是日本公司首次成為中國大公司的大股東<ref>{{cite web |url=http://finance.sina.com.cn/chanjing/gsnews/20090506/20436191237.shtml|title=朝日成青岛啤酒第二大股东|accessdate=2013-07-18|work=|publisher=中国广播网}}</ref><ref>{{cite web |url=http://finance.sina.com.cn/chanjing/gsnews/20090507/10586194664.shtml|title=青啤澄清控股权传闻 称朝日持股19.9%无威胁|accessdate=2013-07-18|work=|publisher=新浪财经}}</ref>。ABInbev 則將保留7.01％的股權。

2009年5月11日百威英博啤酒全資附屬公司A-BJade以每股19.83港元的价格将青岛啤酒的約占已發行總股本的7.01%及已發行H股股份總數的13.99%)，股權出售给[[紫金矿业|紫金矿业]]董事长[[陈发树|陈发树]]，交易金额为2.35亿美元。

2010年12月7日青島啤酒公司與香港培新集團在濟南簽署協議，青島啤酒以18.7亿收購由香港培新集團持有的銀麥啤酒100%股權。

2012年6月5日青島啤酒與日資三得利簽訂合作框架協議，雙方將各自在上海及江蘇省的資產和業務整合到松江公司，青啤透過股權轉讓、增資等方式向「事業合資公司」注入50%股權，涉資13.36億元（人民幣．下同），三得利亦會向其增資13.52億元換取餘下50%股權；同時青啤與三得利各斥1000萬元成立「銷售合資公司」

2017年12月20日，[[復星國際|復星國際]]透過旗下多家附屬公司, 以66.17億港元, 每股作價27.22港元,向日本[[朝日啤酒|朝日啤酒]](Asahi), 收購[[青島啤酒|青島啤酒]]2億4311萬股H股, 交易股份佔青島啤酒37.11%H股已發行股份，相等於總發行股數的17.99%。

2017年12月20日，[[朝日啤酒|朝日啤酒]]與青啤集團及鑫海盛簽署股份購買協議，將其所持公司2702萬股H股(約占公司總股本1.99%)轉讓給鑫海盛，總作價7.35億港元。

== 品種 ==
青岛啤酒目前在市场上的种类为[[拉格啤酒|拉格啤酒]]，其主要成分为[[大米|大米]]、[[麦芽|麦芽]]、[[啤酒花|啤酒花]]以及其他添加剂。有很多个品种。

== 所獲獎項 ==
* 1906年，[[慕尼黑|慕尼黑]]博览会上展出，获得金牌奖
* 1981年、1985年两次，[[美国|美国]][[华盛顿|华盛顿]]举行的国际评酒会上获得冠军；
* 1987年5月，美国[[密西西比州|密西西比州]]杰克逊市举行的国际啤酒评比中名列榜首；
* 1991年9月，[[比利时|比利时]][[布鲁塞尔|布鲁塞尔]]举行的蒙顿国际评比大赛上获金质奖。

== 青島國際啤酒節 ==
[[青島國際啤酒節|青島國際啤酒節]]自1991年開始由中國有關部委和青島市人民政府共同主辦。啤酒節於每年8月第2個周末開幕為期16天。由於正值每年青島的黃金旅遊季節，吸引到海內外的人士到場，參予具[[旅遊|旅遊]]、[[文化|文化]]、[[體育|體育]]、經貿於一體的國家級大型節慶活動。

2005年8月13日開幕的第15屆青島國際啤酒節是首次有[[台灣|台灣]]的[[東森電視台|東森電視台]]和大陸[[山东卫视|山東衛視]]合作，將開幕典禮向亞洲、歐美等地區進行同步連線轉播。
