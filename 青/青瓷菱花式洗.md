'''青瓷菱花式洗'''

== 尺寸 ==
*高：4.3公分
*口徑：17.8公分
*足徑：11.2公分

== 陳列地點 ==
[[國立故宮博物院|國立故宮博物院]]203陳列室

== 年代背景 ==
{{fact|青瓷菱花式洗，杭州修內司官窯。南宋為確立上承[[北宋|北宋]]正統地位，及供應皇室祭祀和日常器物需求，在首都[[临安府|臨安]]修建[[官窯|官窯]]。修內司窯成立時間不會早於紹興二十年(1150年)，產出的瓷器繼承北宋[[汝窯|汝窯]]的特點，器色以粉青最常見，再來為月白色}}。[[明代|明代]]鑑賞家[[曹昭|曹昭]]在《格古論要》中提到：官窯器修內司燒者土臃細潤，色青帶粉紅，濃淡不一，有蟹爪紋紫口鐵足，色好者與汝窯相類，有黑土者為之烏泥窯，偽者皆[[龍泉|龍泉]]所燒者，無紋路。<ref>{{cite book |title=四庫全書.子部雜家類.格古論要| author=曹昭}}</ref>

== 外觀 ==
器形侈口平底淺壁，口沿為弧度圓潤細緻的十二瓣花形，可以想見若是[[筆洗|筆洗]]盛水，將如同[[蓮花|蓮花]]在湖中徐徐開放。有別於以往[[浙江|浙江]]窯場釉色偏黃或綠，[[南宋|南宋]]官窯時興偏藍的青色。這件作品也大致呈現天青色，唯有一圈粉色淡淡分布在盤口和足。[[瓷器|瓷器]]表面釉色柔和飽滿，色澤明亮不失溫潤，呼應宋人追求的青瓷精神。若是仔細觀察，器內和器外細密交錯著棕色[[冰裂紋|冰裂紋]]。<ref> {{Cite web |url=http://theme.npm.edu.tw/exh104/Celadons/ch/index.html | title= 貴似晨星-清宮傳世12至14世紀青瓷特展 | author= 國立故宮博物院}}</ref>瓷器表面的釉因為膨脹係數不同在受熱時產生裂痕，美稱冰裂，雖是自然生成亦可以透過染色加深。青瓷花式洗的冰裂紋為斜向開片，近於[[蟹爪紋|蟹爪紋]]。

== 施釉 ==
菱花式洗表面厚施雙層釉，滿釉支燒，底部釉汁較薄處呈棕色，圈足內有七枚支釘。為求釉汁最大範圍地覆蓋器皿表面，匠人只用幾枚泥釘支撐原胎就送進窯內燒製，燒製出的瓷器底部會有小小的支釘痕跡。這種高難度技法已可見於[[汝窯|汝窯]]，[[青瓷|青瓷]]菱花式洗在此顯示出[[南宋|南宋]][[官窯|官窯]]和[[北宋|北宋]]的傳承關係<ref> {{Cite web | url= http://www.npm.gov.tw/exh99/southernsong/ch_03.html | title= 文藝復興-南宋藝術與文化特展 | author= 國立故宮博物院}}</ref><ref>{{Cite web | url = http://theme.npm.edu.tw/exh104/Celadons/ch/page-4.html |title = 貴似晨星-青瓷特展青瓷菱花式洗 | author= 故宮博物院}}</ref><ref>{{cite web|url = http://antiquities.npm.gov.tw/Utensils_Page.aspx?ItemId=310790 |title = 南宋 官窯 青瓷菱花式洗 |accessdate = 2017-11-10 |publisher = 國立故宮博物院 |website = 器物典藏資料檢索系統 |archiveurl = http://antiquities.npm.gov.tw/Utensils_Page.aspx?ItemId=310790 |archivedate = 2017-11-10 |deadurl = no}}</ref>。
