'''青海省'''，简称'''青'''，为[[中国一级行政区|中国一级行政区]]，位于中国西部、[[青藏高原|青藏高原]]东北部。因境内有全中国最大的内陆[[咸水湖|咸水湖]]─[[青海湖|青海湖]]而得名，省会是[[西宁市|西宁市]]。青海是长江、黄河、澜沧江的发源地，故被称为“三江源”。

== 历史 ==
{{main|青海歷史}}

早在2萬年前青海就有人居住，商周時代形成了羌部落，史稱“[[西羌|西羌]]”。據[[商朝|商朝]][[甲骨文|甲骨文]]記載，商高宗[[武丁|武丁]]出兵征伐西羌，佔領了青海東部大片地區。早在西周時期，就與中原地區發生了政治、經濟聯繫。並且與秦朝來往密切。到了漢初，又分裂成分為[[先零|先零]]、[[燒當|燒當]]、[[發羌|發羌]]、[[党項|党項]]等部。

古代青海東部屬於中原[[王朝|王朝]]統治，[[漢朝|漢]]時曾設[[西海郡|西海郡]]、[[河源郡|河源郡]]、[[湟源|湟源郡]]、[[金城郡|金城郡]]，控制了今天的青海[[貴南|貴南]]、[[貴德|貴德]]以及[[西寧|西寧]]和[[湟源|湟源]]等地，還設立護羌校尉。 [[晉朝|兩晉]]時期，[[鮮卑|鮮卑]][[慕容部|慕容部]]的一支率所部1700戶從[[遼東|遼東]][[前燕|前燕]]西遷，建立[[吐谷渾|吐谷渾]]汗國，除了青海以外，還控制了[[甘肅|甘肅]]一些地區，統治了青海足足300年。在[[吐谷渾|吐谷渾]]帝國的南部還有佔據白蘭道數百年的[[丁零人|丁零人]]和古[[羌族|羌族]]的國家[[白蘭國|白蘭國]]。

[[隋朝|隋朝]]時，[[隋煬帝|隋煬帝]]西征伐吐谷渾，在[[大業|大業]]五年（609年）擊潰吐谷渾，吐谷渾首領被生擒，從此隋朝控制了青海大部。[[唐朝|唐朝]]時，[[吐蕃|吐蕃]]滅吐谷渾，佔領了青海地區。青海其他地區則屬古羌。

11世紀初，青海東部擁立吐蕃普約後裔為王，稱[[唃廝囉|唃廝囉]]，號為宗喀王，建都青唐（今西寧）。史稱青唐王國。12世紀初，[[北宋|北宋]]力量到達河湟地區，並將鄯州改為西寧州，於是西寧名稱開始使用。13世紀初，[[成吉思汗|成吉思汗]]率大軍經臨洮進占西寧州，並在河州（今[[臨夏縣|臨夏縣]]）設置元帥府，當時牧區推行土司制度。這時，在[[蒙古帝國|蒙古帝國]]的支持下，[[藏傳佛教|藏傳佛教]]在青海逐漸盛行起來，並建立了許多寺院。[[元朝|元朝]]时期，青海大部归[[吐蕃等處宣慰司都元帥府|吐蕃等處宣慰司都元帥府]]管辖，隶属于[[宣政院|宣政院]]。

青海[[蒙古族|蒙古族]]在[[明朝|明代]]由[[新疆|新疆]]遷來，其首領[[顧實汗|顧實汗]]，牧於青海湖周圍、柴達木盆地及黃河河曲一帶，勢頗強盛。17世紀，顧實汗控制青海，對[[西藏|西藏]]也有強大的影響。[[清朝|清朝]]政府派員冊封顧實汗為遵行文義敏慧顧實汗，使顧實汗在青、藏的統治地位強化。清[[雍正帝|雍正帝]]平息顧實之孫[[和碩親王|和碩親王]][[羅布藏丹津|羅布藏丹津]]後，改西寧衛為府，設西寧縣、碾伯縣和大通衛，属[[甘肃省|甘肃省]]。青海地区由[[西寧辦事大臣|西寧辦事大臣]]直轄管理。這項措施為後來青海建省鋪平了道路。

清末時，[[回族|回族]]商人[[馬海晏|馬海晏]]起兵反清，創立了[[馬家軍|馬家軍]]，後[[歸順|歸順]]清朝，海晏在[[庚子拳亂|庚子拳亂]]，[[八國聯軍|八國聯軍]]襲來時為了保護[[慈禧太后|慈禧太后]]與[[光緒皇帝|光緒皇帝]]，憂勞而卒，朝廷[[蔭襲|蔭襲]]馬海晏之子[[馬麒|馬麒]]為[[總兵|總兵]]。[[中华民国北洋政府|中华民国北洋政府]]時馬麒升任青海蒙番[[宣慰使|宣慰使]]，成為[[軍閥|軍閥]]，逐漸統治青海，馬麒死後，其子[[馬步芳|馬步芳]]勢力更是深入青海，期间青海省于1928年成立。馬家軍在青海統治了四十年，到1949年，随后由[[中华人民共和国|中华人民共和国]]接管。

== 地理 ==
青海位于[[青藏高原|青藏高原]]东北部，全省均属高原范围之内。地形复杂，地貌多样。全省平均海拔3000多米，最高点昆仑山的布喀达板峰为6860米，最低点在民和下川口村，海拔为1650米。境内山脉高耸，地形多样，河流纵横，湖泊棋布。巍巍昆仑山横贯中部，唐古拉山峙立于南，祁连山矗立于北，茫茫草原起伏绵延，柴达木盆地浩瀚无限。[[长江|长江]]、[[黄河|黄河]]及[[澜沧江|澜沧江]]（[[湄公河|湄公河]]）（因此被称作“三江源”）之源头在青海。[[巴颜喀拉山|巴颜喀拉山]]为长江和黄河的分水岭。

青海属于高原大陆性气候，有太阳辐射强、光照充足；平均气温低，在零下5.7℃～8.5℃之间；降水量少，绝大部分地区年降水量在400毫米以下；地域差异大；雨热同期等特点。

接邻省区有[[四川省|四川省]]、[[甘肃省|甘肃省]]、[[新疆维吾尔自治区|新疆维吾尔自治区]]、[[西藏自治区|西藏自治区]]。

2010年末全省总人口共5,626,722人。汉族人口共2,983,516人，占53.02％；少数民族共2,643,206人，占46.98％，这一比重仅低于西藏和新疆，居全国第三位。截止到2017年末，青海省的总人口为598万人<ref>{{cite web |title=中国统计年鉴-2018 |url=http://www.stats.gov.cn/tjsj/ndsj/2018/indexch.htm |publisher=国家统计局}}</ref>。少数民族：

* [[藏族|藏族]]，共1,375,062人，占全省总人口的24.44%，主要分布在玉树、果洛、海南、黄南、海北五个藏族自治州和海西蒙古族藏族自治州，西宁市的大通县，平安县和海东市也有部分藏族居住。[[青海藏族|青海藏族]]是在长期历史发展中由[[吐蕃|吐蕃]]吸收青藏高原各民族逐渐发展形成的。藏族语言属[[汉藏语系|汉藏语系]][[藏缅语族|藏缅语族]]，[[青海藏族|青海藏族]]除玉树部分人讲[[康巴|康巴]]方言外，均操[[安多|安多]]方言，通用[[藏语文|藏语文]]，信奉[[藏传佛教|藏传佛教]]。
* [[回族|回族]]，共834,298人，占全省总人口的14.83%。主要分布在青海东部地区。
* [[土族|土族]]，共204,413人，佔全省總人口的3.63%。
* [[撒拉族|撒拉族]]，共107,089人，佔全省總人口的1.9%。
* [[蒙古族|蒙古族]]，共99,815人，佔全省總人口的1.77%。
* 其他少數民族，共22,529人，佔全省總人口的0.4%。

===能源===
青海有[[石油|石油]], 主要在[[柴达木盆地|柴达木盆地]]。柴达木盆地石油勘探开采在1954年就进行，1955年第一口油井开钻。至今，青海石油的油气年产量已达459万吨。
