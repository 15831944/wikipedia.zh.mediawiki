'''青海省博物馆'''，位于[[青海省|青海省]][[西宁市|西宁市]][[城西区|城西区]]西关大街58号，是隶属[[青海省文化厅|青海省文化厅]]的综合性博物馆。

==简介==
1957年，第一次成立青海省博物馆筹备处，1962年撤销。1978年8月29日，再次成立青海省博物馆筹备处。1986年9月26日，青海省博物馆正式建馆并且对外开放。馆址设在西宁市[[城东区_(西宁市)|城东区]]为民巷41号原青海军阀[[马步芳|马步芳]]私宅“[[馨庐|馨庐]]”。1996年9月，青海省博物馆被[[中共青海省委|中共青海省委]]、[[青海省人民政府|青海省人民政府]]命名为“省级爱国主义教育基地”。同年10月，被青海省教育委员会命名为“学校德育教育基地”。1997年，被[[国家文物局|国家文物局]]评选为文博系统“全国爱国主义教育基地”。2001年5月1日，青海省博物馆新馆对外开放。新馆在西宁市城西区西关大街58号，[[新宁广场|新宁广场]]的东侧，新馆占地面积17000平方米，建筑面积20800平方米。新馆内设展厅9个，展陈面积9146平方米<ref name=qjjj>{{Cite web|url =http://www.qhei.gov.cn/whysh/whty/201305/t20130524_186956.shtml|title =青海省博物馆|author =|date =2013-05-24|publisher =青海经济信息网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20160823170715/http://www.qhei.gov.cn/whysh/whty/201305/t20130524_186956.shtml|archivedate =2016-08-23}}</ref>。2017年，青海省博物馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref>。

青海省博物馆自筹备时便开始收集整理青海地方历史、民族文物。截至2013年，馆藏文物47000余件，其中国家一级文物150多件。新馆推出四个常设展览：《青海史前文明展》、《青海民族文物展》、《青海[[藏传佛教|藏传佛教]]艺术展》、《可爱的青海》<ref name=qjjj/>。2017年时，常设展览为《江河源文明—青海历史文物展》、《青海省非物质文化遗产展》<ref>{{Cite web |url = http://www.qhmuseum.cn/list-about.html |title =  青博简介|author =  |date =  |publisher =青海省博物馆  |language =  |accessdate =2017-05-30  |quote =  }}</ref>。
