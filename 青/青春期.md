'''青春期'''（[[英語|英語]]：puberty） 是人體（或其他[[生物|生物]]）[[生殖|生殖]]機能及[[生理|生理]]機能成熟、[[心理|心理]]和[[生理|生理]]由[[兒童|兒童]]轉到[[成人|成人]]的時期。普通小型哺乳動物一、兩歲就能性成熟，對人類而言一般指10岁到20岁左右的年龄阶段。女性的青春期比男性早1-2年，即8-9歲開始。

== 概念 ==
青春期（又稱青少年期）是儿童期至成年期的过渡时期，体格、性徵、内分泌及心理等方面都发生巨大变化，人体生长发育是继婴儿期後出现的第二个高峰，各组织器官由稚嫩走向成熟，由能力不足趋向功能健全，世界观及信念逐步形成。

青春期开始的标志是[[第二性徵|第二性徵]]出现。[[褪黑素|褪黑素]]可抑制青春期發育<ref name="我即我脑">{{cite book |title = {{link-nl|我即我脑|Wij zijn ons brein}}：从子宫中孕育，于阿尔茨海默综合症中消亡 |publisher=中国人民大学出版社 |author = {{link-nl|迪克·斯瓦伯|Dick Swaab}} |year=2011 |location=北京}} 王奕瑶、陈琰璟、包爱民译。</ref>。[[卡門氏症候群|卡門氏症候群]]病患也無法進入青春期，還常伴隨有嗅覺障礙<ref name="我即我脑" />。对于青春期何时结束，没有定论，因为它是体格、智力、社会、法律、和心理等多个因素的结合。从认知方面讲，当一个人能够进行抽象思维时，就在这方面达到成熟。从社会学角度讲，如果一个人能够在社会上自立生活，或者选定一种职业，或已婚，或建立家庭，就可以说已经达到成熟期。从政治角度讲，一个人能够行使[[选举权|选举权]]（一般是18岁），没有父母许可也可以[[结婚|结婚]]，可以[[征兵制|参军]]，或者担负[[民事责任|民事责任]]，就可以算作成年。从心理学角度讲，能发现自己的个性特征，不依赖父母，形成自己的[[价值观|价值观念]]，能建立成熟互相依赖的[[友谊|友谊]]和[[爱情|爱情]]关系，即意味着成年。

== 阶段==

[[File:Tanner_scale-female.svg|thumb]]將女性乳房及陰毛的發育分為五個不同的階段|150px]]
对青春期的阶段目前并没有统一的分期标准，有部分学者将其分为：
# 青春期早期：[[第二性徵|第二性徵]]开始出现至女孩出现[[初潮|月经初潮]]，男孩出现首次[[遗精|遗精]]为止，表现是体格生长突增，年龄因人而異。
# 青春期中期：以性器官及第二性徵发育为主，以女孩出现[[初潮|月经初潮]]、男孩出现首次[[遗精|遗精]]为该时期的开始，以第二性征发育成熟为止，年龄因人而異。
# 青春期晚期：自第二性征发育成熟至生殖功能完全成熟、身高增长停止，女孩在这个阶段开始出现[[月经|周期性月经]]，年龄因人而異。
另外[[譚納標準|譚納標準]]也是表示發育階段的標準之一。

===性成熟===
性成熟指性心理或性生理上已經成熟。

===性早熟===
如果孩子因為遺傳（包含性[[激素|激素]]等[[內分泌|內分泌]]失調）、生活環境（經濟、家庭、同儕壓力、教育或攝入過多[[環境荷爾蒙|環境荷爾蒙]]）等等可能原因，在心理或生理一或是兩者層面上較一般孩子成熟，甚至可能是青春期提早結束。

== 生理变化 ==
青春期早期的表现形式是身高明显增长，这种现象一般女孩发生在10到14岁之间，而男孩一般在12到16岁之间。一般男孩比女孩发育更迟2年，骨骼發育速度會更快更久，因此大約16歲後將能超過女孩的身高。青春期的生理發育影響到骨骼和肌肉的所有方面，甚至包括眼睛，大多数人都是这一时期出现[[近视|近视]]，直到青春期結束近視趨勢才會停止。

=== 女性 ===
指从性器官开始发育、第二性徵出现至生殖功能完全成熟的一段时期。

由於[[卵巢|卵巢]]比[[睾丸|睾丸]]发育早，所以[[女孩|女孩]]的身体发育要比男孩早1-2年。女孩青春期时间大约为10至18岁。

身高、体重迅速增长；身体各脏器功能趋向成熟，神经系统的结构已接近成年人，思维活跃，对事物的反应能力提高，分析问题能力和记憶力增强；内分泌系统发育成熟，肾上腺开始分泌雌性激素刺激毛发生长，出现[[阴毛|阴毛]]、[[腋毛|腋毛]]。生殖系统[[下丘脑|下丘脑]]—[[垂体|垂体]]—[[卵巢|卵巢]]轴系统发育成熟，卵巢开始分泌[[雌激素|雌激素]]、[[孕激素|孕激素]]及少量[[雄激素|雄激素]]，刺激机体[[女性生殖系统|内]]、[[女性生殖系统|外生殖器官]]发育，出现第二性徵如：乳房隆起、皮下脂肪丰满、骨盆宽大、嗓音细高等，月经是青春期最显著的标志。

=== 男性 ===
大約13至14歲起，男性[[睾丸|睾丸]]和[[阴囊|阴囊]]开始增大，阴囊变红，皮肤质地改变。大約11到14岁时，阴茎变长，但是周径增大的速度较小，睾丸和阴囊仍在继续生长，出现阴毛，[[前列腺|前列腺]]开始活动。大約14到15岁，阴囊和阴茎开始继续增大，[[龟头|阴茎头根]]充分发育，阴囊颜色较深，睾丸发育成熟，開始出現變聲，長出喉結。<!--這裡屬於生理變化章節，无关内容避免列入-->

== 心理变化 ==
青春期的心理变化有：独立意向及求知欲强；想摆脱父母的监管，爭取團體的認同感，情绪容易不稳定。<ref>{{cite web  |title=如何與青春期的孩子相處，避免衝突和叛逆 |url=http://www.han-lin.tw/puberty-problem/  |accessdate=2018-10-28}}</ref>在性心理上，青春早期會表现出困惑、不安、害羞，对异性疏远和反感，青春中期後则对异性转为好感，喜欢与异性朋友交往。但思想单纯，社会经验不足，易受周围环境的影响，特别需要正确的指导和教育。

在這時候，青少年會非常且特別留意[[个人形象|自我形象]]，心理會出現[[逆反心理|反叛情緒]]。一些青少年會對[[人类性行为|性]]幻想或好奇，或渴望[[青春期恋爱|戀愛]]。
