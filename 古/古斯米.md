'''古斯米'''（{{lang-en|'''couscous'''}}，[[國際音標|國際音標]]：{{IPA|/kʊskʊs/}}；[[柏柏尔語|柏柏尔語]]：''Seksu''；{{lang-ar|كسكس}}）或稱為'''庫斯庫斯'''，又意譯為'''蒸粗麥粉'''，是一種源自[[馬格里布|馬格-{里}-布]][[柏柏尔人|柏柏尔人]]的食物。它由[[粗麵粉|粗麵粉]]製造，形状和颜色都像小米。

[[File:Kuskus.jpg|thumb]]
古斯米在馬格裡布地區是[[主食|主食]]，在[[阿爾及利亞|阿爾及利亞]]、[[摩洛哥|摩洛哥]]東部、[[突尼斯|突尼斯]]和[[利比亞|利比亞]]也是主食，被称作طعام（拉丁化：''ta`aam''，意为“食物”）。古斯米拌鱼是[[西西里|西西里]]西部[[特拉帕尼|特拉帕尼]]省的常见食物，被当地政府认定为传统美食。2011年古斯米被[[法國|法國]]人评选第三受欢迎的食物。

== 語源 ==

== 製造 ==
[[File:Couscous_of_Fes.JPG|thumb]]式古斯米]]

传统手工制法一般是粗麦粉加少量水，用手揉搓，并撒上干面粉，使之分成小米大小的颗粒，大的颗粒要打碎，粉末过筛除去，可重新回到第一步。大小符合要求的颗粒晒干就得到古斯米，可以存放数月。现代已经有机器可以自动化古斯米的生產及乾燥製成品。

== 烹調 ==
古斯米一般蒸熟后作为主食食用，蒸熟后会比较松散，可做成盖浇饭，搭配羊肉、牛肉、鸡肉、[[鸡蛋|鸡蛋]]、[[西葫芦|西葫芦]]、[[胡萝卜|胡萝卜]]、[[鹰嘴豆|鹰嘴豆]]（可依喜好任意添加）、[[洋葱|洋葱]]、[[土豆|土豆]]、[[番茄|番茄]]、[[香芹|香芹]]、[[白萝卜|白萝卜]]、[[青椒|青椒]]、[[茴香|茴香]]、[[辣酱|辣酱]]、[[孜然|孜然]]等食用。现在市场上也有蒸好的“方便古斯米”，食用前只需要加开水搅拌几分钟，十分方便。
