{{日本作品|古書堂事件手帖|ビブリア古書堂の事件手帖}}，是由{{Tsl|ja|三上延}}所寫的[[推理小說|推理小說]]作品，[[越島羽空|越島羽空]]負責插畫，[[MediaWorks文庫|MediaWorks文庫]]（[[ASCII_Media_Works|ASCII Media Works]]）發行。小說中文正體版由[[台灣角川|台灣角川]]<ref>繁體版卷一譯者為[[曉峰|曉峰]]，卷二起為[[黃薇嬪|黃薇嬪]]。</ref>、簡體版則是[[天聞角川|天聞角川]]<ref>简体版前两卷沿用繁体版译文，卷三译者为羽蛇，卷四起为[[朱悦玮|朱悦玮]]。</ref>發行。2013年由[[富士電視台|富士電視台]]改編為「[[月九|月九]]」[[日本電視劇|電視劇]]，[[剛力彩芽|剛力彩芽]]主演。2017年2月公布改編成真人及動畫電影消息<ref name="natalie220190">{{Cite news|url= http://natalie.mu/eiga/news/220190 |title= 「ビブリア古書堂の事件手帖」が実写＆アニメ映画化 |newspaper= 映画ナタリー |publisher= ナターシャ |date= 2017-02-10 |accessdate= 2017-02-10 }}</ref>。

== 劇情簡介 ==
北鎌倉不起眼角落有一家名為文現里亞的舊[[書店|書店]]。店主篠川栞子美麗端莊，且具有深厚的舊書知識和縝密的[[推理|推理]]能力。她單憑舊書主人所留下的蛛絲馬跡，便能些許窺得陌生人們不為人知的過去和[[秘密|秘密]]。某天，患有讀書障礙的失業青年五浦大輔請栞子鑒定去世[[外祖母|外祖母]]留下的藏書，從而瞭解了一段鮮為人知的舊事。以此為契機，他成為文現里亞的店員，而圍繞一本本或平凡或普通的舊書，一段段曲折離奇又饒有趣味的小故事相繼上演。

== 舞台 ==
故事開始於[[2010年8月|2010年8月]]。以[[神奈川縣|神奈川縣]]北鎌倉為背景。出現的地名都是實際存在，但書店與人物則是小說杜撰<ref>第一冊後記。</ref>。

== 登場人物 ==
=== 主要人物 ===
; {{nihongo|篠川栞子|篠川 栞子（しのかわ しおりこ）}}
: 本作女主角，25歲。劇中擔任偵探一角，文現里亞古書堂的店長。故事一開始，原為店長的父親於去年過世，所以由她繼承書店。
: 留著一頭黑色長髮，有著白皙肌膚的美人（與母親非常相似）。身材瘦小，胸部豐滿。有近視，平常會戴眼鏡。
: 性格相當內向，不擅長與他人交談，但擁有龐大的舊書知識，也熱愛書籍，一談到書口條會變得很流暢。
: 登場時因脊椎受傷住院中，出院後平時會拄著一根拐杖行動。
: 對於會為了入手稀有書而不擇手段的母親相當厭惡，有著與母親相似的性格這點自己也非常的恐懼。

; {{nihongo|五浦大輔|五浦 大輔（ごうら だいすけ）}}
: 本作男主角，23歲。
: 身材高壯，學生時代學習過柔道。性格木訥溫和，富有強烈的正義感。
: 因特殊原因從小便無法好好閱讀排滿印刷字體的書籍，長時間盯著看便會感到身體不適，只有漫畫與短篇小說能勉強讀完，本人稱為「閱讀恐懼症」，但其實非常憧憬閱讀。
: 由於內定的公司無預警的倒閉，大學畢業後便一直處於[[失業|失業]]在家狀態，所以被母親稱為「噗輔」。為了鑑定外婆的遺物《漱石全集》，便前往文現里亞古書堂，在那裡與栞子解開了外婆的秘密，也因這個機緣而留在書店打工。

=== 其他人物 ===
; {{nihongo|篠川文香|篠川 文香（しのかわ あやか）}}
: 栞子的妹妹，目前就讀大輔的母校高中。
: 與栞子相反，幾乎不懂舊書，個性也非常外向。負責篠川家的家事，擅長做料理。

; {{nihongo|篠川智惠子|篠川 智恵子（しのかわ ちえこ）}}
: 栞子與文香的母親，十年前[[失蹤|失蹤]]，下落不明。自身持有一定的舊書知識（栞子的舊書知識就是源自於母親），推理能力也高於栞子。
: 會運用自身過人的洞察力找到他人的弱點，並加以利用，為了書甚至會做出不道德的事，因此栞子對她非常反感。

; {{nihongo|五浦絹子|五浦 絹子（ごうら きぬこ）}}
: 《漱石全集·新版》初登場。
: 大浦的外婆。已故。生前經營著『五浦食堂』。死去的丈夫是入贅女婿。

; {{nihongo|五浦惠理|五浦 恵理（ごうら えり）}}
: 《漱石全集·新版》初登場。
: 五浦家的三女。大輔的母親，已離婚。有兩個同母異父的姊姊。

; {{nihongo|田中嘉雄|田中 嘉雄（たなか よしお）}}
: 《漱石全集·新版》初登場。大輔真正的外公，惠理的生父，絹子婚外情的戀人，田中敏雄的祖父。與大輔一样，身材高大。生前為舊書收藏家和《晚年》珍本的前持有者，因生活不繼而不得不以低价賣掉《晚年》。

; {{nihongo|舞子阿姨|舞子}}
: 《漱石全集·新版》初登場。
: 五浦家的長女，大輔的姨母，居住在[[藤澤市|藤澤市]]。遺傳了她父親矮小的特征。

; {{nihongo|志田肇|志田 肇（しだ はじめ）}}（假名）
:《拾穗・聖安徒生》初登場。
: 背取屋，文現里亞古書堂的男性常客。家住在鵠沼的橋底。
: 在「遺失拾穗」的事件解決後，與小菅的關係變得親近，彼此會交換閱讀的心得，被小菅稱呼為「老師」。

; {{nihongo|笠井菊哉|笠井 菊哉（かさい きくや）}}（假名）
: 《拾穗・聖安徒生》初登場。
: 背取屋，『笠井堂』（網絡書店）店主，通稱『男爵』（取自《背取男爵數奇譚》）。
: 令栞子脊椎受傷的犯人，真名為田中敏雄。

; {{nihongo|小菅奈緒|小菅 奈緒（こすが なお）}}
: 《拾穗・聖安徒生》初登場。
: 與文香同年級的女高中生。在事件發生後成為文現里亞古書堂的常客。

; {{nihongo|西野|西野（にしの）}}
: 《拾穗・聖安徒生》初登場。
: 奈緒的同班同學，同時也是暗戀對象。但本人對奈緒囂張高傲的態度感到不悅，拒絕了奈緒送的禮物。

; {{nihongo|坂口昌志|坂口 昌志（さかぐち まさし）}}
: 《邏輯學入門》初登場。

; {{nihongo|坂口忍|坂口 しのぶ（さかぐち しのぶ）}}
: 《邏輯學入門》初登場。昌志的妻子。

; {{nihongo|田中敏雄/大庭葉藏(假名)/笠井菊哉(假名)|大庭 葉蔵（おおば ようぞう）}}
: 《晚年》初登場。
: 千方百計想擁有曾為其祖父田中嘉雄持有、現由文現里亞古書堂收藏的太宰治《晚年》珍本，希望以高價收購未果後，不惜將栞子推下石階，入屋行竊。
: 本名田中敏雄，笠井菊哉和大庭葉藏只是他的其中兩個假身份（方便接近古書堂得知《晚年》的位置），為大輔的表兄弟(但此關係只有栞子和大輔知道)，名字由祖父命名，有關舊書的知识亦是由他傳授的。

; {{nihongo|瀧野蓮杖|滝野 蓮杖（たきの れんじょう）}}
: 港南台「瀧野書店」的兒子。
: 留著一頭分邊的短黑髮，下巴有些鬍渣，戴著金屬框的眼鏡。栞子的舊識，因父母同行，小時候便會到彼此家裡玩。

; {{nihongo|井上太一郎|井上 太一郎（いのうえ たいちろう）}}
: 辻堂古書店「一人書房」的店主。主要經手懸疑和科幻作品。
: 身材如鐵絲般消瘦的白髮男子。栞子母親的舊識，對栞子母女非常反感。

== 評價和反應 ==
在[[2012年1月|2012年1月]]日文版系列發行數量超過103萬本<ref>[http://www.shinbunka.co.jp/news2012/01/120105-06.htm 新文化 - 出版業界紙 - ニュースフラッシュ関連ページ]</ref>，是[[MediaWorks文庫|MediaWorks文庫]]首次出現百萬銷量的作品<ref>[http://mantan-web.jp/2012/01/12/20120111dog00m200047000c.html ビブリア古書堂の事件手帖：第1巻が発売9カ月で初の首位獲得　シリーズ累計は100万部を突破 - MANTANWEB（まんたんウェブ）]</ref>。至[[2012年4月|2012年4月]]超過200萬本，第三冊2012年6月發行時累積超過300萬本<ref>[http://mantan-web.jp/2012/06/22/20120621dog00m200046000c.html ビブリア古書堂の事件手帖：3巻で累計300万部突破　異例のハイペース - MANTANWEB（まんたんウェブ）]</ref>。第四冊2013年發行時累積超過470萬本。

《古書堂事件手帖》是2012年[[書店大獎|書店大獎]]的得獎作品（第8名）<ref>[http://www.hontai.or.jp/history/index.html これまでの本屋大賞 | 本屋大賞]</ref>，也是[[文庫本|文庫本]]首次得到書店大獎的作品。古書堂事件手帖入圍第65回[[日本推理作家協會賞|日本推理作家協會賞]]短編部門，2011年被[[書雜誌社|書雜誌社]]選為最佳小說<ref name="470million">{{Cite web |date=2013-02-19 |url=http://www.kadokawa-hd.co.jp/topics/20130219_biblia.pdf |title=『ビブリア古書堂の事件手帖』2月22日発行第4巻 初版発行80万部、シリーズ累計470万部 |format=PDF |publisher=角川グループ |accessdate=2013-02-23}}</ref>。

=== 與原著小說的差異 ===
* 原著中篠川栞子是身材豐滿的女子，因就讀女校，沒有和男性交往的經驗，因此五浦大輔有時不經意地會看到她的胸部，出現五浦覺得尷尬的場面。電視版似乎沒有這些情節。原著中篠川栞子患有大近視而需要佩戴眼鏡，她母親也需要佩戴眼鏡，母女皆留有長髮。電視版中無論篠川栞子還是她母親都沒有戴眼鏡，兩人皆为短髮。
* 原著中篠川栞子有一位妹妹篠川文香；電視版則是弟弟篠川文也。
* 原著沒有藤波明生的角色，因此他的咖啡店也不存在於小說內。
* 原著中坂口昌志賣《邏輯學入門》時篠川栞子仍在醫院，解謎也在醫院裡進行；電視版篠川栞子則在店內。
* 原著中篠川栞子差不多登上石階頂部時，被「大庭葉藏」用力推下肩膊，從石階跌下而受傷；電視版則被「大庭葉藏」踢下而受傷。原著中「大庭葉藏」曾跑下石階搜查篠川栞子的包裹，而確認篠川栞子沒有攜帶《晚年》；電視版則因為藤波明生剛好經過，「大庭葉藏」因而沒有搜尋篠川栞子的包裹。
* 原著中發現西野企圖燒東西的是坂口忍。
* 原著中五浦大輔在古書堂內識破笠井菊哉就是大庭葉蔵，笠井騎上泊在店外的單車趕到醫院，而五浦則騎上機車追他；電視版則是五浦大輔在笠井菊哉離開後才醒悟菊井就是大庭葉藏，騎單車追上笠井後被笠井搶去單車，只好跑到醫院。原著中五浦趕到醫院門外時，包著《晚年》的綢巾剛好從天台跌下；電視版中則是笠井到達醫院時，篠川栞子的頸巾剛好從天台跌下。
* 原著中找尋「有着貍貓、鱷魚和狗，像繪本般的作品」是坂口忍；電視版則改為五浦大輔的前女友高坂晶穗（原著中高坂晶穗出現在第二冊，委託文現里亞古書店收買亡父的書籍），高坂晶穗的家庭狀況也和原著不同，原著中高坂晶穗的父親剛死亡不久，而她本人是父親的情婦所生，而故事也出現高坂晶穗的同父異母姊姊；電視版則只有高坂晶穗和她的母親。
