'''古瀨車站'''（{{jpn|j=古瀬駅|hg=ふるせえき|rm=Furuse eki|f=y}}）是位於[[北海道|北海道]][[北海道|北海道]][[白糠郡|白糠郡]][[白糠町|白糠町]]和天別的[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[根室本線|根室本線]]的[[鐵路車站|鐵路車站]]。車站編號為'''K46'''。電報略號為'''フセ'''。

== 歷史 ==
本車站在號誌站時代，同時兼任臨時停靠站來處理乘客。而與號誌站併設的職員宿舍，使車站的主要使用對象是職員及其家庭成員（當時約有20間宿舍）。由於現在號誌站已經無人化，員工宿舍亦已沒有職員居住。
* 1954年（[[昭和|昭和]]29年）7月1日 - 開設[[日本國有鐵道|國有鐵道]]的'''古瀨'''號誌站。並配置職員。
* 1971年（昭和46年）8月1日 - 號誌站無人化。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後，由JR北海道繼承。同時升格為車站。
* 1990年（[[平成|平成]]2年）3月10日 - 設定營業距離。

== 車站構造 ==
* 古瀨車站為2面2線、[[側式月台|側式月台]]的地面車站。所有列車主要使用1號線，而需要交換列車時會使用2號車。
* 上下行月台由[[平交道|平交道]]所連接。
* 無人車站，由[[音別車站|音別車站]]所管理（以前由白糠車站管理）。

== 車站周邊 ==
車站主要被周圍的森林所包圍，並有少量的奶農在此生活。本車站亦是其中一個秘境車站。車站道路亦沒有得到很好的維護。
