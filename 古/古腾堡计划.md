'''古腾堡工程'''（Project Gutenberg，缩写：'''PG'''），由志愿者参与，致力于将文化作品的数字化和归档，并鼓励创作和发行[[电子书|电子书]]。该工程肇始于1971年，是最早的[[数字图书馆|数字图书馆]]。其中的大部分书籍都是[[公有领域|公有领域]]书籍的原本，古腾堡工程确保这些原本自由流通、[[自由檔案格式|自由檔案格式]]，有利于长期保存，并可在各种计算机上阅读。截至2018年7月，古腾堡工程声称超过57,000件馆藏。

在可能的情况下，发布的格式是纯[[文本文件|文本文件]]，但也包括其他格式，如[[HTML|HTML]]、[[PDF|PDF]]、[[EPUB|EPUB]]、[[MOBI|MOBI]]、{{link-en|Plucker|Plucker}}。大多数版本使用的语言是英语，但也可以有许多非英语的作品。有多个相关项目，提供更多的内容，包括区域性的和特定于语言的作品。古腾堡计划也密切联系{{link-en|Distributed Proofreaders|Distributed Proofreaders}}，一个校对扫描文本的以互联网为基础的社区。

== 历史 ==
古腾堡工程由{{link-en|Michael S. Hart|Michael S. Hart|Michael Hart}}（1947年－2011年）于1971年启动。当时Hart是[[伊利诺伊大学香槟分校|伊利诺伊大学]]的学生，他获得了学校材料研究实验室中Xerox Sigma V[[大型计算机|大型计算机]]的使用权限。通过和善的操作员，他得到一个几乎没有时限的帐号，后来估算这价值约100,000乃至100,000,000美元不等。Hart想通过做一些有价值的事情，作为这份馈赠"物有所值"。

这台计算机是[[ARPANET|阿帕网]]（全球[[互联网|互联网]]的鼻祖）的15个节点之一。Hart认为有朝一日公众都可以接触计算机，因而他决定将书籍电子化，供人们自由使用。刚好他背包里有一份[[美國獨立宣言|美國獨立宣言]]，该宣言也就成了古腾堡工程的的第一份[[电子文本|电子文本]]。

该工程的命名是为了纪念[[约翰内斯·古腾堡|约翰内斯·古腾堡]]，他是一位[[德国|德国]]15世纪的印刷商，通过推广使用[[活字印刷术|活字印刷术]]推动了[[印刷机|印刷机]]的革命。

1990年代中期，Hart在[[班尼迪克大学|伊利诺伊本笃会学院]]进行该项目，更多的志愿者参与了进来。在[[图像扫描仪|图像扫描仪]]和[[光学文字识别|光学文字识别]]软件得到改进并被广泛使用前，大部分文本都是靠手工输入。

Hart后来与[[卡内基梅隆大学|卡内基梅隆大学]]达成协议，由对方主导古腾堡工程的财务。随着电子文本的增加，志愿者开始替代Hart，进行日常的维护操作。

2000年，[[非營利組織|非營利組織]][[古腾堡工程著作归档基金会|古腾堡工程著作归档基金会]] （The Project Gutenberg Literary Archive Foundation Inc） 在[[密西西比州|密西西比州]]获准成立，以处理相关的法律事务。给古腾堡工程捐款可以得到减税的优惠。工程的长期志愿者Gregory Newby成为基金会的第一任[[首席执行官|首席执行官]]。

同样是在2000年，Charles Franks启动了[[分布校对员|分布校对员]]项目，志愿者从而可以将扫描的文本通过[[互联网|互联网]]共享发布。该举措极大地提高了加入PG的数量与种类，新的志愿者也可以更容易地贡献力量。

[[意大利|意大利]]志愿者Pietro Di Miceli开发、管理了第一个古腾堡工程的网站，并着手对在线工程进行编目。在他参与的10年间（1994年－2004年），网站获得了不少奖项，常被收录到"最佳网页"列表，这也提高了工程的知名度。<ref>[https://www.gutenberg.org/about/credit https://www.gutenberg.org/about/credit]</ref>

2004年开始，启用了新的在线目录，PG内容更容易浏览、获取和链接。

古腾堡工程现在由位于[[北卡罗莱纳大学教堂山分校|北卡罗莱纳大学]]的[[ibiblio|ibiblio]]主办。

== 收录范围 ==
[[Image:Project_Gutenberg_total_books.svg|thumb]]
到2018年4月为止，古腾堡工程已经收录了57,000部书籍，平均每週将新增50部。

其中主要是[[西方文化|西方文化传统]]中的文学作品，比如小说、诗歌、小故事、戏剧，除此之外，PG也收录[[食谱|食谱]]、[[书目|书目]]以及期刊。另外还包括一些非文本内容，比如音频文件、乐谱文件等。

收录中主要是英文作品，但也有相当数量的[[德语|德语]]、[[法语|法语]]、[[意大利语|意大利语]]、[[西班牙语|西班牙语]]、[[荷兰语|荷兰语]]、[[芬兰语|芬兰语]]以及[[中文|中文]]等不同语言的著作。

在可能的情况下，古腾堡以[[纯文本|纯文本]]的格式发布，主要使用[[ASCII|ASCII]][[字符集|字符集]]，也常被扩展为[[ISO/IEC_8859-1|ISO-8859-1]]，而中文书籍几乎全部都是以[[Big5|Big5]]（大五码）纯文本格式发布。志愿者提交时也可能会采用其他格式，最常见的是[[HTML|HTML]]。不容易编辑的格式，比如[[PDF|PDF]]，对于古腾堡工程的目标而言并不合适，但还是有一些这样的格式被提交。近年来，有关于[[XML|XML]]格式的讨论，但相关进展还比较缓慢。

== 理想 ==
Michael Hart在2004年有言“古腾堡工程的使命很简单：‘鼓励电子书的创建与发布。’”<ref>[https://www.gutenberg.org/about/faq0 https://www.gutenberg.org/about/faq0]</ref>

工程的一个口号是“推倒愚昧与无知的藩篱”，就像20世纪早期[[公共圖書館|公共圖書館]]的兴起，古腾堡的志愿者志在继续推广文字读写，从而更好地继承文化遗产。

古腾堡工程的运作方式不是集中式的，例如，没有一项选择政策来指定要添加哪部著作，志愿者只是针对他们感兴趣的或者手头有的著作，进行工作。

== 版权问题 ==
古腾堡工程根据[[美国版权法|美国版权法]]对其电子书进行版权验证。只有版权过期者才可以加到PG档案中来，版权失效的记录将保存以备未来参考。

与其他数字图书馆的项目不同，古腾堡工程不会在其出版物上声明新的版权，从而鼓励自由再加工、再发布。

PG的大部分书籍都依照美国的版权法律，作为[[公有领域|公有领域]]发布。其中有两种书籍，一种书籍的前段会带有古腾堡工程的商标，这意味着对书籍的再利用有少量的限制，限制内容写在电子书的授权条款里头（例如修改后再发布，或者商业用途）。而如果电子书的前段没有使用商标，作为公有领域的书籍就可以不受限地再利用。

古腾堡工程也发布了一些受版权保护的书籍，根据版权所有者的说明，就有进一步的限制。

1998年的美国参议员{{link-en|桑尼·波诺|Sonny Bono}}提出并被投票通过的{{link-en|版权期限延长法案|Copyright Term Extension Act}}，将现存的版权延长了20年。这导致本来在美国将变成公有领域的许多书古腾堡工程将无法添加。

== 批评 ==
古腾堡工程的书籍被指学术上不够严格，比如，缺少版本使用详情介绍，缺少最初版的封面，缺少校订工具。通过比较新舊版本，可以看到新版的电子书已经有较大改进。大部分新版的电子书都保留了版本信息以及封面。

== 姊妹项目 ==
古腾堡工程的所有姊妹项目都是独立的组织，但具有同样的理念，并获准使用古腾堡工程的商标。它们一般专注于某些族群或语种。
* [http://en.wikipedia.org/wiki/Project_Gutenberg_Australia Project Gutenberg Australia] 收录了许多根据[[澳大利亚版权法律|澳大利亚版权法律]]属于公有领域，但在美国仍然版权受限（或者无法确定），特别关注澳大利亚作家以及关于澳洲的书籍。
* [http://www.gutenberg.nl/ PG-EU] 根据的是[[欧盟|欧盟]]的版权法律。它的一个目标是将尽量多的语言包括进来。采用[[Unicode|Unicode]]以确保所有文字都可以正确显示。
* [https://web.archive.org/web/20070824002901/http://www.gutenberg.ph/ Project Gutenberg of the Philippines] 针对[[菲律宾|菲律宾]]及其语言。
* [https://web.archive.org/web/20070820234835/http://pge.rastko.net/ Project Gutenberg Europe] 由{{link-en|Project Rastko|Project Rastko}}维护，位于塞尔维亚－黑山。2005年启动，使用[[分布校对员|分布校对员]]软件，以加速电子文本的制作。
* [http://www.gutenberg.lu/ Project Gutenberg Luxembourg] 主要发布[[卢森堡语|卢森堡语]]的著作
* [http://www.gutenberg.us/ Project Gutenberg Consortia Center]，格式不那么严格一致，按主题收录、包括多种语言是其特色。
* [http://www.lonnrot.net/ Projekti Lönnrot] 由[[芬兰|芬兰]]的古腾堡志愿者启动。
尽管{{link-en|Projekt Gutenberg-DE|Projekt Gutenberg-DE}}数年前被获准使用古腾堡的名称，但由于其理念不同，所以并不是每个人都将其看作古腾堡的姊妹站。Projekt Gutenberg-DE对其产品设定版权，并将其文本限制为可浏览的网页版。

也有一些受古腾堡工程启发而创立的项目，详情可参见[[数字图书馆项目列表|数字图书馆项目列表]]。
