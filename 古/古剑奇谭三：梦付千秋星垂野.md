《'''古剑奇谭三：梦付千秋星垂野'''》是一款由[[上海烛龙|上海烛龙]]开发，[[网元圣唐|网元圣唐]]发行的[[动作角色扮演游戏|动作角色扮演游戏]]，为[[古剑奇谭系列|古剑奇谭系列]]的第三部单人游戏作品。游戏于2014年12月立项，简体中文版于2018年11月23日上市。

==世界觀==

===故事===
{{剧情摘要不完整|time=2018-12-9T14:27:00+08:00|small=left}}
一個只流傳於魔域之中，甚少為人所知的妖族 - 辟邪，其族人突然活躍現身。因辟邪王將死，為了有人能領導其族而尋回幼年流落人間的辟邪少主回到王城，承繼王位<ref>{{cite web | url=https://www.evargame.com/article/gu-jian-qi-tan-3/ |title=《古劍奇譚三》故事起源 | work=Evargame | date= 2018-10-10 | accessdate=2018-11-09}}</ref>。

==主要角色介紹<ref name="GameRobin">{{cite web | url=https://www.evargame.com/article/gu-jian-qi-tan-3/ |title=主要角色介紹 | publisher=Evargame | date= 2018-10-10 | accessdate=2018-11-09}}</ref>==
* 北洛: 男主角，妖族，辟邪王。辟邪王族的原储君，有一孿生哥哥，先王玄戈。<br>
* 雲無月: 故事女主角，魇魅族。妖族尊稱其為霒蚀君。<br>
* 岑纓: 故事女配角，人族。愛讀書，更愛風水堪輿學術，仰慕黄帝。<br>
* [[黄帝|姬轩辕]]: 故事男配角，人族。有熊部落族长，逝世后为镇压魔气，以梦境域主形态帮助主角<br>
* 玄戈: 辟邪王北洛的孿生哥哥，妖族。辟邪先王，后因力量衰竭而死将王位传给弟弟北洛。<br>
* 霓商: 辟邪先王妃，原辟邪王玄戈的王妃，輔助先王和现在的辟邪王處理政務。<br>
* 嵐相: 辟邪王室血脈，武力高強的悍將，后因保卫天鹿城战死。<br>
* 羽林: 受玄戈器重的王族戰士，處事謹慎。后因保卫天鹿城战死。<br>
* 凌星見: 廬山星工晨儀社，天才之格，門派內定的下任掌門。<br>
* 風裡霜: 遙夜灣沙海的寄靈族族長。<br>
* 原天柿: 一隻寵物，小名柿餅，家園的總管。

==开发==
2014年12月，《古剑奇谭三》正式立项<ref name="project">{{cite web | url=http://games.sina.com.cn/ol/n/2014-12-25/1024843882.shtml |title=游戏吧新品牌《九凤RPG》首曝光 《古剑奇谭3》立项 | publisher=[[新浪|新浪]]游戏 | date=2018-02-11 | accessdate=2014-12-25}}</ref>。开发团队认为前作的全即时战斗系统完成度不够高，所以《古剑奇谭三》会朝这个方向继续深化，战斗为全即时，场景中直接开战，不再切换单独的战斗场景<ref name="combat_system">{{cite web | url=http://games.ifeng.com/pcgame/news/detail_2015_07/10/41072373_0.shtml |title=《古剑奇谭》五周年纪念 第三代最新消息公布 | publisher=[[凤凰网|凤凰网]]游戏 | date=2015-07-10 | accessdate=2018-02-11}}</ref>。

《古剑奇谭三》开发实际耗时约3年，项目投资规模约为4000万元人民币，共有约50名开发人员参与其中。<ref>{{cite web |author1=胡正达 |title=与《古剑奇谭三》项目组一席谈 |url=http://www.chuapp.com/article/285967.html |publisher=触乐网}}</ref>

==发行==
[[E3_2017|E3 2017]]上，发行商[[网元圣唐|网元圣唐]]确认游戏将于2018年发行<ref name="release">{{cite web | url=http://www.gamersky.com/news/201706/915973.shtml |title=E3 2017：国产单机《古剑奇谭3》确认2018年发售！ | publisher=[[游民星空|游民星空]] | date= 2017-06-14 | accessdate=2018-02-11}}</ref>。2018年2月7日，《古剑奇谭三》曝光了首部宣传动画，游戏开始进入宣传期<ref name="publicity">{{cite web | url=http://news.yzz.cn/domestic/201802-1303069.shtml|title=《古剑奇谭三》首部宣传动画公开 | publisher=叶子猪 | date=2018-02-07 | accessdate=2018-05-18}}</ref>。2018年8月20日，《古剑奇谭三》试玩版向玩家开放。<ref name="trial">{{cite web | url=http://gjqt3.wangyuan.com/2018/08/21/000017.shtml| title=《古剑奇谭三》试玩版已全面开放| publisher=《古剑奇谭三》官方网站| date=2018-08-21| accessdate=2018-09-12}}</ref>2018年9月，官方宣布游戏将于11月23日发行<ref name="released">{{cite web | url=https://www.gamersky.com/news/201809/1104147.shtml| title=《古剑奇谭3》将于11月23日正式上市| publisher=[[游民星空|游民星空]]| date=2018-09-25| accessdate=2018-09-25}}</ref>。
