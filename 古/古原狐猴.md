'''古原狐猴'''（''Palaeopropithecus''）是[[馬達加斯加|馬達加斯加]]已[[滅絕|滅絕]]的[[狐猴|狐猴]]，[[碳十四|碳十四]]測年顯示古原狐猴可能生存至1500年。<ref name=martin1990>{{Cite book en|author=Martin, R.D.|year=1990|title=Primate Origins and Evolution: A Phylogenetic Reconstruction|publisher=Princeton University Press|location=Princeton, New Jersey}}</ref>

== 形態 ==
古原狐猴的[[上顎|上顎]]及[[下顎|下顎]][[齒式|齒式]]分別是2:1:2:3及2:0:2:3。牠的下[[門齒|門齒]]很細小及垂直，並呈竹片狀。牠的[[臼齒|臼齒]]很窄及有良好發展的剪冠。<ref name=fleagle1988>{{Cite book en|author=Fleagle, J.G.|year=1988|title=Primate Adaptation and Evolution|publisher=Academic Press|location=New York}}</ref>相對於[[大狐猴科|大狐猴科]]，牠的[[頭顱骨|頭顱骨]]略為粗壯，鼻端較長。牠有下顎骨聯合，並[[鼓膜環|鼓膜環]]側伸出管道。牠的前肢較後肢長。拇指很短，而[[指骨|指骨]]很長及彎曲，其[[肢間指數|肢間指數]]為138。

== 食性 ==
根據古原狐猴的[[牙齒|牙齒]]形態，可見牠們是吃[[樹葉|樹葉]]的。
 
== 運動 ==
根據古原狐猴的顱後[[骨骼|骨骼]]，牠們是陸地上四足行走的。
