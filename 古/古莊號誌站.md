'''古莊號誌站'''位於[[台灣|台灣]][[臺東縣|臺東縣]][[大武鄉_(台灣)|大武鄉]]，位於[[中央號誌站|中央號誌]]至[[大武車站|大武]]間，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[南迴線|南迴線]]的[[號誌站|號誌站]]。

== 車站構造 ==
*原設有[[島式月台|島式月台]]一座。

== 利用狀況 ==
*原為[[臺灣鐵路管理局車站等級#三等站|三等站]]，但自2017年10月1日起停辦客運業務，裁撤為[[臺灣鐵路管理局車站等級#號誌站|號誌站]]。<ref name="李姿慧20170922">{{cite news|url=http://www.appledaily.com.tw/appledaily/article/headline/20170922/37789246/ |author=李姿慧|title=改善血汗 台鐵將增2860人|work=[[台灣蘋果日報|蘋果日報]] |date = 2017-09-22|accessdate=2017-09-22}}</ref>。
*先前仍然派駐值班站長監控列車及南迴鐵路電氣化施工之路線封鎖。
*目前已無行車人員駐守，但必要時仍派駐行車人員監控列車及南迴鐵路電氣化施工之路線封鎖。

== 歷史 ==
*1992年10月5日：設站。
*2010年12月起停止電腦售票，此後本站無售票業務。<ref>《[[鐵道情報|鐵道情報]]》209期</ref>。
*2016年6月28日：啟用多卡通刷卡機，接受乘客以[[悠遊卡|悠遊卡]]、[[一卡通|一卡通]]、[[有錢卡|有錢卡]]、[[iCash|iCash 2.0]]付費。
*2017年10月1日：停辦客運，改為號誌站，停辦客運前每日僅有四班車停靠，順逆行各兩列次。<ref name="李姿慧20170922"/>。
