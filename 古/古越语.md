'''古越语'''是古代[[於越人|於越人]]的語言，[[春秋时期|春秋时期]]通行于[[吴国|吴]]、[[越国|越]]两国，与当时附近[[楚国|楚]]及中原諸國言语不同。[[语言学|语言学]]研究表明，古越语可能属于[[侗台语系|侗台语系]]，与现代的[[侗语|侗语]]、[[壮语|壮语]]、[[泰语|泰语]]等关系较近；但也有研究認為應屬於[[南亞語系|南亞語系]]。

古越语留存至今的文献材料极少，内容较完整的有《[[越人歌|越人歌]]》与《[[越绝书|越绝书]]》中的《[[勾践|勾践]]维甲令》，其他材料则包括《越绝书》、《[[吴越春秋|吴越春秋]]》、《[[輶軒使者絕代語釋別國方言|方言]]》等文献中的零星记载以及人名、地名的记录。{{r|wywhz}}

== 语系 ==
语言学家最初猜测古越语属于[[南岛语系|南岛语系]]或侗台语系。1953年，日本学者{{link-ja|泉井久之助|泉井久之助}}透過属于南岛语系的[[占语|占语]]、[[马来语|马来语]]、[[印尼语|印尼语]]对《越人歌》中用汉字标音的古越语进行研究。此后，韦庆稳、[[郑张尚芳|郑张尚芳]]等学者则透過属侗台语系的壮语、泰语等释读《越人歌》、《维甲令》，从而确认了古越语有侗台语系的成分。{{r|wywhz}}{{r|wei}}{{r|ma}}

[[梅祖麟|梅祖麟]]等學者分析古代典籍記錄的越語，認為越語屬於南亞語系<ref>Mei, Tsu-lin and Jerry Norman, “The Austroasiatics in Ancient South China: Some Lexical Evidence,” Monumenta Serica 32(1976): 274-301.</ref>。
== 语法 ==
古越语最显著的语法特征便是定语位于中心词后，与其他侗台语言相同。如《越绝书》中记载的“朱余者，越盐官也。越人谓盐曰余……”即定语后置的例证。又如，《[[左传|左传]]·哀公元年》记载“吴王[[夫差|夫差]]败越于夫椒”，[[杜预|杜预]]注为“夫椒，吴郡吴县西南太湖中椒山”。其中“夫椒”即“椒山”，“夫”为石山之意，亦为定语后置。{{r|wywhz}}{{r|wzh}}

古越语中还使用重叠形式表示摹状修辞，如《越人歌》中用“秦胥胥”表高兴之意。{{r|wywhz}}

== 词汇 ==
作为侗台语系语言，古越语中有许多基本词汇与现今[[侗语|侗]]、[[壮语|壮]]、[[泰语|泰]]、[[傣语|傣]]等侗台语言相同，也有部分与[[京语|京语]]相同。此外，有些词汇与汉语[[同源詞|同源]]，但意义略有差别，如“州”表示摇船、“鬲”表示金属锅等。{{r|wywhz}}

现今吴越旧地的许多地名都是古越语的音译，如[[余姚|余姚]]、[[余杭|余杭]]等中的“餘”为“田地”之意，[[句容|句容]]中的“句”为“族群”之意，[[会稽|会稽]]为“矛山”之意，[[盱眙|盱眙]]为“善道”之意等。{{r|wywhz}}{{r|zhou}}

== 语音 ==
据《[[淮南子|淮南子]]》记载：“胡人有知利者，而人谓之駤。越人有重迟者，而人谓之訬，以多者名之。”其中“訬”为轻快敏捷之意，说明古越语发音轻快急速。{{r|jiang}}

古越语的语音发展接近泰语与汉语[[上古汉语|上古]]晚期。其中还有[[复辅音|复辅音]]。{{r|wywhz}}据[[罗香林|罗香林]]考证，《越绝书》“ 句践乃身被赐夷之甲，带步光之剑，杖物卢之矛，出死士三百人，为阵关下”中的“物卢”便是一个复辅音词，读作plou。{{r|jiang}}

== 古越语遗存 ==
虽然古越语早已消亡，但汉语中仍有许多古越语遗存的成分。

[[吴语|吴语]]中，[[帮母|帮母]]、[[端母|端母]]唸成先喉塞音{{IPA|[ʔb]}}、{{IPA|[ʔd]}}，使用名词定语后置的构词法等语音、语法现象都来自古越语。词汇方面亦有许多古越语遗存，如南方称小河为“滨”，[[温州话|温州话]]将田野称为“垟”等，也与现代[[傣语|傣语]]、[[泰语|泰语]]等发音相近。{{r|wywhz}}{{r|wzh}}[[粤语|粤语]]中，{{IPA|[aː]}}与{{IPA|[a]}}长短两套元音、性别修饰语放在名词之后 （如“鸡公”、“鸡乸”）等现象都与侗台语相同，也属于古越语的遗存。{{r|oyjy}}而其他诸如[[赣语|赣语]]等方言中亦有许多古越语底层词。{{r|xiao}}
