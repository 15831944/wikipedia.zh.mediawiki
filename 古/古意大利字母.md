'''古意大利字母'''指古時存在於[[意大利半島|意大利半島]]上，而現今不再使用的[[印歐語系|印歐語系]]（主要是[[意大利語族|意大利語族]]）和非印歐語系[[字母系統|字母系統]]（如伊特拉斯坎字母），由公元前8世紀的[[優卑亞字母|優卑亞字母]]演化而成。一些屬於[[意大利語族|意大利語族]]分支的[[印歐語系|印歐語系]]初時使用的古意大利字母。伊特拉斯坎字母演變成[[法利希語|法利希語]]、[[奧斯坎語|奧斯坎語]]、[[翁布里亞語|翁布里亞語]]、[[北皮賽恩語|北皮賽恩語]]和[[南皮賽恩語|南皮賽恩語]]所用字母。

== 伊特拉斯坎字母 ==
[[File:Etruscan_cippus_warrior_head_side.jpg|thumb]]
{{alphabet|state=collapsed}}

'''伊特拉斯坎字母'''（Etruscan alphabet）由[[希臘字母|希臘字母]]演化出來，此過程发生在[[希臘|希臘]]在[[意大利|意大利]]的殖民地、[[庫邁|庫邁]]或[[希臘|希臘]]發生，是[[希臘字母|希臘字母]]的一種西支變異體。 在一般的[[希臘字母|希臘字母]]西支變異體中，'''X''' 代表發音 {{IPA|[ks]}} 而 '''Ψ''' 代表發音 {{IPA|[kʰ]}}；伊特拉斯坎字母中，'''X''' 代表發音 {{IPA|[s]}} 而 '''Ψ''' 代表發音 {{IPA|[kʰ]}} 或 {{IPA|[kχ]}}(Rix 202-209)。

最早教授伊特拉斯坎字母的雕刻碑在約公元前700年出現，列出26個與現時希臘字母相對的字母，保留 '''[[Ϻ|san]]''' 和 '''[[Ϙ|qoppa]]''' 但沒有 '''[[Ω|omega]]'''。

直至約公元前600年，伊特拉斯坎字母的古體沒有改變，而書寫方向沒有限制。公元6世紀起，字母隨伊特拉斯坎語的[[音韻學|音韻學]]而演變，廢用代表不是伊特拉斯坎語[[音位|音位]]的字母。到公元前400年，所有伊特拉斯坎人使用由20個字母組成的傳統伊特拉斯坎字母表，由左向右書寫。

對於以前的字母表，新字母表增加譯作 '''F''' 的符號{{script|Ital|𐌚}}，字形像阿拉伯數字8，出現在[[呂底亞語|呂底亞語]]和[[伊特拉斯坎語|伊特拉斯坎語]]，來源可能是B的變化形、H的變化形或新創的符號，取代 '''FH'''來代表發音 ''/f/'' 。有些字母被廢用，如 '''B'''、'''D'''、'''K'''、'''O''' 分別被 '''P'''、'''T'''、'''G'''（或譯作 '''C'''）、'''U''' 所取代。而相連的 '''CE'''、'''KA'''、'''QU''' 也被 '''C'''、'''K'''、'''Q''' 取代。

這個傳統的字母表一直沿用，在公元前2世紀受[[拉丁字母|拉丁字母]]影響而改變，不久伊特拉斯坎語滅亡。

== 奧斯坎字母 ==
奧斯坎人在公元前7世紀開始使用伊特拉斯坎字母，後來在公元前5世紀出現變異體'''奧斯坎字母'''（Oscan alphabet），加入代表長[[元音|元音]]的字母 '''I''' 和 '''U''' （譯為 '''Í''' 和 '''Ú''' ）。 '''U''' 用來表示[[奧斯坎語|奧斯坎語]] ''o''，而 '''Ú''' 用來表示[[奧斯坎語|奧斯坎語]] ''u''。

== 努凱里亞字母 ==
'''努凱里亞字母'''（Nucerian alphabet）在公元前6世紀至公元前5世紀出現，根據[[意大利|意大利]]南部發現的銘刻而成。最重要的字母是狀似杉樹的 /S/ ，可能是[[腓尼基字母|腓尼基字母]]的變異體。

== 盧加諾字母 ==
[[File:Venetic_Raetic_Camunic_Lepontic_alphabets.png|thumb]]北部字母︰ Este字母(古[[威尼托语|维尼提]]字母), Magrè字母(东列托字母)，[[波尔查诺|波尔查诺]]字母（西托字母），[[桑治奧|桑治奧]]字母，[[盧加諾|盧加諾]]字母。]]
根據[[意大利|意大利]]北部和[[提契諾州|提契諾州]]的銘刻而成的'''盧加諾字母'''（Alphabet of Lugano）是公元前7世紀至公元前5世紀時出現，用作記錄[[南阿爾卑高盧語|南阿爾卑高盧語]]的銘刻。

字母系統有由伊特拉斯坎字母發展而成的17個字母︰

字母系統沒有辨別[[清濁音|清]][[塞音|塞音]]和[[清濁音|濁]][[塞音|塞音]]，如 '''P''' 代表 ''/b/'' 或 ''/p/'' 、'''T''' 代表 ''/t/'' 或 ''/d/''、'''K'''代表 ''/g/'' 或 ''/k/''、'''Z'''代表 ''/t<sup>s</sup>/''、'''U''' 代表 ''/u/''、 '''V''' 代表 ''/w/''、'''Θ''' 代表 ''/t/''、 '''X''' 代表 ''/g/''。
