'''古文'''為[[中國|中國]]古代[[散文|散文]]的文體，其形式較[[駢文|駢文]]自由。原指[[先秦|先秦]]、[[秦代|秦代]]、[[兩漢|兩漢]]以來的[[散文|散文]]，但經過[[唐朝|唐]][[宋朝|宋]]的[[古文運動|古文運動]]，在文風方面又有新的突破。

== 歷史 ==
此處的古文是[[中国文学史|中国文学史]]上的术语，自唐代以来，有特殊而狭隘的涵义。并非一切文言文都算古文，同樣的，在某种条件下，“古文”也不一定跟白话对立。

由于[[六朝|六朝]]以来，尚求美感，作文多为[[駢文|駢文]]，过于讲求词藻、[[对偶|对偶]]、[[典故|典故]]，越来越不适应实际应用。

唐初[[陳子昂|陳子昂]]等人已倡古文，韓愈倡[[古文運動|古文運動]]，[[柳宗元|柳宗元]]和之，後至宋[[欧阳修|歐陽脩]]任科舉考官，提拔古文人才，古文運動方告成功。因为他们标榜[[先秦|先秦]]、[[西汉|西汉]]的文章，因此称为“古文”。[[明代|明代]][[前七子|前七子]]、[[後七子|後七子]]、[[唐宋派|唐宋派]]皆對古文的規律，有其獨特的見解。

清代[[桐城派|桐城派]]對古文更加嚴格要求，[[方苞|方苞]]所謂：“古文中忌语录中语、魏晋六朝人藻丽俳语、汉赋中板重字法、诗歌中隽语、南北史佻巧语。”（[[沈廷芳|沈廷芳]]《隐拙轩文钞·方望溪先生传·自记》）后来的桐城派作者更扩大范围，陆续把“注疏”、“尺牍”、“诗话”等语言都列违禁。

清末民初，白話文興起，古文受到一定的打壓。

== 古文特點 ==
古文不重音韻、不重對仗、亦不重辭藻，唯作文時須文以載道，言之有物，古文之文，多合於孔孟儒者之道，因韓愈所謂古文之道，孔孟[[儒家|儒家]]之道也。古文的文體照《[[古文辭類纂|古文辭類纂]]》的分法有「論辯類」、「序跋類」、「奏議類」、「書說類」、「贈序類」、「詔令類」、「傳狀類」、「碑誌類」、「雜記類」、「箴銘類」、「頌贊類」、「辭賦類」、「哀祭類」等。

== 古文流派 ==
=== 明 ===
*[[復古派|復古派]]：[[前後七子|前後七子]]主張「文必秦漢，詩必盛唐」，反抗空泛平庸的[[八股文|八股文]]與[[臺閣體|臺閣體]]。但因擬古，為文缺乏個人風格，文字難懂。
*[[唐宋派|唐宋派]]：反對前後七子的「文必秦漢」主張，推崇[[唐宋八大家|唐宋八大家]]散文的平易通達。代表人物有[[茅坤|茅坤]]、[[王慎中|王慎中]]、[[唐順之|唐順之]]、[[歸有光|歸有光]]等。王慎中、唐順之、歸有光並稱「[[嘉靖三大家|嘉靖三大家]]」；王慎中、唐順之另與六人並稱「[[嘉靖八才子|嘉靖八才子]]」。其中茅坤編有《[[唐宋八大家文鈔|唐宋八大家文鈔]]》。
*[[公安派|公安派]]：代表人物是[[公安三袁|公安三袁]]，主張「獨抒性靈，不拘格套」，強調表達真實情感，文章不再以「載道」為目的。
*[[竟陵派|竟陵派]]：代表人物有[[鍾惺|鍾惺]]、[[譚元春|譚元春]]，繼承公安派「獨抒性靈」的理論，但提倡「幽深估俏」以救治公安派用字俚俗淺陋的弊病，文字亦因此趨於晦澀。

=== 清朝 ===
*[[桐城派|桐城派]]：創始人為[[方苞|方苞]]，[[劉大櫆|劉大櫆]]和[[姚鼐|姚鼐]]為集大成者，三人有「[[桐城三祖|桐城三祖]]」之稱。主張效法《[[左傳|左傳]]》、《[[史記|史記]]》等[[先秦|先秦]][[兩漢|兩漢]]的散文及[[唐|唐]][[宋朝|宋]]散文。提倡古文「義法」：「義」即內容思想，要「言有物」；「法」即形式技巧，要「言有序」。其中姚鼐編有《[[古文辭類纂|古文辭類纂]]》。
*[[陽湖派|陽湖派]]：代表人物有[[惲敬|惲敬]]、[[張惠言|張惠言]]，是桐城派的一個旁支，但思想較活潑，不專主[[孔子|孔]][[孟子|孟]]、[[程朱理學|程朱]]，主張兼採[[諸子百家|諸子百家]]。認為學古文要兼取[[駢文|駢文]]之長；除取法[[六經|六經]]外，也兼取子史。
*[[湘鄉派|湘鄉派]]：創始人為[[曾國藩|曾國藩]]，是桐城派的一個旁支，認為古文創作應兼取[[漢賦|漢賦]]的氣韻，主張剛柔並濟、奇偶互用。提倡「義理」、「辭章」、「考據」、「經濟<ref>經世濟民</ref>」四者並重。其中曾國藩編有《[[經史百家雜鈔|經史百家雜鈔]]》。

== 古文大家 ==
