'''古籍注解体式'''，即'''古注体式'''，又称'''训诂体式'''<ref name=YYXMC>{{Cite book | author = 语言学名词审定委员会 | title = 语言学名词 | location = 北京 | publisher = [[商务印书馆|商务印书馆]] | date = 2011 | ISBN = 9787100068710}}</ref>{{rp|160}}，指[[中文古典典籍|中文古典典籍]][[注释|注释]]的体式。古人作注时角度和侧重点不同，因此产生了不同的体式类型。

[[注解|注解]]古籍的基本原因在於[[語言|語言]]的演變，還有古今[[文化|文化]]的差異，如[[社會制度|社會制度]]、[[人文地理|人文地理]]等。任何時代的多數人都難以讀懂用[[古語|古語]]寫成、反應當時社會制度和人文地理的前代[[文獻|文獻]]，故而需要具有專門知識的[[學者|學者]]來說明這些文獻的含義。

[[中國歷史|中國歷史]]上的[[學者|學者]]對前代[[文獻|文獻]][[典籍|典籍]]，尤其是[[唐代|唐代]]以前的[[古書|古書]]撰寫了大量的[[注解|注解]]。這些注解成為中國古代典籍的重要組成部份，是中文古典典籍的特色之一。直到今天，[[書店|書店]]裏仍然很容易見到各種古書的新注解。

== 传注 ==
'''传注'''是直接解释古籍正文的词语意义、典章制度、历史事实、思想内容的训诂体式<ref name=YYXMC />{{rp|160}}。这类注释名称繁多，常见有诂、训、传、注、笺等。
* '''诂'''、'''训'''侧重于字词、名物解释。其中，“诂”（通作“故”）侧重以今語释古语，“训”侧重用形象化的描绘来说明，但后世常常连用，不加区分<ref name=WN>{{Cite book | author = 王宁 | title = 古代汉语 | location = 北京 | publisher = 北京出版社 | date = 2002 | ISBN = 7200045918}}</ref>{{rp|283}}。如[[东汉|东汉]][[张衡|张衡]]《周官训诂》（已佚）、[[南宋|南宋]][[钱文子|钱文子]]《[[诗训诂|诗训诂]]》等。
* '''传'''指传述，原本指阐述[[儒家|儒家]][[六经|六经]]文义的书，如《[[左传|左传]]》、《[[公羊传|公羊传]]》、《[[谷梁传|谷梁传]]》、[[西汉|西汉]][[毛亨|毛亨]]《[[毛诗|毛诗故训传]]》等。后来“传”也用作一般的注释名称，如南宋[[朱熹|朱熹]]《[[诗集传|诗集传]]》。
* '''注'''是注释的通称，如东汉[[郑玄|郑玄]]《[[周官注|周官注]]》、《[[三礼注|三礼注]]》等。[[西晋|西晋]][[张华|张华]]《[[博物志|博物志]]》说：“上代去先师近，解释经文皆曰传，传师说也。后代去师远，或失其传，故谓之注。注，下己意也。”[[唐代|唐代]][[孔颖达|孔颖达]]《[[五经正义|春秋左传正义]]》说：“毛君、[[孔安国|孔安国]]、[[马融|马融]]、[[王肅_(三國)|王肅]]之徒，其所注书皆谓之传，郑玄则谓之注。”在《[[礼记|礼记]]·曲礼上第一》中又说：“注者，即解书之名。但释义之人多称为传，传谓传述为义。或亲承圣旨，或师儒相传，故云传。今谓之注者，谦也。”<ref name=WN />{{rp|284}}可见“注”和“传”一脉相承，郑玄始称“注”，有谦下之意。
* '''笺'''是表识的意思，对原注隐而不显、略而不详之处加以申发，或表达自己的看法，如郑玄《[[毛诗笺|毛诗笺]]》等。

== 义疏 ==
'''义疏'''是兼释古籍正文与传注的训诂体式<ref name=YYXMC />{{rp|160}}。义疏的词源义是“疏通其义”，又称'''疏'''、'''正義'''。疏是对原注的注。這種體裁出現於[[南北朝|南北朝]]早期，其起源受到了僧人講解佛經的影響。從樹立官方學術的角度，歷代朝廷將其組織學者編寫的疏稱為“正義”。

疏與注的關係是“疏不破注”，即疏的思想內容不去破壞注的體系，只在注的框架內進一步講解。這一點只在南北朝、唐、[[宋朝|宋]]成書疏中被遵守。[[清代|清代]]以來的學者也作有很多名為“疏”的古籍注解，但已不遵守“疏不破注”的原則。

南宋以前注與疏分成二書刊行，南宋[[紹熙|紹熙]]間三山[[黃唐|黃唐]]始將注疏合刻。义疏类注释往往以某一个注本为基础，包含经传原文、注、疏等几部分，按顺序排列，有明显的标记相隔，如唐代[[贾公彦|贾公彦]]《[[周礼注疏|周礼注疏]]》、[[南朝|南朝]][[梁代|梁代]][[皇侃|皇侃]]《[[论语义疏|论语义疏]]》、[[清朝|清朝]][[劉寶楠|劉寶楠]]《論語正義》等。

== 集解 ==
'''集解'''是汇集各家对同一部经典的注释，有时也补充汇集者自己的阐释的训诂体式<ref name=YYXMC />{{rp|160}}。名称有集解、集注等，如南宋朱熹《[[四書集注|孟子集注]]》、《诗集传》等。

西晋[[杜预|杜预]]的《[[春秋经传集解|春秋经传集解]]》虽然也叫“集解”，实指集《[[春秋|春秋]]》之经、《左传》之传而解之，应属于传注类。

== 补注 ==
'''补注'''是在注的基础上，选择一家较好的注本予以补充修订的训诂体式。<ref name=YYXMC />{{rp|160}}名称有补、补疏、补释、补义、补正等，如[[北宋|北宋]][[洪兴祖|洪兴祖]]《[[楚辞补注|楚辞补注]]》、[[清代|清代]][[王先谦|王先谦]]《[[汉书补注|汉书补注]]》等。

== 章句 ==
'''章句'''是以句子为基本训释单位，将字词训释嵌入句子的直译之中，进而分析句读、串讲文章、探讨章旨的训诂体式<ref name=YYXMC />{{rp|160}}。“章句”是“离章辨句”的省称，着重于逐句逐章串讲、分析大意；不重解释词义，对字词的解释则隐含在句意的串讲之中。

汉代一些儒者治学从辨析章句入手，故章句体兴于汉。但汉儒用章句讲经大都支离烦琐，故被斥为“章句小儒”，一般人“羞为章句”。自汉以后，章句日汇渐亡佚。今仅存东汉[[赵岐|赵岐]]的《[[孟子章句|孟子章句]]》和[[王逸|王逸]]的《[[楚辞章句|楚辞章句]]》<ref name=WN />{{rp|285}}。

== 音义 ==
'''音义'''是以辨音释义为本，也兼及比勘文字形体的训诂体式<ref name=YYXMC />{{rp|160}}。名称有音训、音诂、音注、音解、音证、音隐、音释等，如唐代[[陆德明|陆德明]]的《[[经典释义|经典释义]]》中的《周易音义》、《尚书音义》等。

== 徵引 ==
'''徵引'''是以勾稽故实、征引出处的方法来探讨文献中的词语源流、说解语义和阐明文意的训诂体式<ref name=YYXMC />{{rp|160}}。如唐代[[李善|李善]]《[[文选注|文选注]]》。
