《'''古事記'''》（{{jpn|j=古事記}}<ref group="註">一般讀「{{jpn|hg = ふることふみ|rm = Pulukotopumi|kyu = |hgt = |link = no}}」、「{{jpn|hg = ふることぶみ|rm = Pulukotobumi|kyu = |hgt = |link = no}}」或「{{jpn|hg = こじき|rm = Kojiki|kyu = |hgt = |link = no}}」，而原讀未詳。</ref>），[[日本|日本]][[和銅|和銅]]四年（711年）9月18日，日本[[元明天皇|元明天皇]]命[[太安萬侶|太安萬侶]]（{{lang|ja|おお の やすまろ、太安 万侶}}）編撰日本古代史。[[和銅|和銅]]五年（712年）1月28日，將完成的內容《古事記》獻給[[元明天皇|元明天皇]]，為日本最早的歷史書籍。內容大略可分成：「本辭」、「帝紀」兩個項目，以及「上卷」、「中卷」、「下卷」三個部份。全書採用[[漢文|漢文]]寫成。

== 成書經過 ==
本书由太安万侶在712年呈献给元明天皇，记载了憑[[稗田阿禮|稗田阿禮]]口誦之《[[帝紀|帝紀]]》和《[[舊辭|舊辭]]》，以及一些歷代口述相傳的故事。

== 成書目的 ==
如同《古事記》的序中所提到，其編纂原因是為了清除虛構以及建立真實，就如同日本書紀一般，最終的是希望可以強化帝國專制的正當性而成書。《古事記》建立了一個親[[大和|大和]]的歷史敘事，結合將成為官方接受的帝國敘事，《[[日本書紀|日本書紀]]》，則是有助於確保歷史和神聖的合法性和一個能夠在上古生存下來的王朝的優越感。這段歷史敘事清楚地分為神的時代和人類帝王時代，其中誕生了土地的神靈被告知，且轉化成了具有統治正當性的[[天皇|天皇]]。這些敘述也清楚地表示了大和民族具有權力來統治這塊土地，然後這種神給予人類權利的概念大多都借鑑於中國。有些文內的敘述只是為了清除過去的失敗，以及建立過去皇帝的聲譽，例如提到了征服[[朝鮮帝國|朝鮮帝國]]一事。無論《古事記》的最初意圖如何，它最終確定並甚至可能制定了根據皇帝統治考察日本歷史的框架。

書中建構日本宇宙觀，認為宇宙根本是'''天之御中主神'''，生成宇宙的是'''高御產巢日神'''與'''神產巢日神'''<ref group="註">但是，《古事記》中没有記載這事，而只有這些三神的名稱耳。</ref>；日本諸島及島上的山川草木眾神由男神'''伊耶那岐神'''與女神'''伊耶那美神'''生產，並且生下支配日本列島與島上萬物的太陽女神'''天照大神'''，因此日本人自古以太陽民族自居，並視'''日本'''為神國。<ref name = "日本文化史_2012"> {{ cite book | title = 日本文化史 | url = https://books.google.com.tw/books?id=78XFMgEACAAJ&dq=inauthor:%22%E8%91%89%E6%B8%AD%E6%B8%A0%22&hl=zh-TW&sa=X | publisher = 遠足文化, 2012 | author = 葉渭渠 | isbn = 9789865967161 | language = zh-tw | date = 2012 }} </ref>

== 概說 ==
《古事記》大略可分成兩個項目、三個部份：
* 本辭（序言；[[神話|神話]]、[[傳說|傳說]]）
** 上卷：收錄序言、[[神話|神話]]、[[傳說|傳說]]
* 帝紀（1-33代的日本天皇）
** 中卷：收錄第1代天皇（[[神武天皇|神武天皇]]）到第15代天皇（[[應神天皇|應神天皇]]）的[[家譜|家譜]]、事跡。
** 下卷：收錄第16代天皇（[[仁德天皇|仁德天皇]]）到第33代天皇（[[推古天皇|推古天皇]]）的家譜、事跡。
