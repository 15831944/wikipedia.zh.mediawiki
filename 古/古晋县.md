'''古晋县'''（Kuching District）是[[砂拉越|砂拉越]][[古晋省|古晋省]]的其中一个县，总面积1,868.83平方公里<ref>[http://www.kuchingro.sarawak.gov.my/index.php?option=com_content&view=article&id=4&Itemid=2 古晋省公署]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }} 2011年更新. 2011年12月8日查阅</ref>。
<br />
古晋县下辖三个区：古晋, [[巴达旺|巴达旺]]（Padawan）和[[新生村|新生村]]（Siburan），总人口约有617,887人<ref>马来西亚国家统计局 [http://www.statistics.gov.my/portal/download_Population/files/census2010/Taburan_Penduduk_dan_Ciri-ciri_Asas_Demografi.pdf 2010年马来西亚人口和房屋普查：马来西亚人口分布] {{webarchive|url=https://web.archive.org/web/20150301154300/http://www.statistics.gov.my/portal/download_Population/files/census2010/Taburan_Penduduk_dan_Ciri-ciri_Asas_Demografi.pdf |date=2015-03-01 }} 2010年更新 [2011年12月8日查阅]{{ms}}及{{en}}</ref>。


== 行政区划 ==
古晋有三个副县，分别是古晋、巴达旺和新生村。
===古晋===
古晋主要分成两个区：
# 市区- 普遍称为[[古晋|古晋]]市。
# 郊区- 如:盐柴港（Sungai Maong），史戈都（Segedup），新渔村（Sungai Apong），马当（Matang）等。
===巴达旺===
巴达旺在1983年8月11日，宣布成为副县，副县政厅在藤布嘎（Teng Bukap）。<ref>[http://www.kuchingdo.sarawak.gov.my/page16.html 巴达旺副县] {{webarchive|url=https://web.archive.org/web/20110722233646/http://www.kuchingdo.sarawak.gov.my/page16.html |date=2011-07-22 }} 2011年12月8日查阅{{ms}}</ref>
===新生村===
现今的副县政厅在新生村[[巴剎|巴剎]]。新生村是古晋十七哩一带。新生村与巴达旺曾是同一个副县<ref>[http://www.kuchingdo.sarawak.gov.my/page15.html 新生村副县] {{webarchive|url=https://web.archive.org/web/20110722233426/http://www.kuchingdo.sarawak.gov.my/page15.html |date=2011-07-22 }} 2011年12月8日查阅{{ms}}</ref>。
