'''古罗马年表'''按照[[时间|时间]]先后顺序，列举[[古罗马|古罗马]]重要历史事件。时间跨度从古罗马建立至[[拜占庭帝国|拜占庭帝国]]最后试图收复罗马。

== 前8世纪至前6世纪 ==
* [[前753年|前753年]] － 相传，[[罗穆卢斯和瑞摩斯|罗穆卢斯和瑞摩斯]]在这一年建立[[罗马市|罗马市]]，标志'''[[罗马王政时代|罗马王政时代]]'''的开始（[[前509年|前509年]]结束），[[罗穆卢斯|罗穆卢斯]]成为第一任国王。
* 前753年/[[前715年|前715年]] － 国王罗穆卢斯在位。
* 前715年/[[前673年|前673年]] － 国王[[努玛·庞皮留斯|努玛·庞皮留斯]]（[[努马·庞皮里乌斯|努马·庞皮里乌斯]]）在位，创建了[[元老院|元老院]]和[[祭司团|祭司团]]。
* [[前672年|前672年]]/[[前641年|前641年]] － 国王[[托里斯·奥斯蒂吕斯|托里斯·奥斯蒂吕斯]]（[[圖路斯·荷提里烏斯|圖路斯·荷提里烏斯]]）在位。
* 前641年/[[前616年|前616年]] － 国王[[安库斯·玛尔提乌斯|安库斯·玛尔提乌斯]]（[[安古斯·馬奇路斯|安古斯·馬奇路斯]]）在位。
* 前616年/[[前575年|前575年]] － 国王[[卢修斯·塔克文·布里斯库|卢修斯·塔克文·布里斯库]]（Lucius Tarquinius Priscus）在位。
* 前575年/[[前535年|前535年]] － 国王[[塞尔维乌斯·图利乌斯|塞尔维乌斯·图利乌斯]]（[[塞爾維烏斯·圖利烏斯|塞爾維烏斯·圖利烏斯]]）在位。
* 前535年/[[前509年|前509年]] － 国王[[卢修斯·塔克文·苏佩布|卢修斯·塔克文·苏佩布]]（[[卢基乌斯·塔奎尼乌斯·苏培布斯|卢基乌斯·塔奎尼乌斯·苏培布斯]]）在位。
* [[前509年|前509年]] － 国王[[卢修斯·塔克文·苏佩布|卢修斯·塔克文·苏佩布]]被驱逐，'''[[罗马共和国|罗马共和国]]'''建立（[[前27年|前27年]]结束）。

== 前3世纪 ==
* [[前283年|前283年]] － 击败[[伊特鲁里亚人|伊特鲁里亚人]]和柏依人于[[瓦迪莫湖之战|瓦迪莫湖之战]]
* [[前280年|前280年]]/[[前275年|前275年]] － 与[[伊庇鲁斯|伊庇鲁斯]]国王[[皮洛士|皮洛士]]的战争。
* [[前267年|前267年]] － [[财务官|财务官]]的人数从4升至6人。
* [[前264年|前264年]]/[[前241年|前241年]] － [[第一次布匿战争|第一次布匿战争]]对抗[[迦太基|迦太基]]，战事主要在[[西西里岛|西西里岛]]及附近海域进行。
* [[前242年|前242年]] － 创立[[裁判官|裁判官]]（praetor）。
* [[前241年|前241年]] － 罗马和迦太基签定和约。西西里为罗马所有。并且迦太基割让“西西里与意大利之间的岛屿”，不包括[[撒丁岛|撒丁岛]]和[[科西嘉岛|科西嘉岛]]。迦太基赔偿罗马3200他连特。
* [[前238年|前238年]] － [[佣兵战争|佣兵战争]]中的迦太基形势危急。罗马接受迦太基的撒丁岛守军于去年提出的投降要求，进占撒丁岛。
* [[前237年|前237年]] － 迦太基胜利结束佣兵战争。要求罗马归还撒丁岛。罗马以武力威胁。精疲力尽的迦太基被迫屈服。罗马正式吞并撒丁岛和科西嘉岛。
* [[前227年|前227年]] － 财务官的人数从6升至8人；大法官人数从2升至4人。初次设置行省：西西里行省和撒丁行省（包括科西嘉岛）。
* [[前224年|前224年]] － 击败入侵的[[高卢人|高卢人]]
* [[前223年|前223年]] － 击败[[山南高卢人|山南高卢人]]（Cisalpine Gaul）
* [[前218年|前218年]]/[[前201年|前201年]] － [[第二次布匿战争|第二次布匿战争]]对抗[[迦太基|迦太基]]
* [[前216年|前216年]] － [[汉尼拔|汉尼拔]]在[[坎尼战役|坎尼战役]]中给罗马造成了灾难
* [[前214年|前214年]]/[[前205年|前205年]] － [[第一次马其顿战争|第一次马其顿战争]]。战事在希腊半岛进行，罗马主要以其希腊盟友与马其顿作战。前205年的和约规定恢复战前状态。
* [[前213年|前213年]]/[[前211年|前211年]] － 包围并进占[[叙拉古|叙拉古]]
* [[前204年|前204年]]/[[前202年|前202年]] － [[大西庇阿|大西庇阿]]（Scipio Africanus，意为阿非利加的征服者）侵入阿非利加；汉尼拔被召回，于公元前202年[[扎马战役|扎马战役]]败于西庇阿之手
* [[前202年|前202年]]/[[前196年|前196年]] － [[第二次马其顿战争|第二次马其顿战争]]，罗马胜利

== 前2世纪 ==
* [[前197年|前197年]] － 
** 远、近西班牙成为罗马行省
** 财务官的人数从8升至12人；大法官人数从4升至6人
* [[前192年|前192年]]--[[前189年|前189年]] － 对[[塞琉西王朝|塞琉西王朝]]的叙利亚战争
* [[前172年|前172年]]/[[前167年|前167年]] － [[第三次马其顿战争|第三次马其顿战争]], 罗马得胜
* [[前154年|前154年]]/[[前138年|前138年]] － 对吕西坦尼亚的战争
* [[前149年|前149年]]/[[前146年|前146年]] － [[第三次布匿战争|第三次布匿战争]]对抗[[迦太基|迦太基]]
* [[前149年|前149年]]/[[前148年|前148年]] － [[第四次马其顿战争|第四次马其顿战争]]
* [[前146年|前146年]] － [[西庇阿·伊密连那斯|西庇阿·伊密连那斯]]（Scipio Aemilianus）以[[迦太基|迦太基]]与[[科林斯|科林斯]]城的毁灭结束了[[布匿战争|布匿战争]]和[[马其顿战争|马其顿战争]]； [[马其顿|马其顿]]和[[阿非利加|阿非利加]]行省的建立
* [[前133年|前133年]] － 保民官[[提比略·革拉古|提比略·革拉古]]因颁布土地法案被杀害
* [[前121年|前121年]] － 
** 罗马征服并建立[[山外高卢|山外高卢]]（Transalpine Gaul）行省，以确保到西班牙通路的安全
** 元老院颁布首部[[List_of_Roman_laws|Senatus consultum de re publica defenda]]以平息由保民官[[盖约·革拉古|盖约·革拉古]]引起的暴动威胁
* [[前112年|前112年]]/[[前106年|前106年]] － 对[[努米底亚|努米底亚]]国王[[朱古达|朱古达]]（Jughurta）的战争，以[[马略|马略]]（Marius）的胜利告终
* [[前105年|前105年]] － [[西姆布赖|西姆布赖]]（Cimbri）人入侵，重创罗马军团于[[battle_of_Arausio|battle of Arausio]] 
* [[前104年|前104年]]/[[前100年|前100年]] - [[盖乌斯·马略|盖乌斯·马略]]（Gaius Marius） 被选连任五年的执政官
* [[前102年|前102年]] - 马略击败[[条顿|条顿]]人（Teutons）于[[Battle_of_Aquae_Sextae|Battle of Aquae Sextae]]
* [[前101年|前101年]] - 马略和[[昆塔斯·琉喜阿斯·卡塔拉斯|昆塔斯·琉喜阿斯·卡塔拉斯]]（Quintus Lutatius Catulus）击败西姆布赖人于the [[Battle_of_Vercellae|Battle of Vercellae]]

== 前1世纪 ==
* [[前91年|前91年]]/[[前88年|前88年]] － [[同盟者戰爭_(前91到88年)|同盟战争]]（Social wars），末次意大利同盟国对罗马的叛乱
* [[前88年|前88年]] [[苏拉|苏拉]]统军攻入罗马
* [[前88年|前88年]]/[[前85年|前85年]] － 对[[本都|本都]]国王[[米特里達梯六世|米特里達梯六世]]的[[第一次米特里達梯战争|第一次米特里達梯战争]]
* [[前83年|前83年]]/[[前82年|前82年]] － 苏拉与罗马民主党间的第一次内战；苏拉获胜，实行独裁统治；废黜独裁官一职（[[前70年|前70年]]恢复）
* [[前83年|前83年]]/[[前82年|前82年]] － [[第二次米特里達梯战争|第二次米特里達梯战争]]；苏拉回到罗马，任[[独裁官|独裁官]]
* [[前82年|前82年]]/[[前72年|前72年]] － [[塞多留|塞多留]]（Sertorius），最后的马略党人，在西班牙延续内战
* [[前74年|前74年]]/[[前66年|前66年]] － [[第三次米特里達梯战争|第三次米特里達梯战争]]以[[格涅乌斯·庞贝|庞培]]（Pompey）最终获胜
* [[前73年|前73年]]/[[前71年|前71年]] － [[斯巴达克斯起义|斯巴达克斯起义]]，最终被[[克拉苏|克拉苏]]镇压
* [[前67年|前67年]] － 庞培清剿地中海海盗
* [[前63年|前63年]] － 
** 耶路撒冷陷落
** [[西塞罗|西塞罗]]任执政官；粉碎平民派领袖[[喀提林|喀提林]]企图夺权的[[喀提林的阴谋|喀提林的阴谋]]
* [[前59年|前59年]]/[[前54年|前54年]] － [[前三头|前三头]]政治；恺撒、庞培和[[克拉苏|克拉苏]]缔结秘密同盟
* [[前58年|前58年]]/[[前50年|前50年]] － 恺撒发动[[高卢战争|高卢战争]]攻取外高卢
* [[前54年|前54年]]/[[前53年|前53年]] － 首次对[[帕提亚帝国|帕提亚帝国]]作战，克拉苏败死
* [[前44年|前44年]] － [[恺撒|恺撒]]遇刺身亡

== 1世纪 ==
* [[6年|6年]] － [[犹大|犹大]]变成罗马的一个行省。
* [[37年|37年]] －'''[[卡里古拉|卡-{里}-古拉]]'''成为皇帝
* [[41年|41年]] －'''[[克劳狄|克劳狄]]'''成为皇帝
* [[54年|54年]] －'''[[尼禄|尼禄]]'''成为皇帝
* [[79年|79年]] － 
** '''[[提圖斯|提圖斯]]'''成为皇帝
** [[8月24日|8月24日]]，[[维苏威火山|维苏威火山]]爆发毁灭了[[庞贝城|庞贝城]]和[[赫库兰尼姆|赫库兰尼姆]]
* [[81年|81年]] － '''[[多米田王|多米田]]'''（又译作[[图密善|图密善]]）成为皇帝
* [[96年|96年]] － [[多米田王|多米田王]]被元老院派人刺殺，[[夫拉維王朝|夫拉維王朝]]結束。元老院任命[[涅尔瓦|涅尔瓦]]继任，开始了[[五賢帝時代|五賢帝時代]]。
* [[98年|98年]] － '''[[图拉真|图拉真]]'''成为皇帝
