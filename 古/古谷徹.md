'''古谷徹'''（{{bd|1953年|7月31日|||Furuya, Tooru}}）是[[日本|日本]][[動畫|動畫]]界的知名[[聲優|聲優]]與口述員。隸屬於[[青二Production|青二Production]]事務所。

== 經歷 ==
[[File:Tōru_Furuya_and_Lam_Pou_Chuen_in_ACHK_20060728.jpg|right]]於[[香港動漫節|香港動漫節]]上合照。林保全是香港版本[[機動戰士GUNDAM|機動戰士GUNDAM]][[阿姆羅·雷|阿姆羅·雷]]的第一代配音員]]
古谷徹出生於[[神奈川縣|神奈川縣]][[橫濱市|橫濱市]]，在少年時代就加入[[向日葵劇團|向日葵劇團]]進行演出活動，1967年在[[加山雄三|加山雄三]]主演的電影「{{lang|ja|続・何処へ}}」等電影當中演出，1968年擔任動畫[[巨人之星|巨人之星]]的主角星飛雄馬。雖然一般以為巨人之星是古谷徹的聲優出道作品，但實際上他第一部配音作品是「海賊王子」（小學六年級時）。演出巨人之星時升上國中的關係，在巨人之星結束後暫停聲優活動，直到從關東學院六浦高校畢業，進入[[明治學院大學|明治學院大學]]商學系之後才重新開始配音，復出後首部作品為[[鋼鐵吉克|鋼鐵吉克]]。

1979年古谷徹演出[[機動戰士GUNDAM|機動戰士GUNDAM]]的[[阿姆羅·雷|阿姆羅·雷]]，當初的各種名台詞現在仍然是愛好者間的話題，古谷徹清澈的音色也在廣泛的觀眾層當中享有高度的人氣。古谷徹其他的代表作還有[[聖鬥士星矢|聖鬥士星矢]]當中的星矢，[[七龍珠|七龍珠]]當中的[[飲茶_(七龍珠)|阿樂]]（{{lang|ja|ヤムチャ}}）<ref>{{Cite news|url=http://top.tsite.jp/news/anime-song/o/33654560/|title=ヤムチャ愛とネガティブが素晴らしい! 『DB超』ヤムチャ主役回を記念して古谷徹がコメントを発表!|last=|first=|date=2016年12月8日|work=|accessdate=T-SITE}}</ref>等。

在此之後與聲優同伴們結成樂團「{{lang|ja|スラップスティック}}」並與聲優[[小山茉美|小山茉美]]（初代鋼彈[[琪西莉亞·薩比|琪西莉亞·薩比]]，08小隊[[卡蓮·約書亞|卡蓮·約書亞]]，SEED DESTINY[[妲莉雅·古拉迪斯|妲莉雅·古拉迪斯]]）結婚，但後來離婚（卻還是維持友好關係）。現在與前聲優[[間嶋里美|間嶋里美]]再婚並育有一女。

古谷徹從20歲時代就對汽車有相當的愛好，自1986年起擔任「{{lang|ja|カーグラフィックTV}}」節目主述旁白至今將近20年，愛車為[[BMW|BMW]]525i雙凸輪軸。他同時也是電腦的強者，家中的網頁維護與個人開發的資料庫等等也是一手包辦，在線上遊戲[[鋼彈online|GUNDAM NETWORK OPERATION 2]]慕亞伺服器聯邦軍陣營擁有一個角色（於第六輪開始時引退）。

2007年10月15日，他在[[中国|中国]][[北京|北京]]出席个人首次声优见面会。

2008年獲得[[聲優獎|聲優獎]]富山敬賞（話題賞）。

== 古谷徹與鋼彈 ==
* 在GUNDAM NETWORK OPERATION推出當時曾經加入吉翁軍陣營，後來被聯邦玩家以及NPC的自己（阿姆羅）碾到惱羞而轉回聯邦軍（發表於日版[[GUNDAM_ACE|GUNDAM ACE]]人物小雜談）並自嘲說「這大概是我在某公開場合應觀眾要求喊出SIEG ZEON的報應吧（笑）」
* 古谷徹以前曾經說過「GUNDAM系列只願配阿姆羅·雷的聲音」<ref>「スペシャルインタビュー古谷徹」『新・大人のガンダム』日經BP社、2008年、p.4-8、ISBN 978-4-8222-6328-7</ref>，但自從[[池田秀一|池田秀一]]去配[[GUNDAM_SEED_DESTINY|GUNDAM SEED DESTINY]]的[[吉伯特·杜蘭朵|杜蘭朵議長]]之後有所動搖，後來被[[GUNDAM00|GUNDAM00]]監督[[水島精二|水島精二]]邀請擔任旁白，由於旁白不算角色，古谷便答應以自己的名義演出，但中途在導演的懇求外加技癢決定演出正式角色，但因不能違背自己曾公開說過的大話而化名'''蒼月昇'''並以「新人聲優」的名義來替[[GUNDAM00|GUNDAM00]]的里朋斯·阿爾馬克配音。古谷為了隱瞞自己的身分而天天帶著大太陽眼鏡遮大半臉進錄音室保持低調，且從不拿下眼鏡來示眾，就連雜誌媒體參加活動受訪問時也一樣。所以製作期間配音班底只是覺得這個新人「音質跟古谷徹非常像」罷了。一直到兩期正式播放完畢在G-Fes2009的鋼彈00活動的最後才公開真實身分震撼了全體聲優班底與大眾。
* 其蒼月的名稱來自於古谷在GUNDAM NETWORK OPERATION線上遊戲所屬的小隊名稱，分別為「久遠的蒼月」，「久遠蒼月隊」以及『UniversalCentury.net GUNDAM ONLINE』的「BlueMoon」隊。

== 其他小記事 ==
* 在2005年M.O.E製作動畫《[[極限女孩|極限女孩]]》當中演出UFO男一角有許多黃色笑話與黃色的表演，令古谷徹長年愛好者多感到不可思議。不過對於聽過他所主持的廣播節目「{{lang|ja|アクションカメラ}}」的聽眾群則多認為這是表演能力的一部份，沒什麼值得奇怪的。
*在得知曾經與他同台演出，阿姆羅的香港配音員林保全的離世消息後，古谷徹在其Twitter發表悼詞。<ref>[https://twitter.com/torushome/status/551202158775779329 ]</ref>
