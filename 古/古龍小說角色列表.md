==楚留香系列==
;[[楚留香|楚留香]]

;蘇蓉蓉
:[[楚留香|楚留香]]的紅顏知己，和[[李紅袖|李紅袖]]、[[宋甜兒|宋甜兒]]一樣自年少被楚留香收養。精於易容術，使得楚留香隨時都可以用各種不同的面貌和身份在江湖中出現。

;宋甜兒
:廣東人，說廣東話，楚留香的紅顏知己，和[[蘇蓉蓉|蘇蓉蓉]]、[[李紅袖|李紅袖]]一樣自年少被楚留香收養，三人一起住在楚留香的小船，常與[[李紅袖|李紅袖]]鬥嘴，擅長煮食。3名女子之中，宋甜兒活得最開心快活，知足常樂，對什麼事也不在乎，容易滿足。

;李紅袖
:楚留香的紅顏知己，博聞強記，對武林歷史典故瞭如指掌。

;[[胡鐵花_(小說)|胡鐵花]]

; 姬冰雁
: 與[[胡鐵花|胡鐵花]]、[[楚留香|楚留香]]齊名，被稱為「雁蝶為雙翼，花香滿人間」。
: 起初因[[高亚男|高亚男]]之故而孤身远走大[[沙漠|沙漠]]，忍受大漠无情的风沙，凭借自身的精明和强锐，十年艰辛创业，成为[[兰州|兰州]]巨富。
: 其人生性冷傲，不喜多言，亦有几分自私，因而被戏称为“死公鸡”。
: 然却因楚留香一个请求而甘愿抛下亿万家财，再赴大沙漠，其中凶险万分却毫无畏惧，“雁蝶为双翼，花香满人间”果不虚言。
: 似乎也是[[铁血大旗门|铁血大旗门]]的弟子。
: 姬冰雁不仅尊敬[[食物|食物]]，更尊敬[[水|水]]，而他对吃的欲望，已远远超出了对“[[肉|肉]]”的追求，达到一种“惨烈”的境界。姬冰雁有个庞大而舒适的[[马车|马车]]，马车里有两个抽屉：一个装满了[[酒|酒]]，“十个用白银铸成的方瓶子装有十种酒，从[[茅台|茅台]]，[[大曲|大曲]]，[[竹叶青|竹叶青]]，到[[关外|关外]][[羊乳酒|羊乳酒]]都有，瓶子着来虽不大，却可装得下三斤十二两”；另一个却装满了下酒菜，“里面不但有[[江北|江北]]的[[大虾米|大虾米]]，[[金华|金华]]的[[火腿|火腿]]，还有[[福州|福州]][[糟鱼|糟鱼]]，[[福州|福州]][[烧鹅|烧鹅]]，[[海宁|海宁]][[海臭虫|海臭虫]]，[[无锡|无锡]][[肉骨头|肉骨头]]，[[长白山|长白山]]的梅花[[熊掌|熊掌]]……总之，只要你想得出来最好吃的下酒菜，这抽屉里就有”。生活上向来顾头不顾腚的胡铁花对此惊呼：“你这是在变魔术！”而姬冰雁却淡淡哦解释道：为了积累财富，他在途中不知道吃了多少苦，曾有一次饿得恨不得把自己的手剁下来吃，因此他大富之后，无论在哪里，总要先将那里堆满了吃的东西，甚至在他睡觉的床下面，都永远有酒有肉。
: 无论这位朋友有怎样的沉默与怪癖，无论他作出怎样的选择和行为，他总能指望得到楚留香的理解和尊重。在《大沙漠》故事中，楚留香早已发现姬冰雁并未瘫痪，只是不想去大沙漠冒险，楚留香非但没有责怪，反而劝说胡铁花：“你要交一个朋友，就得了解他的脾气，他若有缺点，你应该原谅他，我认识他的时候，就已知道他是个这样的人了，我为何还要生气……。”
: “能令这样的人始终将我当朋友，我已很满足了。”结果姬冰雁果然没有让人失望。其中包含了楚留香交友之道的另一条宝贵原则，那就是对朋友的个性特征及其行为方式的谅解和宽容。

;[[無花|無花]]
:表面上是風雅德高的[[少林|少林]]妙僧，實為詭詐善變的小人。為[[東瀛|東瀛]]高手天楓十四郎與[[石觀音|石觀音]]的長子。曾假死騙過楚香帥，後在大沙漠被香帥制伏，死於畫眉鳥之手。

;[[中原一點紅|中原一點紅]]
:由薛笑人一手养大，武功也从他身上学，在传说中，一點红是天下索价最高，出手最狠，最有信用的杀手。

;[[石觀音|石觀音]]
:據點位於大沙漠中的女魔頭，武功高深莫測,招式巧妙的絕頂高手，連功力高強的香帥都無法招架，絕招「男人見不得」。原名李琦，為[[黄山|黄山]]世家唯一遺孤。後於[[東瀛|東瀛]]習武學成，盡滅[[華山派|華山派]]報仇。

;[[水母陰姬|水母陰姬]]
:神水宮宮主，擁有在水中練出來的無匹內力。為[[女同性戀|女同性戀]]者，但與亦男亦女的雄娘子育有一女。與香帥交手時因在水中不能換氣而敗。

: 司空摘星，[[古龍|古龍]]小說《[[陸小鳳傳奇|陸小鳳傳奇]]》中的角色，[[小偷|小偷]]，貌似[[猴精|猴精]]，長於盜術、易容術，江湖人稱'''偷王之王'''，是[[陸小鳳|陸小鳳]]的朋友。與陸曾出生入死，卻又數度交鋒，雖互有輸贏，卻讓陸頭痛不已，為陸唯一剋星。後人尊其為'''盜仙'''。

: 他是陸小鳳的朋友，可以接受陸小鳳調侃，但是他決不會讓陸小鳳干涉他偷的藝術，更不會因為誰而改變其偷的原則。
: 據《[[白玉老虎|白玉老虎]]》一書提及，司空摘星徒弟為郭雀兒<ref>《[[白玉老虎|白玉老虎]]》第七章〈虎山行〉第六節〈散花天女〉</ref>。

==跨作品人物==
;李尋歡
其小李飛刀百發百中，從不落空，曾殺死金錢幫幫主上官金虹，在百曉生的兵器譜上排名第三。他的傳人為葉開。首見於作品《多情劍客無情劍》之中。其後雖不再在小說登場，卻仍在其他古龍小說中被別的人物提及，名字可見於《邊城浪子》、《天涯‧明月‧刀》、《飛刀，又見飛刀》等作品。
;顧道人
:隐居[[巴山|巴山]]的武林名宿，巴山顧道人的名字雖然經常在古龍小說出現，卻從未登場，古龍也沒有詳細交代他的事跡。
:劍名叫「绿柳」，昔年顾道人曾以[[七七四十九手回風舞柳劍|七七四十九手回風舞柳劍]]縱橫天下，死在这柄剑下的成名剑客，也不知多少，後來飄然隱去、不知仙踪。
:曾经被醉心于剑的年轻人们奉为圣地的[[巴山|巴山]]道观，也已渐渐荒冷没落，所剩下的，惟有一些神话般的传说，和台上的一道剑痕空留凭而已。 
:曾經收下數個子弟，並在不同古龍小說中登場，如[[楚留香|楚留香]][[蝙蝠傳奇|蝙蝠傳奇]]中的[[柳吟松|柳吟松]]、[[蕭十一郎|蕭十一郎]]的[[柳色青|柳色青]]、[[陸小鳳|陸小鳳]][[劍神一笑|劍神一笑]]的[[柳乘風|柳乘風]]，在劍神一笑中提到，巴山顧道人隱去的時候，他的弟子也已四散。
:在陸小鳳[[劍神一笑|劍神一笑]]中提到「七七四十九手回風舞柳劍」是一套需要用輕功配合的劍法。
:除了顧道人之外，上面提及之巴山派弟子也習此劍。
:在[[楚留香傳奇|楚留香傳奇]][[大沙漠|大沙漠]]中[[高亞男|高亞男]]在飲醉的時候曾使出，在[[蝙蝠傳奇|蝙蝠傳奇]]中，回風舞柳劍的劍訣落在蝙蝠島主人[[原隨雲|原隨雲]]手上。

;“人中之龙”海天孤燕
:古龙小说中的神秘人，居於黃海一個孤島上，曾经赤手空拳，连败中原武林各门各派的二十七个掌门人，每个人在他手下都未曾走满十招，被武林尊为千百年来第一人的奇人。

;王憐花
: [[古龍|古龍]]筆下的虛構武林人物，江湖人稱'''千面公子'''，為《[[武林外史|武林外史]]》的一名重要角色。玉面朱唇，風流可人，扮女子亦勾魂盪魄，然最誘人處在其正邪、真假的難辨。其後雖不再在小說登場，卻仍在其他古龍小說中被別的人物提及，如《[[邊城刀聲|邊城刀聲]]》（證實非古龍所著）、《[[多情劍客無情劍|多情劍客無情劍]]》等作品。
: 王憐花是[[快活王|快活王]]與[[雲夢仙子|雲夢仙子]]的兒子，他要報仇，他的仇人是他父親。
: 在《[[多情劍客無情劍|多情劍客無情劍]]》中，'''王憐花'''把一生所學之[[武功|武功]]及[[醫術|醫術]]、下[[毒|毒]]、[[易容|易容]]、放[[蠱|蠱]]、[[攝心術|攝心術]]等雜學寫成《憐花寶鑑》，想交給[[李尋歡|李尋歡]]，卻落在[[林詩音|林詩音]]手中，最後[[龍嘯雲|龍嘯雲]]欲獻予[[上官金虹|上官金虹]]，卻未取出已被殺；《憐花寶鑑》遂與龍嘯雲的屍體自此消失。
