《'''恋爱前规则'''》（{{lang-en|'''My Airhostess Roommate'''}}）中国大陆爱情片、[[青春片|青春片]]、喜剧片，改编自三十《[[和空姐同居的日子|和空姐同居的日子]]》，导演[[蒋钦民|蒋钦民]]、美术设计[[孟禹|孟禹]]，主演王珞丹、陈柏霖、周奇奇。<ref>{{cite web |url=http://gb.cri.cn/27564/2013/04/11/1326s4081289.htm |title=《恋爱前规则的海报》 |accessdate=2013 |publisher= 国际在线|work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/c/2009-11-12/10372766971.shtml |title=陈柏霖王珞丹袒露真实《恋爱前规则》(图) |accessdate=2009 |publisher=新浪网 |work= |language=zh-hans}}</ref>

==剧情==
该片讲述了21世纪青年“陆飞”、“冉静”、“乐乐”追求爱情和幸福人生的故事。

==花絮==
该片曾是一周的票房冠军，首日票房就破了100万元人民币。<ref>{{cite web |url= http://ent.sina.com.cn/m/c/2009-11-05/01442757102.shtml|title= 《恋爱前规则》首日票房破百万 王珞丹拒绝转型|accessdate=2009 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/c/2009-11-18/13212774343.shtml |title=《恋爱前规则》上周票房夺国产电影类冠军(图) |accessdate=2009 |publisher=新浪网 |work= |language=zh-hans}}</ref>

该片标志着王珞丹的转型，从天真浪漫到成熟稳重。<ref>{{cite web |url=http://ent.qq.com/a/20091112/000102.htm |title= 《恋爱前规则》全国热映 王珞丹转型惹争议|accessdate=2009 |publisher=腾讯网 |work= |language=zh-hans}}</ref>

该片-{zh-hans:里;zh-hant:裡;}-，王珞丹出演的女主角“冉静”上演了“女追男”的故事。<ref>{{cite web |url=http://ent.sina.com.cn/r/i/2009-11-11/10152765629.shtml |title= 《恋爱前规则》：倒贴女王珞丹顺利“脱光”|accessdate=2009 |publisher=新浪网 |work= |language=zh-hans}}</ref>
