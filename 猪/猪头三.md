'''猪头三'''（{{Sha|p1=zzydhouse|ipa=[tsɿ̋dɤ̄sᴇ᷆]|ref=1}}）是[[吴语|吴语]]中的常用[[脏话|脏话]]，表示愚蠢、笨头笨脑之义。

在粵語中，豬頭三又名「豬頭丙」。<ref>[http://www.dgslw.net/a/yuanchuangshalong/2014/0603/9004.html “猪头丙”的由来]</ref>

== 来源 ==
“猪头三”一词来源于“猪头三——[[牺牲|牲]]”这一[[歇后语|歇后语]]（缩脚韵）。所谓“三牲”，是指旧时[[祭祀|祭祀]]时所用的猪头、雄鸡、青鱼的合称，因为以猪头为首，故称作“猪头三牲”。吴语中以“猪头三”来歇后“牲”，是骂人“畜牲”之意，后来又泛指笨头笨脑之人。 <ref name="wzx">{{cite book|title=《上海俗语图说》|author=汪仲贤|publisher=上海大学出版社|year=2004}}</ref>也有人认为吴语中“牲”通“生” （吴语中两字同音，都读{{IPA-wuu|sã}}），最初是用来指初来上海而对城市生活陌生的人，后来才引申为今义。<ref name="xly">{{cite book|title=《上海闲话碎语》|author=薛理勇|publisher=上海辞书出版社|year=2005}}</ref>

“猪头三”一词还曾派生出“猪头四”、“者头三”（“者”字起笔为“土”，用其“土头土脑”之义）等词，但现都已不再使用。<ref name="xly" />

「豬頭丙」在粵語中又名「豬頭炳」。二次大戰後上海人移居香港，粵語吸收了滬語詞「豬頭三」,又將「阿炳」(多指草根人物)融合成「豬頭炳」。<ref>[http://silverylines.blogspot.ca/2016/10/blog-post_11.html 從「雞」和「豬頭炳」看粵語之生猛 ]</ref>

== 使用 ==
“猪头三”在吴语中是形容人愚蠢、戆头戆脑、不明事理、不识好歹等义，语气比起“[[瘪三|瘪三]]”、“众牲”等其他吴语訾语较为缓和<ref name="wzx" />，尤为女性所惯用<ref name="scz">{{cite journal|title=上海话中的“猪”|author=申持中|journal=食品与生活|year=2004|issue=3}}</ref>。

“不拿白不拿”、“不吃白不吃”等官话俗语，吴语中多说成“有拿勿拿猪头三”、“有吃勿吃猪头三”，都属“猪头三”一词的常见用法。<ref name="scz" />

在网络语言中，则经常会使用缩写“ZT3”来表示“猪头三”。<ref>{{cite journal|title=从网络旧词新用看名词的语法功能|author=陆月花|journal=文教资料|year=2011|issue=2}}</ref>
