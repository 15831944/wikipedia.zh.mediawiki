'''猪帶绦虫'''（學名：{{lang|la|''Taenia solium''}}；pork tapeworm），也称'''有钩绦虫'''或'''鏈狀帶絛蟲'''，体长2-3米，宽7-8毫米，共有800-900个[[节片|节片]]，后端成熟节片长约10毫米。

== 特徵 ==
*成蟲體分為四部分：[[頭節|頭節]]（scolex），[[未成熟節片|未成熟節片]]（immature proglottids），[[成熟節片|成熟節片]]（mature proglottids）和[[妊娠節片|妊娠節片]]（gravid proglottids）。
*頭節圆球形，直径约1毫米，上有四个吸盘并有顶突和兩圈小钩（armed hooks)，這區別於[[牛肉絛蟲|牛肉絛蟲]]（''Taenia saginata''）頭節的鉤（hooks）（牛肉絛蟲只有一排鉤（hooks），而且不帶齒(not armed)）。頭節既保證蟲體能夠附著在小腸內壁，而且頭節末端的頸部也是蟲體節片的分生區。
*蟲體無口無肛，靠節片體表吸收[[小腸|小腸]]內半消化食物。
*成熟節片即由頭節分生的未成熟節片長成。一個節片內同時長有雄性和雌性生殖系統，並開始排精。受孕後即是妊娠節片。
*妊娠節片的[[子宮|子宮]]有7到13個分支，這區別於牛肉絛蟲的15到20個子宮分支（uterine branches）。子宮分支內充滿了受精卵，卵開始從節片旁邊的孔排出。同時妊娠節片也會脫落，因此在馬桶裡可以看到大約5mm×5mm的白色扁平節片。

== 生活史 ==
[[File:Cysticercosis_by_Taenia_solium_PHIL_3387_lores.jpg|thumb]]

*[[人|人]]是唯一的[[最終宿主|最終宿主]]，[[寄生|寄生]]於[[小腸|小腸]]內，引起豬肉絛蟲病，[[中間宿主|中間宿主]]主要是[[豬|豬]]，人同時也可以成為異常中間宿主（abnormal intermediate host）。
*一般情況下，豬在吃了有絛蟲卵的食物後，絛蟲卵在豬體內[[胃酸|胃酸]]和[[酶|酶]]的作用下開始孵化，生出六鉤蚴（hexacanth）。六鉤蚴接著進入豬的循環系統，最終在肌肉組織、腦組織等地方停留，發育成[[囊尾蚴|囊尾蚴]]，並被豬組織包圍形成一個囊（cyst）。當人吃了未煮熟的有絛蟲囊尾蚴的豬肉（通稱「米心肉」）後感染，囊尾蚴在人體小腸發育成豬肉絛蟲成蟲，頭節掛在小腸壁以吸取營養和並分生出節片以繁殖後代。
*特殊情況下，人偶然進食了豬肉絛蟲卵，蟲卵發育出的六鉤蚴可能通過人體循環系統迴圈到腦組織或者眼球裡發育成包囊，危及人體健康甚至生命。（[[牛肉絛蟲|牛肉絛蟲]]不會有這條異常生活史）

== 感染症狀及危害 ==
*感染豬肉絛蟲成蟲，人體會面黃肌瘦，無力等。
*感染囊蚴，如果囊是在腦，人會出現[[癲癇|癲癇]]，如果數量過多，會死亡。如果囊蚴寄生在[[眼球|眼球]]，會出現[[失明|失明]]。此為「{{le|腦囊蟲病|Neurocysticercosis}}」(Neurocysticercosis)。

== 診斷 == 
*檢查[[糞便|糞便]]，看有沒有蟲卵和妊娠節片。
*做生物化學檢查抗體。
