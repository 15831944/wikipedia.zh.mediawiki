== 养猪生产情况 ==
[[File:Hog_confinement_barn_interior.jpg|thumb]]
*世界养猪生产的情况
:根据[[FAO|FAO]]的统计，2005年全世界的[[猪肉|猪肉]]产量为105,046.36千吨。<ref>{{cite web|url=http://faostat.fao.org/site/336/DesktopDefault.aspx?PageID=336 |title=FAOSTAT |accessdate=2007-12-03 |work=all core data |publisher=FAO }}</ref>
*中国养猪生产的情况
:[[中国|中国]]是[[养猪|养猪]]第一大国，2005年[[中国|中国]][[生猪|生猪]][[存栏|存栏]]占全世界产量的50.90%。<ref>{{cite web |url=http://www.agri.gov.cn/sjzl/nynj/NYNJ200602.pdf |title=各国大牲畜存栏量 |accessdate=2007-12-03 |format=PDF |work=中国农业年鉴2006 |publisher=中华人民共和国农业部 |deadurl=yes |archiveurl=https://web.archive.org/web/20081230081837/http://www.agri.gov.cn/sjzl/nynj/NYNJ200602.pdf |archivedate=2008-12-30 }}</ref> 根据[[中国农业部|中国农业部]]的统计，[[中国|中国]]2005年末和2006年末的[[生猪|生猪]][[存栏|存栏]]分别为503,348千头和494,407千头。<ref>{{cite web|url=http://www.agri.gov.cn/sjzl/2006/197.htm|title=全国牲畜年末存栏头数增减情况|accessdate=2007-12-03|work=历年全国农业统计提要|publisher=中华人民共和国农业部|deadurl=yes|archiveurl=https://web.archive.org/web/20080313114336/http://www.agri.gov.cn/sjzl/2006/197.htm|archivedate=2008-03-13}}</ref> 而中国2005年末和2006年末的生猪出栏数分别为660,986千头和680,504千头。中国2005年末和2006年末的猪肉产量分别为50,106千吨和51,972千吨，分别占肉类总产量的64.71%和64.55%。<ref>{{cite web|url=http://www.agri.gov.cn/sjzl/2006/195.htm|title=全国畜产品产量增减情况|accessdate=2007-12-03|work=历年全国农业统计提要|publisher=中华人民共和国农业部|deadurl=yes|archiveurl=https://web.archive.org/web/20080906193203/http://www.agri.gov.cn/sjzl/2006/195.htm|archivedate=2008-09-06}}</ref>
:中国目前养猪户仍以散养户为主。在中国，99.73%的养猪户年出栏量低于1,000头，却提供了全国猪肉的80.2%<ref name="harmon">{{cite journal|title=中美两国养猪业比较|journal=规模养猪|date=2007-11-01|author=哈默|coauthors= |volume=2007|issue=6|pages=5|id= |url=|format=|accessdate=2007-12-18 }}</ref>。中国2012年人均猪肉消费量为38.7kg，相当于2009年奥地利和香港的59%，德国的71%，台湾的93%。
*台湾养猪业
:生猪存栏数约600万头，年屠宰头数1000万头左右，人均占有猪肉60kg，人均消费猪肉40kg。每头母猪年提供商品猪14头。养殖规模户均千头以上。<ref>“考察台湾养猪业的感受”,《养猪》2010年第2期</ref>
*美国养猪生产的情况
:在美国，养猪业总量仅为中国的五分之一。散养户的养殖数量不低于200头，这类养猪场约占美国猪场的65%。年出栏量低于1,000头的养殖场占85%，但产量不到全美国的1%<ref name="harmon"/>。
*丹麦
:世界最大的猪肉出口国家。年产商品猪2200万头，80%出口。是日本猪肉的主要进口来源地。

== 猪的经济类型 ==
=== 食用 ===
根据食用习惯和市场需求的不同，一般可分为[[脂肪型|脂肪型]]、[[瘦肉型|瘦肉型]]和[[肉脂兼用型|肉脂兼用型]]。
*脂肪型
:也称脂用型。此类型所产[[脂肪|脂肪]]较多，中国多数本地品种均属于此类型。
*瘦肉型
:也称肉用型。这种猪[[屠宰|屠宰]]后的[[瘦肉|瘦肉]]较多，[[脂肪|脂肪]]较少。[[长白猪|长白猪]]、[[约克夏猪|约克夏猪]]、[[杜洛克猪|杜洛克猪]]和[[汉普夏猪|汉普夏猪]]都属于此类型。中国的《瘦肉型猪品种(系)鉴定和验收标准》规定瘦肉型猪的[[胴体瘦肉率|胴体瘦肉率]]应在55%以上，平均[[背膘厚度|膘厚]]3.0cm以下。<ref>{{cite web|url=http://www.meatc.com/law/lawyer_detail.asp?fid=52 |title=瘦肉型猪品种(系)鉴定和验收标准 GB 8468-87 |accessdate=2007-12-03 }}</ref>
*肉脂兼用型
:也称兼用型。这种类型介于脂肪型和瘦肉型之间。

豬隻的各部份也可以加工，作為各項用品的製作原料<ref>[[香港科學館|香港科學館]]內[[食品科學|食品科學]]廳展板</ref>。
*豬[[腺體|腺體]]：
:可加工用於製造[[藥物|藥物]]；
*豬[[脂肪|脂肪]]：
:可加工用於製造[[炸藥|炸藥]]、[[肥皂|肥皂]]、[[蠟燭|蠟燭]]；
*豬[[骨骼|骨骼]]：
:可加工用於製造[[膠水|膠水]]、[[骰|骰]]、[[鈕扣|鈕扣]]、[[瓷器|瓷器]]；
*豬[[軟骨|軟骨]]：
:可加工用於製造[[雪糕|雪糕]][[添加劑|添加劑]]、[[膠卷|膠卷]]、[[膠囊|膠囊]]；
*豬[[皮|皮]]：
:可加工用於製造[[體育|體育]]用品、[[器官移植|移植]][[皮膚|皮膚]]；
*豬[[體毛|毛]]：
:可加工用於製造豬[[鬃毛|鬃毛]]刷、[[床褥|床褥]]、[[汽車|汽車]]{{le|坐墊|cushion}}、[[降落傘|降落傘]]坐墊、[[棒球|棒球]][[手套|手套]]；
*豬[[蹄|蹄]]：
:可加工用於製造[[石膏|石膏]]緩凝劑；
*豬[[血|血]]：
:可加工用於製造[[鞋油|鞋油]]、[[肥料|肥料]]；
*豬[[腸|腸]]：
:可加工用於製造[[油灰|油灰]]（[[玻璃|玻璃]]{{le|膩子|Putty}}）；

==猪场生物安全==
在养猪生产中，[[生物安全|生物安全]]是最为重要的问题之一。

===可能存在的危害===
*鼠害
:[[老鼠|老鼠]]是重要的[[传染源|传染源]]，因此猪场的灭鼠工作也极为重要。
*蚊子
:[[蚊子|蚊子]]也是许多病毒的携带者，尤其在[[夏季|夏季]]。

===常见措施===
*避免饲养其它动物
:一般猪场内应避免饲养其它动物，如[[反刍动物|反刍动物]]、[[家禽|家禽]]、[[猫|猫]]等，以减少病原的传播。
*控制人员出入
:应严重控制人员的进出，尤其要减少外来人员及[[车辆|车辆]]的进入。
*严格消毒
:主要有日常[[消毒|消毒]]、物品[[消毒|消毒]]及进口人员的[[消毒|消毒]]。
*全进全出
:在实际生产中，常对整个[[猪场|猪场]]/[[猪舍|猪舍]]的[[猪|猪]]整批同时转入或转出。这样做的目的是避免不同批次的猪之间造成[[传染|传染]]。
