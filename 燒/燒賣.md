'''燒賣'''，又稱'''稍麦'''、'''燒麥'''、'''捎卖'''、'''稍美'''、'''乾蒸'''、'''烧梅'''<ref name="湖北小吃顺香居重油烧梅的做法详细介绍">{{cite web | url=http://www.ttmeishi.com/xiaochi/231d5d39e18cfac8.htm | title=湖北小吃顺香居重油烧梅的做法详细介绍 | accessdate=2012-04-06}}</ref>、'''肖米'''、'''鬼蓬頭'''<ref>形容頂端蓬鬆的形狀，引自《[[信報|信報]]》2005年11月09日、《飲情食趣》專欄，作者[[唯靈|唯靈]]，香港知名權威食評家。</ref>，是一種以[[燙麵|燙麵]]（以開水和成的半熟麵團）為皮帶餡上籠蒸熟的小吃。傳說源起於[[元代|元代]][[北京|北京]]，一說[[呼和浩特|呼和浩特]]。現在，燒賣在除内蒙古和[[北京|北京]]外，在[[中國東北|中國東北]]、[[天津|天津]]、[[山西|山西]]、[[山東|山東]]、[[湖北|湖北]]、[[江蘇|江蘇]]、[[浙江|浙江]]、[[福建|福建]]、[[廣東|廣東]]<ref>[http://www.fjsq.gov.cn/ShowText.asp?ToBook=38&index=32& 旅游志 二、小　吃]</ref>、[[香港|香港]]、[[台灣|台灣]]、[[澳門|澳門]]乃至[[日本|日本]]、[[越南|越南]]等地均可见到各种风味的烧卖。

燒賣用温水和面（燙麵），其后再加入冷水和麵，以增加成型能力。面团用稍美棰（一種中間粗兩頭有把類似棒槌的特殊擀麵杖）擀开，擀出的皮薄而不平，四邊如同花邊。中間放餡，并不用手包，由下向上一提即可成型。蒸熟。形若杯，底为圓，腰收細，頂部開口為花邊。大小通常約直徑1寸，以一口吃完為標準。宇宙大燒賣除外。

== 命名 ==
燒賣及其他與之谐音的名稱的来由有多種説法，没有定論。
===稍麥===
明代史料记载“北方麦子在四五月间，麦稍有一层白霜”。而稍麦在制作收口处，也有好似白霜的面粉，故而得名。

[[元朝|元]]末[[明朝|明]]初，[[高麗|高麗]]出版的漢語學習書籍《[[朴通事諺解|-{朴}-通事諺解]]》上記有元朝首都[[大都|大都]]有販子熟食「素酸餡稍麥」，是以麥麵製成的薄皮包著肉蒸熟，伴湯而食。在[[清代|清代]][[乾隆|乾隆]]年间，杨米仁在《都门竹枝词》中有“稍麦[[馄饨|馄饨]]列满盘，新添挂粉好[[汤圆|汤圆]]”的诗句。

===燒麥、燒賣===
明清时“稍麥”的称呼仍然被沿用，同时因谐音出现“燒麥”、“燒賣”。關於燒賣的記載包括有《[[扬州畫舫錄|扬州畫舫錄]]》、《[[桐橋椅棹錄|桐橋椅棹錄]]》，清朝《金瓶梅詞話》有“桃花燒賣”的記述<ref>見 [[:s:金瓶梅/第42回|《金瓶梅》第42回]]</ref>；[[清朝|清朝]]著名食譜《調鼎集》亦有收集菜式如“葷餡燒賣”、“豆沙燒賣”、“油糖燒賣”等。

传说清[[光绪|光绪]][[回族|回族]]人[[马玉亮|马玉亮]]所创的燒麥。初名'''撮子包'''，因感不雅，又因其边与将熟的麦穗相似，遂改名为“烧麦”。<ref>[http://hn.ifeng.com/meishi/henanmeishi/detail_2013_10/15/1334421_0.shtml 河南南阳著名小吃 方城美味烧麦]</ref>

===捎卖===
1937年成书的《缓远通志稿》载：“惟（[[歸化城|归化]]，即[[呼和浩特|呼和浩特]]）市内所售捎卖一种，则为食品中之特色，因茶肆附带卖之。俗语谓‘附带’为捎，故称'''捎卖'''。且归化捎卖，自昔驰名远近。外县或外埠亦有仿制以为业者。而风味稍逊矣。”

===燒梅===
湖北武漢附近稱“燒梅”。據說是因為燒賣皮是用[[走槌|走槌]]擀出形似梅花的邊，蒸出来如同一朵朵綻放的梅花，所以稱作“燒梅”。<ref name="重油烧梅馆">{{cite web | url=http://www.whhbx.com/tspp_show.asp?id=95 | title=重油烧梅馆 | accessdate=2017-02-12}}</ref>

== 各地燒賣 ==
在烧麦的原产地中国北方，烧麦的馅料以肉类为主，传到南方之后，[[糯米|糯米]]成了主料。在海外，受到香港僑民的影响，幾乎全部的燒賣都是廣式，廣式燒賣亦早成為西方人普遍認知的品種。
===塞北燒賣===
以[[羊肉|羊肉]]或者其他肉类為主餡，不加糯米。最经典的馅料是羊肉和[[大葱|大葱]]的葱白。[[内蒙古|内蒙古]][[呼和浩特|呼和浩特]]的[[麦香村|麦香村]]、[[山西|山西]][[大同市|大同市]]的[[凤临阁|凤临阁]]等处的烧麦都是远近闻名。
===上海燒賣===
以糯米為主餡，可在其中加入切碎的豬肉，牛肉，蝦，白菜，香菇，以及其它。
===廣東燒賣===
[[File:Shaomai_by_Kent_Wang_in_Seattle,_Vancouver.jpg|thumb]]
[[File:HK_Food_Yuen_Long_Fish_Ball.JPG|thumb]]
廣東[[飲茶|飲茶]]吃的[[點心|點心]]中，包含了燒賣。廣式燒賣一般比較小，長度不過2厘米，一個小籠子內可以放下3至4顆，款式多樣，常見的有：
* 乾蒸燒賣
*: 簡稱「乾蒸」，又稱「蝦肉燒賣」，以切碎[[豬肉|豬肉]]、[[蝦|蝦肉]]為主要餡料，用鮮黃色的薄皮包裹，再在燒賣上加一點[[蟹黃|蟹黃]]來點綴，[[蒸|蒸]]製而成。但由於成本問題，現多以[[蟹子|蟹子]]，甚至[[胡蘿蔔|胡蘿蔔]]取代蟹黃。
* 三星燒賣
*: 乾蒸燒賣的豬肉餡上分別放上[[冬菇|冬菇]]、[[豬肚|豬肚]]和蝦肉，每籠每種各一顆。
* 黃沙豬膶燒賣
*: 乾蒸燒賣的豬肉餡上加上豬肝蒸製而成。現時很少[[茶樓|茶樓]]供應，只有一些老茶樓如[[蓮香樓|蓮香樓]]有售。
* 牛肉燒賣
*: 將切碎[[牛肉|牛肉]]，用白色的薄皮包裹，再上面上加上一顆[[青豆|青豆]]蒸製而成。
* 鵪鶉蛋燒賣
*: 餡料是碎[[豬肉|豬肉]]、蝦肉，上面有一粒去殼熟[[鵪鶉|鵪鶉]]蛋。
* 魚肉燒賣
*: 香港的街頭小吃，體積較一般燒賣細小，主要餡料是碎[[魚肉|魚肉]]，用黃色的雲吞皮包裹蒸成，可以用竹籤串起來進食，通常醮[[醬油|醬油]]和[[辣椒油|辣椒油]]來吃<ref>[http://www.tech-food.com/kndata/1057/0114368.htm 香港鱼肉烧卖王]，中国食品科技网，2013-8-1</ref>。
* 素燒賣
*: 把蘿蔔, 馬蹄, 豆腐乾切碎搞拌至完全混合, 再加入燕麥, 粟米粉, 調味, 搓成一小團, 再以雲吞皮包裹並捏成燒賣狀, 蒸10-12分鐘即可。

===湖北烧梅===
