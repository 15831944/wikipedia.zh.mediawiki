'''灵川县'''是[[中华人民共和国|中国]][[广西壮族自治区|广西壮族自治区]][[桂林|桂林]]市的辖县，位于[[桂林|桂林]]市北25公里，[[漓江|漓江]]西岸，[[湘桂铁路|湘桂铁路]]穿过。县内有[[青狮潭水库|青狮潭水库]]。面积2288平方千米，人口35万。邮政编码541200。县人民政府驻灵川镇。

== 历史 ==

1949年11月间，[[中国人民解放军第四野战军|中国人民解放军第四野战军]]攻占灵川县<ref>{{Cite web |url=http://www.people.com.cn/GB/historic/1122/3974.html |title=1949年11月22日 桂林解放 |accessdate=2014-09-05 |author= |date= |work= |publisher=人民网|language=Zh-hans }}</ref>。

== 行政区划 ==
下辖5镇，3乡，2民族乡：
*镇：灵川镇、三街镇、大圩镇、潭下镇、定江镇。
*乡：潮田乡、海洋乡、灵田乡。
*民族乡：兰田瑶族乡、大境瑶族乡。
