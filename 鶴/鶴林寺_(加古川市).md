'''鶴林寺'''（日語：かくりんじ）是位於日本[[兵庫縣|兵庫縣]][[加古川市|加古川市]]的[[佛教|佛教]][[寺院|寺院]]。[[山號|山號]]是刀田山（日語：とたさん），宗派是[[天台宗|天台宗]]，祭祀本尊為[[藥師佛|藥師佛]]。

== 概要 ==

位置在加古川的市街地外側（位於市役所和明姫幹線間）。 

是屬於近畿地方眾多由[[聖德太子|聖德太子]]開基傳承的寺院之一，也是由太子建立的七大寺之一，但關於創建的詳細事情則不明確。以屬於[[平安時代|平安時代]]建築的太子堂（[[國寶|國寶]]）為主，擁有多數的文化資產，有「西邊的[[法隆寺|法隆寺]]」之稱<ref>{{cite web |last1=Marketing,Inc. |first1=Social Good |title=西の法隆寺と称される鶴林寺、珍しい文化財が多数残る播磨の古刹の魅力に触れる旅 {{!}} ニホンタビ - 日本の奥深い魅力を伝える大人の旅行マガジン |url=https://reki4.com/00273.html |website=ニホンタビ - 日本の奥深い魅力を伝える大人の旅行マガジン |accessdate=2018-08-26 |language=ja}}</ref>，為 [[播磨|播磨]]地方屈指可數的古寺。

在境内和塔頭周圍三方，被整理成鶴林寺公園，展示著日本{{tsl|ja|国鉄C11形蒸気機関車|C11型蒸汽機關車}}等等，在假日是親子一同攜家大小前往的遊樂場。

== 歷史 ==

其傳承據稱是在[[崇峻天皇|崇峻天皇]]2年（[[589年|589年]]）創建，聖德太子為了當時在播磨一帶的[[高句丽|高麗]]僧侶・惠便（日語：えべん）所建立<ref>{{cite book |author1=刀田山鶴林寺 |title=鶴林寺と聖徳太子: 「聖徳太子絵伝」の美 |date=2008 |publisher=法藏館 |page=3}}</ref>。養老2年（[[718年|718年]]）身人部春則（日語：むとべのはるのり/みとべのはるのり）的人一直維持著七堂伽藍，但未離開原先傳承的區域<ref name=市史>{{cite book |author1=八木哲浩 |title=加古川市史 |date=1985 |page=48}}</ref>。

但是，在[[推古天皇|推古天皇]]14年（[[606年|606年]]），聖德太子講說[[法華經|法華經]]<ref name=市史/>，因其功德，播磨國得到天皇賞賜水田百町的這個史實，顯示聖德太子似乎和播磨有某種關聯。創建時的寺號是四天王寺聖靈院，在天永3年（[[1112年|1112年]]）作為[[鳥羽天皇|鳥羽天皇]]的勅願所而改稱為「鶴林寺」<ref>{{cite book |author1=小野正敏 |title=歴史考古学大辞典 |date=2007 |publisher=吉川弘文館 |page=241}}</ref>。

鶴林寺擁有[[飛鳥時代|飛鳥時代]]後期（白鳳期）的銅造[[聖觀音|聖觀音]]像<ref>{{cite book |author1=范作申 |title=日本传统文化 |date=1992 |publisher=新知三联书店 |page=83}}</ref>；本堂的本尊--藥師三尊像，則是可以追溯到平安時代前期・[[10世紀|10世紀]]的古像<ref>{{cite book |author1=久野健 |title=平安初期彫刻史の研究 |date=1974 |publisher=吉川弘文館 |page=184}}</ref>。

戰國時代鄰近的書寫山捲入戰火，當時的[[播磨|播磨]]姫路領主[[黑田職隆|黑田職隆]]，因兒子[[黒田孝高|黒田孝高]]的勸說加入信長派，而未捲入戰火之中，因此仍有不少當時的建築物留存下來。黑田家後來得到福岡藩的大封，在鶴林寺進行大規模的法會，並捐贈了許多金銀供養。寺院殘留不少職隆、孝高往來的信件<ref>{{cite web |title=鶴林寺文書/市指定文化財／加古川市ホームページ |url=http://www.city.kakogawa.lg.jp/soshikikarasagasu/kyoikushidobu/bunkazai_cyosa/bunkazai/site_torokubunkazai/1416295118360.html |website=www.city.kakogawa.lg.jp |accessdate=2018-08-26 |language=ja}}</ref>。

現在也是光主要堂塔就有16棟的大伽藍，鎌倉室町時期曾達到寺坊30以上的規模。

=== 國寶 ===

*本堂（附：宮殿棟札2枚）

:室町時代。[[歇山頂|歇山頂]]、本瓦葺。桁行（正面）7間、梁間（側面）6間（「間」並非長度單位，而是表示柱間數量）。從堂内宮殿（廚子）的[[上樑記牌|棟札銘]]可知是應永4年（1397年）的建築，是[[日本佛教建築#和樣|和樣]]和帶有點[[日本佛教建築#禪宗樣|禪宗樣]]的[[日本佛教建築#折衷樣|折衷樣]]建築之代表作，大量使用棧唐戶（由縱橫棧木組成的門扉）為其特色。内部宮殿則安置有秘佛藥師佛三尊像和二天像（各為重要文化財）<ref name= 國寶>『週刊朝日百科 日本の国宝』32号、朝日新聞社、1997</ref>。

*太子堂

:平安時代。建於本堂右前方，堂内的壁畫是聖德太子像，所以被稱為太子堂，但原本稱為「法華堂」，與本堂左前方的常行堂相對（「法華堂」、「常行堂」同類型的堂並建，是天台宗特有的伽藍配置，如[[延曆寺|延曆寺]]、日光的[[輪王寺|輪王寺]]等等）。屋頂是[[攢尖頂|攢尖頂]]、檜皮葺。桁行（正面）、梁間（側面）都是3間，主屋前面則附加有梁間1間的孫庇（禮拜空間），從側面看的話側面，主屋和孫庇交界的屋簷曲線彎曲形成「縋破風」。從屋頂木板有鎌倉時代的墨跡得知是天永3年（1112年）的建築。堂内安置有本尊釋迦三尊像（重要文化財）<ref name= 國寶/>。

:除了建築，堂内的壁畫也很重要，是平安時代繪畫稀少的遺跡。東側壁面描繪的聖德太子像，被中世的廚子包被，被視為秘佛（因1977年被指定為重要文化財，所以無法公開相片）<ref>在每日新聞社編『重要文化財』、同社編『国宝・重要文化財大全』便沒有刊載這個聖德太子像。『鶴林寺太子堂とその美』（鶴林寺叢書）則刊載了相片。</ref>。來迎壁（在本尊背後的牆壁）內部則有九品來迎圖和佛涅槃圖，雖然因為一片漆黑、肉眼無法確認圖案，但透過紅外線攝影技術而確認全貌<ref>{{cite book |author1=濱田隆 |title=日本の美術２７３ 来迎図 |date=1989 |publisher=至文堂}}</ref>。

== 交通方式 ==

*從[[西日本旅客鐵道|JR]]加古川站起往搭乘加古川市區域巴士「別府路線」約8分，在「鶴林寺」下車，徒步可到達。
*從JR加古川站起搭乘加古川市區域巴士「東加古川路線」或神姫巴士「往安田東」約7分，在「北在家東口」下車，徒步5分到達。
**上述大約1小時有1~2列火車運行。

*從[[山陽電氣鐵道|山陽電氣鐵道]][[山陽電氣鐵道本線|本線]]尾上之松站下車，徒步15分到達。
