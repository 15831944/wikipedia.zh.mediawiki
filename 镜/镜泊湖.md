'''鏡泊湖'''位于[[中華人民共和國|中國]][[黑龙江省|黑龙江省]][[牡丹江市|牡丹江市]][[宁安县|宁安县]]境内，古称“'''忽汗海'''”，是历经五次[[火山|火山]]喷发，[[玄武岩|玄武岩]]浆流阻塞[[牡丹江|牡丹江]]河床而形成的中国最大的高山[[堰塞湖|堰塞湖]]。

== 地理和资源 ==
镜泊湖呈S形，南北长45公里，平均宽度2至3公里；最宽处有6公里，最窄处有1.5公里，号称“百里长湖”，面积约90.3[[平方公里|平方公里]]，湖泊南浅北深，北部最深处达60米，南部最浅处只有1米左右，容水量16亿[[立方米|立方米]]。

== 人文旅游 ==
[[File:Howering_Mansion_Falls.JPG|thumb]]
[[File:Jingpo_Lake_Maogong_Mountain.JPG|250px]]
镜泊湖风景秀丽，气候宜人，是有名的旅游胜地，其著名的景观有[[吊水楼瀑布|吊水楼瀑布]]，道士山，白石砬子，老鹰砬子，珍珠门，大孤山，小孤山等，号称“镜泊八景”。吊水楼瀑布位于镜泊湖北端，落差约20-25米，在每年的秋季，水量最大的时期，瀑布可宽达40余米。颇为壮观。 

鏡泊湖是第一批[[中国国家重点风景名胜区|中国国家重点风景名胜区]]之一。2005年8月被批准为[[国家地质公园|国家地质公园]]，2006年9月18日被[[联合国教科文组织|联合国教科文组织]]评为第三批世界地质公园。<ref>[http://www.jingpohu.com.cn/dizhi/page.asp?id=2 景区概况]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 自然资源 ==
在湖的南部，[[牡丹江|牡丹江]]及其支流的河口有[[三角洲|三角洲]]发育，地势低洼，水网纵横，有大量浮游生物生存，因而适宜鱼类生长发展<ref name="國家地理">{{cite book|title=《中國國家地理精華》|year=2007年|location=吉林出版集團有限公司|isbn=9787807206309|pages=p41}}</ref>，盛产各种淡水鱼类，如：鳌花、[[红尾鱼|红尾鱼]]等。

镜泊湖蕴藏着丰富的水力资源，自20世纪30年代以来，这里已经先后建成了[[镜泊湖发电厂|镜泊湖发电厂]]和[[镜泊湖第二发电厂|镜泊湖第二发电厂]]。<ref name="國家地理" />

地下森林位于镜泊湖西北大约50公里处，为古火山口内的原始森林。

地下熔岩洞位于镜泊湖西北大约30公里，由火山熔岩冷凝而成。

== 文学作品 ==
[[吴兆骞|吴兆骞]]流放[[寧古塔|寧古塔]]時，在《北渚望目》中写道：“徘徊临北渚，永夜目光寒。顾兔飞难定，金波泻欲残。风微频泛滟，浪细不成团。”黃綺《憶江南．鏡泊湖上看姊妹山》：「常結伴，姊妹臥湖邊。天意催妝羞晏起，波平雲抹照雙鬟。破鏡羨人圓。」
