'''紅棉路'''（英文：'''Cotton Tree Drive'''），是由[[香港|香港]][[港島|港島]][[金鐘|金鐘]]前往[[半山區|半山區]]的一條交通幹線。它起始自山下東行線車道的[[金鐘道|金鐘道]]交匯處，及金鐘道西行線車道，而山上尾段分叉接駁到[[堅尼地道|堅尼地道]]西段，及[[花園道|花園道]]的南段（近[[香港動植物公園|香港動植物公園]]東面出口）。

紅棉路舊稱'''木棉徑'''（英文：'''Kapok Drive'''），其名字是因為該處的[[木棉|木棉]]樹而得名。

== 沿路著名景點 ==
紅棉路因為途經的[[紅棉路婚姻註冊處|紅棉路婚姻註冊處]]而著名，在香港幾近成為[[結婚|結婚]]勝地的代名詞。這處的政府婚姻註冊處是粉紅色的，它在[[香港公園|香港公園]]的西面，有露天停泊花車區。

== 霆鋒彎 ==
霆鋒彎，指紅棉路[[美利大廈|美利大廈]]對開上斜路段，名字沿自2002年3月藝人[[謝霆鋒|謝霆鋒]]曾於該彎位駕駛[[法拉利|法拉利]]F360 Modena跑車時發生車禍，並引發哄動一時的[[謝霆鋒#頂包案|頂包案]]。

過去就有多次交通意外發生於霆鋒彎，大部份都是涉及高級、高性能跑車、轎車、電單車。另一方面，由於該處轉彎後為一段斜而闊之上斜路，不少駕駛愛好者都愛趁深夜少車時以高速入彎之後加速上斜，以表現其車之性能<ref>[http://motor.atnext.com/moDspContent.cfm?article_ID='%23\G5Q_K!+0++%0A&CAT_ID=23 壹車搜雙甩]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}，壹蘋果車網</ref><ref>[http://paper.wenweipo.com/2006/06/27/HK0606270068.htm 法拉利掃欄 醉司機被捕]， 2006-06-27，文匯報</ref>
