'''紅隼反甲火箭'''（{{lang|en|Kestrel anti-armor rocket}}）由[[中山科學研究院|中山科學研究院]]製造。提供給[[中華民國陸軍|陸軍]]換裝新式的[[反坦克武器|反坦克武器]]，計畫取代服役超過35年的[[M72輕型反裝甲武器|國造66火箭彈]]。

== 沿革 ==
[[中華民國陸軍|中華民國陸軍]]現役近程反裝甲火箭如外購[[AT4反坦克火箭筒|AT4反坦克火箭筒]]、[[APILAS|APILAS]]、[[肩射多用途攻擊武器|肩射多用途攻擊武器]]及國造66火箭彈均已超過使用年限，嚴重影響「灘岸機動阻絕、城鎮作戰」反裝甲戰力，尤以負責部隊單兵反裝甲戰力之國造66火箭彈服役至今已逾三十年，為有效強化陸軍反裝甲戰具與戰力，[[中山科學研究院|中科院]]歷經兩年的研發測試及環境測試，已順利完成關鍵技術突破（含氣動力、外彈道、錐孔效應及射瞄系統）。

== 初期型 ==
2009年[[中科院|中科院]]在台北航太展展示新型反甲火箭開發雛形，該火箭彈外型似美製M72A4火箭彈，採用[[成型裝藥|成型裝藥]]，彈體約1.6公斤，射程超過300公尺，具有貫穿300公厘[[軋壓均質裝甲|軋壓均質裝甲]]以上之穿甲能力，也可對普通掩體造成傷害；基本威力等同M72A4火箭彈。

設計上要求採拋棄式射擊，因此沒有被敵軍撿拾使用的問題，增加單兵在戰場上的-{zh-hans:生存率; zh-hant:求生率}-，-{zh-hans:瞄具采用模块化设计; zh-hant:瞄具採模組化}-，依照戰場環境改變瞄具組裝。儲放採用真空包裝，以利長期儲存及運輸安全。

== 精進型 ==
[[File:ROC_marines_Kurstal_rocket.jpg|thumb]]

2011年紅隼反甲火箭再度在台北航太展展示，此時外型與2009年展出版本已截然不同；中科院重新設計拋棄式發射管外型，發射管下方並設計有折疊握把及肩靠固定片、發射器上方裝有戰術軌道，可供拆卸式的簡易瞄準具與夜視鏡使用，後端並裝有喇叭開口，可將筒後噴火紅區，控制在15公尺內。改良後的反甲火箭原型聲稱可控制在5公斤以內，且在惡劣環境之下不影響反裝甲火箭的性能和操作。發射前須將握把前端的小拉柄進行上膛，再進行瞄準目標後按下紅色保險鈕，即可射擊。紅隼反甲火箭在2011年後曾少量生產提供[[中華民國陸軍|中華民國陸軍]]與[[中華民國海軍陸戰隊|中華民國海軍陸戰隊]]測評，海軍陸戰隊的紅隼反甲火箭曾在2013年11月初的聯勇操演進行實兵測評。

2013年台北航太展展示時，除了2011年展示的反甲彈頭外，增加了另一型可選配的彈種，中科院稱其為攻堅彈頭，開發代號堅隼專案。據中科院的系列介紹影片，具400公厘穿甲能力，全彈最大射程超過1200公尺，可貫穿30公分厚的磚牆或20公分厚之鋼筋強化[[混凝土|混凝土]]，表面破壞直徑達70～90公分，具400公尺以上之有效射程，系統性能媲美美現製反裝甲火箭，還可用於反恐與城巷戰。

2014年下旬提出的2015年度國防預算書中，海軍陸戰隊編列4,800萬[[新台幣|新台幣]]，採購700枚紅隼反甲火箭。<ref>[http://news.ltn.com.tw/news/politics/breakingnews/1093215 陸戰隊擬增購三項武器 斥資逾54億]</ref>

2018年8月31日，憲兵指揮部計劃編列4979萬元，在108年至109年採購445套含實彈的紅隼反裝甲武器、訓練彈278枚及訓練模擬器8台。<ref>[https://udn.com/news/story/10930/3341094 憲兵進化台北剌蝟軍 採購445枚紅隼反裝甲火箭]</ref>

2019年4月12日，憲兵與中科院簽訂5092萬元紅隼火箭彈訂單，採購445具紅隼型反裝甲火箭彈、278枚訓練彈以及8台訓練模擬器，配屬憲兵202指揮部與憲兵快速反應連，可於衛戍區內利用城鎮大樓的特性，執行部隊反裝甲作戰任務，執行衛戍區反突擊、反滲透與反空（機）降作戰，以遂行衛戍區城鎮防衛作戰。<ref>[https://news.ltn.com.tw/news/politics/breakingnews/2757014 憲兵下單了！5092萬買中科院紅隼火箭彈]</ref>

[[海洋委員會海巡署|海洋委員會海巡署]]，將在108年採購84具紅隼反裝甲火箭彈，裝備[[海洋委員會海巡署東南沙分署|東南沙分署]]<ref>[防越南攻佔太平島！海巡署明年採購「紅隼火箭彈」火力升級 |ETtoday新聞|https://www.ettoday.net/news/20181013/1280265.htm]</ref>。

[[中華民國憲兵|中華民國憲兵]]，採購「紅隼型」反裝甲火箭彈，軍方官員今天證實，憲兵司令部已在日前正式與中科院簽下採購協議，108年5月起開始分批交貨，要在109年11月底之前，交付445具紅隼型反裝甲火箭彈、278枚訓練彈以及8台訓練模擬器，合約總金額為台幣5092萬元。預計裝備憲兵快速反應連與202指揮部。<ref>[https://m.ltn.com.tw/amp/news/politics/breakingnews/2757014 憲兵下單了！5092萬買中科院紅隼火箭彈]</ref>

== 可靠性測試 ==
通過以下測試，符合美軍MIL-STD-810G標準
*防海水測試（時間30分鐘、濃度5%、深度90公分）
*防泥水測試（時間30分鐘、濃度10%、深度30公分）
*防沙土測試（時間30分鐘、深度30公分）
