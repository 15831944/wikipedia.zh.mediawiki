'''紅臉歌鷹（''Micronisus gabar''）'''是生活在非洲及阿拉伯半島的一種[[猛禽|猛禽]]，為[[鷹科|鷹科]]下的一個[[物種|物種]]。

==描述==
紅臉歌鷹具[[多態性_(生物學)|多態性]]，隨著地理區域的不同，其形態會有所不同。較常見的紅臉歌鷹，其上半身為灰色，臀部為明顯的白色，胸部與大腿具白灰相間的條紋，尾巴則具有深灰色的條紋；在幼鳥時，胸部的條紋則會較不清楚。此外，還有一些較少見的紅臉歌鷹，其全身都是黑色的，估計占整體總量的25%。上述兩種類型的成鳥，其眼睛都是黑的，腿部及[[喙|喙]]則都是紅色的；在幼鳥時，其腿部及喙則為黃色的，且翅膀會較偏棕色。雌鳥會明顯比雄鳥還重，雄鳥約重90~173公克，雌鳥則約重167~240公克<ref name = Arkive>{{cite web | url = http://www.arkive.org/gabar-goshawk/melierax-gabar/ | title = Gabar goshawk (''Melierax gabar'') | accessdate = 23 October 2016 | publisher = Wildscreen}}</ref>。身長為28~36公分，展翅寬度則可達63公分<ref name = oiseaux>{{cite web | url = http://www.oiseaux-birds.com/card-gabar-goshawk.html | title = Gabar Goshawk Melierax gabar | accessdate = 23 October 2016 | publisher = Oiseaux-birds}}</ref>。

==分布及亞種==
紅臉歌鷹生活於撒哈拉以南非洲至阿拉伯半島等地<ref name = oiseaux/>，目前已知的亞種如下<ref name = Avibase>{{cite web | url = http://avibase.bsc-eoc.org/species.jsp?avibaseid=922DE9C1DF8597F0 | title = Gabar Goshawk ''Micronisus gabar'' (Daudin, 1800) | accessdate = 23 October 2016 | publisher = Denis Lepage| work = Avibase}}</ref>：

*''M. g. aequatorius''：由[[衣索比亞|衣索比亞]]高地至[[剛果民主共和國|剛果民主共和國]]、尚比亞、莫三比克北部
*''M. g. gabar'': [[安哥拉|安哥拉]]南部至[[尚比亞|尚比亞]]、[[莫三比克|莫三比克]]、[[南非|南非]]
*''M. g. niger'': [[塞內甘比亞|塞內甘比亞]]至[[蘇丹|蘇丹]]、衣索比亞北部、阿拉伯半島西南部
