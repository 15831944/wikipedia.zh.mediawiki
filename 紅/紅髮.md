'''紅髮'''是指多種色調不一的髮色，包括較深的橘紅色、[[橘色|橘色]]，以及[[銅色|銅色]]。

== 地区 ==
在人類族群中，大約有1%到2%的人口擁有紅髮<ref name="redalert">[http://www.washingtonpost.com/ac2/wp-dyn?pagename=article&node=%20&contentId=A47332-2002Mar18&notFound=true Red Alert!] Washington Post: Original Date 2002-03-19. Accessed 2007-02-06.</ref><ref>[[National_Geographic|National Geographic]], September, 2007</ref>。較常出現在北歐與西歐，在這些地區佔約2%到6%；歐洲其他地區，以及亞洲和非洲則相對少見。汉族群中，红髮虽然很少，可是还是会发生。

== 社会歧视现象 ==
在英语文化圈，存在对红发进行歧视和校园欺凌现象，不论红发男性或女性都会被某些社会偏见认为没有“灵魂”，会偷走他人的“灵魂”，对红发男孩歧视尤其严重。

== 化学 ==
紅髮形成自毛髮中高量的紅色[[色素|色素]][[假黑色素|假黑色素]]（英語：pheomelanin）；以及相對較少量的[[真黑色素|真黑色素]]（eumelanin）。擁有紅髮的人在[[16號染色體_(人類)|第16號染色體]]上有兩個[[隱性基因|隱性基因]]，這會使[[黑素皮質素受體1|黑素皮質素受體1]]（melanocortin 1 [[:en:Melanocortin_receptor|receptor]]；MC1R）蛋白發生轉變。由於皮膚和眼睛也有相同的基因，因此也產生了白皙的皮膚與雀斑，紅髮者對[[紫外線|紫外線]]也較為敏感。

== 动物 ==
紅色毛髮也可見於人類以外的[[哺乳動物|哺乳動物]]，如[[紅毛猩猩|紅毛猩猩]]、[[松鼠|松鼠]]、[[山羊|山羊]]，或出產於[[蘇格蘭|蘇格蘭]]的牛隻品種[[高地牛|高地牛]]等。在某些[[狗|狗]]或[[貓|貓]]的品種中，或是部分的[[狐狸|狐狸]]與[[鹿|鹿]]，也有紅毛的存在。
