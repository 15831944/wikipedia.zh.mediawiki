'''紅襟粉蝶'''（[[學名|學名]]：''{{lang|la|Anthocharis cardamines}}''），是[[粉蝶科|粉蝶科]]的一種[[蝴蝶|蝴蝶]]。

== 外觀 ==
雄性紅襟粉蝶的前翅端呈橙色，雌蝶則沒有這個特徵。雄蝶在[[春天|春天]]會在灌木籬牆及濕潤草地上尋找雌蝶。它們的底部呈斑綠色，在[[峨參|峨參]]及[[蒜芥|蒜芥]]等[[花朵|花朵]]是完美的[[偽裝|偽裝]]。雄蝶甚至可以將前翅上的橙色收藏在後翅之下來隱藏。它們斑線色的底部其實是由黑色和黃色的鱗片所組成。

== 分佈及棲息地 ==
紅襟粉蝶橫跨整個[[歐洲|歐洲]]，東至[[亞洲|亞洲]]的[[溫帶|溫帶]]，最遠到[[日本|日本]]。過往30年間因[[氣候|氣候]]轉變，它們在[[英國|英國]]的數量急速增長，尤其是在[[蘇格蘭|蘇格蘭]]及[[愛爾蘭|愛爾蘭]]。它們棲息在濕潤的草坪及草地、林地、河堤、溝渠、堤防、[[沼澤|沼澤]]、鐵道路塹及郊野路徑。

== 生命周期 ==
雌蝶會在[[草甸碎米薺|草甸碎米薺]]、[[蒜芥|蒜芥]]及其他野生[[十字花科|十字花科]]的花頭上產卵，因為它們含有[[硫配醣體|硫配醣體]]。雌蝶會被吸引到較大的[[花朵|花朵]]上，縱然它們並不適合[[毛蟲|毛蟲]]生長。雌蝶的[[繁殖|繁殖]]率會受到尋找適合的[[植物|植物]]影響。故此，它們[[演化|演化]]到會使用多種不同的十字花科。卵開始是呈白色，幾日後就會變成鮮橙色，孵化前就會變得深色。由於毛蟲差不多只吃花朵，所以一顆[[植物|植物]]並不足夠多於一隻毛蟲的生長。若兩條毛蟲相遇，其中一條毛蟲會把另一條吃了，目的是減少競爭者。同一道理，初生的毛蟲也會把未孵化的卵吃掉。為了防止其他雌蝶在同一顆植物上產卵，雌蝶會在產卵的地方留下一種[[信息素|信息素]]。毛蟲有5個幼體階段。毛蟲呈綠色及白色，天敵包括[[寄生蠅|寄生蠅]]及[[小繭蜂科|小繭蜂科]]。毛蟲會於初夏成[[蛹|蛹]]，並於翌年[[春天|春天]]破蛹而出。近期研究顯示成蟲可以延後達兩年才破蛹而出，以期確保它們可以在惡劣環境下生存。
