'''紅樓康普爛'''，簡稱'''康版'''，是一個[[臺灣|臺灣]][[Facebook|Facebook]]匿名留言板（靠北版），採用[[JavaScript|JavaScript]]技術運作，建立在[[全球資訊網|全球資訊網]]的資源之上，以言論自由及匿名性質為基本原則，提供線上匿名言論空間。由[[高雄中學|高雄中學]]匿名學生擔任小編進行管理，目前為一個為高雄中學及[[高雄女中|高雄女中]]服務的匿名留言板。<ref>{{Cite web |url= https://m.facebook.com/kskgcomplain2.0/about/?ref=page_internal&mt_nav=0 |title= 紅樓康普爛「關於」 |author=  |work= [[Facebook|Facebook]] |date=  |publisher= 紅樓康普爛 |language=  |accessdate=  |quote=  }}{{zh-tw}}</ref>

== 歷史 ==
=== 沿革 ===
始於2000年代，網路網路的發達使校園bbs紅極一時。此時雄中雄女的學生也創建一個[[bbs|bbs]]「紅樓論壇」<ref>{{Cite web |url=  http://www.kskg.org |title= 紅樓論壇原網頁 |author=  |work=  |date=  |publisher=  |language=  |accessdate=  |quote=  }}{{zh-tw}}</ref>，並設有數版<ref>{{Cite web |url= https://evctw.fandom.com/wiki/%E7%B4%85%E6%A8%93%E8%AB%96%E5%A3%87 |title= 紅樓論壇-臺灣網絡百科 |author=  |work= FANDOM wikis |date=  |publisher=  |language=  |accessdate=  |quote=  }}{{zh-tw}}</ref>。但隨著[[Facebook|Facebook]]及其他社群網站興起，紅樓論壇因無人維護而關閉，原先其中一版「康版」因人氣過高而開始被討論如何繼續經營下去。

自2011年始，匿名留言板「[[靠北工程師|靠北工程師]]」在[[台灣|台灣]]的Facebook使用者中興起了一股熱潮，於是臉書用戶們也跟著創立了「靠北版」，如各地的各級學校也紛紛創立了許多匿名留言板。在2014年，繼「[[黑特建中|黑特建中]]」之後，由[[高雄中學|高雄中學]][[學生|學生]]藉由臉書粉絲專頁的方法，建立了屬於雄中的靠北版「紅樓康普爛」。

=== 第一代 ===

[[File:RedBuildingKSHS.jpeg|thumb]]

2014年11月5日成立。因遭檢舉而於2017年停止運作，目前貼文皆已刪除。遭檢舉原由眾說紛紜，至今仍未定論。

== 影響 ==
紅樓康普爛在高雄中學的效用，在運行一段時間後成為了一個類似[[PTT|PTT]]之於台大的作用，許多校務公告亦會透過紅樓康普爛進行宣導，而紅樓康普爛也成為外界了解[[雄中|雄中]]內部的途徑之一。（例如：[[高雄中學#雄中校慶接力撤棒事件|雄中運動會大隊接力撤棒事件]]<ref>{{Cite web |url =https://reurl.cc/0KXqY  |title =大隊接力撤棒事件經過 |author = 方子齊|date = 2014-10-25|publisher = |language = zh-hant|accessdate =2019-04-06  |quote =}}</ref>）不過，也有人認為，匿名發言的特性導致了許多的「匿名[[網路霸凌|霸凌]]」，充滿著力量。<ref>{{cite web|url=https://prisperingsenior.wordpress.com/2017/04/29/【囹語觀點姚岳廷高雄中學】論各校黑特版的存在/|title=【囹語觀點/姚岳廷/高雄中學】論各校黑特版的存在|author=姚岳廷|language=zh-hant|accessdate=2019-04-21|quote=}}</ref>目前對於此服務的優劣尚未有統一定論。
