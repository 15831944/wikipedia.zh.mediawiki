'''藍線'''（{{jpn|j=ブルーライン|hg=|rm=Burū Rain|f=y}}），正式名稱為'''橫濱市高速鐵道1號線、3號線'''，是[[日本|日本]][[橫濱市營地下鐵|橫濱市營地下鐵]]路線，連結[[神奈川縣|神奈川縣]][[藤澤市|藤澤市]]{{ST|湘南台|2}}和[[橫濱市|橫濱市]][[青葉區_(橫濱市)|青葉區]][[薊野站_(神奈川縣)|{{ST]]。

橫濱市高速鐵道1號線、3號線以中途的[[關內站|關內站]]為界，湘南台站至關內站為'''1號線'''，關內站至薊野站為'''3號線'''，但關內站非列車起迄站，所有列車皆直通兩線。雖然兩線的正式起點為關內站，但實際的運行系統起點站為湘南台站。依《{{link-ja|橫濱國際港都建設法|横浜国際港都建設法}}》與《{{link-ja|都市計畫法|都市計画法}}》等制定的{{link-ja|都市高速鐵道|都市高速鉄道}}名稱為「橫濱國際港都建設計畫都市高速鐵道第1號市營地下鐵1號線」、「藤澤都市計畫都市高速鐵道1號線」與「橫濱國際港都建設計畫都市高速鐵道第2號市營地下鐵3號線」。

因應[[橫濱市營地下鐵綠線|4號線（綠線）]]通車，交通局將擁有兩條地下鐵路線，因此自該線通車的2008年3月30日起正式使用公開招募的路線暱稱「藍線」。藍色不僅是橫濱的代表色，也是自地下鐵通車起車輛與標示所使用的顏色。列車編號為'''B'''。

==概要==
本路線是從[[湘南台站|湘南台站]]經[[泉區_(橫濱市)|泉區]]南部、[[戶塚站|戶塚站]]、[[港南區|港南區]]中心部、[[上大岡站|上大岡站]]、[[南區_(橫濱市)|南區]]中心部至橫濱[[都心|都心部]]的{{link-ja|伊勢佐木町|伊勢佐木町}}、[[關內_(橫濱市)|關內]]、{{link-ja|櫻木町|桜木町}}、[[橫濱站|橫濱站]]。櫻木町位於{{link-ja|野毛町|野毛町}}與[[橫濱港未來21|港未來]]地區中間，因此實際上貫穿了橫濱都心。接著從橫濱站經[[神奈川區|神奈川區]]中央部、{{link-ja|新橫濱|新横浜|新橫濱都心}}、{{link-ja|港北新城|港北ニュータウン}}至[[青葉區_(橫濱市)|青葉區]]的[[薊野站|薊野站]]。是以「コ字型」連通橫濱都心、副都心群與橫濱市郊的放射型鐵路。由於與眾多連結都心及市郊的各社私鐵路線交會，因此也具有接駁轉乘的機能。

與[[東京地下鐵|東京地下鐵]][[東京地下鐵銀座線|銀座線]]、[[東京地下鐵丸之內線|丸之內線]]一樣採用[[第三軌供電|第三軌供電]]。本路線是日本目前最新一條採用第三軌供電的新建地下鐵路線，本路線通車之後的第三軌供電路線不是既有路線的延伸<ref>如[[大阪市營地下鐵|大阪市營地下鐵]]的[[大阪市營地下鐵御堂筋線|御堂筋線]]、[[大阪市營地下鐵谷町線|谷町線]]、[[大阪市營地下鐵中央線|中央線]]、[[大阪市營地下鐵千日前線|千日前線]]，[名古屋市營地下鐵]]的[[名古屋市營地下鐵東山線|東山線]]、[[名古屋市營地下鐵名城線|名城線]]，[[札幌市營地下鐵南北線|札幌市營地下鐵南北線]]。</ref>，就是以路線連通為目的所建設的路線<ref>[[京阪奈線|京阪奈線]]與{{link-ja|大阪港交通系統|大阪港トランスポートシステム}}[[中央線_(大阪市高速電氣軌道)|中央線]]。皆連結[[大阪市營地下鐵中央線|大阪市營地下鐵中央線]]。</ref>，因此本線是日本目前最新一條純粹全新建設的第三軌供電地下鐵路線。

1972年通車時就已導入當時首都圈稀少的自動驗票閘門（當時首都圈僅導入於1973年開業的[[日本國有鐵道|國鐵]][[武藏野線|武藏野線]]12站、1979年開業的北總開發鐵道、以及其他公司少數實驗車站，首都圈正式導入是在國鐵民營化後的1990年以後）。2000年代全部車站都已完成[[月台門|月台門]]增設。

依據最初（1965年）的計畫，3號線路線為勝田（港北新城附近） - {{link-ja|新橫濱|新横浜}} - 橫濱站 - 伊勢佐木町 - {{link-ja|山下町 (橫濱市)|山下町 (横浜市)|山下町}} - {{link-ja|本牧|本牧}}，勝田可轉乘4號線（[[鶴見站|鶴見]] - [[綱島站|綱島]] - 勝田 - 元石川），橫濱可轉乘2號線（[[神奈川新町站|神奈川新町]] - 橫濱站 - 藤棚 - 吉野町 - 瀧頭 - [[屏風浦站|屏風浦]]），伊勢佐木町可轉乘1號線（伊勢佐木町 - 上大岡 - 戶塚 - 湘南台）。但由於與[[首都高速道路|首都高速道路]]的建設計畫重疊，伊勢佐木町站的預定地從尾上町改為現在的[[關內站|關內站]]，並設計成可與1號線直通運轉的方向別月台。而原先要從關內站延伸至本牧方向的計畫因為改成與1號線直通運轉而宣告中止。往本牧方向的路線之後演變成現在的[[橫濱高速鐵道港未來線|橫濱高速鐵道港未來線]]。不過，現在若是要前往本牧、磯子方向，可直接在關內站轉乘[[根岸線|根岸線]]。

3號線等之後通車的區間，其隧道多採用[[新奧工法|新奧工法]]，多處隧道的天花板較高。湘南台站路段與[[泉野線|相鐵泉野線]]為2段式隧道構造。由於泉野線湘南台站啟用時間比本路線通車還早，因此泉野線在地下三層，本路線在地下二層。

3號線的三澤下町站至岸根公園站區間最初是規劃直線貫穿民有地下方，但因通過住宅下方將有噪音疑慮，於是改成現在的三澤上町 - 片倉町的道路下方路線（即便如此依然比JR線短900公尺）。然而，改道卻使得{{link-ja|六角橋|六角橋}}周邊商店街擔心客群流失，引發抗議運動<ref>大前研一 ｢大前研一の新・国富論｣  p171-172 講談社 1986年</ref><ref>[http://hamarepo.com/story.php?story_id=1377 六角橋商店街が、地下鉄駅の建設誘致に反対したって噂は本当？] - はまれぽ.com、2012年10月4日</ref>。

本線是唯一連結橫濱站、[[櫻木町站|櫻木町站]]、關內站等橫濱市都心，以及新橫濱站、薊野站、港北新城各站、上大岡站、戶塚站等副都心群的唯一鐵道，除了是彼此之間的交通路線外，也可促進橫濱市內人口分布平均化與市區一體化。

隨著港北新城計畫與[[橫濱市營地下鐵綠線|4號線（綠線）]]計畫的進展，[[中心北站|中心北]]與[[中心南站|中心南]]兩站成為4號線轉乘站。由於該線將往南至[[中山站_(神奈川縣)|中山站]]，3號線因此改成延伸至元石川。東急原先強烈要求將元石川設於[[多摩廣場站|多摩廣場站]]。但橫濱市則堅持設於薊野站。結果，橫濱市將3號線終點設於薊野站，然而東急並未配合增停[[東急田園都市線|田園都市線]]的快速（1996年4月廢止）與急行列車，使得使用人次大增的薊野站並未具有一般轉乘站的便利性。但在橫濱市的請求下，東急自[[藤丘站_(神奈川縣)|藤丘站]]上行通過線設置工程完工後的2002年3月28日改點起增停急行（2007年4月5日增停準急）。

新羽 - 薊野間是以'''新市鎮鐵道'''補助金建設，因此嚴格來說該段不屬於地下鐵（另一個例子是[[神戶市營地下鐵|神戶市營地下鐵]][[西神·山手線|西神·山手線]][[名谷站|名谷站]] - [[西神中央站|西神中央站]]段）。

== 運行形態 ==
基本上，1號線與3號線相互直通運轉，但也有薊野站 - 新橫濱站（3號線內）、湘南台站 - 上永谷站（1號線內）等區間列車存在。午間時段每30分有一班湘南台站 - 薊野站的快速列車、兩班湘南台站 - 薊野站間的普通列車、一班踊場站 - 薊野站間的普通列車、一班湘南台站 - 新羽站間的普通列車。

=== 車種 ===
==== 快速 ====
[[戶塚站|戶塚站]] - [[新羽站|新羽站]]間以快速運行。此區間內僅停靠可轉乘其他路線的戶塚站、上大岡站、關內站、櫻木町站、橫濱站、新橫濱站與車輛基地所在站、並可實施[[緩急接續|緩急接續]]的[[上永谷站|上永谷站]]、[[新羽站|新羽站]]。兩端區間的湘南台站 - 戶塚站間與新羽站 - 薊野站間全停。

原先的運行時段為平日的10至16點、週六日9點30分至20點30分。班距30分，1小時有2班（平日、週六日的15點，與週六日湘南台9點發車、薊野10點發車時段僅有一班）。全列車行駛湘南台站 - 薊野站區間。2017年3月4日起，週六日運行時段延長至20點30分<ref name="yokohama20170130" />。

全部列車實施緩急接續，兩方向列車皆可在上永谷站與[[踊場站|踊場站]]起訖的普通列車接續，往湘南台列車可在新羽站與該站起迄的普通列車接續。

湘南台站 - 薊野站間（全線）的普通列車全程需花費69分鐘，快速列車可縮短至60分，其中戶塚站 - 關內站間約需17分鐘，橫濱站 - 新橫濱站間約需8分鐘<ref>[http://www.city.yokohama.lg.jp/koutuu/kigyo/newstopics/2014/topics/20150512094418.html 市営地下鉄ブルーライン　ダイヤ改正後の各駅時刻について] - 横浜市交通局、2015年7月2日</ref>。

車輛的方向顯示器與站內資訊看板以紅色{{Color|#FF3333|■}}代表。

藍線是日本第一條，也是唯一一條運行優等列車的第三軌供電路線。

==== 普通 ====
午間時段每30分有兩班湘南台站 - 薊野站間的直通列車，以及各一班踊場站 - 薊野站間、湘南台站 - 新羽站間的區間列車。

踊場站起迄的區間普通列車會在上永谷站等待快速列車，新羽站起訖的普通列車則在該站與快速接續。

午間以外的時段僅全行駛普通列車，多數班次為薊野站 - 湘南台站間的直通列車，但早晨與深夜有車庫所在站新羽站、上永谷站起迄的出入庫班次，平日亦有薊野站 - 踊場站間的區間列車。此外，早晨還有一班橫濱發往薊野、一班新橫濱發往湘南台，深夜有數班薊野站 - 新橫濱站間的區間列車，並有一班往返湘南台站 - 上大岡站間的區間列車。

平日有1班、週六日有6班薊野往上永谷的列車在終點上永谷站與往湘南台列車接續。

車輛的方向顯示器與站內資訊看板以藍色{{Color|#006bf0|■}}為主，3色LED車輛則使用綠色{{Color|#006600|■}}代表。

== 一人控制 ==
2007年12月15日起，藍線實施[[一人控制|一人控制]]。對此，同年2月起各站開始設置[[月台門|月台門]]，4月起陸續啟用。此外，藍線亦在1月20日啟用[[列車自動運行裝置|ATO]]自動運行。

一人控制開始以前，車門與月台門由司機在控制台控制。

月台門啟用當時，車掌不在列車發車時吹哨，而是播放車輛的乘車廣播，同年11月起配合一人控制開始在各站導入發車鈴聲。

月台門以白色為基底，門上方有藍色色條裝飾。

進站廣播在[[橫濱市營地下鐵綠線|綠線]]通車以前為「（列車進站）危險，請待在白線內側」（危ないですから、白線の内側にお下がり下さい），綠線通車後更改為「（列車進站）請注意」（ご注意ください）。優先席廣播也改為「搭乘市營地下鐵請禮讓座位」（市営地下鉄では、席のゆずりあいをお願いしています。）。

2015年3月14日更新自動廣播，除了加上車種，同時將「（列車進站）請注意」改為「請等待車門開啟」（ドアの前を広く開けて、お待ちください）。

== 車輛 ==
藍線全部列車採用6輛編組，[[湘南台站|湘南台]]側為1號車，[[薊野站|薊野]]側為6號車。一人控制後藍線統一使用3000形車輛。

車輛編號為四位數，個位（湘南台側為1）代表連結位置，百位與十位代表導入順序。

== 延伸計畫 ==
藍線規劃從終點[[薊野站|薊野站]]經[[青葉區_(橫濱市)|青葉區]]{{link-ja|薄野 (橫濱市)|すすき野|薄野}}延伸至[[川崎市|川崎市]][[麻生區|麻生區]][[新百合丘站|新百合丘站]]（現在此區間有[[東急巴士|東急巴士]]與{{link-ja|小田急巴士|小田急バス}}的協定路線「新23」系統運行）。

延伸段規劃在中途新設三座車站（分別位於嶮山一帶、薄野一帶以及川崎市內）。薄野至新百合丘間規劃了三條路徑，分別是通過{{link-ja|YONETY王禪寺|ヨネッティー王禅寺}}附近的「東側路線」、通過王禪寺公園附近的「中央路線」、通過白山附近的「西側路線」，又以東側路線較具效益<ref name="yokohama20190123" />。

2000年{{link-ja|運輸政策審議會答申第18號|運輸政策審議会答申第18号}}將薊野 - 薄野間列為2015年度以前開業路線，薄野 - 新百合丘間列為同年度以前整備動工路線。2016年的{{link-ja|交通政策審議會|交通政策審議会}}「{{link-ja|東京圈今後都市鐵道計畫|東京圏における今後の都市鉄道のあり方について}}」列為「因應地域成長充實鐵道網路專案」。

橫濱市在2014年度預算案納入藍線延伸研究費<ref>[http://news.kanaloco.jp/localnews/article/1401270031/ 地下鉄３号線延伸 事業化に向け１４年度予算に計上へ/横浜市] {{Wayback|url=http://news.kanaloco.jp/localnews/article/1401270031/ |date=20140201193844 }} - 神奈川新聞（カナロコ）、2014年1月28日</ref>，開始進行相關研究<ref>[http://kenplatz.nikkeibp.co.jp/article/const/news/20140802/672689/?ST=building&P=1/ 日本一長い地下鉄に、横浜から川崎へ延伸調査着手] - ケンプラッツ、2014年8月4日</ref><ref>[http://response.jp/article/2015/02/03/243342.html 横浜市、ブルーライン新百合ヶ丘延伸の調査に着手へ] - レスポンス、2014年7月28日、2015年4月28日閲覧</ref><ref>[http://mxt.nikkei.com/?4_37538_561586_5 横浜市営地下鉄に快速 新百合ケ丘への延伸は…] - 日本経済新聞、2015年7月17日</ref><ref name="hamarepo170625">[http://hamarepo.com/story.php?story_id=6172 市営地下鉄グリーンラインの延伸計画は、その後どうなった?] - はまれぽ.com、2017年6月25日</ref>。

2019年1月1日，媒體報導橫濱市將正式推動藍線延伸<ref name="kanaloko2090101" />，同年1月23日與川崎市達成共識，雙方簽訂備忘錄<ref name="nhk190123">{{Cite news|url=https://www3.nhk.or.jp/news/html/20190123/k10011788401000.html|title=横浜市営地下鉄 小田急線「新百合ヶ丘」まで延伸へ|newspaper=NHK NEWS WEB|date=2019-01-23|accessdate=2019-01-23}}</ref><ref name="yokohama20190123" />。目標2030年通車<ref name="kanaloko2090101" />。

雖然[[湘南台站|湘南台站]]沒有往西側延伸的計畫，但與藍線一樣止於湘南台站的[[相鐵泉野線|相鐵泉野線]]有往平塚方向的延伸計畫。
