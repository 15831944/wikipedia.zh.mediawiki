《'''橫衝直撞-{大逃亡}-'''》（{{lang-en|''The Sugarland Express''}}）是一部於1974年上映的美國[[犯罪片|犯罪]][[劇情片|劇情]]電影，由[[史蒂芬·史匹柏|史蒂芬·史匹柏]]執導，並由[[賀爾·巴伍德|賀爾·巴伍德]]與{{link-en|馬修·羅賓斯 (編劇)|Matthew Robbins (screenwriter)|馬修·羅賓斯}}編劇。該片同時也是史匹柏的劇場電影[[處女作|處女作]]<ref name="first-feature.com"/>，由[[歌蒂·韓|歌蒂·韓]]、[[威廉·阿瑟頓|威廉·阿瑟頓]]、[[班·約翰森|班·約翰森]]與{{link-en|麥克·薩克斯|Michael Sacks}}主演。劇情根據真實事件改編，講述一對逃獄的夫妻試圖尋找被送去寄養家庭的孩子。

該片在美國[[德克薩斯州|德克薩斯州]]各地取景。電影在上映後收穫普遍影評人的好評，但在票房方面的表現就沒那麼出色。《橫衝直撞-{大逃亡}-》在{{link-en|第27屆坎城影展|1974 Cannes Film Festival}}上榮獲[[坎城影展最佳劇本獎|最佳劇本獎]]。

== 劇情 ==

1969年5月，剛出獄的盧·琴·帕布倫（{{lang|en|Lou Jean Poplin}}）協助丈夫克洛維斯·麥克·帕布倫（{{lang|en|Clovis Michael Poplin}}）逃出拘留所，去尋找兩人在服刑期間送往寄養家庭的孩子。寄養家庭位在[[德克薩斯州|德克薩斯州]]的[[舒格蘭_(德克薩斯州)|舒格蘭]]。

途中，兩人在公路上引發事故，麥斯維爾·史萊巡警（{{lang|en|Patrolman Maxwell Slide}}）攔下了他們，兩人以為史萊巡警得知了越獄的事，心慌之下挾持了史萊巡警，並搶下了他的警槍。由於兩人持有人質，警方只能一路尾隨在後，並提供食物、汽油給帕布倫夫婦。追捕過程引起了大批媒體的關注。兩人一路來到了寄養家庭，警方在那裡槍擊克洛維斯，克洛維斯試圖逃逸但傷重死在車上。警方將盧·琴逮捕歸案。

== 角色 ==

* [[歌蒂·韓|歌蒂·韓]]飾演盧·琴·帕布倫
* [[威廉·阿瑟頓|威廉·阿瑟頓]]飾演克洛維斯·麥克·帕布倫
* [[班·約翰森|班·約翰森]]飾演哈林·坦納警長（{{lang|en|Captain Harlin Tanner}}）
* {{link-en|麥克·薩克斯|Michael Sacks}}飾演麥斯維爾·史萊巡警
* {{link-en|格雷戈里·沃爾科特|Gregory Walcott}}飾演厄尼·馬史本巡警（{{lang|en|Patrolman Ernie Mashburn}}）
* {{link-en|史蒂夫·卡納利|Steve Kanaly}}飾演傑瑟普巡警（{{lang|en|Patrolman Jessup}}）

角色盧·琴·帕布倫與克洛維斯·麥克·帕布倫分別取材自當時21歲的艾拉·費·荷莉黛（{{lang|en|Ila Fae Holiday}}）和22歲的勞勃·「鮑比」·丹特（{{lang|en|Robert "Bobby" Dent}}）<ref name="Truth"/>。史萊巡警的角色原型是當時27歲的{{link-en|詹姆斯·肯尼斯·克隆|James Kenneth Crone|楚珀·J·肯尼斯·克隆}}，而角色坦納警長則取材自德克薩斯州的公路巡警傑瑞·米勒（{{lang|en|Jerry Miller}}）<ref name="Truth"/>。

在現實中，艾拉·費並沒有協助鮑比逃獄——鮑比是在1969年4月時獲釋出獄的<ref name="Truth"/>。有別於電影，鮑比在艾拉·費的2個孩子（在遇見鮑比前生下來的）所住的寄養家庭中槍後當場斃命<ref name="Truth"/>。艾拉·費被判處5年徒刑，最終服刑5個月便出獄。她於1992年逝世，享年40多歲<ref name="Truth"/>。

== 製作 ==

[[File:Steven_Spielberg_Cannes_2016.jpg|thumb]]的劇場電影[[處女作|處女作]]]]

在作品《{{link-en|飛輪喋血|Duel (1971 film)}}》（1971年）廣受好評後，[[史蒂芬·史匹柏|史蒂芬·史匹柏]]收到了多份邀約，但他一一推辭掉了。史匹柏注意到了一個刊登在《公民新聞報》、塵封已久的新聞案件，報導一對夫妻用盡各種手段奪回住在養父母家中的親生孩子。他將劇本交給[[賀爾·巴伍德|賀爾·巴伍德]]與{{link-en|馬修·羅賓斯 (編劇)|Matthew Robbins (screenwriter)|馬修·羅賓斯}}加以潤色<ref name="大導演史蒂芬史匹柏"/>{{rp|118-119}}。該片在美國[[德克薩斯州|德克薩斯州]]各地取景<ref name="IMDb-Filming Locations"/>。在真實事件中，共有90輛警車參與追捕行動；但因製片預算的緣故，劇組只能用約40輛車來拍片<ref name="大導演史蒂芬史匹柏"/>{{rp|121-122}}。

== 發行 ==

《橫衝直撞-{大逃亡}-》於1974年4月5日在美國上映<ref name="IMDb-Release Info"/>，與同期上映的《[[刺激_(電影)|刺激]]》、《{{link-en|窮山惡水|Badlands (film)}}》及《{{link-en|沒有明天的人|Thieves Like Us (film)}}》一同競爭<ref name="大導演史蒂芬史匹柏"/>{{rp|123-124}}。史匹柏不滿該片的發行方式，認為觀眾一看到[[歌蒂·韓|歌蒂·韓]]的名號便認為該片僅是一部小本電影。他還覺得片名取得不好：「許多人還以為這是部兒童片呢」<ref name="大導演史蒂芬史匹柏"/>{{rp|123}}。該片的票房不盡理想<ref name="電影大師——史蒂芬·史匹柏的少年時光"/>{{rp|159}}。《橫衝直撞-{大逃亡}-》的[[藍光光碟|藍光]]及[[數碼多功能影音光碟|DVD]]版本分別於2011年2月15日及2015年5月5日發布<ref name="DVD/BD"/>。

== 反響 ==

[[爛蕃茄|爛蕃茄]]上收集的33篇專業影評文章中，有30篇給出了「新鮮」的正面評價，「新鮮度」為91%，平均得分7.3分（滿分10分），該網站的共識性評價寫道「也許該片的劇情仿效了時下[[反文化|反文化]]的公路電影，但史蒂芬·史匹柏的劇場電影處女作展現了許多賞心悅目的元素，而他將在日後的作品中對它們加以淬鍊」<ref name="RT"/>。《橫衝直撞-{大逃亡}-》在{{link-en|第27屆坎城影展|1974 Cannes Film Festival}}上榮獲[[坎城影展最佳劇本獎|最佳劇本獎]]<ref name="festival-cannes.com"/>。《{{link-en|芝加哥讀者報|Chicago Reader}}》的戴夫·凱爾（{{lang|en|Dave Kehr}}）指出，該片的步調跟不上結局，但「總體而言，這是一部能抓住觀眾胃口的電影」<ref name="Kehr"/>。《[[紐約時報|紐約時報]]》的諾亞·賽爾（{{lang|en|Nora Sayre}}）表示，《橫衝直撞-{大逃亡}-》有種漫不經心的瘋狂格調，使其顯得格外自然<ref name="Sayre"/>。

然而，《[[芝加哥太陽報|芝加哥太陽報]]》的[[羅傑·伊伯特|羅傑·伊伯特]]給出了2.5顆星（滿分4顆）的評價，認為史匹柏將目光過度-{聚焦}-在那些警車上，且沒有呈現出片中角色的特質及個性<ref name="Ebert"/>。《[[綜藝_(雜誌)|綜藝]]》雜誌的影評人寫道，「不幸的是，該片的片尾部分-{聚焦}-在沉重的社會議題上，噪音及怒氣傾瀉而出」<ref name="RT-Reviews"/>。數年後，史匹柏表示，他應該用另一種方式來呈現該片，認為片中對警方的立場描寫的不夠好，坦納警官的動機顯得過於薄弱，且片中那對夫妻的設定又讓他們看起來太過可憐，削弱了他們所犯罪行的嚴重性<ref name="大導演史蒂芬史匹柏"/>{{rp|124-125}}。
