'''橫濱地標大廈'''（'''{{jpn|j=横浜ランドマークタワー|hg=よこはまランドマークタワー}}'''），位於[[日本|日本]][[橫濱市|橫濱市]]，高296.3米<ref name="CTBUH">[[CTBUH|CTBUH]] - [http://buildingdb.ctbuh.org/?do=building&building_id=547 Landmark Tower, ランドマークタワー]</ref><ref name="Emporis">[http://www.emporis.com/application/?nav=building&lng=3&id=105181 Emporis - Yokohama Landmark Tower, 横浜ランドマークタワー]</ref>，有73樓，1993年至2013年間曾是日本第一高樓，現是日本第二高的大樓（僅次於[[阿倍野Harukas|阿倍野Harukas]]）及第五高建築物。開工於1990年3月20日，開業於1993年7月16日。大樓的49樓以下作為辦公與零售用途，49至70樓為[[旅館|旅館]]，69樓設有觀景台，高273米。橫濱地標塔擁有世界第三快的電梯，速度高達每分鐘750米（時速45公里），僅次於[[台北101大樓|台北101大樓]]觀景台電梯和[[東京晴空塔|東京晴空塔]]。

== 概要 ==
橫濱地標大廈位於港未來地區25街區，可透過橫濱市內第一座正式[[電動步道|電動步道]]（原為{{lang-ja|橫濱博覽會|横浜博覧会}}設施）通往[[櫻木町站|櫻木町站]]方向。橫濱地標大廈可分為地下3層<ref name="CTBUH"/>、地上70層、塔屋3層<ref>{{Cite web |url=http://www.yokohama-landmark.jp/sisetsu/tower/outline.html |title=横浜ランドマークタワー 建築概要 |access-date=2018-12-16 |archive-url=https://web.archive.org/web/20101224214912/http://yokohama-landmark.jp/sisetsu/tower/outline.html |archive-date=2010-12-24 |dead-url=yes }}</ref>的塔棟，以及地下4層、地上5層的廣場棟。建築面積23,208[[平方公尺|m<sup>2</sup>]]。樓板面積392,791m<sup>2</sup><ref name="CTBUH"/>，僅次於[[東京都|東京都]][[豐島區|豐島區]][[太陽城_(東京)|陽光城]]（585,895m<sup>2</sup>）、[[愛知縣|愛知縣]][[名古屋市|名古屋市]][[中村區|中村區]][[JR中央大廈|JR中央大廈]]（416,565m<sup>2</sup>）。

[[美國|美國]][[建築師|建築師]]{{link-en|休伊·史東比斯|Hugh Stubbins}}負責基本設計，三菱地所負責細節設計<ref>{{Cite web |url=https://www.yokohama-landmark.jp/about/ |title=横浜ランドマークタワーについて |publisher =|accessdate =2018-03-10}}</ref>。

最初曾計畫在橫濱地標大廈與{{link-ja|橫濱皇后廣場|クイーンズスクエア横浜}}之間的北側再建設一棟摩天大樓<ref name="taccolin-lmt2">[http://blogs.yahoo.co.jp/taccolin/31127077.html ランドマークタワーの二期棟計画（当面延期?）]（みなとみらい線周辺散歩日記 2006年3月26日）/[http://blogs.yahoo.co.jp/taccolin/39958295.html ランドマークタワーの二期棟]（同ブログ記事 同年7月31日）/[https://blogs.yahoo.co.jp/taccolin/62710789.html 高級車ショールーム　ランドマークタワー二期棟予定地]（同ブログ記事 2009年12月11日）</ref>（後述的二期棟，部分設計與橫濱地標大廈相同<ref>パンフレット『LANDMARK TOWER』（1993年発行） / 『ウォーターフロント・シティ横浜：みなとみらいの誕生』（横浜都市発展記念館 編集、公益財団法人横浜市ふるさと歴史財団 2017年10月7日発行） p.60</ref><ref>{{PDF|[https://www.jstage.jst.go.jp/article/qjjws1943/61/2/61_2_107/_pdf 横浜ランドマークタワーの構造設計]}}（山崎真司 - 溶接学会誌〈61巻2号, 1992年3月5日発行〉 pp.107-112）</ref>），但該計畫已被凍結。另外，原先計畫高度是300公尺，但因受到[[羽田機場|羽田機場]][[標準儀表離場程序|標準離場航線]]高度限制，最終設定為296.3公尺<ref>{{Cite web |url=https://www.mlit.go.jp/tokyo_cab/06_oshirase/index_02_img/haneda.pdf |title=東京空港事務所からのお知らせ |access-date=2018-12-16 |archive-url=https://web.archive.org/web/20090621032353/https://www.mlit.go.jp/tokyo_cab/06_oshirase/index_02_img/haneda.pdf |archive-date=2009-06-21 |dead-url=yes }}</ref>。

地標大廈由{{link-ja|橫濱皇家花園酒店|横浜ロイヤルパークホテル}}、辦公室、地標廣場（購物商場）、Dockyard Garden（廣場、美食街）、Sky Garden（展望台）等設施組成。最上層的70樓有飯店的Sky Lounge與宴會場。

從地上三樓可搭乘最大分速約750公尺（時速約45[[公里每小時|km/h]]）的電梯直達69樓展望台「Sky Garden」（地上273公尺）。在2004年[[台北101|台北101]]（最大分速約1,010m〈時速換算で60.6km/h〉）竣工以前是[[金氏世界紀錄|金氏世界紀錄]]速度最快的電梯<ref>{{Cite book|author = 大成建設「超高層ビル」研究プロジェクトチーム |year = 2009 |title = 超高層ビルの"なぜ"を科学する |url = http://www.ark-gr.co.jp/shuppan/contents/book_sale/4860590767/ |edition = 初版 |publisher= アーク出版 |isbn  =978-4-86059-076-5 |page=14 }}</ref>{{refnest|group="注"|地標大廈電梯是[[三菱電機|三菱電機]]製，台北101電梯是{{link-ja|東芝電梯|東芝エレベータ}}製。2016年[[上海中心|上海中心]]電梯最大分速約1,230m（時速換算73.8km/h），成為世界最快紀錄。三菱電機與地標大廈一樣使用三菱電機製電梯<ref>[https://www.nikkei.com/article/DGXLASDZ09H8I_Z01C16A2000000/ 三菱電機のエレベーター「世界最速・最長」に　ギネス認定]（日本経済新聞 2016年12月9日）</ref>。}}。現在仍是日本前往展望台最快的電梯<ref group="注">東京晴空塔電梯最大分速約600m（時速換算36km/h）。</ref>。本電梯設有強風管制運轉系統，可在強風時維持運轉。

大樓的上層部設有透過電腦控制抑制搖晃的巨型制震器<ref name="hamarepo121003">[http://hamarepo.com/story.php?story_id=1366 超高層ビル「ランドマークタワー」の屋上はどうなってる！？]（はまれぽ.com 2012年10月3日）</ref>，同時大樓本體也採用耐震性高的4根支柱構造。屋頂上亦有緊急用的直升機停機坪<ref name="hamarepo121003"/>。

2004年2月，地標大廈旁橫濱皇后廣場地下的[[橫濱高速鐵道|橫濱高速鐵道]][[橫濱高速鐵道港未來線|港未來線]][[港未來站|港未來站]]開業。2008年<!--[[2001年|2001年]]より始まった-->為現代美術國際展「{{link-ja|橫濱Triennale|横浜トリエンナーレ}}2008」會場，並在地標廣場展示部分作品。

竣工以來，地標大廈位居「日本第一高摩天大樓」寶座將近20年，2012年8月<ref>[http://mw.nikkei.com/tb/#!/article/DGXNASDD3002F_Q2A830C1EB2000/ 大阪「あべのハルカス」高さ日本一のビルに 300メートル到達] {{Wayback|url=http://mw.nikkei.com/tb/#!/article/DGXNASDD3002F_Q2A830C1EB2000/ |date=20160106172413 }} - 日本経済新聞 2012年8月30日</ref>被當時建設中的「阿倍野Harukas」（大阪市[[阿倍野區|阿倍野區]]）超越。現在仍是「日本層數最多的摩天大樓」、「[[中部地方|中部地方]]以東的[[東日本|東日本]]最高摩天大樓」及「關東地方最高大樓」、「神奈川縣最高大樓」。

三菱地所曾計畫建設二期棟（最初規劃為橢圓柱型大樓<ref name="taccolin-lmt2"/>）<ref name="mitsubishi jisho161027">{{PDF|[http://www.mec.co.jp/j/news/archives/mec161027_mm47.pdf 「（仮称）MM21地区47街区開発計画」着手 －みなとみらい21地区にコーエーテクモゲームスが本社を移転－ ]}}（三菱地所株式会社 2016年10月27日）</ref><ref name="mitsubishi jisho171201">{{PDF|[http://www.mec.co.jp/j/news/archives/mec171201_MM47.pdf みなとみらい21地区47街区 コーエーテクモゲームス新本社を含む複合施設着工]}}（三菱地所株式会社 2017年12月1日）</ref><ref name="hamarepo161129">[http://hamarepo.com/story.php?story_id=5699 スーパーオートバックスみなとみらい店の跡地はどうなる？]（はまれぽ.com 2016年11月29日）</ref>，但因景氣衰退而凍結計畫（開發時期未定）。<!--二期棟の-->二期棟建設預定地曾作為巴士停車場與公寓展示間，現為<!--2018年２月時点-->暫定設施<!--[[低層建築物]-->汽車展示間（橫濱[[勞斯萊斯汽車|勞斯萊斯]]<ref name="taccolin-lmt2"/><ref name="nicole-rollsy">[http://www.rolls-roycemotorcars-nicole.com/ ロールス・ロイス・モーター・カーズ横浜]（公式ページ）</ref>與[[法拉利|法拉利]]展示間<ref>[http://www.nicole-competizione.co.jp/showroom/showroom1.html Nicole Competizione フェラーリ 横浜 ショールーム]（公式ページ）</ref><ref name="carwatch131003">[http://car.watch.impress.co.jp/docs/news/20131003_618010.html フェラーリ、ニコルと正規ディーラー契約を締結 2014年春に横浜みなとみらいにフェラーリ・ショールームをオープン]（Car Watch 2013年10月3日）</ref><ref name="kanaloco140213-nicole">[http://www.kanaloco.jp/article/68835 フェラーリ、県内初の正規ディーラーがオープン／神奈川]（神奈川新聞<small>〈カナロコ〉</small> 2014年2月12日）</ref>）。
