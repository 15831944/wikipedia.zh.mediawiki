'''橫田真悠'''（{{jpn|j=横田 真悠|kn=よこた まゆう|rm=Mayuu Yokota}}，{{bd|1999年|6月30日|||catIdx=Mayuu Yokota}}），[[日本|日本]][[模特兒|模特兒]]、[[女演員|女演員]]。出身於[[日本|日本]][[東京都|東京都]]，隸屬{{link-ja|ASIA CROSS|エイジアクロス}}。身高168公分，血型O型。<ref name=2018-12-10>{{cite web|script-title=ja:經紀公司個人介紹頁面|url=http://asiacross.jp/model/mayuu-yokota/|publisher=ASIA CROSS|accessdate=2018-12-10|language=ja}}</ref>
== 經歷 ==
* 2014年，參加了女性潮流雜誌《[[Seventeen|Seventeen]]》舉辦的「Miss Seventeen2014」模特兒甄選會，從6147名報名者中脫穎而出，與樫本琳花、[[川津明日香|川津明日香]]一同獲得大獎，10月成為《[[Seventeen|Seventeen]]》的專屬模特兒。<ref name=2014-08-21>{{cite web |script-title=ja:「ミスセブンティーン2014」3人が決定　西内まりやら先輩モデルがアドバイス|url=https://mdpr.jp/news/detail/1406996|publisher=[[modelpress|modelpress]] |date=2014-08-21|accessdate=2018-12-10|language=ja}}</ref>

* 2019年，首次演出電視劇《{{link-ja|3年A班－從現在起，大家都是人質－|3年A組-今から皆さんは、人質です-}}》<ref name=2018-12-04>{{cite web |script-title=ja:今田美桜・大原優乃・横田真悠ら、生徒役第4弾発表　菅田将暉主演の新ドラマ「3年A組―今から皆さんは、人質です―」|url=https://mdpr.jp/news/detail/1808164|publisher=[[modelpress|modelpress]] |date=2018-12-04|accessdate=2018-12-10|language=ja}}</ref>
== 人物 ==
* 興趣是看足球比賽。特殊才藝是街舞。<ref>{{cite web |script-title=ja:横田真悠が“ティーンのカリスマ”である理由 「Seventeenを私が壊しちゃう」葛藤の過去＜モデルプレスインタビュー＞|url=https://mdpr.jp/interview/detail/1785674|publisher=[[modelpress|modelpress]] |date=2018-08-13|accessdate=2018-12-10|language=ja}}</ref>
* 喜歡的音樂是[[K-POP|K-POP]]，特別是[[Red_Velvet|Red Velvet]]、[[TWICE|TWICE]]、[[Lovelyz|Lovelyz]]的歌曲。<ref name=2016-03-28>{{cite web |script-title=ja:【注目の人物】「Seventeen」横田真悠に抜擢続く ティーン支持急増中で期待|url=https://mdpr.jp/news/detail/1573388|publisher=[[modelpress|modelpress]]|date=2016-03-28|accessdate=2018-12-10|language=ja}}</ref>
* 憧憬的藝人是「Seventeen」的前輩[[古畑星夏|古畑星夏]]<ref name="2016-03-28" />
