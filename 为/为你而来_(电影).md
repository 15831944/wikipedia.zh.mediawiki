《'''为你而来'''》（{{lang-en|'''Come for You'''}}）2012年上映的爱情片，為[[台灣|台灣]]、[[中國大陸|中國大陸]]兩岸三地合拍，由[[夏如芝|夏如芝]]、[[林宗仁|林宗仁]](茂伯)、[[郑凯|郑凯]]、[[霍泥芳|霍泥芳]]等演出。

== 剧情 ==
一位患有[[白血病|白血病]]的台灣女孩谢怡佳，在療養院裡擔任志工，偶然間為了完成療養院裡的一位阿公李水发的遗愿，尋找在大陸宁州家鄉的人事物，开始了一段旅程，意外結識了攝影師林德。<ref>{{cite web |url=http://yule.sohu.com/20120503/n342282493.shtml |title=《为你而来》台湾热映 5月底即将登陆内地 |accessdate=2012年5月3日 |publisher=搜狐网 |work= |language =zh-cn}}</ref>

== 花絮 ==
电影男主角原本是[[潘粤明|潘粤明]]出演，但是由于车祸而中断，重新开拍，换成郑凯。<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2010-11-24/18263157133.shtml |title=郑凯《为你而来》恋上夏如芝 纯爷们为爱痴狂 |accessdate=2010年11月24日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

女主角夏如芝曾演出[[周杰伦|周杰伦]]的《髮如雪》、《千里之外》、《青花瓷》、《蘭亭序》。<ref>{{cite web |url=http://ent.qq.com/a/20120603/000147.htm |title=《为你而来》郑凯黑白大转型 夏如芝延续小清新 |accessdate=2012年6月3日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

该片清新唯美的悲剧爱情故事获得好评。<ref>{{cite web |url= http://yule.sohu.com/20120601/n344617410.shtml|title=《为你而来》大获好评 一部值得反复观赏的影片 |accessdate=2012年6月1日 |publisher=搜狐网 |work= |language =zh-cn}}</ref>

该片在台湾[[宜兰县|宜兰县]]、中國大陸[[福建|福建]][[宁德市|宁德市]]取景。<ref>{{cite web |url=http://www.chinadaily.com.cn/micro-reading/dzh/2012-05-25/content_6008924.html |title= 《为你而来》获好评关注 网友期待无植入的电影|accessdate=2012-05-25  |publisher= 中国日报|work= |language =zh-cn}}</ref>
