'''汀角攀樹蟹'''（學名：''Haberma tingkok''），是屬於[[相手蟹科|相手蟹科]]的一種[[螃蟹|螃蟹]]。於香港的[[紅樹林|紅樹林]]發現，物種命名亦源於發現地點[[汀角|汀角]]，是迄今只在香港出現的[[特有種|特有種]]。汀角攀樹蟹是全球極少數能夠攀樹的螃蟹物種，也是該屬''Haberma''和於亞熱帶氣候生活的首個樹棲蟹種，其攀樹行為的[[進化|進化]]途徑未明。現時整體數量不明。

== 外型 ==
雄雌體型相約，身長約8-9毫米，身體扁平，主要是深棕色，有淡棕色斑紋。甲殼方形，足部幼長。螯為橙色，雄性的較粗大，雌性則比較幼<ref>[https://phys.org/news/2017-03-species-terrestrial-crab-climbing-trees.html phys.org] 2017-03-21. {{en}}</ref>。雄蟹第二和第三對步行足擁有特殊的鉤狀結構，可能是用作在交配時抓住雌性<ref>[http://www.hku.hk/f/upload/16168/170410_power_point.pdf HKU marine ecologists name the first endemic tree-climbing crab, Haberma tingkok, from Tolo Harbour] - hku.hk. 2017-04-10.</ref>。

== 習性 ==
生活於[[秋茄樹|秋茄樹]]的枝幹。攀樹行為可能是因為水底空氣不足，或者逃避獵食者而於樹上生活。

== 發現 ==
2016年，[[香港大學|香港大學]]生物科學學院及太古海洋科學研究所轄下，紅樹林生態與進化實驗室的海洋生態學家發現了這個新蟹種。發現地點為大埔[[吐露港|吐露港]]海岸，汀角[[紅樹林|紅樹林]]，海圖基準面上約1.5至1.8米的樹枝上<ref>[https://www.sciencedaily.com/releases/2017/03/170321122557.htm  ScienceDaily] Pensoft Publishers. 2017, March 21. Retrieved April 11, 2017. {{en}}</ref>，當處亦被評定為[[具特殊科學價值地點_(香港)|具特殊科學價值地點]]。這個新物種是香港首種會攀樹的微型蟹，即樹棲蟹。這次新發現肯定了紅樹林仍然存有豐富的[[生物多樣性|生物多樣性]]，即使是位於高度[[城市化|城市化]]地區。

此物種是第二種由香港描述的紅樹林特有蟹種，第一種是在1975年所發現的帕氏假相手蟹（''Pseudosesarma patshuni''）。

== 流行文化 ==
「汀角攀樹蟹」一名容易令人聯想起香港90年代經典劇集《[[大時代|大時代]]》的男主角「[[丁蟹|丁蟹]]」之名，坊間和媒體報導也會以此作噱頭<ref>[http://hk.apple.nextmedia.com/realtime/news/20170410/56545269 港首現攀樹蟹 全球新品種 以大埔「汀角」為名棲息紅樹林] - 蘋果日報. 2017-04-10.</ref>。
