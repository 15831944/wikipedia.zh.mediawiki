'''滷肉飯'''（[[英語|英語]]：Lo bah png；[[台羅|台羅]]：Lóo-bah-pn̄g；[[白話字|白話字]]：Ló͘-bah-pn̄g）是[[臺灣|臺灣]]流行的一道[[豬肉|豬肉丁]]（或[[絞肉|絞肉]]）[[飯|飯]]菜餚，類似[[燒味|港式燒味]]。近年臺灣官方將滷肉飯英譯，改為[[臺灣話|臺灣話]]發音的直譯（如[[壽司|日本壽司]]、[[韓國泡菜|韓國泡菜]]英譯，以當地語言直譯，突顯專有特色）<ref>[http://mag.udn.com/mag/happylife/storypage.jsp?f_ART_ID=325738 幫滷肉飯正名 北市擬以台語英譯]，[[聯合報|聯合報]]，2011-6-20</ref><ref>2011年[[臺北國際牛肉麵節|臺北國際牛肉麵節]]已將[[台灣牛肉麵|臺灣牛肉麵]]英譯定為[[中華民國國語|現代漢語]]直譯的「New Row Mian」</ref>

卤肉饭的特色和制作的关键部分在于肉酱和肉汁，在台南、台中、台北的制作方法和特点均有差异<ref>[https://home.meishichina.com/recipe-344384.html 美食天下-卤肉饭]</ref>。

==種類==
滷肉飯在[[臺灣|臺灣]]被視為極具台灣特色的民眾[[臺灣小吃|小吃]]，在全臺各地都有店家販售，南北有不同的意義。在[[北臺灣|北臺灣]]，滷肉飯為一種淋上含有煮熟碎豬肉或炒香[[肉燥|肉燥]]（豬絞肉）及醬油滷汁的[[白飯|白飯]]菜餚，有時醬汁裡亦會有香菇丁等的成份在內，與[[炕肉飯|炕肉飯]]不同，看起來像[[南臺灣|南臺灣]][[肉燥飯|肉燥飯]]；「滷肉飯」在臺灣南部是指有著滷汁塊狀肉的切丁滷肉飯，有些店家搭配[[筍乾|筍乾]]醃[[蘿蔔乾|蘿蔔乾]]，與彰化的大塊炕肉飯亦不盡相同。<ref>{{cite web | url=http://life.chinatimes.com/LifeContent/1401/20130827000404.html | title=滷肉飯郵票掀波 南部郵迷：沒三層肉 是肉燥飯｜今日總覽｜中時樂活｜中時電子報 | accessdate=2013-08-27}}</ref>南台灣的肉燥飯通常是以肥瘦一定比例的豬絞肉滷製，但北台灣的滷肉飯通常用含有豬皮的肥肉切丁滷製，兩者的口感和口味略有不同。在[[香港|香港]]的臺式餐館，「滷肉飯」大多是使用滷[[五花肉|五花肉]]塊，而「魯肉飯」或「肉燥飯」才大多是使用滷肉碎。<ref>[https://sites.google.com/site/beef2013220000/gu-zao-wei-de-you-lai 滷&魯傻傻分不清楚]</ref>中国大陆的“台湾卤肉饭”则不作区分，使用小块卤肉或者肉燥的都可以称为“卤肉饭”。

== 名稱爭議 ==
「[[滷|滷]]」者，用濃汁烹調食物也。臺灣常見「魯」肉飯，「魯」字原意為[[鲁国|鲁国]]。臺灣美食作家[[朱振藩|朱振藩]]說，根據最早的字典書[[漢朝|漢朝]]《[[說文解字|說文解字]]》判斷，「-{鹵}-」字是正寫，後來出現「-{滷}-」字。<ref>[https://sites.google.com/site/beef2013220000/gu-zao-wei-de-you-lai 滷&魯傻傻分不清楚]</ref>至於「魯」則是因滷字不常用取其音相近者，數十年前在臺灣出現，久用成為慣用用法；按字意正確用字為「[[滷|滷]]」。<ref>{{Cite news|url =http://chuckorz.pixnet.net/blog/post/64410743 |title = 魯肉飯出自山東 朱振藩：99.8%沒錯|last = 記者蘇湘雲|first = |date = 2011-06-19|work = 聯合報|accessdate = }} 美食家胡天蘭...說魯肉飯起源於山東，不予置評，也不便置評</ref>

《禮記‧內則》中的記載的周代飲食：「煎醢，加於陸稻上，沃之以膏曰淳熬。煎醢，加於黍食上，沃之以膏曰淳毋」。熬煮肉醬澆在米飯之上，拌著吃，稱為淳熬，澆到黍米飯上，則稱為淳毋，此食品廣為流傳並隨著時間產生許多區域變化。《康熙字典》引《廣韻》、《水滸傳》中也有提及「肉哨」一詞，也就是現今「肉燥」，因為「哨」與「燥」兩字讀音相近，在寫錯字的情況下，肉「哨」飯就變成現在的肉「燥」飯。<ref>周惠民  飲膳佳會：餐桌上的文化史 三民出版社 2018-01-05 (ISBN)： 9789571463582</ref>

《[[米其林指南|米其林指南]]》曾誤指台灣滷肉飯發源自[[中國|中國]][[山東|山東]]的「魯肉飯」（山東簡稱「魯」）。<ref>{{cite news|author=孫承武|url=http://www.cna.com.tw/SearchNews/doDetail.aspx?id=201106190095|title=滷肉飯正名 此滷非彼魯|work=[[中央社|中央社]]|date=2011-06-19|accessdate=2011-06-28|deadurl=yes|archiveurl=https://web.archive.org/web/20160304104148/http://www.cna.com.tw/SearchNews/doDetail.aspx?id=201106190095|archivedate=2016-03-04}}</ref><ref>{{cite news|author=羅苑韶|url=http://www.cna.com.tw/SearchNews/doDetail.aspx?id=201106210274|title=米其林：指南不聚焦美食|work=[[中央社|中央社]]|date=2011-06-21|accessdate=2011-06-28|deadurl=yes|archiveurl=https://web.archive.org/web/20160610184614/http://www.cna.com.tw/SearchNews/doDetail.aspx?id=201106210274|archivedate=2016-06-10}}</ref>書中還描述其食材是切塊[[豬肉|豬肉]]再加上「[[洋蔥|洋蔥]]」（實為[[紅蔥頭|紅蔥頭]]）等調味做成的食品，令台灣滷肉飯業者大感不解。經查證，可能是翻譯過程出錯，造成撰文的美食家誤會。
