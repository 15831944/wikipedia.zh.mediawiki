'''蚵仔煎'''（[[臺灣閩南語羅馬字拼音方案|臺羅]]：ô-á-tsian）是一種源於[[中國|中國]]东南沿海地區的海鮮食品<ref>[http://www.chinanews.com/cul/2014/05-03/6126594.shtml "舌尖2"掀起潮汕美食热 小店每天中午煎百鼎蚝烙]，中新网，2014年5月3日</ref><ref>[http://www.tech-food.com/kndata/detail/k0137460.htm 舌尖上的中国2：潮汕小吃蚝烙的做法]，中国食品科技网，2014-4-26</ref>。

== 由來 ==
蚵仔煎是名聞閩臺的知名[[小吃|小吃]]，其來源眾說紛紜，最有名的[[傳奇|傳奇]]是當年[[鄭成功攻臺之役|鄭成功攻臺之役]]時，[[荷蘭人|荷蘭人]]將臺南附近所有的米糧全部藏起，以避免資助[[鄭成功|鄭成功]]。鄭軍為了節約白米，於是利用當時海灘找到的[[牡蠣|牡蠣]]裹以[[番薯粉|番薯粉]]煎炸，演變成為「蚵仔煎」<ref>[http://news.ifeng.com/mil/history/detail_2010_12/27/3716952_0.shtml 臺灣美食渊源：蚵仔煎由断粮的郑成功军队发明]</ref> 。

== 名稱 ==
在[[中國|中國]][[閩南|閩南]]地區、[[臺灣|臺灣]]以及[[東南亞|東南亞]]的[[閩南人|閩南人]]等一般將「-{zh-hant:蠔;zh-hans:蚝}-」稱為「蚵-{}-仔」，闽南地区和[[台灣|台灣]]称之为「蚵-{}-仔煎（[[闽南语|闽南语]]國際音標{{IPA|[o a tsian]}}，台羅拼音：[http://twblg.dict.edu.tw/holodict_new/result_detail.jsp?n_no=7847&curpage=1&sample=%E8%9A%B5%E4%BB%94%E7%85%8E&radiobutton=1&querytarget=1&limit=20&pagenum=1&rowcount=1 Ô-á-tsian]）」。[[中國大陸|中國大陸]]其它地區常稱為「牡-{}-蛎煎」或「牡-{}-蛎烙」<ref>[http://news.takungpao.com.hk/paper/q/2014/0507/2460959.html -{zh-hans:蚝;zh-hant:蠔}-煎]，大公報，2014-5-7</ref>，因為胶辽地区把牡蛎俗称为海蛎子，故也有少數人称之为「海蛎煎」，[[香港|香港]]則稱為「煎-{zh-hans:蠔;zh-hant:蠔}-餅」或「-{zh-hans:蠔;zh-hant:蠔}-餅」。[[马来西亚|马来西亚]]一般寫作「-{zh-hans:蚝;zh-hant:蠔}-煎，[[普通話|普通話]]讀為（háojiān），[[閩南語|閩南語]]則讀為（國際音標{{IPA|[o tsian]}}）。[[新加坡|新加坡]][[潮州人|潮州人]]虽然也把它写作「-{zh-hans:蚝;zh-hant:蠔}-煎」，可是把它读为「-{zh-hans:蚝;zh-hant:蠔}-烙」（國際音標{{IPA|[o lua]}}）。[[英語|英語]]一般譯作（Oyster Omelette），中文直譯為「牡蛎蛋餅」。

== 作法 ==
作法是先用[[平底鍋|平底鍋]]把油燒熱，放上蚵仔、攪拌後的雞蛋、蔬菜（如[[茼蒿菜|茼蒿菜]]、[[菠菜|菠菜]]、[[臺灣萵菜|臺灣萵菜]]、[[小白菜|小白菜]]、[[空心菜|空心菜]]、[[豆芽|豆芽]]、[[九層塔|九層塔]]等）後再淋上[[食用澱粉|食用澱粉]]芡水，待蚵仔熟時盛起，淋上特製醬料後即可食用，製作方便，味道鮮美，幾乎每個台灣[[夜市|夜市]]都可以找到賣蚵仔煎的攤位。

其中食用澱粉的種類（常見的有蕃薯、馬鈴薯和樹薯）和其芡水的黏稠度，煎煮的焦度，各地各家也各有不同。尤其是醬汁的配方，從最簡單的[[醬油膏|醬油膏]]、[[番茄醬|番茄醬]]、[[甜辣醬|甜辣醬]]以至於[[豆瓣醬|豆瓣醬]]，許多店家會混合各種醬料，成為獨門配方。

台灣著名的蚵仔煎販賣地點主要是在各大[[夜市|夜市]]。許多蚵仔煎的攤位會兼賣「蟹肉煎」、「花枝煎」、「蝦仁煎」、「蛋煎」等，顧名思義，就是以[[花枝|花枝]]、[[蝦|蝦]]、[[蟹|蟹]]等海產代替蚵仔，或是只有雞蛋、青菜和澱粉芡水的組合。

中國大陸的蚵仔煎以源自廣東[[潮州|潮]][[汕頭|汕]]地區的「煎蠔餅」最著名，其在[[福建|福建]][[閩南|閩南]]沿海地區也是常見的食物。在[[港澳|港澳]]地區並不是每家中式食肆都會供應煎蠔餅，而是集中在潮式菜館提供。香港的煎蠔餅除了在潮州餐館供應外，也有部分[[大排檔|大排檔]]提供此菜式，香港人常把到潮州餐館吃潮州菜稱為「打冷」<ref>[http://paper.wenweipo.com/2004/06/01/OT0406010003.htm 字詞溯源：「打冷」起源]，文匯報，2014-6-1</ref><ref>[http://hk.apple.nextmedia.com/supplement/food/art/20130307/18186576 食字部：打冷食潮　滷鵝唔慳得]，蘋果日報，2013-3-7</ref>。

廣東地區的蚵仔煎只採用鴨蛋、食用澱粉和蚵仔-{}-煎香；香港的蚵仔煎則較注重香脆程度，除了改用雞蛋和麵粉之外，還將煮食方法改成油炸，比較近似於台灣的[[蚵嗲|蚵嗲]]吃法。不過兩地進食煎蠔餅也時常會配上[[魚露|魚露]]，[[胡椒粉|胡椒粉]]及[[豆瓣醬|豆瓣醬]]<ref>[http://news.stheadline.com/dailynews/headline_news_detail_columnist.asp?id=296661&section_name=wtt&kw=257 食乜好 - 正宗潮州香脆蠔餅]，頭條日報，2014-8-8</ref>。

==類似食品==
除了用蚵做成的蚵仔煎之外，還有用其他材料做成的蜆仔煎 、蛤蜊煎 、蝦仁煎、蟹肉煎、花枝煎、蛋煎等口味。[[蚵嗲|蚵嗲]]則是以類似材料油炸而成。
