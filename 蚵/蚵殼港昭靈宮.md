'''蚵殼港昭靈宮'''，是位於[[臺灣|臺灣]][[桃園市|桃園市]][[新屋區|新屋區]]蚵間里的[[王爺廟|王爺廟]]，主祀七夫人媽，配祀七府王爺、[[保生大帝#保生三真人|保生三真人]]、福德正神等神祇。

== 建立緣由 ==
蚵殼港昭靈宮為1979年興建，1993年間全部完竣<ref name="羅平1"/><ref name="彭新茹"/>。廟位在[[台15線|台15線]]60.5公里處<ref name="羅平2"/>，占地幅員廣闊<ref name="彭新茹"/>，廟址為新屋區蚵間里28-7號<ref name="吳世傑、潘瑞如"/>。廟名取作「昭靈宮」之因，乃得名自光緒年間輪值主辦[[送王船|送王船]]的[[泉州府|泉州府]]廟宇之名<ref name="彭新茹"/>。

其建立緣由是之前蚵間的旅外鄉親出資出力重建[[新屋福興宮|新屋福興宮]]時，意見不一，造成信徒分裂，而另建此廟<ref name="羅平1"/><ref name="羅平2"/><ref name="羅平3"/>。據外傳信徒分裂原因之一，是過去村長選舉時所造成的恩怨<ref name="羅平1"/>。兩廟信徒的政治派系並不同，昭靈宮總幹事兼總務許滿村曾言，該廟事務不少都是許姓，早年都是[[黨外運動|黨外運動]]許家班的支持者，他自己也對[[人事室第二辦公室|國民黨黨工]]在漁會體系的作為有怨言<ref name="曹俊漢3"/>。另一分裂原因，是當時以擲筊決定是否遷廟時，擲筊結果支持原地重建的相關人員反悔，造成信徒集資興建新廟<ref name="羅平1"/>。

新廟的建立消耗新台幣六千萬元以上，而舊廟的重建亦花一億餘元，費用皆出自僅有二、三百戶的蚵間村及在外地創業的村民<ref name="羅平2"/>。昭靈宮舉行建醮儀式時，福興宮尚在重建中，兩廟分庭抗禮、暗中較勁，讓地方人士擔憂<ref name="羅平3"/>。後來1996年採訪時，地方人士則表示分家的真正原因非如各方臆測，且兩廟信徒均已相互走動供奉，無隔閡<ref name="羅平1"/>。在1998年報導時，新屋鄉不到四萬人口，卻就擁有七座大型寺廟，而蚵間村就佔了以上兩間<ref name="羅平2"/>。　

== 祭祀神明 ==
主神龕內：鎮殿蕭府王爺、鎮殿王夫人、鎮殿蘇府王爺、二鎮蕭府王爺、開基七位夫人（王、金、李、黃、薛、何、梁），開基七府王爺（蕭、蘇、邢、沈、羅、池、朱）

主神龕前：令牌、[[五營神將|五營神將]]

主神龕前神桌：[[中壇元帥|中壇元帥]]、[[觀音菩薩|觀音菩薩]]

主神龕下方：[[虎爺|虎爺]]

龍邊神龕：[[保生大帝#保生三真人|保生三真人]]（排序為[[許遜|許府]]、[[孫思邈|孫府]]、[[保生大帝|吳府]]真人）、[[天上聖母|天上聖母]]、[[關聖帝君|關聖帝君]]

虎邊神龕：[[福德正神|福德正神]]、

== 祭祀活動 ==
[[File:蚵殼港昭靈宮王爺聖誕表.jpg|thumb]]
以農曆五月十七日為七位夫人、七府王爺誕辰<ref name="羅平2"/>。其中七位夫人為冊封，王爺則屬於隨從<ref name="曹俊漢5"/>。每年農曆正月十四，蚵間里會有添丁居民的酬神儀式，由福興宮及昭靈宮輪辦，如2008年是後者舉行<ref name="曹俊漢1"/>。
