《'''擒爱记'''》（英語：Diaries of the Cheating Hearts），（又名：《'''擒爱记：回收女人心'''》），2012年6月21日上映于[[中国大陆|中国大陆]]的爱情片。

== 剧情 ==
该片讲述嘉兰（莫小棋）和其好友沈巴黎（巩新亮）、何静婷（谭卓）、梅芸（冯丹滢）4个女人的爱情婚姻故事。

== 花絮 ==
该片中的“[[出轨|出轨]]、[[一夜情|一夜情]]、未婚先孕、[[老少恋|老少恋]]、[[床戏|床戏]]”等话题引发观众的担忧，认为会误导青少年观众，灌输不良情爱观。<ref>{{cite web |url=http://news.mtime.com/2012/06/22/1491209.html |title=《擒爱记》大尺度热映 出轨爱情喜剧引争议 |accessdate=2012-06-22 |publisher=时光网 |work= |language =zh-cn}}</ref>

该片的导演[[韩承桓|韩承桓]]来自[[大韩民国|大韩民国]]。<ref name="fish " >{{cite web |url= http://news.mtime.com/2012/06/12/1490445.html|title=爱情喜剧《擒爱记》首映 莫小棋坦言从未"被出轨" |accessdate= 2012-06-12 |publisher=时光网 |work= |language =zh-cn}}</ref>

在电影的发布会上，莫小棋说自己从来没有“被出轨”。<ref name="fish " />

巩新亮首都出演“床戏”。<ref>{{cite web |url=http://news.mtime.com/2012/06/07/1490027.html |title=《擒爱记》曝桃色版预告片 巩新亮首演床戏 |accessdate=2012-06-07  |publisher=时光网 |work= |language =zh-cn}}</ref>

杨金成被誉为“氧气男生”。<ref>{{cite web |url=http://ent.sina.com.cn/m/2012-06-28/14273670641.shtml |title= 《擒爱记》敏感话题引热议 “氧气男生”杨金承受热捧|accessdate= 2012年6月28日 |publisher= 新浪网|work= |language =zh-cn}}</ref>
