《'''遺忘'''》（{{lang-en|'''Forgotten'''}}）是部2012年拍攝的[[電視電影|電視電影]]，由[[林心如工作室|林心如工作室]]與[[東映製作有限公司|東映製作有限公司]]製作的一部迷你劇，劇中則以[[林心如|林心如]]飾演女主角，並於幕後則兼任製作人，又因拍攝花費500萬元，預計拍攝時間半個多月，為求拍攝順利與提升品質，劇組請來李曉蘋擔任編劇，以及[[連奕琦|連奕琦]]擔任執導，並找來[[新加坡|新加坡]]男星[[李銘順|李銘順]]飾演男主角<ref>[http://news.cts.com.tw/nownews/entertain/201208/201208171074758.html 【林心如NG十多次快崩潰 信心險全毀】] {{webarchive|url=https://web.archive.org/web/20160309205032/http://news.cts.com.tw/nownews/entertain/201208/201208171074758.html |date=2016-03-09 }}華視新聞網 2012年08月17日</ref>。《遺忘》劇組將三項獎項提名[[第47屆金鐘獎|第47屆金鐘獎]]，其結果全部未能獲獎<ref>[http://www.appledaily.com.tw/appledaily/article/entertainment/20121028/34603502 【《遺忘》全軍覆沒 林心如明年再戰】]蘋果日報 2012年10月28日</ref>。

== 簡介 ==
《遺忘》的劇情播放方式，透過劇中人物的回想，將鏡頭拉回過去，爾後又拉回當前的現在，形成劇情將過去與現在的穿插交替。本片主要講述是「一場車禍失憶改變了兩人感情，也挽救瀕臨破碎的婚姻。」

「遺忘」一詞既是本片的片名，也是[[失憶|失憶]]的代名詞。片中女主角因為感情問題，在一場車禍中失去短暫的記憶，經過許醫生會合其它門診醫生討論，解釋出女主角對於自己的婚姻、感情有關的記憶產生「遺忘」，醫學稱為「{{link-en|解離性神遊|Dissociative fugue}}」。

== 劇情大綱 ==
片中開場則以劇中男女主角約在餐廳談判，卻意外被女主角發現小三在隔壁桌竊聽，引發的誤會，造成女主角情緒失控地跑出餐廳外，導致車禍傷及女主角腦部而產生[[失憶症|失憶症]]。而後夫妻兩人的生活產生了巨大的變化，並漸漸找回當初的初衷。<ref>[http://www.pts.org.tw/program/Template1B_About.aspx?PNum=443&PURL=Forget 公共電視《遺忘》]</ref>

== 音樂 ==
劇中結束前，以《牽手》做為配合劇情的一段插曲，於男主角在車上打開收音機所播送這首，透過電話呈現給女主角，是意圖挽回彼此婚姻關係的一種方式。《牽手》收錄於1993年發行專輯《[[牽手_(蘇芮專輯)|牽手]]》，曾為大愛劇場《[[牽手人生|牽手人生]]》主題曲、電影《[[喜宴_(電影)|喜宴]]》片尾曲，歌手[[卓依婷|卓依婷]]、[[區瑞強|區瑞強]]皆曾翻唱過。
