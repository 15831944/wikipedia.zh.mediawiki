'''溴酸钾'''是一个[[无机盐|无机盐]]，室温下为无色[[晶体|晶体]]，分子式为[[鉀|K]][[溴|Br]][[氧|O]]<sub>3</sub>。

其在发酵、醒发及焙烤工艺过程中起到一种[[氧化剂|氧化剂]]的作用，使用了溴酸钾后的面粉更白，制作的面包能快速膨胀，更具有弹性和韧性，在焙烤业被认为是最好的[[面粉|面粉]]改良剂之一。溴酸钾有致癌性，现在已被許多國家（如欧盟）禁用，但在美国仍允许使用。溴酸钾在足够长的烘烤时间和温度下会耗尽，但是如果在面粉中添加的太多就会有残留。

中华人民共和国卫生部于2005年5月30日发布《2005年第9号公告》称，根据溴酸钾危险性评估结果，决定自2005年7月1日起，取消溴酸钾作为面粉处理剂在小麦粉中使用。在此之前按照《食品添加剂使用卫生标准》（GB2760-1996）使用溴酸钾的食品可以在产品保质期内继续销售。

== 氧化剂 ==
KBrO<sub>3</sub>是一种强[[氧化剂|氧化剂]]，酸性溶液其[[氧化还原|氧化还原]][[半反应|半反应]]和[[标准电极电势|标准电极电势]]为：
:BrO<sub>3</sub><sup>−</sup> + 6H<sup>+</sup> + 6e<sup>−</sup> ←→ Br<sup>−</sup> + 3H<sub>2</sub>O; E<sup><s>o</s></sup> = 1.44 V
溴酸钾易提纯，可作为[[基准物质|基准物质]]直接配制标准溶液，其浓度可用间接[[碘量法|碘量法]]标定：
:BrO<sub>3</sub><sup>−</sup> + 6I<sup>−</sup> + 6H<sup>+</sup> → 3I<sub>2</sub> + Br<sup>−</sup> + 3H<sub>2</sub>O
接下来以[[淀粉|淀粉]]为[[指示剂|指示剂]]，用[[硫代硫酸钠|Na<sub>2</sub>S<sub>2</sub>O<sub>3</sub>]]标准溶液[[滴定|滴定]]。

酸性溶液可用KBrO<sub>3</sub>标准溶液直接滴定As（III）、Sb（III）和Sn（II）等[[还原|还原]]性物质。但溴酸钾主要用于测定[[有机物|有机物]]。[[溴|溴]]可与很多有机物发生定量反应（如[[取代反应|取代反应]]、[[加成反应|加成反应]]等），但其水溶液不稳定，不能用作标准溶液。因此在溴酸钾标准溶液中加入过量[[溴化钾|KBr]]并酸化，可生成等计量的Br<sub>2</sub>，相当于Br<sub>2</sub>标准溶液：
:BrO<sub>3</sub><sup>−</sup> + 5Br<sup>−</sup> + 6H<sup>+</sup> → 3Br<sub>2</sub> + 3H<sub>2</sub>O

例如溴酸钾法测定[[苯酚|苯酚]]时，先加入过量KBrO<sub>3</sub>-KBr标准溶液并酸化，使苯酚[[溴化|溴化]]生成[[2,4,6-三溴苯酚|2,4,6-三溴苯酚]]和白色[[2,4,4,6-四溴环己二烯酮|2,4,4,6-四溴环己二烯酮]]沉淀（[[三溴酚溴|三溴酚溴]]）。然后加入过量I<sup>−</sup>还原过量Br<sub>2</sub>为[[溴离子|Br<sup>−</sup>]]及三溴酚溴为三溴苯酚，再用硫代硫酸钠标准溶液滴定生成的[[碘|I<sub>2</sub>]]，即可间接求出样本中苯酚的含量。
