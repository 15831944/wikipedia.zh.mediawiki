'''溴化镨'''是一种[[无机化合物|无机化合物]]，化学式为PrBr<sub>3</sub>。<ref name="a"/><ref name="Pubchem"/>

==制备==
溴化镨可以通过镨和溴反应制得：
: 2 Pr + 3 Br<sub>2</sub> → 2 PrBr<sub>3</sub>


==性质==
===外观===
溴化镨在室温下是绿色固体。<ref name="a"/><ref name="Web elements">{{cite web|last1=Winter|first1=Mark|title=Praseodymium»praseodymium tribromide [WebElements Periodic Table]|url=https://www.webelements.com/compounds/praseodymium/praseodymium_tribromide.html|website=www.webelements.com|accessdate=20 December 2016}}</ref>

===物理性质===
溴化镨的分子量为380.62 g/mol，<ref name="a"/><ref name="Pubchem">{{cite web|title=Praseodymium bromide Br3Pr - PubChem|url=https://pubchem.ncbi.nlm.nih.gov/compound/83561#section=Chemical-Vendors|website=pubchem.ncbi.nlm.nih.gov|accessdate=20 December 2016}}</ref><ref name="Chem">{{cite web|last1=Chambers|first1=Michael|title=ChemIDplus - 0013536533 - PLKCYEBERAEWDR-UHFFFAOYSA-K - Praseodymium bromide (PrBr3) - Similar structures search, synonyms, formulas, resource links, and other chemical information.|url=https://chem.nlm.nih.gov/chemidplus/sid/0013536533|website=chem.nlm.nih.gov|accessdate=20 December 2016}}</ref><ref name="Handbook"/>[[密度|密度]]为5.28 g/cm<sup>2</sup>。<ref>{{cite web|title=Information on praseodymium bromide: Etacude.com|url=http://chemicals.etacude.com/p/praseodymium_bromide.php|website=chemicals.etacude.com|accessdate=20 December 2016}}</ref><ref name="Handbook">{{cite book|last1=Phillips|first1=edited by Dale L. Perry, Sidney L.|title=Handbook of inorganic compounds|date=1995|publisher=CRC Press|location=Boca Raton|isbn=9780849386718|page=323}}</ref>溴化镨具有潮解性。<ref name="Chem"/>

===化学性质===
溴化镨可以发生如下化学反应：
: PrBr<sub>3</sub> + 3 NaOH → Pr(OH)<sub>3</sub>↓ + 3 NaBr
: 2 PrBr<sub>3</sub> + 3 Cl<sub>2</sub> → 2 PrCl<sub>3</sub> + 3 Br<sub>2</sub>
