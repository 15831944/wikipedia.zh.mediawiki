亚汞的溴化物可以存在于一种名为kuzminite（Hg<sub>2</sub>(Br,Cl)<sub>2</sub>）的矿中。

==结构==
亚汞化合物包含直线型的X-Hg-Hg-X单元，Hg<sub>2</sub>Br<sub>2</sub>中Hg-Hg键长249 pm为（Hg-Hg金属键为300 pm），Hg-Br键长为271 pm。<ref name = "Wells">Wells A.F. (1984) ''Structural Inorganic Chemistry'' 5th edition Oxford Science Publications {{ISBN|0-19-855370-6}}</ref>每个Hg原子居于八面体，四个Br原子距离332 pm。<ref name = "Wells"/>该化合物有时候会写作离子形式{{chem|Hg|2|2+|2Br|-}}<ref name = "Cotton">{{Cotton&Wilkinson6th}}</ref>，但它是一种分子化合物。
