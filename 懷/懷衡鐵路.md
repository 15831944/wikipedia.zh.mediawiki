'''怀衡线'''，又名'''懷邵衡鐵路'''，是[[中國|中國]]一條运营中連接[[湖南省|湖南省]][[懷化市|懷化市]]與[[衡陽市|衡陽市]]的[[高速铁路|高速铁路]]。全长约318公里。

==簡介==
懷邵衡鐵路西起[[懷化西站|懷化西站]]，經安江、江口、洞口、石下江（预留）、隆回、岩口铺；與[[洛湛鐵路|洛湛鐵路]]邵永段三線並行至[[邵陽|邵陽]]站，再与新婁邵段经邵阳东站四線並行至邵東新站，然後自[[邵東|邵東]]站貨運場東側分道揚鑣；次第過金兰、西渡、松木園、麻元，最后接入顏家壟，并在此對接[[衡茶吉铁路|衡茶吉鐵路]]，同時將興建聯絡線連入[[衡陽站|衡陽站]]<ref>{{cite web |url=http://www.hnfgw.gov.cn/gmjj/jcss/47638.html |title=存档副本 |accessdate=2013-12-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20131230231708/http://www.hnfgw.gov.cn/gmjj/jcss/47638.html |archivedate=2013-12-30 }}</ref>。

懷邵衡鐵路全長大約318km。該線路于2008年七月初進入《國家中長期鐵路網規劃調整方案》<ref>[http://news.0731fdc.com/fc/20087/31/20080731_09_1758_1.html]</ref>，2010年10月获国家发改委批复立项。原擬於2010年內動工<ref>[http://www.gov.cn/gzdt/2008-12/26/content_1188323.htm]</ref>。2011鐵道部高層人事變動及其後的[[2011年甬台温铁路列车追尾事故|溫州7·23動車追尾事故]]，使得國內所有鐵路新工程上馬延遲<ref>[http://paper.wenweipo.com/2012/03/28/FI1203280019.htm]</ref>。2012年3月列入国家《十二五综合交通运输体系规划》的快速铁路专栏。2012年4月14日鐵道部同意在得到湖南省部分資金墊付的前提下，懷邵衡鐵路與婁邵鐵路四線並行的32km路段可以率先開工<ref>[http://www.sysfgw.gov.cn/view.asp?news_id=1450&classid=4&admin_clas=1]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。懷邵衡鐵路有限責任公司於2012年11月2日成立<ref>[http://www.hnaic.gov.cn/visit/socialservice/a/listEnterprise?marktype=1]</ref>。2013年12月24日可行性研究报告获得国家发改委批复<ref>{{cite web |url=http://www.hnfgw.gov.cn/gmjj/jcss/47638.html |title=存档副本 |accessdate=2013-12-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20131230231708/http://www.hnfgw.gov.cn/gmjj/jcss/47638.html |archivedate=2013-12-30 }}</ref>。2014年1月6日开工<ref>[http://hunan.voc.com.cn/article/201401/201401061729299965.html]</ref>。2014年6月30日全局开工。2018年3月20日上午，全线铺轨贯通。

== 技术标准 ==
懷邵衡铁路設計技術標準<ref>[http://bbs.rednet.cn/a/a.asp?B=74&ID=21759736&q=1&r=3117995&T=0《懷邵衡設計技術標準》]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>：

*鐵路等級：I級；
*正線數目：雙線；
*設計行車速度：2009年邵陽與新化鐵路之爭，邵陽當局在高壓下以放棄滬昆高鐵過境市區為代價，獲得提前開工提升等級的懷邵衡與新婁邵鐵路補償<ref>[http://news.sina.com.cn/c/sd/2009-04-13/140117599861.shtml]</ref>。原設計行車速度為200km/h，預留250km/h<ref>[http://www.chnrailway.com/news/20091217/200912179502220376539.shtml]</ref>；實際設計行車速度線下工程為200km/h，並未預留提升空間<ref>{{cite web |url=http://www.hnfgw.gov.cn/gmjj/jcss/47638.html |title=存档副本 |accessdate=2013-12-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20131230231708/http://www.hnfgw.gov.cn/gmjj/jcss/47638.html |archivedate=2013-12-30 }}</ref>。
*線間距：4.4m；
*最小曲線半徑：一般3500m，困難2800m；
*限制坡度：13‰；
*牽引種類：電力；
*機車類型或動車組類型：貨機HXD3，客機SS9，動車組；
*牽引品質：4000t；
*到發線有效長度：850m（雙機880m）；
*閉塞類型：自動閉塞；
*建筑限界：新建時速200公里客貨共線鐵路建筑限界。
*规划输送能力：客车50对/日，货运5000万吨/年。
