'''土耳其語言協會'''（{{lang-tr|Türk Dil Kurumu}}，'''TDK'''）是[[土耳其語|土耳其語]]的官方{{tsl|en|List of language regulators|語言管理機構列表|管理機構}}，1932年7月12日在土耳其之父[[凱末爾|凱末爾]]的倡議下成立，總部設於[[土耳其|土耳其]][[安卡拉|安卡拉]]。該機構的業務主要包括對土耳其語與其他[[突厥語族|突厥語族]]語言的研究，並負責出版土耳其語官方詞典，《當代土耳其語詞典》（{{lang|tr|Güncel Türkçe Sözlük}}）。

== 歷史 ==
[[File:Meeting_of_the_Turkish_Language_Association,_1933,_Dolmabahçe_Palace.jpg|left]]]]
土耳其語言協會於1932年7月12日由土耳其共和國創始人兼第一任總統[[穆斯塔法·凱末爾·阿塔圖克|穆斯塔法·凱末爾·阿塔圖克]]倡議成立，最初的名字為土耳其語研究學會（{{lang|tr|Türk Dili Tetkik Cemiyeti}}），核心成員如{{tsl|tr|Samih Rifat|薩米·里法}}、{{tsl|tr|Ruşen Eşref Ünaydın|盧申·艾希雷夫·於奈登}}、{{tsl|en|Celâl Sahir Erozan|傑拉爾·薩希爾·埃羅贊}}與{{tsl|en|Yakup Kadri Karaosmanoğlu|雅庫普·卡德里·卡拉奧斯曼盧}}等皆是當時土耳其文學界的著名人物，或是[[土耳其大國民議會|土耳其大國民議會]]的成員。1934年開始，學會的首席專家和秘書長由土耳其－亞美尼亞裔語言學家{{tsl|en|Agop Dilaçar|阿戈普·迪拉恰爾}}擔任，一直到1979年他去世為止。

學會的名稱在1934年改為土耳其語言研究所（{{lang|tr|Türk Dili Araştırma Kurumu}}），兩年後又更名為土耳其語言協會，直至今日。<ref>TDK, [http://www.tdk.gov.tr/index.php?option=com_content&view=article&id=77 Tarihçe]</ref>

== 功能 ==
[[File:TurkishVocabulary.png|thumb]]
土耳其語言協會主導該國的[[土耳其語|土耳其語]]與中亞[[突厥語族|突厥語族]]的學術語言研究，並出版土耳其語官方字典《當代土耳其語詞典》、《寫作指南》（{{lang|tr|Yazım Kılavuzu}}）以及其他數種專業詞典、語言學書籍和期刊。

在1930至1940年間，凱末爾藉由協會的力量推動了許多大型改革，將原本在土耳其語中已使用數世紀的[[阿拉伯語|阿拉伯語]]、[[波斯語|波斯語]]、[[希臘語|希臘語]]與[[法語|法語]]借詞強行廢除，使得當時的書面語與民間口語有明顯的不同。之後的學者將改革前的土耳其語稱為[[鄂圖曼土耳其語|鄂圖曼土耳其語]]，而將改革後流傳開來、逐漸為民間接受的新式語言稱為[[土耳其語|現代土耳其語]]。

由於用來取代舊字彙的新詞過於做作又極富政治色彩，凱末爾和協會的強硬作風招致了國內一些批評聲浪。土耳其語言學家H.C.胡尼曾在1947年提到這個協會「幾乎都由政客組成」，其會議記錄充滿著「荒唐愚蠢的廢話」。胡尼還認為，協會運作的三個目標是「渴望與過去徹底切割」、「渴望被視為歐式國家的一員」和「實踐狹隘且狂熱的民族主義」。<ref>{{cite book| last =Hony| first =H. C.| title =The New Turkish| publisher =Journal of the Royal Asiatic Society| date=December 1947}}</ref>

== 出版刊物 ==
[[File:Turkish_Language_Association_building.jpg|thumb]]的土耳其語言協會大樓]]
土耳其語言協會除了維護《當代土耳其語詞典》和《寫作指南》之外，還出版了850多種語言學相關書籍，主要包括對[[突厥語族|突厥語族]]諸語、專業詞典、[[語文學|語文學]]書籍和文學作品的研究。

此外，該機構還定期出版以下的期刊：
* 《土耳其語》（Türk Dili）, 土耳其語文學期刊，1951年開始發行；
* 《公告》（Belleten），突厥語族研究年刊，1953年開始發行；
* 《土耳其語世界》（Türk Dünyası），關於土耳其語言與文學的半年刊，1996年開始發行。

== 字彙 ==
2005年出版的《當代土耳其語詞典》中，共包含了104,481個詞條，其中有86%是土耳其語原生詞彙，其他14%則來自其他語言；<ref name="Dictionary">{{cite web|url=http://www.tdk.gov.tr/TR/BelgeGoster.aspx?F6E10F8892433CFFAAF6AA849816B2EFB40CE59E171C629F|title=Güncel Türkçe Sözlük|accessdate=2007-03-21|year=2005|publisher=Turkish Language Association|language=tr|deadurl=yes|archiveurl=https://web.archive.org/web/20070312162345/http://www.tdk.gov.tr/TR/BelgeGoster.aspx?F6E10F8892433CFFAAF6AA849816B2EFB40CE59E171C629F|archivedate=2007-03-12|df=}}</ref>字典也明確地指出，在當代土耳其語中，外來語的來源以[[阿拉伯語|阿拉伯語]]、[[法語|法語]]、[[波斯語|波斯語]]、[[義大利語|義大利語]]、[[英語|英語]]及[[希臘語|希臘語]]為最多。<ref name="Loanwords">{{cite web|url=http://www.tdk.gov.tr/TR/BelgeGoster.aspx?F6E10F8892433CFFAAF6AA849816B2EF1A46C5FBFA979D0C|title=Türkçe Sözlük (2005)’teki Sözlerin Kökenlerine Ait Sayısal Döküm (Numerical list on the origin of words in ''Türkçe Sözlük (2005)'')|accessdate=2007-03-21|year=2005|publisher=Turkish Language Association|language=tr|deadurl=yes|archiveurl=https://web.archive.org/web/20070301064559/http://www.tdk.gov.tr/TR/BelgeGoster.aspx?F6E10F8892433CFFAAF6AA849816B2EF1A46C5FBFA979D0C|archivedate=2007-03-01|df=}}</ref>
