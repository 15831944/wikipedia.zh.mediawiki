'''土佐一宮站'''（{{jpn|j=土佐一宮駅|hg=とさいっくえき|rm=Tosa-ikku eki|f=y}}）是一位於[[日本|日本]][[高知縣|高知縣]][[高知市|高知市]]一宮徳谷、隸屬於[[四國旅客鐵道|四國旅客鐵道]]（JR四國）的[[鐵路車站|鐵路車站]]。車站編號為'''D43'''。

==車站結構==
現時站舍為第二代單層[[混凝土|混凝土]]簡易式車站，只設有候車室及自動售票機。車站旁邊設有洗手間。第一代為木製建築，70年代末期至80年代初被拆毀。

穿過站舍後，可沿斜道前往1號月台。但輪椅乘客卻無法前往2號月台候車。

[[File:Tosa-Ikku_station_02.jpg|thumb]][[阿佐線|阿佐線]]的列車正在避車。]]
[[File:Tosa-Ikku_station_01.jpg|thumb]]圖案的下行特急列車正由2號月台通過中。]]

本站最初的設計為側式月台2個，其中原1號月台為避線，並能讓高知開來的列車在此交換駕駛員，1998-99年期間，為配合[[高知運轉所|高知運轉所]]由[[高知站|高知站]]遷往布師田的新址，本站路軌重新安排。<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/dosan/st-ikku_990501a.jpg |title=存档副本 |accessdate=2012-08-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304070841/http://www.shikoku.org.uk/JR/LINE/dosan/st-ikku_990501a.jpg |archivedate=2016-03-04 }} 1999年拍攝的照片，可見到右邊新月台正興建中，以及原兩個側式月台。</ref>改建後，原2號月台變成島式月台（其中靠近新路軌一邊被圍杆所封閉），1號月台為新的側式月台。變成[[側式月台|側式月台]]及[[島式月台|島式月台]]各1個，共提供2面2線的配置。原1號月台的路軌則變成3號線路軌（但不設月台，原月台已拆掉），用作回廠之用。<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/dosan/st-ikku_j2.jpg |title=存档副本 |accessdate=2012-08-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304080049/http://www.shikoku.org.uk/JR/LINE/dosan/st-ikku_j2.jpg |archivedate=2016-03-04 }} 從2號月台向薊野方向，可以看到三軌線道：最左邊為側線；中間為主線往2號月台，右邊為副主線往1號月台。</ref>

兩邊月台的有效長度皆為4-{zh-tw:節;zh-hk:卡;zh-cn:节;}-車廂長。

== 歷史 == 
* 1925年12月5日：開業。
* 1970年10月1日：無人化、簡易委託化。
* 1980年代初：原本站舍改建為簡易車站。<ref>[http://df50shikoku.web.fc2.com/ekisha/dosan/shikakochi.htm 當時的四國站舍]</ref>
* 1987年4月1日：日本國鐵分割民營化，車站的營運由JR四國繼承。
* 1999年：月台及路軌重新編配。
