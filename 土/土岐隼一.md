'''土岐 隼一'''（{{bd|1989年|5月7日||}}）是[[日本|日本]]男性[[声優|声優]]，身高172cm，血型为A型。出身於[[東京都|東京都]]。隸屬於[[EARLY_WING|WITH LINE]]。AMUSEMENT MEDIA綜合學院聲優學科畢業，从2013年开始从事声优活动，曾在数部作品中为主角配音<ref>{{cite web |title=声優・土岐隼一のラジオ「喫茶トキノワ」放送100回記念ゲストは永野由祐 |url=https://news.mynavi.jp/article/20180705-659834/ |website=mynavi.jp |accessdate=2018-08-15}}</ref><ref>{{cite web |author1=アニメ部 |title=「土岐隼一」声優インタビュー＆ミニグラビア【声優図鑑＋】 |url=https://ddnavi.com/interview/438598/a/ |website=ddnavi.com |accessdate=2018-08-15}}</ref>。
