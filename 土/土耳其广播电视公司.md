'''土耳其广播电视公司'''（{{lang-tr|'''Türkiye Radyo Televizyon Kurumu''' / '''TRT'''}}）是土耳其的国营广播电台和电视台，其国际广播呼号为“土耳其之声”。

[[File:Voice_of_Turkey_Interval_Signal.ogg|thumb]]

[[File:İstanbulRadyoevi.JPG|thumb]]

== 历史 ==
土耳其之声前身为1926年成立的土耳其广播电台（Radio Turkey，簡稱RT）。

== 电视节目 ==
目前TRT设有14个[[电视|电视]]频道，分别用[[土耳其语|土耳其语]]、[[英语|英语]]、[[阿拉伯语|阿拉伯语]]等语言播出节目。

== 广播节目 ==
土耳其之声使用41种语言对全球广播，包括[[中文|中文]]和[[维吾尔语|维吾尔语]]。普通话节目播出时间是北京时间19:00-20:00（夏季），频率15240千赫；20:00-21:00（冬季），频率13630千赫。由于频段特性等原因，该台信号在中国东南沿海地区接收较为困难。
