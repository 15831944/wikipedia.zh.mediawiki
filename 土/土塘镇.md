== 人口 ==

全镇人口5.8万多人，外出务工者所占比例大。

== 语言 ==

全镇使用都昌方言，方言的覆盖面达100%，沟通均使用方言。

== 经济作物 ==

农业以水稻为主，兼种油菜、小麦、大豆等。

== 历史发展 ==

以驻地土塘镇命名。建国前夕属濒湖区土塘乡。建国初属第五区土塘乡；杭桥、桂林乡；嘴下、万里等乡。
* 1958年设立土塘公社，属马涧区。1962年分为杭桥、化民、土塘3个公社。
* 1968年合并为土塘[[公社|公社]]。
* 1973年又分为杭桥、化民、土塘3个公社。
* 1984年分别改乡。
* 1993年土塘撤乡设镇。
* 2002年，杭桥乡、化民乡并入土塘镇。
