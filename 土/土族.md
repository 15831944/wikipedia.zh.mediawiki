'''土族'''为[[中国西北|中国西北]]的一个民族，主要聚居在[[青海省|青海省]]东部的[[互助土族自治县|互助土族自治县]]以及[[民和|民和]]、[[大通|大通]]两县，其余散居在[[同仁|同仁]]、[[乐都|乐都]]、[[门源|门源]]以及[[甘肃|甘肃]]的[[天祝|天祝]]等地。土族源于[[鲜卑|鲜卑]]族的一支，曾在中原[[隋朝|隋朝]]时期建立[[吐谷浑|吐谷浑]]政权。

== 族源 ==
[[File:Tibetan_Tu_costumes.jpg|thumb]]

“土族”族名源于部分土族人自称“土昆”、“蒙古尔”，“察罕蒙古爾”，蒙古族称土族为“白韃靼”，藏族称其为“霍尔”。藏族很早就知道蒙古爾人，[[東胡|東胡]]有一部分人前往青海遊牧，藏族稱為[[霍爾|霍爾]]。土族是東胡一部分，因此稱為霍爾。[[沙陀|沙陀]]與[[汪古|汪古]]及吐谷渾是最主要的族源。

土族的祖先是古代的[[鲜卑|鲜卑]]人一个支系，[[隋朝|隋朝]]时曾在[[青海|青海]]一带建立强大的[[吐谷浑|吐谷浑]]国，在[[唐朝|唐朝]]时受到[[吐蕃|吐蕃]]的挤压，沦为唐的附庸。[[宋代|宋代]]时和[[羌族|羌族]]、[[党项|党项]]人联合建立[[西夏|西夏]]帝国。西夏被[[蒙古|蒙古]]人灭亡后，吐谷浑人加入到蒙古的军队，在[[元朝|元朝]]时地位较高，和西域[[色目人|色目人]]同等待遇。他们其实是吐谷浑人与不同时间来到的蒙古与[[汪古|汪古]]军人混血种，[[明代|明代]]以后逐渐定居，改为从事农业生产，并逐渐和[[汉族|汉族]]、[[蒙古族|蒙古族]]等融合，简称为“土人”，清朝因為[[土司|土司]]服從，不用土族[[改土歸流|改土歸流]]，清政府大力推崇[[藏传佛教|藏传佛教]]，导致土族人口减少。到了[[清朝|清朝]]末年只余下不到两万人。[[中华人民共和国|中华人民共和国]]成立后，由于执行了鼓励少数民族文化的政策，人口迅速增加。

== 语言 ==
過去土族只有語言，沒有文字。[[土族語|土族語]]屬[[阿爾泰語系|阿爾泰語系]][[蒙古語族|蒙古語族]]，基本詞彙和蒙古語相同或相近。由於過去通用[[漢語|漢語]]，所以土語的詞彙從漢語吸收了大量的[[借词|借词]]，亦有部分借词從[[藏语|藏语]]而來。1979年，青海省有關部門制定了以中文拼音為基礎的《土文方案（草案）》，已試行推廣。

== 宗教信仰 ==
土族普遍信仰[[藏传佛教|藏传佛教]]，由于受汉族影响，还崇拜祖先，信奉[[关帝|关帝]]、[[二郎神|二郎神]]、[[家神|家神]]、[[灶神|灶神]]、[[门神|门神]]、[[财神|财神]]、[[菩萨|菩萨]]等。 

土族人居住的地方多在山谷间，种的地多半是山地，所以他们非常崇信山神。在村头、村后山头上堆一个土堆，上面插上一块木牌，写着“敕令山神土地守地界”。认为这样能阻挡暴雨，保护庄稼。每代[[章嘉呼图克图|章嘉呼图克图]]出自土族。

== 风俗习惯 ==
土族主要从事[[畜牧业|畜牧业]]和[[农业|农业]]，尤其精于养[[羊|羊]]。
 
土族建筑特点突出，别具一格。农村一般以村落聚居，村庄大多在山脚下，依山傍水搭造房屋。房子是平顶的，上面可储放粮草。房子多以三间为一组，中为堂屋，一侧为卧室，中一侧为佛堂。卧室的炕是暖炕，连着锅灶，烧饭的火可以暖炕。住宅的栋梁和门窗上大多雕刻着或描绘象征牛羊健壮、五谷丰登的彩色花纹图案，鲜艳美观。 

== 重要节日 ==
[[File:Nadun_Picture_1.jpg|thumb]]

土族人民的重要节日有，农历[[正月十四|正月十四]]佑宁寺官经会，[[二月二|二月二]]威远镇擂台会，[[三月|三月]]三、[[四月|四月]]八庙会，[[六月十一|六月十一]]丹麻戏会，[[六月十三|六月十三]]、二十九“少年”会，[[七月廿三|七月廿三]]至九月民和[[三川地区|三川地区]]的“纳顿”（庆丰收会）等等。其中擂台会、丹麻戏会和“纳顿”最具民族特色。届时，除举行赛马、摔跤、武术和唱“花儿”等传统娱乐活动外，还举行物资交流会。此外，与[[汉族|汉族]]一样，也过[[春节|春节]]、[[端午节|端午节]]等。民和三川地区社火「演过」描述了隋唐王朝西征吐谷浑兵败而逃的历史。
