'''土城區'''是[[台灣|台灣]][[新北市|新北市]]的一個[[市轄區|市轄區]]，屬於[[台北都會區|臺北都會區]]的[[衛星城市|衛星城市]]之一，位於[[台北盆地|台北盆地]]西南角，東北與[[板橋區_(台灣)|板橋區]]、[[中和區|中和區]]相鄰；西隔[[大漢溪|大漢溪]]與[[樹林區|樹林區]]相望；東與[[中和區|中和區]]相鄰；越過東南方丘陵地帶與[[新店區|新店區]]毗鄰；南臨[[三峽區|三峽區]]。原為土城鄉，因臨近的板橋發展為都會中心的外溢效應，使土城發展隨之興旺。目前行政區域共劃分47里、1,236鄰。最近有許多大型都市計畫，又有[[捷運板南線|捷運板南線]]支持，近期土城將成為開發完整、適宜居住的地區。

== 歷史 ==
'''土城區'''地名源於當地防備的「城牆」（今區公所一帶），[[臺灣清治時期|清治時期]]行政劃分隸屬於「[[淡水廳|淡水廳]][[擺接堡|擺接堡]]」，[[台灣日治時期|日治時期]]時隷屬「[[臺北州|台北州]][[海山郡|海山郡]]-{[[土城庄|土城庄]]}-」，當時海山郡尚包括[[板橋街|板橋街]]、[[三峽街|三峽街]]、[[鶯歌街|鶯歌街]]、-{[[中和庄|中和庄]]}-（今之[[板橋區_(台灣)|板橋]]、[[三峽區|三峽]]、[[鶯歌區|鶯歌]]、[[樹林區|樹林]]、[[中和區|中和]]、[[永和區|永和]]）等地。戰後地方自治區域重新改制，土城改制為台北縣土城鄉。由於人口持續增加，工商日益發達，於1993年6月26日人口數超過15萬人後，正式改制為台北縣土城市<ref>「[http://web.law.ntpc.gov.tw/Scripts/newsdetail.asp?no=1B0040010 台灣省各縣市實施地方自治綱要]」第七條第二款（此法規於中華民國八十八年十月十二日臺灣省政府 88 府法字第 093796 號令發布廢止）</ref>，為原臺北縣的第7個縣轄市。2010年12月25日[[五都改制|五都改制]]，改為新北市土城區，2016年底人口已上升至23萬多人<ref>{{Cite journal|title=新北市土城區各里鄰戶口數|author=新北市土城區公所|url=http://www.tucheng.ntpc.gov.tw/archive/file/tg105-0622.pdf|date=2017年6月|journal=新北市土城區政統計年報|issue=|doi=|others=|year=2017|volume=|page=10|pmid=}}</ref>。

;主要道路
*[[中央路|中央路]]<ref name="tai3line" >部份路段編入[[台3線|台3線]]</ref>：土城早期街道之一，貫穿土城主要區域，北接[[板橋區_(台灣)|板橋區]]四川路，南接[[三峽區|三峽區]][[介壽路|介壽路]]，是土城公司行號最多、最主要之道路。
*[[中華路|中華路]] ([[台3線|台3線]])：土城外環道路，主要紓解中央路車流，並可透過城林橋至[[樹林區|樹林區]]。<ref name="tai3" >依照行政院公告，台三線已於2007年11月8日起，自中央路1段與中華路1段路口起，訖中央路3段與中華路2段路口止，改行中華路，原台3線中央路路段解編為市區道路。</ref>
*金城路<ref name="88line" >部份路段編入[[北88線|北88線]]</ref>：北起[[中和區|中和區]]連城路，南迄土城[[中央路|中央路]]，為土城區通往[[中和區|中和區]]之主要道路。沿途經[[新北地方法院|新北地方法院]]、[[台北看守所|台北看守所]]、捷運[[土城站_(新北市)|土城站]]、土城區公所等。此地段人口密集、商家眾多，是土城熱鬧的商圈道路。
*裕民路（[[北88-1線|北88-1線]]）：東接金城路，西接中央路、中華路，途經捷運[[海山站_(新北市)|海山站]]、土城綜合運動場。此地段人口密集、商家眾多，是土城熱鬧的商圈道路。
*青雲路（[[北89-1線|北89-1線]]）：清水地區和[[宏國德霖科技大學|宏國德霖科技大學]]地區主要道路，可連接到板橋後埔地區。
*學府路、立德路、[[明德路|明德路]]：三條道路相連接呈圓形狀，[[新北市立清水高級中學|新北市立清水高中]]、[[新北市立新北高工|新北市立新北高工]]、[[新北市立廣福國民小學|新北市立廣福國小]]即位於此路段上。
*[[中山路|中山路]]：貫穿土城工業區之主要道路。
*清水路：為清水地區之主要道路，起訖處皆與金城路相接。

== 工商發展 ==
土城的產業原本以傳統工業（如[[海山煤礦|礦業]]）為主，藍領職缺居多，在前縣長[[蘇貞昌|蘇貞昌]]的積極推動開發下，產業結構開始逐漸轉型，其中以[[頂埔科技園區|頂埔科技園區]]最具代表性，與[[板橋區_(新北市)|板橋區]]的[[台北遠東通訊園區|台北遠東通訊園區]]、[[中華電信雲端產業園區|中華電信雲端產業園區]]、[[樹林區|樹林區]]的[[大同科技園區|大同科技園區]]逐漸形成[[新北市|新北市]]最重要的新興產業[[聚落|聚落]]。由[[捷運永寧站|捷運永寧站]]延伸的[[捷運頂埔站|捷運頂埔站]]通車後與捷運[[板南線|板南線]]連成—氣，進一步帶動土城的升級發展。目前土城與鄰近[[板橋區_(新北市)|板橋區]]的發展關係，近似於[[臺北市|臺北市]][[南港區|南港區]]與[[信義區_(臺北市)|信義區]]的情況。 

*[[土城工業區|土城工業區]]
*[[頂埔科技園區|頂埔高科技園區]]：原址為[[陸軍後勤學校|陸軍後勤學校分部]]（陸軍運輸兵學校），位於土城區西南一隅，後方有國道三號通過，近大暖坑一山區。園區正前方為土城重要幹道中央路四段，接近捷運頂埔站，交通便捷。進駐此園區的廠商包括[[正崴電子|正崴]]、[[大霸電子|大霸]]與[[鴻海精密|鴻海]]，分別建造了正崴科技大樓、大霸電子總部及鴻海高科技中心三棟科技大樓，廠商已進駐運作。

*[[日月光廣場_(臺灣)|日月光廣場]]：位於學府路之購物中心。

== 趣聞 ==
*土城區有條水溝流經「柑林里」、「埤林里」，因此取名為「柑林埤溝」，附近還有座「柑林埤」圖書閱覽室，讀起音發音與台語三字經相近；土城清水路到裕民路一帶，過去盛產柑橘又有池塘，因此被稱為「柑林埤」。
*許多民眾第一次看到土城「柑林埤」，總聯想到台語「Ｘ你爸」的三字經，民眾也會相互開玩笑，但事實上，柑林「埤」正確念法應是注音的「ㄆㄧˊ」，而不是「ㄅㄟ」，希望大家用健康的心態看待。<ref>{{cite web|author=林金池、葉德正、李忠一|url=https://www.chinatimes.com/newspapers/20110712000643-260102|title=北七路柑林埤溪北里怪名蒐奇 居民自娛娛人|work=中國時報|date=2011-07-12}}</ref>
