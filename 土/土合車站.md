'''土合站'''（{{jpn|j=土合駅|hg=どあいえき|rm=Doai eki|f=y}}）是[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[上越線|上越線]]的一座[[鐵路車站|鐵路車站]]，位於[[日本|日本]][[群馬縣|群馬縣]][[利根郡|利根郡]][[水上町|水上町]]。

== 概要 ==
[[File:Doai-Station-0014.jpg|thumb]]
土合站位於{{link-ja|三國山脈|三国山脈}}中[[谷川岳|谷川岳]]的腳下，同時也是群馬縣內的普通鐵道火車站中位於最北方的一座<ref name=navi/>。該站最大的特點是下行月台位於[[清水隧道_(日本)|新清水隧道]]內，要出站或者前往上行月台的話必須先爬上一條462級、長338公尺的樓梯，然後經過一條143公尺長、24個梯級高的聯絡通道，下行月台與[[檢票口|檢票口]]間的高低差大約70公尺。因此，本站又被稱為「日本第一的地下車站」。在日本發行的[[公共交通时刻表|列車時刻表]]中，會在「[[高崎線|高崎線]]·上越線·[[信越本線|信越本線]]下行」頁面旁標註「從土合站的檢票口到下行月台需要大約10分鐘」，而在[[有人車站|有人車站]]時代更是會在發車前10分鐘就停止檢票<ref>{{web cite|url=https://www.sankei.com/west/news/180629/wst1806290003-n1.html|script-title=ja:欄外は情報の宝庫　謎の「ご注意」が上越線ページに|publisher=[[產經新聞|產經新聞]]|accessdate=2018/08/17|date=2018.6.29}}</ref>。另外，在站內的樓梯上還設置有可供乘客休憩的[[長椅|長椅]]<ref name=navi/>。

雖然在一年中大多數時間乘客都很少，但有許多[[登山客|登山客]]會在夏秋時節來此車站攀登谷川岳<ref name=navi>{{web cite|url=http://www.gunmachan-navi.pref.gunma.jp/reserved/07.php|script-title=ja:日本一のモグラ駅「土合駅」|publisher={{lang|ja|ぐんまちゃんナビ - 群馬県広報課}}|accessdate=2018/08/17}}</ref>。在[[橫山秀夫|橫山秀夫]]的小說《[[超越巔峰_(電影)|登山者]]》以及2005年的同名電視劇、2008年的同名電影的開頭，也有對該車站內場景的描寫。<ref>{{web cite|url=http://www.asahi.com/travel/location/TKY200911100289.html|script-title=ja:「クライマーズ・ハイ」　土合駅（群馬県みなかみ町）|publisher=[[朝日新聞|朝日新聞]]|accessdate=2018/08/17|date=2018年3月11日}}</ref>

== 歷史 ==
* 1931年9月1日：上越線水上至越後湯澤站間開通，本站作為號誌站開業<ref name=teishajo>{{Cite web |url=http://sankei.jp.msn.com/life/trend/080426/trd0804260821001-n1.htm|archiveurl=https://web.archive.org/web/20101128103641/http://sankei.jp.msn.com/life/trend/080426/trd0804260821001-n1.htm|script-title=ja:【停車場ストーリー】JR上越線・土合駅 “地底”70メートル 日本一のモグラ駅|publisher=MSN[[產經新聞|產經新聞]] |date=2008-04-26 |archivedate=2010-12-07 |accessdate=2018/08/17}}</ref>。
* 1936年12月19日：升級為車站<ref name=teishajo/>。
* 1967年9月28日：因上越線部分區間改為復線，新清水隧道開通。該隧道內設置的本站下行月台啟用<ref name=teishajo/>。
* 1985年3月：改為無人站<ref name=teishajo/>。
* 1987年4月1日：[[日本國鐵分割民營化|日本國鐵分割民營化]]，車站管理權由東日本旅客鐵道承繼。
*2000年：被選入「[[關東車站百選|關東車站百選]]」<ref>{{Cite web|url=http://wwwtb.mlit.go.jp/kanto/tetudou/station100/gunma/eki100_5gunma.html|script-title=ja:関東の駅百選 - 群馬県|accessdate=2018-08-18|publisher={{lang|ja|[[国土交通省|国土交通省]]関東運輸局}}}}</ref>。
*2013年：JR東日本出於防火考慮而關閉部分可供登山客宿營的站內區域<ref name=tokyo/>。
*2016年4月27日：因有登山客在站舍內的候車室中使用明火，JR東日本出於防火考慮而將候車室關閉，且近期無重開計劃<ref name=tokyo>{{Cite web |url=http://www.tokyo-np.co.jp/article/national/list/201605/CK2016051902000264.html |archiveurl=http://archive.fo/p9Vad |script-title=ja:JR土合駅 登山者愛用の待合室が閉鎖 宿泊者の火気使用相次ぎ「安全のため」 |publisher=[[東京新聞|東京新聞]] |date=2016-05-19 |archivedate=2016-05-26 |accessdate=2018/08/17}}</ref>。
*2017年9月：該站上下行月台間的聯絡通道與附近的清水隧道、新清水隧道以及湯檜曾[[展線|螺旋線]]等一同被{{link-ja|公益社團法人土木學會|土木学会}}指定為{{link-ja|土木学會選獎土木遺產|土木学会選奨土木遺産}}<ref>{{web cite|url=https://www.asahi.com/articles/ASK9V3HYGK9VUHNB001.html|script-title=ja:群馬）ＪＲ清水トンネルなど鉄道７施設、土木遺産に|publisher=[[朝日新聞|朝日新聞]]|accessdate=2018/08/17|date=2017年9月27日}}</ref>。

== 利用狀況 ==
民營化後，本站於1993年度的每日平均乘車人數曾達到最高值43人，此後則呈下降趨勢。另外，群馬縣並未公布2014年以後無人車站的乘車人數<ref name=gunma/><ref>{{web cite|url=http://toukei.pref.gunma.jp/nenkan/|script-title=ja:群馬県統計年鑑各年版|publisher={{lang|ja|群馬県統計情報提供システム}}|accessdate=2018/08/16}}</ref>。
