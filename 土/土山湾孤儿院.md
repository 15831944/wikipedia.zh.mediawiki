'''土山湾孤儿院'''（{{lang-fr|'''L' Orphelinat de T'ou-sè-wè'''}}）是一家曾有的教会孤儿院，1864年由法国[[耶稣会|耶稣会]]建立在[[上海|上海]][[徐家汇|徐家汇]]之南的[[土山湾|土山湾]]地区而得名。

1840年，以法国耶稣会为主的大批外国传教士来到上海，在徐家汇建造天主堂、修道院、天文台、藏书楼、教会学校、慈善机构等，形成了[[:Category:原徐家汇上海天主教教务中心机构|上海天主教教务中心]]<ref name="人民网">{{cite web |url = http://www.people.com.cn/GB/8196/76966/77005/7407997.html |title=上海举办“土山湾文化历史讲坛”|publisher=人民网|date=2008-06-20}}</ref>，土山湾孤儿院也是在此背景下建成的。

== 土山湾孤儿工艺院 ==
土山湾孤儿院在抚养孤儿之余，还创办了土山湾孤儿工艺院，对绘画、雕塑、印刷、木刻、金工等艺术领域都有所涉及，很多新工艺都发源于此，土山湾对中国近代绘画、照相、印刷出版、海派雕刻等手工技艺有着深远的影响，为近代中西方文化的交流产生了十分积极的作用，书画大师[[任伯年|任伯年]]、[[刘海粟|刘海粟]]、[[徐悲鸿|徐悲鸿]]都曾在此任教<ref name="shgov">{{cite web|url=http://www.shanghai.gov.cn/shanghai/node2314/node2315/node17239/node17241/userobject21ai280806.html|title=20日新民晚报与徐汇区联合举办土山湾文化历史讲坛 唤醒“中国西洋画摇篮”记忆|publisher=[[上海市人民政府|上海市人民政府]]|date=2008-06-17}}</ref>。徐悲鸿直言:“土山湾亦有可画之所,盖中国西洋画土山湾老照片之摇篮也，称土山湾画馆为「中国西洋画的摇篮」。<ref name="人民日报海外版">{{cite web|url=http://unn.people.com.cn/GB/9719407.html|title=土山湾牌楼回家记|publisher=人民日报海外版|date=2009-07-24}}</ref>。中国雕塑艺术奠基人[[张充仁|张充仁]]、雕刻工艺大师[[徐宝庆|徐宝庆]]均在土山湾开始了艺术启蒙<ref name="人民网" />。

== 土山湾文化保护现状 ==
[[中华人民共和国|中华人民共和国]]成立后，土山湾孤儿工艺院于1960年关闭，孤儿院的各个工艺部门也在之前的[[公私合营|公私合营]]中被一一拆分。土山湾的众多文化遗迹在[[文化大革命|文化大革命]]中遭受破坏，其历史价值长期不被重视。改革开放之后，特别是2000年代后，政府逐渐加强了对土山湾文化的研究与保护<ref name="shgov" />，开始着手建立[[土山湾博物馆|土山湾博物馆]]，在[[2010年上海世博会|2010年上海世博会]]召开期间的2010年6月12日正式开馆。
