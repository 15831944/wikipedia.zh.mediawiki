'''土卫十九'''又稱為「[[伊米爾|伊米爾]]」(S/2000 S 1, Ymir)，是环绕[[土星|土星]]运行的一颗[[卫星|卫星]]。

== 概述 ==

土卫十九是2000年8月7日被一个由[[布雷特·格莱德曼|布雷特·格莱德曼]]和[[约翰·卡佛拉斯|约翰·卡佛拉斯]]领导的天文学家小组在[[智利|智利]][[拉-西拉|拉-西拉]]的[[欧洲南方天文台|欧洲南方天文台]]发现的。他最初的临时名字是''S/2000 S 1''，后来用[[北欧神话|北欧神话]]中的一个冰巨人命名为[[伊米尔|伊米尔]]。

== 轨道数据 ==

土卫十九环绕土星的椭圆形轨道的平均半径是23,130,000千米，它的公转周期是1315日又8小时。其[[偏心率|偏心率]]为0.3339，相对于土星的赤道它的轨道的倾角为173.104°。因此它的公转方向与土星的自转方向相反。

== 构造和物理数据 ==

土卫十九的直径为16千米，它的密度为2.3g/cm<sup>3</sup>，比土星的其它卫星较高，它可能主要由[[冰|冰]]加上很多[[硅土|硅土]]组成。它的表面比较暗，其[[反照率|反照率]]仅为0.06，也就是说，只有6%的入射阳光被[[反射|反射]]。

它的亮度只有21.7等，因此是一个极暗的天体。
