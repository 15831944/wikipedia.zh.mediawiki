'''-{zh:土库曼斯坦;zh-hans:土库曼斯坦;zh-hant:土庫曼斯坦;zh-hk:土庫曼斯坦;zh-mo:土庫曼斯坦;zh-tw:土庫曼;}-'''（{{lang-tk|Türkmenistan/Түркменистан/تۆركمنيستآن}}）是一個[[中亞|中亞]]國家，前[[蘇聯加盟共和國|蘇聯加盟共和國]]之一，[[蘇聯|蘇聯]]時期的名稱為[[土库曼蘇維埃社會主義共和國|土库曼蘇維埃社會主義共和國]]，1991年從[[蘇聯|蘇聯]]獨立。

== 歷史 ==
{{main|土库曼历史}}
公元前3000年，土库曼境內出現[[社会阶层|階級社會]]。公元前600年左右後，先后被[[阿契美尼德王朝|阿契美尼德王朝]]、[[塞琉古帝国|塞琉古帝国]]、[[大夏_(中亚古国)|大夏]]統治。後來被[[安息帝國|安息帝國]]之[[阿尔沙克王朝|阿尔沙克王朝]]（前247年－224年）統治，期間又被[[贵霜帝国|贵霜帝国]]入侵。

7世紀，阿拉伯人征服土库曼後，當地人民遂漸改信[[伊斯蘭教|伊斯蘭教]]。[[烏古斯人|烏古斯人]]在8-9世紀來到這里。

[[蒙古帝国|蒙古帝国]]在1219年－1221年征服[[花剌子模|花剌子模]]後，土库曼归属[[察合台汗国|察合台汗国]]。1370年－1506年，[[帖木儿帝国|帖木儿帝国]]时代，才基本形成土库曼民族。

16世紀到17世紀，隸屬[[希瓦汗國|希瓦汗國]]和[[布哈拉汗國|布哈拉汗國]]。19世紀60－70年代，併入[[俄罗斯帝国|俄罗斯帝国]]。<ref>{{cite book |author=勒内·格鲁塞 |authorlink=雷纳·格鲁塞 |coauthors=蓝琪译 |title=[[草原帝国|草原帝国]] |year=2007 |publisher=商务印书馆 |location=北京 |isbn=978-7-100-02862-2 }} </ref>

1917年底，主要領土併入[[突厥斯坦蘇維埃社會主義自治共和國|突厥斯坦蘇維埃社會主義自治共和國]]，屬[[俄羅斯聯邦|俄羅斯聯邦]]。1924年10月，成立土库曼蘇維埃社會主義共和國，成為[[蘇聯|蘇聯]]加盟共和國。

1991年10月27日，宣佈獨立，改稱[[土库曼斯坦共和國|土库曼斯坦共和國]]，同年12月21日，加入[[獨立國家聯合體|獨立國家聯合體]]。[[蘇聯解體|蘇聯解體]]後，土库曼成為獨立的國家。[[薩帕爾穆拉特·尼亞佐夫|薩帕爾穆拉特·尼亞佐夫]]于1999年成為[[終身總統|終身總統]]，實行獨裁統治。2005年8月26日，在[[獨聯體|喀山會議]]上宣佈退出[[獨立國家聯合體|獨立國家聯合體]]。

2006年12月21日，總統[[薩帕爾穆拉特·尼亞佐夫|薩帕爾穆拉特·尼亞佐夫]]因心臟驟停逝世。[[古爾班古雷·別爾德穆罕默多夫|古爾班古雷·別爾德穆罕默多夫]]成為代總統。<ref>王作葵，[http://news.xinhuanet.com/world/2006-12/21/content_5518022.htm 土库曼斯坦副總理臨時代行總統職權]，新華網</ref>

2007年2月11日，舉行新的總統選舉。<ref>魏良磊，徐力宇，[http://news.xinhuanet.com/world/2007-02/11/content_5727178.htm 土库曼斯坦舉行總統選舉]，新華網</ref>最終由代總統別爾德穆罕默多夫當選。<ref>王作葵，徐力宇，[http://news.xinhuanet.com/world/2007-02/14/content_5739899.htm 別爾德穆罕默多夫當選土库曼斯坦總統]，新華網</ref>

== 政治 ==
{{main|土库曼斯坦政治|土库曼軍事}}
[[File:Jack-up-rig-in-the-caspian-sea_1.JPG|缩略图]]

土库曼斯坦的前身為[[蘇聯|蘇聯]]的一個中亞地區加盟共和國，在1991年[[蘇聯瓦解|蘇聯瓦解]]之后，成为獨立國家，實行[[總統制|總統制]]。自1985年加盟共和國時期就出任當時的[[土库曼共產黨|土库-{}-曼共产党]]中央[[第一書記|第一書記]]的[[薩帕爾穆拉特·阿塔耶維奇·尼亞佐夫|薩帕爾穆拉特·阿塔耶維奇·尼亞佐夫]]，在1991年12月繼任土库曼斯坦[[總統|總統]]，隨即將擁護其執政的土库曼共產黨改名為[[土库曼斯坦民主黨|土库曼斯坦民主黨]]，落實該黨配合土库曼斯坦政府的施政和政治立場。獨立後的土庫曼斯坦雖然與[[波羅的海三國|波羅的海三國]]一樣沒有加入[[獨立國家聯合體|獨立國家聯合體]]，卻以聯繫國身份加入有關組織，然而有關身份至今仍未正式獲得憲章批准。

尼亞佐夫被尊稱为[[土库曼巴希|土库-{}-曼巴希]]，他在國內推行[[個人崇拜|個人崇拜]]，於1994年起修改該國的[[憲法|憲法]]，逐步將自己行使的總統權力擴大，在1999年獲無限期執行總統職權；傚法[[古羅馬|古羅馬]][[凱撒大帝|凱撒大帝]]更改12個月份名稱—自己名字是一月，母親名字是四月，如此類同。2005年2月，尼亞佐夫宣佈放棄終身總統地位的待遇，並將下屆總統選舉定在2010年舉行，而尼亞佐夫本人在2006年12月21日因[[心臟驟停|心臟驟停]]而逝世，享年67歲。

[[外交|外交]]方面，聯合國在1995年12月12日承認土库曼斯坦為一個[[永久中立國|永久中立國]]。土库曼斯坦奉行中立國和全方位外交政策，故此土库曼政府領導層和民間團體向來只是選擇性地參加區域聯盟活動。土库曼斯坦並没跟鄰近的中亞新興國家一样，在獨立之後靠近[[美國|美國]]，反而由於土库曼斯坦與美國的「仇家」[[伊朗|伊朗]]因為在[[能源|能源]]運輸上有很多大型合作，關係良好而令土、美兩國關係並不緊密。而土库曼斯坦和[[俄羅斯|俄羅斯]]亦因歷史淵源而在經濟和軍事聯繫上有著糾纏不清的關係。

近年土庫曼斯坦在外交上越趨「獨來獨往」，又逼使國內俄羅斯族人只能選擇俄羅斯或土库曼其中之一為國籍，若放棄土库曼斯坦國籍就會喪失在土库曼的所有財產。

== 地理 ==
[[File:Turkmenistan_satellite_photo.jpg|thumb]]
{{main|土库曼斯坦地理}}
土库曼斯坦位於[[伊朗|伊朗]]以北，東南面和[[阿富汗|阿富汗]]接壤、東北面與[[烏茲別克|烏茲別克]]為鄰、西北面是[[哈薩克|哈薩克]]，西邊毗鄰鹹水湖[[裏海|裏海]]，是一個[[內陸國家|內陸國家]]。面積49.12萬平方公里<ref name="routeplanning">[http://news.xinhuanet.com/ziliao/2002-06/18/content_445963.htm 土库曼斯坦概況]。</ref>，是僅次於[[哈薩克斯坦|哈薩克斯坦]]的第二个最大中亞國家；在全世界排名第52，比[[西班牙|西班牙]]略小。

土库曼斯坦全境大部是低地，平原多在海拔200米以下，80%的領土被[[卡拉庫姆沙漠|卡拉庫姆沙漠]]覆蓋，餘下的大多數都屬於橫跨土库曼斯坦、烏茲別克及哈薩克的[[圖蘭低地|圖蘭低地]]的範圍。南部和西部為[[科佩特山脈|科佩特山脈]]和[[帕羅特米茲山脈|帕羅特米茲山脈]]，最高點有2912米。位於最西方的[[土库曼巴爾坎山脈|土库曼巴爾坎山脈]]及位於最東方的[[庫吉唐套山脈|庫吉唐套山脈]]（Kugitang）是其比較重要的高地。主要河流有[[阿姆河|阿姆河]]、[[捷詹河|捷詹河]]、[[穆爾加布河|穆爾加布河]]及[[阿特列克河|阿特列克河]]等，主要分佈在東部。於1967年建成的[[卡拉庫姆運河|卡拉庫姆運河]]長達1400公里，橫貫東南部並灌溉面積約30萬公頃。

土库曼斯坦位處於喜馬拉雅——地中海地震帶上，時常受[[地震|地震]]威脅。首都阿什哈巴德曾經在1948年10月6日發生災難性的地震，導致多人罹難。

土库曼斯坦位處亞洲大陸的中心處，故此屬於典型的[[溫帶大陸性氣候|溫帶大陸性氣候]]，這裡是世界上最乾旱的地區之一。年度平均溫度為14℃-16℃，日夜和冬夏的溫差很大，夏季氣溫長期高達35℃以上（在東南部的Karakum曾經有50℃的極端紀錄），冬季在接近阿富汗的山區Gushgy，氣溫亦可以低至-33℃。年降水量則由西北面沙漠的80毫米，遞增至東南山區的每年240毫米，雨季主要在[[春季|春季]]（1月至5月）。科佩特山脈是全國降雨量最高的地區。

土库曼斯坦靠近[[裡海|裡海]]的海岸線有1768公里長，貨物經水路出口須經過俄羅斯的[[伏爾加河|伏爾加河]]和[[頓河|頓河]]。

主要城市除了首都[[阿什哈巴德|阿什哈巴德]]以外，還有臨海的城市[[土库曼巴希|土库曼巴希]]（Turkmenbaşy，舊名克拉斯諾沃斯克/Krasnovodsk）及[[達沙古茲|達沙古茲]]（Daşoguz）。

== 人口 ==
[[File:Turkman_girl_in_national_dress.jpg|thumb]]
人口505萬人（2011年），當中[[土库曼族|土库曼族]]為大多數佔77%，其次[[烏茲別克族|烏茲別克族]]佔9.2%，[[俄羅斯族|俄羅斯族]]佔6.7%，[[哈薩克族|哈薩克族]]佔2%，其他少數民族包括[[亞美尼亞人|亞美尼亞人]]、[[阿塞拜疆族|阿塞拜疆族]]、[[韃靼人|韃靼人]]等。人口分佈極不平均，絕大部份人口聚居在城市和[[綠洲|綠洲]]，當中有很多非土库曼族人在城市居住，他們是在蘇聯時期移居至土库曼斯坦的技術人員、工人等的後裔。而近三分之二的土库曼族人則住在村落。

土库曼族以往是遊牧民族，現多以務農為生。在沙俄入侵吞併前族人分成了多個部落，但是沙俄的兼併使部落間更團結。土库曼人的定義是父親有土库曼血統以及[[遜尼派|遜尼派]][[穆斯林|穆斯林]]，牧人的地位比農人高，只有窮困的人才做農人，他們出售馬匹，有時出賣[[奴隸|奴隸]]。

== 經濟 ==
[[File:500_Manat_TM_1999.png|thumb]]
國內生產總值（2004年）：988萬億馬納特（約合190億美元）。
人均國內生產總值（2004年）：156萬馬納特。(3000美元
)國內生產總值增長率（2004年）：21.4%。
* 貨幣名稱：馬納特（Manat）
* 匯率：1美元＝23,700馬納特（2006年黑市匯率，官方匯率為1：5200）。

土库曼斯坦擁有豐富的天然氣（世界第五）和石油資源，石油天然氣工業為該國的支柱產業。而農業方面則以種植棉花和小麥為主，亦有畜牧業（[[阿哈爾捷金馬|阿哈爾捷金馬]]、[[卡拉庫爾羊|卡拉庫爾羊]]）。


=== 旅遊 ===
[[File:Darvasa_gas_crater_panorama.jpg|thumb]]氣坑全景圖]]
土库曼的旅遊業，尤其是醫療觀光，在土库曼創造了[[裏海|裏海]]的Avaza旅遊區後快速成長。<ref>{{cite web|url=http://www.aljazeera.com/news/asia/2011/12/20111230155633146491.html|title=Las Vegas on the Caspian?|work=aljazeera.com}}</ref> 每個要進入土库曼的遊客都需要在進入前獲得簽證，而大部分國家的旅客需要透過當地的旅行社獲得簽證。土库曼的旅遊行程時常包含[[達沙古茲|達沙古茲]]、[[庫尼亞烏爾根奇|庫尼亞烏爾根奇]]、[[尼薩|尼薩]]、[[梅爾夫|梅爾夫]]、[[馬雷|馬雷]]、Avaza的海灘旅遊以及 在Mollakara、Yylly suw、Archman等地舉行的慶典。

== 文化 ==
[[File:Ashgabat_museum_8.jpg|thumb]]

[[File:Independence_Day_Parade_-_Flickr_-_Kerri-Jo_(105).jpg|缩略图]]

在18世紀之前，土库曼地區受[[伊斯蘭文化|伊斯蘭文化]]影響。18世紀[[沙俄|沙俄]]侵佔中亞並派俄羅斯人到中亞開墾發展，俄羅斯的文化影響漸漸滲透。但自蘇聯瓦解、土库曼斯坦成為獨立國家之後，又由於俄羅斯在當地影響力逐減和國家大力提倡伊斯蘭文化，令兩者此消彼長。
=== 民族 ===
[[土库曼族|土库曼族]]85％，[[烏茲別克族|烏茲別克族]]5％，[[俄羅斯族|俄羅斯族]]4％

=== 語言 ===
[[土库曼語|土库曼語]]為官方語言。[[俄語|俄語]]在土库曼斯坦仍然通用，另外在烏茲別克族群聚居地方亦流行[[烏茲別克語|烏茲別克語]]。

== 宗教 ==
{{Main article|土庫曼斯坦宗教}}
[[伊斯兰教|伊斯兰教]]為主要宗教，佔總人口89%，主要是[[遜尼派|遜尼派]]信徒。另外的9%人口為[[東正教|東正教]]徒。但在強權總統[[薩帕爾穆拉特·阿塔耶維奇·尼亞佐夫|薩帕爾穆拉特·阿塔耶維奇·尼亞佐夫]]的統治下，大力提倡伊斯蘭文化，使信奉東正教的少數俄羅斯人逐漸移出土库曼斯坦，以致東正教在該國的影響力漸衰。
