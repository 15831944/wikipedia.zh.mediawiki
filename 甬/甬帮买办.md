'''甬帮买办'''，又称'''宁波帮买办'''或'''宁波籍买办'''，是[[中国历史|中国历史]]上最大的地域性买办群体。自[[清朝|清]]末1840年[[鸦片战争|鸦片战争]]兴起，[[日本侵华战争|日本侵华战争]]前后衰弱，活跃时间前后横跨一个世纪之久。经营遍布[[银行业|银行业]]、[[纺织业|纺织业]]、矿业、染料业、[[航运业|航运业]]等。买办多出自[[宁波府|宁波府]]（今[[宁波|宁波]]、[[舟山|舟山]]地区），故名。甬帮买办为中国工商业、金融业的近现代化贡献了力量。

== 历史背景 ==
[[宁波|宁波]]古称[[明州|明州]]，有悠久的对外贸易历史。明州对外贸易，在[[唐朝|唐]][[宋朝|宋]]时已很发达。宋朝政府在明州设置市舶司，是世界最早的官方[[海关|海关]]机构。唐宋时，明州已成为东方大港，也是闻名遐迩的“[[海上丝绸之路|海上丝绸之路]]”的主要起点之一。明州在唐宋时已与[[高丽|高丽]]、[[日本|日本]]、[[东南亚|东南亚]]、[[阿拉伯世界|阿拉伯地区]]建立了活跃的海上贸易。同时，因为宁波地理位置的优势，水路向北可到达[[天津|天津]]，逆[[长江|长江]]可到达[[武汉|武汉]]，逆[[钱塘江|钱塘江]]可达[[杭州|杭州]]，南下可到达[[台湾|台湾]]、[[泉州|泉州]]、[[广州|广州]]等地，所以明州也是国内航运的主要中心之一。

[[明朝|明朝]]时期，宁波曾爆发了严重的中日贸易争端，史称“[[宁波之乱|宁波之乱]]”。清朝实施[[海禁|海禁]]，海路对外贸易衰落。[[乾隆|乾隆]]年间，全中国仅有广州获准成为对外贸易口岸。乾隆二十二年（1757年），清政府明文规定[[英国|英国]]商船“只许在广州收泊交易，不得再赴宁波”，并“不准入浙江海口”。宁波对西方直接贸易因此中断。[[鸦片战争|鸦片战争]]之后，宁波成为“[[五口通商|五口通商]]”口岸之一，对外贸易因此激增。由此宁波地区也诞生了特殊的商人群体，即为“[[买办|买办]]”。宁波籍买办不仅在宁波本地经营，还去[[上海|上海]]、[[天津|天津]]、[[北京|北京]]、[[汉口|汉口]]、[[香港|香港]]等地，上海崛起成为宁波籍买办的活动中心。除地理因素之外，因为宁波本地及宁波籍籍商人有雄厚的[[钱庄|钱庄]]作后盾，并创立了“过帐码头”制度，外商更愿意雇佣宁波籍商人充任买办<ref name="甬帮买办优势">{{cite web|url=http://lz.book.sohu.com/chapter-13848-111257484.html|title=掘金战争——虞洽卿上海滩创富传奇：3 宁波帮第一人（1）|publisher=[[搜狐|搜狐]] 读书|author=作者：白希；出版社：花山文艺出版社|date=|accessdate=2012年7月22日|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20140314085954/http://lz.book.sohu.com/chapter-13848-111257484.html|archivedate=2014年3月14日}}</ref>。

近代[[上海|上海]]首个[[买办|买办]]是宁波府[[定海|定海]]人[[穆炳元|穆炳元]]<ref name="定海商人">{{cite web |url = http://dhnews.zjol.com.cn/dhnews/system/2009/10/14/011485572_01.shtml |title=论定海商人的特殊地位与杰出贡献 |publisher=定海新闻网 |author=金涛 |date=2009-10-14 |accessdate=2012年7月22日 |language=zh-hans}}</ref><ref name="上海市地方志">{{cite web |url = http://www.shtong.gov.cn/newsite/node2/node2245/node74728/node74749/node74773/node74775/userobject1ai89589.html  |title = 五、洋行买办 |publisher=上海市地方志办公室 |author= |date= |accessdate=2012年7月22日 |language = zh-hans}}</ref>。[[余姚|余姚]]人[[王槐山|王槐山]]成为[[英国|英国]][[汇丰银行|汇丰银行]]在[[东亚|东亚]]的第一任买办。据《海关十年报告(1882～1891)》记载，19世纪80年代，活跃在上海的买办“主要来自宁波”<ref>《上海近代社会经济发展概况(1882－1931)——<海关十年报告>译编》，上海社会科学院出版社 出版，徐雪筠等 译编，1985年，第21页。</ref>。20世纪40年代，[[日本|日本]]学者根岸佶系统研究了中国的买办群体，在他列举的90名著名买办中，有浙江籍（主要为宁波籍）买办43名，广东籍买办7名<ref>根岸佶:《买办制度研究》，日本图书株式会社 出版，1948年，第234～240页。</ref>。根据当代学者马学强的研究，20世纪30、40年代，活跃在上海有籍可寻的500余名买办中，有浙江籍买办245人，其中以宁波籍为主；有广东籍买办58人，其中以香山籍为主<ref>《出入于中西之间——近代上海买办社会生活》，马学强、张秀莉 著，第40页。</ref>。
