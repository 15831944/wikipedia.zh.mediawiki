'''甬台温铁路'''是[[中国|中国]][[浙江省|浙江省]]境内的一条铁路，为[[杭福深客運專線|杭福深客運專線]]的一部分。线路自[[宁波站|宁波站]]至[[温州南站|温州南站]]，在宁波站连接[[杭甬客运专线|杭甬客运专线]]，在温州南站连接[[温福铁路|温福铁路]]，全线长279.09公里，是一条以客运为主、客货兼顾的国家一级铁路。甬台温铁路[[宁波站|宁波]]至[[温州南站|温州]]最短运营时间1小时12分<ref name="D3107时刻表">[http://chaxun.shike.org.cn/ti.asp?c=D3107&n=21857&fz=%C9%CF%BA%A3%C4%CF&dz=%B8%A3%D6%DD D3107次动车组时刻表]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。甬台温铁路于2005年10月27日开工建设，2009年9月28日与[[温福铁路|温福铁路]]一同通车运营<ref name="open-operation" />。
{{TOCnowrap}}

== 建设背景 ==
甬台温铁路是浙江开通的第一条高等级快速铁路。这条铁路的开通使[[宁波|宁波]]和[[温州|温州]]从铁路末端成为铁路枢纽，也使[[台州|台州]]结束了没有铁路的历史。<ref>{{cite web|url=http://zjnews.zjol.com.cn/05zjnews/system/2009/09/14/015818978.shtml|title=台州告别无铁路历史 甬台温、温福铁路将正式通车|date=2009-09-14|accessdate=2011-07-24}}</ref>在此之前，浙江的铁路密度仅为每万人0.27公里，不到全国平均数的一半。特别是浙江沿海地区，铁路标准较低，改造现有线路的投资巨大，而效益并不明显。而新建的甬台温铁路作为沿海铁路的组成部分，能够加强东南沿海地区的联系和交流，同时有利于大宗货物的运输。<ref>{{cite web|url=http://news.hsw.cn/gb/news/2004-11/26/content_1449085.htm|title=浙江首条高速铁路 甬台温铁路将于年底正式动工|publisher=今日早报|date=2004-11-26|accessdate=2011-07-24}}</ref>

== 线路 ==

甬台温铁路正线全长282.39公里，线路自[[宁波站|宁波站]]起与[[北仑支线|北仑支线]]并行至[[宁波东站|宁波东站]]，此后从宁波东站出岔，依次经过[[奉化|奉化]]、[[宁海县_(浙江省)|宁海]]、[[三门|三门]]、[[临海|临海]]、[[黄岩区|黄岩]]、[[路桥|路桥]]、[[温岭|温岭]]、[[乐清市|乐清]]、[[永嘉县|永嘉]]，于[[温州南站|温州南站]]接轨[[温福铁路|温福铁路]]并与[[金温铁路|金温铁路]]联络线相连。全线东临东海，西靠[[天台山_(浙江)|天台山]]、[[雁荡山|雁荡山]]。<ref>{{cite web|url=http://zjnews.zjol.com.cn/05zjnews/system/2009/09/19/015835407.shtml|title=甬台温铁路简介|publisher=浙江在线|date=2009-09-18|accessdate=2011-07-24}}</ref>全线共有隧道73座，占线路总长的30%。最长的隧道为温州市境内的凤凰山隧道，长7979米。全线共有桥梁246座，占总长的37%。最长的桥梁为宁波市境内的宁波特大桥，全长10251米。<ref>{{cite web|url=http://zjnews.zjol.com.cn/05zjnews/system/2009/09/23/015849266.shtml|title=甬台温铁路美在哪里|publisher=杭州网|date=2009-09-23|accessdate=2011-07-24}}</ref>

甬台温铁路建设技术标准为一级双线[[电气化铁路|电气化铁路]]，设计时速为250公里，预留时速300公里条件<ref>{{Cite web |url=http://www.casco.com.cn/news/details.aspx?cid=72339069014638592&id=338 |title=甬台温客运专线FZk-CTC开通报道 |accessdate=2018-03-24 |author=卡斯柯信号有限公司 |date=2009-09-28}}</ref>，建成实际运营速度为200公里，测试时最高时速达到292公里，创下中国铁路有砟轨道列车运行的最高速度<ref name="highest-speed">{{cite web|url=http://zjnews.zjol.com.cn/05zjnews/system/2009/08/26/015776034.shtml|title=甬台温、温福铁路接受检测 动车组跑出最高时速|publisher=温州网|date=2009-08-26|accessdate=2011-07-24}}</ref>。近期运量货运1090万吨，客车46对；远期运量1450万吨，客车60对。铁路投资约为170亿元，2005年10月27日动工，2009年9月28日正式運營<ref name="open-operation">{{cite web|url=http://zjnews.zjol.com.cn/05zjnews/system/2009/09/28/015869217.shtml|title=杭州开出甬台温铁路首趟试运营列车|publisher=浙江在线|date=2009-09-28|accessdate=2011-07-24}}</ref>。

== 车站 ==

== 服务 ==
=== 客运 ===
甬台温铁路运行的客运车辆为[[和谐号电力动车组|和谐号]]动车组。目前运行的车辆有[[CRH1B|CRH1B]]、[[CRH1E|CRH1E]]、[[CRH2A|CRH2A]]、[[CRH2B|CRH2B]]、[[CRH2E|CRH2E]]、[[CRH380B|CRH380B]]和[[CRH380D|CRH380D]]，其中部分车辆为重联编组。<ref>{{cite web|url=http://wlnews.zjol.com.cn/wlrb/system/2009/12/11/011662085.shtml|title=多趟动车近期扩编座位将增加一半|publisher=温岭新闻网|date=2010-02-03|accessdate=2011-07-24}}</ref>车辆分属[[北京局集团|北京局集团]]、[[济南局集团|济南局集团]]、[[上海局集团|上海局集团]]、[[南昌局集团|南昌局集团]]、[[广州局集团|广州局集团]]和[[西安局集团|西安局集团]]运营，车次有[[高速动车组列车|G]]、[[普通动车组列车|D]]两种，经甬台温铁路运行的客运列车最北端可达[[北京南站|北京南站]]，最南端可达[[珠海站|珠海站]]。其中，北京局运营[[北京|北京]]往返[[福州|福州]]的[[G55/56次列车|G55/56次高速动车组]]<ref>{{cite web|url=http://fj.sohu.com/20130702/n380423446.shtml|title=G字头列车昨日首次抵福州 平均时速不超300公里|publisher=东南快报|date=2013-07-02|accessdate=2013-07-05}}</ref>，济南局运营[[济南|济南]]往返温州的G63/64次高速动车组，西安局运营[[西安|西安]]往返温州的G1896/1893、G1894/1895次高速动车组，其余经行江苏、浙江、福建、广东境内的动车则由上海局、南昌局和广州局运营。

=== 货运 ===
甬台温铁路运营[[宁波港|宁波港]]至台州和温州的集装箱班列，运行时速100公里，停靠台州南站和温州西站，抵达台州和温州的时间分别为2小时和3小时57分。<ref name="ningbo-wenzhou-containers">{{cite web|url=http://daily.cnnb.com.cn/nbrb/html/2010-01/15/content_157381.htm|title=甬温海铁集装箱班列昨开行|publisher=宁波日报|date=2010-01-15|accessdate=2011-07-24}}</ref><ref>{{cite web|url=http://www.taizhou.com.cn/news/2011-07/18/content_413163.htm|title=今晨首列集装箱班列驶抵台州铁路南站|publisher=台州日报|date=2011-07-18|accessdate=2011-07-24}}</ref>

== 影响 ==
甬台温铁路的开通对沿线其它交通方式造成了冲击。公路方面，通车之后，杭州、宁波前往温州、福州方向的[[大巴|大巴]]票价纷纷打折。航空方面，杭州到福州的航班纷纷打折和取消，而宁波往福州的航班也已取消。<ref>{{cite web|url=http://zjnews.zjol.com.cn/05zjnews/system/2009/11/04/015993445.shtml|title=甬台温、温福动车组开通 杭—温快客多年来首降价|publisher=都市快报|date=2009-11-04|accessdate=2011-07-24}}</ref><ref>{{cite web|url=http://zjnews.zjol.com.cn/05zjnews/system/2009/10/14/015914773.shtml|title=宁波至温州公路客流剧减 动车凶猛大巴无奈再降价|publisher=2009-10-14|accessdate=2011-07-24}}</ref>

甬台温铁路的开通也带热了沿线的“动车游”市场。浙江前往福州、厦门的游客因为甬台温、温福、福厦线的开通而迅速增长。<ref>{{cite web|url=http://zjnews.zjol.com.cn/05zjnews/system/2010/02/01/016286369.shtml|title=宁波、福州两地将再次迎来动车互游高峰|publisher=宁波晚报|date=2010-02-01|accessdate=2011-08-01}}</ref>
