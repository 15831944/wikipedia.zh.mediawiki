{{Vgname|嘉芙蓮|ja=キャサリン|en=Catherine|cn=凯瑟琳|hk=嘉芙蓮|tw=凱薩琳}}是由[[Atlus|Atlus]]開發，並於2011年2月17日發售的[[平台游戏|平台解谜]][[冒險遊戲|冒險遊戲]]。本作為Atlus首款高畫質遊戲，對應[[PlayStation_3|PlayStation 3]]及[[Xbox_360|Xbox 360]]，之後陸續移植到[[Windows|Windows]]、[[PlayStation_4|PlayStation 4]]和[[PlayStation_Vita|PlayStation Vita]]。

2019年1月11日，遊戲登陆Windows平台（[[Steam|Steam]]）<ref name="Catherine Classic" />，Windows版移植自原版遊戲，提供更高畫質、遊戲內容不變。加強版《凱薩琳 Full Body》將在PlayStation 4與PlayStation Vita平台推出，於2019年2月14日在日本地區發售，該版本新增不少遊戲內容，包含一位新角色「琳」，以及具有更多機制的解謎方塊，增加解謎部分的挑戰性。

== 概要 ==

本作由《[[女神異聞錄系列|女神異聞錄系列]]》的開發小組以[[恐怖遊戲|恐怖]]及[[成人遊戲|成人]]為主題的動作冒險遊戲。

製作人、美術指導、歌曲則由開發《女神異聞錄3》、《女神異聞錄4》的橋野桂、副島成記及目黑將司負責。

而遊戲的動畫部份由[[Studio_4°C|Studio 4°C]]負責製作。開場動畫的背景音樂則選用L-VOKAL的《Yo》。於2010年東京遊戲展中[[科樂美|科樂美]]的攤位中展出。

== 故事 ==

32歲的文森是一個單身的上班族，最近困惱他的問題是相戀五年的女朋友凱薩琳（Katherine）開始在催婚了，由於單身生活過慣了，對於歲數不小的兩人到底一同成家立業好還是各自過活好，深深的讓他無法立即對女朋友答覆決定結婚與否。結果在某天他與一位身材火辣的金髮神秘女孩凱薩琳（Catherine）交談後，這個小惡魔女孩一直誘惑文森和她發生關係，在文森婉拒的當天晚上三杯黃湯下肚後沉沉昏睡，卻遇上了最近他所居住的小鎮中流傳的都市傳說。

{{cquote|如果做到崩壞之塔的夢，沒有知悉這一切的話就會死亡。}}

在文森第一次拼死拼活逃出惡夢的糾纏後，隔天早上看到了許多男子在住處死亡的新聞，而這個噩夢的真相，則是一位對劈腿行為相當不滿的魔女對濫情男子的復仇……

== 遊戲系統 ==
主角文森遊走噩夢與現實之間，而遊戲分為現實世界的冒險部分及噩夢的動作部分。冒險部分中需要文森在酒吧與朋友或戀人進行對話。而噩夢的動作部分會出現謎一樣的怪物，文森只能往上爬，因為掉下來的結果就只有死亡（[[Game_Over|Game Over]]）。

== 争议 ==
本作的Windows版发售前宣布解锁帧率<ref name="Catherine Classic" />，但发售后有部分玩家发现帧率其实是锁30帧的，所谓的解锁帧率限制实际上是多次输出同一帧来“伪造”高帧率的假象<ref>{{Cite web |url = https://www.gamersky.com/news/201901/1143518.shtml |title = 《凯瑟琳》Steam特别好评 但游戏帧数被质疑造假 |author = 夜雨陈灯 |date = 2019-01-11 |publisher = 游民星空 |language = zh-cn |accessdate = 2019-01-11 }}</ref>。

== 加強版 ==
2017年12月，Atlus宣布加強版《凱薩琳 Full Body》將在[[PlayStation_4|PlayStation 4]]與[[PlayStation_Vita|PlayStation Vita]]上推出。<ref>{{cite news |title=成人驚悚冒險遊戲《凱薩琳》將於 PS4 / PS Vita 推出完全版《凱薩琳 Full Body》 |url=https://gnn.gamer.com.tw/5/156765.html |accessdate=2019-01-11 |publisher=[[巴哈姆特電玩資訊站|巴哈姆特電玩資訊站]] |date=2017-12-19}}</ref>加強版於2019年2月14日在日本地區發售，歐美地區暫定於2019年發售（僅發售PS4版本）。該版本提供不少新遊戲內容，諸如新增第三位重要角色「琳」、新劇情、改良遊戲系統、針對女主角提供多位新聲優配音...等等。<ref name="ps4 date"/>
