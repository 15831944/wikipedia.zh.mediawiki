'''凱旋瑞田站'''位於[[高雄市|高雄市]][[前鎮區|前鎮區]]，為[[環狀輕軌|環狀輕軌]]的輕軌車站。鄰近[[青年夜市|青年夜市]]。

== 車站概要 ==
車站代碼為'''C2'''。站區位於[[凱旋路_(高雄市)|凱旋四路]]與[[瑞田街|瑞田街]]口。預計完工啟用初期每日可達兩千人次<ref group="註">此資料為最初設計時之估計，當時[[高雄捷運紅線|紅線]]、[[高雄捷運橘線|橘線]]皆尚未通車，故此估計可能不符時宜</ref>。

== 車站構造 ==
*[[側式月台|側式月台]]兩座。

== 歷史 ==
* 以「布魯樂谷站」為計畫站名。
* 2010年因[[布魯樂谷|布魯樂谷]]結束營業，改以「金銀島站」為計畫站名。
* 2014年11月9日，進行[[高雄捷運輕軌電聯車|輕軌電聯車]]靜態測試一週。
* 2015年3月29日，公布定名為「凱旋瑞田站」<ref group="參">[http://www.thenewslens.com/post/144007/ 高雄環狀輕軌C1-C14站有名字了！最快8月通車]</ref><ref group="註">命名活動第一階段建議站名為凱旋瑞田站、凱旋商圈站、瑞祥高中站；第二階段票選選項為凱旋瑞田站、凱旋瑞竹站、瑞田凱旋站。</ref>
* 2015年10月16日，隨著環狀輕軌第一階段[[籬仔內站|籬仔內]]－[[凱旋中華站|凱旋中華]]完工試營運啟用<ref group="參">自由時報—[http://news.ltn.com.tw/news/life/breakingnews/1248359 高雄輕軌 力拚8月試營運]</ref><ref group="參">中時電子報—[https://tw.news.yahoo.com/%E6%B0%B4%E5%B2%B8%E8%BC%95%E8%BB%8C-8%E6%9C%88%E5%BA%95%E9%9A%8E%E6%AE%B5%E9%80%9A%E8%BB%8A-215031837--finance.html 水岸輕軌 8月底階段通車]，2015年3月2日。</ref><ref group="參">[http://travel.ettoday.net/article/573060.htm?from=fb_et_travel 確定了！「高雄輕軌」10/16開放試乘、8日起受理預約]</ref>。
