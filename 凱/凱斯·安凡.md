'''凱斯·安凡'''（'''Cas Anvar'''）是一名[[加拿大人|加拿大]]演員、配音演員和作家。

== 早年 ==
他出生於[[薩斯喀徹溫|薩斯喀徹溫]]的[[里贾纳|里贾纳]]，父母都是[[伊朗人|伊朗人]]<ref>{{Cite web |url = http://iranian.com/Arts/2003/December/Glass2/index.html |title = Simply human nature |date = 2003-12-05 |publisher = The Iranian |language = en}}</ref>，而他在[[魁北克|魁北克]]的[[滿地可|滿地可]]長大。安凡會說[[英語|英語]]、[[法語|法語]]、[[波斯語|波斯語]]，以及有限的[[阿拉伯語|阿拉伯語]]、[[印地語|印地語]]和[[西班牙語|西班牙語]]。

== 生涯 ==
[[File:Cas_Anvar_at_2015_Fan_Expo_Canada_1.jpg|right]]
安凡最出名的是在17集電視劇《''{{link-en|The Tournament (TV series)|The Tournament (TV series)|The Tournament}}''》中飾演Dr Singh。<ref>{{Cite web |url = http://www.theseriouscomedysite.com/showreview.php?r_id=820 |title = TV & sitcom DVDs - The Tournament : The Complete Series - Hockey Comedy |publisher = The Serious Comedy Site |language = en |deadurl = yes |archiveurl = https://web.archive.org/web/20160304095051/http://www.theseriouscomedysite.com/showreview.php?r_id=820 |archivedate = 2016年3月4日 |df =  }}</ref>他出現在[[積·佳蘭賀|積·佳蘭賀]]的電影《[[啟動原始碼|啟動原始碼]]》中，並在2011年的《[[刺客信條：啟示錄|刺客信條：啟示錄]]》中開始他的配音生涯。安凡與[[娜奧美·屈絲|娜奧美·屈絲]]在2013年的《[[戴安娜_(電影)|戴安娜]]》中飾演[[多迪·法耶茲|多迪·法耶茲]]。在2014年，他被揀選在[[Syfy|Syfy]]科幻電視劇《[[太空無垠|太空無垠]]》中扮演角色<ref>{{Cite web |url = http://www.hollywoodreporter.com/news/dominique-tipper-cas-anvar-join-744880 |title = Dominique Tipper, Cas Anvar Join Syfy's 'The Expanse' |author = Etan Vlessing |date = 2014-10-29 |publisher = The Hollywood Reporter |language = en}}</ref>，於2015年12月開始播放。他也加入在2016年的《[[血變|血變]]》第3季，飾演Sanjay Desai。
