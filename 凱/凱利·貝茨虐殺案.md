'''凱利·安妮·貝茨'''（{{lang-en|'''Kelly Anne Bates'''}}，{{bd|1978年|5月18日|1996年|4月16日}}），英國青少年，17歲時在[[曼彻斯特|曼彻斯特]]被男友'''詹姆斯·帕特森·史密斯'''（{{lang|en|'''James Patterson Smith'''}}，{{circa|1948}}出生）<ref name = "byrne">{{cite news | first = Paul | last = Byrne | title = Tortured to death | work = [[每日镜报|Daily Mirror]] | location = [[倫敦|London]] | date = 1997-11-12 | page = 17}}</ref><ref name = "marriner">{{cite journal | first = Brian | last = Marriner | title = Why this Manchester murderer should never be released | url = https://www.truecrimelibrary.com/product/true-crime-october-2010/ | journal = True Crime | publisher = Forum Press | location = London | date = 2010-10 | pages = 26－28}}</ref>謀殺。她被折磨超過四個星期，眼睛在死前三星期內被挖出，最後在浴缸中溺斃<ref name = "ward">{{cite news | first1 = David | last1 = Ward | first2 = Jamie | last2 = Wilson | title = Man who tortured girl to death jailed for life | work = [[衛報|The Guardian]] | location = London | url = https://archive.org/stream/TheGuardian1997UKEnglish/Nov%2020%201997%2C%20The%20Guardian%2C%20%23112%2C%20UK%20%28en%29_djvu.txt | date = 1997-11-20 | page = 9}}</ref>。

此案由大曼彻斯特警察局的警長約瑟夫·莫納漢（{{lang|en|Joseph Monaghan}}）負責，他表示在警隊任職15年中從沒有看過這麽恐怖的案件。負責檢驗貝茨遺體的[[病理学|病理學家]]威廉·勞勒（{{lang|en|William Lawler}}）表示，貝茨身上的傷是他曾檢驗過的謀殺案死者中最糟糕的<ref name = "ward" />。史密斯曾有虐待前性伴侶的歷史。他拒絕承認謀殺貝茨，在1997年11月19日被判處[[無期徒刑|無期徒刑]]<ref name = "greer">{{Cite book | title = The Whole Woman | first = Germaine | last = Greer | authorlink = 吉曼·基爾 | publisher = A. A. Knopf | location = [[紐約|New York]] | year = 1999 | isbn = 978-0-375-40747-5 | pages = 296－297}}</ref><ref>{{cite news | title = Murderer of girlfriend jailed | work = {{tsl|en|Irish Times|愛爾蘭時報|Irish Times}} | location = [[都柏林|Dublin]] | url = https://www.irishtimes.com/news/murderer-of-girlfriend-jailed-1.128517 | date = 1997-11-20 | page = 12}}</ref>。

== 背景 ==
詹姆斯·史密斯住在[[曼彻斯特|曼彻斯特]]{{link-en|戈頓 (大曼徹斯特)|Gorton|戈頓}}，是一名[[離婚|離婚]]的失業人士。身邊的熟人形容他「驕傲自滿」和「整潔乾淨」，他亦是一位禁酒者和非吸菸者。1980年，他的10年婚姻在他對妻子的暴力行為後結束<ref name = "marriner" />。1980年至1982年間，他和當時20歲的蒂娜·華生（{{lang|en|Tina Watson}}）交往，並視對方為「出氣袋」，甚至在對方懷孕期間嚴重毆打對方<ref name = "marriner" /><ref name = "greer" />。她说：「最初只是輕輕拍打，但後來每天如是。他會摑我的臉，或用菸灰缸擊打我的頭。他會踢我的腿或者雙腿之間<ref name = "bunyan">{{Cite news | first = Nigel | last = Bunyan | title = Man who blinded and drowned lover jailed for life | work = [[每日电讯报|The Daily Telegraph]] | location = London | date = 1997-11-20}}</ref>。」史密斯還曾試圖在她沐浴時淹死她，因此她逃離這段關係。1982年，史密斯與15歲的溫蒂·莫特斯克（{{lang|en|Wendy Mottershead}}）展開關係，但溫蒂也成為史密斯的暴力受害者之一<ref name = "ward" />。史密斯曾將她的頭壓進廚房水槽，企圖把她淹死<ref name = "marriner" /><ref name = "ward" /><ref name = "greer" />。

1993年，史密斯遇上14歲的凱利·貝茨，當時她是史密斯朋友的保姆<ref name = "myall">{{cite news | first = Steve | last = Myall | work = {{tsl|en|Manchester Evening News|曼徹斯特晚報|Manchester Evening News}} | location = [[曼徹斯特|Manchester]] | title = I wish I'd killed the monster who murdered our daughter the first time I saw him... | url = http://www.manchestereveningnews.co.uk/news/greater-manchester-news/wish-id-killed-monster-who-8694736 | date = 2015-02-22 |accessdate = 2019-03-11}}</ref>。大約兩年後，貝茨離開學校，隱瞞父母湯米（{{lang|en|Tommy}}）和瑪格麗特（{{lang|en|Margaret}}）與史密斯同住在戈頓弗尼沃爾路（{{lang|en|Furnival Road}}）上史密斯的房子<ref name = "marriner" />。貝茨的母親說第一次看到史密斯時，他和貝茨已同居兩個月：「當我看到史密斯時，我感到背上和頸部毛髮直豎，我嘗試一切方法希望凱莉·安離開他<ref name = "byrne2">{{Cite news | first = Paul | last = Byrne | title = Torture monster is jailed for 20 years | work = Daily Mirror | location = London | date = 1997-11-20 | page = 12}}</ref>。」

雖然貝茨因為和史密斯爭執而離開他，她在1995年11月時回到史密斯身邊，並同居在弗尼沃爾路<ref name = "russell">{{cite news |first = Andrew | last = Russell | title = Blinded, scalded, tortured, stabbed and then drowned | work = [[太陽報_(英國)|The Sun]] | location = London | date = 1997-11-12 | page = 17}}</ref>。她的父母當時已注意到貝茨身上的傷痕，但貝茨解釋這些傷痕是意外造成的<ref name = "byrne" />。她漸漸和家人疏遠並在1995年12月時辭去她的兼職工作。1996年3月，她的父母收到一張據稱是貝茨發出的生日和結婚紀念賀卡，寫卡片的人卻是史密斯。貝茨的兄弟嘗試到史密斯的房子探望她時，史密斯都說她不在家。當一個鄰居詢問貝茨的情況，她也只在樓上的窗口露面<ref name = "marriner" />。

== 謀殺 ==
1996年4月16日，史密斯報告當局，他在與貝茨在浴室爭執時失手殺死了她，並聲稱她因溺水死亡，他試圖救她但不果。他還聲稱，貝茨常常假裝暈倒<ref name = "indie1">{{cite news | title = Teen girl 'horrifically tortured before her murder' | work = [[愛爾蘭獨立報|愛爾蘭獨立報]] | location = Dublin | url = http://www.independent.ie/world-news/teen-girl-horrifically-tortured-before-her-murder-26202114.html | date = 1997-11-12 | accessdate =2019-03-11 | language = en}}</ref>。警察去史密斯的家，發現貝茨全身赤裸躺在卧室，屋內滿佈她的血漬，[[驗屍|驗屍結果]]顯示她身上有超過150處獨立的傷痕。去世前一個月中，她持續被束縛著，通常是用她的頭髮繞著脖子，把她綁在[[暖爐|暖爐]]上或家具上<ref name = "byrne" /><ref name = "marriner" />。

相驗她遺體的[[英國內政部|英國內政部]][[病理学|病理學家]]威廉·勞拉（{{lang|en|William Lawler}}）表示：「我在職業生涯中曾檢查了近600名被殺害者的遺體，但我從未看過有受害者所受的傷害是如此遍佈全身<ref name = "ward" />。」以下是貝茨身上發現的傷痕<ref name = "byrne" /><ref name = "marriner" /><ref name = "ward" />：
* 臀部和左腿被[[灼傷|燙傷]]
* 大腿有[[熨斗|熨斗]]燒傷的傷痕
* 一處手臂[[骨折|骨折]]
* 多處由刀、叉子、剪刀造成的傷痕
* 嘴內部被刺傷
* 兩隻手出現粉碎性骨折
* 耳朵、鼻子、眉毛、嘴唇和[[生殖器|生殖器]][[肉刑|被割掉]] 
* 由[[鏟子|鏟子]]和[[園藝剪|園藝剪]]造成的傷口
* 兩眼被挖出
* 掉了眼球的眼窩被刺傷
* 部分頭皮被[[剝皮|剝皮]]

病理學家確認貝茨的眼球在她去世五天至三週內被移除<ref name = "ward" />。她處於飢餓狀態，失去約20公斤的體重，去世前數天沒有喝過水。本案[[檢察官|檢察官]]彼得·奧彭肖（{{lang|en|Peter Openshaw}}）認為：「史密斯好像故意損壞貝茨的外觀，給她帶來極大的痛苦和苦難……這些傷痕不是一次劇烈的暴力造成的，而是經過長時間造成，所以才會這麽遍佈全身和可怕。被告必定是故意和有系統地折磨該名女孩<ref name = "byrne" />。」貝茨死亡的原因是[[遇溺|溺斃]]，在此之前，她被人以[[蓮蓬頭|蓮蓬頭]]擊打頭部<ref name = "ward" />。奧彭肖並表示，她的死是對她仁慈結局<ref name = "russell" />。

== 審判 ==
史密斯拒絕承認[[謀殺|謀殺]]，並聲稱是貝茨讓他陷入地獄般的困境<ref name = "mirror">{{cite news | title = 'Torture girl gave me hell' | work = Daily Mirror | location = London | date = 1997-11-15 | page = 8}}</ref>。他還聲稱，她嘲笑他已過世的母親，而且有「傷害自己的壞習慣，讓我看起來很糟糕」。當史密斯被要求解釋為什麽他要弄瞎、刺傷和毆打貝茨，他說貝茨挑釁說他不敢這樣傷害她<ref name = "ward" />。[[精神科醫師|精神科]]顧問吉莉安·梅澤（{{lang|en|Gillian Mezey}}）告訴法庭，史密斯有嚴重的病態妄想症，並活在一個被扭曲的現實世界<ref name = "bunyan" />。

曼徹斯特[[皇家法院|皇家法院]][[陪審團|陪審團]]花了一個小時裁定49歲的史密斯謀殺貝茨的罪名成立，判處他[[無期徒刑|終身監禁]]，薩克斯（{{lang|en|Sachs}}）法官建議史密斯至少需服20年刑期<ref name = "indie2">{{cite news | title = Torture killer jailed for 20 years | work = Irish Independent | location = Dublin | url = http://www.independent.ie/world-news/torture-killer-jailed-for-20-years-26202465.html | date = 1997-11-20 | accessdate = 2019-03-11 | language = en}}</ref>。他對史密斯說：「這是一宗可怕的案件，將一個人傷害至體無完膚。你是一個高度危險人物，向婦女施虐，因此我決定，在我權力範圍內，讓你不能再虐待他人<ref name = "marriner" />。」

本案中，陪審團成員接受專業諮商，幫助他們處理看過貝茨身上傷害的照片和這宗「令人欲嘔的暴力」案件後所產生的精神傷害<ref name = "marriner" /><ref name = "byrne2" />。
