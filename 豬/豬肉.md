'''豬肉'''是從[[家猪|家猪]]身上取得的[[肉|肉]]。證據可以追溯到公元前5000年的猪[[畜牧业|畜牧业]]，它是全世界最常吃的肉<ref name="fao.org">[http://www.fao.org/ag/againfo/themes/en/meat/backgr_sources.html]. FAO. 25 September 2012.</ref>。猪肉可以有兩種吃法，一種是吃新鲜的熟食、另一種是將豬肉加工保存後食用。固化可以延长猪肉产品的[[保质期|保质期]]。
 
豬肉的相關加工品有[[火腿|火腿]]、[[臘肉|臘肉]]、[[肉乾|肉乾]]、[[培根|培根]]、和[[香肠|香肠]]。猪肉是东亚和东南亚最受欢迎的肉类，在西方世界也是很常见的。由于它的脂肪含量和质地，在[[亚洲|亚洲]]中被高度重视。

== 歷史 ==

17世紀之後，豬肉陸續成為全世界主要肉品以來，選擇豬肉的標準都大約相同：以淺紅、肉質結實、紋路清晰為主。而最高級的肉，是瘦肉與脂肪比例恰好，吃起來不澀不油的肉品，其部位約在[[里脊|里脊]]，大腿和[[排骨|排骨]]。之後，如果白色脂肪越多，豬肉肉品等級就越低。全脂肪的肥豬肉，亦可熬製成[[豬油|豬油]]。

== 部位 ==
*肩胛：[[梅花肉|梅花肉]]（胛心肉）、頸肉（槽頭肉、雪花肉、霜降肉、松阪豬）、頰肉（嘴邊肉、豬頭皮）、骨仔肉（豬頭肉）。
*背脊：大里肌（背肌肉）、小里肌（腰內肉）、大排（排骨、帶骨里肌）、小排（里肌小排、五花小排、腩排）、二層肉（離緣肉）。
*腹脅：[[五花肉|五花肉]]（三層肉）、肝連肉（隔間肉）。
*腿：腿肉（瘦肉、赤肉、精肉）、大腿肉（蹄膀、腿庫、腳庫）、腱子肉（老鼠肉）、腳蹄（[[豬腳|豬腳]]）。

== 消耗量 ==
根據[[美國農業部|美國農業部]]統計，2011年豬肉總消耗量以[[中華人民共和國|中國大陆]]最多。而若以平均每人食用的豬肉，則以[[捷克|捷克]]第一，其次為[[台灣|台灣]]、[[波蘭|波蘭]]。此外，生產豬肉最多的地方亦為[[中華人民共和國|中國]]，佔全世界豬肉肉品46%以上，接下来則是[[美國|美國]]的7%<ref>{{Cite web |url=http://www.askci.com/news/201204/09/091654427610.shtml |title=2012年中国猪肉行业发展概况 |author=中商情报网 }}</ref>。

== 加工 ==
其實除了鮮肉外，豬肉還大量被加工製作各式[[香腸|香腸]]、[[火腿|火腿]]、[[培根|培根]]、[[醃肉|醃肉]]、[[熏肉|熏肉]]、[[肉罐頭|肉罐頭]]、[[寵物食品|寵物食品]]。加工用的豬肉，甚至比鮮肉還多。

家豬不重要部位屠宰產生的[[豬耳朵|豬耳朵]]、[[豬腳|豬腳]]、[[猪鼻子|猪鼻子]]、[[豬頭|豬頭]]、[[豬舌頭|豬舌頭]]、[[內臟|內臟]]所成的豬肉，還可以與其他蔬菜炒成菜餚。此做法於[[东亚|东亚]]、[[美國南部|美國南方]]、[[欧洲|欧洲]]等地都有。[[豬血|豬血]]在[[中国大陸|中国大陸]]被称为“血豆腐”，[[广东|广东]]和[[香港|香港]]等粤语区称此为「[[猪红|猪红]]」，[[台灣|台灣]]有使用[[豬血|豬血]]與[[米|米]]做成的“[[豬血糕|豬血糕]]”。

== 健康风险 ==
2015年10月，[[世界衛生組織|WHO世界衛生組織的IARC(國際癌症研究機構)]]正式把包括豬肉的[[紅肉|紅肉]]列入2A級別[[致癌物質|致癌物質]]<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20151026/719452/ 加工肉品、紅肉列致癌物　專家籲勿恐慌]，蘋果日報， 2015年10月26日</ref>，而經過加工的紅肉更被列為相當於[[香煙|香煙]]的1級致癌物（代表確定會增加癌症機率，但其致癌機率仍低於香菸）。世界衛生組織的報告指出常吃紅肉會誘發[[大腸癌|大腸癌]]，建議減少進食紅肉及注意均衡飲食<ref>[http://news.wenweipo.com/2015/10/30/IN1510300007.htm 世衛回應加工肉製品致癌質疑]，文匯報，2015-10-30</ref>。

== 禁忌 ==
{{see also|符合教規的食物 (猶太教)|符合教規的食物 (伊斯蘭教)}}
[[犹太教|犹太教]]和[[伊斯兰教|伊斯兰教]]中，猪与诸多动物都是不可食用的，大多数现代[[基督教|基督教]]教派无此[[禁忌|禁忌]]。在[[以色列|以色列]]和某些[[穆斯林国家|穆斯林国家]]，特别是那些以[[可蘭經|可蘭經]]法律作为其宪法一部分的国家，销售猪肉是非法的、或是被严重限制的。[[汉传佛教|汉传佛教]]因不杀生，故也不食用包括猪肉在内的所有肉类。印度人在歷史上則沒有養豬消費豬肉的傳統。

回民及其他穆斯林称猪肉为“大肉”。关于此称呼，一说古代[[满洲人|满洲人]]祭祀时，称猪肉为“大肉”，回民也借用满洲人“大肉”之称呼指代猪肉<ref>{{Cite web |url=http://www.sohu.com/a/162271018_115482 |title=满洲人杀猪方法与对猪肉的称呼|author=满族文化网 }}</ref>；另一说法指旧日回民自称“小教”，称汉族为“大教”，“大肉”一词也由此衍生<ref>左宗棠《请禁绝回民新教折》</ref>。
