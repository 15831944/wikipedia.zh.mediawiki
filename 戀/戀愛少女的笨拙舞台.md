{{日本作品|戀愛少女的笨拙舞台|恋する彼女の不器用な舞台}}是[[CUFFS|CUBE]]在2014年11月28日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]<ref>{{cite web|url=http://moedigi.com/game/2014/07/cube-koikano-site-open/|title=CUBE 4th Products、『恋する彼女の不器用な舞台』が本日発表 ―11月28日発売予定|website=moedigi|date=2014-07-18|accessdate=2019-03-19|language=ja}}</ref>。

== 故事簡介 ==
十川真優原本是非常有名的女舞台演員，在進高中時卻宣告隱退，並和男主角宮國一悟在學校成立文藝社享受學園生活。但在高二的校慶時，文藝社被學生會長愛麗絲·莎德·梅西要求達成她所訂定的目標不然就得廢社。而面臨緊要關頭的文藝社，決定和同樣面對艱難狀況的演劇社合作，完成學生會長要求的課題<ref>{{cite web|url=http://moedigi.com/game/2014/10/cube-koikano-master-up/|title=CUBE最新作『恋する彼女の不器用な舞台』がマスターアップ！ ―マスターアップムービーも公開|website=moedigi|date=2014-10-30|accessdate=2019-03-19|language=ja}}</ref>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|宮國一悟|宮国 一悟（みやくに いちご）}}
: 早久保學園二年級生，文藝社的社長，和十川真優是鄰居。
; {{nihongo|十川真優|十川 真優（とがわ まゆう）|聲：{{lang|ja|白野みゆき}}}}
: 早久保學園二年級生，文藝社社員，是在演劇界非常有名的舞台女演員，但升上高中後以進學為由隱退，但其實是想多爭取跟一悟相處的時間。雖然已經隱退，但因為本身名氣的關係吸引不少學校學生好奇的眼光，所以上學時幾乎都在文藝社的社團教室度過。
; {{nihongo|七瀨千奈|七瀬 千奈（ななせ せな）|聲：{{lang|ja|小早川みや}}}}
: 早久保學園一年級生，一悟的學妹，文藝社社員。本身是商業作家，但並未向人提起。由於名字的念法有著像回文卻又稍嫌不足的感覺，而被真優以還差一步的理由取了「娜塔莉」的綽號。
; {{nihongo|愛麗絲·莎德·梅西|アリス・サード・メイシー|聲：[[森谷實園|森谷實園]]}}
: 早久保學園三年級生，從英國來到日本的留學生，學園的學生會長。
; {{nihongo|千代田百花|千代田 百花（ちよだ ももか）|聲：{{lang|ja|森野まりあ}}}}
: 早久保學園一年級生，演劇社社長。年幼時便喜歡演劇，因此入學時便創立了演劇社。是真優的崇拜者，創社時雖不斷邀請真優入社但都遭到拒絕。
=== 其他角色 ===
; {{nihongo|小野坂光|小野坂 ひかる（おのさか ひかる）|聲：七坂真理}}
: 早久保學園二年級生，一悟的男性友人，新聞社成員，負責新聞社的攝影工作。有著女性化的外表。
; {{nihongo|榎本知紗|榎本 知紗（えのもと ちさ）|聲：[[卯衣|卯衣]]}}
: 早久保學園的保健體育老師。
; {{nihongo|九重真里亞|九重 真里亜（ここのえ まりあ）|聲：{{lang|ja|神辺ちせ}}}}
: 早久保學園三年級生，演劇社的副社長。千代田家的女僕，負責照顧百花的生活起居。
