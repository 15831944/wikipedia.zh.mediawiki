== 故事簡介 ==
因為父母的關係轉學到美丘學園的九條蓮，在轉學首日的放學時遇到正因沒帶傘而困擾著的女孩夕凪雫，好心的蓮將傘借給了雫，當蓮將要拼命跑回家時突然被雷打中，並因此得到了只要觸碰女孩子就能看到那名女孩想像色情場面的能力<ref>{{cite web|url=http://moedigi.com/game/2017/03/chelseasoft-koiwamouretsugirl-dl-version-release/|title=あなたとのエッチな妄想が止まらない！Chelseasoft『恋は夢見る妄烈ガール！』ダウンロード版が配信|date=2017-03-17|accessdate=2019-04-08|language=ja}}</ref>。

== 登場角色 ==
; {{nihongo|九條蓮|九条 レン（くじょう れん）}}
: 作品男主角，美丘學園二年級生。性格體貼，擅於為人著想。轉學首日把傘借給女孩夕凪雫之後忽然被雷打中，之後得到了能看見女孩子色情妄想的能力。與雫是兒時玩伴。
; {{nihongo|夕凪雫|夕凪 シズク（ゆうなぎ しずく）|聲：{{lang|ja|綾瀨あかり}}}}
: 蓮的同班同學，對於男性態度嚴峻的傲嬌女孩。
; {{nihongo|美丘葵|美丘 アオイ（みおか あおい）|聲：{{lang|ja|月星ユナ}}}}
: 美丘學園二年級生，言行舉止帶著不可思議氣質的女孩。
; {{nihongo|汐崎雪|汐崎 ユキ（しおさき ゆき）|聲：{{lang|ja|みたかりん}}}}
: 美丘學園二年級生，個性活潑的女孩。擅於體育並加入田徑社，不擅長和男性相處。
; {{nihongo|貓城艾麗卡|猫城 エリカ（こしろ えりか）|聲：野野村紗夜}}
: 美丘學園三年級生，學生會長，家裡是神社。經常說黃色笑話。
; {{nihongo|花之木愛花|花ノ木 アイカ（はなのき あいか）|聲：{{lang|ja|七瀨こより}}}}
: 美丘學園一年級生，蓮父親朋友的女兒，現在和蓮一起生活，稱呼蓮為哥哥，並負責打理家務。
; {{nihongo|千木野千惠|千木野 チエ（はなのき あいか）|聲：{{lang|ja|水瀨ゆき}}}}
: 美丘學園一年級生，愛花的好友，個性開朗的女孩，學生會成員。
