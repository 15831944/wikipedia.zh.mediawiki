《'''戀愛使用說明'''》（{{lang-ja|トリセツ}}）為日本歌手[[西野加奈|西野加奈]]於2015年9月9日發行的第27張單曲。

== 簡介 ==
*與前作《[[如果有命中註定之人的話|如果有命中註定之人的話]]》相隔約4個月的新作，2015年第二張單曲。
*以初回生産限定盤與通常盤兩版本發售，各版封面不同，初回生産限定盤附送收錄封面拍攝花絮的DVD。
*標題曲為電影「[[女主角失格|女主角失格]]」的主題曲<ref name="my">{{Cite web |date=2015-06-08 |url=http://news.mynavi.jp/news/2015/06/08/294/ |script-title=ja:西野カナ、桐谷美玲主演『ヒロイン失格』主題歌に! タイトルは「トリセツ」 |language=ja|publisher=my navi |accessdate=2015-09-21 }}</ref>、在看過電影的劇本後製作的歌曲<ref name="my" />。歌名帶有給女性的使用說明書的意思，是一首以「男生絕對不會知道的事情」為主題、描述少女心的歌曲<ref>{{cite web|url=https://www.gooume-jp.com/works/170/zh_TW|language=zh |title=小清新正夯！少女教主「西野加奈」教你「戀愛守則」！|publisher=gooume-jp|date=2015-11-09 |accessdate=2017-05-11}}</ref>。
*歌曲於全國巡迴演唱會「with LOVE tour」作初次披露<ref>{{Cite web |date=2015-09-21 |url=http://music-book.jp/music/news/news/90167 |script-title=ja:西野カナ“with LOVE”に溢れたツアーファイナルで新曲「トリセツ」初披露 |language=ja| publisher=music.jp |accessdate=2015-09-21 }}</ref>，並於其後的[[第66回NHK紅白歌合戰|第66回NHK紅白歌合戰]]演唱。

== 商業成績 ==
*CD單曲於發行後次日登上[[公信榜|公信榜]]日榜第3位，其後以超過2.5萬張的銷量登上週間第6位<ref name="oricon"/>，自《[[喜歡_(西野加奈單曲)|喜歡]]》相隔一作後再次打進週榜前10位。
*累積銷量超過7.3萬張，成為自2010年的《[[你喲|你喲]]》後自身銷量最高的單曲，並繼2011年的《[[就算...|就算...]]》後再次獲日本唱片協會認證為金唱片。
*歌曲登上[[iTunes|iTunes]]、[[RecoChoku|RecoChoku]]、[[mora|mora]]等各大付費下載網站的週榜冠軍，其後打進2015年・2016年多個榜單的年榜前10位置，並獲日本唱片協會認證單曲下載為三白金唱片。
