== 故事簡介 ==
清櫻學園的理事長皇城凜將要與白羽學園的理事長柊美紀結婚，同時兩校也要在隔年春天合併。凜知道在兩校合併的過程中一定會有學生感到憂慮，因此要弟弟皇城蓮與學生會長月島沙織幫忙解決兩校學生的煩惱<ref>{{Cite web|url=https://www.game-style.jp/soft/201510/23/01kas_en.php|title=ひとつ屋根の下で甘～い同棲生活を送りながら、個性豊かなお嬢様たちと素敵な恋を育んでいこう！ 『恋する気持ちのかさねかた』11月27日発売！|publisher=Game-Style|date=2015-10-23|accessdate=2018-05-11|language=ja}}</ref>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|皇城蓮|皇城 蓮（おうぎ れん）}}
: 作品男主角，清櫻學園二年級生，大企業皇城集團的社長皇城凜的弟弟。
; {{nihongo|月島沙織|月島 沙織（つきしま さおり）|聲：あじ秋刀魚}}
: 清櫻學園二年級女學生與學生會長，大企業月島集團的獨生女。容姿端麗、成績優秀，在校內十分受歡迎。其實有著怕麻煩的一面，四下無人時經常看漫畫及吃零食。
; {{nihongo|柊美櫻|柊 美桜（ひいらぎ みお）|聲：藤森ゆき奈}}
: 白羽學園二年級女學生，理事長柊美紀的妹妹。責任感強烈且個性認真，但並非難相處的人。對自己的學校將要跟別校合併這件事感到困惑。
; {{nihongo|鳴海朱子|鳴海 朱子（なるみ あかね）|聲：有栖川みや美}}
: 白羽學園二年級女學生，沙灘排球隊的隊長，與美櫻是好朋友。性格直爽。
; {{nihongo|久遠寺日和|久遠寺 ひより（くおんじ ひより）|聲：羽鳥いち}}
: 白羽學園一年級女學生，綜合商社久遠寺商事社長的女兒，目前在白羽學園的食堂打工。喜歡研究咖啡，未來的夢想是成為咖啡師。
; {{nihongo|鏑木由希江|鏑木 由希江（かぶらぎ ゆきえ）|聲：秋野花}}
: 白羽學園三年級女學生與學生會長，住在皇城家對面。喜歡對蓮惡作劇。
; {{nihongo|皇城一花|皇城 一花（おうぎ いちか）|聲：白月かなめ}}
: 蓮沒有血緣關係的妹妹，清櫻學園一年級生。個性冷靜、家事萬能，在學校是優等生。

=== 其他角色 ===
; {{nihongo|皇城凜|皇城 凛（おうぎ りん）|聲：深川綠}}
: 蓮的哥哥，皇城集團的社長，也是清櫻學園的理事長。隔年春天將要與白羽學園的理事長柊美紀結婚。
; {{nihongo|柊美紀|柊 美紀（ひいらぎ みさき）|聲：遠見はるか}}
: 美櫻的姐姐，白羽學園的理事長，溺愛妹妹美櫻。
