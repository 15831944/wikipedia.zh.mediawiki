《'''戀我癖'''》（{{Lang-en|'''Ego-Holic'''}}）是臺灣音樂人[[陳星翰|陳星翰]]的一支單曲，該歌曲由臺灣歌手[[蔡依林|蔡依林]]演唱，由[[環球音樂_(台灣)|環球音樂]]於2016年10月31日發行，收錄於陳星翰的專輯《[[Welcome_to_the_Next_Level|Welcome to the Next Level]]》<ref>{{Cite news|url=http://www.cna.com.tw/news/amov/201610300230-1.aspx|title=蔡依林挺製作人 惡搞塗黑自己雙眼嘴唇 {{!}} 娛樂 {{!}} 中央社即時新聞 CNA NEWS|last=中央通訊社|accessdate=2016-10-30}}</ref>；該歌曲於[[HitFM聯播網|Hit FM]]年度百首單曲排行榜位居第52名<ref>{{Cite news|url=http://www.hitoradio.com/newweb/chart_2.php?ch_year=2016&pageNum_rsList=2|title=Hitoradio‧Hit Fm --華人音樂入口指標|newspaper=Hitoradio.com|accessdate=2016-12-31}}</ref>。

== 背景與發行 ==
臺灣媒體於2016年10月30日發佈消息稱臺灣歌手[[蔡依林|蔡依林]]與臺灣新銳音樂人[[陳星翰|陳星翰]]合作的歌曲將於翌日發行。媒體透露該歌曲名為《戀我癖》，並公佈該單曲的封面。蔡依林與陳星翰的合作始於[[第25屆金曲獎|第25屆臺灣金曲獎]]，陳星翰為蔡依林於該頒獎典禮的表演歌曲重新混音，之後陳星翰與蔡依林保持密切合作，包括擔任蔡依林專輯《[[呸|-{zh-cn:Play; zh-hant:呸; zh-sg:呸}-]]》中歌曲《[[Play_我呸|-{zh-cn:呸; zh-hant:Play 我呸; zh-sg:Play 我呸}-]]》與《I'm Not Yours》的音樂製作工作，同時也擔任蔡依林的巡演「[[Play世界巡迴演唱會|Play世界巡迴演唱會]]」的音樂總監。蔡依林在結束巡演之後陸續在[[兩岸三地|兩岸三地]]出席活動與商演，同時也表示將著手開始準備新專輯。此時，陳星翰主動提出合作邀請，而蔡依林亦爽快答應。該單曲的概念為「無論什麼樣的自己，我都喜歡」，而蔡依林選擇以男女都可唱的音調為基準，以細膩的低音貫穿整首歌曲，用輕聲吟唱的方式倡導聽眾「愛自己」。陳星翰一改之前幾乎使用全[[合成器|合成器]]的做法，在該歌曲的編曲上刻意不使用過於強烈的音色，而大量使用真實的樂器例如電吉他，去營造濃烈的空間感。該歌曲由曾經為[[碧玉_(歌手)|碧玉]]、[[凱文·哈里斯|凱文·哈里斯]]等歌手合作過的英國音樂人Mike Marsh進行母帶後期處理。此外，單曲的封面設計師廖俊裕提出「在蔡依林臉上塗鴉」的想法，而蔡依林亦親手塗鴉。蔡依林透過塗鴉封面強調「『戀我』是內在的自我，不單只是外表，每個人都要有信心去喜歡最真實、最獨特的自我」。該歌曲MV由臺灣導演謝宇恩執導，由汪綺出演被霸凌者，並於單曲發行同日發布<ref>{{Cite web|url=http://ent.ltn.com.tw/news/breakingnews/1871464|title=Jolin遭惡搞變厚唇 兇手竟然是她 - 自由娛樂|accessdate=2016-10-30|last=自由娛樂|work=ent.ltn.com.tw}}</ref><ref>{{Cite news|url=https://video.udn.com/news/586144|title=天后蔡依林毀容照曝光 圓眼嘟唇盡失妳哪位？ {{!}} 聯合影音|last=聯合影音|newspaper=聯合影音|accessdate=2016-10-30|language=zh-TW}}</ref><ref>{{Cite news|url=http://www.appledaily.com.tw/realtimenews/article/entertainment/20161030/978596/Jolin%E6%8C%BA%E5%A9%9A%E5%A7%BB%E5%B9%B3%E6%AC%8A%E3%80%80%E4%B8%8D%E4%BB%8B%E6%84%8F%E6%8A%B9%E9%BB%91%E8%87%AA%E5%B7%B1|title=Jolin挺婚姻平權　不介意抹黑自己　 {{!}} 即時新聞 {{!}} 20161030 {{!}} 蘋果日報|newspaper=蘋果日報|accessdate=2016-10-30|language=zh-TW}}</ref><ref>{{Cite web|url=http://www.playmusic.tw/column_info.php?id=7554&type=news|title=PlayMusic音樂網 - 蔡依林再度攜手陳星翰打造《戀我癖EGO-HOLIC》  音樂和態度完美演繹|accessdate=2016-10-30|work=Play Music}}</ref><ref>{{Cite news|url=http://star.ettoday.net/news/802208|title=蔡依林「自畫像」見不得人　遮臉發單曲《戀我癖》 {{!}} 娛樂星光雲 {{!}} ETtoday東森新聞雲|newspaper=娛樂星光雲|accessdate=2016-10-30|language=zh-TW}}</ref>。

== 現場表演 ==
2016年12月3日，臺灣歌手[[蔡依林|蔡依林]]於第10屆[[音乐盛典咪咕汇|音樂盛典咪咕匯]]頒獎典禮登台首度演唱歌曲《戀我癖》<ref>{{Cite news|url=https://star.ettoday.net/news/823159|title=蔡依林率半裸男首唱《戀我癖》！頂紅唇「撩髮辣舞」嗨炸 {{!}} ETtoday星光雲|last=ETtoday新聞雲|newspaper=ETtoday星光雲|accessdate=2018-01-31|language=zh-TW}}</ref>。

== 爭議事件 ==
该歌曲MV表達「反對[[校園霸凌|校園霸凌]]」主題，但MV內容卻引發臺灣民眾爭議。同年11月6日媒體報導該MV包含女高中生聚眾霸凌胖女孩的劇情，但MV中女演員身著[[臺北市立中山女子高級中學|臺北市立中山女子高級中學]]校服，引發該校部分校友不滿，之後有部分網友於歌手[[蔡依林|蔡依林]]、製作人[[陳星翰|陳星翰]]和MV導演謝宇恩的[[Facebook|Facebook]]專頁留言表達不滿情緒，並要求相關人士公開道歉並刪除MV<ref name=":0">{{Cite web|url=http://www.kwongwah.com.my/?p=221185|title=蔡依林拍MV谈霸凌 竟惹火女高中|accessdate=2016-11-06|work=www.kwongwah.com.my}}</ref>。

臺北市立中山女子高級中學於同日發出聲明嚴正譴責媒體負面教育創作，嚴重影響校風，批評MV內容以校園暴力、欺凌為主調，由穿著中山女高的校服演員，拍攝出誹謗、欺凌、惡意影射之畫面，令大眾極度不適，過度的暴力畫面將引發更多負面的效果，絕非「創意」可以包裝一切，要求MV製作團隊立即下架該片，並公開致歉，以正視聽<ref name=":1">{{Cite web|url=http://sina.com.hk/news/article/20161106/0/1/4/%E5%8F%B0%E7%81%A3-%E6%88%80%E6%88%91%E7%99%96-MV%E6%B6%89%E6%A0%A1%E5%9C%92%E6%AC%BA%E5%87%8C-%E9%81%AD%E5%BD%B1%E5%B0%84%E5%A5%B3%E6%A0%A1%E8%A6%81%E6%B1%82%E8%87%B4%E6%AD%89-6507893.html|title=台灣《戀我癖》MV涉校園欺凌　遭影射女校要求致歉|accessdate=2016-11-06|work=香港新浪}}</ref>。校方亦強調學校是教育的場所，平日教導防止欺凌等議題不遺餘力，媒體宣傳應考慮社會責任，不應以任何學校為影射對象，片中對學校的聲譽損害至為明顯，嚴正表達不妥並深感遺憾<ref name=":1" />。此外，該校教師會Facebook社團發表文章表示：「呼籲！中山女高學生、家長、老師、校友，大家一起表達最嚴厲的抗議！」，並要求「中山人」團結對MV按下檢舉及不喜歡，要求MV下架<ref name=":1" />，並把文章張貼到[[蔡依林|蔡依林]]的Facebook專頁，表示嚴厲的譴責與抗議<ref name=":1" />。

MV導演謝宇恩表示：「采用女高中生制服是希望呈现真实感，贴近大家的生活，借此让大家正视霸凌事件，但並未针对任何学校做出诽谤或恶意影射，让许多学生和校友感到不适或恶意，亦深感抱歉，希望有机会到校拜访，以创作者身份厘清。<ref name=":0" />」製作人陳星翰表示：「誠心向[[蔡依林|蔡依林]]本人、她的團隊以及她廣大的粉絲們道歉。想要針對這件事的人，請瞄準好靶心，我就是那個靶。<ref name=":2">{{Cite news|url=http://star.ettoday.net/news/806279|title=《戀我癖》陳星翰向Jolin道歉「校譽不只建立在衣服上」 {{!}} 娛樂星光雲 {{!}} ETtoday東森新聞雲|newspaper=娛樂星光雲|accessdate=2016-11-06|language=zh-TW}}</ref>」陳星翰亦表示：「在我的認知，一間學校的校譽不是只建立在一件衣服上，而是有其他更穩固的東西在支撐。一個團體的榮譽感並不是靠『同樣的服裝』就能獲得的。我敢保證貴校根本不可能因為一支影像作品，去受到任何負面影響。這一關最難過的。是你們自己內心、跟愛自己所愛的方式。<ref name=":3">{{Cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20161106/983070/|title=【爭議片】蔡依林拍霸凌惹火中山女高　MV馬賽克糊成一片 {{!}} 即時新聞 {{!}} 20161106 {{!}} 蘋果日報|newspaper=蘋果日報|accessdate=2016-11-06|language=zh-TW}}</ref>」蔡依林的經紀人王永良表示：「可以理解同學、校友的感受，有特別請陳星翰的經紀公司務必妥善解決這個爭議。」<ref name=":3" />陳星翰的經紀人高漢玲表示：「對蔡依林感到十分抱歉，《戀我癖》的MV蔡依林從企劃到製作完全沒有參與，感謝蔡依林因為信任我們，並無酬在這首歌中獻聲，卻引來各界對她的誤會，真的很抱歉。<ref name=":3" />」

陳星翰之後亦表示：「大家的心聲我聽到了，也尊重每一個人的想法。因此我加強了同學們反映音樂錄影帶中，00:29-01:08馬賽克的處理，希望大家的焦點放回音樂本質跟要傳達的訊息上。<ref name=":3" />」同日晚上9點陳星翰YouTube官方頻道的《戀我癖》MV已全程進行馬賽克處理，並於視頻左上角添加注釋：「敝公司所製作出版之《戀我癖》MV中，因導演安排演員所著之戲服為中山女高制服，而未將畫面妥善處理所引發爭議一事，為此深表遺憾。針對MV中較具爭議性的畫面，都已再次加強馬賽克處理。對於來自各方的批評及指教，我們都抱持著感謝並虛心接受，並再次表達誠摯的歉意。<ref>{{Cite web|url=http://ent.ltn.com.tw/news/breakingnews/1878649|title=（影音）蔡依林MV惹惱蔡英文母校 《戀我癖》全片馬賽克 - 自由娛樂|accessdate=2016-11-06|last=自由娛樂|work=ent.ltn.com.tw}}</ref>」

在MV已經全程馬賽克，且創作者、導演都已經道歉後，中山女高師生和家長一度強烈要求歌手蔡依林公開道歉，部分網友對此一片諷刺，並紛紛表示：「如果是我們學校出現在MV裡，大家一定樂得不得了」、「中山女高有我母校[[臺北市立建國高級中學|建中]]強嗎？《[[牯嶺街少年殺人事件|牯嶺街少年殺人事件]]》醜化建中，我要號召各位校友抵制啦」、「《[[艋舺|艋舺]]》根本是汙衊我堂堂[[萬華區|萬華]]人，[[鈕承澤|鈕承澤]]給我在[[艋舺清水巖|艋舺祖師廟]]前[[洗門風|洗門風]]」、「《[[教父_(電影)|教父]]》是[[美國人|美國人]][[種族歧視|種族歧視]][[義大利人|義大利人]]啦，[[意大利|義大利]]跟[[美國|美國]][[外交|斷交]]啦」、「MV很貼切啊，中山女高連家長都很懂霸凌啊！」、「MV重點根本不在哪一所學校，觀眾也不會在乎是哪一所學校」。部分網友亦指出蔡依林僅僅是為陳星翰的新專輯獻聲，因此不該把問題牽扯到蔡依林身上。其他網友亦認為如不特別提醒根本不會注意到校服屬於哪間學校，而且MV劇情立意良善，希望校方冷靜客觀看待<ref name=":2" />。但支持中山女高師生行為的部分網友則有「不是玻璃心易碎，是你不該踩到我的底限」、「案例明顯有影響商標所屬人事物之名譽，被禁合理啊」等言論。

事件發生之後亦有部分名人於Facebook聲援《戀我癖》MV，[[國立臺灣大學|國立臺灣大學]]法律系教授[[李茂生|李茂生]]認為事件已經演變成「加害人與被害人間關係的轉換與擴散」，並表示「在名校亦有霸凌的MV製作群，甚至於無辜的歌唱者，反而變成被霸凌的人。<ref>[http://m.appledaily.com.tw/realtimenews/article/new/20161108/984286 法律系教授挺Jolin　「霸凌無所不在」]</ref>」設計師[[聶永真|聶永真]]於Facebook暗諷中山女高，認為不論學校好壞都可能存在霸凌事件，「MV劇本裡的設定，或許本來是虛構的，但終究卻證實了」，並認為強制要求蔡依林道歉的言論，就是在霸凌一位公眾人物<ref>[http://ent.ltn.com.tw/m/news/breakingnews/1879145 《戀我癖》遭「馬」掉要蔡依林道歉 網諷像戒嚴「反霸凌」]</ref>。導演[[廖人帥|廖人帥]]指出，「《[[牯嶺街少年殺人事件|牯嶺街少年殺人事件]]》電影裡穿著建中制服，按照『神邏輯』，建中應該要倒閉，但建中至今依然屹立不搖一樣難考！我們的創作環境，付出的再大努力，都會被一小批自以為是的正義魔人所影響，變得更糟糕！！！…大家的每個言論，都會影響創作者的心情（因為我們很在乎我們的作品），創作者的心情，會影響整個環境，這就是[[蝴蝶效應|蝴蝶效應]]…我們明明可以選擇要當『讓事情更好的推手』，為何重點放錯要當『讓環境變糟的殺手』呢？？？<ref>[http://news.ltn.com.tw/news/life/breakingnews/1878388 打臉中山酸民 廖人帥：建中要抗議《牯嶺街》？]</ref>」Facebook專頁「時尚編輯的真心話」亦針對該事件發表言論：「聽說[[臺灣省戒嚴令|戒嚴]]的時候，[[齊秦|齊秦]]只要出現在電視上，他的長髮是會被打馬賽克的。恭喜中山女高，你們好復古。我重整好幾次以為自己瞎了，整支MV海瞭瞭（海瞭瞭，[[臺灣話|臺語]]「害了了」諧音，意為完蛋）。<ref>[http://photo.chinatimes.com/20161107004071-260806 整支海瞭瞭！中山女高遭酸跟MV一樣公開霸凌]</ref>」
