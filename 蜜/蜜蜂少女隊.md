'''蜜蜂少女隊 '''（英語：'''Lady Bees'''）是姊妹淘（上海）文化传播有限公司於2016年推出的女子演唱組合，由張悠然、許和琪、張希雅、劉雨昕、關凱元、孔雪兒、羅姝涵組成<ref>{{cite web |title=蜜蜂少女队登央视元宵晚会 现场配合魔术表演 |url=http://www.chinanews.com/yl/2017/02-14/8149266.shtml |website=中国新闻网 |accessdate=2018-09-05}}</ref><ref>{{cite web |author1=董文 |title=蜜蜂少女队Ladybees再夺音乐榜上榜周榜冠军 |url=http://music.yule.sohu.com/20171114/n522359052.shtml |website=搜狐音乐 |accessdate=2018-09-05}}</ref>。
