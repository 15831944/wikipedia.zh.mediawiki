{{vgname|紙片瑪利歐 超級貼紙|ja=ペーパーマリオ スーパーシール，Paper Mario Super Seal|en=Paper Mario: Sticker Star|diff=yes|cn=纸片马里奥 贴纸之星|tw=紙片瑪利歐 超級貼紙}}是2012年推出的[[电子角色扮演游戏|电子角色扮演游戏]]，对应[[任天堂3DS|任天堂3DS]]平台。游戏由[[Intelligent_Systems|Intelligent Systems]]开发，是[[纸片马里奥系列|纸片马里奥系列]]的第4作，也是系列的首款[[掌上机游戏|掌上机游戏]]。玩家需要操控一張薄薄的紙片瑪利歐，去打擊邪惡的[[庫巴|庫巴]]，以進行遊戲。

游戏在2013年12月6日发售了官方中文版，在2016年11月发售的《[[精灵宝可梦_太阳／月亮|精灵宝可梦 太阳／月亮]]》之前，本作在3年的时间里一直是任天堂在港台区任天堂3DS上发售的最后一款实体版中文游戏，也是最后一款中国大陆版行货神游3DS XL可以兼容运行的中文游戏。

== 玩法 ==
游戏中事物主要由纸片构成，玩家操控主角纸片马里奥，利用纸片的性质过关。玩家要撕下每关中隐藏的贴纸。

游戏戰斗为回合制。玩家要选择合适的贴纸，并抓好時機按键提升攻擊力防守。此外玩家也可以选择逃走。战斗每回合一般只能使用一张贴纸，后期开启轉轉樂可能使用至多三张贴纸。

== 评价 ==
《纸片马里奥 贴纸之星》所获评价褒贬不一，在[[Metacritic|Metacritic]]的汇总得分为75/100<ref name="mc">{{cite web |url=http://www.metacritic.com/game/3ds/paper-mario-sticker-star |title=Paper Mario: Sticker Star Reviews |accessdate=November 14, 2012 |publisher=[[Metacritic|Metacritic]]}}</ref>，是[[纸片马里奥系列|纸片马里奥系列]]历代最低分。

游戏2012年日本销量为40.2万<ref>{{cite web|url=http://www.gamesindustry.biz/articles/2013-01-28-pokemon-tops-2012-software-charts-in-japan|title=Pokemon tops 2012 software chart in Japan|last=Handrahan|first=Matthew|date=28 January 2013|work=[[GamesIndustry|GamesIndustry]]|publisher=Gamer Network|accessdate=21 April 2013}}</ref>，到2013年3月31日时全球总销量197万<ref name="pm13">{{cite web|url=http://www.nintendo.co.jp/ir/en/sales/software/3ds.html|publisher=[[Nintendo|Nintendo]]|title=Top Selling Software Units - Nintendo 3DS Software|date=2013-03-31|accessdate=2013-04-24}}</ref>。
