'''佬族'''，又稱'''寮族'''、'''僚人'''，是[[东南亚|东南亚]]的一个[[泰語民族|泰語民族]]。大部分居住于[[老挝|老挝]]（约400万）和[[泰国|泰国]]（约190万）。泰国的佬族大部分聚居于东北地区，但也有很多出外打工者在其他地区，如[[曼谷|曼谷]]工作。佬族使用[[老挝语|老挝语]]或者是[[伊森语|伊森语]]的各种方言。很多人认为这两种语言实际上是一种语言。但是由于20世纪的[[泰化|泰化]]运动，在泰国东北地区的人更愿意使用“[[伊森人|伊森人]]”这个词。但实际上佬族作为一个整体，仍有很多文化上的共通之处。

== 历史 ==
早在中国西汉古籍中，就有“寮”族，[[僚人|僚人]]的记载，属[[骆越|骆越]]（[[雒越|雒越]]）族一支，分布在从[[广西|广西]]、[[广东|广东]]、[[四川|四川]]，[[雲南|雲南]]、[[貴州|貴州]]、[[海南|海南]]到[[中南半岛|中南半岛]]的广阔地区。他們最早生活在[[奠邊府市|奠邊府市]]一帶，後南下[[湄公河|湄公河]]平原成為老撾主體民族（神话中[[坤布倫|坤布倫]]一子在[[勐天|勐天]]建国）。他们同化与趕走[[老聽族|老聽族]]，后受[[吳哥王朝|吳哥王朝]]統治，至[[澜沧王国|澜沧王国]]建国后独立。

佬族的歷史包含了[[老挝|老挝]]的歷史以及[[依善地区|伊森地區]]的歷史。二者分殊始自[[十九世紀|19世紀]]，1827年，[[永珍|-{zh-hans:万象;zh-hk:萬象;zh-tw:永珍;}-]][[昭阿努|昭阿努]]王起兵反對[[暹羅|暹羅]]（今[[泰国|泰国]]）遭到镇压之後，大批居民從現今老挝地區迁徙至伊森地區，使得老挝本土人煙稀少。两个地区的分裂被1893年及1904年的[[法暹條約|法暹條約]]确认，使得伊森和老挝兩地區之間的分界成為暹羅和[[法屬印度支那|法屬印度支那]]的邊界。

此後，[[泰國|泰國]]及老挝逐渐向分別以[[泰族|泰族]]及佬族為中心的[[民族國家|民族國家]]轉型。在伊森地區，这意味着「[[泰國化|泰國化]]」，即加強人民對泰國的忠誠度。这尤其导致了許多年輕人寧願自認為[[伊森人|伊森人]]而非佬族，因為「伊森」（原意為「東北部」）暗示隸屬於泰國，而「佬族」則表示對老挝效忠。相對地，在老挝，同樣的過程激发了提倡佬族[[語言|語言]]及[[文化|文化]]作為國家語言及文化的行動。同时他们属于大泰族与泰国的小泰人不同，不愿被混为一谈。

== 分布 ==
老撾的佬族[[人口|人口]]是300萬左右，占该國人口的50.3％左右（其餘大部分人是殘餘的山岳民族，[[老听族|老听族]]與[[老松族|老松族]]）。在老挝，种族学意义上的佬族主要指[[老龙族|老龙族]]（低地佬族）。在泰国，佬族占全国人口约三分之一，主要聚居在伊森（约1500万）和曼谷（据估计至少有100万自伊森移来的佬族），在[[中部泰国|中部泰国]]地区，也有种族学意义上的佬族存在，不过他们已经逐渐同化入占主体地位的泰族中了。在[[柬埔寨|柬埔寨]]，也有小规模的佬族社区，主要是在原老挝国的属地[[上丁省|上丁省]]（老挝称为[[川登省|川登省]]）。在[[越南|越南]]和[[中国|中国]]也有小规模的'''佬族'''分布。各国佬族侨民的人数不详，大约估计在50万人左右，主要是[[第二次印度支那战争|第二次印度支那战争]]和[[巴特寮|巴特寮]]战争造成的难民。

值得注意的是，在暹罗（今泰国）最后的一次官方普查中，佬族作为单独的种族，占到了全国人口的一半。作为“泰国化”的一部分，“佬族”现已不作为单独的种族被统计，因此，究竟泰国人口中多少人有佬族血统，今日已经不再清楚。

== 语言 ==
佬族說的語言是[[寮國語|寮國語]]和[[伊森語|伊森語]]。這兩個語言都存在有許多不同的[[方言|方言]]。其中萬象方言已經被接受為寮國語的標準語言；伊森語當中並沒有標準方言，不過大部分的伊森語方言都能夠和萬象方言相互溝通。[[寮國語|寮國語]]和[[伊森語|伊森語]]大部分的差異是在於伊森語中擁有大量借自[[泰語|泰語]]的字詞，或者是因為19世紀後期寮國族和伊森族分隔開來之後，彼此採用了不同的新詞來稱呼外來的新概念（比如說「[[機車|機車]]」在伊森語中是''lot motorcy''，而在寮國語中是''lot jak''）。

== 文化 ==
伊森族與佬族在經濟上都很貧窮，這是因為他們居住的土地相對而言比較貧瘠而且[[氣候|氣候]]乾燥。因此佬族最常見的生活方式就是[[自給農業|自給農業]]（subsistence farming），另外擁有少數幾個都市中心。

佬族和伊森族信仰的是[[上座部佛教|上座部佛教]]。佬族和[[伊森人|伊森人]]的飲食特色為[[魚露|魚露]]、[[辣椒|辣椒]]和[[糯米|糯米]]。不過它們的飲食也吸收了一些[[法國料理|法國料理]]和[[越南料理|越南料理]]的元素。由於佬族所居住的鄉村地區比較貧窮，這讓他們的食物材料和伊森族比起來更為有限。

佬族和伊森族的[[民俗音樂|民俗音樂]]是Mor Lam風格（樂器是[[蘆笙|蘆笙]]，由貴州傳入）。自從1980年代開始，來自於伊森族的Mor Lam音樂已經在文化上對佬族造成了很大的影響。他們與[[泰族|泰族]]不同，喜歡居住在近山的地方。[[皇清職貢圖|皇清職貢圖]]說老撾人體皆刺花。
