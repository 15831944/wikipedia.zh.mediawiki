'''苟仲文'''（{{bd|1957年|6月||catIdx=G苟}}），[[甘肃省|甘肃省]][[镇原县|镇原县]]人。1974年3月参加工作。1976年4月加入[[中国共产党|中国共产党]]。1982年毕业于西北电讯工程学院（现西安电子科技大学）电子工程系电子对抗专业，获工学学士学位；1989年获西安电子科技大学工学硕士学位<ref>{{cite web|accessdate=2017-06-24|url=http://xyh.xidian.edu.cn/info/1064/3630.htm|publisher=西安电子科技大学校友会|title=西电77级校友苟仲文任国家体育总局局长|deadurl=yes|archiveurl=https://web.archive.org/web/20170628020932/http://xyh.xidian.edu.cn/info/1064/3630.htm|archivedate=2017年6月28日|df=}}</ref>，在职研究生，管理学博士。现任[[中华人民共和国|中华人民共和国]][[国家体育总局|国家体育总局]]局长、[[中国奥林匹克委员会|中国奥林匹克委员会]]主席、[[中华全国体育总会|中华全国体育总会]]主席。[[中国共产党第十九届中央委员会委员列表|中共第十九届中央委员]]。

== 生平 ==

曾任[[中华人民共和国电子工业部|电子工业部]]副司长，计算机与微电子发展研究中心主任；中国电子信息产业发展研究院院长，[[中华人民共和国信息产业部|信息产业部]]副部长，[[中华人民共和国工业和信息化部|工业和信息化部]]副部长等职。

*2008年4月，任[[北京市人民政府|北京市人民政府]]副市长<ref>{{cite web|accessdate=2013-05-01|url=http://www.stnn.cc/china/200804/t20080419_765796.html|title=苟仲文任北京副市长 北京市府人事全部到位|publisher=[[星岛网|星岛网]]}}</ref>。

*2013年7月，晋升[[中共北京市委|中共北京市委]]常委<ref>{{cite web|url=http://china.caixin.com/2013-07-04/100551642.html|publisher=财新网|accessdate=2013-07-04|title=苟仲文任北京市委常委 陈刚另有任用}}</ref>，同年8月兼任市委教工委书记、中关村管委会党组书记<ref>{{cite web|url=http://renshi.people.com.cn/n/2013/0812/c139617-22525234.html|title=苟仲文任北京中关村管委会党组书记|publisher=中国共产党新闻网|accessdate=2013-08-12}}</ref>。

*2016年5月，任[[中共北京市委|中共北京市委]]专职副书记，接替落马的[[吕锡文|吕锡文]]<ref>{{cite web|url=http://news.qq.com/a/20160505/031277.htm|title=苟仲文任北京市委副书记|publisher=腾讯|accessdate=2016-05-05}}</ref>。

*2016年10月，调任[[国家体育总局|国家体育总局]]局长<ref>{{cite web|url=http://district.ce.cn/newarea/sddy/201611/01/t20161101_17391630.shtml|title=北京市委副书记苟仲文任国家体育总局局长 刘鹏不再担任|publisher=中国经济网|accessdate=2016-10-31}}</ref>、[[中国奥林匹克委员会|中国奥委会]]主席、[[中华全国体育总会|中华全国体育总会]]主席、[[2022年冬季奥林匹克运动会|北京2022年冬奥会]]和[[2022年冬季残疾人奥林匹克运动会|冬残奥会]]组织委员会执行主席<ref>{{cite web|url=http://www.thepaper.cn/newsDetail_forward_1565017|title=国务院任免国家工作人员：苟仲文任国家体育总局局长|publisher=澎湃新闻|accessdate=2016-11-20}}</ref>。

*2017年7月8日，苟仲文当选[[中共十九大|中共十九大]]代表<ref name="a1">{{Cite web |url=http://china.dwnews.com/news/2017-07-08/59824335.html |title=中共国家机关十九大代表出炉 蔡振华落选|accessdate=2017-07-23 |author=编辑：南希 |date=2017-07-08 |work= |publisher=多维新闻网|language=Zh-hans }}</ref>。

*2017年10月24日，苟仲文当选中共第十九届中央委员。

== 争议 ==
2015年10月19日，亲自以北京市教育工委书记身份，去[[北京大学|北京大学]]上思想政治理论课<ref>{{Cite web|url=http://zhengwu.beijing.gov.cn/gzdt/bmdt/t1407285.htm|title=市领导为大学生上思想政治理论课-部门动态-首都之窗-北京市政务门户网站|accessdate=2017-06-24|last=贾宁|work=zhengwu.beijing.gov.cn|archiveurl=https://web.archive.org/web/20170626111734/http://zhengwu.beijing.gov.cn/gzdt/bmdt/t1407285.htm|archivedate=2017-06-26|deadurl=yes}}</ref>。在执掌北京市教育系统之后，力主推进取消“择校生”，推动小学、初中“就近入学”。其中取消“共建”，被认为是力度最大的一条<ref>{{Cite web|url=http://news.sina.com.cn/c/zg/2016-05-05/doc-ifxryhhh1630594.shtml|title=苟仲文任北京市委副书记，有哪两点不寻常？|accessdate=2019-02-17|last=新浪|work=news.sina.com.cn}}</ref>。过去，国家部委、央企等强势部门都会通过单位赞助钱或物的方式，和一两所优质中小学签订“共建”协议，以此保障职工子女优先入学。这一做法因有违教育公平广被诟病。但此举也导致对苟仲文的不满声音开始出现，屡有言论指其“外行领导内行”。如在北京推行的中招新政，就有不少家长不满，称违反公平正义原则，“毁了这一届孩子”。

2017年，时任国家体育总局局长的苟仲文，干预主导了[[中国足球协会超级联赛|中国足球协会超级联赛]]U23球员出场和转会新政策<ref>{{Cite news|url=http://www.bbc.com/zhongwen/simp/chinese-news-40046995|title=中国足球：中超将迎来最“激进”制度改革|date=2017-05-25|newspaper=BBC 中文网|accessdate=2017-06-24|language=en-GB}}</ref>。同年又引发了备受争议的[[中国国家乒乓球队|中国国家乒乓球队]]总教练[[刘国梁事件|劉國梁下课、国家乒乓队队员和教练罢赛事件]]<ref>{{Cite web|url=http://sports.163.com/17/0620/22/CNDHIIKJ00058782.html|title=新华社:刘国梁转职如三级跳 为国乒主动求变点赞_网易体育|accessdate=2017-06-24|last=网易|work=sports.163.com}}</ref><ref>{{Cite web|url=http://sports.sina.com.cn/others/pingpang/2017-06-21/doc-ifyhfnqa4537572.shtml|title=刘国梁离任引发三大问号 他究竟是升官还是降职了|accessdate=2017-06-24|work=sports.sina.com.cn}}</ref><ref>{{Cite web|url=http://sports.ifeng.com/a/20170623/51310430_0.shtml|title=体育总局：国乒擅自弃赛极其错误 将严肃处理|accessdate=2017-06-24|work=sports.ifeng.com}}</ref><ref>{{Cite web|url=http://sports.cnhubei.com/2017/0623/362656.shtml|title=人民日报转发国乒应援刘国梁微博 网友:有态度_体育频道_湖北日报网|accessdate=2017-06-24|last=体育中心2|work=sports.cnhubei.com}}</ref>，有网友戏称其为“狗局长”，批评苟仲文“不懂体育、瞎指挥”，甚至有呼声要求苟应引咎辞职<ref>{{Cite web |url=http://www.thepaper.cn/newsDetail_forward_1750478|title=中国新闻周刊：在争议中重启的中国体育改革 |accessdate=2017-08-02 |author= |date=2017-08-02|work= |publisher=微信公众号“中国新闻周刊杂志”|language=Zh-hans }}</ref>。据[[法国国际广播电台|法国国际广播电台]]中文网报道，有传言指国家体育总局局长苟仲文和副局长[[蔡振华|蔡振华]]“不和”，体育总局调换刘国梁、整改国乒教练员队伍，最终目标是现任中国足协主席、体育总局唯一的[[中央候补委员|中央候补委员]]蔡振华，因为蔡振华曾任中国男乒总教练，而刘国梁和[[孔令辉|孔令辉]]是他的“左膀右臂”，孔令辉之前刚受到[[新加坡|新加坡]][[赌场|赌场]]追债事件的负面牵连。法广中文网分析称，[[中共十九大|中共十九大]]给国家体育总局只有一个中央委员名额，苟仲文的主要竞争对手就是蔡振华<ref >{{cite web|url=http://china.dwnews.com/news/2017-06-25/59821839.html | language= | title=中国乒坛乒变 或影响蔡振华入十九大 |publisher=多维新闻 |date=2017-06-25 |accessdate= }}</ref>。7月8日，公布国家机关的[[中共十九大|中共十九大]]代表名单，苟仲文当选，蔡振华未进入名单<ref name="a1"/>。

《足球报》6月26日报道称，前跆拳道奥运冠军陈中的教练[[陈立人|陈立人]]，或将转入到[[中国足协|中国足协]]，具体职位是女足领队或是教练尚未明确。得知这一消息，[[黄健翔|黄健翔]]发表评论称：“中国足协的掌门人位置上曾经坐过主管水上项目的和打乒乓球出身的……现在来个跆拳道教练管女足，八成是更高的领导听说[[兹拉坦·伊布拉希莫维奇|伊布]]练过跆拳道吧。”

== 家庭 ==
父亲名为苟崇廉，曾在甘肃[[银行|银行]]系统任职，1986年任[[中国农业银行|中国农业银行]]甘肃分行金融研究所所长，1992年退休。

苟仲文有一个兄弟叫苟仲武，在甘肃成立过电子信息相关企业。2009年，苟仲武在北京开始经营多家企业，主要是餐饮业。苟仲武曾67次担任企业法人，4次作为股东，48次作为企业高管，旗下拥有2家注册资金超一千万的企业：北京牛魔王餐饮连锁有限公司和兰州珍满楼餐饮连锁有限公司。<ref >{{cite web|url=http://china.dwnews.com/news/2017-06-25/59821906.html | language= | title=乒坛风暴牵出体育局长亲属财富疑云 |publisher=多维新闻 |date=2017-06-25 |accessdate= }}</ref>
