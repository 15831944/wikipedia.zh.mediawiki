'''苟芸慧'''（英文名：{{lang|en|'''Christine Gou'''}}，{{bd|1983年|7月11日|catIdx=G苟}}）<ref>on.cc - 苟芸慧7月11日牛一易記 - 2009年7月2日</ref>，生於[[臺灣|臺灣]][[臺中市|台中市]]，[[華裔加拿大人|華裔加拿大人]]，2008年[[多倫多華裔小姐|多倫多華裔小姐]]和2009年[[國際中華小姐|國際中華小姐]]冠軍，前[[無綫電視|無綫電視]]（TVB）經理人合約藝員兼力捧花旦之一。

== 簡歷 ==
苟芸慧在[[臺灣|臺灣]][[臺中市|台中市]]長大，幼時於台中市立北屯國小就讀，十歲後隨家人舉家移民[[加拿大|加拿大]][[多倫多|多倫多]]，於[[滑鐵盧大學|滑鐵盧大學]]主修數學<ref name="plem">{{Cite web |url=http://wp.plem.com/?p=18 |title=美麗凝聚 真我閃耀 陳娜良子 苟芸慧 -《PLEM 娛樂生活雜誌》 |access-date=2009-06-13 |archive-url=https://web.archive.org/web/20090412001812/http://wp.plem.com/?p=18 |archive-date=2009-04-12 |dead-url=yes }}</ref>，因爸爸生意失敗，又患了癌症，改讀[[卡尔加里大学|卡尔加里大学]]遙距課程，在修讀期間在一間由[[意大利人|意大利人]]經營的公司出任專案經理。<ref name="plem"/>

2008年，苟芸慧參選[[多倫多華裔小姐競選|多倫多華裔小姐競選]]贏得冠軍，於2009年參選[[國際中華小姐競選|國際中華小姐競選]]贏得冠軍，其後加入[[娛樂圈|娛樂圈]]。苟芸慧表示其志向是成為成功女性，和幫助有需要的人，自言參選國際中華小姐全因希望完成母親的願望。<ref>{{cite news|url=http://the-sun.on.cc/cnt/entertainment/20090909/00470_036.html|title=苟芸慧賣甜拒性感|date=2009-09-09|newspaper=太陽報(香港)}}</ref>

近年，苟芸慧因[[甲狀腺|甲狀腺]]出現問題而減產，工作量大降；2015年年底，苟芸慧原本決定約滿TVB後離開，後來經無綫製作資源部總監[[樂易玲|樂易玲]]多番挽留，並訂於11月重新工作。

2016年10月，苟芸慧正式與TVB約滿。曾為無綫電視監製[[林志華|林志華]]的常用演員。

2017年，苟芸慧到米蘭擔任客席編輯作人物專訪、街訪等<ref>[https://www.mings-fashion.com/max-mara-%E8%8B%9F%E8%8A%B8%E6%85%A7-%E6%99%82%E8%A3%9D%E5%91%A8-188051/ 苟芸慧 X MAXMARA：客席米蘭時編大挑戰].MING'S.20 Oct 2017</ref>。

==軼事==
基於苟芸慧[[粵語|粵語]]說得不標準，故曾在《[[情人眼裏高一D|情人眼裏高一D]]》、《[[紫禁驚雷|紫禁驚雷]]》及《[[情逆三世緣|情逆三世緣]]》動用[[配音|配音]]。在《[[怒火街頭II|怒火街頭II]]》和同劇其餘有份演出但粵語不標準的演員[[金剛_(藝人)|金剛]]、[[賈曉晨|賈曉晨]]及[[岑麗香|岑麗香]]被網民戲稱為「鄉音四人組」<ref>{{cite news|title=《怒火街頭2》驚現鄉音四人組 苟芸慧被批唔知噏乜|url=https://hk.entertainment.appledaily.com/entertainment/daily/article/20120813/16597874|accessdate=2018-02-20|work=[[蘋果日報_(香港)|香港蘋果日報]]|date=2012-08-13}}</ref><ref>{{cite news|title=《怒火街頭2》“鄉音四人組”惹網友吐槽|url=http://dailynews.sina.com/bg/ent/tv/sinacn/20120813/15173668255.html|accessdate=2018-02-20|work=[[南方都市報|南方都市報]]（[[新浪|新浪網]]轉載）|date=2012-08-13}}</ref><ref>{{cite news|title=朱璇鄉音No.1|url=http://the-sun.on.cc/cnt/entertainment/20121110/00470_070.html|accessdate=2018-02-20|work=[[太陽報_(香港)|香港太陽報]]|date=2012-11-10|quote=...提到早前在劇集《怒火街頭II》中，金剛、賈曉晨、苟芸慧及岑麗香被嘲「鄉音四人組」...}}</ref>。

苟芸慧與[[鍾嘉欣|鍾嘉欣]]、[[李亞男|李亞男]]、[[王君馨|王君馨]]、[[岑麗香|岑麗香]]因同為[[基督|基督]][[新教徒|新教徒]]，數人自發組成祈禱小組「Amazing Esthers」，亦常有聚會，因此常為傳媒牽連在一起並稱為「TVB五大處女」<ref>{{Cite web |url=http://hk.apple.nextmedia.com/realtime/videonews/20170115/56176929 |title=【唔講陸漢洋】「處女黨」真名叫Amazing Esther 苟芸慧晒事業線為保好姊妹？  |access-date=2017-09-06 |archive-url=https://web.archive.org/web/20170906135314/http://hk.apple.nextmedia.com/realtime/videonews/20170115/56176929 |archive-date=2017-09-06 |dead-url=yes }}</ref>，但現在她們全部已婚。
