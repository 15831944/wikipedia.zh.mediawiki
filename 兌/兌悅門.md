'''兌悅門'''是昔日[[臺灣府城|臺灣府城]]從小西門延伸到小北門之外城的三座城門之中唯一僅存者<ref>另外兩座西北外城門是奠坤門（西南方）與拱乾門（西北方）。</ref>，因位於正西方而以八卦中的「兌」為名，城門橫額提有兌悅門，兌者八卦方位之西方，已應此門為往西。兌悅門今為國定古蹟，位於[[臺南市|臺南市]]中西區，昔日是[[五條港_(臺南)|五條港]]的新港墘港所在的地方。這座城門也是現今臺南市所有城門中唯一還具有通行功能的城門。臺南市另外現存三座城門都已不位於道路之上，[[小西門|小西門]]的[[奠坤門|奠坤門]]/[[小北門|小北門]][[拱乾門|拱乾門]]，三個城門共築一道堅實的外城，由小西門外繞至小北門主要抵抗海盜的騷擾。兌悅門又稱為咾咕石城或甕城。

== 沿革 ==
該門是於清[[道光|道光]]十五年（1835年）時增建的城門，為大西門的外城，主要是因為道光十二年（1832年）發生了[[張丙之役|張丙之役]]後，為安定民心，而大幅在臺灣各城添加半月城之結果。於隔年完工。以磚石構建，城基四角建以花崗岩。成高四公尺，厚三公尺，門洞寬三公尺。基座以珊瑚礁又稱為老古石，所以又名老古石城。

日治時期城牆因為闢建道路而拆除原本相連的城牆，不過城門卻保留了下來。戰後周圍被民房所包圍，且一度因年久失修以致城門上被[[鳥榕|鳥榕]]所覆蓋，不過現已清除整理。

== 建築特色 ==
[[File:臺南兌悅門_(背面).JPG|thumb]]
兌悅門的臺座應是用[[咾咕石|咾咕石]]為材料，上面再砌上紅磚、用白灰粉刷，內部則用夯土而成，而在牆角處則有用[[花崗岩|花崗岩]]來補強。城門的西立面比東立面略高，而且在西立面這邊還設置有[[垛牆|垛牆]][[鎗口|鎗口]]（東立面無鎗口）。原本城門有設置門板，但已不知何時遺失。

== 相關傳說 ==
據說當年兌悅門完成之後，因為其門洞看起來像是一把[[弓|弓]]，而底下鋪著石板的巷道像是一支箭，兩者合一便是一組蓄勢待發的弓箭，帶有[[煞氣|煞氣]]。而為了化解這股煞氣，便在城門旁供奉一座石獅爺來坐鎮（今尚存於門附近）。不過另有一說這所謂的「箭煞陣」其實是由兌悅門的彎曲城垣與巷道所作的組合，是五條港的郊商為了對付在安平的洋行而設下要破壞對方風水的陣式（弓箭對準安平射出）。然而由於在安平有漢人買辦知道了這事，便安置了兩尊「石將軍」<ref>這兩尊石將軍其中之一據說是以前荷蘭人的教堂裡遺留下來的古物，另一尊則推測是延平郡王[[鄭成功|鄭成功]]墓前所安置的石翁仲。</ref>來鎮煞。這兩尊石將軍原先安置在安平天后宮舊址（今[[安平區|安平]]石門國小(安平公學校)附近），現則安置在[[安平開臺天后宮|安平開臺天后宮]]左側的將軍殿裡供奉。
