'''卫立煌'''{{BD|1897年|2月16日|1960年|1月17日|catIdx=Wei,卫}}，字'''俊如'''，又字'''辉珊'''，[[中華民國|中華民國]][[國民革命軍|國民革命軍]]將領，[[安徽|安徽]][[合肥|合肥]]（今[[肥东|肥东]]）人。

== 早年經歷 ==
衛立煌的家庭出生是負責[[田賦|田賦]]登記管理的地方官吏，父亡於1905年，家計靠著其兄長衛立炯在縣城的職務餬口，幼年接受[[私塾|私塾]]教育，有基礎識字知識；[[辛亥革命|辛亥革命]]後的1912年，衛立煌與其兄長一同至和州（今[[安徽|安徽]][[和县|和县]]）加入軍隊，衛立炯任安徽[[和縣|和縣]]軍事訓練班主任，和縣在[[二次革命|二次革命]]時參與革命軍，後兵敗，逃離部隊返鄉；1914年衛立煌在考入湖北陸軍學兵營，再度入伍，在1年的訓練後逃離部隊至[[上海|上海]]，後參加[[肇和號巡洋艦|肇和號巡洋艦]]起事失敗，南下[[廣州|廣州]]加入由[[廣東省|廣東省]]省長[[朱慶瀾|朱慶瀾]]成立的廣東省軍，為粵軍第二軍班長（軍長[[許崇智|許崇智]]）。

由於朱慶瀾立場偏向孫中山，因此他成立的部隊也與孫中山關係密切；1917年9月，衛立煌被編入[[孙中山|孙中山]]的卫队充当卫士、後升任警衛團[[排长|排长]]，1918年調回粵軍第二軍第二支隊。衛立煌的軍事資歷初期均在粵軍第二軍締造，包括東征、廣州商團事件與對抗陳炯明等初期作戰，不久後已升任補充旅旅長。1925年孫中山病逝後，因為許崇智被[[蔣中正|蔣中正]]排擠離開廣州，粵軍第二軍也被收編整併，衛立煌任職的補充旅被縮編為[[國民革命軍第一軍|國民革命軍第一軍]]第三師第9團，衛任少將團長。1926年[[國民革命軍北伐|國民革命軍北伐]]時，第一軍由[[何應欽|何應欽]]率領對抗[[周蔭人|周蔭人]]在福建的部隊，由於第一軍屢戰皆勝，[[松口戰鬥|松口戰鬥]]後衛升任第十四師師長；第十四師在1927年[[龍潭戰鬥|龍潭戰鬥]]對抗[[孫傳芳|孫傳芳]]有功，因此衛立煌在1927年10月升任[[國民革命軍第九軍|國民革命軍第九軍]]副軍長兼南京戍衛副司令，1928年進[[陸軍大學|陸軍大學]]將官特別班第一期深造，補足因由行伍出生而缺少的學歷。

== 中年經歷 ==
1927年，[[南京国民政府|南京国民政府]]成立后，任[[南京|南京]]卫戍副司令。1929年，升任國民革命軍第45師少將[[师长|师长]]，45師當時是新成立部隊，在1930年[[中原大戰|中原大戰]]時擔任[[京滬鐵路|京滬鐵路]]至南京的防禦任務。1932年5月，衛立煌兼任中路軍第6縱隊指揮官，開始受命进攻鄂豫皖[[苏区|苏区]]，1932年9月[[豫鄂皖邊區第三次圍剿|豫鄂皖邊區第三次圍剿]]作戰期間攻占鄂豫皖蘇區軍政中心[[金寨|金寨]]，[[蒋中正|蒋中正]]为彰其功，改[[金寨县|金寨县]]名[[立煌县|立煌县]]；1932年6月，升任[[國民革命軍第十四軍|國民革命軍第十四軍]]中將軍長，下轄第10師（師長[[李默庵|李默庵]]）、第83師（師長[[劉戡|劉戡]]），在[[豫鄂皖邊區第四次圍剿|豫鄂皖邊區第四次圍剿]]擔任中路第六縱隊指揮官，該役紅四軍團戰敗退出鄂豫皖蘇區。

1933年[[第五次圍剿戰爭|第五次圍剿戰爭]]時期，衛立煌擔任北路軍第2路軍第1縱隊指揮官，[[閩變|閩變]]時作為鎮壓[[國民革命軍第十九路軍|國民革命軍第十九路軍]]的主力。1935年11月召開的中國國民黨第5次全國代表大會時，衛立煌被选为[[中國国民党|中國国民党]][[中国国民党中央执行委员会|中央执行委员]]。1936年9月升任陸軍上將。[[西安事變|西安事變]]期間，衛立煌兼任位於[[潼關|潼關]]的第五路軍總指揮，因此與蔣中正一同被囚，後一同釋放。

1937年[[中国抗日战争|中国抗日战争]]爆發初期，衛立煌升任[[國民革命軍集團軍序列|第14集團軍]]總司令，在[[華北|華北]]與日軍作戰，1937年10月升任[[抗日战争第二战区|第二战区]]敵前總指揮，1938年1月升任第二戰區副司令長官兼敵前總指揮，1939年1月升任[[抗日战争第一战区|第一战区]]司令长官，1939年9月兼[[河南省政府|河南省政府]]主席與全省保安司令。由於在第一戰區各種勢力犬牙交錯，衛立煌與[[中國共產黨|中國共產黨]]軍事將領如[[朱德|朱德]]等人在當地有著私交，使重慶國民政府猜疑衛立煌的政治忠誠，在1942年1月衛立煌被調為軍事委員會西安辦公廳主任，河南省政務與軍務轉由[[湯恩伯|湯恩伯]]主控。

1942年3月，衛立煌轉任中國遠征軍第一路軍司令官，由於與共產黨的關係過密，該職務後由[[羅卓英|羅卓英]]擔任。由於衛立煌在當時上將當中屬於戰功不少，且整體派系色彩較不明顯的將領，在各方立場互有僵持時他成為一個較沒有爭議的人選。1943年10月，衛立煌接替罹病的[[陳誠|陳誠]]擔任[[中國远征军|中國远征军]]司令长官，住紮在[[雲南省|雲南省]][[保山縣|保山縣]]，協調雲南省主席[[龍雲|龍雲]]與中緬印戰區總參謀長史迪威等人的相關問題；1944年5月，中國遠征軍發動[[松山戰役|松山戰役]]打通中國對外生命線。

1945年3月2日，中國陸軍總司令部奉令裁撤遠征軍司令長官部、第四戰區司令長官部、黔桂湘邊區總司令部、滇越邊區總司令部，與第一、第九、第十一、第十六、第二十、第二十四、第三十五等集團軍番號及第一集團軍副總司令部，任命衛立煌為中國陸軍副總司令，[[盧漢|盧漢]]、[[張發奎|張發奎]]、[[湯恩伯|湯恩伯]]、[[王耀武|王耀武]]為第一、第二、第三、第四方面軍司令官；蔣介石令調[[俞濟時|俞濟時]]為第三十六集團軍總司令<ref name=民/>{{rp|7673}}。3月芒友會師打通[[中印公路|中印公路]]，名扬中外。

== 晚年經歷 ==
衛立煌對國共問題的立場較偏向以政治談判解決問題，與蔣中正等人的主戰色彩極為格格不入；因此在1946年衛立煌曾一度赴國外旅遊，1947年回國時戰事已轉壞。

1948年1月，[[蔣介石|蔣介石]]任命衛立煌接替陳誠擔任[[中国东北地区|东北]][[剿匪總司令部|剿匪總司令部]]總司令。<ref name="年表">[[陳布雷|陳-{布}-雷]]等編著：《蔣介石先生年表》，台北：傳記文學出版社，1978年6月1日</ref>{{rp|53}}衛立煌未能執行中華民國军事决策部门作出的撤出[[瀋陽|瀋陽]]，集中兵力於[[錦州|錦州]]，以待国际形势变化的策略，导致锦州被[[林彪|林彪]]统领的[[东北野战军|东北野战军]]急进抢占，从此扼住东北咽喉，[[中國共產黨|中國共產黨]]从此掌握战事主动权。因此，蒋中正又派遣徐州剿总副总司令兼第二兵团司令官[[杜聿明|杜聿明]]为东北剿总副总司令兼冀热辽边区司令官，事实上剥夺了卫的战场指挥权，而令其仅能统领在沈阳的國軍。9月24日，衛立煌決定空運第四十九軍到錦州增援<ref name="民">{{cite book |editor=李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編 |title=《中華民國史大事記》 |location=[[北京|北京]] |publisher=[[中華書局|中華書局]] |year=2011 }}</ref>{{rp|8681}}。但是杜聿明的谋划以及蒋中正亲率海陆空军在[[葫芦岛|葫芦岛]]的救援也未能扭转东北战场的颓势，最终[[廖耀湘|廖耀湘]]部在徘徊于[[沈阳|沈阳]]与[[营口|营口]]之间后被全歼，随后沈阳也告陷落，自此国军在东北战场几乎全军覆没，卫立煌仅以个人携家小近从于沈阳城陷之际乘[[飞机|飞机]]逃离，途經[[北平|北平]]会见[[傅作义|傅作义]]后至[[广州|广州]]。11月，蔣以[[东北|东北]][[剿匪總司令部|剿匪總司令部]]總司令衛立煌貽誤戎機，下令撤職查辦，以為奉行軍令不力者戒。<ref name="年表"/>{{rp|56－57}}衛立煌因沒有積極執行蔣介石的作戰命令，被软禁于南京。<ref name="文稿">《建國以來毛澤東文稿》第五冊，北京：中央文獻出版社，1991年</ref>{{rp|49}}
1949年1月，蔣引退後衛立煌去[[香港|香港]]。<ref name="文稿"/>{{rp|49}}后拒绝去[[台湾|台湾]]。在港期间遭遇[[投资|投资]]失败。1955年3月15日，经中共劝说，从香港经广州回[[北京|北京]]。發表《告台灣袍澤朋友書》，全文載1955年3月17日《[[人民日報|人民日報]]》和香港《[[文滙報|文滙報]]》。<ref name="文稿"/>{{rp|49}}曾任[[国防委员会|国防委员会]]副主席，[[全国政协|全国政协]]第二、三届常务委员，第二届[[全国人大代表|全国人大代表]]，[[民革|民革]]中央常委等职。[[周恩来|周恩来]]称之為“[[起义|起义将领]]”。

== 逝世 ==
1960年1月17日0时40分，因罹患[[冠心病|冠心病]]和[[肺炎|肺炎]]在[[北京|北京]]逝世。墓位於北京[[八寶山革命公墓|八寶山革命公墓]]內。
