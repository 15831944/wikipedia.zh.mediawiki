{{日本作品|借宿下的戀愛|かりぐらし恋愛}}是[[HOOKSOFT|ASa Project]]在2018年3月30日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]<ref>{{cite web|url=https://www.game-style.jp/soft/201802/20/01kar_as.php?permission=true|script-title=ja:「Game-Style」特集|website=Game-Style|date=2018-02-20|accessdate=2018-03-31|language=ja}}</ref>。2019年2月21日由Entergram發售[[PlayStation_4|PlayStation 4]]與[[PlayStation_Vita|PlayStation Vita]]版本<ref>{{cite web|url=https://www.4gamer.net/games/437/G043725/20181101171/|title=PS4/PS Vita版「かりぐらし恋愛」の発売日が2019年2月21日に決定。公式サイトで限定版と店舗別特典の情報が公開|website=4Gamer.net|date=2018-11-01|accessdate=2019-01-24|language=ja}}</ref>。

== 故事簡介 ==
瀨戶田拓真為了展開嚮往已久的獨居生活而搬回小時候居住過的城市，卻沒想到自家已變成一座廢屋，而此時四位青梅竹馬對拓真伸出了援手，拓真將要輪流在四位青梅竹馬的家裡開始借宿生活<ref>{{Cite web|url=http://www.tgsmart.jp/article.aspx?a=21053|script-title=ja:【発売直前コメント付き！】ASa Projectの10周年記念タイトル『かりぐらし恋愛』がついにリリース！|publisher=TG Smart|date=2018-03-27|accessdate=2018-03-31|language=ja}}</ref>。

== 登場角色 ==
* 聲優為PC版／PS4、PSV版<ref>{{Cite web|url=http://www.entergram.co.jp/karigurashi/#character|title=登場角色|publisher=PS4/PSVita遊戲「借宿下的戀愛」官方網站|accessdate=2019-01-24|language=ja}}</ref>
=== 主要角色 ===
; {{nihongo|瀨戶田拓真|瀬戸田 拓真（せとだ たくま）}}
: 二年級生，作品男主角，為展開獨居生活而搬回小時候居住的城市。
; {{nihongo|櫻木橋理兔|桜木橋 理兎（さくらぎばし りと）|聲：步サラ／岡本理繪}}
: 二年級生，拓真的青梅竹馬和同班同學，以前被拓真誤認為男生。擅長家事，目前一個人居住。
; {{nihongo|荒波杏|荒波 杏（あらなみ きょう）|聲：花澤さくら／同左}}
: 三年級生，拓真的青梅竹馬，在拓真搬離後唯一有跟他聯繫的人。對料理、家事非常不擅長，房間也非常髒亂。
; {{nihongo|新妻日依|新妻 ひより（にいづま ひより）|聲：くすはらゆい／[[石原舞|石原舞]]}}
: 二年級生，拓真的青梅竹馬和同班同學，性格開朗且不怕生，但經常會說出多餘的話。家裡經營咖啡廳。
; {{nihongo|世計絢花|世計 絢花（よはかり あやか）|聲：鹿野まなか／熊乃可菜}}
: 一年級生，拓真的青梅竹馬，擅長料理且很會照顧人，成績優秀。

=== 其他角色 ===
; {{nihongo|世計丸|世計 丸（よはかり まる）|聲：北見六花／[[小野涼子|小野涼子]]}}
: 可攻略角色。絢花的妹妹，喜歡絢花，討厭跟絢花親近的拓真，並常對拓真吐口水。眼神兇惡。
; {{nihongo|臼島奈奈子|臼島 奈々子（うすじま ななこ）|聲：[[遠野そよぎ|遠野そよぎ]]／[[岡嶋妙|岡嶋妙]]}}
: 可攻略角色。學園的副會長、絢花的好友。經常在中午時偷吃拓真的食物。
; {{nihongo|荒波惣時|荒波 惣時（あらなみ そうじ）|聲：平マサル／轉正浩充}}
: 杏的父親，因為妻子跟女兒懶散的關係而一肩扛起家務，希望拓真成為自己的女婿。
; {{nihongo|荒波透子|荒波 透子（あらなみ とうこ）|聲：京楓／森衣蕗}}
: 杏的母親，性格慵懶，不會做家事。
; {{nihongo|新妻美依|新妻 みより（にいづま みより）|聲：綠栞夏／森谷遙}}
: 日依的母親，自家經營和風咖啡廳「依茶屋」。
; {{nihongo|世計龍之介|世計 龍之介（よはかり りゅうのすけ）|聲：天然才／久保田健介}}
: 絢花的父親，極為疼愛女兒，非常討厭前來借宿的拓真。
; {{nihongo|世計茉莉花|世計 茉莉花（よはかり まりか）|聲：春河あかり／有河春花}}
: 絢花的母親，有極嚴重的潔癖，經常攜帶消臭噴霧，並會對丈夫以外的男性感到恐懼。
; {{nihongo|世計彈|世計 弾（よはかり だん）|聲：花音さつき／瀨田南}}
: 絢花的弟弟，每天都要自慰。
; {{nihongo|飯豐智彥|飯豊 智彦（いいで ともひこ）|聲：鬼龍院アキラ／霧生晃司}}
: 學園的學生會長，喜歡鍛鍊身體。
