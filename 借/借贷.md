'''[https://www.loan-feng.com.tw/coowner-sale 借贷]'''、'''贷款'''、'''借钱'''，俗稱'''調头寸'''，是指[[债权人|债权人]]（或放贷人）向[[债务人|债务人]]（或借款人）让渡资金使用权的一种[[金融|金融]]融资行为。放贷人（银行）无须有指定的贷款额，则可在借款人的银行户口上存款，借款人所还的款会按银行及政府所定的条款进行打扣存回银行（放贷人），若债务人未能还款则可被取回资产，银行常以此方法来获取盈利。

== 贷款要素 ==
交易对象、品种、金额、期限、利率、费率（价格）、担保方式、使用方式、清偿方式、约束条件。

== 貸款人 ==
貸款人指貸出款項的一方：
* [[自然人|自然人]]
* [[法人|法人]]或其[[代表人|代表人]]——企业法人、事业法人
* [[其他经济组织|其他经济组织]]或其[[负责人|负责人]]
* [[自然人|自然人]]、[[法人|法人]]、[[其他经济组织|其他经济组织]]或其[[代理人|代理人]]

== 利息 ==
{{see|高利貸}}
利息是指借款人为取得资金使用权而向贷款人支付的报酬，它是资本（即贷出的本金）在一定期间内的使用价格。在[[民法|民法]]中，利息是本金的[[法定孳息|法定孳息]]。

[[中華人民共和國|中華人民共和國]]法律規定，民間互相借貸最高年利率為年化24%，而24%至36%之間的利息若打官司可以不必支付，若已經支付則無法討回。但超過年化36%的為非法高利貸，無論借據如何簽訂法律皆不承認且必須返還。而砍頭息現象也就是拿借款時先被預扣第一期利息的做法，法律也不承認，只認定以實拿金額數作為本金。<ref>[https://www.youtube.com/watch?v=IYyb1zPcmPI 央視官方頻道-套路貸款]</ref>

== 利率 ==
一定期限內利息與貸款資金總額的比率，是貸款價格的表達形式。即：
:: 利率 = 利息額 / 貸款本金

利率分為日利率、月利率、年度利率。

貸款人依據各國相關法規所公佈的基準利率、利率浮動空間，而與該貸款銀行確定貸款利率。

每間銀行的演算法都有所不同，每個人在各銀行的貸款利率也會有所差異。

個人貸款利率，通常視為「一般成本」＋「機會成本」＋「信用風險」＋「變現力風險」
===負利率貸款===
當貸款利率為負利率時，借款人可從放貸者取得利息。<ref>{{Cite news |url=http://cn.nytimes.com/business/20150228/c28negative/dual/ |title=负利率的欧洲：你借钱，银行倒贴 |work=纽约时报中文网 |date=2015-02-28}}</ref>

* [[臺湾|臺湾]]：[[中華民國|中華民國]][[行政院金融監督管理委員會|行政院金融監督管理委員會]]

* [[澳门|澳门]]：[[澳門金融管理局|澳門金融管理局]]
