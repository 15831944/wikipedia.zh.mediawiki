'''久石讓'''（{{jpn|j=久石 譲|hg=ひさいし じょう|rm=Hisaishi Jō}}，{{bd|1950年|12月6日}}），本名為'''藤澤守'''（{{jpn|j=藤澤 守|hg=ふじさわ まもる|rm=Fujisawa Mamoru}}），是[[日本|日本]][[作曲家|作曲家]]、[[改编_(音乐)|編曲家]]、[[指揮家|指揮家]]、[[鋼琴家|鋼琴家]]。於[[長野縣|長野縣]][[中野市|中野市]]出生，[[日本|日本]][[國立音樂大學|國立音樂大學]]作曲科畢業。其女兒[[藤澤麻衣|麻衣]]為歌手。

藝名「久石讓」名稱源自他的偶像——美國黑人音樂家及制作人[[昆西·瓊斯|昆西·瓊斯]]。<ref>{{cite web |url=http://www.nagano-c.ed.jp/seiho/risuka/2009/2009-9.pdf |script-title=ja:久石譲の曲になる条件 |work={{lang|ja|長野県教育情報}} |author=宮下真實、平野湖納美、丸山寬花 |language=ja |format=[[pdf|pdf]] |page=1 |accessdate=2013年2月7日 |title=存档副本 |archive-url=https://web.archive.org/web/20131017143133/http://www.nagano-c.ed.jp/seiho/risuka/2009/2009-9.pdf# |archive-date=2013-10-17 |dead-url=yes }}</ref>他選用與「昆西·瓊斯」發音相近的漢字「久石讓」（Kuishi Jō），再將「久」由[[音讀|音讀]]改為[[訓讀|訓讀]]，就變成了現在的藝名（Hisaishi Jō）。

== 經歷 ==
從他4歲開始，久石讓在[[鈴木鎮一|鈴木鎮一]]的音樂教室裡學習小提琴，<ref name="jasrac1">{{cite web |url=http://www.jasrac.or.jp/sakka/vol_21/inner1.html |script-title=ja:作家で聴く音楽-JASRAC会員作家、久石譲にインタビューしました。|work=[[日本音樂著作權協會|日本音樂著作權協會]] |language=ja |accessdate=2013年2月7日}}</ref>之後，在中學演奏部社團擔任[[小號|小號]]手，並在此時期興起從事作曲家的念頭。<ref name="jasrac1" />

1969年，久石讓進入[[國立音樂大學|國立音樂大學]]，就讀作曲科，並在校內教授{{tsl|ja|島岡譲|島岡讓}}名下學習。<ref>{{cite web |url=http://www.kunitachi.ac.jp/kunion_cafe/interview/50on/ha/hisaishijoe.html |script-title=ja:久石譲（作曲家） |work=[[國立音樂大學|國立音樂大學]] |language=ja |accessdate=2013年2月7日 |deadurl=yes |archiveurl=https://web.archive.org/web/20101124020301/http://www.kunitachi.ac.jp/kunion_cafe/interview/50on/ha/hisaishijoe.html |archivedate=2010年11月24日 }}</ref>而大學期間久石讓開始受到[[簡約音樂|簡約音樂]]的風格影響，<ref>{{cite web |url=http://www.universal-music.co.jp/hisaishi-joe/biography/ |title=Biography 久石 譲（Hisaishi Joe）|work=[[環球音樂|環球音樂]] |language=ja |accessdate=2013年2月7日 }}</ref>並曾嘗試分析[[菲利普·格拉斯|菲利普·格拉斯]]或[[特里·賴利|特里·賴利]]等簡約音樂樂曲家的作品內容。

自大學畢業後，久石讓首次接手配樂的事務是在1974年替改編自{{tsl|ja|園山俊二|園山俊二}}漫畫作品的電視動畫《[[山林小獵人|山林小獵人]]》配樂。之後在1981年發行了第一張監製的專輯《MKWAJU》，<ref>{{cite web |url=http://joehisaishi.com/biography.php?chrono=198&year=1981 |title=Biography 1981|work=joehisaishi.com |language=ja |accessdate=2013年2月7日 }}</ref>另個人第一張專輯《Information》則在1982年推出。<ref>{{cite web |url=http://joehisaishi.com/biography.php?chrono=198&year=1982 |title=Biography 1982|work=joehisaishi.com |language=ja |accessdate=2013年2月7日 }}</ref>在此期間久石讓以個人本名「藤澤守」名義發表居多，而個人的事務所{{nihongo|「Wonder City」|ワンダーシティ}}則同在1982年成立。，<ref>{{cite web |url=http://www.wondercity.co.jp/company.php |script-title=ja:会社概要 |work=wondercity.co.jp |language=ja |accessdate=2013年2月7日 }}</ref>

之後因動畫室[[Topcraft|Topcraft]]企劃推出電影動畫《[[風之谷|風之谷]]》，該片的導演[[宮崎駿|宮崎駿]]及製作人[[高畑勳|高畑勳]]放棄原先計畫好的配樂人名單，決定採用推薦的人選久石讓負責該作品的音樂內容，<ref>{{cite book |script-title=ja:ジブリの哲学 |publisher=岩波書店 |author=[[鈴木敏夫|鈴木敏夫]] |isbn=978-4-00-023495-5 |page=75頁 |language=ja |date=2011年8月10日 |accessdate=2013年2月7日 }}</ref><ref>{{cite book |title=感動，如此創造 |publisher=麥田出版 |author=[[久石讓|久石讓]] |isbn=978-986-173-380-7 |page=15頁 |language=zh-hant |date=2008年5月 |accessdate=2013年2月7日 }}</ref>而久石讓也因替《風之谷》配樂後開始獲得關注，在後來多次為宮崎駿在[[吉卜力工作室|吉卜力工作室]]執導的動畫配樂。另久石讓也多次獲得[[北野武|北野武]]及[[大林宣彥|大林宣彥]]等電影導演邀約，為他們的作品構思音樂。

在1992至1994年期間，久石讓創下了連三年在[[日本電影金像獎|日本電影金像獎]]奪得音樂大獎的成績。<ref name="jap15"/><ref name="jap16"/><ref name="jap17"/>並在1998年受邀擔任[[1998年冬季帕拉林匹克運動會|長野身障奧運會]]的開幕演出。<ref>{{cite book |title=感動，如此創造 |publisher=麥田出版 |author=[[久石讓|久石讓]] |isbn=978-986-173-380-7 |page=99頁 |language=zh-hant |date=2008年5月 |accessdate=2013年2月7日 }}</ref>而久石讓首次嘗試執導的電影，則為2001年由[[袴田吉彥|袴田吉彥]]主演的{{nihongo|《四重奏》|Quartet カルテット}}<ref>{{cite book |title=感動，如此創造 |publisher=麥田出版 |author=[[久石讓|久石讓]] |isbn=978-986-173-380-7 |page=98頁 |language=zh-hant |date=2008年5月 |accessdate=2013年2月7日 }}</ref>。

除持續為日本的電影、動畫配樂及舉辦個人演奏會外，2004年後久石讓開始嘗試接案亞洲其它地方拍攝電影的配樂工作，其中為2005年南韓電影《[[歡迎來到東莫村|歡迎來到東莫村]]》製作的配樂，創下首位以日本人身份在[[大韓民國電影大獎|大韓民國電影大獎]]上獲得最佳音樂獎的記錄，<ref name="culta">{{cite web |url=http://www.culta.com/healing/healing/healing/upci1052.html |script-title=ja:２００５年韓国興行収入NO.1ヒット！ 日本人初の最優秀音楽賞を受賞（第４回大韓民国映画大賞）！ |work=culta.com |language=ja |accessdate=2013年2月7日 }}</ref>而另一接手的2007年港片《[[姨媽的後現代生活_(電影)|姨媽的後現代生活]]》，也得到第27屆[[香港電影金像獎|香港電影金像獎]]的音樂大獎。<ref name="hk27">{{cite web |url=http://www.hkfaa.com/winnerlist27.html |title=第27屆香港電影金像獎得獎名單 |work=[[香港電影金像獎|香港電影金像獎]] |language=zh-hant |accessdate=2013年2月7日 }}</ref>

2016年與新日本愛樂WORLD DREAM交響樂團一同來台，其中新專輯中的《THE EAST LAND SYMPHONY》和《Princess Mononoke》交響樂作品更是世界首演<ref>{{Cite web|url=http://memeon-music.com/2016/08/30/joe-hisaishi-livereport/|title=遇見大師樂音與風範 久石讓 JOE HISAISHI & WORLD DREAM ORCHESTRA 2016 in TAIPEI|accessdate=2017-02-24|date=2016-08-30|last=memeonmusic|work=MeMeOn Music}}</ref>。
