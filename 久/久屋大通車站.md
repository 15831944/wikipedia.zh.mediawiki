'''久屋大通車站'''（{{jpn|j=久屋大通駅|hg=ひさやおおどおりえき|rm=Hisaya-ōdōri|f=y}}）是位在[[愛知縣|愛知縣]][[名古屋市|名古屋市]][[中区_(名古屋市)|中区]][[錦三丁目|錦三丁目]]。為[[名古屋市營地下鐵|名古屋市營地下鐵]]的車站之一。1999年入選[[中部車站百選|中部車站百選]]（第一回）。

== 可利用之鐵路路線 ==
*名古屋市營地下鐵
**[[名城線|名城線]] - 車站編號為'''M06'''。
**[[櫻通線|櫻通線]] - 車站編號為'''S05'''。

== 車站結構 ==
[[地下車站|地下車站]]。名城線月台可對應6節編組15公尺列車的[[對向式月台|對向式月台]]2面2線，而櫻通線月台可對應8節編組20公尺列車的[[島式月台|島式月台]]1面2線。只限櫻通線月台設有月台閘門，車站顏色為紫紅色。而因離前一站[[榮車站_(愛知縣)|榮站]]僅不到400公尺，兩站間為[[名古屋市營地下鐵|名古屋市營地下鐵]]中相距最短之區間。也因為是筆直的路線區間，可從月台最底端處看得見[[榮車站_(愛知縣)|榮站]]。

== 本站起源 ==
*1989年（[[平成|平成]]元年）9月10日 - 配合櫻通線開通，與名城線交會之故也新設立了本站。久屋大通車站為雙線同時開始營運之站。
