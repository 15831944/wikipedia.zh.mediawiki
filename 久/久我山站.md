'''久我山站'''（{{jpn|j=久我山駅|hg=くがやまえき|rm=Kugayama eki|f=y}}）是一個位於[[東京都|東京都]][[杉並區|杉並區]][[久我山|久我山]]四丁目，屬於[[京王電鐵|京王電鐵]][[井之頭線|井之頭線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''IN14'''。

== 歷史 ==
* 1933年（[[昭和|昭和]]8年）8月1日 - 隸屬於帝都電鐵的車站開始營運。
* 1940年（昭和15年）5月1日 - 與小田原急行鐵道合併，成為該公司帝都線的車站。
* 1942年（昭和17年）5月1日 - [[小田急電鐵|小田急電鐵]]與[[東京急行電鐵|東京急行電鐵]]（大東急）合併。
* 1948年（昭和23年）6月1日 - 從東急分離，成立京王帝都電鐵，成為該公司井之頭線的車站。
* 2005年（[[平成|平成]]17年）
** 2月 - 跨橋式站房（南口）啟用。
** 12月 - 橋跨橋式站房（北口）啟用。
* 2013年（平成25年）12月18日 - 進站音樂採用本地作曲家湯山昭的童謠《おはなしゆびさん》、《山のワルツ》<ref>{{PDFlink|[http://www.keio.co.jp/news/backnumber/news_release2013/nr131216_melody.pdf 童謡の「おはなしゆびさん」と「山のワルツ」を期間限定で久我山駅の列車接近メロディーにします！]}} - 京王電鉄（2013年12月16日付、同月25日閲覧）</ref>。最先是1年限定，期限到後決定繼續使用。

== 車站構造 ==
[[跨站式站房|跨站式站房]]設計，具有1面2線[[島式月台|島式月台]]的[[地面車站|地面車站]]。站房原為一般地面車站，設置至於現今北口的位置，以地下道連接月台。

2005年新站房改建的同時，在站內開設了商業空間。

== 車站周邊 ==
南口為{{link-ja|人見街道|人見街道}}，而正面為通往{{link-ja|岩崎通信機|岩崎通信機}}的[[商店街|商店街]]，統稱「岩通通」。前方是[[神田川_(東京都)|神田川]]，站前的久我山橋同時也作為站前廣場使用。相較於南口，設有商店街的北口較為熱鬧，稍遠離車站則為住宅區。

附近有{{link-ja|國學院大學久我山中學校・高等學校|國學院大學久我山中学校・高等学校}}、{{link-ja|東京都立西高等學校|東京都立西高等学校}}、{{link-ja|杉並區立西宮中學校|杉並区立西宮中学校}}、杉並區立高井戶第二小學校等設施。
