'''颶風何塞'''（{{lang-en|Hurricane Jose}}）是一場強勁且動向飄忽不定的[[熱帶氣旋|熱帶氣旋]]，是自[[2012年大西洋颶風季|2012年]][[飓风纳丁|飓风纳丁]]以來持續時間最長的大西洋颶風。何塞是[[2017年大西洋颶風季|2017年大西洋颶風季]]第10場{{link-en|熱帶氣旋命名|Tropical cyclone naming|獲命名}}的風暴，第5場颶風和第3場[[萨菲尔-辛普森飓风等级|大型颶風]]。何塞在9月5日發展成熱帶風暴，源自一個差不多在一星期前離開[[非洲|非洲]]西岸的[[东风波|东风波]]。隨著何塞達到颶風強度，它在9月6日進入[[快速增强|快速增强]]時期。9月8日，它達到四級颶風上限的最高強度。然而，受[[风切变|风切变]]影響，何塞在接下來的幾天逐漸減弱，並在[[伊斯帕尼奧拉島|伊斯帕尼奧拉島]]以北完成一個逆時針環路。儘管何塞在9月14日減弱為熱帶風暴，但它在第二天重拾颶風強度並開始轉向北移動。何塞在其餘下的生命史內從未增強至超過一級以上的標準，何塞在9月20日再次減弱為熱帶風暴。兩天後，何塞在[[新英格蘭|新英格蘭]]近岸向東北方飄移，並轉變成[[後熱帶氣旋|後熱帶氣旋]]。到了9月26日，何塞的殘留在[[美國東岸|美國東岸]]近海消散。

最初預計會影響已受[[颶風伊爾瑪|颶風伊爾瑪]]影響的安的列斯群島，何塞引發居民從受災難性損害的[[巴布達島|巴布達島]]以及[[圣马丁岛_(加勒比海)|聖馬丁島]]上撤離。最後，隨著何塞改變路徑，其內核心以及最強勁的風力都停留在海上。儘管如此，何塞仍然給這些島嶼帶來了熱帶風暴強度的大風。後來，何塞給美國東岸帶來大雨、湧浪和強勁的海浪，造成海灘侵蝕和一些洪災。有一人在[[阿斯伯里帕克|阿斯伯里帕克]]遇到離岸流而死亡。

==氣象歷史==
{{storm path|Jose 2017 track.png}}
8月31日，一股向西移動的[[东风波|东风波]]離開非洲西岸<ref name="TCR">{{cite web|url=https://www.nhc.noaa.gov/data/tcr/AL122017_Jose.pdf|title=Tropical Cyclone Report: Hurricane Jose|publisher=[[National_Hurricane_Center|National Hurricane Center]]|first1=Robbie|last1=Berg|date=2018-02-20|accessdate=2017-03-10}}</ref><ref>{{cite report|author=Eric Blake|date=2017-08-31|accessdate=2017-09-07|title=Tropical Weather Outlook|publisher=National Hurricane Center|url=http://www.nhc.noaa.gov/archive/text/TWOAT/2017/TWOAT.201708311738.txt|format=TXT}}</ref>。東風波與其結構混亂的雷暴在9月2日經過[[佛得角|佛得角]]以南。但是，環境條件有利於逐步發展，這促使[[國家颶風中心|國家颶風中心]]開始追蹤該系統<ref>{{cite report|author=John Cangialosi|date=2017-09-02|accessdate=2017-09-07|title=Tropical Weather Outlook|publisher=National Hurricane Center|url=http://www.nhc.noaa.gov/archive/text/TWOAT/2017/TWOAT.201709020533.txt|format=TXT}}</ref>。9月4日早上，東風波內的一個表面低氣壓在佛得角群島西南偏南約990公里處形成。系統繼續組織，一個熱帶低氣壓估計在{{lang|en|UTC}}9月5日上午6時形成，6小時後增強至熱帶風暴標準；因此，它被命名為何塞（{{lang|en|Jose}}）<ref name="TCR"/>。在實際操作中，國家颶風中心直到當天{{lang|en|UTC}}下午15時，在實際形成9小時後才為該熱帶風暴發出公告<ref>{{cite report|author=Chris Landsea|title=Tropical Storm Jose Discussion Number 1|publisher=National Hurricane Center|date=2017-09-04|accessdate=2017-09-07|url=http://www.nhc.noaa.gov/archive/2017/al12/al122017.discus.001.shtml?}}</ref>。

當何塞成為熱帶風暴，它在溫暖的[[海面溫度|海面溫度]]，低[[風切變|風切變]]和豐富的水分的有利環境中逐漸增強。<ref>{{cite report|author=Chris Landsea|title=Tropical Storm Jose Discussion Number 2|publisher=National Hurricane Center|date=2017-09-05|accessdate=2017-09-07|url=http://www.nhc.noaa.gov/archive/2017/al12/al122017.discus.002.shtml?}}</ref>。在[[副熱帶高壓脊|副熱帶高壓脊]]的影響下，風暴向西北偏西移動，並形成[[風眼|眼狀特徵]]和對稱的徑向對流<ref name="TCR"/>。9月6日早上，受惠於有利的條件，風暴進入[[快速增強|快速增強]]時期，何塞{{lang|en|UTC}}當天晚上18時達到颶風標準<ref name="TCR"/>。與此同時，何塞連同[[颶風伊爾瑪|颶風伊爾瑪]]和{{link-en|颶風卡蒂婭 (2017年)|Hurricane Katia (2017)|卡蒂婭}}，標誌著自[[2010年大西洋颶風季|2010年]]以來首次在大西洋同時出現三場颶風<ref>{{Cite news|url=http://www.edition.cnn.com/2017/09/06/americas/three-hurricanes-atlantic-basin/index.html|title=Three hurricanes now in the Atlantic basin|last=Grinberg|first=Emanuella|date=2017-09-07|agency=CNN|access-date=2017-09-07}}</ref>。儘管接近位於其西方，規模更大的颶風伊爾瑪的流出，何塞在接下來的兩天繼續迅速增強，最終在{{lang|en|UTC}}9月8日晚上18時於[[背風群島_(加勒比海)|背風群島]]以東達到最高風速每小時250公里和最低氣壓938毫巴（27.7英寸汞柱）<ref name="TCR"/>。何塞連同即將以五級颶風標準登陸古巴的伊爾瑪，標誌著首次同時出現兩場最大持續風速達至少每小時240公里的大西洋颶風<ref>{{Cite news|url=http://edition.cnn.com/2017/09/08/us/hurricane-jose/index.html|title=Hurricane Jose strengthens to 'extremely dangerous' Category 4|last=Levenson|first=Eric|date=2017-09-09|work=CNN|access-date=2017-09-09}}</ref>。

[[File:Jose_2017-09-19_1540Z.jpg|thumb]]近海的熱帶風暴何塞]]
隨着其風眼變為雲塞風眼以及風暴開始受風切變影響，何塞緩慢地減弱<ref>{{Cite report|url=http://www.nhc.noaa.gov/archive/2017/al12/al122017.discus.022.shtml?|title=Hurricane Jose Discussion Number 22|author=Joan R. Ballard|date=2017-09-10|publisher=National Hurricane Center|access-date=2017-09-16}}</ref>，在{{lang|en|UTC}}9月10日晩上18時降至低於四級標準<ref name="TCR"/>。隨着較強的風切變開始侵蝕核心，風暴在第二天{{lang|en|UTC}}上午8時降至低於大型颶風標準，再在{{lang|en|UTC}}9月11日晚上18時降至低於二級標準<ref name="TCR"/><ref>{{Cite report|url=http://www.nhc.noaa.gov/archive/2017/al12/al122017.discus.027.shtml?|title=Hurricane Jose Discussion Number 27|author=Thomas Birchard, David Roth and Chris Sisko|date=2017-09-12|publisher=National Hurricane Center|access-date=2017-09-16}}</ref>。隨着風暴進入逆時針環路，根據德沃夏克的估計，其風速低於颶風標準，何塞在{{lang|en|UTC}}9月15日時凌晨0時被降級為熱帶風暴<ref name="TCR"/>。此時國家颶風中心注意到，北風的風切變把其全部明顯的帶狀保持在東南象限，中心則位於大部分對流的西北方<ref>{{Cite report|url=http://www.nhc.noaa.gov/archive/2017/al12/al122017.discus.037.shtml?|title=Tropical Storm Jose Discussion Number 37|author=Eric Blake|date=2017-09-14|publisher=National Hurricane Center|access-date=2017-09-16}}</ref>。然而，當風暴在當晚完成逆時針環路時，一架偵察機錄得高於颶風門檻的表面風速。因此，國家颶風中心再次將何塞升級為颶風<ref>{{cite report|author=Eric Blake|title=Hurricane Jose Discussion Number 42|publisher=National Hurricane Center|date=2017-09-15|accessdate=2017-09-15|url=http://www.nhc.noaa.gov/text/refresh/MIATCDAT2+shtml/151453.shtml}}</ref>。圍繞着副熱帶高壓脊西部的邊緣，何塞從9月16日開始向北移動<ref>{{cite report|author=Robbie Berg|title=Hurricane Jose Discussion Number 46|publisher=National Hurricane Center|date=2017-09-16|accessdate=2017-09-22|url=http://www.nhc.noaa.gov/archive/2017/al12/al122017.discus.046.shtml?}}</ref>。儘管從衛星圖像上的外觀並不對稱，颶風在9月17日稍微增強，並在{{lang|en|UTC}}中午12時達到風力時速150公里的第二波最高強度<ref name="TCR"/>。

何塞的風場在其北上期間繼續擴大，中心的北部邊緣發展出大型的對流帶，但同時位於中心的雷暴減少<ref>{{cite report|author=Daniel P. Brown|title=Hurricane Jose Discussion Number 51|publisher=National Hurricane Center|date=2017-09-17|accessdate=2017-09-22|url=http://www.nhc.noaa.gov/archive/2017/al12/al122017.discus.051.shtml?}}</ref><ref>{{cite report|author=Dave Roberts|title=Hurricane Jose Discussion Number 52|publisher=National Hurricane Center|date=2017-09-17|accessdate=2017-09-22|url=http://www.nhc.noaa.gov/archive/2017/al12/al122017.discus.052.shtml?}}</ref>。風暴在9月19日於[[北卡羅來納州|北卡羅來納州]]以東再次發展出一片對流區和眼狀特徵<ref>{{cite report|author=Stacy Stewart|title=Hurricane Jose Discussion Number 57|publisher=National Hurricane Center|date=2017-09-19|accessdate=2017-09-22|url=http://www.nhc.noaa.gov/archive/2017/al12/al122017.discus.057.shtml?}}</ref>。9月20日，[[颶風獵人|颶風獵人]]的飛機表明何塞減弱為熱帶風暴，當時風暴正轉向東北方<ref name="Disc 59">{{cite report|author=Lixion Avila|title=Tropical Storm Jose Discussion Number 59|publisher=National Hurricane Center|date=2017-09-20|accessdate=2017-09-22|url=http://www.nhc.noaa.gov/archive/2017/al12/al122017.discus.059.shtml?}}</ref>。此後，隨著風暴經過[[墨西哥灣暖流|墨西哥灣暖流]]以北，該區的水溫較低，令中心的對流減弱<ref>{{cite report|author=Lixion Avila|title=Tropical Storm Jose Discussion Number 60|publisher=National Hurricane Center|date=2017-09-20|accessdate=2017-09-22|url=http://www.nhc.noaa.gov/archive/2017/al12/al122017.discus.060.shtml?}}</ref>。在對流減少超過12小時，以及風暴已經獲得鋒面系統後，國家颶風中心在9月22日早上將何塞重新指定為[[後熱帶氣旋|後熱帶氣旋]]<ref name="TCR"/>。對流帶北部橫過[[新英格蘭|新英格蘭]]，而中心則在[[鳕鱼角|鳕鱼角]]的東南方飄移<ref>{{cite report|author=David Zelinsky|title=Post-Tropical Cyclone Jose Discussion Number 67|publisher=National Hurricane Center|date=2017-09-22|accessdate=2017-09-22|url=http://www.nhc.noaa.gov/archive/2017/al12/al122017.discus.067.shtml?}}</ref>。何塞的殘留徘徊了兩天，然後在9月25日消散<ref name="TCR"/>。

===美國===
[[File:US_Geo_survey_hurricane_jose_2017.jpg|thumb]]
在風暴來臨前，美國三個州的地質調查專家安裝了17個風暴潮感應器——[[康乃狄克州|康乃狄克州]]7個、[[麻薩諸塞州|麻薩諸塞州]]7個、[[羅德島州|羅德島州]]3個——沿著可能受到一些大浪和風暴潮衝擊的海岸線，以收集有關風暴影響的資料<ref>{{Cite web|url=https://www.usgs.gov/news/more-a-dozen-usgs-storm-tide-sensors-deployed-hurricane-jose|title=More than a dozen USGS Storm-Tide Sensors Deployed for Hurricane Jose|publisher=United States Geological Survey|language=en|access-date=2017-09-20}}</ref>。國家颶風中心向大西洋海岸線的部分地區發出熱帶風暴警告，包括[[北卡罗来纳州|北卡罗来纳州]][[外灘群島|外灘群島]]，經過[[德玛瓦半岛|德瑪瓦]]和{{link-en|澤西海岸 (新澤西州)|Jersey Shore|澤西海岸}}。[[長島_(紐約州)|長島]]和[[康乃狄克州|康乃狄克州]]、[[羅德島州|羅德島州]]和[[麻薩諸塞州|麻薩諸塞州]]的海岸線也收到[[熱帶風暴警告|熱帶風暴警告]]。麻薩諸塞州[[南塔克特|南塔克特]]和部分外灘群島也收到風暴潮警報<ref name="Disc 59" />。
