'''颶風卡洛塔'''（{{lang-en|Hurricane Carlotta}}）是自1966年以來登陸位置最偏東面，而且達颶風強度的{{link-en|熱帶氣旋盆地|Tropical cyclone basins|東太平洋}}[[熱帶氣旋|熱帶氣旋]]。作為[[2012年太平洋颶風季|2012年太平洋颶風季]]的第三場[[熱帶氣旋|熱帶氣旋]]和第三個被命名風暴，卡洛塔在6月14日從[[中美洲|中美洲]]西南方的[[东风波|东风波]]緩慢地發展為熱帶低氣壓。它大致向西北偏西移動，第二天增強為熱帶風暴。此後，風暴逐漸增強，風暴在6月15日達到颶風標準。隨著風暴[[快速增强|快速增强]]，卡洛塔在當天達到了每小時175公里的二級颶風的最高強度。UTC翌日上午1時，卡洛塔在[[埃斯孔迪多港_(瓦哈卡州)|埃斯孔迪多港]]附近[[登陸_(氣象學)|登陸]]，這是當時記錄的歷史上登陸位置最偏東面的太平洋颶風<ref name=tcr />。第二天，隨著風暴在墨西哥西南部上岸，風暴開始減弱。卡洛塔繼續快速減弱，最終在6月16日消散。

據報導，整個墨西哥有廣泛的[[停電|停電]]和風力所帶來的損害，特別是在[[瓦哈卡州|瓦哈卡州]]。還有報導說，風和雨造成基礎設施損毀，以及[[農作物|農作物]]受損。卡洛塔的降雨在[[聖胡安包蒂斯塔圖斯特佩克|聖胡安包蒂斯塔圖斯特佩克]]最高達到35.0厘米。暴雨造成許多[[山崩|山崩]]，阻塞了道路和損毀了建築物。至少有29,000個家庭和2,500個企業遭受卡洛塔的損失，大多數在瓦哈卡州。卡洛塔造成7人死亡，瓦哈卡州要求達14億[[墨西哥比索|比索]]（1.077億美元）用於修復公共基礎設施{{#tag:ref|[[墨西哥比索|墨西哥比索]]兌換[[美元|美元]]在[http://www.oanda.com/currency/converter/ 本網站]兌換，兌換日期為2012年6月16日|group="注"}}。

==氣象歷史==
{{storm path|Carlotta 2012 track.png}}
6月12日，[[哥斯達黎加|哥斯達黎加]]西岸發展出熱帶擾動，此擾動向西移動，經過[[巴拿馬|巴拿馬]]後進入東太平洋<ref name=tcr /><ref>{{cite report|author=Kimberlain, Todd|publisher=National Hurricane Center|date=2012-06-12|title=Tropical Weather Outlook|accessdate=2012-10-07|url=http://www.nhc.noaa.gov/archive/text/TWOEP/2012/TWOEP.201206121131.txt|format=TXT}}</ref>。
隨著系統向西北偏西移動，向西的[[開爾文波|開爾文波]]為其發展提供了有利條件，系統變得更加有組織<ref name=tcr /><ref>{{cite report|author=Kimberlain, Todd|publisher=National Hurricane Center|date=2012-06-12|title=Tropical Weather Outlook|accessdate=2012-10-07|url=http://www.nhc.noaa.gov/archive/text/TWOEP/2012/TWOEP.201206121732.txt|format=TXT}}</ref>。{{link-en|大氣對流|Convection (meteorology)|對流}}在發展中的[[低气压|低氣壓區]]内增加，[[國家颶風中心|國家颶風中心]]估計在6月12日晚上有50％的{{link-en|熱帶氣旋發展學|Tropical cyclogenesis|熱帶氣旋發展機會}}<ref>{{cite report|author=Cangialosi, John|publisher=National Hurricane Center|date=2012-06-12|title=Tropical Weather Outlook|accessdate=2012-10-07|url=http://www.nhc.noaa.gov/archive/text/TWOEP/2012/TWOEP.201206122332.txt|format=TXT}}</ref>。進一步組織後，國家颶風中心在{{lang|en|[[协调世界时|UTC]]}}6月14日上午3時發表關於第三{{lang|en|E}}熱帶低氣壓的公告，而該系統位於[[瓦哈卡州|瓦哈卡州]]{{link-en|安赫爾港|Puerto Ángel}}東南偏南約830公里。當時，這個低氣壓位於被認為有利於增強的低[[风切变|风切变]]和溫暖的[[海面溫度|海面溫度]]條件下。此外，它的[[大氣環流|環流]]很小，這增加了[[快速增強|快速增強]]的可能性。由於系統東北面有[[高壓脊|高壓脊]]形成，該系統在形成後向西北移動<ref name="disc1">{{cite report|author=Brown, Daniel|author2=Berg, Robbie|date=2012-06-14|title=Tropical Depression Three-E Discussion Number 1|publisher=National Hurricane Center|accessdate=2012-10-07|url=http://www.nhc.noaa.gov/archive/2012/ep03/ep032012.discus.001.shtml?}}</ref>。形成後約6小時，在中心附近的[[雨帶|雨帶]]增加後，低氣壓增強成熱帶風暴卡洛塔<ref>{{cite report|author=Stewart, Stacy|date=2012-06-14|title=Tropical Storm Carlotta Discussion Number 2|publisher=National Hurricane Center|accessdate=2012-10-07|url=http://www.nhc.noaa.gov/archive/2012/ep03/ep032012.discus.002.shtml?}}</ref>。

在持續的有利條件下，卡洛塔很快就形成了一個粗糙的[[中心密集雲團區|中心密集雲團區]]或集中的對流區域<ref>{{cite report|author=Beven, Jack|date=2012-06-14|title=Tropical Storm Carlotta Discussion Number 4|publisher=National Hurricane Center|accessdate=2012-10-07|url=http://www.nhc.noaa.gov/archive/2012/ep03/ep032012.discus.004.shtml?}}</ref>。
6月15日，卡洛塔開始發展出[[風眼|眼狀特徵]]<ref>{{cite report|author=Stewart, Stacy|date=2012-06-15|title=Tropical Storm Carlotta Discussion Number 6|publisher=National Hurricane Center|accessdate=2012-10-07|url=http://www.nhc.noaa.gov/archive/2012/ep03/ep032012.discus.006.shtml?}}</ref>，當天晩上卡洛塔逼近{{link-en|墨西哥西南部|Southwestern Mexico}}，並增強成颶風<ref>{{cite report|author=Beven, Jack|date=2012-06-15|title=Hurricane Carlotta Discussion Number 7|publisher=National Hurricane Center|accessdate=2012-10-07|url=http://www.nhc.noaa.gov/archive/2012/ep03/ep032012.discus.007.shtml?}}</ref>。6月15日晚，[[颶風獵人|颶風獵人]]的飛機師表明卡洛塔正在快速增強，觀察到風眼中有每小時175公里的風力和良好的[[流出_(氣象學)|流出]]；
機組人員還在飛行高度觀察到每小時183公里的風力。國家颶風中心最初預測颶風將沿著墨西哥沿岸地區移動，這將使卡洛塔能夠保持其大部分的強度<ref>{{cite report|author=Beven, Jack|date=2012-06-15|title=Hurricane Carlotta Discussion Number 8|publisher=National Hurricane Center|accessdate=2012-10-07|url=http://www.nhc.noaa.gov/archive/2012/ep03/ep032012.discus.008.shtml?}}</ref>。在達到最高強度後不久，颶風卡洛塔稍微減弱，其風眼開始與陸地相互作用<ref>{{cite report|author=Avila, Lixion|date=2012-06-16|title=Hurricane Carlotta Discussion Number 9|publisher=National Hurricane Center|accessdate=2012-10-07|url=http://www.nhc.noaa.gov/archive/2012/ep03/ep032012.discus.009.shtml?}}</ref>。{{lang|en|UTC}}6月16日上午1時後不久，颶風在[[瓦哈卡州|瓦哈卡州]][[埃斯孔迪多港_(瓦哈卡州)|埃斯孔迪多港]]附近[[登陸_(氣象學)|登陸]]，估計風速為每小時165公里，最低氣壓為976毫巴（28.8英寸汞柱）<ref>{{cite report|author=Avila, Lixion|date=2012-06-16|title=Hurricane Carlotta Advisory Number 9|accessdate=2012-10-07|publisher=National Hurricane Center|url=http://www.nhc.noaa.gov/archive/2012/ep03/ep032012.public.009.shtml?}}</ref>，成為登陸位置最偏東面，而且達颶風強度的{{link-en|熱帶氣旋盆地|Tropical cyclone basins|東太平洋}}熱帶氣旋，直到一年後被[[2013年飓风芭芭拉|颶風芭芭拉]]超越<ref name=tcr /><ref>{{cite report|author=Daniel P. Brown|title=Tropical Cyclone Report: Hurricane Barbara|url=http://www.nhc.noaa.gov/data/tcr/EP022013_Barbara.pdf|publisher=National Hurricane Center|date=2013-08-19|accessdate=2017-05-03|location=Miami, Florida|format=PDF}}</ref>。卡洛塔在內陸移動時迅速減弱，登陸後12小時已經退化為熱帶低氣壓。雖然雲系模式受到干擾，風暴仍然保持大面積的雷暴<ref>{{cite report|author=Avila, Lixion|date=2012-06-16|title=Tropical Depression Carlotta Discussion Number 11|publisher=National Hurricane Center|accessdate=2012-10-07|url=http://www.nhc.noaa.gov/archive/2012/ep03/ep032012.discus.011.shtml?}}</ref>。
6月17日早上較早時候，國家颶風中心報告卡洛塔已經減弱為[[熱帶氣旋#消散|後熱帶殘留低氣壓]]；該系統當時在墨西哥西南部仍有[[大氣環流|廣泛環流]]<ref>{{cite report|author=Avila, Lixion|date=2012-06-17|title=Post-Tropical Cyclone Carlotta Discussion Number 13|publisher=National Hurricane Center|accessdate=2012-10-07|url=http://www.nhc.noaa.gov/archive/2012/ep03/ep032012.discus.013.shtml?}}</ref>。當天晚上，這個低氣壓退化成墨西哥西岸的一個[[低壓槽|低壓槽]]<ref>{{cite report|author=Brown, Daniel|publisher=National Hurricane Center|date=2012-06-17|title=Tropical Weather Outlook|accessdate=2012-10-07|url=http://www.nhc.noaa.gov/archive/text/TWOEP/2012/TWOEP.201206171153.txt|format=TXT}}</ref>。
