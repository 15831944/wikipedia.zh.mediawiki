《'''鹰隼大队'''》，2009年上映于中国大陆的电视剧，导演[[彭昱凯|彭昱凯]]，主演是沙溢、殷桃、林永健、姬晨牧等。<ref>{{cite web |url=http://ent.qq.com/a/20111014/000422.htm |title=《鹰隼大队》央视重播 胡亚捷温情政委暖军心 |accessdate=2011 |publisher=腾讯网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.qq.com/a/20110712/000446.htm |title=《鹰隼大队》地方台重播 殷桃变身“武器专家” |accessdate=2011 |publisher= 腾讯网|work= |language=zh-hans}}</ref>该剧是“庆祝中国人民解放军空军成立60周年”纪念剧。<ref>{{cite web |url=http://ent.qq.com/a/20101216/000255.htm |title=殷桃倾情演绎《鹰隼大队》 重播余热不减 |accessdate=2010 |publisher=腾讯网 |work= |language=zh-hans}}</ref>

==剧情==
该剧讲述[[中国人民解放军空军|中国人民解放军空军]]为了适应21世纪形势变化，组建的“鹰隼大队”的故事。
