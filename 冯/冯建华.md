'''冯建华'''（{{bd|1958年|12月|||}}），[[山东|山东]][[诸城|诸城]]人，[[中国人民解放军中将|中国人民解放军中将]]。

曾任[[中国人民解放军军事科学院防化研究院|中国人民解放军军事科学院防化研究院]]政委，2011年任[[总装备部|总装备部]]政治部副主任<ref>{{cite web |title=冯建华少将升任战略支援部队政治工作部主任 |url=http://china.caixin.com/2016-02-29/100913753.html |website=财新网 |accessdate=2019-03-02 |date=2016-02-29}}</ref>。2013年，出任[[中国人民解放军总政治部干部部|中国人民解放军总政治部干部部]]部长，2015年担任解放军选举委员会委员<ref>{{cite web |title=全国人民代表大会常务委员会批准任免的名单 |url=http://www.npc.gov.cn/npc/xinwen/2015-02/28/content_1906318.htm |website=中国人大网 |accessdate=2019-03-02 |date=2015-02-27}}</ref>。2015年12月，升任[[中国人民解放军战略支援部队|中国人民解放军战略支援部队]]政治工作部主任<ref>{{cite web |author1=王俊 |title=冯建华少将调任战略支援部队政治部主任 |url=https://news.qq.com/a/20160228/028165.htm |website=腾讯 |accessdate=2019-03-02 |date=2016-02-28}}</ref>。

2017年晋升中将军衔<ref>{{cite web |author1=岳怀让 |title=战略支援部队政治工作部主任冯建华已晋升中将军衔 |url=https://www.thepaper.cn/newsDetail_forward_1838668 |website=澎湃新闻 |accessdate=2019-03-02 |date=2017-10-30}}</ref>。[[中国共产党第十九届中央委员会候补委员列表|中国共产党第十九届中央委员会候补委员]]<ref>{{cite web |title=十九大受权发布：中国共产党第十九届中央委员会候补委员名单 |url=http://www.xinhuanet.com//2017-10/24/c_1121848883.htm |website=新华网 |accessdate=2019-03-02 |date=2017-10-24}}</ref>。
