'''冯行袭'''（{{bd|9世纪||910年|7月31日|catIdx=F}}<ref name=ZZTJ267>[[:s:資治通鑑/卷267|《资治通鉴》卷二百六十七]]</ref>），字'''正臣'''，封'''长乐郡王'''，谥'''忠敬'''，晚唐军阀，后依附取代[[唐朝|唐朝]]的[[后梁|后梁]]。身材魁岸雄壮，脸上有青色胎记，时人称之“冯青面”。<ref name=HFD15>[[:s:舊五代史/卷15|《旧五代史》卷一十五]]</ref>

== 家世背景 ==
冯行袭生年不详，[[均州|均州]][[武当|武当]]人，以有勇有谋闻名当地。<ref name=NBT186>[[:s:新唐書/卷186|《新唐书》卷一百八十六]]</ref>曾任均州都校。[[中和_(唐朝)|中和]]四年（884年）十二月，农民军头领[[孙喜|孙喜]]聚集数千人准备攻均州。刺史[[吕烨|吕烨]]害怕，冯行袭却伏下军队，只身往见孙喜：“本州人会欢迎你成为本州的良牧的。但你的徒众太多，本州人害怕被劫掠，害怕你。你应该把军队留在[[汉江|汉江]]以北，只率心腹人轻骑前进。我冯行袭将做你的前导，安慰士民，他们肯定会归顺你。”孙喜高兴，不怀疑冯行袭。孙喜渡河后，均州士卒都迎上来，作尊敬状。冯行袭部伏兵尽出，冯行袭将孙喜打倒在地，斩之，与伏兵手刃其心腹数人。孙喜留在江北的军队也溃散。冯行袭乘胜驱逐吕烨。<ref name=NBT9>[[:s:新唐书/卷9|《新唐书》卷九]]作“武当贼冯行袭”。</ref>均州上级的[[山南东道节度使|山南东道节度使]][[刘巨容|刘巨容]]将此事上奏因农民军[[黄巢|黄巢]]攻占[[长安|长安]]而停留[[成都|成都]]的[[唐僖宗|唐僖宗]]，冯行袭被任为均州刺史。当时均州西有盗贼，常拦截东面诸镇对成都唐僖宗的上贡。冯行袭攻杀群盗贼，确保贡路畅通。<ref name=ZZTJ256>[[:s:資治通鑑/卷256|《资治通鉴》卷二百五十六]]</ref>[[洋州|洋州]]节度使[[葛佐|葛佐]]辟冯行袭为行军司马，使以兵镇谷口，通秦、蜀道路，冯行袭由此知名。[[凤翔节度使|凤翔节度使]][[李茂贞|李茂贞]]派养子[[李继臻|李继臻]]据[[金州|金州]]，<ref>李茂贞887年才成为凤翔节度使，故此事必发生在887年之后。见《资治通鉴》卷二百五十七。</ref>冯行袭攻李继臻，占领金州。[[唐昭宗|唐昭宗]]任冯行袭为昭信防御使，以金州为治所。<ref name=HFD15/><ref name=NHFD42>[[:s:新五代史/卷42|《新五代史》卷四十二]]</ref><ref name=ZZTJ258>[[:s:資治通鑑/卷258|《资治通鉴》卷二百五十八]]：初，李茂贞养子继臻据金州，均州刺史冯行袭攻下之，诏以行袭为昭信防御使，治金州。杨守亮欲自金、商袭京师，行袭逆击，大破之。</ref>

== 晚唐军阀 ==
[[文德_(唐朝)|文德]]元年（888年）唐僖宗驾崩后，当权宦官[[杨复恭|杨复恭]]助唐昭宗继位。<ref name=ZZTJ257>[[:s:資治通鑑/卷257|《资治通鉴》卷二百五十七]]</ref>[[大顺_(唐朝)|大顺]]二年（891年），杨复恭与昭宗生隙，与养子、侄子[[杨守亮|杨守亮]]、[[杨守忠|杨守忠]]、[[杨守贞|杨守贞]]、[[杨守厚|杨守厚]]反叛昭宗。十二月，杨守亮试图经由冯行袭辖区攻昭宗所在的长安，被冯行袭大败。<ref name=HFD15/><ref name=ZZTJ258/>后诸杨为李茂贞所败。<ref name=ZZTJ259>[[:s:資治通鑑/卷259|《资治通鉴》卷二百五十九]]</ref>

冯行袭被授检校[[太子少保|太子少保]]，封'''长乐县开国子'''，加食邑。<ref>[[:s:授金州刺史冯行袭检校太子少保仍封长乐县开国子加食邑制|钱珝《授金州刺史冯行袭检校太子少保仍封长乐县开国子加食邑制》]]</ref>[[光化|光化]]元年（898年）正月，冯行袭以昭信军防御使、[[特进|特进]]、检校太子太保、使持节金州诸军事、守金州刺史、[[御史大夫|御史大夫]]、[[上柱国|上柱国]]、'''长乐县开国伯'''、食邑七百户的身份被任为昭信节度使，昭信因而升格为军镇。<ref name=HFD15/><ref name=ZZTJ261>[[:s:資治通鑑/卷261|《资治通鉴》卷二百六十一]]：以昭信防御使冯行袭为昭信节度使。</ref><ref>[[:s:授冯行袭昭信军节度使制|张元晏《授冯行袭昭信军节度使制》]]</ref>

[[天复_(唐朝)|天复]]元年（901年），唐昭宗和宰相[[崔胤|崔胤]]屠戮宦官的计划被察觉，宦官[[韩全诲|韩全诲]]挟持昭宗奔凤翔投李茂贞。崔胤召有权势的军阀宣武军节度使[[朱全忠|朱全忠]]前来长安，朱全忠围攻凤翔府。其间，韩全诲派郄文晏等二十余名宦官行经金州，意图前往东南诸镇命他们攻打朱全忠。[[同州|同州]]节度留后[[司马邺_(后梁)|司马邺]]出使说服冯行袭，冯行袭派副使[[鲁崇矩|鲁崇矩]]去朱全忠处听命。<ref>[[:s:舊五代史/卷20|《旧五代史》卷二十]]</ref>十二月，当宦官们到达金州时，冯行袭尽数诛之，<ref name=NHFD42/>将他们携带的韩全诲矫诏送到朱全忠处。<ref name=HFD15/><ref name=ZZTJ262>[[:s:資治通鑑/卷262|《资治通鉴》卷二百六十二]]：朱全忠之入关也，戎昭节度使冯行袭遣副使鲁崇矩听命于全忠。韩全诲遣中使二十馀人分道征江、淮兵屯金州，以胁全忠，行袭尽杀中使，收其诏敕送全忠。</ref>

[[天祐_(唐朝)|天祐]]元年（904年），冯行袭改姓'''朱'''，兼领[[武定军节度使|武定军节度使]]。<ref>《资治通鉴》误作[[武安军节度使|武安军节度使]]。</ref><ref>《续宝运录注》</ref>朱全忠攻克山南东道和荆南时，冯行袭派儿子[[冯勖|冯勖]]率水师在均州、[[房州|房州]]与其会合。<ref name=HFD15/><ref name=NHFD42/>九月，冯行袭奏昭信军犯朱全忠祖父[[朱信|朱信]]讳。<ref name=BT20-2>[[:s:旧唐书/卷20下|《旧唐书》卷二十下]]</ref>

二年（905年）八月，[[西川节度使|西川节度使]][[王建_(前蜀)|王建]]派养子[[王宗贺|王宗贺]]攻戎昭，九月，屡战屡胜，冯行袭弃金州，逃回均州。其亲校[[全师朗|全师朗]]以金州降王宗贺。<ref>全师朗一名，[[李昊|李昊]]《蜀书·高祖纪》作全行思，<后主纪><[[林思谔|林思谔]]传><[[王宗播|王宗播]]传><[[王承规|王承规]]传>作全行宗，<桑弘志传>又作全行朗，《新唐书·冯行袭传》又作金行全，《十国纪年》作全思朗。此从《十国春秋》等。</ref>王建以全师朗为金州观察使，收为养子，改名[[王宗朗|王宗朗]]。但王宗朗自认守不住金州，烧了城池奔成都投王建。十月，昭信军得赐名戎昭军。十二月，冯行袭复夺金州，<ref>[[:s:新唐书/卷10|《新唐书》卷十]]作王宗贺五月攻金州，六月金州被冯行袭所复。</ref>却发现金州已经残破不堪为治所，得昭宗或控制昭宗的朱全忠许可，将军部迁到均州。<ref name=ZZTJ265>[[:s:資治通鑑/卷265|《资治通鉴》卷二百六十五]]</ref>

三年（906年）五月，朱温认为金州人不拥护长官冯行袭，下令废除戎昭军，<ref name=NBT186/><ref>[[:s:停戎昭军额敕|:s:停戎昭军额敕]]</ref>并入山南东道。<ref name=BT20-2/>冯行袭则以助平山南东道之功调任匡国节度使。<ref name=NHFD42/><ref name=ZZTJ265/>刚到任，就杀了大吏[[张澄|张澄]]并声言其罪，当地人震慑。<ref name=HFD15/>

== 后梁年间 ==
四年（907年），朱全忠篡唐，建立后梁，称太祖皇帝，<ref name=ZZTJ266>[[:s:資治通鑑/卷266|《资治通鉴》卷二百六十六]]</ref>冯行袭效命后梁。五月，冯行袭与[[平卢节度使|平卢节度使]][[韩建|韩建]]、[[义武军节度使|义武军节度使]][[王处直|王处直]]请求太祖开内宴，分别获赐方物。<ref>[[:s:舊五代史/卷3|《旧五代史》卷三]]</ref>梁太祖祭祀天地，冯行袭请求入觐，上贡很多，因此也受到很多恩礼。朱全忠派诏[[翰林学士|翰林学士]][[杜晓|杜晓]]撰德政碑以赐之。<ref>[[:s:匡国节度使冯行袭德政碑|:s:匡国节度使冯行袭德政碑]]</ref>冯行袭累官至兼[[中书令|中书令]]，册拜[[司空|司空]]，<ref name=HFD15/>封长乐郡王。<ref name=NHFD42/><ref>[[:s:舊五代史/卷4|《旧五代史》卷四]]</ref>

冯行袭在匡国军三年，贡军粮二十万石。[[开平_(后梁)|开平]]四年（910年）五月，冯行袭病笃，上表请代。梁太祖害怕冯行袭的手下士兵（其中有故大军阀[[秦宗权|秦宗权]][[蔡州_(隋朝)|蔡州]]旧部两千人）或家人试图夺取和后梁都城[[开封|开封]]很近的匡国军的控制权，派崇政院直学士[[李珽|李珽]]为留后，前往匡国军部[[许州|许州]]控制匡国。李珽到许州后，对将吏们说：“天子握百万兵，距此才数舍；冯公忠纯，别使皇上有所疑。你们赤心奉国，何忧不富贵！”冯行袭想派人代受诏，李珽说：“东首加朝服才是礼。”于是在卧室内宣诏，令冯行袭善自保养，若有不测，子孙都可保后福。冯行袭感泣而谢，遂解二印以授李珽，李珽代掌军府事。当时冯行袭严酷，因被从事魏峻切谏触怒，诬陷魏峻贪赃，下狱欲诛。李珽便审理魏峻冤情，立时释放他，回报太祖，太祖览奏喜道：“我本就知道李珽必能为我办事，<ref>[[:s:新五代史/卷54|《新五代史》卷五十四]]</ref>冯行袭门户不朽了。”<ref>[[:s:舊五代史/卷24|《旧五代史》卷二十四]]</ref>冯行袭过世，辍朝一日，赠[[太傅|太傅]]。谥忠敬。<ref name=HFD15/><ref name=NHFD42/>冯行袭部兵被分隶诸校，冒冯姓者皆还本宗。<ref name=ZZTJ267/>

冯行袭性严烈，为政深刻寡恩，但在治所都有好运，境内曾有蝗灾，但不久有群鸟啄食，没有遭害；百姓乏食时，总有野生稻谷出现在田间。虽威福在己，但他一直竭力奉梁室，故得以保其功名。<ref name=HFD15/><ref name=NHFD42/>

== 家庭 ==
长子冯勖，历任蕲、沁二州刺史。次子[[冯德晏|冯德晏]]，官至金吾将军。
