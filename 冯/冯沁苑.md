'''冯沁苑'''（{{bd|1997年|8月28日||}}），网名'''买辣椒也用券'''，中国大陆流行音乐女歌手，黑蕉乐队的主唱，同时也是[[北京师范大学-香港浸会大学联合国际学院|北京师范大学-香港浸会大学联合国际学院]]在校学生。因改编翻唱日本歌曲《吃醋》（日语：ヤキモチ）的中文版本《起风了》而备受关注。<ref name=A>{{cite news |title=回到家乡唱自己喜欢的歌，是件有意义的事 |url=http://www.zstravel.cn/youji/bendi/2018/0905/7590.html |accessdate=2019-03-09 |work=舟山日报 |date=2018-09-05}}</ref><ref name=B>{{cite news |title=舟山姑娘冯沁苑就是大名鼎鼎的“买辣椒也用券” |url=http://www.zstravel.cn/quanwei/2018/0906/7592.html |accessdate=2019-03-09 |work=舟山晚报 |date=2018-09-06}}</ref>

== 生平 ==
=== 早年生活 ===
冯沁苑出生于[[浙江省|浙江省]][[舟山市|舟山市]]，小学就读于沈家门小学，中学先后就读于定海二中、舟山中学。她从小爱好唱歌，在高中时期担任班级文娱委员，也曾做过学生主持人。<ref name=A/>其间，她在读高一时参加了“舟山中学校园十佳歌手比赛”，凭借一首《想你的夜》获得最终的第二名。<ref name=B/><ref name=C>{{cite web |author1=舟山广电 |title=牛！舟山姑娘的歌上了湖南卫视《歌手》，她已有50多万粉丝 |url=https://www.sohu.com/a/293191085_443953 |website=搜狐 |accessdate=2019-03-09 |date=2019-02-03}}</ref>

=== 演艺事业 ===
2016年初，正在[[北京师范大学-香港浸会大学联合国际学院|北京师范大学-香港浸会大学联合国际学院]]读大一的冯沁苑应节目组之邀，参加了第一季《[[中国新歌声_(第一季)|中国新歌声]]》的海选。试唱歌曲期间，她第一次接触了专业的音乐培训。<ref name=A/>之后，她顺利通过海选，也登台参加盲选环节。但因过于紧张而导致发挥欠佳，没有导师为她冲下来{{NoteTag|冯沁苑在接受采访时将其误作“转身”；}}；节目播出时，她的演唱片段也遭到删减。<ref name=B/>

2017年2月2日，冯沁苑在[[网易云音乐|网易云音乐]]发表个人单曲《起风了》。这首歌曲的原版是由日本男歌手[[高桥优_(歌手)|高桥优]]演唱的《ヤキモチ》{{NoteTag|[[高桥优_(歌手)|高桥优]]亦是这首歌曲的作曲人；}}，为日本电视剧《[[深夜食堂3|深夜食堂3]]》的主题曲；大一时期，她和好友米果在一次偶然的机会选中这首歌并进行改编，米果负责填中文词，她负责演唱。歌曲在发表一段时间后，获得了上亿播放量以及40多万评论量，冯沁苑的知名度也因此得到提升。<ref name=A/>

2018年8月底，她带上在大学期间组建的黑蕉乐队以及在《中国新歌声》舞台上认识的好友裘德，赴[[舟山市|舟山]][[朱家尖岛|朱家尖]]参加《[[东海音乐节|东海音乐节]]》，并献唱成名曲《起风了》。<ref name=A/><ref name=C/><ref>{{cite news |title=2018东海音乐节 青年文化在此生长 |url=https://zj.zjol.com.cn/news/1021587.html |accessdate=2019-03-09 |work=浙江新闻 |date=2018-09-02}}</ref>12月3日，发表个人单曲《起风了》的新版本。该版本是她在获得版权方授权之后，受版权方邀请，赴日本录音室完成录制，并且配有歌曲MV。<ref>{{cite web |title=UICer里的音乐人 冯沁苑：一面向前，一面把风装进胸膛 |url=https://uic.edu.hk/cn/home/news/6677-2019-01-14-01-52-07 |website=北京师范大学-香港浸会大学联合国际学院 |accessdate=2019-03-09}}</ref>

== 争议 ==
由于混淆填词翻唱和翻唱的性质，冯沁苑有部分填词翻唱作品{{NoteTag|包括《起风了》在内。}}在未获授权的情况下公开发表，从而引发版权问题。2018年5月5日，冯沁苑通过微博发表声明，决定下架所有填词翻唱作品，并开始争取版权<ref>{{cite web |title=【关于下架填词翻唱作品】 |url=https://m.weibo.cn/2472169050/4236456224692361 |website=新浪微博 |accessdate=2019-03-09 |date=2018-05-05}}</ref>；在经纪公司的协助之下，她的成名曲《起风了》于同年8月31日获得了版权方的授权。<ref>{{cite web |title=【关于《起风了》获得《ヤキモチ》版权方授权以及相关事宜的说明】 |url=https://m.weibo.cn/2472169050/4279016398180642 |website=新浪微博 |accessdate=2019-03-09 |date=2018-08-31}}</ref>

== 影响 ==
冯沁苑演唱的中文版《起风了》在网络上走红之后，[[林俊杰|林俊杰]]、[[吴青峰|吴青峰]]等歌手也相继翻唱了这首歌曲。其中，吴青峰的翻唱版本亦是电视剧《[[加油，你是最棒的|加油，你是最棒的]]》的主题曲。<ref>{{cite news |title=吴青峰超强热单《起风了》 上线网易云音乐19天播放量破亿 |url=http://www.chinanews.com/wap/detail/zw/yl/2019/01-31/8744072.shtml |accessdate=2019-03-09 |work=中国新闻网 |date=2019-01-31}}</ref><ref>{{cite news |title=吳青峰首曝《歌手》主持內幕「抖到看不清手卡」　翻唱網路神曲網跪了！ |url=https://star.ettoday.net/news/1366153 |accessdate=2019-03-09 |work=ETtoday新聞雲 |date=2019-01-26}}</ref>
