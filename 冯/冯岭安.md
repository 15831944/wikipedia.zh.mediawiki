'''冯岭安'''（{{bd|1925年||2002年|11月2日}}）[[河南省|河南省]][[郏县|郏县]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中共中央办公厅|中共中央办公厅]]副主任、[[中共中央直属机关委员会|中共中央直属机关委员会]]书记<ref name=hzw/>。

==生平==
1948年自[[西北大学_(中国)|西北大学]]中文系毕业。1949年1月参加中国共产党领导下的革命工作，同年12月加入中国共产党。1949年歷任[[延安大学|延安大学]]政策研究室组长、[[西北大学_(中国)|西北人民革命大学]]校刊编辑主任。[[中华人民共和国|中华人民共和国]]成立后，历任[[西北行政委员会|西北行政委员会]]办公厅秘书，[[国务院办公厅|国务院秘书厅]]秘书，[[国务院机关事务管理局|国务院机关事务管理局]]党组秘书。1973年后，历任[[国家体委|国家体委]]党组秘书，国家体委[[台湾|台湾]]体育联络处负责人，[[中共中央直属机关事务管理局|中直管理局]]副局长、局长兼[[中共中央办公厅|中共中央办公厅]]特别会计室主任，中共中央办公厅副主任，[[中共中央直属机关委员会|中央直属机关党委]]委员、常委、书记，[[宋庆龄基金会|宋庆龄基金会]]副主席、党组副书记，并兼[[中国国际友谊促进会|中国国际友谊促进会]]副理事长、领导小组副组长等职。是[[中共十三大|中共十三大]]、[[中共十四大|中共十四大]]代表<ref name=hzw>{{Cite web|url =http://www.hangzhou.com.cn/20020127/ca169352.htm|title =冯岭安同志逝世|author =|date =2002-11-05|publisher =杭州网|language =zh-cn|accessdate =|quote =|archiveurl =https://archive.today/20190316050439/http://www.hangzhou.com.cn/20020127/ca169352.htm|archivedate =2019-03-16|dead-url =no}}</ref><ref>{{Cite web |url = http://www.huaxia.com/lasd/hxrwk/ddrw/dl/2003/08/488488.html |title = 冯岭安 |author =  |date = 2003-08-21 |publisher = 华夏经纬网 |language = zh-cn |accessdate =  |quote =  |archiveurl = https://archive.today/20190316050658/http://www.huaxia.com/lasd/hxrwk/ddrw/dl/2003/08/488488.html |archivedate = 2019-03-16 |dead-url = no }}</ref>。

2002年11月2日，冯岭安在[[北京|北京]]病逝，享年78岁<ref name=hzw/>。
