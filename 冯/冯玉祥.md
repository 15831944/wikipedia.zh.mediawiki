'''冯玉祥'''（{{bd|1882年|11月6日|1948年|9月1日|catIdx=F}}），[[官章名|官章名]]'''玉祥'''，[[譜名|譜名]]'''基善'''，[[表字|表字]]'''焕章'''，[[安徽省|安徽]][[巢湖市|巢县]]（今[[巢湖市|巢湖市]][[夏阁镇|夏阁镇]]竹柯村）人，生長於[[直隸省|直隸省]][[保定府|保定府]]（今[[河北省|河北省]][[保定市|保定市]]）。[[民国军阀|民国军阀]]，本屬[[直系|直系]][[軍閥|軍閥]]，[[第二次直奉戰爭|第二次直奉戰爭]]中倒戈，改所部為國民軍，後敗退西北，自成勢力<ref name="學良口述">[[唐德剛|唐德剛]]著、[[張學良|張學良]]口述：《張學良口述歷史》，[[台北|台北]]：[[遠流出版|遠流出版]]，2009年3月1日初版</ref>{{rp|198}}。[[西北军|西北军]]领袖。1935年，晉任[[国民革命军|国民革命军]][[陆军|陆军]][[一级上将|一级上将]]<ref>{{cite book |quote=甲、中央政治會議決議任蔣中正為特級上將，閻錫山、馮玉祥、張學良、何應欽、李宗仁、朱培德、唐生智、陳濟棠為一級上將。 |editor=[[郭廷以|郭廷以]] |title=《中華民國史事日誌·中華民國二十四年三月二十七日》 |location=台北 |publisher=[[中央研究院|中央研究院]]近代史研究所 |year=1979 }}</ref>。馮玉祥軍旅一生，擅长见风使舵，八次临阵倒戈。故民間有“倒戈將軍”之稱。

== 生平 ==
=== 家庭出身 ===
冯玉祥的[[父亲|父亲]]冯有茂，兄弟四人，冯有茂排行第二。1845年生于安徽巢县竹柯村，家庭赤贫，冯有茂做了一个泥瓦匠，冯玉祥的伯父与三叔做了裁缝，冯玉祥的四叔是[[佃农|佃农]]。[[太平軍|太平軍]]攻到安徽时，冯玉祥的父亲外出逃荒，投靠一家地主做[[佣工|佣工]]，陪着地主家的少爷学[[武學|武學]]，考武举，结果冯玉祥的父亲考中了武庠（官办武学）[[生員|生員]]，毕业于投身[[淮军|淮军]]将领[[刘铭传|刘铭传]]的“铭军”，在差遣队当差，后来慢慢地升到哨长和哨官，参加了平定[[陕甘回乱|陕甘回乱]]，之后随[[左宗棠|左宗棠]]进军[[新疆|新疆]]。随军从新疆调防内地进驻[[山东|山东]][[济宁|济宁]]。

冯有茂在济宁与当地妇女游氏结婚，次年生长子冯基道，（后来一共生了七个儿子，其中冯玉祥是次子。）长子出世次年，铭军解散。[[文盲|文盲]]的冯有茂南下返回老家，考武[[举人|举人]]未成，生活无着，不得已，又重新入伍，随淮军至[[直隶|直隶]][[青县|青县]]兴集镇驻防。[[光绪|光绪]]八年（1882年）9月26日，生下了次子冯玉祥，当时按[[族谱|族谱]]起名为冯基善。这时[[李鸿章|李鸿章]]為[[直隶总督|直隶总督]]，淮军在保定府“五营练军”，冯有茂攜家到保定府，因此保定成了冯玉祥儿童时代的养育之地，第二故乡。冯玉祥一生都是保定府口音。冯玉祥的父母早年都染有[[鸦片|鸦片]]的嗜好。这在清末，已成为一种最普遍的风气，尤其是军政界，简直无人不吸。当时因为生活艰难，营养不足，冯有茂的七个儿子死了五个。

[[光绪|光绪]]十七年（1891年）农历九月，冯玉祥的哥哥补上一份[[绿营|绿营]]的[[马兵|马兵]]的空缺从军，冯玉祥接替其哥哥的位子进入村中的私塾读书，因為本来冯家只打算栽培长子读书而已。[[啟蒙|啟蒙]]之后，馮玉祥练习写字，家贫买不起纸與[[毛笔|毛笔]]，于是就用一根细竹管，顶端扎上一束麻，蘸着稀薄的黄泥，在方砖上练习写字。冯有茂千方百计想把次子也补上一个兵额，以缓解家庭经济窘迫。

[[光绪|光绪]]十八年（1892年），营中有一个士兵的缺额，管帶苗某决定补给冯有茂的次子，当时怕耽误了時刻，被别人抢去缺额，来不及问冯有茂次子的名字，就给直接编造了一个名字“冯玉祥”登记入册。從此，冯基善就更名為「冯玉祥」，作為[[官章名|官章名]]，11岁，不用随营操练，发饷时到营中应名领饷外（步兵，每月3[[两|两]]6[[钱|钱]][[银子|银子]]），其余时间仍在家中过活。这在[[保定府|保定府]]的[[淮军|淮军]]，叫做“恩饷”。

12岁时，结束了1年零3个月的[[私塾|私塾]]教育，入保定练军营拿枪操练。后来冯玉祥的父亲因伤丢了差事，断了经济来源。冯玉祥为了多挣一点[[銀兩|銀兩]]，就十年如一日在教场练喊队列口令想成为「传令教习」，结果练出一副惊人的大嗓门。

===民初仕途===
[[File:Feng_Yuxiang4.jpg|thumb]]

冯玉祥1896年加入[[淮軍|淮軍]]，1902年改投[[袁世凯|袁世凯]]的武卫右军。1911年担任由武卫右军改编而成的新编[[陆军第二十镇|陆军第二十镇]]第八十标第三营[[管带|管带]]，曾參與[[滦州起义|滦州起义]]。1912年1月3日，冯玉祥担任北方革命军政府总参谋长。

1914年7月，冯玉祥任陆军第七师第十四旅[[旅长|旅长]]，参加镇压[[白朗_(农民起义领袖)|白朗]]起义。9月任陆军第十六混成旅旅长。1915年[[袁世凱稱帝|袁世凱稱帝]]，奉令率部入川与讨袁的[[蔡锷|蔡锷]]所部[[护国军|护国军]]作战，曾一度击败蔡锷，获袁世凯颁賜[[爵位|爵位]]，為三等[[男爵|男爵]]，但馮卻暗中与蔡锷联络，1915年夏冯玉祥策动四川都督[[陈宦|陈宦]]宣布独立讨袁。1916年3月议和停战。1917年4月被免去第十六混成旅旅长职，7月率旧部响应段祺瑞之號召，讨伐[[张勋|张勋]]的[[辮軍復辟|辮軍復辟]]，复任第十六混成旅旅长。1917年11月，[[孙中山|孙中山]]举起“护法”大旗[[护法运动|护法运动]]，段祺瑞调冯率第十六混成旅南下攻打孙中山、[[陆荣廷|陆荣廷]]的[[护法军|护法军]]。1918年2月14日，冯在[[武穴|武穴]]通电发表主和，主张恢复国会，投向直系[[曹锟|曹锟]]，被免职留任。1918年3月下旬，冯率部驻[[常德|常德]]期间，因為篤信[[基督教|基督教]]，在军中设置[[教會|教會]]，请[[隨軍牧師|隨軍牧師]]向全体官兵[[布道|布道]]、[[探題|探題]]，并为100多名官兵施行[[洗礼|洗礼]]。1918年6月被撤销免职处分。6月14日段祺瑞的心腹[[徐樹錚|徐樹錚]]杀死冯玉祥夫人的姑丈、主张[[南北议和|南北议和]]、支持直系军阀的北洋元老[[陆建章|陆建章]]。11月段祺瑞为了安抚冯玉祥任命他为湘西[[镇守使|镇守使]]。1925年冯玉祥杀死徐樹錚为陆报仇。

===馮玉祥建軍西北===
[[File:Feng_Yuxiang2.jpg|thumb]]》]]

1921年8月任陆军第十一师师长，从属[[直系军阀|直系军阀]]，率其部隊入[[陝西|陝西]]，在陕西[[督军|督军]][[阎相文|阎相文]][[自杀|自杀]]之后，接任陕西督军，並以此地為地盤扩充，受到[[蘇聯|蘇聯]]大力支持壯大，其軍隊因此被稱為“[[西北軍|西北軍]]”。1922年[[第一次直奉战争|第一次直奉战争]]中，出潼关，击败[[河南|河南]]督军[[赵倜|赵倜]]，任河南[[督军|督军]]，因妄杀降将[[宝德全|宝德全]]，被[[吴佩孚|吴佩孚]]撤去河南督军，赴京面見[[曹錕|曹錕]]，曹改任其為陸軍檢閱使。

1924年9月18日[[第二次直奉战争|第二次直奉战争]]爆发，吴佩孚任命冯玉祥为第三军总司令。冯在[[古北口|古北口]]与奉方代表达成协议，接受奉军[[张作霖|张作霖]]50万[[銀元|銀元]]的[[賄賂|賄賂]]<ref>见《唐德刚：张学良口述历史》第五分</ref>，率军返回[[北京|北京]]，发动[[北京政变|北京政变]]，推翻北洋政府，召集部下倒戈改称[[国民军|国民军]]。9月20日清晨，[[鹿钟麟|鹿钟麟]]指挥部队控制了北京全城，监视[[总统|总统]][[曹锟|曹锟]]，逼其弟[[曹銳|曹銳]][[自殺|自殺]]，驱逐[[溥仪|溥仪]]出紫禁城。電請[[孫文|孫文]]北上。导致[[山海关|山海关]]一路的[[吴佩孚|吴佩孚]]失败。11月10日，冯玉祥、张作霖分别到天津推举[[段祺瑞|段祺瑞]]就任[[中华民国临时执政|中华民国临时执政]]。 

1925年1月，冯受命为西北边防督办，率6个师15万人赴张家口，结识了[[李大钊|李大钊]]及苏联驻华大使[[加拉罕|加拉罕]]，并邀请苏联军事顾问进入西北军。1925年底馮玉祥参加[[反奉战争|反奉战争]]，1926年1月1日兵败后通电下野，赴包头。4月9日国民军再度发动政变，逼走段祺瑞，释放曹锟，但随后因军事上失利，不得不撤出北京，在[[南口|南口]]坚守。1926年5月10日，冯赴苏联考察，期间在蒙古[[库伦|库伦]]宣布参加国民党，要为国民革命而战。在苏联期间，冯玉祥与苏联外交人民委员会委员长[[齐切林|齐切林]]、苏联领导人[[加里宁|加里宁]]、阿里科夫、[[伏罗希洛夫|伏罗希洛夫]]、[[卢那察尔斯基|卢那察尔斯基]]、苏联教育委员会副委员长、[[列宁|列宁]]夫人[[克鲁普斯卡娅|克鲁普斯卡娅]]和莫斯科中山大学校长[[拉狄克|拉狄克]]等会面。冯玉祥多次提出要与[[斯大林|斯大林]]会面但被婉拒。1926年8月回國<ref>{{cite web |title=冯玉祥访问苏联 |url=http://hongse.haiwainet.cn/n/2017/1207/c3543031-31199512.html |work=人民政协报 |publisher=海外网 |date=2017-12-07 |accessdate=2018-3-13 |archive-url=https://web.archive.org/web/20180313160645/http://hongse.haiwainet.cn/n/2017/1207/c3543031-31199512.html |archive-date=2018-03-13 |dead-url=yes }}</ref>。国民军在南口坚守四个月后，于8月退回西北。9月16日，冯到达五原后发表了[[五原誓师|五原誓师]]宣言。17日，国民军各部将领公举冯玉祥为国民联军总司令。冯宣布国民军全体将士集体参加[[中国国民党|中国国民党]]，参加[[国民革命|国民革命]]。冯玉祥被委任为广东国民政府委员和军事委员会委员，策应北伐。在击溃[[刘镇华|刘镇华]]的镇嵩军之后，率領西北軍东出[[潼關|潼關]]参加[[國民革命軍北伐|國民革命軍北伐]]，為國民革命軍指揮官之一。

[[File:1927年冯玉祥与蒋介石会面.jpg|150px]]

1927年4月[[宁汉分裂|宁汉分裂]]之后，先在[[郑州|郑州]]与[[汪精卫|汪精卫]]会面，后在[[徐州|徐州]]与[[蒋中正|蒋中正]]会面，最后選擇與身在南京的蒋中正合作，應蒋中正要求，达成协议“分共、清党”。冯将他军队中的共产党员和他管辖地区内的共产党员干部全部“礼送”出境。

===中原毀佛===
1927年，篤信[[基督教|基督教]]的冯玉祥在河南废寺逐僧，將[[大相国寺|大相国寺]]改成[[市場|市場]]。并发动全省毁佛运动，所有[[比丘|比丘]]、[[比丘尼|比丘尼]]一律驱逐。所有寺产[[沒收|沒收]]，寺院改为[[学校|学校]]，或作[[救济院|救济院]]、[[图书馆|图书馆]]，或成为娱乐场所。继河南之后，[[直隸|直]]、[[魯|魯]]、[[秦|秦]]等各[[行省|行省]]，也都纷纷跟从，[[華北|華北]][[佛教|佛教]]因此几乎衰绝。

===与国民党中央的关系===
[[File:Feng_Yu-hsiang_TIME_Cover.jpg|150px]]》封面，1928年7月2日]]
1928年4月，北伐全胜的编遣会议之后，蒋介石、冯玉祥之间由合作转向分裂。

1929年4月，馮玉祥不滿國軍編遣會議比例式裁兵原則的決議，稱病離開南京。5月14日於潼關出任“護黨救國軍西北軍”總司令，但迅速被南京方面內外夾攻而失敗，被迫離職前往[[山西|山西]]。1929年5月22日，冯玉祥的部将韩复榘、石友三联合通电拥蒋叛冯开始，经过中原大战，蒋冯阎李张五大集团中，冯玉祥成为唯一输得精光的总司令。

[[File:Feng_Yuxiang.jpg|150px]]

1930年，聯同[[閻錫山|閻錫山]]、[[李宗仁|李宗仁]]等與蔣中正對抗，引发[[中原大戰|中原大戰]]，兵敗後隐居[[山東|山東]][[泰山|泰山]]。1930年11月4日，冯在太原通电下野，隐居读书。<ref>[[陳布雷|陳-{布}-雷]]等，《蔣介石先生年表》，[[台北|台北]]，[[傳記文學|傳記文學]]出版社，1978年6月1日，第21頁</ref>

1931年7月，冯策动旧部石友三进攻东北军，反而被打败。1931年下半年，冯策动甘肃旧部反蒋，被蒋军打败。

1933年5月26日，与[[吉鸿昌|吉鸿昌]]、[[方振武|方振武]]、[[佟麟阁|佟麟阁]]在[[察哈尔|察哈尔]][[张家口|张家口]]建立[[察哈尔民众抗日同盟军|察哈尔民众抗日同盟军]]，任总司令，時任察省主席的部將[[宋哲元|宋哲元]]並不支持，但又不便反對，因而離職，之後馮率軍攻下由親日軍隊佔領的[[多倫|多倫]]，引起一陣騷動。惟因實力不足且北平軍事委員會分會方面也派遣凃思宗及徐思賢先生，策反馮系將領與給予番號，因此，馮玉祥忖於大勢已去，遂電請北平軍分會派員整編，願意將抗日同盟軍旗幟放下，1933年6月9日北平軍分會代表[[涂思宗|涂思宗]]及徐思賢抵張家口，馮玉祥親迎並請自圖書館由涂將軍擔任主席舉行整編會議，商討善後事宜後返魯。8月7日，冯被迫宣布撤销抗日同盟军总部，辞去总司令职，于8月17日重返泰山。

1933年11月20日，李济深等在福建组成[[中华共和国人民革命政府|中华共和国人民革命政府]]，冯派代表余心清参加，并被选为经济委员会主席，然后积极联络韩复榘、宋哲元呼应反蒋。

[[中國抗日戰爭|中國抗日戰爭]]时任国防最高委员会常委，[[第三戰區|第三戰區]]、[[第六戰區|第六戰區]]司令长官，不久被[[蒋中正|蒋中正]]撤职。1946年4月7日赴[[美国|美国]]考察水利，並發表反蔣言論。1947年6月8日，馮玉祥在美國發表聲明，警告蔣介石政府官員應對國內逮捕、鎮壓學生之暴行負責<ref name="民" >{{cite book |editors=[[李新_(1918年)|李新]]總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編 |title=《中華民國史大事記》 |location=北京 |publisher=[[中華書局|中華書局]] |year=2011 |isbn=9787101079982 }}</ref>{{rp|8369}}。9月7日，馮玉祥在美國群眾集會上發表演說，希望美國人民及政府幫助中國人民爭取和平、民主與自由，而不可幫助中國國民黨內戰獨裁<ref name="民" />{{rp|8407}}。9月9日，馮玉祥在美國發起組織「華僑和平民主協會」<ref name="民"/>{{rp|8409}}。1947年11月5日发表《我为什么与蒋介石决裂》一文说“蒋介石政权是中国所有腐败政府的顶峰，外国的金钱是无法使它免于垮台的”。1948年参加[[中国国民党革命委员会|中国国民党革命委员会]]，任中央政治委员会主任。最终与以[[蔣中正|蔣中正]]为[[總裁|總裁]]及[[總統|總統]]的[[中國國民黨中央委員會|中國國民黨中央委員會]]及[[中华民国政府|中华民国政府]]彻底决裂。

== 逝世 ==
[[File:Feng_yuxiang_tomb_2010_2010_06_04.jpg|thumb]][[泰山|泰山]]脚下的[[冯玉祥墓|冯玉祥墓]]]]

1948年接受[[中國共產黨|中國共產黨]]的邀請，搭[[苏联|苏联]]轮船“胜利”号，由美返国参加[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]，9月1日，中途因轮船出事於[[黑海|黑海]]遇难。“胜利”号正向[[敖德萨|敖德萨]]港口进发时，船上失火，冯玉祥被烟熏窒息致死，與女儿[[冯晓达|冯晓达]]一起罹难，冯玉祥时年65歲。也有人認為他是死於謀殺。<ref name="Sheridan">{{cite book|author=James E. Sheridan|title=''Chinese warlord: the career of Feng Yu-hsiang''| url=http://books.google.com/books?id=te1nnQEACAAJ |date=June 1966 | isbn=978-0804701457 | publisher=[[Stanford_University_Press|Stanford University Press]] | page=281}}</ref>[[冯玉祥墓|其墓]]位于泰山西麓。

== 评价 ==
馮玉祥曾多次謀害政敵，[[張建功|張建功]]、[[郭堅|郭堅]]、[[馬廷勷|馬廷勷]]、[[徐樹錚|徐樹錚]]等人均為其所[[暗殺|暗殺]]。

1917年馮氏[[皈依|皈依]][[基督教|基督教]]并[[受洗|受洗]]，还利用宗教力量來控制軍隊，故有“基督将军”的称号。

1945年8月，毛澤東在重慶說：“煥章先生的豐功偉績，已舉世盡知，尤其在抗日戰争期間，你為反對投降、堅持抗戰，呼籲團結、反對分裂作出了不懈的努力。還望煥章先生為實現祖國和平、民主、團結而努力，不負國人所望。”<ref>{{cite web |title=“置身民主，功在國家” 毛澤東點評馮玉祥|url=http://news.takungpao.com/history/dongjian/2014-06/2555861_7.html |work=光明網 |publisher=大公網 |date=2014-06-23}}</ref>1953年，时任[[中央人民政府政务院|中央人民政府政务院]]总理的[[周恩来|周恩来]]在冯玉祥[[骨灰|骨灰]]安放仪式上评价：“冯玉祥将军是一位从旧军人转变而成的坚定的民主主义战士；虽然和所有的历史人物一样，由于政治视野的局限，在他身上不可避免地存在这样那样的缺陷，但是，瑕不掩瑜，冯玉祥将军为中国民主事业的贡献，将是永垂不朽的。”<ref>{{cite web|title=毛澤東評國民黨著名将領：從李宗仁到馮玉祥 |url = http://news.takungpao.com.hk/history/dongjian/2014-06/2555861.html|work=光明網 |publisher=大公網 |date=2014-06-23}}</ref>

1959年的[[庐山会议|庐山会议]]上，[[林彪|林彪]]称[[彭德怀|彭德怀]]“是野心家、阴谋家、伪君子、是冯玉祥式的人物。”<ref>见《北戴河往事追踪报告》：中央文献出版社 </ref>随后毛泽东对“冯玉祥式的人物”进行解释，对彭德怀说：“人们只看到你简单、坦率、心直口快，初交只看到这一面。久了，就从现象看本质。弯弯曲曲，内心深处不见人。人们（林彪）说你是伪君子，像冯玉祥。真伪有矛盾。不能说全部假，对敌斗争是真的。心中很严重的东西不拿出来。”[[刘少奇|刘少奇]]以“[[魏延|魏延]]的骨头、[[朱可夫|朱可夫]]的党性、冯玉祥的作风...一个一贯反党的伪君子，企图搞军事政变！”批判彭德怀<ref>{{cite web |title=毛泽东为什么看不起民国重要人物冯玉祥？ |url=http://www.sohu.com/a/282351227_621265 |website=搜狐历史 |accessdate=2019-03-18 |archivedate=2018-12-17}}</ref>。

1982年[[乌兰夫|乌兰夫]]（次年担任[[中华人民共和国副主席|中华人民共和国副主席]]）在冯玉祥诞辰100周年纪念大会上评价：“冯玉祥将军是一位杰出的爱国主义者，可敬的民主斗士，著名的军事家和政治家，中国共产党的真挚朋友。”

== 纪念 ==
1926年，北洋军阀[[刘镇华|刘镇华]]包围西安8个月之久，使西安人民饿死4万多人，直到冯玉祥将军率国民军击败刘镇华后西安才得以解围。1928年，陕西省政府主席[[宋哲元|宋哲元]]，于西安城墙[[安定门|西门]]北侧开辟城门，砌单券洞，为纪念[[冯玉祥|冯玉祥]]的功绩，取名“玉祥门”。1934年改称“中正门”，1935年改称“西北门”，后复“玉祥门”。<ref>《古朴遗风——西安碑林与城墙》，吉林出版集团有限责任公司; 第1版 (2011年12月1日)</ref><ref>{{cite book |author=西安市地方志办公室 |date=2012年11月第1版| edition= |title=民国西安词典 |url=http://xadfz.xa.gov.cn/difangzhinew/muluFrame.jsp?bookname=minguo_xian_cidian |location=西安 |publisher=陕西人民出版社 |page=58-59 |isbn=978-7-224-10402-8 |author-link= }}</ref><ref>{{cite web |url=https://mp.weixin.qq.com/s?__biz=MzI0Njc3MDg5NQ==&mid=2247484547&idx=1&sn=117d4da9c0469511042c790db24c5bcd |format= |title= 古城西安十八门 |author=碧海风  |date= 2019-01-31|website= |publisher= 碧海风云|access-date= |quote=}}</ref>
