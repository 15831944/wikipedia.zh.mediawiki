'''冯梦龙'''（{{bd|1574年||1646年|catIdx=F冯}}），[[明代|明代]][[文學家|文學家]]、[[戲曲家|戲曲家]]。字'''犹龙'''，又字'''公鱼'''、'''子犹'''、'''耳犹'''，号龙'''子犹'''、'''墨憨斋主人'''、'''吴下词奴'''、'''姑苏词奴'''、'''前周柱史'''、'''顾曲散人'''、'''绿天馆主人'''等。<ref name=" uisa" /><ref name="rmws " /><ref name="Hxsw " /><ref name="jsx " /> [[南直隶|南直隶]][[苏州府|苏州府]][[长洲县|长洲县]]（今[[江蘇省|江蘇省]][[蘇州市|蘇州市]]）人，同其兄画家[[冯梦桂|冯梦桂]]、其弟诗人[[冯梦熊|冯梦熊]]并称“吴下三冯”。<ref name=" uisa" /><ref name="rmws " /><ref name="Hxsw " /><ref name="jsx " >{{cite web |url=http://www.suzhou.gov.cn/bmdw/swgxj/ggfw_694/bmfw_695/whjy/whmr/201112/t20111202_83879.shtml |title=冯梦龙 |accessdate= 2008年|work= |publisher=江苏网 |language =zh-cn}}</ref> 他的作品比较强调[[感情|感情]]和行为，最有名的作品为《[[喻世明言|喻世明言]]》（古今小说）、《[[警世通言|警世通言]]》、《[[醒世恒言|醒世恒言]]》，合稱“[[三言|三言]]”，[[三言|三言]]与[[凌濛初|凌濛初]]的《[[初刻拍案惊奇|初刻拍案惊奇]]》、《[[二刻拍案惊奇|二刻拍案惊奇]]》合称“三言二拍”，是中国白话短篇小说的经典代表。<ref name="Hxsw " >{{cite web |url=http://cathay.ce.cn/person/200808/28/t20080828_16647030.shtml |title=冯梦龙：挽救民间通俗文学之贡献可媲美孔子 |accessdate= 2008年|work= |publisher=华夏文化网 |language =zh-cn}}</ref>

== 生平 ==
馮夢龍少時有才情，“酷愛李氏（[[李贄|李卓吾]]）之學，奉為蓍蔡”，博學多聞，廣為同輩所欽服。為人曠達，不拘一格。曾與[[文震孟|文震孟]]、[[姚希孟|姚希孟]]、[[錢謙益|錢謙益]]等名人結社作文。屢考科舉不中，落魄奔走，以坐館教書為生。

[[天启_(明朝)|天啟]]六年（1626年），[[魏忠賢|魏忠賢]][[閹黨|閹黨]]逮捕[[周順昌|周順昌]]，馮夢龍也在被迫害之列。馮夢龍發憤著書，完成編纂《[[喻世明言|喻世明言]]》、《[[警世通言|警世通言]]》、《[[醒世恆言|醒世恆言]]》。“三言”各四十篇，共一百二十篇，多為[[宋朝|宋]][[元朝|元]][[明|明]][[話本|話本]]中藝術佳作，歷來被讀者稱譽。同時期的小說家[[凌濛初|凌濛初]]在其《[[拍案驚奇|拍案驚奇]]·序》中指出：{{blockquote|「獨龍子猶氏所輯（喻世）等諸言，頗存雅道，時著良規，一破今時陋習。而宋元舊種，亦被搜括殆盡。肆中人見其行世頗捷，意余當別有秘本，圖出而衡之。不知一二遺者，皆其溝中之斷，蕪略不足陳已。」}}

晚年馮夢龍仍孜孜不倦，繼續創作小说和整理[[戲曲|戲曲]]研究。又曾從事《[[古今譚概|古今譚概]]》、《[[太平廣記鈔|太平廣記鈔]]》、《[[智囊補|智囊]]》（重刊時改名《智囊補》）、《[[情史|情史]]》、《[[太霞新奏|太霞新奏]]》等的評纂工作。

崇禎三年（1630年）補為[[貢生|貢生]]，年已五十七歲，次年破例授[[丹徒縣|丹徒縣]][[訓導|訓導]]，七年至十一年（西元1634年－1638年）任[[壽寧|壽寧]]知縣，當時[[福建|福建]][[壽寧|壽寧]]盛行溺殺女嬰的陋俗，他親自起草〈[[s:禁溺女告示|禁溺女告示]]〉發布，並嚴加禁止。此告示收錄於《[[壽寧待志|壽寧待志]]》：{{blockquote|“一般十月怀胎，吃尽辛苦，不论男女，总是骨血，何忍淹弃。为父者你自想，若不收女，你妻从何而来？为母者你自想，若不收女，你身从何而活？況且生男未必孝順，生女未必忤逆……”}}

崇禎十七年（1644年），[[甲申之變|甲申之變]]，[[李自成|李自成]]破[[北京|北京]]，[[崇禎帝|崇禎帝]][[自縊|自縊]][[煤山|煤山]]。馮夢龍悲痛欲絕，他懷著中興希望編撰《[[甲申紀事|甲申紀事]]》。後清兵南下，他努力宣傳抗[[清|清]]，刊行《[[中興偉略|中興偉略]]》。清[[順治|順治]]三年（1646年）春，夢龍憂憤而死，一說被清兵所殺。

== 文学思想 ==
冯梦龙的文学成就主要体现在通俗文学上，他的作品反映了明朝市井黎民的思想感情和道德观念，同时也反映了明朝末年资本主义萌芽时期反抗封建礼教和追求个性解放的色彩。<ref name="jsx " />

冯梦龙的作品对于女性饱含同情和赞美，有早期女性解放意识，如：杜十娘、赵京娘、白娘子、周胜仙、璩秀秀、张淑儿，这些他作品中女性都闪烁着人性的光芒，表现了对女性的理解和尊重。<ref name=" uisa" >{{cite web |url=http://www.chinadaily.com.cn/hqpl/yssp/2011-12-29/content_4820727_2.html |title= "娱乐大众":冯梦龙的另类才情|accessdate= 2011年|work= |publisher=中国日报网 |language =zh-cn}}</ref>
