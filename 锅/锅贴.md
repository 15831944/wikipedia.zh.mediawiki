'''锅贴'''用少量油水煎熟包有餡細長型兩端開口的麵食，類似餃子。部分地區指生[[煎|煎]]的[[饺子|饺子]]，稱作'''煎餃'''；或指北方農村在大鐵鍋中以雜糧麵發酵後做成的餅。通常的做法是将锅贴排放到[[平底锅|平底锅]]内，锅底涂一些[[食用油|食用油]]以防粘底，在火上浇入水，有时加入一些[[醋|醋]]，盖盖[[焖|焖]]熟。

== 各地鍋貼 ==
認為鍋貼與煎餃不同的地區認為兩者除了包法不同，鍋貼不似煎餃需蒸熟後用油煎，而是直接生煎，且現多使用平底鍋。鍋貼亦指一種料理手法，如鍋貼豆腐、鍋貼乳餅、鍋貼里脊、鍋貼鯽魚片、鍋貼金錢野雞、鍋貼百花雞、鍋貼黃魚、、等，是比煎放稍多油的一種方法，起鍋後食材多半呈長塊狀，與油煎兩面的「𤌙」（音他）有所不同，如鍋𤌙豆腐，煎、貼、燒等易混淆。

锅贴的形状各地不同，[[天津|天津]]锅贴类似[[褡裢火烧|褡裢火烧]]。因梅蘭芳而知名的青島十樂坊鍋貼店與台灣鍋貼相同，一般在秦嶺淮河以南的稻米產區，較多類[[煎餃|煎餃]]的鍋貼。
鍋貼的口味除了一般[[豬肉|豬肉]]外，還有[[韓式泡菜|韓式泡菜]]等其他口味。

== 日本鍋貼 ==
在[[日本|日本]]，所谓的“饺子”指的是煎餃，而日本飲食文化也只會做煎餃，通常和[[日本拉面|日本拉面]]一起经营。煎饺通常是日本境内的中華料理店必备经典菜肴。日本的饺子工厂将饺子用蒸汽蒸熟出售，日本人食用饺子时将买回的熟饺子放入锅中煎一下。

== 起源 ==
鐵鍋盛行於宋代，故鍋貼類菜餚多為河南開封菜系，鍋貼創始則箭垛式的指向宋太祖。

* 第一種說法：[[蒙古|蒙古]]的遊牧民族只帶著少許廚具，於是把生[[水餃|水餃]]拿去煎。 
* 第二種說法：[[慈禧|慈禧]]太后看到餃子不新鮮就會把它丟到皇宮圍牆後，而城外乞丐就會去撿來煎。惟慈禧太后、乾隆皇帝、蘇東坡為三大箭垛式人物，無可靠來源，基本為假說。

== 蘸料 ==
蘸料的通常配方是[[镇江香醋|镇江香醋]]，有人喜欢加入[[辣酱|辣酱]]。海外中餐馆，也提供甜酸酱或酱油。
日本吃法除了蘸[[辣油|辣油]]與[[醬油|醬油]]混和[[柑橘醋|柑橘醋]]調味以外，更有[[白醋|白醋]]加[[胡椒|胡椒]]的吃法受到歡迎。而在东南亚国家，沾醬五花八門，而眾多國家人民習慣以[[辣醬|辣醬]]搭配各種辛香料為佐。
