{{酸鹼}}
{{主條目消歧義列表|可溶於水的鹽基|所有與酸產生中和反應的物質|鹽基|俗称为“碱”或者“纯碱”的化学物质|碳酸钠}}
在各种酸碱理论中，'''碱'''都是指与[[酸|酸]]相对的一类物质。鹼多指[[鹼金屬|鹼金屬]]及[[鹼土金屬|鹼土金屬]]的[[氢氧化物|氢氧化物]]，而对碱最常见的定义是根据[[阿伦尼乌斯|阿伦尼乌斯]]（Arrhenius）提出的酸碱离子理论作出的定义：碱是一种在水溶液中可以[[电离|电离]]出[[氢氧根|氢氧根]][[离子|离子]]并且不产生其它阴离子的化合物。随后这个定义被扩展为提供氢氧根或者吸收氢离子的化合物。

根据不同的酸碱理论，碱有着[[#其他酸碱理论中碱的定义|不同的定义]]。

== 溶液的碱性 ==
在阿伦尼乌斯酸碱离子理论中，碱都是可溶于水的，并且它们的水[[溶液|溶液]]的[[氢离子浓度指数|pH值]]都大于7。碱在溶液里提供氢氧根或者吸收氢离子，造成溶液中的OH<sup>−</sup>比H<sup>+</sup>多，而常温下纯[[水|水]]的pH值为7——即中性时OH<sup>−</sup>的[[浓度|浓度]]为10<sup>−7</sup>mol/L，所以使得pH值上升。

如1mol/L的[[氢氧化钠|氢氧化钠]]溶液，
:<chem>NaOH -> Na+ + OH-</chem>

按上式完全电离，氢氧根浓度也是1mol/L，根据水的离子积得[H<sup>+</sup>]=10<sup>−14</sup>mol/L，所以此时溶液的pH为14。

一些非氢氧化物可以在水中[[水解|水解]]，产生氢氧根而显碱性：
:<chem>Na2CO3 + H2O -> 2Na+ + HCO3- + OH-</chem>
:<chem>NH3 + H2O -> NH4+ + OH-</chem>

但只要是能在水中产生氢氧根的物质，就都能与[[酸|酸]]（尤其是强酸）发生[[中和反应|中和反应]]。以NaOH和HCl的反应为例：
:<chem>NaOH -> Na+ + OH-</chem>
:<chem>HCl -> H+ + Cl-</chem>

然后生成水：
:<chem>H+ + OH- -> H2O</chem>

与Na<sub>2</sub>CO<sub>3</sub>、NH<sub>3</sub>的反应可以仿照写出。

== 其他酸碱理论中碱的定义 ==
=== 酸碱质子理论中碱的定义 ===
在[[酸碱质子理论|酸碱质子理论]]（Brønsted-Lowry理论）中，碱是能接受质子（[[氫離子|氫離子]]）的物质，碱与质子结合后形成的物质叫做碱的共轭酸。例如在以下的反應中，水扮演碱的角色，其共轭酸為 H<sub>3</sub>O<sup>+</sup>。
:<chem>CH3COOH + H2O -> H3O+ + CH3COO-</chem>

酸碱质子理论可以延伸到非水溶液的情形。

=== 酸碱电子理论中碱的定义 ===
在[[酸碱电子理论|酸碱电子理论]]（Lewis理论）中，碱是能给出孤对电子的物质，对于一切有空轨道的物质，碱都可与之进行[[加合反应|加合反应]]。这个定义的范围是最广的。

酸碱电子理论的鹼又稱為[[路易士鹼|路易士鹼]]，如以下的例子中，氯離子提供孤对电子对，即為路易士鹼。
:<chem>AlCl3 + Cl- -> AlCl4-</chem>

== 碱的性质 ==
* 使酸碱指示剂变色：鹼（可溶）使紅色[[石蕊|石蕊]]试液变蓝，无色[[酚酞|酚酞]]变红
* 和[[酸|酸]]发生中和反应：碱 + 酸 → 盐 + 水

::<chem>NaOH + HCl -> H2O + NaCl</chem>

* 和[[盐|盐]]发生[[复分解反应|复分解反应]]：碱（可溶） + 盐（可溶） → 新碱 + 新盐 

::<chem>2NaOH + CuSO4 -> Cu(OH)2 v + Na2SO4</chem>

* 和某些非金属氧化物反应：碱（可溶） + 某些非金属氧化物 → 水 + 盐

::<chem>CO2 + Ca(OH)2 -> CaCO3 v + H2O</chem>

* 通常有滑腻感、味道苦，而且是腐蝕性。因此多用作清潔用途，腐蝕和去除污垢

== 常见的碱 ==
* [[氢氧化钠|氢氧化钠]]（NaOH）- 俗称烧碱、火碱、苛性钠
* [[氢氧化钙|氢氧化钙]]（Ca(OH)<sub>2</sub>）- 俗称熟石灰、消石灰
* [[氨水|氨水]]（NH<sub>3</sub>·H<sub>2</sub>O）- 又叫一水合氨
* [[氢氧化钾|氢氧化钾]]（KOH）

== 碱味 ==
碱在人类口感中，往往呈现生涩的味觉感受，类似未成熟的[[柿子|柿子]]、[[香蕉|香蕉]]的单宁口感。同酸味相对，只是与酸不同的是，一般碱在普通人印象中没有如同酸一样的可让人闻到的气味。
