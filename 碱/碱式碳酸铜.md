'''碱式碳酸铜'''化学式为[[铜|Cu]]<sub>2</sub>([[氢氧根|OH]])<sub>2</sub>[[碳酸根|CO<sub>3</sub>]]，也有写作[[铜|Cu]][[碳酸根|CO<sub>3</sub>]]·[[铜|Cu]]([[氢氧根|OH]])<sub>2</sub>，颜色翠绿，在自然界中[[铜|铜]]通常以此种化合物的形式存在，它是[[铜|铜]]与空气中的[[氧气|氧气]]、[[二氧化碳|二氧化碳]]和[[水|水]]等物质反应产生的物质。不溶于水。

'''碱式碳酸铜'''一般常称為'''銅鏽'''或'''铜绿'''。

== 孔雀石 ==
碱式碳酸铜的结晶为[[孔雀石|孔雀石]]，是一种名贵的矿物宝石。
:2Cu + O<sub>2</sub> + CO<sub>2</sub> + H<sub>2</sub>O → Cu<sub>2</sub>（OH）<sub>2</sub>CO<sub>3</sub>

== 制备 ==

=== 用[[碳酸钠|碳酸钠]]与[[硫酸铜|硫酸铜]]作用制备 ===

反应方程式：
2Na<sub>2</sub>CO<sub>3</sub> + 2CuSO<sub>4</sub> + H<sub>2</sub>O → Cu<sub>2</sub>（OH）<sub>2</sub>CO<sub>3</sub>↓ + CO<sub>2</sub>↑ + Na<sub>2</sub>SO<sub>4</sub>

具体方法：<ref>陈静，石晓波，龚丽维，潘荣楷.碱式碳酸铜的制备[J]大学化学，2012，27（5）:78-82</ref>

按体积比V（CuSO<sub>4</sub>）：V(Na<sub>2</sub>CO<sub>3</sub>)=1:1.2取0.5mol/L的碳酸钠溶液和0.5mol/L的硫酸铜溶液，同时置于70℃的水浴中直到恒温。然后将两者混合，置于70℃的水浴环境下攪拌。可以观察到有浅蓝色沉淀产生。调节pH值为8.50~8.88，不断搅拌悬浊液直到沉淀变为绿色。此时停止搅拌，静置，待沉淀沉降之后进行过滤，并用去离子水洗涤沉淀，直到不含硫酸根离子（硫酸鈉）為止。然後将沉澱至于烘箱中以100℃烘干，即可得到绿色的碱式碳酸铜。

=== 用[[碳酸氢钠|碳酸氢钠]]与[[硫酸铜|硫酸铜]]作用制备 ===
反应方程式：
2CuSO<sub>4</sub> + 4NaHCO<sub>3</sub> → Cu<sub>2</sub>（OH）<sub>2</sub>CO<sub>3</sub>↓ + 3CO<sub>2</sub>↑ + H<sub>2</sub>O + 2Na<sub>2</sub>SO<sub>4</sub>

== 分解 ==
碱式碳酸铜加热会分解为[[氧化铜|氧化铜]]、[[水|水]]和[[二氧化碳|二氧化碳]]。反应式：
:<math>\mathrm{{Cu_2(OH)_2{CO_3}\ {^{^{\;\Delta}_\longrightarrow}}\ 2CuO+CO_2 \uparrow +H_2 O}}</math>

== 除鏽 ==
碱式碳酸铜放入稀硫酸生成[[硫酸铜|硫酸铜]]、[[水|水]]和[[二氧化碳|二氧化碳]]。反应式：
:Cu<sub>2</sub>（OH）<sub>2</sub>CO<sub>3</sub> + 2H<sub>2</sub>SO<sub>4</sub> → 2CuSO<sub>4</sub> + CO<sub>2</sub>↑ + 3H<sub>2</sub>O

==其他反应==
===与盐酸反应===
Cu<sub>2</sub>（OH）<sub>2</sub>CO<sub>3</sub>+4HCl→2CuCl<sub>2</sub>+CO<sub>2</sub>↑+3H<sub>2</sub>O.
