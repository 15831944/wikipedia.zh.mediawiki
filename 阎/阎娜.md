'''阎娜'''{{BD|1987年|10月17日|||catIdx=Yan阎}}出生于[[辽宁省|辽宁省]][[鞍山市|鞍山市]]，毕业于[[中央戏剧学院|中央戏剧学院]]，为中国大陆影视演员。<ref name=" cat" >{{cite web |url=http://www.fjsen.com/o/2011-07/19/content_5237024.htm |title=阎娜大戏连台人气飙升 生活中就像个孩子 |accessdate=2011-07-19 |publisher=东南网 |archiveurl=http://www.fjsen.com/o/2011-07/19/content_5237024.htm |archivedate= 2012年12月27日|work= |language=zh-cn}}</ref>

== 生平 ==
1987年，阎娜出生在[[辽宁省|辽宁省]][[鞍山市|鞍山市]]，4岁时，阎娜被母亲送入[[广州军区|广州军区]]战士杂技团，11岁的时候，阎娜考上广东省舞蹈学校，一直学习到17岁，后在[[中央戏剧学院|中央戏剧学院]]读书，2006年学成毕业。<ref name=" cat" />

== 个人生活 ==
对于媒体曾爆料阎娜与[[周渝民|周渝民]]在《彼岸1945》拍摄时相识相恋，遭到了周渝民经纪人的否认。<ref>{{cite web |url=http://et.21cn.com/star/zhuixing/gangtai/2012/10/31/13457411.shtml |title=周渝民被曝与内地女演员阎娜恋爱 经纪人否认 |accessdate=2012-10 |publisher=21CN |archiveurl=http://et.21cn.com/star/zhuixing/gangtai/2012/10/31/13457411.shtml |archivedate=2012年12月27日 |work= |language=zh-cn}}</ref>
