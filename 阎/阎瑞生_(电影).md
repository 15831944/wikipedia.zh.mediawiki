《'''阎瑞生'''》是中国于1921年拍摄，当年7月1日公映的故事长片。它与同年上映的另外两部故事长片《[[海誓|海誓]]》、《[[红粉骷髅|红粉骷髅]]》被视为[[華語電影#歷史|中国电影]]类型片的肇始，亦是中国电影商业化的开端<ref>{{Cite web |url=http://ent.sina.com.cn/2004-04-16/0915366232.html |title=1921年·《阎瑞生》《海誓》《红粉骷髅》(图) |accessdate=2018-10-20 |author=|date=2004-04-16 |work= |publisher=新浪网，来源：新京报|language=Zh-hans }}</ref>。电影[[拷贝|拷贝]][[散失電影|已散失]]，仅存数张[[剧照|剧照]]<ref name="a1">{{Cite web |url=http://news.163.com/15/0104/15/AF4H8VNP00014AED.html |title=影片《一步之遥》与阎瑞生案件(组图) |accessdate=2018-10-20 |author=张伟|date=2015-01-04 |work=责任编辑：王晓易_NE0011 |publisher=网易，来源：新民晚报(上海)|language=Zh-hans }}</ref>。

== 拍摄 ==
[[File:Yan_ruishen_(film),1921.jpg|thumb]]
电影取材于1920年6月，在上海发生的[[阎瑞生|阎瑞生]]劫杀名妓王莲英一案。此案是当时社会舆论焦点，各类书籍、戏曲亦有演绎。当时，电影[[发烧友|发烧友]][[陈寿芝|陈寿芝]]、[[施彬元|施彬元]]、[[邵鹏|邵鹏]]、[[徐欣夫|徐欣夫]]、[[顾肯夫|顾肯夫]]、[[陆洁|陆洁]]在上海成立[[中国影戏研究社|中国影戏研究社]]。中国影戏研究社筹划、投资，[[商务印书馆|商务印书馆]]活动影戏部提供技术设备、拍摄人员<ref name="a1"/>。

电影于1921年春夏之交正式开拍。场景多用实景拍摄，字幕为[[白话文|白话文]]<ref name="a1"/>。拍摄“枪毙”一场戏时，竟“要求淞沪护军使署，请仍派出如前次执行枪毙时之军队，押送该二犯前赴龙华操场……站齐队伍如法开枪，惟除去子弹，其余情形竟与当时无异”。或认为，这不过电影噱头而已。4月，在《申报》发布广告，称“阎瑞生谋害莲英案喧传社会，无人不知，现闻由中国影戏研究社将全案事实编为影戏……全片制成共费五万元，长约万余尺，不日即可出演，届时必可哄动一时也。又该社以外人影片所摄中国人，均囚首鹄面极形不堪，殊有玷国体，兹拟多摄有价值之影片输往外洋，以介绍我国人真面目云。”<ref name="a3">{{Cite web |url=http://www.takefoto.cn/viewnews-1349695.html |title=《阎瑞生》:一桩命案 催生中国首部电影长片|accessdate=2018-11-07 |author=蔡辉|date=2017-12-15 |work= |publisher=北晚新视觉|language=Zh-hans }}</ref>

剧中角色则由非职业演员扮演。陈寿芝扮演阎瑞生，邵鹏饰演吴春芳，王彩云饰演王莲英<ref name="a1"/>。时长达至100分钟的《阎瑞生》，是中国第一部国产故事长片<ref name="a2"/>。

== 上映 ==

1921年7月1日，电影在上海夏令配克影戏院首映。票价1元和1.5元。电影上映之初获得了巨大的成功，“未至开映时间，包厢已预订一空，池座亦满。<ref name="a3"/>”上映一星期净赚四千大洋。有评论称“中国影戏足以获利之影像，始深映入华人之脑”<ref name="a1"/>。但不久它与同类型影片《[[张欣生|张欣生]]》即因社会舆论被官方禁映，成为中国最早的[[禁片|禁片]]<ref name="a2">{{Cite web |url=http://history.people.com.cn/n/2013/0513/c362055-21461883.html |title=第一部被官方禁映的电影：《阎瑞生》 |accessdate=2018-10-20 |author=|date=2013-05-13 |work=责编：张淑燕、肖静 |publisher=人民网，节选自《国家人文历史》2013年第2期中的《剪刀与电影共同成长》一文|language=Zh-hans }}</ref>。
