'''阎明复'''（{{bd|1931年|||}}），[[辽宁|辽宁]][[海城市|海城]]人，[[阎宝航|阎宝航]]之子。曾任[[中共中央书记处|中共中央书记处]]书记、[[中共中央统战部|中共中央统战部]]长等职，在[[中共十三届四中全会|中共十三届四中全会]]上因其[[六四事件|六四事件]]“表现不好”被撤职，之后一直从事[[慈善事业|慈善事业]]。他曾组织编纂了《[[中国大百科全书|中国大百科全书]]》，并且对中国的民主化进程做出了一定的贡献。

== 生平 ==
阎明复在1931年生于[[北平|北平]]，1947年参加革命，兩年後加入[[中国共产党|中国共产党]]，同年毕业于哈尔滨外国语专门学校（即[[黑龙江大学|黑龙江大学]]）。1949年至1966年在[[全国总工会|全国总工会]]、中共中央办公厅任俄文翻译，曾在1959年，担任[[赫鲁晓夫|赫鲁晓夫]]与毛泽东会谈的翻译<ref>[http://www.doc88.com/p-7052482411555.html 1959年赫鲁晓夫访华的前前后后]</ref><ref>{{cite web |title=胡德平：中苏大论战是极左反对左的“空对空” |url=http://www.21ccom.net/articles/history/xiandai/20150629126213_all.html |website=[[共识网|共识网]] |archiveurl=https://archive.is/20150629100716/http://www.21ccom.net/articles/history/xiandai/20150629126213_all.html |archivedate=2015-06-29 |date=2015-06-29 |deadurl=yes }}</ref>。1967年至1975年在[[文革|文革]]中遭到残酷迫害，曾被非法关押。

1975年至1983年任[[中共中央编译局|中共中央编译局]]毛泽东著作编译室定稿员，[[中国大百科全书|中国大百科全书]]出版社副总编辑。

1983年任第六届[[全国人大常委会|全国人大常委会]]副秘书长，兩年后担任[[中共中央统战部|中共中央统战部]]部长。1986年被增选为第六届[[全国政协|全国政协]]常务委员，翌年在[[中共十三大|中共十三大]]上当选为中央委员、十三届一中全会上当选为[[中共中央书记处|中共中央书记处]]书记。1988年当选为第七届[[全国政协副主席|全国政协副主席]]。<ref>中国网：[http://www.china.com.cn/zhuanti2005/txt/2002-09/22/content_5208107.htm 阎明复]2010年3月4日查阅</ref>

=== 六四事件 ===
1989年5月，在[[六四事件|六四事件]]中，一部分新闻工作者递交了请愿书。于是在1989年5月11至13日之间，阎明复和[[胡启立|胡启立]]、[[芮杏文|芮杏文]]访中国青年报社等新闻单位，并与记者进行了对话。13日，阎明复，胡启立统战部听取了一些新闻界的专家的意见，并且在下午与北京15家新闻单位的主要负责人进行了座谈。同日，示威学生开始[[绝食|绝食]]抗议，并且表示“不达目的决不罢休”。得知消息之后，阎明复约请[[知识分子|知识分子]]进行斡旋，[[严家其|严家其]]、[[戴晴|戴晴]]、[[刘晓波|刘晓波]]等到广场劝说学生停止绝食，未果。

5月14日上午阎明复约请[[王丹|王丹]]，[[吾尔开希|吾尔开希]]等人在统战部进行谈话，决定在下午4点，在统战部礼堂进行对话。但是学生提出对话级别不够，需要全程直播。经过妥协之后，决定改为进行现场录像，并且在电视台播出。下午4点，对话开始，由阎明复主持。对话进行到下午7点，广场绝食学生来到对话现场再次要求电视直播并且形成僵持局面，阎明复宣布休会，第二天继续进行对话<ref name="ref1">[http://64memo.com/disp.aspx?Id=15&Page=1#NT1175  《中国「六四」真相》]2010年3月4日查阅</ref>，但翌日的对话仍然没有进一步的进展<ref name="ref1"/>；直至5月16日，阎明复在得到总书记[[赵紫阳|赵紫阳]]同意之后来到广场，虽然对在广场上的学生说了些打动他们的話，还表示愿意一起静坐和做他们的人质，但是并没有起到扭转局势的作用。反而成为了后来阎明复被撤职的重要依据。<ref name="ref1"/>

結果在同年6月于中共第十三届四中全会上，阎明复与[[胡启立|胡启立]]、[[芮杏文|芮杏文]]一起被免去的中央书记处书记职务<ref>新华网：[http://news.xinhuanet.com/ziliao/2004-10/15/content_2093955_2.htm 中国共产党大事记（1989年）]2010年3月4日查阅</ref>。1990年，他更被免去中共中央统战部部长、全国政协副主席以及全国政协党组副书记等职务<ref>中央政府門戶網站：[http://www.gov.cn/gjjg/2008-10/23/content_1128560.htm 阎明复]2010年3月4日查阅</ref>。

1991年复出，任[[中华人民共和国民政部|中华人民共和国民政部]]副部长。

=== 慈善活动 ===
阎明复参与创办了[[中华慈善总会|中华慈善总会]]，并于1997年至2002年担任会长一职。<ref name="ref2">[http://longquanzs.org/articledetail.php?id=2449 阎明复:一个退休高官的慈善试验 ] {{webarchive|url=https://web.archive.org/web/20100613165332/http://longquanzs.org/articledetail.php?id=2449 |date=2010-06-13 }}2010年3月4日查阅</ref>2013年，因“在总会创立和发展过程中，呕心沥血、无私奉献，为慈善事业作出了重要贡献”，与[[崔乃夫|崔乃夫]]、[[范宝俊|范宝俊]]一起被授予“中华慈善总会终身荣誉会长”称号。<ref>[http://www.xfrb.com.cn/newsf/2013/04/24/1366773458.htm 消费日报：中华慈善总会举行第四次会员代表大会] {{webarchive|url=https://web.archive.org/web/20130603224258/http://xfrb.com.cn/newsf/2013/04/24/1366773458.htm |date=2013-06-03 }}</ref>

在阎明复任中华慈善总会会长之后一直致力于推进中国慈善事业发展。尤其是在98抗洪期间，在阎明复的联络下，成功的进行了“我们万众一心”大型电视义演晚会。筹集善款3亿多人民币。之后，中华慈善总会在2008年中国雪灾，[[汶川大地震|汶川大地震]]中都发挥了重要的作用。<ref>李玉林 :[http://www.szjt.org/fj_renjian/cishan/contents/cishan017.htm 阎明复人生历程随记] {{webarchive|url=https://web.archive.org/web/20110513090858/http://www.szjt.org/fj_renjian/cishan/contents/cishan017.htm |date=2011-05-13 }}2010年3月4日查阅</ref>

== 著作及其他社会活动 ==
阎明复早期曾任[[中国大百科全书出版社|中国大百科全书出版社]]副总编辑，大力推动了《[[中国大百科全书|中国大百科全书]]》的编纂工作。在一次访谈中他把“组织出版了中国的第一部大百科全书”和“参与创办[[中华慈善总会|中华慈善总会]]”作为自己一生中最有意义的两件事。<ref name="ref2"/>

2001年出版《美国慈善事业一瞥》一书。在2005年之后，阎明复一直在《[[炎黄春秋|炎黄春秋]]》杂志不定期的发表一些文章，如《从我亲历的几件事看康生》，《我了解的赫鲁晓夫权力斗争》等。并且作为总顾问指导了电视剧《[[英雄无名|英雄无名]]》的拍摄。

== 家人 ==
*父亲：[[阎宝航|阎宝航]]，曾是[[张学良|张学良]]的密友，1937年[[周恩来|周恩来]]介绍阎宝航加入中国共产党，并且在国民党内帮助[[中共|中共]]搜集了包括[[纳粹德国|纳粹德国]]入侵[[苏联|苏联]]的准确日期等重要情报。解放后阎宝航在[[文革|文革]]中被诬陷为“东北帮叛党投敌反革命集团”，并于1967年11月7日被捕。之后惨死于[[秦城监狱|秦城监狱]]。<ref>阎明复：[http://www.yhcqw.com/html/rwz/2008/417/0841718667HA2E153561G80953161K4D0.html 我父亲阎宝航的情报生涯——炎黄春秋]2010年3月4日查阅</ref>
*二哥：[[阎明智|阎明智]]，俄语翻译，中华人民共和国驻苏使馆一秘，1956年曾负责连夜为中共代表团翻译赫鲁晓夫的[[秘密报告|秘密报告]]。
*姐姐：[[阎明光|阎明光]]，是[[上海阎宝航社会公益基金会|上海阎宝航社会公益基金会]]理事长。
