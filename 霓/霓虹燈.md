'''霓虹燈'''（{{lang-en|Neon Lighting}}）是一種內含[[氣體|氣體]]的[[燈|燈]]，通常用於[[廣告|廣告]]招牌等用途。 “霓虹灯”是半英语音译：“霓虹”发音近似于英语的“neon”（氖）并在汉语裏有彩虹的含义。城市'''霓虹灯招牌'''（{{lang-en|Neon Sign}}）属于一种特殊的大型[[氖灯|氖灯]]。管內氣體的實際成份為99.5%氖氣及0.5%[[氬|氬]]氣，比純氖氣有較低的運作電壓。

== 历史 ==
[[File:(2005)_Crossroads_of_the_World_(96905377).jpg|缩略图]][[時報廣場|時報廣場]]]]
{{fact|霓虹燈首先由Nikola Tesla在1893年的[[芝加哥|芝加哥]]博覽會（World's Columbian Exposition）進行示範，但其創意並未有申請[[專利|專利]]。}}

Georges Claude在1902年發明霓虹燈，並在1910年向公眾展示。

== 原理 ==
[[File:Neon.JPG|thumb]]
充在燈管中能發出紅光的種氣體是氖氣。但是單是紅色的霓虹燈是不夠的。若要霓虹燈產生不同的色彩，這時需要以綠、黃色，藍色、白色的[[螢光粉|螢光粉]]作為輔助。

例如將藍色的螢光粉塗在玻璃管的內壁上，把玻璃管彎製成所需要的文字或花紋圖案後裝上電極，並把玻璃管內的空氣抽乾淨，再充進氖氣，即成粉紅色的霓虹燈。

如果塗上了藍色螢光粉的燈管中充入氬氣和水銀，就成了藍色的霓虹燈；若塗有綠色螢光粉的燈管中充入氖氣，就成了橙紅色；如果把氖氣改為氬氣和水銀，便會成為綠色霓虹燈。

==流行文化==
導演[[王家衛|王家衛]]1994年執導的電影《[[重慶森林|重慶森林]]》中曾出現不少香港霓虹街道作場景。

== 香港現況 ==
香港在[[第二次世界大战|二戰]]後經濟發展迅速，加上霓虹燈為當時較前衛的科技，不論大小公司都以此作為宣傳方式，於[[1980年代香港|1980]]至[[1990年代香港|1990]]年代數量達到高峰，不過近年相反有減無增，可歸納為以下原因：

# LED的出現提供更便宜和省電的選擇，讓不少店舖在添加新招牌或更換招牌時用上此照明方式吸引顧客
# 霓虹燈多見於較早發展的[[香港島|港島]]和[[九龍|九龍]]地區，但舊區逐漸面臨重建，適合懸掛招牌的[[唐樓|唐樓]]會拆卸，而新式建築外牆未預留空間，同時發展商亦未必願意出租相關的位置予店舖
# 2010年政府全面實施「小型工程監管制度」，新制度下的招牌不能自外牆伸出多於4.2米，離地不能少於3.5米。未經[[屋宇署|屋宇署]]批准或根據規定均屬違例，當局將發出清拆令要求拆除<ref>{{Cite web|url=https://www.bd.gov.hk/chineseT/services/index_mwcs_items_c6b.html|title=小型工程項目：招牌|accessdate=2018-11-05|author=|date=2015-11-23|publisher=屋宇署}}</ref>。部份已成地標的霓虹燈招牌因此需要拆除，例如[[深水埗|深水埗]]的「[[信興酒樓|信興酒樓]]」招牌及[[西環|西環]]森美餐廳自1977年豎設的「Sammy's Kitchen」牛形霓虹招牌<ref>{{Cite news|url=https://www.hk01.com/%E7%A4%BE%E5%8D%80%E5%B0%88%E9%A1%8C/30270/%E6%B7%B1%E6%B0%B4%E5%9F%97%E7%92%80%E7%92%A8%E6%A8%99%E8%A8%98%E5%85%89%E8%BC%9D%E5%88%B0%E6%AD%A4-60%E5%B9%B4%E8%80%81%E6%8B%9B%E7%89%8C-%E5%9F%8B%E8%91%AC-%E5%A0%86%E5%A1%AB%E5%8D%80|title=深水埗璀璨標記光輝到此　60年老招牌「埋葬」堆填區|author=林可欣|date=2016-07-09|work=香港01|accessdate=2018-11-05}}</ref>
