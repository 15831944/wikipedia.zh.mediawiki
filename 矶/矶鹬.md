'''矶鹬'''（学名''Actitis hypoleucos''），是[[鷸科|鷸科]]的一种。

== 生态环境 ==
栖息地从沿海滩涂和沙洲至海拔1500米的山地稻田及溪流、河流两岸。行走时头不停地点动，并具两翼僵直滑翔的特殊姿势，定居性 :冬候鳥、留鳥 。

== 分布地域 ==
繁殖于古北界及喜马拉雅山脉；冬季南飞至非洲、印度次大陆、东南亚并远至澳大利亚。

== 特征 ==
体长20cm，[[喙|喙]]短，性活跃，翼不及尾。上体褐色，飞羽近黑；下体白，胸侧具褐灰色斑块。特征为飞行时翼上具白色横纹，腰无白色，外侧尾羽无白色横斑。翼下具黑色及白色横纹。 虹膜－褐色；[[喙|喙]]－深灰；脚－浅橄榄绿；叫声－细而高的管笛音twee-wee-wee-wee。

== 食物 ==
以[[昆蟲、軟體動物|昆蟲、軟體動物]]為食。

== 繁殖 ==
在地上營巢。
