-{zh-hans:'''隔空投送'''（{{lang-en|AirDrop}}）;zh-hant:'''AirDrop'''}-是[[蘋果公司|蘋果公司]]的[[MacOS|MacOS]]和[[iOS|iOS]][[操作系统|操作系统]]中的一个[[隨建即連網路|隨建即連網路]]，自[[Mac_OS_X_Lion|Mac OS X Lion]]（Mac OS X 10.7）和[[iOS_7|iOS 7]]起引入，<ref name="About.com">{{cite web | url=http://ipad.about.com/od/iPad_Guide/ss/What-Is-Airdrop-How-Does-It-Work.htm | title=What Is AirDrop? How Does It Work? | publisher=About.com | accessdate=12 January 2014 | author=Nations, Daniel}}</ref>允许在支持的麦金塔计算机和iOS设备上传输文件，无需透过[[电子邮件|邮件]]或{{tsl|en|mass storage|大容量存储}}设备。<ref name="About.com" />

在OS X Yosemite（OS X 10.10）之前，[[OS_X|OS X]] 中的AirDrop协议不同于iOS的AirDrop协议，因此不能互相传输<ref>{{cite web | url=http://techcrunch.com/2013/09/17/ios-7-review-apple/ | title=Apple iOS 7 Review: A Major Makeover That Delivers, But Takes Some Getting Used To | publisher=[[美国在线|美国在线]] | work=[[TechCrunch|TechCrunch]] | date=September 17, 2013 | accessdate=October 3, 2013 | author=Etherington, Darrell}}</ref>。但是，OS X Yosemite或更新版本支持iOS的AirDrop协议（使用[[Wi-Fi|Wi-Fi]]和[[藍牙|藍牙]]），这适用于一台Mac与一台iOS设备以及两台2012年或更新版本的Mac计算机之间的传输。<ref>{{cite web | url=http://www.iphonehacks.com/2014/06/ios-8-and-os-x-yosemite-airdrop.html | title=iOS 8 and OS X Yosemite Preview: AirDrop finally works across iOS and Mac | publisher=iPhone Hacks | work= iPhone Hacks | date=June 23, 2014 | accessdate=October 14, 2015 | author=Jason}}</ref><ref name="OSX">{{cite web | url=https://support.apple.com/HT203106 | title=Use AirDrop to send content from your Mac | publisher=苹果公司 | date=September 30, 2015 | accessdate=October 14, 2015}}</ref>使用旧AirDrop协议（只使用[[Wi-Fi|Wi-Fi]]）的旧模式在两台2012年或更早的Mac计算机之间传输也是可行的。<ref name="OSX"/>

AirDrop所容纳的文件大小没有限制。苹果用户报告称AirDrop能传输小于10GB的视频文件。{{fact|date=November 2016}}

==环境==
===macOS===
在运行OS X 10.7的Mac上，AirDrop可以在[[Finder|Finder]]視窗側邊欄中使用。<ref name="howto">{{cite web | url=http://www.cnet.com/how-to/how-to-use-os-x-lion-airdrop/ | title=How to use OS X Lion AirDrop - CNET | publisher=CNET | work=[[CNET|CNET]] | date=September 29, 2011 | accessdate=October 14, 2015 | author=Rawlinson, Nik}}</ref>在OS X 10.8.1或更新版本的Mac上，它也可以透过選单“前往”→“AirDrop”或按{{key press/core|Shift}}+{{key press/core|Cmd}}+{{key press/core|R}}来存取。<ref>{{cite web | url=http://macosxautomation.com/workshops/sharing/01.html | title=OS X Automation Workshop: Sharing Content Locally | website=Mac OS X Automation | accessdate=October 14, 2015}}</ref>

必须开启[[Wi-Fi|Wi-Fi]]，AirDrop才能识别其他裝置。其他设备还必须在Finder視窗側邊欄中选择AirDrop才能传输檔案。此外，檔案不会被自动接受；接收用户必须接受传输。<ref name="howto"/>这是为提高安全性和隐私性。<ref name="howto"/>

===iOS===
在iOS 7或更新版本上，AirDrop可以透过{{tsl|en|Control Center (iOS)|控制中心 (iOS)|控制中心}}启用。<ref name="iOS">{{cite web | url=https://support.apple.com/HT204144 | title=Share content with AirDrop from your iPhone, iPad, or iPod touch | publisher=苹果公司 | date=July 23, 2015 | accessdate=October 14, 2015}}</ref>AirDrop启用时Wi-Fi和蓝牙都将自动开启，因为两者都会被使用。<ref name="iOS"/>

控制其他裝置发现AirDrop的选项包括：<ref name="iOS"/>
*看不到此裝置（关闭Airdrop）
*仅限聯絡人能看到此裝置
*所有人都能看到此裝置。

在iOS 7及更新版本中，如果應用程式实现AirDrop支援，它可以透过“共享”按钮使用。<ref>{{cite web | url=http://www.appcoda.com/ios7-airdrop-programming-tutorial/ | title=iOS Programming Tutorial: Adding AirDrop File Sharing Feature to Your iOS Apps | publisher=AppCoda | work= AppCoda | date=October 30, 2013 | accessdate=October 14, 2015 | author=Ng, Simon}}</ref>AirDrop在iOS上受到多项限制，例如无法从內建应用程式分享音乐或影片。<ref>{{cite web | url=http://ioshacker.com/cydia/anydrop-enables-sendreceive-file-including-music-using-airdrop | title=AnyDrop enables you to Send/Receive any file including Music using AirDrop - iOS Hacker | publisher=iOS Hacker | work= iOS Hacker | date=March 24, 2014 | accessdate=October 14, 2015}}</ref>

== 安全与隐私 ==

AirDrop在传输文件时使用[[傳輸層安全協議|傳輸層安全協議]]。<ref>https://www.apple.com/business/docs/iOS_Security_Guide.pdf</ref>将AirDrop设为接收“所有人”的用户可能收到附近的陌生人发送的非期望的露骨色情图片。<ref>{{cite web|url=http://www.bbc.com/news/technology-33889225 |title=Police investigate 'first cyber-flashing' case |date=13 August 2015 |author=Sarah Bell}}</ref>

== 限制及問題 ==
迄今為止，蘋果並未介紹文件通過不同的設備傳輸時經過了何種轉換。

通過隔空投送所傳輸的文件和源文件可能並不一致，網友在不同的Mac和iPhone設備間傳輸同一文件，結果發現在不同設備中的文件並不相同。例如將iPhone 8 Plus所拍攝的4K 60p的視頻通過AirDrop傳輸給iPhone 7 Plus時，最初設備會顯示「正在轉換」；同樣將iPhone 8 Plus所拍攝的4K 60p的視頻傳遞給iPhone 5一類的設備後，其文件大小會有很大改變。
