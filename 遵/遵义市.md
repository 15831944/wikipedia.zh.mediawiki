'''遵义市'''是[[中华人民共和国|中华人民共和国]][[贵州省|贵州省]]下辖的[[地级市|地级市]]，位于中国西南部，贵州省北部，云贵高原东北部。是贵州省第二大城市、新兴工业城市和重要农产品生产基地。黔北政治经济文化中心。。北临[[重庆市|重庆市]]，东接[[铜仁市|铜仁市]]，东南界[[黔东南州|黔东南州]]，南邻[[贵阳市|贵阳市]]、[[黔南州|黔南州]]，西南连[[毕节市|毕节市]]，西毗[[四川省|四川省]][[泸州市|泸州市]]。位处贵州高原北部[[大娄山|大娄山]]区，[[乌江|乌江]]流经南面边界。全市总面积3.08万平方公里，人口800万。遵义是中国首批[[国家历史文化名城|国家历史文化名城]]。“遵义”之名相传出自《[[尚书_(书籍)|尚书]]》：“无偏无陂，遵王之义”<ref>{{Cite web|url=http://www.gog.cn/zonghe/system/2016/12/14/015285048.shtml|title=城市社会学视野下遵义城市精神的传承与弘扬|accessdate=2018-02-18|work=www.gog.cn}}</ref>。

== 历史 ==
古为[[梁州|梁州]]之域。[[春秋战国|春秋战国]]时期，曾是[[巴国|巴]]、[[蜀国|蜀]]、[[夜郎|夜郎]]等邦国的领地。
[[汉朝|汉]][[建元|建元]]六年（前135年）置[[犍为郡|犍为郡]]，治鳖县。[[元鼎|元鼎]]六年（前111年）隶[[牂柯郡|牂柯郡]]。

[[唐朝|唐]][[貞觀_(唐朝)|贞观]]元年（627年）置[[郎州|郎州]]及[[恭水县|恭水县]]。贞观十三年（639年）改为[[播州|播州]]。贞观十四年（640年）改恭水县为[[罗蒙县_(唐朝)|罗蒙县]]。贞观十六年（642年），罗蒙县更名为[[遵义县_(唐朝)|遵义县]]（[[治所|治所]]在今遵义市[[绥阳县|绥阳县]]境）。[[晚唐|晚唐]]时，杨氏家族世袭播州[[刺史|刺史]]职位，形成[[播州杨氏|播州杨氏]]土司政权，一直沿袭至明朝末年。

[[北宋|北宋]][[大观|大观]]二年（1108年）置[[遵义军|遵义军]]。[[宣和|宣和]]三年（1121年）废播州、遵义军。[[南宋|南宋]][[嘉熙|嘉熙]]三年（1239年）设播州安抚司。

[[元朝|元]][[至元|至元]]二十八年（1291年）升为播州宣抚司。

[[明朝|明]][[洪武|洪武]]五年（1372年）改置[[播州宣慰司|播州宣慰司]]，改属[[四川省|四川省]]。洪武十五年（1382年）隶属[[贵州都司|贵州都司]]。洪武二十七年（1394年）又改属四川省。[[万历|万历]]二十八年（1600年），[[明朝政府|明朝政府]]平定杨氏土司[[杨应龙|杨应龙]]的[[播州之役|叛乱]]，对播州宣慰司进行[[改土归流|改土归流]]；置[[遵义军民府|遵义军民府]]（隶[[四川|四川]]）、[[平越军民府|平越军民府]]（隶[[贵州|贵州]]），同时以[[播州长官司|播州长官司]]置[[遵义县_(明朝)|遵义县]]（今遵义市城区），唐朝[[遵义县_(唐朝)|遵义县]]境置[[绥阳县|绥阳县]]。

[[清朝|清]][[康熙|康熙]]二十六年（1687年）改遵义军民府为[[遵义府|遵义府]]。[[雍正|雍正]]六年（1728年），遵义府由四川省划归贵州省。

[[民国|民国]]二年（1913年），废弃府的建置，县属[[黔中道|黔中道]]。民国二十四年（1935年）贵州省设11个行政督察区，黔北为第五行政督察区，专员公署先设[[桐梓|桐梓]]，后迁遵义县城（即今红花岗区）。

[[中华人民共和国|中华人民共和国]]成立后，1949年11月设置[[遵义专区|遵义专区]]。1950年析遵义县城置遵义市，由遵义专区领导，1951年曾撤并，1952年复置。1970年遵义专区改称[[遵义地区|遵义地区]]。1997年6月，撤销遵义地区，改设地级遵义市，原县级遵义市改为[[红花岗区|红花岗区]]；赤水市、仁怀市为省辖市，由遵义市代管。2004年，由红花岗区和遵义县北部划出部分地区，在原遵义经济技术开发区的基础上新成立[[汇川区|汇川区]]。2009年5月4日，由遵义市红花岗区新蒲镇和遵义县新舟镇组成的新蒲新区挂牌成立。2016年3月，撤销遵义县，设立[[播州区|播州区]]。

== 地理 ==
遵义市（地级市）位于中国西南部，贵州省北部，云贵高原东北部。地理位置在北纬27°8´～29°12´、东经105°36´～108°13´之间。市域东西绵延254千米，南北相距230.5千米。全市国土面积30762平方千米，为贵州省总面积的17.5％。北依娄山，与[[重庆|重庆]][[綦江|綦江]]隔山相望，南临[[乌江|乌江]]，可谓黔川要道之咽喉，黔北重镇。

遵义属高原[[亚热带季风气候|亚热带季风气候]]，四季分明，季风影响较重。冬季微凉，低温有时降下0℃。夏季闷热，高温约为30℃。日照时数很少，仅为1050小时。

=== 自然资源 ===
*河流与水资源   全市河流均属长江流域。以大娄山山脉为分水岭，把全市河流分为乌江、赤水河和綦江三大水系。水能资源蕴藏量为350.60万千瓦，可开发量为452万千瓦，是贵州省水能资源较丰富的地区之一。

*矿产资源   市内已探明的矿产有60多种。其中15种藏量居贵州省首位。煤、锰、铝土、汞、硫铁矿有“五朵金花”之称。铝土矿已探明储量和工业保有量居全国同类地区第二、矿石质量居全国之首。探明有硫铁矿3.27亿吨，锰矿5399万吨，镓矿4609.6万吨，页岩2178.5万吨等。

*生物资源   全市森林面积149.37万公顷，森林覆盖率53.9％。高于贵州省和全国比率。仅全市15个自然保护区内，即有野生植物251科831属2894种，其中国家重点保护野生植物一级9种，二级29种，省级重点保护野生植物39种；列入《濒危野生动植物种国际贸易公约》附录Ⅱ的兰科植物有20种。自然保护区内有野生动物87科314种，其中国家重点保护野生植物一级4种，二级34种。银杉、桫椤、珙桐、金花茶、黑叶猴、白冠长尾雉、大灵猫等都是被列为国家一、二级重点保护的珍稀动植物。

*种植养殖   遵义市的粮、油、烟、畜、茶、竹、中药材，均为重要和特色资源，素有“黔北粮仓”之称。烟叶质量优良，是全国三大优质烟区之一。楠竹为全国七大主产区之一。药用植物，已搜集到的有2048种，其中常用中药材200种以上，五倍子占全省的五分之一、全国的六分之一。天麻、杜仲、吴茱萸、石斛、黄连被列为贵州五大名药。赤水金钗石斛是国内唯一获得国家地理标志保护的石斛品种。盛产朝天椒的绥阳县，被誉为中国辣椒之乡。

=== 行政区划 ===
[[File:ZY_Zunyi_Airport.jpg|thumb]]]]
遵义市辖3个[[市辖区|市辖区]]、7个[[县_(中华人民共和国)|县]]、2个[[自治县|自治县]]，代管2个[[县级市|县级市]]、及新蒲新区、南部新区。市人民政府驻地汇川区人民路。

* 市辖区：[[红花岗区|红花岗区]]、[[汇川区|汇川区]]、[[播州区|播州区]]
* 县级市：[[赤水市|赤水市]]、[[仁怀市|仁怀市]]
* 县：[[桐梓县|桐梓县]]、[[绥阳县|绥阳县]]、[[正安县|正安县]]、[[凤冈县|凤冈县]]、[[湄潭县|湄潭县]]、[[余庆县|余庆县]]、[[习水县|习水县]]
* 自治县：[[道真仡佬族苗族自治县|道真仡佬族苗族自治县]]、[[务川仡佬族苗族自治县|务川仡佬族苗族自治县]]

[[新蒲新区|新蒲新区]]是遵义市设立的经济管理区，新区管委会代管[[新蒲镇|新蒲镇]]、[[新舟镇|新舟镇]]、[[虾子镇|虾子镇]]、[[永乐镇|永乐镇]]、[[三渡镇|三渡镇]]和[[喇叭镇|喇叭镇]]。汇川区与[[遵义经济技术开发区|遵义经济技术开发区]]（[[国家级经济技术开发区|国家级]]）实行“[[一个机构多块牌子|两个牌子、一套领导]]”的管理模式。

== 经济 ==
[[File:Tian'ehu_People's_Park_and_buildings_surrounded_in_Xinpu,_Zunyi,_Guizhou.jpg|thumb]]
2018年，遵义GDP为3140亿元人民币，同比增长超过10%，全国城市GDP总量排在全国前100位，在贵州是继省会贵阳之后的第二大经济体<ref>{{cite news|url=http://zy.gog.cn/system/2017/02/08/015400390.shtml|title=2016中国城市GDP排名出炉遵义位列100|publisher=遵义在线|date=2017-02-08|accessdate=2018-01-18}}</ref>，经济总量仅次于贵阳市，继续稳居全省第二。遵义长期的发展目标是积极融入2小时重庆的经济圈，一小时贵阳经济圈，积极利用在[[重庆|重庆]]、[[贵阳|贵阳]]之间的纽带优势，发展生态农业和旅游业。遵义的白酒特别有名，中国国酒茅台就出自遵义下属[[仁怀市|仁怀市]][[茅台镇|茅台镇]]。此外，茶叶种植也有一些名气。随着交通基础设施的完善，目前已被国家划入黔中城市群。2018年1月[[渝貴铁路|渝貴铁路]]通车，自此遵义接入全国高铁网，对外交流更加便捷，经济发展迅速。

== 交通 ==
===公路===
====高速公路====
遵义市拥有四通八达的高速公路网，[[兰海高速公路|兰海高速公路]]、[[杭瑞高速公路|杭瑞高速公路]]分别从市区西部和南部穿过，市区东北部有[[绥遵高速公路|绥遵高速公路]]（'''S81'''），可前往[[绥阳县|绥阳县]]城，[[播州区|播州区]]为[[蓉遵高速公路|蓉遵高速公路]]的起点，可直达[[赤水市|赤水市]]、[[泸州市|泸州市]]等地，另外还有[[遵义北环高速|遵义一环]]（'''S02'''）环绕城区，[[银百高速公路|银百高速公路]][[道真|道真]]至[[瓮安|瓮安]]段自北向南穿过市辖区东部的[[道真|道真]]、[[正安|正安]]、[[绥阳|绥阳]]、[[湄潭|湄潭]]、[[余庆|余庆]]，以及[[德习高速公路|德习高速公路]]（'''S10'''）已完工的[[务川|务川]]至[[正安|正安]]段。2015年底，贵州实现88个县通高速公路，遵义下属的每一个县城都通了高速公路。

====市区公路====
市区内有[[326国道|326国道]]、[[210国道|210国道]]两条重要干线，另外还有南北向道路[[共青大道|共青大道]]、[[遵南大道|遵南大道]]、[[湘江大道|湘江大道]]、[[和平大道|和平大道]]、[[苟江大道|苟江大道]]、[[汇川大道|汇川大道]]将[[汇川区|汇川区]]、[[红花岗区|红花岗区]]、[[播州区|播州区]]及沿线市镇连接在一起，以及[[兴遵大道|兴遵大道]]、[[长新大道|长新大道]]、[[平安大道|平安大道]]、[[新龙大道|新龙大道]]、[[东联二线|东联二线]]将旧城区与[[新蒲镇|新蒲镇]]新开发区相联系。

====公交====
遵义市区有四通八达的公交网络，目前遵义公共交通有限责任公司经营有51条线路联络[[红花岗区|红花岗区]]、[[汇川区|汇川区]]、[[播州区|播州区]]内各地点。市辖区内各县级行政区大多有各自的公交网络（例如[[赤水市|赤水市]]）。各县亦设有客运站，每天有定点班车在各县之间、县城与市区之间行驶。

====长途客车====
遵义市内[[茅草铺|茅草铺]]、[[春天堡|春天堡]]、[[忠庄镇|忠庄镇]]、[[高桥镇|高桥镇]]、[[长征镇|长征镇]]设有长途客运站，可前往省内各市及周边各省。另外各县也设有客运站，除有前往市区的定点班车外，也有前往周边地区的客运班车。

=== 铁路 ===
遵义有[[川黔鐵路|川黔鐵路]]从城区东面经过，并设[[遵义西站|遵义西站]]，另外有[[南关镇站|南关镇站]]（原遵义南站）、[[遵义北站|遵义北站]]两个货运站。由于地處[[云貴高原|云貴高原]]，鐵路复线建設工程複雜，重慶經遵義到貴陽的鐵路至今還是單線鐵路。[[渝贵铁路|渝贵铁路]]并在遵义设新[[遵义站|遵义站]]和[[遵义南站|遵义南站]]（原苟江站），於2018年1月25日正式通车，遵义到重庆和贵阳均可在一小时内到达，接入国家高铁网。

2014年，遵义开始筹划兴建[[城市轨道交通系统|城市轨道交通系统]]，规划8条线路。<ref>[http://dsb.gzdsw.com/html/2014-02/27/content_262965.htm 遵义城区将建轨道交通]</ref>

=== 航空 ===
[[遵义新舟机场|遵义新舟机场]]位于中心城区东部35公里，已于2012年8月28日通航，现已开通北京、上海、广州、西安、三亚、昆明、杭州等地直飞航班。位于遵义下属仁怀市茅台镇的[[遵义茅台机场|遵义茅台机场]]则于2017年10月31日通航。2019年，新舟机场和茅台机场旅客吞吐量实现360万人次以上

== 生活形態 ==
遵義生活形態類似成都重慶貴陽等地，喜好吃喝玩樂，酒吧，夜市，小吃等較豐富。城市生活節奏較慢。遵义市区规模不大，但五脏俱全，购物、美食、娱乐等生活配套设施一应俱全，[[沃尔玛|沃尔玛]]、[[北京华联|北京华联]]、[[百盛|百盛]]、[[肯德基|肯德基]]等国际连锁购物早已进驻遵义。繁华程度在中国三线城市中还算不错，素有“小上海”的称号。
