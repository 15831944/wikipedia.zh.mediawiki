'''娄底市'''是[[中华人民共和国|中华人民共和国]][[湖南省|湖南省]]下辖的[[地级市|地级市]]，位于湖南省中部，是湖南的[[钢铁|钢铁]]、[[能源|能源]]、[[矿物|矿产]]和[[化工|化工]]重镇，也是湖南省内重要的铁路枢纽，[[洛湛铁路|洛湛]]、[[沪昆铁路|沪昆铁路]]、[[沪昆高铁|沪昆高铁]]三条铁路在此交汇<ref>{{Cite web|url=http://www.hnloudi.gov.cn/zjld/ldgk/ldjj/|title=娄底简介|publisher=娄底市人民政府门户网站|accessdate=2012-08-11}}</ref>。

== 历史 ==
[[明代|明]]、[[清代|清]]属[[长沙府|长沙府]]、[[宝庆府|宝庆府]]。1914年属[[湘江道|湘江道]]，1922年直属湖南省。1950年初，东部属[[益阳专区|益阳专区]]，西部属[[邵阳专区|邵阳专区]]。1952年属邵阳专区，1968年属[[邵阳地区|邵阳地区]]。1977年，析邵阳地区北部置[[涟源地区|涟源地区]]，地区行政公署驻蓝田镇。1978年，地区行政公署迁娄底镇，1980年，娄底镇改设娄底市（县级）。1982年，涟源地区改名为[[娄底地区|娄底地区]]。1999年，撤销娄底地区和县级娄底市，改设地级娄底市；原县级娄底市改置娄星区。<ref>{{Cite web|url=http://www.xzqh.org/html/show.php?contentid=15761|title=娄底市历史沿革|publisher=行政区划网|date=2010-12-28|accessdate=2011-08-11}}</ref>

== 地理 ==
娄底市位于湘中腹地，东经110°45′40"－112°31′07"，北纬27°12′31"－28°14′27"之间，西部和西南部属湘西山区；东部属湘中丘陵区，地形起伏平缓。东部[[涟水|涟水]]自西向东注入[[湘江|湘江]]，西部[[资水|资水]]由南向北汇入[[洞庭湖|洞庭湖]]。东邻[[湘潭市|湘潭市]]、[[长沙市|长沙市]]，南傍[[衡阳市|衡阳市]]，北接[[益阳市|益阳市]]，西南与[[邵阳市|邵阳市]]毗连，西北与[[怀化市|怀化市]]接壤。

=== 气候 ===
属中[[亚热带季风气候|亚热带季风气候]]区<ref>{{Cite web|url=http://www.xzqh.org/html/show.php?contentid=15762|title=娄底市概况|publisher=行政区划网|date=2010-09-01|accessdate=2011-08-11}}</ref>，全年平均气温16℃－17.3℃，年平均日照时间1538小时。

== 人口 ==
2010年[[中华人民共和国第六次全国人口普查|第六次人口普查]]初步结果显示，娄底常住人口3,785,627人，占湖南人口的5.76%居'''[[湖南省各市州人口列表|第11位]]'''，人口密度467人/[[平方千米|公里<sup>2</sup>]]；户籍人口4,323,497人，占全省户籍人口的6.11%；人口净流出量537,869人，占户籍人口的 12.44%；男女性别比为106.32比100。按受教育程度分，大学以上文化程度占总人口的6.08%，初中以上文化程度占总人口的62.53%，文盲人口仅占3.20%。辖区内共有1,091,256个家庭户，家庭户人口3,631,452人，占总人口的95.93%；家庭平均人口规模3.33人。年龄构成上，14岁及以下人口726,941人，占总人口的19.20%；15－64年人口2,681,995人，占70.85%；65岁及以上老年人口376,691人，占总人口的9.95%<ref name="2010人口普查数据">2010年湖南与娄底人口普查数据分别出自[http://www.ldtj.gov.cn/tjfx/tjgb/201605/t20160506_1325.html 《娄底市2010年第六次全国人口普查主要数据公报》]与[http://www.hntj.gov.cn/zhuanlan/6th_census/events/in_province/201105/t20110513_85176.htm 《2010年湖南省第六次全国人口普查主要数据公报》] {{webarchive|url=https://web.archive.org/web/20160304041240/http://www.hntj.gov.cn/zhuanlan/6th_census/events/in_province/201105/t20110513_85176.htm |date=2016-03-04 }}</ref>。

== 民族 ==
娄底以[[汉族|汉族]]人口为主，少数民族散居。2000年[[中华人民共和国第五次全国人口普查|第五次人口普查]]数据显示，娄底全市共有[[汉族|汉族]]人口3,778,014人，占地区人口的99.86%；少数民族5,219人，仅占地区人口的0.14%，为湖南省少数民族人口最少的地区居'''[[湖南各市州民族构成列表|第14位]]'''；中国55个少数民族中有33个民族分布。娄底过千人的少数民族有[[苗族|苗]]和[[土家族|土家族]]2个，分别为1,670和1,332人，占地区少数民族人口的比例达到32%和25.52%；居第3位的少数民族[[侗族|侗族]]有681人，占少数民族人口的13.05%。其余30个少数民族总人口1,536人，占地区少数民族人口的29.43%，其中[[回族|回]]、[[瑶族|瑶]]、[[壮族|壮]]、[[布依族|布依]]、[[满族|满]]和[[蒙古族|蒙古]]6个民族人口均在百人以上，其余24个民族人口在10人以下<ref name="2000人口普查数据">2000年第五次人口普查数据出自[http://www.hntj.gov.cn/vital_statistics/census/default.htm#part1 湖南省统计局/人口服务网（子网）-第五次人口普查数据] {{webarchive|url=https://web.archive.org/web/20160331215130/http://www.hntj.gov.cn/vital_statistics/census/default.htm |date=2016-03-31 }}，分县人口数据参考[http://www.stats.gov.cn/tjsj/pcsj/rkpc/dwcrkpc/ 国家统计局-第五次人口普查数据库]</ref>。
