'''娄底站'''位于[[中国|中国]][[湖南省|湖南省]][[娄底市|娄底市]]，始建于1959年，1997年搬迁到新址，为[[广州铁路集团|广州铁路集团]]旗下客运一等车站，主要铁路是[[沪昆铁路|沪昆铁路]]、[[洛湛铁路|洛湛铁路]]。

== 沿革 ==
娄底站始建于1959年。1997年，娄底站搬迁到新址。是沪昆铁路和洛湛铁路交汇点。离上海南站1249公里、离昆明站1411公里（沪昆线），离洛阳东站1031公里、离湛江站955公里(洛湛线)。隶属广州铁路（集团）公司娄底车务段，现为一等站。

== 车站设施 ==
娄底站建有候车站台2座，共有33条股道。1、4、5道是办理客运业务的股道。1道为1站台的落客区，4、5道为2站台落客区。其他股道属于娄底编组站，货运到发场。

== 使用情况 ==
娄底站是沪昆铁路、洛湛铁路经过的重要枢纽。每日旅客列车100次。
