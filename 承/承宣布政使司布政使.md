== 明朝 ==
[[明朝|明朝]]時，布政使是[[承宣布政使司|-{承宣布政使司}-]]（[[中国一级行政区|國家一級行政區]]，大約等於現代的「[[省|省]]」）的最高[[行政長官|行政長官]]，分左右。个别地方只置1员。（明会典卷四，官制三，外官，贵州止设左布政使一员）。明朝后期，布政司之上又设总督、巡抚，布、按二司受其节制。布政使只管经理钱粮等事，地位逐漸低下。<ref>《[[明史|明史]]•职官志》说:“初置藩司，与六部均重。布政使入为尚书、侍郎，副都御史每出为布政使。宣德、正统间犹然，自后无之。”</ref>

承宣布政使司是衙署名，其长官为布政使<ref>见《[[明史|明史]]》卷75“职官四”：“承宣布政使司。左、右布政使各一人”</ref>，全称为承宣布政使司布政使<ref>见《[[清史稿|清史稿]]》卷116“职官三”：“承宣布政使司布政使，省各一人”。</ref>，而“承宣布政使”是错误用法。明代布政使司又俗称藩司，布政使俗称藩台。

== 清朝 ==
[[清朝|清朝]]時，[[承宣布政使司|承宣布政使司]]常簡稱[[布政使司|布政使司]]，為一省最高行政長官——[[巡撫|巡撫]]的下屬，負責省內行政事務，[[從二品|從二品]]，初時設左右布政使，至[[康熙|康熙]]三年，只設布政使一人。

== 阮朝 ==
[[阮朝|阮朝]]明命十二年（1831年）和十三年（1832年），[[明命帝|明命帝]]分两次改全国[[城_(行政区划)|城]][[镇_(越南)|镇]]为省，各省设承宣布政使司，总督所在省专设布政使一员，巡抚所在省由巡抚兼任布政使。
