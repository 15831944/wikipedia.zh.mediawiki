'''藤井實彥'''（1972年－），日本[[實業家|實業家]]、[[保守派|保守派]]、[[右派|右派]]。

==簡介==
藤井實彥曾在飯店任職，自2006年從事[[管理諮詢|管理諮詢]]業，2012年後也從事[[網路漫畫|網路漫畫]]，目前經營Next Door（{{lang|ja|ネクストドアー}}）漫畫出版社、擔任X-brains（{{lang|ja|エックスブレーンズ}}）股份有限公司[[代表董事|代表董事]]，<ref name="98イースト新書">{{Cite web |url = https://www.kinokuniya.co.jp/f/dsg-01-9784781650548 |title = イースト新書 村山談話20年目の真実 著者等紹介 |publisher = [[紀伊國屋書店|紀伊國屋書店]]}}</ref><ref>{{Cite web |url =http://nextdoor-p.com/aboutus/ |title = ネクストドアー出版とは |publisher = Next Door}}</ref><ref>{{Cite web |url = http://icomix.jp/company |title = 会社概要 アイコミックス運営会社 |publisher = icomix}}</ref>以及民間團體{{Link-ja|慰安婦之真相國民運動|「慰安婦の真実」国民運動}}幹事、<ref>{{Cite web |url = https://www.sankei.com/world/news/180906/wor1809060014-n1.html |author = 田中靖人 |date = 2018年9月6日 |title = 台湾・慰安婦像で日本の民間団体が撤去要求 |publisher = [[產經新聞|產經新聞]]}}</ref>反駁計畫（{{lang|ja|論破プロジェクト}}）主席，<ref>{{Cite web |url = http://rom-pa.com/aboutus/ |title = 論破プロジェクトとは |publisher = ROM PA.com}}</ref>以「透過[[漫畫|漫畫]]向世界傳播正確的[[日本歷史|日本歷史]]」為訴求。<ref>{{Cite web |url = https://the-liberty.com/article.php?item_id=6820 |date = 2013年10月23日 |title = 韓国の慰安婦漫画に日本から「倍返し」 - 「論破プロジェクト」始動!　論破プロジェクト 実行委員長 藤井実彦氏インタビュー |publisher = {{Link-ja|The Liberty|ザ・リバティ}}}}</ref>

==事件==
2014年，在[[法國|法國]]舉辦的[[安古蘭國際漫畫節|安古蘭國際漫畫節]]，原本有[[韓國|韓國]]籌畫的[[慰安婦|慰安婦]]事件紀念特展，被藤井實彥不斷抗議，想要干擾展覽。隨後就以自己的名義，也要求在漫畫展上另開席位，要展出「慰安婦的真相」，主辦單位沒有同意，但得到日本外務大臣[[岸田文雄|岸田文雄]]、內閣官房長官[[菅義偉|菅義偉]]、駐法國大使{{Link-ja|鈴木庸一|鈴木庸一 (外交官)}}的聲援。<ref name="漫畫展">{{Cite web |url = http://www.chinatimes.com/realtimenews/20180911003994-260407 |author = 江飛宇 |date = 2018年9月11日 |title = 國際累犯!藤井實彥曾阻止慰安婦漫畫展 |publisher = [[中時電子報|中時電子報]]}}</ref>

2018年9月6日，慰安婦之真相國民運動等多個保守派團體要求撤除位於[[台南市|台南市]]的台灣首座[[慰安妇少女铜像|慰安婦銅像]]，團體代表藤井實彥被[[闭路电视|闭路电视]]拍摄到偷踹銅像，並於8日離境，[[中國國民黨|中國國民黨]]臺南市黨部主委[[謝龍介|謝龍介]]與[[中華統一促進黨|中華統一促進黨]]10日分別前往[[台北市|台北市]]的[[日本台灣交流協會|日本台灣交流協會]]表達譴責並要求道歉。<ref>{{Cite web |url = https://www.ettoday.net/news/20180910/1255182.htm |author = 賴-{于}-榛 |date = 2018年9月10日 |title = 影／竟敢偷踹慰安婦銅像！謝龍介、王炳忠10:30衝日台交流協會抗議 |publisher = [[ETtoday新聞雲|ETtoday新聞雲]]}}</ref>[[臺北駐日經濟文化代表處|臺北駐日經濟文化代表處]]代表[[謝長廷|謝長廷]]11日召開記者會，表示嚴正譴責並反對任何破壞[[臺灣與日本關係史|台日友好]]的行為，但同时称藤井的行为不代表日本人與日本政府。<ref>{{Cite web |url = https://www.upmedia.mg/news_info.php?SerialNo=47894 |author = 蘇欣儀 |date = 2018年9月11日 |title =【日人踹慰安婦銅像】謝長廷嚴正譴責 並反對任何破壞日台友好行為 |publisher = 《[[上報|上報]]》}}</ref>[[中华人民共和国外交部发言人|中华人民共和国外交部发言人]]亦表示，“要提醒个别的日本右翼分子，中国有句话，辱人者必自辱，希望他们能够自省、自重”。<ref>{{Cite web |url = http://news.sina.com.cn/o/2018-09-11/doc-ihiycyfx1533960.shtml |author = 陈慧慧 |date = 2018年9月11日 |title = 日右翼代表脚踢慰安妇头像 外交部:辱人者必自辱 |publisher = [[新浪網|新浪網]]}}</ref>慰安婦之真相國民運動主席{{Link-ja|加瀨英明|加瀬英明}}12日發表道歉聲明，表示藤井實彥已經辭職，但他仍堅持慰安婦的指控有損日本名譽，希望中國國民黨回應相關質問。<ref>{{Cite web |url = https://www.storm.mg/article/494192 |author = 國際中心 |date = 2018年9月13日 |title = 踹慰安婦銅像風波，日本道歉了！日保守派團體低頭認錯：藤井實彥已辭職 |publisher = [[風傳媒|風傳媒]]}}</ref>
