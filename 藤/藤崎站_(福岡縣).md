'''藤崎站'''（{{jpn|j=藤崎駅|hg=ふじさきえき|rm=Fujisaki eki|f=y}}）是一個位於[[日本|日本]][[福岡縣|福岡縣]][[福岡市|福岡市]][[早良區|早良區]]{{link-ja|百道|百道}}2丁目，屬於[[福岡市地下鐵|福岡市地下鐵]][[機場線_(福岡市地下鐵)|機場線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是K03。

== 歷史 ==
* 1981年（昭和56年）7月26日 - 開業。

== 車站結構 ==
[[對向式月台|對向式月台]]2面2線的[[地底車站|地底車站]]。{{link-ja|明治通 (福岡市)|明治通り (福岡市)|明治通}}藤崎交差點附近正下方地下3樓，月台位於地下22公尺，福岡市地下鐵當中最深的車站。
