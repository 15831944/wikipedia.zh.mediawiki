'''藤原京'''，[[飛鳥京|飛鳥京]]的西北部，位于今[[奈良县|奈良县]][[橿原市|橿原市]]，是日本史上最初最大的都城。而且还是[[日本|日本]]史上最初的按照[[条坊制|条坊制]]（{{lang|ja|じょうぼうせい}}）布局的中国式[[都城|都城]]，这座都城还体现了《[[周礼|周礼]]》所传达的思想。

== 概要 ==
690年（[[持統|持統]]4年）开工建设、694年（持統8年）从[[飛鳥浄御原宮|飛鳥浄御原宮]]迁都于此。据说都城建成于迁都后的第10年704年（[[景雲|景雲]]元年），而在676年（[[天武|天武]]5年）就已经开工建设了。

对于迁都的时间、一种说法是[[持統天皇|持統天皇]]在[[十二月初六|十二月初六]]（[[格里曆|格里曆]]12月27日）的上午从飛鳥浄御原宮出发开始迁走。

=== 名称 ===
「藤原京」这个名字是近代所使用的学术用语，《[[日本書紀|日本書紀]]》里面并没有记载。《日本書紀》中关于都城的记载中，[[首都|京]]指「新益京」（持統天皇六年正月十二日条）；[[皇居|宮]]指「藤原宮」。

== 規模・京域 ==
[[大和三山|大和三山]]（[[耳成山|耳成山]]、[[畝傍山|畝傍山]]、[[天香具山|天香具山]]）所包圍的區域。

== 藤原宮 ==
藤原京在九州王朝的假说中，即大宰府。

== 現状 ==
[[奈良県|奈良県]][[橿原市|橿原市]][[高殿町|高殿町]]残留有[[藤原宮|藤原宮]][[大極殿|大極殿]]（{{lang|ja|だいごくでん}}）的[[土台|土台]]、周围还存有古迹公园。藤原宮跡的6成被指定为[[日本国家特别历史遗迹|日本国家特别历史遗迹]]，藤原宮及其遗迹的发掘调查工作仍在继续。

2007年1月、日本政府将飞鸟藤原宫都相关建筑群登录入[[世界遗产|世界遗产]]申请的暂定名单中。
