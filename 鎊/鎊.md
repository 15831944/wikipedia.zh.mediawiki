'''鎊'''（'''{{lang|en|pound}}'''）或'''里拉'''（'''{{lang|it|lira}}'''）是一種[[貨幣|貨幣]]單位，最初是以一[[金衡制|金衡制磅]]重量的高純度[[銀|銀]]的價值作為基準。鎊與里拉的字源與本義相同，都是來自[[拉丁文|拉丁文]]''{{lang|la|libra}}''，[[天平|天平]]之意。鎊與磅的英文都是{{lang|en|pound}}，但翻譯成中文時，[[部首|部首]]有區別。

古時[[歐洲|歐洲]]地區貨幣，多遵從1：20：240的比率，即1[[镑|镑]]=20[[先令|先令]]=240[[便士|便士]]，或1[[里弗尔|里弗尔]]=20[[蘇勒德斯_(錢幣)|蘇爾]]=240[[第纳尔|第纳尔]]；古時與[[镑|镑]]相同價值的貨幣，還有[[索維林|索維林]]、[[里弗爾|里弗爾]]、[[法郎|法郎]]等。

* [[英國|英國]]的[[英鎊|英鎊]]（{{lang|en|Pound sterling}}或{{lang|en|Great Britain Pound}}，簡稱{{lang|en|GBP}}，代表符號為[[£|£]]），由[[英格蘭銀行|英格蘭銀行]]發行，通用於[[英格蘭|英格蘭]]、[[威爾斯|威爾斯]]、[[蘇格蘭|蘇格蘭]]以及[[北愛爾蘭|北愛爾蘭]]。同時，在[[不列顛群島|不列顛群島]]以及[[英國海外領地|英國海外領地]]所發行的貨幣皆與英鎊維持1：1的匯率。

== 使用里拉的國家 ==
* [[土耳其|土耳其]]的[[土耳其里拉|土耳其里拉]]。2005年1月1日發行[[新土耳其里拉|新土耳其里拉]]，1新里拉兌換一百萬舊里拉。
* [[黎巴嫩鎊|黎巴嫩鎊]]在本地語言中可叫做里拉。

== 歷史上曾使用鎊的國家 ==
* [[澳洲|澳洲]]的澳大利亞鎊，1966年2月14日進行貨幣十進化，被[[澳大利亞元|澳大利亞元]]取代。 
* [[愛爾蘭共和國|愛爾蘭共和國]]的愛爾蘭鎊，被[[歐元|歐元]]取代。
* [[以色列|以色列]]的以色列鎊，被[[以色列新謝克爾|以色列新謝克爾]]取代。
* [[紐西蘭|紐西蘭]]的紐西蘭鎊，1967年7月10日進行貨幣十進化，被[[紐西蘭元|紐西蘭元]]取代。
* [[尼日利亞|尼日利亞]]的奈及利亞鎊，被[[奈拉|奈拉]]取代。
* [[南非|南非]]的南非鎊，1961年2月14日進行貨幣十進化，被[[南非蘭特|南非蘭特]]取代。
* [[尼日利亞|尼日利亞]]、[[迦納|迦納]]、英屬[[喀麥隆|喀麥隆]]、[[塞拉利昂|塞拉利昂]]、[[甘比亞|甘比亞]]的[[非洲中央银行及货币列表|西非鎊]]被各國貨幣取代。
* [[賽普勒斯|賽普勒斯]]的[[賽普勒斯鎊|賽普勒斯鎊]]，被歐元取代。

== 歷史上曾使用里拉的國家 ==
* [[義大利|義大利]]的[[義大利里拉|義大利里拉]]，被[[歐元|歐元]]取代。
* [[聖馬利諾|聖馬利諾]]的聖馬利諾里拉，被歐元取代。
* [[梵蒂岡|梵蒂岡]]的梵蒂冈里拉，被歐元取代。
* [[馬爾他|馬爾他]]的[[马耳他镑|馬爾他里拉]]，之前曾稱為馬爾他鎊，被歐元取代。
