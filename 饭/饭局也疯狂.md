《'''饭局也疯狂'''》（{{lang-en|'''Dinner Party'''}}）2012年1月23日上映于[[中国大陆|中国大陆]]的[[喜剧片|喜剧片]]，由著名[[小品|小品]]演员范伟主演。

== 剧情 ==
电影讲述了中国独有的“[[饭局|饭局]]”现象，深刻解剖了饭局文化背后隐藏的[[潜规则|潜规则]]和社会阴暗面。<ref name="duck " >{{cite web |url= http://news.163.com/12/0208/02/7PN4H6VL00014AED.html|title=《饭局也疯狂》挤兑了谁？ |publisher=2012-02-08 |accessdate=网易 |work= |language =}}</ref>

== 主题 ==
[[导演|导演]]尚敬谈到电影的主题时说：“我不想隔靴搔痒，也不想停留在神头鬼脸的人物出场闹一闹”，“我希望它是直指现实的，无情地嘲讽人性的弱点和社会上的丑恶现象，保持原创的激情、批判的锋芒，”他说，“喜剧必须是一把刀子”。<ref name="duck " />

== 花絮 ==
电影播出后，观众认为范伟扮演的[[国学大师|国学大师]]有讽刺[[百家讲坛|百家讲坛]]讲师[[于丹|于丹]]的意味。<ref>{{cite web |url= http://ent.people.com.cn/GB/17039480.html|title=《饭局也疯狂》范伟角色惹争议 被疑讽刺于丹 |publisher=人民网 |accessdate=2012年2月7日 |work= |language =}}</ref>

电影的搞笑成分和剧情吸引了大量八零后和九零后，播出三天票房就达到了两千万。<ref>{{cite web |url=http://ent.ifeng.com/movie/news/mainland/detail_2012_01/27/12154680_0.shtml |title= 《饭局也疯狂》票房破两千万 现实题材辛辣讽刺|publisher=凤凰网 |accessdate=2012年1月27日 |work= |language =}}</ref>
