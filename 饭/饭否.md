'''饭否'''是[[中国大陆|中国大陆]]地区首批提供[[微型博客|微型博客]]服务的类[[Twitter|Twitter]]网站之一，用户可以通过网页、[[WAP|WAP]]页面、手机[[短信|短信]]/[[彩信|彩信]]（现已不可用）、[[IM|IM]]软件（包括[[MSN|MSN]]、[[Google_Talk|Google Talk]]）和上百种[[API|API]]应用<ref>{{cite web|url=http://dev.fanfou.com/2009/04/08/fanfouapi100/ |title=饭否第三方API应用超过100个 | accessdate=2009-04-09 |publisher=饭否 |language=zh-hans }}</ref>发布消息（限140字以内）或上传图片。用户间通过互相关注、私信、或@对话等方式互动。

网站由[[校内网|人人网]]（原校内网）的创始人[[王兴_(网站创办者)|王兴]]于2007年5月12日建立，基于[[LAMP|LAMP]]架构<ref>{{cite web |url=http://www.chinaz.com/Webmaster/Club/0QG43452007.html |title=对话王兴：饭否玩的是一种感觉 |accessdate=2007-08-17 |publisher=草根网 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20071013141425/http://chinaz.com/Webmaster/Club/0QG43452007.html |archivedate=2007-10-13 }}</ref>。饭否团队中有穆荣均、刘丹（已离职）、郭万怀、杨俊（已离职）、王悦蒙等人<ref>{{cite web|url=http://help.fanfou.com/about.html |title=关于饭否_饭否 | accessdate=2009-02-20 |publisher=饭否 |language=zh-hans }}</ref>。自2009年7月7日起，该网站一度[[中华人民共和国网络审查|被关闭]]。2010年11月11日，饭否的网页重新可以打开，显示用户头像墙以及 “2010.11” 的字样。11月25日晚，饭否网站再度开放登入。

== 名称 ==
“饭否”一词来源于[[辛弃疾|辛弃疾]]的《[[永遇乐|永遇乐·京口北固亭怀古]]》中的“廉颇老矣，尚能饭否？”引用其中的“饭否”二字，在这里体现了[[微型博客|微型博客]]的“唠叨精神”，有如[[中国人|中国人]]喜欢见人就唠叨一句“你吃饭了吗？”

== 影响 ==
在饭否的初期阶段，[[中国大陆|中国大陆]]与Twitter类似的中文网站有六七家，其中饭否是各方面功能较为完备，最接近Twitter的一个。与Twitter相比，中文微博客为大多数不习惯英文的中国用户提供了更好的选择。

2009年上半年，饭否的用户数从年初的30万左右激增到了百万。同年6月2日，[[惠普|惠普]]成为饭否首个企业付费用户，开始获得第一笔收入。与此同时，[[陈丹青|陈丹青]]、[[艾未未|艾未未]]、[[梁文道|梁文道]]、[[连岳|连岳]]等一批文化名人的加入，都带动了饭否的快速成长。 <ref>{{cite news|url=http://gcontent.nddaily.com/3/32/3323fe11e9595c09/Blog/198/dabe6f.html|title=Twitter时代：人人都可发新闻|publisher=南方都市报|date=2009-07-12|accessdate=2009-09-22|language=zh-hans|deadurl=yes|archiveurl=http://telegra.ph/%E9%B2%9C%E6%A9%99%E4%BA%92%E5%8A%A8-%E5%8D%97%E9%83%BD%E7%BD%91-%E5%8D%97%E6%96%B9%E9%83%BD%E5%B8%82%E6%8A%A5-%E6%96%B0%E9%97%BB%E4%BA%92%E5%8A%A8%E7%BD%91%E7%AB%99-%E5%8D%97%E9%83%BD%E6%95%B0%E5%AD%97%E6%8A%A5-03-22|archivedate=2009-09-16}}</ref>

在2009年下半年饭否[[中华人民共和国网络审查|被关闭]]以后，[[新浪|新浪]]、[[网易|网易]]、[[腾讯|腾讯]]、[[搜狐|搜狐]]等国内门户网站先后推出了[[微博客|微博客]]服务，其中[[新浪微博|新浪微博]]成为发展最快的一家，众多前饭否用户因此转移到了[[新浪微博|新浪微博]]。

* 2007年5月，饭否正式上线，提供[[API|API]]、私信、[[博客|博客]]插件、[[短信|短信]]发布消息和[[IM|IM]]绑定发布等基本功能。
* 2007年6月－7月，新增@功能、收藏分享功能、[[彩信|彩信]]发布功能和[[Flash|Flash]]插件。
* 2008年12月－2009年2月，去除Logo“测试”字样，结束测试阶段。开始提供自定义模板、全站搜索、回复等功能，改进@功能和Flash插件，更新API。
* 2009年4月－5月，饭否排行榜上线，饭否拍拍发布，新增饭否热词和博客导入功能。
* 2009年6月2日，[[惠普|惠普]]成为饭否首位付费用户，王兴称这是网站获得的首笔收入。<ref>{{cite web|url=http://it.sohu.com/20090602/n264287711.shtml |title=惠普成为中国微博客网站饭否首个付费用户 |accessdate=2009-02-20 |publisher=搜狐IT |language=zh-hans }}</ref>
* 2009年6月3日－6月5日，[[中国互联网维护日|中国互联网维护日]]，网站暂停服务。
* 2009年7月5日，[[乌鲁木齐市|乌鲁木齐市]]发生[[2009年7月乌鲁木齐骚乱|骚乱事件]]。7月6日，[[路透社|路透社]]记者在饭否以“乌鲁木齐”作为关键词搜索，无法显示任何结果。<ref>{{cite news|url=http://www.nydailynews.com/news/us_world/2009/07/06/2009-07-06_aaweraweraweraraew.html|title=Social networking sites Twitter, YouTube are force for revolution|publisher=NYDailyNews|date=2009-07-06|accessdate=2009-09-13|language=en}}</ref>
* 2009年7月7日晚11时44分开始，饭否全站无法更新消息，输入框提示信息为“抱歉，饭否全站暂时不能更新消息，预计7月10日晚恢复。”私信功能并没有被屏蔽。<ref>{{cite news|url=http://www.iterasi.net/openviewer.aspx?sqrlitid=na92gi0mh0m6i5uo2jow1q|title=饭否歇了，Twitter的晚餐在哪里？|publisher=南都周刊|date=2009-07-17|accessdate=2009-09-22|language=zh-hans}}</ref>7月8日零点之后，饭否首页无法登陆，但可以访问WAP版页面和iPhone版页面。7月8日下午16时48分许，饭否各域名均无法解析。<ref>{{cite web|url=http://tech.163.com/09/0713/11/5E3LEC6O000915BF.html |title=微博客网站饭否暂停服务 原因不明|accessdate=2009-07-13 |publisher=网易科技 |language=zh-hans }}</ref>
* 2009年7月22日，中国大陆饭否同类迷你博客网站[[叽歪|叽歪]]和[[嘀咕|嘀咕]]也被关闭。
* 2009年8月11日，饭否团队在其博客上承诺“会带着一个更好的饭否回来”。<ref>{{cite web |url=http://fanfoublog.com/2009/08/11/road/ |title=路在脚下 |accessdate=2009-08-11 |publisher=饭否团队 |language=zh-hans |archive-url=https://web.archive.org/web/20090814114955/http://fanfoublog.com/2009/08/11/road/ |archive-date=2009-08-14 |dead-url=yes }}</ref>
* 2009年9月2日，[[百度|百度]]饭否[[百度贴吧|贴吧]]被关闭（叽歪吧和嘀咕吧也同时被关闭），[[百度百科|百度百科]]上的饭否词条被删除，[[豆瓣|豆瓣]]上的“饭否”“饭否官方”“饭否观光团”“饭否话痨圈”等相关小组被解散。10月2日，百度饭否贴吧重新开放。
* 2009年10月20日，饭否团队在其博客上表示“停了105天，走了2位同事，但我们还在。饭否会回来。”。<ref>{{cite web |url=http://fanfoublog.com/2009/10/20/fanfouupdates200910/ |title=饭否会回来 |accessdate=2009-10-20 |publisher=饭否团队 |language=zh-hans |archive-url=https://web.archive.org/web/20101118081519/http://fanfoublog.com/2009/10/20/fanfouupdates200910/ |archive-date=2010-11-18 |dead-url=yes }}</ref>10月22日起，[http://fanfoublog.com/ 饭否团队博客]无法访问。

* 2010年11月10日，有媒体报道称饭否有望恢复访问，并且已经得到中国经济网的投资。饭否创始人王兴对此消息不置可否。<ref>{{cite web|url=http://www.techweb.com.cn/internet/2010-11-09/718325.shtml |title=消息称独立微博饭否网将恢复正常访问 |accessdate=2010-11-09 |publisher=TechWeb |language=zh-hans }}</ref><ref>{{cite web|url=http://www.techweb.com.cn/internet/2010-11-09/718394.shtml |title=独立微博饭否网即将回归：要靠创新打开局面 |accessdate=2010-11-09 |publisher=TechWeb |language=zh-hans }}</ref>
