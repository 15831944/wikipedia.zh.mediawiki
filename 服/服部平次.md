'''服部平次'''（日語：{{lang|ja|服部 平次}}/{{lang|ja|はっとり へいじ}}，Hattori Heiji），是[[日本|日本]][[漫畫|漫畫]]和[[動畫|動畫]]作品《[[名偵探柯南|名偵探柯南]]》中的[[名偵探柯南角色列表|虛構角色]] 。

命名來自《{{tsl|ja|探偵物語|偵探物語 (電視劇)}}》中的'''服部'''刑事及{{tsl|ja|野村胡堂}}的小說《[[錢形平次捕物控|錢形平次捕物控]]》中的主角錢形'''平次'''。

{{portal|名偵探柯南}}

== 人物 ==
改方學園高中部二年級的學生，關西的高中生偵探，與[[工藤新一|工藤新一]]齊名，並稱為「'''關東工藤，關西服部'''」（但自認為「關西服部，關東工藤」才是正確的）<ref group="參" name="第10卷">漫畫第10卷，TV 48-49《外交官殺人事件》</ref>。父親[[服部平藏|服部平藏]]是{{tsl|ja|大阪府警察}}{{tsl|ja|本部長}}，母親是[[服部靜華|服部靜華]]（舊姓池波），與[[遠山和葉|遠山和葉]]是[[青梅竹馬|青梅竹馬]]的同班同學。

皮膚黝黑（自稱是遺傳自爺爺）<ref group="參">漫畫第28卷，TV 221《滿口謊言的委託人（後篇）》</ref>、具有濃眉及[[美人尖|美人尖]]、說話時帶有引以自豪的濃厚[[關西腔|關西腔]]，外貌相當英俊。有很強的陽光氣場，有著典型的武士氣質，極富有男子氣概和男性魅力。現與父母居住在[[大阪府|大阪府]][[寢屋川市|寢屋川市]]，其住處為傳統[[日式建築|日式建築]]。

曾一直到處尋找新一，想比較兩人的推理能力。但新一早已縮小變成柯南，所以無法如願。在一次的事件之中，意外察覺柯南驚人的推理能力與說話口氣和新一相仿，所以認定兩者為同一人而試探之<ref group="參" name="第10卷"/>。後來柯南為了在破案時不被他發現，決定麻醉他並以其名義破案，卻因此被識破真實身分<ref group="參" name="第13卷">漫畫第13卷，TV 57-58《福爾摩斯迷殺人事件》</ref>。

從此以後，兩人之間有著兄弟般的友情和默契，共同查案時也幾乎形影不離。另外，他經常不自覺地叫柯南為「工藤」，因而造成不少麻煩，還好最後都能裝蒜掩飾過去。也曾偽裝成新一的模樣，幫快要被[[毛利兰|毛利蘭]]識破身分的柯南解圍，但還是被突然出現的[[遠山和葉|遠山和葉]]看穿<ref group="參">漫畫第26卷，TV 190《生死一瞬間 第三個選擇》</ref>。

平次首次登場時，送給毛利家的[[白酒_(蒸餾酒)|白乾酒]]不小心被感冒的柯南喝了一些，卻意外地讓柯南在短時間內變回新一<ref group="參" name="第10卷"/>，也因此成為後來[[灰原哀|灰原哀]]研製解藥不可或缺的參考成分。

== 形象 ==
性格開朗、活潑、陽光、熱血、體貼、衝動<ref group="參">漫畫第54卷，TV 479《與服部平次共度的3天》</ref>好強、不服輸（曾被父親平藏用激將法激起鬥志）<ref group="參" name="第31卷">漫畫第31卷，TV 263《大阪雙重疑案 浪花劍客與太閣城》</ref>，具有大阪人的豪爽與自豪感。

擅長推理、調查刑案、[[劍道|劍道]]（在學校是劍道社社長）<ref group="參" name="第31卷"/>、游泳、潛水、滑雪<ref group="參">漫畫第50-51卷，TV 490《服部平次vs工藤新一 滑雪場的推理對決》</ref>、猜[[比基尼泳衣|比基尼]]花色<ref group="參">漫畫第35卷，TV 291《孤島公主與海龍宮（事件篇）》</ref>、裝蒜（自稱）<ref group="參">漫畫第16卷，TV 78《名家連續慘死事件》</ref>，擁有高超的越野摩托車車技<ref group="參" name="M7">M7《[[迷宮的十字路口|迷宮的十字路口]]》</ref>，英語也說得很流利<ref group="參">漫畫第34卷，TV 278《英文老師vs關西名偵探（後篇）》</ref>，對[[日本傳統色系|日本傳統顏色]]也相當了解<ref group="參">[[名偵探柯南特別篇|漫畫特別篇]]第42卷</ref>；不擅長料理（和柯南一起切菜沒切斷）<ref group="參">漫畫第47卷，TV 406《柯南平次的推理魔術（戲法篇）》</ref>、足球<ref group="參">漫畫第29卷，TV 239《大阪3K事件（後篇）》</ref>。

喜歡的偵探是[[艾勒里·昆恩|艾勒里·昆恩]]，對[[福爾摩斯|福爾摩斯]]並沒有興趣<ref group="參">漫畫第13卷，TV 57《福爾摩斯迷殺人事件（前篇）》</ref>；喜歡的食物有[[咖哩飯|咖哩飯]]、[[章魚燒|章魚燒]]、[[御好燒|大阪燒]]等等。

害怕[[遠山和葉|和葉]]掉眼淚<ref group="參">漫畫第43卷，TV 381-382《誰是推理王》</ref>，討厭和葉被其他男人搭訕<ref group="參">TV 291《孤島公主與海龍宮（事件篇）》、406-407《柯南平次的推理魔術》、763-764《柯南與平次 戀愛的暗號》</ref>、蹩腳的關西腔<ref group="參">TV 49《外交官殺人事件（前篇）》、58《福爾摩斯迷殺人事件（後篇）》、651《柯南vs平次 東西偵探推理對決》</ref>。

與工藤新一的成熟穩重相比，平次的個性較為活潑和隨性，也更調皮。

上學所穿的制服為典型的[[日本校服|日本學生服]]。平時喜歡穿各種休閒風格服裝，並喜歡反戴著一頂有著“SAX”（[[芝加哥白襪|White Sox]]的變形）字樣的棒球帽。當要開始查案時，會習慣將原本反戴的帽子戴正；破案後才會把帽子反戴回去<ref group="參">TV 479《與服部平次共度的3天》、656-660《毒和幻的設計》</ref>。

== 情感 ==
和新一同樣有個[[青梅竹馬|青梅竹馬]]的「初戀情人」—[[遠山和葉|遠山和葉]]；小時候兩人曾因玩手銬，卻意外被銬在一起而引起大騷動，最後甚至連上廁所、洗澡時都在一起<ref group="參" name="第19卷"/>。後來才發現自己的初戀情人其實就是和葉<ref group="參" name="M7"/>。此外也曾經救過和葉幾次<ref group="參">TV 166-168《鳥取蜘蛛之家的古怪》、222-224《人魚失蹤記》、323-324《服部平次一籌莫展》、610-613《犬伏城 炎之魔犬》、830-832《殭屍包圍的別墅》、M17《[[絕海的偵探|絕海的偵探]]》</ref>。

雖然兩人堅決否認彼此為「男女朋友」，但實際上並非如此。平次隨身攜帶和葉送給他的護身符（內藏小時候把兩個人銬在一起的手銬碎片），而護身符每次也都能讓遭逢危難的他保住性命，也曾救過柯南<ref group="參" name="第19卷"/>。

與和葉有着戀人已滿的關係，但雙方卻未曾正式對彼此告白，而平次對感情方面相當遲鈍，曾一度認為和葉是他的手下<ref group="參">漫畫第47卷，TV 408《柯南平次的推理魔術（解決篇）》</ref>。和葉也曾向平次告白，但平次本人卻未聽到<ref group="參">漫畫第75卷，TV 655《毒和幻的設計（Illusion）》</ref>，後來在緊急狀況下對和葉脫口說出「你要對我的和葉做什麼？」，因而明白自己對和葉的心意<ref group="參">漫畫第83卷，TV 764《柯南與平次 戀愛的暗號（後篇）》</ref>。此後就暗自決定要找一個合適的地方告白，但總是受到外來的阻饒而失敗<ref group="參">TV 832《殭屍包圍的別墅（後篇）》、874《柯南與平次的鵺傳說（解決篇）》</ref>。
