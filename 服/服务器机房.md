'''服务器机房'''（{{lang-en|server room}}）是为[[计算机|计算机]][[服务器|服务器]]持续运行而设计的房间  通常装有[[空氣調節|空氣調節]]。专供此目的的建筑物或站点被称为[[数据中心|数据中心]]。

服务器机房中的计算机通常使用[[无头计算机|无头计算机]]，通过如[[KVM切换器|KVM切换器]]或[[Secure_Shell|Secure Shell]]（ssh）、[[VNC|VNC]]、[[远程桌面软件|远程桌面]]等{{tsl|en|Remote administration||远程管理}}软件来远程操作。<ref>Learnthat - [https://web.archive.org/web/20090226230842/http://www.learnthat.com/define/view.asp?id=1996 server room Definition]</ref><ref>Free Online Encyclopedia - [http://encyclopedia2.thefreedictionary.com/server+room server room]</ref><ref>ISP.webopedia.com - [http://isp.webopedia.com/TERM/L/lights_out_server_room.html lights out server room]</ref><ref>[//en.wikipedia.org/wiki/TechRepublic TechRepublic] - [http://content.techrepublic.com.com/2346-10878_11-2044.html What not to do in a server room] {{Wayback|url=http://content.techrepublic.com.com/2346-10878_11-2044.html |date=20090307031024 }}</ref><ref>[//en.wikipedia.org/wiki/CNET_Networks CNET Networks] - [http://news.cnet.com/2300-1010_3-6211253-1.html Photos: Server room cabling overhaul] {{Webarchive|url=https://archive.is/20120714003121/http://news.cnet.com/2300-1010_3-6211253-1.html |date=2012年7月14日 }}</ref>

气候是影响服务器机房能源消耗与环境影响的重要因素之一。气候有利于降温、有大量可再生电力的地区能降低机房带来的环境影响。有此条件的国家、地区有利于吸引企业在当地建设服务器机房。

=== 位置 ===
在考虑房间内布局前，服务器机房的位置是首要考虑因素。大多数设计师认为，服务器机房一般不应建设在有墙壁为建筑物外墙的地方。因为外墙可能比较潮湿，并可能内含可能发生爆裂或渗漏的水管。避免通向外界的窗户也能避免安全和运行环境风险。如果集中式的服务器机房不可行，则逐层的服务器机柜也是可考虑的方案。

除了外墙风险，设计师也要评估机房附近的任何潜在干扰源，例如无线电发射机、发电厂、电梯房等电气干扰。

其他设计时要考虑因素包括：房间大小，门的尺寸和入口坡道（用于设备进出），线缆管理、物理安全和出入管理。

=== 空调 ===
计算机运行时会产生热量，并且对热量、湿度和灰尘很敏感，但还需要很高的弹性和故障转移要求。在严格的公差范围内保持稳定的温度和湿度对IT系统可靠性至关重要。机房室温适宜在18-27°C（64-80°F）之间，湿度在40%-60% rH之间。<ref name="ServersCheck">{{cite web|url=https://serverscheck.com/sensors/temperature_best_practices.asp|title=Best Practices for data center monitoring and server room monitoring|accessdate=2016-10-07|author=ServersCheck}}</ref>
