'''將軍澳公共圖書館'''位於[[香港|香港]][[新界|新界]][[西貢區|西貢區]][[將軍澳|將軍澳]][[寶琳|寶琳]]運隆路9號，為組成將軍澳區內最大綜合場館的設施之一，毗鄰[[將軍澳體育館|將軍澳體育館]]及[[將軍澳游泳池|將軍澳游泳池]]，圖書館提供成人圖書館、兒童圖書館、參考圖書館、多媒體圖書館、電腦資訊中心、報刊閱讀室、推廣活動室及學生自修室的設備。圖書館的外型像一個有蓋的[[羅馬競技場|羅馬競技場]]<ref>[http://www.fso-createhk.gov.hk/tc/locations_search_details.php?File_No=00726&search_type=1 拍攝場地資料庫]</ref>。

==簡介==
將軍澳公共圖書館，於1999年動工興建<ref>[http://www.yearbook.gov.hk/1999/b5/13/13_14_content.htm 建築工程的發展-文娛康樂]香港年報1999</ref>，2001年7月21日正式開放投入服務<ref name = "gov">[http://www.info.gov.hk/gia/general/200107/20/0720145.htm 將軍澳公共圖書館明天投入服務]2001年7月20日，香港政府新聞網</ref>，面積2200平方米<ref name = "gov"/>，建築費用為5300萬港元<ref>[http://www.info.gov.hk/gia/general/200106/11/0611249.htm 將軍澳最大康樂及圖書館綜合場館設施陸續投入服務]2001年6月11日，香港政府新聞網</ref>，視聽資料及中英文藏書約共17萬冊<ref name = "gov"/>。
