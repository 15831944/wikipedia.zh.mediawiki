'''將軍澳站'''（{{lang-en|'''Tseung Kwan O Station'''}}）是[[香港|香港]][[港鐵|港鐵]][[將軍澳綫|將軍澳綫]]的一個地下車站，位於[[新界|新界]][[西貢區|西貢區]][[將軍澳市中心|將軍澳市中心]][[寶邑路|寶邑路]]一帶，於2002年8月18日正式啟用。

==車站設計==
將軍澳站西面將軍澳綫管道下層設有觀塘綫越位隧道<ref name="KTL_LHS">{{cite web|url=http://www.legco.gov.hk/yr08-09/chinese/panels/tp/papers/tp0424cb1-2364-1-c.pdf|title=CB(1)2364/08-09(01)號文件　將軍澳支綫（二期）服務補充資料|date=2009年7月|publisher=香港立法會|language=zh-hant}}</ref>。車站由國際建築事務所[[Aedas|Aedas]]設計。

大堂內設有不同類型的商店供乘客購物，例如有[[便利商店|便利店]]、糖果店、[[麵包糕餅店|麵包糕餅店]]、[[時裝店|時裝店]]等，所有商店主要分佈在非收費區內，以及不少自助服務設施供乘客享用，包括[[自動櫃員機|自動櫃員機]]、[[自動販賣機|自動售賣機]]和自動照相機<ref name="shop"/>。客務中心位於大堂中央C出入口前，旁邊設有[[香港郵政|香港郵政]][[郵箱|郵箱]]方便乘客在車站內投寄郵件<ref>{{cite web|url=http://www.mtr.com.hk/chi/getting_around/post.html|title=鐵路四處遊－郵箱服務|work=港鐵網頁|publisher=港鐵公司|accessdate=2014年4月14日|language=zh-hant}}</ref><ref>{{cite web|url=http://app1.hongkongpost.hk/CGI/location/box_content.jsp?region=%27New%20Territories%27&lang=zh_TW#saikung |title=郵箱位置表－西貢區 |work=香港郵政網頁 |publisher=香港郵政 |accessdate=2014年4月14日 |language=zh-hant |deadurl=yes |archiveurl=https://web.archive.org/web/20140415110401/http://app1.hongkongpost.hk/CGI/location/box_content.jsp?region=%27New%20Territories%27&lang=zh_TW |archivedate=2014年4月15日 }}</ref>。而B2出入口旁亦設有「[[會員服務站|會員服務站]]」<ref>{{cite web|url=http://www.mtr.com.hk/ch/customer/services/csp.html|title=鐵路四處遊－「會員服務站」|work=港鐵網頁|publisher=港鐵公司|accessdate=2016年9月26日|language=zh-hant}}</ref>。

另外，將軍澳站大堂近C出口设有[[港鐵文化|艺术墙]]《層層叠叠的傳統》。觀衆可領略繪畫過程前歷史的變遷。

===出入口===
將軍澳站现时设有5出入口供乘客进出。

另外，C出口在將軍澳站啟用初期為預留出入口，左右兩旁曾為[[自動售票機|自動售票機]]的位置（現已改建為商鋪）。該位置連接[[PopCorn|PopCorn]]商場一期、天晉、[[香港九龍東皇冠假日酒店|香港九龍東皇冠假日酒店]]及[[香港九龍東智選假日酒店|香港九龍東智選假日酒店]]，方便乘客不用繞經A或B出入口的行人路前往。站內店鋪亦已配合改建，C出入口已於2012年3月31日隨PopCorn商場開業而啟用<ref>{{cite web|url=http://mtr.com.hk/eng/corporate/file_rep/PR-12-024-C.pdf|title=港鐵將軍澳站開放新出入口|date=2012年3月31日|publisher=港鐵公司|language=zh-hant}}</ref>。

==利用狀況==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Tsang Kwan O Station Concourse 2010.jpg | caption1 = 2010年的大堂，當時尚未更換願景顯示屏
| image2 = Tsang Kwan O Station Exit C 201009.jpg | caption2 = C出入口預留位置（2010年9月）
| image3 = Tseung Kwan O Station 2017 08 part5.jpg | caption3 = 願景顯示屏（班次顯示器）
}}
將軍澳站啟用初期，因鄰近的住宅（[[寶盈花園|寶盈花園]]、[[將軍澳中心|將軍澳中心]]、[[將軍澳廣場|將軍澳廣場]]、[[君傲灣|君傲灣]]和[[天晉|天晉]]等）仍未完全入伙，距離[[尚德邨|尚德邨]]較遠，因此客量偏低，加上車站附近有不少建築地盤，亦有不少工程車輛出入，路面空氣環境惡劣，部分居民仍選擇以巴士代步。

自[[將軍澳市中心|將軍澳市中心]]周邊住宅陸續入伙及商場啟用、通往尚德邨的一段唐俊街行人路加設上蓋及城巴開辦往返[[將軍澳電視廣播城|將軍澳電視廣播城]]的穿梭-{巴士}-後，客量已大幅增加。

本站與[[調景嶺站|調景嶺站]]相差僅約800米，僅次於[[太子站|太子站]]與[[旺角站|旺角站]]之間，對部分居於兩站之間（將軍澳中心及唐明苑）的乘客來説，他們也樂於步行往來調景嶺站及本站，以便節省車費及轉少一次車。

由於將軍澳綫設有[[寶琳站|寶琳站]]及[[康城站_(香港)|康城站]]兩個上行尾站，而前往以上兩個總站的列車均會停靠在本站的一號月台，因此從康城站前往寶琳站及[[坑口站_(香港)|坑口站]]的乘客抵達本站後，如未看清楚就跑上對面的列車，或會誤乘前往康城站的列車因而返回原處而誤了行程，有網民更以「不能逃出康城」創作了網上笑話。
