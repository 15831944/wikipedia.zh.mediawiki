'''將軍區'''位於[[臺灣|臺灣]][[臺南市|臺南市]]西部偏北的沿海地區，北臨[[北門區|北門區]]，東鄰[[學甲區|學甲區]]、[[佳里區|佳里區]]，西濱[[臺灣海峽|臺灣海峽]]，南接[[七股區|七股區]]。

本區位於濱海地帶，地多[[潟湖|潟湖]]、灘地，[[將軍溪|將軍溪]]自本區與北門區的交界入海，氣候上則屬[[熱帶季風氣候|熱帶季風氣候]]，產業上以[[農業|農業]]及[[漁業|漁業]]為主。[[胡蘿蔔|胡蘿蔔]]為本地特產，有「胡蘿蔔之鄉」的美譽，區徽上的圖騰也以胡蘿蔔為代表。近年來，[[牛蒡|牛蒡]]的農產種植也非常興盛。

== 歷史 ==
'''將軍區'''舊名「漚汪」（[[白話字|白話字]]、[[台羅|台羅]]：Au-âng），現在有不少臺南人仍習慣稱呼此舊名，至於「將軍」之名則相傳與[[施琅|施琅]]有關。1683年，施琅率領清兵攻佔[[臺灣|臺灣]]，[[清|清廷]]以施琅靖臺有功，以跑馬三日為其業地，西起[[馬沙溝|馬沙溝]]，東至烏山頭，不料馬跑至今將軍-{庄}-一帶便斷腳蹄，因而在此地建「將軍府」，所經之田園成為「施侯租」，之後施琅率其族人以及吳王兩姓親戚入墾，因而形成「將軍-{庄}-」。<ref>[http://net2.yhsh.tn.edu.tw/~game/ind.htm 漚汪人.台灣情] , 臺南市私立瀛海中學 [http://www.yhsh.tn.edu.tw/ Ying Hai High School]</ref>當地人亦常簡稱為「將-{庄}-」。

1920年臺灣地方改制，於此地設「[[將軍庄|將軍-{庄}-]]」，劃歸[[臺南州|臺南州]][[北門郡|北門郡]]管轄，戰後改設[[臺南縣|臺南縣]]將軍鄉，2010年12月25日改稱將軍區。另外，區治設於漚汪，而非將軍（將富里、將貴里）。

== 名人 ==
===地方名人===
* 林崑岡：原名碧玉，字爾音，漚汪人，1832年生，1895年逝世。林崑岡曾任漚汪文衡殿董事，為漚汪堡抗日運動領導者，於1895年10月18日與日軍戰鬥中中彈自刎身亡。2005年7月29日，林崑岡紀念館於1895年林氏誓師抗日之漚汪文衡殿內成立。
* 林石：字成玉，1886年生，1938年逝世。林石為[[漢學|漢學]]家，祖籍西甲，早年曾在外受聘為家庭教師及經營縫紉機買賣、堪輿，晚年返回西甲老家定居。林石幼時受教於其父林向前，精通漢學，致力於地方漢學傳承。
* [[林玉体|-{林玉体}-]]：西甲人，1939年生。林玉体為美國愛荷華大學哲學博士、臺灣教育學者，曾任[[考試院|考試院]]考試委員。
* 林照：1892年生，1956年逝世。林照為將軍鄉民代表會第一屆代表主席。
* 吳惠林：漚汪人，1949年生。現為中華經濟研究院研究員。
* [[吳新榮|吳新榮]]：號震瀛，將庄人，1907年生，1967年逝世。吳新榮為[[鹽分地帶|鹽分地帶]]文學家，同時為醫師兼政治人物，曾謂「醫學為本業，文學為情婦」。吳新榮為[[台灣日治時代|日治時代]]漚汪公學校第一屆畢業生。
* 吳全成：筆名「潛誠」，[[馬沙溝|馬沙溝]]人，1949年生，1999年逝世。吳全成為美國[[華盛頓大學|華盛頓大學]]比較文學博士，[[臺灣文學|臺灣文學]]學者，以研究[[愛爾蘭|愛爾蘭]]詩人[[葉慈|葉慈]]（W.B.Yeats,1865-1939）聞名。曾於1999年以《感性定位：文學的想像與介入》一書獲「巫永福文學評論獎」。
* [[陳奇祿|陳奇祿]]：巷口社仔人，1923年生。現為[[中央研究院|中央研究院]]民族學研究所院士、吳三連臺灣史料基金會董事長。
* [[陳冠學|陳冠學]]：1934年生，祖籍西甲。陳冠學為臺灣文學家，著有《田園之秋》、《老台灣》等書。

===歷史名人===
* 黃清舞(1905-1994)：獲選台南市政治類歷史名人，既是醫師又是政治家，曾5度當選將軍鄉長，擔任鄉長達15年，擅長書法、音律，藏書豐富有「圖書館」之美譽。黃清舞的遂園，位在將軍區漚汪里，建於民國33年(1944年)，民國70年代後期漸漸荒廢，由企業家林振豐買下，整修更名為「方圓美術館」。臺南市府於民國106年1月7日於黃清舞故居舉行紀念牌的掛牌儀式。

== 行政區劃與聚落 ==
將軍區的村落大致上分布零散，但主要集中於東半部較內陸之地區。最大的聚落為東南部的「漚汪」，同時也是區治所在地之市區（不在同名之「將軍」聚落）；「將軍」則位於本區中部地帶。本區地名村落之特色是可分為三個部分：四埔、四甲、四寮。「四埔」即將軍、北埔、山仔腳、馬沙溝的「四埔吳」；「四甲」為中甲、東甲、西甲、北甲的「漚汪四甲」；四寮則指頂寮、忠寮（中寮）、下寮和崁頭寮的「苓仔寮四寮」。

[[台灣戰後時期|戰後]]依[[台灣日治時期|日治時代]]的[[保甲|保甲]]而設有24個村；1978年改行大村制：西湖村併入西和村，北嘉村與昌平村併為嘉昌村，忠寮村併入苓和村，苓保村、源頭村則併成保源村，仁巷村與和巷村合併為仁和村，將榮村分割南北各自併入將貴村及將富村，為到目前以來最大的村-{里}-調整，此後為18村，2010年底將軍鄉[[2010年中華民國縣市改制直轄市|改制]]為將軍區、轄下村改里。<ref>[http://web2.tainan.gov.tw/Jiangjyun/CP/10883/history-1.aspx 《地方沿革》，臺南市將軍區公所官網] {{webarchive|url=https://web.archive.org/web/20131004214042/http://web2.tainan.gov.tw/Jiangjyun/CP/10883/history-1.aspx |date=2013-10-04 }}</ref><ref>[http://gissrv3.sinica.edu.tw/search/ 地名檢索系統 - 臺灣地區地名查詢系統 - Academia Sinica]</ref>2018年4月30日因里鄰調整，將西華里、西和里合併成「西甲里」，忠興里、嘉昌里合併成「忠嘉里」，保源里、苓和里合併成「苓仔寮里」，仁和里、北埔里合併成「巷埔里」，將富里、將貴里、三吉里合併成「將軍里」<ref name="里鄰調整">{{cite web|url=http://village.tainan.gov.tw/index.php/Announ/show_pdf/16 |title=將軍區 |work=臺南市里鄰編組及調整專區 |publisher=臺南市政府 |accessdate=2018-10-27}}</ref>。經此調整後，將軍區由18里減為12里。
