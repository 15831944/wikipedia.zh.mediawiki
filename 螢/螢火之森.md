'''螢火之森'''（{{lang-ja|蛍火の杜へ}}），標題直譯為「走進螢火之森」，是[[日本漫畫家|日本漫畫家]][[綠川幸|綠川幸]]创作的[[单篇|单篇]][[少女漫畫|少女漫畫]]。在2002年7月开始連載於[[白泉社|白泉社]]出版的《[[LaLa_DX|LaLa DX]]》[[漫画杂志|漫画杂志]]，故事以6岁时竹川螢在离她爷爷家不远的山林里迷路，邂逅了戴着面具奇怪的神秘人物銀为触发点而展开故事，后来竹川螢得知他是超自然的存在并且一旦被人类触摸便会永远消失。之后每次在夏季时竹川螢回到乡下爷爷家时便与银一起度过，他们之间的感情在努力挣脱各自的束缚的过程中逐渐走向成熟。其改编动画导演[[大森貴弘|大森貴弘]]称这是一个“十年间爱的物语”。

2003年7月时以同样的名称推出重制的短篇集[[单行本|单行本]]，其中书里还包括了4则綠川幸所写的单篇故事。2011年9月漫画被改編為44分钟的[[劇場版動畫|劇場版動畫]]，由大森貴弘执导，配音方面启用了[[内山昂辉|内山昂辉]]与[[佐仓绫音|佐仓绫音]]，配乐由[[吉森信|吉森信]]负责<ref group="參">{{cite web|url=http://movie.walkerplus.com/mv48667/|title=蛍火の杜へ|publisher=Walkerplus|language=ja|accessdate=2013年10月27日}}</ref>。《萤火之森》于9月17日在日本开映，一个月之间过日本国内便大受追捧。电影在欧洲的首映是在2011年10月8日在苏格兰举办的Scotland Loves Animation节上，并并获得了本节评委会特别奖。之后《萤火之森》陆续在英格兰Leeds International Film Festival、[[动画内容博览会|动画内容博览会]]和[[Anime_Expo|Anime Expo]]上放映，并拿下了第66届[[每日电影奖|每日电影奖]]中的最佳动画电影奖。

2012年2月22日《萤火之森》动画电影的[[蓝光光碟|蓝光光碟]]和[[DVD|DVD]]在日本国内开始贩售。《萤火之森》动画电影在日本上映前的9月5日推出了带有纪念意义的、附加的四短篇故事单行本，在其中包含了与《萤火之森》相关联的《萤火之森·特别篇》。蓝光光碟和纪念意义的单行本上市后都很快就蹿升至当年9月5至11日Oricon销量榜第13位。佐倉綾音在为竹川萤配音时坦言她对这个故事产生了强烈的感情共鸣，绿川幸明确表示这个故事为他漫画生涯产生了积极意义。评论家们普遍地称赞了动画电影的美、单纯和柔情，并与宫崎骏与新海诚的作品联系了起来。

== 人物介紹 ==

* 銀（{{lang|ja|ギン}}）（配音員 : [[內山昂輝|內山昂輝]]）
平時都戴著類狐狸面具，一旦被人類碰觸便會消失。由於嬰兒時期被拋棄在森林，後來受到山神大人的恩惠才得以存活。也因為山神大人施下的法術，導致銀既不是人類也不是妖怪，比較像是幽靈的存在，也不會老化。後來喜歡上螢。

* 竹川 螢（配音員 : [[佐倉綾音|佐倉綾音]]）
有點天然呆的少女，六歲的夏天在森林裡迷路時，第一次與銀相遇。每到了暑假便會到爺爺家玩，藉此和銀碰面。後來喜歡上銀。

== 劇情大綱 ==
那是屬於他們彼此的幻夢。
迷路的少女螢無意間進入了妖怪居住的山神森林，遇見了戴著狐狸面具的少年銀。奇異的旅途，就此展開。他們在每年夏日聚首，又分離，然而，彼此間的羈絆卻隨著時間，一點一滴在彼此生命的時間軸上，深深地刻劃下痕跡。但是，唯一可惜的，是螢無法觸碰銀。被施加了法術的銀，被人類觸碰後，便會消失。

{{HideH|详细剧情}}
: 故事講述6歲的女孩竹川螢，在暑假期間固定回到鄉下的爺爺家避暑，有一天，在住滿了妖怪的「山神之森」迷了路，因為迷路而蹲在草地上哭泣，這時卻遇見了戴著狐貍面具的少年銀。銀答應幫助螢走出森林，在途中被螢撲了2次而用木棒打退她2次，因為銀不能被人類碰觸到，會從世界上消失。

: 被帶離森林的螢問銀：「我是竹川螢，你的名字是？」

: 因為過了許久而對方沒有回答時又起了風，螢有些膽怯，匆匆的說明天會帶謝禮過來，轉身就跑開，這時銀才說：「我叫銀」

: 隔天，螢看到在等自己的銀，忍不住興奮而忘記不能碰觸銀而撲了過去，之後螢又被銀用木棒打了一記。從此，螢每年都期待著夏天，每年都期待著能再見到阿銀。年復一年，亭亭玉立的螢與阿銀之間的情愫，逐漸滋長。

: 在螢剛上高中那一年夏天，銀邀請螢去妖怪的夏日祭典，雖然螢看過妖怪，但是在一大群妖怪中會有些害怕，銀說：「我會保護妳的」

: 螢聽有些臉紅的說：「啊…聽你這樣說，害我想撲過去啦！」

: 聽到這番話的銀，拿下面具微笑的說：「那就撲過來吧！」

: 螢愣了愣。

: 「因為這是我的願望。」

: 晚上去赴約的螢和銀一起參加祭典，祭典結束時他們如早上一般，漫步在叢林中，阿銀對螢說：「我等不到下一個夏天，就算得穿越過重重人海，我還是想見到你。」說這話時摘下了自己的面具戴到螢的臉上，隔著面具吻了螢的臉，笑著說：「面具送妳！」

: 這時有小孩從他們身邊跑過，一個小男孩不小心絆倒，銀出手拉住小男孩的手不讓他跌倒，看著小男孩離開後發現自己的手在發光，螢看到後驚恐的說：「那個小孩難道是人類？」

: 銀起初很驚訝，之後微笑的張開手臂說：「來吧，我終於可以碰觸到妳了！」

: 螢跑向銀，拿下面具，抱住銀，流下眼淚，過不到幾秒，銀消失了，螢抱著銀的衣服痛哭。

: 「我很喜歡妳」

: 「嗯，我也是」

: 森林裡的妖怪都很感謝螢，因為銀終於想要碰觸別人，終於可以被擁抱。

== 创作 ==
漫画作者[[綠川幸|綠川幸]]在《爱藏版》的螢火之森的后记中提到，相比她的其他作品，这个故事创作的构思来得更加容易{{Sfn|綠川幸|2011|p=171}}。在已有的豐富漫畫創作经验上，綠川幸计划再挑戰撰写一个故事。在当她打算动工时，故事的构思突然间浮現於脑海。綠川幸为了留住这突如其來的构思，立马将之付诸于绘画创作。虽然随着情节的发展出现了与最初的构思相矛盾的地方，但总而言之她的进展还算顺利{{Sfn|綠川幸|2003|p=69}}。

綠川幸本人繪畫更多风景画的内心渴望促成了《萤火之森》的诞生。她对于画一个男孩闪躲一个向他扑过来的女生的场景很感兴趣{{Sfn|綠川幸|2003|p=140}}，也想画一个蓊郁、神秘的夏日森林场景{{Sfn|綠川幸|2003|p=69}}，那样她就能够延展色调与阴影，再來，她想比她之前的作品多画一些宁静的暑假场景。她也急于为银这个角色创造一个面具，面具的整体样子類似[[狐狸|狐狸]]的面孔，但狐狸那代表性的狭长雙眼则被鹅蛋形眼睛代替，使得这一面具有著狐狸和灵长类混合的“诡异”外貌感{{Sfn|綠川幸|2003|p=140}}。

在摘得每日电影奖最佳动画电影奖后，動畫製作人佐藤由美谈到是他的私自愿望推动了本作的动画化，他也很感谢能遇到这部作品。在2011年6月动画化的消息在日本公开。获评委会特别奖后，电影製作人之一的横山朱子表示最初计划不是改编为电影而是[[OVA|原创视频动画]]，在原班人马制作的《[[夏目友人帐|夏目友人帐]]》大受欢迎的境遇下，制作组们更加坚信他们应将《萤火之森》制作为动画电影。电影便由Brain's Base公司制作，大森貴弘执导。配音启用了佐仓绫音和内山昂辉二人。在之前的一些作品上，横山朱子与大森貴弘之间有过合作经历。《萤火之森》的制作阵营并不庞大，有自身是短片的原因，也因为在制作时仅仅定位为面向本国影院。

2011年3月就在电影制作进展过半时，一场[[2011年日本東北地方太平洋近海地震|大地震]]席卷日本。举国哀悼下，团队还是坚持继续完成电影的制作。导演大森貴弘希望让这部作品能帮到治愈受创需要时间恢复的国民。
