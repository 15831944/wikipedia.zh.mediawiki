《'''垂直活著，水平留戀著。'''》（'''{{lang-en|Fade to Exist}}'''），是[[台灣|台灣]][[歌手|歌手]][[艾怡良|艾怡良]]個人第四張[[錄音室專輯|錄音室專輯]]及首張全創作專輯，於2018年12月21日正式發行<ref name= "iTunes">{{cite web|url= https://itunes.apple.com/hk/album/%E5%9E%82%E7%9B%B4%E6%B4%BB%E8%91%97-%E6%B0%B4%E5%B9%B3%E7%95%99%E6%88%80%E8%91%97/1445993879|title= iTunes - 音樂 - 專輯 - 艾怡良 - 垂直活著，水平留戀著。|work= [[iTunes|iTunes]]}}</ref>，一共收錄十首歌曲；也是她加盟新唱片公司後發行的首張錄音室專輯。

== 創作背景 ==
{{Cquote|所有的話，我自己來說|艾怡良<ref name= "womany">{{cite web|url= https://womany.net/read/article/17474|title= 靈魂女聲艾怡良：人生是垂直活著，離開時就水平留戀著|date= {{start_date|2018|12|24}}|work= 女人迷}}</ref>}}

兩年前憑藉著《說 艾怡良》斬獲金曲最佳女演唱人的艾怡良終於推出全創作專輯《垂直活著，水平留戀著。 》<ref name="早衰少年Felix ">{{cite web|url= https://music.douban.com/review/9917990/|title= 《垂直活著，水平留戀著。》專輯細膩並展現出歌手的才華，艾怡良再度用音樂成功征服聽眾|date= {{start_date|2019|01|25}}|work= 豆瓣音樂}}</ref>，專輯通過「自我和解」的過程轉化淬煉成<ref name= "womany"/>，這張專輯講的是，如何面對“失去”，如何面對“記憶”，歌詞中多次提及生死，例如歌曲《玻璃心》就提到「掛了」、《滅了我》提到「雙腳一伸重頭來過」、《給朱利安》提到了「天堂」、《十四號登機門》提到了「告別」<ref name= "ADguy"/>；而「水平留戀著」本身就暗示了人離開塵世、兩腳一伸的平躺狀態」<ref name= "田中小百合"/>。艾怡良提到年齡不上不下的她，身邊有很多朋友家裡添了新生命，也有很多在乎的人離去，她還是沒學會如何接納“失去”，她能做的就是把她的害怕、她的思考用音樂告訴大家，以音樂會友，共同探討生命中的“失去”<ref name= "ADguy"/>。

== 歌曲簡介 ==
'''《玻璃心》'''於2019年1月18日作為專輯第三主打正式推出，歌曲暗喻著艾怡良自己從出生至今的自我檢討，有些難受的事，其實也可以笑著說。而她正是想要告訴大家，雖然不愜意，但是努力面帶笑意，對人生還是有點幫助的<ref>{{YouTube|4YKNLhU39uk|玻璃心}}</ref>。

'''《滅了我》'''把厭世放到了檯面上，留下自己想留下的，將其他的不好的「我」全部抹去<ref name= "ADguy">{{cite web|url= https://music.douban.com/review/9844024/|title= 《垂直活著，水平留戀著。 》全專歌詞解讀，艾怡良到底想告訴我們什麼？|date= {{start_date|2018|12|22}}|work= 豆瓣音樂}}</ref>。

'''《一整夜》'''於2019年1月3日作為專輯第二主打正式推出，是首輕快、性感的歌曲，講述著一個若即若離的都市遊戲<ref>{{cite web|url= http://www.billboardradiochina.com/story/2019-01-04ai-yi-liang-dou-hui-da-ren-shi-xing-gan-ge-qu-yi-zheng-ye_zh_hk|title= 艾怡良都會大人式性感歌曲《一整夜》|date= {{start_date|2019|01|04}}|work= Billboard Radio China}}</ref>；這首歌曲曾是艾怡良寫給[[張惠妹|張惠妹]]的歌曲<ref>{{cite web|url= http://k.sina.com.cn/article_6426349831_17f0a510700100ep5e.html|title= 艾怡良趕回台北跨年合體張惠妹？經紀人堅定答案笑翻|date= 2018年12月26日|work= [[新浪|新浪看點]]}}</ref>。

'''《給朱利安》'''於2019年2月14日作為專輯第四主打正式推出，想念當時陷入年少輕狂愛情中的自己<ref>{{YouTube|_jOAUWhSQI0|艾怡良 Eve Ai《給朱利安》Official Music Video}}</ref>。

'''《十四號登機門》'''Demo為艾怡良在中國選秀節目《[[中國最強音|中國最強音]]》中演繹的歌曲《Wonderland》，中文版是一首用吟唱的方式來表達旅行後留戀的心情<ref name "樂旻"/>，就像在日常中消磨的人懷念旅途中的燦爛陽光。

'''《討厭的艾瑞斯》'''是一首具時尚感的歌曲，探討一種常常與自己為敵，卻沒有勇氣與世界為敵的矛盾內心<ref name "樂旻"/>。

'''《萊特兄弟有罪》'''於2019年3月21日作為專輯第五主打正式推出，是一首充滿濃厚藍調搖滾氣味的歌曲<ref>{{cite web|url= http://news.pchome.com.tw/entertainment/wownews/20190120/index-15479559049499033006.html|title= 艾怡良新歌演唱會勁黑挖空設計西裝背心+媚黑蕾絲長裙 既帥氣又性感獨特”艾”魅力|date= {{start_date|2019|01|20}}|work= Wow!NEWS新聞網}}</ref>，講述戀人們能輕易跨越距離、不再珍惜得來不易之情感<ref name= "田中小百合"/><ref>{{YouTube|FV6wd-IPcU4|艾怡良Eve Ai《萊特兄弟有罪》Official Music Video}}</ref>。

'''《美術課》'''是寫給來不及告別的人<ref name "樂旻"/>，是怡良對她那突然離世的高中美術老師的紀念<ref name= "田中小百合"/><ref name= "陳于嬙">{{cite web|url= https://www.mirrormedia.mg/story/20181228ent008/|title= 少了憤怒多了體悟　艾怡良交出脫胎換骨之作|date= {{start_date|2019|12|29}}|author= 陳于嬙|work= [[鏡週刊|鏡週刊]]}}</ref>。

'''《夜晚出生的小孩》'''是一首異國風情十足卻十分悲傷的歌曲<ref name= "陳于嬙"/>，編曲上運用了木吉他與電吉他，兩者的互動勾勒出冷調的氛圍，而小號的時隱時現與波斯語的悠遠吟唱，營造出遼闊的時空感，副歌往後軍鼓的推動與嘹亮的小號掃去起初的畏懼與憂傷，帶出晨光灑落一般的溫暖與勇氣<ref name "樂旻"/>。艾怡良指歌曲是寫給在現實生活中覺得被背叛、感覺跟社會格格不入的小孩，其實他們都是善良的，不外乎就是我們一睜開眼，能不能信任這個世界並與他交流，對現代人來說或許都是件很困難的事<ref>{{cite web|url= https://www.niusnews.com/=P0g2kt31|title= 「青春會逝去，但回憶不會。」艾怡良教你先說愛，人生才會無悔|date= {{start_date|2019|01|17}}|work= 妞專訪}}</ref>。

'''《Forever Young》'''是專輯中最後一首歌曲，已於2018年12月14日作為專輯首波主打在各大音樂平台發行。這是一首衝撞內心深處的歌曲，歌曲並不刻意探討存在和離開，而是用「愛人的方式」來表現自己的存在<ref>{{cite web|url= https://music.douban.com/subject/30400344/|title= 豆瓣音樂 - 艾怡良 - Forever Young - 簡介|work= 豆瓣音樂}}</ref>。歌曲推出獲得不少網民盛讚，更封她為「偷眼淚的人」<ref>{{cite web|url= https://www.playmusic.tw/column_info.php?id=13616&type=news|title= 艾怡良《Forever Young》唱出靈魂深處受封『偷眼淚的人』 1/18 Legacy舉辦新歌演唱會|date= {{start_date|2018|12|19}}|work= PlayMusic}}</ref>；更在[[豆瓣|豆瓣音樂]]獲得8.8/10分的正面評價。

專輯《垂直活著，水平留戀著。》於發行之後獲得較正面的評價。PlayMusic音樂網給予該專輯4.5/5的評分，評價該專輯：「細膩並展現出歌手的才華，艾怡良再度用音樂成功征服聽眾<ref name="rating2"/> 。」豆瓣音樂給予該專輯8.1/10的評分，其中一個樂評這樣評價該專輯：「這張專輯裡的歌者被徹底的激活，她的遣詞造句滿含著對人生的思考與體悟，展現出前所未有的深度和廣度；從專輯能聽得到歌者更加細膩的演唱，更加誠實的創作；在每首歌的呼吸咬字、起承轉合間，更直觀的感受到她的語氣和情緒<ref name "樂旻">{{cite web|url= https://music.douban.com/review/9864257/|title= 艾怡良：誠實地寫，用力地活|date= {{start_date|2019|01|01}}|author= 樂旻|publisher= 豆瓣音樂}}</ref>。」《[[GQ|GQ臺灣]]》評選該專輯為「2018 年不能錯過的 12 張華語專輯」之一，並評價該專輯：「10首歌曲，完完全全把艾怡良內心最赤裸的情感一次掏空<ref>{{cite web|url= https://www.gq.com.tw/entertainment/culture/content-38270.html|title= 2018 年不能錯過的 12 張華語專輯|author= Richardbee|date= {{start_date|2018|12|27}}|work= [[GQ|GQ臺灣]]}}</ref>。」《[[南方都市報|南方都市報]]》評選該專輯為「十大華語唱片」第七名，並評價該專輯：「艾怡良的歌曲創作有特別的語法，她的抒情方式令人嘆為觀止，她將自己的表達提升了一個層次，從容地找到了顧左右而言它的方法；這張專輯是強大個性與平易近人的結合，是磨礪珍珠的實驗室，是自我又開放的內在探索<ref>{{cite web|url= http://epaper.oeeee.com/epaper/C/html/2019-01/10/content_805.htm#article|title= 十大華語唱片|work= 南方都市報|date= {{start_date|2019|01|10}}}}</ref>。」田中小百合給予7.8／10分的評價，他這樣評價該專輯：「新專輯顯得歌曲之間的關聯性較高，而她的創作也繼續帶來新的活力、新的色彩，並且跳出了更大的跨度，她譜寫的抒情慢歌易入耳、有經得起時間洗禮之潛質，或具一定的溫度；但是，積極吸收各風格的艾怡良似乎未能夠將不同的音樂風格元素相互混合消化出新的產物，或形成自己特色<ref name= "田中小百合">{{cite web|title=「垂直上」得到進步，「水平上」較突出的艾怡良|url= http://tinzhonglittlelily.blogspot.com/2019/01/blog-post.html|author= 田中小百合|date= {{start_date|2019|01|05}}}}</ref>。」
