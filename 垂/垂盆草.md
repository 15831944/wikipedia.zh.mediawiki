'''垂盆草'''（[[学名|学名]]：{{lang|la|''Sedum sarmentosum''}}）为[[景天科|景天科]][[景天属|景天属]]的植物。

== 形态 ==
多年生肉质[[草本|草本]]，茎纤细，匍匐或倾斜，近地面部分容易生根。倒披针形至长圆形的叶子三片轮生。夏季开黄色小花，无柄，疏松的排列在顶端，呈两歧聚伞花序。

== 分布 ==
分布于[[日本|日本]]、[[朝鲜|朝鲜]]以及[[中国大陆|中国大陆]]的[[河北|河北]]、[[甘肃|甘肃]]、[[福建|福建]]、[[陕西|陕西]]、[[山西|山西]]、[[江西|江西]]、[[浙江|浙江]]、[[贵州|贵州]]、[[河南|河南]]、[[湖南|湖南]]、[[吉林|吉林]]、[[辽宁|辽宁]]、[[江苏|江苏]]、[[湖北|湖北]]、[[四川|四川]]、[[北京|北京]]、[[山东|山东]]、[[安徽|安徽]]等地，生长于海拔500米至1,600米的地区，常生长在山坡阳处以及石上，目前尚未由人工引种栽培。

== 别名 ==
豆瓣菜、狗牙瓣、石头菜、佛甲草（秦岭植物志） 爬景天（北京植物志）卧茎景天（东北植物检索表） 火连草、豆瓣子菜、金钱挂、水马齿苋、野马齿苋（改订植物名汇） 匍行景天（经济植物手册） 狗牙草（湖北植物志）
