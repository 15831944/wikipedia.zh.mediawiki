'''出木杉 英才'''（{{lang-ja|できすぎ ひでとし}}，{{lang-en|Dekisugi Hidetoshi}}，4月14日—），早期台灣亦譯'''王聰明'''、'''優杉'''<ref>台版 大長篇電影《大雄的大魔境》</ref>及'''小杉'''，香港亦譯'''太郎'''、'''彬仔'''，美國版稱'''Ace Goody'''，4月出生，是[[藤子·F·不二雄|藤子·F·不二雄]]的科幻[[漫画|漫画]]《[[哆啦A夢|哆啦A夢]]》及其動畫版中的登場人物之一。

在故事中，出木杉通常以與其姓氏涵義「出眾，第一」相符的完美少年形象出現。<ref name="a">日文中「{{lang|ja|出来過ぎ}}」與「{{lang|ja|出来杉}}」都和「{{lang|ja|出木杉}}」的唸法一模一樣，都是「{{lang|ja|できすぎ}}」，而「{{lang|ja|出来過ぎ}}」翻譯為中文的就是指「太會了，太好了」之意。</ref>並往往擔任有著眾多缺點的主角野比大雄的對比角色與競爭對象。

== 人物 ==
出木杉出生于4月14日。<ref>[http://comic.people.com.cn/GB/123931/9988057.html 热点评论：哆<nowiki></nowiki>啦A梦过生日 童年那个小叮<nowiki></nowiki>当 (7)]，2009年9月4日。</ref>他是[[大雄|大雄]]班级裏的班长<ref>31卷〈試著說一回再見〉</ref>，也是学校裡的帅哥。除了功课好以外，也擅长体育，多次帮助胖虎的棒球队扭转败局，还很会做菜。有想要的东西時，他不依赖父母，而是给别的小孩做家庭教師賺钱<ref>22卷《稅金鳥》</ref>。此外，他也看漫画，玩游戏，还会变魔术。雖然是小學生，但是却與外國笔友史密斯长期通信联络<ref>40卷《大雄的建國用具》</ref>。

可以说是一位極少缺点的少年。对预言之类的东西也能找到合乎逻辑的破解方法<ref>36卷《大預言·地球滅亡之日》</ref>。能够去未来的世界看看是他的梦想，在大雄的帮助下得以实现。<ref>27卷《人腦圖書封套》</ref> 此外，也曾经與大雄和哆啦美一起去白堊紀时代。

在自己的房間内有部电话，曾因好成績遭同班同學嫉妒而被骚扰过。<ref name=dr30mnd>30卷《半夜裏的電話惡魔》</ref>

=== 性格 ===
他性格也非常好，對任何人都很和善，可以原諒曾經攻擊自己的人，心胸寬大，連大雄自己也承認出木杉比他好得多。

由於他在作品的出場大多時候都和靜香相處在一起，因此被大雄當成情敵，但是出木杉本人對大雄並沒有競爭意思。

雖然無追求靜香之舉動，其本人亦對靜香有好感<ref>37卷《蛋殼裡的靜香》</ref>，在電影《[[大雄的結婚前夜|大雄的結婚前夜]]》亦用笑容祝福靜香和大雄結婚。

嗜好有：
* 讀書
* 打[[棒球|棒球]]
* 研究
* 與[[源靜香|靜香]]到[[圖書館|圖書館]]
* 烹飪
* 觀察星象
* 寫生、畫畫

=== 交流 ===
與[[源静香|靜香]]的關係非常好，常與靜香一起出去玩，看書或寫生，甚至與靜香互相[[交換日記|交換日記]]<ref>23卷《透視貼紙大作戰》</ref>。

但是除了靜香，在故事裡與其他主要角色（哆啦A夢、大雄、小夫、胖虎等）的交流並不多見，在電影版中也甚少出場。而雖然受異性歡迎但也較少看到與靜香以外的異性互動，不過受到邀請時不介意加入大雄或其他同學的活動。跟[[胖虎|胖虎]]的關係中，因為頭腦、體能跟個性都很好，劇中幾乎不曾被胖虎欺負。不過接到胖虎武斷命令也會不得不遵守，例如稅金鳥事件與胖虎演唱會等。

在《[[大雄的結婚前夜|大雄的結婚前夜]]》中與胖虎、大雄、小夫等一起慶祝喝酒，及40卷《奪回靜香》的婚後相處來看，出木杉與大雄等人的關係在成人後會有所改善。

== 家族 ==
父母的詳細情況不明，父親只出場過一次<ref>36卷《包租籌碼》</ref>，母親沒有出場過，只是因為使用了哆啦A夢的工具定時傻瓜彈而發出過「嘻嘻哈哈」的怪聲<ref>41卷《定時傻瓜彈》</ref>。

曾經一度因為父親調工作而要搬遷到[[美國|美國]]去，但最后沒實現<ref>電視動畫版《7年後的煩惱》</ref>。

他作為寵物的狗非常強壯，是曾經在比賽裏獲得冠軍的名犬，連胖虎都對其謙讓三分<ref>PLUS1〈想要厲害的寵物〉</ref>。

== 未来 ==
長大后出木杉的體格也變得更強壯，并且留了絡腮胡子。在第44卷《夏威夷越來越近了》中曾經說其夢想是去火星和月球，最后實現了這個夢想，被派到火星工作<ref>40卷《把靜香搶回來》</ref>。因為去火星工作而把兒子托給大雄家，由此看來長大后也與大雄有密切的交流。

在多啦a夢來到20世紀前，原本設定中出木杉與靜香結婚，而自己的孫女會和大雄的孫子結婚。

妻子是金发的白人。兒子的名字叫英世（{{lang|ja|ヒデヨ}}）。

== 其他 ==
; 對故事的影響
*在書籍《{{lang|ja|ド・ラ・カルト}}》裏的記述，在出木杉登場以前，哆啦A夢是以「為了能讓大雄的未來變得幸福」而來。
*但在出木杉出場之后，哆啦A夢出場的目的變成使「現在的大雄可能可以改變未來」，角色演變為以保護大雄為主。因為出木杉的登場讓大雄的未來充滿不確定性。這樣的敘述在單行本上也有<ref>34卷《時間飛速流逝》</ref>。

; 名字的變化
*当初，在單行本第22卷《税金鳥》中出木杉是以「太郎」的名稱登場。在第23卷《透視貼紙大危機》中，明信片的收件人名字變為「英才」（無假名）。而在新版的稅金鳥故事里，「太郎」被修正回「英才」。
*出木杉于月刊[[corocoro|corocoro]]1979年9月號首次在《哆啦A夢和哆啦美》故事裡登場，但當時的名字還是「明智」（原型被認為是[[明智小五郎|明智小五郎]]，在文庫版「哆啦美篇」中，「明智」被修正為「出木杉」）。
*在2006年5月25日播放的電視動畫版裡，出木杉的名字曾經被用來猜謎，正確的答案是「英才」。
*另外，有很多讀者將其名字誤記為「出来過」、「出来杉」，漫畫裡也有「出来杉」這樣的錯誤。<ref name="a"/>

; 在大長篇中的出場
*在電影《[[大雄的恐龍|大雄的恐龍]]》的劇本草稿中出現，與哆啦A夢等五人一起在一億年前的世界進行大冒險。但是在最后制作完成的電影里他并無出現，日後實際登場的數個長篇中主要是擔任知識解說（如老菸槍森林<ref>《[[大雄的大魔境|大雄的大魔境]]》</ref>、魔法<ref>《[[大雄的魔界大冒險|大雄的魔界大冒險]]》</ref>、電影特效<ref>《[[大雄的宇宙小戰争|大雄的宇宙小戰争]]》</ref>、新品種植物<ref>《[[大雄與動物行星|大雄與動物行星]]》</ref>、鳥人傳説<ref>《[[大雄與翼之勇者|大雄與翼之勇者]]》</ref>）而無參與大雄等人的冒險計畫。
*在《[[大雄的創世日記|大雄的創世日記]]》裏，有與出木杉相貌極為相似的出木松博士的出場，進行了世界首次的橫越南極大冒險。之后與和大雄非常相似的野比野美秀和與靜香非常相似的源靜代一起乘坐飛艇來到南極的大洞，前往地下世界探險的故事。
