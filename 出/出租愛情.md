== 概要 ==
在聖誕節當天，山田公介的住所突然出現一名拿著一個大箱子的男人。男人沒有說出任何事情便離去，然後箱內突然出現一位女性，女性的名字叫高杉玲美。

== 登場角色 ==

=== 主要角色 ===
; 高杉玲美 - [[剛力彩芽|剛力彩芽]] (童年：鴫原凛) (香港配音：[[何璐怡|何璐怡]])
: 「Rental Lovers」人氣No.1的出租女友、在出租期間會盡力解決顧客的所有問題。是一個能把所有問題都完美解決的最強女友。
; 山田公介 - [[太賀|太賀]] (香港配音：[[張方正|張方正]])
: 找尋工作失败，一個平凡的大學4年級生。因為一次意外遇上玲美而愛上了她，在此之前沒有任何戀愛經驗。因為玲美的租借費用太貴而無法繳付，最後在「Rental Lovers」兼職來還取債務。
; 道端堇 - {{tsl|ja|岸井ゆきの|岸井雪乃}} (香港配音：[[廖杏茵|廖杏茵]])
: 公介和隼人的青梅竹馬，喜歡公介。

=== Rental Lovers ===

: 「Rental Lovers」社長。
; 天戶悠 - [[原幹惠|原幹惠]]（香港配音：[[詹健兒|詹健兒]]）
: 「Rental Lovers」人氣No.2的出租女友、視玲美為敵人。
; 香川明美 - {{tsl|ja|金田彩奈}}（香港配音：[[魏惠娥|魏惠娥]]）
: 「Rental Lovers」的出租女友，像天戸悠的手下。

=== 其他角色 ===
