'''出雲號護衛艦'''（{{lang-ja|いずも}}，[[日語羅馬字|羅馬字/英文]]：JS Izumo, DDH-183）是[[日本海上自衛隊|日本海上自衛隊]]旗下一艘[[直升機護衛艦|直升機護衛艦]]，為[[出雲級直升機護衛艦|出雲級]]的一號艦，也是日本海上自衛隊的第七艘直升機護衛艦。2015年成軍後成為{{link-ja|海上自衛隊第1護衛隊群|第1護衛隊群|第1護衛隊群}}第1護衛隊的旗艦，以[[橫須賀港|橫須賀港]]為[[母港|母港]]。

== 命名 ==
出雲號的艦名繼承於[[大日本帝國海軍|大日本帝國海軍]]的{{link-ja|出雲級裝甲巡洋艦|出雲型装甲巡洋艦}}一號艦「[[出雲號_(裝甲巡洋艦)|出雲]]」，而其名稱又源自於[[令制國|令制國制度]]時代的[[出雲國|出雲國]]（其範圍約為現今的[[島根縣|島根縣]]東部）。

== 艦艇歷史 ==

出雲號是根據[[日本防衛省|日本防衛省]]的《{{link-ja|2011年中期防衛力整備計劃|中期防衛力整備計画 (2011)}}》預定生產的19,500[[噸位|噸位]]級別、搭載[[直升機|直升機]]的[[護衛艦|護衛艦]]，並由[[IHI海洋聯合|IHI海洋聯合]]橫濱工廠負責建造，2012年1月27日動工。

2013年8月6日，出雲號在日本[[神奈川縣|神奈川縣]][[橫濱市|橫濱市]]舉行下水典禮<ref>{{Cite web|url=http://news.xinhuanet.com/world/2013-08/06/c_116837085.htm|title=日本“出雲”號直升機航母舉行下水儀式|accessdate=2013-08-07|date=2013-08-06|author=[[新華網|新華網]]}}</ref>，並於2015年3月25日正式服役<ref>{{cite web|title=海自最大の護衛艦・いずも就役　ヘリ９機を運用可能|url=http://www.asahi.com/articles/ASH3S5KGRH3SUTIL035.html|publisher=[[朝日新聞|朝日新聞]]|accessdate=2015-03-25|date=2015-03-25}}</ref>。

2018年12月18日，日本內閣18日通過了新的《防衛計劃大綱》以及未來5年(2019年至2023年)的《中期防衛力整備計劃》。當中透露計劃將兩艘"出雲"級直升機護衛艦改裝成能夠搭載美國F-35短距起飛戰鬥機的航空母艦，這樣日本海軍就在70多年來首次擁有海基空中打擊能力。<ref>{{cite web|title=日本：造航母買戰機 強軍計劃抗衡中國|url=https://www.bbc.com/zhongwen/trad/world-46605715|publisher=[[BBC中文網|BBC中文網]]|accessdate=2018-12-18|date=2018-12-18}}</ref>。

== 評價 ==
出雲號長約250米，規模相當於[[二戰|二戰]]期間日本的[[航空母艦|航空母艦]][[加贺号航空母舰|加贺号]]，被稱為日本的「准航母」<ref>{{cite news|url=http://trad.cn.rfi.fr/%E4%B8%AD%E5%9C%8B/20170313-%E6%97%A5%E6%9C%AC%E6%93%AC%E6%B4%BE%E6%9C%80%E5%A4%A7%E8%AD%B7%E8%A1%9B%E8%89%A6%E2%80%9C%E5%87%BA%E9%9B%B2%E8%99%9F%E2%80%9D%E5%88%B0%E5%8D%97%E6%B5%B7%E2%80%9C%E7%A7%80%E5%85%B5%E5%8A%9B%E2%80%9D |title=日本擬派最大護衛艦“出雲號”到南海“秀兵力”(RFI 法廣)|date=2017-03-13 |accessdate=2017-03-14 }}</ref><ref name=medium/>，認為日本藉此規避不得擁有航母的限制<ref>{{cite news|url=https://www.hk01.com/%E5%9C%8B%E9%9A%9B/77621/%E6%97%A5%E6%9C%AC%E6%93%AC%E6%B4%BE%E6%BA%96%E8%88%AA%E6%AF%8D%E5%87%BA%E9%9B%B2%E8%99%9F-%E8%87%B3%E5%8D%97%E6%B5%B7%E6%B8%B8%E5%BC%8B3%E5%80%8B%E6%9C%88-%E5%B1%95%E7%A4%BA%E8%A6%8F%E6%A8%A1%E7%82%BA%E4%BA%8C%E6%88%B0%E5%BE%8C%E6%9C%80%E5%A4%A7 |title=日本擬派準航母出雲號　至南海游弋3個月　展示規模為二戰後最大 |date=2017-03-13 |accessdate=2017-03-14 }}</ref>，亦有中韓媒體質疑日本選[[廣島原爆|廣島原爆]]紀念日下水可能脫離現有的[[和平憲法|和平憲法]]。海外媒體則認為，全世界也只有日本才會為了其政治和軍事發展的目的把如此巨大的[[直升機航空母艦|直升機航艦]]稱為[[直升機|直升機]][[驅逐艦|驅逐艦]]<ref>{{cite web |url=http://nationalinterest.org/blog/the-buzz/when-aircraft-carrier-not-aircraft-carrier-when-its-japans-17735 | title=When Is an Aircraft Carrier Not an Aircraft Carrier? When It's Japan's. | date=2016-09-16 | accessdate=2016-10-27}}{{en icon}}</ref>。該艦上[[甲板|甲板]]能夠提供[[F-35|F-35]]B戰機垂直起降，有評論指出日本的戰機是傳統非艦載型式F-35A，同時出雲號的甲板不適合戰機升降<ref name=medium/>，日本防衛省也回應指「這有違反『專守防衛』的政策，我們沒考慮利用出雲護衛艦進行戰鬥機的起降。」對於選在6日下水，海上自衛隊解釋作出該選擇是因為當日[[潮汐#變動的範圍：大潮和小潮|大潮]]且為大吉日，與廣島原爆紀念日無關<ref name="sina">{{cite web | url=http://dailynews.sina.com/bg/news/int/chinesedaily/20130807/06114836671.html | title=日直升機航母出雲號下水 | accessdate=2013-12-09}}</ref>。

而且有評論認為，出雲號是日本對[[中華人民共和國|中國]]近期發起的[[軍備競賽|軍備競賽]]所做出的相互牽制<ref>{{cite web | url=http://www.ipcs.org/article/east-asia/japan-the-izumo-warship-and-regional-security-4092.html | title=Japan: The Izumo Warship and Regional Security | date=2013-08-19 | accessdate=2013-11-30}}{{en icon}}</ref>，其軍備、使用狀況等資料均具有極大透明度<ref name=medium>{{cite web | url=https://medium.com/war-is-boring/bc29b669fb0e | title=Is It a Duck? Is It an Aircraft Carrier? No! It's Japan's 'Helicopter Destroyer' | accessdate=2013-11-30}}{{en icon}}</ref>，同時可以加強人員以及物資輸送的能力來提高大型救災行動的效率<ref>{{cite web | url=http://www.bbc.co.uk/news/world-asia-23594519 | title=Japan unveils 'largest warship' since World War II | date=2013-08-06 | accessdate=2013-11-30}}{{en icon}}</ref>。
