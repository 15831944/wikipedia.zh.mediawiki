'''沉管式隧道'''一种建造[[隧道|隧道]]的方法。這種方法只適用於建造[[海底隧道|海底隧道]]或[[水底隧道|水底隧道]]。

== 建造程序 ==
# 用[[挖泥船|挖泥船]]挖掘[[海床|海床]]
# 用拖船將沉管式隧道組件拖到預定位置
# 沉降在地基上
# 用鎖緊物料鞏固組件
# 在沉管式隧道組件周圍回填物料
# 放置保護石層保護組件
# 修復海床，抽走隧道積水
'''註'''：建造[[水底隧道|水底隧道]]無需修復海床

== 優點 ==
* [[抗震能力|抗震能力]]較[[鑽挖式隧道|鑽挖式隧道]]出色
* 不影響居民
* 可以不須填海而建造海港內的一段隧道
== 缺點 ==
[[File:Western_Harbour_Tunnel.JPG|250px]][[File:Eastern_Harbour_Tunnel.jpg|250px]]
* 挖掘海床時或會揚起沉積的[[污染物|污染物]]，影響水質，令生物死亡
