{{vgname|沉默之丘：破碎記憶|ja=サイレントヒル シャッタードメモリーズ|en=Silent Hill: Shattered Memories|cn=寂静岭：破碎的记忆|tw=沉默之丘：破碎記憶|aka-hk=}}是由[[科樂美|科樂美]]於2010年3月25日發售的[[驚悚遊戲|驚悚類]]電子遊戲。其開發公司為負責《[[沉默之丘：起源|沉默之丘：起源]]》的[[英国|英国]]游戏制作公司[[Climax_Studios|Climax Studios]]。

== 概要 ==
繼承了[[寂静岭|系列第一部作品]]的主題，再一次重新製作的作品。但是並非[[重製|復刻版]]，從登場人物的設定到故事，「冰世界」的登場等等，都與『[[寂静岭|1代]]』有著大幅的相異點。

主題為「'''無法戰勝的恐怖'''」。如其所示，本作中無法使用武器打倒敵人，另外，視點採用第三人稱視點。主角與1代相同，為哈利‧梅森(Harry Mason)。根據遊玩中的選項，哈利的性格會隨之變化，同時也會影響其他角色的外觀與性格。另外建築物的內部裝潢，外觀與會話的台詞，郵件的內容，怪物的型態等也會隨之有細微的變化，全體的變動數非常大，在如此複雜的變化中，主角哈利將會迎接令人震撼的最後結局。

== 故事 ==
住在有名度假勝地「沉默之丘」的哈利·梅森，在帶著獨生女雪柔從遊樂園回家的路上，因為凍結的路面與下著雪的惡劣天氣發生了車禍，並失去了意識。等他醒來時，女兒已經不見蹤影，在他眼前的只有詭異靜謐的城鎮。

== 登場人物 ==
; 哈利‧梅森(Harry Mason) 
: 本作主角，30歳。住在沉默之丘的作家，溺愛女兒雪柔。車禍後，為了尋找消失的女兒雪柔而於沉默之丘奔走著……
; 雪柔‧梅森(Cheryl Heather Mason)
: 哈利的獨生女，7歳，擁有豐富想像力和行動力的少女。在哈利發生車禍後失去意識的期間消失了身影。 
; K醫生/麥可·考夫曼（Michael Kaufmann）
: 精神分析醫師，58歳。以冷靜的態度來建議玩家。遊戲中會適時的做出建議，玩家也能夠回答他的問題。
; 席柏‧班奈特(Cybil Bennett) 
: 沉默之丘的女警，40歳。非常熟悉沉默之丘，在巡邏途中遇見哈利。
; 米雪兒・巴爾蒂斯(Michelle Valdez)
: 哈利於當地的高中遇見的女性。
; 妲利亞(Dahlia)
: 突然出現在哈利面前的謎之女性。言行舉止像是哈利的熟人。 
; 莉莎‧加蘭德(Lisa Garland) 
: 護士。因為事故而與哈利於醫院中碰面。
; 約翰(John)
: 米雪兒的戀人，職業為[[律師|律師]]。為本作的原創角色。

== 怪物 ==
本作只有一種怪物存在，但其型態有非常多的數量。

=== 刻出眼睛的蒼白怪物 ===
哈利於「惡夢」中遇見的怪物，英語圏以「Raw Shocks」來表示。會往哈利飛撲攻撃而來，沒有辦法打倒。只能一邊逃避一邊尋找逃走的出口。在逃走途中若使用撿到的煙霧罐可以一時嚇退怪物。有[[雄性|雄性]]與[[雌性|雌性]]存在，會因為劇本的分岐而變化外觀，列舉如下：
;Feminine
:擁有從臉的一邊到另一邊的巨大嘴巴，女性型態，有些瘦弱的怪物。腰邊有著像是裙子的[[鰭|鰭]]，[[腳踝|腳踝]]部份隆起，像是穿著[[高跟鞋|高跟鞋]]似的。名稱的意思為「'''女性'''」。
;Bloated
:[[皮膚|皮膚]]上有巨大腫瘤且顏色為灰色，為巨大化・肥胖化的怪物。名稱的意思為「'''肥大化'''」。
;Broken
:臉上有著激烈的凹凸，手腳，[[太陽穴|太陽穴]]的部份有著類似殘枝的怪物。關節部份有洞，而內部是黑色的。名稱的意思為「'''破碎'''」。
;Geometric
:身體非常瘦弱，只剩皮包骨的怪物。儘管如此，還是會以衰弱的型態攻擊而來。頭部異常膨脹，而其中心開了一個大洞，手腳如磁鐵一樣分開。名稱的意思為「'''瘦身化'''」。

== 道具 ==
; [[行動電話|行動電話]]
: 多功能手機，可收發郵件，打電話，拍照，查看地圖等。與之前的[[收音機|收音機]]一樣，要是有怪物靠近就會發出噪音。
; 煙霧罐
: 只在「惡夢」中出現的道具。使用後一定時間怪物無法靠近，也可以放置於地面。
; 回憶
: 可在「現實世界」中獲得的道具。因為可以將登場人物們過去擁有的東西實體化，所以會根據收集的數量使結局有微妙的變化。

===音樂===
''破碎記憶''的音樂再一次地由[[山岡晃|山岡晃]]來作曲，並也是他從科樂美退休後16年為''[[寂静岭系列|寂静岭系列]]''所做的最後貢獻。[[:en:Mary_Elizabeth_McGlynn|Mary Elizabeth McGlynn]]獻唱了遊戲中的四首歌：「[[:en:Always_on_My_Mind|Always on My Mind]]」，「When You're Gone」，「Acceptance」以及「Hell Frozen Rain」。[[:en:Joe_Romersa|Joe Romersa]]也為三首新歌寫詞，其中包括了封面歌曲「Always on My Mind」。[[:en:GameStop|GameStop]]並提供了包含遊戲中21首曲目原聲帶的預先訂購。
