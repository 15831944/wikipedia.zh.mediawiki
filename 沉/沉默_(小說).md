{{日本作品|沉默|沈黙|Silence}}是日本作家[[遠藤周作|遠藤周作]]在1966年出版的歷史小說，係基於部分史實、以及作者身為[[天主教徒|天主教徒]]的哲學思想所著作，講述一名[[耶穌會|耶穌會]]傳教士被派到17世紀的日本，面對日本基督徒在[[島原之亂|島原之亂]]後受到迫害{{link-ja|隐匿的基督徒|隠れキリシタン|轉入地下}}所遭遇的一連串故事。此書使遠藤拿下1966年的[[谷崎潤一郎獎|谷崎潤一郎獎]]，被稱為「遠藤周作的卓越成就」及「二十世紀最好的小說之一」。

== 改編作品 ==
除了周作的舞台劇版本《''The Golden Country''》，還有幾個改編作品：

* 1971年，由日本[[東寶|東寶]]改編為[[沉默_(1971年電影)|同名電影]]，由[[篠田正浩|篠田正浩]]執導。<ref>{{Cite web |url = http://www.imdb.com/title/tt0067755/ |title = Chinmoku (1971) |publisher = IMDb |language = en}}</ref>
* 1996年，改編為電影《''{{link-en|Os Olhos da Ásia|Os Olhos da Ásia|The Eyes of Asia}}''》，由{{link-en|João Mário Grilo|João Mário Grilo}}執導。<ref>{{Cite web |url = https://www.theguardian.com/film/2016/dec/10/silence-review-the-last-temptation-of-liam-neeson-in-scorseses-shattering-epic |title = Silence review: the last temptation of Liam Neeson in Scorsese's shattering epic |author = Peter Bradshaw |date = 2016-12-10 |publisher = The Guardian |language = en}}</ref>
* 2000年改編為同名歌劇在東京的[[新國立劇場|新國立劇場]]公演，劇本與音樂由作曲家和詩人{{link-en|松村禎三|Teizo Matsumura}}編撰。<ref>{{Cite web |url = http://www.nntt.jac.go.jp/english/opera/e20000464_opera.html |title = SILENCE |publisher =  New National Theatre, Tokyo |language = en}}</ref>

* 2016年，由美國導演[[馬丁·史柯西斯|馬丁·史柯西斯]]改編及執導為[[沉默_(電影)|同名電影]]，於2016年11月29日在羅馬的{{link-en|宗座東方學院|Pontifical Oriental Institute}}首映。<ref>{{Cite web |url = http://variety.com/2016/film/global/martin-scorsese-meets-pope-francis-talks-jesuit-history-prior-to-silence-screening-1201930025/ |title = Martin Scorsese Meets Pope Francis and Talks Jesuit History Prior to 'Silence' Screening |author = Nick Vivarelli |date = 2016-11-30 |publisher = Variety |language = en}}</ref>

此外，該小說啟發了蘇格蘭音樂家{{link-en|詹姆斯·麥美倫|James MacMillan}}於2002年創作{{link-en|Symphony No. 3 (MacMillan)|Symphony No. 3 (MacMillan)|Symphony no. 3, "Silence"}}。<ref>{{Cite web |url = https://www.theguardian.com/music/2008/apr/28/classicalmusicandopera1 |title = BBCSSO/Runnicles |author = Rowena Smith |date = 2008-04-28 |publisher = The Guardian |language = en}}</ref>
