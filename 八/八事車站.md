'''八事車站'''（{{jpn|j=八事駅|hg=やごとえき|rm=Yagoto eki|f=y}}）是位在[[日本|日本]][[愛知縣|愛知縣]][[名古屋市|名古屋市]][[昭和區|昭和區]]廣路町字北石坂。為[[名古屋市營地下鐵|名古屋市營地下鐵]]的車站之一。

== 可利用之鐵路路線 ==
*名古屋市營地下鐵
**[[鶴舞線|鶴舞線]] - 車站編號為'''T15'''。
**[[名城線|名城線]] - 車站編號為'''M20'''。

== 車站結構 ==
鶴舞線為1面2線[[島式月台|島式月台]]（往杁中方向設有1條留置線），而名城線則為2面2線[[側式月台|側式月台]]之[[地下車站|地下車站]]。

== 車站週邊設施 ==
本站周圍為高級住宅區域。也因附近有[[中京大學|中京大學]]、[[名城大學|名城大學]]等校區存在，因此車站附近林立了許多適合年輕人喜愛的店家。

*興正寺公園
**八事山興正寺
*八事墓園
*八事商店街
*雲雀岡商店街
*[[三井住友銀行|三井住友銀行]]八事分行
*[[三菱東京UFJ銀行|三菱東京UFJ銀行]]八事分行
*[[愛知銀行|愛知銀行]]八事分行
*名古屋八事郵局
*[[中京大學|中京大學]]名古屋校區
*[[名城大學|名城大學]]八事校區
