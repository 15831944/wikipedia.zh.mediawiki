'''八十場站'''（{{jpn|j=八十場駅|hg=やそばえき|rm=Yasoba eki|f=y}}）是一位於[[日本|日本]][[香川縣|香川縣]][[坂出市|坂出市]]西庄町，隸屬於[[四國旅客鐵道|四國旅客鐵道]]（JR四國）的[[鐵路車站|鐵路車站]]，車站編號為'''Y07'''。本站現時只停靠普通列車，包括每天上下行各1班各站停車的「南風接駁號」。

本站與[[香西站|香西站]]、[[讚岐府中站|讚岐府中站]]及[[讚岐鹽屋站|讚岐鹽屋站]]於1952年設立，然而各站的規模皆為小車站，月台長度只能容納1輛列車，因此絕大部份的列車都會通過而不停靠，使它們的地位和「[[臨時站|臨時站]]」大致相近。直至[[瀨戶大橋線|瀨戶大橋線]]開通及進行[[電氣化|電氣化]]和雙軌化後，月台才得以延長至2輛，並增加停車班次；及後高速化的改善工程繼而再將月台長度延長現時的4輛有效長度。<ref>[http://df50shikoku.web.fc2.com/ekisha/yosan/yasotato.htm] 雙軌化但仍未電化時的八十場站，其中新建往高松月台有效長度仍然為1輛，往坂出的月台下行相對較長。</ref>

車站設立初期，由於月台長度只有兩節長度，因而大部份列車均不停靠。隨著1967年來往[[高松站_(香川縣)|高松站]]及[[端岡站|端岡站]]一段複線化，以及後來[[瀨戶大橋線|瀨戶大橋線]]開通及軌道電化後，車站設備才得到提昇，而停靠的列車亦有所增加。<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/yosan/ST-KOUZAI.htm |title=存档副本 |accessdate=2013-02-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20130624095305/http://www.shikoku.org.uk/JR/LINE/yosan/ST-KOUZAI.htm |archivedate=2013-06-24 }} 非官方車站介紹。</ref>

==車站構造==
自開站起，本站均以簡易車站模式營運，一直以來都沒有興建站舍，車站出入口設於月台末端向[[鴨川站|鴨川站]]方向。在往高松方向的出入口旁設有自動售票機，而兩邊月台間以站外的[[平交道|平交道]]連接。<ref>[http://ku-gyou.net/JR-S/yasoba.html] 另一個非官方車站介紹。</ref>

設有[[側式月台|側式月台]]2個，提供2線2面的配置。月台上並沒有標示月台編號，而是以上、下行的方式來標示，但在售票機上則有顯往高松方向的月台是1號月台；而往坂出方向則為2號月台。<ref>[http://img5.blogs.yahoo.co.jp/ybi/1/b9/11/isemira117/folder/236016/img_236016_9133555_3?1220429229] 圖片攝於2008年。</ref>路軌雖與[[瀨戶大橋線|瀨戶大橋線]]的列車共用，但由於所有瀨戶大橋線列車均不停靠，因此月台的上、下行均以予讚線的方向來命名。

兩邊均設有蓋候車亭，建於舊月台基座上，亦設候車座椅。至於加建部份則皆以鋼板及角鐵所搭建而成。<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/yosan/ST-YASOBA.jpg |title=存档副本 |accessdate=2013-02-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304074014/http://www.shikoku.org.uk/JR/LINE/yosan/ST-YASOBA.jpg |archivedate=2016-03-04 }} 圖片取自非官方車站介紹。</ref>
令各邊月台的有效長度為4輛，另外出入口均為無障礙斜道。
