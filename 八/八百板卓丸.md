'''八百板卓丸'''（{{jpn|j=八百板卓丸|hg=やおいた たくまる|rm=Yaoita Takumaru}}，{{bd|1997年|1月17日|catIdx=Yaoita Takumaru}}）是一名日本職業棒球選手，擔任[[外野手|外野手]]，右投左打，現效力於[[日本棒球機構|日本棒球機構]][[太平洋聯盟|太平洋聯盟]]球隊[[東北樂天金鷲|東北樂天金鷲]]，[[福島縣|福島縣]][[福島市|福島市]]出身<ref name="nikkan141121">{{cite web|url=http://www.nikkansports.com/baseball/professional/draft/2014/news/p-bb-tp0-20141121-1398954.html|title=楽天育成１位八百板「３年以内に支配下」|publisher=[[日刊體育|日刊體育]]|date=2014-11-21|accessdate=2018-05-16|deadurl=no|archiveurl=https://web.archive.org/web/20170808153446/https://www.nikkansports.com/baseball/professional/draft/2014/news/p-bb-tp0-20141121-1398954.html|archivedate=2017-08-08}}</ref>。

== 經歷 ==
{{Link-ja|福島市立鎌田小學校|福島市立鎌田小学校}}1年級時，八百板開始打棒球，升讀{{Link-ja|福島市立福島第三中學校|福島市立福島第三中学校}}後，在當地球隊擔任[[投手|投手]]和[[內野手|內野手]]，中三時晉身全國大會。後來，他升讀[[聖光學院高等學校|聖光學院高等學校]]，並且在高二時擔任後備球員<ref name="nikkan141121"/>。高三時，他[[日本高中棒球錦標賽|日本高中棒球錦標賽]]全國大會，四場賽事中以第一棒[[左外野手|左外野手]]身份，錄得15[[打數|打數]]8[[安打|安打]]、[[打擊率|打擊率]].533，其中在八強戰對{{Link-ja|日本文理高等學校|日本文理高等学校}}的賽事中雖然落敗，但是他在對{{Link-ja|飯塚悟史|飯塚悟史}}時打出三支安打<ref>{{cite web|url=http://www.asahi.com/koshien/game/2014/400/116553/ |title=バーチャル高校野球：聖光学院―神戸国際大付（1回戦） - 試合結果 - 全国高校野球選手権大会 |publisher=[[朝日新聞|朝日新聞]] |date=2014-08-14 |accessdate=2018-05-16}}</ref><ref>{{cite web|url=http://www.asahi.com/koshien/game/2014/400/116718/ |title=バーチャル高校野球：聖光学院―佐久長聖（2回戦） - 試合結果 - 全国高校野球選手権大会 |publisher=朝日新聞 |date=2014-08-19 |accessdate=2018-05-16}}</ref><ref>{{cite web|url=http://www.asahi.com/koshien/game/2014/400/116774/ |title=バーチャル高校野球：聖光学院―近江（3回戦） - 試合結果 - 全国高校野球選手権大会 |publisher=朝日新聞 |date=2014-08-21 |accessdate=2018-05-16}}</ref><ref>{{cite web|url=http://www.asahi.com/koshien/game/2014/400/116851/ |title=バーチャル高校野球：日本文理―聖光学院（準々決勝） - 試合結果 - 全国高校野球選手権大会 |publisher=朝日新聞 |date=2014-08-22 |accessdate=2018-05-16}}</ref>。2014年10月23日，八百板在日本職業棒球選秀會議中獲東北樂天金鷲{{Link-ja|育成選手制度 (日本)|育成選手制度 (日本プロ野球)|育成}}第一指名<ref>{{cite web |url=https://www.rakuteneagles.jp/news/detail/4673.html |title=【2014年ドラフト会議】「契約交渉権獲得」選手のお知らせ |publisher=東北樂天金鷲 |date=2014-10-23 |accessdate=2018-05-16 |deadurl=no |archiveurl=https://web.archive.org/web/20150924120644/http://www.rakuteneagles.jp/news/detail/4673.html |archivedate=2015-09-24 }}</ref>，預付200萬日元，年薪240萬日元為條件加入球隊<ref name="nikkan141121"/><ref>{{cite web|url=http://www.rakuteneagles.jp/news/detail/4755.html |title=2015年度：新入団選手発表会 |publisher=東北樂天金鷲 |date=2014-12-01 |accessdate=2018-05-16}}</ref>。

=== 樂天 ===
2015年5月13日，八百板在{{Link-ja|東部聯賽|イースタン・リーグ}}對{{Link-ja|大塚豐|大塚豊}}打出職業生涯首支安打，該年其在二軍的打率為.230，[[上壘率|上壘率]]則是.400。2016年，他作為二軍出賽70次，取得打率.262、出壘率.355和[[整體攻擊指數|整體攻擊指數]].712的成績。在{{Link-ja|2016亞洲冬季棒球聯盟|2016年のアジアウインターベースボールリーグ}}，他取代因治疗智齒而退出的[[歐可耶瑠偉|歐可耶瑠偉]]，作為東部聯賽選拔成員出賽，取得31打數12安打的成績。2017年7月31日，他成為正式選手，背號由122改為95，在二軍出賽105次，以打率.281取得聯賽第2位。該季結束後，其背號再改為「57」號<ref>{{cite web|url=https://www.rakuteneagles.jp/news/detail/00000856.html|title=背番号の変更に関して|publisher=東北樂天金鷲|date=2017-12-05|accessdate=2018-05-16|deadurl=no|archiveurl=https://web.archive.org/web/20171206135959/https://www.rakuteneagles.jp/news/detail/00000856.html|archivedate=2017-12-06}}</ref>。

== 私生活 ==
八百板在高中時50米跑紀錄為6秒0<ref name="nikkan141121"/>。他的名字卓丸的「卓」字取自前[[讀賣巨人|讀賣巨人]]投手[[江川卓|江川卓]]，“丸”則是指[[棒球|棒球]]<ref name="nikkan141121"/>。

== 數據 ==
=== 紀錄 ===
* 首次出賽：2018年4月15日，對[[埼玉西武獅|埼玉西武獅]]的第3戰（[[宮城球場|樂天生命Park宮城]]），在8局下作為[[茂木榮五郎|茂木榮五郎]]的[[代跑|代跑]]出賽
* 首次先發出賽：2018年4月18日，對[[福岡軟銀鷹|福岡軟銀鷹]]的第5戰（[[福岡巨蛋|福岡雅虎臣蛋]]），以第9棒左外野手身份先發登場
* 首次[[打席數|打席]]：同上，第2局上對[[瑞克·范登赫克|瑞克·范登赫克]]時揮空被[[三振|三振]]
* 首次安打：2018年4月29日，對埼玉西武獅的第5戰（[[西武巨蛋|西武巨蛋]]），第3回上對[[多和田真三郎|多和田真三郎]]時打出中前安打
* 首次盜壘：同上，第3局上盜二壘（投手：多和田、[[捕手|捕手]]：[[森友哉|森友哉]]）
