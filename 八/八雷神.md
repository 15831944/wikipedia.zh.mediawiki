'''八雷神'''（{{lang-ja|ヤクサイカヅチノカミ}}）乃《[[古事記|古事記]]》之記述，《[[日本書紀|日本書紀]]》寫成'''八色雷公'''，祂們是[[日本神話|日本神話]]裡的[[雷神|雷神]]，別名有'''火雷神'''（{{ja}}ホノイカヅチノカミ）、'''火雷大神'''（{{ja}}ホノイカヅチノオオカミ）、'''雷神'''（{{ja}}イカヅチノカミ）等。

== 概要 ==
=== 古事記之記載 ===
根據《古事記》上卷之記錄，[[伊奘諾尊|伊邪那岐命]]砍殺燒死[[伊奘冉尊|伊邪那美命]]的[[迦具土|迦具土]]後，因為非常思念祂，遂追往[[黃泉|黃泉國]]。在黃泉國大殿門前，伊邪那美命由殿內開門迎接，伊邪那岐命對祂說：「我的愛妻呀，我和妳創造的國土尚未完成，請跟我回去吧！」伊邪那美命答：「真是可惜呀！你不早點來，我已經吃了黃泉國的飯食。既然你特意前來找我，我也願意回去呀。且讓我跟黃泉神商量，在這期間不要偷看我。」伊邪那美命邊說邊走入殿內。

伊邪那美命久而未返，伊邪那岐命等得不耐煩，於是取下左髮髻插著的髮梳，折下一齒作燭火，走入殿中。但祂所看見的卻是伊邪那美命腐爛的身軀上佈滿了蛆，且發現頭部被'''大雷'''盤據、胸部被'''火雷'''盤據、腹部被'''黑雷'''盤據、下陰被'''拆雷'''盤據、左手被'''若雷'''盤據、右手被'''土雷'''盤據、左足被'''鳴雷'''盤據、右足被'''伏雷'''盤據，共有'''八雷神'''繞纏其屍身。

伊邪那岐命嚇得拔腿就跑，伊邪那美命怒斥：「你這樣簡直是汙辱我！」立馬派遣[[豫母都志許賣|豫母都志許賣]]追捕。伊邪那岐命丟出黑御鬘變成山[[葡萄|葡萄]]，趁豫母都志許賣撿食之際逃跑，卻仍被追上。再丟出右髮髻插著的髮梳，折下一齒變成[[筍|竹笋]]，又趁豫母都志許賣撿食之際逃逸。後來伊邪那美命遣'''八雷神'''率領一千五百黃泉大軍追趕，嚇得伊邪那岐命抽出[[天之尾羽張|十拳劍]]一邊揮舞一邊逃走。

=== 日本書紀之記載 ===
據《日本書紀》卷第一神代上第九種說法之記載，[[伊奘諾尊|伊奘諾尊]]欲見其妹[[伊奘冉尊|伊奘冉尊]]，乃到殯殮之處。本來伊奘冉尊出來迎接祂，都如同在世般無異狀。接著後者說：「我的夫君，請勿看我。」語畢突然消失且四周陷入一片黑暗，伊弉諾尊舉起火把視之，伊奘冉尊的屍身腫脹，上有'''八色雷公'''。受到驚嚇的伊弉諾尊急忙走避，八色雷公追趕之。伊弉諾尊躲進路邊的桃樹下，並以果實擊退眾雷，這就是[[桃子|桃子]]能驅[[鬼|鬼]]的源由。伊奘諾尊乃投其杖曰：「自此之後，雷不敢來。」便是所謂的[[岐神|岐神]]，也是來名戶之祖神。而所謂的「八雷」，在首曰'''大雷'''、在胸曰'''火雷'''、在腹曰'''土雷'''、在背曰'''稚雷'''、在尻曰'''黑雷'''、在手曰'''山雷'''、在足上曰'''野雷'''、在陰上曰'''裂雷'''。

== 解說 ==
此八位神祇之神名解釋如下<ref>參考{{ja}}[http://www.zb.em-net.ne.jp/~kiokunomori/html/mukashi/shinnmei/shinnmei-4.html 神名釈義　名義ミニ事典]中，「大雷 （オオイカヅチ）」至「伏雷 （フスイカヅチ）」等8個段落。</ref>：
# 大雷（{{lang|ja|オオイカヅチ}}）：「大」表示其威力，「雷（{{ja}}ヅチ）」即為神靈之意。
# 火雷（{{lang|ja|ホノイカヅチ}}）：意表伴隨著雷的火光。
# 黑雷（{{lang|ja|クロイカヅチ}}）：雷鳴時天色昏暗之意。
# 拆雷（{{lang|ja|サクイカヅチ}}）：落雷劈開物體之樣貌，其中「拆（{{ja}}サク）」本義為裂開，代表[[女陰|女陰]]裂縫。
# 若雷（{{lang|ja|ワカイカヅチ}}）：「若」在[[日語|日語]]裡表示年輕，故代表年輕旺盛之力。
# 土雷（{{lang|ja|ツチイカヅチ}}）：落雷落至大地之樣貌。
# 鳴雷（{{lang|ja|ナルイカヅチ}}）：表示雷鳴聲。
# 伏雷（{{lang|ja|フスイカヅチ}}）：表示雷潛伏在雲層間之樣貌。
