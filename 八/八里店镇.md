'''八里店鎮'''是[[浙江省|浙江省]][[湖州市|湖州市]][[吳興區|吳興區]]轄鎮，為吳興區政府駐地。該鎮西靠湖州市區，東臨[[織里鎮|織里鎮]]，南靠[[318國道|318國道]]和[[長湖申航道|長湖申航道]]，[[318國道|318國道]]和湖織大道貫通全境。全鎮區域面積90平方公里，轄38個[[行政村|行政村]]，人口5.5萬。

== 歷史沿革 ==
*[[宋元|宋元]]時期屬[[烏程縣|烏程縣]]德政鄉仁義里、烏山里。
*[[明|明]]屬烏程縣九區七十二都；[[清|清]]屬烏程縣九區七十二都96莊。
*1984年2月，原升山公社（駐地八里店）改為升山鄉，恢複行政鄉建制。
*1993年8月和10月，升山、常路兩鄉相繼撤鄉合並建鎮，召開了首屆人民代表大會，正式成立八里店鎮人民政府。
*2001年6月，撤銷原戴山鎮和八里店鎮，成立新的八里店鎮。

== 行政區劃 ==
八里店鎮轄1個[[社區|社區]]、38個[[行政村|行政村]]<ref>2008年3月10日浙政函〔2008〕33號批複織里鎮東喬村、沈溇村、大溇村等3個行政村劃歸八里店鎮管轄。調整後，八里店鎮轄38個行政村、1個社區，鎮政府駐地不變（烏山村）；織里鎮轄43個行政村、11個社區，鎮政府駐地不變（吳興大道8號）。</ref>（2008年），鎮政府駐地為烏山村。

== 經濟 ==
全鎮共有企業62家，2003年全鎮國內生產總值19.3億元，工農業總產值73.8億元，農民人均收入 6600元。

八里店鎮的「[[新農村建設|新農村建設]]」知名度很高，中國[[央視|央視]]曾於當地採訪報導并作為先進事例像全國宣傳，並被稱為[[社會主義|社會主義]]新農村建設的「'''湖州模式'''」中的集大成者。
