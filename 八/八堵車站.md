'''八堵車站'''位於[[臺灣|臺灣]][[基隆市|基隆市]][[暖暖區|暖暖區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(北段)|縱貫線]]、[[宜蘭線|宜蘭線]]的[[鐵路車站|鐵路車站]]。本站也是縱貫線（[[西部幹線|西部幹線]]）和宜蘭線（[[東部幹線|東部幹線]]）的分歧站。將與計畫中的[[基隆輕軌|基隆輕軌]]山線提供轉乘服務。

== 車站構造 ==
平面車站，[[島式月臺|島式月臺]]一座（西部幹線）、[[側式月臺|側式月臺]]兩座（東部幹線）。

實際上第二月臺及第三月臺相夾同一股道，但是第二月臺B側封閉不開放。且第二、三月台高度有落差，即使第二月臺B側開放亦無法正常使用。

== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#二等站|二等站]]。

*在過去由[[基隆車站|基隆站]]發車或到達的部分[[臺鐵區間車|區間車]]，會經過本站至[[七堵車站|七堵]]折返與[[東部幹線|東部幹線]]直通運行，但目前已無如此運行的車班。東部幹線有部分下行和上行[[對號列車|對號列車]]會在本站停靠。

*1979年起，西部幹線完成[[鐵路電氣化_(十大建設)|鐵路電氣化]]後，但[[宜蘭線|宜蘭線]]及東部幹線仍未完成電氣化之前，不少橫跨東、西線的列車會在此站更換[[鐵路機車|火車頭]]。

*本站為[[平溪線|平溪線]]列車始發站及終點站（不經[[海科館車站|海科館站]]）。

*本站曾為了減少月台曲度而經改建過，將整個站場北移。原本東部幹線宜蘭線的實際定義起點為[[基隆車站|基隆車站]]，並經由八堵折返向東部延伸，但由於站內軌道已修改，折返點改往七堵。原本基隆至八堵間的東部幹線專用路軌，也改為與縱貫線共用。

*本站為少數[[臺灣鐵路管理局車站等級#三等站|三等站]]以上的車站中無辦理行車業務的車站。

*本站因經過閘門設施的升級，因此可使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費<ref>[http://www.railway.gov.tw/tw/CP.aspx?sn=1312&n=1333 臺鐵多卡通電子票證乘車營運規則]</ref>。

== 歷史 ==
*[[臺灣清治時期|清代]]末期即已設置'''八堵火車碼頭'''，不過實際上位於今日[[七堵車站|七堵車站]]的前站。
*1899年7月20日：設立'''八堵假乘降場'''。
*1902年6月1日：改為停車場。
*1919年5月5日：[[瑞芳車站|瑞芳]]－八堵段通車，本站成為分歧站。
*1947年：發生[[八堵車站事件|八堵車站事件]]。
*1951年1月17日：木造站房重建完工。
*1984年7月31日：遷入臨時站房。
*1985年：木造站房拆除。
*1986年：水泥磚造站房完工。
*1994年：二二八紀念碑完工。
*2007年5月8日：因地方民意要求，部份對號列車停靠本站。
