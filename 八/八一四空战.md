'''八一四空戰'''，是[[中國抗日戰爭|中國抗日戰爭]][[淞滬會戰|淞滬會戰]]期間，[[中華民國空軍|中華民國空軍]]和大日本帝国海軍航空隊之間的一次[[空戰|空戰]]，時間發生在1937年8月14日，因而得名。

== 經過 ==
[[File:The_August_14th_bombing_in_front_of_the_Great_World.jpg|thumb]]
繼八一三淞滬會戰開打後，日本空軍對於上海等地的威脅倍增，為強化空防，空軍發出第一號作戰命令，調遣第四大隊飛赴杭州保衛筧橋機場，四大隊出動27架霍克三型飛機。

1937年（民國二十六年）8月14日，中華民國空軍飛上海作戰，曾炸日本[[日本國駐上海總領事館|領事館]]及[[出雲號巡洋艦|出雲號]]旗艦未中；然在[[瀏河|瀏河]]則炸沉日艦一艘、傷四艘，中國飛機一架受傷<ref name="畫史">李怡著，《抗戰畫史》，台北：力行書局，1969年</ref>{{rp|73}}。10時，中華民國空軍首次出動，轟炸[[虹口|虹口]]、[[滙山|滙山]][[碼頭|碼頭]]等處日軍據點<ref name="戰爭史地圖集">武月星主編：《中國抗日戰爭史地圖集》，北京：中國地圖出版社，1995年8月</ref>{{rp|68}}。中國空軍轟炸[[上海海軍特別陸戰隊|日本海軍陸戰隊]]司令部，炸傷10餘艘日艦<ref name="戰爭史地圖集"/>{{rp|67}}。上海上空空戰激烈，日方之高射炮火滿佈天空，[[吳淞口|吳淞口]]炮戰亦烈<ref name="畫史"/>{{rp|73}}。中國飛機一架因彈架損壞，二彈意外墜落娛樂場[[上海大世界|上海大世界]]前，造成三千餘人死傷的事件，稱為[[大世界墜彈慘案|大世界墜彈慘案]]，至為痛惜<ref name="畫史"/>{{rp|73}}。

14時50分，日機最精銳之聯隊鹿屋海軍航空隊18架[[九六式陸上攻擊機|九六式陸上攻擊機]]由[[台北松山機場|台北松山機場]]起飛後分兩路侵襲，第一編隊新田少佐率九架於十二時五十五分起飛轟炸[[杭州|杭州]][[笕桥机场|笕桥机场]]，第二編隊由淺野少佐率九架於十三時零五分起飛轟炸[[廣德|廣德]]機場<ref name="現代史">李守孔著，《中國現代史》，台北：三民書局，1973年9月，ISBN 9571406635</ref>{{rp|112}}。中國空軍則使用美製[[BF2C|BF2C]](霍克III)戰鬥機，由第四大隊隊長[[高志航|高志航]]率隊共計27架起飛迎戰。此時一架日機入場炸中油罐起火燃燒，暴露形蹤被高志航與[[譚文|譚文]]合力擊落，墜毀於半山。二十一隊[[李桂丹|李桂丹]]與[[柳哲生|柳哲生]]、[[王文驊|王文驊]]共同擊中一架日機，使之起火墜毀於[[錢塘江|錢塘江]]口。高大隊長再攻擊大串機，該機中彈七十餘發，左引擎熄火，勉強飛抵台北迫降，此機現於日本海軍館展示。在廣德的二十二隊加油後再起飛，分隊長鄭少愚至翁家埠發現一架日機(在廣德附近投彈後折回的淺野隊)，追擊至[[曹娥江|曹娥江]]，經連續攻擊六、七次，該機受重創，迫降於基隆社寮島(今和平島)附近，重傷報廢。外傳日機進犯大陸當日颱風天候甚差，誤判被擊傷之日機為被擊落，中國方面記載戰果為六比零。日本方面調查則稱，當天日機現場被擊落兩架，另有兩架一是油盡墜海另一是返航但傷重報廢<ref> 吳炳宏 (2017) 814空戰80周年 張文還原史實. 世界日報 20170825 </ref>。中國空軍則是現場沒有被擊落者，但其中一機油盡墜毀，飛行員劉樹籓殉職<ref> 作者不詳。八一四空戰 簡史. 退輔會 光榮雙周刊 2194期 20120808 </ref>。故以擊落論，二比零。以飛機損失論，四比一。此役，[[中華民國空軍|中國空軍]]成功阻止了日軍摧毀機場的意圖，創下六：O輝煌紀錄。

== 事實參考 ==

*前行政院長記述[[衡陽保衛戰|衡陽保衛戰]]歷史小說，轉載二戰前義大利總理[[貝尼托·墨索里尼|墨索里尼]]給高志航的信，信中稱讚高的特技飛行讓他心動，可見高之戰技非比尋常。八一四前，在情報專家[[戴笠|戴笠]]支持下，空軍先已在沿海布設了對空監測站，所以日機進入海岸時已被看到（證明不是風雨天），並由總台長陳一白電告高與基地防空指揮官（航校教育長）[[蔣堅忍|蔣堅忍]]。因而先由南京趕到之高志航，於機隊趕到筧橋時，能用手勢讓其逕自直接迎敵，他自己亦能即時升空，始得殲敵致果。

*筧橋空戰域內無颱風
**外傳這批日機是跟著颱風進犯大陸的。但事實為日本「渡洋爆擊隊」於八月七日隨著颱風尾，以單機追蹤隊形飛到松山機場，十四日下午分兩批進攻杭州筧橋與北面的常德，最後以六比○鎩羽而歸。

# 根據中央研究院氣象所前所長[[竺可楨|竺可楨]]日記記述。竺是庚子退款農學類留學生，與[[胡適|胡適]]同屆，赴美後發現志趣不合，改習氣象，獲哈佛大學博士，返國後先後執教武漢與南京高等師範，後任氣象所長，民國廿五年任浙江大學校長，仍兼氣象業務一把抓（氣象局卅五年才成立）的所長職。他的日記對氣象事件是專家記錄。日記中除記颱風於八月四日過舟山，風速達每秒四十二公尺，次日南京淹水外，也在八月十四日記述當天由杭州坐火車赴蘇州，沿途為多雲時有陣雨的天氣，車中聞飛機聲，未提到另有颱風。
# 中央氣象局百年颱風路徑圖中，經海峽過舟山之颱風發生在八月四日，非十四日。
# 日主力九六式戰轟機，最大航速約每秒一百公尺，無能頂颱風之強風到杭州；當時我霍克三戰機速度約與敵機同，且無座艙罩，亦難於風雨中升空殺敵。
