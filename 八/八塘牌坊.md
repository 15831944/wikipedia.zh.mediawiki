'''八塘牌坊'''，全稱“旌表處士殷璧光之妻六品銜庠生義璜之母李氏坊”，因位於獅子橋旁,也稱為'''獅子橋牌坊'''<ref>《重慶文物總目》，璧山縣，第457頁</ref>，位於[[重慶市|重慶市]][[璧山縣|璧山縣]]八塘鄉西15米，修建於嘉慶十一年(1806年)，是一座[[節孝牌坊|節孝牌坊]]，是重慶市璧山縣文物保護單位。

旌表牌坊是中國社會中特有的表揚性建築，藉以教化人心，多建在舊時的官道。八塘獅子橋牌坊亦為其中之一，位於重慶到成都的古道上，石質仿木結構，重檐山頂。檐下施斗栱四朵，次間各施斗栱兩朵。坊身、斗栱及部分額坊上有淺浮雕圖案。坊為四柱三間。
[[File:璧山縣八塘牌坊2.JPG|thumb]]

在[[文革|文革]]時期險被拆除，但因牌坊兩端的石柱嵌在民房之中，如若拆除將會損毀村民房屋。後經當時的村主任劉太英找人進行粉刷，並寫上毛語錄才得以保存<ref>八塘：老街入選重慶最美小巷50強 {{cite web |url=http://www.bswxw.com/bs_2011/2011-07/27/content_1662534.htm |title=存档副本 |accessdate=2012-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20160305003321/http://www.bswxw.com/bs_2011/2011-07/27/content_1662534.htm |archivedate=2016-03-05 }}</ref>。但由於近年處理除去水泥等填充物方式不恰當，造成了大部分文字的破壞。

== 牌坊文字 ==
牌坊兩面文字一致，大部文字雖能辨認，但都有不同程度破壞痕跡。
* 牌坊斗栱下書“大清嘉慶十一年丙寅歲孟冬月建”
* 正上匾下書銜名“[[四川總督|四川總督]][[勒保|勒保]]、[[提督四川等處學政|提督學政]][[錢栻|錢栻]]<ref>乾隆四十三年戊戌科進士</ref>、[[四川等處承宣布政使司|布政使司]][[楊揆|楊揆]]、[[按察使司|按察使司]][[劉清_(清朝)|劉清]]、[[川東道|四川東道]][[嚴文鋐|嚴文鋐]]<ref>嚴文鋐，[[江蘇|江蘇]][[丹徒|丹徒]]拔貢，嘉慶四年任，十九年再任。《重慶府職官志》，題名，第80頁</ref>、[[重慶|重慶]][[知府|知府]][[石韞玉|石韞玉]]<ref>乾隆五十五年庚戌恩科狀元</ref>、[[璧山|璧山]][[知縣|知縣]][[董淳|董淳]]<ref>董淳，[[山東|山東]][[鄒嶧_(地名)|鄒嶧]]舉人，嘉慶七年署。同治《璧山縣志》卷六職官知縣，第5頁</ref>、璧山知縣[[程齊誥|程齊誥]]<ref>程齊誥，[[湖北|湖北]][[雲夢|雲夢]]拔貢，嘉慶八年任，十五年復任。同治《璧山縣志》卷六職官知縣，第5頁</ref>、璧山[[訓導|訓導]][[劉翀_(清代)|劉翀]]<ref>劉翀，[[華陽|華陽]][[舉人|舉人]]，嘉慶七年任至道光二年陞[[雅州府|雅州府]]教授。同治《璧山縣志》卷六職官知儒學，第9頁</ref>”
* 正上匾題“節孝”二大字
* 其下為坊名“旌表處士[[殷璧元|殷璧元]]之妻六品銜庠生[[殷義璜|義璜]]之母[[李氏|李氏]]坊”
* 左下串坊大部文字已泐，殘餘“庠……曾孫成……”數個字

== 坊主 ==
[[殷璧元|殷璧元]]妻[[李氏|李氏]]，[[同治|同治]]《[[璧山縣志|璧山縣志]]》有載：“殷璧元妻李氏年三十夫亡，事孀姑撫幼子備歷艱辛，子入武庠，堵禦教匪有功，嘉慶八年旌表（通志已載）”

其子[[殷義璜|殷義璜]]，同書也有載：“[[嘉慶|嘉慶]]四年，教匪竄擾[[江北|江北]]、[[合州|合州]]等處，[[知縣|知縣]][[張人龍_(清代)|張人龍]]諭民築寨、練團、製戰器、備城守，復鑄[[大礮|大礮]]數十座，與巴紳[[劉國輔|劉國輔]]率[[徐玉龍|徐玉龍]]、殷義璜及[[左墤|左墤]]、[[羅廷颺|羅廷颺]]等督[[鄉勇|鄉勇]]千餘人於[[澄江口|澄江口]]、[[北碚|北碚]]、[[炭垻|炭垻]]、[[無息溪|無息溪]]、[[馬項頸|馬項頸]]、[[風埡|風埡]]諸處，日夜輪守，相拒年餘，賊不敢渡江，復造渡船數隻，全活北岸難民無算。”<ref>同治《璧山縣志》卷五，武功武備，第13頁</ref>
