《'''八路军'''》，2005年上映于[[中国大陆|中国大陆]]的战争剧，导演[[宋业民|宋业民]]、[[董亚春|董亚春]]，主演[[唐国强|唐国强]]、[[刘劲|刘劲]]、[[王伍福|王伍福]]、[[郭连文|郭连文]]、[[卢奇|卢奇]]、[[姚居德|姚居德]]。

== 剧情 ==
1937年7月7日，[[大日本帝国|大日本帝国]]发动了[[七七事变|七七事变]]，开始全面侵略中国，毛泽东、朱德等领导的[[中国共产党|中国共产党]]和蒋中正领导的[[中国国民党|中国国民党]]结成[[抗日民族统一战线|抗日民族统一战线]]，抗日救国。1945年在[[美国|美国]]、[[苏联|苏联]]的帮助下取得了[[抗日战争|抗日战争]]胜利。<ref>{{cite web |url=http://yule.sohu.com/20060720/n244365598.shtml |title=电视连续剧《八路军》剧情简介  |accessdate=2006年7月20日 |publisher=搜狐网 |work= |language =zh-cn}}</ref>

== 花絮 ==
该剧是“纪念抗日战争胜利暨反法西斯战争胜利60周年”的献礼剧。<ref>{{cite web |url=http://ent.hunantv.com/e/20080720/23487.html |title=电视剧《八路军》：纪念抗战胜利60周年  |accessdate=2008 |publisher=金鹰网 |work= |language =zh-cn}}</ref>

该剧由中国中央电视台、山西省委宣传部、[[八一电影制片厂|八一电影制片厂]]、山西广播电视总台合作拍摄，在[[北京|北京]]、[[河北|河北]]、[[天津|天津]]、[[山西|山西]]、[[湖北|湖北]]等地取景。<ref name=" cat" >{{cite web |url=http://news.xinhuanet.com/ent/2005-07/26/content_3268332.htm |title=《八路军》浴血抗日全景 8月10日CCTV-1播出 |accessdate=2005年7月26日 |publisher=新华网 |work= |language =zh-cn}}</ref>

该剧启用了[[中国人民解放军|军队]]的官兵做群众演员。<ref name=" cat" />

该剧缺点诸多，如八路军在“[[平型关大捷|平型关大捷]]”文艺兵用乐器[[三弦|三弦]]打死日军；刚打仗完，冯玉兰跳[[日本|日本]][[樱花舞|樱花舞]]，老百姓和八路军战士还鼓掌。该剧一些内容直接引用了电视剧《延安颂》中的部分片段。<ref>{{cite web |url=http://yule.sohu.com/20050822/n226744523.shtml |title=《八路军》有硬伤：文艺兵用三弦击倒日本兵 |accessdate=  2005-08-22|publisher=搜狐网 |work= |language =zh-cn}}</ref>

该剧最早2005年8月10日在中国中央电视台综合频道播出。<ref>{{cite web |url= http://news.xinhuanet.com/ent/2005-07/26/content_3268336.htm|title=全面反映八路军抗战史 《八路军》将跃马央视 |accessdate= 2005年7月26日|publisher= 新华网|work= |language =zh-cn}}</ref>
