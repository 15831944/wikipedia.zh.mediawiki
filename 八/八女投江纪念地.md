'''{{FULLPAGENAME}}'''又称'''八女投江殉难地'''，位于[[黑龙江省|黑龙江省]][[牡丹江市|牡丹江市]][[林口县|林口县]]。是为纪念八位在此牺牲的[[东北抗日联军|东北抗日联军]]女战士（[[八女投江|八女投江]]）所修建的一系列纪念性建筑物<ref name="a1">{{Cite web |url=http://www.chinamartyrs.gov.cn/GuoJiaJiKangZhan/GuoJiaJiLingYuanJianJie/jnss31.html |title=“八女投江”殉难地|accessdate=2018-10-09 |author= |date= |work= |publisher=中华英烈网|language=Zh-hans }}</ref>。

1980年，经过调查确认[[林口县|林口县]][[刁翎镇|刁翎镇]]三家子村乌斯浑河畔为八名东北抗联女战士殉难地<ref name="a1"/>。1982年，林口县在此地修建八女投江烈士纪念碑，刻有“八女英魂光照千秋”八个字。由时任黑龙江省长、原东北抗联成员[[陈雷_(省长)|陈雷]]题字<ref>{{Cite web |url=http://dangshi.people.com.cn/n/2015/0806/c85037-27419614.html |title=八女投江弹尽殉国山河恸|accessdate=2018-10-09 |author= |date=2015-08-06 |work= |publisher=中国共产党新闻网，来源：黑龙江日报|language=Zh-hans }}</ref>。1995年，重新修建纪念碑和纪念馆。纪念碑高10米，正面仍为陈雷原有题词。1999年2月1日，[[黑龙江省人民政府|黑龙江省人民政府]]将'''八女投江纪念地'''为[[第四批黑龙江省文物保护单位|第四批黑龙江省文物保护单位]]。2005年，[[国家旅游局|国家旅游局]][[全国红色旅游工作协调小组|全国红色旅游工作协调小组]]列为[[全国红色旅游经典景区|全国红色旅游经典景区]]。2010年，中共黑龙江省委、省人民政府列为黑龙江省[[爱国主义教育基地|爱国主义教育基地]]<ref name="a1"/>。2014年，[[中华人民共和国国务院|中国国务院]]以'''“八女投江”殉难地'''之名列入[[第一批国家级抗战纪念设施、遗址名录|第一批国家级抗战纪念设施、遗址名录]]。
