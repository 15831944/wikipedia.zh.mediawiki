'''八戶自動車道'''（{{jpn|j=八戸自動車道|hg=はちのへじどうしゃどう|rm=Hachinohe jidōshadō|f=y}}）是從[[岩手縣|岩手縣]][[八幡平市|八幡平市]]{{link-ja|安代系統交流道|安代ジャンクション}}至[[青森縣|青森縣]][[八戶市|八戶市]]的高速公路（[[高速自動車國道|高速自動車國道]]）。略稱為'''八戶道'''（{{lang|ja|八戸道／はちのへどう}}，HACHINOHE EXPWY）。高速公路路線編號與[[青森自動車道|青森自動車道]]共同被編為{{Ja Exp Route Sign|E4A}}。

== 概要 ==
[[國土開發幹線自動車道|法定路線名稱]]為'''[[東北縱貫自動車道|東北縱貫自動車道]]八戶線'''。預定於八戶JCT與地域高規格道路[[八戶久慈自動車道|八戶久慈自動車道]]連接。

將來亦預定連接[[百石道路|百石道路]]、[[第二陸奧收費道路|第二陸奧收費道路]]、[[上北道路|上北道路]]、[[三澤天間林道路|三澤天間林道路]]、[[陸奧收費道路|陸奧收費道路]]與[[青森自動車道|青森自動車道]]。

==設施==
* 交流道(IC)編號欄的背景色<span style="color:#BFB">■</span>為已通車區間，設施欄背景色<span style="color:#CCC">■</span>為未通車區間，未通車區間名稱全為暫稱。
* (數字)為其他路線編號，<數字>為預定編號。
* BS（巴士站）：○為使用中、◆為停用中的，未註記者即未設置。
* 終點起的距離 [數字] 為自[[八戶北IC|八戶北IC]]起點的距離。
* 停车区：停車場，SA：服務區，IC：交流道，JCT：系統交流道。

== 歷史 ==
* 1986年11月27日 : 一戶IC-八戶IC間通車。
* 1989年9月7日 : 安代JCT-一戶IC間通車。
* 1995年3月28日 : 一般國道45號百石道路（八戶北IC-下田百石IC）間通車。
* 2002年7月18日 : 八戶JCT-八戶北IC間通車，'''全線通車'''。
* 2005年10月1日 : [[日本道路公團|日本道路公團]]民營化，成為[[東日本高速道路|東日本高速道路株式會社]]管理路線。
* 2010年2月2日：全線路段被指定為高速道路無料化社會試驗對象區間。
