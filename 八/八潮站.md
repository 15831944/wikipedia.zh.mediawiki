'''八潮站'''（{{jpn|j=八潮駅|hg=やしおえき|rm=Yashio eki|f=y}}）是位於[[日本|日本]][[埼玉縣|埼玉縣]][[八潮市|八潮市]]{{link-ja|大瀨 (八潮市)|大瀬 (八潮市)|大瀨}}六丁目，屬於[[首都圈新都市鐵道|首都圈新都市鐵道]][[筑波快線|筑波快線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''TX08'''。此站是八潮市唯一的鐵路車站。

== 歷史 ==
* 2005年（[[平成|平成]]17年）8月24日 - 在八潮市大字大瀬字稗田788番地1開業。
* 2015年（平成27年）1月31日 - [https://www.pref.saitama.lg.jp/a1103/yashionanbu-chuo/index.html 草加都市計劃事業八潮南部中央一體型特定土地區劃整理事業]{{link-ja|換地處分|換地処分}}在前日完成，車站所在地的町名與地號改為八潮市大瀨六丁目5番地1。

== 車站結構 ==
[[島式月台|島式月台]]2面4線的[[高架車站|高架車站]]。入閘的右側是[[秋葉原站|秋葉原]]方向，左側是[[筑波站|筑波]]方向。

== 使用情況 ==
2016年（平成28年）度的1日平均'''上車'''人次為'''20,613人'''<ref name="tx">[http://www.mir.co.jp/company/number.html つくばエクスプレス乗車人員]</ref>。作為筑波快線的快速通過站，是不與其他路線接續的車站當中上車人次最多的。
