'''八坊十三巷'''位于[[甘肃省|甘肃省]][[临夏回族自治州|临夏回族自治州]]首府[[临夏市|临夏市]]主城区，是一片具有[[回族|回族]]民族特色的历史文化街区，[[国家4A级景区|国家4A级景区]]，<ref name="2R">{{Cite web |url = http://m.sohu.com/a/218913861_803374 |title = 临夏市八坊十三巷和积石山大墩峡获批国家AAAA级旅游景区 |website=搜狐网 |date = 2018-01-25 |language = zh |accessdate = 2019-03-09 }}</ref>被称为民族建筑艺术“大观园”<ref name="3R"/>、“甘肃新地标”和临夏文化旅游新名片。<ref>{{Cite web |url = http://www.gslxly.com/neiye_more_gslxyNews.asp?lm_typeID=8838 |title = 八坊十三巷：甘肃文化旅游新地标|website=临夏州文化广电和旅游局|date = 2018-04-23 |language = zh |accessdate = 2019-03-09 }}</ref>街区内居民共2000多户，8200人，回族占98%以上。<ref name="1R"/>

==位置==
八坊十三巷街区位于[[中华人民共和国|中华人民共和国]][[甘肃省|甘肃省]][[临夏回族自治州|临夏回族自治州]][[临夏市|临夏市]]北大街以南，前河沿以北，解放路以西，新西路以东。<ref name="4R"/>

==历史==
临夏古称河州，市内的[[回民|回民]]又称“八坊人”。而八坊十三巷则是[[临夏市|临夏市]]老城区内唯一的保留至今未被破坏性开发的瓦房区。<ref name="3R">{{Cite web |url = http://m.sohu.com/a/222523906_695999 |title = 走进八坊十三巷，探寻老临夏的城市记忆|website=搜狐网 |date = 2018-02-12 |language = zh |accessdate = 2019-03-09 }}</ref>八坊十三巷的历史最早可以追溯到唐朝。<ref name="2R"/>[[大食|大食]]、[[波斯|波斯]]等国的商人及宗教人士来到河州一带经商、传教，一些人在此定居。<ref name="4R">{{Cite web |url = http://image.chinawriter.com.cn/n1/2017/0427/c404019-29240645.html |title = 八坊十三巷：走进时光记忆的深处|website=中国作家网 |date = 2017-04-27 |language = zh |accessdate = 2019-03-09 }}</ref>

八坊十三巷历经了[[宋|宋]]、[[元|元]]、[[明|明]]、[[清|清]]、[[民国|民国]]，随着街区内人口逐步增长，[[回族|回族]]居民开始修建[[清真寺|清真寺]]、设立教坊，逐步形成了一个“围寺而居、围坊而商”的[[穆斯林|穆斯林]]聚居区八坊。<ref name="4R"/>如今的建筑风格始于民国时期，在[[改革开放|改革开放]]后进行的一系列城市建设中被保留了下来。<ref name="1R">{{Cite web |url =
http://www.gslxly.com/neiye_more_gslxyNews.asp?lm_typeID=9379 |title = 临夏有个八坊十三巷|website=临夏州文化广电和旅游局|date = 2018-10-22 |language = zh |accessdate = 2019-03-09 }}</ref>

2015年，临夏市政府开始对八坊十三巷进行保护性开发，修缮了原有的瓦房区。

2017年，临夏市政府进一步开发了总长约3.3公里的街巷，包括专员巷、王寺巷、石桥巷、细巷、坝口巷、小南巷、大南巷、沙尕塄巷、北巷以及红水河两岸。<ref name="1R"/>

2018年1月6日，八坊十三巷获批[[国家4A级旅游景区|国家4A级旅游景区]]。<ref name="2R"/>

==建筑==
八坊十三巷内较完整地保存着41座[[四合院|四合院]]和109座民宅。<ref name="4R"/>
*大公馆，位于大旮巷80号，是一座四合院，占地宽阔。包括了4座正院和后院、南院、车院、花园、库房。该建筑仿照了[[北京四合院|北京四合院]]的建筑形制修建而成，并且融入[[回族|回族]]砖雕、[[汉族|汉族]]木刻、[[藏族|藏族]]彩绘等诸多特色元素，具有较高的建筑水平。现为八坊民俗馆。<ref name="4R"/>
*清真北寺，位于北巷之中，它的影壁建于1741年（清[[乾隆|乾隆]]年间），距今已有276年的历史，是现存最古老的地面大型砖雕影壁。<ref name="4R"/>
