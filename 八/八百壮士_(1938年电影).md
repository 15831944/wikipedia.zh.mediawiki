《'''八百壮士'''》（{{lang-en|'''800 Heroes'''}}）战争片，导演应云卫，主演袁牧之、陈波儿、洪虹。该片曾在[[英属香港|英属香港]]、[[缅甸|缅甸]]、[[菲律宾|菲律宾]]、[[法国|法国]]、[[瑞士|瑞士]]等播出，引发广发反响。<ref>{{cite web |url= http://www.todayonhistory.com/7/3/movie-BaBaiZhuangShi-ZaiGuoNeiWaiYinQiFanXiang.html|title=《八百壮士》 |accessdate= |publisher=历史上的今天 |work= |language=zh-hans}}</ref><ref>{{cite web |url= http://news.163.com/06/0630/09/2KRTOFGD00011EBF.html|title=1938年7月3日 电影《八百壮士》引起反响 |accessdate=2006 |publisher= 网易|work= |language=zh-hans}}</ref>

==剧情==
1937年8月13日，[[大日本帝国|大日本帝国]]侵略[[中华民国|中华民国]][[上海|上海]]，[[中国国民党|中国国民党]]八八师五二四团团长“[[谢晋元|谢晋元]]”带领480名青年军官“[[八百壮士|八百壮士]]”守护四行仓库一共3个月，女[[童子军|童子军]][[杨惠敏|杨惠敏]]同时送食物、送药物，一直到撤退。<ref>{{cite web |url=http://www.china.com.cn/chinese/2005/dybn/1075046.htm |title=八百壮士  |accessdate=2005 |publisher=中国网 |work= |language=zh-hans}}</ref>
