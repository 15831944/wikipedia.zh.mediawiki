'''八一勋章'''是[[中华人民共和国|中华人民共和国]]授予军人的一种[[勳章|勋章]]，是现行[[中华人民共和国军事勋章奖章纪念章列表|中国军事荣誉制度]]中最高的一级，由[[中华人民共和国中央军事委员会|中华人民共和国中央军事委员会]]决定，[[中华人民共和国中央军事委员会主席|中央军事委员会主席]]授予<ref name="报道0424">{{cite web|author1=新华社|title=八一勋章及证书|url=http://www.xinhuanet.com/photo/2017-07/28/c_1121397682_2.htm|website=新华网|accessdate=2018-04-24}}</ref>。授予的对象是[[中国人民解放军|中国人民解放军]]、[[中国人民武装警察部队|中国人民武装警察部队]]和[[公安现役部队|公安现役部队]]的军人和文职人员。

== 历史 ==
2017年6月12日，经中央军委主席[[习近平|习近平]]批准，重新设立“[[八一勋章|八一勋章]]”。新设立的“八一勋章”是由[[中央军事委员会|中央军事委员会]]决定、[[中华人民共和国中央军事委员会主席|中央军委主席]]签发证书并颁授的军队最高荣誉，授予在维护国家主权、安全、发展利益，推进国防和军队现代化建设中建立卓越功勋的军队人员<ref name=xinshe>{{Cite news|url=http://news.xinhuanet.com/2017-06/12/c_1121128933.htm|title=经中央军委主席习近平批准 我军新设立“八一勋章”并组织首次评选|newspaper=新华网|date=2017-06-12}}</ref>。

2017年7月，中央军委主席习近平签署命令，授予麦贤得等10位同志“八一勋章”<ref>{{Cite web|url=http://news.xinhuanet.com/politics/2017-07/28/c_1121395265.htm|title=中央军委主席习近平签署命令 授予麦贤得等同志“八一勋章”|date=2017-07-28|publisher=新华网}}</ref><ref name=elyq/><ref>{{Cite web |url = http://news.ifeng.com/a/20170728/51522244_0.shtml |title = 景海鹏、程开甲、王刚等10人被授予“八一勋章” |author =  |date = 2017-07-28 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。2017年7月28日，中央军委颁授“八一勋章”和授予荣誉称号仪式在北京[[八一大楼|八一大楼]]举行，中共中央总书记、国家主席、中央军委主席习近平向“八一勋章”获得者颁授勋章和证书。这是“八一勋章”首次颁授<ref>{{Cite web |url = http://news.ifeng.com/a/20170728/51524452_0.shtml |title = 习近平向“八一勋章”获得者颁授勋章和证书 |author =  |date =2017-07-28   |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

== 外观 ==
根据官方介绍，“八一勋章”采用了八一军徽、五角星、利剑、旗帜、光芒和长城、橄榄枝等设计元素。五角星衬托八一军徽，表明中国人民解放军从红军走来，并象征军队至高荣誉；五个利剑组合与五角星相呼应，象征我军是听党指挥、能打胜仗、作风优良的人民军队，具有坚不可摧的向心力凝聚力战斗力；旗帜、光芒，象征受勋者在党的旗帜引领下取得辉煌成绩，其事迹和精神也具有旗帜般导向作用，是全军官兵学习的榜样；长城，象征人民军队忠实履行党和人民赋予的神圣使命，坚决捍卫国家主权、安全、发展利益；橄榄枝，象征中国坚定不移走和平发展道路，以及人民军队为捍卫世界和平做出的突出贡献<ref>{{Cite web|url=http://news.xinhuanet.com/mil/2017-07/28/c_129666655.htm|title=军人至高荣誉在强军征程闪耀——“八一勋章”简介|date=2017-07-28|publisher=新华网}}</ref>。
