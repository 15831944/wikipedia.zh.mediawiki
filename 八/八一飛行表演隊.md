'''八一飛行表演隊'''是[[中國人民解放軍空軍|中國人民解放軍空軍]]的飛行表演隊，隶属[[中国人民解放军中部战区空军|中国人民解放军中部战区空军]]。<ref name="zhw">{{Cite web |url = http://military.china.com/news/568/20161118/30043154.html |title = 青春无悔献蓝天 送别“金孔雀”余旭 |author =  |date = 2016-11-18 |publisher =中华网  |language =  |accessdate =  |quote =  }}</ref>

== 简介 ==
1961年，[[印度尼西亚|印度尼西亚]]总统[[蘇加諾|蘇加諾]]应邀訪華，印度尼西亚方面要求按照國際慣例，當总统專機進入中國領空後要有8架戰機護送，[[中华人民共和国国务院总理|中华人民共和国国务院总理]][[周恩來|周恩來]]隨即要求空軍司令员[[劉亞樓|劉亞樓]]迅速抽调8名飞行员，临时组建护航大队，圆满完成了此次护航任务。<ref name="taiguo" />

1962年2月1日，空军司令员[[刘亚楼|刘亚楼]]颁布命令：组建'''中国人民解放军护航表演大队'''，任命[[邵彦魁|邵彦魁]]为护航表演大队队长；[[胡衍林|胡衍林]]为护航表演大队政治委员；[[王正孝|王正孝]]为护航表演大队副队长；[[邹全文|邹全文]]为护航表演大队副政治委员；[[刘光泽|刘光泽]]为护航表演大队参谋长。1962年2月15日，[[中共中央军委|中共中央军委]]颁布《关于成立中国人民解放军空军护航表演大队的命令》。周恩來说：“表演队要与中华人民共和国共存。”[[中华人民共和国主席|中华人民共和国主席]][[刘少奇|刘少奇]]要求表演做到“国家大礼，万无一失。”表演队成立后，[[邓小平|邓小平]]、[[陈毅|陈毅]]、[[贺龙|贺龙]]、[[叶剑英|叶剑英]]、[[聂荣臻|聂荣臻]]、[[罗瑞卿|罗瑞卿]]、[[杨尚昆|杨尚昆]]等人多次观看-{}-表演队的飞行护航表演。1987年8月1日，表演队正式命名为“'''中国人民解放军空军八一飞行表演队'''”，主要任务是对外迎宾及礼仪特技表演飞行。<ref name="wy" /><ref>{{Cite web |url = http://www.81.cn/2016zhhz/2016-10/27/content_7328319.htm |title = 翱翔！中国八一飞行表演队 |author =  |date = 2016-10-27 |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>

八一飞行表演队长期是中国人民解放军空军唯一的飞行表演队。2011年，[[空军航空大学|空军航空大学]]成立“天之翼”飞行表演队，[[空军第三飞行学院|空军第三飞行学院]]成立“红鹰”飞行表演队，使中国人民解放军空军的飞行表演队增加到三支。<ref name="sanzhi" />

2016年[[深化国防和军队改革|深化国防和军队改革]]中，八一飞行表演队的上级单位[[中国人民解放军北京军区空军|中国人民解放军北京军区空军]]撤销，八一飞行表演队转隶[[中国人民解放军中部战区空军|中国人民解放军中部战区空军]]。<ref name=zhw/><ref name="taiguo">{{Cite web |url =  http://www.cankaoxiaoxi.com/mil/20151122/1004687.shtml?fr=pc |title = 中国空军赴泰国飞行表演增进互信和友谊 |author =  |date = 2015-11-22 |website = 参考消息网  |language =  |accessdate =  |quote =  }}</ref>

2015年时，八一飞行表演队飞行员的选拔有三个硬性标准：年龄36岁以下，飞行时间800小时，[[歼-10|歼-10]]飞行总时间200小时以上。<ref name="wy">{{Cite web |url = http://news.163.com/15/0911/05/B375ML5P00014AED.html |title = 本报独家探访八一飞行表演队 |author =  |date =  2015-09-11|publisher =网易  |language =  |accessdate =  |quote =  }}</ref>
