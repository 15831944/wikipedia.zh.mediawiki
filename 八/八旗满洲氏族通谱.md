'''《八旗满洲氏族通谱》'''，简称'''《通谱》'''<ref name=初探>{{cite journal en|quotes = |author = 傅朗云|date =  1990|title =《乾隆<八旗满洲氏族通谱>初探》|journal = 古籍整理研究学刊|issue = 1990年第03期|pages = 14-18|publisher = 东北师范大学古籍整理研究所|location = 吉林省长春市|issn = 1009-1017|url = http://kns.cnki.net/kcms/detail/detail.aspx?dbcode=CJFD&filename=GJZL199003008&dbname=CJFD9093|language = Zh-hans}}</ref>，是清朝[[雍正|雍正]]末年至[[乾隆|乾隆]]初年由清朝官方编纂[[满洲八旗|满洲八旗]][[满族姓氏|氏族]][[谱牒|谱牒]]，由[[弘昼|弘昼]]、[[鄂尔泰|鄂尔泰]]、[[福敏|福敏]]、[[徐元梦|徐元梦]]等人奉[[乾隆帝|乾隆帝]]之命修撰。最早发端可追溯至清初，实际着手始撰于雍正十三年（1735年），于乾隆九年（1744年）刊印成书<ref name=初探/>。

《通谱》收录了除[[爱新觉罗氏|爱新觉罗氏]]外的1114个满洲姓氏，并记载各姓氏人物归附满洲统治者的时间、世居地、家系、人物事迹、封爵等相关情况<ref name=初探/>。每个姓氏均以建立功勋最显赫者为首，共有2240人立传，其建有业绩的子孙附载其后，无事迹可表者则列于所有传记人物之后，共记载乾隆以前的八旗人物约两万余人<ref name=初探/>。《通谱》以[[瓜尔佳氏|瓜尔佳氏]]、[[钮祜禄氏|钮祜禄氏]]、[[舒穆禄氏|舒穆禄氏]]、[[赫舍里氏|赫舍里氏]]、[[他塔喇氏|他塔喇氏]]、[[觉罗氏|觉罗氏]]、[[佟佳氏|佟佳氏]]、[[那拉氏|纳喇氏]]为“满洲八-{著}-姓”<ref>{{harvnb|弘昼等|2002|pp=31，100，115，146，167，181，247，280}}</ref>，被认为是当代[[满族八大姓|满族八大姓]]的原型<ref>{{cite journal en|quotes = |author = 李小雪|date =  |title =浅议“满族八大姓”|journal = 《剑南文学·文史研究》|issue = 2012年第08期|pages = |publisher = 《剑南文学》编辑部|location = |issn = |url = |language = Zh-hans}}</ref>。

此书至成书以来仅在乾隆朝刻印过一次，俗称“武英殿刻本<ref name=初探/>”，[[辽海出版社|辽沈书社]]曾影印出版该刻本<ref name=初探/>。

==瑕疵==
《通谱》偶有将归附之人的前代祖先称作“国初来归”的情况，比如马佳氏始祖马穆敦约为[[明朝|明朝]]中期之人，不可能在[[努尔哈赤|努尔哈赤]]起兵之后归附<ref>{{harvnb|刘庆华|2010|p=144}}</ref>；世居雅尔湖“国初来归”的瓜尔佳氏噶哈也大约为明朝中期人<ref>{{harvnb|刘庆华|2010|p=108}}</ref>。还有世系误载的例子，例如《清史稿》、《八旗通志初集》皆记载[[巴什泰|巴什泰]]死后爵位由其子珠拉岱承袭<ref>{{harvnb|赵尔巽等|1998|p=9200}}</ref>，后再由珠拉岱之子桑格袭爵<ref>{{harvnb|鄂尔泰等|1985|p=1594}}</ref>，而《通谱》则漏记一代，称桑格为巴什泰之子<ref>{{harvnb|弘昼等|2002|p=223}}</ref>。

《通谱》对于清代世家的记载较为全面，而对于社会地位较低的东北满洲旗人家系则常有漏载现象存在<ref>{{harvnb|刘庆华|2010|pp=30，119，249，253}}</ref>。
