'''八打灵再也博物馆''' ([[马来语|马来语]]: ''Muzium Petaling Jaya'')是[[马来西亚|马来西亚]][[雪兰莪州|雪兰莪州]][[八打灵再也|八打灵再也]]的一间[[博物馆|博物馆]]，也是州内首座地方文化博物馆。<ref>{{Cite news|url=http://www.sinchew.com.my/node/725678|title=增加藏物致力发展成新景点．灵市博物馆10万扩建|accessdate=2017-05-22|publisher=[[星洲日报|星洲日报]]}}</ref>馆内展示了八打灵再也的历史发展。
<ref name="malaysiatraveller">{{Cite web|url=http://www.malaysia-traveller.com/petaling-jaya-museum.html|title=Petaling Jaya Museum, Taman Jaya, PJ, Malaysia|accessdate=20 October 2016|publisher=Malaysia Traveller}}</ref><ref>{{Cite web|url=http://www.jmm.gov.my/ms/muzium-petaling-jaya|title=Muzium Petaling Jaya - Jabatan Muzium Malaysia|accessdate=20 October 2016|publisher=Jabatan Muzium Malaysia}}</ref>

== 历史 ==
博物馆于2007年11月开放。后来关闭进行装修，并于2016年4月重新开放。 

== 展览 ==
博物馆内展示了八打灵再也的历史发展，分为三个部分，分别是八打灵再也的起源，行政和发展。<ref>{{Cite web|url=http://www.timeout.com/kuala-lumpur/things-to-do/petaling-jaya-museum|title=Petaling Jaya Museum|accessdate=25 October 2016|date=20 March 2013|publisher=TimeOut}}</ref>

== 开放时间  ==
除了周五和公共假期，博物馆每天开放时间为上午9时至下午5时，入场免费。
