{{日本作品|八月的棒球甜心|八月のシンデレラナイン}}是由{{tsl|ja|アカツキ|Akatsuki}}所發行的手機遊戲，2017年6月27日開始營運<ref>{{Cite web|url=https://hachinai.com/news/post-1060/|title=『八月のシンデレラナイン』本日より配信開始！|work=遊戲「八月的棒球甜心」官方網站|date=2017-06-27|accessdate=2019-01-15|language=ja}}</ref>。2018年7月30日決定改編為電視動畫<ref>{{Cite web|url=http://dengekionline.com/elem/000/001/773/1773739/|title=『八月のシンデレラナイン』がアニメ化決定。コミケ94のグッズ情報が公開|work=電擊Online|date=2018-07-30|accessdate=2019-01-15|language=ja}}</ref>，同年12月29日確定於2019年4月播出。

== 故事簡介 ==
男主角在中學時原本是棒球隊的王牌，但因為受傷不得不放棄夢想，並且離開家鄉到祖母住的地方就讀高中。在學校裡，他和對棒球充滿熱情的女孩子有原翼相遇了，看到翼對棒球的熱情後，讓男主角重新燃起對棒球的熱情，並決定擔任翼所組成的女子棒球隊的監督，一起朝甲子園這個目標邁進。 (動畫版沒有男主角，以女性角色為主。)

== 登場角色 ==
; 男主角
: 小說版定名為「矢上浩太」。中學時原本是棒球隊的王牌，但因為右肩的傷勢不得不放棄棒球，後來在翼的請託下擔任女子棒球隊的監督。 (動畫版沒有男主角。)
