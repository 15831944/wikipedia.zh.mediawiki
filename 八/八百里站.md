'''八百里站'''是[[杭临城际铁路|杭临城际铁路]]的一个高架车站，位于[[杭州市|杭州市]][[临安区|临安区]][[锦北街道|锦北街道]]科技大道陈市线口，沿科技大道东西向布置。该站站名来源自其所处的八百里自然村<ref name="name">{{cite web |title=杭州市人民政府关于杭州至临安城际铁路工程车站站名的批复 _ |url=http://www.hangzhou.gov.cn/art/2019/3/28/art_1256295_31738047.html |website=中国杭州}}</ref>，意为八百岁老人的故里，因传说中[[彭祖|彭祖]]曾在此地居住而得名<ref>{{cite web |title=临安“八百里”美山水 |url=http://news.cri.cn/gb/3601/2005/03/01/1266@465060.htm |website=国际在线}}</ref>。

==结构== 
八百里站为高架三层[[岛式站台|岛式站台]]，位于科技大道中央绿化带内，东侧设有停车线<ref>{{Cite web|url =http://www.crfsdi.com.cn/sysWeb/upfiles/hpgs/20151112111501.pdf|title =杭州至临安城际铁路工程环境影响报告书|accessdate =|author =|date =|publisher =|deadurl =yes|archiveurl =https://web.archive.org/web/20160303035612/http://www.crfsdi.com.cn/sysWeb/upfiles/hpgs/20151112111501.pdf|archivedate =2016年3月3日|df =}}</ref>，建筑面积约13650平方米<ref name="henko">{{cite web |title=省发展改革委关于杭州至临安城际铁路工程初步设计变更的批复 |url=http://www.zjdpc.gov.cn/art/2017/9/14/art_808_1733386.html |website=浙江省发改委}}</ref>。

== 历史 ==
*2017年9月14日，车站由高架二层车站变更为高架三层站，并由科技大道南侧移至科技大道路中绿化带设置。此外，原设于[[青山湖站|高新园区站]]的停车线移至该站东端高架设置。<ref name="henko"/>
*2019年3月25日，车站定名为八百里站。<ref name="name"/>
