'''八路军重庆办事处旧址'''，位于[[重庆市|重庆市]][[渝中区|渝中区]]红岩村13号，一般以'''红岩村'''指代，是[[中共中央南方局|中共中央南方局]]和[[八路军|八路军]]重庆办事处所在地。现隶属[[红岩革命纪念馆|红岩革命纪念馆]]。

==简介==
[[抗日战争|抗日战争]]全面爆发后，1938年10月，[[侵华日军|侵华日军]]占领[[广州|广州]]、[[武汉|武汉]]，[[国民政府|国民政府]]迁都[[重庆|重庆]]，八路军在重庆设立八路军重庆办事处。1939年1月，中共中央南方局在重庆成立，[[周恩来|周恩来]]任书记，[[董必武|董必武]]、[[叶剑英|叶剑英]]、[[秦邦宪|秦邦宪]]、[[凯丰|凯丰]]、[[吴克坚|吴克坚]]等任常委，全面领导中国共产党在[[国统区|国统区]]的工作。因为[[国民政府|国民政府]]不允许中共党组织公开活动，所以中共中央南方局是秘密的，设在公开机关八路军重庆办事处内，最初中共中央南方局和八路军重庆办事处的办公地点在机房街70号<ref name=cqwhw>{{Cite web|url =http://www.cqwhw.gov.cn/Html/1/tpsp/zdww/2013-03-12/10448.html|title =八路军重庆办事处旧址|author =|date =2013-03-12|publisher =重庆市文化委员会|language =|accessdate =|quote =}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

1939年5月3日至4日，日军飞机轰炸重庆，机房街70号被炸毁。董必武、博古等人率南方局和办事处大部分人员迁往红岩嘴，散住在大有农场工人宿舍及堆放柴草、杂物的数处茅草房内。1939年5月中旬，在征得红岩嘴大有农场主人[[饶国模|饶国模]]女士同意后，将原大有农场一幢三层楼重建为南方局、办事处的办公住宿大楼，由办事处人员自行设计并修建，1939年秋完工，南方局、办事处全部迁此办公。自此，周恩来、董必武、叶剑英、[[吴玉章|吴玉章]]、[[王若飞|王若飞]]等领导人一直常驻在此。1945年，[[毛泽东|毛泽东]]自[[延安|延安]]来到重庆进行[[重庆谈判|重庆谈判]]时，也住在这里<ref name=wwj/><ref name=cqwhw/>。这里是抗日战争时期重庆重要的政治活动中心之一<ref>{{Cite web |url =  http://dangshi.people.com.cn/GB/151935/206880/206934/13163667.html|title = 红岩村13号：八路军驻重庆办事处 |author =  |date = 2010-11-09 |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。1947年3月，[[中国国民党|中国国民党]]发动[[第二次国共内战|全面内战]]后，南方局、办事处的工作人员全部返回延安。这里的地名当时是红岩嘴，因其地质成分主要是[[侏罗纪|侏罗纪]]红色页岩而得名。地方政府将这里的门牌号编为红岩嘴13号，1945年改为红岩村13号<ref name=cqwhw/>。

1956年8月16日，“八路军办事处旧址”公佈為四川省第一批歷史及革命文物保護單位<ref>四川省人民委員會關於在工農業生產建設中注意保護文物的通知（〔56〕省二文字第0618號），1956年8月16日</ref>。1980年7月7日，“中共中央南方局及八路軍重慶辦事處舊址-紅岩村13號”重新公佈為四川省第一批文物保護單位<ref>關於重新公布全省文物保護單位名單的通知（川府發（1980）154號），四川省人民政府，1980年7月7日</ref>。1958年，以红岩村13号八路军重庆办事处旧址大楼为主体的[[红岩革命纪念馆|红岩革命纪念馆]]对外开放。1961年3月4日，“八路军重庆办事处旧址”公布為[[第一批全國重點文物保護單位|第一批全國重點文物保護單位]]，该项目包括红岩村13号、曾家岩50号[[周公馆|周公馆]]，后又增加中山三路305号（原263号）[[中共代表团驻地旧址|中共代表团驻地旧址]]、民生路240号（原208号）[[新华日报营业部旧址|《新华日报》营业部旧址]]<ref>國務院關於公布第一批全國重點文物保護單位的通知（國文習字40號），1961年3月4日</ref>。

==建筑==
八路军重庆办事处办公大楼坐落在“大有农场”西北坡上。坐北朝南，是一栋二楼一底的深灰色砖木结构建筑，外看二层、实为三层，占地面积800平方米，进深15.5米，楼宽32.4米，高17米，共有房间54间<ref name=wwj>{{Cite web |url = http://www.cqww.gov.cn/contents/59/1375.html  |title = 八路军重庆办事处旧址 |author =  |date = 2015-04-28 |publisher =重庆市文物局  |language =  |accessdate =  |quote =  }}</ref><ref name=cqwhw/>。

*底层：是公开机关八路军重庆办事处的办公室（[[皖南事变|皖南事变]]前，[[新四军|新四军]]重庆办事处也在此办公）<ref name=cqwhw/>。
*二层：是南方局机关和领导人的办公室兼卧室。周恩来的办公室兼卧室是南方局负责人开会的地方。1945年毛泽东重庆谈判时，在这里居住41天。二层最大的一个房间是南方局和办事处的图书室<ref name=cqwhw/>。
*三层：是南方局、办事处干部的工作间及宿舍。在此处办公的机要科设有秘密电台，主要负责与延安及各地电台的联系及机要文件的传送<ref name=cqwhw/>。
