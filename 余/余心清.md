'''余心清'''（{{bd|1898年||1966年|9月4日|catIdx=}}），[[安徽|安徽]][[合肥|合肥]]人，[[中华人民共和国|中华人民共和国]]政治人物。

==生平==
余心清毕业于[[金陵神学院|金陵神学院]]、[[哥伦比亚大学|哥伦比亚大学]]，为[[冯玉祥|冯玉祥]]部下。回国后，担任[[开封|开封]]训政学院院长、[[察哈尔抗日同盟军|察哈尔抗日同盟军]]总务处长兼[[察哈尔省政府|察哈尔省政府]]民政厅长。1933年，参加[[闽变|闽变]]，并任[[中华共和国|中华共和国]]人民革命政府经济委员会代理主席。[[抗日战争|抗日战争]]时期，历任[[国民革命军|国民革命军]]第三集团军政训处处长等职。后被逮捕，经[[中国共产党|中国共产党]]营救获释，此后出席[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]<ref>{{cite web| url=http://www.minge.gov.cn/txt/2008-09/27/content_2496653.htm| title=余心清| publisher=中国国民党革命委员会中央委员会| date=2008-09-27| deadurl=yes| archiveurl=https://web.archive.org/web/20160305091349/http://minge.gov.cn/txt/2008-09/27/content_2496653.htm| archivedate=2016年3月5日| df=}}</ref>。历任[[中央人民政府办公厅|中央人民政府办公厅]]副主任、典礼局局长，[[国家机关事务管理局|政务院机关事务管理局]]局长、[[中华人民共和国民族事务委员会|中华人民共和国民族事务委员会]]副主任，民革中央常委等职位。[[文化大革命|文化大革命]]开始后被揪斗，于1966年9月4日在北京家中自杀身亡，终年68岁。1978年10月，在[[八宝山革命公墓|八宝山革命公墓]]举行了余心清同志骨灰安放仪式<ref>{{cite web| url=http://www.ah.xinhua.org/jg60nahp/2009-07/07/content_17029891.htm| title=余心清| publisher=安徽省志| date=2009-07-07| deadurl=yes| archiveurl=https://web.archive.org/web/20110903145246/http://www.ah.xinhua.org/jg60nahp/2009-07/07/content_17029891.htm| archivedate=2011-09-03}}</ref><ref name=ifeng>{{Cite web |url = http://news.ifeng.com/history/vp/detail_2012_07/15/16035193_13.shtml |title = 因文革而自杀的党政军领导干部 |author =  |date = 2012-07-15 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

==家庭==
*女婿：[[冯洪达|冯洪达]]，[[冯玉祥|冯玉祥]]之子，[[中国人民解放军少将|中国人民解放军少将]]，曾任[[北海舰队|北海舰队]]副司令员<ref name=ifeng/>。
