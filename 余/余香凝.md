'''余香凝'''（{{lang-en|Jennifer Yu}}，{{bd|1993年|7月5日||}}），[[香港|香港]]女藝人，2011年開始當模特兒，屬{{link-yue|Jam Cast|Jamcast}}旗下
<ref>[https://jamcastblog.com/jennifer-yu-%E4%BD%99%E9%A6%99%E5%87%9D/ Jennifer Yu 余香凝 - Jam Cast Management (HK) Ltd ] {{Wayback|url=https://jamcastblog.com/jennifer-yu-%E4%BD%99%E9%A6%99%E5%87%9D/ |date=20160404110100 }}</ref>，並拍過不少廣告，於2016年開始參演電影，更憑出道作品《[[骨妹|骨妹]]》入圍[[第36屆香港電影金像獎|第36屆香港電影金像獎]][[香港電影金像獎最佳新演員|最佳新演員]]提名。2017年她加入樂壇，推出第一支派台歌《你愛的人上》。

余香凝早年就讀[[保良局朱正賢小學|保良局朱正賢小學]]下午校和[[聖母無玷聖心書院|聖母無玷聖心書院]]。她曾經與男演員[[陳家樂_(藝人)|陳家樂]]交往3年，獲男方認定是結婚對象<ref>{{cite web|url=https://www.hk01.com/article/182230/|title=陳家樂認定余香凝真愛　未求婚已送定鑽戒：會照顧佢一世|work=香港01|date=2018-04-26}}</ref>。2017年3月余駕車時在[[吐露港公路|吐露港公路]]被另一輛車撞，傷及左邊臉並導致鼻骨爆裂<ref>{{Cite web |url=http://hk.on.cc/hk/bkn/cnt/entertainment/20170314/bkn-20170314143243202-0314_00862_001.html |title=影壇新貴余香凝遇車禍　面傷鼻骨裂破相 |date=2017-03-14 |publisher=東網}}</ref><ref>{{Cite web |url=http://www.metrodaily.hk/metro_news/%E4%BD%99%E9%A6%99%E5%87%9D%E9%81%87%E8%BB%8A%E7%A6%8D%E6%81%90%E7%A0%B4%E7%9B%B8-%E9%99%B3%E5%AE%B6%E6%A8%82%EF%BC%9A%E6%9C%83%E7%85%A7%E9%A1%A7%E4%BD%A2%E4%B8%80%E4%B8%96/ |title=余香凝遇車禍恐破相 陳家樂：會照顧佢一世 |date=2017-03-15 |publisher=都市日報}}</ref><ref>{{cite web|url=https://www.hk01.com/article/77782/ |title=《骨妹》余香凝撞車險破相　自嘲十足戲中角色：差點似靈靈咁死 |date=2017-03-14 |publisher=香港01}}</ref>。至2018年年中二人分手<ref>{{Cite web |url=https://hk.entertainment.appledaily.com/entertainment/daily/article/20180615/20420763 |title=陳家樂分手痛哭 余香凝認保險才俊家留宿 |date=2018-06-15 |publisher=蘋果日報娛樂}}</ref><ref>{{Cite web |url=https://www.hk01.com/article/199310 |title=余香凝分手 娛樂圈金童玉女宿命　持續愛到幾多歲又有幾多對？  |date=2018-06-15 |publisher=香港01}}</ref><ref>{{Cite web |url=https://www.hk01.com/article/199346 |title=余香凝稱分手不涉及第三者：嗰位先生係多年好友　最近一齊開餐廳 |date=2018-06-14 |publisher=香港01}}</ref>。她的妹妹是模特兒[[余潔瀅|余潔瀅]]（Zoe Yu），比妹妹大3歲<ref name=cosmo0528>{{cite web|url=https://www.cosmopolitan.com.hk/lifestyle/Zoe-Yu-cooking-programme|title=【美女廚房】美女學徒余潔瀅率先跑出，同家姐余香凝有撞面嗎？|publisher=[[Cosmopolitan|Cosmopolitan]]|date=2018-05-28|accessdate=2018-11-18|language=zh-hant}}</ref>。

她曾於訪問中表示剛入行時身邊的人比較現實，連帶自己也變得現實，只希望能賺錢，於是終日不停想着錢的問題，忽略其他事情<ref>[https://www.mings-fashion.com/余香凝-jennifer-yu-sportmax-50344/ 余香凝：時裝災難 #是福不是禍]</ref>。

2019年，余香凝分別憑電影《[[逆流大叔|逆流大叔]]》以及《[[非同凡響|非同凡響]]》入圍[[第38屆香港電影金像獎|第38屆香港電影金像獎]]「最佳女配角」以及「最佳女主角」，演技備受肯定。
