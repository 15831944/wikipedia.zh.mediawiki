'''余克礼'''（{{bd|1952年|||}}），生於[[湖北|湖北]][[浠水|浠水]]，[[中华人民共和国|中华人民共和国]][[台湾问题|台湾问题]]专家<ref name=zhangxm>章晓明，余克礼：研究两岸关系的高层智囊，决策与信息2005年第05期</ref>

==生平==
余克礼早年在[[武汉大学|武汉大学]]历史系学习。1982年从武汉大学历史系毕业后，进入[[中国社会科学院台湾研究所|中国社会科学院台湾研究所]]从事学术研究工作，主要研究领域是台湾政治、两岸关系等。历任中国社会科学院台湾研究所研究员、副所长、所长。曾任《台湾研究》副主编。兼任[[中国和平统一促进会|中国和平统一促进会]]理事、[[中国亚洲太平洋学会|中国亚洲太平洋学会]]理事、[[全国台湾研究会|全国台湾研究会]]理事、[[北京市政协|北京市政协]]台港澳侨联委员会委员<ref name=zhangxm/>。

余克礼曾先后参加多份对台文件的起草工作。1993年，余克礼参与起草了中国政府发表的首份台湾问题白皮书《台湾问题与中国的统一》。2000年2月21日，[[国务院台湾事务办公室|国务院台湾事务办公室]]、[[国务院新闻办公室|国务院新闻办公室]]发表了余克礼参与起草的《一个中国的原则与台湾问题》白皮书<ref name=zhangxm/>。

1983年5月和1995年11月，余克礼先后两次到[[台湾|台湾]]考察访问。1999年10月，台湾的[[國立政治大學國際關係研究中心|国立政治大学国关中心]]向余克礼发出邀请，并决定了翌年访台行程。因余克礼是《一个中国的原则与台湾问题》白皮书的主要起草者，且被视为大陆方面中央政府对台工作的重要智囊，再加上此行正值台湾举行[[2000年中华民国总统选举|2000年中华民国总统选举]]前后，因此这次学术交流受到台湾媒体关注。2000年4月19日，余克礼一行6人应政治大学国关中心邀请抵达台湾。到台湾后，余克礼接受台湾媒体采访，阐述了[[一个中国原则|“一个中国”原则]]及对两岸关系发展前景的看法。谈话次日，台湾3家晚报头版头条刊登了余克礼的谈话内容，被台湾支持“[[台独|台独]]”的媒体视为态度强硬。此后台湾[[立法院|立法院]]、政治人物、媒体开始围攻余克礼。[[中华民国副总统|中华民国副总统]][[吕秀莲|吕秀莲]]透过台湾媒体指责余克礼有关台湾新领导人[[陈水扁|陈水扁]]应当认同自己是“中国人”的讲话是“放肆”，吕秀莲的讲话使余克礼此次学术交流之行迅速被政治化，一时成为媒体追踪采访的热点。但政治大学国关中心方面未受外界干扰，继续进行学术交流，国关中心主任[[何思因|何思因]]、副主任[[吴钊燮|吴钊燮]]及多位研究员与余克礼一行闭门座谈，余克礼一行还与[[亲民党|亲民党]]人士见面，并拜会了台湾和统会会长[[梁肃戎|梁肃戎]]。余克礼此行未有台湾民众举行抗议活动<ref name=zhangxm/>。

这次“余克礼风波”过后半年多，2000年12月初，台湾[[中華民國國家安全局|国安局]]副局长在[[民主进步党|民进党]]团会议上称，年内曾来台湾进行学术交流的中国社会科学院台湾研究所副所长余克礼“不仅是大陆对台政策的高层智囊，实际身份还是‘[[中华人民共和国国家安全部|国安部]]’第15局副局长。”这引发台湾舆论哗然。为了给自己的胡乱猜测找依据，台方通过各种渠道试图弄清余克礼的真实身份，但最终确认的信息是“国安部”与余克礼无任何关系。在此之前，台湾国安局还曾指中国社会科学院台湾研究所所长[[姜殿铭|姜殿铭]]是“[[中华人民共和国国家安全部|国安部]]”第15局局长，而同样未能拿出任何证据。余克礼表示，台湾有关方面对大陆对台研究学者的这种指控，是抱着[[冷战|冷战]]僵化思维，将两岸正常的学术交流“政治化”<ref name=zhangxm/>。

2012年，在[[张亚中|张亚中]]主导下，[[兩岸統合學會|兩岸統合學會]]申請在[[国立台湾大学|国立台湾大学]]社会科学院舉辦“[[台北會談|台北會談]]”，动员了两岸所有重要智库及学术机关的上百名学者参加，其中邀請了大陆方面[[海峡两岸关系协会|海协]]副会长[[王在希|王在希]]、海协联系部主任于京与余克礼等50多人來台参加；台湾方面，[[马英九政府|马英九政府]]认为此行意在催促两岸政治對話，[[陆委会|陆委会]]乃拒绝王在希、于京等9位具备大陆官方身份的受邀人士入台参会，仅批准了余克礼等不具备大陆官方身份的“纯学者”参会，并规定会议主题不可出现“认同”和“互信”等字。张亚中乃停办台北会谈<ref>{{Cite web |url = http://www.taihainet.com/news/twnews/bilateral/2012-06-28/870774.html |title =  台当局拒绝大陆官员赴台参加学术研讨|author =  |date = 2012-06-28 |publisher = 台海网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.herald-today.com/content.php?sn=4852 |title = 馬英九冷卻兩岸交流意圖 令人不解  |author =  |date =2012-06-25  |publisher =今日導報  |language =  |accessdate =  |quote =  }}</ref>。

余克礼与两岸研究台湾问题的学术界人士有广泛交流，也和台湾蓝绿阵营人士均有广泛接触。2013年，余克礼屆齡退休，卸任中国社会科学院台湾研究所所长职务，仍任研究员<ref>{{cite news|author1=藍孝威|title=余克禮交棒 藍綠通吃|url=http://www.chinatimes.com/newspapers/20131210000439-260108|accessdate=2016-07-28|agency=中國時報|date=2013-12-10}}</ref><ref>{{Cite web |url =  http://news.ifeng.com/taiwan/3/detail_2013_11/07/31033848_0.shtml|title = 王金平宴请余克礼：服贸协议不通过不是我的错 |author =  |date =2013-11-07  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
