'''余世存'''{{BD|1969年|2月|||catIdx=Y余}}出生于[[湖北省|湖北省]][[随州市|随州市]]，毕业于[[北京大学|北京大学]]，为知名作家、编辑、自由撰稿人、[[公共知识分子|公共知识分子]]。<ref name="fish " >{{cite web |url=http://news.163.com/10/0418/00/64GUR08300014AED.html |title=再出江湖，余世存独家讲述五年心路，自称真男，以人格济世 |accessdate=2010-04-18 |publisher=网易 |archiveurl=http://news.163.com/10/0418/00/64GUR08300014AED.html |archivedate=2012年11月7日 |work= |language =zh-cn}}</ref>

== 生平 ==
1969年，余世存出生在[[湖北省|湖北省]][[随州市|随州市]]，1990年，余世存从[[北京大学|北京大学]]毕业，大学毕业后，他曾当过教师、编辑、官员。<ref name="fish " />

== 风格 ==
余世存受到[[李慎之|李慎之]]影响很深，主要为[[知识分子|知识分子]]奔走呼吁，2005年，他出版了《非常道：1840年到1999年的中国话语》，通过讲述[[曾国藩|曾国藩]]、[[左宗棠|左宗棠]]、[[李鸿章|李鸿章]]、[[孙中山|孙中山]]、[[袁世凯|袁世凯]]、[[胡适|胡适]]、[[陈独秀|陈独秀]]、[[钱锺书|钱锺书]]、[[李敖|李敖]]、[[王小波|王小波]]等人的奇闻异事剖析[[中国文化|中国文化]]，2007年，他出版了《常言道：近代以来最重要的话语录》，讲述[[理性|理性]]、[[良知|良知]]、[[自由|自由]]和[[尊严|尊严]]。<ref>{{cite book |language =zh-cn|author=余世存 |title=《常言道：近代以来最重要的话语录》|year= 2007|publisher=新世界出版社 |location= 北京市|pages=前言 |isbn=9787802283169 }}</ref>
2008年成为《[[零八宪章|零八宪章]]》第一批签署者。
