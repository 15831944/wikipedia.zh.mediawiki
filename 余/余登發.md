'''余登發'''{{BD|1904年|9月21日|1989年|9月13日|catIdx=Yu余}}，[[臺灣|臺灣]][[高雄|高雄]][[橋頭|橋頭]]人，曾任[[中華民國|中華民國]]第一屆[[國大代表|國大代表]]、[[高雄縣長|高雄縣長]]、[[中國統一聯盟|中國統一聯盟]]首任名譽主席，1979年曾遭「通匪」罪名逮捕。臺南商業專門學校畢業本科畢業，是[[臺灣戒嚴時期|臺灣戒嚴時期]][[統派|統派]][[政治人物|政治人物]]，因有功於台灣民主運動，被喻為台灣民主先驅、「台灣野生的政治家」<ref>[http://www.eslite.com/product.aspx?pgid=1001110931410045 台灣野生的政治家]</ref>、「赤腳政治家」<ref>陳佳宏, [http://ndltd.ncl.edu.tw/cgi-bin/gs32/gsweb.cgi/login?o=dnclcdr&s=id=%22099NTNU5832008%22.%20&searchmode=basic 余登發與戰後臺灣民主運動], [[國立臺灣師範大學|國立臺灣師範大學]]台灣史研究所碩士論文, 2010</ref>。1989年兒媳競選縣長時，死於自宅，致死原因存有爭議。

==早年生平==

1914年（[[大正|大正]]3年），余登發進入[[楠梓坑公學校|楠梓坑公學校]]就讀。1920年（大正9年），進入臺南商業專門學校本科，並從該校畢業。畢業後，最初到岡山郡役所工商課工作，後來轉到高雄州廳勸業課。

[[昭和|昭和]]十年（1935年），[[日本|日本]]殖民當局舉行[[第一屆市會及街庄協議會員選舉|州庄選舉]]中，余登發當選民選高雄州[[楠梓庄|楠梓庄]]協議會會員。余登發自言隨著[[日中戰爭|日中戰爭]]爆發，民生物資缺乏開始配給。而當時官派的庄長齋藤<ref>據臺灣總督府職員錄查詢，1935年後出任楠梓庄長者有齋藤勝次。見[http://who.ith.sinica.edu.tw/s2g.action?viewer.q_authStr=1&viewer.q_fieldStr=allIndex&viewer.q_opStr=%20&viewer.q_valueStr=%E6%A5%A0%E6%A2%93%E5%BA%84 臺灣總督府職員錄查詢]</ref>卻辦理配給不公，因此引起余登發發起罷免庄長的行動，並獲得六位民選議員全部支持以及一位官選議員田中加入過半支持，因而轟動一時。使齋藤不得不改善他的行為，然而日後齋藤也對余登發挾怨報復，將其徵召為勞務工，送至[[高雄州|高雄州]]桃仔園港灣建教工程做工六十天。{{r|獄|page=1}}

== 戰後 ==
1945年[[二次大戰|二次大戰]][[日本投降|日本投降]]後，台灣被[[國民政府|國民政府]]接管，余登發於1945年當選臺灣省高雄縣岡山鎮橋頭里里長，自費設置電線桿數十隻。{{r|獄|page=3}}由於橋頭鄉與附近十個里當時隸屬於岡山鎮，為了尋求都市化起見，余登發遂與地方人士向省政府申請另成立一鄉。{{r|獄|page=3}}1947年6月1日，新設[[橋頭鄉|橋頭鄉]]成立，余登發獲鄉民推派出任第一任橋頭鄉鄉長。{{r|獄|page=3}}

於同年十二月的國民大會代表選舉中，余登發獲岡山地區人士支持當選[[第一屆國民大會臺灣省代表|第一屆國民大會臺灣省代表]]，在岡山區長[[黃應樞|黃應樞]]勸誘下加入[[國民黨|國民黨]]。{{r|獄|page=4}}並到[[南京|南京]]參與[[國民大會|國民大會]]，在南京見到國民大會大會堂前遭抬[[棺材|棺材]]抗議選舉舞弊，以及聽聞[[李宗仁|李宗仁]]從安徽省主席處以卡車載鈔票到南京活動才當選[[中華民國副總統|副總統]]。使余登發對於國民黨內的分裂以及選舉違法舞弊有了第一手觀察。{{r|獄|page=5}}

1949年5月，余登發於橋頭鄉長屆滿前，競選高雄農田水利會，擊敗官派高雄縣長毛振環等人當選。{{r|獄|page=7}}在主任服務期間致力於地方水利建設並且捐獻私財建設地方。並且制止了原來的總幹事潘葛利用職權賺錢，以及招待時任省水利局長章錫授花天酒地。此舉引發潘葛不滿，遂以自己身為屏東縣民身分於今[[屏東縣|屏東縣]]另外設立屏東農田水利會，自己做會長。然而依照當時水利法，水利會分會必須獲得過半數會員同意使得成立，當時高雄農田水利會會員十萬人，過半數五萬人同意。而潘葛利用與章錫授的關係只得到十餘名會員簽名同意就擅自申請。並得到省主席吳國楨批准，對此余登發於報紙批評[[吳國楨|吳國楨]]濫權，甚至在省政府內與吳國楨互相拍桌。{{r|獄|page=12}}

1951年4月，余登發辦理登記參選[[1950年－1951年中華民國縣市長選舉#高雄縣|首屆高雄縣長選舉]]，一度遭到省府下令撤職水利會長，並送法院查辦。被余登發視為吳國楨為報拍桌之仇之舉。{{r|獄|page=13}}最後余氏獲判無罪，在選舉方面則並未當選。仍保留國大代表的身份。

== 高雄縣長任內 ==
1954年[[1954年中華民國縣市長選舉#高雄縣|第二屆高雄縣長選舉]]，余登發因支持白派的[[陳新安|陳新安]]。最終並未參選。

1960年[[1960年中華民國縣市長選舉#高雄縣|縣長選舉]]，余登發獲得陳新安支持，當選[[高雄縣長|高雄縣長]]。任內開源節流，刪除國民黨高雄縣黨部之補助款以建設學校教室，在在獲得好評。

1963年，[[台灣省政府|台灣省政府]]以「八卦寮地目變更案」、「萬金松丁壩工程案」為由將其強迫停職，並被以[[瀆職罪|瀆職罪]]判處[[有期徒刑|有期徒刑]]兩年，於1973年入獄。此時，余登發的國大代表身份也被政府取消。 稍後，1974年余登發獲准[[保外就醫|保外就醫]]。

== 涉入吳泰安案 ==
余登發在出獄之後，與不少[[黨外|黨外]]人士有所往來；1979年1月21日，余登發與其子[[余瑞言|余瑞言]]因為「涉嫌參與匪諜[[吳泰安|吳泰安]]叛亂」罪名遭逮捕（[[星雲法師|星雲法師]]表示吳泰安本人為精神病患，幻想要推翻政府、革命，還寫許多聘書，造成[[台東海山寺|台東海山寺]]修和法師、余登發冤獄<ref name="林清玄">{{cite book |author=林清玄|chapter=〈改變了世界的藍圖〉|title=《浩瀚星雲》|date=2001-11-01|publisher=圓神出版社|volume=|location=臺灣|language=zh-tw|isbn=9789576076879}}</ref>）引發社會反彈，[[許信良|許信良]]、[[施明德|施明德]]、[[張俊宏|張俊宏]]、[[林義雄|林義雄]]、[[黄順興|黄順興]]、[[邱連輝|邱連輝]]、[[黃信介|黃信介]]、[[陳菊|陳菊]]、[[陳婉真|陳婉真]]、[[胡萬振|胡萬振]]、[[何春木|何春木]]、[[曾心儀|曾心儀]]等黨外人士，前往余登發故鄉[[高雄縣|高雄縣]][[橋頭區|橋頭鄉]]進行示威抗議聲援余登發父子，遊行總策劃人為施明德 ，[[立法委員|立法委員]][[康寧祥|康寧祥]]等人也表示對余的起訴與判決不當，是為「[[橋頭事件|橋頭事件]]」。

1979年4月，余登發因「知匪不報」與「為匪宣傳」等罪，被法院判處有期徒刑8年、褫奪公權5年。1980年，余登發仍再度獲准[[保外就醫|保外就醫]]。晚年余登發居住於[[仁武區|仁武鄉]][[八卦寮|八卦寮]]自宅。1987年，[[許信良|許信良]]返國時，余登發曾經前往[[台灣桃園國際機場|台灣桃園國際機場]]支持。

1989年9月13日，余登發被發現死於自宅臥室血泊中，法醫[[楊日松|楊日松]]和美國知名法醫[[賽瑞爾·韋契特|賽瑞爾·韋契特]]認為是他殺，其他7位法醫認為是意外，後來用投票表決通過是意外，推翻他殺之說，但真實原因仍爭議不決，如今仍是一樁懸宕已久的命案。當時他的兒媳婦[[余陳月瑛|余陳月瑛]]正準備連任第11屆高雄縣長。

==政治理念==
余登發在國族認同上一直認為自己是中國人，並曾在戒嚴時代時主張兩岸和平統一，遭當時的國民黨以「通匪」名義逮捕過。余登發曾在民進黨的場合上喊出中國統一的口號，1988年也曾與[[胡秋原|胡秋原]]共同擔任[[中國統一聯盟|中國統一聯盟]]首任名譽主席。他跟現在的民進黨在兩岸政策主張和國族認同上南轅北轍，但在[[民主|民主]]的追求上一致。

余登發開創了高雄縣的政治家族「[[余家班|余家班]]」，他與女婿[[黃友仁|黃友仁]]、兒媳婦[[余陳月瑛|余陳月瑛]]、長孫[[余政憲|余政憲]]合計擔任過6次[[高雄縣縣長|高雄縣縣長]]；余陳月瑛、余政憲及女兒[[黃余秀鸞|黃余秀鸞]]、孫[[余政道|余政道]]、孫女[[余玲雅|余玲雅]]、孫媳[[鄭貴蓮|鄭貴蓮]]合計擔任過11次立法委員；余陳月瑛、余政道、余玲雅合計擔任過8次[[省議員|省議員]]；另外余政憲更出任過[[中華民國內政部|內政部]]長。其他受過余登發栽培的地方政治人物更不在少數；余家班不但是高雄縣重要的[[地方派系|地方派系]]，也是[[民主進步黨|民主進步黨]]黨內重要的地方派系。
