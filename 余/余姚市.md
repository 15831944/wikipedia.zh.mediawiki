'''余姚市'''为[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]]下辖的一个[[县级市|县级市]]，于余姚秦时建县<ref>{{Cite book|title=一说汉建|last=|first=|publisher=|year=|isbn=|location=|pages=}}</ref>，东汉建安五年（公元200年）始筑县城，为浙东古县城之一。余姚市经济发达，为全国最具竞争力的[[县级行政区|县市]]之一。该市位于宁波市域西北部，中心地理坐标为东经121°09′、北纬30°03′；全市总面积1500.80平方千米，户籍人口104.6万人（2011年）。政府驻[[兰江街道|兰江街道]]北兰江路1号，邮编：315400。行政区划代码：330281。区号：0574。境内现存古代文物遗存与名胜一千余处，有“文献名邦”之称。<ref>[http://www.yy.gov.cn/col/col20899/index.html “文献名邦”的来历]</ref>

== 历史 ==
余姚境内曾发掘出7000年历史的[[河姆渡遗址|河姆渡遗址]]。2004年在三七市又发现同属于[[河姆渡文化|河姆渡文化]]的[[田螺山遗址|田螺山遗址]]。

余姚得名，说法不一。一说，《[[风土记|风土记]]》载，[[舜|舜]]文庶所封之地，舜姓姚，故曰余姚（“舜后支庶所封之地，舜姓姚，故-{云}-余姚”）。另一说，《[[越绝书|越绝书]]》載，[[夏|夏]][[少康|少康]]封少子無餘于會稽，以奉禹祠，姚乃其届邑，故曰余姚。《[[山海经|山海经]]》[[郭璞|郭璞]]注，[[四明山|句餘山]]在[[會稽郡|會稽]]余姚县南，句章县北，故此二县因此为名-{云}-，此为第三种说法。还有一说，“邑有句餘山，又南有[[姚江|姚江]]”，故名。再有一说，明代《郡县释名》说，县隶會稽，在余姚山之西，以河东有姚姓，故曰余姚。近有人主张，余姚为[[越语|越语]]地名，其义石明。

[[秦朝|秦]]时置余姚县（一说[[汉朝|汉]]建），属[[會稽郡|會稽郡]]。[[东汉|东汉]][[建安|建安]]五年（200年）始筑县城，为浙东古县城之一。据《[[太平寰宇记|太平寰宇记]]》所载：唐初“余姚之境东包[[明州|明州]]，西辖[[上虞|上虞]]，为[[越州|越州]]巨镇”，一度升为[[姚州_(浙江)|姚州]]（唐[[武德|武德]]四年（621年）置，武德七年废）。[[宋朝|宋]]为“[[望县|望县]]”，“东南最名邑”。[[元朝|元]][[元贞|元贞]]元年（1295年）改为[[余姚州|余姚州]]。[[明朝|明]][[洪武|洪武]]二年（1369年）废州复县。

1949年5月23日，中共接管政权。中华人民共和国建立后，境域多次变动，最大一次是1954年，大古塘以北棉区划归[[慈谿县|慈谿县]]，慈谿县西部稻区、山区划归余姚县。境内明代所建筑的[[三山所城|三山所城]]则成为新的慈谿县[[县城|县城]]所在。

1985年7月26日撤县设市，隶属宁波市，设9区、8个镇、63个乡。

== 物产 ==
土特产品有[[茶叶|茶叶]]、[[杨梅|杨梅]]、[[干菜笋|干菜笋]]、佛雕、工艺草编。野生动植物资源有[[穿山甲|穿山甲]]、[[水獭|水獭]]、[[丹顶鹤|丹顶鹤]]、[[白鹤|白鹤]]、[[黑鹳|黑鹳]]、[[鸳鸯|鸳鸯]]、[[灰鹤|灰鹤]]、[[银杏|银杏]]等。中草药材有[[络石屯|络石屯]]、[[贯众|贯众]]、[[淡竹叶|淡竹叶]]、[[茯苓|茯苓]]、[[丹参|丹参]]、[[金银花|金银花]]等40餘种。金属矿产资源有[[铜|铜]]、[[铁|铁]]、[[铅|铅]]、[[锌|锌]]等。非金属矿产资源有[[萤石|萤石]]、[[高岭土|高岭土]]、[[粘土|粘土]]、[[石英|石英]]、[[磷|磷]]、[[泥炭|泥炭]]等。

== 经济 ==
2005年实现地区生产总值(GDP) 298.12亿元，按可比价计算，比上年增长14.2％，其中一产22.53亿元，二产175.13亿元，三产100.46亿元，三次产业结构的比例分别为7.6：58.7：33.7。按现行年平均汇率计算的人均生产总值突破4000美元，达到4404美元。

2010年实现地区生产总值565亿元，比上年增长15%；实现财政一般预算收入82.1亿元，其中地方财政收入44亿元，分别增长13%和12.8%；完成全社会固定资产投资210亿元，增长21.1%；实现社会消费品零售总额219.86亿元，增长19.3%；城镇居民人均可支配收入达到29670元，农民人均纯收入达到13770元，分别增长10.4%和13.4%。连续二次蝉联“[[中国最具幸福感城市|中国最具幸福感城市]]”光荣称号。
