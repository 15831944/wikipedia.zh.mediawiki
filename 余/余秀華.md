'''余秀華'''（{{bd|1976年|3月22日||catIdx=Y余}}），诗人。身患脑瘫残疾。她因作品《穿过大半个中国去睡你》在《诗刊》的[[微信|微信]]账号发布而出名。<ref>{{cite news |language = zh-hans |author = 施歌  |url = http://news.xinhuanet.com/photo/2015-01/18/c_127397034.htm |title = 诗人余秀华 |publisher = 新华网  |date = 2015-01-18 |accessdate = 2015-01-28}}</ref>

==简介==
1976年3月22日余秀华出生于[[湖北省|湖北省]][[钟祥市|钟祥市]]，因为乡卫生员的失误，在她睁开眼睛却还没有来得及啼哭之时就将脐带剪断，造成脑部缺氧。出生后的第四天，在她躯干的四个角落打了四针后，她才终于哭了出来。缺氧导致余秀华[[脑瘫|脑瘫]]，行动不便。高中毕业后在家赋闲。<ref>{{cite news |language = zh-hans |author = 新闻晨报 |url = http://news.hexun.com/2015-01-17/172470016.html |title = 余秀华：称我脑瘫诗人，我不难过 |publisher = 和讯网 |date = 2015-01-17 |accessdate = 2015-01-28 |deadurl = yes |archiveurl = https://web.archive.org/web/20150127065408/http://news.hexun.com/2015-01-17/172470016.html |archivedate = 2015-01-27 }}</ref>
她从2009年开始诗歌创作，主题大多关于自己的亲情、爱情以及对生活的感悟，到2015年初为止，共写了2000多首诗。

==家庭==
1995年，余秀华高中毕业，余秀华父母在家门口的公路上给她开了个小卖部。也是这一年，从四川流浪至此的[[尹世平|尹世平]]通过媒人的三次求亲最终说动了余文海夫妇，成了余家倒插门“吃老米”的女婿。余秀华说，“当时自己还差3个月才满20岁，为了结婚还虚报了年龄。好傻，不知道结婚还要做爱。” 余秀华与丈夫育有一子。

余秀华和老公尹世平常年分居，名存实亡的婚姻已经成为公开的秘密。余秀华的父亲余文海告诉记者：“尹世平的钱他自己挣自己花。直到他儿子上了高中，我们才让他每年交钱。以前都是交个几千块钱，今年是最多的一次，带了一万多元回来。他这个人不会说话，爱吹牛，交了些钱，就趾高气昂地骂老婆。”

余秀华有一个年少两岁的弟弟[[余文海|余文海]]。

==评价==
*学者和诗人[[沈睿|沈睿]]称余秀华为中国的[[艾米莉·狄金森|艾米莉·狄金森]]，“余秀华的诗歌是纯粹的诗歌，是生命的诗歌，而不是写出来的充满装饰的盛宴或家宴，而是语言的流星雨，灿烂得你目瞪口呆，感情的深度打中你，让你的心疼痛。”<ref>{{cite news |language = zh-hans |author = 澎湃新闻记者 徐萧  |url = http://news.163.com/15/0116/18/AG3O1TR000014SEH.html |title = 余秀华的诗写得很好，但别急着扣“中国狄金森”的帽子 |publisher = 网易  |date = 2015-01-16 |accessdate = 2015-01-28}}</ref>

*《诗刊》编辑刘年：“她的诗，放在中国女诗人的诗歌中，就像把杀人犯放在一群大家闺秀里一样醒目—别人都穿戴整齐、涂着脂粉、喷着香水，白纸黑字，闻不出一点汗味，唯独她烟熏火燎、泥沙俱下，字与字之间，还有明显的血污。”<ref>{{cite news |language = zh-hans |author = 乌鲁木齐晚报  |url = http://news.163.com/15/0119/02/AG9P66DG00014AED.html |title = 忽然 一个诗人余秀华 |publisher = 网易  |date = 2015-01-19 |accessdate = 2015-01-28}}</ref>

*评论家、上海译文出版社副社长赵武平：“身体患疾为余秀华的创作加上了同情分。”

*中国诗人、北京磨铁图书有限公司总裁[[沈浩波|沈浩波]]：“仅就诗歌而言，余秀华写得并不好，没有艺术高度。这样的文字确实是容易流行的。这当然也挺好，只不过这种流行稍微会拉低一些诗歌的格调。不过再怎么拉低，比起轻浮的乌青体来，总还算不上丢人败兴。”<ref>{{cite news |language = zh-hans |author = 新京报  |url = http://cul.qq.com/a/20150120/016941.htm |title = 脑瘫女诗人余秀华：穿越大半个中国去读你 |publisher = 腾讯网  |date = 2015-01-20 |accessdate = 2015-01-28}}</ref>

*诗人翻译家美国金重选译了余秀华的诗歌， 编入2016年中国当代诗选POETRY CARAVAN《诗歌大篷车》第一卷:“余秀华的诗歌和英美诗歌及文化有着神秘的契合”。
