'''余部橋樑'''（{{jpn|j=余部橋梁|hg=あまるべきょうりょう|rm=Amarube Kyōryō|f=y}}），又稱'''余部橋'''，2010年改建前又稱'''余部鐵橋'''，是位於[[日本|日本]][[兵庫縣|兵庫縣]][[香美町|香美町]]香住區的[[高架橋|高架]][[鐵路橋|鐵路橋]]，由[[JR西日本|JR西日本]]的[[山陰本線|山陰本線]]使用。第一代的{{link-ja|鋼橋|鋼橋|鋼梁}}{{link-en|排架橋|Trestle bridge}}於1909年（[[明治|明治]]42年）12月開工、1912年（明治45年）3月1日竣工；現今採用{{link-en|脊背橋|Extradosed bridge}}設計的第二代橋梁在2007年3月開工、2010年8月12日啟用。在舊橋改建前，由於其不同一般鐵橋的美式構造與特殊的地理位置，吸引許多[[鐵道迷|鐵道迷]]及觀光客參觀攝影，是[[山陰地方|山陰地方]]的觀光名勝之一。

== 概況 ==
* 位置：[[兵庫縣|兵庫縣]][[美方郡|美方郡]][[香美町|香美町]]（原[[城崎郡|城崎郡]][[香住町|香住町]]），山陰本線[[鎧車站|鎧]]、[[餘部車站|餘部]]兩車站之間。
* 橋長度：310.59公尺
* 橋高度：41.45公尺
* 橋墩：舊橋：11座；新橋：4座
* 桁架：舊橋：23連
* 路線：單線
* 橋下方為[[日本國道178號線|國道178線]]穿過。

== 建設經過 ==
余部橋位於[[日本海|日本海]]沿岸，為多山的地形，在山陰本線興建時，日本技師[[古川晴一|古川晴一]]與[[美國|美國]]技師討論後決定興建一座橋梁。由美國技師主導設計，正因為如此，橋梁是十足的美國式風格。鋼材由美國運到日本[[九州|九州]]的[[門司港|門司港]]之後，轉送到施工地點加以施工。以當年的工程技術，艱難的程度僅次於同樣位於山陰本線的[[桃觀隧道|桃觀隧道]]，動員了25萬人次工人參與施工，高危險的程度讓施工單位為施工人員投保了2萬日元的保險。

== 行車事故 ==
[[File:Cenotaph_under_Amarube-tekkyo_bridge.JPG|225px]]
1986年12月28日的下午1時25分左右，從[[香住車站|香住車站]]迴送到[[濱坂車站|濱坂車站]]的{{link-ja|雅號列車|みやび (鉄道車両)|「雅」號列車}}（{{lang|ja|みやび}}，Miyabi）行經余部橋時，突然被日本海方向來的風吹襲，七節客車車廂全部墜落橋下，僅[[鐵路機車|機車]]與一節客車的一組轉向架未墜橋。墜橋後砸中橋下的一座水產加工工廠，造成了工廠員工5人以及列車車長1人喪生。事發原因是因為列車司機漏看了風速超過每秒25公尺時的警示號誌而未停車等待風歇，這是余部橋通車以來最為嚴重的行車事故。事故之後，當年的日本國鐵規定該路段的風速超過每秒20公尺時，該路段所有列車均要停駛，改用公路[[巴士|巴士]]接駁。1988年10月23日，在事故地點立了慰靈碑，以及一尊[[觀世音菩薩|觀世音菩薩]]的石像，每年的12月28日都會舉行法會以慰往生者之靈。

== 改建 ==
[[File:New_amarube_bridge.JPG|225px]]
因為鐵橋使用90年，且當地風大，為了行車安全起見，JR西日本規劃在原橋往山側（南側）6公尺處興建新橋，預算30億日圓。但當地人反應兩極，一方認為搭建新橋後將破壞原橋的美觀，再也無觀光的價值；另一方則贊成盡速改建，以降低列車通過時[[噪音|噪音]]以及[[鐵鏽|鐵鏽]]掉落。

基於安全考量，2007年3月29日開始進行[[預力混凝土|預力混凝土]]新橋興建工程，舊鐵橋於2010年7月16日通過最後列車之後正式停用，濱坂到香住之間暫以公路接駁，並於7月23日開始進行橋東側的切換工程，拆除鐵橋的一端的鋼梁後，一端的水泥橋面（長98公尺，重3800公噸）以油壓機械移動與原路線銜接，30日施作西側的切換工程<ref>{{cite news|url=http://mytown.asahi.com/hyogo/news.php?k_id=29000001007240005|title=橋げた回転　余部鉄橋架け替え工事|publisher=朝日新聞|date=2010-07-24|accessdate=2010-07-31|deadurl=yes|archiveurl=https://web.archive.org/web/20100729141350/http://mytown.asahi.com/hyogo/news.php?k_id=29000001007240005|archivedate=2010-07-29}}</ref>。水泥新橋已於8月12日完工啟用。完工後可讓列車在低於風速每秒30公尺的情形下安全通過。

鐵橋停用後，當地政府保存一部份靠近餘部車站端的橋體，興建「{{link-ja|余部鐵橋「天空之站」|余部鉄橋「空の駅」}}」（{{lang|ja|余部鉄橋「空の駅」}}）做為展望台之用<ref>{{cite news|url=http://www.ryoutan.co.jp/news/2010/08/01/002465.html|title=鉄鋼からコンクリートへ　余部橋梁架け替え着々|publisher=両丹日日新聞|date=2010-08-01|accessdate=2010-08-02}}</ref>，2013年5月3日舊橋體改建為展望台的工程竣工開放參觀，2017年11月26日啟用天空之站的電梯，使得在車站上下車的旅客及參觀天空之站的遊客更為便利。
