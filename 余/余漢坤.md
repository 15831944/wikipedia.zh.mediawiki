'''余漢坤'''{{MH}}{{JP}}，測量師，現任[[離島區議會|離島區議會]]副主席（[[大嶼山_(選區)|大嶼山]]民選議員）兼[[恒基地產|恒基地產]]地產發展部總經理。

==經歷==
余漢坤是大澳原居民，新界鄉事派人物。[[新界鄉議局|新界鄉議局]]前主席[[劉皇發|劉皇發]]的女婿。余漢坤的太太[[劉玉蓮|劉玉蓮]]是一位時裝設計師。<ref >{{Cite web| url = http://orientaldaily.on.cc/cnt/news/20141025/00176_120.html| title = 中環出更：行動失控 藍爺叫學生回家 | work =東方日報| date = 2014-10-25}}</ref>

他曾任離島區委任議員及[[信和置業|信和置業]]有限公司總經理。

2014年，獲委任為[[大嶼山發展諮詢委員會|大嶼山發展諮詢委員會]]委員。<ref >{{Cite web| url = http://www1.hkej.com/dailynews/politics/article/727811/大嶼山諮委逾半持物業土地| title = 大嶼山諮委逾半持物業土地信置余漢坤涉項目最多| work = 信報| date = 2014-05-15}}</ref>被時任[[民主黨_(香港)|民主黨]]立法會議員[[胡志偉|胡志偉]]指涉利益衝突。<ref >{{Cite web| url = http://orientaldaily.on.cc/cnt/news/20140515/00176_045.html| title = 任大嶼山發展委員 劉皇發女婿遭質疑 | work = - 東方日報| date = 2014-05-15}}</ref>

因[[2015年香港區議會選舉|2015年香港區議會選舉]]全面取消區議會委任制，余漢坤辭職參加區議會直選<ref >{{Cite web| url = http://the-sun.on.cc/cnt/news/20150326/00408_006.html|title = 政壇：搞搞政：余漢坤捨厚職心繫選戰？| work = 太陽報| date =2015-03-26}}</ref>，最後以稍多於半數得票成功當選為區議員。<ref >{{Cite web| url = http://www.elections.gov.hk/dc2015/chi/results_islands.html?1457003910916| title = 2015區議會選舉| work = 政府新聞網| date =2015-11-23}}</ref>

2016年，當選為離島區議會副主席。<ref>{{Cite web| title = 十八區區議會選出主席及副主席 | work=香港政府新聞網 | date = 2016-01-07| url = http://www.info.gov.hk/gia/general/201601/07/P201601070744.htm}}</ref>
