'''余若薇'''{{SC}}{{JP}}（{{lang|en|'''Audrey Eu Yuet-mee'''}}，{{bd|1953年|9月11日|catIdx=Y余}}），香港[[資深大律師|資深大律師]]，前[[公民黨_(香港)|公民黨]]黨魁及主席、[[香港立法會|香港立法會]]議員及[[香港大律師公會|香港大律師公會]]主席。

1993年當上英國[[御用大律師|御用大律師]]；[[香港主權移交|1997年中國接收香港主權]]不久[[居港權|香港居留權]]的法理基礎引起爭論，在任期間堅持在[[香港法律制度|香港法律制度]]內解決而不尋求[[全國人大|全國人大]]釋法，自此為人悉知。

2000－2006年[[規劃地政局|建築物上訴審裁處]]主席，1999－2003年[[工商及科技局|消費品安全上訴委員會]]主席，2000－2012年三屆[[香港立法會|立法會]]議員。<REF>[http://www.audreyeu.hk/個人檔案/ 個人檔案]，余若薇官方網站。</REF>2010年香港政制改革爭議不絕，特首[[曾蔭權|曾蔭權]]邀請她上[[政制改革方案辯論|電視直播一對一辯論]]，引起一時風話。2006－2010年是[[公民黨_(香港)|公民黨]]創黨黨魁，2011年改任黨主席，屬[[泛民主派|泛民主派]]。

== 家庭背景 ==
余若薇生於[[香港|香港]]，祖籍[[上海|上海]]，是[[天主教徒|天主教徒]]。曾就讀於[[嘉諾撒聖方濟各書院|嘉諾撒聖方濟各書院]]和[[聖保羅男女中學|聖保羅男女中學]]，為[[香港大學|香港大學]][[法律|法律]][[學士|學士]]，[[倫敦大學|倫敦大學]]法律[[碩士|碩士]]。

已婚，與丈夫[[胡健維|胡健維]][[醫生|醫生]]育有三名女兒，家翁胡百富，曾在1950年代出任[[市政局|市政局]]議員長達13年，胡百富之兄為前[[香港立法局|立法局]][[首席非官守議員|首席非官守議員]][[胡百全|胡百全]]，胡百全之妾為息影女星[[黃夏蕙|黃夏蕙]]。余弟[[余若海|余若海]]亦是資深大律師。

2007年，余若薇在診治中發現[[腦部|腦部]]出現一個3毫米的[[腦動靜脈血管畸形|動脈瘤]]，若血管瘤爆開隨即[[中風|中風]]或致命，她也表示現會積極面對，並如常工作<ref>{{cite web |url=https://www.scmp.com/article/606536/audrey-eu-diagnosed-tumour |title=Audrey Eu diagnosed with tumour |newspaper=[[South_China_Morning_Post|South China Morning Post]] |author=Jimmy Cheung |date=2007-09-03 |language=en}}</ref>。

== 加入政壇 ==
在加入政壇前，她曾擔任[[香港大律師公會|香港大律師公會]]會長。在[[香港主權移交|香港主權移交]]中國，在居港權問題上，反對人大釋法的堅定立場，漸為港人所認識。於2000年9月[[程介南|程介南]]因貪污醜聞辭職，余若薇出選[[2000年香港立法會香港島地方選區補選|立法會議席香港島地方選區補選]]，得到民主派一眾支持，壓倒性贏得議席，從此踏入政壇。

== 《基本法》二十三條關注組 ==
2002年，香港政府有意為《[[香港特別行政區基本法|基本法]]》[[香港特別行政區基本法第二十三條|第二十三條]]立法，當時她與其他出身自大律師公會的法律界人士，包括[[梁家傑|梁家傑]]、[[吳靄儀|吳靄儀]]、[[湯家驊|湯家驊]]、香港大學學者[[陳文敏|陳文敏]]、中文大學政政系教授[[戴大為|戴大為]]等合組[[廿三條關注組|廿三條關注組]]，在公眾未留意二十三條諮詢文件問題，不斷地向香港市民力陳《基本法》二十三條立法的弊端。

她個人在議會上，就二十三條立法的弊端與前[[保安局|保安局]]局長[[葉劉淑儀|葉劉淑儀]]針鋒相對，因而得到了'''民主女神'''的稱號。

由於她與[[廿三條關注組|廿三條關注組]]的成員以理性、溫和而堅定的方法，向香港市民陳述利弊之處，不單令香港市民參與反對二十三條運動，也令她的聲望在[[香港七一遊行|香港七一遊行]]後節節上升。

== 《基本法》四十五條關注組 ==
2004年，港人開始關注香港基本法第四十五條及六十八條，就2007年第三屆行政長官與零八年第四屆立法會選舉方法之規定。余若薇與其他數-{}-位大律師如[[陳文敏|陳文敏]]、[[張健利|張健利]]、[[梁家傑|梁家傑]]、[[李志喜|李志喜]]、[[吳靄儀|吳靄儀]]與[[湯家驊|湯家驊]]，律師[[張達明|張達明]]和[[帝理邁|帝理邁]]，及前[[立法局|立法局]]議員[[陸恭蕙|陸恭蕙]]等組成[[四十五條關注組|四十五條關注組]]，推動零七零八雙普選。因普選議題獲得大部分港人支持，加上其溫和、中產和專業的形象而獲得市民普遍歡迎。

於2004年夥同[[前綫_(1996年)|前綫]]的[[何秀蘭|何秀蘭]]於港島區合組名單出選[[2004年香港立法會選舉|2004年香港立法會選舉]]，因民望高企在民調下一度大幅領先，而有「吸票機」及「票-{后}-」稱號。後卻因另一張泛民主派名單[[李柱銘|李柱銘]]與[[楊森_(香港)|楊森]]的組別宣稱告急而流失大量選票，導致泛民主派配票失敗，最終該名單只得73,844票，余若薇成功出線，但[[何秀蘭|何秀蘭]]則以815票之差敗於[[民建聯|民建聯]]的[[蔡素玉|蔡素玉]]。

四十五條關注組成員及其他人士於 2006年3月19日組成香港公民黨，余若薇出任首任黨魁。2012年12月1日，公民党领导层换届，余若薇众望所归出任党主席。

== 2008年立法會選舉 ==
余若薇參與[[2008年立法會選舉|2008年立法會選舉]]於[[香港島選區|港島區]]出選，合組名單包括新人[[陳淑莊|陳淑莊]]及[[容詠嫦|容詠嫦]]。根據港大民意調查，余若薇一直是前一屆民望最高的立法會議員。她欲於2008年立法會選舉提拔新人，自願擔任名單第二位，最後成功連任，並且把新人[[陳淑莊|陳淑莊]]帶入議會，是港島區得票最多的名單。但名單扣除陳淑莊當選所須的票數後，僅餘30,362票，只能讓余若薇取得港島選區最後一席，較第五位的四年前的參選拍檔何秀蘭還少525票。

擔任立法會議員期間，根據港大民意網站的調查，余若薇一直是評分最高的議員之一，可是在公民黨於2011年被指涉及[[港珠澳大橋|港珠澳大橋]]環境評估報告司法覆核及[[外傭居港權|外傭居港權]]司法覆核後，她的評分大跌。<ref>[http://news.sina.com.hk/news/1542/3/2/2480149/1.html 《余若薇梁家傑民望「插水」》]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}，《星島日報》，2011年11月2日。</ref>
