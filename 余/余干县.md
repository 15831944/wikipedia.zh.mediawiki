'''-{zh-cn:余{{ruby-py|干|gān}};zh-tw:餘{{ruby-py|干|ㄍㄢ}};zh-hk:餘{{ruby-py|干|gon<sup>1</sup>}}}-县'''是[[中华人民共和国|中华人民共和国]][[江西省|江西省]][[上饶市|上饶市]]所辖的一个[[县|县]]。

== 历史 ==
[[File:Jiangxi_Sheng_Bowuguan_20120627-10.jpg|thumb]]出土]]
[[秦|秦]]（前221年）置县，曾改名“餘汗（音“寒”）”、“治干”。

[[元|元]][[元贞|元贞]]二年（1296年），曾升县为州。

1952年属上饶地区至今。

== 地理 ==
西为[[鄱阳湖|鄱阳湖]]区，中部为[[鄱阳湖平原|鄱阳湖平原]]。

== 交通 ==
*[[206国道|206国道]]经县东部
*102省道、208省道
*昌万公路，2004年6月昌万公路建成通车以来，余干至南昌的距由原先的175公里减少为74公里，车程由3个多时减少不到1小时。使得余干成为南昌的经济后花园。
*[[德昌高速公路|德昌高速公路]]，2010年正式动工建设，未来将会更大程度的拉动余干经济的发展。

==宗教==
余干县有[[基督徒|基督徒]]9万余人，占人口十分之一，[[基督教|基督教]]登记堂点324个<ref>复兴的余干教会，《天风》2009年12期</ref>。江埠乡则超过人口的三分之一<ref>段琦：宗教生态失衡对基督教发展的影响，中国民族报，2010年1月19日</ref>。
