'''余莓莓'''（{{bd|1964年|7月14日|catIdx=Y余}}）是[[臺灣|臺灣]][[政治|政治]][[評論家|評論員]]和[[政論節目|政論節目]][[政論節目#政論員|評論員]]，2012年與2016年兩度參與民進黨全國不分區立委選舉，分別排序至27名與26名，均未當選。

== 經歷 ==
*曾任[[台北市議會|台北市議會]]議員助理，1988年加入[[民主進步黨|民主進步黨]]。
*1995年從[[德國|德國]]返臺，曾為[[民主進步黨|民主進步黨]][[王雪峰_(臺灣)|王雪峰]]競選立法委員幫忙助選。
*1996年任民進黨中央黨部文化宣傳部廣電組組長。
*1998年7月任民進黨文宣部副主任<ref name="蘋果日報" />。
*2000年任[[陳水扁|陳水扁]]競選總統總部活動部副主任、代主任。後再任[[民主進步黨|民主進步黨]]黨職。
*2001年6月任民進黨中央黨部中國事務部副主任，2002年7月再續任，2004年5月卸任。
*2003年還任民進黨政策與論述小組執行秘書。與前民進黨中央黨部中國事務部主任[[顏建發|顏建發]]等關係很好。外傳其屬民進黨[[新潮流系|新潮流系]]，其有不實亦有傳聞稱其與[[謝長廷|謝長廷]]關係較為親近，或稱與[[呂秀蓮|呂秀蓮]]關係亦頗良好。
*2012年被排入民進黨不分區立委名單中第27名，落選。
*2015年3月參加民進黨[[新北市第六選舉區|新北市第六選舉區]]（板橋西區）立委初選，被視為謝系代理人，2015年3月21日確定敗給[[新北市議員|新北市議員]][[張宏陸|張宏陸]]。
*2016年被排入民進黨不分區立委名單中第26名，落選。

== 社團任職 ==
“中華港澳之友會”理事。

== 主持 ==
*現任廣播節目主持人、電視政論節目的時事評論員。

==評論風格==
余莓莓經常參加各種政論访谈節目，立場為主張[[台灣民族主義|台灣民族主義]]，會大力抨擊反對台灣民族主義的人，經常在媒體上著文為台灣民族主義價值辯護。

在台灣《[[中共研究|中共研究]]》、《[[共黨問題研究|共黨問題研究]]》（[[展望與探索|展望與探索]]）等雜誌曾發表學術論文多篇。曾多次去[[中國大陸|中國大陸]]，先後到[[廣東|廣東]]、[[西安|西安]]、[[北京|北京]]等地旅遊。在淡江大學大陸所碩專班論文《[[九一一|九一一]]震蕩對中國中亞戰略的衝擊》，指導教授[[張五岳|張五岳]]。

== 家庭生活 ==
父親余瑞成，母親楊素琴，丈夫陳天佑是德國土木工程學博士，現在於臺灣[[新竹市|新竹]][[中華大學|中華大學]]任教。

==相關事件==
2006年8月21日上午，余莓莓在捷運淡水線台北車站月台上車時，因人群擁擠又邊走邊想事情，不慎雙腳踩空跌入月台與車廂間空隙，致左小腿開放性外傷、右腳踝扭傷、右小腿挫傷血腫後，併發蜂窩性組織炎，經三個月治療才痊癒。 
余莓莓認為，當時車廂與月台空隙十五點七公分，超出標準十公分達百分之五十，且月台上只有邊緣地上寫著「小心月台間隙」警語，沒有其他警告標示，該警語又模糊不清，無法達到警示效果。 
余莓莓認為她是電台主持人，並常在電視政論節目擔任來賓，是稍具知名度的公眾人物，因受傷後致她身心嚴重受損，故求償包含計程車代步費、雷射去疤費用及慰撫金共一百一十三萬餘元。 
2010年12月，高等法院審理認定，台北捷運公司沒改善月台間隙太寬的問題，又沒提醒乘客注意危險，判決台北捷運公司賠余的醫療、除疤、交通費共二十四萬七千多元定讞。
