'''余玠'''（{{bd|生年不詳||1253年|catIdx=}}），字'''義夫'''，號'''樵隱'''。[[蘄州|蘄州]][[廣濟|廣濟]]（今[[湖北|湖北]][[武穴市|武穴市]][[余川镇|余川镇]]）人。[[南宋|南宋]]名將。

== 早期事迹 ==

幼時家貧，就讀於[[滄浪書院|滄浪書院]]、[[白鹿洞書院|白鹿洞書院]]。少年時不慎在茶館打死人，失學後投軍，初為[[淮東|淮東]][[制置使|制置使]][[趙葵|趙葵]]幕僚，後因功補進入副尉，不久又擢升作監[[主薄|主薄]]。<ref>《宋史‧余玠傳》</ref>
端平三年（1236年），赵癸任淮东安抚制置使兼[[扬州|扬州]]（今属江苏）知州后，余玠晋谒，被留置幕府。嘉熙二年（1238年）正月，任[[招信军|招信军]]（今安徽[[嘉山|嘉山]]东北）[[知军|知军]]兼淮东制置司参议官。嘉熙四年（1240），余玠率水军由运河进攻至[[开封|开封]]有功，升任淮东[[提点刑狱|提点刑狱]]，后又兼[[淮安|淮安]]知州、淮东制置司参谋官。淳祐元年（1241年），又以率军援救安丰（今安徽寿县）有功，升任淮东制置副使。

== 主政四川 ==

淳祐二年（1242年）六月，余玠晋见[[宋理宗|宋理宗]]，愿以恢复[[四川|四川]]为己任，升任权兵部侍郎、四川[[宣谕使|宣谕使]]。从此，领导四川抗战。淳祐三年十二月，又升任兵部侍郎、四川安抚制置使、兼重慶知府。当时四川已残破，并经常受到蒙古军的侵扰。余玠到任后，招贤纳士，共商大计。[[冉璡|冉璡]]、[[冉璞|冉璞]]兄弟应召入幕，首先建议于合州（今重庆合川）城东钓鱼山上筑城，并移州府于[[钓鱼城|钓鱼城]]。余玠任命冉氏兄弟为[[合州|合州]]长官，负责筑城。此前，已修筑多处山城，此后也修筑了十多座山城，许多州府移居山城，顽强地抗击蒙古军。余玠实行轻徭薄赋政策，俞兴又开屯田于成都（今属四川），经济有所恢复。淳祐十年（1250年），余玠率部巡边，一直进到被蒙古军占领的[[兴元|兴元]]（今陕西汉中）。十二年（1252年），又率部西向[[嘉定府|嘉定府]]（今四川乐山）抗击南犯的蒙古军，出现了[[宋元战争|宋元战争]]爆发四川以来最好的军事、政治、经济形势。

余玠决定在此基础上，逐步清除跋扈的大将，以统一四川的军政。他乘到嘉定府之机，首先将当时也在嘉定的[[利州路|利州路]]都统制[[王夔|王夔]]处死，派[[杨成|杨成]]代统其部属。但移驻云顶山（当在今广元境内）的[[利州|利州]]都统制司，却自行推举[[统制|统制]][[姚世安|姚世安]]接任[[都统制|都统制]]之职。余玠率部至云顶山下，派新任都统制杨成上山任职，遭到姚世安的抵制。姚世安又通过避居云顶山现任宰相[[谢方叔|谢方叔]]的亲属，向谢方叔诬陷余玠。

== 晚年结局 ==

宝祐元年（1253年）五月，南宋朝廷猜疑余玠，决定将他召回。七月，余玠暴卒于四川(另有一說是服毒自盡)。余玠死后，继续受到攻击，还受到追削资政殿学士职名的处分。直至宝祐三年（1255年）八月，谢方叔罢相以后，才又“诏追复余玠官职”。
