'''余洒度'''（{{bd|1898年||1934年|catIdx=Y余}}），出生於[[湖南|湖南]][[平江|平江]]，中國軍事、政治人物，[[中國共產黨|中國共產黨]]、[[中國工農紅軍|中國工農紅軍]]早期領導人。

== 進入黃埔軍校與加入中共 ==
[[1922年|1922年]]平江發生農民運動時期接受共產主義思想，[[1924年|1924年]]考入[[黃埔軍校|黃埔軍校]]第二期，在周恩來的影響下加入了中國共產黨，是黃埔軍校內[[中國青年軍人聯合會|中國青年軍人聯合會]]主要成員，也是黃埔軍校[[血花劇社|血花劇社]]的重要演員。[[1926年|1926年]]7月，受命潛入武漢組織兵變。[[1927年|1927年]]6月，任[[張發奎|張發奎]]的[[國民革命軍|國民革命軍]]第四集團軍第二方面軍總指揮部警衛團中校團附建第一營營長，該團骨幹大部來自[[葉挺|葉挺]]獨立團<ref>[http://www.hoplite.cn/templates/yusadu.htm 黄埔军校二期-余洒度]</ref>。

== 秋收暴動 ==
[[1927年|1927年]]8月，[[盧德銘|盧德銘]]、余灑度率領警衛團參加[[南昌暴動|南昌暴動]]，部隊未開到[[南昌|南昌]]，起義軍已經南下，警衛團遂輾轉在湘鄂贛邊界地區。期間盧德銘離開部隊尋找黨組織，余灑度代理警衛團團長職務。他帶領部隊來到[[修水縣|修水縣]]，與[[羅榮桓|羅榮桓]]帶領的農民武裝取得聯繫，以江西省防軍第一師的名義逐漸擴大力量。

[[1927年|1927年]][[9月9日|9月9日]]，中共在湘赣边界發動[[秋收起义|秋收起义]]<ref>{{Cite book|url=https://books.google.com/books?id=8TVsAgAAQBAJ&pg=PT120&dq=%25E3%2580%25829%25E6%259C%25889%25E6%2597%25A5,%25E6%25B9%2598%25E8%25B5%25A3%25E8%25BE%25B9%25E7%2595%258C%25E7%25A7%258B%25E6%2594%25B6%25E8%25B5%25B7%25E4%25B9%2589%25E7%2588%2586%25E5%258F%2591&hl=zh-CN&sa=X&ved=0ahUKEwj20fOKoK3PAhXFCD4KHTVqA9wQ6AEINjAF#v=onepage&q&f=false|title=影响中国历史进程的战役（世界军事之旅）|last=李宏|date=2013-12-24|publisher=青苹果数据中心|language=zh}}</ref>。駐修水的[[武漢国民政府|武漢国民政府]]警衛團、[[安源|安源]]路礦工人，平江、[[瀏陽|瀏陽]]農民義勇軍纷纷起事，各路起义军先攻克平江、瀏陽，然後包抄[[長沙|長沙]]，但先在平江、瀏陽先後失利，又因收编部队叛乱及临时组成的农军经验不足，導致溃不成军。根據[[中國共產黨|中共黨]]組織的決定，湘贛邊界參加秋收暴動的各支隊伍，集中整編為[[中國工農紅軍|中國工農革命軍]]第一軍第一師，一共三個團，余洒度任師長<ref>[http://cpc.people.com.cn/BIG5/33837/2534129.html 秋收起義--中國共產黨新聞--中國共產黨新聞網 秋收起義--中國共產黨新聞--中國共產黨新聞網]</ref>，[[毛澤東|毛澤東]]任師黨代表，原駐[[修水縣|修水縣]][[武漢國民政府|武漢國民政府]]警衛團團長[[盧德銘|盧德銘]]任秋收暴動總指揮，共5000多人。

9月19日，起义军余部在[[文家市|文家市]]汇合，余灑度主張進攻長沙<ref>[http://www.zgdsw.org.cn/GB/222988/223510/14821354.html 毛泽东怎样引兵井冈--中国共产党历史网]</ref>，毛澤東說服大家放棄，主张「向江西省[[萍乡县|萍乡]]退却」<ref>{{cite web|language=zh-hans |publisher=新華網 |title="政治纪律决议案 （一九二七年十一月十四日）" |url=http://news.xinhuanet.com/ziliao/2004-11/25/content_2259029.htm |deadurl=yes |archiveurl=https://web.archive.org/web/20151102014834/http://news.xinhuanet.com/ziliao/2004-11/25/content_2259029.htm |archivedate=2015-11-02 }}</ref>。毛澤東則計畫將部隊帶往南去一百七十公里的[[井岡山|井岡山]]，指揮官們都不同意進山，但最後勉強服從了毛澤東，因為毛澤東是唯一在場的[[党代表|党代表]]。20日从[[文家市|文家市]]出发之後突然遭到袭击，总指挥盧德銘陣亡。途中在江西萍乡确定向井岡山进发，9月29日，抵达[[江西省|江西省]][[永新县|永新县]]三湾村，在“泰和祥”杂货铺召开中共前敌委员会扩大会议，<ref>{{Cite web|url=http://culture.china.com.cn/lishi/2010-09/16/content_20943778.htm|title=三湾改编——毛泽东加强人民军队中党组织建设|accessdate=2016-09-26|work=culture.china.com.cn}}</ref>原本5000多人的队伍，这时只剩下不足千人<ref>{{Cite book|url=https://books.google.com/books?id=WB04AQAAIAAJ&q=%25E5%258E%259F%25E6%259C%25AC5000%25E5%25A4%259A%25E4%25BA%25BA%25E7%259A%2584%25E9%2598%259F%25E4%25BC%258D%25EF%25BC%258C%25E8%25BF%2599%25E6%2597%25B6%25E5%258F%25AA%25E5%2589%25A9%25E4%25B8%258B%25E4%25B8%258D%25E8%25B6%25B3%25E5%258D%2583%25E4%25BA%25BA&dq=%25E5%258E%259F%25E6%259C%25AC5000%25E5%25A4%259A%25E4%25BA%25BA%25E7%259A%2584%25E9%2598%259F%25E4%25BC%258D%25EF%25BC%258C%25E8%25BF%2599%25E6%2597%25B6%25E5%258F%25AA%25E5%2589%25A9%25E4%25B8%258B%25E4%25B8%258D%25E8%25B6%25B3%25E5%258D%2583%25E4%25BA%25BA&hl=zh-CN&sa=X&ved=0ahUKEwjliprGnK3PAhXJGT4KHYFWBTIQ6AEIIjAB|title=党的建设|last=|first=|date=2008-01-01|publisher=中共甘肃省委《党的建设》 杂志社|year=|isbn=|location=|pages=|language=zh|quote=毛泽东在"泰和祥"杂货铺召开[[中國共產黨|中共黨]]前敌委员会扩大会议，原本5000多人的起义队伍，这时只剩下不足千人。}}</ref>，隨後即進行著名的「[[三灣改編|三灣改編]]」。

10月中旬，余灑度在湖南[[酃县_(南宋)|酃縣]]水口脫逃部隊<ref>{{Cite web|url=http://news.ifeng.com/history/zhongguojindaishi/detail_2010_07/05/1717817_0.shtml|title=井冈山上的黄埔生：余洒度贩毒被国民党枪决_历史频道_凤凰网|accessdate=2016-09-26|author=|date=2010-07-05|work=news.ifeng.com|publisher=凤凰网}}</ref>。10月19日，余灑度輾轉來到上海，向當時設在上海的黨中央寫了《警衛團及平瀏自衛軍合併原委參加此次兩湖戰役報告略書》的工作彙報，該文章刊登在中央內部秘密刊物《中央政治通訊》第13期，此後余灑度逐漸脫離中國共產黨組織。

== 脫離中共、進入國民黨與被蔣介石槍決<ref>[http://www.liuyangshi.cn/liuyangxinwen/wenyi/20181103/197044_4.html 秋收起义重要参与人之余洒度、苏先俊的最后结局(4)_文艺新闻__浏阳之窗新闻资讯中心]</ref> ==
[[1929年|1929年]]，余灑度參加[[譚平山|譚平山]]、[[章伯鈞|章伯鈞]]等人組織的[[中國農工民主黨|中華革命黨]]。[[1930年|1930年]]5月，[[鄧演達|鄧演達]]秘密返回上海，創建了[[中國國民黨臨時行動委員會|中國國民黨臨時行動委員會]]，余灑度加入並受鄧演達的委託籌建[[黃埔革命同學會|黃埔革命同學會]]，號召黃埔學生反對蔣介石。之後[[南京國民政府|南京國民政府]]通緝鄧演達，[[1931年|1931年]][[8月17日|8月17日]]，鄧演達、余灑度先後在上海被捕。[[11月29日|11月29日]]，鄧演達被殺害，而余灑度則表示願意「悔過自新」，之後進入了[[中央軍校特別訓練班|中央軍校特別訓練班]]學習。

[[1932年|1932年]]3月，[[復興社|復興社]]成立，余灑度成為該社的一個重要成員，在政訓研究班學習結束後，他與同班畢業學員五百餘人組成華北宣傳隊，赴華北進行宣傳活動。

[[1933年|1933年]]成為國民革命軍第61軍政訓處少將處長。[[1934年|1934年]][[北平|北平]][[憲兵|憲兵]]團團長[[蔣孝先|蔣孝先]]檢舉余灑度販毒，他旋即被逮捕押往南京，被蔣介石親自下令槍決。
