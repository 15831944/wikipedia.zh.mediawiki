'''余远辉'''（{{bd|1964年|1月28日||catIdx=Y余}}），[[瑶族|瑶族]]，[[广西|广西]][[恭城|恭城]]人。1984年12月加入中国共产党，1986年7月参加工作，武汉大学经济与管理学院经济思想史专业毕业，在职研究生学历，经济学博士，讲师。中共第十七届、十八届中央候补委员，第十届全国政协委员，第十一届全国人大代表。

[[广西农学院|广西农学院]]（现并入[[广西大学|广西大学]]）农学系农学专业毕业，[[大连理工大学|大连理工大学]]社会科学系思想政治教育专业第二学士、[[中央民族大学|中央民族大学]]少数民族经济研究所中国少数民族经济专业法学硕士、[[武汉大学|武汉大学]]经济与管理学院经济思想史专业经济学博士学位。历任中共钦州市委副秘书长，广西壮族自治区人事厅副厅长，共青团广西区委书记，中共梧州市委副书记、市长、市委书记、市人大常委会主任等职。2008年2月获任广西壮族自治区党委常委，次年兼区委秘书长，2010年兼任区委办公厅主任<ref>{{cite web|accessdate=2012-05-30|url=http://www.guilindj.gov.cn:8090/listview.asp?id=6305|title=余远辉任广西壮族自治区党委常委、秘书长|publisher=广西日报}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2013年5月任[[中国共产党广西壮族自治区委员会|广西壮族自治区党委]]常委、[[南宁市|南宁市]]委书记。<ref name="测试">[http://renshi.people.com.cn/n/2013/0504/c139617-21364774.html 余远辉任广西南宁市委书记] {{webarchive|url=https://web.archive.org/web/20150621184128/http://renshi.people.com.cn/n/2013/0504/c139617-21364774.html |date=2015-06-21 }}，人民网。</ref>2015年5月22日，中纪委宣布，广西壮族自治区党委常委、南宁市委书记余远辉涉嫌严重违纪违法，被组织调查<ref>{{cite web|title=广西壮族自治区党委常委、南宁市委书记余远辉涉嫌严重违纪违法接受组织调查|url=http://www.ccdi.gov.cn/xwtt/201505/t20150522_56745.html|publisher=中纪委监察部网站|accessdate=2015-05-22}}</ref>。10月16日，中纪委宣布余远辉被双开，并移送司法机关立案处理<ref name=PanYu>{{cite web |url =http://www.ccdi.gov.cn/xwtt/201510/t20151016_63607.html|title =内蒙古自治区党委原常委、自治区政府原副主席潘逸阳，广西壮族自治区党委原常委、南宁市委原书记余远辉严重违纪被开除党籍和公职 |accessdate = 2015年10月16日 |publisher = 中央纪委监察部网站}}</ref>。11月4日，[[湖南省人民检察院|湖南省人民检察院]]宣布，依法对余远辉以涉嫌受贿罪决定逮捕<ref>{{cite web |url =http://news.china.com.cn/txt/2015-11/05/content_36985769.htm|title =湖南省检察院对原南宁市委书记余远辉决定逮捕 |accessdate = 2015年11月5日 |publisher = 中国网}}</ref>。2017年4月28日，湖南省湘西土家族苗族自治州中级人民法院一身宣判，被告人余远辉利用职务便利，为他人提供帮助，收受贿赂共计折合人民币881万余元，以受贿罪判处有期徒刑11年，并处罚金人民币100万元，对受贿所得财物及孳息予以追缴，上缴国库<ref>{{cite web |url = http://www.ccdi.gov.cn/yw/201704/t20170428_98315.html |title = 广西壮族自治区党委原常委、南宁市委原书记余远辉一审获刑十一年 |accessdate = 2017年4月28日 |publisher = 中央纪委监察部网站}}</ref>。
