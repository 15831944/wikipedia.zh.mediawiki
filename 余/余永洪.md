'''余永洪'''（{{bd|？|||}}）籍贯不详，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
余永洪长期在[[成都军区|成都军区]]服役。曾任[[中国人民解放军陆军第十四集团军|中国人民解放军陆军第十四集团军]]政治部主任。2015年3月，任[[成都军区|成都军区]]政治部副主任<ref>{{cite news|title=余琨、余永洪少将双双升任成都军区政治部副主任|url=http://www.thepaper.cn/newsDetail_forward_1318982|accessdate=2016-08-12|newspaper=澎湃新闻}}</ref>。2015年7月，任[[云南省军区|云南省军区]]政委<ref>{{cite news|title=成都军区政治部原副主任余永洪调任云南省军区政委，补缺石晓|url=http://www.thepaper.cn/newsDetail_forward_1358077|accessdate=2016-08-12|newspaper=澎湃新闻}}</ref>。2015年9月，任陆军第十四集团军政委<ref>{{cite news|title=云南省军区原政委余永洪少将任第14集团军政委，接棒黄集骧|url=http://www.thepaper.cn/newsDetail_forward_1380797_1|accessdate=2016-08-12|newspaper=澎湃新闻}}</ref>。2017年3月，任新组建的[[中国人民解放军陆军第七十九集团军|陆军第七十九集团军]]政委<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1735175 |title = 原第14集团军政委余永洪已任新组建的第79集团军政委 |author =  |date =2017-07-17  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。
