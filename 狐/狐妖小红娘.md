《'''狐妖小紅娘'''》（日語：{{lang|ja|縁結びの妖狐ちゃん}}，英語：Fox Spirit Matchmaker）是庹小新編繪，盤絲大仙上色作品。連載於[[騰訊動漫|騰訊動漫]]，現成為[[騰訊動漫|騰訊動漫]]獨家簽約漫畫，每週雙更。在2015年6月26日推出動畫，並在[[騰訊動漫|騰訊動漫]]第2屆[[PV|PV]]大賽中通過小組賽並在決賽中以35838票贏得第一，獲得長篇化名額。<ref>{{Cite news|url=http://ent.jschina.com.cn/system/2015/06/26/025250431.shtml|title=動畫版《花千骨》：《狐妖小紅娘》首播|language=zh-cn|publisher=中國江蘇網|author=|date=2015-06-26|accessdate=2015-07-24|deadurl=yes|archiveurl=https://web.archive.org/web/20150724082749/http://ent.jschina.com.cn/system/2015/06/26/025250431.shtml|archivedate=2015-07-24}}</ref>講述了以紅娘為職業的狐妖，在為前世戀人牽紅線的故事。<ref>{{Cite news|url=http://finance.ifeng.com/a/20150618/13786593_0.shtml|title=動畫電影《狐妖小紅娘》將上線 故事主軸忠於原著|language=zh-cn|publisher=鳳凰財經|author=|date=2015年6月18日|accessdate=2015-07-24}}</ref><ref>{{Cite news|url=http://ent.workercn.cn/338/201506/26/150626030304478.shtml|title=聲優張傑為《狐妖》配音|language=zh-cn|publisher=中工網|author=|date=2015-06-26|accessdate=2015-07-24}}</ref>2018年7月曾因[[bilibili|bilibili]]被[[央视|央视]]点名批评而暂时下架<ref>[https://www.ithome.com/html/it/371809.htm 求生欲强烈：央视点名批评B站含兄妹恋等低俗动漫 ，後者紧急下架]IT之家 2018-07-20[2018-07-23]</ref><ref>[http://www.expreview.com/62948.html 央视点名批评B站、动漫APP存在低俗内容後，大量动漫视频被下架]超能网 2018-07-21[2018-07-23]</ref>，經检查後重新上架。

==劇情簡介==
《狐妖小紅娘》的故事圍繞人與妖之間的愛情展開。根據古典小說記載，世上有人有妖，妖會與人相戀，妖壽命千萬年，人的壽命有限，人死了，妖活著。人會投胎轉世，但投胎以後，不記得上輩子的愛。妖如果癡情的話，就去找狐妖「購買」一項服務，讓投胎轉世的人，回憶起前世的愛。

狐妖紅娘（红线仙）這一個角色就這樣誕生，作品主要講述了以紅娘為職業的狐妖在為前世戀人牽紅線過程當中發生的一系列有趣、神秘的故事。

: 滅妖神火，500年前就以失传的灭妖神火純質陽炎，專門對付妖怪的神火。東方靈族是唯一可以使用純質陽炎的家族，在東方家破滅後，唯一會使用的人只有東方月初（王權富貴不會是因為沒看過秘笈）。東方月初曾在塗山廢棄倉庫裡留下一大團純質陽炎，只要簡單的召喚術，塗山狐妖就能召喚出那裡的純質陽炎。在百年後，白月初也能使用，因為其身體是傲來國用特殊方式製作的東方肉體血脈（純質陽炎會因為轉世而無法使用涂山案例是例外）。

; 虛空之淚
: 从外形上来看，虚空之泪和普通眼球并无差异，但虚空之泪包含着强大的能力，可以穿破一切物体或转移物体至虚空之境，在一地点放出，王富贵的爷爷会计算其落点，准确率百分之八点二。在五百年前东方月初临死之际曾将其作为与涂山红红“再世续缘”的关键道具。东方月初在世时，黑狐娘娘曾夺走一颗“虚空之泪”，剩下的一颗经过再世续缘，现在作为东方月初的转世─白月初的右眼而存在着。虛空之淚源於至高法力，卻生於至誠之情的法宝，但因為道法力高深者大多清心寡慾，故極難修煉。可切碎空间，触及之物必粉身碎骨，但若泪聚而不散，只会将物或人带到其他空間。目前所知唯一成功煉成的人只有東方月初。

; 再世續緣
: 又稱“轉世續緣”，塗山法術兼主打服務。人與妖共同向苦情樹起誓，以妖和人相愛的記憶和部分妖力為祭品，封入人或妖其中一者的法寶中，再借狐妖之力把此法寶一分為二，人類手持半個法寶死去，法寶就會隨人的靈魂一起輪迴轉世，轉世的人會帶著半個法寶出生，只要再借助狐妖的幫助（狐妖的道具及力量），讓人和妖恢復記憶。再世續緣的人每次轉世只能易姓不易名。

; 红线仙
: 涂山一族有一套完善的培训体系，帮助族人系统地学习红线仙所需要的知识。经过了培训后，新人红线仙就可以去专门的零散缘分任务发布处接任务了。而新人红线仙，需要连续成功两个零散缘分任务后，才能升职为正牌红线仙。红线仙依靠手中的天书，帮助再世續緣的人或妖找到对方，做任务的开销可以刷红线证報。敌人是厄喙獸和黑狐。

; 狐妖之力
: 狐妖之力源於至情，情之所至，力之所生，而因情所生的力量又怎么不会被情所伤。(塗山紅紅誤殺小道士後一位塗山狐妖向塗山雅雅做的解釋)

; 絕緣之爪
: 塗山紅紅(塗山蘇蘇)的雙手可以空手硬接所有法寶，是當時塗山紅紅（塗山蘇蘇）的法寶。

==手機遊戲==
由[[腾讯游戏|腾讯游戏]]的[[北極光工作室|北極光工作室]]自主研发，首款全[[三維空間|3D]]、国风、[[二次元|二次元]]题材的大型多人在线角色扮演移动网络游戏(回合制[[MMORPG|MMORPG]]手游)。游戏使用目前手机端主流的[[Unity3D|Unity3D]]引擎进行开发。以高规格的角色和场景建模，次世代材质渲染效果，带给玩家端游级别的画面感受。
===游戏特色===
*万物有界，爱恨无由——人妖相恋，千载姻缘由你来续！
*高度还原——全[[三維空間|3D]]还原动漫场景，你想要的必将实现！
*海量剧情——将狐妖小红娘的故事全都淋漓尽致的展现在你眼前！
*顶级配音——原班[[声优|声优]]倾情献声，演绎视听盛宴！
*唯美国风——汇聚强大的知名画师团队，打造高规格国风游戏！

====測試及發行延遲原因====
遊戲延遲發行主因為大陸遊戲版號停止審核及發放，下方分析僅與狐妖小紅娘相關部分，事發完整經過詳見[[2018年中国大陆游戏版号冻结|2018年中国大陆游戏版号冻结]]。
* 2018年3月29日：原[[广电总局|广电总局]]发布《[http://shouyoujz.com/archives/46261 游戏申报审批重要事项通知]》，称由于机构改革，所有游戏版号的发放全面暂停，且并未通知暂停期限<ref>[http://shouyoujz.com/archives/46261 原广电总局发布通知称游戏审批进度将推迟]手游矩阵 2018-03-30[2019-02-02]</ref><ref name="版号暂停">[https://new.qq.com/omn/20180919/20180919A0Z528.html 版号发放全面暂停，游戏寒冬持续蔓延]騰訊網 2018-09-19[2019-02-01]</ref><ref>[https://www.huxiu.com/article/257883.html 游戏审批大门已关，自求多福吧]虎嗅网 2018-08-16[2019-02-01]</ref><ref>[https://www.36kr.com/p/5148788 被版号掐住喉咙，腾讯游戏如何破局？]36氪 2018-08-13[2019-02-01]</ref>。此舉動導致[[騰訊遊戲|騰訊遊戲]]製作的多款手遊及網遊無法如期發行，申请版号通常需要至少半年的时间，重啟審查也無法立即改善即將發行而送審遭阻的遊戲所面臨的窘境。使狐妖小红娘手游原訂2018年3月上線被迫延期，但官方未公開點出真實延遲原因，轉而以"為將遊戲畫面品質再提升一個檔次"為由提出延期說明<ref name="1st延期-1"/><ref name="1st延期-2"/><ref name="1st延期-3"/>。
* 游戏版号暂停发放後有收到消息“3月版号全面暂停之后，我们得到的消息是很快会恢复。”<ref name="版号暂停"/>，因此2018年4月23日於[[北京|北京]]北京国家会议中心舉辦的“2018UP大会”中宣布手遊將於2018秋季正式上線<ref name="2018秋1"/><ref name="2018秋2"/>。
* 直到2018年8月15日開啟5th終極限量測試時仍未獲得游戏版号重新開放申請及審核的消息，[[騰訊遊戲|騰訊遊戲]]能做的只有等待並不斷延長本次測試時間，從原訂2個月(8/15~10/15)不斷延長，看來狐妖小红娘手游要在2018年秋季發行也宣告失敗，推出時間遙遙無期。為避免內部測試玩家於等待期間失去遊戲熱度，[[騰訊遊戲|騰訊遊戲]]參考玩家回饋內容後於2018年10月12日於遊戲內公告10月13日10時~15日11時30分停服升級，增加新副本、升級武器及伙伴培養方式、調整裝備技能。
