《'''狐狸新娘'''》是[[日本|日本]][[漫畫家|漫畫家]][[高木聰|高木聰]]筆下的一部[[漫畫|漫畫]]作品，於2008年5月至2011年8月連載於[[講談社|講談社]]的{{lang|ja|[[月刊少年ライバル|月刊少年ライバル]]}}雜誌。 

== 故事概要 ==
擁有不幸體質的主角－高中生葉山櫻介，某日在路上看到一座祭祀[[稻荷神|稻荷神]]的小廟，便對祭祀在那裡的稻荷神許下「希望能夠過著正常生活」的願望。沒想到回家之後，他卻看到一位可愛的女孩子在家裡等她。這位叫做京香的[[狐狸|狐狸]]公主，就是要嫁給他的妻子，從此櫻介的人生就有了巨大的變化……

== 主要出場人物 ==
; {{nihongo|葉山櫻介|葉山 桜介（はやま おうすけ）}}
:本作主角，擁有不幸體質的高中生。走在路上會被鳥屎滴到、踩到香蕉皮，走樓梯會從樓梯上摔下來，生活中完全沒有幸福或福氣的要素。目標是「希望能夠過著正常生活」，卻陰錯陽差的成為京香的未婚夫。
; {{nihongo|京香|京香（きょうか）}}
:嫁給櫻介的白狐狸公主，擁有能讓自己或別人變身；以及「狐火」等法術，外表可愛但很喜歡惡作劇。還是擁有一些身為狐狸的本性，例如喜歡吃[[油豆腐|油豆腐]]。嫁給櫻介後便轉學到櫻介的學校。
; {{nihongo|花音|花音（かのん）}}
:[[狸貓|狸貓]]女孩，自稱天才美少女，京香的死對頭。
; {{nihongo|京子|京子（きょうこ）}}
:京香的母親，興趣是賺錢。
; {{nihongo|大和太郎|大和 太郎（やまと たろう）}}
:櫻介就讀的高中學長及劍道社社長，個性認真。對京香一見鍾情。
; {{nihongo|秋野紅實|秋野 紅実（あきの くみ）}}
:櫻介與京香的同學，是擁有不幸體質而嘴巴惡毒的少女。
; {{nihongo|大和愛羅|大和 愛羅（やまと あいら）}}
:大和太郎的妹妹，是活潑且運動能力極佳的國二女生，似乎有些戀兄情節，後來相當在意櫻介。
