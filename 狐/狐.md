'''狐狸'''（学名：''{{lang|la|Vulpes}}'')，在动物分类学上，属于[[食肉目|食肉目]][[犬科|犬科]]，目前人工主要饲养的有[[银狐|银狐]]，为[[赤狐|赤狐]]的一个亚种。狐狸繁殖率高，抗病力强，食性杂，好饲养。毛长，耳尖，腿相对较短，吻独狭长，似体型中等而尾蓬松的狗。严格意义上指狐属的10个种（真狐），特别是旧大陆的赤狐和新大陆的美洲赤狐。

== 特征 ==
狐体长约70厘米，[[尾|尾]]长约45厘米（厘米=公分）。毛色变化很大，一般呈赤褐、黄褐、灰褐色，耳背黑色或黑褐色，尾尖白色。成年狐每年换毛1次，每年的3-4月份开始，7-8月全部脱完，新的针绒毛开始同时生长，11月份形成长而厚的被毛。

尾基部有一小孔，能分泌恶臭。

狐的体温在38.8-39.6℃，呼吸频率为每分钟21-30次。

蓝狐的寿命为8-10年，繁殖年限为4-5年，赤狐分别为10-14年和6-8年、银狐分别为10-12年和5-6年。

== 习性 ==
狐栖息森林、草原、半沙漠、[[丘陵|丘陵]]地带，居树洞或土穴中，傍晚外出觅食，天明始归。

狐在野生状态下主要以鱼、蚌、虾、蟹、蛆、鼠类、鸟类、昆虫类小型动物为食，有时也采食一些野果。

生殖期结成小群，其他时期单独生活。由於屬於犬科類，狐狸仍保有類似家犬的習性，多數的狐狸可以被人類馴服，僅有少數種類不行。

== 分类 ==
代表为赤狐，赤狐广泛被视为动物诡诈的象征，也是相当数量民间传说的主角。原产于旧大陆的种类实际上分布于整个欧洲，温带亚洲和北非；新大陆种类栖于墨西哥以北的北美大部分地区。赤狐具有长的针毛和柔软纤细的下层绒毛，通常体毛呈浓艳的红褐色，尾梢白色，耳和腿黑色。除了包括狐屬的动物外，狐有时也可以包括狐亞科其它屬的动物：
* [[大耳狐屬|大耳狐屬]] ''Otocyon'' - [[大耳狐|大耳狐]]
* [[灰狐屬|灰狐屬]] ''Urocyon'' - [[灰狐|灰狐]]、[[島嶼灰狐|島嶼灰狐]]
* [[狐屬|狐屬]] ''Vulpes'' - 11種“真正的”狐狸，包括[[赤狐|赤狐]]、[[北極狐|北極狐]]。

== 传说 ==
{{Commons and category|Vulpini|狐|Vulpini}}
在[[東亞|東亞]]古代[[传说|传说]]中，多年修炼的狐狸可以成[[狐狸精|狐狸精]]，变成人形，发人语，譬如章回小說《[[封神演義|封神演義]]》中迷惑紂王的[[妲己|妲己]]就是吸去人的魂魄，再由她的軀殼變成絕色美女。文言短篇小说集《[[聊斋志异|聊斋志异]]》很多篇目都与狐狸精有关。《[[山海经|山海經]]》中亦有記載[[九尾狐|九尾狐]]。中国在[[汉代|汉代]]后有将狐狸视作淫兽<ref>[https://www.hkcd.com/content/2016-03/03/content_988698.html 汉代后狐狸成为淫兽-香港商报]，羊城晚报</ref>。日本亦有[[玉藻前|玉藻前]]的[[白面金毛九尾狐|白面金毛九尾狐]]傳說，同時也因為是務農社會而狐狸會替農家抓捕破壞農作物的田鼠、野兔等，所以亦被視為[[稻荷神|稻荷神]]的使者。
