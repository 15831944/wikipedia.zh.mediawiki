'''陸儼少'''（{{bd|1909年||1993年||catIdx=陸}}），[[嘉定区|嘉定]]人，[[現代|現代]]著名[[國畫|國畫]]家，尤其以[[山水畫|山水畫]]見長，與[[李可染|李可染]]並稱「北李南陸」。小名驥，學名同祖，又名砥，字儼少；後以字名世，改字宛若。<ref name="簡歷">{{Cite web |url=http://art.chinanews.com/news/2012/1205/54.shtml |title=陸儼少 個人簡歷 |accessdate=2013-07-04 |date=2012-12-05 |work=中國新聞網 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20130417181259/http://art.chinanews.com/news/2012/1205/54.shtml |archivedate=2013-04-17 }}</ref>

== 個人經歷 ==
陸儼少1927年考入[[上海美术专科学校|無錫美專]]學習國畫，同時師從[[王同愈|王同愈]]學習詩文及[[書法|書法]]。次年，經王先生介紹，陸儼少正式拜在一代[[海派|海派]]宗師[[馮超然|馮超然]]門下，並且從[[吳湖帆|吳湖帆]]處得益良多。1947年舉辦個人畫展，聲名逐漸在江浙地區傳播。1956年進入[[上海中國畫院|上海中國畫院]]工作，次年成為「右派」，遭受迫害。1962年在[[浙江美院|浙江美院]]兼職任教，1979年升任教授。1981年在[[香港|香港]]開辦畫展，日漸擁有國際聲望。<ref name="簡歷" />

== 藝術風格 ==
陸儼少的[[山水畫|山水畫]]雲氣蒸騰，變化豐富，最為後人稱道。他獨創了將「留白」與「積墨」並用的技法，以墨來反襯白，對比十分強烈，在古典技法上融入了現代的審美意識。其中留白的部份可以是流水，也可以是雲霧或山路；畫面中經常並不能完全確定具體對象，留給觀者想像的空間。<ref name="石雲水">{{cite book |author=瀋明權 |title=陸儼少教授課徒稿：石、雲水法 |accessdate=2013-07-04 |edition=1 |series=中國美術學院教材 |date=2002-12-01 |publisher=中國美術學院出版社 |location=杭州 |language=zh |isbn=9787810831994 }}</ref>
