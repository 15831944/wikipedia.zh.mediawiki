'''陆忠伟'''（{{bd|1953年|10月||}}），[[上海|上海]]人，[[汉族|汉族]]，[[中华人民共和国|中华人民共和国]]政治人物、[[第十一届全国政协委员|第十一届全国政协委员]]。

== 簡歷 ==
文革期间，陆忠伟下放北大荒8510农场四年。 

1977年，毕业于[[黑龙江大学|黑龙江大学]]外语系日语专业，后供职于[[中国现代国家关系研究院|中国现代国家关系研究院]]。1985年，担任东亚研究室副主任。1992年，升任东亚研究室主任。1993年，升任副所长。1999年，担任所长。2003年，担任中国现代国际关系研究院院长。

2008年，当选[[第十一届全国政协委员|第十一届全国政协委员]]<ref>{{cite news |title = 第十一届全国政协委员名单 |url = http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html |accessdate = 2013-04-09 |work = 新华社北京1月25日电 中国人民政治协商会议第十一届全国委员会委员名单(共2237人，2008年1月25日政协第十届全国委员会常务委员会第二十次会议通过) |date = 2008年1月25日 |deadurl = yes |archiveurl = https://web.archive.org/web/20130928014403/http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html |archivedate = 2013年9月28日 }}</ref>，代表社会科学界，分入第三十二组。并担任外事委员会专委<ref>{{cite news |title=全国政协信息-陆忠伟 |url = http://www.cppcc.gov.cn/CMS/wylibary/showJcwyxtInfoWylibary.action?tabJcwyxt.guid=11W0011430&wytype=wjwy |accessdate=2013-04-11 |website = 全国政协 }}</ref>。2009年，两会期间建议中国关注粮食安全问题。

2011年，担任[[中华人民共和国国家安全部|中华人民共和国国家安全部]]副部长；同年，8月陪同[[王乐泉|王乐泉]]访问[[巴哈马|巴哈马]]。当年，其秘书李辉被国安部识破为[[美国中情局|美国中情局]]间谍<ref name="外參編輯部2012">{{cite journal |author = 外參編輯部 |journal =《[[外參|外參]]》 |issue = 第26期 |title = 諜案誤傷胡總親信 |url = http://books.google.com/books?id=vVEOBAAAQBAJ&pg=PA5 |date=2012-06-17 |publisher=外參雜誌社 |pages=5–6 |id = GGKEY:3A07PJ53Q3U }}</ref>。
