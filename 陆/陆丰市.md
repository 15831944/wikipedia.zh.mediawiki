'''陆丰市'''是[[中國|中國]][[广东省|广东省]][[汕尾市|汕尾市]]下辖的县级市，位于[[碣石湾|碣石湾]]畔，北接[[陆河县|陆河县]]、[[普宁市|普宁市]]，东接[[惠来县|惠来县]]，西接[[海丰县|海丰县]]和汕尾市城区，属海陆丰地区。

== 地理 ==
陆丰市北面和[[陆河县|陆河县]]、[[普宁市|普宁市]]交界；东与[[汕尾市|汕尾市]]华侨管理区及[[惠来县|惠来县]]接壤；西与[[海丰县|海丰县]]和[[汕尾市|汕尾市]]城区为邻；南濒南海，毗邻港澳，介于[[深圳|深圳]]与[[汕头|汕头]]两个经济特区之间。全市陆地面积1681km<sup>2</sup>，海岸线长116.5km，海域面积1.256万km<sup>2</sup>。南北向长，东西向短，全境周长372km。海岸曲折，港湾众多，有乌坎、甲子、碣石、湖东、金厢5个港口。

== 历史 ==
[[清朝|清]][[雍正|雍正]]九年四月己亥（1731年5月12日）自[[海豐縣|海豐縣]]析出，置陸豐縣。属[[惠州府|惠州府]]，[[民國|民國]]時期屬[[潮循道|潮循道]]，1958年屬[[汕頭|汕頭]]地區，1983属[[惠阳地区|惠阳地区]]，1988年改属[[汕尾|汕尾]]市。

== 語言與文化 ==

===語言===

陆丰的[[方言|方言]]，大部分地区通行的是[[陆丰话|陆丰话]]（福佬話），三甲地区[[潮州话|潮州话]]（相近[[惠來|惠來]]方言），以上均属[[闽南语|闽南语]]。陆丰的北部部分地区讲的是[[海陸客語|陆丰客家话]]、[[军话|军话]]等。其中，[[军话|军话]]是[[明代|明代]]初期“卫”、“所”军制的直接产物，有陆丰市[[西南镇|西南镇]][[青塘村|青塘村]][[良军村|良军村]]、[[大安镇|大安镇]][[坎石潭|坎石潭]]和[[海丰县|海丰县]][[平东镇|平东镇]][[龙吟塘|龙吟塘]]3个[[方言岛|方言岛]]，使用人口约9000人。

== 交通 ==
以[[海运|海运]]和[[公路|公路]]为主。

===铁路运输（[[陆丰站|陆丰站]]）===
陆丰站办理D、G字头列车的客运业务，其客运场具有3条到发线和2条通过线（正线）。然而，本站的客流量略少于同城的[[汕尾站|汕尾站]]。所有途经本站的车次之中，超过半数不停本站，其中来往广东省和上海市的高铁动卧也无一停靠。2017年4月16日起，深圳北站至南京站的D3126（上海局担当）增加停靠陆丰站，同时取消汕尾站客运业务，南京站也因此成为从陆丰站上车后能到达的最远车站。高峰期，深圳北站会开行终到陆丰站D9750（经停汕尾站）及其返程车次D9749（经停惠东站和惠州南站），广铁集团担当。

== 经济 ==
工业有机械、化工、水产加工等。

從1980年代到1990年代初期，陸豐由于周边城市如[[深圳|深圳]]在政策上的照顾，这段时期经济上也得到一定的发展，如电器产业、食品产业等各方面有比较好的知名度。1990年代开始，由于地方官员贪污、规划、治安等方面的问题，使发展前景不错的工厂都先后宣布倒闭，并失去以后的外来投资。

1990年代后期到2000年以后，陆丰很多重要产业链都断掉，造成大量失业现象，大部份本地年轻人都涌向周边城市寻求发展，如深圳、广州等城市。

== 治安 ==
陆丰在治安上有一定的問題，是[[广东省|广东省]]确定的全省八个社会治安重点整治地区之一<ref>[http://www.chinanews.com.cn/2000-08-22/26/42573.html 三千警力狂扫粤东陆丰黑窝点（中国新闻网）]（2009年8月）</ref>。

由於管理上存在[[漏洞|漏洞]]，當地曾有[[農民|農民]]的土地被人私自買賣，有人通過网上[[发帖|发帖]]的方式，希望能申訴自己的問題，並能得到重視<ref>[http://www.ce.cn/xwzx/gnsz/gdxw/200903/29/t20090329_18648193.shtml 广东农民指村干部"卖地" 悬赏百万惩办贪官(图)（北京青年报）]（2009年3月）</ref>。當地大部分市民對官員的治理不滿意，大部份認為因為官員裏有[[貪官|貪官]]，所以造成陸豐的經濟低迷，在网上，甚至还有“著名贪官”一说<ref>[http://www.hailufeng.net/html/22/n-1422.html 说“著名贪官”，很搞笑（海陆丰在线）] {{webarchive|url=https://web.archive.org/web/20100110092616/http://www.hailufeng.net/html/22/n-1422.html |date=2010-01-10 }}（2009年1月）</ref>。

陆丰是中国制贩毒严重的地区，2012年统计，陆丰市制造了全中国约1/3的[[冰毒|冰毒]]。<ref>{{cite web | language = zh-cn | publisher = 新浪 | title =陆丰毒情实录:84个制毒窝连成片 冰毒占全国1/3| url = http://gd.sina.com.cn/news/m/2014-01-03/075468981.html| date =  | accessdate = 2014-01-03 }}</ref>而廣東省其中一個蔡姓大毒梟及其餘有份參與製毒的蔡姓人在2013年12月29日被捕，而其所持有的製毒工場也隨後被瓦解。本次拘捕行動轟動全國，顯示中央打擊毒販的決心。
