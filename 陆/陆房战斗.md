'''陆房战斗'''是[[中国抗日战争|中国抗日战争]]中，中日双方的发生战斗<ref name=":0">{{Cite web|url=http://www.feicheng.gov.cn/about/contents/283/29694.html|title=陆房战斗遗址|accessdate=2016-06-15|author=|date=2013-08-05|publisher=}}</ref>。

== 背景 ==
1939年初，为开辟抗日根据地，八路军115师代师长[[陈光|陈光]]、政委[[罗荣桓|罗荣桓]]率东进支队挺进[[山东|山东]]，日伪军收到沉重打击。

四月底，驻山东日本侵华军第十二军司令官尾高龟藏，纠集近两个旅的主力和[[济南市|济南]]、[[泰安市|泰安]]、[[肥城市|肥城]]、[[宁阳县|宁阳]]、[[东平县|东平]]、[[长清区|长清]]、[[平阴县|平阴]]等十几个城镇的日军五千余人，分九路向泰肥山区合围<ref name="中国人民解放军全史">《中国人民解放军全史》军事历史研究部 编，军事科学出版社，2000年，ISBN 7-80137-315-4，卷“中国人民解放军战役战斗总览”</ref><ref name=":1">{{Cite book|title=桃乡颂歌|last=|first=|publisher=肥城市矿区教育办公室|year=|isbn=|location=|pages=}}</ref>。

== 经过 ==

1939年5月10日，日军合围圈缩至山東陆房山区，中国军队主力部队及党政机关三千余人被困。1939年5月11日拂晓，中国军队先在牛家庄一帶与日军遭遇，随后全线抗击。八六八团占领陆房以西肥猪山一线阵地，津浦支队占领陆房以东及东北之凤凰山阵地，师指挥所设在安家林。九时，日伪军在炮火掩护下发起全面进攻。中国军队连续击退敌人轮番攻击。经一天激战，日军伤亡1300余人，中国军队伤亡150余人。傍晚，中国军队趁夜幕在当地群众配合下分三路突围，津浦支队南过[[大汶河|大汶河]]，其余部队则先后到[[东平县|东平县]][[无盐村|无盐村]]一带修整。
