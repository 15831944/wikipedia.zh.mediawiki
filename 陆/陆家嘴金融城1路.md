'''上海公交陆家嘴金融城1路'''，是由[[浦东杨高|浦东杨高]]运营的一条专线线路，该线路为环线，由[[东金线轮渡|东昌路银城中路]]（东昌路渡口）到东昌路银城中路止，单圈长5.3[[公里|公里]]。

== 历史 ==
该线路为[[陆家嘴金融城|陆家嘴金融城]]白领通勤线路。开设该线路前，早在2010年4月中旬，陆家嘴金融城管委会完成了一份针对[[陆家嘴街道|小陆家嘴地区]]白领上下班公交需求的调研。在陆家嘴，高峰通勤需求人群为4.5万人次/小时，其中，轨道分担1.5万人次，常规公交分担1万人次，其他人群可能要步行“最后一公里”上下班<ref>{{cite web |title=陆家嘴3条巴士线串起商务楼 让白领上班路更从容 |url=http://news.163.com/10/1221/08/6ODQSL9800014AEE.html |website=网易新闻 |accessdate=2018-07-13}}</ref>。

2011年4月28日，陆家嘴金融城1路正式开通运营<ref name="wyxw" />。

2011年12月14日，该线路增加站点“陆家嘴环路花园石桥路”<ref>{{cite web |author1=钟晖 |title=明起浦东开通4条穿梭巴士 陆家嘴金融城1路增加站点 |url=http://news.163.com/11/1214/05/7L7A4DJE00014AEE.html |website=网易新闻 |accessdate=2018-07-13}}</ref>。

== 特点 ==
为方便乘客，该线路车厢和站点配备了陆家嘴金融城LED信息发布系统，方便乘客随时获取各类金融行情与新闻咨询。同时，站点Wi-Fi全覆盖，乘客可在站内无线上网。有别于浦东公交的蓝色涂装，陆家嘴金融城1路拥有独立标识，车体色彩为红色并辅助金色装饰，设计主题为“我爱金融城”，借鉴了伦敦红色巴士的设计。<ref name="wyxw" />

== 争议 ==
该线路早晚高峰间隔3分钟，低谷时段8至9分钟<ref name="wyxw" />。早高峰时段，大量市民换乘陆家嘴金融城1路上班，而线路间隔时间很长，乘客们只能排起长队<ref>{{cite web |title=陆家嘴的专线车让人等得好辛苦 |url=http://news.ifeng.com/gundong/detail_2014_02/22/34071682_0.shtml |website=news.ifeng.com |publisher=解放日报 |accessdate=2018-07-13}}</ref>。

==走向和停靠站点==
行驶路线自东昌路渡口站起，经花园石桥路、银城西路、银城北路、陆家嘴环路、银城中路、金洲街、东城路（折返）、陆家嘴环路、银城中路、东园路、银城北路、银城西路、拾步街、富城路至东昌路渡口终点站<ref name="wyxw">{{cite web |title=解决白领“最后一公里” 陆家嘴金融城1路今开通(组图)|url=http://news.163.com/11/0428/17/72OBL3B800014AEE.html |website=网易新闻 }}</ref>。

该线路停靠如下站点：东昌路渡口、花园石桥路银城中路、陆家嘴环路名商路、陆家嘴环路丰和路、陆家嘴环路东园路、金洲街银城中路、东城路陆家嘴环路、银城中路东园路、东园路银城中路、陆家嘴环路丰和路、陆家嘴环路名商路、陆家嘴环路花园石桥路、东昌路渡口<ref>{{Cite book|title=公共交通图|last=|first=|publisher=中华地图学社|year=2016|isbn=9787800319181|location=上海|pages=}}</ref>。
