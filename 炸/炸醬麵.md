'''炸醬麵'''{{efn|不可寫作「醡」醬麵}}，是指一種[[鲁菜|起源于山东的]]麵條食用方法，将煮熟的[[麵條|麵條]]和炸醬混合均匀食用。炸醬之所以為「炸」是因為鍋中「油」的量所定義的烹調用字，而在製作炸醬的最後階段，因為比重的關係，醬料都會沉在鍋底，上頭浮起一層的油，看上去就像把醬丟在油裏頭炸一樣，所以才會用「炸」作為動詞，稱為「炸醬」。<ref name=":0">{{Cite journal|url = |title = 美食節目主持人焦志方上菜 / 從小最愛傳家炸醬麵|journal = 台北畫刊|issue = 569期|doi = |others = |year = 2015/6|volume = |page = 第12~14頁|pmid = |author = 焦志方}}</ref>

炸醬一般用肉末（或肉丁、肉丝）和[[黄酱|黄酱]]（或[[甜面酱|甜面酱]]，[[晉語|晉語]]區多用[[老黑醬|老黑醬]]），加盐、糖或蜂蜜炒製的。可以配以[[鸡蛋丝|鸡蛋丝]]或其他配菜，称菜码或面码。一般有[[黄瓜|黄瓜]]絲、[[青豆|青豆]]、[[豆芽|豆芽]]、[[菠菜|菠菜]]等。

== 各地炸酱面  ==
炸酱面是中国[[山東|山東]]富有特色的食物。另外，[[北京|北京]]、[[天津|天津]]、[[上海|上海]]、[[辽宁|辽宁]]、[[吉林|吉林]]、[[臺灣|臺灣]]、[[廣東|廣東]]、[[香港|香港]]乃至[[日本|日本]]、[[韓國|韓國]]也有不同製法的炸醬麵。

=== 山东 ===
山东炸酱面特点是，炸酱里面除肉丁外，还会有豆角丁。做法是以葱薑爆锅后，下肉丁炸白，再加酱炒成炸酱，再加入事先炒好的豆角丁，一同成为炸酱。另外还有以鲜虾代替肉丁的鲜虾炸酱面。春季时令菜码有腌香椿芽。

=== 东北 ===
在辽宁和吉林，炸酱传统上使用[[大酱|大酱]]炒製。大酱和肉丝或肉末炒制成的炸酱叫肉酱，比较常见；另外还有素食版的鸡蛋酱，由大酱和炒[[鸡蛋|鸡蛋]]制成，为当地特色。

=== 北京 ===
北京炸酱面是由早期的[[魯菜|魯菜]]館帶入當地，后經傳播推廣后逐漸成為當地家喻戶曉的一道名吃。老北京炸醬麵的作法主要是由炸醬，面及菜碼等佐料拌勻而食，食感清爽可口濃郁不膩。北京的炸醬的做法是切小塊的五花肉，搭配以薑末爆炒，而後佐以乾黄酱、鲜黄酱、甜麵醬各三分之一调和，再加入蔥蒜炖製。北京炸酱面讲究菜码，在食用時除了將煮熟的手擀麵拌上炸醬外，也會佐以小黃瓜絲、蔥白、豆芽等等码料。也有人喜欢加入少許麵汤使面条拌起来更润些，並依個人喜好加入醋、麻油調味。整瓣的蒜头则是佐餐的调味品，最好的是独头蒜<ref>{{Cite web|url = http://blog.xuite.net/chiao939/10thfoods/213521416-5%E6%9C%8821%E6%97%A5%E6%9D%B1%E9%A2%A8%E6%96%B0%E6%96%99%E7%90%86%E7%BE%8E%E9%A3%9F%E7%8E%8B%E9%A3%9F%E8%AD%9C(1246%E9%9B%86)|title = 5月21日東風新料理美食王食譜(1246集) / 蔡季芳老師-老北京炸醬麵|accessdate = 2014-05-21|author = chiao939|date = 2014-05-21|publisher = }}</ref>。

=== 粤港 ===
港式炸醬麵是以辣[[豆瓣酱|豆瓣酱]]及[[蕃茄酱|蕃茄酱]]作主要醬料，紅蘿蔔絲及烏醋、糖和鹽熱煮，食用時搭配廣式生面及肉絲拌吃，味道較甜辣。<ref>[http://www.caipuwu.cn/tesecaipu/zhushi/02262XR009.html 港式炸醬麵的做法] {{webarchive|url=https://web.archive.org/web/20111012111403/http://caipuwu.cn/tesecaipu/zhushi/02262XR009.html |date=2011-10-12 }}</ref>

=== 臺灣 ===
[[File:Billyzhajiang1.jpg|thumb]]炸醬麵]]
台灣炸醬製作：先將五花肉切小丁塊，豆瓣醬及甜麵醬分別以水調致稀糊狀，起鍋熱油放入薑末爆香，豆干丁用小火煸得微微焦香，先加入豆瓣醬，以小火炒至香味飄出，再加入甜麵醬，續以小火翻炒至醬汁發亮。<ref name=":0" />搭配麵條有一般扁麵條，還有嚼勁十足的手工家常麵等，十足道地風味。另有知名維力食品的[[速食麵|速食麵]]維力炸醬麵，且陸續推出維力炸醬罐，深受台灣人喜愛。

=== 韓國 ===
{{main|韩式炸酱面}}
[[韓國|韓國]]亦有炸醬麵（{{lang-ko|자장면}}），早在清朝时期即由來自山東的[[華僑|華僑]]們自[[仁川|仁川]]傳入韓國。韓式炸醬麵以[[春醬|春醬]]（[[黑豆|黑豆]]醬）拌著醬油、砂糖與太白粉作為炸醬基底，酱质较为稀薄，放上黃瓜絲、佐洋蔥丁、醃黃蘿蔔及[[辣白菜|辣白菜]]一起食用。風味獨特，已經自成一格。<ref>{{Cite book|title = 韓國料理超簡單！人氣料理家高賢哲的韓式家常菜1．2．3|last = 高賢哲|first = |publisher = 積木|year = 2012-07-13|isbn = 9789866595813|location = 台灣|pages = 第87頁}}</ref>
韓式炸醬麵的由來


炸醬麵，是指一種起源於山東的麵條食用方法，將煮熟的麵條和炸醬混合均勻食用。炸醬之所以為「炸」是因為早期都使用豬油，買不到成形的豬肉。所以師父買回豬皮來炸油，鍋中「油」再來抄黒面醬。因為早期入境到韓國的僑民大部份由山東過去。在韓戰後，大部份僑民以開中華料理的小餐舘維生。當時與韓式细麺抗衡的就是炸醬麵， 後來也變成韓國的國民美食。幼年時，記憶中炸醬麵是一個銅板價，韓圓500.-，現代一碗炸醬麵是韓圓5,000。原文來自https://cristtravel.blogspot.com/2019/02/blog-post_23.html

=== 日本 ===
{{see also|ja:盛岡じゃじゃ麺}}
炸醬麵目前在[[日本|日本]]的高级餐館也有賣，口味是照搬台灣式的。樣子是: 在盤子周圍放上綠色[[黄瓜|黄瓜]]絲，中間放麵條，碟子放上黄色的炒[[鴨蛋|鴨蛋]]，最中央是一撮紫色的炸醬。

== 庶民文化 ==
[[File:Fried_sauce_noodles_in_Ganjiakou_(20140302144640).JPG|220px]]
老[[北京|北京]]也有顺口溜记述了炸醬麵的配料：
:青豆嘴儿、香椿芽儿，
:焯韭菜，切成段儿；
:芹菜末儿、莴笋片儿，
:狗牙蒜要掰两瓣儿；
:豆芽菜，去掉根儿，
:顶花带刺儿的黄瓜要切细丝儿；
:心里美，切几批儿，
:焯江豆剁碎丁儿，小水萝卜带绿缨儿；
:辣椒麻油淋一点儿，芥末泼到辣鼻眼儿。
:炸酱面虽只一小碗，七碟八碗是面码儿。
