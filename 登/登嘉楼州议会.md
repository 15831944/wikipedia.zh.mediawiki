'''登嘉楼州议会'''（{{Lang-ms|Dewan Undangan Negeri Terengganu}}）是[[马来西亚|马来西亚]][[登嘉樓|登嘉楼州]]的州立法机构，由32名议员组成[[一院制|一院制]]议会。每届州议会的任期为5年，届满后会自动解散，但州务大臣可以向拉惹建议提前解散州议会，解散后必须在60天内举行选举。州议会选举一般会和国会下议院选举同时进行。

州议会是在州首府[[瓜拉登嘉楼|瓜拉登嘉楼]]的达鲁伊曼大夏举行。

== 角色 ==
登嘉楼州议会的主要角色是颁布州法律和政策，并负责监督州政府的部门和机构。州议会议员负责检查和批评州政府的行政。<ref>{{Cite web|title=Peranan dan Keahlian Dewan Undangan Negeri|url=http://dun.terengganu.gov.my/maxc2020/agensi/article.php?cid=78&aid=6636|language=Malay|publisher=Terengganu State Assembly|accessdate=16 April 2011}}</ref>

掌握州议会多数议席的党或联盟领导人，由[[登嘉楼苏丹|登嘉楼]]苏丹任命为[[州务大臣|州务大臣]]，州务大臣再委任州行政议员。<ref>{{Cite web|title=Pengenalan|url=http://dun.terengganu.gov.my/maxc2020/agensi/article.php?cid=78&aid=6635|language=Malay|publisher=Terengganu State Assembly|accessdate=16 April 2011}}</ref>

== 委员会 ==
州议会由委员会来处理行政事项，包括委员会的特权<ref>{{Cite web|title=Jawatankuasa Hak and Kebebasan Dewan Undangan Negeri|url=http://dun.terengganu.gov.my/maxc2020/agensi/article.php?cid=78&aid=6641|language=Malay|publisher=Terengganu State Assembly|accessdate=16 April 2011}}</ref> 和公共账户委员会。<ref>{{Cite web|title=Jawatankuasa Kira-kira Wang Negeri|url=http://dun.terengganu.gov.my/maxc2020/agensi/article.php?cid=78&aid=6642|language=Malay|publisher=Terengganu State Assembly|accessdate=16 April 2011}}</ref>
