'''登封觀星台'''（{{lang-en|Dengfeng Observatory}}），也稱為'''告成天文台'''（{{lang-en|Gaocheng Astronomical Observatory}}），最早是[[周朝|周公]]建造的觀星台，位於中國[[河南省|河南省]][[登封|登封]]，鄰近[[告成縣|告成縣]]，是[[世界遺產|世界遺產]]之一。這個觀星台有悠久的歷史，從[[西周|西周]]開始到[[元朝|元朝]]的初期。元朝的[[天文學家|天文學家]][[郭守敬|郭守敬]]在AD1276年將它重建成一個巨大的天文台，是[[中国|中国]]现存最早的古代[[天文台|天文台]]<ref>{{cite web|url=http://news.qq.com/a/20100801/000513.htm|title=河南登封观星台见证嵩山“天下之中”|publisher=腾讯网|date=2010年8月1日| language = zh-hans }}</ref>。附近還有建於[[唐朝|唐朝]][[開元十一年|開元十一年]]（AD723年）的[[周公測景台|周公測景台]]。

== 西周 ==
據說[[周公|周公]]在這個地方樹立了[[圭表|圭表]]（使用陰影或[[晷針|晷針]]測量太陽位置），他在數學、天文與星占的成就，記載在[[周髀算經|周髀算經]]。

== 唐朝 ==
唐朝的天文學家[[一行|一行]]在國內各地建立了20餘個標準化的晷針來測量不同地點的[[均時差|均時差]]。後續的劉焯從AD604年起，在在現今的[[本初子午線|本初子午線]]以東114° 上，沿著子午線從中亞到越南建立的10個測站，在理想的球面上測量地球這個球體。其中一個測站就建在現今的告成鎮附近，而觀測的成果被用來創建[[大衍曆|大衍曆]]。

在觀星台的南方可以找到由一行建造的石龜圖，根據《[[周禮|周禮]]》的記載，這個地方是地球的中心。

== 元朝 ==
巨大的觀星台始建於1276年，是[[元朝|元朝]]初期的君王[[忽必烈|忽必烈]]命令[[郭守敬|郭守敬]]和[[王恂_(元朝)|王恂]]（1235年-1291年）建造的，用來觀測太陽和星星，以紀錄時間。
 
觀星台由青石和磚塊建造，分成兩個部分：主體和青石塊的[[圭表|圭]]，也稱為量天尺。主體是高9.46米的平台，加上建於其上兩間小閣樓的高度是12.62米，在兩間閣樓之間的空隙水平的安裝了一根晷針。兩間閣樓的一间放[[漏壶|漏壶]]，另一间放[[浑仪|浑仪]]。作為量天尺的圭長31.19米，寬0.53，由36塊方形的青石組成，並有兩條水道以校準它的水平。量天尺的位置常向正北，就如同子午線的方向。在測量時，晷針的投影落在量天尺上，使用安置在水道上移動的[[圭表#郭守敬的改良|景符]]（利用針孔成像使晷針影子清晰的儀器）測量投影的位置，可精確至2mm。在冬至，正午時晷針的影子落在靠近量天尺尾端的位置。通过测量一年当中影子长度的变化，可以确定一[[年|年]]的长度。<ref>{{cite web|url=http://news.qq.com/a/20100801/000513.htm|title=河南登封观星台见证嵩山“天下之中”|publisher=腾讯网|date=2010年8月1日| language =zh-hans }}</ref>

這種非常準確的觀測維新的[[授時曆|授時曆]]提供服務，從1281年起共使用了364年。它的[[回歸年|回歸年]]年長度是365日5小時49分20秒，與現行[[格里曆|格里曆]]的數值相同，但是早300年就得到這個數值。

在1787年，[[拉普拉斯|拉普拉斯]]利用這些測量資料來檢查黃赤交角和地球軌道離心率的變化。

它也是元朝初期建立的27座觀星台中的第一座。

== 現況 ==
观星台是由台身和量天尺组成的一座青砖石结构建筑，台身状如覆斗，其作用是 “昼参日影，夜观极星，以正朝夕”。观星台不仅保存了中国古代[[圭表|圭表]]测影的实物，也是自周公土圭测影以来测影技术发展的高峰，对中国天文史和建筑史都具重要研究价值。<ref>{{cite web|url=http://www.gmw.cn/content/2009-04/01/content_904249_5.htm|title=河南登封：展现中原文化无限魅力|publisher=光明网|date=2010年8月1日| language =zh-hans }}</ref>
