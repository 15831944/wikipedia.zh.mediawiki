'''登封市'''是[[中华人民共和国|中华人民共和国]][[河南省|河南省]][[郑州市|郑州市]]代管的一个[[县级市|县级市]]，登封之名由[[武周|武周]]女皇帝[[武则天|武则天]]所封。位于河南省中部偏西，地处[[嵩山|嵩山]]南麓，东北临[[省会|省会]]郑州，西北靠古都[[洛阳市|洛阳]]。辖域东西长约56公里，南北宽约35公里，面积1220平方公里。全市辖3个街道、8个镇、5个乡。人口65.52万人。主要方言是[[中原官话|中原官话]][[河南方言|河南方言]]。

登封市不仅境内的文物及古遗址遗迹不胜枚举，而且是[[少林功夫|少林功夫]]源起地少林寺的所在地，是中国著名的“文物之乡”和“武术之乡”<ref>{{cite web|url=http://www.hnsc.com.cn/news/2007/07/16/190157.htm|title=登封简介|publisher=人民网|date= 2007-07-16}}</ref>。近年又相继成为国家园林城市、国家卫生城市、中国县域旅游品牌百强市<ref>{{cite web|url=http://www.dahe.cn/xwzx/zt/hnzt/whdf/xgqk/t20070716_1038769.htm|title=奋力跨入百强　建设魅力登封|publisher=大河网2007年07月16日|date= | language =zh-hans }}</ref>和中国优秀旅游城市，拥有国家5A级旅游景区[[少林寺|少林寺]]、国家级重点风景名胜区嵩山以及世界文化遗产[[登封“天地之中”历史建筑群|登封“天地之中”历史建筑群]]<ref name="dfls"/>。

== 历史 ==
=== 古代史 ===
登封是[[中华文化|中华文化]]的主要发祥地之一，古代这里曾生活居住着[[共工氏|共工部落]]和[[四岳|四岳部落]]，经长期繁衍生息，其后代不断迁徙散居在中国各地。根据史料记载及考古印证，公元前2000多年前，夏禹王朝生产生活和政治活动的中心阳城就位于今日登封市[[告成镇|告成镇]]。西周时期，[[周公旦|周公旦]]在阳城立[[土圭|土圭]]、测[[日影|日影]]，制定[[历法|历法]]。秦代推行[[郡县制|郡县制]]，在这一地区设立[[阳城县_(秦朝)|阳城县]]，西汉武帝[[刘彻|刘彻]]游[[嵩山|嵩山]]，正式设立崇高县。隋代改为嵩阳县。696年，女皇[[武则天|武则天]]登嵩山、封中岳，改嵩阳县为登封县，改阳城县为告成县（寓意大功告成）。金代将登封、告成两县合并为登封县。<ref name="dfls">{{cite web|url=http://newpaper.dahe.cn/hnrb/html/2011-08/01/content_556023.htm|title=“天地之中”叫响全国 “文化圣山”走向世界|publisher=大河网|date=2011年8月1日| language = zh-hans }}</ref>

=== 近代史 ===
中华民国[[北洋政府|北洋政府]]时期，河南省是包括[[中国国民党|中国国民党]]在内的军阀内战北方主要战场，处于洛阳、郑州和许昌之间的登封屡遭战火之殃，造成土匪横行，于是地方豪绅自发组建武装。[[国共内战|国共内战]]时期，1948年年初，在共军3路大军的打击下，中国国民党军队在河南地区转入全面防御，3月初，[[中国人民解放军|中国人民解放军]]在登封、临汝、宜阳以南集结，至5月16日登封全境基本被中国共产党控制。1983年[[洛阳地区|洛阳地区]]所辖[[登封县|登封县]]划归[[郑州市|郑州市]]管辖。[[新安县|新安]]、[[孟津县|孟津]]、[[偃师县|偃师]]三县由洛阳地区划归洛阳市。1994年5月30日经国务院批准登封县撤县设市。

== 地理 ==
登封处于山地向平原的过渡地带，境内地形复杂，山地、丘陵、盆地、河谷平原兼有。中岳[[嵩山|嵩山]]是境内北部的主要山脉，分为太室、少室二山。太室山位于城北，环山36峰，其中峻极峰海拔1494米，为嵩山主峰。源于少室山的[[颖河|颖河]]为境内的主要河流，东经[[白沙水库|白沙水库]]，至安徽省的[[颍上县|颍上县]]流入淮河。境内南有[[箕山_(河南)|箕山]]和熊山山脉。

登封处在北半球暖温带，属大陆性气候，干旱天气常见，明清时期，大的旱灾曾发生过22次。降水多集中在每年的七八月份。无霜期238天，年均降水量614毫米。年平均日照时数2297小时，年均温度14.2℃。

== 经济 ==
登封市为较具实力的[[县级行政区|县（市）]]之一，其经济体系可概况为以文化旅游为主的现代服务业，以特色种植、养殖为主的现代都市农业，以装备和汽车零部件制造、新材料、生物医药为主的现代工业。综合竞争力曾在2009年位居[[中国百强县#县域经济论坛评价|全国县域经济百强县（市）的第六十四位]]，[[国内生产总值|经济总量]]位居河南省各县（市）[[河南省各县市国内生产总值列表|第九位]]，GDP总量为2,585,811万元（[[人民币汇率|折合378,541万美元]]），人均GDP为39,573元（合5,793美元）。<ref>人口与经济数据来源于《河南统计年鉴2010》</ref>

登封目前有3个经济发展聚集区，包括产业集聚区、文化旅游产业集聚区和农业产业集聚区，2个专业园区包括三里庄高新技术工业园区和循环经济工业园，还有22个乡镇特色产业园。2012年，装备及汽车零部件制造、新材料、生物医药产业实现产值121亿元；实现旅游总收入55亿元；农业总产值突破20亿元。 

== 交通 ==
登封向周边的郑州、洛阳、焦作、平顶山、许昌已形成“一小时快速交通圈”，[[207国道|207国道]]与217、316省道贯通全境，郑少洛、禹登高速公路纵横交错，汝州至登封高速、[[郑州新郑国际机场|郑州新郑国际机场]]至登封高速及郑州至登封快速通道正在建设。登封铁路与京广、陇海、[[焦枝铁路|焦枝铁路]]干线相连，形成了四通八达、方便快捷的交通网络。

== 名胜古迹 ==
{{see also|登封市各级文物保护单位列表}}
[[File:Zhongyuemiao.JPG|right]]]]
登封是[[华夏文明|华夏文明]]的重要发祥地之一，佛道儒三教荟萃，留下众多文物古迹。境内现存文物史迹1127处，其中国家级重点文物保护单位16处，省级16处，名列全国县级第一，有天下第一名刹[[少林寺|少林寺]]，中国现存最早的天文台[[登封观星台|登封观星台]]，中国现存最古老的佛教砖塔[[嵩岳寺塔|嵩岳寺塔]]，中国现存规模最大、数量最多的古塔建筑群[[少林寺塔林|少林寺塔林]]，宋代四大书院之一的[[嵩阳书院|嵩阳书院]]，中国最早的佛寺[[法王寺_(登封)|法王寺]]，五岳道教圣地[[中岳庙|中岳庙]]等。<ref>{{cite web|url=http://www.dengfeng365.com/gaik/|title=登封概况|publisher=登封在线|deadurl=yes|archiveurl=https://web.archive.org/web/20160304070612/http://www.dengfeng365.com/gaik/|archivedate=2016-03-04}}</ref>

中岳[[嵩山|嵩山]]，是国家级重点风景名胜区和国家级森林公园（国务院首批公布），有人文景观150处，自然景观30余处。1991年被[[联合国科教文组织|联合国科教文组织]]首轮通过为“世界地质遗产保护单位”。

[[登封“天地之中”历史建筑群|登封“天地之中”历史建筑群]]，2010年被列为[[世界文化遗产|世界文化遗产]]，古迹群包括[[太室阙|太室阙]]和[[中岳庙|中岳庙]]，[[少室闕|少室闕]]，[[啟母闕|啟母闕]]，[[嵩岳寺|嵩岳寺塔]]，[[少林寺|少林寺]]建築群（包括[[少林寺#建筑|常住院]]、[[少林寺塔林|塔林]]、[[初祖庵|初祖庵]]），[[會善寺|會善寺]]，[[嵩陽書院|嵩陽書院]]，[[登封觀星台|观星台]]。
