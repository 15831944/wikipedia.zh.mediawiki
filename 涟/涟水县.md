'''涟水县'''是[[中华人民共和国|中华人民共和国]][[江苏省|江苏省]][[淮安市|淮安市]]所辖的一个[[县|县]]。总面积1678平方公里，人口约120万。县政府驻[[涟城镇|涟城镇]]。

== 历史 ==
*[[夏朝|夏]]、[[商|商]]、[[周|周]]时期，涟水境为“[[夷|夷]]”聚居之地。
*[[File:涟水河畔_-_panoramio.jpg|缩略图]][[春秋|春秋]][[战国|战国]]时期，涟水境先属[[吴|吴]]，后属[[越|越]]，再属[[楚|楚]]。
*[[秦|秦]]时涟水境属[[东海郡|东海郡]]。
*[[汉|汉]]初，涟水境属[[韩信|韩信]]封地楚。
*[[汉武帝|汉武帝]][[元狩|元狩]]六年（公元前117）在涟水境置[[淮浦县|淮浦县]]，建淮浦城，涟水始设[[县|县]]。
*[[王莽|王莽]]时期改淮浦为淮敬。
*[[三国|三国]]时期淮浦属魏，隶属[[广陵郡|广陵郡]]。
*建武三年（495）并淮浦入襄贲。
*[[隋|隋]][[开皇|开皇]]五年（585）改襄贲为涟水县。这是涟水的始称。
*[[唐|唐]]武德四年（621）在涟水境置涟州，州治涟水城，辖涟水、金城二县。
*[[贞观|贞观]]元年（627）废涟州，并金城县，并划归[[泗州|泗州]]管辖。
*[[宋|宋]]太平兴国三年（978）涟水县升为涟水郡。
*[[元|元]]代划归山东路。
*[[明|明]][[洪武|洪武]]二年（1369）涟水境改称[[安东县|安东县]]，属[[淮安府|淮安府]]。
*[[清朝|清朝]]时期，涟水仍称[[安东县|安东县]]。
*[[中华民国|中华民国]]三年（1914），因与[[奉天省|奉天省]][[安东县|安东县]]同名，复名涟水县至今。

== 地理 ==
涟水县位于[[江苏省|江苏省]]北部，[[淮安市|淮安市]]东北部。东南隔[[古黄河|古黄河]]与[[盐城市|盐城市]][[阜宁县|阜宁县]]、[[淮安市|淮安市]][[淮安区|淮安区]]相望，东北毗邻[[盐城市|盐城市]][[响水县|响水县]]和[[滨海县|滨海县]]，西南与[[淮安市|淮安市]][[淮阴区|淮阴区]]接壤，西北毗邻[[宿迁市|宿迁市]][[沭阳县|沭阳县]]，北部与[[连云港市|连云港市]][[灌南县|灌南县]]相接。

涟水县在[[亚热带|亚热带]]和[[暖温带|暖温带]]气候的过渡地带上，月平均温度最低0度，年均降水量约1000毫米，气候四季分明。

涟水县河流、湖泊较多，城内涟漪湖、东湖、茵湖三湖相连，设[[五岛公园|五岛公园]]，为[[江苏省|江苏省]][[黄嘴白鹭|黄嘴白鹭]]自然保护区。

==行政区划==
涟水县辖17个镇、2个乡：[[涟城镇|涟城镇]]、[[高沟镇|高沟镇]]、[[唐集镇|唐集镇]]、[[保滩镇|保滩镇]]、[[大东镇|大东镇]]、[[五港镇|五港镇]]、[[梁岔镇|梁岔镇]]、[[石湖镇|石湖镇]]、[[朱码镇|朱码镇]]、[[岔庙镇|岔庙镇]]、[[东胡集镇|东胡集镇]]、[[南集镇|南集镇]]、[[义兴镇|义兴镇]]、[[成集镇|成集镇]]、[[红窑镇|红窑镇]]、[[陈师镇|陈师镇]]、前进镇、徐集乡、黄营乡。

== 经济 ==
涟水县是农业大县，[[全国商品粮生产基地|全国商品粮生产基地]]县之一。

在工业方面，酿酒业是涟水工业经济的支柱，上市公司、骨干企业为位于[[高沟镇_(涟水县)|高沟镇]]的[[江苏今世缘酒业有限公司|江苏今世缘酒业有限公司]]。

涟水亦是[[淮扬菜|淮扬菜]]的发源地之一，涟水的两大特产：[[鸡糕|鸡糕]]和[[高沟捆蹄|高沟捆蹄]]，制作方法独特，口味别致，深得当地人和他乡客的喜爱。

== 教育 ==
涟水历来有崇尚教育的传统，因此涟水人才辈出。

位于涟水的[[涟水中学|涟水中学]]为国家重点示范高中、[[郑梁梅高级中学|郑梁梅高级中学]]为省重点示范高中、[[炎黄职业技术学院|炎黄职业技术学院]]是全日制民办高等院校、涟水一中、涟水红日中学、涟水怀文学校、涟水金城外国语学校。

== 旅游 ==
能仁寺始建于唐初，距今至少有1400年历史；寺内妙通塔，始建于公元1023年，至今已有980多年历史。
涟水县五岛湖旅游风光带
今世缘中国缘文化主题公园位于本县高沟镇。
涟水百花园观光景区位于涟水县保滩镇。
