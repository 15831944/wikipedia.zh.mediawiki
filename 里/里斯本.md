'''里斯本'''（{{lang-pt|Lisboa}}），亦稱為'''葡京'''，是[[葡萄牙共和國|葡萄牙共和國]]的[[首都|首都]]和最大都市。其位置位于葡萄牙中南部[[大西洋|大西洋]]沿岸，城北为[[辛特拉|辛特拉山]]，城南为[[特茹河|特茹河]]出海口，与[[伦敦|伦敦]]、[[巴黎|巴黎]]、[[罗马|罗马]]等同为[[西欧|西欧]]历史最悠久的城市。市區面積84.6平方公里、人口547,631人（2011年）。包含[[衛星城|衛星城]]的[[都会区|都会区]]人口超過300萬，相當於葡萄牙人口的27%左右。公元前205年起为[[罗马人|罗马人]]统治，当时的统治者[[凯撒|凯撒]]把这个地区升格为市，并命名为{{lang-la|Felicitas Julia}}（意为“祝贺凯撒”）。1256年起正式成为[[葡萄牙王国|葡萄牙王国]]的首都，从此发展成为[[欧洲|欧洲]]和[[地中海|地中海]]一带重要的港口与贸易城市。地理大发现时代很多航海家都是由里斯本出发到世界不同的地方探险的，此后成为[[葡萄牙帝国|葡萄牙殖民帝国]]富甲一方的政治与商业中心，现今与[[波尔图|波尔图]]同为葡萄牙两大代表性城市。

== 歷史 ==
[[File:1755_Lisbon_earthquake.jpg|thumb]]]]
[[File:PacoRibeira-18thCentury.jpg|left]]
[[File:Lisbon_monument.jpg|left]]

自[[新石器時代|新石器時代]]開始，-{里}-斯本的地區已經有[[伊比利亞人|伊比利亞人]]居住，最早興建起來這座古城的是[[腓尼基人|腓尼基人]]。至公元前205年起為[[羅馬人|羅馬人]]統治，當時的統治者[[凱撒|凱撒]]把這個地區升格為市，並命名為「Felicitas Julia」（意為「祝賀凱撒」）。可是到公元5世紀起相繼被[[蠻族|蠻族]]（包括[[斯維比人|斯維比人]]、[[汪达尔人|汪达尔人]]和[[阿兰人|阿兰人]]）占領。

公元711年，来自北非的[[摩爾人|摩爾人]]入侵[[伊比利亚半岛|伊比利亚半岛]]，占领里斯本。信奉伊斯蘭教的摩爾人除了在市內興建了很多[[清真寺|清真寺]]外，還建了很多房屋和新的城牆。而在伊斯蘭的統治期間，城市的經濟獲得了長足的發展，成為這一帶重要的海運港口。與此同時，市內住了信奉不同宗教的人如[[基督徒|基督徒]]、[[穆斯林|穆斯林]]和[[猶太人|猶太人]]等。

1147年，葡萄牙国王[[阿方索一世_(葡萄牙)|阿方索一世]]率領的[[十字軍|十字軍]]擊敗摩爾人，使得-{里}-斯本重回基督徒的手中。1256年，-{里}-斯本正式成為葡萄牙的首都，從此就躍升成為歐洲和地中海一帶重要的貿易城市。在[[地理大發現|地理大發現]]時代，很多航海家如[[瓦斯科·達·伽馬|瓦斯科·達·伽馬]]都是由-{里}-斯本出發到世界不同的地方探險的，因此，到了16世紀可算是-{里}-斯本最輝煌的時期，大量黃金從當時葡萄牙的殖民地巴西運到-{里}-斯本，而這裡亦盛產優質的[[陶瓷|陶瓷製品]]，使得-{里}-斯本成為歐洲富甲一方的商業中心。（<small>詳見：[[葡萄牙殖民帝國|葡萄牙殖民帝國]]</small>）

1506年4月19日，里斯本发生一场针对犹太人的大屠杀，夺走了多达4000名占当时城市1/4人口的犹太人生命<ref>{{citation|contribution=A 500-Year-Old Memory – Another tragic date in Jewish history|author=Rabbi Jules Harlow|publisher=Comunidade Judaica Masorti – Lisboa|title=Jewish Week|year=2011|accessdate=2011-06-22|language=pt|location=Lisbon, Portugal}}</ref>。1531年1月26日，里斯本发生大地震<ref>Eva Missler: ''Lissabon.'' 2005, S. 28.</ref>。

1755年，-{里}-斯本發生了一場欧洲史上规模最大的[[里斯本大地震|九级强震]]，火灾和海啸造成九万人丧失，造成城市及港口的嚴重损毁，從此里斯本远洋贸易的中心地位不再，葡萄牙的國力也開始逐渐衰落。重建后的里斯本，除了加宽道路，新的{{le|庞巴尔式建筑|Pombaline style}}依地形高低排列，采用洛可可和新古典主义装饰，但予以简化、自成一格，并在建筑结构嵌入木桩支架防震。<ref>{{cite book |author1=細腿男 |title=里斯本，沒落的美感 |date=2018-06-28 |publisher=商周出版 |isbn=9789864774838 |page=012}}</ref>

在19世紀，[[拿破崙|拿破崙]]入侵-{里}-斯本，當時的皇室集體逃亡至[[巴西|巴西]]，而城市也受到一定程度的破壞。[[第二次世界大戰|第二次世界大戰]]期间，-{里}-斯本則是少數開放的大西洋港口城市。現在她是葡萄牙的政治、文化中心，第一大港與第一大城。

1974年里斯本爆发[[康乃馨革命|康乃馨革命]]，实现了葡萄牙的民主化<ref>{{cite book |author1=塞繆爾‧杭廷頓 |title=第三波：二十世紀末的民主化浪潮 |date=2014 |publisher=五南出版 |location=台北 |page=16}}</ref>。1983年，[[貝倫塔|貝倫塔]]及熱諾尼莫修道院（Mosteiro dos Jerónimos）成為[[世界文化遺產|世界文化遺產]]<ref>[http://whc.unesco.org/en/list/263/ Monastery of the Hieronymites and Tower of Belém in Lisbon] - 聯合國教科文組織 {{en|icon}}</ref>。

1998年，-{里}-斯本主辦[[世界博覽會|世界博覽會]]，主題為「海洋，未來的資產」，以配合-{里}-斯本海洋城市的特色。博覽會舉行共132天，總計一千萬人次參觀<ref>[http://expomuseum.com/1998/ ExpoMuseum / 1998 Lisbon] -ExpoMuseum {{en|icon}}</ref>。

==地理==
=== 概况 ===
里斯本位于欧洲大陆最西端，是濒大西洋、坐落于[[特茹河|特茹河]]河口的港城。市內的特茹河段因地形起伏而蜿蜒曲折，街市大多建在山丘之上。因市區內坡街遍佈，故有別名「七丘之城」。在坡道上下行走的市營電車和纜車成為里斯本的特色標誌。市區分為沿特茹河岸建的旧城，以及北部高地上的新城。
=== 氣候 ===
里斯本是典型的[[地中海氣候|地中海氣候]]。[[夏季|夏季]]晴熱而少雨，不時有自[[非洲|非洲]]来的[[熱浪|熱浪]]帶來近40度的高溫，白天乾燥而日烈，夜晚氣溫驟降而微寒，晝夜反差較大。[[冬季|冬季]]天氣多變，雨水豐盈，氣溫較歐洲大多數城市來得溫暖，偶而寒流來襲致成落雪。

===航空===
[[File:Aeroporto_Lisboa_-_panoramio.jpg|thumb]]
[[里斯本機場|里斯本機場]]有两条沥青跑道，可以满足如[[波音747|波音747]]类型的大型飞机的起降要求。機場距市區约7公里，到達需搭乘穿梭巴士、地铁或[[計程車|計程車]]，由市中心搭計程車前往約需10歐元。<ref>{{cite web |title=里斯本机场 |url=http://data.carnoc.com/corp/airport/lis.html |website=CARNOC |accessdate=2019-01-17}}</ref>

===輪渡===
特茹河北岸有四个轮渡码头，南面有6个轮渡码头。其中，海关码头(Cais da Alfandega)是很多轮船的始发站，而南面对江Barreiro码头则是通向葡萄牙南部的列车总站。除此之外，在Belem码头乘坐游轮航线可以沿特茹河北上去达伽玛大桥，西去卡斯卡伊斯，观赏两岸的风光。

== 人口資料 ==
[[File:TelhadosLisboa1.JPG|thumb]]
[[File:Lisbon_(Lisboa)_historic_elevator_Santa_Justa_Luca_Galuzzi_2006.jpg|thumb]]]]
-{里}-斯本市的人口大約是564,477人，人口密度為6,658人/ km²，而大-{里}-斯本地區的人口就高達2,800,000人<ref>[http://www.ine.pt/ Instituto Nacional de Estatística] – 葡萄牙國家統計局{{pt icon}}</ref>。大-{里}-斯本地區是歐洲其中一個人口增長得最快的地區，根據[[聯合國|聯合國]]的估計，到2050年該地區的人口會增至450萬{{Fact|time=2007-11-01T17:09:12Z|date=February 2007}}。此外，據葡萄牙報紙《[[快報|快報]]》Expresso對生活水平的統計，-{里}-斯本是全葡萄牙最適合居住的城市<ref>[http://semanal.expresso.clix.pt/imagens/ed1784/fotos/pdfs/IPDF-u0321.pdf ''Classificação Expresso das melhores cidades portuguesas para viver em 2007''] - Expresso</ref>。

== 經濟 ==
-{里}-斯本區是葡萄牙最富庶的地區，這地區的[[人均GDP|人均GDP]]遠高於[[歐盟|歐盟]]平均人均GDP的水平，這地區的GDP佔整個葡萄牙的45%。
另外這裏還號稱是西歐景色最優美的港口城市，故-{里}-斯本的經濟主要是由服務業支持。此外，也有很多大公司的總部設於大-{里}-斯本地區，而特茹河的南岸也是相當工業化的。

-{里}-斯本的股票交易所是[[泛歐股票交易所|泛歐股票交易所]]的成員之一，2014年上市公司有50家，其证券交易量占葡萄牙GDP的35%~40%。<ref>{{cite web |title=葡萄牙证券市场初探 |url=http://www.mofcom.gov.cn/article/i/dxfw/jlyd/201412/20141200845893.shtml |website=中华人民共和国商务部 |publisher=驻葡萄牙经商参处 |accessdate=2019-01-17 |date=2014-12-24}}</ref>
