'''里昂中法大学'''（Institut Franco-Chinois de Lyon）是位于[[法国|法国]][[里昂|里昂]]的一所[[大学|大学]]。

== 历史 ==
1921年7月8日在[[法国|法国]][[里昂|里昂]]—圣·伊雷内堡正式成立。[[吴敬恒|吴敬恒]]担任首任校长。学校的管理由中法双方共同负责。资金由法方和中方共同筹集。自1926年起，中法[[庚子赔款|庚子赔款]]部分资金开始资助该校。里昂中法大学是中国在海外设立的唯一一所大学类机构，曾为中国培养了大批优秀人才。

第一批127名中国[[留学生|留学生]]于1921年9月25日抵达里昂。学生人数在1936年之前一直处于稳定的水平。后因国际局势的变动，学校的招生工作也逐步停顿，1946年录取了最后一批学生，而后学校终止运作。1946年该大学停办。在里昂中法大学25年的历史中，官方登记在册的共计有473名学生。中国留学生虽然在该校注册，但学校实际上只是学生寄宿舍，学生则在里昂大学各学院或专门学校进行学习。<ref name="f">[http://xmwb.xinmin.cn/html/2015-03/15/content_31_1.htm 里昂中法大学话旧]</ref> 

在老校友的多方努力之下，中法大学终于在1980年2月25日重新成立。新的中法大学(中国方面称之为[[中法学院|中法学院]])，它挑选录取已经获得学位的中国青年来法国，完成一学年的高水平实习。

自1980年至1999年间，共计有197名[[奖学金|奖学金]]获得者来到里昂实习。

== 与中法大学的关系 ==
位于法国的里昂中法大学、比利时的晓露槐工业专修馆、位于巴黎的留学事务所以及华侨教育事务所。它们同是隶属于[[中法大学|中法大学]]海外部。

== 历史上的作用 ==
里昂中法大学在中法两国的教育、文化交流方面发挥了重要作用，它在促进法国学术文化在中国传播的同时，也向法国人民介绍了中国文化，加深了法国人民对中国的了解。

出名學生：[[常書鴻|常書鴻]]、[[戴望舒|戴望舒]]、[[敬隱漁|敬隱漁]]、[[林克明|林克明]]、[[潘玉良|潘玉良]]、[[蘇雪林|蘇雪林]]、[[張若名|張若名]]、[[朱锡侯|朱锡侯]]等。

[[周恩來|周恩來]]、[[鄧小平|鄧小平]]和[[陳毅|陳毅]]，雖然在法國有留學過，都不是里昂中法大學的學生。<ref name="r">[http://www.bm-lyon.fr/nos-blogs-et-sites-thematiques/le-fonds-chinois/bibliographies-et-ressources/bibliographies/bibliographies-sur-l-institut/article/liste-des-etudiants-inscrits-a-l 里昂中法大學的官方名單]</ref>
