'''里宾斯基五规则'''，有时简称'''五规则'''，是[[辉瑞|辉瑞]]公司资深[[药物化学|药物化学]]家{{en-link|克里斯多夫·里宾斯基|Christopher A. Lipinski}}在1997年提出的筛选[[类药分子|类药分子]]的五条基本法则，符合里宾斯基五规则的[[化合物|化合物]]会有更好的药代动力学性质，在生物体内代谢过程中会有更高的[[生物利用度|生物利用度]]，因而也更有可能成为口服药物。在[[药物|药物]]研发领域，里宾斯基五规则被用于对化合物库的初筛，以期摒除那些不适合成为药物的分子，缩小筛选的范围并降低药物研发成本。在长期的实践过程中，药物化学家们对里宾斯基五规则作出简化，形成“四规则”和“三规则”，但是四规则和三规则有时仍然被称作“五规则”，这里的五指的是各条规则的判别值均为5或5的整数倍。

== 五规则的内容和变体 ==
里宾斯基五规则的内容是：
# 化合物的[[分子量|分子量]]小于500[[原子质量单位|道尔顿]]
# 化合物结构中的氢键给体 (包括[[羟基|羟基]]、[[氨基|氨基]]等)的数量不超过5个
# 化合物中[[氢键|氢键]]受体的数量不超过10个
# 化合物的脂水[[分配系数|分配系数]]的对数值（logP）在-2到5之间
# 化合物中可旋转[[化学键|键]]的数量不超过10个

简化后的四规则去掉了关于可旋转键的数量限制；三规则进一步去掉了对氢键受体数量的限制。

一些药物化学家还仿照里宾斯基五规则提出了用于筛选[[类先导化合物|类先导化合物]]的五规则：
# 化合物的分子量在100至380道尔顿之间
# 化合物中有尽可能少的氢键受体
# 化合物中有尽可能少的[[芳香环|芳香环]]
# 化合物脂水分配系数的对数值在1到3之间
# 化合物能够与靶[[受体|受体]]相结合
