'''巩新亮'''{{BD|1987年|10月11日|||catIdx= G巩}}原名'''巩永华'''，出生于[[甘肃省|甘肃省]][[平凉市|平凉市]]，毕业于[[天星影视艺术学院|天星影视艺术学院]]，为著名的[[演员|演员]]、[[模特|模特]]和[[歌手|歌手]]。<ref name="ant " />

== 简介 ==
巩新亮为[[甘肃省|甘肃省]][[平凉市|平凉市]]人，早年曾在[[甘肃省|甘肃省]]联合中专[[万里中学|万里中学]]读幼师中专，<ref name="lion " /><ref name="ok ">{{cite web|url=http://ent.cqnews.net/mx/nd/200903/t20090330_3133626.htm|title=巩新亮：很早就认识王中军，我只是中专生|accessdate=2009年3月30日|publisher=华龙网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20110601211932/http://ent.cqnews.net/mx/nd/200903/t20090330_3133626.htm|archivedate=2011年6月1日}}</ref>
2003年8月，巩新亮在[[四川省|四川省]][[天星影视艺术学院|天星影视艺术学院]]影视表演大专班学影视专业。<ref name=" cat">{{cite web|url=http://www.hlgzs.com/v/p/7175-0-0.html|title=巩新亮被潜规则|accessdate=2012-03-20|publisher=华龙网|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120328181919/http://www.hlgzs.com/v/p/7175-0-0.html|archivedate=2012-03-28}}</ref><ref name="ok " />
16岁时，巩新亮只身闯荡[[北京|北京]]，在[[房地产|房地产]]公司当文员，20岁时担任公司的总裁助理。<ref name="lion ">{{cite web|url=http://bjyouth.ynet.com/article.jsp?oid=49945679|title=巩新亮承认我确实没读过大学|accessdate=2009-03-31|publisher=北青网|work=|language=zh-cn}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="ok " />
中间曾经参加选秀节目《绝对唱响》，为西安赛区八强。<ref name=" cat" /><ref name="ok " /><ref name="bed " />
2005年，巩新亮参加[[中国中央电视台|中国中央电视台]]《[[星光大道|星光大道]]》第十五期比赛，为三号选手。<ref name="lion " />
因参演《非诚勿扰》一蹴而红，出名后，她经常提及恩师[[冯小刚|冯小刚]]和[[葛优|葛优]]，“有今天的片约不断，我特别感谢恩师冯小刚、葛优！”<ref name="ant " />

== 个人生活 ==
2010年7月份，巩新亮在[[上海机场|上海机场]]被一个青年男子拥着，媒体曝光说是其恋人。<ref>{{cite web |url=http://ent.ifeng.com/photo/bigpicture/detail_2011_01/07/4080320_0.shtml |title= 巩新亮不爱富二代 被神秘年轻男蜜拥疑传新恋情|accessdate=2011-01-07  |publisher=凤凰网 |work= |language =zh-cn}}</ref>

2012年3月份，钢琴家[[郎朗|郎朗]]和巩新亮被曝光两人出入[[酒店|酒店]]，事后郎朗回应说：“我们是认识，但只见过两面而已。”<ref>{{cite web |url=http://news.sohu.com/20120318/n338087830.shtml |title=郎朗回应称与巩新亮“只是朋友” 网友批其炒作 |accessdate=2012年3月18日 |publisher=搜狐网 |work= |language =zh-cn}}</ref> 对于此事，[[搜狐网|搜狐网]]认为是她借助郎朗炒作自己。<ref name="bed " >{{cite web |url= http://astro.women.sohu.com/20120320/n338205485.shtml|title=巩新亮将借朗朗爆红 |accessdate=2012年3月20日 |publisher=搜狐网 |work= |language =zh-cn}}</ref>
