'''巩汉林'''{{bD|1957年|10月20日||}}，[[男|男]]，[[沈阳|沈阳]]人，毕业于[[沈阳师范学院|沈阳师范学院]]。[[中国民主促进会|中国民主促进会]]会员<ref>[http://ent.163.com/16/0509/11/BMKBOTT800031H2L.html#p=9HDPTG6O00AJ0003 相声演员于谦已低调加入中国国民党革命委员会]</ref>。中国著名[[相声|相声]]、[[影视|影视]]及[[小品演员|小品演员]]。<ref>{{cite web |url= http://ent.163.com/14/0129/07/9JO7SQJT00031GVS.html|title=小沈阳宋小宝上演"爷们"之战 潘长江巩汉林同台 |accessdate=2014 |publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://news.xinhuanet.com/ent/2014-01/29/c_126077225.htm |title=辽视春晚冯巩调侃“缝小缸” 潘长江巩汉林争黑马 |accessdate=2014 |publisher=新华网 |work= |language=zh-hans}}</ref>

== 生平 ==
巩汉林毕业于沈阳师范学院，原从事[[相声|相声]]创作，后师从著名相声[[表演|表演]][[艺术家|艺术家]][[唐杰忠|唐杰忠]]。1979年调入[[沈阳曲艺团|沈阳曲艺团]]。巩汉林表演过小品《[[如此竞争|如此竞争]]》、《[[对缝|对缝]]》、《[[打麻将|打麻将]]》、《[[如此包装|如此包装]]》、《[[鞋钉|鞋钉]]》、《[[打工奇遇|打工奇遇]]》、《[[老将出马|老将出马]]》、《[[街头奇遇|街头奇遇]]》、《[[都市外乡人|都市外乡人]]》。《如此竞争》是其在[[辽宁|辽宁]]地区的成名作，也是东北[[小品|小品]]具有里程碑意义的作品。《[[打麻将|打麻将]]》是其在央视春晚的处女作。曾经与[[赵本山|赵本山]]、[[潘长江|潘长江]]、[[岳红|岳红]]、[[赵丽蓉|赵丽蓉]]等合作。在巩汉林的搭档赵丽蓉去世之后，巩汉林多次同[[黄宏|黄宏]]一起表演小品。巩汉林同时表演过影视剧《[[百姓秀才官|百姓秀才官]]》、《[[火狐|火狐]]》、《[[乔迁之喜|乔迁之喜]]》、《[[而立之年|而立之年]]》等。

其夫人[[金珠_(演员)|金珠]]也是著名小品艺术家之一<ref>{{cite web |url=http://news.163.com/14/0218/01/9LB3J68C00014AED.html |title=巩汉林展现“巧夫”特质(图) |accessdate=2014 |publisher=网易 |work= |language=zh-hans}}</ref>，两人1983年结婚<ref>{{cite web|url=http://ent.sina.com.cn/s/m/2013-02-01/12473850337.shtml |title=巩汉林与夫人庆结婚30年 儿子曝光(图)|publisher=[[新浪|新浪]]|date=2013年2月1日|accessdate=2014年9月22日}}</ref>。
