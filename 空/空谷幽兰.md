'''空谷幽兰'''，[[美国|美国]]著名[[汉学家|汉学家]][[比尔·波特|比尔·波特]]介绍[[中国|中国]][[隐士|隐士]]的著作，[[明洁|明洁]]翻译。<ref name="fhwsm " >{{cite web |url=http://news.ifeng.com/gundong/detail_2011_04/28/6055346_0.shtml?_from_ralated |title=美国汉学家撰就中国隐士游记 将拍纪录片冲击国际大奖 |accessdate=2012年2月21日  |work=|publisher= 凤凰网|language=zh-hans}}</ref>

== 内容简介 ==
该书为美国汉学家比尔·波特在中国寻找隐士的记录书籍，主要围绕他在[[终南山|终南山]]寻找隐居的隐士而作。然后对中国的[[隐士文化|隐士文化]]做了自己的看法和评价。<ref name="fhwsm " />

== 作者简介 ==
比尔·波特，美国著名作家、汉译英翻译家和汉学家。其三大著作《[[寒山诗集|寒山诗集]]》，《[[石屋山居诗集|石屋山居诗集]]》和《[[菩提达摩禅法|菩提达摩禅法]]》扬名海内外。<ref name="fhwsm " />

== 影响 ==
该书出版之后，引起了海外学习和研究[[中华文化|中华文化]]的浪潮。<ref name="fhwsm " />

中国[[峨影集团|峨影集团]]和比尔·波特签订了影视出版合作协议，将把此书拍摄为电影。<ref name="fhwsm " />

== 评价 ==
作者比尔·波特说：“中国的隐士和美国的隐士不一样，美国隐士是为了自己在修行，而中国隐士修行的目的是为了帮助别人。”<ref name="fhwsm " />，“一般人轻易不宜去体验隐士生活，没有修行的基础一般人会承受不了，甚至可能会发疯，这就像要读[[博士学位|博士学位]]首先得大学毕业。”<ref name="fhwsm " />
