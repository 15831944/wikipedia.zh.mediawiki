'''太空也入樽'''（{{lang-en|Space Jam}}）由[[華納兄弟|華納兄弟]]於1996年推出，是一套混合動畫和真人演出的電影，由[[樂一通|樂一通]]（{{lang|en|Looney Tunes}}）的[[兔寶寶|兔寶寶]]和籃球明星[[麥可·喬丹|麥可·喬丹]]領銜主演。2016年，續集宣布開拍，由[[勒布朗·詹姆士|勒布朗·詹姆士]]主演，並由臺裔導演[[林詣彬|林詣彬]]指導。

== 故事 ==

故事從遙遠的太空中開始。一些名為{{lang|en|Nerdlucks}}的外星人要把樂一通朋友帶到他們的上司{{lang|en|Mr. Swackhammer}}手中，以增加其主題公園的營業額。眼見這些外星人的體型微小，領袖[[兔寶寶|兔寶寶]]便和他們協議一場籃球比賽，以身高力壓對手。

然而，這些外星人利用了特殊能力，偷去五位[[NBA|NBA]]明星（包括[[查爾斯·巴克利|巴克利]] ({{lang|en|Charles Barkley}})、[[派屈克·尤英|-{zh-hans:尤因;zh-hk:伊榮;zh-tw:尤英;}-]] ({{lang|en|Patrick Ewing}})、[[马格西·博格斯|-{zh-hans:波古斯; zh-hk:波古斯}-]] ({{lang|en|Muggsy Bogues}})、[[拉里·约翰逊_(1969年出生)|-{zh-hans:賴瑞·強森; zh-hk: 賴瑞·強森}-]] ({{lang|en|Larry Johnson}})和[[肖恩·布拉德利|-{zh-hans:肖恩·布萊德利; zh-hk:巴特利}-]] ({{lang|en|Shawn Bradley}})）的球技，從而變得壯大起來。樂一通朋友恐懼起來，便召來已轉職[[棒球|棒球]]的麥可·喬丹前來協助。在決賽中，雙方勢均力敵。最後，麥可·喬丹在樂一通朋友、舊隊友和家人的支持下，投入了關鍵性一球，使樂一通得到勝利。

而落敗的{{lang|en|Nerdlucks}}，就和{{lang|en|Mr. Swackhammer}}决裂，把他拋向太空，然後想加入[[樂一通|樂一通]]的行列。至於麥可·喬丹，決定重新接觸[[籃球|籃球]]，繼續其職業生涯。

== 外界反應 ==
許多有關本片的評論都較負面。首先，普遍評論都認為這套電影比起對前的'''[[威探闖通關|威探闖通關]]'''({{lang|en|Who Framed Roger Rabbit}})為遜色。另外，[[籃球|籃球]]迷都認為貶低體育和麥可·喬丹的價值與人品。他們亦覺得電影內容偏離現實和標準。

不過，亦有一些觀眾讚揚電影的視覺效果是當時的重要突破。而一些專業的評論家，如{{lang|en|Roger Ebert}}和{{lang|en|Leonard Maltin}}，都對電影持正面態度。另一方面，電影的原創音樂都在當時流行起來。當中最為熟悉的是[[I_Believe_I_Can_Fly|I Believe I Can Fly]]，其次就有{{lang|en|Fly Like an Eagle}}、{{lang|en|Hit 'em High}} 和{{lang|en|For You I Will}}等等。

而在電影播放以後，{{lang|en|Acclaim Entertainment}}亦推出此遊戲的彈珠球機和[[電子遊戲|電子遊戲]]。

== 特別事物 ==
# 一個全新角色[[蘿拉兔|蘿拉兔]]正式登場，身份是[[兔寶寶|兔寶寶]]的情人。
# 麥可·喬丹在電影中說他會在隊衣下穿著[[大學|大學]]的[[籃球|籃球]][[短褲|短褲]]。在現實中，-{zh-hant:麥可·喬丹; zh-cn:迈克尔·乔丹; zh-tw:麥可·喬登;}-亦會是這樣，純粹是一種好運象徵。
# 今次故事指出，樂一通朋友住在一個奇異的地底世界—「樂一通地帶」（{{lang|en|Looney Tune Land}}）中。而所有動畫亦是在當地獨立製作。
# 對華納的經典動畫來說，這是其第一套混合動畫和真人演出的電影。
