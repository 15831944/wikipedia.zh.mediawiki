'''空军新疆航空队纪念馆'''，位于[[新疆维吾尔自治区|新疆维吾尔自治区]][[乌鲁木齐市|乌鲁木齐市]][[天山区|天山区]]幸福路，是展示[[新疆航空队|新疆航空队]]历史的专题博物馆。

==简介==
2010年3月18日，[[中国人民解放军空军政治工作部|中国人民解放军空军政治部]]副主任[[林红松|林红松]]一行，在[[中国人民解放军兰州军区空军|中国人民解放军兰州军区空军]]政治部副主任[[卫明|卫明]]、[[中国人民解放军空军乌鲁木齐基地|中国人民解放军空军乌鲁木齐指挥所]]政委[[钟卫国|钟卫国]]等人陪同下，到乌鲁木齐市协调中国工农红军新疆航空队纪念馆建设问题，并同乌鲁木齐市领导座谈<ref>{{Cite web |url = http://news.163.com/10/0319/10/624PUBA3000146BC.html |title = 新疆航空队纪念馆将落户乌鲁木齐 |author =  |date = 2010-03-19 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

此后，由[[中国人民革命军事博物馆|中国人民革命军事博物馆]]负责设计和组织施工，在新疆航空队所在地原址兴建了空军新疆航空队纪念馆。纪念馆占地面积8亩，建筑面积777.22平方米<ref name=sanjia>{{Cite web |url =http://news.iyaxin.com/content/2013-05/27/content_4025908.htm  |title = 乌鲁木齐市幸福路空军新疆航空队纪念馆有三架退役战斗机 |author =  |date =2013-05-27  |publisher = 亚心网 |language =  |accessdate =  |quote =  }}</ref>。纪念馆坐落在原新疆航空队飞机场旧址幸福路小东门外机场<ref>{{Cite web |url = http://roll.sohu.com/20121220/n361048216.shtml |title =城迹：共和国雄鹰从幸福路起飞  |author =  |date = 2012-12-20 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

2011年11月10日，位于乌鲁木齐市幸福路的空军新疆航空队纪念馆正式向公众开放。馆楼前有“人民空军党缔造、人民空军忠于党”14个烫金大字，纪念馆以此为主题，分成“高瞻远瞩英明决策”、“红军骨干 蓝天英豪”、“风云突变 烈火真金”、“历尽艰险 回到延安”、“雄鹰展翅 壮志凌云”五部分，采用37本（篇）图书资料、1200多张图片、多件实物复制品，展示了[[第二次国共合作|第二次国共合作]]时期，[[中国共产党|中国共产党]]选派人员到[[盛世才|盛世才]]统治下的新疆加入新疆航空队，学习飞行和维护技术，为建立[[中国人民解放军空军|中国人民解放军空军]]奠定基础的历史<ref>{{Cite web |url = http://www.zjww.gov.cn/news/2011-11-14/406810446.shtml |title =空军新疆航空队纪念馆在乌鲁木齐开馆  |author =  |date = 2011-11-14 |publisher =浙江省文物局  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://kj.81.cn/content/2012-12/20/content_5151461.htm |title =空军新疆航空队纪念馆解说词  |author =  |date = 2012-12-20 |publisher =  中国空军网|language =  |accessdate =  |quote =  }}</ref>。

纪念馆开放后，在纪念馆院子的东北角和西北角分别停放有三架飞机：一架[[歼-5|歼-5]]战斗机、一架[[歼-6|歼-6]]战斗机、一架[[初教-6|初教-6]]教练机<ref name=sanjia/>。
