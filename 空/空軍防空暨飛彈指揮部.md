'''空軍防空暨飛彈指揮部'''（簡稱'''防空飛彈部'''、'''防空部'''或'''飛彈指揮部'''、'''飛指部'''）為空軍防空砲兵部隊之最高指揮機構，負責策劃與督導空軍防空飛彈砲兵部隊之兵力整建、部署、派遣、戰力整備、陣地配置、任務訓練等，以遂行防空作戰任務。隸屬[[國防部空軍司令部|國防部空軍司令部]]，指揮部位在[[臺南市|臺南市]][[仁德區|仁德區]]長安營區，現任指揮官為[[范大維|范大維]]中將。

== 歷史 ==
* 1959年8月16日，[[中華民國國防部|中華民國國防部]]因防衛[[臺澎金馬|臺澎金馬]]地區之重要，開始著手部屬[[美援|美援]][[勝利女神飛彈|勝利女神飛彈]]力士，編號SAM-A-25，後改為MIM-14，編成「陸軍力士飛彈第一營」，擔任臺北地區防空任務。<ref>[http://tw.myblog.yahoo.com/jw!LaA52ZWCAhjWHtqVC207BlN5YUejYHUS/article?mid=9666 防空飛彈部隊 改隸屬國防部「防空飛彈指揮部」]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
* 1960年12月6日，編成陸軍飛彈第二營（鷹式飛彈）<ref>[http://www.cna.com.tw/news/aipl/201202130094-1.aspx 防空飛彈部隊 改隸屬國防部]</ref>。
* 1961年8月1日，陸軍飛彈第1營改編為飛彈第661營，第2營改編為飛彈第662營。
* 1962年，成立陸軍防空飛彈605[[群_(軍隊)|群]]，下轄飛彈第661營、第662營、[[本部連|群部連]]、防空管制隊第1隊。
* 1969年12月29日，增編力士飛彈第663營及防空管制第2隊。
* 1976年2月1日，增編綜合保修第一、二廠提供飛彈系統裝備三級補保支援。同年7月1日，編成飛彈664營（南寧案）。
* 1978年5月1日，編成飛彈作戰管制組。
* 1979年3月1日，增編飛彈第665營（煉石案）。
* 1979年9月1日，陸軍防空飛彈605群奉令擴編為「'''陸軍防空飛彈指揮部'''」，下轄本部連、電子訓練中心、飛彈作戰管制組、靶勤隊，並奉令成立兩個飛彈群（606、608群）。
* 1981年7月，[[中山科學研究院|中山科學研究院]]成立「天弓計畫室」研發[[天弓飛彈|天弓飛彈]]；1989年9月測試完畢。
* 1982年9月16日，增編飛彈666營（天蠍案），擔任[[東臺灣|臺灣東部地區]]防空任務。同年11月24日，配合國軍兵力部署及陣地位置，奉令調整各飛彈營連番號與隸屬關係。
* 1993年9月30日，自行研發之[[天弓一型防空飛彈|天弓一型]]飛彈系統正式成軍部署，編成天弓飛彈營並擔任戰備。
* 1996年10月1日，向美國採購之[[愛國者飛彈系統|愛國者飛彈系統]]，編成愛國者飛彈營。10月28日，力士飛彈系統正式除役。
* 2004年4月1日，配合國軍「[[精進案|精進案]]」組織變革，納編海軍[[海鋒大隊|海鋒大隊]]編成「'''國防部參謀本部飛彈司令部'''」直屬[[國防部參謀本部|國防部參謀本部]]，擔任制海、防空之作戰任務。
* 2006年1月1日，將飛彈司令部所屬制海、防空部隊移回海軍、空軍，改編為「'''[[空軍防空砲兵司令部|空軍防空砲兵司令部]]'''」，下轄3個指揮部、2個飛彈群、1個訓練中心、17個防空砲兵營。<ref>[http://www.libertytimes.com.tw/2006/new/jun/23/today-p2.htm 《自由時報》2006年6月23日]</ref>
* 2006年3月1日，應[[s:國防部組織法_(民國91年)|國防組織法]]（[[精進案|精進案]]）規定，更銜為「'''空軍防空砲兵指揮部'''」。
* 2012年2月16日，配合國軍「[[精粹案|精粹案]]」組織變革，國防部從空軍（空軍防空砲兵指揮部）收回陸基中長程防空飛彈部隊的指揮權<ref>[http://www.nownews.com/2011/03/23/91-2699105.htm 空軍指揮權被拔？高華柱：愛國者、天弓、鷹式部隊將移轉]，今日新聞網，2011年3月23日]</ref>，以原「飛彈司令部」為母體，整併[[天弓飛彈|天弓飛彈]]、[[鷹式飛彈|鷹式飛彈]]及[[愛國者飛彈|愛國者飛彈]]等中、高防空飛彈部隊，編成「'''國防部參謀本部防空飛彈指揮部'''」。
* 2017年3月1日，單位自國防部參謀本部移編空軍司令部，更銜「'''空軍防空飛彈指揮部'''」。
* 2017年9月1日，為統一指揮體系「空軍防空飛彈指揮部」和「空軍防空砲兵指揮部」兩個「[[旅|旅]]」級單位併編升格為「[[軍_(軍隊)|軍]]」級單位，新單位名稱為「'''[[空軍防空暨飛彈指揮部|空軍防空暨飛彈指揮部]]'''」<ref>[http://news.ltn.com.tw/news/politics/breakingnews/2170443  防空飛彈指揮部 九月正式成軍]</ref>，指揮官改由[[中將|中將]]出任，2位副指揮官由[[少將|少將]]出任，下轄5個防空旅<ref>[http://news.ltn.com.tw/news/politics/paper/1082383  統一防空指揮權 飛指部改隸空軍]</ref>、1個防空管制中心和1個綜合保修總廠。防空旅旅長為上校編階，副旅長為中校編階，防空管制中心主任為上校編階，副主任為上校編階；綜合保修總廠廠長為上校編階，3個綜合保修廠廠長為中校編階；16個防空營，防空營營長為中校編階；50多個防空連，防空連連長為少校編階。

== 武器系統 ==
由防空暨飛彈指揮部管轄的[[中華民國國軍|中華民國國軍]]陸基防空戰力，以購自[[美國|美國]]的[[愛國者飛彈|愛國者飛彈]]以及我國[[中山科學研究院|中山科學研究院]]自主研製的[[天弓飛彈|天弓飛彈]]為骨幹，輔以較老舊的美製[[MIM-23鷹式飛彈|MIM-23鷹式飛彈]]，構成[[臺灣|臺灣]]本島及若干外島、離島的中低空防禦網，以輔助空軍的第一線攔截。此外，媒體<ref>[https://udn.com/news/story/10930/2693694 「特種飛彈」791旅曝光 雄二E部隊徽代表部隊特性]</ref>推測，其也負責操作對地攻擊的[[雄風二E巡弋飛彈|雄風二E巡弋飛彈]]。

== 部徽意涵 ==
[[File:Air_defence_missile_command.png|150px]]
一、飛鷹圖騰:鷹是精、氣、神的表徵，凸顯本部隊機警、迅速、主動、積極及勇猛的戰鬥精神，將給予來犯敵機致命的一擊，使其聞之喪膽。<br />
二、箭矢圖示:展現我防空部隊各型武器裝備「勇、猛、狠、準」之特性，以揚神箭之威，捍衛國家領空。<br />
三、環形標誌:代表整合及有效運用各種防空武器系統，象徵本部隊協調密切、合作團結、組織嚴密，戰鬥意志高昂，是一支戰無不勝、攻無不克的常勝部隊。<br />
四、外環黃邊:象徵我防空部隊淵遠流長，生生不息、屹立不搖、上下齊心、團結一致的忠勇建軍精神。<br />
五、紅底:代表我全體官兵朝氣蓬勃、犧牲奉獻、捍衛國家、忠貞不二與持續堅強的戰鬥意志。<br />

== 臂章意涵 ==
[[File:ROCAF_ADMC.jpg|thumb]]
一、外型為盾牌，代表飛指部精實壯大，所向披靡，足以殲滅犯敵，確保台、澎、金、馬地區安全。<br />
二、電子軌跡所形成的圓弧，象徵本部團結一致、紀律嚴明、精益求精、日新又新的現代化部隊。<br />
三、藍、白、紅之底色象徵中華民國國旗亦表徵我神箭(飛彈)部隊貫徹國軍忠誠、團結、風氣、效率的軍風<br />
四、箭矢代表飛彈具遠程飛行能力。<br />
五、閃電即表我飛彈部隊能以迅雷不及掩耳的速度及精準機動打擊力。
