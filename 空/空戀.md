{{日本作品|空戀|ソラコイ}}是Chelseasoft在2015年12月18日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]，也是Chelseasoft的第一部作品。<ref>{{cite web |title=Chelseasoftデビュー作『ソラコイ』は幼なじみの恋愛バトル♥ |url=https://web.archive.org/web/20150816004009/http://dengeki-hime.com:80/2015/08/07/3783/ |website=電撃姫.com |language=ja |date=2015-08-14}}</ref><ref>{{cite web |title=Chelseasoftデビュー作『ソラコイ』がマスターアップ！幼馴染vs幼馴染の恋愛バトルADV |url=http://moedigi.com/game/2015/11/chelseasoft-sorakoi-master-up/ |website=moedigi |language=ja |date=2015年11月27日}}</ref>

==故事==
拓斗和光兩人從小時候就開始一起生活，但是有天兩人遇到與光外貌相似的少女空，由於光與空的競爭讓拓斗的生活開始不平靜。

==角色==
;{{nihongo|拓斗|タクト}}
:本作的男主角，希森學園（{{lang|ja|希ノ森学園}}）二年級學生。夢想能成為電影導演而加入電影同好會。
;{{nihongo|光|ヒカリ|聲優：秋野花}}
:身高：155cm，三圍：B82/W52/H81，血型：O型
:拓斗的同學和青梅竹馬，電影同好會的成員。
;{{nihongo|空|ソラ|聲優：浜穂志香}}
:身高：156cm，三圍：B83/W52/H82，血型：O型
:自稱是光，擁有與光相似的面貌。在奈美的幫忙下與拓斗同班並加入電影同好會。
;{{nihongo|愛莉|アイリ|聲優：白月かなめ}}
:身高：152cm，三圍：B90/W50/H84，血型：A型
:拓斗的學妹，電影同好會的成員。
;{{nihongo|奈美|ナミ|聲優：雪村とあ}}
:身高：160cm，三圍：B80/W53/H83，血液型：B型
:拓斗的學姊，電影同好會的成員，學生宿舍「夕凪寮」的管理人。
