'''空军一号'''（'''Air Force One'''）是任何接載着[[美國總統|美國總統]]的空軍飛機的航空[[無線電台呼號|無線電台呼號]]。

== 空军一号简介 ==
目前美国总统最常用的是两架[[VC-25|VC-25]]A飞机，尾号SAM 28000和SAM 29000。其中一架为主用机，一架是备用机。这两架飞机进行过大规模改装且换用了[[波音747|波音747-400]]使用的通用电气[[通用电气CF6发动机|CF6-80C2B1]]发动机<ref>{{cite web |title=VC-25 Air Force One - Fact Sheet Display |url=http://www.af.mil/About-Us/Fact-Sheets/Display/Article/104588/vc-25-air-force-one}}</ref>的[[波音747|波音747-200B]]民航机，而且，这两架飞机是一模一样的。在SAM 28000和SAM 29000于1990年开始服役之前，空军一号是两架[[波音707|波音707-300]]飞机（VC-137-1），尾号26000和27000，这两架飞机分别从1962，1972年起开始服役。

== 服役歷史 ==
空軍一號起源於美國第32任總統[[富蘭克林·羅斯福|富蘭克林·羅斯福]]總統時期所搭乘的[[道格拉斯|道格拉斯]][[DC-4|DC-4]]。從此，往後的[[美國總統|美國總統]]皆有專機。1950年代，[[艾森豪|艾森豪]]總統搭乘專機出訪時遇到相同代號的客機在同一空域，搞得塔台人員無法分清真的總統專機，在此之後，總統專機便有新的代號：空軍一號(Air Force One)。
在[[艾森豪|艾森豪]]總統時期的專機則採用[[洛克希德星座|洛克希德星座]]作為專機，並增加空對地電話與空對地電傳打字機來提升空軍一號的通訊技術。
到了[[約翰·甘迺迪|約翰·甘迺迪]]總統任期時，總統專機進入噴射客機時代，在1962年10月，美國空軍購入[[波音|波音]][[C-137|C-137]]2架，並稱為SAM 26000以及SAM 27000，當時專機塗裝為紅色帶有金色的樣式，被[[甘迺迪|甘迺迪]]總統認為觀感上過於豪華，在其妻子[[賈桂琳·甘迺迪|賈桂琳·甘迺迪]]的建議下，請來[[傑蒙德‧洛威|傑蒙德‧洛威]]重新設計空軍一號的外部塗裝與內部裝飾。

其中SAM 26000服役時間為1962年至1998年，從[[甘迺迪|甘迺迪]]總統到[[柯林頓|柯林頓]]總統，其中1963年到德州達斯訪問時發生[[甘迺迪遇刺案|甘迺迪遇刺案]]，導致甘迺迪總統身亡，副總統[[詹森|詹森]]隨即在專機上宣誓就職並將甘迺迪總統的遺體運回[[華盛頓特區|華盛頓特區]]。另一架編號SAM 27000則在1972年12月由[[尼克森|尼克森]]總統使用，並用至1990年由兩架[[VC-25|VC-25]]專機取代，最後在1998年退役。

到了[[雷根|雷根]]總統時期，仍以[[波音707|波音707]]作為總統專機，但[[美國空軍|美國空軍]]在1985年發布採購新一代專機的申請書，並要求至少要有三台發動機以及能最少要續航9700公里的需求，[[波音|波音]]公司與[[麥道|麥道]]公司分別提出[[波音747|波音747]]以及[[DC-10|DC-10]]的方案，最後則採購兩架[[波音747|波音747]]來取代老化的[[波音707|波音707]]專機。兩架編號為SAM 28000和SAM 29000

到了2015年1月28日，[[美國空軍|美國空軍]]宣布由於[[VC-25|VC-25]]已經接近使用期限，決定採購[[波音747-8|波音747-8]]作為新一代空軍一號，預算約為40億美元(約新台幣1217億元，或人民币278亿元)。

== 安全 ==
作为美国总统专机，空军一号拥有先進的防护措施以保护总统的安全。该飞机足以抵御核弹爆炸后所产生的[[核辐射|核辐射]]及[[冲击波|冲击波]]。另外，空军一号还有先进的防干扰雷达、反[[地对空飛彈|地对空飛彈]]系统，如果飞机遭到飛彈袭击，飞机会发射红外線诱饵。該種誘餌在飞机周围空域飘荡，从而让[[红外線導引|红外線導引]]飛弹錯認目標。此外，任何人携带任何行李登机时，都要经过十分严格的安检，以确保飞机和总统的安全。空军一号上的总统套房、通讯中心以及驾驶室也是只有少数的特許人士才能够进入。

== 设施 ==
[[File:President_George_W._Bush_confers_with_White_House_Chief_of_Staff_Andrew_Card_aboard_Air_Force.jpg|thumb]]
[[File:VC-137-1_Air_Force_One.jpg|thumb]]
空军一号主要改装了飞机的内部装潢和设施，更适合总统在空中行使其职责。另外，空军一号还奉行“总统在白宫能做什么，空軍一號也亦然”的原则（除了發表演說和參與地面上的會議）。在机上，有两个现代化厨房可以烹饪料理，同时，为了保障食品安全，在起飞前一天晚上，食材通常在普通超市采购，空军一号上的餐厅能同时满足200人就餐。机上的医务室配备了医疗器械，机上的通讯中心也装备了较为先进通讯设备，如果美国一旦發生核战争，空军一号的通讯中心足以在空中对地面发出指令。空军一号上还有87部电话（其中28部防窃听），19台电视机。空軍一號也會載運總統座車直接送達目的地給總統乘坐。

除此之外，空军一号给总统留下了一些私人空间，机上的总统套房标配有一个[[席梦思|席梦思]]床和半圆办公桌及一台电视机。机上还配有会议室和休息室。

== 保养 ==
为了保证空军一号的飞行安全，一旦引擎出现磨损，[[马里兰州|马里兰州]]的[[安德鲁斯空军基地|安德鲁斯空军基地]]的总统飞行组就要把引擎等设备换一个新的，在起飞前又要把飞机全面检查一遍，空军一号的工程师几乎一天到晚都在忙，这种频繁的养护，成本相当高。

== 其他 ==
[[File:Lyndon_B._Johnson_taking_the_oath_of_office,_November_1963.jpg|thumb]]
[[File:Nixons_exit_AFO_in_China_1972.jpg|thumb]]期間，空軍一號SAM 26000在[[北京首都国际机场|北京首都国际机场]]]]
空军一号的所有飞行都属于空军的军事行动，由位于[[马里兰州|马里兰州]]的[[安德鲁斯空军基地|安德鲁斯空军基地]]的总统飞行组负责。从白宫到安德鲁空军基地总统一般先要搭乘[[美国海军陆战队|美国海军陆战队]]的专用直升机[[海军陆战队一号|海军陆战队一号]]。同样的，搭載总统的陆军交通工具称为[[陆军一号|陆军一号]]，搭載总统的海军交通工具称为[[海军一号|海军一号]]，而搭载总统的民航飞机则被称为[[行政一号|行政一号]]（Executive One）。

将总统的飞机称为“空军一号”是在[[艾森豪威尔|艾森豪威尔]]总统时代开始的，主要是为了安全原因。当年有一个拥有与总统飞机同样代号的民航飞机（代号：8610）正好与总统专机（代号：Air Force 8610）一起进入同一个空域，从此之后总统的飞机就有了一个特殊的名字，藉以和其他航班区别。

两架总统专机只有当总统在飞机上时（跨入机舱一刻起）才称为空军一号。当1974年[[尼克松|尼克松]]总统辞职，然后搭乘总统专机离开安德鲁空军基地时，飞机的代码被改成了飞机的尾号，只有当继任的[[傑拉爾德·福特|福特]]总统正式接任后，才恢复了空军一号的代号。

“空军一号”已经成为总统权力的象征，它伴随美国总统多次执行外交任务，并见证了许多历史性时刻：1963年11月22日，空军一号SAM 26000将总统[[约翰·肯尼迪|约翰·肯尼迪]]带到了[[达拉斯|达拉斯]]，肯尼迪却不幸[[肯尼迪遇刺|被暗杀]]，空军一号載著他的遺體返回華盛頓，副总统[[林登·约翰逊|约翰逊]]在機上宣誓继任美国总统；1972年，空军一号又陪伴了[[尼克松|尼克松]]总统完成了[[1972年尼克松访华|历史性的中国之行]]；2004年、2006年及2007年，在高度保密的狀態下，空軍一號搭載[[喬治·沃克·布殊|乔治·沃克·布希]]總統到[[伊拉克|伊拉克]]探望駐紮當地的美軍部隊。

空军一号机组乘务员有2组，每组26名，全部是军队编制。最高指挥官是[[机长|机长]]，[[军衔|军衔]]为[[上校|上校]]。

===九一一襲擊事件===
[[九一一襲擊事件|九一一襲擊事件]]中[[紐約|紐約]][[世界貿易中心|世界貿易中心]]南塔遭到襲擊後，[[乔治·沃克·布什|布希總統]]在[[佛羅里達州|佛羅里達州]]的薩拉索塔艾瑪 E.布克小學的訪問被打斷，搭乘空軍一號由薩拉索塔-布雷登頓國際機場起飛。在那之後布希搭乘空軍一號飛往了[[路易斯安那州|路易斯安那州]]和[[內布拉斯加州|內布拉斯加州]]的空軍基地躲避攻擊，接著才飛回[[華盛頓哥倫比亞特區|華盛頓哥倫比亞特區]]。

== 流行文化 ==
*1997年由[[哈里森·福特|哈里森·福特]]主演的一部動作電影《[[空軍一號_(電影)|空軍一號]]》，由于特勤内奸导致飞机被独裁领袖追随者劫持，最后福特所扮演的美国总统在斗智斗勇，清除了全部劫机者并成功被救，内奸和飞机一同坠毁。

*2004年在街機射擊遊戲《[[魅影小隊|魅影小隊]]》第二關，國際[[恐怖组织|恐怖组织]]“蒼之狼”[[劫机|劫持]]了從歐洲舉行會談回歸A國的空軍一號。玩家需要重新壓制全機，並且救出A國的總統。

*2013年美國電影《[[鋼鐵人3|-{zh-cn:钢铁侠; zh-hk:鐵甲奇俠; zh-tw:鋼鐵人;}-3]]》，反派奧德區穿上鋼鐵愛國者綁架空軍一號上的美國總統，最後空軍一號在空中起火爆炸。

*2013年美國電影《[[惊天危机|-{zh-cn:惊天危机; zh-hk:白宮末日; zh-tw:白宮末日;}-]]》，劇中副總統艾爾文·哈蒙德直接與自己的人馬坐空軍一號離開白宮。其後劇中總統詹姆斯·索耶被外界認定死亡（其實存活）時，機上的副總統艾爾文成爲新一任美國總統。但最後空軍一號被雇傭兵中的電腦駭客斯基普·泰勒以核飛彈控制系統的飛彈直接擊中墜毀。
