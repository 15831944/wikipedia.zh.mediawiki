'''空肠弯曲菌'''（'''Campylobacter jejuni'''）菌体轻度弯曲似逗点状，长1.5～5 μm,宽0.2～0.8 μm。菌体一端或两端有单[[鞭毛|鞭毛]]，运动活泼，在暗视野镜下观察似飞蝇。无[[荚膜|荚膜]]，不形成[[芽胞|芽胞]]。[[微需氧菌|微需氧菌]]，在含5% O2或10% CO2和85% N2的环境中生长最好。最适生长温度42℃。在正常大气或[[无氧|无氧]]环境中均不能生长。<ref>{{Cite journal|last=Crushell|first=Ellen|last2=Harty|first2=Sinead|last3=Sharif|first3=Farhana|last4=Bourke|first4=Billy|date=2004-01-01|title=Enteric Campylobacter: Purging Its Secrets?|url=http://www.nature.com/pr/journal/v55/n1/full/pr20042a.html|journal=Pediatric Research|language=en|volume=55|issue=1|pages=3–12|doi=10.1203/01.PDR.0000099794.06260.71|issn=0031-3998}}</ref>

== 简介 ==
本菌在普通[[培养基|培养基]]上难以生长，在凝固[[血清|血清]]和血[[琼脂培养基|琼脂培养基]]上培养36小时可见无色半透明毛玻璃样小[[菌落|菌落]]，单个菌落呈中心凸起，周边不规则，无[[溶血|溶血]]现象。空肠弯曲生化反应不活泼，不发酵[[糖类|糖类]]，不分解[[尿素|尿素]]，靛基质阴性。可还原[[硝酸盐|硝酸盐]]，氧化酶和过氧化氢酶为阳性。能产生微量或不产生[[硫化氢|硫化氢]]，[[甲基红|甲基红]]和vp试验阴性，[[枸椽酸盐|枸椽酸盐]]培养基中不生长,在弯曲菌中唯一马尿酸呈阳性反应。
空肠弯曲菌抵抗力不强，易被干燥、直射日光及弱消毒剂所杀灭，56℃5分钟可被杀死。对[[红霉素|红霉素]]、[[新霉素|新霉素]]、[[庆大霉素|庆大霉素]]、[[四环素|四环素]]、[[氯霉素|氯霉素]]、[[卡那霉素|卡那霉素]]等抗生素敏感。近年发现了不少[[耐药菌株|耐药菌株]],及多重耐药性菌株。
空肠弯曲菌抗原构造与肠道杆菌一样具有o、h和k[[抗原|抗原]]。根据o抗原，可把空肠弯曲菌分成45个以上血清型，第11、12和18血清型最为常见。空肠弯曲菌是多种动物如牛、羊、狗及禽类的正常寄居菌。在它们的生殖道或[[肠道|肠道]]有大量细菌，故可通过分娩或排泄物污染食物和饮水。人群普遍易感，5岁以下儿童的发病率最高，夏秋季多见。[[苍蝇|苍蝇]]亦起重要的媒介作用。亦可经接触感染。感染的产妇可在[[分娩|分娩]]时传染给[[胎儿|胎儿]]。
空肠弯曲菌有内毒素能侵袭小肠和[[大肠粘膜|大肠粘膜]]引起急性[[肠炎|肠炎]]，亦可引起[[腹泻|腹泻]]的暴发流行或集体[[食物中毒|食物中毒]]。[[潜伏期|潜伏期]]一般为3～5天，对人的致病部位是[[空肠|空肠]]、[[回肠|回肠]]及[[结肠|结肠]]。主要症状为痉挛性腹痛、腹泻、血便或果酱样便，量多；头痛、不适、发热。细菌有时可通过肠粘膜入血流引起[[败血症|败血症]]和其他脏器感染，如[[脑膜炎|脑膜炎]]、[[关节炎|关节炎]]、[[肾盂肾炎|肾盂肾炎]]等。孕妇感染本菌可导致[[流产|流产]]，[[早产|早产]]，而且可使新生儿受染。感染后能产生特异性血清[[抗体|抗体]]，可增强[[吞噬细胞|吞噬细胞]]功能。目前尚未测得肠道局部slga抗体。空肠弯曲菌对多种[[抗生素|抗生素]]敏感，常用红霉素、四环素治疗。<ref name=CDC>{{Cite web|url=https://www.cdc.gov/foodsafety/diseases/campylobacter/|title=Food Safety: Campylobacter|publisher=U.S. Centers for Disease Control and Prevention|access-date=2016-04-18}}</ref>
空肠弯曲菌是一种[[人畜共患病病|人畜共患病病]]原菌，可以引起人和动物发生多种疾病，并且是一种食物源性病原菌，被认为是引起全世界人类[[细菌性腹泻|细菌性腹泻]]的主要原因，对空肠弯曲茵的致病机理的研究越来越多。其致病因素包括粘附、侵袭、产生毒素和分子模拟机制等四个方面，通过分子模拟机制可以引起最严重的并发症一格林一巴利综合征。空肠弯曲茵可以通过产生细胞紧张性[[肠毒素|肠毒素]]、[[细胞毒素|细胞毒素]]和细胞致死性膨胀毒素而致病。

== 生化特性 ==
约有40%的菌株可以水解络蛋白、[[核糖核酸|核糖核酸]]和[[脱氧核糖核酸|脱氧核糖核酸]]，90%~95%的菌株具有[[碱性磷酸酶|碱性磷酸酶]]活性，6%的菌株芳香基硫酸脂酶阳性。在0.1%[[亚硒酸钠|亚硒酸钠]]斜面上生长，还原亚硒酸盐。水解[[吲哚酚|吲哚酚]]。在含有0.1%[[胆汁|胆汁]]或1.5%Nacl的培养基中可生长。

== 分离培养 ==
取服用抗生素前的腹泻[[粪便|粪便]]或[[宫颈|宫颈]]粘液等，短时间用棉拭子沾取样品将样品放如内加入[[万古霉素|万古霉素]]10mg,[[多粘菌素|多粘菌素]]B 2500iu，[[两性霉素B|两性霉素B]]13mg，[[头孢氨苄|头孢氨苄]]15mg的肉汤中进行增菌培养，后接种于哥伦比亚血(5%脱纤维绵羊血)平板上,培养24H观察,如果有杂菌则选取单个菌株纯化,后使用[[马尿酸|马尿酸]]和氧化酶检测,有条件可以做血清凝集试验(成本高).初筛将阳性以及疑似菌株进行[[多聚酶链式反应|PCR]]检测(16S,mapA,hipO)最终确定为空肠弯曲菌。<ref>{{cite journal|url=http://jcp.bmj.com/content/35/4/462.full.pdf|format=pdf|title=A selective medium for isolating Campylobacter jejuni/coli|journal=J Clin Pathol|year=1982|pages=462–476|volume=35|author=FJ Bolton|author2=L Robertson|doi=10.1136/jcp.35.4.462}}</ref>

== 菌种保藏 ==
在-20度时一般可以保存6个月,-70度冻干保存时间可以长达两年。

== 检测方法 ==
用于弯曲菌检测的经典方法常用的为[[生化鉴定|生化鉴定]]方法，但检测时间长、试剂成本高、尤其空肠弯曲菌的微需氧特性，使其分离培养更为困难，需要额外的维持微需氧条件的设施和试剂,近年来,PCR等分子生物学检测方法取得了长足发展,它具有敏感性高、特异性强、检测快速等优点。研究发现，通过对空肠弯曲菌及结肠弯曲菌16s [[rRNA|rRNA]][[基因|基因]]相同保守序列的扩增, 即可实现对样品中空肠弯曲菌及结肠弯曲菌的检测，经大量实验表明，该方法具有较强的特异性及灵敏性能够直接从鸡肉样品中直接检测出这两种弯曲菌,具有较强的实际应用价值。<ref name=sb0>[http://www.cnki.com.cn/Article/CJFDTotal-RDYZ200907008.htm 实时荧光PCR方法快速检测空肠弯曲菌方法的建立</span>]</ref>

== 检查 ==
发病一周后，血清内可出现[[抗体|抗体]]，主要为lgM，可用[[间接血凝试验|间接血凝试验]]及间接[[免疫荧光试验|免疫荧光试验]]等检测特异性抗体[[效价|效价]]，正常人或带菌者[[血清效价|血清效价]]可达1：2～1：8，急性期病人抗体效价可达1：8～1：32，恢复期可达1：80～1：320以上。由于血清抗体效价不高，须采取双份血清检测，以效价增高4倍作为诊断依据。
