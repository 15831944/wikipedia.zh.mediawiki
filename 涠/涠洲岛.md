'''涠洲岛'''是一座位于[[广西壮族自治区|广西壮族自治区]][[北海市|北海市]]南方[[北部湾|北部湾]]海域的[[海岛|海岛]]，是[[中国|中国]]最大、地质年龄最轻的[[火山岛|火山岛]]<ref>{{cite web |url=http://www.cigem.gov.cn/ReadNews.asp?NewsID=7840 |title=涠洲岛：中国最大最年轻的火山岛 |accessdate=2009-03-07 |work= |publisher=地质勘查导报 |date=2006年7月20日 |deadurl=yes |archiveurl=https://web.archive.org/web/20131203013821/http://www.cigem.gov.cn/ReadNews.asp?NewsID=7840 |archivedate=2013年12月3日 }}</ref>。该岛位于北部湾中部，北临广西北海市，东望[[雷州半岛|雷州半岛]]，东南与[[斜阳岛|斜阳岛]]毗邻，南与[[海南岛|海南岛]]隔海相望，西面面向[[越南|越南]]。

涠洲岛目前由北海市[[海城区|海城区]]管辖，设[[涠洲镇|涠洲镇]]。

== 历史 ==
[[File:Church_of_shengtang_village_Weizhou_Island.jpg|thumb]]]] 
[[新生代|新生代]][[第四纪|第四纪]]时期，[[火山|火山]]喷发，堆积形成涠洲岛<ref>{{cite journal zh|first=亓发庆|authorlink=|coauthors=|date=2003-01|month=|title=北部湾涠洲岛地貌的基本特征|journal=海洋科学进展|volume=21|issue=|pages=|id=|url=http://scholar.ilib.cn/A-hbhhy200301005.html|accessdate=2009-03-07|quote=}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

涠洲岛自[[汉朝|汉朝]]始，属[[合浦郡|合浦郡]]管辖。[[唐朝|唐]]、[[宋朝|宋]]两朝，归[[雷州|雷州]]巡检司管辖。[[元朝|元]][[元贞|元贞]]元年（1294年）设立涠洲巡检司。[[明朝|明]]初，涠洲巡检司属雷州府管辖。[[洪武|洪武]]七年（1374年），涠洲巡检司迁往雷州遂溪。[[万历|万历]]六年（1578年），自雷州移民于岛上垦耕。

[[清朝|清]][[康熙|康熙]]元年至[[嘉庆|嘉庆]]11年（1692年-1806年）涠洲居民三度[[迁界令|内迁]]，驻岛行政机构裁撤。[[同治|同治]]六年（1867年）重开岛禁，移雷州、廉州二州船户客民于岛上。同年起，[[法国|法国]][[天主教|天主教]][[传教士|传教士]]到涠洲岛传教。并于1880年和1882年，分别建成[[涠洲天主教堂|盛塘村天主教堂]]和[[涠洲城仔教堂|城仔村天主圣母教堂]]。[[光绪|光绪]]二十年（1894年），涠洲岛划归[[合浦县|合浦县]]管辖。

[[民国|民国]]初至1931年，先后属合浦靖海团和合浦第二自治区辖镇。1938年9月至1945年6月，[[日军|日军]]占领涠洲岛，并在岛上修筑有[[机场|机场]]。 

1950年3月6日，國軍主動撤出，[[中国人民解放军|中国人民解放军]]占领涠洲岛，设涠洲乡，隶属于合浦县。1950年8月，涠洲乡划归北海镇管辖，并改设涠洲镇。1993年7月8日<ref>{{cite web |url=http://www.beihai.gov.cn/81/2008_3_18/81_55826_1205830777234.html |title= 北海市政府门户网站：涠洲岛优惠政策|accessdate=2009-03-07 |work= |publisher= |date=1993年7月8日 }}</ref>，涠洲岛被辟为广西壮族自治区区级旅游度假区。2004年3月19日，广西北海涠洲岛火山国家地质公园成立，为第三批[[中国国家地质公园|中国国家地质公园]]<ref>{{cite web|url=http://www.jingtai.gov.cn/tour/ShowArticle.asp?ArticleID=422|title=第三批中国国家地质公园一览表|accessdate=2009-03-07|work=|publisher=|date=|archive-url=https://web.archive.org/web/20180125015516/http://www.jingtai.gov.cn/tour/ShowArticle.asp?ArticleID=422|archive-date=2018-01-25|dead-url=yes}}</ref><ref>{{cite web |url=http://www.southcn.com/news/china/zgkx/200403190682.htm |title= 中国公布第三批国家地质公园 |accessdate=2009-03-07 |work= |publisher=中新社 |date=2004年3月19日}}</ref>。2005年，涠洲岛被《[[中国国家地理杂志|中国国家地理杂志]]》“选美中国”特辑评为“中国十大最美丽海岛”之一，名列第二<ref>{{cite web |url=http://travel.people.com.cn/GB/41636/41637/73953/73961/5036451.html |title= 普陀山入选中国最美十大海岛|accessdate=2009-03-07 |work= |publisher=人民网旅游频道 |date=2006年11月14日 }}</ref>。

== 地理 ==
[[File:Stele_park_weizhou_island.JPG|thumb]]
[[File:Weizhou_island_rock_of_pig_and_xieyang_island.jpg|thumb]]
涠洲岛距[[北海市|北海市]]市区36[[海里|海里]]，距[[斜阳岛|斜阳岛]]9海里。岛屿南北方向长6.5[[千米|千米]]，东西方向宽6千米，总面积24.74[[平方千米|平方千米]]，[[海岸线|海岸线]]全长36千米，南面海岸最高海拔79米。地势由南向北倾斜，岛屿北部为熔岩沉积形成的[[丘陵|丘陵]]地带，南部为古火山口形成的[[港湾|港湾]]，港湾沿岸有岛上最大的居民聚居点涠州镇。全岛无河流，岛屿西北部的西角[[水库|水库]]是主要淡水来源。沿岛屿海岸线分布有[[火山|火山]]地貌 (有“[[蓬莱岛|蓬莱岛]]”之称)、[[海蚀|海蚀]]地貌、古[[地震|地震]]遗迹，古海洋风暴遗迹等，岛周边海域生长有大片[[珊瑚礁|珊瑚礁]]。

岛上主要植物有[[仙人掌|仙人掌]]、[[木麻黄|木麻黄]]、[[台湾相思树|台湾相思树]]、[[菠萝蜜|菠萝蜜]]、[[银合欢|银合欢]]和[[桉类|桉类]]等，全岛目前森林面积为4.31平方公里，主要为[[沿海防护林|沿海防护林]]<ref name="baohuqu" />。

岛上自然风光景点有猪仔石、火山公园、芝麻滩、地质博物馆等，古建筑有三婆庙、圣母教堂、法国天主教堂等。

== 气候 ==
涠洲岛属[[亚热带|亚热带]][[季风|季风]]气候，年平均[[气温|气温]]23[[℃|℃]]，绝对低温2.9℃，沿岸海水年均水温24.6℃，年平均[[降水量|降水量]]1297.2毫米<ref name="baohuqu" />。

== 居民 ==
涠洲岛人口为1.6万余人，2000多户。

== 旅游业 ==
涠洲岛2007年接待游客数目超过10万人，超过20%的岛上居民从事与[[旅游业|旅游业]]相关的工作。旅游业促进了当地经济的发展，居民人均纯收入从2005年的2577元增长到2007年的3317元<ref>{{cite web |url=http://www.xinbbw.com/?action-viewnews-itemid-1945 |title=北海涠洲岛：旅游业使中国最年轻火山岛走向富裕 |accessdate=2009-03-07 |work= |publisher=新华网 |date=2008年8月21日 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 交通 ==
岛上有环岛公路和多条乡村公路，环岛公路全长13.6km。道路较狭窄，主要交通工具为三轮[[摩托车|摩托车]]、[[自行车|自行车]]、微型[[面包车|面包车]]。

涠洲岛到大陆的交通运输主要依靠[[海运|海运]]，旧港口位于岛南部港湾内，2008年6月在岛西侧新修建了[[客运滚装船|客运滚装船]]码头<ref>{{cite web |url=http://www.gx.xinhuanet.com/misc/2008-06/27/content_13662636.htm |title=涠洲岛客货码头开通 |accessdate=2009-03-07 |work= |publisher=南国早报 |date=2008年6月27日 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304133809/http://www.gx.xinhuanet.com/misc/2008-06/27/content_13662636.htm |archivedate=2016年3月4日 }}</ref>。涠洲岛到北海市航线有数艘[[渡轮|渡轮]]往返，包括：飞逸号、飞鱼号、飞达号、飞跃168号。航行时间1个半小时到三个小时不等。

== 石油和天然气开采 ==
南海西部及涠洲岛海域目前已探明[[天然气|天然气]]储量约260亿[[立方米|立方米]]<ref name="tianranqi">{{cite web |url=http://news.cnpc.com.cn/system/2008/12/11/001213280.shtml |title=北海建成广西最大清洁能源生产基地 |accessdate=2009-03-07 |work= |publisher=广西日报 |date=2008年12月9日 |deadurl=yes |archiveurl=https://web.archive.org/web/20131205002217/http://news.cnpc.com.cn/system/2008/12/11/001213280.shtml |archivedate=2013年12月5日 }}</ref>，[[中国海洋石油有限公司|中国海洋石油有限公司]]下辖的南海西部石油公司在涠洲岛西南海域建设有多座[[石油钻井平台|石油钻井平台]]。1998年，在岛上建设了油气终端处理厂，目前正在建设30万吨级原油码头。

为解决中海油油气终端处理厂生产过程中排放的废气，由[[新奥集团|新奥集团]]投资在涠洲岛北部建设的燃气液化[[天然气|天然气]]工程项目，目前已完成两期工程。完全建成后日处理天然气能力将达60万立方米<ref name="tianranqi" />。除供应岛上居民外，天然气还输送到大陆其他城市。液化气的使用改变了岛民砍树烧柴的习惯，对岛上植被起到了保护作用。

2008年8月，涠洲岛周边海域和临近的[[北海银滩|北海银滩]]发生溢油事故，造成沿岸沙滩大面积污染。污染源头尚未查清，可能是输油管道泄露或原油自然泄露<ref>{{cite web |url=http://news.xinhuanet.com/mrdx/2008-09/06/content_9806989.htm |title=溢油污染涠洲岛，警钟敲向北部湾 |accessdate=2009-03-07 |work= |publisher=新华每日电讯  |date=2008年9月6日 }}</ref>。
