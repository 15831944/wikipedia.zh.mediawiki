'''劉慕裳'''（{{lang-en|'''LAU Mo Sheung, Grace'''}}，{{bd|1991年|10月19日||}}），[[香港|香港]]女子[[空手道|空手道]]運動員。女子個人形世界排名第三位（2019年1月）<ref>{{en}}{{cite web |title=WKF CURRENT RANKING - Female Kata |url=https://setopen.sportdata.org/wkfranking/ranking_main.php#ranking |website=sportdata.org |publisher=WKF}}</ref>。其兄[[劉知名|劉知名]]同為香港空手道運動員。

== 簡歷 ==
劉慕裳年幼時在哥哥[[劉知名|劉知名]]的薰陶下學習空手道，因為她覺得空手道的拳姿很有型<ref>{{Cite web|url=https://www.hk01.com/%E9%AB%94%E8%82%B2/9993/-%E5%A5%B9%E8%AA%AA-%E5%8A%89%E6%85%95%E8%A3%B3-%E7%A9%BA%E6%89%8B%E9%81%93%E5%A5%B3%E7%94%9F%E7%9A%84%E7%8B%A0%E6%BA%96%E8%88%87%E7%94%9C%E7%BE%8E-|title=【她說．劉慕裳】空手道女生的狠準與甜美|accessdate=2017-08-30|last=楊宇翹|work=香港01|language=en}}</ref>。她在11歲開始正式練習空手道，一開始練習「對打」，亦參加過比賽，不過因為曾經令臉部受傷，她不想媽媽擔心就改練「形」（套拳），她認為套拳動作漂亮，講求速度和動感，很適合其個人特色及風格<ref name="蘋果日報0804">{{cite news |title=【印尼亞運】嬌滴滴女生道場一秒變惡煞 劉慕裳隔「空」對談 |url=https://hk.sports.appledaily.com/sports/realtime/article/20180804/58521260 |work=蘋果日報 |date=2018-08-04}}</ref>。

劉慕裳在16歲開始代表香港參加國際比賽，19、20歲開始每年到日本集訓兩次，打下不錯的基礎。及至2015年，她開始成為全職空手道運動員。

== 個人生活 ==
劉慕裳曾讀於[[保良局王賜豪（田心谷）小學|保良局王賜豪（田心谷）小學]]和[[林大輝中學|林大輝中學]]<ref>{{Cite web|url=http://www.ltfc.edu.hk/Website../index.php/campus-news/1056-2016-05-28-09-51-48|title=本校校友劉慕裳於世界空手道排名躍升至第三名|accessdate=2017-08-30|work=www.ltfc.edu.hk}}</ref>，大學時期則在[[香港城市大學|香港城市大學]]修讀創意媒體榮譽理學士，2016年畢業<ref>{{Cite web|url=http://www.cityu.edu.hk/aro/alumnistories/50.htm|title=香港城市大學校友專訪|accessdate=2016|author=|date=|publisher=}}</ref>。
