== 生平 ==
刘强东祖籍湖南，祖父辈移居至[[江苏省|江苏省]][[宿迁县|宿迁县]]。刘强东于1973年3月10日或1974年2月14日<ref group="注" name="birthdate" />出生于宿迁的一个普通乡村——光明村，今属[[宿迁市|宿迁市]][[宿豫区|宿豫区]][[来龙镇|来龙镇]]。在村镇接受小学和中学教育<ref>{{Cite web |url=http://tech.qq.com/a/20150217/043645.htm |title=刘强东为家乡老人发650万春节红包|accessdate=2018-09-09 |author= |date=2015-02-17 |work=责任编辑：jimmonzang |publisher=腾讯网|language=Zh-hans }}</ref>。1992年，刘强东毕业于[[江苏省宿迁中学|宿迁中学]]，同年考入[[中国人民大学|中国人民大学]]社會學系。刘强东从大三时就开始写程序赚钱，曾是班里第一个买[[大哥大|大哥大]]的人，大四开餐厅，到毕业时餐厅赔了20多万<ref>{{Cite web|url = http://edu.163.com/14/0423/10/9QGS5VMT00294MA1.html|title = 刘强东讲述3个血泪段子：一夜竟然愁白了头|accessdate = 2015-08-10|author = |date = 2014-04-23|publisher = 网易教育|language = zh-cn}}</ref>。

1998年6月18日，刘强东在北京[[中关村|中关村]]创办了京东公司，最初主营光磁产品，后来开始经营IT连锁店，最后受2003年[[非典型肺炎|非典]]的影响而放弃了扩张连锁店的计划<ref name=":0">{{Cite web|url = http://money.163.com/11/0117/08/6QJAM8N200254L5E.html|title = 刘强东：电子商务现在正是到了泡沫期|accessdate = 2015-08-10|author = |date = 2011-01-17|publisher = 网易财经|language = zh-cn}}</ref>。

2004年，创办“京东多媒体网”，尝试涉足[[电子商务|电子商务]]领域<ref name=":0" />。

2005年，刘强东关闭零售店面，京东正式转型为一家专业电商公司<ref>{{Cite web|url = http://news.zol.com.cn/456/4560113.html|title = 还原一个真实的京东创始人刘强东|accessdate = 2015-08-10|author = |date = 2014-05-22|publisher = 中关村在线|language = zh-cn}}</ref>。

2009年，刘强东再次入学，就读[[中欧国际工商学院|中欧国际工商学院]][[EMBA|高级管理人员工商管理硕士]]（[[EMBA|EMBA]]）<ref>{{Cite web|url = http://www.chinadaily.com.cn/hqcj/xfly/2014-08-01/content_12121685_3.html|title = 全方位揭秘：极端富豪刘强东|accessdate = 2015-08-10|author = |date = 2014-08-01|publisher = 中国日报网|language = zh-cn}}</ref>。

2012年，荣登[[胡润百富榜|胡润少壮派富豪榜]]，排名第八<ref>{{Cite web|url = http://resource.emagecompany.com/newmoney/huruishaozhuangpai2012.html|title = 2012胡润少壮派富豪榜|accessdate = 2015-08-10|author = |date = |publisher = |language = zh-cn}}</ref>。

2013年，担任第十二届上海市政协委员<ref>{{Cite web|url = http://tech.sina.com.cn/i/2013-01-25/15048016216.shtml|title = 京东商城CEO刘强东出任上海政协委员|accessdate = 2015-08-10|author = |date = 2013年1月25日|publisher = 新浪科技|language = zh-cn}}</ref>，同年留学美国[[哥伦比亚大学|哥伦比亚大学]]<ref>{{Cite web|url = http://www.ctjin.com/shenxinling/nengliang/youxue/2013-12-30/5606.html|title = 刘强东：我为什么要去哥伦比亚留学|accessdate = 2015-08-10|author = |date = 2013-12-30|publisher = 1财金网|language = zh-cn}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 个人生活 ==
“京东”的名称，来自于刘强东自己以及同样毕业于[[中国人民大学|中国人民大学]]的前女友龚小京（又名“龚晓京”）的名字。他们于2003年分手。<ref name="people1">{{cite news |author1=吴亚雄 |title=刘强东晒儿子照片前妻曾挽回婚姻 奶茶妹妹甘作后妈? |url=http://ent.people.com.cn/n/2014/0415/c1012-24898955.html |accessdate=2018-09-02 |publisher=人民网 |date=2014-04-15}}</ref><ref name="sina2">{{cite news|author1=南都娱乐周刊|title=传奶茶妹倒追刘强东 男方有婚史儿子8岁|url=http://ent.sina.com.cn/s/m/2014-04-14/20144126785.shtml|publisher=新浪娱乐|accessdate=2018-09-02}}</ref><ref name="sina1">{{cite web|title=刘强东恋爱史：三个时期的三个女人|url=http://tech.sina.com.cn/i/2014-04-07/19409302803.shtml|website=新浪科技|accessdate=2016-05-25}}</ref><ref>{{cite web|title=8张图带你认识刘强东初恋龚小京|url=http://money.hexun.com/2014-05-24/165095528.html|website=和讯网|accessdate=2016-05-25}}</ref>

刘强东育有一子，于2006年出生。孩子的母亲是一名未对外公开姓名的女性，刘强东可能与其有过一段婚姻。<ref name="people1" /><ref>{{cite news |author1=中华网编辑 |title=刘强东承认有个儿子，11岁了 |url=https://news.china.com/news100/11038989/20170704/30906916_all.html |accessdate=2018-09-02 |publisher=中华网 |date=2017-07-04}}</ref><ref name="sina2"/>

2012年7月15日，刘强东在[[新浪微博|微博]]上晒出自家阳台上的番茄；11分钟后，京东小家电釆销总监庄佳也在微博上晒出一张番茄照。“西红柿门”由此拉开序幕。7月20日，刘强东在一次行业活动上回应称：“西红柿门绝对不是一种营销，是一种无奈。我和庄佳谈恋爱三年了。”后来两人分手<ref>[http://tech.sina.com.cn/i/2014-04-07/19409302803.shtml 刘强东恋爱史：三个时期的三个女人]</ref>。

=== 悉尼派对客人性侵事件 ===
2015年12月26日，刘强东在其[[澳大利亚|澳大利亚]][[悉尼|悉尼]]的公寓中邀客举行了一场[[派对|派对]]。在派对后，一名派对上被灌醉的女性被另一派对客人徐龙伟（音译）带回徐的酒店房间性侵。徐龙伟被判七项罪名成立。刘强东没有被控任何罪行或不当行为。刘强东曾通过律师申请不公布他的身份，理由是可能让他的婚姻和生意受损。法官虽然一开始同意禁止公布可以或倾向可以识别刘身份的任何内容，并随后扩大禁制令到“亿万富翁（billionaire）”和“顶层公寓（penthouse）”两词，但后来推翻了这些命令。刘的律师向刑事上诉法院上诉，但被拒绝，媒体随后（2018年7月）公布了他的名字。<ref>{{cite news |author1=JACQUELINE WILLIAMS |author2=孟宝勒 |title=刘强东被卷入悉尼性侵案：豪宅派对后宾客犯案 |url=https://cn.nytimes.com/asia-pacific/20180725/jd-richard-liu-sydney/ |publisher=纽约时报 |accessdate=2018-09-03 |date=2018-07-25}}</ref><ref>{{Cite web|url =https://www.smh.com.au/national/nsw/court-lifts-suppression-on-name-of-chinese-billionaire-entangled-in-rape-trial-20180723-p4zt4c.html | title= Court lifts suppression on name of Chinese billionaire entangled in rape trial}}</ref><ref>{{Cite web|url =http://news.sina.com.cn/s/2018-09-03/doc-ihiqtcan0877857.shtml| title= 刘强东在美被拘留照曝光 美媒称其涉性行为犯罪}}</ref><ref>{{cite web |title=刘强东涉嫌“一级强奸”：如案件持续“将冲击京东” |url=https://www.bbc.com/zhongwen/simp/chinese-news-45418676 |website=BBC|accessdate=2018-09-19 |date=2018-09-05}}</ref>
