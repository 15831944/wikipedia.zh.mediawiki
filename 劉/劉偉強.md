'''劉偉強'''（英文名：{{lang|en|Andrew Lau}}，{{bd|1960年|4月4日|catIdx=L刘}}<ref name="5000萬">{{cite web |title = 任它票房5000萬——專訪《無間道》導演劉偉強 - 中國新聞網 |url = http://big5.chinanews.com.cn:89/n/2003-01-23/26/266750.html |date = 2003年1月23日 }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}{{cite web |title = 東方新聞 |url = http://www.eastday.com/epublish/big5/paper148/20030123/class014800007/hwz872325.htm |date = 2003年1月23日 }}</ref>），[[香港|香港]]電影[[導演|導演]]以及[[攝影師|攝影師]]。

==生平==
劉偉強自小對拍攝[[電影|電影]]產生興趣。<ref name="话事人">{{cite web |title = 刘伟强：黑帮片“话事人”--南都娱乐周刊 |url = http://nbweekly.oeeee.com/Print/Article/7152-0.shtml |date = 2009-02-11 |deadurl = yes |archiveurl = https://web.archive.org/web/20090728090635/http://nbweekly.oeeee.com/Print/Article/7152-0.shtml |archivedate = 2009-07-28 }}</ref>父親劉國榮是一名建築公司的規劃師。而胞弟劉偉民則是一名設計師。劉偉強早年在[[新界|新界]][[元朗|元朗]]大旗嶺村生活，並在村內的嶺文學校讀小學。後來一家移居[[香港島|香港島]][[銅鑼灣|銅鑼灣]]禮頓道嘉柏大廈。<ref>{{web reference|URL=https://hk.lifestyle.appledaily.com/nextplus/magazine/article/20020530/2_2627969/%E4%BA%BA-%E6%B0%A3-%E8%A8%AD-%E8%A8%88-%E5%B8%AB-%E7%8E%A9-%E8%BD%89-%E6%9C%AC-%E5%9C%9F-%E6%96%87-%E5%8C%96|title=人氣設計師玩轉本土文化|work=香港蘋果日報|year=2002年|date=5月30日}}</ref>他曾就讀[[聖貞德中學|聖貞德中學]]<ref>{{web reference|URL=http://www1.hkej.com/features/article?q=%23%E6%A0%A1%E9%95%B7%E7%B3%BB%E5%88%97%23&suid=1341630674|title=聖貞德中學校長 留學生涯學會面對自己|work=信報|year=20155年|date=12月7日}}</ref>
和[[嶺南中學_(香港)|嶺南中學]]（1975年入讀），至1980年中學畢業<ref name="《校園通訊》第29期">[http://www.lingnan.edu.hk/com/dl/1516/no29/index.pdf 嶺南中學《校園通訊》第29期，2015年]</ref>。在校時曾是游泳學會和攝影學會成員，又為校內的話劇比賽做配樂。他之後到[[中環|中環]]一所交易行當了一年黃金買賣員。<ref name="《校園通訊》第29期">[http://www.lingnan.edu.hk/com/dl/1516/no29/index.pdf 嶺南中學《校園通訊》第29期，2015年]</ref>劉偉強在1981年進入[[邵氏公司|邵氏公司]]，由攝影小工做起，慢慢地升至助理攝影師及副攝影師。最後在1985年正式成為攝影師，1988年為[[王家衛|王家衛]]負責《[[旺角卡門|旺角卡門]]》的攝影工作。1990年擔任導演，1995年與[[文雋|文雋]]、[[王晶_(導演)|王晶]]合組[[最佳拍檔電影公司|最佳拍檔電影公司]]，並把漫畫[[古惑仔|古惑仔]]改編成一系列古惑仔電影，獲得票房佳績。<ref name="话事人" /><ref name="5000萬" />其後劉偉強籌組[[基本映畫|基本映畫]]製作公司<ref name="Golden Scene">{{cite web| title = 強捕犯 - Golden Scene | url = http://www.goldenscene.com/flock/flock-cast-ch.html }}</ref>，拍攝改編自其他漫畫作品的電影，包括《[[風雲雄霸天下|風雲雄霸天下]]》及《[[中華英雄_(1999年電影)|中華英雄]]》等。<ref name="话事人" />

2002年，由商人[[林建岳|林建岳]]投資，劉偉強與[[麥兆輝|麥兆輝]]及編劇[[莊文強|莊文強]]合作編導製作的[[警匪片|警匪片]]《[[無間道|無間道]]》，再在香港、[[中國|中國]]等地取得票房佳績，并获得香港金像与台湾金马的最佳电影和最佳导演等多项大奖<ref name="《無間道》威盡林建岳">{{cite web | title = 蘋果日報 - 20070304 - 《無間道》威盡林建岳、劉偉強叩謝白龍王 | url = http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20070304&sec_id=462&art_id=6867829 | date = 2007年3月4日 }}</ref>，其後並拍攝《[[無間道II|無間道II]]》及《[[無間道III|無間道III]]》等續集。故事版權獲得[[荷里活|荷里活]]片商垂青，並由導演-{zh:[[馬田·史高西斯|馬田·史高西斯]];zh-hans:[[马丁·斯科塞斯|马丁·斯科塞斯]];zh-hk:[[馬田·史高西斯|馬田·史高西斯]];zh-tw:[[馬丁·史柯西斯|馬丁·史柯西斯]];}-重新拍攝西片版本《[[無間道風雲|無間道風雲]]》。<ref name="《無間道》威盡林建岳" /><ref name="三字經">{{cite web | title = 蘋果日報 - 20071129 - 三字經　劉偉強 | url = http://hk.apple.nextmedia.com/template/apple_sub/art_main.php?iss_id=20071129&sec_id=12187379&art_id=10472969 | date = 2007年11月29日 }}</ref>2005年執導改編自日本賽車漫畫的電影《[[頭文字D_(電影)|頭文字D]]》。

在2005年第二十四屆香港電影金像獎頒獎典禮頒發最佳視覺效果獎項時，劉偉強作為頒獎嘉賓，竟在台上將得獎者之一洪毓良的名字讀成「洪唔知咩野良」（洪「不知什麼」良），被批評有欠莊重，以及中文程度欠佳。<ref>http://paper.wenweipo.com/2005/03/28/EN0503280003.htm</ref><ref name="三字經" />

2006年，劉偉強首次執導[[韓國|韓國]]電影《[[雛菊_(電影)|雛菊]]》，3月9日在南韓上映，其後在香港、中國大陸及[[台灣|台灣]]上映。<ref>{{imdb title|id=0468704|title=雛菊}}</ref>2007年，劉偉強進軍美國荷里活，執導由-{zh-hans:[[理查·基尔|理查·基尔]];zh-hk:[[李察·基爾|李察·基爾]];zh-tw:[[李察·吉爾|李察·吉爾]];}-主演的電影《[[強捕犯|強捕犯]]》。<ref name="Golden Scene" /><ref name="三字經" />

2008年，劉偉強接替[[黃岳泰|黃岳泰]]成為[[香港專業電影攝影師學會|香港專業電影攝影師學會]]會長<ref>{{cite web | title = 刘伟强接替黄岳泰 任香港摄影师学会会长 | url = http://ent.sina.com.cn/m/c/2008-04-21/13021996289.shtml | date = 2008年4月21日 }}</ref>。

2014年推出第二部美国电影《[[青龙复仇|青龙复仇]]》，[[马丁·斯科塞斯|马丁·斯科塞斯]]任监制，影片取材于真实事件，讲述两个中国兄弟在[[纽约市|纽约市]][[黑帮|黑帮]]里挣扎的故事。<ref >{{cite web |url=http://www.1905.com/news/20140814/791410.shtml | language= | title=刘伟强《青龙复仇》发预告 马丁·斯科塞斯任监制 |publisher=1905电影网 |date=2014-08-14 |accessdate= }}</ref>

2015年3月，劉偉強接替[[爾冬陞|爾冬陞]]成為[[香港導演協會|香港導演協會]]會長<ref>{{cite web | title = 尔冬升卸任香港导演会会长 刘伟强接棒 | url = http://ent.sina.com.cn/m/c/2015-03-19/doc-iawzuney0869640.shtml | date = 2015年3月19日 }}</ref>。代表香港專業電影攝影師學會任香港電影金像獎董事局副主席。
