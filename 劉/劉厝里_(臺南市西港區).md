'''劉厝里'''位於[[臺南市|臺南市]][[西港區|西港區]]西北部，與[[七股區|七股區]]、[[佳里區|佳里區]]相鄰{{r|縣|page=587}}{{r|西港常民文化|page=109}}。里內原有劉厝、蚶西港、中社仔三個聚落{{r|縣|page=597}}{{r|西港常民文化|page=109}}{{r|地名}}，2018年1月29日-{里}-界調整後，原屬新復里的麻豆寮仔被改劃入劉厝里的範圍{{r|里鄰調整}}。

== 沿革 ==
該里在日治時期屬於[[西港庄|西港庄]][[大字|大字]]劉厝，二次大戰設為劉厝村{{r|縣|page=587}}{{r|西港常民文化|page=109}}。2010年12月25日，因臺南縣市合併升格，改為劉厝里。2018年1月29日，調整了與新復里之間的里界{{r|里鄰調整}}。

== 聚落與地名 ==
劉厝在劉厝大排（七股溪上游）南邊，因為是劉姓家族所建立的聚落而得名{{r|縣|page=587}}{{r|西港常民文化|page=110}}{{r|地名}}。劉姓家族的開臺祖名為劉登魁，來自「福建省泉州府[[同安縣|同安縣]]廿四都後山尾窗裡社」，傳有三子，分別是大房劉國禎、二房劉國棟、三房劉國樑{{r|地名}}。庄廟是聖帝宮{{r|地名}}。

蚶西港在劉厝北邊，舊名「含西港」{{NoteTag|含換成蚶據說是後人因發音相近而誤植{{r|西港常民文化|page=109}}。}}，是[[臺江內海|臺江內海]]港口之一，以黃姓為大姓{{r|縣|page=588}}{{r|西港常民文化|page=109}}{{r|地名}}。劉厝劉家開臺祖劉登魁據說即是從含西港上岸，而昔日港口的位置在今劉厝橋一帶，該處有著「舢板頭」的舊地名{{r|縣|page=588}}{{r|西港常民文化|page=109}}。此外相對於佳里區的「外渡頭」，含西港內有稱為「內渡頭」的小地名{{r|縣|page=588}}{{r|西港常民文化|page=109}}。另外過去七股溪往上游在西港分成兩條，北邊叫「頂頭港」，即含西港{{r|西港常民文化|page=109}}。南邊叫下頭港，現為南寶樹脂南側的水圳{{r|西港常民文化|page=109}}{{r|地名}}。而「含西」據說是魚名，有傳說過去有一對大含西從大海游到此地，破壞了當地的風水，結果一隻被當地居民打死，另一隻則逃到了八份（[[港東里_(臺南市西港區)|港東里]]）才死去{{r|西港常民文化|page=109}}。過去聚落規模很大，當時庄廟開仙真宮參與[[西港香|西港香]]時可以出兩組宋江陣、兩組跳鼓陣和一組天子門生{{r|縣|page=588}}{{r|西港常民文化|page=109}}{{r|地名}}。

中社仔則在劉厝西邊，居民多為王姓，名稱由來是因為位在劉厝與外渡頭之間的關係{{r|縣|page=588}}{{r|西港常民文化|page=110}}{{r|地名}}。另外過去在劉厝東邊曾有一個名叫「許厝」的聚落，但後來說因為風水不佳不安寧的關係，在日治時期該聚落就散庄了{{r|縣|page=588}}{{r|西港常民文化|page=109}}。

麻豆寮仔（麻豆寮）原屬新復里，2018年1月29日調整里界後改屬劉厝里{{r|里鄰調整}}。居民來自七股區竹橋里的「麻豆寮仔（麻豆寮）」，相對於七股的麻豆寮仔又稱為「東寮仔（東爿寮仔）」{{r|縣|page=594}}{{r|西港常民文化|page=103}}{{r|地名}}。

== 宗教信仰 ==
該里境內有劉厝聖帝宮、蚶西港開仙真宮等廟宇{{r|縣|page=588}}。

== 景點 ==
*劉厝古墓：2001年2月所發現的兩座古墓，分別是劉厝劉姓開臺祖劉登魁墓與來臺二代祖媽劉陳孝淑孺人墓{{r|西港常民文化|page=110}}。因為過去[[曾文溪|曾文溪]]多次的改道淤積，兩座古墓被埋在地底多年，現離地面有13臺尺（約3.93公尺）的差距{{r|西港常民文化|page=110}}。而根據參與發掘工作的[[成功大學|成功大學]]歷史系教授林明瑞表示，古墓所用磚頭都是從泉州所運來{{r|西港常民文化|page=111}}。
*劉家古厝：位在劉厝聖帝宮旁，又稱劉楩楠古厝{{r|西港常民文化|page=112}}。該古厝約建於光緒年間，正門門額寫著「薰風南來」，兩邊牆上則用磚頭砌了六個壽字{{r|西港常民文化|page=112}}。
