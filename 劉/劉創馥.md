'''劉創馥'''（{{lang-en|Chong-Fuk Lau}}），香港哲學家，[[香港中文大學|香港中文大學]]哲學系教授<ref name = "zoung man daai hok">{{cite web |author1= |title=LAU，Chong-fuk|url =http://www.phil.arts.cuhk.edu.hk/web/academic/lau-chong-fuk/|agency=[[香港中文大學|香港中文大學]]|accessdate=2019-03-28|}}</ref>，[[滑翔傘|滑翔傘]]飛行員，曾任職於德國[[達姆施塔特工業大學|達姆施塔特科技大學]]哲學系。
==生平==
劉創馥早年於香港中文大學完成訊息工程學學士及哲學碩士學位，後考入[[海德堡大學|海德堡大學]]攻讀哲學博士學位。曾於[[達姆施塔特工業大學|達姆施塔特科技大學]]哲學系任教。劉創馥自2004年起，於香港中文大學哲學系任教，現職哲學系教授及研究學部主任，其研究興趣包括德國哲學、形上學、心靈哲學與宗教哲學。

劉創馥學術著作甚豐，論文著有三十多篇，書籍自著兩本，合著一本。<ref name = "zoung man daai hok"/>

劉創馥是業餘滑翔傘飛行員。
==思想==
===批判黑格爾===
[[黑格爾|黑格爾]]之[[理性主義|理性主義]]，以嚴密之辯證（Dialektik），建立了一個思想體系，號稱幾乎「思維」是無所不包，所有具體事物存在與發展，皆於思維之中，並遵從思維法則，所以可以完全解決自古希臘以來之眾多哲學難題，並吸收融合歷代哲學之精髓，而化成一個[[絕對觀念論|絕對觀念論]]。劉創馥認為，就算黑格爾之系統有多精密，想法有多深刻，但如果宣稱人類理性，可以建立一個哲學體系，足以擁有「絕對真理」，並可完全解決所有問題，就實在是高估了人類理性，並超越了哲學本身之能力。<ref>{{cite book |last= |editor1-last=劉國英|editor2-last=張燦輝|date=2009年|title= 修遠之路：香港中文大學哲學系六十周年系慶論文集（同寅卷）|location=香港 |publisher=中文大學出版社|page= |isbn=978-962-996-425-2 |author-link= }}</ref>
===研究黑格爾===
劉創馥之著作《黑格爾新釋》，研究方式，獨樹一幟。劉創馥將黑格爾之想法，視為一個整體。劉創馥沒有將黑格爾青年時之觀點與晚年時之思想互相排斥，而是視兩者之關係為，心靈內在發展。劉創馥將黑格爾之想法，詮釋為一種方法，而不是一種完滿之知識系統。<ref>{{cite journal|author=楊德立|title= 黑格爾作為方法：評劉創馥《黑格爾新釋》 |location=[[臺北市|臺北市]]|journal=東吳哲學學報 |publisher=[[東吳大學_(臺灣)|東吳大學]]|volume=38 |year=2018 |pages=115-123 }}</ref>
==批判==
劉創馥認為，科學與宗教有極大衝突，而科學是會勝出。<ref>{{cite web |author1=[[周保松|周保松]]|title=寫於「宗教與科學」大討論之後|url =https://thestandnews.com/culture/%E5%AF%AB%E6%96%BC-%E5%AE%97%E6%95%99%E8%88%87%E7%A7%91%E5%AD%B8-%E5%A4%A7%E8%A8%8E%E8%AB%96%E4%B9%8B%E5%BE%8C/|agency=[[立場新聞|立場新聞]]|date=2017-01-15|accessdate=2019-03-28|}}</ref>劉創馥提出三個命題嘗試證明，科學和宗教基本上是互不相容：一、兩者之方法與態度不相容；二、兩者之內容間有直接衝突；三、兩者之內容間有間接衝突。就劉創馥之論證方式，[[陳啟恩|陳啟恩]]指劉創馥理據並不有力，因為劉創馥未能證明科學與基督教在內容上有任何直接衝突或間接衝突，亦未能證明科學與基督教之態度與方法於認識世界上，彼此不相容。<ref>{{cite web |author1=陳啟恩|title=再論科學與基督教的關係——回應劉教授及王教授|url =https://thestandnews.com/philosophy/%E5%86%8D%E8%AB%96%E7%A7%91%E5%AD%B8%E8%88%87%E5%9F%BA%E7%9D%A3%E6%95%99%E7%9A%84%E9%97%9C%E4%BF%82-%E5%9B%9E%E6%87%89%E5%8A%89%E6%95%99%E6%8E%88%E5%8F%8A%E7%8E%8B%E6%95%99%E6%8E%88/|agency=[[立場新聞|立場新聞]]|date=2017-03-12|accessdate=2019-03-28|}}</ref>
