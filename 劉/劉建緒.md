'''劉建緒'''（1892年10月2日—1978年3月22日），[[号|号]]恢先，[[字|字]]子養，[[光绪|光绪]]十八年(1892年)10月2日出生于[[湖南省|湖南省]][[长沙府|长沙府]][[醴陵縣|醴陵縣]]北乡的[[清安铺|清安铺]]。屬[[中央俱樂部|CC派]]。

== 早年生平 ==
光緒二十九年（1903年）就读[[湖南省省立师范馆附小|湖南省省立师范馆附小]]，光緒三十三年（1907年）转学新成立的[[兰谊小学|兰谊小学]]。[[宣統|宣統]]元年（1909年）考进湖南省[[長沙市長郡中學|长沙府中学堂]]。[[民國|民國]]二年（1913年）进[[陆军第二预备学校|陆军第二预备学校]]。民國四年（1915年）进[[保定陆军军官学校|保定陆军军官学校]]，炮科第三期。

== 中年經歷 ==
毕业后参加一系列战役，包括1923年的湖南护宪，1926年的[[國民革命軍北伐|北伐]]，以及[[抗日战争|抗日战争]]。1937年1月6日，第四路軍總指揮劉建緒在浙江[[衢縣|衢縣]]主持開閩、浙、皖、贛邊區綏靖會議，討論進攻中國工農紅軍游擊部隊辦法；會期兩日，到會者70餘人<ref name="民" >{{cite book |editors=[[李新_(1918年)|李新]]總主編，中國社會科學院近代史研究所中華民國史研究室，韓信夫、姜克夫主編 |title=《中華民國史大事記》 |location=北京 |publisher=[[中華書局|中華書局]] |year=2011 |isbn=9787101079982 }}</ref>{{rp|5331}}。5月7日，閩浙贛皖四省邊區主任劉建緒訓令各縣縣長，在向紅軍游擊隊進攻期間，各縣長必須親自辦理善後工作<ref name=民 />{{rp|5421}}。6月23日，蔣電令劉建緒，加緊進攻紅軍南方游擊部隊，「早完任務為要」<ref name=民 />{{rp|5451}}。

民國31年（1942年）起奉命离军从政，任中華民國[[福建省|福建省]]省政府委员兼主席，先在临时省会[[永安市|永安]]，后回到[[福州|福州]]。1947年7月19日，國民政府派劉建緒為福建選舉事務所主任委員；國民政府同時派定其為[[國民大會代表|國民大會代表]]，[[立法院|立法院]][[立法委員|立法委員]]<ref name=民 />{{rp|8385-8386}}。1948年辞退[[福建省省主席|福建省省主席]]职，携眷回湖南长沙。1949年去[[香港|香港]]。1951年举家移民[[巴西|巴西]]。

== 逝世 ==
1978年3月病逝于巴西，终年86岁。
