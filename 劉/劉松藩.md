'''劉松藩'''（{{bd|1931年|12月3日|2016年|11月18日|catIdx=L劉}}），[[臺灣|臺灣]][[臺中|臺中]][[大甲區|大甲區]]人，[[中華民國|中華民國]][[立法院|立法院]]第一屆（第1～6次增額立委）、第二、三、四、五屆[[立法委員|立法委員]]，並擔任第9任[[立法院長|立法院長]]、第7任[[立法院副院長|立法院副院長]]。他是立法院1992年全面改選後首位来自台湾的立法院長。

== 早年 ==
1931年，劉松藩出生於台中大甲，他繼承父親劉雲騰在台中縣大甲鎮農會總幹事一職，一直在農會任職。因劉家長久以來積極經營農會系統，地方勢力強大。

== 立法院 ==
1973年，劉松藩擔任第一屆增額立委後，政壇一路平步青雲，成為台中紅派掌門人。他總共擔任32年立委(1973年—2004年，即第一屆增額立委—第五屆立委)，其中7年是擔任[[立法院院長|立法院院長]]，直到第4屆時，敗給當時的副院長王金平，劉因此退出國民黨。2000年，他加入宋楚瑜競選總統寶座的陣營，合組親民黨。

== 卸任後 ==
1999年，劉松藩雖連任立委，但未能蟬聯院長。其院長職位由上一任期的副院長，時由[[高雄縣|高雄縣]]選區選出的國民黨立委[[王金平|王金平]]當選就任。2001年，劉松藩脫離中國國民黨後，改由親民黨以不分區形式提名為立法委員。

他在1998年院長任內，涉嫌協助[[台中商業銀行|台中商銀]][[董事長|董事長]][[曾正仁|曾正仁]]，居中介紹經營不善的[[知慶公司|知慶公司]]向台中商銀非法超貸15億元，劉松藩並從中取得1億5000萬元的傭金，在2004年9月7日被[[臺灣高等法院|臺灣高等法院]]臺中分院判決[[背信罪|背信罪]]成立，当時劉松藩在美國執行立法委員考察，他同日宣佈退出[[親民黨|親民黨]]及辭去[[不分區立法委員|不分區立法委員]]職務，結束32年立委生涯。

劉松藩叱咤政壇數十年，2004年時，因[[廣三超貸背信案|廣三超貸背信案]]，從中賺取1.5億元新台幣(約3,600萬港元)佣金，遭判刑4年。劉曾以患糖尿病和心血管疾病為由，希望申請保外就醫，但遭法務部拒絕，為逃避牢獄之災，劉松藩潛逃出境，政壇曾傳出他身處美國洛杉磯，但後因不適應當地生活，輾轉到大陸，之後久居流亡當地。2007年胞弟過世與2009年劉妻蔡雲卿過世，都一度傳出劉松藩可能返台奔喪，但最後都未有現身。依據台中地檢署在2007年2月發出通緝令，有效期15年，劉松藩的通緝令到2022年才到期。曾有臺灣平面媒體記者在[[香港|香港]]九龍尖沙咀[[半島酒店|半島酒店]]遇見劉松藩，但不能証實其最終下落。2009年其妻蔡雲卿在台病逝喪禮，也未見劉松藩身影，晚年情況不詳。

== 被通緝及流亡海外 ==
2004年9月7日，劉松藩因[[廣三案|廣三案]]依背信罪判刑4年、併科罰金3000萬元定讞。同年8月底潛逃[[美國|美國]]遭到通緝，追訴期至2022年2月。曾有[[中國時報|中國時報]]記者在[[香港|香港]][[九龍|九龍]]發現其行蹤。

== 逝世 ==
2016年11月18日，劉松藩因心肺慢性阻塞、心律不整等疾病於於[[美國|美國]]洛杉磯病逝，享年84歲<ref>[https://udn.com/news/story/1/2154255 前立法院長劉松藩病逝美國 享壽85歲]</ref><ref>[https://udn.com/news/story/1/2154372 叱吒政壇半世紀 立院強人「劉松藩」] [[联合新闻网|联合新闻网]]</ref><ref>[https://udn.com/news/story/1/2154707 王金平：劉松藩盼睡夢中離去 如願了][[联合新闻网|联合新闻网]]</ref><ref>[http://udn.com/news/story/1/2154284 劉松藩在美病逝 骨灰今安奉新北市納骨塔]
