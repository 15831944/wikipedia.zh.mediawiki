'''劉桂標'''（{{lang-en|Kwai-piu Lau}}），香港哲學家，[[香港人文學會|香港人文學會]]理事長，曾任[[香港中文大學|香港中文大學]]哲學系兼任[[講師|講師]]<ref name = "appledaily1">{{cite_news|url=https://hk.news.appledaily.com/local/realtime/article/20180612/58306044|title=中大哲學系講師批系方以權謀私　被要求道歉否則解僱|work=《[[蘋果日報_(香港)|蘋果日報]]》|date=2018-06-12|accessdate=2019-04-05}}</ref>。

==事業==
2018年6月，劉桂標時任[[香港中文大學|香港中文大學]]哲學系兼任[[講師|講師]]，其遭系方削減其任教之科目，遂因不滿而發出公開信，指課程負責人「私相授受」與「以權謀私」。香港中文大學哲學系系主任[[張錦青|張錦青]]則稱，劉桂標陳述有謬誤，要求劉桂標公開道歉並移除信件，倘不就公開信內之謬誤道歉，就不會續聘劉桂標。<ref name = "appledaily1"/>香港中文大學哲學系系方則指，據校方統一課檢成績計算方法，於過去七個學期，劉桂標之課檢分數，乃於哲學系二十位兼任老師之中，最低之四位，所以劉桂標並非教學表現得優秀。2018年6月11日，中大哲學系已決定停止聘用劉桂標。<ref>{{cite_news|url=https://thestandnews.com/society/%E4%B8%8D%E6%BB%BF%E8%A2%AB%E5%89%8A%E8%AA%B2-%E5%85%BC%E8%81%B7%E6%95%99%E5%B8%AB%E6%89%B9%E7%A7%81%E7%9B%B8%E6%8E%88%E5%8F%97-%E4%B8%AD%E5%A4%A7%E5%93%B2%E5%AD%B8%E7%B3%BB-%E4%B8%8D%E9%81%93%E6%AD%89%E4%B8%8D%E5%AE%89%E6%8E%92%E6%8E%88%E8%AA%B2/|title=不滿被削課　兼職教師批私相授受　中大哲學系：不道歉不安排授課|work=《[[立場新聞|立場新聞]]》|date=2018-06-11|accessdate=2019-04-05}}</ref>[[黎明_(社會學家)|黎明]]認為，劉桂標是「敢言或起來反抗之講師」，疑似受到秋後算帳，而不獲續約或變相解僱。<ref>{{cite web |author1=陳娉婷|agency=[[關鍵評論網|關鍵評論網]]|title=黎明：高教界金錢及威權至上，賤視大學講師價值|url =https://hk.thenewslens.com/feature/hk-lecturer-crisis/98011|date=2018-06-17|accessdate=2019-04-05|}}</ref>

==思想==
===批判邏輯實證主義===
[[邏輯實證主義|邏輯實證主義]]將解釋自然事件與歷史事件視為相同，劉桂標認為，這是邏輯實證主義之謬誤，因為自然事件與歷史事件之間根本就不同，所以要解釋後者，不可應用到解釋前者時所應用到之普遍法則。劉桂標指，即使邏輯實證主義解釋歷史亦有部份之真理，例如能應用自然事件之因果法則，解釋有關人之外部之事件，但未能充份解釋歷史人物一切行動。<ref>{{cite journal|author=劉桂標|title= 論實證論的歷史解釋理論（下）|location=[[香港|香港]]|journal=《人文月刊》 |publisher=香港人文哲學會|volume=16 |year=1995 }}</ref>
