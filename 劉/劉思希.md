'''劉思希'''（{{lang|en|'''Lisa Lau See Hei'''}}，{{bd|1985年|9月22日<ref name=LL>{{Cite web |url=http://misshk.tvb.com/2011/contestant/lisalau |title=11. 劉思希 - 2011香港小姐競選 - tvb.com |access-date=2013-09-12 |archive-url=https://web.archive.org/web/20130925064722/http://misshk.tvb.com/2011/contestant/lisalau |archive-date=2013-09-25 |dead-url=yes }}</ref>}}），原名'''劉雅儀'''<ref>[http://hk.apple.nextmedia.com/entertainment/art/20110612/15333513 港姐港男鬥起底 嫩口安麗變臉改名]，[[香港蘋果日報|香港蘋果日報]]，2011年6月12日</ref>，香港女演員及主持，現為[[無綫電視藝員列表|無綫電視經理人合約]]藝員<ref>[http://artiste.tvb.com/main/artistedetail/lisalau 劉思希 Lisa Lau - TVB藝人資料 - tvb.com]</ref>。

== 簡介 ==
劉思希曾赴美國[[西北大學_(伊利諾伊州)|西北大學]]及[[林肯大學|林肯大學]]留學，分別修讀凱洛格行政管理碩士和工商管理碩士課程，畢業後成為市場總監，2009年便參與《2009年[[中華小姐環球大賽|中華小姐環球大賽]]》，進入決賽12強，並獲得「友誼小姐」獎項<ref name=LL2>[http://www.hihoku.com/news/20110518/94865.html 港姐翻版廖安丽 曾夺选美友谊小姐] {{Wayback|url=http://www.hihoku.com/news/20110518/94865.html |date=20140723012952 }}，[[Hi好酷|Hi好酷]]，2011年5月18日</ref>。另外，她亦曾參選《[[2011年度香港小姐競選|2011年度香港小姐競選]]》<ref name=LL/>，因長相貌似前無綫電視女藝員[[廖安麗|廖安麗]]而獲傳媒稱為「年輕版[[廖安麗|廖安麗]]」<ref name=LL2/>，之後則签約[[無綫電視|無綫電視]]，曾於2011至2012年擔任[[TVB娛樂新聞台|TVB娛樂新聞台]]主播。

2014年，劉思希在該台電視劇《[[忠奸人_(電視劇)|忠奸人]]》中首次擔任戲份較重的角色「杜綽姿（Gillian）」<ref>[http://news.mingpao.com/pns/%E5%8A%89%E6%80%9D%E5%B8%8C%E6%A8%82%E5%81%9A%E6%9D%8F%E5%85%92%E5%9B%A7%E6%A8%A3%E6%8E%A5%E7%8F%AD%E4%BA%BA/web_tc/article/20140822/s00016/1408644201313
