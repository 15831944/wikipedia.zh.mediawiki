'''劉偉健'''（{{lang-en|Wai Kin Kurtis Lau}}，{{bd|1992年|6月9日}}，遊戲代號：'''Toyz'''）出生於[[英屬香港|英屬香港]]，曾經為[[英雄聯盟|英雄聯盟]][[電子競技|電子競技]]職業選手。過往曾效力於Cross Gaming、[[台北暗殺星|台北暗殺星]]（Taipei Assassins）、[[香港電子競技戰隊|香港電子競技戰隊]]（Hong Kong Esports）、[[Raise_Gaming|Raise Gaming]]<ref name="ltntw-1827500">{{Cite web |url =http://sports.ltn.com.tw/news/breakingnews/1827500|title =電競》酷寒戰士Toyz回歸實況界 觀看人數30分鐘破8萬|author = |date =2016年9月15日 |publisher =[[自由電子報|自由電子報]] |language =zh-tw }}</ref>。隸屬[[英皇娛樂|英皇娛樂]]子公司「[[英皇娛樂|英皇電競]]」電競總監。

==簡介==
熱衷電腦遊戲主要參與[[英雄聯盟|英雄聯盟]]職業賽事，曾為[[台北暗殺星|台北暗殺星]]職業隊伍中路選手並在2012年獲得S2[[英雄聯盟世界大賽|英雄聯盟世界大賽]]冠軍，後轉隊於[[香港電子競技戰隊|香港電子競技戰隊]]，後期因合約糾紛長達1年處於無法上場比賽或直播實況的冷凍狀態。2016宣布復出擔任[[銳思娛樂有限公司|銳思娛樂有限公司]]旗下Raise Gaming戰隊教練，2017年由所屬的銳思娛樂有限公司向LMS聯盟委員會提出的經營權轉移申請已審核通過，全數轉移給香港英皇娛樂集團旗下之英皇電競。現擔任旗下戰隊電競總監兼總教練<ref>{{Cite web |url =https://gnn.gamer.com.tw/0/152610.html|title =前選手 Toyz 率領《英雄聯盟》戰隊 Raise Gaming 經營權轉移|author = Jessica |date =2017年9月15日 |publisher =[[巴哈姆特電玩資訊站|巴哈姆特電玩資訊站]] |language =zh-tw }}</ref><ref>{{Cite web |url =http://sports.ltn.com.tw/news/breakingnews/2195269|title =LOL》打出好表現被看見 英皇集團買下RG 2018春季賽經營權|author = 陳耀宗 |date =2017年9月16日 |publisher =[[自由電子報|自由電子報]] |language =zh-tw }}</ref>。
2018年6月卸下總教練職務轉任監督。<ref>{{cite web |title=Toyz卸下GRX主教練一職 轉任戰隊監督 |url=https://game.udn.com/game/story/10454/3185735}}</ref>。同時也是位知名[[遊戲實況|遊戲實況]]玩家，於[[Twitch|Twitch]]實況頻道直播最高同時收看人數多達十萬人。<ref>{{cite web |title=Toyz開實況的期間人數一度衝破了10萬大關 |url=https://game.upmedia.mg/LOL/text.php?SerialNo=4175}}</ref>

==糾紛==
*劉偉健於[[HKES|HKES]]後期，因與HKES在合約方面無法達成共識，長達1年處於無法上場比賽或直播實況的冷凍狀態，故走上法律途徑，最終以210萬港幣的和解金予HKES換取終止合約<ref>{{cite web |title=前《英雄聯盟》選手 Toyz 與 HKES 合約糾紛告一段落 Toyz 將賠償 210 萬港幣 |url=https://gnn.gamer.com.tw/3/149153.html}}</ref>，而後創立主要由TPA 2隊成員組成的Raise Gaming 。
