'''刘仲敬'''（{{bd|1974年|}}），旅[[美國|美]]作家。著有《民国纪事本末》、《从华夏到中国》、《经与史：华夏世界的历史建构》等书籍。

== 经历 ==
=== 就学时期 ===
生于1974年，[[四川省|四川省]][[资阳市|资阳市]]人。1996年于[[华西医科大学|华西医科大学]]毕业，在乌鲁木齐公安局任职[[法医|法医]]十年。2009年后攻读四川大学的世界史硕士学位，主修英国历史。2012年入[[武汉大学|武汉大学]]攻读博士学位<ref name="thepaper">{{cite web|title=刘仲敬：中国人从本质上来讲不近神学 |url=https://www.thepaper.cn/newsDetail_forward_1275174 |publisher=[[澎湃新闻|澎湃新闻]] |author=李宗陶 |date=2014-11-08 |accessdate=2018-09-28}}</ref>。

2015年初在共识网开设点评近世人物专栏，7月出任凤凰网新闻客户端主笔<ref name="epaper2016">{{cite web |title=大事记|url=http://epaper.oeeee.com/epaper/C/html/2016-01/11/content_2463.htm |author=|publisher=[[南方都市报|南方都市报]] |date=2016-01-11 |archiveurl=https://web.archive.org/web/20180913055949/http://epaper.oeeee.com/epaper/C/html/2016-01/11/content_2463.htm |archivedate=2018-09-13 |access-date=2018-09-28 |dead-url=no }}（附于李尔克文章《刘仲敬的三个面相》）</ref>。12月12日，刘仲敬参加[[香港浸会大学|香港浸会大学]]主办的论坛，也是多名演讲者之一<ref>{{cite web|title=兩地學者暢談中國模式與文明共融 |url=http://paper.wenweipo.com/2015/12/12/OT1512120001.htm |publisher=[[文匯報|文匯報]] |date=2015-12-12 |archiveurl=https://web.archive.org/web/20181127234311/http://paper.wenweipo.com/2015/12/12/OT1512120001.htm |archivedate=2018-11-27 |dead-url=no}}</ref>。

2016年，刘仲敬出走美国<ref name="jiemian2017"/>。

=== 网络出名 ===
刘仲敬在武汉大学其间完成6卷[[大卫·休谟|大卫·休谟]]的英国史中译本，后创作《民国纪事本末》获得好评<ref name="thepaper"/>。

2007年开始在[[豆瓣|豆瓣]]活跃（网名数卷残编），贴文有时遭管理员删除仍不断发帖。网友因此整理他一系列文章多达38万字。2014年刘仲敬接受[[共识网|共识网]]的采访表示对中国的民族问题感到忧心，不支持大一统的立场，自此渐有名气<ref name="thepaper"/><ref>{{cite web|title=刘仲敬：我们在世界中的位置 |url=http://www.21ccom.net/articles/sxwh/shsc/article_20140513105889.html |publisher=[[共识网|共识网]] |date=2014-05-19 |archiveurl=https://web.archive.org/web/20160415225325/http://www.21ccom.net/articles/sxwh/shsc/article_20140513105889.html |archivedate=2016-04-15}}</ref>。由于刘仲敬长年浸泡于网络[[论坛|论坛]]，在与网友论战时曾自称阿姨，粉丝对其常以阿姨相称<ref name="163-2016"/>。

2014年3月25日，[[凤凰卫视|凤凰卫视]][[开卷八分钟|开卷八分钟]]以专题特别介绍《民国纪事本末》<ref>{{cite web|title=刘仲敬：恶专制而恋大一统无异于爱苗条而不舍甜食 |url=http://phtv.ifeng.com/program/kjbfz/detail_2014_03/25/35107314_0.shtml |publisher=[[凤凰卫视|凤凰卫视]] |date=2014-03-25 |accessdate=2018-11-28}}</ref>。

2016年，刘仲敬应邀出席北京的演讲，因技术问题原本的现场演讲改为线上[[直播|直播]]，讲述[[古罗马|罗马]]历史的进程引起群众的注意。粉丝中流传其日均阅读120万字，以及拥有无需作笔记的超强记忆力，刘仲敬本人在接受采访时未予否认<ref name="163-2016">{{cite news |author=窦昊|title=刘仲敬：不说我是异端才奇怪 |url=http://news.163.com/16/0128/16/BEE8Q0UL00014Q4P.html |newspaper=山东商报 |publisher=[[网易|网易]] |date=2016-01-28}}</ref>。

== 评价 ==
[[华东师范大学|华东师范大学]]教授[[许纪霖|许纪霖]]称《民国纪事本末》为「奇人奇书」，还说<ref>{{cite web|title=不确定的历史？ ——一本奇书《民国纪事本末》引发的争议|url=http://www.bjnews.com.cn/book/2013/11/16/292956.html |publisher=[[新京报|新京报]] |date=2013-11-16 |archiveurl=https://web.archive.org/web/20181127232245/http://www.bjnews.com.cn/book/2013/11/16/292956.html |archivedate=2018-11-27 |dead-url=no}}</ref>：
{{cquote|这位刚过而立之年的学子是一位难得的怪才、学痴，读了这本书，深为震撼，个中评点如同老吏断狱，字字见血，妙语隽句，俯拾皆是。其大识大见打通古今中西，乃当今学界罕见之天才也。}}

[[同济大学|同济大学]]副教授王晓渔评刘仲敬《民国纪事本末》：“通过宪制的视野，刘仲敬对民国史有很多纵深的观察”<ref>{{cite web|title=跳出“党派叙事”——评刘仲敬《民国纪事本末》 |url=http://book.ifeng.com/shuping/detail_2014_07/03/104121_0.shtml |newspaper=东方早报 |author=王晓渔 |publisher=[[凤凰网|凤凰网]] |date=2014-07-03 |accessdate=2018-10-01}}</ref>。 

自由撰稿人李尔克称：“就思想性而言，刘仲敬也并非自诩的‘先知’，而只不过是以一套自创而又未必高明的语言重新包装、兜售别人的观点而已。”<ref>{{cite web |title=有一种“先知”让人如何评判？ |url=http://epaper.oeeee.com/epaper/C/html/2015-11/15/content_11410.htm |author=李尔克 |publisher=[[南方都市报|南方都市报]] |date=2015-11-15 |archiveurl=https://web.archive.org/web/20170910172728/http://epaper.oeeee.com/epaper/C/html/2015-11/15/content_11410.htm |archivedate=2017-09-10}}</ref>

[[河南大学|河南大学]]教师梅祖蓉表示刘仲敬《英国史》中译本有多处遗漏：“译本只保留了标题的事件排序，而原著的补充有的时间和事件提示全部不见。此外，在第5章至少有20处漏译以及其他的错译部份。”梅祖蓉还认为刘仲敬将原著改译成[[文言文|文言文]]的行为是在「着意卖弄」<ref>{{cite web|title=荒唐错漏何其多：谈刘仲敬译大卫·休谟《英国史》 |url=https://www.thepaper.cn/newsDetail_forward_1421888 |publisher=[[澎湃新闻|澎湃新闻]] |date=2016-01-12 |accessdate=2018-11-28}}</ref>。

[[北京师范大学|北京师范大学]]讲师田方萌认为刘仲敬的《安·兰德传》涉嫌大量抄袭：“凡是牵涉兰德生平的史料，作者都不提供出处，只在参考来源中列出几十种英文资料，而其中一本便是美国作家安妮·海勒的英文传记《{{link-en|安·兰德和她创造的世界|Ayn Rand and the World She Made}}》”<ref>{{cite web|url=http://www.infzm.com/content/125663 |title=劉仲敬《安·蘭德傳》抽檢報告 |author=田方萌 |date=2017-07-04 |publisher=[[南方周末|南方周末]] |archiveurl=https://web.archive.org/web/20180218211936/http://www.infzm.com/content/125663 |archivedate=2018-02-18}}</ref><ref name="jiemian2017">{{cite web |url=https://www.jiemian.com/article/1455771.html |title=《安·蘭德傳》抄襲指控背後 是保守派內部的思想對峙？|author=張之琪 |date=2017-07-08 |publisher=[[界面新聞|界面新聞]]}}</ref>。
