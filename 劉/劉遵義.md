'''劉遵義'''{{GBSHK}}{{JP}}（{{lang-en|'''Lawrence Lau'''}}，{{bd|1944年|12月12日|||catIdx=L刘}}），香港[[經濟學家|經濟學家]]，生於[[中國|中國]][[貴州|貴州]][[遵義市|遵義]]，籍貫[[廣東|廣東]][[汕頭|汕頭]][[潮阳|潮阳]]，後移居[[香港|香港]]。劉遵義的母親[[于志賢|-{于}-志賢]]是已故[[中華民國監察院|中華民國監察院]]院長[[于右任|于右任]][[姪女|姪女]]<ref>右老羅東避壽 偷閒流覽海山，聯合報，1955.04.13，第一版</ref><ref>劉世達夫人 -{于}-志賢病逝，香港工商日報，1960-01-30</ref>，妻子是香港已故歌手[[張國榮|張國榮]]的外甥女[[麥嘉軒|麥嘉軒]]。他自2004年7月1日出任[[香港中文大學|香港中文大學]]校長至[[2010年6月|2010年6月]]，並於2009年1月21日起出任[[香港行政會議|香港行政會議]]非官守成員。他也是第十一屆[[中國人民政治協商會議|全國政協]]經濟委員會委員。<ref>http://www.cppcc.gov.cn/page.do?pa=2c90829522e164290122e16d726c0040&guid=11W001892&og=2c90489523198f38012319c65d4a00b7{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>2010年7月1日，剛卸任中大校長的劉遵義，即就任[[中國中信集團公司|中國中信集團公司]]轄下的[[中信資本控股有限公司|中信資本控股有限公司]]副董事長<ref>{{cite news| title=劉遵義交棒沈祖堯 即任中信資本副董事長| url=http://hk.news.yahoo.com/article/100630/4/ixow.html| publisher=明報| date=2010年7月1日| deadurl=yes| archiveurl=https://web.archive.org/web/20100705124249/http://hk.news.yahoo.com/article/100630/4/ixow.html| archivedate=2010年7月5日}}</ref>。他也是[[中華民國|中華民國]][[中央研究院|中央研究院]][[中央研究院院士|院士]]。刘遵义教授现任[[香港中文大學（深圳）|香港中文大学（深圳）]]高等金融研究院理事会主席。

==簡歷==
===學術===
劉遵義1944年生於[[貴州|貴州]][[遵義市|遵義]]，在[[香港|香港]]接受教育，小學就讀[[聖保羅男女中學附屬小學|聖保羅男女中學附屬小學]]，中學畢業於[[聖保羅男女中學|聖保羅男女中學]]。後負笈[[美國|美國]]，1964年取得[[史丹福大學|史丹福大學]][[物理|物理]]及[[經濟|經濟]]學理學士學位，並先後於1966年及1969年取得[[加州大學柏克萊分校|加州大學柏克萊分校]]的[[經濟學|經濟學]]文學碩士和[[哲學博士|哲學博士]]學位。

劉遵義自1966年起任教於[[史丹福大學|史丹福大學]][[經濟|經濟]]系，1976年晉升為[[講座教授|講座教授]]，1992年出任該系首任[[李國鼎|李國鼎]]經濟發展[[講座教授|講座教授]]。他於1992年至1996年間擔任[[史丹福大學|史丹福大學]]亞太研究中心的共同主任(Co-Director)；1997年至1999年間出任[[史丹福經濟政策研究所|史丹福經濟政策研究所]]主任；2006年獲[[史丹福大學|史丹福大學]]頒授[[李國鼎|李國鼎]]經濟發展榮休講座教授銜。2004年起，劉遵義出任[[香港中文大學|香港中文大學]]校長，任期6年，現兼任[[香港中文大學|香港中文大學]][[藍饒富|藍饒富]]暨[[藍凱麗|藍凱麗]]經濟學[[講座教授|講座教授]]。

劉遵義專研[[經濟發展|經濟發展]]和[[經濟增長|經濟增長]]，以及包括[[中國|中國]]在內的[[東亞|東亞]]經濟。最早之[[中國|中國]][[計量經濟|計量經濟]]模型由他於1966年建立，自此他更不斷將之加以改良。劉遵義著有專書五種，曾於學術期刊發表[[論文|論文]]一百六十多篇。 

劉遵義現為 Phi Beta Kappa學會和 Tau Beta Pi學會會員、[[計量經濟學會|計量經濟學會]][[院士|院士]]、中華民國[[中央研究院|中央研究院]]院士、 Conference for Research in Income and Wealth 會員、[[劍橋大學|劍橋大學]][[劍橋大學丘吉爾學院|邱吉爾學院]]海外院士、[[中國社會科學院|中國社會科學院]]榮譽院士，以及[[國際歐亞科學院|國際歐亞科學院]]院士。他於1999年獲[[香港科技大學|香港科技大學]]頒授榮譽社會科學博士學位，2007年獲日本[[創價大學|創價大學]]榮譽博士學位、日本[[早稻田大學|早稻田大學]]榮譽法學博士學位，2010年獲臺灣[[國立中央大學|國立中央大學]]榮譽管理學博士學位，2011年獲[[復旦大學|復旦大學]]榮譽經濟學博士學位，復於2014年獲[[香港中文大學|香港中文大學]]榮譽法學博士學位。

劉遵義以其專長服務多所學術機構。他是[[上海社會科學院|上海社會科學院]]的名譽研究員，以及內地多所高等院校和學術機構的名譽[[教授|教授]]，包括[[中國科學院|中國科學院]]系統科學研究所、[[吉林大學|吉林大學]]、[[南京大學|南京大學]]、[[中國人民大學|中國人民大學]]、[[汕頭大學|汕頭大學]]、[[東南大學|東南大學]]，以及[[清華大學|清華大學]]經濟管理學院。他現任[[國際歐亞科學院|國際歐亞科學院]]中國科學中心副主席及[[中國金融學會|中國金融學會]]副會長，也是[[國家統計局|國家統計局]]的國際顧問，以及[[台北|台北]][[蔣經國|蔣經國]]國際交流基金的董事會成員。

===香港中文大學校長===
劉遵義擔任香港中文大學(簡稱中大)校長期間(2004年7月1日－2010年6月)，提出把大部分學系改以[[普通話|普通話]]及[[英語|英語]]授課，以增加大學的國際化程度。此外，劉遵義批准了五所新成員書院，以及「校園發展計劃」（最後採納[[凱達環球|凱達環球]]方案），以因應2012年中文大學本科回復四年制後，學生人數增加的問題。<ref>{{cite web |url=http://www.cusu.hk/cecg/blog/?cat=8 |title=存档副本 |accessdate=2010-02-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20100207130454/http://www.cusu.hk/cecg/blog/?cat=8 |archivedate=2010-02-07 }}</ref> 

劉遵義於[[2010年6月|2010年6月]]任滿退休，由[[逸夫書院|逸夫書院]]院長[[沈祖堯|沈祖堯]]教授擔任中文大學校長。

===公職===
劉遵義亦擔任多項公職，包括[[中國人民政治協商會議|中國人民政治協商會議]]第十一屆全國委員會委員、[[香港特別行政區政府|香港特別行政區政府]][[策略發展委員會|策略發展委員會]]委員、[[廉政公署|廉政公署]]貪污問題諮詢委員會委員、[[創新及科技督導委員會|創新及科技督導委員會]]委員、[[外匯基金諮詢委員會|外匯基金諮詢委員會]]及其轄下管治委員會和貨幣發行委員會之委員、[[授勳評審委員會|授勳評審委員會]]委員，並於2007年七月獲委任為[[太平紳士|太平紳士]]。此外，他也是[[中信資本控股有限公司|中信資本控股有限公司]]副董事長、[[香港科技園公司|香港科技園公司]]董事、[[中國海洋石油有限公司|中國海洋石油有限公司]]獨立非執行董事、台灣[[遠傳電信|遠傳電信]]股份有限公司獨立董事、[[新光金融控股股份有限公司|新光金融控股股份有限公司]]獨立監察人，以及美國[[大展集團|大展集團]]董事。<ref>http://news.mingpao.com/20100305/ggb1.htm</ref>

==爭議==
劉遵義教授在中大校長任內，有不少決策遭輿論、教職員與學生批評。

===哭中大事件===
{{main|哭中大事件}}
2005年初，一篇名為「哭中大」的文章，掀起香港中文大學學生對大學核心課程英語化的反對聲浪，並引發了一連串事件。

===劉遵義施政監察===
一群意圖影響大學發展的校友及同學發起了「[[劉遵義施政監察|劉遵義施政監察]]」網誌，記錄劉遵義任內受爭議之中大施政<ref>http://ourcuhk.blogspot.com/</ref><ref>{{cite web |url=http://hk.news.yahoo.com/article/100612/4/ila6.html |title=存档副本 |accessdate=2010-06-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20100615010412/http://hk.news.yahoo.com/article/100612/4/ila6.html |archivedate=2010-06-15 }}</ref>。這些人認為中大向[[董建華|董建華]]授予榮譽法學博士是[[政治獻媚|政治獻媚]]，所以破天荒地發起針對該校校長的監察組織。劉遵義被中央委任為「[[中國人民政治協商會議全國委員會香港地區委員|全國政協委員]]」時，「劉遵義施政監察」組織做了紅底白字的「'''''-{热烈祝贺刘遵义同志荣任全国政协委员}-'''''」簡體字，目的是嘲諷他「赤裸裸的政治分贓」。<ref>[http://news.sina.com/hk/mingpao/103-101-101-102/2008-02-05/15042654190.html 劉遵義任政協 中大生嘲政治分贓]，明報，2008年02月05日</ref>

===中大學生報事件===
{{main|中大學生報情色版事件}}
2007年5月，中大學生會出版的《[[中大學生報|中大學生報]]》[[情色|情色版]]被指刊登不雅內容，送往[[淫褻及不雅物品審裁處|淫褻及不雅物品審裁處]]進行評級。校方在淫審處進行評級前，即召開紀律委員會向全體《中大學生報》出版委員發警告信，指內容不雅及粗鄙違反大學學則，作出嚴重警告，並禁止在校內發佈有關刊物，其後被淫審處暫定評為「不雅」。2008年10月，[[高等法院|高等法院]]推翻淫審處的暫定不雅評級，校方拒絕撤回警告信，並指「同學應注意社會可接受的道德底線，當中沒有任何同學被處分」<ref>{{cite news| title=《明報》《中大學生報》平反 高院裁淫審處評級違例| url=http://hk.news.yahoo.com/article/081021/4/8u2n.html| publisher=[[明報|明報]]| date=2008-10-22| accessdate=2010-06-17| deadurl=yes| archiveurl=https://web.archive.org/web/20081101230141/http://hk.news.yahoo.com/article/081021/4/8u2n.html| archivedate=2008-11-01}}</ref>。

===拒絕安放新民主女神像===
[[File:抗議劉遵義_Protest_against_Lau_Juen-yee_in_Hong_Kong.jpg|thumb]]
2010年6月（[[六四事件|六四事件]]二十一週年前夕），行將卸任的劉遵義主持中大行政與計劃委員會，以「維持大學政治中立的原則」為由，拒絕讓新[[民主女神像|民主女神像]]在校園永久擺放<ref>{{cite news | url=http://hk.news.yahoo.com/article/100602/4/iduy.html | title=中大校方拒納民主女神 稱堅守政治中立 學生會竟斥其向中央示好 | date=03-06-2010 | work=[[明報|明報]] | deadurl=yes | archiveurl=https://web.archive.org/web/20100606235333/http://hk.news.yahoo.com/article/100602/4/iduy.html | archivedate=2010-06-06 }}</ref>。由於劉的身份是全國政協委員、香港行政會議非官守成員，他聲稱「大學應堅守政治中立」成為一些中大師生員工和社會各界一些人攻擊中大的原因。[[維園六四燭光晚會|維園六四燭光晚會]]之後，[[中大學生會|中大學生會]]和數千香港市民迎接民主女神像到中大，並不斷批評劉遵義校長及校方「[[起錨_(口號)|超錯]]」。<ref>{{cite web |url=http://hk.news.yahoo.com/article/100606/4/igbi.html |title=存档副本 |accessdate=2010-06-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20100611041643/http://hk.news.yahoo.com/article/100606/4/igbi.html |archivedate=2010-06-11 }}</ref>劉則聲稱，雖然他本人有政治立場，但大學校方是政治中立的，於[[民主女神像|民主女神像]]一事並沒有政治偏袒。<ref>{{cite web |url=http://hk.news.yahoo.com/article/100608/4/ihk5.html |title=存档副本 |accessdate=2010-06-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20100611173308/http://hk.news.yahoo.com/article/100608/4/ihk5.html |archivedate=2010-06-11 }}</ref>後來的中大校長[[沈祖堯|沈祖堯]]認為校方處理不當。<ref>{{cite web |url=http://hk.news.yahoo.com/article/100607/3/ih9p.html |title=存档副本 |accessdate=2010-06-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20100611180423/http://hk.news.yahoo.com/article/100607/3/ih9p.html |archivedate=2010-06-11 }}</ref>

===拖延安放孫中山像===
在新民主女神像事件爆發後，[[中國國民黨|中國國民黨]]創始人、[[中華民國|中華民國]]國父[[孫中山|孫中山]]先生的孫女[[孫穗芳|孫穗芳]]說，自己在2010年初在中大演講[[三民主義|三民主義]]思想時，（單方面）說會把一個孫中山像贈送給中大，以紀念[[辛亥革命|辛亥革命]]100週年，而銅像在5月送到中大時，劉遵義以自己離任在即為理由，要求在下半年再安放銅像，並要候任校長[[沈祖堯|沈祖堯]]教授上任後才決定。<ref>{{cite web |url=http://hk.news.yahoo.com/article/100610/4/ijux.html |title=存档副本 |accessdate=2010-06-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20100614222211/http://hk.news.yahoo.com/article/100610/4/ijux.html |archivedate=2010-06-14 }}</ref>

==私人生活==
劉遵義曾經歷兩段婚姻，育有一名兒子<ref>{{cite web |url=http://hk.news.yahoo.com/article/100111/4/g28m.html |title=存档副本 |accessdate=2010-06-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20100114164159/http://hk.news.yahoo.com/article/100111/4/g28m.html |archivedate=2010-01-14 }}</ref>。現任太太[[麥嘉軒|麥嘉軒]]是[[張國榮|張國榮]]的外甥女、其母為首任[[消費者委員會|消費者委員會]]總幹事張綠萍，生父為香港大學比較文學系前系主任亞巴斯（[http://en.wikipedia.org/wiki/Ackbar_Abbas Ackbar Abbas]）<ref>http://news.hkheadline.com/dailynews/headline_news_detail_columnist.asp?id=93166&section_name=wtt&kw=15</ref>，養父是前[[香港殖民地時期#香港政府|香港政府]][[運輸司|運輸司]]麥法誠。劉遵義與現任太太年齡相差22歲，兩人於2010年1月11日成婚。
