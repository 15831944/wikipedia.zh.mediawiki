'''劉正'''，又名'''亨利'''（英語：'''Henry LAU Ching'''），是一位任職香港中文大學日本研究學系的講師，他畢業於[[香港中文大學|香港中文大學]]，獲得文學碩士學位。<ref>{{cite web |url=http://www.jas.cuhk.edu.hk/zh-TW/people/mr-lau-ching |title=香港中文大學日本研究學系：劉正 }}</ref> 也是一位[[香港|香港]]的[[博客|博客]]及[[網台|網台]]主持。<ref>{{cite web |url=http://paper.wenweipo.com/2017/09/12/HS1709120027.htm |title=【街談網議】兩講師「抽水」 網民斥「非人也」|publisher=文匯報 |date=2017-09-12}}</ref><ref>{{cite web |url=http://paper.wenweipo.com/2018/02/05/HK1802050011.htm |title=【網議政事】咖喱飯又抽水 網民寸搶錢|publisher=文匯報 |date=2018-02-05}}</ref><ref>{{cite web |url=http://www.boyangu.com/2016/02/turmoil_mongkok/ |title=胡啟敢：魚蛋騷亂的幾點想法|date=2016-02-11}}</ref><ref>{{cite web |url=https://martinoei.com/article/tag/%E7%84%A1%E7%A5%9E%E8%AB%96%E8%80%85%E7%9A%84%E5%B7%B4%E5%88%A5%E5%A1%94 |title=黃世澤的筆記簿：無神論者的巴別塔 }}</ref> 他於2016年4月的《股掌世情》正式承認自己是「無神論者的巴別塔」博客。<ref>{{cite web |url=https://www.youtube.com/watch?v=Uhffpwzn6Fc&t=5m35s |title=無神論巴別塔與亨利/沈旭暉離港與陳智遠發炮〈股掌世情〉(5:35-5:50) |date=2016-04-25}}</ref>。立場原為泛民派，但近年已轉投本土派，主要批評泛民主派議會中的表現。
