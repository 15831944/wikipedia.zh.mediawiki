'''劉梓潔'''（[[英文|英文]]：{{lang|en|Marula Liu}}，{{bd|1980年|10月20日||}}<ref>{{cite journal|author=劉梓潔|date=2016年12月|title=〈小樽重遊日記〉|journal=《皇冠》|issue=754期|pages=55頁、57頁|publisher=鑫濤出版事業有限公司|location=臺北市|issn=1019-3774|quote=Day 3 / 十月二十日 / 祝我自己生日快樂。……今天剛滿三十六歲的我突然明瞭，……}}</ref>），[[台灣電影|台灣電影]][[導演|導演]]、[[作家|作家]]。[[彰化縣|彰化縣]][[田尾鄉|田尾鄉]]人。

== 生平 ==
[[國立台中女中|國立台中女中]]、[[台灣師範大學|台灣師範大學]]社教系新聞組畢業，[[清華大學|清華大學]][[台灣文學|台灣文學]]研究所碩士班肄業。曾任《誠品好讀》編輯、[[琉璃工房|琉璃工房]]文案、[[中國時報|中國時報]][[開卷|開卷]]週報記者。

== 作品 ==
=== 文學作品 ===
* 2010年 出版《[[父後七日|父後七日]]》散文集。
* 2012年 出版《此時此地》散文集。
* 2013年 出版《親愛的小孩》短篇小說集。
* 2014年 出版《遇見》短篇小說集。
* 2016年 出版《真的》長篇小說。
* 2017年出版《愛寫》散文集。
* 2018年出版《外面的世界》長篇小說。

=== 影視作品 ===
* 2004年 電視劇本〈野百合〉（與[[蔡宗翰|蔡宗翰]]合著）入選[[公共電視|公共電視]]百萬原創劇本。
* 2010年 擔任電影[[父後七日|《父後七日》]][[編劇|編劇]]，並與[[王育麟|王育麟]]共同[[導演|執導]]。
* 2014年 偶像劇《[[:徵婚啟事_(電視劇)|徵婚啟事]]》編劇統籌。
* 2016年 《滾石愛情故事》編劇統籌。

== 得獎紀錄 ==
* 2003年 小說《失明》獲[[聯合文學|聯合文學]]小說新人獎。
* 2006年 四千字散文〈[[父後七日|父後七日]]〉獲《自由時報》[[林榮三文學獎|林榮三文學獎]]散文首獎。
* 2010年 電影《[[父後七日|父後七日]]》獲第十二屆[[台北電影節|台北電影節]]最佳編劇。
* 2010年 電影《[[父後七日|父後七日]]》獲第四十七屆[[金馬獎|金馬獎]]最佳改編劇本。
* 2016年 長篇小說《真的》獲香港亞洲週刊年度十大小說。
