'''劉仲-{恒}-'''（{{lang-en|Dr. Kevin Lau Chung-Hang}}，{{bd|1981年|12月22日}}），香港放射科專科醫生<ref>{{cite web |url=https://hk.on.cc/hk/bkn/cnt/news/20180831/bkn-20180831063024152-0831_00822_001_cn.html |title=Dr.東：3大造影檢查邊款啱？放射科醫生話你知 |publisher=[[ON.CC|東網]] |date=2018-08-31 |accessdate=2019-02-04}}</ref>及商人。

== 簡歷 ==
劉仲-{恒}-於1986年隨父母移民到[[加拿大|加拿大]][[多倫多|多倫多]]居住{{noteTag |name=劉仲-{恒}-個人資料及相關學歷 |關於劉仲-{恒}-個人資料及相關學歷，可見於下方「外部連結」中的社交網站。}}，1998年因[[亞洲金融危機|亞洲金融危機]]導致家庭經濟拮据，兩人決定離婚，而他則回流入讀[[港島中學|港島中學]]<ref name=K1>{{cite web |url=http://hd.stheadline.com/news/daily/hk/541703/ |title=歷巨變重新領悟人生 暖男醫生大難不死 |publisher=[[頭條日報|頭條日報]] |date=2017-02-17 |accessdate=2019-02-04}}</ref>；1999 - 2005年修讀[[香港大學李嘉誠醫學院|香港大學李嘉誠醫學院]][[內外全科醫學士|內外全科醫學士]]課程期間曾一度留班{{noteTag |name=劉仲-{恒}-個人資料及相關學歷}}<ref name=K1/>，畢業後獨自去[[美國|美國]][[史丹福大學|史丹福大學]]、[[杜克大學|杜克大學]]及[[北卡羅萊納大學|北卡羅萊納大學]]繼續深造。他擁有[[英國皇家|英國皇家]]放射科學院、香港[[放射科|放射科]]學院、香港醫學科學院等三間院校的醫學專業資格，2015及2016年亦分別取得[[香港大學|香港大學]]公共衛生碩士與公共管理碩士學位<ref>{{cite web |url=https://trinitymedical.com.hk/zh/%E5%9C%98%E9%9A%8A/ |title=劉仲-{恒}-醫生 - Trinity Medical Centre 全仁醫學診斷中心 |publisher=[[全仁醫學診斷中心|全仁醫學診斷中心]] |date=2017}}</ref>。

2009年，劉仲-{恒}-往[[泰國|泰國]]潛水的時候遇上車禍，引致[[腦出血|腦出血]]且陷入昏迷狀態，須立即動手術救治<ref name=K1/>。他曾任職[[廣華醫院|廣華醫院]]放射診療科，從事顧問醫生工作<ref name=K2>{{cite web |url=http://dailynews.sina.com/bg/ent/hktwstar/sinacn/20111226/15383016418.html |title=馬時亨女兒婚宴黎姿隨夫出席 伏明霞子女當花童 |publisher=[[北京新浪網|北京新浪網]] |date=2011-12-26 |accessdate=2019-02-04}}</ref>，至2017年便和朋友鄭文輝選擇在[[中環|中環]][[娛樂行|娛樂行]]設立全仁醫學診斷中心<ref>{{cite web |url=https://hk.news.appledaily.com/local/realtime/article/20170110/56152270 |title=【李八方online】肥馬女婿中環開檔 高官名醫齊撐場 |publisher=[[蘋果日報_(香港)|蘋果日報]] |date=2017-01-10 |accessdate=2019-02-04}}</ref>；現時也擔任多個政府機構委員，包括[[扶貧委員會|扶貧委員會]]與社創基金專責小組、[[婦女事務委員會|婦女事務委員會]]<ref>{{cite web |url=https://www2.hkej.com/instantnews/current/article/1474025/%E9%99%B3%E5%A9%89%E5%AB%BB%E7%AD%89%E7%8D%B2%E4%BB%BB%E5%A9%A6%E5%A5%B3%E4%BA%8B%E5%8B%99%E5%A7%94%E5%93%A1%E6%9C%83%E6%88%90%E5%93%A1 |title=陳婉嫻等獲任婦女事務委員會成員 |publisher=[[信報財經新聞|信報財經新聞]] |date=2017-01-14 |accessdate=2019-02-04}}</ref>、[[關愛基金|關愛基金]]專責小組、兒童發展基金督導委員會、暴力及執法傷亡賠償委員會、行業培訓諮詢委員會（安老服務業）、[[促進母乳餵哺委員會|促進母乳餵哺委員會]]<ref>{{cite web |url=https://hk.on.cc/hk/bkn/cnt/news/20170210/bkn-20170210015827067-0210_00822_001_cn.html |title=政Whats噏：刘仲-{恒}-夫妇身体力行撑母乳喂哺 |publisher=東網 |date=2017-02-10 |accessdate=2019-02-04}}</ref>等，又常常參加慈善活動<ref name=K1/><ref>{{cite web |url=http://www.metrodaily.hk/metro_news/%E5%B0%88%E8%A8%AA%E9%A6%AC%E6%99%82%E4%BA%A8%E5%A5%B3%E5%A9%BF%E5%8A%89%E4%BB%B2%E6%81%92-%E5%8A%A9%E7%97%85%E7%AB%A5%E5%A4%A2%E6%88%90%E7%9C%9F/ |title=專訪馬時亨女婿劉仲-{恒}- 助病童夢成真 |publisher=[[都市日報|都市日報]] |date=2017-08-02 |accessdate=2019-02-04}}</ref>。

== 個人生活 ==
2009年，劉仲-{恒}-經友人介紹下，認識了[[港鐵|港鐵]]董事局非執行主席[[馬時亨|馬時亨]]長女[[馬露明|馬露明]]，兩人同為[[基督徒|基督徒]]，對談生論死方面十分投契，可直至2011年1月才開始拍拖<ref name=K2/>，並於同年12月26日在[[君悅酒店|君悅酒店]]註冊結婚<ref name=K1/>，共育有一子劉智-{恒}-（Christian）<ref>{{cite web |url=https://www.mpweekly.com/entertainment/focus/local/%E5%90%8D%E4%BA%BA%E5%B0%88%E7%B7%9A/20170128-35323 |title=岳丈馬時亨幫手湊仔　劉仲-{恒}-樂助病童圓心願 |publisher=[[明報周刊|明報周刊]] |date=2017-01-28 |accessdate=2019-02-04}}</ref><ref>{{cite web |url=http://www.ohpama.com/173583/pamajunior/%E3%80%90%E9%A6%AC%E6%99%82%E4%BA%A8%E5%A5%B3%E5%A9%BF%E3%80%91%E5%8A%89%E4%BB%B2%E6%81%92%E9%86%AB%E7%94%9F%EF%BC%9A%E5%B0%8A%E9%87%8D%E5%AD%A9%E5%AD%90%E4%B9%9F%E6%98%AF%E4%B8%80%E5%80%8B%E4%BA%BA/ |title=【馬時亨女婿】劉仲-{恒}-醫生：尊重孩子也是一個人 |publisher=[[Oh!_爸媽|Oh! 爸媽]] |date=2017-04-07 |accessdate=2019-02-04}}</ref><ref>{{cite web |url=http://paper.wenweipo.com/2017/05/12/RW1705120002.htm |title=劉馬露明 Christine Ma-Lau |publisher=[[香港文匯報|香港文匯報]] |date=2017-05-12 |accessdate=2019-02-04}}</ref>。
