'''劉康'''（{{lang-en|Lau Hong}}，{{bd|2001年}}），香港學生。因外貌與[[余澎杉|余澎杉]]相似，且同為持不同政見者，故此有「香港余澎杉」之稱。2017年與行政長官林鄭月娥合照時，展示「香港獨立」標語，抗議香港政府不斷打壓獨立運動及言論自由。同年，劉康於[[連儂牆|連儂牆]]被警察拘捕。

==生平==
劉康就讀於[[葵涌循道中學|葵涌循道中學]]，為[[香港獨立|香港獨立]]支持者。2016年新東補選時本土思潮興起，啟發劉康的港獨主張。在政治主張上，劉康強調港人和中國人是不同民族，文化和價值觀有差異，故有必要做好港中區隔。<ref>[https://hk.news.appledaily.com/local/realtime/article/20180414/58072711 蘋果日報:曾與林鄭合照展示港獨標語 中五生劉康被控涉管有仿製火器  ，2018年4月14日]</ref>

==事件== 
====在和行政長官合照時高舉香港獨立標語====
2017年11月16日，於第21屆明報校園記者開學禮之合照環節中，劉康於行政長官[[林鄭月娥|林鄭月娥]]後方，高舉「香港獨立」標語，抗議香港政府不斷打壓獨立運動及言論自由。<ref>{{cite_news|url= https://www.hk01.com/社會新聞/134151/政壇諸事町-突襲-林鄭成功-中五生合照時展示-港獨-標語 |title=【政壇諸事町】「突襲」林鄭成功　中五生合照時展示「港獨」標語|work=《[[香港01|香港01]]》|date=2017-11-16|accessdate=2018-09-24}}</ref><ref name = "appledaily1">{{cite_news|url= https://hk.news.appledaily.com/local/daily/article/20171124/20224042 |title= 撐港獨中五生遭校方警告|work=《[[蘋果日報_(香港)|蘋果日報]]》|date=2017-11-24|accessdate=2018-09-24}}</ref>

事後，葵涌循道中學校方警告劉康，並要求見家長，又表明，若再次於林鄭月娥後方，舉「香港獨立」標語，就會記過。
<ref name = “7+1-1”>{{cite_news|url= https://www.hk01.com/社會新聞/136256/突襲示威學生遭校方警告-團體到校聲援-批校方政治打壓|title=突襲示威學生遭校方警告　團體到校聲援　批校方政治打|work=《[[香港01|香港01]]》|date=2017-11-24|accessdate=2018-09-24}}</ref><ref>{{cite_news|url= https://hk.news.appledaily.com/local/realtime/article/20171124/57499670 |title= 學生校外派港獨傳單 遭教局嚴厲譴責 親中團體踩場對罵|work=《[[蘋果日報_(香港)|蘋果日報]]》|date=2017-11-24|accessdate=2018-09-24}}</ref>劉康認為，示威是應有的表達權利，不認為當日抗議港府是有錯，並對校方感到不滿。劉表示，不會怕被打壓而放棄示威權，並計劃於11月24日於葵涌循道中學校外，派發宣傳香港獨立之單張。<ref name = "appledaily1"/>

11月24日，[[香港民族陣線|香港民族陣線]]及[[學生動源|學生動源]]於葵涌循道中學外，聲援劉康，並批評校方政治打壓，干預言論自由。<ref name = “7+1-1”/>同時，校方就以所謂安全理由，超過2小時不讓劉康離開學校。劉康批評，校方做法不合理。<ref>{{cite_news|url= https://hk.news.appledaily.com/local/daily/article/20171125/20225138 |title= 撐港獨中五生一度被禁放學|work=《[[蘋果日報_(香港)|蘋果日報]]》|date=2017-11-25|accessdate=2018-09-24}}</ref>

====涉藏有氣槍被捕====
在[[立法會|立法會]]修改議事規則期間，劉康在立法會附近的[[連儂牆|連儂牆]]因藏有氣槍被捕。<ref>[https://www.hk01.com/%E6%94%BF%E6%83%85/178270/%E8%88%87%E6%9E%97%E9%84%AD%E5%90%88%E7%85%A7%E6%99%82%E5%B1%95%E7%A4%BA%E6%B8%AF%E7%8D%A8%E6%A8%99%E8%AA%9E-%E4%B8%AD%E4%BA%94%E7%94%9F%E5%8A%89%E5%BA%B7%E6%B6%89%E8%97%8F%E6%9C%89%E4%BB%BF%E8%A3%BD%E6%A7%8D%E6%A2%B0%E8%A2%AB%E6%8D%95 香港01:與林鄭合照時展示港獨標語　中五生劉康涉藏有仿製槍械被捕 , 2018年4月14日]</ref>事後劉康被控告一項藏有仿製火器罪，於同年8月22日被裁定藏有仿製火器罪名成立，判入[[壁屋懲教所|壁屋懲教所]]索閱感化、勞教中心、教導所、青少年罪犯評審委員會報至9月12日。劉康透過律師表示，這次是因政治理念而受到違反[[社會契約|社會契約]]的政府打壓。[[青年新政|青年新政]]召集人[[梁頌恆|梁頌恆]]表示，對裁判結果感到失望，一名16歲的中學生帶着「BB彈槍」竟然被告仿製火器罪名，質疑是否全香港的中學生也有機會被定此罪名。案件定於9月12日判刑。<ref>[http://hk.on.cc/hk/bkn/cnt/news/20180605/bkn-20180605112957244-0605_00822_001.html 東網:涉立會外藏氣槍　港獨青年劉康否認控罪 , 2018年6月4日]</ref>最終法庭判處他18個月感化令，並需參與小組計劃活動和心理輔導。<ref>[https://m.mingpao.com/ins/instantnews/web_tc/article/20180912/s00001/1536725271915 青年立法會外藏氣槍罪成　判感化18個月 (12:16) 2018年9月12日]</ref>
