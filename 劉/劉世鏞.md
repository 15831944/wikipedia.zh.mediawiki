'''劉世鏞'''（[[英文|英文]]：'''Lau Sai Yung'''<ref>{{web reference|URL=http://www.scaacpa.org.hk/_cms/files/pdf/424.pdf|title=會員名單|work=香港華人會計師公會|year=2015年|date=8月7日}}</ref>)，香港會計師（1984年創立劉世鏞會計師行）、香港中文大學校友評議會前主席<ref name="立法會CB(2)993/09-10(03)號文件">[http://www.legco.gov.hk/yr09-10/chinese/hc/papers/hc0226cb2-993-3-ec.pdf 香港立法會《立法會CB(2)993/09-10(03)號文件》]</ref>[[香港中文大學|香港中文大學]][[崇基學院|崇基學院]]前校董會成員、[[香港中文大學校友會聯會陳震夏中學|香港中文大學校友會聯會陳震夏中學]]校董會主席兼校監<ref name="榮譽院士劉世鏞先生讚辭">[https://www.cpr.cuhk.edu.hk/honfellow/Citation/chi/5_lausaiyung.pdf 香港中文大學〈榮譽院士劉世鏞先生讚辭〉]</ref>、[[香港中文大學校友會聯會張煊昌中學|香港中文大學校友會聯會張煊昌中學]]榮譽校董<ref name="廿五周年校慶聚餐席位編排">[http://www.tcss.edu.hk/chinese/25/file/list.pdf 香港中文大學校友會聯會張煊昌中學〈廿五周年校慶聚餐席位編排〉]</ref>，香港中文大學校友會聯會張煊昌學校前校董和香港中文大學校友會聯會張煊昌幼稚園榮譽校董。<ref name="榮譽院士劉世鏞先生讚辭" />

== 背景 ==
1967年，劉世鏞畢業於[[金文泰中學|金文泰中學]]，其後於1971年完成[[香港中文大學|香港中文大學]][[崇基學院|崇基學院]]工商管理學士（會計及財務）學位。<ref name="十年辛苦也尋常">[http://www.alumni.cuhk.edu.hk/magazine/200512/htm/p05-16.htm 香港中文大學《中大校友》第5至16頁〈十年辛苦也尋常〉，2005年12月]</ref>五年後，即1976年，他成為[[英國|英國]]特許秘書及行政人員公會會員。在1979年至1984年期間，劉氏也成為[[英國|英國]]特許公認會計師公會會員及資深會員。<ref name="榮譽院士劉世鏞先生讚辭" />之後在1983年至1985年期間，他擔任[[香港中文大學|香港中文大學]]校友會評議會常務委員會副主席，[[崇基學院|崇基學院]]校董以及校友會會長。又開始出任[[香港中文大學|香港中文大學]]校友會聯會教育基金會有限公司[[董事|董事]]（2011至2014年）<ref name="香港中文大學校友會聯會教育基金會30周年紀念特刊 1982-2012">[http://www.tcps.edu.hk/CustomPage/21/CUF/CUF_30_Anniversary.pdf 香港中文大學校友會聯會《香港中文大學校友會聯會教育基金會30周年紀念特刊 1982-2012》]</ref>及義務司庫一職（現時為香港順龍仁澤基金會義務司庫<ref name="雲南騰冲大壩完全小學重建計劃香港順龍仁澤基金會捐資建校監理工作報告">[http://cpecf.org.hk/docu/11_Report_Yunnan_SchoolRebuild_20120306.pdf 中華精忠慈善基金《雲南騰冲大壩完全小學重建計劃香港順龍仁澤基金會捐資建校監理工作報告》]</ref><ref name="恒管通訊6 • 2016">[https://www.hsmc.edu.hk/old/images/People/Media/Newsletter/201606.pdf 恒生管理學院《恒管通訊6 • 2016》]</ref>）。

另外，劉世鏞熱心教育事業。他於1995年為[[崇基學院|崇基學院]]設立「崇基校友至善獎學金」以獎勵成績優異的學生。同年也發起「小扁擔勵學行動」以扶助內地偏遠地區貧困農村基礎教育發展<ref name="恒管通訊6 • 2016" />。劉世鏞於2004年成立才礇會計師事務所有限公司，並再進修考獲[[香港會計師公會|香港會計師公會]]破產管理文憑。同時，劉氏亦是加多寶集團執行董事及財務總經理<ref>{{web reference|URL=http://www.jdbchina.com/cn/new/jdb_news.asp?id=208|title=福建加多寶開工慶典儀式|work=加多寶集團|year=2007年|date=4月5日}}</ref>、[[路勁基建有限公司|路勁基建有限公司]]獨立非執行董事<ref>{{web reference|URL=http://www.hkexnews.hk/listedco/listconews/sehk/2017/1016/LTN20171016380_C.pdf|title=截至二零一七年九月三十日止九個月未經審核營運數據|work=路勁基建有限公司|year=2007年|date=10月16日}}</ref><ref name="董事名單與其角色和職能">{{Cite web |url=http://www.roadking.com.hk/zh_TW/clist%20of%20directors%20and%20their%20role%20and%20function%20May%202013.pdf |title=路勁基建《董事名單與其角色和職能》 |access-date=2017-11-05 |archive-url=https://web.archive.org/web/20140514092450/http://www.roadking.com.hk/zh_TW/clist%20of%20directors%20and%20their%20role%20and%20function%20May%202013.pdf |archive-date=2014-05-14 |dead-url=yes }}</ref>、[[新疆|新疆]]天山毛紡織股份有限公司非執行董事<ref name="新疆天山毛紡織股份有限公司第六屆董事會第十六次會議決議公告">[http://q.stock.sohu.com/gg/20161188476763.pdf 新疆天山毛紡織《新疆天山毛紡織股份有限公司第六屆董事會第十六次會議決議公告》]</ref>以及其他[[香港|香港]]及海外成立的[[有限公司|有限公司]]董事職務。

除此之外，劉世鏞先後成為[[香港會計師公會|香港會計師公會]]會員及資深會員（1980年、2004年）、香港稅務學會會員（1982年）、香港特許秘書公會會士（1994年）、香港華人會計師公會會員（2001年）及[[英格蘭|英格蘭]]與[[威爾斯|威爾斯]]特許會計師公會會員（2005年）。<ref name="榮譽院士劉世鏞先生讚辭" />直到2006年，他成為[[香港中文大學|香港中文大學]]第五屆榮譽院士<ref name="香港中文大學第五屆榮譽院士頒授典禮">[http://www.cuhk.edu.hk/cpr/pressrelease/060514c.htm 香港中文大學〈香港中文大學第五屆榮譽院士頒授典禮〉，2006年]</ref>。

== 爭議 ==
劉世鏞由2005年起擔任[[香港中文大學|香港中文大學]]校友評議會主席。但在2008年競逐連任時，他錯誤地讓其中一位提名人以筆名填寫表格，結果被質疑提名有效性。最後事件不了了之。他仍然繼續當主席。<ref>{{web reference|URL=http://www.inmediahk.net/%E6%96%B0%E6%B0%91%E5%A5%B3%E8%A8%8E%E8%AB%96%E8%A2%AB%E8%85%B0%E6%96%AC-%E9%81%B8%E8%88%89%E4%B8%8D%E5%85%AC%E7%85%A7%E5%8E%BB%E2%80%94%E2%80%94%E7%9B%B8%E9%9A%94%E5%85%AB%E5%80%8B%E6%9C%88%E7%9A%84%E4%B8%AD%E5%A4%A7%E8%A9%95%E8%AD%B0%E6%9C%83%E6%9C%83%E5%93%A1%E5%A4%A7%E6%9C%83|title=黃俊邦〈新民女討論被腰斬 選舉不公照去——相隔八個月的中大評議會會員大會〉|work=獨立媒體（香港） inmediahk|year=2011年|date=2月27日}}</ref>2010年，他更被指在未經常委會議決下，以[[香港中文大學|香港中文大學]]校友評議會主席身份向[[立法會|立法會]]去信支持委任[[陳茂波|陳茂波]]為[[香港中文大學|香港中文大學]]校董。去信中聲稱是代表校友評議會中多數校友的身份。而事後他辯稱是以個人身份去信。<ref>{{web reference|URL=http://www.inmediahk.net/%E4%B8%AD%E5%A4%A7%E8%A9%95%E8%AD%B0%E6%9C%83%E4%B8%BB%E5%B8%AD%E6%94%B9%E9%81%B8%EF%BC%8C%E8%AE%8A%E7%9B%B8%E6%A0%A1%E5%8F%8B%E7%9B%B4%E9%81%B8%E6%A0%A1%E8%91%A3|title=黃俊邦〈中大評議會主席改選，變相校友直選校董〉|work=獨立媒體（香港） inmediahk|year=2011年|date=7月3日}}</ref>
