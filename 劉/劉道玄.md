'''劉道玄'''（{{bd|1986年|3月14日||catIdx=P}}），臺灣[[男演員|男演員]]，出生於[[臺灣|臺灣]][[臺北市|臺北市]]，2012年與[[陳妍希|陳妍希]]拍攝寶島眼鏡男主角出道<ref>{{cite web |language=zh-TW |url=https://www.youtube.com/watch?v=jYtinqzITDQ&feature=youtu.be | title=寶島眼鏡廣告 |publisher=[[Youtube|Youtube]] |date=2013-05-03 |accessdate=2018-03-10}}</ref>，並開始參與多部廣告、微電影、戲劇男主角演出。

== 經歷 ==
===2013年===
* 受邀擔任台北兩年度捷運局「推廣台北觀光大使」形象廣告男主角，並被相中主持旅遊外景 「星星帶你去旅行」第三季與第四季的外景主持，也因此受邀主持[[吳奇隆|吳奇隆]]能量發表會。
===2014年===
* 參與電影「大樓有鬼」<ref>{{cite web |language=zh-TW |url=https://www.facebook.com/search/top/?q=%E5%A4%A7%E6%A8%93%E6%9C%89%E9%AC%BC | title=大樓有鬼粉絲專頁 |publisher=[[www.facebook.com|www.facebook.com]] |date=2014-10 |accessdate=2018-03-10}}</ref>男主角演出，獲得洛杉磯[[亞太影展|亞太影展]]與[[金穗獎|金穗獎]]的肯定，因「星星帶你去旅行」節目特輯受邀專訪「[[吳奇隆|吳奇隆]]」四川特輯與熊貓能量發表會。<ref>{{cite web |language=zh-cn |url=https://3g.163.com/ent/article/9SMCAMS2000333J5.html#adaptation=pc&refer=https%3A%2F%2Fm.baidu.com%2Ffrom%3D1099a%2Fbd_page_type%3D1%2Fssid%3D0%2Fuid%3D0%2Fbaiduid%3DFEF17EE667D2C5C9573C3130E10E96C1%2Fw%3D0_10_%2Ft%3Dzbios%2Fl%3D1%2Ftc%3Fref%3Dwww_zbios%26pu%3Dsz%25401320_480%252Ccuid%254038D07475FFFC0CFBA16CCA1F8637447EACBD0E28DFCKDEKPRPE%252Ccua%25401242_2208_iphone_9.3.0.11_0%252Ccut%2540iPhone8%25252C2_10.1.1%252Cosname%2540baiduboxapp%252Cctv%25401%252Ccfrom%25401099a%252Ccsrc%2540home_box_txt%252Ccud%2540NzI0NTE1N0YtNTBFNy00MDM5LUI0REQtNDNCMkY1RjE4RkEx%252Cta%2540zbios_1_10.1_6_9.3%252Cusm%25402%252Cvmgdb%25400020100228y%26lid%3D9610805263569329521%26order%3D7%26fm%3Dalhm%26dict%3D-1%26tj%3Dh5_mobile_7_0_10_title%26w_qd%3DIlPT2AEptyoA_yk5rBEc8gO6FSCXt9h0nzRisg35%26tcplug%3D1%26sec%3D28187%26di%3D449fd0d7a28350cf%26bdenc%3D1%26tch%3D124.65.56.1187.1.548%26nsrc%3DIlPT2AEptyoA_yixCFOxXnANedT62v3IIxiJ_85CQXSxokDyqRKuVBEsRG3aAifTUS3cdCPQpxZijUPKYkMo7RJ0ra6nmW-Eli_rvKChbM8_U1e%26eqid%3D856070778c812000100000015aa26fa2%26wd%3D%26clk_info%3D%257B%2522srcid%2522%253A1599%252C%2522tplname%2522%253A%2522h5_mobile%2522%252C%2522t%2522%253A1520594857541%252C%2522xpath%2522%253A%2522div-a-h3%2522%257D | title=主持人劉道玄首訪偶像吳奇隆 |publisher=[[網易|網易]] |date=2014-05-20 |accessdate=2018-03-10}}</ref><ref>{{cite web |language=zh-cn |url=http://ent.ifeng.com/a/20140729/40207698_0.shtml | title=劉道玄獲吳奇隆授鴕鳥功四爺零距離接觸粉絲 |publisher=[[北青網|北青網]] |date=2014-07-29 |accessdate=2018-03-10}}</ref>
===2015年===
* 獨挑大梁主持TVBS年度製作優質旅遊行腳節目-「背包客來尞<ref>{{cite web |language=zh-TW |url=http://campaign.tvbs.com.tw/activity/backpackers/ |title=背包客來尞官方網站 |publisher=[[Copyright_©_2016_TVBS_Media_Inc._All_Rights_Reserved.|Copyright © 2016 TVBS Media Inc. ===2016年===
* 與[[sony|sony]]品牌合作拍攝ActionCamX3000愛玩不設限10種超狂拍法4K廣告男主角，挑戰飛行傘、潛水、衝浪、泡溫泉、釣魚、攀樹、騎車各項極限運動。並到韓國拍攝愛奇藝宣傳片男主角-男神之愛。
===2017年===
* 在[[蘇州|蘇州]]拍攝[[愛奇藝|愛奇藝]]網劇-瑪莉蘇遇上杰克蘇飾演-禁慾系男神「李克力」
* 在[[北京|北京]]拍攝電影-暗遊記不良廚娘-男主角-「伍六七」
* 在[[佛山|佛山]]拍攝電影-無敵而已-男主角-「李默森」
* 受邀吉林觀光局擔任觀光大使-推廣吉林文化
===2018年===
* 在[[象山|象山]]拍攝電影-斗墓占花魁男主角-「花易天」
* 受邀寧夏觀光局擔任觀光大使-推廣寧夏文化
