'''劉交'''（{{Bd|？||前178年}}），字'''游'''，沛郡丰邑人，[[漢高祖|漢高祖]]劉邦之弟，被封為楚王，諡曰元王。劉交是[[荀子|荀子]]的學生[[浮丘伯|浮丘伯]]門下弟子，他曾為《詩經》作傳注，號为《元王詩》。

== 生平 ==
劉交是劉氏四兄弟中文化程度最高的一個。年輕時即好讀書，而且多才多藝，有大志。曾與魯人[[穆生|穆生]]、[[白生|白生]]、[[申公|申公]]一起到[[荀子|荀子]]的學生[[浮丘伯|浮丘伯]]門下學習《[[詩經|詩經]]》。後因秦始皇[[焚書坑儒|焚書坑儒]]，才被迫離去。

在兄弟四人當中，劉交的思想與三哥劉邦最為接近。劉邦起事後，劉交跟隨劉邦，是劉邦最為信任的人。入關後，劉邦封劉交為文信君。之後劉交又隨劉邦轉戰各地。因此劉交也是漢王朝的締造者之一。

漢高祖六年（西元前201年），劉邦在[[消滅異姓王風潮|消滅異姓王風潮]]中偽裝到[[雲夢澤|雲夢澤]]遊玩，召見楚王[[韓信|韓信]]，並立即廢之為淮陰侯，把楚國一分為二，[[劉賈|劉賈]]為荊王，劉交為楚王。同年劉交到楚國就封。楚國的都城為[[彭城|彭城]]（今[[江蘇|江蘇]][[徐州|徐州]]）。

楚王劉交的家族，在所有的西漢劉姓諸王中，是一個非常重要的家族。楚王國轄[[東海郡|東海郡]]、[[彭城郡|彭城郡]]和[[薛郡|薛郡]]，3郡36縣，是各王國中僅次於齊國的大國。

另外，他的家族自楚元王開始就一直有濃厚的學術氣質。楚元王到了封國以後，派自己的同學白生、穆生、申公和自己的兒子[[劉郢客|劉郢客]]到[[長安|長安]]向自己的老師浮丘伯學習。他自己本人則與其他兒子們一起誦讀儒家經典，也為《詩經》作傳注，號为《元王詩》，是當時《詩經》學的一個重要派別。劉郢客因學有所成被[[漢文帝|漢文帝]]任命為[[宗正|宗正]]。在劉交的治理下，楚國成為當時最為重要的學術中心。

漢高后八年（前180年）[[呂太后|呂太后]]去世之後，呂家外戚掌握[[朝廷|朝廷]]大權，楚元王會同齊王[[刘襄_(齐王)|劉襄]]、朱虛侯[[刘章_(城阳王)|劉章]]兄弟起兵，參與[[誅滅諸呂|誅滅諸呂]]的兵變。

在漢文帝元年（前179年）去世，在位23年，[[諡號|諡號]]'''元'''。

楚元王劉交的後裔是中國劉姓中最为繁盛的一支，號稱[[彭城劉氏|彭城劉氏]]。隋唐時期[[曹州南华劉氏|曹州南华劉氏]]也是楚元王的後裔。

== 儿子 ==
*[[劉辟非|劉辟非]]：王太子，早卒。
*[[劉郢客|劉郢客]]：初封為上邳侯，漢朝初定，受其父派遣，往長安師從[[浮丘伯|浮丘伯]]，學有所成，被任命為[[宗正|宗正]]。楚元王去世後，繼任為[[楚王|楚王]]，是為楚夷王。
*[[劉禮|劉禮]]：平陸侯，漢文帝時為[[宗正|宗正]]。匈奴入寇，以宗正出任將軍，與祝茲侯[[徐厲|徐厲]]、條侯[[周亞夫|周亞夫]]共備匈奴。吳楚[[七國之亂|七國之亂]]後被立為楚王，是為楚文王。
*[[劉富|劉富]]：初封休侯，[[七國之亂|七國之亂]]時被廢，被从宗室中除名。[[七國之亂|七國之亂]]結束後，改封為[[紅侯|紅侯]]，恢復宗室身份。死後諡為“懿侯”。
*[[劉歲|劉歲]]：沈猷夷侯。
*[[劉埶|劉埶]]：宛朐侯，參與[[七國之亂|七國之亂]]，失敗被殺。
*[[劉調|劉調]]：棘樂敬侯。

== 后裔 ==
*[[劉隗|劉隗]]：東晉時名將。
*[[劉牢之|劉牢之]]：東晉時名將。
*[[劉裕|劉裕]]：南北朝时期劉宋的开国皇帝。
