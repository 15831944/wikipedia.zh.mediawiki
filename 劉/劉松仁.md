'''劉松仁'''（{{lang-en|Damian Lau Chung Yan}}，{{bd|1949年|10月14日}}），香港资深男艺人，前[[亞洲電視|亞洲電視]]及[[電視廣播有限公司|無綫電視]]藝人。他自出道以來不喜歡拍照、過社交生活以及接受訪問，所以他每當受訪時都會感到緊張。坊間亦因此較少有關於他的個人專訪。{{noteTag|關於劉松仁不喜歡拍照、過社交生活以及接受訪問的資料出自香港電台第一台廣播節目《守下留情》於2019年4月8日播出的劉松仁訪問。}}

==背景==
劉松仁為父親長房的兒子，在家中排行第五<ref>{{cite web |url=https://www.ihktv.com/sudden-902-damian.html |title=春夢……了無痕 劉松仁 |last= |first= |date=2012-11-9 |website= |publisher=忽然1周 第902期 |access-date= 2019-4-8 |quote=}}</ref>。於1969年畢業於[[聖若瑟書院|聖若瑟書院]]<ref>{{cite web |url=https://www.sjc.edu.hk/oldboys.php?content=perform |title=Prominent Old Boys > Performing Arts |last= |first= |date= |website= |publisher=聖若瑟書院 |access-date= 2019-4-8 |quote=}}</ref>，在校時與[[麥當雄|麥當雄]]為同學。劉松仁少年時由恩保德（Giovanni Giampietro）神父帶領受洗成為天主教教徒，又曾長期參與聖堂事工<ref>{{cite web |url=https://hk.lifestyle.appledaily.com/nextplus/magazine/article/20121025/2_16702658/%E4%B8%8D%E6%98%AF%E9%99%B8%E5%B0%8F%E9%B3%B3-%E6%98%AF%E9%99%B3%E8%BF%91%E5%8D%97-%E5%8A%89%E6%9D%BE%E4%BB%81 |title=不是陸小鳳 是陳近南 劉松仁 |last= |first= |date=2012-10-25 |website= |publisher=壹週刊 |access-date= 2019-4-8 |quote=}}</ref>。{{noteTag|關於劉松仁受洗的資料出自香港電台第一台廣播節目《守下留情》於2019年4月8日播出的劉松仁訪問。}}他於1970年考入[[麗的映聲|麗的映聲]]第五屆電視藝員訓練班而入行<ref>{{cite web |url=https://jet.my-magazine.me/article/detail/interview/10971 |title=劉松仁 松仁的法則 |last= |first= |date=2019-4 |website= |publisher=JET Magazine |access-date= 2019-4-8 |quote=}}</ref>。起先他在[[麗的電視|麗的電視]]演出，1971年演出首部電視劇《[[煙水寒|煙水寒]]》，但直至1976年簽約[[無綫電視|無綫電視]]後才開始為人熟悉。《[[陸小鳳|陸小鳳]]》一劇不但令劉松仁聲名鵲起，由其演繹的「[[陸小鳳|陸小鳳]]」也被認為無可替代，其時他的代表作包括1980年首播的《[[京華春夢|京華春夢]]》。

劉松仁於1980年拍畢《[[京華春夢|京華春夢]]》後重返[[麗的電視|麗的電視]]演出《[[風塵淚|風塵淚]]》一劇。在整個1980年代，劉松仁主要在[[亞洲電視|亞洲電視]]和台灣拍攝劇集，他和著名演員[[米雪|米雪]]合作大量經典電視劇，包括《[[大地恩情|大地恩情]]》、《[[萍蹤俠影錄|萍蹤俠影錄]]》、《[[秦始皇|秦始皇]]》、《[[法網柔情|法網柔情]]》和《[[啼笑因緣|啼笑因緣]]》等，二人更被視為最佳螢幕情侶。劉松仁比較突出的電影作品乃《[[愛的世界|愛的世界]]》、《[[黃蜂尾後針|黃蜂尾後針]]》、《[[東方三俠|東方三俠]]》、《[[現代豪俠傳|現代豪俠傳]]》和《[[東京審判|東京審判]]》還有《[[鹿鼎記_(1992年電影)|鹿鼎記]]》等。其時他的演出遍及[[中國大陸|中國大陸]]和[[台灣|台灣]]，因此曾於兩地演出多套劇集。

他於1992年再亮相[[無綫電視|無綫電視]]，甫回歸便在《[[大時代|大時代]]》取得成功，雖然全劇40集僅6集戲分，但精湛演技獲觀眾讚同，其中被老闆炒魷魚跪地哀求一幕更令不少人流淚。此後他同於兩台演出，但隨著亞視開始連年虧損減少劇集製作，劉松仁在亞視之演出開始減少。

2012年與[[英皇電影|英皇電影]]簽定經理人合約。他在2015年與恩保德神父開始構思音樂劇《利瑪竇》，過程中聯同[[杜琪峯|杜琪峯]]共同策劃細節。該劇於2019年4月上演。<ref>{{cite web |url=https://hk.news.appledaily.com/local/daily/article/20190321/20637840 |title=
蘋人誌：原來以前我是死人一個　劉松仁 |last= |first= |date=2019-3-21 |website= |publisher=香港蘋果日報 |access-date= 2019-4-8 |quote=}}</ref>

==個人生活==
劉松仁的妻子蘇嘉燕為一位[[國泰航空|國泰航空]]的資深[[空姐|空姐]]。他們於往[[加拿大|加拿大]]的航班上邂逅，於1992年結婚。二人婚後沒有生兒育女。
