'''劉人語'''<ref name="百度百科">{{cite web |title=刘人语 |url=https://baike.baidu.com/item/%E5%88%98%E4%BA%BA%E8%AF%AD/22564447|website=百度百科 |date=2018-10-03}}</ref>（{{lang-en|'''Ren Yu-Liu'''}}，{{bd|2001年|10月10日}}），中國女歌手，現為[[ETM活力時代|ETM活力時代]]旗下藝人。

== 經歷 ==
劉人語在2015年5月進入[[ETM活力時代|ETM活力時代]]成為付費練習生，同年7月成為簽約練習生<ref name="ETM活力時代">{{cite web |title=etm活力时代 - ETMSkies |url=http://www.etmskies.com/Skies.asp |website=ETM活力時代官方網站 |date=2018-08-02 }}</ref>。2017年7月，她以[[ETM_OrangE|ETM OrangE]]首批小分隊成員活動，11月18日發表首支廣告曲《咪咕圈圈》<ref name="搜狐娛樂網">{{cite web |title=咪咕圈圈发布会—ETM家族OrangE小分队首演圈圈主题曲 |url=http://www.sohu.com/a/205675849_100022609 |website=搜狐娱乐 |date=2017-11-21}}</ref>。2018年2月26日，她又成為[[ETM氧氣少女計畫|ETM氧氣少女計畫]]其中一員，同年4月21日參加[[騰訊視頻|騰訊視頻]]女子偶像團體養成競技類節目「[[創造101|創造101]]」<ref>{{Cite web|url=https://v.qq.com/detail/7/78483.html|title=创造101|date=2018-06-27|work=v.qq.com|language=zh-CN}}</ref>，最終獲得了第13名的成績；8月24日於[[成都|成都]]ETM公演星耀秀上宣布跟[[蘇芮琪|蘇芮琪]]組成雙人小分隊出道，9月30日則與蘇芮琪、[[羅奕佳|羅奕佳]]、[[張靜萱|張靜萱]]及[[吉利|吉利]]組成女團[[Chic_Chili|Chic Chili]]，並在10月10日發行首支個人單曲《LRY》。

=== 創造101時期 ===
劉人語在第一期以評級考核第二天首位評級A成為A班選手晉級，但因節目剪輯導致畫面幾乎遭刪減<ref name="中國娛樂">{{cite web |title=ETM活力时代得罪了谁 ？《创造101》首期亮相 “一剪没” |url=http://news.yule.com.cn/html/201804/269160.html |website=中国娱乐 |date=2018-04-25}}</ref>。第二期主題曲評級，成功留守A班，成為節目中僅有5位導師評級的"雙A"選手。第三期以導師評選能力C位第3名成為第一次公演C位。第四期順位發布以413萬票位列第22名，成功晉級55強。第五期被分配至創作組，導師[[胡彥斌|胡彥斌]]提議使用劉人語原創歌曲《麻煩少女》下去修改。第六期公演以526票成為該組"點讚王"。第七期順位發布以1500萬票位列第15名，成功晉級36強。第九期順位發布以接近2800萬票位列第13名，晉級前22強。第十期決賽，最終以71191950萬票，位列第13名，未能成團。

此外，劉人語原為出道熱門人選，但在[[創造101|創造101]]決賽直播途中顯示點贊數為73853280萬票<ref>{{Cite web|url=https://v.qq.com/detail/7/78483.html|title=创造101|date=2018-06-27|work=v.qq.com|language=zh-CN}}</ref>，最終顯示得票數卻不增反減的少了270萬票，大批網友懷疑節目組造假黑幕，隨後節目組僅在官方微博聲稱選手[[劉人語|劉人語]]的點贊數據由於技術原因發生錯誤，並進行了票數修正，由7300多萬票變為6500多萬票，但此聲明並未讓眾人信服<ref name="Yahoo新聞">{{cite web |title=《101》選手讚數變少 網怒批黑幕 |url=https://tw.news.yahoo.com/101-%E9%81%B8%E6%89%8B%E8%AE%9A%E6%95%B8%E8%AE%8A%E5%B0%91-%E7%B6%B2%E6%80%92%E6%89%B9%E9%BB%91%E5%B9%95-045504506.html |website=Yahoo新聞 |date=2018-06-24 }}</ref>。
