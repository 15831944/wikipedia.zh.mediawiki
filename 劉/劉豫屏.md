'''劉豫屏 '''（{{Bd|1965年|||}}），[[中華民國海軍陸戰隊|中華民國海軍陸戰隊]][[少將|少將]]、[[本省人|本省人]]，生於[[臺灣|臺灣]][[臺東|臺東]]，籍貫[[臺灣|臺灣]][[臺東|臺東]]，畢業於[[國立臺東大學|國立臺東大學]]附設國小、[[臺東|臺東]]新生國中、[[中正預校|中正預校]]、[[陸軍官校|陸軍官校]]56期，現任[[海軍陸戰隊|海軍陸戰隊]]少將副指揮官，曾任海軍陸戰隊少將參謀長、[[海軍陸戰隊陸戰九九旅|海軍陸戰隊陸戰九九旅]]少將旅長、海軍陸戰隊上校副參謀長、海軍陸戰隊指揮部後勤處上校處長、海軍陸戰隊補保上校大隊長<ref>[http://m.appledaily.com.tw/appledaily/article/headline/20150522/36564669 美軍首辦兩棲研討 台受邀]</ref><ref>[http://www.epochtimes.com.tw/n203932/%E5%8F%B0%E6%B5%B7%E8%BB%8D%E9%99%B8%E6%88%B0%E9%9A%8A%E5%8F%83%E8%AC%80%E9%95%B7-%E5%96%84%E5%BF%B5%E7%9A%84%E5%82%B3%E6%89%BF-%E5%95%9F%E4%BA%BA%E5%90%91%E5%96%84.html 台海軍陸戰隊參謀長：善念的傳承 啟人向善]</ref>。

== 紀錄 == 
2015年5月，時任海軍陸戰隊陸戰九九旅旅長劉豫屏參加[[美國海軍陸戰隊|美國海軍陸戰隊]]主辦的[[夏威夷|夏威夷]]首屆「太平洋司令部兩棲領導人研討會」（Pacific Command Amphibious Leaders Symposium，簡稱PALS），亦觀摩了[[美軍|美軍]][[兩棲登陸|兩棲登陸]][[軍事演習|軍事演習]]。2016年6月，中華民國與美國斷交38年後，劉豫屏再次領軍一個[[連|連級]][[兵力|兵力]]編制的[[地面部隊|地面部隊]]，前往美國進行協同演習<ref>[https://tw.mobi.yahoo.com/tech/%E5%8F%B0%E7%BE%8E%E6%96%B7%E4%BA%A438%E5%B9%B4%E5%BE%8C-%E6%B5%B7%E8%BB%8D%E9%99%B8%E6%88%B0%E9%9A%8A%E5%8F%97%E9%82%80%E5%8F%83%E5%8A%A0%E7%BE%8E%E8%BB%8D%E5%85%A9%E6%A3%B2%E6%BC%94%E7%BF%92-010000083.html 中華民國與美國斷交38年後 海軍陸戰隊受邀參加美軍兩棲演習]</ref>。
