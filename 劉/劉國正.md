'''刘国正'''{{BD|1980年|3月7日||}}是[[中国|中国]]退役乒乓球运动员，[[湖北|湖北]][[武汉|武汉]]人，曾为[[中国国家乒乓球队|中国国家乒乓球队]]男子二队主教练。2017年4月6日起正式担任[[中国国家乒乓球队|中国国家乒乓球队]]男子一队主教练。

== 简历 ==
刘国正6岁开始打球，1991年进省队，1994年进国家二队，1995年进入国家队。1999年国际乒联巡回赛总决赛，刘国正获得男单冠军，就此崭露头角。一年后，刘国正入选中国队出征[[2000年世界乒乓球锦标赛|2000年世乒赛]]的阵容，并于决赛上击败[[彼得·卡尔松|卡尔松]]，为中国队赢得一局。同年，刘国正在队内竞争中战胜势头正盛的马琳，与孔令辉、刘国梁携手出战[[2000年奥林匹克运动会|悉尼奥运会]]男子单打赛事。刘国正以1:3的总比分不敌瑞典名将[[约尔根·佩尔森|佩尔森]]，止步八强。<ref>[http://sports.sina.com.cn/others/200009/2388008.shtml 乒乓球-刘国正、吕林谈1/4决赛负于佩尔森原因]</ref>

2001年，刘国正再次入选中国队征战[[世乒赛|世乒赛]]的阵容。志在重夺冠军的中国队，于半决赛遭遇[[韩国国家乒乓球队|韩国队]]的强力阻击。双方总比分战成2:2时，刘国正在决胜局对阵韩国首席选手[[金泽洙|金泽洙]]。在比赛中，刘国正成功挽救了七个赛点，最终以2:1的比分击败金泽洙。<ref>{{Cite web |url=http://sports.21cn.com/integrate/2012/03/31/11340784.shtml |title=中韩男团对抗史：刘国正演绎经典 宿敌难撼国乒 |access-date=2013-01-27 |archive-url=https://web.archive.org/web/20160908234130/http://sports.21cn.com/integrate/2012/03/31/11340784.shtml |archive-date=2016-09-08 |dead-url=yes }}</ref>此战是刘国正球员生涯中最经典的一战，并让其人气急升，被球迷赠予“抗韩英雄”的称号。有球迷甚至说出：“嫁人就嫁刘国正”的赞美之词。<ref>[http://sports.sohu.com/94/61/sports_news163156194.shtml 刘国正:演绎奇迹的后来者 刘国梁:让人感动的失意者]</ref>

此后，刘国正曾参加[[2004年世界乒乓球团体锦标赛|2004年世界乒乓球团体锦标赛]]，再次随队获得冠军，但由于伤病的困扰，刘国正逐渐淡出了国家队。

2008年，刘国正宣布退役，后进入国家队教练组成为实习教练。2012年，刘国正正式任职中国乒乓球队男子二队主教练。<ref>[http://news.enorth.com.cn/system/2013/01/04/010480694.shtml?fr=hao10086.com 刘国正：80后个性教练]</ref>
