'''斐迪南一世'''（'''{{lang|ru|Фердинанд I}}'''，{{bd|1861年|2月26日|1948年|9月10日|catIdx=Ф}}），全名'''斐迪南·马克西米利安·卡尔·利奥波德·马利·萨克森-科堡-哥达斯基'''（'''{{lang|ru|Фердинанд Максимилиан Карл Леополд Мари Сакскобургготски}}'''）。[[保加利亞|保加利亞]][[沙皇|沙皇]]，[[保加利亚王国|保加利亚王国]]的开国君主。

== 家族与早年生活 ==
斐迪南一世是[[萨克森-科堡-哥达|萨克森-科堡-哥达]]-科哈里的奥古斯特王子（''Аугуст фон Сакс-Кобург-Гота-Кохари''，1818年—1881年）与波旁-奥尔良的克莱门汀公主（''Клементина Бурбон-Орлеанска''，1817年—1907年）的第三子，也是年纪最小的孩子。奥古斯特王子是萨克森-科堡-哥达王子斐迪南·格奥尔格·奥古斯特与[[匈牙利|匈牙利]]大贵族科哈里家族的小姐安东妮的次子。奥古斯特王子的长兄是[[葡萄牙|葡萄牙]]女王[[玛丽亚二世_(葡萄牙)|玛丽亚二世]]的丈夫[[费尔南多二世_(葡萄牙)|费尔南多二世]]；[[英国君主列表|英国女王]][[维多利亚女王|维多利亚]]的丈夫[[阿尔伯特亲王|阿尔伯特亲王]]是奥古斯特王子的堂弟；[[比利时君主列表|比利时第一任国王]][[利奥波德一世_(比利时)|利奥波德一世]]是他的叔叔。克莱门汀公主是[[法国|法国]][[奥尔良王朝|奥尔良王朝]][[法国君主列表|国王]][[路易·腓力|路易·腓力]]与王后两西西里的玛丽·阿美莉亚的第四个女儿。所以斐迪南一世有着萨克森-科堡-哥达王室、法国[[波旁王朝|波旁王室]]、两西西里波旁王室、[[奥地利|奥地利]][[哈布斯堡家族|哈布斯堡皇室]]及匈牙利科哈里家族的血统，又与葡萄牙王室、英国王室、比利时王室有着亲戚关系。

斐迪南一世于1861年2月26日生于奥地利首都[[维也纳|维也纳]]，他与父母在这里度过了童年，并在这里完成了学业。

== 保加利亚的君主 ==
1886年8月，保加利亚亲王[[亞歷山大一世_(保加利亞)|亚历山大一世]]被迫退位。许多国家的王公贵族觊觎保加利亚的王位。[[奥匈帝国|奥匈帝国]]于是推荐了斐迪南·冯·萨克森-科堡-哥达为新的保加利亚亲王，并得到了其他列强的首肯。1887年7月7日，斐迪南·冯·萨克森-科堡-哥达成为保加利亚亲王斐迪南一世。由于[[保加利亚|保加利亚]]是[[东正教|东正教]]国家斐迪南一世由[[天主教|天主教]]改宗东正教。

斐迪南一世是一位专制的君主，生活奢靡。在他统治的31年中，政府频繁更迭，最短的一届政府只有十几天；他在农村恢复的封建时代的[[什一税|什一税]]。在外交上，保加利亚与[[俄罗斯|俄罗斯]]的关系日渐降温，与[[奥匈帝国|奥匈帝国]]日渐亲密。

1908年9月22日，斐迪南一世在国民议会上宣布，保加利亚独立，不再是[[土耳其|土耳其]]的保护国。10月5日，他宣布保加利亚成为王国，保加利亚公国成为第三保加利亚王国，斐迪南一世也从保加利亚亲王变成了'''全体保加利亚人的沙皇'''。

称沙皇之后，斐迪南一世一心想恢复中世纪保加利亚王国的辉煌。1908年保加利亚与[[黑山|黑山]]、[[塞尔维亚|塞尔维亚]]和[[希腊|希腊]]结成[[巴尔干联盟|巴尔干联盟]]，共同反抗土耳其。10月8日[[第一次巴尔干战争|第一次巴尔干战争]]打响，保加利亚于10月17日对土宣战。保加利亚军队迅速取得了[[色雷斯|色雷斯]]，并于次年3月与塞尔维亚共同占领了阿德里安堡（今[[埃迪尔内|埃迪尔内]]）。土耳其被迫投降。保加利亚本想一举拿下土耳其首都[[君士坦丁堡|君士坦丁堡]]，但遭到了俄罗斯、希腊等国的反对。战争虽然结束，和谈的结果却让巴尔干联盟中的其他三国对保加利亚很不满。同年7月保加利亚对马其顿的塞尔维亚军队进攻，塞尔维亚和希腊对保宣战，[[第二次巴尔干战争|第二次巴尔干战争]]爆发。保加利亚遭到惨败，失去了大量土地。

斐迪南一世并没有放弃他的梦想，1914年，奥匈帝国皇储在[[萨拉热窝|萨拉热窝]]遇刺，奥匈帝国向塞尔维亚宣战，塞尔维亚向保加利亚求救。斐迪南一世由于第二次巴尔干战争的失败，秘密与奥匈帝国谈判，企图趁火打劫，夺回当年失去的领土。1915年10月1日，斐迪南一世不顾部分议员和人民的反对，向塞尔维亚宣战。次年9月保加利亚又向罗马尼亚宣战，使保加利亚陷入了腹背受敌的地步。

[[一战|一战]]末期的1918年9月24日，保加利亚军队发生哗变。9月27日，起义军宣布成立[[共和国|共和国]]。斐迪南一世的[[亚历山大·马林诺夫|马林诺夫]]政府派出代表团与起义军进行谈判，不料谈判代表中的[[亚历山大·斯坦姆鲍利斯基|亚历山大·斯坦姆鲍利斯基]]（''Александър Стамболийски''）加入起义军并成为共和国总统。起义军在9月29日逼近首都[[索非亚|索非亚]]，政府宣布退出战争，并请来德国军队镇压了起义。

10月4日，斐迪南一世在[[协约国|协约国]]的要求下退位，王位让给了长子[[鲍里斯三世|鲍里斯]]。斐迪南流亡奥地利。

== 晚年生活 ==
斐迪南后来生活在他的故乡[[德国|德国]]科堡，依靠德国政府的抚恤金生活。1948年在那里去世，享壽87岁，葬于科堡的圣奥古斯汀天主教大教堂。

斐迪南一世还是一位作家、植物学家和集邮家。

== 婚姻与子女 ==
[[File:Les_princesses_Eudoxia_et_Nadejda_de_Bulgarie.jpg|thumb]]
斐迪南一世于1893年4月20日与[[帕尔马公爵列表|帕尔马公爵]][[罗伯托一世_(帕尔马)|罗伯托一世]]的长女'''玛利亚·路易萨'''（意大利文：''{{lang|it|'''Maria Luisa''' Pia Teresa Ana Ferdinanda Francesca Antonietta Margherita Giuseppina Carolina Bianca Lucia Apolonia}}''，保加利亚文：'''''{{lang|bg|Мария-Луиза Бурбон-Пармска}}'''''，1870年—1899年）结婚，有二子二女：
* 长子[[鲍里斯三世|'''鲍里斯·克莱门特·罗伯特·玛利亚·庇护·斯坦尼斯拉夫''']]（'''{{lang|bg|Борис Клемент Роберт Мария Пий Станислав}}'''，1894年1月30日—1943年8月28日），1918年成为沙皇鲍里斯三世。
* 次子[[基里尔_(保加利亞摄政)|'''基里尔·亨利·弗朗茨·路德维希·安东·卡尔·菲利普''']]（'''{{lang|bg|Кирил Хайнрих Франц Лудвиг Антон Карл Филип}}'''，1895年11月17日—1945年2月1日），普雷斯拉夫亲王，1943年至44年为摄政王，1945年被共产党处决。
* 长女'''叶夫多基娅·奥古斯塔·菲利品娜·克莱门蒂娜·玛利亚'''（'''{{lang|bg|Евдокия Аугуста Филипина Клементина Мария}}'''，1898年1月17日—1985年10月4日）
* 次女'''纳杰日达·克莱门蒂娜·玛利亚·皮娅·玛热拉'''（'''{{lang|bg|Надежда Клементина Мария Пия Мажелла}}'''，1899年1月30日—1958年2月15日），1924年与[[符腾堡统治者列表|符腾堡公爵]][[阿尔布雷希特_(符腾堡)|阿尔布雷希特]]的次子阿尔布雷希特·欧根·玛利亚·菲利普·卡尔·约瑟夫·福图纳图斯结婚，有三子二女。
由于玛丽·路易萨在生下纳杰日达后第二天去世，斐迪南于1908年3月28日在科堡与罗伊斯-科斯特里茨亲王海因里希四世和罗伊斯长系侯国公主路易丝·卡洛琳的唯一女儿'''埃莱奥诺尔'''（德文：''{{lang|it|'''Caroline Gasparine Louise }}''，保加利亚文：'''''{{lang|bg|Елеонора Каролина Гаспарина Луиза}}''''' ，1860年8月22日-1917年9月12日)结婚，两人没有任何子嗣。
