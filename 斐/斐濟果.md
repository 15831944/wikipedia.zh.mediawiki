'''斐濟果'''（Feijoa）、'''[[费约果|费约果]]'''、'''[[鳳梨番石榴|鳳梨番石榴]]'''、'''非油果'''、'''菠萝番石榴'''<ref>{{cite web |url=http://www.sohu.com/a/19470238_126525 |title=这些变态水果你吃过十种就是大牛！ |date=2015-06-19 |website=[[搜狐|搜狐]] |publisher= |access-date=2017-06-03 }}</ref>，在中國大陸又称'''纳粹瓜'''（因其果肉的纵切面像[[纳粹党|纳粹党]]的[[卐|卐字标志]]而得名）。原產地在中南美洲[[哥伦比亚|哥伦比亚]]、 [[乌拉圭|乌拉圭]]和[[阿根廷|阿根廷]]北部，盛產於[[紐西蘭|紐西蘭]]，外皮深綠色，內部類似[[芭樂|芭樂]]。含豐富[[維他命C|維他命C]]。

== 消費與使用 ==
斐濟果通常是用[[水果刀|水果刀]]削半，然後用[[湯匙|湯匙]]舀出果肉食用。果實香甜多汁，無比芳香，接近外皮的部份比較堅韌。在新西蘭，斐濟果會被製成果汁飲料，[[果醬|果醬]]，[[冰淇淋|冰淇淋]]等，還可以用來製作[[葡萄酒|葡萄酒]]或[[蘋果酒|蘋果酒]]，也可以注入[[伏特加|伏特加]]。

水果從樹上跌落時是最佳成熟時，不久其內部肉質開始褐色，果實開始過分成熟。

== 運輸和銷售 ==
斐濟果成熟後, 果實很容易擦傷，因此斐濟果雖然好吃，卻不適合大量出口。斐濟果可以被冷凍長達一年, 無-{zh-tw:品質;zh-cn:質量}-損失。

== 栽培 ==
在紐西蘭，斐濟果多靠[[蜜蜂|蜜蜂]]或[[黃蜂|黃蜂]]以及中型鳥類傳遞花粉。在美國[[加州|加州]]，由[[知更鳥|知更鳥]]或[[蜂鳥|蜂鳥]]傳遞花粉。
