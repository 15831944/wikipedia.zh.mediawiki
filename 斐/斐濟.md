'''斐濟共和國'''（{{lang-en|Republic of Fiji}}，{{lang-fj|Matanitu Tugalala o Viti}}，[[斐濟印地語|斐濟印地語]]：रिपब्लिक ऑफ फीजी） 通稱'''斐濟'''，是位於南[[太平洋|太平洋]]，[[瓦努阿圖|瓦努阿圖]]以東、[[東加|東加]]以西、[[吐瓦魯|吐瓦魯]]以南的[[群島|群島]]國家，由330個[[島嶼|島嶼]]組成（一半為[[無人島|無人島]]，而[[維提島|維提島]]和[[瓦努阿島|瓦努阿島]]兩個主要島嶼的人口佔全國的87%）。國家名稱是源自[[東加語|東加語]]中的「島嶼」，並且轉為斐濟文中的“Viti”。

== 歷史 ==
[[File:Fijian_mountain_warrior,_Kai_Colo.jpg|thumb]]
[[File:Urville-Viti-Lebouka2.jpg|thumb]]，1842年]]
{{main|斐濟歷史}}
早在17世紀，便有[[東南亞|東南亞]]裔人士抵達斐濟定居，比[[歐洲|歐洲]]的探險家要早一些。

1643年，[[荷蘭|荷蘭]]探險家[[阿貝爾·塔斯曼|阿貝爾·塔斯曼]]抵達斐濟，是首位踏足此地的歐洲人。

1871年，[[斐济王国|斐济王国]]成立，这是首个统一[[斐济群岛|斐济群岛]]的政权。

1874年，[[英国|英国]]吞并斐济，成立[[斐济殖民地|斐济殖民地]]。

斐濟於1970年從英國獨立，1987年發生了兩起軍事政變，第一次是有人不滿政府由當地的印度裔壟斷，第二次是要求廢除[[君主立憲|君主立憲]]，實施[[共和制|共和制]]，以總統取代總督，其國家名稱也從當時的「[[斐濟自治領|斐濟自治領]]」改為'''「斐濟共和國」'''。1990年改名'''「斐濟主權民主共和國」'''。1998年再改為'''「斐濟群島共和國」'''。

2006年5月6日，议会选举举行。5月18日，选举委员会确认，在众议院的71个议席中，斐济团结党获得36个席位，[[斐济工党|斐济工党]]得到31个议席，联合[[人民党|人民党]]获得两个议席，其余两个议席为独立竞选人获得。斐济团结党领导人[[莱塞尼亚·恩加拉塞|莱塞尼亚·恩加拉塞]]宣誓就职总理<ref>[http://news.xinhuanet.com/newscenter/2006-05/18/content_4567516.htm] </ref>。

2006年12月5日，斐济武装部队司令[[弗兰克·姆拜尼马拉马|弗兰克·姆拜尼马拉马]]称，军方已完全[[2006年斐济军事政变|接管政府并控制了全国]]，解除总理恩加拉塞的职务，他本人暂时代行总统职务。<ref>[http://news.xinhuanet.com/world/2006-12/05/content_5438878.htm]</ref>12月6日，过渡政府总理乔纳·塞尼兰加卡利宣誓就职<ref>[http://news.xinhuanet.com/world/2006-12/06/content_5444435.htm] </ref>。2007年1月4日，姆拜尼马拉马宣布将总统职务交还给[[约瑟法·伊洛伊洛|约瑟法·伊洛伊洛]]<ref>[http://news.xinhuanet.com/world/2007-01/04/content_5565097.htm]</ref> 。1月5日，姆拜尼马拉马宣誓就任过渡政府总理。

2011年，恢復原國號「斐濟共和國」<ref>[http://news.xinhuanet.com/world/2007-01/05/content_5569014.htm]</ref> 。

== 政治 ==

斐济的政治体制为代议民主制，斐济首相为政府首脑，总统为国家领袖，并实行多党制。行政权归属政府，立法权由国会与政府共有，司法权独立于行政权与立法权。斐济为[[英联邦|英联邦]]成员。

== 地理 ==
[[File:Kulu_Bay_Resort1.jpg|thumb]]
[[File:Fiji_map.png|thumb]]
{{main|斐济地理}}
斐济总面积（含海域面积）为194000平方公里，其中10%面积为陆地。

斐济包含322个岛屿，主要为珊瑚礁环绕的火山岛，其中1/3有人居住。最大的岛屿为[[维提岛|维提岛]]与[[瓦努阿岛|瓦努阿岛]]。首都[[苏瓦|苏瓦]]位于维提岛上。四分之三的人口也居住在维提岛。该岛多山，最高峰托马尼维峰（Tomaniivi），海拔1324米，并有热带雨林。其他重要城镇有[[納迪|納迪]]、[[萨武萨武|萨武萨武]]与[[劳托卡|劳托卡]]。

===氣候===
斐濟屬於溫暖的[[熱帶海洋氣候|熱帶海洋氣候]]，幾乎全年的溫度都在一個很小的範圍內。溫暖的季節出現在11月到4月，涼爽的季節則是從5月至10月。即使在在涼爽的季節，平均溫度仍然高達22°C（72°F）。降雨是季節性的到來，溫暖的季節會有較多的雨量，尤其是在內陸地區。雖然一年平均會有一個熱帶氣旋來襲（十年出現10-12次），但一般說來，風是溫和的。

== 軍事 ==
[[File:Dabur881.JPG|thumb]]
{{main|斐濟軍事}}
'''斐濟共和國軍事部隊'''（RFMF），總兵力約3500人，是一支小型軍隊。其中，有3200名現役軍人為陸軍部隊，組成六個步兵營和一個工兵營；另外，大約有6000名陸軍後備部隊組成三個步兵營。過去有一個"祖魯"（Zulu）{{link-en|反革命特戰部隊|Counter Revolutionary Warfare Unit (Fiji)}}（約80名士兵），在2000年年底因涉及軍事政變被裁撤。

== 經濟 ==
[[File:Tree_map_exports_2009_Fiji.jpeg|thumb]]
[[File:Suva_from_harbour.jpg|thumb]]市區]]
斐濟經濟以農業、漁業、林業、製造業、礦產業、資訊與通訊產業、影視工業為主。農作物部份，包括栽培農作物、新鮮水果、蔬菜，如：芋頭、椰子、可可豆、木瓜、各類水果和蔬菜、卡瓦（Kava）和稻米。畜牧部份，包括牛肉，乳品，豬肉，家禽，綿羊，山羊和養蜂。漁業對斐濟經濟具有重大意義，因為它不但是蛋白質的主要來源，也是就業和外匯收入重要項目。林業可供砍伐的林地面積總計約956,860公頃，占國內產值（GDP）的1.3%。製造業多為基礎的工業。採礦和採石業平均占國民生產總值的1.4%。斐濟運用資訊與通訊技術，吸引世界上知名的企業將其後台作業（back office services）交給位在斐濟的公司運作。影視工業則包括電影、電視、視訊、音訊、電腦軟體、電子商務、電訊等的製作與輸出。

== 人口 ==
[[File:Fijian_women_ceremonial.jpg|thumb]]
2003年的统计数据显示斐济有人口837,000人。人口密度45.8人/平方公里。人均寿命72.1岁。从二十世纪三十年代开始，每年人口增长率为1.1%。斐济人口主要由15-64岁的青壮年组成，年龄[[中位数|中位数]]为27.9岁。人口性别比为103（女性=100）。

[[美拉尼西亞|美拉尼西亞]]人佔54.3%。[[印度|印度]]裔佔38.1%。

=== 语言 ===
斐济的宪法规定了三个语言为官方语言，分別是[[英语|英语]]、[[斐济语|斐济语]]和[[印度斯坦语|印度斯坦语]]（{{lang|en|Hindustani}}）。其中，英语和印度斯坦语皆由英国[[殖民者|殖民者]]所引入，印度斯坦语是居住在斐济的印度族裔的主要语言，而斐济语则是斐济本民族的语言。据斐济宪法规定，斐济公民有权利使用其中任何一种语言同政府机构沟通，需要的话政府应及时提供翻译人员。

由于斐济受英国[[殖民统治|殖民统治]]超过一百年，[[英语|英语]]成为了斐济本民族和印度族裔最常用的语言。不同族群间沟通一般是用英语，政府机构、教育机构、金融和法律机构也一般以英语为主。

斐济语是斐济的民族语言，属于[[南岛语系|南岛语系]]。与同属南岛语系的[[美拉尼西亚语|美拉尼西亚语]]较为接近，但由于长期与[[波利尼西亚|波利尼西亚]]民族接触，因此也受到[[马来-波利尼西亚语族|马来-波利尼西亚语族]]中的[[汤加语|汤加语]]的影响。

[[印度斯坦语|印度斯坦语]]是在斐济居住的印度族裔和信仰[[伊斯兰教|伊斯兰教]]的[[穆斯林|穆斯林]]们所讲的主要语言。与[[印度次大陆|印度次大陆]]所讲的[[印地语|印地语]]和[[乌尔都语|乌尔都语]]有一定区别，但其区别并不妨碍相互交流，關於斐濟所使用的印度斯坦語，請參見[[斐濟印地語|斐濟印地語]]。

除了上述三种官方语言，斐济还有很多其他的语言。其中在该国西部的很多方言，同官方的标准语（Bau Standard）差别很大，甚至可以被认为是另一种语言。另外，很多斐济的印度人除了说印度斯坦语外，也说[[泰米尔语|泰米尔语]]。还有少数说[[比哈里语|比哈里语]]（Bihari）、[[孟加拉语|孟加拉语]]和其他语言。而在该国的[[罗图马岛|罗图马岛]]上，主要使用[[罗图马语|罗图马语]]（属于波利尼西亚语的一种）。

== 新闻媒体 ==
斐济的主要报纸有[[斐济时报|斐济时报]]（英文版）、每日邮报、斐济太阳报，华文报纸有斐济日报和斐济华声报。

斐济主要杂志有岛国商务、太平洋岛屿（月刊），在太平洋岛国地区发行，发行量约1万册。

斐济有一家广播公司，成立于1985年，是斐济的官方电台，也是斐济全国性的广播网，由6个电台组成，使用斐济语、英语、印地语向全国播音。此外斐济还有电视台，成立于1993年，共有1个免費频道（{{link-en|Fiji One|Fiji One}}）和付費電視服務{{link-en|SKY Pacific|Sky Pacific}}，向全国播出<ref>[http://news.xinhuanet.com/ziliao/2002-06/23/content_453267_6.htm 斐济概况]</ref>。
