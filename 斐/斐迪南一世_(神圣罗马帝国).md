'''斐迪南一世'''（[[德语|德语]]：'''Ferdinand I'''，{{bd|1503年|3月10日|1564年|7月25日|catIdx=F}}）[[哈布斯堡王朝|哈布斯堡王朝]]的[[奥地利大公|奥地利大公]]和[[神圣罗马帝国|神圣罗马帝国]]皇帝（1556年起；1558年加冕）。他也是[[匈牙利王国_(1526年-1867年)|匈牙利]]和[[波希米亚王国|波希米亚]]的国王（1526年-1564年）。

== 事迹 ==
[[File:Hans_Bocksberger_der_Aeltere_001.jpg|thumb]]
斐迪南一世是[[卡斯蒂利亚|卡斯蒂利亚]]国王[[腓力一世_(卡斯蒂利亚)|腓力一世]]与其妻卡斯蒂利亚女王[[疯女胡安娜|胡安娜]]的儿子，生于[[马德里|马德里]]附近的[[埃纳雷斯堡|埃纳雷斯堡]]。

斐迪南早年在[[西班牙|西班牙]]接受教育。1521年，他的哥哥西班牙国王兼神圣罗马帝国皇帝[[查理五世_(神圣罗马帝国)|查理五世]]任命他为[[奥地利大公国|奥地利]]的统治者，并且是皇帝在德意志的全权代理人。

1526年[[匈牙利王国_(1301年–1526年)|匈牙利]]兼[[波希米亚王国|波希米亚]]国王[[拉约什二世|拉约什二世]]在与[[奥斯曼帝国|奥斯曼帝国]]的[[第一次摩哈赤战役|第一次摩哈赤战役]]中阵亡后，斐迪南当选为这两国的国王（他的妻子安娜是拉约什二世的姊姊）。由于大片匈牙利国土为奥斯曼帝国占领，斐迪南实际仅能控制匈牙利的一小部分。在与反对哈布斯堡王朝的[[特兰西瓦尼亚|特兰西瓦尼亚]]统帅[[扎波尧伊·亚诺什|扎波尧伊·亚诺什]]长期斗争之后，他才在这部分领地上站住脚。[[土耳其人|土耳其人]]在1529年和1533年两次围攻[[维也纳|维也纳]]，造成巨大危机；到局势稳定时，斐迪南控制匈牙利西部（[[皇家匈牙利|皇家匈牙利]])），扎波尧伊·亚诺什（在奥斯曼帝国保护下）控制匈牙利东部（{{le|匈牙利东部王国|Eastern Hungarian Kingdom|匈牙利东部王国}}），而南部地区被奥斯曼帝国吞并。

1531年，德意志[[选侯|选侯]]选举斐迪南为“[[罗马人民的国王|罗马人民的国王]]”，使他成为查理五世皇帝的直接继承者。由于斐迪南企图利用捷克军队去镇压德意志[[新教|新教徒]]，波希米亚在1547年曾发生一次叛乱。

當[[施馬爾卡爾登聯盟|施馬爾卡爾登聯盟]]進行反查理五世的戰爭時，斐迪南极支持其兄壓制德意志地區的[[新教|新教]][[诸侯|诸侯]]。1552年新的宗教战争爆发时，他又充当查理五世与[[施馬爾卡爾登聯盟|施馬爾卡爾登聯盟]]之间的调解人（查理在这次战争中失败）。在[[扎波尧伊·亚诺什|扎波尧伊·亚诺什]]去世后，他的遗孀{{le|伊莎贝拉·雅盖隆|Isabella Jagiellon|伊莎贝拉·雅盖隆}}将特兰西瓦尼亚和匈牙利东部割让给斐迪南（1551年{{le|魏森堡条约|Treaty_of_Weissenburg}}）；但由于奥斯曼帝国的压力，斐迪南并不能进行有效统治。1554年斐迪南曾派遣使者到[[伊斯坦布尔|伊斯坦布尔]]去，希望和[[苏莱曼一世_(奥斯曼帝国)|苏莱曼一世]][[苏丹_(称谓)|苏丹]]商讨缔结一份边界条约。

1556年9月12日，查理五世宣布让位于斐迪南，后者遂成为[[神圣罗马帝国|神圣罗马帝国]]皇帝（一般认为到1558年查理五世去世后才是斐迪南的正式在位时期）。可能是迫于[[大臣|大臣]]與诸侯的压力，查理五世排除了他的儿子[[謹慎者腓力|謹慎者腓力]]成为皇帝的可能，这就永远阻止了[[奥地利大公国|奥地利]]-[[西班牙帝国|西班牙]]的合併或者成為[[共主邦聯|共主邦聯]]，因為奥地利和神圣罗马帝国的皇位由斐迪南的后代继承。

作为皇帝，斐迪南一世的统治比较安定。他小心地维持着与[[奥斯曼帝国|奥斯曼帝国]]的相对和平。在内政方面，斐迪南倾向于[[中央集权|中央集权]]。他在1527年颁布过一部对哈布斯堡家族世袭领地有效的[[宪法|宪法]]。但是，来自匈牙利和捷克大贵族的反对迫使斐迪南放弃了将这些国家完全纳入奥地利统治的做法，他在1559年承认这些国家的独立地位。

1564年斐迪南一世在维也纳去世，遗体葬于[[布拉格|布拉格]][[圣维特大教堂|圣维特大教堂]]。

== 家庭 ==
斐迪南在1521年5月25日与波希米亚兼匈牙利国王[[弗拉迪斯拉夫四世|弗拉迪斯拉夫四世]]的女儿安娜结婚，这桩婚姻使波希米亚与匈牙利两国成为哈布斯堡王朝领地。斐迪南夫妇的子女有：
# 奥地利的伊丽莎白
# [[马克西米连二世_(神圣罗马帝国)|马克西米连二世]] 
# 奥地利的安娜
# [[斐迪南二世_(上奧地利)|斐迪南二世]]
# 奥地利的玛丽娅
# 奥地利的玛格达莱娜
# 奥地利的凯瑟琳
# 奥地利的埃丽奥诺拉
# 奥地利的玛格丽特
# 奥地利的约翰（夭折）
# 奥地利的巴巴拉
# [[查理二世_(内奥地利)|查理二世]]大公 
# 奥地利的乌尔苏拉
# 奥地利的海伦
# 奥地利的约翰娜
