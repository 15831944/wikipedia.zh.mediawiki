'''储安平'''（{{bd|1909年|||1966年？|catIdx=Chu Anping}}），[[江苏|江苏]][[宜兴|宜兴]]人，[[中国|中国]]现代学者、知识分子。出身望族，曾留学[[英国|英国]]，后任[[复旦大学|复旦大学]]教授。[[民国|民国]]时期著名评论家，《[[观察_(杂志)|观察]]》社长和主编。曾因批判[[国民政府|国民政府]]政策而导致杂志被查封。[[中华人民共和国|中华人民共和国]]成立後曾出任[[新华书店|新华书店]]经理、《[[光明日报|光明日报]]》社总编、[[九三学社|九三学社]]宣传部副部长等职。

1957年因响应所谓“[[百花齐放、百家争鸣|百花齐放、百家争鸣]]”方针，發表了《向毛主席、周總理提些意見》（即“党天下”）一文。[[毛泽东|毛泽东]]对此不满发文反驳，并迅速发起了[[反右运动|反右运动]]。儲安平被作为右派典型打倒，其家人和诸多相关人士均受牽連。1966年[[文化大革命|文化大革命]]中储安平遭受残酷迫害而失踪，后被宣布死亡。1978年後，55萬右派被平反改正，但儲安平依然是不予改正的中央级“[[未获改正的右派|五大右派]]”之一。

==生平简历==
===早年===
儲安平出身于宜兴望族。宜興儲氏為唐代詩人[[儲光羲|儲光羲]]之後，自[[南宋|南宋]]以來，科名長盛不衰。儲安平十二世祖[[儲昌祚|儲昌祚]]、九世祖[[儲方慶|儲方慶]]、八世祖[[儲大文|儲大文]]皆為進士<ref>《豐義儲氏家譜》</ref>。儲安平出生后六天即丧母<ref>{{cite journal|title = 母亲 |accessdate =2009-06-10 |author = 储安平 |date = 1930年|language =zh-hant |publisher =《真美善》杂志7卷1号|pages =101}}</ref>，14岁丧父，依靠祖母、伯父儲南強抚养成人。儲南強肄業於[[江陰南菁書院|江陰南菁書院]]，與[[黃炎培|黃炎培]]是同學，曾官南通知縣。

1928年，储安平入[[上海|上海]][[光华大学|光华大学]]（今[[华东师范大学|华东师范大学]]）英文系学习，爱好文学创作，为“[[新月派|新月派]]”后起之秀，1932年毕业。

===媒体生涯===
1933年任[[南京|南京]]《[[中央日报|中央日报]]》副刊编辑，同时在戏剧学校兼课。1934年，储安平与光华大学同学[[端木露西|端木露西]]结婚<ref name="luci">{{cite journal|title=端木露西：蔚蓝中的一点黯淡|author=[[章诒和|章诒和]]|date=2010年3月25日|language=zh-hans|publisher=《[[南方周末|南方周末]]》|url=http://www.infzm.com/content/43006|deadurl=yes|archiveurl=https://archive.is/20130426164207/http://www.infzm.com/content/43006|archivedate=2013年4月26日}}</ref>。1936年赴[[英国|英国]]采风，后入英国[[伦敦政治经济学院|伦敦政治经济学院]]学习，或受到当时著名的[[自由主义|自由主义]]思想家[[哈罗德·拉斯基|哈罗德·拉斯基]]教授的影响<ref>{{cite journal|title = 储安平的英国情节|accessdate =2009-06-10 |author = 谢泳 |date = 2006年1月16日|language =zh-hans |publisher =《[[南方都市报|南方都市报]]》|url =http://www.southcn.com/nfsq/ywhc/ds/200601160632.htm}}</ref>。在英期间担任无薪酬、只领稿酬的《中央日报》驻欧记者。

1938年因[[中國抗日戰爭|抗战]]学业未完，回[[重庆|重庆]]，先后任《中央日报》撰述、编辑，[[复旦大学|复旦大学]]教授，[[中央政治大学|中央政治大学]]研究员。由于不堪忍受写作“遵命文章”及拒绝入[[中國國民黨|党]]等原因，离开重庆。1940年8月後，辗转于[[湖南省|湘]]、[[廣西壯族自治區|桂]]之间。曾任[[国立师范学院|国立师范学院]]教师（今[[湖南师范大学|湖南师范大学]]，学校时在湖南[[安化县|安化县]][[蓝田镇|蓝田镇]]），讲授[[英國歷史|英國史]]和世界政治概论。后在[[桂林|桂林]]担任《[[力报|力报]]》主笔和湖南[[辰溪|辰溪]]《[[中国晨报|中国晨报]]》主笔。日军侵占桂林后返重庆。1945年11月与[[张稚琴|张稚琴]]创办《[[客觀_(重慶雜誌)|客观]]》周刊，任主编。1944年或1945年，与端木露西离婚<ref name="luci" />。

1946年春，储安平离开《客观》杂志社，9月1日创办《观察》半月刊，任社长和主编，并使得《观察》成为国共内战时期著名的自由刊物。同时兼任复旦大学教授，讲授《比较宪法》、《各国政府与政治》等课程。1948年12月25日，[[蔣介石|蔣介石]]親自下令查封《觀察》[[雜誌|雜誌]]和逮捕《觀察》工作人員，這就是中國近代史上著名的「《觀察》事件」。此後，著名「七君子」之一[[王造時|王造時]]親自出面，以自己的影響力多方奔走，給當局施壓。並於1949年2月親自擔保和營救了包括美術家[[朱宣咸|朱宣咸]]在內數位《觀察》進步人士出獄。《觀察》是宣傳「第三條道路」的代表，是當時中國一面進步民主運動的旗幟。查封《觀察》是近代中國[[自由主義|自由主義]]悲歌。儲安平曾批判[[中國國民黨|中国國民黨]]“七十天是一场小烂污，二十年是一场大烂污，烂污烂污，二十年来拆足了烂污”<ref>储安平《一场烂污》，《观察》第5卷第11期第1页。</ref>。

[[中华人民共和国|中华人民共和国]]成立後，《观察》复刊，储继续担任主编至1950年终刊。文章多以游记杂感为主，鲜见时评。1950年《观察》终刊后，任新华书店副总经理，1952年改任[[中央出版总署|中央出版总署]]发行局副局长。1954年任九三学社中央委员兼宣传部副部长，并任第一届[[全国人大|全国人大]]代表。

===“黨天下”言論與遭迫害===
1957年4月1日，经[[胡乔木|胡乔木]]推荐任[[光明日报|《光明日报》]]总编。4月30日，[[毛泽东|毛泽东]]在天安门城楼上召集民主党派负责人座谈，请他们积极提意见，帮助共产党[[百花運動|整风]]<ref>{{cite web |url = http://cpc.people.com.cn/GB/64162/64165/78561/79779/5637497.html|title = 毛泽东等中央领导人邀请各民主党派负责人和无党派民主人士谈话 |accessdate = 2009-06-16 |language = zh-hans |publisher = 中国共产党新闻网}}</ref>。为消除党外人士的思想顾虑，[[中国共产党|中国共产党]]再次重申了“知无不言，言无不尽；言者无罪，闻者足戒；有则改之，无则加勉”的原则。在这种政策影响下，经历[[胡风案|胡风案]]等一系列批判事件后的知识分子开始逐渐向中共提出批评和意见。6月1日，在中共中央统一战线工作部召集的座谈会上，储安平以《向毛主席和周总理提些意见》为题发言，称“宗派主义的突出，党群关系的不好，是一个全面性的现象。”而且与中央也有很大关系。并且称在百花政策後“大家对小和尚（基层和一般党员）提了不少意见，但对老和尚（中共高层）没有人提意见。”并委婉地批评政府已经成为一党天下，國務院12位副總理中無黨外人士，最后总结“这个‘党天下’的思想问题，是一切宗派主义现象的最终根源，是党和非党之间矛盾的基本所在”<ref>{{cite web |url = http://news.china.com/zh_cn/history/all/11025807/20050418/12249456.html |title = 质疑“党天下”：“大右派”储安平的传奇人生 |accessdate = 2009-06-08 |date = 2005年4月18日 |language = zh-hans |publisher = [[中华网|中华网]]转自《福建党史月刊》 |deadurl = yes |archiveurl = https://web.archive.org/web/20090317181854/http://news.china.com/zh_cn/history/all/11025807/20050418/12249456.html |archivedate = 2009年3月17日 }}</ref>。此文一出石破天惊，动撼朝野。次日，《[[人民日报|人民日报]]》、《光明日报》以醒目标题显著位置全文刊登。文章发表後，使毛澤東「一連幾天沒睡好覺」（據[[胡乔木|胡乔木]]兒子透露）。

6月6日，[[中華人民共和國國務院|国务院]][[中華人民共和國國務院秘書長|秘书长]][[习仲勋|习仲勋]]邀请党外人士举行座谈。国务院秘书长助理[[卢郁文|卢郁文]]首先发言，批评储安平的“党天下”的提法是严重的错误。6月8日《人民日报》发表了毛泽东亲自执笔的、题为《[[这是为什么？|这是为什么？]]》的社论，反右序幕从此拉开<ref name="died">{{cite web |language = zh-hans |publisher = 人民网 |title = 中国“十大右派”之一──储安平的失踪之谜 |url = http://news.0898.net/2005/09/20/190368.html |author = 闻文 |date = 2005-09-20 |accessdate = 2009-06-14 |deadurl = yes |archiveurl = https://web.archive.org/web/20131005002404/http://news.0898.net/2005/09/20/190368.html |archivedate = 2013-10-05 }}</ref>。此后，对储安平铺天盖地的揭发、批判与控诉接踵而至。子女被迫划清界限，同事纷纷“检举揭发”，储安平陷入天下围攻的局面。6月21日，储安平在[[九三学社|九三学社]]中央常务委员会第十六次扩大会议结束时“承认错误”，他说，他看到[[台湾|台湾]]和[[香港|香港]]的“反动报纸”为他喝彩时“感到很沉重”，并表示“要认真地挖掘自己的思想根源，并且交代同[[章伯钧|章伯钧]]、[[罗隆基|罗隆基]]的关系”。7月召开的第一届全国人大第四次会议上，储安平作了题为《向人民投降》的发言，《人民日报》全文发表<ref name="何与怀">{{cite web |url = http://www.tecn.cn/data/detail.php?id=7574 |title = 北望长天祭英魂──储安平诞辰96周年纪念 |author = 何与怀 |authorlink =  |accessdate = 2009-06-15 |date = 2005-07-18 |language = zh-hans |publisher = 天益网 |deadurl = yes |archiveurl = https://web.archive.org/web/20080511030214/http://www.tecn.cn/data/detail.php?id=7574 |archivedate = 2008-05-11 }}</ref>。

然而对储安平的批判并未因此而结束。1957年11月12日，作为《光明日报》总编辑的储安平和《光明日报》社长[[章伯钧|章伯钧]]被同时免职。在1958年1月18日至24日召开的九三学社第四届中委会第三次全会上，储安平中央委员、中央宣传部副部长的职务被撤销。1月31日，第一届[[全国人民代表大会|全国人大]]第五次会议召开，决定罢免储安平[[中華人民共和國人大代表獨立候選人|人大代表]]的资格。此后，储安平被一顶特大号的“大右派分子”的帽子死死地扣在头上<ref name="何与怀" />。

1966年[[文化大革命|文化大革命]]开始后，储被残酷迫害。1966年6月1日，《人民日报》发表评论《[[横扫一切牛鬼蛇神|横扫一切牛鬼蛇神]]》，作为著名“大右派”，储安平也成为被“扫荡”对象。8月31日，遭受多次批斗后的储安平投河自杀未遂，被造反派押回[[九三学社|九三学社]]，看管起来<ref>{{cite web |url = http://blog.sina.com.cn/s/blog_470bf2570100ajx2.html |title = 父亲储安平之死 |date =2008年10月13日 |accessdate =2009-06-10 |author1 = 李菁 |author2 = 储望华 |language = zh-hans |publisher =《[[三联生活周刊|三联生活周刊]]》}}</ref>。1966年9月上旬失踪，生死不明。同时对于其失踪也有不同的说法：出家说<ref name="died" />、出走说、自杀说以及被打死说等不同的猜测和怀疑<ref>{{cite web |url = http://www.edubridge.com/erxiantang/library/chuanping.htm |title = 储安平生死之谜又一说 |date = 1999年 |issue = 1999年第2期 |accessdate = 2009-06-10 |author = 余开伟 |language = zh-hans |journal = 《[[书屋|书屋]]》 |deadurl = yes |archiveurl = https://web.archive.org/web/20100121181502/http://www.edubridge.com/erxiantang/library/chuanping.htm |archivedate = 2010-01-21 }}</ref>。也有人说他是先在青龙桥投河自杀未遂，接着又跳海自沉了，然而何时何地都是未知<ref>{{Cite book|title=余时书话|last=姜|first=德明|publisher=复旦大学出版社|year=2016|isbn=9787309121872|location=上海|pages=231}}</ref>。

==家庭==
1933年，储安平与[[端木露西|端木露西]]结婚，后离异，育有四子女。1949年后，与[[易吟先|易吟先]]<ref>易吟先1960年与[[宋希濂|宋希濂]]结婚，1980年後赴美定居。</ref>结婚，1957年[[反右运动|反右运动]]后离婚。

储安平子女有[[储望英|储望英]]（长子）、[[储望德|储望德]]（次子）、[[储望瑞|储望瑞]]（女儿）、[[儲望華|儲望華]]（幼子，音乐家，现居[[澳大利亚|澳大利亚]]）。
