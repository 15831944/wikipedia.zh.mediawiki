== 文月學園的學生 ==
=== 高中部二年級 ===
==== F班 ====
成績最糟糕的班級，有著最差的設備，包括壞掉的[[榻榻米|榻榻米]]、缺乏填充物的座墊和紙箱做成的課桌。由47名男性、2名女性和秀吉組成，二年級有149名男學生，因此約有三分之一在F班。

: 故事的主角，第一人稱敘事者。他是無可救藥的笨蛋，成績奇差無比。擁有笨蛋的代名詞頭銜「觀察處分者」，這個頭銜使他的召喚獸能夠接觸到物體，但代價是召喚獸所受傷害會轉移到自己身上。
: 他的雙親長期在海外工作，雖然有定時匯生活費給他，但他總是拿去買新遊戲。這使得他以鹽水、糖和油為主食，偶爾會以泡麵度日，但仍會將一包泡麵分成64份以延長食用時間。由於他獨自生活，因此他擅長烹飪和家務
: 有許多女性和男性都愛慕他，包括姬路瑞希、島田美波、島田葉月、久保利光和吉井玲，不過因為他太笨了而沒有察覺。他和雄二有著深厚的友誼，不需言語就能了解彼此想法。
: 在第二集學園祭時獲得了白金手環，可以同時召喚出兩隻召喚獸。
:在動畫第二季第3集中說明了如何得到觀察處分者

==== A班 ====
成績最好的學生所組成的一班，享有高級設備，包括個人電腦、飲料和點心吃到飽。其中霧島翔子、木下優子和工藤愛子三位學生合稱A班三子。

: 雄二的青梅竹馬，長髮飄飄的美人，是成績最頂尖的學生。她相當有自信且聰明，說話輕聲細語，在小說中，她的對白並不是以[[引號|引號]]而是以[[括弧號|括弧號]]包夾。她自小便瘋狂地喜歡雄二，對其它男性的告白一律拒絕，因此曾被認為是[[女同性戀|女同性戀]]。她經常會戳雄二的眼睛以防止他去看其它女孩穿泳裝或換衣服的樣子，她會使用鎖鍊、[[項圈|項圈]]、電擊棒等物品控制他。她也規劃了兩人結婚後的計劃，更將小孩的名字也取好了。她不擅長機械，任何機械到了她手中都會毀壞。
: 她的召喚獸身穿日本武士鎧甲、手持日本刀。

; {{nihongo|木下優子|木下 優子（きのした ゆうこ）|Kinoshita Yuuko|聲優：日本：[[加藤英美里|加藤英美里]]／[[水原薰|水原薰]]（廣播劇CD）；台灣：[[林筱玲|林筱玲]]}}

; {{nihongo|工藤愛子|工藤 愛子（くどう あいこ）|Kudou Aiko|聲優：日本：[[南條愛乃|南條愛乃]]；台灣：[[穆宣名|穆宣名]]}}

== 文月學園的教職員 ==
; {{nihongo|藤堂薰|藤堂 カヲル（とうどう カヲル）||聲優：[[仲村香織|仲村香織]]／[[文月君|文月君]]（廣播劇CD）；台灣：[[李明幸|李明幸]]}}
: 文月學園的學園長。試驗召喚系統的發明者，秉持「教育者也必需有相應的學力」的教育理念，要求教師也必需參加測驗及使用召喚獸，常常將學園當成自身的發明品實驗場。

; {{nihongo|西村宗一|西村 宗一（にしむら そういち）||聲優：[[大塚明夫|大塚明夫]]／[[佐佐木大輔|佐佐木大輔]]（廣播劇CD）；台灣：[[林谷珍|林谷珍]]}}
: 掌管紀律的嚴格教師，在F班敗給A班後、成了F班的導師。外號「鐵人」，因為他的興趣是[[鐵人三項|鐵人三項]]，他的個性嚴格、肉體經過千錘百鍊，會將召喚分數歸零的學生拖入補習室教育一番，身受所有學生的敬畏。雖然無法使用召喚獸，但能擊出媲美召喚獸的鐵拳。

; {{nihongo|高橋洋子|高橋 洋子（たかはし ようこ）||聲優：[[嘉數由美|嘉數由美]]；台灣：[[林筱玲|林筱玲]]}}
: 2年A班的導師，通稱「高橋女士」。她是整個學院中最聰明的人，她所有科目的總分是7800分，近乎翔子的兩倍。但她完全不懂得打棒球。
: 她的召喚獸外型類似第一次世界大戰時德國軍官的制服，以鞭子做武器。

; {{nihongo|福原慎|福原 慎（ふくはら しん）||聲優：[[津田健次郎|津田健次郎]]；台灣：[[鍾少庭|鍾少庭]]}}
: 原二年F班的導師，在F班試召戰爭敗北後，導師職務就交給了西村老師。在動畫版中、自第一季第三集起任旁白。

; {{nihongo|大島武|大島 武（おおしま たけし）||聲優：[[田中雅之|田中雅之]]；台灣：[[鍾少庭|鍾少庭]]}}
: 保健體育科目的教師。有著優異行動力，能利用繩索從屋頂垂降突入教室。召喚獸裝備為現代軍服以及軍刀。

; {{nihongo|長谷川|長谷川（はせがわ）||聲優：[[景浦大輔|景浦大輔]]；台灣：[[鈕凱暘|鈕凱暘]]}}
: 數學教師。似乎喜歡高橋老師。召喚獸裝備武器為[[三角尺|三角尺]]。

; {{nihongo|布施文博|布施 文博（ふせ ふみひろ）||聲優：[[德本英一郎|德本英一郎]]；台灣：[[鈕凱暘|鈕凱暘]]}}
: 化學教師。說話都用[[日語敬語|敬語]]，彬彬有禮且對危機泰然處之。召喚獸的武器是實驗用[[錐形瓶|錐形瓶]]。

; {{nihongo|船越|船越（ふなこし）}}
: 數學教師。四十五歲還是單身，是一個會拿學分威脅學生跟她交往的老師。

; {{nihongo|竹中|竹中（たけなか）||聲優：[[高橋伸也|高橋伸也]]（動畫第一季）→[[株田裕介|株田裕介]]（動畫第二季）；台灣：[[鈕凱暘|鈕凱暘]]}}
: 古典文學教師。

== 文月學園學生家人 ==
; {{nihongo|島田葉月|島田 葉月（しまだ はづき）|Shimada Hazuki|聲優：[[平田真菜|平田真菜]]／[[辻步美|辻步美]]（廣播劇CD）；台灣：[[林筱玲|林筱玲]]}}
: 美波的妹妹，小學五年級。稱明久是「笨蛋哥哥」，但她也對明久有好感，希望成為他未來的新娘。

; {{nihongo|吉井玲|吉井 玲（よしい あきら）||聲優：[[井上喜久子|井上喜久子]]；台灣：[[李明幸|李明幸]]}}
: 明久的姊姊，23歲。雖然是哈佛大學畢業，但她毫無道德觀念，可以只圍著浴巾就外出搭火車。她有戀弟情結，經常會勾引明久並企圖與他發生性關係。她和瑞希在許多地方很類似，包括聰明、烹飪技巧很差、巨乳、以及喜歡明久。

; {{nihongo|坂本雪乃|坂本 雪乃（さかもと ゆきの）||聲優：[[永田依子|永田依子]]；台灣：[[林筱玲|林筱玲]]}}
: 雄二的母親，外表年輕、常會被誤認為雄二的姉姊。極度[[天然呆|天然呆]]，會搞混七月和十月，分不清[[海膽|海膽]]和[[鬃刷|鬃刷]]。

; {{nihongo|姬路瑞穗|姫路 瑞穂（ひめじ みずほ）}}
: 瑞希的母親，儘管實際上已經四十一歲，但身高只到明久的胸口附近。因為女兒瑞希的抱枕和坐垫上印著明久的圖，因此斷定女兒喜歡他、並對他相當滿意。

; {{nihongo|久保良光|久保 良光（くぼ よしみつ）}}
: 久保利光的弟弟，本作品中少數擁有正常人思維的角色。與土屋康太的妹妹土屋陽向是同班同學。
