'''卡瑟姆若马尔特·克米列维奇·托卡耶夫'''（{{lang-kz|Қасым-Жомарт Кемелұлы Тоқаев}}，{{bd|1953年|5月17日|||Tokayev, Kassym-Jomart}}），生于[[阿拉木图|阿拉木图]]，是[[哈萨克斯坦|哈萨克斯坦]]政治家，政治学博士，[[哈萨克斯坦共和国|哈萨克斯坦共和国]]前总理（1999－2002年），2013年10月16日至2019年3月19日任[[哈萨克斯坦议会|哈萨克斯坦议会]]参议院议长。2019年3月19日，哈萨克斯坦时任总统[[努尔苏丹·纳扎尔巴耶夫|努尔苏丹·纳扎尔巴耶夫]]宣布辞职，托卡耶夫繼任总统。<ref name="1R">{{Cite web |url = https://news.sina.cn/gn/2019-03-19/detail-ihsxncvh3870185.d.html?from=wap |title = 哈萨克斯坦总统纳扎尔巴耶夫宣布辞职 执政近30年 |website=新浪网 |date = 2019-03-19 |language = zh |accessdate = 2019-03-19 }}</ref>

==人物生平==
===早年经历===
1970年，卡·托卡耶夫考入苏联[[莫斯科国立国际关系学院|莫斯科国立国际关系学院]]，后进入[[苏联|苏联]]外交部工作。在大学五年级时被派到苏联驻[[中华人民共和国大使馆|中华人民共和国大使馆]]实习6个月。

===政治生涯===
1975年，卡·托卡耶夫进入苏联外交部工作，任职[[苏联驻新加坡共和国大使馆|苏联驻新加坡共和国大使馆]]。1979年回到苏联外交部。1983年到中华人民共和国[[北京语言学院|北京语言学院]]进修中文10个月。

1984年至1985年在苏联外交部任职，后被派到苏联驻华大使馆工作至1991年，历任二秘、一秘、参赞。1991年考入[[莫斯科|莫斯科]]苏联外交部外交学院，参加高级外交官培训班。

1992年出任哈萨克斯坦共和国外交部副部长。1993年任外交部第一副部长。1994年任外交部长。1999年3月晋升为副总理。

1999年10月，经议会批准，根据哈萨克斯坦共和国总统令，托卡耶夫出任总理。2002年1月辞职，随后被任命为国务秘书兼外交部长。卡·托卡耶夫任职外交部长时间共计10年（1994－1999，2002－2007）。

托卡耶夫任外长期间在核不扩散领域发挥了积极作用。在1995年与2005年，托卡耶夫出席了在纽约召开的《[[不扩散核武器条约|不扩散核武器条约]]》审议大会。1996年在纽约签署了《[[全面禁止核试验条约|全面禁止核试验条约]]》。2005年在塞米巴拉金斯克签署了《[[中亚无核武器区条约|中亚无核武器区条约]]》。

2003年，其主持了内陆发展中国家国际部长级会议。 会议制定了“阿拉木图行动纲领”，并通过了“内陆发展中国家的阿拉木图宣言”。<ref>{{Cite web|title=Kassym-Jomart Tokayev biography, Career in the United Nations|url=http://www.fampeople.com/cat-kassym-jomart-tokayev|work=www.fampeople.com|accessdate=2019-02-22}}</ref>

卡·托卡耶夫参加了十届[[联合国大会|联合国大会]]，曾当选为[[独联体|独联体]]外长理事会和[[上海市|上海]][[上海合作组织|合作组织]]主席，托卡耶夫的外交官级别为特命全权大使。

2007年1月，当选为[[哈萨克斯坦议会|哈萨克斯坦议会]]参议院议长。

2008年，托卡耶夫以[[哈萨克斯坦议会|哈萨克斯坦议会]]参议院议长的身份当选为[[欧洲安全与合作组织|欧洲安全与合作组织]]议会副议长。

2011年3月，[[联合国秘书长|联合国秘书长]]宣布任命托卡耶夫为联合国日内瓦办事处总干事、联合国副秘书长、联合国秘书长在裁军谈判会议个人代表，兼任裁军谈判会议秘书长。他还是负责瑞士联合国工作人员安全的高级官员。

2013年10月16日，当选为哈萨克斯坦议会参议院议长。

2019年3月20日，哈萨克斯坦时任总统[[纳扎尔巴耶夫|纳扎尔巴耶夫]]辞职後，托卡耶夫繼任总统。<ref name="1R"/>

2019年3月20日，提议将首都阿斯塔纳改名为“努尔苏丹”，以纪念[[哈萨克斯坦|哈萨克斯坦]]独立后的第一任总统[[努尔苏丹·纳扎尔巴耶夫|努尔苏丹·纳扎尔巴耶夫]]<ref name="改名">{{Cite web |url = http://world.huanqiu.com/exclusive/2019-03/14578961.html |title = 哈萨克斯坦新总统提议：以纳扎尔巴耶夫之名命名首都|website=环球网 |date = 2019-03-20 |language = zh |accessdate = 2019-03-20 }}</ref>随后[[哈萨克斯坦议会|哈萨克斯坦议会]]通过宪法修正案，将首都阿斯塔纳更名为努尔苏丹。<ref name="改名2">{{Cite web |url = http://www.chinanews.com/gj/2019/03-20/8785617.shtml |title = 哈萨克斯坦通过宪法修正案 将首都更名为努尔苏丹|website=中国新闻网 |date = 2019-03-20 |language = zh |accessdate = 2019-03-20 }}</ref><ref name="change name">{{Cite web |url = https://tass.ru/mezhdunarodnaya-panorama/6238253 |title = Парламент Казахстана принял законопроект о переименовании Астаны в Нурсултан |date = 2019-03-20 |language = ru |accessdate = 2019-03-20 }}</ref><ref name="change name2">{{Cite web |url = https://www.svoboda.org/a/29832026.html |title = одобрил переименование Астаны в Нурсултан |date = 2019-03-20 |language = ru |accessdate = 2019-03-20 }}</ref>

==家庭状况==
托卡耶夫已婚，育有一子。其父[[科梅·托卡耶夫|科梅·托卡耶夫]]（1923－1986）曾参加过[[第二次世界大战|第二次世界大战]]，同时也是著名的哈萨克斯坦作家。其母[[图拉·莎巴帕耶娃|图拉·莎巴帕耶娃]]（1931－2000）任教于[[阿拉木图外国语学院|阿拉木图外国语学院]]。

==个人成就==
托卡耶夫精通[[俄语|俄语]]，[[英语|英语]]和[[中文|中文]]，懂[[法语|法语]]。除此之外还拥有政治学博士学位，出版过九本专著，发表过大量有关国际事务的论文。托卡耶夫现为世界人文与自然科学院的成员，[[慕尼黑|慕尼黑]]安全会议“智者”理事会成员，[[深圳大学|深圳大学]]（中国）名誉教授，俄罗斯联邦外交部外交学院名誉教授和博士、校董事会董事，哈萨克斯坦对外关系委员会名誉主席。他也是世界和传统宗教领袖大会秘书处的负责人，以及日内瓦外交与国际关系学院名誉院长。被授予哈萨克斯坦共和国外交部N. Tyryakulov纪念奖章，俄罗斯联邦外交部果里恰科夫奖章，[[日内瓦大学|日内瓦大学]]“学士”学位，以及洛里奇纪念奖章。根据俄罗斯传记协会的统计，他进入《[[2018年度人物|2018年度人物]]》获奖名单。 <ref>{{Cite web|title=Geneva School of Diplomacy {{!}} Kassym-Jomart Tokayev|url=https://genevadiplomacy.ch/kassym-jomart-tokayev/|accessdate=2019-02-22|language=en-US}}</ref>

托卡耶夫曾担任[[哈萨克斯坦乒乓球联合会主席|哈萨克斯坦乒乓球联合会主席]]13年。

他曾在哈萨克斯坦共和国和其他国家荣获多个奖项。<ref>{{Cite web|title=Пожизненное заключение для наркодилеров предложил Токаев|url=https://tengrinews.kz/kazakhstan_news/pojiznennoe-zaklyuchenie-dlya-narkodilerov-predlojil-tokaev-359946/|work=Tengrinews.kz|date=2018-12-20|accessdate=2019-02-22|language=ru}}</ref>

==个人观点==

2014年11月24日，在[[阿斯塔纳|阿斯塔纳]]举行的议会会议上，他表示哈萨克斯坦政治结构的特点为“强有力的总统，权威的议会，负责的政府”。

2016年4月29日，在[[哈萨克斯坦参议院|哈萨克斯坦参议院]]关于土地租赁和出售土地法修正听证会上，他强调：“我们不仅要保证，农业用地不会卖给外国人。并且我们要通过立法和其他方式来确保这一重要条款的实施，以便没有人利用各种狡猾的方法来规避此要求。最重要的是，必须确保国家元首对于国家土地，以及高效合理使用土地管辖权的要求。”

2017年9月22日的议会教育听证会上，关于将[[哈萨克语|哈萨克语]]转写为[[拉丁字母|拉丁字母]]的问题，他指出：“我们必须完成具有战略意义的任务：翻译出拉丁字母表。就像国家元首指出的那样，将哈萨克语用拉丁字母转写是朝着融入全球科学和教育体系迈出的一步，确保我们精神统一。我们需要认真，渐进地开展这项工作，而不是‘大跃进’。但是推迟它是没有意义的。最重要的是，我们没有犯错误的权利。”

卡·托卡耶夫曾批评哈萨克语对应拉丁字母的初始版本：“国家委员会没有就拉丁字母表做出最终的决定，因此在报纸或者其他地方使用拉丁字母还为时过早。”2018年2月19日，新的字母表通过以后，卡·托卡耶夫向所有参与“历史事件”的人表示祝贺。<ref>{{Cite web|title=Kazakhstan adopts new version of Latin-based Kazakh alphabet|url=https://astanatimes.com/2018/02/kazakhstan-adopts-new-version-of-latin-based-kazakh-alphabet/|work=The Astana Times|date=2018-02-26|accessdate=2019-02-22|language=en|first=Elya Altynsarina in Nation on 26|last=February 2018}}</ref>

2018年12月19日，在参议院的一次会议中，托卡耶夫提议加强对毒贩的处罚为[[终身监禁|终身监禁]]或者[[死刑|死刑]]，因为他们的活动像[[恐怖活动|恐怖活动]]一样威胁到国家利益。
