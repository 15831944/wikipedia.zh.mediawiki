'''卡那卡那富族'''（[[卡那卡那富語|卡那卡那富語]]：'''{{lang|xnb|Kanakanavu}}'''），舊稱'''卡那卡那布'''，人口約345人，主要居住在[[高雄市|高雄市]][[那瑪夏區|那瑪夏區]]。傳統語言為[[卡那卡那富語|卡那卡那富語]]，但目前通用語是[[布農語|布農語]]。以往被各統治政府歸類於[[鄒族|鄒族]]，和[[拉阿魯哇族|拉阿魯哇族]]合稱「南鄒」，[[中華民國行政院|中華民國行政院]]於2014年6月26日承認其為台灣原住民族的第16族。<ref>[http://www.cna.com.tw/news/aipl/201406260230-1.aspx 正名成功 原住民再增2族變16族]，中央社，2014年6月26日</ref><ref>[http://news.ltn.com.tw/news/politics/breakingnews/1040436 原住民再多2族 增至16族]</ref><ref name=Hlaalua>原住民族委員會,"

== 簡介 ==
「卡那卡那富」（Kanakanavu）意為住在Kanakanavu一地的人，古稱「簡仔霧番」或「干仔霧番」，在[[台灣日治時期|日治時代]]認定為鄒族，戰後亦沿用此分類至今。但學界在日治時代即對於該族群和鄒族，有各種分類法。[[伊能嘉矩|伊能嘉矩]]、[[鳥居龍藏|鳥居龍藏]]等學者根據語言與風俗習慣的異同，將鄒族視為單一族群，但對族群內部的區分並不一致；[[小川尚義|小川尚義]]則由[[比較語言學|比較語言學]]的觀點，將[[鄒族|鄒族]]、卡那卡那富族、[[拉阿魯哇族|拉阿魯哇族]]視為三個獨立的族。<ref>[http://www.dmtip.gov.tw/Index.aspx 台灣原住民數位博物館] {{webarchive|url=https://web.archive.org/web/20130826004238/http://www.dmtip.gov.tw/Index.aspx |date=2013-08-26 }}</ref>卡那卡那富族人在2001年原住民族語言能力認證時，發現他們在語言上無法溝通，進一步比較發現與鄒族在祭典與食材上也有很大差異，加上受到[[賽德克族|賽德克族]]正名運動激勵，決定發起正名運動<ref>[http://www.chinatimes.com/newspapers/20140224000375-260102 新故鄉願景－爭正名叫我卡那卡那富族]</ref>，並於2014年6月26日正式獨立為一族。<ref name=Hlaalua/>

==祭儀==
卡那卡那富族的重要祭儀有「Mikong（米貢祭）」、「Pasikarai（河祭）」等。

傳統狩獵的方式有三種 Uma'anupu (帶狗狩獵) / Rumaringee (陷阱) / Aracakaan (狩獵)

== 民族起源 ==
Kanakanavu的口傳族史中，關於其祖先的原居地在何處有幾種不同的說法，按照其地理位置可大致分成兩大類：第一大類是「東來說」，有關此說的傳說有多種版本，不過其內容大同小異，現舉一例：Kanakanavu與[[沙阿魯阿族|Hlaʼalua]]昔日同住在Nacunga，後來遭遇了大洪水侵襲，前者逃往Tanungintsu，後者逃往Nausulana（[[籐包山|籐包山]]，在今[[那瑪夏區|那瑪夏區]]東南方）。待洪水退後，Kanakanavu再下山至Natanasa 。<ref>移川子之藏。1935</ref>第二大類則為「西來說」：一說Kanakanavu的祖先原本住在[[嘉南平原|嘉南平原]]。後來因平原上的野獸被獵得越來越少，為了追捕野獸之故，就漸漸地往山區遷移，最後來到[[楠梓仙溪|楠梓仙溪]]上游一帶建立聚落。在遷移的路途上，大約走到[[臺南市|台南市]][[楠西區|楠西區]]一帶時，有一群人與Kanakanavu的祖先分手而往[[旗山區|旗山]]的方向走去，這群人就是今日的[[沙阿魯阿族|Hlaʼalua]]的祖先。另一說是Kanakanavu的祖先原本住在台南附近，與同住平原的其它族群和諧相處，後來受[[台灣荷治時期|荷蘭人逼迫]]才遷往山區（1995, 2002）。<ref>，林曜同，〈建構與分類：「南鄒族」（Kanakanavu）的族屬論述〉，
《國立臺灣大學考古人類學刊》，2005年，頁97–140。</ref>
