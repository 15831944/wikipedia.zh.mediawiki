'''卡尔十六世·古斯塔夫'''（{{lang-sv|Carl XVI Gustaf}}，全名'''卡尔·古斯塔夫·富尔克·休伯特斯'''、{{lang|sv|Carl Gustaf Folke Hubertus}}，{{bd|1946年|4月30日|}}）是当今的[[瑞典|瑞典]]国王。他是已故王太子[[古斯塔夫·阿道夫亲王|古斯塔夫·阿道夫亲王]]和[[薩克森-科堡-哥達|薩克森-科堡-哥達]]-的施比拉公主（{{lang|sv|Sibylla av Sachsen-Coburg-Gotha}}）的唯一儿子。由於其父在即位前便因遭遇[[1947年凱斯楚普機場空難|空难]]去世，而他的叔父皆失去王位繼承權，故而他於1973年9月15日继承了祖父[[古斯塔夫六世|古斯塔夫六世]]的王位，而他是現時僅次於[[大英國協王國|大英國協十六個君主國]]的[[伊莉莎白二世|伊莉莎白二世]]和[[丹麥|丹麥]]的[[瑪格麗特二世_(丹麥)|瑪格麗特二世]]為最長任期君主，也是[[瑞典歷史|瑞典歷史]]上在位最久之君主。

== 教育 ==
[[File:Vasa_Carl_Gustaf_1961.jpg|thumb]]旁]]
卡爾完成高中課程後，他便到[[瑞典陆军|瑞典陆军]]、[[瑞典海軍|瑞典海軍]]和[[瑞典空軍|瑞典空軍]]接受兩年半的軍事訓練。他曾在[[乌普萨拉大學|乌普萨拉大學]]和[[斯德哥爾摩大學|斯德哥爾摩大學]]攻讀歷史、社會學、政治科學、財務權益及經濟學。

為了準備擔當國家元首的角色，卡爾接受了一系列极为广泛的教育，其中主要關於法律系統、社會組織和機關、工會、和雇主的協會。另外，他还认真學習了解了議會、政府及其下辖各部的职能和工作方式以及如何处理外交事務。卡爾·古斯塔夫还经常花时间出国参与实践活动：他曾亲自参与在瑞典常驻聯合國代表團和瑞典國際發展代辦處的工作；还曾在倫敦的一家銀行、{{Link-en|瑞典駐英國大使館|Embassy of Sweden, London}}、瑞典駐法國商會以及Alfa Laval公司在法国的分廠工作过些许時日。

== 婚姻和家庭 ==
卡尔十六世·古斯塔夫在1976年6月19日与当今王后[[希爾維亞王后|希尔维亚]]結婚。希尔维亚王后是一位德國、巴西混血兒。卡尔十六世·古斯塔夫跟她在1972年慕尼黑夏季奧林匹克運動會上首次見面，她當時是一名翻譯。他們現在居住在斯德哥爾摩郊外的[[卓寧霍姆宮|卓寧霍姆宮]]，並以斯德哥爾摩市中心的[[斯德哥爾摩王宮|斯德哥爾摩王宮]]作為日常的辦公地點。

他们有三个孩子：
*[[維多利亞王儲|维多利亚·英格丽德·爱丽丝·德西蕾]]（Victoria Ingrid Alice Désirée，1977年7月14日─），瑞典女王储，西约特兰女公爵
*[[卡尔·菲利普王子_(瑞典)|卡尔·菲利普·爱德华·贝蒂尔]]（Carl Philip Edmund Bertil，1979年5月13日─），韦姆兰公爵
*[[马德琳公主_(瑞典)|马德琳·特蕾丝·阿梅丽·约瑟芬]]（Madeleine Thérèse Amelie Josephine，1982年6月10日─），海尔辛兰及耶斯特里克蘭女公爵

按照瑞典原来的王位继承法，卡爾·菲利普王子本是当然的王位承繼人，但是在他出生后不久的根據[[性别平等|性别平等]]的原則通过的憲法修正案使得他的姐姐——維多利亞公主于1980年1月1日成为瑞典歷史上首位女王储。

== 王室职责和个人兴趣 ==
根據[[瑞典憲法|瑞典憲法]]，國王是[[虛位元首|虛位元首]]，其地位纯粹是代表性和禮儀性的。他代表瑞典对外国进行國事訪問并接待对瑞典的访问；主持瑞典国會每年的开会仪式；在政府更迭期間主持特別委員會；根据法规向委員會與大臣提出咨询意见；主持外交事務委員會的會議，接受和簽署外国使节的国书，簽署瑞典對其它國家的国书。國王名义上还是瑞典三军的[[最高统帅|最高统帅]]。

自1975年起，他幾乎沒有仼何政治實權：首相人選先由議長提名，而非由國王直接任命；議會通過的法案亦無須得到[[御准|御准]]。少數的保留權力為冊封或去除其他王室成員的爵位。

卡尔十六世·古斯塔夫國王热心于環保、科技、農業、貿易和工业等事务。1994年，他对于邻国[[挪威|挪威]]猎杀[[海豹|海豹]]幼崽，表示“一個連海豹幼崽都不能关怀的人，也不會是一個很能治國的首相”，此言论受到瑞典人民普遍欢迎。在他的积极倡导下，一個定期舉行的國際环境專題討論會最终得以创立。他是瑞典皇家學院的保護者，并且经常积极地參與学院的许多活动。他还是[[世界童军运动组织|世界童军运动组织]]的名譽主席，經常參加在瑞典和海外童军活動。他在世界上最为人知的身份是每年[[諾貝爾獎|諾貝爾獎]]的颁发者。

== 诵读困难 ==
多年以来，瑞典民间廣泛傳言國王有[[诵读困难|诵读困难]]。新聞工作者注意到，當国王簽署文件时，时常拼錯他自己的名字；此外，當1973年国王參觀一個銅礦后在石墙上的签名也拼错了。1997年，在一次瑞典的電視採訪节目上，希尔维亚王后公開地承認这个传言。她说：“當国王年幼时，没有人注意到这个問題，因而他也没能及时得到必要的帮助。”王后还透露，他们孩子们身上也“多少”存在着閱讀困難的状况。
