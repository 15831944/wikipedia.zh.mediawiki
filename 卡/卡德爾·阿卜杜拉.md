'''卡德爾·阿卜杜拉'''（{{lang-fa|قادر عبدالله}}，{{lang-nl|Kader Abdolah}}；{{bd|1954年|11月12日}}）本名'''侯賽因·沙加地·加耶馬哈米·法拉哈尼'''（{{lang-fa|حسین سجادی قائم‌مقامی فراهانی}}，{{lang-nl|Hossein Sadjadi Ghaemmaghami Farahani}}），[[波斯人|伊朗裔]][[荷蘭人|荷蘭籍]]作家、詩人和专栏作家。他以在他的[[荷蘭文學|荷蘭語作品]]中融入波斯文學主題著稱，最知名的作品包括《[[清真寺之家|清真寺之家]]》、《國王》和《天書：我父親的筆記本》，已被翻譯成三十種語言<ref name="Quran" /><ref name="trouw1">{{cite web |title='De ontdekking van de hemel' beste Nederlands boek |trans-title=《發現天堂》被評為史上最佳的荷蘭文學著作 |url=https://www.trouw.nl/home/-de-ontdekking-van-de-hemel-beste-nederlands-boek~af670bbb/ |date=2007-03-11 |website=trouw.nl |accessdate=2018-05-04 |location=Amsterdam |language=nl |archive-url=https://web.archive.org/web/20180504225742/https://www.trouw.nl/home/-de-ontdekking-van-de-hemel-beste-nederlands-boek~af670bbb/ |archive-date=2018-05-04 |dead-url=no }}</ref>。

== 生平 ==
侯賽因·沙加地·加耶馬哈米·法拉哈尼於1954年出生於[[伊朗|伊朗]][[阿拉克|阿拉克]]，自幼於[[伊斯蘭|伊斯蘭]]家庭中成長，十二歲開始著迷於西方文學<ref name="vpro">{{cite web |title = Kader Abdolah (Arak, 1954) is schrijver |url = https://www.vpro.nl/programmas/zomergasten/lees/gasten/1999/kader-abdolah.html |website = vpro.nl |accessdate = 2018-07-23 |language = nl |archive-url = https://web.archive.org/web/20180723100431/https://www.vpro.nl/programmas/zomergasten/lees/gasten/1999/kader-abdolah.html |archive-date = 2018-07-23 |dead-url = no }}</ref>。1977年，他自阿拉克科學院（今{{link-en|阿拉克大學|Arak University}}）物理系畢業，之後到巴列維港（今[[安扎利港|安扎利港]]）的伊朗海軍服兵役。在[[伊朗伊斯蘭革命|伊朗伊斯蘭革命]]時，他加入左翼陣營，反對[[穆罕默德-礼萨·巴列维|沙王]]和後來的[[魯霍拉·穆薩維·何梅尼|何梅尼]]政權。他結合了兩位在抗戰時期被殺害的[[庫德族|庫德族]]裔人士－卡德爾和阿卜杜拉－的名字，以「卡德爾·阿卜杜拉」作為自己的筆名，以紀念這兩位朋友<ref name="vpro" /><ref name="Quran">{{cite news |last = Mahamdallie |first = Hassan |url = https://www.independent.co.uk/arts-entertainment/the-qur-an-and-the-messenger-by-kader-abdolah-book-reviews-a6865461.html |title = The Qur'an: A Journey and The Messenger: A Tale Retold, by Kader Abdolah; book reviews |trans-title = 《古蘭經：一個旅程和信使：故事再述》，卡德爾·阿卜杜拉著；書評 |publisher = [[The_Independent|The Independent]] |date = 2016-02-10 |language = en |accessdate = 2018-07-23 |archive-url = https://web.archive.org/web/20161227104521/http://www.independent.co.uk/arts-entertainment/the-qur-an-and-the-messenger-by-kader-abdolah-book-reviews-a6865461.html |archive-date = 2016-12-27 |dead-url = no }}</ref>。阿卜杜拉隨後於1988年以[[難民|政治難民]]身分逃難到荷蘭，最初落腳在[[阿珀爾多倫|阿珀爾多倫]]的一個難民庇護中心<ref name="vpro" /><ref name="LowCountry">{{cite web |last = Abdolah |first = Kader |translator = 保羅·文森（Paul Vincent） |url = http://www.literaturefromthelowcountries.eu/and-then-in-that-low-country/ |title = And then in that Low Country |website = Literature from the Low Countries |year = 2001 |language = en |accessdate = 2018-07-23 }}</ref>。短短五年，阿卜杜拉便精熟了荷蘭文，並在1993年出版了他的第一本荷蘭文著作《鷹》（{{lang|nl|De adelaars}}）<ref name="LowCountry" />。阿卜杜拉以這本短篇故事集獲得「{{link-en|金驢耳獎|Gouden Ezelsoor}}」的肯定。四年後，阿卜杜拉再出版他的首本小說作品《空瓶之旅》（{{lang|nl|De reis van de lege flessen}}）。2005年，他出版了他的代表作《[[清真寺之家|清真寺之家]]》，登上荷蘭暢銷書榜。該書並在2007年{{link-en|鹿特丹商報|鹿特丹商報}}和{{link-en|荷蘭廣播基金會|Nederlandse Programma Stichting}}（NPS）主辦的網路投票活動中，被票選為荷蘭史上最佳小說第二名<ref name="trouw1" />。

阿卜杜拉於2006年在[[萊頓大學|萊頓大學]]擔任客座作家<ref>{{cite web |url = https://www.universiteitleiden.nl/dossiers/gastschrijver/alle-gastschrijvers |title = Alle gastschrijvers |trans-title = 所有客座作家 |website = [[萊頓大學|Universiteit Leiden]] |language = nl |accessdate = 2018-07-23 }}</ref>。他現定居於[[台夫特|台夫特]]<ref>{{cite news |last = Knols |first = Karolien |url = https://www.volkskrant.nl/nieuws-achtergrond/kader-abdolah~bdad92c2/ |title = Kader Abdolah |publisher = de Volkskrant |date = 2008-04-26 |accessdate = 2018-07-23 |language = nl }}</ref>。2009年，卡德爾·阿卜杜拉獲頒[[格羅寧根大學|格羅寧根大學]]神學與宗教研究學院的[[榮譽博士|榮譽博士]]學位，因為他「成功且獨樹一格地向世人展現了伊斯蘭教的另一面」<ref name="vpro" />。
