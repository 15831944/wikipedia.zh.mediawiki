'''卡姆航空904號班機'''（[[英文|英文]]：'''Kam Air Flight 904'''）是一架於2005年2月3日當地時間4時墜毀於[[阿富汗|阿富汗]][[帕米爾高原|帕米爾高原]]地區的[[卡姆航空|卡姆航空]]（Kam Air）[[波音737|波音737-200型客機]]，該飛機原本預計計畫從阿富汗西部城鎮[[赫拉特|赫拉特]]飛往[[喀布爾|喀布爾]]，但是飛機在降落前受到暴風雪影響而在雷達上消失。這次墜毀事件造成飛機上96名乘客和8名機組人員全數死亡，救援人員於[[喀布爾|喀布爾]]以東的[[帕米爾高原|帕米爾高原]]處找到飛機殘骸和遺體，這也是迄今阿富汗歷史上最嚴重的空難事故。

== 背景 ==
卡姆航空公司是一家於2003年成立的私營航空公司，其主要工作是負責為阿富汗當地提供[[波音|波音]]或者[[安东诺夫航空科学技术联合体|安托諾夫航空科學技術聯合體]]製造飛機之國內和國際航線服務。發生空難事故的904號班機為該家航空公司所租貸、註冊編號EX-037[[波音737|波音737-200型客機]]，最早飛機是在1980年時由[[加拿大|加拿大]][[Nordair|Nordair]]所引進，之後在2004年4月時由公司總部設於[[阿拉伯聯合大公國|阿拉伯聯合大公國]][[沙迦_(城市)|沙迦]]（Sharjah）的[[鳳凰航空|鳳凰航空]]（Phoenix Aviation）買下以應付[[吉爾吉斯|吉爾吉斯]]航線需求，並且在同年11月時租借給卡姆航空公司調度使用<ref>{{Cite web |url= http://www.dailytimes.com.pk/default.asp?page=story_10-2-2005_pg4_18 |title= Owners of Afghan crash plane blame weather |author= [[路透社|路透社]] |date= 2005-02-10 |publisher= 《[[每日時報_(巴基斯坦)|每日時報]]》（Daily Times） |language= en |accessdate= 2013-01-23 |quote=  }}</ref><ref>{{Cite web |url= http://aviation-safety.net/photos/displayphoto.php?id=20050203-1&vnr=1&kind=PO |title= Photo of Boeing 737-242 N238TA |author=  |date= 2006-01-30 |publisher= [[航空安全網|航空安全網]] |language= en |accessdate= 2013-01-23 |quote=  }}</ref>。

== 事故 ==
民營航空公司卡姆航空904號班機於2005年2月3日從赫拉特國際機場起飛後，在剩15分鐘便能準備於[[喀布爾國際機場|喀布爾國際機場]]著陸時突然遭遇大規模[[冬季風暴|冬季風暴]]（Winter storm）影響。隨後在當地時間（[[世界統一時間|UTC]] +4:30）4時在暴風雪影響下與空中交通管制人員失去聯繫，之後緊接著雷達螢幕上也失去該架波音737-200型客機飛行時的反射訊號<ref>{{Cite web |url= http://aviation-safety.net/database/airport/airport.php?id=HEA |title= Herat Airport profile |author=  |date= 2008-07-13 |publisher= [[航空安全網|航空安全網]] |language= en |accessdate= 2013-01-23 |quote=  }}</ref><ref>{{Cite web |url= http://aviation-safety.net/database/airport/airport.php?id=KBL |title= Kabul-Khwaja Rawash Airport profile |author=  |date= 2008-07-13 |publisher= [[航空安全網|航空安全網]] |language= en |accessdate= 2013-01-23 |quote=  }}</ref>。

== 救援 ==
在惡劣的氣候條件[[北大西洋公約組織|北大西洋公約組織]]部隊和[[阿富汗國民軍|阿富汗國民軍]]（Afghan National Army）隨即展開救援行動，到了當地時間1時30分左右2架[[荷蘭皇家空軍|荷蘭皇家空軍]]的[[AH-64阿帕契直昇機|AH-64阿帕契直昇機]]發現大量飛機尾部的殘骸。到了2005年2月5日時，則在阿富汗首都[[喀布爾|喀布爾]]以東33公里（約20英哩）、海拔高3,400公尺（約11,000英呎）的山區高處發現飛機的主體殘骸<ref name="Mack Davis">{{Cite web|url=http://www.defense.gov/news/newsarticle.aspx?id=25895 |title=Afghan National Army Assists in Plane Crash Aftermath |author=Mack Davis |date=2005-02-14 |publisher=[[美國新聞局|美國新聞局]]（American Forces Press Service） |language=en |accessdate=2013-01-23 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20120414145144/http://www.defense.gov//News/NewsArticle.aspx?ID=25895 |archivedate=2012-04-14 }}</ref><ref>{{Cite web |url= http://aviation-safety.net/database/record.php?id=20050203-1 |title= Accident description |author=  |date= 2013-01-22 |publisher= [[航空安全網|航空安全網]] |language= en |accessdate= 2013-01-23 |quote=  }}</ref>。隨後駐阿富汗國際維和部隊嘗試展開多次直升機救援行動，但是受限於氣候環境的影響而無法順利進行；對此[[國防部_(阿富汗)|阿富汗國防部]]則要求阿富汗國民軍組織團隊，嘗試直接從陸路前往營救可能還存活的倖存者<ref name = "Mack Davis"/>。之後數10名阿富汗[[ANA突擊旅|ANA突擊旅]]（ANA Commando Brigade）便嘗試前往墜毀地區探查，但是受到暴風雪的影響而被迫撤離。過了第四天以後，國際維和部隊的救援部隊才能前往飛機墜毀處進行調查，並且確認飛機上96名乘客和8名機組人員全數遇難，並且在尋獲104具屍體的過程之中僅有1名乘客的遺體仍較為完整。

除了包含104名乘客和機組人員喪生外，飛機機體也很大程度上地遭到破壞。客機的[[飛行數據記錄器|飛行數據記錄器]]已經被尋獲並且移交由[[美國國家運輸安全委員會|美國國家運輸安全委員會]]分析，然而紀錄駕駛艙是否有向巴格拉姆空軍基地的[[美國空軍|美國空軍]]提出緊急迫降需求的[[飛行紀錄儀|飛行紀錄儀]]則沒有被尋獲<ref>{{Cite web |url= http://news.airwise.com/story/view/1108408951.html |title= Troops Start Moving Afghan Plane Crash Bodies |author= [[路透社|路透社]] |date= 2005-02-14 |publisher= Airwise |language= en |accessdate= 2013-01-23 |quote=  }}</ref>。

飛機上喪生的104人之中有96位乘客以及8名機組人員，其中至少有25人是並不是阿富汗當地居民，這包括有9名[[土耳其人|土耳其人]]、6名[[美國|美國人]]、4名[[俄羅斯|俄羅斯人]]、3名[[義大利|義大利人]]、1名[[荷蘭|荷蘭人]]、1名[[伊朗|伊朗人]]以及同時擁有[[加拿大|加拿大]]和俄羅斯[[多重國籍|國籍]]的副駕駛<ref>{{Cite web |url= http://www.cbsnews.com/stories/2005/02/03/world/main671642.shtml |title= Afghan Crash Killed All On Board |author= [[美聯社|美聯社]] |date= 2009-03-23 |publisher= [[哥倫比亞廣播公司|哥倫比亞廣播公司]] |language= en |accessdate= 2013-01-23 |quote=  }}</ref>。其中根據報導指出這些外國人分別是俄羅斯籍的機組人員、駐阿富汗之土耳其公司工作的土耳其籍文職人員、[[聯合國|聯合國]]為重建工程所邀請的義大利籍[[建築師|建築師]]安德烈·波拉斯特里（Andrea Pollastri）、1名義大利籍平民與[[義大利海軍|義大利海軍]][[海軍上校|海軍上校]]、1名負責西部盆地水資源運用團領導的荷蘭籍工程師，另外還有3名美國人則是為了[[麻薩諸塞州|麻薩諸塞州]][[劍橋_(麻薩諸塞州)|劍橋]]當地[[非政府組織|非政府組織]]活動而前往阿富汗工作<ref>{{Cite web |url= http://www.thedenverchannel.com/news/recent-du-grad-killed-in-afghan-airline-crash |title= Recent DU Grad Killed In Afghan Airline Crash |author=  |date= 2005-02-07 |publisher= [[KMGH-TV|KMGH-TV]] |language= en |accessdate= 2013-01-23 |quote=  }}</ref>。

== 原因 ==
目前對於為何會跟波音737-200失去聯繫以及[[航空器|航空器]]墜毀的原因仍然無法確定，對此[[塔利班|塔利班]]領導人[[杜拉赫|杜拉赫]]（Dadullah）則表示自己轄下的[[游擊戰|游擊部隊]]並沒有擊落飛機，並且對於飛機墜毀一事感到哀傷。對此[[巴基斯坦|巴基斯坦]]當地報紙和線上新聞網站更傳出卡姆航空904號班機在面臨天氣惡劣情況下，機組人員決定放棄由負責喀布爾空中交通管制之[[駐阿富汗國際維和部隊|駐阿富汗國際維和部隊]]管理的喀布爾國際機場，改轉往在[[美國軍事|美國軍方]]所管理的[[巴格拉姆空軍基地|巴格拉姆空軍基地]]（Bagram Airfield）緊急迫降。但是在飛行員報告只剩下15分鐘的燃料後，機場內的美軍空中交通管制人員仍拒絕飛機降落的請求，最後飛機因為燃料耗盡而墜毀<ref>{{Cite web |url= http://rense.com/general62/kam.htm |title= US Held Responsible For KAM AIR Disaster |author= Jeff Rense |date= 2005-02-08 |publisher= Rense |language= en |accessdate= 2013-01-23 |quote=  }}</ref>。
