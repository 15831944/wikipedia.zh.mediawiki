'''卡拉哈斯機場'''（{{airport codes|CKS|SBCJ}}） 是一座位於[[巴西|巴西]][[帕勞阿佩巴斯|帕勞阿佩巴斯]]市{{link-en|卡拉哈斯礦場|Carajás (proposed Brazilian state)}}外圍的機場，目前營運公司為國營的{{link-en|巴西機場基建公司|Infraero}}（Infraero）。

== 歷史 ==
機場是由[[淡水河谷公司|淡水河谷公司]]於1982年8月23日擇定在距[[帕勞阿佩巴斯|帕勞阿佩巴斯]]市中心18公里處設立，主要為提供[[帕勞阿佩巴斯|帕勞阿佩巴斯]]市對外交通及支援卡拉哈斯礦場作業。1985年由國營的{{link-en|巴西機場基建公司|Infraero}}負責營運。

== 飛安事故 ==
*1987年9月8日，[[巴西空軍|巴西空軍]]所屬[[英國航太|英國航太]]{{link-en|BAe125|BritishAerospace125}}噴射客機（國籍標誌及登記號碼：FAB-2129）在起飛時墜毀，機上載有3名機組員及6名乘客全數罹難<ref>{{Cite web | url=http://aviation-safety.net/database/record.php?id=19870908-0 | title=Accident description FAB-2129 |language=en| publisher=Aviation Safety Network | accessdate=2011-06-17}}</ref>。
*1997年2月14日，[[巴西航空|巴西航空]]所屬[[Boeing_737|Boeing 737-241]]（國籍標誌及登記號碼：PP-CJO）執飛由{{link-en|馬拉巴機場|Marabá Airport}}至卡拉哈斯機場的定期航班，在降落時遭遇雷雨天氣右側主起落架斷裂並向後擊中機翼導致飛機向右偏移出跑道，1名機組員死亡<ref>{{Cite web | url=http://aviation-safety.net/database/record.php?id=19970214-0 | language=en| title=Accident description PP-CJO | publisher=Aviation Safety Network | accessdate=2011-06-17}}</ref>。
