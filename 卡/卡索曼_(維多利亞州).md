'''卡索曼'''乃[[維多利亞州|維多利亞州]]的一座城市，地屬當年金礦區域（Goldfield）之內，位於距離[[府城|府城]][[墨爾本|墨爾本]]之西北120公-{里}-、[[本迪戈|本迪戈]]40公-{里}-處。根據統計資料，2006年卡索曼有14%的市民生於海外，3%於[[英國|英國]]。

== 沿革 ==
卡索曼最初應1851年的淘金熱而生，當年鎮名為'''Forrest Creek'''。

== 產業 ==
淘金熱退潮後，本市轉以發展食品業為主。

== 文化 ==
本市的文化節慶多與淘金熱和食品業有關，且以此發展觀光業。
