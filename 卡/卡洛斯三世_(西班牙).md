'''卡洛斯三世'''({{lang-es|'''Carlos III'''}}，1716年1月20日—1788年12月14日）[[西班牙波旁王朝|波旁王朝]]的[[西班牙君主列表|西班牙国王]]（1759年—1788年在位），即位前封号为[[帕爾馬公爵列表|帕尔马公爵]]（称'''卡洛一世'''，1731年—1735年）。他也是[[西西里和那不勒斯君主列表|那不勒斯国王]]（称'''卡洛七世'''，1735年—1759年）和[[西西里和那不勒斯君主列表|西西里国王]]（称'''卡洛四世'''，1735年—1759年）。

== 早年事迹 ==
卡洛斯三世是西班牙波旁王室的第一位国王[[腓力五世_(西班牙)|腓力五世]]第二次婚姻所生的儿子，1716年出生于[[马德里|马德里]]，他的母亲是帕尔马公国的郡主[[埃麗莎貝塔·法爾內塞|埃麗莎貝塔·法爾內塞]]。1731年，[[法尔内塞家族|法尔内塞家族]]绝嗣，15岁的卡洛斯因为母系血缘继承了得到帕尔马公国作为自己的封地。 他在1734年[[波兰王位继承战争|波兰王位继承战争]]期间趁[[哈布斯堡王朝|哈布斯堡王室]]无暇南顾，从[[奥地利|奥地利]]夺取那不勒斯和西西里，并以帕尔马公爵身份兼任两国君主。
== 那不勒斯與西西里國王 ==
十九歲的卡洛斯成為那不勒斯兼西西里國王之後，並未展現他後來那樣傑出的治國才幹與精力，而是將王位作為奢侈享受的特權。因此到1755年之前，他對政務細節頗為疏忽，將半數時間花費在打獵上，更一味滿足口腹之慾，把自己吃得肥肥胖胖，與後來的瘦長身材恰成對比。即使如此，因為知人善任，他仍有出色的政績。從1741年開始，卡洛斯在首席大臣{{Tsl|en|Bernardo Tanucci|貝爾納多·塔努奇}}（1752、1754年先後兼任司法部長與外交部長）的勸說鼓舞下，緩和了當地嚴酷的封建等級制度，打擊了長久以來把持當地的三個特權階層──貴族（控制國會、管制稅收並打擊改革）、教士（以接近迷信的方式使人民精神臣服）、律政人士（掌握地方行政權並對貴族和主教負責）。<ref name="杜蘭 ">（美）威爾·杜蘭著、幼獅文化公司譯，《世界文明史‧第31‧盧梭時代的南歐》，頁56-57</ref>

為了制衡貴族的強大勢力，卡洛斯吸引貴族來宮廷以便就近監視，並催生、成立支持他的新貴族；他勸阻年輕人流入修道院，把教士人數從十萬降低為八千一百人，課教產百分之二的稅，更限制教士的合法免稅權。貝爾納多·塔努奇限制貴族之司法權，向司法腐敗戰鬥、改革法律程序，又將嚴厲的刑法修改成溫和許多。本來卡洛斯還對[[猶太人|猶太人]]實施赦免及宗教寬容，但修道士以上帝之名警告他之後，他於是將赦免取消，維持對猶太人的禁令。<ref name="杜蘭 "/>

1759年异母兄长[[费迪南多六世|费迪南多六世]]国王去世后，卡洛斯三世继承了西班牙的王位。在卡洛斯啟程離開那不勒斯的時刻，那不勒斯人民為之悲傷難過，沿岸聚集著大批人群為國王送行，當人民送別這位「愛民如子的君主」時，據說大多數人都哭了出來。<ref>（美）威爾·杜蘭著、幼獅文化公司譯，《世界文明史‧第31‧盧梭時代的南歐》，頁70</ref>

== 国内政治 ==
{{Main|開明專制}}
卡洛斯三世起用[[冈波斯|冈波斯]]、[[洛里达布兰卡|洛里达布兰卡]]等大臣整顿国政，执行一种[[开明专制|开明专制]]政策。他还设立一个国务会议以协调各大臣的工作。在军事方面，卡洛斯三世把雇佣兵制度改为征兵制，并学习[[陆军|陆军]]强国[[普鲁士|普鲁士]]的训练方法。一时间，西班牙的国势颇有起色。卡洛斯三世采取的另一件有影响的大举措是：将[[耶稣会|耶稣会]]会士从西班牙及西班牙的海外领地驱逐出境。长期以来，耶稣会卷入一系列的政治阴谋之中，已经令许多国家，包括最重要的[[天主教|天主教]]国家西班牙和[[法国|法国]]感到不满。卡洛斯三世的举动可能直接影响了[[教皇|教皇]]后来解散耶稣会。

== 对外行动 ==
卡洛斯三世的对外政策相比之下显得有些冒进。他的主要方针是，联合同为[[波旁家族|波旁家族]]统治的法国反对[[英国|英国]]。 1761年，卡洛斯三世与法国国王[[路易十五|路易十五]]签订[[波旁家族条约|波旁家族条约]]，约定共同遏制英国。 但这个条约却使西班牙卷入了与英国直接对抗的、极不明智的[[七年战争|七年战争]]之中。 由于在战争中失败，西班牙失去了在[[北美洲|北美洲]]的重要殖民地[[佛罗里达|佛罗里达]]。 1770年，卡洛斯三世又险些在[[马尔维纳斯群岛|马尔维纳斯群岛]]（福克兰群岛）与英国发生冲突。

在[[北美独立战争|北美独立战争]]中，卡洛斯三世又站在殖民地人民一边反对宿敌英国，虽然有些勉强。 1779年，卡洛斯三世正式对英宣战，并为北美殖民地提供援助。 这次他找对了阵营：随着[[美国|美国]]的胜利与独立，西班牙又收回了佛罗里达并获得[[梅诺卡岛|梅诺卡岛]]（[[巴黎和约|巴黎和约]]）。
