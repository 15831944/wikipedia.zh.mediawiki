'''卡洛·杰苏阿尔多'''（{{lang-it|'''Carlo Gesualdo'''}}，{{bd|1566年|3月8日|1613年|9月8日}}），<ref>出生时间有争议。</ref>[[韦诺萨|韦诺萨]][[亲王|亲王]]、[[孔扎|孔扎]]（Conza）[[伯爵|伯爵]]。[[意大利|意大利]][[文艺复兴音乐|文艺复兴]]晚期杰出的[[作曲家|作曲家]]、[[鲁特琴|鲁特琴]]演奏家。

== 生平 ==
作为韦诺萨亲王，卡洛·杰苏阿尔多是在各种[[贵族|贵族]]特权的拥护下成长起来的，他生于[[那不勒斯|那不勒斯]]，最初使他名满意大利的是他的鲁特琴演奏技艺。由于他并非家中长子，他起初并不需要为娶妻生子而烦恼，但是1585年他兄长的突然死亡使卡洛得到亲王宝座的同时也导致了他的悲剧性婚姻——他的未婚妻d'Avalos不仅是他表妹，并且已经结过了两次婚，虽然卡洛与之结合，但是他却发现了妻子竟然与[[安德里亚|安德里亚]][[公爵|公爵]]有私情，恰好这时d'Avalos生了一个儿子，但是卡洛无论如何也不相信是他的孩子，所以他命人将这对奸夫淫妇连同孩子一起捉到他面前，并且将3人全部处死，更甚的是他还将他们的尸体公开示众。<ref>http://www.musicweb-international.com/gesualdo.htm</ref>尽管整个意大利震惊于他的暴行，但是他的生活并没有因此而受到一定程度的精神创伤以外的改变，1594年他搬到[[费拉拉|费拉拉]]，后又返回杰苏阿尔多城堡，并在该处去世。杰苏阿尔多的晚年一直受着[[抑郁症|抑郁症]]的折磨。

== 音乐 ==
他于1594年出版了两卷牧歌集子，正是这些的牧歌确立了他的日后盛名，接下来的几年间，他又发表了两卷牧歌集，1611年又发表了两卷。此外杰苏阿尔多的作品还包括一些宗教音乐等。他的音乐风格有着高度的[[半音|半音]]化特征以及怪异的[[和声|和声]]，这在当时相当超前。他的个人经历很有可能是他音乐充满了怪诞和阴暗色彩的重要原因。

== 后世影响 ==
杰苏阿尔多的音乐在20世纪越来越受重视，很多艺术家都对其表现出浓厚的兴趣：

* [[斯特拉文斯基|斯特拉文斯基]]曾整理改编过杰苏阿尔多的作品。

* 杰苏阿尔多的人生经历曾被写成各种体裁的文学作品，以及搬上[[歌剧|歌剧]]舞台或改编为其他形式的戏剧作品，包括俄罗斯先锋派作曲家[[施尼特凯|施尼特凯]]的歌剧以及德国导演[[韦纳·荷索|韦纳·荷索]]为[[ZDF|ZDF]]制作的电影等。
