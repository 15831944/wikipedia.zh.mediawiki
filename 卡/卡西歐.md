'''-{zh-hans:卡西欧计算机株式会社;zh-hant:卡西歐計算機株式會社;}-'''（{{lang-ja|'''カシオ計算機株式会社'''}}，{{lang-en|'''Casio Computer Company, Limited'''}}，{{tyo|6952}}）是一家跨國的消費性電子產品和商用電子產品製造公司，總部設在東京澀谷。其產品包括[[计算器|計算器]]，移動電話，[[数码相机|數碼相機]]，電子音樂儀器和[[手表|數碼手錶]]。該公司始建於1946年，並於1957年發布了世界上第一個全電子緊湊型計算器，同時亦是數碼相機的早期創新研發者之一。在20世紀的80和90年代，卡西歐開發出多款實惠的家用電子鍵盤。

2013年，卡西欧开通官方商城，针对[[中国大陆|中国大陆]]用户展开销售。



==歷史==
該公司於1946年4月由[[樫尾忠雄|樫尾忠雄]]創立，他是一名精通裝配的[[工程師|工程師]]。公司的名字來自「樫尾」的日語讀音「Kashio」，首款產品是[[香煙|香煙]]指環，使用者可以把香煙套在指環上，從而可在沒有煙灰缸時，不需用嘴或手指拿煙。當時剛結束[[二戰|二戰]]，香煙對當時的日本來說是一種價值商品，因此該款香煙指環十分成功，為樫尾帶來第一桶金。

樫尾在1949年於東京[[銀座|銀座]]舉行的商業展看到電動[[計算機|計算機]]後，便與他的兄弟合作，用他從指環賺到的錢開發計算機。在當時，不少計算機是手動或以[[馬達|馬達]]推動。樫尾有見於此，便以自己對電子學的知識，以[[螺線管|螺線管]]製造計算機，並於1954年成功研製首台電子計算機，當時的售價高達48.5萬日圓。與同期的機械式計算機比較，他的電子產品只有10個單位數字鍵，而機械產品除了個位數外，還需要另設10、100等多位數字鍵。

1957年，卡西歐推出14-A型計算機，也是全球首款全電子計算機。公司於同年更名称'''-{zh-hans:卡西欧计算机株式会社;zh-hant:卡西歐計算機株式會社;}-'''，專門研發電子產品。

== 公司沿革 ==
[[File:CASIO_14-A.jpg|thumb]][[国立科学博物馆|国立科学博物馆]]]]
* 1946年4月：[[樫尾忠雄|樫尾忠雄]]在[[東京都|東京都]][[三鷹市|三鷹市]]成立'''樫尾製作所'''。
* 1957年6月：開發出世界第一部沒有[[齒輪|齒輪]]的純電器式計算機「14-A型」。<ref>[http://casio.jp/dentaku/info/history/beginning/ 電卓の歴史（カシオ計算機のWebサイト）]</ref><ref>[http://www.dentaku-museum.com/calc/calc/2-casio/1-casiod/casiod.html 電卓博物館]</ref><ref>[http://www.casio.com.cn/company/nenpyo.html 主要产品一览] {{webarchive|url=https://web.archive.org/web/20141202060410/http://www.casio.com.cn/company/nenpyo.html |date=2014-12-02 }}</ref>樫尾忠雄、俊雄、和雄、幸雄四兄弟共同創設'''-{zh-hans:卡西欧计算机株式会社;zh-hant:卡西歐計算機株式會社;}-'''（{{lang-ja|カシオ計算機株式会社}}），第一任社長為四兄弟之父樫尾茂。
* 1960年4月：在東京都[[北多摩郡|北多摩郡]]大和町（現稱[[東大和市|東大和市]]）成立東京工廠（現改成東京事務所Tokyo Product Control and Technical Center）。
* 1965年9月：電子式桌上計算機「001型」問世。
* 1966年6月：將總部遷移至東京都北多摩郡大和町（現稱東大和市）。
* 1969年10月：在[[山梨縣|山梨縣]][[中巨摩郡|中巨摩郡]]玉穂町建立甲府工廠。
* 1972年8月：發表世界第一部個人計算機「{{link-ja|卡西歐迷你|カシオミニ}}」。
* 1974年5月：總部搬遷到東京都[[新宿區|新宿區]]。
* 1974年11月：推出「CASIOTRON」（{{lang-ja|カシオトロン}}）電子手錶，這是世界第一隻不須手動調整大小月或[[閏年|閏年]]的全自動電子錶。
* 1978年1月：發售第一部名片大小的計算機「卡西歐迷你名片LC-78型」（{{lang-ja|カシオミニカード}}），厚度僅3.9mm。
* 1978年2月：在臺灣成立海外生產工廠「臺灣刻時豪股份有限公司」。
* 1979年7月：成立八王子（Hachioji）工廠（現改成八王子研究所），且在東京都[[西多摩郡|西多摩郡]]羽村町（現稱[[羽村市|羽村市]]）成立羽村技術中心。
* 1979年10月：在[[山形縣|山形縣]][[東根市|東根市]]建立「山形卡西歐有限公司」。
* 1980年1月：推出電子樂器「卡西歐Tone」（{{lang-ja|カシオトーン}}）。
* 1980年2月：推出首支結合計算機功能的手錶Data Bank系列C-80-1型。
* 1983年4月：推出首支具防震功能的手錶G-Shock DW-5000C-1A型。
* 1989年2月：推出首支具有天氣預測感測器的數位手錶BM-100WJ-3型（PRO TERK系列尚未獨立之前身）。
* 1998年1月：將總部遷至東京都[[澁谷區|澁谷區]]初台。
* 2000年：[[移動電話|行動電話]]事業部成立，並發售{{link-ja|G'zOne|G'zOne}} C303CA型手機。
* 2004年4月：旗下行動電話事業部與[[日立|日立製作所]]合併，成為{{link-ja|卡西歐日立移動通訊有限公司|カシオ日立モバイルコミュニケーションズ}}。
* 2010年6月：卡西歐日立移動通訊有限公司與[[日本電氣|NEC]]旗下行動電話事業部整合。
* 2018年4月：宣佈退出小型數位相機，終止小型數位相機的生產<ref>{{Cite web |url = https://www.nikkei.com/article/DGXMZO29783480U8A420C1TJ3000/ |title = カシオ、コンパクトデジカメから撤退へ |author =  |date = 2018-04-24 |publisher = 日本経済新聞 |language = jp |accessdate = 2018-08-19 |archive-url = https://archive.today/20180818161423/https://www.nikkei.com/article/DGXMZO29783480U8A420C1TJ3000/ |archive-date = 2018-08-18 |dead-url = no }}</ref><ref>{{Cite web |url = https://casio.jp/dc/end/ |title = デジタルカメラ生産終了のお知らせ |author =  |date =  |publisher = カシオ |language = jp |accessdate = 2018-08-19 |archive-url = https://archive.today/20180515044947/https://casio.jp/dc/end/ |archive-date = 2018-05-15 |dead-url = no }}</ref>
