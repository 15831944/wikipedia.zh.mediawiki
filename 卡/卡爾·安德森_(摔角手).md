'''查德·艾雷葛拉'''（{{lang-en|Chad Allegra}}，{{bd|1980年|1月20日|||Karl Anderson}}）是出生於[[美國|美國]][[北卡羅來納州|北卡羅來納州]]的職業摔角手，以'''卡爾·安德森'''（Karl Anderson）作為個人主要擂台名稱。

他過去有在{{tsl|ja|新日本プロレス|新日本職業摔角}}裡拿下過四次雙打冠軍，並寫下過新日本雙打冠軍的最長持有記錄。他也是摔角反派團體{{tsl|en|Bullet Club|子彈俱樂部}}的創始成員之一，且擔任過該團體的第二任領導者。

== 經歷 ==
=== 早年時期 ===
在學生時代查德主要投入的運動是[[棒球|棒球]]，等到搬家至[[辛辛那提|辛辛那提]]時他開始接觸摔角界，並在{{tsl|en|Les Thatcher|萊斯·蔡徹}}下接受短期摔角訓練。<ref name="FIM">{{cite web |url=http://www.footinmouth.net/machinegun/history.html |archiveurl=https://web.archive.org/web/20070927094816/http://www.footinmouth.net/machinegun/history.html |dead-url=yes |title=Machine Gun Anderson History |work=Foot In Mouth Productions |language=en |archivedate=2007年9月27日 |accessdate=2018年2月3日}}</ref>在萊斯·蔡徹的訓練期間，查德曾於2000年意外遭到另一名訓練生{{tsl|en|Derick Neikirk|德瑞克·涅科克}}所傷引發[[腦震盪|腦震盪]]，而休養了將近一年。<ref name="FIM"/>

後續查德有在羅傑·盧芬（Roger Ruffen）以及北方摔角聯盟（Northern Wrestling Federation、NWF）下接收更長期的教導。<ref name="FIM"/>

=== NWF時期 ===
查德首次正式露面是2002年5月期間的NWF賽事上，之後曾有和{{tsl|en|Chris Harris (wrestler)|克里斯·哈里斯}}、{{tsl|en|Shark Boy|鯊魚小子}}，以及[[傑利·勞勒|傑利·勞勒]]等人對戰過。<ref name="OWW">{{cite web |url=http://www.onlineworldofwrestling.com/bios/k/karl-anderson/ |title=Karl Anderson |work=Online World of Wrestling |language=en |date=2018年1月30日 |accessdate=2018年2月3日}}</ref>

2005年夏季NWF有和中心區域摔角協會（Heartland Wrestling Association、HWA）交流對戰，查德當時是以NWF冠軍身份對上HWA冠軍選手寇帝·霍克（Cody Hawk）。<ref name="OWW"/>

=== NWA時期 ===
2005年10月查德在全國摔角同盟（National Wrestling Alliance、NWA）的賽事上露面時，受到當時新日本職業摔角洛杉磯道場分館人員戴夫·馬奎茲（Dave Marquez）的注意。<ref name="Slam">{{cite web |url=http://slam.canoe.com/Slam/Wrestling/2013/01/30/20538151.html |title=Getting to know the unknown 'Machine Gun' Karl Anderson |work=Slam! Sports |author=Greg Oliver |language=en |date=2016年4月3日 |accessdate=2018年2月3日}}</ref>在2006年1月1日防衛NWF冠軍腰帶的比賽失利後，查德接收到戴夫·馬奎茲的邀約前往洛杉磯道場受訓，並在美國西岸當地摔角圈登場。<ref name="Slam"/>此期間查德在擂台上身份被重新包裝成是摔角團體{{tsl|en|Anderson family|安德森家族}}的成員，除了開始採用「卡爾·安德森」的擂台名稱之外，也開始以「機關槍」作為個人外號。<ref name="OWW"/>

在NWA闖蕩時期，安德森有與{{tsl|en|Joey Ryan (wrestler)|喬伊·萊恩}}合組雙打團隊，之後有將團隊名稱改成「真美國英雄」（Real American Heroes）。他們的組合曾奪下過NWA雙打冠軍，並且有多次腰帶防衛成功記錄。<ref name="Slam"/>

=== PWG時期 ===
2007年起安德森開始有在職業游擊戰摔角（Pro Wrestling Guerrilla、PWG）上露面，之後2008年2月期間他參加了PWG的世界冠軍錦標賽，最後是於決賽裡失利。<ref>{{cite web |url=http://www.prowrestlingguerrilla.com/results/2008/show0085.html |title=Dynamite Duumvirate Tag Team Title Tournament Round Robin Qualifier Series |work=Pro Wrestling Guerrilla |language=en |accessdate=2018年2月3日}}</ref>

安德森在休養了約11個月時間後，於2009年2月21日再次復出，並在當年8月與老搭檔喬伊·萊恩合作，對上當時的PWG雙打冠軍{{tsl|en|The Young Bucks|揚·巴克斯}}兄弟，但最後未能挑戰腰帶成功。<ref>{{cite web |url=http://www.prowrestlingguerrilla.com/results/2009/show0104.html |title=Speed of Sound |work=Pro Wrestling Guerrilla |language=en |accessdate=2018年2月3日}}</ref>

2009年9月4日安德森登場了他在PWG的最後一場賽事，結果是輸給了對手{{tsl|en|Roderick Strong|羅德瑞克·史壯}}。<ref>{{cite web |url=http://www.prowrestlingguerrilla.com/results/2009/show0105.html |title=Guerre Sans Frontières |work=Pro Wrestling Guerrilla |language=en |accessdate=2018年2月3日}}</ref>

安德森首次在新日本職業摔角的登場是在2008年3月23日裡，當時他取代了{{tsl|ja|永田裕志|永田裕志}}來與{{tsl|ja|金本浩二|金本浩二}}舉行單打賽。<ref>{{cite web |url=https://www.cagematch.net//?id=1&nr=18677 |title=NJPW Circuit 2008 New Japan Cup ~Who Is The Highest~ - Tag 1 |work=Cage Match |language=en |accessdate=2018年2月3日}}</ref>最後安德森雖輸了那場比賽，但他的演出讓新日本願意向他簽下一年合約。<ref name="Slam"/>

加入新日本的安德森，最初是加入當時由{{tsl|ja|天山廣吉|天山廣吉}}率領的「最兇惡偉大反派」（Great Bash Heel、GBH）團體，<ref>{{cite web |url=http://www.puroresucentral.com/anderson.html |title=Karl Anderson |work=Puro Resucentral |language=en |accessdate=2018年2月3日}}</ref>之後改跳脫到[[中邑真輔|中邑真輔]]所領軍的「Chaos」裡，與裡頭成員之一{{tsl|en|Matt Bloom|麥特·布魯恩|巨人伯納德}}合組名為「Bad Intentions」的雙打組合，並在2009年《G1雙打錦標賽》上成為積分最多的獲勝團隊。<ref>{{cite web |url=http://www.prowrestlinghistory.com/supercards/japan/newjapan/g1tag.html#2009 |title=G-1 Climax Tag Team League 2009 |work=Pro Wrestling History |author= |language=en |accessdate=2018年2月3日}}</ref>

2010年4月4日上演了Chaos將Bad Intentions給驅逐的戲碼，離開了Chaos後的Bad Intentions則在當年6月19日所舉行《Dominion 6.19》大賽上拿下IWGP雙打冠軍。<ref>{{cite web |url=http://www.puroresufan.com/njpw/2010/06/19/results-new-japan-dominion-619-619/ |title=[Results] New Japan "DOMINION 6.19", 6/19  |work=Puroresufan |author= |language=en |date=2010年6月19日 |accessdate=2018年2月3日}}</ref>Bad Intentions隨後展開了長期的腰帶防衛成功戰績，其中在2011年6月18日於《Dominion 6.18》上擊敗來自Noah職業摔角聯盟的{{tsl|ja|佐野巧真|佐野巧真}}與{{tsl|ja|高山善廣|高山善廣}}雙打組合外，還一併奪下對方所持有的GHC雙打冠軍腰帶。<ref>{{cite web |url=https://www.pwtorch.com/artman2/publish/Other_News_4/article_50865.shtml#.WnUfy3UjFD8 |title=NJPW News: Results from Dominion PPV Saturday - MVP's first IWGP IC Title defense, TNA wrestler appears, key title change |work=Pro Wrestling Troch |author=James Caldwell |language=en |date=2011年6月18日 |accessdate=2018年2月3日}}</ref>到了2011年9月9日Bad Intentions的IWGP雙打腰帶防衛成功已長達446天，達到之前[[蝶野正洋|蝶野正洋]]與天山廣吉的雙打組合「蝶天」所保持記錄。

2012年1月4日《Wrestle Kingdom VI》上Bad Intentions腰帶防衛失敗，將IWGP雙打冠軍腰帶易手給天山廣吉與{{tsl|ja|小島聡|小島聰}}團隊「Tencozy」，結束了長達564天，在此之前最長的持有IWGP雙打腰帶時間。<ref>{{cite web |url=http://www.njpw.co.jp:80/match/detail_result_game.php?e=470&c=3714 |archiveurl=https://web.archive.org/web/20120108184339/http://www.njpw.co.jp/match/detail_result_game.php?e=470&c=3714 |dead-url=yes |script-title=ja:NJPW 40th anniversary Tour. レッスルキングダム? in 東京ドーム |work=New Japan Pro-Wrestling Co,.Ltd |language=ja |archivedate=2012年1月8日 |date=2012年1月4日 |accessdate=2018年2月3日 }}</ref><ref>{{cite web |url=https://www.cagematch.net/?id=5&nr=148 |title=IWGP Tag Team Championship |work=Cage Match |language=en |accessdate=2018年2月3日}}</ref>Bad Intentions之前所奪下的Noah聯盟GHC雙打冠軍腰帶，則在當月22日輸給了Noah所屬選手{{tsl|ja|斎藤彰俊|齋藤彰俊}}與[[秋山準|秋山準]]。而Bad Intentions的GHC雙打腰帶持有時間則長達218天。<ref>{{cite web |url=http://www.noah.co.jp/result.php?tour_id=898 |script-title=ja:2012年1月22日(日) 試合開始:17:00 大阪府立体育会館(ボディメーカーコロシアム) 3,400人 |work=N O A H. |language=ja |accessdate=2018年2月3日}}</ref>

輸掉兩項雙打腰帶沒多久後，巨人伯納德在3月改以「天災大帝」（Lord Tensai）的身份回歸[[WWE|WWE]]聯盟，結束與安德森多年的合作。<ref>{{cite web |url=http://www.wrestleview.com/viewnews.php?id=1332217948 |archiveurl=https://web.archive.org/web/20120322234244/http://www.wrestleview.com/viewnews.php?id=1332217948 |dead-url=yes |title=SPOILER: Identity of new WWE wrestler |work=Wrestle View |author= Mike Tedesco |language=en |archivedate=2012年3月22日 |date=2012年3月20日 |accessdate=2018年2月3日}}</ref>

單飛後的安德森有在當年夏季《[[G1_Climax|G1 Climax]]》循環賽上打出5勝3敗的成績，最後是在總決賽上輸給了{{tsl|ja|オカダ・カズチカ|岡田和睦}}。

==== 子彈俱樂部 ====
[[File:BULLET_CLUB.JPG|thumb]]
2013年5月3日的《Wrestling Dontaku 2013》大賽上，安德森輸了與{{tsl|ja|棚橋弘至|棚橋弘至}}的一對一單打賽，但賽後另幾名外籍選手{{tsl|en|Finn Bálor|芬恩·貝勒|戴比特王子}}、{{tsl|en|Bad Luck Fale|厄運費勒}}，以及原先在旁觀看比賽的{{tsl|en|Tama Tonga|塔瑪·東加}}闖入了擂台與安德森一同圍攻棚橋弘至。隨後他們在節目上宣佈成立了以戴比特王子為首的反派團體「{{tsl|en|Bullet Club|子彈俱樂部}}」。<ref>{{cite web |url=http://www.njpw.co.jp/match/detail_result_game.php?e=708&c=5320 |archiveurl=https://web.archive.org/web/20130601033010/http://www.njpw.co.jp/match/detail_result_game.php?e=708&c=5320 |dead-url=yes |script-title=ja:レスリングどんたく 2013 |work=New Japan Pro-Wrestling Co,.Ltd |language=ja |archivedate=2013年6月1日 |date=2013年5月3日 |accessdate=2018年2月3日}}</ref><ref>{{cite web |url=http://www.njpw.co.jp/news/detail.php?nid=9478 |archiveurl=https://web.archive.org/web/20130509181451/http://www.njpw.co.jp/news/detail.php?nid=9478 |dead-url=yes |script-title=ja:「イッツ・リィィィアル!! 」なんとアンダーソン、トンガとも合体!!　デヴィットが外国人軍団“BULLET CLUB”結成!! |work=New Japan Pro-Wrestling Co,.Ltd |language=ja |archivedate=2013年5月9日 |date=2013年5月3日 |accessdate=2018年2月3日}}</ref>

當年10月期間，安德森將目標瞄準去年在《G1 Climax》所輸給的對手——岡田和睦身上的IWGP冠軍腰帶。之後於11月9日《Power Struggle》的IWGP冠軍腰帶爭奪戰裡，安德森最後仍未戰勝岡田和睦，<ref>{{cite web |url=http://www.njpw.co.jp/match/detail_result.php?e=772 |archiveurl=https://web.archive.org/web/20131111230216/http://www.njpw.co.jp/match/detail_result.php?e=772 |dead-url=yes |script-title=ja:POWER STRUGGLE |work=New Japan Pro-Wrestling Co,.Ltd |language=ja |archivedate=2013年11月11日 |date=2013年11月9日 |accessdate=2018年2月3日}}</ref>不過在11日裡子彈俱樂部引進了新成員[[路克·蓋洛斯|蓋洛斯]]，安德森與蓋洛斯的雙打組合在23日《世界雙打錦標賽》上，成為了最後優勝的隊伍。<ref>{{cite web |url=http://www.njpw.co.jp/match/detail_result.php?e=792 |archiveurl=https://web.archive.org/web/20131212005338/http://www.njpw.co.jp/match/detail_result.php?e=792 |dead-url=yes |script-title=ja:WORLD TAG LEAGUE 2013 |work=New Japan Pro-Wrestling Co,.Ltd |language=ja |archivedate=2013年12月7日 |date=2013年12月12日 |accessdate=2018年2月3日}}</ref>

2014年1月4日《Wrestle Kingdom 8》上，安德森與蓋洛斯擊敗了[[哈利·史密斯_(摔角手)|史密斯 Jr.]]和{{tsl|en|Lance Hoyt|蘭斯·霍伊特|蘭斯·亞瑟}}的組合，奪下IWGP雙打冠軍。<ref>{{cite web |url=https://www.pwtorch.com/artman2/publish/otherppvs/article_75461.shtml#.WnUsA3UjFD8 |title=CALDWELL'S NJPW TOKYO DOME RESULTS 1/4: Complete "virtual-time" coverage of New Japan's biggest show of the year - four title changes, former WWE/TNA stars featured, more |work=Pro Wrestling Troch |author=James Caldwell |language=en |date=2014年1月4日 |accessdate=2018年2月3日}}</ref>在4月6日《Invasion Attack 2014》的大賽裡安德森有成功守下雙打冠軍腰戴，但賽程結束後戴比特王子宣佈將離開新日本，讓安德森成為子彈俱樂部的新領導者。<ref>{{cite web |url=https://www.njpw.co.jp/news/detail.php?nid=11490 |archiveurl=https://web.archive.org/web/20140502032030/https://www.njpw.co.jp/news/detail.php?nid=11490 |dead-url=yes |script-title=ja:5月度のスマホサイトは、いま一番悪いヤツら“BULLET CLUB”が完全ジャック!!　アンダーソンの心中は? |work=New Japan Pro-Wrestling Co,.Ltd |language=ja |archivedate=2014年5月2日 |date=2014年4月30日 |accessdate=2018年2月3日}}</ref>隔週安德森有參與新日本在[[台灣|台灣]]的海外巡迴表演賽，當時他與蓋洛斯擊敗了{{tsl|ja|後藤洋央紀|後藤洋央紀}}與特別打扮成{{nihongo|「台灣隊長」|キャプテン·台湾}}造型的{{tsl|ja|平澤光秀|平澤光秀}}，繼續保有雙打冠軍。<ref>{{cite web |url=http://www.njpw.co.jp/match/detail_result_game.php?e=852&c=6238 |archiveurl=https://web.archive.org/web/20140414190948/http://www.njpw.co.jp/match/detail_result_game.php?e=852&c=6238 |dead-url=yes |script-title=ja:WRESTLING WORLD 2014 in 台湾 |work=New Japan Pro-Wrestling Co,.Ltd |language=ja |archivedate=2014年4月14日 |date=2014年4月13日 |accessdate=2018年2月3日}}</ref>

9月23日安德森再次輸給岡田和睦，失去IWGP冠軍腰帶的挑戰資格。到了年底《G1雙打錦標賽》裡，他與蓋洛斯這回在打入總決賽裡輸給後藤洋央紀與{{tsl|ja|柴田勝頼|柴田勝賴}}。<ref>{{cite web |url=http://www.njpw.co.jp/match/detail_result.php?e=913 |archiveurl=https://web.archive.org/web/20141211051544/http://www.njpw.co.jp/match/detail_result.php?e=913 |dead-url=yes |script-title=ja:WORLD TAG LEAGUE 2014 |work=New Japan Pro-Wrestling Co,.Ltd |language=ja |archivedate=2014年12月11日 |date=2014年12月7日 |accessdate=2018年2月3日}}</ref>到了2015年1月4日《Wrestle Kingdom 9》雙打冠軍防衛賽上，他們同樣輸給後藤洋央紀與柴田勝頼，將持有剛好滿一年的雙打腰帶易手。<ref>{{cite web |url=https://www.pwtorch.com/artman2/publish/otherppvs/article_82579.shtml#.WnUyYnUjFD8 |title=CALDWELL'S NJPW TOKYO DOME SHOW RESULTS 1/4: Complete "virtual-time" coverage of live Wrestle Kingdom PPV - Tanahashi vs. Okada, Bullet Club, Nakamura, Jim Ross, more |work=Pro Wrestling Troch |author=James Caldwell |language=en |date=2015年1月4日 |accessdate=2018年2月3日}}</ref>當年夏季安德森再次加入《G1 Climax》的巡迴賽，此回他在積分方面以2分之差輸給了[[中邑真輔|中邑真輔]]與岡田和睦。

2016年1月4日《Wrestle Kingdom 10》裡安德森與蓋洛斯輸掉他們第3次得到的雙打冠軍後，隔天開始有消息傳出他與蓋洛斯即將離開新日本前往[[WWE|WWE]]發展。<ref>{{cite web |url=http://www.pwtorch.com/site/2016/01/04/pwtorch-report-four-big-names-leaving-new-japan-heading-to-wwe/ |archiveurl=https://web.archive.org/web/20160105055406/http://www.pwtorch.com/site/2016/01/04/pwtorch-report-four-big-names-leaving-new-japan-heading-to-wwe/ |dead-url=yes |title=PWTorch Report – Four big names leaving New Japan, heading to WWE |work=Pro Wrestling Troch |author=Sean Radican、James Caldwell |language=en |archivedate=2016年1月5日 |date=2016年1月4日 |accessdate=2018年2月3日}}</ref>安德森與蓋洛斯在2月20日《Honor Rising: Japan 2016》大賽裡出席了他們在新日本的最後一次比賽，當時他們與子彈俱樂部另兩名成員{{tsl|en|Bad Luck Fale|厄運費勒}}以及{{tsl|en|Tama Tonga|塔瑪·東加}}，一同聯手對戰後藤洋央紀、柴田勝頼、{{tsl|en|Bobby Fish|巴比·費許}}，以及{{tsl|en|Kyle O'Reilly|凱勒·歐萊禮}}等四人。雖然安德森那一方最後輸了比賽，但賽後費勒與東加有向他與蓋洛斯祝福，他與蓋洛斯最後也有以[[叩頭|叩頭]]方式向觀眾致謝。

=== WWE時期 ===
==== 與A·J的重組 ====
[[File:Luke_Gallows_and_Karl_Anderson_in_September_2016.jpg|thumb]]（左）是安德森從新日本到WWE裡長期合作的夥伴。]]
2016年4月11日安德森與蓋洛斯在WWE的《[[WWE_Raw|RAW]]》節目上，以亂入的方式登場攻擊了[[烏索兄弟|烏索兄弟]]。<ref>{{cite web |url=http://www.wwe.com/shows/raw/2016-04-11#full-detail-40003948 |title=The Usos def. Heath Slater & Curtis Axel to advance in the No. 1 Contenders' Tag Team Tournament, sponsored by Booty-O’s |work=[[WWE|WWE]] |language=en |date=2016年4月11日 |accessdate=2018年2月3日}}</ref>隔週過去曾與安德森一同於子彈俱樂部合作的[[A·J·史泰爾斯|A·J·史泰爾斯]]在和[[羅曼·瑞恩斯|羅曼·瑞恩斯]]互相在擂台上對峙後，安德森與蓋洛斯同樣採用亂入的方式闖進會場攻擊了羅曼。<ref>{{cite web |url=http://www.wwe.com/shows/raw/2016-04-18#full-detail-40004263 |title=Luke Gallows & Karl Anderson attacked WWE World Heavyweight Champion Roman Reigns |work=WWE |language=en |date=2016年4月18日 |accessdate=2018年2月3日}}</ref>之後安德森、蓋洛斯，與A·J三人則宣佈成立了名為「俱樂部」（The Club）的組合團體。

安德森在俱樂部成立後在節目上被分配到的定位，主要是與蓋洛斯合作協助A·J與羅曼·瑞恩斯之間的對抗，以搶下對方當時持有的[[WWE世界冠軍|WWE世界冠軍]]腰帶。之後因無法奪得羅曼身上腰帶的緣故，出現了A·J想離開俱樂部的劇情線。到了5月30日另一名WWE選手[[約翰·希南|約翰·希南]]久違重返到節目上時，A·J與安德森及蓋洛斯再次攜手合作一同攻擊希南，也開始呈現安德森與蓋洛斯協助A·J向希南對戰的情節。<ref>{{cite web |url=http://www.wwe.com/shows/raw/2016-05-30#full-detail-40006146 |title=AJ Styles confronted John Cena |work=WWE |language=en |date=2016年5月1日 |accessdate=2018年2月3日}}</ref>

之後在夏季期間裡，WWE規劃出讓旗下選手分別各在《RAW》與《[[WWE_Friday_Night_SmackDown!|SmackDown]]》兩邊節目登場的方案，當時因安德森與蓋洛斯被分配到《RAW》，A·J被調到《SmackDown》裡情況，讓俱樂部的劇情線暫此結束。

俱樂部的劇情結束後，安德森與蓋洛斯將目標放在{{tsl|en|The New Day (professional wrestling)|The New Day}}持有的雙打冠軍上，在8月21日《[[夏日衝擊|夏日衝擊]]》大賽上他們雖然擊敗了The New Day，但因為是以取消資格方式獲勝，故沒有成功奪下對方的腰帶。<ref>{{cite web |url=http://www.wwe.com/shows/summerslam/2016/wwe-tag-team-champions-new-day-vs-gallows-anderson-result |title=Luke Gallows & Karl Anderson def. WWE Tag Team Champions The New Day via disqualification |work=WWE |author=Matthew Artus |language=en |date=2016年8月21日 |accessdate=2018年2月3日}}</ref>到了11月20日的安德森與蓋洛斯有被列為代表《RAW》一方的雙打隊伍之一，與來自《SmackDown》節目的其他雙打組合在《[[強者生存|強者生存]]》展開一場十對十的對抗大戰，最後安德森隸屬的《RAW》有拿下勝利。<ref>{{cite web |url=http://www.wwe.com/shows/survivorseries/2016/raw-vs-smackdown-live-traditional-survivor-series-tag-team-elimination |title=Team Raw def. Team SmackDown LIVE in a 10-on-10 Traditional Survivor Series Tag Team Elimination Match |work=WWE |author=Michael Burdick |language=en |date=2016年11月20日 |accessdate=2018年2月3日}}</ref>

2017年1月29日安德森與蓋洛斯在《[[皇家大戰|皇家大戰]]》上擊敗了[[席莫斯|席莫斯]]與[[安東尼奧·希薩羅|希薩羅]]的組合，拿下他們第一次在WWE的雙打冠軍。<ref>{{cite web |url=http://www.wwe.com/shows/royalrumble/2017/cesaro-sheamus-vs-luke-gallows-karl-anderson-results |title=Luke Gallows & Karl Anderson def. Cesaro & Sheamus to win the Raw Tag Team Championships |work=WWE |author=Greg Adkins |language=en |date=2017年1月29日 |accessdate=2018年2月3日}}</ref>不過他們在4月2日第33屆《[[摔角狂熱|摔角狂熱]]》的四組雙打團體攻防戰上落敗，把腰帶易手給[[哈迪兄弟|哈迪兄弟]]。<ref>{{cite web |url=http://www.wwe.com/shows/wrestlemania/raw-tag-team-championship-triple-threat-ladder-match-results |title=The Hardy Boyz won a Fatal 4-Way Ladder Match to win the Raw Tag Team Championship |work=WWE |author=WWE.com Staff |language=en |date=2017年4月2日 |accessdate=2018年2月3日}}</ref>

== 擂台資料 ==
=== 個人招式 ===
==== 終結技 ====
*電擊槍（Gun Stun）
:為摔角裡的「切擊」（Cutter）類型技巧，是站在背對著對手正面的位置後，兩手緊扣住對手頸部來使勁拉向地面痛砸的招式。

*火箭彈踢擊（Rocket Kick）
:以衝刺的力道跳躍起來後，來朝向對手頭部攻擊的踢擊。

*毀滅之踢（Boot of Doom）
:與蓋洛斯合作使出的招式。先讓蓋洛斯將對手扛在肩上後，再以火箭彈踢擊來攻擊對手頭部。

*魔幻殺手（Magic Killer）
:與蓋洛斯合作使出的招式。先和蓋洛斯分別抓住對手上下半身來舉到半空後，把對手甩向一旁的地面重摔。

==== 其它代表招式 ====
*迴旋背脊轟炸（Spinning Spinebuster）
*垂直落下式（Brainbuster）
*死亡谷炸彈摔（Death Valley Bomb）
*頸部粉碎技（Neckbreaker）
*千噸落下撞擊（Senton）
*逆蝦型固定（Boston Crab）
*跳躍小腿踢擊（Calf Kick）
