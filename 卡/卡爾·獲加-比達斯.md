'''卡爾·獲加-比達斯'''（{{lang-en|'''Kyle Walker-Peters'''}}，{{bd|1997年|4月13日|catIdx=Walker-Peters, Kyle}}），[[英格蘭|英格蘭]]足球運動員，現時效力[[英格蘭超級足球聯賽|英格蘭超級足球聯賽]]球會[[托定咸熱刺足球會|熱刺]]。

==生涯==
===熱刺===
[[托特纳姆热刺足球俱乐部|热刺]]於2013年7月1日簽下獲加-比達斯。<ref name="spurs-profile">{{cite web |url=http://www.tottenhamhotspur.com/development-squad/kyle-walker-peters/ |title=Kyle Walker-Peters Profile |publisher=Tottenham Hotspur |accessdate=11 June 2017 |deadurl=yes |archiveurl=https://web.archive.org/web/20170428063001/http://www.tottenhamhotspur.com/development-squad/kyle-walker-peters/ |archivedate=2017年4月28日 }}</ref>2015年5月，他在球隊於馬來西亞及澳洲的季前賽首次為一隊上陣。<ref name="spurs-profile" />2015年12月，他被選為英超U21每月最佳球員。<ref name="thesun-scouting-report">{{cite web |url=https://www.thesun.co.uk/sport/football/2139724/kyle-walker-peters-tottenham-wonderkid-right-back/ |title=Who is Kyle Walker-Peters? The Tottenham wonderkid set to rival Kyle Walker for the right-back role |publisher=The Sun |author=Wally Downes Jr |date=6 June 2017 |accessdate=21 June 2017 }}</ref>Wally Downes Jr在報紙「The Sun」形容獲加-比達斯為一個「優秀的熱刺青訓右閘」及「有作為10號位的思路，同時亦有翼鋒所要的速度」<ref name="thesun-scouting-report" />2017年2月，獲加-比達斯與球會續約至2019年。<ref> {{cite web |url=http://www.standard.co.uk/sport/football/kyle-walkerpeters-over-the-moon-after-signing-new-tottenham-contract-a3466861.html |title=Kyle Walker-Peters 'over the moon' after signing new Tottenham contract |publisher=Evening Standard |location=London |date=14 February 2017 |accessdate=21 June 2017 }}</ref>

==國家隊==
獲加-比達斯被教練{{link-en|保羅·森遜|Paul Simpson (footballer)}}徵召參加[[2017年國際足協U-20世界盃|2017年國際足協U-20世界盃]]，他共上陣5場及決賽，協助英格蘭自1966年以來再奪該項錦標。<ref name="telegraph-wc2017">{{cite web |url=http://www.telegraph.co.uk/football/2017/06/11/england-vs-venezuela-u20-world-cup-final-live-score-updates/ |title=England U20s seal World Cup glory as Dominic Calvert-Lewin scores and Freddie Woodman saves penalty |publisher=Daily Telegraph |date=11 June 2017 |accessdate=11 June 2017 }}</ref><ref>{{cite web |url=http://www.fifa.com/fifa-tournaments/players-coaches/people=403064/index.html |title=FIFA Tournaments - Kyle Walker-Peters |publisher=FIFA.com |accessdate=11 June 2017}}</ref>
