'''卡里姆·阿卜杜勒-贾巴尔'''（{{lang-ar|'''كريم عبد الجبار'''}}，{{lang-en|'''Kareem Abdul-Jabbar'''}}），原名為'''小费迪南德·刘易斯·阿尔辛多'''（{{lang-en|'''Ferdinand Lewis Alcindor, Jr.'''}}，{{bd|1947年|4月16日|catIdx=J}}）是[[美国|美国]]最伟大的[[NBA|NBA]][[篮球|篮球]]运动员之一，从高中开始直到NBA都是一名出色的[[中鋒_(籃球)|中鋒]]球员。生涯20年效力過[[密爾瓦基公鹿|密爾瓦基公鹿]]及[[洛杉磯湖人|洛杉磯湖人]]。在職業生涯中，贾巴尔十九次入選[[NBA全明星賽|NBA全明星賽]]，拿過六次[[NBA總冠軍|NBA總冠軍]]和[[NBA年度MVP|NBA年度MVP]]，兩次被選為[[NBA總決賽最有價值球員|NBA總決賽最有價值球員]]。1996年，他被評為[[NBA50大巨星|NBA50大巨星]]之一 。

贾巴尔至今仍保持着NBA比赛总得分的最高纪录，退役后成为一名成功的[[作家|作家]]和兼职[[演员|演员]]。他在球員時期最出众的技巧是「'''[[勾手|勾手]]'''」，因此綽號「'''天鉤贾巴尔'''」比全名更為有名。退役后他出版了多本著作，在种族、宗教问题上经常发言，成为[[公共知识分子|公共知识分子]]。他在是[[时代杂志|时代杂志]]的专栏作家。 2012年，他被[[國務卿|國務卿]]-[[希拉蕊·柯林頓|希拉蕊·柯林頓]]選為美國全球文化大使 。2016年，他獲頒了[[總統自由勳章|總統自由勳章]]。

== 高中时代 ==
賈霸生于纽约市哈林區的黑人社區，是百貨公司價格檢查員Cora Lillian唯一的孩子。因為身高的關係，所以从中学时代的場上位置便為中锋。他曾带领母校的球队夺得三次纽约市天主教学校比赛的冠军，并创造了连胜71场的辉煌战绩，獲得了高年級亞軍。球队該年的总成绩为96胜6负。

== 大学时代 ==
賈霸在1967年進入[[加利福尼亚大学洛杉磯分校|加利福尼亚大学洛杉磯分校]]的篮球队打球，一年級時曾在一次練習賽中帶領新生隊擊敗了當年拿到冠軍的高年級生組成的隊伍<ref>灌篮高手动画第六集13分左右介绍</ref>，且三年來为球队取得了88胜2负的傲人战绩。[[哈林篮球队|哈林篮球队]]曾开出100万美元的酬劳邀请其加盟，但是贾巴尔还是拒绝了这个优厚的条件，毅然决定参加NBA。

在加州大学洛杉磯分校比赛期间，賈霸的左[[眼球|眼球]]在1968年1月12日的比賽中被湯姆·亨德森擊中並劃傷了左角膜。賈霸錯過了接下來的兩場對陣史丹佛大學和波特蘭大學的比賽，之后他几乎都戴著防护[[眼罩|眼罩]]参加比赛。

== 改信伊斯蘭教和抵制1968奧運 ==
在加州大学读书期间，賈霸改为信仰[[伊斯兰教|伊斯兰教]]，直到1971年才開始公開使用他的阿拉伯名字。当年[[NBA总决赛|NBA总决赛]]，賈霸带领雄鹿队以4比0轻松战胜巴尔的摩子弹队（现名为[[华盛顿巫師队|华盛顿巫師队]]）后的第二天（5月1日），他对外公布了自己的改名。但是他反复否认了自己与伊斯兰国家的任何联系，他皈依伊斯兰教是因为他在加州大学就读时所跟随的一位[[土耳其|土耳其]]长老的影响。

賈霸抵制了[[1968年夏季奧林匹克運動會|1968年夏季奧林匹克運動會]] ，決定不加入[[美國國家男子籃球隊|美國國家男子籃球隊]] ，抗議非裔美國人遭受的不平等待遇。

== NBA时期 ==
在1969年的[[NBA选秀|NBA选秀]]大会上，[[菲尼克斯太阳队|菲尼克斯太阳队]]与[[密尔沃基公鹿队|密尔沃基公鹿队]]用猜硬币的方式来争夺贾巴尔。结果后者幸运地赢得了賈霸的加盟。贾霸在公鹿队待了5个赛季。1975年，公鹿队为了获得[[中锋|中锋]]埃尔莫尔·史密斯、后卫布里安·温特斯以及自由球员戴夫·梅尔斯和朱尼尔·布莱吉曼，将贾巴尔交换给[[洛杉矶湖人队|洛杉矶湖人队]]。

在湖人队期间，为了提高自己的灵活性，賈霸从1976年开始练习[[瑜伽|瑜伽]]功。

除了他的眼罩，賈霸最出名的就是他的大[[勾手|勾手]]投篮，其他球员几乎很难防守其勾手，他也因此被称为“'''天勾賈霸'''”。

賈霸在NBA整整打了20个赛季，到1989年才宣布退役，总得分38,387分，是NBA历史最高纪录。

== 教练生涯 ==
1998年，賈霸以1美元的象征性薪水义务执教[[亚利桑那州|亚利桑那州]]白河市的一所高中篮球队。2000年，他担任[[洛杉矶快船队|洛杉矶快船队]]副教练，协助[[吉姆·托德|吉姆·托德]]工作。2002年，他执教[[USBL|USBL]]的[[俄克拉荷马|俄克拉荷马]]暴风队。2004年，又执教[[纽约尼克斯队|纽约尼克斯队]]。2005年9月，賈霸宣佈擔任洛杉機湖人隊的球隊助教。

== 個人生活 ==
=== 健康問題 ===
賈霸患有偏頭痛，他曾利用大麻來減輕自己的症狀城染色體陽性慢性粒細胞白血病 ，血液和骨髓癌。 該疾病於2008年12月被診斷出來，但賈霸說他的病情可以透過每天服用口服藥物來治療，每隔一個月回診，並定期分析血液。他在新聞發布會上表示，他不認為這種疾病會影響他的正常生活。賈霸成為[[諾華公司|諾華公司]]的代言人，該公司生產他的癌症藥物[[伊馬替尼|伊馬替尼]]。

2011年2月，賈霸透過Twitter宣布，他體內的癌細胞已經「100%被清除」。幾天后，他澄清了他的錯誤陳述，表示他的癌症並不是完全治癒了，只是被控制在最低限度。賈霸說：「你不可能完全擺脫癌細胞，我本應該知道這點的。」日前，賈霸在科學公園高中作了一次演講，其中談到了他的健康狀況。「現在我體內的癌細胞只是被控制在了最低限度。」現年70歲的賈霸在2008年被診斷患了慢性髓細胞白血病。他表示：「在我生命的這個階段，現在癌細胞不會造成生命威脅了。」

=== 非運動榮譽 ===
2011年，賈霸榮獲[[Double_Helix_Medal|雙螺旋勳章]]，致力於提高癌症研究的意識。同樣在2011年，賈霸獲得了紐約理工學院的榮譽學位。2016年底，賈霸被前美國[[總統|總統]][[巴拉克·歐巴馬|巴拉克·歐巴馬]]授予[[總統自由勳章|總統自由勳章]]。

=== 武術興趣 ===

賈霸自年輕起即熱衷於學習武術及[[瑜珈|瑜珈]]作為養生健身之用，並因此在60年代末成為在[[洛杉磯|洛杉磯]]教授[[截拳道|截拳道]]的[[李小龍|李小龍]]的弟子。由於兩者間的師徒關係，賈霸在李小龍[[1972年|1972年]]的遺作[[死亡遊戲|死亡遊戲]]中客串演出了死亡塔頂的巨人武術家與李小龍交手。
