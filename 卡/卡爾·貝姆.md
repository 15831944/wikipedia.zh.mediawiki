'''卡尔·伯姆'''（{{lang-de|Karl Böhm}}，{{bd|1894年|8月24日|1981年|8月14日|catIdx=B}}），[[奥地利|奥地利]][[指揮家|指揮家]]，被认作是[[德奥|德奥乐派]]管弦乐和歌剧最权威的诠释者。

== 生平 ==
貝姆出生於奧地利[[格拉茨|格拉茨]]一個熱愛音樂的家庭，父親雖然知道他有音樂天賦並為之高興，但並不十分同意讓音樂成為卡爾·貝姆的職業。貝姆於是在做指揮助理、學習音樂的同時學習法律，後就讀於[[維也納音樂與表演藝術大學|維也納音樂學院]]、{{link-de|格拉茨音樂與表演藝術大學|Universität für Musik und darstellende Kunst Graz|格拉茨音樂學院}}學習鋼琴和音樂理論。卡爾·貝姆1917年在家鄉格拉茨開始職業指揮生涯，1921年在[[布魯諾·瓦爾特|布魯諾·瓦爾特]]的支持下成為慕尼克[[巴伐利亞國立歌劇院|巴伐利亞國立歌劇院]]指揮。1927年在[[達姆施塔特|達姆施塔特]]擔任音樂總監，1931年遷職[[漢堡|漢堡]]，1934至1943年任職[[德累斯頓薩克森國立樂團|德累斯頓薩克森國立樂團]]首席指揮，1943年起執導[[維也納國立歌劇院|維也納國立歌劇院]]等樂團。[[二次大戰|二次大戰]]末期，1944年8月，卡爾·伯姆被希特勒擬定為受保護的藝術家之一，免於上前線演出。由於和納粹黨靠近的歷史，卡爾·伯姆在戰後被聯軍免去維也納國立歌劇院指揮職務，禁止上臺指揮直至1947年。

貝姆於1928年首演[[貝爾格|貝爾格]]的《[[伍采克|伍采克]]》，並成為《伍采克》與《[[露露|露露]]》的權威詮釋者。1933年因《{{le|阿拉貝拉|Arabella}}》的演出，貝姆與[[理查·史特勞斯|理查·史特勞斯]]結下不解之緣。之後首演理查·施特勞斯的《{{le|沉默的女人|Die schweigsame Frau}}》以及題獻給貝姆的《{{le|達佛涅 (歌劇)|Daphne (opera)}}》。此外貝姆亦在各地不斷指揮演出[[莫札特|莫札特]]、[[華格納|華格納]]、理查·施特勞斯等人為數不少的[[歌劇|歌劇]]，及德奧作曲家為主的[[交響曲|交響曲]]作品。1970年因為他對[[維也納|維也納]]音樂貢獻卓著而榮獲{{link-de|奧地利共和國科學和藝術榮譽勳章|Österreichisches Ehrenzeichen für Wissenschaft und Kunst}}。

1981年8月14日貝姆在[[薩爾茲堡|薩爾茲堡]]排練理查·史特勞斯的《[[Elektra_(歌劇)|Elektra]]》時逝世，享年86歲。

== 成就 ==
卡爾·貝姆指揮過數量眾多的著名交響樂團，與[[維也納愛樂樂團|維也納愛樂樂團]]的關係尤其緊密，和[[柏林愛樂樂團|柏林愛樂樂團]]也合作錄製過許多唱片，其中包括莫札特的交響曲全集。伯姆還指揮過歐美各地許多著名歌劇院的演出，比如[[大都會歌劇院|大都會歌劇院]]、[[柏林國立歌劇院|柏林國立歌劇院]]，與[[拜羅伊特音樂節|拜羅伊特音樂節]]、[[薩爾茨堡音樂節|薩爾茨堡音樂節]]等很多音樂節保持著幾十年的緊密聯繫。卡爾·伯姆錄製了許多唱片，其中一些還排成了電影，他指揮演出的重點是德奧古典和浪漫時期的作品。人們認為他對[[莫札特|莫札特]]有偏愛，貝姆本人認為莫札特是一個「革命者」。在德累斯頓的時候，他與理查·施特勞斯相交甚好，因而指揮過他很多作品的首演。在公眾心目中，與同胞[[卡拉揚|卡拉揚]]涉足甚廣的風格不同，貝姆更多的被認為是一位本土音樂家，他們同為維也納愛樂至今為止僅有的兩位榮譽指揮(Ehrendirigent)。

== 指揮風格 ==
總體來說，貝姆的指揮樸實大方、四平八穩，被譽為“精準大師”。雖然有人說他「學究氣」重，可能是針對他忠實原譜而言，不會擅自加入戲劇性效果；而他這種自然不造作的風格，其實反倒為絕大多數樂迷所稱道。特別是莫札特的作品，貝姆即使年事已高，也不斷對之研究探索。他演奏的莫札特簡潔而不浮華，卻又不失幽默。

== 家庭 ==
卡尔·伯姆的儿子是[[演员|演员]]、[[慈善家|慈善家]][[卡尔海因茨·伯姆|卡尔海因茨·伯姆]]。
