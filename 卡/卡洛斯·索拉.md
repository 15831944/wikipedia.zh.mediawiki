'''卡洛斯·索拉'''（{{lang-es|'''Carlos Saura Atarés'''}} ，{{bd|1932年|1月4日|catIdx=Saura}}）是一位[[西班牙|西班牙]][[導演|導演]]與[[攝影家|攝影家]]，曾以《[[蕩婦卡門|蕩婦卡門]]》（''Carmen''）入圍[[奧斯卡最佳外語片獎|奧斯卡最佳外語片獎]]，也曾獲得[[坎城影展評審團大獎|坎城影展評審團大獎]]與[[最佳导演银熊奖|柏林影展最佳導演獎]]。他與[[查理·卓別林|查理·卓別林]]的[[女兒|女兒]][[潔洛汀·卓別林|潔洛汀·卓別林]]（Geraldine Chaplin）在1967年開始成為親密伴侶，並維持12年之久。

==早期生活==
卡洛斯·索拉出生於一個[[藝術家|藝術家]]家庭（他的母親是一位[[鋼琴家|鋼琴家]]，他的弟弟[[安東尼奧·索拉|安東尼奧·索拉]]則是[[畫家|畫家]]），他的藝術感從童年時期就開始發展，當時他是一個攝影愛好者。

1957年，他在[[馬德里|馬德里]]電影研究與實驗學會得到了導演文憑，他也在此教書直到1963年為止。

==導演生涯==
卡洛斯·索拉在1957年至1958年完成他的第一部電影《昆卡》。1962年，他的電影《[[小流氓|小流氓]]》（''Los Golfos''）被公認對西班牙青年[[社會|社會]]產生強大的影響，探討馬德里的最貧窮地區的[[青少年|青少年]]犯罪。四年後（1966年），他在[[第16屆柏林國際電影節|第16屆柏林國際電影節]]以《[[狩獵|狩獵]]》（''La Caza''）獲得了[[最佳導演銀熊獎|最佳導演銀熊獎]]<ref name="Berlinale 1966">{{cite web |url=http://www.berlinale.de/en/archiv/jahresarchive/1966/03_preistr_ger_1966/03_Preistraeger_1966.html |title=Berlinale: 1966 Prize Winners |accessdate=2010-02-23 |work=berlinale.de}}</ref>。在1967年，他的電影《薄荷冰沙》也獲得了[[第18屆柏林國際電影節|第18屆柏林國際電影節]][[最佳導演銀熊獎|最佳導演銀熊獎]]<ref name="Berlinale 1968">{{cite web |url=http://www.berlinale.de/en/archiv/jahresarchive/1968/03_preistr_ger_1968/03_Preistraeger_1968.html |title=Berlinale: 1968 Prize Winners |accessdate=2010-03-05 |work=berlinale.de}}</ref>。1981年，他以《趕快》（''Deprisa, Deprisa''）贏得了第31屆柏林國際電影節[[金熊獎|金熊獎]]<ref name="Berlinale 1981">{{cite web |url=http://www.berlinale.de/en/archiv/jahresarchive/1981/03_preistr_ger_1981/03_Preistraeger_1981.html |title=Berlinale: 1981 Prize Winners |accessdate=2010-08-29 |work=berlinale.de}}</ref>。

1973年，卡洛斯·索拉拍攝的電影《[[安琪莉卡表妹|安琪莉卡表妹]]》獲得高度讚譽，奪得[[坎城電影節評審團獎|坎城電影節評審團獎]]。1975年電影《[[飼養烏鴉|飼養烏鴉]]》獲得了[[坎城電影節評審團大獎|坎城電影節評審團大獎]]。他的電影於1979年被提名[[奧斯卡獎|奧斯卡獎]]最佳外語片獎。

卡洛斯·索拉的電影以[[弗拉門戈舞|弗拉門戈舞]]和其他[[西班牙舞蹈|西班牙舞蹈]]而著名。20世紀80年代，他的弗拉門戈三部曲包括《血婚》、《[[蕩婦卡門|蕩婦卡門]]》和《[[魔愛|魔愛]]》。他與西班牙弗拉門戈舞蹈家[[克里斯蒂娜·奧約斯|克里斯蒂娜·奧約斯]]共同合作。他後來也執導電影《[[弗拉門戈|弗拉門戈]]》（1995年）、《[[情慾飛舞|情慾飛舞]]》（1998年）和《Fados》（2007）。

1989年，他執導的電影《黑夜》也參加[[第39屆柏林國際電影節|第39屆柏林國際電影節]]正式競賽<ref name="Berlinale 1989">{{cite web |url=http://www.berlinale.de/en/archiv/jahresarchive/1989/02_programm_1989/02_Programm_1989.html |title=Berlinale: 1989 Programme |accessdate=2011-03-13 |work=berlinale.de}}</ref>。

卡洛斯·索拉認為以[[超現實主義|超現實主義]]大師[[路易斯·布努埃爾|路易斯·布努埃爾]]為主軸的電影《布努埃爾與所羅門王》是他執導過最出色的電影作品。卡洛斯·索拉在接受線上電影雜誌《[[DearCinema.com|DearCinema.com]]》[[記者|記者]]採訪時<ref>{{cite web |url=http://dearcinema.com/camera-is-my-memory-carlos-saura/ |title=DearCinema.com |publisher=DearCinema.com |date=2008-04-16 |accessdate=2013-02-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20100115031421/http://dearcinema.com/camera-is-my-memory-carlos-saura |archivedate=2010-01-15 }}</ref>，他說：「《布努埃爾與所羅門王》是我所執導過的最偉大的電影。我喜歡這部電影，但是其他人似乎都不喜歡它。我敢肯定，布努埃爾會喜歡這部電影。但是，也許只有他會喜歡它。你在電影中看到的一切，實際上是基於我和他的談話內容。」

1990年，他以《[[卡梅拉！|卡梅拉！]]》獲得了[[哥雅獎|哥雅獎]]最佳導演和最佳劇本。卡洛斯·索拉被選為1992年[[巴塞羅那奧運會|巴塞羅那奧運會]]官方電影《馬拉松》的導演。2004年，卡洛斯·索拉獲得[[歐洲電影獎|歐洲電影獎]]終身成就獎。

2008年，卡洛斯·索拉榮獲第10屆[[孟買國際電影節|孟買國際電影節]]頒發的終身成就獎，該影展由[[孟買科學院|孟買科學院]]（MAMI）所舉辦<ref>{{cite web |url=http://dearcinema.com/spanish-director-carlos-saura-to-get-lifetime-achievement-award-at-mumbai-fest/ |title=Spanish director Carlos Saura to get Lifetime Achievement Award at Mumbai Fest |publisher=DearCinema.com |date=2008-02-22 |accessdate=2013-02-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20091201111000/http://dearcinema.com/spanish-director-carlos-saura-to-get-lifetime-achievement-award-at-mumbai-fest/ |archivedate=2009-12-01 }}</ref>。

==私生活==
卡洛斯·索拉結過兩次[[婚姻|婚姻]]。他首先於1957年在[[巴塞羅那|巴塞羅那]]與阿黛拉·梅德拉諾結婚。他們有兩個兒子，卡洛斯（生於1958年）和安東尼奧（生於1960年）。1982年12月27日，他迎娶了梅賽德斯·佩雷斯。他們有三個兒子，曼努埃爾（生於1980年），阿德里安（生於1984年）和聖地亞哥（生於1987年）。

卡洛斯·索拉與[[查理·卓別林|查理·卓別林]]的[[女兒|女兒]][[潔洛汀·卓別林|潔洛汀·卓別林]]（Geraldine Chaplin）育有一個[[兒子|兒子]]謝恩（生於1974年）。一些人認為，他可能曾有過更多的孩子。他的第二次婚姻後，他也有一位的女兒名為安娜（生於1994年12月）。
