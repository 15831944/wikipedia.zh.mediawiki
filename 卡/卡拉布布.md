'''卡拉布布'''（[[印尼语|印尼语]]：'''{{lang|id|Kalabubu}}'''或'''{{lang|en|Kala bubu}}'''）是居住在[[印度尼西亚|印度尼西亚]][[北苏门答腊|北苏门答腊]][[北尼亚斯县|北尼亚斯县]]地区的{{link-en|尼亚斯人|Nias people|尼亚斯战士}}佩戴的类似于[[项圈|项圈]]的圆形[[项链|项链]]。卡拉布布象征着[[男性气概|男性气概]]、勇敢顽强、战争的胜利和[[英雄主义|英雄主义]]，而被普遍称为“[[猎首|猎首]]的项链”。从历史上看，只有那些在战争中表现出高超能力，并把敌对部落首领斩首的英勇战士才有资格佩戴卡拉布布。正因如此，佩戴卡拉布布的人通常被看作是部落的英雄。除此之外，在神圣的战争仪式期间，舞蹈者也会佩戴卡拉布布{{sfn|Muhammad Husni|Tiarma Rita Siregar|2000|p=47}}。如今，大多数{{link-en|尼亚斯人|Nias people}}已经将佩戴卡拉布布视为习以为常的事情{{sfn|Richter|Carpenter|2012|p=428}}。

== 形式 ==
卡拉布布是一个直径大约在{{convert|22|cm|inch}}到{{convert|25|cm|inch}}之间的圆形项链，类似于[[项圈|项圈]]。卡拉布布的厚度是不均匀的，其中心部分的厚度最大，然后向两端逐渐变小。其收缩的两端连接形成一个磁盘，一般由黄铜和少量黄金装饰。而卡拉布布的内部框架，则是由铁或黄铜制成的，并在其末端塑造一个类似于圆盘的象盘。圆形的椰子壳（有时可用龟壳替代）通常被镶嵌在内部的黄铜框架旁边<ref>{{cite web |url=http://www.museum-nias.org/istiadat-nias/ |title=Istiadat Nias |trans-title=Nias Tradition |language=[[印尼语|印尼语]] |author=<!--Not stated--> |date=2017 |website=Museum Pusaka Nias |publisher=Yayasan Pusaka Nias |access-date=2017-11-20 |quote=}}</ref>。用磁盘片将它连接在一起之后进行打磨和抛光的工序，直至打磨到其有一个较为平滑的表面才算完成。最后，卡拉布布会被染成黑色{{sfn|Marschall|2013|p=128}}。卡拉布布内部的黄铜框架象征着[[尼亚斯神话|尼亚斯神话]]中伟大的金蛇{{sfn|Richter|Carpenter|2012|p=428}} 。

== 象征 ==
卡拉布布与部落之间的战争还有猎首有关。卡拉布布象征着[[男性气概|男性气概]]、勇敢顽强、战争的胜利和[[英雄主义|英雄主义]]。从历史上看，只有那些在战争中表现出高超能力，并把敌对部落首领斩首的英勇战士才有资格佩戴卡拉布布。正因如此，佩戴卡拉布布的人通常被看作是部落的英雄。在神圣的战争仪式期间，舞蹈者也会佩戴卡拉布布{{sfn|Muhammad Husni|Tiarma Rita Siregar|2000|p=47}}。如今，大多数{{link-en|尼亚斯人|Nias people}}已经将佩戴卡拉布布视为习以为常的事情{{sfn|Richter|Carpenter|2012|p=428}}。

== 变体 ==
在[[印度尼西亚|印度尼西亚]][[北苏门答腊省|北苏门答腊省]][[北尼亚斯县|北尼亚斯县]]，被称作尼费塔里－塔里（[[印尼语|印尼语]]：{{lang|id|nifitali-tali}}）的项链被视为卡拉布布的珍贵金属变体。尼费塔里－塔里通常是用贵金属（如[[银|银]]或[[黄金|黄金]]）制作的，被视为高地位的象征，所以尼费塔里－塔里都是高地位的男人在喜庆的场合穿戴的{{sfn|Richter|Carpenter|2012|p=428}}。
