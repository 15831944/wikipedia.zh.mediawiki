'''卡爾十五世'''（{{lang|sv|Karl XV}}，{{bd|1826年|5月31日|1872年|9月18日}}），原名'''卡爾·盧德維格·歐根'''（{{lang|sv|Karl Ludvig Eugen}}），是1859年至1872年間的[[瑞典國王|瑞典國王]]和[[挪威國王|挪威國王]]，他在挪威的稱號為'''卡爾四世'''（{{lang|no|Karl IV}}）。

== 早年 ==
卡爾於1826年在[[斯德哥爾摩王宮|斯德哥爾摩王宮]]出生，即獲封為斯科訥公爵。他曾受教於哲學家[[克里斯托法·雅各布·博斯特倫|克里斯托法·雅各布·博斯特倫]]和歷史學家[[弗雷德里克·費爾德南德·卡爾松|弗雷德里克·費爾德南德·卡爾松]]，1843年和1845年入讀[[烏普薩拉大學|烏普薩拉大學]]，1844年春在奧斯陸求學。他喜歡藝術，藝術學院教授隆德奎斯特和水彩畫家[[約翰·克里斯蒂安·貝爾耶|約翰·克里斯蒂安·貝爾耶]]都是他的藝術老師。他在1856年獲任命為挪威副王，在挪威期間常常在郊外寫生。

1850年代時，斯堪的納維亞各國關係十分良好，瑞典和挪威的卡爾王儲和丹麥國王[[弗雷德里克七世|弗雷德里克七世]]自1846年在[[哥本哈根|哥本哈根]]會面後，一直都是好朋友。卡爾希望推動各國在政治上的合作。1857年夏，[[奧斯卡一世|奧斯卡一世]]患上重病，[[瑞典議會|瑞典]]和[[挪威議會|挪威議會]]通過由卡爾王儲攝政。<ref>Louis de Geer, Minnen del 1 Sthlm 1892 s 157 f.</ref>

== 國王 ==
奧斯卡一世在1857年9月25日病逝，卡爾繼位為瑞典及挪威國王。他在位期間，瑞典和挪威的關係轉趨惡化。

他熱心於[[泛斯堪的納維亞主義|泛斯堪的納維亞主義]]，但未能如願。他在1863年7月和[[弗雷德里克七世|弗雷德里克七世]]在斯庫德斯堡會談後，答應[[丹麥首相|丹麥首相]][[卡爾·克里斯蒂安·霍爾|卡爾·克里斯蒂安·霍爾]]，瑞典會協助丹麥防衛[[石勒蘇益格|石勒蘇益格]]。然而，兩國正式協商前，卡爾便因國內政治形勢而被迫放棄支援丹麥。

他的北歐政策未能成功，但國內法治則見強化。正如他的格言「國家以法律建成（{{lang|sv|Land skall med lag byggas}}）」<ref>''Svenska kungliga valspråk'', [[Lars_O._Lagerqvist|Lars O. Lagerqvist]], Nils Åberg, Kenneth Andersson, Vincent Förlag, Stockholm 1996 ISBN 91-87064-13-8 s. 26</ref>，他進行了瑞典的法律改革，社區法（1862年）、教會法（1863年）和刑罰法（1864年）都沿用至今。他也進行了以[[路易斯·德·吉爾_(1818年－1896年)|路易斯·德·吉爾]]為首的議會改革，1866年廢除了[[四院制|四院制]]，建立了現時的[[瑞典議會|瑞典議會]]。1876年，他任命路易斯·德·吉爾為瑞典第一任[[瑞典首相|首相]]。

他也推動基礎建設和工業化，下令建造了瑞典最大的鐵路。1868年11月，瑞典、挪威和日本簽訂「友誼、貿易和航海條約」。根據《橫濱條約》，日本開放[[函館|函館]]、[[橫濱|橫濱]]、[[長崎市|長崎]]、[[神戶|神戶]]和[[大阪市|大坂]]給瑞典商旅經商，[[瑞典-挪威|瑞典-挪威]]可往這些商埠派駐外交人員，而且瑞典人和挪威人享有[[領事裁判權|領事裁判權]]。

他在1872年9月18日死於[[馬爾默|馬爾默]]。由於他的兒子早夭，所以由三弟[[奧斯卡二世|奧斯卡二世]]繼位。

== 個人 ==
卡爾十五世的詩作於1863年和1865年出版，後來翻譯為法語、英語、匈牙利語、意大利語、荷蘭語、德語、丹麥語和挪威語。他鼓勵瑞典和挪威的藝術發展，除了加入了藝術學院外，自己也購入了很多藝術品。他的收藏包括400多名瑞典畫家的500多張畫作，死後捐為國有，大幅增加了[[瑞典國立博物館|國家博物館]]的斯堪的納維亞藝術品。他的畫作可見於國家博物館、議會大樓、藝術學院、[[哥德堡博物館|哥德堡博物館]]、[[挪威藝術、建築和設計國家博物館|挪威國家美術館]]和[[瑞典王宮|瑞典各王宮]]。

== 後裔 ==
卡爾十五世與[[荷蘭的露易莎|荷蘭的露易莎]]結婚，共有一子一女：
*[[瑞典的路易莎公主|'''路易莎'''·约瑟芬娜·尤金妮娅]]（'''Lovisa''' Josefina Eugenia，1851年10月31日－1926年3月20日），丹麥王后
*[[南曼蘭公爵卡爾·奧斯卡王子|'''卡爾·奧斯卡'''·威尔海姆·弗雷德里克]]（'''Carl Oscar''' Vilhelm Fredrik，1852年12月14日－1854年3月13日），南曼蘭公爵

雖然卡爾十五世的愛女路易莎，未能繼承瑞典王位，但日後許多丹麥和挪威君王也同時是路易莎的後裔。路易莎嫁予丹麥國王[[弗雷德里克八世|弗雷德里克八世]]，誕下了卡爾王子和克里斯蒂安王子。1905年[[瑞典-挪威聯合解體|挪威獨立]]後，卡爾王子被[[挪威國會|挪威國會]]立為國王[[哈康七世|哈康七世]]；克里斯蒂安王子於1912年繼承丹麥王位，是為[[克里斯蒂安十世|克里斯蒂安十世]]。

現任挪威國王[[哈拉爾五世|哈拉爾五世]]的父系和母系先祖都擁有卡爾十五世的血統。

卡爾十五世有一个私生子[[卡尔·约翰·勃兰德|卡尔·约翰·勃兰德]]（{{lang|sv|Carl Johan Bolander}}）。著名[[赞美歌|赞美歌]]作家，主教[[尼尔斯·勃兰德|尼尔斯·勃兰德]]是他的儿子。
