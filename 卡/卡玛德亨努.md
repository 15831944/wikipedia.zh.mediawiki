'''卡玛汗奴'''（{{lang|en|Kamadhenu}}，{{lang|sa|कामधेनु}}），也被称为苏罗毗（{{lang|sa|सुरभि}}），[[印度教|印度教]]中神圣的牛女神，是所有奶牛的母亲。它也是一头神奇的“财富之牛”、“如意神牛”，能满足主人的一切愿望，通常认为它还是十一楼陀罗（{{lang|en|Rudras}}）的母亲。在画像中，她常被描绘为一头长着女人头和胸的白牛，或体内包含了各种神灵的白牛。在印度教中，[[瘤牛|瘤牛]]被看成是卡玛汗奴在尘世的化身而受到崇拜，因此，卡玛汗奴不是一位受独立崇拜的女神，也没有专门供奉它的寺庙。而是通过虔诚的印度教徒普遍对牛的尊敬而受到尊崇。

印度教经文提供卡玛汗奴诞生的不同说法。有些神话说它诞生于诸神和阿修罗搅乳海之时；另说她是造物主{{tsl|en|Daksha|达刹}}之女，[[迦叶波|迦叶佛]]（Kashyapa）之妻。还有一些经文讲述了诸神将卡玛汗奴送给了仙人{{tsl|en|Jamadagni|阇摩陀尼}}或{{tsl|en|Vashista|婆私吒}}（二者均是古代圣人），而国王迦陀毗里耶想把它从圣人那里偷来，最终遭遇可怕的后果-被阇摩陀尼之子波罗舒罗摩（Parashurama）杀死。卡玛汗奴主要为圣主提供供奉的牛奶和奶制品；她也能变出凶猛的武士来保护圣主。它除了住在圣人的隐居之处，还住在奶牛之国{{tsl|en|Goloka|高洛卡}}和地府“[[帕塔拉|帕塔拉]]”。

==崇拜==
一些寺庙和住宅中有用于崇拜的卡玛汗奴画像<ref name = "monier"/>。但是，从没有对她的专门祭仪，也没有任何一座寺庙把她作为主神来祭拜<ref name = "monier"/><ref>{{cite book|last=怀特|first= 大卫·戈登|title=瑜伽母之吻: 南亚"密宗性爱"|publisher=芝加哥大学出版|year=2003年|page = 54页|isbn=978-0-226-89483-6|chapter =苏罗毗,众牛之母}}</ref>。按照莫尼尔-威廉姆斯的说法：“她相当于活的动物[牛]的永恒代表”<ref name = "monier">{{cite book|last=莫尼尔-威廉姆斯|first=莫尼尔|authorlink=莫尼尔-威廉姆斯|title=婆罗门教和印度教:印度的宗教思想和生活|url=http://www.archive.org/stream/brahmanismhindu00moniuoft#page/n337/mode/2up|year=1887|publisher=London Murray}}</ref>。在印度，寺庙四周经常饲养着牛，并定期在礼拜五和特殊场合举行祭仪。每一头牛都被“虔诚的印度教徒”视为是卡玛汗奴下凡（俗世的化身）<ref>{{cite book|last=拉奥|first=T.A. 哥琵纳塔|title=印度教的意象元素|volume=1: Part I|year=1916年|publisher=Law Printing House|location=马德拉斯|page = 13页}}</ref>。
