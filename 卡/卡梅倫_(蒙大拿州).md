'''卡梅倫'''（[[英語|英語]]：'''Cameron'''）是[[美國|美國]][[蒙大拿州|蒙大拿州]][[麥迪遜縣_(蒙大拿州)|麥迪遜縣]]一個[[非建制地區|非建制地區]]，位於一處能夠建設牧場的肥沃山谷之上，海拔為1774米（5822英尺）。<ref>{{cite web | url=https://www.zip-codes.com/zip-code/59720/zip-code-59720.asp | title=Zip Code 59720, Cameron, MT | publisher=Zip-Code | accessdate=2017年12月11日 | language=en}}</ref>卡梅倫鄰近麥迪遜山脈（Madison Range）以及[[獺頭-鹿棧國家森林|獺頭-鹿棧國家森林]]，並能夠觀賞到色調略紅的斯芬克斯山（Sphinx Mountain）。<ref>{{cite web | url=http://www.southwestmt.com/communities/cameron.htm | title=Cameron, Montana. | publisher=Southwest Montana Tourism | accessdate=2017年12月11日 | language=en}}</ref>
