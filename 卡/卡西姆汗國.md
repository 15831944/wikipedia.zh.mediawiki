'''卡西姆汗國'''，又名卡西莫夫汗国。是一个以俄国人为[[宗主|宗主]]的[[鞑靼|鞑靼]][[汗国|汗国]]。由1452年生存至1681年，这国家在[[奥卡河|奥卡河]]中游[[梁赞州|梁赞州]]一带。这国在[[瓦西里三世|瓦西里三世]]形成。

== 之前历史 ==
这地方的居民最早是[[芬人|芬人]]。受[[基辅罗斯|基辅罗斯]]与[[伏尔加保加利亚|伏尔加保加利亚]]影响。他们向罗斯大公交毛皮税。这里是[[苏兹达尔|苏兹达尔]]的地方。他们首领是[[尤里·多尔戈鲁基|尤里·多尔戈鲁基]]。之后这里是[[金帐汗国|金帐汗国]]mishar部（[[保加尔人|保加尔人]]）。[[喀山汗国|喀山汗国]]建国后，[[马赫穆图克|马赫穆图克]]的兄弟[[卡西姆汗|卡西姆汗]]避开他前往莫斯科，俄国人把这地方划给他缓冲莫斯科与喀山。内政外交受俄国控制。

== 人口 ==
这地方主要居民是俄国人，[[摩尔达维亚|摩尔达维亚]]人，喀山鞑靼人中的米沙爾人（芬人已溶入鞑靼）。也有一千个[[卡西莫夫鞑靼人|卡西莫夫鞑靼人]]。汗国由四大贵族主持：曼吉特、阿尔根、[[札刺亦儿|札刺亦儿]]、[[钦察|钦察]]。他们的汗在喀山、[[克里米亚汗国|克里米亚汗国]]、[[西伯利亚汗国|西伯利亚汗国]]中是亲莫斯科派。

==社會的組成==
卡西莫夫汗國最特權階層有汗、[[維齊爾|維齊爾]]、[[穆爾扎|穆爾扎]]。受尊敬的人還有[[阿塔里克|阿塔里克]]、法官、[[哥薩克|哥薩克]]、宮廷侍長。

神職人員是最有影響力的階層之一，包括：毛拉（牧師）、（“古蘭經”的學者）、（教師）、最高等神職人員是[[赛義德|赛義德]]。

==稅收支付類==
卡西莫夫稅務系統和註冊表是非常先進的。大多數人口納稅，包括商人，手工業者，小農場主和農民。

==奴隸==
奴隸（Chura）使用的是有限的，主要包括俄羅斯人和[[摩尔达维亚人|摩尔达维亚人]]。在十六世紀，卡西莫夫在有一個奴隸市場。

==經濟==
汗國生產的葡萄酒，皮革，蠟，採鹽和黃金、農業，養蜂，毛皮狩獵，捕魚對汗國經濟起到了的重大作用。

汗國有發達的手工業和手工藝。

== 历史 ==
卡西莫夫的[[汗|汗]]多次与罗斯出兵喀山。他们的汗曾三次派到喀山。而卡西莫夫本身半自治，他们的汗1574年受洗。赛德·布尔罕时代（1627－1679）大规模[[基督教化|基督教化]]。[[伯克|伯克]]成了俄国[[波雅爾|波雅爾]]。1656年曾经叛乱反同化。最后一个皇后在1681年死后，汗国也并入俄国。那些招降哈萨克的鞑靼人多来自这。他们被称为[[服役韃靼人|服役韃靼人]]。
