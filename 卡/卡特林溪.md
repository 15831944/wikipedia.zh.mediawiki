'''卡特林溪'''（{{lang-en|Catlin Brook}}），是美国[[宾夕法尼亚州|宾夕法尼亚州]][[懷俄明縣_(賓夕法尼亞州)|怀俄明县]]{{le|北支梅霍帕尼河|North Branch Mehoopany Creek}}的一条支流。 它大约有1.5{{convert|1.5|mi|km}}, 流经{{le|北支镇|North Branch Township, Wyoming County, Pennsylvania}}<ref name = "nationalmap">{{citation|author = [[United_States_Geological_Survey|United States Geological Survey]]|url = https://viewer.nationalmap.gov/advanced-viewer/|title = The National Map Viewer|accessdate = January 6, 2018}}</ref>。这条河流的分水岭面积为{{convert|1.16|sqmi|km2}}。该小溪本身非常小且陡峭，并以极其难以接近的瀑布而闻名。

==特征==
卡特林溪起源于{{le|巴特利特山|Bartlett Mountain}}北部的一个不知名的湖泊。它向北流动了一小段距离，然后流入另一个未命名的湖泊。这条小溪随后向东北偏北转了几十英里，开始向下流入巴特利特山脉。随后它向北转了十分之一英里，然后穿过魔鬼之肘路（{{lang-en|Devil's Elbow Road}}），并到达山脚。在下游几十英里的地方，河流与北支流梅霍普尼河汇合<ref name = "nationalmap"/><ref>{{citation|author = [[United_States_Geological_Survey|United States Geological Survey]]|url = http://www.usgwarchives.net/maps/pa/county/wyomin/usgs/northbra.jpg|title = northbra.jpg|year = 1987|accessdate = January 6, 2018}}</ref>。

卡特林溪加入北支流梅霍普尼河的地点，位于后者上游的{{convert|5.27|mi|km}}位置<ref name = "gaz">{{Citation|url = http://www.lycoming.edu/cwi/pdfs/paGazetterOfStreams.pdf|title = Pennsylvania Gazetteer of Streams|page = 45|date = November 2, 2001|accessdate = January 6, 2018}}</ref>。

==水文学==
卡特林溪没有被指定为受损水体<ref>{{citation|author = [[United_States_Environmental_Protection_Agency|United States Environmental Protection Agency]]|url = https://oaspub.epa.gov/tmdl/attains_waterbody.control?p_list_id=&p_au_id=PA8970&p_cycle=2006&p_state=PA&p_report_type=A|title = 2006 Waterbody Report for Catlin Brook|accessdate = January 6, 2018}}</ref>。一年中这条溪流大部分时间都有水，但在夏季通常会变成涓涓细流<ref name = "mitchell"/>。然而在2001年夏天，{{le|宾夕法尼亚州渔船委员会|Pennsylvania Fish and Boat Commission}}的生物学家在研究北部梅霍帕尼河流域时，发现这里已经干涸<ref name = "pafbc"/>。

==地理和地质学==
卡特琳溪口附近海拔{{convert|1050|ft|m}}<ref name = "gnis">{{citation|author = [[Geographic_Names_Information_System|Geographic Names Information System]]|url = https://gnis.usgs.gov/apex/f?p=138:3:0::NO::P3_FID,P3_TITLE:1171345,Catlin%20Brook|title = Feature Detail Report for: Catlin Brook|accessdate = January 6, 2018}}</ref>。溪源头附近的海拔为海拔{{convert|2260|ft|m}}<ref name = "nationalmap"/>。

卡特林溪是一条非常小且高度梯度的溪流，以{{convert|193.5|m/km|ft/mi|sp=us}}的速度下降。这使它成为北部梅霍普尼河命名支流中的最高梯度<ref name = "pafbc">{{citation|author = Robert Wnuk, Robert Moase, Matthew Gearhart|url = http://files.dep.state.pa.us/Water/Drinking%20Water%20and%20Facility%20Regulation/WaterQualityPortalFiles/Class_A_PFBC_Reports/Burgess_Brook.pdf|title = North Branch Mehoopany Creek Basin (404G) Fisheries Management Report|pages = 9–10, 16–17 23|date = November 2001|accessdate = January 6, 2018}}</ref>。

卡特林溪有一组称为卡特林瀑布。杰夫·米切尔（{{lang-en|Jeff Mitchell}}）在他的著作《徒步无尽的山脉》(徒步旅行) 中，描述其为宾夕法尼亚州瀑布的“[[圣杯|圣杯]]”。虽然距离最近的公路只有{{convert|1|mi|km}} ，但要进入它们却极其困难。米切尔称这一地区“险峻而崎岖”（{{lang-en|insanely steep and rugged}}）<ref name = "mitchell"/>。除了有许多瀑布、巨石和悬崖外, 卡特林溪还经过许多悬崖峭壁。此外在冬季，在溪流的瀑布上可以形成{{le|蓝色冰柱|Blue ice (glacial)}}，并一直持续到五月初。小溪的源头在一片[[潮湿草地|潮湿草地]]上，水位高的时候，那里可能是一个又大又浅的池塘。此湿地高原被描述为具有“惊人的孤立感”<ref name = "mitchell"/>。

==分水岭和生物学==
卡特林溪的分水岭面积为{{convert|1.16|sqmi|km2}}<ref name = "gaz"/> 。这条小溪完全位于[[美国地质调查局|美国地质调查局]]中詹宁斯维尔四分位中<ref name = "gnis"/>。它最后汇入北部梅霍帕尼河<ref name = "gaz"/>。

卡特林溪主要以北向方位经流林地。这条公路的总长度的20%为{{convert|100|m|ft|sp=us}}以内，46%为{{convert|300|m|ft|sp=us}}以内；52%为{{convert|500|m|ft|sp=us}}以内。2000年，该流域的人口密度为每平方千米3人（每平方英里8人），为北梅霍普尼河流域人口密度最小的地区<ref name = "pafbc"/>。它也被列为[[科尔德沃特渔场|科尔德沃特渔场]]（{{lang-en|Coldwater Fishery}}）范围内<ref>{{citation|author = Pennsylvania Environmental Council, Wyoming County Office of Community Planning, Mehoopany Creek Watershed Association|url = http://pecpa.org/wp-content/uploads/Mehoopany-Creek-and-Little-Mehoopany-Creek-Watersheds.pdf|title = MEHOOPANY CREEK and LITTLE MEHOOPANY CREEK WATERSHEDS RIVERS CONSERVATION PLAN|pages = 18, 57|date = March 31, 2007|accessdate = January 6, 2018}}</ref>。

==历史与娱乐==
1979年8月2日,卡特林溪被收录入[[地名信息系统|地名信息系统]]，其标识符为1171345<ref name = "gnis"/>。

到达卡特林溪峡谷的地方尚未存在已知的徒步小径。尽管如此，由于这些瀑布对宾夕法尼亚州瀑布爱好者的意义重大，卡特林溪瀑布是杰夫·米切尔的《徒步无尽的山脉》中的一个主题。这是一个{{convert|7|mi|km|adj=on}}的徒步旅行，耗时5到8个小时<ref name = "mitchell">{{citation|author = Jeff Mitchell|publisher = [[Stackpole_Books|Stackpole Books]]|url = https://books.google.com/books?id=1FqtMQ4Z2rsC&pg=PA68|title = Hiking the Endless Mountains: Exploring the Wilderness of Northeastern|page = 68|date = December 13, 2010|accessdate = January 6, 2018}}</ref>。小溪上游位于宾夕法尼亚州第57号游乐场<ref name = "pafbc"/>。

卡特林河的源头也是[[观鸟|观鸟]]的合适地点<ref name = "mitchell"/>。
