《'''卡路里'''》是中国女子组合[[火箭少女101|火箭少女101]]的歌曲，喜剧电影《[[西虹市首富|西虹市首富]]》中的插曲，于2018年7月26日发行。本歌曲由李聪、Akiyama Sayuri和彭飞共同创作。本歌曲是火箭少女101继成团曲《Rocket Girls》之后发行的首支单曲。本歌曲发行后空降中国《[[公告牌_(杂志)|公告牌]]》V榜第一名。

== 发行 ==
2018年7月19日，电影《[[西虹市首富|西虹市首富]]》的官方微博分享了一张宣传图，图中饰演“王多鱼”的[[沈腾|沈腾]]怀抱印有[[火箭少女101|火箭少女101]]标志的火箭。7月20日，电影官方微博公布了组合为电影献唱的插曲。<ref>{{cite web|title=《卡路里》来袭 火箭少女101献声《西虹市首富》|url=http://ent.163.com/18/0720/11/DN5HJFRF000380D0.html|website=网易娱乐|date=2018-07-20}}</ref>7月25日，火箭少女101在[[Youtube|Youtube]]上分享了一段音乐视频片段。<ref>{{YouTube|xfeFBpNPCCc|Rocket Girls - Calorie / TEASER}}</ref>7月26日，歌曲与音乐视频一并发行。<ref>{{cite web|title=夏天到了，你还在为小赘肉发愁吗，你还在苦... 来自电影西虹市首富 - 微博|url=https://weibo.com/6433313037/GrF36ybxQ|website=新浪微博|date=2018-07-26}}</ref>

《卡路里》拥有活泼动感的曲风和简单轻快的旋律，其歌词具有写实风格。<ref name="XIANG">{{cite web|title=西红柿首富插曲是什么 火箭少女歌曲走红 - 问剧|url=http://www.wenjutv.com/article/2018/0805/0435263|website=向剧|date=2018-08-05}}</ref>编曲彭飞认为这是一首“运动感的，俏皮的，女性化的，贴近生活的又略有些毒性的”歌曲。<ref name="ZHI">{{cite web|title=「火箭少女」新歌《卡路里》的编曲是彭飞老师吗？ - 知乎|url=https://www.zhihu.com/question/286949887/answer/452612421|website=知乎|date=2018-07-26}}</ref>歌词讲述了少女为了甩掉卡路里而所做的种种努力的故事。<ref>{{cite web|title=《西虹柿首富》插曲《卡路里》正式发布，火箭少女MV装束成为焦点|url=http://ent.tom.com/201808/4789673279.html|website=tom.com|date=2018-08-06}}</ref>据词作者李聪老师透露：“歌曲之所以命名为《卡路里》是因为‘卡路里’这三个字念起来非常有跳跃感，很有魔性。这种‘我又想吃东西，但是我还是很努力’的状态，是每个人都会有的共同心理映射。但这种需要减肥的普通人的心情、这种努力拼搏的状态，是歌曲和演唱者之间的一个契合点。”<ref name="XIANG"/>

== 打榜 ==
