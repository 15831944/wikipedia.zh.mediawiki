'''吕厚民'''（{{bd|1928年|9月9日|2015年|3月9日}}），[[黑龙江省|黑龙江]][[依兰县|依兰]]人，中国摄影师，曾在[[中南海|中南海]]担任中国[[党和国家领导人|党和国家领导人]]的专职摄影师<ref name="中红网">{{cite web |title=专职摄影师吕厚民、钱嗣杰眼中的毛泽东——纪念毛泽东同志诞辰119周年 |url=http://www.crt.com.cn/sheying/ |website=中红网}}</ref>。

== 生平 ==
1948年7月，吕厚民考入[[东北电影制片厂|东北电影制片厂]]，在照相科工作。1949年11月，调到新成立的[[北京电影制片厂|北京电影制片厂]]照相科工作。吕厚民身高1.66米，在毛泽东面前显得矮小<ref name="刘红庆"/>，毛泽东曾评价吕厚民称“你还不错嘛，短小精干。<ref name="范博韬"/>”

76岁时，罹患[[前列腺癌|前列腺癌]]，未进行手术，一直用[[药物治疗|药物控制]]。2013年10月，发现肺部出现[[肿瘤|肿瘤]]，医生判断是[[恶性肿瘤|恶性]]，为避免手术后癌细胞扩散，选择[[保守治疗|保守治疗]]。2014年11月15日，吕厚民因咳嗽住进[[北京医院|北京医院]]检查，结果是“[[肺癌|肺癌]]晚期扩散”。2015年3月9日凌晨0时36分，吕厚民因肺癌在医院逝世，享年88岁。3月15日上午，吕厚民遗体告别仪式在北京医院告别厅举行，毛泽东孙子[[毛新宇|毛新宇]]等前往送别。<ref name="刘红庆">{{cite web |author1=刘红庆 |title=这一次出发，吕厚民没带摄影包(组图) |url=http://news.163.com/15/0319/03/AL1RIIPU00014AED.html |website=网易-北青网}}</ref><ref name="范博韬">{{cite web |author1=范博韬 |title=著名摄影家吕厚民 曾为毛泽东专职摄影记者 定格无数历史瞬间 追悼会上午举行—— 毛新宇送别摄影界泰斗(组图) |url=http://news.163.com/15/0315/14/AKONLD1000014Q4P.html |website=网易-法制晚报}}</ref>

== 荣誉 ==
作品《欢送志愿军归国》先后获德国摄影银牌奖、苏联国际摄影艺术展览金牌奖、古巴国际摄影艺术展览一等奖。作品《放鸭图》获比利时皇家摄影学会一等奖<ref name="中红网" />。

== 亲人 ==
吕厚民妻子刘钟云，[[河北|河北]][[保定|保定]]人，曾在中南海做机要工作，两人经中南海摄影组[[侯波|侯波]]介绍后结婚，育有两子一女。长子吕坚1954年生，《北京日报》摄影记者；女儿吕方1960年生，《中国民航杂志》摄影记者；次子吕东1961年生，曾在照相馆做暗房<ref name="刘红庆" />。
