'''呂基亞語'''（{{lang|xlc|𐊗𐊕𐊐𐊎𐊆𐊍𐊆}} {{transl|xlc|''Trm̃mili''}}）<ref name="bry30">Bryce (1986) page 30.</ref> 是[[铁器时代|铁器时代]]在[[安那托利亞|安那托利亞]]的[[吕基亚|吕基亚]]地區（今[[土耳其|土耳其]]西南部）的呂基亞人所使用的語言。公元前4世紀時這種語言就已經受到強烈的希臘化影響，呂基亞人已經開始給自己取希臘名字，安那托利亞希臘化後這種語言被[[古希臘語|古希臘語]]取代，現在早已經成為[[絕跡語言|絕跡語言]]。“呂基亞”（{{lang-grc|Λυκία}} 轉寫：''Lykía''，{{lang-tr|Likya}}）這個地名是古希臘語記載的，得名於[[雅典國王|雅典國王]]{{link-en|潘狄翁二世|Pandion II}}之子{{link-en|吕科斯|Lycus (mythology)}}。而呂基亞語對本地區的自稱“Trm̃mis-”，其中一種可能是源自意為“崎嶇多山的”的構擬詞“*tarmašši-”。

== 地區 ==
呂基亞地區覆蓋的範圍包括現今土耳其西南部的[[安塔利亚|安塔利亚]]和[[费特希耶|费特希耶]]兩城，尤其是費特希耶灣和安塔利亞灣之間多山的海岬。

== 分類 ==
呂基亞語是一種印歐語，[[安那托利亞語族|安那托利亞語族]]下的[[卢维语|卢维语]]支系里的一種。盧維語支還包括使用[[楔形文字|楔形文字]]和[[圣书体|圣书体]]的盧維語、卡里亞語、西代語、皮西迪亞語。<ref>Adiego (2007) page 763.</ref> 前字母形式的盧維語可以追溯到晚期[[青铜时代|青铜时代]]，[[赫梯|赫梯]]帝國覆滅之前，大約在安那托利亞南部出現新的敘利亞-赫梯國家時期消失了，因此鐵器時代的這個語支的成員都是盧維語的本地化衍生物。

呂基亞語發現於一些較長的碑文中，學者從中分辨出了至少兩種方言：其中一種被認為是標準的呂基亞語，也稱為“呂基亞語A”（Lycian A）；另一種方言在{{link-en|克桑托斯碑|Xanthian Obelisk}}的D面（朝向西北偏西的面）得到了證實，記為“呂基亞語B”（Lycian B）或稱“彌呂亞語”（Milyan），但現在後者也常被認為是另一種語言而不只是前者的變體。<ref>{{Cite web |url = http://multitree.org/codes/imy |title = MultiTree - Milyan |language = en |accessdate = 2017-11-15}}</ref>

對呂基亞文字的解讀并不完全，很多都是基於假設；其音系也不甚明了。有[[元音|元音]] a、e、i、u 和[[鼻化元音|鼻化元音]] ã ẽ，[[辅音|辅音]]中“h”發自[[原始印歐語|原始印歐語]]的“*s”（彌呂亞語仍保留為“s”），而原始印歐語“i”前的“*kʷ”變成了“t”，這一點和古希臘語一致。<ref>松本(1992) p.892</ref>

== 資料 ==
對呂基亞語的瞭解來源於以下這些資料：<ref>Adiego (2007) page 764.</ref><ref>Bryce (1986) page 42.</ref>
* 希臘語記錄的地名和人名；
* 從公元前5世紀末到公元前4世紀末的約172處呂基亞語碑文，<ref>Bryce (1986) page 50.</ref>這些碑文可以分類成：
** 150處石墓墓文
** 20處功德碑文
* 製於[[克桑托斯|克桑托斯]]的約100件[[鑄幣|鑄幣]]銘文，從庫普里利（Kuprili）統治時期（公元前485-440年）到伯里克勒（Pericle）統治時期（公元前380-360年）；<ref>Bryce (1986) pages 51–52.</ref>
* {{link-en|勒托翁三語碑|Letoon trilingual}}，上面寫有呂基亞語A、[[古希臘語|古希臘語]]和[[亞拉姆語|亞拉姆語]]，1973年被法國考古隊發現；<ref>松本(1992) p.891</ref>
* {{link-en|克桑托斯雙語碑|Xanthian Obelisk}}。克桑托斯一處墳墓的上部刻有文字，這被稱作是“克桑托斯石碑”或“克桑托斯方尖碑”。此碑上南面、東面、和部分北面刻滿了呂基亞語A；北面還刻有12行古希臘詩句；此外還有呂基亞語B主要刻在西面。總共有255行文字。
上述這些碑刻資料時間跨越約170年，從公元前500年至公元前330年。
