'''吕基亚'''（[[古希腊语|古希腊语]]：Λυκία）[[安纳托利亚|安纳托利亚]]历史上的一个地区，位于今[[土耳其|土耳其]][[安塔利亚省|安塔利亚省]]境内。在[[罗马帝国|罗马帝国]]时期，这里曾是帝国在[[亚洲|亚洲]]的一个[[罗马行省|行省]]。
== 地理 ==
吕基亚位于土耳其西南部海岸，是一个多山而且植被茂盛的地区。它的西面和西北面与[[卡里亚|卡里亚]]交界，东面与[[潘菲利亚|潘菲利亚]]接壤，东北面是[[皮西迪亚|皮西迪亚]]。

在古代，吕基亚地区的主要城市是[[克桑托斯|克桑托斯]]，[[帕塔拉|帕塔拉]]，[[米拉_(土耳其)|米拉]]和[[法塞利斯|法塞利斯]]。
== 居民 ==
吕基亚地区在史前时代就有人类居住。当地的土著是[[吕基亚人|吕基亚人]]，他们说一种属于[[印欧语系|印欧语系]]的[[语言|语言]]（[[吕基亚语|吕基亚语]]）。与吕基亚语关系最近的语言是[[卢维语|卢维语]]，它或许是吕基亚语的直接祖先。后来[[希腊|希腊]]移民逐渐取代了当地居民成为吕基亚的主人。希腊人在吕基亚建起了一系列殖民城邦。直到[[土耳其|土耳其]]统治时代，该地区仍有希腊人居住；最后一批希腊居民是因1919年～1922年的[[希土战争|希土战争]]而被迫离开。

== 历史 ==
[[古埃及|古埃及]]文献告诉我们吕基亚人是[[赫梯|赫梯人]]的盟友。在赫梯帝国崩溃后，吕基亚作为一个独立王国发展起来。按照[[希罗多德|希罗多德]]的说法，“吕基亚”这个名字来源于[[雅典|雅典]]国王[[潘狄翁二世|潘狄翁二世]]的儿子[[吕科斯|吕科斯]]的名字。在古典时代，吕基亚地区从未被统一于一个政权之下，而是由许多独立[[城邦|城邦]]组成的联合体。

[[荷马|荷马]]在《[[伊利亚特|伊利亚特]]》中多次提到吕基亚，并说它是[[特洛伊|特洛伊人]]的盟友。在《伊利亚特》里，参战的吕基亚人由[[萨尔珀冬|萨尔珀冬]]（[[宙斯|宙斯]]与[[拉俄达弥亚|拉俄达弥亚]]之子）和[[格劳科斯|格劳科斯]]（[[希波格科斯|希波格科斯]]之子）两位英勇善战的英雄领导。<ref>荷马，《伊利亚特》，第6卷</ref>非常值得注意的是，在其它一些神话中提到过另一位[[萨尔珀冬|萨尔珀冬]]（宙斯与[[欧罗巴|欧罗巴]]之子）的故事，他因与兄长、[[克里特|克里特]]国王[[米诺斯|米诺斯]]争权失败而被迫流亡吕基亚，并在那里建立了一个王朝。<ref>希罗多德，I</ref>古代人经常把这两个[[萨尔珀冬|萨尔珀冬]]相混同。这些神话可能暗示了[[克里特文明|克里特文明]]与吕基亚（或[[安纳托利亚|安纳托利亚]]）之间的某种未知联系。

前546年，[[居鲁士大帝|居鲁士大帝]]的将领[[哈尔帕古斯|哈尔帕古斯]]征服了吕基亚，将其并入[[波斯帝国|波斯帝国]]。哈尔帕古斯的后裔世袭了小亚细亚总督的职位，他们统治吕基亚直到前468年，那年雅典人将波斯帝国的势力赶出了这一地区。波斯人在前387年又夺回了吕基亚，并占据该地直到整个波斯帝国被[[亚历山大大帝|亚历山大大帝]]征服。亚历山大死后，其帝国迅速瓦解。[[塞琉古一世|塞琉古一世]]在[[继业者战争|继业者战争]]中几乎夺得了帝国在[[亚洲|亚洲]]的全部领地，包括吕基亚；他在亚洲建立起最大的一个[[希腊化国家|希腊化国家]][[塞琉古帝国|塞琉古帝国]]。前189年，塞琉古国王[[安条克三世|安条克三世]]被新兴强国[[罗马共和国|罗马]]打败，包括吕基亚在内的大片土地被割让给罗马。随着罗马的发展，整个[[地中海|地中海]]地区都被并入这个帝国。公元43年，[[罗马皇帝|罗马皇帝]][[克劳狄一世|克劳狄一世]]将吕基亚与[[潘菲利亚|潘菲利亚]]合并为罗马的一个[[羅馬行省|行省]]，后又单独建省。

在[[西罗马帝国|西罗马帝国]]灭亡后，吕基亚属于东部帝国[[拜占廷帝国|拜占廷]]，因而保留了古典文化。[[奥斯曼帝国|奥斯曼帝国]]兴起后征服了拜占廷在亚洲的全部领土并最终灭亡了拜占廷，吕基亚亦被并入奥斯曼帝国；从此吕基亚即成为土耳其的一部分。

吕基亚在古代是一个重要的宗教活动中心，以崇拜[[勒托|勒托]]、[[阿波罗|阿波罗]]和[[阿耳忒弥斯|阿耳忒弥斯]]为主。

== 吕基亚联盟 ==

吕基亚联盟是吕基亚各城邦在前168年组成的一个政治同盟，它包括已知的23个成员。吕基亚在安条克三世失败后一度被[[罗得岛|罗得岛]]所控制，但罗马人在[[第三次马其顿战争|第三次马其顿战争]]结束后决定让吕基亚各城邦在罗马庇护下获得独立。这些城邦于是建立了一个联邦式的政府，并组建了由各邦成员组成的元老院来选举联盟执政官（执政官的任期为一年）。这个联盟的主要成员是克桑托斯，帕塔拉，[[皮纳拉|皮纳拉]]，[[奥林波斯|奥林波斯]]，米拉和[[特洛斯|特洛斯]]。后来法塞利斯也加入了联盟。有趣的是，在吕基亚被正式并入罗马之后，这个联盟仍然存在并发挥着功能，直到4世纪。
