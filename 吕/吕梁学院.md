'''吕梁学院'''是[[山西省|山西省]]的一所全日制本科高校，位于[[吕梁市|吕梁市]][[离石区|离石区]]学院路1号。吕梁学院的前身是1978年在[[汾阳市|汾阳]]建立的'''[[山西师范学院|山西师范学院]]吕梁师专班'''，1979年迁到离石，1984年成立'''吕梁师范专科学校'''，1990年改为'''吕梁高等专科学校'''，2010年升格为本科吕梁学院<ref>{{cite web |url=http://www.llhc.edu.cn/xxgk/lsyg.htm |title=吕梁学院-历史沿革 |last= |first= |date= |website=www.llhc.edu.cn |publisher=吕梁学院 |access-date=2017-12-18 |quote=}}</ref><ref>{{cite web |url=http://www.sxrb.com/sxrb/aban/a2/2265563.shtml |title=吕梁学院举行成立挂牌仪式 |last= |first= |date=2010-06-27 |website=www.sxrb.com |publisher=山西新闻网 山西日报 |access-date=2017-12-18 |quote=}}</ref>。2007年开始扩建校园，工程结束后增设了[[煤化工|煤化工]]、[[电子技术|电子]]、[[冶金|冶金]]、[[煤矿|采煤]]、[[管理|管理]]、[[地质矿产|地矿]]、[[环境科学|环境科学]]等吕梁社会相关人才急需的一批专业<ref>{{cite web |url=http://news.sina.com.cn/c/2009-04-26/061515528513s.shtml |title=吕梁学院新校址工程建设加速前进 |author=王佳丽 |date=2009-04-26 |website=news.sina.com.cn |publisher=山西新闻网 山西经济日报 |access-date=2017-12-19 |quote=}}</ref>。现有校园面积1145亩，建筑面积48万平方米。共有17个教学系和42个本科专业，教职工984人，现任院长是熊继军<ref>{{cite web |url=http://www.llhc.edu.cn/xxgk/xxjj.htm |title=吕梁学院-学校简介 |last= |first= |date= |website=www.llhc.edu.cn |publisher=吕梁学院 |access-date=2017-12-18 |quote=}}</ref>。2012年，校园内曾发生物理系主任在校内被暴力杀害的事件，在吕梁引起轰动<ref>{{cite web |url=http://sxllzy.chinacourt.org/article/detail/2014/05/id/1289975.shtml |title=吕梁学院校园杀人案一审宣判 |author=康照明 |date=2014-05-08 |website=sxllzy.chinacourt.org |publisher=吕梁市中级人民法院 |access-date=2017-12-19 |quote=}}</ref>。2016年，吕梁学院开设的地下室餐厅因消防安全隐患突出，被有关部门查封，承办方和监管方不顾学生人身和饮食安全的作为引起媒体热议<ref>{{cite web |url=http://www.wschina.net/lm51/2016/410799.htm |title=吕梁学院设地下室餐厅违规经营 消防安全隐患突出 |last= |first= |date=2016-08-25 |website=www.wschina.net |publisher=中国网山西 |access-date=2017-12-19 |quote=}}</ref>。
