'''盂唇'''，亦作'''盂韌帶'''，位於位於手臂骨與肩胛骨之間<ref>{{cite journal |url=http://www.healthsmart.com.hk/article/issues/joints/I120171.aspx |title=肩膀「盂唇」若撕裂，手臂難發力 |date=2012-02-28 |accessdate=2012-04-04 |journal=[[樂本‧健|樂本‧健]] |author= |language=zh-hant }}</ref>的[[肩關節|肩關節]]，是一個附著在[[肩胛骨盂腔|肩胛骨盂腔]]周圍的[[纖維軟骨|纖維軟骨]]組織<ref name=antoine669>{{cite web |url=http://www.wretch.cc/blog/antoine669/24092759 |title=肩關節盂唇撕裂之手術過程(照片來源為中國骨科運動醫學網) - |publisher=無名小站：王老師教室 |accessdate=2012-04-04 |language=zh-hant}}</ref>，約3-4毫米寬。

== 解剖 ==
盂唇的橫切面呈三角形，其底部固定於關節腔，而其自由面薄且尖。

盂唇與[[肱二頭肌|肱二頭肌]]頂端的長肌腱連在一起，其纖維組織使兩個纖維束得以融合在一起。 

== 功用 ==
肩關節被歸類為[[球窩關節|球窩關節]]，但實際上肩胛窩（又名[[肩臼窩|肩臼窩]]或[[肩胛骨關節窩|肩胛骨關節窩]]）很淺和細小，那是因為它其實是由一條大約在我們一歲時出現的[[肩胛骨喙突|肩胛骨喙突]]與肩胛骨在我們約17歲時癒合形成<ref>{{cite web |url=http://www.books.com.tw/exep/prod/books/show_series.php?item=0010409840&page=6 |title=博客來書評：《退化原來是進化─智齒、盲腸、副乳……留在你身上的演化痕跡》 |language=zh-hant |accessdate=2012-04-04 }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。因此，肩胛窩只能把[[肱骨|肱骨]]的關節球的三分一覆蓋，其餘部份由盂唇覆蓋着，以穩定肩膀<ref name=antoine669 />。

盂唇令[[關節腔|關節腔]]加深之餘，亦保護了肩胛骨的邊緣。
