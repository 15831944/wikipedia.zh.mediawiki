'''普林塞萨港地下河国家公园'''、'''公主港地下河國家公園'''（{{lang-en|'''Puerto Princesa Subterranean River National Park'''}}）位于[[菲律宾|菲律宾]][[巴拉望省|巴拉望省]][[普林塞萨港|普林塞萨港]]市中心以北50公里处，由普林塞萨港市政府管理，1999年由[[聯合國教科文組織|聯合國教科文組織]]獲選為[[世界自然遺產|世界自然遺產]]<ref name = "臺灣世界遺產潛力點-世界遺產_2018"> {{ cite web | title = 臺灣世界遺產潛力點-世界遺產 | url = https://twh.boch.gov.tw/world/intro.aspx?id=646 | publisher = 文化部文化資產局 | language = zh-tw | accessdate = 2018-08-11 }} </ref><ref name = "普林塞萨港地下河国家公园_2018"> {{ cite web | title = 普林塞萨港地下河国家公园 | url = http://whc.unesco.org/zh/list/652#top | language = zh-cn | accessdate = 2018-08-11 }} </ref>，2011年11月11日，普林塞萨港地下河入选[[世界新七大自然奇观|世界新七大自然奇观]]。<ref name = "公主港地下河國家公園_2018"> {{ cite web | title = 公主港地下河國家公園 | url = https://www.expedia.com.tw/Puerto-Princesa-Subterranean-River-National-Park-Puerto-Princesa.d6115184.Place-To-Visit | publisher = Expedia.com.tw | language = zh-tw | accessdate = 2018-08-11 }} </ref>

== 登錄基準 ==
公主港地底河國家公園因為符合以下標準，而被聯合國科教文組織登錄為世界自然遺產：
*（vii）包含出色的自然美景與美學重要性的自然現象或地區。
*（x）擁有最重要及顯著的多元性生物自然生態棲息地，包含從保育或科學的角度來看，符合普世價值的瀕臨絕種動物種。
