'''普利沙集团'''（{{lang-es|'''PRISA'''}}，全称：{{lang|es|Promotora de Informaciones, S.A}}）是西班牙的一家[[传媒集团|传媒集团]]<ref>{{cite web |author1=张蕾 |title=走进民族报报业集团 |url=http://media.people.com.cn/GB/40606/6235249.html |publisher=[[人民网|人民网]] |archiveurl=https://web.archive.org/web/20070914232259/http://media.people.com.cn/GB/40606/6235249.html |archivedate=2007-09-14 |language=zh-hans |date=2007-09-08 |access-date=2019-01-08 |dead-url=no }}</ref>。

==历史==
普利沙集团由西班牙[[實業家|實業家]]{{tsl|en|Jesús de Polanco|赫苏斯·德波朗科}}和何塞·奥特嘉·斯波托尔诺两人于1972年创立<ref name=tcor>{{cite news|author=Julia Pastor|title=Prisa Group: Dismantling of Spain’s Top Media Giant Means End of an Era|url=http://www.thecorner.eu/companies/prisa-group-leading-spanish-media-conglomerates-dismantling/34255/|accessdate=2014-11-26|work=The Corner|date=2013-12-27|location=Madrid}}</ref>。斯波托尔诺是西班牙著名哲学家[[何塞·奧特嘉·伊·加塞特|加塞特]]的儿子<ref name=imc/>，波朗科则是军人家庭出身，后来一直担任集团的大[[股东|股东]]和[[总裁|总裁]]，直到2007年7月21日去世<ref name=anmee>{{cite book|author1=Andrea Czepek|author2=Melanie Hellwig|author3=Eva Nowak|title=Press Freedom and Pluralism in Europe: Concepts and Conditions|url=https://books.google.com/books?id=nZHATHfMo7gC&pg=PA275|accessdate=2014-12-12|year=2009|publisher=Intellect Books|isbn=978-1-84150-243-4|page=278}}</ref>。在这个过渡期，由他的儿子{{tsl|en|Ignacio Polanco|伊格纳西奥·波朗科}}担任总裁<ref name=anmee/>。2012年，{{tsl|es|Juan Luis Cebrián|胡安·路易斯·塞夫里安}}被任命为新总裁，伊格纳西奥·波朗科转任名誉总裁<ref>{{cite web |title=Juan Luis Cebrián, presidente de PRISA |url=https://elpais.com/economia/2012/07/20/actualidad/1342803905_976072.html |publisher=EL PAÍS |accessdate=2019-01-08|language=es |date=2012-07-20}}</ref>。

{{As of|2010}}，它的最大股东持有者为投资家[[尼古拉斯·伯格鲁恩|尼古拉斯·伯格鲁恩]]<ref>{{cite web|url=https://www.nytimes.com/2010/11/22/technology/22iht-prisa.html?ref=libertyacquisitionholdingscorporation|author=Raphael Minder|title=Prisa Looks for Return to Financial Health|publisher=[[纽约时报|纽约时报]]|date=2010-11-21}}</ref>。之后该集团的股权构成发生了很大变化，截至2017年底，普利沙集团的主要[[股东|股东]]为英国Amber Capital（19.29％），波朗科家族的Rucandio（17.53％）和[[西班牙電信|西班牙電信]]（13.06％）<ref>{{cite web |title=ANNUAL REPORT ON CORPORATE GOVERNANCE |url=http://www.prisa.com/uploads/2017/02/iagc-2016-24-02-17-completo-ing.pdf |publisher=PROMOTORA DE INFORMACIONES, S.A. |language=en |date=2017|accessdate=2019-01-08}}</ref>。

==旗下出版物==
{{As of|2012}}，普利沙集团持有《[[赫芬顿邮报|赫芬顿邮报]]》西语版50%的股权<ref name=imc/><ref>{{cite web |title=Tráfico El Huffington Post |url=https://www.prisa.com/informe-anual-2015/wp-includes/flip/Informe_de_Sostenibilidad_ES_PRISA/#/24/ |website=prisa.com |accessdate=2019-01-08 |language=es}}</ref>。{{As of|2014}}，普利沙集团持有[[葡萄牙|葡萄牙]]媒体公司{{tsl|en|Media Capital||Media Capital}}95%的股权，也收购了葡萄牙[[獨立電視台_(葡萄牙)|獨立電視台]]以及一些电台频道<ref>{{cite web|title=Portuguese Media|url=https://www.bpiequity.bpi.pt/others/PDF.aspx?id=49681|publisher=BPI Equity|accessdate= 2015-02-02|date=2014-03-05|language=en}}</ref>。
