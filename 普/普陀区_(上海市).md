'''普陀区'''是[[中華人民共和國|中華人民共和國]][[上海市|上海市]]的一个[[市辖区|市辖区]]，地处上海市中心区西北部，面积55.47平方公里。北接[[宝山区_(上海市)|宝山区]]，东临[[闸北区|闸北区]]，南面与[[静安区|静安区]]、[[长宁区|长宁区]]接壤，西面毗邻[[嘉定区|嘉定区]]。

== 概述 ==
普陀区2010年第六次全国人口普查结果户籍人口879,213人，常住人口1,288,881人。以境内普陀路而得名，境内有[[苏州河|苏州河]]横穿。[[上海市普陀区人民政府|上海市普陀区人民政府]]驻大渡河路。普陀区还是上海市区各少数民族特别是[[回族|回族]]最为集中的地区，据2007年末数据，共有少数民族31个，8942人。区内文化、教育、卫生、体育、社区等公益事业设施众多，著名的景点有[[真如寺_(上海)|真如寺]]，[[玉佛寺|玉佛寺]]以及[[长风公园|长风公园]]等。区内有立体的交通网络，由[[上海地铁|地铁]]、地面交通、高架道路构成。

== 历史 ==
[[唐朝|唐朝]]之前普陀区内部分地区就已经形成陆地。民国16年（1927年），建立了上海特别市并辖有17个区。今普陀区区域分属闸北、彭浦、真如、蒲松、法华等地区。抗日战争期间，汪精卫政府将公共租界重新分划，并设立了31个警区，苏州河南区域分划为第十三警区，由普陀路警察局管理。抗日战争胜利后，民国34年（1945年）12月25日，国民政府上海市政府在原第十三警区的基础上，建立了第十三行政区，并设置[[区公所|区公所]]。<ref>{{Cite book |title = 普陀指南|last = |first = |publisher = 上海社会科学院出版社 |date = 2015年1月 |ISBN = 978-7-5520-0725-1|location = 上海|pages = 3}}</ref>民国36年（1947年），以境内的普陀路得名，改称普陀区。当时区辖面积仅2.65平方公里，全部位于[[吴淞江|吴淞江]]南岸，总人口13.05万。1949年5月，[[中国共产党|中国共产党]]占领上海全境，同月底设立了“普陀区接管委员会”。1950年6月8日成立了普陀区人民政府。此后行政区域不断扩大，先后共6次从[[吴淞江|吴淞江]]以北的临近区县划入土地共计52.46平方公里，至今总面积已达55.47平方公里。同时，常住人口也增长至1,288,881人（2010年第六次全国人口普查结果）。

== 经济 ==
近代，普陀区是上海市中心区中经济最不发达的行政区。然而，近几年来，随着上海市政府将普陀区[[真如镇街道|真如镇街道]]规划为上海市区第四大副中心，将其规划成为上海市区西北部物流中心以后，经济建设速度正不断加快。

== 交通 ==
19世纪末，普陀区区境内为河网密布的农村水乡，大小河道多可通航，陆上依靠乡间小道通连各地。横贯全区的吴淞江是主要航道，但江上无桥梁，过江只得唯赖船渡。公共租界扩展至境内吴淞江南岸之后，民族资本家和外商利用吴淞江水运方便，纷纷在南北两岸兴建工厂与仓库，同时还增设船埠码头，辟筑道路，逐步改善了交通条件。20世纪初，沪宁、沪杭铁路修剪成，设上海西站及其他车站。<ref name="交通">[http://www.shtong.gov.cn/newsite/node2/node4/node2249/putuo/node41331/index.html 普陀区志]，第二十卷交通邮电。</ref>20世纪初，境内吴淞江以南开始辟筑城市道路，逐步形成道路网。解放前夕约有道路40余条，至1990年全区共有道路121条。<ref name="陆上交通">[http://www.shtong.gov.cn/newsite/node2/node4/node2249/putuo/node41331/node41346/node62594/userobject1ai26821.html 普陀区志]，第二十卷交通邮电，第三章陆上交通。</ref>

公共交通方面，约有公共汽车线路34条、无轨电车线路4条、长途客运巴士线路85条。

=== 公共汽车沿革 ===
* 民国15年（1926年）2月30日，[[英商上海电车公司|英商上海电车公司]]辟16路、19路无轨电车。
* 民国27年（1938年）12月15日增辟的24路无轨电车。
* 民国37年（1948年），[[上海市公共交通公司|上海市公共交通公司]]开行的公交线路有：10路甲（自大夏大学至中山公园）、14路（自外滩至造币厂桥），一直经营到上海解放为止。
* 1960年10月8日，穿行境内的13路无轨电车开始通行。<ref name="陆上交通" />

2012年3月，成立了上海市邮政公司普陀区邮政局。2014年3月根据《中华人民共和国邮政法》、《国务院关于印发邮政体制改革方案的通知》等有关规定，更名为上海市邮政公司普陀区分公司。
