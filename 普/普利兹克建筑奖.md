'''普立茲克建築獎'''（{{lang-en|Pritzker Architecture Prize}}）是一年一度由凱悅基金會頒發，以表彰「在世建築師，其建築作品展現了其天賦、遠見與奉獻等特質的交融，並透過建築藝術，立下對人道與建築環境延續且意義重大的貢獻」<ref>{{Cite web | url = http://www.pritzkerprize.com/about/purpose.html | title = The Purpose of the Pritzker Architecture Prize | work = Pritzker Prize official site | publisher = The Hyatt Foundation | accessdate = June 24, 2009 | deadurl = yes | archiveurl = https://web.archive.org/web/20090628142227/http://www.pritzkerprize.com/about/purpose.html | archivedate = 2009年6月28日 | df =  }}</ref>，於1979年由{{link-en|傑·普立茲克|Jay Pritzker}}和妻子辛蒂設立，由{{link-en|普立茲克家族|Pritzker family}}資助；普利兹克建筑奖被公認是全球最主要的[[建築|建築]]獎項之一，有「建築界的[[諾貝爾獎|諾貝爾獎]]」的美譽<ref name=eightyeight>{{Cite web | url = http://www.nytimes.com/1988/05/29/arts/architecture-view-what-pritzker-winners-tell-us-about-the-prize.html | title = Architecture View; What Pritzker Winners Tell Us About the Prize | work=The New York Times|accessdate=June 26, 2009|date=May 28, 1988|first=Paul|last=Goldberger}}</ref><ref name=eighty>{{Cite web | url = http://www.sfgate.com/cgi-bin/article.cgi?f=/c/a/2006/10/14/HOG51LMROS1.DTL |title = The Mexican garden revisited|accessdate=June 26, 2009|date=October 14, 2006|first=Katherine |last=Endicott|work=San Francisco Chronicle}}</ref>。

普立茲克獎授獎「無關國籍、種族、宗教或思想」<ref name=nom>{{Cite web |url =http://www.pritzkerprize.com/about/nomination.html |title =Nomination Process |work =Pritzker Prize official site |publisher =The Hyatt Foundation |accessdate =July 3, 2009 |deadurl =yes |archiveurl =https://web.archive.org/web/20090628135630/http://www.pritzkerprize.com/about/nomination.html |archivedate =2009年6月28日 }}</ref>；受獎者可獲得獎金十萬[[美元|美元]]、獎狀，以及自1987年起增頒的銅質獎章一只<ref name=proc>{{Cite web |url =http://www.pritzkerprize.com/about/history.html |title =History |work =Pritzker Prize official site |publisher =The Hyatt Foundation |accessdate =June 27, 2009 |deadurl =yes |archiveurl =https://web.archive.org/web/20100110053937/http://www.pritzkerprize.com/about/history.html |archivedate =2010年1月10日 }}</ref>。獎章背面的拉丁文銘刻—''firmitas, utilitas, venustas''（中文：坚固、适用、美观）—源自[[古羅馬|古羅馬]]建築師[[維特魯威|維特魯威]]<ref>{{Cite web| url= http://www.pritzkerprize.com/about/ceremony.html| title= Ceremony and Medal| publisher= The Hyatt Foundation| work= Pritzker Prize official site| accessdate= June 29, 2009| deadurl= yes| archiveurl= https://web.archive.org/web/20090628142829/http://www.pritzkerprize.com/about/ceremony.html| archivedate= 2009年6月28日}}</ref>。1987年前，受獎者可獲得限量[[亨利·摩爾|亨利·摩爾]]雕像與獎金<ref name=proc/>。

自2009年起，執行董事{{le|瑪莎·索恩|Martha Thorne}}徵求往年得獎者、學者、評論家與其他「在建築領域具專業知識與興趣」人士的提名<ref name=nom/><ref>{{Cite web | url = http://www.pritzkerprize.com/about/jury.html|accessdate=July 3, 2009| work= Pritzker Prize official site|publisher=The Hyatt Foundation |title = 2009 Jury Members}}</ref>。領有執照的建築師亦可在每年11月1日之前提出個人申請。評審委員會由五至九位「專家……公認在其建築、經濟、教育、出版與文化等個人領域具專業者」組成，負責商討、裁定並在隔年春季向公眾宣布獲獎人選<ref name=nom/>。

== 爭議 ==
2013年，{{link-en|哈佛大學設計研究生院|Harvard Graduate School of Design}}的一個學生組織「在設計之中的女性」的代表向普立茲克獎官方遞交請願書，要求表彰1991年得主[[羅伯特·文丘里|羅伯特·文丘里]]的妻子兼拍擋{{link-en|丹尼絲·斯科特·布朗|Denise Scott Brown}}，此外他們亦進一步推動建築界的[[性別歧視|性別歧視]]辯論。《[[紐約時報|紐約時報]]》的報導稱該請願書「再次點燃了醞釀已久的女性們是否仍然一貫否認她們的地位是否可以獲得建築界最負盛名的獎項，而該獎項又是否不該給2004年得主[[扎哈·哈迪德|扎哈·哈迪德]]的問題」<ref>{{Cite web | url = http://www.nytimes.com/2013/04/18/arts/design/bid-for-pritzker-prize-to-acknowledge-denise-scott-brown.html?ref=design|work=The New York Times |title = Partner Without the Prize| accessdate= April 18, 2013}}</ref>。雖然該請願書獲得了一些普立茲獎得主的支持，但普立茲獎的頒獎委員會卻稱無法重審查過去的評判們的決定，不過他們表示肯定斯科特·布朗和2012年得主[[王澍|王澍]]的妻子{{link-en|陸文宇|Lu Wenyu}}的工作，認為她們不比她們的丈夫遜色<ref>{{Cite web | url = http://www.architectmagazine.com/design/pritzker-architecture-prize-committee-refuses-to-honor-denise-scott-brown.aspx |work=architectmagazine.com|title = Pritzker Architecture Prize Committee Denies Honors for Denise Scott Brown| accessdate= June 18, 2013}}</ref>。在官方回應後，斯科特·布朗透過[[CNN|CNN]]說：「作為一名女性，我覺得我被排除在建築界的精英之外」，除此之外，她也表示普立茲克獎只基於頒發給以合作為代價建出偉大建築的「單身男性天才」的謬論而頒發<ref>{{cite web |url=http://edition.cnn.com/2013/05/01/business/denise-scott-brown-pritzker-prize |title=Denise Scott Brown: Architecture favors 'lone male genius' over women |publisher=CNN |author= Catriona Davies |date=May 29, 2013}}</ref>。

== 歷屆得主 ==
首任得主[[菲力普·強生|菲力普·強生]]受表彰其「五十年來體現於無數博物館、劇院、圖書館、房屋、庭園及企業建築的創造力與熱情」<ref>{{Cite web | url  =http://www.pritzkerprize.com/1979/jury| accessdate=November 25, 2014|work= Pritzker Prize official site| publisher=The Hyatt Foundation |title = Philip Johnson - 1979 Laureate - Jury Citation}}</ref>。2004年獲獎者[[薩哈·哈帝|薩哈·哈帝]]是第一位女性得主<ref>{{Cite web | url = http://news.bbc.co.uk/1/hi/england/london/4167077.stm| title = Hadid designs landmark building | date = January 15, 2005|accessdate=June 29, 2009|publisher = BBC News}}</ref>。[[西泽立卫|西泽立卫]]是最年輕的得主，2010年獲獎時年僅四十四歲。他當時與[[妹島和世|妹島和世]]一起得獎，獲獎理由為「結構精緻而強大，精確而流暢，巧妙而不會精巧過頭」<ref>{{Cite web| url = http://www.pritzkerprize.com/2010/announcement| work = | publisher = The Hyatt Foundation| title = Announcement: Kazuyo Sejima and Ryue Nishizawa| accessdate = February 26, 2015| deadurl = yes| archiveurl = https://web.archive.org/web/20120418152536/http://www.pritzkerprize.com/2010/announcement| archivedate = 2012年4月18日| df = }}</ref>。第33任得主[[彼得·卒姆托|彼得·卒姆托]]則被嘉許其「將建築精簡至最簡潔、摒棄最奢華的要素」<ref>{{Cite web | url = http://www.pritzkerprize.com/2009/jury|  work= Pritzker Prize official site|publisher=The Hyatt Foundation |title = Peter Zumthor - 2009 Laureate - Jury Citation| accessdate= November 25, 2014}}</ref><ref>{{Cite web | url = http://www.nytimes.com/2009/04/13/arts/design/13pritzker.html|work = The New York Times|accessdate= June 29, 2009| date=April 12, 2009|title = Pritzker Prize Goes to Peter Zumthor|first=Robin|last=Pogrebin}}</ref>。2015年得主德國建築師[[弗萊·奧托|弗萊·奧托]]是目前唯一一位於公佈前逝世的得主，他在頒獎前一天逝世，終年89歲<ref name = Otto/>。而最近的得主（2019年）則為日本建築師[[磯崎新|磯崎新]]<ref name="Isozaki"/>。
