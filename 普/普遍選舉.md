'''普遍選舉'''（{{lang-en|Universal suffrage}}，{{lang|en|Universal adult suffrage}}，{{lang|en|General suffrage}}，或{{lang|en|Common suffrage}}），又稱'''普通選舉權'''、'''普選'''、'''普及選舉'''、'''普通選舉'''、'''普遍选举权'''，在[[民主|民主]]政體中，將[[选举权|选举权]]延伸至任何一個[[成年|成年]][[公民|公民]]，稱為普通選舉，但是這個權利通常不包括未成年人，以及未擁有[[國籍|國籍]]的非[[公民|公民]]與特定人士。擁有普選權，也代表，在這個國家中，无论[[性别|性别]]、[[年龄|年龄]]、[[种族|种族]]、[[信仰|信仰]]、社会状况，只要是成年公民，都有[[选举|选举]][[投票|投票]]。

== 歷史 ==
在[[光榮革命|光榮革命]]和[[法國大革命|法國大革命]]之後短命的革命政府，都有某種程度的普選制，但實際上都是以向舊統治階級報復的[[雅各宾专政|恐怖統治]]。在不久為了穩定下來，變成只有能賦交重稅的富人，即[[資產階級|資產階級]]成年男性獨佔的所有選舉和被選權的[[代議制|代議制]]。

19世纪中期，[[英国|英国]]有以男性普选权为主要诉求的[[宪章运动|宪章运动]]，提倡无论[[男性|男性]]的种族、阶级都有参政选举的权利。19世纪的領導民主运动的[[自由主义|自由主义]]者和[[社会民主主义|社会民主主义]]者，尤其在[[北欧|北欧]]，使用了口号“均等共有选举权”。普选权运动包括社会、经济和政治运动，目标在把选举权扩展到所有种族。但对于[[女性|女性]]平等的普选权或[[投票权|投票权]]、[[选举权|选举权]]等等则要在20世纪初一戰後才被重视。而最早的普选权运动发生在19世纪早期，聚焦于减除选举所要求的财产条件。1907年俄罗斯属[[芬兰大公国|芬兰大公国]]议会第一届议会议员的选举可视为第一次普选。

许多社会原先都对投票权有种族要求。比如，非白种人不能在1952年前的[[玻利維亞|玻利維亞]]投票，[[种族隔离制度|种族隔离制度]]的[[南非|南非]]投票，这种情况在1994年多党选举后才结束。1960年代[[非裔美国人民权运动|非裔美国人民权运动]]之前的[[非裔美国人|非裔美国人]]只是在理论上有[[投票权|投票权]]，之前他们並无完全保障的投票权。[[3K党|3K党]]在[[美国内战|美国内战]]之后成立，很大程度是要求胁迫阻止非裔美国人在南方州份投票，南方各州在1890年代陸續通過法律包括投票稅來限制非裔美国人投票。至今[[以色列|以色列]]為確保猶太人選民佔國內多數，仍然堅持[[巴勒斯坦人|巴勒斯坦人]]要改變身分為[[阿拉伯裔以色列人|阿拉伯裔以色列人]]才可以投票。

有一些普选系统其实还是排除一些人的选举权。比如，拒绝承认犯人投票权和精神上有疾病的人。几乎所有司法系統都拒绝非[[公民|公民]]居住者和未成年[[公民|公民]]的[[投票权|投票权]]。

由全民普選歷史看來，雖然不同地方的制度或多或少有制度上的問題，受人為的黑幕和貪污和人民的素養影響，後果不見得理想，例如未普選產生的英國國會投票同意對中國發動[[鴉片戰爭|鴉片戰爭]]，以國家的力量公然販賣毒品；美國在[[南北戰爭|南北戰爭]]後政府竟同意戰時支持聯邦政府的北方部分的奴隸州可繼續實施奴隸制度（此後有種族隔離法律），1882年[[美國國會|美國國會]]又通過[[排華法案|排華法案]]迫害[[華裔美國人|華裔美國人]]，加拿大則對華裔民眾課特別重的税，二戰後美國國會為了反共通過一系列法案迫害同情社會主義和共產主義的支持者，部分指控言過其實，公然違背[[美國憲法|美國憲法]]內[[權利法案|權利法案]]保障[[言論自由|言論自由]]的條款，形成[[白色恐怖|白色恐怖]]；德國則在二戰前選出希特勒的[[納粹黨|納粹黨]]作執政黨，實行[[一黨制|一黨制]]獨裁統治。

== 世界各地普選權 ==
根據[[瑞典|瑞典]]一個國際組織The International Institute for Democracy and Electoral Assistance (International IDEA)紀錄和研究，至2007年，全球已有185個國家的人民可投票選舉[[國會|國會]]（包括[[埃及|埃及]]、[[印度|印度]]、[[南非|南非]]等）<ref>{{cite web |url=http://www.idea.int/vt/parl.cfm |title=存档副本 |accessdate=2007-09-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20070928005920/http://www.idea.int/vt/parl.cfm |archivedate=2007-09-28 }}</ref>，但有些君主立憲制的國家如許多[[中東|中東]]國家國王或宗教領袖作為國家元首可推翻國會決議，國會並無實權，有108個[[國家|國家]]的人民可投票選舉[[最高領導人|最高領導人]]（[[總統|總統]]或[[總理|總理]]）<ref>{{cite web |url=http://www.idea.int/vt/pres.cfm |title=存档副本 |accessdate=2007-09-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20070821230621/http://www.idea.int/vt/pres.cfm |archivedate=2007-08-21 }}</ref>，如[[埃及|埃及]]1980年代後[[穆巴拉克|穆巴拉克]]曾五度當選總統，選舉過程有嚴重弊端，前四次選舉均無反對黨候選人，實際上等於獨裁統治。

普选权最先于哪个[[國家|國家]]或[[州|州]]实现的问题上存在一些争议。[[紐西蘭|紐西蘭]]、[[芬蘭|芬蘭]]、[[瑞典|瑞典]]和[[挪威|挪威]]和其他一些国家都有资格主张本国是最先实现完全平等的民主选举的国家（指最先允許女性擁有投票權利的國家，儘管存在限制）。一些[[國家|國家]]或[[州|州]]在不同時期授予和取消过[[公民普选权|公民普选权]]。[[文萊|文萊]]奉行[[君主專制|君主專制]]，所以男性公民和女性公民都沒有普選權。

== 女性的普选權 ==
{{see also|女性參政權}}
1776年，[[新澤西|新澤西]]授予女性普选權，資產資格和[[男性|男性]]一樣（詞語「居民」代替「男人」），但后来在1807年被廢止。1838年，[[皮特凯恩群岛|皮特凯恩群岛]]授予了有限的女性普选權。在19世紀後期，世界有一半國家和地區授予了有限的女性參政普选權，1861年，從[[澳洲|澳洲]][[南澳大利亞|南澳大利亞]]開始，是第一個無限制女性投票权的地方，但女性最初未被允許參加競選。

第一個授予女性參政權的國家是[[紐西蘭|紐西蘭]]，在1893年的大選僅僅數星期之前，成年女性的參政權[[法案|法案]]被通過了。1894年，澳洲南澳大利亞是第一個授予普遍女性公民投票权並允許她們競選國會議席的州。

1919年，[[美國憲法第十九修正案|美國憲法第十九修正案]]獲得[[美國國會|美國國會]]及四分之三的州議會支持通過，成年女性在[[1920年美國總統選舉|1920年美國總統選舉]]獲得平等的投票權。

1931年，[[西班牙第二共和国|西班牙第二共和国]]允許女性有參政權，當年共有三名女性在選舉中當選。在一次擴大女性政治權利在現行選舉的辯論中，激進[[女性主義|女性主義]]和[[社會主義|社會主義]]者肯特認為，[[西班牙|西班牙]]女性並未準備好投票，他們會被[[天主教會|天主教會]]影響，去投票支持[[右翼|右翼]]保守派候選人。然而Clara Campoamor以女權不被注意的定位辯護，最後她的論點佔了上風。在1933年的選舉中，所有23歲或以上的公民均獲得普选權的政治權利。但是最後Campoamor和肯特皆失去了他們的議席。
