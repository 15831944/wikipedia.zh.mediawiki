以下為介紹《[[偵探學園Q|偵探學園Q]]》的登場角色

== 「DDS」 ==
=== Q班學生 ===
; {{nihongo|小Q／連城究<ref>其本名於Vol.22由團老師揭曉。</ref>|連城 究（キュウ：れんじょうきゅう）|聲優：日本：[[绪方惠美|绪方惠美]]；台灣：[[王瑞芹|王瑞芹]]（台視）／[[汪世瑋|汪世瑋]]（AXN/Animax）|演員：[[神木隆之介|神木隆之介]]、[[中村咲哉|中村咲哉]]（童年時期）}}
:本作主角。暱稱「Q」，國三生
:學業成績一般，語言能力並較差，對社會所發生的事物一無所知，而且又不懂游泳，但其推理能力出眾
:曾於年幼時被某犯罪集團綁架，但幸好最後卻被其稱為「偵探叔叔」的其父親連城曉救出，其後連城曉更將其作為偵探心得的筆記本交給Q。後來Q便下定決心，立志成為偵探
:其深綠色頭髮和於頭部頂端的白髮為其特徵
:個性樂天，而且有些好色，經常因為看見惠的「[[裙底風光|裙底風光]]」而被惠拳打腳踢，不過當遇上案件時他卻會變得認真
:其手上戴著的錶經常無故停掉，並多次讓Q誤導正確時間
:其名言和口頭禪是「如果連偵探也放棄的話，那麼案情將永遠陷入迷宮之中！」、「提示有（數字）個！」和「答案只有一個！」。不過這幾句話也不時會被同伴借用，並等同本作的招牌台詞。
:多次與惠有心靈交流，而直到最後更成為其男友

; {{nihongo|美南惠|美南 恵（メグ：みなみ めぐみ）|聲優：日本：[[桑島法子|桑島法子]]；台灣：[[林美秀_(配音員)|林美秀]]（台視）／[[楊凱凱|楊凱凱]]（AXN/Animax）|演員：[[志田未來|志田未來]]}}
:擁有「瞬間記憶能力」的女孩，國三生，並是Q班唯一的女性
:其能力被「國立能力開發研究所」所認同，而且總是能夠幫助Q班等人
:雖然其「瞬間記憶能力」為她帶來不少的痛苦（原因是當惠一目擊某些恐怖場面時，其「瞬間記憶能力」並會將這些場面記下，而讓惠感到懼怕），但是她仍然覺得擁有「瞬間記憶能力」是一件很偉大的事，並希望自己能夠藉著其能力可以幫助到別人
:其粉紅色頭髮和紮雙馬尾為其特徵。個性熱心，而且對Q特別有好感，但整天也會對他說：「你真的很天真！」
:其經常穿著短裙，但有時當裙底走光時卻被Q和金太看見其「[[裙底風光|裙底風光]]」
:與身為大學生的姐姐茜同居
:年幼時曾進入「國立能力開發研究所」，並於該處與流相識，但後來此段記憶卻被哈迪斯王以催眠的方式消除。而惠直到團老師將其催眠解開後，惠才回想到其過去與流相識的記憶
:多次與Q有心靈交流，而直到最後更成為其女友。

; {{nihongo|天草流|天草 流（リュウ：あまくさ りゅう）|聲優：日本：[[遠近孝一|遠近孝一]]；台灣：[[李景唐|李景唐]]（台視）／[[劉傑_(臺灣)|劉傑]]（AXN/Animax）|演員：[[山田涼介|山田涼介]]、[[櫻井遼太郎|櫻井遼太郎]]（童年時期）}}
:「DDS」入學考試第一名，其推理能力與Q不分上下，是凡事也會冷靜的全能天才
:其頸上經常戴著一條頸飾為其特徵。個性冷漠，但有時卻會出現純真的一面
:動畫中流初時正處於紐約，直至第5話才歸國，第8話加入「DDS」
:其真正身份是「冥王星」首領哈迪斯王的孫子，同時也是未來的繼承人（後來揭曉並不是親生孫子<ref>於Vol.22哈迪斯王親口承認</ref>）
:過去曾經因爲受到深度催眠而失去其年幼時的記憶，後來催眠被哈迪斯王解開。
: 起初流本身不願與Q班等人成為朋友，但後來卻逐漸對Q卸下心防，並與Q交情很好，彼此信賴。於原作Vol.10（動畫第40話）中，為了脫離「冥王星」的控制而開始寄居於Q家。後來當流於與哈迪斯王對決時，他並選擇聯同「DDS」並肩作戰，最後更成功瓦解「冥王星」。

; {{nihongo|鳴澤數馬|鳴沢 数馬（カズマ：なるさわ かずま）|聲優：日本：[[川上倫子|川上倫子]]；台灣：[[李明幸|李明幸]]（台視）／[[詹雅菁|詹雅菁]]（AXN/Animax）|演員：[[松川尚瑠輝|松川尚瑠輝]]（單發電視劇）；[[若葉克實|若葉克實]]（連續劇）}}
: 「鳴澤產業」的三公子，小五生，家境富裕。是個天才程序員，曾經創作幾款很流行的遊戲軟件，並經常用電腦進行推理，但其體能於Q班等人中最差。個性較與其同齡的小孩成熟，但仍會出現其小孩子的一面。經常戴著眼鏡和一頂橙色的帽子為其特徵。擅長解讀各種密碼，對密碼的知識並非常豐富。

; {{nihongo|遠山金太郎|遠山 金太郎（キンタ：とおやま きんたろう）|聲優：日本：[[石川英郎|石川英郎]]；台灣：[[吳文民|吳文民]]（台視）／[[魏伯勤|魏伯勤]]（AXN/Animax）|演員：[[要潤|要潤]]}}
: 暱稱「金太」，高中生，空手道三段、柔道三段、劍道四段，也得過游泳和田徑的獎項。是遠山流隱術的繼承人、[[遠山景元|遠山金四郎]]的後裔之一、警視正的獨生子。擁有超人般的直覺、視覺（視力4.0）、嗅覺和聽覺，但其推理能力於Q班等人中最差。其經常瞇眼和紮馬尾為其特徵。個性既衝動、又魯莽，經常為Q班等人惹上麻煩，而且與Q一樣好色。為了不再依賴父母的照顧，決定搬離家園獨自一人生活，以兼職打工為生。

=== 學園講師 ===
; {{nihongo|團守彦|団 守彦（だん もりひこ）|聲優：日本：[[田中秀幸|田中秀幸]]；台灣：[[吳文民|吳文民]]（台視）／[[李香生|李香生]]（AXN/Animax）|演員：[[陣内孝則|陣内孝則]]}}
: 「DDS」校長；並是「DDC」（團偵探社）的創立人之一，是能被日本警方允許持槍的唯一一位的偵探（使用槍械：{{tsl|en|FN Model 1910|白朗寧M1910}}）。雙腳因受重傷而必須依靠輪椅行動。因為其有感時日無多而決定開辦Q班，以示將自己和其拍檔的信念傳續至下一代。
: 學生時代曾與哈迪斯王為摯友，但後來因爲兩人之間的信任破裂而失和。他並因此這樣而刻意告誡Q一定要相信流。最後團老師於原作最終話將被子彈一分為五的「警視廳公認偵探徽章」平均分給Q班等人後，便病逝於Q班教室內。

; {{nihongo|七海光太郎|七海 光太郎（ななみ こうたろう）|聲優：日本：[[三木真一郎|三木真一郎]]；台灣：[[李香生|李香生]]（AXN/Animax）|演員：[[山本太郎|山本太郎]]（連續劇）}}
: 「DDS」講師，並是「DDC」的偵探，自稱「團守彥的右手」。第一屆「DDS」畢業生，卻為小混混出身，年少時也常常撞板。擁有高超的易容術，有時並會喬裝跟蹤學生的任務。有時很會裝傻、搞笑，對仙人掌有莫名其妙的愛好，並經常以仙人掌的樣子來監視學生的行動。
: 曾數度與「冥王星」的成員交手，雖曾擒拿塞伯拉斯但後來卻被其逃脫。後來七海於團老師病重時曾暫任「DDS」校長和「DDC」的領袖位置，其後當團老師過世後便正式成為「DDS」的校長，並仍指導Q班。

; {{nihongo|片桐紫乃|片桐 紫乃（かたぎり しの）|聲：[[久川綾|久川綾]]|單發電視劇演員：[[鈴木砂羽|鈴木砂羽]]}}
: 團老師的秘書，並是「DDS」第一屆畢業生。擁有柔道及空手道二段的資格。原作中經常隨侍於團老師左右，卻曾一度被懷疑是「冥王星」的間諜。後來當團老師病重時片桐並擔任其看護，其後於團老師過世後仍以「DDC」偵探身份行動。
: 動畫中當片桐目睹Q、惠、金太三人解決事件的經過後，她便邀請三人參加「DDS」的入學考試。後來片桐卻被「冥王星」綁架，並被「冥王星」成員厄客德娜取代其身份於「DDS」擔任間諜，但片桐最後於「DDS」船隻「DD Sunrise」號中被眾人發現和救起。

; {{nihongo|本鄉巽|本郷 巽（ほんごう たつみ）|聲：[[梁田清之|梁田清之]]}}
: 「DDC」的偵探兼「DDS」講師，同時並是第一屆「DDS」畢業生。個性嚴謹，教學並較嚴厲。從小與七海維持著亦敵亦友的關係。其手上經常戴著一隻傷痕累累的手錶，Q班等人並透過該點而得知本鄉的實戰經驗豐富。

; {{nihongo|真木慎太郎|真木 慎太郎（まき しんたろう）|聲：[[成田劍|成田劍]]|單發電視劇演員：[[東根作壽英|東根作壽英]]}}
: 「DDS」監察醫生。過去曾於警視廳工作。曾被「冥王星」綁架，並被取代其身份於「DDS」擔任間諜。

; {{nihongo|鬼首獨郎|鬼首 独郎（おにこうべ どくろう）}}
: 「DDS」科學班主任，自稱「骷髏頭博士」。經常為「DDS」制作教材。雖然其個性愛捉弄人，又常有天馬行空的理念，但其實他仍有認真的一面。

=== A班學生 ===

; {{nihongo|遠矢邦子|遠矢 邦子（とおや くにこ）|聲：[[金田朋子|金田朋子]]|連續劇演員：[[田島有魅香|田島有魅香]]}}

; {{nihongo|霧雨右近|霧雨 右近（きりゅう うこん）}}
: 金太的兒時玩伴。其外貌雖然似笨蛋，並愛作弄人，但實際上其智慧超凡，繪畫技術高超。曾複畫九頭龍匠的幽靈畫作品「雪月花」，並得知九頭龍匠第4幅幽靈畫的存在。後來於原作最後加入「DDS」，卻於入學第一天因捉弄同學而遭罰留堂；但後來其成績於《PREMIUM》中更超越雪平而成為A班第一名。

: 三郎丸曾於《[[金田一少年之事件簿|金田一少年之事件簿]]》的「煉金術殺人事件」中短暫登場。

; {{nihongo|近松蓳|近松 すみれ（ちかまつ すみれ）}}

==「冥王星」==
; {{nihongo|黑王星彦|キング・ハデス（黒王星彦：こくおう ほしひこ）|聲：[[納谷悟朗|納谷悟朗]]|連續劇演員：[[若松武史|若松武史]]}}
: 「冥王星」首領，暱稱「哈迪斯王（キング・ハデス）」。九頭龍匠與黑王百合華之子（私生從母姓）。
: 學生時代與團老師為摯友，於兩人失和後則被父親囚禁於棲龍館之一（後來Q班使用的校舍）的地下室中。但後來他利用其方法逃出後創立「冥王星」。最後欲招攬流不果，他於棲龍館事件計劃完全敗露後便啟動於棲龍館內的放火機關，揚言要於眾人面前消失，最後不知去向。
: 雖然一向對著流稱為其祖父，但實際上為流親生祖母的同父異母弟。

; {{nihongo|賽柏拉斯|ケルベロス（ケン・L・ベルローズ）|連續劇演員：[[鈴木一真|鈴木一真]]}}
: 「冥王星」幹部。擅長催眠，高智商，而且擁有迷人的外表。
: 於「雪月花殺人事件」中曾被「DDS」教師們反用其計而被捕，而於獄中賽柏拉斯並向前來探視時其的Q班等人提供關於「DDS」内的間諜的綫索（此綫索有多種理解方法，結果是所有老師皆有可能……）。
: 後來風光逃獄，於「冥王星」解散時救出了小流，其後並解開流的父親身上的催眠術；事後並沒有再被捕獲。
: 有一番外《冥界偵探賽柏拉斯》（收錄於第20卷）以其為主角。
: 其名字源自希臘神話中負責看守冥界入口的多頭犬——[[刻耳柏洛斯|刻耳柏洛斯]]。

; {{nihongo|Sir·阿努比斯|サー﹒アヌビス|聲：[[綠川光|綠川光]]}}
: '''動畫原創'''的「冥王星」幹部，其角色地位則取自於原作的賽柏拉斯和卡龍。
: 「魔矢姬連續殺人事件」後，他卻私下委託三個綁匪綁架團老師，並將其運送到「DDS」船隻「DD Sunrise」號內囚禁，而阿努比斯更向團老師施以催眠。然而當阿努比斯與流於船上對峙時，阿努比斯便啟動船上的計時炸彈裝置。最後當團老師被眾人救走後，阿努比斯卻自願留下於船上直接被炸彈炸死。
: 其名字源自埃及神話中負責木乃伊死後生活的胡狼頭神。

; {{nihongo|Miss·由里繪|ミス﹒由利絵|聲：[[野田順子|野田順子]]|連續劇演員：[[奧貫薰|奧貫薰]]}}
: 「冥王星」成員。充當流的監護人，而流並曾向Q班等人謊稱Miss由里繪是其母親。

; {{nihongo|Miss·薰|ミス﹒カオリ|聲：[[茂呂田薰|茂呂田薰]]}}

; {{nihongo|連城曉|連城 暁|聲：[[關俊彦|關俊彦]]；連續劇演員：[[細川茂樹|細川茂樹]]}}

; {{nihongo|連城瑞枝（連城小百合）|連城 瑞枝（連城 小百合）|聲：[[龜井芳子|龜井芳子]]}}
