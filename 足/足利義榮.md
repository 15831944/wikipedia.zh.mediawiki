'''足利義榮'''（{{bd|1538年||1568年}}）是[[日本戰國時代|日本戰國時代]]的[[室町幕府|室町幕府]]第14代將軍。在職時期是[[永祿|永祿]]11年（1568年）2月至9月。父親是[[足利將軍家|足利將軍家]]的一族兼{{tsl|ja|堺公方}}[[足利義維|足利義維]]（{{tsl|ja|平島公方}}）

== 生平 ==
在[[天文_(後奈良天皇)|天文]]7年（1538年）（一說是天文9年（1540年））於[[阿波國|阿波國]][[平島莊|平島莊]]出生，家中長男。初名'''義親'''、'''義勝'''（與第7代將軍同名）。

[[永祿|永祿]]8年（1565年），在從兄兼第13代將軍[[足利義輝|義輝]]在[[永祿之變|永祿之變]]中被[[三好三人眾|三好三人眾]]和[[松永久秀|松永久秀]]殺死後，被三好三人眾和久秀以父親[[足利義維|義冬（義維）]][[中風|中風]]而不適合成為將軍為由，於是以將軍候補的身份被擁立。不過在同年11月開始，三人眾和久秀的櫂力鬥爭開始，於是被三人眾所逼，在12月發出討伐久秀的命令。

翌年（1566年）6月，被三人眾方的[[篠原長房|篠原長房]]、[[三好康長|三好康長]]等人擁立，渡海至[[淡路國|淡路國]]，並在9月23日進入[[攝津國|攝津國]][[越水城|越水城]]。12月5日，進入[[攝津國|攝津]]富田的[[總持寺|總持寺]]，在7日進入[[普門寺|普門寺]]。24日，被許可任官[[從五位下|從五位下]][[左馬頭|左馬頭]]。翌年（1567年）1月5日，正式敘任，並改名為'''義榮'''。

11月，向[[朝廷|朝廷]]申請「[[將軍宣下|將軍宣下]]」，不過因為沒有答應朝廷的獻金要求而遭拒絕。翌年（1568年）2月8日，因為三人眾推舉，受朝廷允許成為第14代將軍，於是就任將軍，並被賜予[[禁色|禁色]]，及允許[[昇殿|昇殿]]。不過三人眾和久秀的鬥爭沒有停止，而義榮自身亦因為背上有腫物，所以連就任将軍後，亦沒有入[[京都|京]]。

永祿11年（1568年）9月，義輝的次弟[[足利義昭|義昭]]被[[織田信長|織田信長]]擁立並[[上洛|上洛]]，於是三人眾在[[畿內|畿內]]與信長戰鬥，不過被擊敗，並失去畿內勢力，於是逃到[[阿波國|阿波]]。久秀向信長臣服，在沒有阻礙下，義昭在10月18日，就任第15代將軍。

此後，因為腫物惡化而病死。享年29歲（一說是31歲）。死去日期有9月13日、9月30日（『[[公卿補任|公卿補任]]』）、10月1日（『[[重編應仁記|重編應仁記]]』）、10月8日、10月20日（『[[阿州將裔記|阿州將裔記]]』）、10月22日（『[[平島記|平島記]]』、『[[嶋公方阿波公方譜|嶋公方阿波公方譜]]』）等諸多説法。而死去的地方亦有[[阿波國|阿波國]]、[[淡路國|淡路國]]、[[攝津國|攝津國]]的[[普門寺|普門寺]]等諸種説法。

== 義榮時期的室町幕府 ==
從[[阿波國|阿波國]]進入[[攝津國|攝津國]]的義榮之下，有仕於父親[[足利義維|義冬]]（{{tsl|ja|堺公方}}）和父親的養父[[足利義稙|義稙]]（第10代將軍）的[[幕臣|幕臣]]及其子孫以家臣身份出仕，不過在義冬、義榮兩代，被付到[[御內書|御內書]]的副狀發出者[[畠山維廣|畠山維廣]]等人，數量被限制著，因此勢力脆弱。當時義冬因為[[中風|中風]]而隱居，完全沒有發言力（[[足利義視|足利義視]]作為[[足利義材|足利義材]]的[[大御所|大御所]]，亦沒有權勢），因此希望換入仕於[[足利義輝|義輝]]的幕臣。當時在京幕臣的所領中，許多都集中在[[三好氏|三好氏]]的勢力圈[[京都|京都]]周圍，所領的{{tsl|ja|安堵}}和交換都置於義榮之下，而回應這個行動的有[[大舘輝光|大舘輝光]]、[[伊勢貞助|伊勢貞助]]、以及[[小笠原稙盛|小笠原稙盛]]、[[小笠原秀清|秀清]]父子。而且允許在[[永祿之變|永祿之變]]前背叛義輝而被殺死的政所頭人（[[執事|執事]]）[[伊勢貞孝|伊勢貞孝]]的孫兒伊勢虎福丸（後來的[[伊勢貞為|伊勢貞為]]）歸返，承認[[伊勢氏|伊勢氏]]宗家的再興。更提拔[[三好三人眾|三好三人眾]]之一的[[三好長逸|三好長逸]]為[[御供眾|御供眾]]。另一方面，伊勢貞孝的後任頭人[[攝津晴門|攝津晴門]]和負責政所實務的[[奉行|奉行]]眾，則厭惡推舉許多殺死義輝的三好三人眾的義榮，部份人在前往[[越前國|越前國]]與義昭交往後，再沒有返回。當時，[[室町幕府|室町幕府]]的幕臣有執行[[武家故實|武家故實]]而仕官的階層，以及裁許辯論和執行行政事務而仕官的階層（奉行衆），義榮雖然成功得到伊勢氏、[[大舘氏|大舘氏]]等武家故實階層，但是沒能得到[[諏方氏|諏方氏]]、[[飯尾氏|飯尾氏]]、[[松田氏|松田氏]]等奉行眾階層，於是在就任將軍後，造成再建幕府機構的不安形勢。即使如此，與父親義冬時期不同，沒有能爭奪將軍之位的人（雖然前將軍的實弟義昭在地位上比較合適，但是不是「將軍候補」，加上沒能[[上洛|上洛]]，而且當時仍是僧籍），因此對義榮申請「將軍宣下」比較有利。

從沒向[[春日大社|春日大社]]或朝廷獻上[[太刀|太刀]]、[[馬|馬]]等物品，被認為完全是[[三好三人眾|三好三人眾]]和[[松永久秀|松永久秀]]的傀儡，不過在三人眾與久秀對立後，被三人眾一方擁立，而且介入[[石清水八幡宮|石清水八幡宮]]的人事中，與朝廷對立。[[永祿|永祿]]10年（1567年）5月，在京都住民與[[大德寺|大德寺]]對立的事件中，由義榮派遣的幕府奉行人[[松田藤弘|松田藤弘]]與由朝廷派遣的[[勸修寺晴右|勸修寺晴右]]一同仲裁。三人眾和久秀的對立，從結果上來看，義榮的受到制約減少，發言力亦有提高，不過「將軍宣下」的延遲，以及沒能得到在京幕臣的全體（特別是奉行眾）支持，而作為主要支援的三好家中，亦因為[[三好實休|三好實休]]和三人眾對立而沒有團結等，這些情況亦有很大影響，自身生病、在任期間短暫、將軍在任中以自己意向而2次發出的奉行人奉書亦沒有被確認，並沒能發揮將軍的主體性。

== 人物 ==
*在[[室町幕府|室町幕府]]歷代將軍中，唯一一個沒有踏足本據地[[京都|京都]]。在[[日本史|日本史]]上，沒有進入本據地的將軍，就只有義榮和[[幕末|幕末]]的[[德川慶喜|德川慶喜]]（在[[大坂城|大坂城]]中就任將軍，於[[大政奉還|大政奉還]]後，被解職後才可以進入[[江戶城|江戶城]]）2人。

*[[永祿之變|永祿之變]]後，三人眾和[[松永久秀|松永久秀]]的對立原因，其中一個理由就是義榮是由誰人擁立。而且在{{tsl|ja|堺公方}}以來，人才基礎亦很脆弱，沒能得到在[[幕府|幕府]]中執行行政實務的[[奉行|奉行]]眾支持，因此即使能進入京都，亦沒可能令幕府機構活動。

*木像由[[栃木縣|栃木縣]][[足利市|足利市]][[家富町|家富町]]的[[鑁阿寺|鑁阿寺]]所藏。而仿造的像則由{{tsl|ja|平島館|阿南市立阿波公方・民俗資料館}}所藏，日常亦有展示。

*一說指久秀令從弟[[松永喜內|松永喜內]]暗殺義榮，不過被撃退，於是命令毒殺。（『阿州將裔記』）

*根據『[[江源武鑑|江源武鑑]]』中的說法，義榮的存在完全被無視。

*一說指與[[足利義輝|足利義輝]]不和。
