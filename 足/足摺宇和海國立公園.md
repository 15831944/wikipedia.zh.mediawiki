'''足摺宇和海國立公園'''（假名：{{lang|ja|あしずりうわかいこくりつこうえん}}），是位在[[日本|日本]][[高知縣|高知縣]]、[[愛媛縣|愛媛縣]]的[[日本國立公園|國立公園]]。

== 年表 ==
* 1955年4月1日 - 被指定為'''足摺國定公園'''。
* 1972年11月10日 - 追加宇和海地域，變更為國立公園，更名為'''足摺宇和海國立公園'''。

== 主要景點 ==
* [[足摺岬|足摺岬]]
* [[龍串|龍串]]
* [[宇和海|宇和海]]
* [[滑床溪谷|滑床溪谷]]
* [[篠山|篠山]]
* [[法華津峠|法華津峠]]

== 面積 ==
* 面積 - 11,345ha （陸域部分）
** 高知縣 - 6,041ha
** 愛媛縣 - 5,304ha

== 關連市町村 ==
* 高知縣 - [[四萬十市|四萬十市]]、[[宿毛市|宿毛市]]、[[土佐清水市|土佐清水市]]、[[大月町|大月町]]
* 愛媛縣 - [[宇和島市|宇和島市]]、[[西予市|西予市]]、[[松野町|松野町]]、[[鬼北町|鬼北町]]、[[愛南町|愛南町]]
