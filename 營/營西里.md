'''營西里'''位於[[臺南市|臺南市]][[西港區|西港區]]，是因後營聚落人多而分割而成的里{{r|縣|page=590}}{{r|西港常民文化|page=77}}。2018年1月29日調整跟西港里之間的-{里}-界{{r|里鄰調整}}。

== 沿革 ==
該里在日治時期屬於[[西港庄|西港庄]][[大字|大字]]後營，二次大戰後將後營聚落分成營西村與後營村，兩者合稱「後營二村」{{r|縣|page=590}}{{r|西港常民文化|page=77}}{{r|地名}}。2010年12月25日，因臺南縣市合併升格，改為營西里。2018年1月29日，調整了與西港里之間的里界{{r|里鄰調整}}。

== 聚落與地名 ==
營西里境內主要的聚落有為後營（西半部），另外過去還有打鐵、那菝宅等舊地名{{r|縣|page=590}}{{r|西港常民文化|page=77}}。打鐵在營西里南部，過去糖鐵鐵路西邊約100公尺處，據說是因過去有打鐵舖而得名{{r|縣|page=590}}{{r|西港常民文化|page=77}}。該聚落在日治時期還存在，現已散庄{{r|縣|page=590}}{{r|西港常民文化|page=77}}。那菝宅在營西、後營、港東三里交會處，後營永安宮附近，據說因過去種滿那菝而得名{{r|縣|page=590}}{{r|西港常民文化|page=77}}。

== 教育 ==
境內設有後營國小，是西港區歷史最悠久的學校{{r|縣|page=590}}。

== 宗教信仰 ==
里內有後營慈鳳宮、後營永安宮等廟宇，另外以位在後營里的後營普護宮為信仰中心{{r|地名}}{{r|西港常民文化|page=77}}。而在後營國小西邊有一間福德正神廟，據說是過去後營跟荔枝林庄鬥風水而興建{{r|地名}}{{r|西港常民文化|page=80}}。荔枝林庄位在後營東南，和後營關係不好{{r|地名}}{{r|西港常民文化|page=80}}。後來荔枝林在聚落北邊開了一條正對著後營的大溝，要敗後營的風水，後營人乃向普護宮保生大帝請示，而蓋了這間朝向荔枝林的福德正神廟{{r|地名}}{{r|西港常民文化|page=80}}。之後荔枝林敗下陣來，居民四散{{r|地名}}{{r|西港常民文化|page=80}}。
