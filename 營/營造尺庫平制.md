'''營造尺庫平制'''，是[[清朝|清朝]][[康熙|康熙]]年間制定發布，[[中華民國|中華民國]][[北洋政府|北洋政府]]繼續的[[度量衡|度量衡]]制度。

== 清朝 ==
清[[康熙|康熙]]五十二年（1713年）制的《[[御製律呂正義|御製律呂正義]]》以及《[[御製數理精蘊|御製數理精蘊]]》都定度量衡表，而[[光緒|光緒]]三十四年（1908年）《推行劃一度量衡章程四十條》，形成[[營造尺|營造尺]]、[[庫平兩|庫平兩]]的標準化。<ref>{{Cite web |url=http://www.shtong.gov.cn/newsite/node2/node2245/node71621/node71626/node71635/node71714/userobject1ai76193.html |title=第一篇计量 第一章计量单位制 第一节 营造尺库平制 |accessdate=2012-10-16 |author=责任编辑 过文瀚 |date=2003年12月第1版 |work=上海质量技术监督志|publisher=上海市地方志办公室}}</ref>

== 中華民國北洋政府 ==
民國4年（1915年）1月7日，北洋政府公布《權度法》。其第二條明定甲制營造尺庫平制以及乙制[[米制|萬國權度通制]]並行，長度基本單位是[[營造尺|營造尺]]，等於0.32公尺（米），而重量基本單位是[[庫平兩|庫平兩]]，等於0.037301[[公斤|公斤]]（千克），直接法定的換算用'''粗體'''，列表如下：<ref>{{Cite web |url=http://gaz.ncl.edu.tw/detail.jsp?sysid=D1500002 |title=公布「權度法」|author=大總統令法律第1號 |date=民國4年（1915年）1月7日 |work=政府公報(重印本)957期第85-94頁 |publisher=中華民國國家圖書館政府公報資訊網}}</ref>

== 中華民國國民政府 ==
{{Wikisource|度量衡法 (民國18年)}}

[[国民革命|国民革命]]之後，取代北洋政府的[[國民政府|國民政府]]以[[市制|市用制]]取代營造尺庫平制，於1929年（民國18年）2月2日制定2月16日公布、1930年（民國19年）1月1日施行《度量衡法》，其第二條：「中華民國度量衡採用[[米制|萬國公制]]為標準制，並暫設輔制稱曰市用制。」<ref>[http://lis.ly.gov.tw/lghtml/lawstat/version2/01926/0192618020200.htm 立法院法律系統：度量衡法 (民國18年)] {{webarchive|url=https://web.archive.org/web/20140425025351/http://lis.ly.gov.tw/lghtml/lawstat/version2/01926/0192618020200.htm |date=2014-04-25 }}</ref>
