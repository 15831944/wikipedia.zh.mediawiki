'''丰镐房'''是[[蒋中正|蒋中正]]位于家乡[[奉化|奉化]][[溪口镇_(宁波市)|溪口]]的故居建筑之一，位于今[[浙江省|浙江省]][[宁波市|宁波市]][[奉化区|奉化区]][[溪口镇_(宁波市)|溪口镇]]武岭路下街，占地4800平方米，建筑面积1850平方米。丰镐房最初为蒋家祖传的三间旧屋和蒋中正儿时居所，1929年冬蒋中正重建此屋，[[蒋经国|蒋经国]]、[[毛福梅|毛福梅]]也曾在此居住<ref name="王天苍">{{cite journal zh|title=蒋氏丰镐房|author=王天苍|journal=宁波通讯|year=2003|issue=12}}</ref>。1949年[[中国人民解放军|中国人民解放军]]占领溪口后，丰镐房由解放军使用并得到保护，后作为中共溪口区委员会办公地，1979年按原貌恢复<ref name="王舜祁">{{cite book|chapter=蒋氏故居保护和开放|chapterurl=http://www.nbzx.gov.cn/art/2015/9/18/art_4055_690833.html|accessdate=2017-09-02|title=宁波文物古迹保护纪实|author=王舜祁|publisher=宁波出版社|year=2000|isbn=9-787-80602-418-8|deadurl=no|archiveurl=https://web.archive.org/web/20180222043756/http://www.nbzx.gov.cn/art/2015/9/18/art_4055_690833.html|archivedate=2018-02-22|df=}}</ref>。1996年，丰镐房与[[玉泰盐铺|玉泰盐铺]]、[[小洋房_(溪口)|小洋房]]一道以[[蒋氏故居|蒋氏故居]]名义被列入[[第四批全国重点文物保护单位|第四批全国重点文物保护单位]]<ref name=":2">{{cite web|url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=830&type=2|title=蒋氏故居|publisher=宁波市文物保护管理所|date=2006-06-22|accessdate=2017-09-02|deadurl=no|archiveurl=https://web.archive.org/web/20180222043927/http://www.nbwb.net/pd_wwbh/info.aspx?Id=830&type=2|archivedate=2018-02-22|df=}}</ref>。

== 历史 ==
丰镐房原为[[蒋中正家族|蒋中正家族]]拥有的三间旧屋。清光绪十四年（1889年），因出生地[[玉泰盐铺|玉泰盐铺]]遇火焚毁，蒋中正一家搬迁到旧屋正房“报本堂”西侧居住。蒋中正父亲[[蒋肇聪|蒋肇聪]]去世后，蒋中正与其弟蒋瑞青分得此房。因蒋中正与其弟辈分为“周”，因而以西周都城[[丰镐|丰镐]]将两人的房名定为“丰房”和“镐房”。后蒋瑞青早逝，房屋为蒋中正所得，故房名改为“丰镐房”，蒋经国的字“建丰”和[[蒋纬国|蒋纬国]]的字“建镐”也是由房号而来<ref name="王天苍" />。1929年蒋中正发迹后，使用以房换房的方式，以溪口上街楼房替换原有的旧房，并拆除旧房营建新房，形成当下的布局。此后蒋中正每次返乡，均会到丰镐房巡视<ref name="王舜祁" />。

1949年1月28日除夕，蒋中正最后一次在丰镐房报本堂吃年夜饭，同席的有[[陈立夫|陈立夫]]、[[張群|张群]]等政要。5月25日，[[中国人民解放军|中国人民解放军]]占领溪口，丰镐房先后由二十一军和二十二军管理。1958年，丰镐房交由地方管理，成为中共溪口区委员会办公用房，[[文化大革命|文化大革命]]期间成为造反派指挥中心。1979年起，丰镐房进行了维修和陈设的恢复，并于1984年对公众开放<ref name="王舜祁" />。

== 建筑 ==
[[File:Plan_of_Fenghao_House,_Chiang_Kai-shek's_house.svg|thumb]]
丰镐房扩建由上海孙余生营造厂建造，蒋中正舅父孙琴凤和姐夫宋周运管理建设过程。建筑占地面积4800平方米，建筑面积1850平方米，共有房间49间，为江南旧式世家府第建筑<ref name="王舜祁" />。建筑中轴线由南向北依次为大门、前庭、内门、前厅、内庭和后堂，内庭左右两侧有东西厢房，西厢房西有独立小楼，后堂西有六间平屋。大门、素居、报本堂、独立小楼为清代建筑，其余为1929年扩建<ref name=":2" />。

前厅题额为“素居”，因蒋家世代信佛吃素而得名<ref>{{Cite web|url=http://nb.zjol.com.cn/nb/system/2013/11/18/019711093.shtml|title=奉化溪口：民国文化第一镇|accessdate=2018-02-21|author=|date=2013-11-18|work=|publisher=浙江在线-宁波频道|deadurl=no|archiveurl=https://web.archive.org/web/20180408210257/http://nb.zjol.com.cn/nb/system/2013/11/18/019711093.shtml|archivedate=2018-04-08|df=}}</ref>。建筑为硬山顶两层房屋，下层为接待间和账房工作间，上层中部为佛堂，两侧为居室，[[王采玉|王采玉]]、[[毛福梅|毛福梅]]在此念佛<ref name="王天苍" />。后堂建筑为硬山顶，面阔三间，[[风火墙|风火墙]]为五马头。后堂题额为“报本堂”，由[[吳敬恆|吳敬恆]]题写<ref name=":2" />。建筑用于祭祖，设有蒋中正曾祖以下[[牌位|牌位]]。屋顶有福禄寿三星堆塑，柱头有《[[三国演义|三国演义]]》故事雕塑，走廊、墙壁上亦有《[[封神演义|封神演义]]》故事雕刻。堂前柱上楹联“报本尊亲是谓至德要道，光前裕后所望孝子顺孙”为[[沙孟海|沙孟海]]撰，蒋中正书写。走廊上匾“寓理帅气”亦为蒋中正手书，左侧为跋文。东西厢房分别为[[宋美龄|宋美龄]]和毛福梅居室，分别为西式和中式陈设。四合院东的两层楼房为工作人员住所，西厢房西的独立楼房为蒋母王采玉的居所，其楼梯较小，为蒋中正为方便母亲上下楼扶手而设计。报本堂西的房屋为客房和佣人居所，毛福梅晚年腿脚不便时也曾在此居住<ref name="王天苍" /><ref name="王舜祁" />。建筑物内还有2座小花园，设圆洞门，上有“格物”、“致知”门额<ref name=":2" />。

== 保护与展示 ==
{{external media|width=250px|image1=[http://www.nbwb.net/UploadFile/images/2018012213421576.jpg 丰镐房保护区划图]，原始内容[https://web.archive.org/web/20180711154324/http://www.nbwb.net/UploadFile/images/2018012213421576.jpg 存档]于2018年7月11日}}1949年，蒋中正离开中国大陆后，[[毛泽东|毛泽东]]曾经指示进军的中国人民解放军“在占领奉化时，要告诫部队，不要破坏蒋介石的住宅、祠堂及其他建筑物”<ref>{{cite book|title=毛泽东军事文集第五卷|chapter=占领吴淞嘉兴等地应注意的问题|publisher=军事科学出版社|year=1993|isbn=9-787-8-00217-371}}</ref><ref>{{Cite web|url=http://dangshi.people.com.cn/GB/144956/12588548.html|title=毛泽东为何亲自下令保护蒋介石故居？|accessdate=2018-02-21|author=|date=2010-08-31|work=|publisher=人民网-中国共产党新闻|deadurl=no|archiveurl=https://web.archive.org/web/20180222043919/http://dangshi.people.com.cn/GB/144956/12588548.html|archivedate=2018-02-22|df=}}</ref>，浙江省军管会主任[[谭震林|谭震林]]也曾指示军队不要随便移动蒋氏故居中的陈设，并对丰镐房派专人看管<ref name="王天苍" />。文化大革命期间，红卫兵破坏了一部分蒋氏相关遗迹，但丰镐房保存完好。1979年，溪口区委迁出丰镐房后，丰镐房得到修葺，雕塑上的石灰被剔除，彩画得到修复，蒋中正亲笔书写的楹联和牌匾被重新悬挂，报本堂的牌位也被重新摆放。1999年，补充了蒋中正家室和家庭生活相关的资料。1996年，丰镐房作为蒋氏故居的一部分被列入第四批全国重点文物保护单位<ref name="王舜祁" />。

== 轶事 ==
据传说，丰镐房扩建时，旁边的周顺房千层饼店不愿意搬迁，蒋中正并没有勉强，后来千层饼店被称为“民国最大的钉子户”<ref>{{Cite web|url=http://www.cctv.com/program/zbzg/topic/geography/C17621/20070201/107785_1.shtml|title=蒋介石与溪口|accessdate=2018-02-21|author=|date=2007-02-01|work=|publisher=央视国际|deadurl=no|archiveurl=https://web.archive.org/web/20170326083425/http://www.cctv.com/program/zbzg/topic/geography/C17621/20070201/107785_1.shtml|archivedate=2017-03-26|df=}}</ref>。但周顺房后人周孝裕则称，其父曾为蒋中正修建丰镐房让出了今东厢房前的一间小屋，且当下丰镐房的范围是成为旅游景点后扩建才形成的，因而周家并非钉子户<ref>{{Cite web|url=http://www.fhtv.net.cn/folder16/folder84/2016-06-22/69824.html|title=周孝裕：我不是最牛钉子户|accessdate=2018-02-21|author=|date=2016-06-22|work=|publisher=锦凤网-奉化网络广播电视台|deadurl=no|archiveurl=https://web.archive.org/web/20180711164223/http://www.fhtv.net.cn/folder16/folder84/2016-06-22/69824.html|archivedate=2018-07-11|df=}}</ref>。
