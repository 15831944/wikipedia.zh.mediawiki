== 解剖結構 ==
臀部由[[臀大肌|臀大肌]]和[[臀中肌|臀中肌]]兩塊臀肌組成，上層被[[脂肪組織|脂肪組織]]覆蓋。臀部的最上伸延至上[[髂骨|髂骨]]，最下到水平臀肌摺紋；往下就是[[大腿|大腿]]。

在臀部分布著臀上神經與臀下神經，表現於臀部肌肉群，尤其是臀大肌上。往下走則有大腿後皮神經，主控大腿後肌肉群、[[股二頭肌|股二頭肌]]、[[腿後腱|半腱肌與半膜肌]]等。再往下的{{link-en|坐骨神經|Sciatic nerve}}（sciatic nerve），分脛骨神經與腓骨總神經，掌控小腿的肌肉群。<ref>{{Cite book|author=李家雄 |title=《男女本性》 |year=2000 |publisher=書泉 |page=頁26 }}</ref>

[[臀大肌|臀大肌]]有兩個附著處：[[股骨|股骨]][[粗線|粗線]]的三分一部分，和iliotibial tractus的部分。臀大肌中間被一個溝分隔，稱為[[股溝|股溝]]，其中位置就是[[肛門|肛門]]所在。如此，臀部容許[[靈長類動物|靈長類動物]]舒適地坐下，而不需要將重心放在[[腿|腿]]上。相反，[[貓|貓]]和[[狗|狗]]要坐下時，不能這樣做。

一些[[狒狒|狒狒]]和所有[[長臂猿|長臂猿]]都長有長毛，但牠們的臀部皮膚硬而沒有毛。

== 服裝 ==
[[File:Drake_Jaden_3.jpg|thumb]]Drake Jaden身著露臀的[[性虐|性虐]]服飾。]]

很多文化都對性[[裸體|裸體]]有[[忌諱|忌諱]]，而展露臀部也是一種忌諱，正如一般的[[性感帶|性感帶]]。

主流服裝之設計，一般把臀部完全遮掩，無論是否必須。相反，日本的傳統男性腰布「{{lang|ja|褌}}」（fundoshi，亦作「[[兜襠布|兜襠布]]」）幾乎展露整個臀部，但並不顯示穿著者裸露的癖好。

一些衣物則用來突出臀部，無論是外衣、內衣或內褲，例如[[丁字褲|丁字褲]]。另一例子是，一些女性穿著特別設計的褲（例如[[牛仔褲|牛仔褲]]），因為令褲子的腰部布料減少，故此顯露出[[股溝|股溝]]。此外，如果[[比基尼泳衣|比基尼泳衣]]的布料較少，穿著時也會顯露股溝。

== 健美 ==
在大腿近臀部的連接位，當一個人過胖時，會出現一種類似橙皮的組織。這些組織，俗稱「橙皮紋」，學名叫作「[[蜂窩組織|蜂窩組織]]」，是一種脂肪組織。這些組織，被一般坊間民眾認為「不雅」，而務求在[[減肥|減肥]]的過程中消除。
