'''钦北高速铁路'''位於[[中华人民共和国|中华人民共和国]][[广西壮族自治区|广西壮族自治区]]境内，是连接[[钦州市|钦州市]]和[[北海市|北海市]]的一条客货共线[[高速鐵路|高速鐵路]]。该铁路属於广西沿海铁路钦州至北海段扩能改造工程，旨在弥补既有既有单线铁路[[钦北铁路|钦北铁路]]的不足，可以显著提高广西沿海城市及港口的铁路客货运输能力。

该线与[[南钦铁路|南钦线]]一道又合称“邕北线”。

== 简介 ==
钦北高速铁路自钦州北引出，止於北海港前，为国铁Ⅰ级[[复线铁路|双线]][[電氣化鐵路|電氣化鐵路]]，设计时速250公里/小时，以客运为主、兼顾货运。线路走向与[[钦北铁路|钦北铁路]]大致平行，正线全长99.6公里，其中钦州北至合浦段新建双线76公里并保留既有铁路，形成三线模式；合浦至北海段增建第二线23.6公里，形成双线格局；北海至北海港前段进行电气化改造，作为本线货运系统的延伸线<ref>{{Cite web |url=http://www.beihai.gov.cn/15913/2013_12_9/15913_388646_1386551460938.html |title=广西沿海铁路钦州北至北海段扩能改造工程项目简介 |publisher=北海市政府门户网站 |date=2013年12月1日}}</ref><ref name="tljsgc">{{Cite web |url=http://www.rebcenter.com/projectInfo/projectInfoContent.action?id=305 |title=广西沿海铁路钦州北至北海段扩能改造工程 |publisher=铁路建设工程网}}</ref>。

全线设{{站|钦州东}}、{{站|大马}}、{{站|合浦}}、{{站|北海}}和{{站|北海港前}}共5个车站，其中大马为越行站，钦州东、合浦、北海和为中间站，北海港前站为货运站。

== 主要技术指标 ==
*铁路等级：国铁Ⅰ级
*正线数目：双线
*旅客列车设计行车速度：250 km/h
*最小曲线半径：4500 m，引入钦州北、北海地区根据运营需要合理选定
*限制坡度：6‰
*牵引种类：电力牵引
*牵引质量：4000t
*到发线有效长度：850 m
*桥隧比例：35%
*闭塞类型：[[自动闭塞|自动闭塞]]
*线间距：4.6 m
