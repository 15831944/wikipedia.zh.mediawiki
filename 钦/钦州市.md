'''钦州市'''（[[邮政式拼音|邮政式拼音]]：{{lang|la|Yamchow}}、{{lang|la|Yen Chow}}）位于[[廣東|廣東]]西端，广西南部，[[北部湾|北部湾]]沿岸。西界[[防城港市|防城港市]]，北接[[南宁市|南宁市]]，东邻[[玉林市|玉林市]]，南临[[北海市|北海市]]及[[北部湾|北部湾]]。前身是廣東下四府八鄉之一的[[钦县|欽縣]]。1965年後被當局劃[[廣西|廣西]]管轄，現時是[[广西壮族自治区|广西壮族自治区]]管辖的[[地级市|地级市]]，地处桂南台地[[丘陵|丘陵]]区，北有罗阳山，东北为[[六万大山|六万大山]]，西北为[[十万大山|十万大山]]。[[钦江|钦江]]自东北向西南流经市区后出北部湾。总面积1.09万平方公里，人口320.93万。钦州为区内重要[[交通枢纽|交通枢纽]]，是[[中国—东盟自由贸易区|中国—东盟自由贸易区]]前沿城市，[[南防铁路|南防铁路]]、[[钦北铁路|钦北铁路]]和[[钦防铁路|钦防铁路]]于市区交汇，[[钦州港|钦州港]]为北部湾重要[[港口|港口]]。

== 历史 ==
钦州古称'''古越'''、'''安州'''，有1400多年的历史。[[先秦|先秦]]时期，两广地区为[[百越|百越]]民族所居住。[[秦始皇|秦始皇]]最早在全国范围内推行[[郡县制|郡县制]]，统一[[岭南|岭南]]地区（今两广）后设立[[象郡|象郡]]，并将现在的钦州一带划归象郡管辖。[[汉朝|汉朝]]时期开始设立“[[州|州]]”的建制，两广一带归[[交州|交州]]管辖，今钦州一带归交州[[合浦郡|合浦郡]]管辖。[[三国|三国]]时期，交州是[[吴_(三国)|吴国]]辖地，一直到[[晋朝|晋朝]]，今钦州都属于[[合浦郡|合浦郡]]管辖。[[南北朝|南北朝]]时，[[刘宋|宋]][[元嘉|元嘉]]年间设[[宋寿郡|宋寿郡]]，[[南齐|南齐]]时为[[宋寿县|宋寿县]]地，属[[交州|交州]]管辖；[[梁朝|梁]]、[[陈朝|陈]]时成为[[安京郡|安京郡]]地，属[[安州|安州]]管辖。

[[隋朝|隋]][[开皇|开皇]]九年（589年）废宋寿郡，安京郡改置[[安京县|安京县]]。开皇十八年（598年）将安州改为钦州，取“钦顺”之义，亦为“取钦江为名”（《[[元和郡县志|元和郡县志]]》），这是钦州的最早得名，治所为[[钦江县|钦江县]]（今钦州市东北）。[[大业|大业]]初又改置为[[宁越郡|宁越郡]]。[[唐朝|唐]]时推行[[道|道]]的建制，今两广地区设立[[岭南道|岭南道]]。[[武德|武德]]四年（621年）撤宁越郡改置钦州。[[至德|至德]]二年（757年）改安京县为[[保京县|保京县]]，辖今广西钦州、[[灵山|灵山]]（今灵山县西），后数次更名。[[五代十国|五代十国]]时期，两广地区为[[南汉|南汉]]国辖地。[[宋朝|宋朝]]统一中国后推行[[路|路]]的建制，今广西、海南等地归[[广南西路|广南西路]]管辖，同时今钦州一带仍设置钦州。[[北宋|北宋]]初保京县复为安京县，[[开宝|开宝]]五年（972年）废钦江县，[[景德|景德]]三年（1006年）改安京县为[[安远县|安远县]]。[[南宋|南宋]]移治安远（今钦州）。

[[元朝|元朝]]时始设[[行省|行省]]，钦州一带归[[湖广行省|湖广行省]]管辖，同时将钦州改为[[钦州路|钦州路]]。[[明朝|明朝]]时钦州划归[[广东布政使司|广东布政使司]]管辖，[[洪武二年|洪武二年]]（1369年）为[[钦州府|钦州府]]，[[洪武|洪武]]七年（1374年）降为钦州，废[[安远县|安远县]]并入州；洪武十四年（1381年）属[[廉州府|廉州府]]。[[清朝|清]]初时钦州继续归广东管辖，同时钦州不设县。[[光绪十四年|光绪十四年]]（1888年）划出钦州西部，设置[[防城县|防城县]]。[[光绪|光绪]]中钦州升为[[直隶州|直隶州]]，辖今钦州、防城两地，仍隶属[[广东省|广东省]]。1912年改州为县，1914年属[[钦廉道|钦廉道]]。

[[民国|民国]]时期至共和国建立初期，钦州一带仍属广东省辖。[[中华人民共和国|中华人民共和国]]成立后，属广东省[[南路专区|南路专区]]，1950年改[[钦廉专区|钦廉专区]]，1951年改称[[钦州专区|钦州专区]]。1952年钦州专区划归[[广西省|广西省]]。1955年改为[[合浦专区|合浦专区]]，并复归广东省。1959年改属[[湛江专区|湛江专区]]。1964年，设立钦州[[僮族|僮族]]（[[壮族|壮族]]的旧称）自治县（1963年批准）。1965年，钦州、防城港一带划归广西壮族自治区，设立钦州专区，钦州壮族自治县改为[[钦州县|钦州县]]。1971年改称[[钦州地区|钦州地区]]。1983年划出[[北海市|北海市]]。1987年将合浦县划归北海市。1993年划出[[防城|防城]]、[[上思|上思]]两县。1994年6月28日，国务院批准，撤销钦州地区，改设[[地级|地级]]钦州市。

== 地理 ==
北与[[南宁|南宁]]，西与[[防城港|防城港]]，东与[[玉林|玉林]]，东北与[[贵港|贵港]]，东南与[[北海市|北海]]相邻。位于北纬20°54′至22°41′，东经107°27′至109°56′之间。其陆地海岸线长311.44公里，东北和西北分别有[[六万大山|六万大山]]和[[十万大山|十万大山]]，海拔均过千米。

=== 气候 ===
钦州属南[[亚热带季风气候|亚热带季风气候]]，气候温暖，降水充沛。冬季短而温和，夏季长而闷热。1月平均气温13.6℃，极端最低气温-1.8℃（1955年1月12日）。7月平均气温28.4℃，极端最高气温37.9℃（2005年7月19日）。年平均气温22.2℃。降雨集中在5月~8月，降水量占全年的三分之二。年均降水量2150.4毫米。年均日照时数1721.1小时。冬春季节日照较少，夏秋则较为晴朗。<ref>[http://cdc.cma.gov.cn/home.do 中国气象科学数据共享服务网] {{webarchive|url=https://web.archive.org/web/20150302053152/http://cdc.cma.gov.cn/home.do |date=2015-03-02 }}</ref>

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为307.97万人<ref>{{cite book |author=钦州市统计局|title=《钦州市2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加10.49万人，增加3.53%。年平均增加0.35%。其中，男性人口为163.83万人，占53.20%；女性人口为144.14万人，占46.80%。人口性别比（以女性为100）为113.66。0－14岁人口为87.52万人，占28.42%；15－59岁人口为182.09万人，占59.13%；60岁以上人口为38.36万人，占12.46%；其中，65岁及以上人口为27.57万人，占8.95%。

=== 民族 ===
全市常住人口中，汉族人口为275.46万人，占89.44%；各[[少数民族|少数民族]]人口为32.51万人，占10.56%。

== 物产 ==
* [[北部湾|北部湾]]大陆架蕴藏总储量约5亿吨的石油和丰富的天然气。有总储量约1930万吨的石英砂、储量约30亿吨的陶土。
* 大垌煤矿 
* 农作物有水稻、花生、香蕉、沙柑、扁柑、沙梨、甘蔗、火龙果、荔枝、龙眼、百香果和各类蔬菜等。 
* 钦州所濒临的北部湾是中国沿海四大渔场之一。主要鱼类500多种，虾类43种，另有蟹类、贝类、头足类、爬行类及哺乳类等种类繁多的海生动物和藻类等多种植物，海特产有珍珠、海参、文蛤、沙虫 、鱿鱼、地鱼 、红鱼、鳝肚、对虾、大蚝、海蛰等。

== 经济 ==
* 工业有化肥、机械、制糖、农机、采矿、生物科技、药材加工以及临海石油工业。
* 农业以渔业、养殖业和水稻种植业为主。
* 并积极发展对外贸易。

== 文化 ==
===语言===
钦州为[[粤語|粤語]]区，俗稱[[白话|白话]]，通行[[w:en:Qin-Lian_Yue|钦廉片粤语]]（又謂'''欽州話'''），与[[省港澳|省港澳]]通行之[[廣州話|廣州話]]相通。辖区内同时并存[[客家话|客家话]]、[[广西平话|平话]]以及[[壮语|壮语]]。

===名人===
钦州近代名人有清末著名军事家，[[黑旗军|黑旗军]]领袖[[刘永福|刘永福]]、[[冯子材|冯子材]]，北宋诗人[[苏东坡|苏东坡]]、清末民初著名画家[[齐白石|齐白石]]和现代诗人[[田汉|田汉]]都到过钦州，并有佳作为证。
