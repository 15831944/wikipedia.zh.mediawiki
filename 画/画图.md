'''画图'''是一个[[计算机图形学|图像绘画]]程序，是微软公司[[Windows|Windows]]操作系统的預載軟件之一。自從發佈以來，大部分的[[Windows|Windows]]操作系统都內建了這個軟體。它的外號通常是'''MS Paint'''或者'''Microsoft Paint'''。這個軟體可以打开并查看[[WMF|WMF]]和[[EMF|EMF]]文件，打开並保存[[BMP|BMP]]（24-bit、256色、16色和[[單色|單色]]）、[[JPEG|JPEG]]、[[GIF|GIF]]（不包括透明和動畫數據，但Windows 98和更新後的Windows 95可以支援後者）、[[PNG|PNG]]（不包括[[阿尔法通道|阿尔法通道]]）以及[[TIFF|TIFF]]格式的图像文件。比较舊的小畫家版本不支持PNG格式，也只能在使用[[图像过滤|图像过滤]]的情况下打开GIF、JPEG、和TIFF文件。另外，新版本不再支持[[PCX|PCX]]格式，以及其他比较古老的格式（例如[[RLE|RLE]]）。新版本也不再支持在Windows 3.x中的画笔以及Windows 1和2中的画图所支持的[[MSP_(檔案格式)|MSP]]文件格式。

這個軟體有兩個模式，分別是彩色和單色（[[黑白|黑和白]]），而不包括灰階模式。

2017年7月，微软预告在秋季创意者更新中以[[画图_3D|画图 3D]]取代画图，同时将在[[Windows商店|Windows商店]]中提供画图的免费下载<ref>{{cite web |url=http://www.bbc.com/news/technology-40705466 |title=Microsoft signals end of Paint program |date=24 July 2017 |last=Kleinman |first=Zoe |publisher=BBC |accessdate=24 July 2017}}</ref><ref>{{cite web |url=https://blogs.windows.com/windowsexperience/2017/07/24/ms-paint-stay/#PTsP85lW5H4CtLSS.97 |title=MS Paint is here to stay |date=24 July 2017 |last=Saunders |first=Megan |work=Windows Blog |publisher=Microsoft |accessdate=24 July 2017}}</ref>。

== 歷史 ==
画图的第一個版本，內建於[[Windows_1.0|Windows 1.0]]中。這個版本只支援[[MSP_(檔案格式)|MSP]]和[[BMP|BMP]]檔案格式。前者已不被新版本的小畫家支援。同樣不被支援還有[[PCX|PCX]]和[[Run-length_encoding|RLE]]檔案。較舊版本的小畫家不支援開啟和編輯PNG檔案，配合某些圖像插件，都只可以開啟GIF、JPEG和TIFF檔案。

在Windows 3.1的繁體中文版中被稱為'''調色盤'''，简体中文版中则被称为'''画笔'''，直到[[Windows_95|Windows 95]]後更為今名。同時，新的画图一同發佈。此後，相同的圖示和調色板，一路沿用到[[Windows_Vista|Windows Vista]]中。

Windows 95所带的画图可以在颜料盒中显示索引颜色图片使用的颜色。在[[Windows_98|Windows 98]]、[[Windows_2000|Windows 2000]]或者[[Windows_Me|Windows Me]]版本的小画图，如果安裝了必須的微軟圖像濾光鏡，就可以將圖片儲存成JPEG和GIF。微軟圖像濾光鏡通常是其他的微軟產品，例如[[Microsoft_Office|Microsoft Office]]或者[[Microsoft_PhotoDraw|Microsoft PhotoDraw]]。還有，如果開啟或貼上的圖像比畫布大，畫布能自動放大，去適應圖像。

在[[Windows_XP|Windows XP]]和後期的作業系統中，画图是建基於[[图形设备接口|GDI+]] <ref> [http://support.microsoft.com/kb/917498 Paint in Windows XP and onwards uses GDI+] </ref>上。即是画图可以直接將圖像儲存成JPEG、GIF、TIFF和PNG格式，不用安裝任何圖像濾光鏡。但是，阿尔法通道仍然不被支援，因為画图只可以支援[[24-bit|24-bit]]深色的圖像。另外，由於Windows XP不再內建Imaging這個軟體，所以要從掃描機或數碼相機擷取圖片的話，就要利用画图。但是，複合色功能亦被移除，所以画图不能建立透明背景的GIF圖像。另外，'儲存'和'開啟'色板顏色檔案（.pal）功能亦被移除。

在[[Windows_Vista|Windows Vista]]中，圖示和調色板都被更新。另外，增加了復原次數，新增了縮放滑動拉桿和裁剪功能。在[[Windows_7|Windows 7]]中使用了[[Ribbon|Ribbon]]界面，并包含有更多功能，如增加消除锯齿以使绘出的图形边界平滑的选项。

== 軟體特色 ==
[[File:Painttoolboxvista.png|thumb]]中，小畫家的工具箱。]]
小畫家的工具箱包含了以下的繪圖選項：
* 選擇任意範圍
* 選擇（矩形）
* [[橡皮擦|橡皮擦]]／彩色橡皮擦
* 填入[[顏色|顏色]]
* 挑選顏色
* [[放大鏡|放大鏡]]
* [[鉛筆|鉛筆]]
* [[粉刷|粉刷]]
* [[噴槍|噴槍]]
* [[文字|文字]]
* [[直線|直線]]
* [[曲線|曲線]]
* [[矩形|矩形]]
* [[多邊形|多邊形]]
* [[橢圓形|橢圓形]]
* 圓角矩形

在Windows 7中包含有更多的自选图形。按小畫家「說明」功能表的「說明主題」，可以查詢其使用方法。

「影像」功能表有以下功能：翻轉/旋轉、延展/扭曲、色彩對換、屬性、消除影像和不透明處理。「色彩」功能表容許使用者去編輯顏色（它是該功能表唯一的選項）。編輯顏色的對話方塊，顯示出一個48色的基本色彩色板，另外有一個12色的自訂色彩色板。選擇「定義自訂色彩」後，就會出現一個正方形的顏色盤。使用者可以利用該十字符號（+），選擇顏色。亦可以輸入色調/濃度/亮度數值，或者紅/綠/藍數值。

色盤的預設顏色有[[黑色|黑色]]、[[白色|白色]]、[[灰色|灰色]]、[[銀色|銀色]]、[[褐紫紅色|褐紫紅色]]、[[紅色|紅色]]、[[橄欖色|橄欖色]]、[[黃色|黃色]]、[[深綠色|深綠色]]、[[綠色|綠色]]、[[茶色|茶色]]、[[青綠色|青綠色]]、[[海军蓝|海軍藍色]]、[[藍色|藍色]]、[[紫色|紫色]]、[[品紅色|品紅色]]、[[古金色|古金色]]、[[檸檬黃色|檸檬黃色]]、[[青灰色|青灰色]]、[[暗綠色|暗綠色]]、[[深卡莱罗纳蓝|深卡莱罗纳蓝]]、[[鼠灰色|鼠灰色]]、[[黄绿色|黄绿色]]、[[藍綠色|藍綠色]]、[[午夜藍|午夜藍]]、[[長春花色|長春花色]]、[[紫罗兰_(颜色)|紫罗兰]]、[[珊瑚_(颜色)|珊瑚]]、[[褐色|褐色]]和[[南瓜橙|南瓜橙]]。并提供一个调色板。
