《'''大蟒蛇2：血蘭花'''》（{{lang-en|'''''Anacondas: The Hunt for the Blood Orchid'''''}}）2004年上映于美国的[[恐怖片|恐怖片]]和[[惊悚片|惊悚片]]。

== 剧情 ==
一群[[科学家|科学家]]打着寻找神秘植物“血兰”的名号进入尚未开发的[[婆罗洲|婆罗洲]]地区，科学家相信血兰具有一种神奇的功效，可以使人永保青春，永不衰老。[[美国|美国]][[纽约州|纽约州]][[纽约市|纽约市]]的一家公司对血兰的药理功效非常感兴趣，他们希望找到这种植物，然后大赚一笔，于是慷慨解囊，资助了这些缺乏资金的科学家。科学家选择了一个不恰当的时机出发，就是蟒蛇交配和[[雨季|雨季]]，而血兰这种神奇的植物生长在蟒蛇的地盘上，为了掠夺血兰，他们开始和蟒蛇一较高下，决一生死。

== 花絮 ==
1997年，《狂蟒之灾》的第一部[[蟒蛇|蟒蛇]]特技主要是依赖于电子动画学来制作的，而这部电影是用[[计算机图形学|计算机图形学]]（CG）打造特技效果的<ref name="cow" >{{cite web |url = http://ent.163.com/ent_2003/editor/movie/film/040823/040823_303109.html |title = 物竞天择：《狂蟒之灾2：搜寻血兰》 |accessdate = 2004年8月23日 |publisher= 网易|work= |language = zh-cn}}</ref>。

这部电影的色调阴暗沉郁，目的是渲染一种[[热带雨林|热带雨林]]的原始状态的效果，以符合热带雨林的真实性<ref name="cow" />。

== 票房 ==
这部电影在推出第二周就获得12,812,287美元的票房<ref>[http://www.boxofficemojo.com/weekend/chart/?yr=2004&wknd=35&p=.htm boxofficemojojo]</ref>。

它在2905个电影院站点中，平均票房为4410美元，获得了国内总票房收入32,238,923美元的门票销售额<ref name="boxofficemojo.com">[http://www.boxofficemojo.com/movies/?page=main&id=anacondas.htm boxofficemojo]</ref>。

在世界范围内，影片票房收入是70,992,898美元<ref name="boxofficemojo.com" />。
