《'''狂賭之淵'''》（{{lang-ja|賭ケグルイ}}）是由[[河本焰|河本焰]]擔任原作、[[尚村透|尚村透]]作畫的[[賭博|賭博]]題材[[日本漫畫|日本漫畫]]作品，自2014年4月起在《[[月刊GANGAN_JOKER|月刊GANGAN JOKER]]》連載，曾獲得第二屆[[NEXT漫畫大獎|NEXT漫畫大獎]]第3名，在2017年2月單行本在日本銷售量已經突破200萬本以上。除了正傳之外，還有：外傳漫畫《'''狂賭之淵 雙'''》、《'''狂賭之淵 妄'''》，搞笑[[四格漫畫|四格漫畫]]《'''狂賭之淵（假）'''》，官方[[同人_(ACG)|同人]]漫畫《'''狂賭之淵 萬'''》，小說《'''狂賭之淵 悅'''》，設定集《'''狂賭之淵 愛'''》。[[跨媒體製作|跨媒體製作]]方面，同名[[電視動畫|電視動畫]]於2017年夏季檔（7月至9月）播出，並且於2018年由[[Netflix|Netflix]]在全球上線，其續篇《'''狂賭之淵××'''》也於2019年冬季檔（1月至3月）播映，而[[電視劇|電視劇]]真人版則是在2018年冬季檔播映。

== 故事大綱 ==

私立百花王學園是以賭博決定學生階級的貴族學校，賭博由[[學生會|學生會]]管理，學生需向學生會繳納金錢。繳納金額在全校學生中排名倒數一百名的學生，將會淪為「家畜」，不但在校內受到[[歧視|歧視]]與[[霸凌|霸凌]]，其中欠下鉅額債務者還會收到「人生計畫書」，畢業後也被學生會控制，直至還清債務。

蛇喰夢子進入百花王學園後，讓因為敗給早乙女芽亞里而成為家畜的鈴井涼太的學校生活起了變化。夢子在半數以上的同學都是芽亞里暗樁的情況下勝過她，讓芽亞里欠她880萬元，更讓原本是班上領導者的芽亞里地位一落千丈，此外也免除了涼太的家畜身份。之後，夢子以撲克牌擊敗學生會成員皇伊月，因而受學生會注意。另一方面，芽亞里為了翻身而向學生會成員西洞院百合子發起挑戰，卻慘敗並成為家畜。百合子在學生會會長桃喰綺羅莉暗中協助下勝過夢子，讓她欠了3億元並成為家畜。

欠下鉅款的夢子和芽亞里參加了學生會準備的救濟方案「債務整理大集會」，她們合作擊敗了對手，並成為朋友。對夢子的賭技抱有期待的學生會成員生志摩妄因為不希望夢子被綺羅莉盯上而先一步出手，她們用[[齊訥卡片|齊訥卡片]]和[[俄羅斯輪盤|俄羅斯輪盤]]對賭，賭局在夢子的安排下以平局收場，導致學生會的威信降低。身兼學生會成員的偶像夢見弖優芽美以落敗後必須成為偶像為條件向夢子提出賭局，但夢子的手中也握有優芽美的把柄。在優芽美敗給了夢子以後，夢子推測這是學生會成員豆田生的計畫，因此使用家畜僅有一次的正式比賽（公式戰）挑戰權向他提出賭局，兩人以撲克對賭。落敗的豆田生被綺羅莉開除，之後向夢子挑戰的學生會書記五十嵐清華也以落敗作收。

綺羅莉開辦下屆學生會會長選舉，以重整學生會。除了學生，「百喰一族」各分家的代表也參與了選舉，勝者同時可以成為百喰一族的大當家。所有學生和百喰一族分家代表都各有一票，以自己的選票當賭注對賭。

: 早乙女芽亞里的同班同學，曾與芽亞里就讀同一所小學，自中學起即就讀百花王學園，愛慕芽亞里，認定她是自己的王子。

; {{nihongo|戶隱雪見|戸隠 雪見（とがくし ゆきみ）}}
: 時任文藝社社長的三年級女學生，擁有學園最沒人氣的賭場，在將被回收之際，芽亞里贏走了使用權，後來與芽亞里和津津麗共同經營。

==== 學生會 ====
; {{nihongo|聚樂幸子|聚楽 幸子（じゅらく さちこ）}}
: 時任學生會的風紀股長代表，喜歡看到天才淪落成敗者，因而在桃喰綺蘿莉與壬生臣葵的鬥爭中保持中立。把芽亞里當作新的玩具，在芽亞里參與的多次賭博中都有旁觀。

; {{nihongo|壬生臣葵|壬生臣 葵（みぶおみ あおい）}}
: 𡈼生臣家族由平安时代發跡至今已有一千年歴史，在上流階級中擁有強大的發言力。看似平易近人，實則是冷酷的現實主義者，毫不留情地拋棄了失敗者，與未婚妻三春瀧咲良的關係被歸類為有利益的政治婚姻，並且對她沒有愛。時任學生會的執行長，暗地成立善咲會與學生會長對峙。表面上要扭轉不合理的家畜制度，實際上是替他個人謀取更大的權力。赌博能力相当杰出，从来没有输过，聚乐幸子评价他跟桃喰绮罗莉一样善于赌博。

; {{nihongo|三春瀧咲良|三春滝 咲良（みはるたき さくら）}}
: 時任學生會的美化股長代表，同時也是善咲會的評議員。與壬生臣葵有婚約，本人很在意葵，對於葵欲拉攏芽亞里感到不服氣，因而向芽亞里發出賭博挑戰。

==== 善咲會 ====
壬生臣葵創立的祕密組織，以「善於綻放若百花笑」（「咲」為「笑」的古字，日文裡意為「綻放」）為口號，組織結構表面上為合議制，實際上是由壬生臣獨攬大權。

; {{nihongo|久留米来未|久留 米來未}}
: 善咲会的评议员，擅于和他人打交道的辣妹。在「配对遊戲」赌局中認識早乙女芽亚里等人，后來为了观察芽亚里而加入文艺社，试图说服她加入善咲会，同時對她抱有极大的兴趣。

; {{nihongo|下月卖奥理|下月 売奧理}}
: 成绩优异的特待生，班級的图书股長，在𡈼生臣葵招攬芽亞里失敗後，作為代替被招為善咲会评议员。对壬生臣抱有好感，對芽亞里懷有極大敵意。具有照相機記憶之能力，能够把看到的东西以影像形式原封不動记录下来，中學时期長年与五十岚清华争夺年级第一。

; {{nihongo|妪之头直子|嫗之頭 直子}}
: 妪之头姐妹中的姐姐，垒球社成员，兼善咲会的評議員。

; {{nihongo|妪之头傍子|嫗之頭 傍子}}
: 妪之头姐妹中的妹妹，垒球社成员，兼善咲会的評議員。

; {{nihongo|六条惠音留|六條 惠音留}}
: 摄影社社长兼善咲会的評議員，超級喜歡看助手清水露出和迷路小孩子一樣的為難表情。經常指示部員出售女同學出洋相的照片予一些變態，喜欢拍摄女性裸体。以文艺社存亡为筹码向花手毬津津麗和户隐雪见发起“偷拍棒球拳”赌局，敗北後被戶隱雪见拍光女體。

; {{nihongo|伊吹壯太郎|伊吹 壯太郎}}
: 二年級生，劍道社社長兼善咲會評議員，為人謹慎。一開始與上下凪只是普通同班同學的關係，壯太郎因欠下學生會債務成為家畜而被一年級生欺負時，上下凪竟出手幫助他，並且邀請他加入善咲會。後來，因為𡈼生臣葵有着強大的志願去打敗學生會會長桃喰綺羅莉，所以不論正義和罪惡也想要幫助他。上下凪唆使他去打敗早乙女芽亞里，證明他是有用的人。後來，又被上下凪唆使他去打敗三春瀧咲良和早乙女芽亞里，展示他的必要性。若失敗，他要離開文艺社不再搞事，並且要放棄要求早乙女芽亞里加入善咲會。上下凪暗中吐嘈他是個容易使喚的人，以及為最喜歡的葵同學辦事會很高興。他痛恨所有背叛葵同學的人，曾建議讓他放逐和處理妪之头姐妹。

; {{nihongo|上下凪|上下 凪}}
: 二年級生，善咲会的评议员，與伊吹壯太郎搭檔。一開始與壯太郎只是同班同學的關係，壯太郎因欠下學生會債務成為家畜而被一年級生欺負時，竟出手幫助他，並且邀請他加入善咲會。

==== 其他學生 ====
; {{nihongo|爱浦心|愛浦 心}}
: 在早乙女芽亚里转学到百花王学园的第一天，虐待成为家畜的花手毬津津麗。之后，與芽亚里进行“时短大富翁”赌局，惟败北。

; {{nihongo|佐渡魅久良|佐渡 美久良}}

;{{nihongo|琉璃鸟抚子|瑠璃鳥 撫子（るりちょう なでしこ）}}

; {{nihongo|清水|}}
: 摄影社成員兼六條惠音留的助手，六條惠音留超級喜歡看她露出和迷路小孩子一樣的為難表情，惟已對她的女體厭倦。

== 電視動畫 ==
電視版動畫於2017年7月1日在[[TBS系|TBS系]]電視台開始播出，並於2018年在[[Netflix|Netflix]]上映。蛇喰夢子在2015年的單行本宣傳預告中的配音為[[能登麻美子|能登麻美子]]<ref>[https://www.youtube.com/watch?v=1Rvlz9sib_k 2015年12月ＳＥコミックスＰＶ]</ref>，但動畫播出時是由[[早見沙織|早見沙織]]配音。在動畫開始之前，有標示「該節目充滿描寫賭博的行為。（在日本）法律規定是全面禁止賭博的。」「請兒童、青少年自行斟酌觀賞。」等等節目一般常見的警告提醒標語。

2018年1月6日，於{{link-ja|新宿文化中心|新宿文化センター}}的相關特別活動「私立百花王學園文化祭」<ref>{{Cite web|url=http://anime.eiga.com/event/119284/|title=TVアニメ「賭ケグルイ」スペシャルイベント～私立百花王学園文化祭～【1部】|work=アニメハック|date=2017-11-15|accessdate=2018-01-06}}</ref>中，發表決定製作第2期的消息<ref>{{Cite web|url=https://natalie.mu/comic/news/264166|title=もっと賭け狂いましょう！アニメ「賭ケグルイ」第2期の制作決定|work=[[Natalie_(網站)|Comic Natalie]]|date=2018-01-06|accessdate=2018-01-06|language=ja}}</ref>，2018年8月9日發表第二期於2019年春播放，並命名為《狂賭之淵××》。※註：「×」為代表「錯誤」或「隱諱」的「叉」字符號，而非英文字母。
