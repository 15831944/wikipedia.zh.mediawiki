'''藝術中心設計學院'''（{{lang-en|'''ArtCenter College of Design'''}}，2015年後，校方正式將校名中{{lang|en|Art}}與{{lang|en|Center}}之間的空白去除），又簡稱為{{lang|en|'''ArtCenter'''}}或{{lang|en|'''ACCD'''}}，是一所[[非營利|非營利]]的[[私立|私立]]藝術學院，座落於[[美國|美國]][[加州|加州]]的[[帕萨迪纳_(加利福尼亚州)|帕薩迪納]]。

目前藝術中心設計學院有兩個主要校區: 山邊校區（{{lang|en|Hillside Campus}}） 與南校區（{{lang|en|South Campus}}），山邊校區座落於舉辦過數次[[超級盃|超級盃]]的[[玫瑰盃球場|玫瑰盃球場]]後方，南校區則是由三棟不同風格的建築組成，鄰近[[帕萨迪纳_(加利福尼亚州)|帕薩迪納]]市中心。

== 歷史 ==
[[Image:accdp1.jpg|thumb]]

'''藝術中心設計學院'''（ArtCenter College of Design）成立於1930年，原址是位於[[洛杉磯市中心|美國洛杉磯市區]]的'''設計中心學校'''（Art Center School)。創辦人愛德華·A·汀克·亞當斯（Edward A. "Tink" Adams）是一位南加州的廣告人。當年創辦該校的主要想法是藉由在職藝術家與設計師去帶領學生，讓學生能夠在畢業前即做好踏入業界教備。因此，即使在[[經濟大蕭條|經濟大蕭條]]的1930年代，藝術中心的畢業生仍能夠快速的在業界找到工作機會。<ref>{{cite web |url=http://www.artcenter.edu/about/get-to-know-artcenter/history.html |title=ArtCenter Mission}}</ref>

1935年，[[好萊塢|好萊塢]]的人像攝影師弗萊德·阿澈（Fred R. Archer）創建了該校的攝影系所，並由他的好友，以[[優勝美地國家公園|優勝美地國家公園]]黑白系列照聞名的攝影師[[安塞尔·亚当斯|安塞爾·亞當斯]]擔任客座講師。<ref>{{cite web |url=http://www.artcenter.edu/about/get-to-know-artcenter/history.html |title=ArtCenter History}}</ref>

[[第二次世界大戰|第二次世界大戰]]期間，藝術中心與[[加州理工學院|加州理工學院]]共同進行了一項技術性插圖的計畫（Technical Illustration Program）。<ref>{{cite web |url=http://blogs.artcenter.edu/dottedline/tag/world-war-ii/ |title=World War II Archives}}</ref> 1947年，二戰結束後，藝術中心的學生激增，校方為了容納更多學生而將學校搬遷至[[漢考克公園|漢考克公園]]附近，在現今[[洛杉磯郡藝術博物館|洛杉磯郡藝術博物館]] （LACMA）附近。1949年，藝術中心正式開始授與藝術[[學士學位|學士學位]]以及[[碩士|碩士]]學位，並於1955年正式成為[[西部学校和学院协会|西部學校和學院協會]]會員校。

1965年，學校正式將其校名更改為藝術中心設計學院 （Art Center College of Design）。學校也開始大規模的擴辦系所，其中[[攝影|攝影]]系於 1973年落成。 1973年，學校正式搬遷至位於[[帕萨迪纳_(加利福尼亚州)|帕薩迪納市]]山邊的校區（Hillside Campus）。2003年，藝術中心設計學院成為第一所被[[聯合國|聯合國]]公共資訊部門 (Department of Public Information) 授與[[非政府组织|非政府組織]]資格之藝術學校。 2004年，藝術中心的南校區 （South Campus）正式啟用，主要作為大學部[[藝術|純藝術]]與[[插畫|插畫]]以及許多[[研究生|研究生]]創作和公開活動所使用。

== 學術成就 ==
[[File:ACCD1111.jpg|thumb]]

藝術中心的辦學理念獲得了許多業界設計師與藝術家的認可。從創校開始[[安塞尔·亚当斯|安塞爾·亞當斯]]坐鎮攝影系，到知名藝術家[[凱斯·哈林|凱斯·哈林]]在1989年參訪該校時，替該校畫了一幅巨大的壁畫及可知。目前學校是全美藝術與設計學校協會（NASAD）成員，也因為該校校友在產業及社會的影響力，成為第一所藝術學院獲聯合國頒[[非政府組織|非政府組織]]（NGO）頭銜。<ref>{{cite web |url=http://www.artcenter.edu/about/get-to-know-artcenter/artcenter-at-a-glance.html |title=ArtCenter at a Glance}}</ref>

目前藝術中心設計學院大學部共設有11種不同領域的主修，研究所則分成7種不同的領域。大學部方面，與[[加州理工學院|加州理工學院]]和[[西方學院|西方學院]]都有教育協議，學生可以跨校修課。<ref>{{cite web |url=http://www.artcenter.edu/about/get-to-know-art-center/frequently-asked-questions/details/does-artcenter-have-a-relationship-with-caltech.html |title=ArtCenter FAQ}}</ref> 研究所方面，則是與克萊蒙特高等學院(Claremont Graduate University)的管理學院 (Drucker School of Management)設有合作的 MS/MBA 商科碩士學程。<ref>{{cite web |url=http://www.artcenter.edu/about/get-to-know-artcenter/artcenter-at-a-glance.html |title=ArtCenter Postgraduate Studies}}</ref>

== 學校狀況 ==
[[File:ACCD890.jpg|thumb]]

學生組成方面，以2015年入學新生來說，52%為加州本地生，18%為美國外州生，28%則為來自超過45國的國際學生。新生男女比例約為7:3。依據2015年秋季的數據來說，學校老師人數為439人，非講師相關工作人員為263人，大學部學生人數為1,915人（其中有超過一半是轉學生），研究生為218人。<ref>{{cite web |url=http://www.artcenter.edu/about/get-to-know-artcenter/artcenter-at-a-glance.html |title=ArtCenter Enrollment Profile}}</ref> 轉學進入藝術中心學習是很常見的一件事情，因為藝術中心入學要求嚴苛，許多高中應屆畢業生若沒有經過特殊的指導，往往很難達到入學所需要的門檻。因此，許多學生會選擇先去其他學校進修一至三年，準備好作品集後再進入藝術中心設計學院；在藝術中心大學部遇到年紀較長的新生是頗為常見的。<ref>{{cite web |url=https://www.quora.com/Is-the-Art-Center-College-of-Design-a-good-school |title=Quora, Dec 3, 2015}}</ref>

大學部人數最多的系為[[插畫|插畫]]，緊接著為[[平面設計|平面設計]]以及運輸設計 (Transportation)。人數最少的系則為[[互動設計|互動設計]]、[[廣告設計|廣告設計]]，以及純藝術 (Fine Art)，但不論系的大小，系與系之間的跨系修課很常見。<ref>{{cite web |url=http://www.artcenter.edu/about/get-to-know-artcenter/artcenter-at-a-glance.html |title=ArtCenter Undergraduate Enrollment by Major}}</ref>

學校目前授與的學位有藝術學士（Bachelor of Fine Arts）, [[理學學士|理學學士]]（Bachelor of Science），藝術碩士 (Master of Fine Arts)以及理學碩士 (Master of Science)。

== 學校設備 ==
校方在佔地162.66英畝的雙校園中，有213,771.2淨平方英尺是作為精密及最新設計與藝術用儀器的使用地。校內有各種最新的器材供師生使用。從攝影與電影所需的各式器材和空間，到最新的各種3D列印機，石膏製造間，大型裁縫機，雷射切割機，各式材質的大型切割機，以及純藝術學生獨立的創作空間，學校都有完整的設備供師生租借使用。<ref>{{cite web |url=https://www.youtube.com/watch?v=sCSoWsBPn1I |title=美最頂尖藝術學校}}</ref>

此外，學校[[圖書館|圖書館]]（Fogg Library）坐擁近10萬本藝術與設計相關書籍，各式電影DVDs以及數量龐大的線上數據資料庫供在校生使用。<ref>{{cite web |url=http://www.artcenter.edu/about/get-to-know-artcenter/artcenter-at-a-glance.html |title=ArtCenter Library}}</ref>

目前學校的南校區已經完工並開始使用，未來藝術中心計劃擴建學生宿舍來提供給外地學生做使用。洛杉磯以外地區，藝術中心位於[[德國|德國]]以及其他許多國家也都設有專門給學校師生使用的工作室。
