'''藝予群島'''（{{jpn|j=芸予諸島|hg=げいよしょとう|rm=Geiyo shotō|f=y}}）是位於[[日本|日本]][[瀨戶內海|瀨戶內海]]西部、[[廣島縣|廣島縣]]和[[愛媛縣|愛媛縣]]之間的群島。名稱取自兩縣的[[令制國|舊國名]][[安藝國|安藝國]]和[[伊予國|伊予國]]的各一字。

== 地理 ==
包含舊[[備後國|備後國]]的[[廣島縣|廣島縣]]的全部島嶼和[[愛媛縣|愛媛縣]]舊[[越智郡|越智郡]]島嶼部及若干付屬島嶼，由大小數百的[[島|島]]嶼構成。西邊是[[防予諸島|防予諸島]]，東邊是[[笠岡諸島|笠岡諸島]]。

== 歷史 ==
藝予諸島是古來[[京都|京都]]・[[畿內|畿內]]連結[[西日本|西日本]]及[[朝鮮半島|朝鮮半島]]的海上要衝，各地港口發達。安藝國、伊予國的[[一宮|一宮]][[嚴島神社|嚴島神社]]和[[大山祇神社|大山祇神社]]（皆是航海之守護）設在藝予諸島，可知此海域的重要性。中世以[[村上水軍|村上水軍]]為首的[[水軍|水軍]]（[[海賊|海賊]]）大為活躍，砦・居館跡散佈各地。

== 行政區劃 ==
[[愛媛縣|愛媛縣]]側的全部島嶼屬於[[今治市|今治市]]和[[上島町|上島町]]，[[廣島縣|廣島縣]]側的全部島嶼屬於[[江田島市|江田島市]]、[[吳市|吳市]]、[[大崎上島町|大崎上島町]]、[[尾道市|尾道市]]、[[福山市|福山市]]。
