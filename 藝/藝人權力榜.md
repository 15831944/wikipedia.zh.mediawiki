'''藝人權力榜'''（日語：{{lang|ja|'''タレントパワーランキング'''}}）是株式會社Architect所實施統計，[[日本|日本]]國內最大規模基於電視[[藝人|藝人]]知名度、人氣度調查作成的[[排名|排行榜]]。<ref>{{cite news|url=http://www.talentsearch.jp/whats/01.html|title=タレントパワーランキングのここに注目！|work=株式会社アーキテクト|accessdate=2015-01-08|deadurl=yes|archiveurl=https://web.archive.org/web/20141223073649/http://www.talentsearch.jp/whats/01.html|archivedate=2014-12-23}}</ref>

此排行榜目的是為了方便[[電視節目|電視節目]]和[[電視廣告|電視廣告]]選角，調查了在電視上活躍的約1000名藝人或名人的知名度和人氣度。一開始曾稱為「'''日本藝人輿論調查'''」，2010年1月改名為「藝人權力榜」。自2008年開始，{{link-ja|日經BP|日経BP}}旗下雜誌《{{link-ja|日經娛樂！|日経エンタテインメント!}}》就在每年的5月號基於這個調查的數據發表「藝人權力榜特集」。

== 調查方法 ==
每年進行4次調查（2月、5月、8月、11月），在一都三縣（[[東京都|東京都]]、[[神奈川縣|神奈川縣]]、[[埼玉縣|埼玉縣]]、[[千葉縣|千葉縣]]）抽取在當地居住10歳至69歳男女個人為對象進行調查（以下為詳細方法）<ref name="a">{{cite news|url=https://reposen.jp/3705/8/85.html|title=アーキテクト“タレントパワーランキング” ｜最新調査結果ダイジェスト|work=ポセン|accessdate=2015-01-08|deadurl=yes|archiveurl=https://web.archive.org/web/20150321021545/http://reposen.jp/3705/8/85.html|archivedate=2015-03-21}}</ref>。

*分為調査A、調査B、調査C、調査D四個調查小組（每小組調查1100樣本，合共4400樣本）
*性別和年代（10歲至59歲每5歲一屬性，60至69歲單獨一屬性）共22種属性，每種屬性50樣本

以下為調查項目：<ref name="nikkei">{{Cite journal | date=2014-05-02| journal=日経エンタテインメント!| issue=2014年6月号| publisher=Nikkei BP}}</ref>

'''認知率'''：以下問題回答為第1項的比例
# 知道其名字及樣貌
# 知道其名字，但想不起是什麼樣子
# 完全不認識這位藝人

'''關注率'''：以下問題回答為第1、第2項的比例（以認知率問題回答為第1項的人為調查對象）
# 想更多地看到、聽到、了解
# 想看到、聽到、了解
# 不想看到、聽到、了解
# 完全不想看到、聽到、了解

'''藝人權力分數'''：認知率乘以關注率的加權點數（即關注率問題回答第1項加上回答第2項的1/3的合計值）。理論上的最大値為100。

'''藝人權力榜'''：基於藝人權力分數從高到底排列的排行榜。

== 影響 ==
由於《[[日經娛樂！|日經娛樂！]]》每年的「藝人權力榜特集」都會將藝人按橫坐標為認知率、縱坐標為關注率標準在[[坐標圖|坐標圖]]上，並以當旬圈、注目圈、次世代圈、大御所及安全圈等劃分藝人的當前受關注程度和人氣度檔次，因此在中文網絡上，這個排行榜常被戲稱為「'''圈地榜'''」或「'''日經圈地榜'''」。

受此影響，[[中國|中國]]的[[上海任性文化傳播有限公司|上海任性文化傳播有限公司]]也模仿藝人權力榜，在2016年創造了「星盤·中國明星偶像新媒體圈地榜」。<ref>{{Cite news|url = http://news.xhby.net/system/2017/06/09/030690820.shtml |title = 星盘·中国明星偶像新媒体圈地榜2017春榜发布 |newspaper = 新华报业网 |date = 2017-06-09 }}</ref>
