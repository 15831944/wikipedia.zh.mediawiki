'''鷹鵰'''（[[學名|學名]]：''Nisaetus nipalensis''）又名'''-{zh-cn:熊鹰;zh-tw:鷹雕;}-'''、'''赫氏角鷹'''，是一種[[猛禽|猛禽]]，屬於[[角鷹屬|角鷹屬]]。一般生活於山中的常绿森林。该物种的{{le|模式产地|Type (biology)#type_locality}}在[[尼泊尔|尼泊尔]]。<ref name='id1569'/>

== 外觀 ==
鷹鵰是一種偏大的猛禽，一般身長70至72[[公分|公分]]。[[成年|成年]]的鷹鵰的上半身呈[[棕色|棕色]]，有[[白色|白色]]的下半身、[[羽毛|羽毛]]和[[尾巴|尾巴]]。牠們的[[胸部|胸部]]、[[腹部|腹部]]和[[後翅|後翅]]有很明顯的[[條紋|條紋]]。牠們的[[翅膀|翅膀]]很寬，在[[飛行|飛行]]時呈V型，[[未成熟|未成熟]]的鷹鵰通常擁有白色的[[頭|頭]]。生態習性：主要棲息於海拔1,000～3,000公尺左右之原始闊葉林或針闊混合林內，其族群數量極稀少。[[覓食|覓食]]時，大都採伏擊型之狩獵方式，先停於視野遼闊之枝椏上，當獵物出現時，即悄然滑行接近獵物再急速俯衝突擊。

面臨問題：族群稀少，又是養鷹者熱衷的對象，賴以生存之中、高海拔原始森林頻遭砍伐及人為開發壓力，本種的生存岌岌可危。<ref name=":0">{{Cite news|url=https://raptor.org.tw/taiwan-diurnal-raptors/304-2014-08-12-09-14-15.html|title=熊鷹|last=小編B|newspaper=台灣猛禽研究會|accessdate=2017-10-18|language=zh-tw}}</ref><ref>{{Cite news|url=http://e-info.org.tw/node/93955|title=脆弱卻舉足輕重的存在─ ─熊鷹|newspaper=台灣環境資訊協會-環境資訊中心|accessdate=2017-10-18|language=zh-hant}}</ref>

== 生物學 ==
鷹鵰是一種在[[山脈|山脈]][[森林|森林]]中的猛禽，牠們一般會在[[樹|樹]]上建小[[鳥窩|鳥窩]]和每次只生一隻[[蛋|蛋]]。牠們一般會吃[[哺乳動物|哺乳動物]]、[[鳥|鳥]]和[[爬行動物|爬行動物]]做食物。<ref name=":0" />

== 亚种 ==
* '''鹰鵰指名亚种'''（[[学名|学名]]：{{lang|la|''Nisaetus nipalensis nipalensis''}} <small>Hodgson, 1836</small>）。分布於[[巴基斯坦|巴基斯坦]]東北部、[[喜馬拉雅山脈|喜馬拉雅山脈]]、[[中國|中國]][[雲貴高原|雲貴高原]]至[[東南丘陵|東南丘陵]]與[[海南島|海南島]]一帶、[[台灣|台灣]]、[[中南半島|中南半島]]北部山區至[[緬甸|緬甸]]南部的[[德林達依省|德林達依省]]及[[馬來西亞|馬來西亞]]的北部與[[蘭卡威|蘭卡威]]等地區。<ref name='id1571'>{{cite web| url =http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=1571| title =鹰鵰指名亚种| work =《中国动物物种编目数据库》| publisher =中国科学院微生物研究所| author =中国科学院动物研究所| accessdate =2009-04-04| deadurl =yes| archiveurl =https://web.archive.org/web/20160305045659/http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=1571| archivedate =2016-03-05}}</ref>
* '''鹰鵰东方亚种'''（[[学名|学名]]：{{lang|la|''Nisaetus nipalensis orientalis''}} <small>（Temminck & Schlegel, 1844）</small> ）。主要分布於[[日本|日本]]，可能也分布於[[中國東北地區|中國東北]]與[[內蒙古|內蒙古]]、[[朝鮮半島|朝鮮半島]]與[[俄羅斯|俄羅斯]][[遠東聯邦管區|遠東地區]]。该物种的模式产地在日本。<ref name='id1572'>{{cite web| url =http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=1572| title =鹰鵰东方亚种| work =《中国动物物种编目数据库》| publisher =中国科学院微生物研究所| author =中国科学院动物研究所| accessdate =2009-04-04| deadurl =yes| archiveurl =https://web.archive.org/web/20160305094409/http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=1572| archivedate =2016-03-05}}</ref>
* '''鹰鵰印南亚种'''（[[学名|学名]]：{{lang|la|''Nisaetus nipalensis kelaarti''}} <small>（Legge, 1878）</small> ）。分布於[[印度|印度]][[西高止山脈|西高止山脈]]與[[斯里蘭卡|斯里蘭卡]]。

== 文化 ==
* 在台灣原住民[[排灣族|排灣族]]及[[魯凱族|魯凱族]]的部分地區的[[神話故事|神話故事]]中，族人在[[死|死]]後會化為[[百步蛇|百步蛇]]，而百步蛇[[老年|年老]]後會[[羽化|羽化]]為熊鷹；因此傳統上禁止獵捕熊鷹，若意外捕獲須將其[[羽毛|羽毛]]獻給[[貴族以上身分的族人|貴族以上身分的族人]]。<ref name=":0" />
