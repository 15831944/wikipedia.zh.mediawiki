'''鷹記電視台'''（RCTI）是[[印度尼西亚|印度尼西亚]]第一个商业和私营[[電視聯播網|電視聯播網]]，总部设在[[西雅加達行政市|西雅加达]]。这个电视台播出《{{link-en|印尼偶像|Indonesian Idol}}》、《{{link-en|X因素|X Factor Indonesia}}》、《{{link-en|印尼大廚|MasterChef Indonesia}}》、《{{link-en|新星印度尼西亞|Rising Star Indonesia}}》、《{{link-en|印尼之音|The Voice Indonesia}}》、《{{link-en|印尼達人|Indonesia's Got Talent}}》（2018年年底），以及电影、新闻和时事、真人秀和宗教节目。该网络的旗舰新闻节目是Seputar Indonesia，后来在2009年2月扩大到4个版本。

== 背景 ==
印尼政府在1988年11月6日第四個五年建設計劃仍在實行期間開放電視市場，准許民間資本開辦電視台，結束國營電視台——[[印度尼西亞共和國電視台|印度尼西亞共和國電視台]]的壟斷地位。1989年8月24日早上，蘇哈托總統為印尼第一家私營地面電視台主持正式啟播的儀式，晚間節目的啟播儀式則由新聞部長[[哈爾莫科|哈爾莫科]]主持。鷹記電視台起初在大雅加達地區以收費電視台的模式播放節目，至1990年獲發許可證，開始在全國範圍播放節目。

目前鷹記電視台在印尼全國設有48個轉播站，观众人数达到1.8亿。該電視台由[[MNC有限公司|MNC有限公司]]全資擁有，姐妹頻道還包括MNC全資擁有的全球电视台和MNC公司持股75%的私营电视台[[MNCTV|MNCTV]]。

== 传送时间 ==

=== 早期历史 ===

==== 当地电台（1988年11月6日至8月23日） ====
播放时间为每日下午5:30至10:30，在拉马丹，播放国歌和爱国歌曲。

== 节目 ==
旗舰新闻节目是 Seputar Indonesia，它有早间，中午，晚上和深夜版本。RCTI还播出问答娱乐节目，包括从2001年到2006年播放的印尼版的《百万富翁》。卡通节目包括[[哆啦A梦|哆啦A梦]]和[[蜡笔小新|蜡笔小新]]。{{Fact|date=June 2012}}

== 其他部分 ==
RCTI只在一些城市播出马格里布祈祷，包括雅加达和[[万隆|万隆]]。
一些 RCTI频道播出了本地节目。例如，在[[泗水|泗水]]，RCTI播出了Seputar Jatim，而在万隆，RCTI播出了Seputar Jabar，巴邻旁火车站播出了Seputar Sumsel，Batam火车站播出了Seputar Kepri。 它们主要出现在 Seputar Indonesia Pagi 的两个闭合段（05:45-06:15 / 06:30 WIB）。

== 海外广播 ==
RCTI是[[东帝汶|东帝汶]]的一个免费电视频道，但只在迪利、Ermera和包考等其它城市使用。RCTI在马来西亚也可以作为 Johor Bahru 的一个免费频道，在新加坡作为一个使用天线的免费频道。RCTI是印度尼西亚唯一一个在印度尼西亚境外自由广播的免费电视频道。印尼电视剧系列，如 Bunga di Tepi Jalan，Liontin 和 Pernikahan Dini，目前正在Astro Aruna播出。此外，该频道还在[[新加坡电信|新加坡电信]]IPTV上播出，该频道分别于2001年、2005年和2006年播出。
