{{作品名稱|name=''Lost River''|hk=謎河|tw=遺落流域|cn=迷河}}是一部2014年的美國[[奇幻作品|奇幻]][[新黑色電影|新黑色電影]]，由[[賴恩·高斯寧|賴恩·高斯寧]]執導、監製和編劇，這是他第一次執導的電影。本片主演是[[克莉絲汀娜·韓翠克絲|克莉絲汀娜·韓翠克絲]]、[[茜爾莎·羅倫|茜爾莎·羅倫]]、[[伊恩·德·卡斯泰克|伊恩·德·卡斯泰克]]、[[馬特·史密斯|馬特·史密斯]]、[[賓·曼迪臣|賓·曼迪臣]]、[[芭芭拉·斯蒂爾|芭芭拉·斯蒂爾]]和[[伊娃·曼德絲|伊娃·曼德絲]]。主要攝影於2013年5月6日在[[底特律|底特律]]開始<ref>{{Cite web |url = https://www.onlocationvacations.com/2013/05/10/how-to-catch-a-monster-directed-by-ryan-gosling-is-filming-on-location-in-detroit-on-friday/ |title = ‘How To Catch A Monster’, directed by Ryan Gosling, is filming on location in Detroit on Friday |author = Christine |date = 2013-05-10 |publisher = On Location Vacations |language = en}}</ref>。本片在[[第67屆康城影展|2014年康城影展]]的[[一種注目|一種注目]]單元中首映，並於2015年4月10日在美國上映。<ref>{{Cite web |url = https://variety.com/2014/film/news/ryan-goslings-lost-river-will-not-get-theatrical-release-1201389122/ |title = Ryan Gosling's ‘Lost River’ Will Get Day-and-Date VOD Release |author = Dave McNary |date = 2014-12-29 |publisher = Variety |language = en}}</ref><ref>{{Cite web |url = http://www.slashfilm.com/lost-river-trailer-ryan-gosling/ |title = ‘Lost River’ Trailer: Ryan Gosling Makes His Directorial Debut |author = Angie Han |date = 2015-02-03 |publisher = Slashfilm.com |language = en}}</ref>
