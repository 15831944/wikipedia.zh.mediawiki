《'''昴宿七星'''》（{{lang|ja|七星のスバル}}）是{{link-ja|田尾典丈}}所作的[[日本|日本]][[輕小說|輕小說]]作品，[[插圖|插圖]]由{{lang|ja|ぶーた}}繪畫，由[[小學館|小學館]]出版。2017年9月宣佈改編為電視動畫作品，將於2018年7月5日起[[TBS|TBS]]電視台開始播放<ref>{{cite web|script-title=ja:幼馴染×青春バトルオンライン「七星のスバル」キービジュアル＆放送日時が公開|url=https://animeanime.jp/article/2018/04/18/37439.html|website=Anime Anime|accessdate=2018年4月21日|language=ja|date=2018年4月18日}}</ref>。

== 故事簡介 ==
曾經紅遍全世界的人氣MMORPG《UNION》中，有一支隊名叫做昴宿的傳說級隊伍。這支隊伍是由一群互為兒時玩伴的小學生們所組成，他們憑藉著各自的才能，登上這款線上遊戲的巔峰——不過以某起死亡意外為開端，《UNION》宣布結束營運，這群兒時玩伴們也各奔東西……六年後，身為高中生的陽翔，登入新生《ReUNION》之後，竟與一名少女再次重逢。她就是昴宿的成員之一，應當於六年前「過世」的兒時玩伴──旭姬。她究竟是電子組成的幽靈，還是……?由現實與電玩遊戲交織而成，嶄新的青春故事正式上線！

== 登場角色 ==
===主要人物===
; {{nihongo|天羽陽翔|天羽 陽翔（あもう はると）|extra=聲：[[高梨謙吾|高梨謙吾]]、[[飯田友子|飯田友子]]（兒童）}}
: 本作主角，「昴宿」團長。不管在哪都是積極往前的中心人物。星象為鬥氣
: 旭姬死後渾渾噩噩度過了空虛的6年。6年後重新登入帳號，打完Boss後在寶箱中發現空閑旭姫。

; {{nihongo|空閑旭姬|空閑 旭姫（くが あさひ）|extra=聲：[[大森日雅|大森日雅]]}}
: 「昴宿」成員，非常開朗的少女，卻在遊戲中死亡的同時現實也因為突發性心力衰竭死亡。星象是心奏，還有類似心奏的能力'''先知'''。
: 6年後被天羽陽翔在寶箱中發現，且不知道自己在6年前已死亡的事實。

; {{nihongo|碓冰咲月|碓氷 咲月（うすい さつき）|extra=聲：[[鬼頭明里|鬼頭明里]]}}
: 「昴宿」成員，曾是大剌剌的少女。星象為魔導
:對陽翔抱有好感 ，常常關心得了心病的陽翔，在遊戲舉辦的舞會強硬要求陽翔要邀請自己，並在舞會當天被陽翔拒絕了，也在此認輸對陽翔的感情，承認陽翔與旭姬是一對的。

; {{nihongo|御門貴法|御門 貴法（みかど たかのり）|extra=聲：[[石川界人|石川界人]]、[[小市眞琴|小市眞琴]]（兒童）}}
: 總是很冷靜的少年。對旭姬抱有好感，與陽翔在現實世界的打鬥中，在此認輸對旭姬的感情，承認陽翔與旭姬是一對的，在事件發生後開始遷怒憎恨陽翔。星象為天理
: 6年後在遊戲中成立了光明會，並且是光明會的領導者，曾經不惜以骯髒手段陷害陽翔奪回旭姬。與「昴宿」團的團員們一起打倒真知會的兩位成員並從中解救希後，最終與希在一起。

; {{nihongo|日下希|日下 希（くさか のぞみ）|extra=聲：[[花守由美里|花守由美里]]}}
: 畏畏縮縮的少女，對貴法抱有好感。星象為夢境。
:6年之後的現在卻成對了雜誌的讀者模特兒，為了見到很久沒見的貴法與夥伴們而登入遊戲，意外在遊戲中的舞會看到貴法與咲月跳舞，誤以為兩人在一起，而難過的走到某一處。被真知會的成員第四者和第五者劫走和變成他們的一夥，在貴法的努力下終於成功使希變回原來的樣子，被真知會的的人擄走並被控制，當「昴宿」團的團員們一起打倒真知會的兩位成員後，被貴法解救，最終與貴法在一起。

; {{nihongo|克萊布·維巴里|クライヴ＝ヴィヴァリー|romaji=Clive Beverly|extra=聲：[[寺島拓篤|寺島拓篤]]}}
: 青梅竹馬中唯一從國外登入的人，個性自由奔放。星象為變幻。

; {{nihongo|艾莉西亞|エリシア|romaji=Elicia|extra=聲：[[安濟知佳|安濟知佳]]}}
: 阳翔重新登录游戏后遇到的神秘少女，是時間改變者，從旭姫的記憶碎片裡看到艾莉西亞曾拯救過旭姫。

== 遊戲公會 ==
;{{nihongo|[[昴宿星團|昴星團]]|スバル}}
:曾經紅遍全世界的人氣MMORPG《UNION》中的傳說級團隊，但自從空閑旭姬的意外死亡後，昴星團也就跟著沒落了，團員夥伴們各奔東西去，就連《UNION》也因為這原因，而宣布結束營運。

;{{nihongo|[[光照派|光明會]]|イルミナティ}}
:遊戲中的高級公會之一，由御門貴法親自率領的公會。

;{{nihongo|[[南十字座|南十字星座]]|サザンクロス|Southern Cross}}
:遊戲中的高級公會之一，領導者為雷鬥斯。與維利會聯盟且合併為新的公會。

;{{nihongo|維利會|ブリルソサエティ}}
:遊戲中的高級公會之一，領導者為安傑洛斯，與南十字星座聯盟且合併為新的公會。

;{{nihongo|天賜|ディバイン|Divine}}
:由南十字星座和維利會兩個公會聯盟合併為一體所打造出的公會。

;{{nihongo|[[諾斯底主義|真知會（諾斯底）]]|グノーシス}}
:還在MMORPG《UNION》時期是一流的公會，團員向來都保持得非常神秘又可疑，再轉換成新的MMORPG《ReUNION》之後，所有的公會成員都被抹除了，所有的紀錄都消失的無影無蹤。

;{{nihongo|弗安德斯|そしてヒュアデス}}
:遊戲中的更高級的工會之一，傳說他們能直接與營運方取得聯繫。
