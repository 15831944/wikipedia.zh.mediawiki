'''崙頂里'''是[[中華民國|中華民國]][[臺灣|臺灣]][[嘉義縣|嘉義縣]][[太保市|太保市]]轄下的行政區。位於嘉義縣西南方，面積約1.979617平方公里，含7個鄰、329戶、人口數947人。家己人公園、藝術小湖為該村景點。

== 地理 ==
先民開墾時，這裡是四邊低而中間高的小土丘，居民在中央建屋，外人看來像是在崙仔上而得名。<ref name=":0">{{Cite web|url=http://taibao.cyhg.gov.tw/cp.aspx?n=68AF3B64ECE864A9&s=9BB844180349DC88|title=里辦公室|accessdate=2017-12-17|author=|date=|publisher=}}</ref>

本村有家己人公園、安東國小、藝術小湖、高鐵站。

== 聚落歷史 ==
崙頂里原本隸屬於[[嘉義廳|嘉義廳]]朴子腳支所大糠鄉東下堡太保庄崙子頂區；[[明朝|明朝]]永曆年間由漳州府詔安縣或附近鄉鎮先民登陸後，沿朴子溪之東石、樸子腳、[[六腳鄉|六腳佃]]漸漸移居到目前崙仔頂；初期是一片荒蕪沙丘，而且本庄又為半月形沙丘之頂，村里名稱因而定之<ref name=":0" />。

== 教育 ==
本區域內目有一安東國小，國小學區為安東國小；國中學區為太保國中。

安東國小創立於1960年(民國四十九年)，學區含括崙頂、安仁、東勢。由於學校已位於嘉朴公路邊的崙頂，故以安仁、東勢命名為安東國民小學。

太保國中在1968年(民國57年)建立，。太保國中在嘉義縣政府附近，也離高鐵嘉義站不遠，開車只需3分鐘<ref>{{Cite web|url=http://edu.cyhg.gov.tw/News4.aspx?n=8B04CBD4471A6D45&sms=745127AC0E5C0036|title=嘉義縣國民小學學區劃分|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。

== 景點 ==
藝術小湖：原本是一個髒亂的區域，但在經過藝術家及里民的合作製作裝置藝術下成為一個壁畫彩繪的景點<ref>{{Cite web|url=http://news.ltn.com.tw/news/local/paper/250224|title=〈中部〉真好命！ 太保市崙頂里坐擁藝術小湖|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。
