'''帕哈里语'''、'''帕哈尔语'''或'''帕哈迪语'''（{{lang|hi|पहाड़ी}} {{lang|inc|pahāṛī}}）曾被用以称呼多种不同的方言、语言或语支，它们大多分布于[[喜马拉雅山脉|下喜马拉雅山脉]]地区。

在最常见的用法中，它可能指：
* {{le|帕哈里-波特瓦里语|Pahari-Pothwari}}：[[巴控克什米尔|巴控克什米尔]]地区和邻近[[旁遮普|旁遮普]]地区、[[印控克什米尔|印控克什米尔]]的主要语言
* 任何属于{{le|西帕哈里语支|Western Pahari}}（又称喜马偕尔语支）的单一语言，主要分布在印度的[[喜马偕尔邦|喜马偕尔邦]]<ref name="Masica1991">{{Cite book| last = Masica| first = Colin P.| title = The Indo-Aryan languages| series = Cambridge language surveys| date = 1991| publisher = Cambridge University Press| isbn = 978-0-521-23420-7| page = 439| ref = harv}}</ref>
* {{le|北印度-雅利安语支|Northern Indo-Aryan languages}}：一个有争议的[[印度-雅利安语支|印度-雅利安语支]]的亚语支，下属语言主要分布在[[尼泊尔|尼泊尔]]和印度的[[北阿坎德邦|北阿坎德邦]]和喜马偕尔邦，在语言学中经常被称为“帕哈里语”或“帕哈里语支”

在不甚常见的用法中，它可能指：
* [[多格拉语|多格拉语]]：一些平原地区的多格拉语使用者对高海拔地区多格拉语变体的称呼<ref>{{cite report|last1 = Brightbill| first1 = Jeremy D.| last2= Turner| first2 = Scott D.| year = 2007 | title = A sociolinguistic survey of the Dogri language, Jammu and Kashmir| series = SIL Electronic Survey Reports| url = https://www.sil.org/resources/publications/entry/9015| page = 7}}</ref>
* {{le|比拉斯普尔语|Bilaspuri}}的一种变体，分布于印度[[旁遮普邦|旁遮普邦]]的某些丘陵地区<ref name="Masica1991"/>
* 部分尼泊尔农村地区对[[尼泊尔语|尼泊尔语]]的称呼<ref>{{Cite book| last = Riccardi| first = Theodore| editors = George Cardona, Dhanesh Jain (eds.)| date = 2003| title = The Indo-Aryan languages| chapter = Nepali| series = Routledge language family series. Y| publisher = Routledge| location = London| page = 540| isbn = 978-0-7007-1130-7| ref = harv |url=https://books.google.co.uk/books?id=C9MPCd6mO6sC}}</ref>
* {{le|比尔语|Bhili language}}的方言之一，分布于[[古吉拉特邦|古吉拉特邦]]东部<ref name="Masica1991"/>

另外一个被译为帕哈里（{{lang|ne|पहरी}}）的单词常用于称呼尼泊尔境内的[[藏缅语族|藏缅语族]]语言[[尼瓦尔语|尼瓦尔语]]<ref>{{Cite book| last = van Driem| first = George| date = 2001| title = Languages of the Himalayas| series = Handbuch der Orientalistik| volume = 1| publisher = Brill| location = Leiden| isbn = 978-90-04-10390-0| pp = 758, 1098|ref = harv}}</ref><ref>{{Cite book| last = Shreshtha| first = Rudra Laxmi| year = 2003| chapter = Verbal morphology of the Badikhel Pahari dialect of Newar| title = Themes in Himalayan languages and linguistics| editor-last1 = Kansakar| editor-first1 = Tej Ratna|  editor-last2 = Turin| editor-first2 = Mark| publisher = South Asia Institute ; Tribhuvan University| location = Heidelberg ; Kathmandu| isbn = 978-99933-54-16-1| pages = 145–62| ref = harv}}</ref>。
