'''帕爾馬1913足球會'''（{{lang-it|Parma Calcio 1913}}，前稱為{{lang|it|Parma Associazione Calcio}}、{{lang|it|Parma Football Club s.p.a.}}，可简称{{lang|it|AC Parma}}及{{lang|it|Parma FC}}），简称'''帕爾馬足球會'''，是意大利的一家足球俱樂部。主场是坐落在意大利[[帕爾馬_(意大利)|帕尔玛]]市东南的[[塔爾迪尼球场|塔爾迪尼球场]]。该队在1990年代升入甲级联赛，联赛最好成绩是第二名。球隊在欧洲赛场上的最好成绩是 1992–93 賽季的[[歐洲優勝者杯|歐洲優勝者杯]]冠軍和两届[[欧洲联盟杯|欧洲联盟杯]]冠军（1994–95 賽季和 1998–99 賽季）。

== 歷史 ==
帕爾馬足球俱樂部成立於1913年7月，原名威爾第足球俱樂部（Verdi Football Club）。1969年夏天，該隊與另一足球隊帕爾門塞隊（A.C. Parmense）合併為帕爾馬足球俱樂部（Parma A.C.）。球隊長期徘徊於[[意大利足球乙級聯賽|意大利足球乙級聯賽]]（Serie B）和丙級聯賽（Serie C）。1989至1990賽季，帕爾馬在乙級聯賽中表現不俗，最終於1990年5月27日以二比零的比分在主場擊敗[[雷吉纳足球俱乐部|里賈納]]（Reggina），升入甲級聯賽。在這一時期，當地的奶類製品行業巨頭──[[帕瑪拉特集團|帕瑪拉特集團]]买下了球隊45％股份，為俱樂部主要贊助商。

九十年代屬於帕爾馬的黃金年代，由于球队老板[[斯蒂法诺·坦济|斯蒂法诺·坦济]](Stefano·Tanzi)对球队的大力支持和热爱，球队在甲組的成績一直穩步上揚，最典型的就是 1994–95 赛季与尤文图斯的三线决赛。并出產了不少著名球星，較著名的例子有法國的[[利利安·图拉姆|杜林]]、意大利的[[卡纳瓦罗|簡拿華路]]、[[布冯|保方]]、[[基耶萨|基艾沙]]、[[佐拉|蘇拿]]，阿根廷的[[胡安·塞巴斯蒂安·贝隆|華朗]]、[[埃尔南·克雷斯波|基斯普]]，瑞典的{{le|布連|Tomas Brolin}}，哥倫比亞的[[阿斯普里拉|艾斯派拿]]，巴西的[[阿莫罗索|阿莫罗索]]等，著名教練有[[斯卡拉|斯卡拉]]（Scala）、[[安切洛蒂|安切洛蒂]]（Ancelotti）等。1993年獲得[[歐洲優勝者盃|歐洲優勝者盃]]，1995年，1999年帕爾馬兩度贏得[[歐洲聯盟杯|歐洲聯盟杯]]，乃球會史上最佳的成績。

自2003年该队的原赞助商帕玛拉特（Parmalat）集团因涉嫌金融违规而破产后，老板坦济锒铛入狱，球队成绩也大不如以往，甚至有降级的危险。但在02-03、03-04赛季，[[切薩雷·普蘭德利|普兰德利]]（Prandelli）治下打出了意甲青年近卫军的风采，[[阿德里安·穆圖|穆图]]和[[阿德里亚诺|阿德里亚诺]]成为了球队史上最高产锋线搭档。2004-2005赛季通过附加赛击败[[博洛尼亚足球俱乐部|博罗尼亚]]（Bologna）而保住甲级球队资格。同时当年也打入了[[欧洲联盟杯|欧洲联盟杯]]4强，在被[[莫斯科中央陆军队|莫斯科中央陆军队]](Moskow CSKA)淘汰后，结束了连续15年有资格参加欧洲赛事的历史。2005年，曾任[[皇家馬德里|皇家馬德里]]會長的[[西班牙|西班牙]]老板[[洛伦佐·桑斯|洛伦佐·桑斯]]（Lorenzo Sanz）打算收購球隊，但在最後放棄了出價。此後球隊一直由政府托管，直到2007年1月24日，31岁的企業家吉拉爾迪（Tommaso Ghirardi）收購了球隊，結束了3年的政府托管，并成爲了俱樂部主席，同年他請來了著名教練[[拉涅利|拉涅利]]（Claudio Ranieri）中途接手了瀕臨降級的球隊，并成功保級。

2007年夏，拉涅利離開球隊成為[[尤文圖斯|尤文圖斯]]（Juventus）隊主教練，而球隊則迎來了長期在意乙執教戰績出色的迪卡洛（Domenico Di Carlo）。2008年3月10日，在主場負於[[桑普多利亞|桑普多利亞]]（Sampdoria）后，前[[國際米蘭|國際米蘭]]主教練[[庫珀|庫珀]]（Hector Cuper）接替了迪卡洛成爲了當時意甲唯一一名外教，而他的目標當然是在當年聯賽中保級。

2008年5月18日，帕尔玛主场0:2负于国际米兰，历史上第一次从意甲降级，同时国际米兰夺冠。而在赛前，库珀也被解职。2009年5月17日，帕尔玛客场2:2战平[[施达迪那|施达迪那]](Cittadella)，提前两轮宣布重回意甲。

球會於2015年2月20日破產，并因此从甲级联赛被降入[[意大利足球丁級聯賽|意大利足球丁級聯賽]]。然而在接下來的兩年時間中,帕爾馬連續晉級，從丁級聯賽一路升回了乙級聯賽。在2018年5月18日，帕爾馬在第三個賽季完成了他們的第三次晉級，重返意大利甲級聯賽。他們也是第一支做到三年連續晉級的意大利足球俱樂部。

== 球會榮譽 ==
* [[意甲|意甲]]亞军（2次）：1994–95，1996–97
* [[意大利盃|意大利盃]]冠军（3次）：1991–92，1998–99，2001-02；亞軍（2次）：1994–95，2000–01
* [[意大利超級盃|意大利超級盃]]冠军（1次）：1999年；亞軍（3次）：1992，1995，2002
* [[歐洲聯盟盃|歐洲聯盟盃]]冠军（2次）：1994–95，1998–99
* [[歐洲優勝者盃|歐洲優勝者盃]]冠军（1次）：1992–93；亞軍（1次）：1993–94
* [[欧洲超级杯|欧洲超级杯]]冠军（1次）：1993年
