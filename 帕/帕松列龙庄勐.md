'''帕松列龙庄勐'''{{BD|1960年|6月1日|||catIdx= Long龙}}生于[[云南省|云南省]][[西双版纳|西双版纳]][[勐海县|勐海县]]，[[傣族|傣族]]，[[中国佛教协会|中国佛教协会]]副会长。出家前叫'''岩仔龙庄'''（巖仔、巖里意为生人），初出家时叫'''帕龙庄'''（帕意为沙弥<ref name=huba>[http://www.chinabuddhism.com.cn/xw/jl/2016-10-09/11656.html 郑筱筠：帕松列和帕祜巴升座的重大意义兼论中国南传佛教僧阶制度，中国佛教协会官网]</ref>），职位上升之后叫'''都龙庄'''（都意为比丘<ref name=huba/>），后晋升为“祜巴”（都统長老<ref name=huba/>），尊称为'''祜巴龙庄勐'''（勐爲[[西双版纳|西双版纳]]，祜巴勐爲全版纳的祜巴）<ref name="shark ">{{cite web |url = http://fo.ifeng.com/zhuanti/shijiefojiaoluntan2/people/200903/0321_350_54016.shtml |title=祜巴龙庄勐长老：中国佛教协会副会长 |publisher= 凤凰网 |accessdate=2009年3月21日 |work= |language =}}</ref><ref name="calf " >{{cite web |url=http://www.zhfgwh.com/gaoseng/xdqita/4902.html |title=祜巴龙庄勐长老 |publisher=中华佛光文化网 |accessdate=2010-10-12 |work= |language = }}</ref>。2016年晋升为'''帕松列龙庄勐'''。<ref>{{cite|url=http://bodhi.takungpao.com.hk/sspt/sramana/2015-03/2940079.html|title=中佛協副會長祜巴龍莊勐：中國南傳佛教的唯一祜巴勐|publisher=大公佛教}}</ref>

==生平==
1960年，龙庄出生于[[云南省|云南省]][[西双版纳傣族自治州|西双版纳傣族自治州]][[勐海县|勐海县]]。1973年，13岁的龙庄远赴[[缅甸|缅甸]]勐养龙旺坝岗佛寺剃发受[[沙弥戒|沙弥戒]]。1980年，20岁的龙庄在缅甸景栋旺贺弘佛寺受[[比丘戒|比丘戒]]。1993年，龙庄担任[[西双版纳总佛寺|西双版纳总佛寺]]住持<ref name="shark " /><ref name="calf " />。2004年1月28日至31日，西双版纳总佛寺举行都龙庄晋升“西双版纳祜巴勐”庆典仪式，龙庄勐晋升为“祜巴”，被尊称为“祜巴龙庄勐”<ref>桑吉扎西，西双版纳总佛寺隆重举行都龙庄晋升“西双版纳祜巴勐”庆典仪式，法音2004 (2):35-37</ref>。

2016年2月17日至19日，由[[云南省佛教协会|云南省佛教协会]]、[[中国社会科学院世界宗教研究所|中国社会科学院世界宗教研究所]]主办，西双版纳傣族自治州佛教协会、西双版纳总佛寺承办的首届南传佛教高峰论坛暨帕祜巴和帕松列升座庆典法会在西双版纳傣族自治州[[景洪市|景洪市]]西双版纳总佛寺举行。其中2月19日举行了升座庆典法会。此次升座的高僧共六位：祜巴龙庄勐升为帕松列、西滴看堂升为帕祜巴、玛哈香升为帕祜巴、都香达升为帕祜巴、都炳升为帕祜巴、都罕听升为帕祜巴<ref>{{Cite web |url = http://www.fjnet.com/jjdt/jjdtnr/201602/t20160218_238608.htm |title =西双版纳总佛寺举行中国南传佛教帕松列和帕祜巴升座庆典法会  |author =  |date = 2016-02-18 |publisher =  佛教在线|language =  |accessdate =  |quote =  }}</ref>。此次升座庆典是继1956年之后规模最大的一次。[[中国佛教协会|中国佛教协会]]副会长、云南省佛教协会副会长、西双版纳总佛寺住持祜巴龙庄勐由此晋升为帕松列龙庄勐<ref>孙云霞、梁晓芬，荷如来慧命 弘南传佛教 记新升座的帕松列龙庄勐，中国宗教2016年04期</ref>。

帕松列龙庄勐任中国佛教协会副会长、云南省佛教协会副会长、西双版纳傣族自治州佛教协会会长、西双版纳总佛寺住持、[[云南佛学院|云南佛学院]]西双版纳分院院长<ref>{{Cite web |url = http://www.pusa123.com/pusa/news/fo/2017111342.shtml |title = 帕松列龙庄勐携四众弟子与儿童共庆六一 |author =  |date =2017-06-01  |publisher = 菩萨在线 |language =  |accessdate =  |quote =  }}</ref><ref name=shsta>{{Cite web |url =  http://www.chinabuddhism.com.cn/xw/jj/2017-03-08/12629.html|title = 帕松列龙庄勐提案《加强中国南传佛教建设维护民族团结促进边疆稳定》 |author =  |date =2017-03-08  |publisher = 中国佛教协会 |language =  |accessdate =  |quote =  }}</ref>。他还是西双版纳傣族自治州政协副主席<ref>{{Cite web |url =http://www.bndaily.com/c/2016-09-07/44405.shtml  |title = 帕松列龙庄勐为戒毒学员佛法开示 |author =  |date =  2016-09-07|publisher =西双版纳新闻网   |language =  |accessdate =  |quote =  }}</ref>，第十一、十二、十三届[[全国政协委员|全国政协委员]]<ref name=shsta/><ref>{{Cite web |url = http://www.china.com.cn/guoqing/2017-03/08/content_40427141.htm |title = 帕松列·龙庄勐为“有寺无僧”献策：加强南传佛教僧才培养 |author =  |date =2017-03-08  |publisher =  中国网|language =  |accessdate =  |quote =  }}</ref>。
