'''帕特里斯·盧蒙巴'''（{{lang|fr|Patrice Émery Lumumba}}，{{bd|1925年|7月2日|1961年|1月17日|catIdx=Lumumba, Patrice Émery}}），[[非洲|非洲]][[政治家|政治家]]，[[剛果民主共和國|剛果民主共和國]]的締造者之一。刚果民主共和国（扎伊尔）首任总理（1960）。1925年7月2日生于[[开赛省|开赛省]]桑库鲁县，1961年1月17日或18日被杀害于[[加丹加省|加丹加]]。巴特特拉部族人。

== 早年 ==
盧蒙巴出生於比屬剛果一農家，中學畢業後在郵局工作。1955年当选为非洲雇员联合会东方省分会主席。1958年協助创建了刚果第一个全国性政党-剛果民族運動 （Mouvement National Congolais）。盧蒙巴致力於不同族群的團結，使得剛果民族運動的聲勢日漸壯大。
== 剛果獨立運動 ==
1959年，盧蒙巴成為剛果民族運動的領導人。同年末，当选为全非人民大会常设委员会委员。1959年11月，殖民当局镇压斯坦利维尔群众运动时，以唆使骚乱罪逮捕了卢蒙巴，但迫于舆论压力，不久即释放。1960年1月，卢蒙巴出席了关于刚果独立的[[布鲁塞尔|布鲁塞尔]]圆桌会议，会上，他坚决反对分裂刚果，坚持建立独立、统一、民主国家的原则。6月，剛果举行首次議會選舉，剛果民族運動於下議院的137席取得40席成為最大黨。6月23日，卢蒙巴当选为总理，组织首届政府。6月30日，剛果民主共和國成立。

== 內戰 ==
但随后重要的礦產出產地、南方的加丹加省发生叛乱，叛軍首領[[莫伊兹·冲伯|莫伊兹·冲伯]]邀请比利时军队重新进驻刚果，新生的共和国面临危机。7月間比利時軍隊攻擊首相官邸，占領[[金沙薩|利奥波德维尔]]機場，盧蒙巴與比利時[[斷交|斷交]]，並要求[[聯合國|聯合國]]予以支援。但美国控制的联合国军抵达后，拒绝与合法政府合作以恢复刚果统一。9月14日，刚果国民军参谋长[[蒙博托·塞塞·塞科|蒙博托·塞塞·塞科]]发动政变，联合国军以保护为名软禁了卢蒙巴。

== 遇害 ==
11月27日，卢蒙巴潜离利奥波德维尔，前往东方省，想和政变后迁往[[基桑加尼|斯坦利维尔]]的合法政府会合，但途中被[[蒙博托|蒙博托]]部队绑架，并交予[[莫伊兹·冲伯|冲伯]]。冲伯不顾联合国秘书长[[达格·哈马舍尔德|达格·哈马舍尔德]]等人的呼吁，於1961年1月17日至18日間指使其白人傭兵将卢蒙巴殺害。卢蒙巴遇害后，中华人民共和国政府通过人民日报发表了悲痛的讣告，并断绝与刚果（金）的外交关系，在北京举行了数十万人的游行抗议卢蒙巴的遇害。

== 遺產 ==
1961年2月位于苏联莫斯科的人民友谊大学被正式更名为'''盧蒙巴'''人民友谊大学；该校成立于1960年，专为[[第三世界|第三世界]]国家培训青年共产党员。后于1992年再度更名为[[俄罗斯人民友谊大学|俄罗斯人民友谊大学]]。

1961年3月，第三届全非人民大会宣布他为非洲英雄。

2000年，[[海地|海地]][[導演|導演]][[拉烏爾·佩克|拉烏爾·佩克]]拍攝了一部[[電影|電影]]《{{le|盧蒙巴(電影)|Lumumba (film)|盧蒙巴}}》，由[[艾瑞克·艾伯納尼|艾瑞克·艾伯納尼]]飾演盧蒙巴，描述盧蒙巴領導剛果獨立運動到遇害的過程，全片在[[莫三比克|莫三比克]]和[[辛巴威|辛巴威]]拍攝，2000年在[[第53屆坎城電影節|坎城影展]]首映<ref>[http://www.imdb.com/title/tt0246765/ 網際網路電影資料庫（IMDb）上《Lumumba》的資料]</ref>。
