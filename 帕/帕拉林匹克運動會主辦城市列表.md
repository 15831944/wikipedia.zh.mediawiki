第一屆帕拉林匹克運動會於1960年在[[意大利|意大利]][[羅馬|羅馬]]舉辦，至今有17個城市成功舉辦15屆[[夏季帕拉林匹克運動會|夏季帕拉林匹克運動會]]及11個城市成功主辦12屆[[冬季帕拉林匹克運動會|冬季帕拉林匹克運動會]]。

由於現時獲得該屆城市[[奧林匹克運動會|奧林匹克運動會]]的主辦權則自動獲得[[帕拉林匹克運動會|帕拉林匹克運動會]]主辦權，因此[[倫敦|倫敦]]及[[里約熱內盧|里約熱內盧]]兩個城市分別已被[[國際奧委會|國際奧委會]]選為[[2012年夏季帕拉林匹克運動會|2012年]]及[[2016年夏季帕拉林匹克運動會|2016年]]夏季帕拉林匹克運動會的主辦城市，而[[索契|索契]]及[[平昌郡|平昌]]被選為[[2014年冬季帕拉林匹克運動會|2014年]]及[[2018年冬季帕拉林匹克運動會|2018年]]冬季帕拉林匹克運動會的主辦城市。國際奧委會於2013年9月選出[[2020年夏季帕拉林匹克運動會|2020年]]夏季帕拉林匹克運動會主辦城市為[[東京|東京]]，并更進一步於2015年7月選出[[2022年冬季帕拉林匹克運動會|2022年]]冬季帕拉林匹克運動會主辦城市為[[北京|北京]]，再之後於2017年9月選出[[2024年夏季帕拉林匹克運動會|2024年]]與[[2028年夏季帕拉林匹克運動會|2028年]]夏季帕拉林匹克運動會主辦城市分別為[[巴黎|巴黎]]與[[洛杉磯|洛杉磯]]。

== 主辦城市的產生 ==
{{main|奧林匹克運動會主辦城市列表#主辦城市產生過程}}
帕拉林匹克運動會源於於1948年在[[英國|英國]]斯托克·曼特維爾舉行的世界輪椅及截肢者運動會，是為[[輪椅|輪椅]]運動員而設的運動會；而在1960年舉行的[[1960年夏季帕拉林匹克運動會|羅馬世界輪椅及截肢者運動會]]被視為第一屆[[帕拉林匹克運動會|帕拉林匹克運動會]]<ref>{{en}}{{cite web | title=Rome 1960 |publisher=國際帕拉林匹克委員會 |accessdate=2010年7月11日 |url=http://www.paralympic.org/Paralympic_Games/Past_Games/Rome_1960/index.html }}</ref>，並和[[奧林匹克運動會|奧林匹克運動會]]在同年、同地點舉辦，但只維持了兩屆；有關的主辦權也是由國際斯托克曼德維爾運動會聯合會和相關的國際單項體育總會決定<ref name="IPC1984" />。[[1968年夏季奧林匹克運動會|1968年奧運]]主辦國[[墨西哥|墨西哥]]政府表示因技術問題而放棄主辦該屆帕運會；後來得到以色列政府同意主辦第三屆[[帕拉林匹克運動會|帕拉林匹克運動會]]<ref>{{en}}{{cite web | title=Tel Aviv 1968 |publisher=國際帕拉林匹克委員會 |accessdate=2010年7月11日 |url=http://www.paralympic.org/Paralympic_Games/Past_Games/Tel_Aviv_1968/index.html }}</ref>；1972年帕運也因為住宿問題而由[[慕尼黑|慕尼黑]]改在[[海德堡|海德堡]]舉行<ref>{{en}}{{cite web | title=Heidelberg 1972 |publisher=國際帕拉林匹克委員會 |accessdate=2010年7月11日 |url=http://www.paralympic.org/Paralympic_Games/Past_Games/Heidelberg_1972/index.html }}</ref>；因此，自1968年起的帕運會都因種種不同原因而在非[[奧林匹克運動會|奧林匹克運動會]]主辦城市進行至1984年帕運。雖然有體育組織希望1984年夏季帕拉林匹克運動會可以在洛杉磯進行，但[[1984年夏季奧林匹克運動會|洛杉磯奧運會組委會]]和美國奧林匹克委員會沒有一個正式的溝通橋樑而最終導致美國國家輪椅田徑協會和國際斯托克曼德維爾運動會聯合會在1980年宣布該屆運動會在分別在[[美國|美國]][[紐約|紐約]]及[[英國|英國]]斯托克·曼德維爾進行<ref name="IPC1984">{{en}}{{cite web | title=Stoke Mandeville & New York 1984 |publisher=國際帕拉林匹克委員會 |accessdate=2010年7月11日 |url=http://www.paralympic.org/Paralympic_Games/Past_Games/Stoke_Mandeville_New_York_1984/index.html }}</ref>。

自1988年[[漢城|漢城]][[帕拉林匹克運動會|帕拉林匹克運動會]]起，每屆帕拉林匹克運動會均會同年在[[奧林匹克運動會|奧林匹克運動會]]的主辦城市進行；2001年6月19日，[[國際帕拉林匹克委員會|國際帕拉林匹克委員會]]和[[國際奧林匹克委員會|國際奧林匹克委員會]]簽署協議，授權由國際奧林匹克委員會決定主辦權誰屬；而在申辦2010年的[[奧林匹克運動會|奧林匹克運動會]]主辦權開始，一旦獲選為[[奧林匹克運動會主辦城市|奧林匹克運動會主辦城市]]必須兼辦帕拉林匹克運動會，即同時獲得奧林匹克運動會及帕拉林匹克運動會兩個運動會之主辦權<ref>{{en}}{{cite web | title=Paralympic Games |publisher=國際帕拉林匹克委員會 |accessdate=2010年6月19日 |url=http://www.paralympic.org/Paralympic_Games/ }}</ref>。代表舉辦奧運會的同時須考量選手設施的完善性。

=== 洲 ===
帕拉林匹克運動會曾在[[歐洲|歐洲]]國家舉辦12次，而其第13及14次將於2012年及2014年舉辦；[[亞洲|亞洲]]舉辦過5次帕拉林匹克運動會，而[[東南亞|東南亞]]、[[南亞|南亞]]及[[中亞|中亞]]地區均未曾舉辦帕拉林匹克運動會；北美洲則舉辦過4次，單是美國已佔四分之三。[[大洋洲|大洋洲]]也因[[澳洲|澳洲]]而舉辦過1次；而由於[[巴西|巴西]]將會主辦2016年夏季帕拉林匹克運動會，因此將會成為[[南美洲|南美洲]]國家第一次主辦帕拉林匹克運動會。另一方面，[[非洲|非洲]]成為[[洲|五大洲]]唯一一個曾未舉辦帕拉林匹克運動會的[[洲|洲]]。
