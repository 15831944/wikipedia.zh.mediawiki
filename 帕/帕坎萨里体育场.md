'''帕坎萨里体育馆'''({{lang-id|Stadion Pakansari}})是一个位于[[印尼|印尼]][[西爪哇省|西爪哇省]]帕坎萨里的{{link-en|多用途体育馆|multi-purpose stadium}}。该体育场主要用于足球比赛，亦是{{le|伯希卡波茂物|Persikabo Bogor}}足球队用于取代旧主场{{link-en|伯希卡波体育场|Persikabo Stadium}}的新主场体育场<ref>{{cite web|title=Mencari Nama Stadion Baru Persikabo|url=http://www.ligaindonesia.co.id/index.php/read/divisi-utama/Mencari-Nama-Stadion-Baru-Persikabo-5220#.UlAkvo0W3UY|publisher=ligaindonesia.co.id|accessdate=5 October 2013|language=id|archive-url=https://web.archive.org/web/20171024183729/http://ligaindonesia.co.id/index.php/read/divisi-utama/Mencari-Nama-Stadion-Baru-Persikabo-5220#.UlAkvo0W3UY|archive-date=2017-10-24|dead-url=yes}}</ref>。该体育馆可以容纳3万名观众<ref>{{cite news|url=http://www.thejakartapost.com/news/2012/04/14/west-java-build-new-sporting-venue.html|title=West Java to build new sporting venue|work=[[雅加达邮报|The Jakarta Post]]|date=14 April 2012|accessdate=15 December 2016}}</ref>。该足球场是[[2018年亚运会|2018年亚运会]]的[[2018年亞洲運動會男子足球比賽|男子足球比赛]]赛场之一<ref>{{cite web|url=https://www.bbc.com/zhongwen/simp/sports-45243499|title=2018亚运会：40个项目的比赛场馆互动指南|publisher=BBC中文网}}</ref>。
