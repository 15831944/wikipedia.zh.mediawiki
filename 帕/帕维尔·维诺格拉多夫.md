'''帕维尔·弗拉基米罗维奇·维诺格拉多夫'''（{{lang-ru|Па́вел Влади́мирович Виногра́дов}}，{{bd|1953年|8月31日|}}）是[[俄罗斯|俄罗斯]]第87位[[宇航员|宇航员]]，世界第360位宇航员。他曾于1997年、2006年和2013年三度飞往[[空间站|空间站]]，一共进行了7次太空行走。

==早年==
1953年出生在[[苏联|苏联]]远东[[马加丹州|马加丹州]][[马加丹|马加丹]]。1970年毕业于[[阿纳德尔|阿纳德尔]]高中。1971年进入[[莫斯科航空学院|莫斯科航空学院]]，1977年毕业于[[机械工程|机械工程]]专业，1980年毕业于通信工程专业，获得[[计算机|计算机]]系统分析学位。从1977年4月起，他在飞机自动化设计系统部门实验室担任工程师，并于1978年5月至1983年8月担任高级工程师，专门从事可回收[[航天器|航天器]]自动交互设计系统的开发，[[空气动力学|空气动力学]]和计算机图形学软件开发。同时，他还在[[俄罗斯科学院生物医学问题研究所|俄罗斯科学院生物医学问题研究所]]参与实验与测试。自1983年8月起，他在[[科罗廖夫能源火箭航天集团|科罗廖夫能源火箭航天集团]]担任工程师和项目负责人<ref name=nasabio>{{cite web |title=Pavel Vladimirovich Vinogradov |url=https://web.archive.org/web/20170414153551/https://www.jsc.nasa.gov/Bios/htmlbios/vinogradov.html|publisher=[[NASA|NASA]] |accessdate=2019-04-18}}</ref>。
==宇航员生涯==
1992年5月成为宇航员候选人。1992年10月至1994年2月参与[[加加林宇航员培训中心|加加林宇航员培训中心]]基础太空飞行训练。在成功通过考验后获得测试宇航员资格。

===联盟TM-26===
协调世界时1997年8月5日15时35分，联盟TM-26发射升空，他作为飞行工程师，与指令长[[阿纳托利·索洛维约夫|阿纳托利·索洛维约夫]]一同前往[[和平号空间站|和平号空间站]]<ref>{{cite book |author=Ben Evans |title=The Twenty-first Century in Space |date=2014 |publisher=Springer |isbn=9781493913077 |page=405 |url=https://books.google.com/books?id=BH7ZBQAAQBAJ&pg=PA405&}}</ref>。在任务期间，他一共进行了5次太空行走。联盟TM-26任务共持续197天17小时34分<ref>{{cite web |title=Pavel Vladimirovich VINOGRADOV |url=https://www.energia.ru/en/iss/iss35/vinogradov.html |publisher=S.P. Korolev RSC "Energia" |accessdate=2019-04-18}}</ref>。

===远征13===
[[File:ISS_expedition_13_crew_with_reiter.jpg|thumb]]
北京时间2006年3月30日上午10时30分，TMA-8载人飞船在哈萨克斯坦境内的拜科努尔发射场成功发射，前往国际空间站<ref>{{cite web |title=俄罗斯联盟TMA-8载人飞船发射成功 |url=http://tech.sina.com.cn/d/2006-06-08/1205980929.shtml |publisher=[[CCTV|CCTV]] |accessdate=2019-04-18 |archiveurl=https://web.archive.org/web/20060704210525/http://tech.sina.com.cn/d/2006-06-08/1205980929.shtml |archivedate=2006-07-24 |date=2006-06-08}}</ref>。12时19分，飞船与国际空间站[[曙光号功能货舱|曙光号功能货舱]]完成了自动对接<ref>{{cite web |author=魏忠杰 |title=俄“联盟”号抵达国际空间站并完成对接 |url=http://scitech.people.com.cn/GB/1057/4260769.html |publisher=新华社 |accessdate=2019-04-18 |archiveurl=https://web.archive.org/web/20070824074632/http://scitech.people.com.cn/GB/1057/4260769.html |archivedate=2007-08-24 |date=2006-04-01}}</ref>，3名宇航员在检查了对接口的密封性能并确认安全后，于13时59分打开曙光号舱门并陆续进入空间站<ref>{{cite web |author=魏忠杰 |title=“联盟TMA-8” 宇航员 进入国际空间站 |url=http://zjrb.zjol.com.cn/html/2006-04/02/content_66977.htm |publisher=[[浙江日报|浙江日报]] |archiveurl=https://web.archive.org/web/20190418083801/http://zjrb.zjol.com.cn/html/2006-04/02/content_66977.htm |archivedate=2019-04-18}}</ref>。

北京时间2006年6月2日6时48分，他和威廉姆斯走出国际空间站，开始太空行走。整个行走过程共持续了6个半小时<ref>{{cite web |title=俄宇航员推迟太空高尔夫表演 |url=http://news.sina.com.cn/w/2006-06-03/02249104380s.shtml |publisher=[[重庆晨报|重庆晨报]] |accessdate=2019-04-18 |archiveurl=https://web.archive.org/web/20060622211613/http://news.sina.com.cn/w/2006-06-03/02249104380s.shtml |archivedate=2006-06-22 |date=2006-06-03}}</ref>。

根据俄航天署与加拿大一[[高尔夫球|高尔夫球]]公司签署的协议，维诺格拉多夫原定于在太空行走时打上一杆高尔夫球。但[[美国国家航空航天局|美国国家航空航天局]]认为高尔夫球被击到太空后，会对轨道上的卫星和国际空间站形成威胁，因此活动被迫取消<ref>{{cite web |title=太空高尔夫表演被迫取消 因会对国际空间站形成威胁 |url=http://news.sina.com.cn/w/2006-06-03/02059104177s.shtml |publisher=新浪新闻 |accessdate=2019-04-18}}</ref>。

协调世界时2006年9月29日1时13分，携带维诺格拉多夫等队员的的联盟号返回舱降落在哈萨克斯坦[[阿尔卡雷克|阿尔卡雷克]]附近<ref>{{cite web |url=http://www.space.com/missionlaunches/060928_exp13_landing.html |title=Ansari, Expedition 13 Crew Return to Earth |author=Ker Than |publisher=SPACE.com |accessdate=2010-06-07}}</ref>。
===远征35/36===
[[协调世界时|协调世界时]]2013年3月28日20时43分，[[联盟-FG|联盟-FG]]运载火箭携[[联盟TMA-08M|联盟TMA-08M]]载人飞船从哈萨克斯坦[[拜科努尔航天发射场|拜科努尔航天发射场]]发射升空，把维诺格拉多夫、[[亚历山大·米苏尔金|米苏尔金]]和[[克里斯托弗·卡西迪|卡西迪]]送往[[国际空间站|国际空间站]]<ref>{{cite web |title=俄“联盟TMA-08M”载人飞船发射升空 |url=http://news.163.com/air/photoview/56NT0001/33180.html?from=tj_xgtj#p=8R7JHU8356NT0001 |publisher=[[网易|网易]]新闻 |accessdate=2019-04-18 |archiveurl=https://web.archive.org/web/20130518140826/http://news.163.com/air/photoview/56NT0001/33180.html?from=tj_xgtj#p=8R7JHU8356NT0001 |archivedate=2013-05-18 |date=2013-03-30}}</ref>。联盟TMA-08M之前的载人飞船均采用的是两昼夜的飞行模式，而这次飞行首次采用仅为6小时的“快速”飞行模式<ref>{{cite web |title=三名宇航员乘飞船赴国际空间站 采用新飞行模式 |url=http://news.sciencenet.cn/htmlnews/2013/3/276192.shtm |publisher=中国新闻网 |accessdate=2019-04-18 |archiveurl=https://web.archive.org/web/20130920155855/http://news.sciencenet.cn/htmlnews/2013/3/276192.shtm |archivedate=2013-09-20 |date=2013-03-29}}</ref>，因此他们是航天史上第一批乘坐“特快载人飞船”前往国际空间站的宇航员。

协调世界时2013年4月19日下午14时03分，他和[[罗曼·罗曼年科|罗曼·罗曼年科]]打开舱门，进行了6小时38分钟的太空行走任务<ref>{{cite web |author=Pete Harding |title=Cosmonauts successfully conclude Russian spacewalk |url=https://www.nasaspaceflight.com/2013/04/cosmonauts-russian-spacewalk-iss-rs32/ |publisher=nasaspaceflight |accessdate=2019-04-18 |date=2013-04-19}}</ref>。

2013年4月22日，他通过[[俄罗斯联邦储蓄银行|俄罗斯联邦储蓄银行]]的在线交易系统缴纳了自己的土地[[税|税]]，成为世界上首位从太空缴税的[[纳税|纳税]]人<ref>{{cite web |title=Российский космонавт впервые оплатил налог из космоса |url=https://web.archive.org/web/20181122171902/https://www.nalog.ru/rn22/news/smi/4287277/ |publisher=[[俄罗斯联邦税务局|ФНС России]] |accessdate=2019-04-18 |language=ru |date=2013-05-16}}</ref><ref>{{cite web |author1=张文春|author2=余彦龙|author3=徐小兵 |title=揭示全球税收征管新趋势 |url=http://www.cfen.com.cn/old_7392/paper/6/201401/t20140107_1748878.html |publisher=[[中国财经报|中国财经报]] |archiveurl=https://web.archive.org/web/20190418071512/http://www.cfen.com.cn/old_7392/paper/6/201401/t20140107_1748878.html |archivedate=2019-04-18 |date=2014-01-07}}</ref>。

2013年9月11日，他和队员乘坐联盟-TMA08M飞船的返回舱安全返回[[地球|地球]]<ref>{{cite web |title=国际空间站三宇航员返回地球 |url=http://news.sina.com.cn/c/2013-09-12/154028200165.shtml |publisher=[[羊城晚报|羊城晚报]] |archiveurl=https://web.archive.org/web/20190418083211/http://news.sina.com.cn/c/2013-09-12/154028200165.shtml |archivedate=2019-04-18 |date=2013-09-12}}</ref>。

==相关活动==
2007年2月，他在俄罗斯《[[劳动报|劳动报]]》发表了以一篇题为《太空困局》的文章，回顾了俄罗斯载人航天史，批评了[[俄罗斯政府|俄罗斯政府]]没有制定面向[[21世纪|21世纪]]的明确发展计划和战略目标，表达了对俄罗斯航天业濒临灭亡的忧虑<ref>{{cite web |title=俄王牌宇航员自白：俄罗斯航天业濒临灭亡 |url=http://news.sohu.com/20070228/n248405499.shtml |publisher=[[新华网|新华网]]|archiveurl=https://web.archive.org/web/20180813162207/http://news.sohu.com/20070228/n248405499.shtml |archivedate=2019-04-18 |date=2007-02-28}}</ref>。

2016年4月10日，他和[[安德烈·鲍里先科|安德烈·鲍里先科]]参加了[[圣彼得堡|圣彼得堡]]纪念[[尤里·加加林|加加林]]太空飞行55周年庆祝活动，与市民们举行了见面会<ref>{{cite web |author1=姚心原 |title=圣彼得堡举行纪念加加林飞行55周年庆祝活动 |url=http://sky.cssn.cn/gj/gj_gjzl/gj_ggzl/201604/t20160412_2962990.shtml |publisher=中国社会科学网 |archiveurl=https://web.archive.org/web/20190418065330/http://sky.cssn.cn/gj/gj_gjzl/gj_ggzl/201604/t20160412_2962990.shtml |archivedate=2019-04-18 |date=2016-04-12}}</ref>。

2018年4月21日至25日，他以俄罗斯联邦宇航员联合会副主席和[[科罗廖夫能源火箭航天集团|科罗廖夫能源火箭航天集团]]航天中心副主任的身份，与中国宇航员[[劉伯明_(航天員)|劉伯明]]出席了在[[哈尔滨工业大学|哈尔滨工业大学]]举办的“2018阿斯图中俄大学生微纳卫星研习营”活动<ref>{{cite web |title=“2018阿斯图中俄大学生微纳卫星研习营”在校举行 |url=http://news2.hit.edu.cn/2018/0425/c421a205521/pagem.htm |publisher=哈尔滨工业大学 |archiveurl=https://web.archive.org/web/20190418063940/http://news2.hit.edu.cn/2018/0425/c421a205521/pagem.htm |archivedate=2019-04-18 |date=2018-04-26}}</ref>。4月24日参与了第二届中国航天科技教育校长论坛<ref>{{cite web |author=霍枭涵 |title=第二届中国航天科技教育校长论坛举办 中俄航天员同台对话莘莘学子 |url=http://www.zgkjjy.cn/index.php?c=show&id=219 |publisher=东北网 |archiveurl=https://web.archive.org/web/20190418063429/http://www.zgkjjy.cn/index.php?c=show&id=219 |archivedate=2019-04-18 |date=2018-04-25}}</ref>。
