'''瑞典学院丑闻'''是[[瑞典学院|瑞典学院]]自2017年起，因[[瑞典學院院士列表|学院院士]]{{link-en|卡塔琳娜·佛洛斯登松|Katarina Frostenson}}的丈夫{{link-en|让-克劳德·阿尔诺|Jean-Claude Arnault|}}被指控性侵而陷入的一系列丑闻。而学院对佛洛斯登松夫妇的“庇护”，引发内部分裂。院士陆续请辞，致使评奖人数缺失，最终，停办2018年[[诺贝尔文学奖|诺贝尔文学奖]]。

==相关指控 ==
=== 对院士夫妇的指控 ===

对{{link-en|让-克劳德·阿尔诺|Jean-Claude Arnault|}}，除了性侵指控，有报道称<ref name="a1"/>，学院内部报告指他自1996年起，曾7次洩漏[[诺贝尔文学奖得主列表|诺贝尔文学奖得主]]消息。

=== 对让-克劳德·阿尔诺的性侵指控 ===

自2017年底起陆续有18名女性通过瑞典报纸《每日新闻》，指控{{link-en|让-克劳德·阿尔诺|Jean-Claude Arnault|}}在1996年至2017年间对多名女性做出了[[性騷擾|性騷擾]]等行為，並宣稱其行为在[[瑞典|瑞典]]文化界广为人知，学院院士、院士妻女、学院员工亦受其害，而[[瑞典学院|瑞典学院]]则助长了他的恶行<ref name="a3"/>。有相关报道称<ref>{{Cite web |url=http://www.bbc.com/news/world-europe-43942106 |title=Photographer 'groped' princess at Swedish Academy event |accessdate=2018-05-01 |author= |date=2018-04-29 |work= |publisher=BBC.com|language=en }}</ref>，他在2006年也对瑞典[[王储|王储]][[維多利亞王儲|维多利亚公主]]进行了性骚扰。而公主的父亲瑞典国王[[卡尔十六世·古斯塔夫|卡尔十六世·古斯塔夫]]本身是瑞典学院的[[赞助人|赞助人]]<ref name="a4"/>。

瑞典检察机关对事件进行初步调查。在2018年3月，[[斯德哥尔摩|斯德哥尔摩]]检察署宣布，因[[证据|证据不足]]和超过[[追溯期|追溯期]]，撤销2013年至2015年对阿尔诺的指控<ref name="a1">{{Cite web |url=http://grinews.com/news/%E8%AB%BE%E8%B2%9D%E7%88%BE%E6%96%87%E5%AD%B8%E7%8D%8E%E6%88%90metoo%E9%87%8D%E7%81%BD%E5%8D%80%EF%BC%81%E6%80%A7%E4%BE%B5%E3%80%81%E6%B4%A9%E5%AF%86%E9%86%9C%E8%81%9E%E5%BB%B6%E7%87%92%E7%91%9E/ |title=諾貝爾文學獎成MeToo重災區！性侵、洩密醜聞延燒瑞典文化圈 |accessdate=2018-05-01 |author=劉俞妗 |date=2018-05-01 |work= |publisher=|language=Zh-hant }}</ref>。4月，《每日新闻》刊登了18位女性的证词<ref name="a4"/>。

== 学院处理 ==

瑞典学院在1996年得知对{{link-en|让-克劳德·阿尔诺|Jean-Claude Arnault|}}的指控<ref name="a2"/>。1997年时，有媒体报道，受害女性向瑞典学院举报其行为。学院未予调查，时任常务秘书的斯图雷·阿伦拒绝向报道此事的媒体证实是否收到举报信。学院知晓他的丑闻后，在历任四位常务秘书的管理下，持续向{{link-en|卡塔琳娜·佛洛斯登松|Katarina Frostenson}}夫妇主办的文化俱乐部——“论坛”提供资金<ref name="a3">{{Cite web |url=http://epaper.gmw.cn/zhdsb/html/2018-04/11/nw.D110000zhdsb_20180411_1-04.htm |title=三大院士同日宣布离席，瑞典学院遭到致命一击|accessdate=2018-05-05 |author=记者：康慨 |date= |work= |publisher=光明网，来源：《中华读书报》2018年04月11日04版|language=Zh-hans }}</ref>。

2017年底，丑闻再度爆发。2018年4月，丑闻升级<ref name="a4"/>。除瑞典检察机关的调查外，学院委托一间律师事务所展开内部调查<ref name="a1"/>，切断与“论坛”的经济联系<ref name="a4"/>。学院院士亦投票决定不驱逐{{link-en|卡塔琳娜·佛洛斯登松|Katarina Frostenson}}。此举引发学院院士的分裂。4月6日，三位[[瑞典學院院士列表|学院院士]]皮特·英格朗、克拉斯·厄斯特格伦和谢尔·艾斯普马克，因此事离席。皮特·英格朗表示“我既不相信也不知道怎样为之辩护的决定已经作出”，“所以我决定不再参与瑞典学院的工作。”克拉斯·厄斯特格伦反对学院处理危机的方式 ，称这无异于对“对学院创办者的背叛”。谢尔·艾斯普马克则说“当学院内部的要人选择将友谊和其他不相干的考虑置于诚实责任之上时，我已不能再继续自己的工作。”<ref name="a3"/>

[[诺贝尔基金会|诺贝尔基金会]]董事会亦批评学院破坏自身信誉，甚至危及[[诺贝尔奖|诺贝尔奖]]的声誉，发声明表示：“瑞典学院的信誉已遭到破坏，董事会要求学院采取具体措施改善，恢复各界的信任。”董事会的要求包括“以可靠方式”来评审本年度的诺贝尔奖、按诺贝尔法规和指引来解决利益、冲突及保密性等问题，及将可疑的犯罪行为移交执法人员<ref>{{Cite web |url=http://www.kwongwah.com.my/?p=499566 |title=瑞典学院卷性侵丑闻 诺贝尔基金会：信誉受损 |accessdate=2018-05-05 |author= |date=2018-04-12 |work= |publisher=光华网|language=Zh-hans }}</ref>。12日，学院常务秘书、院士{{le|萨拉·丹尼尔斯|Sara Danius}}在批评学院处理性侵丑闻的态度后，宣布辞职。她表示“这是学院的愿望，我应该离开这一职务。” “我立即做出了这一决定。”“这起丑闻已经严重影响了诺贝尔奖。<ref name="a2">{{Cite web |url=http://wemedia.ifeng.com/56443076/wemedia.shtml |title=瑞典学院首位女性常务秘书辞职，诺贝尔文学奖遭遇信任危机 |accessdate=2018-05-01 |author= |date=2018-04-14 |work= |publisher=凤凰网，来源：澎湃新闻|language=Zh-hans }}</ref>”

瑞典国王[[卡尔十六世·古斯塔夫|卡尔十六世·古斯塔夫]]以学院赞助人的身份，并介入此次风波。开始外界认为，丑闻将不会影响颁发2018年诺贝尔文学奖<ref name="a4"/>。在4月25日，瑞典学院宣布，可能会暂停颁发2018年诺贝尔文学奖<ref>{{Cite web |url=http://news.ifeng.com/a/20180426/57894718_0.shtml |title=德媒：今年的诺贝尔文学奖可能没了|accessdate=2018-05-01 |author= |date=2018-04-26 |work= |publisher=凤凰网，来源：澎湃新闻网|language=Zh-hans }}</ref>。5月3日，学院做出停办2018年诺贝尔文学奖的决定，并计划在2019年补发奖项。5月4日，瑞典学院对外宣布了这一决定。对此，[[诺贝尔基金会|诺贝尔基金会]]发声明表示：“瑞典学院决定推迟颁发2018年诺贝尔文学奖......在过去的几周里，我们一直在与瑞典学院进行持续沟通，我们支持于本周四（5月3日）所做决定......当前瑞典学院的危机对诺贝尔奖的声誉产生不利影响，基于事态的严重性，并维持[[诺贝尔奖|诺贝尔奖]]长久以来的声誉，做出（不颁发文学奖）的决定。但这不会影响到2018年诺贝尔其他各奖项的颁发。诺贝尔基金会认为，瑞典学院现在必须尽一切努力恢复其作为颁奖机构的公信力，此外学院还需汇报后续具体措施。我们还必须清楚意识到，目前对于组织架构的广泛改革都必须做到公开透明。<ref name="a4">{{Cite web |url=https://www.thepaper.cn/newsDetail_forward_2111295 |title=今年诺贝尔文学奖不发了，明年发两个|accessdate=2018-05-05 |author= |date=2018-05-04 |work= |publisher=澎湃新闻|language=Zh-hans }}</ref>”
