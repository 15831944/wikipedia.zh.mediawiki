'''瑞丽航空'''是一家总部位于[[中国|中国]][[云南省|云南省]]的[[航空公司|航空公司]]，以[[昆明长水国际机场|昆明长水国际机场]]为基地机场。由云南景成集团公司独资，从事国内公共航空运输经营，注册资金10亿元人民币。瑞丽航空于2013年5月经国家民航局批准筹建，计划2014年4月开航，<ref>{{Cite web |url=http://job.carnoc.com/other/ruiliair.html |title=与瑞丽航空一起飞 |publisher=民航资源网 |access-date=2013-10-08 |archive-url=https://web.archive.org/web/20131017025952/http://job.carnoc.com/other/ruiliair.html |archive-date=2013-10-17 |dead-url=yes }}</ref>但最後延至2014年5月18日才首航。<ref>{{cite news|url=http://news.carnoc.com/list/282/282168.html|title=瑞丽航空首航仪式圆满成功 美丽空姐快闪庆祝|work=民航资源网|date=2014-05-18|accessdate=2014-05-20}}</ref>

自2007年“暂停受理新设立航空公司申请”的禁令执行5年后，[[中国民用航空局|中国民用航空局]]对新建航空公司的受理首次敞开大门。瑞丽航空成为近5年来民航局受理并获批筹建的第一家民营航空公司。<ref>{{Cite web |url=http://www.caac.gov.cn/C1/201305/t20130506_55626.html |title=关于拟批准瑞丽航空公司筹建的公示 |publisher=中国民用航空局 |date=2013-05-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20131004062901/http://www.caac.gov.cn/C1/201305/t20130506_55626.html |archivedate=2013-10-04 }}</ref>

2013年8月15日，瑞丽航空与[[美国|美国]][[波音公司|波音公司]]签订了8架[[波音737-700|波音737-700]]型客机、6架[[波音737-800|波音737-800]]型客机的采购协议；与[[德国|德国]][[柏林航空|柏林航空]]签订了2013年购买2架波音737-700型客机、2014年购买2架波音737-800型客机的采购协议。<ref>{{Cite web |url=http://news.carnoc.com/list/259/259209.html |title=瑞丽航空购18架飞机　争取2014年春节通航 |publisher=民航资源网 |date=2013-08-16 }}</ref>
2013年11月瑞丽航空接收来自柏林航空的两架737-700客机，编号为B-5811,B-5812。飞机将停留在上海浦东机场至12月底，完成委托给上海波音的过渡检、瑞丽航空LOGO喷涂等工作。<ref>{{cite news|last=潘 |first=林虎 |url=http://news.163.com/13/1204/06/9F7U69CF00014AEE.html |title=瑞丽航空德国购波音737“落地” 两架飞机暂留上海做喷涂 |publisher=网易 |date=2013-12-04|accessdate=2013-12-06}}</ref>

2014年11月25日，首架直购全新波音737-700型客机降落在昆明长水国际机场。这也成为了瑞丽航空第四架飞机。<ref>http://www.chinanews.com/shipin/2014/12-01/news528238.shtml</ref>

2015年9月23日19时18分，瑞丽航空新引进的波音737-700型飞机降落在昆明长水国际机场，瑞丽航空机队规模由此增至7架。
<ref>{{Cite web |url=http://www.yn.xinhuanet.com/reporter/2015-09/24/c_134654505.htm |title=瑞丽航空引进第七架民航飞机 |publisher=新华网 |date=2015-09-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20151008131133/http://www.yn.xinhuanet.com/reporter/2015-09/24/c_134654505.htm |archivedate=2015-10-08 }}</ref>

2017年9月20日，云南景成集团与美国波音公司在北京签署了8架B737-MAX飞机购机合同，瑞丽航空使用中的机队规模达到14架，而景成集团至此共与美国波音公司签署59架购机合同（瑞丽航空58架）。<ref>{{cite web |url=http://www.cnepaper.com/dhtjb/html/2017-09/28/content_4_5.htm |title=瑞丽航空再签8架波音737-MAX订购合同 |last=张 |first=恒钊 |date=2017-09-28 |website=2017-09-29 |publisher=德宏团结报 |access-date=2017-09-29 |quote=}}</ref>

2017年12月18日，瑞丽航空第15架飞机在昆明长水国际机场交付使用。<ref>{{cite web |url=http://www.cnepaper.com/dhtjb/html/2017-12/20/content_1_5.htm |title=瑞丽航空机队迎来第15架飞机 |author=张恒钊 |date=2017-12-20 |website=www.cnepaper.com |publisher=[[德宏团结报|德宏团结报]] |access-date=2018-01-19 |quote=}}</ref>

2018年1月15日，注册号为B-1598的全新波音737-800客机在[[昆明长水国际机场|昆明长水国际机场]]交付瑞丽航空，是为瑞航第16架飞机。<ref>{{cite web |url=http://www.cnepaper.com/dhtjb/html/2018-01/18/content_4_6.htm |title=瑞丽航空迎来今年首架新飞机 |author=杨奇 |date=2018-01-18 |website=www.cnepaper.com |publisher=[[德宏团结报|德宏团结报]] |access-date=2018-01-19 |quote=}}</ref>
