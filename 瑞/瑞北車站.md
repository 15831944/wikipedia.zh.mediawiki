'''瑞北車站'''位於[[臺灣|臺灣]][[花蓮縣|花蓮縣]][[瑞穗鄉|瑞穗鄉]]，曾為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]（現已廢止）的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*岸式月臺一座。

== 利用狀況 ==
*廢止前為招呼站，由[[瑞穗車站_(台灣)|瑞穗車站]]管理。
*[[富源車站|富源]]＝[[瑞穗車站_(台灣)|瑞穗車站]]間改線後裁撤。

== 車站周邊 ==
;教育單位
*花蓮縣瑞穗鄉瑞北國民小學

;道路
*[[台九線|台九線]]：[[花東公路|花東公路]]、中正北路二段
*瑞北部落聯絡道路
*祥北路一段
*馬遠聯絡道路

;其他
*東光部落
*[[富源溪|富源溪]]
*馬遠部落
*鶴岡溪

== 歷史 ==
*1956年：設站。
*2003年8月23日：廢止。
*現為自行車道服務站，2009年12月13日完成啟用<ref>[http://blog.yam.com/yoyoimage/article/25822326 瑞穗賞櫻單車道啟用]</ref>。
