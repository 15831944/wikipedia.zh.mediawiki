== 歷史上的政黨 ==
* 民主黨 (1867-1971, 併入[[瑞士人民黨|瑞士人民黨]])
* [[農民、商人和獨立派黨|農民、商人和獨立派黨]] (1917-1971, 併入[[瑞士人民黨|瑞士人民黨]])
* 共產反對黨 (1930-1935)
* [[民族陣線_(瑞士)|民族陣線]] / Eidgenössische Sammlung (1933-1943)
* 獨立聯盟 (1936-1999)
* [[民族行動|民族行動]] (1961-1990, 併入[[瑞士民主派|瑞士民主派]])
* [[瑞士進步組織|瑞士進步組織]] (1969-1995)
* 社會主義工人黨 (1969-1990)
* 共和派 (1971-1990, 併入[[瑞士民主派|瑞士民主派]])
