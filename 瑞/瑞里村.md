'''瑞里村'''是[[臺灣|臺灣]][[嘉義縣|嘉義縣]][[梅山鄉|梅山鄉]]轄下的一個[[村里|村]]級行政區，位於[[梅山鄉|梅山鄉]]東南部，北鄰[[瑞峯村|瑞峯村]]、東鄰[[太和村|太和村]]、西鄰[[太興村|太興村]]、南接[[竹崎鄉|竹崎鄉]]<ref name="1.1">{{cite web |author1=內政部國土測繪中心 |title=臺灣村里界圖 |url=https://whgis.nlsc.gov.tw/GisMap/NLSCGisMap.aspx}}</ref>。瑞里村面積約15.95平方公里，其中包含8個鄰、227戶、人口數706人<ref>{{cite web |author1=嘉義縣竹崎戶政事務所 |title=嘉義縣梅山鄉村里鄰人口數107年9月 |url=https://chuchi-hro.cyhg.gov.tw/News_Content.aspx?n=D3F80864C1D1F68B&sms=BBD9BB2302A9D07D&s=ED9A818F987F3181}}</ref>。瑞里村位於[[阿里山國家風景區|阿里山國家風景區]]內，村內瑞里部落為現今主要聚落，公共設施上有[[瑞里國民小學|瑞里國民小學]]、[[瑞里源興宮|源興宮]]、瑞里村活動中心。景點上以[[瑞里風景區|瑞里風景區]]內之「瑞里八景」聞名，且有燕子崖、蝙蝠洞等知名景點。

== 地理 ==
瑞里村是[[臺灣|臺灣]][[嘉義縣|嘉義縣]][[梅山鄉|梅山鄉]]轄下的一個[[村里|村]]級行政區，位於[[梅山鄉|梅山鄉]]東南部，北鄰[[瑞峯村|瑞峯村]]、東鄰[[太和村|太和村]]、西鄰[[太興村|太興村]]、南接[[竹崎鄉|竹崎鄉]]<ref name="1.1"/>。瑞里村位在[[阿里山國家風景區|阿里山國家風景區]]內，同時處於[[阿里山山脈|阿里山山脈]]周邊，平均海拔高度約為1000公尺。村內有毛生樹溪流經，而東側則有海拔1634公尺的雲戴山<ref name="1.2">{{cite web |author1=內政部 |title=地名資訊服務網：瑞里村 |url=http://gn.moi.gov.tw/geonames/GNMap/map_Admin/map/MapMain.aspx#}}</ref>。

== 聚落歷史 ==
瑞里村是昔日[[阿里山|阿里山]]開發最早的區域，村內主要由幼葉林、科仔林、九芎坪等三聚落構成，三大聚落皆於西元1800年前後開始開發，現今人口則集中於本村中心，瑞里部落<ref name="1.2"/><ref name="1.3">{{cite web |author1=阿里山國家風景區管理處 |title=瑞里 |url=https://www.ali-nsa.net/user/Article.aspx?Lang=1&SNo=04003720}}</ref>。

幼葉林為本村最主要之聚落，閩南人開墾初期，此聚落皆為幼葉楠木原始林，故而得名。產業方面，最早以天然資源為主，後轉型至造林、竹子相關產業，現今則以種植茶葉、觀光發展為主<ref name="1.2"/>。

科仔林聚落最早長滿了水科樹，故得其名。此聚落早期居住人口甚多，但因地形陡峭，容易導致天災，故人口大量外移。產業方面以發展觀光為主<ref name="1.2"/>。

九芎坪聚落最早長滿了九芎樹，故得其名。產業方面，此地原先為原始林，後則以種植茶葉與檳榔為主<ref name="1.2"/>。

== 教育 ==
瑞里村學區內僅有一所[[瑞里國民小學|瑞里國民小學]]，早期因山區交通不便，本村教育發展較為落後，但經過地方人士的努力後，於西元1927年(昭和2年)興建瑞里國民小學前身「小梅公學校分教場」。西元1936年(昭和11年)，由小梅公學校分教場獨立為「生毛樹公學校」，當時開設五個班級。西元1941年(昭和16年)改制為「幼葉林國民學校」，接著又接連改了三次校名，最後於西元1968年(民國57年)正式更名為「嘉義縣梅山鄉瑞里國民小學」<ref name="1.4">{{cite book |author1=嘉義縣政府 |coauthors=李奉儒 |editor=雷家驥 |title=嘉義縣志《教育志》 |date=2015 |publisher=嘉義縣政府 |page=275-276}}</ref>。

瑞里國民小學屬於小型學校，據西元2007年(96學年度)之統計，全校共有6班54人。校內主要社團為扯鈴隊、兒童樂隊及直笛隊。學校另設有附設幼稚園，解決本村居民教養後代之問題<ref name="1.4"/>。

== 文化資源(文化資產) ==
瑞里村內的文化資源有二：[[瑞里源興宮|瑞里源興宮]]以及瑞里村活動中心。

瑞里村內的信仰中心為源興宮，是一間道教寺廟。源興宮由村民共同集資，建於西元1871年(同治10年)，並曾於西元1982年(民國71年)重新翻修過一次，現址為翻修4年竣工後之樣貌。廟內原主奉神明為吳鳳，後改為池王爺及三官大帝，其他神明則有五穀王、延平郡王、觀世音菩薩、地藏王菩薩等<ref>{{cite web |author1=地理資訊科學研究專題中心 |title=文化資源地理資訊系統：瑞里源興宮 |url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/meishan/1015016-YXG}}</ref><ref>{{cite web |author1=嘉義縣文化觀光局 |title=源興宮 |url=http://ruili.sp-store.com/itinerary10.html}}</ref>。

瑞里村活動中心位於幼葉林部落中，其建於1918年(民國7年)，曾於1999年(民國88年)重建。內設有村辦公處、社區發展協會、集會所、閱覽室及休閒室等，具有多功能之用途，瑞里村活動中心是為本村居民主要的活動場所<ref name="1.2"/>。

== 景點 ==
瑞里村內劃有[[瑞里風景區|瑞里風景區]]，以「瑞里八景」聞名。而其中以燕子崖、蝙蝠洞、瑞太古道及[[雲潭瀑布|雲潭瀑布]]等景點最廣為人知<ref name="1.3"/>。

燕子崖因曾有燕子棲息而得名，是一大佈滿條狀岩石的峭壁，有『臺灣敦煌石窟』之美名，景色雄偉壯麗<ref name="1.2"/><ref name="1.3"/>。

蝙蝠洞與燕子崖位於同一條步道上，其因曾有蝙蝠棲息於此而得名，是一座高達百餘尺的大型石灰岩壁，其中因為長年侵蝕的因素，還產生了數千個小石洞，是全國最大的蝙蝠洞之一，極為壯觀<ref name="1.2"/><ref name="1.3"/>。

瑞太古道位於瑞里村與太和村間，是以前兩村通聯農產品的重要道路，整條古道以枕木鋪設，沿途皆是孟宗竹林與原始林木，景色精美雅緻<ref name="1.2"/><ref name="1.3"/>。

[[雲潭瀑布|雲潭瀑布]]是雲潭溪因遭遇斷崖而形成的三層瀑布，因為當地經常雲霧飄渺，加上瀑布於河底鑿成的深潭，故得其名。此瀑布十分雄偉壯麗，號稱「瑞里第一景」<ref name="1.2"/><ref name="1.3"/>。

== 其他特色 ==
*產業

瑞里村除了漂亮的景觀外，還有豐富的農產，包括[[茶|茶]]、[[竹筍|竹筍]]、[[檳榔|檳榔]]、[[愛玉|愛玉]]、[[梅子|梅子]]、[[金針|金針]]及[[板栗|板栗]]等，其中的茶園與金針園構成的風光更為瑞里景觀增添不同的色彩。而料理上當地更以茶葉、金針及梅子入菜，有特色且別具風味<ref name="1.3"/>。

*紫藤花季

瑞里村曾於西元2014年3月(民國103年)舉辦瑞里紫藤花季，嘉義縣政府觀光局邀請日籍藝人[[大久保麻梨子|大久保麻梨子]]擔任代言人，吸引民眾至瑞里村欣賞紫藤花海<ref>{{cite news |author1=EToday旅遊中心 |title=瑞里紫藤花季登場　15個最佳賞花處大公開 |url=https://travel.ettoday.net/article/331880.htm |accessdate=2014-03-06 |work=EToday旅遊雲}}</ref>。
