'''瑞和車站'''位於[[台灣|台灣]][[臺東縣|臺東縣]][[鹿野鄉|鹿野鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[侧式月台|侧式月台]]一座。
*本站無天橋及地下道，旅客需跨越軌道才能進入月台乘車。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]，並指定由[[瑞源車站|瑞源站]]管理。
*停靠狀態：
**僅停靠[[臺鐵區間車|區間車]]，為台東線停靠班次最多的招呼站，共13班。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1923年7月14日：設「大埔停車場」開始營業。
*1945年：因行政區域改稱為[[鹿野鄉|鹿野鄉]]瑞豐村，站名和地名改稱為「瑞豐」。
*1979年1月1日：因瑞和村由瑞豐村中分出，站名和地名再改為「瑞和」。
*1986年3月24日：降為簡易站，並指定由[[瑞源車站|瑞源站]]管理。
*1989年5月15日：降為招呼站。
*2016年6月28日：啟用多卡通刷卡機。
