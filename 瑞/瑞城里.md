'''瑞城-{里}-'''，位於大里區南境偏東,為全區人口最多的里。

== 里名由來 ==
以「塗城」之「城」字，另加雅字「瑞」，定名為「瑞城」。

取名之初，原本取名「銀城」，但是鄉民代表會認為此名不妥，所以經由當地耆老、士紳及地方人士再三考慮後，訂名為「瑞城」，取其祥瑞的意思，為當地帶來和諧、安詳。

== 地理 ==
瑞城里劃分為44鄰，4215戶，總人口14,135人（2014年1月）。本里東為[[塗城里|塗城-{里}-]]，西隔中興路鄰[[東湖里_(臺中市)|東湖-{里}-]]，南臨[[霧峰區|霧峰區]]吉峰里，北為[[金城里|金城-{里}-]]、[[仁德里_(臺中市大里區)|仁德里]]。

== 行政區劃沿革 ==
瑞城里於清代時隸屬藍興堡土城莊；明治34年（1901年）時，改隸臺中廳太平區塗城庄，至大正9年（1920年）則隨行政區調整，隸屬大屯郡大里庄塗城大字。二次大戰結束後，國民政府接收臺灣，以原塗城大字範圍置'''塗城村'''，隸屬臺中縣大里鄉。民國71年（1982年）析塗城村置'''瑞城村'''。民國82年（1993年）大里鄉升格為縣轄市'''大里市'''，隨之改制為'''瑞城-{里}-'''。
