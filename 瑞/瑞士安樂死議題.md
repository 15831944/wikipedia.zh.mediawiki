{{安樂死}}
[[瑞士|瑞士]]是全世界第一個、也是少數可合法實施被動[[安樂死|安樂死]]的國家。瑞士在1941年通過安樂死相關法案，合法化[[協助自殺|協助自殺]]；協助方式局限於為被動（例如除去病人的維生系統或讓病人停止服藥）而非主動（例如透過注射方式），限定須由醫師以外人士執行。<ref name="Hurst Mauron">{{cite journal |author=Hurst SA, Mauron A |title=Assisted suicide and euthanasia in Switzerland: allowing a role for non-physicians |journal=BMJ |volume=326 |issue=7383 |pages=271–3 |pmid=12560284 |pmc=1125125|date=February 2003}}</ref><ref name="appledaily">{{cite web |url=http://www.appledaily.com.tw/appledaily/article/international/20110517/33392818 |title=幫外國人被動安樂死 瑞士公投通過 |work=《[[蘋果日報|蘋果日報]]》 |date=2011年5月17日 |accessdate=2012年8月25日}}</ref><ref>{{cite web |url=http://news.bbc.co.uk/chinese/trad/hi/newsid_2950000/newsid_2951500/2951567.stm |title=瑞士組織助英國夫婦被動安樂死 |work=BBC中文網 |date=2003年4月15日 |accessdate=2012年8月25日}}</ref>

瑞士被動安樂死制度的接受者不限為[[瑞士公民|瑞士公民]]，但隨著前往瑞士尋求被動安樂死的外籍人士漸增，瑞士也因此蒙上「自殺觀光業」的陰影。根據[[瑞士聯邦統計局|瑞士聯邦統計局]]資料顯示，1998年有43人在協助自殺下過世，2009年則來到近300人<ref name="yahoohk" />；而單在[[蘇黎世|蘇黎世]]，每年有約200人尋求被動安樂死協助，其中很大比例為外國人<ref name="appledaily" />。此外，尋求協助自殺的案例中，9成接受者的年齡超過55歲<ref name="yahoohk">{{cite web |url=http://hk.news.yahoo.com/%E7%91%9E%E5%A3%AB%E5%AE%89%E6%A8%82%E6%AD%BB%E4%BA%BA%E6%95%B810%E5%B9%B4%E7%BF%BB6%E5%80%8D-073000562.html |title=瑞士被動安樂死人數10年翻6倍 |work=[[路透社|路透社]] |date=2012年3月28日 |accessdate=2012年8月25日}}</ref>。

== 主動安樂死 ==
所有的主動安樂死，例如對病患注射致命藥物，在瑞士仍是受禁止的。瑞士立法只允許為接受者提供藥物或器具，且不得涉於利益。<ref>{{cite web |editor=Derek Humphry |url=http://www.assistedsuicide.org/suicide_laws.html |title=Tread Carefully When You Help to Die - Assisted Suicide Laws Around the World |date=2005-03-01 |accessdate=2012年8月25日}}</ref>

== 瑞士刑法第115條 ==
瑞士在1937年通過刑法第115條，1942年生效。刑法第115條明定[[協助自殺|協助自殺]]只有在動機是為自身利益的條件下，方才被認定為犯罪。<ref name="Hurst Mauron" />

協助自殺案件發生後，警方將主動介入調查。如果協助者沒有私人意圖，警方便會結束案件偵查。如果調查顯示，協助自殺並非病患的主動選擇，或協助者具有特別意圖，導致接受者權益受損時，案件便將導向訴訟。

1980年代，即在刑法第115條生效的40年後，瑞士社會逐漸引用法條承認協助自殺的機構的合法性。
