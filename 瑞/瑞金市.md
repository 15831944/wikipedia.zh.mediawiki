'''瑞金市'''是位于[[中华人民共和国|中华人民共和国]][[江西省|江西省]]东南部的一个[[县级市|县级市]]，现由[[赣州市|赣州市]]代管。

==历史==
[[File:Ruijin_Shazhouba_Geming_Jiuzhi_2014.05.30_16-01-35.jpg|缩略图]]
[[File:Ruijin_Shazhouba_Geming_Jiuzhi_2014.05.30_16-15-31.jpg|缩略图]]
[[东汉|东汉]][[建安_(东汉)|建安]]七年（202年），置象湖镇，属[[雩都县|雩都县]]。[[唐朝|唐朝]][[天祐_(唐朝)|天佑]]元年（904年）析雩都县置瑞金监，因“掘地得金，金为瑞”得名。据《[[太平寰宇记|太平寰宇记]]》：“瑞金场，淘金之地也。”又据《瑞金县志》：“置（瑞金）监时，有航浮水面，色如黄金，人目为瑞”，故名。[[南唐|南唐]][[保大_(南唐)|保大]]十一年（953年）升监为县。

1931年11月，[[第一次全国苏维埃代表大会|第一次全国苏维埃代表大会]]在[[瑞金县|瑞金县]]召开。[[中华苏维埃共和国|中华苏维埃共和国]]成立后，瑞金作为首都，曾更名'''瑞京'''。其时曾有宣传口号为「北京南京不如瑞京，美国英国不如兴国」。中华人民共和国建立后，与另一个[[革命圣地|革命圣地]][[延安|延安]]类似，有上海[[瑞金一路|瑞金一路]]、[[瑞金二路|瑞金二路]]、[[上海瑞金医院|上海瑞金医院]]，以及南京[[瑞金路|瑞金路]]，以瑞金命名，以纪念此段历史。

1994年5月18日，瑞金撤县设市。2014年7月1日起，为江西省试点省管县。<ref>{{cite web |url=http://www.ruijin.gov.cn/ruijinoverview/ruijinhistory/200810/t20081030_8196.htm |title=存档副本 |accessdate=2011-09-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20110715213556/http://www.ruijin.gov.cn/ruijinoverview/ruijinhistory/200810/t20081030_8196.htm |archivedate=2011-07-15 }}</ref>

== 地理 ==
瑞金位于江西省东南部，[[赣江|赣江]]东源[[贡水|贡水]]上游，邻接福建省。

境内主要河流有：绵江、九堡河、梅江、万田河、元坑河。<ref>{{cite web |url=http://www.ruijin.gov.cn/ruijinoverview/zrzy/200907/t20090708_17747.htm |title=存档副本 |accessdate=2011-09-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20110715213609/http://www.ruijin.gov.cn/ruijinoverview/zrzy/200907/t20090708_17747.htm |archivedate=2011-07-15 }}</ref>

== 经济 ==
=== 工业 ===
瑞金工业产品中烤鳗、九华痔疮栓、萤石精粉、电线电缆等在市场竞争中属强势品牌，形成了食品、建材、医药生化、矿产品加工等支柱产业。发制品、箱包、家俱生产成为工业经济新的增长点。
   
=== 农业 ===
瑞金农业初步形成了烤烟、肉牛、鳗鱼、脐橙等四大产业，正努力争取成为赣南脐橙的主产区和全国绿色食品的生产基地。

=== 商业 ===
瑞金地处赣闽粤三省通衢之地，自古以来商贸云集，不仅成为辐射周边10多个县市的消费品集散地，也是福建沿海地区辐射中西部腹地的中转枢纽。大型百货商场、大型超市蓬勃发展，培育了摩托车、服装、钢材、水果等专业市场21个。红都商业城为赣东南、闽西边际最大的商业城。

*[[罗汉岩|罗汉岩]]：[[丹霞地貌|丹霞地貌]]景区，是江西省风景名胜区
*[[密溪村|密溪村]]：江西省历史文化名村
