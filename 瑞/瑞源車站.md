'''瑞源車站'''位於[[台灣|台灣]][[臺東縣|臺東縣]][[鹿野鄉|鹿野鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。本站配合「花東新車站運動」計畫改建完工。

== 車站構造 ==
*[[島式月台|島式月台]]一座。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]。
*2018/10/12 年度改點之後 ，實施週日班表。
*停靠狀態：
**[[莒光號列車|莒光號]]：順行602、640(週六行駛)/642（逢週日停駛)、6642次（逢週日行駛）；逆行653（逢週日停駛）、6653（逢週日行駛）、611次停靠。
**[[臺鐵區間車|區間車]]：皆停靠。
*對號列車停靠率：16.0%；所有車種停靠率：33.3%。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1922年4月20日：設「大原」驛開始營業，「大原」原稱為「大埔尾」。[[台灣日治時期|日治初期]]日人移居此地，改名為「大原村」，設站時取其村名作為站名。
*1945年：因本站行政區域改為[[鹿野鄉|鹿野鄉]]瑞源村，故站名也隨地名更改為「瑞源」站。
*2016年6月28日：啟用多卡通刷卡機。
