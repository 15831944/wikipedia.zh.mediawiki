'''瑞丽景成直升机场'''又称'''瑞丽直升机起降点'''，是[[云南省|云南省]][[德宏傣族景颇族自治州|德宏傣族景颇族自治州]][[瑞丽市|瑞丽市]]的一座[[直升机场|直升机场]]，是[[中国西南地区|西南地区]]规模最大的通用航空机场<ref>{{cite web |url=http://news.sina.com.cn/o/2014-05-19/074530166720.shtml |title=瑞丽、南亚通用航空昨首航 瑞丽开建西南最大通用航空机场 |author=项陆才 |date=2014-05-19 |website=新浪新闻 |publisher=云南网 |access-date=2017-08-27 |quote=}}</ref><ref>{{cite web |url=http://yn.yunnan.cn/html/2014-05/18/content_3215506.htm |title=瑞丽航空、南亚航空同日首航 云南民营航空企业展翅高飞 |author=王楠 |date=2014-05-18 |website=yn.yunnan.cn |publisher=云南网 |access-date=2017-08-27 |quote= |accessdate=2017-08-28 |archiveurl=https://web.archive.org/web/20171001031619/http://yn.yunnan.cn/html/2014-05/18/content_3215506.htm |archivedate=2017-10-01 |deadurl=yes }}</ref>，位于瑞丽城区西北侧[[勐秀乡|勐秀乡]]南京里的刀坝<ref>{{cite web |url=http://www.ruili.gov.cn/tzrl/dt/content-166-240-1.html |title=瑞丽景成直升机公司迎来第一架直升机 |author=李卫昌 |date=2013-08-28 |website=www.ruili.gov.cn |publisher=瑞丽江 |access-date=2017-08-27 |quote=}}</ref>，距离市区10公里，对民众开放<ref name="www.ethcy.com"></ref>。占地面积约120亩，其中进场道路占地约20亩，有1万平方米停机坪一个，可同时停放六架直升机，包括航站楼、机务办公楼、后勤办公楼各一幢，配备停机库、指示塔、航油库、消防中心等设施，另有一条长340米的跑道。<ref name="建设纪实">{{cite web |url=http://www.jcjtgs.com/content-20-122-1.html |title=瑞丽景成直升机场建设纪实 |last= |first= |date=2013-09-23 |website=www.jcjtgs.com |publisher=云南景成集团有限责任公司 |access-date=2017-08-27 |quote=}}</ref><ref name="www.sirenji.com">{{cite web |url=http://www.sirenji.com/article/201308/36442.html |title=瑞丽2亿元直升机S92A落地 明年开始运营 |last= |first= |date=2013-08-28 |website=私人飞机网 |publisher=昆明信息港 |access-date=2017-08-27 |quote=}}</ref>

瑞丽景成直升机场于2013年1月18日开工建设<ref name="建设纪实"></ref>，2014年落成启用<ref>{{cite web |url=http://www.sirenji.com/article/201308/36442.html |title=瑞丽2亿元直升机S92A落地 明年开始运营 |last= |first= |date=2013-08-28 |website=www.sirenji.com |publisher=昆明信息港 |access-date=2017-08-27 |quote=}}</ref>。

2013年8月26日，[[德宏南亚通用航空|瑞丽景成直升机有限公司]]<ref group="注">今[[德宏南亚通用航空|德宏南亚通用航空]]</ref>的一架[[西科斯基飞机公司|西科斯基]]{{le|西科斯基 S-92|Sikorsky S-92#S-92A|S-92A}}自[[德宏芒市机场|德宏芒市机场]]飞抵瑞丽景成直升机场，为瑞丽景成直升机场首航<ref>{{cite web |url=http://society.yunnan.cn/html/2013-08/27/content_2860238.htm |title=芒市到瑞丽可坐直升机 飞行时间需20余分钟 |author=胡静 |date=2013-08-27 |website=云南网 |publisher=春城晚报 |access-date=2017-08-27 |quote= |archive-url=https://web.archive.org/web/20170903072508/http://society.yunnan.cn/html/2013-08/27/content_2860238.htm |archive-date=2017-09-03 |dead-url=yes }}</ref>。直升机运营后，将开展瑞丽、芒市、[[腾冲市|腾冲]]、[[保山市|保山]]等地的飞行，还将开通飞往缅甸的国际直升机航线<ref name="www.sirenji.com"></ref><ref name="www.ethcy.com">{{cite web|url=http://www.ethcy.com/htmls/qyxw/15422.html|title=直升机S-92 首抵瑞丽景成直升机场|last=|first=|date=2014-03-12|website=通航产业网|publisher=云新网（昆明）|access-date=2017-08-27|quote=}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。
