'''瑞士信贷集团'''（{{lang-en|Credit Suisse Group AG}}，簡稱'''瑞信'''，也稱'''瑞士信貸銀行'''）（{{SWX|CSGN}}, {{nyse|CS}}）是一家在1856年创立的国际投资银行和金融服务公司，總部位于[[瑞士|瑞士]]的[[蘇黎世|蘇黎世]]。截至2017年底，瑞信在全球50多个国家拥有雇员近4万7千人，管理资产达1.376万亿[[瑞郎|瑞郎]]（约9.6万亿[[人民币|人民币]]），是仅次于[[瑞银集团|瑞银]]的瑞士第二大银行。<ref>{{cite web |url=https://www.mg21.com/cs.html|title=瑞士银行龙头：瑞士信贷/瑞信银行Credit Suisse(CS)|publisher=美股之家|accessdate=2018年2月6日}}</ref> <ref>{{cite web |url=https://www.credit-suisse.com/corporate/sc/about-us.html|title=瑞信 - 关于我们|publisher=瑞信|accessdate=2018年10月18日}}</ref>

== 历史 ==
瑞信的雏形要追溯到1856年由瑞士政治家[[阿尔弗里特·爱舍|阿尔弗里特·爱舍]]于[[苏黎世|苏黎世]]成立的瑞士信贷机构 (德语：Schweizerische Kreditanstalt，简称：SKA）。

1870年，SKA 成功在[[美国|美国]][[纽约|纽约]]开设代理业务。随后，陆续在世界各地陆续开设代理，如法国巴黎（1910年）、英国伦敦（1954年）、阿根廷的布宜诺斯艾利斯（1959年）、中国香港（1969年）、巴林（1975年）等。<ref>{{Cite book|title=德语：Von der Schweizerischen Kreditanstalt zur Credit Suisse Group. Eine Bankengeschichte.|last=Jung|first=Joseph|publisher=NZZ-Verlag|year=2000|isbn=ISBN 978-3-85823-815-3|location=苏黎世|pages=}}</ref> 1964年，SKA 位于纽约分部得到美国许可成为[[全能银行|全能银行]]。<ref>{{Cite web|url=http://markenlandschaft.ch/markenlexikon/credit-suisse|title=Schweizer Markenlandschaft - Credit Suisse（德语）|accessdate=2019年1月17日|author=|date=|publisher=Markenlandschaft}}</ref>

自1873年起，瑞信的总部一直位于瑞士经济中心[[苏黎世|苏黎世]]市的[[阅兵广场_(苏黎世)|阅兵广场（Paradeplatz）]]。1905年，瑞士信贷机构在[[巴塞尔|巴塞尔]]开设分行，并逐步主要通过兼并地方银行的方式在瑞士国内扩张。

==瑞信在亚洲==

==== 瑞信在中国 ====
瑞信在华业务最早始于1955年与中国银行建立关系，并于1985年及1992年先后在[[北京|北京]]及[[上海|上海]]设立代表处，1994年，上海代表处改为瑞士信贷[[中国|中国]]分行。1998年6月，瑞士信贷上海分行获准经营人民币业务<ref>{{cite web |url=http://zx.finchina.com/fcasp/jgfl/default.aspx?companycode=80048301|title=瑞士信贷 - 公司简介|accessdate=2018年10月18日}}</ref>。

瑞信在资产管理领域通过和中方合作的方式，逐步进入中国的券商及公募基金等业务。2005年6月，工银瑞信基金管理有限公司中国工商银行和瑞信联手创办。2008年10月，瑞信同方正证券合资成立了瑞信方正证券有限责任公司，经营投行业务，方正证券和瑞银分别控股66.7%和33.3%。<ref>{{Cite web|url=http://fund.cnfol.com/smjj/simudongtai/20181205/27075242.shtml|title=瑞信加入“洋私募”队伍 中国资管市场加速国际化|accessdate=2018年12月6日|author=|date=2018年12月5日|publisher=国际金融报社}}</ref>
