'''瑞娃·斯廷坎普'''（{{lang-en|'''Reeva Steenkamp'''}}，{{bd|1983年|8月19日|2013年|2月14日|catIdx=Donald Richie}}），南非人，女[[模特|模特]]，2013年2月14日被其男友南非残疾短跑运动员[[奥斯卡·皮斯托利斯|奥斯卡·皮斯托利斯]]枪杀。

== 早年生活 ==
瑞娃·斯廷坎普生于南非[[开普敦|开普敦]]，后举家迁至[[伊莉莎白港|伊莉莎白港]]。

瑞娃·斯廷坎普2005年于[[纳尔逊曼德拉都市大学|纳尔逊曼德拉都市大学]]获[[法学士|法律学士]]学位。

== 职业 ==
瑞娃·斯廷坎普14岁开始模特生涯。

2011年和2012年分别位列《[[男人装|男人装]]》最性感美女排行榜第40和第45位<ref>{{cite news |author= |url=http://fhm.co.za/in-the-mag/fhm-100-sexiest-issue-reeva-steenkamp-no-45/8698 |title=《男人装》最性感美女排行榜：瑞娃·斯廷坎普（第45位） |work=男人装 |date=2012-06-22 |accessdate=2013-02-21}}</ref>。

== 个人生活 ==
瑞娃·斯廷坎普2012年11月成为南非残疾短跑运动员被誉为"刀锋战士"的[[奥斯卡·皮斯托利斯|奥斯卡·皮斯托利斯]]的女友。

== 被杀 ==
2013年2月14日在奥斯卡·皮斯托利斯位于[[比勒陀利亚|比勒陀利亚]]的家中被其枪杀。

葬礼于2月19日在伊莉莎白港举行。
