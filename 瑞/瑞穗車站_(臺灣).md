'''瑞穗車站'''位於[[台灣|台灣]][[花蓮縣|花蓮縣]][[瑞穗鄉|瑞穗鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
瑞穗車站是設置有[[島式月台|島式]]、[[岸式月台|岸式月台]]各一座的地面車站，站房與月台間是以[[行人地下道|行人地下道]]相連。

在瑞穗車站設立當初就已建造的站房原本位於路軌的東側。2009年時，[[中華民國交通部|交通部]]推出「花東新車站運動」（正式計畫名「花東線鐵路整體服務效能提升計畫」）開始進行台灣東岸各鐵路車站的改建工程，雖然對於大部分的車站而言改建工程主要著眼在站房外觀的更新（拉皮）與站內設備的升級，但瑞穗車站是少數更新規模較大的車站之一。為了爭取較大的腹地與平衡鎮區發展，瑞穗車站的整個站區皆向東移動，除了原有的站房進行外觀與設備的更新及內部空間的重設外，也在軌道西側新建了一座整合遊客中心與[[自行車|自行車]]補給站功能的西站房，並增設連結東西兩站區的地下道{{R|KSN_130706}}{{R|vf2012}}{{R|SL203}}。工程預算原訂2.7億[[新台幣|新台幣]]，但在追加調整後上修為3億元{{R|KSN_130706}}{{R|SL203}}。改建工程於2012年時正式開始，並計畫預定在2017年6月完成，截至2017年3月西站的興建工程已完成並繼續進行東站的內外觀工程，與站前道路的整修等收尾工作{{R|KSN_170315}}。

== 營運狀況 ==
根據台鐵的分級瑞穗車站屬於[[臺灣鐵路管理局車站等級#三等站|三等站]]等級，停靠部分的[[普悠瑪列車|普悠瑪號]]與大部分的[[自強號列車|自強號]]班次，與所有的[[莒光號列車|莒光號]]及[[臺鐵區間車|區間車]]。瑞穗車站的[[閩南語|閩南語]]名廣播是依瑞穗的古名「水尾」發音，而非以源自[[日文|日文]]的站名直譯。站內設有可供旅客蓋戳留念之「鐵路之旅 - 小站巡禮紀念章」戳章，且因隔鄰的[[舞鶴車站|舞鶴車站]]早已在1982年[[台東線#臺東線拓寬計畫|東拓]]完成時裁撤並由瑞穗車站託管，因此站內也一併收藏著舞鶴的戳章。
*停靠
**[[普悠瑪列車|普悠瑪號]]：除北上445次(週日停駛)不停靠外，其餘列車皆停。
**[[太魯閣列車|太魯閣號]]：除南下408次、426次；北上421次（週日停駛） 、441次 （週日停駛）、6421次（週日行駛） 、6441次（週日行駛）不停靠外，其餘列車皆停。
**[[自強號列車|自強號]]：除南下418次（逢週五、六、日行駛）不停靠外，其餘列車皆停。
**[[莒光號列車|莒光號]]：皆停靠。
**[[臺鐵區間車|區間車]]：皆停靠。
**[[觀光列車|觀光團體列車]]：停靠南下72（逢週六行駛）、82（逢週一、五至日行駛）次；北上6073（逢週日行駛）、81（逢週一、五、六行駛）、6081（逢週日行駛）次。
在2016年站內票務設備升級之後，瑞穗車站開始提供使用[[一卡通_(台灣)|一卡通]]、[[悠遊卡|悠遊卡]]及[[icash|icash 2.0]]等[[電子票卡|電子票卡]]付費的服務。

== 歷史 ==
*1914年：設立'''水尾乘降場'''。
*1915年1月26日：改為'''水尾停車場'''。
*1917年9月22日：改為'''瑞穗驛'''。
*1948年：因颱風侵襲，原木造站房全倒，災後以鐵皮屋翻修補強。
*1968年：原有鐵皮屋站房拆除，改建為[[鋼筋混凝土構造|鋼筋混凝土]]站房。
*1970年：改建站房啟用。
*2012年：配合台東線沿線各車站站房重新翻修改造，開始進行站區的重新整建，包含原有東站房及周遭相關設施的整修，與新建西站房{{R|vf2012}}。
*2015年：西站房完工{{R|KSN_170315}}。
*2016年6月28日：啟用多卡通刷卡機。
