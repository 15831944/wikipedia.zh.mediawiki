'''瑞典國家手球隊'''是由[[瑞典手球總會|瑞典手球總會]]於1948年所創立，乃[[北歐|北歐]]成績最佳的[[手球|手球]]隊。

== 世界锦标赛成績 ==
* 冠軍：4 (1954年, 1958年, 1990年及1999年) 
* 亞軍：3 (1964, 1997, 2001)
* 季軍：3 (1961, 1993, 1995) 

== 歐洲锦标赛成績 ==
* 冠軍：4 (1994年, 1998年, 2000年及2002年) 

== 奧運會成績 ==
* 亞軍：3 (1992, 1996, 2000)
