'''瑞安街'''為[[臺灣|臺灣]][[臺北市|臺北市]]市區的街道，大致呈東北－西南方向，全境位於[[大安區_(台北市)|大安區]]；北端始於[[臺北捷運|臺北捷運]][[大安站_(臺灣)|大安站]]後方，先是往南約200公尺後，轉向西南方向，穿過[[復興南路|復興南路]]二段，直到[[和平東路|和平東路]]二段為止，全長約1公里。

==歷史文化==
[[File:Taipei_ruian_st_1905.jpg|thumb]]中，可在紅色箭頭處看見現今瑞安街路段與一旁的水圳。]]
[[File:City_Plans_Taihoku-Matsuyama.jpg|thumb]]

瑞安街過去曾屬於[[臺北市|臺北市]]東區南北向的主要道路[[安東街|安東街]]的一部分<ref name="Appledaily-20140125">{{Cite news| url=https://tw.appledaily.com/new/realtime/20140125/333554/| title=羅文嘉：風華過後| newspaper=蘋果日報| date=2014-01-25| accessdate=2019-01-02}}</ref>，這條道路早在[[臺灣清治時期|清治時期]]就已存在，由於過去位於臺北城以外的道路大多沿著水圳道開闢，安東街即是沿著[[霧裡薛圳#路線|第一霧裡薜支線]]如此的隨興貫穿於當時台北的東部區域中，在[[台灣日治時期|日治時期]]的「[[臺灣堡圖|臺灣堡圖]]」中已經能見到他的蹤跡<ref>{{cite web |url=http://gissrv4.sinica.edu.tw/gis/taipei.aspx|title=臺北市百年歷史地圖 |accessdate=2019-01-02}}</ref><ref>{{cite web |url=http://blog.udn.com/marvinfann/5124062 |title=水城台北 之二 |accessdate=2019-01-02| work=Marvin和我們一起閱讀}}</ref>，在當時也是本地區唯一南北向的連通道路<ref name="NTNUHS">{{cite web |url=https://www.hs.ntnu.edu.tw/great/change/change_4/ |title=附中變遷史（四）|publisher=國立臺灣師範大學附屬高級中學 |accessdate=2019-01-02}}</ref>，也因此街道內商家林立，車輛與行人絡繹不絕。

1975年因[[復興南路|復興南路]]闢建[[信義路_(台北市)|信義路]]至[[和平東路|和平東路]]之間的路段，原本安東街自[[忠孝東路|忠孝東路]]至信義路之間的路段經拓寬後成為復興南路的一部分<ref>{{Cite book| url=https://www-ws.gov.taipei/001/Upload/public/Attachment/17414333478.pdf| title=臺北市大安區志| chapter=卷五 交通篇 | year=2011| publisher=臺北市大安區公所| page=203| accessdate=2019-01-02}}</ref>，導致安東街被分為位於忠孝東路以北的北段與信義路以南的南段；因此安東街南段改名成'''瑞安街'''<ref name="NTNUHS"/>，名稱則源自中國[[浙江省|浙江省]]的瑞安縣（現在的[[瑞安市|瑞安市]]）。

瑞安街起點原本位於現在的[[信義路_(台北市)|信義路]]與[[復興南路|復興南路]]路口的東南角，1988年為興建[[臺北捷運|臺北捷運]][[大安站_(臺北市)|大安站]]被徵收，使得瑞安街出現門牌號碼斷頭的怪現象。瑞安街中段與[[復興南路|復興南路]]的交叉路口，也由於在1990年代被徵收供[[臺北市政府消防局|臺北市政府消防局]]第二大隊大安中隊復興分隊（1999年成立，2002年啟用）使用，使得街道無法直接穿越[[復興南路|復興南路]]。而1987年新闢[[新龍公園|新龍公園]]於瑞安街214巷旁，更直接佔用了瑞安街的部分路段，使得整條瑞安街可以說是柔腸寸斷。

; 反民墓地

現在的瑞安街256巷內在日治時代中期曾設有「反民墓地」，包括[[羅福星|羅福星]]、[[陳阿榮|陳阿榮]]、[[賴來|賴來]]、[[張火爐_(台灣日治時期)|張火爐]]、[[李阿齊|李阿齊]]等多位參與[[苗栗事件|苗栗事件]]的抗日人士於[[臺北刑務所|臺北刑務所]]處刑後曾被葬於此地，直到1952年才將遺骸遷至苗栗[[大湖義民廟|大湖義民廟]]<ref>{{Cite journal| title=台灣歷史名人──印尼華僑羅福星的故事| author=邱炫煜| journal=印尼僑聲雜誌| issue=114| date=2007-12-20| accessdate=2019-01-03| url=http://www.ezsino.org/index.php?mod=journal&action=show&cid=8&pid=53&id=1137}}</ref><ref>{{Cite news| title=苗栗縣揭羅福星畫像| date=2014-03-04| author=胡蓬生| newspaper=聯合報| accessdate=2019-01-03}}</ref>。

; 日式住宅群

[[File:臺大宿舍－大安區瑞安街264巷16、18號.jpg|thumb]]。1937年由營造商千歲町建築信用購買利用組合興建，日治時期分別為下津浦廾、塩谷巖三自宅，戰後分別配住予台大森林系教授黃希周、化學系教授張苕旭。]]

日治時代末期在瑞安街南段靠近現在和平東路附近的區域，開始有民間營造商營建日式住宅群及住宅[[團地|團地]]，但隨著城市的開發，大多當時的住宅都已被拆除重建，目前僅有位於264巷16、18號的兩棟建築在戰後因做為[[國立臺灣大學|國立臺灣大學]]教師宿舍使用而被保存至今，這兩棟建築已於2014年被列為[[臺北市文化資產|臺北市文化資產]]<ref>{{Cite web| url=https://nchdb.boch.gov.tw/assets/overview/historicalBuilding/20140421000001| title=國立臺灣大學日式宿舍－大安區瑞安街264巷16、18號| publisher=文化部文化資產局| work=國家文化資產網| accessdate=2019-01-03}}</ref>。

; 名人住所

瑞安街周邊在[[臺灣戰後時期|戰後時期]]曾經住著許多文學人士與[[將軍|將軍]]，包括散文作家[[梁實秋|梁實秋]]、[[陳之藩|陳之藩]]、[[夏元瑜|夏元瑜]]、[[姚宜瑛|姚宜瑛]]、詩人[[夏菁|夏菁]]都曾居住於瑞安街北段附近<ref>{{Cite news| url=https://tw.finance.appledaily.com/daily/20120929/34539780/| title=明水路 水岸豪宅政商愛 瑞安街 綠意簇擁藝人多| date=2012-09-29| author=周承諺| accessdate=2019-01-03| newspaper=蘋果日報}}</ref><ref>{{Cite web| url=https://www.nlpi.edu.tw/JournalDetailC003313.aspx?Cond=c4b57680-e398-425b-ba92-912cb93fab73| title= 台灣文學苑－亮軒 快人快語 下筆千言| author=林家成 | publisher=國立公共資訊圖書館| accessdate=2019-01-03}}</ref>。而在現在瑞安街與[[復興南路|復興南路]]二段111巷巷內（消防隊對面）原有四戶平房（部份已遭改建為停車場）過去都曾是將軍官邸，曾住在此區域的將軍包括曾擔任[[中華民國國防部|中華民國國防部]]部長的[[陳大慶|陳大慶]]上將<ref name="Appledaily-20140125"/>、[[鄭為元|鄭為元]]上將<ref>{{Cite news| url=http://news.ltn.com.tw/news/local/paper/436642| title=軍方官舍 被貼競選看板| author=許紹軒、黃忠榮| newspaper=自由時報| date=2010-10-19| accessdate=2019-01-02}}</ref>，及在[[中華民國空軍|中華民國空軍]]官方紀錄中擁有擊落最多敵機紀錄的[[柳哲生|柳哲生]]少將（現址為89號）<ref name="ohsir">{{cite web|url=https://ohsir.tw/2933/|title=第三幕：柳哲生將軍家的流言陰霾（三）|first=|last=艾德嘉|date=2018-09-27|publisher=|accessdate=2019-01-02}}</ref>等。

1961年發生遭當時媒體密切關注的[[瑠公圳分屍案|瑠公圳分屍案]]，由於兇手及死者住所緊鄰瑞安街（當時仍為安東街），初期偵辦期間在目擊證人提供線索後，警方曾密集在瑞安街周邊進行調查，也造成包括柳哲生將軍之家僕等部分附近居民一度受到冤枉成為嫌疑人。<ref name="ohsir"/>

瑞安街內目前也是許多名人居住的地方，包括[[蔡依林|蔡依林]]、[[蕭亞軒|蕭亞軒]]、[[藍心湄|藍心湄]]、[[桂綸鎂|桂綸鎂]]、[[吳宗憲|吳宗憲]]等知名藝人<ref>{{cite web |url=https://news.housefun.com.tw/jill/article/392767182639 |title=名人鍾愛大安區豪宅　Jolin看屋處緊鄰陶喆家 | date=2017-12-21| author=Mira| work=好房網| accessdate=2019-01-02}}</ref><ref>{{cite web |url=https://news.housefun.com.tw/jill/article/392767107755 |title=阮慕驊起家厝近瑞安街名人巷天后置產房價沾光 || author=Mira | work=好房網| date=2015-09-18| accessdate=2019-01-02}}</ref><ref>{{Cite news| title=吳宗憲妻出面 證實結婚十年| author=徐紀琤| newspaper=中國時報| date=2000-08-15| accessdate=2019-01-03}}</ref>，政治人物[[羅文嘉|羅文嘉]]所經營的[[水牛出版社|水牛書店]]也位於此處<ref>{{cite web |url=https://www.thenewslens.com/article/35287 |title=【獨立書店與街區風貌】水牛書店：夢想在故鄉開一間書店，但不賣書 | author=陳怡慈| work=The News Lens 關鍵評論網| date=2016-02-15| accessdate=2019-01-02}}</ref>。
