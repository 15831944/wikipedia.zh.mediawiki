'''瑞丽—孟连高速公路'''简称'''瑞孟高速'''，是[[云南省|云南省]]规划建设的一条高速公路，全长521.359公里，其中部分路段与[[杭瑞高速|杭瑞高速]]、[[镇清高速|镇清高速]]和[[临清高速|临清高速]]共线，新建里程368.756公里，项目总投资607.2亿元。计划2017年12月开工建设，2020年通车。高速采用技术标准为双向四车道，设计速度80公里/小时，路基宽25.5米。<ref>{{cite web |url=http://www.ynjtt.gov.cn/Item/181838.aspx |title=省政府专家组实地踏勘瑞孟高速 |author=匡鹏 |date=2017-10-30 |website=www.ynjtt.gov.cn |publisher=云南省交通运输厅 |access-date=2017-12-10 |quote=}}</ref><ref name="云南高速通">{{cite web |url=http://www.read138.com/archives/100/iaf31v8lnb7kh5us/ |title=瑞丽至孟连高速将开建，建成后将增强沿边经济带的发展活力 |author=启胜 |date=2017-12-01 |website=www.read138.com |publisher=云南高速通 |access-date=2017-12-10 |quote= }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

瑞孟高速连接昆明至[[猴桥镇|猴桥]]、昆明至[[瑞丽市|瑞丽]]、昆明至[[孟定清水河口岸|清水河]]3条出境大通道，串联了[[瑞丽口岸|瑞丽]]、[[畹町口岸|畹町]]、[[南伞口岸|南伞]]、[[孟定清水河口岸|清水河]]、[[沧源口岸|永和]]、[[孟连口岸|孟连]]等6个口岸，连接[[德宏芒市机场|德宏芒市机场]]、[[沧源佤山机场|沧源佤山机场]]、[[澜沧景迈机场|澜沧景迈机场]]等3个机场。<ref>{{cite web |url=http://k.sina.cn/article_1069205631_3fbac87f0200063yt.html?cre=tianyi&mod=wpage&loc=14&r=32&doct=0&rfunc=0&tj=none&tr=32 |title=瑞丽至孟连将建高速 全长520.17公里 |last= |first= |date=2017-12-05 |website=k.sina.cn |publisher=新浪看点 |access-date=2017-12-10 |quote=}}</ref>

==路线==

瑞孟高速经过[[德宏傣族景颇族自治州|德宏]]、[[保山市|保山]]、[[临沧市|临沧]]、[[普洱市|普洱]]4个州市共9个县份。起点位于[[瑞丽市|瑞丽市]]城北的姐勒互通，与[[瑞陇高速|瑞陇高速]]衔接，与[[杭瑞高速|杭瑞高速]]共线75千米至[[芒市|芒市]]户育枢纽，连接在建的[[芒梁高速|芒梁高速]]，后转向南，沿[[芒市坝|芒市坝]]外沿绕向东，后穿隧道抵达[[龙陵县|龙陵县]][[象达镇|象达镇]]。向南经[[平达乡|平达乡]]、[[勐糯镇|勐糯镇]]，在田坡村附近设怒江特大桥跨越怒江，之后设12.5公里特长隧道穿越山梁到达[[镇康县|镇康县]][[勐捧镇_(镇康县)|勐捧镇]]，向西南抵达[[镇康县|镇康县]]城。芒市至镇康段新建里程为148.175公里。<ref name="路线">{{cite web |url=http://news.sina.com.cn/o/2017-09-06/doc-ifyktzim8384532.shtml |title=好消息！瑞丽至孟连将建高速 全长520公里 看看经过你家没？ |last= |first= |date=2017-09-06 |website=news.sina.com.cn |publisher=云南网 |access-date=2017-12-10 |quote=}}</ref><ref>{{cite web |url=http://www.cnepaper.com/dhtjb/html/2017-09/11/content_1_3.htm |title=瑞丽至孟连拟建高速 |last= |first= |date=2017-09-11 |website=www.cnepaper.com |publisher=德宏团结报 |access-date=2018-01-04 |quote=}}</ref>

高速从镇康向南，与在建的[[镇清高速|镇清高速]]共线，共线里程34.222公里。到南捧村与在建的[[临清高速|临清高速]]衔接，后与临清高速共线，向东北至[[勐撒镇|勐撒镇]]，共线里程77.603公里。<ref name="路线"></ref>

后从勐撒折向西南，经过[[耿马傣族佤族自治县|耿马]]县城后向南，在[[糯良乡|糯良乡]]设立沧源互通，另建连接互通与[[沧源佤族自治县|沧源]]县城的一级路。高速折向东南到[[孟连傣族拉祜族佤族自治县|孟连]]，向南到[[景信乡|景信乡]]郎勒互通与规划中的[[孟富高速|孟富高速]]衔接。勐撒至孟连段新建185.17公里。<ref name="路线"></ref>

全线共设南相章、法帕、芒市北、象达、平达、勐糯、勐捧、耿马北、 耿马南、勐省、沧源、孟连12条互通立交配套连接线，共长80.495公里，其中一级公路50.68公里、二级公路长29.82公里。<ref name="云南高速通"></ref>
