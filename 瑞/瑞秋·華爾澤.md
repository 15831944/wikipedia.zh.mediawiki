'''瑞秋·華爾澤'''（'''Rachel Walzer'''）是[[美國|美國]]出身的[[女性|女性]][[演員|演員]]。

== 人物來歷 ==
在佛蒙特州的國際教育大學完成碩士學位。在1990年來到日本，此後活躍於日本。自1994年在[[青山學院大學|青山學院大學]]擔任戲劇指導。

有美國與[[以色列|以色列]]的[[多重國籍|雙重國籍]]<ref>{{cite web |url=http://soliconsultants.com/female/487 |title=存档副本 |accessdate=2012年6月6日 |deadurl=yes |archiveurl=https://web.archive.org/web/20120316073339/http://soliconsultants.com/female/487 |archivedate=2012年3月16日 |df= }} 公開簡歷。</ref>。<ref>{{cite web |url=http://talent.yahoo.co.jp/pf/detail/pp462581 |title=存档副本 |accessdate=2012-06-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304230350/http://talent.yahoo.co.jp/pf/detail/pp462581 |archivedate=2016-03-04 }} 日本雅虎公開簡歷。</ref>。特技是[[希伯來語|希伯來語]]、[[日語|日語]]。
