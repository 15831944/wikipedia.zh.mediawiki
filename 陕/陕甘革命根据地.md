'''陕甘革命根据地'''或称'''西北根据地'''<ref name="a1"/>形成于1935年，由[[陕北根据地|陕北根据地]]与[[陕甘边根据地|陕甘边根据地]]组成。9月间，[[长征|长征]]中的[[毛泽东|毛泽东]]得知相关消息后，最终[[中央红军|中央红军]]（红一方面军）将陕甘根据地定为目的地<ref>{{Cite web |url=http://history.people.com.cn/GB/217006/217939/15882014.html |title=长征落脚点选在陕北 毛泽东为何向国民党的报纸致谢？ |accessdate=2016-06-01 |author=韩三洲，责任编辑：吴皓 |date=2011-10-13 |work= |publisher=人民网，原载于《同舟共进》2011年第9期|language=Zh-hans }}</ref>。1936年5月，红一方面军进行[[西征战役|西征]]，陕甘根据地扩展为[[陕甘宁根据地|陕甘宁根据地]]。10月，红一、[[红二方面军|二]]、[[红四方面军|四方面军]][[会师|会师]]后抵到陕甘宁根据地<ref name="a2"/>。

== 建立 ==

1935年1月21日，陕北根据地创建者[[刘志丹|刘志丹]]与[[谢子长|谢子长]]会面，两人对[[陕北根据地|陕北根据地]]与[[陕甘边根据地|陕甘边根据地]]的统一领导达成一致。2月5日，在[[赤源县|赤源县]]（今陕西省[[子长县|子长县]]）周家崄的刺儿圪塔村举行中共陕甘边特委和陕北特委联席会议，成立中共西北工委和西北军委。[[惠子俊|惠子俊]]任工委书记，刘志丹任军委主席。标志两个根据地的政治领导权、军事指挥权的统一<ref name="a1">{{Cite web |url=http://www.htqly.org/detail.aspx?Id=49 |title=陕甘边、陕北革命根据地简介 |accessdate=2016-06-01 |author= |date= |work= |publisher=西北革命历史网|language=Zh-hans }}</ref>。

同年，在刘志丹的领导下，[[西北红军|西北红军]]攻占[[安定县_(元朝)|安定县]]、[[延长县|延长县]]、[[延川县|延川县]]、[[安塞县|安塞县]]、[[靖边县|靖边县]]、[[保安县_(金朝)|保安县]]六座县城，使陕北根据地与陕甘边根据地连成一体，形成陕甘根据地<ref name="a2">{{Cite web |url=http://dangshi.people.com.cn/n/2013/0618/c85037-21884537.html |title=陕甘边革命根据地的军事斗争 |accessdate=2016-06-01 |author=整理：李俊杰、许瑞源，责编：程宏毅、常雪梅 |date=2013-06-18 |work= |publisher=中国共产党新闻网，来源：甘肃日报|language=Zh-hans }}</ref>。
