'''陕西汽车控股集团有限公司'''（简称'''陕汽控股'''），总部位于[[陕西省|陕西省]][[西安市|西安市]]<ref name="a1">{{Cite web |url=http://www.sxqc.com/jtjj/index_13.aspx|title=集团简介|accessdate=2018-11-15 |author=|date=|work= |publisher=陕西汽车控股集团有限公司网站|language=Zh-hans }}</ref>，是[[陕西省人民政府国有资产监督管理委会员|陕西省人民政府国有资产监督管理委会员]]监管的省属企业<ref>{{Cite web |url=http://www.sxgz.gov.cn/newstyle/pub_newsshow.asp?id=2114252&chid=100057|title=西安市与省属企业投资合作座谈会成功举办|accessdate=2018-11-15 |author= |date=2018-10-24 |work= |publisher=陕西省人民政府国有资产监督管理委会员网站|language=Zh-hans }}</ref>。主要从事重型军用越野车、重型卡车、中轻型卡车、大中型客车、微型车、重微型车桥、[[康明斯|康明斯]]发动机及其零部件的开发、生产、销售及汽车金融业务<ref name="a1"/>。

集团前身是1968年创建的陕西汽车制造厂。现有员工2.8万人，总资产533亿元。集团下辖'''陕西汽车集团有限责任公司'''（简称'''陕汽集团'''）、陕西汽车实业有限公司、金龙汽车（西安）有限公司等100余家参控股子公司<ref name="a1"/>。

== 子公司——陕汽集团 ==

'''陕西汽车集团有限责任公司'''（简称'''陕汽集团'''）是中国最大的[[重卡|重卡]]生产商之一。雇员超过1万5千人。

=== 参股、控股子公司 ===

陕汽集团拥有[[陕西重汽|陕西重汽]]、宝鸡华山工程车辆公司、陕西欧舒特汽车股份有限公司、西安康明斯发动机公司、陕西汉德车桥公司、陕西重汽汽车零部件公司等15个参股控股子公司<ref name="a2">{{Cite web |url=http://finance.sina.com.cn/stock/newstock/zxdt/20091016/21506849674.shtml|title=陕汽集团与潍柴争夺控股权谋求上市|accessdate=2018-11-15 |author=记者：郑重 |date=2009-10-16|work= |publisher=新浪网|language=Zh-hans }}</ref>。通家是陕汽的下属企业，主要生产[[微型客车|微型客车]]。

=== 陕西重汽控制权之争 ===

在陕汽集团诸多子公司中，上市公司[[陕西重汽|陕西重汽]]是陕汽集团最为优质的资产。[[潍柴动力|潍柴动力]]则通过抢购[[湘火炬|湘火炬]]进而控股陕西重汽<ref>{{Cite web |url=http://auto.sohu.com/20100707/n273353376.shtml|title=潍柴独霸 陕汽一直无法击破三大困局|accessdate=2018-11-15 |author=裴达军 |date=2010-07-07|work= |publisher=搜狐网，来源：《中国商人·汽车人》杂志|language=Zh-hans }}</ref>。2009年时，在陕西省国资委的支持下，陕汽集团正与潍柴动力争夺陕西重汽的[[控股权|控股权]]<ref name="a2"/>。
