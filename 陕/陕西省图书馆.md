'''陕西省图书馆'''（'''Shaanxi Library'''，简称'''陕图'''），创建于[[清|清]]·[[宣统|宣统]]元年八月（即公元1909年9月），是中国成立最早的省级图书馆之一，也是[[陕西省|陕西省]]最大的公共图书馆。目前，图书馆拥有一所总馆，少年儿童分馆、陕西省艺术学校分馆等19个分馆，以及15个流动服务点。截至2014年底累计馆藏总量达444.67万余册（件），其中图书343.84万余册，古籍32万余册，报刊56万余件。另有电子图书107万余册，视听文献、缩微制品等其他文献11万余件。目前年入藏新书29万余册，年订中外文印刷型报刊5000余种。截至2015年底，陕西省图书馆设有23个部(科)，在职职工176人，其中专业技术人员166人<ref>{{Cite web |url= http://news.cnwest.com/content/2009-07/09/content_2206874.htm |title= 西安十大城市地标建筑候选：陕西省图书馆 |author=  |date= |publisher=  |language= 中文 |accessdate= 2012-12-15  }}</ref>。

2001年9月30日，陕西省图书馆位于[[西安市|西安市]]长安北路与南二环路交汇处的新馆正式对外开放。新馆紧邻[[陕西省体育场|陕西省体育场]]，占地2.03公顷，建筑面积4.2万平方米；主楼12层，裙楼6层，设计藏书容量400万册，阅览座位2000个，<ref>{{Cite web |url= http://www.sxlib.org.cn/stgk/index.htm |title= 陕图简介 |author=  |date= |publisher=  |language= 中文 |accessdate= 2012-07-25 |quote= 新馆位于长安北路与南二环路交汇处，占地2.03公顷，建筑面积4.2万平方米，主楼12层，裙楼6层，设计藏书容量400万册，阅览座位2000个。 }}</ref>是陕西省最大的综合性、研究型公共图书馆。2005年、2010年、2013年，陕图被[[文化部|文化部]]评定为'''[[国家一级图书馆|国家一级图书馆]]'''<ref>{{Cite web |url= http://www.shaanxichina.org/web/11041018/stsg/ |title= 陕西省图书馆简介 |author=  |date= 2012-06-25 |publisher= 《网上陕西》 |language= 中文 |accessdate= 2012-07-25 |quote= 2005年、2010年两次被国家文化部命名为“一级图书馆”。 |deadurl= yes |archiveurl= https://web.archive.org/web/20121001113650/http://www.shaanxichina.org/web/11041018/stsg/ |archivedate= 2012-10-01 }}</ref>。

==历史沿革==
*[[清|清]][[宣统|宣统]]元年八月（公元1909年9月）创馆，附设于西安梁府街（今青年路）的学务公所内。
*1915年，迁馆至西安市[[南院门|南院门]]。
*1927年，更名为“'''陕西省立中山图书馆'''”。
*1931年7月，更名为“陕西省立第一图书馆”。
*1937年4月，更名为“陕西省立西京图书馆”。
*1949年5月，中共进驻西安后，更名为“陕甘宁边区西安图书馆”。
*1950年4月，更名为“陕西省西安人民图书馆”。
*1953年7月2日，定名为“陕西省图书馆”。
*2001年9月30日，位于西安市长安北路与南二环路交汇处的新馆正式对外开放。

==各馆介绍==
===总馆===
2001年9月30日对外开放。总馆设各类借阅区和阅览室16个，一年365日天天开放，每天连续开放12小时。同时，总馆拥有报告厅、多功能厅、展览厅、多媒体教室、读者餐厅等设施；并采用无线射频识别（RFID）技术进行业务管理与服务。

===少年儿童分馆===
专为3至16岁的少年儿童提供少儿图书；实行无证阅览、持证外借的方法。馆址位于西大街南院门53号，陕西省图书馆的旧址。2005年6月1日正式对外开放。馆舍面积近1600平方米，阅览座位172个，设有儿童借阅室、少年借阅室。

===省艺术学校分馆===
2006年3月建成，目前主要面向陕西省艺术学校的学生开放。

==对外交流==
*陕西省图书馆是[[联合国教科文组织|联合国教科文组织]]出版物和美国高等教育资料在中国的收藏单位之一。
*与[[美国|美国]]国会图书馆、[[日本|日本]]奈良县立图书馆、[[荷兰|荷兰]]莱登大学汉学研究院图书馆等，建立了国际书刊交换关系。
