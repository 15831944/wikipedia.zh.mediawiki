'''陕南行政区'''或称'''陕南行署区'''、'''陕南区'''，是中共政权于1948年6月7日到1951年2月22日，在[[陝西省_(中華民國)|陕西]]、[[湖北省_(中華民國)|湖北]]设置的一个[[行署区|行署区]]，亦是省级行政区<ref>{{Cite web |url=http://www.htq.gov.cn/zjhs/lisyg/lsyg/117360.htm |title=历史上的陕南公学 |accessdate=2019-03-06 |author= |date=2018-10-24 |work=发布人：信息中心-侯晓斐|publisher=汉中市汉台区人民政府网站，来源：汉中日报|language=Zh-hans }}</ref>。1949年时，下辖[[南郑市|南郑市]]及[[南郑分区|南郑]]、[[安康分区|安康]]、[[商雒分区|商雒]]、[[两郧分区|两郧]]四个分区，所辖区域已并入[[陕西省|陕西省]]和[[湖北省|湖北省]]<ref name="a1">{{Cite web |url=http://www.sohu.com/a/197436461_799711 |title=沙建国【岁月留痕】红旗飘飘 —— 陕南区暨汉中解放初期纪事 |accessdate=2019-03-06 |author=沙建国 |date=2017-10-11 |work=|publisher=搜狐网|language=Zh-hans }}</ref>。

1948年6月，奉[[中共中央中原局|中共中央中原局]]、[[中原军区|中原军区]]命令，在湖北省[[郧县|郧县]]组建陕南军区、陕南区党委、陕南行政主任公署。[[刘金轩|刘金轩]]任军区司令员、[[陈先瑞|陈先瑞]]任副司令员，[[汪锋|汪锋]]任军区政治委员、陕南区党委书记，行署主任为[[时逸之|时逸之]]。当时，陕南行政区拥有二百余万人口<ref name="a1"/>。

1949年春，[[张邦英|张邦英]]任陕南区常委第一书记兼军区第一政委。5月1日，奉中原军区命令，陕南军区成立[[中国人民解放军第19军|中国人民解放军第19军]]。刘金轩任军长，张邦英兼任第一政委，汪锋兼任政委，陈先瑞任副军长。6月，依中共中央电令，由中原局改为[[西北局|西北局]]、[[西北军区|西北军区]]领导。此后，解放军的军事行动，使辖区进一步扩大。[[中国人民解放军第19军|解放军第19军]]55师，于11月28日攻占[[岚皋县|岚皋县]]，30日攻占[[紫阳县|紫阳县]]，12月17日攻占[[镇巴县|镇巴县]]。第19军57师于11月26日攻占[[洵阳县|洵阳县]]，27日攻占[[安康县|安康县]]，12月4日攻占[[西乡县|西乡县]]，7日攻占[[城固县|城固县]]。12月2日，解放军18兵团攻占[[留坝县|留坝县]]，6日，攻占[[褒城县|褒城县]]、[[沔县|沔县]]、南郑城。7日，[[中国人民解放军第7军|解放军第7军]]攻占[[略阳县|略阳县]]，8日，咸阳军区部队攻占[[佛坪县|佛坪县]]，11日，入川部队攻占[[宁强县|宁强县]]<ref name="a1"/>。

1950年，安康分区改名[[安康专区|安康专区]]，属陕西省管辖。3月，两郧分区划入湖北省，为[[两郧专区|两郧专区]]。5月1日，撤消南郑地委、专署、军分区，所辖县、区由陕南行署区直辖。2日，商雒分区改名为商雒地区专员公署（[[商雒专区|商雒专区]]），属陕西省。年底，第19军军长刘金轩兼任陕西军区代司令员，[[马明芳|马明芳]]任政治委员<ref name="a1"/>。

1951年2月22日，经[[中央人民政府政务院|中央人民政府政务院]]批准，撤销陕南区党委、军区、行署。同时，成立陕西省人民政府南郑区专员公署（[[南郑专区|南郑专区]]），和安康专区直属陕西省管辖。
