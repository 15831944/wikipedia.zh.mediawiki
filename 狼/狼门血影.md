《'''狼门血影'''》（{{lang-en|''The Gates''}}，又译：'''灵异社区'''）是一部由Richard Hatem和Grant Scharbo创造的[[美国|美国]][[电视剧|电视剧]]，于2010年6月20日到2010年9月19日期间在[[美国广播公司|美国广播公司]]播出。

== 简介 ==
Nick Monohan和他的家人从芝加哥搬家到了一个安静的、高档的名为盖茨（Gates）社区，在那里他将是警察局长。他们很快意识到，自己的邻居并非是表面上看出去的样子<ref name=abcmedia>{{Cite web |url=http://www.abcmedianet.com/web/showpage/showpage.aspx?program_id=002988&type=lead  |title=The Gates: Show Description |work=ABCmedianet |publisher=ABC Television Network |accessdate=April 27, 2010}}</ref><ref name=futonMar2510>{{Cite web |url=http://www.thefutoncritic.com/news/2010/03/25/development-update-thursday-march-25/8588/ |title=Development Update: Thursday, March 25 |date=March 25, 2010 |publisher=The Futon Critic |accessdate=April 26, 2010}}</ref><ref name=futonFeb1010>{{Cite web |url=http://www.thefutoncritic.com/news/2010/02/10/development-update-wednesday-february-10/8522/ |title=Development Update: Wednesday, February 10 |date=February 10, 2010 |publisher=The Futon Critic |accessdate=April 26, 2010}}</ref> 。盖茨社区充满了诸如吸血鬼、女巫，狼人，和一个女妖。

== 演员和角色 ==
*[[:en:Luke_Mably|路克·馬柏利]] 饰 Dylan Radcliff，[[吸血鬼|吸血鬼]]。
*[[蘿娜·米莎|蘿娜·米莎]] 饰 Claire Radcliff，吸血鬼, Dylan的妻子。
*[[法蘭克·葛里洛|法蘭克·葛里洛]] 饰 Nick Monohan，盖茨镇新警察局长。
*[[:en:Marisol_Nichols|Marisol Nichols]] 饰 Sarah Monohan，Nick的妻子
*[[:en:Travis_Caldwell|Travis Caldwell]] 饰 Charlie Monohan，Nick的儿子
*[[:en:McKaley_Miller|McKaley Miller]] 饰 Dana Monohan，Nick的女儿
*[[:en:Chandra_West|Chandra West]] 饰 Devon Buckley, 一个坏女巫，剧集的主要反面角色。
*Victoria Platt 饰 Dr. Peg Mueller, 一个好女巫，Devon的克星。
*[[絲凱樂·山繆斯|絲凱樂·山繆斯]] 饰 Andie Bates，女[[魅魔|魅魔]]（succubus）。
*[[科尔顿·海恩斯|科尔顿·海恩斯]] 饰 Brett Crezki，[[狼人|狼人]]。
*Justin Miles 饰 Marcus Jordan，在盖茨镇警察局工作。
*[[:en:Janina_Gavankar|Janina Gavankar]] 饰 Leigh Turner，在盖茨镇警察局工作，但是她有一些秘密。
*[[:en:James_Preston_(actor)|James Preston]] 饰 Lukas Ford，狼人。
