'''狼山'''，位于[[江苏省|江苏省]][[南通市|南通市]]南郊，是[[江苏省省级风景名胜区|江苏省省级风景名胜区]]，[[国家AAAA级旅游区|国家AAAA级旅游区]]，被称为“江海第一山”<ref>{{cite web |title=谒骆宾王墓(图)_网易新闻 |url=http://news.163.com/11/0823/07/7C4HBN4I00014AED.html |website=news.163.com |accessdate=2019-03-05}}</ref>。與[[马鞍山|马鞍山]]、[[剑山|剑山]]、[[军山|军山]]和[[黄泥山|黄泥山]]组成，合称[[狼五山|狼五山]]。為中国[[佛教八小名山|佛教八小名山]]之一<ref>{{cite web |title=南通狼山风景区桂花飘香 |url=http://news.sina.com.cn/c/2003-10-05/0920865130s.shtml |website=news.sina.com.cn |accessdate=2019-03-05}}</ref>，同时狼山也是天主教在中国的十二大朝圣地之一<ref>{{cite web |title=江苏这些山值得爬 - 网易苏州 |url=http://js.news.163.com/suzhou/15/0320/11/AL58RVLU03300PO7_7.html |website=js.news.163.com |accessdate=2019-03-05}}</ref>。

其[[海拔|海拔]]106.94米，於狼五山中最为峻拔挺秀，其他四山如众星拱月，狼山成为五山之首。[[狼山风景区|狼山风景区]]共有43处古建筑、8处古墓、23处题刻石刻、13处古洞穴。狼山之北的[[狼山圣母堂|狼山圣母堂]]为苏北重要的[[天主教|天主教]]活动场所，1930年代，罗马教廷几下诏书颁赐“全大赦”使狼山列入中国天主教十二大朝圣地之一，在远东地区都有着崇高的宗教地位和影响力，每年有大量的天主教徒寻访而来，到此朝拜。

=== 剑山 ===
剑山又名剑脊山，高87.33米，海拔74.8米，位于狼山东南，军山之西，其高度在狼五山中排在第三的位置。整座山[[山脊|山脊]]突如卧置出鞘的宝剑，相传[[秦始皇|秦始皇]]曾经在剑山东南的一块石头上磨淬过宝剑，故而得名剑脊山。[[刘名芳|刘名芳]][[《南通州五山全志》|《南通州五山全志》]]中载：“剑山山脊若剑，一名剑脊，亦谓祖龙尝于此淬剑。” 

=== 马鞍山 ===
马鞍山是[[隋炀帝|隋炀帝]]征辽放马的马鞍山<ref>{{cite web |title=城市名片_新闻中心_新浪网 |url=http://news.sina.com.cn/o/2006-09-19/004810048935s.shtml |website=news.sina.com.cn |accessdate=2019-03-05}}</ref>。

=== 军山 ===
军山为海中岛屿时，相传曾为秦王屯兵之地，因形似伏象，又名象山。军山上古迹有：燕真人洞、白云泉、四贤祠、包公祠、董其昌碑刻及中国历史上最早的民办气象台等。

==宗教文化==
===佛教===
{{seealso|大势至菩萨#江苏狼山}}
狼山的开山祖师是[[僧伽大圣|僧伽大圣]]禪師，在这里开始奉祭大势至菩萨。传说当时此山为白狼精占据，故稱狼山，僧伽与白狼精斗法，以一袭袈裟遮遍全山，降伏白狼，白狼只得让出此山。并无私心的僧伽，在狼山上建寺，即今日的广教寺，为大势至菩萨建了首座道场。

广教寺始建于唐总章二年(669年)，是一座有1300多年历史的古刹。据《通州志》记载：“唐总章二年，山上即建大雄宝殿、殿阁、方丈室”，当时“山在巨浸中，设舟以济，号慈航院，后改广教寺。”狼山奉祀的开山祖师是僧伽，又称狼山大圣、大圣菩萨，爲唐代高僧。唐高宗时，他曾到长安、洛阳游历，为人治病，名声大噪。南游江淮时，医病治水，为百姓称道。唐中宗尊为国师。后世尊他为“大圣菩萨”。宋朝太平兴国年间（976-983年），智幻法师住持广教寺，他弘法创业，修建寺庙，主持建造了大圣殿、支云塔，并塑僧伽像进行供奉，此后江淮一带许多寺庙供奉僧伽像。智幻法师圆寂时，留下一首偈言：“当初不肯住长安，现像西归泗水间，今日又还思展化，东来海上镇狼山。”后人称他为僧伽化身。为纪念智幻法师，明嘉清年间在寺内建幻公塔，至今保存。
