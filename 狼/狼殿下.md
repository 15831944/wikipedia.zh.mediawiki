《'''狼殿下'''》（{{lang-en|'''''The Wolf'''''}}），是一部[[中國大陸電視劇列表_(2018年)|2018年中國大陸電視劇]]。[[陳玉珊|陳玉珊]]任總製作人和總導演，王為任導演，主要演員有[[王大陆|王大陆]]、[[李沁|李沁]]、[[辛芷蕾|辛芷蕾]]、[[肖戰|肖戰]]、[[郭書瑤|郭書瑤]]、臧洪娜和[[林佑威|林佑威]]<ref>{{cite web |url=http://news.yule.com.cn/html/201808/277784.html |title=《狼殿下》的主要演員 |date=2018-08-17 |publisher=中國娛樂網}}</ref>。《狼殿下》是原創的故事，以五代十國[[後梁|後梁]][[後晉|後晉]]的真實故事作為歷史背景，講述後梁奎州城郡主摘星與生長於狼群中的少年相識，錯別八年後再次相遇，少年已成為狠厲的渤王，兩人需在權謀中為了愛而力戰天命。

預計於2018年下半年播出。

==劇情簡介==
唐朝末年，[[朱溫|朱溫]]篡位稱帝，建立[[後梁|後梁]]。後梁奎州城主馬瑛之女摘星，結識了自小生長於山林間的少年。不諳世事的少年為救幼狼遭到追殺，失足墜崖，被[[朱溫|朱溫]]收為義子，封為渤王。八年後，渤王偶遇摘星，他喜歡摘星的聰慧與勇敢；摘星也發現，出身草根的渤王雖然身居高位，卻依然保有著天性中的善良和正義感。在摘星的支持下，渤王反對苛政、扶助百姓、匡扶正義、阻止兄弟相爭。在歷經重重的磨難和考驗之後，兩人終於有了屬於自己的幸福。

==製作==
2015年7月，[[國家廣播電視總局|廣電總局]]備案<ref>{{cite web |url=http://dsj.sarft.gov.cn/tims/site/views/include/quickSearch/checkBlueprint/result.shanty?ids=014e008fb1db0aa34028e4a04dfbf323 |title=《狼殿下》廣電總局備案 |date=2015-07 |website= [[中華人民共和國國家新聞出版廣電總局|廣電總局]]}}</ref>。

===劇本===
《狼殿下》是原創故事，以五代十國為歷史背景<ref>{{cite web |url=https://www.facebook.com/pg/frankie.chen.official |title=陳玉珊導演於2017年4月9日的發文 |date=2017-04-09 |website=[[臉書|臉書]]}}</ref>，整個劇本寫了2年半，於2018年6月4日完成最後一集劇本<ref>{{cite web |url=https://www.facebook.com/frankie.chen.1694 |title=陳玉珊導演(Yu Shan Chen)於2017年6月4日的發文  |date=2017-06-04 |website=[[臉書|臉書]] }}</ref>。

===選角===
陳玉珊導演表示：“王大陸那種男性荷爾蒙感覺，皮膚黑黑、很健康的男生就特別像劇中的狼仔。肖戰是近期在綜藝節目裡看到他的表現特別好，我們想把他改造成不那麼只走顏值的演員。李沁這一次的挑戰是一個女生的成長史，我特別相信她有這個能力，成為一個戰神般的女孩子<ref>{{cite web |url=http://www.kaixian.tv/gd/2017/0410/154680.html |title=《狼殿下》的選角 |author=馬軍艷 |date=2017-04-1 |website=www.kaixian.tv |publisher=春城晚報 }}</ref>。”

===拍攝===
2017年4月9日，《狼殿下》在[[雲南|雲南]][[香格里拉市|香格里拉市]]的[[普達措國家公園|普達措國家公園]]開機拍攝，主演李沁、肖戰、郭書瑤、臧洪娜等人，共同參加了開機儀式<ref>{{cite web|url=http://xw.kunming.cn/a/2017-04/09/content_4580365.htm |title=《狼殿下》開機拍攝 |author=袁思思 |date=2017-04-09 |publisher=昆明信息港 }}</ref><ref>{{cite web |url=http://ent.cri.cn/20170410/e2699c06-dcad-39ed-edc8-65bde8841572.html |title=《狼殿下》開機 |author=徐明輝 |date=2017-04-10 |publisher=國際在線娛樂 }}</ref><ref>{{cite web |url=http://www.kaixian.tv/gd/2017/0410/154680.html |title=3億打造《狼殿下》|author=馬軍艷 |date=2017-04-10 |publisher=漢豐網 }}</ref>。拍攝地點從雲南香格里拉市到[[浙江|浙江]][[象山影視城|象山影視城]]，歷經五個多月，共163天<ref>{{cite web |url=https://www.weibo.com/1809054937/Fmw0JvQ9F?type=comment#_rnd1534602815960 |title=《狼殿下》拍攝地點和周期 |author=[[李沁|李沁]] |date=2017-09-19 |website=[[新浪微博|微博]]}}</ref>。
