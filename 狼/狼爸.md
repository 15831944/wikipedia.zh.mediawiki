'''狼爸'''是家庭一种对于父亲对儿子或女儿产生[[棍棒教育|棍棒教育]]或语言暴力教育的一种，是属于家庭暴力的范畴。这一种教育行为是父亲对子女产生任何生活、物质主义、传统和社交的限制性，目前该行为已经受到中国社会质疑。因为以棍棒或语言暴力出孝子为法宝<ref>{{cite web |url=http://baby.sina.com.cn/z/humalangba/ |title=虎妈、狼爸、猫爸式的教育反思 |publisher=新浪教育}}</ref>，已经引起中国国产教育模式的讨论。

== 产生狼爸的原因和后果 ==
主要的就是有以下四个原因：<br />
1. 由于中国的[[專制|專制]]統治时期过长，加上当时的[[棍棒教育|棍棒教育]]的傳統思想观念根深蒂固，导致家长的子女出现歪曲或唱反调的人格、还有逆反的心理；<br />
2. 在[[20世纪70年代|20世纪70年代]]末至今，由于[[应试教育|应试教育]]的逼迫，加上家长急于望子成龙或望女成凤的心态，以及现实评价体系重视知识和分数，相反反而轻视综合素质、实践能力和人格<ref>{{cite web |url=http://learning.sohu.com/20051215/n240971528.shtml |title=应试教育造成的十大恶果 |publisher=搜狐教育}}</ref>，反而助长了子女做出很多反社会（例如：因成绩不满意而自杀<ref>{{cite web |url=http://edu.qq.com/a/20120502/000221.htm |title=2名少女相约自杀 留遗书称成绩差看不到未来 |publisher=腾讯教育 |date=2012年5月26日}}</ref><ref>{{cite web |url=http://news.qq.com/a/20091026/000972.htm |title=16岁高二女生因考试成绩不理想割腕自杀 |publisher=腾讯 社会万象 |date=2009年12月26日}}</ref>）的事情；<br />
3. 社会的跟风和推波助澜，助长了新的事物出现而吸引了子女，但家长不满意子女的表现而出现的暴力状况<ref>{{cite web |url=http://tv.sohu.com/20121010/n354560890.shtml |title=温州“狼爸”惩罚6岁女儿跑步6小时致死 |publisher=搜狐视频（转自辽宁卫视） |date=2012年10月10日}}</ref>；<br />
4. 最近的商业推手的唯利是图，导致家长的子女出现多新鲜的物质主义吸引；

== 对狼爸的各种看法 ==
=== 支持 ===
该行为的可喜之处是按照目前实行的传统教育而言，目的是对孩子的行为举止各方面严格规范，符合伦理道德。<ref>{{cite web |url=http://kid.baby.sina.com.cn/2011-11-16/081149685.html |title=“打进北大”不证明“狼爸”有理  |publisher=新华网 |date=2011年11月16日}}</ref>

=== 反对 ===
该教育的本质上是用暴力强迫孩子服从家长的意志来培养孩子，只会成为唯唯诺诺，没有独立思想的人<ref>{{cite web |url=http://kid.baby.sina.com.cn/2011-11-16/081149685.html |title=“打进北大”不证明“狼爸”有理  |publisher=新华网 |date=2011年11月16日}}</ref>。如果说按棍棒教育的方法来管孩子就能成功，那么天下父母都可以是“成功父母”了。其中，狼爸的本质是属于一种奴性教育<ref>{{cite web |url=http://kid.baby.sina.com.cn/2011-11-16/081149685.html |title=狼爸的奴性教育受到质疑  |publisher=新浪网 |date=2011年11月16日}}</ref>，不仅扭曲孩子的心灵，向北大输送的未必就是有创造力的人才，更有违法之嫌<ref>{{cite web |url=http://news.ifeng.com/opinion/gundong/detail_2011_11/16/10692549_0.shtml |title=梁江涛：千万不要让“中国狼爸” 建私塾  |publisher=凤凰网 |date=2011年11月16日}}</ref>。被调教得不敢越雷池一步的“好”孩子、“好”学生，无论考试成绩如何光鲜，却鲜少拥有冒险求变的内在力量。他们中间又怎会诞生中国的[[比尔盖茨|比尔盖茨]]和[[乔布斯|乔布斯]]？正是这种教育方式造成人才凋萎的后果，引来沉痛的[[钱学森|钱学森]]之问。假如他真是棍棒教育那样简单粗暴地对待孩子，那他的孩子一定是有问题的，即使学习成绩好，心理也不会健康。”有专家也认为，孩子倘若没有幸福的童年，就难有幸福的人生<ref>{{cite web |url=http://d.wanfangdata.com.cn/Periodical_hqcj201112048.aspx |title= 三天一顿打，孩子进北大 中国需要“狼爸式”教育吗？  |publisher=万方数据}}</ref>。自己是主，孩子是民———有这样的爸爸，注定没有幸福的童年。没有幸福的童年，很难有一个幸福的人生———因为童年的经历影响人的一生。其可怕之处在于：孩子们可能要花很多时间才能去除头脑里的“臣民”意识，认同现代社会的基本价值观，明白民主、自由这样基本的现代价值的重要性并维护它，具备公民意识和品格，成为国家公民。因此，为避免出现棍棒教育出现狼爸的情况，只能在子女小时悉心培养调教，避免孩子叛逆厌世，难以管教<ref>{{cite web |url=http://news.ifeng.com/society/special/langba/content-3/detail_2011_11/22/10821856_0.shtml |title=华人热议“狼爸”:美国孩子“伤不起”管教打不得  |publisher=凤凰网 |date=2011年11月22日}}</ref>。孩子是不能太宠太溺爱，适当体罚、防止孩子做错事确有必要，但这不是无理由的打孩子，用棍棒逼他们学习的理由。否则，即使他们在应试教育中获得成功，也未必会成为对社会真正有用的人才，充其量只是死读书的料子。<ref>{{cite web |url=http://news.ifeng.com/opinion/society/detail_2011_11/16/10688458_0.shtml |title=杨国栋：“狼爸”的成功算不上经验  |publisher=凤凰网 |date=2011年11月16日}}</ref>
