《'''狼王羅伯'''》（{{lang-en|Lobo the King of Currumpaw}}）是收錄於美國作家[[歐尼斯特·湯普森·西頓|歐尼斯特·湯普森·西頓]]發表在1898年的創作《西頓動物記》（Wild Animals I Have Known）裡的一篇故事。內容是描述西頓過去在[[新墨西哥州|新墨西哥州]]科倫坡（Currumpaw）一帶，受人委託追捕一頭頭腦精明，入侵多處牧場進行獵殺家畜而引發危害，被當地人稱作為「羅伯」（Lobo）<ref group="註">「Lobo」在[[西班牙語|西班牙語]]的意思即為「狼」。</ref>的[[灰狼|灰狼]]之過程經歷。

作品推出後曾被[[華特迪士尼公司|華特迪士尼公司]]改編為[[電影|電影]]，在日本當地也有改編為[[電視動畫|電視動畫]]、[[漫畫|教育漫畫]]，[[舞台劇|舞台劇]]等多種媒體作品。

== 事蹟過程 ==
[[File:Lobo_(The_King_of_Currumpaw).jpg|thumb]]
1890年期間，於美國新墨西哥州科倫坡北部一帶的河谷裡，有一頭被當地人稱為羅伯的灰狼，牠多次帶領狼群攻擊科倫坡各牧場的羊群家畜。當地人雖曾試著用槍枝或毒餌等方法來對付羅伯，但都被羅伯給躲避掉或識破。

1893年秋季西頓受了委託，前往科倫坡找尋對付羅伯的方法。起初西頓曾利用毒餌去除人味的方法，不過並沒有騙到羅伯。之後西頓購買了獸夾，設法在獸徑上暗藏陷阱，但也遭到羅伯看穿。

毒餌和陷阱都對付不了羅伯的情況下，西頓修改了策略，改朝向羅伯的妻子，一頭被稱為「布蘭卡」（Blanca）<ref group="註">「Blanca」在西班牙語的意思即為「白色」。</ref>的白色母狼下手。在西頓抓到了布蘭卡並處死後，想在羅伯情緒混亂的狀態下補捉牠，便擺置了一百個以上的獸夾陷阱在布蘭卡的屍體附近等候。

雖然西頓在捕抓到羅伯時並未當場將牠處死，但羅伯在喪失生存意志的情況下拒絕被餵食，再被捕抓沒多久的某日凌晨便死去。

1898年西頓將他過去所接觸到野生動物事蹟或其它故事寫成《西頓動物記》，並把羅伯的事件收錄至該著作裡。羅伯死後遺留的毛皮現今保留在位於新墨西哥州[[科爾法克斯縣_(新墨西哥州)|科爾法克斯縣]]的西頓紀念館裡。<ref>{{cite web |url=http://www.philmontscoutranch.org/Museums/SetonMemorialLibrary/lobo.aspx |title=Lobo, The King Of Currumpaw |work=Boy Scouts of America - Philmont |language=en |date=年月日 |accessdate=2017年12月30日}}</ref>

狼王羅伯的奇特之處在於除了過人的智慧外，還有牠並不因為獵食而襲擊人類牧場：據記載牠曾率領狼群在一夜之間殺死300頭羊，卻一口都沒吃。

羅伯死後不久的20世紀初期，牠的種群從美國全土滅絕。1995年美國野生動物局從加拿大引進狼群到黃石公園（為了控制當地鹿群的數量)野放<ref>{{cite web |url=https://www.youtube.com/watch?v=ysa5OBhXz-Q  |title=狼如何改變河流|date=年月日 |accessdate=2014年2月13日}}</ref>，雖然復育順利，但目前只分布在洛磯山脈以北，狼王羅伯生存過的新墨西哥州已有近一世紀未曾發現過狼群的蹤跡。

== 改編作品 ==
美國[[華特迪士尼公司|華特迪士尼公司]]有將羅伯的事蹟改編成名為《羅伯傳奇》（The Legend of Lobo）的電影於1962年公開，<ref>{{cite web |url=https://letterboxd.com/film/the-legend-of-lobo/ |title=The Legend of Lobo |work=Letter BoxD |language=en |accessdate=2017年12月30日}}</ref>往後英國生物學家[[大衛·艾登堡|大衛·艾登堡]]受此電影影響，與[[BBC|BBC]]電視台合作策劃出以羅伯為主題的紀錄影集於2008年播出，其中大衛·艾登堡有親自替該影集擔任旁白。<ref>{{cite web |url=http://www.bbc.co.uk/programmes/b009s6s9 |title=Lobo: The Wolf that Changed America |work=[[BBC|BBC]] |language=en |accessdate=2017年12月30日}}</ref>

在日本有多部將《狼王羅伯》改編的動物教育漫畫，包含1974年由漫畫家[[谷口治郎|谷口治郎]]所繪製；<ref>{{cite web |url=http://www.jiro-taniguchi-fan.com/setonanimalsstory.htm |script-title=ja:'74「 学習漫画　シートン動物記」について |work=Jiro-Taniguchi Fan |language=ja |accessdate=2017年12月30日}}</ref>1991年，由石川球太所繪製；<ref>{{cite web |url=http://www.jiro-taniguchi-fan.com/ishikawakyutasakuhinnenpyou2.htm |script-title=ja:石川球太氏作品リスト |work=Jiro-Taniguchi Fan |language=ja |accessdate=2017年12月30日}}</ref>以及2012年，由姬川明所繪製等多個不同版本。<ref>{{cite web |url=https://www.shogakukan.co.jp/books/09296603 |script-title=ja:シートン動物記 |work=[[小學館|小學館]] |language=ja |accessdate=2017年12月30日}}</ref>動畫部份方面，[[日本電視台|日本電視台]]有在1989至1990年期間播放由[[白土武|白土武]]所執導的《西頓動物記》改編電視動畫，《狼王羅伯》則是收錄在該系列裡的第17及18集。<ref>{{cite web |url=http://www.allcinema.net/prog/show_c.php?num_c=89173 |script-title=ja:シートン動物記(1989～1990) |work=All Cinema |language=en |accessdate=2017年12月30日}}</ref>

西頓之後在1900年的著作《給孩子們的野生動物扮演》（The Wild Animal Play for Children）裡，提供了在音樂劇表演上扮演《狼王羅伯》的角色造型。<ref>{{cite web |url=http://www.archive.org/stream/cihm_33037/#page/n39/mode/2up |title=The wild animal play for children |work=[[網際網路檔案館|網際網路檔案館]] |author=[[歐尼斯特·湯普森·西頓|歐尼斯特·湯普森·西頓]] |language=en |page=35頁 |accessdate=2017年12月30日}}</ref>日本當地也有多部忠於《狼王羅伯》原著或是改編的音樂劇作品，<ref name="Musical">{{cite web |url=http://www.musical.ne.jp/seton2013/casts |script-title=ja:キャスト「新オオカミ王ロボ」キャスト紹介 |work= Triangle co.,Ltd |language=ja |date=2014年5月30日 |accessdate=2017年12月30日}}</ref><ref>{{cite web |url=https://natalie.mu/stage/news/239049 |script-title=ja:札幌演劇シーズン2017-夏、「あっちこっち佐藤さん」がレパートリー作品に |work=[[Natalie_(網站)|Natalie]] |language=ja |date=2017年7月3日 |accessdate=2017年12月30日}}</ref>如藝人[[大山真志|大山真志]]便有在2013年春季於[[東京|東京]][[澀谷|澀谷]]公演的戲劇裡飾演羅伯。<ref name="Musical"/>

電子遊戲方面，在2015年手機遊戲《[[Fate/Grand_Order|Fate/Grand Order]]》裡所登場名為{{nihongo|「新宿的Avenger」|新宿のアヴェンジャー}}角色，採用結合了羅伯、與19世紀美國小說《[[沉睡谷传奇|沉睡谷传奇]]》故事中的無頭騎士之造型設計。
