'''嫩江省'''，是[[中華民國|中華民國]][[東九省|東九省]]之一，簡稱'''嫩'''，省會[[齊齊哈爾市|齊齊哈爾市]]。

== 名稱由來 ==
由於[[嫩江|嫩江]]貫穿本省全境，故名。

== 地理 ==
全省面積為77326平方公里。東面接[[合江省_(中華民國)|合江省]]，南鄰[[松江省_(中華民國)|松江省]]、[[吉林省_(中華民國)|吉林省]]和[[遼北省_(中華民國)|遼北省]]，西界[[興安省_(中華民國)|興安省]]，北連[[黑龍江省_(中華民國)|黑龍江省]]。

以平原為主體，除了西邊接近[[大興安嶺|大興安嶺]]是丘陵地外，全境都屬[[松遼平原|松遼平原]]的一部分。[[呼蘭河|呼蘭河]]和[[嫩江|嫩江]]是主要河流，全境屬[[松花江|松花江]]的流域範圍。屬[[大陸性氣候|大陸性氣候]]，年雨量約在400公釐左右。西部靠大興安嶺地區則是[[草原氣候|草原氣候]]。
== 經濟 ==
*境內多為[[黑土區|黑土區]]，土壤肥沃，宜農宜牧。其中[[呼蘭河|呼蘭河]]一帶，不僅土壤肥沃，再加上氣候比較濕潤，所以農業發達，是東北地區的重要穀倉，也是該省的精華區，農產品以小麥、玉米、大豆、小米及高粱等，此外還有出產甜菜，可供製糖。
*西部[[嫩江|嫩江]]河流域地廣人稀，草原上散布著不少鹽池。

== 交通 ==
國民政府時期的嫩江省相對東北其他省區而言，交通相對便利。
=== 鐵路 ===
*[[中國長春鐵路|中國長春鐵路]]由西北往東南斜貫本省區。
*[[洮北鐵路|洮北鐵路]]縱貫西部。
*[[濱北鐵路|濱北鐵路]]縱貫東部。
=== 公路 ===
*以[[齊齊哈爾|齊齊哈爾]]和[[呼蘭|呼蘭]]為中心對外放射。
=== 水運 ===
*嫩江自[[齊齊哈爾|齊齊哈爾]]以下，[[呼蘭河|呼蘭河]]自[[呼蘭|呼蘭]]以下均可行船，進入[[松花江|松花江]]，但冬季河面結冰，則無法行駛。
== 歷史沿革 ==
嫩江省自古即是[[蒙古|蒙古]]、[[滿族|滿族]]等民族居住之地。[[清|清]]代[[齊齊哈爾|齊齊哈爾]]為[[黑龍江將軍|黑龍江將軍]]駐守之地，光緒三十三年（西元1907年），設[[黑龍江省|黑龍江省]]，[[齊齊哈爾|齊齊哈爾]]為[[省會|省會]]。

1932年[[九一八事變|九一八事變]]爆發後，[[日軍|日軍]]侵入，淪為「[[滿洲國|滿洲國]]」領土，設[[龍江省|龍江省]]。

1945年[[抗日戰爭_(中國)|中國抗日戰爭]]勝利後，為免和[[黑龍江省|黑龍江省]]混淆，改稱'''嫩江省'''。國民政府於9月4日，在重慶宣布任命[[彭濟群|彭濟群]]為嫩江省主席，主持在齊齊哈爾市組建嫩江省政府，併在重慶成立嫩江省政府籌備處，開始策劃接收工作。而[[中共|中共]]在[[蘇聯|蘇聯]]的協助下，率先進入東北。11月9日，东北局派[[刘锡五|刘锡五]]、[[于毅夫|于毅夫]]等十五名干部到达[[齐齐哈尔市|齐齐哈尔市]]。11月9日，中共嫩江地区工委在齐齐哈尔市成立，刘锡五任书记。11月14日，嫩江省政府在齐齐哈尔市宣告成立，于毅夫任主席。嫩江人民自卫军司令部改称东北人民自治军嫩江军区，[[王明贵|王明贵]]任司令员，刘锡五兼政委。11月下旬，成立中共齐齐哈尔市委和齐齐哈尔市政府，朱光任书记，朱新阳任市长。

11月27日，蘇聯通知中共地方黨、軍領導機關和武裝力量撤出齊齊哈爾，準備將城市移交給[[國民政府|國民政府]]接管。1945年12月30日，中共嫩江省工委、省政府、嫩江军区、齐齐哈尔市等党、政、军机关、部队撤离齐齐哈尔市，转移至[[甘南|甘南]]县城。

1946年1月1日中共嫩江地区工委改为中共嫩江省工委，隶属东北局北满分局领导。

1946年1月，嫩江省新任主席[[彭濟群|彭濟群]]及其所率全體接收委員保安隊等，於21日由[[哈爾濱|哈爾濱]]出發，抵達省城齐齐哈尔，24日正式成立省政府，並開始接管省政。隨著蘇聯軍隊撤離東北，國軍和共軍在東北地區展開了激烈爭奪。4月中下旬，蘇聯軍隊撤出齊齊哈爾。[[共產黨|共產黨]]將領[[黃克誠|黃克誠]]命令[[新四軍|新四軍]]第3師一個團，在4月24日攻下齊齊哈爾，嫩江省政府頃刻瓦解。彭濟群率領部分省府人員逃奔瀋陽，建立嫩江省政府辦事處。1947年7月19日，國民政府派彭濟群為嫩江選舉事務所主任委員；國民政府同時派定其為[[國民大會代表|國民大會代表]]，[[立法院|立法院]][[立法委員|立法委員]]。<ref name="民國史大事記">{{cite book|editor=李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編|title=《中華民國史大事記》|location=[[北京|北京]]|publisher=[[中華書局|中華書局]]|date=2011年7月}}</ref>{{rp|8385-8386}}

1946年5月14日，中共嫩江省工委由中共北满分局划归中共西满分局领导。1946年6月30日至7月7日，嫩江省临时参议会在齐齐哈尔市召开，选举了嫩江省行政委员会，[[于毅夫|于毅夫]]、[[王明贵|王明贵]]等十一人为行政委员，于毅夫当选为省政府主席兼省高等法院院长，[[车向忱|车向忱]]、[[张瑞麟|张瑞麟]]当选为省临时参议会正、副议长。[[倪志亮|倪志亮]]接任嫩江军区司令员。1946年7月，中共嫩江省工委改为中共嫩江省委。省委书记刘锡五兼嫩江军区政委，副书记[[邵式平|邵式平]]兼嫩江军区副政委。

1946年12月，为适应战争环境的需要，中共西满分局决定撤销中共嫩江省委，成立中共嫩江、三肇（哈西）两个地委，由西满分局直接领导。1947年2月2日，东北行政委员会常委会决定合并黑龙江省、嫩江省为[[黑嫩联合省|黑嫩联合省]]，简称为黑嫩省政府，任命于毅夫为黑嫩省政府主席、[[杨英杰|杨英杰]]为副主席。省府设在齐齐哈尔市，同时撤销中共黑龙江省委。西满分局和黑嫩省政府下辖五个地委、五个专署、一个市委与市政府。1947年9月16日，黑嫩联合省撤销，恢复嫩江省与黑龙江省，并将原属黑龙江省的安达县划归嫩江省。同时，嫩江、三肇两地委（即二、四地委）合并成立中共嫩江省委，省委书记刘锡五（1948年10月20日由[[顾卓新|顾卓新]]接任），于毅夫任嫩江省政府主席。

1948年年底[[遼瀋會戰|遼瀋會戰]]後，大部分地區為[[中共|中共]]所控。11月2日[[解放軍|解放軍]]正式進入瀋陽，中華民國嫩江省政府機構徹底瓦解，前後僅兩年。

1949年4月21日，东北行政委员会决定，撤销嫩江省，并入黑龙江省。1949年5月15日，两省机构正式合并。

== 行政區劃 ==
=== 省會 ===
民國三十六年（1947年）6月，國民政府公布東北九省的行政區域，嫩江省省會為齊齊哈爾市。

=== 縣、市、設治局 ===
民國三十六年（1947年）6月，國民政府公布東北九省的行政區域，全省轄有1市、18縣、2旗，由於省境較小，不設行政督察區。由於本省因抗戰結束後，全境己被中共東北民主聯軍控制，故國民政府公佈的行政區劃並未得到執行。
