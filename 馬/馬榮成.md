'''馬榮成'''（{{bd|1961年|1月16日|||M马}}），原名'''馬榮城'''，籍貫[[廣東|廣東]][[潮陽|潮陽]]。[[香港|香港]]著名[[漫畫家|漫畫家]]。 自小熱愛繪畫，14歲加入《[[喜報|喜報]]》，踏入漫畫行業，曾用筆名馬雲龍，早期作品有《[[魔鬼實驗|魔鬼實驗]]》、《[[五兄弟|五兄弟]]》及《[[風流|風流]]》等，在1980年代當紅。馬榮成和另外9位漫畫家於2012年入選由[[香港動漫畫聯會|香港動漫畫聯會]]主辦，[[香港政府|香港政府]][[創意香港|創意香港]]撥款[[資助|資助]]的[[香港漫畫星光大道|香港漫畫星光大道]]留下[[手印|手印]]。馬榮成接受訪問時表示童年時很受日本漫畫的影響，變得很愛看漫畫，亦為及後成為漫畫家而舖路<ref>[https://bka.mpweekly.com/interview/%E6%B1%AA%E6%9B%BC%E7%8E%B2-%E6%9B%BC%E9%8F%A1%E9%A0%AD/20180312-104885 【馬榮成專訪1】十四歲入行 畫出美麗人生]</ref>。

== 早年 ==
馬榮成三歲喪父，與媽媽，兩個姊姊和一個妹妹[[馬淑珠|馬淑珠]]一家五口居於[[油塘|油塘]]，小學畢業於樂善堂油塘灣學校（即現在的樂善堂楊仲明學校）。後來升讀[[九龍城|九龍城]]中西英文書院（曾於嘉林小築位置，現為私人住宅）。中二輟學後加入《喜報》任學徒。1980年與朱迺怡(朱祖兒)、沈立雄及尹慶祥等編繪《新浪潮》漫畫雜誌，屬新派漫畫，是當年主流打鬥書之外的另一風格。

== 中華英雄時期 ==
師承[[上官小威|上官小威]]的馬榮成，1980年8月轉投「玉郎機構」（[[文化傳信|文化傳信]]前身）在[[金報|金報]]寫「[[中華英雄|中華英雄]]」；1981年在[[黃玉郎|黃玉郎]]的「[[醉拳|醉拳]]」連載，漸受歡迎後獲黃玉郎賞識，在1982年尾，令他初嘗當[[主筆|主筆]]的滋味，中華英雄獨立推出單行本雙週刊。在[[黃玉郎|黃玉郎]]壟斷畫壇的時代下，他猶如平地一聲雷，創造出十八萬（有說是二十萬）的驚人銷量，並開創寫實型漫畫之風。個人亦曾在[[灣仔區|灣仔]]的[[香港藝術中心|香港藝術中心]]開畫展，是首位香港漫畫家開辦個人畫展，標示著香港人接納港漫為藝術的一部份。 

由於中華英雄的理想成績，為馬帶來豐厚的收入（一說他年薪超過500萬）<ref>[[中華英雄|中華英雄]]第151期專欄，第35頁「有關《中華英雄的一段澄清》」</ref>。在1980年代，馬是玉郎機構旗下年薪最高的主筆。至1988年初，馬打算在翌年約滿後離開玉郎機構，另謀發展，期間不停遭人恐嚇騷擾。1988年5月，馬在[[灣仔|灣仔]]遭人毆打受傷；及至1989年元旦日，馬與其新任助手[[崔源發|崔源發]]在[[太古城|太古城]]停車場，遭三名不明凶徒持刀襲擊，兩人均手部受創，其中崔的傷勢較馬嚴重。<ref>1989年1月3日，第4版，[[《華僑日報》|《華僑日報》]]</ref> 當期時中華英雄出版至第143期，受是次事件影響停刊半年，其後馬傷瘉後在6月復刊，之後不久馬亦約滿離開玉郎機構。

* [[無線電視|無線電視]]於1984年即[[中華英雄|中華英雄]]銷量達高峰之時，曾以電視劇「[[畫出彩虹|畫出彩虹]]」描繪馬榮成奮鬥經過，由[[詹秉熙|詹秉熙]]、[[張曼玉|張曼玉]]主演。
* [[亞洲電視|亞洲電視]]於1993年亦曾以馬榮成由出道到建立公司之經歷拍成電視單元劇「[[紙上英雄|紙上英雄]]」，由[[張家輝|張家輝]]主演，被斬一事亦有被收錄其中。

== 天下時期 ==
1989年7月，馬榮成脫離「玉郎機構」，創立了「[[天下出版社|天下出版社]]」，出版「[[天下畫集|天下畫集]]」，最初四期故事為「[[兩極|兩極]]」，之後連載長篇武俠漫畫「[[風雲_(漫畫)|風雲]]」，得到當時讀者們甚高的評價，不但在「天下畫集」中連載超過二十年，它更多次被改編成為電影、小說和廣播劇，台灣方面更把它改編成為電視劇集。 可惜銷量始終無法突破「中華英雄」年代的成績。

1995年，馬榮成與《黑豹列傳》主編[[溫日良|溫日良]]發生勞資糾紛，雙方亦鬧上法院，溫日良最終獲判勝訴，資方馬榮成須賠償合約內之酬金及支付全數堂費。

2016年，馬榮成致力於中國市場推動炒賣漫畫。

== 影響 ==
馬榮成的寫實風格，是受[[日本漫畫家|日本漫畫家]][[池上遼一|池上遼一]]、[[永安巧|永安巧]]和[[松森正|松森正]]、與及香港「[[山月畫室|山月畫室]]」的[[陳中樞|陳中樞]]所影響。而「中華英雄」的空前成功，使寫實風格成為了[[香港漫畫|港漫]]畫面製作的標準，「天下畫集」的出現更是深化了此風氣。日本漫畫家[[井上雄彥|井上雄彥]]亦曾參觀「天下出版社」。

== 個人生活 ==
馬榮成的初戀情人叫陳潔瑜，他們拍拖超過十年。當年他創作中華英雄時就將女主角命名「陳潔瑜」（主角華英雄之青梅竹馬戀人、後結成夫婦。單行本第一期被毒死。）90年兩人分手，一年後重遇[[玉郎集團|玉郎集團]]時期的舊同事趙美鳳（筆名：一怡），是漫畫少年的助理。 兩人拍拖至96年結婚至今。兩人育有一子、一女。趙美鳳1991年曾參與[[李克勤|李克勤]][[紅樓夢|紅樓夢]][[MV|MV]]拍攝，為女主角之一（另一女主角是[[周慧敏|周慧敏]]）。（MV：http://www.youtube.com/watch?v=RRm3VKl2oMY）
