'''馬自達Flair Wagon'''（{{lang-ja|マツダ・フレアワゴン}}）是2012年起由[[日本|日本]][[馬自達|馬自達]]汽車公司委託[[鈴木公司|鈴木汽車]][[貼牌生產|代工生產]]、[[换牌工程|貼上馬自達廠徽銘牌]]的[[輕型高頂旅行車|輕型高頂旅行車]]，這款車是以[[鈴木Palette|鈴木Palette]]（{{ja}}スズキ・パレット）為基礎而開發出來的，僅在日本國內市場銷售。

除了鈴木Palette外，另一款兄弟車為{{link-ja|日產Roox|日産・ルークス}}。這是繼[[鈴木Alto#第六代HA24S/24V型（2004年-2009年）|第六代鈴木Alto]]、[[馬自達Carol#第五代（HB24型_2004年-2009年）|第五代馬自達Carol]]和{{link-ja|日產Pino|日産・ピノ}}三兄弟車款以來，第二次出現三兄弟車款。

長年以來[[馬自達|馬自達]]和[[鈴木公司|鈴木汽車]]之間存在著輕型車[[貼牌生產|OEM]]代工的協議，這款以[[鈴木Palette|鈴木Palette]]為基礎而開發的輕型車於2012年6月26日正式發表上市<ref>參看{{ja}}[http://www.mazda.co.jp/corporate/publicity/release/2012/201206/120626a.html 【MAZDA】新型「マツダ フレアワゴン」を発売]。</ref>。上市初期時兩者不論外觀或內裝完全一樣，差別只在廠徽銘牌不同而已，這一點和[[馬自達Scrum#第四代（2005年迄今）|第四代Scrum]]、[[馬自達AZ-Offroad|AZ-Offroad]]相同<ref>不過，另一款兄弟車日產Roox則自行重新設計車頭。</ref>。

動力方面分成兩種：自然進氣的658c.c.直列三缸DOHC VVT K6A型[[發動機|引擎]]最大[[馬力|馬力]]54ps / 6,500rpm、扭力峰值6.4kgf・m / 3,500rpm；配備渦輪增壓的658c.c.直列三缸DOHC K6A型引擎最大馬力64ps / 6,000rpm、最大扭力9.7kgf・m / 3,000rpm。至於[[變速器|變速箱]]系統，則採用[[日產汽車|日產汽車]]與[[Jatco|Jatco]]（{{ja}}ジヤトコ株式会社）共同開發的[[無段自動變速器|CVT]]附副變速機構。全車系最大的賣點在於後車門採滑門式設計，部分高階車款甚至設置電動滑門。此外，「IS Limited」車款裝置了[[怠速熄火系統|怠速熄火系統]]，以{{link-ja|JC08模式|JC08モード}}測試可達22.2km/L的油耗表現<ref>請見{{ja}}[http://www.mazda.co.jp/corporate/publicity/release/2012/201206/120626a.html 【MAZDA】新型「マツダ フレアワゴン」を発売]，第三段。</ref>。

2013年2月因鈴木Palette的後繼車款[[鈴木Spacia|鈴木Spacia]]上市，第一代Flair Wagon隨著停產。此代車款僅販售約9個月的時間，這在日本汽車史上是相當罕見的。

2013年 - 3月18日發表大改款，同年4月25日開始發售，其原型車款從鈴木Palette變為[[鈴木Spacia|鈴木Spacia]]。和同系列的[[馬自達Flair|Flair]]、[[馬自達Carol|第六代Carol]]一樣，搭載鈴木汽車新開發的「eNe-CHARGE」[[動能回收系統|動能回收系統]]（{{ja}}エネチャージ）、「Eco-Cool」蓄冷技術（{{ja}}エコクール）<ref>在汽車[[怠速熄火系統|怠速熄火]]狀態下，該蓄冷技術將冷氣的冷風保存住，俟引擎再度發動的瞬間將冷風送往引擎以散熱，提高引擎作功效率，詳情請參考{{ja}}[http://response.jp/article/2012/08/09/179457.html 【スズキ 低燃費技術発表】蓮池常務、エコクールで「実用面の性能を高めたい」]內文。</ref> 和[[怠速熄火系統|引擎怠速自動啟閉系統]]，以提升油耗表現。[[發動機|引擎]]改採658c.c.直列三缸DOHC VVT [[馬自達輕型車引擎列表#R06A型|R06A型]]，最大[[馬力|馬力]]52ps / 6,000rpm、扭力峰值63N·m / 4,000rpm。側門採用滑門設計，且使用隱藏式滑軌，使得車門寬度擴大。

同年6月20日發表運動化車型「Custom Style」，其實為鈴木Spacia Custom[[換牌工程|換牌]]而來，具有全新設計水箱罩、透明尾燈組、全車空力套件等配備；同時「XS」可加價選購附防夾功能的電動滑門、附自動啟閉的放電式頭燈組。9月19日部分改良，雷達煞車輔助系統、防暴衝機能、緊急煞車信號、DSC動態穩定控制系統、TCS循跡控制系統等皆成為標準配備，更換新水箱罩，「XG」新增前拉桿。11月23日與鈴木Spacia同時獲得2013年度[[日本年度風雲車|日本年度風雲車]]。

2014年 - 6月19日發售「XS Limited」特仕車，以「XS」為基調外加附防夾功能的右後電動滑門、附自動啟閉的放電式頭燈組、防[[紫外線|紫外線]]前擋玻璃、雙前座加熱功能、後座熱風出口、助手席化妝鏡、全車空力套件、附LED方向燈之後視鏡、14吋[[鋁合金|鋁合金]]輪圈等配備。

2015年 - 5月28日實施小改款，原「eNe-CHARGE」[[動能回收系統|動能回收系統]]的馬達改為ISG整合式啟動馬達（Integrated Starter Generator之[[縮寫|縮寫]]）與[[鋰電池|鋰電池]]的「S-eNe-CHARGE」，新增雙攝影鏡頭，當中包含[[車道偏離警示|車道偏離警示]]系統、前方碰撞預警系統（含自動煞車）、前方車輛移動警告、車輛路線不穩警告與油門誤踩警告系統等配備。8月27日推出「Custom Style」[[渦輪增壓器|渦輪增壓]]版，搭載658c.c.直列三缸DOHC VVT [[馬自達輕型車引擎列表#R06A型|R06A型]][[渦輪增壓器|渦輪增壓]]引擎，配合ISG整合式啟動馬達（Integrated Starter Generator之[[縮寫|縮寫]]）與[[鋰電池|鋰電池]]，同時新設[[巡航定速|巡航定速]]系統和換檔撥片。最大馬力為64ps / 6,000rpm，最大扭力為9.7kg・m / 3,000rpm。主動式安全系統方面新增雙攝影鏡頭，當中包含[[車道偏離警示|車道偏離警示]]系統、前方碰撞預警系統（含自動煞車）、前方車輛移動警告、車輛路線不穩警告與油門誤踩警告系統等配備，加上四顆[[安全氣囊|安全氣囊]]與ESP車身穩定控制系統。

2017年 - 1月12日改良<ref>{{ja}}[http://www2.mazda.com/ja/publicity/release/2017/201701/170112a.html 【MAZDA】「マツダ フレアワゴン」を一部改良]。</ref>，新增防眩式後視鏡、倒車顯影等配備，追加兩種新車色，將以往二種渦輪增壓車型合併成「Custom Style XT」一種車型。同年12月由於兄弟車改朝換代，全車系中止生產，不過福祉車仍繼續販售。

2017年 - 12月22日原廠宣佈翌年2月8日起開始販售第三代Flair Wagon<ref>{{ja}}[http://www2.mazda.com/ja/publicity/release/2017/201712/171222a.html 【MAZDA】マツダ、新型「フレアワゴン」「フレアワゴン　カスタムスタイル」を発表]。</ref>，由於採用新式[[底盤|底盤]]，車身尺寸比前代更為放大，車室空間也因而擴大。動力心臟分成二種：658c.c.直列三缸DOHC VVT [[馬自達輕型車引擎列表#R06A型|R06A型]]自然進氣與658c.c.直列三缸DOHC VVT [[馬自達輕型車引擎列表#R06A型|R06A型]][[渦輪增壓器|渦輪增壓]]引擎，配合CVT無段變速系統。此外，全車系標配具有WA05A型直流同步馬達、微型鋰電池模組的Mild Hybrid動力系統。在安全輔助方面，倒車自動緊急煞車、倒車防爆衝、後方駐車雷達等皆列為全車系標配；另可加價選購DSBS預防碰撞系統（Dual Sensor Brake Support），由單眼鏡頭與雷射感應器構成，並包含自動切換遠近燈之功能。

2018年 - 5月21日原廠發佈福祉車「[[輪椅|輪椅]]移動車」的改款消息<ref>{{ja}}[http://www2.mazda.com/ja/publicity/release/2018/201805/180521a.html 【MAZDA】新型「マツダ フレアワゴン 車いす移動車」を発売]。</ref>。
