'''（偉大者）马克西米利安一世'''（[[德文|德文]]：'''{{lang|de|Maximilian I}}'''，或譯'''馬克西米連一世'''，{{bd|1459年|3月22日|1519年|1月12日|catIdx=M}}），[[神圣罗马帝国|神圣罗马帝国]][[神聖羅馬皇帝|皇帝]]，[[德国君主列表|罗马人民的国王]]，[[奧地利大公國|奥地利]][[奥地利大公|大公]]（1493年－1519年），也被稱作「马克西米利安[[大帝|大帝]]」。神聖羅馬帝國皇帝及奧地利大公[[腓特烈三世_(神圣罗马帝国)|腓特烈三世]]的長子，是神圣罗马皇帝[[查理五世_(神圣罗马帝国)|查理五世]]（罗马人民的国王卡尔五世，[[西班牙|西班牙]]王卡洛斯一世）的祖父，亦是[[哈布斯堡王朝|哈布斯堡王朝]]鼎盛时期的奠基者。马克西米利安通过自己和子女的婚姻，使得孙子成功地取得[[西班牙|西班牙]]这个殖民帝国的王位，再加上自己的神圣罗马帝位，使查理五世成为[[欧洲|欧洲]]的盟主，更令哈布斯堡王朝成为一「[[日不落帝国|日不落帝国]]」。

== 生平 ==
1477年，馬克西米利安一世同[[勃艮第统治者列表|勃艮第公爵]][[大胆的查理|大胆的查理]]之女[[勃艮第的瑪麗|瑪麗]]订婚，获得[[荷兰|尼德兰]]和法国东部边境一带的[[勃艮第公國|勃艮第]]领地。大胆的查理在[[南锡|南锡]]战死后，馬克西米利安一世於1479年在[[吉内加特战役|吉内加特战役]]中击败法兰西国王[[路易十一|路易十一]]的进犯，保護了妻子的領地。1482年3月27日，他的妻子骑马摔死，他的年幼的長子[[腓力一世_(卡斯蒂利亞)|腓力一世]]繼承尼德兰而他担任攝政，后来因要与法国开战，与尼德兰议会发生冲突，被困在宫中，直到他父亲派兵把他救出来，他被迫接受“[[阿拉斯会议|阿拉斯条约]]”，允许尼德兰三级议会充当儿子的摄政，又允许把女儿[[奥地利的玛格丽特|玛格丽特]]嫁给法兰西王太子以保持和平。馬克西米利安打算娶[[布列塔尼的安妮|布列塔尼的安妮]]为妻，當通过代理人举行过婚礼後，但他的准女婿[[查理八世_(法兰西)|查理八世]]因怕布列塔尼落入馬克西米利安手中，居然毁婚而抢走了馬克西米利安的未婚妻。他不久就違反誓言，重新获得摄政权利。1486年继承父位，成为罗马人民的国王。

1490年，当[[匈牙利|匈牙利]]的[[马加什一世|马加什一世]]死后，他重新控制[[奥地利|奥地利]]地区。他參與争奪[[匈牙利王国_(1301年–1526年)|匈牙利]]和[[波希米亞王國|波希米亞]]的王位，却被[[弗拉迪斯拉夫四世|弗拉迪斯拉夫四世]]抢先奪得王位，根据1491年的“[[普雷斯堡和约|普雷斯堡和约]]”若弗拉迪斯拉夫没有后裔，波西米亚和匈牙利的王位应该由哈布斯堡家族来继承。1493年，[[腓特烈三世_(神圣罗马帝国)|腓特烈三世]]去世，他成了德意志唯一的统治者和哈布斯堡家族的首领。

1494年，法兰西國王查理八世侵犯意大利，开启了长达半个世纪的意大利战争，为了寻求对付法兰西王国的同盟者，馬克西米利安一世娶了[[米兰公国|米蘭]][[斯福尔扎家族|斯福尔扎家族]]的[[比安卡·玛丽亚·斯福尔扎|瑪麗亞]]，还同[[阿拉贡|阿拉贡]]的[[费尔南多二世_(阿拉贡)|费尔南多二世]]结成双重的儿女亲家。1495年同教皇、[[西班牙|西班牙]]、[[威尼斯共和国|威尼斯]]、米兰结成同盟，次年赶走入侵意大利的法军。1499年，[[舊瑞士邦聯|瑞士联邦]]不服从帝国法令，他下令征讨，但他还没有动身，前锋已经被瑞士人夜袭而大败，只好承认瑞士独立。实际上，德意志诸侯害怕他扩大权力，甚至准备串通起来废黜他，他只好同意将帝国权力交给议会，满足做一个虚君，有段时间他竟然异想天开的想去竞选教皇。但他终于渡过了难关，在1504年的王朝战争中得胜，重振声威。1508年2月，教皇[[儒略二世|尤利乌斯二世]]授予其[[神聖羅馬皇帝|神圣罗马帝国皇帝]]称号，自此羅馬人民的國王当选后即成为神圣罗马帝国皇帝，无须去罗马接受教皇加冕。同年和教皇国、法国、西班牙组成[[康布雷同盟|康布雷同盟]]，对威尼斯共和国开战，战胜后他获得[[维罗纳|维羅纳]]、[[维琴察|维琴察]]和[[伦巴第|伦巴第]]的[[帕多瓦|帕多瓦]]。

1511年经他那个杰出的女儿，尼德兰摄政－[[奥地利的玛格丽特|奥地利的玛格丽特]]的外交才干，解除同法蘭西王國的盟约，召回法军中的德国雇佣军，同英国、西班牙和教皇结成新的神圣同盟，1513年在斯普尔斯战役中战胜法军，把路易十二的法军赶出意大利。但1515年法蘭西國王[[弗朗索瓦一世_(法兰西)|弗朗索瓦一世]]重新带兵出现在意大利平原，在[[马里尼亚诺战役|马里尼亚诺战役]]中击溃了他的同盟者米兰公爵[[馬克西米利安·斯福爾扎|馬克西米利安·斯福爾扎]]，根据《[[布鲁塞尔和约|布鲁塞尔和约]]》，米兰归属[[法蘭西王國|法蘭西王國]]，维羅纳归属威尼斯，马克西米利安一世仅保有[[蒂罗尔|蒂罗尔]]地区。

== 联姻安排 ==
马克西米利安本人，于1477年8月18日迎娶[[勃艮第公国|勃艮第公国]]公爵[[勇士查理|大胆查理]]的独女[[勃艮第的瑪麗|勃艮第的玛丽]]。在玛丽的父亲去世后，这段姻缘将属于[[勃艮第公国|勃艮第公国]]的，自[[法国|法国]]南部至[[荷兰|荷兰]]的领地统统并入哈布斯堡皇室领地。第一任妻子瑪麗過世後，1497年他再婚，與[[米蘭公爵|米蘭公爵]]之女[[比安卡·玛丽亚·斯福尔扎|比安卡·玛丽亚·斯福尔扎]]的婚姻讓他從義大利獲得大量現金，幫助馬克西米利安在1508年被確認為皇帝。

马克西米利安的儿子[[費利佩一世_(卡斯蒂利亞)|美男子菲利普]]于1496年10月20日迎娶卡斯蒂利亚王储[[胡安娜_(卡斯蒂利亚)|疯女胡安娜]]，开创了西班牙哈布斯堡王朝。

马克西米利安的孙子，日后的神圣罗马皇帝[[斐迪南一世_(神圣罗马帝国)|斐迪南一世]]于1521年5月25日迎娶了波希米亚公主安妮，次年，斐迪南的妹妹玛利亚公主嫁与[[匈牙利|匈牙利]]兼波希米亚国王[[拉約什二世|拉約什二世]]，这两段婚姻为日后奥地利吞并波希米亚和奥匈合组二元帝国埋下伏线。

马克西米利安的儿女，多与外国王族通婚，哈布斯堡王朝在欧洲的影响，大大增强。

== 联姻结果 ==
[[File:Bernhard_Strigel_003.jpg|200px]]

马克西米利安精心安排的政治婚姻，使他的孙子查理五世成为欧洲的霸主。

1506年，马克西米利安的儿子，查理的父亲英俊王菲利浦英年早逝，查理便继承了勃艮地公爵一职，成为了[[西屬尼德蘭|西属尼德兰]]（今日的荷兰和比利时）的首位统治者。

1516年，马克西米利安的亲家，西班牙国王费迪南二世病逝，查理因此成为了西班牙国王卡洛斯一世。自此，西班牙全国、意大利南部的[[西西里岛|西西里岛]]、[[撒丁岛|撒丁岛]]、[[那不勒斯王国|那不勒斯王国]]以及西班牙在[[美洲|美洲]]的殖民地都成了哈布斯堡王朝的管治领域。1526年8月29日，匈牙利兼波西米亚国王路易二世战死沙场，马克西米利安另一个孙子，查理五世的弟弟，奥地利大公斐迪南以路易的姐夫身分继承匈牙利和波西米亚国王。虽然在匈牙利出现了争位者，但很快便被平定。

== 逝世和身后安排 ==

1519年，马克西米利安病逝，长孙查理继承帝位成为[[查理五世_(神圣罗马帝国)|查理五世]]，并承继了家族对奥地利和[[阿尔萨斯|阿尔萨斯]]的管治。马克西米利安同时嘱咐女儿，奥地利的玛格丽特女大公负责养育两个孙儿卡尔和费迪南。马克西米利安的两个孙儿相继成为神圣罗马皇帝，而且都有所作为。

== 贡献与影响 ==

马克西米利安一世被称为“最后的骑士”，既具有中世纪风范，又有[[文艺复兴|文艺复兴]]君主的气质，是一位学者和诗人，是人文主义者和艺术家的保护人。他在战争中则是一位胸怀韬略的统帅，也是[[德国雇佣兵|德国雇佣兵]]（Landsknechts）和帝国炮兵的缔造者，火炮技术的专家。著有半小说性质的个人自传《白色的国王》（Weiss König）。另外，马克西米利安还因倡导了一种最重要的铠甲制作风格而闻名。以表面的多条开槽和镂刻线为特征的“[[马克西米利安式铠甲|马克西米利安式铠甲]]”在1500－1530年间风行欧洲。

他品德高尚，长相英俊，性格浪漫，当然，也免不了有些随便，他喜欢女人和酒，喜欢打猎游玩，他会寻欢作乐，也能忍耐，而他晚年怕自己随时死掉，甚至带着棺材伪装成书箱或财宝箱出游，1519年，当他还不满60岁时，为债务所迫，怀着极坏的心情跑到因斯布鲁克，但这个城市居然拒绝为他开城门，这位两鬓斑白的皇帝不得不继续前行，不久就死在他经常滑雪的维尔斯山间小屋里。因斯布鲁克还留有一个他的墓地，但他并没有埋在那里。他的墓地几乎是个艺术馆，全是雕像，这在西方是很少见的。 虽然马克西米连的一生充满挫折和磨难，但是他的奋斗为哈布斯堡王朝的扩展开拓的了广阔的前景。他有14个私生子女。和父亲一样，他经常债台高筑，一旦有了烦恼，他便去打猎消遣。他平时威严而又乐于接进民众，他是文艺复兴事业的赞助者和热心参与者，他是虔诚的基督徒，又相信占星术。

马克西米利安的奋斗为哈布斯堡王朝的扩展开拓的了广阔的前景。在加强德意志中央集权方面，马克西米利安也做出过很大努力，但因无法将哈布斯堡家族利益与帝国利益区分清楚，诸侯并不支持他的政策。

“在你们的国王领导下团结起来，保卫德意志使之免遭法国和土耳其的侵犯，否则就是灭亡”，这是马克西米利安雄辩的向帝国议会，向诸侯们，向城镇议会不断提出的问题。他没有能够达到这一目标，德意志以后三个世纪的历史证实了他的警告。
