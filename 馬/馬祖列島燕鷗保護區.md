'''馬祖列島燕鷗保護區'''位於[[馬祖列島|馬祖列島]]中的八座無人島礁，包括各島礁低潮線向海域延伸一百公尺的範圍，為依據《野生動物保育法》公告之野生動物保護區及重要棲息環境<ref name="林務局">{{Cite web|url=http://conservation.forest.gov.tw/ct.asp?xItem=59934&ctNode=178&mp=10　|title=馬祖列島燕鷗保護區|accessdate=2013-01-08|publisher=農委會林務局}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web|url=http://conservation.forest.gov.tw/ct.asp?xItem=3358&ctNode=176&mp=10　|title=馬祖列島野生動物重要棲息環境|accessdate=2013-01-08|publisher=農委會林務局}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。保護區的陸域面積近12公頃；海域面積近60公頃，散布在[[連江縣_(中華民國)|連江縣]][[東引鄉|東引]]、[[北竿鄉|北竿]]、[[南竿鄉|南竿]]、[[莒光鄉|莒光]]等四個行政區內。每年4月1日起至9月30日止列為管制期，一般民眾嚴禁登島，可藉搭船巡航方式自遠處賞鳥<ref>{{Cite web |url=http://www.matsu-nsa.gov.tw/User/Article.aspx?a=103&lang=1 |title=季節賞鳥四日遊 |publisher=交通部觀光局馬祖國家風景區 |accessdate=2013-01-08}}</ref>。

==保護區範圍==
包括[[連江縣_(中華民國)|連江縣]][[東引鄉|東引鄉]]之雙子礁，[[北竿鄉|北竿鄉]]之三連嶼、中島、鐵尖島、白廟、進嶼，[[南竿鄉|南竿鄉]]之劉泉礁，[[莒光鄉|莒光鄉]]之蛇山等八座島礁<ref name="林務局"/>。

==保育物種==
主要為保護在此繁殖的[[白眉燕鷗|白眉燕鷗]]、[[紅燕鷗|紅燕鷗]]、[[蒼燕鷗|蒼燕鷗]]、[[鳳頭燕鷗|鳳頭燕鷗]]、[[黑尾鷗|黑尾鷗]]、[[岩鷺|岩鷺]]、[[叉尾雨燕|叉尾雨燕]]等七種鳥類<ref name="林務局"/>。攝影師梁皆得在2000年時應連江縣政府委託進行拍攝時，發現已瀕臨絕跡的「神話之鳥」[[黑嘴端鳳頭燕鷗|黑嘴端鳳頭燕鷗]]在此處繁殖<ref>{{Cite web|url=http://tc.gio.gov.tw/ct_12576_39|title=梁皆得|work=臺灣電影網|publisher=文化部影視及流行音樂產業局|accessdate=2013-01-08}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。這是1937年後首次確認的採集或觀測紀錄，也是首次繁殖紀錄<ref>{{Cite web |url=http://www.matsu-nsa.gov.tw/Backpacker/Article.aspx?a=536&l=1 |title=神話之鳥—黑嘴端鳳頭燕鷗 |author=梁皆得 |publisher=連江縣政府 |accessdate=2013-01-08}}</ref><ref>{{Cite web |url=http://www.libertytimes.com.tw/2005/new/aug/11/today-life10.htm |title=神話鳥黑嘴端鳳頭燕鷗雙飛 驚動世界 |author=鄭旭凱 |publisher=自由電子報 |date=2005-08-11 |accessdate=2013-01-08}}</ref>。

==賞鷗活動==
近年來，馬祖每年8、9月有固定的賞鷗船航班，全程大約90分鐘，當地也有許多觀光漁船可提供包船賞鷗行程。<ref>{{Cite web |url=https://zetaspace.win/2016-matsu-tern/ |title=馬祖●北竿-馬祖燕鷗保護區 夏日 與神話有約 |work=北方羽林 |accessdate=2016-07-02}}</ref>。
