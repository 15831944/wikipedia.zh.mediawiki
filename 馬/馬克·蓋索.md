'''马克·加索尔·萨厄斯'''（[[加泰隆尼亞語|加泰隆尼亞語]]：'''Marc Gasol Sáez'''，{{bd|1985年|1月29日|||Gasol}}），出生於[[西班牙|西班牙]][[巴塞隆納|巴塞隆納]]，現役西班牙職業[[籃球|籃球]]运动员，目前效力於[[NBA|NBA]]聯盟的[[多倫多暴龍|多倫多暴龍]]，場上位置為[[中锋|中锋]]。此外他也是[[密爾瓦基公鹿|密爾瓦基公鹿]]中鋒[[保罗·加索尔|保罗·加索尔]]的二弟。

小蓋索在[[2007年NBA選秀|2007年NBA選秀]]大會中於第二輪第四十八順位被[[洛杉磯湖人|洛杉磯湖人]]選中，隨後被交易至[[曼菲斯灰熊|曼菲斯灰熊]]。2012年首次入選[[2012年NBA全明星賽|全明星賽]]，[[2012-13_NBA賽季|2012–13賽季]]榮獲[[NBA年度最佳防守球員|NBA年度最佳防守球員]]並入選[[NBA最佳陣容|NBA年度第二隊]]以及[[NBA最佳防守陣容|NBA最佳防守陣容第二隊]]，2015年職業生涯首次以先發球員身份入選[[2015年NBA全明星賽|全明星賽]]，以及[[NBA最佳陣容|NBA年度第一隊]]，2017年以替補球員身份入選[[2017年NBA全明星賽|全明星賽]]。

小蓋索是[[西班牙國家男子籃球隊|西班牙國家男子籃球隊]]的成員，他贏得了兩枚[[奧運會|奧運會]][[銀牌|銀牌]]，一枚[[世界盃籃球賽|世界盃籃球賽]][[金牌|金牌]]，兩枚[[歐洲籃球錦標賽|歐洲籃球錦標賽]]金牌，一枚[[歐洲籃球錦標賽|歐洲籃球錦標賽]]銀牌和一枚[[歐洲籃球錦標賽|歐洲籃球錦標賽]][[銅牌|銅牌]]。

==交易==
在多倫多時間2019年2月7號，[[曼菲斯灰熊|曼菲斯灰熊]]與[[多倫多暴龍|多倫多暴龍]]發生了一單交易，曼菲斯灰熊用馬克·蓋索換來了[[尤納斯·瓦蘭丘納斯|尤納斯·瓦蘭丘納斯]]、[[德隆·萊特|德隆·萊特]]、[[C·J·邁爾斯|C·J·邁爾斯]]和一張2024年第二輪選秀簽

== 西班牙國家隊表現 ==
蓋索曾為[[西班牙國家男子籃球隊|西班牙國家男子籃球隊]]在2006年、2009年和2011年的[[世界男籃錦標賽|世界男籃錦標賽]]奪得金牌，而在2007年歐洲男籃錦標賽和[[2008年北京奧運|2008年北京奧運]]奪得銀牌。

== NBA选秀 ==
马克·加索尔在[[2007年NBA选秀|2007年NBA选秀]]大会上，第二轮第48顺位被[[洛杉磯湖人|洛杉磯湖人]]选中。

== 技术特点 ==
马克·加索尔在选秀后并没有登陆NBA，选择继续在西班牙联赛效力。

在2008年，因為哥哥[[保羅·蓋索|保羅·蓋索]]對灰熊提出2點要求，分別是「交易我」和「給我弟弟一個機會」，他被湖人交易至孟菲斯灰熊。在灰熊队效力的几个赛季当中，马克·加索尔逐渐成长为NBA最优秀的内线球员之一。拥有2米16身高的他，拥有出色的球场视野和无私的球风，加上出色的传球能力，登陆NBA至今他每赛季平均助攻均没有低于1个。马克·加索尔的篮下技术并不输给哥哥保罗加索尔，并且马克的身体更加强壮，也并不惧怕身体对抗。马克·加索尔同时拥有出色的中距离投篮的能力，在篮筐12尺左右的距离拥有接近6成的命中率。

但马克·加索尔最擅长的其实是防守低位背打。 在2012-2013赛季中，所有防了至少150个回合低位背打的球员中，他每回合只让对手得到0.665分，命中率为35.8%，两项数据分别排名联盟第一和第二。凭借他出色的防守表现，马克·加索尔在2012-2013赛季，在最佳防守球员的比拼上击败了此前呼声最高的球员热火球星勒布朗-詹姆斯以及马刺老将蒂姆-邓肯。同时马克·加索尔也是灰熊队史上首次获得这一奖项的球员。
