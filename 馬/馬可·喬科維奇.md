'''馬可·喬科維奇'''（塞尔维亚语：Марко Ђоковић / Marko Đoković，英语：'''Marko Djokovic'''，{{bd|1991年|8月20日}}），生于南斯拉夫联邦贝尔格莱德市，当今[[塞爾維亞|塞爾維亞]]男子[[網球|網球]]運動員，而且他也是球王[[诺瓦克·喬科維奇|诺瓦克·喬科維奇]]的亲弟弟。

==家庭==
他是斯爾詹（Srdjan）和狄亞娜（Dijana）之子，他在他們三個兄弟之中是排行中間<ref>{{cite web |url=http://www.iht.com/articles/2007/06/02/sports/03tennis.php |title=Serbian players emerge from a broken country |accessdate=2009-02-17 |publisher=International Herald Tribune}}</ref>，哥哥是著名網球選手[[諾瓦克·喬科維奇|諾瓦克·喬科維奇]]，他還有一個弟弟Djordje<ref>{{cite web |url=http://sports.espn.go.com/espn/print?id=2912069&type=story |title=Ascent of Novak Djokovic has only}}</ref>。

== 網球生涯 ==
=== 早期生涯 ===
2006年，喬科維奇開始參加[[國際網球總會|ITF]]級別巡迴賽。2007年7月，他參加克羅埃西亞公開賽，在會外賽第一輪被[[巴勃羅·安杜哈爾|巴勃羅·安杜哈爾]]以6–1、6–2擊敗<ref>{{PDFlink|1=[http://www.atptennis.com/en/common/TrackIt.asp?file=http://www.atptennis.com/posting/2007/439/QS.pdf]|Croatia Open Umag – 2007 Singles qualify drawn}}</ref>，他曾與哥哥諾瓦克參加雙打比賽，不過被[[愛德華·羅傑-瓦塞林|愛德華·羅傑-瓦塞林]]與[[馬蒂厄·蒙考特|馬蒂厄·蒙考特]]擊敗。

喬科維奇曾在未來賽輸掉兩場比賽，在家鄉塞爾維亞[[诺威萨|诺威萨]]兩次數給同胞[[薩薩·斯托吉薩耶維奇|薩薩·斯托吉薩耶維奇]]。2008年，他參加澳洲網球公開賽青少年比賽，在第一輪被[[克利福德·馬斯蘭|克利福德·馬斯蘭]]以7–5、6–1所淘汰。

喬科維奇參加位於[[波德戈里察|波德戈里察]]蒙特內哥羅公開賽，決賽擊敗[[柳普科·切萊比奇|柳普科·切萊比奇]]（Ljubomir Čelebić）<ref>{{cite web |url=http://www.juniortennis.com/ajt/tournamentinfo.php?t_id=16626&h_id=2515 |title=Montenegro Open – Drawsheet |accessdate=2008-08-23 |publisher=juniortennis.com |deadurl=yes |archiveurl=https://web.archive.org/web/20110725115753/http://www.juniortennis.com/ajt/tournamentinfo.php?t_id=16626&h_id=2515 |archivedate=2011-07-25 }}</ref>2008年7月，喬科維奇收到克羅埃西亞公開賽外卡，不過被義大利籍[[弗朗切斯科·皮卡里|弗朗切斯科·皮卡里]]（Francesco Piccari）擊敗<ref>{{cite web |url=http://www.itftennis.com/mens/tournaments/drawsheetbyRound.asp?event=1100102741&tournament=1100018223 |title=Umag Open qualifying draw |accessdate=2008-08-23 |publisher=itftennis.com}}</ref>。

=== 職業生涯 ===
2008年9月，他收到[[泰國網球公開賽|泰國網球公開賽]]外卡，開始參加ATP級別比賽<ref>{{cite web |url=http://www.thailandopen.org/news_detail.php?nid=18&lang=eng |title=存档副本 |accessdate=2008-09-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20080913034528/http://www.thailandopen.org/news_detail.php?nid=18&lang=eng |archivedate=2008-09-13 }}</ref>，他在首輪被[[亞爾科·涅米寧|亞爾科·涅米寧]]以6–2、6–0所清盤淘汰。

2009年5月7日，喬科維奇與[[達科·馬扎諾維奇|達科·馬扎諾維奇]]參加家鄉[[塞爾維亞公開賽|塞爾維亞公開賽]]的雙打比賽，首輪擊敗雙打高手[[丹尼爾·內斯特|丹尼爾·內斯特]]與[[內納德·齊莫尼奇|內納德·齊莫尼奇]]，這是他在ATP級別比賽的第一勝。

2012年2月，喬科維奇收到ATP500級別比賽[[杜拜網球錦標賽|杜拜網球錦標賽]]外卡，不過在第一輪被[[安德烈·戈盧別夫|安德烈·戈盧別夫]]以6–3、6–2淘汰<ref>{{cite web |url=http://www.independent.co.uk/sport/tennis/djokovics-little-brother-fails-to-make-his-marko-7447221.html |title=Djokovic's little brother fails to make his Marko |accessdate=2012-02-28 |publisher=atpworldtour.com}}</ref>。
