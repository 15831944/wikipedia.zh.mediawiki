'''馬場站'''（{{lang-en|'''Racecourse Station'''}}）位於[[香港|香港]][[沙田區|沙田區]][[沙田|沙田]]，是[[港鐵|港鐵]][[東鐵綫|東鐵綫]]的[[鐵路車站|鐵路車站]]。於1978年10月7日随[[沙田馬場|沙田馬場]]正式開幕而启用。

==概述==
馬場站是專為[[沙田馬場|沙田馬場]]而設站，只在該處舉行賽馬時、進行雙邊投注及有大型活動舉行才開放使用。

馬場站在港鐵票務系統屬於「東鐵綫本地路段」；不過由於其只服務馬場，因此大部份優惠都不適用此站。不過馬場站在[[兩鐵合併|兩鐵合併]]時有減價，學生亦可繳付「特惠車費」，[[長者及合資格殘疾人士公共交通票價優惠計劃|長者及合資格殘疾人士公共交通票價優惠計劃]]亦適用於此站；而[[上水／烏溪沙－尖東全月通加強版|上水／烏溪沙－尖東全月通加強版]]<ref>{{Cite web|language=zh-hant|publisher=港鐵公司|title=全月通加強版|url=http://www.mtr.com.hk/ch/customer/tickets/monthly_pass_extra.html|author=港鐵公司|accessdate=2015-09-07}}</ref>或部份港鐵紀念車票<ref>{{Cite web|language=zh-hant|publisher=港鐵公司|title=機場快綫十周年THOMAS & FRIENDS 港鐵紀念車票|url=http://www.mtr.com.hk/chi/whatsnew/ael10anniticket.html|author=港鐵公司|accessdate=2008-12-30}}</ref>並不適用於此站。

另外，因應[[香港賽馬會|香港賽馬會]]的規定，一般情況下未滿18歲的人士在賽馬日不可進入馬場。因此供3歲至11歲小童遊客使用的「[[遊客全日通|小童遊客全日通]]」'''無法'''於此站使用。同時小童[[八達通|八達通]]及未滿18歲的人士所持有的個人八達通（包括「學生身分」或「殘疾人士身分」個人八達通）亦不適用於此站（沙田馬場舉行的非賽馬活動除外）。

由于本站與[[火炭站|火炭站]]平行，列車若停靠本站就無法停靠火炭站。因此在東鐵綫所有停靠本站的班次，其班次信息显示屏显示其终点站上会附加“经马场”的指示。

==車站設計==
本站位於[[港鐵車廠#何東樓車廠|港鐵何東樓車廠]]和[[大埔公路|大埔公路]]之間，月台位於沙田馬場看台對面[[駿景園|駿景園]]平台下方（但沒有直接通道連接駿景園）；大堂則並非位於車站之內，而是位於馬場看台。本站有3條[[行人天橋|行人天橋]]連接。

===大堂===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Racecourse Station Entrance Gate 1 201406.jpg | caption1 = 大堂及沙田馬場的閘口
}}
馬場站大堂位於[[沙田馬場|沙田馬場]]看台內，因此大堂並非位於月台的正上方，並被[[大埔公路|大埔公路]]相隔<ref name="sta_info">{{cite book|author=港鐵公司|title=《馬場站車站指南》|publisher=港鐵公司|date=2009年8月}}</ref>。乘客如需往返大堂及月台，必須利用橫跨大埔公路的多條行人天橋往返。由於該行人天橋連接看台的中央及南面，因此車站分為北面及南面兩個大堂，兩個大堂均個別設有客務中心及出入閘機。

由於馬場站不是經常開放，加上大堂位於馬場看台內，所以它是東鐵綫裡唯一一個不設公眾[[廁所|洗手間]]、[[自動售票機|自動售票機]]、[[扶手電梯|扶手電梯]]及[[升降機|升降機]]的車站，乘客需在其他車站多購一張[[車票|車票]]或使用八達通作回程之用，否則便要在客務中心排隊購票。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Racecourse Station 2019 03 part2.jpg | caption1 = 1號月台﹙往羅湖／落馬洲方向﹚
| image2 = Racecourse Station 2019 03 part1.jpg | caption2 = 2號月台﹙往紅磡方向﹚
}}

馬場站只設2個月台，位於[[港鐵車廠|港鐵何東樓車廠]]旁，以[[島式月台|島式月台]]排列。月台上沒有標示特定列車目的地，通常2號月台為南行列車；1號月台為北行列車，有部分由本站始發的南行列車由1號月台開出。

由於駛經馬場站的列車並不途經[[火炭站|火炭站]]，如果乘客需要由該站前往火炭站，必須在[[沙田站|沙田站]]或[[大學站_(香港)|大學站]]轉乘經火炭的列車前往火炭站。而從沙田馬場可以步行取道跨越大埔公路之行人天橋和[[銀禧花園|銀禧花園]]通道前往火炭站。

當有城際客運列車通過火炭站而前一班列車途經本站，南北行列車將於本站待避通過火炭站的城際客運列車。

==利用情況==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Racecourse Station 2017 10 part13.jpg | caption1 = 看台出入口站牌
}}
由於馬場站只在[[沙田馬場|沙田馬場]]賽馬日（一般在星期六、日或假期的日間，以及星期三的夜間）、進行雙邊投注及有大型活動舉行開放予乘客，因此車站只會在沙田馬場賽馬日、進行雙邊投注及有大型活動舉行才有乘客使用。平日並不提供服務（除了在清晨及晚上抵達或離開香港的直通車頭、尾班車，或進入何東樓北廠貼近馬場站北行軌道暫泊的直通車車卡會使用馬場站軌道，避免妨礙通勤列車調度進出[[港鐵何東樓車廠|港鐵何東樓車廠]]，以及特别情況例外）。

雖然本站是專為沙田馬場而設。但由於在馬場舉行賽事時並不是每班東鐵綫列車都會停靠本站，部分仍然會停靠[[火炭站|火炭站]]，即使停靠本站，班次也不是十分頻密，故部分前往沙田馬場的乘客會選擇在火炭站下車步行前往。

===翻新月台===
兩鐵合併後，港鐵公司翻新東鐵綫各月台，以遮掩東鐵綫車站月台殘舊的觀感。工程包括重新為各車站配主題色，並加入代表花種，為月台橫樑及柱子翻上月台配色的油漆，貼上印有車站主題花、建築物的牆紙，加設/更新直立站牌、路線圖、指示牌、告示等等。此站配色為綠色，主題花為馬尾草，街景為沙田馬場及賽馬。

==未來發展==
===月台閘門===
東鐵綫沿線各車站未有安裝[[月台閘門|月台閘門]]，有關工程將會與[[沙田至中環綫|沙田至中環綫]]一同進行，以解決技術困難，預計於2020年完成<ref>[http://www.mtr.com.hk/eng/corporate/file_rep/PR-12-016-C.pdf 港鐵「用心聽·用心做」計劃 提升車站設施 並訂出新方向在轉綫車站設置公共洗手間]</ref><ref>[http://hk.news.yahoo.com/港鐵東鐵線加裝月台閘門預計2020年運作-050300003.html 港鐵東鐵線加裝月台閘門預計2020年運作]</ref>，馬場站將率先進行。<ref>{{cite news|url=http://orientaldaily.on.cc/cnt/news/20130707/00176_094.html|title=政情：軟功冧議員 港鐵衰咗|work=東方日報|date=2013年7月7日|language=zh-hk}}</ref>
