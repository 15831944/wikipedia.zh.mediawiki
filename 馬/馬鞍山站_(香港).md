'''馬鞍山站'''（{{lang-en|'''Ma On Shan Station'''}}）為[[香港|香港]]的[[港鐵|港鐵]][[馬鞍山綫|馬鞍山綫]]的一個[[架空車站|架空車站]]，位於[[沙田區|沙田區]][[馬鞍山_(香港市鎮)|馬鞍山]][[新港城中心|新港城中心]]與[[馬鞍山廣場|馬鞍山廣場]]之間一段[[西沙路|西沙路]]地面及架空路段。

==結構==
馬鞍山站設有兩條[[樓梯|樓梯]]與[[電動扶梯|扶手電梯]]，以及一部[[升降機|升降機]]，連接月台及大堂兩層。

===大堂===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Ma On Shan Station 2017 11 part6.jpg | caption1 = 大堂
}}
大堂收費區設有7間商舖，包括[[麵包糕餅店|麵包糕餅店]]、[[便利商店|便利店]]、外帶壽司店、自動櫃員機<ref name="shop"/>。另外亦設有3個報紙櫃供乘客於早上免費取閱。另外，大堂近B出口的非付費區設有「[[會員服務站|會員服務站]]」，供乘客享用港鐵友禮會的會員優惠<ref>{{cite web|title=鐵路四處遊－「會員服務站」|url=http://www.mtr.com.hk/ch/customer/services/csp.html|work=港鐵網頁|publisher=港鐵公司|accessdate=2016年9月26日|language=zh-hant}}</ref>。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Ma On Shan Station 2018 06 part2.jpg | caption1 = 1號月台
| image2 = Ma On Shan Station 2018 06 part6.jpg | caption2 = 月台佈置
}}
馬鞍山站設有兩個月台，以島式月台設計排列。月台可容納8卡列車，初時月台只建成其中4至5卡長度，其餘部分會於沙中綫興建時於預留結構上擴建，現時擴建工程已經完成。

===出入口===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Ma On Shan Station 2018 06 part7.jpg | caption1 = A出入口
| image2 = Ma On Shan Station 2018 06 part8.jpg | caption2 = B出入口
}}
馬鞍山站是唯一所有出入口均連接商場的港鐵站。車站啟用初期的設計，基於已有行人天橋連接新港城中心及馬鞍山廣場，兩邊出入口互不相通。港鐵在2013年進行車站改善工程，除在大堂加建商店外，亦將向鞍源街一邊大堂改建為非付費區，打通兩個出入口，並增加商店。

==鄰接車站==
馬鞍山站是馬鞍山綫的中途站，乘客可在該站乘坐往[[烏溪沙站|烏溪沙]]或[[大圍站|大圍]]列車，往返新界東各地。
