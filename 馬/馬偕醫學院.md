'''馬偕學校財團法人馬偕醫學院'''（{{lang-en|'''Mackay Medical College'''}}），是一所由[[台灣長老教會|台灣基督長老教會]]創立於[[台灣|台灣]]新北三芝的私立[[醫學院|醫學院]]。2009年3月30日，[[中華民國教育部|教育部]]通過該校的立案申請，准予立案招生，成為全台第12所醫學院。<ref>{{cite news|url=http://health.chinatimes.com/contents.aspx?cid=5,68&id=5810|title=馬偕醫學院 8月加入招生|publisher=[[中國時報|中國時報]]|author=陳至中|date=2009年4月18日|accessdate=2009-04-18|deadurl=yes|archiveurl=https://web.archive.org/web/20131014233530/http://health.chinatimes.com/contents.aspx?cid=5,68&id=5810|archivedate=2013年10月14日}}</ref>
擁有1.5公頃的自然森林。

2017年7月下旬傳出與[[真理大學|真理大學]]、[[馬偕護專|馬偕護專]]整併消息<ref>{{Cite web |url = https://pttweb.tw/thread/m-1354662811-a-459 |title = 紀念創辦人，真理大學改名馬偕大學 |publisher = 中國時報  |date = 2012-12}}</ref>，2020年前併校並更名為馬偕大學，成為馬偕大學醫學院校區<ref>{{Cite web |url = https://money.udn.com/money/story/5641/2598328 |title = 真理馬偕3校 擬3年內合併 |publisher = 聯合財經網 |date = 2017-07-22 |access-date = 2017-07-22 |archive-url = https://web.archive.org/web/20171201042215/https://money.udn.com/money/story/5641/2598328 |archive-date = 2017-12-01 |dead-url = yes }}</ref>。 此消息校方已證實為誤傳

==概要==
馬偕醫學院，是為了紀念[[馬偕|馬偕]][[博士|博士]]在台灣的[[醫療|醫療]]事業、以及培養醫療人才而設立。校慶日定為3月21日，即馬偕的[[生日|生日]]。

目前馬偕醫學院部分醫療課程與[[國立陽明大學|國立陽明大學]]採[[策略聯盟|策略聯盟]]方式合作，通識課程則與[[北二區區域教學資源中心|北二區區域教學資源中心]]所開設的「夏季學院」相互承認。

==校史==
===大事年表===
*1995年10月，成立醫學院籌備小組。
*1997年3月，向教育部提出設校申請。 
*1999年3月，成立醫學院籌備處，位於台北馬偕醫院及淡水院區。
*1999年5月，聘高銘憲教授為籌備處主任。 
*2000年1月，向教育部提出詳細籌設計畫書修訂本。
*2003年3月，內政部區域計劃委員會通過校園整體開發計畫。
*2005年2月，通過教育部許可籌設。
*2006年12月，財團法人設立登記。
*2007年7月，取得建造執照。
*2008年4月，第一期建築工程施工。
*2008年5月，聘方菊雄教授擔任籌備處主任。
*2008年9月20日，上樑感恩禮拜。
*2009年1月，第2屆董事會選出黃俊雄先生擔任董事長。
*2009年2月，通過立案申請。
*2009年3月，通過招生辦法審核，通過教育部立案並准予招生(台高(四)字第0980050130號)。
*2009年4月，教育部通過任用魏耀揮校長。
*2009年4月18日，第一期校舍落成感恩典禮。
*2009年7月，招收第一屆學生。
*2009年9月12日，首任校長授職暨創校開學典禮。
*2010年3月20日，第一屆校慶，訂於馬偕的誕辰前一日。
*2011年10月，獲教育部通知准予籌設'''生物醫學研究所'''<ref>{{cite news|url=http://news.pchome.com.tw/living/awakening/20120313/index-13316353204732243009.html |title= 馬偕醫3週年 朝醫學大學邁進|publisher= [[PChome|PChome]] |author= |date=2012年3月，13日， |accessdate=2012-03-13}}</ref>。
*2012年9月，招收第一屆碩士班研究生，增設聽力暨語言治療學系。
*2013年3月23日，第二期校舍新建工程落成。
*2013年6月15日，舉行第一屆護理學系畢業典禮。
* 2015年1月20日，更名為馬偕學校財團法人馬偕醫學院。
