'''馬-{啓}-仁'''{{NoteTag|關於'''馬啓仁'''的名字中間的為「啓」字的正確寫法，並非「啓」字寫法。因技術所限「啓」字在不同字型會有不同寫法，但「啓」字則能在不同字型正確顯示。馬啓仁是使用「啓」字為名，所以以此為正寫。}}（{{lang-en|Keyman Ma}}，{{bd|1970年|6月9日|catIdx=M马}}），[[香港|香港]]的[[運動|運動]]評述員、[[主持人|主持人]]及[[演員|演員]]。

== 簡歷 ==
[[File:HK_KeymanMa_2011.JPG|thumb]]
馬啓仁小學就讀[[馬頭涌官立小學|馬頭涌官立小學]]，中學就讀於[[余振強紀念中學|余振強紀念中學]]，1993年於香港樹仁學院（現改稱[[香港樹仁大學|香港樹仁大學]]）社會系畢業。畢業後加入[[香港有線電視|香港有線電視]]擔任體育賽事評述及主持。1999年完成[[澳洲|澳洲]][[臥龍崗大學|臥龍崗大學]]的新聞學碩士課程。曾任[[華僑日報|華僑日報]]體育報記者，2003年加入[[香港賽馬會|香港賽馬會]]任職市場傳訊經理，幫助賽馬會推廣[[足球|足球]]運動。

2007年8月，馬啓仁轉投至[[無綫電視|無綫電視]]旗下擔任評述員及節目主持人。2011年，再轉投[[now寬頻電視|now寬頻電視]]擔任遊戲節目《[[撳錢|撳錢]]》主持，但將不會負責體育節目（now sports）{{NoteTag|now寬頻電視於宣傳片中曾將馬列為2012年歐洲國家盃的星級評述員之一，但仍未能確實馬會為2012年歐洲國家盃擔任旁述。}}。2011年4月17日，馬聯同[[彭晴|彭晴]]擔任[[now香港台|now香港台]]節目《[[香港電影金像獎|香港電影金像獎]]》錄影廠部分的主持<ref>http://www.am730.com.hk/article.php?article=53666  {{Wayback|url=http://www.am730.com.hk/article.php?article=53666 |date=20160304221608 }}</ref>。

2014年5月，馬啓仁重返無綫電視擔任主持[[2014年國際足協世界盃|世界盃]]至2015年。

2016年，他成立了製作公司並和Now sports合作，製作[[2016歐洲國家杯|2016歐洲國家杯]]的節目。

2018年夏天，他為[[ViuTV|ViuTV]]與[[Now_TV|Now TV]]擔任[[2018年國際足協世界盃|世界盃]]主持。

== 個人生活 ==
馬啓仁於1997年結婚，現時育有一女兒名冬晴。

== 評述 ==
===有線電視===
馬啓仁為有線電視青訓產品，有線電視體育台成立不久後，於1993年被發掘入行，原先是當《[[華僑日報|華僑日報]]》的記者。首次直播評述足球球賽是在1995年的[[欧洲冠军联赛|欧洲冠军联赛]]賽事，[[拜仁慕尼黑足球俱乐部|拜仁慕尼黑]]對[[阿積士|阿積士]]；而最難忘的賽事是[[1998年世界盃足球賽|1998年世界盃]]中[[阿根廷國家足球隊|阿根廷]]被[[荷蘭國家足球隊|荷蘭]]淘汰出局。在足球評述賽事時，大多數是結紫色[[領帶|領帶]]，以表明自己支持[[西汉姆联足球俱乐部|-{zh-hk:韋斯咸;zh-tw:西漢姆聯;}-]]（-{zh-hk:韋斯咸;zh-tw:西漢姆聯;}-WEST HAM的代表顏色是[[紫色|紫色]]）。他曾評述1998年、2002年及2006年的[[世界盃足球賽|世界盃]]及1996年、2000年及2004年[[欧洲足球锦标赛|歐洲國家盃]]等重要賽事，此外主要評述[[英格蘭足球超級聯賽|英超聯]]。在有線電視任職時，經常與[[李德能|李德能]]及[[蔡文堅|蔡文堅]]合作評述英超足球比賽。除了被安排足球比賽外，曾評述[[NBA|NBA]]、排球及[[2006年亞洲運動會|亞運會]]。在2007年7月，馬啟仁與有線電視合約期滿，有線電視開出合約，但馬啓仁沒有續約，結束在[[有線電視|有線電視]]的14年職業生涯。

===其他===
另外，馬啓仁曾被[[新城電台|新城財經台]]節目《足球王》邀請作嘉賓，在2006年[[賀歲盃足球賽|賀歲盃足球賽]]賽事中與[[李德能|李德能]]評述足球賽事。

===無綫電視===
後來轉會至無綫電視擔任旁述一職，第一場與[[鍾志光|鍾志光]]合作評述[[巴塞罗那足球俱乐部|-{zh-hans:巴塞罗那;zh-hk:巴塞隆拿;zh-tw:巴塞隆納;}-]]對[[北京国安足球俱乐部|北京国安]]比賽。現時主要評述[[香港甲組足球聯賽|港甲]]、[[荷兰足球甲级联赛|荷甲]]精華等賽事，另外主持節目「[[體育世界|體育世界]]」，以及主持[[2008年夏季奧林匹克運動會|北京奧運]]節目。馬啓仁透露，他將會朝着主持人身份工作。在10月21日，停播已久的足球節目－「球迷世界」復播，由馬啓仁夥拍[[伍晃榮|伍晃榮]]及鍾志光一同主持。馬啟仁加入無線電視的第二場評述在2008年2月7日及10日所舉行的賀歲盃足球賽。當時所有足球比賽均由馬啓仁及鍾志光評述。當中馬啓仁主要負責評述重要賽事，如[[2008年欧洲足球锦标赛|-{zh-hans:欧洲足球锦标赛;zh-hk:歐洲國家盃;zh-tw:歐洲足球錦標賽;}-]]。另外在[[2007年至2008年香港足總盃|2007年至2008年香港足總盃]]決賽的為[[汶川大地震|汶川大地震]]的籌款拍賣。

除了足球比賽外，馬啓仁也有評述[[NBA|NBA]]籃球賽及[[NBA中國賽|NBA中國賽]]，主持明珠台節目《NBA精華速遞》。首場評述NBA賽事是[[芝加哥公牛|芝加哥公牛]]對[[休斯頓火箭|休斯頓火箭]]。

2014年5月重返無綫電視並主持世界盃至2015年約滿。

===ViuTV / Now TV===
2018年6月加盟[[ViuTV|ViuTV]]/[[Now_Sports|Now Sports]]擔任2018世界盃主持及評述。

== 瑣事 ==
=== Keyman 的意思 ===
馬啓仁自己解釋Keyman這名字是其中文名的意譯，是馬啓仁父親在他小時候改的，Key有開啓的意思，而仁的讀音是人，所以譯為man，合起來即是Keyman，在他讀大專時才開始使用。

=== 愛好 ===
馬啓仁最喜愛的[[足球|足球]]隊是[[西汉姆联足球俱乐部|-{zh-hans:西汉姆联;zh-hk:韋斯咸}-]]和[[阿根廷国家足球队|阿根廷國家隊]]。最欣賞球員則為[[胡安·巴勃罗·索林|-{zh-hans:胡安·帕伯罗·索林;zh-hk:蘇連}-]]（[[阿根廷|阿根廷]]球員）和[[保罗·迪卡尼奥|-{zh-hans:迪卡尼奥; zh-hk:迪肯尼奧;}-]]（[[意大利|意大利]]球員、曾是-{zh-hans:西汉姆联;zh-hk:韋斯咸}-球員）。平日的喜好則是[[跑步|長跑]]、[[卡拉OK|卡拉OK]]、閱讀、踢足球及游泳。

=== 電玩 ===
他平時亦會玩一些[[電玩|電玩]]，而他最愛的遊戲是[[冠軍足球經理|冠軍足球經理]]（{{lang|en|Championship Manager}}）。[[冠軍足球經理|冠軍足球經理]]現而易名為[[足球經理人|足球經理人]]（{{lang|en|Football Manager}}），他一直都有收藏。另外還有[[胜利十一人|-{zh-hans:胜利十一人;zh-hk:實況足球}-]]。

== 雜誌創辦 ==
馬啓仁在小時候已希望創辦一本雜誌，在1996年一個偶然的機會下，跟一間將結業的遊戲雜誌社出版一本[[國家籃球協會|NBA]]的籃球雜誌，名為《'''入樽'''》，當時市面上並沒有相關中文的籃球雜誌，這本書出版了八千冊，全部售罄。但馬啓仁當時並沒有把自己的名字加入這本雜誌內，引致他決心在下一次有機會創辦雜誌時，一定要有自己的名字。而他最希望尋回《'''入樽'''》這本雜誌。

===Keyman Show===
馬啓仁後來出版了共九期的足球雜誌—《Keyman Show》，在2001年9月1日首次出版，初時定價為20元，以雙週刊出版，後來再以月刊出版，定價增加至25元，在2002年4月1日出版第九期後，後來原定出版第十期關於世界盃的特刊時，但街上報攤關於世界盃的書籍甚多，馬啓仁認為沒有生存空間，另外工作繁多和人手不足等問題下，於是第十期結果沒有出版，第九期成為《Keyman Show》的最後一期，不會再次出版《Keyman Show》。現時馬啓仁在[[蘋果日報_(香港)|蘋果日報]]和《Milk》雜誌中MikiSport的專欄名稱也是《Keyman Show》，專欄內容主要圍繞其喜愛球隊和一些足球趣聞為主，亦有球迷投稿。而蘋果日報中的專欄簽名為「馬啓仁，愈講愈有味道，堅持『鎚仔』先係正路的球評家。」

===Keyman Soho===
馬啓仁在推出九期《Keyman Show》共蝕了30萬港元，但他捲土重來夥拍綜合體育雜誌《運動版圖》（Sportsoho）的老闆陳冠英合作推出雜誌《足球版圖》（Keymansoho），初時以一書兩冊月刊形式出版，兩書的營運成本由陳冠英包辦，馬啓仁只須負責《足球版圖》的編採工作，盈利由二人對分，2009年11月創刊，在馬啓仁的名人效應下，賣出超過13,000本，於2010年5月第七期才獨立分拆以單行本出售，但於2011年2月第十六期出版後因長期虧損而宣佈停刊。2011年5月透過與「505 United」合作用《keymansoho》名義復刊，但在〈13個出書的藉口〉中馬啓仁表示雜誌隨時會「摺埋」。2011年10月，馬啓仁在 facebook宣佈《keymansoho》停刊。
