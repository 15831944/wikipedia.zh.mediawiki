'''馬修·查爾斯·桑德斯'''（英語：Matthew Charles Sanders，{{bd|1981年|7月31日}}），藝名'''Matt Shadows'''，通常被稱為'''M. Shadows'''，是[[美国|美國]][[歌手|歌手]]、[[作曲家|作曲家]]、[[音乐家|音樂家]]。其最主要身份為美國[[重金属音乐|重金屬]]樂團[[七級煉獄|七級煉獄]]的主唱及創始成員。在2017年，被{{Link-en|Ultimate Guitar|Ultimate_Guitar}}列為「當代25大傑出主唱」第3位。<ref>{{Cite web|url=https://www.ultimate-guitar.com/articles/features/friday_top_25_greatest_modern_frontmen-59851|title=Friday Top: 25 Greatest Modern Frontmen|accessdate=2017-12-19|author=|date=2017-3-19|publisher=Ultimate Guitar}}</ref>

== 早期經歷 ==
Shadows在1981年7月31日出生於[[美国|美國]][[加利福尼亚州|加州]]的[[亨廷頓比奇|亨廷頓比奇]]（英語：Huntington Beach）。他在很小的時候就開始學習[[歌唱|歌唱]]，而他是開始學[[电吉他|吉他]]及年齡稍大後才漸漸萌生對[[摇滚乐|搖滾樂]]、[[重金属音乐|金屬樂]]的興趣。他將他早期獲得的音樂經驗及鋼琴結合起來，作為他提升自己吉他及歌唱技巧的養分。他進入{{Link-en|亨廷頓比奇中學|Huntington_Beach_High_School}}，並在那裡短暫地組了一個叫「[[Successful_Failure|Successful Failure]]」的[[朋克搖滾|龐克]]樂團，擔任的是吉他手。在之後的1999年與中學同學[[Zacky_Vengeance|Zacky Vengeance]]、[[The_Rev|The Rev]]以及[[Matt_Wendt|Matt Wendt]]一起組了[[七級煉獄|七級煉獄]]。

== 藝名 ==
如同七級煉獄裡的其他團員一樣，M. Shadows這個名字是藝名。在接受採訪時，M. Shadows提到關於當初取藝名的原因：「Matt」取自其本名的「Matthew」，而本名念起來並不順口；而「Shadows」則是因為，他認為他是樂團裡面的黑暗人物（darker character of the group）。他還補充道，之所以樂團中的大家都取了藝名，是因為影響他們的許多樂團的成員也都使用了藝名（如[[枪与玫瑰|槍與玫瑰]]的[[史萊許|Slash]]、[[Korn|Korn]]的{{Link-en|Munky|James_Shaffer}}以及[[滑結樂團|滑結樂團]]的成員們）。

== 演唱風格 ==
雖然Shadows是屬於[[男中音|男中音]]，但是經過他運用自己的技巧，他能夠在他所選擇的音樂風格內，舒服而驚人地演唱男高音。他的演唱技巧及風格經過了若干年，有顯著的進化和發展。在樂團的第一張錄音作品''{{Link-en|Sounding the Seventh Trumpet|Sounding_the_Seventh_Trumpet}}''中，他展現了渾厚而粗重的[[金屬蕊|金屬蕊]]吼腔技巧，同時清腔技巧卻有所限制。2003年發行的''{{Link-en|Waking the Fallen|Waking_the_Fallen}}''中，他展現了他在具有旋律性的聲線上的進展，但仍保持了吼腔技巧帶來的影響。風格改變最明顯的，莫過於他在樂團2005年發行的大碟''{{Link-en|City of Evil|City_of_Evil}}''中的表現，該專輯以減少許多比重的吼腔當作[[和声|和聲]]，主要以強有力的旋律清腔構成人聲的部分。在這張專輯錄製前，Shadows找上曾經與[[埃克索尔·罗斯|Axl Rose]]、[[克里斯·康奈爾|Chris Cornell]]合作過的聲樂指導[[Ron_Anderson|Ron Anderson]]，並與之合作了幾個月，因為Shadows想在自己的聲線中加入一些顆粒感更重的元素。<ref>{{Cite web|url=http://www.blistering.com/fastpage/fpengine.php/link/1/templateid/10992/|title=Avenged Sevenfold Feature Interview|accessdate=2017-12-24|author=|date=2006|publisher=Blistering|deadurl=yes|archiveurl=https://archive.is/20120529161111/http://www.blistering.com/fastpage/fpengine.php/link/1/templateid/10992/|archivedate=2012-05-29}}</ref>這個變化也導致了這張專輯中含有大量樂團其他成員的和聲，尤其是現場表演的時候，而這一特色也一直保留至今。

有謠傳說，因為2003年的[[Warped巡迴|Warped巡迴]]之後進行的[[咽喉|喉嚨]][[外科手術|手術]]，Shadows在那之後就喪失了吼腔的能力。但是，[[音樂製作人|製片人]]{{Link-en|Andrew Murdock|Andrew_Murdock}}如此回覆來否認這些謠言：「當''Sounding the Seventh Trumpet''發行後，我遇見他們……Matt將CD拿給我並說道：『這張專輯是吼的。接下來我們想做的音樂，是要半吼半唱的，我不想再吼了，之後我們的歌都會是用唱的。』」<ref>{{Citation|last=Steve Rennie|title=M Shadows, Avenged Sevenfold Singer - Renman LIVE #098|date=2014-10-28|url=https://www.youtube.com/watch?v=CxUgRVyq1Jg|accessdate=2017-12-24}}</ref>

== 影響 ==
Shadows借鑒了傳統重金屬音樂的許多元素來重新演繹他的演唱風格。他提到槍與玫瑰是影響他最大的樂團，並說：「我很愛那個團！你可以隨你想要的來比較我們──因為那就是我玩樂團、做音樂的原因！」此外，他提到過深重影響他的樂團還有[[金屬製品|金屬製品]]、[[麦加帝斯|麥加帝斯]]、{{Link-en|Pantera|Pantera}}、[[超級殺手合唱團|超級殺手]]、[[鐵娘子樂團|鐵娘子]]、[[奥齐·奥斯本|Ozzy Osbourne]]、[[黑色安息日|黑色安息日]]、[[齐柏林飞船_(乐团)|齊柏林飛船]]、[[AC/DC|AC/DC]]、{{Link-en|Blind Guardian|Blind_Guardian}}、[[夢劇院_(美國)|夢劇場]]、[[滑結樂團|滑結樂團]]、[[野獸男孩|野獸男孩]]，以及吉他手[[Scott_Taylor|Scott Taylor]]。

== 其它計畫 ==
Shadows曾在許多音樂人的專輯中客串過。他曾在{{Link-en|Steel Panther|Steel_Panther}}的''{{Link-en|Feel the Steel|Feel_the_Steel}}''專輯中「Turn out the Lights」一曲唱了主歌。他也在2007年{{Link-en|The Confession|The_Confession_(band)}}的''Requiem''專輯中擔任製作人，這也是七級煉獄在2007年自行製作同名專輯前跨出的第一步。[[狂野夏洛特|狂野夏洛特]]的''{{Link-en|Good Morning Revival|Good_Morning_Revival}}''專輯中「The River」一曲，Shadows在第二段主歌後客串出現，而七級煉獄的吉他手Synyster Gates也在吉他solo中客串出現。在2012年，Shadows和Gates在戰爭[[电子游戏|電玩遊戲]][[決勝時刻：黑色行動II|決勝時刻：黑色行動II]]中以[[配音|配音]]的方式客串。在[[史萊許|Slash]]的同名solo專輯中的「Nothing to Say」一曲，Shadows也擔任了客座主唱。

==私人生活 ==
2009年10月17日，桑德斯與[[Valary_DiBendetto|Valary DiBendetto]]結婚。他們在2012年及2014年生下兩個兒子。{{Link-en|Amy Sanders|Amy_Sanders}}是大Shadows兩歲的姊姊<ref>{{Cite web|url=http://www.a7xes.com/page540.php|title=Archived copy|accessdate=2017-12-24|author=|date=2015-4-25|publisher=|deadurl=yes|archiveurl=https://web.archive.org/web/20150425045023/http://www.a7xes.com/page540.php|archivedate=2015-04-25}}</ref>。Shadows在七級煉獄的團員Synyster Gates，與Valary DiBendetto的姊妹[[Michelle_DiBendetto|Michelle DiBendetto]]結婚。


在2018年7月，Avenged Sevenfold的Facebook帐户发布了桑德斯写的一篇文章，声称他患有严重的声带病毒感染，这暂时影响了他的唱歌能力。 Avenged Sevenfold取消了接下来三个月的所有巡演/音乐会日期，以便让桑德斯能够恢复。<ref>{{cite web|url=https://www.rollingstone.com/music/music-news/avenged-sevenfold-cancel-summer-tour-after-infection-leaves-m-shadows-voiceless-702260/|first=Daniel|last= Kreps| title = Avenged Sevenfold Cancel Tour After Infection Leaves M. Shadows ‘Voiceless’|date=JULY 20, 2018 |publisher=rollingstone |accessdate=March 28, 2018|via=Rolling Stone}}</ref>
