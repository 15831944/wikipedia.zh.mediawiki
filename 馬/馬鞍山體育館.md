'''馬鞍山體育館'''（{{lang-en|'''Ma On Shan Sports Centre'''}}）位於[[香港|香港]][[新界|新界]][[沙田區|沙田區]][[馬鞍山_(香港市鎮)|馬鞍山]]，是區內第5所體育場地，於2004年10月1日啟用，總面積6,390[[平方米|平方米]]，其橢圓形的天花頂以及中間凹槽做成的天窗，以採納天然光照射的新穎設計最具特色。天花頂全長70[[米_(單位)|米]]，最高點有22.1米高。

== 場內設備 ==
該體育館設有1個多用途主場，同時可以用作2個[[籃球|籃球]]場、2個[[排球|排球]]場、8個[[羽毛球|羽毛球]]場或1個[[手球|手球]]場。有1個可以容納1,500人的觀眾席。其他設施包括3個多用途活動室、乒乓球室、[[舞蹈|舞蹈]]室及[[康文署健身室|健身室]]，更有一個以「奇趣礦洞」為主題的兒童遊戲室。另外，亦設有一間體育用品商店。

=== 臨時體操訓練中心 ===
2004年體育學院清拆全港唯一合規格的體操館（騰龍館），體操運動員被迫暫用馬鞍山體育館作為訓練基地，直到[[順利邨體育館|順利邨體育館]]的體操館於2005年12月15日落成後才再次獲得永久訓練場地。

== 奧運馬術 ==
因舉辦[[2008年夏季奧林匹克運動會馬術項目|2008年奧運馬術項目]]而遷拆的[[香港體育學院|香港體育學院]]，暫時遷往[[烏溪沙青年新村|烏溪沙青年新村]]，需使用[[康文署|康文署]]轄下位於[[沙田區|沙田區]]的體育設施作為個別精英項目的臨時訓練場地，馬鞍山體育館暫時被利用為[[羽毛球|羽毛球]]及[[武術|武術]]運動員訓練場地，[[公眾人士|公眾人士]]在這段期間，只能夠在精英運動員訓練時段以外使用。

== 開放時間 ==
開放時間為每日07:00-23:00。
