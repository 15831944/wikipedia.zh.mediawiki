'''馬略卡島'''（{{lang-ca|'''Mallorca'''}}）是[[西班牙|西班牙]][[巴利阿里群島|巴利阿里群島]]的最大島嶼，位於西[[地中海|地中海]]，是著名的旅遊點和[[觀鳥|觀鳥]]去處，旅客主要來自[[英國|英國]]、[[德國|德國]]、[[愛爾蘭|愛爾蘭]]和[[斯堪的納維亞半島|斯堪的納維亞半島]]。中世紀以[[瓷器|瓷器]]著名（意大利花飾瓷器），多[[古羅馬|古羅馬]]、[[腓尼基|腓尼基]]和[[迦太基|迦太基]]遺址。其首府[[帕爾馬_(馬略卡)|帕爾馬]]位於島西南方一海灣處，它同時是整個自治區的首府。贯通全岛北部的Tramuntana山脉地区由于独特的文化景觀被联合国教科文组织于2011年评选为世界文化遗产。 马洛卡与875277个居民（2015）是巴利阿里群岛人口最多的岛屿，也是西班牙人口第二多的岛屿，仅次于[[加那利群岛|加那利群岛]]的[[特内里费岛|特内里费岛]]<ref>[https://www.boe.es/boe/dias/2016/12/17/pdfs/BOE-A-2016-11994.pdf Datos oficiales del Instituto Nacional de Estadística, ver 01/01/2016]</ref>。

== 人口 ==
该岛人口共約75萬，其中37萬人居於首府，其次是[[馬納科爾|馬納科爾]]（28,000）、Inca（21,500）、Llucmajor（19,000）和（11,000）。此外還有3萬至8萬名旅客暫居。島上居民主要說[[加泰羅尼亞語|加泰羅尼亞語]]和[[西班牙語|西班牙語]]，兩者同時亦是官方語言，年輕的馬略卡居民也懂[[英語|英語]]。

== 地理 ==
此島面積為3,640.11平方公里，馬約卡岛南北長75公里，东西100公里，西北部的山脈長70公里、海岸多懸崖且山石崎嶇。全島最高點是Puig Major（1445米），但由於是軍事禁區，所以鄰近的Puig de Massanella（1364米）是可到達的最高峰。中部由帕爾馬伸延出來的是肥沃的平原。

== 氣候 ==
馬略卡島屬[[地中海氣候|地中海氣候]]，夏天陽光充沛而溫暖，冬天清涼而多雨，但北部的山脈則較寒冷，偶爾下雪。一月氣溫8至15度，七月21至29度。冬天平均降雨日數約5天，雨量在50毫米左右。

== 歷史 ==
[[File:Mallorca_in_summer.jpg|thumb]]的馬略卡島]]

* 450年[[汪達爾人|汪達爾人]]佔領巴利阿里群島。
* 534年[[拜占庭|拜占庭]]人佔領馬略卡島。
* 902年被[[摩爾人|摩爾人]]征服。
* 1229年被[[阿拉貢王國|亞拉岡]]国王[[海梅一世|海梅一世]]佔領，在此之前為[[阿拉伯人|阿拉伯]]或摩爾人管治。
* 1276年[[馬略卡王國|馬略卡王國]]成立。
* 1349年至19世紀成為自治區，馬略卡島直接由內陸管理。
* 19世紀旅遊業興起，波蘭作曲家[[弗雷德里克·肖邦|蕭邦]]和法國作家[[喬治·桑|喬治·桑]]曾逗留在[[法德摩薩|法德摩薩]]。
* 1936年[[西班牙內戰|西班牙內戰]]中，馬略卡島為[[長槍黨|長槍黨]]所佔據。
* 1950年代開始發展旅遊業，馬略卡島漸漸吸引外國遊客和加泰隆尼亞的工人前往當地旅遊和生活。
* 1960年馬略卡島接收了50萬的遊客，到了1997年遊客達670萬人次。

== 經濟 ==
[[旅遊業|旅遊業]]幾乎支撐著馬略卡全島的經濟，亦是最主要的收益來源，佔去了近八成的[[GDP|GDP]]。另有[[採礦業|採礦業]]和[[漁業|漁業]]，產[[陶器|陶器]]、[[白蘭地|白蘭地]]、[[珠寶|珠寶]]、[[牲畜|牲畜]]和[[木材|木材]]等，貨幣單位為[[歐元|歐元]]。2005年島上有大大小小的餐廳2,400多家。
