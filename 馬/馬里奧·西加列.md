'''馬里奧·阿諾·西加列'''（{{lang-en|'''Mario Arnold Segale'''}}，{{bd|1934年|4月30日|2018年|10月27日}}）是一名[[美國|美國]]商人和房地產開發商。自1950年代以來，他一直參與[[西雅圖|西雅圖]]地區的各種開發項目<ref>{{Cite web |url = http://old.seattletimes.com/html/businesstechnology/2011460764_segale28.html |title = Powerful Segale family has massive vision for Tukwila expanse |author = Eric Pryne |date = 2010-03-27 |publisher = The Seattle Times |language = en}}</ref>。[[任天堂|任天堂]]的[[馬里奧_(角色)|馬里奧]]角色是以他的名字來命名，而他的公司當時正在出租一個倉庫給任天堂<ref name="Mario Myths with Mr Miyamoto">{{cite web |title= Mario Myths with Mr Miyamoto |url= https://www.youtube.com/watch?v=uu2DnTd3dEo |website= YouTube |publisher= Nintendo UK |language = en}}</ref>。

== 逝世 ==
西加列於2018年10月27日在華盛頓州[[塔克維拉_(華盛頓州)|塔克維拉]]的家中去世，享年84歲，死因未明確。<ref>{{Cite web |url = https://kotaku.com/marios-real-life-namesake-mario-segale-dies-at-84-1830154543 |title = Mario's Real-Life Namesake, Mario Segale, Dies At 84 |author = Cecilia D'Anastasio |publisher = Kotaku |language = en}}</ref>他留下妻子Donna Segale、四個孩子（Lisa Atkins、Mark Segale、Tina Covey和Nita Johnson），以及九個外孫。<ref>{{Cite web |url = https://www.legacy.com/obituaries/seattletimes/obituary.aspx?page=lifestory&pid=190623316 |title = Mario A. Segale |publisher = Legacy.com |language = en}}</ref>
