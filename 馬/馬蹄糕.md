'''馬蹄糕'''是以'''馬蹄粉'''（即'''[[荸薺|荸薺]]'''磨成的粉）製成的[[糕點|糕點]]。其他材料還包括[[馬蹄粒|馬蹄粒]]、[[蔗糖|蔗糖]]、[[豬油|豬油]]和[[水份|水份]]。馬蹄糕一般都是晶瑩剔透的金黃色糕塊，夾雜著白色的馬蹄粒。

== 基本製法 ==
通常先將馬蹄粉用水開勻，製成馬蹄粉水。另把蔗糖用熱水煮至溶化，加入豬油和馬蹄粒。再將馬蹄水撞落煮滾的蔗糖水之中，快動作攪拌成糊狀。將糕糊倒入已經抹了食油的糕盆內，隔水蒸半小時即成。

== 食法 ==
馬蹄糕宜凍食，或將雪凍的馬蹄糕下煎鑊煎至金黃熱食。

== 賀年糕點 ==
馬蹄糕與[[蘿蔔糕|蘿蔔糕]]、[[芋頭糕|芋頭糕]]都是中國南方[[春節|春節]]期間不可缺少的賀年糕點。馬蹄糕有著「馬到功成、步步高陞」的喻意。
