'''馬場典子'''（{{jpn|j=馬場 典子|kn=ばば のりこ|rm=Baba Noriko|f=h}}，{{bd|1974年|4月27日|||Baba Noriko}}），日本自由[[播报员|播报员]]，[[大阪藝術大學|大阪藝術大學]]教授和[[Amuse|Amuse]]旗下[[艺人|艺人]]。曾在1997年至2014年供职于[[日本電視台|日本電視台]]。
==经历==
[[东京都|东京都]][[豐島區|豐島區]]出身<ref name=music>{{cite web |title=フリーアナ・馬場典子が“雅楽”同級生の現在に驚愕！ |url=http://music-book.jp/video/news/news/190516 |publisher=music.jpニュース |accessdate=2019-02-11 |archiveurl=https://web.archive.org/web/20190211141411/http://music-book.jp/video/news/news/190516 |archivedate=2019-02-11 |date=2018-06-20}}</ref>。中学就读于{{tsl|ja|女子学院中学校・高等学校|女子学院中学校・高等学校|女子学院}}，本科毕业于[[早稻田大学|早稻田大学]]商学院[[市场营销|市场营销]]专业<ref name=music/>，1997年入职[[日本電視台|日本電視台]]。高中时代与[[膳場貴子|膳場貴子]]（主播）和{{tsl|ja|辛酸なめ子||辛酸なめ子}}（漫画家）是同年级同学<ref>{{cite web |title=女子学院同窓会で年齢退行…辛酸なめ子 |url=https://www.yomiuri.co.jp/kyoiku/support/ranger/20180803-OYT8T50013/ |publisher=[[讀賣新聞|読売新聞]] |accessdate=2019-02-11 |date=2018-08-03}}</ref>。曾参与2008年[[東京馬拉松|東京馬拉松]]，并以5小时42分57秒的成绩跑完全程<ref>{{cite web |title=東京マラソン2008 著名・芸能人の完走タイム一覧 |url=http://honolulu.bhlife.net/tokyo2008/post_124.html |publisher=ホノルルマラソンへの道2017 |accessdate=2019-02-11 |date=2009-03-24|archiveurl=https://web.archive.org/web/20190212041217/http://honolulu.bhlife.net/tokyo2008/post_124.html|archivedate=2019-02-12}}</ref>。

2014年3月28日，她不再主持[[晨間新聞#日本|晨間新聞]]节目『{{tsl|ja|ZIP!||ZIP!}}』<ref>{{cite web|url=http://www.cyzo.com/2014/03/post_16588.html|title=日テレ馬場アナ降板はなぜ？|publisher=日刊サイゾー|date=2014-03-25}}</ref>。4月11日，她发表将在6月底从日本电视台辞退，并在7月转职为自由播报员的消息<ref>{{cite web|url=http://www.sponichi.co.jp/entertainment/news/2014/04/11/kiji/K20140411007953700.html|title=日テレ・馬場典子アナ 6月末で退社「新たな挑戦したい」] |publisher=Sponichi Annex|accessdate=2014-04-11}}</ref><ref>{{cite web|url=http://www.daily.co.jp/newsflash/gossip/2014/04/11/1p_0006855071.shtml|title=馬場典子アナ 6月末で日テレ退社|publisher=デイリースポーツ|accessdate=2014-04-11}}</ref><ref>{{cite web|url=http://www.cyzo.com/2014/04/post_16843.html|title=馬場典子アナ フリー転向の勝算|publisher=日刊サイゾー |accessdate=2014-04-16}}</ref>。之后与[[Amuse|Amuse]]娱乐公司签约，并成为旗下艺人<ref>{{cite web|url=http://www.sponichi.co.jp/entertainment/news/2014/06/23/kiji/K20140623008422940.html|title=6月退社の日テレ馬場アナ アミューズと契約「世界広がる」|publisher=[[體育日本|スポーツニッポン]]|accessdate=2014-06-23}}</ref>。2014年9月5日出演[[富士電視台|富士電視台]]综艺节目『{{tsl|ja|ライオンのごきげんよう|狮子的祝您健康}}』<ref>{{cite web |author=Techinsight |title=桝太一アナの真面目すぎる一面を馬場典子アナが暴露 飲み会後に「反省会」 |url=http://news.livedoor.com/article/detail/9228272/ |publisher=[[活力门|livedoor]] |accessdate=2019-02-12 |date=2014-09-07}}</ref>，这是她从日本电视台辞退以来首次出演其他电视台的节目。

2015年4月就任[[大阪藝術大學|大阪藝術大學]][[广播|广播]]专业[[教授|教授]]<ref name=music/>。
