'''馬納瓜'''（{{lang-es|'''Managua'''}}）是[[尼加拉瓜|尼加拉瓜]]的首都及第一大城市，也是[[中美洲|中美洲]]地區第二大城市。位於尼加拉瓜的西部以及尼加拉瓜第二大湖－[[馬那瓜湖|馬那瓜湖]]的沿岸。該城在1855年被訂為尼加拉瓜的首都。1972年該地曾遭受大地震的侵襲，造成超過一萬人的死亡。

馬納瓜的人口有1,617,096人(2004年) 其中大多數為說[[西班牙語|西班牙語]]的[[麥士蒂索人|麥士蒂索人]]，除此之外尚有[[敘利亞|敘利亞]]人、[[巴勒斯坦|巴勒斯坦]]人、[[亞美尼亞|亞美尼亞]]人、[[日本|日本]]人、[[華人|華人]]的居住社群。

在文化上，馬納瓜是著名的[[尼加拉瓜手語|尼加拉瓜手語]]的誕生地。

== 历史 ==
== 地理 ==
马那瓜位于[[马那瓜湖|马那瓜湖]]南岸，也被称为马那瓜湖。湖中包含与东南部湖泊相同的鱼种，除了只在后者中发现的淡水鲨鱼。湖水已自1927年以来的新排水系统和废水到拉斯梅塞德斯一个新的污水处理厂是由德国政府资助，以净化的污染。

== 气候 ==
马那瓜，像很多[[尼加拉瓜|尼加拉瓜]]西部的城市，除了南方的山脉属[[热带气候|热带气候]]，平均气温28至32℃（82至90°F）。在[[柯本气候分类法|柯本气候分类]]中，全市拥有热带干湿气候。十一月至四月之间存在着明显的旱季，而大部分的降雨是五月至十月间。温度最高是三、四月份，当太阳位于头顶但夏季降雨尚未开始。
