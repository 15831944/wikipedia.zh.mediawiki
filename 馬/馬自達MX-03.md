'''馬自達MX-03'''（{{lang-ja|'''マツダ・MX-03'''}}）是一款由[[日本|日本]][[馬自達|馬自達]]公司開發的[[概念車|概念車]]。

== 概要 ==
該款車於1985年第26屆[[東京車展|東京車展]]展出，是一輛雙門GT大型旅行概念車，也是[[馬自達Cosmo#第四代（JC系_1990年-1996年）|第四代馬自達Cosmo]]的雛型。該車配置654c.c. X 3 [[馬自達轉子引擎#13B-DEI型|13B型]]轉子引擎，最大[[馬力|馬力]]達320hp / 7,000rpm，一舉超越日本車廠的280hp馬力自主上限。馬自達宣稱該款概念車從0加速至60[[英哩|英哩]]僅需5秒，極速可達300km/hr。另外該款車也搭載了全時[[四輪驅動|四輪驅動]]和SS-VGR速度感應式四輪轉向系統，可惜後者未曾運用在日後的量產車型上。內裝設計方面，採用類似操縱[[飛機|飛機]]的[[方向盤|方向盤]]和排檔桿、恆溫空調、音響、數位式顯影儀表板等，具有未來科技感。

== 車輛規格 ==
* 全長：4,510mm
* 全寬：1,800mm 
* 全高：1,200mm
* [[軸距|軸距]]：2,710mm 
* 座位：2+2人 
* 引擎：654c.c. X 3 [[馬自達轉子引擎#13B-DEI型|13B型]]轉子引擎
* 最大[[馬力|馬力]]：320hp / 7,000rpm
* 極速：300km/hr
* 變速系統：四速自動排檔
