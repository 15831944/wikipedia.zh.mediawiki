'''阿尔马里奥·沃纳德·“马里奥”·查尔莫斯'''（{{lang-en|'''Almario Vernard "Mario" Chalmers''' }}，{{bd|1986年|5月19日}}），美国职业篮球运动员，曾随[[迈阿密热火|迈阿密热火]]获得2012年、2013年NBA总冠军。<ref>{{Cite web |url =http://bleacherreport.com/articles/1228748-nba-finals-2012-miami-heats-mario-chalmers-takes-oklahoma-city-thunder-down|title = NBA Finals 2012: Miami Heat's Mario Chalmers Comes Up Clutch in Game 4 |date = 2012-06-20||publisher = Bleacher Report }}</ref>

== 邁阿密熱火 ==
查爾莫斯於菜鳥球季便成為球隊先發，並全勤出賽，繳出場均10.0分、4.9 助攻、2.8籃板的成績。2010-11年賽季，球星[[雷霸龍·詹姆斯|雷霸龍·詹姆斯]]加盟熱火，查爾莫斯讓出原先球衣背號6號給詹姆斯，自己改穿15號球衣。

2011-12賽季，查爾莫斯與熱火隊續約。在2012總冠軍賽中，查爾莫斯於第四戰表現出色，整場比賽攻下25分，其中12分為第四節所得。熱火隊最終擊敗對手[[俄克拉何马城雷霆|俄克拉何马城雷霆]]，獲得總冠軍，並於隔年達成衛冕。2014年總冠軍賽，查爾莫斯表現有所下滑，先發位置於第五戰被[[雷·艾倫|雷·艾倫]]取代，熱火隊最終敗給[[馬刺隊|馬刺隊]]，無緣三連霸。2014-15賽季，由於[[戈蘭·達拉積|戈蘭·卓吉奇]]的加盟，查爾莫斯在球隊的地位不如以往，於2015年9月10日被交易至灰熊隊。

== 曼斐斯灰熊 ==
這樁交易熱火送出查尔莫斯與前鋒詹姆斯·恩尼斯，換來灰熊後衛[[貝諾·烏德里|貝諾·烏德里]]與前鋒Jarnell Stokes。<ref>{{Cite web |url =http://www.nba.com/grizzlies/news/chalmers-and-ennis-151110|title =Grizzlies acquire Mario Chalmers and James Ennis from Heat |date = 2015-09-10||work = NBA.com }}</ref>2015年3月10日，查尔莫斯因右腿[[跟腱|跟腱]]傷勢而賽季報銷，隨後被灰熊隊裁掉。<ref>{{Cite web |url =http://www.nba.com/grizzlies/news/chalmers-injury-update-160311|title =Update on Mario Chalmers |date = 2016-03-10||work = NBA.com }}</ref>

2017年7月19日，據體育記者Adrian Wojnarowski報導，灰熊隊將與Chalmers簽下一年部分保障的合約。
