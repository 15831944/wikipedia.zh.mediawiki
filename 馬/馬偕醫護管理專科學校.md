'''馬偕學校財團法人馬偕醫護管理專科學校（Mackay Junior College of Medicine, Nursing and Management）'''，簡稱'''馬偕醫專'''、'''馬偕護專'''，是一間位於[[臺灣|臺灣]][[臺北市|臺北市]][[北投區|北投區]]及[[新北市|新北市]][[三芝區|三芝區]]的[[專科學校|專科學校]]，目前有6個學科。該校董事會有意與[[馬偕醫學院|馬偕醫學院]]共同改制或部分合併改制'''馬偕大學''' <ref>[http://www.mmc.edu.tw/ugC_AboutUs.asp 馬偕醫學院官網校區介紹提到「將來兩校合併成立馬偕大學」]</ref>。

2017年7月下旬傳出與[[馬偕醫學院|馬偕醫學院]]、[[真理大學|真理大學]]整併消息<ref>{{Cite web |url = https://pttweb.tw/thread/m-1354662811-a-459 |title = 紀念創辦人，真理大學改名馬偕大學 |publisher = 中國時報  |date = 2012-12}}</ref>，2020年前併入<ref>{{Cite web |url = https://money.udn.com/money/story/5641/2598328 |title = 真理馬偕3校 擬3年內合併 |publisher = 聯合財經網 |date = 2017-07-22 |access-date = 2017-11-21 |archive-url = https://web.archive.org/web/20171201042215/https://money.udn.com/money/story/5641/2598328 |archive-date = 2017-12-01 |dead-url = yes }}</ref>。

== 沿革 ==
*1913年，設立馬偕紀念醫院看護婦訓練班。
*1948年，[[國民政府接收台灣|國民政府接收台灣]]後，更名為馬偕紀念醫院護士訓練班。
*1970年，'''臺北市私立馬偕高級護理職業學校'''立案創校。
*1999年，改制為'''馬偕護理專科學校'''。
*2004年，因增加科系更名為'''馬偕醫護管理專科學校'''。
*2017年，改制為'''馬偕財團法人馬偕醫護管理專科學校'''。
