'''马绍尔群岛共和国'''（{{langWithName|mh|马绍尔语|Aolepān Aorōkin M̧ajeļ}}） 通稱'''马绍尔群岛'''（{{lang-en|Marshall Islands}}），是位於北[[太平洋|太平洋]]的[[島嶼國家|島嶼國家]]。马绍尔群岛是[[密克罗尼西亚群岛|密克罗尼西亚群岛]]的一部分，土地面積为181平方公里，人口为68,480人，分散定居在由1,156个岛屿组成的34个珊瑚环礁上。与之在海上相邻的地区分别是北部的[[威克島|威克島]]，南部的[[諾魯|諾魯]]，西部的[[密克罗尼西亚联邦|密克罗尼西亚联邦]]和东南部的[[基里巴斯|基里巴斯]]。人口最多的环礁为[[馬久羅|馬久羅]]，亦为马绍尔群岛的首都。

[[前2千年|公元前两千年]]左右，密克罗尼西亚人陆续到马绍尔群岛定居，他们借助由木棒编织的浮网在岛屿间航行。[[欧洲人|欧洲人]]在16世纪20年代发现马绍尔群岛，[[西班牙|西班牙]]探险家阿隆索·萨拉萨尔于1526年8月在海上看到这一环礁。随后其他西班牙和[[英国|英国]]探险家也陆续发现了这一岛屿。马绍尔群岛的名称源于英国探险家约翰·马绍尔。1874年，马绍尔群岛成为了[[西屬東印度群島|西屬東印度群島]]的一部分。1884年这些岛屿被出售给德国，并在1885年成为[[德屬新幾內亞|德屬新幾內亞]]的一部分。[[日本|日本]]在[[第一次世界大战|第一次世界大战]]中占领了马绍尔群岛，并于1919年由[[國際聯盟|國際聯盟]]将其正式移交日本，马绍尔群岛因此受日本[[南洋廳|南洋廳]]管辖。[[第二次世界大战|第二次世界大战]]中，[[美国|美国]]在[[吉爾伯特及馬紹爾群島戰事|吉爾伯特及馬紹爾群島戰事]]中占领了该群岛。战后，马绍尔群岛与其他的太平洋群岛一同受美国治理的[[太平洋群岛托管地|太平洋群岛托管地]]管辖。1979年马绍尔群岛实现自治，并于1986年和美国签署《自由联合协定》而获得独立主权。

马绍尔群岛是一个[[总统制|总统制]][[共和制|共和国]]，并与美国达成[[聯繫邦|自由联合]]关系。美国提供马绍尔群岛的国防、资金和社会服务。由于岛上天然资源并不丰富，马绍尔群岛的经济依赖于[[第三产业|服务业]]以及部分渔业和农业。其[[国内生产总值|国内生产总值]]的很大一部分来自美国的援助。马绍尔群岛采用[[美元|美元]]作为货币。马绍尔群岛的大多数公民具有马绍尔血统，小部分移民来自[[菲律宾|菲律宾]]及其他[[太平洋|太平洋]]岛屿。马绍尔群岛的两种官方语言是[[马绍尔语|马绍尔语]]和[[英语|英语]]。几乎全体国民信仰宗教，约四分之三的国民加入[[联合基督教会|联合基督教会]] - 马绍尔群岛[[公理会|公理会]]（UCCCMI）或[[神召会|神召会]]。

== 歷史 ==
{{main|马绍尔群岛历史}}
[[File:Kwajalein-closing_in.jpeg|thumb]]在[[瓜加林環礁|瓜加林環礁]]檢查一個敵人碉堡]]
[[File:Castle_Bravo_Blast.jpg|thumb]]在馬紹爾群島上進行核子試爆]]
1885年，[[德意志帝國|德國]]佔領馬紹爾群島，屬[[德屬新幾內亞|德屬新幾內亞]]保護，[[第一次世界大战|第一次世界大战]]時被[[日本|日本]]佔領，後由日本託管。

1944年，[[美国|美国]]攻佔馬紹爾群島，戰後[[联合国|联合国]]委由[[美國|美國]]進行託管，成為[[太平洋群島託管地|太平洋群島託管地]]其中一區。

1946年至1968年間，美國建立[[太平洋试验场|太平洋试验场]]，曾於馬紹爾群島上進行多達66次的[[核试验|核子試爆]]。

1979年，馬紹爾群島否決《[[密克罗尼西亚联邦|密克罗尼西亚联邦]]憲法》公投，自行成立自治政府和準備建國。

1986年，與[[美國|美國]]簽訂《[[聯繫邦|自由聯合]]條約》，同年10月21日宣布獨立。

1991年，[[联合国|联合国]]終止[[美國|美國]]託管，接納馬紹爾群島共和國為[[聯合國會員國|聯合國會員國]]。

== 地理 ==
[[File:Marshall_islands-map.png|thumb]]
馬紹爾群島共計有29座[[環礁|環礁]]及5座偏遠島嶼，地理上分為東面的[[拉塔克礁鏈|拉塔克（日出）島鏈]]和西面的[[拉利克礁鏈|拉利克（日落）島鏈]]。海拔平均高度2公尺。<ref>{{cite web|url=http://www.rmiembassyus.org/Geography.htm|title=Geography|accessdate=2015年9月16日|language=en|deadurl=yes|archiveurl=https://web.archive.org/web/20131115173707/http://rmiembassyus.org/Geography.htm|archivedate=2013年11月15日}}</ref>全國三分二人口住在馬久羅和伊拜（Ebeye）兩環礁的島上。

== 行政區劃 ==
{{main|马绍尔群岛行政区划}}
馬紹爾群島未正式劃分行政區，但可分為26個立法選區（legislative districts）。

== 經濟 ==
[[File:Marshall_Islands_treemap.png|thumb]]
國內約有6萬人，以'''[[渔业|漁業]]'''和'''[[农业|农业]]'''為主，海域內富有鈷殼和[[錳結核|錳結核]]等資源，魚產豐富。出口貨品主要為[[冷凍魚|冷凍魚]]、[[椰子產品|椰子產品及加工品]]和土產的[[手工艺|手工藝品]]等。

*主要進口項目：食品、機械設備、燃料、一般民生用品、菸草
*主要進口來源：美國、日本、澳大利亞、紐西蘭、新加坡、斐濟、中国、菲律賓 
*主要出口項目：椰子油、椰乾餅及手工藝品、魚 
*主要出口市場：美國、日本、澳大利亞、中國
*主要港口及自由貿易區：Majuro Island、Majuro Atoll

== 交通 ==
位於首都馬久羅的[[馬紹爾群島國際機場|馬紹爾群島國際機場]]是主要機場。
== 军事 ==
马绍尔群岛的军事地理位置十分重要。第二次世界大战期间，美国与日本曾在此进行过激烈的战斗。1946-1962年，美国在群岛西北部的比基尼和埃克威托克两面三岛上进行过数十次核试验。夸贾林上设有美国的导弹军事基地。马绍尔岛上有卫星通讯设备。首都有可供波音707、727和737飞机起降的机场。马绍尔群岛航空公司有定期飞往国内十个岛屿和邻近的南太平洋岛国。国内各主要岛屿之间均有客轮定期往来。首都马朱罗有深水码头。
== 外交 ==
{{main|马绍尔群岛共和国建交列表}}
馬紹爾群島迄今與全球90個國家保持正式外交關係，[[美国|美国]]、[[日本|日本]]、[[中華民國|中華民國]]在馬紹爾群島設有大使館，[[菲律宾|菲律宾]]、[[以色列|以色列]]、[[土耳其|土耳其]]在馬紹爾群島設有榮譽領事，馬紹爾群島亦於[[美國|美國]]、[[日本|日本]]、[[斐濟|斐濟]]、[[中華民國|中華民國]]設置大使館。

海外的馬紹爾群島公民如在沒有大使館的地區需要急難救助，可以向當地的[[美國駐外機構列表|美國大使館]]提出協助，美國大使館應提供馬紹爾公民與美國公民相同的對待。

===與{{ROC}}關係===
{{see also|中華民國－馬紹爾群島關係}}
[[中华人民共和国|中华人民共和国]]曾在1990年11月16日與馬紹爾群島建交，1998年11月20日馬紹爾群島轉而承認[[中華民國|中華民國]]，次月與中华人民共和国斷交，馬紹爾是中華民國現今17個邦交國之一。

=== 與{{USA}}關係 ===
馬紹爾群島被[[美國|美國]]託管近40年，至今仍與美國關係密切。

馬紹爾群島使用美元，各種現代商品均相當依賴美國進口。根據《自由聯繫條約》規定，馬紹爾群島軍事防衛任務交由美國執行，國民也可參加[[美國軍隊|美國軍隊]]。另外，馬紹爾群島國民也可自由進出美國和在美居住、就業和就學，無需簽證（但犯事者可被美國驅逐）。馬紹爾群島的對外郵政由[[美國郵政|美國郵政]]負責，郵資也根據美國標準，寄往[[帕劳|帕劳]]、[[密克羅尼西亞聯邦|密克羅尼西亞聯邦]]、美國各州、屬地和軍事外交駐地的郵件均按國內郵資計算。

《自由聯繫條約》也規範美國對馬紹爾群島政府的各項財政支援，以及[[美國陸軍|美國陸軍]]租借夸贾林環礁的條件等。因為各項的支援和撥款項目，美國對馬紹爾群島的事務主要由內政部的島嶼事務辦公室（而非國務院）負責。
