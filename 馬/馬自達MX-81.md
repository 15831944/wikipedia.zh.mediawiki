'''馬自達MX-81'''（{{lang-ja|'''マツダ・MX-81'''}}；'''Mazda MX-81'''）是一款由[[日本|日本]][[馬自達|馬自達]]汽車公司開發的[[概念車|概念車]]。

== 概要 ==
歷經[[第一次石油危机|1973年石油危機]]及[[第二次石油危机|1979年石油危機]]兩次能源危機後，日本製造、比較省油的[[紧凑型轿车|小型車]]在全球無往不利。比如1980年全日本車廠總共出產703萬8千輛[[汽車|汽車]]，勝過美國車廠的637萬6千輛。在這種成功背景的催化下，日本車廠陸續推出天馬行空的夢幻概念車。

1981年第24屆[[東京車展|東京車展]]上，馬自達發表了名為'''MX-81'''的概念車。這部雙門四人座的夢幻概念車由[[義大利|義大利]]{{link-en|博通汽車|Gruppo Bertone}}公司（Gruppo Bertone）首席設計師馬克·德尚（Marc Deschamps）操刀，以[[馬自達323#第四代（1977年-1980年）|第四代323]]之[[底盤|底盤]]為基礎，大面積[[玻璃|玻璃]]雙門、上掀啟閉式頭燈組（retractable headlight）等新潮的外型吸引不少年輕族群的目光。縱然當時馬自達並未公布其規格數據，但是MX-81所要展現的是科技概念：
# 可旋轉式前乘客座：方便前座乘客上下車，且防止穿著短裙的女性走光。此外，這種座椅設計正是目前福祉車輛（[[身心障礙|殘障人士]]使用的交通工具）的標準配備。
# [[阴极射线管|映像管]]螢幕：可顯示各種行車資訊，並以立體圖像顯現。
# 方向盤：以類似皮帶所組成。

1989年問世的[[馬自達323_Astina|323 Astina]]五門[[快背車|快背式]][[轎車|轎車]]，運用了不少MX-81的概念。也由於其時髦流線的車身外觀，在[[歐洲|歐洲]]市場獲得了好評。
