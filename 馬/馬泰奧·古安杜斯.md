'''馬泰奧·古安杜斯'''（{{lang-fr|Mattéo Guendouzi Olié}}，{{bd|1999年|04月14日}}），是一名[[法國|法國]]職業足球員，現時效力[[英格蘭超級足球聯賽|英格蘭超級足球聯賽]]球會[[阿仙奴足球會|阿仙奴]]和{{link-en|法國20歲以下國家足球隊|France national under-20 football team|法國 U20}}。司職[[中場|中場]]。

== 球會生涯 ==
=== 早期生涯 ===
古安杜斯曾效力[[巴黎圣日耳曼足球俱乐部|巴黎圣日耳曼]]的青年軍，於2014年，他轉投[[洛里昂足球俱樂部|洛里昂]]的青年軍，其後於2016年由[[洛里昂足球俱樂部|洛里昂 B隊]]被提升至一線隊。

=== 羅連安特 ===
2016年10月15日，古安杜斯於主負[[南特足球會|南特]]1比2的聯賽中，首次為羅連安特一線隊上陣。於2016-17賽季，他於各項賽事上陣9次<ref>{{cite web |url= http://www.soccerbase.com/players/player.sd?player_id=97495&season_id=149 |title=Matteo Guendouzi in 2016/2017 |publisher=Soccerbase |date= 2018-07-09 |accessdate= 2018-07-09|language= en}}</ref>。可是，羅連安特於聯賽以第18名完成賽季，於次季需降班至[[法國乙組足球聯賽|法國乙組足球聯賽]]<ref>{{cite web |url= http://www.beinsports.com/au/football/video/troyes-promoted-to-ligue-1/554193|language= en|title= Troyes promoted to Ligue 1|date= 2017-05-29|accessdate= 2017-05-29|work= Bein Sports}}</ref><ref>{{cite web |url=http://www.ligue1.com/club/fc-lorient|language= en|title= FC LORIENT|accessdate= 2018-07-09}}</ref>。

於2017-18賽季，古安杜斯於各項賽事上陣20次，球隊於聯賽亦以第7名完成，失去升班資格<ref name="Skysports">{{cite web |url= http://www.skysports.com/football/news/11670/11430116/arsenal-in-talks-with-france-u20-midfielder-matteo-guendouzi |title=Arsenal in talks with France U20 midfielder Matteo Guendouzi|publisher=Sky Sports |date= 2018-07-09 |accessdate= 2018-07-09|language= en}}</ref>。

=== 阿仙奴 ===
2018年7月11日，古安杜斯轉會至[[英格蘭超級足球聯賽|英格蘭超級足球聯賽]]球會[[阿仙奴足球會|阿仙奴]]，轉會費為700萬鎊，將穿上29號球衣<ref>{{cite web|url= https://www.arsenal.com/news/matteo-guendouzi-joins-club|title= Matteo Guendouzi joins the club|work= Arsenal.com|date= 2018-07-11|accessdate= 2018-07-11|language= en}}</ref>。8月12日，他於對陣[[曼彻斯特城足球俱乐部|曼城]]的聯賽中，首次於英超上陣，但球隊於主場以0比2落敗；古安杜斯全場觸球72次，為全隊之首<ref>{{Cite news|url=https://www.bbc.com/sport/football/45138012|title=Arsenal 0-2 Man City: Unai Emery's opener is a reality check for Arsenal|date=2018-08-12|work=BBC Sport|access-date=2018-08-13|language=en}}</ref>。

10月4日，古安杜斯於[[欧足联欧洲联赛|欧足联欧洲联赛]]客勝[[卡拉巴克足球會|卡拉巴克]]3比0的比賽中，接應[[阿歷山大·拿卡錫迪|-{zh-cn:亚历山大·拉卡泽特;zh-hk:阿歷山大·拿卡錫迪;zh-tw:亞歷山大·拉卡傑特;}-]]的傳球，攻入職業生涯首個入球<ref>{{cite web |url= https://www.bbc.co.uk/sport/football/45684940|title= Qarabag 0 - 3 Arsenal|work= BBC|date= 2018-10-05 |accessdate= 2018-10-05|language= en}}</ref>。

== 國家隊生涯 ==
古安杜斯於法國出生，曾代表{{link-en|法國18歲以下國家足球隊|France national under-18 football team|法國 U18}}、[[法國19歲以下國家足球隊|法國 U19]]和{{link-en|法國20歲以下國家足球隊|France national under-20 football team|法國 U20}}上陣<ref name="Skysports"/>。他擁有[[摩洛哥|摩洛哥]]血統，[[摩洛哥國家足球隊|摩洛哥國家足球隊]]領隊{{link-en|埃爾韋·勒納爾|Hervé Renard}}曾於2017年接觸古安杜斯，希望他改為摩洛哥隊上陣。2017年3月，古安杜斯的父親與勒納爾見面後，他決定繼續為法國隊上陣<ref>{{cite web |url=https://fr.starafrica.com/football/articles/maroc-renard-tourne-page-de-guendouzi/|title=Morocco: Renard turns the page of Guendouzi|publisher=Orange|date= 2018-03-01 |accessdate= 2018-07-09|language= en}}</ref>。
