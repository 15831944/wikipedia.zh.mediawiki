'''馬努斯·埃克雷姆'''（{{lang-no|Magnus Wolff Eikrem}}；{{bd|1990年|8月8日}}）是一位[[挪威|挪威]]足球運動員。在場上的位置是中場。他現在效力於[[瑞典足球超級聯賽|瑞典足球超級聯賽]]球隊[[馬爾默足球俱樂部|馬爾默足球俱樂部]]<ref>{{cite web |url=http://www.mff.se/Aktuellt/Nyheter/2015-01-26_Valkommen_till_MFF_Magnus_Wolff_Eikrem |title=Välkommen till MFF, Magnus Wolff Eikrem |publisher=Malmö FF |work=mff.se |date=26 January 2015 |accessdate=26 January 2015 |language=sv |deadurl=yes |archiveurl=https://web.archive.org/web/20150128114347/http://www.mff.se/Aktuellt/Nyheter/2015-01-26_Valkommen_till_MFF_Magnus_Wolff_Eikrem |archivedate=28 一月 2015 }}</ref><ref>{{cite web |url=http://svenskfotboll.se/cuper-och-serier/svenska-cupen-herrar/grupp-1/?scr=result&fmid=2906520 |title=Malmö FF 3–0 Assyriska FF |publisher=The Swedish Football Association |work=svenskfotboll.se |date=22 February 2015 |accessdate=23 February 2015 |language=sv }}</ref>。他也代表挪威國家足球隊參賽<ref>{{cite web |url=http://www.vg.no/sport/fotball/norsk/landslaget/artikkel.php?artid=10075559 |language=no |title=Elyounoussi reddet Norge: - Dette lover godt |work=vg.no |publisher=[[Verdens_Gang|VG]] |date=15 January 2012 |accessdate=15 January 2012 }}</ref>。
