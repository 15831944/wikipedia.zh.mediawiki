'''馬來穿山甲'''（[[學名|學名]]：''{{lang|la|Manis javanica}}''），又名'''爪哇穿山甲'''或'''爪哇鯪鯉'''，是[[東南亞|東南亞]]的一種[[穿山甲|穿山甲]]，分佈在[[泰國|泰國]]、[[印尼|印尼]]、[[菲律賓|菲律賓]]、[[越南|越南]]、[[老撾|老撾]]、[[柬埔寨|柬埔寨]]、[[馬來西亞|馬來西亞]]及[[新加坡|新加坡]]的[[森林|森林]]。牠們大部份時間都是在樹上休息或覓食。<ref name=Nowak1999>{{Cite book en|author=Nowark, R.|year=1999|title=Walker's Mammals of the World|edition=Sixth Edition|location=Baltimore|publisher=The Johns Hopkins University Press}}</ref>

== 特徵 ==
馬來穿山甲的爪厚及強壯，可以挖開[[螞蟻|螞蟻]]的巢及撕開[[白蟻|白蟻]]的土堆。牠們沒有[[牙齒|牙齒]]，[[舌頭|舌頭]]很長及帶有黏性，用來捕食螞蟻及白蟻。牠們的身體由一列列的鱗片及纖毛覆蓋。腳部的[[皮膚|皮膚]]呈粿粒狀的，前肢上有墊。牠們的尾巴上有30個鱗片。牠們體長約65厘米，尾巴長56厘米，重達10公斤。雄性較雌性大隻。牠們的近親是[[中華穿山甲|中華穿山甲]]，但體型較大，顏色較淺，前爪較短。

== 行為 ==
馬來穿山甲每年會產1-2隻幼穿山甲。牠們會在[[秋天|秋天]][[交配|交配]]，到了[[冬天|冬天]]就產子。出生後的幼穿山甲約需3個月的照顧。

馬來穿山甲很多時都是獨居的，晚間活動及很膽小。牠們的天敵有[[虎|虎]]及[[雲豹|雲豹]]。當感到威脅時，牠們會捲曲成球狀來保護[[腹部|腹部]]柔軟的部位。牠們會在近[[白蟻|白蟻]]的土堆及[[螞蟻|螞蟻]]的巢穴築巢，並以[[植物|植物]]覆蓋來保溫<ref name=Medway1969>{{Cite book en|author=Medway, L|year=1969|title=The Wild Mammals of Malaya|location=London|publisher=Oxford University Press}}</ref>，其巢穴可以幫助[[土壤|土壤]]通風。<ref name=Heath1992>{{Cite journal en|author=Heath, M|year=1992|title=''Manis pentadactyla''|journal=Mammalian Species|volume=414|pages=1-6}}</ref>

== 與人類關係 ==
馬來穿山甲的[[皮膚|皮膚]]、皮革及肉都是[[人類|人類]]獵殺的目的。一些部位更有醫藥的療效。<ref name=Nowak1999/>當地人用牠們的皮革來製成驅散[[風濕熱|風濕熱]]的符咒，皮膚則用來製鞋。[[尼泊爾|尼泊爾]]人認為牠們[[子宮|子宮]]的抽取物可以安胎。人類會挖開牠們的巢穴來捕獵牠們，或是以[[辣椒|辣椒]]的氣味來逼出牠們。

== 保育 ==
馬來穿山甲列在[[瀕危野生動植物種國際貿易公約|瀕危野生動植物種國際貿易公約]]的附錄II中，表示任何從野外捕獵的商業用途均被禁止。在多個不同國家及地區均禁止出口及貿易。<ref name="iucn"/>
