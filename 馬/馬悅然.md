'''馬悅然'''（[[瑞典语|瑞典语]]：{{lang|sv|'''Nils Göran David Malmqvist'''}}，{{bd|1924年|6月6日|catIdx=M马}}），[[瑞典|瑞典]]籍[[汉学家|汉学家]]。[[瑞典學院|瑞典學院]]院士、[[諾貝爾文學獎|諾貝爾文學獎]]評審委員。<ref>[http://www.apdnews.com/news/73132.html 马悦然谈诺贝尔奖]，亚太日报，2014年3月4日</ref>

==生平==
[[1924年|1924年]]，马悦然出生于[[瑞典|瑞典]]南部[[延雪平|延雪平]]。

[[1946年|1946年]]，馬悅然考入[[斯德哥尔摩大学|斯德哥尔摩大学]]，做了瑞典大学者[[高本汉|高本汉]]的学生，[[1946年|1946年]]跟着老师高本汉学习[[古汉语|古汉语]]和中国音韵学。

[[1948年|1948年]]来到中国[[四川|四川]]做[[方言|方言]]调查，[[1949年|1949年]]到中国西南，过[[塔尔寺|塔尔寺]]，拜见[[活佛|活佛]][[班禅额尔德尼|班禅额尔德尼]]。

[[1950年代|1950年代]]马悦然曾一度在瑞典驻中国大使馆任文化专员。

[[反右|反右]]时期，马悦然在中国工作，目睹了这场政治文化风暴，他说：“[[杨绛|杨绛]]女士的《干校六记》，[[钱锺书|钱锺书]]的《小引》和[[李锐_(小说家)|李锐]]的《无风之树》给我的印象非常深。这三个文本帮助我了解十年浩劫给我的第二祖国带来的不幸。”<ref>{{cite web|url=http://news.xinhuanet.com/book/2004-07/13/content_1595215.htm|title=《马悦然的《另一种乡愁》》|accessdate=2004年7月13日|work=李国涛|publisher=中华读书报|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20120318041457/http://news.xinhuanet.com/book/2004-07/13/content_1595215.htm|archivedate=2012年3月18日}}</ref>

[[1958年|1958年]]，他離開中國前往[[澳洲國立大學|澳洲國立大學]]教書，直到六年後回到瑞典。后来马悦然历任斯德哥尔摩大学东方语言学院中文系汉学教授和系主任，[[欧洲汉学协会|欧洲汉学协会]]会长。

[[1985年|1985年]]，马悦然当选为[[瑞典学院|瑞典学院]]院士。

==貢獻==
馬悅然是諾貝爾文學獎評委中唯一精通[[中文|中文]]的評委，他曾翻譯過《[[诗经|诗经]]》、《[[春秋繁露|春秋繁露]]》、《[[西遊記|西遊記]]》、《[[水滸傳|水滸傳]]》、《[[辛棄疾詞|辛棄疾詞]]》《[[道德经|道德经]]》等中国古典著作，亦翻譯了[[鲁迅|鲁迅]]、[[高行健|高行健]]、[[沈從文|沈從文]]以及[[北岛_(诗人)|北岛]]的诗歌和李锐的小说等當代中文作品，致力于提升中國文學在國際的地位。

== 观点 ==
*2004年，在被问道“中国人为什么至今没有拿到诺贝尔文学奖，难道中国文学和中国作家真落后于世界么？”时，马悦然说：“中国的好作家好作品多得是，但好的翻译太少了！”<ref name="multiple1">{{cite web |url=http://news.sina.com.cn/w/2004-12-09/12574477694s.shtml|title=专访马悦然：中国作家何时能拿诺贝尔文学奖？|accessdate=2004年12月9日|work=王洁明|publisher=参考消息特刊|language=zh-hans}}</ref>

*“如果上个世纪20年代有人能够翻译《[[彷徨|彷徨]]》、《[[呐喊|呐喊]]》，鲁迅早就得奖了。但鲁迅的作品只到30年代末才有人译成捷克文，等外文出版社推出[[杨宪益|杨宪益]]的英译本，已经是70年代了，鲁迅已不在人世。而诺贝尔奖是不颁给已去世的人的。”<ref name="multiple1"/>

*1987年和1988年，[[沈从文|沈从文]]两次被提名为诺贝尔文学奖候选人，1988年瑞典诺贝尔文学奖准备颁发给沈从文，1988年的5月10日，[[龙应台|龙应台]]打电话告诉马悦然沈从文过世的消息，马悦然给中国驻瑞典大使馆文化秘书打电话确认消息，又给他的好友文化记者李辉打电话询问消息，最终确认沈从文过世了。<ref>{{cite web |url=http://cul.sohu.com/20071010/n252580088.shtml|title=沈从文如果活着就肯定能得诺贝尔文学奖|accessdate=2007年10月10日16|work=|publisher=南方周末|language= zh-hans}}</ref> 马悦然屡次想说服瑞典学院破例把诺奖授予死去的人，最后一次使出浑身解数劝说无效后，他哭着离开了会场。<ref>{{cite web |url=http://book.ifeng.com/special/nobelwxj2008/list/200810/1007_4772_820468.shtml|title=马悦然喜欢“乡巴佬作家”曹乃谦|accessdate=2008年10月7日|work=|publisher=深圳商报|language= zh-hans}}</ref>

==家庭==
馬悅然第一任妻子為中国籍陈宁祖，1996年陈宁祖去世。<ref>{{cite web |url=http://www.chinawriter.com.cn/bk/2005-11-25/4408.html|title=马悦然牵手陈文芬|accessdate=2005年11月17日|work=楼乘震|publisher=深圳晚报|language= zh-hans}}</ref>1998年馬悅然與《[[中國時報|中國時報]]》記者[[陳文芬|陳文芬]]相识，2005年在中国山西宣布婚讯。<ref>{{cite web |url=http://www.bjdclib.com/subdb/laneculture/famousperson/201004/t20100416_32233.html|title=《名人：马悦然》|accessdate=|work=|publisher=东华流韵|language= zh-hans}}</ref>
