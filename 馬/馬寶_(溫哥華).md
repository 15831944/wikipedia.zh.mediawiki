'''馬寶'''<ref>{{cite news |title=溫馬寶區地盤暫禁施工 馬斯琴原住民歡迎 發展商或告省府 |url=http://news.singtao.ca/vancouver/2012-09-29/canada1348913457d4116677.html |newspaper=《星島日報》 |date=2012-09-29 |accessdate=2013-01-03}}</ref>（{{lang-en|Marpole}}）是[[加拿大|加拿大]][[卑詩省|卑詩省]][[溫哥華|溫哥華]]市一個社區，北面以西57街與[[渥烈治|渥烈治區]]為鄰，東西分別以[[緬街_(溫哥華)|緬街]]和安格斯路為界，南面則瀕臨[[菲沙河|菲沙河]]的北支流與[[列治文|列治文]]的[[魯魯島|魯魯島]]和[[海島_(卑詩省)|海島]]對望<ref>{{Cite web |url=http://vancouver.ca/community_profiles/marpole/index.htm |title=Vancouver - communityWEBpages - Marpole |accessdate=2013-01-03 |publisher=溫哥華市政府 |archiveurl=https://web.archive.org/web/20060418013939/http://vancouver.ca/community_profiles/marpole/index.htm |archivedate=2006-04-18 |deadurl=yes }}</ref>，總面積為561公頃<ref name="Profile">{{Cite web |url=http://vancouver.ca/commsvcs/Census2001/Marpole03.pdf |title=Marpole Community Statistics |accessdate=2013-01-03 |publisher=溫哥華市政府 |archiveurl=https://web.archive.org/web/20060418075300/http://vancouver.ca/commsvcs/Census2001/Marpole03.pdf |archivedate=2006-04-18 |deadurl=yes }}</ref>；[[固蘭湖街|固蘭湖街]]、[[渥街|渥街]]和[[甘比街|甘比街]]則為區内另外三條主要南北向道路。[[溫哥華架空列車|架空列車]][[加拿大線|加拿大線]]於2009年開通，並於馬寶區東部甘比街與西南海旁大道交界設有[[海旁大道站|海旁大道站]]。

==歷史==
現馬寶區的人類聚落史可追溯至公元前3500年的原住民聚落；非原住民族居民則從1860年代起到此定居<ref name="history">{{Cite web |url=http://vancouver.ca/community_profiles/marpole/history.htm |title=History and Heritage |accessdate=2013-01-03 |publisher=溫哥華市政府 |archiveurl=https://web.archive.org/web/20050219122807/http://vancouver.ca/community_profiles/marpole/history.htm |archivedate=2005-02-19 |deadurl=yes }}</ref>。1881年，哈里·伊本（{{lang|en|Harry Eburne}}）在現馬寶區開設一家商店，再於翌年加開一家郵局，並將之名為「北支流郵局」（{{lang|en|North Arm Post Office}}），一個社區亦圍繞著他的商店和郵局而成型<ref name="GeoBC">{{Cite web |url=http://apps.gov.bc.ca/pub/bcgnws/names/49998.html |title=BC Geographical Names: Eburne |accessdate=2013-01-03 |publisher=GeoBC}}</ref>。隨著跨越菲沙河北支流通往海島的橋樑開通，伊本亦於1891年將他的商店和郵局遷至海島的東北端；坐落菲沙河北支流兩岸的社區亦於往後年間以他命名為伊本<ref name="GeoBC"/>。

連接溫哥華和[[史提夫斯頓|史提夫斯頓]]的市際電車於1905年開通，並於現馬寶區設立伊本站，刺激了該帶的經濟<ref name="history"/>。鋸木廠和其他工廠陸續進駐該區，而另一家郵局則以「伊本站郵局」之名於1911年開業<ref name="GeoBC"/>。為了與海島上的伊本區作出區別，伊本站郵局周圍的社區於1916年改以[[加拿大太平洋鐵路|加拿大太平洋鐵路]]公司高層理查·馬寶（{{lang|en|Richard Marpole}}）命名為馬寶區<ref name="GeoBC"/>。馬寶區所屬的格雷角自治區（{{lang|en|Point Grey Municipality}}）於1929年聯同南溫哥華自治區併入溫哥華市，該區亦隨之成為溫哥華市的主要工業區之一<ref name="history"/>。

[[渥街橋|渥街橋]]於1957年開通後，來往列治文和溫哥華的交通改經渥街，令馬寶區原有坐落於哈德遜街和西南海旁大道交界的商業區生意一落千丈<ref name="history"/>。[[亞瑟·萊恩橋|亞瑟·萊恩橋]]於1975年開通後，與其接駁的固蘭湖街南端則變成馬寶區的商業活動焦點所在<ref name="history"/>。

==人口統計==
2011年馬寶有23,832名居民，較2006年只錄得些微增長。就居民歲數而言，45至64歲居民為區内最大群體，其次為30至44歲居民。區内居民有42%以[[漢語|漢語]]（沒有指定哪一種方言）為母語，是眾語言之首，其次為[[英語|英語]]（37%）。區内家庭入息中位數為$41,125，而32.2%居民為低收入戶。<ref>{{Cite web |url=http://vancouver.ca/files/cov/marpole-community-plan-open-house-display-boards-2012-may.pdf |title=Marpole Community Plan Open House Display Boards |accessdate=2013-01-03 |publisher=溫哥華市政府 |date=2012-05}}</ref>
