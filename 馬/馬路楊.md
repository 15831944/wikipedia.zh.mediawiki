'''馬路楊檳榔會社'''（店名：'''楊-{双}-冬檳榔'''）是位於[[台南市|台南市]][[中西區_(台灣)|中西區]]的一間檳榔攤，經營者為'''楊馬路'''，因為[[一青妙|一青妙]]的書籍介紹而在日本暴紅<ref name="udn">{{cite web |title=馬路楊的檳榔會社沒「西施」 |url=https://udn.com/upf/newmedia/2015_vist/11/20151116_tainan_09/index.html |website=[[聯合新聞網|聯合新聞網]] |accessdate=2018-08-27 |language=zh-Hant}}</ref>，3年無償招待500多組日本遊客而聞名<ref name=":1">{{cite web |title=【人情檳榔會社一】沒西施只有怪老頭　迷倒數千日本客 |author1=謝祝芬 |url=https://www.mirrormedia.mg/story/20180801bus010/ |website=[[鏡週刊|鏡週刊]] |accessdate=2018-08-27 |language=zh-TW |archive-url=https://web.archive.org/web/20180810130810/https://www.mirrormedia.mg/story/20180801bus010/ |archive-date=2018-08-10 |dead-url=no }}</ref>。被一青妙譽為「台南人情味代表」<ref name="玩咖Playing">{{cite web |title=日本人遊台南私景點！跟著美女作家「一青妙」感受濃濃府城人情味 |url=http://playing.ltn.com.tw/article/10331/1 |website=自由時報玩咖Playing |accessdate=2018-08-27 |language=zh |date=2018-08-03 |archive-url=https://web.archive.org/web/20180803114448/http://playing.ltn.com.tw/article/10331/1 |archive-date=2018-08-03 |dead-url=no }}</ref>。

==簡介==
楊馬路（1965年—）身高158公分，戴眼鏡、穿白色麻衫、蓄短鬍。<ref name="中"/>楊馬路原本身高171公分，因為[[糖尿病|糖尿病]]導致[[骨質疏鬆|骨質疏鬆]]而矮了13公分。<ref name=":2">{{cite web |title=【人情檳榔會社二】歡喜被吃「豆乾」　日本作家認他當爸爸 |url=https://www.mirrormedia.mg/story/20180801bus011/ |website=[[鏡週刊|鏡週刊]] |accessdate=2018-08-27 |author1=謝祝芬 |language=zh |date=2018-08-09 |archive-url=https://web.archive.org/web/20180810122900/https://www.mirrormedia.mg/story/20180801bus011/ |archive-date=2018-08-10 |dead-url=no }}</ref>小時候因父親生病而放棄學業繼承檳榔攤。<ref name="中"/>因為作家[[一青妙|一青妙]]到檳榔攤問路與談話而結識。<ref name="中">{{cite web |author1=陳治交 |title=馬路楊檳榔攤 日客愛朝聖 |url=http://www.cdns.com.tw/news.php?n_id=1&nc_id=87786 |website=[[中華日報|中華日報新聞網]] |accessdate=2018-08-27 |date=2016-04-13}}</ref>一青妙覺得楊馬路抽菸喝酒的樣子像她父親[[顏惠民|顏惠民]]，因此認楊馬路為「[[乾親|第二個爸爸]]」<ref name=":2"/>，一青妙將這段經歷寫入書中，此書出版後吸引需多日本遊客前來拜訪，截至2018年8月已經有超過500組（3000多人次<ref name="玩咖Playing"/>）遊客慕名而來，楊馬路也樂意擔任解說員導覽。<ref name="line">{{cite web |title=【台灣老店】檳榔裡的人情案內所　馬路楊檳榔會社 |url=https://today.line.me/TW/pc/article/yLlvY1?utm_source=lineshare |website=LINE TODAY |accessdate=2018-08-27 |language=zh |date=2018-08-10}}</ref>楊馬路經營檳榔三十餘年，十分熟識此地的文化，{{tsl|ja|佐藤春夫記念館|佐藤春夫記念館}}館長辻本雄一等研究[[佐藤春夫|佐藤春夫]]的台灣鬼怪小說〈女誡扇綺譚〉的團隊曾請楊馬路帶領團隊尋找書中所描述的地點位置等內容。<ref>{{cite book |author1=佐藤春夫 |authorlink1=佐藤春夫 |coauthors=詹慕如 |title=〔新譯〕帝國旅人佐藤春夫行腳台灣：收錄〈女誡扇綺譚〉一個臺南府城的奇幻傳說 |date=2016-11-03 |publisher=紅通通文化 |url=https://www.books.com.tw/products/0010733698 |access-date=2018-08-27 |archive-url=https://web.archive.org/web/20170905211238/http://www.books.com.tw/products/0010733698 |archive-date=2017-09-05 |dead-url=no }}</ref><ref>{{cite web |author1=黒羽夏彦 |title=台湾史を知るためのブックガイド#21 |url=http://motto-taiwan.com/2015/05/0514/ |website=もっと台湾 |accessdate=2018-08-27 |language=ja |date=2015-05-14 |archive-url=https://web.archive.org/web/20170324010412/http://motto-taiwan.com/2015/05/0514/ |archive-date=2017-03-24 |dead-url=no }}</ref>

楊馬路不會日語，但是能用比手畫腳的方式介紹給日本遊客。<ref name="中"/>楊馬路以熱情和人情味著名。楊馬路也曾經在一青妙的作品《日本媽媽的臺菜物語》的台日合作電影《[[媽媽，晚餐吃什麼？|媽媽，晚餐吃什麼？]]》客串演出。<ref name=":2"/>

楊馬路認為[[檳榔|檳榔]]並不是不好的事物<ref name=":5">{{cite web |author1=謝祝芬 |title=【人情檳榔會社五】背負檳榔原罪半世紀　他終敢抬頭挺胸 |url=https://www.mirrormedia.mg/story/20180801bus014/ |website=[[鏡週刊|鏡週刊]] |accessdate=2018-08-27 |language=zh-TW |archive-url=https://web.archive.org/web/20180810123800/https://www.mirrormedia.mg/story/20180801bus014/ |archive-date=2018-08-10 |dead-url=no }}</ref>，並表示檳榔本來是中藥、早期台灣的檳榔是貴重的商品，現在卻被少數掛羊頭賣狗肉的賭場和情色場所汙名化。<ref name="udn"/><ref name="我的台南"/>這間檳榔攤就是一間沒有[[檳榔西施|檳榔西施]]的檳榔攤。<ref name="udn"/><ref name="line"/>

楊-{双}-冬檳榔攤店面位於[[台南市|台南市]][[中西區_(台灣)|中西區]]民族路和康樂街交叉口，是經營50多年的老店面<ref>{{cite book |author1=CREA Traveller編集部 |title=CREA Traveller 2016 Spring NO.45 |date=2016 |accessdate=2018-08-27 |language=ja|ASIN=B01COSC3WS|asin-tld=jp}}</ref>。招牌中的[[雙冬|-{双}-冬]]是台灣[[南投縣|南投縣]]的檳榔產地<ref name="我的台南"/>，招牌中的楊為「◯」內有「楊」字樣，在日治時代這種「◯」標誌稱為「{{lang|ja|[[圈號|丸]]}}（{{lang-ja|まる|maru}}）」發音接近中文「馬路」的發音。<ref name="我的台南">{{cite book |author1=一青妙 |authorlink1=一青妙 |title=我的臺南：一青妙的府城紀行 |date=2015年6月22日 |publisher=聯經出版 |isbn=9789570845778 |accessdate=2018-08-27}}</ref><ref>{{cite web |title=《秋刀魚》X《正興聞》首度聯名！最道地的街頭臺式日語會話集 |url=http://qdymag.com/news/129 |website=秋刀魚 |accessdate=2018-08-27 |language=ja |date=2016-08-03 |archive-url=https://web.archive.org/web/20170721065709/http://qdymag.com/news/129 |archive-date=2017-07-21 |dead-url=no }}</ref><ref>{{youtube|KU9ndMrLZrM|m=2|s=29|台湾ファンの間では知る人ぞ知る台南のビンロウ店経営者、馬路楊さんの話題|language=zh}},[[櫻花頻道|櫻花頻道]],2016-03-03</ref>

在結識一青妙之後便成立[[臉書|臉書]]粉絲團「馬路楊檳榔會社」<ref name="我的台南"/>，並於粉絲團紀錄來訪的遊客、推廣台日交流和檳榔文化。
<ref>{{cite web |title=台南で3千人の日本人をもてなした檳榔おじさんが語る「日台友好」の心 |url=https://www.nippon.com/ja/column/g00433/ |website=nippon.com |accessdate=2018-08-27 |language=ja |date=2017-08-28 |archive-url=https://web.archive.org/web/20171115230449/http://www.nippon.com/ja/column/g00433/ |archive-date=2017-11-15 |dead-url=no }}</ref><ref name="我的台南"/>不乏一些知名人士前來，如[[八田與一|八田與一]]的孫子八田修一與[[金澤市|金澤市]]觀光大使德光重人來訪。<ref name=":5"/>日本電視台{{tsl|ja|アナザースカイ|Another Sky}}曾經拍攝一青妙的妹妹[[一青窈|一青窈]]來訪的節目 <ref>{{cite web |title=another sky-アナザースカイ- |url=http://www.ntv.co.jp/anothersky/contents/2014/11/post-1575.html |website=www.ntv.co.jp |accessdate=2018-08-27 |date=2014-11-28 |archive-url=https://web.archive.org/web/20160302112648/http://www.ntv.co.jp/anothersky/contents/2014/11/post-1575.html |archive-date=2016-03-02 |dead-url=no }}</ref>。甚至有一些遊客一年會前來拜訪數十次。<ref name="中"/>
