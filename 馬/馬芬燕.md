'''馬芬燕'''，（英文名︰Stephanie Ma），綽號「STEP」，[[香港|香港]]出生，是香港聾人機構「[[龍耳|龍耳]]」的手語翻譯員<ref>{{Cite news|language=zh-hk |url=http://stedu.stheadline.com/sec/sec_news.php?aid=16336&friendly_print=1|title=溝通橋梁 助求診求學 手語翻譯員 打破聽說障礙|publisher=[[星島日報|星島日報]]|date=2016-11-09|accessdate=2018-06-03}}</ref>，亦是[[龍耳電視|龍耳電視]]的編輯。現她符合《[[香港手語翻譯員名單|香港手語翻譯員名單]]》基本條件的手語翻譯員。她亦有參與[[亞洲電視|亞洲電視]]節目「[[感動香港|感動香港]]」、[[香港電台|香港電台]]電視節目「[[早辰。早晨|早辰。早晨]]」及[[無綫電視|無綫電視]]新聞節目「[[手語新聞報道|手語新聞報道]]」<ref>{{Cite news|language=zh-hk |url=https://www.singpao.com.hk/index.php?fi=news1&id=75247|title=手語翻譯員：與聾人溝通了解最重要|publisher=[[成報|成報]]|date=2018-07-04|accessdate=2018-07-04}}</ref><ref>{{Cite news|language=zh-hk |url=http://hd.stheadline.com/news/daily/hk/681177/|title=明珠台每晚七時正配中文字幕 ﻿無線手語新聞﻿﻿方便聽障觀眾﻿|publisher=[[頭條日報|頭條日報]]|date=2018-07-04|accessdate=2018-07-04}}</ref>的[[手語傳譯|手語傳譯]]工作。

== 家庭背景 ==
馬芬燕的兄姊均為聾人，為協助他們跟健聽父母溝通，她從小開始學習[[手語|手語]]<ref>{{Cite news|language=zh-hk |url=https://hk.news.appledaily.com/local/daily/article/20140626/18777092|title=手語旁述世界盃 聽障人士更投入|publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]]|date=2014-06-26|accessdate=2018-06-03}}</ref>。

在[[2014年國際足協世界盃|2014年國際足協世界盃]]，趁世界盃熱潮推動共融社會。馬芬燕指出兄姊屬於天生遺傳性失聰，慶幸自己和妹妹是健聽人士，自覺要把握所有機會，回饋上天眷顧。她期望把一向寂靜的世界，改造成歷來最熱鬧的足球派對，希望藉此在社會推動共融訊息<ref>{{Cite news|language=zh-hk |url=https://hk.sports.appledaily.com/sports/daily/article/20140521/18726923|title=話題：手語世盃無聲仿有聲|publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]]|date=2014-05-21|accessdate=2018-06-03}}</ref><ref>{{Cite news|language=zh-hk |url=http://203.80.0.224/pdf_district/20140530/HLD20140530.C01-4C.PDF|title=同享世盃樂 增設手語旁述 不再孤立聽障|publisher=[[星島日報|星島日報]]|date=2014-05-30 至 06-04|accessdate=2018-06-03}}</ref>。

馬芬燕指出，[[香港政府|香港政府]]有很多新聞，宣傳片及廣告都沒有配上[[手語傳譯|手語傳譯]]，但手語和文字的文法有不同，單靠字幕很難理解全部內容。聾人經常誤墮騙局，她的哥哥就因為缺乏資訊，被非法的傳銷公司欺騙。她希望透過龍耳電視製作配有手語傳譯的節目，協助聾人理解新聞資訊<ref>{{Cite news|language=zh-hk |url=http://www.mingpaocanada.com/tor/htm/News/20141205/HK-gzd1_r.htm|title=聽障組織搞視頻 鬼古都有手語版|publisher=[[明報_(香港)|明報 (香港)]]|date=2014-12-05|accessdate=2018-06-03}}</ref><ref>{{Cite news|language=zh-hk |url=http://std.stheadline.com/daily/news-content.php?id=216648&target=2|title=首個手語頻道「龍耳」電視啟播|publisher=[[星島日報|星島日報]]|date=2014-12-05|accessdate=2018-06-03}}</ref><ref>{{Cite news|language=zh-hk |url=https://hk.lifestyle.appledaily.com/lifestyle/special/daily/article/20161231/19881809|title=【專題籽】無聲勝有聲　六人撐起聾人電視台|publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]]|date=2016-12-31|accessdate=2018-06-03}}</ref>。

== 龍耳電視 ==
馬芬燕指出，有些電視台直播節目時，把手語傳譯框刪走，導致聾人無從理解節目內容，反映有必要製作手語版本的電視節目。龍耳電視於2014年11月成立，製作配有手語傳譯的節目，協助聾人理解新聞資訊。

她亦指出，聾人以手語為母語，與平常的語言文法相差很大，以致現時很多資訊都未能傳達給他們。她認為，單靠看在電視下方顯示的字幕，根本不能讓他們把內容消化，因為他們在閱讀文字時，比一般人需要更多時間去理解<ref>{{Cite news|language=zh-hk |url=https://jmc.hksyu.edu/shuo/%E6%A9%9F%E6%A7%8B%E6%8E%A8%E6%B8%AF%E9%A6%96%E9%96%93%E6%89%8B%E8%AA%9E%E9%9B%BB%E8%A6%96%E5%8F%B0/|title=機構推港首間手語電視台|publisher=說在線|date=2014-12-05|accessdate=2018-06-03}}</ref>。

== 手語傳譯 ==
她指出，[[香港手語|香港手語]]主要有兩種，一種是中文手語，即是直接將整句句字的每一個字順序打出，而另一種則是自然手語，以「重點在前，問句量詞在後」為原則，方便聾人迅速理解說話內容。例如想問對方有沒有facebook，中文手語的打法是直接將「你有沒有facebook」依次打出，而自然手語則是先打「facebook」，再打「有沒有」<ref>{{Cite news|language=zh-hk |url=https://hk.lifestyle.appledaily.com/lifestyle/special/daily/article/20170210/19923414|title=【潮語手語】考吓你「你識條鐵」　點樣打？|publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]]|date=2017-02-10|accessdate=2018-06-03}}</ref>。

她亦指出，每個聾人都有自己一套手語的打法，因各人成長背景、文化及學歷水平等不同，以致對不同字詞的理解也有不同，故演繹出來的手語亦有所不同。她認為要成為一個好的翻譯員，最重要是要多接觸聾人，才可了解到那位聾人所打的手語，是否和自己學的意思相同<ref>{{Cite news|language=zh-hk |url=http://hk.on.cc/hk/bkn/cnt/news/20160318/bkn-20160318103553931-0318_00822_001.html|title=手語不統一政府不支援　翻譯認可路難走|publisher=[[東方日報_(香港)|東方日報 (香港)]]|date=2016-03-18|accessdate=2018-06-03}}</ref>。

馬芬燕認為香港政府除欠缺手語翻譯員的人手外，前線的警務人員亦缺乏對聾人的基本認知，加上內部溝通不足，令警務處對申請手語傳譯員支援感到陌生。她指並非要求前線人員能與聾人溝通，但希望他們能夠有正確態度，並清楚申請手語翻譯員的程序<ref>{{Cite news|language=zh-hk |url=https://jmc.hksyu.edu/ourvoice/?p=9486|title=短訊難報案　前線欠認知　港聾人無從求救|publisher=仁聞報|date=2018-03-13|accessdate=2018-06-03}}</ref>。

== 龍耳手譯寶 ==
聾人機構「龍耳」2017年4月推出[[龍耳手譯寶|龍耳手譯寶]]的手機應用程式，馬芬燕指出會員可免費使用服務，非會員可註冊成為會員後免費使用<ref>{{Cite news|language=zh-hk |url=https://jump.mingpao.com/resources/detail/20170328/s00012/1490666997141|title=真人翻譯App 助弱聽健聽者溝通|publisher=[[明報_(香港)|明報 (香港)]]|date=2017-03-28|accessdate=2018-06-03}}</ref>。她接受《政府新聞網》訪問時說，最常收到的要求是回撥電話，就是有聾人不斷收到電話來電，但又不知道來電者的來意，於是利用這手機應用程式聯絡，要求協助詢問來電者的目的。詢問後才知道他們曾訂購床褥或其他物品，對方準備送貨，只是他們遺忘了<ref>{{Cite news|language=zh-hk |url=https://www.hk01.com/%E7%A4%BE%E6%9C%83%E6%96%B0%E8%81%9E/119675/%E6%89%8B%E8%AA%9E%E7%BF%BB%E8%AD%AF%E5%93%A1%E7%BE%A9%E5%B7%A5%E9%96%8B%E7%99%BC%E7%A8%8B%E5%BC%8F-%E7%82%BA%E8%81%BE%E5%95%9E%E4%BA%BA%E5%A3%AB%E6%8F%90%E4%BE%9B%E5%8D%B3%E6%99%82%E7%BF%BB%E8%AD%AF|title=手語翻譯員義工開發程式　為聾啞人士提供即時翻譯|publisher=[[香港01|香港01]]|date=2017-09-17|accessdate=2018-06-03}}</ref><ref>{{Cite news|language=zh-hk |url=http://paper.wenweipo.com/2017/09/25/HK1709250050.htm|title=真人App手譯寶 傷健「溝通到」|publisher=[[文匯報_(香港)|文匯報 (香港)]]|date=2017-09-17|accessdate=2018-06-03}}</ref>。
