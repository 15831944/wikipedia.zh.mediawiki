《'''齊木楠雄的災難'''》（{{lang-ja|斉木楠雄のΨ難}}）是由[[麻生周一|麻生周一]]所作的一部[[漫畫|漫畫]]作品。於《[[週刊少年Jump|週刊少年Jump]]》2012年24號開始連載<ref name="flash">{{Cite web |url=http://comic.qq.com/a/20130704/010863.htm |title=《齐木楠雄的灾难》FLASH动画化！ |date=2013-07-04 |work=腾讯动漫}}</ref>至2018年13號完結，單行本全26卷。

關於本作標題中的[[Ψ|Ψ]]是希臘字母、讀做{{IPA|/ˈsaɪ/}}，與日文的「災」同音，同時也表示超能力。

== 劇情大綱 ==
{{main|齊木楠雄的災難角色列表}}

能夠窺視別人的內心、看透牆壁、以念力移動物品、瞬間移動，一出生就擁有一切的超能力！他就是沉默寡言的超能力者·'''齊木楠雄'''。在一個平凡的日本家庭，父母都很普通，齊木楠雄的出生卻不平凡。齊木楠雄天生下來就是個超能力者，長大後逐漸明白自己與別人不同，雖然擁有足以毀滅世界的力量，但齊木楠雄每天只是想辦法讓自己默默地隱藏在人群之中。不想引人注目因此極力做個平凡人，直到國中都順利當個不起眼的學生。

然而事與願違，他上了PK學園遇上了許多特別的同學後，楠雄的父母以及笨蛋·'''燃堂力'''、中二病·'''海藤瞬'''等人卻一點也不打算讓他安安靜靜地過活，甚至連全校第一的完美美少女·'''照橋心美'''都愛上了他！平凡的日子逐漸變成對他來說是個多災多難的生活…究竟楠雄能不能使用他的超能力，在持續不斷的災難中繼續過著不受矚目的平穩生活？

== 發展 ==
2011年起初在《[[週刊少年Jump|週刊少年Jump]]》以短篇發表，初期的標題則是《'''超能力者齊木楠雄的災難'''》（{{lang|ja|超能力者斉木楠雄のΨ難}}），而後2012年開始連載，連載前的讀切版收錄在第0冊。

於2012年24號開始正式連載並更名為《'''齊木楠雄的災難'''》（{{lang|ja|斉木楠雄のΨ難}}），直至2018年13號完結（合計全279話+番外篇5話），單行本合計全26卷。完結後以四格漫畫版於2018年14號到31號期間進行連載（合計17話，收錄在最終冊）。

另有合作作品《'''殺老師VS齊木楠雄 ～入間市最終決戰～'''》（{{lang|ja|殺せんせーVS斉木楠雄 ～入間市最終決戦～}}），是作者[[麻生周一|麻生周一]]和《[[暗殺教室|暗殺教室]]》的作者[[松井優征|松井優征]]同為[[埼玉縣|埼玉縣]][[入間市|入間市]]出身，2位同鄉推出合作短篇。第一次合作收錄在暗殺教室單行本第2冊，故事圍繞殺老師與齊木楠雄爭奪「入間饅頭」的決戰。第二次合作則收錄在暗殺教室單行本第7冊，故事以殺老師與齊木楠雄兩個角色和以「茶道」為背景展開。

另有特別篇作品《'''照橋心美的災難'''》（{{lang|ja|照橋心美のΨ難}}），以照橋心美為主角描繪的特別篇作品，於《Cocohana》2015年8月號公開，收錄在單行本15卷。

於2018年5月25日至7月26日在《[[週刊少年Jump的增刊號|JUMP GIGA]]》2018 SUMMER vol.1及vol.3刊載共2話的後日談《'''齊木楠雄的再啟動'''》（{{lang|ja|斉木楠雄のΨ起動}}），收錄於最終冊。

== 發行書籍 ==
漫畫單行本合計全26冊，此外第0冊為《麻生周一短篇集》，日文原書由[[集英社|集英社]]發行；台灣中文版由[[東立出版社|東立出版社]]發行，詳細資訊請參考以下書籍資料。

此外另有兩部小說，書名為《{{lang|ja|斉木楠雄のΨ難 EXTRA STORY OF PSYCHICS}}》。作者為{{lang|ja|ひなたしょう}}，由集英社〈JUMP j-BOOKS〉發行。第一集發售日為2013年5月2日（ISBN 978-4-08-703289-5）；第二集發售日為2014年7月4日（ISBN 978-4-08-703321-2）。

另有公式角色導讀手冊，書名為《{{lang|ja|斉木楠雄のΨ難 左脇腹町防Ψハンドブック}}》，於2015年8月4日發行（ISBN 978-4-08-908247-8）。其他書籍有，書名為《{{lang|ja|斉木楠雄のΨ難 超能力エクサΨズ}}》，於2013年12月4日發行（ISBN 978-4-08-908247-8）。

== 衍生作品 ==
=== VOMIC ===
於2012年6月在[[集英社|集英社]]Voice Comic Station以[[VOMIC|VOMIC]]放映，全12話。除童年楠雄為[[田中愛美|田中愛美]]配音以外，與電視動畫版聲優不同陣容<ref>{{Cite web |url=http://www.dongman.gov.cn/xinwen/guoji/201306/t20130628_692983.htm |title=JUMP连载漫画《齐木楠雄的Ψ难》动画化？ |date=2013-06-28 |work=国家动漫产业网 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes}}</ref>，齊木楠雄為[[淺沼晉太郎|淺沼晉太郎]]；燃堂力為[[三宅健太|三宅健太]]；海藤瞬為[[江口拓也|江口拓也]]；照橋心美為[[阿澄佳奈|阿澄佳奈]]；齊木國春為佐佐木啓夫；齊木久留美為三上遙香；高橋為三宅貴大；旁白為[[西山宏太朗|西山宏太朗]]；其餘角色有不良少年為[[齊藤壯馬|齊藤壯馬]]。

此外另有FLASH動畫，於2013年8月在「週刊少年JUMP LIVE」以FLASH動畫放映，全5話，聲優與VOMIC同一陣容。

=== 電視動畫 ===
{{main|齊木楠雄的災難 (動畫)}}

於2016年5月9日公佈製作電視動畫，並於同年7月開播至12月播畢同時宣布決定製作續篇；第二期續篇於2018年1月開播至6月底播畢，並公布製作完結篇並於2018年12月底播出。

=== 電影 ===
{{main|齊木楠雄的災難 (2017年電影)}}

於2015年6月15日公佈拍攝真人電影，由[[福田雄一|福田雄一]]导演的日本动作喜剧片，改编自[[麻生周一|麻生周一]]所创作的著名漫画《[[齊木楠雄的災難|齊木楠雄的災難]]》，主角齊木楠雄由[[山崎賢人|山崎賢人]]主演，电影於[[2017年|2017年]][[10月21日|10月21日]]在日本上映，臺灣於[[2018年|2018年]][[4月20日|4月20日]]上映。

=== 電子遊戲 ===
