《'''-{zh-tw:雙面人魔; zh-cn:汉尼拔;zh-hk:沉默的羔羊前傳;zh-sg:人魔再起·HANNIAL}-'''》（{{lang-en|''Hannibal''}}），是[[湯瑪斯·哈里斯|湯瑪斯·哈里斯]]的小說《{{link-en|紅龍 (小說)|Red Dragon (novel)|紅龍}}》及《{{link-en|人魔 (小說)|Hannibal (Harris novel)|人魔}}》基礎上改編的美国{{link-en|心理學驚悚|Psychological thriller|心理驚悚}}[[电视剧|电视剧]]。2012年2月14日[[國家广播公司|NBC]]宣布跳过试映环节直接预订本剧第一季13集<ref>{{Cite web|url=http://www.deadline.com/2012/02/nbc-gives-series-order-to-hannibal-picks-up-notorious-drama-pilot/|title=NBC Gives Straight-To-Series Order To 'Hannibal', Picks Up 'Notorious' Drama Pilot|work=Deadline|last=Andreeva|first=Nellie|date=February 14, 2012|accessdate=April 21, 2012}}</ref>，{{link-en|大衛·史雷德|David Slade}}執行製片及執導第一集，2013年4月4日首播。第二季也是13集，在2014年2月28日播映。2014年5月9日，NBC續訂第三季，於2015年6月4日播放。

2015年6月22日，NBC[[國家廣播公司|國家廣播公司]]因收視率低落的緣故於第三季後取消本劇。<ref>{{Cite web |url = http://tvline.com/2015/06/22/hannibal-cancelled-nbc-season-4/ |title = ''Hannibal'' Cancelled at NBC |author = Michael Ausiello |date = 2015-06-22 |publisher = TVLine |language = en}}</ref>

==簡介==
《漢尼拔》是一齣美國心理驚悚連續劇，由[[布萊恩·富勒|布萊恩·富勒]]監制。此劇是根據[[湯瑪斯·哈里斯|湯瑪斯·哈里斯]]的著作[[紅龍|紅龍]]改編。故事主要講述美國聯邦調查局探員威爾·葛蘭姆（ [[休·丹西|休·丹西]] 飾演）和精神病學家-{漢尼拔}-（ [[麦德斯·米科尔森|麦德斯·米科尔森]] 飾演）之間的關係和其周邊展開的凶殺案。該系列於2013年4月4日首播。
