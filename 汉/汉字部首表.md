《'''汉字部首表'''》，標準名GF 0011-2009是現時[[中國大陸|中國大陸]]的[[汉字|汉字]][[部首|部首]]標準，共规定'''主部首'''201个，'''附形部首'''100个<ref>{{cite web |url=http://www.china-language.gov.cn/9/2009_9_23/1_9_4344_0_1253687722552.html |title=存档副本 |accessdate=2016-01-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20150923202814/http://www.china-language.gov.cn/9/2009_9_23/1_9_4344_0_1253687722552.html |archivedate=2015-09-23 }}</ref>。現時中國大陸主要字書如《[[新華字典|新華字典]]》和《[[現代漢語詞典|現代漢語詞典]]》的新版本都使用此標準。

由于中国大陆推行[[漢字簡化|简体字]]，《康熙字典》的[[康熙部首|214部首]]不能适应实际需求，[[中国文字改革委员会|中国文字改革委员会]]和[[中国国家出版局|国家出版局]]於1983年发布了《'''汉字统一部首表(草案)'''》<ref>{{cite web |url=http://www.china-language.gov.cn/wenziguifan/scanning/gfsc/gfbz10.htm |title=存档副本 |accessdate=2013-03-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20130524144704/http://www.china-language.gov.cn/wenziguifan/scanning/gfsc/gfbz10.htm |archivedate=2013-05-24 }}</ref>。2009年[[中华人民共和国教育部|教育部]]和[[国家语言文字工作委员会|国家语言文字工作委员会]]重新整理及发布。<ref>{{cite web |url=http://www.moe.gov.cn/s78/A19/yxs_left/moe_810/s230/201001/t20100115_75694.html |title=漢字部首表|accessdate=2018-6-3}}</ref>
