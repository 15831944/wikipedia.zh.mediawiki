汉诺威作为[[神圣罗马帝国|神圣罗马帝国]]的一个邦国，前身是不伦瑞克-卡伦贝格公国，不伦瑞克-卡伦贝格是不伦瑞克-吕讷堡的幼系。该公国1636年被封给[[不伦瑞克统治者列表|不伦瑞克-吕讷堡公爵]][[威廉四世_(不伦瑞克-吕讷堡)|「胜利者」威廉四世]]的第六子[[格奥尔格_(不伦瑞克-卡伦贝格)|格奥尔格]]。不久，该家系将首府迁移到了[[汉诺威|汉诺威]]城。1692年，不伦瑞克-卡伦贝格公爵[[恩斯特·奧古斯特_(漢諾威选侯)|恩斯特·奧古斯特]]被晋升为[[选侯|选侯]]，他的长子[[喬治一世_(大不列顛)|格奥尔格·路德维希]]更于1714年继承了[[英国|英国]]的王位。1814年10月，在[[维也纳会议|维也纳会议]]上，汉诺威升为[[王国|王国]]。

以下是[[汉诺威_(地区)|汉诺威]]历代君主的名单。


== 汉诺威选侯国 ==
*1692年-1698年：[[恩斯特·奧古斯特_(漢諾威选侯)|恩斯特·奧古斯特]]
*1698年-1727年：[[乔治一世_(大不列颠)|格奥尔格一世·路德维希]]（Georg I Ludwig）  <span style="font-size:smaller;">1714年後兼任[[英国君主列表|大不列顛和爱尔兰國王]]</span>
*1727年-1760年：[[乔治二世_(英国)|格奥尔格二世·奧古斯特]]（Georg II August）  <span style="font-size:smaller;">兼任大不列顛和爱尔兰國王</span>
*1760年-1802年：[[乔治三世_(英国)|格奥尔格三世·威廉·弗里德里希]]（Georg III Wilhelm Friedrich）  <span style="font-size:smaller;">兼任大不列顛和爱尔兰國王，1814年成为汉诺威国王</span>
1802年，汉诺威被法国占领，1807年并入[[威斯特法伦王国|威斯特法伦王国]]。

== 汉诺威王国 ==
*1814年-1820年：[[乔治三世_(英国)|格奥尔格三世]]
*1820年-1830年：[[喬治四世_(英國)|格奥尔格四世·奧古斯特·弗里德里希]]（Georg IV August Friedrich）  <span style="font-size:smaller;">兼任大不列顛和爱尔兰联合王国國王</span>
*1830年-1837年：[[威廉四世_(英國)|威廉·亨利]]（Wilhelm  Heinrich）  <span style="font-size:smaller;">兼任大不列顛和爱尔兰联合王国國王</span>
*1837年-1851年：[[恩斯特·奧古斯特一世_(漢諾威)|恩斯特·奧古斯特一世]]（Ernst August I）  <span style="font-size:smaller;">兼任[[英国|英国]][[坎伯蘭公爵|坎伯蘭公爵]]</span>
*1851年-1866年：[[格奥尔格五世_(漢諾威)|格奥尔格五世]]（Georg V） <span style="font-size:smaller;">兼任[[英国|英国]][[坎伯蘭公爵|坎伯蘭公爵]]</span>
1866年，汉诺威王国被[[普魯士|普魯士]]吞并。[[格奥尔格五世_(漢諾威)|格奥尔格五世]]的孙子[[恩斯特·奧古斯特三世_(漢諾威)|恩斯特·奧古斯特]]曾在1913年至1918年间任不伦瑞克公国的公爵。

== 汉诺威王位继承人 ==
*1866年-1878年：[[格奥尔格五世_(漢諾威)|格奥尔格五世]]
*1878年-1923年：[[恩斯特·奧古斯特二世_(漢諾威)|恩斯特·奧古斯特二世]]（Ernst August II）  <span style="font-size:smaller;">1884年至1885年成为挂名的[[不伦瑞克统治者列表|不伦瑞克公爵]]，1878年至1919年为英国坎伯蘭公爵</span>
*1923年-1953年：[[恩斯特·奧古斯特三世_(漢諾威)|恩斯特·奧古斯特三世]]（Ernst August III）  <span style="font-size:smaller;">1913年至1918年为不伦瑞克公爵</span>
*1953年-1987年：[[恩斯特·奧古斯特四世_(漢諾威)|恩斯特·奧古斯特四世]]（Ernst August IV）
*1987年至今：[[恩斯特·奧古斯特五世_(漢諾威)|恩斯特·奧古斯特五世]]（Ernst August V）
