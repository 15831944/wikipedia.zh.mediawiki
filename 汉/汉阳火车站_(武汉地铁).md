'''汉阳火车站'''是[[武汉地铁4号线|武汉地铁4号线]]一座车站<ref name="武汉61座地铁站名已定">{{cite web |url = http://news.cnhubei.com/xw/wuhan/201208/t2187330.shtml |title = 武汉地铁4号线实现全线动工 串起三大火车站 | accessdate=2012-03-23}}</ref>，位于[[武汉市|武汉市]][[汉阳区|汉阳区]]。

== 車站 ==
车站位于汉阳大道与五琴路交叉口的东北象限内，为地下两层岛式站台：地下一层站厅，地下二层站台。共设四个出入口<ref name="地铁官网">{{cite web |url = http://www.whrt.gov.cn/dtjs/shxeq/hyhcz/ |title = 武汉地铁4号线各站简介 |accessdate = 2012-03-25 |deadurl = yes |archiveurl = https://web.archive.org/web/20120913023416/http://www.whrt.gov.cn/dtjs/shxeq/hyhcz/ |archivedate = 2012-09-13 }}</ref>。本站虽名为“汉阳火车站”，可由于国铁[[汉阳站|汉阳站]]并不办理客运业务，所以无法像[[武汉火车站_(武汉地铁)|武汉火车站]]和[[武昌火车站_(武汉地铁)|武昌火车站]]那样为乘客提供国铁-地铁转乘服务。故本站只是一座普通的[[地铁站|地铁站]]。
