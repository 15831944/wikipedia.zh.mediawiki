'''汉剧'''（也称'''广东汉剧'''）是一种传统[[客家|客家]][[戏剧|戏剧]]，也是[[客家人|客家人]]喜闻乐见的艺术品种之一。该剧种主要流行于[[粤东|粤东]]、[[闽西|闽西]]、[[赣南|赣南]]等地部分[[客家地区|客家地区]]，演绎以[[中州|中州]]古调为准，发音近似于现在的[[普通话|普通话]]。

== 简介 ==
汉剧，原名“'''外江戏'''”或“兴梅汉戏”，又称“乱弹”，在[[晚清|晚清]]被诸多举人士绅崇为雅乐。

1933年，[[汕头|汕头]][[公益国乐社|公益国乐社]]出版社刊《公益社乐剧月刊》，主要介绍外江戏剧目、[[汉乐|汉乐]]曲调、[[潮州音乐|潮州音乐]]等相关资讯<ref>{{cite web|url=http://www.choifookkee.com.hk/library/chaoyuebainiandashiji.htm#1|title=潮樂百年大事記（第1至50條）|3=|author=鄭志偉|date=}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。作为刊物主编的[[大埔縣|埔籍]]学者[[钱热储|钱热储]]，在内文《汉剧提纲》中，提及此剧种似有创自[[汉口|汉口]]之可能，遂倡议并将其定名作“汉剧”<ref>{{cite web |url=http://www.chaoju.com/lianzai/jinxianzaicsxj/67.htm |title=第四章　外江戏（广东汉剧）(1) | |author=陈韩星 |date=}}</ref>。

1950年代末，漢劇又按分布地區不同，逐漸分成「廣東漢劇」和「[[閩西漢劇|閩西漢劇]]」兩大類，領奏的[[胡琴|胡琴]]（主弦）均為[[頭弦|頭弦]]，都屬於板式變化豐富的[[皮黃|皮黃]]劇種。

关于汉剧，其源流始祖说法不一，有称“来自[[徽班|徽班]]”<ref>{{cite book|editor=上海艺术研究所、中国戏剧家协会上海分会|title=《中国戏曲曲艺词典》|origyear=1981年9月|url=http://www.cnhakka.com/bbs/read-htm-tid-49957.html|edition=一版三印|date=1985年2月3日|publisher=上海辞书出版社|pages=第217页|deadurl=yes|archiveurl=https://web.archive.org/web/20140407073342/http://www.cnhakka.com/bbs/read-htm-tid-49957.html|archivedate=2014年4月7日}}</ref>、“源于[[湖北汉剧|湖北汉剧]]”<ref>{{cite web |url=http://www.huadu.gov.cn:8080/was40/detail?record=2&channelid=4374&searchword=%BE%E7%D6%D6%D4%B4%C1%F7 |title=广东汉剧不是源于徽班而是源于湖北汉剧| |author=丘煌、罗恒报 |date=}}</ref>、或是“秦戏班底”等等。但据现有资料记载和考证，当数与[[湖南|湖南]][[祁阳戏|祁阳戏]]（亦称“楚南戏”）关系密切。

== 臺灣客家大戲 ==
{{more|台灣傳統戲劇表演團體列表#客家採茶戲、客家大戲}}
在[[臺灣|臺灣]]，當地的客家人在各種民間[[廟會|廟會]]（如酬神會、[[中元節|中元節]]、甚至年底的收冬祭等等）裏，都會上演「客家大戲」。這種「大戲」常以「四縣客語」（[[客語|四縣腔]]）來發音演繹，不僅與[[中國大陸|中國大陸]]的傳統客家漢劇有明顯差異，連使用的語言和曲調都有所不同。

從演出題材和內容方面看，台灣的客家戲曲主要以「歷史故事」或「民間傳說」為背景，借以宣揚「忠、孝、節、義」四項為本、「勤儉」之大傳統為輔，對下一代進行道德教化，保持社會風氣之優良。
