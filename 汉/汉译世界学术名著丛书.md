'''汉译世界学术名著丛书'''是由[[商务印书馆|商务印书馆]]出版发行的一套丛书，截至2013年已出14辑600种。主要为[[马克思主义|马克思主义]]诞生以前<ref>[见该套丛书出版说明]</ref>世界各国的各种古典学术作品的中文译作，也有少量是现代或当代的外国学术作品。从1950年代起印行至今，包含五大类：[[哲学|哲学]]类，[[历史学|历史]]、[[地理学|地理]]类，[[政治学|政治]]、[[法律学|法律]]、[[社会学|社会]]类，[[经济学|经济]]类，[[语言学|语言]]类。除常规版本外，其中一些作品还有精装本和节选本。因收录数目及翻译质量頗佳，該丛书在中国颇受好评。

对于原有单行本的书籍，被编入汉译世界学术名著丛书后，一般仍用原纸型，并不重新排版，译文及序跋都一仍其旧，故这套丛书体例并不统一。某些书籍的字体字型与现今出版物有差异。丛书中包括[[古希腊|古希腊]]名著在内的各种学术名著大多是从原著直接汉译，也有部分版本参照各种语言译本翻译，如企鹅丛书等。

2009年，商务印书馆推出该丛书的“珍藏版”，共400种、490册，并于9月24日在北京召开出版座谈会<ref>{{cite web |url=http://theory.people.com.cn/GB/10122964.html |title=汉译世界学术名著丛书（珍藏版）出版座谈会举行 |accessdate=2009-10-22 |publisher=[[中国青年报|中国青年报]]|language=zh-hans }}</ref>。2011年，商务印书馆推出该丛书的“分科本”，分科本在2009年“珍藏本”的基础上又增补了从2009年到2011年新出版的第十一、十二辑书目，共500种，以软精装形式出版，并于11月19日在北京[[商务印书馆|商务印书馆]]礼堂召开“汉译世界学术名著丛书”分科本出版座谈会暨第十四辑专家论证会<ref>{{cite web |url=http://www.chinanews.com/cul/2011/11-19/3472497.shtml|title= 汉译世界学术名著丛书出分科本 2020年出2000种|date=2011-11-19|publisher=中国新闻网|language=zh-hans}}</ref><ref>{{cite web |url=http://book.sina.com.cn/news/c/2011-11-21/1044292918.shtml|title= 汉译名著丛书分科本出版座谈会在京召开|date=2011-11-21|publisher=新浪|language=zh-hans}}</ref>。
