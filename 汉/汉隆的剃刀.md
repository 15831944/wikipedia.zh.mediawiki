'''汉隆的剃刀'''（{{lang|en|Hanlon's razor}}）是一句[[哲学剃刀|哲学剃刀]][[格言|格言]]：{{cquote|能解释为愚蠢的，就不要解释为恶意<ref name="Wigosky-2004"/>。}}汉隆的剃刀可以作為[[經驗法則|經驗法則]]，用於排除一個現象的可能解釋選項。

== 起源 ==
1990年，{{tsl|en|Jargon File|Jargon File|黑客字典}}收录了汉隆的剃刀，将其注释为“[[墨菲定律|墨菲定律]]版的[[奥卡姆剃刀|奥卡姆剃刀]]”<ref>{{cite web| url = https://jargon-file.org/archive/jargon-2.1.1.dos.txt| title = The Jargon File, Version 2.1.1 (Draft)| editor1 = Guy L. Steele| editor2 = Eric S. Raymond| date = 1990-06-12| publisher = jargon-file.org| accessdate = 2017-07-19| archive-url = https://web.archive.org/web/20170731093549/http://jargon-file.org/archive/jargon-2.1.1.dos.txt| archive-date = 2017-07-31| dead-url = no}}</ref>，汉隆的剃刀自此闻名<ref name="Wigosky-2004">{{cite book| url = https://books.google.com/books?id=tmyz9S-hLm8C&pg=PA5| title = RAPID Value Management for the Business Cost of Ownership| author = Andrew S. Wigosky| publisher = Digital Press| year = 2004| page = 5| quote = [...] Hanlon's Razor: 'Never attribute to malice that which can be adequately explained by stupidity.' This definition comes from 'The Jargon File' (edited by Eric Raymond), but one poster attributes it to Robert Heinlein, in a 1941 story called 'Logic of Empire.'| isbn = 9781555582890}}</ref><ref>{{cite book| title=Il potere della stupidità| first=Giancarlo| last=Livraghi| publisher=Monti & Ambrosini SRL| location=Pescara, Italy| year=2004| page=1| isbn=9788889479131}}</ref><ref>{{cite web| url = http://www.catb.org/jargon/html/H/Hanlons-Razor.html| title = Hanlon's Razor| work = {{tsl|en|Jargon File|Jargon File|黑客字典}}| date = 2002-03-03| publisher = [[Eric_S._Raymond|Eric S. Raymond]]| accessdate = 2017-07-19| archive-url = https://web.archive.org/web/20170704093715/http://www.catb.org/jargon/html/H/Hanlons-Razor.html| archive-date = 2017-07-04| dead-url = no}}</ref>。同年，黑客字典的编者又在该[[词条|词条]]内注明该格言的起源资料缺失，且另有一句[[美国|美国]][[哲学家|哲学家]][[威廉·詹姆士|威廉·詹姆士]]的格言表达了相似的意思<ref name="Wigosky-2004"/><ref>{{cite web| url = https://jargon-file.org/archive/jargon-2.2.1.dos.txt| title = The Jargon File, Version 2.2.1| editor2 = Guy L. Steele| editor1 = Eric S. Raymond| date = 1990-12-15| publisher = jargon-file.org| accessdate = 2017-07-19| archive-url = https://web.archive.org/web/20160407071055/http://jargon-file.org/archive/jargon-2.2.1.dos.txt| archive-date = 2016-04-07| dead-url = no}}</ref>。1996年，黑客字典的同名词条再次更新，注明[[羅伯特·海萊因|羅伯特·海萊因]]在1941年發表的小說《{{tsl|en|Logic of Empire|帝國的邏輯}}》中也有一句类似的格言：{{cquote|你把愚蠢所導致的許多情況，都歸咎於邪惡<ref name="Wigosky-2004"/>。}}黑客字典的编者推测汉隆（{{lang|en|Hanlon}}）可能是海萊因（{{lang|en|Heinlein}}）的讹传<ref>{{cite web| url = https://jargon-file.org/archive/jargon-4.0.0.dos.txt| title = The Jargon File, Version 4.0.0| editor = Eric S. Raymond| date = 1996-07-24| publisher = jargon-file.org| accessdate = 2017-07-19| archive-url = https://web.archive.org/web/20160407071529/http://jargon-file.org/archive/jargon-4.0.0.dos.txt| archive-date = 2016-04-07| dead-url = no}}</ref>。

2001年，{{tsl|en|Quentin Stafford-Fraser|昆汀·斯塔福德·弗雷澤}}在[[博客|博客]]上发表了两篇文章，引述了来自約瑟夫·E·比格勒（{{lang|en|Joseph E. Bigler}}）的邮件，阐述了汉隆的剃刀的起源<ref>{{cite web| url = https://statusq.org/archives/2001/11/26/| title = [untitled]| first = Quentin| last = Stafford-Fraser| date = 2001-11-26| accessdate = 2017-07-19| archive-url = https://web.archive.org/web/20170718001115/https://statusq.org/archives/2001/11/26/| archive-date = 2017-07-18| dead-url = no}}</ref><ref>{{cite web| url = https://statusq.org/archives/2001/12/04/| title = The origins of Hanlon’s Razor| first = Quentin| last = Stafford-Fraser| date = 2001-12-04| accessdate = 2017-07-19| archive-url = https://web.archive.org/web/20160314062939/http://statusq.org/archives/2001/12/04/| archive-date = 2016-03-14| dead-url = no}}</ref>。文中写道，《墨菲定律》（{{lang|en|''Murphy's Law, and Other Reasons Why Things Go WRONG!''}}）一书的作者{{tsl|en|Arthur Bloch|亞瑟·布洛赫}}曾向公众征集自创“[[定律|定律]]”的投稿，而汉隆的剃刀正是約瑟夫的好友，来自[[賓夕法尼亞州|賓夕法尼亞州]][[斯克蘭頓_(賓夕法尼亞州)|斯克蘭頓]]的罗伯特·J·汉隆（{{lang|en|Robert J. Hanlon}}）的作品，被收录在《墨菲定律第二册》（{{lang|en|''Murphy's Law Book Two: More Reasons Why Things Go Wrong!''}}）中<ref>{{cite book| title = Murphy's Law Book Two: More Reasons Why Things Go Wrong!| publisher = Price Stern Sloan| author = Arthur Bloch| year = 1980| page = 52| isbn = 9780417064505}}</ref>。

==类似格言==
[[約翰·沃爾夫岡·馮·歌德|歌德]]在《[[少年維特的煩惱|少年維特的煩惱]]》中著有意思相似的一句話：{{cquote|誤解和忽視在世上所造成的混亂，比惡意和欺騙要多上許多。無論如何，後兩者出現的頻率都低得多<ref>{{cite book |last1=introduction |first1=Johann Wolfgang von Goethe ; translated with an |last2=Constantine |first2=notes by David |title=The sorrows of young Werther |date=2012 |publisher=Oxford University Press |location=Oxford |isbn=978-0199583027}}</ref>。}}

[[英国|英国]]前首席{{tsl|en|press secretary|新闻秘书}}{{tsl|en|Bernard Ingham|伯纳德·英厄姆}}爵士同样說過一段类似的話：{{cquote|許多記者沉醉在政府陰謀論中。我可以擔保，如果他們支持的是“政府搞砸論”，報導就会更准确一些<ref>{{cite news |url = http://books.google.co.uk/books?id=SoyalAxDItYC&pg=PA17&lpg=PA17#v=onepage&q&f=false |title = Conspiracy Theories: The Philosophical Debate (Chapter 3 - Popper Revised, or What is Wrong with Conspiracy Theories?) |editor = David Coady |first = Charles |last = Pigden |year = 2006 |page = 17}}</ref>。}}
