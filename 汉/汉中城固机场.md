'''汉中城固机场'''（{{airport codes|HZG|ZLHZ}}），又称'''柳林机场'''，是[[中华人民共和国|中华人民共和国]]一座军民合用[[机场|机场]]，位於[[陕西省|陕西省]][[汉中市|汉中市]][[城固县|城固县]][[柳林镇_(城固县)|柳林镇]]，距离汉中市区18公里。<ref name="xa">{{cite web |url=http://news.xiancn.com/content/2013-09/09/content_2799116.htm |title=汉中城固机场明年通航 将开通至北京、重庆等航线 |publisher=西安新闻网 |date=2013-09-09 |author=张松}}</ref>

==历史沿革==
* 1944年秋，[[中华民国|中华民国]][[國防部空軍司令部|航空委员会]]三十九工程处处长[[祝巍|祝巍]]率领技术人员100余人，到城固县柳林镇[[五渠寺|五渠寺]]组织勘设修建飞机场。汉中[[行政督察专员公署|专员公署]]调动城固、[[南郑|南郑]]两县各300名[[民工|民工]]开始修建小型民用机场。
* 1945年春，专员公署抽调全区7县民工8万多人将其续建扩建为军用机场。机场占地1,000多[[亩|亩]]，完成土方692,300[[立方米|立方米]]，石方297,600立方米。
* 1945年8月，[[抗日战争|抗日战争]]胜利，机场停止扩建，建有一条环形跑道及[[停机坪|停机坪]]等，但未曾使用。
* 1950年，机场被辟为新汉[[劳改|劳改]][[农场|农场]]。
* 1966年，[[中华人民共和国航空工业部|中华人民共和国航空工业部]]所属[[劲松机械厂|劲松机械厂]]在五渠寺筹建，设立国产[[运输机|运输机]]总装试飞区，同时，对原五渠寺机场进行了改建。
* 1967年，[[中国人民解放军空军|中国人民解放军空军]]某部进驻机场附近。
* 1971年，机场改建工程完成，称“柳林机场”，由解放军空军某部管理使用。
* 1979年2月，劲松机械厂、[[彤辉机械厂|彤辉机械厂]]和[[012基地|012基地]]设计所合并成立[[陕西飞机制造公司|陕西飞机制造公司]]，柳林机场同时成为陕西飞机制造公司的试飞机场，设有[[试飞站|试飞站]]等机构。

==军民合用改扩建==
===背景===
由于汉中社会经济的快速发展，原有[[汉中西关机场|汉中西关机场]]的规模、设施、环境已不能满足日益增长的社会需求；同时，机场距离市区太近，飞行安全与城市发展造成相互制约。从1998年起，汉中市[[政府|政府]]就已开始规划新的机场建设项目；建设方案历经迁建、改建、[[军民合用|军民合用]]的多次调整，于2007年6月，最终确定了对柳林机场进行军民合用改扩建的方案，建设新的汉中机场。<ref name="xb">{{cite web |url=http://news.cnwest.com/content/2008-12/01/content_1600648.htm |title=汉中与西部机场集团签订机场建设合作协议 |publisher=西部网 |date=2008-12-01 |author=}}</ref>

===进展===
* 2008年11月29日，汉中市与[[西部机场集团|西部机场集团]]正式签约，合作建设运营汉中城固军民合用机场项目。<ref name="xb"/>
* 2011年8月31日，[[中华人民共和国国务院|国务院]]、[[中央军委|中央军委]]批复同意空军城固机场实行军民合用并进行扩建。<ref>{{cite web|url=http://www.hanzhong.gov.cn/xwzx/zwyw/201410/t20141024_75474.html|title=汉中城固机场军民合用项目获批立项|publisher=汉中市人民政府|date=2011-09-08|author=}}</ref>
* 2012年4月18日－19日，受[[国家发改委|国家发改委]]的委托，[[中国国际工程咨询公司|中国国际工程咨询公司]]组成专家组，在北京主持召开了汉中城固机场军民合用项目[[可行性研究|可行性研究]]报告评估会，通过了项目可研报告专家审查。<ref>{{cite web |title=汉中城固机场军民合用项目可研报告通过专家审查 |url=http://www.hanzhong.gov.cn/xwzx/zwyw/201410/t20141024_76037.html |publisher=汉中市人民政府 |date=2012-04-23}}</ref>
* 2012年8月24日，汉中城固机场军民合用改扩建工程正式奠基。汉中[[市委书记|市委书记]][[张会民|张会民]]宣布奠基令。<ref name="dj">{{cite web |author= |title=汉中城固机场军民合用改扩建工程隆重奠基 |url=http://cwag-xaghs.com/newsDetail.aspx?id=452&classID=98 |publisher=西部机场集团航空地勤（西安）有限公司 |date=2012-11-07}}</ref>
* 2013年2月27日，汉中城固机场军民合用改扩建项目军航及军民合用工程开工仪式正式举行，根据军队相关规定，这部分工程由市政府委托[[兰州军区空军|兰空]]后勤部管理、空军第四空防工程处负责承建。<ref>{{cite web |title=汉中新机场军民合用工程开工建设 |url=http://www.hanzhong.gov.cn/xwzx/zwyw/201410/t20141024_76703.html |publisher=汉中市人民政府 |date=2013-02-28}}</ref>
* 2013年7月15日，国家发改委、[[总参谋部|总参谋部]]对汉中城固机场军民合用项目可行性研究报告联合作出批复，同意实施城固机场军民合用改扩建工程。<ref name="ky">{{cite web |title=国家发改委、总参谋部批复同意汉中城固机场军民合用项目可研报告 |url=http://www.hanzhong.gov.cn/xwzx/zwyw/201410/t20141024_76945.html |publisher=汉中市人民政府 |date=2013-07-23}}</ref>
* 2013年10月18日，机场跑道主体工程顺利完工。<ref>{{cite web |title=新机场跑道主体工程完工 |url=http://www.hanzhong.gov.cn/xwzx/zwyw/201410/t20141024_77080.html |publisher=汉中市人民政府 |date=2013-10-17}}</ref>
* 2014年4月7日－13日，[[中国民航|中国民航]]飞行校验中心对汉中城固机场进行了校飞。4月13日12时15分，随着一架{{le|奖状C680|Cessna Citation Sovereign}}型飞机平稳降落，为期一周的校飞工作结束，共起降9架次、飞行近30小时。
* 2014年6月10日9时12分，[[中国东方航空|中国东方航空]]一架[[空客A320|空客A320]]客机降落在汉中城固机场，开始对机场进行试飞，经过1小时40分钟的测试飞行，对机场的[[导航|导航]]设施、助航灯光、通信设备、飞行程序、净空环境、保障能力等进行了全面验证。结果显示，各项参数和指标正常，机场相关设施满足通航条件。<ref>{{cite web|url=http://news.dichan.sina.com.cn/2014/06/23/1138216_all.html|title=汉中城固机场试飞成功|publisher=中国民用航空网|date=2014-06-23|author=}}</ref>
* 2014年8月13日，汉中城固机场正式通航。首航由中国东方航空MU2217航班执飞，机型为空客A320，共有128名乘客乘坐首航班机从[[西安咸阳国际机场|西安咸阳国际机场]]飞抵汉中。民航陕西监管局向汉中机场颁发了机场使用许可证。<ref name="th">{{cite web|url=http://www.chenggu.gov.cn/news/cgyw/34668.htm |title=汉中城固机场正式通航 |publisher=城固县人民政府 |date=2014-08-14 |author=房辰}}</ref>

===规模===
汉中城固机场军民合用改扩建工程，新征土地1,315.5亩，总投资5.63亿元，其中国家发改委安排中央预算内投资1亿元，国家民航局安排民航发展基金1.8亿元，其余投资由市政府自筹解决<ref name="ky" />，建设工期两年。近期主要新建一条长2,500米、宽45米的军民合用[[跑道|跑道]]，新建29,000平方米的[[机坪|站坪]]，民航站坪机位5个（2B3C）；一条长303米、宽18米的民航专用联络[[滑行道|滑行道]]，[[飞行区等级|飞行区等级]]4C；新建民航助航灯光系统；新建600平方米的[[塔台|航管楼]]、990平方米的[[消防|消防]]用房以及300平方米的货运站；新建民航[[航站楼|航站楼]]5,500平方米，设有6个[[值机|值机]]柜台、1个超大行李值机柜台、2条[[安检|安检]]通道、2条[[行李传送带|行李传送带]]，以及其他公用配套设施；同时还建设和完善部分军航设施。远期规划建设平行滑行道、联络滑行道和垂直联络道，新建站坪机位9个（2B7C）。机场建成后，可供[[波音737|波音737]]、[[空客A320|空客A320]]及其他[[支线飞机|支线飞机]]起落，近距离航线可直飞[[西安|西安]]、[[成都|成都]]、[[重庆|重庆]]、[[武汉|武汉]]，远距离航线可直飞[[北京|北京]]、[[上海|上海]]、[[广州|广州]]，预计到2020年每年旅客流量30万人次，货邮[[吞吐量|吞吐量]]1,300吨、飞机起降量3,300架次。<ref name="xa"/><ref name="dj" /><ref name="th" />

==机场交通==
现已开通汉中高速客运站至城固机场的直达专线[[巴士|巴士]]，同时为[[出租车|出租车]]提供免费停车服务。<ref name="th" />

==未来规划==
2017年8月25日，汉中市人民政府办公室发布了关于成立汉中城固机场扩建项目领导小组的通知，将展开机场扩建项目的前期工作。<ref>{{cite web |title=汉中市人民政府办公室关于成立汉中城固机场扩建项目领导小组的通知 |url=http://www.hanzhong.gov.cn/xxgk/gkml/zfwj/zfbwj/hzfwj_13928/201709/t20170912_432700.html |publisher=汉中市人民政府 |date=2017-09-12}}</ref>
