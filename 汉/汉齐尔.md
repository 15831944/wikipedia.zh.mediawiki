'''汉齐尔'''（{{lang-en|Khanzir}}；{{lang-ar|خنزير}}；{{lang-fa|خوک}}）在[[阿拉伯语|阿拉伯语]]及[[普什图语|普什图语]]<ref name=washington/>中表“[[猪属|猪]]”之意，同时也是在[[阿富汗|阿富汗]]{{tsl|en|Kabul Zoo||喀布尔动物园}}中展览的一头从[[中华人民共和国|中国]]进口的猪的名字。[[伊斯兰教|伊斯兰教]]《[[可蘭經|可蘭經]]》嚴禁食用[[豬肉|豬肉]]，故[[穆斯林|穆斯林]]占主导地位的阿富汗，售卖[[猪肉|猪肉]]的行为被严令禁止，这使得汉齐尔作为阿富汗境内唯一的一头猪而闻名世界<ref>{{cite web |url=http://news.china.com/socialgd/10000169/20161106/23855006_all.html |title=它是阿富汗全国唯一的一头猪 还是中国送的 |date=2016-11-06 |accessdate=2018-01-15}}
</ref><ref>{{cite web |url=http://www.tanews.org.tw/info/11583 |title=牠，阿富汗境內唯一的豬….. |date=2016-11-13 |accessdate=2018-01-15}}</ref>。

2002年，汉齐尔从中国被送往[[喀布尔|喀布尔]]动物园<ref name="BBC">{{cite web |url=http://news.bbc.co.uk/2/hi/south_asia/8038368.stm |title=Quarantine for lonely Afghan pig |date=2009-05-07 |publisher=[[英國廣播公司新聞|BBC News]] |accessdate=2012-05-01}}</ref>。起初它与另一头猪结伴而来，並生下一群小猪，但由于园方管理员疏忽，同是中国赠送的熊进入饲养栏内杀死所有小猪。待园方赶到时母猪已经重伤，不久后死去<ref>{{cite web |url=https://www.theguardian.com/world/2009/may/06/kabul-pig-quarantined |title=Life goes from bad to worse for Kabul's only pig |first=Peter |last=Walker |date=2009-05-06 |work=[[衛報|The Guardian]] |accessdate=2012-05-01}}</ref><ref>{{cite web |url=http://news.xinhuanet.com/english/2009-05/07/content_11329139.htm |title=Afghanistan's lone pig quarantined over swine scare |date=2009-05-07 |publisher=Xinhua News |accessdate=2012-05-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20121024064512/http://news.xinhuanet.com/english/2009-05/07/content_11329139.htm |archivedate=2012年10月24日 |df= }}</ref><ref>{{cite web |url=http://blogs.telegraph.co.uk/news/emmahartley/9729687/Afghanistan_only_has_one_pig/ |title=Afghanistan only has one pig |first=Emma |last=Hartley |date=2009-05-07 |work=[[每日电讯报|The Daily Telegraph]] |accessdate=2012-05-01}}</ref><ref name=washington>{{cite web |url=https://www.washingtonpost.com/news/animalia/wp/2016/11/03/there-is-only-one-pig-in-afghanistan-his-name-is-khanzir/?utm_term=.0a33f7659f5e |title=There is only one pig in Afghanistan. His name is Khanzir |first=Kumar |last=Ruchi |date=2016-11-03 |work=[[华盛顿邮报|The Washington Post]] |accessdate=2018-01-15}}</ref>。

2009年5月，汉齐尔接受隔离[[检疫|检疫]]，并被关入{{link-en|独居禁闭室|Solitary confinement}}。其“阿富汗唯一一头猪”之特殊地位一度吸引了国际关注<ref>{{cite web |url=https://www.reuters.com/article/2009/05/05/us-afghanistan-pig-idUSTRE5444XQ20090505 |title=Afghanistan's only pig quarantined in flu fear |first=Golnar |last=Motevalli |date=2009-05-05 |publisher=[[路透社|Reuters]] |accessdate=2012-05-01}}</ref>。由于彼时正值[[甲型流感病毒H1N1亞型|猪流感]][[2009年H1N1流感大流行|大爆发]]，此举可能是因为怀疑其有传播猪流感风险的参观者不在少数。对此，动物园负责人阿齐兹·古勒·萨吉布（Aziz Gul Saqib）称汉齐尔实际上“身强体壮”，并表示“隔离仅是因为阿富汗国民在猪流感方面知识匮乏，故担心他们见到猪会担心自己患病”<ref name="BBC"/><ref>{{cite web |url=http://www.irishtimes.com/newspaper/breaking/2009/0507/breaking73.html |title=Lonely Afghan pig seeks similar |date=2009-05-05 |work={{tsl|en|The Irish Times||The Irish Times}} |accessdate=2012-05-01}}</ref>。两个月后，汉齐尔被释放<ref>{{cite web |url=https://www.reuters.com/article/2009/07/08/us-flu-afghanistan-idUSTRE5672NW20090708 |title=Afghanistan says 14 H1N1 cases on U.S. military base |date=2009-07-08 |publisher=[[路透社|Reuters]] |accessdate=2012-05-01}}</ref>。
萨吉布称动物园希望为汉齐尔觅得一个雌性伴侣，但他同时也承认了此举“在此时非常危险且难以执行”<ref>{{cite web |url=http://www.irishtimes.com/newspaper/breaking/2009/0507/breaking73.html |title=Lonely Afghan pig seeks similar |date=2009-05-05 |work={{tsl|en|The Irish Times||The Irish Times}} |accessdate=2012-05-01}}</ref>。
