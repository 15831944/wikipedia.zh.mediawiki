'''汉江'''，亦称'''汉水'''，又名'''襄河'''，古称'''{{Ruby|沔|-{zh-hans:{{lang|pinyin|miǎn}};zh-hant:{{lang|zh-tw|ㄇ丨ㄢˇ}};zh-hk:{{lang|pinyin|min5}};}-}}水'''，是[[長江|長江]]最长的[[支流|支流]]。汉水位处[[长江中游|长江中游]]左岸，發源於[[中国|中国]][[陝西省|陝西省]][[秦岭|秦岭]]南麓的[[沮水_(汉水)|沮水]]，[[干流|干流]]自西向東流经[[陕南|陕南]]和[[湖北省|鄂]]西北，于[[武汉|武汉]][[汉口|汉口]]注入長江。长1577公里，流域面积15.9万平方公里。<ref name=HHDD>{{cite encyclopedia |year=2010 |title =汉江 |encyclopedia=中国河湖大典·长江卷 |publisher=中国水利水电出版社 |location=北京 |ISBN = 978-7-5084-6512-8}}</ref>

[[汉族|汉族]]的族称追根溯源即来自汉水。[[秦惠文王|秦惠文王]]置[[汉中郡|汉中郡]]，名字取自汉水。<ref>“漢中郡……秦惠文王置郡。因水名也。”{{cite web |url = http://ctext.org/wiki.pl?if=gb&chapter=401191 |title=華陽國志 漢中志 一}}</ref>后[[刘邦|刘邦]]受[[项羽|项羽]]封于巴、蜀、汉中，都于汉中郡[[南郑|南郑]]，因称汉王，统一天下后亦以“汉”为[[国号|国号]]。<ref>“更立沛公為漢王，王巴、蜀、漢中，都南鄭。”{{cite web |url = http://ctext.org/shiji/gao-zu-ben-ji/zh |title=史記 高祖本紀}}</ref><ref>“漢中郡……縣十二：……南鄭……”{{cite web |url = http://ctext.org/han-shu/di-li-zhi/zh |title = 漢書 地理志}}</ref>[[汉朝|汉朝]]前后400余年，[[经济|经济]]、[[文化|文化]]和[[疆域|疆域]]都有了大发展，原称[[华夏|华夏]]的[[中原|中原]]居民称为汉人，汉人从此成为[[中国|中国]]主体民族的通称。<ref>{{cite encyclopedia |year=2009 |title = 汉族 |encyclopedia=中国大百科全书 |publisher=中国大百科全书出版社 |location=北京 |edition=第二版 |ISBN = 978-7-5000-7958-3}}</ref>

== 名称 ==
'''汉江'''、'''汉水'''之名的源头可以追溯到[[先秦|先秦]]时代。《[[尚书_(书)|書]]·[[禹貢|禹貢]]》中就有“嶓冢<sup>（嶓冢山）</sup>導漾<sup>（[[漾水河|漾水河]]）</sup>，東流為'''漢'''”<ref name="YG">{{cite web |url = http://ctext.org/shang-shu/tribute-of-yu/zh |title = 尚書 禹貢}}</ref>。《[[诗经|詩]]·小雅·四月》中有“滔滔江'''漢'''，南國之紀”<ref>{{cite web |url = http://ctext.org/book-of-poetry/si-yue/zh |title=詩經 小雅 四月}}</ref>，将汉江与长江并称为南方的两条大河。

'''沔水'''之称，亦可追溯至《書·禹貢》：“浮于潛，逾于'''沔'''”<ref name="YG" />。《[[史记|史记]]》记载：“於是自[[殽|殽]]以東……大川祠二……水曰[[济水|濟]]，曰[[淮河|淮]]。……自[[华山|華]]以西……名川四……水曰[[黄河|河]]……'''沔'''……湫淵……[[长江|江水]]。”<ref>{{cite web |url = http://ctext.org/shiji/feng-chan-shu/zh |title = 史記 封禪書}}</ref>《[[水经|水经]]》中只称“沔水”，不称“汉”；[[北魏|北魏]][[郦道元|郦道元]]《[[水经注|水经注]]》中两种称呼都有使用，“互相通稱”<ref>{{cite web |url = http://ctext.org/shui-jing-zhu/27/zh |title = 水經注 沔水}}</ref>。但也有把沔水看成是汉水的上游段的，如[[东晋|东晋]][[梅赜|梅赜]]所献尚书伪[[孔安国|孔]]传称“漢上曰沔”<ref>{{cite web |url = http://ctext.org/library.pl?if=gb&file=80131&page=91 |title=尚書正義 禹貢 影印本}}</ref>、“東南流為沔水，至[[漢中|漢中]]東流為漢水”<ref>{{cite web |url = http://ctext.org/library.pl?if=gb&file=80131&page=107 |title = 尚書正義 禹貢 影印本}}</ref>。

'''襄江'''、'''襄河'''，特指汉江[[襄阳|襄阳]]段或襄阳以下。《[[明史|明史]]》记载：“襄陽[[附郭县|倚]]……漢水在城北，亦曰襄江。”<ref>{{cite web |url = http://ctext.org/wiki.pl?if=gb&chapter=340963 |title=明史 地理志}}</ref>《[[讀史方輿紀要|讀史方輿紀要]]》记载：“襄河者，漢水自襄陽來也。”<ref>{{cite web |url = http://ctext.org/wiki.pl?if=gb&res=569304 |title=讀史方輿紀要 湖廣方輿紀要}}</ref>

== 流域范围 ==
汉江[[流域|流域]]之北为[[黄河|黄河]]流域，以秦岭、[[外方山|外方山]]、[[伏牛山|伏牛山]]为[[分水岭|分水岭]]；东北为[[淮河|淮河]]流域，以伏牛山、[[桐柏山|桐柏山]]为分水岭；西南隔[[大巴山|大巴山]]、[[荆山|荆山]]与[[嘉陵江|嘉陵江]]、[[沮漳河|沮漳河]]为邻；东南为[[江汉平原|江汉平原]]，与长江无明显的天然分水岭。

[[流域面积|流域面积]]达15.9万[[平方公里|平方公里]]，从西北至东南长约820[[公里|公里]]，涉及陕西、湖北、[[甘肃|甘肃]]、[[四川|四川]]、[[重庆|重庆]]、[[河南|河南]]等6省的21[[地级行政区|地]]、79[[县级行政区|县]]。流域内总人口达3955万人（2006年），其中湖北、陕西、河南分别占53.76%、21.14%和24.53%。<ref name="HHDD" />

== 水系特征 ==
[[File:Dan_river_g.png|thumb]]]]

干流全长1577公里，总落差1964米。上中游以[[丹江口|丹江口]]为界，中下游以[[钟祥|钟祥]]为界。上游长约925公里，流域面积约9.52万平方公里，[[盆地|盆地]]与[[峡谷|峡谷]]相间，河深水急；中游长约270公里，流域面积约4.68万平方公里，流经[[丘陵|丘陵]][[河谷|河谷]]盆地，河床不稳定，时冲时淤；下游长约382公里，流域面积约1.7万平方公里，多曲流、[[洲滩|洲滩]]，两岸筑有[[堤防|堤防]]。襄阳以下，河槽上宽下窄，河道逐渐收拢，再加上可能的长江洪水[[顶托|顶托]]，遇大水时水流难以宣泄，深受[[洪灾|洪灾]]威胁。

流域水系呈[[水系#叶脉状|叶脉状]]，[[支流|支流]]大多较短小，左右岸分布不均衡。流域面积在0.5万-1.0万平方公里的支流有[[旬河|旬河]]、[[夹河|夹河]]、唐河（[[唐白河|唐白河]]支流）、[[南河_(漢水支流)|南河]]、[[汉北河|汉北河]]；流域面积在1.0万平方公里以上的支流有[[堵河|堵河]]、[[丹江|丹江]]、唐白河。其中，唐白河流域面积居首，丹江长度第一，堵河水量最大。<ref name="HHDD" />

== 水文要素 ==
=== 降水、蒸发===
[[降水|降水]]以降[[雨|雨]]、降[[雪|雪]]为主，鲜有[[冰雹|冰雹]]、积雪。流域内年均[[降水量|降水量]]约873[[毫米|毫米]]，下游地区可达1100毫米以上，上中游地区约700-900毫米。时间分配不均匀，连续最大4个月降水（下游约为4-7月，上游约为7-10月）占年降水的55%-65%。

年均水面[[蒸发量|蒸发量]]约为700-1100毫米，空间上西少东多，时间上1或12月最少。<ref name="HHDD" />

=== 径流 ===
流域内1956-1998年平均年[[径流量|径流量]]为566亿[[立方米|立方米]]。年际变化大，如皇庄[[水文站|水文站]]<sup>钟祥</sup>最枯年径流量（182亿立方米，1999年）约为最丰年径流量（1060亿立方米，1964年）的六分之一。年内变化大，主汛期（7-10月）径流量占年径流量的65%，约为枯水期（11月-次年6月）的2倍；最大月径流量约为最小月的10倍。<ref name="HHDD" />

=== 泥沙 ===
[[File:Wuhan_114.27328E_30.58114N.jpg|thumb]]清浊分明。]]

汉江本是长江中游的主要产沙区，为湖北省内[[输沙模数|输沙模数]]最大的地区，干流年均含沙量为0.5-2.0[[千克|千克]]每立方米。7、8、9月的来沙量占全年的80%以上。[[丹江口水库|丹江口水库]]建成后，泥沙淤积库内不出，中下游沙情大幅改善，黄家港水文站<sup>丹江口</sup>年均输沙量减至建库前的1.2%，皇庄年均输沙量减至建库前的16.5%。<ref name="HHDD" />

=== 水质 ===
汉江干流[[水质|水质]]总体较好：数据表明，水质达[[水质#水质分类|Ⅱ类]]以上河段占评价河长的78%；Ⅲ类河段占13.82%，多在中下游地区；Ⅳ类河段占7.78%，主要在上游的[[汉中|汉中]]、[[安康|安康]]地区。支流中，丹江水质较好，所有测站全年水质均达到Ⅱ类，唐白河水质较差。<ref name="HHDD" />

== 地形地势 ==
汉江流域地势西高东低。西部为[[秦岭|秦]][[大巴山|巴]]山地，[[海拔|海拔]]1000-3000米；中部为[[南襄盆地|南襄盆地]]，海拔100-300米；东部为江汉平原，平均海拔20-40米。干流总落差1964米，河口处海拔18米。流域内[[山地|山地]]占55%，主要是西部的中低山区；丘陵占21%，多分布于南襄盆地和江汉平原的边缘地带；平原占23%，主要为南襄盆地、江汉平原和汉江河谷；湖泊1%，基本分布于江汉平原。<ref name=HHDD/>

== 自然灾害 ==
=== 水灾 ===
汉江流域早在公元前就有关于[[洪水|洪水]]的记载。平均约一百年就可能发生一次特大洪水。1416年以来，特大洪水的记述更多，1583年至今发生了7次特大洪水。1822年-1949年，汉江干流堤防发生溃决的有76年，平均2年不到溃决一次。

1583年（[[万历|万历]]十一年），汉江满溢，覆没[[金州_(陕西)|金州]]城。

1832年、1852年、1867年、1921年、1935年，汉江均发生了特大洪灾。

1954年，长江流域发生[[1954年长江洪水|特大洪水]]，汉江洪水与长江洪水遭遇，[[潜江市|潜江市]]汉江右岸堤坝溃决，造成45.3万人受灾。

1983年，汉江全流域大暴雨，安康市城区遭受[[1983年安康水灾|灭顶之灾]]，直接损失达8亿元；下游局部地区受灾，受灾人口接近9万。
 
1990年，汉江上游发生百年一遇特大洪水，汉中平川段100万人受灾。<ref name="HHDD" />

=== 旱灾 ===
汉江流域的[[旱灾|旱灾]]多为轻旱。上游多夏旱，[[商洛|商洛]]、安康、汉中等地区平均受旱率在20%上下；中下游多[[三伏|伏]]旱，襄阳为重旱灾害多发区，极端干旱机遇最大的则数[[十堰|十堰]]。<ref name="HHDD" />

=== 水土流失 ===
汉江全流域都存在不同程度的[[水土流失|水土流失]]，水土流失面积约为6.26万平方公里，其中以汉江干流沿岸、丹江中上游和汉江源头区最为严重。<ref name="HHDD" />

=== 地震 ===
汉江流域大部分处于弱[[地震|震]]区，很少发生较强地震。只有汉中-安康一个中强[[地震带|地震带]]；788年（[[贞元_(唐朝)|贞元]]四年）安康曾发生[[黎克特制地震震級|6.5级]]大地震。

丹江口水库1967年蓄水后，产生[[诱发地震|诱发地震]]，十几年内大小地震500余次。最大地震发生在1973年，震级4.7级。1980年后，库区地震数量减少，强度降低，趋于平稳。<ref name=HHDD/>

== 水能开发 ==
汉江流域[[水力资源|水力资源]]全流域理论总蕴藏量1083.16万[[千瓦|千瓦]]，其中技术可开发量装机容量约820万千瓦，经济可开发量装机容量约780万千瓦。水力资源集中于上游，占全流域的60%以上；干流占水力资源总量的47.1%，支流52.9%；中小型电站占62.4%，大型电站占37.6%。

汉江全流域已经或在建水电站168座，总装机容量367.49万千瓦，年发电量126.86亿[[千瓦时|千瓦时]]。干流现已建造[[石泉水电站|石泉]]、[[喜河水电站|喜河]]、[[安康水电站|安康]]、[[丹江口水电站|丹江口]]、[[王甫洲水电站|王甫洲]]、[[蜀河水电站|蜀河]]、[[崔家营水电站|崔家营]]、[[兴隆水利枢纽|兴隆]]等8座水电站。<ref name="HHDD" />

== 航运情况 ==
[[File:0275-Wuhan-Hanjiang.jpg|thumb]]航運。]]
汉江干流从汉中[[洋县|洋县]]开始可以通航，通航里程1342公里。

各河段[[中国内河航道等级|航道等级]]：[[洋县|洋县]]—黄瓜架，等级以下；黄瓜架—安康水库，Ⅳ级；安康水库-[[白河县|白河]]，Ⅵ级；白河—[[神定河|神定河]]<sup>[[郧阳区|郧阳区]]</sup>，Ⅶ级；神定河—丹江口，Ⅴ级；丹江口—皇庄，Ⅳ级；干流下游段，全年Ⅴ级，中洪水期Ⅳ级；河口段（[[蔡甸|蔡甸]]—河口）全年Ⅳ级，中洪水期Ⅲ级。<ref name="HHDD" />

== 各段分述 ==
=== 源头 ===
汉江有北源[[沮水_(汉水)|沮水]]、中源漾水河、南源[[玉带河|玉带河]]三个源头，均位于陕西省[[汉中市|汉中市]]内。

中源漾水河发源于[[宁强县|宁强县]]东北部，流程较短。南源玉带河发源于宁强县南部，长101公里。北源沮水发源于[[留坝县|留坝县]]与[[宝鸡市|宝鸡市]][[凤县|凤县]]交界的[[紫柏山|紫柏山]]南麓的黄花坪沟，向西南流，穿过[[勉县|勉县]]后，在[[略阳县|略阳县]]内的[[黑河坝乡|黑河坝乡]]与白河汇合，然后又向东南流回勉县境内；以白河[[河口|口]]为界，上游称黑河，长112公里，下游称沮水，长18公里。

在勉县，漾水河与玉带河先在[[新铺镇|新铺镇]]炭厂寺汇合，随后沮水在[[土关铺乡|土关铺乡]]沮水村汇入，三源汇合后始称汉江，已颇具大河规模，自西向东流至[[武侯镇|武侯镇]]后进入[[汉中盆地|汉中盆地]]。<ref name="HHDD" />

==== 正源 ====
中源漾水河长度、流域面积、流量均较小，却在传统上被视为正源。《書·禹貢》：“嶓冢導漾，東流為漢。”<ref name=YG/>学者认为，在《尚書》时代之后，古汉水上游发生了重大的水系变迁，[[西汉|西汉]]初[[武都|武都]]地区的大地震，最终导致原为古汉水上游的[[西汉水|西汉水]]被[[袭夺|袭夺]]，成了嘉陵江上游；这也可以解释现在的漾水河为何会拥有跟河流大小不相符的宽广[[河谷|河谷]]和普遍[[沉积|沉积]]现象。<ref name=SYC>{{cite journal zh |author=沈玉昌|year=1956|title=汉水河谷的地貌及其发育史|journal=地理学报|issue=4|pages=295-323}}</ref><ref>{{cite journal zh |author=周宏伟|year=2010 |title=汉初武都大地震与汉水上游的水系变迁 |journal=历史研究 |issue=4 |pages=49-69}}</ref>

今汉江的正源，《中国河湖大典》根据“河源唯远”原则，定为北源沮水。<ref name=HHDD/>也有学者根据河谷的一贯性，排除出自逼仄峡谷的南北两源，定正源为河谷宽广的中源漾水河。<ref name="SYC" />也有工具书定正源为南源玉带河，如《[[中国大百科全书|中国大百科全书]]》认为汉江源自[[米仓山|米仓山]]西麓。<ref>{{cite encyclopedia |year=2009|title = 汉江 |encyclopedia=中国大百科全书 |publisher=中国大百科全书出版社 |location=北京 |edition=第二版 |ISBN = 978-7-5000-7958-3}}</ref>

=== 上游 ===
汉江流出河源区后，从勉县武侯镇开始，穿越[[南郑县|南郑县]]、[[汉中|汉中]]市区、[[城固县|城固县]]，至洋县[[龙亭镇|龙亭镇]]大龙河口才又进入峡谷。该段河长约105公里，宽300-400米（洪水期1000-2000米），流域面积万余平方公里，中间即为[[汉中盆地|汉中盆地]]。该河段内，穿过盆地进入汉江的支流，北有[[褒河|褒河]]、[[湑水河|湑水河]]、[[灙水河|灙水河]]，南有[[漾家河|漾家河]]、[[濂水河|濂水河]]、[[冷水河_(汉江)|冷水河]]、[[南沙河|南沙河]]等。

汉江流出汉中盆地，即进入一段长约600公里的峡谷河段，河床狭窄，河谷陡峻，水流湍急，一泻千里，水力资源丰富，丹江口水电站、安康水电站等均坐落于这片峡谷中。
* 从大龙河口到[[黄金峡镇|黄金峡镇]]渭门村的一段，长50余公里，是汉江上游最为险峻的峡谷段，上段称小峡，北纳[[酉水河|酉水河]]，下段即著名的黄金峡，又称大峡，先向北、再向东、再向东南、再向南兜一个大圈，收纳[[金水河|金水河]]、[[子午河|子午河]]。
* 从渭门村出，汉江往东南方向流去，纳[[牧马河|牧马河]]，然后流入石泉水库。[[石泉县|石泉县]]至[[白河县|白河县]]为峡谷盆地交错段，包括石泉盆地、[[紫阳县|紫阳]]峡谷、[[安康|安康]]盆地、[[旬阳|旬阳]]峡谷、白河峡谷等，安康城上游建有[[安康水库|安康水库]]。该段左岸有[[池河_(汉水)|池河]]、[[月河_(汉水支流)|月河]]、[[旬河|旬河]]、[[夹河|夹河]]等汇入，右岸有[[黄洋河|黄洋河]]、[[岚河|岚河]]、[[坝河_(汉水支流)|坝河]]等汇入。从陕西省旬阳县[[兰滩乡|兰滩乡]]/湖北省[[郧西县|郧西县]]兰滩开始，汉江成为两省的[[界河|界河]]。
* 出白河县后，汉江干流全部进入湖北省境内，至郧阳区[[柳陂镇|柳陂镇]]堵河口，纳右岸的[[堵河|堵河]]，然后流至[[丹江口市|丹江口市]]，纳左岸的[[丹江|丹江]]，从白河县至丹江口市河段长223公里。丹江口建有[[丹江口水库|丹江口水库]]，[[南水北调中线工程|南水北调中线工程]]即从丹江口水库引汉江水向北。<ref name="HHDD" />

=== 中游 ===
[[File:The_Town_of_Fancheng,_on_the_Bank_of_the_Han_Jiang_(Han_River)._Hubei_Province,_China,_1874_WDL2101.png|thumb]]汉江河岸（1874）]]

汉江自丹江口开始，流向由西-东变为西北-东南，穿行于低山丘陵间，然后流入[[老河口市|老河口市]]，市区下游建有王甫洲水库，作为丹江口水库的反调节水库。出老河口市区后，划过一个弧形，流入[[谷城县|谷城县]]境内，谷城下游有[[南河_(汉水支流)|南河]]从右岸注入。然后继续东南流，流至[[襄阳市|襄阳市]]，[[唐白河|唐白河]]在左岸入汇。丹江口-襄阳段为丘陵平原，河道弯曲，河床开阔，江面低水位宽300-400米，洪水期宽2000-3000米，[[心滩|心滩]]、洲滩发育。

出襄阳后，江面逐渐加宽，流经[[宜城|宜城]]，[[蛮河|蛮河]]自右岸注入。继续东南流，进入[[钟祥市|钟祥市]]境内。襄阳-钟祥段位于汉江[[地堑|地堑]]，河道弯曲，水流散乱，江面低水位宽300-400米，洪水期宽2000-3000米、甚至8000-10000米；心滩、洲滩众多，枯水时河汊密布，洪水时汪洋一片。<ref name="HHDD" />

=== 下游 ===
[[File:Wuhan,_Han_river-2.jpg|thumb]]]]

钟祥以下，汉江进入江汉平原。下游两岸均修筑堤防，汉江在其约束下，从钟祥南流至[[沙洋县|沙洋县]]，然后转向东流。至[[潜江市|潜江市]]泽口，右岸有汉江最大[[分流_(水文学)|分流]][[东荆河|东荆河]]。然后经过[[天门市|天门市]]、[[仙桃市|仙桃市]]，流至[[汉川市|汉川市]]，左岸纳[[汈汊湖|汈汊湖]]、[[汉北河|汉北河]]。最后进入[[武汉市|武汉市]]，于[[汉口|汉口]]龙王庙汇入长江。

据1995年统计，汉江干流下游堤防总长约727公里，左岸堤防自钟祥至汉口，长约369公里；右岸堤防自沙洋至[[汉阳|汉阳]]，长约358公里。下游左岸的[[遥堤|遥堤]]，上起钟祥市罗汉寺，下至天门市多宝湾，是汉江最重要的堤防。1935年，汉江特大洪灾，中游左岸堤防溃决达4公里，淹没汉北平原33.33万公顷农田，死亡8万余人，重修时放弃原堤段，留下200余平方公里的自然滞洪区，修建罗汉寺至旧口的遥堤18.2公里，后将旧口至多宝湾的堤段并入，合计56.26公里。

下游堤防从明代开始修筑，历经演变，河道越往下越收拢，遥堤末端约宽3500米，至汉川市仅300-400米，汉口附近只剩100米左右，遇上洪水，排泄不畅，容易酿成洪灾。汉川河段的河道安全泄量只有9000[[立方米|m³]]/[[秒|s]]，长江水位较高时则只有5000m³/s，而1935年特大洪灾中游洪峰流量超过50000m³/s。因此，为预防洪灾，1956年于仙桃市附近汉江右岸修建了[[杜家台分洪工程|杜家台分洪工程]]。<ref name="HHDD" />
