'''漢諾威王朝'''（{{lang-de|Haus Hannover}}），是於1692年－1866年間統治[[德意志聯邦共和國|德意志]][[汉诺威_(地区)|漢諾威地區]]與在1714年－1901年間統治[[英国|英国]]的王朝。

== 起源 ==

汉诺威王朝是[[德意志|德意志]][[不倫瑞克王朝|不倫瑞克王朝]]的分支之一，因此又稱為不倫瑞克王朝漢諾威分支。漢諾威王朝和[[不倫瑞克王朝|不倫瑞克王朝]]的前身都是[[韋爾夫王朝|韋爾夫王朝]]，祖先最早可以追溯到[[薩克森及巴伐利亞公爵|薩克森及巴伐利亞公爵]][[獅子公爵亨利|獅子公爵亨利]]。而亨利的幼子威廉成為[[吕讷堡公爵|吕讷堡公爵]]（Herzog von Lüneburg）。

不倫瑞克王朝的呂訥堡分支，經過多年發展後，在1692年成為[[不伦瑞克-吕讷堡选侯国|漢諾威選侯國]]。自此以後，王室人員以國為姓氏，改稱為漢諾威分支，即為漢諾威王朝。

== 漢諾威 ==
{{See Also|汉诺威统治者列表}}
=== 漢諾威選帝侯 ===
1692年，[[哈布斯堡王朝|哈布斯堡王朝]]的神聖羅馬皇帝[[利奧波德一世_(神聖羅馬帝國)|利奧波德一世]]，委任[[不伦瑞克统治者列表|不倫瑞克-呂訥堡公爵]]為第9位世襲[[選帝侯|選帝侯]]，是為不倫瑞克-呂訥堡選帝侯，以增加哈布斯堡王室成員當選[[神聖羅馬帝國皇帝列表|皇帝]]的勝算機會。自此以後，不倫瑞克-呂訥堡公爵及其後代便以封國為姓，改姓漢諾威。

自1714年起，漢諾威選帝侯[[喬治一世_(大不列顛)|喬治]]入主[[英國|英國]]，兼任[[英王|英王]]（詳見[[汉诺威王朝#大不列顛國王及愛爾蘭國王|下文]]），漢諾威和[[大不列顛|大不列顛]]以及[[愛爾蘭|愛爾蘭]]結成[[共主邦聯|共主邦聯]]。

而[[不伦瑞克-吕讷堡选侯国|漢諾威選帝侯國]]，在1803年被法軍佔領，直至1814年[[拿破崙|拿破崙]]倒台後，漢諾威在列強同意下，被升級為[[王國|王國]]。

=== 漢諾威國王 ===
1814年10月，[[維也納會議|維也納會議]]上決定，為了給予[[英國國王|英國國王]]在[[德意志|德意志]]事務上的發言權，決定將漢諾威選帝侯國提昇為[[漢諾威王國|漢諾威王國]]，並與英國組成[[共主邦聯|共主邦聯]]。

但這個共主邦聯很快瓦解，當1837年英王[[威廉四世_(英國)|威廉四世]]駕崩後，英國王位傳給了血緣關係較近的姪女[[維多利亞女王|肯特郡主維多利亞]]。不過卻因德意志地區施行[[歐洲大陸|歐洲大陸]]的[[薩利克法|薩利克法]]，禁止[[女性|女性]]繼承王位。維多利亞三伯父的漢諾威君主寶座，傳給了她的五叔[[恩斯特·奥古斯特一世_(汉诺威)|坎伯蘭公爵恩斯特·奧古斯特一世]]。

但此時，[[普魯士|普魯士]]已經開始密謀統一[[德國|德國]]的大業。1866年，[[普奧戰爭|普奧戰爭]]期間，普魯士突然派兵佔領漢諾威王國，並改編為普魯士的一個[[行省|行省]]。漢諾威王國滅亡。

== 英國國王 ==
1688年，英國發生[[光榮革命|光榮革命]]，英王[[詹姆斯二世_(英國)|詹姆斯二世]]和其信奉[[天主教|天主教]]的後裔被逐離英國。詹姆斯信奉[[新教|新教]]的女兒[[瑪麗二世_(英國)|瑪麗]]和[[安妮女王|安妮]]陸續繼位，但不幸地二人駕崩後都無存活子嗣可繼承王位。

在[[安妮女王|安妮女王]]長子[[格洛斯特公爵威廉王子|威廉王子]]夭折後，[[英國國會|英國國會]]為避免兼領[[英國國教|英國國教]]的國王寶座再次落入[[天主教徒|天主教徒]]手上，通過《[[1701年王位繼承法|1701年王位繼承法]]》，規定：

* 若《1689年權利法案》中所規定的繼承人全部無子而亡後，王位由英王[[詹姆士一世_(英格蘭)|詹姆士一世]]的外孫女，漢諾威選帝侯夫人[[索菲_(汉诺威选侯夫人)|索菲]]和她的新教後代繼承。

這部法律，令當時在[[英國王位繼承|英國王位繼承]]序列頭50名的王室成員都失去王位繼承權。雖然法案導致數次因王位爭端而起的衝突，但位於王位繼承序列第52名的[[漢諾威|漢諾威]][[選帝侯|選帝侯]]，[[喬治一世_(大不列顛)|乔治一世]]（George I），最終還是於1714年繼位為大不列顛國王及愛爾蘭國王，直至1727年駕崩。而[[大不列顛王國|大不列顛王國]]和[[愛爾蘭王國|愛爾蘭王國]]於1801年合併，組成[[大不列顛及愛爾蘭聯合王國|大不列顛及愛爾蘭聯合王國]]。

=== 大英帝國和印度女皇 ===
{{See Also|大英帝國}}

遠在[[英國國王|英國國王]][[亨利八世|亨利八世]]在位時期，英國就開始實施對外擴張政策。在[[維多利亞時期|維多利亞時期]]英國的[[殖民|殖民]]疆域達致最廣，也曾一度是全球唯一的工业化强权。但英國在法律上一直不是「帝國」，因為英國君主一直僅稱'''國王'''或'''女王'''，而非皇帝。因此大英帝國一直是一個「非正式帝國」。

據說，維多利亞女王因為她在[[俄罗斯帝国|俄國]]和[[德意志帝国|德國]]的親戚，都可以稱為[[皇帝|皇帝]]，而自己的國家比他們的強大多了，卻只可稱為女王，因此女王的其中一位[[首相|首相]][[本杰明·迪斯雷利|本杰明·迪斯雷利]]有鑑於此，便借英國剛滅印度[[莫臥爾帝國|莫臥爾帝國]]，於1877年1月1日，獻上'''印度女皇'''之尊號與'''維多利亞女王'''。自此，英國君主有了帝號，[[大英帝國|大英帝國]]因為征服印度而成了真正的帝國。

維多利亞女王去世後，其長子[[愛德華七世_(英國)|愛德華]]繼位，由於愛德華[[冠父姓|冠父姓]]，[[漢諾威王朝|漢諾威王朝]]在英、印等地的統治正式結束。因此漢諾威王室僅有維多利亞一人擔任過印度女皇(1877年-1901年)。此後，帝位隨英國王位傳予[[萨克森-科堡-哥达王朝|萨克森-科堡-哥达王朝]]。

== 家族首領 ==
1866年後，漢諾威王室的首領都冠以漢諾威親王的稱號，而在1919年前，漢諾威家族依然持有英國坎伯蘭公爵的爵位。但在1919年3月28日，英王[[乔治五世_(英国)|喬治五世]]因漢諾威王室的親戚，不願放棄德國國籍，並涉嫌幫助德皇，對付英國，而動用《1917年削除爵位法》，廢除坎伯蘭公爵的稱號。
