'''战菁一'''（{{bd|1981年|3月17日}}），[[中国大陆|中国大陆]]知名电影电视演员、[[成都军区|成都军区]]战旗歌舞团舞蹈演员，毕业于[[中央戏剧学院|中央戏剧学院]]，曾获得[[中华人民共和国文化部|中华人民共和国文化部]]颁发的[[群星奖|群星奖]]。

== 生平 ==
战菁一早年就读于[[中央戏剧学院|中央戏剧学院]]。战菁一擅长出演各种性情的角色，在《网球王子》中，是热辣活力的女子小朋；在《我的兄弟是顺溜》是温婉可人的荷花；在《碧波仙子》是隐忍内敛的赵秀。<ref>{{cite web |url= http://ent.sina.com.cn/v/m/2010-06-01/14162975107.shtml|title= 战菁一热拍《碧波仙子》 古装造型“非仙似仙”|accessdate=2010年6月1日 |publisher= 新浪网|work= |language =zh-cn}}</ref>《后宫甄嬛传》出演流朱一角，使得她好评如潮。<ref>{{cite web |url=http://roll.sohu.com/20111214/n329004803.shtml |title=《后宫甄嬛传》热播 战菁一伶牙俐齿大受欢迎 |accessdate=2011年12月14日 |publisher=搜狐网 |work= |language =zh-cn}}</ref>
