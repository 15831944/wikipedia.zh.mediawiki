{{Vgname|戰地風雲5|tw=戰地風雲5|cn=战地5|en=Battlefield V}}是一款由[[EA_DICE|EA DICE]]开发，[[艺电|艺电]]发行的[[第一人称射击游戏|第一人称射击游戏]]。本作是《[[战地系列|战地]]》系列的第14款作品，以[[第二次世界大战|第二次世界大战]]中“未闻，未见，未被呈现”的各个方面<ref name="IGN"/>为游戏主题。

== 游戏设定 ==
《战地5》依旧保留了小队作战的核心玩法，玩家会被按照轴心国与同盟国分为两个敌对阵营，在[[坦克|坦克]]的掩护下战斗。玩家将可以建造掩体或破坏建筑物，摧毁敌方载具或击杀敌方玩家。玩家可以成为“突击兵”，“医护兵”，“支援兵”和“狙击兵”四种兵种，并可以驾驶飞机和坦克等多种载具。小队玩家可以互相重生和复活。

本作取消了付费货币微交易和高级通行证，所有的武器皮肤和角色皮肤都会进行标价，而新的内容会进行免费更新。<ref>{{cite web |url=http://www.vgtime.com/topic/948155.jhtml |title=《战地5》发布会信息汇总 |date= 2018-05-26|website=VGTime |publisher=Limboo |access-date=2018-05-29 |quote=}}</ref>这一设计受到了《[[星球大战：前线II|星球大战：前线II]]》中开箱系统的恶评的影响。<ref>{{cite web |url=https://new.qq.com/omn/20180416/20180416A1RGBS.html |title=EA：新作绝不会再犯和《星球大战：前线2》同样的错误
 |date= 2018-04-16|website=腾讯新闻 |publisher=游骑兵 |access-date=2018-06-13 |quote=}}</ref>

== 开发 ==
《战地5》是由EA旗下DICE团队开发、2002年问世至今已16年的《战地》系列全新作品，采用不同于前一代《战地1》第一次世界大战战场的场景，加入前所未见的大规模多人模式，提供逼真的画面与音效，导入新的模式与新的体验，以及收录引人入胜的战争故事。

EA公布《战地5》（Battlefield V）的发行消息，此续作将于11月20日推出，并将玩家带回第二次世界大战的世界。我们也非常兴奋地宣布，GeForce GTX是《战地 5》的 PC 游戏平台。

DICE 的资深製作人，Andreas Morell表示:「针对《战地5》，我们希望能与和我们一样对 PC 游戏充满热情的伙伴进行合作。」「结合我们才华洋溢的开发团队，与 NVIDIA先进的游戏技术和可靠的GeForce PC游戏平台，我们能创造最真实、富临场感且视觉效果令人震撼的《战地5》游戏体验。」

NVIDIA的工程师已与DICE的《战地5》开发人员合作，以便将GeForce游戏平台之最带给 PC 游戏玩家，也就是说，GeForce Experience将为《战地5》提供Game Ready驱动程式、最佳游戏设定与其他NVIDIA平台功能。

== 评价 ==
=== 争议 ===
在《战地5》宣传片发布后，有玩家就花俏的服装和女性士兵的登场，以及没有展示納粹旗幟等各方面与历史不符向艺电表示了反对，並批評開發商“過份[[政治正確|政治正確]]”。EA DICE的总经理Oskar Gabrielson在[[Twitter|Twitter]]上回应，他希望《战地5》在能代表人类历史上一部分伟大时刻的同时，让玩家有选择和自订角色的权利。

==股價==

雖然 EA 非常有誠意的向玩家解釋《戰地風雲 5》延遲上市的理由，但因為下調了玩家預訂淨收入預測值，加上遊戲推遲的消息，也讓 EA 的股價一開盤就下跌了 7.7%。<ref>{{Cite tweet |user=ogabrielson |number=1000042209247416322 |date=2018-05-28 |title= choose and customize}}</ref>
