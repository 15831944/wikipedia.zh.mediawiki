'''{{zy|冼|-{zh-tw:ㄒㄧㄢˇ;zh-cn:xiǎn}-}}夫人'''（{{bd|512年||602年||catIdx=Sheng冼}}），[[高凉郡_(东汉)|高涼郡]]（今[[广东省|广东省]]境）[[俚人|俚人]]（[[百越|百越]]的分支），為[[俚人|俚人]]傑出的女領袖和[[軍事家|軍事家]]。[[南梁|南梁]]宋康郡夫人、[[南陳|陳朝]][[石龍|石龍]]太夫人、[[隋朝|隋朝]][[譙國|譙國]]夫人，諡誠敬夫人。

== 生平 ==

冼氏世代均為[[俚人|俚人]]首領。冼夫人生於[[梁武帝|梁武帝]]初年。[[大同_(南梁)|大同]]（535年—546年）初年，[[罗州|罗州]][[刺史|刺史]][[冯融|冯融]]听闻洗氏的品行，为其子[[高凉郡_(东汉)|高涼郡]][[太守|太守]][[馮寶|馮寶]]娉以为妻。婚后，洗夫人輔佐馮寶平息了今[[广东省|广东省]]境内的俚族[[原住民|原住民]]與[[中原|中原]]人士的衝突，促進了[[中原|中原]]人和[[俚族|俚族]]的和解，引[[海南島|海南島]]各部落歸附[[南梁|南梁]]，並開始設立[[崖州|崖州]]，使海南島再次成為古代[[中國|中國]]中央王朝設立的正式政區。

[[南梁|南梁]][[侯景之亂|侯景之亂]]時，冼夫人率兵擊破[[高州_(广东)|高州]][[刺史|刺史]]李遷仕，並與都督[[陳霸先|陳霸先]]聯合。[[南陳|陳朝]]建立後，冼夫人即率眾歸附陳朝。陳朝[[太建|太建]]二年（570年），[[广州_(古代)|廣州]][[刺史|刺史]][[歐陽紇|歐陽紇]]反，將冼夫人兒子馮僕騙去，欲誘迫他同反。冼夫人不以兒子之安危為念而肯負國家，立即布兵拒境，并與陳朝遣討軍隊全力擊潰叛軍，獲冊封為[[中郎將|中郎將]]、[[石龍|石龍]][[太夫人|太夫人]]，儀仗比照刺史。

[[隋文帝|隋文帝]]出兵南下[[隋灭陈|滅陳朝]]後，冼夫人保境安民，被俚族尊為「聖母」。[[隋朝|隋朝]][[總管|總管]][[韋洸|韋洸]]被陳朝[[嶺南|嶺南]]守將徐璒所阻，於嶺下停滯不前，無法進入嶺南。源陳主帥[[楊廣|楊廣]]命令[[陳後主|陳後主]]致書冼夫人，使其歸隋朝，為了證明是真的，還把冼夫人曾經獻給陳朝皇帝的扶南犀杖及陳後主的兵符拿給她看。冼夫人見到信及物件，始確信陳朝已滅亡，便派孫子馮魂迎韋洸嶺南，各地才聞風歸附。從此[[嶺南|嶺南]]地區全部歸隋朝管轄。

不久[[番禺|番禺]]人王仲宣謀反，越族首領皆響應，圍韋洸於嶺南首府[[南海|南海]]，並派兵進屯[[衡山|衡山]]，斷絕中原和嶺南的交通。冼夫人派遣孫子馮暄領兵救援韋洸。由於馮暄與叛軍將領陳佛智素來友善，故意停留不解南海之圍。冼夫人知道之後，派人將馮暄監禁，改派另一名孫子[[馮盎|馮盎]]討伐陳佛智，擊敗陳佛智並將其斬首。再進兵至南海，與鹿願軍會合，共敗王仲宣。是役，冼夫人親自被甲，率領軍隊，保護[[隋文帝|隋文帝]]使節[[裴矩|裴矩]]招撫[[嶺南|嶺南]]各州，使[[蒼梧郡|蒼梧郡]]南越族[[首領|首領]][[陳坦|陳坦]]、[[岡州|岡州]]馮岑翁、梁化鄧馬頭、[[藤州|藤州]]李光略、羅州龐靖等皆來參謁[[裴矩|裴矩]]，歸順隋朝。[[隋文帝|隋文帝]]大表贊賞，拜馮盎為高州刺史，並赦免馮暄，拜為羅州刺史，追贈馮寶為廣州總管、譙[[國公|國公]]，冊封冼夫人為譙[[國夫人|國夫人]]，讓[[譙國|譙國]]夫人開創[[幕府_(中國)|幕府]]，可以自由任命[[長史|長史]]及下屬官員，給予印章（有實權），可以調動俚族各部及嶺南六州兵馬，若遇有機急，可便宜行事。將宋康邑再次授予馮僕的[[遺孀|遺孀]]（冼夫人亡夫馮寶原為南梁宋康郡公，冼夫人本為南梁宋康郡夫人。隋朝修改地方體制，不設[[宋康郡|宋康郡]]）。

不久，[[番州_(隋朝)|番州]]總管[[趙訥|趙訥]]貪汙，苛待人民，使南越諸[[俚|俚]][[獠|獠]]叛亂。冼夫人派遣長史[[張融|張融]]向朝廷報告，提出安撫之法，並告發趙訥罪狀。隋文帝派遣使者調查，查獲趙訥的贓款，將趙訥法辦。降旨委託冼夫人招撫叛民。冼夫人親自帶著詔書，自稱[[使者|使者]]，遍曆十餘州，宣述隋文帝之意，勸諭俚獠各部，所至皆降。隋文帝為表嘉獎，賜冼夫人[[臨振縣|臨振縣]][[湯沐邑|湯沐邑]]一千五百戶，並追贈馮僕為[[岩州|岩州]]總管、[[平原郡|平原郡]]公。

[[仁壽_(隋)|仁壽]]二年（602年），冼夫人去世，虚岁九十一，谥号誠敬夫人。

[[中華人民共和國總理|中華人民共和國總理]][[周恩來|周恩來]]曾讚譽冼夫人為「中國歷史上第一位巾幗英雄」。

== 纪念 ==
清朝光緒《茂名縣誌》載：“（高涼山）山半有高涼庵，祀冼夫人。”
详见:[[隋谯国夫人冼氏墓|隋谯国夫人冼氏墓]]、[[冼夫人紀念館|冼夫人紀念館]]、[[冼夫人廟|冼夫人廟]][[高涼嶺冼太廟|高涼嶺冼太廟]]
===高州冼太廟===

後人為紀念冼夫人而在各地興建了許多冼太廟，僅在茂名[[地區|地區]]就超過200座，其中尤以高州冼太廟為最大、最有影響力。

冼太廟位於高州城區人民路，[[潘州公園|潘州公園]][[北|北]]面，始建于[[明朝|明朝]][[嘉靖|嘉靖]]十四年（1535年），嘉靖四十三年和[[清朝|清朝]][[同治|同治]]年間先後重修。整座[[廟宇|廟宇]][[建築面積|建築面積]]達800餘平方米，磚木結構。全廟分前殿、中殿、正殿和馮公廟（馮公即馮寶）四部分，總進深49.5[[米|米]]，總面闊13.4米，殿內冼太夫人[[雕像|雕像]]、[[石刻|石刻]]、[[碑刻|碑刻]]等保存完好，有較高的[[歷史|歷史]][[文化|文化]][[價值|價值]]，為省級重點文物保護單位。

2000年2月，[[江澤民|江澤民]][[總書記|總書記]]曾到冼太廟參觀，指出要把冼太廟辦成[[愛國主義教育|愛國主義教育]]的[[基地|基地]]。

== 民族 ==

{{fact|冼夫人後廣東境內的大部分[[俚人|俚人]]（漢化不深的[[南越族|南越族]]人）在明清時代後陸續融入[[漢族|漢族]]，成為漢族的組成部分。但在冼夫人所處的時代，[[俚人|俚人]]屬[[壯族|壯族]]先民的分支，至到[[宋代|宋代]]，[[俚人|俚人]]的民族風俗和語言依然與[[壯族|壯族]]一致。[[清代|清代]]後，留在粵西[[茂名|茂名]]一帶的'''俚人'''多數已經融入汉族，而另一支向西發展的俚人在[[广西|广西]]与[[僚人|僚人]]合为一体，成為[[壯族|壯族]]、[[布依族|布依族]]等[[民族|民族]]的一部分。故冼夫人在历史上被視為正宗的[[壮族|壮族]]人。}}
