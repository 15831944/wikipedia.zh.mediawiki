'''麟洛車站'''位於[[屏東縣|屏東縣]][[麟洛鄉|麟洛鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[屏東線|屏東線]]的[[鐵路車站|鐵路車站]]。此站事實上距麟洛市區約2[[公里|公里]]，但冠以鄉名。

== 車站構造 ==
*[[側式月台|側式月台]]兩座。
*配合[[高雄潮州鐵路捷運化計畫|高雄潮州鐵路捷運化計畫]]，本站已於2013年6月25日改為高架車站。

== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]，僅停靠[[臺鐵區間車|區間車]]
*由於車站距位於[[台1線|台1線]]上之鬧區頗遠、列車較少。故麟洛鄉民多利用公路運輸至[[屏東市|屏東市]]（約5公里）再轉乘其他交通工具。
*已於2013年6月25切換為高架路段，高架車站開始營運。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。<ref>{{cite news|author=|title=歸來─潮州電子票證多卡通　10月15日啟用|url=http://www.appledaily.com.tw/realtimenews/article/new/20151010/708645/|accessdate=2015-10-10|newspaper=蘋果日報|date=2015-10-10}}</ref>

== 車站歷史 ==
*1941年7月1日：設立'''[[隘寮溪|隘寮溪]]信號場'''，管理石碴支線。
*1949年6月1日：「隘寮溪站」改為客貨營業站，辦理一般客貨運。
*1951年5月1日：因本站位置地處[[屏東縣|屏東縣]][[麟洛鄉|麟洛鄉]]，故改稱今名。
*1986年3月1日：由三等站降為甲種簡易站，並指定由[[屏東車站|屏東站]]管理。
*1989年3月1日：由甲種簡易站降為招呼站。
*2013年6月25日：改為高架車站。
*2015年8月23日：高架雙軌電氣化通車。
*2015年10月15日：啟用多卡通刷卡機。
