{{日本作品|半澤直樹|半沢直樹|Hanzawa Naoki}}是[[日本|日本]][[TBS電視台|TBS電視台]]於2013年7月7日於[[周日劇場|周日劇場]]時段（21:00 - 21:54，[[日本標準時|JST]]）播出的[[電視連續劇|電視連續劇]]，改編自[[池井戶潤|池井戶潤]]的两部小說作品，以第一部《我們是泡沫入行組》（{{lang|ja|オレたちバブル入行組}}）為基礎再加上第二部《我們是花樣泡沫組》（{{lang|ja|オレたち花のバブル組}}）的前後篇改編而成，並以書中主角「半澤直樹」作為劇名<ref name="簡介">{{cite web |url=http://www.xinmedia.com/n/news_article.aspx?newsid=317998&type=1|title=日劇一族／堺雅人新婚氣旺 對抗上司成英雄 |accessdate=2013-07-16|work= |publisher=欣新聞 |date=2013-07-09 }}</ref>，由[[堺雅人|堺雅人]]、[[上戶彩|上戶彩]]主演。

==劇情簡介==
本劇描寫於[[日本泡沫經濟|泡沫時期]]入行的銀行員半澤直樹，與銀行內外的「敵人」戰鬥的故事。

===第一部：大阪篇===
[[File:Mido-suji_Osaka01s4s3200.jpg|thumb]]
原產業中央銀行因有大筆[[呆帳|呆帳]]而與東京第一銀行合併為東京中央銀行。合併後[[銀行|銀行]]內部有著「舊東京第一」與「舊產業中央」的派系鬥爭。大阪西分行行長為了自身的利益和晉升，以及提升該分行的名聲，因而裸貸給西大阪鋼鐵，而在融資一個月後即倒閉致使銀行留下五億元債務。分行長為了自身利益開始推卸責任給當時為融資課長的半澤，與此同時[[國稅局|國稅局]]亦開始介入事件，加上相關人士的不合作態度，半澤等人要收回五億變得困難重重。然而在半澤等人鍥而不捨的追回之下，五億元債務最終成功給收回，並同時揪出幕後黑手。而後半澤以此為契機晉升東京本部，擔任次長一職。

===第二部：東京篇===
[[File:Mitsui_Main_Building_2009.jpg|thumb]]
晉升東京本部後一年，銀行批出伊勢島飯店的[[融資|融資]]200億，但批出後沒多久竟發現飯店有著120億元的[[资金|资金]]運用失誤，再加上[[金融廳|金融廳]]檢查局要在兩周內對東京中央銀行進行監察，迫使銀行必須設法使伊勢島飯店不被金融廳認為是問題企業，而避免銀行名譽受損且被迫改組，於是伊勢島飯店社長與行長將此事托付因成功收回5億元債務而備受囑目的半澤直樹來處理。

==製作背景==
導演{{link-ja|福澤克雄|福澤克雄}}表示「起初是讀到池井戶潤的《[[下町火箭|下町火箭]]》（{{lang|ja|下町ロケット}}）而相當感動，因而讀了池井戶潤的全部小說，看到《我們是泡沫入行組》的標題時卻完全不被吸引，但讀了之後覺得很有趣，覺得不得不將這本書拍成影片」。他特別選擇了小說中半澤最活躍的部分來拍成電視劇。此外，他亦希望將《半澤直樹》拍成[[電影|電影]]。他表示原著的第三部作品很適合拍成電影版，也想在電影中描寫更深刻的主題<ref name="導演訪談">{{cite web |url=http://www.oricon.co.jp/news/movie/2026328/full/ |title=<span lang=ja>【インタビュー】7月期ドラマ『半沢直樹』演出</span> |accessdate=2013-07-15|work= |publisher=[[Oricon|Oricon]] |date=2013-07-06 }}</ref>。<ref>[http://www.insight-post.tw/?p=3988 【書評影評】日劇《半澤直樹》反映出的金融制度與哲學 ]，洞見Insight－國際事務評論網</ref>

當初挑選飾演「半澤直樹」的演員時，非常快地就決定由[[堺雅人|堺雅人]]擔任，導演福澤克雄表示「在《[[南極大陸_(電視劇)|南極大陸]]》合作時，對堺雅人台詞的節奏以及他作為演員的態度十分佩服……池井戶潤先生腦海中也浮現他的臉<ref name="導演訪談" />。」《半澤直樹》是堺雅人在同年4月跟[[菅野美穗|菅野美穗]]新婚後的首部電視劇<ref>{{cite web |url=http://www.oricon.co.jp/news/movie/2026328/full/ |title=<span lang=ja>堺雅人“既婚”初ドラマ！上戸彩と新婚タッグ！</span> |accessdate=2013-07-16|work= |publisher=[[體育日本|體育日本]] |date=2013-05-15}}</ref>。半澤直樹的妻子則是由[[上戶彩|上戶彩]]飾演。這次除了是上戶跟堺雅人繼10年前在《{{link-ja|獻給一個夏天的爸爸|ひと夏のパパへ}}》一起演出後的再一次合作外，也是她在2012年9月跟[[EXILE|EXILE]]的[[五十嵐廣行|HIRO]]結婚後的首次電視劇演出<ref name="收視紀錄1" />。

==播出==
首集延長一小時並創下19.4%的高平均收視率，在該季僅次於《[[最強名醫|最強名醫]]》的19.6%。第二集的平均收視率21.8%較上集高了2.4%，刷新了該季的收視紀錄，瞬間最高收視率於9點51與52分，皆達到了24.4%，該劇亦為堺雅人所主演的收視最高紀錄<ref name="收視紀錄1">{{cite web |url=http://www.sponichi.co.jp/entertainment/news/2013/07/16/kiji/K20130716006228740.html |title=大台突破！堺雅人主演「半沢直樹」第２話で２１・８％ |accessdate=2013-07-16|work= |publisher=[[體育日本|體育日本]] |date=2013-07-16 }}</ref><ref name="收視紀錄">{{cite web|url=http://mainichi.jp/mantan/news/20130716dyo00m200004000c.html |title=半沢直樹:第2話視聴率は21．8％　堺雅人主演作が第2話で20％超 |accessdate=2013-07-16 |work= |publisher=[[每日新聞|每日新聞]] |date=2013-07-16 |archiveurl=https://archive.is/20130717063456/http://mainichi.jp/mantan/news/20130716dyo00m200004000c.html |archivedate=2013-07-17 |deadurl=yes }}</ref>。

從第一集到[[最後的百倍奉還下跪者是誰！～衝撃的結果！！是友情？還是背叛？|最終回]]的收視率不曾下跌，而最終回的平均收視率達到了42.2%，達到了本世紀日本電視劇最終回的最高紀錄，打破了2011年[[日本電視台|日本電視台]]連續劇《[[家政婦三田|家政婦三田]]》最終回40.0%的收視紀錄。而最終回瞬間收視率最高的那段為當晚10點17分，達到了46.7%，也打破了《家政婦三田》的42.8%<ref name="收視紀錄2">{{cite web |url=http://www.oricon.co.jp/news/movie/2028958/full/ |title=『半沢直樹』最終回で『ミタ』超え、“紅白”並みの視聴率42.2％、瞬間最高46.7％|accessdate=2013-09-24|work= |publisher=[[Oricon|Oricon]] |date=2013-09-24 }}</ref>。

台灣[[緯來日本台|緯來日本台]]以《王牌銀行員》为劇名引進，并於2013年10月7日至10月18日期间播出。10月18日播出的完結篇平均收視率達到2.0（約70萬人收看），為台灣有線電視中日劇單集的最高收視<ref name="台灣收視">{{cite web|url=http://iservice.libertytimes.com.tw/liveNews/news.php?no=888031&type=%E5%BD%B1%E5%8A%87&Slots=Live|title=「半澤直樹」結局收視高 打趴木村拓哉|accessdate=2013-11-02|work=|publisher=[[自由時報|自由時報]]|date=2013-10-22|deadurl=yes|archiveurl=https://web.archive.org/web/20131103163309/http://iservice.libertytimes.com.tw/liveNews/news.php?no=888031&type=%E5%BD%B1%E5%8A%87&Slots=Live|archivedate=2013-11-03}}</ref>。[[八大戲劇台|八大戲劇台]]於2014年3月17日播出<ref>[http://www.gtv.com.tw/News/20140225001/ 八大戲劇台【TBS劇場】] {{webarchive|url=https://web.archive.org/web/20141204203825/http://www.gtv.com.tw/News/20140225001/ |date=2014-12-04 }}，八大電視台</ref>。

香港[[无线电视|无线电视]]以原名引进，於2013年11月16日在旗下收费电视频道[[煲劇1台|煲劇1台]]首播，并於2014年1月28日在旗下收费电视频道[[TVB_Window|TVB Window]]重播。另外，于2014年7月安排在免费频道[[翡翠台|翡翠台]]及[[高清翡翠台|高清翡翠台]]播映，并作为[[TVB_Amazing_Summer|TVB Amazing Summer]]2014其中一個推介的節目。

第一章遭到大阪西支店分店長淺野陷害，被迫承擔西大阪鋼鐵社長東田欺詐[[融資|融資]]5億，造成銀行損失的責任，為了向東田討回金錢而四處奔走。對於長官、上司不當的做法，會抓住足夠的證據後而予以反撲。

第二章以東京中央銀行總行為背景，針對往來對象伊勢島酒店爆發的巨額虧損、及內部融資作業不法情事進行調查，進而與過去有極深的因緣，現今位居總行高層的大和田常務正面交鋒。

其態度和處事方式深得一部分人(主要是其好友、下屬)尊敬、認同，如同期好友的渡真利和近藤、大阪篇中的竹下社長、部下中西、東田的情婦未姬、東京篇中的湯淺社長、戶越會計課長等。與此同時亦有一部分人(主要是其上司)對此相當不滿，如大和田常務、岸川部長、黑崎檢察官、大阪篇中的淺野分店長、東田社長、小木曾次長，東京篇的羽根專務、貝瀨分店長等。

==與原作的不同點==
===基本設定===
*原作中半澤直樹有個弟弟「和樹」，但連續劇中的家族組成只有半澤、母親、妻子、兒子的四人家庭。
*半澤的父母在原作中的名字不詳，另外因銀行拒絕融資而自殺的橋段為連續劇的原創部份，在原作中，直樹被銀行內定的時期父親仍健在。
*大和田曉、中野渡謙、黑崎駿一皆為《我們是花樣泡沫組》中才登場的人物。
*半澤直樹與妻子花的相處態度與原作不同。
*行員妻子聚集的「妻子聯誼會」為連續劇原創。

===第一部（大阪篇）===
*在原作有登場的半澤的同期同事苅田，並沒出現在連續劇中。
*撤回半澤父親融資的銀行行員改成大和田。原作為木村，與淺野關係密切的業務統括部的代理部長，劇中未出現。
*竹下企圖自殺以及板橋搶奪資料的部份，為連續劇原創。
*連續劇中東田策劃在越南另起爐灶，原作地點則為中國。
*原作中，並沒有半澤於老人看護中心與小村接觸的場景。
*原作中半澤的內部調職令公佈前，半澤已經把5億元收回，另外，半澤向淺野要求讓他與他部下調職的詳細部門，在原作中沒有明示。
*原作中，黒崎駿一並非大阪國税局統括官。

===第二部（東京篇）===
*伊勢島飯店專務的性別由男性變為女性，姓名也從原作的羽根夏彥改成連續劇版的羽根夏子。
*古里的非法證據資料放置場所，原作是在貝瀨自宅，連續劇改為銀行金庫內。原作為古里個人潛入貝瀨自宅中拿取資料。
*小花回直樹老家的劇情為連續劇原創。
*半澤將資料放置在家，而小花擅自將資料寄回娘家的劇情為連續劇原創。
*田宮電機將3000萬元轉借給LaFite（ラフィット），原作的公司名為La Fayette（ラファイエット）。另外，原作中近藤與La Fayette社長對話的場景，並沒有在連續劇中出現。
*半澤在董事會要求大和田下跪的場景為連續劇原創。

==影響==
原著《{{link-ja|我們是泡沫入行組|オレたちバブル入行組}}》（{{lang|ja|オレたちバブル入行組}}）、《{{link-ja|我們是花樣泡沫組|オレたち花のバブル組}}》（{{lang|ja|オレたち花のバブル組}}）在連續劇播出後，半澤直樹系列已熱賣170萬冊並追加増印120萬冊<ref name="原著">{{cite web |url=http://www.rbbtoday.com/article/2013/08/15/110801.htmll |title=「半沢直樹」人気で、原作も120万部の増刷 |accessdate=2013-09-25 |work= |publisher=RBB Today |date=2013-08-15 }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，而最新的第三部《失落一代的逆襲》（{{lang|ja|ロスジェネの逆襲}}）已賣出超過50萬冊<ref name="原著2">{{cite web |url=http://www.sponichi.co.jp/entertainment/news/2013/09/25/kiji/K20130925006682930.html |title=「倍返し饅頭」は当面販売継続　半沢第３弾小説は５０万部突破|accessdate=2013-09-25|work= |publisher=[[體育日本|體育日本]] |date=2013-09-25 }}</ref>。

由于半泽直树电视剧的大热，电视台曾制作“加倍奉还馒头”在电视台大楼楼下销售，上面包装印上半泽直树的卡片，结果大受欢迎<ref>{{cite web | url=http://www.chinanews.com/yl/2013/09-03/5238420.shtml | title=《半泽直树》收视突破30点 主题馒头一小时售罄 | accessdate=2013-09-21}}</ref>，而其他相關的周邊商品合計總營收超過1億5千萬日圓<ref name="周邊">{{cite web |url=http://www.daily.co.jp/gossip/2013/09/25/0006366005.shtml |title=半沢グッズもバカ売れ！１億５千万円超|accessdate=2013-09-25|work= |publisher=daily sports online |date=2013-09-25 }}</ref>。

而在[[香港免費電視牌照爭議|香港免費電視牌照爭議]]中，為了抗議香港政府黑箱作業沒有對發牌一事作出解釋，香港電視員工更上演此劇以作諷刺<ref>[http://hk.news.yahoo.com/%E6%B8%AF%E8%A6%96%E8%97%9D%E4%BA%BA-%E6%94%BF%E7%B8%BD%E6%BC%94-%E5%8D%8A%E6%BE%A4%E7%9B%B4%E6%A8%B9-224546382.html  港視藝人 政總演《半澤直樹》]</ref>。

在台灣因該劇的高人氣在網路上竄紅，而在正式播出後，劇中半澤直樹的著名金句「加倍奉還」在許多場合被拿來使用，許多政治的場合也常被拿來惡搞或使用。日本媒體亦注意到台灣的半澤風潮，並加以報導<ref name="台灣影響">{{cite web|url=http://www.rbbtoday.com/article/2013/08/15/110801.htmll|title=半沢直樹:台湾でも放送　「加倍奉還」は流行語に|accessdate=2013-10-12|work=|publisher=[[每日新聞|每日新聞]]|date=2013-11-02}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2013年11月26日，中华全国总工会中国职工对外交流中心邀请一批研究中日关系的著名专家和学者，与正在中国访问的日本劳工运动研究会代表团举行了一次题为“如何维系中日关系、开展民间友好活动”的座谈会。在这个座谈会上，[[中国中央电视台|中国中央电视台]]外事处业务主管邱鹏鸣表示，将可能于2014年引进《半泽直树》，正在同TBS就播放权问题进行谈判<ref>[http://www.dfdaily.com/html/150/2013/11/29/1090950.shtml 央视明年有望引进《半泽直树》] {{webarchive|url=https://web.archive.org/web/20140320115907/http://www.dfdaily.com/html/150/2013/11/29/1090950.shtml |date=2014-03-20 }}，东方早报，2013年11月29日</ref>。

2013年12月2日，第30屆日本年度[[新語、流行語大賞|新語、流行語大賞]]評選活動公布得獎名單，日劇「半澤直樹」中男主角講的「加倍奉還」（{{lang|ja|倍返し}}）一詞順利獲選<ref>[http://www.apdnews.com/news/51837.html 年度日本流行语揭晓 “加倍奉还”上榜] {{webarchive|url=https://web.archive.org/web/20131212025419/http://www.apdnews.com/news/51837.html |date=2013-12-12 }}，亚太日报，2013年12月6日</ref><ref>[http://www.cna.com.tw/news/firstnews/201312020024-1.aspx 日本年度流行語 加倍奉還獲選]，中央社，2013年12月2日</ref>。
