'''半次元'''是王耀网络<ref>{{Cite web|url=https://www.tianyancha.com/company/2310304426|title=杭州王耀网络科技有限公司 工商登记注册资料|accessdate=2018-03-29|author=|date=|publisher=天眼查}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>旗下运营的一个定位于[[次元_(次文化用語)|二次元]]爱好者[[用户生成内容|UGC（用户原创内容）]]分享互动社区，成立于2014年。

目前主要提供的服务是以[[Cosplay|Cosplay]]、文字、绘画等原创作品的发布与社群交流，用户可以通过网页版和客户端APP（支持[[Android|Android]]和[[iOS|iOS]]系统）访问半次元社区。<ref>{{Cite web|url=https://bcy.net/static/about|title=关于我们 — 半次元  第一中文COS绘画小说社区|accessdate=2018-03-29|author=|date=|work=|publisher=bcy.net|language=|deadurl=no|archiveurl=https://web.archive.org/web/20171001075650/https://bcy.net/static/about|archivedate=2017-10-01}}</ref>

== 运营历史 ==

2014年6月25日，为了庆祝《[[最终幻想XIV|最终幻想14]]》国服正式运营，半次元社区联合游戏运营方[[盛大游戏|盛大游戏]]，举办了最终幻想绘画大赏暨《最终幻想14》官方绘画比赛。<ref>{{Cite web|url=http://news.17173.com/content/2014-06-25/20140625122519771.shtml|title=FF14二测在即 携手半次元开启全民绘画大赛|accessdate=2018-03-29|author=|date=2014-06-25|last=|work=|publisher=17173|deadurl=no|archiveurl=https://web.archive.org/web/20150609213822/http://news.17173.com/content/2014-06-25/20140625122519771.shtml|archivedate=2015-06-09}}</ref>

2016年2月，半次元创办人王伟（网名：PT）接受媒体专访称，半次元社区成立一年半，已经拥有注册用户150万，注册作者20万，并收到了[[创新工场|创新工场]]等机构数百万美元的天使投资，社区估值上亿，计划于2016年年中开启第2轮融资。<ref>{{Cite web|url=http://www.donews.com/201602/2916945.shtm|title=互联网逻辑办二次元社区 创业一年半半次元估值过亿|accessdate=2018-03-29|author=|date=2016-02-04|work=|publisher=donews游戏|deadurl=no|archiveurl=https://web.archive.org/web/20180329221721/http://www.donews.com/201602/2916945.shtm|archivedate=2018-03-29}}</ref>

== 对外投资 ==
2018年3月，半次元中国大陆独家引进4月的春季动画《[[銀河英雄傳說#Die_Neue_These|银河英雄传说 全新命题]]》。<ref>{{Cite web|url=https://bcy.net/bangumi/episode|title=银河英雄传说 全新命题 在线观看|accessdate=2018-03-29|author=|date=|work=|publisher=半次元|language=|deadurl=no|archiveurl=https://web.archive.org/web/20180329221721/https://bcy.net/bangumi/episode|archivedate=2018-03-29}}</ref>

== 融资 ==
2014年7月，半次元获得了来自[[创新工场|创新工场]]、[[正时资本|正时资本]]的数百万美元天使轮投资。<ref name=":0" />

2015年，半次元获得了来自[[联创永宣|联创永宣]]的数千万元A轮投资。<ref name=":0" />

2018年2月7日，[[36氪|36氪]]独家报道了[[今日头条|今日头条]]收购以Cosplay和手绘为主的[[用户生成内容|UGC]]社区半次元的新闻，而经查证半次元运营主体杭州王耀网络确实已于1月22日办理了工商资料变更登记手续。<ref>{{Cite news|url=http://36kr.com/p/5118331.html?ktm_source=feed|title=今日头条进军二次元，收购COS绘画小说社区“半次元”|author=|date=2018-02-07|work=36氪|accessdate=2018-03-29|language=|deadurl=no|archiveurl=https://web.archive.org/web/20180329221721/http://36kr.com/p/5118331.html?ktm_source=feed|archivedate=2018-03-29}}</ref>根据工商登记资料显示，杭州王耀网络的股东变更为北京闪星科技有限公司，法定代表人则由半次元创始人王伟变更为张利东。北京闪星科技是[[字节跳动|字节跳动]]的全资子公司，而张利东则是[[今日头条|今日头条]]的合伙人兼高级副总裁。<ref name=":0">{{Cite web|url=http://www.donews.com/technology/detail/29546101|title=收购“半次元”，今日头条和它的二次元布局|accessdate=2018-03-29|author=|date=2018-02-08|work=|publisher=donews科技|deadurl=no|archiveurl=https://web.archive.org/web/20180329221721/http://www.donews.com/technology/detail/29546101|archivedate=2018-03-29}}</ref>
