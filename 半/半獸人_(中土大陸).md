在[[J·R·R·托爾金|J·R·R·托爾金]]的小說中，'''半獸人'''（Orc）是[[中土大陸|中土大陸]]的一個邪惡[[中土大陸的種族|種族]]，數量眾多，先後成為[[魔苟斯|魔苟斯]]、[[索倫_(魔戒)|索倫]]和[[薩魯曼|薩魯曼]]的爪牙。

半獸人的起源最常見的解釋為，魔苟斯透過折磨那些被他所俘虜的[[精靈_(中土大陸)|精靈]]族，使他們扭曲墮落成為半獸人<ref name="艾倫·李">{{Cite book|title=紙上中洲：艾倫·李的《魔戒》素描集|last=艾倫·李|publisher=上海人民出版社|year=2015年8月|isbn=978-7-208-13055-5|location=上海}}</ref>{{Rp|130}}<ref>{{Cite book|title=魔戒魅影：魔戒首部曲小說·電影導讀本|last=裘德·費雪|publisher=聯經|year=2002年1月|isbn=957-08-2346-1|location=臺北}}</ref>{{Rp|64}}。半獸人汙穢而扭曲，他們懼怕陽光，[[血液|血液]]是黑色的<ref name="克理斯·史密斯">{{Cite book|title=魔戒武器聖戰|last=克理斯·史密斯|publisher=奇幻基地|year=2003年12月|isbn=986-7576-12-8|location=臺北}}</ref>。

托爾金曾描述半獸人的外表是「蜷伏着，骯髒的，大鼻子的，灰黄皮肤的，嘴巴寬闊，眼睛歪斜」<ref>托尔金书信，第210封</ref>。

== 名字 ==
托尔金曾说：“半兽人一词来自[[古英语|古英语]]的orc（一种恶魔），但只是引用了它的读法。” <ref>托尔金书信，第144封，1953年</ref>

《魔戒》的附錄中指出「Orc」一詞源自[[洛汗語|洛汗語]]，該詞在[[辛達林語|辛達林語]]中的對應詞是「Orch」，在[[黑暗語|黑暗語]]的對應詞則是「uruk」（即[[強獸人|強獸人]]），但事實上「uruk」（強獸人）這個詞只被用來稱呼[[第三紀元|第三紀元]]時期[[魔多|魔多]]和[[艾辛格|艾辛格]]培育出來的新種半獸人，而這些強獸人皆一律稱呼其他的半兽人為「snaga」（史那加，意為[[奴隸|奴隸]]）<ref name="王者再臨">{{Cite book|title=魔戒三部曲：王者再臨|last=托爾金|publisher=聯經|year=2012年12月|isbn=978-957-08-4102-2|location=臺北}}</ref>。

在2013年[[中國大陸|中國大陸]][[世纪文景|世纪文景]]出版的《魔戒》中文譯本當中將「Orc」譯為'''奥克'''，而非半獸人，譯者[[鄧嘉宛|鄧嘉宛]]表示她是依據托爾金訂下的翻译原则來翻譯的，因為「Orc」一詞有被規定必須音譯，以顯示異化感<ref>{{Cite web|url=http://www.chinanews.com/cul/2013/11-15/5505329.shtml|title=《魔戒》译者：托尔金语言特殊 需多注释|accessdate=2018-10-29|author=刘婷|date=2013-11-15|publisher=中新网（北京晨报）}}</ref>。

== 歷史 ==
[[第一紀元|第一紀元]]，半獸人是魔苟斯麾下的主要軍力，魔苟斯在[[安格班|安格班]]培養了大量的半獸人，用於跟精靈族戰鬥。在[[憤怒之戰|憤怒之戰]]中，半獸人幾乎全被消滅。[[第二紀元|第二紀元]]時期，曾是魔苟斯部下大將的索倫佔據魔多，半獸人再度為他效力。大量半獸人在[[最後聯盟戰役|最後聯盟戰役]]中陣亡。<ref>{{Cite book|title=精靈寶鑽|last=托爾金|first=|publisher=聯經|year=2015年1月初版第二十一刷|isbn=978-957-08-2524-4|location=臺北}}</ref>

第三紀元，不少半獸人聚集在[[迷霧山脈|迷霧山脈]]，以[[剛達巴山|剛達巴山]]為他們的首都，他們的領袖是[[阿索格|阿索格]]。由於阿索格殺死[[矮人_(中土大陸)|矮人]]國王索爾，引起矮人與半獸人之間的多年戰爭，迷霧山脈的大多數半獸人被殺死或驅逐，但一段時間後，半獸人又重新回歸。在[[孤山_(托爾金小說)|孤山]]任務期間，半獸人領袖[[波格_(中土大陸)|波格]]帶領剛達巴半獸人向孤山發動進攻，[[五軍之戰|五軍之戰]]爆發，中土大陸有三分之一的半獸人死於此役，換皮人[[比翁|比翁]]暫時肅清了在[[大荒原|大荒原]]一帶橫行的半獸人<ref name="王者再臨" /><ref name="哈比人小說">{{Cite book|title=哈比人|last=[[托爾金|托爾金]]|publisher=聯經|year=2014年1月初版第四刷|isbn=978-957-08-4103-9|location=臺北}}</ref>。

[[魔戒聖戰|魔戒聖戰]]期間，索倫正不斷集結半獸人大軍，以為佔領中土大陸作準備；也有部分的迷霧山脈半獸人為白袍[[巫師_(中土大陸)|巫師]][[薩魯曼|薩魯曼]]效力。半獸人大軍近乎參與了魔戒聖戰中的每一場戰役，包括[[艾辛河渡口|艾辛河渡口]]之役、[[聖盔谷之戰|聖盔谷之戰]]、[[羅瑞安之戰|羅瑞安之戰]]、[[帕蘭諾平原戰役|帕蘭諾平原戰役]]等。索倫在[[黑門之戰|黑門之戰]]中敗亡後，魔多半獸人近乎全部被殺，倖存者也逐漸消聲匿跡<ref name="王者再臨" />。

== 創作構想 ==
托尔金十分厭惡戰爭與科技，他的朋友喬治·塞爾曾提及當他們見到[[工業污染|工業污染]]的痕跡時，托尔金就會「談起半兽人或半兽人族」。另一次托尔金在論及[[第二次世界大戰|第二次世界大戰]]時說：「我們試圖征服擁有[[至尊魔戒|魔戒]]的索倫，我們將（看似）成功。但是，懲罰會培育出新的索倫們，並且慢慢將人類和精靈變為半獸人」<ref name="魔戒的鍛造者：托爾金傳">{{Cite book|title=魔戒的鍛造者：托爾金傳|last=柯林·杜瑞茲|publisher=黑龍江教育出版社|year=2015年2月|isbn=978-7-5316-7894-6|location=哈爾濱}}</ref>。

二戰期間，托爾金在寫給兒子[[克里斯托夫·托爾金|克里斯托夫]]的信中說：「我認為半獸人和『[[現實主義|現實主義]]』作品中的任何創造物一樣，是真實存在的」<ref>{{Cite web|title=托尔金痛恨战争，他在《魔戒》系列里写下了那些悲悯的句子|url=https://www.thepaper.cn/newsDetail_forward_1296182|accessdate=2019-01-29|date=2015-01-21|publisher=澎湃新闻}}</ref>。
