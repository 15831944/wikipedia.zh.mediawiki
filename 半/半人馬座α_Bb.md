'''半人馬座αBb'''（'''Alpha Centauri Bb'''），即'''南門二Bb'''，是環繞[[恆星光譜|光譜類型]] K 型主序星[[南門二|南門二B]]的一顆可能的[[系外行星|系外行星]]，距離地球約4.37光年，位於[[半人馬座|半人馬座]]。目前該行星的存在可能性已被推翻<ref name = "Dumusque">{{cite journal

一個來自歐洲的觀測者團隊於2012年10月聲稱發現該行星，消息廣為傳媒報導，不過有一些天文學家則抱持懷疑態度，認為該歐洲團隊把數據過度解讀。

至2015年10月，[[牛津大學|牛津大學]]的天文學家發表科學論文，推翻了該行星的存在可能性，他們把一些隨機產生的合成數據，與真實的天文數據進行相同的統計分析，分析發現得出相同的結果，令聲稱發現行星的論文作者杜穆斯克，不得不承認自己不能百分百肯定該發現，並稱該行星可能不存在。

==物理特性==
該行星並不是位於母恆星的[[適居帶|適居帶]]，並且距離母恆星只有0.04天文單位，公轉周期則只有3.236日<ref name = "Dumusque"/>。它的[[質量下限|質量下限]]至少是地球的1.13倍，但實際值尚待[[軌道傾角|軌道傾角]]確定後才能得知<ref name = "Dumusque"/>。它的表面溫度預估大約至少是1200[[摄氏温标|°C]]或1500 [[熱力學溫標|K]]<ref name="NYT">{{cite web | url=http://www.nytimes.com/2012/10/17/science/space/new-planet-found-in-alpha-centauri.html?_r=0 | title=New Planet in Neighborhood, Astronomically Speaking | publisher=New York Times | date=October 17, 2012 | accessdate=October 17, 2012 | author=Overbye, Dennis}}</ref><ref name="BBC">{{cite web | url=http://www.bbc.co.uk/news/science-environment-19959531 | title=Exoplanet around Alpha Centauri is nearest-ever | date=October 17, 2012 | accessdate=October 17, 2012 | author=Palmer, Jason | publisher=BBC}}</ref>，這遠高於液態水能存在的溫度，甚至高於許多含矽酸鹽岩石熔化成[[岩漿|岩漿]]的溫度。相較之下，太陽系中表面溫度最高的行星金星表面溫度是462°C或735 K。 

[[星震學|星震]]分析、[[色球|色球]]活動和恆星自轉（[[迴轉年代學|迴轉年代學]]）研究南門二A和B的結果，判定這兩顆恆星的年齡稍高於太陽系，大約是45到70億年<ref name=MH08>{{cite journal | author=E. E. Mamajek; L. A. Hillenbrand | title=Improved Age Estimation for Solar-Type Dwarfs Using Activity-Rotation Diagnostics | journal=Astrophysical Journal | year=2008 | bibcode=2008ApJ...687.1264M | doi=10.1086/591785 | volume=687 | issue=2 | page=1264 |arxiv = 0807.1686 }}Age discussion on page 1284</ref>。

==偵測==
南門二B的適居帶範圍大約是0.5到0.9天文單位。2009年時電腦模擬顯示行星在適居帶內最可能存在的位置是在適居帶內界線。模擬中必須要有特殊假設以得到距離恆星較遠，讓行星形成吸積過程易於進行的環境。然而，如果南門二A和B原本是在結構緊密的[[疏散星團|疏散星團]]內形成，則可能發生兩者形成時位置距離較遠，之後兩者逐漸靠近的狀況。這個狀況下適合行星形成的位置將距離南門二B較遠<ref name="Thebault 2009">{{cite journal

|arxiv = 0811.0673
