'''曹操讨伐袁术之战'''是[[三国|三国]]前的197年—199年的一场由汉朝朝廷发起對僭號稱帝的[[袁术|袁术]]做出的讨伐。最終以袁术及其[[仲家|仲氏]]政权败亡而告终。

== 背景 ==
[[董卓|董卓]]劫掠都城[[洛阳|洛阳]]时，象征君权的[[传国玺|传国玺]]丢失，作为[[董卓讨伐战|讨董联军]]一员的[[孙坚|孙坚]]意外在洛阳废墟中找到了传国玺并将其保管；当时孙坚为[[袁术|袁术]]部将，当袁术得知孙坚得到传国玺，便劫持孙坚的妻子，迫使孙坚将传国玺交给他。

大约196年，繼承父業的孙坚之子[[孙策|孙策]]攻打[[刘繇|刘繇]]、[[王朗|王朗]]等治下的[[江东|江东]]地区，使袁术自认为已稳稳掌控了[[华中|华中]]地区。197年初，袁术在[[淮南|淮南]]僭號稱帝，建立了[[仲家|仲氏]]朝。<ref>[[司马光|司马光]]《[[资治通鉴|资治通鉴]]》卷六十二：[獻帝建安二年]……袁術稱帝於壽春，自稱仲家，……</ref>袁术此举被视为叛汉，使孙策与其决裂，也给了其他军阀攻打他的理由；曹操控制下的汉廷向孙策和自称[[徐州|徐州]]牧的[[吕布|吕布]]下诏，敦促他们对袁术展开攻势。

== 过程 ==
得知袁术自称“天子”，孙策写信给袁术帐下的舅父[[广陵|广陵]]太守[[吴景|吴景]]和堂兄袁术部将[[孙贲|孙贲]]，要他们和袁术决裂，吴景、孙贲响应孙策并倒戈投靠之，袁术因此失去了广陵和孙策攻占的江东地区，对华中的影响力骤降，同时[[吕布|吕布]]在[[淮北|淮北]]击败袁术并在当地劫掠，袁术为扭转局势派军侵[[陳國_(東漢)|陈]]，被[[曹操|曹操]]率领的汉军击退，使袁术转移到淮南。<ref>[[陈寿|陈寿]]《[[三国志|三国志]]》卷一武帝纪：袁術欲稱帝於淮南，使人告呂布。布收其使，上其書。術怒，攻布，為布所破。秋九月，術侵陳，公東征之。術聞公自來，棄軍走，留其將橋蕤、李豐、梁綱、樂就；公到，擊破蕤等，皆斬之。術走渡淮。公還許。</ref>

当袁术的影响力已被削弱到微小时，反袁术联军出现了内讧，吕布在袁术要求下攻打刘备，曹操支持刘备反击，这场冲突导致了198年曹刘联军与吕布之间的[[下邳之战|下邳之战]]；吕布处境危险，求助袁术，但袁术仅派一千骑兵相助，未至下邳即被击败，眼看吕布灭亡不可避免，袁术派使者煽动[[山越|山越]]部落和匪徒头领[[祖郎|祖郎]]攻打孙策，然而孙策击破敌军，继续加强自己在江东的影响。

另一方面，袁术軍的處境恶化，他的仓库空虚，军力也被削弱，袁术焚烧宫室，逃入[[潜山|潜山]]投奔旧部[[雷薄|雷薄]]、[[陈兰|陈兰]]，但雷薄、陈兰拒绝接纳，袁术写信给哥哥[[袁绍|袁绍]]，许诺一旦袁绍肯助他就轉讓帝號，袁绍派儿子[[袁谭|袁谭]]迎袁术入[[青州|青州]]，袁术去青州途中，曹操派[[刘备|刘备]]和[[朱灵|朱灵]]截击，袁术只得返回淮南，最终在回都城[[寿春|寿春]]的途中病死。<ref>陈寿《三国志》卷一武帝纪：袁術自敗於陳，稍困，袁譚自青州遣迎之。術欲從下邳北過，公遣劉備、朱靈要之。會術病死。</ref>

== 后續 ==
袁术死后，其家人投奔旧部[[刘勋_(东汉)|刘勋]]。[[杨弘_(东汉)|杨弘]]和[[张勋_(东汉)|张勋]]计划率本部投靠孙策，却在[[庐江|庐江]]被刘勋截留。199年，孙策败刘勋，克庐江，释放了袁术的家人和部众。

== 演义 ==
曹操讨伐袁术之战在[[罗贯中|罗贯中]]的历史小说《[[三国演义|三国演义]]》第十七回中表现如下：

袁术派[[张勋_(东汉)|张勋]]率军二十万攻吕布于[[徐州|徐州]]。军队分七路：张勋领中军，[[桥蕤|桥蕤]]、雷薄、[[韩暹|韩暹]]居左，[[陈纪_(袁术)|陈纪]]、陈兰、[[杨奉|杨奉]]居右。每个将领都分别受命攻陷特定城镇。

吕布由哨探得知张勋的目标是攻克徐州；其他被攻打的城镇有[[小沛|小沛]]、[[沂都|沂都]]、[[琅琊|琅琊]]、[[碣石|碣石]]、[[下邳|下邳]]和[[浚山|浚山]]。袁术军每日前进二十里，于路劫掠乡野。当吕布与谋士商议时，[[陈宫|陈宫]]指责[[陈珪_(东汉)|陈-{珪}-]]和[[陈登|陈登]]招祸。但陈登却笑陈宫懦弱，并献击败袁术之计。吕布同意了，并命其实施。吕布上表朝廷传达对付袁术的战略。

吕布追击，遭遇袁术亲自率军。马背上的袁术身披金甲，腕悬两刀，厉声责骂吕布并派部将[[李丰_(东汉)|李丰]]出战。战不三合，李丰手受伤，弃枪而走。吕布麾兵冲杀并获胜，敌军逃散，丢下马匹衣甲无数。袁术军没跑远就遭遇[[关羽|关羽]]所部军队，伤亡惨重，仅率少数士兵逃走。吕布获胜后和关羽及背叛袁术来投的韩暹、杨奉回徐州。吕布设宴庆功并保举杨奉为琅琊牧，韩暹为沂都牧。在陈-{珪}-建议下，他们立即各自赴任。

战败的袁术遭到曹操、吕布、孙策围攻。袁术乏粮，劫掠[[陈留|陈留]]。曹操亲率军队攻袁术，并写信给刘备、吕布、孙策，要他们协同进攻。刘备首先响应，与曹操会师时展示韩暹、杨奉的首级，称他们在徐州任上纵兵掠民。曹操感谢刘备除掉了他们。吕布到后，曹操善言抚慰，许诺回到[[许昌|许昌]]后就给以徐州牧官印，吕布大喜过望。三军合兵，曹操领中军，吕布在左，刘备在右，[[夏侯惇|夏侯惇]]、[[于禁|于禁]]为先锋。

桥蕤率袁术先锋军在寿春界口遭遇夏侯惇。夏侯惇出马，战不三合杀死桥蕤，桥蕤军撤入城中。孙策军奔西面而来，即将到达。其余三军也各攻一面，曹操攻北面，吕布攻东面，刘备攻南面。袁术见身处绝境，命李丰、[[乐就|乐就]]、[[梁纲|梁纲]]、陈纪率军十万守寿春，收拾库藏率余部渡淮出逃。

曹操军每日需要大量粮食，寿春周边又遭数年饥荒，无以接济。因此，曹操催促军队速战，寿春守军却试图尽量拖延战事。围城一月有余，毫无进展。曹操粮食将尽，写信向孙策借得粮米十万斛。此后曹操的战略确保联军不停攻城以尽早结束围城。联军登城破门，被围的守军被俘。李丰、乐就、梁纲、陈纪都被生擒并当众处决。袁术政权所造的宫室殿宇和犯禁之物都被焚毁，城池被劫掠一空。袁术未被追杀，三军也各自班师。
