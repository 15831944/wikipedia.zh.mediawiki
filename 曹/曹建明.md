'''曹建明'''（{{bd|1955年|9月24日|catIdx=C曹}}），[[江苏|江苏]][[南通|南通]]人，出生于[[上海|上海]]；[[华东政法大学|原华东政法学院]]法律系毕业，法学硕士，教授；[[比利时|比利时]][[根特大学|根特大学]]名誉法学博士。1973年12月加入[[中国共产党|中国共产党]]；是中共第十六届[[中央候补委员|中央候补委员]]，第十七、十八、十九届中央委员。曾为[[中华人民共和国一级大法官|中华人民共和国一级大法官]]（时任[[中华人民共和国最高人民法院副院长|中华人民共和国最高人民法院常务副院长]]）、[[中华人民共和国最高人民检察院检察长|中华人民共和国最高人民检察院检察长]]（[[中华人民共和国首席大检察官|中华人民共和国首席大检察官]]）；现任第十三届[[全国人大常委会副委员长|全国人大常委会副委员长]]。

现任妻子是[[央视|央视]]著名主持人[[王小丫|王小丫]]<ref>[http://wenhua.youth.cn/kt/201208/t20120813_2352657_19.htm 44岁王小丫有两次婚姻仍未孕内幕 曾与潘虹争夫]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==早年经历==
曹建明出生在上海的一个普通干部家庭；父亲从军队转业后，曾在地方一家国营工厂当了30年的厂长；母亲则在某国营棉纺织厂，长期从事基层党务工作。“[[文化大革命|文化大革命]]”开始后，曹的父亲被打成“走资派”。因家庭出身问题，中学毕业后的曹建明进入一家国营饮食店当学徒工。1975年到1976年间，曹建明先是在[[安徽|安徽]]参与“小三线”建设；后又投入到[[唐山大地震|唐山大地震]]后的城市重建工作，一直过着漂泊的生活，直到文革结束。

==研习法律==
[[高考|高考]]制度恢复后，已经中断学习六年多的曹建明决心考大学。1979年，他踩着[[华东政法学院|华东政法学院]]的最低录取分数线，进入该校法学院。四年本科学习结束后，又攻读国际法研究生。在华东政法学院学习的七年时间里，曹建明有六年都被评为“上海市[[三好學生|‘三好’学生”]]。

1986年研究生毕业後，曹建明留校任教；历任国际法系教师，系副主任、主任；期间，还曾到[[比利时|比利时]]根特大学法学院进修。在留学比利时期间，[[六四事件|六四事件]]的发生，促使很多中国海外[[留学生|留学生]]希望能够留在当地。但是，在一年学习结束后，曹建明坚持回到了中国。回国后不久，曹建明被破格晋升为副教授；并于1990年8月，被派赴[[美国|美国]][[旧金山大学|旧金山大学]]法学院，作为访问教授进行短期教学工作。1991年，再次被破格晋升为教授，成为当时中国最年轻的法学教授，时年36岁。<ref>[http://211.151.90.1/newchisa/web/5/2004-04-07/news_30703.asp 曹建明：我与祖国息息相关]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }} 神州学人</ref> 

1994年12月9日，作为国际法的研究学者，曹建明受邀到[[中南海|中南海]]，为当时中共领导人[[江泽民|江泽民]]、[[乔石|乔石]]、[[李瑞环|李瑞环]]、[[胡锦涛|胡锦涛]]等讲授有关《国际商贸法律制度与关贸总协定》的课题；由此，进入中共高层领导人的视野。一年后，即被安排出任[[华东政法学院|华东政法学院]]常务副院长；1997年，晋升为院长。1998年5月12日，曹建明第二次进入[[中南海|中南海]]，为中共领导讲授有关《金融安全与法制建设》的课题。<ref>[http://www.law-lib.com/fxj/fxj.asp?id=763 曹建明——法律学人] 西湖法律书店</ref>

== 最高人民法院 ==
1999年11月，曹建明被[[第九届全国人大常委会|第九届全国人大常委会]]任命为[[中华人民共和国最高人民法院副院长|中华人民共和国最高人民法院副院长]]；同时兼任[[国家法官学院|国家法官学院]]院长，[[中国法学会|中国法学会]]审判理论研究会会长，[[中国国际法学会|中国国际法学会]]副会长（後被选为会长，任职高检後退为名誉会长），[[中国知识产权研究会|中国知识产权研究会]]副理事长，[[中国法官协会|中国法官协会]]副会长等职<ref>[http://news.xinhuanet.com/ziliao/2002-02/28/content_295052.htm 曹建明简历] 新华网</ref>；2002年，升任最高人民法院常务副院长。

== 最高人民检察院 ==
2008年3月16日，在[[第十一届全国人民代表大会第一次会议|第十一届全国人民代表大会第一次会议]]上，53岁的曹建明當選為[[中华人民共和国最高人民检察院检察长|中华人民共和国最高人民检察院检察长]]，晋升副国级。2013年3月15日在[[第十二届全国人民代表大会第一次會議|第十二届全国人民代表大会第一次會議]]上获得連任，任期至2018年3月18日结束。

== 全国人大常委会副委员长 ==
2018年3月17日，当选第十三届[[全国人大常委会副委员长|全国人大常委会副委员长]]<ref>{{cite web|title=十三届全国人大一次会议选举产生全国人大常委会副委员长、秘书长|url=http://www.xinhuanet.com/2018-03/17/c_1122550486.htm|website=新华网|accessdate=2018-03-17}}</ref>。

==学术成就==
曹建明曾先后主编、参与编辑出版《国际经济法概论》、《国际产品责任法概论》、《国际经济法新论》等著作、辞书共28部；并在国内外报刊杂志上发表论文两百多篇。
