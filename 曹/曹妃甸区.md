'''曹妃甸区'''，是[[中国|中国]][[河北省|河北省]][[唐山市|唐山市]]所辖的一个[[市辖区|市辖区]]。原为[[唐海县|唐海县]]，2012年7月，国务院批准撤销唐海县设立曹妃甸区，和[[丰南区|丰南区]][[滨海镇_(唐山市)|滨海镇]]合并，<ref>{{Cite web |url=http://news.163.com/12/0914/11/8BC1ILSH00014JB6.html |title=河北省唐山市部分行政区划调整前后示意图公布 |accessdate=2012-09-14 |publisher=网易新闻 |date=2012-09-14 }}</ref>设立曹妃甸区<ref>{{Cite web |url=http://www.heb.chinanews.com/tangshan/22/2012/0728/47498.shtml |title=唐山市撤销唐海县设立曹妃甸区 |accessdate=2012-07-29 |publisher=燕赵晚报 |date=2012-07-28 }}</ref>。总面积为725平方公里（原唐海县），2003年人口为14万人。

曹妃甸区人民政府驻[[唐海镇|唐海镇]]垦丰大街23号。

==历史==
原為中國河北省唐山市以南80千米处一座帶狀的小沙島，据说当年唐太宗东征时,一为姓曹的妃子死于此地，因此建立了一座曹妃廟，岛因此得名。自1993年开始筹建一個中國新工業區，2005年10月8日正式成立管委会，作为唐山市政府的派出机构，开始了大规模的建设工作。2006年7月29日，胡錦濤視察[[曹妃甸|曹妃甸]]時指出：“曹妃甸是一塊黃金寶地”。2012年设立唐山市曹妃甸区。

[[File:河北省唐山市部分行政区划调整前后示意图.jpg|thumb]]

2012年7月，国务院批准设立唐山市曹妃甸区，行政区划包括原来的唐海县全境、曹妃甸工业区、生态城、柳赞镇、滨海镇。

2015年1月，曹妃甸工业区党工委委员、管委会副主任王永山涉嫌违纪违法，正在接受组织调查<ref>{{cite news|title=河北唐山曹妃甸工业区管委会副主任王永山被查|url=http://www.chinanews.com/fz/2015/01-06/6941755.shtml|accessdate=2016-07-04}}</ref>。
2016年，[[天津市人民政府|天津市政府]]与[[河北省人民政府|河北省政府]]联合申报将曹妃甸区纳入天津自贸试验区，成为天津自贸区河北曹妃甸片区<ref>{{cite news|title=天津自贸区将建曹妃甸片区 推进京津冀深度融合|url=http://news.enorth.com.cn/system/2016/07/25/031079651.shtml|accessdate=2016-08-05|agency=北方网|date=2016-07-25}}</ref>。

== 南堡油田 ==
{{main|南堡油田}}
南堡油田，隶属于[[中国石油天然气集团公司|中国石油天然气集团公司]]，是位于曹妃甸区的[[油田|油田]]。2007年5月3日，中石油高调宣称冀东南堡油田油气总储量为10.2亿吨，探明储量4.05亿吨，约为30亿桶，媒体称之为“40多年来中国石油勘探最激动人心的发现”。但数月之后，被证实其实际储量远小于这一数据，该油田经济可采储量仅为6.35亿桶。<ref>{{cite web | url=http://www.21cbh.com/HTML/2010-5-28/5NMDAwMDE3OTU5NQ.html | title=中石油冀东南堡油田储量或少于10亿吨预期 | publisher=21世纪网 | accessdate=2016-07-04 | deadurl=yes | archiveurl=https://web.archive.org/web/20100608040021/http://www.21cbh.com/HTML/2010-5-28/5NMDAwMDE3OTU5NQ.html | archivedate=2010-06-08 }}</ref>2010年5月，[[路透社|路透社]]报道，冀东南堡油田储量低于预期。<ref>{{cite web | url=http://cn.reuters.com/article/CNAnalysesNews/idCNCHINA-2379920100527 | title=冀东大油田储量低于预期 中国仍将依赖石油进口 | publisher=路透社 | accessdate=2016-07-04}}</ref>此后，关于冀东南堡油田的报道越来越少。2013年，[[蒋洁敏|蒋洁敏]]被双规后，南堡油田被认为是其在中石油任内的“注水政绩”。<ref>{{cite web | url=http://news.qq.com/a/20130917/006891.htm | title=蒋洁敏境外运油重庆建厂 为薄熙来政绩做文章 | publisher=腾讯网 | accessdate=2016-07-04}}</ref>南堡油田的储量骗局令曹妃甸区陷入困局，香港媒体指蒋洁敏是骗局的罪魁祸首<ref>{{cite news|title=南堡油田造假蔣潔敏屬禍首|url=http://hk.apple.nextmedia.com/realtime/7015342/20131008/51794468|accessdate=2016-07-04|agency=香港苹果日报}}</ref>。
