'''曹玉清'''（{{bd|1906年||1991年|4月28日}}）[[侗族|侗族]]，[[湖南省|湖南省]][[新晃侗族自治县|新晃县]]人，[[中国人民解放军开国少将|中国人民解放军开国少将]]。

==生平==
1926年参加[[北伐战争|北伐战争]]。1929年参加[[中国工农红军|中国工农红军]]，同年加入[[中国共产党|中国共产党]]。[[第一次国共内战|第一次国共内战]]中，参加和指挥了[[鄂豫皖苏区|鄂豫皖苏区]]第一、二、三、四次反围剿的战斗，参加了[[二万五千里长征|二万五千里长征]]。[[抗日战争|抗日战争]]时期，参与领导开辟了[[洪湖根据地|洪湖根据地]]。[[第二次国共内战|第二次国共内战]]时期，参加和领导了[[中原突围|中原突围]]、两淮保卫战、[[枣庄战役|枣庄战役]]、[[莱芜战役|莱芜战役]]、[[孟良崮战役|孟良崮战役]]和解放安阳、新乡、邯郸的战斗。他历任分队长、连长、副营长、团长、团政委、旅长、师长、山西省兵役局局长、[[山西省军区|山西省军区]]参谋长、山西省军区副司令员、山西省军区顾问等职。1955年，被授予[[中国人民解放军少将|中国人民解放军少将]]军衔<ref name=cpc/><ref>{{cite news|title=揭秘：中共首批开国将军的级别是怎样划分的？ |url=http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml |accessdate=2012-06-15 |newspaper=凤凰网（转自摘自《党史博览》2010年第5期） |date=2010-06-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20120620091504/http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml |archivedate=2012-06-20 }}</ref>。

1991年4月28日，曹玉清在[[北京|北京]]病逝，享年85岁<ref name=cpc>{{Cite web |url =  http://dangshi.people.com.cn/GB/165617/165618/166491/167909/12263282.html|title =曹玉清  |author =  |date =  |publisher = 中国共产党新闻网|language =  |accessdate =  2017-04-29|quote =  }}</ref>。
