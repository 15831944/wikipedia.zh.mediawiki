'''曹仲达'''（{{bd|882年||943年|12月8日|catIdx=C}}<ref name=SGCQ86>[https://archive.org/stream/06080961.cn#page/n140/mode/2up 《十国春秋》卷八十六]</ref><ref name=SGCQ80>[https://archive.org/stream/06075838.cn#page/n118/mode/2up 《十国春秋》卷八十]</ref>），本名'''曹弘达'''，[[中国|中国]][[五代十国|五代十国]]时期[[吴越国|吴越国]][[官员|官员]]，在第二任国王[[钱元瓘|钱元瓘]]和第三任国王[[钱弘佐|钱弘佐]]年间为[[宰相|宰相]]。

== 背景和效力钱镠 ==
父[[曹圭|曹圭]]，是后来建立吴越国的[[钱镠|钱镠]]的部将。[[唐僖宗|唐僖宗]][[中和_(唐朝)|中和]]二年（882年），曹弘达生于[[临平|临平]]。其母分娩时，室内有紫光，家人都以为异。长大后，曹圭想锻炼他的身体，让他和仆人们吃同样的饭，即使严冬也不给他足够保暖的衣物。曹圭还每天命他搬砖，以磨炼筋骨。<ref name=SGCQ86/>

不久，曹弘达在钱镠治下的[[镇东军|镇东军]]为押牙。时曹圭为[[苏州|苏州]][[刺史|刺史]]，请求与[[睦州_(仁寿)|睦州]]刺史[[陈询_(唐朝)|陈询]]结亲，<ref>曹圭于898年任苏州刺史，907年唐朝灭亡时仍在任；陈询任睦州刺史在900年—905年间。</ref>由曹弘达娶陈询的女儿。于是他派曹弘达去睦州，想让他在睦州亲自迎亲成婚。但有占卜者说：“陈氏不是合适的结亲对象，应该和其他家族结亲以致荣显。”曹弘达行经钱镠主要领地[[镇海军|镇海军]]军部[[钱塘|钱塘]]，钱镠奇其貌，把二妹嫁给了他。<ref>《[[钱氏家乘|钱氏家乘]]》，将曹弘达误作'''曹宏逵'''。</ref>历任[[台州|台州]]、[[处州|处州]]刺史。<ref name=SGCQ86/>

== 效力钱元瓘 ==
[[宝正|宝正]]七年（932年）三月，时为吴越国王的钱镠去世。子钱元瓘继位。钱元瓘起初不称国王，仅作为[[后唐|后唐]]封臣称镇海军、镇东军节度使。他命曹弘达权知政事，为实际上的宰相，仅因钱元瓘未称王而没有宰相名义。<ref name=ZZTJ277>[[:s:資治通鑑/卷277|《资治通鉴》卷二百七十七]]</ref>[[天福_(后晋)|天福]]二年（937年）四月，时为取代后唐的[[后晋|后晋]]封臣的钱元瓘称国王，任曹弘达、[[沈崧|沈崧]]、[[皮光业|皮光业]]为[[丞相|丞相]]。<ref name=ZZTJ281>[[:s:資治通鑑/卷281|《资治通鉴》卷二百八十一]]</ref>同时他立[[钱弘僔|钱弘僔]]为世子，曹弘达大约此时为[[避讳|避讳]]改名曹仲达。曹仲达性仁厚，喜欢布施食物。钱元瓘很器重他，经常呼为丞相而不呼其名。<ref name=SGCQ86/>

== 效力钱弘佐 ==
天福六年（941年）九月，钱元瓘去世，因钱弘僔已先卒，由钱弘僔的弟弟钱弘佐继位。钱弘佐居丧期间，曹仲达摄行军府事。当时军队大受赏赐，因认为赏赐不均而哗然拒绝受赐，诸将不能制止。曹仲达亲自出去告谕，言辞流畅，士兵放下武器作罢。<ref name=ZZTJ282>[[:s:資治通鑑/卷282|《资治通鉴》卷二百八十二]]</ref>八年（943年），曹仲达去世，谥'''安成'''。<ref name=SGCQ80/>
