'''曹姓'''为[[中國姓氏|中國姓氏]]之一，在[[百家姓|百家姓]]中排第26。此[[姓氏|姓氏]]在[[韓國|韓國]]亦有分佈。但「曹」字在[[韓語|韓語]][[漢字|漢字]]作「-{曺}-」，但在[[異體字|異體字]]。

==來源==
曹姓在[[中國|中國]]的來源有以下几個：
# 出自古曹姓，[[陆终|陆终]]第五子[[晏安|晏安]]，分化为[[祝融八姓|祝融八姓]]中的曹姓。
# 出自[[姬姓|姬姓]]，[[周文王|周文王]]第十三子[[曹叔振鐸|叔振鐸]]，受封於曹（今[[山東省|山東省]][[定陶縣|定陶縣]]西南一帶），建立[[曹國_(諸侯國)|曹國]]，後為[[宋國|宋國]]所滅，子孫便以國為氏。
# 出自[[姒姓|姒姓]]。三國[[曹丕|魏文帝曹丕]]之祖父[[曹嵩|曹嵩]]本姓夏侯氏, 夏侯氏是本來夏王朝[[姒姓|姒姓]]中一。
# 出自[[昭武九姓|昭武九姓]]的[[曹國_(昭武九姓)|曹國]]（今[[烏茲別克|烏茲別克]][[撒馬爾罕|撒馬爾罕]]東北一帶）。
#出自中國國民黨對臺政策對臺灣島原住民的漢化改姓，部份人改姓曹。

==郡望堂号==
===郡望===
曹姓在长期发展中，形成许多郡望，最著名的当数：<br />
*[[谯郡|谯郡]]：东汉末年从沛郡分出置郡。治所在亳州（今[[安徽省|安徽省]][[亳州市|亳州市]]）。治辖相当于今天的安徽、河南两省的灵璧、蒙城、太和、鹿邑、永城之间的地方。
*[[彭城郡|彭城郡]]：西汉时设置，东汉时改为彭城国。治所在彭城（今江苏省徐州市）。治辖大约在今天的山东省微山县，江苏省徐州市、沛县东南部。
*[[高平郡|高平郡]]：晋时设置，治所在今山东省金乡。治辖相当于今天的山东独山湖、金乡、巨野、邹城之间的地方。后来有多次变迁。汉时设置了巨野县，在今天的山东西南部、万福河北岸。

*[[曹仁|曹仁]]：東漢末軍事家，三國時期曹魏重要將領，官至大司馬
*[[曹丕|曹丕]]：魏國建立者，魏文帝，曹操子
*[[曹叡|曹叡]] :  魏明帝，曹丕子
*[[曹昂|曹昂]] :  曹操長子
*[[曹彰|曹彰]] :  曹操子，曹魏將軍
*[[曹植|曹植]]：三国时代著名诗人，曹操子，曹丕弟
*[[曹冲|曹冲]]：著名神童，曹操子
*[[曹訓|曹訓]]：三國曹魏重臣
*[[曹爽|曹爽]]：三國時代曹魏大將軍
*[[曹妙达|曹妙达]] : [[粟特|粟特]]人，北齐至隋文帝时期的宫廷音乐家
*[[曹善才|曹善才]] : 西域[[昭武九姓|昭武九姓]]的曹国人，唐代琵琶演奏家
*[[曹松|曹松]]：唐昭宗时诗人
*[[曹霸|曹霸]]：唐代画家，據說是曹操後裔

*[[曹贵妃_(西夏崇宗)|曹氏]]，西夏仁宗之母

*[[曹國舅|曹國舅]]：傳說中的八仙之一

*[[曹雪芹|曹雪芹]]：清代文學家，《红楼梦》作者
*[[曹謹|曹謹]]：清代政治家，於福建[[臺灣府|臺灣府]][[鳳山縣|鳳山縣]]知縣時任內建造著名水利工程《[[曹公圳|曹公圳]]》
