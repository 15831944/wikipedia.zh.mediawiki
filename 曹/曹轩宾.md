'''曹轩宾'''（{{bd|1980年|7月19日||}}），[[西安|西安]]人，是中国音乐人、华语流行歌手、音乐制作人。他集演唱、作曲、编曲和制作于一身，曲风细腻柔和，以抒情慢歌见长。他所属的经纪公司为[[海蝶音樂|海蝶音樂]]。

== 生平 ==
毕业于[[中央音乐学院|中央音乐学院]]。他曾在幕后工作了近8年，为华语流行乐坛创作了许多脍炙人口的歌曲，如《可惜不是你》、《爱因为在心中》等。

2011年参加[[山东卫视|山东卫视]]音乐节目《[[歌声传奇|歌声传奇]]》受到广泛关注和支持。2013年他与[[张含韵|张含韵]]联合演唱电影《[[初恋未满|初恋未满]]》的同名主题曲「初恋未满」<ref>{{cite web |url=http://m2.21cn.com/newdisc/listen/a/2013/0607/15/22082618.shtml |language= |title=试听：张含韵/曹轩宾《初恋未满》 |publisher=21CN音乐 |date=2013-06-07 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20150512162555/http://m2.21cn.com/newdisc/listen/a/2013/0607/15/22082618.shtml |archivedate=2015-05-12 }}</ref>。官方[[MV|MV]]由[[鲍比达|鲍比达]]全程监制录制，MV在首播不到两天的时间里获点击播放超过一百万次<ref>{{cite web |url=http://ent.dzwww.com/yulezhuanti/yanpo/201306/t20130621_8581392.htm |language= |title=《初恋未满》MV点击过百万 电影未映歌先红(图) |publisher=新华娱乐 |date=2013年6月21日 |accessdate= |deadurl=yes |archiveurl=https://archive.is/20130624141919/http://ent.dzwww.com/yulezhuanti/yanpo/201306/t20130621_8581392.htm |archivedate=2013年6月24日 }}</ref>。
