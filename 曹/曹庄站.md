'''曹庄站'''位于[[天津市|天津市]][[西青区|西青区]][[外环线|外环线]]与阜盛道交口，该站为[[天津轨道交通|天津轨道交通]][[天津地铁二号线|天津地铁二号线]]车站之一。站台形式为侧式站台，地面段设有曹庄停车场。今后，[[天津地铁二号线|二号线]]将西延至[[杨柳青|杨柳青]]。

== 车站构造 ==

== 车站出入口 ==
该车站共设有两个出入口，均没有标识。
*{{港鐵出口|A}}--停车场、曹庄车辆段
*{{港鐵出口|B}}--阜盛道、永旺梦乐城（西青店）

== 换乘公交 ==
