'''曹炳良'''（{{bd|1951年|7月||}}）[[山东|山东]][[新泰|新泰]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员。

==生平==
曹炳良是[[中国共产党|中国共产党]]党员，[[河北地质学院|河北地质学院]]经济管理专业毕业，本科学历。1969年入伍，在[[兰州军区|兰州军区]]空军工程兵第十二团。1975年，进入[[中国人民解放军基本建设工程兵|基建工程兵]]水文地质部队第九十二支队（师）。1984年，由部队正团职转业至[[中华人民共和国地质矿产部|中华人民共和国地质矿产部]]任处长。

1989年，任[[中国老龄协会|中国老龄问题全国委员会]]办公室副主任。1992年，任中国老龄问题全国委员会副秘书长、办公室主任。1993年，当选中国老年学学会第二届理事会常务理事。1994年4月，任[[全国人大|全国人大]]《[[中华人民共和国老年人权益保障法|中华人民共和国老年人权益保障法]]》起草小组成员。1995年3月，当选中国质量协会第六届理事会理事。1996年1月，任[[中国老龄协会|中国老龄协会]]办公室主任。1999年10月，任[[全国老龄工作委员会|全国老龄工作委员会]]联络员、中国老龄协会办公室主任。2001年8月，任全国老龄工作委员会办公室（中国老龄协会）综合部主任。2002年4月，任[[联合国|联合国]]第二次世界老龄大会中国政府代表团成员。2003年10月，当选中国老龄事业发展基金会第二届理事会常务理事。

2004年1月，任全国老龄工作委员会办公室主任助理、中国老龄协会会长助理。2004年9月，任中国老龄协会副会长。2004年11月，任全国老龄工作委员会办公室副主任、中国老龄协会副会长<ref>{{Cite web |url = http://www.gov.cn/zhibo39/2006-12/12/content_467235.htm |title = 全国老龄工作办公室副主任曹炳良 |author =  |date = 2016-12-12 |publisher = 中国政府网|language =  |accessdate =  |quote =  }}</ref>。2005年4月，任中国榜样公益活动组委会副主任。2006年3月，任《中国老龄事业的发展》（[[白皮书|白皮书]]）起草领导小组组长。2007年5月，任《中华人民共和国老年人权益保障法》修订工作领导小组副组长兼办公室主任<ref>{{Cite web |url =http://www.beijingreview.com.cn/zgsy/txt/2007-12/12/content_90207.htm  |title =曹炳良：新“老年法”将强化政府责任  |author =  |date =2007-12-12  |publisher =北京周报  |language =  |accessdate =  |quote =  }}</ref>。2007年11月，当选[[中国联合国协会|中国联合国协会]]第四届理事会理事。2008年2月，任[[联合国社会发展委员会|联合国社会发展委员会]]第46届大会中国政府代表团顾问<ref>{{Cite web |url = http://www.fmprc.gov.cn/ce/ceun/chn/lhghywj/fyywj/2008/t406854.htm   |title = 中国代表团顾问曹炳良在联合国社会发展委员会第46届会议关于“审查与各社会群体状况有关的联合国行动计划和行动纲领”的发言 |author =  |date = 2008-02-11 |publisher = 中华人民共和国外交部 |language =  |accessdate =  |quote =  }}</ref>。2008年11月，任全国老龄工作委员会办公室党委常委、副主任，中国老龄协会第一副会长，华龄国际培训交流中心理事长。2009年1月，当选中国老龄事业发展基金会第三届理事会理事。2009年2月，任全国老龄工作委员会办公室党组成员、副主任、中国老龄协会第一副会长、华龄国际培训交流中心理事长。2009年3月，任《中华人民共和国老年人权益保障法》立法项目起草工作小组组长。2009年10月，任国家应对人口老龄化战略研究秘书组组长。2010年5月，任全国老龄工作委员会办公室党组成员、副主任、中共全国老龄工作委员会办公室直属机关委员会书记（2010年5月10日由民政部直属机关党委批准任命<ref>{{Cite web |url = http://www.cncaprc.gov.cn/contents/766/156310.html |title = 关于转发民政部直属机关党委《关于曹炳良等同志任职的通知》的通知 |author =  |date =2010-05-20  |publisher =全国老龄工作委员会办公室  |language =  |accessdate =  |quote =  }}</ref>），中国老龄协会第一副会长、华龄国际培训交流中心理事长。

2011年12月，任中国老龄产业协会副会长<ref>{{Cite web |url = http://hlj.ifeng.com/news/detail_2014_10/28/3070045_0.shtml |title =黑龙江省养老独具四大优势 30处省直单位房产招商办养老  |author =  |date = 2014-10-28 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。此后兼任国家应对人口老龄化战略研究秘书组组长、华龄国际培训交流中心理事长、中国联合国协会理事、中国老龄事业发展基金会理事、中国榜样公益活动组委会副主任。
