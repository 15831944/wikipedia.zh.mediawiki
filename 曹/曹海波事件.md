== 当事人 ==
曹海波，出生于[[江苏省|江苏省]][[盐城市|盐城市]]，27岁；妻子张念，[[四川省|四川省]][[宜宾市|宜宾市]]人，24岁，夫妻结婚之后在[[云南省|云南省]][[昆明市|昆明市]]工作。<ref name=" fish" /><ref name="tiger " />

== 事件经过 ==
=== 背景 ===
2010年11月，身为[[网吧|网吧]]管理员的曹海波在云南省昆明市用QQ建立QQ群，取名叫做“振华会”，同时还建立了同名网站，组建了“中国共和党”，主要宣传[[民主|民主]]、[[宪政|宪政]]、[[三民主义|三民主义]]。<ref name=" fish" /><ref name="bull " /><ref name="tiger " /><ref name=" nanwei" >{{cite web |url= http://www.peacehall.com/news/gb/china/2012/11/201211012314.shtml|title=曹海波宣扬民主理念被判监八年
 |accessdate= 2012年11月1日|publisher= 博讯网|archiveurl=http://www.peacehall.com/news/gb/china/2012/11/201211012314.shtml |archivedate=2012年11月8日 |work= |language =zh-cn}}</ref>

=== 逮捕 ===
2011年10月21日，曹海波被昆明市西山区20多个警察抄家搜罗罪证“[[电话机|电话机]]3个、[[电脑|电脑]]1台、[[U盘|U盘]]1个”之后抓走。<ref name="bull " /><ref name="tiger " /><ref name=" nanwei" /><ref>{{cite web |url=http://www.peacehall.com/news/gb/china/2011/10/201110211625.shtml |title=民主人士曹海波先生今天中午12点被公安带走! |accessdate=2011年10月21日 |publisher=博讯网 |archiveurl=http://www.peacehall.com/news/gb/china/2011/10/201110211625.shtml |archivedate=2012年11月8日 |work= |language =zh-cn}}</ref>

2011年11月25日，云南省昆明市西山区分局公安正式用“[[煽动颠覆国家政权罪|煽动颠覆国家政权罪]]”罪名逮捕曹海波，羁押在西山区看守所。<ref name="bull " /><ref name="tiger " />

=== 判决 ===
2012年5月22日，云南省昆明市一家法院秘密对曹海波案件进行一审审判，并且禁止其妻子张念以及其他所有人进入法庭，只有曹海波的辩护律师马晓鹏，理由是涉及中华人民共和国国家机密，判决结果未予对外公布。<ref name="bull " /><ref name="tiger " /><ref name=" nanwei" /><ref>{{cite web |url=http://www.peacehall.com/news/gb/pubvp/2012/05/201205281011.shtml |title=妻儿不得旁听，对曹海波的审判见不得光？ |accessdate= 2012年5月28日|publisher=博讯网 |archiveurl=http://www.peacehall.com/news/gb/pubvp/2012/05/201205281011.shtml |archivedate=2012年11月8日 |work= |language =zh-cn}}</ref>

2012年10月，昆明市中级人民法院宣布判处曹海波有期徒刑8年，判决书至今没有传递给家属和律师。<ref name="bull " /><ref>{{cite web |url=http://www.peacehall.com/news/gb/china/2012/11/201211011705.shtml |title=振华会创始人曹海波被以“颠覆罪”秘密判刑8年
 |accessdate=2012年11月1日 |publisher=博讯网 |archiveurl= http://www.peacehall.com/news/gb/china/2012/11/201211011705.shtml|archivedate=2012年11月8日 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://www.rfa.org/mandarin/yataibaodao/hc-11052012145710.html |title=昆明网吧老板曹海波因言获刑 律师正在考虑提出上诉 |accessdate=2012-11-05
