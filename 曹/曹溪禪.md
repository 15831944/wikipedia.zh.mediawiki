'''曹溪禪'''，是[[佛教|佛教]][[禪宗宗派|禪宗宗派]]之一，又稱為[[南宗|南宗]]，為[[惠能|曹溪惠能]]禪師所開創。'''曹溪禪'''所谓「一花開五葉，结果自然成」，[[晚唐|晚唐]]至[[北宋|北宋]]初期，共分[[临济宗|临济宗]]、[[曹洞宗|曹洞宗]]、[[雲門宗|雲門宗]]、[[法眼宗|法眼宗]]、[[沩仰宗|沩仰宗]]五宗，再加上臨濟門下分出的[[黃龍派|黃龍派]]、[[楊岐派|楊岐派]]兩派，合稱五家七派，为[[唐朝|唐朝]]以后的[[禪宗|禪宗]]主流。但[[宋朝|宋朝]]后，以[[临济宗|临济]]与[[曹洞宗|曹洞]]二宗，較有影響。

==法門==
[[禅宗|禅宗]]从[[达摩|达摩]]始，百餘年之间皆以《[[楞伽经|楞伽经]]》印証弟子的修為，故[[禪師|禪師]]亦称为[[楞伽師|楞伽師]]。达摩的三传弟子[[道信|道信]]开始兼以《[[金刚经|金刚经]]》等经为典据，道信徒孫[[弘忍|弘忍]]較看重《金刚经》，到了惠能即以文句简单的《金刚经》义代替了《楞伽经》，其目的在于摆脱[[名相|名相]]烦琐的思想束缚，而单刀直入求得开悟。<ref>《中国佛教》中国佛教协会编/林子青</ref>

惠能禅法以[[定慧|定慧]]为本。定與慧犹如灯光，有灯即有光，这是所谓定慧一体观。觉性本有，烦恼本无。直接契证觉性，便是[[顿悟|顿悟]]<ref>《佛教人物传》，中华佛典宝库编，</ref>。惠能曾告戒僧侶[[法達|法達]]，不要錯解《[[法華經|法華經]]》中意義。<ref>《[[六祖壇經|六祖壇經]]》載：「心迷法華轉，心悟轉法華，誦經久不明，與義作讎家；無念念即正，有念念成邪，有無俱不計，長御白牛車。」</ref>[[神秀|神秀]]把佛教理論“戒、定、慧”解釋為：“诸恶莫作名为戒，诸善奉行名为慧，自净其意名为定。”<ref>《[[法华经|法华经]]·寿量品》</ref>而惠能則另有不同解說：“心地无非自性戒，心地无痴自性慧，心地无乱自性定。”<ref>《[[六祖坛经|六祖坛经]]·偈》</ref><ref>吉州志诚禅师《五灯严统》</ref>然而，惠能亦一再強調「法無頓漸，人有利鈍，故名頓漸」、「法即無頓漸。迷悟有遲疾。」，所以認為頓漸只是假名，佛法只為[[一乘|一乘]]，「說即雖萬般。合理還歸一。」<ref>《六祖坛经》</ref>

惠能说自心既不攀缘善恶，不以静坐敛心才算是禅，就是一切时中行住坐卧动作裏，也可体会禅的境界。<ref>法海集记《[[法宝坛经|法宝坛经]]》</ref>又曰“先立无念为宗”，“佛法在世间，不离世间觉。”<ref>《[[坛经|坛经]]·智论》</ref>所谓无念，即虽有见闻觉知，而心常空寂之意。“自心归依自性，是皈依真佛”<ref>《[[坛经|坛经]]·智论》</ref>

== 源流 ==
中國禪宗由始祖[[達摩|達摩]]、二祖[[神光|神光]]、三祖[[僧粲|僧粲]]、四祖[[道信|道信]]、五祖[[弘忍|弘忍]]，一脈相傳。五祖弘忍有[[惠能|惠能]]與[[神秀|神秀]]兩大弟子，而神秀早年博覽[[五經|五經]]，學通[[三藏|三藏]]，人稱為[[上座|上座]]，擔任「教授[[阿闍梨|阿闍梨]]」；以「身是菩提樹，心如明鏡臺，時時勤拂拭，勿使惹塵埃」一偈呈現，但沒有得到弘忍的讚許。[[惠能|惠能]]是[[文盲|文盲]]，但了悟[[佛心|佛心]]，見了神秀的詩，直言「菩提本無樹，明鏡亦非臺。本來無一物，何處惹塵埃」，弘忍大喜，入室為其宣講《[[金剛經|金剛經]]》，認為惠能已經[[覺悟|覺悟]]了，傳給惠能衣缽。

弘忍[[圓寂|圓寂]]後，神秀在北方大開禪法，人稱[[北宗|北宗]]，教義傳遍[[西京|西京]][[長安|長安]]、[[東京|東京]][[洛邑|洛邑]]，並曾獲得[[武后|武后]]、[[中宗|中宗]]、[[睿宗|睿宗]]三帝之[[皈依|皈依]]，有「两京[[法主|法主]]、三帝[[帝師|門師]]」<ref>{{Cite web |url=http://www.ichanfeng.com/news/tianxiazongfeng/2015/0618/787.html |title=曾为“两京法主，三帝门师”的神秀禅师 |access-date=2018-07-13 |archive-url=https://web.archive.org/web/20180714021419/http://www.ichanfeng.com/news/tianxiazongfeng/2015/0618/787.html |archive-date=2018-07-14 |dead-url=yes }}</ref>之尊號。得到衣缽的惠能到[[曹溪|曹溪]][[宝林寺|宝林寺]]（今[[广东|广东]][[韶关|韶关]][[南华寺|南华寺]]）弘法，主張「頓悟」，影响[[華南|華南]]諸宗派，人稱「南宗」（曹溪禪），當時[[惠能|惠能]]與與[[神秀|神秀]]並駕齊驅，人稱[[南能北秀|南能北秀]]。

[[唐明皇|唐明皇]]時，惠能之徒[[荷澤神會|荷澤神會]]在「明定南北總是非大會」（[[滑臺大會|滑臺大會]]）中與[[北宗|北宗]]辯論勝利，使[[曹溪禪|曹溪禪]]後來居上，成為了中國禪宗的主流，後世禪宗大多出自此系統，從此禪宗以[[曹溪禪|曹溪禪]]為正宗，尊[[惠能|惠能]]為六祖。六祖完成了禪的中國化道路，使禪宗成為了中國佛教的主流<ref>印順著：《中國禪宗史》，北京：團結出版社，2009年版，第5-6頁</ref>。因为惠能之后不再传衣钵，並無明定七祖是何人，雖然[[唐德宗|唐德宗]][[追贈|追贈]]荷澤神會為禪宗七祖，並親撰《七祖贊》，但其實曹溪禪已經各開枝散葉。

隨著曹溪禪影響的擴大，禪宗也開始產生了分流，六祖慧能門下分為[[荷澤神會|荷澤神會]]、[[南岳懷讓|南岳懷讓]]、[[青原行思|青原行思]]等宗系<ref>麻天祥著：《中國禪宗思想發展史》，長沙：湖南教育出版社，1997年版，第37頁</ref>。南岳懷讓的弟子[[馬祖道一|馬祖道一]]播化[[江西|江西]]。馬祖的弟子[[百丈懷海|百丈懷海]]制訂叢林清規，建立禪院組織制度。懷海門下又分出[[溈仰宗|溈仰]]、[[臨濟宗|臨濟]]二宗。

青原行思的弟子[[石頭希遷|石頭希遷]]在[[湖南|湖南]]弘法，大闡宗風。石頭希遷從「本體無相，即休即用」進行發揮，提倡隨順自然的修行方法，以「心體靈昭，湛然圓滿，一源含萬派，萬象現心中」為法門大要。石頭希遷門下有[[藥山惟儼|藥山惟儼]]與[[天皇道悟|天皇道悟]]{{noteTag|1=道悟的法統歸屬至宋代起開始出現爭端，有道悟出於南岳下一說，或有天皇道悟與天王道悟二道悟之說，該爭議一直持續到清初。}}，惟儼再傳弟子[[洞山良价|洞山良价]]與三傳弟子[[曹山本寂|曹山本寂]]創[[曹洞宗|曹洞宗]]。道悟三傳至[[雪峰義存|雪峰義存]]，義存門下[[雲門文偃|雲門文偃]]創立[[雲門宗|雲門宗]]。義存另一弟子[[玄沙師備|玄沙師備]]傳[[羅漢桂琛|羅漢桂琛]]，桂琛傳[[清涼文益|清涼文益]]，文益弟子廣布，創立法眼宗<ref>釋道原撰：《景德傳燈錄》，CBETA, T10, no. 279</ref>。

溈仰、臨濟、曹洞、雲門、法眼五宗的出現，被禪宗稱之為「一花開五葉，道果自然成」。禪門五宗均出於曹溪禪法，思想上並無不同，但在接引方式方面各有特色<ref>麻天祥著：《中國禪宗思想發展史》，長沙：湖南教育出版社，1997年版，第40頁</ref>。

== 支系 ==
曹溪禪后分为曹溪北宗、曹溪南宗、[[保唐宗|保唐宗]]。曹溪北宗即為[[荷澤宗|荷澤宗]]，曹溪南宗分化出[[石頭宗|石頭宗]]、[[洪州宗|洪州宗]]、保唐宗三支，而以石頭、洪州為禪門正宗。

*'''[[楞伽師|楞伽師]]'''（又稱南天竺一乘宗、南宗），因[[求那跋陀羅|求那跋陀羅]]《[[楞伽經|楞伽阿跋多羅寶經]]》为名。
**'''[[東山法门|東山法门]]'''（又稱[[黃梅禪|黃梅禪]]），因[[弘忍|弘忍]]居[[黃梅|黃梅]]雙峰山之東为名。
***北宗，因[[神秀|神秀]]的“[[渐悟|渐悟]]”为名。
***南宗（又称[[曹溪禪|曹溪禪]]），因六祖[[惠能|惠能]]的“[[顿悟|顿悟]]”、居于[[曹溪|曹溪]]为名。
****曹溪北宗：[[荷澤宗|荷澤宗]]，始於[[荷澤神會|荷澤神會]]，後衰。
****曹溪南宗，分二：
*****[[石頭禪|石頭禪]]，因[[石頭希遷|石頭希遷]]为名。後出[[雲門宗|雲門]]、[[法眼宗|法眼]]、[[曹洞宗|曹洞]]三宗。
******[[雲門宗|雲門宗]]：出自[[雲門文偃|雲門文偃]]禅师，以以[[韶州|韶州]]云门山寺（在今[[广东|广东]][[乳源|乳源]]瑶族自治县北）得名。
******[[法眼宗|法眼]]：始於[[法眼文益|法眼文益]]，為禪宗五派中最晚成立的一派。
******[[曹洞宗|曹洞]]：[[曹山本寂|曹山本寂]]（840年－901年）所立，因曹山（今江西[[宜黄|宜黄]]境内）得名。
*****[[洪州禪|洪州禪]]，始於[[馬祖道一|道一法师]]，因居于[[江西|江西]][[洪州|洪州]][[開元寺|開元寺]]为名。下開[[臨濟宗|臨濟]]、[[溈仰宗|溈仰]]兩宗。
******[[臨濟宗|臨濟宗]]：始於[[臨濟义玄|臨濟义玄]]（？－867年）大師，因镇州[[临济寺|临济院]]（今[[河北|河北]][[正定|正定]]，滹沱河畔）得名。
******[[溈仰宗|溈仰宗]]：[[溈山靈祐|溈山靈祐]]及其弟子[[仰山慧寂|仰山慧寂]]在的沩山以及仰山举扬宗风，后世就称它为沩仰宗。以[[棲隱禪寺|棲隱禪寺]]為祖庭。
****[[保唐宗|保唐宗]]，因[[成都|成都]][[保唐寺|保唐寺]][[無住禪師|無住禪師]]为名。
