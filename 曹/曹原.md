'''曹原'''（{{bd|1996年||||catIdx=曹}}），[[四川|四川]][[成都|成都]]人，是一名正在[[美国|美国]][[麻省理工学院|麻省理工学院]]攻读[[博士|博士]]学位的中国籍学生，其主要研究领域为二维材料的输运性质。<ref name="aboutme" /><ref name="rti" /> 2018年因入选《[[自然_(期刊)|自然]]》杂志“2018年度科学人物”并位列榜首而知名，他被《自然》赞誉为“石墨烯驾驭者（{{lang|en|Graphene Wrangler}}）”。<ref>{{cite web |author1=ELIZABETH GIBNEY |title=Nature’s 10 - Ten people who mattered this year. |url=https://www.nature.com/immersive/d41586-018-07683-5/index.html |publisher=[[自然_(期刊)|自然]] |accessdate=2018-12-22}}</ref><ref>{{cite web |author1=刘世东 |title=“石墨烯驾驭者”：中国95后博士登上《自然》年度科学人物榜首 |url=http://stdaily.com/index/kejixinwen/2018-12/22/content_741760.shtml |website=中国科技网 |publisher=[[中国日报|中国日报网]]}}</ref><ref>{{cite web |author1=虞涵棋 |title=《自然》年度十大人物:天才曹原居首 贺建奎来去匆匆 |url=http://tech.sina.com.cn/d/i/2018-12-19/doc-ihqhqcir8089491.shtml |website=[[新浪网|新浪网]] |publisher=[[澎湃新闻|澎湃新闻]] |accessdate=2018-12-22}}</ref><ref>{{cite web |author1=林庭瑤 |title=「自然」科學人物 中國天才曹原居首、賀建奎亦入列 |url=https://udn.com/news/story/7332/3547376 |website=[[联合新闻网|联合新闻网]] |publisher=[[联合报|联合报]] |accessdate=2018-12-22}}</ref><ref>{{cite web |url=https://www.ettoday.net/news/20181219/1334899.htm |title=《自然》年度科學人物 「石墨烯駕馭者」曹原名列榜首 |publisher=[[ETtoday新聞雲|ETtoday新聞雲]]}}</ref>
