{{vgname|生化危機5|ja=バイオハザード5，Biohazard 5|en=Resident Evil 5|cn=生化危机5|hk=cn|tw=惡靈古堡5}}是一款由[[卡普空|卡普空]]开发并发行在PlayStation 3、Xbox360和Windows平台的[[第三人称射击游戏|第三人称射击]][[恐怖游戏|恐怖游戏]]。本作是《[[生化危机系列|生化危机]]》系列的第6部正统作品，并首次把游戏舞台设定在非洲。

游戏於2005年7月12日由[[卡普空|卡普空]]公司正式宣佈開發[[PlayStation_3|PlayStation 3]]版本，其後表示[[Xbox_360|Xbox 360]]版本將會同時開發，在2009年七月也宣佈電腦版同年推出，遊戲已於2009年3月5日在日本發售及3月13日在北美和歐洲發售。在日本，PS3版首週銷量為36萬，為PS3當時歷來首週銷量第二位，而XBOX360版則為8萬7千。<ref>電擊Online，''[http://news.dengeki.com/elem/000/000/145/145733/ 3分鐘排名]''。2009年3月14日引用。(日文)</ref>卡普空于2016年6月在[[PlayStation_4|PlayStation 4]]和[[Xbox_One|Xbox One]]平台上重新发售了本作；该版本的简体中文版于2018年2月27日发售PlayStation 4版本，同时Xbox One版本免费更新。<ref>[http://tv.duowan.com/1801/380562561606.html 《生化危机4/5/6》简体中文版2月27日正式发售 X1版可免费更新].多玩电视游戏.2018-01-22.[2019-03-05].</ref>

== 歷史 ==
《生化危機5》最初由[[小林裕幸|小林裕幸]]負責初期策劃及開發工作，其後便將《生化危機5》交由曾經擔任《[[鬼武者|鬼武者]]》監督及《[[失落的星球：極限狀態|失落的星球：極限狀態]]》製作人的[[竹內潤|竹內潤]]負責擔任[[執行製作人|執行製作人]]。而曾經擔任《[[生化危機2|生化危機2]]》宣傳監督及《[[生化危機4|生化危機4]]》([[PlayStation_2|PlayStation 2]]版本)執行製作人的[[稻船敬二|稻船敬二]]亦負責監視整個遊戲的開發進度。其後[[卡普空|卡普空]]公司全資擁有的[[Clover_Studio|Clover Studio]]亦加入並且協助繼續開發。

遊戲開發初期，由於大量開發人員借調到開發[[Wii|Wii]]專用遊戲《[[生化危機：安布雷拉編年史|生化危機：安布雷拉編年史]]》（Biohazard Umbrella Chronicles）的工作，於是《生化危機5》的開發非常緩慢，在2006年的[[E3|E3電玩展]]上沒有提供任何新的展示片段。其後Clover Studio於2007年三月被卡普空解散，進一步拖慢了《生化危機5》的開發進度。

直至2007年的E3電玩展，卡普空於場內展了新的《生化危機5》片段，片段以實機遊戲畫面進行，玩家可以於[[PlayStation_Network|PlayStation Network]]及[[Xbox_Live|Xbox Live]]下載欣賞。

== 概要 ==
2009年，已成為生化恐怖主義安全評估聯盟（Bioterrorism Security Assessment Alliance，簡稱BSAA）成員的克里斯·雷德菲爾被部署到非洲小鎮吉鳩就（Kijuju），與席娃·亞洛瑪合作逮捕試圖在黑市出售生化武器的里卡德·亞文（Ricardo Irving）。當地被強迫餵食寄生體的居民迅速變成馬吉尼（Majini）並多次向二人作出攻擊。在前往亞文交易地點時，克里斯和席娃發現Alpha隊伍被不明生命體殺死，在場奄奄一息的隊長迪恰恩特（Dechant）在死前交出了一個裝有亞文資料的硬盤。二人在消滅了該不明生命體後把硬盤內的資料傳回本部並奉命繼續追捕亞文。因此兩人的正式任務就此展開。

== 遊戲系統 ==

《生化危機5》的畫面結構與《[[生化危機4|生化危機4]]》相同，男主角克里斯在舉槍的狀態下會在畫面的左方，而女主角席娃舉槍時的狀態會在畫面的右方，這次加入Online和Offline的兩人合作的模式，而且可以透過互聯網下載追加的遊戲內容。

=== 合作模式（Co-op Mode） ===
《生化危機5》包含2人合作遊玩主篇的模式，此模式分為連線及離線兩種，離線時主機需要兩個控制器，而畫面則會上下分割顯示。<ref>CAPCOM Official Website ''[http://www.capcom.co.jp/bio5/coop.html Bio Hazard 5 Co-op Mode]''。2009年4月9日引用。(日文)</ref>

=== 對戰模式（Versus） ===
此為付費的追加下載內容，玩家可以組成最多4人的連線對戰遊戲，此模式細分為兩種，分別是「SLAYERS」和「SURVIVORS」。SLAYERS 以打倒"Majini"（生化危機5的一般敵人）為目標而獲取分數的模式; SURVIVORS 則以打倒其他玩家作目標而獲取分數。SLAYERS和SURVIVORS都可以選擇個人戰或團隊戰（例如2對2）。<ref>CAPCOM Official Website ''[http://www.capcom.co.jp/bio5/versus.html Bio Hazard5 VERSUS]''。2009年4月9日引用。(日文)</ref>(PC版没有)

=== 傭兵模式（The Mercenaries） ===
《生化危機5》保留了此第三代開始出現的傭兵模式，玩家操作不同的角色挑戰不同關卡，以追求高分爲目的。所得分數可用於開啟遊戲內的一些隱藏要素，此模式共八個關卡，獲得一定分數後會自動開啟新關卡。另外，亦可以透過互聯網下載THE MERCENARIES DUO模式，通過該模式2位玩家可以在線協力挑戰傭兵模式。<ref>新浪遊戲 ''[http://game.sina.com.hk/cgi-bin/nw/show.cgi/463/3/1/24481/1.html Bio Hazard 5 傭兵模式免費下載] {{webarchive|url=https://web.archive.org/web/20090318011635/http://game.sina.com.hk/cgi-bin/nw/show.cgi/463/3/1/24481/1.html |date=2009-03-18 }}''。2009年4月9日引用。</ref>

== 特點 ==
=== 光源 ===

[[File:BIOHAZARD_5_1.jpg|thumb]]

曾經說明《生化危機5》的陽光及燈光效果將會實現對眼睛有一定的影響，例如由光明的地方進入黑暗的地方，主角的眼睛會先看不清楚內裡的環境，待眼睛適應之後才會慢慢看得清楚。相反，走出陽光猛烈的地方時，主角的雙眼感到耀眼便會有瞬間是雪白，看不到周圍的環境。

=== 集體瘋狂 ===
《生化危機5》的敵人不再是[[喪屍|喪屍]]（殭屍），他們的智力非常高，將會以「集體瘋狂」的方式去攻擊主角，比起前作更會懂得利用武器。他們是寄生了Plagas Types 2&3，還有感染了烏洛波羅斯（Uroboros，有銜尾蛇的意思）的不明生物。

=== 神秘女角 ===
《生化危機5》在[[E3|E3電玩展]]展示的最新片段出現了一個神秘女角，身穿黑色斗篷，帶上一個鳥形面具，外國玩家稱她為Birdlady。由執行製作人[[竹內潤|竹內潤]]的口中得知她將會是本集遊戲的關鍵人物。
而這個女角就是《[[生化危機|生化危機]]》和《[[生化危機3|生化危機3]]》的女主角[[吉兒·范倫廷|吉兒·范倫廷]]。

=== 武器系統 ===
《生化危機5》取消了《生化危機4》的武器商人系統，改以在關卡開始或RESTART時處整理和購買或者升級武器。并且本作的武器也比前作多很多。Capcom将前作《生化危机4》中的無限子弹Thompson改为無限子彈系統，只要將武器改造至上限，即可用分數換取。

== 電腦版的新內容 ==
《生化危機5》電腦版有一些內容是XBOX360®版以及PlayStation®3版是沒有的，例如對應「NVIDIA GeForce 3D Vision」 技術（3D立體眼鏡）；升級版「傭兵模式」「No Mercy」（3倍敵人數壓倒性登場）；針對PC版新添加了「鍵盤+滑鼠」的操作模式；新增男女主角新服裝；以及電腦畫質和按鍵的設定項目以及NVIDIA®發佈的遊戲3D運作測試項目等都是電腦版獨特應有的內容。

== 製作人的看法 ==
===三上真司===
生化危機系列的創造者[[三上真司|三上真司]]因前作的平台移植問題而離開CAPCOM，所以沒有參與製作本遊戲。他接受訪問時表示《生化危機5》與之前的《生化危機4》過分相似，系統相似在《生化危機5》還可接受，但如果要使系列延續，就必須要開發一套新的模式。他也表示自己不會玩此作品，因為《生化危機5》不是他想要做的遊戲，而且劇情令他很失望。
<ref>太平洋游戏网
 ''[http://www.pcgames.com.cn/tvgames/bao/other/0812/1290406.html 三上真司不玩《生化5》!彻底革新势在必行]''。2009年7月26日引用。</ref>

===竹內潤===
在《惡靈古堡5》在上市以來，玩家對此遊戲的評論都有正有負。而《惡靈古堡5》執行製作人竹內潤接受訪問時表示開發本作過程時的壓力很大，指製作時有參考三上真司的《惡靈古堡4》，但得不到他的支持，又指希望全面集中開發《失落在星球2》。因此他不會參與《惡靈古堡6》的任何開發。
<ref>新浪遊戲
 ''[http://game.sina.com.hk/cgi-bin/nw/show.cgi/463/3/1/57586/1.html Bio Hazard5製作人:我不會做生化6了] {{webarchive|url=https://web.archive.org/web/20140726041643/http://game.sina.com.hk/cgi-bin/nw/show.cgi/463/3/1/57586/1.html |date=2014-07-26 }}''。2010年5月7日引用。</ref>
