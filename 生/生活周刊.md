'''《生活周刊》'''，是[[上海市|上海市]]一份已经停刊的周报，自称为1925年创刊的《[[生活_(周刊)|生活]]》周刊的后继者。该报1985年由[[青年报|青年报社]]复办，逢周日出版。2018年12月25日，青年报社发行最后一期《生活周刊》（1763期），2019年1月1日起停刊，并入《[[青年报|青年报]]》周日刊。

== 1925年创办 ==
{{see|生活 (周刊)}}

1925年10月11日，《生活》周刊创刊，1926年10月起改由[[邹韬奋|邹韬奋]]主编<ref>{{cite book |author1=上海市卢湾区志编纂委员会 |title=卢湾区志 |date=1998 |publisher=上海社会科学院出版社 |isbn=7-80618-445-7 |url=http://www.shtong.gov.cn/Newsite/node2/node2245/node4522/node5500/index.html |accessdate=2018-12-29 |chapter=大事记}}</ref>。报刊宣传抗日思想，曾是中国发行量最大的报刊，但于1933年12月16日遭到查封，邹韬奋远走国外。<ref>{{cite book |title=上海通志·第四卷 |publisher=上海人民出版社 |url=http://www.shtong.gov.cn/Newsite/node2/node2247/node4568/node79140/node79146/userobject1ai103323.html |accessdate=2018-12-29 |chapter=第二节 重大决策、活动}}</ref>

== 1985年复办 ==
1985年，经过邹韬奋的夫人沈粹缜首肯，青年报社于1985年1月6日发行新的《生活周刊》<ref name="qfgj">{{cite journal|title=千帆过尽，江河宛若初见——《生活周刊》致读者|journal=生活周刊|date=2018-12-26|issue=1763}}</ref>。不同于《青年报》的其他副刊，该报刊单独出版发行。发行初期，《生活周刊》版面为4开16版，系改革开放后中国第一张综合生活类新闻周刊。该报以继承邹韬奋精神自命，报头沿用[[黄炎培|黄炎培]]手书“生活”二字，办报宗旨有两条，即“传播现代生活信息，传播现代生活方式，传播现代生活观念”和“寓教于生活，真诚地为生活着的人们服务”。该报纸曾创立“大特写”栏目，截取社会生活的一个侧面进行放大细化并追根溯源，《101大车祸》《天上有个太阳，水中有个月亮》《一首被肢解的颂歌》《印度洋幸存者》等文有较强的社会反响，栏目的形式也曾被国内大量媒体所仿效。该报纸也曾被多次评为“全国最佳报纸”和“全国十佳报纸”。据1995年统计数字，《生活周刊》的年发行量曾达到16万份。<ref>{{cite book |author1=《上海新闻志》编纂委员会 |title=上海新闻志 |date=2000 |publisher=上海社会科学院出版社 |isbn=7-80618-739-1 |edition=第1版 |url=http://www.shtong.gov.cn/Newsite/node2/node2245/node66268/node66281/node66340/node66416/userobject1ai62394.html |accessdate=2018-12-29 |section=新闻出版}}</ref><ref>{{cite book |author1=《上海青年志》编撰委员会 |title=上海青年志 |date=2002 |publisher=上海社会科学院出版社 |isbn=7-80681-005-6 |url=http://www.shtong.gov.cn/Newsite/node2/node2245/node66268/node66281/node66340/node66416/userobject1ai62394.html |accessdate=2018-12-29 |section=新闻出版}}</ref>

2018年12月18日，青年报社宣布，《生活周刊》自2019年1月1日（周日）起休刊，内容并入《青年报》周日刊出版<ref name="qfgj" /><ref>{{cite web |author1=青年报社 |title=告读者 {{!}}《生活周刊》休刊公告 |url=https://mp.weixin.qq.com/s?src=11&timestamp=1546081613&ver=1317&signature=vmsE4alnygvL-enlhTeaV6OIvHdy11LqX4B8KUHOjsz2AdUDiVpi8EQqDm2BTUcQ8PsKj6AaWzgxKdfqlPSxOp2O1UG24i7-w*uIe6mBZT5RfQMgKcrW5vgADO7ZpMuD&new=1 |website=微信公众平台 |publisher=青年报社《生活周刊》编辑部 |accessdate=2018-12-29 |archivedate=2018-12-29 |archiveurl=http://archive.fo/8PIPs }}</ref>。
