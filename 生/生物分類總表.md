===[[植物界|植物界]](Plantae)===
{{主條目|植物分類表 (NCBI)}}
包含[[綠藻|綠藻]]、[[輪藻|輪藻]]、[[苔蘚植物|苔蘚植物]]、[[蕨類植物|蕨類植物]]、[[種子植物|種子植物]]等。<br>
植物依据不同的分类方式可分为不同的类型，依据营养来源的不同分为自养型和异养型；依据是否形成种子分为孢子植物和种子植物；还可依据种子结构、主要特征进行分类。现介绍如下：

==== 自养植物与异养植物 ====

根据[[生命|生命]]活动所需[[有机物|有机物]]来源的不同进行分类，能自己合成有机物的是自养型植物；依赖于现成有机物的是异养植物。

类型包括有自养植物及异养植物，[[绿色|绿色]][[植物|植物]]包括有[[寄生植物|寄生植物]]、[[腐生植物|腐生植物]]、[[食虫植物|食虫植物]]、[[食菌植物|食菌植物]]。而常见代表有[[孢子植物|孢子植物]]和[[种子植物|种子植物]]的[[菟丝子|菟丝子]]、[[腐生龙胆|腐生龙胆]]、[[苁蓉|苁蓉]]、[[猪笼草|猪笼草]]、[[天麻|天麻]]。
 

==== 孢子植物与种子植物 ====

[[植物|植物]]依据能否形成[[种子|种子]]可分为种子植物和孢子植物。孢子植物又可分为[[藻类|藻类]]、[[苔藓|苔藓]]、[[蕨类|蕨类]]，均通过孢子进行生殖，[[进化|进化]]顺序由低到高依次为藻类→苔藓→蕨类。种子植物对[[陆地|陆地]]的适应能力很强，依据种子外面有无果皮包被可分为被子植物（有果皮）和裸子植物（无果皮）。

藻类植物分別有衣藻、水绵、海带、紫菜，它們大都生活在水中，无根、茎、叶等器官分化，而植物的类型及常见主要特征如下：

===== 苔藓植物 =====
* [[墙藓|墙藓]]、[[葫芦藓|葫芦藓]]、[[地钱|地钱]]
**大多生活在[[潮湿|潮湿]]环境中，植株矮小，有[[茎|茎]]、[[叶|叶]]分化，有假[[根|根]]；无输导组织，叶片仅有一层[[细胞|细胞]]。
 
===== 蕨类植物 =====
*蕨、肾蕨、卷柏、满江红
**生活在[[潮濕|潮濕]][[环境|环境]]中，有[[根|根]]、[[茎|茎]]、[[叶|叶]]的分化；[[根|根]]、[[茎|茎]]、[[叶|叶]]中均有专门的输导组织，因此可以长高。
 
===== 裸子植物 =====
*[[松|松]]、[[杉|杉]]、[[柏|柏]]、[[苏铁|苏铁]]、[[银杏|银杏]]
**[[种子|种子]]祼露，无果皮包被，易受[[昆虫|昆虫]]叮咬及不良因素的危害；对[[陆地|陆地]]环境的适应能力比[[孢子植物|孢子植物]]强，比[[被子植物|被子植物]]弱。
 
===== 被子植物 =====
被子植物的[[種子|種子]]由种皮、[[胚|胚]]和胚乳构成。其中，胚由子[[叶|叶]]、[[胚芽|胚芽]]、[[胚轴|胚轴]]和[[胚根|胚根]]组成。有的种子有胚乳，有的种子无胚乳。主要类型有相同点是子叶、不同点是胚乳。

*种子不裸露，有果皮包被；种子免受昆虫叮咬及不良因素的危害；是对陆地环境的适应能力最强的绿色植物。 
**例如[[玉米|玉米]]、[[水稻|水稻]]、[[菊|菊]]、[[牡丹|牡丹]]、[[苹果|苹果]]、[[梨|梨]]、刺槐

*双子叶无胚乳是都有种皮和胚，胚都由胚芽、胚轴、胚根和子叶构成。
**例如[[菜豆|菜豆]]、[[荠菜|荠菜]]、[[花生|花生]]。
 
*两片无双子叶有胚乳。
**例如[[蓖麻|蓖麻]]、[[莲|莲]]、[[荞麦|荞麦]]、[[胡萝卜|胡萝卜]]、[[苋菜|苋菜]]、[[柿|柿]]

*两片有单子叶无胚乳。
**例如[[慈姑|慈姑]]、[[泽泻|泽泻]]、[[眼子菜|眼子菜]]

*有一片单子叶有胚乳。
**例如[[玉米|玉米]]、[[小麦|小麦]]、[[水稻|水稻]]、[[高粱|高粱]]
