'''生茂'''（{{bd|1928年||2007年|11月24日}}），原名'''娄盛茂'''，生于[[河北|河北]][[晋县|晋县]]，[[中华人民共和国|中华人民共和国]]作曲家<ref name=caijing>唐诃，生茂，财经2007年第26期</ref><ref name=qushi>{{Cite web |url = http://ent.sina.com.cn/y/2007-11-29/01321811932.shtml |title = 《学习雷锋好榜样》曲作者生茂去世 享年79岁 |author =  |date =2007-11-29  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1945年参加[[八路军|八路军]]。先后在冀中区火线剧社、华北野战军三纵队文工团任指挥、乐队队长、编导等职。[[中华人民共和国|中华人民共和国]]成立后，到[[中央音乐学院|中央音乐学院]]作曲系进修。1950年代中期，他刚刚随六十三军从朝鲜的[[抗美援朝战争|抗美援朝战争]]前线回国，驻[[石家庄|石家庄]]。不久，他向[[中国人民解放军陆军政治工作部文工团|北京军区战友文工团]]的[[唐诃|唐诃]]寄来新作《真是乐死人》，唐诃看后转交团长[[张非|张非]]，张非亲自修改，并交[[马国光|马国光]]演唱。这首歌在1956年第一届北京音乐周上演出，随后流传开来<ref name=caijing/><ref name=qushi/>。 

1958年，调到[[中国人民解放军陆军政治工作部文工团|北京军区战友文工团]]任作曲和艺术指导。他和[[唐诃|唐诃]]经常合作作曲，有时[[晨耕|晨耕]]也参加，后来他们取了个联合笔名“唐茂耕”<ref name=caijing/>。

1965年，为纪念红军[[长征|长征]]胜利30周年，[[肖华|肖华]]写出组诗《[[长征组歌|长征组歌]]——红军不怕远征难》并点名要求[[中国人民解放军陆军政治工作部文工团|北京军区战友文工团]]组织作曲家谱曲，晨耕、生茂、唐诃、[[遇秋|遇秋]]4人遂参加谱曲。在创作排练中，他们获得了[[周恩来|周恩来]]总理及老红军们的关心及指导。周恩来还专门接见了唐诃等4位曲作者，并对每段提出具体意见<ref name=tangtang>{{Cite web |url =http://www.qdlgb.gov.cn/whyd/details.aspx?id=1420160214155243  |title = 堂堂正正一大家——忆唐诃 |author =刘真骅  |date =  |publisher = 中共青岛市委老干部局  |language =  |accessdate = 2017-08-27 |quote =  }}</ref><ref>{{Cite book|title=你不可不知道的音乐常识|last=吴刚主编；李金祥等编|first=|publisher=南海出版社|year=2013年1月|isbn=978-7-5442-6403-7|location=海口|pages=272}}</ref>。

1966年[[文革|文革]]爆发后，晨耕、生茂、唐诃随即被[[造反派|造反派]]打成战友文工团的反革命小集团“唐茂耕[[三家村_(文革用语)|三家村]]”。1966年10月，唐诃在《[[解放军报|解放军报]]》上发表了[[毛主席语录|毛主席语录]]歌《完全彻底为人民》，见报后随即传遍全国。此后，唐诃又把《[[为人民服务|为人民服务]]》全文谱成一首弹唱歌曲，生茂随即也谱写出毛主席语录《再版前言》全文。这两首歌发表在《解放军歌曲》上。此后，造反派对他们的批斗缓和了，[[大字报|大字报]]也变少了。唐诃便找生茂提议写一部大作品《[[毛主席诗词|毛主席诗词]]组歌》，生茂表示赞同。他们选出八首毛主席诗词，并选出八段与这些诗词有关的毛主席语录。该创意获得战友文工团团长[[唐江|唐江]]以及作曲家[[吕韧敏|吕韧敏]]、[[王竹林|王竹林]]等人支持，经过编伴奏、练合唱，最后在[[京西宾馆|京西宾馆]]彩排。[[周恩来|周恩来]]得知后，前来观看了彩排，看后召集作者、指挥及主要演员座谈。周恩来说，在《[[沁园春·雪|沁园春·雪]]》中，最重要的一句是“数风流人物还看今朝”。唐诃、生茂的创作未突出这句。此后他们按周恩来的意见，对结尾进行了较大修改。后来该组歌没传开，但《沁园春·雪》这首歌曲却传开，后来还被编入中国的音乐学院声乐教材<ref name=caijing/>。

此后，生茂、唐诃又合作了《贫下中农最爱毛主席》、《毛主席永远和我们在一起》、《革命风雷激荡》、《毛主席是各族人民心中的红太阳》等歌曲。包括《长征组歌》在内，生茂、唐诃合作过70多首歌曲<ref name=caijing/><ref>{{Cite book|title=当前声乐创作问题|last=中国音乐家协会编|first=|publisher=北京：人民音乐出版社|year=1979年10月|isbn=|location=|pages=71}}</ref>。

2007年11月24日，生茂因肝癌晚期、肺部感染造成呼吸衰竭在北京病逝，享年79岁<ref name=caijing/><ref name=qushi/>。

==作品==
生茂创作了两千多首声乐作品。代表作有歌曲《马儿啊，你慢些走》、《学习雷锋好榜样》、《老房东查铺》等，以及《[[长征组歌|长征组歌]]——红军不怕远征难》<ref name=qushi/>。

生茂常年到中国各地采风，收集了数百种地方戏曲、小调、民歌。《长征组歌》中由生茂作曲的第一曲至第七曲，分别采用[[赣南|赣南]][[采茶戏|采茶戏]]、[[川江号子|川江号子]]、[[云南|云南]][[花灯戏|花灯戏]]、[[瑶族|瑶族]]和[[苗族|苗族]]等多种音乐元素，还有[[秦腔|秦腔]]和[[蒲剧|蒲剧]]的音乐风格<ref name=caijing/>。
