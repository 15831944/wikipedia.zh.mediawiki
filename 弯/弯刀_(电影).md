{{作品名稱|name=''Machete''|hk=攞命．彎刀．罪惡城|tw=絕煞刀鋒|cn=弯刀}}是一部2010年上映的美國[[動作片|動作片]]，為[[勞勃·羅里葛茲|勞勃·羅里葛茲]]和[[伊森·曼尼奎斯|伊森·曼尼奎斯]]執導。本片為[[昆汀·塔倫提諾|昆汀·塔倫提諾]]的2007年電影《[[刑房_(電影)|刑房]]》的假預告片延伸而出，並延續了《刑房》的[[B級片|B級片]]風格和一些素材<ref name="AutoMR-4">{{Cite news|title=The unlikely politics of Robert Rodriguez's 'Machete' |first=Steven |last=Zeitchik |newspaper=Los Angeles Times |date=July 23, 2010 |url=http://articles.latimes.com/2010/jul/23/entertainment/la-et-comic-con-machete |accessdate=August 23, 2010}}</ref>。電影由[[丹尼·崔喬|丹尼·崔喬]]擔任主演，其他配角包括[[史蒂芬·席格|史蒂芬·席格]]、[[蜜雪兒·羅德里奎茲|蜜雪兒·羅德里奎茲]]、[[勞勃·狄尼洛|勞勃·狄尼洛]]、[[切奇·馬林|切奇·馬林]]、[[琳賽·蘿涵|琳賽·蘿涵]]、[[唐·強生|唐·強生]]、[[潔西卡·艾巴|潔西卡·艾巴]]與[[傑夫·法漢|傑夫·法漢]]。本片為史蒂芬·席格繼2002年電影《[[黑獄風雲_(2002年電影)|黑獄風雲]]》後的八年首部在戲院上映的電影。《絕煞刀鋒》由[[二十世紀福斯|二十世紀福斯]]定於2010年9月3日在美國上映。續集《[[殺千刀重出江湖|殺千刀重出江湖]]》於2013年10月11日推出。

==剧情==
故事围绕墨西哥人科尔特斯（丹尼·特乔）展开，讲述他在墨西哥和自己的兄弟们拯救那些被绑架的女孩子们的故事。

==花絮==
该片的原型是《刑房》，导演是[[罗伯特·罗德里格兹|罗伯特·罗德里格兹]]，灵感是《恐怖星球》和《死亡证据》。

该片的续集《弯刀2》已经在2012年4月开机拍摄。<ref>{{cite web |url= http://news.mtime.com/2012/02/08/1481340.html|title=《弯刀》续集正式定名 有望今年四月开机 |accessdate=2012-02-08  |publisher=时光网 |work= |language =zh-cn}}</ref>同时开拍的还有《罪恶之城2》。<ref>{{cite web |url=http://news.mtime.com/2011/08/10/1467220.html |title=罗德里格斯：《罪恶之城2》《弯刀2》即将开拍 |accessdate=2011-08-10  |publisher=时光网 |work= |language =zh-cn}}</ref>

杰西卡·阿尔巴在该片中沐浴的镜头原本是穿了[[内衣|内衣]]和[[内裤|内裤]]的，因为她拒绝全裸，经过电脑特技，看上去是全裸。<ref>{{cite web |url= http://news.mtime.com/2010/10/20/1442909.html|title=阿尔芭拒绝全裸 《弯刀》中裸浴原是后期制作 |accessdate=2010-10-20 |publisher=时光网 |work= |language =zh-cn}}</ref>

该片参加2010年9月的第67届[[威尼斯电影节|威尼斯电影节]]，导演罗伯特·罗德里格兹、丹尼·特乔和杰西卡·阿尔巴穿礼服出席。<ref>{{cite web |url= http://news.mtime.com/2010/09/02/1439966.html|title=《弯刀》威尼斯首映 俏娇娘阿尔芭猛男相伴 |accessdate=2010-09-02  |publisher=时光网 |work= |language =zh-cn}}</ref>

该片中的限制级镜头已经删节。<ref>{{cite web |url=http://news.mtime.com/2010/09/01/1439851.html |title= 《弯刀》限制级血腥片段 疤脸猛男彪悍杀戮|accessdate=2010-09-01  |publisher= 时光网|work= |language =zh-cn}}</ref>

该片于2010年8月25日在[[加利福尼亚州|加利福尼亚州]][[洛杉矶|洛杉矶]]首映，导演罗伯特·罗德里格兹、丹尼·特乔和杰西卡·阿尔巴出席。<ref>{{cite web |url=http://news.mtime.com/2010/08/26/1439515.html |title= 《弯刀》洛城首映礼 猛男特乔左拥右抱两美人|accessdate=2010-08-26 |publisher=时光网 |work= |language =zh-cn}}</ref>

琳赛·罗韩在该片中化身为[[修女|修女]]杀手。<ref>{{cite web |url= http://news.mtime.com/2010/08/22/1439205.html|title=《弯刀》琳赛·洛翰造型曝光 化身修女舔枪管 |accessdate= 2010-08-22 |publisher= 时光网|work= |language =zh-cn}}</ref>

该片中，杰西卡·阿尔巴将延续《罪恶之城》中的镜头，在车顶跳舞。<ref>{{cite web |url=http://news.mtime.com/2009/08/20/1414294.html |title=阿尔芭现身《弯刀》片场 车顶独舞对戏猛男 |accessdate= 2009-08-20|publisher=时光网 |work= |language =zh-cn}}</ref>
