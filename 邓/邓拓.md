'''邓拓'''（{{bd|1912年|2月26日|1966年|5月18日}}），原名邓子健、邓云特，笔名马南邨、左海等<ref>{{cite web|url = http://news.fznews.com.cn/zt/2007/lhfz/glp/2007-9-28/2007928uo_BobuIlk164914.shtml|title = 联话邓拓 |accessdate = 2009-06-07 |date =2007年9月28日 |language =zh-hans |publisher =福州新闻网}}</ref>。[[福建|福建]][[闽县|闽县]]（今[[福州|福州]]）人。中共宣传战线重要成员，长期担任《[[人民日报|人民日报]]》社长等中央主要宣传机构领导职务。后因多次未能领会[[毛泽东|毛泽东]]政治部署之意而遭到训斥，并被撤销人民日报社实际职务，[[文革|文革]]前夜因政治批判而自杀身亡。

==生平==
===早年===
1912年2月26日出生于福建闽县竹屿村（今为福州市区），父邓鸿予为[[举人|举人]]出身。民国15年（1926年）入[[福建省福州第一中学|省立福州第一中学]]，北伐军进入福州后，全市高中调整，邓入省立第一师范。民国17年，和[[傅衣凌|傅衣凌]]等组织“野草社”，出版自编自印的刊物《野草》<ref name="福州市档案局">{{cite web|url = http://www.fzda.gov.cn/tslmshow.asp?id=3077|title = 文学家--邓拓|date = 2004年10月8日|accessdate = 2009-06-08|language = zh-hans|publisher = 福州市档案局|deadurl = yes|archiveurl = https://web.archive.org/web/20110707032643/http://www.fzda.gov.cn/tslmshow.asp?id=3077|archivedate = 2011年7月7日}}</ref>。

1929年进入[[上海光华大学|上海光华大学]]（今[[华东师范大学|华东师范大学]]）社会经济系，1931年转入[[上海政法学院|上海政法学院]]，1933年加入[[中國共產黨|中國共產黨]]，民国21年，参加上海纪念“[[广州暴动|广州暴动]]”五周年游行时被捕。翌年秋，保释出狱。是年冬，参加“[[福建事变|福建事变]]”，在“人民革命政府”文化委员会、外交部任职。“闽变”失败，逃回上海，后经其兄邓伯宇的介绍，转入[[河南大学|河南大学]]社会经济系续学，1937年毕业并进入[[晋察冀边区|晋察冀边区]]，改名邓拓<ref name="福州市档案局"/>。

此后曾陆续使用邓云特、丁曼公、殷洲、萧斯、鸥子、以及左海、马南邨、吴南星（三人合用）、卜无忌、于道安、向阳生等多个笔名。

曾到晉察冀邊區担任《晋察冀日报》社社长、总编，做過[[新華社|新華社]]晉察冀分社社長。1944年夏，时任晋察冀日报社长兼主编的邓拓被调入分局党校学习，适逢审干运动，邓拓也遭到政治审查<ref >{{cite book|title = 红太阳是怎样升起的：延安整风运动的来龙去脉|date =2000-2 |accessdate = 2012-11-26|language = zh-hant|publisher =香港中文大學 |isbn=962201920X |author=高华|page=449}}</ref>。

[[中华人民共和国|中华人民共和国]]成立后，主持過首部《[[毛澤東選集|毛澤東選集]]》的編印工作<ref name="王彬彬">{{cite web|url = http://www.infzm.com/content/trs/raw/42911|title = 邓拓与第一部《毛泽东选集》的出版|author = 王彬彬|date =2008年4月16日 |accessdate = 2009-06-08|language = zh-hans|publisher =[[南方周末|南方周末]]}}</ref>，他在建国后初期當過[[人民日报|人民日报]]社长兼總編輯，也主編過《[[前綫_(雜誌)|前綫]]》，當過北京委文教書記。

===从巅峰滑向深渊===
1956年《人民日报》改版，6月10日脱稿《要反对[[保守主义|保守主义]]，也要反对急躁情绪》的社论，先后经过[[陆定一|陆定一]]、[[胡乔木|胡乔木]]、[[刘少奇|刘少奇]]三人审阅，送至[[毛泽东|毛泽东]]处，毛收到后批了三个字：“不看了”。时任《人民日报》总编的邓拓收到退回的稿件后十分为难，经过再三考虑，决定用新5号字（比老五号字小一号）发表，在6月20日头版显著位置刊登<ref>对于“6.20”社论，毛本人一直耿耿于怀，后来在1958年1月南宁会议上总爆发：严厉指出1956年反冒进是错误的，“6.20”社论有原则性错误。</ref>。1957年2月27日，毛泽东在最高国务会议上作了题为《关于正确处理人民内部矛盾》的报告，《人民日报》由于未接指示，故没有宣传，更没有发表社论。同样，在[[双百方针|双百方针]]开始后，《人民日报》依然没有动静，让毛泽东很不满意<ref>{{cite journal|url =http://www.cntheory.com/news/Djftzjjd/2007/1017/071017151230GKHKFKB3F9IKF2110140_2.html|title =陈维仁 给邓拓、林枫、胡耀邦当秘书|author =程冠军|date =2007年10月17日|language =zh-hans|publisher =[[中共中央党校|中共中央党校]]理论网|deadurl =yes|archiveurl =https://web.archive.org/web/20110519121915/http://www.cntheory.com/news/Djftzjjd/2007/1017/071017151230GKHKFKB3F9IKF2110140_2.html|archivedate =2011年5月19日}}</ref>。自4月10日后毛泽东多次批评邓拓和人民日报编辑，指斥邓拓为“书生办报、死人办报”<ref name="王彬彬"/><ref>{{cite journal|url =http://d.wanfangdata.com.cn/Periodical_xwdx200604005.aspx|title = "健笔终存天地间":论邓拓与书生办报|author = 吴廷俊 阳海洪 |date =2006年第4期 |language =zh-hans |publisher =新闻大学}}</ref>。邓拓感到自己处境十分难堪，请求辞职。毛泽东说，“你只知道养尊处优，不能占着茅坑不拉屎。”6月13日，邓拓被撤销总编职务，只保留社长名义，总编由[[吴冷西|吴冷西]]继任。

===“三家村”与燕山夜话===
[[File:1966-10_1966年对三家村的批判.jpg|right]]
马南邨是他1961年在《[[北京晚报|北京晚報]]》副刊寫《燕山夜话》專欄用的筆名。同年還跟[[吳晗|吳晗]]、[[廖沫沙|廖沫沙]]合寫《三家村札记》，共用吳南星筆名，寫到1964年7月總共六十五篇，其中邓拓寫了十八篇，最有名的一篇是《偉大的空話》。著有《燕山夜话》、《三家村札记》等杂文集。

邓拓在《燕山夜话》当中对毛泽东的大跃进政策进行了尖锐的评判，在《爱护劳动力的学说》一篇当中借古讽今对强迫使用农业劳动力和大规模无计划的工程建设提出抗议，他写道「早在春秋战国及其前后时期，许多古代的大政治家已经知道爱护劳动者的重要意义……《礼记·王制》写道：“用民之力，岁不过三日”……我们应当从古人的经验中得到新的启发，更加注意在各方面努力爱护劳动力。」<ref>邓拓：《爱护劳动力的学说》，燕山夜话，1，58</ref>

《燕山夜话》的另一篇则迂回地为当时被打倒的[[彭德怀|彭德怀]]而辩护，试图为彭争取公正的待遇。《为李三才辩护》当中描述了高级官员李三才，他因为在朝廷上勇敢揭发宦官的罪恶而被罢官，李一再上书要求皇帝亲自审问，但被拒绝，这个情节恰巧影射了毛泽东拒绝彭德怀后来写的八万字自辩书。

邓拓以古代直言进谏的学者自诩，称赞的“杂家”（具有非正统观点的知识分子）对国家工作的重要意义，并且屡次举明末的[[东林党|东林党]]人积极参政的范例，邓拓写过若干首诗来称许东林党人“莫谓书生空议论，头颅掷处血斑斑”“力抗权奸志不移，东林一代好男儿”<ref>{{cite news|url =|title = 太湖吟|date= 1960-09-07|accessdate =2012年6月5日 |publisher =[[光明日报|光明日报]]|language = zh }}</ref>。

自1966年5月8日《[[解放军报|解放军报]]》发表署名“高炬”的文章《向反党反社会主义的黑线开火》点明邓拓“为资产阶级复辟鸣锣开道”后<ref>{{cite web|url =http://news.ifeng.com/history/1/midang/200903/0329_2664_1081987_2.shtml|title =《人民日报》社前社长邓拓自杀前的抗争|language = zh-hans |author =钱江 |accessdate =2009年6月5日 |publisher =[[凤凰网|凤凰网]]转自《世纪》 }}</ref>，5月10日、5月14日、5月16日先后发表了[[姚文元|姚文元]]、[[戚本禹|戚本禹]]等针对邓拓的批判文章，揭发邓拓是“反党反社会主义头目”，并且声称：“邓拓是什么人？现在已经查明，他是一个叛徒。”<ref>{{cite web |url =http://skb.hebnews.cn/20050714/ca534296.htm |title =邓拓为何在“文革”序幕中死去 |accessdate =2009年6月5日 |publisher =《人物周报》 |language =zh-hans |deadurl =yes |archiveurl =https://web.archive.org/web/20110707044408/http://skb.hebnews.cn/20050714/ca534296.htm |archivedate =2011年7月7日 }}</ref><ref>{{cite journal|url =|title = 评《前线》、《北京日报》的资产阶级立场|accessdate =1966-05-11|publisher =《红旗》|author=[[戚本禹|戚本禹]] }}5月16日《人民日报》转载</ref>

5月17日夜，邓拓在给[[彭真|彭真]]、[[刘仁|刘仁]]写信时，用不少篇幅分析自己写《燕山夜话》和《三家村札记》时的背景和不足，竭力表白自己并非“反党反社会主义反毛泽东思想”<ref name="“砸烂旧世界”">{{cite book|title = 中华人民共和国史·卷六（1966-1968）“砸烂旧世界”|accessdate =2011年4月24日 |publisher =[[香港中文大学出版社|香港中文大学出版社]]|language = zh-hant |author=卜伟华|pages =109-110}}</ref>。

邓拓被打成“反党集团”，1966年5月18日留下绝笔后在北京家裏服藥自尽<ref>{{cite book |title = 兩個知識分子之死，《甲申年紀事》|author = [[董橋|董橋]]|date = 2004|language =zh-hant |publisher =[[香港|香港]]：[[牛津大學出版社|牛津大學出版社]]|pages =269-271}}</ref>。

===身后===
5月18日清晨，丁一岚发现丈夫死后，首先想到的不是悲痛，而是恐惧。据丁一岚回忆：

{{cquote|我当时也不敢失声痛哭，家里还住着警卫员呢。邓拓临死之前，放在桌上一个信封，里面装的好像稿纸，上面写着让我立即给这个东西销毁。我看见这个信封，让我马上销毁，哎呀，那时候我都想不出该怎么办，又怕惊动了警卫员。那时，邓拓的姐姐在我们家里，所以，我就急急忙忙地给她叫起来。我说，现在有这么个东西，要马上销毁，然后，我们俩就嘁哩喀喳地连剪带撕地弄碎，扔到厕所，还有澡盆里，拿水冲掉了。当时不敢烧哇，怕出火光，怕有味，可我就没看看销毁的是什么，是稿纸，那么一叠，字都朝里，我都来不及看了，我就想着，在警卫员醒来之前，我得处理完。<ref>{{cite journal|url=http://media.people.com.cn/GB/192301/192377/192680/193005/193194/12538878.html|title = 邓拓自杀的前前后后|author = 任捷|date =2010-08-25|language =zh-hans |publisher =[[人民网|人民网]]转载自[[南方周末|南方周末]]1999年6月25日刊}}</ref>
}}

邓拓死后，市委又来人抄家，按党的规定，邓拓的书信、照片、作品都要拿走。邓拓之死也被要求绝对保密，丁一岚甚至只能告诉子女他们的父亲因病住院了。直到两年后，子女们从社会上得知邓拓已死的消息，找母亲证实，丁一岚在请示北京市委后，才敢向子女们说出真相。<ref name="“砸烂旧世界”"/>

1966年6月2日，《人民日报》发表《歡呼北大的一張大字報》，开头写道：“[[聶元梓|聶元梓]]等同志的[[大字報|大字報]]，揭穿了“[[三家村_(文革用語)|三家村]]”黑幫分子的一個大陰謀！” ““三家村”黑店的掌櫃鄧拓被揭露出來了，但是這個反黨集團並不甘心自己的失敗。他們仍然負隅頑抗，用“三家村”反黨集團分子宋碩的話來說，叫作“加強領導，堅守崗位”。”<ref>[http://www.singtaonet.com:82/global/wg/wg10/t20060511_210383.html 歡呼北大的一張大字報] {{webarchive|url=https://web.archive.org/web/20070813125436/http://www.singtaonet.com:82/global/wg/wg10/t20060511_210383.html |date=2007-08-13 }}</ref>

1979年2月，中共为其平反并恢复名誉。1979年9月6日下午，在[[北京|北京]][[八宝山革命公墓|八宝山革命公墓]]礼堂举行邓拓追悼大会，由[[胡耀邦|胡耀邦]]主持，给予了邓拓极高的评价，称其为“优秀党员、忠诚的无产阶级革命战士”，后安葬于[[八宝山|八宝山]]<ref>{{cite news|url =http://www.hybsl.cn/xinwen/2007-10-31/4728.html|title =1979年9月5日 胡耀邦主持邓拓追悼会|accessdate =2009年6月9日|date =1979年9月6日|publisher =《人民日报》}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
