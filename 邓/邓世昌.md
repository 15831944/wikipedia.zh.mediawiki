'''邓世昌'''（{{bd|[[道光|道光]]二十九年（1849年）|8月20日|1894年|9月17日（[[农历|农历]]八月十八日）|catIdx=D鄧}}），原名'''永昌'''，字'''正卿'''，谥'''壮节'''，[[广东|广东]][[番禺|番禺]]龙导尾人（今[[广东省|广东省]][[广州市|广州市]][[海珠区|海珠区]]龙珠直街龙延里2号），[[中国|中国]][[北洋水师|北洋水师]][[将领|将领]]，[[甲午戰爭|甲午戰爭]]時[[陣亡|陣亡]]，[[追贈|追贈]][[太子少保|太子少保]]，入祀[[昭忠祠|昭忠祠]]，被譽為[[民族英雄|民族英雄]]。

== 早年生涯 ==
[[清|清]][[道光|道光]]二十九年（1849年）8月20日，邓世昌生于[[广东|广东]][[番禺|番禺]]一茶商之家，为近代中国第一批自己培养出来的海军将领之一。家境优越，幼年被父亲送去上海的教会学校，接受西式教育。1867年考入[[马尾船政学堂|马尾船政学堂]]驾驶班第一期学习，1871年被派至“[[建威|建威]]”舰练习航海，经过5年课堂教育、2年舰上实习，1874年以优异成绩毕业，被船政大臣[[沈葆桢|沈葆桢]]嘉奖以五品军功任命为“[[琛航|琛航]]”运船帮带，次年任“[[海东云|海东云]]”炮舰[[管带|管带]]。1874年5月[[牡丹社事件|牡丹社事件]][[日军|日军]]侵台，邓世昌奉命扼守[[澎湖|澎湖]]，[[基隆|基隆]]等要塞，得补[[千总|千总]]。

== 北洋水师 ==
[[File:致远号.JPG|200px]]

1880年调[[北洋水师|北洋水师]]。同年到英国接“[[揚威|揚威]]”巡洋舰，回国後任“扬威”管带。1887年再到英国接“[[致遠艦|致远]]”巡洋舰，任“致远”管带，職中軍中副將。是北洋海軍管帶中，唯一一位未曾出洋[[留學|留學]]或實習的管帶。1891年[[李鸿章|李鸿章]]检阅[[北洋海军|北洋海军]]，邓世昌因训练有功，获“葛尔萨[[巴图鲁|巴图鲁]]”勇名。

== 黃海海戰 ==
{{main|黄海海战 (1894年)}}
在中日[[甲午战争|甲午战争]][[黃海海戰|黃海海戰]]中，因“致远”舰受伤，彈藥用盡，难以继续战斗，邓世昌语曰：“倭舰专恃吉野，苟沉是舰，则我军可以集事。”便意图撞沉日舰主力“[[吉野號防護巡洋艦|吉野號巡洋艦]]”（据《东乡平八郎と秋山真之》记载为浪速），但“[[致远|致远]]”舰因为日舰攻击引起[[鱼雷|鱼雷]]爆炸而沉没，邓世昌决心与战舰同存亡，从而在落水后幾次推開救生圈，拒绝救援。<ref>[[:s:zh:清史稿/卷460|清史稿·卷460·列傳二百四十七·鄧世昌]]</ref>洋將[[馬吉芬|馬吉芬]]在书中则提到邓世昌的狗生性恶劣，在邓世昌抓住浮木时，爬上邓世昌身上，强使其放开浮木，以致其溺水。<ref>{{cite magazine|author1=Philo Norton McGiffin|magazine=''The Century Illustrated Monthly Magazine''|title=''The Battle of the Yalu''|url=https://books.google.com/books?id=A7glcjIh6qUC&pg=PA599|volume=50 |year=1895年|publisher=Century Company|page=599}}</ref>

邓世昌牺牲后举国震动，[[光绪帝|光绪帝]]垂泪撰联“此日漫挥天下泪，有公足壮海军威”，清廷[[諡號|谥]]以“壮节”，按提督例从优议恤并，还赐给其母一块用1.5[[公斤|公斤]][[黄金|黄金]]制成的“[[教子有方|教子有方]]”大匾，拨给邓家白银10万两以示抚恤，追赠[[太子少保|太子少保]]衔，入祀京师昭忠祠。邓家用此款在原籍[[番禺|番禺]]（今[[广东省|广东省]][[广州市|广州市]][[海珠区|海珠区]]）为邓世昌修了[[衣冠冢|衣冠冢]]，建起邓氏宗祠。[[抗日战争|抗日战争]]日军侵占广州期间，日军士兵慑于邓世昌的威望和英气，不敢破坏邓氏宗祠。

== 紀念船 ==
1996年12月28日，[[中国人民解放军海军|中国人民解放军海军]]以「世昌」命名[[世昌號國防動員艦|世昌號國防動員艦]]，以示纪念民族英雄邓世昌。

== 家庭 ==
鄧世昌育有三兒兩女，長子[[鄧浩鴻|鄧浩鴻]]，1947年去世；次子[[鄧浩洋|鄧浩洋]]，青年早逝；三子[[鄧浩乾|鄧浩乾]]是遺腹子，曾在[[中華民國|中華民國]]海軍部供過職，1969年逝于[[無錫|無錫]]。
