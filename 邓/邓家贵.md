== 风波 == 
* 2009年10月，香港《[[壹週刊_(香港)|壹周刊]]》报道，一座价值一亿五千万港元的香港豪宅，买家是[[张燕南|张燕南]]及其拍档[[北京中民信房地产|北京中民信房地产]]总经理邓家贵<ref>{{cite web |title = 习近平亲人在港购豪宅引发微词 |date = 2009-10-26 |url = http://www.rfi.fr/actucn/articles/118/article_17047.asp |author = 《[[壹周刊|壹周刊]]》 |publisher = [[法国国际广播电台|法国国际广播电台]]}}</ref>，而张和邓均为习近平的亲戚。
* 针对外界关于万达“背景”的质疑，2015年10月30日，[[万达集团|万达集团]]董事长[[王健林|王健林]]在美国[[哈佛大学|哈佛大学]]的公开课上回答主持人提问时表示，2009年7、8月，万达集团旗下的万达商业地产为解决资金问题，展开了一轮股权私募，委托[[银河证券|银河证券]]和[[中金证券|中金证券]]两家券商代为寻找投资人，在最终入股的十几家公司中包括习近平的姐夫邓家贵的秦川大地公司，邓和其他十几家公司以同样价格进来，采用的是市场化的购买价格。而在万达商业地产公司2014年12月份于香港上市前的两个月，邓家贵将其所持有的股份以较低的价格转让，王健林称其“牺牲了巨大利益”。王健林同时表示，他认为邓家贵“应该是受到了某种影响或者压力”，暗指邓受到了来自习家方面的压力。他还认为，这件事与腐败无关，反而证明了习近平“治国严治家更严的风格”。
*2016年曝光的[[巴拿马文件|巴拿马文件]]显示邓家贵在[[习近平|习近平]]成为中国最高领导人之前就拥有三家离岸公司，之前的2012年[[彭博社|彭博社]]也曾报道关于邓家贵及其妻齐桥桥的商业帝国新闻。<ref>{{cite web|title=巴拿马文件再撼中共高层：刘云山张高丽家人上榜|url=http://cn.nytimes.com/china/20160407/c07chinapanama/?utm_source=mostviewed-weekly&utm_medium=cpc&utm_campaign=mostviewed|publisher=纽约时报中文网}}</ref>随后在中国网络上，“[[姐夫|姐夫]]”一词成为[[敏感词|敏感词]]。<ref>{{cite web|url=https://chinadigitaltimes.net/chinese/2016/04/%E3%80%90%E7%BD%91%E7%BB%9C%E6%B0%91%E8%AE%AE%E3%80%91-19/|title=【网络民议】看敏感词库已到姐夫|publisher=中国数字时代}}</ref>
