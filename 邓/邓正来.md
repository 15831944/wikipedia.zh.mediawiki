'''邓正来'''{{BD|1956年|2月|2013年|1月24日|catIdx=D}}上海人，毕业于[[四川外语学院|四川外语学院]]、[[外交学院|外交学院]]，在[[复旦大学|复旦大学]]执教并担任多项公职，为知名的[[翻译家|翻译家]]、[[法学家|法学家]]、[[政治学家|政治学家]]，被誉为“中国第一个学术个体户”，是[[西学东渐|西学东译]]重要人物，他和[[梁治平|梁治平]]、[[贺卫方|贺卫方]]、[[朱苏力|朱苏力]]一同被誉为“中国法学四剑客”。<ref name=" fish" >{{cite web |url= http://yule.sohu.com/20130125/n364578218.shtml|title=法学家邓正来病逝 被称“第一个学术个体户” |accessdate=2013年1月25日 |publisher=搜狐网 |work= |language=zh-cn}}</ref><ref name=" tiger">{{cite web |url=http://www.chinadaily.com.cn/hqgj/jryw/2013-01-25/content_8123633.html |title=著名法学家邓正来去世 |accessdate=2013年1月25日 |publisher=中国日报网 |work= |language=zh-cn }}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web|url=http://cn.ibtimes.com/articles/20380/20130125/deng-zhenglai-he-weifang-zhusuli-liangzhiping-law-die.htm|title=邓正来和法学“四剑客”梁治平、贺卫方、朱苏力|accessdate=2013年1月25日|publisher=IBtimes中文网|work=|language=zh-cn}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url= http://news.cnhubei.com/xw/gn/201301/t2433511_1.shtml|title= 著名学者邓正来逝世 是我国西学东译重要代表人物|accessdate=2013年1月25日 |publisher=荆楚网 |work= |language=zh-cn}}</ref>

1956年，邓正来出生在[[上海市|上海市]]。<ref name=" fly" />1978年，[[邓小平|邓小平]]恢复高考制度，邓正来考上[[四川外语学院|四川外语学院]]，<ref name=" fly" />1982年，邓正来又考上[[外交学院|外交学院]]。<ref name=" fly" />大学毕业后曾长期在民间研究法学和政治学。1989年，在“六四”风波中前往公安机关秘密自首，上交了他假借“秘书处”名义偷偷录制“首都知识界联合会”等组织活动的全部录音带，结果导致多人被捕、判刑。<ref name=" sb" >{{cite web |url=http://www.boxun.com/news/gb/pubvp/2009/07/200907140900.shtml |title=季羡林与“六四”大屠杀 |accessdate=2014年8月13日 |publisher=博讯网 |work= |language=zh-cn}}</ref><ref name=" sb2" >{{cite web |url=http://www.boxun.com/news/gb/pubvp/2013/02/201302020228.shtml |title=纪念邓正来的窃听使命 |accessdate=2014年8月13日 |publisher=博讯网 |work= |language=zh-cn}}</ref>2003年，好友邀请他到[[吉林大学|吉林大学]]执教，入校前提出“只教书带学生不任职”的两个原则，<ref>{{cite web |url=http://www.chinadaily.com.cn/hqgj/jryw/2013-01-25/content_8124754.html |title=做学问，关乎独立更关乎生命 |accessdate=2013年1月25日 |publisher=中国日报网 |work= |language=zh-cn |archive-url=https://web.archive.org/web/20130128103243/http://www.chinadaily.com.cn/hqgj/jryw/2013-01-25/content_8124754.html |archive-date=2013年1月28日 |dead-url=yes }}</ref>2008年转入[[复旦大学|复旦大学]]，把[[英国|英国]][[经济学家|经济学家]][[弗里德里希·哈耶克|弗里德里希·哈耶克]]著作研究引入复旦大学。<ref name=" fish" /><ref>{{cite web |url=http://culture.ifeng.com/whrd/detail_2013_01/24/21553425_0.shtml |title=冯克利：邓正来以民间学者身份将哈耶克推荐到中国 |accessdate=2013年1月25日 |publisher=凤凰网 |work= |language=zh-cn}}</ref>2013年1月24日，邓正来因[[胃癌|胃癌]]在复旦大学附属肿瘤医院病逝。<ref name=" tiger" />

== 作品与思想 ==
邓正来一生有著作25部，译著19部，主编书刊43种，论文58篇。<ref name=" fly" >{{cite web |url=http://news.hexun.com/2013-01-25/150562114.html |title= 邓正来教授逝世 学界豪杰曾为“学术个体户”|accessdate=2013年1月25日 |publisher=和讯网 |work= |language=zh-cn}}</ref>邓正来长期研究西方法学和政治学，主持翻译西方[[人文科学|人文社会科学]]经典著作，致力于中国大陆法学和政治学启蒙，主张[[学术自救|学术自救]]。<ref name=" fish" /><ref name=" man" />
