'''邓六条'''为於1983年6月26日，时任[[中国共产党中央顾问委员会|中国共产党中央顾问委员会]]主任的[[邓小平|邓小平]]会见美国西东大学教授杨力宇时，所提出的解决[[台湾问题|台湾问题]]的六条方针。此次谈话的要点收入《[[邓小平文选|邓小平文选]]》第三卷，题为《中国大陆和台湾和平统一的设想》<ref name="a1">{{Cite web |url=http://dangshi.people.com.cn/n/2014/1124/c85037-26078979-3.html |title=邓小平殚精竭虑解决台湾问题|accessdate=2019-03-27 |author=蒋永青 |date=2014-11-24 |work=责编：张湘忆、谢磊 |publisher=中国共产党新闻网|language=Zh-hans }}</ref>。其方针思想大致与[[一国两制|一国两制]]构想相同。

1979年至1982年间，杨力宇曾11次访问中国大陆。1983年3月，美国旧金山亚洲问题研究协会年会上，杨力宇将“中国统一之展望”列为讨论专题之一。会上，被视为[[蔣經國|蒋经国]]代表的[[丘宏达|丘宏达]]提出了被认为代表中华民国方面的和谈条件。邓小平认为他的误解太深，愿意当面澄清。6月26日的会谈中，邓小平的陪同人员有中共中央政治局委员、中央军委常务副主席兼秘书长[[杨尚昆|杨尚昆]]（负责[[中央对台工作|中央对台工作]]），中央书记处书记、中宣部部长[[邓力群|邓力群]]，[[中央对台工作领导小组|中央对台工作领导小组]]副组长[[汪锋|汪锋]]，中国社会科学院院长[[马洪|马洪]]等人<ref name="a1"/>。

1987年，[[蔣經國|蒋经国]]的[[一國良制|一国良制]]则被视为是对1983年[[一国两制|一国两制]]的回应。

==内容==

原载于1983年6月26日的《[[人民日报|人民日报]]》<ref>{{Cite web |url=http://www.huaxia.com/zt/2003-54/pl/00140185.html |title=邓小平中国大陆和台湾和平统一设想（邓六条）|accessdate=2019-03-27 |author= |date=2003-03-11 |work=|publisher=华夏经纬网，来源：《人民日报》|language=Zh-hans }}</ref>

#台湾问题的核心是祖国统一。和平统一已成为国共两党的共同语言。
#制度可以不同，但在国际上代表中国的，只能是中华人民共和国。
#不赞成台湾“完全自治”的提法，“完全自治”就是“[[两个中国|两个中国]]”，而不是一个中国。自治不能没有限度，不能损害统一的国家的利益。
#祖国统一後，台湾特别行政区可以实行同大陆不同的制度，可以有其他省、市、自治区所没有而为自己所独有的某些权力。司法独立，终审权不须到北京。台湾还可以有自己的军队，只是不能构成对大陆的威胁。大陆不派人驻台，不仅军队不去，行政人员也不去。台湾的党、政、军等系统都由台湾自己来管。中央政府还要给台湾留出名额。
#和平统一不是大陆把台湾吃掉，当然也不能是台湾把大陆吃掉，所谓“[[三民主义统一中国|三民主义统一中国]]”不现实。
#要实现统一，就要有个适当方式。建议举行两党平等会谈，实行国共第三次合作，而不提中央与地方谈判。双方达成协定後可以正式宣布，但万万不可让外国插手，那样只能意味着中国还未独立，後患无穷。

== 为统一大业，更改国号 ==

香港《[[文汇报_(香港)|文汇报]]》的[[社论|社论]]指，邓小平曾在此次会谈中表示，只要能够统一，中华人民共和国的[[国号|国号]]可以改{{efn|搜狐网转载[[香港文汇报|香港文汇报]][[社评|社评]]<ref>{{Cite web |url=http://text.news.sohu.com/40/76/news147117640.shtml |title=香港文汇报社评：陈水扁倒打一耙的伎俩|accessdate=2019-03-27 |author= |date=2001-11-05 |work= |publisher=搜狐网|language=Zh-hans }}</ref>“祖国大陆多次明确表示，在“[[一個中國_(中華人民共和國)|一个中国]]”原则下，什么都可以谈。这当然包括两岸统一之后的国号、国旗、国歌等。而邓小平早在八三年会见美国华人学者杨力宇时就曾表示，“问题的核心是祖国统一”，“如果能够统一，国号也可以改”。可见，祖国大陆为了促进两岸和平统一，在祖国统一即“一个中国”的核心原则之下，展示出了极大的诚意和包容性与灵活性。”}}，或称，邓小平说只要承认一个中国，国号、国旗、国歌都可以谈<ref name="a3"/>。但邓小平为统一大业，可以更改国号的观点确实未收入当日《人民日报》的邓六条，而“在‘[[一个中国|一个中国]]’原则下，什么都可以谈”的观点则被普遍的接受。2002年时，继任领导人[[江泽民|江泽民]]在[[中共十六大|中共十六大]]报告中，重申“在‘一个中国’原则下，什么都可以谈”，后被总结为[[三个可以谈|三个可以谈]]。

2012年12月，台湾举办“九二共识”二十周年论坛，会议中，大陆学者表示等两岸达成“一中共表”的共识，可协商国号。对于协商国号的建议，[[国台办|国台办]][[国务院台湾事务办公室新闻发言人|新闻发言人]][[范丽青|范丽青]]在例行新闻发布会上的回应，重申了“只要坚持一个中国，两岸之间什么都可以谈<ref>{{Cite web |url=http://politics.people.com.cn/n/2012/1226/c70731-20025125.html |title=国台办回应两岸可以协商“国号”建议|accessdate=2019-03-27 |author= |date=2012-12-26 |work= |publisher=人民网，来源：中国网|language=Zh-hans }}</ref>”。但有分析指，进入2010年代后，随中华人民共和国经济实力增强，已非1980年代远逊于中华民国（台湾）的状况。中华人民共和国方面已无为追求[[中国统一|中国统一]]，而更改国号的可能性<ref name="a3">{{Cite web |url=http://phtv.ifeng.com/a/20160822/44440310_0.shtml |title=李杰：解决台湾问题有三种可能 偏绿人士称不会坐以待毙|accessdate=2019-03-27 |author= |date=2016-08-22 |work= |publisher=凤凰网，来源：凤凰卫视|language=Zh-hans }}</ref>。
