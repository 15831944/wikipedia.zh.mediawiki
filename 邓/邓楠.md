'''鄧楠'''{{BD|1945年|10月|||catIdx=Deng鄧}}，籍贯[[四川|四川]][[广安|广安]]，生于[[河北|河北]][[涉县|涉县]]。[[邓小平|邓小平]]之女，[[中國共產黨|中國共產黨]]党員。[[中国人民政治协商会议|中国人民政治协商会议]][[中国人民政治协商会议第十二届全国委员会|第十二届全国委员会]][[中国人民政治协商会议全国委员会常务委员会|常务委员]]、教科文卫体委员会副主任委员。

== 生平 ==

1945年出生于[[涉县|涉县]][[八路军一二九师司令部旧址|129师司令部家属院]]。

1964年毕业于原北京实验中学（现[[北京师范大学附属实验中学|北京师范大学附属实验中学]]前身）高中1班。1970年畢業於[[北京大學物理系|北京大學物理系]]。

曾在[[中國科學院|中國科學院]]自動化研究所及[[中国科学院半导体研究所|中国科学院半导体研究所]]任研究實習員，擔任原中华人民共和国科学技术委员会（现[[中华人民共和国科学技术部|中华人民共和国科学技术部]]前身）政策局副局長、社會發展司司長。

1991年11月，就任國家科委副主任。1998年任[[中華人民共和國科學技術部|中華人民共和國科學技術部]]副部長。

2003年，当选[[中国人民政治协商会议第十届全国委员会委员名单|中国人民政治协商会议第十届全国委员会委员]]。

2004年10月起任[[中国科学技术协会|中国科学技术协会]][[党组书记|党组书记]]，11月任中国科协书记处[[第一书记|第一书记]]<ref name="ifeng">{{cite web |url = http://news.ifeng.com/mainland/detail_2011_05/05/6173791_0.shtml |title = 中科协证实陈希接替邓楠任中国科协党组书记 |publisher = 凤凰网资讯}}</ref>。

2005年任[[中國科協|中國科協]]常務副主席。

2007年10月當選[[中国共产党第十七届中央委员会|中国共产党第十七届中央委员会]]委員。

2011年4月，不再担任中国科协职务<ref name="ifeng" />。

2013年，当选全国政协十二届常委及科教文卫体委员会副主任。

== 邓楠与赵紫阳下台 ==
据[[鲍彤|鲍彤]]分析，邓小平与[[赵紫阳|赵紫阳]]分裂，导致赵紫阳下台，原因中邓小平的子女起了非常坏的作用。邓小平的子女不知道邓小平、赵紫阳、[[中联部|中联部]]在中苏最高级会谈上的有关安排与意外，因此误解了赵紫阳对[[戈尔巴乔夫|戈尔巴乔夫]]所说“关于邓小平在党内作用”的一席话<ref name="rqzkdn">{{cite web |url = http://biweekly.hrichina.org/article/8124 |title = 八九民运与赵紫阳（下）|publisher=《中国人权》双周刊}}</ref>。曾担任政府官员的张刚回忆，邓楠给赵紫阳打了一个小时电话，从头到尾都是骂<ref>{{cite web |url = http://cn.nytimes.com/china/20140512/c12tiananmen/ |title = 天安门事件的核心一幕：赵紫阳的下台 |publisher=纽约时报 |date=2014年5月12日}}</ref>。声称赵紫阳把邓小平抛出去了，骂完直接把电话挂掉；邓小平的三女儿[[邓榕|邓榕]]则拒绝接听鲍彤试图通过她来沟通解释的电话。邓小平耳聋，没有这两个女儿做中介，无法与外界沟通，因而出现了赵紫阳所说的“小孩子干政”<ref name="rqzkdn" />。

== 家庭 ==
*丈夫：[[张宏|张宏]]，出生于江苏省泰州一个普通的农民家庭，3儿一女中张宏排行老二，毕业于[[北京大学|北京大学]]。
*女儿：[[邓卓芮|邓卓芮]]，1972年11月14日在江西出生；小名眠眠。
*女婿：[[吴小晖|吴小晖]]，1966年10月18日出生于平阳县萧江镇周宅村。安邦保险董事长兼CEO，北京国通高盛投资有限公司董事长。[[邓卓芮|邓卓芮]]是其第三任妻子。
