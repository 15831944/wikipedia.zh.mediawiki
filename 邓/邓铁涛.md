'''邓铁涛'''（{{bd|1916年|11月6日|2019年|1月10日|catIdx=Deng鄧}}），[[广东|广东]][[开平县|开平]]人，中国[[中醫師|中醫師]]，[[广州中医药大学|广州中医药大学]]终身教授，中医内科学专业博士研究生导师，首届[[國醫大師|國醫大師]]。2019年1月10日凌晨六点06分逝世，享寿104岁<ref>{{Cite web|url=https://news.sina.com.cn/c/2019-01-10/doc-ihqfskcn5745739.shtml|title=国医大师邓铁涛今晨逝世|accessdate=2019-01-10|date=2019-01-10|last=新京报|work=news.sina.com.cn}}</ref>，追悼会于1月16日在广州殡仪馆白云厅举行<ref>{{cite web |title=社会各界送别“国医大师”邓铁涛 |url=http://www.chinanews.com/tp/hd2011/2019/01-16/862538.shtml |website=中国新闻网 |accessdate=2019-01-20 |date=2019-01-16}}</ref><ref>{{cite web |title=104岁国医大师邓铁涛追悼会在广州举行 |url=http://www.chinanews.com/sh/2019/01-16/8730803.shtml |website=中新网 |accessdate=2019-01-20 |date=2019-01-16}}</ref>。

== 争议 ==
中國作家[[方舟子|方舟子]]曾转载医学专业网站[[丁香园|丁香园]]上的網路文章，其中聲称邓铁涛不具有现代医学的常识，且邓的儿子和妻子均被其误诊而死。传言其子患肠梗阻便血，其诊断为“大厂湿热”，利用传统中医药治疗半年无果，后查出结肠癌晚期。其妻胸口疼痛，其诊断为“胃腕痛”，后由于心脏病去世。<ref>{{Cite web|url=http://xysblogs.org/fangzhouzi/archives/884|title=方舟子的Blog » “中医泰斗”邓铁涛妻子、儿子因误诊死亡情况|accessdate=2019-01-10|language=en}}</ref>
