《'''邓小平'''》，中国大陆电影，总导演[[丁荫楠|丁荫楠]]，主演卢奇、王苏娅。

== 剧情 ==
该片讲述[[文化大革命|文化大革命]]结束之后，[[邓小平|邓小平]]复出主持工作，主持召开[[中共十一届三中全会|中共十一届三中全会]]，拨乱反正、[[平反|平反]]冤案、恢复[[高考|高考]]等事迹。<ref>{{cite web |url=http://news.qq.com/a/20110520/000369.htm |title=电影《邓小平》 |accessdate=2011年5月20日 |publisher=腾讯网 |archiveurl= http://news.qq.com/a/20110520/000369.htm|archivedate=2013年1月3日 |work= |language=zh-cn}}</ref>

== 花絮 ==
该片主要呈现[[改革开放|改革开放]]之后邓小平的形象。<ref>{{cite web |url=http://yule.sohu.com/20090710/n265131670.shtml |title=《邓小平》中“邓小平”形象 |accessdate=2009 |publisher=搜狐网 |archiveurl=http://yule.sohu.com/20090710/n265131670.shtml |archivedate=2013年1月3日 |work= |language=zh-cn}}</ref>

该片2003年开始筹备摄制，2007年才上映。<ref>{{cite web |url=http://news.sina.com.cn/c/2007-02-01/122512201388.shtml |title= 电影：《邓小平》|accessdate= 2007年2月1日 |publisher=新浪网 |archiveurl=http://news.sina.com.cn/c/2007-02-01/122512201388.shtml |archivedate=2013年1月3日 |work= |language=zh-cn}}</ref>

2008年，该片参加“第九届中国[[长春电影节|长春电影节]]”。<ref>{{cite web |url= http://yule.sohu.com/20080705/n257965729.shtml|title= 长春电影节 与观众同温《邓小平》|accessdate=2008年7月5日 |publisher=搜狐网 |archiveurl=http://yule.sohu.com/20080705/n257965729.shtml |archivedate=2013年1月3日 |work= |language=zh-cn}}</ref>
