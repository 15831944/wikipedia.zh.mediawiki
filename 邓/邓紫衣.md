'''邓紫衣'''{{BD|1986年|5月21日|||catIdx=D邓}}出生于[[湖南省|湖南省]][[益阳市|益阳市]]，为[[中国|中国]]影视演员、歌手。

== 生平 ==
1986年，邓紫衣出生在[[湖南省|湖南省]][[益阳市|益阳市]]，15岁时，邓紫衣独自北漂，由于嗓音天赋，他在小饭馆100元一场开始唱歌，一直唱到2004年喉咙发炎才停止。<ref name=" cat" /><ref name=" fish" />2004年，导演[[崔子恩|崔子恩]]找到了她，拍摄了她的处女作《[[少年花草黄|少年花草黄]]》，从未学过表演的她大获成功，随后，独立电影教父[[吕建民|吕建民]]邀她加入了“北京春秋时代文化有限公司”，开始了她的职业生涯。<ref name=" cat" >{{cite web |url=http://ent.163.com/12/0514/14/81FK3JJP00031H2L.html |title=邓紫衣《金鹰报》专访 “影坛乖乖女”十年蜕变 |accessdate=2012-05-14 |publisher= 网易|archiveurl=http://ent.163.com/12/0514/14/81FK3JJP00031H2L.html |archivedate=2012年12月27日 |work= |language=zh-cn}}</ref><ref name=" fish" >{{cite web |url=http://yue.ifeng.com/news/detail_2012_05/14/14516854_0.shtml |title= 邓紫衣接受专访 “影坛乖乖女”十年蜕变成蝶|accessdate=2012年5月14日 |publisher=凤凰网 |archiveurl=http://yue.ifeng.com/news/detail_2012_05/14/14516854_0.shtml |archivedate=2012年12月27日 |work= |language=zh-cn}}</ref><ref name=" bull" >{{cite web |url=http://et.21cn.com/collect/2011/12/06/10042206.shtml |title= 邓紫衣录制《身边的幸福》筹备专辑进军乐坛|accessdate=2011-12-06 |publisher=21CN |archiveurl=http://et.21cn.com/collect/2011/12/06/10042206.shtml |archivedate= 2012年12月27日|work= |language=zh-cn}}</ref>
