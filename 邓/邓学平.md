'''邓学平'''（{{bd|1982年|||}}）是一名中国律师，毕业于[[上海财经大学|上海财经大学]]，曾任职检察官七年，后辞职以律师身份工作<ref name="qingfeng">{{cite web |title=邓学平：做律师更贴近我的法治理想 |url=http://www.lsqingfeng.com/news_view.php?Id=11852 |website=www.lsqingfeng.com |accessdate=2019-04-05 |language=en}}</ref>。现任[[上海财经大学|上海财经大学]]和[[上海政法学院|上海政法学院]]兼职硕士生导师，[[华东师范大学|华东师范大学]]法律教育与法律职业研究院研究员，兼任[[中国法学会|中国法学会]]会员、中国律师协会会员。曾代理[[中华人民共和国公安部|公安部]]挂牌督办、[[国际刑警组织|国际刑警组织]]红色通缉令海外引渡等影响较大案件<ref>{{cite web |author1=民主与法制时报 |title=邓学平：社会热点案件让法律人谨守理性 |url=http://www.mzyfz.com/index.php/cms/item-view-id-1380428?from=singlemessage&isappinstalled=0 |website=http://www.mzyfz.com |accessdate=2019-04-05}}</ref>，也是[[张扣扣案|张扣扣案]]和[[雷洋事件|雷洋]]案的代理律师。著有《法影下的中国》，《给正义一点时间》，同时是《精英的浮沉——中国企业家犯罪报告》一书的主要作者<ref>{{Cite web|url=http://opinion.caixin.com/dengxueping_shwx/|title=邓学平_观点频道_财新网|accessdate=|last=|work=|author=|date=|publisher=}}</ref>。

== 生平 ==
邓学平出生于[[湖北|湖北]][[十堰|十堰]]的农村，13岁时离开家乡到郧西县第二中学上初中。后进入郧阳中学。2001年，进入[[上海财经大学|上海财经大学]]，主修法律。后来在上海财经大学获得经济法硕士学位。2007年，在苏州一检察院开始其检察官工作。2014年6月，辞去苏州市某区检察院公诉处副处长的职务，进入京衡律师集团上海事务所，成为律师。 <ref name="qingfeng" />
