1997年2月19日21时08分，[[中国共产党|中国共产党]]“第二代[[中国共产党集体领导制度|中央领导集体]]的[[领导核心|核心]]”、一度的实际[[中华人民共和国最高领导人|最高领导人]][[邓小平|邓小平]]在北京逝世，享年93岁。死因為[[帕金森病|帕金森病]]晚期併發肺部感染，致使呼吸循环功能衰竭，最終抢救无效。

==病情发展==
邓小平自1994年10月1日后没有公开露面<ref>{{cite web  |url = http://cpc.people.com.cn/GB/64093/64387/5382541.html|title=激情与宁静：邓小平的最后五年 (2)|work=[[南方周末|南方周末]] |date=2007-02-08|accessdate=2014-07-24}}</ref>。1996年12月12日，邓小平因呼吸道疾病住进[[中国人民解放军总医院|中国人民解放军总医院]]南楼病房<ref name="nfzm">{{cite web|author=张悦|coauthors=赵蕾|url=http://www.infzm.com/content/6015|title=十年前，那个令人悲伤的夜晚|work=[[南方周末|南方周末]]|date=2007-02-08|accessdate=2014-07-24|deadurl=yes|archiveurl=https://web.archive.org/web/20160412120203/http://www.infzm.com/content/6015|archivedate=2016-04-12}}</ref>，此前邓小平被多次误报已经逝世。[[中国中央电视台综合频道|中国中央电视台第一套节目]]于1997年1月1日起播出纪录片《邓小平》，当时已患[[帕金森氏症|帕金森氏症]]晚期的邓小平在病房收看了全部12集。<ref>{{cite web |url = http://news.sina.com.cn/c/2004-08-12/19213384892s.shtml |title=大型电视文献纪录片《邓小平》背后的故事 |work=东方网 |date=2004-08-12 |accessdate=2014-07-24}}</ref>1997年2月初（即[[春節|春節]]前夕），病情略有平稳<ref>{{cite news |url = http://www.people.com.cn/GB/shizheng/252/7443/7444/20020210/667773.html |title = 解放军总医院：怀念“老南楼”灯光 |work=[[人民日报|人民日报]] |date=1997-02-26 |accessdate=2014-07-24}}</ref>的邓小平向前去探望的[[中国共产党中央委员会总书记|中国共产党中央委员会总书记]][[江泽民|江泽民]]传达了自己对全国各族人民的节日祝贺，并希望在以江泽民为核心的[[中共中央|中共中央]]领导下“把今年恢复[[香港回歸|对香港行使主权]]和召开[[中国共产党第十五次全国代表大会|党的十五大]]这两件大事办好”<ref name="nfzm" />；同月，离京的[[党和国家领导人|党和国家领导人]]全部赶回北京<ref name="nfzm"/>。

1997年2月15日，邓小平夫人[[卓琳|卓琳]]及子女致信江泽民，向其转告邓小平的临终嘱托<ref name="df">{{cite web |url = http://news.sina.com.cn/c/2004-08-13/16003391305s.shtml |title=捐献角膜和遗体——邓小平的临终时刻与身后事 |work=东方网 |date=2004-08-13 |accessdate=2014-07-24}}</ref><ref>{{cite web |url = http://www.people.com.cn/GB/shizheng/252/7443/7446/20020209/667194.html |title = 邓小平同志的家属致江总书记并党中央的信 |publisher=新华社 |date=1997-02-15 |accessdate=2014-07-24}}</ref>。2月19日，邓小平的呼吸功能已经衰竭，只能借助机器呼吸；当晚21时08分，医疗组宣布停止抢救，邓小平以93岁高龄逝世。

== 媒体报道 ==
2月19日21时许，[[中共中央办公厅|中共中央办公厅]]紧急通知[[中国中央电视台|中国中央电视台]]新闻中心派两组记者赶往解放军总医院，中国中央电视台台长[[杨伟光|杨伟光]]亦从[[顺义区|顺义县]]（后改为顺义区）赶回总台以等待正式[[新闻稿|新闻稿]]<ref name="luojing">{{cite web |url = http://www.infzm.com/content/29806 |title = “再复制一个罗京不那么容易” 专访中央电视台副台长孙玉胜 |work=[[南方周末|南方周末]] |date=2009-06-11 |accessdate=2014-07-24}}</ref>。

香港[[傳訊電視|傳訊電視]]（CTN）创办人[[于品海|于品海]]于2月19日23时许得知邓小平逝世的消息后，紧急召回[[中天新聞台|CTN中天频道]]编辑部工作人员。中天频道编辑部获悉北京2月19日23时许的反常情况，并向多名省部级领导求证后，于2月20日1时18分许播出邓小平逝世的消息<ref>{{cite web |url=http://www.oeeee.com/Channel/content/2007/200702/20070217/456350.html |title=CTN中天频道陈庆源回忆十年前那夜的新闻大战|work=[[南方都市报|南方都市报]] |date=2007-02-17 |accessdate=2014-07-24}}</ref><ref>{{cite web |url = http://news.ifeng.com/history/1/200702/0217_335_78007.shtml |title=曹景行：独家率先披露邓小平去世消息前后 |website = 凤凰网 |date = 2007-02-17 |accessdate=2014-07-24}}</ref>。

美国[[有线电视新闻网|有线电视新闻网]]（CNN）于2月20日凌晨0时之后，引述“尚未证实的消息”称邓小平已经逝世。20日凌晨2时42分，[[路透社|路透社]]播发邓小平逝世的消息，[[新华社|新华社]]于2分钟后正式公布此消息；中国中央电视台同时插播“重要新闻”，由[[播音員|播音員]][[罗京|罗京]]播报《[[告全党全军全国各族人民书|告全党全军全国各族人民书]]》<ref name="luojing" />。

== 悼念 ==
1997年2月20日早7时01分，北京[[天安门广场|天安门广场]]升旗仪式[[半旗|降半旗]]以悼念邓小平<ref name="df" />，纽约[[联合国总部大楼|联合国总部大楼]]当天亦下半旗。由于1997年[[元宵节|元宵节]]为2月21日（邓小平逝世后两天），民众在21日当天停止原定的元宵节活动。

{{Wikisource|中共中央、全国人大常委会、国务院、全国政协、中央军委告全党全军全国各族人民书}}
2月25日上午10时，中共中央、全国人大常委会、国务院、全国政协、中央军委在[[人民大會堂|人民大會堂]]大禮堂舉行“鄧小平同志追悼大會”，[[中华人民共和国国务院总理|国务院总理]][[李鹏|李鹏]]宣布大会开始，中共中央總書記江澤民致悼詞，大会在《[[国际歌|国际歌]]》歌声中结束<ref>{{cite web|title=邓小平同志追悼大会在北京隆重举行|url=http://www.people.com.cn/GB/shizheng/252/7443/7446/20020209/667243.html|website=人民日報|date=1997-02-26|accessdate=2017-02-25}}</ref>。

2月25日上午10时（即邓小平追悼大会开始时），[[九廣鐵路|九廣鐵路]]上进出[[紅磡站|紅磡站]]的列车鸣笛3分钟，以示哀悼。

[[中國民運|中國民運]]人士[[王軍濤|王軍濤]]專程去[[中华人民共和国驻美国大使馆|中华人民共和国驻美国大使馆]]送花圈公開悼念鄧小平，此事當時在民運圈頗有非議。

前[[民主進步黨|民主進步黨]]資深黨員[[王老養|王老養]]（後來的[[台湾共产党_(2008年)|台湾共产党]]創黨主席）想去北京祭拜邓小平，但在[[澳门机场|澳门机场]]转机时突然被[[李登輝政府|李登輝政府]]方面告知不能去，最后只好在澳门机场遥寄哀思。王老養說，他沒有讀過[[马克思|马克思]]與[[列宁|列宁]]的著作、也不了解中共老一辈知名领袖，但他十分敬佩邓小平，因为邓小平“为国家和人民做了很多贡献”<ref>{{Cite web|url=http://www.chinanews.com/tw/twyw/news/2008/08-04/1334239.shtml|author=[[環球時報|環球時報]]|title=台湾共产党主席：不懂马列，但很敬佩邓小平|publisher=[[中新网|中新网]]|date=2008-08-04|accessdate=2018-06-21}}</ref>。

== 火化 ==
2月24日上午9時34分，鄧小平的遺體被送至[[八寶山革命公墓|八寶山革命公墓]]火化。从解放军总医院到[[八宝山|八宝山]]的[[复兴路_(北京)|复兴路]]至[[石景山路|石景山路]]一线聚集了前去悼念的百萬民众，[[北京大学生命科学学院|北京大学生命科学学院]]部分大学生更打出写有“再道一声：小平您好”的横幅，而在1984年国庆游行时举出“小平您好”标语的正是来自北京大学生物系的大学生<ref>{{cite news |url = http://www.people.com.cn/GB/shizheng/252/7443/7446/20020209/667237.html |title=长街洒泪 万众同悲 |newspaper = [[人民日报|人民日报]]|date=1997-02-25 |accessdate=2014-07-25}}</ref><ref>{{cite news |url = http://www.people.com.cn/GB/shizheng/252/7443/7446/20020209/667242.html |title = 小平，我们永远爱您——首都十多万群众挥泪送小平 |newspaper = [[人民日报|人民日报]] |date=1997-02-25 |accessdate=2014-07-25}}</ref>。3月2日，遵遗嘱，骨灰撒入大海，其[[角膜|角膜]]亦捐出<ref>{{cite news |language = zh-hans |author = |coauthors = |url =http://news.sina.com.cn/c/2004-08-13/16003391305s.shtml |title = 捐献角膜和遗体——邓小平的临终时刻与身后事 |work = |publisher = 新浪 |pages = |date = 2004-08-13|accessdate = 2013-11-03 }}</ref>。
