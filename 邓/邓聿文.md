'''邓聿文'''{{BD|1968年|3月|||catIdx=D}}，男，[[江西|江西]][[新余|新余]]人，[[中国国民党革命委员会|中国国民党革命委员会]]党员，[[记者|记者]]、[[作家|作家]]、[[评论员|评论员]]，著有《幸福的权利》、《中国必须赢》、《中国经济大突围》等，主张[[自由|自由]]和[[民主|民主]]，曾是[[中共中央党校|中共中央党校]]校刊《[[学习时报|学习时报]]》副编审。[[中国国民党革命委员会|民革]]中央社会与法制委员会委员、北京改革和发展研究会理事。英国诺丁汉大学中国政策研究所访问学者。<ref>{{cite web |url= http://news.takungpao.com/special/dyw/|title=邓聿文_大公网 |accessdate= 2014年6月2日|publisher= 大公网|work= |language=zh-cn}}</ref>
<ref>{{cite web |url= http://news.hexun.com/2012/dyw/|title=作者简介：邓聿文 |accessdate= 2013年4月3日|publisher= 和讯网|work= |language=zh-cn}}</ref>2012年曾经撰写长文《胡温的政治遗产》，对胡温执政的十年进行了较为严厉的批评。2013年3月，邓聿文在[[英国|英国]]《[[金融时报|金融时报]]》公开撰文《中国应该抛弃朝鲜》批评[[中华人民共和国|中华人民共和国]]对[[朝鲜|朝鲜]]的现行政策，提出丢弃朝鲜，不久之后，其在学习时报的职务被停。<ref>{{cite web |url= http://www.rfa.org/mandarin/yataibaodao/junshiwaijiao/ch-04022013155259.html|title= 中共党校校刊副编审邓聿文呼吁“弃朝鲜”遭停职|accessdate=2013年4月3日 |publisher=自由亚洲电台 |work= |language=zh-cn}}</ref><ref>{{cite web |url= http://cn.nytimes.com/china/20130402/c02editor/|title= 党校编审质疑中朝关系被停职|accessdate=2014年6月2日 |publisher=纽约时报中文网 |work= |language=zh-cn}}</ref>2018年，邓聿文在《纽约时报》发表《给习近平的七个忠告》<ref>{{cite news |author1=邓聿文 |title=给习近平的七个忠告 |url=https://cn.nytimes.com/opinion/20181204/bits-of-advice-chairman-xi/?utm_source=top10-in-article&utm_medium=email&utm_campaign=web |accessdate=2018-12-04 |work=纽约时报 |date=2018年12月4日}}</ref>。
