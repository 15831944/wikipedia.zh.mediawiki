{{日本作品|糖調！-sugarfull tempering-|しゅがてん！-sugarfull tempering-}}是Recette在2017年2月24日<ref>{{Cite web|url=https://www.game-style.jp/special/201701/24/00sug_re.php|title=Game-Style「糖調！-sugarfull tempering-」特集|publisher=Game-Style|date=2017-01-24|accessdate=2018-05-14|language=ja}}</ref>發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]。

== 故事簡介 ==
笛谷町每年都會發生一次[[鑽石塵|鑽石塵]]現象，被當地居民稱呼為「妖精之夜」，並相傳在妖精之夜許願的話願望就會實現。在笛谷町經營數十年的西式糕點店「佛克羅爾」由於老闆受傷住院而陷入經營困難的情況，老闆的女兒古倉芽瑠和在店裡打工寄宿的女孩聖代橋冰織於妖精之夜這天出外許願，希望有人可以拯救佛克羅爾，當芽瑠回到店裡時，發現店裡的屋頂上掉下了一名男子。這名男子在醒來之後，為了感謝芽瑠和冰織救了自己，決定展現做蛋糕的技術以拯救佛克羅爾<ref>{{Cite web|url=https://www.game-style.jp/soft/201701/17/01sug_re.php|title=新ブランド「Recette（ルセット）」のデビュー作は、妖精が棲む雪の街で紡がれる癒し系ハートフルラブストーリー！ 『しゅがてん！ -sugarfull tempering-』2月24日発売！ |publisher=Game-Style|date=2017-01-17|accessdate=2018-05-14|language=ja}}</ref>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|九郎|クロウ}}
: 妖精之夜這天在笛谷町出現的男子，沒有記憶，為了感謝芽瑠和冰織救了自己而在佛克羅爾工作。製作蛋糕的技巧非常高明。
; {{nihongo|聖代橋冰織|聖代橋 氷織（みよはし こおり）|聲：[[杏子御津|杏子御津]]<ref name="CHARACTER">{{cite web|url=http://recette.clearrave.co.jp/chara.html|title=CHARACTER|publisher=Recette|accessdate=2018年6月5日|language=ja}}</ref>}}
: 在佛克羅爾打工寄宿的少女。個性認真且冷靜，擅長泡紅茶與咖啡，吃了甜食後會醉。
; {{nihongo|古倉芽瑠|古倉 める（ふるくら める）|聲：くすはらゆい<ref name="CHARACTER"/>}}
: 佛克羅爾老闆的孫女，好奇心旺盛且活潑。非常喜歡甜食。
; {{nihongo|巧克力·涅朱|ショコラ・ネージュ|聲：藤咲ウサ<ref name="CHARACTER"/>}}
: 大企業的大小姐，原本為了收購佛克羅爾而來到笛谷町，在吃過九郎做的蛋糕後改變了想法。

; {{nihongo|村崎小町|村崎 小町（むらさき こまち）|聲：[[鈴谷真矢|鈴谷真矢]]<ref name="CHARACTER"/>}}
