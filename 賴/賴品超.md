'''賴品超'''教授<!--（{{bd|19xx年||||catIdx=L}}）-->（{{lang-en|Prof. LAI Pan Chiu}}），是宗教研究學者，現任[[香港中文大學|香港中文大學]]文學院副院長、文化及宗教研究系教授（宗教研究）、漢語基督教文化研究所學術委員會主席。

==簡歷==
賴品超，1985年畢業於[[香港中文大學|香港中文大學]]宗教系獲文學士學位，1987年於[[崇基神學組|崇基神學組]]獲得道學碩學位，之後遠赴[[英國|英國]]深造，研究系統神學，1991年獲[[倫敦國王學院|倫敦大學國王學院]]哲學博士學位。他曾任英國[[密德薩斯大學|密德薩斯大學]]哲宗系講師。<ref>{{ cite web |url=https://www.theology.cuhk.edu.hk/tc/academics/teaching-staff/lai-pan-chiu  |title=教授團隊：賴品超教授 |date= |publisher=香港中文大學崇基學院神學院 |author= }}</ref>

1996起，賴品超任教於[[香港中文大學|香港中文大學]]宗教系、文化及宗教研究系（2004年，宗教系與現代語言及文化研究學系的文化研究組別合併，成為「文化及宗教研究系」<ref>{{ cite web |url=http://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=22702&Pid=2&Version=862&Cid=179&Charset=big5_hkscs |title=中大宗教系與文化研究系合併 |date=2004-03-07 |publisher=時代論壇 |author= }}</ref>），歷任系主任（2002-2008），天主教研究中心主任（2005-2008）、人間佛教研究中心主任（2005-2008），2004年起擔任文學院副院長。主要研究包括：中國基督教神學、現代基督教思想、宗教對話、基督教與中國文化、環境倫理、宗教與自然科學。<ref>{{ cite web |url=http://www2.crs.cuhk.edu.hk/tc/faculty-staff/faculty-members/lai-pan-chiu |title=賴品超教授 |date= |publisher=香港中文大學文化及宗教研究系 |author= }}</ref><ref>{{ cite web |url=http://www.iscs.org.hk/Common/Reader/News/ShowNews.jsp?Nid=98&Pid=4&Version=0&Cid=28&Charset=big5_hkscs |title=賴品超教授 |date= |publisher=漢語基督教文化研究所 |author= }}</ref>
