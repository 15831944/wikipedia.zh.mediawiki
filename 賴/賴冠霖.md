'''賴冠霖'''（{{kor|'''라이관린'''|rm='''Lai Kuanlin'''}}，{{lang-en|'''Lai Kuan-lin'''}}<ref name=":0">{{Cite web|url=http://wannaonego.mwave.me/wannaone|title=WANNA-ONE|accessdate=2017-11-28|last=E&M|first=CJ|work=wannaonego.mwave.me|language=ko}}</ref>，{{lang-ja|'''ライ・グァンリン'''}}；{{bd|2001年|9月23日||}}），[[台灣|台灣]][[新北市|新北市]]人，所屬經紀公司為[[Cube娛樂|Cube娛樂]]。2017年參加韓國[[Mnet媒體|Mnet]]選秀節目《[[PRODUCE_101_第二季|Produce 101 S2]]》，最終以第7名的身份成為限定男子團體[[Wanna_One|Wanna One]]的成員出道。

== 生涯 ==
''' 早年生活 '''

賴冠霖於2001年9月23日出生於中華民國台灣新北市，家庭成員除了父母之外還有一位姊姊。<ref>{{cite news |language = zh-tw |url = http://ent.ltn.com.tw/news/breakingnews/2102997 |title = 台灣第二個周子瑜是他！ 賴冠霖韓國順利出道 |publisher = 自由時報 |date = 2017-06-17 |accessdate = 2017-07-12}}</ref>賴冠霖小時候曾在[[美國|美國]][[加州|加州]]居住過。<ref>{{Cite web |url = http://www.wishnote.tw/#!/menu=landing&content_id=73916 |title = 賴冠霖簽名會上「超不謙虛」的純真反應讓粉絲噴笑 原來他「英文好的原因」是？ |author =  米丘嫂 |date = 2017-08-14 |publisher = Wishnote |language = zh-tw |accessdate = 2017-08-27 }}</ref>他畢業於新北市立麗園國民小學，休學前往韓國當練習生之前，曾就讀於新北市立林口國民中學。<ref>{{cite news |language = zh-tw |author =  |coauthors =  |url = http://www.setn.com/News.aspx?NewsID=250078 |title = 同學變藝人！國中同窗心碎告白賴冠霖：來不及說出口的話… |publisher = 三立新聞網 |pages =  |date = 2017-05-06 |accessdate = 2017-05-21}}</ref>

==演藝生涯==
===出道前===

2016年4月，賴冠霖通過Cube Star World Audition in Taiwan後，到韓國當練習生。<ref>[http://m.entertain.naver.com/read?oid=382&aid=0000597662 워너원 프리퀄⑧라이관린, ‘천상 아이돌’의 표본 ]2017-10-2</ref> 

{{main|PRODUCE 101 第二季}}

2017年3月，賴冠霖以[[Cube娛樂|Cube]]練習生的身份參演大型男團生存實境節目《[[PRODUCE_101_(第二季)|PRODUCE 101第二季]]》。節目中，101位來自52間不同經紀公司的練習生與個人練習生將會接受唱歌、rap、舞蹈等嚴格訓練並競爭，每集由國民製作人經過網路投票選出練習生排名，而最後一集排名前11的練習生，將以[[Imagine_Asia|YMC娛樂]]旗下藝人的身份進行為期一年6個月的活動。在6月16日，節目進入尾聲，賴冠霖在第十一集共獲得905,875票數，以排名第7的成績，確認成為限定組合Wanna One的一員。

===2017年===
'''出道後：Wanna One'''
{{main|Wanna One}}
8月7日，賴冠霖以[[Wanna_One|Wanna One]]成員的身份正式出道，並發行首張迷你專輯《[[1X1=1_(TO_BE_ONE)|1X1=1 (TO BE ONE)]]》，主打曲為〈{{lang|en|Energetic}}〉及〈{{lang|en|Burn It Up}}〉。

===2018年===
1月26日，賴冠霖受品牌邀請，赴[[香港|香港]]參加時尚快閃店的活動，首度個人活動受到粉絲高度關注，同時他也是第一個受邀時尚品牌活動站台的[[Wanna_One|Wanna One]]成員。<ref>{{cite news |language = zh-tw |url = https://tw.appledaily.com/new/realtime/20180129/1288147/|title = 賴冠霖拚靚張藝興　VLTN開店圈粉星級小 |publisher = 蘋果日報 |date = 2018-02-15 |accessdate = 2018-02-15}}</ref>

===2019年===
'''解散後：Wanna One，進軍中國當演員'''

1月13日，出席電視劇《初見•未來》的新聞發布會，正式宣布接演《初戀那件小事》男主角。

1月15日，以[[Wanna_One|Wanna One]]成員的身分出席了第28屆首爾歌謠大賞（28th Seoul Music Awards）。

1月24日至27日也在高尺巨蛋舉行告別演唱會「2019 Wanna One Concert [Therefore]」。

'''小分隊：WOOSEOK X KUANLIN'''

3月11日，與[[Pentagon_(男子組合)|PENTAGON]]成員[[鄭禹奭|禹奭]]，組成小分隊[[禹奭_X_冠霖|禹奭 X 冠霖]]
，公開迷你專輯《9801》，主打歌《별짓》。

== 爭議事件 ==
===來自「中國臺灣」===
賴冠霖第一次參加《[[PRODUCE_101_第二季|Produce 101 S2]]》錄影時（2017年2月26日錄影，4月7日播出）自我介紹來自臺灣，其後節目官方於網路釋出的自我介紹影片裡表示自己來自[[臺北|臺北]]<ref>{{cite web |language = en |author = Mnet Official|url = https://www.youtube.com/watch?v=PXRK3nD2Za8|title=PRODUCE 101 season2 큐브ㅣ라이관린 타이페이 힙합소년 @자기소개_1분 PR 161212 EP.0 |publisher = [[YouTube|YouTube]] |date = 2017-03-10 |accessdate = 2017-05-21}}</ref>。2017年3月9日，《Produce 101 S2》練習生全員出演Mnet《M COUNTDOWN》，結束後在上岩洞CJ E&M 中心廣場與粉絲見面並分送事先準備好之小禮物及自介紙卡，賴冠霖的自介紙卡後經公開於微博，屬名[[中國臺灣|中國臺灣]]賴冠霖，此舉動引起部分臺灣網友的批判<ref>{{cite news |language = zh-tw |author =  |coauthors =  |url = http://ent.ltn.com.tw/news/breakingnews/2000355 |title = 台17歲嫩男赴韓選秀自貶國格 竟稱來自「中國台灣」 |publisher = [[自由時報|自由時報]] |pages =  |date = 2017-03-10 |accessdate = 2017-05-21}}</ref>，不過介紹文為他親筆手寫之事真實性有待商榷。<ref>{{cite news |language = zh-tw |author = 田暐瑋 |url = http://star.ettoday.net/news/882367 |title = 17歲台男孩赴韓選秀自稱「中國人」　網戳破陰謀論！ |publisher = ETtoday |date = 2017-03-11 |accessdate = 2017-05-21}}</ref><ref>{{cite news |language = zh-tw |author =  |url = http://ent.ltn.com.tw/news/breakingnews/2000688 |title = （影音）來自中國台灣？賴冠霖官方自介影片這樣說... |publisher = [[自由時報|自由時報]] |date = 2017-03-11 |accessdate = 2017-08-29}}</ref>

2017年8月，賴冠霖於韓國廣播節目《Kiss The Radio》裡說出中國臺灣四個字，此舉動再次造成部分臺灣網友的批評。<ref>{{cite news |language = zh-tw |author = 周詠晨、李芷萱 |url = http://www.setn.com/News.aspx?NewsID=287989 |title = 賴冠霖親口說出「中國台灣」　鄉民崩潰：我真的不行 |publisher = 三立新聞網 |date = 2017-08-28 |accessdate = 2017-08-29}}</ref>

2018年2月14日，賴冠霖於第七屆Gaon榜音樂頒獎典禮，代表團體用中文發表得獎感言。他首先表示：「謝謝主辦方給我們這個獎，以這個獎為動力，2018也會展現更好的Wanna One給你們！」，但在之後說道「也謝謝網易雲音樂！大家請多多用網易雲音樂」，得獎感言突如其來的業配贊助廠商而引發了爭議。<ref>{{cite news |language = zh-tw |url = https://tw.entertainment.appledaily.com/realtime/20180215/1299141/ |title = 子瑜得獎中文謝粉絲　賴冠霖這發言又惹議 |publisher = 蘋果日報 |date = 2018-02-15 |accessdate = 2018-02-15}}</ref>
