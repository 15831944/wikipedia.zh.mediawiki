== 簡歷 ==
=== 社會活動 ===
[[File:賴品妤與華光社區強拆事件.jpg|thumb]]後於[[華光社區|華光社區]]拍下的攝影作品，希望能夠藉此吸引更多人關注該議題。]]
賴品妤出生於[[雲林縣|雲林縣]]，其父親是曾經擔任過[[民主進步黨|民主進步黨]]籍[[立法委員|立法委員]]和[[行政院勞工委員會|行政院勞工委員會]]副主任委員的[[賴勁麟|賴勁麟]]，而母親是媒體人[[吳如萍|吳如萍]]<ref>{{Cite web |url = http://www.nexttv.com.tw/news/realtime/politics/10985827/%E4%B8%8D%E8%AE%93%E9%AC%9A%E7%9C%89!%20%E8%B3%B4%E5%93%81%E5%A6%A4%E4%BF%8F%E9%BA%97%E5%89%BD%E6%82%8D%E3%80%80%E5%89%8D%E7%AB%8B%E5%A7%94%E6%84%9B%E5%A5%B3 |title = 不讓鬚眉! 賴品妤俏麗剽悍　前立委愛女 |author =  |date = 2014年3月20日 |publisher = [[壹電視|壹電視]] |language = zh-hant |accessdate = 2014年3月26日 |quote =  }}</ref>。賴勁麟在進入大學後曾積極參與學生運動，並且是1981年主張普通選舉以及言論自由的大論五人小組一員，後來加入民主進步黨並且投入勞工運動<ref>{{Cite web |url = http://big5.china.com.cn/chinese/zhuanti/mjdrw/954186.htm |title = 賴勁麟 |author =  |date = 2005年8月29日 |publisher = [[中國網|中國網]] |language = zh-hant |accessdate = 2014年3月26日 |quote =  }}</ref>。這樣的生活環境使得她長期關心著許多類似的社會運動，而她也表示「我爸爸年輕的時候有參與過工運，我的家人不會特別排斥我參與社會運動」<ref>{{Cite web |url = http://news.ebc.net.tw/apps/newsList.aspx?id=1395324860 |title = 上火線對峙警 法律系賴品妤「嬌」點 |author =  |date = 2014年3月20日 |publisher = [[東森新聞台|東森新聞台]] |language = zh-hant |accessdate = 2014年3月26日 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20140328023427/http://news.ebc.net.tw/apps/newsList.aspx?id=1395324860 |archivedate = 2014年3月28日 }}</ref>。而在2012年因為因為[[洪仲丘事件|洪仲丘事件]]爆發引起的[[白衫軍運動|白衫軍運動]]以及[[反媒體壟斷運動|反媒體壟斷運動]]所興起的[[反媒體壟斷大遊行|反媒體壟斷大遊行]]出現後，賴品妤也開始親自參與許多社會運動<ref>{{Cite web |url = http://news.sina.com.hk/news/20140324/-9-3221441/1.html |title = 台灣抗議學生群體多為民進黨外圍分子 |author =  |date = 2014年3月24日 |publisher = [[新浪|新浪]] |language = zh-hant |accessdate = 2014年3月28日 |quote =  }}</ref>。對此她則提到：「你進入了一個圈子，你難免開始關注不同方面的議題，所以久了，的確對自己過去的價值觀會有很大的影響，譬如說我覺得更能去看到過去沒有看到，可能比較弱勢的族群。<ref>{{Cite web |url = http://www.ettoday.net/news/20140320/336975.htm |title = 反黑箱服貿占立院　長髮辣妹賴品妤掛彩沒喊痛 |author =  |date = 2014年3月20日 |publisher = [[ETtoday_東森新聞雲|ETtoday 東森新聞雲]] |language = zh-hant |accessdate = 2014年3月26日 |quote =  }}</ref>」

在2013年8月臺灣學生運動團體[[黑色島國青年陣線|黑色島國青年陣線]]宣告成立後，賴品妤便開始積極參與相關的活動。同年11月26日，中國[[海峽兩岸關係協會|海峽兩岸關係協會]]會長[[陳德銘|陳德銘]]前往[[海峽交流基金會|海峽交流基金會]]並且拜訪董事長[[林中森|林中森]]，而不滿陳德銘對《兩岸服務貿易協議》施加壓力的黑色島國青年陣線20多名學生便計畫在海峽交流基金會附近高舉標語牌抗議<ref>{{Cite web |url = http://www.chbcnet.com/pl/content/2014-03/28/content_806451.htm |title = 杨平：“反服贸”学运是民进党操弄的结果 |author = 艾然 |date = 2014年3月28日 |publisher = 華廣網 |language = zh-hans |accessdate = 2014年3月28日 |quote =  }}</ref>。但是隨後便立刻在海峽交流基金會前方和側邊小巷與警方爆發衝突，其中賴品妤、黃燕茹和賴郁棻3名女學生以管制區名義遭到10名女警手勾手圍在牆壁上，後來賴品妤更被警察一路拖行至海峽交流基金會門口才放走<ref>{{Cite web |url = http://www.greenpeace.com.tw/index.php?option=com_content&task=view&id=2652 |title = 對陳德銘高喊反服貿 學生遭警察「鎖喉」 |author = 王立柔 |date = 2013年11月26日 |publisher = [[綠色和平電台|綠色和平電台]] |language = zh-hant |accessdate = 2014年3月28日 |quote =  }}</ref>。當天她接受採訪時也表示受到中國勞工相對便宜以及環境法令寬鬆的影響，將導致臺灣產業為了與之中國抗衡而將使得臺灣勞工薪資水準停滯不前<ref>{{Cite web |url = http://www.idn.com.tw/news/news_content.php?artid=20131126abcd026 |title = 看見台灣 請陳德銘停止干涉台灣內政 |author = 張廷玉和陳金寶 |date = 2013年11月26日 |publisher = 《[[自立晚報|自立晚報]]》 |language = zh-hant |accessdate = 2014年3月28日 |quote =  }}</ref>。

由於賴品妤本身是個動漫迷並且自己也喜歡自拍和[[Cosplay|Cosplay]]裝扮，這使得她在[[Facebook|Facebook]]上有許多粉絲關注著<ref>{{Cite web |url = http://www.setnews.net/News.aspx?PageGroupID=6&NewsID=17375&PageType=3 |title = 服貿／抗議「嬌」點　甜美正妹是社運狂　剽悍抗警察沒在怕 |author = 劉家綺和徐士庭 |date = 2014年3月20日 |publisher = [[三立新聞台|三立新聞台]] |language = zh-hant |accessdate = 2014年3月28日 |quote =  }}</ref>。而在2013年12月[[臺北市政府都市計畫委員會|臺北市政府都市計畫委員會]]所引發的[[華光社區|華光社區]]拆除事件中，賴品妤便決定將該拆除議題與Cosplay結合而讓自己扮演成《[[新世紀福音戰士|新世紀福音戰士]]》中的角色[[綾波零|綾波零]]，並且希望透過在華光社區建築遺骸中拍照留下的影像作品引起更多人的重視<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/politics/20140319/363058/ |title = 【獨家】22歲女大生　為民主做悍妞 |author =  |date = 2014年3月19日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2014年3月28日 |quote =  }}</ref>。對此她後來表示：「我總是覺得自己在社會運動這塊，能做的事情好像不多，後來華光社區要全面強拆，覺得可以用自己的方式留下紀錄，如果能引起更多動漫迷的注意也不錯。」同時她認為一般自拍照比較注重他人對於照片的評價，但是Cosplay則是著重自己跟喜愛的角色之間對話，並且也能夠藉由上傳照片來與同好討論動漫作品<ref name = "丁宛臻、李孟儒、莊惠琪和蘇俞璇"/>。而在2014年1月發生的[[中華民國教育部|中華民國教育部]]其[[臺灣高中歷史課綱微調案|高中歷史課綱調整]]引起[[去臺灣化|去臺灣化]]的爭議中，她則指出在中華民國教育部提出的課綱新草案中當代臺灣歷史部分僅強調《[[開羅宣言|開羅宣言]]》以及臺灣在於國際法地位確定的過程，但是卻淡化了原先《[[舊金山和約|舊金山和約]]》與《[[臺灣關係法|臺灣關係法]]》的內容<ref>{{Cite web |url = http://www.lihpao.com/?action-viewnews-itemid-137309 |title = 高中歷史爭議課綱 教部定案 |author =  |date = 2014年1月27日 |publisher = 《[[台灣立報|台灣立報]]》 |language = zh-hant |accessdate = 2014年3月26日 |quote =  }}</ref>。
