'''共匪'''是對[[共產黨|共產黨]]的一種敵對稱呼，於中文環境中多用於指[[中國共產黨|中國共產黨]]、中國共產黨黨員、或支持中國共產黨的[[親中共|親中共]]人士。

== 概述 ==
此名詞主要用在[[中國國民黨|中國國民黨]]右派領導的[[中華民國政府|中華民國政府]]與中國共產黨（包含[[國共內戰|國共內戰]]后建立的[[中华人民共和国|中华人民共和国]]）嚴重對立的時期。也有以[[共狗|共狗]]、[[土共|土共]]、[[中国大陆网络用语列表#政治类_2|黄俄]]、共產黨[[匪諜|匪諜]]及[[國家|國家]][[黑社會主義|黑社會主義]]<ref>[http://hk.apple.nextmedia.com/news/art/20120310/16144187 蘋論：選唐是災難，黑社會主義是更大災難]，李怡，蘋果日報，2012-3-10</ref><ref>[http://www.epochtimes.com/b5/6/4/12/n1284989.htm 郭飛熊： 「國家黑社會主義」的肆虐及其前景]，2006年4月12日</ref>等稱呼中國共產黨、黨員及其支持者。

不承認中華人民共和國之人也會使用「中共」稱呼「中华人民共和國」<ref>陳茂雄. [http://www.appledaily.com.tw/appledaily/article/headline/20081225/31260466/%E9%A6%AC%E8%8B%B1%E4%B9%9D%E4%B8%8D%E8%A9%B2%E9%81%8E%E5%BA%A6%E4%BE%9D%E8%B3%B4%E4%B8%AD%E5%85%B1%28%E9%99%B3%E8%8C%82%E9%9B%84%29 馬英九不該過度依賴中共]. 蘋果日報. 2008年12月25日.</ref>，或作为「中华人民共和国」的简称，但該用法較常見於中華民國國防部說辭，此時的中共一詞也是貶稱。<ref>蔣中正，[http://www.chungcheng.org.tw/thought/class07/0035/0004.htm 中華民國五十年國慶紀念告全國軍民同胞書] {{webarchive|url=https://web.archive.org/web/20040103043340/http://chungcheng.org.tw/thought/class07/0035/0004.htm |date=2004-01-03 }}</ref>亦有將中國共產黨統治的[[中國大陸|中國大陸]]地區稱作「匪區」或者「[[北平|北平]][[傀儡政權|匪偽政權]]」<ref>秦孝儀. [http://www.chungcheng.org.tw/thought/class10/0035/0004.htm 美國真正的敵人——共匪] {{webarchive|url=https://web.archive.org/web/20050218203200/http://chungcheng.org.tw/thought/class10/0035/0004.htm |date=2005-02-18 }}. 卷三十九＼談話＼中華民國五十四年. 《先總統　蔣公思想言論總集序》第303頁.</ref>、「中共匪區」<ref>蔣中正，《蘇俄在中國》第一編　中俄和平共存的開始與發展及其結果：[http://www.chungcheng.org.tw/thought/class05/0009/0001/0003.htm 第三章　中俄和平共存的發展] {{webarchive|url=https://web.archive.org/web/20031223061200/http://chungcheng.org.tw/thought/class05/0009/0001/0003.htm |date=2003-12-23 }}</ref><ref>蔣中正，《蘇俄在中國》補編　俄共在中國三十年來所使用的各種政治鬥爭的戰術，及其運用辯證法的方式之綜合研究：[http://www.chungcheng.org.tw/thought/class05/0009/0004/0006.htm 第六章　貿易戰與基地戰] {{webarchive|url=https://web.archive.org/web/20030824173858/http://chungcheng.org.tw/thought/class05/0009/0004/0006.htm |date=2003-08-24 }}</ref>、「大陸匪區」、「大陸[[淪陷地區_(國共內戰)|淪陷區]]」、「中共匪國」、「伪中华人民共和国」、「中共」（以中国共产党简称称之）。

共匪和[[赤匪|赤匪]]亦可以指向中國共產黨附屬組織及其他國家或者地區[[共產主義|共產主義]]組織和政黨，包括「越南共匪」（[[越南共產黨|越南共產黨]]）<ref>[http://www.chungcheng.org.tw/thought/class10/0035/0004.htm 越南共匪搶地引發抗共暴亂 7000苗人揭竿而起要求建立獨立國] {{webarchive|url=https://web.archive.org/web/20050218203200/http://chungcheng.org.tw/thought/class10/0035/0004.htm |date=2005-02-18 }} 中華評述5月7日</ref>，「朝鮮共匪」（[[朝鮮勞動黨|朝鮮勞動黨]]），「古巴共匪」（[[古巴共產黨|古巴共產黨]]）。<ref>「{{lang|ko|[[:ko:공비|공비]]（共匪）}}」</ref>

近代中國內戰頻繁，盜匪猖獗，政府通常將各種[[造反|造反]]的武裝團體與盜匪等同。

[[辛亥革命|辛亥革命]]後，新生的中華民國政治動盪，軍閥混戰，各派政治勢力常以「匪軍」之名加諸於對方。[[國民政府|國民政府]]在1927年宣佈[[剿匪|剿匪]]後，[[賀龍|賀龍]]、[[朱德|朱德]]、[[毛澤東|毛澤東]]在[[井岡山|井岡山]]等地建立獨立的武裝與國民政府對抗，國民政府遂將中共紅軍及本身稱為「赤匪」、「紅匪」、「共匪」、「毛匪」、「毛赤」、「黃俄」、「土共」<ref>這裡的「土共」常指鄉村較沒文化的共匪黨徒</ref>等。

== 用法 ==
=== 中華民國 ===
「赤匪」一詞用來指稱抗戰爆發前的[[中国共产党|中国共产党]]及其[[中國工农紅軍|红军]]，而「共匪」除指稱共產黨人外，還用作指1949年前後的中華人民共和國政府：

{{Quotation|中國目前最大之禍患，厥為赤匪。國民政府與全國人民當前最急要之工作，亦莫過於撲滅赤匪。赤匪之禍患，其已發露於江西、湖南等處者，固在政府與當地人民協力撲滅之中，而其潛伏於他處，冀圖使各省均為江西與湖南之續者，亦已漸露其端，而實不容吾人之忽視。以各地受匪殘害之種種事實言之，赤匪之存在與蔓延，不惟於中國民族生存與發展不能相容，且於全國人民各個人之生命與生計不能並立。|《[[s:國民政府向國民會議提出剿滅赤匪報告案|國民政府向國民會議提出剿滅赤匪報告案]]》，民國二十年五月十二日}}

{{Quotation|我全國同胞當此赤匪、軍閥、叛徒與[[帝國主義|帝國主義]]聯合進攻、生死存亡間不容髮之秋，自應以臥薪嘗膽之精神作安內攘外之奮鬥；以忍辱負重之毅力雪黨、國百年之奇恥……惟攘外必先安內，去腐方能防蠹。此次如無粵中叛變，則朝鮮慘案必無由而生，法權收回問題亦早已解決，不平等條約取消自無疑義……故不先滅赤匪，恢復民族之元氣，則不能禦侮；不先削平粵逆，完成國家之統一，乃不能攘外。|蔣中正，《[[s:告全國同胞一致安內攘外書|告全國同胞一致安內攘外書]]》，民國二十年七月二十三日}}

與此類似的詞彙還有「[[新華社|匪新華社]]」、「[[殺朱拔毛|殺朱拔毛]]」等。中共統治的中國大陸則被稱作「中共匪區」、「大陸匪區」、或簡稱「匪區」。<ref>《國民政府頒布的全國總動員方案》《[[國民政府公報|國民政府公報]]》第2869號（1947年7月4日）</ref><ref>《動員戡亂完成憲政實施綱要》（1947年7月）</ref><ref>《[[動員戡亂時期臨時條款|動員戡亂時期臨時條款]]》（1948年5月10日）</ref>「共匪」的[[間諜|間諜]]曾被稱作「'''[[匪諜|匪諜]]'''」。<ref>《[[戡亂時期檢肅匪諜條例|戡亂時期檢肅匪諜條例]]》</ref>中共國政府則被稱作「匪偽政府」或簡稱「匪偽」，例如《戒嚴時期台灣地區查禁匪偽郵票實施辦法》。

1983年，[[蔣經國|蔣經國]]在總統文告首次以「中共」而非「共匪」稱呼對岸政權。此後，在描述[[中華人民共和國政府|北京政府]]機構、職稱及文件時，仍要將相關名稱用引號包住以作不承認之意，如：<ref>《[[s:中共黨政軍機關企業學術機構團體旗歌及人員職銜統一稱謂實施要點|中共黨政軍機關企業學術機構團體旗歌及人員職銜統一稱謂實施要點]]》，民國八十一年（1992年）十一月十九日[[行政院大陸委員會|行政院大陸委員會]](８１)陸法字第五二七二號函</ref>

臺灣廢止《[[動員戡亂時期臨時條款|動員戡亂時期臨時條款]]》後，共匪一詞就不在正式場合使用，「匪區」一般改稱「中共佔領區」。隨著[[臺灣獨立運動|臺灣獨立運動]]的發展，臺灣有越來越多民眾直接使用「中國」一詞來指稱中國大陸，但在中華民國官方正式公文中通常仍避稱「中華人民共和國」，仍以「大陸」或「中國大陸」稱之（[[陳水扁政府|陳水扁政府]]時期例外）。<ref>[[中華民國司法院大法官|中華民國司法院大法官]]司法院釋字第328、476、572號解釋相關附件有中共國三字，沒有加引號。《[[s:司法院釋字第618號解釋|司法院釋字第618號解釋]]》相關附件有「中華人民共和國」七字加引號。</ref>1994年，中國大陸[[浙江省|浙江省]]發生[[千島湖事件|千島湖事件]]後，[[中華民國總統|中華民國總統]][[李登輝|李登輝]]曾經公開批評中共是「土匪政權」，「像土匪一樣害死我們那麼多同胞」。<ref>[http://taipedia.cca.gov.tw/Dict/DispText.aspx?id=111&CurrPage=1&menu=0&n=3&key=%20&type=1&flags=2 台灣歷史詞典：千島湖事件]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

[[臺灣人|臺灣人]]民間常用「阿共仔」，作為對[[中華人民共和國政府|中華人民共和國政府]]或者[[中國共產黨|中國共產黨]]的習慣稱呼用語，一如稱[[日本人|日本人]]為「阿本仔」，早年稱[[外省人|唐山人]]（外省人）為「阿山仔」。

===韓國===
{{Seealso|:ko:공비|l1=공비}}
韓國與中國一樣，長期將與政府敵對的勢力稱為「匪」，比如1894年發動[[東學黨起義|東學黨起義]]的[[天道教|東學黨]]人就被稱為「東匪」。而[[大韓民國|大韓民國]]的官方、媒體常用「{{lang|ko|[[:ko:공비|공비]]（共匪）}}」以及「{{lang|ko|공산 비적（共産匪賊）}}」形容[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]人物、組織、反政府行為或間諜行為，或是專指[[韓戰|韓戰]]在[[智異山|智異山]]、[[太白山|太白山]]一帶活動的朝鮮人民游擊隊、[[朝鮮日治時期|日本殖民時代]]在[[偽滿洲國|偽滿洲國]]一帶活動的抗日遊撃隊派。

===香港===
香港一些傳媒和[[泛民主派|泛民主派]]人士有時以「土共」作為对香港親共團體或人士的贬称，也指香港的[[建制派|建制派]]或[[建制派_(澳門)|澳門建制派]]<ref>[http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20100127&sec_id=4104&art_id=13668301 公投五俠選票起義第一擊冒着土共炮火前進] 蘋果日報-2010年01月27日</ref><ref> [http://hk.epochtimes.com/6/3/10/19575.htm 《香港焦點》凌鋒：中央支持特首 土共緊急轉呔]</ref>。

自1949年至1971年之間，除了[[左派|左派]]報紙外，香港的[[反共|反共]]報紙（[[香港親國民黨團體|香港親國民黨團體]]或[[港英政府|港英政府]]）均以「中共」稱呼大陸的政權，以區分統治台灣的「中華民國」（又稱「自由中國」）政權。

[[香港親共團體|香港親共團體]]另有一些蔑稱是「港共」、「左仔」，最早見於[[六七暴動|六七暴動]]期間，親國報紙（如《[[工商日報|工商日報]]》）以此詞彙抨擊左派暴徒的[[恐怖主義|恐怖主義]]行為<ref>[http://1967riot.wordpress.com/tag/港共/ 1967年間使用「港共」詞彙的報章社論]</ref>。及至2010年代隨着新一屆香港特區政府班子上任，有高官被指為共產黨員，「[[港共|港共]]」這一稱呼再被[[本土派_(香港)|本土派]]人士或民主派傳媒（如[[黃毓民|黃毓民]]、《[[蘋果日報|蘋果日報]]》）所使用<ref>[http://hk.apple.nextmedia.com/news/art/20030325/3191676 李怡專欄：港共更左]</ref>。

== 类似用法 ==
=== 共軍 ===
{{主條目|共軍}}
'''共軍'''，意為[[共产党|共产党]]的[[軍隊|軍隊]]，在現代漢語特指[[中国共产党|中国共产党]]所領導的軍隊。在普遍概念中，與「共軍」相對的是[[中國國民黨|中國國民黨]]創建的[[國民革命軍|國民革命軍]]（[[中華民國|中華民國]][[中華民國國軍|國軍]]之前身）。「共軍」包括中國現代歷史上的[[中国工农红军|中国工农红军]]、[[八路军|八路军]]、[[新四軍|新四軍]]甚至后来的[[中国人民解放军|中国人民解放军]]、[[中国人民志愿军|中国人民志愿军]]等共產黨軍隊，当今用语中特指[[中国人民解放军|中国人民解放军]]。<ref>[http://www.nsb.gov.tw/documents%5C%E4%B8%80%E8%88%AC%E6%80%A7%E5%A0%B1%E5%91%8A%5C2015%5C06%E7%BE%8E%E5%9C%8B%E5%9C%8B%E9%98%B2%E9%83%A8%E3%80%8C2015%E5%B9%B4%E4%B8%AD%E5%85%B1%E8%BB%8D%E5%8A%9B%E3%80%8D%E5%A0%B1%E5%91%8A.pdf 中華民國國家安全局-美國防部2015 年中共軍事與安全發展報告]</ref>

中國共產黨與其領導的[[中華人民共和國政府|中華人民共和國政府]]較少使用「共軍」這個詞語（一般直接称“我军”），而[[中華民國政府|中華民國政府]]與[[台灣|台灣]]民間普遍使用「共軍」來代指[[中国人民解放军|中国人民解放军]]，性質較「[[共匪|共匪]]」溫和<ref>{{Cite web |url = http://www.mnd.gov.tw/publish.aspx?cnid=480&p=45175 |title=No. 616 解讀共軍兵力規模 |publisher = [[中華民國國防部|中華民國國防部]]}}</ref>。至今中華民國國防部仍在使用「共軍」一詞。「共軍」這個詞也可以指[[世界|世界]]各[[國|國]]共產黨所擁有的軍隊[[武力|武力]]，意思近於[[紅軍|紅軍]]。如[[苏联红军|蘇軍]]、[[越南軍事|越軍]]也會被稱爲「共軍」。此外，[[韓國|韓國]]官方还將[[朝鮮人民军|朝鮮人民军]]稱爲「武裝共軍」。{{Fact}}
