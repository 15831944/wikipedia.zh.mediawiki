'''共青城市'''，简称'''共青城'''，是[[江西省|江西省]]的[[省直管市|省直管市]]，地处南昌与九江中段、[[鄱阳湖|鄱阳湖]]之滨，面积308平方千米，人口19万（其中常住人口12万）<ref>[http://www.epi88.com/master/News_View.asp?NewsID=10287 江西共青城：政治名城 生态美城] {{webarchive|url=https://web.archive.org/web/20120311185039/http://www.epi88.com/master/News_View.asp?NewsID=10287 |date=2012-03-11 }}</ref>。共青城是[[中国内地|中国]]唯一以[[共青团|共青团]]命名的城市。前[[中共中央总书记|中共中央总书记]][[胡耀邦|胡耀邦]]逝世后按其遗孀李昭的愿望安葬于此。<ref>{{cite web|url=http://news.163.com/12/0404/07/7U7TPL7J00014AEE.html|title=记者清明探访胡耀邦墓地|work=网易|date=2012-04-04}}</ref>

== 历史 ==
1955年10月18日，[[中国共产主义青年团上海市委员会|上海共青团]]青年志愿者到此垦荒，创建共青社；40天后，时任[[共青团中央|共青团中央]]书记处书记的[[胡耀邦|胡耀邦]]到此视察。1984年12月，时任[[中共中央总书记|中共中央总书记]]的胡耀邦再次到此，效仿[[苏联|苏联]][[阿穆尔河畔共青城|阿穆尔河畔共青城]]并定名为“共青城”。

1992年[[江西省人民政府|江西省人民政府]]批准成立江西省共青城开放开发区，1994年成立江西省共青台商投资区。2002年，共青城被赋予政府职能，行政级别升为副厅级。2002年10月，共青城被确定为九江市人民政府派出机构——共青城开放开发区管委会（副地级局委）。<ref name="共青城总体规划">{{cite web |url=http://www.gbzc.com/?action-viewnews-itemid-723 |title=共青城总体规划 |accessdate=2009-11-03|author= |publisher=九江赣北之窗 |language=zh-hans }}</ref>

2010年，共青城开放开发区获[[中华人民共和国国务院|国务院]]批准升格为[[县级市|县级市]]，更名为'''共青城市'''。<ref name="ce.cn">[http://www.ce.cn/xwzx/gnsz/gdxw/201009/27/t20100927_21849742.shtml 胡耀邦命名的江西共青城获批县级市]</ref><ref>[http://jiangxi.jxnews.com.cn/system/2010/09/26/011486595.shtml 已获国务院批准！ 江西“共青城市”呼之欲出]</ref><ref>[http://xxgk.jiangxi.gov.cn/bmgkxx/smzt/gzdt/zwdt/201011/t20101115_263197.htm 江西省政务网]：《民政部批复同意江西省设立共青城市》：...将[[德安县|德安县]][[茶山街道|茶山街道]]、[[甘露镇|甘露镇]]、[[金湖乡|金湖乡]]，[[永修县|永修县]][[江益镇|江益镇]]和[[燕坊镇|燕坊镇]]坪塘村、燕坊村，[[星子县|星子县]][[苏家垱乡|苏家垱乡]]、[[泽泉乡|泽泉乡]]划归共青城市管辖...</ref>。2014年7月1日共青城市被江西省列为首先的6个试点直管县市之一，为省直管市。<ref name="直管县改革">{{cite web |url=http://jx.people.com.cn/n/2014/0529/c359142-21311620.html |title=共青城市被列入省直管县改革试点 |publisher=[[人民网|人民网]] |date=2014年5月29日 |accessdate=2018年6月8日}}</ref>

== 地理及交通 ==
共青城市东邻鄱阳湖，[[京九铁路|京九铁路]]、[[昌九城际铁路|昌九城际铁路]]、[[105国道|105国道]]及[[昌九高速公路|昌九高速公路]]穿城而过，距[[九江庐山机场|九江庐山机场]]，[[南昌昌北机场|南昌昌北机场]]各约30公里，交通便捷。<ref name="共青城总体规划"/>

== 行政区划 ==
辖1街道3镇2乡：[[茶山街道|茶山街道]]、[[甘露镇|甘露镇]]、[[金湖乡|金湖乡]]、[[江益镇|江益镇]]、[[苏家垱乡|苏家垱乡]]、[[泽泉乡|泽泉乡]]

[[苏家垱乡|苏家垱乡]]和[[泽泉乡|泽泉乡]]并未立即移交共青城市，继续由[[星子县|星子县]]管理<ref>[http://www.jxlaw.com.cn/html/2015-08/47006.html 行政区划调整5年未执行到位 九江两乡镇发展管理陷尴尬] {{webarchive|url=https://archive.is/20150813015838/http://www.jxlaw.com.cn/html/2015-08/47006.html |date=2015-08-13 }}</ref>；直至2016年6月，随着[[庐山市|庐山市]]成立，两乡的行政管理权才正式移交共青城市管理<ref>[http://www.jjxw.cn/2016/0616/253798.shtml 共青城市正式接管苏家垱乡和泽泉乡]</ref>。

== 经济 == 
共青城地理位置优越，由于2010年以前体制、区划不完善等，经济一直没有大的发展，知名企业有[[鸭鸭集团|鸭鸭集团]]（羽绒制品专业生产企业）。2009年10月，共青团中央授予共青城“[[全国青年创业基地|全国青年创业基地]]”的称号。<ref name="ce.cn"/>

2009年12月“[[鄱阳湖生态经济区|鄱阳湖生态经济区]]”正式出台，共青城是重点建设地。[[金淞电器|金淞电器]]、[[钜清电子|钜清电子]]、[[传宝光电|传宝光电]]等一批台资企业落户共青城<ref>[http://news.xinhuanet.com/fortune/2010-09/18/c_12583797.htm 鄱阳湖生态经济区成台企投资江西的新热点]</ref>。

2010年9月29日，太阳能光伏、服装羽绒、[[手机通讯|手机通讯]]以及[[江西师范大学|江西师范大学]]共青校区等19个重大项目集中开工，投资总额达[[人民币|人民币]]136.5亿元。<ref>[http://jiangxi.jxnews.com.cn/system/2010/10/01/011490106.shtml 共青城19个重大项目开工]</ref>
