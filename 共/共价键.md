'''共价键'''（{{lang-en|Covalent Bond}}），是[[化学键|化学键]]的一种。两个或多个[[非金屬|非金屬]][[原子|原子]]共同使用它们的外层[[电子|电子]]（砷化鎵為例外），在理想情况下达到电子饱和的状态，由此组成比较稳定和坚固的化学结构叫做共价键。与[[离子键|离子键]]不同的是进入共价键的原子向外不显示电荷，因为它们并没有获得或损失电子。共价键的强度比[[氢键|氢键]]要强，比[[离子键|离子键]]小。同一種[[元素|元素]]的原子或不同元素的原子都可以通過共​​價鍵結合，一般共價鍵結合的產物是[[分子|分子]]，在少數情況下也可以形成[[晶體|晶體]]。

[[吉爾伯特·路易斯|吉爾伯特·路易斯]]于1916年最先提出共价键。他提出在简单的原子轨道模型中进入共价键的原子互相提供单一的电子形成电子对，这些电子对围绕进入共价键的原子而属它们共有。

在[[量子力学|量子力学]]中，最早的共价键形成的解释是由电子的复合而构成完整的轨道来解释的。第一个量子力学的共价键模型是1927年提出的，当时人们还只能计算最简单的共价键：氢气分子的共价键。今天的计算表明，当原子相互之间的距离非常近时，它们的[[电子轨道|电子轨道]]会相互作用而形成整个分子共用的电子轨道。

== 吸引和排斥 ==
在共价键中，被共用的电子被所有进入共价键的原子吸引，由此使得这些原子结合在一起。虽然其原子核之间和电子之间會因电荷相同而互相排斥，但这些排斥作用被位于原子核间的电子减弱，而电子与原子核之间的相互作用更加强。

== 电子饱和 ==
按照简单的电子壳模型一个原子的外层电子在达到饱和状态下最稳定。对大多数原子来说，外层电子数为8时它们达到饱和，即“[[八隅體|八隅體]]”。这时它们的外层电子数与同周期的[[稀有氣體|惰性气体]]元素的外层电子数相同。

以[[氯化氢|氯化氢]]为例，在氯化氢分子中[[氢|氢]]原子并没有将它的外层电子交给[[氯|氯]]原子。而是两个原子共用一对外层电子而达到饱和状态。

== 共价键的分类 ==
共价键是[[电子雲|电子雲]]的重叠，所以共价键最本质的分类方式就是它们的重叠方式。现在已知有3种重叠方式，分别称作：
*[[σ键|σ键]]
*[[π键|π键]]
*[[δ键|δ键]]

在[[有机化合物|有机化合物]]中，通常把共价键以其共用的电子对数分为'''[[單鍵|單鍵]]'''、'''[[雙鍵|雙鍵]]'''以及'''[[叄鍵|叄鍵]]'''。单键是一根σ键；双键和叄鍵都含一根σ键，其余1根或2根是π键。

但[[无机物|无机化合物]]不用此法。原因是，无机化合物中经常出现的[[共轭|共轭]]体系（[[离域π键|离域π键]]）使得某两个原子之间共用的电子对数很难确定，因此无机物中常取'''平均键级'''，作为[[键能|键能]]的粗略标准。

== 极性共价键 ==
假如组成共价键的原子的[[电负性|电负性]]不同，那么它们共用的电子对可能被其中的一个原子核吸引，由此而来它们在分子中的分布也不相等，电子被吸引比较集中的地方显负性，电子比较少集中的地方显正性。这样整个分子就会显示出一定的[[极性|极性]]。一个分子的电极的分布除其原子的电负性外还与其分子的组成有关。

==网状共价键==
{{Link-en|网状共价键|Network covalent bonding}}，英文為Network covalent bonding，是[[原子晶体|原子晶体]]的主要化学键，是[[晶体结构|晶体结构]]的主要连接键。

网状共价键是共价键中的一种，它所形成的物质不是一个个的分子，而是一个大的[[原子晶体|原子晶体]]。网状共价键的连接之力很强，若想使键断开则需要有很大的能量，所以以网状共价键作为化学键的物质的熔点和沸点都较高、硬度也很大，最典型的例子之一就是[[金刚石|金刚石]]。

== 配位键 ==
{{main|配位键}}
[[配位键|配位键]]是一种特殊的共价键，它的特点在于共用的一对电子出自同一原子。形成配位键的条件是，一个原子有孤电子对，而另一个原子有空轨道。[[配位化合物|配位化合物]]，尤其是过渡金属配合物，种类繁多，用途广泛，现已形成[[配位化学|配位化学]]。
