'''共命之鸟'''（{{lang-sa|जीवजीवक}} ''{{IAST|Jīvajīvaka}}''），又名'''同命鸟'''、'''命命鸟'''<ref>{{Cite wikisource|佛學大辭典/命命鳥|wslanguage=zh}}</ref>、'''耆婆耆婆'''<ref>{{Cite wikisource|佛學大辭典/耆婆耆婆|wslanguage=zh}}</ref>，是[[佛经|佛经]]中记载的[[双头动物|双头鸟]]。

== 记载 ==
《[[佛本行集经|佛本行集经]]》中有关于共命鸟的[[譬喻_(佛教)|譬喻]]故事：在过去雪山中有一只长着两个头的鸟，一个头叫“迦嘍嗏”，能吃到美味的果实；而另一个头叫“優波迦嘍嗏”，只能吃到烂果实。一次在觅食果实时，優波迦嘍嗏为迦嘍嗏品尝毒果，被双双毒死<ref>{{cite web |url=http://tripitaka.cbeta.org/ko/T03n0190_059 |title=佛本行集經卷第五十九 |last= |first= |date= |website= |publisher= |access-date= |quote=}}</ref><ref>{{cite web |url=https://www.ctworld.org.tw/sutra_stories/story057-gb.htm |title=佛典故事 :: 共命鸟 |last= |first= |date= |website= |publisher= |access-date= |quote=}}</ref>。

《[[佛说阿弥陀经|佛说阿弥陀经]]》中也有提及共命鸟：
{{cquote|彼国常有种种奇妙杂色之鸟。[[白鹤|白鹤]]、[[孔雀|孔雀]]、[[鹦鹉|鹦鹉]]、[[:s:佛學大辭典/舍利|舍利]]、[[迦陵频伽|迦陵频伽]]、'''共命之鸟'''，是诸众鸟。昼夜六时。出和雅音。其音演畅五根、五力、七菩提分、八圣道分、如是等法。其土众生。闻是音已。皆悉念佛念法念僧。<ref>{{Cite wikisource|佛說阿彌陀經|wslanguage=zh}}</ref>}}

== 形象 ==
共命鸟的头部分为一雄一雌，共用一个身体。由于没有固定形象，在有些画作中共命鸟却是双头[[凤凰|凤凰]]或者[[双头鹰|双头鹰]]。在安西[[榆林窟|榆林窟]]第25窟南壁壁画《观无量寿经变》中，共命鸟是上半身是人、下半身是鸟的弹奏[[琵琶|琵琶]]伎乐天形象。<ref>{{cite web |url=http://news.ifeng.com/a/20180116/55170318_0.shtml |title=佛教中的神鸟：这对双头共命鸟竟是死于“吃独食” |last= |first= |date=2017-12-29 |website=凤凰佛教 |publisher= |access-date= |quote=}}</ref>半人半鸟的共命鸟在古代又称作“鸱鸮”，常装饰在金刚宝座的下部<ref>{{cite web |url=http://fo.sina.com.cn/culture/painting/2013-10-09/112913344.shtml |title=迦陵频伽与共命鸟 |last= |first= |date= |website= |publisher= |access-date= |quote=}}</ref>。
