'''獅子山下獎學金'''（英語：'''Lion Rock Scholarship'''）由[[香港浸會大學|香港浸會大學]][[傳理學院|傳理學院]][[校友|校友]]發起，旨在獎勵就讀傳理學院、並且為[[香港永久性居民|香港永久性居民]]的[[本科生|本科生]]。獅子山下獎學金不以[[學分|學分]]或成績為頒授標準，評審基於學生的作品，分為[[電影|電影]]、[[新聞|新聞]]、[[廣告|廣告]]三項。獅子山下獎學金在2014年12月10日至17日一週之內，籌募超過港幣三十萬元。<ref>{{Cite web | url = http://www.facebook.com/lionrockscholarship | title = 專頁 |publisher =獅子山下獎學金專頁}}</ref>

== 發展 ==
2014年12月7日，6名傳理學院校友在[[香港|香港]]舉行第一次[[籌備委員會|籌備委員會]]會議，決定命名「獅子山下獎學金」。<br />

2014年12月10日，獅子山下獎學金籌備委員會在其[[Facebook|Facebook]]呼籲認捐。<br />

2014年12月12日，《明報》報導，[[文匯報_(香港)|文匯報]]因不滿浸會大學傳理學院新聞系學生支持佔領，撤回捐予學生4個共值2萬元的獎學金。<ref>{{Cite web | url = http://news.mingpao.com/ins/浸大：《文匯報》撤回新聞系學生獎學金/web_tc/article/20141212/s00001/1418317068228 | title = 浸大：《文匯報》撤回新聞系學生獎學金 |publisher =《明報》}}</ref>同日，獅子山下獎學金專頁轉載《明報》的報導，校友[[柳俊江|柳俊江]]在Facebook撰文支持獅子山下獎學金，並且公開認捐。<br />

2014年12月13日，《蘋果日報》報導文匯報撤回獎學金一事，報導提及獅子山下獎學金，是該獎學金首次見報。<ref>{{Cite web | url = http://hk.apple.nextmedia.com/news/art/20141213/18967555 | title = 不滿學生佔中　《文匯》撤浸大獎學金 |publisher =《蘋果日報》}}</ref>同日，[[劉夢熊|劉夢熊]]向獅子山下獎學金認捐一萬元，是單一最大筆捐款。其後，《蘋果日報》引述劉夢熊指：「捐得錢畀大學，就唔應該有附帶條件，有政治目的就更加唔應該，所以要捐錢以示支持學生嘅價值判斷」。<ref>{{Cite web | url = http://hk.apple.nextmedia.com/news/art/20141216/18970952 | title = 尖啤捐錢浸大新聞系
 |publisher =《蘋果日報》}}</ref><br />

2014年12月17日，獅子山下獎學金籌得三十萬元。

== 名稱和徽號 ==

《明報》引述獅子山下獎學金發言人，形容[[獅子山_(香港)|獅子山]]象徵香港人同舟共濟、無懼風雨的精神，而香港浸會大學正正位處獅子山下，故以「獅子山下獎學金」為名。<ref>{{Cite web | url = http://news.mingpao.com/ins/浸會舊生籌組「獅子山下獎學金」%20批企業撤獎學金%20扼殺言論自由/web_tc/article/20141213/s00001/1418416517935 | title = 浸會舊生籌組「獅子山下獎學金」 批企業撤獎學金 扼殺言論自由 |publisher =《明報》}}</ref>

獅子山下獎學金的徽號以黃色和黑色為主色，中心是獅子山的形象，中間豎排「唯真為善」四字是傳理學院的院訓，下方是從右至左「獅子山下獎學金」七個大字。
