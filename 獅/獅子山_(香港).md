'''獅子山'''（英語：{{lang|en|Lion Rock}}）是[[香港|香港]]一個著名的[[山峰|山峰]]，位於[[九龍|九龍]]和[[新界|新界]]之間，為[[九龍群山|九龍群山]]之一。獅子山因為形狀以及外貌像似一隻[[獅子|獅子]]而得名。獅子山海拔495米，在九龍不少地方都可望到，是香港其中一個著名[[地標|地標]]。獅子山為[[獅子山郊野公園|獅子山郊野公園]]的主要部分，亦為該郊野公園的名稱由來。

獅子山為昔日[[醉酒灣防線|醉酒灣防線]]的一部分，[[山腰|山腰]]部分依然留有一些軍事遺跡，包括[[座標|座標]]石和[[機槍|機槍]]堡。

[[獅子山隧道|獅子山隧道]]穿越獅子山以西的[[九龍坳|九龍坳]]，為一條連接[[九龍塘|九龍塘]]及[[大圍|大圍]]的行車隧道，亦是[[香港1號幹線|1號幹線]]的一部分；而副峰下則有[[屯馬綫|屯馬綫]]的[[獅子山鐵路隧道|獅子山鐵路隧道]]通過。

== 傳說 ==
民間相傳九龍過去有九條[[龍|龍]]為患，所以[[上天|上天]]派了一隻獅子下凡鎮壓。現時的[[八仙嶺|八仙嶺]]就是收拾了其中八條龍，餘下一條則被獅子壓著在腳下，而那一隻獅子就是現在的獅子山，那麽被壓著的第九條龍就是[[九龍山|九龍山]]。

獅子山又有虎頭山，駱駝山等別名。<ref>{{cite book|author1=黃佩佳|title=新界風土名勝大觀|date=2016年7月|publisher=商務印書館（香港）|pages=71|edition=第一版第一次印刷|quote="虎頭山，在九龍城之北，高一六二〇英尺，其名至多，若獅子山……"}}</ref>

== 地理和旅遊 ==
[[File:Kwun_Tong_Road_1945.jpg|thumb]]
[[File:Lok_Fu_2010.jpg|thumb]]望向獅子山]]

獅子山為九龍和新界之天然分界線，主要是[[花崗岩|花崗岩]]結構；山嶺的北面為[[大圍|大圍]]、南面為[[樂富|樂富]]和[[黃大仙|黃大仙]]、東面為[[雞胸山|雞胸山]]和[[慈雲山|慈雲山]]、西面為[[筆架山_(香港)|筆架山]]（與之隔著[[九龍坳|九龍坳]]）。獅子山的山嶺由東至西伸展，高而狹長。其北面坡度較為平緩，但南面則十分陡峭，山頂一帶更是一片懸崖峭壁。山頂一帶約可分為三個高峰，三峰高度相若。在山的北面和西面都有正式的路徑上山，連接山頂及[[麥理浩徑|麥理浩徑]]第五段。不過因為部分路段較為危險，遊人緊記量力而為。

由麥理浩徑第五段，走過[[電塔|電塔]]下，再經一段0.5公里的石級可到達東面的山峰；亦可經九龍坳前往獅子山，只是因沒有維修，且多亂石，才較右邊上山的路難行。兩者的起點均設有警告牌，內容：此段路艱險難行，只宜有經驗及裝備良好的人士前往。山峰上有一由[[漁農自然護理署|漁農自然護理署]]竪立的標高牌。山峰的東面有一巨型的警告牌，足有一個人高，牌上寫著「懸崖危險，切勿前進」。不過，牌後的地方才是獅子山的最高點，比牌前地面的高度高出一呎左右。此外，牌後的地方亦是[[獅尾脊|獅尾脊]]的起點，可前往[[石馬|石馬]]及[[傻人塔|傻人塔]]。

遊人沿山脊西走，經一小山谷後便可到達中部的山峰。在該山峰上可跟西面的山崖對望，兩峰相距不過百米。

西面的山崖即獅子的「[[头|頭部]]」。山崖的底部亦竪立了跟其他地方一般樣式的懸崖警告牌。不過不難找到遊人爬上崖頂拍照留念。從九龍坳附近的麥理浩徑第五段，沿山脊登上西崖只需走0.4公里。

很多有關獅子山的風景照，都是從九龍以至[[香港島|香港島]]向北面拍攝的，可以看到獅子山整個南面的山坡。人們從市區眺望獅子山的角度亦多數如此。不過從該角度看獅子山其實並不太像獅子。如要觀看獅子山的「盧山真面目」，應從獅子山以西的地方眺望。在西崖一面的山路為一個上佳的觀賞位置。在該處可發現西崖岩石磷峋，似如獅子蹲伏在山嶺，並面向西方，雄視著香江，鎮守著九龍。

== 大眾文化與政治 ==
[[File:Roc_flag_in_lion_rock.jpg|thumb]]
[[File:Lion_Rocks_Umbrella_Revolution_Banner_20141024.jpg|thumb]]」巨型條幅]]
因為[[第二次世界大戰|第二次世界大戰]]、[[國共內戰|國共內戰]]和之後中國大陸政治運動，很多低下階層的人移居香港。獅子山南面的九龍是其中一個最多人住的地方，有山邊[[寮屋|寮屋]]、[[唐樓|唐樓]]、[[公共房屋|公共房屋]]、[[工廠大廈|工廠大廈]]和[[山寨廠|山寨廠]]等。因此「獅子山下」代表了當時典型的香港社會。1972年至1994年期間，[[香港電台|香港電台]]製作了一套名為《[[獅子山下|獅子山下]]》的電視劇。而這個名稱自1979年起亦成為該劇主題曲的名稱（獅子山下），由[[顧嘉煇|顧嘉煇]]作曲，[[黃霑|黃霑]]填詞，[[羅文|羅文]]主唱。

1950至1970年代期間，每逢[[雙十節|雙十節]]前後，都會有親[[國民黨|國民黨]]人士到獅子山頂懸掛[[中華民國國旗|中華民國國旗]]。

2014年10月23日上午11時左右，一群「香港蜘蛛仔」攀山愛好者在獅子山頂懸掛一幅長28米、寬6米的巨型黃底黑字條幅，上面有一把[[雨傘|雨傘]]標誌，並寫上「[[真普選|我要真普選]]」，下寫「UMBRELLA MOVEMENT」以聲援「[[雨傘運動|雨傘運動]]」。翌日（24日）中午，在[[政府飛行服務隊|政府飛行服務隊]]、[[消防處|消防處]]、[[民安隊|民安隊]]的聯合行動中被移除。此後，飛鵝山、大帽山等香港多個山峰被多次掛上同類型條幅，以支持「雨傘運動」。
