'''易安迪F59PH型柴電機車'''是由[[通用汽車|通用汽車]][[易安迪|易安迪]]於1988年至2001年生產的旅客用{{convert|3000|hp|MW|0}}四軸[[柴油機車#電力傳動|柴電機車]]，[[UIC鐵路機車軸式分類|UIC車輛軸式]]為[[Bo-Bo|Bo-Bo]]。本型機車在設計時曾於[[加拿大|加拿大]][[多倫多|多倫多]]的[[GO運輸|GO運輸]]討論並取得成功，成為GO運輸的主力機車。[[南加利福尼亞州|南加州]]的[[南加州都會鐵道|南加州都會鐵道]]（{{lang|en|Metrolink}}）於1992年引進本型機車，為另一個主要客戶。本型車於1994年停產，共生產73輛。

'''易安迪F59PHI型柴電機車'''是為{{link-en|加州美鐵|Amtrak California}}城際鐵路運輸設計的，於1994年開始生產，與F59PH最大的不同是流線形外觀的駕駛室。本型機車在1994年至2001年間共生產83輛，大部份用在通勤鐵路上。

'''易安迪F59PH型柴電機車'''是易安迪（{{lang|en|EMD}}）F59系列首款機車。1988年5月至1994年5月間，{{lang|en|EMD}}共為[[加拿大|加拿大]][[多倫多|多倫多]]的[[GO運輸|GO運輸]]與[[美國|美國]][[南加州|南加州]][[南加州都會鐵道|南加州都會鐵道]]（{{lang|en|Metrolink}}）生產73輛機車<ref name = "McDonnell-2015-208">{{Harvnb|McDonnell|2015|p=208}}</ref>。{{lang|en|Metrolink}}仍使用本型機車，但GO運輸自2008年起逐漸以[[MPI_MPXpress_柴電機車#MP40PH-3C|MPI MP40PH-3C]]柴電機車取代本型機車<ref name = "Solomon-2011-159">{{Harvnb|Solomon|2011|p=159}}</ref>，僅保留8輛在2011年更新的機車；這8輛也將被[[MPI_MPXpress_柴電機車#MP54AC|MPI MP54AC]]柴電機車取代。

本型機車採用有[[渦輪增壓器|渦輸增壓]]的[[二行程循環|二行程]][[V12引擎|12汽缸]][[易安迪710系列柴油機|EMD 12-710G3A]]引擎。外觀上，本型機車前後端均設有平臺，全寬的駕駛室端面有3片駕駛窗<!--，有別於{{lang|en|EMD}}既有斗篷式機車（{{lang|en|cowl unit}}）的兩片或採用加拿大式駕駛室（{{lang|en|Canadian comfort cab}}）機車的4片--><ref name = "Solomon-2011-159" />。

本型機車目前主要在通勤鐵路使用。Metrolink將5輛機車改造，將冷卻系統升級，引擎改為符合美國二級排放標準的12-710G3Eco，並改用電子警鐘，形式改為F59PHR。[[北卡羅萊納州|北卡羅萊納州]]交通局的1859號機車正以相同標準改造，預計2018年完工。

'''易安迪F59PHI型柴電機車'''是活躍在[[北美洲|北美洲]]鐵路幹線的旅客列車用[[柴油機車#電力傳動|柴電機車]]<ref name="age94">{{cite web | title = F59PHI debuts in California | url = http://www.highbeam.com/doc/1G1-15991684.html | publisher = ''Railway Age'' {{Subscription required | via = HighBeam Research}} | accessdate = 2018-10-19 | date = 1994-12-01 | language = en | archive-url = https://web.archive.org/web/20150924142642/http://www.highbeam.com/doc/1G1-15991684.html | archive-date = 2015-09-24 | dead-url = yes }}</ref>。本型機車自1994年起生產，採用有[[渦輪增壓器|渦輸增壓]]的[[二行程循環|二行程]][[V12引擎|12汽缸]][[易安迪710系列柴油機|EMD 12-710G3C-EC]]引擎，引擎功率{{convert|3200|hp|MW|1}}。傳動系統採用交－直流電傳動，柴油引擎輸出軸通過聯軸器驅動[[交流發電機|交流發電機]]，發出的交流電通過大功率矽整流裝置轉換為直流電，然後供給四台D87BTR型直流牽引馬達。馬達採用[[軸懸式驅動方式|軸懸式驅動方式]]，牽引齒輪傳動比為56:21，最高車速可達{{convert|110|mph|km/h}}。本型機車採用箱形車體，車體內設有通往引擎室與車尾的走道，使機車能在使用時進行維護保養。外觀最顯眼的是使用複合材料做成的流線形鼻端<ref name = "age94" />。

本型機車另有發電機以提供客車照明、[[電熱|電熱]]，與[[空氣調節|空調]]用電。此發電機可產生480[[伏特|伏特]]頻率60赫茲的[[交流電|交流電]]，功率{{convert|500|and|750|kW|hp|lk=on}}。此發電機由副引擎驅動，故主引擎只需產生牽引用動力，不因[[列車供電|列車供電]]影響輸出功率。

本型機車於1994年後期在[[加利福尼亞州運輸部|加利福尼亞州運輸部]]初登場時，是美國唯一一款符合加州嚴格廢氣排放標準的鐵路機車<ref name="age94" />。
