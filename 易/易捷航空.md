'''易捷航空'''（'''{{lang|en|easyJet}}'''）是一間英國的航空公司，总部位于[[英国|英国]][[伦敦卢顿机场|伦敦卢顿机场]]<ref>http://uk.reuters.com/business/quotes/companyProfile?symbol=EZJ.L</ref>，營運超過600條的國內、國際航線航線，航點遍布32個國家<ref>{{Cite web |url=http://www.easyjet.com/EN/Routemap/ |title=存档副本 |access-date=2018-09-10 |archive-url=https://web.archive.org/web/20150304174300/http://www.easyjet.com/EN/routemap/ |archive-date=2015-03-04 |dead-url=yes }}</ref><ref name="ADB">{{cite web | title = easyJet airline details |publisher=theAirDB| url = http://www.theairdb.com/airline/easyJet.html | accessdate =26 January 2010}}</ref>，載客數首屈一指，是英國最大的航空公司，欧洲第二大[[低成本航空公司|低成本航空公司]]，仅次于[[瑞安航空|瑞安航空]]<ref>{{cite web |url=http://www.ryanair.com/doc/investor/2009/Annual_report_2009_web.pdf |publisher=Ryanair |page=3 |accessdate=25 January 2010 |title=Annual Report 2009}}</ref>。
[[File:Easyjet_A319_G-EZEZ_AMS_2016-09-04.jpg|thumb]]客机，摄于[[阿姆斯特丹斯希普霍尔机场|阿姆斯特丹斯希普霍尔机场]]。]]
1995年成立後，易捷航空發展快速，[[併購|併購]]了其他公司<ref name="GO1">{{cite news  | title = easyJet buys Go for £374m  | publisher = BBC  | date = 16 May 2002  | url = http://news.bbc.co.uk/1/hi/business/1990691.stm  | accessdate = 28 January 2010  | archiveurl = https://www.webcitation.org/5n7M8djCe?url=http://news.bbc.co.uk/2/hi/business/1990691.stm  | archivedate = 2010-01-28  | dead-url = no  }}</ref><ref name="GB1">{{cite news  | title = easyJet agrees to buy GB Airways  | publisher = BBC  | date = 25 October 2007  | url = http://news.bbc.co.uk/1/hi/business/7061246.stm  | accessdate = 28 January 2010  | archiveurl = https://www.webcitation.org/5n7MJ5yGV?url=http://news.bbc.co.uk/2/hi/business/7061246.stm  | archivedate = 2010-01-28  | dead-url = no  }}</ref>，由於選擇低成本航空旅遊的旅客持續增長，便在歐洲各地設置樞紐機場。目前，易捷航空與子公司[[瑞士易捷航空|瑞士易捷航空]]擁有超過200架飛機的機隊，<ref name=fleet>{{cite web |title = Annual report and accounts 2009 |url = http://2009annualreport.easyJet.com/files/pdf/easyJet_AR09.pdf |page = 20 |accessdate = 25 January 2010 |publisher = easyJet |archive-url = https://web.archive.org/web/20100202062706/http://2009annualreport.easyjet.com/files/pdf/easyJet_AR09.pdf |archive-date = 2010-02-02 |dead-url = yes }}</ref>多數為[[空中巴士A319|空中巴士A319]]；<ref name=fleet/>在歐洲各地設置共23座樞紐機場，規模最大為[[蓋威克機場|蓋威克機場]]<ref name=bases>{{cite web |title = Annual report and accounts 2009 |url = http://2009annualreport.easyJet.com/files/pdf/easyJet_AR09.pdf |page = 13 |accessdate = 25 January 2010 |publisher = easyJet |archive-url = https://web.archive.org/web/20100202062706/http://2009annualreport.easyjet.com/files/pdf/easyJet_AR09.pdf |archive-date = 2010-02-02 |dead-url = yes }}</ref>。2012年，易捷航空一共運載超過5000萬名旅客<ref>{{cite web |title =All Services 2012 |url =http://www.caa.co.uk/docs/80/airline_data/2012Annual/Table_0_1_6_All_Services_2012.pdf |accessdate=23 June 2013 | publisher=Civil Aviation Authority}}</ref>。

== 历史 ==
英国易捷航空（easyJet）成立于1995年。创始人斯泰·利奥斯。最初，易捷航空机队运营单一机型[[波音737|波音737]]。2004年，易捷航空订购了120架[[空中客车A319|空中客车A319]]，易捷航空成为使用空中客车A319作为主力机型的低成本航空公司。

2017年10月，以4,000萬[[歐元|歐元]]收購[[德國|德國]][[柏林航空|柏林航空]]在[[泰格爾機場|泰格爾機場]]的業務。
　　

== 业务 ==

易捷航空(easyJet)提供的飞航点遍及全欧洲，航线采取的是点对点的直航，而不是辐射状的网络。

2012年，易捷航空客运量达到58,399,840人次，列欧洲廉价航空第二，仅次于瑞安航空。

== 貴賓會員 ==

易捷航空的貴賓會員制度為EasyJet Plus，加入EasyJet Plus的費用為每年199歐元，會員可以獲得優先免費選位、優先報到手續、額外手提行李、快速通關、優先登機與免費更改航班等特別服務。
