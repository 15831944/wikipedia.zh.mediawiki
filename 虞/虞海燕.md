'''虞海燕'''（{{bd|1961年|7月||catIdx=Y虞}}），男，[[浙江|浙江]][[义乌|义乌]]人。1982年8月参加工作，1987年4月加入[[中国共产党|中国共产党]]。[[西安建筑科技大学|西安冶金建筑学院]]（今[[西安建筑科技大学|西安建筑科技大学]]）冶金系炼钢专业毕业，研究生学历。曾任[[中共甘肃省委|中共甘肃省委]]常委、常务副省长及兰州市委书记等职。

历任[[酒泉钢铁|酒泉钢铁]]公司炼钢厂副厂长、厂长，酒泉钢铁公司总经理助理兼生产处处长，酒泉钢铁(集团)有限责任公司副总经理兼总工程师、总经理，甘肃省政府国有资产监督管理委员会主任，中共[[天水市|天水市]]委书记、市人大常委会主任，酒泉钢铁（集团）有限责任公司董事长等职。2011年4月，任[[中共甘肃省委|中共甘肃省委]]常委<ref>{{cite web|accessdate=2012-04-21|url=http://gs.people.com.cn/n/2012/0428/c183283-16991879.html|title=王三运当选中共甘肃省委书记|publisher=[[人民网|人民网]]|deadurl=yes|archiveurl=https://web.archive.org/web/20120505234933/http://gs.people.com.cn/n/2012/0428/c183283-16991879.html|archivedate=2012年5月5日|df=}}</ref>，5月当选[[甘肃省人民政府|甘肃省人民政府]]副省长<ref>{{cite web|accessdate=2012-04-21|url=http://www.chinanews.com/gn/2011/05-30/3075272.shtml|title=虞海燕、李建华获任甘肃省副省长|publisher=[[中新网|中新网]]}}</ref>。2012年10月，任[[中共甘肃省委|中共甘肃省委]]常委、[[兰州市|兰州市]]委书记<ref>{{cite web|title=甘肃省委常委、副省长李荣灿任兰州市委书记，虞海燕另有任用|url=http://www.thepaper.cn/newsDetail_forward_1557355|publisher=澎湃新闻|accessdate=2016-11-09}}</ref>。2016年11月，任[[中共甘肃省委|中共甘肃省委]]常委、[[甘肃省人民政府|甘肃省人民政府]]党组副书记、常务副省长<ref>{{cite web|title=甘肃省委常委虞海燕任省政府党组副书记，此前为兰州市委书记|url=http://www.thepaper.cn/newsDetail_forward_1564613|publisher=澎湃新闻|accessdate=2016-11-20}}</ref>。

==落马与受审==
2017年1月11日，[[中共中央纪委|中央纪委]][[中华人民共和国监察部|监察部]]网站宣布，虞海燕涉嫌严重违纪，接受组织调查<ref>{{cite web|title=甘肃省委常委、副省长虞海燕涉嫌严重违纪接受组织调查|url=http://www.thepaper.cn/newsDetail_forward_1598398|publisher=澎湃新闻|accessdate=2017-01-11}}</ref>。 虞海燕是中共十八大以来甘肃省落马的第二位省部级干部，也是连续第二任落马的中共兰州市委书记，前一位是[[陆武成|陆武成]]。2017年6月4日被开除党籍、开除公职，移送司法机关<ref>{{cite web |url = http://www.ccdi.gov.cn/xwtt/201706/t20170604_100472.html |title = 甘肃省原省委常委、副省长虞海燕严重违纪被开除党籍和公职 |accessdate = 2017年6月4日 |publisher = 中央纪委监察部网站}}</ref>。

2018年5月24日，重庆市第一中级人民法院一审公开开庭审理了虞海燕受贿一案。指控1998年至2016年，虞海燕利用职务上的便利，为有关单位和个人在产品销售、工程承揽、房地产开发、职务调整晋升等事项上谋取利益，直接或通过其妻子李岩华收受相关单位和个人给予的财物，共计折合人民币6563万余元。7月18日，重庆市第一中级人民法院公开宣判，对被告人虞海燕以受贿罪判处有期徒刑十五年，并处罚金人民币六百万元。对虞海燕受贿所得财物及其孳息，予以追缴，上缴国库<ref>{{cite web |url = https://news.sina.cn/2018-07-18/detail-ihfnsvza2023997.d.html?from=wap |title = 甘肃省原省委常委、副省长虞海燕被判处有期徒刑15年 |accessdate = 2018年7月18日 |publisher = 新浪网}}</ref>。

2019年1月，中纪委主管的《中国纪检监察杂志》发文称虞海燕在任兰州市委书记期间，违规从酒泉钢铁集团公司调入一批干部，形成了一个政治小圈子，被戏称为“酒钢号”<ref>{{cite web|url=https://zgjjjc.ccdi.gov.cn/bqml/bqxx/201901/t20190127_187745.html|date=2019-01-27|publisher=中国纪检监察杂志|title=有权必有责 用权受监督——习近平总书记关于加强权力监督的重要论述揽要}}</ref>。
