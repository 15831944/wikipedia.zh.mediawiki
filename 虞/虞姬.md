'''虞姬'''，又稱'''虞美人'''（{{bd|約前233年|||前202年|catIdx=Y}}），名虞<ref name="a1"/>或[[虞姓|姓虞]]<ref name="a2"/>，西楚霸王[[項羽|項羽]]的[[後宮美人|美人]]<ref name="a1">《史记·卷七·项羽本纪第七》：有美人名虞，常幸从；骏马名骓，常骑之。於是项王乃悲歌慷慨，自为诗曰：“力拔山兮气盖世，时不利兮骓不逝。骓不逝兮可柰何，虞兮虞兮柰若何！”歌数阕，美人和之。项王泣数行下，左右皆泣，莫能仰视。</ref><ref name="a2">《汉书·卷三十一·陈胜项籍传第一》：有美人姓虞氏，常幸从；骏马名骓，常骑。乃悲歌慷慨，自为歌诗曰：“力拔山兮气盖世，时不利兮骓不逝。骓不逝兮可奈何！虞兮虞兮奈若何！”歌数曲，美人和之。羽泣下数行，左右皆泣，莫能仰视。</ref><ref>《史记正义·卷七·项羽本纪第七》：楚汉春秋云：“歌曰‘汉兵已略地，四方楚歌声。大王意气尽，贱妾何聊生’。”</ref>。與[[白娘子|白娘子]]、[[孟姜女|孟姜女]]、[[王寶釧|王寶釧]]並稱中國古代四大情女。

== 生平 ==

虞姬的身世不详，仅知她是项羽身边的美人，常伴随项羽左右。

前202年，項羽在[[垓下之戰|垓下之戰]]中被[[劉邦|劉邦]]、[[韓信|韓信]]、[[彭越|彭越]]、[[英布|英布]]四面八方大軍合圍困於[[垓下|垓下]]（今[[安徽省|安徽省]][[靈璧縣|靈璧縣]]城南沱河北岸城後村），身陷[[十面埋伏|十面埋伏]]，兵孤糧缺，夜聞[[四面楚歌|四面楚歌]]，[[楚|楚]]軍士氣盡失。項羽認為大勢已去，帳中酌酒，對著虞姬唱起悲壯的「[[垓下歌|垓下歌]]」：『力拔山兮氣蓋世。時不利兮騅不逝。騅不逝兮可奈何！虞兮虞兮奈若何！』。<ref>{{Cite web|url=https://zh.wikisource.org/wiki/%E5%8F%B2%E8%A8%98/%E5%8D%B7007|title=史記/卷007 - 维基文库，自由的图书馆|accessdate=2018-02-06|work=zh.wikisource.org|language=zh}}</ref>虞姬唱和：『漢兵已略地，四方楚歌聲。大王意氣盡，賤妾何聊生。』

最终垓下之战，项羽自杀身亡。但史书没有记载虞姬的结局。后世文艺作品通常设定为，虞姬自杀[[殉情|殉情]]。

== 歷史記載 ==
《[[史记|史記]]》中[[司馬遷|司馬遷]]的記載虞姬經常伴於項羽（並未提及死亡原因、司馬遷是最接近虞姬年代的史官）：

項王軍壁垓下，兵少食盡，漢軍及諸侯兵圍之數重。夜聞漢軍四面皆楚歌，項王乃大驚曰：「漢皆已得楚乎？是何楚人之多也！」項王則夜起，飲帳中。有美人名虞，常幸從；駿馬名騅，常騎之。於是項王乃悲歌慨，自為詩曰：「力拔山兮氣蓋世，時不利兮騅不逝。騅不逝兮可柰何，虞兮虞兮柰若何！」歌數闋，美人和之。項王泣數行下，左右皆泣，莫能仰視。<ref>{{Cite web|url=https://zh.wikisource.org/wiki/%E5%8F%B2%E8%A8%98/%E5%8D%B7007|title=史記/卷007 - 维基文库，自由的图书馆|accessdate=2018-02-06|work=zh.wikisource.org|language=zh}}</ref>

《[[太平寰宇记|太平寰宇記]]》中提到是項羽在敗戰之際，殺死虞姬：

卷一二八「濠州鍾離縣」：「虞姬冢在縣東南六十里，高六丈，即項羽敗，殺姬葬此。」

== 墳墓 ==
關於虞姬的墳墓，《[[括地志|括地誌輯校]]》中記載在濠州定遠縣東六十里<ref>{{Cite web|url=https://zh.wikisource.org/zh-hant/%E6%8B%AC%E5%9C%B0%E5%BF%97%E8%BC%AF%E6%A0%A1/004|title=括地誌輯校/004 - 維基文庫，自由的圖書館|accessdate=2018-02-06|work=zh.wikisource.org|language=zh-Hant}}</ref>但《[[太平寰宇记|太平寰宇記]]》認為是在濠州鍾離縣東南六十里。《[[方輿勝覽|方舆胜览]]》則寫到：「虞姬冢在定远县南，今宿州也有墓，相传是灵壁葬其身，此葬其首。」

== 虞姬廟 ==
位於[[安徽省|安徽省]]宿州市境内的插花山。

《歷陽典錄》記載：「魯妃廟在陰陵山之陽，有小山曰插花山，上有廟曰插花廟，又曰魯妃廟，蓋項羽為魯公，故虞姬稱魯妃也。」<ref>{{Cite web|url=http://ctext.org/wiki.pl?if=gb&res=394135&searchu=%E9%AD%AF%E5%A6%83|title=歷陽典錄 - 中國哲學書電子化計劃|accessdate=2018-02-06|last=Sturgeon|first=Donald|work=ctext.org|language=zh-TW}}</ref>

《直隸和州志》書卷四中《輿地志·山川》記載：「陰陵山，州北80里，項王迷道處，上有刺槍坑，為項王立槍點，旁有澤，名紅草湖。春夏之交，湖水漲發，瀰漫無際，即陰陵九曲澤，澤中有項王村，項王失路於澤中，周圍九曲，後人因以為名。」

== 影響 ==
[[File:2001年“京韵铿锵”专场演出《霸王别姬》.jpg|thumb]]》塑造的虞姬形象]]
[[File:Consort_Yu2.jpg|200px]]

雖然歷史上有關虞姬的記述不多，但[[中國|中國]]的[[文學|文學]]、[[戲劇|戲劇]]、[[小說|小說]]、[[電影|電影]]，均對她有深刻的描寫刻劃。
*北宋詩人[[蘇軾|蘇軾]]《[[虞姬墓|虞姬墓]]》：「布叛增亡國已空，摧殘羽翮自令窮。艱難獨與虞姬共，誰使西來破沛公。」
*清朝詩人[[何浦|何浦]]《[[虞美人_(词牌)|虞美人]]》：「遺恨[[江東|江東]]應未消，芳魂零亂任風飄。八千子弟同歸漢，不負君恩是楚腰（虞姬）。」
*清朝詩人[[袁枚|袁枚]]《[[過虞溝游虞姬廟|過虞溝游虞姬廟]]》：「為欠虞姬一首詩，白頭重到古靈祠。三軍已散佳人在，六國空亡烈女誰？死竟成神重桑梓，魂猶舞草濕胭脂。座旁合塑烏騅像，好訪君王月下騎。」
*清朝[[曹雪芹|曹雪芹]]《[[紅樓夢|紅樓夢]]》中的《五美吟·虞姬》：「肠断乌骓夜啸风，虞兮幽恨对重瞳。黥彭甘受他年醢，饮剑何如楚帐中？」
*《[[霸王别姬_(京剧)|霸王別姬]]》：[[京劇|京劇]]的經典名劇，[[作家|作家]][[李碧華|李碧華]]所著的[[霸王別姬_(小說)|同名小說]]，更被[[導演|導演]][[陳凱歌|陳凱歌]]改編成[[霸王別姬_(電影)|同名電影]]，獲得多个國際電影獎項。
