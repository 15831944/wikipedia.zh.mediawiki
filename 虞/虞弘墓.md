'''虞弘墓'''是一座位於[[山西省|山西省]][[太原市|太原市]]的[[隋朝|隋朝]]官員虞弘與其妻子的合葬墓，是中原地區至今發現的唯一有準確紀年的[[西域|西域]]、[[中亞|中亞]]文化墓葬<ref name=太原虞弘墓>{{cite web |url=http://www.kaogu.cn/cn/kaoguyuandi/kaogubaike/2013/1025/34131.html |title=太原虞弘墓 |author=<!--Staff writer(s); no by-line.--> |date=2009年10月21日 |website=kaogu.cn |publisher= |accessdate=2018年7月15日}}</ref>。

== 墓主簡介 ==
[[File:Rubbing_of_Yü_Hung’s_Epitaph.jpg|thumb]]
墓主的墓誌蓋上刻有「大隋故儀同虞公墓誌」九個[[篆字|篆字]]，墓誌又記載：「公諱弘，字莫潘，魚國尉紇驎城人也。」因此可知他本名叫虞弘，字莫潘，來自魚國的尉紇驎城；石棺上的浮雕，如蓮臺上的{{Link-en|祆教聖火|Atar|聖火}}，則表明他是一位[[祆教|祆教]]信徒。虞弘曾奉[[茹茹|茹茹]]國王之命出使[[薩珊王朝|波斯]]、[[吐谷渾|吐谷渾]]、[[月氏|月氏]]等西域國家以及曾經的[[安息帝國|安息帝國]]所統治的區域。後出使[[北齊|北齊]]，以後就留在北齊、[[北周|北周]]和[[隋朝|隋朝]]做官。在北周曾任「檢校薩保府」，「薩保」音譯自[[粟特語|粟特語]]「{{transl|sog|s′rtp′w}}」，本意是「商隊首領」。因為當時進入中原的[[粟特人|粟特人]]為數衆多，所以政府設立了「薩保」這種官職來管理進入中土的外國人事務，是中原地區唯一一個以外來語命名的官職。

虞弘終年59歲，於隋[[開皇|開皇]]十二年（公元592年）下葬，臨終前是隋朝的正五品[[儀同三司|儀同三司]]。從其石槨上帶濃郁波斯風格的浮雕推測，魚國可能位於中亞<ref name=太原虞弘墓 />，但究竟在何處，至今尙無準確定論。

== 墓葬概述 ==
虞弘墓位於太原市[[晉源區|晉源區]]（中國古代[[晉陽|晉陽]]）的王郭村，1999年出土，為磚砌單室墓，墓頂已毀，現存墓道、甬道、墓門及墓室，全長13.65米。墓室中央偏北處安置一座仿木構[[漢白玉|漢白玉]]石槨，並有八棱漢白玉石柱、石俑、陶俑、銅錢、白瓷碗、墓誌銘等隨葬品80餘件。這具長2.48米、寬1.38米、通高2.17米的漢白玉石槨由槨頂、槨身、底座三部份數十塊構件組成，頂部為三開間[[歇山頂|歇山頂]]建築式樣，四周佈滿浮雕，浮雕畫面施彩並局部[[描金|描金]]，內容有宴飲圖、樂舞圖、狩獵圖、家居圖、釀酒圖、行旅圖等。浮雕裡的人物皆是高鼻深目、鬍鬚濃密的[[高加索人種|高加索人種]]，其中不少人物頭戴波斯日月冠。其他圖案如銜綬鳥、帶綬馬、胡騰舞、頭部的扇形長帔等擁有濃郁的[[薩珊王朝|波斯]]、中亞風格<ref>{{cite web |url=http://big5.huaxia.com/zhwh/kgfx/2017/11/5547648.html |title=虞弘墓漢白玉石槨首露眞容 墓主虞弘何許人也？|author=<!--Staff writer(s); no by-line.--> |date=2017年11月23日 |website=huaxia.com |publisher= |accessdate=2018年7月15日}}</ref>。此外，曾在[[羅馬帝國|羅馬帝國]]受到崇拜的印度-伊朗神祇[[密特拉|密特拉]]也現身於浮雕中<ref>{{cite book |editor=Sarah Stewart; Firoza Punthakey Mistree; Ursula Sims-Williams |date=2013-12-18 |title=The Everlasting Flame: Zoroastrianism in History and Imagination |trans-title= |url=https://books.google.com/books?id=IUVTAQAAQBAJ&pg=PA23&lpg=PA23&dq=tomb+of+yu+hong+mithra&source=bl&ots=6cOVg3ot8W&sig=UgVtL4L1jUd_mxy1xqQam_ps_aM&hl=en&sa=X&ved=0ahUKEwjWiuXig6LcAhXO16QKHe8YDHsQ6AEIPzAH#v=onepage&q=tomb%20of%20yu%20hong%20mithra&f=false |page=23 |language=en |location=London |publisher=I.B.Tauris |isbn=9781780768090}}</ref>。這些浮雕畫不僅再現了墓主的故鄉風物，也為研究虞弘所信仰的自[[六朝|六朝]]初期就已經[[絲綢之路|絲綢之路]]傳入中土的[[祆教|祆教]]提供了第一手資料。此外，虞弘的石槨有槨無棺，與中原的喪葬習俗完全不同。

== 浮雕石板 ==
[[File:Panel_5_of_the_Sarcophagus_of_Yü_Hung.jpg|thumb]]
[[File:Sarcophagus_Platform_Panel_from_the_Tomb_of_Yü_Hung.jpg|thumb]]
虞弘石槨四圍內外佈滿大小不等的浮雕及彩繪圖案54個，主體部份的九幅石板浮雕畫可細分為兩組，第一組（前四幅）可能表現墓主的人世生活，第二組（後五幅）應該是描繪墓主死後在天國中的場景<ref>{{cite web |url=http://www.sohu.com/a/166030194_649680 |title=絲綢之路與太原隋代虞弘墓 |author=張慶捷 |date=2017年8月20日 |website=sohu.com |accessdate=2018年7月16日}}</ref>。九幅石板浮雕畫分別是：
# 牽馬及神馬圖
# 釀酒及猛獅鬪神馬圖
# 騎駱駝獵獅及持角形器的胡人圖
# 騎駱駝獵獅及大角羊圖
# 宴飲及人獅搏鬪圖
# 騎象搏獅及銜綬鳥圖
# 行旅飲食及山羊圖
# 出行休憩及奔鹿圖
# 騎馬出行及牛獅搏鬪圖，亦名「奉果圖」<ref>{{cite web |url=http://www.his.ntnu.edu.tw/files/publish/723_0029a7eb.pdf |title=影視教材在高中歷史教學的應用─以隋唐史教學為中心 |author=陳登武 |date=2009 |website=his.ntnu.edu.tw |page=第5頁（PDF頁數）|accessdate=2018年7月16日}}</ref>

不過經中國人民大學國學院副教授畢波考證，她認為第五幅「宴飲圖」（全稱「夫婦宴飲圖」）並非是描繪虞弘夫妻正對坐宴飲，其中的女子應是[[瑣羅亞斯德教|瑣羅亞斯德教]]（祆教）的一種天界神靈{{Link-en|妲瑞娜|Daena}}，而這幅浮雕畫的主題是表現墓主的靈魂進入天國後的美好情景<ref>{{cite web |url=http://www.dpm.org.cn/Uploads/pdf/5056/T00066_00.pdf |title=虞弘墓所謂「夫婦宴飲圖」辨析 |author=畢波 |date=2006 |website=dpm.org.cn |accessdate=2018年7月15日}}</ref>。
