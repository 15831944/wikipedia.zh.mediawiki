'''史君墓'''是一座位於[[西安|西安]][[未央區|未央區]]大明宮鄉井上村東的[[北周|北周]][[粟特人|粟特人]]史君及其妻康氏的合葬墓，發現於2003年。該墓西距[[漢長安城|漢長安城]]遺址約6.6千米，南距[[大明宮|大明宮]]遺址約1.6千米，同北周[[安伽墓|安伽墓]]相距約2.5千米。

== 墓主簡介 ==
[[File:Rubbing_of_Epitaph_of_the_Sa-pao_Wirkak_(Part_2).jpg|thumb]]
史君的墓誌位於房屋形石槨（石堂）門楣上，為一長方形石刻墓誌，以[[粟特語|粟特文]]和[[古代漢語|漢文]]雙語書寫。漢文部份開篇以[[篆書|篆書]]題記「大周[[涼州|涼州]]薩保史君石堂」，並記載「君（諱）……，[[史國|史國]]人也，本居西土，……，遷居長安，……，授涼州薩保，……[[大象_(年號)|大象]]元年……薨於家，年八十六，妻康氏……」可知這位史君以其國名「史」為姓，漢文名因墓誌殘缺不存，只知他的粟特名叫{{transl|sog|Wirkak}}（尉各伽），妻子康氏是[[康居|康國]]人，粟特名{{transl|sog|Wiyusi}}（維耶尉思）。他的祖父阿史盤陁（{{transl|sog|Rštßntk}}）就曾擔任史國薩保，父親阿奴伽（{{transl|sog|Wn’wk}}）也在很年輕時就「立功立事」。尉各伽繼承祖輩志向，擔任薩保判事曹主（事），後來北周皇帝下詔授命他為涼州薩保。他於北周大象元年（公元579年）逝世，其長子毗沙（{{transl|sog|ßr’yšmnßntk}}）、次子維摩（{{transl|sog|δrymtßntk}}）、三子富□多（{{transl|sog|Pr’wtßntk}}）在580年將史君夫妻合葬。

== 墓葬簡介 ==
[[File:Stone_Sarcophagus_in_the_Sa-pao_Wirkak’s_Tomb_3.jpg|thumb]]、[[密特拉|密特拉]]神、韋施帕卡神等場面]]
史君墓坐北朝南，由墓道、天井、過洞、甬道及墓室組成。墓道和墓室之間有兩重封門，第一重為磚砌，第二重為石製。門楣和兩側立柱上有纏枝葡萄、忍冬、伎樂、天王等浮雕圖像。

墓室中部偏北安置一石堂，就是房屋形石槨，頂部為[[歇山頂|歇山頂]]建築式樣，長2.46米、寬1.55米、高1.58米。石堂周身佈滿浮雕，殘留貼金彩繪的痕跡，有類似天王的神像、宴飲、狩獵、出行等場景，還有[[祆教|祆教]]的半人半鳥祭司、{{Link-en|祆教聖火|Atar|聖火壇}}、[[欽瓦特橋|欽瓦特橋]]、坐於蓮臺上的[[阿胡拉·馬茲達|阿胡拉·馬茲達]]、盤旋於天空的[[密特拉|密特拉]]神、迎接逝者靈魂的大氣神或風神韋施帕卡（{{transl|sog|Weshparkar}}，又譯「維什帕卡」），<ref>{{cite web |url=http://www.turfanological.com/ft/article/article.php?articleid=519 |title=高昌火祆教遺跡考 |author=林梅村 |date=2012年4月19日 |website=www.turfanological.com |access-date=2018年8月2日}}</ref><ref>{{cite web |url=http://www.silkroadfoundation.org/newsletter/vol9/srjournal_v9.pdf |title=The Silk Road: Zoroastrian Funerary Beliefs and Practices Known from the Sino-Sogdian Tombs in China |author=Judith A. Lerner |date=2011 |website=www.silkroadfoundation.org |page=22 |access-date=2018-07-31 |language=en}}</ref>[[薩珊王朝|波斯]]宗教的色彩相當濃郁。雖然石堂的基本外觀已漢化，但相較另外兩座同為北周的粟特人墓葬——[[康業墓|康業墓]]和[[安伽墓|安伽墓]]，史君墓的漢化程度最低。<ref>{{cite book |editor=吳天泰 |date=2015 |title=《族群與社會》 |url=https://books.google.com/books?id=bsYgpuud5gMC&pg=PA290&lpg=PA290&dq=%E5%BA%B7%E6%A5%AD%E5%A2%93+%E6%BC%A2%E5%8C%96&source=bl&ots=x6FACl0aD-&sig=243EbRyh7rP4XmmNZVN8EFhd8Yw&hl=en&sa=X&ved=0ahUKEwjIr-KbxbHcAhXOIlAKHQQyBGMQ6AEIJzAA#v=onepage&q=%E5%BA%B7%E6%A5%AD%E5%A2%93%20%E6%BC%A2%E5%8C%96&f=false |location=臺北 |publisher=五南圖書出版 |page=290 |isbn=9789571141756 |accessdate=2018年7月31日}}</ref>專事研究[[粟特|粟特]]文化的法國學者[[魏義天|魏義天]]認為，史君石堂上這些豐富的圖像並非單純的祆教象徵，而是混合了很多[[摩尼教|摩尼教]]元素。比如欽瓦特橋的審判場面同[[元朝|元朝]]摩尼教絹畫《[[冥王聖幀|冥王聖幀]]》所描繪的審判場景驚人地相似，所以史君墓的圖像呈現了一種{{Link-en|綜攝宗教|Religious syncretism|宗教綜攝化}}傾向。<ref>{{cite web |url=http://www.academia.edu/20031662/_Wirkak_Manichaean_Zoroastrian_Khurram%C3%AE_On_bilingualism_and_syncretism_in_Sogdian_funerary_art_Papers_in_Honour_of_Professor_Yutaka_YOSHIDA_and_Professor_Masaharu_ARAKAWA_on_Their_60th_Birthday_Studies_on_the_Inner_Asian_Languages_30_2015_p._95-112 |title=Wirkak: Manichaean, Zoroastrian, Khurramî? On bilingualism and syncretism in Sogdian funerary art |last=La Vaissière |first=Étienne de |date=2015 |website={{Link-en|academia.edu}} |access-date=2018-07-31 |language=en |author-link=魏義天}}</ref>

史君墓的壁畫主要分佈在過洞上方、墓道、天井靠近過洞的兩側、甬道和墓室牆壁，每幅畫都有分欄。但保存狀況很差，大多都已模糊不淸。<ref>{{cite web |url=http://www.chugoku-shoten.com/mokuji/cmokuji/71777/71777.pdf |title=北周史君墓 |author=楊軍凱 |date=2003 |website=www.chugoku-shoten.com |access-date=2018年7月31日}}</ref>此外，墓葬還出土金[[耳璫|耳璫]]、金[[指環|指環]]和一枚仿製[[拜占庭帝國|拜占庭]][[蘇勒德斯_(錢幣)|索里達]]的金幣。<ref>{{cite web |url=http://www.nxkg.org.cn/index.php?m=content&c=index&a=show&catid=15&id=442 |title=關於西安北周史君墓出土金幣仿製品的一點補充 |author=郭雲艷 |date=2018年7月9日 |website=www.nxkg.org.cn |access-date=2018年7月31日}}</ref>
