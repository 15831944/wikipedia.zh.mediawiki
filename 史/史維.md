'''史維'''（{{Lang-en|Wei Shyy}}，{{bd|1955年|7月19日||}}），[[台灣裔美國人|台灣裔美國人]]，[[航空太空工程學|航空太空工程學]]家，目前擔任[[香港科技大學|香港科技大學]]第四任校長
<ref>{{cite web|url=http://www.ust.hk/zh-hant/about-hkust/senior-adm/president/|title=史維簡歷}}</ref>
<ref>{{cite web |title=畢業禮默哀悼鍾士元 科大校長史維冀與廣州分校互補 |url=https://hk.news.appledaily.com/local/realtime/article/20181116/58921323 |website=蘋果日報 |accessdate=2018-11-30}}</ref>。

出生於[[台灣|台灣]]，從[[新竹市|新竹市]][[國立清華大學|國立清華大學]]取得理學士後，先後從[[密歇根大學|密歇根大學]]取得理學碩士和博士學位。其後，任職於[[佛罗里达大学|佛罗里达大学]]及位处纽约的[[通用电气研发中心|通用电气研发中心]]，任職[[香港科技大學|香港科技大學]]校長前是[[密歇根大學|密歇根大學]]Clarence L.“Kelly” Johnson讲座教授及航天工程学系系主任<ref>{{cite web |title=有台灣背景 史維獲薦任香港科大校長 |url=https://udn.com/news/story/7332/2936826 |website=聯合新聞網 |accessdate=2018-11-30}}</ref><ref>{{cite web |author1=黃雲娜 |title=大灣區 科大籌辦廣州分校 校長史維 冀與本校區發揮協同效應 |url=https://www.hk01.com/%E5%91%A8%E5%A0%B1/190644/%E5%A4%A7%E7%81%A3%E5%8D%80-%E7%A7%91%E5%A4%A7%E7%B1%8C%E8%BE%A6%E5%BB%A3%E5%B7%9E%E5%88%86%E6%A0%A1-%E6%A0%A1%E9%95%B7%E5%8F%B2%E7%B6%AD-%E5%86%80%E8%88%87%E6%9C%AC%E6%A0%A1%E5%8D%80%E7%99%BC%E6%8F%AE%E5%8D%94%E5%90%8C%E6%95%88%E6%87%89 |website=HK01 |accessdate=2018-11-30}}</ref>。
