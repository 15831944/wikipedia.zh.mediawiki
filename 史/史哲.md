'''史哲'''（{{bd|1969年|7月14日||}}），生於[[臺灣|臺灣]][[臺北市|臺北市]]，臺灣[[政治人物|政治人物]]，[[民主進步黨|民主進步黨]]黨員，為[[人本教育基金會|人本教育基金會]]董事長[[史英|史英]]之子。已婚，育有三子，曾任[[行政院勞工委員會勞工保險局|行政院勞工委員會勞工保險局]]總經理，[[高雄市政府文化局|高雄市政府文化局]]局長。前高雄市副市長<ref>[http://udn.com/news/story/6/1953209 高雄市副市長 史哲、許銘春接任]</ref>。

==經歷==
*史哲就讀[[東海大學_(台灣)|東海大學]]資訊科學系期間，為[[野百合學運|野百合學運]]的一份子，曾經成立「東海[[人間工作坊|人間工作坊]]」，積極投入政治，後肄業於[[國立中興大學|國立中興大學]][[應用數學|應用數學]]研究所。
*1995年，史哲至[[民進黨|民進黨]][[臺灣省議會|臺灣省議會]]「開拓辦公室」任職，進入政壇。曾擔任[[臺中縣長|臺中縣長]][[廖永來|廖永來]]機要秘書。2001年，廖永來落選，史哲於是離開[[臺中縣政府|臺中縣政府]]。
*2004年，史哲跟隨[[陳菊|陳菊]]進入[[行政院勞工委員會|行政院勞工委員會]]，擔任[[勞保局|勞保局]]總經理，在職期間曾負責推動勞退新制，兩年後主動請辭離開。
*2006年，陳菊當選[[高雄市市長|高雄市市長]]，史哲隨之進入[[高雄市政府|高雄市政府]]，先任[[高雄市政府新聞處|高雄市政府新聞處]]處長，後轉任[[高雄市政府文化局|高雄市政府文化局]]局長。
*2016年，原副市長[[陳金德|陳金德]]接任[[台灣中油|中油]]董事長，而升任副市長。

==爭議事件==
*2001年3月21日，因辦理[[臺中縣|臺中縣]]大安鄉[[松柏漁港|松柏漁港]]工程，涉嫌圍標，2002年被[[臺中地檢署|臺中地檢署]]依貪汙罪嫌起訴，但獲判無罪。
*2008年12月18日，在[[高雄市|高雄市]][[文化局|文化局]]長期間，曾遭[[臺灣壹週刊|臺灣壹週刊]]報導<ref>{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20081218/31240097/ |title = 高市文化局長 車上熱吻女記者 |author = 周昭平、涂建豐 |date = 2008年12月18日  |publisher = 蘋果日報 |language = zh-hant |accessdate = 2015年10月22日 |quote = 昨出刊的《壹週刊》拍到高雄市文化局長史哲與一名平面媒體女記者在車上有親密互動，甚至熱吻。史哲昨強調，他曾任高市府新聞處長，與媒體記者友好，否認有《壹週刊》照片描述行為。被指涉的女記者則回應：「與史哲是哥們，絕無男女感情！」史哲已婚並育有三名子女；該女記者則是單身。 }}</ref>，他與一名女記者深夜同車約會並熱吻，引發外遇緋聞風暴，但史哲堅決否認此事。
