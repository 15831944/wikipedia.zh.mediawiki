'''史氏魮脂鯉'''，為[[輻鰭魚綱|輻鰭魚綱]][[脂鯉目|脂鯉目]][[脂鯉科|脂鯉科]]的一個[[種|種]]。

== 分布 ==
本魚分布於[[南美洲|南美洲]][[奧里諾科河|奧里諾科河]]流域。

== 特徵 ==
本魚體色為透明的紅色，背鰭上有一些黑色的斑點(雌魚則有白色尖端)，肩上有一塊黑斑。雄魚的背鰭發育良好。眼眶有金邊。體長約3.4公分。

== 生態 ==
本魚棲息在溪流和湖泊中，性情溫和，屬雜食性，以[[蠕蟲|蠕蟲]]、小的[[昆蟲|昆蟲]]與[[甲殼動物|甲殼動物]]等為食。

== 經濟利用 ==
為觀賞性魚類，需要酸性軟性，以便於生長和繁殖。
