'''史蒂芬·文森·布西密'''（{{lang-en|Steven Vincent Buscemi}}，{{IPAc-en|b|uː|ˈ|s|ɛ|m|i}}，{{bd|1957年|12月13日|||catIdx=Buscemi, Steven}}），暱稱'''史蒂夫·布西密'''（{{lang|en|Steve Buscemi}}），是一個[[美國|美國]][[演員|演員]]與電影導演。在他漫長的職業生涯當中，他出演了多部[[獨立電影|獨立電影]]、商業電影和[[電視連續劇|電視連續劇]]的演出，如《[[紐約故事|紐約故事]]》、《[[落水狗|落水狗]]》、《[[絕世天劫|絕世天劫]]》、《{{link-en|黐孖妹|Ghost World (film)}}》和《[[大魚奇緣|大魚奇緣]]》。他也知名於演出[[高安兄弟|高安兄弟]]的電影，如《[[風雲再起時|風雲再起時]]》、《[[才子夢驚魂|才子夢驚魂]]》、《{{link-en|影子大亨|The Hudsucker Proxy}}》、《[[雪花高離奇命案|雪花高離奇命案]]》、《[[大保齡離奇綁架|大保齡離奇綁架]]》。布斯美在《[[怪獸公司|怪獸公司]]》系列中為角色Randall Boggs配音。

2010年至2014年，他在被受讚賞的電視劇《[[酒私風雲|酒私風雲]]》中飾演[[努基·湯普森|努基·湯普森]]<ref>{{Cite web |url = http://collider.com/steve-buscemi-interview-boardwalk-empire-2/ |title = Steve Buscemi Interview BOARDWALK EMPIRE |author = Christina Radish |date = 2010-09-09 |publisher = Collider |language = en}}</ref><ref>{{Cite web |url = https://www.telegraph.co.uk/culture/tvandradio/8812019/Steve-Buscemi-and-Kelly-Macdonald-Boardwalk-Empire-interview.html |title = Steve Buscemi and Kelly Macdonald: Boardwalk Empire interview |author = Jane Mulkerrins |date = 2011-10-07 |publisher = The Telegraph |language = en}}</ref>，為他獲得兩項[[美國演員工會獎|美國演員工會獎]]、一項[[金球獎|金球獎]]及提名兩項[[艾美獎|艾美獎]]。他在1996年首次執導電影《{{link-en|伤心树屋|Trees Lounge}}》，他也在其中主演。其他執導的電影包括《{{link-en|野獸工場|Animal Factory}}》（2000年）、《{{link-en|孤独的吉姆|Lonesome Jim}}》（2005年）和《{{link-en|跟蹤大明星|Interview (2007 film)}}》（2007年）。他也執導多個電視劇的集數，包括《''{{link-en|Homicide: Life on the Street|Homicide: Life on the Street}}''》、《[[黑道家族|黑道家族]]》、《[[監獄風雲_(美國電視劇)|監獄風雲]]》、《[[娛樂揸Fit人|娛樂揸Fit人]]》和《[[護士當家|護士當家]]》。

== 早年 ==
布西密出生於[[紐約|紐約]]的[[布魯克林|布魯克林]]，父親是John Buscemi，曾是垃圾清潔工和[[韓戰|韓戰]][[退伍軍人|退伍軍人]]，而母親是Dorothy（娘家姓Wilson），是{{link-en|Howard Johnson's|Howard Johnson's}}酒店的老闆。布斯美的父親有[[意大利|意大利]]血統，他的祖先是來自西西里島的[[門菲|門菲]]鎮。布斯美的母親是愛爾蘭、英格蘭和荷蘭的世系。他有三個兄弟，Jon、Ken和Michael，而Michael也是演員。布斯美在[[天主教會|羅馬天主教]]的環境下長大。<ref>{{Cite web |url = http://www.johnlahr.com/stevebuscemi.html |title = Steve Buscemi profile |author = John Lahr |language = en |deadurl = yes |archiveurl = https://web.archive.org/web/20111029094615/http://www.johnlahr.com/stevebuscemi.html |archivedate = 2011-10-29 }}</ref>

他於1975年畢業於紐約州[[納蘇縣_(紐約州)|納蘇縣]]的{{link-en|谷溪中央高中|Valley Stream Central High School}}，他與女演員{{link-en|派翠西亞·沙邦努|Patricia Charbonneau}}一起上學。在高中時，布斯美為校隊打摔角，並參加了話劇團。布斯美在1996年電影《{{link-en|伤心树屋|Trees Lounge}}》中演出並擔任編劇和導演，內容設定在他兒時的[[谷溪|谷溪]]村，而且大部分時間在那裡拍攝。<ref>{{Cite web |url = https://www.nytimes.com/1997/06/01/nyregion/cinema-arts-film-festival-stresses-the-independents.html |title = Cinema Arts Film Festival Stresses the Independents |author = Barbara Delatiner |date = 1997-06-01 |publisher = The New York Times |language = en}}</ref>

布斯美在搬到[[曼哈頓|曼哈頓]]加入{{link-en|李·史特拉斯堡戲劇和電影機構|Lee Strasberg Theatre and Film Institute|李·史特拉斯機構}}之前曾短暫就讀{{link-en|納蘇社區學院|Nassau Community College}}。1980年，布斯美成為紐約市的一名消防員前，18歳的他在1976年進行公務員考試。四年來，布斯美在曼哈頓[[小意大利_(曼哈頓)|小意大利]]的{{link-en|紐約市消防局|New York City Fire Department}}（FDNY）的Engine Co. 55中服務。[[911恐怖襲擊|911恐怖襲擊]]後，布斯美回到Engine 55，連續幾天與其他的消防員一起工作12小時，仔細檢查[[世貿中心|世貿中心]]的瓦礫。2003年，他在一次工會集會上發表了支持消防員爭取更高工資的演講<ref>{{Cite web |url = http://www.indieking.com/E55.html |title = Steve Buscemi – Former Firefighter |publisher = indieking.com |language = en |deadurl = yes |archiveurl = https://web.archive.org/web/20080409210850/http://www.indieking.com/E55.html |archivedate = 2008-04-09 }}</ref>。2014年，他被任命為FDNY的榮譽大隊長<ref>{{Cite web |url = https://mobile.twitter.com/fdny/status/530151048577417217 |title = FDNY Twitter |date = 2014-11-06 |publisher = Twitter |language = en}}</ref>。

== 生涯 ==
=== 從演 ===
[[File:Steve_Buscemi_2017.jpg|thumb]]]]
1985年開始演員生涯。一般是在電視連續劇演出，曾參演作品包括《[[邁阿密風雲|邁阿密風雲]]》、《[[洛城法網|洛城法網]]》、《[[魔界奇譚|魔界奇譚]]》、《[[黑道家族|黑道家族]]》，以至《[[仁心仁術|仁心仁術]]》等等。他亦有參演電影，於1989年以《{{link-en|三個藍月亮|Mystery Train (film)}}》獲得[[獨立精神獎|獨立精神獎]]最佳男配角提名，1992年憑低成本電影《[[落水狗|落水狗]]》贏得該獎項。

布斯美在HBO電視劇《[[酒私風雲|酒私風雲]]》中飾演[[努基·湯普森|努基·湯普森]]（基於真實歷史人物[[伊努·L·強森|伊努·L·強森]]），這角色是一名在[[禁酒令時期|禁酒令時期]]管治[[大西洋城|大西洋城]]的腐敗政客，為他贏得了一項{{link-en|金球獎最佳戲劇類電視男主角|Golden Globe Award for Best Actor – Television Series Drama}}。2011年，他主持[[全國廣播公司|NBC]]的《[[週六夜現場|週六夜現場]]》。<ref>{{Cite web |url = https://www.hollywoodreporter.com/news/saturday-night-live-steve-buscemi-snl-review-269443 |title = 'Saturday Night Live' Hosted by Steve Buscemi: What the Critics Are Saying (Video) |date = 2011-12-04 |publisher = The Hollywood Reporter |language = en}}</ref>

=== 執導 ===
布斯美曾經擔任過導演，在1990年代首次執導。他執導過的電影包括：
*《''What Happened to Pete''》（1992年）（短片）
*《{{link-en|伤心树屋|Trees Lounge}}》（1996年）
*《{{link-en|野獸工場|Animal Factory}}》（2000年）
*《{{link-en|孤独的吉姆|Lonesome Jim}}》（2005年）
*《{{link-en|跟蹤大明星|Interview (2007 film)}}》（2007年）

== 私人生活 ==
1987年，與{{link-en|乔·安德瑞斯|Jo Andres}}（Jo Andres）結婚，兩人現育有一子，卢西恩·布西密（Lucian Buscemi）。
