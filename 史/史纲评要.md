《'''史纲评要'''》三十六卷，五十八万余字，是从上古到元代的编年体史评，1613年刊印于南京（癸丑本），次年再版（甲寅本）。癸丑本卷首有“霞漪阁校订”、“明温陵卓吾李贽评纂，新都宁野吴从先参订，武林仙郎何伟然校阅”等字。<ref>{{Cite journal|title=介绍李贽的一部重要著作—明刻本《史纲评要》|author=福建泉州市文物管理委员会|url=|journal=文物|issue=9|doi=|coauthors=厦门大学历史系|year=1974|volume=|page=20-23|pmid=}}</ref>

== 出版 ==
1974年12月，[[中华书局|中华书局]]根据泉州文物管理委员会所藏癸丑本刊印普通平装本三册，特制大字本十册，仿古线装本二函十七册。次年台湾[[大通书局|大通书局]]也刊印了《史纲评要》。2008年9月中华书局再版《史纲评要》，由张友臣作注。张建业主编的《李贽全集注》（2010年5月社会科学文献出版社出版）也收入了《史纲评要》。

== 争议 ==
1974年平装本《出版说明》称：“与《[[史纲要领|史纲要领]]》一书，二者评语有的内容相近，有的则针锋相对，可见其中混杂有他人评语。”<ref>{{Cite book|title=史纲评要.上册|author=李贽评纂|first=|publisher=中华书局|year=1974|isbn=|location=|pages=}}</ref> [[王重民|王重民]]1974年鉴定认为《史纲评要》是托名李贽的伪书。<ref>{{Cite journal|title=王重民教授生平及学术活动年表(附《著述目录》)|author=刘脩业|url=|journal=图书馆学研究|issue=5|doi=|others=|year=1985|volume=|page=43|pmid=}}</ref> “文革”结束后，[[崔文印|崔文印]]（受业于王重民，曾任《史纲评要》责任编辑）、[[王利器|王利器]]力证《史纲评要》系伪托。<ref>{{Cite journal|title=谈《史纲评要》的真伪问题|author=崔文印|url=|journal=文物|issue=8|doi=|others=|year=1977|volume=|page=29-34|pmid=}}</ref><ref>{{Cite journal|title=《史纲评要》是吴从先假李卓吾之名以行|author=王利器|url=|journal=社会科学战线|issue=3|doi=|others=|year=1982|volume=|page=343-345|pmid=}}</ref> [[张岱年|张岱年]]认可崔文印的意见。<ref>{{Cite book|title=中国哲学史史料学|author=张岱年|publisher=三联书店|year=1982|isbn=|location=|pages=172|quote=《史纲评要》，也不是李贽作的。近年崔文印著文考证此书不是李贽的著作。}}</ref> [[郑良树|郑良树]]《续伪书通考》收入崔文印文章。<ref>{{Cite book|title=续伪书通考|author=郑良树|first=|publisher=学生书局|year=1984|isbn=|location=|pages=1183}}</ref> [[陈泗东|陈泗东]]则认为“《史纲评要》基本上是李贽所评纂”。<ref>{{Cite journal|title=《史纲评要》基本上是李贽所评纂|author=陈泗东|url=|journal=泉州文史|issue=5|doi=|others=|year=1981|volume=|page=23|pmid=}}</ref><ref>{{Cite book|title=幸园笔耕录 上|author=陈泗东|first=|publisher=鹭江出版社|year=2003|isbn=|location=厦门|pages=345}}</ref>

== 其他 ==
1974年7月，[[毛泽东|毛泽东]]命人辑录印制大字本《史纲评要》二十三条，亲自选辑，加以删节，[[朱永嘉|朱永嘉]]是办理人之一。<ref>{{Cite book|title=论李贽|chapter=说明：关于毛泽东晚年阅读李贽《史纲评要》|author=朱永嘉|publisher=中国长安出版社|year=2018|isbn=9787510709241|location=|pages=174}}</ref> [[朱永嘉|朱永嘉]]2018年出版《论李贽》，详细解读毛泽东选录部分。陈荣凯发表文章认为朱永嘉跳过《史纲评要》真伪公案，<ref>{{Cite web|url=http://www.takungpao.com.hk/231106/2018/0716/188719.html|title=论李贽——文章异代有知音|accessdate=|author=陈荣凯|date=2018-07-16|publisher=大公报}}</ref> 朱永嘉则回应称这二十三条与《史纲评要》真伪关系不大。
