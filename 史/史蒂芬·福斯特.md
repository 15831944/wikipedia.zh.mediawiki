'''史蒂芬·柯林斯·福斯特'''（{{lang-en|Stephen Collins Foster}}，{{bd|1826年|7月4日|1864年|1月13日|catIdx=F}}）是[[美國|美國]]19世紀[[鄉謠|鄉謠]]、[[民歌|民歌]]作家，產量豐富。

== 生平 ==
福斯特生於美國[[賓夕凡尼亞州|賓夕凡尼亞州]]的中產家庭，出生當天正好是[[美國獨立日|美國獨立日]]。他從來沒受過正統音樂訓練，於16歲時作成了第一首歌曲《戀人請打開窗》，18歲時正式踏入作-{曲}-生涯，但一直都沒有太好的際遇，後來更因為喜歡喝酒，於37歲時因喝醉了，在[[紐約市|紐約市]]一家旅館中不幸跌倒，失救致死。

在他死後，在他的皮包中，發現一些尚未發表的作品手稿，當中有一首《Beautiful Dreamer》就是福斯特死後才被發表的，到現在亦為人所熟悉。

== 家庭 ==
*'''父親'''威廉·巴克萊·福斯特（William Barclay Foster）（1779年9月7日-1855年7月27日）
*'''母親'''愛麗莎·克萊蘭（湯姆林生）·福斯特（Eliza Clayland（Tomlinson）Foster）（1788年1月-1855年1月）
*'''姐妹'''蘇珊娜（Susanna）、愛麗莎（Eliza）與 韓麗葉塔（Henrietta）
*'''兄弟'''詹姆斯（James）（幼年早逝）、（鄧寧·麥克奈爾）Dunning McNair、莫里遜（Morrison）和威廉（William）。

在1850年，福斯特與珍·丹妮·麥克道威爾（Jane（Jeanie）Denny McDowell）結婚。育有一個女兒。

== 作品 ==
他的眾多歌曲中，最為人所熟悉有：

*1848年的《[[哦，苏珊娜|哦，苏珊娜]]》[http://www.youtube.com/watch?v=7PrnUwDWy_o]（許冠傑曾改編成為《有酒今朝醉》）
*1850年的《De Campton Races》[http://www.youtube.com/watch?v=1zPqzT-C4CU]
*1851年《Old Folks at Home[aka Swanee River]》
*1853年《My Old Kentucky Home》[http://www.youtube.com/watch?v=a8tqbXkT1tU]、《Good-Night》
*1854年《肯塔基洲洲歌》、《Jeanie With the Light Brown Hair》
*1856年《Gentle Annie》
*1862年《Beautiful Dreamer》[http://www.youtube.com/watch?v=bCuVZPhqBcE]
*1860年《Old Black Joe》[http://www.youtube.com/watch?v=rDrs2-m7Q0o]
*1865年《The Voices That Are Gone》。

== 其他 ==
他的生平，曾被[[日本|日本]]製成動畫「[[風中少女_金髮珍妮|風中少女 金髮珍妮]]」《風の中の少女 金髪のジェニー》，[[香港無線電視台|香港無線電視台]]亦有播放，名為《歲月飄零》，劇中的配樂則以福斯特的歌曲及黑人[[靈歌|靈歌]]為主。
