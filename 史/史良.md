'''史良'''（{{bd|1900年|3月27日|1985年|9月6日|catIdx=S史}}），女，[[表字|字]]'''存初'''，生於[[江苏省|江苏省]][[常州市|常州市]]，[[中国|中国]]律师、法律家、政治家、社會運動家，[[中华人民共和国司法部|中华人民共和国司法部]]首任部长。她於1942年加入“中国民主政团同盟”（1944年改为[[中国民主同盟|中国民主同盟]]，简称“民盟”），后历任民盟第一届中央常委、副主席，第二、三届中央副主席。1979年10月，当选为民盟第四届中央主席。在担任民盟第五届中央主席期间，于1985年9月6日在[[北京|北京]]逝世，享年85岁。她还曾担任[[全国人大常委会副委员长|全国人大常委会副委员长]]、[[全国政协副主席|全国政协副主席]]等职。

== 早年活动 ==
{{fact|史良極早參與民主政治活动，1919年即參加[[五四运动|五四运动]]，并担任“[[常州|常州]]女师同学会”会长，参加学界联合会。1922年，史良从[[武進女師|武進女師]]畢業後，進入[[上海法政学院|上海法政学院]]学习法律。1925年在[[五卅運動|五卅運動]]中被捕。释放后主编《雪耻》，宣传民族独立，反对列强侵略。1926年在学习就读期间，曾参加学潮，组成护校团，后脱离法政大学进入[[上海法科大学|上海法科大学]]。}}

== 参与政事 ==
{{fact|1927年上海法科大学法律专门部毕业后，史良被配到[[南京|南京]]国民革命军总政治部政治工作人员养成所工作；1930年又出任[[青岛|青岛]]国民党特别市党部训政科主任；1931年后，回[[上海|上海]]开业任律师，是[[上海|上海]]律师公会的执行委员；并加入了[[中共|中共]]在[[上海|上海]]的外围组织“革命人道互济总会”，任该会的律师，办理营救[[邓中夏|邓中夏]]、[[任白戈|任白戈]]、[[方知达|方知达]]等[[中共|中共]]地下党员的多起案件。}}

[[九一八事件|九一八事件]]後，史良则積極參加抗日運動，参与发起组织“[[中国战时儿童保育会|中国战时儿童保育会]]”、“[[妇女界救国会|妇女界救国会]]”、“[[文化界救国会|文化界救国会]]”等民间组织。1936年擔任[[全國各界救國聯合會|全國各界救國聯合會]]常務委員，11月被捕，是「[[七君子之獄|七君子之獄]]」中唯一的女性。[[抗日戰爭|抗日戰爭]]期間，史良擔任[[國民參政會|國民參政會]]參政員，1942年加入[[中国民主同盟|中国民主同盟]]的前身[[中國民主政團同盟|中國民主政團同盟]]，1949年9月21日，以民盟代表的身份參加[[中國人民政治協商會議第一屆全體會議|中國人民政治協商會議第一屆全體會議]]<ref>{{cite news|title=史良：共和国首任司法部长|url=http://news.ifeng.com/history/special/38funvjie/renwuziliao/detail_2010_03/04/360276_0.shtml|accessdate=2011-11-06|newspaper=凤凰网}}</ref>。

== 共和國成立後 ==
新中國成立後，史良擔任第一任[[中央人民政府司法部|司法部]]部長，是當時仅有的两位女性部長之一（另一為[[中华人民共和国卫生部|衛生部]]部長[[李德全|李德全]]）。1957年幫助共產黨「揭發」民盟內「[[右派|右派]]」（發言刊於1957年6月15日《人民日報》等主要報紙上）；1979年任民盟主席。曾历任司法部部长和政务院政治法律委员会委员，[[全国妇联|全国妇联]]执行委员、副主席；第二、三、四届[[全国人大|全国人大]]常委，第五、六届全国人大常委会副委员长；第二、三、四届[[全国政协|全国政协]]常委，第五届[[全国政协|全国政协]]副主席等职<ref>{{cite news|title=著名爱国民主人士史良|url=http://www.js.xinhuanet.com/zhuanlan/2004-07/27/content_2571247.htm|accessdate=2011-11-06|newspaper=新华网|deadurl=yes|archiveurl=https://web.archive.org/web/20070426155232/http://www.js.xinhuanet.com/zhuanlan/2004-07/27/content_2571247.htm|archivedate=2007-04-26}}</ref>。

==个人生活==
在[[抗日战争_(中国)|抗戰]]至1946年前曾與「右派」之一[[羅隆基|羅隆基]]相好，其丈夫為[[陸殿東|陸殿東]]，但並無子女。
