'''史久镛'''（{{bd|1926年|10月9日|||S史}}），生于[[浙江|浙江]][[寧波|寧波]]。中国[[国际法|国际法]][[法学家|法学家]]，曾任[[国际法院|国际法院]]资深法官、院长{{NoteTag|[[国际法院|国际法院]]中排序仅次于院长、副院长的法官称为“资深法官”。}}。是国际法院成立以来首位华人院长<ref>{{cite web |url = http://www.taiwan.cn/twrwk/ywysh/201008/t20100811_1487036.htm |title =史久镛 海牙国际法院的中国院长 |via = 台湾网 |work = 中国青年报 |date=2010-08-11 |deadurl=yes |archiveurl = https://web.archive.org/web/20150519001924/http://www.taiwan.cn/twrwk/ywysh/201008/t20100811_1487036.htm |archivedate = 2015-05-19 }}</ref>。

== 生平 ==
与[[商务印书馆|商务印书馆]]经理[[史久芸|史久芸]]、民国著名银行家[[史久鳌|史久鳌]]、著名石油工程师[[史久光|史久光]]等均为家族中“久”字辈，籍貫浙江宁波鄞县<ref>{{cite book |title=《董浩雲日記: 1948-1982》|author=董浩云  |date=2007年6月1日 |location=香港 |page=315 |isbn=9787108027276 |publisher=生活·读书·新知三联书店}}</ref>。史久镛曾分别在[[上海圣约翰大学|上海圣约翰大学]]和[[美国|美国]][[哥伦比亚大学|哥伦比亚大学]]攻读[[法律|法律]]，1956至1958年间曾在中国多所高等学校的[[国际关系|国际关系]]学院担任教职，1984年受聘为[[外交学院|外交学院]]教授。1980年起史久镛就在众多国际会议与组织中担任过中国政府的法律代表或顾问，曾参与过中[[英国|英]]关于[[香港|香港]]问题的谈判。

=== 国际法院法官 ===
1994年2月6日史久镛被选为国际法院法官，2003年连任，任期至2012年，并曾于2003年至2006年期间出任院长。2010年1月，史久镛提交了辞呈，并于5月28日提前退任法官，已从荷兰海牙回到北京。<ref>[http://www.chinataiwan.org/twrwk/ywysh/201008/t20100811_1487036.htm 史久镛 海牙国际法院的中国院长]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

史久镛表示，《国际法院规约》没有按地区分配法官名额的规定，但各国代表之间形成了一种“[[君子协定|君子协定]]”：按照[[联合国安全理事会|联合国安全理事会]]理事国的[[联合国区域集团|地区分配]]选举15名法官<ref>[http://shifeike.blog.sohu.com/615864.html 史久镛：华人国际法官不代表中国政府]</ref>。
