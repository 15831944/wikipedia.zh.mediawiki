'''史密斯威森M340PD'''（{{Lang-en|Smith & Wesson Model 340PD}}）是一款由[[美国|美国]]槍械公司[[史密斯&威森|史密斯&威森]]旗下的性能中心以[[史密斯威森M640左輪手槍|史密斯威森M340]]（史密斯威森M640的[[钪|钪]]合金[[機匣|底把]]版本）為藍本研製及生產的5發式“J型[[機匣|底把]]”[[扳機#雙動操作|雙動操作]]式{{Link-en|短槍管左輪手槍|Snubnosed revolver|短槍管}}[[左輪手槍|左輪手槍]]，發射[[.357麥格農|.357 S&W馬格南]]或火力相對較弱的[[.38_Special|.38 S&W特種彈]]這二種[[子彈|子彈]]。

== 設計細節 ==
史密斯威森M340PD的[[機匣|底把]]由[[钪|钪]]增強[[鋁合金|鋁合金]]、[[钛|钛]][[合金|合金]][[弹巢|弹巢]]和耐[[腐蚀|腐蚀]][[钢|钢]]製[[槍管|槍管]]內襯所構成。空槍時重量僅為12盎司（340克），上滿彈以後仍保持在1磅（0.45千克）以下。<ref name="Ramage2006">{{cite book|last=Ramage|first=Ken|title=Handguns 2007|url=https://books.google.com/books?id=aBIFulEGU98C&pg=PA185|date=16-11-2006|publisher=Gun Digest Books|location=Iola, Wisconsin|isbn=0-89689-415-0|page=185}}</ref>

由於無外置式[[擊錘|擊錘]]，它以純[[扳機#雙動操作|雙動操作扳機]]進行射擊。它亦採用了集光HI-VIZ[[红色|红色]][[光導纖維|光纖]][[照準器|準星]]和可調節的V型缺口式[[照門|照門]]。整把左輪手槍表面具有消減防眩眩光的[[磨砂|磨砂]][[黑色|黑色]]外觀，以及磨砂[[灰色|灰色]]的[[弹巢|弹巢]]。

== M&P340 ==
'''M&P340'''是M340的衍生型，是史密斯&威森的M&P手槍系列的一部分。它具有[[物理气相沉积|物理气相沉积]]（PVD）塗層和[[不鏽鋼|不鏽鋼]]彈巢。<ref name="Shideler2010">{{cite book|last=Shideler|first=Dan|title=Guns Illustrated 2011: The Latest Guns, Specs & Prices|url=https://books.google.com/books?id=eT2t8PBVmVMC&pg=PA188|date=20-0802010|publisher=F+W Media, Inc|location=Iola, Wisconsin|isbn=1-4402-1624-X|page=188}}</ref>它裝有XS Sights 24/7[[氚|氚光]]夜間[[照準器|準星]]，重13.8盎司（391.22克，0.86磅）。它可以裝上或是不帶內置保險鎖。<ref>{{Cite web|title = Product: Model M&P340|url = http://www.smith-wesson.com/webapp/wcs/stores/servlet/Product4_750001_750051_765779_-1_757783_757781_757781_ProductDisplayErrorView_Y|website = www.smith-wesson.com|accessdate = 2015-09-25}}</ref>

== 設計限制 ==
禁止使用[[彈丸|彈丸]]重量小於120[[格令|格令]]（7.8克）的[[彈藥|彈藥]]，以免輕質[[拋體|射彈]]過快飛出以後，[[裝藥|裝藥]]仍然在燃燒而造成{{Link-en|火焰切斷 (槍械)|Flame cutting (firearms)|底把侵蝕}}。在用戶手冊以上指出的另一個警告就是其[[後座力|後座力]]有著足夠的力度導致彈巢以內未被擊發的彈藥的彈殼後移導致彈丸移位，進而引致彈巢卡死。還有評論指出，在這些超輕型左輪手槍類型當中，因為該槍的槍管其實是鋼製內襯管而非單一實心的固體鋼製件，以致[[準確與精密|準確度]]下降。<ref>Ayoob, Massad. (2002). ''The Gun Digest Book of Combat Handgunnery'', 5th edition: Krause Publications. {{ISBN|0-87349-485-7}}</ref>
