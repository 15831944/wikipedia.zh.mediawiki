'''史密斯威森M59'''（{{Lang-en|'''Smith & Wesson Model 59'''}}）是一款由[[美国|美国]][[槍械|槍械]]公司[[史密斯&威森|史密斯&威森]]所研製與生產以後於1971年推出的[[扳機|雙動操作]]式[[半自動手槍|半自動手槍]]；它的設計開發基於先前的[[史密斯威森M39手槍|史密斯威森M39手槍]]（S&W M39），並且採用了更寬闊的槍身和直形握把背護片以改進為可對應容量達到14發的雙排[[彈匣|彈匣]]的手槍<ref name="EPR2002">{{cite book| last = Hartink| first = A.E.| title = The Complete Encyclopedia of Pistols and Revolvers| publisher = Chartwell Books, Inc| year = 2002| pages = 87–88| location = Edison, New Jersey| isbn = 978-0-7858-1519-8}}</ref>；而且與S&W M39一樣，發射[[9×19mm魯格彈|9×19毫米]][[手枪|手枪]][[子彈|子彈]]。

== 歷史和用戶 ==
史密斯威森M59最初就為[[美國海軍|美國海軍]]設計，用作Mk 22 Mod 0「靜寂的小狗」（Hush Puppy）[[抑制器|消聲]]手槍的原型史密斯威森M39的加大容量版本。1965年，美國海軍委託了史密斯威森，研製史密斯威森M39的可使用[[白朗寧大威力手槍|白朗寧大威力手槍]]的13發可拆卸式雙排[[彈匣|彈匣]]版本。1970年初，史密斯威森生產了十多把實驗型全[[不鏽鋼|不鏽鋼]]大容量原型槍，並且提供給海軍[[海豹部隊|海豹部隊]]隊員進行現場實戰評估。但結果該槍並無獲得採用。

1971年，史密斯威森M59在民用槍械市場上市。<ref name=Trzoniec/><ref name=Hartink/>直到十年以後的1980年代，隨著史密斯威森推出其改進過的第二代半自動手槍系列（如以下的史密斯威森M459）而停產。

== 設計 ==
史密斯威森M59採用[[9×19mm魯格彈|9×19毫米]][[口徑|口徑]]生型，具有加大寬度的[[陽極處理|陽極]][[氧化|氧化]][[鋁合金|鋁合金]]製[[機匣|底把]]（可容納雙排彈匣）、直形握把背護片、彈匣斷路裝置（除非彈匣到位，否則手槍無法開起火）以及裝有手動保險的烤藍[[碳鋼|碳鋼]][[手槍套筒|套筒]]。三片式握把是由兩片[[尼龙|尼龙]]製[[塑料|塑料]]板所製成，並且與[[金属|金属]]背護片連接。它採用了與[[M1911手槍|M1911A1]]類似的、位於[[扳機護環|扳機護圈]]後部的彈匣釋放按鈕。

== 彈匣 ==
史密斯威森M59的可拆卸式彈匣採用了與[[白朗寧大威力手槍|白朗寧大威力手槍]]的彈匣類似設計。最初推出時的彈匣容彈量為14發，後來進一步推出的衍生型將容量增加為15發。史密斯威森亦生產了其延長型20發版本雙排彈匣。其他生產商中亦有許多槍械採用了M59所用的彈匣，包括{{Link-en|馬林營用卡賓槍|Marlin Camp Carbine}}，[[大宇K5手槍|大宇K5手槍]]以及{{Link-en|Kel-Tec|Kel-Tec}} {{Link-en|Kel-Tec P11手槍|Kel-Tec P-11|P-11}}、[[Kel-Tec_SUB-2000卡賓槍|SUB-9和SUB-2000]]。

== M459 ==
史密斯威森M459是史密斯威森M59的的更新型版本，配有可調節式瞄準具和方格式尼龍握把。該型號在1988年停產。當中有803把根據[[联邦调查局|联邦调查局]]規格生產，採用{{Link-en|拉絲金屬|Brushed metal|拉絲}}表面處理並且裝有特殊握把。
