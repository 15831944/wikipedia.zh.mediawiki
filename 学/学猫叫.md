《'''学猫叫'''》是一首由[[小潘潘_(歌手)|小潘潘]]和[[小峰峰|小峰峰]]共同制作及演唱的[[爱情歌曲|爱情歌曲]]，在[[抖音短视频|抖音]]等短视频平台作为一些视频的[[背景音乐|背景音乐]]后开始流行传唱。<ref>{{cite web |title=主播小潘潘《学猫叫》 引发全民卖萌热潮 |url=http://ent.people.com.cn/n1/2018/0709/c1012-30135931.html |website=北方网 |accessdate=2018-10-22 |date=2018-07-09}}</ref><ref>{{cite web |author1=劉芝宇 |title=暴紅歌曲「學貓叫」 手勢舞襲捲網路 |url=http://gotv.ctitv.com.tw/2018/08/955586.htm |website=中天快点TV |publisher=中天新聞 |accessdate=2018-12-06 |language=zh-TW |date=2018-08-28}}</ref>此外，网络上也有其它语言，如日语版的翻唱。<ref>{{cite web|url=http://it.gmw.cn/2018-10/16/content_31743745.htm|title=TikTok登上日本王牌新闻节目：受各个年龄段用户喜爱|publisher=光明网|date=2018-10-16|accessdate=2019-1-12}}</ref>

==批评==
有评论认为，《学猫叫》是一首毫无营养的口水歌，符合旋律简明、节奏强烈、无意义的歌词、重复等[[网络神曲|网络神曲]]的特点。<ref>{{cite web |author1=剁椒娱投 |title=除了制造神曲，抖音还能给音乐产业留下什么？
