'''学政试院'''又称'''扬郡试院'''，位于[[江苏省|江苏省]][[泰州市|泰州市]][[海陵区|海陵区]]府前路和鼓楼北路交叉口西北侧、府前路2号，西侧原为泰州州衙（今泰州市海陵区政府），其旧址曾为南唐永宁宫和明代都察院、提督军务巡抚凤阳都御史衙门（凤阳巡抚衙门）等，清代[[康熙|康熙]]时（具体时间不详，约在康熙二十五年之前）改为[[扬州府|扬州府]]所属州县<ref group="注">扬州府一度辖三州七县，雍正后演变为二州六县，即泰州、高邮州、江都、甘泉、仪征、宝应、兴化和东台，因泰州地处扬州府所属州县的适中区位，故自明万历四十年起时院试即在泰州举行，但最初场所不在该处。</ref>的[[院试|院试]]场所，院试由[[提督江苏学政|提督江苏学政]]主持，除两次在[[仪征|仪征]]、一次在[[江阴|江阴]]外，其他均在泰州举行<ref name="历史">{{cite web |url=http://sz.tznews.cn/tzwb/html/2011-12/11/content_4555.htm|title=《泰州学政试院旧事寻踪》|accessdate=2012-12-17|work=汪维寅|publisher=泰州晚报}}</ref>。

该建筑是江苏省内仅存的较为完整的院试试院，原布局为大门前以照壁和东西辕门、东西吹鼓亭围合成广场，其后中轴线上依次为大门五间、仪门五间、大堂五间、二堂（思补堂）五间、内正门五间、正房五间（嘉庆年间因火灾将原楼房改建为平房），考棚设在仪门和大堂之间的院子两侧，另有东西楼、巡房、福神祠、幕舍和厨舍等其他辅助用房<ref name="历史" />。[[科举|科举]]废止后曾一度作为泰州机关团体的办公场所，原建筑只存大门和二堂，泰州市政府于2005年5月至2006年初投资约1500万元、参考续纂《泰州志》上的扬郡试院图进行了修缮复原<ref>{{cite web |url=http://www.xinhuanet.com/chinanews/2005-03/14/content_3870018.htm|title=《计划投资1500万元 泰州将修缮复原“扬郡试院”》|accessdate=2012-12-17|work=|publisher=新华网}}</ref>，2008年8月作为展示科举程序和沿革等内容的'''泰州科举院试博物馆'''正式对外开放<ref>{{cite web |url=http://js.xhby.net/system/2008/08/01/010308841.shtml|title=《泰州清代学政试院开门迎客》|accessdate=2012-12-17|work=|publisher=新华报业网}}</ref>。
