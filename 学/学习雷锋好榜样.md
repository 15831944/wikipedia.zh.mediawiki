'''学习雷锋好榜样'''，由[[吴洪源|洪源]]作词，[[生茂|生茂]]作曲的一首歌颂[[雷锋|雷锋]]，号召大家向雷锋学习的歌曲，创作于1963年3月5日，创作当日由[[北京军区战友文工团|北京军区战友文工团]]在北京[[天安门|天安门]]前首唱。该歌曲在中国大陆有很高的传唱度，特别是每年3月5日“学雷锋纪念日”活动多会使用<ref>{{cite web |author1=解辰巽 |title=弘扬雷锋精神 榜样再度唱响(图) |url=http://news.163.com/12/0301/01/7RFNFDA400014AED.html |website=网易 |publisher=北京晨报}}</ref>。该歌曲的传唱极大地推动了“学习雷锋活动”的发展<ref name="yishi"/>。

== 背景 ==
1962年8月15日，[[中国人民解放军|中国人民解放军]][[沈阳军区|沈阳军区]]工程兵第10团的中士[[雷锋|雷锋]]因公殉职。雷锋早在1960年11月就被沈阳军区树为学习《[[毛泽东选集|毛泽东选集]]》的标兵进行宣传。1963年3月2日，《[[中国青年|中国青年]]》杂志在[[学雷锋|学习雷锋]]专辑中刊登了[[毛泽东|毛泽东]]、[[周恩来|周恩来]]的题词，[[董必武|董必武]]、[[郭沫若|郭沫若]]、[[罗瑞卿|罗瑞卿]]、[[谢觉哉|谢觉哉]]等写的诗和文章<ref>{{Cite book|title=毛泽东题词背后的故事|last=周炳钦编著|first=|publisher=杭州：浙江人民出版社|year=2015.01|isbn=978-7-213-06406-7|location=|pages=261}}</ref><ref>{{Cite book|title=宣传雷锋轶事|last=本书编写组编|first=|publisher=沈阳：白山出版社|year=1997|isbn=7-80566-585-0|location=|pages=20-22}}</ref>。1963年3月5日，毛泽东的题词“向雷锋同志学习”在《[[人民日报|人民日报]]》《[[解放军报|解放军报]]》等媒体刊登，[[学雷锋活动|学雷锋活动]]在全国范围内展开，“向雷锋同志学习”逐渐成为树立社会主义新风尚的动员口号<ref name=rmzxb>{{cite web |author1=孟素 |title=《学习雷锋好榜样》的诞生 |url=http://www.rmzxb.com.cn/c/2016-03-06/723460.shtml |website=人民政协网}}</ref>。

== 创作 ==
1963年3月5日上午，也就是毛泽东题词在《解放军报》发表当天的上午，[[北京军区战友文工团|北京军区战友文工团]]团长[[晨耕|晨耕]]、政委[[王引龙|王引龙]]召集全体成员学习毛主席等领导题词，并通知当日下午2点在[[天安门|天安门]]有游行活动，全团参加宣传。接近上午10点钟时散会，有人提议作为文工团在下午游行应该有首歌，提议得到一致赞同，并决定由作曲家[[生茂|生茂]]和词作家[[吴洪源|吴洪源]]创作。<ref name="ci2"/><ref name=rmzxb/>

决定后，洪源找出《[[雷锋日记|雷锋日记]]》和刊载雷锋事迹的报纸、资料，并决定将“好榜样”三个字定为歌曲主题，用两个小时创作了《学习雷锋好榜样》歌词，交给生茂谱曲<ref name="ci2">{{cite web |author1=刘超群 |title=词作者揭“学习雷锋好榜样”创作过程 ：只用了2小时 |url=http://ent.ifeng.com/a/20120305/12978208_0.shtml |website=凤凰娱乐-每日新报}}</ref>。生茂则一直在等洪源的歌词，连中午饭都没顾上吃<ref>{{cite web |author1=石祥 |title=“词坛名家”洪源：年轻的老兵 永恒的歌 |url=http://www.81.cn/ysj/2015-08/09/content_6613431.htm |website=中国军网 |publisher=解放军报}}</ref>，拿到歌词后生茂不到一个小时便完成了创作。歌曲完成后，他们让人抄在文工团食堂门前的黑板上征求意见，得到了大家的一致好评<ref>{{cite web |title=“雷锋之歌”的来历 |url=http://ent.163.com/13/0306/00/8P8C54PE00032DGD.html |website=网易-华商报}}</ref>。

1963年3月5日下午，战友文工团成员们在[[手风琴|手风琴]]的伴奏下，在天安门前合唱了《学习雷锋好榜样》<ref name="yishi">{{cite web |title=《学习雷锋好榜样》轶事 |url=http://news.163.com/14/1121/10/ABIOU4I200014AED.html |website=网易 |publisher=中老年时报(天津)}}</ref>。

== 传唱 ==
据词作者洪源回忆，1963年3月5日下午上街宣传演唱《学习雷锋好榜样》时就有很多群众要歌词，不久就有电台请他们去教歌<ref>{{cite web |title=词作者吴洪源：两小时写就《学习雷锋好榜样》 |url=http://news.cntv.cn/2013/03/10/ARTI1362871642867540.shtml |website=央视网-中新网}}</ref>，“过了四五天，《人民日报》、《红旗》杂志就发表了这首歌曲，一时间下至农村上至城镇，几乎全国都在唱这首歌<ref name="ci2"/>。”

1963年10月，《学习雷锋好榜样》词曲在《解放军歌曲》上发表，接着战友文工团首唱的录音，由中央及各地广播电台向国内外播放<ref name="yishi"/>。1965年3月5日，[[八一电影制片厂|八一电影制片厂]]出品，[[董兆琪|董兆琪]]执导的黑白电影《[[雷锋_(1965年电影)|雷锋]]》开始在中国各地城乡放映，《学习雷锋好榜样》为该电影的片尾曲<ref>{{cite web |title=雷锋精神的时代印象——践行社会主义核心价值观 |url=http://politics.people.com.cn/n/2014/0224/c70731-24450426.html |website=人民网 |publisher=浙江日报}}</ref>。

== 获奖 ==
1964年5月，合唱歌曲《学习雷锋好榜样》在中国人民解放军全军第三届文艺汇演中荣获“优秀奖”<ref name="yishi"/><ref>{{cite web |title=学雷锋好榜样歌曲 唱雷锋学雷锋 |url=https://edu.pcbaby.com.cn/342/3427226.html |website=PCGROUP}}</ref>。1989年，《学习雷锋好榜样》获得建国40周年“唤起我美好回忆的那些歌”优秀作品奖<ref name="yishi"/>。

== 轶事 ==
1966年，周恩来总理到大连视察时，曾亲自向宾馆服务员和文工团演员推荐并教唱《学习雷锋好榜样》<ref name="yishi"/>。
