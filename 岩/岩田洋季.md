'''岩田 洋季'''（{{bd|1983年|12月13日}}）是[[日本|日本]]的[[輕小說|輕小說]]作家，[[廣島縣|廣島縣]]出身。

== 簡介 ==
* 曾在2001年參加第八屆[[電擊小說大獎|電擊遊戲小說大獎]]，但在最後輪選時被淘汰而未有得獎。
* 在2002年7月時以作品《'''灰色愛麗斯'''》出道，並在[[電擊文庫|電擊文庫]]展開連載。由於當時尚未成年，而被譽為天才。
* 在2003年，由電擊文庫出版代表作《'''[[獻上女神的祝福！|獻上女神的祝福！]]'''》，作品在2006年被製成全24話的電視動畫。
* 在《灰色愛麗斯》完結後，由電擊文庫出版另一作品《'''月之盾'''》。
* 在2008年，由電擊文庫出版作品《{{lang|ja|'''めしあのいちにち'''}}》。
* 在2010年推出作品《'''[[花×華|花×華]]'''》。

《'''花×華'''》（{{lang-ja|花×華}}）是岩田洋季撰寫，[[涼香|涼香]]插畫的日本[[輕小說|輕小說]]作品，[[ASCII_Media_Works|ASCII Media Works]]的[[電擊文庫|電擊文庫]]出版發行。2010年3月10日發售第1集，同年9月10日發表第2集。繁體中文版由台灣角川代理，原本預計2011年3月底出，但後來改為4月29日發售<ref>{{Cite web |url=http://www.kadokawa.com.tw/details.asp?id=6737   |title=台灣角川 KADOKAWA 花×華 01 岩田洋季  |language=zh }}</ref>。作品是描寫高中男女學生[[三角戀情|三角戀情]]，着重描寫「青春」<ref>根據岩田洋季在《花×華》後記所言。</ref>的戀愛喜劇。

====故事簡介====
普通高中生園端夕在升上高二前的春假時，收到一封署名為「Hana」的情書，而在他的身邊卻有兩名同樣名為「Hana」的女孩，一個是受人愛慕的千金小姐「華公主」東雲華，而另一個則是擅長運動，活潑好動的成宮花<ref>華與花的日文發音同為「Hana」</ref>。兩名個性南轅北轍的少女同時向夕告白，並開始熱烈追求夕，一起加入夕所屬的電影研究會，也讓夕決定開拍以兩人為主角的電影。

====登場人物====
===== 主要人物 =====
;{{nihongo|園端 夕|園端 夕（そのはな ゆう）}}
:本作主角，血型是B型。美櫻高中2年D班的男學生。成績不怎麼樣，運動神經非常差，講話有點毒，而且是一名「愛哭鬼」。和母親兩人一起住在美櫻車站旁邊的公寓的707號房。已離世的父親「園端朝地」是有名的電影導演。自幼受父親創作的電影「美櫻」影響，十分喜愛電影，在中學時常拿着攝影機拍攝影像。目前隸屬於父親在學生時代創立的電影研究會。
: 在負責拍攝戲劇社「灰姑娘」的公演紀錄影片時初次認識劇社主角東雲華，又在拍攝女子籃球社的冬季選拔預賽紀錄時初次認識非籃球社正式社員的成宮花。
: 喜歡中華料理，特別是「東佳苑」的中華拉麵。
: 為了吸引新生加入影研會，擔任導演拍攝短片，決定以「杜鵑絕景」作主題，並同時起用華及花一起當女主角。在進行影研會的拍攝時，加深了對華及花的認識，發現自己對兩人都有好感。其後從花的表白中知道了華與花的事，及情信的真相，並向牡丹駿平宣言要證明華及花的演出與血緣無關。由於杜鵑花園被封鎖，於是與影研會的同伴在夜裏偷偷潛入花園進行拍攝。完成品獲得蓮井福助的認同。

;{{nihongo|東雲 華|東雲 華（しののめ はな）}} 
:大企業「東雲造船」千金，學校的模範生，在開學典禮上代表新生上臺演講，號稱校內第一大美女，老師、同學都以「華大人」稱呼她。雖然家境富裕，卻不會因此具有優越感，進退應對相當得體。小時候開始上演戲課程，是演劇社的重要人物，在高中1年級戲劇社「灰姑娘」的公演中擔任女主角，完美詮釋了灰姑娘，支配了整個舞臺，讓全場觀眾起立喝采。然而由於影研社陷入被廢部的危機，決定離開演劇社加入影研社。
:和花自小相識，兩人同在十六年前的十月四日早上十一點十四分在末松婦產科出生<ref name="1a">岩田洋季(2011)，《花×華》，台灣角川</ref>。兩人事實上是姊妹，雖然自稱十分討厭花，但實際上非常在意花。
:身材嬌小，身高147公分<ref name="1b">岩田洋季（2011），《花×華》彩頁，台灣角川</ref>，面容如同袖珍娃娃屋的可愛人偶，給人清秀高雅的感覺。於升上高二前的春假最後一天時，在百合峰瞭望台向夕告白，亦是情信的作者<ref>根據《花×華》花對夕的告白所言。</ref>。然而，因為知道花也同樣喜歡夕，不願「不戰而勝」而故意使用「Hana」作為署名。高中1年級時就讀1年A班，在升上高中2年級進入2年D班，成為夕的同班同學。在班會上直接向導師申請坐在夕的隔鄰，並拚命的在夕面前顯示自己的優點。
: 平日十分文靜優雅，但在與花爭奪夕時卻會露出真實強悍的一面。得知花曾單獨到夕家中時感到不快，同樣稱呼夕為「夕同學」。
: 由於牡丹駿平將「美櫻」留在東雲家，因此自小受到「美櫻」的影響，立志成為像女主角般的人。
: 在杜鵑花園被封鎖後，向夕告知自身及花的事。在得知夕為了自己及花而向牡丹駿平宣言後，與華一起決定全力以赴，投入全部感情。

;{{nihongo|成宮 花|成宮 花（なるみや はな）}}
:升上高中2年級後和夕變成同班同學的少女。平日個性開朗活潑，實際上很易害羞，不太會讀書，但運動神經十分出色，跑五十公尺只需六秒五七<ref name="1a"/>。常去各個運動社團支援比賽，不過沒有加入任何社團。在去年校慶時舉辦的「校內人氣大賽」上獲得第三名<ref name="1a"/>。
:和華自小相識，兩人同在十六年前的十月四日早上十一點十四分在末松婦產科出生<ref name="1a"/>。兩人事實上是姊妹，在缺乏自信的同時十分討厭華。
: 家位於商店街，家裡經營名為「中華食堂 柏林」的中華料理店。成宮家有花、雙親、祖母、3個弟弟及1個妹妹，共計8人。
: 身高約156公分<ref name="1b"/>，有着用電棒卷卷過的雙馬尾，宛如模特兒的可愛臉容，散發出充滿活力的存在感。得知華有喜歡的人後進行調查，不知不覺間同樣喜歡上夕。在百合峰瞭望台華向夕告白之際，突然闖入並向夕告白。
: 在升上高中2年級進入2年D班，成為夕的同班同學。為了坐在夕的隔鄰而強行要求和武志交換位置，並拚命的在夕面前顯示自己的優點。
: 雖然加入了影研社，但因為親生父親的關係，本身很討厭演戲。不過在和夕交談過後，決定出演影研社作品的女主角，並自此稱呼夕為「夕同學」。可是，由於太在意旁人的眼光而導致全身僵硬，因此拍攝並不順利。
: 在杜鵑花園被封鎖後，與牡丹駿平爭吵後連續數天沒有上學，向趕來探望的夕表白。在得知夕為了自己及華而向牡丹駿平宣言後，與華一起決定全力以赴，投入全部感情。

:華的母親。華不太喜歡她。一頭充滿光澤的秀髮，苗條得不尋常的身材，外表看來像二十多歲卻又帶點蒼老感。據說在生下孩子後，行為變得怪異。為了牡丹駿平而將「杜鵑花園」封鎖。

;{{nihongo|牡丹 駿平|牡丹 駿平（ぼたん しゅんぺい）}}
:美櫻町出身，長相俊美，頭髮很長，皮膚很白，兼具陰與陽的氣息，具有獨特魅力的的著名舞臺劇演員。美櫻高中畢業生，在學時和園端朝地、相澤櫻一起加入影研會，討厭朝地但喜歡櫻。是華和花的親生父親。

;{{nihongo|城本老師|城本先生（しろもとせんせい）}}
: 2年D班的班導，男性英語教師。因為身材肥胖而被男學生們昵稱他為「伊比利城本（{{lang|ja|イベリー城本}}）」。

;{{nihongo|月野老師|月野先生（つきのせんせい）}}
:影研會顧問，準備退休，是美櫻高中資歷最深的教師。

====名詞解釋====
;{{nihongo|美櫻鎮|美桜町（びおうちょう）}}
:人口約七萬人，從縣廳乘JR普通列車約需48分鐘車程。城鎮位處於百合峰和太平洋中間，接近海邊，四季景色千變萬化，擁有觀光名勝「美櫻十二景」，是知名的花都。城鎮被國道分為東西兩側，東側是沿海平地，也是城鎮繁忙地區集中地，西側是百合峰，山坡上有着復古風情的房舍。

;{{nihongo|百合峰|百合ヶ峰（ゆりがみね）}}
:位於美櫻町，標高212公尺的山。附近有能夠將美櫻鎮與太平洋一覽無遺的「百合峰展望台」以及東雲家管理的「百合峰展望公園」。在櫻花盛放時，瞭望台將有如「飄浮在櫻花橋上」，被稱為「櫻花陸橋」<ref name="1a"/>。

;{{nihongo|美櫻高中|美桜高校（びおうこうこう）}}
:夕等人就讀的高中。夕的父親朝地和牡丹駿平等人也是美櫻高中出身。

;{{nihongo|電影研究會|映像研究会}} 
: 一般稱為「影研會（{{lang|ja|映像研}}）」，由園端朝地創立的社團。在花與華加入前，只有社長知佳和夕兩名成員，由於學校規定社員人數必須五人以上，所以一度陷入被廢社的危機。

;{{nihongo|電影社|映画部}} 
: 由於過去影研會太散漫，於是部分新生創立了「新．影研會」，即現在的電影社。現任社長是蓮井福助。

;{{nihongo|「美櫻」|「美桜」}}
:在二十年前，園端朝地高三時和影研會成員一起花了一年時間創作的無聲電影，沒有任何配樂和旁白，女主角是夕的母親。電影充滿了實驗性、藝術性的影片，和朝地日後作品的風格有點不同。作品以「櫻花陸橋」、「杜鵑絕景」、「向日葵街道」、「黃昏坡道」等美櫻鎮的著名美景為舞臺，故事從女主角寄情書給心儀的男孩子開始，是一部記錄了隨四季改變氛圍的電影。

;{{nihongo|中華食堂伯林|中華食堂 伯林（ちゅうかしょくどう べるりん）}}
:成宮家經營的中華料理店。店名由來是母方的姓「林田」，在開店時想取跟林字有關的店名而叫伯林，和德國首都無關。

;{{nihongo|Toy Box|トイボックス}}
:夕的母親經營的酒館。知佳在酒館裡打工，化名為「日向向日葵」。
