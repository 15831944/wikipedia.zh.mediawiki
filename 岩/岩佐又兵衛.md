'''岩佐又兵衛'''（{{jpn|j=岩佐 又兵衛|hg=いわさ またべえ|rm=Iwasa Matabē|f=h}}，[[天正_(日本)|天正]]6年（1578年） - [[慶安_(後光明天皇)|慶安]]3年六月二十二日（1650年7月20日））是[[江户时代|江户时代]]初期[[畫師|畫師]]。“又兵衛”是[[表字|表字]]也是一般稱呼，[[本名|本名]]為“{{lang|ja|{{ruby|勝以|かつもち}}}}”，[[号|号]]“{{lang|ja|{{ruby|道蘊|どうおん}}}}”，其餘還有{{lang|ja|岩佐光輔}}、{{lang|ja|憂世又兵衛}}、{{lang|ja|浮世又兵衛}}等名。<ref name="tukuba_thesaurus">{{cite web |url=https://www.tulips.tsukuba.ac.jp/jart/mokuji/2ismb001.html|title=2ismb001 岩佐又兵衛|publisher=筑波大学日本美術シソーラスデータベース作成委員会|access-date=2017-11-19}}</ref>又称“{{lang|ja|{{ruby|吃|ども}}の{{ruby|又平|またへい}}}}”（意為“[[口吃|口吃]]的又平”）。後人稱之為“{{lang|ja|浮世絵の元祖}}”<ref>在福井市松本三丁目興宗寺中的岩佐又兵衛勝以墓旁邊，立有一篇介紹，上面第一句為“{{lang|ja|『山中常磐絵巻』等、数々の名作を遺し、浮世絵の元祖とも言われる岩佐又兵衛勝以は、元和元年（１６１５）頃から２０年余り福井に住んだ。}}” 翻譯：“有《山中常磐繪卷》等諸多名作流傳，被稱為“浮世繪之元祖”的岩佐又兵衛勝以，於元和元年（1615年）始20餘年在福井居住。”</ref><ref>{{cite web |url=http://www2.kokugakuin.ac.jp/zyokoym/iwasam.html|title=岩佐又兵衛は、浮世絵の元祖として蘇るか|author=横山実|access-date=2017-11-19}}</ref>（浮世繪之元祖，意為比“祖”更早的發祥者），而真正確立[[浮世繪|浮世繪]]這一藝術形式的畫師是後來的[[菱川師宣|菱川師宣]]，人稱“{{lang|ja|浮世絵の祖}}”（浮世繪之祖，開山鼻祖）。
== 經歷 ==
又兵衛是[[攝津國|攝津國]][[川邊郡|川邊郡]]{{link-ja|有岡城|伊丹城}}[[荒木村重|荒木村重]]的小兒子<ref name="岩佐家譜">記自『岩佐家譜』（[[享保|享保]]4年（[[1731年|1731年]]））。辻惟雄『日本美術絵画全集 第十三巻 岩佐又兵衛』（集英社，1980年。翌年普及版）及『岩佐派のゆくえ』展図録（福井県立美術館，1998年）中收錄。</ref><ref>黒川道祐『遠碧軒記』（『日本随筆大成 第一期第十巻』所收，吉川弘文館，1975年。新装版1993年 ISBN 4-642-09010-X。2007年 ISBN 978-4-642-04076-1）</ref>，出生第二年即天正7年，[[有岡城之戰|有岡城之戰]]結束，有岡城被攻陷之後荒木家族幾乎被殺盡，兩歲的又兵衛被乳母救出，可能是在[[石山本願寺|石山本願寺]]受到庇護。<ref>《岩佐家譜》中記載為“[[西本願寺|西本願寺]]”，但当時還沒有西本願寺，推測為石山本願寺。</ref>

據說成人後又兵衛以母方的岩佐為姓<ref name="岩佐家譜" /><ref>雖然他的母親被認為是[[荒木夫人|荒木夫人]]“{{lang|ja|だし}}”，但她很可能是川那部氏，如何都不可能姓岩佐。堀直格『扶桑名画伝』把岩佐當作是乳母的姓氏，而畠山浩一意識到可能是土佐派的關係（「岩佐又兵衛と荒木一族」，東北大学大学院文学研究科美術史学講座 『美術史学』30号所收，2009年）。</ref>，在[[織田信長|織田信長]]之子[[織田信雄|織田信雄]]手下當[[小姓|近習小姓]]，可能是集文藝、繪畫諸藝於一身的[[御伽眾|御伽眾]]一樣的存在。織田信雄{{link-ja|改易|改易}}之後，成為[[浪人|浪人]]的又兵衛自稱“勝以”，開始了在京都的畫師生涯。<ref>又兵衛自寫的紀行文《廻国通之記》，把京都稱為“古郷”，寫下“{{lang|ja|わかく盛んなる時、みやこに久しく住なれ}}”。</ref>

他曾在京都、福井和江戶居住過，死後葬在福井{{link-ja|興宗寺|興宗寺_(福井市)}}。[[1948年福井地震|福井大地震]]後寺廟移走，1987年墓又移進興宗寺，而原址留下石碑。

== 繪畫風格 ==
又兵衛據說曾師從其父荒木村重舊時的{{link-ja|家臣|家臣団}}{{link-ja|狩野内膳|狩野内膳}}。狩野内膳是可以於[[俵屋宗達|俵屋宗達]]齊名的江戶初期[[大和繪|大和繪]]畫師，他吸收包括[[牧谿|牧谿]]、[[梁楷|梁楷]]的中國[[水墨画|水墨画]]和日本的[[狩野派|狩野派]]、[[海北友松|海北派]]、[[土佐派|土佐派]]等多方風格形成自己的流派。也可能效法了{{link-ja|土佐光則|土佐光則}}的繪畫風格。<ref name="tukuba_thesaurus"/>又兵衛的特色表現在畫人物上，他畫的人物魁梧，強調失去平衡的極端動態，人物臉頰豐滿、長下巴，是所謂“{{lang|ja|{{ruby|豊頬長頤|ほうぎょうちょうい}}}}”。
