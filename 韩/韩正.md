'''韩正'''{{BD|1954年|4月|catIdx=Han韩正}}，籍貫[[浙江省|浙江]][[慈溪市|慈溪]]，生于[[上海市|上海]]，[[中国共产党|中国共产党]]和[[中华人民共和国|中华人民共和国]]主要领导人之一，中共第十九届[[中国共产党中央政治局常务委员会|中央政治局常务委员会]]委员、[[中华人民共和国国务院副总理|国务院副总理]]（排名第一）。现任[[中央全面深化改革委员会|中央全面深化改革委员会]]副主任、[[中共中央港澳工作协调小组|中共中央港澳工作协调小组]]组长、[[推进“一带一路”建设工作领导小组|推进“一带一路”建设工作领导小组]]組長、[[中华人民共和国国务院|国务院]][[粤港澳大湾区建设领导小组|粤港澳大湾区建设领导小组]]组长。曾任[[中国共产主义青年团|共青团]]上海市委书记、[[卢湾区|卢湾区]]区长、[[上海市人民政府|上海市人民政府]][[上海市行政长官列表|市长]]、[[中共上海市委|中共上海市委]]书记。

韩正于1975年12月开始参加工作，1979年5月加入[[中国共产党|中国共产党]]。[[复旦大学|复旦大学]]大专班毕业，[[华东师范大学|华东师范大学]]夜大学政教系政教专业毕业，[[华东师范大学|华东师范大学]]国际关系与世界经济专业经济学硕士。[[中国共产党第十六届中央委员会委员列表|第十六]]、[[中国共产党第十七届中央委员会委员列表|十七]]、[[中国共产党第十八届中央委员会委员列表|十八]]、[[中国共产党第十九届中央委员会委员列表|十九届中共中央委员]]，第十八届、十九届[[中国共产党中央政治局|中共中央政治局]]委员，第十九届[[中国共产党中央政治局常务委员会|中共中央政治局常委]]<ref>{{cite web|url=http://district.ce.cn/newarea/sddy/201705/12/t20170512_22780562.shtml|title=上海新一届市委常委简历(书记韩正 副书记应勇、尹弘)|publisher=中国经济网|accessdate=2017-05-13}}</ref>。

==生平==
=== 早年经历 ===
1975年12月，在上海市[[徐汇区|徐汇区]]任职超重安装队仓库管理员，后调任供销股办事员、[[中国共产主义青年团|共青团]]总支部副书记。1980年8月，任上海市化工装备工业公司干事、团委负责人。1982年11月，任[[上海市化工局|上海市化工局]]团委书记。1986年2月，任[[上海市化工专科学校|上海市化工专科学校]]党委副书记。1987年5月，任[[上海胶鞋厂|上海胶鞋厂]]副厂长、党委书记。1988年8月，任[[上海大中华橡胶厂|上海大中华橡胶厂]]副厂长、党委书记，期间时任上海市长[[朱镕基|朱镕基]]曾前往考查，并对韩正的工作予以了肯定<ref name="jianli">{{cite web |url = http://sh.xinhuanet.com/2012-05/22/c_131604326.htm |title = 韩正同志简历 |publisher = 新华网 |date = 2012-05-22 |accessdate = 2013-09-06 |deadurl = yes |archiveurl = https://web.archive.org/web/20140104232502/http://sh.xinhuanet.com/2012-05/22/c_131604326.htm |archivedate = 2014-01-04 }}</ref><ref name="zhurongji">{{cite web |url = http://www.yicai.com/news/2013/08/2939288.html |title = 韩正回忆朱镕基下工厂：第一句话就是批评 |publisher = 一财网 |date = 2013-08-14 |accessdate = 2013-09-06 |author = 徐燕燕 |archive-url = https://web.archive.org/web/20130824193842/http://www.yicai.com/news/2013/08/2939288.html |archive-date = 2013-08-24 |dead-url = yes }}</ref>。

1990年6月，任[[中国共产主义青年团|共青团]]上海市委副书记（主持工作）。1991年5月，任共青团上海市委书记。1992年11月，任中共[[卢湾区|卢湾区]]委副书记、代区长（1993年5月正式当选区长，期间1991-1994年，在[[华东师范大学|华东师范大学]]国际问题研究所国际关系与世界经济专业学习，获得[[经济学|经济学]]硕士学位）。韩正任职卢湾区时年仅39岁，任职期间韩正着手对辖区的著名商业街—[[淮海路_(上海)|淮海路]]进行改造和升级，并重点和海外势力雄厚的企业财团合作，使淮海路的升级改造基本没有受到之后爆发的[[亚洲金融危机|亚洲金融危机]]的影响，此外韩正任职期间还对卢湾区以前社区环境不佳的地区进行了集中整治，基本解决了辖区内的环境问题<ref name="2003.2">{{cite web|url=http://www.takungpao.com/mainland/content/2012-10/23/content_1285728.htm|title=【2003年2月】韩正上海最年轻市长|accessdate=2013-09-06|date=2012-10-23|publisher=文汇网|archive-url=https://web.archive.org/web/20130909060756/http://www.takungpao.com/mainland/content/2012-10/23/content_1285728.htm|archive-date=2013-09-09|dead-url=yes}}</ref>。

1995年7月，任[[上海市人民政府|上海市人民政府]]副秘书长，并先后兼任市综合经济工作党委副书记、[[上海市计划委员会|上海市计划委员会]]主任、[[上海市证券管理办公室|上海市证券管理办公室]]主任等职。1997年12月，任[[中共上海市委|中共上海市委]]常委。1998年2月，任中共上海市委常委、上海市副市长。2002年5月，任中共上海市委副书记、上海市副市长。2003年2月，在上海市第十二届人民代表大会第一次会议上，年仅49岁的韩正当选为新一届[[上海市市长|上海市市长]]，同时韩正也成为了上海50年来最年轻的市长。2006年9月，代理中共上海市委书记职务<ref>{{cite news |language = zh-hans |author = |url =http://news.163.com/06/0925/13/2RSA9TQ30001124J.html | title =韩正同志代理上海市委书记职务 |publisher =[[新華網|新華網]] | date =2006-09-25 |accessdate =2011-11-12}}</ref>。2007年3月，不再代理上海市委书记职务。2008年1月30日，在上海市第十三届人民代表大会第一次会议上被选举连任上海市市长<ref>{{cite news |language = zh-hans |author =  |url = http://www.sh.xinhuanet.com/2008-01/30/content_12359747.htm |title = 韩正当选上海市市长 |publisher = [[新華網|新華網]] |date = 2008-01-30 |accessdate = 2011-11-12 |deadurl = yes |archiveurl = https://web.archive.org/web/20091005110356/http://www.sh.xinhuanet.com/2008-01/30/content_12359747.htm |archivedate = 2009-10-05 }}</ref><ref>{{cite news |language = zh-hans |author = |url = http://shanghai.kankanews.com/kuaibao/2012-05-22/1172044.shtml |title = 俞正声当选上海市委书记 韩正殷一璀当选为副书记 | publisher = [[看看新聞網|看看新聞網]] |date = 2011-05-22 |accessdate =2012-05-31}}</ref>。2010年，适逢台北市与上海市的「双城论坛」，韩正來台访问四天，成為中国大陆首位访台的直辖市长<ref>{{Cite news|url=http://www.chinatimes.com/newspapers/20100405000664-260301|title=韓正來台 寫台北上海交流史|date=2010-04-05|newspaper=中時電子報|accessdate=2018-03-20|language=zh-hant}}</ref>。
