'''韩红'''（{{bd|1971年|9月26日|catIdx=Han韩红}}），藏名'''央金卓玛'''（{{bo|z=དབྱངས་ཅན་སྒྲོལ་མ|p=Yangjian Zhuoma|w=dbyangs can sgrol ma}}），生于[[中国|中国]][[西藏|西藏]][[昌都市|昌都市]][[卡若區|卡若區]]，祖籍[[山東|山東]][[德州市|德州]]，少年成长于[[北京城|北京城]]，父親為[[漢族|漢族]][[知青|知青]]和[[相声|相聲]]演員，母親為[[藏族|藏族]]歌手雍西。她是中国知名的少數民族女歌手，中央国家机关青联常委，创作型唱将，[[空军少校|空军少校]][[军衔|军衔]]。韩红是[[中国人民解放军空军政治部文工团|空军政治部文工团]]副团长，[[全国政协|全国政协]]委员、[[全国青联|全国青联]]常委、中华海外联谊会理事。[[韩红|韩红]]和[[梦鸽|梦鸽]]、[[谭晶|谭晶]]、[[许飞|许飞]]被誉为[[李双江|李双江]]“四大关门女弟子”<ref>{{cite web |url = http://edu.sina.com.cn/bschool/2013-02-26/1713372183.shtml |title = 商院人物：盘点李双江四大关门女弟子 |accessdate = 2013年2月26日 |publisher = 新浪网 |work = 人民网|language=zh-cn}}</ref>。

== 生平 ==
韓紅因為父親早逝，加上母親常常登台演出，9歲後到北京與奶奶和叔叔相依為命。韓紅因為體型粗胖，未能考進文藝兵，反而成為通信兵，所以自學彈吉他和打鼓。儘管如此，她參加各項軍內歌唱比賽都沒有得名，此情況持續十幾年，2005年，奶奶過世後，因為也常常登台，每年過年都在不同地方度過。1988年韩红进入歌坛。1995年，因為親友幫忙，而錄製《喜瑪拉雅》。1997年底，遇見[[张越_(主持人)|張越]]，在《[[半邊天|半邊天]]》的《別為你的相貌發愁》一集演唱。其後，韩红再借助親友幫忙，推出了第一张个人专辑《[[雪域光芒|雪域光芒]]》。2003年后，韩红逐渐受到关注，多次获得各种“年度最受欢迎女歌手奖”和“最佳女歌手奖”，成为中国大陆最红的女歌手之一。2009年9月韩红被特招入伍，加入空军政治部文工团，[[中国人民解放军文职干部|专业技术五级]]，任[[中国人民解放军空军政治部文工团|空军政治部文工团]]副团长。

2014年12月18日，韩红正式确认加盟[[我是歌手_(第三季)|《我是歌手》第三季]]，並獲得歌王。

2015年12月7日11时19分，韩红在微博发布一张自己的军装照，并表示“离开”（离开军队。经证实她已经转业）。

2016年1月，韩红宣佈加盟香港的[[華星唱片|華星唱片]]轉戰香港樂壇，時隔六年再出新專輯，並表明會嘗試唱廣東歌。還說她會唱《浮誇》、《海闊天空》二首廣東歌曲。1月9日，前往[[臺北小巨蛋|臺北小巨蛋]]錄製[[農曆新年|農曆新年]][[除夕|除夕]]特別節目《[[2016超級巨星紅白藝能大賞|2016超級巨星紅白藝能大賞]]》，演唱〈天天想你〉、〈我愛故我在〉，為出道多年，首次在台演出。

== 社会活动 ==
1999年10月3日，在[[貴州|貴州]][[马别河|馬嶺河]]風景區，正在運行的纜車突然墜毀，在纜車墜落的那一剎那間，車廂內來自南寧市的潘天麒、賀艷文夫婦，不約而同地使勁將年僅兩歲半的兒子高高舉起。結果，這名名叫潘子灝的孩子只是嘴唇受了點輕傷，而他的雙親卻先後死去。 

1999年10月3日的這場災難讓當時只有2歲半的潘子灝變成了孤兒這個生命的故事，深深打動了韓紅，經過多方聯繫，她領養了這個大難不死的小孩。2000年，应[[中国中央电视台|中国中央电视台]][[3.15晚会|3.15晚会]]要求，韩红根据潘子灏的故事，亲自编曲，填词并演唱了《天亮了》一曲，歌曲被收录在韩红专辑《醒了》中，于次年7月发行。

2010年11月28日，韩红在赴[[舟曲泥石流|舟曲泥石流]]灾区的途中，遭遇车祸，所乘坐的[[越野车|越野车]]翻滚至路边的沟中，所幸未造成人员伤亡<ref>{{cite news|title=韩红劫后余生回京痛哭 赴医院进行全面体检(图)|url=http://ent.ynet.com/view.jsp?oid=74185521&pageno=1|accessdate=2010-12-01|newspaper=北京晨报|date=2010-12-01|deadurl=yes|archiveurl=https://web.archive.org/web/20110109012238/http://ent.ynet.com/view.jsp?oid=74185521|archivedate=2011-01-09}}</ref>。韩红称，之所以未受到严重的伤害，是因为她繫了[[座位安全带|安全带]]<ref>{{cite news|author=练红宁|title=系全带才大难不死 韩红车祸躲过一劫|url=http://gb.cri.cn/27564/2010/11/30/1326s3073110.htm|accessdate=2010-11-30|date=2010-12-01}}</ref>。

韩红热衷于慈善事业，2007年发起“爱心西藏行”等公益活动，2008年5月14日成立“韩红爱心救援团队”前往[[汶川大地震|汶川灾区]]支持救援， 2012年5月9日注册成立了韩红爱心慈善基金会。
