'''韩永进'''（{{bd|1958年||||catIdx=Han韩永进}}），[[河北|河北]][[顺平|顺平]]人，文学博士，曾任[[中国国家图书馆|中国国家图书馆]]馆长。

== 人物简介 ==
    
韩永进先生于1958年2月出生于中国[[北京市|北京市]]，河北顺平人。1982年毕业于[[南开大学|南开大学]]历史系。现任中国国家图书馆馆长、党委书记，中国[[国家古籍保护中心|国家古籍保护中心]]主任。

== 个人简历 ==

* 2018年9月，由于年龄原因，不再担任中国国家图书馆馆长。<ref>[https://baike.baidu.com/redirect/a970Oh94SebUhUt0RHp5LrnTBAx55X1WfOmlVr2xKLfUHvGOhpk2-Nztq6bKeYzQLh_vO0pzK1lYKeHBb8Oyk7YiJ4KRx3YPxyoECXMEerXzZY79NWc 韩永进理事长在2018年中国图书馆年会开幕式上的致辞]，国家图书馆[。</ref>
* 2014年1月：任国家图书馆馆长、党委书记兼国家古籍保护中心主任。
* 2003年11月至2014年1月：在国家文化部工作，历任教育科技司司长、全国艺术科学规划领导小组办公室主任，政策法规司司长、文化部文化体制改革领导小组办公室主任。
* 1995年3月至2003年11月：在中央办公厅调研室工作，历任中央宣传思想工作领导小组秘书组调研员、副组长、组长（正局级）。
* 1989年12月至1995年3月，在辽宁省[[绥中县|绥中县]]工作，历任副县长、县委副书记。
* 1982年2月至1989年12月：先后在中央办公厅和国务院办公厅工作。

== 教育背景 ==
    
* 1982年2月毕业于南开大学历史系。

== 社会兼职 ==
* 国家语言文字工作委员会委员
* 国家科学技术进步奖、国家技术发明奖、全国哲学社会科学基金、中宣部五个一工程奖等评委
* 国家行政学院、哈尔滨工程大学、山东艺术学院、武汉音乐学院等院校兼职教授
