'''韩占文'''（{{bd|1965年|10月|}}），[[河北|河北]][[顺平县|顺平]]人，中国[[天体物理学|天体物理学]]家，[[中国科学院云南天文台|中国科学院云南天文台]]研究员。<ref>{{Cite web|url=http://sourcedb.ynao.cas.cn/rencaiku/201008/t20100812_2923526.html|title=韩占文——中国科学院云南天文台|accessdate=2017年12月|author=|date=|publisher=}}</ref>2017年当选为[[中国科学院院士|中国科学院院士]]。<ref>{{Cite web|url=http://www.casad.cas.cn/doc/17503.html|title=关于公布2017年中国科学院院士增选当选院士名单的公告|accessdate=2017年12月|author=|date=|publisher=}}</ref>

== 教育经历 ==
1984年毕业于[[河北大学|河北大学]]物理系，1987在中国科学院云南天文台获硕士学位，1995年在英国[[剑桥大学|剑桥大学]]获博士学位。<ref name=":0">{{Cite web|url=http://www.casad.cas.cn/aca/371/sxwlxb-201711-t20171129_4625042.html|title=韩占文——中国科学院学部与院士|accessdate=2017年12月|author=|date=|publisher=}}</ref>

== 研究方向 ==
主要从事[[恒星结构和演化|恒星结构和演化]]的研究。发展了双星星族合成研究，并应用于特殊恒星大样本观测结果的理论解释；建立了[[热亚矮星|热亚矮星]]、[[钡星|钡星]]、[[双白矮星|双白矮星]]等特殊恒星的形成模型，阐述了双星在特殊恒星形成研究中的重要地位；将双星星族合成用于星系研究，发现双星演化形成的热亚矮星等天体对年老星族的短波光谱有重要贡献，解释了早型星系的紫外超。<ref name=":0" /><ref>{{Cite web|url=http://210.72.91.131/~zhanwenhan/chinese.html|title=研究方向——欢迎光临韩占文之网页 ！|accessdate=2017年12月|author=|date=|publisher=}}</ref>

== 荣誉 ==
1998年获第六届中国青年科技奖。1999年获国家“[[杰出青年基金|杰出青年基金]]”资助。2004年入选首批“[[新世纪百千万人才工程|新世纪百千万人才工程]]”国家级人选。<ref>{{Cite web|url=http://210.72.91.131/~zhanwenhan/chinese.html|title=个人简历——欢迎光临韩占文之网页 ！|accessdate=2017年12月|author=|date=|publisher=}}</ref>

曾获何梁何利基金科学与技术进步奖（天文学奖），国家自然科学奖二等奖等。
