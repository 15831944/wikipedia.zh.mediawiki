== 移民 ==
[[在韓蒙古人|在韓蒙古人]]形成了最大的蒙古国移民团体。<ref name="TaipeiTimes">{{Template:Citation|url = http://www.taipeitimes.com/News/world/archives/2003/10/13/2003071505|periodical = Taipei Times|date = 2003-10-13|accessdate = 2007-08-17|title = Mongolians, Koreans share ancient bonds}}</ref><ref name="Seabreeze">{{Template:Citation|url = http://english.chosun.com/w21data/html/news/200503/200503290013.html|date = 2005-03-29|accessdate = 2007-09-05|periodical = The Chosun Ilbo|title = Sea Breeze Helps Korea's Mongolians Speak with One Voice}}</ref>2008年的统计数据表明大概有33000蒙古人组成了这一团体。<ref name="KoreanDream">{{Template:Citation|url = http://english.chosun.com/w21data/html/news/200804/200804240009.html|title = 'Korean Dream' fills Korean classrooms in Mongolia|periodical = The Chosun Ilbo|date = 2008-04-24|accessdate = 2009-02-06}}</ref> 同样地，韩国也拥有大概3,500[[在蒙韓國人|在蒙韩国人]]。<ref name="3500SouthKoreans">{{Template:Citation|periodical = YTN News|url = http://www.ytn.co.kr/global_korean/global_view.php?key=200807050858441453&s_mcd=0930&s_hcd=19|date = 2008-07-05|accessdate = 2009-01-27|title = 유혈 사태, 평온 회복 중}}</ref> 根据2006年签署的双边协议, 每个国家的公民居住在其他国家免除；否则强制贡献他们居住的国家养老金计划。<ref>{{Template:Citation|title = South Korean president in Mongolia|date = 2006-05-18|accessdate = 2009-01-27|periodical = The Mongol Messenger|url = http://www.mongolmessenger.mn/issue/060518.php}}</ref>
