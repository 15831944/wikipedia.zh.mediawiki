'''韩国铁道公社311000系电力动车组'''（{{kor|諺=한국철도공사 311000호대 전동차|漢=韓國鐵道公社 311000號帶 電動車}}）是[[韩国铁道公社|韩国铁道公社]]的一款[[通勤型电力动车组|通勤型电力动车组]]，在[[首都圈电铁1号线|首都圈电铁1号线]]运营。

== 概况 ==
311000系电力动车组是[[广域电铁|广域电铁]]的列车，是交直两用车辆，一共分为3代，由[[现代Mobis|现代Mobis]]、[[大宇重工业|大宇重工业]]和[[现代Rotem|现代Rotem]]负责生产，在1996年正式使用。主要是为了替代部分老旧的[[韩国铁道1000系电力动车组|1000系]]。最初运行于[[首都圈电铁4号线|首都圈电铁4号线]]的[[安山线|安山线]]、[[首都圈电铁1号线|首都圈电铁1号线]]的[[京元线龙山-城北运行模式|京元线龙山-城北运行模式]]（已废止）和[[京釜高速线|京釜高速线]]。现在运行于[[首都圈电铁1号线|首都圈电铁1号线]]的[[京元线|京元线]]、[[京釜线|京釜线]]、[[京仁线|京仁线]]、[[长项线|长项线]]、[[饼店基地线|饼店基地线]]区间（为25000V、60Hz交流电区间），亦包括[[首尔地铁|首尔地铁]]营运的[[首尔地铁1号线|首尔地铁1号线]]（为1500V直流电区间）。列车客室车门为内藏门，每节车厢共4对车门，列车在两端设有[[紧急出口|紧急出口]]，目前均为10编组。列车的[[VVVF|VVVF]]由[[日本|日本]][[东芝|东芝]]生产。列车的型号开始更改为'''311000系'''。

== 使用情况 ==
=== 第1代(납작이)===
第1代列车在1996年至1999年期间生产，初期外涂装为黄、绿色。在2005年1月1日国营公司化后，列车的涂装更换为红、蓝、白三色。在2003年2月18日[[大邱地铁纵火案|大邱地铁纵火案]]发生后，列车材料全部改为了防火材料。列车一共分为2批，而列車車門採用氣動內藏門，而車門上的車窗的面積較小。而部分列車的制御裝置已由[[GTO|GTO]]-[[VVVF|VVVF]]改為[[IGBT|IGBT]]-[[VVVF|VVVF]]。
==== 第1批 ====
编号为31101-31127（原编号为501-528），由[[现代Mobis|现代Mobis]]负责生产。
==== 第2批 ====
编号为31128-31141（原编号为529-542），由[[大宇重工业|大宇重工业]]负责生产，是为了替代1974年投入服務的[[韩国铁道1000系电力动车组|韩国铁道1000系]]第1代列车。较第1批相比，增加了轮椅位，其中车组编号为31139-31141中的部分车厢由[[韩国铁道1000系电力动车组|1000系]]的第2代车厢改造插入而成。

=== 第2代(동글이) ===
第2代列车在2002年-2004年期间生产，外观与第1代列车不同。在2003年2月18日[[大邱地铁纵火案|大邱地铁纵火案]]发生后，列车材料全部改为了防火材料。一共分为3批，配属于[[饼店车辆基地|饼店车辆基地]]和[[九老车辆基地|九老车辆基地]]。車門是採用氣動方式的，不過部分第2代列車的車門驅動方式改為電動形式以及部分列車的制御裝置已由[[GTO|GTO]]-[[VVVF|VVVF]]改為[[IGBT|IGBT]]-[[VVVF|VVVF]]。
==== 第3批 ====
编号为31142-31147（原编号为543-548），是在[[首都圈电铁1号线|首都圈电铁1号线]]服务区段南延伸至[[饼店站|饼店站]]后开始服役。在初期外涂装为黄、绿色。在2005年1月1日国营公司化后，列车的涂装更换为红、蓝、白三色。
==== 第4批 ====
第4批车组编号为31148-31163（原编号为549-564），较第3批相比，增加了[[列车综合信息系统|列车综合信息系统]]（TGIS），编号为31154（原编号为555）的列车在[[2011年7月韩国暴雨|2011年7月韩国暴雨]]中受损严重，后于2012年3月修复完成。
==== 第5批 ====
第5批车组编号为31164、31165（原编号为565、566），较之前的列车拥有LED显示屏，而制御裝置在出厂時已改為[[IGBT|IGBT]]-[[VVVF|VVVF]]。

=== 第3代(뱀눈이) ===
第3代列车在2005年-2013年期间生产，外涂装为红、蓝、白三色，外观与第1、2代列车不同，另外車門採用電動式(第1、2代列车是採用氣動式)。一共分为4批，配属于[[里门车辆基地|里门车辆基地]]、[[饼店车辆基地|饼店车辆基地]]和[[九老车辆基地|九老车辆基地]]。
==== 第6批 ====
第6批车组编号为31166-31177（原编号为567-578），于2005年由[[现代Rotem|现代Rotem]]负责生产。
==== 第7批 ====
第7批车组编号为31178-31182（原编号为579-592），于2006年生产。
==== 第8批 ====
第8批车组编号为31183-31189，于2012年生产。
==== 第9-10批 ====
第9-10批车组编号为31190-31194，于2013年生产，是为了替代部分[[韩国铁道1000系电力动车组|韩国铁道1000系]]第2代列车。

=== 第4代(삼눈이) ===
第4批列車於2016年投入服務，編號為31195-31203，是为了替代餘下[[韩国铁道1000系电力动车组|韩国铁道1000系]]第2代列车。另外這一批列車採用單臂式集電弓，車廂亦裝設閉路電視。

== 事故 ==
*2007年，车组编号为519的列车在[[永登浦站|永登浦站]]与[[韩国铁道4400型柴油机车|4404号柴油机车]]发生了严重碰撞，导致损毁严重，目前剩余8节列车改为[[韩国铁道公社319000系电力动车组|319000系]]。
*2010年5月1日，车组编号为575(今31174)的列车与[[韩国铁道1000系电力动车组|1000系]]车组编号为158的列车在[[九老车辆基地|九老车辆基地]]发生了严重碰撞。31174受損的車廂已修复完成並重新投入服務，而158號列車因1158車廂嚴重毀損不獲復修而整列直接退役。
*2012年6月3日，车辆编号为31147号（原编号为548）列车于[[天安站|天安站]]出轨，导致[[集电弓|集电弓]]受损，后于同年9月修复完成<ref>[http://news.sbs.co.kr/section_news/news_read.jsp?news_id=N1001213802 천안역서 열차 탈선…장항선 열차 운행 중단]</ref><ref>[http://news.sbs.co.kr/section_news/news_read.jsp?news_id=N1001213841 천안역 탈선사고 9시간만에 복구…항의 빗발쳐]</ref>。
