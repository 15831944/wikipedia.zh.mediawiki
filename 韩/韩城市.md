'''韩城市'''，位於[[中華人民共和國|中国]][[陕西省|陕西省]]东部，是[[国家历史文化名城|国家历史文化名城]]，也是一座重要的[[能源|能源]]工业城市。现在是陕西省[[省直辖县级行政区|省直辖县级市]]。

== 历史 ==
相传夏禹“导河积石，至于龙门”，因而史以“龙门”为韩城地域的代称。[[西周|西周]]时为韩侯封邑。[[春秋|春秋]]时属[[晋国|晋国]]，称韩原。[[战国|战国]]时属[[魏國_(戰國)|魏国]]，称少梁。战国末期[[秦国|秦国]]在此设夏阳县。至[[隋代|隋代]]改称韩城，并延续至今。1983年撤县设市，1986年被定为[[国家历史文化名城|国家历史文化名城]]，2007年被命名为中国优秀旅游城市。2012年被评选为陕西省计划单列市，享受地级市待遇。

== 地理 ==
韩城地处[[黄河|黄河]]西岸。

== 交通 ==
韩城龙门位于市境东北的[[黄河|黄河]]边，自古以来就是兵家重地。隋末[[李渊|李渊]]、明末[[李自成|李自成]]，以及近代的[[中国共产党|中国共产党]]军队，都由这里渡过黄河，现建有[[侯西铁路|侯西铁路]]。韩城交通基础设施完善，境内有[[108国道|108国道]]、[[304省道|304省道]]、[[沿黄公路|沿黄公路]]和[[阎禹高速公路|阎禹高速公路]]。西（安）太（原）铁路穿境而过，开通了全国唯一的县级市直通北京的列车。完善的交通及通信网络，极大地促进了当地经济的发展。2003年，韩城综合经济实力位居陕西省第二，在全国县域经济基本竞争力评价中进入西部百强行列。

== 经济 ==
韩城地区[[煤炭|煤炭]]资源丰富，到今天，已发展成以煤炭、机焦、钢铁工业为主的[[重工业|重工业]]基地。

韩城地处渭北地区。地形西部为山区、丘陵，东部紧依黄河为平原、河谷，林木资源丰富。全市年平均气温13℃，降水量559毫米。主要农产品有[[小麦|小麦]]、[[玉米|玉米]]、[[花椒|花椒]]等。特产[[韩城大红袍花椒|韩城大红袍花椒]]，是受国家保护的[[原产地域产品|原产地域产品]]。

== 名人 ==
韩城是[[史记|史记]]作者[[司马迁|司马迁]]的故乡。春秋时期赵武、乾隆年间名相王杰、现代著名作家杜鹏程等也都诞生在这里。

== 教育 ==
[[明|明]][[清|清]]时期，县内有名的书院有十几座。民国早年，全县仅小学就有200多所。

== 风景名胜 ==
[[File:Dangjiacun.JPG|thumb]]
[[File:P6190112.JPG|thumb]]
韩城是[[国家历史文化名城|国家历史文化名城]]，古城内至今仍保存有大量历史街道和住宅，有[[唐|唐]]至清各代的古建筑140多处。[[司马迁墓和祠|司马迁墓和祠]]等十五处古建筑群被列为[[全国重点文物保护单位|全国重点文物保护单位]]，其数量在中国所有[[县|县]]、县级市和[[地级市|地级市]]辖区中位居前几名。其中，位于城北的[[党家村|党家村]]更因密布着明清[[四合院|四合院]]式宅院，而被命名为[[中国历史文化名村|中国历史文化名村]]。位于昝村镇的梁带村两周遗址被评为2005年度中国十大考古发现。目前，韩城市已在老城区外兴建了新城区，以保护老城的历史风貌和布局。韩城的文化遗产极为丰富，最具代表性质的有门楣题字、百面锣鼓、耍神楼、秧歌等。近年来，韩城市依托丰富的文物资源，旅游业得到了迅速发展，初步形成了南司马迁祠墓、中古城和党家村、北龙门的旅游格局，并融入了全省旅游东环线。

===  韩城八景 ===
*[[禹门春浪|禹门春浪]]：禹门口禹门三叠浪。相傳鲤鱼跳龙门之说发于此；
*[[猴山秋韵|猴山秋韵]]：[[薛峰山|薛峰山]]因為其西北处有一风化石的外形跟顽猴非常相像而又名猴山，再配當地深秋的美景；
*[[横山烟雨|横山烟雨]]：板桥火炬村横山道观之烟雨与香火交互产生之幻化美景；
*[[象岭春晓|象岭春晓]]：位於[[象山中学|象山中学]]西边的象山，因其形似俯卧之大象，象首南面饮狮山潭中之水，故称。旧时每年三月游客满山，现时有今人开辟象岭山庄供游人歇脚住宿；
*[[嵬峰摩霄|嵬峰摩霄]]：位於韩城西25公里处，旧时建有庙宇，多植松柏，多为文人骚客造访，另，其西香山亦为韩城一名胜，诗人白居易晚年隐居于此，后人有“九老图”传世，诗人故后葬于龙门山；
*[[圆觉晨钟|圆觉晨钟]]：即今烈士陵园，旧时为圆觉寺，寺东为钟，以西为鼓，古人所称晨钟暮鼓是也；
*[[苏柏南柯|苏柏南柯]]：位於韩城西北处，汉[[苏武墓|苏武墓]]建於其上，古时合抱松柏常青，柯树皆南向；
*[[龙泉秋稼|龙泉秋稼]]：因泉处于圆觉寺内龙王神座下得名，[[唐初四杰|唐初四杰]]之一[[王勃|王勃]]与韩城好友曾观龙泉之美景。
