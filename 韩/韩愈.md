'''韓愈'''（{{bd|768年||824年12月25日|catIdx=Han韓愈}}），字'''退之'''，[[河南|河南]][[河陽|河陽]]（今[[河南|河南]][[孟州|孟縣]]）人<ref> 牛广臣《韩愈籍贯·郡望考究》 《锦州文史资料 第21辑 锦州文史考辨》</ref>，自稱[[郡望|郡望]]昌黎，世稱'''韓昌黎'''；晚年任[[吏部|吏部]][[侍郎|侍郎]]，又称'''韩吏部'''。卒[[谥|谥]]文，世称'''韩文公'''。唐代文学家，与[[柳宗元|柳宗元]]是当时[[古文运动|古文运动]]的倡导者，合稱「韓柳」。[[苏轼|苏轼]]称赞他“文起八代之衰，道濟天下之溺，忠犯人主之怒，勇夺三军之帅”（八代：[[漢朝|東漢]]、[[曹魏|魏]]、[[晉朝|晉]]、[[刘宋|宋]]、[[南朝齐|齐]]、[[南朝梁|梁]]、[[南朝陈|陈]]、[[隋朝|隋]]）。[[散文|散文]]、[[诗|诗]]，均有名。著作收錄《[[s:昌黎先生集|昌黎先生集]]》。

== 家世 ==
汉弓高侯[[韩颓当|韩颓当]]裔孙韩寻，后汉陇西太守，世居[[颍川|颍川]]，生司空[[韩棱|韩棱]]，字伯师，其后徙安定武安。后魏有常山太守、武安成侯[[韩耆|韩耆]]，字黄耇，永兴中，自[[赫连屈丐|赫连屈丐]]来降，位常山太守，假安武侯，仍居常山之九门。卒，赠泾州刺史，谥曰成。耆生茂，字元兴，尚书令、征南大将军、安定桓王。生子：韩备、韩均、韩延寿、韩天生。
*七世祖：[[韩耆|韩耆]]，字黄耇，魏常山太守、安武子。
*六世祖：[[韩茂|韩茂]]，字元兴，北魏安定公，赠泾州刺史、安定王，谥号桓王。
*五世祖：韩均，字天德，定州刺史、安定康公。
*高祖：韩皎，雅州都督。
*曾祖：韩仁泰，曹州司马。
*祖：韩叡素，桂州长史。生晋卿、季卿、子卿、仲卿、云卿、绅卿、升卿。
*父：韩仲卿，秘书郎。生韩会、韩介、韩愈。

== 生平 ==
韓愈自稱是[[漢代|漢代]][[諸侯|諸侯]][[韓王信|韓王信]]的後裔，父[[韓仲卿|韓仲卿]]。韓愈出生不到幾歲，母親過世，三歲喪父，受大哥[[韓會|韓會]]（即十二郎[[韓老成|韓老成]]之伯父兼[[養父|養父]]，韓老成被韓愈二哥韓介[[过继|过继]]给韓會）撫育，隨兄長為官轉徙[[長安|長安]]、[[韶州|韶州]]（今[[廣東|廣東]][[韶關|韶關]]）等地。後韓會病逝韶州，隨嫂鄭氏護喪返回河陽。後又避難[[宣城|宣城]]（今[[安徽|安徽]][[宣城|宣城]]），與侄韓老成，同由鄭氏撫養成人，情逾手足。

韓愈至七歲才開始讀書，十三歲能寫文章，自言“前古之興亡，未嘗不經於心也，當世之得失，未嘗不留於意也”<ref>《與鳳翔邢尚書書》</ref>。唐朝[[貞元_(唐朝)|貞元]]二年（786年）赴長安應試，無[[門第|門第]]資蔭，三試不第。贞元八年（792年）始中[[进士|进士]]<ref>洪兴祖《韩子年谱》引《科名记》云：“贞元八年陆贽主司，试《明水赋》、《御沟新柳诗》。其人贾棱、陈羽、欧阳詹、李博、李观、冯宿、王涯、张季友、齐孝若、刘遵古、徐季同、侯继、穆贽、韩愈、李绛、温商、庾承宣、员结、胡谅、崔群、邢册、裴光辅、万当，是年一榜多天下孤隽伟杰之士，号‘龙虎榜’。”</ref>。應[[吏部|吏部]]試，又三次不中。貞元十一年（795年），三次上書[[宰相|宰相]]，希得薦舉。

貞元十二年（796年），[[汴州|汴州]][[宣武軍|宣武軍]]亂，隨宣武軍[[節度使|節度使]][[董晉_(唐朝)|董晉]]赴任，擔任「觀察[[推官|推官]]」。期間與[[孟郊|孟郊]]相識交游，[[李翱|李翱]]、[[張籍|張籍]]入其門下。董晉卒，改任[[武寧|武寧]]節度使[[張建封|張建封]]屬下「節度推官」。張建封死，遷居[[洛邑|洛邑]]。

貞元十七年（801年），任[[國子監|國子監]]四門博士，貞元十九年(803)前，著《師說》。貞元十九年（803年）任[[监察御史|监察御史]]，因[[關中|關中]]旱災，上《御史臺上論天旱人饑狀》，糾彈國戚[[京兆尹|京兆尹]][[李實|李實]]，遂贬阳山令，深受百姓愛戴，百姓甚至以「韓」字，為兒取名。這一年侄子韓老成去世，寫《[[祭十二郎文|祭十二郎文]]》。

[[元和_(唐朝)|元和]]六年（811年）任[[國子博士|國子博士]]，作〈進學解〉，受[[裴度|裴度]]賞識，擢為[[禮部|禮部]][[郎中|郎中]]。815年隨裴度-{征}-[[淮西|淮西]]，因功擢任[[刑部|刑部]][[侍郎|侍郎]]，並作〈平淮西碑〉。

轉任[[刑部侍郎|刑部侍郎]]时，元和十四年（819年）正月，[[唐憲宗|唐憲宗]]將[[釋迦牟尼佛|釋迦牟尼佛]][[佛骨|佛骨]]迎入了宮中供養三日，舉國若狂，甚有百姓燒指灼背而供養者。因谏阻[[天子|天子]]迎佛骨，耗費[[銀錢|銀錢]]，作《諫迎佛骨表》說明“唯[[梁武帝|梁武帝]]在位四十八年，前後三度捨身施[[佛|佛]]，[[宗廟|宗廟]]之祭，不牲宰，晝日一食，止於菜果，其後竟為[[侯景|侯景]]所[[侯景之乱|兵逼]]，餓死[[台城|台城]]，國亦尋滅。事佛求福，乃更得禍。由此觀之，佛不足信，亦可知矣。”憲宗聞之大怒，將處以[[死刑|極刑]]，[[裴度|裴度]]、[[崔群|崔群]]力救道：「愈言訐牾，罪之誠宜。然非內懷至忠，安能及此。願少寬假，以求諫爭。」帝曰：“愈言我奉佛太過，猶可容；至謂東漢奉佛以後，天子咸夭促，言何乖剌邪？愈，人臣，狂妄敢爾，固不可赦。”<ref>《新唐书·韩愈传》</ref>乃贬为[[潮州_(古代)|潮州]][[刺史|刺史]]（今[[广东省|廣東]][[潮州市|潮州]]）。

往潮州路上，已經生病的愛女也因為不堪路程折磨而病死，韓愈傷感不已，來到了藍關（今[[陕西省|陝西]][[藍田縣|藍田]]）時，大雪紛飛，韓愈見到姪孙[[韓湘|韓湘]]（傳說韓湘就是[[八仙|八仙]]之一的[[韓湘子|韓湘子]]）。不禁再三嗟歎道：「吾為汝成此詩。」詩吟：「一封朝奏九重天，夕貶潮陽路八千；願為聖明除弊事，肯將衰朽惜殘年！雲横秦嶺家何在？雪擁藍關馬不前；知汝此來應有意，好收吾骨瘴江邊。」

到了潮州之後，韓愈用心治民興學、又藉以工抵債釋放[[奴婢|奴婢]]，與[[潮州大顛|潮州大顛]]和尚成為好友。後來在潮州又寫〈[https://zh.wikisource.org/wiki/%E7%A5%AD%E9%B1%B7%E9%AD%9A%E6%96%87 祭鱷魚文]〉，往河裏扔了一[[豬|豬]]一[[羊|羊]]，據聞[[鱷魚|鱷魚]]就此絕跡。事實上，後來[[宰相|宰相]][[李德裕|李德裕]]、宋朝[[陳堯佐|陳堯佐]]在潮州時，看見鱷魚仍在。韓愈卒後，當地乃建[[韓文公廟|韓文公廟]]供奉。

在潮州任內，韓愈上表懺罪謝恩曰：“臣以狂妄戆愚，不識禮度，陳佛骨事，言涉不恭，正名定罪，萬死莫塞。陛下哀臣愚忠，恕臣狂直，谓言雖可罪，心亦無它，特屈刑章，以臣為潮州刺史。既免刑誅，又獲禄食，聖恩寬大，天地莫量，破腦刳心，豈足為謝！”並陳述邊地困境，希冀憐憫之心：“臣所領州，在廣府极東，過海口，下惡水，濤瀧壮猛，難计期程，颶風鱷魚，患禍不測。州南近界，漲海連天，毒霧瘴氛，日夕發作。臣少多病，年纔五十，髪白齒落，理不久長。加以罪犯至重，所处遠惡，憂惶惭悸，死亡無日。單立一身，朝無親黨，居蠻夷之地，與魑魅同群，苟非陛下哀而念之，誰肯為臣言者？”<ref>《新唐书·韩愈传》</ref>表進，[[敕|敕]]令改任[[袁州|袁州]]（今[[江西|江西]][[宜春|宜春]]）。

[[唐穆宗|唐穆宗]]即位後，奉旨回京，歷任[[國子監祭酒|國子監祭酒]]、[[兵部|兵部]]侍郎、[[吏部|吏部]]侍郎、[[京兆尹|京兆尹]]兼[[御史大夫|御史大夫]]等職，是人稱其為「韓吏部」

長慶四年（824年），八月因疾告假。十二月二日（西曆12月25日），於長安靖安里的家中與世長辭，年五十七 。贈為禮部尚書，謚為文。  

寶曆元年（825年），三月葬於河陽。

宋元豐元年（1078年），神宗追封為昌黎伯，並從祀孔廟。

== 身後 ==
[[File:Chaozhou_Hanwengong_Ci_2013.10.27_09-14-50.jpg|thumb]]正殿]]
當時伽藍四萬所，僧尼二十六萬五千餘人，王公與士民花費過多金錢且盲目[[迷信|迷信]][[佛教|佛教]]的風氣，一度對唐朝造成頗大的經濟壓力，雖然韓愈曾有逼迫所有僧侶還俗的言論，但事實上他並不完全反佛，所以他也不避諱與[[僧人|僧人]]應酬唱和，如[[潮州大顛|潮州大顛]]、[[三平義中|三平義中]]師徒。他也反對[[道教|道教]]對[[炼丹|仙丹]]的迷信，要人們食用丹藥，說因此“杀人不可计”，但並不是針對道家思想。韓愈以“道統”自命，三十六歲時已經“而視茫茫，而髮蒼蒼，而齒牙動搖”（〈[[祭十二郎文|祭十二郎文]]〉），晚年有“落齒”詩：“去年落一牙，今年落一牙。俄然去六七，落勢殊末已”。後來在長安城南興建過豪宅，也有絳桃、柳枝等妓，能歌善舞<ref>宋代[[王谠|王谠]]《唐语林》卷六《补遗》-{云}-：“韩退之有二妾，一日绛桃，一日柳枝，皆能歌舞。初使王庭凑，至寿阳驿，绝句云：‘风光欲动别长安，春半边城特地寒。不见园花兼巷柳，马头惟有月团团。’盖有所属也。柳枝后逾垣遁去，家人追获。及镇州初归，诗曰：  ‘别来杨柳街头树，摆弄春风只欲飞。还有小园桃李在，留花不放待郎归。’自是专宠绛桃矣。”</ref>。長慶四年（824年）敬宗即位，同年十二月韓愈因病去世，年五十七<ref>李翱《韩公行状》云：“长庆四年（824）得病，满百日假，既罢，以十二月二日卒于靖安里第。……享年五十七，赠礼部尚书。”《旧唐书》卷一六○《韩愈传》云：“长庆四年十二月卒，时年五十七，赠礼部尚书，谥曰文。”</ref>。妻子出身范陽盧氏，二子[[韩昶|韩昶]]、[[富平|富平]]令[[韩州仇|韩州仇]]，有一女为[[李漢_(唐朝)|李汉]]妻，一女为[[蒋系|蒋系]]妻。韓愈去世後，〈韓文公墓誌銘〉是由其生前指定的高徒[[皇甫湜|皇甫湜]]撰寫，[[李翱|李翱]]令作[[行狀|行狀]]。

现在[[孟州市|孟州市]]西虢乡韩庄（传为韩愈老家）修建有韩文公墓。广东潮州市[[韩文公祠|潮州韩文公祠]]是中国大陆现存保存最完整历史最久远的纪念韩愈的专祠。臺灣[[屏東縣|屏東縣]][[內埔鄉|內埔鄉]]的[[昌黎祠|昌黎祠]]（韓愈廟）是臺灣唯一主祀韓愈的廟宇。

== 文學 ==
===古文成就===
韓愈散文內容豐富，形式多樣化，眾體兼長，不落俗套，力求創新、構思奇巧，詞鋒銳利，雄奇奔放，氣勢磅礡，汪洋恣肆，曲折多變，波瀾起伏，想像豐富，感情充沛，析理透闢，邏輯嚴密，設譬巧妙，筆觸幽默，句法則駢散交錯。

韓愈文章以排斥[[佛教|佛]][[老子|老]]，闡明儒家之道為宗旨，《[[wikisource:zh:師說|師說]]》、《[[送董邵南序|送董邵南序]]》、《[[原性|原性]]》、《[[原道|原道]]》、《[[諫迎佛骨表|諫迎佛骨表]]》、《[[進學解|進學解]]》、《[[送窮文|送窮文]]》、《[[wikisource:zh:柳子厚墓誌銘|柳子厚墓誌铭]]》，備受傳誦。後世尊為“[[唐宋八大家|唐宋八大家]]”之首。

== 主張 ==
[[File:Han_Yu.jpg|thumb]]
韓愈主張「文道合一，明道為主」，柳冕首先提出(文以載道)，周敦頤首先寫出(文以載道)四字，宣揚儒道，攘斥佛、老，「破駢為散」，反對六朝以來的[[駢文|駢文]]文風，主張恢復三代兩漢自然質樸的文體，「言貴獨到」，「詞必己出」，要「文從字順」。

韓愈主張思想復古、文學復古，以復古道為目的，復古文為手段，而所謂道，是指儒家的道。韓在繼承的基礎上亦有所創造和革新，主張學古文“師其意不師其辭”“唯陳言之務去”。

===領導古文運動===
貞元、元和年間，韓愈和[[柳宗元|柳宗元]]開始寫作古文，由於有理論有創作，故有成就。韓愈有自信，百折不回，又善宣傳，廣收門徒，遂形成聲勢浩大的社會運動。

韓愈繼承前人之經驗，意志堅強，大力提倡古文。正如李漢「昌黎先生集序」說：「時人始而驚，中而笑且排，先生益堅。」他既有理論，又有優秀作品示人，得到[[李翱|李翱]]、皇甫湜、[[張籍|張籍]]、[[孟郊|孟郊]]等文友和後進支持，故能領導古文運動。

===宋明理學的先驅===
儒學在魏晉南北朝以來，受佛道兩教大盛所影響，漸漸失去統治地位。韓愈本身主張復古，反對佛教。為儒教爭取統治的地位，以代替佛教或道教統治的地位。為後來[[宋明理學|宋明理學]]產生了先驅作用。韓愈在他的道統傳承的說法中把[[孟子|孟子]]說成[[孔子|孔子]]的繼承人，並認為聖人之道在孟子以後失傳，使[[孟子|孟子]]在道統中具有了與孔子同等的地位，這就把孟子的地位大大提高了。影響到後來宋明理學的學者把《孟子》一書列入四書，使孟子在中國文化中的地位真正提高起來。而另外在宋代《[[大學_(經傳)|大學]]》被尊為四書之一，獲得了儒家重要經典的地位，而闡揚《大學》其實在韓愈已經開始。<ref>陳來 《宋明理學》</ref>。

===詩歌===
其诗有论者以为可以列[[李白|李白]][[杜甫|杜甫]]之后，居全唐第三。韩诗以文为诗，以论为诗，求新求奇，有气势，对纠正大历诗风起到了一定作用，对宋诗产生了较大影响。[[王安石|王安石]]以韩愈《荐士》评[[孟郊|孟郊]]诗反過來評韓愈本人：“横空盘硬语，妥贴力排奡”，概括其诗风。[[葉燮|葉燮]]論韓詩：“韓愈為唐詩之一大變。其力大，其思雄，崛起特為鼻祖。”<ref>《原诗》内篇上</ref>[[赵翼|赵翼]]《瓯北诗话》云：“诗家好作奇句警语，必千锤百炼而后能成。如……昌黎之‘巨刃磨天扬’、‘乾坤摆雷硠’等句，实足惊心动魄，然全力搏兔之状人皆见之”。他的代表作有《南山诗》、《调张籍》、《听颖师弹琴》、《左迁至蓝关示侄孙湘》、《早春呈水部张十八员外》、《春雪》、《晚春》等。

===墓誌===
韓愈亦善寫[[墓誌銘|墓誌銘]]，是時“[[长安|长安]]中争为碑志，若市买然。”韓愈文名日盛，達官貴人常求其為[[先人|先人]]撰墓誌銘，[[潤筆|潤筆]]之金甚高，韓愈亦來者不拒。[[司马光|司马光]]《颜乐亭颂》一文指出，韩愈“好悦人以铭志，而受其金”。韩愈寫一篇《谢许受王用男人事物状》，[[润笔|润笔費用]]是“受马一匹，并鞍、衔及白玉腰带一条”。《谢许受韩弘物状》記載曾收得[[韓弘|韓弘]]致贈的“絹五百匹”，相當於四百[[貫|貫]][[文_(貨幣)|錢]]，而時韓愈一個月的[[薪水|薪水]]只有二十五貫錢，當官反成副業。部分“諛墓”之作，在當時頗受譏斥。其友[[劉叉|劉叉]]故意取其[[黃金|黃金]]數斤而去，-{云}-：“此諛墓中人得耳，不若與劉君為壽。”[[元代|元代]][[王若虚|王若虚]]说：“[[韩退之|韩退之]]不善处穷，哀号之语，见于文字。”又说：“退之不忍须臾之穷。”

===反对佛教===
韩愈 在《原道》和《论佛骨表》中，尖锐地抨击了佛教，认为：佛教耗费大量财富，加重了百姓负担，使“民不穷且盗” ；“佛本夷狄之人”，佛徒“口不言先王之法言，身不眼先王之法服，不知君臣之义、父子之情”，让佛教凌驾于儒学之上，有被同化为夷狄的危险。韩愈还认为佛教自外于天下国家，灭弃封建伦常，“子焉而不父其父，臣焉而不君其君，民焉而不事其事”。他主张“人其人，火其书，庐其居”。819 年（元和十四年），[[唐宪宗|唐宪宗]]要将凤翔[[法门寺|法门寺]]的“佛骨”迎至宫中供养，崇佛之风再次兴起，有人“焚顶烧指，百十为群，解衣散钱，自朝至暮，转相仿效，惟恐后时”。韩愈上表反对，主张将佛骨“投诸水火，永绝根本，断天下之疑，绝后代之惑”，从而触怒了唐宪宗，几乎因此被杀。他主张对于僧尼：“请令匹配，即成十万余户，产育男女，十年长养，一纪教训，自然益国，可以足兵”<ref>中国古代史 北京师范大学出版社出版</ref>。

== 評價 ==
* [[蘇軾|蘇軾]]在《潮州韓文公廟碑》盛稱其“匹夫而为百世师，一言而为天下法，是皆有以参天地之化，关盛衰之运。”“独韩文公起布衣，谈笑而麾之，天下靡然从公，复归于正，盖三百年于此矣。文起八代之衰，道济天下之溺。忠犯人主之怒，而勇夺三军之帅。此岂非参天地，关盛衰，浩然而独存者乎？”，并作詩稱讚韓愈：“公昔騎龍白雲鄉，手抉雲漢分天章。天孫為織雲錦裳，飄然乘風來帝旁。下與濁世掃粃糠，西遊咸池略扶桑。草木衣被昭回光，追逐李杜參翱翔。汗流籍湜走且僵，滅沒倒景不可望。作書詆佛譏君王，要觀南海窺衡湘。歷舜九嶷弔英皇，祝融先驅海若藏，約束蛟鱷如趨羊。鈞天無人帝悲傷，謳吟下招遣巫陽。犦牲雞卜羞我觴，於粲荔丹與蕉黃。公不少留我涕滂，翩然披髮下大荒。”
* [[蘇洵|蘇洵]]稱讚韓愈文章“如長江大河，渾浩流轉”。
* [[吳虎臣|吳虎臣]]《能改齋漫錄》卷十謂荊公“不以退之為是”。
* 韓退之所撰寫《[[祭十二郎文|祭十二郎文]]》與[[李密_(西晉)|李密]]的《[[陳情表|陳情表]]》、[[諸葛亮|諸葛亮]]的《[[出師表|出師表]]》並列為中國三大抒情文之一，南宋[[謝枋得|謝枋得]]《[[文章軌範|文章軌範]]》引用[[安子順|安子順]]之說：“讀《出師表》不哭者不忠，讀《陳情表》不哭者不孝，讀《祭十二郎文》不哭者不慈。”
* 韓愈說自己反對閹黨，“日與宦者為敵”<ref>韓愈：《上鄭尚書相公啟》</ref>，但是貞元十三年，韓愈曾作〈送汴州監軍俱文珍序〉，對宦官[[俱文珍|俱文珍]]歌頌備至。後來又作《順宗實錄》，對俱文珍亦多加褒辭。[[魏了翁|魏了翁]]嘲笑他“韓公每是有求於人，其詞輒卑諂不可據”。
* 韓愈善寫“谀墓”之文，清初[[顾炎武|顾炎武]]在书信中评他：“韩文公文起八代之衰，若但作《原道》、《原毁》、《争臣论》、《平淮西碑》、《张中丞传后序》诸篇，而一切铭状概为谢绝，则诚近代之泰山北斗矣；今犹未敢许也”。
* [[张耒|张耒]]论韩愈“以为文人则有余，以为知道则不足”（《韩愈论》）。
* [[朱熹|朱熹]]指责韩愈“裂道与文以为两物”（《读唐志》）。
* [[章學誠|章學誠]]稱韓愈的碑誌文是「心識古人源流，隨時通其變化」（《[[文史通義|文史通義]]‧墓銘辨例》）。
* [[茅坤|茅坤]]说：“世之论韩文者，共曾称碑蒜；予独以韩公碑文多奇崛险谲，不得《史》《汉》叙事法，故于风神处或少道逸。”（《唐宋八大家支钞·论例》）
* [[章太炎|章太炎]]说：“韩对死生利禄之念，刻刻不忘：登华山大哭，作《送穷文》，是真正的证据。”韩愈登华山，“度不可返，乃作遗书，发狂恸哭”，最後被华阴县令救下。
* [[钱锺书|钱锺书]]《[[谈艺录|谈艺录]]》认为：“韩昌黎之在北宋，可谓千秋万岁，名不寂寞矣。”又說“古来薄韩者多姓王”。
* [[陳寅恪|陳寅恪]]在《論韓愈》中論及韓愈排斥佛教，“呵抵釋迦，申明夷夏之大防”。
* [[周作人|周作人]]對韓退之則不以為然：“講到韓文我壓根兒不能懂得他的好處”，“總是有舊戲似的印象”，“但見其裝腔作勢，搔首弄姿而已”<ref>周作人：《談韓退之與桐城派》</ref>。他在《秉烛谈谈韩文》又說：“假如我们不赞成统治思想，不赞成青年写新八股，则韩退之暂时不能不挨骂，盖窃以为韩公实系该运动的祖师，其势力至今尚弥漫于全国上下也”、“如有人愿学滥调古文，韩文自是上选”。
* [[錢仲聯|錢仲聯]]：“韓愈的散文，氣勢充沛，縱橫開合，奇偶交錯，巧譬善喻，或詭譎，或嚴正，具有多樣的藝術特色。”
*[[毛泽东|毛泽东]]1936年在[[保安|保安]]会见[[斯诺|斯诺]]时说：“学校里有一个国文老师（[[袁吉六|袁吉六]]），学生给他起了个“袁大胡子”的外号。他嘲突我的文章，说它是新闻记者的手笔。他看不起我祝为楷模的[[梁启超|梁启超]]，认为他是一个半通不通的人。我不得不改变我的文风，去钻研韩愈的文章，学会了古文的措词。所以，多亏袁大胡子，今天我如果需要的话，仍然能够写出一篇过得去的古文。”晚年毛泽东读[[欧阳修|欧阳修]]《[[新唐书|新唐书]]·[[李汲|李汲]]传》时写下批语：“韩愈文集，为李汲编辑得全，欧阳修得之于[[随县|随县]]，因以流传，厥功伟哉。”<ref>{{cite web |url=http://www.xiexingcun.com/maozedong/dsbj015.htm |title= 我改变文风，去钻研韩愈的文章（读韩愈《韩昌黎诗文全集》）|accessdate=|work=|publisher=毛泽东读书笔记解析|language= zh-hans}}</ref>
* [[錢穆|錢穆]]：其排釋老而返之儒，倡言師道，確立道統，則皆宋儒之所濫觴也。嘗試論之，唐之學者，治詩賦取進士第，得高官，卑者漁獵富貴，上者建樹功名，是謂入世之士。其遯跡山林、棲心玄寂，求神仙，溺虛無，歸依釋老，則為出世之士……獨昌黎韓愈氏，進不願為富貴功名，退不願為神仙虛無，而倡言乎古之道……此皆宋學精神也，治宋學者首昌黎，則可不昧乎其所入矣。<ref>錢穆：《中國近三百年學術史》上冊</ref>
*[[陳來|陳來]]認為韓愈和他的弟子李翱提出的復興儒家的基本口號與發展方向，確乎是北宋慶曆時期思想運動的先導。韓愈在他的道統傳承的說法中把孟子說成孔子的繼承人，把孟子的地位大大提高，影響後來宋明理學的研究。<ref>陳來《宋明理學》P17-19</ref>

== 軼聞 ==
{{wikisource author|韓愈|韩愈}}
* 韓愈的畫像多半將他畫為有著美髯、戴著[[紗帽|紗帽]]的男子，然而據[[沈括|沈括]]《[[夢溪筆談|夢溪筆談]]》（卷四）所記，韓愈其實是位肥胖且寡髯的男子，而現在的形象其實是與五代十國時的[[韓熙載|韓熙載]]搞混了。因為韓熙載[[諡號|諡號]]文靖，江南人稱呼他為'''韓文公'''，而韓愈亦被尊稱為韓文公的關係。”<ref>沈括《夢溪筆談‧卷四‧辯證二》「世人畫韓退之，小面而美髯、著紗帽，此乃江南韓熙載耳。尚有當時所畫，題誌甚明。熙載諡文靖，江南人謂之韓文公，因此遂謬以為退之。退之肥而寡髯。元豐中，以退之從享文宣王廟，郡縣所畫，皆是熙載，後世不復可辯，退之遂為熙載矣。」</ref>

* 五代时人[[陶榖|陶榖]]《[[清異錄|清異乡錄]]》载：昌黎公逾晚年頗親脂粉，故可服食；用硫磺末攪粥飯，啖雞男，不使交，千日，烹庖，名 火靈庫 ，公間日進一隻焉，但是，始亦見功，終致絕命。

* 與[[孟郊|孟郊]]齊名的[[詩人|詩人]][[賈島|賈島]]在赴試時，嘗騎一[[驢|驢]]趕路，於路上突詩興一發，做得一詩句:「鳥宿池邊樹，僧推月下門」。然而不知要用「'''推'''」字為好，抑或是「'''敲'''」字為佳，於是在驢上做動作。正在思惱時，正遇韓愈出巡，而賈島不知此事，渠之驢竟走至隊伍之間，於是被韓愈的護衛捉拿，而後被押帶到韓愈暫宿的[[驛所|驛所]]訊問。韓愈問賈島：「為何衝撞本尹？」島回：「大人，下生實不知大人出，於做一詩，正琢磨用字，因而失禮，望大人恕罪！」愈：「是何詩句？」島:「大人，學生做『鳥宿池邊樹，僧推月下門』，不知欲用『推』字，或為『敲』字，甚為苦惱，故此失禮。」韓愈思忖良久，回曰：「做『'''敲'''』字佳矣。『'''推'''』表有盜竊之意，是罪矣。故以『敲』字甚好」。成了「'''推敲'''」一詞的來源。
