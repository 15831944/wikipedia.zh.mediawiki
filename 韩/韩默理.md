'''費迪南德·哈默'''（{{lang-nl|Ferdinand Hamer}}，1840年8月21日－1900年7月24日），漢名'''韩默理'''，比利时[[圣母圣心会|圣母圣心会]]会士，[[天主教|天主教]]西南[[蒙古|蒙古]]教区主教，1900年在[[义和团运动|义和团运动]]中被杀。

== 故乡比利时 ==
1840年，韩默理生于[[荷兰|荷兰]][[奈梅亨|奈梅亨]]。1864年升为神父，同年，[[罗马教廷|罗马教廷]]把蒙古教区划归1862年刚刚成立的比利时[[圣母圣心会|圣母圣心会]]，一个无人知晓但充满进取心的传教会。

== 长城以外 ==

1865年12月，[[南怀义|南怀义]]（[[圣母圣心会|圣母圣心会]]的创始人）、韩默理和另外3名不顾一切的传教士，穿越[[长城|长城]]，来到蒙古教区的中心西湾子村（后来以教堂为中心发展成河北[[崇礼|崇礼]]县城），接管了原属法国[[遣使会|遣使会]]的教务。

== 河西走廊 ==

1878年，罗马教廷将[[甘肃|甘肃]]划出[[陕西|陕西]]教区。1882年罗马教廷委派韩默理为甘肃主教，主教堂设在[[武威|武威]]城西松树庄。　

== 二十四顷地 ==

由于信徒大量增加，1883年，罗马教廷把广阔的蒙古教区分为东蒙古、中蒙古和西南蒙古三个教区。1889年，韩默理调任西南蒙古教区主教，主教座堂起初在三道河，1898年韩默理将主教座堂迁至河套地区萨拉齐厅（[[土默特右旗|土默特右旗]]）二十四顷地。在那一带有天主教1890年以后陆续购进的大片地产（多达260多顷），有许多从关内前来谋生的汉族农民，租种教会的土地，其中有不少因此加入了天主教。

== 庚子年 ==

1900年7月5日（六月初九日），韩默理面对敌视天主教的形势，和萨拉齐厅境内6名外籍教士，在二十四顷地主教座堂内祈祷后，决定在该教堂固守。7月6日，韩主教命令6名外籍传教士逃往三道河，自己单独留下陪伴在此避难的数千名教徒。

此后萨厅东段所有教堂均被[[义和团|义和团]]攻破，并全部焚毁。每天都有惊魂未定的教徒前來投奔二十四顷地主教座堂。7月19日（六月二十三日）天未亮时，归化城派出的200名清军马队趁守卫教堂的教徒疲劳，攻进村内，大批天主教徒被杀，包括圣婴会所有的孩子，义和团则放火烧毁村庄与教堂。韩默理被捕后，遭义和团“手足合繫，贯以竹竿，抬之遊街”，受尽百般凌辱，然后与“教友五人，同置一车……跪于车中，头上有大刀一，横悬颈后，车动则刀切六人之颈”，运到托克托厅城，又“以铁索穿肩骨，囚以笼……背上插一小旗，上书‘老洋魔’三字”，去河口镇游街示众。7月24日，回到托城，又往陶村[[游街示众|游街示众]]，最后在托厅南滩被脱去衣服，用浸满油的棉布擦拭全身，随后放火烧死，首级悬于木<ref>《拳祸记》</ref>。
