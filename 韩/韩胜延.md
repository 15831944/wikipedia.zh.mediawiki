==生平==
1963年5月出生於河北涿鹿县，1979年7月入伍，1979年7月历任空军第一航空预备学校学员，空军第十三航空学校学员，空军航空兵第三师飞行员，飞行中队长，空战射击主任，飞行大队长，副团长，团长，副师长，南京军区空军司令部军训处处长，2002年12月任南京军区空军航空兵第29师师长，2003年10月任南京军区空军航空兵第3师师长，2008年2月任成都军区空军副参谋长，2011年2月任兰州军区空军参谋长，2013年2月任成都军区空军副司令员，2014年2月任空军试验训练基地司令员，2016年1月任中國人民解放軍西部战区专职空军副司令员；2009年7月晋升空军少将军衔，2017年7月晋升空军中将军衔。

1990年代初，韩胜延曾随团留学国外。他长期在[[中国人民解放军空军|中国人民解放军空军]]服役，2006年被评为全军优秀指挥军官。曾任[[中国人民解放军成都军区空军|中国人民解放军成都军区空军]]副参谋长。2011年7月任[[中国人民解放军兰州军区空军|中国人民解放军兰州军区空军]]参谋长。2013年6月任中国人民解放军成都军区空军副司令员。后任[[中国人民解放军空军试验训练基地|中国人民解放军空军试验训练基地]]司令员。2016年，升任新组建的[[中国人民解放军西部战区|中国人民解放军西部战区]]副司令员<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1533297 |title =空军某试验训练基地原司令员韩胜延少将升任西部战区副司令员  |author =  |date = 2016-09-23  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref name=caixin/>。2018年7月，任[[中国人民解放军中部战区空军|中国人民解放军中部战区空军]]司令员<ref>{{cite web |title=中部战区空军司令员换将：韩胜延接替庄可柱 |url=http://news.163.com/18/1202/15/E21FUEUL0001899N.html |website=网易 |accessdate=2018-12-04 |date=2018-12-02}}</ref>。
