'''韩俊'''（{{bd|1963年|12月||}}），[[山东|山东]][[高青|高青]]人。1989年5月参加工作，[[中国共产党|中国共产党]]党员。农村经济及农村发展研究专家。[[山东农业大学|山东农业大学]]农业经济系本科毕业，[[西北农业大学|西北农业大学]]农业经济系硕士、博士毕业，研究生学历，农学博士学位。

==生平==
1979年9月到1983年7月，在[[山东农业大学|山东农业大学]]农业经济系学习，获学士学位。1983年9月到1986年7月，为[[西北农业大学|西北农业大学]]农业经济系硕士研究生。1986年7月到1989年5月，为西北农业大学（现[[西北农林科技大学|西北农林科技大学]]）农业经济系农业经济管理专业博士研究生，毕业获得农学博士学位<ref name=han/>。

1989年5月，在[[国务院农村发展研究中心|国务院农村发展研究中心]]工作。1989年12月，调到[[中国社会科学院|中国社会科学院]]农村发展研究所任职，历任《中国农村经济》杂志社编辑部副主任、主任、副总编、社长、副所长。2001年3月，任[[国务院发展研究中心|国务院发展研究中心]]农村经济研究部部长（其间，2006年3月到2007年1月，在[[中共中央党校|中共中央党校]]第22期[[中央党校一年制中青年干部培训班|一年制中青年干部培训班]]学习）。2008年10月，任国务院发展研究中心党组成员、农村经济研究部部长。2010年11月，任国务院发展研究中心副主任、党组成员<ref name=han/>。

2014年10月，任[[中央财经领导小组办公室|中央财经领导小组办公室]]副主任<ref name=han>{{Cite web |url =  http://cpc.people.com.cn/gbzl/html/121001639.html|title =  韩俊|author =  |date =  |publisher = 中国领导干部资料库 |language =  |accessdate =2017-05-15  |quote =  }}</ref>。2017年4月，兼任[[中央农村工作领导小组办公室|中央农村工作领导小组办公室]]主任<ref name=ifeng>{{Cite web |url = http://news.ifeng.com/a/20170515/51095465_0.shtml |title =韩俊任中央农村工作领导小组办公室主任  |author =  |date = 2017-05-15 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。2018年3月，任新成立的[[中华人民共和国农业农村部|中华人民共和国农业农村部]]党组副书记、副部长
