'''韩少功'''（{{bd|1953年|1月1日|||catIdx=Han韩少功}}），[[湖南|湖南]][[长沙|长沙]]人，[[中国|中国]][[作家|作家]]。

== 简介 ==
韩少功出生于[[湖南|湖南]][[长沙|长沙]]，1968年初中毕业后到[[湖南|湖南]]省[[汨罗|汨罗]]县下乡插队，1974年调汨罗县文化馆工作，1978年考入[[湖南师范大学|湖南师范大学]]中文系，1982年毕业，在湖南省总[[工会|工会]]工作，任《主人翁》杂志编辑、副总编。<ref name="smex " >{{cite web |url=http://www.library.hn.cn/hxrw/xdrw/wxlmr/hanshaogong.htm |title=韩少功 |accessdate= |work=|publisher=湖大图书馆官方网 |language = zh-hans}}</ref> 1985年在[[武汉大学|武汉大学]]英文系进修，随后任湖南省[[作家协会|作家协会]]专业作家，并当选为[[中国作家协会|中国作家协会]]理事。<ref name="smex " /> 1988年调到海南省，任《[[海南纪实|海南纪实]]》杂志主编。<ref name="smex " /> 1995年任《[[天涯_(杂志)|天涯]]》杂志社长。<ref name="smex " /> 1996年任海南省[[作家协会|作家协会]]主席。<ref name="smex " /> 2000年任海南省[[文联|文联]]主席，曾兼任省政协委员、常委和省人大代表等职。他现在过着“半世俗半隐居的生活”，一半时间生活在汨罗农村，一半时间生活在城市。

他的作品主要以文革时的有关经历为素材。八十年代他首倡[[寻根文学|寻根文学]]，是中国[[新时期文学|新时期文学]]代表作家之一。他的作品曾多次获得中国和外国的奖项。

*1983年：《飞过蓝天》（中短篇小说集）　
*1986年：《诱惑》（中短篇小说集）　　　
*1988年：《空城》（中短篇小说集） 　
*1989年：《谋杀》（中短篇小说集）　　
*1993年：《[[爸爸爸|爸爸爸]]》（中篇小说集）　　
*1994年：《北门口预言》（中短篇小说集）
*1996年：《[[马桥词典|马桥词典]]》（长篇小说）　　　
*2002年：《[[暗示|暗示]]》（长篇小说）
*2005年：《[[报告政府|报告政府]]》（中短篇小说集）
*2013年：《[[日夜书|日夜书]]》（长篇小说）

===散文===
*1986年：《面对神秘而空阔的世界》（随笔集）
*1993年：《夜行者梦语》（散文随笔集）
*1994年：《圣战与游戏》（随笔集）
*1996年：《心想》（散文集）；《灵魂的声音》（散文集）
*2003年：《完美的假定》，昆仑出版社
*2006年：《山南水北——八溪峒笔记》（散文集）

===电影剧本===

*1999年：《[[惶然录|惶然录]]》

== 所获奖誉 ==
*1980年：中国全国优秀短篇小说奖

*1998年：上海中长篇小说大奖
*1998年：台湾最佳图书奖
*2000年：《[[马桥词典|马桥词典]]》入选海内外专家共同推选的“[[二十世紀中文小說一百強|二十世紀中文小說一百強]]”

*2002年：憑《[[暗示|暗示]]》獲 第二屆 [[華語文學傳媒大獎|華語文學傳媒大獎]] 年度小說家獎
*2007年：《山南水北》获第四届[[鲁迅文学奖|鲁迅文学奖]]之全国优秀散文杂文奖
*2011年：第二届[[纽曼华语文学奖|纽曼华语文学奖]]
*2014年：《日夜書》获第五届[[紅樓夢獎|紅樓夢獎]]專家推薦獎
