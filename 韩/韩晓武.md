'''韩晓武'''（{{bd|1958年|11月||}}）[[北京市|北京市]]人，[[中华人民共和国|中华人民共和国]]政治人物。

==生平==
毕业于[[日本|日本]][[新潟大学|新潟大学]]法学专业，获得法学硕士学位。[[中国共产党|中国共产党]]党员。2008年12月，任[[第十一届全国人大常委会|第十一届全国人大常委会]]副秘书长<ref name=liuhan>{{Cite web |url =http://news.xinhuanet.com/newscenter/2008-12/27/content_10568406.htm  |title = 十一届全国人大常委会第六次会议在京闭会 |author =  |date = 2008-12-27 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref>{{cite news|title=全国人大常委会副秘书长韩晓武|url=http://lianghui2009.people.com.cn/GB/145766/146770/148515/148517/8931541.html|accessdate=2013-03-06|newspaper=人民网}}</ref>。2013年，任[[第十二届全国人大常委会|第十二届全国人大常委会]]委员、[[全国人大内务司法委员会|全国人大内务司法委员会]]委员<ref>{{cite news|title=全国人大代表信息-韩晓武|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=2008120052|accessdate=2013-03-06|newspaper=全国人大网}}</ref>，并继续任全国人大常委会副秘书长<ref name=wangwanbin>{{Cite web |url = http://news.xinhuanet.com/renshi/2013-03/19/c_115075489.htm |title =王万宾等被任命为全国人大常委会副秘书长  |author =  |date =2013-03-19  |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。
