'''韩国铁道公社331000系电力动车组'''（{{kor|諺=한국철도공사 331000호대 전동차|漢=韓國鐵道公社 331000號帶 電動車}}）是[[韩国铁道公社|韩国铁道公社]]的一款[[通勤型电力动车组|通勤型电力动车组]]，在[[首都圈電鐵京義·中央線|首都圈電鐵京義·中央線]]运营。

== 概况 ==
331000系电力动车组是[[广域电铁|广域电铁]]的列车，是交流用车辆，列车自2009年7月1日开始在[[首都圈電鐵京義·中央線|首都圈電鐵京義·中央線]]使用。列车客室车门为電動内藏门，每节车厢共4对车门，列车两端拥有[[紧急出口|紧急出口]]。本款列车共27组，配属于[[汶山车辆基地|汶山车辆基地]]。

== 列车特点 ==
本款列车外观与[[韩国铁道公社361000系电力动车组|361000系]]相似，不过其外涂装和客室座位颜色不同。

== 使用情况 ==
===第1批===
编号33101-33113，于2009年生产，列车材料为[[铝合金|铝合金]]，车内安装有[[液晶电视|液晶电视]]；而所有編組均為8卡編組。

===第2批===
编号33114-33127，于2012年生产，列车材料为[[不锈钢|不锈钢]]，车内安装有[[液晶电视|液晶电视]]；除了33114-33122為4卡編組外，其他編組均為8卡編組。
