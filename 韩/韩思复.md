'''韩思复'''（{{bd|652年||725年|catIdx=H}}），[[表字|字]]'''绍出'''，京兆长安人。[[唐朝|唐朝]][[官员|官员]]。

== 家世及早期仕途 ==
祖父韩伦，[[贞观_(唐朝)|贞观]]年间为左卫率，赐爵'''长山县男'''。<ref name=BT101>[[:s:旧唐书/卷101|《旧唐书》卷一百零一]]</ref><ref name=NBT118>[[:s:新唐书/卷118|《新唐书》卷一百一十八]]</ref>

韩思复年少丧父。十岁时，其母对他说父亲去世的情状，他感触呜咽几乎断气，韩伦因此特别喜爱他，曾说：“此儿必大吾宗。”家富有，从未吝惜金玉、车马、玩好之物。好学，举[[秀才|秀才]]中高第，承袭祖父封爵。<ref name=BT101/>[[永淳|永淳]]年间，家道中落，遭遇饥荒，京兆人杜瑾送韩思复一百匹绫，韩思复正穷得两三天才有一天的粮食，却始终没有将绫拆封。<ref name=NBT118/>

调任梁府仓曹参军，正逢大旱，就开仓赈民，被州府劾责，说：“不如救活他们，别让他们为盗贼。”州府不能责难。又任[[汴州|汴州]]司户参军，为政宽恕，不行杖罚。在任上因[[丁母忧|丁母忧]]去职，家贫，卖柴服完丧期。当时[[姚崇|姚崇]]为夏官侍郎、[[宰相|宰相]]，深嘉叹之，擢授其为司礼博士。<ref name=BT101/>[[圣历|圣历]]年间，被吏部郎[[郑杲_(唐朝)|郑杲]]任为[[太常博士|太常博士]]，定南郊仪注，建昌王[[武攸宁|武攸宁]]丧母，请求鼓吹，韩思复坚持不可，于是没请求到。韩思复排挤奸邪小人，守大体，国家赖之。郑杲同时任[[元希声|元希声]]为京兆士曹，曾对人说：“今年掌选，得韩元二子，则吏部不负朝廷矣。”<ref>《[[册府元龟|册府元龟]]》</ref>由司礼博士五迁至礼部郎中。[[神龙_(年号)|神龙]]二年（706年）[[唐中宗|唐中宗]][[驸马|驸马]][[王同皎|王同皎]]获罪被杀，韩思复因是王同皎所荐，受到连累，贬[[始州|始州]]长史。迁[[滁州|滁州]][[刺史|刺史]]，州有铜矿，百姓铲凿很辛苦，韩思复将它卖到边境，省下很多费用。州署五次生出黄芝，百姓刻碑歌颂其祥瑞。改任[[襄州_(西魏)|襄州]]。<ref name=NBT118/>

== 唐睿宗年间 ==
[[唐睿宗|唐睿宗]][[景龙|景龙]]年间，累迁[[给事中|给事中]]。睿宗建造景龙观，韩思复上谏认为内乱刚平定，不宜大兴土木，未果。中宗子谯王[[李重福|李重福]]在东都[[洛阳|洛阳]]作乱，任左散骑常侍[[严善思|严善思]]为[[礼部尚书|礼部尚书]]，知吏部选事。后李重福事败，[[景云|景云]]元年（710年），严善思也因涉事被下诏狱，大理寺奏称严善思在任[[汝州|汝州]]刺史时就素与李重福交游，被召至京师也不说李重福谋逆，只奏称“东都有兵气”，是隐匿谋反事，请求处以[[绞刑|绞刑]]。韩思复驳奏，指出严善思先前在唐中宗[[韦皇后_(唐中宗)|韦皇后]]专权图谋篡位时投靠相王（即唐睿宗），即使与李重福交游也是为了谋害韦氏，且被下敕追拿时就投案了，如果心怀谋逆之心为何不逃命？建议付刑部集群官议定奏裁，慎重量刑。<ref>[[:s:驳严善思绞刑奏议|《驳严善思绞刑奏议》]]</ref>当时议者多认为严善思应被赦免，有司仍坚持先前所议请求诛之。韩思复又恳切切直地驳斥。<ref>[[:s:又驳严善思绞刑奏议|《又驳严善思绞刑奏议》]]</ref>睿宗纳其奏，免严善思死，配流[[静州_(武周)|静州]]。严善思也没有对韩思复道谢，时人称赞严善思为长者。<ref>[[:s:旧唐书/卷191|《旧唐书》卷一百九十一]]</ref>韩思复也没有以其恩人自居。<ref>[[:s:新唐书/卷204|《新唐书》卷二百零四]]</ref>曾任大理卿，执行法律时出错，被权臣弹劾，被给事中[[崔沔|崔沔]]宽恕。<ref>[[:s:通议大夫守太子宾客东都副留守云骑尉赠尚书左仆射博陵崔孝公宅陋室铭记|颜真卿《通议大夫守太子宾客东都副留守云骑尉赠尚书左仆射博陵崔孝公宅陋室铭记》]]</ref>不久转中书舍人，数次上疏陈说政事得失，多被纳用。<ref name=BT101/><ref name=NBT118/>

== 唐玄宗年间 ==
[[开元|开元]]初年，为[[谏议大夫|谏议大夫]]。二年（714年）二月，姚崇被任为持节灵武道行军大总管，北伐[[后突厥|突厥]]，韩思复被任为兼行军司马。<ref>[[:s:命姚崇等北伐制|苏颋《命姚崇等北伐制》]]</ref>四年（716年），山东蝗虫大起，[[姚崇|姚崇]]时任中书令，奏请遣使分别去河南、河北诸道杀蝗虫而埋之。韩思复以为蝗虫是天灾，应当修德祈祷消灾，恐怕不是人力所能消除的，上疏建议[[唐玄宗|唐玄宗]]悔过自责、派使者宣慰、召至公之人上下同心、君臣一德，停止派出驱蝗使等，收揽人心。<ref>[[:s:谏捕蝗疏|《谏捕蝗疏》]]</ref>玄宗深以为然，将韩思复的奏疏给姚崇。姚崇请求遣韩思复去山东察看被蝗虫所损之处，韩思复回来后奏以实情。<ref>[[:s:旧唐书/卷37|《旧唐书》卷三十七]]</ref>姚崇又请求令[[监察御史|监察御史]][[刘沼|刘沼]]复查，刘沼揣摩姚崇心思，鞭挞百姓，修改情状回奏，河南数州因而不得免灾。韩思复遂为姚崇所排挤，出为[[德州_(山东)|德州]]刺史，八年（720年）前后在任。约九年（721年）、十年（722年）转[[绛州|绛州]]刺史。<ref>[[郁贤皓|郁贤皓]]《[[唐刺史考全编|唐刺史考全编]]》</ref>入为[[黄门侍郎|黄门侍郎]]，加[[银青光禄大夫|银青光禄大夫]]。十一年（723年）春，玄宗北巡，以韩思复为行在巡问赈给大使。韩思复作于汾阴祭皇地祇所用乐章。<ref>[[:s:旧唐书/卷30|《旧唐书》卷三十]]</ref>代[[裴漼|裴漼]]为[[御史大夫|御史大夫]]。<ref name=BT101/><ref name=NBT118/>

韩思复性恬淡，爱好谈玄，安心于实行仁道、亲行正道，不能担任纠察官员之职。不久，转[[太子宾客|太子宾客]]。进爵'''伯爵'''。累迁吏部侍郎。复为襄州刺史，政绩闻名天下，被代任回京，仍为太子宾客。十三年卒。<ref name=BT101/>玄宗亲自题其碑“有唐忠孝韩长山之墓”。故吏[[卢僎|卢僎]]、襄州人[[孟浩然|孟浩然]]立其碑于石岘山。谥号'''文'''。<ref name=NBT118/>

之前韩思复年轻时曾随太白山隐士郑仁杰、李无为游玩，二人说：“你识清貌古，可惜做不到宰相。”<ref name=NBT118/>

== 影响 ==
后来玄宗降罪于直言的谏官，起居郎[[吴兢|吴兢]]上疏劝谏，举玄宗登基初年[[褚无量|褚无量]]、[[张廷珪|张廷珪]]、韩思复、[[辛替否|辛替否]]、[[柳泽|柳泽]]、[[袁楚客|袁楚客]]等数次上疏争论时政得失的例子，要玄宗虚心纳谏。<ref>[[:s:新唐书/卷132|《新唐书》卷一百三十二]]</ref><ref>[[:s:上元宗皇帝纳谏疏|《上元宗皇帝纳谏疏》]]</ref>
