'''韩寓群'''（{{bd|1943年|9月|catIdx=Han韩寓群}}），[[江苏省|江苏省]][[徐州市|徐州市]][[睢宁县|睢宁县]]人；[[南京工业大学|南京化工学院]]毕业。1975年12月加入[[中国共产党|中国共产党]]；曾任[[中共|中共]][[山东|山东]][[省委书记|省委副书记]]，[[山东省|山东省]][[省长|省长]]。现任第十一届[[全国人大常委会|全国人大常委会]]委员、财政经济委员会副主任委员。

== 早年经历 ==
1967年7月大学毕业后，韩寓群被分配到[[山东|山东]]“[[济宁|济宁]]合成洗涤剂厂”工作；从普通技术员做起，历任车间副主任、主任，副厂长，工程师，厂党委副书记、兼厂长等职。

1980年代初，[[中共|中共]]全面推行干部年轻化的政策，韩寓群和很多在国有企业成长起来，既有学历、又懂技术的年轻厂长一样，被列为地方党政干部的培养对象。1983年，已在“[[济宁|济宁]]合成洗涤剂厂”前后工作十三年的韩寓群，被任命为[[中共|中共]][[济宁|济宁]][[市委书记|市委副书记]]；1986年，出任[[济宁市|济宁市]][[市长|市长]]；五年后，晋升[[中共|中共]][[济宁|济宁]][[市委书记|市委书记]]。

== “三起三落” ==
1993年4月，韩寓群出任[[山东省|山东省]]政府经济委员会主任；七个月后，在当年11月举行的[[中共|中共]][[山东省|山东省]]六届一中全会上，当选省委常委，并出任省委统战部部长。然而，在1995年6月转任[[山东省|山东省]]副省长后，韩寓群不仅不再担任省委常委；而且在八位副省长中，仅排名第六位。 直到2000年，常务副省长[[宋法棠|宋法棠]]调[[黑龙江省|黑龙江省]]任职后，韩寓群才再次入选中共山东省委常委，并接任常务副省长职务。2000年底，因对[[烟台11·24特大海难|烟台11·24特大海难]]事故负有领导责任，受到行政记过处分。

按照副省级干部任职年龄的规定，到2002年6月[[中共|中共]][[山东省|山东省]]第八次党代会召开的时候，已近59岁的韩寓群，没有继续留任省委常委，更没有在当年11月召开的[[中共|中共]]十六大上，入选[[中央委员会|中央委员会]]。然而，就在[[山东省|山东省]]进行换届选举时，2003年1月9日，[[中共中央|中共中央]]决定再度启用韩寓群，任命他为[[中共|中共]][[山东|山东]][[省委书记|省委副书记]]。韩寓群第三次进入中共山东省委常委会。三天后，接替[[张高丽|张高丽]]出任[[山东省|山东省]][[省长|代理省长]]；并在2003年4月，正式当选[[山东省|山东省]][[省长|省长]]。

作为主政一方的中共高级行政官员，既不是中央委员、也不是候补中央委员，韩寓群的职务经历在现今的中共体制中，是不多见的。但这样的安排，也使得韩寓群的政治生命被延续了十年。

== 退休 ==
2007年6月13日，年届64岁的韩寓群，在山东省人大常委会十届二十七次会议上，辞去了省长的职务；两周后，被增补为第十届[[全国人大常委会|全国人大常委会]]委员，财政经济委员会副主任委员。2008年3月，在第十一届[[全国人大|全国人大]]一次会议上，连任了人大常委会委员和财政经济委员会副主任委员的职务。

2013年7月26日，任[[中国重汽|中国重汽]]（香港）有限公司[[独立董事|独立董事]]<ref>{{cite web|title=贵州和山东前省长加盟中国重汽 年薪18万元|url=http://news.ifeng.com/mainland/detail_2013_07/28/27996562_0.shtml|publisher=凤凰网|accessdate=2013-07-28}}</ref>。
