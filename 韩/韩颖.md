'''韩颖'''（{{bd|1973年|12月29日}}），北京市人，维权人士、人权捍卫者、微笑公益创始人<ref name=":0">{{Cite web|url = http://www.rfa.org/mandarin/Xinwen/10-11092014174509.html|title = 因声援香港 微笑公益发起人韩颖被正式逮捕|author = 吴晶|publisher = 自由亚洲}}</ref>，现居北京市。曾因支持香港真普选被羁押8个月。
** 2003年，因为祖宅面临非法拆迁而辞职维权。
** 2009年，与其他公益人士在北京南站发放食品救助访民。<ref>{{Cite web|url = https://www.youtube.com/watch?v=AR08Lzx7k1c|title = 阳光公益在北京南站附近救助访民|accessdate = |author = |date = 07/04/2010|publisher = 博讯}}</ref>
** 2010年，围观黑监狱，参与打击黑监狱的活动。
** 2010年，开始帮助其他拆迁维权人士，代理多起行政诉讼。<ref>{{Cite web|url = http://wqw2010.blogspot.com/2011/05/blog-post_7758.html|title = 北京海淀房管局拒绝信息公开，韩颖向法院起诉|date = 2011-05-05|author = 甘宕|publisher = 维权网}}</ref>
** 2011年，任[[公盟|公盟]]行政助理。
** <ref>{{Cite web|url = http://www.boxun.com/news/gb/china/2011/12/201112280639.shtml|title = 博讯推出2011年中国维权运动“25”|accessdate = |author = 博讯|date = 2011-12-28|publisher = 博讯网}}</ref>2011年作为独立参选人参加北京市区县人大代表选举，海淀区国保曾试图绑架她，因为群众的救援导致国保绑架失败。在参选期间多次受到骚扰、抹黑，甚至被警察堵门，不允许外出。<ref>{{Cite web|url = http://wqw2010.blogspot.com/2011/11/blog-post_9190.html|title = 北京独立参选人韩颖受到骚扰威胁|author = 刘涛|publisher = 维权网|date = 11/02/2011}}</ref>   <ref>{{Cite web|url = http://soundofhope.org/node/46021/print|title = 竞选期间 北京人大独立参选人再失自由|date = 2011-10-30|author = 李亦菲|coauthors = 李阳|publisher = 希望之声}}</ref>  <ref>{{Cite web|url = http://www.canyu.org/n36544c6.aspx|title = 北京独立参选人韩颖又一次遭到警察围追骚扰|author = 何德普|date = 2011-12-14|publisher = 参与网}}</ref><ref>{{Cite web|url = http://m.voachinese.com/a/article-20111013-deputy-harressed-131800843/789056.html|title = 北京公民谈参选人大代表遭遇|author = 申华|date = 2011-10-13|publisher = 美国之音}}</ref>
** 2012年，发起成立公益机构微笑公益，关注儿童权利。<ref>{{Cite web|url = http://hanyingha.blogspot.com/2013/01/blog-post.html|title = 微笑公益，温暖人心|author = 韩颖|date = 01/03/2012}}</ref>
