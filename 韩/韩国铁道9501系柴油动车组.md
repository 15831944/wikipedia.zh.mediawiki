'''韩国铁道9501系柴油动车组'''，或称'''通勤型柴油动车组'''（[[朝鲜语|朝鲜语]]：통근형 디젤 동차，[[朝鲜汉字|朝鲜汉字]]：{{lang|ko|通勤型디젤動車}}；[[英语|英语]]：Commuter Diesel Car，简称'''CDC'''）。是[[韩国铁道公社|韩国铁道公社]]的一款[[柴油动车组|柴油动车组]]，现在普遍运营于韩国尚未电气化的干线铁路区段及部分支线铁路上。

== 概况 ==
本列车开始于1996年投入使用，主要是为了取代已经老化的旧式[[柴油动车组|柴油动车组]]和已经停用的部分[[铁路客车|铁路客车]]，本列车与[[韩国铁道1000系电力动车组|韩国铁道1000系电力动车组]]的第3代列车外观相似，只不过在列车两侧无紧急出口。目前[[韩国铁道公社|韩国铁道公社]]总共保有131辆列车，全部由[[大宇重工业|大宇重工业]]负责制造。在2003年2月18日[[大邱地铁纵火案|大邱地铁纵火案]]发生后，列车材料全部改为了防火材料。

== 营运情况 ==
目前列车在韩国分别以[[统一号|统一号]]、[[无穷花号|无穷花号]]和[[观光列车|观光列车]]的名义运营。
=== 统一号 ===
{{main|统一号}}
本列车是为了取代[[Bidulgi号|Bidulgi号]]和[[统一号|统一号]]客车而编入。最初在1996年于[[东海南部线|东海南部线]]和[[京釜线|京釜线]]的[[海云台站|海云台]]-[[龟浦站|龟浦]]运行，在1997年6月16日又开始运行于[[京义线|京义线]]的[[首尔站|首尔]]-[[汶山站|汶山]]以及[[京元线|京元线]]的[[议政府站|议政府]]-[[新炭里站|新炭里]]，1998年之后又运行于[[湖南线|湖南线]]和[[庆全线|庆全线]]。现在仅运行于[[京义线|京义线]]和[[京元线|京元线]]的部分区间上。

=== 无穷花号 ===
{{main|无穷花号}}
本列车是为了取代[[韩国铁道9211系柴油动车组|9211系]]而编入。2008年4月15日开始在[[京釜线|京釜线]]和[[庆全线|庆全线]]运行。现在运行于韩国南部的大部分线路上。

=== 观光列车 ===
目前，观光列车被分为了[[海列车|海列车]]和[[庆北观光循环主题列车|庆北观光循环主题列车]]，各1组，分别宣传地方的旅游主题。现在运行于韩国东部的部分线路上。

== 编组 ==
列车为全动车，可以3-5节灵活编组。

=== 各名号详细分配 ===
==== 统一号 ====
自2010年7月开始运行，目前共28节：
*带有驾驶室的动车（Mc）：9549-9561
*动车（M）：9629，9630，9634，9635，9640，9648，9652，9656-9661

==== 无穷花号 ====
本列车于2009年2月之前曾在[[京义线|京义线]]服役，在[[首都圈电铁京义线|首都圈电铁京义线]]开通后，列车在2009年12月经过翻新，现改为在韩国南部运行，目前共96节：
*带有驾驶室的动车（Mc）：9011-9032，9035-9060
*动车（M）：9111-9132，9135-9160

==== 观光列车 ====
本列车分为[[海列车|海列车]]和[[庆北观光循环主题列车|庆北观光循环主题列车]]，目前共7节：
*[[海列车|海列车]]：9564（Mc）-9666（M）-9565（Mc）
*[[庆北观光循环主题列车|庆北观光循环主题列车]]：9033（Mc1）-9133（M1）-9134（M2）-9034（Mc2）

== 列车外涂装 ==
目前本列车拥有4种外涂装，其中[[统一号|统一号]]是以绿色、蓝色涂装为主，[[无穷花号|无穷花号]]是以红色、蓝色、白色涂装为主，[[观光列车|观光列车]]中的[[海列车|海列车]]为大海的蓝色涂装、[[庆北观光循环主题列车|庆北观光循环主题列车]]为更复杂的涂装。
