'''韩国铁道公社3000系电力动车组'''（{{kor|諺=한국철도공사 3000호대 전동차|漢=韓國鐵道公社 3000號帶 電動車}}）是[[韩国铁道公社|韩国铁道公社]]的一款[[通勤型电力动车组|通勤型电力动车组]]，在[[首都圈电铁3号线|首都圈电铁3号线]]运营。

== 概况 ==
3000系电力动车组是[[韩国铁道公社|韩国铁道公社]]在[[首都圈电铁3号线|首都圈电铁3号线]]的第1款列车，是直流用车辆，由[[现代Mobis|现代Mobis]]负责生产，在1996年正式使用。运行于[[首都圈电铁3号线|首都圈电铁3号线]]的[[一山线|一山线]]区间，亦包括[[首尔地铁|首尔地铁]]营运的[[首尔地铁3号线|首尔地铁3号线]]。列车客室车门为氣動内藏门，每节车厢共4对车门，列车在两端设有[[紧急出口|紧急出口]]，目前均为10辆编组，配属于[[纸杻车辆基地|纸杻车辆基地]]。是用来取代已经停用的部分[[首尔地铁3000系斩波控制电力动车组|3000系斩波控制]]列车。值得一提的是，本系列列车在[[首都圈电铁4号线|首都圈电铁4号线]]（[[首尔地铁4号线|首尔地铁4号线]]）也有服役，型号为[[首尔地铁4000系电力动车组|首尔地铁4000系电力动车组]]，只不过在外涂装上有所不同。

== 列车改造 ==
在2003年2月18日[[大邱地铁纵火案|大邱地铁纵火案]]发生后，列车的材料全部更换为防火材料。

== 事故 ==
