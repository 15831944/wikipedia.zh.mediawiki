'''韩国核能'''自20世纪50年代以来经过迅速的发展，已经成为世界核能的后起之秀。2007年，韩国成为世界上第三个具备自行研发[[第三代核电技术|第三代核电技术]]的国家<ref name=star>{{cite web|url=http://news.xinhuanet.com/world/2010-04/10/c_1225892.htm|title=韩国：核电领域后起之秀|publisher=新华网|date=2010-04-10|deadurl=yes|archiveurl=https://web.archive.org/web/20100505010421/http://news.xinhuanet.com/world/2010-04/10/c_1225892.htm|archivedate=2010-05-05}}</ref>。

韩国是个能源资源贫乏的国家。发展核能是韩国的国家战略。截止2014年底，韩国一共拥有23座核动力堆，数量世界第五。2013年，核能在韩国总发电量的比例为27.6%<ref>{{cite web |url=http://www.sgcc.com.cn/xwzx/nyzx/2015/03/322946.shtml |title=《2014年全球核电综述》 |publisher=国家电网 |date=2015-03-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20150617060837/http://www.sgcc.com.cn/xwzx/nyzx/2015/03/322946.shtml |archivedate=2015-06-17 }}</ref>。韩国核反应堆最近几年的运营功率平均保持在96.5%，是世界最高的国家之一<ref>{{cite web |url=http://www.360doc.com/content/13/0613/11/1630322_292532164.shtml  |title=核能档案39--走进韩国核电 |publisher=360DOC |date=2013-06-13}}</ref>。多年来，韩国核能保持着高效的运行效率，平均功率损失率仅为3.6%，为世界最低（[[经济合作与发展组织|世界经合组织]]国家的平均值为6.4%）。与此同时，韩国核能持续保持核电厂平均非计划停堆率世界最低。2010年，韩国核电厂的平均非计划停堆率仅为0.1%（[[美国|美国]]、[[法国|法国]]和[[加拿大|加拿大]]2008年的指标都在1%以上）<ref name=energy>{{cite web |url=http://www.china5e.com/news/news-882250-1.html  |title=韩国核电“走出去”的历程及主要经验 |publisher=中国能源网 |date=2014-08-29}}</ref><ref>{{cite web |url=https://www.koreatimes.co.kr/www/common/printpreview.asp?categoryCode=291&newsIdx=64190 |title=Korea Boasts Global Competitiveness in Nuclear Tech |publisher=The Korea Times |accessdate=2015-06-08}}</ref>。

==历史==
[[File:Kori_NPP.jpg|thumb]]
韩国的核能发展开始于20世纪50年代。1956年，韩国在与[[美国|美国]]签署和平利用核能的合作协定后，派出大量的人员赴美进行核能研究方面的培训{{r|energy}} 。1957年，韩国加入[[国际原子能机构|国际原子能机构]]<ref>{{cite web |url=https://www.iaea.org/about/memberstates |title=Member States of the IAEA |publisher=International Atomic Energy Agency |accessdate=2015-06-08}}</ref>。1958年，[[韩国国会|韩国国会]]通过了《原子能法》，为核能发展提供了法规体系。根据《原子能法》成立的原子能委员会是韩国核能政策的最高决策机构。原子能委员会由9到11名代表政府、学术界和产业界部门的成员组成，主席由[[韩国总理|韩国总理]]兼任<ref name=energy />。

1962年，韩国建成首座小型核实验反应堆。1971年，韩国首座核电站——{{link-ko|古里核电站|고리원자력발전소}}一号机组在[[釜山广域市|釜山广域市]][[机张郡|机张郡]]长安邑开工，于1977年建成，1978年4月实现了商业运营。在之后的几十年间，韩国的核工业得到迅速的发展，先后建立或扩建了[[月城核电站|月城核电站]]、古里核电站、新古里核电站、{{link-ko|灵光核电站|한빛원자력발전소}}、{{link-ko|蔚珍核电站|한울원자력발전소}}、新蔚珍核电站<ref>{{cite web |url=http://uzone.univs.cn/news2_2008_401530.html  |title=韩国核电档案 |publisher=教育部中国大学生在线网 |date=2012-10-17}}</ref>。2007年，韩国成为世界上第三个具备自行研发第三代核电技术的国家{{r|star}}。

2008年，韩国电力公司工程建设公司就赢得了一份价值300亿韩元的向[[西屋公司|西屋公司]]提供AP1000核岛和辅助设施的设计服务合同， 开始了韩国核技术的出口<ref name=energy />。2009年底，韩国力压[[美国|美国]]、[[法国|法国]]等世界老牌核电出口国，成功赢得[[阿联酋|阿联酋]]价值200亿美元的4座[[轻水核反应堆|轻水核反应堆]]核电站建设合同<ref>{{cite web|url=http://news.xinhuanet.com/world/2010-01/13/content_12805426.htm|title=综述：韩国欲打造世界核电大国|publisher=新华网|date=2010-01-13|deadurl=yes|archiveurl=https://web.archive.org/web/20100405060218/http://news.xinhuanet.com/world/2010-01/13/content_12805426.htm|archivedate=2010-04-05}}</ref><ref>{{cite web |url= http://www.chinadaily.com.cn/hqgj/2009-12/28/content_9235515.htm |title=韩国在阿联酋签订200多亿美元核电站建设协议 |publisher=中国日报 |date= 2009-12-28}}</ref>。2010年3月10日，韩国与[[土耳其|土耳其]]签署了为土耳其承建两座核电站的《核电项目合作相关的联合公报》。2013年，韩国被获准在约旦科技大学建设[[约旦|约旦]]首座核研究反应堆<ref name=energy />。2015年，韩国与[[沙特阿拉伯|沙特阿拉伯]]达成协议在沙國建设两座韩国自主开发的“SMART”核反应堆，试运行后，两国将联手出口第三国<ref>{{cite web |url=http://www.chinanews.com/gj/2015/03-04/7101218.shtml |title=韩国将耗资20亿美元在沙特建设核电站 |publisher=中国新闻网 |date=2015-03-04}}</ref>。2015年3月，[[韩国水电与核电公司|韩国水电与核电公司]]的“{{link-en|APR1400|APR-1400}}” 新型轻水反应堆设计通过了[[美国核能管理委员会|美国核能管理委员会]]的预备审批。韩国核电有望出口美国<ref>{{cite web |url=http://world.people.com.cn/n/2015/0306/c157278-26650137.html |title=韩核电设计首次通过美国预备审批 有望进军美国 |publisher=人民网 |date=201-03-06}}</ref>。

2011年10月26日，监管韩国核安全的独立机构核安全与安保委员会正式成立。该机构直接受[[韩国总统|韩国总统]]领导，全面负责韩国核安全、安保及保障工作<ref name=energy />。2015年4月22日，韩美修改了1973年生效的核能协定。韩国核乏燃料处理被解禁，为韩国开展核能活动提供了更大空间<ref>{{cite web |url=http://www.chinanews.com/gj/2015/04-22/7226813.shtml |title=韩美修改核能协定 为韩国核乏燃料处理解禁 |publisher=中国新闻网 |date=2015-04-22}}</ref><ref>{{cite web |url=http://world.newssc.org/system/20150422/000557748.html |title=韩美签署新原子能协定 有效期从41年缩短为20年 |publisher=中国新闻网 |date=2015-04-22 |access-date=2015-06-09 |archive-url=https://web.archive.org/web/20150720100700/http://world.newssc.org/system/20150422/000557748.html |archive-date=2015-07-20 |dead-url=yes }}</ref>。

韩国致力于发展比[[核裂变|核裂变]]能量更大，而且近乎零污染的[[核聚变|核聚变]]发电技术，计划在21世纪40年代建造商业性热核聚变发电厂。20世纪60年代，韩国开始开展小规模的实验室[[等离子|等离子]]实验。70年代晚期，韩国大学先后建造了SNUT79、KAIST、 KT 1、HANBIT等[[托卡马克|托卡马克]]装置。1995年，韩国基础科学研究院根据美国[[麻省理工大学|麻省理工大学]]的TARA串级磁镜，建造并安装了中型装置HANBIT。2009年12月，被称为“韩国太阳”的[[KSTAR|KSTAR]]在1000万摄氏度的温度下成功获得了电流为320千安的等离子体放电，持续时间约3.6秒。2010年11月8日，KSTAR提前一年首次成功实现了等离子体约束状态的H模式。这是世界首次用超导热核实验装置实现H模式，对[[国际热核聚变实验反应堆|国际热核聚变实验反应堆]]（ITER）项目的进展具有非常重要的价值<ref>{{cite web |url= http://kejichu.sjtu.edu.cn/info/1012/5943.htm |title= 韩超导装置首次实现高能量模式 |publisher= 上海交通大学 |date= 2010-11-26 |deadurl= yes |archiveurl= https://web.archive.org/web/20150715114559/http://kejichu.sjtu.edu.cn/info/1012/5943.htm |archivedate= 2015-07-15 }}</ref><ref name="韩联社">{{cite web |url=http://chinese.yonhapnews.co.kr/techscience/2009/12/09/8400000000ACK20091209002500881.HTML |title=热核聚变研究设备KSTAR实现超标 |publisher=韩联社 |date=2009-12-09}}</ref>
<ref name="现状">{{cite journal |url=http://www.globesci.com/fileup/PDF/20140224.pdf |title=韩国的核聚变研究现状及发展战略 |author=康卫红 |journal=《世界科技研究与发展》 |date=2014年第36卷第2期}}</ref>。
