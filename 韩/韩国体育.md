'''韩国体育'''是[[韩国文化|韩国文化]]的重要部分。[[跆拳道|跆拳道]]是源于[[韩国|韩国]]的一种集强身健体，防身自卫和心灵修养为一身的搏击武术，被称为韩国的国技。目前已经成为[[奥运会|奥运会]]的比赛项目。[[韩国|韩国]]的传统体育运动有[[韩式摔跤|韩式摔跤]]、[[斗牛|斗牛]]、[[风筝|风筝]]等。与大多数国家一样，现代竞技体育是韩国体育的主体。韩国已经成功举办了[[1988年夏季奥林匹克运动会|1988年夏季奥林匹克运动会]]、[[2002年世界杯足球赛|2002年世界杯足球赛]]、[[2003年夏季世界大学生运动会|2003年夏季世界大学生运动会]]、[[2011年世界田径锦标赛|2011年世界田径锦标赛]]、[[2018年冬季奥林匹克运动会|2018年冬季奥林匹克运动会]]等多个世界体育赛事和数个[[亚运会|亚运会]]，并取得优异的成绩<ref>{{cite web |url = http://slide.sports.sina.com.cn/o/slide_2_57057_70948.html#p=1  |title=盘点近年韩国举办的大型体育赛事 |publisher = 新浪网|accessdate=2014-10-26}}</ref>。

韩国的体育发展最初由1954年成立的文教部负责。1982年，韩国曾设立了专门负责体育事业的体育部，1988年6月，新设青少年局。1990年9月，体育部和青少年局合并成体育青少年部。1993年后，负责体育事业的部门开始和负责文化和旅游的部门合并。目前，韩国的体育事业由[[大韩民国文化体育观光部|文化体育观光部]]负责{{r|概况|韩文化|page1=134|page2=220}}。

==源于韩国的体育运动==
[[跆拳道|跆拳道]]既是一项能够强身健体，又能够于防身自卫的传统搏击武术，更是一项新兴的集健身、竞技及娱乐为一体的现代体育，集[[力学|力学]]、[[兵学|兵学]]、[[哲学|哲学]]、[[医学|医学]]及[[伦理学|伦理学]]为一体，以技击格斗为基础，以修心养性为核心，以磨练人的意志、振奋人的内在精神气质、培训练习者良好的礼仪及道德为目标。「跆拳道」一詞，是1955年由[[崔泓熙|崔泓熙]]创造。其中「跆」指踢擊、「拳」指拳擊，「[[道|道]]」則是代表道行、自己對禮儀的修練<ref>{{english}} {{cite web |url= http://www.worldtaekwondo.com/history.htm|title=  THE HISTORY OF TAEKWONDO|accessdate=2008-12-23 |work= A Report for Recommendation Black Belt Testing 1994|publisher=Glen R. Morris |date= }}</ref>。在1986年第10届亚运会，[[跆拳道|跆拳道]]成为亚运会正式比赛项目。2000年悉尼奥运会，跆拳道成为奥运会正式比赛项目。跆拳道已经在世界各国广为推广{{r|概况|韩国|page1=139|page2=211-212}}。 

[[跆跟|跆跟]]是[[朝鮮半島|朝鮮半島]]古代的一種傳統武術。跆跟以腿脚为主，注重利用对方弱点或借对方之力反击并使其摔倒，是一种柔中带刚的武艺<ref name="跆拳">{{cite web |url=http://chinese.visitkorea.or.kr/chs/CU/CU_CHG_3_5_2.jsp |title=韩国体育世界 |publisher=韩国旅游发展局 |accessdate=2015-08-31}}</ref>。

[[合气道_(韩国)|韩国合气道]]是参考[[日本|日本]][[大东流合气柔术|大东流合气柔术]]并加入一些其他武术混合而成的新流派武术。与[[日本|日本]][[合气道|合气道]]之间的主要差异在于融入了较多的打击技。但与其它武术不同的是，韩国合气道的打击技术并不强调出招的速度，而在于力度及准确度，并且主要用于使用关节技前的连接招式，或是最后给予对方必杀的一击。对资深的修习者，更强调面对中短距离武器如匕首、短棍、剑等武器的防卫<ref>[http://www.encyber.com/search_w/ctdetail.php?gs=ws&gd=%20&cd=%20&d=%20&k=%20&inqr=%20&indme=%20&p=1&q=%C7%D5%B1%E2%B5%B5&masterno=774999&contentno=774999 "합기도 ④"] {{webarchive|url=https://web.archive.org/web/20150323054815/http://www.blayney.nsw.gov.au/ArticleDocuments/329/3h%20-%20Hobbys%20Yards%20Heritage%20Items.pdf.aspx |date=2015-03-23 }} at Doosan EnCyber & EnCyber.com (두산 백과사전) {{ko icon}}</ref>。

===棒球===
[[棒球|棒球]]是[[韩国|韩国]]最受欢迎的体育运动之一。1905年，[[美国|美国]][[传教士|传教士]]菲利普·吉伯特将棒球传授给皇城基督教青年团，当时被称为“打球”或“击球”。1906年开始，韩国就有棒球比赛。1982年，韩国组建了职业棒球队。目前，韩国有[[LG双子|LG双子]]、[[NC恐龍|NC恐龍]]、[[韩华鹰|韩华鹰]]、[[起亚虎|起亚虎]]、[[三星狮|三星狮]]、[[罗德巨人|罗德巨人]]、[[斗山熊|斗山熊]]、[[SK飛龍|SK飛龍]]、[[KT巫師|KT巫師]]、[[耐克森英雄|耐克森英雄]]10支职业棒球队。[[2008年夏季奥林匹克运动会|2008年夏季奥林匹克运动会]]，韩国取得了棒球比赛冠军的好成绩{{r|韩文化|概况|page1=210-211|page2=137}}。

韩国有许多棒球运动员在[[美国职业棒球大联盟|美国职业棒球大联盟]]比赛。1994年，[[朴赞浩|朴赞浩]]加盟[[洛杉矶道奇|洛杉矶道奇]]，成为第一个进军美国职棒大联盟的韩国人<ref>{{cite web|title=Chan Ho Park Player Page |url= http://korea.people.com.cn/205155/205168/8041722.html |date=2012-11-30}}</ref>。其它在美国职棒大联盟打球的韩国棒球运动员还有[[崔熙燮|崔熙燮]] 、[[金炳贤|金炳贤]]、[[奉重根|奉重根]]、[[秋信守|秋信守]]、[[柳贤振|柳贤振]]、[[姜正浩|姜正浩]]、[[李大浩|李大浩]]等。

===足球===
韩国古代的[[蹴鞠|蹴鞠]]运动是一种与[[足球|足球]]很相像的体育运动。现代足球是1882年由到访[[仁川|仁川]]港的英国海军士兵传入韩国的<ref>{{cite web |url= http://www.kfa.or.kr/eng/record/history.asp |title= History |publisher= Korea Football Association |accessdate= 2015-08-31 |archive-url= https://web.archive.org/web/20150804074846/http://www.kfa.or.kr/eng/record/history.asp |archive-date= 2015-08-04 |dead-url= yes }}</ref>。1904年，足球被官办的外国语大学设为正式的体育项目。1906年，大韩体育俱乐部与皇城基督青年会在汉城的三善坪举行了韩国历史上第一场现代足球赛。1929年，[[汉城|汉城]]与[[平壤|平壤]]的“京平对抗赛”使普通百姓开始对足球运动产生兴趣{{r|韩文化|page=209}}。

1933年，[[韩国足协|韩国足协]]正式成立。1948年，韩国足协成为[[国际足联|国际足联]]的成员。同年，[[韩国国家足球队|韩国国家足球队]]首次在[[英国|英国]][[伦敦|伦敦]]举行的奥运会足球比赛中亮相。1971年以来，韩国开始每年举办国际足球锦标赛（原名为总统杯足球赛）。1983年，韩国在[[墨西哥|墨西哥]]举办的世青赛中杀进了前四强，并由此得到了“红魔”的美誉。同年，韩国成立了亚洲首个职业足球联赛。1994年，职业足球联赛更名为韩国足球联赛（[[K联赛|K联赛]]）。从1986年墨西哥世界杯开始，韩国队连续8次杀入世界杯决赛圈，是亚洲足球世界杯最好的成绩。2002年，[[韩国|韩国]]与[[日本|日本]]联合举办了[[2002年世界杯足球赛|2002年世界杯足球赛]]，并取得第四名的成绩{{r|韩文化|概况|韩国|page1=209|page2=136-137|page3=249}}。

===篮球===
[[篮球|篮球]]是另一个在[[韩国|韩国]]越来越受欢迎的体育运动。篮球运动是美国人吉礼泰在1903年传入韩国的，20年代就已经在全国范围普及。目前，韩国有10支职业篮球队在[[韩国篮球联赛|韩国篮球联赛]]比赛{{r|韩文化|概况|page1=212|page2=139}}<ref>{{cite web |url=http://www.kbl.or.kr/main/main.asp |title=Korean Basketball League |publisher=韩国篮球联赛官网 |accessdate=2015-08-31}}</ref>。

===高尔夫球===
[[高尔夫球|高尔夫球]]在[[韩国|韩国]]是个很流行的运动。 [[高尔夫球|高尔夫球]]会员的会费要比[[美国|美国]]和[[日本|日本]]的还贵，因此[[高尔夫球|高尔夫球]]在韩国也是地位的象征。2009年8月16日，韩国高尔夫球球手[[梁容银|梁容银]]击败了[[泰格·伍兹|泰格·伍兹]]赢得了[[PGA锦标赛|PGA锦标赛]]冠军，成为第一个取得此殊荣的亚洲人<ref>{{cite web |url=http://sports.qq.com/a/20090817/000166.htm |title=PGA锦标赛梁容银大比分夺冠 伍兹爆冷大崩盘 |publisher=腾讯网 |date=2009-08-17}}</ref>。韩国女子高尔夫球球员[[朴世莉|朴世莉]]更是获得了23个LPGA冠军，其中包含五个大满贯冠军。2007年，朴世莉成为首位进入{{link-en|世界高尔夫球名人堂}}的韩国人<ref>{{cite web |url=http://sports.enorth.com.cn/system/2007/09/26/002050606.shtml |title=LPGA为朴世莉开庆功会 成首位韩籍名人堂成员 |publisher=北方网 |date=2007-09-26}}</ref>。

===技击运动===
[[射箭|射箭]]被称为韩国的国技<ref>{{cite web|url=http://news.xinhuanet.com/world/2014-09/28/c_127043651.htm|title=射箭为啥是韩国“国技”|publisher=新华网|date=2014-09-28}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。韩国在这个比赛项目中保持着世界领先水平<ref>{{cite web |url=http://sports.163.com/15/0704/21/ATN9PRSS00051C8U.html|title= 韩国射箭展现恐怖实力 排位赛刷新两项世界纪录|publisher=网易|date=2015-07-04}}</ref>。

[[射擊|射擊]]近年來成為韓國最著名一項運動之一<ref>{{cite web |url=http://mt.sohu.com/20140920/n404487999.shtml |title= 韩国射击露崛起前兆 上届奖牌榜回顾已威胁中国 |publisher=搜狐网 |date=2014-09-20}}</ref>。[[擊劍|擊劍]]普遍韓國民眾最熱愛的運動之一，在奧林匹克運動會也是有不錯的表現<ref>{{cite web |url=http://sports.sohu.com/20131012/n388047851.shtml |title=东亚运击剑奖牌榜：韩国居首 中国5金7银4铜第2 |publisher=搜狐网 |date=2013-10-12}}</ref>。

===其它===
[[短道速滑|短道速滑]]是韩国的体育强项之一。在2006年2月[[2006年冬季奥林匹克运动会|都灵冬奥会]]和3月世锦赛上，韩国取得了一个又一个的短道速滑冠军。[[2010年冬季奥林匹克运动会|2010年温哥华冬奥会]]，韩国获得了2金、4银、2铜的好成绩。2010年世界团体短道速滑锦标赛上，韩国男女队分获男女团体冠军{{r|韩文化|page =213}}。

[[电子竞技|电子竞技]]在[[韩国|韩国]]很盛行。[[星际争霸职业联赛|星际争霸职业联赛]]是韩国最大，也是全球最大的电子竞技赛事项目，游戏打破了《[[吉尼斯世界纪录大全|吉尼斯世界纪录大全]]》中“职业联赛收入最高的游戏”、“比赛观众最多的游戏”两项纪录<ref>{{en}}{{cite book|title=Guinness World Records Gamer's Edition|publisher=Hit Entertainment|year=2008|isbn=9781904994206}}</ref>。韩国大公司赞助的电子竞技队有''OnGameNet Starleague'' 、''MBCGame StarCraft League''（已退役）和''Proleague''。韩国还有播放[[电子竞技|电子竞技]]的电台，像''Ongamenet''、''GomTV''等。

2010年以来，[[韩国|韩国]]每年都在[[全罗南道|全罗南道]][[灵岩郡|灵岩郡]]举办[[一级方程式赛车|一级方程式赛车]]<ref>{{cite web |url=http://chinese.visitkorea.or.kr/chs/SI/SI_CHG_2_11.jsp?cid=1114752 |title=F1一级方程式锦标赛韩国站 |publisher=韩国旅游发展局 |date=2013-09-25}}</ref>。
