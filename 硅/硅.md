'''硅'''（{{lang-en|'''silicon'''}}，台湾、香港及澳門称為'''-{矽}-'''，舊訛稱為'''釸'''，中國大陸稱為'''-{硅}-'''）是一种[[类金属|类金属]][[元素|元素]]，[[化学符号|化学符号]]為'''Si'''，[[原子序數|原子序數]]為14，属于[[元素周期表|元素周期表]]上的IVA族。

硅原子有4个外圍电子，与同族的[[碳|碳]]相比，硅的化学性质相對稳定，活性較低。硅是极为常见的一种元素，然而它极少以[[單質|單質]]的形式存在於自然界，而是以复杂的[[硅酸盐|硅酸盐]]或[[二氧化硅|二氧化硅]]等[[化合物|化合物]]形式广泛存在于岩石、砂砾、尘土之中。在宇宙储量排名中，矽位於第八名。<ref>见{{Harvnb|Greenwood|1984|p=517}}(位于上册第9章“硅”第2节“硅元素”)。</ref>在[[地壳|地壳]]中，它是第二丰富的元素，佔[[地壳|地壳]]总质量25.7%，仅次于第一位的[[氧|氧]]（45.4%）。<ref>见{{Harvnb|Greenwood|1984|p=517}}(位于上册第9章“硅”第2节“硅元素”)。</ref>

硅是重要的[[半导体|半导体]]材料，广泛应用于制造各种电子仪器，是现代工业的材料基石之一。<ref>见{{Harvnb|Greenwood|1984|p=516}}(位于上册第9章“硅”第1节“导论”)。</ref>

==性質==
===物理性質===
结晶型的硅是暗蓝色的，很脆，是典型的[[半导体|半导体]]。

===化學性質===
硅的化学性质非常稳定，常与[[氧|氧]]元素形成化合物。<ref>见{{Harvnb|Greenwood|1984|p=516}}(位于上册第9章“硅”第1节“导论”)。</ref>存在于地表的硅几乎总以含氧化合物的形式存在，尤以包含4个配位键的结构居多，少有例外。<ref>见{{Harvnb|Greenwood|1984|p=518}}(位于上册第9章“硅”第2节“硅元素”)。</ref>每1个硅元素搭配4个氧元素的组合可以单独形成基团，也可以形成链、带、环、层等复杂结构。<ref>见{{Harvnb|Greenwood|1984|p=518}}(位于上册第9章“硅”第2节“硅元素”)。</ref>在常温下，除[[氟化氢|氟化氢]]以外，很难与其他物质发生反应。

固态硅单质不太活泼，不易于气体或液体试剂反应；液态硅单质则相反，能与多数金属发生反应。<ref>见{{Harvnb|Greenwood|1984|p=524}}(位于上册第9章“硅”第2节“硅元素”)。</ref>

=== 同素异形体 ===
[[同素异形体|同素异形体]]有两种，一种为暗棕色无定形粉末，用[[镁|镁]]使二氧化硅还原而得，性质比较活泼，能够在空气中燃烧，称为[[无定形硅|无定形硅]]；另一种为性质稳定的晶体（结晶硅），是用炭在电炉中使二氧化硅还原而得。
[[File:Quartz_Brésil.jpg|thumb]]，其主要成分為[[二氧化矽|二氧化矽]]]]

== 发现 ==
1787年，[[拉瓦锡|拉瓦锡]]首次发现硅存在于岩石中。然而在1800年，[[戴维|戴维]]将其错认为一种[[化合物|化合物]]。1811年，[[约瑟夫·路易·盖-吕萨克|盖-吕萨克]]和{{link-en|路易·雅克·特纳|Louis Jacques Thénard|路易·特纳}}可能已经通过将单质[[钾|钾]]和[[四氟化硅|四氟化硅]]混合加热的方法制备了不纯的[[无定形硅|无定形硅]]。1823年，硅首次作为一种元素被[[贝采利乌斯|贝采利乌斯]]发现，并于一年后提炼出了[[无定形硅|无定形硅]]，其方法与[[约瑟夫·路易·盖-吕萨克|盖-吕萨克]]使用的方法大致相同。他随后还用反复清洗的方法将单质硅[[提纯|提纯]]。

== 名称由来 ==
[[英文|英文]]中的{{lang|en|silicon}}一词，来自[[拉丁文|拉丁文]]的{{lang|lt|''silex, silicis''}}，意思为[[燧石|燧石]]（即火石，富含硅元素）。在1831年，苏格兰化学家{{link-en|托马斯·汤姆森|Thomas Thomson}}創造了這個名詞。<ref>见{{Harvnb|Greenwood|1984|p=516}}(位于上册第9章“硅”第1节“导论”)。</ref>

在1823年純化出來後，瑞典化学家[[永斯·贝采利乌斯|永斯·贝采利乌斯]]利用[[新拉丁文|新拉丁文]]規則造出{{lang|la|silicium}}這個單字，再加上-ium字根，以代表它是一種[[金屬|金屬]]。歐洲許多國家都採用{{lang|la|silicium}}這個名稱，但英文名稱最終採用了在1817年提出的{{lang|en|silicon}}這個拼法，去除了代表金屬的-ium字根，因為它的物理特性更接近於[[碳|碳]]（carbon）與[[硼|硼]]（boron）這一類元素。<ref>见{{Harvnb|Greenwood|1984|p=516}}(位于上册第9章“硅”第1节“导论”)。</ref>

1837年，[[日本|日本]]第一部西方化学译著，[[宇田川榕庵|宇田川榕庵]]的《[[舎密開宗|舎密開宗]]》中首先以「珪土」作为硅元素的名称。<ref name=":0">{{Cite book|title=舎密開宗|last=宇田川榕菴|first=|publisher=|year=1837|isbn=|location=|pages=}}</ref>该书译自[[荷蘭語|荷兰语]]，由于当时荷兰语硅的元素名为「keiaarde」，是「keisteen-aarde」（燧石土）的缩略，日本就专门找了一个玉字旁的同音字「珪」（日语：けい，汉语：-{zh-hans:guī;zh-tw:ㄍㄨㄟ;zh-hk:guī}-，是「圭」的异体字）来音译「kei」。因此「keisteen」就译成「珪石」，作为[[二氧化硅|SiO<sub>2</sub>]]的名称<ref name=":1">{{Cite web|url=https://zhuanlan.zhihu.com/p/22339684|title=「硅」「矽」新考|accessdate=2016-09-07|author=古逢平|date=2016-09-07|publisher=}}</ref>。由于当时荷兰语中的[[镁|镁]]、[[铝|铝]]、硅、[[钡|钡]]等元素都是以「aarde」（土）结尾，因此日语也分别照译成苦土、礬土、珪土、重土。<ref name=":0" /><ref name=":1" />后来日本摒弃了荷兰语的「土」，改译为「珪素」。19世纪后期，出现了「-{硅}-素」的写法，<ref>{{Cite journal|title=关于“珪素、硅素”的溯源[J]|author=太田泰弘, 孙丽平|url=|journal=中国科技术语|issue=|doi=|others=|year=2013|volume=15 (3)|page=58-59|pmid=}}</ref>不过19世纪末日本规定以「珪素」为准。「珪/-{硅}-」进入中国是20世纪初。原因有一派学者考虑土壤基本上就是混杂的硅酸盐。他们想到的是菜畦（qí）的畦字。畦是土壤，正好是硅酸盐类组成的<ref name=":2">{{Cite journal|title=关于“硅”和“矽”来历的补充[J]|author=邵靖宇|url=|journal=中国科技术语|issue=|doi=|others=|year=2008|volume=10 (4)|page=64|pmid=}}</ref><ref name= xiandgui></ref>，由于它属于[[固体|固态]][[非金属元素|非金属]]元素，所以中国采用了石字旁的写法（「燐」改成「[[磷|磷]]」也是同理）。<ref name=":3">{{Cite book|title=化學命名原則|last=國立編譯館|first=|publisher=國立編譯館|year=1933|isbn=|location=|pages=}}</ref>「-{硅}-」字古也有之，是「砉」的[[异体字|异体字]]，讀作-{zh-hans:huò;zh-tw:ㄏㄨㄛˋ;zh-hk:huò}-<ref>[http://dict.variants.moe.edu.tw/yitib/frb/frb02950.htm 教育部異體字典-砉]</ref>）<ref name="清华人">[http://www.tsinghua.org.cn/alumni/infoSingleArticle.do?articleId=10034286&columnId=10012607 “-{硅}-”字的来历和变迁] {{webarchive|url=https://web.archive.org/web/20141218203608/http://www.tsinghua.org.cn/alumni/infoSingleArticle.do?articleId=10034286&columnId=10012607 |date=2014-12-18 }}，《清华人》2009-4期</ref>，与今义无关。

「-{矽}-」一字的出现是在1871年，中国第一部西方化学译著，中国近代化学的启蒙者[[徐壽|徐寿]]的《[[化學鑑原|化學鑑原]]》中作为硅元素（silicon）的音译<ref>{{Cite book|title=化學鑑原|last=徐壽|first=|publisher=|year=1871|isbn=|location=|pages=}}</ref>。20世纪初，中国「-{硅}-」、「-{矽}-」混用，“硅”表示silicon最早见于1906年顾琅、周树人《中国矿产志·导言》：“以水成者，有砂硅粘板石灰等。”。直到1933年[[中華民國教育部|中華民國教育部]]公布《[[化學命名原則|化學命名原則]]》，将硅元素的译名定为「-{矽}-」，其中写道「Silicon 舊譯一作-{硅}-，一作-{矽}-，-{硅}-由日名珪素孳演而成，因爲固體，故改王旁爲石；於義旣無可取，不如用諧聲之-{矽}-」。<ref name=":3" />

[[中国科学院|中国科学院]]于1953年2月在北京组织召开了一次全国性的化学物质命名扩大座谈会。会议邀集全国各地的化学专家以及从事文字改革工作的学者参加。据会议纪要显示，当日会场上围绕-{矽}-字等同音字是否需要变更有着两种不同的看法。一派是以[[顾翼东|顾翼东]]、[[方柏容|方柏容]]、[[刘泽先|刘泽先]]三位先生为代表，主张改；另一派则以[[符绶玺|符绶玺]]、[[侯毓汾|侯毓汾]]二位先生为代表，主张不改。<ref>{{Cite web|url=http://dag.ecust.edu.cn/s/233/t/254/dd/60/info56672.htm|title=废矽改硅：避免中译化学名词同音字的一次选择|accessdate=2016-09-07|author=王力|date=2015-01-08|publisher=}}</ref>座谈会结束后不久，《化学通报》连载了化学名词审查小组成员陶坤的文章——《化学新字的读音》上、下两篇。在下篇中，陶文宣布将-{矽}-字改为-{硅}-字，在注解中他是这样陈述更改理由的：“-{矽}-音夕，与硒、烯、醯、锡不易分辨”。<ref>{{Cite journal|title=化学新字的读音（下）[J]|author=陶坤|url=|journal=化学通报|issue=8|doi=|others=|year=1953|volume=|page=359|pmid=}}</ref>真正全国性的统一变更是在1957年。这一年，[[中国科学院|中国科学院]]编译出版委员会名词室下发《关于几个化学名词订名问题的通知》，正式宣布废-{矽}-改-{硅}-的决定。这一通知中提到了审定理由和过程：“1953年化学名词审查小组建议将‘-{矽}-’改为‘-{硅}-’，1955年，无机化合物名词审查小组认为此项建议甚为正确，在征求全国各有关单位意见后，决议将‘-{矽}-’改为‘-{硅}-’。”<ref>{{Cite journal|title=关于几个化学名词订名问题的通知[J]|author=中国科学院编译出版委员会名词室|url=|journal=化学通报|issue=1|doi=|others=|year=1957|volume=|page=70|pmid=}}</ref>，值得注意的是：最初-{硅}-字并不念作圭(guī)而要求念作夕(xì)。因为宣传不够，人们不知道该怎么念，而想当然地按圭、桂、闺等字的读音。後來的人沿用错误读法，这样，Si这种元素在中国大陆有两个名称，-{硅}-和-{矽}-。<ref name= xiandgui>邵靖宇《中国科技术语》硅字的来历和变迁，2008，一節原文“硅這種元素的拉丁名是Silicium，元素符號是Si，當時把它設計為硅字，要求的讀音是xi，為的是與Si的音盡量接近。請注意：最初-{硅}-字並不念作gui而要求念作xi。當年的學者考慮到硅是土壤的主要組成成分，土壤基本上就是混雜的硅酸鹽。他們想到的是菜畦（xi）的畦字。畦是土壤，正好是硅酸鹽類組成的。因此讓硅讀畦的音可以聯想到土壤，兼有諧音和會意的意思。”</ref>，[[矽肺|矽肺]]與[[矽钢片|矽钢片]]等民间常用詞彙至今仍常用-{矽}-字（在大陸讀作-{zh-hans:xī;zh-tw:ㄒㄧ;zh-hk:xī}-<ref>[http://dict.baidu.com/s?wd=%E7%9F%BD 百度词典_矽]</ref>）<ref>[http://www.edubridge.com/letter/xizi.htm -{硅}-字的来历和变迁]</ref>。在香港，兩用法皆有，但「-{矽}-」較通用。

== 分布 ==
硅主要以含氧[[化合物|化合物]]的形式，作为仅次于[[氧|氧]]的最丰富的[[元素|元素]]存在于[[地壳|地壳]]中，约占地表岩石的四分之一，广泛存在于[[硅酸盐|硅酸盐]]和[[硅石|硅石]]中。

== 制备 ==
工业上，通常是在电炉中由高纯度[[碳|碳]]还原[[二氧化硅|二氧化硅]]（常见于[[石英|石英]]或沙子）而制得：<ref>见{{Harvnb|Greenwood|1984|p=518}}(位于上册第9章“硅”第2节“硅元素”)。</ref>

:<chem>SiO2 + 2C -> Si + 2CO</chem>

为减少[[副产物|副产物]]碳化硅的产生，需要加入过量的二氧化硅：<ref>见{{Harvnb|Greenwood|1984|p=518}}(位于上册第9章“硅”第2节“硅元素”)。</ref>
:<chem>SiO2 + 2SiC -> 3Si + 2CO</chem>

这样制得的硅纯度为97~98%，<ref>见{{Harvnb|Greenwood|1984|p=519}}(位于上册第9章“硅”第2节“硅元素”)。</ref>叫做粗硅。再将它融化后[[重结晶|重结晶]]，用酸除去杂质，得到纯度为99.7~99.8%的純硅。如要将它做成半导体用硅，还要将其转化成易于提纯的[[液体|液体]]或[[气体|气体]]形式，再经蒸馏、分解过程得到'''多晶硅'''。如需得到高纯度的硅，则需要进行进一步的提纯处理。

== 同位素 ==
已发现的硅的[[同位素|同位素]]共有12种，包括[[硅25|硅25]]至[[硅36|硅36]]，其中只有[[硅28|硅28]]，[[硅29|硅29]]，[[硅30|硅30]]是稳定的，其他[[同位素|同位素]]都带有[[放射性|放射性]]。

== 用途 ==
硅是一种[[半导体|半导体]]材料，可用于制作半导体器件、[[太陽能|太陽能]]電板、[[光導纖維|光纖]]和[[積體電路|積體電路]]。还可以[[合金|合金]]的形式使用（如硅铁合金），用于汽车和机械配件。也与[[陶瓷材料|陶瓷材料]]一起用于[[金属陶瓷|金属陶瓷]]中。还可用于制造[[玻璃|玻璃]]、[[混凝土|混凝土]]、[[砖|砖]]、[[耐火材料|耐火材料]]、[[矽氧烷|矽氧烷]]、[[硅烷|硅烷]]。與鐵結合，可以成為[[矽鋼|矽鋼]]，這是一種耐磨的鋼件，常用在各種工具上。此外，矽也是[[不鏽鋼|不鏽鋼]]的主成分之一，用來使不鏽鋼具有耐磨的特性。

== 重要化合物 ==
* [[碳化矽|碳化矽]]，耐磨耐热，<ref>见{{Harvnb|Greenwood|1984|p=525}}(位于上册第9章“硅”第2节“硅元素”)。</ref>用於[[半導體|半導體]]、[[避雷針|避雷針]]、電路元件、高溫應用、紫外光偵檢器、結構材料、[[天文|天文]]、碟剎、離合器、柴油微粒濾清器、細絲高溫計、陶瓷薄膜、裁切工具、加熱元件、核燃料、珠寶、[[鋼|鋼]]、護具、觸媒擔體等領域。
* [[二氧化矽|二氧化矽]]，是[[沙|沙]]和[[石英|石英]]的主要成分。在半導體和太陽能板等應用中，是目前主要的原料。
* [[矽烷|矽烷]]，在醫學和工業領域有著廣泛的應用。
* [[四氯化矽|四氯化矽]]，應用在半導體工業和光電池中。
