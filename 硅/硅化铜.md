'''硅化铜'''（[[铜|Cu]]<sub>5</sub>[[硅|Si]]）又名'''硅化五铜'''，是一种[[铜|铜]]的[[二元硅化合物|二元硅化合物]]，為一种[[金属互化物|金属互化物]]，这意味着它的性质介于[[离子化合物|离子化合物]]和[[合金|合金]]之间。可以通过加热铜与硅的混合物得到。

== 物理性质 ==
[[立方晶系|立方晶体]]，其中''a'' = 0.6224 [[纳米|nm]]。不溶于水。

== 应用 ==
硅化铜[[薄膜|薄膜]]可用于[[钝化|钝化]][[铜基芯片|铜基芯片]]，抑制其[[扩散作用|扩散]]和[[电子迁移|电子迁移]]，及作为[[扩散阻挡层|扩散阻挡层]]。<ref>{{cite web |url=http://www.patentstorm.us/patents/6869873.html |title=存档副本 |accessdate=2007-02-10 |0=2007-09-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20070929082943/http://www.patentstorm.us/patents/6869873.html |archivedate=2007-09-29 }}</ref>

硅化铜也可用于工业上[[直接合成|直接合成]][[有机硅化学|有机硅化合物]]。在反应中，硅化铜可以使[[氯代甲烷|氯代甲烷]]硅化。例如，可以制得工业上有用的[[二甲基二氯硅烷|二甲基二氯硅烷]]：<ref>{{Greenwood&Earnshaw2nd}}</ref>
:2 CH<sub>3</sub>Cl + Si → (CH<sub>3</sub>)<sub>2</sub>SiCl<sub>2</sub>
