'''宝坻南站'''位于[[天津市|天津市]][[宝坻区|宝坻区]][[潮阳街道|马家店镇]]，是建设中的[[京滨城际铁路|京滨城际铁路]]和[[京唐城际铁路|京唐城际铁路]]以及规划中的[[津承城际铁路|津承城际铁路]]的一个车站<ref>{{Cite web|url=http://www.gov.cn/xinwen/2016-12/30/content_5154837.htm|title=京唐城际铁路正线工程开建_地方政府政务联播_中国政府网|accessdate=2018-09-08|last=张维|work=www.gov.cn}}</ref>。

香河站为4台10线车站<ref>{{cite web |title=新建北京至唐山铁路环境影响报告书全本 |url=http://hps.mee.gov.cn/jsxm/xmsl/201608/W020160819560914540113.pdf |website=中华人民共和国生态环境部 |accessdate=2018-09-09}}</ref><ref>{{cite web |title=新建北京至天津滨海新区城际铁路（宝坻-滨海新区）环境影响报告书 |url=http://hjbh.tj.gov.cn/root16/mechanism/development_mana/201604/P020160427597834866683.pdf |website=中华人民共和国天津市环境保护局 |accessdate=2018-09-09}}</ref>，是建设中的京唐城际铁路工程中规模最大的一座车站。
