《'''宝贝儿'''》（{{lang-en|'''''Baby'''''}}）是2018年中国大陆剧情片，由[[刘杰_(导演)|刘杰]]导演，[[杨幂|杨幂]]、[[郭京飞|郭京飞]]和[[李鸿其|李鸿其]]主演。片中，杨幂饰演的孤儿江萌拯救另一位被父母抛弃的缺陷婴儿，这是楊冪首次素颜出演电影。影片入围[[2018年多倫多國際影展|第43届多伦多国际电影节]]“特别展映”单元和[[圣塞瓦斯蒂安国际电影节|圣塞瓦斯蒂安国际电影节]]主竞赛单元。中国大陆定于2018年10月19日上映。

== 制作 ==
影片于[[江苏省|江苏省]][[南京市|南京市]]和[[安徽省|安徽省]][[马鞍山市|马鞍山市]]取景，全片采用[[南京话|南京话]]台词，镜头大多聚焦于郊区农村、菜市场、汽车站、市井街巷等各个真实的生活场景。阵容方面，除了杨幂、郭京飞和李鸿其三位专业演员，其余角色均为素人，其中包括真实生活中的警察、福利院长。这是杨幂出演的首部文艺片<ref>{{cite web |title=《宝贝儿》用平等眼光直面不完美人生 |url=http://epaper.bjnews.com.cn/html/2018-09/19/content_732433.htm?div=0 |website=新京报 |accessdate=2018-10-02 |date=2018-09-19}}</ref><ref>{{cite web |title=《宝贝儿》多伦多首秀 杨幂文艺片实现大反差 |url=http://www.1905.com/news/20180910/1312471.shtml |website=1905电影网 |accessdate=2018-10-02 |date=2018-09-10}}</ref>。

电影于2017年10月开机，2018年3月17日正式宣布杀青<ref>{{cite web |title=杨幂郭京飞联袂主演 刘杰微博透露《宝贝》阵容 |url=http://ent.sina.com.cn/zz/2018-03-07/doc-ifxshkpn0657920.shtml |website=新浪网 |accessdate=2018-10-02 |date=2018-03-07}}</ref>。

== 发行 ==
*影片定档2018年10月19日在中国大陆上映<ref>{{cite web |title=《宝贝儿》定档10月19日 杨幂文艺片首秀 |url=http://www.xinhuanet.com/ent/2018-08/14/c_1123265276.htm |website=新华网 |accessdate=2018-10-02 |date=2018-08-14}}</ref>，首支预告片于9月6日公布<ref>{{cite web |title=杨幂主演《宝贝儿》曝光首支预告 饰演医院护工"偷小孩" 折射弃婴成长种种遭遇 |url=http://news.mtime.com/2018/09/06/1584008.html |website=Mtime时光网 |accessdate=2018-10-02 |date=2018-09-06}}</ref>。
*電影上映首日票房即突破1000万人民币，截至目前為止，票房近2500萬人民幣。

== 评价 ==
*《[[综艺_(杂志)|综艺]]》杂志认为影片“细致入微的审视了[[中国计划生育政策|中国计划生育政策]]的后果，含蓄的批判了中国国家福利政策的僵化，展现了社会对女性与残疾人的歧视，以及在一个冷酷的世界，善良该如何自处的问题。”<ref>{{cite web |last1=Lee |first1=Maggie |title=Film Review: ‘Baby’ |url=https://variety.com/2018/film/reviews/baby-review-baobei-1202934531/ |website=Variety |accessdate=2018-10-02 |date=2018-09-08}}</ref>
*《[[人民網|人民網]]》亦指出，该片原本被视为杨幂的转型之作，让观众对其演技有所改观，但目前看来，网友却并不为杨幂的“扮丑”、“牺牲形象”买单；影评人“水怪”也表示，杨幂以往拍商业片较多，第一次愿意花时间去演绎一个不熟悉的角色，对演员本身来说是一种进步，也能让更多观众关注有出生缺陷的儿童群体，評價產生兩極化。<ref>[http://ent.people.com.cn/n1/2018/1022/c1012-30354819.html 杨幂《宝贝儿》票房口碑双输 给流量明星敲响警钟]</ref>
