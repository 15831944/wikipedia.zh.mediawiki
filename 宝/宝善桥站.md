'''宝善桥站'''是[[杭州地铁5号线|杭州地铁5号线]]市区段的一个站点，位于杭州市下城区建国北路体育场路交叉口以北<ref>{{cite web |title=抓铣结合相得益彰 SX40双轮铣助力上海远方杭州地铁5号线宝善桥站完美收官-基础工程网 |url=http://www.jcgcw.com/news/qiye/11/54541.html |website=基础工程网}}</ref>。


== 车站设计 ==
宝善桥站为地下岛式站台，沿建国北路布置，有3个出入口和一个消防疏散口<ref>{{cite web |title=本周六晚起 杭州建国北路（头营巷-体育场路段）改为北向南单行 - 杭州新闻中心 - 杭州网 |url=http://hznews.hangzhou.com.cn/chengshi/content/2017-07/27/content_6612247.htm |website=杭州网}}</ref>。
