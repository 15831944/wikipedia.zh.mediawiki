'''宝鸡中学'''（通常简称为'''宝中'''）位于[[中国|中国]][[陕西省|陕西省]][[宝鸡市|宝鸡市]]东开发区高新大道29号，是一所陕西省省级重点中学。2010年9月10日成为[[陕西省普通高中示范性学校列表|陕西省普通高中示范性学校]]<ref name="宝中建校70周年">{{Cite web |url= http://www.sxbjedu.com/info.jsp?urltype=news.NewsContentUrl&wbnewsid=4277&wbtreeid=1218 |title= 宝鸡中学隆重举行建校70周年暨省级示范高中命名大会-宝鸡教育网 宝鸡市教育局主办 |date=2010年9月10日 |accessdate= 2013年10月3日 }}</ref>。

== 学校历史 ==

宝鸡中学创建于1940年。六十多年来，曾经三次搬迁，五次更名<ref name="六十校庆讲话">
{{cite web|url=http://www.baozhong.com.cn/info/ShowContent.asp?Programtype=news&ID=1537|title=走过沧桑岁月，面向辉煌未来——在宝鸡中学建校六十周年庆祝大会上的讲话|author=常雅宁|publisher=宝鸡中学|date=2000年10月6日|accessdate=2008年6月21日}}</ref>。

=== 校名及校址 ===

学校最初创立时的校名是“大华初级中学”<ref name="六十校庆讲话"/>，而后改名为“宝鸡县立中学”<ref name="六十校庆讲话"/>。在此之后还曾经使用过“宝鸡中学”，“宝鸡市第一中学”等校名。其现行校名为“宝鸡中学”<ref name="宝中校史">{{Cite web |url= http://www.sxbjedu.com/info.jsp?urltype=news.NewsContentUrl&wbnewsid=4286&wbtreeid=1217 |title= 宝鸡中学·建校七十周年 （宝中校史） -宝鸡教育网 宝鸡市教育局主办 |date=2010年9月13日 |accessdate= 2013年10月3日 }}</ref>。

学校最初创立时的校址在原宝鸡县（今宝鸡市[[陈仓区|陈仓区]]）{{fact|time=2008-6-21}}，后迁至宝鸡市[[金台区|金台区]]宝中路1号，所在街道因校得名。在此期间，宝鸡中学逐步发展至县立中学，再到市省两级重点中学<ref name="六十校庆讲话"/>。

20世纪五十年代前期，宝鸡中学作为关中四大重点中学，合并了[[陇县|陇县]]、[[彬县|彬县]]、[[虢镇|虢镇]]三校，办学规模空前扩大，教学质量不断提高。<ref name="六十校庆讲话"/>

2001年，宝鸡中学开始筹备吸纳社会资金搬迁新建和进行国有民办改制。2003年9月，由诚誉公司出资，宝鸡中学完成国有民办改制，并从宝中路1号迁至宝鸡市东开发区高新大道29号<ref name="新校落成讲话">{{cite web|url=http://www.baozhong.com.cn/info/ShowContent.asp?Programtype=news&ID=1538|title=更新观念　迎接挑战　齐心创造宝中新的辉煌——常校长在新校落成暨开学典礼上的讲话|author=常雅宁|publisher=宝鸡中学|date=2003年9月6日|accessdate=2008年6月21日}}</ref><ref name="宝鸡中学简介">
{{cite web|url=http://www.baozhong.com.cn/info/ShowContent.asp?Programtype=news&ID=1996|title=西秦明珠，人才摇篮(宝鸡中学简介)|publisher=宝鸡中学|date=2004年12月14日|accessdate=2008年6月21日}}</ref>。

== 教学结构 ==
