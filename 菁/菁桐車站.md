'''菁桐車站'''位於[[臺灣|臺灣]][[新北市|新北市]][[平溪區|平溪區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[平溪線|平溪線]]的[[鐵路車站|終點站]]。

本站建於1929年，並在2003年5月1日由臺北縣政府（今[[新北市政府|新北市政府]]）文化局公告為古蹟<ref>{{cite web|url=https://nchdb.boch.gov.tw/assets/advanceSearch/monument/20030501000001|title=菁桐車站|publisher=文化部文化資產局|date=}}</ref>。

== 車站構造 ==
*[[岸式月台|岸式月台]]一座。
*木造車站。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#簡易站|簡易站]]。
*行駛車輛為[[台鐵DR1000型柴油客車|DR1000型柴油客車]]，等級為[[臺鐵區間車|區間車]]。
*本站有"鐵路之旅-小站巡禮紀念章"之戳章,可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1929年10月1日：設立，初名'''菁桐坑驛'''。
*1962年12月15日：與[[保安車站|保安]]、[[嶺腳車站|嶺腳]]、[[十分車站|十分]]、林榮（廢止）、[[南平車站_(臺灣)|南平]]、[[月美車站|月眉（月美]]，已廢止）等車站同時改稱今名。
*1989年3月20日：由三等站降為簡易站並指定由[[十分車站|十分站]]管理。<ref>{{cite book |author = 戴震宇 |authorlink =  |title = 《台灣的老火車站》 |publisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>
*2001年7月16日：管理站由[[十分車站|十分站]]改為[[瑞芳車站|瑞芳站]]。
*2013年9月30日：啟用多卡通刷卡機。<ref>{{cite web|url=http://www.railway.gov.tw/tw/news_Detailnw.aspx?SN=24275| title=臺鐵局推動電子票證多卡通，9月30日起擴大服務範圍|publisher=交通部臺灣鐵路管理局|date=2013-10-01|accessdate=2014-05-02|language=中文}}</ref>
*2014年2月：火車站修復暨月台加長工程動工<ref>{{cite web|url=http://news.ltn.com.tw/news/local/paper/763378| title=菁桐車站擴增月台 不影響上、下車|publisher=自由時報電子報|date=2014-03-19|accessdate=2014-05-07|language=中文}}</ref>，於2015年3月2日完工啟用<ref>{{cite web|url=http://udn.com/news/story/7323/1009419-%E8%8F%81%E6%A1%90%E8%BB%8A%E7%AB%99%E4%B8%8D%E8%A6%8B%E9%9D%92%E8%8B%94-%E3%80%8C%E7%AD%89%E5%AE%83%E9%95%B7%E5%9B%9E%E4%BE%86%E3%80%8D| title=菁桐車站不見青苔 「等它長回來」|publisher=聯合新聞網|date=2015-06-23|accessdate=2015-08-27|language=中文}}</ref>。
