《'''历史的天空'''》，中国大陆史诗剧，改编自[[徐贵祥|徐贵祥]]长篇小说《[[历史的天空|历史的天空]]》，导演[[高希希|高希希]]，主演是[[张丰毅|张丰毅]]、[[李雪健|李雪健]]、[[林永健|林永健]]、[[殷桃|殷桃]]。<ref>{{cite web |url=http://www.btv.org/btvweb/07btv4/2007-09/03/content_216186.htm |title=32集电视剧《历史的天空》 |accessdate=2007 |publisher=北京卫视 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://cpc.people.com.cn/GB/64093/64387/15136113.html |title=徐贵祥谈《历史的天空》：共产党员首先是有血有肉的人 |accessdate=2007 |publisher=中国共产党新闻网 |work= |language=zh-hans}}</ref>

==剧情==
该剧讲述从[[抗日战争|抗日战争]]、[[国共内战|国共内战]]、[[中华人民共和国|中国共产党接管中国大陆]]、[[朝鲜战争|朝鲜战争]]、[[三年自然灾害|大饥荒]]、[[反右运动|反右运动]]、[[文化大革命|文化大革命]]、[[上山下乡运动|上山下乡运动]]、到[[华国锋|华国锋]]、[[叶剑英|叶剑英]]等一举粉碎[[四人帮|四人帮]]的这段时间里的历史。<ref>{{cite web |url=http://military.people.com.cn/GB/other3615/3617/6059078.html |title=电视剧《历史的天空》故事简介  |accessdate=2007 |publisher=人民网 |work= |language=zh-hans}}</ref>
