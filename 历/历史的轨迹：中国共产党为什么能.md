《'''历史的轨迹：中国共产党为什么能'''》（{{lang-en|''' Why and How the CPC Works in China & China!'''}}）由[[中共中央党校|中共中央党校]]著名学者和教授谢春涛主编的一本纪实文学作品，该书是研究[[中国共产党|中国共产党]]执政历史的重要书籍，已经翻译了12种语言，14个版本在全球发售，曾参与第63届[[法兰克福国际书展|法兰克福国际书展]]和2012年[[伦敦国际书展|伦敦国际书展]]。

== 内容 ==
该书共分为13个章节，起点为[[中华人民共和国|中华人民共和国]]建立一直到[[和谐世界|和谐世界]]概念的提出和应用，囊括了中华人民共和国六十年的历史大事件。<ref>{{cite web |url= http://www.jxnews.com.cn/jxrb/system/2011/05/20/011667078.shtml|title=中国共产党为什么能？ |publisher=江西日报 |accessdate= 2011-05-20  |work= |language =zh-cn}}</ref><ref>{{cite web |url= http://news.163.com/11/0728/12/7A23QPJF00014AED.html|title= 中国共产党为什么能|publisher=网易 |accessdate=2011-07-28 |work= |language =zh-cn}}</ref>

== 版本 ==
2011年10月12日，该书参加德国第63届法兰克福国际书展，英文版由英国新经典出版社发行，也是英文版首次发布。<ref>{{cite web|url=http://www.china.com.cn/info/zhuanti/2011sz/2011-10/12/content_23606783.htm|title=《历史的轨迹:中国共产党为什么能》英文版首发|publisher=中国网|accessdate=2011-10-12|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20111015031109/http://www.china.com.cn/info/zhuanti/2011sz/2011-10/12/content_23606783.htm|archivedate=2011-10-15}}</ref>

2012年4月12日，该书参加了2012年度伦敦国际书展，在书展上推出了12个语种，14个版本，包括英文、德文、土耳其文、韩文、日文、俄文、西班牙文等。<ref>{{cite web |url=http://news.china.com.cn/txt/2012-04/12/content_25128814.htm |title= 《历史的轨迹：中国共产党为什么能？》将出姐妹篇|publisher= 中国网|accessdate= 2012-04-12|work= |language =zh-cn}}</ref>
