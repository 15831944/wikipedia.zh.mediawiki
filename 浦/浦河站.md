'''浦河站'''（{{jpn|j=浦河駅|hg=うらかわえき|rm=Urakawa eki|f=y}}）位於[[北海道|北海道]][[浦河郡|浦河郡]][[浦河町|浦河町]]昌平町站通，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[日高本線|日高本線]]上的站。電報略號是'''{{lang|ja|ウカ}}'''。以前是急行「襟裳」的停靠站。

== 車站構造 ==
[[側式月台|側式月台]]1面1線的地面站。以前是[[相對式月台|相對式月台]]2面2線並建有一座天橋，但之後拆除了其中一線。

站房出入口並非面對於國道上，故由國道前往可利用天橋。

有站員駐守，營業時間為8:15 - 16:20（逢星期三、六及每月的第二及四個星期一的營業時間為12:00 - 16:20），但不進行剪票。

設有[[綠窗口|綠窗口]]，且因為沒有設置自動售票機，可利用綠窗口購買短距離的車票。

== 車站周邊 ==
車站靠海側是填海造地的一部份，靠山側則是俗稱西伯利亞街道的舊國道。
在現在的國道與車站間尚存有部分防波堤的遺跡。
* [[國道235號|國道235號]]、[[國道236號|國道236號]]
* 日高振興局廳舍
* 浦河町公所
* 浦河警察署
* 浦河郵局（與日本郵便苫小牧支店浦河集配中心併設）
* 日高信用金庫本店
* 北洋銀行浦河分店
* 日高東農業協同組合（JA日高東）本所
* 日高中央漁業協同組合本所
* 浦河町立堺町小學
* 浦河町立浦河第一中學
* 浦河簡易裁判所
* 浦河大壩
* 浦河神社
* JR北海道巴士、道南巴士「浦河站通」、「公所前」站牌

== 歷史 ==
* 1935年（[[昭和|昭和]]10年）10月24日 - 設立。
* 1982年（昭和57年）12月15日 - 廢止貨物處理的業務。
* 1984年（昭和59年）2月1日 - 廢止行李處理的業務。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為北海道旅客鐵道（JR北海道）轄下的車站。
