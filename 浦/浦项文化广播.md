'''浦项文化广播'''（[[朝鲜语|朝鲜语]]：포항문화방송，[[朝鲜汉字|朝鲜汉字]]：{{lang|ko|浦項文化放送}}；[[英语|英语]]：Pohang Munhwa Broadcasting Corporation）是位于[[韩国|韩国]][[庆尚北道|庆尚北道]]的[[文化广播_(韩国)|文化广播]]（MBC）所屬[[電台|電台]]。

== 所在地 ==
760-905[[庆尚北道|庆尚北道]][[浦项市|浦项市]][[南区_(浦項市)|南区]]大岑洞907-4

== 沿革 ==
*1971年10月1日 - 开局，开始播音（呼号'''HLAV'''）。
*1983年9月1日 - MBC FM4U开始播音（呼号'''HLAV-FM'''，频率为97.9MHz）。
*1983年11月1日 - 开通电视节目。
*1999年11月17日 - MBC标准FM开始播音（呼号'''HLAV-SFM'''，频率为100.7MHz）。
*2012年11月6日 - 模拟电视停播。
