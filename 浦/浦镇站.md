'''浦镇站'''，位于[[江苏省|江苏省]][[南京市|南京市]][[浦口区|浦口区]][[顶山街道|顶山街道]]，现仅办理专用线、专用铁路整车到发货运业务。车站内有南京扬子物流有限责任公司专用铁路、[[中车南京浦镇车辆|中车浦镇]]专用铁路、南京振强工贸中心（大桥四处）专用铁路和中铁十五局集团南京混凝土制品有限公司专用线共4条专用线/铁路）<ref>[http://hyfw.12306.cn/hyinfo/action/FwcszsAction_index?type=1 12306货运站点查询 上海铁路局]</ref>，站场现存英式站房为[[南京市文物保护单位|南京市文物保护单位]]。

==简介==
浦镇火车站建于[[清朝|清朝]][[宣统|宣统]]三年（1911年），是[[津浦铁路|津浦铁路]]最早的火车站之一。西北距[[林场火车站|林场火车站]]4.5公里，东南距南京北站（即[[浦口站|浦口站]]）1.5公里，21世纪初是境内多个大型企业铁路专用线的出岔站。1986年下半年，英式老站房被拆除。现存英式建筑是过去该火车站的仓库，占地面积360平方米，建筑长38米、高6.8米、宽7.4米，青砖外墙，红色彩钢瓦顶，它的砖砌形式、门窗、雨棚形制都是英式建筑风格。2012年，“浦镇火车站英式建筑”被公布为南京市文物保护单位<ref name=ppxw>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1696539  |title = 原南京浦镇火车站文保建筑遭电影剧组破坏，已被叫停并立案 |author =  |date = 2017-05-29 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web|url =http://govinfo.nlc.gov.cn/jssnjfz/zfgb/750525a/201210/t20121009_2829889.shtml?classid=428|title =市政府关于公布第四批南京市文物保护单位的通知|author =|date =2012-03-20|publisher =中国政府公开信息整合服务平台南京市分站|language =|accessdate =|quote =}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{Cite web |url = http://news.dichan.sina.com.cn/2012/03/29/463733.html |title =  第四批南京市文物保护单位名单|author =  |date = 2012-03-29 |publisher =  新浪|language =  |accessdate =  |quote =  }}</ref>。浦镇火车站英式建筑位于江北快速化通道高架桥下方<ref name=fakuan/>。

==事件==
2017年3月，南京秀比特文化传播有限公司受北京某文化传媒有限公司委托，在浦口区为电影《人来人往》前期拍摄搭建场景。为此，南京秀比特文化传播有限公司与南京铁道发展集团有限公司（隶属[[上海铁路局|上海铁路局]][[南京站|南京站]]）签订了该建筑为期一年的租赁合同。随后该公司在未办理任何手续的情况下，擅自拆改浦镇火车站英式建筑。2017年5月，在接到群众举报后，南京市浦口区公安局、文广局、城管局和顶山街道到现场处理，责成相关单位立即停工，报批修复方案后恢复原状，并对违法行为立案查处。2017年6月2日，南京市浦口区文广局对渉事公司处以行政处罚20万元人民币，并要求其遵照法律规定程序复原被破坏的建筑<ref name=ppxw/><ref name=fakuan>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1700469 |title = 南京“浦镇火车站文保建筑遭电影剧组破坏”后续：罚款20万 |author =  |date =  2017-06-04 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。
