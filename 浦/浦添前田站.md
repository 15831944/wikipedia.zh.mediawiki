'''浦添前田站'''（{{jpn|j=浦添前田駅|hg=うらそえまえだえき|rm=Urasoe-maeda eki|f=y}}）是位於[[日本|日本]][[沖繩縣|沖繩縣]][[浦添市|浦添市]]的一個興建中的[[鐵路車站|鐵路車站]]，為日後[[沖繩都市單軌鐵路|沖繩都市單軌鐵路]]伸延時所設置的新站，最初車站名稱曾稱為「'''前田站'''」<ref name="response131103">{{Cite news|url=http://response.jp/article/2013/11/03/209948.html|title=沖縄都市モノレールの延伸区間が着工···2019年春開業目指す|publisher=Response.|date=2013-11-03|accessdate=2013-11-12}}</ref>及「'''浦添御城站'''」等<ref name="ryukyu1">{{Cite web|url=http://www.city.urasoe.lg.jp/article.php/s20090222182151702 |title=てだこの新しい道〜沖縄都市モノレール延長（ご案内）〜 |publisher=[[浦添市|浦添市]] |accessdate=2012-12-11 |deadurl=bot: unknown |archiveurl=https://web.archive.org/web/20100213110328/http://www.city.urasoe.lg.jp/article.php/s20090222182151702 |archivedate=2010-02-13 }}</ref>，2014年12月26日，最終發表了正式的車站名稱，決定稱為「'''浦添前田站'''」<ref>[http://ryukyushimpo.jp/news/storyid-236496-storytopic-82.html てだこ浦西、浦添前田など　ゆいレール浦添延伸、４駅名決まる，]琉球新報，2014年12月26日。{{ja}}</ref> 。

== 車站結構 ==
將採用[[島式月台|島式月台]]設計，共設有1面2線的月台配置，與現時[[儀保站|儀保站]]的設計相同<ref>[http://www.pref.okinawa.jp/site/doboku/toshimono/enchou.html 延長事業について] - 沖縄県、2014年5月2日，2015年7月5日閲覧</ref>。

== 歷史 ==
* 2012年1月25日：[[國土交通省|國土交通省]]批准[[沖繩都市單軌鐵路|沖繩都市單軌鐵路]]經營[[首里站|首里]]～[[日子浦西站|浦西]]共4.1公里的軌道事業特許認可<ref>{{Cite web |date=2012-01-25 |url=http://news.mynavi.jp/news/2012/01/26/005/index.html |title=ゆいレール首里駅から延伸へ - 2019年春開業予定、バスとの乗り継ぎも考慮 |publisher=マイナビニュース|accessdate=2013-11-12}}</ref>。
* 2013年11月2日：進行車站興建起工式<ref name="times131103">{{Cite web |date=2013-11-03 |url=http://article.okinawatimes.co.jp/article/2013-11-03_56124 |title=モノレール延長着工　首里から4駅新設 |publisher=沖縄タイムス |accessdate=2013-11-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20131111212633/http://article.okinawatimes.co.jp/article/2013-11-03_56124 |archivedate=2013-11-11 }}</ref>。
* 2014年12月26日：站名決定採用「浦添前田站」<ref name="ryukyu1"/>。
* 2019年：預計開業<ref name="times131103"/>。
