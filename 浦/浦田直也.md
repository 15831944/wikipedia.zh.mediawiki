'''浦田直也'''（{{jpn|j=浦田 直也|hg=うらた なおや}}，1982年11月10日 - ）是日本的[[歌手|歌手]]、[[演員|演員]]、[[舞者|舞者]]。所屬公司愛貝克思集團，為團體AAA的團長兼主唱，東京都出身。其中，以'''URATA NAOYA'''的身份進行個人活動。<ref>[http://talent.yahoo.co.jp/pf/detail/pp192009 浦田直也] {{webarchive|url=https://web.archive.org/web/20150210180624/http://talent.yahoo.co.jp/pf/detail/pp192009 |date=2015-02-10 }}Yahoo（人物）</ref>

== 來歷 ==
在3歲的時候，在親戚的婚禮上演唱了吉幾三的雪國後，並得到讚賞後就以成為一名歌手作為目標。是AVEX藝術家學院的畢業生。

有過50次參加甄選會而落選的經驗，之後參加過曾經造就過組合[[CHEMISTRY|CHEMISTRY]]的選秀節目，[[東京電視台|東京電視台]]節目ASAYAN舉辦的「男主唱甄選」，結果在1次審查落選。

2001年，參加了為選出擔任[[濱崎步|濱崎步]]的伴舞員的「DOME TOUR 2001」，亦在ASAYAN舉辦的『濱崎步甄選會』中合格，並且在合格者中選出了浦田作為領隊。

之後擔任[[濱崎步|濱崎步]]LIVE、「DOME TOUR 2001」、「STADIUM TOUR 2002」的伴舞。

與AAA的成員西島隆弘、日高光啓、與真司郎、末吉秀太一起擔任[[鈴木亞美|鈴木亞美]]「Eventful」的伴舞。 

在AAA3周年記念武道館演唱會上宣佈將進行個人活動。

== 人物 ==
*歌唱的時候會有獨特的高音域，擅長假音。
*手非常的巧妙，能夠製作指甲彩繪，水鑽商品。擁有非常豐厚的性格。
*與[[濱崎步|濱崎步]]姐弟相稱。<ref>[http://mdpr.jp/news/detail/1406309 AAA浦田直也、浜崎あゆみのバトンに「務めさせていただきます」]modelpress</ref>
*和宇野實彩子被與真司郎視作AAA的哥哥和姐姐。
