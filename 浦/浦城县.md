'''浦城县'''位于中华人民共和国[[福建省|福建省]]最北端，隶属于[[南平市|南平市]]。踞八闽上游，[[仙霞岭|仙霞岭]]山脉与[[武夷山脉|武夷山脉]]交接处，为江西，浙江，福建三省交界处。面积3383.02平方千米。浦城县历来就是兵家必争之地，为闽北交通要地。

== 历史沿革 ==
[[秦朝|秦]]属[[闽中郡|闽中郡]]。[[汉朝|汉]][[始元_(西汉)|始元]]年间设'''冶县'''。[[东汉|东汉]][[建武_(东汉)|建武]]年间改冶县为侯官，[[建安_(东汉)|建安]]年间置'''汉兴县'''。[[三国|三国]][[永安_(孫休)|永安]]年间改吴兴。[[隋朝|隋]][[开皇|开皇]]年间吴兴并入[[建安县|建安县]]。[[唐朝|唐]][[建武_(东汉)|武德]]年间置'''唐兴县'''。[[天授_(武周)|天授]]年间改名'''武宁'''。[[神龍|神龙]]年间复名唐兴，[[天宝_(唐朝)|天宝]]年间定名'''浦城'''。

== 行政区划 ==
全县辖2个街道、9个镇、8个乡：
*街道：南浦街道、河滨街道。
*镇：富岭镇、石陂镇、临江镇、仙阳镇、水北街镇、永兴镇、忠信镇、莲塘镇、九牧镇。
*乡：万安乡、古楼乡、山下乡、枫溪乡、濠村乡、管厝乡、盘亭乡、官路乡。

== 地理 ==
以[[山地|山地]]为主。

最高峰[[营盘尖|营盘尖]]，海拔1664米，位于东北部。属亚热带季风气候。

主要河流[[南浦溪|南浦溪]]。

属中亚热带季风湿润气候区，年平均气温17.4℃，年平均降雨量1780.2毫米，为福建省多雨地带之一。常年平均日照时数为1893.5小时。

== 经济 ==
以农业工业为主，为福建省最大的商品粮基地，近年来工业与服务业在经济中的比重不断上升。

== 方言 ==
浦城县南部通行[[闽北语|闽北语]]，北部通行[[吴语|吴语]]的[[处衢片|处衢片]]方言。
== 文化教育 ==
以闽北文化为主，近十年来县府在小学增设乡土课程学习闽北文化。

==物产==
*盛产稻麦，浦城[[薏米|薏米]]为[[中国地理标志产品|中国地理标志产品]]<ref>[http://kjs.aqsiq.gov.cn/dlbzcpbhwz/bhml/200906/t20090622_118997.htm 《地理标志产品保护规定》实施以来批准的地理标志名录（2005年7月至2009年6月）] {{webarchive|url=https://web.archive.org/web/20110311120900/http://kjs.aqsiq.gov.cn/dlbzcpbhwz/bhml/200906/t20090622_118997.htm |date=2011-03-11 }}</ref>。。
