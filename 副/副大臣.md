'''副大臣'''（{{jpn|j=副大臣|hg=ふくだいじん}}）是[[日本|日本]][[中央省廳再編|中央省廳再編]]後於[[內閣府|內閣府]]與各省設置的[[官職|官職]]。

== 概要 ==
其地位依《{{link-ja|內閣府設置法|内閣府設置法}}》第13條、《{{link-ja|國家行政組織法|国家行政組織法}}》第16條所規定，是廢除原有的政務次官所新設的{{link-ja|政治任用制|政治任用制|政治任用職}}，慣例上由國會議員出任<ref>與國務大臣一樣也可採用國會議員以外的人士，但至今尚未有此情況產生。</ref>。

依各省的[[主任大臣|主任大臣]]、[[內閣官房長官|內閣官房長官]]或[[內閣府特命擔當大臣|內閣府特命擔當大臣]]命令，負責規劃政策及企劃或處理政務。若大臣不在時，可依大臣事前命令代為執行其職務（[[內閣總理大臣|內閣總理大臣]]不在時的代理是由擔任{{link-ja|內閣總理大臣臨時代理|内閣総理大臣臨時代理}}的[[國務大臣|國務大臣]]出任，而非{{link-ja|內閣府副大臣|内閣府副大臣}}）。但由於無法代理國務大臣職權，無法代理出席{{link-ja|閣議|閣議}}。

員額依各府省而異（1人至3人）。其任免由各府省首長大臣提出，[[內閣|內閣]]決行，最後由[[天皇|天皇]]認證（{{link-ja|認證官|認証官}}）。內閣總辭時與大臣一樣失去其職務。是《{{link-ja|國家公務員法|国家公務員法}}》上的{{link-ja|特別職|特別職}}。

此外，設有'''副大臣會議'''以便於內閣府及各省相互協調政策。副大臣會議設於[[內閣官房長官|內閣官房長官]]之下，由[[內閣官房副長官|內閣官房副長官]]（政務）及全體副大臣組成，內閣官房長官負責監督，內閣官房副長官（政務）擔任議長。
副大臣地位高於{{link-ja|大臣政務官|大臣政務官}}與[[事務次官|事務次官]]。然而，副大臣及副大臣會議無法影響{{link-ja|閣議|閣議}}決議案件，閣議決議案件的事前協調是由{{link-ja|事務次官等會議|事務次官等会議}}負責（事務次官等會議在2009年由[[鳩山由紀夫內閣|鳩山由紀夫內閣]]廢除，但在2011年[[東日本大震災|東日本大震災]]後事務次官等事務方再次組成「各府省連絡會議」）。2012年2月10日，[[野田內閣_(第1次改造)|野田內閣]]由部分內閣府副大臣兼任各省副大臣，成為之後的慣例。
英語名稱依各府省廳有Senior Vice-Minister、Parliamentary Senior Vice-Minister、State Minister、State Secretary等譯名。

== 沿革 ==
過去做為政治任用的政務次官，權限小、腳色不明確，被稱為「省廳的盲腸」。1999年，《{{link-ja|國會審議活性化及政治主導政策決定制度確立相關法律|国会審議の活性化及び政治主導の政策決定システムの確立に関する法律|國會審議活性化法}}》廢止國會的政府委員制度及政務次官，新設副大臣與大臣政務官。並依同法第9條創設副大臣會議取代無法律依據的政務次官會議，賦予權力進行實質的省廳間政策協調。

然而在導入新制度後，一反過去[[第2次橋本內閣|第2次橋本內閣]]、[[小渕內閣|小渕內閣]]時代由具閣僚經驗者出任重要官廳的政務次官（外務政務次官[[高村正彥|高村正彥]]、大藏政務次官[[谷垣禎一|谷垣禎一]]、外務政務次官[[町村信孝|町村信孝]]等），小泉內閣的副大臣、大臣政務官多由[[自由民主黨派閥|派閥]]分配。繼任的[[第1次安倍內閣|安倍內閣]]則不再由派閥推薦，改由總理與政黨幹部決定。之後，[[福田康夫內閣_(改造)|福田改造內閣]]任命[[高市早苗|高市早苗]]（經濟產業）與[[鴨下一郎|鴨下一郎]]（厚生勞動），[[第2次安倍內閣|第2次安倍內閣]]任命{{link-ja|小渕優子|小渕優子}}（財務）與[[鈴木俊一_(眾議院議員)|鈴木俊一]]（外務），再次由閣僚經驗者出任副大臣。

== 其他 ==
[[內閣官房副長官|內閣官房副長官]]不僅處理[[內閣官房|內閣官房]]事務，也參與了內閣府部分特定事務，同時也是副大臣會議成員，其薪俸等待遇、職位也與副大臣相當，然而依照職位設置依據、內閣總辭時連帶辭職的有無等，從法律、學術上來看不被視為副大臣。

除了內閣官房副長官外，其薪俸等待遇與副大臣相當的職位有{{link-ja|內閣法制局長官|内閣法制局長官}}、國家公務員倫理審査會常任會長、[[公平交易委員會_(日本)|公平交易委員會委員長]]、[[宮內廳|宮內廳長官]]、部分[[大使|大使]]等。此外，[[日本國會|國會]]的各議院{{link-ja|事務總長 (國會)|事務総長 (国会)|事務總長}}與{{link-ja|法制局長|法制局長}}、[[國立國會圖書館|國立國會圖書館長]]、[[東京高等裁判所|東京高等裁判所]]{{link-ja|高等裁判所長官|高等裁判所長官|長官}}的薪俸等待遇也與副大臣相當。
