《'''如果有一件事是重要的'''》是[[臺灣|臺灣]]創作歌手[[陳珊妮|陳珊妮]]於2008年11月22日發行的[[錄音室專輯|錄音室專輯]]，繼2004年發行《後來 我們都哭了》後睽違四年的全新創作，[[陳珊妮|陳珊妮]]也以這張專輯拿下[[第20屆金曲獎|第20屆金曲獎]]最佳女歌手獎<ref>{{Cite web |url = http://ent.ltn.com.tw/news/breakingnews/234182 |title = 陳珊妮 獲最佳國語女歌手獎 |author = 自由娛樂 |date = 2009年6月27日 |publisher = 自由時報 |language = zh-tw |accessdate = 2016年1月22日 |quote = 第20屆華語金曲歌后出爐，由陳珊妮奪下「最佳國語女歌手獎」。陳珊妮在陳奕迅和孫燕姿手中接過這項獎座。陳珊妮發表感言時，自嘲「我的臉看起來似乎不是很謙虛，但是我真的很謙虛來接受這個獎」。據悉，此次入圍者還包括梁靜茹、蔡健雅、蔡琴和A-Lin，都是極強勁的對手。 }}</ref>。

==發行紀事==
*2008年10月12日：於臺北公館[[THE_WALL|THE WALL]]舉辦「公主同一國祕密搶聽會」首次披露專輯全曲給樂迷試聽
*2009年2月25日：華語音樂聯盟2008年度單項獎4項提名：最佳女歌手、最佳國語專輯、最佳國語歌曲、最佳作詞人
*2009年8月13日：[[陳珊妮|陳珊妮]]以《如果有一件事是重要的》英文版專輯進軍韓國音樂市場，樂曲在Melon、M.net、Daum提供付費下載
*2009年5月15日：《如果有一件事是重要的》入圍第20屆金曲獎最佳專輯製作人、最佳國語專輯、最佳國語女歌手獎
