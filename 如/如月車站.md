'''如月車站'''（{{jpn|j=きさらぎ駅|hg=きさらぎえき|rm=Kisaragi-eki}}）是[[日本|日本]]論壇網站[[2ch|2ch]]上流傳的[[都市傳說|都市傳說]]<ref>[http://llike.net/2ch/fear/kisaragi.htm 2ch原文]</ref>。由於事件沒有實質的證據、疑點重重。有人更認為是為[[鐵道迷|鐵道的愛好者]]編造出來的故事，對於都市鐵道愛好者來說，最後一班電車具有獨特的美感，因此很多人認為如月車站是虛構的。2010年開始，該都市傳說透過網絡傳至[[中國|中國]]，引起網民的熱烈討論。

== 概要 ==
日本論壇網站2ch上流傳數年的都市傳說：一名網名叫蓮實的男子（真實身份或許是一個叫作葉純的女子）在深夜搭乘[[電車|電車]]時，無意間搭到一個不存在的車站「如月車站」；起初她很熱切地將在[[車站|車站]]的所見所聞與2ch網友即時分享，最後卻沒想到自己會下落不明。

事件發生於2004年1月8日晚上，蓮實在晚間11時搭上一列電車，經過20分鐘的行駛後，她疑惑地在2ch上留言說：平時這班車只要7、8分鐘就會停站，今天卻遲遲未停下來。有網友建議他去車長室詢問，蓮實照作後卻沒有獲得[[車長|車長]]的回應。在列車通過一條陌生的[[隧道|隧道]]後，於凌晨12時停在如月車站月台<ref>[http://matome.naver.jp/odai/2139223931502227001 如月車站貼文]”</ref>。

蓮實在車站裡發現空無一人，出站後外面也是空蕩蕩，她用手機網絡搜尋也找不到自己的所在位置。接著她困惑地向家人求救，希望能夠幫忙報警，但卻被警方當作惡作劇電話，原因是事實根本沒有“如月”這個車站。

隨著時間越來越晚，蓮實不斷在2ch上與網友聊天，她在凌晨2時留言說他在附近聽到[[鈴鐺|鈴鐺]]及[[太鼓|太鼓]]的聲音，還出現一名單腳的老伯。網友們驚覺不對並指示蓮實順著隧道迅速離開。最後蓮實在走出隧道後，碰上一個陌生人，而對方還好心表示願意免費載他一程。

蓮實無視網友的阻止上了陌生人的車，並於凌晨3時44分留下最後一篇文章，她說：“駕駛把車往山上開，始終沉默不語，我打算趁機逃跑。”從此以後，蓮實就再也沒在2ch發過文。<ref>[https://www.youtube.com/watch?v=k6l-yZIuYe8 如月車站影片]</ref>

7年以後，有個自稱是蓮實的網友，在原文底下留言，說自己回到一開始搭車的車站了，但是時間轉眼就過去了七年，父母像是甚麼都沒發生過的來車站接他。<ref> [https://home.gamer.com.tw/creationDetail.php?sn=1432595 7年後的留言]</ref>

== 疑點 ==
*2004年的手機可以上網嗎？那時候的基地台在深山普遍嗎？
* 蓮實（或葉純）在晚上11時14分在火車上發文，可是卻說：「我坐的電車應該是11點40分的車。」
* 蓮實在11時44分的留的話不是私鐵沒停，而是問：「我該去車長室看看嗎？」
* 蓮實坐了91分鐘的車後才發現停靠在不明的如月車站。
* 有網友要求蓮實走鐵軌回去，可是如月站與上車的車程全長91分鐘，走路的話要走多久呢？
* 蓮實從火車開往前的地方聽到太鼓跟鈴鐺的聲音，於是他往反方向跑。這時候是凌晨兩點，但當他出火車隧道的時候，是凌晨三點。<ref>[http://system.princediamond.com.hk/coco/archiver/?tid-338.html 首個案疑點] {{webarchive|url=https://web.archive.org/web/20160403184624/http://system.princediamond.com.hk/coco/archiver/?tid-338.html |date=2016-04-03 }}</ref>
* 網友根據敘述推理出蓮實是乘坐位於靜岡縣的私鐵——[[遠州鐵道|遠州鐵道]]鐵道線，從[[新濱松站|新濱松站]]出發的列車，但是查閱列車時刻表發現，從新濱松出發的列車，1到2分鐘就可以到站，全列車不存在5到8分鐘的區間，這與蓮實的敘述（平時差不多5分鐘就到下一站了，最長也不過7,8分鐘而已）矛盾。<ref>
[https://www.zhihu.com/question/24037363/answer/30249622 知乎網友的推理]</ref>

== 車站名稱 ==
2ch原始對話裡沒有出現「如月」這兩個漢字，所謂的如月車站是日文翻譯中文版過來的。原始對話中另有一種解說，認為「Kisaragi」是「[[鬼|鬼]]」的意思，這是當作姓氏的時候，但也有可能翻譯成其他東西，如「木皿儀」。「Kisaragi」和[[漢字|漢字]]發音毫無關係，古語對應的含義有可能是「{{Lang|ja|衣更著}}」（天寒添衣）或者「{{Lang|ja|生更ぎ}}」（草木更生）。這是網上一些解說者的說法之一。<ref>[https://www.zhihu.com/question/24037363 知乎討論]</ref>

== 發展 ==
事件過後，網上出現第二名網友「@radio_buna」<ref>[http://matome.naver.jp/odai/2131242126037914001 第二名造訪者] {{webarchive|url=https://web.archive.org/web/20160322153519/http://matome.naver.jp/odai/2131242126037914001 |date=2016-03-22 }}</ref>自稱造訪了“如月車站”，並在網上公開該車站及電車的照片。這第二名訪客指出，他是在[[關東地區|關東地區]]搭乘電車，在過了[[千葉|千葉]]以後，才發覺自己到了一個奇怪的車站；這裡的時間比網路上快了1個多小時，且當晚關東地區明明傳出有5級地震發生，他卻一點都沒感覺。幸運的是，第二名造訪者順利離開“如月車站”，他在進入一間超商後，車站就消失了，後來他查看票卡上的乘車資訊，也沒有“如月車站”的出站紀錄。有網友對比他上傳的照片，發現車站的站牌樣式竟是屬於[[關西地區|關西地區]]，而電車也不是都會區附近的車種，有網友認為這是一個玩笑，但也有人堅信先後兩名同是“[[受害者|受害者]]”，確實進入了“如月車站”的平行空間。

在2012年，網上出現第三位<ref>[http://togetter.com/li/358241 第三位造訪者]</ref>自稱人進入了「如月車站」，暱稱為「{{lang|ja|びゃくらん}}」（現改為闢藍）<ref>[https://twitter.com/__Byakuran__ 第三位造訪者網誌]</ref>，他的[[Twitter|Twitter]]往前紀錄被隱藏起來。事發2012年8月19日晚上11時，電車上原本是女生的廣播變成男生的聲音廣播：「接下來是終點站如月車站」。[[月台|月台]]上沒有站牌，[[剪票口|剪票口]]是關閉的，完全沒有任何人。打電話給任何人都無法接通。第三位的他開始在Twitter上向人求助。他表示只要拍照，手機會顯示記憶體不足而強制退出，即使刪除了手機裡的照片還是一樣不能拍照，更無法用手機取得目前的所在位置。他更表示自己身上帶有手機的電池和充電器，確保手機是有足夠的電源。原本他想試著從販賣機上找到此處地址，卻發現地址的部分已經被刮除了，在時刻表旁發現了一些無法辨認的文字。這時他想離開車站，卻被網友阻止。由於無論打電話，重要的人回電接起來後卻沒有聲音，令他情緒一度變得很暴躁。最後他睡在車站。半個小時後因為朋友「{{lang|ja|ゆめちゃん}}」的電話驚醒，醒來後發現自己睡在[[神奈川縣|神奈川縣]]的[[小田原站|小田原站]]。<BR><ref>[http://news.gamme.com.tw/338057 都市傳說的第三位目擊者]</ref>

有人認為如月車站只是網上的創作，其原型為[[東海地方|東海地方]]的[[天龍濱名湖鐵道|天龍濱名湖鐵道]]，其中在[[浜北站|浜北站]]有一個支線可以到[[宮口站|宮口站]]，中間有一個大[[神社|神社]]，由於當地的神主和[[巫女|巫女]]寧願開車也不要搭鐵路，此車站後來就荒廢了，廢線後偶爾會有通往舊路的不明班車，當地居民認為這是[[狐仙|狐仙]]的玩笑，所以就傳出了"不要搭該車"的都市傳言。<ref>[http://matome.naver.jp/odai/2134539117836413201?&page=1 naver討論]</ref>

== 目擊者 ==
* 2004年1月8日晚上11時14分，網名為「はすみ(葉純) ◆KkRQjKFCDs」的女性網民貼文說從靜岡縣濱松市新松濱車站乘電車回家，自稱進入如月車站。
* 2011年8月2日晚上10時29分，用戶為「@radio_buna」<ref>[http://yoshizokitan.blog.shinobi.jp/Entry/6450/ 第二名日記]</ref>說自己在電車上睡過頭，醒來發現電車停在「きさらぎ駅」，事後證實是惡作劇。
* 2012年8月19日晚上8點19分，用戶為「@__Byakuran__」<ref>[http://news.gamme.com.tw/338057 第三名日記]</ref>在[[Twitter|Twitter]]留言說玩手機時玩過頭，察覺時電車停在「きさらぎ駅」事後在車站睡著，醒來後才發現自己在小田原站。
