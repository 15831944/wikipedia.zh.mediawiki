《'''如果世界巡迴演唱會'''》是[[台灣|台灣]]女[[歌手|歌手]]、[[S.H.E|S.H.E]]成員[[田馥甄|田馥甄]]的第一次大型巡迴演唱會，並且是首次以個人身份站上[[台北小巨蛋|台北小巨蛋]]，創下新的單飛里程碑。此次巡演自2014年12月6日、7日在[[台灣|台灣]][[台北|台北]]揭開序幕，並於2017年9月2日、3日在[[台灣|台灣]][[高雄|高雄]]作為最終場結束本次巡演，巡演場次總共有38場演出。

[[File:Hebe_WuHan_03SEP2016.jpg|thumb]]

== 巡演歷程 ==
2014年8月10日在S.H.E《[[Together_Forever世界巡迴演唱會|Together Forever世界巡迴演唱會]]》臺北Encore場上，Selina和Ella宣佈12月田馥甄在小巨蛋有個唱，引爆台下全場氣氛<ref>[http://ent.ltn.com.tw/news/paper/803459 S.H.E攻蛋Selina忙作媒 Hebe黃腔逗尤秋興]</ref>，之後接著此次巡迴演唱會自2014年12月6日、12月7日於[[台北小巨蛋|台北小巨蛋]]揭開序幕<ref>[http://www.chinatimes.com/realtimenews/20140903005155-260404 田馥甄挑大梁 年底唱進小巨蛋]</ref>，兩場共2.2萬張門票於售票日當天10分鐘內完售<ref>[http://www.appledaily.com.tw/realtimenews/article/entertainment/20140906/464959 Hebe演唱會2.2萬張票　10分鐘賣光]</ref>。演唱會由[[必應創造|必應創造]]製作。

2016年3月5日，於[[新加坡室內體育館|新加坡室內體育館]]舉辦《如果世界巡迴演唱會》1.0版最終場，並宣布《如果世界巡迴演唱會》改版第二階段2.0 Plus版將於[[香港|香港]][[紅磡體育館|紅磡體育館]]展開。

2016年6月17日，《如果世界巡迴演唱會》在[[香港紅磡體育館|香港紅磡體育館]]展開改版第二階段《如果世界巡迴演唱會》2.0 Plus版，同時也是田馥甄首次以個人身分登上紅館開唱。

2016年12月9日、10日、11日，《如果世界巡迴演唱會PLUS》相隔兩年唱回家鄉台灣於台北小巨蛋舉行，也恰巧是《如果巡演Plus》的第9、10、11場，並且3天票房高達8250萬元<ref>[http://ent.ltn.com.tw/m/news/breakingnews/1912748 田馥甄化身「銀幻女神」 演唱會吸金8250萬]</ref>，之後於台北三場演唱會後接著宣布2017年將繼續展開第三階段的海外巡演。

2016年12月9日，於《如果世界巡演》Plus台北場首場當天，發行收錄了2014年12月6日、7日在台北小巨蛋以及2015年9月19日高雄巨蛋舉行的「如果世界巡迴演唱會」的現場影音《[[IF_ONLY_如果田馥甄巡迴演唱會|IF ONLY 如果田馥甄巡迴演唱會LIVE]]》。

2017年5月底，宣布《如果世界巡迴演唱會》plus將再度唱回家鄉台灣，並將於2017年9月2日、3日在高雄的[[高雄巨蛋|高雄巨蛋]]連續舉辦兩場演唱會，並且此兩場演唱會也是《如果世界巡迴演唱會》的最終場，兩場演唱會2.2萬張門票於門票開賣後30分鍾內完售 ，而最後這兩場演唱會總票房共4800萬元<ref>[http://ent.ltn.com.tw/m/news/breakingnews/2182013 田馥甄高雄開唱吸金4800萬 水滴吊台唯美登場]</ref>。

此次巡演自2014年12月巡迴至2017年9月，歷經1002天，共走過23個城市，共計有38場演出<ref>[http://star.ettoday.net/news/1003431?from=fb_et_star 田馥甄「獨自跑巡演3年」　想念Selina、Ella一起瘋！]</ref><ref>[http://m.cna.com.tw/news/amov/201709040122.aspx 走過23個城市 田馥甄想伴家人品味日常]</ref>。

2017年12月8日，發行收錄了於2016年12月9日、10日、11日在台北小巨蛋舉行的「如果世界巡迴演唱會PLUS」的現場影音《[[IF＋如果田馥甄巡迴演唱會plus_LIVE|IF＋如果田馥甄巡迴演唱會plus LIVE]]》。

== 名人評價 ==
女藝人[[劉若英|劉若英]]：「田馥甄Hebe演唱會。抽離自我，逃離現實。非常特別的演唱會。值得思考的演唱會。」<ref>[http://www.weibo.com/1681213010/BzMsu6NyL?type=comment#_rnd1418049988354 12月7日劉若英新浪微博提及]</ref>

資深媒體人[[陳樂融|陳樂融]]：「不是Hebe粉，卻發現田馥甄實力佳。可冷可熱可頑皮可灑脫，多數歌都像新歌，但緊盯著大銀幕的歌詞也靜靜被她收服。滿座粉拍手少，因為一手拿螢光棒一邊忙尖叫。女多於男（很多），不再計較是否唱腔有王菲影子，這一晚，七八年級心目中的田馥甄差不多就是五六年級心目中的王菲（吧）。」<ref>[http://www.weibo.com/chenlerong?1418050139859#1418050155724 12月7日陳樂融新浪微博提及]</ref>

著名藝術家兼大學教師[[曲家瑞|曲家瑞]]：「我覺得田馥甄是一個努力、用功、有天份、有才華、才氣，然後她很願意去嘗試、去突破的一個女生，沒有令在場任何一個人失望，而是讓我們獲得更多更多帶走。」「她聲音是天賴，當她飆到高音時，幾乎是無可取代，你就會覺得說她的可能性很大。」<ref>節目《媒事來哈啦》 2014.12.19</ref>
