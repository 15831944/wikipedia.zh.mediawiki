《'''如果早知道男生也會被性侵'''》是中華民國[[國家教育研究院|國家教育研究院]]制作的一部[[性別平等|性別平等]]教育宣傳影片<ref name="东森1">{{cite web|url=http://www.ettoday.net/news/20130715/240793.htm|title=「杰哥，不要啊......」性侵宣導片暴紅　劇情確有其事|date=2013-07-15|accessdate=2017-06-23|work=东森新闻云|deadurl=no|archiveurl=https://web.archive.org/web/20171223102533/https://www.ettoday.net/news/20130715/240793.htm|archivedate=2017-12-23}}</ref>，由陳北川導演，傳愛編劇小組編劇，在[[基隆市|基隆市]]拍攝。剧情講述了一名男學生阿瑋被男子杰哥[[性侵犯|性侵犯]]的故事，在網路上引起巨大反響。

== 製作 ==
《如果早知道男生也會被性侵》由[[中華民國教育部|中華民國教育部]]委託[[國家教育研究院|國家教育研究院]]製作<ref name="东森1"/>，由傳愛家族傳播文化事業有限公司承製，導演陳北川執導。在基隆市取景拍摄<ref name="nownews">{{cite web|url=http://www.nownews.com/n/2013/07/13/211802|title=「杰哥不要啦！」　教部性侵宣導片逆勢宣傳爆紅|date=2013-07-13|accessdate=2017-06-23|work=nownews|deadurl=no|archiveurl=https://web.archive.org/web/20170316051823/http://www.nownews.com/n/2013/07/13/211802|archivedate=2017-03-16}}</ref><ref name="东森1"/>，取景地点包括[[暖暖高中|暖暖高中]]、新豐街等<ref name="appledaily1">{{cite web|url=http://www.appledaily.com.tw/realtimenews/article/new/20140616/417413/|title=杰哥在這獵走阿瑋　6拍攝景點公開|author=饒偉生|date=2014-06-16|accessdate=2017-06-23 |work=苹果日报}}</ref>，主題曲由傳愛創意小組作詞、馬路大作曲。

== 劇情大綱 ==
[[File:杰哥不要.jpg|thumb]]
阿瑋（黃耀緯飾）與阿嬤吵架後，翹家和朋友彬彬（黃韶麒飾）去網咖玩。兩人在網咖被杰哥（伍嘉緯饰）盯上，在出網咖後被杰哥搭訕。杰哥以「家中有很多吃的」、「幫助過很多逃家少年」等理由，誘使他們去自己家裡玩。杰哥帶兩人去超商買東西，並趁機帶了幾罐啤酒回家。在家中，杰哥將兩人灌醉後企圖對阿瑋性侵犯，雖然阿瑋大喊「杰哥不要！」，但杰哥斥喝「聽話，讓我看看！」後性侵阿瑋得逞，並威脅其不-{准}-將這件事說出去<ref name="东森2">{{cite web|url=http://www.ettoday.net/news/20130625/230107.htm|title=男生也會被性侵　阿瑋淚崩：杰哥，不要啊......|date=2013-06-25|accessdate=2017-06-23|work=東森新聞雲|deadurl=no|archiveurl=https://web.archive.org/web/20170611174330/http://www.ettoday.net/news/20130625/230107.htm|archivedate=2017-06-11}}</ref>。事後，阿瑋的身心受到創傷，他在公園遇見志工媽媽淑惠（張昌緬飾）時，聽見她提及性侵犯相關詞彙而大叫，因而被淑惠注意，在她的帮助下，阿瑋吐露出了自己的遭遇，並接受淑惠的建議，準備報警、驗傷、通報學校，讓全案進入司法-{程序}-<ref name="appledaily2">{{cite web|url=http://www.appledaily.com.tw/realtimenews/article/new/20140609/413013/|title=「杰哥不要」2年後　揪出淑惠阿姨是主謀|author=饒偉生|date=2014-06-09|accessdate=2017-06-23|work=蘋果日報|deadurl=no|archiveurl=https://web.archive.org/web/20170624135320/http://www.appledaily.com.tw/realtimenews/article/new/20140609/413013|archivedate=2017-06-24}}</ref>。影片最後由旁白說明性侵害的定義。

== 反響 ==
《如果早知道男生也會被性侵》2013年被人上載至網路<ref name="nownews"/>{{efn|一說2012年<ref name="appledaily1"/>。}}，由於主角杰哥和阿瑋逼真的演技<ref name="東森2"/>，加上網友認為劇情老哏、可笑<ref name="nownews"/>，影片引起了民衆熱烈討論<ref name="東森2"/>。影片在2013年7月點閱率达到了近80萬，創下中華民國政府單位宣導片的紀錄<ref name="nownews"/>。2014年6月已獲得超過200萬次點閱<ref name="appledaily2"/>。片中的部份臺詞如「如果早知道…」、「杰哥，不要啦」等也成爲了網路流行語<ref name="nownews"/>，會在提及[[強姦與性別#對男性的強姦|男性對男性性侵害]]時的文章使用<ref>{{cite news |url = http://news.ltn.com.tw/news/society/breakingnews/1845973 |author = 陳慰慈 |title = 杰哥不要翻版！11歲男童到網友家玩 被彈小雞雞 |publisher = 自由時報 |date = 2016-10-04 |accessdate = 2016-12-29 |deadurl = no |archiveurl = https://web.archive.org/web/20161229171957/http://news.ltn.com.tw/news/society/breakingnews/1845973 |archivedate = 2016-12-29 }}</ref>。网友亦將本片改編成各種版本<ref name="nownews"/>。

但是《如果早知道男生也會被性侵》也因為缺乏性別平等教育的功能而遭到部分人士批評，「愛與和平與幹」成員李佳霖撰文評論，認為該片不但沒有教學功能，反而加深「年長男同志會誘拐、侵害少年」的刻板印象，並認為該片強化了愛滋感染者與性侵犯受害者間的連結，扼殺青少年的性慾實踐。<ref name="苦勞">{{cite web|url=http://www.coolloud.org.tw/node/74844|title=「杰哥，不要」──性別平等教育的荒誕與可笑|author=李佳霖|date=2013-07-03|accessdate=2017-06-24|work=[[苦勞網|苦勞網]]|deadurl=no|archiveurl=https://web.archive.org/web/20170627135741/http://www.coolloud.org.tw/node/74844|archivedate=2017-06-27}}</ref>

飾演杰哥的伍嘉緯和飾演阿瑋的黃耀緯也因此走紅。有人照著影片中簡訊的電話號碼回撥找伍嘉緯<ref name="東森2"/>，有人甚至上門邀約伍嘉緯演杰哥<ref name="專訪">{{cite web|url=http://www.ettoday.net/news/20130715/240838.htm|title=專訪／「杰哥」伍嘉緯：我現在麵包供不應求|author=賴淑惠、高鴻銘|date=2013-07-15|accessdate=2017-06-28|work=ettoday|deadurl=no|archiveurl=https://web.archive.org/web/20170510115113/http://www.ettoday.net/news/20130715/240838.htm|archivedate=2017-05-10}}</ref>。而黃耀緯則表示，這部短片走紅之後，他走在路上、吃滷肉飯，都會有人過來問：「請問你是被性侵的那個嗎？」、「欸你被性侵我們全家都有看喔！」。而他的好朋友亦經常拿影片對他開玩笑，讓他「都要發火了」。<ref>{{cite web |author1=林映妤 |title=杰哥不要！本尊出現了　4年來性侵玩笑「開到我要發火」 |url=https://star.ettoday.net/news/1036241 |website=ETtoday |accessdate=2018-10-04 |language=zh-TW}}</ref>
