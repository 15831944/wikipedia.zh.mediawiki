'''宋真宗趙恒'''（{{bd|968年|12月23日|1022年|3月23日|catIdx=Z趙}}），原名'''趙德昌'''，又曾名'''趙元休'''、'''趙元侃'''，[[北宋|北宋]]的第三位皇帝。他是[[宋太宗|宋太宗]]的第三个儿子，登基前曾被封为[[宋真宗|韩王]]、[[宋真宗|襄王]]和[[宋真宗|寿王]]，[[淳化_(年號)|淳化]]五年（994年）九月，加檢校太傅行[[開封府尹|開封府尹]]，[[至道|至道]]三年（997年）四月登基後離任，以太子身份继位，在位25年。

宋真宗是著名諺語「書中自有黃金屋，書中自有顏如玉<ref>'''原文：'''富家不用买良田，书中自有千钟粟。安居不用架高堂，书中自有黄金屋。出门无车毋须恨，书中有马多如簇。娶妻无媒毋须恨，书中有女颜如玉。男儿欲遂平生志，勤向窗前读六经。</ref>」的作者。

== 生平 ==
宋[[乾德_(北宋)|乾德]]六年十二月初二日（968年12月23日）趙恒生於東京開封府第，是趙光義第三子，與長兄楚王趙元佐同母，初名趙德昌。幼時英睿，姿表特異。與諸王嬉戲時，喜歡作戰陣之狀，自稱元帥。宋太祖喜愛他，將他養在宮中。

趙恆最初並非皇位繼承人，宋太宗最初屬意長子[[趙元佐|趙元佐]]為太子，但因趙元佐自秦王廷美亡后精神失常，而且因病傷人及在宮內縱火，最後被廢。太宗本計劃立次子[[趙元僖|趙元僖]]為太子，但趙元僖又早逝。趙元僖死後，太宗立三子趙元侃為太子，赐名恆，[[至道|至道]]三年(997年)宋太宗箭傷復發而駕崩，趙恆繼位為帝，是為宋真宗。

[[景德|景德]]元年（1004年）[[辽国|辽国]]入侵宋，宋朝大多数大臣建议不抵抗，以宰相[[寇準|寇準]]为首的少数人极力主张抵抗，最后宋真宗御驾亲征，双方在澶州（今河南濮阳附近）相交，宋胜後，真宗决定就此罢兵，以每年向辽纳白银十万両、绢二十万匹来換取与辽之間的和平，定[[澶渊之盟|澶渊之盟]]。这是宋朝以[[岁币|岁币]]换取和平的开始。

真宗時，[[鐵器|鐵製工具]]製作進步，土地耕作面積增至5.2億畝（[[宋太宗|太宗]][[至道|至道]]二年，耕地有3億多畝），又引入[[暹罗|暹罗]]良种水稻，農作物產量倍增，紡織、染色、造紙、製瓷等手工業、商業蓬勃發展，景德年间，專門製作[[瓷器|瓷器]]（原名白崖场）的昌南镇遂改名为[[景德镇|景德镇]]，貿易盛況空前，史称[[咸平之治|咸平之治]]。

宋真宗统治后期，以[[王钦若|王钦若]]和[[丁谓|丁谓]]为宰相，信奉[[道教|道教]]和[[佛教|佛教]]，[[大中祥符|大中祥符]]元年（1008年）称受天书，封[[泰山|泰山]]、祀[[汾阳|汾阳]]，詔令[[丁谓|丁谓]]修建了玉清昭应宫，极侈土木，七年始成，有房屋近三千间，“小不中程，虽金碧已具，必毁而更造，有司不敢计其费。”，[[封禪|封禪]]给民众造成极大的负担<ref>《[[續資治通鑑|續資治通鑑]]》卷三十三記載天禧元年（1017年）宰相[[王旦|王旦]]临终時，对儿子说：“我別無過，惟不諫天書一節，為過莫贖。我死之后，当削发披缁以敛。”諸子欲奉遺令，[[楊億|楊億]]以為不可，乃止。</ref>。但亦有认為封禪其實只是為了要震懾強鄰遼國，在軍事屢居下風之際進行一種天命正當性的競爭<ref>《宋史》：“他日修《[[辽史|辽史]]》，见契丹故俗而后推求宋史之微言焉。宋自太宗幽州之败，恶言兵矣。契丹其主称天，其后称地，一岁祭天不知其几，猎而手接飞雁，鸨自投地，皆称为天赐，祭告而夸耀之。意者宋之诸臣，因知契丹之习，又见其君有厌兵之意，遂进神道设教之言，欲假是以动敌人之听闻，庶几足以潜消其窥觎之志欤？”</ref>，但之後發生了[[北宋帽妖案|北宋帽妖案]]引發社會動盪，其內在原因史家至今各有主張。

宋真宗好文學<ref>《庚溪诗话》：“真宗皇帝听断之暇，唯务观书。每观一书毕，即有篇咏，命近臣赓和，可谓好文之主也。”</ref>，也是一名诗人，他比较著名的诗有《励学篇》、《[[勸學詩|勸學詩]]》等。一般人常說的「書中自有黃金屋、書中有女顏如玉」就是出自他的[[勸學詩|勸學詩]]。

[[乾興|乾興]]元年二月十九日（1022年3月23日）于汴京延慶殿駕崩，享年五十五歲，在位共二十五年。群臣為其上諡號為文明章聖元孝皇帝，廟號真宗。十月十三日，葬於永定陵；二十三日附祭太廟。太子[[宋仁宗|宋仁宗]]继位，史称“仁宗以天书殉葬山陵，呜呼贤哉！”七年後，昭应宫遭雷击，被大火焚為灰烬。《宋史》稱真宗“及澶洲既盟，封禅事作，祥瑞踏臻，天書屢降，導迎奠安，一國君臣如病狂然，吁，可怪也。

== 評價 ==
* [[元朝|元朝]]官修[[正史|正史]]《[[宋史|宋史]]》[[脱脱|脱脱]]等的評價是：“真宗英悟之主。其初践位，相臣李沆虑其聪明，必多作为，数奏灾异以杜其侈心，盖有所见也。及澶洲既盟，封禅事作，祥瑞沓臻，天书屡降，导迎奠安，一国君臣如病狂然，吁，可怪也。他日修《辽史》，见契丹故俗而后推求宋史之微言焉。宋自太宗幽州之败，恶言兵矣。契丹其主称天，其后称地，一岁祭天不知其几，猎而手接飞雁，鸨自投地，皆称为天赐，祭告而夸耀之。意者宋之诸臣，因知契丹之习，又见其君有厌兵之意，遂进神道设教之言，欲假是以动敌人之听闻，庶几足以潜消其窥觎之志欤？然不思修本以制敌，又效尤焉，计亦末矣。仁宗以天书殉葬山陵，呜呼贤哉！”<ref>[http://www.guoxue123.com/shibu/0101/00songs/007.htm 《宋史·真宗本紀》]</ref>

== 逸事 ==
宋真宗派[[曹利用|曹利用]]去辽国签订[[澶渊之盟|澶渊之盟]]之际，告诉曹“迫不得已，虽百万亦可!”。[[寇准|寇准]]知道后，指着曹怒道“超过30万两，提人头来见”。最后，经过曹利用再三讨价还价，以每年白银10万两、绢帛20万匹，订立澶渊之盟。曹利用回到宋朝之后，真宗急问金额多少，曹利用不敢直说，只竖起3根指头，真宗以为是300万两，大惊失声脱口而说，“太多了”，过了一会又安慰道：“金额是太多了，但就此把事情了结也好”，待知道是30万时，如释重负，转忧为喜，对曹利用大加赏赐。

另外，宋真宗亦是著名戲曲'''《[[貍貓換太子|貍貓換太子]]》'''第三主角。（另兩位為[[宋仁宗|宋仁宗]]和[[李宸妃|李宸妃]]，相關人物為[[太監|太監]]陳琳、郭槐、[[章献皇后|劉太后]]、王親貴族[[趙元儼|八賢王]]、狄太后及[[婢女|婢女]]寇珠。）
