'''宋時選'''（{{bd|1922年|1月15日|2010年|4月18日|catIdx=Song宋時選}}），[[中国国民党|中国国民党]]大老。[[浙江省|浙江省]][[奉化|奉化]]县（今[[宁波市|宁波市]][[奉化区|奉化区]]）人，曾任[[反共救国团|反共救国团]]主任、[[裕台企业|裕台企业]]董事长、[[中国广播公司|中国广播公司]]董事长、[[朝阳科技大学|朝阳科技大学]][[董事长|董事长]]。

== 身世 ==
宋时选出身显赫。宋时选[[籍贯|籍贯]]和出生地均与[[蒋介石|蒋介石]]相同，是浙江省宁波府奉化縣溪口镇人，生于下跸驻村。宋时选的祖母是[[毛福梅|毛福梅]]（蒋介石原配夫人）的姐姐。宋时选的母亲是[[毛思诚|毛思诚]]（蒋介石早期导师）的女儿[[毛英梅|毛英梅]]。<ref name="宋时选家世">{{cite web|url=http://wangxiaofang1.blshe.com/post/13103/606641 |title=国民党大老宋时选魂归故土 |date=2010年10月25日 |accessdate=2011年8月1日}}</ref>宋时选与国民党高层人物诸如[[毛高文|毛高文]]（教育部长）、[[周宏涛|周宏涛]]（[[总统府|总统府]]机要秘書）均为远房亲戚。

== 早年 ==
宋时选早年就讀於家乡[[奉化|奉化]]的[[武岭学校|武岭学校]]、而後畢業於江西省私立政法專科學校市政科。

== 政治生涯 ==
宋时选是国民党内部“太子系”的重要人物，跟随[[蒋经国|蒋经国]]多年，曾担任蒋经国的机要秘书。政治生涯发端于“[[中国青年反共救国团|中国青年反共救国团]]”（救国团）。1960年代，宋时选追随[[李焕|李焕]]，负责主持救国团。1967年，宋出任救国团总部主任秘书，后升任“救国团”执行长，继而升任救国团副主任，在救國團內創辦[[張老師|張老師中心]]<ref>{{Cite news|url=http://www.cna.com.tw/news/aedu/201404180147-1.aspx |title=一號張老師宋時選 銅像揭牌 |date=2014-04-18 |publisher=中央社 }}</ref>，以對青少年心理輔導，並解答學業、就業、家庭等問題。。1978年。宋时选出任救国团主任。

宋时选曾获选国民党第十二和十三届中央委员。

宋时选退出政界后，服務于企业界，曾先后担任[[裕台企业|裕台企业]]董事长、[[中国广播公司|中国广播公司]]董事长等职务。宋时选也涉足教育，逝世前是朝阳科技大学董事长。

== 晚年 ==
2010年4月18日晚10时，宋时选病逝于[[台北|台北]][[振兴复健医学中心|振兴复健医学中心]]，享寿90。<ref name="宋時選逝世">{{cite web|url=http://www.libertytimes.com.tw/2010/new/may/3/today-p7.htm|title=宋時選 出生在浙江的台灣歐吉桑|publisher=自由电子报|date=2010-05-03|accessdate=2011年12月23日|deadurl=yes|archiveurl=https://web.archive.org/web/20100506034023/http://www.libertytimes.com.tw/2010/new/may/3/today-p7.htm|archivedate=2010年5月6日}}</ref>宋时选临终思念家乡，慨叹生前未能为身在奉化家乡的父母尽孝道，遗嘱骨灰归葬大陆家乡，埋于父母墓地旁边，永远陪伴父母。<ref name="宋时选遗愿">{{cite web|url=http://zfxx.ningbo.gov.cn/gk/jcms_files/jcms1/web50/site/zfxxgk/article.jsp?jdid=50&cid=8445&infoid=88503|title=宋时选叶落归根遗愿实现 骨灰归葬家乡奉化|publisher=中国宁波网|accessdate=2011年8月1日}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 评价 ==

宋时选虽然出身显赫，在[[中國国民党|中國国民党]]高层人脉广阔，國民黨內大多數人稱他為「宋公」（國民黨黨內一般以名的首字尊稱，而非姓，如[[吳伯雄|吳伯雄]]「伯公」、[[許水德|許水德]]「水公」。）<ref>[http://dailynews.sina.com/bg/tw/twpolitics/phoenixtv/20110125/16282186574.html  國民黨“公”“老”滿天飛 有些馬英九聽了就想笑]，2016年9月15日 </ref>但因行事低调、平易近人、不搞特权，宋也被誉为“台灣歐吉桑”（歐吉桑：[[日語|日語]]，對男性長者的尊稱）。<ref name="宋时选往事">{{cite web|url=http://www.libertytimes.com.tw/2010/new/may/9/today-p6.htm|title=追思宋公 兒外島當兵 宋時選不搞特權干預|publisher=自由电子报|date=2010-05-09|accessdate=2011年8月1日|deadurl=yes|archiveurl=https://web.archive.org/web/20100512175814/http://www.libertytimes.com.tw/2010/new/may/9/today-p6.htm|archivedate=2010年5月12日}}</ref><ref name="宋时选评价">{{cite web|url=http://www.etaiwannews.com/etn/news_content.php?id=1249736&lang=tc_news |title=宋時選行事謙遜 被譽台灣歐吉桑 |publisher=Taiwan News Online  |date=2010-05-07 10:46 PM  |accessdate=2011年8月1日}}</ref>宋时选逝世后获各方高度评价和追思。<ref name="宋时选追思">{{cite web|url=http://news.ifeng.com/taiwan/1/detail_2010_05/11/1506817_0.shtml |title=國民黨大老宋時選追思悼念會舉行(圖) |publisher=台灣頻道_鳳凰網  |date=2010年5月11日  |accessdate=2011年8月1日}}</ref>
