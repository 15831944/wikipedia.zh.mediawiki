'''宋宝罗'''（{{bd|1916年|10月21日|2017年|9月2日}}），原名'''保罗'''，生于[[北京|北京]]，[[中华民国|中华民国]]及[[中华人民共和国|中华人民共和国]][[京剧|京剧]]老生演员、书画家、篆刻家<ref name=gongren>{{Cite web |url = http://epaper.zjgrrb.com/html/2017-03/04/content_2579705.htm?div=1 |title = 梨园百岁老寿星宋宝罗——粉墨丹青皆入戏 |author =  |date =2017-03-04  |publisher =  浙江工人日报|language =  |accessdate =  |quote =  }}</ref>。

==生平==
宋宝罗的父亲宋永珍自幼学戏，16岁搭班演出，后由[[侯俊山|侯俊山]]推荐，来北京投奔[[田际云_(京剧演员)|田际云]]，搭入“玉成班”，因擅演《红梅阁》而出名。宋宝罗的母亲艺名叫“金翠凤”，是京剧丑角演员，擅演《十八扯》，被誉为“坤伶第一名丑”。青年“坤伶”[[孟小冬|孟小冬]]、[[孟丽君|孟丽君]]、[[碧云霞|碧云霞]]、[[张蕴新|张蕴新]]等都是她弟子，称她“干妈”，故有人戏称她是戏曲界“名妈”<ref name=gongren/>。

宋宝罗在他母亲所生的十个子女中排行第四。6岁时起每晚或每周日，母亲带着他与兄弟姐妹到北京[[城南游艺园|城南游艺园]]后台看戏。他遂对演戏感兴趣，回家以后独自模仿唱戏，父母乃决定让宋宝罗学戏。宋宝罗先后师从[[张立英|张立英]]、[[黄少山|黄少山]]、[[张春彦|张春彦]]、[[朱殿卿|朱殿卿]]、[[宋继亭|宋继亭]]等老师，学了20余出戏。7岁时，宋宝罗首次登台，在[[北京天桥|北京天桥]]东歌舞台剧场，演老生戏《上天台》，被观众视为“梨园小神童”。次日，演[[老旦|老旦]]戏《滑油山》、《游六殿》。第三日，演[[花脸|花脸]]戏《探阴山》。《[[益世报|益世报]]》、《群强报》等报纸报道了宋宝罗演出的消息<ref name=gongren/>。

1924年2月19日，[[冯玉祥|冯玉祥]]和[[李德全|李德全]]在[[南苑机场|南苑机场]]举行婚礼，群益社以三天堂会戏助演。第一天，由入科不足一年的13岁的宋遇春（宋宝罗的二哥）和7岁的宋宝罗演《[[张松|张松]]献地图》，宋遇春饰[[关羽|关羽]]，宋宝罗饰张松。第三天，哥俩演《斩颜良》，宋宝罗饰关羽，宋遇春饰[[颜良|颜良]]，因宋宝罗耍不动关羽的[[青龙偃月刀|青龙偃月刀]]，饰颜良的哥哥宋遇春刚开打便倒下了，引得台下的冯玉祥和官兵们大笑<ref name=wy/>。

9岁时，宋宝罗与哥哥在父亲率领下，到各地巡回演出，有场花脸戏因化妆师不慎用了烟锅灰勾脸，使宋宝罗右眼中毒失明，一时无法登台。直到十三、四岁时，经治疗眼睛才恢复视力，宋宝罗重新登台。15岁时，宋宝罗在[[河南|河南]][[郑州|郑州]]连演数月而失声，再次无法登台<ref name=gongren/>。

这时，宋宝罗家刚从北京[[施家胡同|施家胡同]]迁至北京[[延寿街_(北京)|延寿街]]三眼井胡同21号。该地区有戏曲演员[[杨小楼|杨小楼]]、[[张君秋|张君秋]]、[[王玉蓉|王玉蓉]]、[[李洪春|李洪春]]、[[吴素秋|吴素秋]]等人居住，且距离北京[[琉璃厂|琉璃厂]]很近，有[[朵云轩|朵云轩]]、[[荣宝斋|荣宝斋]]等书画古玩店。书画家[[马谌汀|马谌汀]]租住在他家的南屋，因喜欢他而教他学画画。在马谌汀的画室，宋宝罗还结识了[[齐白石|齐白石]]、[[李苦禅|李苦禅]]、[[王青芳|王青芳]]、[[陈半丁|陈半丁]]等书画家。经马谌汀介绍，宋宝罗后来又拜[[于非闇|于非闇]]为师，学习工笔花鸟画。受于非闇启发，宋宝罗又学习[[篆刻|篆刻]]<ref name=gongren/>。

[[北京中山公园|北京中山公园]]里有个水榭林园，是书画社——[[湖社|湖社]]的活动场所。湖社每日举办笔会，并常办画展。宋宝罗常来水榭林园，因此结识[[徐悲鸿|徐悲鸿]]、[[张大千|张大千]]、[[徐燕荪|徐燕荪]]等书画家。1934年春某天，几位画家合作一幅画，徐悲鸿画了数只麻雀，题款后发现自己忘带印章，宋宝罗利用水榭里的石头和刻刀现场刻了方“悲鸿”印章，徐悲鸿用这枚印画。此后徐悲鸿不少画用的就是这方印章<ref name=gongren/>。

宋宝罗曾为不少名人刻过印章，如[[张大千|张大千]]、[[于右任|于右任]]、[[孔祥熙|孔祥熙]]以及[[周信芳|周信芳]]等人，这些印章被他收入印谱本，此谱后请[[张伯苓|张伯苓]]题跋，[[徐世昌|徐世昌]]题“铁划银钩”，[[金锡侯|金锡侯]]题“直追秦汉”。他每月能刻二、三十方印章，月收入一、二百元。有5年时间他都没唱戏<ref name=gongren/>。

1937年[[七七事变|七七事变]]发生后，[[侵华日军|侵华日军]]占领[[北平|北平]]、[[天津|天津]]及[[华北|华北]]大部分地区。日本控制下的华北当局为安定人心，大搞娱乐活动。见[[荀慧生|荀慧生]]、[[马连良|马连良]]、[[谭富英|谭富英]]、[[金少山|金少山]]等名角在[[天津|天津]][[中国大戏院|中国大戏院]]演出，宋宝罗又重燃演戏的念头。5年没演戏，他的嗓音竟比之前更好，[[刘鸿升|刘（鸿升）]]派、[[汪桂芬|汪（桂芬）]]派的戏都能唱<ref name=gongren/>。

不久，[[程砚秋|程砚秋]]率团来天津公演，演出《阳平关》、《[[锁麟囊|锁麟囊]]》，戏票迅速卖光。但演出前一天，有位二牌老生演员突发高烧无法来天津参演，程砚秋心急如焚。宋宝罗在画界的忘年交[[姜妙香|姜妙香]]向程砚秋推荐宋宝罗演《阳平关》中的[[黄忠|黄忠]]一角。宋宝罗因此与程砚秋同台演出。后来他又和[[郑冰如|郑冰如]]、[[宋玉茹|宋玉茹]]合作演出。后来，宋宝罗到[[中国东北|东北]]、[[上海|上海]]、[[江苏|江苏]]、[[浙江|浙江]]等地巡演<ref name=gongren/>。

1949年4月，[[上海战役|上海战役]]结束后，宋宝罗在上海[[复兴公园|复兴公园]]办了场[[扇面|扇面]]义卖画展，义卖所得全部捐给[[上海市人民政府|上海市人民政府]]慰问[[中国人民解放军|中国人民解放军]]。随后他又参与[[梅兰芳|梅兰芳]]、[[周信芳|周信芳]]等人举办的慰问中国人民解放军义演活动。1953年到1957年[[反右运动|反右运动]]前，宋宝罗分别在上海、江苏、[[江西|江西]]等地演出。1957年[[反右运动|反右运动]]中，一批戏曲名家被打成“右派分子”，宋宝罗虽未被划为“右派分子”，但也受到冲击。1958年，杭州京剧团邀宋宝罗加盟，此后他常在浙江城乡演戏，[[浙江省|浙江省]]及[[杭州市|杭州市]]领导多次看他演出。1958年至1964年，他连续被评为杭州市先进工作者<ref name=gongren/>。

1958年9月某晚，宋宝罗在杭州[[新中国剧院|新中国剧院]]主演《碰碑》等剧，演出结束后，前台看门的工作人员告诉他，[[周恩来|周恩来]]刚才站在后面看戏。数日后某晚，[[浙江省公安厅|浙江省公安厅]]警卫处人员用轿车将他接到[[西泠饭店|西泠饭店]]，见到了周恩来。他为周恩来唱了《碰碑》中反[[二黄|二黄]]唱段，周恩来鼓掌说“好极了，好极了！”不久，周恩来将此事向[[毛泽东|毛泽东]]进行了介绍。此后宋宝罗多次为毛泽东献艺<ref name=gongren/>。

1958年秋某晚，浙江省公安厅警卫处的轿车将宋宝罗从[[东坡剧院|东坡剧院]]接到[[杭州饭店|杭州饭店]]，走进三楼大厅他遇见周恩来和[[中共浙江省委|中共浙江省委]]书记[[江华_(政治人物)|江华]]，再往里走便见到毛泽东、[[刘少奇|刘少奇]]、[[王光美|王光美]]、[[陈毅|陈毅]]和[[十世班禅|十世班禅]]，当时毛泽东、刘少奇、周恩来等中央领导是陪[[金日成|金日成]]率领的[[朝鲜劳动党|朝鲜劳动党]]代表团来杭州访问。宋宝罗为毛泽东等中央首长和朝鲜贵宾唱戏，演出结束后，毛泽东、刘少奇、周恩来、金日成、陈毅、[[崔庸健|崔庸健]]、班禅等中朝领导人与宋宝罗等演职员合影<ref name=gongren/>。

1962年12月26日，毛泽东七十寿辰。宋宝罗应邀为毛泽东演出《[[朱耷|朱耷]]卖画》，边唱戏边画画。当宋宝罗边唱二黄原板边画时，毛泽东从沙发上站起来，轻轻走到宋宝罗身旁。宋宝罗回忆道，“我正在握着大笔画鸡身的时候，无意中碰到了毛主席的身体，才知道主席背着手在仔细看我的画稿。我激动极了，原来要唱6句或者8句才能将公鸡画好，这次有如神助一般，唱了4句便画好了。”毛泽东说：“戏唱得好，画也不错，用笔很准。”宋宝罗用[[小楷|小楷]]笔题款：“敬献给毛主席”。毛泽东高兴地说：“你可以写上一句‘一唱雄鸡天下白’嘛！”<ref name=gongren/>

1958年春至1963年春，宋宝罗先后40余次为毛泽东、刘少奇、周恩来、[[朱德|朱德]]、[[陈云|陈云]]、[[叶剑英|叶剑英]]、[[李先念|李先念]]、陈毅等中央领导演唱<ref name=gongren/>。

1966年“[[文化大革命|文化大革命]]”爆发后，杭州京剧团被[[造反派|造反派]]称作“浙江省最大的放毒剧团”，宋宝罗被称作“最大的反动权威和[[牛鬼蛇神|牛鬼蛇神]]”。1967年春夏之交，造反派在[[南京图书馆|南京图书馆]]发现一张1945年10月的报纸，头版头条是[[蒋介石|蒋介石]]在舞台上与宋宝罗握手的照片，旁边有[[宋美龄|宋美龄]]、[[何应钦|何应钦]]、[[陈诚|陈诚]]及[[美国|美国]][[乔治·卡特利特·马歇尔|马歇尔]]将军，那是庆祝抗日战争胜利的一场演出。因为该“罪证”，宋宝罗被造反派打成“历史反革命分子”，“永世不得翻身”，妻子被迫和宋宝罗离婚，宋宝罗差点自杀<ref name=gongren/>。

1970年冬，毛泽东到杭州视察，在一次晚会上，毛泽东问[[浙江省革命委员会|浙江省革命委员会]]第一副主任[[陈励耘|陈励耘]]：“那位会唱京戏、会画大公鸡的宋宝罗怎么没有来？”陈励耘答：“他的问题还没搞清楚。”毛泽东问：“他有什么问题？”陈励耘答：“听说他给蒋介石唱过戏。”毛泽东生气地说：“演戏是演员的工作，蒋介石叫他唱戏他敢不唱？解放后我和总理多次请他来为我们唱戏，他不是一叫就来嘛。你们真是乱弹琴！”此后造反派不敢再像以前那样迫害宋宝罗，但仍未给他平反<ref name=gongren/>。

1978年冬，[[中共中央副主席|中共中央副主席]][[叶剑英|叶剑英]]来浙江视察，在一次晚会上对在场的浙江省、杭州市及有关部门的领导说：“宋宝罗在‘文革’中被造反派打成所谓的‘反革命’，这是错误的。他是我们的好演员、好同志，我在这里为他平反了！”叶剑英的话引起了掌声，宋宝罗当即流泪。当天是叶剑英八十寿辰，宋宝罗演唱了叶剑英的诗作《八十抒怀》，当他唱到“满目青山夕照明”时，全场掌声雷动，叶剑英也起立鼓掌<ref name=gongren/>。

宋宝罗退休后，仍时常登台唱戏，并继续绘画、篆刻。他喜爱画大公鸡，他刻的“百寿印”将100个寿字错落排列。他不仅读书、看报，还学会了上[[互联网|互联网]]，开了[[博客|博客]]。他曾被评为“中国百佳健康老人”、“浙江省十佳健康老人”、“杭州市十佳健康老人”<ref name=gongren/>。

2017年9月2日，宋宝罗在杭州家中无疾而终，享年101岁<ref name=shishi>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1782933 |title = 101岁京剧表演艺术家宋宝罗去世，曾与梅兰芳程砚秋同台 |author =  |date = 2017-09-02  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

==家庭==
*父：[[宋永珍|宋永珍]]，艺名“毛毛旦”，梆子旦角<ref name=wy/>。
*母：[[宋凤云|宋凤云]]，原姓杨，早年工旦角，艺名“金翠凤”，嗓败之后改演丑角，更名“宋凤云”，成为京剧第一位女丑演员。宋永珍、宋凤云夫妇育有四子二女，均在戏曲界工作，其中以次子宋遇春、四子宋保罗最知名<ref name=wy/>。
