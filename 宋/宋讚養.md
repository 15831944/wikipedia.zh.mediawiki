'''宋讚養'''（{{lang-ko|송찬양}}，{{bd|1990年|3月24日}}），來自[[韓國|韓國]][[首爾|首爾]]，音樂方面具有相當造詣，擅長樂器演奏，擁有13種以上的樂器演奏技能。

== 個人生活 ==
大學期間創立了街頭運動社團“Barcrobatic Soulz”<ref>{{Cite web |url = https://www.facebook.com/barsoulz/|title =Barcrobatic Soulz Fan Page|language=ko}}</ref>

== 經歷 ==
2011年～2013年在首爾的「首都防衛司令部」以憲兵的身份服完兵役。

2016年，考取台中街頭藝人證照(有效期限:107/07/31)，為全台首位韓國街頭藝人。<ref>{{Cite web |url =http://www.buskers-taichung.com.tw/info_detail.php?id=2223|title =臺中市街頭藝人 官方網站|language=zh-TW}}</ref>

2016年8月20日，於台中一中街進行街頭表演。

2017年6月，加盟並於[[台中|台中]]逢甲開設全[[臺灣|臺灣]]第一家“碰咕食米漢堡店”。

2017年7月19日，台中逢甲心齋橋餐廳發生氣爆意外，宋讚養提供店面讓傷者休息，且主動提供餐點給救災人員。<ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/2135871|title=逢甲商圈大火開放店內供急救 韓國帥哥足感心|newspaper=自由時報|accessdate=2017-07-18|language=zh-TW}}</ref><ref>{{Cite news|url=http://news.ebc.net.tw/news.php?nid=71327&fb_comment_id=1254766974646272_1255498337906469&noredirect=1|title=滿身鮮血求救！韓籍帥哥庇護氣爆傷者被讚「暖男」|newspaper=東森新聞|accessdate=2017-07-19|language=zh-TW}}</ref><ref>{{Cite news|url=https://udn.com/news/story/11326/2591785|title=逢甲氣爆傷者滿身是血 韓籍暖男開店慰問警消|newspaper=聯合新聞網|accessdate=2017-07-19|language=zh-TW}}</ref>

2017年7月23日，宋讚養製作歌曲追悼氣爆亡者，並決定北上生活，將米漢堡店交回韓國總公司。<ref>{{Cite news|url=http://www.setn.com/News.aspx?NewsID=275846|title=氣爆暖心關店供餐…韓店長不捨告別逢甲：對不起我不會在了|newspaper=三立新聞網|accessdate=2017-07-23|language=zh-TW}}</ref>

2017年8月19日，於[[台北|台北]][[西門町|西門町]]熱心幫助陌生老人，被路過民眾認出，善行曝光上網，引起新聞討論。<ref>{{Cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20170820/1186518/|title=好心韓籍老闆又做善心　幫拾荒奶奶返家|newspaper=蘋果日報|accessdate=2017-08-20|language=zh-TW}}</ref>

2017年10月，因YouTube個人頻道“韓國讚養綜合台”發佈之「韓國男人好羨慕台灣的戀愛方式」<ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/2211932|title=戀愛觀差很大 韓國歐巴吐苦水：台灣這幾件事讓我很羨慕！|newspaper=自由時報|accessdate=2017-10-03|language=zh-TW}}</ref>及「韓國人第一次去吃丹丹漢堡時的反應」<ref>{{Cite news|url=https://www.ettoday.net/news/20171016/1032030.htm?from=fb_et_news|title=小當家美味！第一次吃丹丹漢堡　歐爸激推給南韓人：簡直開外掛|newspaper=東森新聞雲|accessdate=2017-10-16|language=zh-TW}}</ref>而獲得關注。

2017年12月14日，開始東森星光雲「韓星爆爆」談論韓國娛樂消息的網路直播節目「韓國阿拉唷」的代班主持工作。

== 演藝經歷 ==
大學畢業後決定來到台灣，住在台中逢甲，以多種管樂表演，通過2016台中[[街頭藝人|街頭藝人]]徵選，並獲得街頭藝人證。
