'''宋琨'''（{{bd|1977年}}），[[内蒙古|内蒙古]]人，是当代[[中国|中国]]的[[画家|画家]]、艺术家。工作生活于北京，职业艺术家，被誉为“最令人瞩目的70后女性艺术家”。<ref>{{cite web|url=http://art.ifeng.com/2017/0503/3313981.shtml|title=中国嘉德2017春拍:二十世纪及当代艺术专场|publisher=凤凰网| language =zh}}</ref><ref>{{cite web|url=http://sh.eastday.com/m/20130727/u1a7549651.html|title="当代艺术与当代城市"展览在沪开幕|publisher=东方网| language =zh}}</ref><ref>{{cite web|url=http://cul.qq.com/a/20131227/007011.htm|title=宋琨：艺术是一种暗号 藏在真实里|publisher=腾讯网| language =zh}}</ref><ref>{{cite web|url=http://art.people.com.cn/n/2015/0716/c206244-27312788.html|title=青年艺术家项目还能怎么玩|publisher=人民网| language =zh}}</ref><ref>{{cite web|url=https://www.lw5u.com/zz/hugangjingji/news/itemid-1243206.html|title=中国当代艺术新生代|publisher=沪港经济杂志| language =zh}}</ref>

== 教育经历 ==
1996 毕业于中央美术学院附属中学

2002 毕业于中央美术学院油画系第三工作室, 学士

2006 毕业于中央美术学院油画系第三工作室，硕士，师从刘小东，谢东明
