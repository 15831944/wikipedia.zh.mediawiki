'''宋雨琦'''（{{kor|k=''' 송우기'''|rm='''Song Ugi'''}}，{{lang-en|'''Song Yuqi'''}}，{{bd|1999年|9月23日||}})，出生于[[中国|中国]][[北京市|北京市]]，在[[中国|中]][[韩国|韩]]两地活动的中国女歌手。现为[[Cube娱乐|Cube娱乐]]旗下女子组合'''[[(G)I-DLE|(G)I-DLE]]'''成员，担任领舞<ref>{{cite web |title=(G)I-DLE 5th Week of May The Show (5/29/2018) [CC: ENG SUBS] |url=https://www.youtube.com/watch?v=tFVtqwtpjcI |website=Youtube [6]K-POP WORLD-WIDE CH.6 |date=2018-05-30}}</ref>和副唱。

== 演艺生涯 ==
2015年，参加Cube Star World Audition北京站<ref>{{cite web |title=(여자)아이들((G)-I-DLE) - 나의 이름은。우기(YUQI) |url=https://www.youtube.com/watch?v=MkkbLKWx_GY |website=Youtube  United CUBE |date=2018-04-26}}</ref>，表演是跳[[CLC|CLC]]《变得漂亮》舞蹈、唱[[鄧紫棋|鄧紫棋]]《泡沫》<ref>{{cite web |title=爱奇艺独家专访(G)I-DLE |url=https://www.iqiyi.com/v_19rqydjhvs.html |website=爱奇艺 |date=2018-07-31}}</ref>。

2016年7月20日，进入Cube娱乐<ref>180816[[欢乐在一起|Happy Together 3]]节目中透露在拍摄前一天刚好在韩国两年整. KBS. August 16,2018.{{ko icon}} </ref>。

2017年6月，参加Rising Star化妆品广告拍摄<ref> {{cite web|title=비포앤애프터바이오 라이징스타 워터뱅뱅앰플미스트 걸그룹연습생 화장품광고 영상 (Jiwoo, Shuwha, Woogi, Minnie |url=https://www.youtube.com/watch?v=NuJy4SGOd_w |website=Youtube BEFORE & AFTER BIO |date=2017-06-01}}</ref>。8月23日，作为Cube娱乐练习生首次公开<ref>{{cite web |title=#SYQ |url=https://www.instagram.com/p/BYIuKwwARtm/?taken-by=addxcubetree |website=Instagram addxcubetree |date=2017-08-23}}</ref>。

2018年2月，参与综艺节目《世上的温度》<ref>{{cite web |title=아이돌 연습생이 노래를 들어 달라고 한다면? 반전 목소리네 꼭 데뷔했으면 좋겠다 세상의 온도 | |url=https://www.youtube.com/watch?v=MbHVHQstS9w |website=Youtube dingo story |date=2018-02-27}}</ref>。4月，参与综艺节目《首次亮相之前的女子组合》<ref>{{cite web |title=걸그룹 데뷔를 앞둔 한 소녀에게 어마무시한 일이 일어났다?! 데뷔 전부터 살벌한(?) 신인 걸그룹 '아이들' ! |url=https://www.youtube.com/watch?v=FoiDU23wjO4 |website=Youtube dingo |date=2018-04-26}}</ref>和进行舞蹈翻跳活动<ref>{{cite web |title=(이거 대박!!!) 홍대 뒤집어진 신인 걸그룹 댄스 실화?! 씨엘부터 방탄까지 [댄스버스킹] (여자)아이들 Dance Busking - (G)I-DLE (@Hongdae) |url=https://www.youtube.com/watch?v=WUK4X97Fzzc |website=Youtube dingo music |date=2018-04-15}}</ref><ref>{{cite web |title=2018 아는만큼 보이는 아이돌 댄스! 이거 다 맞추면 춤잘알 인정 (여자)아이들 (G)I-DLE DANCE MEDLEY |url=https://www.youtube.com/watch?v=YkrS9kfFO_s |website=Youtube dingo music |date=2018-04-20}}</ref><ref>{{cite web |title=이렇게 아름다운 춤이?! 글로벌한 신인 걸그룹의 고전미 세련미 쩌는 매력발산 영상!
