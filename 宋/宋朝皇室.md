==家族史==
'''宋宣祖'''─'''武昭皇帝'''[[趙弘殷|趙弘殷]]，涿郡人（今[[河北省|河北省]][[保定|保定]][[涿州市|涿州市]]），後遷居[[洛陽|洛陽]]，為是宋開國君主[[宋太祖|宋太祖]][[趙匡胤|趙匡胤]]、與繼位者[[宋太宗|宋太宗]][[趙光義|趙光義]]的父親。官拜[[後漢|後漢]]護聖軍都指揮使，據聞「少時驍勇異常，擅長騎馬、弓箭」。<ref>中華頌 -> 百家姓 -> 名姓望族，{{cite web |url=http://big5.ccnt.com.cn/china/surname/tribe/cw-zhao-2.htm |title=存档副本 |accessdate=2011-03-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20110518231307/http://big5.ccnt.com.cn/china/surname/tribe/cw-zhao-2.htm |archivedate=2011-05-18 }} </ref> 後來因其次子[[趙匡胤|趙匡胤]]投靠[[郭威|郭威]][[幕府|幕府]]，[[趙弘殷|趙弘殷]]被派護駕出征，在兒子勸說之下倒戈，又在[[郭威|郭威]]和[[周世宗|周世宗]][[柴榮|柴榮]]手下效力，先後任鐵騎第一軍都指揮使助周太祖[[郭威|郭威]]起兵推翻[[後漢|後漢]]，建立[[後周|後周]]。後又出任檢校司徒，統率禁兵，封[[天水|天水]][[縣男|縣男]]。<ref>中華頌 -> 百家姓 -> 名姓望族，{{cite web |url=http://big5.ccnt.com.cn/china/surname/tribe/cw-zhao-2.htm |title=存档副本 |accessdate=2011-03-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20110518231307/http://big5.ccnt.com.cn/china/surname/tribe/cw-zhao-2.htm |archivedate=2011-05-18 }} </ref>

[[趙弘殷|趙弘殷]]之次子[[趙匡胤|趙匡胤]]受[[周世宗|周世宗]][[柴榮|柴榮]]臨終託孤，在[[周世宗|周世宗]][[柴榮|柴榮]][[駕崩|駕崩]]之後，輔佐幼主[[後周恭帝|後周恭帝]][[柴宗訓|柴宗訓]]。後受部下擁立，在[[陳橋兵變|陳橋兵變]]中[[黃袍加身|黃袍加身]]，[[稱帝|稱帝]]，是為[[宋太祖|宋太祖]]。建[[宋|宋]]。柴氏後人並未遭不測，而是獲得宋室厚待。[[趙匡胤|趙匡胤]]南征北討，先後滅[[后蜀|后蜀]]、[[南汉|南汉]]和[[南唐|南唐]]等國，再次建立了相對統一[[中原|中原]]王朝。

根據[[宋太宗|宋太宗]]的說法，他表示母親[[昭憲太后|昭憲太后]]杜氏認為後周滅亡皆因幼主臨朝，所以留下[[兄終弟及|兄終弟及]]的遺言，要求[[趙匡胤|趙匡胤]]立下[[金櫃之盟|金櫃之盟]]，死後將[[帝位|帝位]]傳給光義，宋太祖被迫接受。歷史學家鄧廣銘、張蔭麟等論証「金櫃之盟為虛構」，影響至今，成為最有影響力的說法。

[[開寶|開寶]]九年（976年），[[宋太祖|宋太祖]]在北征[[契丹|契丹]]的途中，與其弟[[趙光義|趙光義]]飲酒，共宿宮中；翌晨，太祖猝崩，得年50岁。據[[湘山野錄|湘山野錄]]記載，有「[[燭影斧聲|燭影斧聲]]」的說法，普遍猜測[[宋太祖|宋太祖]][[趙匡胤|趙匡胤]]是被意圖[[篡位|篡位]]的[[宋太宗|宋太宗]][[趙光義|趙光義]]所弒。[[宋太祖|宋太祖]][[趙匡胤|趙匡胤]]之次子[[趙德昭|趙德昭]]後于[[太平兴国|太平兴国]]四年（979年）被迫[[自殺|自殺]]而死，[[宋太祖|宋太祖]]四子[[趙德芳|趙德芳]]则在[[太平兴国|太平兴国]]六年（981年）因病英年早逝。朝野皆認為死因與[[宋太宗|宋太宗]][[趙光義|趙光義]]有關。

[[宋朝|宋朝]]文強武弱，因失去[[燕云十六州|燕云十六州]]作為屏障，被迫與[[遼朝|遼朝]]/[[契丹|契丹]]及[[西夏|西夏]][[黨項|黨項]]盟約納[[歲幣|歲幣]]，维持百餘年和平。

[[靖康|靖康]]元年（1126年），金兵滅[[遼朝|遼朝]]後南下，[[北宋|北宋]][[靖康|靖康]]二年（金朝天會五年）二月初六 (1127年3月20日)，[[北宋|北宋]]滅亡。趙宋皇室與文武官貴族遭受滅頂之災，宋徽宗，宋欽宗，一眾皇室宗親，及官員眷屬被迫「北狩」─ 被俘虜至金國。史稱[[靖康之禍|靖康之禍]]。「北狩」途中，已有皇子皇女或夭折，或死於顛沛折磨。到達金國後，趙宋皇室依舊備受凌辱，皇子為奴，皇女入洗衣院(金人軍妓營)為娼婢，供金人奴役或淫樂，<ref>宋徽宗：史上子女最多皇帝， 2009年12月22日，【僑報】http://dailynews.sina.com/bg/chn/chnlocal/chinapress/20091222/0220983034.html </ref>更有被賤賣到蒙古，朝鮮等地。趙宋皇室後又被金人遷至[[五國城|五國城]]（今[[黑龍江省|黑龍江省]][[依蘭縣|依蘭縣]]城北舊古城）。[[宋徽宗|宋徽宗]]與[[宋欽宗|宋欽宗]]後駕崩於[[五國城|五國城]]。

與此同時，[[宋高宗|宋高宗]][[趙構|趙構]]倖免於難，並在河南[[商丘|商丘]]登基，[[南宋|南宋]]開始。高宗畏懼[[金朝|金朝]]及担心一旦北伐成功導致[[宋欽宗|宋欽宗]]回国爭位，因此在1141年稱臣求和，達成[[紹興和議|紹興和議]]。由於宋高宗唯一之子早夭，南渡後不育，和群臣商議後，宋高宗復立[[趙德芳|趙德芳]]之六世孫[[趙昚|趙昚]]為儲君，史稱[[宋孝宗|宋孝宗]]。於是宋室天下在186年又回[[宋太祖|宋太祖]]一脈。[[趙德芳|趙德芳]]七世孫為[[宋光宗|宋光宗]]，八世孫為[[宋寧宗|宋寧宗]]。[[趙德昭|趙德昭]]之九世孫亦為皇帝，稱[[宋理宗|宋理宗]]，十世孫為[[宋度宗|宋度宗]]，十一世孫為宋末三幼主。

[[南宋|南宋]]與[[蒙古帝國|蒙古帝國]]後來聯兵滅[[金朝|金朝]]，但又因唇亡齒寒的緣故，失去[[金朝|金朝]]作為其與蒙古之間的屏障，後於45年後[[崖山海戰|崖山海戰]]中亡於[[蒙元|蒙元]]。陸秀夫背著8歲的[[趙昺|趙昺]]跳海自殺。不少後宮和大臣亦相繼跳海自殺。[[宋史|宋史]]記載七日後，十餘萬具屍體浮海。[[楊淑妃_(宋度宗)|楊太后]]在聽聞宋帝昺的死訊在後亦赴海自殺，[[張世傑|張世傑]]將其葬在海邊。不久張世傑亦舉家自殺。[[文天祥|文天祥]]早前已在海豐被俘，後在元軍船艦上目睹宋軍大敗，曾作詩悼念。四年後仍拒降蒙元，從容就義。自此趙宋宗室自319年後在中國失去政權。

==分封制度==
宋承唐制，'''宗王'''襁褓即裂土而爵之。然名存實亡，無補於事。降至疏屬，宗正有籍，玉牒有名，宗學有教，郊祀、明堂，遇國慶典，皆有祿秩。所寓州縣，月有廩餼。至於'''宗女'''適人，亦有恩數。然國祚既長，世代浸遠，恆產豐約，去士庶之家無甚相遠者。[[靖康之亂|靖康之亂]]，諸王駢首以弊於金人之虐，論者咎其無封建之實，故不獲維城之助焉。<ref>【宋史‧列传第三‧宗室一】</ref>

===皇室男性===
[[宋太祖|宋太祖]]分封其父親五子為王，王長子封[[國公|國公]]，拜[[節度使|節度使]]，後襲[[郡王|郡王]]名號。<ref>【宋史‧列传第三‧宗室一】</ref>

[[熙寧|熙寧]]二年，詔宣祖、太祖、太宗之子，皆擇其後一人為宗，世世封公，以奉其祀，不以服屬盡故殺其恩禮。<ref>【宋史‧列傳第三‧宗室一】</ref>

[[熙寧|熙寧]]三年，太常禮院言：「本朝近制，諸王之後，皆用本宮最長一人封公繼襲。去年詔祖宗之子皆擇其後一人為宗，世世封公，即與舊制有異。按禮文，諸王、公、侯、伯、子、男，皆子孫承嫡者傳襲。若無嫡子及有罪疾，立嫡孫；無嫡孫，以次立嫡子同母弟；無母弟，立庶子；無庶子，立嫡孫同母弟；無同母弟，立庶孫。曾孫以下準此。合依禮令，傳嫡承襲。」詔可。<ref>【宋史‧列传第三‧宗室一】</ref>

===皇室女性===
宋朝公主為國公主。 宋制，“皇姑母”曰[[大長公主|大長公主]]，“皇姊妹”曰[[長公主|長公主]]，“皇女”曰[[公主|公主]]，“王女”曰[[郡主|郡主]]。“王孫女”曰[[縣主|縣主]]。<ref>《宋史‧列傳第七‧公主》</ref>

[[政和|政和]]三年（1113年）因[[蔡京|蔡京]]上奏，宋廷應仿照[[周朝|周朝]]“[[王姬|王姬]]”位號，改帝女“[[公主|公主]]”為“帝姬”。《宋史·禮志十八》載：「徽宗改公主為姬，下詔曰：『在熙寧初，有詔釐改公主、郡主、縣主名称，當時群臣不克奉承。近命有司稽考前世，周稱「王姬」，見於《詩》、《雅》。「王姬」雖周姓，考古立制，宜莫如周。可改“公主”為“帝姬”、“郡主”為“宗姬”、“縣主”為“族姬”。其称“大長”者，為“大長帝姬”，仍以美名二字易其國號，内兩國者以四字。』」“帝姑祖母”亦封“大長帝姬”，為與“帝姑母”有所區別，雖仍封“大長帝姬”，但改以四字美名為封號。<ref>《宋史·禮志十八》</ref> 而時人認為帝姬此號與「帝飢」同音，引為不吉。後宋室因[[靖康之難|靖康之難]]南遷，於[[建炎|建炎]]元年（1127年）復舊制。

==現今分布==
太宗派與魏王派在南宋灭亡時，隨朝廷南下到[[潮汕|潮汕]]與[[珠江三角洲|珠江三角洲]]一带，但绝大多数到了[[崖山海战|崖山海战]]战场所在地的[[廣東|廣東]][[四邑|四邑]]一帶定居，如[[台山|台山]][[斗山镇|斗山镇]]浮石赵氏，[[新会|新会]][[古井镇|古井镇]][[霞路|霞路]][[赵氏|赵氏]]，[[新会|新会]][[三江趙氏|三江趙氏]]，[[江門|江門]][[滘頭|滘頭]][[趙氏|趙氏]]及[[珠海|珠海]][[斗門|斗門]][[崑山趙氏|崑山趙氏]]。
