'''宋碩芸'''（{{lang|en|'''SUNG Shuo Yun'''}}，{{bd|1997年|6月15日|||catIdx=Sung宋}}），為[[台灣|台灣]]女子[[羽球|羽球]]運動員，目前就讀[[臺北市立大學|臺北市立大學]]球類系。

== 簡歷 ==
2011年8月，代表[[台北市|台北市]]參加在[[蘇格蘭|蘇格蘭]]舉行的[[國際少年運動會|國際少年運動會]]，獲得女子單打金牌、女子雙打和團體賽銀牌<ref>{{cite news |language = zh-tw |author = 黃秀仁 |coauthors =  |url = https://tw.news.yahoo.com/%E7%BE%BD%E7%90%83-%E5%8C%97%E5%B8%82%E5%A4%A7%E5%90%8C%E5%9C%8B%E7%94%B7-%E5%A5%B3%E6%8C%91%E6%88%B0%E7%B4%80%E9%8C%84-%E7%AC%AC-%E5%96%AE%E6%89%93%E5%91%82%E5%AE%B6%E5%BC%98-%E5%AE%8B%E7%A2%A9%E8%8A%B8%E9%A0%98%E8%BB%8D-113014285.html |title = 羽球》北市大同國男、女挑戰紀錄　第一單打呂家弘、宋碩芸領軍 |publisher = 麗台運動報 |pages =  |date = 2011-12-01 |accessdate = 2016-07-10}}</ref><ref>{{Cite web |url = http://www.openreport.taipei.gov.tw/OpenFront/report/report_detail.jsp?sysId=C100AW349 |title = 臺北市參加2011年蘇格蘭拉納克郡國際少年運動會成果報告 |author =  |date = 2011-11-08 |publisher = 臺北市政府教育局  |language = zh-tw |accessdate = 2016-07-10 |quote =報告內容摘要：...三、羽球隊由大同高中林峻永教練帶隊...（六）宋碩芸榮獲羽球女子雙打銀牌。...（九）宋碩芸榮獲羽球混合雙打金牌。...（十五）宋碩芸榮獲羽球團體賽銀牌。}}</ref>。

2014年5月，代表[[台北市立大同高中|台北市立大同高中]]出戰[[世界中學生羽毛球錦標賽|世界中學生羽毛球錦標賽]]，並幫助該校獲得3面金牌和3面銀牌<ref>{{cite news |language = zh-tw |author = 黃秀仁 |coauthors =  |url = http://www.ltsports.com.tw/other/complex/104440-2014-05-11-10-20-00 |title = 世界中學生羽球錦標賽》囊括四冠 　中華隊金圓滿 |publisher = 麗台運動報 |pages =  |date = 2015-09-13 |accessdate = 2016-07-10}}</ref>。

2015年9月，參加[[2015年奧克蘭羽毛球國際賽|奧克蘭羽球國際賽]]女子單打項目，在決賽以0比2（12-21、14-21）敗給同隊的[[李佳馨|李佳馨]]，屈居亞軍<ref>{{cite news |language = zh-tw |author = 黃秀仁 |coauthors =  |url = http://www.ltsports.com.tw/other/complex/113104-3-5-5 |title = 羽球》台灣潛優選手奧克蘭國際系列賽一舉囊括3冠5亞5季 |publisher = 麗台運動報 |pages =  |date = 2014-05-11 |accessdate = 2016-07-10}}</ref>。

2017年7月，宋碩芸在[[中華民國106年度第2次全國羽球排名賽|全國羽球排名賽]]女子單打決賽以2比0（21-19、21-12）擊敗隊友[[李佳馨|李佳馨]]，生涯首度贏得全國賽冠軍<ref>{{cite news|title=羽球排名賽／吞黃牌爆氣 宋碩芸女單首封后|url=https://udn.com/news/story/7005/2591046|agency=聯合報|date=2017-07-18}}</ref>。
