'''宋代汴洛语音'''即[[宋代|宋代]][[汴京|汴京]]及[[洛阳|洛阳]]一带的[[汉语方言|方言]][[音系|音系]]。

== 研究材料 ==
{{seealso|皇极经世声音唱和图}}
[[北宋|北宋]][[邵雍|邵雍]]《[[皇极经世声音唱和图|皇极经世声音唱和图]]》中的音系，和《[[切韵|切韵]]》、《[[广韵|广韵]]》所反映的[[切韵音系|音系]]不同，却能和北宋汴洛人士邵雍、[[程颢|程颢]]、[[程颐|程颐]]等人的诗文用韵两相吻合，可以据此来构拟宋初汴洛语音。

两人构拟的主要区别在于：
#《声音唱和图》中，同一部位的塞音、塞擦音分为四套，如“卜、步、普、旁”。周祖谟认为对应的音值是p、p、pʰ、pʰ，只是邵雍墨守旧说，把原来属“並”母的“步”、“旁”仍列维浊音<ref name=ZZM/>{{rp|591}}。李荣则认为对应的音值是p、b、pʰ、bʱ，其中b和bʱ的区别不辨义，是两个[[音位变体|音位变体]]<ref name=LR/>{{rp|170}}。
#同理，原图中同一部位的擦音分为清浊两套，周祖谟认为已经并为一套，而李荣分列两套。
#原图中有独立的舌上音。周祖谟的构拟中，舌上音已经和正齿音合并。
#原图中，次浊声母（鼻音和边音）分两套，一套配上声，一套配平、去、入三声。周祖谟的构拟中，轻唇部位分为相配的ɱ和v，牙喉部位分为相配的ŋ和ɣ，其他部位只有一套。李荣的构拟中，次浊声母只有一套，但有两个音位变体，平去入三声字带浊流，上声字[[声门|声门]]紧缩，不带浊流<ref name=LR/>{{rp|171}}。

== 特点 ==
宋代汴洛语音与之前的[[切韵音|《切韵》音系]]和之后的《[[中原音韵|中原音韵]]》音系相比，擁有如下特点：
# 全浊声母按照声调分为两组，分别接近同部位的全清和次清声母，可能已经清化。
# 上声次浊声母归入阴类，类似后代的“[[浊上归去|浊上归去]]”。
# 歌、麻两韵合流。
# 切韵音中的一二等韵合流，三四等韵合流。出现了类似后代[[四呼|四呼]]的格局。
# 韵母i和ts、tʃ等相拼时变为ɿ、ʅ，从细音三等变为洪音一等。
# 切韵音中，[[入声韵|入声韵]]字总是和[[阳声韵|阳声韵]]字相配。宋代汴洛语音中的入声字，部分和[[阴声韵|阴声韵]]字相配，此时-t尾和-k尾的入声字可能已经合流变为-ʔ尾。
