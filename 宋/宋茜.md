'''宋茜'''（{{lang-pinyin|'''Song Qian'''}}；藝名：'''Victoria'''；{{bd|1987年|2月2日|catIdx=Victoria茜}}），出生於[[山东省|山东省]][[青岛|青岛]]市，[[中国|中国]]及[[大韓民國|大韓民國]]女演员、歌手，亚洲多栖发展女艺人。2009年以[[南韓|南韓]]女子組合[[F(x)_(組合)|f(x)]]出道，在隊內擔任隊長、主領舞及副唱。2015年正式在中國成立宋茜工作室。<ref>{{cite web | url=http://ent.sina.com.cn/v/m/2015-06-27/doc-ifxenncn6752213.shtml | title=独家：宋茜工作室合作贾士凯 将主攻影视 | publisher=新浪娱乐 | language=zh}}</ref>

== 早年生活与教育程度 ==
2005年，宋茜考入[[北京舞蹈学院|北京舞蹈学院]]中国民族民间舞系，学习藏族、蒙古族、维吾尔族等各民族风格舞蹈，期间多次随北京舞蹈学院进行国内外交流，登台表演。2006年8月，在第八届桃李杯舞蹈比赛中以作品《红珊瑚（山东秧歌）》获得群舞组一等奖。

2007年9月，在北京市舞蹈大赛上以现代舞《画聆》获得一等奖，彝族群舞《花腰新娘》获得二等奖  ，在此次比赛中，宋茜被[[韓國|韓國]][[SM娛樂|SM娛樂]]星探发掘；12月，宋茜开始了为期两年的练习生生活，学习语言和HipHop舞蹈，同时进行演戏、歌唱、舞蹈等多方面训练。

== 演艺生涯 ==
=== 早期生涯 ===
2008年3月，出道前的宋茜曾出演由[[Rain|Rain]]演唱的北京奥运会火炬传递主题曲《Any Dream》MV<ref>{{cite web | url=http://sports.sina.com.cn/o/2008-03-21/15053546870.shtml | title=三星公布奥运火炬手名单 RAIN劲爆歌曲搭档中国美女 | publisher=新浪体育 | language=zh}}</ref>；5月，宋茜开始在亚洲作为广告模特和演员展开活动；8月20日，宋茜在北京奥林匹克公园参加奥运宣传活动，为中国奥运健儿加油<ref>{{cite web | url=http://2008.163.com/08/0821/12/4JSB4R3400742QDT.html | title=SJ-M劲歌热舞与宋茜亮相奥林匹克公园 | publisher=网易娱乐 | language=zh}}</ref>；同年，宋茜先后出演了歌曲《[http://www.youtube.com/watch?v=VTASffPQGhY 姐姐真漂亮]》、《[https://www.youtube.com/watch?v=bmS-sBfQjLA U]》、《[http://www.youtube.com/watch?v=m-V0xjcHDkQ 某一天心诉说着]》等MV。

=== 2009年－2011年：[[F(x)_(組合)|f(x)]]出道、积极出演综艺节目 ===
2009年9月，作為[[F(x)_(組合)|f(x)]]組合以歌手身份出道，在组合中担任隊長，主要负责领舞的工作。出道初期，由于[[F(x)_(組合)|f(x)]]知名度不高、成员们还未成年，对韩语还生疏的宋茜积极参加了各种综艺节目，因为外国人的身份受到不少韩国观众的关注，[[F(x)_(組合)|f(x)]]渐渐为人所熟悉。

2010年6月，以固定嘉宾身份出演[[KBS第2頻道|KBS 2TV]]農村實境綜藝節目《[[青春不敗|青春不敗]]》。同时，與韓國男子流行音樂團體[[2PM|2PM]]的成員之一[[Nichkhun|Nichkhun]]共同在[[文化廣播_(韓國)|MBC]][[綜藝節目|實境綜藝節目]]《[[我們結婚了|我們結婚了]]》中演出，並飾演一對假想夫婦，在韓國被稱為「Khuntoria夫婦」；在中國被稱為「[[我們結婚了#維尼夫婦|維尼夫婦]]」，該名字來自兩人的名字Nichkhun和Victoria的首個字母。同年年底在《[[MBC演藝大獎|MBC演藝大獎]]》一起獲選綜藝部最佳人氣獎。2011年12月31日，两人与节目中其他假想夫妇一起在《MBC歌谣大战》中担任主持人，维尼夫妇更特别与[[湖南卫视|湖南卫视]]跨年演唱会进行连线采访<ref>{{cite web | url=http://ent.163.com/12/0101/01/7ML61F9O00034O8M.html | title=湖南卫视跨年演唱会落幕 巨星云集引爆全场| publisher=网易娱乐 | language=zh}}</ref>。

=== 2012年－2014年：初入影視、节目主持 ===
2012年1月5日，受邀出演由中國上海辛迪加影視有限公司和台灣[[八大電視|八大電視股份有限公司]]聯合製作的偶像劇《[[愛情闖進門|愛情闖進門]]》，並首次擔當女主角。8月27日，《[[愛情闖進門|愛情闖進門]]》正式開播，獲得同時段收視率居首位的成績。12月31日，宋茜憑著飾演沈雅音一角於《2012[[國劇盛典|國劇盛典]]》上獲頒「最佳新人女演員」。<ref>{{cite web | url=http://video.sina.com.cn/p/ent/v/m/2012-12-31/201561962401.html | title=《国剧盛典》宋茜获最佳新人女演员 | publisher=新浪视频 | language=zh}}</ref>

2014年6月，主持[[江苏卫视|江苏卫视]]访谈综艺节目《[[最强天团|最强天团]]》<ref>{{cite web | url=http://sd.china.com.cn/a/2014/yule_0611/23498.html | title=最强天团即将播出 SJ成员与F(x)组合同台PK | publisher=中国网山东 | language=zh}}</ref>；9月，搭檔[[车太贤|車太賢]]出演个人首部电影《[[我的新野蛮女友|我的新野蛮女友]]》，饰演新一代的野蛮女友“星星”<ref>{{cite web | url=http://ent.sina.com.cn/m/c/2014-09-24/10464215420.shtml?qq-pf-to=pcqq.group | title=《我的新野蛮女友》开拍 车太贤搭宋茜 | publisher=新浪网 | language=zh}}</ref>。

=== 2015年－2016年：国内真人秀、成立個人工作室、主攻影视 ===
2015年2月，参加[[东方卫视|东方卫视]]明星旅行节目《花样姐姐》，这也是宋茜在中国的首个真人秀节目<ref>{{cite web | url=http://ent.sina.com.cn/tv/zy/2015-01-28/doc-iavxeafs0513588.shtml | title=韩女团f(x)宋茜出演中国版《花样姐姐》 | publisher=新浪娱乐 | language=zh}}</ref>。

2015年5月29日，[[中國大陸|中國大陸]][[社交網站|社交網站]][[微博|微博]]傳出宋茜單方面向[[韓國|韓國]][[經紀公司|經紀公司]][[SM娛樂|SM娛樂]]提出解約的消息。5月30日，SM娛樂否認宋茜提出解約的傳聞，同時表示將為宋茜開設工作室並全力支持她在中國國內的活動，而工作室隸屬於SM娛樂，SM娛樂和宋茜仍然保持經紀全約。<ref>{{cite web | url=https://share.iclient.ifeng.com/news/shareNews?forward=1&aid=99056387#backhead | title=宋茜：从来没想过与公司解约 | publisher=广州日报 | language=zh}}</ref>6月27日，SM娛樂表示宋茜确实已经在中国成立了个人工作室，宋茜会以工作室的形式和贾士凯合作，但宋茜依旧是SM娛樂旗下艺人，韩国地区的工作也继续由SM娛樂打理。宋茜未来会大大增加在中国各种演艺事业的活动，中韩双方会协调宋茜的行程安排。<ref>{{cite web | url=http://ent.sina.com.cn/v/m/2015-06-27/doc-ifxenncn6752213.shtml | title=独家：宋茜工作室合作贾士凯 将主攻影视 | publisher=新浪娱乐 | language=zh}}</ref>

==个人生活==
宋茜出生在山东省青岛市，是家中独生女，父母都是普通工人；小时候因为家离着学校远，宋茜便住在外公外婆家，考虑到外公外婆年纪大了，宋茜在生活上学会了自理，并培养出独立的性格。

== 社會活動 ==
2012年11月1日，宋茜参加了在北京昌平雨竹学校举行的CJ集团携手[[SM娛樂|韓國SM娛樂有限公司]]打造的农民工子弟音乐课堂活动。此次活动是为纪念中韩建交20周年而举办的活动之一，也是中国友好和平发展基金会“CJ CGV和谐基金”创立以来举办的首次公益活动<ref>{{cite web | url=http://ent.sina.com.cn/s/j/k/2012-11-02/23433778997.shtml | title=安七炫宋茜助阵农民工子弟梦想音乐教室 | publisher=新浪娱乐 | language=zh}}</ref>。

2014年12月21日，宋茜在SMTOWN Coex Artium开办跳蚤市场，进行慈善义卖活动，并将当天所有收入捐献给了联合国儿童基金会<ref>{{cite web | url=http://cqm.people.com.cn/news_zk/news/201514/2015141411878943.htm | title=f(x) Victoria宋茜低调行善 捐出跳蚤市场所得 | publisher=人民网 | language=zh }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2015年4月27日，宋茜携同《[[美丽的秘密|美丽的秘密]]》剧组发起“美丽童年的秘密”关爱听障儿童公益众筹项目，为听障儿童定制专属于自己的故事绘本，所得收益捐赠给中国聋儿康复研究中心，用于听障儿童救助<ref>{{cite web | url=http://ent.163.com/15/0427/10/AO7038M4000333J5.html | title=《美丽的秘密》杀青 宋茜与何润东启动公益 | publisher=网易娱乐 | language=zh}}</ref>；4月30日，宋茜在联合国儿童基金会网站拍卖由自己亲手制作的人偶“宋宋”，并将拍卖所得的所有款项捐助给联合国儿童基金会，用于救济儿童<ref>{{cite web | url=http://www.unicef.or.kr/join/awoo/auction_view.asp?Did=30044 | title=아우인형 경매하기 | publisher=unicef | language=ko}}</ref>；6月，宋茜通过参与联合国儿童基金会“帮助尼泊尔儿童”公益手链慈善项目，为尼泊尔贫困儿童筹集善款<ref>{{cite web | url=http://www.unicef.or.kr/helpnepalchildren/?TrackCode=helpnepal_mainban | title=HelpNepalChildren | publisher=unicef | language=ko}}</ref>；10月2日，宋茜参与在首尔江南区三成洞COEX举行的首尔综合艺术学校明星私人物品竞拍及慈善义卖会，当日所得资金全部捐给TGC江南文化基金，用以帮助社会弱势群体。

2016年9月3日，宋茜参加时尚芭莎联合一直播推出的公益直播《画出生命线》活动，在线直播筹集善款963万金币，约合人民币9.63万元；9日，宋茜参加Bazzar明星慈善夜，与现场嘉宾一同为爱开拍，助力慈善<ref>{{cite web | url=http://hb.people.com.cn/n2/2016/0912/c192237-28989724-7.html | title=芭莎慈善夜 刘诗诗杨幂领衔女神红毯争艳 | publisher=人民网 | language=zh}}</ref><ref>{{cite web | url=http://world.people.com.cn/n1/2016/0912/c57506-28709605.html | title=“思源工程”携手2016芭莎明星慈善夜助力脱贫攻坚战 | publisher=人民网 | language=zh}}</ref>。

2018年2月2日，宋茜生日当天携手芭莎公益慈善基金创立“宋茜爱心舞蹈教室”公益项目，为山村的孩子们送去丰富的艺术类美育教育资源<ref>{{cite web | url=https://www.toutiao.com/i6517995813757518344/ | title=宋茜×芭莎公益 爱心舞蹈教室圆梦起航！ | publisher=芭莎公益慈善基金 | language=zh}}</ref>。3月11日，宋茜在2017 BAZAAR明星慈善夜上捐赠10辆的芭莎爱心救护车被证实即将启程到内蒙古自治区，为需要的人送去温暖和保障<ref>{{cite web | url=http://www.sohu.com/a/225313377_267454 | title=落实行动送温暖 宋茜捐赠爱心救护车即将启程 | publisher=搜狐娱乐 | language=zh}}</ref>。
