== 简介 ==
宋英杰毕业于[[北京气象学院|北京气象学院]]天气动力专业。<ref name=" cat" >{{cite web |url=http://gongyi.people.com.cn/GB/151132/151666/153789/185565/11259335.html |title=宋英杰简历 |publisher= 人民网|accessdate=2010-03-30 |work= |language =zh-cn}}</ref>

1993年，宋英杰成为[[中国中央电视台|中国中央电视台]]《[[新闻联播天气预报|新闻联播天气预报]]》主持人，被誉为“中国第一气象先生”。<ref name=" cat" />

== 个人生活 ==
宋英杰的妻子叫刘珩，是他的师妹，比他小12岁。<ref name="lion " /><ref name="duck " />

2018年5月10日，宋英杰在微博上發佈信息，稱其4歲兒子自稱是從[[汶川大地震|汶川大地震]]時的[[北川羌族自治县|北川]]轉世而來，擁有“[[前世記憶|前世記憶]]”，在網上引發熱議<ref>{{Cite news|url=https://www.hk01.com/%E5%A4%A7%E5%9C%8B%E5%B0%8F%E4%BA%8B/191052/%E5%A4%AE%E8%A6%96-%E5%A4%A9%E6%B0%A3%E5%85%88%E7%94%9F-%E7%A8%B14%E6%AD%B2%E5%AD%90%E7%96%91%E5%BE%9E%E6%B1%B6%E5%B7%9D%E5%9C%B0%E9%9C%87%E8%BD%89%E4%B8%96-%E8%A6%81%E5%9B%9E%E5%8C%97%E5%B7%9D%E8%80%81%E5%AE%B6%E6%89%BE%E5%A5%B3%E5%85%92|title=央視「天氣先生」稱4歲子疑從汶川地震轉世：要回北川老家找女兒|last=朱加樟|date=2018-05-22|newspaper=香港01|accessdate=2018-06-15|language=zh-HK}}</ref>。
