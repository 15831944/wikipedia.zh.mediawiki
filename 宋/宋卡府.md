'''宋卡府'''（{{thai|t=จังหวัดสงขลา|r=Changwat Songkhla}}，{{IPA-th|t͡ɕāŋ.wàt sǒŋ.kʰlǎː}}）是[[泰國|泰國]]南部其中的一個府。

==歷史==
宋卡府的名字源於馬來語「星若拉」（{{lang-ms|Singgora}}），意思是「獅子山」。古代曾為馬來人建立的{{link-en|信哥拉蘇丹國|Sultanate of Singora}}（宋卡蘇丹國），元代[[汪大渊|汪大渊]] 《[[岛夷志略|岛夷志略]]》称之为[[东冲古剌|东冲古剌]]，《[[郑和航海图|郑和航海图]]》称之为[[孙姑那|孙姑那]]<ref>[[许云樵|许云樵]] 《[[北大年史|北大年史]]》二十页 </ref>

原名沙庭城，建於[[素可泰王朝|素可泰王朝]]前的千年古城，曾於[[大城王朝|大城王朝]]時代宣佈獨立，後被敉平而隸於[[那空是貪瑪叻府|那空是貪瑪叻城]]。[[鄭昭|鄭王大帝]]時，被昇為一等城，直轄於[[曼谷|曼谷]]。[[拉瑪五世|拉瑪五世]]時代，採用道省制，將博他侖、宋卡歸入那空是貪瑪叻道省（Monthon Nakhon Si Thammarat）的管轄，至1933年[[泰国|泰国]]取消省制為止，宋卡才成為府治。中世紀的[[海盜|海盜]]以此為據點，海上[[貿易|貿易]]繁忙，是通向[[新加坡|新加坡]]和[[馬來西亞|馬來西亞]]的南大門。

== 地理 ==
[[File:Songkhla_Laem_Samila.jpg|left]]

[[面積|土地面积]]約7393平方公里，距離[[曼谷|曼谷]]約950公里。宋卡府北連[[博他侖府|博他侖府]]和[[那空是貪瑪叻府|那空是貪瑪叻府]]；南毗[[也拉府|也拉府]]和[[馬來西亞|馬來西亞]]的[[玻璃市|玻璃市]]；東瀕[[北大年府|北大年府]]和[[泰國灣|泰國灣]]；西接[[博他侖府|博他侖府]]和[[沙敦府|沙敦府]]。

主要城市为[[合艾|合艾市]]和[[宋卡市|宋卡市]]。

本府东北部瀕临一个方圆一千多[[平方公里|平方公里]]的内陆[[湖泊|湖泊]]叫[[宋卡湖|宋卡湖]]，湖内盛产[[鱼|鱼]][[虾|虾]]，原先生活着一种叫Irrawaddy Dolphins的[[白骥豚|白豚]]，但是近年来白豚因所食[[鱼|鱼类]]被过量捕捞而濒于绝迹。

南部大部份為[[山|山巒]]和原始山脈及高阜地帶，北部為[[平原|平原]]，[[河流|河流]]多且[[流量|流量]]大，背山面海，[[漁業|漁業]]發達，盛產[[錫|錫]]、[[錳|錳]]、[[鎢|鎢]]、[[重晶石|重晶石]]等礦，[[山林|山林]]里出产[[木材|木材]]。

== 行政与管理 ==
本府行政上被划分为16个縣（Ampoe），又进一步被划分为127个区（Tambon）及987條村（Muban）。

今日的占那縣（Chana）、沙巴銳縣（Saba Yoi）和貼帕縣（Thepa）曾隶属[[北大年府|北大年府]]，近年才由泰国政府将她们划归宋卡府管辖。

== 社会动乱 ==
本府东南部的[[乍納区|乍納区]]（จะนะ ，Chana）、[[堤坝区|堤坝区]]（เทพา，Thepa）和[[沙波俞|沙波俞]]区（สะบ้าย้อย ，Saba Yoi）原先是从[[北大年府|北大年府]]中划分出来的，所以在广义上往往也将这三个小区域看作是[[泰南三府|泰南三府]]的一部份。这里由于民族关系错综复杂，社会较为动荡，人民的生命安全缺乏保障，近年来[[泰国|泰国]]和[[世界|世界各国]]的传媒对此地区发生的凶杀、暴力、恐怖事件有很多相关的报道。

泰国警方、军方一直怀疑有恐怖分子在这三个地区藏身，而且在合艾市中心、[[合艾国际机场|合艾国际机场]]、[[合艾火车站|合艾火车站]]等地制造过多起炸弹爆炸事件。然而，进入2007年，本府治安局势渐渐趋向稳定。

== 觀光熱點 ==
* [[合艾市|合艾市]]：[[新加坡|新加坡]]及[[馬來西亞|馬來西亞]]居民喜歡的購物及召妓的市鎮，市內色情按摩院及[[妓院|妓院]]林立，可看鬥牛表演。
* 東安撐瀑布及莫里拍瀑布：位於叻德蓬縣，為兩個美麗[[瀑布|瀑布]]，共有七層，水花飛濺四周如[[象牙|象牙]]。
* 蓮莎密拉半島：距直轄縣市場約三公里，水清沙白，岸邊為松林，附近是著名的貓鼠群島，海濱有座[[美人魚|美人魚]]石像作為地標。
* 考奈山及考登功山：位於蓮莎密拉半島附，山麓為花圃[[花園|花園]]，山頂有佛塔及[[燈塔|燈塔]]旗杆，遙望宋卡湖及全城美景。
* 告線沙灘：距蓮莎密拉半島約三公里，大小鎮石塊散布[[海灘|海灘]]，風光秀麗。
* [[鴿子|鴿子]]飼養場：乍納縣的居民流行飼養[[鴿子|鴿子]]，且每年1月及7月舉行鴿子比賽。
