'''宋丹丹'''（{{bd|1961年|8月25日|catIdx=S宋}}），[[北京|北京]]人，[[中国|中国]]著名女[[演员|演员]]，参加了大量著名的[[喜剧小品|喜剧小品]]和[[影视剧|影视剧]]作品演出和创作。她的主要演出作品包括，小品《[[懒汉相亲|懒汉相亲]]》、《[[超生游击队|超生游击队]]》、《[[手拉手|手拉手]]》、《[[昨天今天明天|昨天今天明天]]》、《[[钟点工|钟点工]]》等；电影《[[男妇女主任|男妇女主任]]》、《[[十面埋伏|十面埋伏]]》；电视剧《[[我爱我家|我爱我家]]》、《[[爱你没商量|爱你没商量]]》、《[[家有儿女|家有儿女]]》等；话剧《回归》、《万家灯火》。<ref>宋丹丹介绍</ref>

== 从艺经历 ==
宋丹丹1961年出生在北京一个普通[[知识分子|知识分子]]家庭，1981年考入[[北京人民艺术剧院|北京人民艺术剧院]]演员训练班<ref>{{cite web|url=http://www.bjry.com/actor/html/2015/06/20150605110.html|title=宋丹丹|publisher=[[北京人民艺术剧院|北京人民艺术剧院]]|date=|accessdate=2017年2月24日}}</ref>。毕业后不久，因在话剧《红白喜事》中扮演“灵芝”一角色，宋丹丹开始了她的艺术生涯。1985年因在电视剧《寻找回来的世界》中，饰演宋小丽，而获大众电视[[飞天奖|飞天奖]]“最佳女配角奖”。

在1989年的[[中国中央电视台春节联欢晚会|中国中央电视台春节联欢晚会]]上，宋丹丹参与出演了小品《懒汉相亲》，并开始在中国的小品界崭露头角。继而在1990年的央视元旦晚会上，宋丹丹和[[黄宏|黄宏]]出演的小品《[[超生游击队|超生游击队]]》，受到观众更强烈的喜爱。

== 家庭 ==
24岁为摆脱初戀情伤而闪电结婚，該段婚姻只维持了一年；离婚一年后，宋丹丹1989年和[[导演|导演]][[英达|英达]]结婚，婚后育有一子[[巴图_(演员)|英巴图]] ; 两人曾在中国内地影视界，被称为“模范夫妻”，但1997年两人突然宣布离婚。同年，再次閃電嫁给现任老公赵玉吉。2007年，宋丹丹出版自传《[[幸福深处|幸福深处]]》（[[长江文艺出版社|长江文艺出版社]]，2007年3月），首次披露了以上的感情生活<ref>宋丹丹英达十年前的婚变真相 原载《江淮晨报》</ref>。

宋丹丹的[[父亲|父亲]][[宋汎|宋汎]]，曾任北京市[[文联|文联]]党组书记，副主席。

宋丹丹的长兄[[宋北杉|宋北杉]]，曾任[[山西省|山西省]]副[[省长|省长]]，现任[[全国工商联|全国工商联]]副[[主席|主席]]。

== 趣事 ==
* 2011年，宋丹丹与[[潘石屹|潘石屹]]在[[微博|微博]]上争议潘公司在北京市中心城区建楼影响市容的问题。宋丹丹格式为“××，我就是个××，没多少钱，我请你××，别再××了，真的，求你了！”的文字被网友戏称为“丹丹体”。
