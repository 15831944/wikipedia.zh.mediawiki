== 用途 ==
胺甲酸是制備尿素中的中間產物，是氨和二氧化碳反應時產生的<ref name="Ullmann">{{Ullmann|author=Meessen, J. H.|author2=Petersen, H.|title=Urea|doi=10.1002/14356007.a27_333}}</ref>
: CO<sub>2</sub>  +  NH<sub>3</sub> → H<sub>2</sub>N-CO<sub>2</sub>H
: H<sub>2</sub>N-CO<sub>2</sub>H  +  NH<sub>3</sub>   →  CO(NH<sub>2</sub>)<sub>2</sub>  +  H<sub>2</sub>O

胺甲酸轉移酶（Carbamoyltransferases）屬於[[轉移酶|轉移酶]]，[[EC編號|EC編號]] 2.1.3。
