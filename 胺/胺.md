'''胺'''（英語：'''amine'''）是[[氨|氨]]分子（NH<sub>3</sub>）中的[[氢|氢]]被[[烃基|烃基]][[取代基|取代]]后形成的一类[[有机化合物|有机化合物]]。'''氨基'''（－NH<sub>2</sub>、－NHR、－NR<sub>2</sub>）是胺的[[官能团|官能团]]。

如果氮原子连着[[羰基|羰基]](C=O)，那么该化合物则称为[[酰胺|酰胺]]，其化学性质与胺并不相同。

==胺的分類==
胺分子可根据烃基的种类，分为[[脂肪族化合物|脂肪]]胺和[[芳香族化合物|芳香]]胺。

此外，胺還可根据氨分子上被取代的氫原子数量，順次分为'''伯胺'''（'''一級胺'''）、'''仲胺'''（'''二級胺'''）、'''叔胺'''（'''三級胺'''）。此外，还有[[季铵盐|季铵盐]]（四級铵鹽），可以看成是[[铵根|铵根]]离子（NH<sub>4</sub><sup>+</sup>）的四个氫都被取代的产物。

==物理性質==
一级胺及二级胺的物理性质主要受[[氢键|氢键]]影响，然而因為[[氮|氮]]與[[氫|氫]][[陰電性|陰電性]]的差別(3.0-2.1=0.9)並無[[氫|氫]]和氧間(3.5-2.1=1.4)那麼大，所以一般   N-H...H氫鍵較不像O-H...H鍵那麼強，故胺的[[沸点|沸点]]一般较相对的磷烃为高，却低于相对应的[[醇|醇]]。
例如，

[[甲胺|甲胺]]和[[乙胺|乙胺]]在室温之下为气体，但[[甲醇|甲醇]]和[[乙醇|乙醇]]在室温之下为液体。甲醇與甲胺分子量相近，其[[沸點|沸點]]差異卻甚大。
幾乎所有胺類皆與水形成氫鍵，所以比同[[分子量|分子量]]的[[烴|烴]]易溶於水，低分子量的胺類與水完全互溶，高分子胺類只適量溶於水。
气体胺带有一种类似[[氨|氨]]的气味，液体胺则带有一种容易辨出的鱼腥味。

氮原子连有三个不同基团形如NHRR′与NRR′R″的胺是有[[手性|手性]]的，因氮有一对孤对电子，分子呈三角锥形，氮是手性中心。唯其手性翻转能垒相对很低，对于三烷基胺通常只需活化能25-37.6 kJ/mol，两个對映體在室温下就能迅速转化故對映體不能分离。<ref name="Xing">邢其毅等．《基础有机化学》第三版 下册．北京：高等教育出版社，2005年．ISBN 978-7-04-017755-8</ref>若季铵盐氮所连四个基团不同，则能拆分出旋光异构体。有些环状的胺因位阻不能翻转成對映體也能拆分。

==合成==
===鹵化物氨解（Ammonolysis of halides）===
{{seealso|氨解}}
氨或胺氮上有孤对电子，作为亲核试剂与卤代烷发生[[亲核取代反应|亲核取代反应]]，按S<sub>N</sub>2机理进行。許多有機鹵化物以[[氨水|氨水]]溶液或氨溶液處理則變成胺類：
: RX + NH<sub>3</sub> → RNH<sub>3</sub><sup> + </sup>X<sup>-</sup>
: RNH<sub>3</sub><sup>+</sup>X<sup>-</sup> + NH<sub>3</sub> → RNH<sub>2</sub> + NH<sub>4</sub><sup>+</sup> + X<sup>-</sup>
: （X為鹵素）

===用醇制备===
胺的主要合成方法是氨的[[烷基化|烷基化]]。工业上使用醇与氨合成有机胺：
:ROH + NH<sub>3</sub> → RNH<sub>2</sub> + H<sub>2</sub>O
这些反应需要使用[[催化剂|催化剂]]、特制仪器及额外纯化，因为得到的是一、二、三级胺的混合物，需要提高反应的选择性。
