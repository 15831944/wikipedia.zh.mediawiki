== 習性 ==
地生，生長在林下遮蔭且腐植質豐富的環境。<ref name=":1">{{Cite book|title=蕨類圖鑑1|last=郭|first=城孟|publisher=遠流出版事業股份有限公司|year=2001年11月30日 初版一刷 2013年9月15日 三版三刷|isbn=957-32-4520-5|location=台北市南昌路2段81號6樓|pages=421}}</ref>

== 分布 ==
中國大陸西南部、喜馬拉雅山區、中南半島、臺灣低、中海拔山區（約海拔1000到2000公尺）之闊葉樹林陰濕地可見。<ref name=":1" /><ref>{{Cite web|url=http://taibif.tw/zh/namecode/205153|title=Monachosorum henryi Christ, 1898 稀子蕨 {{!}} 台灣生物多樣性資訊入口網|accessdate=2017-03-29|work=taibif.tw|language=zh-hant}}</ref> 

稀子蕨的分布狀況和臺灣多種裸子植物類似，產於喜馬拉雅山東部一帶和臺灣，這個範圍是200萬至1萬年前冰河期之際全球最大的避難所，因此稀子蕨可能是冰河期當時生物避難所遺留下來的物種。<ref name=":1" /><ref name=":0" />

== 特徵 ==
[[File:稀子蕨不定芽.jpg|缩略图]]
多年生蕨類植物；根莖短且斜生（ascending）；葉叢生，葉柄長25～50cm，基部暗褐色，密被腺毛；葉片卵形至三角形，長30～60cm，寬20～35cm，三到四回羽狀複葉，草質，先端銳尖（acute），末羽片橢圓形，葉軸表面常有拳頭狀不定芽，羽軸表面有時也有，落地後可萌出新株，葉近軸面與遠軸面相似；孢子囊群無孢膜、圓形，著生於側脈頂端。<ref>{{Cite book|title=FLORA OF TAIWAN SECOND EDITION VOLUME ONE|last=|first=|publisher=臺灣植物誌第二版編輯委員會|year=1994|isbn=957-9019-52-5|location=中華民國 臺灣 臺北|pages=648}}</ref><ref name=":1" />

有些蕨類的不定芽成熟後會被風或水帶走，如東方狗脊蕨，然而稀子蕨的芽太大太重，無法乘風離去，且長在林下遮蔭處、不靠水邊，很難藉著水離開。那麼，花了比別人更多的能量來生長比其他蕨類都顯得龐大的不定芽，似乎不是明智之舉，幸好這種在冰河時期於臺灣避難的蕨類，於中海拔找到了適合它的環境，所以族群不少。<ref>{{Cite news|url=http://e-info.org.tw/node/51209|title=「蕨」對有趣：觀察蕨類的「不定芽」|newspaper=台灣環境資訊協會-環境資訊中心|accessdate=2017-03-29|language=zh-hant}}</ref>

== 應用 ==
稀子蕨的不定芽可以食用，不但是許多野生動物的美食，也有紀錄表示原住民族會食用稀子蕨的不定芽<ref>{{Cite journal|title=台東縣卑南鄉魯凱族達魯瑪克部落傳統有用植物之調查研究|author=劉炯錫|url=http://jtc.nttu.edu.tw/upload/admin/journal_paper/011a-02.pdf|journal=台東師院學報|issue=11|doi=|others=|year=民國89年|volume=1|page=p.29-60|pmid=|access-date=2017-03-29|archive-url=https://web.archive.org/web/20170329143540/http://jtc.nttu.edu.tw/upload/admin/journal_paper/011a-02.pdf|archive-date=2017-03-29|dead-url=yes}}</ref>，烤熟後味道如帶有草味的花生。
