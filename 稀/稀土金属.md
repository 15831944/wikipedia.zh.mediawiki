'''稀土金属'''，或称'''稀土元素'''，是[[元素週期表|元素週期表]]第Ⅲ族副族元素[[钪|钪]]、[[钇|钇]]和[[镧系元素|镧系元素]]共17种[[化学元素|化学元素]]的合称。[[钪|钪]]和[[钇|钇]]因为经常与[[镧系元素|镧系元素]]在矿床中共生，且具有相似的化学性质，故被认为是稀土元素。

与其名称暗示的不同，稀土元素（[[钷|钷]]除外）在地壳中的豐度相当高，其中[[铈|铈]]在地壳元素豐度排名第25，占0.0068%（与[[铜|铜]]接近）。稀土元素並不稀有，但其傾向於兩兩一起生成合金，且難以將稀土元素單獨分離。另外，稀土元素在地殼中的分佈相當分散，很少有稀土元素集中到容許商業开采的礦床。人类第一种发现的稀土矿物是从[[瑞典|瑞典]][[伊特比村|伊特比村]]的矿山中提取出的{{link-en|硅铍钇矿|Gadolinite}}，许多稀土元素的名称正源自于此地。

== 性质 ==
[[File:Rareearthoxides.jpg|thumb]]
多数呈银灰色，有光泽，[[晶体结构|晶体结构]]多为[[六方最密堆積|六方最密堆積]]（HCP）或[[面心立方|面心立方]]（FCC）。性质较软，在潮湿空气中不易保存，易溶于稀酸。

原子价主要是+3价（[[铈|铈]]正四价较稳定，[[镨|镨]]和[[铽|铽]]也有极个别的四价[[氧化物|氧化物]]，[[钐|钐]]、[[铕|铕]]、[[镱|镱]]有二价化合物），能形成稳定的[[配合物|配合物]]及微溶于水的[[草酸盐|草酸盐]]、[[氟化物|氟化物]]、[[碳酸盐|碳酸盐]]、[[磷酸盐|磷酸盐]]及[[氢氧化物|氢氧化物]]等。

在三价稀土氧化物中，[[氧化镧|氧化镧]]的吸水性和碱性与[[氧化钙|氧化钙]]相似，其余则依次转弱。三价稀土的化学性质除[[钪|钪]]的差异较显著外，其余都很相似，所以分离较难。

== 稀土金属所含元素 ==
具体的稀土金属包括：[[镧|镧]]（La）、[[铈|铈]]（Ce）、[[镨|镨]]（Pr）、[[钕|钕]]（Nd）、[[钷|钷]]（Pm）、[[钐|钐]]（Sm）、[[铕|铕]]（Eu）、[[钆|钆]]（Gd）、[[铽|铽]]（Tb）、[[镝|镝]]（Dy）、[[钬|钬]]（Ho）、[[铒|铒]]（Er）、[[铥|铥]]（Tm）、[[镱|镱]]（Yb）、[[镥|镥]]（Lu）以及与[[镧系|镧系]]的15个元素密切相关的两个元素：[[钪|钪]]（Sc）和[[钇|钇]]（Y），称为稀土元素（Rare Earth），简称稀土（RE）。

=== 分类 ===
根据稀土元素[[原子|原子]][[电子层|电子层]]结构和物理化学性质，以及它们在矿物中共生情况和不同的[[离子|离子]]半径可产生不同性质的特征，十七种稀土元素通常分为二组：

'''轻稀土'''包括：[[镧|镧]]、[[铈|铈]]、[[镨|镨]]、[[钕|钕]]、[[钷|钷]]、[[钐|钐]]、[[铕|铕]]、[[钆|钆]]。

'''重稀土'''包括：[[铽|铽]]、[[镝|镝]]、[[钬|钬]]、[[铒|铒]]、[[铥|铥]]、[[镱|镱]]、[[镥|镥]]、[[钪|钪]]、[[钇|钇]]。

也可分为'''铈组'''（镧、铈、镨、钕、钷、钐）和'''钇组'''（铕、钆、铽、镝、钬、铒、铥、镱、钇）。

== 分布 ==
[[File:元素豐度.svg|thumb]]
[[File:Rareearth_production.svg|thumb]]
[[File:Lutetium_sublimed_dendritic_and_1cm3_cube.jpg|thumb]]的加工形態]]

各种稀土元素常以差别很大的含量存在于同一矿石中，如[[独居石|独居石]]及氟碳铈矿中以轻稀土为主，钪钇矿石以钪和钇为主，黑稀金矿以钇及重稀土为主。

全世界已知有约9261万吨稀土矿，其中有约占36%储藏在[[中华人民共和国|中国]]。[[内蒙古|内蒙古]][[包头市|包头市]]的[[白云鄂博|白云鄂博]]、[[江西|江西]]、[[广东|广东]]、[[四川|四川]]等地混合矿储量巨大。中国是目前世界第一大稀土矿生产国，也是稀土矿第一大出口国，约占世界出口额总量的90%。2009年中国开始管控和限制稀土矿石出口，此举有利于保护资源环境和遏制过度开发，但同时也引起美日欧等国家的不满。2012年3月美国表示将就中国限制稀土矿出口问题向[[世贸组织|世贸组织]](WTO)提起诉讼，欧盟和日本将会支持并参与其中<ref>[http://finance.china.com/html/c/20123/15/1333958_1_418.html 美国将就中国稀土出口问题向世贸组织提出诉讼] {{webarchive|url=https://web.archive.org/web/20121007061635/http://finance.china.com/html/c/20123/15/1333958_1_418.html |date=2012-10-07 }}</ref>。中国报道称美国和其它国家虽有大量稀土矿资源，但因其国内工资远高于中国，开采经济效益不高，故长期向中国廉价进口。各国普遍认为中国限制稀土出口与中日领土争端有关，而中国则指责美国和其它国家为战略储备和保护珍贵资源的目的长期向中国廉价进口，中国以此为理由限制稀土出口规模<ref>[http://opinion.people.com.cn/GB/223228/17401411.html 人民网评：应对稀土贸易战，中国理直气壮]</ref>。

其它拥有稀土资源的国家和地区有[[独联体|独联体]]国家（19%）、[[美国|美国]]（13%）、[[澳大利亚|澳大利亚]]（5%）、[[印度|印度]]（3.14%）、[[加拿大|加拿大]]、[[南非|南非]]、[[巴西|巴西]]、[[马来西亚|马来西亚]]和[[越南|越南]]等，而[[欧盟|欧盟]]和[[日本|日本]]基本没有稀土资源。它们的稀土来源主要从[[中国|中国]]进口<ref>[http://money.163.com/special/view140/ 网易特别策划_ 遭遇美日欧围剿 中国打响“稀土之战”]</ref>。

日本科學家於2011年7月4日聲稱，在[[太平洋|太平洋]]海底發現巨大的稀土蘊藏，可開採量約是全球陸地上已證實藏量的1000倍。<ref>[http://hk.news.yahoo.com/%E6%97%A5%E7%99%BC%E7%8F%BE%E5%A4%AA%E5%B9%B3%E6%B4%8B%E7%A8%80%E5%9C%9F%E7%A4%A6-%E8%97%8F%E9%87%8F%E9%AB%98%E9%99%B8%E5%9C%B0%E5%8D%83%E5%80%8D-220825236.html 日發現太平洋稀土礦 藏量高陸地千倍]</ref>又2012年6月28日[[東京大學|東京大學]]研究團隊在日本最東端的[[南鳥島|南鳥島]]周邊海域，發現含大量稀土的泥礦床，其蘊藏量相當於日本約230年的消費量。<ref>[http://www.bbc.co.uk/zhongwen/trad/world/2012/06/120628_japan_rare_earth.shtml 日本宣佈發現大規模海底稀土礦床]</ref>但目前并无任何国家到太平洋海底开采出稀土矿资源，而从经济角度来讲，相较于目前稀土矿价格，自海底开采并不符合经济效益。

=== 稀土金屬礦之開採 ===
由週期表特性可知稀土元素不易形成大規模礦區，必以大量挖掘後再予以過濾。副作用是損失大量土方及飛塵污染，容易引發居民抗議，因此多被苛徵巨額環保稅。

扣除上列因素，原則上能挖掘的古大陸地形都能開採稀土，早期伴隨煤鐵石油鑽石礦區一起挖掘，例如：近[[北極圈|北極圈]]各國與[[中歐|中歐]]；機械工具發達後便不受限制，多半由運輸成本及出口稅率而定，例如[[前蘇聯|前蘇聯]]、中國與澳洲等。

迄2010年底為止主要稀土出口地為中國（約佔97%{{citation needed}}），以鉛、鋅、高嶺土等初級品為主；可精鍊治金品目前由大型礦業或鋼鐵公司掌握專利，例如：[[必和必拓|必和必拓]]（BHP Billiton）、[[力拓集團|力拓]]（Rio Tinto）、[[淡水河谷公司|淡水河谷]]（Vale）等等。此外為增加工作人口，澳洲、中國、巴西等地都有意提高原料出口價格，以吸引礦商與金屬業者到地主國投資先進[[冶金|冶金]]技術。

== 应用 ==
稀土已被广泛应用于[[國防工業|國防工業]]、[[冶金|冶金]]、[[机械|机械]]、[[石油|石油]]、[[化工|化工]]、[[玻璃|玻璃]]、[[陶瓷|陶瓷]]、[[纺织|纺织]]、[[皮革|皮革]]、[[农牧养殖|农牧养殖]]等各传统方面领域，在社会生活中几乎随处可见。

作为改性添加元素在[[钢铁|钢铁]]和[[有色金属|有色金属]]中加入极少量稀土就能明显改善金属材料性能，提高钢材的强度及[[耐磨性|耐磨性]]和[[抗腐蚀性|抗腐蚀性]]能力。<ref>2009-2012年中国稀土市场投资分析及前景预测报告（09年5月）</ref>

== “稀土金属”一词存在的争议 ==
“稀土”中的“土”字实际上指的是[[氧化物|氧化物]]。这些元素被发现时人们以为它们在地球上分布非常稀少。实际上它们在[[地壳|地壳]]内的含量并不算太低，最高的[[铈|铈]]是地壳中第25丰富的元素，比[[铅|铅]]还要高，而“稀土金属”中稳定元素里丰度最低的[[镥|镥]]在地壳中的含量比[[金|金]]甚至还要高出200倍。
