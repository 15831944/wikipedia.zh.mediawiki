{{japanese|japanese=候文|kana=そうろうぶん|romaji=sōrōbun}}
'''候文'''，是[[日本語|日本語]]在[[中世紀|中世紀]]至[[近代|近代]]期間使用的一種[[文語體|文語體]]（[[文言文|文言文]]），屬{{nihongo|變體漢文|変体漢文}}。因其在句末使用[[丁寧語|丁寧]][[助動詞|助動詞]]{{nihongo|「候」|そうろう、そろ|[[歷史假名遣|歷史假名遣]]作「{{lang|ja|サウラフ}}」}}，故而得名。
== 历史 ==
{{nihongo|「候」|古代作サモラフ、サブラフ等}}一詞本意是服侍貴人，[[平安時代|平安時代]]成為{{nihongo|「有」或「在」|居り}}的[[日語敬語|謙讓語]]，後來又轉為表示丁寧的助動詞。[[鐮倉時代|鐮倉時代]]日本開始普遍在官方文書中使用文語體（文言文）。到了[[江戶時代|江戶時代]]，候文成為日本最常用的文語體。

[[明治維新|明治維新]]以降，候文的地位開始被官方[[漢文訓讀體|漢文訓讀體]]、民間口語體取代。[[二戰|二戰]]結束之後，日本官方文書一律採口語體，徹底廢除了候文及漢文訓讀體。
== 特征 ==
江户时期候文的特征在可以从文字和文体两方面来看。

所用文字包括[[汉字|汉字]]的行草体及其异体字，[[变体假名|变体假名]]、用行草体书写的[[文言文|文言文]]助词、[[平假名|平假名]]、[[片假名|片假名]]以及[[合字|合字]]等等。

候文的文体特征是在按照日文语序排列词汇的文章里夹杂书写源自汉文文言文的{{nihongo|返读字|返し読み}}，通常在句末使用「候」字而被称为「候文」。没有[[浊音符|浊音符]]，也没有句读等[[标点符号|标点符号]]。

{{nihongo|返读字|返し読み}}例：[[助动词|助动词]]如、如（ごとし）、不（ず）、為（す・さす・たり）、令（しむ）、可（べし）、被（る・らる）等等。其他还包括动词、助词等等。

二战后国语改革全部改用「平假名」书写的[[連詞|連詞]]、[[副詞|副詞]]、[[代词|代词]]、助动词，大部分在候文中都用汉字及其略体（比如「候」字简写为一点・）。
* 连词　「あるいは」（或者）、「しかれば」（然者）、「なおまた」（尚又）、「もっとも」（尤）、「または」（又者）など 
* 副詞　「いささか」（聊）、「いまもって」（今以）、「いよいよ」（弥）、「かねて」（兼而）、「もし」（若）、「しかのみならず」(加之)、「おそれながら」（乍恐）、「もとより」（固より）など 
* 代詞　「この」（此）、「これ」（之・是）、「その」（其）、「それ」（夫）など 
* 助动词词　「そうろう」（候）、「なり」（也）、如（ごとし）、不（ず）、為（す・さす・たり）、令（しむ）、可（べし）、被（る・らる）、度（たし）など<ref>[[林英夫|林英夫]]监修『おさらい古文書の基礎・文例と語彙』柏書房</ref>
* 前缀　「あい」（相）
* 辅助动词　「いたし」(致)、「ほうじ」(奉)、「し」(仕)、「もうし」（申）、「いり」（入）、「いたしおり」（致居）
* 连词　「よう」（様）、「ところ」（所）、「おもむき」（趣）、「ゆえ」（故）、「ども」（共）、「につき」（に付）
* 其他　「～にぞんじそうろう」（～に存候）、「～にそうろう」（～に候）、「これあり」（有之）、「これなし」（無之）、「ござそうろう」（御座候）、「ござなくそうろう」（無御座候）

越正式的文章越多使用汉文的问题，而不写假名。根据作者和文章的性质，汉字、假名的用法都不甚相同。不仅女性书写的书信多使用假名，当时即便是男性，对于私人、内部的文章也可以更多地使用假名书写。<ref>[[日本歴史学会|日本歴史学会]]編『演習古文書選』吉川弘文館</ref>。

不仅行政司法等公文，其他各种商业文章、日記、记录、通关文书等各种文献都用的是「候文」。由于是书面语而非[[方言|方言]]，因此能在克服方言障碍、通用全国的层面上来看，在当时也是一种非常方便的一种文体<ref>摘自「方言与候文的关系」，[[五十嵐力|五十嵐力]]监修《手紙講座第1巻』平凡社、昭和10年より。</ref>。

== 例文 ==

*法令・庆应三年第十三
此文是[[王政復古|王政復古]]大号令相关法令・庆应三年（[[1867年|1867年]]）第十三（庆应三年十二月九日公布）后半的部分摘录。<ref>原文出处：「[[近代デジタルライブラリー|近代デジタルライブラリー]][[法令全書|法令全書]][[慶応|慶応]]3年」で検索。2012年8月23日閲覧。（10/383頁）。</ref>。当时的[[法令|法令]]都是用'''候文'''书写的。

{{quotation|
一　{{lang|ja|舊弊御一洗に付言語の道被洞開'''候間'''見込有之向は不拘貴賤無忌憚可致獻言且人才登庸第一之御急務に'''候故'''心當之仁有之'''候者'''早々可有言上'''候事'''

一　舊弊（きゅうへい）御一洗に付、言語の道 被洞開'''候間'''（どうかいせられそうろうあいだ）、見込有之（これある）向は不拘貴賤（きせんにかかわらず）、無忌憚可致獻言（きたんなくけんげんいたすべく）、且人才登庸第一之御急務に'''候故'''、心當之仁有之'''候者'''（こころあたりのじんこれありそうらわば）、早々可有言上（ごんじょうあるべく）'''候事'''。}}
}}

*翻译成现代日语：「{{lang|ja|旧弊一新につき言論の道を広く開くこととなったので考えがあるものは身分にかかわらず忌憚なく上申せよ。また、人材登用を急務とするので心当たりの人物がいれば早々に推薦せよ}}」

*翻译成中文文言：「舊弊已一洗，需洞開言語之道。若有預見人選，不拘貴賤、無需忌憚，可致獻言，且人才登庸為第一急務，若有心當之仁，需早早推薦。」
