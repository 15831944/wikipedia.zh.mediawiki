《'''候任、現任及離任行政長官及主要官員的保障制度'''》法案<ref>法案 -《候任、現任及離任行政長官及主要官員的保障制度 {{cite web |url=http://www.al.gov.mo/proposta/chefe/cn.htm |title=存档副本 |accessdate=2014-05-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20140529052740/http://www.al.gov.mo/proposta/chefe/cn.htm |archivedate=2014-05-29 }} </ref>，簡稱'''「離補」法'''<ref>法案的簡稱取自法案的其中一項重點「一次性'''離'''任'''補'''償」</ref>，是於2013年12月16日由澳門特區政府提交于立法會審議，在「一般性討論及表決」獲得通過的法案。之後政府再向立法會建議將公務員或非公務員出身主要官員，一次性離任補償由原先的14%調升至30%，在表決前引起澳門市民強烈不滿，於2014年5月25日下午舉行[[2014年澳門反離補大遊行|2014年澳門反離補大遊行]]，要求政府撤回法案。

==法案重點==
建議立法設立特首及9名主要官員的福利制度，包括：
* 特首離任後可取得相等於月薪七成的長俸
* 主要官員離任則可獲得額外一次過補償，相當於任職月份乘以月薪的14%至30%
* 建議特首在任期內可享有刑事豁免權

以現任澳門特首[[崔世安|崔世安]]為例，他於1999年至2009年擔任社會文化司長10年，可獲700多萬元離任補償，離任特首時另可領取每月約18.9萬元「長俸」。

==進展==
# 2013年12月16日交于澳門立法會審議，在「一般性討論及表決」獲得通過<ref>特首及主要官員離任保障一般性表決 http://almacau.net/vote/?bill=24</ref>
# 政府再向立法會建議將公務員或非公務員出身主要官員，一次性離任補償由原先的14%調升至30%，並會於2014年5月27日表決。
# 2014年5月25日下午逾二萬名市民參與[[2014年澳門反離補大遊行|2014年澳門反離補大遊行]]，要求政府撤回法案。
# 2014年5月25日晚上，行政會召開緊急會議，要求將有關高官保障制度法案重新提交至立法會。
# 2014年5月26日，澳門特首[[崔世安|崔世安]]，提出取消原定於2014年5月27日討論的「離補」法案議程。
# 2014年5月27日由澳門議員[[吳國昌|吳國昌]]提出「撤回《高官離補法案》致意動議」，被否決<ref>撤回「高官離補法案」致意動議 http://almacau.net/vote/?bill=33</ref>。
# 2014年5月27日下午澳門越七千名市民包圍立法會，要求政府撤回法案。
# 2014年5月29日上午澳門行政長官崔世安宣佈，將致函澳門立法會撤回《候任及離任行政長官及主要官員的保障制度》草案。<ref>	
特首宣布撤回高官保障制度法案 http://www.tdm.com.mo/c_news/radio_news.php?id=217466</ref>

==爭議==
澳門市民對於「離補」法的推行，有下列爭議：
* 為何一般公務員都沒有「長俸」制度下，特首及主要官員卻可擁有「長俸」制度。
* 主要官員表現未達市民預期，為何還可收取更多褔利？
* 沒有作「全民咨詢」就草草推出法案
* 為何特首可享「刑事豁免權」，是否為自己訂做「保護傘」?
