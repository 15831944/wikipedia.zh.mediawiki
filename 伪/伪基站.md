'''伪基站'''，又称'''假基站'''、'''假基地台'''，是一种利用[[GSM|GSM]]单向认证缺陷的非法[[无线电|无线电]]通信设备，主要由[[主机|主机]]和[[笔记本电脑|笔记本电脑]]组成，能够搜取以其为中心、一定半径范围内的GSM移动电话信息，并任意冒用他人手机号码强行向用户手机发送[[诈骗|诈骗]]、[[推销|推销]]等[[垃圾短信|垃圾短信]]，通常安放在[[汽车|汽车]]或者一个比较隐蔽的地方发送。伪基站运行时，用户手机信号被强制连接到该设备上，无法连接到公用电信网络，以影响手机用户的正常使用。<ref name="伪基站盗手机号推送诈骗信息">
{{cite web |url = http://bjyouth.ynet.com/3.1/1308/06/8185932.html |title = 伪基站盗手机号推-{}-送诈骗信-{}-息 |website = 北青网 |deadurl = yes |archiveurl = https://web.archive.org/web/20141006085023/http://bjyouth.ynet.com/3.1/1308/06/8185932.html |archivedate = 2014-10-06 }}</ref><ref name="谁发的诈骗短-{}-信">{{cite web |url = http://jingji.cntv.cn/2014/09/23/VIDE1411486556416731.shtml |title = 《经济半小时》 20140923 聚焦网络电信骗局：谁发的诈骗短信 |author=王玉飞 |work=[[中国中央电视台财经频道|中国中央电视台财经频道]] }}</ref>

==工作原理==
伪基站利用[[移动信令监测系统|-{zh-tw:行動;zh-hk:流動;zh-cn:移动;}-信令监测系统]]监测移动通讯过程中的各种信令过程，获得手机用户当前的位置信息。伪基站启动后就会干扰和屏蔽一定范围内的[[运营商|运营商]]信号，之后则会搜索出附近的手机号，并将短信发送到这些号码上。屏蔽运营商的信号可以持续10秒到20秒，短信推送完成后，对方手机才能重新搜索到信号。大部分手机不能自动恢复-{}-信号，需要重启。{{vc}}伪基站能把发送号码显示为任意号码，甚至是邮箱号和[[特服号码|特服号码]]。载有伪基站的车行驶速度不高于60公里/小时，可以向周边用户群发短信，因此，伪基站具有一定的流动性。<ref>[http://www.hinews.cn/news/system/2014/04/30/016638455.shtml 伪基站的工作、设备原理]，南海网</ref>

一些功率大的伪基站，辐射的范围很广，只要伪基站不关闭发射，就会不断的有手机被吸入。离基站越近，被吸入的可能性越大。当伪基站正在发送诈骗短信的时候，附近有一万人经过，至少有5000人会受到影响。<ref name="谁发的诈骗短-{}-信"/>

==生产和贩卖==
伪基站可以在[[淘宝网|淘宝网]]上买到，一台“伪基站”的价格不足5万元[[人民币|人民币]]，大都是私人厂家自行组装。伪基站的产地主要以江苏、上海和广东居多。但事实上，淘宝网规定不能交易此类产品，若在淘宝搜索“伪基站”一词则会屏蔽，并显示“根据相关法律法规和政策，无法显示”的消息。<ref name="伪基站盗手机号推送诈骗信息" />

==危害==
伪基站的危害巨大。根据[[奇虎|360]]互联-{}-网安全中心最新发布的《全国首份伪基站短-{}-信治理报告》显示，中国大陆70.2%的伪基站诈骗短信冒充运营商诱导用户点击恶意网址。<ref>{{cite web |url = http://finance.people.com.cn/n/2014/0906/c1004-25616239.html |title = 超七成伪基站短信冒充运营商诈骗 |website = 人民网 }}</ref>

除中国外，美国、印度、捷克等国家均存在伪基站的安全威胁<ref>[http://digi.163.com/14/0924/16/A6U0T3RD001618JV.html 外国人怎么看中国式诈骗？360报告震惊美国网友]</ref>。

==事件==
由于伪基站的出现成为[[垃圾短信|垃圾短信]]的一大元凶，近年来，[[中国大陆|中国大陆]]出现了众多涉及伪基站的案件。<ref>[http://www.sznews.com/tech/content/2014-06/16/content_9660441.htm 伪基站年发垃圾短信近千亿条危害巨大]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
* 2012年9月至11月，汪某团伙在[[深圳宝安机场|深圳宝安机场]]利用一台伪基站设备发送[[机票|机票]]广告短信，非法获取62万部手机信息，导致机场众多手机用户无法通话，造成区域手机通话业务量损失达100余万元。一名司机到机场后发现手机信号经常不好，有一次中断了2个小时。<ref name="北上广等12省区市发现“伪基站”">{{cite web |url = http://money.163.com/13/0806/06/95ISVCBU00252G50.html |title=北上广等12省区市发现“伪基站” |website = 网易 }}</ref>
* 2013年6月19日，南京市[[江宁区|江宁区]]上元路一超市附近有一个不明信号，经过跟踪监测，发现发射源为停在该超市地面停车场的银灰色别克车。涉案车辆在人群密集的街道小区行驶或停留，伪基站设备自动骗取其覆盖范围内的手机登录伪基站，盗取手机号码和电子串号，然后强行发送垃圾短信。<ref name="伪基站盗手机号推送诈骗信息"/>
* 2013年8月1日，江苏省淮安市无线电管理处执法人员在市区某大厦发现了可疑信号，在民警的配合下，在某广告传媒有限公司内现场查获“伪基站”设备。<ref name="北上广等12省区市发现“伪基站”"/>
* 2013年8月5日，[[中华人民共和国公安部|中华人民共和国公安部]]部署指挥北京、辽宁、湖南、广东等12省区市公安机关集中开展打击“伪基站”行动。<ref name="北上广等12省区市发现“伪基站”"/>
* 2013年11月初，[[深圳市公安局|深圳市公安局]]以涉嫌破坏公用电信设施罪名，刑拘两名在某楼盘群发30余万条广告短信的嫌疑人。同年9月9日，[[福建省|福建省]]三级公安机关刑侦、技侦、网安等部门联合行动，在泉州、厦门控制20名违法犯罪嫌疑人，查获伪基站设备3套、电脑19台，破获泉州腾星信-{}-息技术有限公司利用伪基站非法经营案。<ref>[http://news.xinhuanet.com/newmedia/2013-11/18/c_125718027_4.htm 伪基站潜伏 垃圾短信围困手机]，新华网</ref>
* 2014年1月11日，著名演员[[汤唯|汤唯]]在[[上海|上海]]拍《[[三城记|三城记]]》期间，收到[[電話詐騙|電話詐騙]]的簡訊，被骗21万元人民币。根据媒体报道，剧组所有工作人员都收到声称公安机关发来的短信，短信内容为“欠了一笔钱，不投案要罚款”。有传言称，汤唯收到的短信可能是通过伪基站发送的。<ref>{{cite web |url = http://ent.qq.com/a/20140115/002360.htm |title = 汤唯损失21万 被骗短信内容曝光：不投案要罚款 |website = qq.com |deadurl=yes |archiveurl = https://web.archive.org/web/20140202163623/http://ent.qq.com/a/20140115/002360.htm |archivedate=2014-02-02 }}</ref>
* 2014年3月25日，[[中共中央宣传部|中共中央宣传部]]、[[中央网信办|中央网信办]]等9部门在全国范围内部署开展打击整治专项行动，严打非法生产、销售和使用伪基站设备的违法犯罪活动，专门出台了《关于依法办理非法生产销售使用“伪基站”设-{}-备案件的意见》。<ref>{{cite web |url = http://news.163.com/14/0326/01/9O7QNPTU00014AED.html |title=公安部严打“伪基站”犯罪团伙 |website = 网易 }}</ref>
* 2014年4月11日，[[陕西省|陕西省]][[榆林市|榆林市]][[榆阳区|榆阳区]]人民法院集中公开审理了7起利用“伪基站”群发短消息案，8名被告人一审被判处有期徒刑3年2个月至4年2个月不等。法院审判称，2013年9月到12月期间，武某、冯某等8人在榆阳区利用从外地购得的短信群发设备，向城区手机用户发送广告短信。<ref>[http://hsb.hsw.cn/2014-04/13/content_8533339.htm 榆林首次公审“伪基站”案]，[[华商报|华商报]]</ref>
* 2014年5月21日上午，北京一犯罪嫌疑人张国领，因购买5套伪基站设备，沿三四环群发房产、拍卖等垃圾短信16万条，超过8万中国移动用户受到骚扰，获利3万元人民币，后因破坏公用电信设施罪被通州区法院判处四年有期徒刑。<ref>[http://news.sina.com.cn/c/2014-05-21/124530186435.shtml 北京首例伪基站案宣判 被告流泪走出法庭]，法制晚报</ref>

据[[奇虎360|奇虎360]]公司统计，在全国范围内，河南[[郑州|郑州]]的伪基站最多，其次是北京。<ref name="谁发的诈骗短-{}-信"/>

==相关法规==
《[[中华人民共和国刑法|中华人民共和国刑法]]》第288条和《中华人民共和国治安处罚法》第28条分别规定，擅自使用[[无线电台|无线电台]]，经无线电管理部门责令停止使用后拒不停止使用的，或故意干扰无线电业务经有关部门指出后拒不采取有效措施消除的，才能移送司法部门处罚。

2014年4月4日，[[国家工商总局|国家工商总局]]公布《禁止生产销售使用窃听窃照专用器材和“伪基站”设-{}-备的规定(征求意见稿)》，规定对经公安机关认定的生产、销售窃听窃照专用器材、“伪基站”设备行为，由质量监督部门责令停止生产、销售，处以3万元人民币以下罚款。<ref>{{cite web |url = http://tech.sina.com.cn/t/2014-04-04/17079300179.shtml |title = 中国拟规定非法制售伪基站设备可处3万以下罚款 |website = sina.com.cn }}</ref>
