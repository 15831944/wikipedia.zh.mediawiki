'''蘿桑尼·「蘿西」·哈特'''（'''Roxanne "Roxie" Hart'''）是一名在1926年舞台劇《{{link-en|芝加哥 (舞台劇)|Chicago (play)|芝加哥}}》中的虛構角色。劇作家、記者{{link-en|莫琳·達拉斯·瓦特金斯|Maurine Dallas Watkins}}受到真實事件的1924年{{link-en|貝拉·阿南|Beulah Annan}}和{{link-en|碧娃·加特納|Belva Gaertner}}的謀殺罪審判而啟發，瓦特金斯為《[[芝加哥論壇報|芝加哥論壇報]]》報導她們（兩位女人都被判無罪）。蘿西·哈特是以貝拉·阿南作基礎而成的。<ref>{{Cite web |url = http://www.clydefitchreport.com/2010/10/the-girls-of-murder-city-the-women-who-inspired-roxie-hart-and-velma-kelly/ |title = The Women Who Inspired Roxie Hart and Velma Kelly |author = Leonard Jacobs |date = 2010-10-21 |publisher = Clyde Fitch Report |language = en}}</ref>

該舞台劇於1926年在[[百老匯劇院|百老匯]]表演，並且演出了172場。碧娃·加特納有份參加《芝加哥》的開幕禮。舞台劇的改編作品包括：
*《[[芝加哥_(1927年電影)|芝加哥]]》，1927年的無聲電影，{{link-en|菲利斯·赫佛|Phyllis Haver}}飾演蘿西
*《{{link-en|艷姬逢春|Roxie Hart (film)}}》，1942年的電影，[[琴吉·羅傑斯|琴吉·羅傑斯]]飾演蘿西
*《[[芝加哥_(音樂劇)|芝加哥]]》，1975年的百老匯[[音樂劇|音樂劇]]，{{link-en|格溫·沃頓|Gwen Verdon}}飾演蘿西
*《[[芝加哥_(電影)|芝加哥]]》，2002年[[奧斯卡金像獎|奧斯卡金像獎]]得獎電影，改編1975年音樂劇，[[雲妮·絲維嘉|雲妮·絲維嘉]]飾演蘿西<ref>{{Cite web |url = https://www.theguardian.com/stage/gallery/2009/jul/14/hart-roxie-musical-chicago?picture=350259025 |title = Hart to Hart: a lineup of Roxies in the musical Chicago |date = 2009-07-15 |publisher = guardian |language = en}}</ref>，[[嘉芙蓮·薛達-鍾斯|嘉芙蓮·薛達-鍾斯]]飾演敵對的{{link-en|薇瑪·姬利|Velma Kelly}}，[[李察·基爾|李察·基爾]]飾演律師{{link-en|比利·弗林 (芝加哥)|Billy Flynn (Chicago)|比利·弗林}}
