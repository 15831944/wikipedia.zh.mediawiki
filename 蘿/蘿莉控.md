== 名稱與起源 ==
「{{lang|ja|ロリータ・コンプレックス}}」（Lolita Complex的日語）一詞，日文中一般會以「{{lang|ja|ロリコン}}」為其簡稱；在中文语境中，則以[[音譯|音譯]]而來的「蘿莉控」為主要的表記方式來使用著。而[[拉丁字母|拉丁字母]]使用地區多以「lolicon」表記。

== 相關歷史==
這個辭語在日本最遲是在1970年代開始被廣泛使用；{{lang|ja|[[吾妻ひでお|吾妻ひでお]]}}等人的漫畫世界中，一部分的媒體似乎已經廣泛的流行著這種用法。從1980年代開始，「{{lang|ja|ロリコン}}」這個詞彙流行起來，在[[内山亞紀|内山亞紀]]的漫畫「{{lang|ja|ロリコン・ラブ}}」（《蘿莉控的愛》，1983）的影響之下，「{{lang|ja|体験告白 僕のロリコン・ラブ}}」系列（《體驗告白：我的蘿莉控愛》，1983）等等作品隨之出版；而同時期「{{lang|ja|ロリコン大全集}}」（1983）、「{{lang|ja|ロリコン白書}}」（1982）等等的書籍也問世了。此外，[[日活|日活]]（原「[[日本活動寫真株式會社|日本活動寫真株式會社]]」，在改組為日活時曾有段時間專出成人電影）曾以當時的招牌「{{lang|ja|にっかつロマンポルノ}}」成人電影系列作品，在1983年製作了名為「{{lang|ja|ロリコンハウス おしめりジュンコ}}」（青木琴美主演）的作品。

因媒體大幅報導這樣的觀點，社會大眾傾向產生「蘿莉控」是「具有問題的人格」的刻板印象。對於[[文藝創作|文藝創作]]的法律，也屢次因為受到這樣的觀點影響而制定。另一方面，也有駁斥這種說法的聲音存在，認為這些廣泛流傳於社會大眾的觀點，僅僅是偏頗的意見。

== 觀點 ==
[[File:YuuseiShonenPanel.jpg|thumb]]國際上，歐美國家對戀童行為是持擔憂態度的。但[[精神科|精神科]]醫師及[[犯罪心理學|犯罪心理學]]專家認為，男性擁有這樣的喜好是一種在正常范畴中的情形，不代表他們真的就與女童有性慾望或性關係。

日本的法律認為，[[漫畫|漫畫]]、[[動畫|動畫]]及[[游戏|游戏]]中有關二次元的蘿莉[[性愛|性愛]]並不違法。然而，也有擔憂這類出版物會鼓勵兒童賣淫或增多性侵行為的聲音。日本[[爱知县|爱知县]]有位外地来的大叔向一小学女学生问路，就被小女孩当成变态并且报警，这种情况被媒体称作“萝莉控恐惧症”<ref>[http://www.ettoday.net/news/20121212/138851.htm 「蘿莉控恐懼症」蔓延　日本阿宅：這世界好危險！] [[东森新闻云|东森新闻云]] 2012年12月12日</ref>，這是過度輸入觀點所造成的，孩童的家長必要時提醒，但過度介入管理會產生許多問題。
