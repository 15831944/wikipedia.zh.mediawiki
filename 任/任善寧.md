'''任善寧'''（{{lang-en|'''Yum Sin-ling'''}}，{{bd|1948年|11月23日}}），[[香港|香港]]殖民地年代政治人物，祖籍[[廣東|廣東]][[東莞|東莞]]，畢業於[[德明中學|德明中學]]，臺灣省立成功大學(現稱[[國立成功大學|國立成功大學]])電機工程學系工學學士(1970年畢業)及[[香港中文大學|香港中文大學]]工商管理碩士<ref>http://www.alumni.cuhk.edu.hk/magazine/dec95/p3-8/yum.html</ref>。曾創立[[一二三民主聯盟|一二三民主聯盟]]，並且擔任[[立法局|立法局]]議員及中華民國僑務顧問<ref>http://h9902485.mysinablog.com/index.php?op=ViewArticle&articleId=205435</ref><ref>http://www%20.epochtimes.com/b5/9/10/17/n2691931p.htm{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，現為美中三民主義大同盟秘書長。<ref>{{cite web |url=http://www.worldjournal.com/view/full_news/6868744/article-%E7%B4%80%E5%BF%B5%E5%AD%AB%E4%B8%AD%E5%B1%B1-%E8%8A%9D%E5%9F%8E23%E5%83%91%E5%9C%98%E7%8D%BB%E8%8A%B1?instance=il |title=存档副本 |accessdate=2010-04-07 |deadurl=yes |archiveurl=https://archive.is/20130428154821/http://www.worldjournal.com/view/full_news/6868744/article-%E7%B4%80%E5%BF%B5%E5%AD%AB%E4%B8%AD%E5%B1%B1-%E8%8A%9D%E5%9F%8E23%E5%83%91%E5%9C%98%E7%8D%BB%E8%8A%B1?instance=il |archivedate=2013-04-28 }}</ref>

== 政治生涯 ==
任善寧[[1994年香港區議會選舉|1994年]]當選[[九龍城區議會|九龍城區議員]]，在[[1995年香港立法局選舉|1995年]]的區議員互選中當選[[香港立法局|香港立法局]]議員。任內，他曾為立法局文康廣播事務委員會主席。

但由於立法局直通車問題，他於[[香港主權移交|香港主權移交]]後會被迫「下車」，未能成為[[臨時立法會|臨時立法會]]成員。任領導之一二三民主聯盟曾參加了以[[泛民主派|民主派]]為主導的「反對臨時立法會大聯盟」。1996年7月1日，任善寧和另外幾名立法局議員及區議員赴北京抗議，對中央政府縮短任期和成立臨時立法會表達不滿。在抗議者中，因任善寧曾就讀於台灣，操一口比較流利的普通話，出頭機會較多，自然而然被北京視為抗議活動的領導者；為此，任的[[回鄉證|回鄉證]]被吊銷了八年。

1997年7月1日香港主權移交午夜，任聯同一班民主派立法會議員爬上[[立法會大樓_(香港)|立法局大樓]]露台，誓言「我一定會回來」。不過，任於主權移交後曾表示，「一二三民主聯盟」一切依法行事，不會做任何違反特區法律的事情。<ref>http://www.china.com.cn/overseas/zhuanti/xgtwgx/2007-07/06/content_8487540.htm</ref>他於1998年連同[[朱祖恩|朱祖恩]]、[[麥業成|麥業成]]、[[宋景輝|宋景輝]]組成名單，參與[[1998年香港立法會選舉|1998年香港立法會選舉新界西直選]]落敗，自此退出香港政壇。<ref>http://www.info.gov.hk/gia/general/199805/25/0525203.htm</ref>

因為擔心香港巨變，他在1999年移居[[美國|美國]][[芝加哥|芝加哥]]，後來任之母親去世，他要回深圳料理安葬事宜，才重新取得回鄉證。<ref>{{cite web |url=http://politics.dwnews.com/news/2009-03-04/4770704.html |title=存档副本 |accessdate=2010-04-07 |deadurl=yes |archiveurl=https://archive.is/20120715002154/http://politics.dwnews.com/news/2009-03-04/4770704.html |archivedate=2012-07-15 }}</ref>

== 退出香港政壇後 ==
至今，任仍然致力關心台灣政局，於親台團體工作，又曾擔任及成大校友會會長<ref>http://www.epochtimes.com/b5/5/11/15/n1121225p.htm</ref>，也有關心香港政治。他曾直指「香港民主發展如逆水行舟，不進反退」。<ref>{{cite web |url=http://www.ocac.gov.tw/unit/unit_pop_oversea.asp?no=32107&nodeno=3305 |title=存档副本 |accessdate=2010-04-12 |deadurl=yes |archiveurl=https://archive.is/20120805223956/http://www.ocac.gov.tw/unit/unit_pop_oversea.asp?no=32107&nodeno=3305 |archivedate=2012-08-05 }}</ref>又指中共在香港的「一國兩制」其實是「政經分離」，經濟上是百分之百的兩個制度，但在政治上偏向「一國」，只有經濟是「兩制」。<ref>{{cite web |url=http://38.101.236.102/b5/8/4/16/n2084106.htm |title=存档副本 |accessdate=2010-04-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20100513065957/http://38.101.236.102/b5/8/4/16/n2084106.htm |archivedate=2010-05-13 }}</ref>

== 家庭 ==
其子為網名「馬草泥」的[[任亮憲|任亮憲]]，是一名信託基金公司的聯席董事。近年其於[[維多利亞公園|維園]]之出位行為，被人稱為「[[維園阿哥|維園阿哥]]」。

「馬草泥」曾表示，於2010年，其父（任善寧）起初不支持[[公民黨_(香港)|公民黨]]及[[社民連|社民連]]所發動之[[五區總辭|五區總辭]]，但被自己游說後也變得支持。連同任自身於主權移交前之政治立場，任善寧也算是泛民一員。
