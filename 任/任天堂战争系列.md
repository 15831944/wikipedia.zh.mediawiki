{{vgsname|任天堂战争|ja=ファミコンウォーズ，Famicom Wars|en=Advance Wars|diff=yes|歐美版稱為「'''高级战争'''」}}是军事[[回合制战略游戏|回合制战略游戏]]，主要由[[Intelligent_Systems|Intelligent Systems]]开发，[[任天堂|任天堂]]发行。系列1988年8月12日在日本首发。和[[火焰之纹章|火焰之纹章]]等其他Intelligent Systems系列类似，早期作品僅在日本发行；《[[高级战争|高级战争]]》（2001）是首款在美欧市场发行的作品。《高级战争》2001年9月11日在美国发行，但因为[[911恐怖袭击|911恐怖袭击]]而推延在日本和欧洲发行<ref>{{cite web |url=http://www.eurogamer.net/article.php?article_id=2393 |title=Advance Wars Review - Page 1 // GBA /// Eurogamer - Games Reviews, News and More |publisher=www.eurogamer.net |accessdate=2008-08-08 }}</ref>。尽管欧洲版在2002年1月，但日本版直到2004年11月25日才以《Game Boy战争Advance 1+2》名义发行 Game Boy Advance作品。

2005年《[[任天堂战争DS|任天堂战争DS]]》在[[任天堂DS|任天堂DS]]平台发行，其沿用了前作的基本形式。同年的《[[突击！！任天堂战争|突击！！任天堂战争]]》由[[Kuju_Entertainment|Kuju Entertainment]]开发，发行于[[任天堂GameCube|任天堂GameCube]]，被视作主系列的衍生作品。此作为[[三维计算机图形|3D]]动作即时战略游戏，并非主系列传统的回合制战略。

== 游戏 ==

; {{Vgname|任天堂战争|ja=ファミコンウォーズ}} : 1988年发行于日本[[红白机|红白机]]。2009年再版于[[Wii|Wii]] [[Virtual_Console|Virtual Console]]，2012年再版于[[任天堂3DS|任天堂3DS]] Virtual Console。
; {{Vgname|Game Boy战争|ja=ゲームボーイウォーズ}} : 1991年发行于[[Game_Boy|Game Boy]]。
; {{Vgname|Game Boy战争Turbo|ja=ゲームボーイウォーズTURBO}} : 1997年发行于Game Boy。[[Hudson_Soft|Hudson]]制作发行。
; {{Vgname|超级任天堂战争|ja=スーパーファミコンウォーズ|又翻译为“超级任天堂大战”}} : 1998年发行于[[超级任天堂|超级任天堂]]。2010年再版于Wii Virtual Console，2013年再版于[[Wii_U|Wii U]] Virtual Console。
; {{Vgname|Game Boy战争2|ja=ゲームボーイウォーズ2}} : 1998年行于[[Game_Boy_Color|Game Boy Color]]。[[Hudson_Soft|Hudson]]制作发行。
; {{Vgname|Game Boy战争3|ja=ゲームボーイウォーズ3}} : 2001年发行于Game Boy Color。
; {{Vgname|[[高級戰爭|高級戰爭]]|en=Advance Wars}} : 2001年发行于北美和澳洲的[[Game_Boy_Advance|Game Boy Advance]]，2002年发行欧洲版。2014年再版于Wii U Virtual Console。
; {{Vgname|高級戰爭2：黑洞堀起|en=Advance Wars 2: Black Hole Rising}} : 2003年在北美、澳洲和欧洲Game Boy Advance发行。
; {{Vgname|Game Boy战争Advance|ja=ゲームボーイウォーズアドバンス1+2}} : 2004年11月25日在日本Game Boy Advance发行，上两部游戏的合辑。2014年再版于Wii U Virtual Console。
; {{Vgname|高级战争DS|ja=ファミコンウォーズDS|en=Advance Wars: Dual Strike|diff=yes}} : 2005年在全球[[任天堂DS|任天堂DS]]发行。
; {{Vgname|突击！！任天堂战争|ja=突撃!!ファミコンウォーズ|en=Battalion Wars|diff=yes}} : 2005年在全球[[任天堂GameCube|任天堂GameCube]]发行。
; {{Vgname|突击！！任天堂战争VS|ja=突撃!!ファミコンウォーズVS|en=Battalion Wars 2|diff=yes}} : 2007年在北美[[Wii|Wii]]发行，2008年在日本、欧洲和澳洲发行。
; {{Vgname|高級戰爭：毁灭之日|ja=ファミコンウォーズDS 失われた光|na=Advance Wars: Days of Ruin|eu=Advance Wars: Dark Conflict}} : 2008年在西方发行[[任天堂DS|任天堂DS]]发行，日版原定发行但取消，后于2013年通过[[任天堂eShop|任天堂eShop]]为任天堂3DS下载发行。
