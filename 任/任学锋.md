'''任学峰'''（{{bd|1965年|10月||catIdx=R任}}），[[河北省|河北]][[邢台|邢台]]人。男，汉族。1988年12月参加工作，1985年加入[[中国共产党|中国共产党]]。[[河北工业大学|河北工学院]]（现[[河北工业大学|河北工业大学]]）化学工程系有机化工专业毕业，高分子材料专业硕士，[[南开大学|南开大学]]国际商学院企业管理专业博士。[[中国共产党第十八届中央委员会候补委员列表|中共第十八]]、[[中国共产党第十九届中央委员会候补委员列表|十九届中央候补委员]]<ref>{{cite web|url=http://news.xinhuanet.com/18cpcnc/2012-11/14/c_113690803.htm|title=中国共产党第十八届中央委员会候补委员名单|accessdate=2012-11-15|author=|date=2012-11-14|publisher=新华网}}</ref>。

1988年从[[河北工业大学|河北工学院]]毕业后留校工作，先后任河北工学院团委副书记、团委书记。1992年起，先后任[[天津滨海高新技术产业开发区|天津新技术产业园区]]管委会业务处工程师、开发总公司工业投资部副部长、公司副总经理。1995年，任天津新技术产业园区开发总公司总经理。2002年，任天津新技术产业园区管委会副主任。2004年起，先后任[[香港津联集团有限公司|香港津联集团有限公司]]党委书记、副董事长、董事长。2008年1月，任[[天津市人民政府|天津市人民政府]]副市长。

2014年8月，任[[中国共产党广东省委员会|中共广东省委]]常委、广州市委书记<ref>{{cite web|url=http://news.qq.com/a/20140827/041037.htm?tu_biz=1.114.1.0|title=任学锋出任广东省委常委、广州市委书记|accessdate=2014-08-27|author=|date=2014-08-27|publisher=腾讯网}}</ref>。2017年2月，任[[中国共产党广东省委员会|中共广东省委]]副书记、[[中国共产党广州市委员会|中共广州市委]]书记<ref>{{cite web|url=http://www.thepaper.cn/newsDetail_forward_1622918|title=任学锋任广东省委副书记，邹铭任省委组织部部长|accessdate=2017-02-20|author=|date=2017-02-20|publisher=澎湃新闻}}</ref>。

2018年7月，任学锋不再担任中共广州市委书记职务，仍任中共广东省委副书记<ref>{{cite web |author1=新华网 |title=张硕辅、任学锋同志职务任免 |publisher=南方网 |accessdate=2018-07-23 |url=http://news.southcn.com/rsdd/content/2018-07/20/content_182659175.htm}}</ref>。同年10月17日，调任[[中共重庆市委|中共重庆市委]]委员、常委、副书记<ref>{{cite web |author1=华龙网  |title=任学锋任重庆市委委员、常委、副书记|publisher=新华网 |accessdate=2018-10-18 |url=http://www.xinhuanet.com/renshi/2018-10/17/c_1123574030.htm}}</ref>。
