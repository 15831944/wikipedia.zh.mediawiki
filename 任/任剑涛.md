'''任剑涛'''（{{bd|1962年|8月|||R任}}）男，[[四川省|四川省]][[广元市|广元市]][[苍溪县|苍溪县]]人<ref name="中国人民大学政治学系教授：任剑涛">{{cite web | url=http://edu.qq.com/a/20110503/000035.htm | title=中国人民大学政治学系教授：任剑涛 | accessdate=2011-05-04}}</ref>，曾任[[中山大学|中山大学]]政治与公共事务管理学院院长、教授、博士生导师；美国哈佛大学燕京学社访问学者。2009年9月起<ref name="任剑涛：以法治国与中华文明的自我突破">{{cite web | url=http://theory.gmw.cn/2014-12/22/content_14254494.htm  | title=任剑涛：以法治国与中华文明的自我突破 | accessdate=2014-12-22}}</ref>调入[[中国人民大学|中国人民大学]]，曾任中国人民大学国际关系学院政治学系二级教授、博士生导师<ref name="任剑涛">{{cite web | url=http://economy.caixin.com/2011/9393/100323792/ | title=任剑涛 | accessdate=2012-01-08}}</ref>。现为[[清华大学|清华大学]]社会科学学院政治学系教授，教育部“长江学者”特聘教授<ref name="清华任剑涛：潘恩如何将茶杯里的风波演化成浩大的革命？（中）">{{cite web | url=http://news.ifeng.com/a/20170816/51653623_0.shtml | title=清华任剑涛：潘恩如何将茶杯里的风波演化成浩大的革命？（中） | accessdate=2017-08-16}}</ref>。其主要研究领域为政治哲学、西方政治思想史、中国政治以及当代中国政府与政治研究。代表作：《政治哲学讲演录》《伦理政治研究》《从自在到自觉——中国国民性研究》《中国现代思想脉络中的自由主义》等<ref name="清华任剑涛：潘恩如何将茶杯里的风波演化成浩大的革命？（中）">{{cite web | url=http://news.ifeng.com/a/20170816/51653623_0.shtml | title=清华任剑涛：潘恩如何将茶杯里的风波演化成浩大的革命？（中） | accessdate=2017-08-16}}</ref>。

==人物生平==

*1962年8月生于四川苍溪； 
*1982年本科毕业于南充师范学院（现易名为西华师范大学）；
*1996年获得中山大学哲学博士学位<ref name="任剑涛：以法治国与中华文明的自我突破">{{cite web | url=http://theory.gmw.cn/2014-12/22/content_14254494.htm  | title=任剑涛：以法治国与中华文明的自我突破 | accessdate=2014-12-22}}</ref>；
*1998年11月－1999年11月哈佛大学燕京学社高级访问学者；
*2005年荣获第五届“广东省十大杰出青年”称号<ref name="中国人民大学教授任剑涛——从城市文化入手定位深圳城市品格">{{cite web| url=http://sztqb.sznews.com/html/2010-08/25/content_1206608.htm| title=中国人民大学教授任剑涛——从城市文化入手定位深圳城市品格| accessdate=2010-08-25| deadurl=yes| archiveurl=https://web.archive.org/web/20100906144515/http://sztqb.sznews.com/html/2010-08/25/content_1206608.htm| archivedate=2010-09-06}}</ref> <ref name="任剑涛：政府体制改革不是单纯的行政改革">{{cite web| url=http://www.chinareform.net/special_detail.php?id=359| title=任剑涛：政府体制改革不是单纯的行政改革| accessdate=2011-08-26| deadurl=yes| archiveurl=https://web.archive.org/web/20150203120017/http://www.chinareform.net/special_detail.php?id=359| archivedate=2015-02-03}}</ref>；
*2010年10－12月台湾大学访问研究员；
*2009年9月起<ref name="任剑涛：以法治国与中华文明的自我突破">{{cite web | url=http://theory.gmw.cn/2014-12/22/content_14254494.htm  | title=任剑涛：以法治国与中华文明的自我突破 | accessdate=2014-12-22}}</ref>调入[[中国人民大学|中国人民大学]]，现任中国人民大学国际关系学院政治学系二级教授、博士生导师。

== 2012年瑞士信贷银行中国投资研讨会上的演讲 ==

2012年，任剑涛在瑞士信贷银行中国投资研讨会上公开发表演讲，瑞信中国投资研讨会上，认为，30年来中国经济将没有预警性崩溃，权势集团挟持国家，政策调试变成利益集团的妥协结果，期望体制改革不可能<ref name="大陆网站全面封杀 人大教授任剑涛政经演讲视频">{{cite web | url=http://www.aboluowang.com/2012/0814/256836.html  | title=大陆网站全面封杀 人大教授任剑涛政经演讲视频 | accessdate=2012-08-14 }}</ref>。这一言论随后遭到中国大陆网站全面封杀<ref name="大陆网站全面封杀 人大教授任剑涛政经演讲视频">{{cite web | url=http://www.aboluowang.com/2012/0814/256836.html  | title=大陆网站全面封杀 人大教授任剑涛政经演讲视频 | accessdate=2012-08-14 }}</ref>。
