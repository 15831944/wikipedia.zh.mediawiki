'''任創均'''<ref>{{Cite web |url =http://ww1.sinaimg.cn/mw690/73217002jw1f5if9t196uj20sg0zv10k.jpg |title =MONSTA X 全員確認後的正名 | date = 2016-07-05}}</ref><ref>{{Cite web |url =http://www.weibo.com/1931571202/DDjAI4YdQ?type=comment#_rnd1481452395256 | title =MONSTA X 全員確認後的正名:詳細全文 | date = 2016-07-05}}</ref>（{{kor|k='''임창균'''|rm=}}，{{bd|1996年|1月26日|}}），藝名為'''I.M'''（{{kor|k='''아이엠'''|rm=}}），是一名韓國男歌手、舞者。為[[MONSTA_X|MONSTA X]]七位成員之一，擔任Rapper，是隊內的忙內和英文擔當。任創均曾是Nubility成員。
