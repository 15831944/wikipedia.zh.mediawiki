'''抗磁性'''（Diamagnetism，亦作'''反磁性'''）是一些類別的[[物質|物質]]，當處在外加[[磁場|磁場]]中，會對磁場產生的微弱[[斥力|斥力]]的一種[[磁性|磁性]]現象。

== 起因 ==
抗磁性的成因，是當物質處在外加磁場中，外加磁場使得物質[[電子|電子]][[軌道|軌道]]（更精確的講法：[[軌域|軌域]]）運動產生改變的連帶效應。當施加一外源磁場'''B'''時，會對運動中的電子（電荷''q''）產生了磁力'''F'''：'''F''' = ''q'''''v''' × '''B'''。此力改變了電子所受的[[向心力|向心力]]，使得電子軌道運動或是加速，或是減慢。電子速度因此受到改變，而連帶改變了其與外加磁場相反方向上的軌道[[磁矩|磁矩]]。

考慮兩個電子軌域：一個順時針運動，一個逆時針運動。一進入頁面方向的外加磁場會使順時針轉動電子的[[向心力|向心力]]增加，而使其自頁面出來方向上的[[磁矩|磁矩]]增加。同樣的外加磁場則會使[[逆時針|逆時針]]轉動電子的向心力減少，而使其進入頁面方向上的磁矩減少。兩者的改變都與進入頁面方向的外加磁場相抗衡。然而，外加磁場對於多數日常物質所感生的磁矩卻非常小，因此淨效應會是一種斥力。

所有物质都会对外加磁场作出不同程度的抗磁性反应；但是对于同时拥有其他磁性性质的材料来说（如[[铁磁性|铁磁性]]和[[顺磁性|顺磁性]]），抗磁性可以完全忽略不计。那些仅仅或者很大程度显示抗磁性的物质被称之为抗磁性材料或者抗磁性子。那些被认为具有抗磁性的材料通常被非物理学家作为非磁性物质看待。它们包括[[水|水]]，[[DNA|DNA]]，绝大多数[[有机化合物|有机化合物]]如[[石油|石油]]和一些塑料，和[[金属|金属]]如[[水银|水银]]（元素），[[金|金]]和[[铋|铋]]。

== 理论解释 ==
尽管物质抗磁性本质上是量子效应，但通过纯经典的朗之万抗磁性理论可以获得一致的解释。

=== 朗之万抗磁性理论 ===
朗之万抗磁性理论可用于解释闭壳层原子构成的物质的抗磁性。强度<math>B</math>的磁场作用在电荷量为<math>e</math>质量为<math>m</math>的电子上，电子受洛伦兹力作用将进行频率为<math>\omega = e B / 2 m </math>的[[拉莫爾進動|拉莫爾進動]]。单位时间内转动速度为<math>\omega / 2\pi</math>，含<math>Z</math>个电子的原子所产生的环状电流为（採用[[国际单位制|国际单位制]]）

:<math> I = -\frac{Ze^2B}{4 \pi m}.</math>

环状电流产生的磁矩等于电流强度与闭合环包含的面积。假定外场沿<math>z</math>轴方向。平均的环内面积为<math>\pi\langle\rho^2\rangle</math>，其中<math>\langle\rho^2\rangle</math>为电子到<math>z</math>轴的均方距离。可知[[磁矩|磁矩]]为

:<math> \mu = -\frac{Ze^2B}{4 m}\langle\rho^2\rangle.</math>

若电荷分布呈球对称，可设<math>\langle x^2\rangle=\langle y^2\rangle=\langle z^2\rangle=\langle r^2\rangle/3</math>，其中<math>\langle r^2\rangle</math>为电子到核的均方距离。则<math>\langle\rho^2\rangle=\langle x^2\rangle+\langle y^2\rangle=(2/3)\langle r^2\rangle</math>。
若<math>N</math>为单位体积原子数，抗磁性[[磁化率|磁化率]]为

:<math>\chi = \frac{\mu_0 N \mu}{B} = -\frac{\mu_0 N Z e^2}{6 m}\langle r^2\rangle.</math>

=== 金属中的抗磁性 ===

因为金属中含非定域电子，朗之万理论不适用于金属。自由电子气的抗磁性理论被称为朗道抗磁性理论，它包含了洛伦兹力下电子轨线被曲化后形成的弱反作用场。朗道抗磁性应当与泡利顺磁性相区别，后者与非定域电子自旋的极化相关联。

== 歷史 ==
虽然早在1778年，[[S._J._Brugmans|S. J. Brugmans]]就发现了金属[[铋|铋]]和金属[[锑|锑]]在磁场中存在某些抗磁性现象。但是直到1845年9月，[[迈克尔·法拉第|迈克尔·法拉第]]发现在外在施加磁场中，所有天然物质拥有不同程度的抗磁性，抗磁性这个词才正式在文献中使用。
