'''抗美援朝纪念馆'''，位于中國[[辽宁省|辽宁省]][[丹东市|丹东市]]，建于1958年，其前身是1953年建立的'''辽东省地质博物馆筹备处'''。该馆除陈列[[抗美援朝|抗美援朝]][[志愿军|志愿军]]的一些列历史文物外，还有一些其他时期的历史文物。

== 紀念館設置 ==
紀念館包含抗美援朝戰爭館，抗美援朝運動館，中朝人民友誼館和英雄模範烈士館。另建有抗美援朝紀念塔。塔高53米，象徵1953年[[朝鮮停戰協定|朝鮮停戰協定]]簽字，[[抗美援朝戰爭|抗美援朝戰爭]]結束。紀念館的露天兵器陳列場陳列有中國人民志願軍所用部分武器，及志願軍在戰鬥中繳獲的部分武器。

2008年5月16日，由[[國家文物局|國家文物局]]列入[[國家一級博物館|國家一級博物館]]。<ref>{{Cite web |url = http://www.sach.gov.cn/art/2008/6/20/art_1329_97233.html |title = 关于发布首批国家一级博物馆名单的决定（文物博发〔2008〕35号） |author =  |date =2008-06-20  |publisher =  国家文物局}}</ref>2013年5月，降為[[國家二級博物館|國家二級博物館]]。<ref>{{Cite web |url = http://www.chinanews.com/cj/2013/05-15/4820002.shtml |title =北京天文馆被降为国家二级博物馆  |author =  |date =2013-05-15  |publisher =  中国新闻网}}</ref>

该馆2014年開始扩建工程而關閉。截至2018年4月為止，该馆尚未重新開放。<ref>{{cite news | url=http://chinanews.sina.com/gb/2018-04-24/doc-ixkifkew7730032.shtml | title=抗美援朝纪念馆长被查：3.5亿改扩建 至今未开馆 |  work=凤凰网}}</ref>

== 志愿军烈士名单 ==
馆中收藏了完整的[[中國人民志愿军|中國人民志愿军]]烈士名单，共183,108人<ref>[http://www.chinanews.com.cn/news/2006/2006-04-28/8/723736.shtml 抗美援朝纪念馆推新数字 志愿军烈士名单增万人] 华商晨报 2006年04月28日</ref>。
