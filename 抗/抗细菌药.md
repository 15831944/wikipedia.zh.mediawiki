'''抗细菌药'''（{{lang-en|antibacterial}}）也称为“'''抗细菌剂'''”，是一类用于抑制细菌生长或杀死细菌的药物。<ref name="urlDorlands Medical Dictionary:antibacterial">{{cite web |url=http://www.mercksource.com/pp/us/cns/cns_hl_dorlands_split.jsp?pg=/ppdocs/us/common/dorlands/dorland/one/000005889.htm |title=Dorlands Medical Dictionary:antibacterial |work= |accessdate=2010-10-29 |archiveurl=https://www.webcitation.org/5uJysabSO?url=http://www.mercksource.com/pp/us/cns/cns_hl_dorlands_split.jsp?pg=%2Fppdocs%2Fus%2Fcommon%2Fdorlands%2Fdorland%2Fone%2F000005889.htm |archivedate=2010-11-18 |deadurl=no }}</ref>在不引起歧义的情况下，抗细菌药也可简称为“'''抗菌药'''”，包括'''抗生素'''（{{lang-en|antibiotic}}） 由微生物（包括细菌、真菌、放线菌属）所产生的具有抑制其它类微生物生长、生存的一类[[次级代谢产物|次级代谢产物]]，以及用化学方法合成或半合成的类似化合物。在定義上是一較廣的概念，包括抗细菌抗生素、抗真菌抗生素以及對付其他微小病原之抗生素；但臨床實務中，抗生素常常是指抗細菌抗生素。抗生素的副作用之一是使肠道正常菌群失调。<ref>{{Cite journal|title=Uncovering effects of antibiotics on the host and microbiota using transkingdom gene networks.|author=Morgun A|url=https://www.researchgate.net/profile/Andrey_Morgun/publication/271213616_Uncovering_effects_of_antibiotics_on_the_host_and_microbiota_using_transkingdom_gene_networks/links/554d5a7f08ae3e860c2975b5.pdf?inViewer=0&pdfJsDownload=0&origin=publication_detail|journal=Gut|issue=|doi=|others=|year=2015|volume=|page=|pmid=25614621}}</ref>

== 用途 ==
* 抗细菌药最主要用於醫療方面，可以用來殺死細菌。
* 對抗在人或動物體內的致幹菌等病原體，可治療大多數[[細菌|細菌]]、[[立克次體|立克次體]]、[[支原體|支原體]]、[[衣原體|衣原體]]、[[螺旋體|螺旋體]]等微生物感染導致的[[疾病|疾病]]。
* 對於[[病毒|病毒]]、[[朊毒體|朊毒體]]等結構簡單的病原體所引起的[[疾病|疾病]]沒有效用。
* 除了抗細菌性的[[感染|感染]]外，某些抗细菌药還具有抗[[腫瘤|腫瘤]]活性，用於腫瘤的[[化學治療|化學治療]]。
* 有些抗细菌药還具有[[免疫|免疫]]抑制作用。
* 抗细菌药除用於[[醫療|醫療]]，還應用於生物科學研究、[[農業|農業]]、[[畜牧業|畜牧業]]和[[食品工業|食品工業]]等方面。
* 在畜牧業和農業中非治療用途的抗细菌药，稱為[[抗生素生長促進劑|抗生素生長促進劑]]。

== 製造方法 ==
[[File:Pyrazinamide_ball-and-stick.png|缩略图]]的[[吡嗪醯胺|吡嗪醯胺]]化學式]]
抗細菌藥的主要製造方法為[[发酵|发酵]]，也可以通過[[化學合成|化學合成]]和[[半合成|半合成]]的方法製造出來。
先採用適當的菌種，經輻射處理，選擇最具潛力的變異菌株，將其從試管轉移到燒瓶再到發酵槽，在溫度與通風控制的環境中培育，然後從培養基中分離、純化、析離成抗生素的結晶。經研判化學結構、藥理實驗與臨床試驗後，才告完成。

== 歷史 ==
人类合成的第一种抗菌药是[[磺胺|磺胺]]，1932～1933年间德国病理与细菌学家[[格哈德·多馬克|格哈德·多馬克]]发现其具有体内抗菌活性，他因此获得1939年[[诺贝尔生理学或医学奖|诺贝尔生理学或医学奖]]。

人类发现的第一种抗生素——[[青黴素|青黴素]]（盤尼西林），是英国微生物学家[[亞歷山大·弗莱明|亞歷山大·弗莱明]]于1928年偶然發現的，但当时并没有提纯出有效成分和分析化学结构。他從被黴菌汙染的[[葡萄球菌|葡萄球菌]]培養皿中，觀察到黴菌附近的細菌都無法生長，推測黴菌中可能有殺菌的物質，1929年，弗萊明將這個發現發表在《英國實驗病理學期刊》，但沒有得到重視。直到1939年，牛津大學的佛洛里（Howard Florey）和柴恩（Ernst Chain）想開發能醫療細菌感染的藥物，才在聯絡弗萊明取得菌株後，成功提纯出青黴素。1939年至1940年酪毛黴素、盤尼西林及放線黴素先後被分離出，開創了抗生素的新紀元。弗萊明、佛洛里與柴恩因此於1945年共同獲得[[诺贝尔生理学和医学奖|诺贝尔生理学和医学奖]]<ref>{{cite journal | url = http://www.kmu.edu.tw/~kmcj/data/9204/10.htm | title = 抗生素與抗藥性 | author = 盧柏樑 | journal = 高醫醫訊 | volume = 22 | issue = 11 | publisher = 高醫醫訊雜誌社 | location = [[高雄市|高雄市]] | date = 2003-04-01}} </ref><ref>{{cite journal | title = 抗生素的研究 | author = 林天送 | date = 2010-08-10 | journal = 科學發展 | issue = 452 | publisher = [[行政院國家科學委員會|行政院國家科學委員會]] | location = [[臺北市|臺北市]] | page = 72-75 | issn = 0250-1651}} </ref>。

== 分类 ==
作为抗菌剂使用的抗生素有以下几个主要类别：
* [[β-内酰胺类抗生素|β-内酰胺类抗生素]] 
** [[青黴素|青黴素]]
** [[头孢菌素|头孢菌素]]
** 非典型的β-内酰胺类抗生素
* [[氨基糖苷类抗生素|氨基糖苷类抗生素]]
* [[大环内酯类抗生素|大环内酯类抗生素]]
* [[四环素类抗生素|四环素类抗生素]]
* [[氯霉素|氯霉素]]（全合成抗生素）

另外，既不是微生物分泌物又不是其类似物的人工全合成抗菌剂有：
* [[喹诺酮|喹诺酮]]
* [[磺胺类药物|磺胺类药物]]
在不严格的情况下，有时候也把这两类抗菌剂并称为“抗生素”。

== 作用机理 ==
抗生素等抗菌剂有抑菌或杀菌作用，但抗生素對微生物的作用具選擇性，依其作用，可分為廣效或是專一。主要是针对“细菌有而人（或其它高等动植物）没有”的机制进行杀伤，有4大类作用机理：
# 阻碍[[细菌|细菌]][[细胞壁|细胞壁]]的合成，导致细菌在低[[渗透压|渗透压]]环境下溶胀破裂死亡，以这种方式作用的抗生素主要是[[β-内酰胺类抗生素|β-内酰胺类抗生素]]。[[哺乳动物|哺乳动物]]的细胞没有细胞壁，不受这类药物的影响。
# 与细菌[[细胞膜|细胞膜]]相互作用，增强细菌细胞膜的[[通透性|通透性]]、打开膜上的[[离子通道|离子通道]]，让细菌内部的有用物质漏出菌体或电解质平衡失调而死。以这种方式作用的抗生素有[[多粘菌素|多粘菌素]]和[[短杆菌肽|短杆菌肽]]等。
# 与细菌[[核糖体|核糖体]]或其反应[[底物|底物]]（如[[tRNA|tRNA]]、[[mRNA|mRNA]]）相互所用，抑制[[蛋白质|蛋白质]]的[[不对称合成|合成]]——这意味着细胞存活所必需的[[结构蛋白|结构蛋白]]和[[酶|酶]]不能被合成。以这种方式作用的抗生素包括[[四环素类抗生素|四环素类抗生素]]、[[大环内酯类抗生素|大环内酯类抗生素]]、[[氨基糖苷类抗生素|氨基糖苷类抗生素]]、[[氯霉素|氯霉素]]等。
# 阻碍细菌[[脫氧核醣核酸|脫氧核醣核酸]]的[[DNA复制|复制]]和[[转录|转录]]，阻碍DNA复制将导致细菌细胞分裂繁殖受阻，阻碍DNA转录成[[mRNA|mRNA]]则导致后续的mRNA[[翻译_(遗传学)|翻译]]合成蛋白的过程受阻。以这种方式作用的主要是人工合成的抗菌剂[[喹诺酮|喹诺酮]]类（如[[氧氟沙星|氧氟沙星]]）。

與細胞壁或細胞膜作用的兩類抗生素，是以破壞菌體完整性的方式殺死細菌，故可稱為殺菌劑（Bactericidal agent）；另外兩類抗生素則是靠抑制細菌大分子合成的方式，阻斷其繁殖，故又可稱之為抑菌劑（Bacteriostatic agent）<ref>{{cite journal |  title = 感染性疾病和抗生素的使用 |  author = 蔡嘉寅 | journal = 科學月刊 | publisher = 科學月刊雜誌社 | date = 1971-1 |volume = 2 | issue = 1 | page = 19-23 |  issn = 0250-331X }}</ref>。
