'''抗台风暴潮殉职同志纪念碑'''，是[[金湾区文物保护单位|金湾区文物保护单位]]之一，位于中国[[广东省|广东省]][[珠海市|珠海市]][[金湾区|金湾区]][[红旗镇_(珠海市)|红旗镇]]。纪念碑建于1980年4月，为纪念在1972年11月[[台风|台风]]和[[风暴潮|风暴潮]]抢险工作中殉职的36名[[知识青年|知识青年]]而建。

== 历史 ==
1972年11月8日晚间（[[UTC+8|UTC+8]]），[[珠江口|珠江口]]以西地区遭受[[颱風柏美娜_(1972年)|強颱風]]侵袭，当时正值[[潮汐|天文大潮]]，出现严重的风灾与水灾情况；珠海市金湾区[[红旗农场_(广东)|红旗农场]]面临建立以来危害最大的天灾，红旗农场军建分场被猛烈的风势正面袭击，水位在警戒线的四米上，道路、耕地被洪水淹没，灾民众多<ref name="珠海市档案局"/><ref name="中山商报"/>。九个连队、共六百多名[[知识青年|知识青年]]（知青）受命投入抢险，负责堆砌堤坝、筑人墙、堵住堤坝缺口、搬运物资等工作<ref name="中山商报"/>。其中36名知青为保护财产和拯救战士，被卷进漩涡急流，遇难殉职；他们的遗体由知青船队运到红旗农场码头，整理遗容，举行近百人出席的追悼会后，安葬于邻近金湾区的珠海[[斗门区|斗门区]]白藤山上<ref name="中山商报"/>。

到了1980年4月，红旗农场（当时已更名为“[[华侨农场|红旗华侨农场]]”）的[[中国共产党|中国共产党]]委员会为了纪念1972年台风期间的殉职党员，一致通过建造纪念碑<ref name="南方都市报"/>。同年，“抗台风暴潮殉职同志纪念碑”在红旗农场广场上建成，以纪念36名殉职知青<ref name="珠海市档案局"/>。

== 结构 ==
抗台风暴潮殉职同志纪念碑的结构分为碑身和台座，其中碑身大约高四米；纪念碑四旁有低矮的水泥围墙，墙面铺上瓷砖，墙体中间有三级台阶，供参访者走到碑前凭吊<ref name="南方都市报"/>。台座正面刻有36名殉职知青的姓名，亦刻有悼词。

== 保护 ==
2012年6月，金湾区政府按照法律公布不可移动文物名录，其中把抗台风暴潮殉职同志纪念碑列为[[金湾区文物保护单位|金湾区文物保护单位]]<ref name="珠海市文化体育旅游局"/>。《[[南方都市报|南方都市报]]》同年报道指：纪念碑年代久远，碑身因风雨侵蚀而出现外观陈旧；水泥围墙上的的瓷砖不少出现开裂、隆起，或被夹缝里的小植物挤裂<ref name="南方都市报"/>。
