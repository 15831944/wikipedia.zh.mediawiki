《'''一九四二'''》（英語：Back to 1942），是一部2012年[[馮小剛|馮小剛]]執導的中国電影，劇本改編自[[劉震雲|劉震雲]][[小說|小說]]《[[溫故一九四二|溫故一九四二]]》。[[華誼兄弟|華誼兄弟]]發行，2012年11月29日於[[中國大陸|中國大陸]]上映<ref>{{cite news|author=sunny|title=《一九四二》宣布过审 定档11月29日(图)|url=http://ent.sina.com.cn/m/c/2012-11-04/21013779549.shtml|accessdate=2012-11-11|newspaper=新浪娱乐|date=2012年11月4日}}</ref>。


== 剧情 ==
影片改編自小說《溫故一九四二》，講述了1942年[[中國抗日戰爭|中國抗日戰爭]]背景下的一場悲劇。1942年[[1942年河南饥荒|河南大饑荒]]，此時戰爭正處於僵持階段，數百萬民眾忍受飢荒的痛苦，展開背井離鄉逃荒的旅程。

== 史实争议 ==
*《[[大公报|大公报]]》被国民政府停刊三日，非电影中所说是报道[[河南大饥荒|河南大饥荒]]受到国民政府的处分，而是因为“《大公报》因指摘限制物价之失败，受停刊三日之处分”。记录于国民政府中央宣传部部长[[王世杰|王世杰]]1943年2月4日日记<ref name=M1942>{{cite web |url=http://view.news.qq.com/zt2012/1943wls/index.htm |title=戳破《1942》里的伪历史 |accessdate=2012年12月14日 |publisher=腾讯网 |archiveurl=https://web.archive.org/web/20130818052744/http://view.news.qq.com/zt2012/1943wls/index.htm |archivedate=2013年8月18日 |work= |language=zh-cn |deadurl=yes }}</ref>。
*国民政府并没有禁止和封杀媒体报纸对河南大饥荒的报道。當時[[重庆|重庆]]《[[新华日报|新华日报]]》（[[中共|中共]]官方报纸）报道此事就有至少40多篇，而且可以发行<ref name=M1942/>。
*电影里称蒋介石“不许报灾”、“不相信河南有灾”，来源是[[冯玉祥|冯玉祥]]1947年的[[回忆录|回忆录]]《[[我所知道的蒋介石|我所知道的蒋介石]]》；后经过查证，李培基根本没去重庆报灾，係冯玉祥捏造、杜撰。第三届[[国民参政员|国民参政员]][[郭仲隗|郭仲隗]]晚年在《江流天地外：郭仲隗、郭海长纪念文集》的回忆录中记载“中央不准报灾，亦不救济”，但河南军方多次报灾，所以亦不符合史实。[[汤恩伯|汤恩伯]]、[[蒋鼎文|蒋鼎文]]、[[李家钰|李家钰]]等均向国民政府报灾，只有[[河南省政府|河南省政府]]瞒报灾情。[[王芸生|王芸生]]、[[曹谷冰|曹谷冰]]合著《1926年至1949年的旧大公报》记载“委员长根本不相信河南有灾”，也不符合史实，因为国民政府派遣了[[张继|张继]]和[[张厉生|张厉生]]等人前往河南调查灾情<ref name=M1942/>。
*电影中，国民政府救灾是迫于[[白修德|白修德]]曝光河南大饥荒事件。事实上，在白修德報導之前，国民政府和河南省政府就已经展开了救灾工作<ref name=M1942/>。
*电影中，蒋介石将灾民遗弃给日本人。事实上就《河南省政府救灾工作总报告》紀錄：「各县应切实督饬所属团队协同国军，在通往地区地点设哨盘查，制止良民逃往敌区；如遇敌军抽集我壮丁难民，应随时随地予以武力驱逐，以粉碎敌寇阴谋。」防止灾民流亡到日治地区。<ref name=M1942/>。
*电影中，冈村宁次下令开仓放粮赈灾。部分地区在1938年已经变成日据地区，灾民是饿死在日军的统治下<ref name=M1942/>。
*[[中央研究院近代史研究所|中央研究院近代史研究所]]研究員[[朱浤源|朱浤源]]以[[國史館|國史館]]的文獻證明，蔣介石親筆電文指揮官員發糧救災。朱浤源批評：「這些劇作家根據千千萬萬的資料裡面，哪些部分我們不知道，但是是以共產黨的宣傳為主，因為整個大陸是中華人民共和國所治理的。」<ref>{{cite web |url=http://news.tvbs.com.tw/entry/209712 |accessdate= 2014年1月13日|publisher=TVBS新聞 |title= 〈獨家〉醜化老蔣沒救災？ 中研院教授批「1942」|date=2013年5月10日 |author=陳乃瑜 |language =zh-tw}}</ref>
*亦有人批评，影片过分渲染人性阴暗面<ref>[http://ent.cn.yahoo.com/ypen/20121214/1490674.html “1942”被批渲染人性恶 冯小刚：太平绅士慎入] {{webarchive|url=https://archive.is/20130105140954/http://ent.cn.yahoo.com/ypen/20121214/1490674.html |date=2013-01-05 }}，yahoo.cn.</ref>。
