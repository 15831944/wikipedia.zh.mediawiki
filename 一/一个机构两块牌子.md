'''一个机构两块牌子'''，又稱'''一套人马两块牌子'''，是中华人民共和国的一种机构编制方式<ref>{{Cite web|url=http://www.china.org.cn/china/leadership/2013-03/11/content_28206251.htm|title=State Bureaus administered by Ministries or Commissions|accessdate=2018-02-03|date=2013-03-11|publisher=中国网|archive-url=https://web.archive.org/web/20180408055057/http://www.china.org.cn/china/leadership/2013-03/11/content_28206251.htm|archive-date=2018-04-08|dead-url=no}}</ref>。按照[[中央机构编制委员会办公室|中央机构编制委员会办公室]]的解释，「所谓一个机构，就是一个[[法人代表|法人代表]]、一个财务帐号、一套领导班子和一个队伍。所谓两块牌子，是指机构有两个名称，根据工作需要，以不同的名义对外使用相应的名称」<ref>{{Cite web|url=http://axsyglj.my.gov.cn/ax/218735752308064256/20080411/180515.html|title=中央编办对事业单位“一个机构两块牌子”含义做出政策解释|accessdate=2018-06-12|date=2008-04-11|publisher=安县事业单位登记管理局|archiveurl=https://web.archive.org/web/20141108100030/http://axsyglj.my.gov.cn/ax/218735752308064256/20080411/180515.html|archivedate=2014-11-08}}</ref>。

一个机构两块牌子的形式主要分为“'''加挂牌子'''”和“'''对外保留牌子'''”两种，相应被撤并的机构分别称为“'''挂牌机构'''”和“'''保留牌子机构'''”<ref name="报道1230A"/><ref name="释义"/>。当一个机构被撤销并转为挂牌机构（保留牌子机构）时，该机构不再是法律上的独立实体机构，仅是在特定条件下可以单独使用“牌子”所指称的名义<ref name="报道1230A"/>。因此，被撤并机构在[[中华人民共和国民法总则|民法]]上的独立机关法人地位、在行政法上的独立行政主体地位彻底丧失，其法律人格被相应的职能部门所吸收<ref name="报道1230A"/>。

== 加挂牌子 ==
'''加挂牌子'''，指一个机构承担多项职责，因工作需要加挂牌子，有两个或者两个以上名称，在不同场合以相应名称对外开展工作的机构编制方式。被加挂牌子的机构称为“挂牌机构”<ref name="释义"/>。挂牌机构往往是以其中的一块牌子为主，同时由于承担某项职责，又加挂另一块牌子。挂牌机构一般不单独核定编制和领导职数，不设置相对独立的工作体系<ref name="释义">{{cite book |author1=中央机构编制委员会办公室 |title=机构编制工作用语释义 |date=2014 |location=北京 |accessdate=2018-12-30 |language=zh-cn}}</ref>。

加挂牌子一般是由于在改革中给相关机构增加了职能，但原有的机构名称无法较好地体现新增的职能内容，因而产生的一种机构编制形式<ref name="报道1230A">{{cite web |author1=蓝辉 |title=“对外保留牌子”的行政法内涵 |url=http://share.iclient.ifeng.com/shareNews?aid=52967228&fromType=vampire |website=凤凰网新闻 |accessdate=2018-10-03 |location=北京 |language=zh-cn |archive-url=https://web.archive.org/web/20181230233729/http://share.iclient.ifeng.com/shareNews?aid=52967228&fromType=vampire |archive-date=2018-12-30 |dead-url=no }}</ref>。

=== 行政机构行使不同职责时加挂牌子 ===
某些政府行政机构在国际交流、国际执法时以国家全权代表机构的身份运作，此类加挂牌子称之为“'''涉外时称'''”或“'''对外时称'''”。例如，[[交通运输部海事局|交通运输部海事局]]对外称[[交通运输部海事局|中华人民共和国海事局]]，是[[中华人民共和国交通运输部|中华人民共和国交通运输部]]直属机构，为国务院[[航运|海事]]主管部门，该局在在涉外行政执法、[[行政管理|行政管理]]时使用中国海事局的名义<ref name="报道1231A">{{cite web |author1=大公网编辑 |title=揭秘「一個機構兩塊牌子」 中國特色的機構編制 |url=http://news.takungpao.com.hk/mainland/focus/2013-07/1779741.html |website=大公网新闻 |accessdate=2018-10-03 |location=香港 |language=zh-cn |archive-url=https://web.archive.org/web/20181231093158/http://news.takungpao.com.hk/mainland/focus/2013-07/1779741.html |archive-date=2018-12-31 |dead-url=no }}</ref>。其前身中国港务监督局与交通部港务监督局、中国船舶检验局与交通部船舶检验局，都是类似的一个机构两块牌子<ref name="报道1231A"/>。

政府机构担负不同但有密切联系的多个工作任务，在履行各自职责时，可能使用不同的机构名字<ref name="报道1231A"/>。例如，原[[中华人民共和国国家新闻出版总署|中华人民共和国国家新闻出版总署]]与[[国家版权局|国家版权局]]是一个机构两块牌子，新闻出版总署在履行版权稽查职能时使用国家版权局名义<ref name="报道1231A"/>。

=== 行政机构加挂企事业单位、社会团体牌子 ===
部分政府机构同时以社会团体（行业协会）、企业单位、事业单位的身份参与社会活动或商业活动<ref name="报道1231A"/>。例如，[[国家体育总局|国家体育总局]]加挂[[中华全国体育总会|中华全国体育总会]]牌子；[[新疆生产建设兵团|新疆生产建设兵团]]加挂[[新疆生产建设兵团|中国新建集团公司]]牌子<ref name="报道1231A"/>。

受[[计划经济|计划经济]]体制的影响，部分行业的管理机构与经营企业亦属一个机构两块牌子<ref name="论文1"/>。中华人民共和国对烟草行业实行国家专卖体制，[[国家烟草专卖局|国家烟草专卖局]]与[[中国烟草总公司|中国烟草总公司]]曾实行一个机构两块牌子，烟草局既承担烟草行业行政管理职能，又承接卷烟生产销售任务<ref name="论文1">{{cite journal |author1=莫炜 |title=新形势下我国烟草专卖制度存在问题与发展对策研究 |journal=西南交通大学 |accessdate=2019-01-05 |location=成都 |language=zh-cn}}</ref>。而在盐业体制改革前，食盐运销由国家专营，食盐生产企业必须由省盐务局分配生产计划，且省盐务局与盐业公司两块牌子一套人马，盐务局既是盐业的管理者、又是经营者<ref name="报道0105C">{{cite web |author1=赵展慧 |title=关注中国盐改：改了啥？ 改革后食盐安全咋保障？ |url=http://www.scopsr.gov.cn/zxdd/zygg/201701/t20170107_293989.html |website=中国机构编制网 |accessdate=2019-01-05 |location=北京 |language=zh-cn}}</ref>。

=== 党务机构加挂行政机构牌子 ===
中华人民共和国成立之初，中共中央作出规定，包括中共中央在内的各级党委，都要建立一些和政府对应的部门，比如国务院里原有[[中华人民共和国农业部|农业部]]，中共中央也曾设有[[中共中央农村工作部|农村工作部]]<ref name="报道0106A"/>。1980年8月，[[邓小平|邓小平]]发表《党和国家领导制度的改革》谈话，提倡党政分开<ref name="报道0106A"/>。随后，中共中央陆续撤销了一些与国务院对应的机构<ref name="报道0106A">{{cite web |author1=王玮 |title=党政机构“挂牌子”的讲究 |url=http://www.shm.com.cn/jcld/html/2014-07/13/content_3029694.htm |website=烟台日报 |accessdate=2019-01-06 |location=烟台 |language=zh-cn |archive-url=https://web.archive.org/web/20170613092805/http://www.shm.com.cn/jcld/html/2014-07/13/content_3029694.htm |archive-date=2017-06-13 |dead-url=no }}</ref>。[[改革开放|改革开放]]后，出于对外交往的需要，党政机关有时对外以政府名义交流，这样同样一个机构，在不同的时候就挂上不同的牌子，成为“一个机构两块牌子”<ref name="报道1231A"/>。

例如，[[中共中央宣传部|中共中央宣传部]]加挂[[国务院新闻办公室|国务院新闻办公室]]牌子；[[国务院台湾事务办公室|中共中央台湾工作办公室]]与[[国务院台湾事务办公室|国务院台湾事务办公室]]一个机构两块牌子；[[中央档案馆|中央档案馆]]与[[中央档案馆|国家档案局]]一个机构两块牌子；[[中央保密委员会办公室|中央保密委员会办公室]]与[[中央保密委员会办公室|国家保密局]]一个机构两块牌子<ref name="报道1231A"/>。

=== 其他情形 ===
[[天津市|天津市]]在建设和开发新区域的过程中，为借助成熟的区域开发经验以及相对雄厚的资金实力，而由成熟地区的机构管理待开发新区域，形成了“一个机构两块牌子”<ref name="报道1231B"/>。[[天津经济技术开发区|天津经济技术开发区]]管理委员会加挂[[天津南港工业区|天津南港工业区]]管理委员会牌子，以促进后者复制吸收前者发展经验<ref name="报道1231B">{{cite web |author1=马晖 |title=天津南港暗战环渤海 |url=http://www.21cbh.com:80/HTML/2009-9-7/HTML_II5R4AJTFBSB_1.html |website=21世纪经济报道 |accessdate=2018-12-31 |archiveurl=https://web.archive.org/web/20090908021418/http://www.21cbh.com/HTML/2009-9-7/HTML_II5R4AJTFBSB_1.html |archivedate=2009-09-08 |location=北京 |language=zh-cn |dead-url=no }}</ref>。

== 对外保留牌子 ==
'''对外保留牌子'''，指根据行政体制改革的需要对一个机构进行调整或将相应机构合并到另一个机构后，调整、合并后的机构可以继续延用原机构名称或隶属原机构的其他机构名称，对外继续开展工作<ref name="释义"/>。

对外保留牌子一般是由于机构撤并后，被撤并机构的名称具有较强的不可替代性，从而折衷采取的编制设计方案<ref name="报道1230A"/>。某些被撤并机构的专业性、行业特殊性、重要性相对较强，或原机构规模相对较大，因而一定历史时期内保留其牌子，以保证相关行业的发展和改革前后工作的衔接、队伍的稳定<ref name="报道1230A"/>。保留牌子的机构可以继续延用原机构名称或隶属原机构的其他机构名称，对外继续开展工作<ref name="释义"/>。如[[中华人民共和国核工业部|中华人民共和国核工业部]]撤销后，改为在承接核工业管理职能的[[中华人民共和国工业和信息化部|中华人民共和国工业和信息化部]]加挂[[国家原子能机构|国家原子能机构]]牌子<ref name="报道1230A"/>。

某些机构的设置系由法律、法规和其它规范性文件所规定，故在机构调整后保留其牌子<ref name="报道1230A"/>。行业立法时惯常采用“国务院某某行业主管部门”或类似的表达技术，因此相关部委开展特定行业管理工作时，采用“对外保留”的“牌子”名义<ref name="报道1230A"/>。例如，《[[中华人民共和国国家通用语言文字法|中华人民共和国国家通用语言文字法]]》规定“国家通用语言文字工作由国务院语言文字工作部门负责规划指导、管理监督”，承接语言文字管理职能的[[中华人民共和国教育部|中华人民共和国教育部]]则对外保留[[国家语言文字工作委员会|国家语言文字工作委员会]]牌子<ref name="报道0105B">{{cite web |author1=国家语言文字工作委员会 |title=国家语言文字工作委员会工作职责和任务 |url=http://www.china-language.gov.cn/jg/jgjxx/201704/t20170412_5217.html |website=中国语言文字网 |accessdate=2019-01-05 |location=北京 |language=zh-cn |archive-url=https://web.archive.org/web/20180504035910/http://www.china-language.gov.cn/jg/jgjxx/201704/t20170412_5217.html |archive-date=2018-05-04 |dead-url=no }}</ref><ref name="书目0105A">{{cite book |author1=全国人大教科文卫委员会教育室 |title=中华人民共和国国家通用语言文字法释义及实用指南 |date=2001-02 |location=北京 |accessdate=2019-01-05 |language=zh-cn}}</ref>。

一些对外保留牌子机构的设置，是为了便于国际交流，方便与国际组织对接，而需要以特殊身份独立对外开展工作<ref name="释义"/>。例如，[[国家市场监督管理总局|国家市场监督管理总局]]对外保留[[中国国家标准化管理委员会|中国国家标准化管理委员会]]和[[中国国家认证认可监督管理委员会|中国国家认证认可监督管理委员会]]的牌子<ref name="报道0105A">{{cite web |author1=王勇 |title=关于国务院机构改革方案的说明——2018年3月13日在第十三届全国人民代表大会第一次会议上 |url=http://www.npc.gov.cn/npc/dbdhhy/13_1/2018-03/14/content_2048552.htm |website=中国人大网 |accessdate=2019-01-05 |location=北京 |language=zh-cn |archive-url=https://web.archive.org/web/20180609103422/http://www.npc.gov.cn/npc/dbdhhy/13_1/2018-03/14/content_2048552.htm |archive-date=2018-06-09 |dead-url=no }}</ref>。

此外，部分机构在一个机构两个牌子的同时，对保留牌子机构兼设单独领导班子，并由所属机构领导兼任<ref name="释义"/>。例如，[[国家语言文字工作委员会|国家语言文字工作委员会]]主任由教育部副部长兼任<ref name="报道1230A"/>。[[国家核安全局|国家核安全局]]局长由生态环境部副部长兼任。另有部分保留牌子机构仍保留实体内设机构，如[[国家航天局|国家航天局]]仍保留大量的内设机构、直属事业单位、主管社会团体等，具有一定实体地位<ref name="报道1230A"/>。
