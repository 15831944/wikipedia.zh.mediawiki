《'''一步之遥'''》（{{lang-en|'''Gone with the Bullets'''}}）是一部2014年[[姜文|姜文]]执导的中国3D喜剧片，為北洋三部曲的第二部。影片改编自1920年上海史上的[[阎瑞生|阎瑞生案]]，由[[姜文|姜文]]、[[葛优|葛优]]、[[周韵|周韵]]及[[舒淇|舒淇]]主演。2013年9月開拍，2014年1月拍攝完成。中国于2014年12月18日上映。入选[[第65届柏林影展|第65届柏林影展]]主竞赛单元。

== 剧情 ==
马走日（[[姜文|姜文]]饰）和项飞田（[[葛优|葛优]]饰）创办了“花国选美（妓女选美）大赛”以帮助武大帅之子武七[[洗钱|洗钱]]，吸引多国佳丽参赛和世人关注，最后有三人参加决选。在马走日和项飞田的操纵下，在台上承诺拍卖出嫁权并计划捐出全部身家的前届花魁完颜英（[[舒淇|舒淇]]饰）最后获胜。马走日在与完颜英[[约会|约会]]时被对方逼婚，因马走日没有答应导致对方拿枪相逼迫，两人之后抽了[[大烟|大烟]]并开车去兜风，在郊外车翻导致完颜英意外惨死，马走日之后选择[[逃亡|逃亡]]。

随着媒体对完颜英死亡事件的报道，马走日成为众人心目中的[[杀人犯|杀人犯]]受到警察[[通缉|通缉]]，马走日于是寻求武大帅的相救。马走日在武家与小時候的老师、如今的大帅夫人覃赛男，和旧情人武大帥之女武六一一见面，并意外发现项飞田因武七一起去[[嫖娼|嫖娼]]而受到了武大帅“马办”之惩罚，马走日于是出主意，带领武大帅等人做了一个双手摸地的游戏，救了项飛田一命，馬走日離開武家後才发现自己的此行目的没有达到，他依旧身处危險之中。而项飞田是个忘恩负义的小人，后来反而拼命追查马走日的下落。在马走日逃亡的两年中，以王天王为代表的许多[[戏剧|剧团]]均改编馬走日和完颜英之事為劇本，更拍成电影《枪毙马走日》，在这些戏中马走日成为一个劫财害命的无耻凶手。

因看不下王天王的劇作对其形象的卑鄙刻画，马走日以隐藏身份对王天王进行威逼利诱劝其不要再继续演下去，可是对方不为所动。后来马走日在表演现场动手打了王天王，被埋伏的警察抓获。被抓后，武六与马走日对话后，认为他是被冤枉之人，开始同情起马走日的遭遇；另一面，武七和项飞田却坚信马走日是杀人凶手，希望最后[[枪毙|枪毙]]马走日。在武六和武七因不同原因，向其父武大帥恳求將马走日被从法租界[[引渡|引渡]]到武大帅所管的地区。武七力劝武大帥处死马走日以平息民愤和提升威望。武六却对马走日的感情越来越深，最后还与其母产生激烈争执。

在武大帥迎娶一位[[白俄|白俄]][[贵族|贵族]]女子为妾的夜晚，武六与武大帥对现场来宾表演了一首外国[[歌剧|歌剧]]。之后武六偷偷放走了被关押的马走日，并与之一同开车逃走。武大帅等人在后面乘车紧追不舍，一路上武六为打中后面车顶上的[[喇叭|喇叭]]还与其母玩起了枪战。最后马走日与武六逃进一个[[风车|风车]]所在的屋子里。为了避免武六最后落得像完颜英一样的悲惨下场，马走日将武六打昏并出外[[投降|投降]]。对着外面正在举行[[婚礼|婚礼]]的许多对新人，马走日承认是自己害死了完颜英，武六也是他[[绑架|绑架]]来的。在讲述自己遭世人误解之经历的长篇[[演讲|演讲]]中，他受到多次枪击最终死亡。

故事的最后是武六幻想出来的画面：她在[[站台|站台]]上与坐在[[列车|列车]]里的马走日分手告别。

== 制作 ==
[[第65届戛纳电影节|第65届戛纳电影节]]，姜文发布了《一步之遥》的海报和宣传单。<ref>{{cite web |url=http://ent.qq.com/a/20120523/000460.htm |title=姜文公布新片《一步之遥》 海报“席卷”戛纳 |accessdate=2012年5月23日 |publisher= 腾讯网|work= |language =zh-cn}}</ref>

姜文表示，故事原型来自于1920年轰动全国的上海“[[阎瑞生|阎瑞生]]杀害花国总理”案，这桩谜案背后的冒险、悬疑、荒诞、爱情的多重元素是令他着迷的原因。影片只选取了“花国大选”和“总统遇害”两个核心元素，整个故事被重新编织得更具悬疑和荒诞感。姜文认为北洋年代虽然短暂但疯狂梦幻。另一件“施剑翘替父报仇刺杀孙传芳案”也在其拍摄计划中，将与《[[让子弹飞|让子弹飞]]》、《一步之遥》组成“北洋三部曲”。<ref >{{cite web|url=http://ent.163.com/14/0519/08/9SJJMPLO000300B1.html | language= | title=姜文谈《一步之遥》 故事源自北洋阎瑞生案 |publisher=: 网易娱乐 |date=2014-05-19 |accessdate= }}</ref>

影片由获11次[[奥斯卡金像奖|奥斯卡金像奖]]提名的[[迈克尔·明克勒|迈克尔·明克勒]][[混音|混音]]，{{link-en|尤金·吉尔蒂|Eugene Gearty}}担任[[音效|音效]]设计，采用[[杜比全景声|杜比全景声]]<ref>{{cite news|title=迈克尔明克勒三夺小金人 赞葛优让其捧腹|url=http://news.mtime.com/2014/12/11/1535121.html|accessdate=2014-12-13|newspaper=时光网|date=2014-12-11}}</ref>。

== 发行 ==
为了给电影的首映造势，片方联合[[IMAX|IMAX]]公司将北京奥体中心[[国家体育馆_(北京)|国奥体育馆]]改造为临时IMAX影院，体育馆中挂起的一座宽26米高14.5米IMAX银幕<ref>{{cite news|title=《一步之遥》为首映做准备 北京奥体大变身 IMAX临时影院成功挂幕|url=http://news.mtime.com/2014/12/06/1534919.html|accessdate=2014-12-13|newspaper=时光网|date=2014-12-06}}</ref>。新作原定于12月8日在北京举行全球首映礼正式确认推迟，仅在7日当晚进行的媒体看片会上放映部分片段。首映礼推迟的原因，有圈内人士猜测是影片尚未通过终审，未拿到“龙标”（[[电影片公映许可证|电影片公映许可证]]）<ref>{{cite news|title=离《一步之遥》还有“几步之遥” 发布会推迟 制片人马珂:"保证18日上映"|url=http://news.mtime.com/2014/12/07/1534941.html|accessdate=2014-12-13|newspaper=时光网|date=2014-12-07}}</ref>。

2014年12月12日，[[姜文|姜文]]透过微博发布消息，确认影片过审，将如期上映，首映礼改在15日举行<ref>{{cite news|title=《一步之遥》已经过审 院线"人肉快递"拷贝　12月18日如期上映|url=http://news.mtime.com/2014/12/12/1535190.html|accessdate=2014-12-13|newspaper=时光网|date=2014-12-12}}</ref>。《一步之遥》于12月15日夜在北京举办了全球首映礼，有2828名观众观看了影片，当晚的反响并不好<ref >{{cite web|url=http://ent.163.com/14/1216/09/ADIVHTF2000300B1.html | language= | title=姜文电影《一步之遥》被批 除配乐几乎一无是处 |publisher=网易娱乐 |date=2014-12-16 |accessdate= }}</ref>。

由于首周的3亿多票房不够理想，片方决定自12月23日下午15点起增加上映2D版《一步之遥》。<ref >{{cite web|url=http://www.1905.com/news/20141222/836186.shtml | language= | title=《一步之遥》推2D版救市 疑避3D《智取威虎山》 |publisher=1905电影网 |date=2014-12-22 |accessdate= }}</ref>
