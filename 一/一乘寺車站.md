'''一乗寺車站'''（{{jpn|j=一乗寺駅|hg=いちじょうじえき|rm=Ichijōji eki|f=y}}）是位於[[日本|日本]][[京都市|京都市]][[左京區|左京區]]一乘寺里之西町的[[叡山電鐵|叡山電鐵]][[叡山本線|叡山本線]]的車站。

== 車站結構 ==
擁有[[相對式月台|相對式月台]]2面2線。平常雖然沒有站員進駐，但平日的早晨或乘客較多的時候則會配置站員於此，因此在往出町柳車站方向的月台出入口處設有剪票口狀的柵欄。且僅在往出町柳車站方向的月台設有售票機，可於8點至18點30分之間使用。

入口位在月台靠出町柳車站的一側，僅鄰「曼殊院道」上的平交道（但是要去[[曼殊院|曼殊院]]的話則相鄰的修學院車站比較近）。往[[八瀨比叡山口車站|八瀨比叡山口車站]]、[[鞍馬車站|鞍馬車站]]方向的月台幾乎全部都設有遮蓬，往出町柳車站方向的月台則設有約一節車箱長的遮蓬，隆罩住一人控制電車的登車口。

兩個方向的月台皆沒有無障礙設施。從道路至月台間只有階梯（參照圖片），使用輪椅的話必需要有人從中協助。

== 車站周邊 ==
車站面向曼殊院道，位於商店街。在啟用當時有許多商家羅列，在戰後市場被開拓，曼殊院道上也開始有公車行駛。1960年代，車站西側發展成了商店街、風化區，而且擁有在當時除了市中心之外非常稀有的電影院、柏青哥店。從1980年左右開始衰退，電影院倒閉，但商店街的模樣則保留了下來。現在，以大學生為對相的店家較多，特別是拉麵店的數量在京都被說是數一數二，京都拉麵在日本全國的拉麵迷中知名度相當高<ref>一乘寺車站周邊的拉麵[http://www.hotpepper.jp/A_11100/smd1_stc2204515_dst3_svcSA22_grcG013.html]（hotpepper.jp）{{ja}}</ref><ref>Google マップ: [http://maps.google.co.jp/maps?sourceid=navclient-ff&ie=UTF-8&rlz=1B2GGGL_jaJP209JP209&um=1&q=%E3%83%A9%E3%83%BC%E3%83%A1%E3%83%B3&near=%E4%B8%80%E4%B9%97%E5%AF%BA%E9%A7%85%EF%BC%88%E4%BA%AC%E9%83%BD%EF%BC%89&fb=1&view=text&sa=X&oi=local_group&resnum=1&ct=more-results&cd=1 一乗寺車站（京都）附近拉麵的搜尋結果]{{ja}}</ref><ref>但最近由於意識到有人是開車前來的，周邊白川通、東大路通沿路的拉麵店也在增加中。</ref>。

附近有[[詩仙堂|詩仙堂]]、一乘寺下松等許多觀光景點。

現在最近的[[京都市營巴士|京都市營巴士]]公車站有東方約400公尺處面向[[白川通|白川通]]的5、65、北8路「一乘寺下松町」，以及西方約300公尺處面向[[東大路通|東大路通]]的31路「一乘寺高槻町」。

雖然叡電的普通運費是採用區間制度，但考慮到商店街的使用者，曾經有段時期將本站～三宅八幡車站劃為一區。

*一乘寺下松（以[[宮本武藏|宮本武藏]]和吉岡一門在此處的決鬥而聞名）
*詩仙堂（車內廣播會提及）
*狸谷山不動院（車內廣播會提及）
*惠文社一乘寺店（以美術書籍等聞名的書店）
*[[京都工藝纖維大學|京都工藝纖維大學]]（通常是使用公車。鐵路的話最近的是[[松崎車站_(京都府)|松崎車站]]）
*京都市立修學院中學校
*京都市立修學院第二小學校
*[[滋賀銀行|滋賀銀行]]一乘寺分店
*將軍山城

== 歷史 ==
* 1925年（[[大正|大正]]14年）9月27日　開業。當時為[[京都電燈|京都電燈]]的車站。
* 1942年（[[昭和|昭和]]17年）3月2日　由於公司的讓渡，成了[[京福電氣鐵道|京福電氣鐵道]]的車站。
* 1986年（昭和61年）4月1日　由於公司的讓渡，成了叡山電鐵的車站。
