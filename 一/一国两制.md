'''一國兩制'''意指「'''一個國家，兩種制度'''」（{{lang-en|One country, two systems}}，{{lang-pt|Um país, dois sistemas }}），是[[中華人民共和國|中華人民共和國]]第二代[[中華人民共和國最高領導人|領導人]][[鄧小平|鄧小平]]在1980年代為了實現[[中國統一|中國統一]]之目標所提出的[[中華人民共和國憲法|憲法]]原則<ref>{{Cite web |url = http://www.nanzao.com/tc/opinion/36984/xiang-gang-shi-shi-hou-mian-dui-yi-guo-yi-zhi-de-xian-shi |title = 香港是時候面對“一國一制”的現實 |author = Doreen Dang |date = 2014-09-03 |publisher = 《[[南華早報|南華早報]]》 |language = zh-hant |accessdate = 2014-07-23 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20141214235844/http://www.nanzao.com/tc/opinion/36984/xiang-gang-shi-shi-hou-mian-dui-yi-guo-yi-zhi-de-xian-shi |archivedate = 2014-12-14 }}</ref>。最早是為了改變中華民國政府撤退至[[臺灣|臺灣]]後的海峽兩岸關係，總統[[蔣經國|蔣經國]]則以[[一國良制|一國良制]]（One China, Better System）回應鄧小平<ref>{{cite news|author1=海基會首任秘書長陳長文|title=陳長文／辜汪會談23週年：放下兩國論，追求一國良制|url=http://www.ettoday.net/news/20160502/690431.htm|agency=Ettoday東森新聞雲|date=2016-05-02}}</ref>。

後來相繼適用於過去分別為[[英國|英國]]和[[葡萄牙|葡萄牙]][[殖民地|殖民地]]、實施[[資本主義|資本主義]]制度的[[香港|香港]]及[[澳門|澳門]]，將兩地变為[[特别行政区|特别行政区]]。邓小平主張中華民國持有的[[南京條約|南京條約]]等[[大清|大清]]與[[列強|列強]]所簽之條約無效，在[[中国统一|中国和平统一]]時只有中华人民共和国能夠代表[[一個中国|一個中国]]，並且是中国唯一合法政府，而中华人民共和国也多次對外主張其對港澳之主权政策，並且要求其他國家應該要認識到一国两制政策。1990年代起，[[港澳|港澳]]保留[[資本主義|資本主義]]經濟和社會政治制度，繼續擁有自己的政治管治制度、法律體系、金融經濟政策事務以及對外關係50年，而中华人民共和国其他地区則繼續實行[[中国特色社会主义|中国特色社会主义]]制度<ref>{{Cite web |url = http://www.china.org.cn/english/features/china/203730.htm |title = One Country, Two Systems |author = |date =  |publisher = [[中國網|中國網]] |language = en |accessdate = 2014-07-23 |quote =  }}</ref>。

學界認為[[西藏|西藏]]與[[中共|中共]]所簽訂的《[[和平解放西藏十七條協定|和平解放西藏十七條協定]]》是北京地方自治精神的初嘗試，並形容和平協定是一国兩制的濫觴<ref>董樹藩，民國48年西藏反共抗暴後達賴喇嘛言行之研析，蒙藏委員會，第57頁</ref><ref>林照真認為一般人誤以為香港的「一国两制」是出自邓小平，但一国两制實際上只是西藏＜十七條協議＞的另一種說法，換句話說，一国两治第一次實施地區應該是西藏，＜十七條協議＞才是北京地方自治精神的初嘗試。林照真，最後的達賴喇嘛，時報文化，ISBN 957-13-3263-1，第328頁</ref>。

== 歷史 ==
=== 兩岸分治 ===
1949年12月7日，於[[國共內戰|國共內戰]]中不斷戰敗的[[中華民國政府|中華民國政府]]正式由[[成都市|成都市]]遷往[[臺北市|臺北市]]，並繼續統治[[臺灣|臺灣]]、[[澎湖群島|澎湖群島]]、部分[[福建省|福建省]]離島（[[金門|金門]]和[[馬祖|馬祖]]）、[[東沙群島|東沙群島]]和[[太平島|太平島]]等地區，而剛由[[中國共產黨|中國共產黨]]成立的[[中華人民共和國政府|中華人民共和國政府]]在初期決定採取「武力解放台灣」方針。<ref>{{Cite web |url =http://history.people.com.cn/n/2013/0523/c198865-21584349.html |title =毛泽东筹划用武力解放台湾为何没能实现？|author =来源：中国共产党新闻网  |date = 原文发表日期 |publisher = 人民网 |language = zh-cn |accessdate = 2016-10-06 |quote =朝鲜停战协定签署，毛泽东认为是该解决台湾问题的时候了。10月，毛泽东在中央军委会议上说：“朝鲜停战了，我们身上的担子一下轻了很多……这两年，我们那位在台湾的蒋先生趁我们抗美援朝无暇他顾之际，仗着有‘山姆大叔’撑腰，很是兴风作浪，在那里做反攻大陆的美梦哩！我们现在已经可以腾出手来了，我看该集中力量去解决台湾的问题了。”}}</ref>直到1950年，中華人民共和國基本上已經控制中國大陸大部分地區後，中華民國則僅能控制臺灣地區以及剩餘部分島嶼。然而這時雙方政府皆照著各自憲法條文聲稱自己為「中國」的唯一合法政府，此後雙方到1960年代末期前仍然發生多次小規模衝突，不過先前獲得優勢的[[中國人民解放軍|中國人民解放軍]]在美國派[[第七艦隊|第七艦隊]]進入[[台灣海峽|台灣海峽]]後，始終未能成功取得臺灣和其周邊島嶼。而這時聯合國創始會員國的中華民國政府仍然被[[聯合國|聯合國]]認可為中國合法政府並保有[[中國與聯合國|中國代表權]]，同時繼續擔任[[聯合國安全理事會|聯合國安全理事會]][[聯合國安全理事會常任理事國|常任理事國]]。

之後中華民國政府長期認定中華人民共和國政府是非法叛亂組織，認為自身仍然擁有中國大陸的主權，並且中華人民共和國不能取代中華民國所代表的中國席次。中華民國政府的主張受到大部分非[[社會主義國家|社會主義國家]]的認同，同時中華民國政府也長期不承認中華人民共和國政府的國際活動，並且要求其他國家尊重和理解。不過相對地中華人民共和國政府在1949年成立後也不承認中華民國政府，聲稱其已被消滅，並且堅持臺灣地區為自身領土的一部分。1951年，大陆政府與西藏簽署《[[十七條協議|十七條協議]]》，承諾保留西藏制度，一国兩制之概念在西藏實行，最終實施了8年<ref>{{Cite news |title= 兩岸如何一家親？ |url= http://news.now.com/home/international/player?newsId=332903 |work= now新聞 |date= 2019-01-03 }}</ref>。1956年，[[中國共產黨中央委員會主席|中國共產黨中央委員會主席]][[毛澤東|毛澤東]]提出了「和平解放臺灣」的構想，希望透過第三次[[國共合作|國共合作]]來統一臺灣。1961年6月，毛澤東在與[[印尼總統|印尼總統]][[蘇卡諾|蘇卡諾]]會談時首次談及容許臺灣保持原來社會制度，並且提到「如果臺灣歸還祖國，中國就可以進聯合國。如果臺灣不作為一個國家，沒有中央政府，它歸還祖國，那麼臺灣的社會制度也可以留待以後談。我們容許臺灣保持原來的社會制度，等臺灣人民自己來解決這個問題。」<ref>{{Cite web | url=http://www.china.com.cn/chinese/zhuanti/ffl/733754.htm | title=毛澤東的「和平解放台灣」| publisher=中國網 | work=《台灣百科》}}</ref>

之後在1963年時，[[中華人民共和國總理|中華人民共和國總理]][[周恩來|周恩來]]將其對臺政策歸納為[[一綱四目|一綱四目]]，其內容已經隱含後來一國兩制的概念。到了1971年10月25日的聯合國大會上，[[阿爾巴尼亞|阿爾巴尼亞]]、[[阿爾及利亞|阿爾及利亞]]等23國所提出「關於恢復中華人民共和國在聯合國合法權利之提案」獲得通過，之後中華人民共和國便取代中華民國在聯合國的中國代表權。之後中華人民共和國更進一步與多國建立正式外交關係，相對地中華民國的邦交國家則逐漸減少。而在1979年美國政府正式與中華民國政府斷交，並且和中華人民共和國政府建立正式全面的外交關係後，大部分國家已經承認中華人民共和國政府為代表中國的唯一合法政府。雖然海峽兩岸仍然繼續處於政治對立並且各自統治的狀態，但是後來中國大陸[[改革開放|改革開放]]促使海峽兩岸的經濟與文化交流漸趨頻繁。

1978年10月8日，中华人民共和国副總理[[鄧小平|鄧小平]]在會見日本文學評論家[[江藤淳|江藤淳]]時提到：「如果實現祖國統一，我們在臺灣的政策將根據臺灣的現實來處理。比如說，美國在臺灣有大量的投資，日本在那裡也有大量的投資，這就是現實，我們正視這個現實。」其中鄧小平在最初構想中認為中華人民共和國政府在統一後將會尊重台灣的現實情況，並且繼續保護外商投資。

1978年11月14日，鄧小平在結束對[[泰國|泰國]]、[[馬來西亞|馬來西亞]]和[[新加坡|新加坡]]的國是訪問並且準備途經[[緬甸|緬甸]]返國時，在與[[緬甸總統|緬甸總統]][[吳奈溫|吳奈溫]]會談中談到臺灣問題時表示：「在解決臺灣問題時，我們會尊重臺灣的現實。比如，臺灣的某些制度跟生活方式可以不動，但是要统一。」

1978年12月，[[中國共產黨第十一屆中央委員會第三次全體會議|中共十一屆三中全會]]的全會公報在提及臺灣問題時，首次以「臺灣回到祖國懷抱，實現統一大業」來代替「解放臺灣」的說法。

1979年1月1日，[[全國人大常委會|全國人大常委會]]發表了《[[告台灣同胞書|告台灣同胞書]]》，同時[[中華人民共和國國防部|中華人民共和國國防部]]部長[[徐向前|徐向前]]也發表了關於停止自1958年起對[[金門縣|金門縣]]炮擊的聲明。在聲明中表示海峽兩岸應該儘快結束分裂局面，並且呼籲結束海峽兩岸軍事對峙局勢、開放海峽兩岸[[三通|三通]]以及擴大海峽兩岸關係交流等方針。在這次公開信再度重申中華人民共和國期望和平統一的態度，並且被視為是中華人民共和國針對臺灣最早制定的政策內容<ref>{{Cite web |url = http://news.xinhuanet.com/ziliao/2003-01/23/content_704733.htm |title = 全国人大常委会《告台湾同胞书》 |author = [[全國人民代表大會常務委員會|全國人民代表大會常務委員會]] |date = 1979-01-01 |publisher = [[新華網|新華網]] |language = zh-hans |accessdate = 2014-07-23 |quote =  }}</ref>。

1981年9月底，[[全國人大常委會委員長|全國人大常委會委員長]][[葉劍英|葉劍英]]在向新華社記者發表的談話中提到：「國家實現統一後，臺灣可作為特別行政區，享有高度的自治權，並可保留軍隊……臺灣現行社會、經濟制度不變，生活方式不變，同外國的經濟、文化關係不變。私人財產、房屋、土地、企業所有權、合法繼承權和外國投資不受侵犯。」

1982年1月，鄧小平更進一步表示：「『一個國家，兩種制度』，兩制是可以允許的，他們不要破壞大陸的制度，我們也不要破壞他那個制度。」

1983年6月4日，於[[中國人民政治協商會議|中國人民政治協商會議]]第六屆全國委員會第一次會議，主席[[鄧穎超|鄧穎超]]：「在統一的大前提下，一切問題都好商量，總會求得合情合理的解決。」

1983年6月26日，[[中共中央军委主席|中共中央军委主席]]、[[中國共產黨中央顧問委員會|中共中央顧問委員會]]主任[[鄧小平|鄧小平]]會見[[美國|美國]][[新澤西州|新澤西州]][[薛頓賀爾大學|西東大學]][[政治學|政治學]][[教授|教授]]楊力宇時重申：「祖國統一後，[[台灣特別行政區|台灣特別行政區]]可以有自己的獨立性，可以實行同大陸不同的制度。[[司法獨立|司法獨立]]，終審權不須到北京。台灣還可以有自己的軍隊，只是不能構成對大陸的威脅。大陸不派人駐台，不僅軍隊不去，行政人員也不去。台灣的黨、政、軍等系統，都由台灣自己來管。中央政府還要給台灣留出名額<ref>{{cite web|url=http://big5.huaxia.com/zt/pl/08-068/1270802.html |title=鄧小平中國大陸和台灣和平統一設想（1983.06.26）|author=[[人民網|人民網]]|date=2008-12-25|publisher = [[華夏經緯網|華夏經緯網]]|accessdate=2014-07-23}}</ref>。」<ref name="鄧3"/>{{rp|30}}。而在楊力宇的文章中則提到鄧小平針對臺灣問題所提出的一國兩制作法被指實際上取自於[[聯邦制|聯邦制]]想法，不過後來這一論點則是被中華人民共和國官方所否認。後來鄧小平對於[[海峽兩岸關係|海峽兩岸關係]]未來統一的政治構想被歸為[[鄧小平理論|鄧小平理論]]的一部份，其中主張以特別行政區的方式管理臺灣、但是比後來實施的香港和澳門地區擁有如軍隊、司法、政治制度等有更多自治權。此次会谈要点归纳为[[邓六条|邓六条]]，虽然邓小平为了统一，可以更改中华人民共和国[[国号|国号]]的论点未再宣扬。但形成“在‘一个中国’原则下，什么都可以谈”的论调。2012年，国台办在回答更改国号的问题时，仍持“只要坚持一个中国，两岸之间什么都可以谈”的论调<ref>{{Cite web |url=http://politics.people.com.cn/n/2012/1226/c70731-20025125.html |title=国台办回应两岸可以协商“国号”建议|accessdate=2019-03-27 |author= |date=2012-12-26 |work= |publisher=人民网，来源：中国网|language=Zh-hans }}</ref>。

:歷任[[中國共產黨中央委員會總書記|中國共產黨中央委員會總書記]]對台政策的論述：

*1992年10月12日，中共中央總書記[[江澤民|江澤民]]於[[中共十四大|中共十四大]]報告中，提及「在一個中國的前提下，什麼問題都可以談，……。」

*1997年9月12日，中共中央總書記[[江澤民|江澤民]]於[[中共十五大|中共十五大]]報告中，提及「在一個中國的前提下，什麼問題都可以談。只要是有利於祖國統一的意見和建議，都可以提出來。」

*2002年11月8日，中共中央總書記[[江澤民|江澤民]]於[[中共十六大|中共十六大]]報告中，提及「在一個中國的前提下，什麼問題都可以談，……。」总结为[[三个可以谈|三个可以谈]]。

*2007年10月15日，中共中央總書記[[胡錦濤|胡錦濤]]於[[中共十七大|中共十七大]]報告中，提及「台灣任何政黨，只要承認兩岸同屬一個中國，……，什麼問題都可以談。」

*2012年11月8日，中共中央總書記[[胡錦濤|胡錦濤]]於[[中共十八大|中共十八大]]報告中，提及「對台灣任何政黨，只要不主張台獨、認同一個中國，我們都願意同他們交往、對話、合作。」

*2014年10月20日，中共中央總書記[[習近平|習近平]]於[[中共十八屆四中全會|中共十八屆四中全會]]中，提及「運用法治方式鞏固和深化兩岸關係和平發展，完善涉台法律法規，依法規範和保障兩岸人民關係、推進兩岸交流合作。運用法律手段捍衛一個中國原則、反對台獨，增進維護一個中國框架的共同認知，推進祖國和平統一。」

一國兩制政策提到儘管在[[香港回歸|香港回歸]]和[[澳門回歸|澳門回歸]]並且作為中華人民共和國的領土一部分後，中華人民共和國長期統治的[[中國大陸|中國大陸]]地區仍然施行[[中國特色社會主義|中國特色社會主義]]以及[[民主集中制|民主集中制]]。中華人民共和國政府依照《[[中華人民共和國憲法|中華人民共和國憲法]]》第三十一條的規定能夠在國家有其必要時設立[[特別行政區|特別行政區]]，並且[[全國人民代表大會|全國人民代表大會]]將會根據具體情況制定特別行政區自治區域內施行的法律規定。而為此所個別制定的[[中華人民共和國特別行政區基本法|中華人民共和國特別行政區基本法]]則作為特別行政區的最高法律，並且作為特別行政區地方性法律的基礎以實行[[自治地方|高度自治]]。而[[全國人民代表大會常務委員會|全國人民代表大會常務委員會]]亦表示經由全國人民代表大會全體會議通過的[[基本法|基本法]]本身有憲-{制}-性質，這也使得基本法又有「小憲法」之稱。

在基本法中主要規範了特別行政區與中央政府的關係、居民的權利和義務、政治體制、經濟社會事務和對外事務等，也明訂在特別行政區內施行的各項制度。其中一國兩制政策一方面肯定中國作為政治實體的獨特性，另外一方面在單一主權下各個地區可以安排不同的政治和經濟制度。根據基本法內容香港和澳門兩個省級特別行政區政府負責自身地區事務，並且經常稱作「港人治港，高度自治」及「澳人治澳，高度自治」。而在和平時期絕大部分全國性法律不適用於特別行政區內，各個特別行政區因應需要自行訂定法律，並且實施依照基本法規定適用於特別行政區的全國性法律。而當地特別行政區政府領導人則被稱為[[行政長官|行政長官]]，俗稱「特別行政區首長」或者是「特區首長」、「特首」。

而過去分別為[[英國|英國]]與[[葡萄牙|葡萄牙]][[殖民地|殖民地]]的[[港澳地區|港澳地區]]得以繼續保留其既定-{}-制度，並且在統一後至少50年內維持高度自治系統，不過特別行政區的[[外交|外交]]與[[軍事|軍事]]則是由[[中華人民共和國政府|中華人民共和國政府]]負責<ref>{{Cite web |url = http://www.basiclaw.gov.hk/tc/basiclawtext/chapter_2.html |title = 第二章: 中央和香港特別行政區的關係 |author =  |date =  |publisher = [[香港特別行政區政府|香港特別行政區政府]] |language = zh-hant |accessdate = 2014-09-01 |quote =  }}</ref><ref>{{Cite web |url = http://bo.io.gov.mo/bo/i/1999/leibasica/index_cn.asp |title = 中華人民共和國澳門特別行政區基本法 |author =  |date = |publisher = [[印務局|印務局]] |language = zh-hant |accessdate = 2014-09-01 |quote =  }}</ref>。其中在香港特別行政區的[[憲法|憲制文件]]《[[中華人民共和國香港特別行政區基本法|中華人民共和國香港特別行政區基本法]]》第一章第五條中便提到：「香港不實行社會主義制度和政策，保持原有的資本主義制度和生活方式，五十年不變。<ref>{{Cite book |author = Jing Luo |title = Over a Cup of Tea: An Introduction to Chinese Life and Culture |location = 美國朗漢（Lanham）|publisher = {{Link-en|美國大學出版社|University Press of America}} |date = 2004-08-11 |pages = |ISBN = 978-0761829379 |accessdate = 2014-07-23 |url = http://books.google.com.tw/books?id=QESTbCP0Kz0C&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false |language = en |quote =  }}</ref><ref>{{Cite book |author = Wong Yiu-chung |title = One Country, Two Systems in Crisis: Hong Kong's Transformation since the Handover |location = 美國朗漢（Lanham）|publisher = {{Link-en|Rowman & Littlefield|Rowman & Littlefield}} |date = 2004-10-01 |pages = |ISBN = 978-0739104927 | accessdate = 2014-07-23 |url = http://books.google.com.tw/books?id=NIcq7y3oBQYC&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false |language = en |quote = }}</ref>」不過，針對香港和澳門分別在2047年以及2049年過後是否有政策改動，中華人民共和國政府以及官方文件上從來沒有公開表示。在1988年6月3日，鄧小平在會見「『九七』年代的中國與世界」國際會議全體與會者時指出：<ref>{{Cite web |url=http://zg.people.com.cn/GB/33839/34943/34944/34947/2617639.html |title=要吸收国际的经验 |accessdate=2018-03-22 |author=邓小平 |date=1988-06-03 |work=著作选登 - === 實施情況 ===
{{Main|特別行政區}}

在《[[中英聯合聲明|中英聯合聲明]]》和《[[中葡聯合聲明|中葡聯合聲明]]》發表後，香港與澳門地區決定實施各自的[[基本法|基本法]]以維持一國兩制原則。根據雙方個別的基本法規定特別行政區擁有多項事務高度自治的權利，享有行政管理權、立法權、獨立司法權以及終審權。其中憑藉著基本法使得香港與澳門分別成立[[香港特別行政區政府|香港特別行政區政府]]和[[澳門特別行政區政府|澳門特別行政區政府]]作為[[行政机构|行政機構]]，以及包括作為[[立法機構|立法機構]]的[[香港立法會|香港立法會]]與[[澳門特別行政區立法會|澳門特別行政區立法會]]和作為檢察機關的[[律政司|律政司]]與[[澳門特別行政區檢察院|澳門特別行政區檢察院]]。同時作為司法機構的[[香港司法機構|香港司法機構]]與[[澳門特別行政區司法機關|澳門特別行政區司法機關]]擁有獨立的審判權和終審權，其中香港繼續實施英國的[[普通法|普通法]]，而澳門則繼續使用葡萄牙的[[歐陸法系|民事法律制度]]<ref>{{Cite web |url = http://www.info.gov.hk/info/sar5/claw_1.htm |title = 法治 |author =  |date =  |publisher = [[香港特別行政區政府|香港特別行政區政府]] |language = zh-hant |accessdate = 2014-07-23 |quote =  }}</ref>。雖然[[香港特別行政區行政長官|香港特別行政區行政長官]]和[[澳門特別行政區行政長官|澳門特別行政區行政長官]]會經由特別行政區行政長官選舉選出，但是連同特別行政區的主要官員都必須提報[[中華人民共和國國務院|中華人民共和國國務院]]（中央人民政府）任命，這意味著實際上中華人民共和國政府對於主要官員人選仍然擁有決定權。

特別行政區政府可以自行處理各種內部事務，制定[[外來移民|出入境]]、[[海關|海關]]政策、[[公共財政|公共財政]]政策、社會治安以及[[引渡法|引渡法]]規定。同時特別行政區政府在基本法框架內得以制定訂定有關該地區[[教育|教育]]、[[文化|文化]]、[[體育|體育]]、社會福利等政策，而在特別行政區中不適用在中國大陸施行的文化產業相關法律規定<ref>{{Cite web |url = http://www.info.gov.hk/info/sar5/cmessage.htm |title = 行政長官董建華獻辭 |author = [[董建華|董建華]] |date =  |publisher = [[香港特別行政區政府|香港特別行政區政府]] |language = zh-hant |accessdate = 2014-07-23 |quote = }}</ref><ref>{{Cite web |url = http://www.info.gov.hk/info/sar5/c12.htm |title = 一國兩制 |author =  |date = |publisher = [[香港特別行政區政府|香港特別行政區政府]] |language = zh-hant |accessdate = 2014-07-23 |quote = }}</ref>。在公眾教育以及大眾傳播也未強制使用[[簡化字|簡化字]]以及[[現代標準漢語|現代標準漢語]]，其中香港地區的法定語言為[[中文|中文]]以及[[英語|英語]]，而澳門地區的法定語言則為中文以及[[葡萄牙語|葡萄牙語]]，而香港和澳門則以[[繁體中文|繁體中文]]書寫、以[[粵語|粵語]]作通用語言。在經濟和民生上，特別行政區按基本法規定保持原有的[[資本主義|資本主義]]制度，而不實行中國大陸地區的[[社會主義|社會主義]]制度和政策{{NoteTag|《[[中華人民共和國香港特別行政區基本法|中華人民共和國香港特別行政區基本法]]》第五條：「香港特別行政區不實行社會主義制度和政策，保持原有的資本主義制度和生活方式，五十年不變。」}}{{NoteTag|《[[中華人民共和國澳門特別行政區基本法|中華人民共和國澳門特別行政區基本法]]》第五條：「澳門特別行政區不實行社會主義制度和政策，保持原有的資本主義制度和生活方式，五十年不變。」}}。同時特別行政區可以自行制訂自己的[[通貨|流通貨幣]]和財政方針，[[港元|港元]]和[[澳門幣|澳門幣]]分別繼續作為香港與澳門的流通貨幣。另外基本法還提供憲法所保障的[[人權|人權]]和自由等權利，同時基本法的條文也讓《[[公民及政治权利国际公约|公民權利和政治權利國際公約]]》獲得憲法保障地位<ref>{{Cite web |url = http://www.info.gov.hk/gia/general/200011/09/1109239.htm |title = 律政司司長致辭全文 |author = [[梁愛詩|梁愛詩]] |date = 2000-11-09 |publisher = [[香港特別行政區政府|香港特別行政區政府]] |language = zh-hant |accessdate = 2014-07-23 |quote =  }}</ref>。

由於香港特別行政區和澳門特別行政區過去曾經分別被英國以及葡萄牙管治過，在這般背景下使得主要官員、政府機構乃至於法律設計上有部分相似地方，例如雙方都有[[廉政公署|廉政公署]]、[[審計署|審計署]]、[[終審法院|終審法院]]和[[民政總署_(澳門)|民政總署]]（港稱[[民政事務總署|民政事務總署]]）等機構，同時兩個地區都不同於中國大陸地區而採行[[道路通行方向|靠左行駛]]的方式。不過兩個特別行政區在制度上也有不同處，分別包括有保留傳統的政府架構{{NoteTag|[[香港特別行政區政府|香港特別行政區政府]]轄下分成[[香港特別行政區政務司司長|政務司]]、[[香港特別行政區財政司司長|財政司]]和[[律政司|律政司]]三司，而[[澳門特別行政區政府|澳門特別行政區政府]]轄下則分成[[行政法務司|行政法務司]]、[[經濟財政司|經濟財政司]]、[[保安司|保安司]]、[[社會文化司|社會文化司]]以及[[運輸工務司|運輸工務司]]五司，同時後者在職能方面僅相當於香港司長轄下的各局。}}、法律體系不同{{NoteTag|[[香港法律制度|香港法律制度]]繼續實施英國的{{Link-en|英國法律|English law|普通法}}，而[[澳門司法制度|澳門司法制度]]則繼續沿用葡萄牙的[[歐陸法系|民事法律制度]]。}}、澳門沒有高等法院等{{NoteTag|一直到1999年6月30日時，葡萄牙把所有訴訟案件的終審權移交給澳門高等法院，並且改組成為[[澳門特別行政區終審法院|澳門特別行政區終審法院]]。}}。同時《中華人民共和國澳門特別行政區基本法》是以較早制定的《中華人民共和國香港特別行政區基本法》為藍本所通過的修訂版本，因此香港與澳門兩者的基本法在內容上有些許不同之處，不過這也避免了很多憲-{制}-法律中的問題與漏洞{{NoteTag|[[全國人民代表大會常務委員會|全國人民代表大會常務委員會]]分別在1999年針對[[香港居留權爭議|香港居留權爭議]]、2004年針對[[香港政治制度改革|香港政治制度改革]]、2005年針對[[香港特別行政區行政長官|香港特別行政區行政長官]][[香港政治|任期問題]]和2011年針對[[香港對外事務|香港對外事務]]問題進行基本法解釋，而澳門則是在2011年時才有關於[[澳門政治制度改革|澳門政治制度改革]]的基本法解釋。}}。

2004年3月8日，[[全國人民代表大會常務委員會副委員長|全國人民代表大會常務委員會副委員長]][[王兆國|王兆國]]説明《中華人民共和國憲法修正案（草案）》時稱：憲法修正案（草案）在憲法第十四條增加第四款「國家建立健全同經濟發展水平相適應的社會保障制度。」；在憲法第三十三條中增加第三款「國家尊重和保障人權。」；在憲法第五十九條第一款關於全國人民代表大會組規定中增加「特別行政區」，將這一款修改為：「全國人民代表大會由省、自治區、直轄市、特別行政區和軍隊選出的代表組成。各少數民族都應當有適當名額的代表。」<ref>{{cite book |title=《「一國兩制」有關文憲》 |publisher=香港《基本法》敎育協會 |year=2004 |ISBN = 988-97873-1-8 }}</ref>{{rp|31-32}}

總部位於首都北京市的[[中華人民共和國國務院|中華人民共和國國務院]]（中央人民政府）透過設立在香港的[[中央人民政府駐香港特別行政區聯絡辦公室|駐香港特別行政區聯絡辦公室]]與香港特別行政區政府保持聯繫，同樣地中華人民共和國國務院也在澳門設立[[中央人民政府駐澳門特別行政區聯絡辦公室|駐澳門特別行政區聯絡辦公室]]來與澳門特別行政區政府進行商談。而香港特別行政區政府與澳門特別行政區政府則為了能夠與中華人民共和國國務院建立雙向溝通管道，分別在北京市設立了[[香港特別行政區政府駐北京辦事處|香港特別行政區政府駐北京辦事處]]與[[澳門特別行政區駐北京辦事處|澳門特別行政區駐北京辦事處]]作為對口。而由於中華人民共和國基本法的性質使得其最終解釋權由全國人民代表大會常務委員會擁有，而其下還分別設立了[[香港特別行政區基本法委員會|香港特別行政區基本法委員會]]以及[[澳門特別行政區基本法委員會|澳門特別行政區基本法委員會]]以提供相關意見。而《中華人民共和國香港特別行政區基本法》第一百五十八條則指出，如果香港法院審理案件時需要對基本法有關中央人民政府管理事務或者是中央人民政府和香港關係的條款進行解釋、而該條款的解釋又影響案件判決，[[香港終審法院|香港終審法院]]應該提請全國人民代表大會常務委員會要求解釋{{NoteTag|《[[中華人民共和國香港特別行政區基本法|中華人民共和國香港特別行政區基本法]]》第一百五十八條：「本法的解釋權屬於全國人民代表大會常務委員會。全國人民代表大會常務委員會授權香港特別行政區法院在審理案件時對本法關於香港特別行政區自治範圍內的條款自行解釋。香港特別行政區法院在審理案件時對本法的其他條款也可解釋。但如香港特別行政區法院在審理案件時需要對本法關於中央人民政府管理的事務或中央和香港特別行政區關係的條款進行解釋，而該條款的解釋又影響到案件的判決，在對該案件作出不可上訴的終局判決前，應由香港特別行政區終審法院請全國人民代表大會常務委員會對有關條款作出解釋。如全國人民代表大會常務委員會作出解釋，香港特別行政區法院在引用該條款時，應以全國人民代表大會常務委員會的解釋為準。但在此以前作出的判決不受影響。全國人民代表大會常務委員會在對本法進行解釋前，徵詢其所屬的香港特別行政區基本法委員會的意見。」}}<ref>{{cite web |url = http://www.octs.org.hk/c_page_BasicLaw.htm |title = 中華人民共和國香港特別行政區基本法 |author =  |date =  |publisher = [[一國兩制研究中心|一國兩制研究中心]] |language = zh-hant |accessdate = 2014-07-23 |quote =  }}</ref>。

而根據[[中華人民共和國香港特別行政區基本法|中華人民共和國香港特別行政區基本法]]的規定，有關特別行政區的外交以及國防事務由[[中華人民共和國中央人民政府|中華人民共和國中央人民政府]]負責{{NoteTag|《[[中華人民共和國香港特別行政區基本法|中華人民共和國香港特別行政區基本法]]》第十三條：「中央人民政府負責管理與香港特別行政區有關的外交事務。中華人民共和國外交部在香港設立機構處理外交事務。中央人民政府授權香港特別行政區依照本法自行處理有關的對外事務。」}}{{NoteTag|《[[中華人民共和國香港特別行政區基本法|中華人民共和國香港特別行政區基本法]]》第十四條：「中央人民政府負責管理香港特別行政區的防務。香港特別行政區政府負責維持香港特別行政區的社會治安。中央人民政府派駐香港特別行政區負責防務的軍隊不干預香港特別行政區的地方事務。香港特別行政區政府在必要時，可向中央人民政府請求駐軍協助維持社會治安和救助災害。駐軍人員除須遵守全國性的法律外，還須遵守香港特別行政區的法律。駐軍費用由中央人民政府負擔。」 }}。其中在外交談判或者是僅限國家參加的國際組織與會議可能直接影響特別行政區時，特別行政區政府可以派遣代表加入中華人民共和國代表團。另一方面，中華人民共和國政府也在兩個特別行政區分別派遣[[中國人民解放軍駐香港部隊|中國人民解放軍駐香港部隊]]與[[中國人民解放軍駐澳門部隊|中國人民解放軍駐澳門部隊]]，而其費用則由中華人民共和國中央人民政府承擔。不過在特別行政區對外事務部分，中華人民共和國中央人民政府則授權特別行政區政府得以依照基本法內容自行處理相關事務。其中在不限國家參與的情況時，特別行政區則可以使用「中國香港」（Hong Kong, China）和「中國澳門」（Macau, China）的名義參與國際組織、國際事務與體育競賽<ref>{{cite web |url = http://www.fmcoprc.gov.hk/eng/zcjs/yglz/t17768.htm |title = What are the main articles in the Basic Law concerning Hong Kong's foreign affairs? |author = |date = 2000-11-15 |publisher = [[中華人民共和國外交部駐香港特別行政區特派員公署|中華人民共和國外交部駐香港特別行政區特派員公署]] |language = en |accessdate = 2014-07-23 |quote =  }}</ref>。例如香港特別行政區和澳門特別行政區便由於作為獨立的經濟實體而同為[[世界貿易組織|世界貿易組織]]的會員，同時香港也是[[亞太經濟合作會議|亞太經濟合作會議]]的經濟體成員之一，不過澳門在亞洲太平洋經濟合作會議中則沒有決策權<ref>{{cite web |url = http://www.economia.gov.mo/zh_TW/web/public/pg_eetr_ieto_apec?_refresh=true |title = 亞太經濟合作組織(APEC) |author = |date = 2014-03-05 |publisher = [[經濟局_(澳門)|經濟局]] |language = zh-hant |accessdate = 2014-07-23 |quote = }}</ref>。

}}</ref>。例如2003年[[香港特別行政區基本法第二十三條|香港特別行政區基本法第二十三條]]準備立法時，便被擔憂{{By whom}}可能是為了大規模打擊反對黨並且破壞自主權<ref>{{Cite web |url = http://www.aisixiang.com/data/67223.html |title = 陈弘毅：一国两制的法治实践：十年的回顾与反思 |author = [[陳弘毅|陳弘毅]] |date = 2013-08-31 |publisher = [[愛思想|愛思想]] |language = zh-hans |accessdate = 2014-07-23 |quote = }}</ref>。2013年3月時，[[全國人民代表大會法律委員會|全國人民代表大會法律委員會]]主任委員[[喬曉陽|喬曉陽]]提出的「管理香港的人不能與中央對抗，不能企圖推翻中國共產黨的領導」論點，也引來香港部分民主派人士的批評<ref>{{Cite web |url = http://www.bbc.co.uk/zhongwen/trad/china/2013/03/130328_qiao_xiaoyang_speech.shtml |title = 喬曉陽言論嚴重打擊香港自治|author = |date = 2013-03-28 |publisher = [[BBC中文網|BBC中文網]] |language = zh-hant |accessdate = 2014-07-23 |quote = }}</ref>。

2014年6月10日，[[国务院新闻办公室|中華人民共和國國務院新聞辦公室]]發表的《[[一國兩制白皮書|“一國兩制”在香港特別行政區的實踐]]》白皮書中重申香港的高度自治權来源于中央授权，并强调“中央拥有对香港特别行政区的全面管治权，既包括中央直接行使的权力，也包括授权香港特别行政区依法实行高度自治。对于香港特别行政区的高度自治权，中央具有监督权力。”<ref>{{Cite web |url = http://www.gov.cn/xinwen/2014-06/10/content_2697833.htm |title = 《“一國兩制”在香港特別行政區的實踐》白皮書 |author = [[國務院新聞辦公室|國務院新聞辦公室]] |date = 2014-06-10 |publisher = [[中華人民共和國中央人民政府門戶網站|中華人民共和國中央人民政府門戶網站]] |language = zh-hant |accessdate = 2014-07-23 |quote =  }}</ref>这引來部分香港反对派人士的批評和質疑，認為中國共產黨領導高層是背棄一國兩制的承諾，進而破壞在中華人民共和國統治下香港實施民主自治的政策<ref>{{Cite web |url = http://sinosphere.blogs.nytimes.com/2014/06/11/beijings-white-paper-sets-off-a-firestorm-in-hong-kong/?_php=true&_type=blogs&_r=0 |title = Beijing’s ‘White Paper’ Sets Off a Firestorm in Hong Kong |author = Alan Wong |date = 2014-06-11 |publisher = 《[[紐約時報|紐約時報]]》 |language = en |accessdate = 2014-07-23 |quote = }}</ref>。例如[[香港大學|香港大學]][[香港大學專業進修學院|專業進修學院]]人文及法律學院助理教授賈廷思表示難以就「監督權力」評論。被問到有沒有任何法律條文提及白皮書所謂「監督權力」，他表示若要辯論此問題，对方或會辩解《基本法》第12條暗示了有關權力，但賈認為該條文絕對沒有作此表述。公民黨法律界議員[[郭榮鏗|郭榮鏗]]指，白皮書所謂「監督權力」之說沒有明確法律基礎，強調白皮書不能取代《基本法》成為憲法一部份，也不能作為解釋《基本法》的文件<ref>[http://hk.apple.nextmedia.com/news/art/20140611/18753598 未有明確法律基礎　僭建中央監督權力]，苹果日报 2014-06-11{{zh-hk}}</ref>。

就此问题，中国内地的一些法学专家强调，香港特别行政区的高度自治权来源于中央授权，并非香港固有，中央与香港不是分权体制，不存在所谓“剩余权力”问题，希望香港的反对派人士理性面对中央与香港的关系<ref>{{cite web |url=http://www.chinadaily.com.cn/hqgj/jryw/2014-06-23/content_11882510.html |title=法学专家：香港不存在所谓“剩余权力”，中国日报网，2014-06-23 |publisher=Chinadaily.com.cn |date= |accessdate=2017-05-23 |archive-url=https://web.archive.org/web/20160316010945/http://www.chinadaily.com.cn/hqgj/jryw/2014-06-23/content_11882510.html |archive-date=2016-03-16 |dead-url=yes }}</ref><ref>{{cite web|author=作者：张定淮 |url=http://opinion.china.com.cn/opinion_34_101034.html |title=“剩余权力论”的荒谬，中国网，2014-06-11 |publisher=Opinion.china.com.cn |date=2014-06-11 |accessdate=2017-05-23}}</ref>。此前有内地學者研究認爲，1986年初，在《基本法》起草过程中，港方起草委员[[李柱铭|李柱铭]]等人提出的“剩余权力”问题，後來並沒有被《基本法》承認。<ref>李元起、黄若谷，论特别行政区制度下的“剩余权力”问题，北方法学2008年第2期</ref>

而对于白皮书抛出的对自治层拥有指挥权的概念，《[[明报|明报]]》不署名评论指若中央依法直接對港行使管治權，香港與其他中國城市并无分別，而在白皮書中出现中央人民政府「向行政長官發出指令的權力」的說法，這個權力如何操作、由哪一個部門向特首發號施令未有明言，评论认为體現高度自治的特首變成受令於中央，客觀上特首的管治威信勢必大打折扣<ref>[https://news.mingpao.com/pns/%E9%A6%99%E6%B8%AF%E6%B7%AA%E7%82%BA%E6%99%AE%E9%80%9A%E5%9F%8E%E5%B8%82%EF%BC%8C%E5%B0%8D%E5%9C%8B%E5%AE%B6%E6%9C%89%E4%BB%80%E9%BA%BC%E5%A5%BD%E8%99%95%EF%BC%9F/web_tc/article/20140611/s00003/1402422492394 香港淪為普通城市，對國家有什麼好處？]，[[明報|明報]] 2014-06-11{{zh-hk}}</ref>。

對中國中央政府來說，一國兩制是一個整體，因此它不能僅僅對香港好，還是中國現代化、國際化整體戰略方案的一部分。起草《基本法》時以保障香港法制優先，但中央保留[[全國人大常委會解釋基本法|人大釋法]]條文（《基本法》第158條）<ref name="01周報"/>。

2015年9月11日，中聯辦主任[[張曉明|張曉明]]出席《基本法》頒布25周年研討會時就香港政制表示，草擬基本法時的主導思想，是以特區政府作為核心的主要體制，「行政長官具有超然於行政、立法、司法三個機關之上的特殊法律地位」。張曉明於研討會發表致謝辭說，香港於回歸前及回歸後，也沒有實行過任何三權分立的政治體制。他說，「作為領導特區政府的行政長官，他所具有的雙首長身分和雙負責制的責任，使得行政長官具有超然於行政、立法、司法的三個機關之上的特殊法律地位，處於特別行政區權力運行的核心位置，在中央政府之下，特別行政區的三權之上起着連結的樞紐作用」。<ref>{{cite web|title =張曉明「行政長官具有超然於行政、立法、司法三個機關之上的特殊法律地位」|url =http://news.mingpao.com/pns/%E5%BC%B5%E6%9B%89%E6%98%8E%EF%B9%95%E7%89%B9%E9%A6%96%E8%B6%85%E7%84%B6%E4%B8%89%E6%AC%8A%E4%B9%8B%E4%B8%8A%20%E6%9D%8E%E6%9F%B1%E9%8A%98%EF%BC%9A%E6%B8%AF%E8%A1%8C%E6%99%AE%E9%80%9A%E6%B3%95%20%E5%B1%AC%E4%B8%89%E6%AC%8A%E5%88%86%E7%AB%8B/web_tc/article/20150913/s00001/1442080836494|publisher =明報|date =2015-09-13|deadurl =yes|archiveurl =https://web.archive.org/web/20160822230609/http://news.mingpao.com/pns/%E5%BC%B5%E6%9B%89%E6%98%8E%EF%B9%95%E7%89%B9%E9%A6%96%E8%B6%85%E7%84%B6%E4%B8%89%E6%AC%8A%E4%B9%8B%E4%B8%8A%20%E6%9D%8E%E6%9F%B1%E9%8A%98%EF%BC%9A%E6%B8%AF%E8%A1%8C%E6%99%AE%E9%80%9A%E6%B3%95%20%E5%B1%AC%E4%B8%89%E6%AC%8A%E5%88%86%E7%AB%8B/web_tc/article/20150913/s00001/1442080836494|archivedate =2016-08-22}}</ref>

有關言論引發巨大爭議，多位政界人士對此作出評論。9月14日，[[香港大律師公會|香港大律師公會]]發表聲明，對張曉明的演講內容的討論表示深切關注。聲明提到，基本法清晰界定了三權的關係和權責。大律師公會堅信「[[權力分立|權力分立]]」的法律概念在基本法底下得到充分的落實。張曉明「超然於三個機關之上」的描述，難免被解讀為特首「凌駕於三權之上」，引起公眾憂慮，大律師公會對此深表遺憾。大律師公會又促請張曉明及律政司司長早日作出澄清，「消除香港市民以及國際社會對香港在一國兩制下秉行法治的疑慮」。<ref>{{cite web|title=香港大律師公會就張曉明主任在九月十二日《香港特別行政區基本法》頌布25周年研討會的演講內容的聲明|url=http://www.hkba.org/whatsnew/press-release/20150914%20-%20Press%20Statement%20of%20HKBA%20-%20Chinese.pdf|publisher=香港大律師公會|accessdate=2015-10-10|deadurl=yes|archiveurl=https://web.archive.org/web/20160306062916/http://www.hkba.org/whatsnew/press-release/20150914%20-%20Press%20Statement%20of%20HKBA%20-%20Chinese.pdf|archivedate=2016-03-06}}</ref>

同日，23名泛民立法會議員以[[泛民會議|泛民會議]]名義發表聯合聲明，引用基本法三項條文，說明特首權力受多重制衡，因此超然一說是無中生有。泛民會議表示希望張曉明能「尊重『一國兩制』及《基本法》，勿再發表損害香港核心價值的言論」，又促請梁振英「公開重申會遵守本地法律，並按基本法條文接受立法與司法的制衡」。<ref>{{cite web|title=泛民會議就中聯辦主任張曉明「特首超然論」的聲明|url=http://www.ipkinyuen.org.hk/file_7nevts/newEvts_20150914.htm|publisher=立法會葉建源議員辦事處|accessdate=2015-10-10}}</ref>9月15日，[[梁振英|梁振英]]在政府總部會見記者時，評論了張曉明的演說內容，指張曉明在演說中並無使用「凌駕」字眼，有人沒在現場聽而作出批評，出現斷章取義或誤導的情況。<ref>{{cite news|title=為張曉明超然論辯解 梁振英斥外界斷章取義|url=http://www.am730.com.hk/article-281467|newspaper=AM730|date=2015-09-16|accessdate=2015-10-10}}</ref><ref>{{cite news|title=梁振英：張曉明「香港特首超然」論遭曲解 |url=http://www.bbc.com/zhongwen/trad/china/2015/09/150915_hongkong_two_system|newspaper=BBC|date=2015-09-15|accessdate=2015-10-10}}</ref>9月16日，梁振英又對傳媒表示行政長官地位「確實是超然」。<ref>{{cite news|title=香港特首梁振英：特首地位「確實是超然」|url=http://www.bbc.com/zhongwen/trad/china/2015/09/150916_hongkong_zhang_xiaoming_responses|newspaper=BBC|date=2015-09-16|accessdate=2015-10-10}}</ref>9月20日，[[香港律師會|香港律師會]]就張曉明的言論發表聲明，重申兩個基本原則：「司法獨立對法治的重要」和「香港司法機構的專業及其制度健全」。<ref>{{cite web|title=香港律師會的聲明|url=http://www.hklawsoc.org.hk/pub_c/news/press/20150920.asp|publisher=香港律師會|accessdate=2015-10-10|deadurl=yes|archiveurl=https://archive.is/20151010093936/http://www.hklawsoc.org.hk/pub_c/news/press/20150920.asp|archivedate=2015-10-10}}</ref>
