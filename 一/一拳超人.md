{{日本作品|一拳超人|ワンパンマン|ONE-PUNCH MAN}}，是日本網路漫畫家[[ONE_(漫畫家)|ONE]]的幽默格鬥[[漫畫|漫畫]]，2009年初開始網絡發佈，系列迅速傳播，2012年6月超過790萬次點閱。「重製版」（作画：[[村田雄介|村田雄介]]）《Young Jump Web Comics》（集英社）於2012年6月14日連載中。由[[MADHOUSE|MADHOUSE]]製作的[[電視動畫|電視動畫]]自2015年10月至12月期間在日本播出。於2019年4月開始播放改由[[J.C.STAFF|J.C.STAFF]]製作的第二季動畫。

== 劇情大綱 ==
{{Expand section||time=2019-03-17T01:05:12+00:00}}
故事發生於一個類似於地球，但有著超能力與各種奇怪生物的星球。各種具有超自然力量的「怪人」不斷地出現並造成災難，為了打擊它們，由某富豪所創立的「英雄協會」開始招募全世界的能人異士成為職業英雄與怪人對抗。

故事主角'''埼玉'''便是一名英雄，他具有無敵的力量，但因為能力太過大，卻沒了當初能打敗怪人那激昂的情緒，只有在他遇見有挑戰性的對手時才能興奮起來。在一連串事件中，埼玉遇見了各式各樣的英雄、反派和怪人，收了改造人'''傑諾斯'''做為徒弟，並加入英雄協會以獲得官方認可。與眾多英雄一起開始了對抗各種怪人以及災難的生活，但更多時是在無人知曉的情況下做英雄的工作。

== 作品設計 ==
該作品原本是作者為了練習一個漫畫原稿軟件[[ComicStudio|ComicStudio PC]] ，參考《[[麵包超人|麵包超人]]》的角色設定完成第一話，於2009年7月3日上傳至他的個人網頁（[[FC2|FC2]]）並不定期更新，雖然畫功粗糙，但由於幽默的設定和相當精彩故事，因此於日本網路大熱。根據日本[[NHK|NHK]]2012年9月2日播出的「網路漫畫革命」調查，《一拳超人》於該年觀看總數超過1000萬次。

=== 重製版 ===
2012年初，漫畫家[[村田雄介|村田雄介]]（代表作品：《[[光速蒙面俠21|-{zh-hans:光速跑者21号;zh-tw:光速蒙面俠21;zh-hk:衝鋒21}-]]》）在[[Twitter|Twitter]]上與[[ONE_(漫畫家)|ONE]]討論合作創作事宜<ref>{{cite web|url=http://bizmash.jp/articles/38287.html|title=「ONE 」というWebコミック界のエース（前編）|archive-url=https://web.archive.org/web/20130110052111/http://bizmash.jp/articles/38287.html|archive-date=2013年1月10日|access-date=2019年3月17日|language=ja}}</ref>。2012年6月14日，村田雄介以原作為藍本創作「一拳超人 重製版」，英文名為「ONE PUNCH MAN re-make」，於[[集英社|集英社]]的網上漫畫雜誌《Young Jump Web Comics》<ref>{{cite web|url=https://tonarinoyj.jp/|title=となりのヤングジャンプ|archive-url=https://web.archive.org/web/20121230145537/http://tonarinoyj.jp/|archive-date=2012年12月30日|access-date=2019年3月17日|language=ja}}</ref>上連載。

集英社於2012年12月開始為「一拳超人﹣重製版」發行[[單行本|單行本]]。由[[東立出版社|東立出版社]]取得台灣代理<ref>{{cite web|url=http://isbn.ncl.edu.tw/NCL_ISBNNet/Ultra_MessageTrace.php?PHPSESSID=h1bfbl2u4e3umbm0n0c6u7fbh6&Pact=view&Ppub=P00039&Pkey=2400094017&KeepThis=true&TB_iframe=true&width=880&height=400|title=出版單位：東立出版社有限公司，ONE PUNCH MAN 一拳超人，ONE原作/村田雄介漫畫/吳勵誠譯|access-date=2019年3月17日}}</ref>，香港代理則由[[文化傳信|文化傳信]]奪得。

大部分镜头中角色都经过大幅美化，加入許多細節，場面更有魄力，以及村田將幾個分鏡特別修改後的跨頁圖。連載中也加入一些原創回及特別篇，令一拳超人的設定及世界觀得以擴充以及更加合理（如介紹協會的一些原作未提及的運作、執法人員如警察的立場，一些關於埼玉的性格和過去等），所有新增內容都有與原作One討論過。

2015年3月12日15號Young Jump公開TV動畫化，並在同年的3月21日至3月22日開展的「Anime Japan 2015」公開詳細情報。7月23日電視動畫的官方網站正式宣布播出時間和播放電視台，將從10月開始在[[東京電視台|東京電視台]]等台播出。台灣由[[木棉花國際|木棉花國際]]官網正式宣布與日本同步播出。

除此之外村田與One也有在Jump合作短篇《彈丸天使粉絲俱樂部》及《怒濤的勇者們》。其中埼玉還有在《勇者們》穿越。
