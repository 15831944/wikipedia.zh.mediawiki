'''一氧化鍺'''（GeO），是[[鍺|鍺]]與[[氧|氧]]的一種化合物。把[[二氧化鍺|二氧化鍺]]與鍺加熱至1000℃高溫後，可得一黃色昇華物，即一氧化鍺。黃色的昇華物加熱至600℃時會轉成棕色<ref name = "Greenwood">{{cite book
 |last1=Greenwood |first1=Norman  N.
 |last2=Earnshaw |first2=Alan
 |year=1997
 |title=Chemistry of the Elements
 |edition=2nd
 |publisher=Butterworth–Heinemann
 |isbn=0080379419
}}</ref>。對於一氧化鍺具體特性的研究仍有待加強<ref name = "Greenwood"/> 。它是一種[[兩性_(化學)|兩性]]化合物：溶於酸中生成亞鍺酸鹽，溶於鹼中則生成含Ge(OH)<sub>3</sub>離子的“三水鍺酸鹽”或“鍺石”<ref name = "Wiberg&Holleman">Egon Wiberg, Arnold Frederick Holleman (2001) ''Inorganic Chemistry'', Elsevier ISBN 0-12-352651-5</ref>。
