'''一氧化鉛'''，化學式'''PbO'''，俗稱'''鉛黃'''、'''黃丹'''、'''密陀僧'''。為[[兩性氧化物|兩性氧化物]]，常溫時為淡黃色[[結晶|結晶]]粉末。

一氧化鉛在[[唐代|唐代]]由[[波斯|波斯]]传入中国，密陀僧就是[[波斯语|波斯语]]mirdasang的对音，当时用作治疗[[痔疮|痔疮]]的药物。<ref>（美）爱德华·谢弗.《唐代的外来文明》</ref>

== 结构 ==
一氧化铅有两种[[晶型|晶型]]，α-PbO为红色晶体，四方晶系，密度9.35g/cm<sup>3</sup>。β-PbO为黄色晶体，正交晶系，密度为9.64g/cm<sup>3</sup>。β-PbO在水中煮沸可以得到α-PbO<ref>周为群 朱琴玉. 普通化学（第二版）. 苏州大学出版社,2014.9. 第六章 重要元素及化合物. pp.201. ISBN 978-7-5672-1076-9</ref>，在常温下研磨β-PbO也能得到α-PbO；α-PbO在475~583℃转变为β-PbO。<ref name=z />
== 製備 ==
在空氣中加熱[[铅|铅]]、[[硝酸铅|硝酸铅]]、[[硫酸铅|硫酸铅]]、[[硫化铅|硫化铅]]都可以製得一氧化铅。反應式：
:2 Pb + O<sub>2</sub> → 2 PbO
:2 Pb(NO<sub>3</sub>)<sub>2</sub> → 2 PbO + 4 NO<sub>2</sub>↑ + O<sub>2</sub>↑
高纯的一氧化铅通常则以[[乙酸铅|乙酸铅]]为原料，用浓[[氨水|氨水]]处理后加热分解得到。<ref name=z>朱文祥. 无机化合物制备手册. 化学工业出版社,2006. pp581-582. 【XIV-368】氧化铅 [Lead(II) oxide]</ref>
== 性质 ==
PbO易溶于[[醋酸|醋酸]]。
== 用途 ==
可用於製造[[玻璃|玻璃]]、[[琺瑯|琺瑯]]、[[釉藥|釉藥]]以及其他[[:Category:铅化合物|鉛化合物]]。
