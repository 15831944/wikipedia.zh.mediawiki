《'''一人之下'''》（原名《'''异人'''》），是米二创作的[[中国大陆|中国大陆]][[网络漫画|网络漫画]]，改编动画由[[日本|日本]]动画公司Pandanium负责制作。2017年宣布於10月27日正式推出一人之下the outcast動畫第二季《一人之下 羅天大醮篇》，於中國地區[[腾讯视频|腾讯视频]]、[[bilibili|bilibili]]、[[腾讯动漫|腾讯动漫]]、[[爱奇艺|爱奇艺]]每周五网络平台首播,第二季的动画制作更换为[[上海绘界文化传播有限公司|上海绘界文化传播有限公司]]。

==故事簡介==
随着爷爷尸体被盗，神秘少女冯宝宝的造访，少年张楚岚的平静校园。急于解开爷爷和自身秘密的张楚岚和没有任何记忆“不死少女”冯宝宝开启了“异人”之旅……

==人物介绍==
; 张楚岚
: 聲：（普通话）[[夏侯落枫|夏侯落枫]]；（日语）[[田丸笃志|田丸笃志]]、[[饭田友子|饭田友子]]（幼年）
: 虽然脾气暴躁，但是是非常普通的大学生。小时候被祖父张锡林命令「要过普通人的生活」。但为了解开祖父死亡的谜题，开始使用超能力踏入了“异人”们的世界。<ref name=character>{{cite news|url=http://cn.hitorinoshita.com/character|title=人物介绍}}</ref>
; 冯宝宝
: 聲：（普通话、[[四川话|四川话]]）[[小连杀|小连杀]]；（日语）[[早见沙织|早见沙织]]
: 对于“气”的操纵非常的拿手，身体的能力也相当异于常人。因为不谙世事也完全不懂表达情感的方法，所以脸上没什麽表情。<ref name=character></ref>说话十分直接，不会像正常人一样思前想后，甚至不能理解正常人的道德规则。在徐四的教导下，信奉"能动手尽量不动嘴"，劝人和表达感情的方式基本上就是往死里揍。
; 夏禾

; 徐三
: 聲：（普通话）[[藤新|藤新]]；（日语）[[野岛健儿|野岛健儿]]
: 哪都通快递干部，徐四之兄。先天异人，能力为念动力。
; 徐四
: 聲：（普通话）[[图特哈蒙|图特哈蒙]]；（日语）[[立花慎之介|立花慎之介]]
: 现任华北大区负责人，徐三之弟，直接负责张楚岚和冯宝宝。暴力，痞气，教给冯宝宝很多奇怪的东西，并将冯宝宝的功夫命名为阿威十八式(电光毒龙钻、中出、冈本零点零一等等)。
; 风正豪
: 天下集团(天下会)的创立者，风莎燕、风星潼之父，十佬之一。先人是凉山大觋(习)，掌握八奇技之一拘灵遣将，但并不完整。十分看好张楚岚，真心赞赏其行为，能够看出来张楚岚一些流氓行为背后的用心，曾鼓励风莎燕和张楚岚交往。
: 拥有枭雄的心性，在得知风家与王家的过往后，表面对王家感恩戴德，甘愿当王家的狗，实则忍辱负重，另有谋划。

: 与夏禾有一段不为人知的往事，因而在修炼雷法前失去了童子之身，无法修炼阳五雷，只能修炼阴五雷。看似对夏禾十分憎恨，但貌似对其有旧情而下不了手。奉老天师之命下山邀请张楚岚参加罗天大醮，厌恶"肮脏"的自己，对练成阳五雷的张楚岚心存嫉妒，后在与张楚岚的较量中解开心结，接受了真实的自己并与张楚岚和解。天真善良到了一定境界，待人谦和有礼，非常尊敬老天师，被老天师评价为太过阳光而难以把阴五雷的阴毒发挥到化境。也许也是因为这个原因，张天师才让其继承通天箓。
