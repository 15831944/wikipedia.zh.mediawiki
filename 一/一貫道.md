'''一貫道'''是[[中國民間宗教|中國民間宗教]]、[[新興宗教|新興宗教]]<ref>{{Cite book|url=https://hdl.handle.net/11296/np242x|title=從壓迫到擁抱：一貫道與國民黨|last=吳鳳章|first=|publisher=[[一貫道天皇學院|一貫道天皇學院]]|year=2017|isbn=|location=|pages=}}</ref>，又名'''天道教'''，[[羅教|羅教]]系宗教，主要信仰[[無極老母|無極老母]]，以發揚[[儒家思想|儒家思想]]、實踐中國傳統倫理道德為其宗旨。一貫道的教義自稱融合[[儒教_(宗教)|儒]]、[[佛教|釋]]、[[道教|道]]、[[基督宗教|耶]]、[[伊斯蘭教|回]]五教，也有借竅臨壇（[[扶乩|扶乩]]）等類[[道術|道術]]儀式。該教信徒稱為「道親」，彼此間自稱「後學」。<ref name=":0">{{Cite web|url=http://www.3fo.org/article/article.jsp?index=10&item=66&bookid=2c907d4945f411dc0145f4a891090004&ch=1&se=11&f=1|title=佛教與一貫道|accessdate=2018-09-21|author=星雲|date=|authorlink=釋星雲|publisher=}}</ref>

其主神無極老母是創造天地者、萬靈主宰者，全名為「明明上帝無量清虛至尊至聖三界十方萬靈真宰」，一般以「先天母字」符號代表之。教義將人類歷史分成青陽、紅陽、白陽三期，無極老母派遣[[彌勒佛|彌勒佛]]（轉世為祖師[[路中一|路中一]]）拯救凡間。亦有[[濟公|濟公]]（轉世為祖師[[張天然|張天然]]）、[[南海古佛|南海古佛]]、[[关羽|關法律主]]、[[吕洞宾|呂法律主]]、[[孔子|孔子]]、[[穆罕默德|穆罕默德]]、[[耶稣|耶稣]]等仙佛神聖。主要儀軌典籍是[[中華民國_(大陸時期)|民國年間]]由張天然頒訂的《暫定佛規》，信徒應祈禱、懺悔、[[茹素|茹素]]並修行「三寶」獲得解脫。<ref name=":0" />

該教原本僅在[[山东省|山东省]]流傳，[[中華民國_(大陸時期)|民國年間]]，張天然將之傳佈開至全中國大陸，後來在[[文化大革命|文化大革命]]時期銷聲匿跡。1946年傳入[[臺灣|臺灣]]，[[戒嚴時期|戒嚴時期]]被政府取締，1987年才被允許合法立案。<ref name=":0" />今日[[臺灣|臺灣]]的一貫道信徒約有80萬人<ref name=":22">{{Cite news|url=https://www.ettoday.net/news/20180306/1124854.htm|title=一貫道學院超強！註冊率92.86%、人均經費又贏台清交成|date=2018-03-06|newspaper=ETtoday新聞雲|accessdate=2018-10-13|language=zh-TW}}</ref>。

一貫道相信回歸「道」是各宗教的共同目標，「道」也可以透過各種形式顯現，故世上五大宗教[[儒教_(宗教)|儒]]、[[佛教|釋]]、[[道教|道]]、[[基督宗教|耶]]、[[伊斯蘭教|回]]其實是殊途同歸。一貫道也尊[[伏羲氏|伏羲氏]]、[[黄帝|黃帝]]、[[孔子|孔子]]等中國上古先賢及[[漢傳佛教|漢傳佛教]]祖師[[惠能|惠能]]等人為其祖師。<ref>{{Cite web|url=http://www.gses.ylc.edu.tw/~history/ch5/5-1-1-5.htm|title=一貫道|accessdate=2018-05-30|author=|date=|publisher=雲林縣西螺鎮廣興國民小學}}</ref>

== 名稱 ==
一貫道的名稱為其第十六代祖[[劉清虛|劉清虛]]取《[[論語|論語]]》〈里仁第四〉中「吾道一以貫之」之義而名。1940年，第18代祖[[張天然|張天然]]復將一貫道易名「天道」，迄今兩名並用。在[[中華民國|中華民國]]境內，以「中華民國一貫道總會」的名義登記立案。<ref>{{Cite web|url=http://www.3fo.org/article/article.jsp?index=198&item=257&bookid=2c907d4944dd5ce70144e285bec50005&ch=11&se=12&f=1|title=新興宗教介紹|accessdate=2018-10-13|author=|date=|last=釋星雲|publisher=|quote=一貫道……名稱則是劉清虛取法《論語‧里仁篇》「參乎！吾道一以貫之」的意義。1940年，十八代祖張天然再將一貫道易名為「天道」。其後，一貫道與天道兩種名稱並用，以迄於今。一九八七年獲內政部核准成立「中華民國一貫道總會」，為合法的宗教團體。|archive-url=https://web.archive.org/web/20181013090659/http://www.3fo.org/article/article.jsp?index=198&item=257&bookid=2c907d4944dd5ce70144e285bec50005&ch=11&se=12&f=1|archive-date=2018-10-13|dead-url=no}}</ref>

== 沿革 ==
{{Main|一貫道歷史}}[[File:Zhang_Tianran.PNG|thumb]]畫像。]]清[[光緒|光緒]]年間，[[先天道|先天道]]信徒[[王覺一|王覺一]]自開支派「先天無生教」，後又改稱「末後一著教」，是一貫道實際創始者。1886年其弟子[[劉清虛|劉清虛]]將教名改為現名。教務後來傳給[[路中一|路中一]]（該教經典敘述是[[彌勒菩薩|彌勒佛]]轉世<ref name=":03">{{Cite web|url=http://www.oneline88.com/show.aspx?id=1503&cid=248|title=一貫道|accessdate=2018-06-18|author=|date=|work=信仰問答彙編}}</ref>），在其逝世後由妹妹路中節暫時掌理。民國19年（1930年），以濟南為傳教根據地的[[張天然|張光璧]]（又名張天然，該教經典敘述為[[济公活佛|济公活佛]]分靈倒裝轉世<ref>[http://www.oneline88.com/show.aspx?id=2017&cid=248 師尊]</ref>）與妾[[孫素貞|孫素貞]]（張的元配劉率貞是陳興海之妹）以[[扶乩|飛鸞]]方式自稱獲最高神祇[[無極老母|無極老母]]認可，領導一貫道。兩人傳道足跡遍布中國，時值日本侵略，大量民眾遂信奉一貫道。[[中華民國_(大陸時期)|民國年間]]，一貫道各地的道堂公益慈善事業做得很多，如施粥、施茶、舍藥、助葬、濟貧、救難等善舉<ref name=":14">一貫道概要-慕禹，ISBN13：9789868010000</ref>。較有名的有「崇華堂慈善會」、「中華慈善道德會」，對慈善事業的推行績效卓著，尤其是對戰時傷患的醫療及難民救助貢獻極大，屢蒙當局的獎勵<ref name=":14" />。

香港的道務主要自中共建政後，離開大陸的孫慧明等人領導發展，逐漸設立的道壇、佛堂使香港與台灣並列，成為一貫道兩大本營。1997年[[香港回歸|香港回歸]]中國後，部分道場移往臺灣和海外發展。

海外的道務為戰後由於[[孙素贞|孙素贞]]師母及許多支線的前人、點傳師、人才避居香港，其地位遂日愈重要，與台灣成為一貫道的兩個大本營，許多國家的道場都由此開出。各道場主要開荒地區有韓國、日本、東南亞、南亞、澳洲、紐西蘭、美國、加拿大、中南美洲、非洲、歐洲等。<ref name="voa"/>許多國家也設有一貫道總會。

== 組線及衍生宗教 ==
一貫道傳來台灣的師母派可分為十八組線，另有師兄派的正义辅导委员会（後來的[[天道|天道]]）。該教也有天道、原[[文化組|文化組]][[王好德|王好德]]開創的[[彌勒大道|彌勒大道]]和[[中华圣教|中华圣教]]、[[天真亥子道|天真亥子道]]等相關衍生宗教。<ref name="zhongping2010" />

== 道義 ==
一貫道主要教義為：「[[道|道]]是創生宇宙萬物的本體、永恆不變的真理，也是返回理天的唯一道路。在人身上稱為性理心法、根本佛性。現今末法時期，世人迷失根本，因此上天降道，挽救善男信女，唯有求道授記，才能超生了死。」以回「無極理天」為目標，並認為「宗教同源」，皆追求回到天堂。

'''考'''是一貫道稱呼試煉與磨難的用詞，被視為是修行的課題。其中，「官考」指的是政府單位的取締。<ref name="Li2014">{{cite journal |author1=李欣霖 |title=一貫道初期在雲彰地區的人文考察—— 以斗六崇修堂為核心之探索 |journal=興大人文學報 |date=2014-09 |issue=第五十三期 |pages=193-234}}</ref>{{rp|200}}

一貫道有以下宗旨<ref>{{Cite web|url=http://www.ikuantao.org.tw/modules/tadbook2/view.php?book_sn=1&bdsn=31#A|title=一貫道的宗旨|accessdate=2018-09-21|author=中華民國一貫道總會|date=|publisher=}}</ref>：
{{Quote|text=敬天地，禮神明，愛國忠事，敦敬崇禮，孝父母，重師尊，信朋友，和鄉鄰，改惡向善，講明五倫八德，闡發五教聖人之奧旨，恪遵四維綱常之古禮，洗心滌慮，借假修真，恢復本性之自然，啟發良知良能之至善，已立立人，己達達人，挽世界為清平，化人心為良賢，冀世界為大同。}}

=== 經典 ===
一貫道雖使用佛教、道教、儒教、基督教、回教等經典，<ref name="voa">{{Cite web|url=https://www.voachinese.com/a/article-20100624-uncut-news-97087734/502686.html|title=对比新闻：一贯道在中国|accessdate=2010-06-24|author=美國之音}}</ref>但另有自己的經典，如《彌勒真經》、以及經過[[扶乩|扶乩]]的方式所產生的訓文等。

1939年，張天然為了規範一貫道各地道壇的儀軌，頒布《暫定佛規》，為其代表性著作。

== 神職人員 ==
經過一貫道的求道儀式者稱為「道親」，指「因道而親」，也是信徒間的互稱<ref name=":02">{{Cite web|url=https://religion.moi.gov.tw/Knowledge/Content?ci=2&cid=516|title=道親|accessdate=2018-07-02|author=李紀勳|date=|publisher=內政部全國宗教資訊網|author2=劉怡君}}</ref>。在該教經典中，以「乾道」稱呼男道親，女性則稱「坤道」。其組織層次分明<ref name=":02" />，神職人員的職級依序可以分為：

* 一、祖師：專指「師尊」[[張天然|張天然]]及「師母」[[孫慧明|孫慧明]]和歷代祖師，包括[[路中一|路中一]]。
* 二、道長：一貫道元老人物，通常是大陸時期各省市領導人，職務由祖師授予。在臺灣對道務傳播有貢獻的張文運、韓雨霖也被當地道親尊為道長。
* 三、老前人：一貫道資深人物，如張文運、韓雨霖等。
* 四、前人：資深的代表點傳師。經祖師授權後，可依其核准的名額來任命點傳師。習慣上，臺灣外省點傳師都稱為前人。
* 五、代表[[點傳師|點傳師]]：常簡稱點傳師，意涵為代表祖師傳道者，1936年前稱代表師，今亦稱經理。
* 六、壇主（堂主）：佛堂負責人。
* 七、講師：講述倫理道德。擔任講師以上的職級，必須發「清口願」。
* 八、辦事人員：佛堂中幫忙協助者。
* 九、三才：借竅時扶乩者為天才，抄字者為地才，報字者稱為人才。
* 十、道親：道親間又可依是否發願茹素分為「求道道親」、「清口道親」。<ref>{{Cite web|url=http://www.oneline88.com/show.aspx?id=1019&cid=178|title=一貫道的職級|accessdate=2018-07-02|author=|date=2016-10-30|publisher=彌勒書院}}</ref>

== 戒律 ==
一貫道的戒律主要來自[[漢傳佛教|漢傳佛教]]。臺灣地區資深教徒多[[素食主義|蛋奶素]]，發「清口願」者則終身茹素。有五戒：殺、盜、淫、妄、酒。

== 佛堂 ==
[[File:Sanbao_Temple_三寶宮.jpg|thumb]][[雲林縣|雲林縣]][[虎尾鎮|虎尾鎮]]的三寶宮即為一貫道佛堂|替代=|200x200像素]]一貫道的禮拜場所為佛堂，又稱佛壇、道場，是一貫道「傳道、禮佛、講經、習儀及道親共修聚會之場所」，也奉祀該教主神無極老母。依據其設置地點、大小，可分為家庭佛堂、公共佛堂和道場。家庭佛堂設於家庭私宅中，清口道親如欲投身道務，可以在前賢{{Efn|參本條目「職級」一節}}允許後設置，便利家庭共同參與道務。公共佛堂、道場多為信徒集資興建，面積不一，道場多大於公共佛堂。<ref name=":1">{{Cite web|url=http://www.ikuantao.org.tw/modules/tadbook2/view.php?book_sn=1&bdsn=34#A|title=一貫道的佛堂|accessdate=2018-09-21|author=中華民國一貫道總會|date=|work=一貫道簡介|publisher=}}</ref>

[[File:Ikuantaoist_oil_lamp.PNG|thumb]]的油灯（母灯）]]
佛堂擺設有佛位、供桌。佛位或書寫無極老母全稱「明明上帝，無量清虛至尊、至聖，三界、十方萬靈真宰」，或僅書寫先天母字、「佛」字，也可能懸掛南海古佛、太上老君或孔子之畫像，亦可能供有彌勒佛、南海古佛、濟公活佛、月慧菩薩、關帝、呂祖等仙佛神聖的塑像。佛位兩邊懸掛對聯，多和教義有關。供桌分為上、下桌，上桌擺放佛燈，代表無極老母；下桌放置香爐，左右亦置佛燈。<ref name=":13">{{Cite web|url=http://www.ikuantao.org.tw/modules/tadbook2/view.php?book_sn=1&bdsn=34#A|title=一貫道的佛堂|accessdate=2018-09-21|author=中華民國一貫道總會|date=|work=一貫道簡介|publisher=}}</ref>

== 文教公益 ==
一貫道创立大專院校[[一貫道天皇學院|一贯道天皇学院]]<ref name=":2">{{Cite news|url=https://www.ettoday.net/news/20180306/1124854.htm|title=一貫道學院超強！註冊率92.86%、人均經費又贏台清交成|date=2018-03-06|newspaper=ETtoday新聞雲|accessdate=2018-10-13|language=zh-TW}}</ref>、[[一貫道崇德學院|一貫道崇德學院]]，主要供神職人員、信徒就读，一贯道天皇学院106学年度招生率达92.86%<ref name=":2" />。位於[[台灣|台灣]][[屏東縣|屏東縣]]之[[屏東縣私立崇華國民小學|屏東縣私立崇華國民小學]]（ 原[[高鳳數位內容學院|高鳳數位內容學院]]）亦為該教所創。
