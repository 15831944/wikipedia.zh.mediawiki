《'''一個故事'''》是[[黎明|黎明]]的國語[[音樂專輯|音樂專輯]]，由[[李宗盛|李宗盛]]製作<ref>{{Cite web |url =http://www.iwant-radio.com/showinformation.php?sisn=12362|title = 黎明『一個故事』專輯介紹|publisher =銀河網路電台|language = 中文}}</ref><ref>{{Cite web|url =http://www.epochtimes.com/b5/5/9/23/n1062058.htm|title =《一個故事》上市 黎明變換歌路脫胎換骨|author =曾歲春|date = 2005年9月23日|publisher =大紀元新聞網|language =中文}}</ref>，在[[北京|北京]]進行為期兩個的錄音，於2005年9月21日年推出。《一個故事》專輯收錄了十一首國語歌曲<ref>{{Cite web |url =https://www.kkbox.com/hk/tc/album/oe4J8N3AmP178P40FZE0008l-index.html|title = 黎明(Leon Lai)1個故事-特別版專輯|publisher =KKbox|language = 中文}}</ref>，主打歌曲包括《有情郎》、《赤腳男人的山谷》、《別來無恙》、《Good Day》。2005年10月，推出《一個故事》全新特別版，加錄了《Crazy Classic》演唱會主題曲「Crazy Classic」之廣東歌前奏版，加輯《赤腳男人的山谷》、《Good Day》和《別來無恙》的MV，以及《一個故事》的製作花絮。<ref>{{Cite web |url =https://www.yesasia.com/global/%E4%B8%80%E5%80%8B%E6%95%85%E4%BA%8B-%E5%85%A8%E6%96%B0%E7%89%B9%E5%88%A5%E7%89%88/1004076157-0-0-0-zh_TW/info.html|title =一個故事(全新特別版)鐳射唱片-黎明,東亞唱片-國語音樂|publisher =YsaAsia|language = 中文}}</ref>。
