'''一號九龍道'''（{{lang|en|'''Madison Park'''}}），是位於[[香港|香港]][[九龍|九龍]][[深水埗|深水埗]]<ref>根據市建局官方定為深水埗，[https://www.ura.org.hk/tc/project/redevelopment/dl-4-kln-rd-kiu-yam-st 九龍道/僑蔭街項目]</ref>[[九龍道|九龍道]]的單幢住宅，樓高共24層，由[[市區重建局|市區重建局]]、[[信和置業|信和置業]]發展，[[信和物業管理|信和物業管理]]管理，預計於[[2020年|2020年]][[11月|11月]]入伙。項目於[[2018年8月|2018年8月]]獲批[[預售樓花同意書|預售樓花同意書]]，同月上載[[售樓說明書|售樓說明書]]及公怖首50伙售價，包括1房40伙及2房10伙單位，折實售價由557.4萬至829.8萬元，平均呎價18,517元至21,901元<ref>[https://inews.hket.com/article/2144467/%E9%95%B7%E6%B2%99%E7%81%A3%E4%B8%80%E8%99%9F%E4%B9%9D%E9%BE%8D%E9%81%93%E5%B8%82%E5%83%B9%E6%8E%A8%201%E6%88%BF%E6%8A%98%E5%AF%A6557%E8%90%AC 長沙灣一號九龍道市價推 1房折實557萬]</ref>，及後同月再推30伙，結果累積收票逾500票，超額逾5倍<ref>[https://hk.finance.appledaily.com/finance/realtime/article/20180828/58617550 一號九龍道截收逾500票　超額5倍]</ref>，項目終在8月下旬開售首日共沽出55伙或69%，套現近3.5億元<ref>[https://hk.finance.appledaily.com/finance/daily/article/20180830/20487721 一號九龍道首批僅沽69%]</ref>。至10月上旬項目累售65伙，共套現約4.3億<ref>[https://ps.hket.com/article/2177320/%E4%B8%80%E8%99%9F%E4%B9%9D%E9%BE%8D%E9%81%93%E7%B4%AF%E5%94%AE65%E4%BC%99 一號九龍道累售65伙]</ref>。而[[示範單位|示範單位]]設於[[奧海城#3期商場|奧海城3期]]。

== 歷史 ==
大廈前身為1幢建於[[1959年|1959年]]，位於「[[九龍街|九龍街]]1-3B號及[[僑蔭街|僑蔭街]]1-5號」樓高8層的[[唐樓|唐樓]]<ref>[https://news.now.com/home/local/player?newsId=64985&refer=%20 深水埗重建項目可提供二百多個單位]</ref>，地盤面積約599[[平方米|平方米]]，為[[市區重建局|市區重建局]]第二輪[[市區重建局#需求主導重建項目|需求主導重建項目]]之一；[[2013年4月|2013年4月]]，市區重建局公佈共收到34份第二輪需求主導申請書，當中選取四個完全符合計劃先決條件的項目，納入市建局2013/2014業務計劃內，並得到[[財政司司長|財政司司長]]批准，而九龍道/僑蔭街項目則為兩個首批公布項目之一<ref>[https://www.ura.org.hk/tc/media/press-release/20130412 市建局啟動第二輪「需求主導」先導計劃首兩個重建項目]</ref>。

[[2013年6月|2013年6月]]，市建局向該項目約79個業主發出「有條件收購建議」，以相等於類似地區一個大約七年樓齡的假設重置單位的呎價作收購，即每平方呎[[實用面積|實用面積]]9,857元，業主須於75日考慮和接受有關收購建議，如成功獲得上述八成業權的業主接納和得到政府授權進行，及後沒有收到任何上訴或所有上訴已被駁回後，市建局便會與業主完成收購程序和發放補償。否則項目便會終止<ref>[https://www.ura.org.hk/tc/media/press-release/20130627 市建局向第二輪兩個「需求主導」重建項目發出「有條件收購建議」]</ref>。最終市建局於[[2013年9月|2013年9月]]宣佈得到每一個地段超過八成不分割份數業權的的業主簽署買賣[[合約|合約]]，並於11月[[刊憲|刊憲]]獲[[發展局局長|發展局局長]]授權進行<ref>[https://www.ura.org.hk/tc/media/press-release/20131122 市建局兩個深水埗區的重建項目獲發展局局長授權進行]</ref>；當中，市建局總監（收購及就置）[[黃偉權|黃偉權]]指，游說業主簽約的過程甚艱巨，有業主同意收購，簽約時卻反口，該項目於限期屆滿前兩周仍不足六成業主同意建議，至限期屆滿前夕才僅湊夠八成<ref>[http://the-sun.on.cc/cnt/news/20130911/00407_020.html 深水埗兩重建項目料過關]</ref>。

[[2015年3月|2015年3月]]，[[地政總署|地政總署]][[刊憲|刊憲]]根據《[[收回土地條例|收回土地條例]]》收回土地進行重建作住宅用途，並提供零售及附屬設施，包括地面上落客貨區連[[停車場|停車場]]<ref>[http://www.info.gov.hk/gia/general/201503/06/P201503060262.htm 政府刊憲收回深水埗土地進行需求主導巿區重建項目]</ref>。

[[2016年8月|2016年8月]]，市建局邀請有興趣參與的[[發展商|發展商]]提交合作發展意向書，終接獲32份意向書，市建局董事會設立的招標遴選小組經詳細考慮後，於同年9月決定邀請27家符合資格的發展商入標<ref>[https://www.ura.org.hk/tc/media/press-release/20160906 市建局深水埗九龍道／僑蔭街需求主導重建項目招標]</ref>。至10月上旬截止時，一共接獲11份標書，同月下旬市建局公布決定與[[信和置業|信和置業]]合作發展該項目，當時預計可提供約4,070[[平方米|平方米]]住宅樓面面積（約80個住宅單位），以及約814平方米商業樓面面積<ref>[https://www.ura.org.hk/tc/media/press-release/20161025-dl4 信和置業有限公司投得市建局深水埗九龍道／僑蔭街需求主導重建項目]</ref>。而市場消息透露，項目招標條款列明，當項目的賣樓收入超過6.8億元，[[市建局|市建局]]可獲分紅，首2500萬元的分紅兩成，其後每2500萬元增一成，遞增至7500萬元或以上分紅五成。條款亦提及，將來單位面積最小要在260方呎或以上，並至少一半單位面積於480方呎以下，預料該地皮估值介乎2.7億至3億[[港元|港元]]，每呎地價由5200元起<ref>[http://std.stheadline.com/daily/news-content.php?id=1496073&target=2 信和奪市建深水埗細地]</ref>。

[[2017年6月|2017年6月]]，[[屋宇署|屋宇署]]獲批建1幢25層商住物業，住宅樓面涉4.38萬方呎，另設8766方呎商業樓面<ref>[https://www.mpfinance.com/fin/instantp2.php?node=1502781907361&issue=20170815 【豪宅有市】建灝赤柱項目批建11幢4層高豪宅及32幢洋房]</ref>。

== 簡介 ==
一號九龍道 共設24層（不包括天台、高層天台1、高層天台2及高層天台3，並不設4、13、14及24樓），於5至27樓提供100伙實用面積介乎301方呎至443方呎的1房至2房住宅單位，當中1房佔60伙，其餘則為2房戶，其中有10伙為連平台或連天台的特色單位，有報導指為「[[市建局|市建局]]近年最大入場戶」<ref>[https://news.mingpao.com/pns/dailynews/web_tc/article/20180917/s00004/1537122522086 一號九龍道300呎起 市建局近年最大入場戶]</ref>，其餘樓層分別設有
*地下：住宅入口大堂、商舖、停車場（1個訪客停車位、1個傷健人士停車位、1個電單車停車位及1個上落貨車位）
*1-2樓：商舖、機房
*3樓：[[住客會所|住客會所]]（面積約1,366平方呎，包括[[游泳池|游泳池]]、遊樂場及園景區等）

== 命名 ==
[[2018年6月|2018年6月]]發展商公佈樓盤中英文名稱，表示「項目擁有市區難得一見的綠化景觀，是鬧市少有的自然美景」，故英文名為“Madison Park”，即「美麗的公園」，中文則以地址來定名<ref>[http://std.stheadline.com/daily/news-content.php?id=1814717&target=2 深水埗新盤命名直接易記]</ref>。
