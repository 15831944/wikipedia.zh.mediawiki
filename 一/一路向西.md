==分級制度==

依[[香港電影分級制度|香港電影分級制度]]本片屬於第Ⅲ級，18歲以上人士收看。

== 制作 ==
影片投资1000万，其中3D制作费用占7成。剧本100%是用“公仔画”画出来的，以更直观形象地描述拍摄的环境状况。胡导解释：“因为我是一个新导演，为了事前更好地与演员、制作人员沟通，我特意请了一位香港著名的漫画家(林祥焜)帮我画了70%，剩下的部分由我自己来画。这样做虽然花费的成本大了，但是就节省了沟通成本，大家做起事来事半功倍。”<ref>{{cite web|url=http://ent.163.com/12/0913/20/8BAE1U3000032KMI.html#p=8A7UPM7I00B50003 | language= | title=导演解读3D情色片《一路向西》拍摄幕后故事 |publisher= 网易娱乐 |date=2012-09-13 |accessdate= }}</ref>

== 首影禮 ==
於2012年9月11日 (星期二) 晚上9:30於香港九龍尖沙咀iSQUARE UA戲院全部5座[[電影院|電影院]]舉行。

== 評價 ==

由F級巨乳女星王李丹妮主演，當中露乳，揸奶情節比美同期作品。

[[明報|明報]]的石琪給予2顆半星（最高5星）。他說：「《一路向西》始終是迎合男嫖客的賣淫戲，比起過去那些港產妓女名作，例如《[[廟街皇后|廟街皇后]]》、《[[榴槤飄飄|榴槤飄飄]]》、《[[金雞_(電影)|金雞]]》等拍出社會滄桑和女性感受，實在相差很遠。」<ref>{{cite web|title=石琪說戲﹕向西賣淫稱職 獵頭鬥智可觀|url=http://ol.mingpao.com/cfm/star20.cfm?File=20120923/sae03/myb1.txt|date=2012年9月23日|author=石琪|publisher=明報}}</ref>

[[爽報|爽報]]的爽影評給予2個讚(最高5個)：「電影技巧上，極度忠於原著，但太依賴自白及[[對白|對白]]，營造不少悶場，萬一[[荷李活|荷李活]]買下《一路向西》翻拍，背景改成[[美國|美國]]青年跨境到[[中美洲|中美洲]]召妓，說不定會拍出神級[[Cult片|Cult片]]。」<ref>{{cite web|title=爽影評：《一路向西》忠於原著添悶場|url=http://hk.apple.nextmedia.com/entertainment/art/20120926/18025339|date=2012年9月26日|publisher=爽報 (香港)}}</ref>
