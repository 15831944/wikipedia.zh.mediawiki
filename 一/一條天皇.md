'''一條天皇'''（{{jpn|j=一条天皇|hg=いちじょう てんのう|rm=Ichijō Tennō}}；{{bd|980年|7月15日|1011年|7月25日|catIdx=Y一}}），是[[日本|日本]]第66代[[天皇|天皇]]，986年7月31日－1011年7月16日在位。名'''懷仁'''（{{jpn|j=懐仁|hg=やすひと|rm=Yasuhito}}）。

== 系譜 ==
他是[[圓融天皇|圓融天皇]]的第一皇子，母親是[[藤原兼家|藤原兼家]]之女[[藤原詮子|詮子]]。一条天皇的父帝[[圓融天皇|圓融天皇]]除了他以外，沒有其他子女，因此一条天皇為獨生子。

== 略歷 ==
[[永觀|永觀]]2年（984年），還在[[花山天皇|花山天皇]]的時代，就被立為[[皇太子|皇太子]]。986年8月1日，因為花山天皇出家，七歲的皇太子即位（一說是兼家為了讓孫子早點即位而實行的陰謀）。立[[冷泉天皇|冷泉天皇]]的皇子居貞親王（[[三條天皇|三條天皇]]）為皇太子，[[攝政|攝政]]由藤原兼家擔任（之後為[[關白|關白]]）。

兼家死後由長男[[藤原道隆|道隆]]等外戚繼續擔任攝政與關白，並且讓女兒[[藤原定子|定子]]入宮做[[皇后|皇后]]，道隆於[[長德|長德]]元年（995年）病逝。取而代之的是由其弟[[藤原道兼|道兼]]就任關白，但他七天後也死了，之後道隆之子[[藤原伊周|伊周]]與道隆、道兼之弟[[藤原道長|道長]]相爭，道長得到勝利，掌握實權。道長將女兒[[藤原彰子|彰子]]也送入宮，成為與皇后定子位階相等的[[中宮|中宮]]，開了一帝二-{后}-的先例。

一條天皇的時代，也是道隆、道長兄弟與[[藤原氏|藤原氏]]權勢達到頂峰的時代，而服侍皇后定子的[[清少納言|清少納言]]與服侍中宮彰子的[[紫式部|紫式部]]、[[和泉式部|和泉式部]]等人，使平安時代女流文學得到發展。天皇本人對文藝方面也相當關心，留下《[[本朝文粹|本朝文粹]]》等詩文。他對音樂也很有專長，擅長吹笛。此外，他為人溫和而好學，得到很多人的仰慕。

== 后妃.皇子女 ==

*皇后：[[藤原定子|藤原定子]]，[[關白|關白]][[藤原道隆|藤原道隆]]之女。
**第一皇女：[[脩子內親王|脩子內親王]]，一品准三宮
**第一皇子：[[敦康親王|敦康親王]]，一品式部卿
**第二皇女：[[媄子內親王|媄子內親王]]，東三條院藤原詮子的養女

*中宮：[[藤原彰子|藤原彰子]]，攝政[[藤原道長|藤原道長]]之女。
**第二皇子：敦成親王，後為[[後一條天皇|後一條天皇]]
**第三皇子：敦良親王，後為[[後朱雀天皇|後朱雀天皇]]。

*女御：[[藤原義子|藤原義子]]，[[太政大臣|太政大臣]][[藤原公季|藤原公季]]之女。稱「[[弘徽殿|弘徽殿]]女御」。
*女御：[[藤原元子|藤原元子]]，[[左大臣|左大臣]][[藤原顯光|藤原顯光]]之女。
*女御：[[藤原尊子|藤原尊子]]，關白[[藤原道兼|藤原道兼]]之女。
*宫人：[[御匣殿_(藤原道隆四女)|御匣殿]]：藤原道隆第四女，皇后定子之妹。

== 諡號・追號・異名 ==
根據他在位時的里內裡（臨時[[皇居|皇居]]）之名「一條院」，取其成為他的[[院號|院號]]。明治以後，廢除院號，從此正式稱為「一條天皇」。
