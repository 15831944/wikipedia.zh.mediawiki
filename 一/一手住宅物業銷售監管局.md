'''一手住宅物業銷售監管局'''（{{lang-en|Sales of First-hand Residential Properties Authority}}，縮寫為SRPA）為[[香港政府|香港政府]][[運輸及房屋局|運輸及房屋局]]轄下的[[香港政府部門|部門]]，主要負責執行《[[香港法例|香港法例]]》第621章《[[一手住宅物業銷售條例|一手住宅物業銷售條例]]》，監管[[發展商|發展商]]必須按照條例提交合適的售樓說明書、價單及提供[[示範單位|示範單位]]等。

== 組織 ==
*一手住宅銷售物業監管局：由一名首長級乙級[[政務官|政務官]]（首長級薪級第三點）出任專員，由一名首席行政主任出任副專員，另外由一支由30名綜合[[專業人士|專業人士]]所組成的隊伍，包括屋宇測量師及產業測量師等，共32名人員。
**巡查及監控組，負責查核售樓書、價單及樓盤廣告。
**調查及檢視條例遵守情況組，負責調查違反條例的個案。
**投訴組，負責接收投訴。
**行政及公眾教育組，負責[[行政|行政]]及公眾教育。

== 歷史 ==
為了加強規管新樓宇的銷售，運輸及房屋局於2012年11月表示計劃於2013年4月29日成立一手住宅銷售物業監管局。

2012年12月月初，立法會房屋事務委員會將會討論開設首長級職位的事宜<ref>[http://orientaldaily.on.cc/cnt/news/20121127/00176_009.html 新執法機關管一手樓銷售] 《東方日報》 2012年11月27日</ref>。

== 批評 ==
銷監局指由新例生效至2015年3月底，累積有160個樓盤推售，接獲119宗投訴，以銷售安排、失實陳述及樓書投訴最多，惟至今仍未正式作出檢控，文件亦未有披露詳細情况。局方強調已經就投訴個案、巡查發現的違規情況及傳媒查詢的懷疑違例個案，進行立案調查，亦已經陸續將涉嫌違規的調查結果，提交律政司刑事檢控科考慮。<ref>銷監局兩年接119投訴 未作檢控 余敏欽 《[[香港經濟日報|香港經濟日報]]》 2015年4月28日</ref>
