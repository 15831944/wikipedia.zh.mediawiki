《'''姆姆抱抱'''》（MuMuHug），為[[台灣動畫|臺灣動畫]]公司「[[首映創意|首映創意]]」（SOFA Studio）的作品，<ref name=singtao01>{{Cite web |url=http://news.singtao.ca/toronto/content_column_tor.php?IndexID=3753982 |title=讓小朋友一起來追夢 .兒童國際電影節百片爭輝 |work=加拿大多倫多中文新聞網- 星島日報 |date=2012-03-15}}</ref>製作歷時三年，目前發展一～四季，共52集。全作品無對話，僅靠肢體表演，於2008年12月在[[東森幼幼台|東森幼幼台]]首播，日後[[臺視|臺視]]也在2011年播放該片。至2012年，海外授權超過八十國，<ref name=singtao01/>尤其在[[南韓|南韓]]的表現最為突出。

== 內容 ==
故事敘述一隻藍色的海洋生物姆姆獨自在島上生活，每天都有不同的朋友來找他彼此互動，通過淺顯的故事教導小朋友為人處世的道理。<ref name=singtao01/>

== 角色介紹 ==
*姆姆(MuMu)：主角，小男生，是鯨魚和海豚的綜合體，最喜歡抱抱。雖然住在島上，但不會游泳。
*波波(BoBo)：第一配角，小女生，活潑的小海豹。初出場時尚未脫毛，全身雪白，後來在姆姆的陪伴下，脫毛變成鮮豔的粉紅。
*凸泡(Tupow)：全身會發光閃爍類似彈塗魚的魚類。
*波奇(Boki)：大企鵝，多疑喜猜忌，不過是個好爸爸。
*史努努(Snoonoo)：小鯨魚，老是脫隊玩耍，不過當姆姆在海上漂流時，救了姆姆一命。
*姆弟(Mudy)：小章魚個性調皮，愛玩捉迷藏。最喜歡在沙上畫畫。
