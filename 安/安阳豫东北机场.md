'''安阳豫东北机场'''位于[[中国|中国]][[河南省|河南省]][[安阳市|安阳市]][[汤阴县|汤阴县]]瓦岗乡。是河南省目前已获批的四个新建民用支线机场之一。

== 地理区位 ==
机场位于[[安阳市|安阳市]][[汤阴县|汤阴县]]瓦岗乡火龙岗，距市区约28千米。

== 设计标准 ==
规划总占地面积约4500亩，投资约10亿元。该机场按照4D标准设计，跑道长度为2.8千米，机场定位为支线客运、通用航空和航空体育运动相结合的综合多用途机场，同时将作为[[郑州|郑州]][[新郑国际机场|新郑国际机场]]的备降机场。

== 历史 ==

===安阳第一座机场===
*安阳第一座机场修建于1937年（[[民国26年|民国26年]]）。1937年11月4日，安阳沦陷，安阳机场被日军占领。
*1945年日军投降后，[[国民党政府|国民党政府]]接管安阳机场。

===新机场筹备历程===
*2004年，[[国家民航总局|国家民航总局]]《关于全国民用机场布局规划》指出，河南省在未来可以新建3个[[支线机场|支线机场]]，引起安阳、[[信阳|信阳]]、[[商丘|商丘]]、[[周口|周口]]、[[焦作|焦作]]等6市争“飞”，都宣称要建民用机场。后仅有[[信阳明港机场|信阳明港机场]]被列入规划，但是安阳方面一直没有放弃，仍通过各种途径，为筹建新机场而努力。
*2009年，安阳计划与邯郸联合建[[安邯机场|安邯机场]]，按照该计划，安阳引进马来西亚的[[赛诺美嘉集团|赛诺美嘉集团]]投资，在安阳建一座[[异地航站楼|异地航站楼]]，该计划后因邯郸方面迟迟不愿更名而搁浅。
*2011年4月，安阳豫东北机场正式被列入国家十二五规划。
*2013年，机场建设前期准备工作开始进行。
*2017年12月29日，安阳机场项目战略合作协议签约暨奠基仪式举行。
