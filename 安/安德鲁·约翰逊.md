'''安德魯·约翰逊'''（{{lang|en|Andrew Johnson}}，{{bd|1808年|12月29日|1875年|7月31日|Johnson}}），[[美國|美國]]政治人物，第17任[[美國總統|總統]]。

== 早期生活 ==
生於[[北卡羅萊納州|北卡羅萊納州]]，1843－1853年任[[田納西州|田納西州]]聯邦眾議員，1853－1857年任田納西州州長，1857－1862年任田納西州聯邦參議員，其後出任戰時州長，1864年被[[共和黨_(美國)|共和黨]]的[[亚伯拉罕·林肯|林肯]]總統提名為聯邦副總統候選人。

== 繼任總統 ==
1865年4月[[亚伯拉罕·林肯|林肯]]總統遇刺時，主謀者[[約翰·威爾克斯·布思|約翰·威爾克斯·布思]]另外安排{{tsl|en|George Atzerodt|喬治·阿澤羅德特}}刺殺约翰逊，企圖加上對時任[[美國國務卿|美國國務卿]][[威廉·亨利·西華德|威廉·亨利·西華德]]的刺殺，使[[美國聯邦政府|美國聯邦政府]]陷入癱瘓。但是喬治·阿澤羅德特因為害怕，沒有對约翰逊下手，刺殺行動失敗。<ref name="george atzerodt"/>事後布斯在追捕過程中被聯邦士兵射殺，喬治·阿澤羅德特等四名同夥均被處[[絞刑|絞刑]]。

林肯遇刺身亡後，安德魯·约翰逊以副總統繼任為[[美國總統|總統]]。就任總統後，作为民主党人的他繼續沿用林肯內閣，因此仍被视为共和党政府。之後的[[美國重建時期|重建時期]]，作為南方民主黨人，由於在南方重建採取妥協及同情南方的立場（他曾兩次擔任[[田納西州|田納西州]]州長），與控制[[美國國會|國會]]絕大多數的[[共和黨_(美國)|共和黨]]議員意見不合，使他于1868年成為[[美國歷史|美國歷史]]上首位被[[彈劾|彈劾]]的總統，但最終在[[美國參議院|參議院]]僅以一票之微避過[[彈劾|彈劾]]的命運，而他亦顺利完成總統任期。

== 卸任後 ==
卸任後，安德魯·约翰逊于1874年以民主党人身份參選國會參議員，成功當選，但他在就任幾個月便逝世。
