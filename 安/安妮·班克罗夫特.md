'''安妮·班克罗夫特'''（英语：'''Anne Bancroft'''，{{bd|1931年|9月17日|2005年|6月6日|catIdx= Bancroft}}），生于[[美国|美国]][[纽约|纽约]]，美国著名女演员，其作品《[[毕业生_(电影)|毕业生]]》蜚声世界影坛，1962年[[奥斯卡最佳女主角奖|奥斯卡最佳女主角奖]]得主。

== 生平 ==
=== 早年生活 ===
1931年，班克罗夫特出生于[[美国|美国]][[纽约州|纽约州]][[纽约市|纽约市]][[布朗克斯|布朗克斯]]，她的母亲米尔·德里德，是一个电话接线员，父亲迈克尔是一个服装设计师。<ref>[http://www.filmreference.com/film/4/Anne-Bancroft.html Anne Bancroft Biography (1931–)<!-- Bot generated title -->]</ref>

班克罗夫特的父母都是意大利移民，他们全家信奉[[天主教|天主教]]。<ref>{{cite news|url=http://www.bbc.co.uk/dna/h2g2/alabaster/A650594|title=Mel Brooks – Director, Actor, Writer and Producer|publisher=BBC|work=[[h2g2|h2g2]]|accessdate=September 19, 2010}}</ref>

4岁时，班克罗夫特就开始学习[[表演|表演]]和[[舞蹈|舞蹈]]。<ref name=" bear" >{{cite web |url=http://www.cctv.com/news/entertainment/20050608/101952.shtml |title= 美国著名女影星安妮·班克罗夫特逝世 |accessdate= 2005年6月8日|publisher=CCTV新闻频道 |work= |language =zh-cn}}</ref>

9岁时，班克罗夫特在自己的卧室墙上写下“我要当演员”的誓言。<ref name="rat " >{{cite web |url=http://ent.sina.com.cn/r/m/2005-06-09/1133747194.html |title=东方早报：再见，罗宾逊太太 |accessdate=2005年6月9日 |publisher= 新浪网|work= |language =zh-cn}}</ref>

班克罗夫特毕业于[[布朗克斯|布朗克斯]]区的克里斯托弗·哥伦布高中。

1948年，班克罗夫特加入HB工作室，后来陆续加入[[洛杉矶|洛杉矶]]的[[加利福尼亚大学|加利福尼亚大学]]旗下的美国科学院戏剧艺术、演员工作室和美国电影协会。

2005年6月6日，班克罗夫特在[[美国|美国]]西奈山医院因[[子宫癌|子宫癌]]病逝，享年73岁。<ref name=" wolf">{{cite web|url= http://bjyouth.ynet.com/article.jsp?oid=5522371|title= 奥斯卡影后安妮·班克罗夫特去世享年73岁|accessdate= 2005-06-09|publisher= 北青网|work= |language= zh-cn|deadurl= yes|archiveurl= https://archive.is/20130706021236/http://bjyouth.ynet.com/article.jsp?oid=5522371|archivedate= 2013-07-06}}</ref>

=== 职业生涯 ===
19岁时，班克罗夫特就出现电影上，21岁时，她正式和[[20世纪福克斯|20世纪福克斯]]电影公司签约。<ref name=" bear" />

1950年，班克罗夫特首次用“安妮·玛诺”的艺名出现在电视荧屏之上，随后在好莱坞主演了许多B级电影。<ref name=" fish" >{{cite web |url=http://datalib.ent.qq.com/star/887/starinfo.shtml |title= 安妮·班克罗夫特|accessdate=2005年 |publisher= 腾讯网|work= |language =zh-cn}}</ref>

1962年，班克罗夫特以《热泪心声》受封奥斯卡影后。<ref name=" fish" />

1964年，班克罗夫特以《吃南瓜的人》融合戛纳最佳女演员奖。<ref name=" fish" />

1968年，班克罗夫特和[[德斯汀·荷夫曼|達斯汀·霍夫曼]]，凯瑟琳·罗丝一起主演了《毕业生》，这是她一生最为著名的一部电影，为她赢得了巨大的声誉。<ref name=" fish" />

== 个人生活 ==
班克罗夫特一生有过2次婚姻经历。第一任丈夫是建筑工人马丁·梅，两人只在一起度过了3年。<ref name="rat " /><ref name=" wolf" /><ref name=" bear" />

1964年，班克罗夫特和“当代最滑稽的作家”的梅尔·布鲁克斯结婚，1972年，两人育有儿子马克思。<ref name=" bear" /><ref name=" fish" /><ref name="rat " /><ref name=" wolf" />两人的相识也极具有传奇色彩，他们是在一次音乐剧的彩排上认识的，班克罗夫特后来回忆说：“那时候我就在想，我遇到了真命天子。我从来没有和一个人相处得那么愉快，我也希望他喜欢我。”<ref name="rat " />
