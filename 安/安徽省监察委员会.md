'''安徽省监察委员会'''，简称'''安徽省监察委'''，是[[中华人民共和国|中华人民共和国]][[安徽省|安徽省]]的省级地方[[中华人民共和国国家监察机关|国家监察机关]]，与[[中国共产党安徽省纪律检查委员会|中国共产党安徽省纪律检查委员会]][[合署办公|合署办公]]，成立于2018年1月31日<ref>{{cite web |title = 安徽省监察委员会挂牌成立 |url = http://www.ah.gov.cn/UserData/DocHtml/1/2018/2/1/3285810179048.html |publisher=安徽省人民政府 |date=2018-02-01 |accessdate=2018-03-04 }}</ref>，取代[[安徽省监察厅|安徽省监察厅]]的职能。安徽省监察委主任由[[安徽省人民代表大会|安徽省人民代表大会]]选举产生，并向其和安徽省人大常委会以及中华人民共和国监察委员会负责。安徽省监察委现任主任为中共安徽省委常委、省纪委书记[[刘惠_(1966年)|刘惠]]<ref>{{cite web |url = http://www.ahrd.gov.cn/npcweb/web/info_view.jsp?strId=1517308547431185 |title=安徽省人民代表大会常务委员会任命人员名单 |website = 安徽人大 |date=2018-01-30 |accessdate = 2018-02-26 }}</ref>。
