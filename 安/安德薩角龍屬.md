'''安德薩角龍屬'''（屬名：''Udanoceratops''）又名'''峨丹角龍'''，是種[[角龍下目|角龍下目]][[恐龍|恐龍]]。牠們生存於晚[[白堊紀|白堊紀]]的[[桑托階|桑托階]]晚期或[[坎潘階|坎潘階]]早期，[[化石|化石]]發現於[[蒙古|蒙古]]。

== 發現與種 ==
[[File:Udanoceratops_size_comparison.png|thumb]]
安德薩角龍的化石是一個幾乎完整的大型頭顱骨，長度為60公分，保存狀態相當好。安德薩角龍似乎是目前所發現的最大型二足[[新角龍類|新角龍類]]。安德薩角龍的頭顱骨擁有非常小的角與頭盾，而身體長估計約4公尺。[[正模標本|正模標本]]發現於[[南戈壁省|南戈壁省]]的[[牙道黑達組|牙道黑達組]]（Djadokhta Formation），地質年代約8350萬到7060萬年前。

在1992年，Sergei Kurzanov將這個頭顱骨進行敘述、命名，[[模式種|模式種]]是''U. tschizhovi''。屬名是以化石發現處的Udan-Sayr為名，''ceras/κέρας''則是角龍類的學名常見字根。

在2007年，[[東戈壁省|東戈壁省]]的[[巴魯恩戈約特組|巴魯恩戈約特組]]發現一個標本，被歸類於模式種的親近種（''U. aff. tschizhovi''），地質年代為[[坎潘階|坎潘階]]。

== 分類 ==
安德薩角龍屬於角龍下目，角龍下目恐龍是群[[草食性|草食性]]恐龍，擁有類似[[鸚鵡|鸚鵡]]的喙狀嘴，生存於[[白堊紀|白堊紀]]的[[北美洲|北美洲]]與[[亞洲|亞洲]]。所有的角龍類恐龍在[[白堊紀|白堊紀]]末期滅絕。

== 食性 ==
如同所有角龍類恐龍，安德薩角龍是[[草食性|草食性]]。在[[白堊紀|白堊紀]]期間，[[開花植物|開花植物]]的地理範圍有限，所以安德薩角龍可能以當時的優勢植物為食，例如：[[蕨類|蕨類]]、[[蘇鐵|蘇鐵]]、[[針葉樹|針葉樹]]。牠們可能使用銳利的喙狀嘴咬下樹葉或針葉。
