'''安藝龜山站'''（{{jpn|j=あき亀山駅|hg=あきかめやまえき|rm=Aki-kameyama eki|f=y}}）是[[日本|日本]][[廣島縣|廣島縣]][[廣島市|廣島市]][[安佐北區|安佐北區]]龜山南二丁目的車站，屬於[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）[[可部線|可部線]]，於2017年3月4日開業。正式名稱決定前曾暫名「'''新河戶站'''」（{{lang|ja|新河戸駅}}）<ref>{{Cite web |publisher=[[國土交通省|國土交通省]] |date=2014-02-25 |url=http://www.mlit.go.jp/common/001028379.pdf |title=西日本旅客鉄道（株）申請の第一種鉄道事業許可について（概要） |format=PDF |accessdate=2014-02-25}}</ref>。

由於在[[可部站|可部站]]至[[三段峽站|三段峽站]]路段於2003年停駛廢止之前，距離可部站約5公里另有一[[安藝龜山站_(舊站)|同名車站]]，為避免混淆，新設安藝龜山站的站名前兩字改以[[平假名|平假名]]表示。

== 概要 ==
設有[[島式月台|島式月台]]1座，提供1面2線配置。另外兩邊月台側均各有1條側線。有效長度為4輛。而當末班列車當到達後，亦會在月台上留置至翌日清晨首班車為止。

== 歷史 ==
* 2014年（平成26年）
**2月25日 - 包含新河戶站的鐵道工程獲得[[國土交通大臣|國土交通大臣]]許可<ref>{{Cite web |date=2014-02-25 |url=http://response.jp/article/2014/02/25/217984.html |title=国交相、可部線延伸事業を許可…廃線を復活 |publisher=レスポンス |accessdate=2014-02-25}}</ref>。
**[[12月19日|12月19日]] - 由於車站建設用地的手續進行延遲，決定將開業時間延期1年至2017年春天<ref name="NHK20141219">{{cite web|url=http://www3.nhk.or.jp/hiroshima-news/20141219/4117611.html|title=可部線延伸計画 １年延期に|publisher=NHK|date=2014-12-19|accessdate=2014-12-23|deadurl=yes|archiveurl=https://web.archive.org/web/20141223055108/http://www3.nhk.or.jp/hiroshima-news/20141219/4117611.html|archivedate=2014-12-23}}</ref>。
* 2015年（平成27年）2月6日 - 車站設計完成<ref>{{cite press release|url=http://www.westjr.co.jp/press/article/2015/02/page_6784.html|title=可部線電化延伸区間の新駅について|date=2015-02-06|publisher=西日本旅客鉄道}}</ref>。
* 2016年（平成28年）7月8日 - JR西日本將正式站名命名為「'''安藝龜山站'''」<ref>[https://www.westjr.co.jp/press/article/2016/07/page_8915.html 可部線および山陽線新駅の駅名決定について] 西日本旅客鉄道 ニュースリリース 2016年7月8日</ref>。
* 2017年（平成29年）3月4日 - 伴隨[[可部線|可部線]]的{{link-ja|可部站|可部駅}}到本站的區間開通，車站開業<ref>{{Cite press release|url=http://www.westjr.co.jp/press/article/items/161216_00_hiroshima.pdf|format=PDF|title=平成29年春のダイヤ改正について|publisher=西日本旅客鉄道広島支社|date=2016-12-16|accessdate=2016-12-19}}</ref>。
