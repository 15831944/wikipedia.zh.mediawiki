'''安宁市'''是[[中华人民共和国|中华人民共和国]][[云南省|云南省]][[昆明市|昆明市]]下辖的一个[[县級市|县級市]]，位于昆明城区以西33公里，全市总面积1301平方千米，2012年末全市常住人口35.1万人。

安宁古城连然，是昆明通往滇西的门户重镇，自古称“诸爨要冲”，被誉为“连然金方，螳川宝地”。境内矿产资源丰富，是云南省最大的[[钢铁|钢铁]]、[[盐|盐]]和[[磷|磷]]化工基地。

== 历史沿革 ==
[[汉|汉]]武帝[[元封|元封]]二年（公元前109年），置'''连然县'''。[[唐|唐]]高祖[[武德|武德]]四年（公元621年），改称安宁县。[[元|元]][[至元|至元]]十二年（公元1275年），升县为州。[[民国|民国]]2年（公元1913年），复改县。1950年4月20日，安宁县人民政府成立。1956年10月，改安宁县为昆明市安宁区。1959年9月，复称安宁县。1995年10月13日撤县设市，归昆明市管辖。<ref>{{cite web | language = zh-cn | publisher = 安宁市人民政府 | title = [建置区划] | url = http://www.ynan.gov.cn/Category_65/Index.aspx | date =  | accessdate = 2013-06-22 | archive-url = https://web.archive.org/web/20141025024829/http://www.ynan.gov.cn/Category_65/Index.aspx | archive-date = 2014-10-25 | dead-url = yes }}</ref>

==地理==
===区位===
安宁市地处昆明市西郊，距昆明主城区33千米。介于东经102°8'～102°37'和北纬24°31'～25°6'之间。南北长约66.5千米,东西宽约46.4千米,总面积1301平方千米。东面和东北面与昆明市[[西山区_(昆明市)|西山区]]接壤，西面和西北面与[[禄丰县|禄丰县]]交界，南面和东南面与[[晋宁县|晋宁县]]相连，西南面与[[易门县|易门县]]毗邻。<ref>{{cite web| language =zh-cn| publisher =安宁市人民政府| title =安宁的地理位置及气候（2012年）| url =http://ynxxgk.yn.gov.cn/M1/view.aspx?int_Document_ID=1823576| date =| accessdate =2013-06-22}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

===地形===
安宁地形呈东南高、西北低，有八街、连然、禄脿3个山间盆地，其余为山区半山区，属于[[乌蒙山|乌蒙山]]余脉。最高点位于市境西南部与易门县交界处的黑风洞，海拔2617.7米，最低点在[[草铺街道|草铺街道]]王家滩村委会[[大河_(扒河)|扒河]]出安宁境处的鲁家山河谷，海拔1680米。<ref name="安宁水务局">{{cite web| language =zh-cn| publisher =安宁水务局| title =安宁市水利十一五规划和2020年长远规划| url =http://ynxxgk.yn.gov.cn/M1/view.aspx?int_Document_ID=1261664| date =| accessdate =2013-06-22| deadurl =yes| archiveurl =https://web.archive.org/web/20160304115711/http://ynxxgk.yn.gov.cn/M1/view.aspx?int_Document_ID=1261664| archivedate =2016-03-04}}</ref>全市平均海拔1800米。

===水系===
[[File:Tanglang_River_near_Baita_Village_in_Anning_City.jpg|200px]]
安宁境内河流除[[禄脿街道|禄脿]]和草铺境内的[[大河_(扒河)|扒河]]（九渡河）属于[[红河|元江－红河]]水系外，其他河流均属[[金沙江|金沙江]]水系。主要河流是流经市区的[[螳螂川|螳螂川]]，境内所有属于金沙江水系的河流最后都汇入螳螂川。全境金沙江水系的流域总面积为1206平方公里，红河水系流域面积为115平方公里。<ref name="安宁水务局"/>

===气候===
安宁市属于高原低纬度亚热带季风气候，年平均气温14.7[[摄氏度|摄氏度]]。

== 行政区划 ==
安宁市于2011年4月正式在全域撤镇设[[街道办事处|街道办事处]]，实现了“一级政府、两级管理”的城市管理体系，这在昆明市和整个云南全省县(市)中均尚属首家。目前，安宁现有街道办事处9个：[[连然街道|连然街道]]、[[金方街道|金方街道]]、[[八街街道|八街街道]]、[[县街街道|县街街道]]、[[温泉街道|温泉街道]]、[[太平新城街道|太平新城街道]]、[[青龙街道|青龙街道]]、[[草铺街道|草铺街道]]、[[禄脿街道|禄脿街道]]<ref>{{cite web | language = zh-cn | publisher = 云南经济日报 | title =安宁全域撤镇设街道办事处| url = http://jjrbpaper.yunnan.cn/html/2011-04/19/content_330434.htm| date = 2011-04-19

== 交通 ==
安宁是昆明通往滇西的重要通道，境内交通四通八达。境内主要公路有{{中国国道名|320}}、[[昆安高速公路|昆安高速公路]]、{{China Expwy Name|G56|[[安楚高速公路|安楚高速公路]]}}（属于[[杭瑞高速公路|杭瑞高速公路]]的一部分）和[[安晋高速公路|安晋高速公路]]（昆明绕城高速公路西南段）。截止2012年全市公路里程达1308公里，行政村公路硬化率达100%，<ref>{{cite web | language = zh-cn | publisher = 云南省政府信息公开门户网站 | title = 2012年安宁交通运输发展情况 | url = http://ynxxgk.yn.gov.cn/M1/view.aspx?int_Document_ID=1823586 | date = 2013-04-24 | accessdate = 2013-06-22 | deadurl = yes | archiveurl = https://web.archive.org/web/20140222173758/http://ynxxgk.yn.gov.cn/M1/view.aspx?int_Document_ID=1823586 | archivedate = 2014-02-22 }}</ref>并实现100%行政村公交全覆盖。<ref>{{cite web | language = zh-cn | publisher = 云南省交通厅 | title =安宁实现100%行政村公交全覆盖| url = http://www.ynjtt.com/Item.aspx?id=14119| date = 2013-05-10 
| accessdate = 2011-09-27 }}</ref>

铁路方面，[[file:China_Railways.svg|15px]][[成昆铁路|成昆铁路]]经过安宁市北部地区，同时境内还有盐矿、昆钢、云化、大黄磷、长坡石油专用线等各类铁路运输线，为各大工厂、矿山提供物资运输。过去安宁市有铁路客运服务，现已取消。建设中[[file:China_Railways.svg|15px]][[昆明市域铁路|昆明市域铁路]]将连接安宁市、昆明城区和[[嵩明县|嵩明县]]。<ref>{{cite web | language = zh-cn | publisher = 云南网 | title = 铁路横穿昆明市中心 25分钟到安宁 | url = http://yn.yunnan.cn/html/2011-11/19/content_1916454.htm | date = 2011-11-19 | accessdate = 2013-06-22 | deadurl = yes | archiveurl = https://web.archive.org/web/20130312192219/http://yn.yunnan.cn/html/2011-11/19/content_1916454.htm | archivedate = 2013-03-12 }}</ref>

== 经济 ==
[[File:The_main_gate_of_Kunming_Iron_&_Steel_Co._Ltd.jpg|280px]]。]]
安宁是云南省最大的冶金、盐、磷化工基地，是云南最大的钢铁企业[[昆钢集团|昆钢集团]]的所在地，也是云南最大的化工企业[[云天化集团|云天化集团]]主要的生产基地。随着[[中石油云南石化1000万吨炼油项目|中石油云南石化1000万吨炼油项目]]落户安宁工业园区草铺片区，安宁将成为云南石化产业中枢。<ref>{{cite web | language = zh-cn | publisher = 搜狐 | title =安宁将成西南石油中枢 | url = http://roll.sohu.com/20130206/n365651737.shtml| date = 2013-02-06 | accessdate = 2013-06-22 }}</ref>然而，由于担心该项目含有[[PX|PX]]项目，2013年[[2013年昆明市反对PX项目事件|昆明市爆发了反对PX项目事件]]。

2016年，全年实现地区生产总值(GDP)272.87亿元，比上年增长8.0%。人均生产总值(按常住人口计算)73810元，比上年增长6.4%。 在地区生产总值中，第一产业实现增加值13.31亿元，比上年增长5.9%，拉动GDP增长0.3个百分点;第二产业实现增加值109.38亿元，比上年增长0.9%，拉动GDP增长0.4个百分点，其中工业实现增加值90.5亿元，比上年下降1.1%;第三产业实现增加值150.18亿元，比上年增长14.8%，拉动GDP增长7.3个百分点;一、二、三产业增加值比重分别为4.9%、 40.1 %和55.0%。非公经济实现增加值110.11亿元，增长8.9%，占全市地区生产总值的40.4%。

2016年，全市地方财政总收入达39.39亿元，比上年增长,3.9%。其中：公共财政预算收入29.13亿元，比上年增长10.1%;上划中央“四税”收入10.26亿元，比上年下降10.5%;全年地方财政支出35.02亿元，比上年下降23.2%，其中：公共财政预算支出30.74亿元，比上年增长5.3%。

== 人口 ==
2012年末，全市常住人口35.1万人，全市户籍人口26.76万人。全市有少数民族人口37414人，占总人口的14.0％。世居少数民族人口主要有[[白族|白族]]11196人，[[彝族|彝族]]13795人，[[苗族|苗族]]3918人．[[回族|回族]]3140人。2012年全市人口出生率为10.48‰；死亡率为5.23‰，自然增长率为5.25‰。<ref>{{cite web | language = zh-cn | publisher = 云南省政府信息公开门户网站 | title = 2012年安宁的行政区划及人口情况 | url = http://ynxxgk.yn.gov.cn/M1/view.aspx?int_Document_ID=1823577 | date = 2013-04-24 | accessdate = 2013-06-22 }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 教育 ==
安宁市拥有中国西南地区类型最为丰富，云南省最大的职业教育基地－安宁职业教育基地<ref>{{cite web | language = zh-cn | publisher = 云南新闻网 | title = 云南安宁职业教育基地探索中国西部职教发展新模式 | url = http://www.yn.chinanews.com/html/jiaoyu/20110309/154931.html | date = 2011-03-09 | accessdate = 2012-05-11 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。基地位于安宁主城区西南3.5公里，规划区总面积15.52平方公里。计划入驻学校达到8～10所，在校师生达到10～12万人的规模。<ref>{{cite web | language = zh-cn | publisher =网易新闻| title =8学校入驻安宁职教基地| url = http://news.163.com/11/0629/09/77N3348D00014AED.html| date = 2011-06-29 | accessdate = 2012-05-11 }}</ref>目前入驻的学校主要有[[昆明冶金高等专科学校|昆明冶金高等专科学校]]、[[云南技师学院|云南技师学院]]（[[云南工贸职业技术学院|云南工贸职业技术学院]]）、[[云南经济管理职业学院|云南经济管理职业学院]]、[[云南工程职业学院|云南工程职业学院]]、[[云南轻工职业学院|云南轻工职业学院]]、[[云南工艺美术职业学院|云南工艺美术职业学院]]、[[昆明扬帆职业技术学院|昆明扬帆职业技术学院]]、[[云南师达商贸管理学校|云南师达商贸管理学校]]、[[昆明现代工商旅游学校|昆明现代工商旅游学校]]等。此外安宁还拥有[[云南省交通高级技工学校|云南省交通技师学院]]（[[云南交通运输职业技术学院|云南交通运输职业技术学院]]）、[[昆明工业职业技术学院|昆明工业职业技术学院]]等职业学校。

截止2012年，安宁全市学龄前儿童毛入学率114.04%，初中毛入学率117.8％，普通高中录取率42.9%，高考综合上线率98.5％，高考录取率98.8％。2012年末，全市幼儿在园人数9825人，小学在校学生26740人，初中在校学生11991人，高中在校学生3888人，职教基地专任教师2860人，在校学生46351人。<ref>{{cite web | language = zh-cn | publisher = 云南省政府信息公开门户网站 | title = 2012年安宁教育事业发展情况 | url = http://ynxxgk.yn.gov.cn/M1/view.aspx?int_Document_ID=1823600 | date = 2013-04-24 | accessdate = 2013-06-22 }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 城市发展 ==

2017年成功创建成为第五届[[全国文明城市|全国文明城市]]和[[全国未成年人思想道德建设工作先进城市|全国未成年人思想道德建设工作先进城市]]。城市化进程明显加快，均衡化、绿色化、特色化发展水平明显提升，建成区面积达到37.39平方公里，成为全国新型城镇化质量百强县(市)、城乡交通运输一体化示范县和云南省可持续发展试验区、云南省首个装配式建筑示范城市。<ref>{{Cite web|url=http://an.km.gov.cn/c/2018-02-09/2331037.shtml|title=安宁概况|accessdate=2018-319|author=|date=|publisher=}}</ref>

== 风景名胜 ==
[[File:Caoxi_Temple,_Anning.jpg|250px]]]]
* 温泉·金色螳川旅游区：范围是从安宁“天下第一汤”牌坊至青龙峡与[[富民县|富民县]]交界处螳螂川两岸40余千米沿线，是一个开放式的生态旅游景区。
* [[安宁溫泉|安宁溫泉]]：位于[[温泉街道|温泉街道]]，有“天下第一汤”的美誉。
* [[温泉摩崖石刻群|温泉摩崖石刻群]]：位于温泉街道[[螳螂川|螳螂川]]右岸的环云崖上，多为明、清两代文人游温泉写景抒情之作，是云南省级文物保护单位。
* [[曹溪寺_(安宁)|曹溪寺]]：位于温泉街道以西1公里的龙山上，始建于[[大理国|大理国]]时期，其“宝函映月”久负盛名，为国家级重点文物保护单位。
* [[唐代|唐代]][[王仁求碑|王仁求碑]]：位于[[县街街道|县街]]小石庄村西南的葱蒙卧山中，立于[[唐朝|唐]][[武周|武周]][[圣历|圣历]]元年（698年），是目前云南昆明地区唯一留存的唐碑，2006年被[[国务院|国务院]]公布为全国重点文物保护单位。
* [[安宁法华寺|法华寺]]：[[连然街道|连然街道]]东部桃花村委会小桃花村以南约1公里的洛阳山中，拥有云南省第二大的石窟群，是云南省级文物保护单位。
* [[连然文庙|连然文庙]]：位于安宁市区连然街122号，始建于[[元朝|元]][[大德|大德]]六年（1302年），为国家级重点文物保护单位。
* 青龙峡风景区：位于[[青龙街道|青龙街道]]，距安宁市区29公里
* 玉龙湾东南亚影视城
* 三柱香公园
