'''安康市'''是[[中华人民共和国|中华人民共和国]][[陕西省|陕西省]]下辖的[[地级市|地级市]]，位于陕西南部偏东，北依[[秦岭|秦岭]]，南沿[[大巴山|大巴山]]-{A|zh-hans:余脉;zh-hant:餘脈}-。

== 历史 ==

[[西汉|西汉]]时，[[西城县_(西汉)|西城县]]隶汉中郡。[[北朝|北朝]]北周初撤县。[[隋|隋]]复设西城县，唐朝是[[金州|金州]]州治，之后州、县名多变。[[清|清]]乾隆四十八年（1783年）置安康县。1950年设安康市，1954年撤消，1988年复设安康市。

安康历史悠久，人文史可上溯到[[石器时代|石器时代]]。[[夏代|夏代]]，安康属[[梁州|梁州]]的一部分。据中国最早的地理著作《[[尚书|尚书]]·[[禹贡|禹贡]]》记载，将全国分为九州，称“华阳黑水惟梁州”，唐代学者认为梁州为今[[汉中|汉中]]、安康、[[商洛|商洛]]一带。[[商|商]][[周|周]]时安康成为[[庸国|庸国]]的封地，史称上庸。据《太平环宇记》载：“金州于[[战国|战国]]为[[楚|楚]]地，附庸地，后为楚所灭，复为楚地”。

公元前312年，[[秦惠王|秦惠王]]在安康[[汉江|汉江]]北岸台地设西城县，[[汉中郡|汉中郡]]设在西城。[[秦|秦]]统一六国（公元前221年），西城县为汉中郡治，辖安康、汉阴、石泉、紫阳、岚皋、平利、镇坪七县。

[[西汉|西汉]]沿袭秦制，汉中郡下设五县，[[东汉|东汉]][[建武|建武]]元年(公元25年)将汉中郡治迁移至汉中南郑县。东汉[[建安|建安]]二十一年(公元215年)，[[曹操|曹操]]攻占汉中，分郡之东部即安康为西城郡。

[[三国|三国]][[曹魏|曹魏]][[黄初|黄初]]二年（221年），取“曹魏兴盛”之义，设[[魏兴郡|魏兴郡]]，辖七县，[[西晋|西晋]]沿之。据《兴安府志》载：[[晋武帝|晋武帝]][[太康|太康]]元年（公元280年）为安置巴山一带流民，取“万年丰乐、安宁康泰”之意，将安阳县更名为安康县，“安康”从此得名。

[[南北朝|南北朝]]时，安康先属南朝，后属北朝，先称直州，[[西魏|西魏]]废帝三年（554年）设[[金州_(西魏)|金州]]，因越河川道出麸金得州名，隶属魏兴郡。

[[隋朝|隋朝]]复设西城郡。[[唐|唐]]、[[五代|五代]]、[[宋|宋]]设金州安康郡，辖六县。[[元朝|元朝]]设金州，[[明代|明代]]前仍设金州，[[万历|万历]]十一年（公元1583年）汉江洪水覆没金州城，遂于城南赵台山下筑新城，并易名为兴安州。

[[清|清]][[顺治|顺治]]四年（公元1647年）州府迁回老城。[[乾隆|乾隆]]四十七年（公元1782年）改设[[兴安府|兴安府]]。

[[辛亥革命|辛亥革命]]后，废府设道，在清代六县基础上新增四县归[[汉中道|汉中道]]。[[民国|民国]]二十二年（公元1933年），废道而直隶于省；二十四年（公元1935年）设为陕西省第五行政区行政督察专员公署。

[[第二次国共内战|第二次国共内战]]后期，中共政权逐步攻占中华民国政府辖下县份，于1949年设[[安康分区|安康分区]]，隶属于[[陕南行政区|陕南行政区]]。[[陕南军区|陕南军区]][[中国人民解放军第19军|第19军]]于5月24日攻占[[白河县|白河县]]，7月10日攻占[[平利县|平利县]]，7月25日第一次攻占安康县。11月28日，第19军55师攻占[[岚皋县|岚皋县]]，30日攻占[[紫阳县|紫阳县]]。第19军57师于11月26日攻占[[洵阳县|洵阳县]]，27日再度攻占安康县<ref>{{Cite web |url=http://www.sohu.com/a/197436461_799711 |title=沙建国【岁月留痕】红旗飘飘 —— 陕南区暨汉中解放初期纪事 |accessdate=2019-03-06 |author=沙建国 |date=2017-10-11 |work=|publisher=搜狐网|language=Zh-hans }}</ref>。

[[中华人民共和国|中华人民共和国]]成立后（1949年10月），于1949年12月，安康分区改为陕西省安康行政督察专员公署。1950年1月，解放军攻占今安康市最南瑞的[[镇坪县|镇坪县]]，至此，安康暨陕西全境“解放”。1950年7月，改称陕西省人民政府[[安康专区|安康专员公署]]，“文革”期间改称[[安康地区|安康地区]]革命委员会，“文革”结束后的第三年（1979年）改为[[安康地区|安康地区]]行政公署。中共安康地委和地区行政公署驻地安康县。1988年，安康县撤县建市，是为县级安康市。

2000年，中华人民共和国国务院批准安康地区撤地设市。新组建的中共安康市委员会、中共安康市纪律检查委员会于2000年12月正式对外办公，撤地建市后，原县级安康市改为[[汉滨区|汉滨区]]。地级安康市于2001年元月1日成立。

== 经济 ==
工业以丝纺、化工、建材、食品、采矿为主，全市電力均由在[[嵐河|嵐河]]沿岸所建立的水力發電廠提供<ref name=epochtimes>{{cite news |url= http://cn.epochtimes.com/b5/10/5/19/n2912513.htm |title=百里建16電站 陝嵐河斷流魚類滅絕 |date=2010-05-19 |accessdate=2010-05-29}}</ref>。不過由於這些電站大多是[[引水式電站|引水式電站]]，裝機容量通常較小，發電機組裝在河道旁，上游則建有[[水壩|水壩]]，水壩將水攔截後，通過人工渠道注入發電機組，利用自然落差發電<ref name=epochtimes />，所以發電規模也較小之餘，嵐河的盲目斷流令河流的魚類滅絕，造成生態大災難。

== 交通 ==
*[[襄渝铁路|襄渝铁路]]（[[湖北省|湖北省]][[襄陽市|襄陽市]]—[[重慶市|重慶市]]）、[[西康铁路|西康铁路]]（[[陝西省|陝西省]][[西安市|西安市]]—陝西省[[安康市|安康市]]）、[[阳安铁路|阳安铁路]]（陝西省[[漢中市|漢中市]][[陽平關|陽平關]]—安康市）三条电气化铁路在此交汇，是陕西省南部的重要交通枢纽。
**未来还将建成和推进西康二线、阳安复线、安康至[[湖南省|湖南省]][[张家界|张家界]]、西安至[[四川省|四川省]][[成都|成都]]、西安至武汉、西安至重庆客运专线6条铁路。<ref name="si">[http://sn.ifeng.com/shanxidifangzixun/ankang/detail_2013_10/30/1391791_0.shtml 安康富强机场年内开工 规划5条航线]鳳凰網，2013年10月30日</ref>
*[[210国道|210国道]]和[[316国道|316国道]]通过本市，[[包茂高速|包茂高速]]（[[內蒙古|內蒙古]][[包頭市|包頭市]]—[[廣東省|廣東省]][[茂名市|茂名市]]）中的[[西康高速公路|西康高速公路]]（陝西省西安市—安康市）2009年5月28日正式通车。
**未来还将建成和推进陕鄂安康至[[湖北省|湖北省]][[十堰市|十堰市]][[竹溪縣|竹溪縣]]、安康至[[廣西|廣西]][[北海市|北海市]]、[[寧夏|寧夏]][[银川市|银川市]]至[[廣西|廣西]][[百色市|百色市]]高速安康段。<ref name="si"/>
*最靠近的機場是位於安康市東方205公里的[[湖北省|湖北省]][[十堰武当山机场|十堰武当山机场]]及西方偏北210公里的[[陝西省|陝西省]][[汉中城固机场|汉中城固机场]]。
**位於安康市[[漢濱區|漢濱區]]的[[安康富强机场|安康富强机场]]將取代2013年起停航的[[安康五里舖機場|安康五里舖機場]]，目前规划中有5条航线，分别飞往[[西安|西安]]、[[北京|北京]]、[[重庆|重庆]]、[[成都|成都]]和[[武汉|武汉]]。<ref name="si"/>
*未来还将建成和推进[[汉江_(中国)|汉江]]高等级区域航道。<ref name="si"/>

== 物产 ==
*矿产有沙金、重晶石、金红石、叶蜡石、膨润土、硅石、矾矿等。
*可作工业原料的林产品有蚕茧、生漆、油桐等。
*果品有核桃、板栗、柑橘、猕猴桃等。
*中草药主要有天麻、黄连、当归、党参，绞股蓝等。
*土特产品有白厂丝、中华猕猴桃干酒等。

== 名人 ==
*[[祝塏|祝塏]]，清朝政治人物、進士出身。 生於道光六年（1826年），道光二十七年（1847年）中進士。曾任大順廣道、長蘆鹽運使等職。以戰功獲二品頂戴。 
*[[张紫樵|张紫樵]]，晚清进士。
*[[陈树藩|陈树藩]]，清末民初安康县人，曾任北洋政府督理陕西军务兼民政长。
*[[张宝麟|张宝麟]]，陕西省紫阳县洞河乡人。清末民国时期军事、商界人物。
*[[周正龙|周正龙]] 镇坪县城关镇文采村村民, 参见[[华南虎照片事件|华南虎照片事件]]
*[[谢亚龙|谢亚龙]] 2002-2004年挂职任市委副书记，之后被调任为中国足协专职副主席，国家足球运动管理中心主任、党委副书记。
*[[邹东涛|邹东涛]] 汉阴县人，现任中国社会科学文献出版社总编辑，曾任[[中国社会科学院研究生院|中国社会科学院研究生院]]常务副院长。
*[[杨达才|杨达才]]，微笑哥，表哥。
*[[张涛_(化学家)|张涛]]，化学家，[[中国科学院院士|中国科学院院士]]。
*[[柯尊平|柯尊平]]（1956年9月－），陝西安康人；[[西南交通大學|西南交通大學]]物理師資班畢業，工學博士。
*[[劉建明|劉建明]]（1953年－），陝西安康人，漢族，中華人民共和國政治人物、第十一屆全國人民代表大會陝西地區代表。
*[[南嶽懷讓|南嶽懷讓]]， 俗姓杜，唐代金州安康（今陝西安康）人，禪宗高僧，為六祖惠能門下，與青原行思形成兩大支派。
*[[徐啟方|徐啟方]]（1965年8月—），陝西漢濱人，1986年7月參加工作，1986年7月加入中共，在職中南財經政法大學工商管理碩士。
*[[馬青年|馬青年]]（1917年－1997年），又名馬青廉，經名撒力海，祖籍河南孟縣，出生陝西省安康縣人，中華人民共和國政治人物。
*[[徐山林|徐山林]]（1935年10月－），陝西安康市人，中華人民共和國政治人物。
*[[張寶麟|張寶麟]]（1883～1946），原名允贊，字仲仁，陝西省紫陽縣洞河鄉人。清末民國時期軍事、商界人物。
*[[雷鍾德|雷鍾德]]（1842年12月30日－1910年），派名潤身，字子脩，又字仲宣，號禹門，行二，陝西省興安府安康縣人，原籍西安府渭南縣，同治三年甲子科舉人，同治十年辛未科進士，欽點翰林院庶吉士。
*[[李襲志|李襲志]]，字重光，本隴西郡狄道人。五葉祖李景避地安康，稱金州安康縣人。隋朝、唐朝官員。
*[[劉小燕|劉小燕]]（1958年11月－），陝西安康人。女，漢族。1980年2月參加工作，1985年11月加入中國共產黨。
