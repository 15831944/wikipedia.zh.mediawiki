'''安徽中国徽州文化博物馆'''，又称'''中国徽州文化博物馆'''，是[[安徽省|安徽省]][[黄山市|黄山市]]的一所以徽州文化为主题的综合性博物馆，也是中国唯一全面展现徽州文化的博物馆，是安徽省第二大社会科学类综合性博物馆，馆址位于[[屯溪区|屯溪区]]迎宾大道50号。馆舍占地面积10.47万平方米，建筑面积1.4平方米<ref name="intro-ah">{{Cite web |url=http://ah.wenming.cn/agzyjyjd/201501/t20150105_2384952.htm |title=安徽中国徽州文化博物馆 |accessdate=2018-07-23 |author=安徽文明网 |date=2015-01-05 |publisher=中国文明网}}</ref>。博物馆沿革于'''黄山市博物馆'''，隶属黄山市文化委员会，新馆于2008年1月8日正式开馆。博物馆是[[国家一级博物馆|国家一级博物馆]]、[[全国古籍重点保护单位|全国古籍重点保护单位]]，文物近10万余件，其中[[徽墨|徽墨]]、[[歙砚|歙砚]]、[[新安画派|新安书画]]、徽州文献、[[徽州三雕|徽州三雕]]（木雕、石雕、砖雕）是馆内的特色藏品<ref name="intro-official">{{Cite web |url=http://www.hzwhbwg.com/index.php/list-2.html |title=本馆介绍 |accessdate=2018-07-23 |author=安徽中国徽州文化博物馆}}</ref>。

== 历史沿革 ==
安徽中国徽州文化博物馆的前身最早是'''徽州地区博物馆'''，博物馆于1963年成立，隶属于徽州地区文化局，馆址位于[[屯溪老街|屯溪老街]]。博物馆于1978年并入了徽州地区展览馆。1987年黄山市成立，徽州地区博物馆改名为'''黄山市博物馆'''，行政关系也相应转隶黄山市文化局。博物馆于当年迁址屯溪区徽山路24号坞山脚下占地16500平方米的新馆址，馆舍是钢筋水泥结构的殿宇式三层建筑，融合了[[徽派建筑|徽派建筑]]风格<ref name="history1">{{Cite web |url=http://www.ah.xinhuanet.com/2018-07/03/c_1123073480.htm |title=安徽中国徽州文化博物馆 梦里徽州此处寻 |accessdate=2018-07-23 |author=人民日报 |date=2018-07-03 |publisher=新华网}}</ref><ref name="history2">{{Cite web |url=http://www.beijingmuseum.gov.cn/art/2015/7/16/art_11061_261548.html |title=安徽中国徽州文化博物馆 |accessdate=2018-07-23 |author=安徽中国徽州文化博物馆 |date=2015-07-16 |publisher=北京数字博物馆}}</ref>。

2008年1月8日博物馆新馆舍建成投入使用，'''安徽省徽州文化博物馆'''成立，黄山市博物馆、黄山市文物商店并入安徽省徽州文化博物馆。2008年5月28日，根据[[国务院办公厅|国务院办公厅]]、[[安徽省人民政府|安徽省人民政府]]办公厅复函，同意安徽省徽州文化博物馆改名为'''安徽中国徽州文化博物馆'''<ref name="history1"/>。

2009年6月9日，安徽中国徽州文化博物馆被列入第二批全国古籍重点保护单位名单<ref name="gujibaohu">{{Cite web |url=http://www.nlc.cn/pcab/gjml/gjzdbhdw/wjtz/201412/t20141203_93310.htm |title=国务院关于公布第二批国家珍贵古籍名录和第而批全国古籍重点保护单位名单的通知(国发[2009]28号) |accessdate=2018-07-23 |author=国务院 |date=2009-06-09 |publisher=中国古籍保护网}}</ref>。2017年，安徽中国徽州文化博物馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref name="level">{{Cite web |url=http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html |title=关于发布第三批国家一级博物馆名单的通知 |accessdate=2018-07-23 |author=中国博物馆协会 |date=2017-01-20}}</ref>。此外，安徽中国徽州文化博物馆也是徽州三雕[[国家级非物质文化遗产|国家级非物质文化遗产]]项目保护单位、全国人文社会科学普及基地、安徽省社会科学知识普及基地、安徽省级及黄山市级爱国主义教育基地<ref>{{Cite web |url=http://hs.wenming.cn/wmcj/wmdw/201705/t20170519_3838587.html?COLLCC=3039268458& |title=安徽中国徽州文化博物馆被正式授予“国家一级博物馆” |accessdate=2018-07-23 |author=黄山市委宣传部, 黄山市文明办 |date=2017-05-19 |publisher=黄山文明网}}</ref>。

== 馆藏陈列 ==
=== 馆藏文物 ===
安徽中国徽州文化博物馆馆藏丰富，藏有文物近10万余件，其中国家一级文物34件、二级文物144件、三级文物3913件。藏品中囊括了具有徽州特色的[[徽墨|徽墨]]、[[歙砚|歙砚]]、[[新安画派|新安书画]]、[[徽州三雕|徽州三雕]]（木雕、石雕、砖雕），这其中包括有着“镇馆之宝”之称的中国考古出土唯一北宋时期墨锭——北宋文府墨<ref name="intro-official"/><ref name="relic1">{{Cite web |url=http://www.ahwang.cn/anhui/20180311/1746310.shtml |title=探访中国徽州文化博物馆“镇馆之宝”——北宋文府墨 |accessdate=2018-07-23 |author=吴永泉 |date=2018-03-11 |publisher=新安晚报}}</ref>，藏品总数和珍贵文物在安徽省均居第2名，仅次于[[安徽博物院|安徽博物院]]<ref name="intro-hs">{{Cite web |url=http://hs.wenming.cn/zt/2018bowuguan/201805/t20180524_5227185.html |title=安徽中国徽州文化博物馆简介 |accessdate=2018-07-23 |author=黄山文明网 |date=2018-05-24}}</ref>。
=== 陈列 ===
博物馆常设展览为《徽州人与徽州文化》，该主题的陈列分为<ref name="history1"/><ref name="intro-hs"/><ref>{{Cite web |url=http://www.hzwhbwg.com/index.php/list-9-1.html |title=常设展览 |accessdate=2018-07-23 |author=安徽中国徽州文化博物馆}}</ref>：
*走进徽州
*天下徽商：展示和介绍中国[[明朝|明]][[清朝|清]]两代最具势力和最有影响的商帮之一——[[徽商|徽商]]。
*礼仪徽州：展示徽州在古代特别是明清时期在文化、教育、学术方面取得的成就。
*徽州建筑：明清时期徽州府辖区具有地域特色的徽派建筑。
*徽州艺术：展示徽州特色工艺品，包括文房瑰宝、新安画派、徽派版画、篆刻艺术等。
*徽州科技：展示徽州人在医学、数学、地理学、物理等领域的成就。
此外，馆内相继举办过《徽州文房四宝展》、《黄宾虹、汪采白书画展》、《黄山市中共党史资料展》、《徽州雕刻艺术展》、和《徽派三雕艺术陈列》等文物专题展览<ref name="history2"/>。
