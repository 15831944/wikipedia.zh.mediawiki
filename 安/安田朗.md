'''安田朗'''（{{jpn|j=安田 朗|hg=やすだ あきら|rm=Yasuda Akira}}，{{bd|1964年|7月21日||}}），[[卡普空|卡普空]]公司出身，現在是個人工作室「（有）{{lang|ja|あきまん}}」代表的[[日本|日本]]資深[[遊戲開發者|遊戲開發者]]、[[插畫家|插畫家]]、[[人物設計|人物設計師]]、[[漫畫家|漫畫家]]、原[[動畫師|動畫師]]。男性。出身於[[北海道|北海道]][[釧路市|釧路市]]。筆名「'''{{lang|ja|あきまん}}'''（'''AKIMAN'''）」。

== 人物 ==
安田從小繪畫藝術受到[[薩爾瓦多·達利|達利]]和[[埃德加·德加|德加]]的影響。上小學的時候，他的美術資質被美術老師發掘和得到老師讚揚。然而上了中學之後，卻見識到職業插畫家[[安彥良和|安彥良和]]的插圖風格，而改變了安田繪畫的發展方向。之後安田想找尋繪畫相關的工作，但是當時在他的家鄉沒有在地就業的服務機關，於是作為新聞獎學生進入{{link-ja|專門學校東京設計學院|専門学校東京デザイナー学院|東京設計學院}}（現{{link-ja|專門學校東京Net Wave|専門学校東京ネットウエイブ|東京Net Wave}}）就讀。此外他有睡覺的嗜好，經常因為在課堂上睡覺對於老師的課題卻回答不出來。

但是在那之後，安田卻從東京設計學院申請辭退，而家人對於他退學一事表示尊重。退學之後的安田，短暫期間曾經在報社上班，原本將動畫的電腦繪製作為業餘的興趣。成為[[動畫師|動畫師]]之後，他開始參加下游動畫工作室承包《{{link-ja|金比羅小子|コンポラキッド}}》和《[[北斗神拳|北斗神拳]]》等等電視動畫的原畫工作，卻覺得所得工資低於搭電車通勤的交通費，結果只做數個月就離職。之後又曾經想過要當漫畫藝術家和插畫家，但說每天一直盯著天花板等理由而放棄。直到1985年，在友人的介紹下，安田從求職雜誌上看到遊戲開發公司[[卡普空|卡普空]]正在徵才，但他誤以為該公司是廣告設計公司，欲立即前往卻獲得錄取。

成為卡普空的社內員工之後，安田筆名決定使用AKIMAN（他在玩《[[勇者鬥惡龍_(遊戲)|勇者鬥惡龍]]》輸入遊戲名字「{{lang|ja|アキラマン}}」的縮減）。並從卡普空時期開始，他以藝術指導一職作為《[[街頭快打|街頭快打]]》和《[[街頭霸王II_－世界勇士－|快打旋風II]]》的提攜企劃者使其才能得到了發展的空間。並讓遊戲的知名度在海外國家世界傳開來。

現在，安田已經從卡普空離職，[[自由職業|自由身]]，並將AKIMAN這暱稱在主要著作上作為著作者名稱使用。在繪畫方面，安田說是受到安彥良和、[[法蘭克·法拉捷特|法蘭克·法拉捷特]]、{{link-en|N. C. 韋耶思|N. C. Wyeth}}的影響。並表示「我很喜歡安彥先生，是位想當動畫師未成卻讓我進入了遊戲業界的人」（出自《∀鋼彈全記録集2》）。

安田雖然作為[[人物設計|人物設計師]]而有名，但是讓他大顯身手的地方是在[[像素畫|像素畫]]繪製的技術，以《[[街頭霸王III|快打旋風III]]》登場的[[隆|隆]]和[[春麗|春麗]]等代表性的精緻角色圖像在遊戲業界內外得到許多粉絲的支持。安田在遊戲角色的像素畫製作與作業効率上，更要求不必要的裝飾不用畫出、明確掌握色彩的記號位置編排與強而有力的影像構成之魅力，集合了以上這些像素畫繪製的表現在格鬥遊戲中紛紛而出，成功吸引大批粉絲，並刺激了後進的創作者。但在製作的背後，有說他因為諸多事情多次發生沒有趕上已定的完成日期，並從開發現場中誕生出“等等安田先生”一句話在公司的員工之間不斷流傳的逸事。1992年，卡普空的[[電玩遊戲|電玩遊戲]][[製作人|製作人]][[岡本吉起|岡本吉起]]提出說「就讓我們引進像《[[波斯王子_(1989年遊戲)|波斯王子]]》的動畫之某種技術畫出角色動畫吧」「將賽璐上色的方式用在像素畫上最有效率了」，但安田覺得「我們需要一個可靠的繪圖方式來減少圖形本身的信息接收量，除非我們製作比以往更细致的動畫樣本，否則不可能有所作為。」「在開發《[[上尉密令|上尉密令]]》的時候，我不斷被追究調整至柔和{{notetag|在角色圖像的輪廓邊緣加上暗色系色彩陰影表現出角色立體感，是卡普空《[[街頭霸王II_－世界勇士－|快打旋風II]]》亮相時期使用的傳統像素畫繪製技法。}}的程度已經我是極限，想說我不能再這樣做了」，本來安田是這樣子想，但在岡本面對壓力而無法避免之下，欲在新作品《[[魔域幽靈|魔域幽靈]]》開發時決定挑戰進行賽璐上色。安田說現在回想起來在當時徹底的進行人物設定監修是值得的，至今，據本人表示《魔域幽靈》已經成為令人信服的賽璐上色與動畫樣本之實驗作<ref>《快打旋風 X 鐵拳 Artworks》，第184頁，[[卡普空|卡普空]]，2012年4月5日發行 ISBN 978-4862333445</ref>。後來，《魔域幽靈》系列的賽璐上色手法之特色也活用推在之後出的《{{link-ja|漫威超級英雄 (遊戲)|マーヴル・スーパーヒーローズ|漫威超級英雄}}》對戰格鬥系列上。

安田在卡普空對後輩的栽培也有所貢獻，有來自跟他來自社內設計室出身的[[西村絹|西村絹]]<ref name="page234">《Capcom Design Works －{{lang|ja|カプコンデザインワークス}}－》，Fami通書籍編輯部 (著)、卡普空 (監修)，2001年8月發行，[[enterbrain|enterbrain]]出版，第234頁 ISBN 978-4-757-70412-1</ref>及{{link-ja|BENGUS|BENGUS}}等等人氣設計師輩出。此外《[[龍息之焰|龍息之焰]]》負責人物設計的[[吉川達哉|吉川達哉]]說自身的目標是要脫離安田的美術設計之影響，因此在參加《[[龍戰士II_使命之子|龍息之焰2]]》的時候畫風一轉，成為吉川現在自身特有的風格。儘管促銷用插圖和人物設計很容易被外界誤解全部都說是由安田擔當，但是卡普空所屬時期的安田當時擔任職位的是董事和設計主管，並非所屬任何一間設計室。

此外，安田說自身受到影響的還有功夫類電影、動畫師[[荒木伸吾|荒木伸吾]]作品及《[[甜心戰士|甜心戰士]]》。並表示一旦開始繪畫時就不會停止，完成一張圖需要三個星期的時間。

== 經歷 ==
; 1980年
* {{link-ja|北海道釧路北陽高等學校|北海道釧路北陽高等学校}}入學。
; 1983年
* {{link-ja|專門學校東京設計學院|専門学校東京デザイナー学院}}商業設計科入學。
; 1984年
* 同校中退。
; 1985年
* 進入卡普空東京支社作為工讀生採用後，第一份工作是負責《{{link-ja|SIDE ARMS (射擊遊戲)|サイドアーム (シューティングゲーム)|SIDE ARMS}}》的人物設計與背景繪製。除了這款遊戲之外，還曾經繪製過[[紅白機|Super Famicon]]機遊戲《[[1942_(遊戲)|1942]]》的包裝插圖。
; 1986年
* 轉屬卡普空大阪總部第3企劃室。
; 1988年
* 擔當《{{link-ja|失落世界 (遊戲)|ロストワールド (ゲーム)|失落世界}}》的企劃和[[平面設計|平面設計]]。這是安田他身為企劃所開發而製作出來的第一個作品。
; 1989年
* 參加《街頭快打》製作。他在這款遊戲提出了角色設定和Bonus Stage的點子，不僅在平面設計上，在企劃上也有所貢獻。
; 1990年
* 幫《[[魔劍_(1990年遊戲)|魔劍]]》繪製促銷宣傳用的海報插圖{{notetag|安田在幫大型電玩機台版繪製插圖時，有畫出女性的[[乳頭|乳頭]]，後來出[[超級任天堂|超級任天堂]]移植版本的時候已經將這個隱私部位去除。}}。
; 1991年
* 幫《快打旋風II》設計出[[春麗|春麗]]和{{link-ja|蓋爾 (快打旋風)|ガイル (ストリートファイター)|蓋爾}}等等許多象徵性的角色，以及繪製出玩家操控角色的像素圖片。之後同名作品系列化，築起了卡普空的大型電玩機台的黃金時代。
; 1994年
* 參加《魔域幽靈》的製作，作為圖形製作團隊的中心人物。
* 幫《[[X戰警_磁場原子人|X戰警 磁場原子人]]》角色[[金鋼狼|金鋼狼]]的像素畫進行繪製。同時幫忙繪製促銷宣傳用的海報插圖。
; 1997年
* 第一次為《快打旋風III》的所有角色做造型設計，並幫所有角色繪製玩家選角畫面用的圖像跟對戰畫面中的動態像素畫。
; 1998年
* 個人工作室「有限公司AKIMAN」成立。並作為自身的辦公室，從卡普空的設計室得到兩張桌子。
* 擔當[[富士電視網|富士電視網]]播出、[[日昇動畫|日昇動畫]]製作電視動畫《[[∀GUNDAM|∀鋼彈]]》人物設計。從此與[[富野由悠季|富野由悠季]]加深友誼。
; 2001年
* 移居[[美國|美國]][[加利福尼亞州|加利福尼亞州]][[聖迭戈縣|聖地牙哥郡]]的[[卡爾斯巴德_(加利福尼亞州)|卡爾斯巴德]]，並參加《[[紅色死亡左輪|碧血狂殺]]》的開發製作，同時擔任該作的人物設定。
* 個人第一本畫冊《安田朗∀鋼彈DESINGS》發售，由[[角川書店|角川書店]]出版。
; 2002年
* 受邀擔任[[WOWOW|WOWOW]]、日昇動畫製播電視動畫《[[帝皇戰紀|帝皇戰紀]]》的機械設計。
; 2003年
* 結束《碧血狂殺》的開發製作回到日本。之後從卡普空離職，並移居至東京。成為自由身。
