'''安妮宝贝'''（{{bd|1974年|7月11日|catIdx=L勵}}），本名'''励婕'''，出生于[[浙江|浙江]][[宁波市|宁波]]，中國當代女性[[小說家|小說家]]、[[散文家|散文家]]和[[編輯|雜誌編輯]]。

2014年6月16日，安妮宝贝在[[新浪微博|新浪微博]]上宣布改笔名为“'''庆山'''”。

== 生平 ==
安妮宝贝1992年高中毕业于浙江省宁波中学。在从事写作之前，曾在北京从事过[[金融|金融]]、[[编辑|编辑]]等工作<ref name="nanfang">{{cite web|author =名牌|title =安妮宝贝：突然十年便过去|url =http://nf.nfdaily.cn/mpzz/content/2011-10/17/content_31548962.htm|publisher =南方网|accessdate =2012-05-10|language =zh-cn|deadurl =yes|archiveurl =https://web.archive.org/web/20120305185344/http://nf.nfdaily.cn/mpzz/content/2011-10/17/content_31548962.htm|archivedate =2012-03-05}}</ref>。2000年，她曾在榕树下网站工作，榕树下网站创始人朱威廉专门为她成立了“安妮宝贝工作室”。

1998年開始，她在[[榕樹下|榕樹下]]發表了《七月和安生》、《告別薇安》等短篇小說，引發讀者關注。安妮寶貝的作品以描寫都市男女愛情為主，文章風格傾向於[[悲觀主義|悲觀主義]]和頹廢主義。安妮寶貝早期作品屬於“網絡文學”，而後期的作品則走向傳統的出版。

2014年1月，在中国大陆广电总局的电影备案中显示安妮宝贝的小说《[[七月与安生_(电影)|七月与安生]]》将拍电影。翌年六月，香港导演[[陈可辛|陈可辛]]被爆担任电影监制<ref>{{Cite web|url = http://ent.163.com/14/0212/16/9KT7BCD300032DGD.html|title = 安妮宝贝小说将被搬上银幕 改编难度大|accessdate = 2015-07-20|author = |date = 2015-06-25|publisher = 网易娱乐|language = zh-cn}}</ref>。

== 個人生活 ==
安妮寶貝具體婚期不詳，其丈夫資料亦不詳，媒體上僅僅有安妮寶貝在2008年接受《深圳青年》採訪時透露她在2007年10月1日誕下一個女兒<ref>{{cite journal|author = 励婕|coauthors = 林皓哲|title = 安妮宝贝，得夫如此万事足|journal = 深圳青年|year = 2008年|issue = 15|url = http://www.qikan.com.cn/Article/szqn/szqn200815/szqn20081519.html|accessdate = 2012-05-09|language = zh-cn}}</ref>。

== 评价 ==

=== 正面评价 ===
安妮宝贝语言凝练，短促平快，极具张力。运用大量比喻和搭配等修辞手法，作品呈现出空灵绮丽、简洁蕴藉的风格。随着阅历的加深丰富，安妮宝贝作品日渐成熟，风格上从阴郁逼仄、桀骜不驯，渐趋明朗清新、稳重平和。

=== 负面评价 ===
凤凰网读书作者张丙认为安妮宝贝是一个感性而缺乏逻辑的人，其文章中经常有对词义理解出现偏差导致的病句<ref>{{Cite web|url = http://book.ifeng.com/shupingzhoukan/special/duyao54/wenzhang/detail_2011_09/07/9012860_0.shtml|title = 张丙：挑挑安妮宝贝的毛病|accessdate = 2015-07-20|author = 张丙|date = 2011年9月7日|publisher = 凤凰网读书|language = zh-cn}}</ref>。
