'''安乐死'''（{{lang-en|Euthanasia}}，源自於{{lang-el|εὐθανασία}}，「好的死亡」；{{lang|el|εὖ}}為「好的」，{{lang|el|θάνατος}}為「死亡」，此名稱與實際作法不斷地受到爭議）是一种给予患有不治之症的人以无痛楚、或更嚴謹而言「盡其量減小痛楚地」致死的行为或措施，一般用于在个别患者出现了无法医治的长期显性病症，因病情到了晚期或不治之症，对病人造成极大的负担，不愿再受病痛折磨而采取的了结生命的措施，经过医生和病人双方同意后进行，为减轻痛苦而进行的提前死亡。

但无可否认，安乐死确有可能会增加处于无尽痛苦之中的诸如病人、穷人或者其他生存压力过大人群的自杀倾向，因其主要作用正是为了使自身能够永久性的摆脱这样的痛苦之折磨。被安乐死是指在未经病人自身同意的情况下，由其亲属向医生申请或医生主观判断实行安乐死，违背死者本人意愿的死亡事件也会因之而发生。

== 醫學界 ==
目前醫學界對「安樂死」無統一的定義，不過在操作層面，主要可分為<ref>陳明進，[http://jtc.nttu.edu.tw/upload/admin/journal_paper/012a-09.pdf 安樂死問題初探] {{webarchive|url=https://web.archive.org/web/20131228163221/http://jtc.nttu.edu.tw/upload/admin/journal_paper/012a-09.pdf |date=2013-12-28 }}，台東師院學報，12卷1期，頁211-241，2001年6月。</ref>：
;* 主動安樂死（Active Euthanasia）
:: 主動為病人結束生命（例如透過注射方式）。
;* 被動安樂死（Passive Euthanasia）
:: 被動安樂死是停止療程（例如除去病人的維生系統或讓病人停止服藥），使其自然死亡。
;* 協助自殺（Assisted suicide）
:: 協助自殺是病人主動要求，並親自執行結束生命（例如通過服藥方式）。

== 執行與否 ==
以[[荷蘭|荷蘭]]為例，目前在荷蘭要執行安樂死，必須出自病人的个人意願，并且有醫生證明病人正在處於「不能減輕」和「不能忍受」的痛苦中，醫生和病人之間也得先達成共識，確認安樂死已經是他們的唯一選擇<ref>孫效智，1996年12月，[http://www.christianstudy.com/data/pastoral/euthanas04.html 安樂死的倫理反省]，《國立臺灣大學文史哲學報》，第四十五期，頁85-113。</ref>。安乐死在许多国家引发了很大的争议（例如[[美國|美國]]的[[特麗·夏沃|特麗·夏沃]]案）。

一、目前已立法容許主動安樂死的國家（或地区）：
* 歐洲：{{NED}}、{{BEL}}、{{LUX}} (醫師不得親自注射，需由病人自行服藥)
* 南美洲：{{COL}} (醫師需親自注射)

二、目前已立法容許被動安樂死的國家（或地区）：
* 歐洲：{{GBR}}、{{IRL}}、{{FIN}}、{{NOR}}、{{FRA}}、{{ESP}}、{{AUT}}、{{GRE}}、{{DEN}}、{{SWE}}
* 亞洲：{{KOR}}、{{flagicon|ROC}}[[台灣|台灣]]（[[中華民國|中華民國]]）、{{flagicon|India}} [[印度|印度]]

三、目前已立法容許[[協助自殺|協助自殺]]的國家<ref>
[https://en.m.wikipedia.org/wiki/Legality_of_euthanasia]</ref>

* 歐洲：{{GER}}、{{flagicon|SUI}} [[瑞士安樂死議題|瑞士]]
* 北美洲：{{flagicon|USA}} [[安樂死_(美國)|美國部分州]](華盛頓特區、加州、奧勒岡州、科羅拉多州、佛蒙特州、華盛頓州、蒙大拿州、夏威夷州)、{{CAN}}<ref>[http://www.chinapress.com.my/20170420/%E8%AB%8B%E6%B1%82%E5%AE%89%E6%A8%82%E6%AD%BB%E6%9C%AA%E5%A6%82%E9%A1%98-%E5%8A%A0%E6%8B%BF%E5%A4%A727%E6%AD%B2%E5%81%A5%E7%BE%8E%E5%B8%A5%E5%93%A5%E8%B5%B0%E4%BA%86/ 加拿大在2016年6月，通過了稱為Bill C-14的允許病人在醫生協助下死亡俗稱安樂死的法律]</ref>
*大洋洲：{{AUS}}（{{flagicon|Victoria}} [[維多利亞州|維多利亞州]]）<ref>[http://www.upmedia.mg/news_info.php?SerialNo=30108 澳大利亚昨日通過安樂死合法化]</ref>

[[丹麥|丹麥]]於1992年通過「預立遺囑法」，由[[哥本哈根大學|哥本哈根大學]]醫學院管理，並在1998年時通過「病患權利法」，確認其約束性。任何末期重症、嚴重意外等無自主能力的患者，其負責醫師都必須向病人本人詢問，該患者是否有登錄預立遺囑。如果有登錄，則根據預立遺囑內的條件進行安樂死。<ref>[http://www.epochtimes.com/b5/5/3/28/n869036.htm 安樂死 丹麥強調預立遺囑約束性]，大紀元，2005年3月28日</ref>

[[德國|德國]]則禁止主動安樂死，但是重症病人可以以口頭形式或者書面形式要求被動安樂死。當病人因病重無法表達意願時，其親屬可以代替他做出決定。

[[比利时眾議院|比利时眾議院]]于2014年2月13日通過安樂死合法範圍擴及未成年人一案，將成為繼荷蘭之後，全球第2個在嚴格條件下許可未成年的孩童实施安樂死的國家。<ref name="cna_201402140002-1">[http://www.cna.com.tw/news/firstnews/201402140002-1.aspx 全球第2 比利時允孩童安樂死]，中央社，2014年2月14日</ref><ref name="appledaily_20140215_35642465">{{cite news |url =http://www.appledaily.com.tw/appledaily/article/international/20140215/35642465/ |title= 全球首例 比利時解除安樂死限齡 - 18歲以下能適用 旁聽民眾喊「謀殺」 | date = 2014年2月15日 | author = 李寧怡 | publisher = [[蘋果日報_(台灣)|蘋果日報 (台灣)]] }}</ref>

2017年10月22日，[[韩国|韩国]]保健福祉部称，从2017年10月23日至2018年1月15日将试行《维持生命医疗决定法》，临终患者可以在病情严重并且不可逆转的情况下，自己决定是否继续接受维持生命的治疗。<ref>{{Cite web|url=http://www.xinhuanet.com/world/2017-10/23/c_129725195.htm|title=韩国今起试行《安乐死法》 年满19岁即可提前决定-新华网|accessdate=2018-09-01|work=www.xinhuanet.com}}</ref>

== 安樂死辯論 ==

{{来源请求|安樂死辯論往往集中在一些關鍵問題上。 根據安樂死反對者Ezekiel Emanuel ，安樂死的支持者提出了四個主要論點：
* 人們有自決權 ，因此應該允許他們選擇自己的命運; 
* 協助受試者死亡或許好過他們繼續受苦; 
* 經常允許的被動安樂死與主動安樂死之間的區別，這種安樂死不是實質性的（或者基本原則 - 雙重效應原則 - 不合理或不合理）; }}

安樂死支持者經常指向像[[荷蘭|荷蘭]]和[[比利時|比利時]]這樣的國家，以及像安樂死合法化的[[俄勒岡州|俄勒岡州]]這樣的地方，認為它基本上沒有問題。

同樣，[[芝加哥|芝加哥]]第55任市長[[拉姆·伊曼纽尔|拉姆·伊曼纽尔]]認為，反對安樂死的主要觀點有四個：
* 並非所有的死亡都是痛苦的; 
* 提供替代方案，如停止積極治療，並結合使用有效的止痛藥; 
* 主動安樂死和被動安樂死之間的區別在道德上是重要的;
* 使安樂死合法化會使社會處於一個滑坡 ，將導致不可接受的後果。

== 接受或寻求安乐死的著名个案 ==
* [[西格蒙德·弗洛伊德|西格蒙德·弗洛伊德]]（1856-1939年，奧地利出生、英國接受自愿主動安乐死）
* [[特麗·夏沃|特麗·夏沃]]（1963-2005年，美國、非自愿被動安乐死）
* [[鄧紹斌|鄧紹斌]]（1969-2012年，香港人，頸部以下癱瘓、因为敗血病于2012年12月9日離世）
*{{link-ja|東海大學安樂死事件|東海大学安楽死事件}}（1991年，日本案例）
* [[克里斯汀·德·迪夫|克里斯汀·德·迪夫]]，諾貝爾醫學獎得主，2013年5月4日採取安樂死離世，享年96歲。
* [[雨果·克劳斯|雨果·克劳斯]]，比利時作家，2008年離世，享年79歲。
*[[巴金|巴金]]晚年患有帕金森氏症、慢性氣管炎、高血壓、惡性間皮細胞瘤等多種疾病，曾经表示要求安樂死。2005年10月17日在上海逝世，享年101歲。
*王明成因目睹母親夏素文受肝病折磨，要求醫院于1986年6月29日為她實施安樂死。他和執行安樂死的醫生蒲連升被控故意殺人罪，成為中國內地第一宗安樂死案例。 1991年4月6日，漢中市人民法院判决蒲連升、王明成虽然违法，但不构成犯罪。<ref>{{Cite web |url=http://news.sohu.com/28/97/news209869728.shtml |title=我国首例“安乐死”案始末 |accessdate=2018-04-10 |work=news.sohu.com}}</ref>2003年，王明成罹患胃癌后也要求安乐死未果，于8月3日去世。<ref>{{Cite web |url=http://ah.sina.com.cn/news/2015-01-27/detail-icczmvun5444532.shtml |title=我国首例安乐死详尽报道 王明成与“安乐死”_新浪安徽_新浪网 |accessdate=2018-04-10 |work=ah.sina.com.cn}}</ref>
*[[王曉民|王曉民]] 二戰後台灣第一個受社會廣泛注目的植物人案例。1963年王曉民因車禍成為植物人，長期臥床由父母照料數十年之久，母親趙錫念擔心自己死後王曉民無人照護，於1983年向立法院請願，要求速訂安樂死法律，引發立法委員激烈辯論，由於多數委員反對，且學者對於日後可能的安樂死濫用存有疑慮，因此未獲結論，王曉民直到2010年逝世都未清醒。
*[[傅達仁|傅達仁]] （1933年4月3日－2018年6月7日，自願主動安樂死）台灣知名體育主播傅達仁於2016年12月6日籌辦「生前追思會」，上書總統蔡英文呼籲立法院通過安樂死法案，願以自身當台灣合法安樂死首例，其主張安樂死可以免除自身及家人痛苦，同時減少因高齡社會配套長照政策所造成的國家資源浪費。2018年6月7日在瑞士蘇黎世實行安樂死，享壽85歲。<ref>{{Cite web |url=http://ent.ltn.com.tw/news/breakingnews/2451271 |title=「生命鬥士」傅達仁瑞士安詳離世 享壽85歲 |accessdate=2018-06-07 |work=自由時報}}</ref>
