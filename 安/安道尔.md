'''安道尔公国'''（{{lang-ca|Principat d'Andorra}}），也译作'''安道拉[[親王國|親王國]]'''，中文通稱'''安道爾'''，为一[[袖珍国家|袖珍国家]]，国土面积468平方公里。是[[西欧|西]][[南欧|南欧]]的[[內陸國家|內陸]][[親王國|親王國]]，位于[[比利牛斯山脉|比利牛斯山脉]]东南部，毗邻[[法国|法国]]和[[西班牙|西班牙]]。首都安道尔城（Andorra la Vella）。官方语言为[[加泰罗尼亚语|加泰罗尼亚语]]，通用语言为[[西班牙语|西班牙语]]和[[法语|法语]]。国内居民多信奉[[天主教|天主教]]。

安道尔高山峡谷遍布全境，全国平均海拔高达{{convert|1100|m}}，是欧洲地势最高的国家。

安道尔宪法规定，安道尔为[[君主立宪制|君主立宪制]]国家，首相为[[政府首脑|政府首脑]]。安道尔没有军队和自卫队，只存在警察，所以国防由法国和西班牙一起负责。安道尔主要有两大政党：安道尔民主党（Demòcrates per Andorra）和社会民主党（Partit Socialdemòcrata），安道尔民主主义者党现为安道尔的执政党。安道尔的旅游、商业及金融业极其发达，服务业在1950年代很快取代了传统农牧业，成为经济支柱。安道尔的[[旅遊業|旅遊業]]相當發達，亦為[[避稅天堂|避稅天堂]]，因属山地气候，冬季寒冷漫长，高山积雪一年可达8个月，是個名副其实的滑雪胜地。安道尔也是全世界[[各国人口预期寿命列表|國民預期壽命]]第四長的國家。其國際代碼為AD。

== 名称 ==
“安道爾”（{{lang|en|Andorra}}）源自[[納瓦拉|納瓦拉]]的''andurrial''，意含為覆蓋灌木的土地（{{lang|en|shrub-covered land}}）。安道尔本名「Valls d'Andorra」，字面意义为「安道拉谷地」，大概因其境内有多条山谷。

== 历史 ==
{{main| 安道爾歷史}}
[[File:Borís_Skósyrev.jpg|left]]
傳說[[法兰克王国|法兰克王国]][[查理大帝|查理大帝]]為回報安道尔人与[[摩尔人|摩尔人]]作战，頒布特許狀，在安道爾設立教區，命[[西班牙|西班牙]][[烏格爾|烏格爾]]的伯爵擔任主教一職。11世纪時，乌格尔主教和法国开始争论安道尔的主权问题。

1278年双方达成协议，安道爾由法国[[富瓦|富瓦]]伯爵和乌格尔主教兩方共同擁有主權（協議內的範圍是今日安道爾政治和國土的雛型），後來富瓦伯爵的主權轉到[[纳瓦拉|纳瓦拉]]國王手中，法王[[亨利四世_(法兰西)|亨利四世]]于1607年下令法国国家元首与乌格尔主教同时擁有安道尔的主权。

1933年法國以世界[[經濟大蕭條|經濟大蕭條]]導致社會動盪之名，佔領安道爾。在1934年7月12日，一名俄罗斯冒险家名为鲍里斯·斯科瑟廖夫（{{lang-ru|Борис Скосирев}}）於烏格爾向大眾宣佈，他擁有安道爾的主權，同時對烏格爾主教宣戰。但他在八天之後的7月20日即被西班牙當局逮捕，并立刻被驅逐出境。1936到1940年間，法國因[[西班牙內戰|西班牙內戰]]而派兵駐守安道爾。

[[二次大戰|二戰]]期間，安道爾維持中立，成為[[维希法国|维希法国]]與[[西班牙国|西班牙]]之間的走私重鎮，二戰中西班牙亦曾想佔領安道尔，但西班牙政府沒有這樣做。

安道尔长时间与世隔绝，只保持對法国和加泰罗尼亚的关系。近年来安道爾致力改善交通和通訊，並大力發展[[旅遊業|旅遊業]]。1993年，安道爾舉行[[公投|公投]]，通過安道爾憲法，並成為[[聯合國|聯合國]]的一員。

== 政治 ==
{{main|安道尔政治}}
安道​​爾[[憲法|憲法]]於1993年生效，憲法定明安道爾為[[君主立憲制|君主立憲制]]國家，[[國家元首|國家元首]]設兩位，稱為亲王，分別是[[法國總統|法國總統]]（現任：[[埃马纽埃尔·马克龙|艾曼紐·馬克宏]]）與[[加泰隆尼亞|加泰隆尼亞]][[天主教烏赫爾教區|羅馬天主教烏格爾教區]]（{{lang-ca|Bisbat d'Urgell}}）的主教（現任：[[霍安·恩里克·比韋斯·西西利亞|霍安·恩里克·比韋斯·西西利亞]]），亲王為虛位職。

立法則由議會負責，議會設有個28席位，一半由全國選舉選出，另一半則由各地區選出。[[政府|政府]]由議會多數黨組成。

安道​​爾的國防由法國和西班牙負責。

== 地理 ==
{{main|安道爾地理}}

[[File:Andorra_topographic_map-en.svg|230px]]
安道尔位于[[比利牛斯山|比利牛斯山]]东部，境内高山环绕，地形崎岖，平均海拔是{{convert|1996|m}}，最高峰[[科马佩德罗萨山|科马佩德罗萨山]]，海拔{{convert|2,946|m}}。整个国家被三条Y形的狭谷分割，在这里两条河会合成[[瓦利拉河|瓦利拉河]]並流向加泰罗尼亚。

安道尔氣候屬於[[山地气候|山地气候]]，因境內地勢影響，夏季涼爽，大部分地区冬季漫长寒冷，年平均气温[[攝氏|攝氏]]{{convert|9.1|℃}}。

== 經濟 ==
[[File:Andorra_Vall_dels_Cortals_05_JMM.JPG|230px]]]]

[[File:Andorra_Encamp_rio_JMM.JPG|thumb]]
[[旅遊業|旅遊業]]佔[[國內生產毛額|GDP]]的80%，為安道尔經濟的主要來源。每年約有九百萬人因免稅及冬夏兩季的度假設施前往安道爾旅遊。近來因法國和加泰罗尼亚在其境內實施經濟改革，令安道爾失去一些優勢。

作为免稅天堂，安道尔的[[金融业|金融业]]也很重要。[[農產品|農產品]]因僅有2%土地可耕種的關係，大多數自國外進口；[[畜牧業|畜牧業]]以[[羊部|羊部]]為主，工业产品主要有[[烟|烟]]和家具。

安道尔並非[[欧盟|欧盟]]成員國，公民在沒有有效簽證下也不能自由進出被包圍的[[申根區|申根區]]，但兩者間有特殊关系（歐盟對安道爾出口的產品實施零關稅）。此外安道爾本身並無貨幣，1999年之前採用法國[[法郎|法郎]]及西班牙[[比塞塔|比塞塔]]，之後使用[[歐元|歐元]]。安道爾於2011年6月30日與歐盟達成協議前不能自製歐元。安道爾自2013年起鑄造自身的歐元，總額為230萬歐元<ref>http://myeurohobby.eu/profiles/blogs/andorra-euro-coins-a-step</ref>。

自然資源有[[水力發電|水力發電]]、[[礦泉水|礦泉水]]、[[木材|木材]]、[[鐵|鐵礦]]及[[鉛|鉛]]等。

== 人口 ==
{{main|安道尔人口}}

安道爾只有33%的人口是[[加泰羅尼亞人|安道爾人]]，外國僑民以[[西班牙人|西班牙人]]為主（佔43%），還有[[葡萄牙人|葡萄牙人]]（11%）和[[法國人|法國人]]（7%）等。

== 語言 ==
[[官方語言|官方語言]]為[[加泰羅尼亞語|加泰羅尼亞語]]，與[[加泰羅尼亞|加泰羅尼亞]]相同。多數民眾亦能通[[西班牙語|西班牙語]]。[[英語|英語]]、[[法語|法語]]是當地最重要的[[外語|外語]]。

==體育==
安道尔以其在冬季运动项目上的表现而著称。流行的体育项目包括足球、英式橄榄球、篮球和滑轮曲棍球。

安道尔参加滑轮曲轮球欧洲杯及[[滑轮曲棍球世界杯|世界杯]]的比赛。2011年，安道尔主办了当届欧洲联赛的八强赛。

由于国内人口少，[[安道尔国家足球队|安道尔国家足球队]]在国际赛事中战绩不佳。[[安道尔足球总会|安道尔足球总会]]成立于1994年，负责组织足球（包括[[安道尔甲组足球联赛|甲级联赛]]、[[安道尔杯|安道尔杯]]、[[安道尔超级杯|安道尔超级杯]]）及室内五人制足球的国家级比赛。安道尔城的俱乐部{{tsl|en|FC Andorra||安道尔FC}}，成立于1942年，目前征战于[[西班牙足球联赛系统|西班牙联赛]]。

橄榄球是安道尔的传统运动，这是因为受到法国南部的影响。绰号“Els Isards”的安道尔国家橄榄球队在英式橄榄球和七人制橄榄球的国际赛场上都有亮眼表现。安道尔城的橄榄球队“VPC安道尔XV”参加法国冠军杯的比赛。

篮球运动自1990年代起在安道尔逐渐流行，当时安道尔篮球队是[[西班牙篮球甲级联赛|西班牙篮球甲级联赛]]的成员。阔别18年后，该队于2014年重返西甲联赛。

安道尔国内开展的其他项目还包括自行车、排球、柔道、澳式橄榄球、手球、游泳、体操、网球和赛车。2012年，安道尔组成了第一支板球队伍，史上第一场比赛是主场迎战荷兰俱乐部“Fellowship Of Fairly Odd Places Cricket Club”，这场比赛在海拔1300米的场地进行。

安道尔于1976年第一次参加奥运会，此后每一届冬奥会都不曾缺席。安道尔还参加[[欧洲小国运动会|欧洲小国运动会]]，并且于1991年及2005年两度承办该赛事。

作为加泰罗尼亚文化区的一部分，安道尔也拥有一支[[叠人塔|叠人塔]]队伍。安道尔人塔队由加泰罗尼亚人塔协会组织，其大本营在小城[[圣科洛马|圣科洛马]]。

===重要成绩===
Ariadna Tudel Cuberes和Sophie Dusautoir Bertrand在2009年欧洲滑雪登山锦标赛女子团体比赛中获得铜牌。Joan Verdu Sanchez在2012冬季青年奥运会上获得了高山滑雪的铜牌。2015年，Marc Oliveras在世界大学生冬季运动会上获得高山滑雪银牌，而Carmina Pallas斩获女子组一银一铜两枚奖牌。
