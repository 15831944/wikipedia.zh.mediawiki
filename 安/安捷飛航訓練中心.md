'''安捷飛航訓練中心'''為[[中華民國|中華民國]]第一間通過[[交通部民用航空局|交通部民用航空局]]五階段審查作業認證，同時擁有學科及飛行術科的民營飛航訓練中心。
營運總部設立在[[台北市|台北市]][[內湖區|內湖區]]，航機務主基地設置在台東[[豐年機場|豐年機場]], 並在美國德州喬治城設有衛星基地。

== 事紀 ==

*2014年9月 - 獲得台灣民航局營業許可
*2015年7月 - 與台灣[[威航|威航]]簽訂培訓合約
*2015年8月 - 與[[台灣虎航|台灣虎航]]簽訂培訓合約
*2016年5月 - 與澳商Ansett Aviation Training合資成立模擬機訓練中心
*2017年4月 - 美國德州分校成立
*2018年1月 - 与法國[[国立民用航空学院|国立民用航空学院]] ENAC 簽訂合作意向書 
*2018年1月 - 與台灣[[華信航空|華信航空]]簽訂培訓合約
*2018年7月 - 獲得柬埔寨國家民航局(SSCA)認證的飛航訓練中心(ATO)<ref>{{cite web|url=https://news.rti.org.tw/news/view/id/415750|title=機師培訓台灣首獲柬埔寨官方認證|publisher=中央廣播電台}}</ref>
*2018年12月 - 澳商Ansett Aviation Training買回模擬機訓練中心所有股份

== 機隊 ==

== 事故 ==
*2016年5月5日,安捷飛航訓練中心一架 DA40NG 型機，國籍標誌及登記號碼 B-88002，於 0802 時由臺東豐年機場起飛執行空域單飛訓練。該機於 0901 時返場落地。於落地過程中航機鼻輪折斷脫落前機身底部磨損，無人傷亡。<ref>{{cite web|url=https://www.asc.gov.tw/upload/cont_att/f4545c72-36b4-4a5b-94a3-9455a585b821.pdf|title=調查中飛航事故彙整表|publisher=中華民國行政院飛航安全委員會}}</ref>
*2018年7月9日，安捷飛航訓練中心一架 DA40NG 型機，國籍標誌及登記號碼 B-88123，於1632時由臺東豐年機場起飛，執行訓練任務，目的地為高雄機場，機上載有飛航教官1人，飛行學員2人，在1702時告知航管發動機出現問題，隨後迫降在海上，飛機的雷達光點消失處，約在高雄機場西南方11浬，無人受傷。<ref>{{cite web|url=https://www.asc.gov.tw/main_ch/docDetail.aspx?uid=227&pid=227&docid=987|title=安捷飛航訓練中心AFA72飛航事故進度報告|publisher=飛航安全調查委員會}}</ref>
