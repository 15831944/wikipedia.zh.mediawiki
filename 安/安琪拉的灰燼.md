《'''安琪拉的灰燼'''》（{{lang-en|''Angela's Ashes''}}）是[[愛爾蘭|愛爾蘭]]裔[[美國|美國]]作家{{tsl|en|Frank McCourt|法蘭克．麥考特}}（1930年8月19日-2009年7月19日）在1996年完成的[[回憶錄|回憶錄]]文學作品。書中內容主要敘述作者童年趣聞軼事，從幼兒時期在[[紐約|紐約]][[布魯克林|布魯克林]]生活開始描述，主要篇幅描述他在[[愛爾蘭|愛爾蘭]][[利默里克|利默里克]]勞工生活，其中也包括他掙扎於貧窮生活與父親[[酗酒|酗酒]]問題。

== 創作成就 ==

本書在1996年出版，並於隔年1997年贏得[[普立茲獎|普立茲獎]]的{{tsl|en|Pulitzer Prize for Biography or Autobiography|傳記文學獎}}、{{tsl|en|National Book Critics Circle Award|國家書評獎}}及{{tsl|en|Los Angeles Times Book Prize|洛杉磯時報書卷獎}}，直到2018年仍是全球最大線上購物[[亞馬遜公司|亞馬遜網站]]，愛爾蘭[[傳記|傳記]]類書籍暢銷榜首<ref>{{Cite web | title = Angela's Ashes: A Memoir Paperback | url = https://www.amazon.com/Angelas-Ashes-Memoir-Frank-McCourt/dp/068484267X | publisher = Amazon.com| language = en| date = 1999年5月25日| accessdate=2018年6月11日}}</ref>。本書在世界各國譯為25種語言，全球銷售量超過四百萬本<ref>{{Cite news| url = https://tw.appledaily.com/finance/daily/20090721/31800232| title = 《安琪拉的灰燼》作者病逝 | publisher = 蘋果日報| location = 台灣 | date = 2009年7月21日}}</ref><ref>{{Cite web| title = 安琪拉的灰燼 Angela's Ashes| url = http://www.eslite.com/product.aspx?pgid=1001272362579919&kw=%E5%AE%89%E7%90%AA%E6%8B%89%E7%9A%84%E7%81%B0%E7%87%BC&pi=0 | publisher = 誠品網路書店 | location = 台灣 | date = 2017年3月31日}}</ref>，1999年法蘭克．麥考特出版了《安琪拉的灰燼》續集《{{tsl|en|'Tis}}》描述他與妻子及家人生活，書中結尾作者將母親安琪拉．麥考特的骨灰安置於愛爾蘭，2005年出版他在紐約高中與大學的教學經驗傳記{{tsl|en|Teacher Man|夢想教室}}，在北美創下千萬本銷售量。

== 故事情節 ==
作者法蘭克．麥考特出生於1930年8月19日的紐約布魯克林貧民區，他是家中長子，父母都是從愛爾蘭移民到美國並因為懷了法蘭克．麥考特而[[奉子成婚|奉子成婚]]，母親安琪拉來自愛爾蘭[[利默里克|利默里克]]，喜愛音樂、唱歌與跳舞。法蘭克．麥考特幼年因父母貧困不堪、無法在美國生活舉家搬遷回家鄉愛爾蘭，他們的住處鼠患成災，下雨天可能造成化糞池滿溢，酒鬼父親喝光家裏所有錢財，連生存都有困難。在母親安琪拉．麥考特的教養與對於宗教信仰的支持下，法蘭克．麥考特知道不要怨天尤人，以堅強的意志與生命力撐過悲慘貧困的年少時光，成年後返回美國在紐約高中與大學執教，並陸續接回家人到美國團聚生活。
