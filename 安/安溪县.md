'''安溪县'''（{{lang-nan|An-khoe-koān}}），古稱'''小溪'''、'''清溪'''，位于[[福建省|福建省]][[东南|东南]][[沿海|沿海]]，[[厦门市|厦]]、[[漳州市|漳]]、[[泉州市|泉]][[閩南金三角|閩南金三角]]西北部，隶属[[泉州市|泉州市]]。县域范围东经117°36′－118°17′，北纬24°50′－25°26′，东接[[南安市|南安市]]，西连[[华安县|华安县]]，南毗[[同安区|同安区]]，北邻[[永春县|永春县]]，西南与[[长泰县|长泰县]]接壤，西北与[[漳平市|漳平市]]交界。全县总面积3057.28平方千米，辖24个乡镇460个村居，人口108万<ref>{{Cite web|title=安溪概述-安溪县人民政府|url=http://www.fjax.gov.cn/zjax/axgl/201707/t20170717_789807.htm|work=www.fjax.gov.cn|accessdate=2019-01-30}}</ref>(2017年)，有[[汉族|汉族]]、[[畲族|畲族]]等多个民族，通行[[普通話|普通話]]與[[閩南語|閩南語]]，以[[茶|茶]]業聞名全[[中國|中國]]，號稱[[中國茶都|中國茶都]]。

安溪县是"国家级园林县城"，素有"龙凤名区"之美誉，全县拥有各级文物保护单位88处，其中以[[安溪清水岩|清水岩]]、[[安溪文庙|文庙]]、[[安溪城隍|城隍庙]]最为著名。

== 歷史 ==
* [[唐懿宗|唐懿宗]][[咸通|咸通]]五年（西元864年），置[[南安縣|南安縣]][[小溪場|小溪場]]。
* [[南唐|南唐]][[保大_(李璟)|保大]]十三年（[[後周|後周]][[显德|显德]]二年，西元955年），小溪场监[[詹敦仁|詹敦仁]]向[[清源军|清源军]]节度使[[留从效|留从效]]申请建县获准，以境内溪水清澈之意，命名为“清溪县”，属[[泉州|泉州]]，詹敦仁为首任县令。
* [[北宋|北宋]]，[[宋徽宗|宋徽宗]][[宣和|宣和]]三年（西元1121年），以「[[睦州|睦州]][[清溪縣_(永貞)|清溪縣]]」（今[[浙江省|浙江省]][[淳安縣|淳安縣]]一帶）[[方腊起義|方腊盜起]]，[[官家|官家]][[禁忌|諱之]]，故改稱「安溪縣」，历代沿袭。

== 概況 ==
安溪縣，可粗略分為內安溪和外安溪，内安溪是安溪县中[[海拔|海拔]]较高的[[乡镇|乡镇]]的民间统称。内安溪一般包括[[长坑乡|长坑乡]]、[[剑斗镇|剑斗镇]]、[[蓝田乡|蓝田乡]]、[[祥华乡|祥华乡]]和[[感德镇|感德镇]]等乡镇，海拔较高，地势较陡，交通相对落后，经济水平比较低。

外安溪是安溪县中海拔较低的[[乡镇|乡镇]]的民间统称，[[凤城镇_(安溪县)|安溪县城]]所在地也属外山。外安溪一般包括[[凤城镇_(安溪县)|凤城镇]]、[[湖头镇_(安溪县)|湖头镇]]、[[官桥镇_(安溪县)|官桥镇]]、[[城厢镇_(安溪县)|城厢镇]]、[[龙门镇_(安溪县)|龙门镇]]、[[参内乡|参内乡]]等乡镇。外安溪地势较平坦，交通便利，经济发展水平比较高，百姓生活水平相对较高。

安溪縣號稱中國茶都，以產茶聞名，其中，[[鐵觀音|鐵觀音]]是最具盛名的一種。「鐵觀音」是烏龍茶的一種，別稱「紅心觀音茶」。[[清世宗|清世宗]][[雍正|雍正]]年間，「鐵觀音」品種在安溪西坪堯陽發現，安溪茶人隨即推廣，成為中國名茶之一。「鐵觀音」天性嬌弱，産量不多，有“好喝不好栽”的說法。

同時，安溪縣也是風靡閩、[[台灣|台]]、[[南洋|南洋]]的[[清水祖師|清水祖師]]信仰發源地，以清水祖師為主要奉祀對象的廟宇，在[[台灣|台灣]]有一百五十座以上。另外，安溪縣的其他鄉土神信仰：[[顯應祖師|顯應祖師]]、[[法主真君|法主真君]]、[[董公真人|董公真人]]、[[雙忠|雙忠]]神等[[安溪諸聖|安溪諸聖]]，亦隨著移民播遷，流傳至世界各地。

安溪是著名的僑鄉，許多安溪先民遠渡重洋到[[台灣|台灣]]，甚至[[南洋|南洋]]拓展新天地。[[清朝|清朝]]時，因安溪本地的[[大學士|大學士]][[李光地|李光地]]家族反對台灣的[[明鄭王朝|明鄭王朝]]政權之故，安溪移民是泉州諸縣邑之中，最晚來台的。在台灣，有兩百餘萬人口[[祖籍|祖籍]]安溪，約佔今日[[台灣人口|台灣人口]]的十分之一。[[安溪|安溪]]移民定居之地區，以[[大臺北|大臺北]]地區為多，[[日治台灣|日治台灣]][[昭和|昭和]]元年（1926年）的調查，當時[[台北市|台北市]]約有53.3%的人祖籍安溪縣，[[新店|新店]]、[[坪林|坪林]]、[[深坑|深坑]]、[[景美|景美]]、[[木柵|木柵]]、[[松山|松山]]、[[七堵|七堵]]、[[瑞芳|瑞芳]]、[[平溪|平溪]]、[[樹林|樹林]]、[[鶯歌|鶯歌]]、[[三峽|三峽]]等地區的居民，有四成到九成祖籍安溪縣。<ref>[http://ir.lib.ntnu.edu.tw/retrieve/47584/metadata_02_06_s_05_0008.pdf 劉豐極.艋舺清水巖祖師廟研究 ] {{webarchive|url=https://web.archive.org/web/20141213033726/http://ir.lib.ntnu.edu.tw/retrieve/47584/metadata_02_06_s_05_0008.pdf |date=2014-12-13 }}</ref>

而台灣，也有許多名人祖籍安溪縣，如大企業家：[[台塑|台塑]]集團創辦者[[王永慶|王永慶]]、[[王永在|王永在]]兄弟，[[新光集團|新光集團]]創辦者[[吳火獅|吳火獅]]，[[大同集團|大同集團]]創辦者[[林尚志|林尚志]]、[[林挺生|林挺生]]父子，[[台灣五大家族|台灣五大家族]]的[[基隆顏家|基隆顏家]]，[[瑞芳區|瑞芳]]礦業名人[[李建興|李建興]]、[[李建和|李建和]]兄弟。政治方面，則有：[[民主進步黨|民主進步黨]]創黨元老[[黃信介|黃信介]]，台灣[[總統府|總統府]]資政前[[台北市長|台北市長]][[高玉樹|高玉樹]]，曾任[[台北市|台北市]]代理市長的[[周百鍊|周百鍊]]等。在[[印尼|印尼]]、[[馬來西亞|馬來西亞]]、[[新加坡|新加坡]]，各約有20萬人口祖籍安溪。在[[東南亞|東南亞]]，也有許多名人祖籍安溪縣，如大企業家：[[馬來西亞|馬來西亞]][[雲頂集團|雲頂集團]]的創辦人[[林梧桐|林梧桐]]、安裕钢铁创办人[[林生枝|林生枝]]等。政治方面，則有：[[马来西亚|马来西亚]][[敦_(頭銜)|敦]][[林敬益|林敬益]]医生，祖籍安溪县[[官桥镇|官桥镇]]赤岭村；馬來西亞前[[民政党|民政党]]党主席，曾任多届内阁部长，[[新加坡|新加坡]]國會議員[[王世豐|王世豐]]等。

== 文化 ==
風靡[[臺灣|臺灣]]、[[南洋|南洋]]的[[清水祖師|清水祖師]]信仰，即發源於此。其餘[[安溪諸聖圖|安溪鄉土神靈]]，亦頗受重視。

安溪县的[[乌龙茶|乌龙茶]]制作技艺（[[铁观音|铁观音]]制作技艺）被列入[[国家级非物质文化遗产名录|国家级非物质文化遗产名录]]，湖头米粉亦有名。

== 旅游景点 ==
* “蓬莱仙境”清水岩安溪[[安溪清水岩|清水岩]]位于福建省安溪县[[蓬莱镇_(安溪县)|蓬莱镇]]的蓬莱山，始建于北宋，至今已有950多年，内奉中国百仙之一[[清水祖師|清水祖师]]，乃清水祖师文化之发源地。是[[国家4A级旅游景区|国家AAAA级]]风景名胜旅游区，又是享誉海内外的朝圣旅游胜地。清水岩是全国惟一主殿建设呈“帝”字形结构的岩寺，景区内楼阁殿宇巍峨宏伟，水色山光秀丽神奇，文物古迹星罗棋布。现存文物古迹中，宋代29处，元代1处，明代8处，清代5处，近代拓展23处，加上天造地设的自然景观，构成了令人神往的“蓬莱仙境”。
* 革命烈士纪念碑革命烈士纪念碑　位于[[凤城镇_(安溪县)|凤城镇]]北面凤冠山。建于1958年5月。石构，平面方形，由碑座、碑身、碑顶等组成。碑座束腰，碑顶作四垂檐式。通高7米，四周环绕平座栏杆。坐北向南。碑面阴楷镌刻“革命烈士纪念碑”7个大字，左右分别镌刻朱德、邓子恢的题词：“革命烈士永垂不朽！”、“英名不朽浩气长存！”背文记述：中共安溪中心县委书记李实、中国工农红军闽南游击队第二支队政委陈凤伍、二支队支队长李世全、官彭区[[中华苏维埃共和国|苏维埃政府]]主席傅有智等12位革命烈士的英雄事迹。1985年10月21日，安溪县人民政府公布为第一批县级重点文物保护单位。
* [[安溪文庙|安溪文庙]]始建于公元1001年，现存建筑为清代。在县城东南隅。省重点文物保护单位。北背踞凤山，前滨龙津，笔峰拱峙。有“名冠八闽，秀甲江南”这美誉。安溪文庙荟萃古代[[木雕|木雕]]、石雕之精华，其中莲花斗拱构成的八卦形[[藻井|藻井]]悬空倒挂，堪称一绝，八根石雕龙柱，价值连城。宽阔的庭院，铺有3000块石块，72根石柱，意喻孔圣3000个弟子，72个门生贤人，大殿上方有康熙皇帝御笔《万世师表》，屋顶的如意斗拱计108目，喻意108文曲星。[http://www.baike.com/wiki/安溪文庙]
* 安溪凤山风景旅游区     安溪凤山[[森林公园|森林公园]]位于安溪城关，以现有凤山公园为中心，向周边辐射，经营总面积180公顷（2700亩）。站在山顶可以看到整个安溪县城，该公园以自然、人文景观为根本，强化茶文化、宗教文化主题，集茶文化旅游、寻根旅游、朝圣旅游、健身旅游、观光旅游、生态旅游6大功能为一体，公园规划由“一园两区”构成，即茶叶大观园、宗教朝圣区、健身游览区，共有景点71处，其中现有45处，规划建设26处。
* [[安溪城隍|安溪城隍庙]]，始建于后周显德三年（956年），即置县次年，位于县治东，即今小东街。为全省较早建的城隍庙之一，素有“[[八闽|八闽]]第一，五邑无双”美誉，是安溪重要的道教景点和旅游目的地。      庙宇历尽沧桑，几经迁建：明[[洪武帝|洪武]]元年（1368年）重建；明[[景泰_(年号)|景泰]]二年（1451年）邑令谷廷怡、[[吴英|吴英]]修，典史蔡珍建后堂三间；明[[嘉靖帝|嘉靖]]十八年（1539年）邑令殷骛、邑尉邹奇重建；明嘉靖四十四年（1565年）邑令蔡常毓重建；清[[康熙|康熙]]十二年（1673年）邑令谢宸荃重修前后殿宇；清康熙四十年（1701年）邑令戎式弘重修，邑令曹续修；康熙五十四（1715年）邑令曾之传新建后堂住持丈室；乾隆十年（1745年）邑令何隆遇铺拜亭前石坪；[[乾隆|乾隆]]二十年（1755年）邑令庄成重修；1941年，庙中神像被国民党兵团所毁；1953年，迁在东岳寺檀樾祠奉祀。1990年新加坡侨亲陈美英女士捐巨资重建，定东岳寺东侧为新庙址。三进殿宇完工之后，又有新加坡韭菜芭城隍庙及杨桃园城隍庙联谊会捐资续建四、五两进殿宇。继而再有诸多善信捐建其他设施。新庙重建工程于1992年完工。城隍新庙与东岳寺并肩联臂五进殿堂，顺山势递升。主体建筑为重檐歇山式，穿斗式，面宽、进深各五间，全庙红墙碧瓦雕龙刻凤，巧夺天工，更胜旧观。      安溪城隍庙主神“敇封钦加普护清溪显佑伯主”，自[[宋朝|宋朝]]以来因有功于朝廷，皇帝钦赐封号、黄龙袍、玉印，明洪武元年再封“监察司威灵公”。座前差役八爷、九爷（俗称大爷伯、二爷伯或七爷、八爷）。安溪城隍自清初在[[臺灣|台湾]]传衍至今分炉已达三百多座，含嘉義縣鹿草鄉中寮城隍廟及鹿草安溪二隍殿等等, 目前海内外分炉有近千处。农历五月廿八日为城隍爷诞辰日，每年清明节前后举行春巡迎傩盛会，海内外善信近万人参加，盛况空前。
