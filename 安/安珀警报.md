'''安珀警报'''（{{lang-en|'''AMBER Alert'''}}）主要用于[[美國|美國]]和[[加拿大|加拿大]]，是當確認發生[[兒童|兒童]][[綁架|綁架]]案件時，透過各種[[大眾媒體|媒體]]向[[社會|社會]]大眾傳播的一種警戒告知。「{{lang|en|AMBER}}」是「{{lang|en|America's Missing: Broadcasting Emergency Response}}」（直譯為：美國失蹤人口：廣播緊急回應）的[[縮寫|縮寫]]，也是以一名於1996年在美國[[德克萨斯州|德州]][[阿灵顿_(得克萨斯州)|阿靈頓]]（Arlington）被綁架並[[謀殺|殺害]]的九歲女童[[安珀·海格曼|安珀·海格曼]]（Amber Hagerman）命名。在[[喬治亞州|喬治亞州]]則稱為「'''Levi's Call'''」<ref>{{cite web
 |title       = Levi's Call: Georgia's Amber Alert
 |accessdate  = 2006-09-04
 |url         = http://www.state.ga.us/gbi/levi.html
 |publisher   = Georgia Bureau of Investigation
 |deadurl     = yes
 |archiveurl  = https://web.archive.org/web/20060925193436/http://www.state.ga.us/gbi/levi.html
 |archivedate = 2006-09-25
}}</ref>在某些[[美國州份|州]]，這樣的計劃以在當地失蹤的兒童命名。安珀警戒使用美國[[緊急警報系統|緊急警報系統]]（EAS）透過[[電台|商業廣播電台]]、[[衛星電台|衛星電台]]、[[電視台|電視台]]，以及[[有線電視|有線電視]]向全國發佈，並同時會利用[[電子郵件|電子郵件]]、[[電子交通狀況號誌|電子交通狀況號誌]]以及[[無線通訊|無線]]裝置的[[簡訊|簡訊]]發佈。安珀警戒的發佈由負責調查该起綁架案的警察機構決定。對大眾發佈的安珀警戒內容通常包含被綁架者的描述、綁架嫌犯的描述，以及对綁匪[[車輛|車輛]]的描述及[[車牌號碼|車牌號碼]]。
== 歷史 ==
1996年1月13日，九歲的[[安珀·海格曼|安珀·海格曼]]和她的哥哥瑞奇（Ricky）在位於[[美國|美國]][[德克萨斯州|德州]][[阿灵顿_(得克萨斯州)|阿靈頓]]（Arlington）附近的祖母家騎著腳踏車，一位鄰居目睹了綁架過程，隨即報警，安珀的哥哥告訴他媽媽及祖父母這件事，當安珀的爸爸理察·海格曼一聽到女兒被綁架，隨即聯絡了Marc Klaas（其女兒Polly在1993年被綁架且殺害）。

理察·海格曼和安珀的媽媽堂娜‧懷生通知了媒體及[[FBI|FBI]]，接著和他們的鄰居一同尋找安珀的下落。四天後，一位正在遛狗的男子在溝渠裡發現了安珀的屍體，而兇手卻一直都沒有找到。

安珀的家人隨後成立人民對抗性犯罪者組織（People Against Sex Offenders，简称P.A.S.O.），希望可以藉此逼迫[[德克萨斯州|德州]]議會通過更多[[保護兒童|保護兒童]]的法律。上帝之家國際教會（God's Place International Church）也提供了一個給組織辦公的地方，並且持續的尋找那名兇手，當地媒體幾乎每天都在報導這件事。一位當地議員{{tsl|en|Martin Frost|馬丁‧佛斯特}}在Marc Klaas的幫助下起草了安珀·海格曼兒童保護法（Amber Hagerman Child Protection Act）。

1996年，當時的美國總統[[比尔·克林顿|柯林頓]]，簽署此法通過。

1996年7月，布魯斯‧賽伯特和理察·海格曼在[[阿灵顿_(得克萨斯州)|阿靈頓]]參加了一場[[記者會|記者會]]，雖然理察·海格曼早就準備好演講稿，但當天主辦單位卻要求讓布魯斯‧賽伯特發言。在布魯斯‧賽伯特20分鐘的發言中，他提到了當地警察有多努力的去尋找失蹤的小女孩，以及媒體是如何在這件事上幫了大忙。不久之後，一位從{{tsl|en|KRLD|KRLD}}電台來的記者向[[達拉斯|達拉斯]]的警察局長提出了賽伯特的發言內容，這導致了[[安珀警報|安珀警報]]系統的正式產生。

之後兩年內，警報系統主要在[[電台|電台]]內靠人民傳播。1998年，'''兒童警報基金會'''（{{tsl|en|Child Alert Foundation|Child Alert Foundation}}）建立了第一個全自動化的警報通知系統（Alert Notification System （ANS）），當有兒童失蹤或被綁架時，周圍所有的社區皆可收到通知。這警報發送至電台、電視、執法單位、報紙還有當地有參與警報系統的組織，警報將會透過所有的[[傳真機|傳真機]]、電子郵件及手機傳播，所有的民眾可以在第一時間收到相關的訊息。

在兒童警報基金會將安珀警報自動化之後，國際失蹤及被剝削兒童保護中心（National Center for Missing and Exploited Children （NCMEC））擴大了安珀警報的通知範圍。

2002年，安珀警報已經廣泛的在[[緊急廣播|緊急廣播]]及[[電視|電視]]和[[互聯網|網路]]運用。

《安珀的故事》是一部根據此事改編而成的電影。

== 啟用標準 ==
為了避免發出錯誤警報而讓這個系統被當作「[[狼來了|狼來了]]」而忽略，發佈警戒的標準相當嚴格。

每一個州或郡都有自己的安珀警戒發佈標準。而[[美國司法部|美國司法部]]發行了以下的「指引」，大部分的州都表示將「嚴格遵守」<ref>{{cite web
| title=發佈安珀警戒標準指引（Guidance on Criteria for Issuing AMBER Alerts）（PDF）
| date=2004年4月
| accessdate=2006-09-05
| url=http://www.amberalert.gov/docs/AMBERCriteria.pdf
| publisher=美國司法部
| deadurl=yes
| archiveurl=https://web.archive.org/web/20041015003050/http://amberalert.gov/docs/AMBERCriteria.pdf
| archivedate=2004-10-15
}}</ref>：

# 司法機構必須確認綁架實際發生
# 兒童必須有受到重傷或死亡的危險
# 必須要有被綁架兒童、綁架嫌犯、或綁架嫌犯的車輛詳細描述資料來發佈警報
# 被綁架者必須是17歲以下（包含17歲）的兒童

指引中建議將安珀警戒的即時資料通報[[聯邦調查局|聯邦調查局]]（FBI）的{{tsl|en|National Crime Information Center|國家犯罪資料中心}}（National Crime Information Center）。關於兒童綁架情況的文字描述必須輸入通報中，並將案件標示為兒童綁架（child abduction）。

在加拿大，[[加拿大皇家騎警|皇家騎警]]的要求和上述類似，只是將通報對象聯邦調查局改為皇家騎警。<ref name="Our Missing Children Canada">{{cite web

== 錯誤警報 ==
關心失蹤兒童的人士擔心警方並未嚴格遵守美國司法部的規範來發佈安珀警戒，造成大量的錯誤警報，而使社會大眾對安珀警戒系統失去信心。

美國 {{tsl|en|E. W. Scripps Company|Scripps}} 公司觀察了在2004年期間於美國發佈的233件安珀警戒，發現大部分的警戒並未達到司法部的發佈標準。其中有50%（117件警戒）被「國家失蹤及被剝削兒童中心」（National Center for Missing & Exploited Children）歸類為「家庭綁架」（family abductions），通常是父母爭奪監護權所致。其中有48起警戒的對象是完全未被綁架的兒童，這些兒童僅是逃家、失蹤，或牽扯到家庭誤會（例如有兩起案件僅是兒童在祖父母家中），也有因惡作劇造成的錯誤警報。其他23件警報是警察不知道據傳遭綁架的兒童姓名，通常是因為報案的目擊者誤會所致。

在2004年發佈的233件警戒中，僅有70件（佔30%）是兒童被陌生人帶走，並且和合法監護人以外的成人違法行動。<ref>{{cite web

=== 電子郵件的惡作劇 ===
有大量的[[惡作劇|惡作劇]]電子郵件偽裝成安珀警戒的通報郵件。其中一個特別的例子是'''潘妮·布朗'''（'''Penny Brown'''），關於潘妮被綁架的惡作劇警戒郵件在網路上流傳許久，使得安珀警戒網站必須貼出澄清，告知大眾從來沒有發佈這樣的警戒，也沒有叫做這個名字的小孩被綁架。<ref>{{cite web

==臉書的安柏計畫==
[[Facebook|Facebook]]（也称“臉書”）是一个相當普及的社群網站，因此已有多個國家的警政機構與臉書合作，只要有兒童遭到綁架，或是涉及刑案無法尋獲的失蹤兒童，臉書便會發布警報，若臉書使用者距離失蹤兒最後一次現身的位置160公里以內，就會在動態時報內收到臉書傳送失蹤兒的照片、各項特徵和失蹤地點等資訊的連結，以讓使用者協助搜尋。從2015年起，美國、加拿大、墨西哥、南韓、中華民國（台灣）和馬來西亞等國均已加入此計劃<ref>[http://udn.com/news/story/2/2114809 創舉！兒少遭綁架 臉書發警報到動態時報]</ref>。
