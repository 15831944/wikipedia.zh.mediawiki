'''安彦良和'''（{{bd|1947年|12月9日|||Yasuhiko, Yoshikazu}}）[[日本|日本]][[漫画家|漫画家]]、[[动画|动画]][[原畫師|原畫師]]及導演、[[插画家|插画家]]、[[小说|小说]][[作家|作家]]，出生自[[北海道|北海道]][[纹别郡|纹别郡]][[遠輕町|遠輕町]]。

==生平==
1966年前，安彦良和於[[大学|大学]][[人文学科|人文学科]]专攻西洋史，1970年因卷入[[大学|大学]]的[[学潮|学潮]]而于毕业前夕退学。1971年加入[[手塚治虫|手塚治虫]]的[[虫制片公司|虫制片公司]]。1973年加入创映社（现在的[[SUNRISE|SUNRISE]]）。


== 作品 ==
1971年，安彦良和加入[[虫制片公司|虫制片公司]]，从这里走上制作[[动画|动画]]的道路；1979年开始连载第一部长篇[[漫画|漫画]][[作品|作品]]《[[亞利安_(安彥良和作品)|亞利安]]》。

安彦良和代表作為《[[機動戰士GUNDAM|機動戰士GUNDAM]]》的人物设定和作画[[监督|监督]]，擔任高千穗遙科幻小說插畫的《[[宇宙先鋒|宇宙先鋒]]》，則成為他的動畫導演處女作。[[漫画|漫画]]代表作《[[亞利安_(安彥良和作品)|亞利安]]》和《[[金星战记|金星战记]]》都由他自己亲自执导改编成[[动画|动画]]電影。他不但在动漫畫製作方面成就卓越，还从事[[插画|插画]]工作，並且也有小說創作，作品包括《[[钢马章传|钢马章传]]》《[[圣王子库库路坎|圣王子库库路坎]]》《[[坦格里大战|坦格里大战]]》等。

== 获奖經歷 ==
*1979年～1982年蝉联动画最高赏[[动画|动画]][[制作人|制作人]]首位。
*1983年获[[日本|日本]][[动画|动画]]大赏作画赏。
*1985年获[[日本|日本]][[动画|动画]]大赏FAN大赏。
*1990年以《[[大國主神|大國主]]》获第19届[[日本|日本]]漫画家协会赏优秀赏。
