'''安德烈·克考夫'''（Андрій Юрійович Курков）是一位[[烏克蘭|烏克蘭]][[小說家|小說家]]和[[思想家|思想家]]，他用[[俄語|俄語]]寫作，作品暢銷書《[[企鵝的憂鬱|企鵝的憂鬱]]》，9本兒童書籍以及約20部[[紀錄片|紀錄片]]、[[小說|小說]]與[[電視劇|電視劇]][[劇本|劇本]]。安德烈·克考夫的作品已經翻譯成[[英語|英語]]，[[日語|日語]]，[[法語|法語]]，[[德語|德語]]，[[義大利語|義大利語]]，[[中文|中文]]，[[瑞典語|瑞典語]]和[[希伯來語|希伯來語]]在內的37種[[語言|語言]]，並在65個[[國家|國家]]出版<ref>{{Cite web|url = http://www.radiosvoboda.org/content/article/24949786.html|title = Radio Liberty interview with Andrey Kurkov|date = |accessdate = 2015-10-01|website = |publisher = |last = |first = }}</ref>。安德烈·克考夫的著作充滿[[黑色幽默|黑色幽默]]，後[[蘇聯|蘇聯]]現實和[[超現實主義|超現實主義]]元素。

==生平==
安德烈·克考夫的父親是一名試飛員，母親則是[[醫生|醫生]]<ref name = wroe> {{cite web | author = Nicholas Wroe | url = https：//www.theguardian.com/culture/2011/jul/29/andrey -kurkov-life-books-profile | title =書中的生活：Andrey Kurkov |出版=衛報|日期= 2011年7月30日| accessdate = 2015年10月21日}} </ref>。

安德烈·克考夫從基輔外國語學院畢業後，成為訓練有素[[日語|日語]][[翻譯|翻譯]]人員，被指派協助[[KGB|KGB]]。然而，他設法改為替警察服務，因此在服役期間和之後獲得更大程度的自由。之後他被分配到[[敖德薩|敖德薩]]的監獄。在這期間，安德烈·克考夫寫下一些兒童文學。

安德烈·克考夫的第一部小說是在[[蘇聯解體|蘇聯解體]]兩週前出版的，在隨後的社會和政治動盪中，他自費出版著作。<ref name = wroe />

==家族==
安德烈·克考夫和[[英國|英國]]妻子伊麗莎白住在基輔，育有三個孩子。
