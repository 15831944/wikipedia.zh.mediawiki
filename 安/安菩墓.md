'''安菩墓'''是一座位於[[河南省|河南省]][[洛陽市|洛陽市]][[龍門山_(洛陽)|龍門山]]的[[唐朝|唐朝]][[粟特人|粟特人]]古墓葬，建於唐[[景龍|景龍]]三年（公元709年），為安菩及其妻何氏的合葬墓<ref name=安菩墓>{{cite web |url=http://www.chnmus.net/dcjp/node_12360.htm |title=安菩墓出土三彩馬 |author=王軍花 |date= |website=www.chnmus.net |access-date=2018年8月11日}}</ref>。發掘於1981年4月<ref>{{cite web |url=http://www.dpm.org.cn/Uploads/File/pdf/61/5c/0f/615c0ff985387ea79b3a8e103abf3918.pdf |title=重讀安菩墓 |author=沈睿文 |date=2009 |website=www.dpm.org.cn |access-date=2018年8月11日}}</ref>，出土文物129件。

== 墓主簡介 ==
根據墓誌記載，安菩，字薩，曾祖父叫鉢達干，祖父叫係利，其父名諱失載。他原為粟特[[昭武九姓|昭武九姓]]之一的[[安國_(昭武九姓)|安國]]首領，後因不滿[[突厥人|突厥人]]的統治而歸依唐帝國，獲封大唐定遠將軍。據說安菩驍勇善戰，率領軍隊縱橫大漠，保衞着唐帝國的邊境。史書描述他為「以一當千，獨掃蜂飛之衆」<ref>{{cite web |url=http://www.siluyou.org/index/citywwinfo/id/194.html |title=三彩釉陶束髮胡人俑 |author1=王軍花 |author2=顧凡穎 |website=www.siluyou.org |access-date=2018年8月11日}}</ref>。公元664年，安菩於西安去世。40年後，其妻何氏在東都洛陽病逝。他們的獨子[[安金藏|安金藏]]將安菩的遺骨遷至洛陽與何氏合葬，因而建造了這座安菩墓。

經學者考證，認為安金藏在為父母擧行合葬儀式時使用了[[火祆教|火祆教]]葬儀「{{Link-en|犬視|Sagdid}}」，並對祆教戰神牡鹿和豐饒女神{{Link-en|阿娜希塔|Anahita}}致祭奉祀<ref>{{cite web |url=http://www.xuehuile.com/thesis/a41e2f13247b491e8711e957782f7400.html |title=重讀安菩墓 |author=沈睿文 |date=2014年2月1日 |website=www.xuehuile.com |access-date=2018年8月11日}}</ref>。此外，安菩墓中發現的一件唐三彩雙峰駝，其駝鞍中間有一神像，經[[中山大學|中山大學]]歷史系教授[[姜伯勤|姜伯勤]]考證，認為是當時流行的一種「盛於皮袋」的祆神<ref>{{cite web |url=https://www.xueshu.com/nfww/201401/3494979.html |title=從唐章懷太子墓壁畫「狩獵出行圖 (四)」駱駝圖像看所謂「唐安菩墓三彩駱駝所見『盛於皮袋』的祆神」 |author=沈騫 |date=2014 |website=www.xueshu.com |access-date=2018年8月12日}}</ref>。由此可知安菩一家均是祆教信徒。

== 墓葬簡介 ==
安菩墓座落在洛陽龍門山東山北麓，方向坐南朝北，西距[[伊水|伊水]]約1公里，北距[[隋唐洛陽城|隋唐洛陽城]]的南城牆約8公里。由於歷時久遠，墓葬地面已無任何遺存。但隨葬品豐富，合計達129件。該墓由墓道、墓門、甬道和墓室組成，墓門以靑石構築，在門楣、門框、門板、門檻上刻有紋飾，墓內東西兩邊並置雙棺牀。在出土的129件文物中，[[唐三彩|唐三彩]]佔50件，單釉器61件，瓷器11件，陶器2件，銅器3件，金器1件，瑪瑙1件。唐三彩中有高達1.12米的文官俑和武士俑，有面目猙獰的鎭墓獸，還有各種鷄、鴨、馬匹、駱駝、牽馬俑、牽駝俑、男女侍俑等。其中出土的一枚[[東羅馬|東羅馬]][[福卡斯|福卡斯]]皇帝時期所鑄造的金幣，再一次見證唐朝時東西方貿易的繁盛<ref name=安菩墓 />。
