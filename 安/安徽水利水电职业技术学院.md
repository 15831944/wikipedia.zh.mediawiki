'''安徽水利水电职业技术学院'''是位于[[中华人民共和国|中华人民共和国]][[安徽省|安徽省]][[合肥市|合肥市]]的一所高等职业院校，学校前身是1952年建立的淮河水利专科学校，2000年经安徽省政府批准成立安徽水利水电职业技术学院，是全国100所国家示范性高职院校之一。

== 沿革 ==
* 1952年，淮河水利专科学校成立，隶属于[[中华人民共和国水利部|国家水利部]]<ref name=jj>{{cite web|url=http://www.ahsdxy.ah.edu.cn/portal/singlecontent.jsp?id=302768&cid=10844|title=学院简介|publisher=安徽水利水电职业技术学院|accessdate=2018-04-17}}</ref>。
* 1984年，成立安徽水利职工大学<ref>{{cite news|url=http://ah.anhuinews.com/qmt/system/2012/10/19/005266356.shtml|title=安徽水利水电职业技术学院举办60周年校庆|date=2012-10-19|publisher=中安在线}}</ref>。
* 2000年6月18日，经安徽省政府批准，安徽水利水电职业技术学院正式成立<ref>{{cite news|url=http://www.ahsl.gov.cn/websites11/detail/533e3ac5e2ede8c01a6a20e9.html|title=安徽省水利水电职业技术学院成立|date=2000-10-25|publisher=安徽省水利厅}}</ref>。
* 2014年，学校成为安徽省首批“四年一贯制”本科教育改革试点院校<ref>{{cite news|url=http://ah.anhuinews.com/system/2014/06/20/006464223.shtml|title=安徽试点"四年一贯制"技术技能型本科人才培养模式|date=2014-06-20|publisher=中安在线}}</ref>，与[[合肥学院|合肥学院]]联合培养本科人才<ref>{{cite news|url=http://www.ahedu.gov.cn/28/view/256012.shtml|title=安徽水利水电职业技术学院今秋开始招收本科层次学生|date=2014-09-22|publisher=安徽省教育厅}}</ref>。

== 院系设置 ==
学院设有9个二级学院和本科教育部、基础教学部、马克思主义学院及继续教育学院，开设53个专业。<ref name=jj/>

== 学校排名 ==
根据中国科学评价研究中心、武汉大学中国教育质量评价中心和中国科教评价网发布的“2018中国专科院校排行榜”显示，该校位列专科院校'''第85位'''，安徽省'''第3位'''。<ref>{{cite news|url=http://www.nseac.com/eva/CUcompspeE.php|title=2018-2019年中国专科院校排名|publisher=中国科教评价网|accessdate=2018-04-17}}</ref>
