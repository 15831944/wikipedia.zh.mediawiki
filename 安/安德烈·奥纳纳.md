'''安德烈·奥纳纳'''（{{lang-fr|'''André Onana'''}}，{{bd|1993年|7月20日}}），是一名[[喀麥隆|喀麥隆]]職業足球員，現時效力[[荷蘭足球甲組聯賽|荷蘭足球甲組聯賽]]球會[[阿贾克斯足球俱乐部|阿贾克斯]]和[[喀麥隆國家足球隊|喀麥隆國家足球隊]]。司職[[門將|門將]]。

== 球會生涯 ==
奧拉拿於[[喀麥隆|喀麥隆]]Nkol Ngok出生<ref name = "SW">{{soccerway|andre-onana/372465|accessdate= 2015-02-09|language= en}}</ref>，曾為[[西班牙|西班牙]]球隊[[巴塞罗那足球俱乐部|巴塞罗那]]的青年軍上陣。於2015年1月，[[阿贾克斯足球俱乐部|阿贾克斯]]宣佈球員將於7月來投<ref name = "Ajax">{{cite web|url=http://www.bbc.co.uk/sport/0/football/30672547|title=Cameroon's Onana to join Ajax from Barcelona|date= 2015-01-04|accessdate=2016-12-27|publisher=BBC Sport|language= en}}</ref>。其後，阿積士改稱球員會提早在1月冬季[[轉會窗|轉會窗]]加盟<ref>{{cite web|url=http://www.vi.nl/nieuws/talentvolle-onana-lost-keepersprobleem-ajax-op.htm|title=Talentvolle Onana lost keepersprobleem Ajax op|date=2015-01-14|accessdate= 2015-01-18|language=nl|publisher=Voetbal International}}</ref><ref>{{cite web|url=http://uk.soccerway.com/news/2015/January/14/onana-makes-immediate-ajax-switch/n530463/|title=Onana makes immediate Ajax switch|date= 2015-01-14|accessdate= 2015-01-18|publisher=Soccerway|language= en}}</ref><ref>{{cite web|url=http://www.fourfourtwo.com/news/onana-makes-immediate-ajax-switch|title=Barca's Onana makes immediate Ajax switch|date= 2015-01-14|accessdate= 2015-02-29|publisher=FourFourTwo|language= en}}</ref>。2月，奧拉拿首次為[[阿積士青年隊|阿積士青年隊]]上陣<ref name = "SW"/>。

== 國家隊生涯 ==
奧拉拿曾為喀麥隆的青年國家隊上陣<ref name = "Ajax"/>。2016年5月，奧拉拿被列入對陣[[法國國家足球隊|法國國家足球隊]]的大軍名單，但沒有上陣<ref>{{cite web|url=http://www.bbc.co.uk/sport/football/36321424|title=Cameroon's Moukandjo to miss friendly against France|date= 2016-05-22|accessdate= 2016-07-11|publisher=BBC Sport|language= en}}</ref>。9月，他首次代表[[喀麥隆國家足球隊|喀麥隆國家足球隊]]上陣，對手為[[加蓬國家足球隊|加蓬國家足球隊]]，最後喀麥隆小勝2比1<ref name = "NFT">{{NFT player|pid=65205|accessdate= 2016-12-27|language= en}}</ref>。
