'''安宰亨'''（{{bd|1965年|1月8日||}}）是一名韩国男子乒乓球运动员。他在1988年汉城奥运会中，参加了男子乒乓球比赛，并获得男子乒乓球比赛双打铜牌。<ref>{{cite web |url=http://www.sports-reference.com/olympics/athletes/an/an-jae-hyeong-1.html |title=An Jae-Hyeong Biography and Olympic Results |publisher=Sports Reference |accessdate=27 December 2010}}</ref><ref>{{cite web |url=http://www.ittf.com/ittf_stats/All_events3.asp?ID=91 |title=ITTF Database |publisher=International Table Tennis Federation |accessdate=27 December 2010 |deadurl=yes |archiveurl=https://web.archive.org/web/20121004144349/http://www.ittf.com/ittf_stats/All_events3.asp?ID=91 |archivedate=2012年10月4日 }}</ref>

安宰亨的妻子[[焦志敏|焦志敏]]是原中国女子乒乓球运动员。其子[[安秉勋|安秉勋]]则是一位职业[[高尔夫球|高尔夫球]]运动员。
