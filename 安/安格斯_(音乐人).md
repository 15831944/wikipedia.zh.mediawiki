'''安格斯'''（{{lang|en|Angus Mak}}），本名'''麥偉珍'''{{r|皇妃}}，是香港著名男音乐人、编曲，曾经发掘过[[黎瑞恩|黎瑞恩]]、[[陈慧娴|陈慧娴]]等著名歌手，{{fact|1990年代曾经因为胞弟欠款而被[[高利贷|高利贷]]威胁编曲而被当时[[亚洲电视|亚洲电视]][[今日睇真D|今日睇真D]]节目组报导|time=2016-01-11}}。他曾與[[李振權|李振權]]、[[趙偉健|趙偉健]]和[[趙潤勤|趙潤勤]]三人合作開設法安利製作公司，制作包括《[[少女雜誌|少女雜誌]]》和《[[故事的感覺|故事的感覺]]》多張粵語專輯<ref>{{Cite web |url = http://comebacktolove.blogspot.com/2014/05/1984.html |title = 陳慧嫻 黎芷珊 陳樂敏 - 少女雜誌 (1984)}}</ref><ref>{{Cite web |url = http://comebacktolove.blogspot.com/2014/05/1984_7406.html |title = 陳慧嫻 - 故事的感覺 (1984)}}</ref>。陈慧娴的出道作《[[逝去的諾言|逝去的諾言]]》亦是由安格斯填词作曲編曲。而黎瑞恩的出道作《发誓》便是由安格斯填词作曲及監制。

1985年，安格斯曾與李振權、林志楊和 Melchior Sarreal 合組皇妃樂隊，並推出樂隊唯一大碟《[[Lady_Diana|Lady Diana]]》<ref name="皇妃">{{Cite web |url = http://comebacktolove.blogspot.com/2014/05/lady-diana-1985.html |title = 皇妃 - Lady Diana (1985)}}</ref>。該大碟以大膽露骨著稱，內容涉及色情和同性戀等被視為犯禁的題材，因此並無電台或電視台願意播放該大碟所收錄的歌曲<ref>{{cite news |language = zh-hk |url =  http://hk.apple.nextmedia.com/supplement/culture/art/20100920/14467264 |title = 代 Cult2：八十前 不禁政治禁鹹濕歌 |publisher = 蘋果日報 |date = 2010-09-20}}</ref>。

*安格斯在網上常被誤稱為陳慧嫻法安利製作時期的經理人，但實際上安格斯僅負責陳慧嫻的填词,作曲,編曲工作，而並非陳慧嫻的經理人，陳慧嫻在法安利製作公司時期的經理人實為[[趙潤勤|趙潤勤]]，1986年法安利製作公司因為試圖再次推動樂隊熱而失敗，從唱片公司改組成為經理人公司，安格斯選擇離開法安利。
