'''安東尼·拉休拉'''（{{lang-en|'''Anthony Laciura'''}}，{{bd|1951年|9月27日||}}）是一名美國歌劇[[男高音|男高音]]，他的才能以{{link-en|歌劇配角|Comprimario}}和演員而聞名。他出生在[[新奧爾良|新奧爾良]]，與Charles Paddock在那裡一起學習歌喉，Paddock也是{{link-en|Ticho Parly|Ticho Parly}}的老師。

== 生涯 ==
2010年到2013年，他被看中在[[HBO|HBO]]電視劇《[[酒私風雲|酒私風雲]]》中演出，由[[馬田·史高西斯|馬田·史高西斯]]監製。他飾演Eddie Kessler，是[[史提夫·布斯美|史提夫·布斯美]]飾演[[努基·湯普森|努基·湯普森]]的男管家。<ref>{{Cite web |url = http://www.hbo.com/boardwalk-empire/episodes/04/41-erlkonig/interview/anthony-laciura.html |title = Interview With Anthony Laciura |publisher = HBO |language = en}}</ref>
