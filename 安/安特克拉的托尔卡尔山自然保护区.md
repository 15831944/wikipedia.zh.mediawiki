'''安特克拉的托尔卡尔山自然保护区'''（{{lang-es|El Paraje Natural Torcal de Antequera}}）位于[[托尔卡尔山|托尔卡尔山]]（{{lang-es|Sierra del Torcal}}），简称'''托尔卡尔'''（{{lang-es|El Torcal}}），是[[西班牙|西班牙]][[安达卢西亚|安达卢西亚]][[马拉加省|马拉加省]][[安特克拉|安特克拉]]南部的一个[[自然保护区|自然保护区]]。{{lang|es|Torcal}}意思是有{{lang|es|torca}}的地方，<ref>{{Cite web |url = https://es.oxforddictionaries.com/definition/torcal |title = torcal |publisher = Oxford Dictionaries |language = es |accessdate = 2019-02-08 }}</ref>而{{lang|es|torca}}是指[[落水洞|落水洞]]，托尔卡尔即得名于当地的落水洞。<ref>{{Cite web |url = https://www.benahavislife.com/lugares-de-interes-benahavis-blog/2018/1/18/torcal-de-antequera-malaga |title = Torcal - Antequera, Malaga |date = 2018-01-18 |publisher = Benahavis Life |language = en |accessdate = 2019-02-08 }}</ref>

托尔卡尔于2016年被收入[[联合国教科文组织|联合国教科文组织]][[世界遗产|世界遗产]][[安特克拉石墓遗址|安特克拉石墓遗址]]。<ref>{{Cite web |url = http://whc.unesco.org/en/list/1501 |title = Antequera Dolmens Site |date = 2016-07-17 |publisher = UNESCO |language = en |accessdate = 2019-02-08 }}</ref>
