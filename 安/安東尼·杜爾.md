'''安東尼·杜爾'''（Anthony Doerr，{{bd|1973年|10月27日}}），美國小說家和短篇故事作家，2015年憑藉《[[我们看不到的所有光明|我们看不到的所有光明]]》獲得[[普利茲小說獎|普利茲小說獎]]。

== 早年生活與教育 ==
杜爾於1973年10月27日出生於[[俄亥俄州|俄亥俄州]][[克利夫蘭|克利夫蘭]]，並在{{link-en|拉塞爾鎮區|Russell Township, Geauga County, Ohio}}長大<ref>{{cite news |first=Karen R. |last=Long |title=Anthony Doerr Wins Lucrative Short-story Prize |url=http://www.cleveland.com/books/index.ssf/2011/04/anthony_doerr_wins_lucrative_s.html |newspaper=The Plain Dealer |date=2011-04-09 |accessdate=2011-04-27 |archive-url=https://web.archive.org/web/20110413204813/http://www.cleveland.com/books/index.ssf/2011/04/anthony_doerr_wins_lucrative_s.html |archive-date=2011-04-13 |dead-url=no }}</ref>。他之後到附近的{{link-en|大學學院|University School}}就讀，在1991年畢業。多爾隨後遠走到[[緬因州|緬因州]]{{link-en|布朗斯維克 (緬因州)|Brunswick, Maine|布朗斯維克}}的[[鲍登学院|鲍登学院]]主修歷史，讀至1995年後獲得[[文學士|文學士]]學位，他之後回到俄亥俄州，在[[鮑林格林|鮑林格林]]的[[鮑林格林州立大學|鮑林格林州立大學]]攻讀[[藝術創作碩士|藝術創作碩士]]<ref>{{cite web|url=http://www.bestyoungnovelists.com/Anthony-Doerr |title=Anthony Doerr |publisher= |deadurl=yes |archiveurl=https://web.archive.org/web/20100509162510/http://www.bestyoungnovelists.com/Anthony-Doerr |archivedate=2010-05-09 }}</ref>。

== 作家生涯 ==
杜爾第一本出版的書籍是2002年的短篇故事集《拾貝人》（''The Shell Collector''），其中的許多故事都取材於他曾工作或生活過的[[新西蘭|新西蘭]]和非洲<ref>{{cite web |url=https://www.nytimes.com/2002/03/03/books/rivers-run-through-it.html |title=Rivers Run Through It |publisher=[[The_New_York_Times|The New York Times]] |author=Nancy Willard |date=2002-03-03 |accessdate=2018-07-25 |archive-url=https://web.archive.org/web/20160527022833/http://www.nytimes.com/2002/03/03/books/rivers-run-through-it.html |archive-date=2016-05-27 |dead-url=no }}</ref>。他之後又在2010年寫了一本名叫《記憶牆》（''Memory Wall''）的短篇故事集<ref>{{cite web |url=https://www.nytimes.com/2010/08/01/books/review/Rafferty-t.html |title=Time and Inner Space |publisher=[[The_New_York_Times|The New York Times]] |author=Terrence Rafferty |date=2010-07-30 |accessdate=2018-07-25 |archive-url=https://web.archive.org/web/20180202224156/http://www.nytimes.com/2010/08/01/books/review/Rafferty-t.html |archive-date=2018-02-02 |dead-url=no }}</ref>。其第一本小說為2004年的《關於恩典》（''About Grace''）<ref>{{cite web |url=https://bookpage.com/reviews/3735-anthony-doerr-about-grace#.W1hRc9UzZhE |title=ABOUT GRACE |publisher=BookPage |author=Thane Tierney |accessdate=2018-07-25 |archive-url=https://web.archive.org/web/20151011234538/http://bookpage.com/reviews/3735-anthony-doerr-about-grace#.W1hRc9UzZhE |archive-date=2015-10-11 |dead-url=no }}</ref>，三年後杜爾則發行了名叫《羅馬四季》（''Four Seasons in Rome''）的[[回憶錄|回憶錄]]<ref>{{cite web |url=https://bookpage.com/reviews/5294-anthony-doerr-four-seasons-rome#.W1hSH9UzZhE |title=FOUR SEASONS IN ROME |publisher=BookPage |author=Kelly Koepke |accessdate=2018-07-25 |archive-url=https://web.archive.org/web/20150927205845/http://bookpage.com/reviews/5294-anthony-doerr-four-seasons-rome#.W1hSH9UzZhE |archive-date=2015-09-27 |dead-url=no }}</ref>。

杜爾的第二本小說《[[我们看不到的所有光明|我们看不到的所有光明]]》把故事舞台設在[[二戰|二戰]]時期的法國，講述法國女孩瑪麗-蘿兒（Marie-Laure）和德國男童韋納（Werner）故事。其於2014年發行，隨即獲得廣大的好評，並入圍[[美國國家圖書獎|美國國家圖書獎最佳小說類獎]]<ref>{{cite web |url=http://www.mhpbooks.com/national-book-awards-shortlists-announced/ |title=National Book Awards shortlists announced |work=Melville House Publishing |author=Alex Shephard |date=2014-10-15 |accessdate=2014-10-15 |archive-url=https://web.archive.org/web/20141020051257/http://www.mhpbooks.com/national-book-awards-shortlists-announced/ |archive-date=2014-10-20 |dead-url=no }}</ref>。作品為《[[紐約時報|紐約時報]]》的[[紐約時報暢銷書榜|暢銷書]]之一，亦被該報評為2014年值得留意的書籍<ref>{{cite web|url=https://www.nytimes.com/2014/12/14/books/review/the-10-best-books-of-2014.html|title=The 10 Best Books of 2014|work=nytimes.com|access-date=2015-09-26|archive-url=https://web.archive.org/web/20150923212235/http://www.nytimes.com/2014/12/14/books/review/the-10-best-books-of-2014.html|archive-date=2015-09-23|dead-url=no}}</ref>。2015年，《我们看不到的所有光明》為杜爾羸得當屆的[[普利茲小說獎|普利茲小說獎]]，同時也使其取得{{link-en|戴頓文學和平獎|Dayton Literary Peace Prize}}小說類獎的亞軍<ref>{{cite web|url=http://daytonliterarypeaceprize.org/#|title=Dayton Literary Peace Prize - An International Award|author=D. Verne Morland|publisher=|access-date=2015-12-02|archive-url=https://web.archive.org/web/20151210215712/http://daytonliterarypeaceprize.org/|archive-date=2015-12-10|dead-url=no}}</ref>和俄亥俄州圖書獎（Ohioana Book Award）<ref>{{cite web|url=http://ohioana.org/about/media/prjuly1415.pdf |title=Archived copy |accessdate=2015-12-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20150905065528/http://www.ohioana.org/about/media/prjuly1415.pdf |archivedate=2015-09-05 }}</ref>。

在2007年到2010年，杜爾都是愛達荷州的駐地作家<ref>{{cite web|url=http://www.arts.idaho.gov/writers/tonydoerr.aspx|title=ICA|publisher=|deadurl=yes|archiveurl=https://web.archive.org/web/20120512031242/http://www.arts.idaho.gov/writers/tonydoerr.aspx|archivedate=2012-05-12}}</ref><ref>{{cite web|url=http://www.boiseweekly.com/boise/anthony-doerr-is-a-recognized-and-slightly-wealthier-fellow/Content?oid=1566790|title=Anthony Doerr Is A Recognized (And Slightly Wealthier) Fellow|work=Boise Weekly|access-date=2010-04-29|archive-url=https://web.archive.org/web/20120310165604/http://www.boiseweekly.com/boise/anthony-doerr-is-a-recognized-and-slightly-wealthier-fellow/Content?oid=1566790|archive-date=2012-03-10|dead-url=no}}</ref>。

== 個人生活 ==
杜爾已經結婚，並育有一對雙胞胎兒子，一家四口現在居住於[[愛達荷州|愛達荷州]]的[[波夕|波夕]]<ref name="idahostatesmanboisesanthonydoerr">{{cite news|last1=Oland|first1=Dana|title=Boise’s Anthony Doerr wins the 2015 Pulitzer Prize for Fiction|url=http://www.idahostatesman.com/entertainment/arts-culture/article40854669.html|accessdate=2017-09-29|work=Idaho Statesman|date=2015-04-20}}</ref>。
