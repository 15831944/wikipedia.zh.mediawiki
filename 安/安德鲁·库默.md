'''安德鲁·库默'''（{{lang-en|'''Andrew Mark Cuomo'''}}，{{IPA-en|ˈkwoʊmoʊ|pron}}；{{bd|1957年|12月6日|catIdx=Cuomo}}），是现任[[美国|美国]][[纽约州|纽约州]][[纽约州州长|州长]]，于2011年1月1日就任。父亲[[馬力歐·库默|馬力歐·库默]]为第52任纽约州州长（1983年－1994年）。

== 早期生活 ==
生于[[纽约|纽约]][[皇后区|皇后区]]，他是家中的长子，父亲[[馬力歐·库默|馬力歐·库默]]为前任纽约州州长，母亲是Matilda Raffa，弟弟Christopher是[[美国广播公司|美国广播公司]]（ABC）的记者。

中学就读于Archbishop Molloy High School，1979年毕业于纽约[[福坦莫大学|福坦莫大学]]，1982年从纽约[[奥尔巴尼法学院|奥尔巴尼法学院]]毕业。

其父亲馬力歐·库默于1982年开始参选，成功當選並擔任纽约州州长長達12年，那一年他作为父亲的助手开始踏入政治之路。1984－1985年他从事[[律师|律师]]职业。

== 从政生涯 ==
[[File:Andrew_Cuomo.png|left]]
1993－2001年他曾在[[比爾·克林顿|比爾·克林顿]]政府内阁中任職，在1997－2001年担任过[[美国住房与城市发展部|美国住房与城市发展部]]部长。1999年1月19日，比爾·克林顿總統率領閣員到國會發表[[國情諮文|國情諮文]]期間，安德鲁·库默被指定為[[指定倖存者|指定倖存者]]（{{lang|en|Designated survivor}}）在一處秘密安全地點待命，以便在聚集在眾議院會議廳中的總統、副總統以及其他的官員面臨災難時，能按照[[美國總統繼任順序|美國總統繼任順序]]繼任總統職位。。


2002年曾參與紐約州州長提名，2006年當選紐約州檢察總長。

2010年州長選舉，他代表民主黨，以62%得票率當選[[纽约州州长|纽约州州长]]。當選州長後，他于2011年推動[[紐約州同性婚姻|紐約同性婚姻合法化]]，並對企業減免稅款。

2014年州長選舉，庫默競選連任。<ref>[http://www.ntdtv.com/xtr/gb/2014/09/11/a1137379.html 纽约州初选 库默胜出 艾维乐击败刘醇逸]</ref>

== 家庭生活 ==
