'''安平區'''是[[臺灣|臺灣]][[臺南市|臺南市]]三十七個行政區之一，為[[臺南市政府|臺南市政府永華市政中心]]所在地，與[[新營區|新營區]]共為[[臺南市|臺南市]]的[[行政中心|行政中心]]。地理位置北以[[鹽水溪|鹽水溪]]和[[安南區|安南區]]為界，東以[[台南運河|台南運河]]及民權路、[[中華西路_(台南市)|中華西路]]與[[中西區_(台灣)|中西區]]相鄰，南則隔[[台南運河|運河]]與[[南區_(臺南市)|南區]]毗鄰，西臨[[台灣海峽|台灣海峽]]，也因為安平區四周被台灣海峽(西)、鹽水溪(北)、台南運河(東、南)環繞，對外皆須以橋樑相連，固以市政府周遭為發展中心的新市區素有「安平島」之稱。區內北端為老安平聚落，為漢人最早開發地之一，在[[台灣荷蘭統治時期|荷蘭時期]]時這裡建有大員市鎮，是臺灣最古老的市鎮，而「臺灣」之名也起源於「大員」（[[荷蘭語|荷語]]：Tayouan），因此此區可說是[[台灣歷史|臺灣歷史]]的起源地。

== 命名由來 ==
安平在[[台灣荷蘭統治時期|荷蘭時期]]之前已被稱為「[[大員|大員]]」、[[臺灣|臺灣]]，例如《[[東番記|東番記]]》。[[明朝|明朝]][[永曆|永曆]]15年（西元1661年），[[鄭成功|鄭成功]]取代[[荷蘭人|荷蘭人]]管轄[[臺灣|臺灣]]，此批來臺[[漢人|漢人]]改名大員為安平，取自[[福建省|福建省]][[泉州府|泉州府]]連接[[晋江|晋江]]與[[南安|南安]]的[[安平橋|安平橋]]；時設有[[安平鎮_(明鄭)|安平鎮]]，同時為鄭氏[[安平城|王城]]所在。

== 歷史 ==
[[File:Zeelandia_model.JPG|right]]
[[File:前安平區公所.JPG|thumb]]）]]
在荷蘭時期之前，上鯤鯓等沙洲是位於下林仔（今臺南市南區夏林路一帶，荷蘭時期的克利(Celij)森林）的大員社(Tayouan)狩獵之地，並建有簡單的草寮。

而在明朝中葉之後，安平一帶了聚集明朝和日本的走私客，以及葡萄牙、西班牙與英國等歐洲商人互相或與[[平埔族|平埔族]]進行貿易，因而在今安平延平街形成市街。不過當時市街的長度似乎只有從今天延平街104巷往東到石門國小而已，至於往西的部分則在今[[臺南周龍殿|周龍殿]]處設有仲裁所來處理商業糾紛，而當無法解決時，則在仲裁所與上鯤鯓沙崙之間的決鬥場解決。

而在1604年[[荷蘭東印度公司|荷蘭東印度公司]]派遣[[韋麻郎|韋麻郎]](Wijbrant van Waerwijck)到澎湖時，他除了派人到[[廈門|廈門]]與明朝交涉通商事宜外，也派人到安平海域尋找其他的良港。

=== 日治時期 ===
1895年[[乙未戰爭|乙未戰爭]]末期，臺南一帶駐軍主帥[[劉永福|劉永福]]潛逃至廈門後，安平駐軍士氣低落，10月20日，在安平海關稅務司主管嘉麥林（C.A.MacCallum）勸告下交出武器裝備，收藏於海關鴉片倉庫準備投降{{r|張}}。次日（10月21日），日軍{{link-ja|山口素臣|山口素臣}}少將率軍進入臺南府城，而海軍常備艦隊有地中將於清晨則派部隊於四鯤鯓附近登陸，之後佔領[[二鯤鯓砲臺|二鯤鯓砲臺]]（億載金城），進入安平市街{{r|張}}。10月23日，因運輸船「旅順號」在前往香港修理的路上剛好途經安平，日軍利用該船搭載安平一帶5千多人的俘虜遣返到金門{{r|張}}。而後該年10月27日創立臺南民政支部安平出張所，所長是[[西鄉菊次郎|西鄉菊次郎]]，而開始運作則是在11月1日{{r|張}}。安平出張所的職員有日籍12名，臺籍5名，臺灣人主要是擔任輔助的工作{{r|張}}。此外在11月2日時，安平出張所選出蔡崇玉為安平六社的總理{{r|張}}。

==地理==
除安平舊社區外，安平區大部分土地皆為[[臺江內海|臺江內海]]淤積所形成的海埔新生地及市地重劃後填土所產生，日治時期後，淤積更為嚴重，遂修築[[臺南運河|臺南運河]]供[[臺南府城|城內]]船隻進出[[安平港|安平舊港]]，淤積後部分之新生地初期則供給養殖漁業使用，故安平區尚未進行市地重劃填土前，須乘坐竹筏方能到達各[[鯤鯓|鯤鯓]]。

現行行政區主要由安平舊社區、三鯤鯓及[[臺南市第五期市地重劃區|五期重劃區]]所形成，其中安平舊社區與三鯤鯓之間的原有的陸地連接道路，因為安平新漁港開闢的關係，已經挖除為港區海域，三鯤鯓地區（漁光里）目前並無與臺南市區陸地相連，成為安平港區間的小島，主要的對外道路倚賴位於健康路末端與億載金城之間，新闢建的漁光大橋（於2009年8月通車）。其包含過去的一鯤鯓（含舊社區）、三鯤鯓（億載金城南邊對岸處，今[[秋茂園|秋茂園]]所在處，目前原址部分已成為安平新漁港漁船進出港區）以及二鯤鯓（[[億載金城|億載金城]]至安平國中一帶）和與臺南運河圍繞之新生地、重劃地產生之[[臺南市第五期市地重劃區|五期重劃區]]。

=== 新舊之分 ===
[[File:Street_View_of_Yonghua_Road.jpg|thumb]]與[[中華西路|中華西路]]口]]
[[台灣戰後時期|二戰後時期]]的安平區，係合併[[台灣日治時期|日治時期]]的[[臺南市_(州轄市)|臺南市]]直轄的「[[安平_(臺南市地名)|安平]]」、「上鯤鯓」兩個[[大字_(行政區劃)|大字]]而來，沿續至今產生「舊安平」、「新安平」的差別。

一般語境下所稱的安平，僅單指舊安平。其為歷史悠久、古蹟眾多的知名觀光區，由6個角頭（海頭社、王城西社、灰窯尾社、港仔尾社、十二宮社、囝子宮社）組成。老安平聚落旁有安平舊港，[[臺灣清治時期|清治時期]]至日治初期曾為[[南臺灣|南臺灣]]第一大港，今已封淤。

而上鯤鯓即所謂的新安平，為[[台江內海|台江內海]]遺跡，自清治時期以來逐漸轉為魚塭使用。1979年的都市計畫中，確立填平魚塭以擴大市區的方針，稱為「臺南新市區」，並編為[[臺南市第五期市地重劃區|第五期市地重劃區]]，衍生'''五期'''成為當地俗稱的地名。1997年將市政府與市議會遷至五期，與舊安平相隔[[臺南運河|運河]]。2009年5月18日起，安平區公所亦由安平，搬遷至五期的區政大樓辦公。

另本區西南端與鄰近的[[南區_(臺南市)|南區]]設有安平工業區與[[安平港|安平商港]]（大部分工業區及實際港口位置設於南區，安平區之[[鯤鯓|三鯤鯓]]為新港北防波堤、南區之[[鯤鯓|四鯤鯓]]為新港南防波堤），臺南人另稱工業區或安平工業區以便與安平區區別，而安平商港又稱新港口、安平新港，與舊港區別。

== 民俗文化 ==
安平區三百餘年來，歷經[[臺灣荷蘭統治時期|荷蘭]]、[[鄭氏王朝|明鄭]]、[[臺灣清治時期|清朝]]，及[[臺灣日治時期|日治]]各時期，滄桑屢經，變革相乘，可是安平的舊街道市仔街，依然保持著若干清代古老房屋的建築，許多古時居民的習俗，在安平仍沿襲著，較臺南市猶為守舊。比方安平劍獅、安平風獅爺與臺南市區內已漸式微的擋煞用「刀劍屏」，安平仍到處可見。其為住宅在庭院（埕斗）對著門口處，立一木屏風，上畫獅頭（口含劍）、八卦，屏風上插有木製刀劍戟五支至七支，它與「石敢當」一樣，是作為擋煞用的。

== 交通 ==
每逢年節及假日期間進入安平舊社區之車輛頗多，除依賴[[安平路|安平路]]→[[民生路_(台南市)|民生路]]通往市區之外，假日建議採用四草大橋往北，或從安億橋往南經過五期重劃區為建議路線，避開塞車路段。近年已有規劃[[臺南捷運|臺南捷運]]的構想。市府也為了安平地區交通問題，已規劃於城平路與永華路二段間興建「安平跨港大橋」來紓解交通，預計於2019年中旬動工。

四草大橋與漁光大橋為安平區內兩跨港大橋，四草大橋位鹽水溪西流至安平附近，與[[嘉南大圳|嘉南大圳]]匯流進入臺灣海峽，而四草大橋橫跨其上（一鯤鯓西段），直通[[安南區|安南區]][[鹿耳門|鹿耳門]]，可於橋頂觀賞潟湖生態及觀賞臺灣十大美景-安平夕陽；漁光大橋則跨接三鯤鯓與五期重劃區，往北可達[[秋茂園|秋茂園]]，但目前已無道路直通安平舊社區（一鯤鯓）。
