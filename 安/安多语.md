== 概况 ==
'''安多方言'''（{{Bo|z=ཨ་མདོ་སྐད་|w=A-mdo skad}}）是一种主要使用于[[安多地区|安多地区]]的[[藏缅语族|藏缅语族]]藏语的方言。 

'''条目标题涉嫌错误，语是语言分类的一个大系，而此条目所说的“<s>安多语</s>”仅为藏语的具体方言，且藏语本身并没有独立的形态标准，所以姑且应该称之为安多方言。其他的卫藏语和康巴语也应该改为卫藏方言和康巴方言。''完整应该称作藏语安多方言，而非安多语。'''''

安多方言语音上迄今较多的[[古藏语|古藏语]][[复辅音|复辅音]]声母，部分发音没有[[聲調|聲調]]变化。从语言进化的角度而言，这一点区别于[[卫藏方言|卫藏方言]]、[[康方言|康方言]]。

<br />

== 分布范围 ==
安多方言在其分布范围上可以分成四个次方言区：

*牧区土语：分布在[[安多地区|安多地区]]牧业生产区，跨青海省藏族聚居区、甘肃省藏族聚居区、四川省藏族聚居区三省从事牧业的藏族群体、使用人数较为分散，但语言共同性很强。
*半农半牧区土语：分布在[[同仁县|同仁县]]、[[夏河县|夏河县]]等半农半牧为经济主体的藏族聚居区
*农区土语：分布在青海省黄河以北的农业生产区，含化隆藏人居区区（人数最多）、尖扎县、循化县大部分地区、乐都县局部地区、西宁部分藏人、以及天祝藏人聚居区
*道孚土语：分布在[[道孚县|道孚县]]、[[炉霍县|炉霍县]]（地理上属于[[康区|康区]] ，在语言学研究上称其为方言岛）

<br />

== 媒体使用 ==
目前青海藏语广播、青海[[安多卫视|安多卫视]]都使用安多語播音、中央人民广播电台也有每天两小时安多方言新闻和娱乐内容播报。

<br />

== 参考价值 ==
安多方言因为保留着较多的古藏语语音特征和文法特征，在国际藏学学术上占据很重要的地位，在解读敦煌古藏文和上古汉语语音上，也有很高的参考价值。

<br />

== 语言现状 ==
目前安多方言使用人数因为民族同化的原因，在境内使用人数呈减少趋势。
