'''安纳布尔纳环线'''（{{Lang-en|'''Annapurna Circuit'''}}）是一条环绕[[尼泊尔|尼泊尔]]中部[[安纳布尔纳峰|安纳布尔纳峰]]地区的{{Link-en|野外徒步|Backpacking (wilderness)}}路线。<ref>[http://www.backpacker.com/trips/international/the-perfect-circle-hiking-the-annapurna-circuit/#bp=0/img1 The Annapurna Circuit]</ref> 整条徒步路线的总长度在{{convert|160|-|230|km|mi}}之间变换，这具体取决于起点位置和终点位置的不同。路线穿越了两个不同的河谷并环绕整个[[安纳布尔纳峰|安纳布尔纳峰]]。安纳布尔纳环线的最高点在{{Link-en|驼龙垭口|Thorong La}}{{efn|1=中文译名参考自[[纽约时报中文网|纽约时报中文网]]报道《[https://cn.nytimes.com/world/20141016/c16nepal/ 喜马拉雅山雪崩至少20人遇难]》。}}，其海拔高度为{{convert|5416|m|ft}}，已经触及[[青藏高原|青藏高原]]的边缘。实际上，绝大部分徒步旅行者都会选择逆时针行走这条路线，因为这样每日增加的海拔高度会比较缓慢，所以通过驼龙垭口也会相对更安全和更容易。<ref>{{citeweb|url=http://travel.lumbinimedia.com/2017/01/annapurna-short-trek-range.html|title=Annapurna Short Trek Range|publisher=Lumbini Media}}</ref>

通过安纳布尔纳环线可以近距离观赏的山景包括：[[安纳布尔纳峰|安纳布尔纳峰]]、[[道拉吉里峰|道拉吉里峰]]、[[鱼尾峰|鱼尾峰]]、[[马纳斯卢峰|马纳斯卢峰]]、[[安纳布尔纳峰|刚嘎普尔那峰]]（{{lang|en|Gangapurna}}）{{efn|name=CMA|1=中文译名参考自中国登山协会资料《[http://cmasports.sport.org.cn/zlk/2013-03-07/392752_10.html 十四座8000米以上山峰介绍（图文）]》。}}、[[蒂利乔峰|蒂利乔峰]]、{{Link-en|皮桑峰|Pisang Peak}}和{{Link-en|庞达丹达峰|Paungda Danda}}，以及其他安纳布尔纳地区海拔在6000米到8000米的山峰。

安纳布尔纳环线的徒步旅行一般开始于[[马沙阳蒂河|马沙阳蒂河]]河谷的{{Link-en|贝西沙尔|Besisahar}}或布贝尔（{{lang|en|Bhulbhule}}），并在{{Link-en|卡利甘达基峡谷|Kali Gandaki Gorge}}结束。从[[加德满都|加德满都]]驾车7小时即可抵达贝西沙尔。这条路线经过稻田，穿过亚热带森林，并经过几条瀑布和巨大悬崖，以及几个村落。

安纳布尔纳环线被誉为是世界上几条最好的野外长途徒步路线之一。因为它以完整未开发的形态展示了各种气候带景色，从海拔600米的[[热带气候|热带气候]]到海拔5,416米的驼龙垭口的[[极地气候|极地气候]]；同时这里也有丘陵地区的[[印度教|印度教]]村落，还有位于[[馬南_(尼泊爾)|马南谷]]和下慕斯唐（{{lang|en|Lower Mustang}}）的[[西藏文化|西藏文化]]村落。随着当地政府在这里修建道路，导致整条路线的步行时间被缩短，但同时也让这里成为[[山地自行车|山地自行车]]骑行者的胜地。特别是慕斯唐是他们的最爱的目的地之一。<ref>{{Cite web|title = Annapurna Bike tour in Kagbeni, Mustang|url = http://mytrip2nepal.com/annapurna-bike-tour.html|website = MyTrip2Nepal|language = en-gb}}</ref>

== 标准耗时 ==
{{wide image|Annapurna Massif Panorama.jpg|990px|[[尼泊尔|尼泊尔]][[馬南_(尼泊爾)|马南]]附近的[[安纳布尔纳峰|安纳布尔纳峰]]景色。}}

徒步走完安纳布尔纳环线大约需要15到20天，<ref name=":0">{{Cite web|url=http://www.alienadv.com/faq/10/Annapurna-Circuit-Trek/|title=Annapurna Circuit Trek|access-date=2016-08-23}}</ref> 从[[加德满都|加德满都]]离开后在[[博克拉|博克拉]]停留，然后再返回加德满都。<ref name=":0" /> 这条徒步旅行路线相对比较温和但又具有一定的挑战性，沿路的河流之上修建了许多钢制或木制的[[悬索桥|悬索桥]]。沿路的茶馆及小屋大多可提供住宿及餐饮服务。某些徒步人士或团体也会选择携带帐篷，但他们选择的路线通常远离村落，或有登山计划。<ref name=":0"/><ref>{{Cite web|url=http://wanderingsasquatch.com/how-to-hike-the-annapurna-circuit/|title=How to Hike the Annapurna Circuit|date=2013-06-13|language=en-US|access-date=2016-08-23}}</ref>

=== 大致行程 ===
[[File:Marsyangdi_river_valley_near_Pisang.jpg|thumb]]。]]

# 第一天：从[[加德满都|加德满都]]出发，搭乘私家车或大巴向西至{{Link-en|贝西沙尔|Besisahar}}（{{convert|820|m|ft|disp=or}}），耗时大约6至7个小时。
# 第二天：步行至{{Link-en|库迪 (尼泊尔)|Khudi, Nepal|库迪}}（{{convert|790|m|ft|disp=or}}）。
# 第三天：步行至{{Link-en|巴浑丹达|Bahundanda}}（{{convert|1310|m|ft|disp=or}}）。
# 第四天：步行至{{Link-en|加贾特|Jagat}}（{{convert|1290|m|ft|disp=or}}）。
# 第五天：步行至{{Link-en|达拉巴尼 (甘达基)|Dharapani, Gandaki|达拉巴尼}}（{{convert|1920|m|ft|disp=or}}）。
# 第六天：步行至{{Link-en|茶么 (尼泊尔)|Chame, Nepal|茶么}}（{{convert|2630|m|ft|disp=or}}）。
# 第七天：步行至下{{Link-en|皮桑|Pisang}}{{\}}上皮桑（{{convert|3190|m|ft|disp=or}}）。
# 第八天：步行至[[馬南_(尼泊爾)|马南]]（{{convert|3520|m|ft|disp=or}}）。
# 第九天：在马南（{{convert|3520|m|ft|disp=or}}）休整一日。
# 第十天：步行至利勒达（{{lang|en|Letdar}}）（{{convert|4250|m|ft|disp=or}}）。
# 第十一天：步行至索隆菲迪（{{lang|en|Thorung Phedi}}）（{{convert|4500|m|ft|disp=or}}）。
# 第十二天：步行至{{Link-en|慕克缇娜斯|Muktinath}}（{{convert|3800|m|ft|disp=or}}），途中将穿越{{Link-en|驼龙垭口|Thorong La}}（{{convert|5416|m|ft|disp=or}}）。
# 第十三天：步行至{{Link-en|玛法|Marpha}}（{{convert|2665|m|ft|disp=or}}）。
# 第十四天：步行至{{Link-en|莱特 (尼泊尔)|Lete, Nepal|莱特}}（{{convert|2470|m|ft|disp=or}}）；现在通常会在这段行程使用汽车或山地车，故不会选择在莱特停留一日。
# 第十五天：步行至{{Link-en|塔托帕尼 (苗地)|Tatopani, Myagdi|塔托帕尼}}（{{convert|1160|m|ft|disp=or}}）；如果选用汽车或山地车，则会和上面的行程合并成第十四日行程。
# 第十六天：步行至{{Link-en|果瑞巴尼|Ghorepani}}（{{convert|2775|m|ft|disp=or}}）。
# 第十七天：步行至比瑞丹提（{{lang|en|Birethanti}}）（{{convert|1050|m|ft|disp=or}}），从这里搭车返回[[博克拉|博克拉]]。
# 第十八天：返回加德满都。
<small>* 注：''括号内的高度数字为当地[[海拔高度|海拔高度]]。''</small>
