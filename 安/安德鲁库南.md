'''安德鲁·菲利普·库纳南''' （英文：Andrew Philip Cunanan，1969年8月31日 – 1997年7月23日） ，美国[[連環殺手|連環殺手]]<ref>{{cite web|url=https://www.fbi.gov/news/stories/2014/march/serial-killers-part-6-andrew-cunanan-murders-a-fashion-icon|title=FBI — Serial Killers, Part 6: Andrew Cunanan Murders a Fashion Icon|publisher=FBI|archiveurl=https://web.archive.org/web/20160702022814/https://www.fbi.gov/news/stories/2014/march/serial-killers-part-6-andrew-cunanan-murders-a-fashion-icon|archivedate=July 2, 2016|deadurl=yes|df=}}</ref> 。在1997年中旬的三个月期间，至少杀害五人，其中包括时尚设计师[[詹尼·范思哲|詹尼·范思哲]] 以及芝加哥大亨 [[李·米格林|李·米格林]]。 1997年7月12日，库纳南成为第449名[[美國聯邦調查局十大通緝要犯|美國聯邦調查局十大通緝要犯]]。此次连环杀人事件以库南在7月23日的吞枪自杀作为结束。此时他27岁。 

在库纳南生命的最后几年中，他一直没有固定的工作。他与富有的年长男士结交，并通过在俱乐部吹嘘自己，或在餐馆为他人买单，使自己在当地同性恋社区获得声誉。<ref name="CT">{{cite web|url=http://articles.chicagotribune.com/1997-05-16/news/9705160213_1_andrew-cunanan-jeffrey-trail-artsy-crowd/2|title=The Many Faces of Andrew Cunanan: 'He Could Win Anyone Over'|accessdate=April 30, 2017|date=May 16, 1997|work=[[Chicago_Tribune|Chicago Tribune]]|publisher=[[Tronc|Tronc]]||first1=Dion|last1=Haynes|first2=Bob|last2=Secter|location=Chicago, Illinois}}</ref> 案发前一年，即1996年，库南被一位百万富翁抛弃。 <ref name="CT" />

== 早年生活 ==
[[File:Andrew_Phillip_Cunanan_FBI_pictures.jpg|左]]
库纳南生于[[美国|美国]][[加利福尼亚州|加利福尼亚州]][[纳雄耐尔城_(加利福尼亚州)|纳雄耐尔城]]。父亲，莫德斯托·“皮特”·库纳南（Modesto "Pete" Cunanan），[[菲律賓裔美國人|菲律宾裔美国人]]；<ref>{{cite web|url=http://articles.latimes.com/1997/sep/19/entertainment/ca-33799|title=Cunanan's Father Plans Documentary on Son's Life|accessdate=March 2, 2018|date=September 19, 1997|last=Welkom|first=Robert W.|work=[[洛杉磯時報|Los Angeles Times]]|publisher=[[Tronc|Tronc]]||location=Los Angeles, California}}</ref>母亲玛丽·安妮·希拉奇（Mary Anne Schillaci），[[意大利裔美国人|意大利裔美国人]]。库纳南是四个孩子中最小的一个。在库纳南出生时，父亲莫德斯托·库纳南正服役于[[美國海軍|美国海军]]参加[[越南战争|越南战争]]中。

1981年，父亲将他送往在[[聖地牙哥_(加利福尼亞州)|圣地亚哥]][[拉霍亚|拉霍亚]]附近的[[独立学校|独立学校]][[加州毕夏普学校|加州毕夏普学校]]（The Bishop's School） 。在校期间，库纳南给人一种活泼、健谈的印象，[[智商|I.Q.]]测试值147。<ref name="Orth">{{cite book|url=http://www.randomhouse.com/catalog/display.pperl?isbn=978-0-440-22585-0&view=excerpt|title=Vulgar Favors|last=Orth|first=Maureen|authorlink=Maureen Orth|publisher=[[Dell_Publishing|Dell Publishing]]|year=2000|isbn=978-0-440-22585-0|location=New York City}}</ref> 然而，他逐渐变成一个说谎成性的青年，常用谎言讲述他的家庭和他的个人生活。他也擅长改变自己的外貌，把自己装扮成他自认为好看的样子。

19岁时，父亲抛弃了家人，以逃避因[[贪污|贪污]]被捕。同年，母亲也意识到库纳南是[[同性戀|同性恋]]。在之后的争论中，库纳南将母亲扔到墙上导致母亲手臂[[脫臼|脱臼]]。在后来的报告中指出，他的行为表明他可能患有[[反社会人格障碍|反社会人格障碍]]，这是一种以异常缺乏[[同情心|同情心]]为特征的人格障碍（早期称为[[精神病|精神病]]）。

1987年高中毕业后，他就读于美国[[聖地牙哥加利福尼亞大學|圣地亚哥加利福尼亚大学]]美国历史专业。 <ref>{{cite web|url=http://www.crimeandinvestigation.co.uk/famous_criminal/6/biography/1/Andrew_Cunanan.htm|title=Famous Criminals: Andrew Cunanan|accessdate=October 13, 2010|publisher=Crimeandinvestigation.co.uk|archiveurl=https://web.archive.org/web/20090709185826/http://www.crimeandinvestigation.co.uk/famous_criminal/6/biography/1/Andrew_Cunanan.htm|archivedate=July 9, 2009|deadurl=yes|df=}}</ref> 辍学后，他定居于[[旧金山|旧金山]][[卡斯楚街|卡斯楚街]]。

==案件经过==

1997年4月27日，库纳南在[[明尼阿波利斯|明尼阿波利斯]]开始了他的杀戮狂欢。首名被害者是库纳南在圣地亚哥的熟人杰弗里·特里奥（Jeffrey Trail），美国海军军官以及[[丙烷|丙烷]]销售员。由于一次争论，库纳南用[[羊角錘|羊角锤]]将特里奥砸死，随后将他的尸体用地毯裹好后藏匿于建筑师大卫·麦德逊的[[閣樓|阁楼]]中。<ref name=CT/>麦德逊是第二名被害者，曾一度是库纳南的伴侣。1997年5月3日，他的尸体在[[明尼苏达州|明尼苏达州]] [[拉什城|拉什城]]拉什湖东岸附近被发现。尸体头部和背部均有枪伤。<ref>{{cite web |url = http://www.amw.com/fugitives/case.cfm?id=55650 |title = 'America's Most Wanted': Andrew Cunanan |publisher = Amw.com |accessdate = October 13, 2010 |deadurl = yes |archiveurl = https://web.archive.org/web/20121025161155/http://www.amw.com/fugitives/case.cfm?id=55650 |archivedate = October 25, 2012 |df = }}</ref><ref>{{cite news |first1=William|last1=Recktenwald|first2=Andrew|last2=Martin|title=New Twist in Miglin Case |url = http://articles.chicagotribune.com/1997-05-08/news/9705080105_1_andrew-phillip-cunanan-david-madson-minneapolis-architect |work = [[芝加哥論壇報|Chicago Tribune]]|publisher=[[Tronc|Tronc]]|location=[[芝加哥|Chicago, Illinois]]|date = May 8, 1997|accessdate = October 8, 2014 }}</ref>1997年五月四日，库纳南驾车开往[[芝加哥|芝加哥]]，杀害了七十二岁的著名房地产开发商[[李·米格林|李·米格林]]。米格林被其用胶带捆住手脚，头部被胶带缠住。之后被螺丝钉戳刺二十多次，喉咙被[[钢锯|钢锯]]割开。<ref name="www.washingtonpost.com">{{cite news |last1 = Kastor |first1 = Elizabeth |last2 = Weeks |first2 = Linton |url = https://www.washingtonpost.com/wp-srv/national/longterm/cunanan/stories/victims.htm |title = Five Lives Cut Short |work = [[华盛顿邮报|Washington Post]] |publisher=[[杰弗里·贝索斯|Nash Holdings LLC]]|location=Washington, DC|date = July 17, 1997|accessdate=March 2, 2018 }}</ref> 此次案件之后，库纳南被联邦调查局列入[[美國聯邦調查局十大通緝要犯|美国联邦调查局十大通缉要犯]]。

{{anchor|William Reese}}五天后，库纳南驾驶着米格林的车来到[[新泽西州|新泽西州]][[彭斯威尔镇|彭斯威尔]]的[[芬恩国家公墓|芬恩国家公墓]]。射杀了45岁的看管员威廉·瑞斯，并偷走了他的红色皮卡。但在进行第五次也是最后一次谋杀之前，库纳南在[佛罗里达州迈阿密海滩] 藏匿了两个月。而此时的搜捕行动着重于瑞斯被盗的红色皮卡上<ref>{{cite web |last = Orth |first = Maureen |url = https://www.vanityfair.com/magazine/1997/09/cunanan199709 |title = The Killer's Trail|work=[[名利场_(杂志)|Vanity Fair]]|publisher=[[康泰纳仕|Condé Nast]]|location=New York City|date=September 1997|accessdate=March 2, 2018}}</ref> 虽然他知道警方经常检查[[典當業|当铺]]的记录来找寻失窃物品，但他还是用自己的真名典当了一件被盗物品。<ref name="Versace's Killer Kills Self">{{cite web |last = Phillips |first = Andrew |url = https://thecanadianencyclopedia.ca/en/article/versaces-killer-kills-self/ |title = Versace's Killer Kills Self |work = [[麦克林|Maclean's]] |publisher=[[羅渣士傳媒|Rogers Media]]|location=[[多伦多|Toronto, Ontario]]|date = August 4, 1997|archive-url = https://web.archive.org/web/20090630201120/http://www.thecanadianencyclopedia.com/index.cfm?PgNm=TCE&Params=M1ARTM0011375 |archive-date = June 30, 2009 }}</ref>
1997年7月15日，库纳南在意大利时尚设计师[[詹尼·范思哲|詹尼·范思哲]]位于迈阿密豪宅的门口连开两枪射杀了他。<ref>{{cite news |last = Lecayo |first = Richard |url = http://www.time.com/time/magazine/article/0,9171,138062,00.html |title = Tagged for Murder |work = [[時代雜誌|Time]] |publisher=[[Meredith_Corporation|Meredith Corporation]] |location=New York City|date = June 21, 2001|accessdate=March 2, 2018 }}</ref>  一位目击者曾尝试追赶他，但是没有成功。接警警察在附近的[[停車場|停车场]]发现了瑞斯被盗的皮卡，以及库纳南的衣服、备用护照和库纳南谋杀案的简报。<ref name="douglas">{{cite book |first1 = Danielle |last1 = Esposito |first2 = John E. |last2 = Douglas |first3 = Ann W. |last3 = Burgess |first4 = Allen G. |last4 = Burgess |editor1-first = John E. |editor1-last = Douglas |editor2-first = Ann W. |editor2-last = Burgess |editor3-first = Allen G. |editor3-last = Burgess |title = Crime Classification Manual: A Standard System for Investigating and Classifying Violent Crimes |url = https://books.google.com/books?id=DIc9F5ZzO7YC&dq=andrew+cunanan+spree+killer+FBI |accessdate = April 10, 2011 |edition = 2nd |year = 2006 |publisher = John Wiley and Sons |location=Hoboken, New Jersey|isbn = 978-0-7879-8501-1 |pages = 448–452 |chapter = Case Study: Andrew Cunanan }}</ref>

==死亡==

1997年7月23日，在杀死范思哲八天后。库纳南在[[迈阿密海滩|迈阿密海滩]][[船屋|船屋]]楼上右侧的卧室里开枪自杀身亡。他使用的是从第一名受害者杰弗里·特里奥处偷得的一把[[.40_S&W|.40 S&W]][[口径|口径]]的[[金牛座PT92手槍|金牛座PT92]][[半自動手槍|半自动手枪]]。他也是用这把手枪杀害了麦迪逊、瑞斯和范思哲。<ref name="douglas"/><ref name="Versace's Killer Kills Self"/><ref>{{cite news |url = https://www.nytimes.com/1997/07/25/us/suspect-s-suicide-brings-relief-and-normality.html |title = Suspect's Suicide Brings Relief and Normality |last = Janofsky |first = Michael |work = [[纽约时报|The New York Times]] |publisher = The New York Times Publishing Company | location = New York City | date = July 25, 1997 |accessdate = August 4, 2009 }}</ref>  死后葬于[[加州圣地亚哥圣十字天主教墓地|加州圣地亚哥圣十字天主教墓地]]。

==动机==

库纳南的明确动机至今未知。在案件发生时期，普遍大众和媒体猜测库纳南之所以会杀人是因为他感染了[[人類免疫缺陷病毒|人類免疫缺陷病毒]];<ref>{{cite journal|last1=Cenite|first1=Mark|title=The Obligation to Qualify Speculation|journal=Journal of Mass Media Ethics|date=1 March 2005|volume=20|issue=1|page=43-44|publisher=Taylor & Francis Group|location=Abingdon, Oxford|doi=10.1207/s15327728jmme2001_4|url=https://doi.org/10.1207/s15327728jmme2001_4}}</ref> 然而，验尸报告表明库纳南的HIV检测呈阴性。<ref>{{cite web |url = http://www.cnn.com/WORLD/9707/17/cunanan/ |title = Who is Andrew Cunanan? |archive-url = https://web.archive.org/web/20060112181305/http://www.cnn.com/WORLD/9707/17/cunanan/ |archive-date = January 12, 2006 |publisher = CNN |date = July 17, 1997 }}</ref><ref>Cunanan, Andrew – Autopsy report #1997-01742, Miami Medical Examiner.</ref>

尽管警察搜查了库纳南自杀的船屋，但并没有发现遗书，只找到少量个人物品。<ref name=Gibson138>{{cite book |first = Dirk Cameron |last = Gibson |title = Serial Murder and Media Circuses |location = |publisher = Greenwood Publishing Group |year = 2006 |page = 138 }}</ref> 这使得调查人员诧异，因为他经常用他的名声从富有的老年男子处获得金钱。<ref name=Gibson138/> 警方认为除了多管[[皮质醇|皮质醇]]乳膏和相当多的[[C·S·路易斯|C·S·路易斯]]的小说集外，没有其他重要的发现。<ref name=Gibson138/><ref>{{cite web |first=Ben|last=Raworth|department = This Day in History |title = July 15: Gianni Versace Killed |work = [[美信|Maxim]] |publisher=Biglari Holdings|location=San Antonio, Texas|date = July 2009 |accessdate=March 3, 2018|archivedate = March 18, 2012 |archiveurl = https://web.archive.org/web/20120318060459/http://www.maxim.co.uk/features/articles/17299/this_day_in_history.html |url = http://www.maxim.co.uk/features/articles/17299/this_day_in_history.html }}</ref><ref>{{cite book |first=Tyler|last=Stoddard Smith|title = Whore Stories: A Revealing History of the World's Oldest Profession |date=July 18, 2012|publisher=Adams Media|location=Adams, Massachusetts|page = 172 }}</ref>
