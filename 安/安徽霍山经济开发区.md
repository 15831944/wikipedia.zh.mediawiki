'''安徽霍山经济开发区'''<ref name="a1"/><ref name="a3"/>是位于{{PRC admin/fullname|34/15/25/400/000|3}}东部的一个省级经济开发区，辖区以'''经济开发区'''之名列为霍山县的一个[[类似乡级单位|类似乡级单位]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|34/15/25/400/000}}}}</ref>。前身是霍山县生态工业园区<ref name="a2"/>。

开发区的成立时间可追溯至2002年<ref name="a3">{{Cite web |url=http://www.ahhuoshan.gov.cn/openness/detail/content/55a308b1e1d5bdd3069bac66.html |title=安徽霍山经济开发区简介|accessdate=2018-09-12 |author= |date=2018-06-16  |work= |publisher=霍山县人民政府网站|language=Zh-hans }}</ref>。2005年时，[[安徽省人民政府|安徽省人民政府]]将之升级为经济开发区。中共霍山县委、县政府将[[衡山镇|衡山镇]]的城东社区（设置于2004年）划归其管辖<ref name="a2">{{Cite web |url=http://www.ahhuoshan.gov.cn/content/detail/53b22bede1d5bde00c701f59.html |title=第二节 行政区划 |accessdate=2018-09-12 |author= |date=2018-03-01 |work= |publisher=霍山县人民政府网站|language=Zh-hans }}</ref>。2006年4月，升级为省级开发区<ref name="a3"/>。

现仅领城东社区，面积28平方公里，下设59个村民组，户籍人口7235人，常住人口14061人。城东社区东邻[[但家庙镇|但家庙镇]]，南邻衡山镇<ref>{{Cite web |url=http://www.ahhuoshan.gov.cn/openness/detail/content/558d1266e1d5bd17b6dd0aa7.html|title=城东社区简介|accessdate=2018-09-12 |author= |date=2015-06-26 |work= |publisher=霍山县人民政府网站|language=Zh-hans }}</ref>。行政机关是安徽霍山经济开发区管委会<ref name="a1">{{Cite web |url=http://www.ahhuoshan.gov.cn/openness/detail/content/5af256471ea8ace424000025.html|title=县经济开发区机构设置|accessdate=2018-09-12 |author= |date=2015-06-26 |work= |publisher=霍山县人民政府网站|language=Zh-hans }}</ref><ref name="a3"/>。
