'''安東尼·阿爾巴尼西'''（{{lang-en|'''Anthony Albanese'''}}，{{bd|1963年|3月2日|catIdx=Albanese}}），全名'''安東尼·諾曼·阿爾巴尼西'''（{{lang-en|'''Anthony Norman Albanese'''}}），[[澳大利亞|澳大利亞]]工党籍政治家，曾担任过[[澳大利亞工黨|澳大利亞工黨]]副黨魁、[[澳大利亞副總理|澳大利亞副總理]]及众议院领袖等要职。

== 生平 ==
阿爾巴尼西出生于[[悉尼|悉尼]]，是[[意大利|意大利]]和[[愛爾蘭|愛爾蘭]]人後裔，幼年父母離異，在悉尼内城區的貧困環境中長大。幼年就讀于悉尼天主教教會學校。中學畢業后，阿爾巴尼西進入[[澳洲聯邦銀行|澳洲聯邦銀行]]工作，兩年后入讀[[悉尼大學|悉尼大學]]，在校期間開始接觸左派政治，入選學生代表理事會，經濟學專業畢業后便擔任州政府廳長幕僚。1989年至1995年任[[澳大利亞工黨|澳大利亞工黨]][[新南威爾士州|新南威爾士州]]黨部助理總書記，1995年至1996年為時任州長[[鲍勃·卡尔|鲍勃·卡尔]]的高級幕僚。1996年當選聯邦議員（代表悉尼内西區的[[格賴恩德勒_(澳大利亞國會選區)|格賴恩德勒]]選區）一直至今。曾擔任基建、運輸、鄉村地區發展和地方政府部長，以及衆議院領袖等。

在2013年6月26日的工黨黨團表決中當選為副黨魁，并在次日獲任命為[[第二次陆克文内阁|第二次陆克文内阁]]中的副總理，成为总理、工党党魁[[陆克文|陆克文]]的副手。

[[2013年澳洲联邦大选|2013年澳洲联邦大选]]中工党失利后，陆克文辞去党魁职务。2013年9月13日阿爾巴尼西宣布参选党魁职务。

2013年9月18日[[艾伯特内阁|新政府]]宣誓就职后其副总理、众议院领袖任期结束。他轉為影子內閣擔任影子基礎設施、交通及旅遊部長。

== 個人生活 ==
阿爾巴尼西的妻子是新南威爾士議員、前副州長{{link-en|卡梅爾·特布特|Carmel Tebbutt}}。

== 中文音译 ==
“阿爾巴尼西”是其姓氏在澳大利亚英语中的普遍读音。此姓氏出于[[意大利语|意大利语]]。如果按照其姓氏在意大利语中的读音，中文音译一般为“阿尔巴内塞”。阿爾巴尼西本人也使用另一种讀音“阿尔巴尼斯”。
