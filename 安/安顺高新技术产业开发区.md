'''安顺高新技术产业开发区'''前身为[[黎阳高新技术工业园区|黎阳高新技术工业园区]]<ref>{{cite web |url=http://www.gywb.cn/content/2017-06/29/content_5548320.htm|title=安顺国家高新技术产业开发区正式挂牌成立 |accessdate=2017-06-29 |publisher=贵阳网}}</ref>，位于[[贵州省|贵州省]][[安顺市|安顺市]][[平坝区|平坝区]]南侧，2001年4月经由贵州省人民政府批准成立，系贵州省第一个省级高新技术产业园区<ref>{{cite web |url=http://gz.people.com.cn/n2/2017/0624/c222152-30373971.html|title=安顺高新技术产业开发区升级为国家级高新区 |accessdate=2017-06-24|publisher=人民网}}</ref>。2011年12月，安顺高新区启动国家高新区创建工作，并于2017年2月13日<ref>{{cite web |url=http://www.most.gov.cn/dfkj/gz/zxdt/201703/t20170302_131569.htm|title=贵州省安顺高新技术产业开发区获批国家高新技术产业开发区 |accessdate=2017-03-03|publisher=中华人民共和国科学技术部网}}</ref>获得批准予以升级，核准规划面积422公顷<ref>{{cite web |url=http://www.gz.xinhuanet.com/2017-03/03/c_1120564768.htm|title=安顺高新区正式升格国家级高新区 |accessdate=2017-03-03|publisher=新华网}}</ref>。2017年6月23日，安顺国家高新技术产业开发区挂牌成立</ref>获得批准予以升级，核准规划面积422公顷<ref>{{cite web |url=http://www.anshun.gov.cn/xwzx/jrtt/201706/t20170626_2561108.html|title=安顺国家高新技术产业开发区挂牌成立 |accessdate=2017-06-26|publisher=安顺市人民政府网}}</ref>，系贵州省第二个国家级高新区。
