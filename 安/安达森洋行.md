'''安达森洋行'''，位于[[重庆市|重庆市]][[南岸区|南岸区]][[南滨路|南滨路]]海狮支路2号。

==历史==
19世纪末重庆开埠初期，[[瑞典|瑞典]]商人安达森在[[长江|长江]]边依山兴建安达森洋行，由8栋建筑组成。截至2017年，其中4栋建筑2013年成为第二批[[南岸区文物保护单位|南岸区文物保护单位]]“安达森洋行”，另有1栋为优秀历史建筑，3栋为传统风貌建筑<ref name=zhenshi>{{Cite web |url = http://cq.sina.com.cn/news/b/2017-03-31/detail-ifycwyns3851961.shtml |title =南滨路上的“故宫” 安达森洋行10月对外开放 |author =  |date = 2017-03-31 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web|url =http://www.cqna.gov.cn/html/zjna/jrna/13/11/22397.html|title =郑向东主持召开区政府第72次常务会议审议通过南岸区第二批区级文物保护单位名单|author =|date =2013-11-26|publisher =重庆市南岸区人民政府|language =|accessdate =|quote =}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。安达森洋行成立后，主要经营腊肉、鬃毛、百货等<ref name=zhenshi/>。

1931年[[九一八事变|九一八事变]]后，[[日军|日军]]占领[[中国东北|中国东北]]，[[北平|北平]][[故宫博物院|故宫博物院]]文物被迫南迁。1938年5月，故宫博物院南迁至重庆的文物全部运至重庆后，分三处保存：（1）[[川康平民商业银行|川康平民商业银行]]，即今渝中区打铜街邮局大楼二楼仓库，存放3830箱；（2）南岸王家沱吉时洋行仓库，存放1814箱；（3）安达森洋行的4个仓库，存放3694箱<ref name=zhenshi/><ref>{{Cite web |url = http://news.sina.com.cn/o/2013-05-09/050927067587.shtml |title =这里保护过故宫的国宝  |author =  |date =2013-05-09  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。1939年4月起，日军飞机不断轰炸重庆。洋行老板安达森在每次轰炸时，都安排将[[瑞典国旗|瑞典国旗]]挂到安达森洋行房顶旗杆上，以保安全。最终安达森洋行幸免被炸，存放的故宫博物院文物因此未受损<ref name=zhenshi/>。

2016年11月，[[故宫博物院|故宫博物院]]院长[[单霁翔|单霁翔]]赴安达森洋行现场指导，由建筑师[[张永和|张永和]]主持设计。安达森洋行被纳入“慈云寺—米市街—[[龙门浩街道|龙门浩]]历史文化街区”首开区的范围内，准备引进“紫禁书院、数字故宫、文物南迁纪念馆”等故宫博物院主题的文创业态。2017年3月21日，[[重庆市文物局|重庆市文物局]]召开安达森洋行修缮设计方案评审会。修缮后的安达森洋行将与北京故宫博物院合作，建设“故宫南迁博物馆”以及“[[故宫研究院|故宫研究院]]重庆分院”，并引进“紫禁书院”、“数字故宫”等故宫博物院主题的文创业态，定期举办故宫文化讲座，成为艺术文化中心<ref name=zhenshi/><ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1689609 |title = 重庆将建“故宫南迁博物馆”及“故宫研究院重庆分院” |author =  |date =2017-05-19  |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref>。2018年1月19日，“[[故宫学院|故宫学院]]（重庆）”签约揭牌暨故宫文化传承活动在安达森洋行旧址举行<ref>{{Cite web |url = http://cq.cqnews.net/cqqx/html/2018-01/19/content_43693220.htm |title = 故宫学院落户重庆南岸 家门口就能看“国家宝藏” |author =  |date = 2018-01-19 |publisher = 华龙网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1960227  |title =  重庆将建故宫学院分院，计划在2019年春节对外开放|author =  |date =2018-01-20  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

==建筑==
安达森洋行的建筑为土木结构，各仓库都采用大梁穿斗结构，人字坡小青瓦屋面，土墙，石质基座。在2017年的修缮中，建筑采用嵌入式改造，将空地整合为大跨展示空间。文物保护建筑在临江面架观景平台<ref name=zhenshi/>。
