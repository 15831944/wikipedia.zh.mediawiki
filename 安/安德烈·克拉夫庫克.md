'''安德烈·克拉夫庫克'''（{{lang-en|'''Andrei Kravchuk'''}}，{{bd|1962年|4月13日}}）是一名[[俄羅斯|俄羅斯]]男導演和編劇。他較知名的作品如2016年電影《{{le|維京傳說：寒冰交鋒|Viking (film)}}》<ref name="@movies">{{@movies|id=fvru85157456|title=維京傳說：寒冰交鋒 Viking}}</ref>。

== 早期生活 ==
安德烈·克拉夫庫克的父親是一名海軍工程師，他的母親是一名醫生。

== 個人生活 ==
克拉夫庫克的妻子是一名設計師，並在結婚前就育有兩名年幼兒子。
