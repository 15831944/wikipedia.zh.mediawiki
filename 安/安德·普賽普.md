== 早年生活 ==
[[File:Karl_Puusepp.jpg|thumb]].]]
安德·普賽普出生于[[西伯利亚|西伯利亚]][[叶尼塞省|叶尼塞省]]一个爱沙尼亚籍农民家庭里。由于俄罗斯政府承诺提供他们力所能耕的大片土地，他的双亲受到举家迁至这里。有上千爱莎尼亚和[[拉脱维亚|拉脱维亚]]人来到西伯利亚并建立农场（[[:en:khutor|khutor]]），在那里他们传承语言和文化。 <ref name=Red /> 从童年开始，普赛普就梦想成为一名飞行员。但是他的父母却为他规划了一条不同的人生道路：当教师或者[[农艺家|农艺家]]。在念完七年级后，普赛普前往[[列宁格勒|列宁格勒]]的爱沙尼亚-芬兰师范学院进修。

== 飞行生涯 ==
在师范学院完成了一年学业后，普赛普转学到飞行员培训学校，最初在[[沃利斯克|沃利斯克]]训练，后来又转到[[奥伦堡|奥伦堡]]。毕业后，他作为飞行教官留在了奥伦堡。后来他又转到了一个新建立的特别盲飞（指由于视野受限而只能使用仪器协助飞行）中队。

=== 二战经历 ===
[[File:Puusepp_-_Ogonyok.jpg|thumb]]）杂志的封面照]]
1941年8月8日，在[[:en:Mikhail_Vasilyevich_Vodopianov|Mikhail Vodopianov]]的指挥下，普赛普参加了他生平第一次轰炸任务。在成功投弹袭击[[柏林|柏林]]返航途中，飞机被防空炮击中严重受损，结果只能在当时受[[纳粹|纳粹]]德国控制的[[爱沙尼亚|爱沙尼亚]]紧急迫降。在逃出飞机后，机组成员遇到了一个牧童。普赛普利用自己的母语和牧童交流，得知了纳粹军队的位置，从而成功避开追捕者，逃到了苏联控制地区。<ref name=Kp>{{cite web

== 晚年生活 ==

战后，普赛普决定迁往[[塔林|塔林]]，并成为[[爱沙尼亚苏维埃社会主义共和国|爱沙尼亚苏维埃社会主义共和国]]交通运输署的长官。1951年，他被任命为共和国[[最高苏维埃|最高苏维埃]]的副主席，之后又担任社会事务部部长。<ref name=peoples>{{cite web
