'''安全搜索'''（{{lang-en|SafeSearch}}）是[[Google搜索|Google搜索]]的一大特性，可以自動過濾掉[[色情|色情]]和具有潛在冒犯性的內容。Google對色情內容的過濾成為維持與中共當局合作的一大重要因素。<ref name=owen>{{cite news|last1=Fletcher|first1=Owen|title=Google porn filter gained China's thumbs-up|url=http://www.networkworld.com/article/2249577/applications/google-porn-filter-gained-china-s-thumbs-up.html|work=Network World|date=2009-09-07}}</ref>

2003年，[[哈佛法学院|哈佛法学院]]{{tsl|en|Berkman Center for Internet & Society|伯克曼互聯網與社會研究中心|伯克曼互聯網與社會研究中心}}的一份報告顯示，安全搜索過濾掉了[[白宮|白宮]]、[[IBM|IBM]]、[[美國圖書館協會|美國圖書館協會]]和{{tsl|en|Liz Claiborne|麗姿·克萊本|麗姿·克萊本}}等許多無害的網站。<ref>{{cite web |url=http://cyber.law.harvard.edu/archived_content/people/edelman/google-safesearch/ |title=Empirical Analysis of Google SafeSearch |date=2003-04-14 |author=Benjamin Edelman |publisher=Harvard University |accessdate=2013-02-03 }}</ref>而另一方面，許多色情圖片也成為了過濾器的漏網之魚。關鍵詞過濾機制不能阻擋用[[同形詞|同形詞]]（如“beaver”，女性的陰毛或陰部的一種粗俗說法，但該詞同時可以表示[[河狸|河狸]]）繞過過濾的情形。<ref>{{cite web |url=https://www.google.com/hostednews/afp/article/ALeqM5hJrLjBHj_y8G7OFPAZZmOhUL10Bw |title=Canada's The Beaver magazine renamed to end porn mix-up |date=2010-01-12 |author= |publisher=AFP |accessdate=2013-02-03 }}</ref>同時，阻擋色情網站網址的機制也會因為網站網址的更替而失效。而由於膚色的多樣化，加上嬰兒的照片也通常帶有大量肉色，用軟體將帶有肉色的圖片識別為色情圖片亦引發了許多問題。<ref>{{cite web |url=http://news.cnet.com/2100-1023-269290.html |title=Porn sneaks past search filters |date=2001-07-02 |author=Paul Festa |publisher=CNET News |accessdate=2013-02-03 }}</ref>

2009年11月11日，Google引入了安全搜索鎖定功能，<ref>{{cite web |url=http://googleblog.blogspot.co.uk/2009/11/locking-safesearch.html |title=Locking SafeSearch |date=2009-11-11 |author=Pete Lidwell |publisher=Google Official Blog |accessdate=2013-02-03 }}</ref> 該功能能讓持有Google賬戶的使用者鎖定在安全搜索中的高強度模式。一旦開啟這項功能，用戶在登出Google賬戶之後，搜索過濾強度不會發生任何變化。而另一項功能KidzSearch則讓安全搜索總是開啟。這項功能保證兒童在使用Google進行搜索時，搜索結果會被嚴格過濾。<ref>{{Cite news|url=http://www.telegraph.co.uk/technology/11794464/Cyber-safety-How-protected-are-your-children-online.html|title=Cyber safety: How protected are your children online?|newspaper=Telegraph.co.uk|access-date=2017-02-02}}</ref>2012年12月12日，Google移除了完全關閉安全搜索的功能。<ref>{{cite web |url=http://news.cnet.com/8301-1023_3-57558795-93/google-tweaks-image-search-to-make-porn-harder-to-find/ |title=Google tweaks image search to make porn harder to find |date=2012-12-12 |author=Casey Newton |publisher=CNET News |accessdate=2013-02-03 }}</ref><ref>{{cite web |url=http://thenextweb.com/google/2012/12/12/google-changes-search-image-safesearch-explicit/ |title=Google tweaks image search algorithm and SafeSearch option to show less explicit content |date=2012-12-12 |author=Matthew Panzarino |publisher=TNW |accessdate=2013-02-03 }}</ref><ref>{{cite web |url=http://www.webpronews.com/google-preventing-u-s-users-from-disabling-safesearch-2012-12 |title=Google No Longer Allows You to Disable SafeSearch, and That Makes Google Search Worse |date=2012-12-16 |author=Josh Wolford |publisher=Web Pro News |accessdate=2013-02-03 }}</ref>
