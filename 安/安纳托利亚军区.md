'''安纳托利亚军区'''（{{lang-el|{{lang|grc|Άνατολικόν θέμα, θέμα Άνατολικῶν}}}}）,或者叫做'''安纳托利孔军区'''，为[[拜占庭帝国|拜占庭帝国]]7世纪在[[安纳托利亚|安纳托利亚]]中部的一个[[军区制|军区]]。其自从被建立的那一刻起，就是帝国内最大且最重要的军区，长官通常都是人中豪杰，在帝国的漫长历史里数次起兵反叛登上皇位，在长达三个世纪的对阿拉伯人的战争中扮演着中流砥柱的角色，在曼齐刻尔特之前此处长期保持和平，与边陲要冲区不同，极少经敌军蹂躏。

== 地理及行政管理 ==
在八到九世纪时期，最早的安纳托利亚军区地跨传统的古地区例如{{tsl|en|Lycaonia|利考尼亚}}、[[皮西迪亚|皮西迪亚]]、[[伊蘇利亞|伊蘇利亞]]以及[[弗里吉亚|弗里吉亚]]大部和[[加拉塔|加拉塔]]西部<ref>{{harvnb|Haldon|1997|p=157}}.</ref><ref name="Pertusi">{{harvnb|Pertusi|1952|pp=114–115}}.</ref>，之后在720年左右剥离出[[色雷斯西亚军区|色雷斯西亚]]和[[基比拉奥特军区|基比拉奥特]]军区<ref name="DOAKS">{{harvnb|Nesbitt|Oikonomides|1996|p=144}}.</ref><ref name="ODB90">{{harvnb|Kazhdan|1991|p=90}}.</ref>。在[[狄奥斐卢斯|狄奥斐卢斯]]统治时期，其东部和东南直面阿拉伯人（当时[[奇里乞亚|奇里乞亚]]仍在阿拔斯手中）的地区，包括奇里乞亚山口北部的堡垒卫戍区，从原军区中剥离出来，被设立成两个新的[[要冲区|要冲区]]（之后形成{{tsl|en|Cappadocia (theme)|卡帕多西亚军区}}和[[塞琉西亚军区|塞琉西亚军区]]）<ref>{{harvnb|Haldon|1999|p=114}}.</ref><ref name="EHW5.2">{{harvnb|Gyftopoulou|2003|loc=[http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=%20&kefalaia_State=#chapter_7 Chapter 5.2] {{Cite web |url=http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=&kefalaia_State= |title=Archived copy |access-date=2012-01-27 |archive-url=https://web.archive.org/web/20130511174446/http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=&kefalaia_State= |archive-date=2013-05-11}}}}</ref>。[[利奥六世_(拜占庭)|利奥六世]]之后又将图兹湖的西岸划给卡帕多西亚<ref name="ODB90"/><ref name="EHW5.2"/>，军区首府在838年在其失陷于[[阿拔斯王朝|阿拔斯]]之前一直在{{tsl|en|Amorium|阿莫里乌姆}}<ref name="ODB90"/><ref>{{Harvnb|Pertusi|1952|p=115}}.</ref>。在此之后大概迁至{{tsl|en|Polybotos|波利波图斯}}（今土耳其[[博爾瓦丁|博爾瓦丁]]）的堡垒附近地区<ref name="EHW5.1">{{harvnb|Gyftopoulou|2003|loc=[http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=%20&kefalaia_State=#chapter_6 Chapter 5.1] {{Cite web |url=http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=&kefalaia_State= |title=Archived copy |access-date=2012-01-27 |archive-url=https://web.archive.org/web/20130511174446/http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=&kefalaia_State= |archive-date=2013-05-11}}}}.</ref>。

据10世纪[[阿拉伯人|阿拉伯]]地理学家记载，安纳托利亚军区为“罗马最大的省份”。下辖有15,000名士兵，共有34处要塞<ref name="Pertusi"/>。其[[將軍_(古希臘)|将军]]职位首次是在690年被证实。在所有将军里位列第一级。安纳托利亚将军是帝国最高职位之一，年薪为40金镑。但其也是少数禁止宦官担任的职位。除此之外，还被指派为monostrategos，掌管亚洲各个军区<ref name="EHW3"/><ref name="ODB">{{harvnb|Kazhdan|1991|pp=89–90}}.</ref><ref>{{harvnb|Bury|1911|pp=39–41}}.</ref>。

== 历史 ==
军区的详细设立日期仍然是谜，但与其他几个的初始设立的军区相似，安纳托利亚军区也是因{{tsl|en|Battle of Yarmouk|雅尔穆克战役}}之后，撤退的[[羅馬陸軍|东罗马野战军]]的残余部队驻扎在此而设立。安纳托利亚军区也正由过去三大野战军（阿非利加，意大利和亚美尼亚）之一的东方野战军（希腊语：{{lang|grc|Άνατολή}}，Anatolē，即“东方”）而命名<ref name="Pertusi114">{{harvnb|Pertusi|1952|p=114}}.</ref><ref>{{harvnb|Haldon|1999|p=73}}; {{harvnb|Treadgold|1995|p=23}}.</ref><ref name="EHW1">{{harvnb|Gyftopoulou|2003|loc=[http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=%20&kefalaia_State=#chapter_1 Chapter 1] {{Cite web |url=http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=&kefalaia_State= |title=Archived copy |access-date=2012-01-27 |archive-url=https://web.archive.org/web/20130511174446/http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=&kefalaia_State= |archive-date=2013-05-11}}}}.</ref>。在669年，该军区被第一次证实存在，与当地的军队一起被提及（Exercitus Orientalis，即东方野战军的拉丁语名字）。而当地所驻扎的部队之后在687年演化成[[查士丁尼二世|查士丁尼二世]]的御林军<ref name="Pertusi114"/><ref name="EHW1"/><ref name="ODB89">{{harvnb|Kazhdan|1991|p=89}}.</ref>。

=== 与阿拉伯人和突厥人的战争 ===
在7到8世纪的与阿拉伯人的长期拉锯战中，安纳托利亚军区，尤其是最东边的[[卡帕多西亚|卡帕多西亚]]经常成为阿拉伯人入侵的主要目标，而在8世纪中期开始卡帕多西亚则成为抗击阿拉伯人劫掠的最前线<ref name="EHW6.1">{{harvnb|Gyftopoulou|2003|loc=[http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=%20&kefalaia_State=#chapter_9 Chapter 6.1] {{Cite web |url=http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=&kefalaia_State= |title=Archived copy |access-date=2012-01-27 |archive-url=https://web.archive.org/web/20130511174446/http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=&kefalaia_State= |archive-date=2013-05-11}}}}.</ref>。

阿莫里乌姆作为军区的首府也频繁成为阿拉伯人打击的目标。早期644年就遭受到攻击，646年失陷而又在669年收复。阿拉伯人之后在708年再次打到该处，并且尝试驶向[[君士坦丁堡|君士坦丁堡]]进行{{tsl|en|Siege of Constantinople (717–718)|君士坦丁堡之围 (717-718年)|第二次围攻}}，然而经过8年的围攻后阿拉伯人仍是无功而返<ref name="EHW6.1"/><ref name="ODB79">{{harvnb|Kazhdan|1991|p=79}}.</ref>。阿拉伯人的攻势持续了二十余年，直到40年代，[[倭马亚|倭马亚]]在740年的{{tsl|en|Battle of Akroinon|阿柯洛伊努战役}}决定性输给了东罗马帝国，同时内部又经历了第三次内战，以及来自阿拔斯家族的叛乱之后，外侵的势头才逐渐削弱<ref>{{harvnb|Lilie|1976|pp=97–162}}</ref>。与此同时原先阿拉伯人控制的某些区域，也在[[利奥三世_(拜占庭)|利奥三世]]的统治时期被并入安纳托利亚军区，这同时激怒了改朝换代不久的阿拔斯哈里发国。25年后的780年阿拔斯再次开始侵入安纳托利亚，788年在{{tsl|en|Battle of Kopidnadon|柯皮那顿战役}}中击败拜占庭的军队，797年又再次进攻至阿莫里乌姆，当地遭受极为严重的破坏<ref name="EHW6.1"/>。9世纪早期，卡帕多西亚成了阿拔斯进攻的核心区域，806年由哈里发[[哈伦·拉希德|哈伦·拉希德]]发起的大征服使之达到了高潮，阿拔斯占领了{{tsl|en|Heraclea Cybistra|赫拉克利亚-基比斯特拉|赫拉克利亚-基比斯特拉城}}和附近的一系列堡垒<ref name="EHW6.2">{{harvnb|Gyftopoulou|2003|loc=[http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=%20&kefalaia_State=#chapter_20 Chapter 6.2] {{Cite web |url=http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=&kefalaia_State= |title=Archived copy |access-date=2012-01-27 |archive-url=https://web.archive.org/web/20130511174446/http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=&kefalaia_State= |archive-date=2013-05-11 }}}}.</ref>。

伴随着阿拉伯人带来的战火毁灭了这片地区的城市，社会构造因此崩坏，出现了去城市化的现象。在军区内侧的弗里吉亚和皮西迪亚绝大多数的城市勉强幸存，但是规模急剧缩水；而东部地区靠近阿拉伯人的地区，类似于卡帕多西亚的东部城市，则如同皮西迪亚的[[安条克|安条克城]]一样几乎沦为废墟<ref name="EHW5.3">{{harvnb|Gyftopoulou|2003|loc=[http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=%20&kefalaia_State=#chapter_8 Chapter 5.3] {{Cite web |url=http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=&kefalaia_State= |title=Archived copy |access-date=2012-01-27 |archive-url=https://web.archive.org/web/20130511174446/http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=&kefalaia_State= |archive-date=2013-05-11}}}}.</ref>。于是东部地区，尤其是几乎毁灭了的卡帕多西亚地区，在9世纪变成了帝国的东部要冲区，这意味着阿拉伯人的攻势到此为止，几乎无法再侵入到帝国的安纳托利亚军区。

然而在838年，哈里发[[穆阿台绥姆|穆阿台绥姆]]大举进攻，打穿了帝国的防线侵入到安纳托利亚的内部，直到878年在密斯忒亚（今土耳其[[貝伊謝希爾|貝伊謝希爾]]）被军区防御部队打退。之后的888年，894年，897年的一系列两国间持续拉锯战都集中在该军区的东南部{{tsl|en|Iconium|以哥念}}（今土耳其[[科尼亞_(土耳其)|科尼亚]]）附近<ref name="EHW6.2"/>。但到了十世纪，除了931年{{tsl|en|Thamal al-Dulafi|塔玛尔·阿尔·杜拉菲}}和963年{{tsl|en|Sayf al-Dawla|赛义夫·阿尔·道莱}}针对以哥念的劫掠以外，军区内部长期保持和平<ref name="EHW6.2"/>。

1069年[[突厥人|突厥人]]建立的[[塞尔柱帝国|塞尔柱帝国]]开始对帝国展开第一次攻势，突厥人打到了以哥念。而在[[曼齐刻尔特战役|曼齐刻尔特战役]]后突厥人占领了安纳托利亚大部，以哥念便成为[[罗姆苏丹国|罗姆苏丹国]]的首都所在地<ref name="EHW7">{{harvnb|Gyftopoulou|2003|loc=[http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=%20&kefalaia_State=#chapter_13 Chapter 7] {{Cite web |url=http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=&kefalaia_State= |title=Archived copy |access-date=2012-01-27 |archive-url=https://web.archive.org/web/20130511174446/http://asiaminor.ehw.gr/forms/fLemmaBodyExtended.aspx?lemmaid=3587&boithimata_State=&kefalaia_State= |archive-date=2013-05-11  }}}}.</ref>。安纳托利亚最后于1077年被提及，当时的将军即为后来的皇帝[[尼基弗鲁斯三世|尼基弗鲁斯三世]]<ref name="ODB90"/>。[[科穆宁王朝|科穆宁王朝]]时期的皇帝们成功收复该军区西部及北部的领土，但军区从未被重建<ref name="EHW7"/>。
