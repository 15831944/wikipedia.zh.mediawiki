'''安兔兔'''是一款中國的基準測試工具軟件，通常用於手機的基準測試。

== 開發 ==
該公司的軟件開發位于[[朝阳区_(北京市)|北京朝陽區]]<ref>{{cite web|title=關於我們|url=http://www.antutu.com/about.htm|website=安兔兔的官方網站|accessdate=2016年2月20日|quote=地址：北京朝阳区朝阳北路237号复星国际中心20层}}</ref>，由中國企業家邵英<ref>{{cite web|author1=卓克|title=重新定义跑分意义 安兔兔6.0版本发布|url=http://www.pcpop.com/doc/1/1230/1230943.shtml|website=PCPOP.com|accessdate=2016年2月29日|date=2015年11月18日|quote=安兔兔的创始人邵英介绍：在之前版本的积累中，这款软件已经在200多个国家使用，总用户超过2亿...}}</ref>和梁斌<ref>{{cite web|author1=Hobo C|title=从手机跑分到数据服务，专访安兔兔创始人梁斌|url=http://www.pcdigest.com/2014/05/antutu/|website=PCDigest.com|accessdate=2014年5月5日|date=2016年2月29日|deadurl=yes|archiveurl=https://web.archive.org/web/20160305171512/http://www.pcdigest.com/2014/05/antutu/|archivedate=2016年3月5日}}</ref>共同創辦。

== 挑戰 ==
安兔兔基準測試非常流行，有些手機硬件製造商採取手段使基準测试變得不可靠。<ref>{{cite web|url=http://appleinsider.com/articles/13/10/03/rampant-cheating-by-android-knockoffs-prompts-antutu-to-launch-new-benchmark|title=Android模仿作弊猖獗導致安兔兔推出新的基準測試|date=2013年10月3日|work=AppleInsider}}</ref><ref name="antutu.com">{{cite web|url=http://www.antutu.com/view.shtml?id=7242|title=反對作弊基準AnTuTu發布AnTuTu X基準|publisher=}}</ref>為了應對這種欺詐行為，安兔兔創建了一個名為安兔兔X的新基準，這使得製造商更難以作弊。<ref name="antutu.com"/><ref>{{cite web|url=https://play.google.com/store/apps/details?id%3Dcom.emobile.flashlight%26hl%3Den |title=存檔副本 |accessdate=2014-11-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20141224142503/https://play.google.com/store/apps/details?id=com.emobile.flashlight&hl=en |archivedate=2014-12-24 |df= }}</ref><ref>{{cite web|url=http://phandroid.com/2013/10/07/antutu-x-benchmark/|title=AnTuTu X基準防止製造商作弊|author=Chris Chavez|date=2013年10月7日|work=Phandroid - Android新聞和評論}}</ref>
