'''安房國分寺 '''({{jpn|j='''安房国分寺'''|kk=あわこくぶんじ}})，是位於[[千葉縣|千葉縣]][[館山市|館山市]]國分的[[真言宗智山派|真言宗智山派]]的佛寺。[[山號|山號]]是日色山，[[本尊|本尊]]為[[藥師佛|藥師佛]]。

[[奈良時代|奈良時代]][[圣武天皇|聖武天皇]]下詔在[[日本|日本]]各地建立[[國分寺|國分寺]]，據說是[[安房國|安房國]]國分僧寺的後續繼承之寺院。

== 概要 ==
位在館山市市中心的东邊，距離海岸3公里的沙丘上（海拔19公尺）。{{Sfn|安房国分寺跡（平凡社）|1996年}}繼承了安房國分僧寺和傳承，除了現在寺內和舊國分僧寺遺跡重覆的部分外，推測北方約900公尺處是舊國分尼寺的位置。但是安房國成立的情况特別，因为和其他國家的國分寺不同，是稍晚於下令建立國分寺的詔令頒布所創建的。

被推定為舊國分寺遺跡的區域，在[[1957年|1957年]]（[[昭和|昭和]] 32年）被指定為館山市歷史遺跡， 在[[1992年|1992年]] （[[平成|平成]] 4年）被指定為千葉縣的歷史遺跡。 该区域從[[1976年|1976年]] （[[昭和|昭和]]51年）起進行挖掘調查，發現了[[大雄寶殿|金堂]]，並看的到基壇，并已經檢測到劃分寺廟區域的凹槽。此外，國分寺所處的沙丘，也已知道有從[[繩文時代|繩文時代]]到[[日本历史|中世紀]]遺址的分佈。 {{Sfn|千葉県の歴史 資料編 考古3|1998年|pp=16-19}}{{Sfn|第6次調査報告書|2011年}}

</div>一般而言，全國的國分寺，是在[[天平_(聖武天皇)|天平]]13年（[[741年|741年]]）下詔建立國分寺的時候所創建的成立。 但安房國的背景，在[[養老_(日本年號)|養老]]2年（[[718年|718年]]）從[[上總國|上總國]]分出成立，在天平13年12月又與上总國重新合并，並在[[天平勝寶|天平勝寶]]9年（[[757年|757年]]）5月又再次重新分出獨立。可看到安房國國分寺是比其他國家較為晚期建立。{{Sfn|安房国分寺跡（平凡社）|1996年}}{{Sfn|千葉県の歴史 資料編 考古3|1998年|pp=16-19}}

和[[安房國|安房國]]同樣歷經並合分立的[[能登國|能登國]]、[[和泉國|和泉國]]，前者在[[承和|承和]]10年（[[843年|843年]]），將定額寺（[[奈良時代|奈良]]到[[平安時代|平安時代]]的一種官寺名稱）大興寺充當做為能登國分寺；后者在承和6年（[[839年|839年]]）將泉郡安樂寺充當作為和泉國分寺。 {{Sfn|安房国分寺跡（平凡社）|1996年}}關於安房國的情況，雖然看不到安房國關於安房國分寺設置的紀錄，有一種理論認為應該和能登國、和泉國一樣，是以既有的定額寺充作國分寺。{{Sfn|安房国分寺跡（平凡社）|1996年}}

而且關於日色山國分寺（即現在的國分寺）的來歷，傳說是在詔令之前的神龜4年（[[727年|727年]]），順從[[國司|國司]]指令下鄉到安房的史生鹿屋脛代，將[[行基|行基]]製作的藥師佛作為本尊建立了寺廟。{{Sfn|安房国分寺跡（平凡社）|1996年}}{{Sfn|第6次調査報告書|2011年}}但本身即不清楚现在國分寺和舊國分寺的关系，也並不清楚其傳承 。{{Sfn|安房国分寺跡（平凡社）|1996年}}

=== 古代 ===
在《類聚三代格》中，有[[仁和_(光孝天皇)|仁和]]2年（[[886年|886年]]）6月2日的太政官符（一種公文書），可看到「彼國定國分寺、已置十僧」，安房國分寺被認為在時期即存在（史料上初見）。{{Sfn|安房国分寺跡（平凡社）|1996年}}{{Sfn|千葉県の歴史 資料編 考古3|1998年|pp=16-19}}

[[延長_(醍醐天皇)|延长]]5年（[[927年|927年]]）設置的「[[延喜式|延喜式]]」有關於主稅的规定，和其他國家不同，沒有安房國的國分寺费的記載。{{Sfn|千葉県の歴史 資料編 考古3|1998年|pp=16-19}}关于这一点，有一種推測是從上总國分配經費的，但並不明确。{{Sfn|千葉県の歴史 資料編 考古3|1998年|pp=16-19}}

=== 中世・近世 ===
[[中世纪|中世纪]]时期的變遷不詳。{{Sfn|中世諸国一宮制|2000年|p=203}}舊國分寺等的後續也是不明确， [[慶安_(後光明天皇)|慶安]]2年（[[1369年|1369年]]）安房國分寺的僧侶抄錄[[大般若经|大般若经]]，{{Sfn|第6次調査報告書|2011年}}，在現在國分寺內，也發現了從[[南北朝时代_(日本)|南北朝時代]]左右流傳至今的五輪塔。{{Sfn|安房国分寺跡（平凡社）|1996年}}

[[江户时代|江戸時代]]的[[享保|享保]]20年（[[1735年|1735年]]）重建了本堂。{{Sfn|中世諸国一宮制|2000年|p=203}}

=== 近代以後 ===
* [[1932年|1932年]]和[[1934年|1934年]]（[[昭和|昭和]]7年和9年），平野元三郎、滝口宏進行調查。發掘出大量的瓦。{{Sfn|第6次調査報告書|2011年}}
* [[1957年|1957年]]（昭和32年）12月16日，「安房國分寺跡」被指定為館山市史蹟。
* [[1976年|1976]]-[[1978年|1978年]]（昭和51-53年），進行第1次-第3次發掘調査（早稲田大学考古学研究室等）。發掘出金堂基壇、寺域區畫溝（推定）等等。{{Sfn|第6次調査報告書|2011年}}
* [[1992年|1992年]]（[[平成|平成]]4年）2月28日，「安房國分寺跡」被指定為千葉縣指定史跡。
* [[1994年|1994年]]（平成6年），第4次發掘調査。伴隨墓地建設工程的調査，挖掘出寺域區畫溝（推定）等。{{Sfn|第6次調査報告書|2011年}}
* [[1998年|1998年]]（平成10年），第5次發掘調査。伴隨民房建設的確認調査發掘寺域區畫溝（推定）。{{Sfn|第6次調査報告書|2011年}}
* [[2010年|2010年]]（平成22年）、第6次發掘調査。溝的確認調査。{{Sfn|第6次調査報告書|2011年}}

== 安房國分寺遺址 ==
[[File:Awa_Kokubunji_(Tateyama),_kondo-ato.JPG|右]]
僧寺遺址是和現在國分寺境内重複的地方（[//tools.wmflabs.org/geohack/geohack.php?language=ja&pagename=%E5%AE%89%E6%88%BF%E5%9B%BD%E5%88%86%E5%AF%BA&params=34_59_37.37_N_139_53_21.74_E_region:JP-12_type:landmark&title=%E5%AE%89%E6%88%BF%E5%9B%BD%E5%88%86%E5%AF%BA%E8%B7%A1 <span title="この位置の地図や航空写真などをリンクするページを表示します">位置</span>]）。在此之前實行了數次的發掘調查，並不認為有下總國分寺跡、上總國分寺跡的複數寺院的伽藍配置。從金堂遺址可看到被公認只有一個建築。{{Sfn|千葉県の歴史 資料編 考古3|1998年|pp=16-19}}被推定的金堂遺跡，推測其夯土基壇的東西向長度為公尺、南北向長度為15公尺。{{Sfn|千葉県の歴史 資料編 考古3|1998年|pp=16-19}}除此之外在寺院的東邊和西邊可看到寺域畫線用的溝。{{Sfn|第6次調査報告書|2011年}}

此外，從僧寺遺跡挖掘出土大量的瓦片。關於瓦的樣貌様是以「素緣七葉素弁蓮花紋」的紋飾和製作技法被認為有強烈的在地色彩。{{Sfn|千葉県の歴史 資料編 考古3|1998年|pp=16-19}}其他的出土品有三彩獸腳（是放在金堂本尊前的香爐腳），在日本也被視為罕見。{{Sfn|千葉県の歴史 資料編 考古3|1998年|pp=16-19}}現在國分寺境内也發現被認為是舊國分寺遺跡的4個礎石。
[[File:Awa_Kokubunji_(Tateyama),_soseki.JPG|右]]


== 安房國分尼寺遺址 ==
關於尼寺遺跡位址並不明確。在北方約900公尺的萱野地区殘留有「アマンボウ」（發音：amanbou）的地名，推測可能是被稱為「尼坊」的國分尼寺位置。{{Sfn|中世諸国一宮制|2000年|p=203}}<ref name="説明板">境内説明板。</ref>

除此之外，在[[南房總市|南房總市]]僧間的僧間廢寺（有出土布目瓦）中殘有「尼堂」的流傳，在南房總市府中的寶樹院也留有尼寺等的相關流傳，但仍不清楚其關係。{{Sfn|中世諸国一宮制|2000年|p=203}}
