'''安通車站'''位於[[台灣|台灣]][[花蓮縣|花蓮縣]][[富里鄉|富里鄉]]，曾為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]，於2007年3月30日裁撤。

== 車站構造 ==
*島式月臺一座。

== 利用狀況 ==
*廢止前為招呼站。
*[[玉里車站|玉里站]]＝[[東里車站|東里站]]間截彎取直後裁撤。
*目前是[[玉里鎮|玉里鎮]]環鎮自行車道的起迄點。

== 車站周邊 ==
*安通溫泉
*[[秀姑巒溪|秀姑巒溪]]
*[[樂樂溪|樂樂溪]]
*[[台9線|台9線]]
*[[台30線|台30線]]

== 歷史 ==
*1924年4月27日：設站。
*1986年4月24日：降等為招呼站。
*2007年3月30日：玉里-東里截彎取直完成後本站裁撤，走入歷史。
