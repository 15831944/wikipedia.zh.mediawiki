'''安远县'''在[[中国|中国]][[江西省|江西省]]南部、[[贡水|贡水]]支流濂江流贯，是[[赣州市|赣州市]]所辖的一个县。

== 历史沿革 ==
安远县始建于[[南朝|南朝]][[南朝梁|梁]]。[[大同_(南梁)|大同]]十年（公元 544 年）析[[雩都县|雩都县]]置安远县，因境内「安远水」（今濂江）得名。

[[隋|隋]][[开皇|开皇]]年间并入雩都县，属[[虔州|虔州]]（[[大业|大业]]三（607）年改称[[南康郡|南康郡]]，唐又改虔州）管辖。

[[唐|唐]][[贞元|贞元]]四（788）年，划雩都县南部三乡及[[信丰县|信丰县]]一里重建安远县。

唐[[天佑|天佑]]十（910）年，[[十国|十国]]之[[杨吴|吴]]合并虔州及[[韶州|韶州]]称百胜军，县属百胜军。[[南唐|南唐]][[升元|升元]]元（937）年改百胜军为昭信军。

[[宋朝|宋]][[开宝|开宝]]八（975）年，改昭信军为军州；[[太平兴国|太平兴国]]元（976）年复名虔州。[[绍兴_(南宋)|绍兴]]二十三（1153）年，改虔州为[[赣州|赣州]]。

[[元|元]]代实行行省制，改赣州为赣州路。[[至元|至元]]十四（1277）年，安远县并入[[会昌县|会昌县]]。[[至大|至大]]三（1310）年又析会昌县置安远县。

[[明|明]][[洪武|洪武]]二（1369）年改赣州路为赣州府。

[[民国纪年|中华民国]]元（1912）年，[[北洋政府|北洋政府]]废除府制，安远县直属[[江西省|江西省]]；次年实行道制，县属[[赣南道|赣南道]]；民国 15（1926）年废除道制，再次直属江西省管辖。
民国 19（1930）年 6 月，[[中国共产党|中国共产党]]设立赣西南苏维埃政府赣南革命委员会，安远县归其管辖；11 月初，赣西南苏维埃政府赣南革命委员会改称江西省苏维埃政府赣南办事处。次（1931）年[[中华苏维埃共和国|中华苏维埃共和国]]成立，县隶江西省。民国 22（1933）年 8 月改隶粤赣省。
为应对第一次国共内战，[[南京国民政府|南京国民政府]]自民国 21（1932）年开始实行[[行政督察区|行政督察区]]制度。民国 23（1934）年，县隶江西省第九行政督察区；民国 24（1935）年 4 月以后，改隶第四行政督察区。

1949 年 8 月，[[中国人民解放军|中国人民解放军]]占领安远县。20 日起，县隶赣州分区；11 月起隶属赣西南行政区；1951 年起隶属赣州专区；1954 年改赣州专区为赣南行政区；1964 年复名赣州专区；1971 年改为赣州地区；1999 年赣州撤地建市，安远县隶赣州市。

== 行政区划 ==
安远县下辖8个镇10个乡。

== 地理 ==
[[File:Mount_Sanbai,_source_of_the_Dongjiang_River.jpg|thumb]]
安远县地处[[南岭|南岭]]山脉的延续地带，属中低山与丘陵区。面积2375平方公里，其中山地面积297万亩，占土地总面积的83.43%森林覆盖率达83.4%。<ref>{{cite web |url=http://www.ay.gov.cn/aygk/xqjs/200512/t20051202_5995.htm |title=存档副本 |accessdate=2009-09-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20100929183252/http://www.ay.gov.cn/aygk/xqjs/200512/t20051202_5995.htm |archivedate=2010-09-29 }}</ref>

安远县属[[中亚热带季风气候|中亚热带季风气候]]。

== 文化教育 ==
至2004年底，全县拥有幼儿园21所，小学212所，普通中学23所，职业学校1所。<ref>http://www.ay.gov.cn/aygk/dlwz/200811/t20081113_7387.htm{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
安远县是中国采茶戏的发源地。
