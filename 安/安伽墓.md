'''安伽墓'''是一座位於[[西安|西安]][[未央區|未央區]]大明宮鄉炕底寨村的[[北周|北周]][[粟特人|粟特人]]古墓葬，於2000年5月至7月間發掘出土，<ref>{{cite web |url=http://www.sxhm.com/index.php?ac=article&at=read&did=10699 |title=安伽墓石榻 |author=<!--Staff writer(s); no by-line.--> |date=2016年12月9日 |website=www.sxhm.com |access-date=2018年7月31日}}</ref>同北周[[史君墓|史君墓]]相距約2.5千米。

== 墓主簡介 ==
[[File:Rubbing_of_An_Chia’s_Epitaph.jpg|thumb]]
根據墓誌記載，墓主安伽，字大伽，來自姑藏[[昌松|昌松]]（位於甘肅[[武威市|武威市]]，曾是[[絲綢之路|絲綢之路]]要衝，北朝時期聚居大量中亞、西亞人），從他的姓氏來看，應是源自中亞[[安國_(昭武九姓)|安國]]的粟特人。<ref name=西域>{{cite book |editor=鄭培凱 |date=2009 |title=《西域：中外文明交流的中轉站》 |url=https://books.google.com/books?id=ahsfAwAAQBAJ&pg=PA85&lpg=PA85&dq=%E5%AE%89%E4%BC%BD%E5%A2%93&source=bl&ots=-NPd_BkHYr&sig=-L8E68S4-ruLTvFcinr308YWPvg&hl=en&sa=X&ved=2ahUKEwj-gNKGoMjcAhWK4YUKHbMOCvU4ChDoATAAegQIABAB#v=onepage&q=%E5%AE%89%E4%BC%BD%E5%A2%93&f=false |location=香港 |publisher=香港城市大學 |pages=82, 85 |isbn=978-962-937-166-1 |accessdate=2018年7月31日}}</ref>曾任北周大都督、[[同州|同州]]薩保。其父安突建曾任「冠軍將軍」、[[眉州|眉州]]刺史，母親杜氏為昌松縣君，也就是[[涼州|涼州]]漢人。安伽於[[大象_(年號)|大象]]元年（公元579年）五月逝世，終年62歲。

== 墓葬簡介 ==
[[File:Fire_Altar_Scene_from_the_Tomb_of_An_Chia_1.jpg|thumb]]火壇浮雕線摹圖]]
安伽墓由斜坡墓道、五個天井、五個過洞、磚砌甬道以及墓室組成，石製墓門門額有彩繪雕刻的[[祆教|祆教]]{{Link-en|祆教聖火|Atar|火壇}}和半人半鳥祭司圖像。墓室內發現一具圍屏石榻，由11塊石板構成，共雕繪56幅圖案，全部彩繪貼金，內容有出行、狩獵、樂舞、宴飲等，其中人物多為[[胡人|胡人]]，着胡服，作胡舞，充滿異域風情。<ref>{{cite web |url=http://mall.cnki.net/magazine/Article/WENW200101000.htm |title=西安發現的北周安伽墓 |author1=尹申平 |author2=邢福來 |author3=李明 |date=2001 |website=mall.cnki.net |access-date=2018年7月31日}}</ref>這些圖案的組成和佈局都有明顯的對稱性，特別是左右兩側圖案的對稱性更強。<ref>{{cite web |url=http://www.nxkg.org.cn/index.php?m=content&c=index&a=show&catid=15&id=296 |title=北朝至隋初入華粟特貴族墓隨葬用圍屏石榻研究 |author=邢福來 |date=2017年12月4日 |website=www.nxkg.org.cn |access-date=2018年7月31日}}</ref>

粟特人大多信仰[[祆教|祆教]]，通常以火燒的方式處理遺體。安伽的遺骨就經過火燒，而且放置在墓門之外。同[[隋朝|隋朝]]的[[虞弘墓|虞弘墓]]相比，安伽墓只有一具圍屏石榻，虞弘墓只有石槨，兩者都沒有棺，這與漢人葬俗大不相同。<ref name=西域 />與同為北周時期的另外兩座粟特人墓葬——[[康業墓|康業墓]]及[[史君墓|史君墓]]相比，安伽墓的漢化程度高於史君墓，但低於康業墓。<ref>{{cite book |editor=吳天泰 |date=2015 |title=《族群與社會》 |url=https://books.google.com/books?id=bsYgpuud5gMC&pg=PA290&lpg=PA290&dq=%E5%BA%B7%E6%A5%AD%E5%A2%93+%E6%BC%A2%E5%8C%96&source=bl&ots=x6FACl0aD-&sig=243EbRyh7rP4XmmNZVN8EFhd8Yw&hl=en&sa=X&ved=0ahUKEwjIr-KbxbHcAhXOIlAKHQQyBGMQ6AEIJzAA#v=onepage&q=%E5%BA%B7%E6%A5%AD%E5%A2%93%20%E6%BC%A2%E5%8C%96&f=false |location=臺北 |publisher=五南圖書出版 |page=290 |isbn=9789571141756 |accessdate=2018年7月31日}}</ref>
