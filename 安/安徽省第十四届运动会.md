'''安徽省第十四届运动会'''，简称'''第十四届省运会'''或'''蚌埠省运会'''，于2018年在[[蚌埠市|蚌埠市]]举办，这也是继1986年后，蚌埠市第二次举办[[安徽省运动会|省运会]]。安徽省第十四届运动会于2018年10月16日晚举办开幕式，但比赛项目自当年3月就已陆续展开，包括[[安徽省|安徽省]]16个地级市及98所高校组成的共114个代表团参与了本届省运会，参赛运动员约2.6万人。

==申办过程==
2014年，安徽省体育局印发《安徽省运动会申办办法(试行)》，标志着省运会开始采取申办制，有意愿承办省运会的城市，可向省政府提交申办请示。随后，阜阳、淮北、蚌埠三市申请承办第十四届省运会。2014年9月12日，蚌埠在申办候选城市陈述报告会中胜出，获得安徽省第十四届运动会的举办权<ref>{{cite web |title=揭秘 走进安徽省第十四届省运会举办城市——“珠城”蚌埠 |url=https://www.sohu.com/a/203383243_500723 |publisher=搜狐 |accessdate=2018-07-24}}</ref>。

==赛事文化==
2018年3月22日，安徽省第十四届运动会的会徽、会歌、吉祥物、主题口号公布，主题口号为“运动安徽 奔跑蚌埠”<ref>{{cite web |author1=姚庆林 |title=安徽省第十四届运动会会徽、会歌、吉祥物、主题口号发布 |url=http://ah.people.com.cn/n2/2018/0322/c358266-31373598.html |publisher=人民网-安徽频道 |accessdate=2018-07-24 |date=2018-03-22}}</ref>。
===会徽===
会徽以运动员为主体，其中头部为珍珠，象征蚌埠市被誉为珠城，身体部分以“十四”的造型，象征本届省运会的届数。
===吉祥物===
[[File:14th_Anhui_Games_mascot.jpg|缩略图]]
吉祥物是两位花鼓灯形象的福娃，分别为“蚌蚌”和“珠珠”。[[花鼓灯|花鼓灯]]是蚌埠市的独特文化，被列为国家级非物质文化遗产，吉祥物将体育形象与地域文化相结合，同时佩挂珍珠项链，进一步增强吉祥物的蚌埠特色。

===主题曲===
本届省运会的主题曲为《相约珠城》，由刘爱斌作词、陈雄作曲。

==开闭幕式==
本届省运会开幕式于2018年10月16日在[[蚌埠体育中心|蚌埠体育中心]]主体育场举行，由暖场、仪式和文体展演三部分组成，邓锐斌担任总导演。其中暖场部分包括龙狮欢腾、天地人和、石榴花开、青春梦想、鼓舞飞扬等五部分，主要展示蚌埠当地的传统文化，展现“禹风厚德 孕沙成珠”的城市精神。仪式部分包括嘉宾、裁判员、运动员入场，升旗、领导致辞等。文体展演部分由《序》、《文明之源 家国故里》、《诗意山水 大美安徽》、《运动健康 活力飞扬》、《新时代 新未来》等五大篇章组成，这部分不仅展现蚌埠当地特色，也对安徽全省的自然、文化、现代经济和体育发展进行演绎<ref>{{cite web |author1=刘晨,苗成韬 |title=省运会开幕式紧锣密鼓筹备中 |url=http://www.bbnews.cn/zt/ahsdssjydh/sydt/586288.shtml |publisher=蚌埠新闻网 |accessdate=2018-10-05 |date=2018-10-15}}</ref>。中国首枚[[奥运金牌|奥运金牌]]获得者[[许海峰|许海峰]]、体操运动员[[邓琳琳|邓琳琳]]、[[世界击剑锦标赛|世界击剑锦标赛]]冠军张亮亮、武术散打世界杯女子冠军蒋先婷等运动员，以及蚌埠籍著名演员[[蒋雯丽|蒋雯丽]]等人出席了开幕式。

本届省运会闭幕式于2018年10月24日在蚌埠体育中心体育馆举行，闭幕式对获奖代表团进行了表彰，并将省运会会旗传递给[[滁州市|滁州市]]市长许继伟，滁州是下届省运会举办城市。最后，闭幕式上举行了名为《“淮”节拍 青春“派” 梦未来》的文体表演，伴随着主题曲《相约珠城》的歌声，本届省运会的所有活动结束<ref>{{cite web |author1= |title=安徽省第十四届运动会在蚌圆满落幕|url=http://www.bbnews.cn/zt/ahsdssjydh/sydt/588284.shtml |publisher=蚌埠新闻网 |accessdate=2018-10-25 |date=2018-10-28}}</ref>。
