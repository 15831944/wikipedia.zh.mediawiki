'''安小晶'''（英文名：{{lang|en|'''Axel AN'''}}，{{bd|1974年|9月3日}}）<ref>[http://www.nfcmag.com/article/7166.html 安小晶：以归零心态重新出发]，南风窗网，2017年04月11日</ref>，祖籍[[甘肃省|甘肃省]][[天水市|天水市]]，出生于甘肃省天水市，[[中国|中国]]互联网整合营销专家，亿玛在线副董事长（股票代码：836346）<ref>[http://data.eastmoney.com/notices/detail/836346/AN201603250014087382,JUU0JUJBJUJGJUU3JThFJTlCJUU1JTlDJUE4JUU3JUJBJUJG.html 亿玛在线:2015年年度报告]，东方财富证券，2016年03月25日</ref>，现任登云健康美业集团执行董事兼联席CEO。<ref>[http://news.xinhuanet.com/fashion/2017-02/21/c_1120504887.htm 美业与互联网融合：机会还是陷阱？]，新华网，2017年02月21日</ref>

== 生平 ==
# 2004年，安小晶和柯细兴一起创立亿玛在线。<ref>[http://news.hexun.com/2016-10-24/186553888.html 亿玛在线创始人柯细兴 “春泥”的情怀]，和讯网，2016年10月24日</ref>
# 2008年，先后为京东、58同城、唯品会、聚美优品等多家赴美上市企业提供网络营销解决方案。亿玛在线于2016年3月登陆中国新三板。<ref>[http://net.chinabyte.com/zt/emar/ 专题:2011第三届效果整合营销高峰论坛]，天极传媒，2011年12月13日</ref>
# 2012年在香港科技大学读EMBA期间，安小晶对“互联网+美业”的创业投资方向产生了浓厚兴趣。<ref>[http://www.nfcmag.com/article/7166.html 安小晶：以归零心态重新出发]，南风窗网，2017年04月11日</ref>
# 2015年底，安小晶选择加入登云健康美业集团，将集团旗下的健康管理、生活美容、中医养生、医学美容等传统线下服务与互联网相结合，主导开发了服务美容师的培训平台“妃子校”和针对高端女性用户的O2O平台“每天美耶”。<ref>[http://bschool.hexun.com/2017-02-06/188012317.html 安小晶：用互联网思维成就美业帝国]，和讯网，2017年02月06日</ref>
