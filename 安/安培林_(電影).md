《'''安培林'''》（{{lang-en|''Amblin<nowiki>'</nowiki>''}}）是一部於[[1968年電影|1968年]]上映的美國[[短片|短片]]，由[[史蒂芬·史匹柏|史蒂芬·史匹柏]]編劇、執導及剪輯，並由理查·萊文與帕米拉·麥克麥勒主演，劇情講述一個愛情故事。《安培林》獲得了一系列的獎項，且令[[環球影業|環球影業]]的[[西德尼·希恩伯格|西德尼·希恩伯格]]感到印象深刻，進而與史匹柏簽下了為期7年的合約。

== 劇情 ==

該片敘述了一個愛情故事<ref name="Spielberg on Spielberg"/>，並透過畫面和音效來呈現，完全沒有任何一句對白<ref name="OpenCulture.com"/>。劇情講述一名男孩和一名女孩從沙漠一路搭便車到太平洋畔<ref name="大導演史蒂芬史匹柏"/>{{rp|90}}<ref name="OpenCulture.com"/>。

== 製作 ==

史匹柏用了一天的時間撰寫劇本，隨後，他詢問了一名在[[比佛利山_(加利福尼亞州)|比佛利山]]圖書館工作的年輕人理查·萊文是否有意擔任男主角，並在一個演員名錄中找到了女主角的演員人選帕米拉·麥克麥勒<ref name="OpenCulture.com"/>。該片由{{link-en|艾倫·達維奧|Allen Daviau}}掌鏡；日後達維奧還出任了史匹柏的作品《[[E.T.外星人|E.T.外星人]]》（1982年）的攝影師<ref name="大導演史蒂芬史匹柏"/>{{rp|90}}。《安培林》由[[35毫米底片|35毫米底片]]拍成<ref name="Fischer"/>{{rp|548}}。

部分資金由史匹柏的有錢同學丹尼斯·霍夫曼提供。霍夫曼一直很想擔任電影製片人<ref name="大導演史蒂芬史匹柏"/>{{rp|90}}，他一共資助了10,000美元<ref name="OpenCulture.com"/>。《安培林》花費10天的時間來製作<ref name="大導演史蒂芬史匹柏"/>{{rp|90}}，預算為15,000美元<ref name="IMDb"/>。

== 發行 ==

《安培林》於1968年12月18日在美國上映<ref name="IMDb"/>。1970年，該片與《[[愛情故事_(1970年電影)|愛情故事]]》一同在全國聯映<ref name="大導演史蒂芬史匹柏"/>{{rp|91}}。該片在{{link-en|亞特蘭大影展|Atlanta Film Festival}}和[[威尼斯影展|威尼斯影展]]上得獎<ref name="大導演史蒂芬史匹柏"/>{{rp|91}}。

== 反響 ==

當時，史匹柏希望有人能給他一個擔任導演的機會，便四處將作品展示給製片人看<ref name="OpenCulture.com"/>。1965年，史匹柏將《安培林》展示給[[環球影業|環球影業]]的剪輯部門的負責人恰克·斯爾文看。斯爾文對該片頗為讚賞，但除此之外愛莫能助<ref name="大導演史蒂芬史匹柏"/>{{rp|88}}。隨後，史匹柏被環球影業的[[西德尼·希恩伯格|西德尼·希恩伯格]]召見，簽下了為期7年的合約<ref name="大導演史蒂芬史匹柏"/>{{rp|90}}<ref name="Fischer"/>{{rp|548}}。

[[傑利·路易斯|傑利·路易斯]]曾在[[南加州大學|南加利福尼亞州大學]]的電影班任教多年；史匹柏曾是他門下的學生之一<ref name="Jerry Lewis-TV Guide Biography"/>。1968年，路易斯在課堂上播了《安培林》，並告訴他的學生們，「這就是有關電影製作的一切」<ref name="McBride"/>{{rp|168}}。

史匹柏在日後的評論中批評該片，稱其為「粗俗商業主義的產物」，並把《安培林》比做[[百事可樂|百事可樂]]的廣告。他認為《安培林》揭示了當時的自己對時事是多麼的不屑一顧<ref name="大導演史蒂芬史匹柏"/>{{rp|90}}，「難怪我沒有去[[肯特州立大學#事件|肯特州]]或[[越戰|越南]]」，他說，「我現在無法直視它」<ref name="OpenCulture.com"/>。
