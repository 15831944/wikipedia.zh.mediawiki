'''安德烈·歐弗蘭多'''（{{lang-en|'''André Øvredal'''}}，{{bd|1973年|}}）是一名[[挪威|挪威]]男導演、編劇、監製和剪輯師。

== 職業生涯 ==
歐弗蘭多的職業生涯始於2000年。2010年，他所執導的黑色奇幻片《[[追擊巨怪|追擊巨怪]]》是一部[[拾得錄影_(片種)|拾得錄影]]的[[偽紀錄片|偽紀錄片]]，並獲得了正面好評<ref>{{cite web |url=http://illusion.scene360.com/movies/58773/scandinavian-films |title=10 Great Modern Scandinavian Films |last=Smail |first=Christopher |work=Illusion Magazine |publisher=Scene360 |date=2014-04-17 |accessdate=2016-11-13}}</ref><ref>{{cite web |url=http://www.thisishorror.co.uk/see-horror/film-reviews/the-troll-hunter-trolljegeren-2010 |title=The Troll Hunter (Trolljegeren) (2010) |last=Brookes |first=Raven |date=2011-12-19 |work=This Is Horror |accessdate=2016-11-13}}</ref>，後來更成為了一部[[邪典電影|邪典電影]]。歐弗蘭多執導的[[美國|美國]]恐怖片《{{le|驗屍官 (電影)|The Autopsy of Jane Doe|驗屍官}}》於2016年上映，同樣也贏得了普遍讚譽。
