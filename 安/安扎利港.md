'''安扎利港'''（[[波斯语|波斯语]]：<span contenteditable="false"> </span><span dir="rtl" contenteditable="false" lang="fa">بندرانزلی</span>，[[罗马化|罗马化]]：'''Bandar-e Anzalī'''；在[[伊朗伊斯兰革命|伊朗伊斯兰革命]]前曾被[[穆罕默德-礼萨·巴列维|穆罕默德-礼萨·巴列维]]改名为巴列维港，[[罗马化|罗马化]]：Bandar-e Pahlavi）<ref><span>Bandar-e Anzali can be found at </span>GEOnet Names Server<span>, at </span>[http://geonames.nga.mil/namesgaz/ this link]<span>, by opening the Advanced Search box, entering "-3055109"  in the "Unique Feature Id" form, and clicking on "Search Database".</span></ref> 是[[伊朗|伊朗]][[吉蘭省|吉兰省]]的一座城市。截至2006年，安扎利共有109,687人，32,424户。<ref><cite class="citation web">[https://web.archive.org/web/20111114233307/http://www.amar.org.ir/DesktopModules/FTPManager/upload/upload2360/newjkh/newjkh/01.xls "Census of the Islamic Republic of Iran, 1385 (2006)"]. </cite></ref>

'''安扎利港'''是一个坐落于里海海岸的港口城市，靠近[[拉什特|拉什特]]。安扎利港是伊朗北部最重要的海港之一。它包含了''Mianposhteh''岛及周围的诸岛屿。安扎利港的旅游景点包括一个叫''Manareh''的钟楼、海港廊道、[[萨菲德河|萨菲德河]]的沿岸河滩以及一个表面有积水的三角洲。安扎利港也是1942年4月1日至10月[[波兰|波兰]][[安德军|安德军]]自[[苏联|苏联]]撤退后上岸的地方。城市里的波兰公墓也建立于同一年。

== 历史 ==
安扎利是古伊朗一座古老的城市，与[[卡度辛|卡度辛]]类似，出于与[[塞浦路斯|塞浦路斯]]的友好关系及他们在战斗中的合作，更名为安赞（[[罗马化|罗马化]]：Anshan-e Pars），希腊人称他们为Anzaluy。在巴列维语中是Anzalag，它的阿拉伯语形式为Anzalaj。安扎利湾是一处商船和渔船的安全港。1935年时被重新命名为巴列维港。

1919年，由于[[安东·伊万诺维奇·邓尼金|安东·伊万诺维奇·邓尼金]]将军率领的[[沙皇俄國海軍|白海军]]的溃败，他手下的18艘舰艇前往安扎利港寻求庇护。1920年5月18日，一支由13艘舰艇组成的[[红海军|红海军]][[区舰队|区舰队]]对安扎利发动了一起突然袭击，俘虏了当地的英国驻军和前来避难的18艘白海军舰艇。这催生了短命的[[波斯苏维埃社会主义共和国|波斯苏维埃社会主义共和国]]和[[波斯共产党|波斯共产党]]。苏联政府声称不为这起攻击负责，而将其归咎于当地海军军官的自行决断。<ref>A Peace to End All Peace, David Fromkin, 1989, pp.458-459.</ref>

安扎利港美丽的滨海湖、海关与市政厅（[[波斯语|波斯语]][[罗马化|罗马化]]：Shahrdari）、Mian Poshte宫和 Motamedi楼是它主要的旅游景点。直到1980年以前，安扎利还是[[里海海军科学大学|里海海军科学大学]]的所在地（该校后迁至[[瑙沙赫爾|瑙沙赫尔]]）。由Moayer-ol-mamalekk为[[納賽爾丁·沙|纳萨尔丁·沙阿]]建造的宫殿和它著名的石浴受到了人为及自然因素的破坏。

石浴的一大奇特之处是其蓄水池中水的加热与循环系统。一些人认为，Motamedi楼曾受到过改造——它现在是警察局的所在地。这个[[卡扎尔王朝|卡扎尔王朝]]时期的两层建筑建造时曾受到Mirza Abd-ol-Vahab的帮助。<ref>{{Template:Cite web|url=http://anzaliclub.ir/%D8%AA%D8%A7%D8%B1%DB%8C%D8%AE-%D8%A7%D9%86%D8%B2%D9%84%DB%8C/|title=انزلی کلاب|accessdate=8 October 2015|work=انزلی کلاب}}</ref>

== 地理 ==
[[File:AC-Borj-e-Tavoos.jpg|缩略图]]
[[安扎利滨海湖|安扎利滨海湖]]将安扎利港分为两块。城镇两部分由两座桥连接起来。安扎利港有一个鱼子酱加工工厂，还有一些19世纪的遗址以及著名的周六[[巴扎|巴扎]]。''Tourbebar''是一个距离安扎利港城区大约40km的村庄，靠近安扎利滨海湖。

== 气候 ==
安扎利的气候在伊朗城市中属于最潮湿的，它深秋和初冬时期的降水、持续高度湿润和低日照度有些类似于[[日本|日本]]的[[北陸地方|北陆地方]]，但相比于[[北陸地方|北陆地方]]有着更少的夏季降水。它的气候类型被划归为[[副热带湿润气候|亚热带湿润性气候]]（[[柯本气候分类法|柯本气候分类法]]'' Cfa''）。这种温暖而又潮湿的天气使得该区域得以种植譬如[[水稻|水稻]]和[[茶|茶]]这类需要大量水分的作物，特别是还受到来自[[厄尔布尔士山脉|厄尔布尔士山脉]]额外落水的影响。

== 鱼子酱 ==
安扎利港是一个鱼子酱生产中心。鱼子酱的生产与销售由国家政府通过财政部控制下的[[伊朗渔业公司|伊朗渔业公司]]垄断。公众不被允许进入那些储存着成吨的被取出鱼籽（约占鱼重的1/10）后的[[鱘科|鲟鱼]]——有些甚至长达3米、重100千克——的巨大冷库。

== 运动 ==
安扎利港有一个职业俱乐部——[[安扎利港水手足球俱乐部|伊朗水手足球俱乐部]]，参加[[伊朗職業足球聯賽|伊朗职业足球联赛]]。

== 住民和文化 ==
安扎利住民的自然母语是[[吉拉基语|吉拉基语]]，[[波斯语|波斯语]]是他们的国家民族语言。
