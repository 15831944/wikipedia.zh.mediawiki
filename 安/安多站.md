'''安多站'''位于[[中国|中国]][[西藏自治区|西藏自治区]][[那曲地区|那曲地区]][[安多县|安多县]]县城南部，[[海拔|海拔]]4702米，于2006年7月1日启用<ref name="r1">中华建筑资讯网：〈[http://www.rpccn.com/news/33/200673162447.htm 藏域风格建筑群 璀璨“新天路”] {{webarchive|url=https://web.archive.org/web/20070930190113/http://www.rpccn.com/news/33/200673162447.htm |date=2007-09-30 }}〉，于2007年6月25日存取</ref>。该站为[[青藏铁路|青藏铁路]]进入西藏的第一大客货两用车站，由[[青藏铁路集团公司|青藏铁路集团公司]]营运<ref name="r2">鼎言商旅网：〈[http://xz.d0086.com/SKM/sljt/xxlcz.htm 安多火车站]〉，于2007年6月25日存取</ref>。

== 车站设施 ==
安多站全长1601米，占地14万[[平方米|平方米]]<ref name="r2"/>。根据中华建筑资讯网的资料，安多站站房设计模仿帐篷及毡房，屋顶的挑檐以对称的两翼上扬设计，仿如草原上随处可见的雄鹰，也隐喻[[安多县|安多]]以至[[西藏自治区|西藏]]的飞升之势<ref name="r1"/>。

== 使用情况 ==
安多站是[[青藏铁路|青藏铁路]]的客货两用车站，有14列旅客列车经过此站，其中4列（'''K917'''、'''K918'''、'''K9801'''、'''K9802'''）停靠此站，主要为[[拉萨站|拉萨]]到发的各类旅客列车。

== 始发车次 ==
目前安多站未有任何始发车次。
