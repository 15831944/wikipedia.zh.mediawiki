'''安娜·西利亞·德哈瑪斯·卡索'''（{{lang-es|'''Ana Celia de Armas Caso'''}}，{{bd|1988年|4月30日}}）是一位[[古巴|古巴]]女演員。2015年，她演出首部美國電影《[[夜半女敲門|夜半女敲門]]》，之後在《''{{link-en|Exposed (2016 film)|Exposed (2016 film)|Exposed}}''》、《[[光榮擂台|光榮擂台]]》和《[[軍火狗|軍火狗]]》中也有演出。

2017年，她在受好評的新黑色科幻電影《[[銀翼殺手2049|銀翼殺手2049]]》中飾演[[人工智能|人工智能]]立體投影Joi，她在裡面的表演令全世界的觀眾認識了她。<ref>{{Cite web |url = https://www.forbes.com/sites/markhughes/2017/10/05/review-blade-runner-2049-miraculously-accomplishes-the-impossible/#75765d5e6e4a |title = Review: 'Blade Runner 2049' Miraculously Accomplishes The Impossible |author = Mark Hughes |date = 2017-10-05 |publisher = Forbes |language = en}}</ref><ref>{{Cite web |url = https://www.esquire.com/entertainment/movies/a12184443/ana-de-armas-2017-interview-blade-runner-2049/ |title = Ana de Armas Is Complex and Incredible in Blade Runner 2049 |author = Emily Poenisch, By |date = 2017-10-03 |publisher = Esquire |language = en}}</ref><ref>{{Cite web |url = https://www.hollywoodreporter.com/news/hollywoods-next-big-thing-blade-runner-2049-breakout-ana-de-armas-1044837 |title = Hollywood's Next Big Thing: 'Blade Runner 2049' Breakout Ana de Armas |author = Rebecca Ford |date = 2017-10-06 |publisher = The Hollywood Reporter |language = en}}</ref>

== 早期生活 ==
安娜·德哈瑪斯出生於[[哈瓦那|哈瓦那]]。她畢業於[[古巴國家劇院|古巴國家戲劇學校]]。在18歲時，她搬到了[[西班牙|西班牙]]。

== 個人生活 ==
德哈瑪斯目前住在[[美國|美國]][[加利福尼亞州|加利福尼亞州]]的[[洛杉磯|洛杉磯]]。她在2011年时与西班牙演员{{link-en|馬克·克洛特|Marc Clotet}}结婚，2013年两人离婚。
