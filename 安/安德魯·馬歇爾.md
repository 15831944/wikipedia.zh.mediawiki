'''安德魯·馬歇爾'''（{{lang-en|Andrew Marshall}}；{{bd|1921年|9月13日|2019年|3月26日}}）於1973年到2015年擔任[[美國國防部|美國國防部]][[淨評估辦公室|淨評估辦公室]][[外交政策|外交政策]]策略家，由美國總統[[理查·尼克森|理查·尼克森]]任命，並歷經數屆總統任期，直到2015年1月2日退休，接任者為吉姆·貝克（{{lang-en|Jim Baker，James H. Baker}}）主任。

== 學歷 ==
* 1949年取得[[芝加哥大學|芝加哥大學]]經濟系的[[文學碩士|文學碩士]]。<ref>{{cite journal |last=Marshall |first=A. W. |year=1950 |title=A Test of Klein's Model III for Changes of Structure |journal=[[Econometrica|Econometrica]] |volume=18 |issue=3 |pages=291 |jstor=1905802 }}</ref><ref>{{cite book |last=Qin |first=Duo |title=The Formation of Econometrics: A Historical Perspective |location= |publisher=Clarendon Press |year=1993 |isbn=0-19-828388-1 |pages=137–138 }}</ref><ref>{{cite journal |first=Carl F. |last=Christ |authorlink=Carl Christ |title=The Cowles Commission's Contributions to Econometrics at Chicago, 1939–1955 |journal=[[Journal_of_Economic_Literature|Journal of Economic Literature]] |volume=32 |issue=1 |year=1994 |pages=30–59 |jstor=2728422 }}</ref>

== 經歷 ==
* 1949年任職於[[蘭德公司|蘭德公司]]
* 1973年到2015年任職於美國國防部淨評估辦公室
* 提供資金協助[[白邦瑞|白邦瑞]]出版書籍包括1998年的'''Chinese Views of Future Warfare'''<ref>{{cite web |url=https://www.foreignaffairs.com/reviews/capsule-review/1997-11-01/chinese-views-future-warfare |title=Chinese Views of Future Warfare |work=Foreign Affairs}}</ref>，'''China Debates the Future Security Environment'''<ref>{{cite web |url=https://www.amazon.com/China-Debates-Future-Security-Environment/dp/1410218562 |title=China Debates the Future Security Environment}}</ref>，以及2015年的'''2049百年馬拉松：中國稱霸全球的祕密戰略'''<ref>{{cite web |url=https://www.wsj.com/articles/book-review-the-hundred-year-marathon-by-michael-pillsbury-1424996150 |title=Panda Hugger Turned Slugger |work=Wall Street Journal}}</ref>。

== 評價 ==
* [[中國人民解放軍|中國人民解放軍]]陳舟少將稱讚'''安德魯·馬歇爾'''為解放軍在研究[[軍事事務變革|軍事事務變革]]時的英雄，他指出：我們逐字翻譯了'''安德魯·馬歇爾'''陳述的所有言論。<ref name="EconomistChina">{{cite web | url=http://www.economist.com/node/21552193?fsrc=nlw%7Chig%7C4-5-2012%7C1303226%7C36310463 | title=The dragon’s new teeth | publisher=The Economist | date=Apr 7, 2012 | accessdate=April 6, 2012}}</ref>
* [[外交政策_(雜誌)|外交政策]]將'''安德魯·馬歇爾'''列入2012年度的[[FT_100全球知識分子|FT 100全球知識分子]]。<ref>{{cite web |url=https://foreignpolicy.com/articles/2012/11/26/the_fp_100_global_thinkers?page=0,33 |title=The FP Top 100 Global Thinkers |date=26 November 2012 |work=Foreign Policy |accessdate=28 November 2012 |archivedate=2012-11-28 |archiveurl=https://www.webcitation.org/6CViUyRpk?url=http://www.foreignpolicy.com/articles/2012/11/26/the_fp_100_global_thinkers?page=0,33 |deadurl=yes |df= }}</ref>

== 過世 ==
'''安德魯·馬歇爾'''以高齡97歲逝世於[[阿靈頓郡_(維吉尼亞州)|阿靈頓郡]]。<ref>{{Cite web|url=https://www.nytimes.com/2019/03/26/us/politics/andrew-marshall-dead.html|title=Andrew Marshall, Pentagon's Threat Expert, Dies at 97|work=[[The_New_York_Times|The New York Times]]|author=Barnes, Julian E.|date=March 26, 2019|accessdate=March 28, 2019}}</ref><ref>{{cite web |url=https://www.rand.org/news/press/2019/03/26.html |title=Andrew Marshall, RAND Researcher Who Founded Department of Defense's 'Internal Think-Tank,' Dies at 97 |date=March 26, 2019 |work=RAND |accessdate=March 26, 2019}}</ref>[[美國眾議院軍事委員會|美國眾議院軍事委員會]]的代表[[麥克·索恩貝里|麥克·索恩貝里]]議員在聽證會提及'''安德魯·馬歇爾'''過世的消息，並稱讚'''安德魯·馬歇爾'''是少數正確指引美國國防與國家安全方向的重要人物，是值得美國人感到榮耀的卓越外交政策策略家。<ref name = "John Bolton已認證帳戶_2019"> {{ cite web | title = John Bolton已認證帳戶 | url = https://twitter.com/AmbJohnBolton/status/1111786852204331008 | publisher = Twitter | language = en | date = 2019年3月29日 }} </ref>
