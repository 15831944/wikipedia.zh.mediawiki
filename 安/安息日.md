'''安息日'''（{{lang-he|שַׁבָּת}}，含有“休息<ref>[http://www.jewishjournal.com/you_tai_ren/item/seven_days_a_week_20100404/ 经上说“上帝在第七天停了他造化之工……他休息了……”，其中的“休息”和“沙巴特”在拼写上是一样的（一个是动词，一个是名词，读音上有点区别）。这 “沙巴特”就是“休息日”的意思，在汉语中常被称为“安息日”，在英语中是写作Sabbath。]</ref>，停止，終止”的意思）是[[猶太教|猶太教]]每週一天的休息日，根据神的有关命令纪念神创世六日后休息的第七日。安息日在[[摩西|摩西]]获得的[[十诫|十诫]]中有明确指出。

根據[[摩西律法|摩西律法]]所定的安息制度，以色列人在每七天一次的聖日、每年幾個特定的日子、 每七年一次和每五十年一次的聖年，都必須守安息。猶太人每七天所守的安息日，按照[[猶太曆|猶太曆]]，是從星期五日落起到星期六日落。當安息日開始時猶太教徒會點起[[蠟燭|蠟燭]]，而時間按當地日落時間而定。安息日是一個喜慶的日子。它是一個記念上帝和思考生命的精神層面，並花時間陪伴家人的一天。現代社會的[[星期六|星期六]]作為多數國家的公眾假期，便是受此影響。

== 创立和目的 ==
《創世記》记载，[[耶和華|耶和華]]使地球適宜人類居住之後就安歇，停止了地球上的創造工作。而當初並沒有分為猶太人與外邦人之別。所以安息日是屬於全人類。<ref>参考創世記2章1-3节</ref>

《出埃及記》记载，耶和華透過十誡吩咐人們紀念上帝六日創造萬物的事蹟，並且與上帝一起休息的日子。<ref>参考 出埃及記20章8-11节</ref>

《申命記》记载，“不要忘記你在埃及地也做過奴隸。耶和華你的上帝用强壯的手、伸出的臂膀把你從那裏領出來，所以耶和華你的上帝吩咐你要守安息日。” 从这里可以看出，耶和華吩咐以色列人守安息日，是跟上帝拯救他們脱離[[埃及|埃及]]的奴役有關，而不是跟伊甸園發生的事有關。<ref>参考 申命記5章15节</ref>

《出埃及記》也记载，“以色列全會衆從埃及地出來以後第二個月的十五日，到了以琳和西奈之間的遜曠野。摩西對他們説：‘這件事耶和華早已説過。明天要守安息日，就是耶和華所定的聖安息日。……六天你們要出去撿食物，第七天是安息日，必沒有食物可撿。’……耶和華對摩西説：‘……要記住，耶和華把安息日賜給了你們’。”<ref>参考出埃及記16：1，23-29</ref>在此之前，人已經把七天定為一週，但妥拉在這裏才首次提到守安息日的規定。

《以賽亞書》記載，當上帝的新天新地降臨的時候，所有的人都要在安息日敬拜上帝，並且共同觀看違背上帝旨意之人的屍首。<ref>参考 以賽亞書66章22-24节</ref>

== 希伯來聖經中的安息日 ==
[[犹太教|犹太教]][[希伯来圣经|希伯来圣经]][[正典|正典]]《[[塔納赫|塔納赫]]》多次提及要遵守安息日，最明顯是[[十誡|十誡]]中的第四誡 ([[出埃及記|出埃及記]]20:8-11 和[[申命記|申命記]]5:12-15)。 其它提有提及的包括出埃及記31:12-17 和 35:2-3、 [[利未記|利未記]] 19:3 及 19:30, 23:3 and [[民數記|民數記]]28:9-10 。 [[以賽亞書|以賽亞書]] 56:4,6 、 [[以西結書|以西結書]] (20, 22, 23章) 和 [[尼希米記|尼希米記]] 9:14都有提及。

== 意義 ==

===猶太教===
塔納赫及和[[猶太教公禱書|猶太教公禱書]]表示安息日有以下四個目的：
# 記念創世記中記述[[上帝|上帝]]的創造和第七天竭工。（[[創世記|創世記]]2章：1-3節）
# 到新天新地之後敬拜[[上帝|上帝]]的日子。（[[以賽亞書|以賽亞書]]66章：22-24節）
# 記念以色列人從[[古埃及|古埃及]]為奴之地得釋放（[[申命記|申命記]]5章：15節）。
# 預表猶太教彌賽亞來到之時。

猶太教視安息日為神聖的歡樂日子。猶太律法規定安息日為其曆法中最重要日子：
* 它是聖經所記首個聖日，而上帝借終止創造之工來遵守它（創世記 2:1-3節）。
* [[猶太禮儀書|猶太禮儀書]] 視之為日子中的新娘和女王。
* 星期六早上禮儀唸誦的[[摩西五經|妥拉]]經卷比平日長，而安息日當日是比其它聖日更長。
* 古代猶太律法規定違反安息日規定者要以石頭擲死，這是最嚴重的懲罰。<ref>參看 民數記 15:32-36。

=== 基督宗教 ===
在[[新約聖經|新約聖經]]中安息日的目的與生活方式：
# 休息（[[路加福音|路加福音]]23章：54-56節）
# 聚會，有外邦的婦女基督徒在外邦之地的河邊聚會。（[[使徒行傳|使徒行傳]]16章：9-13節）
# 為耶穌的真理辯論。（[[使徒行傳|使徒行傳]]17章：1-2節）
# 作善良的事情的日子。（[[馬太福音|馬太福音]]12章：9-12節）
