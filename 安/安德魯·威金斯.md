'''安德鲁·威金斯'''（{{lang-en|'''Andrew Wiggins'''}}，{{bd|1995年|2月23日}}），[[加拿大|加拿大]]职业篮球运动员，目前为[[明尼蘇達灰狼|明尼蘇達灰狼]]效力，綽號圍巾。在[[2014年NBA选秀|2014年NBA选秀]]上，他以第1顺位被[[克里夫兰骑士|克里夫兰骑士]]选中，成为继[[安东尼·贝内特|安东尼·贝内特]]之后第二位加拿大籍狀元，隨後被交易至[[明尼蘇達灰狼|明尼蘇達灰狼]]以交換[[凱文·洛夫|凱文·洛夫]]。他不但被誉为自勒布朗·詹姆士後最強的高中球员，而且更被譽爲NBA明日之星<ref>[http://www.foxsports.com/nba/story/minnesota-timberwolves-andrew-wiggins-720-dunk-540-miss-080316 Wolves' Andrew Wiggins nearly throws down 720-degree dunk]foxsports.com</ref>。結果在[[2014-15_NBA賽季|2014-15年賽季]]一如外界所料，奪得[[NBA年度最佳新秀|NBA年度最佳新秀]]。安德鲁·威金斯同時也是[[加拿大國家男子籃球隊|加拿大國家男子籃球隊]]的成員。

== 早年生活 ==
威金斯出生在[[加拿大|加拿大]][[多伦多|多伦多]]。他是前NBA球員米切爾·威金斯（Mitchell Wiggins）的兒子，威金斯的母親瑪麗塔·佩恩·維金斯（Marita Payne-Wiggins）為前奧運田徑運動員，優秀的籃球基因正是威金斯成名的基礎。當他九歲時，開始接觸了籃球，並在多倫多加入了一個10歲以下的球隊。此時，他的身高已經有5英尺7英寸（1.70米），一年後，威金斯成長為6英尺6英寸（1.98米），並充分展現了他的爆發力，威金斯曾在某次扣籃時震碎了籃球框的玻璃背板。

== 高中時期 ==
在2013年2月，威金斯在高中聯賽中狂砍57分13籃板4火鍋，創造了自己在杭廷頓高中效力期間的單場最高分。由於威金斯擁有異於常人的體格，他一直在“ESPN高中生排名榜”中排名第一。

== 大学時期 ==
高中畢業後，威金斯選擇在[[堪萨斯大學|堪萨斯大學]]打球。在代表堪薩斯大學征戰的唯一一個賽季中，威金斯在場均32.8分鐘的出場時間內能拿到17.1分5.9個籃板球1.5次助攻1.2次抄截，命中率44.8%，三分球命中率34.1%。不過在NCAA錦標賽中，威金斯所在的堪薩斯大學第二輪爆冷不敵斯坦福大學，他的表现和外界的期待有落差，加上隊友[[喬爾·恩比德|喬爾·恩比德]]打出驚人表現，令威金斯的選秀行程下跌。幸好在球季中段他表现回勇，順利成為2014年選秀狀元。

== 職業生涯 ==
=== 明尼蘇達灰狼（2014年-至今） ===
在2015年2月14日舉行的新秀挑戰賽中，威金斯代表國際隊出戰23分鐘，11投8中得到22分6籃板4助攻1抄截1火鍋，幫助國際隊以121-112戰勝美國新秀隊，威金斯被評為新秀挑戰賽最有價值球員，這也是他NBA生涯首座冠軍獎盃。

2015年4月30日，NBA正式宣佈威金斯獲得了[[2014-15_NBA賽季|2014-15年賽季]]的年度最佳新秀獎。威金斯新秀賽季場均能夠得到16.9分、4.6個籃板和2.1次助攻，為灰狼隊史第二的新秀表現，僅次於1992-93賽季[[克里斯汀·雷特納|克里斯汀·雷特納]]的18.2分。其中場均得分為本屆新秀中最高。他是NBA歷史上首位加拿大籍的最佳新秀球員，同時也是繼2001-02賽季的[[保羅·蓋索|保羅·蓋索]]後，歷史上第2位獲得年度最佳新秀的外籍球員。

威金斯是明尼蘇達灰狼隊史26年來首位最佳新秀球員、NBA史上第5位在19歲或以下的年齡場均砍下16分以上的球員，另外4名球員分別是[[勒布朗·詹姆斯|勒布朗·詹姆斯]]、[[卡梅羅·安東尼|卡梅羅·安東尼]]、[[凱里·厄文|凱里·厄文]]和[[凱文·杜蘭特|凱文·杜蘭特]]。

==球風==
威金斯擁有得天獨厚的小前鋒身材。身高6呎8吋卻有著長達7呎的驚人臂展，搭配上絕佳的彈跳能力，高人一等的敏捷性，身體素質極為強悍。他有著良好的身材去配合他的精英級運動能力。在球場上快速的第一步使他幾乎是無往不利，而他的運動能力更是讓他提高了整體水平，也使他能夠在球場上的任何一處將球投出。關鍵時刻的大心臟，也使他投進過不少關鍵球，背打技術也十分嫻熟，連Kobe也曾領教過。
