'''安亭事件'''指1966年11月10日，“[[上海工人革命造反总司令部|工总司]]”的1000名成员北上赴京告状，导致不能组织跨行业的地区性工人群众组织的禁令废除的事件。

== 过程 ==
11月9日[[上海国棉17厂|上海国棉17厂]]（[[王洪文|王洪文]]）、[[上海国棉30厂|上海国棉30厂]]（[[王秀珍|王秀珍]]）、[[上海国棉31厂|上海国棉31厂]]（[[黄金海|黄金海]]）、[[上海良工阀门厂|上海良工阀门厂]]（[[陈阿大|陈阿大]]）、[[上海玻璃机械厂|上海玻璃机械厂]]（潘国平）、[[上海合成纤维研究所|上海合成纤维研究所]]（[[叶昌明|叶昌明]]）等17家工厂造反组织组成的“工总司”在文化广场召开成立大会。会议受到冲击。会前他们曾要求[[上海市委|上海市委]]承认其组织、[[曹荻秋|曹荻秋]]到会接受批判、提供宣传工具，得到上海市委根据《十六条》的精神和周恩来一再强调的关于不要成立跨行业组织文件规定做出的“不参加、不承认、不支持”“三不”答复。

11月10日，为达到造反组织合法化的[[政治|政治]]目的，他们率队赴京告状。[[潘国平|潘国平]]率1000人登上去北京的特快列车，王洪文率1000人登上驶往[[郑州|郑州]]的[[69次特快列车|69次特快列车]]（运行中被改为[[602次慢车|602次慢车]]）。先后受阻于[[南京|南京]]和[[安亭|安亭]]。当日中午12点他们将[[13/14次列车_(中国)|14次特快列车]]拦阻于[[安亭站_(京沪铁路)|安亭站]]{{NoteTag|[[安亭站_(京沪铁路)|安亭站]]：[[嘉定县|嘉定县]]境内与[[昆山|昆山]]、[[青浦|青浦]]交界地方[[京沪线|京沪线]]上一个小站，距上海市中心西北约30公里。}}，造成[[京沪线|京沪线]]中断20小时（一说30小时）。企图以扩大事态，迫使中央解决问题。事件当即引起北京震动、外电的报道。

11月11日，[[中共中央华东局|中共中央华东局]]第三书记[[韩哲一|韩哲一]]、上海市副市长[[李干成|李干成]]到安亭劝说前，14次特快列车被放行。同时[[陈伯达|陈伯达]]发来电报：“你们的这次行动，不单影响本单位的[[生产|生产]]，而且大大影响全国的[[交通|交通]]”，希望“立即回到[[上海|上海]]去，有问题就地解决”，[[张春桥|张春桥]]11日深夜乘专机抵沪，与“工总司”负责人接触，允诺回上海解决他们提出的各项要求,11月12日下午大部分造反队员回沪。

11月13日上午[[张春桥|张春桥]]与工总司会谈前，参加的上海市委常委会再次重申“三不”决定。下午张春桥在“工总司”“承认组织合法、承认上京告状是革命行动、告状后果由上海市委负责、曹荻秋公开检查、对工总司提供各方面条件”的五项条件上签字。这被印成传单在全上海市散发。[[中央文革小组|中央文革小组]]、[[毛泽东|毛泽东]]获悉消息后迅速同意了这一处理意见。11月16日毛说“可以先斩后奏,总是先有事实，后有概念。”不能组织跨行业的地区性工人群众组织的禁令由此废除，学生[[红卫兵|红卫兵]]与工人[[造反派|造反派]]合流。

== 影响 ==
该事件后，[[多米诺骨牌|多米诺骨牌]]接连倒下：[[解放日报事件|《解放日报》事件]]、[[康平路事件|康平路事件]]相继发生。这一事件成了全面“炮打”[[中共上海市委|中共上海市委]]，导致后来轰动全国的所谓夺权的“[[一月革命|一月革命]]”的导火索、转折点。

[[中共中央华东局|中共中央华东局]]及[[上海市委|上海市委]]的领导[[陈丕显|陈丕显]]、曹荻秋、[[魏文伯|魏文伯]]、[[常溪萍|常溪萍]]等先后被[[打倒|打倒]]、迫害，[[常溪萍|常溪萍]]甚至自杀而亡。

== 怀疑 ==
* [[黄金海|黄金海]]：“如果确如官史所说{{notetag|指工总司拦截14次列车。}}，那么，14次列车上的列车长、乘务员、驾驶员，还有数以千计的乘客，为什么没一个人站出来证实呢？要知道，这正是揭批四人帮的好材料呀！为什么在几十年的高压审查之下，竟没有一个人出来证明当时的拦截情况？四十年过去了，为什么“卧轨”的数百人中，竟也没一个人肯出来讲一讲他们当初是怎样受王洪文指使的？又是怎么采取卧轨行动的？”<ref>黄金海：《“四人帮”兴亡》一书的得失——与叶永烈商榷，《记忆》2013年9月30日第102期</ref>
