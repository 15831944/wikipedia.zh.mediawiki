'''安備墓'''是一座位於[[河南省|河南省]][[洛陽市|洛陽市]]的[[隋朝|隋朝]]（公元589年）[[粟特人|粟特人]]古墓葬，2007年被私人發掘出土。

== 概述 ==
由於是私人建樓時挖地基發現，所以出土文物的相關資訊不明，衹知有一方墓誌和圍屏石榻，墓葬具體位置也不淸楚，墓葬所出文物後來被[[西安|西安]][[大唐西市博物館|大唐西市博物館]]徵集收藏<ref>{{cite web |url=http://xuewen.cnki.net/CJFD-MUSE200903004.html |title=祆教聖火藝術的新發現——隋代安備墓文物初探 |author=葛承雍 |date=2009 |website=xuewen.cnki.net |access-date=2018年8月6日}}</ref>。不過史學博士毛陽光在考證墓誌後，認為安備墓位於[[漢魏洛陽故城|漢魏洛陽故城]]城西張分橋附近<ref name=安備墓誌>{{cite web |url=http://www.heluowenhua.net/heluowenhuayanjiu/kaogu/2016-05-15/689.html |title=洛陽新出土隋《安備墓誌》考釋 |author=毛陽光 |date=2011 |website=www.heluowenhua.net |access-date=2018年8月6日}}</ref>。

墓葬所出的漢白玉圍屏石榻佈滿浮雕，並鎏金彩繪。其上刻繪的守護神頭戴寶冠，腦後帛帶飛揚，飾耳環、項圈、手鐲、瓔珞，腰束銙帶，右手持住三叉戟，立於圓形[[波斯地毯|波斯地毯]]上<ref>{{cite web |url=https://www.thepaper.cn/newsDetail_forward_1943480 |title=唐代護法神式鎭墓俑的來源及其他 |author=李星明 |date=2018年2月6日 |website=www.thepaper.cn |access-date=2018年8月6日}}</ref>。旁邊雕刻的[[壼門|壼門]]中有盤腿而坐的人像，同守護神的裝扮幾乎一樣，整座石榻具有濃郁的[[祆教|祆教]]色彩及[[薩珊王朝|波斯]]風格。因此，葛承雍在《祆教聖火藝術的新發現：隋代安備墓文物初探》一文中認為安備是一名祆教信徒<ref>{{cite journal |author=葛承雍 |date=2009 |title=〈祆教聖火藝術的新發現——隋代安備墓文物初探〉 |url= |journal=《美術研究》 |volume= |issue=3 |pages=第14–17頁 |doi= |access-date=}}</ref>。而毛陽光則通過考釋墓誌，認為安備信仰佛教<ref name=安備墓誌 />。

== 墓誌 ==
[[File:Rubbing_of_An_Pei’s_Epitaph.jpg|thumb]]
安備的墓誌銘共18行，滿行18字，合計315字，以[[魏體|魏體]]刻寫，有界格。墓誌開篇題記「故開府長兼行參軍安君墓誌銘」，全文記載「君名備，字五相，陽城縣龍口鄉曹劉里人。其先出於安居耶尼國，上世慕中夏之風。大魏入朝，名沾典客。父知識，齊車騎大將軍、直盪都督、千乘縣散男。君種類雖胡，入夏世久，與漢不殊。此即蓬生麻中，不扶自直者也。善於白圭之術，蘊而不為，玄髙之業，棄而不慕。訥言慜行，唯事安親。室名龍駒，鄉號指南。孝悌之響，聞於邦國。武平之末，齊許昌王莫府初開，牒為長兼行參軍。一參府寮，備經驅使。雖未執斷，小心恭奉。時輩之中，謙直遜順。屢展懃誠，漸望昇進。但事與願違，遇周統齊，許昌失寵，歸於廉之第。君便義絕，遂還舊廬。斂志東皋，歸田二頃。忽縈疾，醫僚無工。大命運窮，奄從朝露。時年卅有四，以大隋[[開皇|開皇]]九年歲次己酉十月辛酉朔廿四日甲申葬於瀔水之南，張分橋側。恐山壑時移，乃為銘曰：門標貴冑，世代高良。比蘭斯馨，譬蕊能芳。弱冠釋褐，奉事君王。年始過立，奄歸元常。」<ref>{{cite web |url=http://csid.zju.edu.cn/tomb/stone/detail?id=40288b95607358b20160beb326640016&rubbingId=40288b95607358b20160beb3266c0017 |title=隋安備墓誌 |author=<!--Staff writer(s); no by-line.--> |website=csid.zju.edu.cn |access-date=2018年8月6日}}</ref>

大意是安備，字五相，來自陽城縣龍口鄉曹劉里。這裡的陽城縣屬於[[陽城郡|陽城郡]]，轄境相當於今河南省[[登封市|登封市]]、[[禹州市|禹州市]]等地一帶。他的祖上源自安居耶尼國，這個國家在[[南北朝|北朝]]隋唐時期的文獻資料中還從未出現過，經學術界研究，認為可能是[[昭武九姓|昭武九姓]]之一的中亞粟特城邦國[[安國_(昭武九姓)|安國]]。祖上因為仰慕[[中夏|中夏]]，於[[北魏|北魏]]時期來到[[中原|中原]]。安備的父親安知識，曾任[[北齊|北齊]]的[[車騎大將軍|車騎大將軍]]、直盪都督、千乘縣散男等職。安備雖是[[胡人|胡人]]，但在中土生活很久，受[[中原文化|中原文化]]的影響，已經漢化，同漢人沒有區別了。雖善於經商，但志不在此。許昌王[[叱列長叉|叱列長叉]]開府置僚屬後，安備被任命為開府長兼行參軍，開始了仕途生涯。[[北周|北周]]滅北齊，叱列長叉地位下降，安備也「君便義絕，遂還舊廬。斂志東皋，歸田二頃」，回歸了田園生活。後因染疾，僅34歲就去世，於隋[[開皇|開皇]]九年（589年）安葬<ref name=安備墓誌 />。
