{{japanese|japanese=安倍 晴明|kana=あべ の せいめい、あべ の はるあきら|romaji=Abe no Seimei, Abe no Haruakira}}
'''安倍晴明'''（{{lang|ja|あべ の せいめい/はるあき/はるあきら}}、[[延喜|延喜]]21年[[正月十一|1月11日]]<ref>宝賀寿男『古代氏族系譜集成』古代氏族研究会、1986年、369頁。また卒年85歳（「安倍氏系図」（『続群書類従』巻第170 所収）による）とも一致する。</ref>〈921年2月21日〉 - [[寬弘|寛弘]]2年[[九月廿六|9月26日]]〈1005年10月31日〉），是[[平安时代|平安時代]]有名的[[陰陽師|陰陽師]]，從[[鎌倉時代|鎌倉時代]]至[[明治|明治時代]]初期統轄[[阴阳寮|陰陽寮]]的{{tsl|ja|土御門家||土御門家}}始祖。({{tsl|ja|土御門家||土御門家}}偏祖 至- 寬弘2年12月9日，因實力大過於他人，才改為始祖)

安倍晴明在當時處於[[科技|科技]]與[[魔法|咒術]]領域的最尖端，是一位在「{{tsl|ja|天文道||天文道}}」與以占卜為主的「[[阴阳道|陰陽道]]」領域內有著卓越知識的專家，更是一位受到平安[[貴族|貴族]]們信賴的大[[陰陽師|陰陽師]]。

他生前與[[道摩法師|道摩法師]]（蘆屋道滿）是競爭對手，生平事跡多被神秘化，因此衍生許多傳說，更有一說為他可能是[[平將門|平將門]]之子[[平將國|平將國]]。{{來源請求}}

成為後世陰陽術經典的《[[秘传|秘传]]》一書（{{lang|ja|ほきないでん}}、別名《[[金烏玉兔集|金烏玉兔集]]》），雖非他所著，卻也冠上他的名字。

由於近年來[[夢枕獏|夢枕獏]]的小說《[[陰陽師_(小說)|陰陽師]]》改編成[[阴阳师_(电影)|電影]]大受歡迎，因此位於京都的[[晴明神社|晴明神社]]也充滿人潮，大受歡迎。安倍晴明也是長年以來日式動漫遊戲熱門的借鑑人物。

== 史實中的安倍晴明 ==
安倍晴明、據安倍氏所傳系圖、為{{tsl|ja|大膳職||大膳大夫}}官之下級貴族'''安倍益材'''（{{lang|ja|あべのますき}}）之子，生於[[攝津國|攝津國]]阿倍野（現[[大阪市|大阪市]][[阿倍野區|阿倍野区]]）。據稱為[[阿倍仲麻呂|阿倍仲麻呂]]之子孫。生年未詳、而以[[寬弘|寛弘]]二年九月（1005年）亡身，享年85歲之記錄反推，或許生於[[延喜|延喜]]廿年（920年）。也有一說為得年45歲。（土御門家紀錄則享年有異。）幼少時期沒有詳細的紀錄、先後在陰陽師[[贺茂忠行|賀茂忠行]]、{{tsl|ja|賀茂保憲||保憲}}父子的門下學習陰陽道、受{{tsl|ja|賀茂保憲||保憲}}賞識而被傳授{{tsl|ja|天文道||天文道}}。

關於安倍晴明的確實紀錄出現在西元960年，當時[[村上天皇|村上天皇]]任命當時還是'''天文得業生'''（所屬於[[阴阳寮|陰陽寮]]，跟{{tsl|ja|天文博士||天文博士}}學習天文道的學生的職稱）的晴明占卜。雖然晴明出人頭地得晚，但他的占卜才能已經得到了貴族社會的承認。在那之後，他被任命為天文博士。

西元979年，59歲的晴明受皇太子（之後的[[花山天皇|花山天皇]]）的命令，到{{tsl|ja|那智山||那智山}}舉行封印[[天狗_(日本)|天狗]]的儀式。從那時起，晴明似乎得到花山天皇的信賴，紀錄中詳盡的記載了晴明執行陰陽道儀式的情況。花山天皇退位後，晴明也得到了[[一條天皇|一条天皇]]和[[藤原道長|藤原道長]]的信賴，這可以從道長的日記『[[御堂關白記|御堂關白記]]』和其他貴族的日記之中得知。

身為陰陽師極負盛名的晴明，擔任過{{tsl|ja|京職||左京權大夫}}，{{tsl|ja|穀倉院||穀倉院別当}}，[[播磨國|播磨守]]等職位，地位從[[從四位下|從四位下]]一路攀升。另外，晴明的兩個兒子'''吉昌'''和'''吉平'''，也被任命為天文博士和陰陽助（陰陽寮的次官），安倍家族也在晴明這一代成為能跟師父忠行的賀茂一族相提並論的陰陽道世家。

=== 平安、中世文學 ===
在晴明過世的11世紀，他就已經被神格化。在歷史傳記『[[大镜|大鏡]]』和民間傳說『[[今昔物語集|今昔物語]]』『[[宇治拾遺物語|宇治拾遺物語]]』裡皆記載不少有關晴明的神秘逸話。

==== 《大鏡》 ====
* 帝紀「花山天皇」
*: 花山天皇在捨棄帝位出家時，晴明從天象中察知，試著利用[[式神|式神]]跟[[朝廷|朝廷]]急報，那時正好是天皇向寺廟前進的時候。

==== 《今昔物語》 ====
* 「安倍晴明随忠行習道語」
*# 晴明幼少之時、供隨[[贺茂忠行|賀茂忠行]]夜行、於夜道而能見鬼，豫知忠行。忠行悟晴明之才優、傾身相授陰陽之道。
*# [安倍晴明随忠行学习阴阳之道的故事]
*# 晴明年幼尚轻之时，某夜，师傅贺茂忠行外出到下京一带。晴明无意间发现前方有怪异的东西，便向忠行报告了所见的情况。忠行见晴明内蕴灵气，便将自己平生所学悉数传授给了晴明。
* 一日來自[[播磨國|播磨国]]之陰陽師來到陰陽道大宗家晴明之處，挑戰競術、立即敗北。
*# 時有播磨人智德者，亦能方術，欲試晴明。佯為習術者，造其家，二童子從焉。晴明以為彼欲試我術，二童子蓋式神也。於是心念手印，密匿二童，而謂曰：「今日有事，請竢他日。」智德去，少一時復來曰「我失二童，意卿之匿也。」晴明笑曰：「我不知也。」智德叩頭不已，晴明曰：「子欲試我，而謂我不之知邪。何料我之淺也？」乃誦咒，二童忽出。智德歎服曰：「自古使式神也易，而匿他式神則難。卿可謂神也。」遂師事之。【今昔物語。】
* 「播磨国陰陽師智徳法師語」
*# 播磨国陰陽師、{{tsl|ja|智徳法師||智徳法師}}以[[方術|方術]]捕[[海盗|海賊]]之物語。其末尾云「智徳為陰陽師，其才優秀如此。然亦不敵晴明。」。可見，前術物語中所登場之播磨陰陽師是即{{tsl|ja|智徳法師||智徳法師}}。
* 在[[仁和寺|仁和寺]]的{{tsl|ja|寛朝僧正||寛朝僧正}}之所，同席[[公卿|公卿]]們看到使用陰陽道殺死一隻青蛙。
*# 有一日，安倍晴明前去一个居住在广泽、名叫宽朝僧正的人的住处，年轻的贵族公子、僧人们叫晴明以咒术杀死庭院中的青蛙。晴明摘下一片树叶念咒，手沒碰觸青蛙，青蛙当场壓扁死掉。
* 晴明於家使役式神於家事。無人而門戶自開、自閉。
*# 晴明在家中没有其他人时使用式神，家里明明没人在，办窗却能自动打开、关闭，即使没人去开门关门，房门也能自行打开。

==== 《宇治拾遺物語》 ====
* 「晴明蔵人少将封ずる事」
* 「御堂関白の御犬晴明等奇特の事」
* [[藤原道長|藤原道長]]家畜一犬，每出必隨。一日，將往{{tsl|ja|法成寺||法成寺}}，犬遮前而吠，銜衣牽之。[[藤原道長|道長]]異之，不入寺門。召[[晴明|晴明]]問之。[[晴明|晴明]]曰：「有咒咀相府者，埋厭物於此。」因指示一所，掘之得泥封土器，內有朱書一字。[[晴明|晴明]]曰：「世無知此術者，必[[道滿法師|道滿法師]]之所為也。」乃結紙作鳥形，誦咒投之，化為鷺飛去。因遣人從其後，止于萬里小路河原院側民家。眾執其主而來，果[[道滿|道滿]]也。鞫問得實，放道滿於{{tsl|ja|播磨||播磨}}。【[[宇治拾遺物語|宇治拾遺物語]]、[[東齋隨葦|東齋隨葦]]、[[十訓鈔|十訓鈔]]。】

==== 《續古事談》====
* 安倍晴明與賀茂光榮相爭，相較孰較為師父賀茂保憲所看中。
* 光榮傳家學，為曆博士。與安倍晴明並稱。嘗與晴明較短長。晴明曰：「師之在世，常以我置卿右。且百家集傳，在我家。」光榮曰：「愛憎不以親疏。如百家集，我亦傳之。至曆道，我獨得之，卿之所不知也。」世謂：「晴明長方術， 才學不如光榮。」

==ACG作品==

《[[東京闇鴉|東京闇鴉]]》中的重要角色土御門夜光爲安倍晴明之後裔。

《[[少年陰陽師|少年陰陽師]]》中「安倍昌浩」的祖父，同時也為安倍吉昌、安倍吉平之父親。十二神將騰蛇、青龍等的主人。

《[[妖怪少爺|妖怪少爺]]》中羽衣狐附身的少女-山吹乙女所生下的「從地獄回來的安倍晴明」。

《[[獵命師傳奇|獵命師傳奇]]》中樂眠七棺之一，被譽為「最接近神的男人」。

《[[雙星之陰陽師|雙星之陰陽師]]》中陰陽連土御門有馬的祖先，被譽為最強的陰陽師。

《[[银魂|银魂]]》中被恶搞为“结野晴明”，为阴阳师结野众的首领，出现在漫画和动画的“阴阳师篇”。

《[[漂流武士|漂流武士]]》中穿越到异世界，成为管理和集结“漂泊者”的特殊组织“十月机关”的领导人。

《[[陰陽師_(遊戲)|陰陽師 (遊戲)]]》中重要的主角。

《[[妖怪學校的新人教師|妖怪學校的新人教師]]》中的主角同名，為安倍晴明的轉世。

《[[在茜色世界與君詠唱|在茜色世界與君詠唱]]》中的其中角色之一。

《[[通靈王|通靈王]]》中的麻倉葉王的原型。
