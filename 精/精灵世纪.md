'''精灵世纪'''（英语：'''Century Sonny'''）是2006年上映的[[中国大陆|中国大陆]][[三维|三维]]动画片，为中国大陆首部原创高清晰三维卡通动画，代表性的配音员有[[香港|香港]]巨星[[成龙|成龙]]、主持人[[李咏|李咏]]。<ref name=" ios" >{{cite web |url=http://yule.sohu.com/20060705/n244114141.shtml |title=《精灵世纪》大打明星牌 牵手梦想中国  |accessdate= 2006年7月5日|publisher=搜狐网 |archiveurl= http://yule.sohu.com/20060705/n244114141.shtml|archivedate=2012年12月19日 |work= |language=zh-cn}}</ref><ref name=" bv" >{{cite web |url=http://bjyouth.ynet.com/3.1/0607/09/1698579.html |title= 《精灵世纪》大打明星牌|accessdate=2006-07-09 |publisher=北青网 |archiveurl=http://bjyouth.ynet.com/3.1/0607/09/1698579.html |archivedate=2012年12月19日 |work= |language=zh-cn}}</ref><ref name=" xvac" >{{cite web |url=http://www.ccdy.cn/yule/dianshi/201109/t20110927_111775.htm |title= 原创三维动画《精灵世纪》首播|accessdate= 2006-05-22|publisher= 中国文化传媒网|archiveurl=http://www.ccdy.cn/yule/dianshi/201109/t20110927_111775.htm |archivedate=2012年12月19日 |work= |language=zh-cn}}</ref>原计划四季，现播出两季，共52集。

== 背景 ==
该片投资1.5亿元[[人民币|人民币]]，2006年5月开始在中国大陆近400家电视台播出，创造了中国动画片的播放频道数量里程碑。<ref name="fish">{{cite web |url=http://zjdaily.zjol.com.cn/qjwb/html/2006-07/28/content_1081797.htm |title=中国首部3D功夫动画巨片《精灵世纪》  |accessdate=2006年7月28日 |publisher= 钱江晚报|archiveurl=http://zjdaily.zjol.com.cn/qjwb/html/2006-07/28/content_1081797.htm |archivedate=2012年12月19日 |work= |language=zh-cn}}</ref><ref name="qs">{{cite web |url=http://news.sohu.com/20060812/n244762960.shtml |title=《精灵世纪》全国热播第二部将于月底完成 谢霆锋演唱片头曲  |accessdate= 2006年8月12日|publisher=搜狐网 |archiveurl=http://news.sohu.com/20060812/n244762960.shtml |archivedate=2012年12月19日 |work= |language=zh-cn}}</ref>该片播出后迅速受到了中国大陆媒体的广泛关注，并引起了日本、美国及欧洲诸多电视台的极大兴趣。<ref>{{Cite web|url = http://video.sina.com.cn/z/jinglingshiji/05.shtml|title = 关于龙马|accessdate = 2015-07-30|author = |date = 2006|publisher = 新浪宽频}}</ref>

该片的制作方是龙马世纪国际文化有限公司和[[北京电影学院动画学院|北京电影学院动画学院]]。

==剧情==
该片以魔幻、励志为主题，讲述了“精灵小王子卡卡”在文具王国的冒险故事。
