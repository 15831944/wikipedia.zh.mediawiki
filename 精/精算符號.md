'''精算符號'''是[[精算|精算]]用的一套速記法，以便寫出與[[利率|利率]]和[[生命表|生命表]]相關的數學公式。

現有的記法，把[[上標|上標]]和[[下標|下標]]放在主要符號的左或右邊，像光暈一樣圍繞著。以下會列出一些例子。

有些建議的方法改變記法為線性，令符號可以橫向書寫，不需用到上標和下標。這種方法可以免除現有方法在計算上的困難，但現在仍未有一套標準的線性標記方法。

== 例子 ==

=== 利率 ===

<math>\,i\,</math>是一年後得到的利息的利率。所以如果年利率是8%，那麼<math>\,i = 0.08\,</math>。

<math>\,i^{(12)}\,</math>是當利息為每月支付時，把一個月的利率按單利轉為一年的利率，就是把它乘上12。它與<math>\,i\,</math> 不同，因為每個月所得利息，可以用來在下一月賺取利息。事實上，它們的關係式是：
: <math>\!1+i=\left(1+i^{(12)}/12\right)^{12}\!</math>

<math>\,v\!</math> 是一年的貼現率。<math>\,v\!</math>可以從下式算出：<math>\,v = 1/(1+i)\,</math>。它是用來計算若將來要獲得一定的金額，現在須投資的金額。例如如果要一年後獲得1的金錢，現在需要<math>\,v\,</math>；若要在5年後獲得25，現在需要<math>\,25v^5\,</math>。

=== 生命表 ===

<math>\,l_x\,</math>是在''x''歲活著的人的數目。

<math>\,d_x\,</math>是在''x''歲在1年間逝世的人的數目：<math>\,d_x = l_x - l_{x+1}\,</math>。

<math>\,p_x\!</math>是在''x''歲的人活到''x''+1歲的概率：   <math>\,p_x = l_{x+1} / l_x\!</math>。

<math>\,q_x\!</math>是''x''歲的人在1年間逝世的概率：    <math>\,q_x = d_x / l_x\,</math>。

<math>\, {}_2 p_x\,</math>是在''x''歲的人活到''x''+2歲的概率：   <math>\, {}_2 p_x = p_x p_{x+1} \,</math>。

<math>\, {}_2 q_x\,</math>是在''x''歲的人在2年間逝世的概率：   <math>\, {}_2 q_x = 1 - {}_2 p_x \,</math>。
