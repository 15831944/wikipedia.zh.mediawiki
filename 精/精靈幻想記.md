《'''精靈幻想記'''》是[[北山結莉|北山結莉]]所作的[[日本|日本]][[輕小說|輕小說]]作品，[[插圖|插圖]]由Riv繪畫，本為投稿網站《[[成為小說家吧|成為小說家吧]]》連載的網絡小說，後來在2015年經由[[Hobby_Japan|Hobby Japan]]出版書籍<ref>{{cite web|script-title=ja:HJ文庫のマンガサイトがリニューアル、うるりひ描く「sin 七つの大罪」開幕|url=http://natalie.mu/comic/news/199901|website=[[Natalie_(網站)|Natalie]]|accessdate=2017年1月16日|language=ja|date=2016年8月29日}}</ref>，中文版由[[東立出版社|東立出版社]]發行。

== 故事簡介 ==
少年利歐是住在貧民窟的孤兒。他在七歲時的某一天，突然想起曾身為日本大學生「天川春人」的記憶，然後發現自己轉生到了劍與魔法的異世界。利歐因想起這份記憶，腦袋非常混亂，但他也在同時意識到體內擁有強大魔力，於是使用那份魔力為解決綁架事件做出了貢獻。此功績讓他獲得高度評價，於是破例得以進入各地貴族子女雲集的名校就讀！

: 雖人在異國，但她的內心還是一直想與春人再見一面，所以那個勇者也沒嘗到什麼甜頭。反而身陷於美春與親弟弟的冷戰之中。（美春在異國擔任該國王女莉莉安娜的日語老師）

; {{nihongo|艾西雅|アイシア}}
: 寄宿在利歐身上的人型高等精靈，知曉利歐所有的事物。
; {{nihongo|萊娣法／遠藤涼音|ラティーファ}}
: 原本是在某公爵下的接受暗殺者教育的女獸人奴隸（幼女），在暗殺失敗後被利歐所解放，把奴隸之環給去除，而後面又與自己的親人們相遇。現在正努力成為利歐的好妹妹（靠山）而努力。
; {{nihongo|千堂亞紀|千堂 亜紀（せんどう あき）}}

; {{nihongo|千堂雅人|千堂 雅人（せんどう まさと）}}
