{{vgname|精靈寶可夢 Let's Go！皮卡丘／Let's Go！伊布|ja=ポケットモンスター Let's Go! ピカチュウ・Let's Go! イーブイ|en=Pokémon Let's Go, Pikachu! and Let's Go, Eevee!|diff=yes}}，是一款由[[Game_Freak|Game Freak]]制作，任天堂在[[任天堂Switch|任天堂Switch]]上发行的角色扮演类游戏。本作是基于1998年发售的《[[精灵宝可梦_皮卡丘|精灵宝可梦 皮卡丘]]》的重制作品，也是《[[精靈寶可夢系列|精靈寶可夢]]》系列游戏首次登陆任天堂Switch平台。

游戏采用全新的3D游戏引擎制作，在系统上向《[[精灵宝可梦GO|精灵宝可梦GO]]》靠拢，取消了系列正统RPG作品中可以在野外与野生宝可梦战斗的设定，将仅可捕捉野生的宝可梦，并只能与训练家进行战斗；此外游戏将可以与《[[精灵宝可梦GO|精灵宝可梦GO]]》联动，玩家可以将《精灵宝可梦GO》中的第一世代宝可梦传送到本作中。<ref>[https://www.polygon.com/2018/5/29/17407414/pokemon-nintendo-switch-lets-go-pikachu-version-eevee-version Pokémon Let’s Go! is the series’ big Switch debut, and it’s targeting newcomers].Polygon.2018-05-29.[2018-05-30].</ref>开发商还为游戏制作了一个“精灵球Plus”的控制器，不仅能在游戏中进行操控，在现实生活中也可以作为挂饰，并能将游戏中的宝可梦传到该控制器中，进行培育。<ref>[http://www.ign.com/articles/2018/05/30/pokemon-lets-go-pikachu-and-lets-go-eevee-confirmed-for-nintendo-switch-releases-november-16 Pokemon Let's Go Pikachu and Let's Go Eevee Confirmed  for Nintendo Switch, Releases November 16].IGN.2018-05-29.[2018-05-30].</ref>

《精靈寶可夢 Let's Go！皮卡丘／Let's Go！伊布》作为《精灵宝可梦 皮卡丘》的重制作品，仅收录了系列第一世代的151只宝可梦，以及阿羅拉地區形態，但将會公開從未登場的秘密宝可梦，最後公佈為美錄坦以及進化版本美錄梅塔<ref>[https://www.forbes.com/sites/olliebarder/2018/10/25/the-new-crossover-pokemon-meltan-can-evolve-into-melmetal/#2b9017767f3a https://www.forbes.com/sites/olliebarder/2018/10/25/the-new-crossover-pokemon-meltan-can-evolve-into-melmetal/#2b9017767f3a] 福布斯</ref>。

游戏于日本时间2018年5月30日在精灵宝可梦发布会上正式公布，并将于2018年11月16日全球发售。<ref>[https://www.g-cores.com/articles/98834 来了！NS版新作确定为《精灵宝可梦 Let's GO 皮卡丘 / 伊布》].机核网.2018-05-30.[2018-05-30].</ref>

== 遊戲機制 ==
{{main|精靈寶可夢遊戲機制}}
《精靈寶可夢 Let's Go！皮卡丘》和《精靈寶可夢 Let's Go！伊布》的故事舞台是關都地區，有該地區的151種寶可夢及它們的阿羅拉型態。玩家在遊戲中可以透過单个[[Joy-Con|Joy-Con]]或精靈球外型的控制器（精灵球Plus）控制玩家或捕捉寶可夢，並以寶可夢對抗[[非玩家角色|非玩家角色]]的訓練家及道館館主。<ref name="Polygon">{{cite web |url=https://www.polygon.com/2018/5/29/17407414/pokemon-nintendo-switch-lets-go-pikachu-version-eevee-version |archiveurl=https://web.archive.org/web/20180530042425/https://www.polygon.com/2018/5/29/17407414/pokemon-nintendo-switch-lets-go-pikachu-version-eevee-version |title=Pokémon Let’s Go! launches on Nintendo Switch in November |last=Frank |first=Allegra |website=[[Polygon_(website)|Polygon]] |date=2018-05-29 |accessdate=2018-05-30 |archivedate=2018-05-30 |deadurl=no}}</ref><ref name="announcement">{{cite web |url=https://www.gamespot.com/articles/pokemon-lets-go-pikachu-and-lets-go-eevee-announce/1100-6459249 |archiveurl=https://web.archive.org/web/20180530041241/https://www.gamespot.com/articles/pokemon-lets-go-pikachu-and-lets-go-eevee-announce/1100-6459249 |title=Pokemon Let's Go Pikachu And Let's Go Eevee Announced For Nintendo Switch |last=Plagge |first=Kallie |website=[[GameSpot|GameSpot]] |date=2018-05-29 |accessdate=2018-05-30 |archivedate=2018-05-30 |deadurl=no}}</ref><ref>{{Cite news|url=https://www.eurogamer.net/articles/2018-06-07-motion-controls-are-compulsory-for-catching-pokemon-in-lets-go-pikachu-and-eevee|title=Motion controls are compulsory for catching Pokémon in Let's Go Pikachu and Eevee|work=Eurogamer.net|access-date=2018-06-22|language=en}}</ref><ref name="Eurogamer post-E3 2018">{{cite web |url=https://www.eurogamer.net/articles/2018-06-12-pokemon-lets-go-pikachu-eevee-guide-release-date-5010 |archiveurl=https://web.archive.org/web/20180614013403/https://www.eurogamer.net/articles/2018-06-12-pokemon-lets-go-pikachu-eevee-guide-release-date-5010 |title=Pokémon Let's Go Pikachu and Eevee release date, confirmed feature list, plus everything we know about the 2019 Pokémon Switch RPG |last=Tapsell |first=Chris |website=[[Eurogamer|Eurogamer]] |date=2018-06-12 |accessdate=2018-06-14 |archivedate=2018-06-14 |deadurl=no}}</ref>另外精靈球Plus亦可於[[精靈寶可夢GO|精靈寶可夢GO]]當作Pokemon Go Plus使用。

玩家只需要一個Joy-Con就能進行遊戲，並支援兩名玩家一同遊玩，在雙玩家模式時，玩家的戰鬥將對變成雙打對戰，捕捉時也會各投擲一的精靈球，捕捉寶可夢的機率會翻倍。{{来源请求}}

依據版本不同，玩家的初始寶可夢可能是[[皮卡丘|皮卡丘]]或[[伊布_(精靈寶可夢)|伊布]]，在畫面處於第三人稱鳥瞰視角時，它將會位在玩家的肩膀或頭上，這個機制類似於《[[精靈寶可夢_皮卡丘|精靈寶可夢 皮卡丘]]》中首次引進的步行寶可夢機制<ref name="Polygon" />。當玩家靠近隱藏道具時，玩家的搭檔寶可夢會擺動它的尾巴，也可以透過換裝功能來裝飾寶可夢的外觀。玩家還可以像《[[精靈寶可夢_心金·魂銀|精靈寶可夢 心金·魂銀]]》一樣選擇一個寶可夢當作自己的夥伴，讓它們跟隨自己<ref>{{cite web |url=https://www.theverge.com/2018/6/12/17455722/pokemon-lets-go-preview-poke-ball-plus-controller-e3-nintendo |archiveurl=https://web.archive.org/web/20180613205353/https://www.theverge.com/2018/6/12/17455722/pokemon-lets-go-preview-poke-ball-plus-controller-e3-nintendo |title=Pokémon: Let’s Go is a simple game improved by its pricey pokéball controller |last=Farokhmanesh |first=Megan |website=[[The_Verge|The Verge]] |date=2018-06-12 |accessdate=2018-06-13 |archivedate=2018-06-13 |deadurl=no}}</ref>，如果選擇的寶可夢的體型比較大，那麼玩家將會像《[[精灵宝可梦_X·Y|精灵宝可梦_X·Y]]》中的設計一樣騎乘寶可夢。如同《[[精靈寶可夢_皮卡丘|精靈寶可夢 皮卡丘]]》，玩家的初始寶可夢無法進化，不過玩家捕獲的其他同種類寶可夢可以進化。寶可夢蛋機制不會出現在遊戲中。<ref name="GameSpot post-E3 2018">{{cite web|url=https://www.gamespot.com/articles/pokemon-lets-go-pikachu-eevee-everything-we-know-s/1100-6459253|title=Pokemon Let's Go Pikachu, Eevee: Everything We Know So Far|last=Plagge|first=Kallie|date=2018-06-12|website=[[GameSpot|GameSpot]]|archiveurl=https://web.archive.org/web/20180613205331/https://www.gamespot.com/articles/pokemon-lets-go-pikachu-eevee-everything-we-know-s/1100-6459253|archivedate=2018-06-13|deadurl=no|accessdate=2018-06-13}}</ref>在本作中，玩家可以通过中文输入法为自己的角色以及捕捉到的宝可梦起名，这是系列首次支持繁体和简体中文输入，也是任天堂Switch上首款带有中文输入法的游戏。

《Let's Go！皮卡丘》和《Let's Go！伊布》裡一個和主系列遊戲很大的不同是，除非遇上特別情況，遊戲中的野生寶可夢無法對戰，必須透過與寶可夢交流來捕捉它們，捕捉它們後可以獲得經驗值。<ref>{{cite web |url=https://www.engadget.com/2018/06/14/pokemon-lets-go-pikachu-e3-impressions |archiveurl=https://web.archive.org/web/20180619050942/https://www.engadget.com/2018/06/14/pokemon-lets-go-pikachu-e3-impressions |title='Pokémon: Let’s Go, Pikachu!' doesn't feel like a remake |website=[[Engadget|Engadget]] |last=Souppouris |first=Aaron |date=2018-06-14 |accessdate=2018-06-19|archivedate=2018-06-19}}</ref>
