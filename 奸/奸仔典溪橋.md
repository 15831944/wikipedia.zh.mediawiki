[[臺灣鐵路管理局|台鐵]]'''奸仔典溪橋'''是位於[[台灣|台灣]][[臺東縣|臺東縣]][[加典溪|加典溪]]上的一座鐵路[[橋樑|橋樑]]，連接[[臺東縣|臺東縣]][[關山鎮|關山鎮]][[關山車站|關山車站]]與[[鹿野鄉|鹿野鄉]][[瑞和車站|瑞和車站]]，1986年於舊橋原址重建新橋。

== 沿革 ==
[[臺東線|臺東線]]鐵路臺東（今稱[[台東舊站|台東舊站]]）至里壠（今[[關山車站|關山]]）間軌距762mm的輕便鐵路，由[[台東開拓會社|台東開拓會社]]興建，1919年12月16日通車，推測跨越加典溪的「奸仔典溪橋」於當時即已存在，其長度與型式不詳。1920年代初期，[[臺灣總督府|臺灣總督府]]在交通系統一元化政策下，收購[[台東開拓會社|台東開拓會社]]興建的這段鐵路，改為官營，並自1921年起續築里壠（今[[關山車站|關山]]）至璞石閣（今[[玉里車站|玉里]]）路段，銜接1919年5月17日即已完工通車的璞石閣（今[[玉里車站|玉里]]）至[[花蓮車站|花蓮]]段762mm輕便鐵路，並在1926年完成[[臺東線|臺東線]]鐵路全線通車。

1978年7月1日起，台鐵局進行東線鐵路拓寬工程，將[[臺東線|臺東線]]鐵路由原來的762mm軌距拓寬為1067mm，全線於1982年6月26日竣工、27日通車<ref>〈東線鐵路改舖寬軌 全部施工今天完成 寬軌火車明起「上道」承辦貨運 窄軌火車從此「休息」永懷留念〉，《[[中國時報|中國時報]]》，1982年6月26日，第6版。</ref>（但舞鶴－三民間[[自強隧道_(花蓮縣)|自強隧道]]前後路段除外），其中奸仔典溪橋採就地加固橋台及[[橋墩|橋墩]]方式辦理，並將鋼鈑梁抽換為4孔16.7公尺及2孔13.7公尺長之[[鋼筋混凝土|鋼筋混凝土]]丁字梁，全長94.2公尺{{r|老橋技術}}，1980年10月完工{{r|通志}}。

[[日本時代|日本時代]]後期及[[臺灣戰後時期|戰後]]，[[臺灣鐵路管理局|台鐵局]]陸續改建或補強老舊鐵路橋梁，並自1980年代起，大規模辦理舊橋重建，除了已完成重建者{{NoteTag|1=未納入重建計畫的老舊橋梁，舉例如下：
*已於日本時代後期重建者，例如配合[[民雄車站|民雄車站]]至[[嘉義車站|嘉義車站]]間雙線工程，於1941年完工的第二代[[臺鐵牛稠溪橋|臺鐵牛稠溪橋]] <ref>[[臺灣總督府交通局鐵道部|臺灣總督府交通局鐵道部]]：《[[臺灣總督府交通局鐵道部|臺灣總督府交通局鐵道部]] 昭和十六年度年報》，（出版地不詳），1941年，頁102。</ref>。
*已配合戰後[[美援|美援]]時期重建者，例如1953年竣工通車的第二代[[臺鐵曾文溪橋|臺鐵曾文溪橋]]。
*已列入其他鐵路擴建雙線計畫重建者，例如配合高屏鐵路擴建雙軌工程，於1987年6月17日完工通車的新[[高屏溪|高屏溪]]橋<ref>交通部：[[中華民國|中華民國]]77年《交通年鑑》，[[臺北市|臺北市]]，1989年10月10日，頁329。</ref>。}}之外，計羅列出29座橋梁，分別納入「[[五大橋梁重建工程|五大橋梁重建工程]]」及「[[鐵路沿線老舊橋梁重建工程|鐵路沿線老舊橋梁重建工程]]」兩項計畫{{r|百週年}}。奸仔典溪橋由於受到1982年起實施「東部治山防洪計畫」整治河川上游、興建堤防並加寬堤距的影響，導致鐵路落於行水區內，影響排洪與行車安全，[[臺灣鐵路管理局|台鐵局]]遂配合堤防法線及最高洪水位，規劃改建奸仔典溪橋，並列入「鐵路沿線老舊橋梁重建工程」項下。

== 設計施工 ==
奸仔典溪橋採取就地重建的方式施工，為了維持列車運行，工程單位先在原舊橋下游側另建施工便線（即臨時軌）與臨時橋。另外，因為奸仔典溪橋北方四百多公尺也有一座須重建的[[水拔溪橋|水拔溪橋]]（跨越加鹿溪），所以兩橋的重建工程同步進行，並共同鋪設一條施工便線。施工便線在1985年12月16日撥接通車，列車改駛便線之後，工程單位隨即動工拆除舊橋，重建新橋。

重建的奸仔典溪橋為6孔跨距22.9公尺單線箱型預力梁橋，橋長增加為137.4公尺，橋梁高程提高1.9公尺，前後引道也配合抬高，橋墩為單圓柱懸臂式，基礎為圓形開口沉箱，於1986年11月21日完工，待鋪軌完成後，在1987年2月23日自施工便線撥接至新橋通車{{r|老橋技術}}，並使用至今。

== 近況 ==
*2014年6月28日，包含奸仔典溪橋在內的[[臺東線|臺東線]]完成電氣化通車營運<ref>[[交通部鐵路改建工程局|交通部鐵路改建工程局]][https://www.rrb.gov.tw/04100.aspx?id=49&lan=ch 花東線鐵路瓶頸路段雙軌化暨全線電氣化計畫] </ref>。
