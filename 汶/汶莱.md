'''汶萊達魯-{}-薩-{}-蘭國'''（{{lang-ms|Negara Brunei Darussalam}}；[[爪夷文|爪夷文]]：{{lang|ar|بروني دارالسلام}}），也作'''汶萊和平-{}-之國'''{{efn|馬來文的 Negara 意為「國家」，而 Darussalam 則意為「和平之邦」，寓意警惕，並求安定。 }}，簡稱'''-{汶}-萊'''或'''-{文}-萊'''{{efn|「汶」的[[汉语|華語]]讀音應該為「wèn（ㄨㄣˋ）」，而非「wén（ㄨㄣˊ）」，不過[[汶萊華人|當地華人]]常讀作「wén（ㄨㄣˊ）」，而當地華文報章均使用「-{汶}-萊」作為本國譯名。 }}，舊亦稱'''婆羅乃'''<ref>{{cite news|author1=少翁|title=石油小邦婆羅乃|work=聯合報|date=1962-12-14}}</ref>，是一个位于[[婆羅洲|婆羅洲]]北岸的东南亚国家。汶萊是[[北婆三邦|北婆三邦]]之一，位於[[南中國海|南中國海]]南岸，整個國土被[[砂拉越|砂拉越]]所分割、環繞。 1984年1月1日文莱脫離[[英國|英國]]和平獨立，1月7日加入[[东南亚国家联盟|東盟]]。文莱經濟狀況頗佳，2014年人均GDP名列[[亞洲|亞洲]]總第五位、除[[波斯灣國家|波斯灣國家]]外的第二位（列[[新加坡|新加坡]]之後），石油和天然氣的生產和出口約佔國內生產總值的67％，社會福利之程度與範圍皆非常完善，更以不需要繳稅而聞名。

== 历史 ==
[[File:Map_of_Brunei_Demis.png|thumb]]
{{Main|汶萊歷史}}
汶莱从8世纪开始有人定居，在[[中國歷史|中國歷史]]的[[唐朝|唐朝]]史籍中称'''婆利'''，[[明朝|明朝]]史籍中称-{汶}-莱<ref>《[[东西洋考|东西洋考]]》卷五云：“文莱，即婆罗国，东洋尽处，西洋所自起也。”</ref>，亦有古籍稱之為'''勃泥'''或'''渤泥'''。

[[梁朝|梁朝]]時代，一位名叫Pinka和他的家人，即Kaundinya，是一名[[佛教徒|佛教徒]]。據載他是統治了136個區域，他的皇座是用金造的，踏板是用銀造的，皇冠用鑽石做的，相传皇冠有1尺高。他曾經在518年派遣使者去中國，並送了地毯給皇帝。在523年，他也派了使者去其他國家。

在[[隋代|隋代]]，汶萊国王Hu Lu Na Po和他的家人，即Chiari Ya Ka在616年，他們也派了使者去中國。[[唐朝|唐朝]]時，汶萊国王被认为是很有錢的，在[[貞觀_(唐朝)|貞觀]]年間（669年），曾派遣很多使者去中國。

[[宋朝|宋朝]]年間，一個據載叫Hiang Ta的国王，他擁有過百隻船，還有一些士兵。在[[宋太宗|宋太宗]]年間，Hiang Ta送了使者去中國，使者團叫阿布阿里，Sheikh Noh和Kadhi Kasim。在北宋[[元豐|元豐]]年間1082年2月，也曾送過使者去中國。根據傳說，這使者叫Seri Maharaja，他繼承者是Raha Makatunaw和Sang Aji，而這Sang Aji也就是後來蘇丹穆罕默德沙的祖父了。

1395年[[明太祖|明太祖]]朱元璋所著的《[[皇明祖訓|皇明祖訓]]》提到勃泥為15個[[不征之國|不征之國]]之一

汶莱长期受中国文化影响，13世纪后伊斯兰化。当代汶莱由[[渤泥国|渤泥国]][[苏丹|苏丹]][[马合谟沙|马合谟沙]]（Sultan Muhammad Shah）和福建移民[[黄森屏|黄森屏]]率领的华人势力联合组建而成。[[明代|明代]][[永樂|永樂]]年間浡泥王[[麻那惹加那|麻那惹加那]]曾經親自率使臣來中國拜見[[明成祖|明成祖]]，不幸病故，葬於[[南京|南京]]，即今[[浡泥国王墓|浡泥国王墓]]。历史上汶莱王室便奉[[黄森屏|黄森屏]]为始祖，至今文莱仍有所谓“黄总兵墓”。

14世纪，文莱伊斯兰教君主国脫離[[爪哇|爪哇]]的控制，在14至16世纪期間一度非常强大，国土包括[[菲律宾|菲律宾]]南部以及[[砂拉越|砂拉越]]和[[沙巴|沙巴]]；[[欧洲|欧洲]]人的影响使得这一政权走上末路。

18世紀70年代，汶萊受到[[英国|英国]]势力侵入；1888年，汶萊成為[[英國殖民地|英國殖民地]]；1941至1945年，汶萊被[[日本|日本]]佔領；1946年，汶萊恢復為[[英屬|英屬]][[保護國|保護國]]；1971年，汶萊實現内部自治；1984年1月1日，汶萊獲得獨立，但仍然是[[英联邦|英联邦]]成员国。

== 政治 ==
{{main|文莱政治}}

===政治體制===
汶莱的[[蘇丹_(稱謂)|蘇丹]]，沿襲自14世紀以來世襲的王朝，是國家和政府的領袖和軍隊司令。蘇丹以[[國家元首|國家元首]]身份兼任[[文萊首相|文萊首相]]。全部[[文萊立法會|文萊立法會]]立法议员由苏丹任命，没有民选立法机关。

統治國策是馬來伊斯蘭君主憲制（Melayu Islam Beraja，爪夷文是ملاي إسلام براج）。

=== 文莱苏丹 ===
{{main|文莱苏丹}}

現時蘇丹是第29世，[[哈吉·哈桑納爾·博爾基亞|哈吉·哈桑納爾·博爾基亞]]，1946年生於汶萊市。王后乃其表妹薩拉赫。1967年两人结婚，萨拉赫生有四女二子，长子比拉赫为王储。1981年再娶米丽亚姆·阿卜杜勒·阿齐兹，生有两子两女。2003年2月，和米丽亚姆离婚并收回米丽亚姆所有封号与头衔。2005年8月20日在[[马来西亚|马来西亚]]首都[[吉隆坡|吉隆坡]]与亞茲麗娜·瑪茲结婚。育有一子一女。2010年6月，两人离婚。

=== 蘇丹王室 ===
{{main|汶萊歷史}}
[[汶萊王室|汶萊王室]]積極推行[[王族|王族]]內部通婚政策，即是為了鞏固[[馬來人|馬來人]]政權以及維持其[[王室|王室]]政權的合法性，也是保持其王室[[血統|血統]]純正的重要舉措。現任蘇丹[[哈吉·哈桑納爾·博爾基亞|博爾基亞]]的妻子{{link-en|汶萊王后莎麗哈|Queen Saleha of Brunei|莎麗哈}}（{{lang|ms|Saleha binti Mohamed Alam}}）是其表妹，而母親達米特（{{lang|ms|Damit binti [[阿都拉曼|Abdul Rahman]]}}）與莎麗哈的爸爸{{link-en|莫哈末·阿南|Pengiran Anak Mohammad Alam|}}（{{lang|ms|[[馬來語稱謂列表|Pengiran Pemancha Pengiran Anak Haji]]}} {{lang|ms|Mohammad Alam}}）為兄妹關係；他們的祖父[[奧瑪阿里賽義夫汀三世|賽義夫汀三世]]（{{lang|ms|Omar Ali Saifuddien III}}）是蘇丹{{link-en|哈欣·再如·阿南·阿馬汀|Hashim Jalilul Alam Aqamaddin|}}（{{lang|ms|Hashim Jalilul Alam Aqamaddin}}）的兒子。換言之，達米特亦是蘇丹賽義夫汀三世的[[堂亲|堂]][[姪女|姪女]]。因此可以說，達米特和莎麗哈既是[[姑|姑]][[姪|姪]]又是[[婆媳|婆媳]]。

== 行政區域 ==
{{main|汶莱行政区划}}
[[File:Brunei_districts_english.png|thumb]]
汶萊國內共分為4個县（daerah）：

* [[都東县|都東县]]，面積1,166平方公里，人口38,649。

== 经济 ==
[[原油|原油]]和[[天然气|天然气]]是国家的主要支柱，总产值几乎占整个国家[[國內生產總值|國內生產總值]]90%。在[[东南亚|东南亚]]，石油储量和产量仅次于[[印度尼西亚|印度尼西亚]]，居第2位。文莱是世界上最富有的国家之一，同時也和[[新加坡|新加坡]]是唯二最富有的東南亞國家，但汶萊目前仍被視為[[開發中國家|開發中國家]]。 

2005年汶萊人均[[國內生產總值|國內生產總值]]為15,964美元，國民無須繳交[[個人所得稅|個人所得稅]]。

2010年文莱人均國內生產總值为51,600美元。{{fact|汶萊也是東南亞唯一真正没有[[農業|農業]]的國家(另一個類似的是新加坡)}}。

汶莱投资公司（Brunei Investment Company）在欧美拥有的9家饭店座落于美国、英国、法国和意大利，包括比贝弗利山酒店、伦敦多尔切斯特饭店（The Dorchester）和巴黎雅典娜广场饭店（Plaza Athenee）。

== 法律 ==
=== 历史 ===
1959年9月29日颁布第一部宪法。1971年和1984年曾进行重大修改。宪法规定，苏丹为国家元首和宗教领袖，拥有全部最高行政权力和颁布法律的权力。设宗教委员会、继承与册封委员会、枢密院、立法院和内阁部长会议协助苏丹理政。

2004年9月，立法院第一届会议审议并通过宪法修正案，内容涉及司法、宗教、民俗等多个方面，共13项内容，包括赋予苏丹无须经立法院同意而自行颁布紧急法令等法令的权利；制定选举法令，让人民参选从政；增加立法院议员人数；伊斯兰教仍为国教，但人民有宗教信仰自由；仍以马来语作为官方语言，英语可作为法庭办案语言从等。

从2014年起，文莱引入较严格的伊斯兰法作为新法律体系并逐渐推行。第一阶段在2014年实施，当中涵盖涉及监禁和罚款的犯罪。之后汶莱延迟后面两个阶段的进程，当中涵盖的是以[[截肢|截肢]]和[[石刑|石刑]]为刑事的犯罪。2019年4月3日，文莱发表声明，宣布伊斯兰法律全面施行。<ref>http://www.aseanlaw.com.cn/info/da0df32897784df1bf339cc21d112ac7</ref>
=== 法律制度 ===
汶萊同時採用兩種法律制度，一種以[[英国|英國]]法律為根基，另一種則是伊斯蘭司法制度，伊斯蘭法庭一般只處理穆斯林私人與家庭問題，如婚姻、遺產糾紛等。汶莱虽然保留死刑，但是自从1957年以来从未执行过。入境汶莱需要注意以下法规：

* 因是回教國家，汶萊航空機上不會供應酒，境內也沒有夜生活。
* 回教徒女性必將頭、手、膝都遮起來，異性也不得與之握手。
* 在汶萊境內也吃不到豬肉，更要避談豬與豬肉的問題。
* 入清真寺前須換上過膝黑袍及拖鞋，步行其間不可踩到祈禱地毯，[[週五|週五]]聚禮日不開放參觀。
* 攜帶、吸食、販賣、藏匿毒品刑罰最高處以[[死刑|死刑]]。
* 汶萊為單一國籍國家，倘被查獲持有汶萊以外之他國護照，汶萊國籍將會立即喪失。

* 未在週五的聚礼日禮拜的話，都將遭到罰款或是入監服刑。
2019年4月3日，汶萊宣布修改法律，以实施更严格的伊斯兰法律，修改与新增法律包括：
* 同性恋及通奸者将会在“一群穆斯林的见证下”被处以乱石砸死的死刑。此前同性恋、雙性戀及跨性別者的最高刑法是10年监禁。
* 犯下窃盗罪者，最高刑法将被砍断手脚。
* 强奸、通奸、鸡奸，侮辱或诽谤伊斯兰先知穆罕默德，皆会处以死刑。
* 堕胎的女性会面临公开鞭刑。
* 让穆斯林儿童接触伊斯兰教以外的宗教被视为犯罪行为。<ref>https://tw.mobi.yahoo.com/news/lgbt-恐遭亂石砸死-人權組織籲汶萊打消念頭-145431408.html</ref>

== 軍事 ==
[[文莱皇家武装部队|文莱皇家武装部队]]成立於1961年5月31日。在1965年5月31日，當時被稱為Askar Melayu Brunei；全名為汶萊皇家馬來兵團。到了1984年1月1日的汶萊獨立紀念日，汶萊皇家馬來兵團再次改名為汶萊皇家武裝部隊。

在傳統上，只有汶萊公民的馬來族（原住民）可以爭取在汶萊皇家武裝部隊服役。這些馬來族包括Belait、Bisaya、Brunei、Dusun、Kedayan、Murut及和Tutong等汶萊憲法所界定的人民。兵役不是強制性的，汶萊也沒有任何徵兵制。

汶萊皇家武裝部隊所使用的外國軍事設備範圍廣泛，有很大比例來自[[英國|英國]]、[[法國|法國]]等[[歐洲|歐洲]]國家及[[美國|美國]]。{{tsl|en|Royal Brunei Land Forces|文萊皇家陸軍}}是汶萊規模最大的[[武裝力量|武裝力量]]，{{tsl|en|Royal Brunei Air Force|汶萊皇家空軍}}和{{tsl|en|Royal Brunei Navy|汶萊皇家海軍}}的規模則相對較小。汶萊缺乏軍事打擊或實戰經驗，但最近已積極參與國際性的人道主義與維持和平任務的區域部署行動。汶萊也廣泛與[[新加坡|新加坡]]進行軍事交流。

== 人口 ==
文莱总人口约41万，约2/3为[[汶萊馬來人|马来人]]，土著佔6％，人口最多的少数民族是[[汶萊華人|華人]]，约占总人口15%，大多数华人信奉[[佛教|佛教]]，部分为[[回族|回族]][[穆斯林|穆斯林]]，文莱華人中又以[[福建省_(中華民國)|福建]][[金門縣|金門]][[閩南民系|閩南人]]占大多数<ref>[http://tran.httpcn.com/Html/1405/2661901553.shtml 奋斗在高福利国家：文莱侨情述略]</ref>，其餘12％人口為其他民族（包含[[印裔汶萊人|印度人]]）。

官方语言是[[马来语|马来语]]，也使用[[英语|英语]]和[[閩南話|閩南話]]，因為文莱华人绝大多數是来自于[[福建|福建]]和[[广东|广东]]的移民及後裔，所以他們之間的通用语是[[福建話|福建話]]（[[閩南語|閩南語]]）或[[客家话|客家话]]而不是[[华语|华语]]（[[現代標準漢語|現代標準漢語]]、[[中華民國國語|國語]]和[[普通話|普通話]]）。

在文字使用上以[[馬來文|馬來文]]和[[正體中文|正體中文]]為主，而馬來文除了和[[馬來西亞|馬來西亞]]一樣使用羅馬字外，傳統阿拉伯字母書寫的[[爪夷文|爪夷文]]也具有相同的官方文字地位，是少數目前仍然通行[[爪夷文|爪夷文]]的國家。

== 領土争议 ==
{{main|南中国海领土争端}}
[[File:Kampong_Ayer(3).JPG|缩略图]]
文莱与邻国[[马来西亚|马来西亚]]存在领土领海争端，包括[[婆罗洲|婆罗洲]]沿海的深水J和K区块，离陆地80公里，并向外延伸100公里。文莱政府声称J区块和K区块完全位于其专有经济区域（EEZ）内。2002年初，以道达尔公司为首的国际财团被文莱政府授予了文莱深水J区块的勘探许可证，道达尔公司作为该区块的作业者拥有60%的股权。2003年4月，马来西亚的海军巡逻艇驱逐了[[道达尔|道达尔]]公司的勘探船。两国已就解决争议进行过多次协商，但一直未达成任何协议<ref>[http://bn.mofcom.gov.cn/article/ztdy/200304/20030400083711.shtml 文莱石油天然气工业现状]. 2005-01-10</ref>。

目前文莱以200海里专属经济区为由声称拥有约五万平方公里的[[南沙群岛|南沙群岛]]海域主权，其範圍包括[[南通礁|南通礁]]；而[[中华人民共和国|中华人民共和国]]、[[中华民国|中华民国]]、[[馬來西亞|馬來西亞]]以及[[越南|越南]]均宣稱擁有该群岛海域主權。
