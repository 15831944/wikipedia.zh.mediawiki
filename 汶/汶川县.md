'''汶川县'''{{NoteTag|「汶」的现代汉语拼音为Wèn，[[注音符号|注音符号]]：ㄨㄣˋ，部分地区四川话发音同“文”，粵語发音：man4，音同“民”<ref>{{cite web |title=“民” |url = http://www.zdic.net/z/1c/js/6C11.htm |website=汉典网 |accessdate=2015-07-03 }}</ref><ref>{{cite web |title=“汶” |url = http://www.zdic.net/z/1c/js/6C76.htm |website=汉典网 |accessdate=2015-07-03 }}</ref>。《资治通鉴·后梁纪一》注：“汶川，汉绵虒地，晋置汶川县，……汶读曰岷。”，所以岷江亦读作汶江<ref name="one">{{cite web |title = > 汶川资讯 > 汶川概况 > 历史沿革 |url = http://www.wenchuan.gov.cn/NewsDefault.aspx?TabId=www_14 |website=汶川县政府网站 |accessdate=2015-07-03 }}</ref>。汶与岷在古汉语中为同地异称，古籍中关于这个字意时，岷与汶两字也相通，《中国地名手册》中也认为古汶与岷相同<ref>{{cite journal |author = 李旭芳 |title=“汶川”之“汶”读音小考 |journal=《阅读与写作》 |year = 2010年 |issue=10 |page=第33页 }}</ref>。}}是[[中华人民共和国|中国]][[四川省|四川省]][[阿坝藏族羌族自治州|阿坝藏族羌族自治州]]辖[[县|县]]。县治威州镇，是中国四个[[羌族|羌族]]聚居县之一，[[汉朝|汉朝]]时称'''绵虒'''。汶川县因[[汶水|汶水]]得名，据《元和郡县志》：“梁置汶川县，因县西汶水为名。”汶川县的映秀镇是2008年大地震的震中区域，2012年汶川灾后重建工程完毕。现代的汶川以工业与旅游业为其主导产业发展。

汶川县的[[地理坐标|地理坐标]]在[[北纬|北纬]]30°45′—31°43′与[[东经|东经]]102°51′—103°44′之间，东西宽84公里，南北长105公里。县域面积4,084[[平方千米|平方千米]]；人口105,436人（2009年），主要民族为[[汉族|汉]]、[[羌族|羌]]、[[藏族|藏]]和[[回族|回族]]。东邻[[彭县|彭县]]、[[都江堰市|都江堰市]]，南接[[崇州市|崇州市]]、[[大邑县|大邑县]]，西界[[宝兴县|宝兴县]]与[[小金县|小金县]]，西北至东北分别与[[理县|理县]]、[[茂县|茂县]]相连。境内有[[卧龙自然保护区|卧龙自然保护区]]，为[[大熊猫|大熊猫]]的研究和主要繁殖地，[[四姑娘山|四姑娘山]]为主要旅游景点。

==地理==
汶川，地处[[四川盆地|四川盆地]]西北缘，距离成都约160公里，距离都江堰市约90公里。[[龙门山脉|龙门山脉]]和[[邛崃山|邛崃山脉]]分别位于县域东北与西南部；[[岷江|岷江]]及其支流[[杂谷脑河|杂谷脑河]]、[[草坡河|草坡河]]、[[寿江|寿江]]为境内主要河流，属于岷江[[水系|水系]]。汶川西部的[[四姑娘山|四姑娘山]]海拔6250米，为最高点，汶川最低点位于东南方海拔780米的[[漩口镇|漩口镇]]，林地面积26.7hm²，森林活木蓄积总量为2123.8m³{{cfn|李浩|y=2008|p=81}}。汶川以及茂县等汶川地区属于龙门山断裂带部分，至今仍处于活动状态，汶川县境内分布有龙门山断裂带的活动构造茂汶韧性断裂带，与另两断裂带成叠瓦状排列。

汶川县地质构造复杂，地层发育完整，岩浆岩分布广，[[矿产资源|矿产资源]]丰富，特别是非金属矿产品种较多。其中汶川县[[花岗石|花岗石]]储量约75亿m³，[[铁|铁]]矿储量约25445亿吨以及铁铝石榴石储量约为5000万吨{{cfn|李浩|y=2008|p=80}}。[[气候|气候]]随东南向西北地势上升，呈比较完整的垂直，可分为8个不同的[[自然气候区|自然气候区]]以及有暖温带、温带、寒温带、亚热带和寒带气候带，故有“十里不同天”之说<ref> {{cite web |url=http://travel.newssc.org/system/2006/08/17/010094632.shtml |title=熊猫故乡：汶川县 |accessdate=2008-05-13 |date=2006-08-17 |publisher=四川新闻网 |language=zh-hans }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==历史==
汶川地区出土了多处[[新石器时代|新石器时代]]遗址。位于汶川县城威州北端的姜维城遗址经考古确认为岷江上游地区的新石器时代遗址，该遗址延长时期长分三期。估计是黄河上、中游地区[[仰韶文化|仰韶文化]]中晚期的一支的传入岷江上游产生的新文化类型。[[彩陶|彩陶]]片纹饰方面和发掘的石棺葬的古墓与甘肃、青海地区的[[马家窑文化|马家窑文化]]有相似之处{{cfn|李浩|y=2008|p=80}}。姜维城遗址与成都平原的遗址出土的彩陶器、陶罐等属于不同谱系，但也出土了同样谱系的花边口延砂罐{{cfn|黄家祥|y=2006|p=14}}。

汶川因县西汶水（现[[岷江|岷江]]上游）得名，据《[[尚书_(书)|尚书]]》<禹贡>记载汶川地区在夏商时期隶属于[[梁州|梁州]]。[[西汉|西汉]]元鼎六年（公元前111年）时置汶川郡辖绵虒县等5县，绵虒县的县治与今县治所在同一地点。《[[后汉书|后汉书]]》记载“蜀郡……绵虒道”<ref>[[s:後漢書/卷113|後汉书 志第二十三 郡国五]]</ref>。[[蜀汉|蜀汉]]为汶山郡郡治，[[西晋|西晋]]改名汶山县，[[南北朝|南朝]][[齐建|齐建]]元年（公元449年）时期包括汶川在内的北川大部分区域由[[氐人|氐人]]所建的[[阴平国|阴平国]]统治，[[北周|北周]]天和三年（568年）始名汶川县，距今已有一千四百余年历史。

前后蜀（公元907～965年）时期，今汶川岷江以西地区由吐蕃占据。宋朝时期曾在汶川地区置威戎军。元代时汶川地区为安抚司后改为为宣抚司以及之后的宣抚司，隶属于[[陕西等处行中书省|陕西等处行中书省]]。明代汶川地区划入四川，明初时在绵虒地区置汶川长官司。在[[明英宗|明英宗]]时期（15世纪前期）汶川地区曾多次爆发反抗明廷的起义运动。

在民国期间由[[川军|川军]]28军军长邓锡侯兼督办，设“松理懋茂汶屯殖督办署”管辖汶川区域。1933年8月25日，在汶川以北的茂县叠溪镇发生了一场[[黎克特制|黎克特制]]7.5级的[[叠溪大地震|大地震]]，在汶川有6人因山体崩塌被岩石击中身亡，地震后的次生水灾中汶川有480多人人遇难<ref>{{cite journal|author1=洪时中|title=对叠溪地震死亡人数的一些考证与分析|journal=《四川地震》|date=2014年9月|issue=3|pages=第7页至第8页|accessdate=2016-04-04}}</ref>。

1950年1月14日，中国人民解放军进驻汶川县城绵虒，2月24日正式成立汶川县人民政府，隶属茂县专区。2月1952年，县城由绵虒迁威州；1958年[[茂县|茂县]]、汶川县、[[理县|理县]]合并成立“茂汶羌族自治县”；1962年恢复汶川县至今。

==基础设施==
=== 经济 ===
在20世纪中期汶川地区包括汶川均以[[农业|农业]]为主导的县份，在1950年汶川县农业占比97.9%，社会生产总值为456万元。汶川县工业的发展从当地丰富的资源和当地小型的工业生产的基础开始。1957年建成了方解石厂和煤矿，随着之后电能的开发，1976年汶川县工业在工农业总产值（1421万元）的27.6%。1978年随着国家经济层面的[[改革开放|改革]]，汶川工业发展迅速，并形成了较为完整的工业体系，汶川县的中小形乡村企业也有较大发展。因此，汶川县工商税从1950年的1.44万元增长到了1985年的720.59万元，县的财政也基本实现了自给。到2000年汶川县工业总产值达到1.26亿元，占工农业总产值的60.9%{{cfn|李浩|y=2008|p=83-84}}。汶川也成为了[[阿坝州|阿坝州]]的工业经济走廊，在2007年实现工业总产值40.47亿元，工业比重占全州的70%多{{cfn|王晓倩|y=2010|p=329}}。

2008年大地震对汶川经济产生直接损失139.1亿元，工业减值达到67.2%。2010年汶川地区生产总值比上年增长37.8%，自2009年和2010年增速激增后的2011开始增速放缓，2011年增长为10.7%。2013年在遭受特大山洪泥石流灾害后仍比上年增长了3.9%，2014年全县实现地区生产总值549934万元，工业增加值358158万元，工业化率达65.1%，旅游总收入289665万元，比上年增长10.2%，全年城镇居民人均可支配收入为24153元，全年农民人均纯收入8835元<ref>{{cite web|author1=汶川县统计局|title=2014年汶川县国民经济和社会发展统计公报|url=http://www.wenchuan.gov.cn/p/st_news_items_i_9bac1d64a5b54241874e7b41a584c3a0/|website=汶川县人民政府网|accessdate=2016-01-15|date=2015-05-04}}</ref>。2015年四川省划分的全省五大经济区中，汶川县处于[[川西北经济区|川西北经济区]]。

汶川县有丰富的自然景观以及地震旅游资源，同时汶川是阿坝州的东南门户，也是前往卧龙、九寨黄龙的必经之地，也是距离成都及其周边城市最近的羌族聚居县。羌族文化上，在震后国家设立了国家级羌族文化生态保护实验区。汶川县主要旅游景点如下：
*[[卧龙自然保护区|卧龙自然保护区]] - 为[[大熊猫|大熊猫]]的研究和主要繁殖地
*[[四姑娘山|四姑娘山]]

*[[汶川博物馆|汶川博物馆]] - 震后汶川县最大的文化援建项目，主要包括：地震文物、历史文物、民俗文物、革命文物。一级文物11件，二级文物4件，三级文物193件，一般文物12083件。

同时也因为汶川县工业的快速发展，汶川县自然生态环境也遭受极大的破坏。在2008年大地震前，国土资源部在生态破坏区域调查出有4929处大型地质灾害隐患点。地震后汶川县转移了部分城镇的功能，调整了城镇空间结构，形成纵横发展的两轴，纵轴主要为城镇建发展轴，横轴为生态保护和遗址旅游发展轴，并划分了五个不同发展的片区，以威州镇和雁门乡、水磨镇和漩口镇为汶川的南北中心，原重工业发达的映秀镇调整为地震旅游及纪念中心、公交换乘中心和水电服务基地，重要工业区则转向漩口镇{{cfn|李运哲|y=2012}}。震后汶川建设起的广东现代产业园区承接广东省的产业转移，为汶川引进人才、资金和技术管理。同时援建项目还建设了阿坝州最现代化和大型的覆盖汶川县城的污水处理厂。自2015年来汶川县坚持“产业融合发展”的理念，推动汶川可持续的发展，计划打造集农业生产与农业观光、旅游休闲与康体疗养为一体的生态康养目的地。现在汶川已形成以三江镇、水磨镇、漩口镇为核心、1000平方公里的生态旅游目的地<ref>{{cite web|title=汶川：生态与产业“联姻”|url=http://www.wenchuan.gov.cn/p/st_news_items_i_878e362faa8c4aabaca6e7f296c42853/|website=汶川县政府网站|accessdate=2016-02-12|date=2015-10-03}}</ref>。

=== 交通设施 ===
汶川县的道路交通以[[国道213线|国道213线]]、[[国道317线|国道317线]]为主干，在2012年连接[[都江堰市|都江堰市]]、通往川内盆地城市的长82公里的[[都汶高速公路|都汶高速]]全线通车，汶川至[[马尔康|马尔康]]的高速公路[[汶马高速|汶马高速]]正在修建中。汶川县汽车站设在汶川威州镇，每天有发往[[成都|成都]]、[[都江堰|都江堰]]、[[彭县|彭县]]、[[理县|理县]]、[[马尔康|马尔康]]、[[九寨沟|九寨沟]]等地的客车。除了[[七盘沟树林公园|七盘沟树林公园]]和[[西羌大峡谷|西羌大峡谷]]尚未开放不通车外，到各景区皆可乘车到达<ref> {{cite web |url=http://www.57uu.com/sc/abz/abzjt/11_50_24_435.html |title=汶川县交通 |accessdate=2008-05-13 |publisher=西部旅行网 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20080517171143/http://www.57uu.com/sc/abz/abzjt/11_50_24_435.html |archivedate=2008-05-17 }}</ref>。2013年，覆盖映秀、漩口、水磨、三江等全县11个乡镇的公交线路全面开通<ref>{{cite web|author1=史付莉 卢国彤|title=2013年汶川县着力完善基础设施，改善城乡面貌|url=http://www.abazhou.gov.cn/jrab/zwyw/201402/t20140215_941845.html|website=中国阿坝州政府网站|accessdate=2016-01-15|date=2014-02-15}}</ref>。2014年末，汶川县境内公路总里程为696公里，高等级公路619公里，高速公路51公里。

铁路交通方面，汶川县并未通铁路，距离汶川县最近的大型[[火车站|火车站]]为[[都江堰站|都江堰火车站]]。在建中的[[成兰铁路|成兰铁路]]会经过汶川县北的茂县。
