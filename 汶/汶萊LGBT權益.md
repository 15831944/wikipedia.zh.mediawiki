'''[[文莱|文莱]]的[[LGBT|LGBT]]人群'''（即[[女同性恋者|女同性恋者]]、[[男同性恋者|男同性恋者]]、[[双性恋者|双性恋者]]和[[跨性别者|跨性别者]]）需要面对严峻的法律挑战，因为[[同性恋|同性恋]]在文莱是违法行为。男性之间的性行为将会遭致[[石刑|石刑]]处死；而女性之间的性行为则会面临[[监禁|监禁]]或[[鞭刑|鞭刑]]处罚。<ref name="dw" /><ref name="AFP" />{{Link-en|国际立即行动|OutRight Action International|国际立即行动组织}}认为文莱是“东南亚国家中，LGBT人群最担忧其权利的国家”，文莱的LGBT人群认为有必要对其性取向保持谨慎。<ref name="huffpost">{{cite web |author1=Dominique Mosbergen |title=Brunei’s LGBT Community Faces Terrifying Future |url=https://www.huffpost.com/entry/lgbt-brunei_n_561501f9e4b0fad1591a1167 |publisher=[[赫芬顿邮报|赫芬顿邮报]] |accessdate=2019-04-13 |archive-url=https://web.archive.org/web/20190409021410/https://www.huffpost.com/entry/lgbt-brunei_n_561501f9e4b0fad1591a1167 |archive-date=2019-04-09 |dead-url=no }}</ref>成立于2015年的“文莱项目（{{lang|en|Brunei Project}}）”旨在通过社交媒体促进文莱国内[[人权|人权]]，包括：[[信仰自由|信仰自由]]、[[言论自由|言论自由]]和LGBT权利等方面的进步。该组织曾于2016年组织起了一次私人社区活动，庆祝文莱首次举办反恐同活动。<ref>{{cite web |title=Welcome to IDAHOT Newbies! |url=https://dayagainsthomophobia.org/welcome-to-idahot-newbies/ |publisher=May17.org |accessdate=2019-04-13 |archive-url=https://web.archive.org/web/20190404053700/https://dayagainsthomophobia.org/welcome-to-idahot-newbies/ |archive-date=2019-04-04 |dead-url=no }}</ref>

== 同性恋合法地位 ==

当文莱宣布实施伊斯兰教法的时候，[[联合国|联合国]]曾敦促该国重新审视该法律，而许多媒体将这些法律斥为“中世纪”、“不文明”和“重返石器时代”等。<ref name="huffpost" /><ref name="beast" />但是文莱政府却宣布这些法律条款是源自真主的“特别指引”，但他们还是将法律的实施日期推迟到了2019年4月。<ref name="independent" />如果男性之间发生[[男男性行为|男男性行为]]，那么他们将被判处[[石刑|石刑]]。只要他们主动承认，或是有四人声明目击，即可对他们实施处罚。<ref name="bbc" />而如果是女性之间发生[[女女性行为|女女性行为]]，那么她们将被判处40鞭[[鞭刑|鞭刑]]或最高10年的[[有期徒刑|有期徒刑]]。<ref name="AFP" />

国际人权活动家建议文莱的LGBT人群以及[[基督徒|基督徒]]、[[佛教徒|佛教徒]]等少数群体在该国保持谨慎，因为“任何玷污伊斯兰教形象”的人都将遭受严惩。<ref name="beast" />

== 跨性别者权利 ==
文莱不允许在官方文件上修改自己的姓名或性别。<ref name="ILGA">{{cite web |title=ILGA Trans Legal Mapping Report 2017 |url=https://ilga.org/downloads/ILGA_Trans_Legal_Mapping_Report_2017_ENG.pdf |publisher=[[国际LGBTI联合会|国际LGBTI联合会]] |accessdate=2019-04-13 |archive-url=https://web.archive.org/web/20190403092250/https://ilga.org/downloads/ILGA_Trans_Legal_Mapping_Report_2017_ENG.pdf |archive-date=2019-04-03 |dead-url=no }}</ref>[[性别重置手术|性别重置手术]]也被禁止。<ref name="outright">{{cite web |title=LGBTIQ Rights in Southeast Asia - Where We Stand and Pathway Forward |url=https://www.outrightinternational.org/content/lgbtiq-rights-southeast-asia-where-we-stand-and-pathway-forward |publisher={{Link-en|国际立即行动|OutRight Action International}} |accessdate=2019-04-13 |archive-url=https://web.archive.org/web/20190403104508/https://www.outrightinternational.org/content/lgbtiq-rights-southeast-asia-where-we-stand-and-pathway-forward |archive-date=2019-04-03 |dead-url=no }}</ref>

2015年3月11日，文莱依据《2013年伊斯兰刑法（{{lang|en|Syariah Penal Code Order}}）》判处一名公务员因[[变装|变装]]而罚款1000[[文莱元|文莱元]]。<ref>{{cite news |last=HARUN |first=AK MD KHAIRUDDIN PG |last2=BEGAWAN |first2=BANDAR SERI |url=http://www.bt.com.bn/news-national/2015/03/11/bruneian-civil-servant-fined-1-000-cross-dressing |title=Bruneian civil servant fined $1,000 for cross-dressing |work=文莱时报 |date=2015-03-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20150313161723/http://www.bt.com.bn/news-national/2015/03/11/bruneian-civil-servant-fined-1-000-cross-dressing |archivedate=2015-03-13 |accessdate=2019-04-13 }}</ref><ref>{{cite web |title=Brunei |url=https://www.humandignitytrust.org/country-profile/brunei/ |publisher=Human Dignity Trust |accessdate=2019-04-13 |archive-url=https://web.archive.org/web/20190403203116/https://www.humandignitytrust.org/country-profile/brunei/ |archive-date=2019-04-03 |dead-url=no }}</ref>

== 生存状况 ==
文莱的LGBT社区非常隐蔽和秘密，而文莱社会倾向于将同性恋者与“娘娘腔”联系到一起。<ref name="GlobalGayz">{{cite web |title=Gay Life in Brunei |url=https://www.globalgayz.com/gay-life-in-brunei/ |publisher=GlobalGayz.com |accessdate=2019-04-13 |archive-url=https://web.archive.org/web/20190330052412/https://www.globalgayz.com/gay-life-in-brunei/ |archive-date=2019-03-30 |dead-url=no }}</ref>

2011年，{{Link-en|文莱达鲁萨兰国大学|Universiti Brunei Darussalam}}的学者启动了一项针对文莱[[同性恋者|同性恋者]]的调查研究。该研究说明了文莱的同性恋者是如何对自己[[性取向|性取向]]选择沉默或保持谨慎态度，研究者只在文莱找到了29名同性恋者，而其中一部分还并非文莱公民。<ref name="GlobalGayz" />
