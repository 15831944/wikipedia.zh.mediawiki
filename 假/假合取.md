'''假合取'''（{{lang-en|False conjunction}}），又稱'''捆包謬誤'''（{{lang-en|Package-deal fallacy}}），指的是假定在傳統或文化上常常會分類在一起的事物必然會一起出現的謬誤。

一個例子是「他是反對[[社福|社福]]和加[[稅|稅]]的[[美國|美國]][[保守派|保守派]]人士，因此他也反對[[槍枝管制|槍枝管制]]和[[墮胎|墮胎]]」，盡管在美國，「反對社福」、「反對加稅」、「反對槍枝管制」和「反對墮胎」都被認為是[[保守派|保守派]]的政治觀點，但這不表示說在美國相信其中一種保守派觀點的人，也必然會相信其他保守派的觀點。

假合取可說是誤用了「且」的推理；關於誤用了「[[邏輯異或|或]]」的推理，可見[[假兩難推理|假兩難推理]]。

==其他例子==
;例一
: 這個[[蘿莉|蘿莉]]是[[雙馬尾|雙馬尾]]，所以一定是[[傲嬌|傲嬌]]。
: 說明：「雙馬尾」、「蘿莉」和「傲嬌」是常常一起出現的[[萌屬性|萌屬性]]，但這不表示說雙馬尾蘿莉總是傲嬌角色。

;例二
: [[濟公禪師|濟公禪師]]是有修為的[[和尚|和尚]]，因此一定是光頭，而且一定很守佛門清規，後世把他做成披頭散髮、吃肉喝酒的形象，一定是搞錯了！
: 說明：由於傳統上多數的佛教和尚和尼姑出家後都會剃光頭的緣故，和尚和光頭是常常一起出現的屬性、有修為和恪守清規也常常在一起，但這不表示和尚一定是光頭，有修為的和尚也必然恪守清規，也有像濟公禪師這種不拘小節、不剃髮但確實有修為的僧人存在。

;例三
: [[司馬光|光]]言唯[[青苗法|青苗]]、[[免役法|助役]]為[[陕西路|陜西]]之患
: 說明：青苗法和助役法都是[[王安石變法|王安石變法]]的一部分，但這不表示所有的新法法案都總是一起出現。

;例四
: 他有[[亞斯伯格症|亞斯伯格症]]，又是高智商，所以一定很適合寫[[程式|程式]]。
: 說明：亞斯伯格症患者擅長搞電腦程式是一個常見的關聯，但這不表示亞斯伯格症加上高智商的人一定適合寫程式。

;例五
: 她是[[女生|女生]]，又是[[巨乳|巨乳]]，所以[[數學|數學]]成績一定很差。
: 說明：社會上常常把女性以及女性外表的某些特徵和數理方面的負面說法連結起來，但是是女生又是巨乳不表示數學成績一定很差，況且女生不擅長數學以及胸大無腦等說法，涉及[[刻板印象|刻板印象]]和[[性別歧視|性別歧視]]。

;例六
: 他支持[[死刑|死刑]]、支持[[體罰|體罰]]、支持[[髮禁|髮禁]]，所以一定支持[[威權|威權]]體制。
: 說明：死刑、體罰和髮禁常被認為和威權體制相關，但這不表示支持這些事物的也一定支持威權體制。

;例七
: 白勇太是男的，小橘子是女的，白勇太和小橘子是很要好的朋友，又是同一個賽車車隊的，所以白勇太和小橘子一定是一對情侶。
: 說明：一對感情很要好且常常在一起的男女常常也是情侶，但這不表示他們一定是情侶。

;例八
: 你數學很厲害，還拿到美國名校的數學博士，你說你不會電腦程式、不會統計，因此不敢應徵[[大數據|大數據]]相關產業的工作？少騙人了，你一定很會電腦程式，以你的學歷你一定能找到很好的工作的！
: 說明：盡管一般認為數學系的電腦程式也很強，但數學很厲害，不代表電腦程式也很厲害。

;例九
: 他是某大學法律系的高材生，邏輯一定很強
: 說明：法律系的學生邏輯很是一個刻板印象，但這不表示法律系的邏輯一定很強。

==不是謬誤的狀況==
如果講的是在傳統或文化上常常會分類在一起的事物，「可能」而不是「必然總是」會一起出現，那就不算是謬誤。

一個不是謬誤例子是「廣志喜歡科幻電影，因此他'''可能'''也喜歡[[星際大戰|星際大戰]]」；另一個例子是「他是反對[[社福|社福]]和加[[稅|稅]]的[[美國|美國]][[保守派|保守派]]人士，因此他'''有可能會'''反對[[槍枝管制|槍枝管制]]和[[墮胎|墮胎]]」
