'''假戰'''（{{lang-en|Phoney War}}），'''怪戰'''（{{lang-fr|Drôle de guerre}}），又稱為'''静坐战'''（{{lang-de|Sitzkrieg}}），是指1939年9月開始到1940年4月之間，[[英国|英]][[法国|法]]雖然因為[[納粹德國|納粹德國]]對[[波蘭|波蘭]]的入侵而宣戰，可是兩方實際上只有極輕微的軍事衝突。

這段期間雙方有小規模的空中與海上的衝突，英國開始將遠征軍調動至法國，法國則開始進行一些動員的工作。1940年初法國有14個師防守有名的[[馬奇諾防線|馬奇諾防線]]，同時有80個師在後方予以支援，英國陸續增援到9個師的部隊到了1940年5月英法共集結超過3,100輛[[戰車|戰車]]與2,000多架[[戰鬥機|戰鬥機]]。

當時德軍正與波蘭交戰只有少量軍隊駐守與法國交接的[[齊格菲防線|齊格菲防線]]，在另一端的[[馬奇諾防線|馬奇諾防線]]，英法聯軍與德軍相對，但只爆發零星交戰。[[英國皇家空軍|英國皇家空軍]]向德國投擲宣傳單，[[加拿大軍隊|加拿大軍隊]]也踏上不列顛，然而西歐在7個月間並無任何戰爭，直至兩個陣營終於在[[北歐戰役|北歐戰役]]中爆發衝突，結束這一段沒有戰爭的宣戰期。在重整軍備期間，驚惶的英法除依靠本身的生產外，都向美國購買大量武器。美國作為[[非參戰國|非參戰國]]，向英法租借及提供折扣出售武器。然而在1930年代，英美的私人公司曾隱瞞政府，向德軍提供武器，德軍戰機的引擎在英國製造，而原材料則購自美國。德國為了阻止這跨越大西洋的交易，觸發了[[大西洋戰爭_(二戰)|大西洋戰爭]]。

== 薩爾衝突 ==
{{Main|薩爾衝突}}

1939年9月，在純政治考量下，法軍向[[國際聯盟|國際聯盟]]轄下的[[薩爾|薩爾]]推進了3英里。法軍動用98個師和2,500輛坦克，而德軍動用43個師，沒有坦克。

== 冬季戰爭 ==
{{Main|冬季戰爭}}

在西歐假戰期間，[[蘇聯|蘇聯]]於1939年11月30日入侵[[芬蘭|芬蘭]]，東歐爆發了[[冬季戰爭|冬季戰爭]]。普遍輿論都支持民主國家芬蘭，由於芬蘭的抵抗似乎比波蘭[[德波戰爭|對抗德軍]]更成功，英法國民甚至要求其政府支援芬蘭對抗強大的蘇聯。於是，蘇聯被驅逐出國際聯盟，英法兩國曾計劃遠征[[斯堪的納維亞|斯堪的納維亞]]。英國開始征召軍隊，但於冬季戰爭結束前仍未到達芬蘭，便改為在[[北歐戰役|北歐戰役]]中支援[[挪威|挪威]]。1940年3月，冬季戰爭結束後，法國總理[[達拉第|達拉第]]因支援芬蘭失敗而引咎辭職。

== 德軍入侵丹麥、挪威 ==
{{Main|北歐戰役}}

[[同盟國_(第二次世界大戰)|同盟國]]公開討論遠征[[斯堪的納維亞|斯堪的納維亞]]北部，卻得不到中立的北歐國家同意；2月的[[阿爾特馬克事件|阿爾特馬克事件]]影響德國鐵礦供應，德國便強烈要求保衛挪威海岸。4月9日，德國發起「[[威瑟演習作戰|威瑟演習作戰]]」，開始入侵丹麥和挪威，丹麥隨即戰敗。4月12至15日間，盟軍登陸挪威，但德軍在兩個星期內佔領整個挪威，並把盟軍驅逐出境。

== 英國變天 ==
{{Main|挪威爭議}}

英國因支援芬蘭失敗，而改為支援挪威，其帶來的災難使[[英國首相|英國首相]][[內維爾·張伯倫|張伯倫]]在[[英國下議院|下議院]]備受攻擊。雖然下議院以281對200通過對張伯倫政府的[[信任動議|信任動議]]，但其支持者很多都轉投了反對票。張伯倫感到失去支持，難以繼續領導英國政府或另組[[聯合政府|聯合政府]]，便於5月10日辭任首相一職。英國國王[[喬治六世|喬治六世]]委任反對[[綏靖|綏靖]]的[[邱吉爾|邱吉爾]]出任首相。邱吉爾重組聯合政府，包括[[保守黨_(英國)|保守黨]]、[[工黨_(英國)|工黨]]、[[自由黨_(英國)|自由黨]]和一些無政黨人士。

當日，即英法對德[[宣戰|宣戰]]後8個月，德軍進攻[[比利時|比利時]]、[[荷蘭|荷蘭]]和[[盧森堡|盧森堡]]，結束了假戰。

==影响==
假战是[[二战|二战]]时期西方[[绥靖政策#西方国家的绥靖政策|绥靖政策]]的延续。它使[[法西斯|法西斯]]的侵略野心进一步膨胀，削弱了[[反法西斯同盟|反法西斯战线]]的力量，并间接导致了[[法国|法国]]的沦陷。
