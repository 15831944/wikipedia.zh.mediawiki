'''假离婚'''是完成[[法律|法律]]上的[[离婚|离婚]]程序，但仍维持实质上的夫妻关系。

== {{PRC}} ==

[[中华人民共和国政府|中华人民共和国各级政府]]或相关机构制定的[[公共政策|公共政策]]<ref>{{Cite web |url=http://finance.ifeng.com/a/20170620/15470031_0.shtml |title=假离婚现象高发，该反思的是政策 |accessdate=2017-09-04 |author= |date=2017-06-20 |work= |publisher=凤凰网|language=Zh-hans }}</ref>造成单身人士比已婚者获得更多权益，是造成假离婚在中国大陆高发的因素。中国大陆常见的假离婚与[[假结婚|假结婚]]类似，发生于[[房屋|房屋]][[征收|征收]]、购置[[房产|房产]]等过程中。

在房屋交易过程中，获得购房资格<ref name="a1"/>、降低[[房屋贷款|房屋贷款]]的[[首付款|首付款]]比例和[[节税|节税]]是个人选择假离婚的重要原因。2016年2月22日，实行的[[中华人民共和国财政部|中华人民共和国财政部]]、[[国家税务总局|国家税务总局]]、[[中华人民共和国住房和城乡建设部|中华人民共和国住房和城乡建设部]]《关于调整房地产交易环节契税、营业税优惠政策的通知》后，针对已婚人士的房产税下调，假离婚不再起节税作用，但仍能降低首付款比例<ref>{{Cite web |url=http://news.163.com/16/0506/08/BMCCRQDP00014Q4P.html |title=为买房逃税假离婚 小心人房两空 |accessdate=2018-05-09 |author= |date=2016-05-06 |work= |publisher=网易，来源：大江网-江南都市报(南昌)|language=Zh-hans }}</ref>。2017年来自北京的假离婚实例中，一对夫妻通过假离婚，取得购房资格，由此得到了超过一百万元人民币的经济收益，获得近十年的个人收入<ref name="a1">{{Cite web |url=http://news.ifeng.com/a/20170622/51301786_0.shtml |title=中国式假离婚：“一次就挣一百万，后悔没再离一次” |accessdate=2018-05-09 |author= |date=2017-06-22 |work= |publisher=凤凰网，来源：Vista看天下|language=Zh-hans }}</ref>。与之相比，2016年北京市全市居民人均可支配收入仅5.2万元<ref>{{Cite web |url=http://www.bjstats.gov.cn/zxfb/201701/t20170122_367497.html |title=2016年我市居民收支特点 |accessdate=2017-09-04 |author= |date=2017-01-22 |work= |publisher=北京市统计局、国家统计局北京调查总队网站|language=Zh-hans }}</ref>。
