== 假面騎士 ==
=== 本作假面騎士 ===
'''{{Nihongo|常磐莊吾|常磐 ソウゴ（ときわ・ソウゴ）}}'''（{{tsl|ja|奥野壮}}、[[高橋琉晟|高橋琉晟]]（幼年期）飾，大陆配音：[[李兰陵|李兰陵]]）<ref>因其名字為片假名，所以劇集初期並沒有正式的漢字譯名，有莊吾及宗吾等之説。直至《假面騎士時王 補完計劃》第5.5話才確認其漢字譯名為莊吾。</ref>
: '''假面騎士ZI-O'''、'''逢魔時王'''的變身者，2000年4月28日生，18歲。
: 就讀光之森高等學校高中三年級，擅長科目是各種歷史，但不擅長科目是化學和物理<ref>第1話</ref>，因此使用Build裝甲必殺技時所出現的彈幕並不是數學算式，而是對理科不滿的感想<ref>出現的彈幕內容亦令戰兔感到不快</ref>。
: 同時亦是一個遊戲白癡，玩遊戲時經常Game Over。所以雖然可變成EX-AID裝甲對抗另類EX-AID，但使用必殺技時因不懂「Mighty Action X」的玩法而令永夢覺得無奈。
: 在高中的進路指導中也說「我要成為王」的高中3年生，即使陷入苦境也會笑着面對，擁有正面的思想。
: 不知說是天才還是天然呆，只要說了「這樣的事情感覺能夠完成」这句座右铭，不論是甚麼事情也能很好地做完。
: 於第15話初次披露夢想成為王的原因：兒時夢到巨大的機器人<ref>即逢魔時王用來破壞并征服世界的大魔神</ref>四處破壞，幼小的自己無能為力時一位神秘男子出現并告之其「你生來註定要成為王、拯救世界於破滅之中」，第28話中了解到实际上是在巴士惨剧当天被斯瓦鲁兹连同随车的所有小孩带走，在大魔神四处破坏的残骸即将掉落砸死昏迷中的加古川飞流之时无意间爆发能力而被斯瓦鲁兹看中，对其说了「你生來註定要成為王、拯救世界於破滅之中」这句话后消除其记忆并将其与加古川飞流送回车祸现场。
: 隨著故事的進展越發展現出作為王的氣度和領導能力，於第8話起已能夠出色地同時指揮月津和沃茲的行動并協助自己解決另類騎士的事件：
: 在第9話先後利用檀黎斗王和月津的性格特點制定調查檀黎斗王的計劃（假意投誠而使月津以為自己選擇成為魔王而主動挑起戰鬥、并通過打倒月津博得檀黎斗王的信任），作為ZI-O亦展現出壓倒Geiz的強橫戰鬥能力。
: 於第11、12話，三日後的莊吾回到三日前獨自一人制定營救月津的計劃，並未經歷過被來到三日前的自己改變的歷史卻能推斷出鎧武手錶會被丟至海姆冥界<ref>因沃茲對自己想要營救月津的態度不滿而向時劫者洩密。</ref>甚至據此令月津與自己取得聯繫、用激將法<ref>並非對他說「要幫助你」而是「你要阻止我成為魔王」這個月津一直在堅持的目標</ref>令月津以自己的力量逃離海姆冥界，展現出令人驚訝的行動力和對所有人性格了如指掌。
: 第13話因追擊另類Ghost途中被假面騎士DECADE打敗，甚至靈魂被擊出身體，後在遭到另類Ghost吸收自己靈魂之際，得到天空寺尊作法幫助下將自身靈魂飛離，避開一劫。
: 第14話教導尊使用時間魔神穿越自2015年，並在2015年擊敗另類Ghost後靈魂返回肉體内。
: 第15話親眼目睹未來的逢魔ZI-O所操縱的大魔神在各國主要城市破壞，原本欲搭救被家臣襲擊的月津，卻遭到假面騎士Decade攻擊而來到2068年、與當時的自己=逢魔ZI-O對峙。
: 第16話因為知道自己的未來後要求月津把自己的時空驅動器破壞，但後被時刧者追殺，之後因月津保證<ref>及因戰力不足</ref>若自己踏入成為逢魔ZI-O的路的話會被其拼盡全力地去打敗而再次成為假面騎士ZI-O。
: 有透過夢境創造未來的能力<ref>雖然假面騎士Shinobi和假面騎士Kikai只是在莊吾的夢境中出現，但夢境之後另類Shinobi和另類Kikai確實在該作的時間段出現。</ref>，但自己不知情，於第24話被月津和月讀得知。
: 第28話在另類ZI-O事件的最後與月津互道心聲后冰釋前嫌，將月讀與月津二人帶回九時五時屋。
: 第30話向白沃茲挑明“若要讓未來成為自己想要的樣子就應努力戰鬥并創造”后接受了心意改變的白沃茲託付的ZI-O三重手錶，與月津和沃茲合體為ZI-O三重形態時使軒轅十四閃耀，被黑白沃茲認為是新的“逢魔之日”。
: 即使突然在眼前出現的月讀向他告訴了「你在50年後將會作為魔王君臨世界」這樣的未來後，也很平靜地接受（因為未知2068年的自己=逢魔ZI-O有多麼可怕）。後於第15話親身面見2068年作為魔王的自己時顯得極為痛苦而不願接受。
: 為了避免成為魔王，而是成為「能夠實現人們的幸福、最高最善的王」，這樣的夢想而促使他決定成為假面騎士ZI-O戰鬥。
: 將月讀和月津視為同伴，并為向他們表訴中心而說出「假如我真的成為魔王、無論什麼時候也可以來打倒我」的發言。
: 本人有著天生的“脫離感”，即並不過度參與事件與人際交流，而是以近似旁觀者的態度觀察身邊人的個性和言行，并根據相關信息獨自一人制定計劃并實施<ref>這一點在第11、12話得到成為神的葛葉纮汰點撥而知曉相信夥伴的重要性后有所好轉。</ref>。
:名字由來是作者[[石之森章太郎|石之森章太郎]]老師以前住過的公寓[[常盤莊|常盤莊]]

'''{{Nihongo|明光院月津|明光院 ゲイツ（みょうこういん・ゲイツ）}}'''（{{tsl|ja|押田岳}}飾，大陆配音：[[刘明月|刘明月]]）
: '''假面騎士Geiz'''的變身者，18歲。
: 來自50年後的未來的反抗軍的戰士之一，而時空驅動器還有Geiz、Ghost和Drive 騎士手錶從逄魔時王偷來<ref>第13話月津提及，現實情況是本劇拍攝時，[[西銘駿|西銘駿]]因拍攝電影和舞台劇，與本劇拍攝行程有衝突，故Ghost騎士手錶先於Ghost篇出現。</ref>，在與魔王的戰鬥中經歷了一番苦戰。因此決定要在魔王覺醒之前就消滅他，為了奪去還没成为魔王的莊吾的生命而現身2018年的世界。但被同樣來自2068年的世界的月讀說服了「有必要監視莊吾會否成為真正的魔王」，因此不情願地同意了。
: 在第2話至第9話之間和月讀一起生活在莊吾居往的九時五時屋的2樓，直到第10話自行搬走，隨後在第12話時再次回到九時五時屋居住。
: 第11話有與時劫者合作的意願，但因無法接受其放任另類鎧武傷人反被其放逐至海姆冥界，並遇到了同被放逐於此的驅紋戒斗，而第12話時和戒斗成功發現了通往現實的缺口，在其激發下成功逃脫並將被丟至海姆冥界的鎧武手表還給莊吾。
: 雖然對莊吾為解決另類騎士的事件而積極變身為假面騎士一事感到不快，但總是有意無意地說出與解決事件有關的話語從而引導天賦異稟的莊吾發現線索。
: 對莊吾始終充滿敵意，只要後者有稍許與自身主張相違背的行動就會認為其有走上魔王之路的可能性而惡語相向甚至大打出手。於第12話結尾態度已有所緩和。於第19話起因長期的相處而猶豫，於第24集得知他有透過夢境創造未來的能力
: 第13話因追擊另類Ghost途中被假面騎士Decade打敗
: 於第15話開頭勸說莊吾捨棄時空驅動器以規避成為逢魔時王的未來未果，之後因大魔神和家臣先後現身而徹底覺悟要殺死莊吾，為此更不惜藉助身為時劫者協力者的門矢士的力量。
: 第22話後段因追擊另類龍牙而死亡，幸莊吾用ZI-O II騎士手錶倒流時間而逃過一劫。
: 第26話目擊2009年車禍事件，認為月讀是因為自己的搖擺不定導致下落不明而下定決心，成功使用Geiz Revive手錶變身。但事后被黑沃兹告知Geiz Revive手錶的原理是压缩时间并透支其生命力，而使用Geiz Revive手錶数次后出现了七窍流血的症状。
: 第28话与庄吾冰释前嫌后和月读两人搬回九时五时屋，但是对于黑沃兹也入住九时五时屋感到不满。
: 第29話由月讀之口透露與黑沃茲關係不佳的原因，並在之後的30話互道心聲，同時也表示想要見證自己與莊吾能創造怎樣的未來。

'''{{Nihongo|白沃茲|白ウォズ}}'''（{{tsl|ja|渡邊圭祐}}二飾，大陆配音：[[皇贞季|皇贞季]]）

'''{{Nihongo|黑沃茲|黑ウォズ}}'''（{{tsl|ja|渡邊圭祐}}飾，大陆配音：[[皇贞季|皇贞季]]）

:: 但在第8話莊吾放棄Wizard手錶的繼承權時仍然表示莊吾作為魔王的力量增強并已超出他能力的測量範圍，其判定標準尚未明確。

: 於第7話救下被斯瓦鲁兹扔下楼的月津，因为以前与月津有过交情<ref>29话中由月读透露其此前为反抗军，且为月读与月津所在小队之队长，自愿接受前往逢魔時王身边担任卧底之任务，但放出假情报引诱反抗军进攻导致反抗军伤亡惨重，直接导致月读与月津回到2018年寻找庄吾。</ref>。
:: 但隨後便多次強調因月津與莊吾的成王之道並無關係而無需在乎他的生死。
: 於第8話起已被莊吾摸清行動模式而成為隨時隨地應莊吾召喚而現身解答問題的角色，最初的些許不滿情緒因莊吾承認其為助力者而消退。
: 「逢魔降临-{曆; zh-hk:曆; zh-cn:历;}-」一書似乎只記載了「莊吾正確地成為了魔王」的歷史，因此沃茲在面對書上未有記載的未來時會陷入迷惑<ref>例如第11、12話因三日後的莊吾回到現在而使歷史發生改變、發生沒有記載的事時。更有甚者因為這次的歷史改變而令原本與莊吾成王無關的月津出現在書中也令沃茲大為驚訝。</ref>。
: 正因爲三日後的莊吾回到現在而改變歷史從而令到其不滿，因而於第13話開始與時劫者合作。<ref>但第15話2018年的莊吾被送往2068年時卻並無慌亂反而大為興奮。</ref>
: 第17話開始時出現在莊吾等人面前解釋白沃茲，並説出自己的顧慮 ----- 「逢魔降临-{曆; zh-hk:曆; zh-cn:历;}-」沒有記載到的未來<ref>因爲按照原本的歷史，在假面騎士ZI-O之後的未來並沒有出現假面騎士，不過卻因爲歷史改寫的關係而出現了假面騎士Shinobi、假面騎士Quiz和假面騎士Kikai。</ref>，而且在白沃茲出現時與其對峙。

: 第29話由月讀之口透露與月津關係不佳的原因，並在之後的30話互道心聲，見證了莊吾變身ZI-O三重、迎來新的逢魔之日后與白沃茲交換意見并目睹其消散。
