'''富山號誌站'''位於[[臺灣|臺灣]][[臺東縣|臺東縣]][[大武鄉_(台灣)|大武鄉]]，曾為[[臺灣鐵路管理局|臺灣鐵路管理局]][[南迴線|南迴線]]的鐵路號誌站。

== 車站構造 ==
*岸式月台兩座。
*因通車後運量不大，車次不多，海側軌道在南迴線通車後不久即拆除，僅保留一股通過線。

== 利用狀況 ==
*目前已廢除。

== 車站周邊 ==
*富山部落舊址（[[八八水災|八八水災]]後已遷大武國小舊址，並正名[[斯卡拉比部落|斯卡拉比部落]]）
*[[太平洋|太平洋]]。
*距[[File:TW_PHW9.svg|23px]][[台9線|台9線]]426.5公里岔路口約 250 公尺。

== 公車資訊 ==
由號誌站聯絡道路走到台9線路口即有站牌，站牌為富山
*[[鼎東客運|鼎東客運]](山線)：

== 歷史 ==
*原計畫為[[號誌站|號誌站]]，但1991年[[南迴線|南迴線]]通車時並未實際使用<REF name="最美遺落車站"/>。
*[[屏東車站|屏東車站]]曾短暫販賣過至此站之普通、快車通用[[埃德蒙森式鐵路車票|名片式車票]]，被[[鐵道迷|鐵道迷]]視為珍品。
*台鐵曾安排郵輪式列車於富山站臨停觀光<ref name="最美遺落車站">{{Cite news|url=https://tw.news.yahoo.com/%E7%84%A1%E6%95%B5%E6%B5%B7%E6%99%AF-%E5%8D%97%E8%BF%B4-%E6%9C%80%E7%BE%8E%E9%81%BA%E8%90%BD%E8%BB%8A%E7%AB%99-194700531.html|title=無敵海景 南迴「最美遺落車站」|language=zh-tw|publisher=聯合新聞網|author=尤聰光、許俊偉|date=2014-12-14|accessdate=2014-12-14}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
