'''富兰克林·皮尔斯'''（'''{{lang|en|Franklin Pierce}}'''，{{bd|1804年|11月23日|1869年|10月8日|Pierce, Franklin}}），[[美国|美国]]政治人物、[[美國陸軍|陸軍]]退役[[准將|准將]]，[[民主黨_(美國)|民主黨]]黨員，第14任[[美国总统|美国总统]]（1853年－1857年）。皮尔斯是[[美国|美国]]第一位出生于19世纪的美国总统，任內試圖解決美國南北之間的衝突，但最終失敗。而他任內簽訂的[[內布拉斯加法案|內布拉斯加法案]]成為他最為非議的法案，也使他成為[[廢奴主義|廢奴主義]]者眾矢之的，該法案也成為[[美國內戰|美國內戰]]的導火線。皮尔斯被認為是[[美国总统历史排名|最差的美國總統之一]]。

== 早年生涯 ==
[[File:Franklin_Pierce_-_1852.jpg|thumb]]戎裝照，1852年]]
皮尔斯1804年11月23日生于[[新罕布什尔州|新罕布什尔州]]的[[希尔兹巴勒|希尔兹巴勒]]。他是[[鲍登大学|鲍登大学]]的学生。1829年當選[[新罕布什尔州|新罕布什尔州]]议会议员。1831年成为新罕布什尔州议会议长。1838年當選[[美国众议员|美国众议员]]，时年33岁，是当时美国最年轻的众议员。之后又成为参议员。不久又从事[[律师|律师]]行业。1846－1848年[[墨西哥战争|墨西哥战争]]期间，晉升[[准將|准將]]。

== 總統任期内 ==
在1852年民主党全国大会的第49轮投票中被提名为“[[黑马|黑马]]”候选人。在当年的总统竞选中，皮尔斯同他的竞选伙伴[[威廉·魯福士·金|威廉·魯福士·金]]以普选的55％比44％、选举人票的254比42的压倒性优势战胜輝格黨的Winfield Scott，成为那时最年轻的总统（1869年被格蘭特取代）。直到今天為止，他仍是唯一一位來自[[新罕布什爾州|新罕布什爾州]]的總統。

在總統任內，身為北方民主黨人的他曾企圖廢除奴隸制度，但受制于民主黨的南方白人勢力而無實行，而他為避免[[南北戰爭|南北戰爭]]而簽訂的[[內布拉斯加法案|內布拉斯加法案]]，該法案是为了调停反对在这一地区修建铁路的人的利益，然而许多美国人认为，这项建议法案的真正目的在于奴隶制的扩张。<ref>[http://blog.ifeng.com/article/2836099.html 堪萨斯――内布拉斯加法案分裂了美国] {{webarchive|url=https://web.archive.org/web/20141129054542/http://blog.ifeng.com/article/2836099.html |date=2014-11-29 }}</ref>內布拉斯加法案最終成為南北戰爭的導火線。

1856年，由於在民主黨內無法再次贏得總統選舉的黨內提名，無法連任總統。

== 卸任后 ==
卸任后僅四年，南北戰爭隨即爆發，他支持維護聯邦的統一。但他仍然直言不諱的批評總統[[林肯|林肯]]在內戰時的政策。
