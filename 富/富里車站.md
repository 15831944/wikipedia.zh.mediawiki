'''富里車站'''位於[[台灣|台灣]][[花蓮縣|花蓮縣]][[富里鄉|富里鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。本站配合「花東新車站運動」計畫改建完工。

== 車站構造 ==
*[[岸式月台|岸式月台]]、[[島式月台|島式月台]]各一座。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]
*停靠狀態：
**[[普悠瑪列車|普悠瑪號]]：停靠順行406、412、422(週日停駛)、448次(週日停駛);逆行425(週日停駛)、431次(週日停駛)。
**[[自強號列車|自強號]]：除逆行301、427(逢週五、六行駛)、6443(逢週日行駛)次、6473（逢週日行駛）、順行418(逢週五至日行駛)、410、324、484(逢週五行駛)次外皆停靠。
**[[莒光號列車|莒光號]]：除[[觀光列車|觀光列車]]、[[團體列車|團體列車]]、南下616次（逢週五行駛）、北上6615次（逢週日行駛）外皆停靠。
**[[臺鐵區間車|區間車]]：皆停靠。
*對號列車停靠率：50.0%；所有車種停靠率：59.7%。
*站內設有「鐵路之旅－小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1926年1月20日：設「公埔」停車場；昔日少有私有放牧場，而此地建有一平埔族公有放牧場，故將此地稱為「公埔」。
*1937年11月1日：因此地土地肥沃，地名和站名同時改為「富里」。
*2016年6月28日：啟用多卡通刷卡機。
