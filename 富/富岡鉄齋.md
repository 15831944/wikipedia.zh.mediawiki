'''富岡 鉄齋 '''(とみおか てっさい；1837年1月25日 - 1924年12月31日)，生於京都（三條通新町東），字'''無倦'''，號'''鉄齋'''，是日本[[明治|明治]]至[[大正|大正]]年間的文人畫家、儒學學者。被謳歌為日本最後的[[文人|文人]]<ref>{{cite web |author1=產經新聞 |title=日本畫家富岡鐵齋14幅未發表作品被發現 |url=http://japan.people.com.cn/BIG5/n/2013/1118/c35465-23577441.html |publisher=環球網 |accessdate=2018-10-09}}</ref>。別號'''鉄人'''、'''鉄史'''、'''鉄崖'''等。

== 生涯 ==
文久元年(1861年)，前往[[長崎|长崎]]遊學，接受長崎南畫派的祖門鉄翁、木下逸雲、小曾根乾堂等人的指导。

[[座右銘|座右铭]]是「讀萬卷書、行萬里路」，並在日本各地旅行而得到實踐<ref>{{cite web |title=富岡鐵齋 |url=http://163.17.79.102/%A4%A4%B0%EA%A4j%A6%CA%AC%EC/Content.asp?ID=76695 |publisher=中國大日科全書智慧藏 |accessdate=2018-10-09}}</ref>。明治7年(1874年)，與松浦武四郎等人的交流而到[[北海道|北海道]]旅行，在那時便描繪了有關[[阿伊努族|阿伊努人]]風俗題材的代表作「舊蝦夷風俗圖」。

明治二年([[1869年|1869年]])，成為私塾立命館的一名教师。

明治26年(1893年)，成為京都市美术学校 的教师。
