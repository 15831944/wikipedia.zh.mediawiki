'''富士見丘站'''（{{jpn|j=富士見ヶ丘駅|hg=ふじみがおかえき|rm=Fujimigaoka eki|f=y}}）是一個位於[[東京都|東京都]][[杉並區|杉並區]][[久我山|久我山]]五丁目，屬於[[京王電鐵|京王電鐵]][[井之頭線|井之頭線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''IN13'''。

== 歷史 ==
* 1933年（[[昭和|昭和]]8年）8月1日 - 隸屬於帝都電鐵的車站開始營運。
* 1940年（昭和15年）5月1日 - 與小田原急行鐵道合併，成為該公司帝都線的車站。
* 1942年（昭和17年）5月1日 - [[小田急電鐵|小田急電鐵]]與[[東京急行電鐵|東京急行電鐵]]（大東急）合併。
* 1948年（昭和23年）6月1日 - 從東急分離，成立京王帝都電鐵，成為該公司井之頭線的車站。
* 2010年（[[平成|平成]]22年）12月26日 - 車站遷至新建的跨橋式站房。

== 車站構造 ==
[[跨站式站房|跨站式站房]]設計，具有1面2線[[島式月台|島式月台]]的[[地面車站|地面車站]]。現今車站站房於2010年3月動土，2010年12月26日開始營運。車站中央通道與連接的月台、南口、北口之間，設有電梯與電扶梯（跨橋站房至南口地面之間沒有設置電扶梯）。

2010年12月以前曾連接舊站房的地下通道被保存下來，開放供民眾作為南北向的自由通路。

車站西側設有{{link-ja|富士見丘車廠|富士見ヶ丘検車区}}。另外設置於主要幹線西側的[[側線|側線]]，與上、下幹線連接作為入車線，在早上的通勤尖峰時段以當站作為發車站，並藉由此側線將列車駛進主線用以消化通勤人潮。也因此在早上的尖峰時段時，有許多為求空席可坐的乘客皆在此站轉乘當站始發列車，進而造成月台的混雜狀況。

[[公用洗手間|洗手間]]設置於2樓車站閘口内，同時也設置有無障礙廁所。

部分列車以此站作為起點或終點。該站發車至吉祥寺的列車需由入車線逆向駛進1號月台，因此該月台同時設有順向及逆向的停車位置標點。預備由吉祥寺發車的執勤車輛，出富士見丘車庫後也以同樣方式利用該月台作為轉折點運行至吉祥寺站投入勤務。

雖然僅有普通車停靠此站，但由於設有車廠的關係，仍然屬於井之頭線線上的主要車站。月台也較早裝設[[發光二極管|LED]]的列車資訊看板（之後井之頭線已全面裝設）。<!-- 原文提到：普通車停靠站的列車進站廣播僅有通知「列車將要進站」，該站的廣播則為「OO行的列車即將進站」-->

== 車站周邊 ==
{{See also|久我山|高井戶西}}
南側、北側皆有[[商店街|商店街]]，站前沒有大型道路交叉。商店街外即是大範圍的住宅區。
* 杉並富士見丘[[郵便局|郵便局]]
* {{link-ja|NHK富士見丘Club House|NHK富士見ヶ丘クラブハウス}} - {{link-ja|前川國男|前川國男}}設計。1954年竣工。
* 京王電鐵富士見丘車廠
* 杉並區立久我山小学校
* {{link-ja|浴風會|浴風会}}本館 - [[內田祥三|內田祥三]]、土岐達人設計。1926年竣工。於2001年被選為{{link-ja|東京都選定歷史的建造物|東京都選定歴史的建造物}}。

== 其他 ==
1960年，京王帝都電鐵原計畫建設新路線，由當站連接至[[三鷹站|三鷹站]]方向的三鷹線，以及連接[[西國立站|西國立站]]方向的立川線等2條路線，最終皆未實現計畫。
