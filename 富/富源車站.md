'''富源車站'''位於[[台灣|台灣]][[花蓮縣|花蓮縣]][[瑞穗鄉|瑞穗鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。本站已配合「花東新車站運動」計畫改建完工。

== 車站構造 ==
*[[側式月台|側式月台]]、[[島式月台|島式月台]]各一座。

== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]。
*停靠狀態：
**[[自強號列車|自強號]]：停靠逆行407次、順行328次（逢周五、日行駛）、340次（逢周五、日停駛）。
**[[莒光號列車|莒光號]]：停靠南下602、642次(週六，日停駛) ；北上653(週日停駛)、611次
**[[臺鐵區間車|區間車]]：皆停靠。
*對號列車停靠率：16.0%；所有車種停靠率：30.0%。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1914年3月8日：設「拔仔停車場」。「拔仔」，音譯自阿美語「真白的溪流」。
*1921年6月1日：改為「拔子停車場」。
*1937年11月1日：將「真白的溪流」以日文意譯為「白川」，並將站名和地名一併改為「白川驛」。
*1947年8月23日：因[[花蓮縣|花蓮縣]]政府認為「白川」站名含有強烈日本色彩，建議改名；之後[[臺灣鐵路管理局|臺鐵]]花蓮管理處取其諧音改為「百-{庄}-」站。
*約1950年：因本站行政區域位於[[瑞穗鄉|瑞穗鄉]]富源村，應當地村民強烈要求站名配合地名改為「富源」站。
*2016年6月28日：啟用多卡通刷卡機。
