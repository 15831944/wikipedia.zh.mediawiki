富王大飯店坐落於台中[[西屯區|西屯區]]之老字號飯店，除了座落地點為台中市重要交通樞紐地帶，毗鄰[[新光三越百貨|新光三越百貨]]商圈、[[臺中國家歌劇院|臺中國家歌劇院]]、[[秋紅谷廣場|秋紅谷]]、[[臺中市政府|市政府]]、[[愛買|愛買]]、[[臺中市第七期市地重劃區|臺中市第七期市地重劃區]]及許多文化空間、[[獨立書店|獨立書店]]等，為台中市最現代化的重要社經中心地段。自1985年開幕至今，歷經[[Sars|SARS]]、[[921大地震|921大地震]]風暴歷久不衰。

== 歷史 ==

====== '''官方網站顯示中國台灣''' ======
2018年8月，臺北市西華飯店、[[永豐棧酒店|永安棧酒店]]、福君海悅酒店、臺中市富王大飯店、高雄市漢華酒店被媒體報導，其官方網站簡體版本中，飯店位置顯示為[[中國台灣|中國台灣]]，引起許多民眾不滿。<ref>{{Cite news|url=https://www.thenewslens.com/article/101712|title=不只林口亞昕，六家用了同樣「網站系統」的飯店也出現「中國台灣」|author=Abby Huang|date=|work=|accessdate=2018.8.10}}</ref>

對此，台北西華飯店與永安棧酒店表示，由於官網是委外第三方技術廠商製作，目前已將錯誤的部分撤除，並緊急聯繫委外廠商要求調整相關資訊。而其他被點名的飯店，目前官網中也已經把簡體版網頁拿掉，現在已經看不到頁面出現「中國台灣」的字眼。<ref>{{Cite news|url=https://news.tvbs.com.tw/life/971839|title=飯店網頁現「中國台灣」遭砲轟　業者喊冤：廠商擅自更改|author=李育道|date=|work=|accessdate=2018/08/11}}</ref>

== 飯店特色 ==

====== 貫通石 ======

民間認為[[貫通石|貫通石]]具高能量，能有趨吉避邪和助孕生男效果，[[草埔隧道|草埔隧道]]貫通工程結束後，全台已無隧道貫通計畫<ref>{{Cite news|url=https://udn.com/news/story/11322/3151738|title=南迴隧道貫通石助孕？ 900份1小時內搶光|author=潘欣中|date=|work=|accessdate=2018-05-20}}</ref>，貫通石遂成稀有之物。目前富王飯店仍保有一顆，因為據說摸了就能生兒子，所以受到新人和夫妻的喜愛。

====== 白鷺鷥雕像 ======
1999年票選出[[白鹭|白鷺鷥]]為台中市市鳥<ref>{{Cite news|url=http://news.ltn.com.tw/news/life/paper/216891|title=小白鷺是中市「市鳥」|author=林良哲、蔡淑媛|date=|work=|accessdate=2008-06-05}}</ref>，因此當時富王大飯店聘請知名設計師[[朱魯青|朱魯青]]製作三隻白鷺鷥大型雕塑作品，贈送台中市作為精神指標。製作完成後展覽於[[台中市立文化中心|台中市文化中心]]作為城市景點。但後來台中文化中心改建，白鷺鷥雕像重回富王飯店入口處。

====== 636藝文中心 ======

[[民歌餐厅|民歌餐廳]]曾在台灣紅極一時，當時常有鋼琴師承租636廳，作為[[鋼琴餐廳|鋼琴餐廳]]（Piano Bar），但因為生意太好被同業頻頻檢舉，不堪其擾而暫停演出。現的636廳作為中型[[會議廳|會議廳]]，室內陳設仍保有當時的時代感，也常作為藝文活動之用。
