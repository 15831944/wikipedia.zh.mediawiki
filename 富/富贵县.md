'''富贵县'''（{{vie|v='''huyện Phú Quý'''|hn=縣富貴}}），俗称'''岣嶗快驩'''（{{lang-vi|cù lao Khoai Xứ}}），'''秋岛'''。是[[南中国海|南中国海]]（越南称之为“东海”）上的一个岛屿，主权上属于[[越南|越南]]，行政区划上属越南[[平顺省|平顺省]]，1930年单独建县，本县一共管辖9个岛屿。

== 行政区划 ==
富贵县下辖3社，县莅在五凤社。
*五凤社（Xã Ngũ Phụng）
*三清社（Xã Tam Thanh）
*龙海社（Xã Long Hải）

== 教育 ==
富贵县一共有14所中小学、早教机构、托儿所，三清中学、五凤中学、龙海中学、吴权中学等中学。

== 交通 ==
富贵县是[[海岛|海岛]]县，对外只能靠[[海运|海运]]链接越南大陆。

== 旅游景点 ==
岛上有[[灵光寺|灵光寺]]，[[万安所城|万安所城]]，[[朝阳亭|朝阳亭]]等名胜，而富贵群岛本身就是一个自然景点。
