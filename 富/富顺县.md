'''富顺县'''，是[[中华人民共和国|中华人民共和国]][[四川省|四川省]][[自贡市|自贡市]]的下辖[[县级行政区|县]]，县治为[[富世镇|富世镇]]，历史悠久，全县面积1602.92平方公里，在2011年人口为107.71万人。

古代“富世县”一名来源自古代出盐最多一口“富世盐井”。

== 历史 ==

[[北周|北周]][[天和|天和]]2年（567年）将富世盐井周边地区，设[[雒原郡|雒原郡]]及所辖'''富世县'''。[[唐代|唐代]][[贞观|贞观]]23年（公元649年）将其更名为富义县。[[北宋|北宋]][[乾德|乾德]]4年（公元966）将其升格与州同级的富义监，境域从原来的1000平方公里扩大到了3000平方公里。[[太平兴国|太平兴国]]元年（公元976年）改名富顺盐监。[[元代|元]]世祖[[至元|至元]]12年（公元1275年）改富顺盐监为富顺安抚使司，[[至元|至元]]20年（公元1283）升为富顺州。

[[明|明]][[洪武|洪武]]4年（公元1371）降州为富顺县。[[民国|民国]]24年（公元1935年），隶于四川省第七行政督察区。中华人民共和国成立后，1983年3月，正式由[[自贡市|自贡市]]管辖。

== 地理 ==
富顺县位于[[四川盆地|四川盆地]]南部，介于东经104<sup>o</sup>40'-105<sup>o</sup>15'，北纬28<sup>o</sup>55'-29<sup>o</sup>28'之间。县位于[[沱江|沱江]]下游，并将富顺县分城为东湖、富世二镇。东接[[内江市|内江市]][[隆昌县|隆昌县]]，西接[[自贡市|自贡市]][[沿滩区|沿滩区]]，北与自贡市大安区和隆昌县接壤。

== 气候 ==
富顺属亚热带湿润季风气候，四季分明，温和多雨，因四川盆地地理响，终年多雾，年平均气温18<sup>o</sup>C，常年降雨量1,058mm，日照率1,275小时，无霜期355天。

== 名人 ==
历来文教兴盛，人物杰出，古代进士、举人多至1,000余人，素有“富顺才子内江官”、“才子之乡”的美誉。

明代，富顺人考中进士 139人、举人492人、贡生386人，著名人物有景泰十才子之首的晏锋、嘉靖八才子之一的熊过等。

清代，全县考中进士31人、举人315人、贡持477人。

此外，戊戌六君子之一的[[刘光第|刘光第]]、民国监察委员[[谢持|谢持]]、[[红军|红军]]将领[[邓萍|邓萍]]、革命烈士[[江竹筠|江竹筠]]、四川报业第一人[[宋育仁|宋育仁]]、[[四川|四川]]省委书记[[陈刚|陈刚]]（[[刘家镇|刘家镇]]、又名[[刘作抚|刘作抚]]）[[厚黑学|厚黑学]]创始人[[李宗吾|李宗吾]]、[[中央党校|中央党校]]常务副校长[[郑必坚|郑必坚]]、川南才女[[赖雨|赖雨]]、巴蜀鬼才[[魏明伦|魏明伦]]、[[郭敬明|郭敬明]]等均为富顺人。
