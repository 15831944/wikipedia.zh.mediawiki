'''富平县'''是[[陕西省|陕西省]][[渭南市|渭南市]]下辖的一个县。位于陕西中部，[[渭河平原|渭河平原]]北部。面积1233[[平方公里|平方公里]]，2006年人口76万，为陕西省人口最多的的行政县之一，邮编711700，县政府驻[[杜村镇|杜村镇]]人民路。

==历史==
[[秦厉共公|秦厉共公]]二十一年（前456年）始在频山以南设置频阳县治（故址在今美原镇古城村一带）。

[[金太宗|金太宗]][[天会|天会]]九年（1131年），即宋金两军“富平之役”后，金人以富平、美原二县赐齐（金人立刘豫为齐帝，建都大名）。

== 交通 ==
富平境内有[[西禹高速公路|西禹高速公路]]和[[咸铜铁路|咸铜铁路]]、[[西延铁路|西延铁路]]通过，西铜高速，108国道，210国道穿境而过。

== 资源 ==
石灰石、大理石储量丰富。富平是渭北高原上最大商品粮生产基地，盛产[[苹果|苹果]]。

== 古迹 ==
有唐献陵陪葬区、定陵、[[唐元陵|元陵]]、丰陵、章陵、簡陵、法源寺塔等古迹。唐代帝陵详见[[唐十八陵|唐十八陵]]。
