'''富山縣'''（{{jpn|j=富山県|hg=とやまけん|rm=Toyama ken|f=y}}）是[[日本|日本]]的一个縣，位于[[本州|本州]]中部[[日本海|日本海]]沿岸的[[北陸地方|北陸]]地區，轄區範圍相當於[[令制國|令制國]]時的[[越中國|越中國]]。縣廳所在地为[[富山市|富山市]]。富山縣地形一面環海三面環山，縣內高低差極大。富山縣擁有世界文化遺產[[五箇山|五箇山]]，兼具信仰聖地和豐富自然生態景觀的[[立山_(日本)|立山]]等著名景點。擁有、魚津市至滑川市的海岸地區以[[海市蜃楼|海市蜃楼]]現象而著稱。富山縣還是[[北陸工業地域|北陸工業地域]]的中心地區，是日本海沿岸縣中工業較發達的縣，醫藥產業的歷史尤為悠久。

岐阜縣北部的[[飛驒國|飛驒地區]]距離太平洋沿岸地區較遠，富山縣反而是距該地最近的出海口，使得兩地在經濟、文化上關係密切，有時合稱為「飛越地區」<ref>{{cite web |url=http://www.hietsu.jp/block/ |script-title=ja:飛越地方とは |accessdate=2017-06-05 |publisher={{lang|ja|飛越協議会}} |language=ja}}</ref>。

== 歷史 ==
=== 古代史 ===
[[File:ToyamaCastle000.jpg|thumb]]
富山縣最早的人類活動遺跡可以追溯至[[舊石器時代|舊石器時代]]，這一時期的主要遺跡有安養寺遺跡等<ref name="dictionary">{{cite book |author={{lang|ja|竹内理三}}|script-title=ja:《角川日本地名大辞典 16 富山県》 |year=1979 |publisher={{lang|ja|角川書店}} |location=東京都 |isbn=4040011600|language=ja}}</ref>{{rp|22}}。進入[[繩文時代|繩文時代]]後，海平面上升並更靠近人類居住區，使得人們開始採集海產作為食物。富山縣也發現了這一時期的[[貝塚|貝塚]]<ref name="dictionary"></ref>{{rp|22}}。富山縣的主要繩文時代遺跡有魚津市的天神山遺跡、朝日町的不動堂遺跡等地<ref name="dictionary"></ref>{{rp|23}}。[[彌生時代|彌生時代]]之後，日本開始出現農耕文化，富山縣內也有一些遺跡出土了彌生時代的土器<ref name="dictionary"></ref>{{rp|23}}。富山縣最早的古墳出現在4世紀，縣內的大型古墳有羽根山古墳群、櫻谷古墳群等地<ref name="dictionary"></ref>{{rp|23}}。

在畿內地方已形成統一國家時，[[北陸道|北陸道]]地區屬於[[越國_(日本)|越國]]。日本在實施[[日本律令制|律令制]]之後將越國分為[[越前國|越前]]、[[越中國|越中]]、[[越後國|越後]]三國，富山縣即屬於越中國<ref name="dictionary"></ref>{{rp|24}}。著名歌人[[大伴家持|大伴家持]]曾任越中國守，位於高岡市的越中國府和國分寺遺跡記錄了這一時期的歷史<ref name="dictionary"></ref>{{rp|24}}。8世紀之後，莊園勢力日漸壯大，成為越中國的實力者<ref name="dictionary"></ref>{{rp|26}}。富山縣的象徵自8世紀初期開始即有佛寺，立山信仰也自[[平安時代|平安時代]]就開始形成<ref name="dictionary"></ref>{{rp|27}}。

在平安時代末期的[[源平合戰|源平合戰]]中，今富山、石川兩縣交界處的[[俱利伽羅峠|俱利伽羅峠]]是[[俱利伽羅峠之戰|俱利伽羅峠之戰]]的戰場<ref name="dictionary"></ref>{{rp|27}}。[[鎌倉時代|鎌倉時代]]時，名越氏曾擔任越中守護<ref name="dictionary"></ref>{{rp|28}}。鎌倉幕府滅亡之後，越中國爭戰不斷，守護也頻繁變動。直到[[南北朝_(日本)|南北朝]]戰亂結束之後，[[足利義滿|足利義滿]]任命{{link-ja|畠山基國|畠山基国}}為越中守護之後，越中國才進入畠山氏的長期政權時期<ref name="dictionary"></ref>{{rp|28}}。越中國和越前國、[[加賀國|加賀國]]等地都是[[淨土真宗|淨土真宗]]繁盛的地區，因此宗教性質的[[一向一揆|一向一揆]]時有發生，寺院實際支配了越中國的部分地區<ref name="dictionary"></ref>{{rp|28}}。

[[戰國_(日本)|戰國]]時代，越中國的大名{{link-ja|神保氏|神保氏}}、{{link-ja|椎名氏|椎名氏}}外受[[上杉謙信|上杉謙信]]、[[武田信玄|武田信玄]]、[[織田信長|織田信長]]等強敵環繞，內有一向一揆勢力的牽制，勢力弱小<ref name="dictionary"></ref>{{rp|31}}。上杉謙信在1560年（[[永祿|永祿]]3年）首次侵入越中國，此後控制了越中國東部地區<ref name="dictionary"></ref>{{rp|32}}。而織田信長也在1581年（[[天正_(日本)|天正]]9年）遣[[佐佐成政|佐佐成政]]、[[前田利家|前田利家]]攻入越中並佔領越中國西部地區<ref name="dictionary"></ref>{{rp|32}}。[[本能寺之變|本能寺之變]]後，控制了中央政權的[[豐臣秀吉|豐臣秀吉]]在越中國封給佐佐成政。後因佐佐成政謀反，豐臣秀吉將越中國大部分地區改封給前田利家<ref name="dictionary"></ref>{{rp|33}}。

[[關原之戰|關原之戰]]後，[[德川家康|德川家康]]獎賞[[前田利長|前田利長]]投靠德川陣營，將越中、加賀、[[能登國|能登]]三國都封給前田氏，此後至幕末，越中國都由前田氏治理，屬於[[加賀藩|加賀藩]]及加賀藩的支藩[[富山藩|富山藩]]<ref name="dictionary"></ref>{{rp|33}}。[[江戶時代|江戶時代]]，藩政當局積極進行新田開發，使得越中國的農業有一定發展<ref name="dictionary"></ref>{{rp|35}}。自江戶時代開始，富山和高岡就是越中國的兩大城市。當時富山屬於[[城下町|城下町]]，也是飛驒街道的起點。而高岡則在[[高岡城|高岡城]]被廢之後演變為商業城市<ref name="dictionary"></ref>{{rp|37}}。富山的製藥產業也是自江戶時代就開始興起<ref name="dictionary"></ref>{{rp|39}}。

=== 富山縣的成立 ===
[[File:Fushiki_factory_of_OJI_PAPER_CO.,_LTD.jpg|thumb]]
1871年廢藩置縣之後，富山藩被富山縣取代，後又很快改名新川縣。1876年，新川縣被併入石川縣，舊加賀藩版圖復活。然而越中國在加賀藩時代飽受差別待遇，被併入石川縣一事引發了當地民眾的強烈不滿，發起了分縣運動。1883年，富山縣成功復縣<ref name="dictionary"></ref>{{rp|40}}。富山縣的近代化開始於交通的近代化。1897年，富士縣首條鐵路通車<ref name="dictionary"></ref>{{rp|41}}。富山縣是[[北海道|北海道]]移民的最大來源縣之一。在1897年至1907年期間，有近6萬富山縣民移民北海道，是同時期各縣之冠<ref name="dictionary"></ref>{{rp|42}}。1918年，震撼日本全國，以至於政府派出軍隊的[[米騷動|米騷動]]就開始於富山縣<ref name="dictionary"></ref>{{rp|42}}。另一方面，多急流的地理特徵使得富山縣適合開展水力發電。富山縣自20世紀初期就開始扮演近畿地方電源地的角色，並且豐富的電力也促進了富山縣本地產業的發展<ref name="dictionary"></ref>{{rp|43}}。在第二次世界大戰時期，富山縣是日本重要的軍需品生產地。二戰末期，太平洋沿岸地區眾多工廠為躲避空襲而搬遷至富山，並且許多大都市的兒童也被疏散到富山縣。然而自1945年5月開始，富山縣也開始被美軍空襲。1945年，富山市遭到{{link-ja|富山大空襲|富山大空襲|大規模空襲}}，超過2,700人在空襲中遇難，市區99.5%的面積被燒毀。這場空襲也使得富山市是三大都市圈以外城市中除了被原子彈轟炸的[[廣島市|廣島市]]、[[長崎市|長崎市]]之外受害最嚴重的城市<ref>{{cite web |url=http://matsukawa-cruise.jp/reading/toyama-air-raid-and-matsukawa/ |script-title=ja:富山大空襲と松川 |accessdate=2017-06-05 |publisher={{lang|ja|松川遊覧船}} |language=ja}}</ref>。

富山縣在戰後快速開始復興。1952年，富山縣制定了綜合開發計劃，提出了在常願寺川、黑部川上游興建水庫等目標，是較早制定大規模開發計劃的縣之一<ref name="dictionary"></ref>{{rp|43}}。1964年，富山·高岡地區被指定為新產業都市，使得富山縣沿海地區成為北陸地方工業開發的據點<ref name="dictionary"></ref>{{rp|43}}。1968年，富山新港竣工，成為富山縣的一大工業中心<ref name="dictionary"></ref>{{rp|44}}。然而急速的工業化也帶來負面影響。1950年代開始，富山縣出現[[痛痛病|痛痛病]]，工廠將廢水排入河川後引發下游居民罹患[[鎘中毒|鎘中毒]]，造成嚴重的健康問題<ref name="dictionary"></ref>{{rp|44}}。嚴重的公害問題也促使富山縣自1970年代開始修正過於重視工業化的發展路線。2015年，[[北陸新幹線|北陸新幹線]]通車，使得富山縣前往[[東京|東京]]的交通便利度大幅提升，也對富山縣今後的發展有重大作用<ref name="shinkansen">{{cite web |url=http://www.pref.toyama.jp/cms_sec/1002/kj00015308.html |script-title=ja:祝 北陸新幹線開業 |accessdate=2017-06-05 |publisher={{lang|ja|富山県}} |language=ja |title=存档副本 |archive-url=https://web.archive.org/web/20150708023344/http://www.pref.toyama.jp/cms_sec/1002/kj00015308.html |archive-date=2015-07-08 |dead-url=yes }}</ref>。

== 地理 ==
[[File:Tsurugidake_from_bessan_22_1995_8_20.jpg|thumb]]
富山縣東西寬約90公里，南北長約76公里，面積4,247.61平方公里<ref>{{cite web |url=http://www.pref.toyama.jp/cms_sec/1002/kj00015308.html |script-title=ja:富山県の概要 位置・面積・人口 |accessdate=2017-06-05 |publisher={{lang|ja|富山県}} |language=ja |title=存档副本 |archive-url=https://web.archive.org/web/20150708023344/http://www.pref.toyama.jp/cms_sec/1002/kj00015308.html |archive-date=2015-07-08 |dead-url=yes }}</ref>。富山縣北側是[[富山灣|富山灣]]，東側隔[[飛驒山脈|飛驒山脈]]和[[長野縣|長野縣]]、[[新潟縣|新潟縣]]接壤，南側通過飛驒高地和[[岐阜縣|岐阜縣]]接壤，西側通過兩白山地和[[石川縣|石川縣]]接壤<ref name="Chishi">{{cite book |author={{lang|ja|藤田佳久・田林明}}|script-title=ja:《日本の地誌7　中部圏》 |year=2007 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=978-4-254-16767-2|language=ja}}</ref>{{rp|490}}。富山縣的地形呈現以富山灣為中心的半圓形構造，最靠近富山灣的是平原地形[[富山平原|富山平原]]，外側是台地和丘陵，最外側則是山地<ref name="Chishi"></ref>{{rp|491}}。位於富山縣中部的富山平原是由眾多沖積扇形成的平原，和西部的砺波平原並為富山縣的主要平地<ref name="Chishi"></ref>{{rp|491}}。富山縣內有五個一級水系，主要的河川有{{link-ja|黑部川|黒部川}}、{{link-ja|片貝川|片貝川}}、{{link-ja|早月川|早月川}}、[[常願寺川|常願寺川]]、{{link-ja|神通川|神通川}}、{{link-ja|庄川|庄川}}、{{link-ja|小矢部川|小矢部川}}<ref>{{cite web |url=http://www.pref.toyama.jp/cms_sec/1503/kj00000327-002-01.html |script-title=ja:河川の概要 |accessdate=2017-06-05 |publisher={{lang|ja|富山県}} |language=ja}}</ref>。由於富山縣內高低差極大，因此這些河流也大多極為湍急，在治水工程未完善的時代經常發生洪水<ref name="Chishi"></ref>{{rp|491}}。但另一方面，眾多急流使得富山縣適合修建水庫進行水力發電。縣內主要的大型水庫有[[黑部水庫|黑部水庫]]、{{link-ja|有峰水庫|有峰ダム}}、{{link-ja|境川水庫|境川ダム}}等地<ref>{{cite web |url=http://www.pref.toyama.jp/cms_sec/1503/kj00015226-002-01.html |script-title=ja:富山県のダム事業 |accessdate=2017-06-05 |publisher={{lang|ja|富山県}} |language=ja}}</ref>。富山縣的海岸線全長約145公里，東部以石灘為主，西部以沙灘和石灘為主<ref name="Chishi"></ref>{{rp|491}}。富山灣的最深處水深超過1,000米，這一在近海處就急劇變深的地形適合採集[[海洋深層水|海洋深層水]]<ref>{{cite web |url=http://taffrc.pref.toyama.jp/nsgc/suisan/link_flat.phtml?TGenre_ID=321&t=blog2 |script-title=ja:富山湾の海洋深層水-富山湾の深層水 |accessdate=2017-06-05 |publisher={{lang|ja|富山県農林水産総合技術センター/水産研究所}} |language=ja}}</ref>。富山縣東部的{{link-ja|立山黑部地質公園|立山黒部ジオパーク}}內地形變化多端，高地差達4,000米，是日本地質公園<ref>{{cite web |url=http://www.city.kurobe.toyama.jp/attach/EDIT/010/010688.pdf |script-title=ja:立山黒部ジオパーク |accessdate=2017-06-05 |publisher={{lang|ja|黒部市}} |language=ja}}</ref>。
=== 氣候 ===
富山縣的氣候屬於[[日本海側氣候|日本海側氣候]]，內又可分為山地氣候區和平原氣候區<ref name="Chishi"></ref>{{rp|491}}。富山縣平地雖然在冬季多豪雪天氣和陰天，日照時間較短，但並不十分寒冷。在夏季的日照時間甚至超過太平洋沿岸的部分地區<ref name="Chishi"></ref>{{rp|491}}。富山縣是日本降水量較多的地區，平地地區的年降水量在2,100毫米至2,600毫米之間，山地的年降水量超過3,000毫米，全年大部分降水集中在冬季時的降雪<ref name="Chishi"></ref>{{rp|491}}。在日本海海域低氣壓發達時，富山縣內常有[[焚風|焚風]]現象發生<ref name="Chishi"></ref>{{rp|492}}。魚津市的沿海地區在4月至5月期間常有[[海市蜃楼|海市蜃楼]]現象發生<ref>{{cite web |url=https://www.city.uozu.toyama.jp/nekkolnd/shinkiro/index.html |script-title=ja:蜃気楼 |accessdate=2017-06-05 |publisher={{lang|ja|魚津埋没林博物館}} |language=ja}}</ref>。
=== 行政區劃 ===
富山縣傳統上以{{link-ja|吳羽丘陵|呉羽丘陵}}為界，分為吳東和吳西兩個地區<ref>{{cite web |url=http://toyama-teiju.jp/knowledge/others/others-1 |script-title=ja:呉西、呉東ってなに？ |accessdate=2017-06-05 |publisher={{lang|ja|くらしたい国、富山}} |language=ja}}</ref>。吳東地區包括[[富山市|富山市]]、[[滑川市|滑川市]]、[[上市町|上市町]]、[[立山町|立山町]]、[[舟橋村|舟橋村]]、[[魚津市|魚津市]]、[[黑部市|黑部市]]、[[朝日町_(富山縣)|朝日町]]、[[入善町|入善町]]。吳西地區包括[[高岡市|高岡市]]、[[射水市|射水市]]、[[冰見市|冰見市]]、[[砺波市|砺波市]]、[[小矢部市|小矢部市]]、[[南砺市|南砺市]]。富山縣共轄有10市4町1村，是日本自治體數量最少的縣。

== 人口 ==
1920年日本首次人口普查時，富山縣有人口約72.4萬人<ref name="population1">{{cite web |url=http://www.pref.toyama.jp/sections/1015/lib/kokucyo/h22/toukeihyou/jikeiretu/ji1hyou.xls |script-title=ja:人口 |accessdate=2017-06-05 |publisher={{lang|ja|富山県}} |language=ja}}</ref>。富山縣人口在二戰前穩定增加，並在戰後因嬰兒潮和軍人返鄉而出現人口暴增，在1950年時突破百萬大關<ref name="population1"></ref>。然而在1950年代和1960年代，受人口外流影響，富山縣人口增長陷入停滯，直到1970年代才恢復增加<ref name="population1"></ref>。然而1980年代之後，由於人口老齡化和少子化，富山縣人口增長再次開始趨緩。1998年，富山縣人口達到112.6萬人的峰值<ref name="population2">{{cite web |url=http://www.pref.toyama.jp/cms_pfile/00015864/00872424.pdf |script-title=ja:富山県人口ビジョン |accessdate=2017-06-05 |publisher={{lang|ja|富山県}} |language=ja}}</ref>。2017年5月1日時，富山縣有人口1,057,570人<ref>{{cite web |url=http://www.pref.toyama.jp/sections/1015/lib/jinko/_news/jinko170501/jinko170501.html |script-title=ja:富山県の人口と世帯（推計）|accessdate=2017-06-05 |publisher={{lang|ja|富山県}} |language=ja}}</ref>。富山縣的人口自1920年代中期曾一度超過鄰近的[[石川縣|石川縣]]。但在1980年代後，石川縣人口反超富山縣，且差距仍在拉大<ref name="population2"></ref>。在1990年代中期，富山縣一度是人口流入縣，但在其他時間則一直是人口流出縣<ref name="population2"></ref>。2014年，富山縣的[[總和生育率|總和生育率]]有1.45，略高於日本平均水準<ref name="population2"></ref>。

== 政治 ==
[[File:Toyama_Prefectural_Office_Building01st3200.jpg|thumb]]
1996年日本導入[[小選舉區制|小選舉區制]]之後，在[[眾議院_(日本)|眾議院]]選舉中，富山縣被劃分為3個選舉區<ref>{{cite web |url=http://www.soumu.go.jp/main_content/000240056.pdf　 |script-title=ja:富山県 |accessdate=2017-06-05 |publisher={{lang|ja|総務省}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。這三個選舉區均是[[自由民主黨_(日本)|自民黨]]佔據優勢的{{link-ja|保守王國|保守王国}}。除了[[富山縣第1區|富山縣第1區]]在2009年曾有[[民主黨_(1998年)|民主黨]]候選人當選之外<ref>{{cite web |url=http://www.yomiuri.co.jp/election/shugiin2009/kaihyou/ya16.htm　|script-title=ja:小選挙区 富山  |accessdate=2017-06-05 |publisher={{lang|ja|読売新聞}} |language=ja}}</ref>，這三個選舉區在歷次選舉中都是由保守派人士當選。在[[參議院_(日本)|參議院]]選舉中，富山縣屬於[[富山縣選舉區|富山縣選舉區]]，共有2名議員。該選舉區自1970年代以來只有過兩次非自民黨候選人當選記錄。

[[富山縣知事列表|富山縣知事]]也多是由保守派人士擔任。戰後富山縣只經歷了六位不同的知事，第五位知事{{link-ja|中沖豐|中沖豊}}自1980年開始執政長達24年<ref>{{cite web |url=http://www.toyama-brand.jp/TJN/?tid=102250　|script-title=ja:No.170-1:中沖前知事が任期満了で退任し、石井県政がスタート |accessdate=2017-06-05 |publisher={{lang|ja|TJN編集室}} |language=ja}}</ref>。現任富山縣知事是{{link-ja|石井隆一|石井隆一}}，他在2004年首次當選，並且已經三次連任成功<ref>{{cite web |url=http://www.pref.toyama.jp/cms_cat/401010/　 |script-title=ja:No.170-1:知事室へようこそ |accessdate=2017-06-05 |publisher={{lang|ja|富山県}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。{{link-ja|富山縣議會|富山県議会}}共有40名議員，其中自民黨是最大黨派，有30名議員<ref>{{cite web |url=http://www.pref.toyama.jp/sections/0100/members/kaiha.html　 |script-title=ja:会派別議員名簿 |accessdate=2017-06-05 |publisher={{lang|ja|富山県議会}} |language=ja }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2015年度，富山縣的財政力指數有0.45，位居日本中等水準<ref>{{cite web |url=http://www.soumu.go.jp/main_content/000456281.xls |script-title=ja:全都道府県の主要財政指標 |accessdate=2017-06-03 |publisher={{lang|ja|総務省}} |language=ja}}</ref>。

== 經濟 ==
2014年，富山縣縣內生產總值44,526億日元，人均縣民所得有318.5萬日元，比日本平均水準高約一成<ref>{{cite web |url=http://www.pref.toyama.jp/sections/1015/lib/k3/_rep28/report01.html |script-title=ja:平成26年度県民経済計算の概要 |accessdate=2017-06-06 |publisher={{lang|ja|富山県}} |language=ja}}</ref>。

=== 第一級產業 ===
[[File:Combine_Jyohana_05a9119cv.jpg|thumb]]
2016年，富山縣有耕地面積5.87萬公頃，耕地面積率有13.8%，富山縣的農業就業人口有2.1萬人<ref>{{cite web |url=http://www.maff.go.jp/hokuriku/stat/attach/xls/index-1.xls |script-title=ja:ミニ農林水産統計 |accessdate=2017-06-06 |publisher={{lang|ja|北陸農政局}} |language=ja}}</ref>。

富山縣農業特徵之一是專精於稻米生產，不僅農業生產額中以稻米比例最大<ref>{{cite web |url=http://www.maff.go.jp/hokuriku/stat/attach/xls/index-4.xls |script-title=ja:ミニ農林水産統計 |accessdate=2017-06-06 |publisher={{lang|ja|北陸農政局}} |language=ja}}</ref>，耕地面積中水田比例，也是日本第一<ref name="Chishi"></ref>{{rp|492}}。除稻米外，富山縣主要農作物還有砺波市的[[鬱金香|鬱金香]]球根、入善巨無霸[[西瓜|西瓜]]等<ref>{{cite web |url=http://www.shoku-toyama.jp/season/s030/ |script-title=ja:富山食ブランド |accessdate=2017-06-06 |publisher={{lang|ja|富山県}} |language=ja}}</ref>。富士縣雖然67%面積被森林覆蓋，大部分是保安林，在水土保持上有重要作用，因此林業並不發達<ref name="Chishi"></ref>{{rp|493}}。

2014年，富山縣水產業生產額約144億日元<ref>{{cite web |url=http://www.pref.toyama.jp/cms_sec/1615/kj00015984.html |script-title=ja:富山県の水産業の概要 |accessdate=2017-06-06 |publisher={{lang|ja|富山県}} |language=ja}}</ref>。複雜的海底地形使得富山縣定置網漁業發達，{{link-ja|日本白蝦|シラエビ}}、[[螢火魷|螢火魷]]是富山縣著名水產<ref name="Chishi"></ref>{{rp|493}}。

=== 第二級產業 ===
[[File:Kurobe_dam_01.jpg|thumb]]
2014年，富山縣工業品生產額有35,672億日元，其中以化學業比重最大（16.8%），其次依序是非鐵金屬（11.2%）、金屬製品（10.1%）、生產用機械（9.4%）、電子零件（8.5%），這五類產業合計佔比56%<ref>{{cite web |url=http://www.pref.toyama.jp/sections/1015/lib/kogyo/rep26/01outline.htm |script-title=ja:平成26年工業統計調査結果の概況 |accessdate=2017-06-06 |publisher={{lang|ja|富山県}} |language=ja}}</ref>。富山縣是日本海沿岸地區的工業大縣，人均工業品生產額位居日本前列<ref name="Chishi"></ref>{{rp|493}}。

富山縣水電產業特別發達，不僅北陸三縣的電力供應企業[[北陸電力|北陸電力]]總部位於富山，[[關西電力|關西電力]]等外地企業也在富山縣內有黑部水庫等發電廠<ref>{{cite web |url=http://www.kurobe-dam.com/whatis/index.html |script-title=ja:黒部ダムを知る |accessdate=2017-06-06 |publisher={{lang|ja|黒部ダム}} |language=ja |title=存档副本 |archive-url=https://web.archive.org/web/20100325121930/http://www.kurobe-dam.com/whatis/index.html |archive-date=2010-03-25 |dead-url=yes }}</ref>。豐富的電力和水資源也使得富山縣適合開展化工業、鋁建材等生產過程需要大量電力和水的產業。自1964年被指定為新產業都市之後，富山和高岡地區的鋁建材產業快速發展，富山新港地區形成了沿海型工業區，是日本第一的鋁建材產地<ref>{{cite web |url=http://www3.boj.or.jp/toyama/pdf/toya.pdf |script-title=ja:富山県経済の特徴 |accessdate=2017-06-06 |publisher={{lang|ja|日本銀行}} |language=ja}}</ref>。{{link-ja|YKK AP|YKK AP}}等鋁業企業在富山縣設有工廠<ref>{{cite web |url=http://www.ykkap.co.jp/company/japanese/base/ |script-title=ja:主要拠点 |accessdate=2017-06-06 |publisher={{lang|ja|YKK AP}} |language=ja}}</ref>。

富山地區自[[元祿|元祿]]時代開始製藥業就特別發達，人均醫藥品生產金額是日本首位<ref>{{cite web |url=http://www3.boj.or.jp/kanazawa/kouhyou/report/report16j.pdf |script-title=ja:ほくりくのさくらリポート |accessdate=2017-06-06 |publisher={{lang|ja|日本銀行}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20170722095132/http://www3.boj.or.jp/kanazawa/kouhyou/report/report16j.pdf |archivedate=2017年7月22日 |df= }}</ref>。2015年，富山縣更首次成為日本最大的醫藥品生產縣<ref>{{cite web |url=http://www.nikkei.com/article/DGXLZO14664890Z20C17A3LB0000/ |script-title=ja:15年医薬品生産額 初の首位　富山県、受託拡大  |accessdate=2017-06-06 |publisher={{lang|ja|日本経済新聞}} |language=ja}}</ref>。

=== 第三級產業 ===
[[北陸銀行|北陸銀行]]總部位於富山市，是日本大型地方銀行之一，以北陸三縣和[[北海道|北海道]]為主要營業範圍，並且在[[東京都|東京都]]、[[名古屋市|名古屋市]]、[[大阪市|大阪市]]等地也設有分行<ref>{{cite web |url=http://www.hokugin.co.jp/company/profile.html |script-title=ja:会社情報 |accessdate=2017-06-06 |publisher={{lang|ja|北陸銀行}} |language=ja}}</ref>。2003年，北陸銀行和[[北海道銀行|北海道銀行]]實現統合經營，兩者共創{{link-ja|北北金融集團|ほくほくフィナンシャルグループ}}<ref>{{cite web |url=http://www.hokuhoku-fg.co.jp/info/overview/history/ |script-title=ja:沿革 |accessdate=2017-06-06 |publisher={{lang|ja|ほくほくフィナンシャルグループ}} |language=ja}}</ref>。富山縣內還有{{link-ja|富山銀行|富山銀行}}<ref>{{cite web |url=http://www.toyamabank.co.jp/pages/info/bankdata.htm |script-title=ja:会社概要 |accessdate=2017-06-06 |publisher={{lang|ja|富山銀行}} |language=ja}}</ref>和{{link-ja|富山第一銀行|富山第一銀行}}<ref>{{cite web |url=https://www.first-bank.co.jp/outline/company/ |script-title=ja:会社情報 |accessdate=2017-06-06 |publisher={{lang|ja|富山第一銀行}} |language=ja}}</ref>兩家規模較小的地方銀行。2014年，富山縣商品銷售額有27,584億日元<ref>{{cite web |url=http://www.pref.toyama.jp/sections/1015/lib/shogyo/rep26/gaiyo/tyosakekka.pdf |script-title=ja:調査結果の概要  |accessdate=2017-06-06 |publisher={{lang|ja|富山県}} |language=ja}}</ref>。富山縣六成的批發業銷售額和四成的零售業銷售額集中在富山市<ref>{{cite web |url=http://www.meti.go.jp/statistics/tyo/syougyo/dms/2009/pdf/ken16.pdf |script-title=ja:富山県の事業所数、年間商品販売額はともに富山市に集中  |accessdate=2017-06-06 |publisher={{lang|ja|経済産業省}} |language=ja}}</ref>。由於富山縣是日本私家車最普及的地區之一且公路交通便利，利於郊外大型購物中心的發展，導致富山、高岡兩市的中心地區出現嚴重的空洞化問題。富山縣已通過發展公共交通等方式試圖解決這一問題<ref name="Chishi"></ref>{{rp|506-509}}。2015年，受北陸新幹線開業的影響，富山縣的觀光客數大幅增加17.5%，達3412.6萬人<ref>{{cite web |url=http://www.pref.toyama.jp/cms_pfile/00013393/00918794.pdf |script-title=ja:平成２７年富山県観光客入込数（推計） |accessdate=2017-06-06 |publisher={{lang|ja|富山県}} |language=ja }}{{Dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

== 文化 ==
=== 方言 ===
富山縣地處日語東西方言的分界線，是不同方言融合之後產生的接觸方言<ref name="Dialect">{{cite book |author={{lang|ja|平山輝男}}|script-title=ja:《富山県のことば》 |year=1998 |publisher={{lang|ja|明治書院}} |location=東京都 |isbn=4-625-52216-1|language=ja}}</ref>{{rp|6}}。{{link-ja|富山方言|富山弁}}可以分為吳東方言、吳西方言、五箇山方言三個地區<ref name="Dialect"></ref>{{rp|2}}。雖然富山縣方言屬於{{link-ja|西日本方言|西日本方言}}中的{{link-ja|北陸方言|北陸方言}}，但吳東方言有較強的{{link-ja|東日本方言|東日本方言}}色彩<ref name="Dialect"></ref>{{rp|2}}。
=== 飲食 ===
富山縣地處東西日本交界地這一特徵也體現在富山縣的飲食文化上，例如正月的[[雜煮|雜煮]]就同時融合了關東、關西的口味<ref name="food">{{cite web |url=http://www.pref.toyama.jp/branches/1133/derukui/vol200503/dk21p04_07.pdf |script-title=ja:とやまの食文化について |accessdate=2017-06-06 |publisher={{lang|ja|富山県}} |language=ja}}</ref>。由於富山在江戶時代是{{link-ja|北前船|北前船}}[[海帶|海帶]]貿易的據點，因此人均海帶消費量位居日本前列<ref>{{cite web |url=http://www.shoku-toyama.jp/gourmet/kaidou/ |script-title=ja:グルメ |accessdate=2017-06-06 |publisher={{lang|ja|富山県}} |language=ja}}</ref>。富山灣豐富的海產使得富山縣飲食文化以水產品為主，白蝦等富山特有的食材在當地的飲食文化中有重要地位<ref name="food"></ref>。{{link-ja|冰見烏冬|氷見うどん}}是日本三大烏冬之一<ref>{{cite web |url=http://himi-aoyagi.com/himi_udon.html |script-title=ja:氷見うどんについて |accessdate=2017-06-06 |publisher={{lang|ja|氷見 民宿青柳}} |language=ja}}</ref>。

=== 教育 ===
[[富山大學|富山大學]]創建於1949年，在富山市、高岡市均有校區，設有人文學院、人間發達科學院、經濟學院、理學院、醫學院、藥學院、工學院、藝術文化學院八個學院，擁有超過8,000名學生，是富山縣唯一的國立大學<ref>{{cite web |url=http://www.u-toyama.ac.jp/outline/about/index.html |script-title=ja:富山大学概要 |accessdate=2017-06-06 |publisher={{lang|ja|富山大学}} |language=ja}}</ref>。[[富山縣立大學|富山縣立大學]]是富山唯一的公立大學，創建於1990年。只有工學院一個學院<ref>{{cite web |url=http://www.pu-toyama.ac.jp/about/kengaku-outline/2013/03/18/1003/ |script-title=ja:建学の理念と沿革 |accessdate=2017-06-06 |publisher={{lang|ja|富山県立大学}} |language=ja}}</ref>。此外富山縣還有高岡法科大學、富山國際大學等私立大學<ref>{{cite web |url=http://www.cn-toyama.jp/university/index.html |script-title=ja:県内大学一覧 |accessdate=2017-06-06 |publisher={{lang|ja|富山県大学連携協議会}} |language=ja}}</ref>。

== 交通 ==
[[File:E7_F12_Shin-Takaoka_20150314.jpg|thumb]]
富山縣內大部分鐵路由[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）運營，包括南北縱貫富山縣中部[[高山本線|高山本線]]（豬谷站以南部分由[[JR東海|JR東海]]營運）、富山縣西部的[[城端線|城端線]]及[[冰見線|冰見線]]三條在來線<ref>{{cite web |url=http://johana-himisen.com/about/ |script-title=ja:城端線・氷見線について |accessdate=2017-06-06 |publisher={{lang|ja|城端線・氷見線観光ガイド}} |language=ja}}</ref>。2015年，北陸新幹線通車，乘坐最快的[[光輝號列車|光輝號列車]]只需約2小時10分就可從[[東京站|東京站]]到達[[富山站_(日本)|富山站]]<ref name="shinkansen"></ref>。在北陸新幹線通車之後，富山縣內的[[北陸本線|北陸本線]]變為[[日本第三部門鐵道|第三部門鐵路]][[愛之風富山鐵道線|愛之風富山鐵道線]]，由[[愛之風富山鐵道|愛之風富山鐵道]]運營<ref>{{cite web |url=http://ainokaze.co.jp/station |script-title=ja:各駅情報 |accessdate=2017-06-06 |publisher={{lang|ja|あいの風とやま鉄道}} |language=ja}}</ref>。[[富山地方鐵道|富山地方鐵道]]是富山縣最大的私鐵公司，經營有[[富山地方鐵道本線|富山地方鐵道本線]]、[[立山線|立山線]]、[[不二越線|不二越線]]、[[上瀧線|上瀧線]]四條普通鐵路路線<ref>{{cite web |url=http://www.chitetsu.co.jp/?page_id=18887 |script-title=ja:鉄道 |accessdate=2017-06-06 |publisher={{lang|ja|富山地方鉄道}} |language=ja}}</ref>，以及{{link-ja|富山市内軌道線|富山地方鉄道富山市内軌道線|富山市內的路面電車}}<ref>{{cite web |url=http://www.chitetsu.co.jp/english/jp/trams/ |script-title=ja:路面電車 |accessdate=2017-06-06 |publisher={{lang|ja|富山地方鉄道}} |language=ja}}</ref>。{{link-ja|萬葉線|万葉線}}是第三部門鐵路公司，在高岡市經營有{{link-ja|高岡軌道線|万葉線高岡軌道線}}、{{link-ja|新湊港線|万葉線新湊港線}}兩條路線<ref>{{cite web |url=http://www.manyosen.co.jp/timetable/goingup/index.html |script-title=ja:時刻表・運賃案内 |accessdate=2017-06-06 |publisher={{lang|ja|万葉線}} |language=ja}}</ref>。[[富山輕軌|富山輕軌]]的前身是JR富山港線，現在富山輕軌[[富山港線|富山港線]]已是富山北部重要的公共交通方式<ref>{{cite web |url=http://www.t-lr.co.jp/time/ |script-title=ja:ポートラム 時刻表・路線図案内 |accessdate=2017-06-06 |publisher={{lang|ja|富山ライトレール}} |language=ja}}</ref>。[[黑部峽谷鐵道|黑部峽谷鐵道]]原本是為修建水力發電站而設置的鐵路，現在是觀光鐵路<ref>{{cite web |url=http://www.kurotetu.co.jp/ensen/ |script-title=ja:沿線案内 |accessdate=2017-06-06 |publisher={{lang|ja|黒部峡谷鉄道}} |language=ja}}</ref>。

2012年，富山縣的道路整備率達74.9%，位於日本第一<ref>{{cite web |url=http://www.soumu.go.jp/main_content/000321481.pdf |script-title=ja:富山県 |accessdate=2017-06-06 |publisher={{lang|ja|総務省}} |language=ja }}{{Dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。富山縣內有東西橫貫沿海地區的[[北陸自動車道|北陸自動車道]]、連接高山地區和富山縣西部的[[東海北陸自動車道|東海北陸自動車道]]、連接能登半島和富山縣的[[能越自動車道|能越自動車道]]三條高速公路<ref>{{cite web |url=http://www.pref.toyama.jp/cms_sec/1501/kj00000273-002-01.html |script-title=ja:県内の高速道路 |accessdate=2017-06-06 |publisher={{lang|ja|富山県}} |language=ja}}</ref>。{{link-ja|伏木富山港|伏木富山港}}是日本的重要港灣，雖然以工業和貨物貿易為主，但亦有郵輪停靠，是本州日本海沿岸僅有的兩個國際據點港灣之一<ref>{{cite web |url=http://www.pref.toyama.jp/sections/1504/port/about/ |script-title=ja:伏木富山港のご案内 |accessdate=2017-06-06 |publisher={{lang|ja|伏木富山港}} |language=ja}}</ref>。[[富山機場|富山機場]]開業於1963年，跑道全長2,000米<ref>{{cite web |url=http://www.toyama-airport.jp/other/about |script-title=ja:富山きときと空港について |accessdate=2017-06-06 |publisher={{lang|ja|富山きときと空港}} |language=ja}}</ref>，運行有飛往[[東京國際機場|羽田]]、[[新千歲機場|札幌]]、[[仁川國際機場|首爾]]、[[大連國際機場|大連]]、[[浦東國際機場|上海]]、[[桃園國際機場|桃園]]的定期航線<ref>{{cite web |url=http://www.toyama-airport.jp/flight |script-title=ja:飛行機に乗る |accessdate=2017-06-06 |publisher={{lang|ja|富山きときと空港}} |language=ja}}</ref>。北陸新幹線開業之後，富山機場的乘客數急劇減少，2015年度的乘客數有67.5萬人次，只有前一年的68.4%<ref>{{cite web |url=http://www.toyama-airport.jp/wp-content/uploads/2016/06/00019790.pdf |script-title=ja:平成27年度 富山きときと空港の利用状況について |accessdate=2017-06-06 |publisher={{lang|ja|富山きときと空港}} |language=ja}}</ref>。

== 觀光 ==
[[File:20_Tateyama_from_Mikurigaike_1998-7-17.jpg|thumb]]
[[南礪市|南礪市]][[五箇山|五箇山]]以傳統民居建築[[合掌造|合掌造]]而著稱，因地理環境封閉而產生了獨特的文化。但自大正時期之後，五箇山因水電站開發而開通了公路，交通狀況大幅改善<ref name="Chishi"></ref>{{rp|547}}。1995年，五箇山和[[白川鄉|白川鄉]]被列入世界文化遺產，觀光客也因此大幅增加<ref>{{cite web |url=http://www.gokayama-info.jp/?page_id=76 |script-title=ja:世界遺産 五箇山 |accessdate=2017-06-06 |publisher={{lang|ja|五箇山彩歲}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20170916140652/http://www.gokayama-info.jp/?page_id=76 |archivedate=2017年9月16日 |df= }}</ref>。高岡市的{{link-ja|瑞龍寺 (高岡市)|瑞龍寺 (高岡市)|瑞龍寺}}是近世禪宗建築的代表作，也是富山縣唯一的[[日本國寶|日本國寶]]<ref>{{cite web |url=http://www.info-toyama.com/spot/21009/ |script-title=ja:国宝　高岡山瑞龍寺 |accessdate=2017-06-06 |publisher={{lang|ja|とやま観光推進機構}} |language=ja}}</ref>、[[高岡大佛|高岡大佛]]是「日本三大大佛」之一<ref>{{cite web |url=http://www.info-toyama.com/spot/21108/ |script-title=ja:高岡大仏 |accessdate=2017-06-06 |publisher={{lang|ja|とやま観光推進機構}} |language=ja}}</ref>。高岡市的{{link-ja|山町筋|山町筋}}是[[重要傳統的建造物群保存地區|重要傳統的建造物群保存地區]]<ref>{{cite web |url=http://www.info-toyama.com/spot/21120/ |script-title=ja:山町筋（土蔵造りのまち並み）|accessdate=2017-06-06 |publisher={{lang|ja|とやま観光推進機構}} |language=ja}}</ref>。{{link-ja|雨晴海岸|雨晴海岸}}是[[能登半島國定公園|能登半島國定公園]]的一部分，晴天時在此可一覽立山連峰景色<ref>{{cite web |url=http://www.info-toyama.com/spot/21011/ |script-title=ja:国定公園　雨晴海岸 |accessdate=2017-06-06 |publisher={{lang|ja|とやま観光推進機構}} |language=ja}}</ref>。冰見市沿海的冰見番屋街銷售有眾多在地美食，是冰見市觀光的中心<ref>{{cite web |url=http://www.info-toyama.com/spot/21006/ |script-title=ja:氷見漁港場外市場『ひみ番屋街』　氷見温泉郷『総湯』 |accessdate=2017-06-06 |publisher={{lang|ja|とやま観光推進機構}} |language=ja}}</ref>。

富山市的松川兩岸種植有約460株櫻花，松川遊覽船在春季是賞櫻勝地<ref>{{cite web |url=http://matsukawa-cruise.jp/about/nanahashi/ |script-title=ja:コースと見どころ |accessdate=2017-06-06 |publisher={{lang|ja|松川遊覧船}} |language=ja}}</ref>。[[富岩運河環水公園|富岩運河環水公園]]利用運河修建親水公園，是近年日本都市公園的成功典範<ref>{{cite web |url=http://www.kansui-park.jp/guide/ |script-title=ja:施設案内 |accessdate=2017-06-06 |publisher={{lang|ja|富岩運河環水公園}} |language=ja}}</ref>。富山縣東部的[[立山黑部阿爾卑斯路線|立山黑部阿爾卑斯路線]]於1971年全線開通<ref name="Chishi"></ref>{{rp|529}}，是世界著名的高山觀光路線，隨季節變化而在各時期呈現截然不同的景色。立山黑部阿爾卑斯路線每年4月至5月的「雪之大谷」的最深積雪可達近20米<ref>{{cite web |url=http://www.alpen-route.com/enjoy_navi/snow_otani/ |script-title=ja:雪の大谷を楽しむ |accessdate=2017-06-06 |publisher={{lang|ja|立山黒部アルペンルート}} |language=ja}}</ref>。夏季的立山黑部阿爾卑斯路線則以豐富的高山植物景觀<ref>{{cite web |url=http://www.alpen-route.com/enjoy_navi/field_note/alpine_plant.html |script-title=ja:可憐で色鮮やか！貴重な高山植物たち |accessdate=2017-06-06 |publisher={{lang|ja|立山黒部アルペンルート}} |language=ja}}</ref>和黑部水庫的觀光放水而著稱<ref>{{cite web |url=http://www.kurobe-dam.com/kankou/housui.html |script-title=ja:黒部ダム観光放水 |accessdate=2017-06-06 |publisher={{lang|ja|黒部ダム}} |language=ja}}</ref>。秋季的立山黑部阿爾卑斯路線是觀賞紅葉的勝地<ref>{{cite web |url=http://www.alpen-route.com/enjoy_navi/seasons/autumn.html |script-title=ja:秋　紅葉の季節　|accessdate=2017-06-06 |publisher={{lang|ja|立山黒部アルペンルート}} |language=ja}}</ref>。黑部峽谷內的宇奈月溫泉是日本透明度最高的溫泉<ref>{{cite web |url=http://www.unazuki-onsen.com/hotspring |script-title=ja:美肌の湯　|accessdate=2017-06-06 |publisher={{lang|ja|宇奈月温泉旅館協同組合}} |language=ja}}</ref>，昔日難以抵達，但隨著水電開發的鐵路興建，現在已發展為著名觀光地。
