'''株式會社富士媒體控股'''（{{Lang-ja|'''株式会社フジ・メディア・ホールディングス'''}}，{{Lang-en|'''Fuji Media Holdings, Inc.'''}}）是日本大型传媒集团[[富士产经集团|富士产经集团]]的[[控股公司|控股公司]]，也是日本首家基於{{Link-ja|广播法 (日本)|放送法|广播法}}成立的{{Link-ja|廣播控股公司|放送持株会社|特许广播控股公司}}。其[[法人|法人]]前身為1957年成立的[[富士電視台|株式會社富士電視]]，2008年因公司重組而改為現名。

== 公司概要 ==
2008年10月1日，原富士电视有限公司进行业务重组。原富士电视有限公司改名为富士媒体控股有限公司，并成为日本首家“广播控股公司”；同时公司的广播业务交由新成立的同名子公司[[富士电视|富士电视有限公司]]运营。<ref>テレビ朝日や日本テレビ放送網・中部日本放送と異なり、準備会社を新設して一定期間の後に放送事業や放送免許等を移管する方法を採用しなかった。</ref> 公司本部所在的[[富士产经集团大楼|富士产经集团大楼]]及东京{{Link-ja|产经大厦|サンケイビル}}因牵连到[[活力门事件|活力门事件]]而被视为赔偿抵押物，而新成立的富士媒体控股公司需要继续承担该法律义务。<ref>{{cite web |url=http://qoe.jp/file/ES0000CQ7 |title=存档副本 |accessdate=2017-04-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20090310010027/http://qoe.jp/file/ES0000CQ7 |archivedate=2009-03-10 }}</ref>

作为广播持股公司，富士媒体控股有限公司将[[关东地方|关东地区]]的地面电视台[[富士电视台|富士电视台]]、卫星电视台[[BS富士|BS富士]]、地面电视台[[日本放送|日本放送]]列为自己的子公司；2006年12月，[[富士电视网|富士电视网]]旗下[[仙台放送|仙台放送]]，以及原本的一些地方电视台也成为子公司。<ref>{{Cite news|title=＜フジＨＤ＞仙台放送を子会社化|newspaper=河北新報オンラインニュース|date=2016-12-01|url=http://www.kahoku.co.jp/tohokunews/201612/20161201_12058.html|accessdate=2016-12-12|deadurl=yes|archiveurl=https://web.archive.org/web/20161202114943/http://www.kahoku.co.jp/tohokunews/201612/20161201_12058.html|archivedate=2016-12-02}}</ref> 而其他诸如[[WOWOW|WOWOW]]和富士电视网关联电视台则只是持股公司。
