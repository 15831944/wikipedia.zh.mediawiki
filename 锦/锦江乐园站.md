'''锦江乐园站'''位于[[上海市|上海市]][[徐汇区|徐汇区]][[沪闵路|沪闵路]][[虹梅路|虹梅路]]东侧，近[[锦江乐园|锦江乐园]]和铁路[[梅陇火车站|梅陇火车站]]，是[[上海轨道交通1号线|上海轨道交通1号线]]的地面[[侧式站台|侧式车站]]（两个方向在付费区内不连通）。由上海铁路局勘测设计院设计<ref name="上海市政工程志设计">[http://www.shtong.gov.cn/newsite/node2/node2245/node68289/node68299/node68387/node68393/userobject1ai65937.html 《上海市政工程志》>>第六篇 地下铁道>>第二章 一号线前期工作>>第三节 设计] {{webarchive|url=https://web.archive.org/web/20110827021106/http://www.shtong.gov.cn/newsite/node2/node2245/node68289/node68299/node68387/node68393/userobject1ai65937.html |date=2011-08-27 }}</ref>。

==车站结构==
{{multiple image|align=right|direction=vertical|width=220
|image1=Platform of Jinjiang Park Station.jpg|caption1=1号线站台
}}
该站与虹梅商务大厦共构，位于大厦一层。车站的构造是上海轨道交通较为特有的。由于它并非[[岛式站台|岛式站台]]，加上它是地面车站，因此开往[[莘庄站|莘庄]]方向的列车的乘客只需在虹梅路的出入站口进站或出站，但往[[富锦路站|富锦路]]的列车的乘客就必须先到车站的二楼，然后再下楼乘搭列车。售票厅也位于二楼。

==接驳交通==
=== “P+R”停车换乘 ===
自2009年7月21日起，锦江乐园站加入上海“P+R”停车换乘试点，自驾车的市民在每天的5时30分至23时30分时段内，使用[[上海公共交通卡|公共交通卡]]刷卡进入锦江乐园站的虹梅路停车楼，并用该卡换乘轨道交通，即可享受停车收费优惠。试行收费标准为在换乘服务时段内按次收费，每次[[人民币|人民币]]10元。<ref>{{cite web |title = 上海试点"P+R" 停车换乘轨交享优惠 |publisher = xinhuanet.com |date = 2009-07-20 |url = http://www.xinhuanet.com/chinanews/2009-07/20/content_17140460.htm |accessdate = 2009-07-22 |language = zh }}</ref>

== 历史 ==
锦江乐园站刚启用时位于[[沪闵路|沪闵路]][[虹漕南路|虹漕南路]]西侧。由于当时是作为起讫站，因此车站形式为地面[[侧式站台|单侧式站台]]，后因兴建南延伸工程而在现址兴建虹梅商务大厦和新地铁站，原锦江乐园站则拆除。由于新站位置在虹梅路，而最初被称为虹梅路站。2001年5月1日，虹梅路站重新改名为锦江乐园站<ref>{{cite web |title = 上海虹梅路站改名锦江乐园站 |publisher = Sina.com |date = 2001-04-25 |url = http://news.sina.com.cn/c/239682.html |accessdate = 2014-08-18 |language = zh }}</ref>
