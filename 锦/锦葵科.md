'''锦葵科'''（[[學名|學名]]：''{{lang|la|Malvaceae}}''）是[[锦葵目|锦葵目]]的一个[[科|科]]。锦葵科大约有75属，1000－1500种，分布于[[温带|温带]]及[[热带|热带]]。[[中国|中国]]有16属，81种，36变种或变型。

== 形态 ==
* [[草本|草本]]、[[灌木|灌木]]或[[乔木|乔木]]，常有星状毛，有粘液；
* 叶互生，单叶，掌状脉；
* 花常两性，辐射对称，萼片5枚，其下屡有总苞状小苞，花瓣5枚，雄蕊常多数，花丝合成一柱，多少与花瓣基部合生，花药一室，子房上位，二至多室；
* 果实为[[蒴果|蒴果]]，或分离成数个小干果，偶然有肉质果。

== 分类 ==
'''锦葵科'''包括以下一些属：
* [[秋葵属|秋葵属]]（''Abelmoschus'' Medik.）：例如：[[秋葵|秋葵]]（''A. esculentus'' (L.) Moench）、[[黄蜀葵|黄蜀葵]]（''A.manihot'' (L.) Medic.）
* [[苘麻属|苘麻属]]（''Abutilon'' Mill. ）：例如：[[苘麻|苘麻]]（''A. thephrasti'' Medicus）
* [[蜀葵属|蜀葵属]]（''Althaea'' L.）：例如：[[蜀葵|蜀葵]]（''A. rosea'' (L.) Cavan）
* [[木槿属|木槿属]]（''Hibiscus'' L.）：例如：[[木芙蓉|木芙蓉]]（''H. mutabilis'' L.）、[[朱槿|朱槿]]（''H. rosa-sinensis'' L.）、[[野西瓜苗|野西瓜苗]] （''H. trionum'' L.）
* [[锦葵属|锦葵属]]（''Malva'' L.）：例如：[[锦葵|锦葵]]（''M. sylvestris'' L.）、[[冬葵|冬葵]]（''M. verticillata'' L.）
* [[梵天花属|梵天花属]]（''Urena'' L.）：例如：[[肖梵天花|肖梵天花]]（''U. lobbata'' L.）、[[梵天花|梵天花]]（''U. proccumbens'' L.）
* [[棉花属|棉属]]（''Gossypium'' L.）：例如：[[树棉|树棉]]（''G.arboreum'' L.）、[[海岛棉|海岛棉]]（''G. barbadense'' L.）

在[[APG_分类法|APG 分类法]]和修订后的[[APG_II_分类法|APG II 分类法]]中，将[[梧桐科|梧桐科]]、[[木棉科|木棉科]]、[[椴树科|椴树科]]合并到锦葵科中间来，将锦葵科分为几个[[亚科|亚科]]：
# [[木棉亚科|木棉亚科]]，原来属于[[木棉科|木棉科]]的植物
# [[杯萼亚科|杯萼亚科]]，原来属于[[椴树科|椴树科]]的植物
# [[刺果藤亚科|刺果藤亚科]]，原来属于[[梧桐科|梧桐科]][[刺果藤属|刺果藤属]]的植物
# [[非洲芙蓉亚科|非洲芙蓉亚科]]，原来属于[[梧桐科|梧桐科]]的[[非洲芙蓉属|非洲芙蓉属]]植物
# [[扁担杆亚科|扁担杆亚科]]，原来属于[[椴树科|椴树科]][[扁担杆属|扁担杆属]]的植物
# [[山芝麻亚科|山芝麻亚科]]，原来[[梧桐科|梧桐科]]中[[山芝麻属|山芝麻属]]的植物和[[木棉科|木棉科]]中[[榴莲属|榴莲属]]的植物
# [[锦葵亚科|锦葵亚科]]，原来传统中属于锦葵科的植物
# [[梧桐亚科|梧桐亚科]]，原来属于[[梧桐科|梧桐科]]的植物
# [[椴树亚科|椴树亚科]]，原来属于[[椴树科|椴树科]]的植物
