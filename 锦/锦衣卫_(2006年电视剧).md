《'''锦衣卫'''》（{{lang-en|'''Imperial Guards'''}}），2006年中国大陆电视剧，导演[[张黎|张黎]]，主演马伊琍、文章、黄志忠、谭凯，2006年10月23日'''广州综合频道'''首播。

==剧情==
该剧讲述明朝中晚期明神宗到明思宗之间，明朝三大[[锦衣卫|锦衣卫]]“萧天云、钱宁、杨天石”三人和与美女“客印月”之间的爱恨情仇。<ref>{{cite web |url= http://yule.sohu.com/20100123/n269789592.shtml |title=资料：《锦衣卫》内容简介 |accessdate=2010 |publisher=搜狐网 |work= |language=zh-hans}}</ref><ref name="bee" >{{cite web |url=http://ent.sina.com.cn/v/2007-07-10/09551632076.shtml |title=资料：电视剧《锦衣卫》故事梗概 |accessdate=2007 |publisher=新浪网 |work= |language=zh-hans}}</ref>

==花絮==
该片收视率很低，而且由于过分不尊重历史事实，导致被观众批评“违背史实”<ref name="bull" >{{cite web |url=http://cul.sohu.com/20070723/n251193440.shtml |title=电视剧《锦衣卫》不尊重史实 引起观众诟病 |accessdate=2007 |publisher=搜狐网 |work= |language=zh-hans}}</ref>，导演张黎回应说“这是一部商业剧”。<ref>{{cite web |url=http://www.china.com.cn/culture/txt/2007-07/24/content_8570881.htm |title= 张黎：《锦衣卫》纯属商业片|accessdate=2007 |publisher=中国网 |work= |language=zh-hans}}</ref>

编剧张建伟曾是《[[走向共和|走向共和]]》和《[[军人机密|军人机密]]》的编剧；马伊琍一人分别出演三个角色。<ref>{{cite web |url=http://ent.china.com/zh_cn/movie/news/205/20050828/12607255.html |title=张黎开拍新片《锦衣卫》 马伊俐一人饰三角(图) |accessdate=2005 |publisher=中国网 |work= |language=zh-hans}}</ref>

该剧耗费人民币2000万元，拍摄地在[[北京市|北京市]]。<ref name="cat" >{{cite web |url=http://ent.sina.com.cn/x/2007-07-16/01361639139.shtml |title=每周电视质量报告——电视剧《锦衣卫》新说成胡说 |accessdate=2007 |publisher=新浪网 |work= |language=zh-hans}}</ref>
