'''屯門'''（{{lang-en|Tuen Mun}}），舊名'''青山'''，位於[[香港|香港]][[新界|新界]]西部，是[[香港行政區劃|香港十八區]]之一的[[屯門區|屯門區]]的主要區域，[[屯門新市鎮|屯門新市鎮]]座落其中。「屯門」或「屯門市」的範圍通常指包括北至[[兆康苑|兆康苑]]、[[富泰邨|富泰邨]]，西至[[蝴蝶灣|蝴蝶灣]]，東至[[三聖邨|三聖邨]]的區域；屯門新市鎮所有[[香港公共房屋|公共房屋]]及主要社區設施均位於這個範圍。不過如果根據香港政府「分區計劃大綱圖」，屯門的範圍包括西至[[踏石角|踏石角]]，東至[[深井|深井]]丶[[青龍頭|青龍頭]]附近一帶，北至[[藍地|藍地]]。

==歷史及發展==
{{main|屯門軍鎮|屯門區|屯門新市鎮}}
「屯門」意即「屯兵之門」，也是「屯田防衛的海門」，是[[中國歷史|中國歷史]][[文獻|文獻]]最早出現的[[香港|香港]]地名<ref>《[[新唐書|新唐書]]》卷四十三「地理誌」：「有府二：曰綏南、番禺；有經略軍，'''屯門'''鎮兵」</ref>，比「香港」一詞還早。[[新石器時代|新石器時代]]已有人類居住{{fact}}。[[南朝宋|南朝宋]][[元嘉_(劉宋)|元嘉]]年間，相傳[[杯渡禪師|杯渡禪師]]到了屯門禪修。[[唐代|唐代]]開元二十四年，唐室發展南方，設[[屯門軍鎮|屯門軍鎮]]，駐軍二千人，屬於安南都護府，視為[[廣州|廣州]]南方的[[港口|港口]]重鎮、交通樞紐。唐代詩人[[韓愈|韓愈]]的《贈別元十八協律詩》中寫：「峽山逢颶風，雷電助撞捽。乘潮簸扶胥，近岸指一-{發}-。兩巖雖-{云}-牢，水石互飛發。屯門雖-{云}-高，亦映波浪沒。」描寫屯門海面的水勢湍急險峻的情景。而[[劉禹錫|劉禹錫]]的《踏潮歌》寫道：「屯門積日無風飆，滄浪不歸成踏潮。轟如鞭石屹且搖，互空欲駕黿鼍橋。」描寫屯門巨浪滔天的海面<ref name="香港歷史之謎"/><ref>{{Cite web |url=https://kwantailo.wordpress.com/2013/02/18/%E9%A6%99%E6%B8%AF%E6%95%85%E4%BA%8B%EF%BC%9A%E3%80%8A%E5%B1%AF%E9%96%80%E3%80%8B/ |title=香港故事：《屯門簡史》 |author=經濟日報文化生活網  |date=2013-02-18}}</ref>。有些歷史學家認為韓愈被[[唐玄宗|唐玄宗]]貶官至[[潮州|潮州]]任刺史時就是循海路經香港前往赴任，途中途經屯門，但有關說法仍存爭議，部分史家認為他循[[東江|東江]]前往，並未路經屯門<ref name="香港歷史之謎"/>。

屯門三面環山，屬優良的天然避風港，古代商船（如從[[波斯|波斯]]、[[印度|印度]]和[[南洋|南洋]]來的船泊）來華經商時，都會選擇先停駐屯門補給，再向北前往[[廣州|廣州]]等地<ref name="香港歷史之謎">{{cite book|author=爾東|editor=周海燕|title=《香港歷史之謎》|date=2007年4月|publisher=明報出版社有限公司|location=香港|isbn=978-962-973-634-7|page=頁74-77}}</ref>。[[五代十國|五代十國]]時，[[南漢|南漢]]在屯門設市舶部門，對各國來華經商船隻抽解（意思是舊時對沿海港口進出口貿易徵收稅款），這個市舶部門便逐漸成為南漢主要稅收來源之一<ref name="香港歷史之謎"/>。

1898年，[[英國|英國]]向[[清政府|清政府]]租借新界時，屯門併入[[元朗|元朗約]]，以該地山峰[[青山_(香港)|青山]]為名，稱為'''青山分約'''。

1910年從元朗獨立出來，設立'''青山區'''，由[[理民府|青山理民府]]管轄。

1960年代，[[香港政府|香港政府]]開始計劃在其上發展第一代衛星城市（後改稱[[香港新市鎮|新市鎮]]），名為'''青山新市鎮'''；及後於1972年，青山區恢復使用舊名屯門，改名[[屯門區|屯門區]]；於1973年正式宣佈發展新市鎮<ref>香港亞洲電視，《[[當年今日|當年今日]]》，12月23日</ref>，名為'''[[屯門新市鎮|屯門新市鎮]]'''。

時至今天，屯門仍然在香港航運交通上擔當重要角色，[[屯門內河碼頭|屯門內河碼頭]]即設於鄰近[[珠江口|珠江口]]的屯門新市鎮西側。在2013年2月27日，時任[[財政司司長|財政司司長]][[曾俊華|曾俊華]]透過《[[2013年至2014年度香港政府財政預算案|2013年至2014年度香港政府財政預算案]]》宣佈，為了進一步發展[[物流|物流]]及[[旅遊|旅遊]]兩大支柱產業，將會預留屯門西一塊達10.8公頃的土地，開發新工業邨，發展物流[[基礎建設|基礎建設]]，使到不同的物流工序及服務公司集中，形成高端服務群組，項目暫定2019年展開，2023年完成，最多製造約6600個職位。<ref>[http://hk.news.yahoo.com/政府預留青衣屯門地發展物流業-055548853.html 政府預留青衣屯門地發展物流業]</ref><ref>[http://hk.news.yahoo.com/青衣屯門發展現代物流業-032617150.html 青衣屯門發展現代物流業] {{webarchive|url=https://web.archive.org/web/20130303010932/http://hk.news.yahoo.com/%E9%9D%92%E8%A1%A3%E5%B1%AF%E9%96%80%E7%99%BC%E5%B1%95%E7%8F%BE%E4%BB%A3%E7%89%A9%E6%B5%81%E6%A5%AD-032617150.html |date=2013-03-03 }} 《明報》 2013年2月27日</ref><ref>[http://hk.news.yahoo.com/預算案-預留屯門西土地發展物流設施-031900127.html 預算案:預留屯門西土地發展物流設施] 《星島日報》 2013年2月27日</ref>。

19世紀末[[青山灣|青山灣]]一帶（今[[三聖墟|三聖墟]]）已有人居住，[[后角天后廟|-{后}-角天后廟]]（今[[屯門舊墟|舊墟]]）及[[屯門新墟|新墟]]一帶是當時的[[市集|市集]]，新墟人口僅有約250人<ref name="屯門居民手冊" />。屯門的[[商業|商業]][[購物|購物]]中心於新市鎮發展後始移往'''[[屯門市中心|屯門市中心]]'''。 洪橋，亦名為4B街。洪橋，富泰，良景，治安極差，皆因一名販賣人口頭子呂偉信，他主要犯案路線在幼稚園，小學門口等地點拐幼童上內地賺取暴利。

== 著名學府 ==
[[File:Lingnan_University_Campus_Overview_201410.jpg|thumb]]
屯門區有38所[[中学|中學]]，36[[小學|小學]]及多間[[幼稚園|幼稚園]]。區內設有1995年遷入[[虎地|虎地]]的[[嶺南大學_(香港)|嶺南大學]]。區內亦有兩所大專院校，包括[[香港專業教育學院|香港專業教育學院]]（屯門分校）及2016年遷入的[[珠海學院|珠海學院]]。

到2012年起，屯門區有國際著名學校進駐，包括[[哈羅香港國際學校|哈羅香港國際學校]]。

== 交通運輸 ==
[[File:Light_Rail_Track_in_Tuen_Mun_201207.jpg|thumb]]是屯門區內主要交通工具之一]]
1899年尚未興建馬路，只靠水路運輸<ref name="屯門居民手冊">
{{cite book zh |author=屯門區議會 |title=《屯門居民手冊》 |publisher=屯門區議會 |location=香港 |date=2005 }}</ref>。

[[青山公路|青山公路]]元朗至青山段1914年落成、荃灣至青山段1919年落成<ref>
{{cite book zh |author=屯門區議會 |title=《屯門風物志》 |location=香港 |publisher=屯門區議會 |date=2007 |pages=42 }}</ref>。

為配合新市鎮發展，1970年代斥資數億港元動工興建[[屯門公路|屯門公路]]，1983年全線通車。但由於彎多路斜，屯門公路每天繁忙時間必定[[塞車|塞車]]，屯門居民過了十年對外交通不便之日子，直至[[三號幹線|三號幹線]]1998年上半年全面通車，西鐵（今[[西鐵綫|西鐵綫]]）2003年通車，前往港九區之交通才大大舒緩。

屯門內部交通工具有[[巴士|巴士]]、[[香港的士#.E6.96.B0.E7.95.8C.E7.9A.84.E5.A3.AB|新界的士]]及[[小巴|小巴]]等，其中較有特色的是於80年代建成的[[香港輕鐵|輕鐵]]。輕鐵於1988年9月18日首次通車<ref>[http://www.mtr.com.hk/chi/lr_bus/intro_index.html 輕鐵及巴士服務 簡介 － 港鐵]</ref>，並於其後作出多次擴展。由於有專利的「輕鐵專區」，輕鐵一直是接駁屯門和[[元朗|元朗]]的主要交通工具。直至2003年12月20日[[西鐵綫|西鐵]]的建成，輕鐵在區內的角色漸漸變成接駁西鐵的交通工具。於2007年12月2日，[[九廣鐵路|九廣鐵路]]與[[香港地鐵|香港地鐵]]正式[[兩鐵合併|合併]]，西鐵和輕鐵亦成為[[港鐵|港鐵系統]]的一部份，西鐵亦易名為西鐵綫。過往[[屯門碼頭|屯門碼頭]]有飛翔船往返[[中環碼頭|中環碼頭]]，不少上班一族會乘坐上班。但在2000年代這現象逐漸式微，由過海巴士和西鐵綫取而代之。港鐵建議輕鐵發展到天水圍北面，但未獲當局接納。
