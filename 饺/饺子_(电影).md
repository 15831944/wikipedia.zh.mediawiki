《'''饺子'''》（{{lang-en|'''Dumplings'''}}）香港[[恐怖片|恐怖片]]，导演陈果，由[[杨千嬅|杨千嬅]]領銜主演。<ref>{{cite web |url=http://ent.sina.com.cn/s/h/2005-04-30/1831715627.html |title=杨千嬅曾是三级女星？ 原是受“饺子”无辜连累 |accessdate=2005 |publisher=新浪网 |work= |language=zh-hans}}</ref>

==剧情==
该片是香港、日本、韩国合拍片《[[三更2|三更2]]》的一个单元。

该片讲述过气女星艾菁菁面对色衰爱弛的情况，用吃婴儿药引子的做法养颜，慢慢恢复青春，但是最后，她全身散发出腐肉气味。

==軼事==
*此電影是[[楊千嬅|楊千嬅]]首次擔任[[三级片|三级片]]第一女主角。
*部分观众认为这是一部[[三级片|三级片]]，但是杨千嬅否认。<ref>{{cite web |url=http://hn.rednet.cn/c/2004/08/23/601292.htm |title=杨千嬅演三级片《三更》 严父一直黑脸(图)  |accessdate=2004 |publisher=红网 |work= |language=zh-hans}}</ref>
*该片改变了杨千嬅的婚姻观。<ref>{{cite web|url=http://news.xinhuanet.com/ent/2005-01/05/content_2417132.htm|title=杨千嬅的婚姻观:被"一顿"《饺子》改写了(图)|accessdate=2005|publisher=新华网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20170130090206/http://news.xinhuanet.com/ent/2005-01/05/content_2417132.htm|archivedate=2017-01-30}}</ref>
*该片参加[[柏林国际电影节|柏林国际电影节]]、[[威尼斯国际电影节|威尼斯国际电影节]]。<ref>{{cite web |url=http://ent.sina.com.cn/x/2005-02-13/1135653599.html |title=资料：柏林电影节全景单元参赛片《饺子》简介 |accessdate=2005 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/2004-09-07/0139496358.html |title=中国恐怖电影威尼斯上映 “饺子”吓坏外国老记 |accessdate=2004 |publisher=新浪网 |work= |language=zh-hans}}</ref>
*该片是杨千嬅的第一次“[[床戏|床戏]]”。<ref>{{cite web |url=http://ent.163.com/ent_2003/editor/movie/film/040817/040817_300981.html |title= 《三更2之饺子》：杨千嬅恐怖第一次|accessdate=2004 |publisher=网易 |work= |language=zh-hans}}</ref>
*白靈憑本片獲[[第24屆香港電影金像獎|第24屆]][[香港電影金像獎最佳女配角|香港電影金像獎最佳女配角]]、第41屆金馬獎最佳女配角、第10屆香港電影金紫荊獎最佳女配角、第5屆華語電影傳媒大獎最佳女配角。
