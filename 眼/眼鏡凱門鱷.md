'''眼鏡凱門鱷'''（[[學名|學名]]：''{{lang|la|Caiman crocodilus}}''），又名'''南美短吻鱷'''、'''眼鏡鱷'''，是一種[[鱷魚|鱷魚]]，因為眼球前端有一條橫骨，就像[[眼鏡|眼鏡]]架一樣，因而得名。

眼鏡凱門鱷體長最大可達2.5[[米|米]]，初孵出來的幼鱷有20至25厘米長。眼鏡凱門鱷[[嘴|嘴]]長，背面呈[[橄欖|橄欖]][[綠色|綠色]]，[[頭|頭]]部、身體、[[尾|尾]]部上有個多深色斑紋，[[背|背]]部及尾部上則有深[[棕色|棕色]]或[[黑色|黑色]]的橫帶紋，肚皮的顏色則是純[[米色|米色]]或者淺[[黃色|黃色]]。初孵出來的幼鱷下[[頜|頜]]兩側有淡色斑，長大到大約35厘米長時，這些斑紋會全部消失。

== 分佈地 ==
眼鏡凱門鱷原產於[[中美洲|中美洲]]及[[南美洲|南美洲]]一帶，不過其後牠們被人類帶到[[美國|美國]][[佛羅里達州|佛羅里達州]]及[[古巴|古巴]]。

== 習性 ==

眼鏡凱門鱷一般棲居於廣泛的水域棲息地。與其他鱷魚一樣，食物隨年齡、季節及棲息地的不同而變化。幼鱷主要以[[無脊椎動物|無脊椎動物]]為食，特別是[[鞘翅目|鞘翅目]][[昆蟲|昆蟲]]，也會進食[[螺|螺]]、[[蝦|蝦]]及[[蟹|蟹]]等水生動物。成年鱷則主要以[[脊椎動物|脊椎動物]]為食，包括水生及陸生的脊椎動物。牠們的捕食策略包括靜伏不動，偷襲路過的陸生脊椎動物，於水中偷襲游近的[[魚類|魚類]]與其他水生脊椎動物。此外，牠們有時也用身體和尾巴驅趕魚類到淺水處，或者於開闊的水體將魚類驅趕到狹窄的岸邊後再捕食。

眼鏡凱門鱷的求愛行為包括躍出水面、炫耀尾巴，以及輕咬和摩擦頭部與頸部等。牠們於不同地點的造巢活動的高峰期也有不同，一般多在潮濕季節。牠們的巢由[[葉子|葉子]]、小[[樹枝|樹枝]]、雜草以及泥堆成小丘狀。

雌性的眼鏡凱門鱷每次產卵可達40顆，而整個孵化期雌性會有規律地關心和保護巢址。幼鱷出殼時發出叫聲，雌性將巢挖開護送雛鱷進入水中，而有時雄鱷也協助。雌鱷也會用嘴幫助胚鱷破殼而出，並用嘴攜帶幼鱷進入水中。

眼鏡凱門鱷於其一生中不同時期均有不同敵人。捕食鱷卵的動物包括有[[黑點雙領蜥|黑點雙領蜥]]、[[發冠長腳鷹|發冠長腳鷹]]、[[食蟹狐|食蟹狐]]、[[浣熊|浣熊]]、[[長鼻浣熊|長鼻浣熊]]和[[戴帽猴|戴帽猴]]等。而以雛鱷和幼鱷為食的則有[[鯰魚|鯰魚]]等食肉性魚類、[[龜|龜]]、[[鱷魚|鱷魚]]、[[夜鷺|夜鷺]]、[[鸛|鸛]]、[[鷹|鷹]]、[[虎貓|虎貓]]以及[[家豬|家豬]]等。[[美洲豹|美洲豹]]及[[森蚺|森蚺]]就會捕食成年鱷魚。

== 種群動態 ==

眼鏡凱門鱷分佈廣且數量多，是提供國際[[鱷皮|鱷皮]]貿易的最主要種類。於原生地以外地方，牠們是極具威脅性的[[外來物種|外來物種]]。
