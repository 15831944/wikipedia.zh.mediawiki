'''尹中卿'''（{{bd|1959年|2月||}}）[[河南|河南]][[桐柏|桐柏]]人。[[中国共产党|中国共产党]]党员。[[中华人民共和国|中华人民共和国]]官员。<ref name=ifeng>[http://finance.ifeng.com/people/official/yinzhongqing.shtml 尹中卿，凤凰网，于2013-03-16查阅]</ref>

==生平==
1985年，尹中卿自[[武汉大学|武汉大学]]政治经济学研究生毕业。1986年11月后，任[[全国人大常委会办公厅研究室|全国人大常委会办公厅研究室]]经济组副组长、组长。1993年8月后，任全国人大常委会办公厅研究室副主任。2004年10月后，任全国人大常委会办公厅研究室主任、常委会机关党组成员。2008年3月，出任第十一届[[全国人民代表大会财政经济委员会|全国人大财经委员会]]委员。2008年12月起，任第十一届全国人大财经委员会副主任委员<ref name=ifeng/><ref name=wy>{{Cite web |url = http://money.163.com/special/2015naec_t_yzq/ |title =尹中卿  |author =  |date =  |publisher =网易  |language =  |accessdate = 2018-03-04 |quote =  }}</ref>。2013年3月起，任[[第十二届全国人大常委会|第十二届全国人大常委会]]委员、财经委员会副主任委员<ref name=diy>[http://www.court.gov.cn/xwzx/rdzt/2013qglh/ldr/201303/t20130315_182710.html 中华人民共和国全国人民代表大会公告（第一号），中华人民共和国第十二届全国人民代表大会第一次会议主席团，2013年3月14日] {{webarchive|url=https://web.archive.org/web/20130517145057/http://www.court.gov.cn/xwzx/rdzt/2013qglh/ldr/201303/t20130315_182710.html |date=2013年05月17日 }}</ref>。

2015年12月26日，[[第十二届全国人大常委会|第十二届全国人大常委会]]第十八次会议举行联组会议，就[[中华人民共和国国务院|国务院]]关于2014年度中央预算执行和其他财政收支审计查出问题整改情况的报告进行专题询问。据此次审计报告反映，部分地区彩票资金管理不严格，抽查发现虚报、套取、挤占、挪用等问题金额达169.32亿元人民币，占抽查资金的26%。其中的31.47亿元人民币违规用于购买办公楼、建立培训中心等，4.55亿元人民币违规用于购车、组织出国出境旅游、发放津贴补贴。尹中卿在专题询问中，向[[中华人民共和国民政部|民政部]]部长[[李立国|李立国]]和[[国家体育总局|国家体育总局]]局长[[刘鹏|刘鹏]]提问，要求解释为何存在这些问题，如何整改，违纪违规违法资金是否已全部收回，对有关责任人员是否已问责、追责<ref>{{Cite web |url =http://news.cntv.cn/2015/12/30/VIDE1451478775739863.shtml  |title =监督的力量  |author =  |date =2015-12-30  |publisher = 中国网络电视台 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://sports.qq.com/a/20151227/011519.htm |title = 170亿彩票问题资金去哪了？人大常委们怒了！ |author =  |date =2015-12-27  |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。

==著作==

*《中国未来五年走向》
*《社会主义法制理论读本》
*《人大研究文萃》
*《中国人大组织构成和工作制度》<ref name=wy/>
