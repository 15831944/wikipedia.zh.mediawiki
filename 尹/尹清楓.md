'''尹清楓''' {{BD|1946年|2月8日<ref>[https://www.youtube.com/watch?v=mY8l_rExGfY 2010-05-04公視晚間新聞(拉法葉軍購 衍生佣金弊案.尹清楓案)]1:05新聞資料畫面的尹清楓[[中華民國國民身分證|中華民國國民身分證]]出生日期</ref>|1993年|12月9日|catIdx=Yin尹清楓}} ，[[中華民國海軍|中華民國海軍]][[上校|上校]]（佔[[少將|少將]]缺），生於[[甘肅省|甘肅省]][[蘭州市|蘭州市]]，籍貫[[山東省|山東省]][[濟南|濟南]]，在1949年[[第二次國共內戰|第二次國共內戰]]期間遷居[[臺灣省|臺灣省]][[高雄縣|高雄縣]][[岡山區|岡山鎮]]，畢業於[[海軍工程學院|海軍工程學院]]機械工程系57年班（就讀期間該校被併入剛改名的[[中正理工學院|中正理工學院]]）<ref>[http://mypaper.pchome.com.tw/mayer009/post/1247538809 軍中人事派系的大傾軋 - 拉法葉疑雲] - PChome 個人新聞台</ref>，被害前其階級為[[中華民國海軍|中華民國海軍]][[上校|上校]]，但是佔[[少將|少將]]缺。在擔任[[國防部海軍司令部|海軍總司令部]]武器系統獲得管理室執行長期間，為他人所殺害，1993年12月10日被漁民發現浮屍於[[臺灣省|臺灣省]][[宜蘭縣|宜蘭縣]][[東澳車站|東澳車站]]附近烏岩角外約四至五百公尺處外海。

==案情==
尹清楓生前因為執行二代艦採購業務（1991年8月就經簽訂採購合約），包含[[拉法葉艦|拉法葉艦]]在內的四件[[拉法葉軍購案|艦艇軍購案]]總預算達新台幣1152億之譜，臺灣社會普遍相信尹清楓命案與[[軍購案|軍購案]]當中的龐大利益有關，一開始時[[國防部海軍司令部|海軍總司令部]][[軍法處|軍法處]]僅簡單判定死者為[[自殺|自殺]]，而沒有大力調查此案，但隨著家屬抗議以及[[媒體|媒體]]注意後，因死亡過程充滿各種疑問，以及在死亡後陸續牽連出許多當時中華民國政府的採購問題與弊端。[[汪傳浦|汪傳浦]]說，他根本不認識尹清楓。尹清楓1993年5月才調到海軍武獲室，遇害時間是1993年12月，但早在1991年8月底，台、法已簽訂拉法葉艦採購案合約，尹清楓並未參與，只是一個軍方的[[替死鬼|替死鬼]]而已。

==調查過程==
法國前外交部部長{{en-link|杜马斯|Roland Dumas}}已因收受-{佣}-金下台，中華民國海軍前[[中華民國海軍司令|海軍總司令]][[葉昌桐|葉昌桐]]、前艦管室主任[[雷學明|雷學明]]和[[姚能君|姚能君]]等高階將領，遭到彈劾。監察委員[[康寧祥|康寧祥]]表示，前行政院長[[郝柏村|郝柏村]]在購艦決策轉向，將原定的[[蔚山艦|蔚山艦]]變更為拉法葉艦過程中，違反多項程序規定，對此郝柏村發表一篇「我心無愧，願受公評」的聲明，郝柏村在聲明中強調，「面對昔日長官否定知情之言，我深表遺憾。對監察院『對統帥權顯欠尊重』的評語，我心坦然，唯願以我60年公職生涯，始於從軍報國，止於依憲總辭，無悔無愧之心情，接受國人公評」。總之，在此案調查中所涉層級之高，時任總統[[陳水扁|陳水扁]]甚至有「動搖國本」的說法，<ref>[http://tw.news.yahoo.com/軍方終於低頭-不代表真相大白-20110201-103253-853.html 中國時報－軍中黑暗]</ref>可說是臺灣在1990年代最大的懸案之一，此命案迄今仍未偵破。

== 影響 ==
甚至，在台灣社會中「尹清楓」一度成為離奇死亡的懸案的代名詞，如前省議員[[宋艾克|宋艾克]]在脫黨競選立委時聲明不甘做「[[新黨|新黨]]的尹清楓」，[[聯勤|聯勤]]工程弊案中的[[李錚|李錚]]也強調不做「第二個尹清楓」，上任後不斷處理爭議案件的台北市長[[柯文哲|柯文哲]]也在廣播節目中表示避免「被[[陳文成|陳文成]]掉或尹清楓掉」，甚至在金融風暴中跳樓自殺的銀行職員、捲入弊案而投河自盡的警員，乃至於各式各樣離奇死亡者，幾乎全被冠上是「尹清楓」。

== 家庭 ==
尹清楓的家人也有不少人有參政經驗。尹清楓的妻子[[李美葵|李美葵]]曾經於1994年由[[泛藍陣營|泛藍陣營]][[新黨|新黨]]提名參選[[台北市議員|台北市議員]]，尹清楓的妹妹楊尹星雯也曾經於2001年於[[高雄市_(1945年-2010年)|高雄市]]以無黨籍身份參選第五屆立法委員，但都未能當選。
