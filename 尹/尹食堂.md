《'''尹食堂'''》（{{kor|k='''윤식당'''}}，{{lang-en|'''Youn's Kitchen'''}}）為[[韓國|韓國]][[tvN|tvN]]的[[真人實境秀|實境真人-{zh-tw:秀;zh-cn:秀;zh-hk:騷}-]]節目，由《[[花樣爺爺|花樣爺爺]]》的[[羅䁐錫|羅䁐錫]]PD製作，[[尹汝貞|尹汝貞]]、[[李瑞鎮|李瑞鎮]]、[[鄭有美|鄭有美]]及[[申久|申久]]在'''[[印尼|印尼]]'''的'''[[峇里島|峇里島]]'''附近的'''德拉娜安島'''({{lang-en|'''Gili Trawangan Island'''}}) 共同營運一家韓餐廳，節目於2017年3月24日起逢星期五晚間9點20分 ([[韓國標準時|KST]]) 播出。<br />

《[[尹食堂|尹食堂]]》於2017年4月28日(第6期)創下最高平均收視率{{color|red|'''14.141%'''}}(AGB全國)及{{color|red|'''16.376%'''}}(AGB首都圈)的記錄，打破《[[一日三餐|一日三餐]]漁村篇第一季》於2015年2月20日(第5期)創下的最高平均收視率'''13.338%'''(AGB全國)的紀錄，成為韓國有線電視台綜藝節目歷年收視紀錄排名第1位。<br/>
<br/>
後續則有續作《[[尹食堂2|尹食堂2]]》。

== 節目概要 ==
2017年2月1日，[[tvN|tvN]]電視台回應了關於[[羅䁐錫|羅䁐錫]]PD全新旅遊綜藝節目的消息，節目的企劃目前尚處於初步構想階段，關於節目名稱、風格、日程等都尚未確定<ref>{{cite news|author=Laura|url=https://www.koreastardaily.com/tc/news/90513|title=李瑞鎮將與羅PD合作新節目 與尹汝貞、鄭有美搭檔形成新鮮組合|publisher=韓星網|date=2017-02-02}}{{zh-tw}}</ref>。2月21日，[[tvN|tvN]]電視台相關人員透露，演員[[尹汝貞|尹汝貞]]、[[李瑞鎮|李瑞鎮]]、[[鄭有美|鄭有美]]三人確定加盟[[羅䁐錫|羅䁐錫]]PD最新綜藝節目，並且將在近日出國錄製，三位演員的新鮮組合，讓粉絲們都相當期待<ref>{{cite news|author=Laura|url=https://www.koreastardaily.com/tc/news/91380|title=尹汝貞、李瑞鎮、鄭有美三人確定出演羅PD最新綜藝 近日出國錄製|publisher=韓星網|date=2017-02-22}}{{zh-tw}}</ref>。2月23日[[尹汝貞|尹汝貞]]、[[李瑞鎮|李瑞鎮]]、[[鄭有美|鄭有美]]與製作組一起出發前往印度尼西亞進行拍攝，[[申久|申久]]也隨後在結束了電視劇的拍攝，秘密前往與製作組會合<ref>{{cite news|author=邵天翔|url=http://m.allthatstar.com/tw/view.php?action=5205&postID=317147|title=花樣成員聚首《尹食堂》傳申久秘密出演|publisher=全星網|date=2017-03-03}}{{zh-tw}}</ref>。3月3日，[[tvN|tvN]]方面公佈節目定名《尹食堂》，製作組方面表示 :「或許很多人都有過離開複雜喧鬧的都市，到一個全新的地方開始不同的日常生活的想法，《尹食堂》是以很多人心中的夢想 :『脫離熾烈的生活現場，到一個悠閒寧靜的地方開設專屬自己的小店』為概念而企劃的」。節目在'''[[印尼|印尼]]'''的'''[[峇里島|峇里島]]'''附近的'''德拉娜安島''' ({{lang-en|'''Gili Trawangan Island'''}}) 拍攝，島嶼景色充滿了異國熱帶風情，堪稱是旅遊度假的聖地、潛水愛好者的天堂。[[尹汝貞|尹汝貞]]、[[李瑞鎮|李瑞鎮]]、[[鄭有美|鄭有美]]及[[申久|申久]]四人在這美麗的島嶼開設一家韓餐廳，節目記錄了四人共同營運餐廳的過程，以及和外國食客之間的交流<ref>{{cite news|author=邵天翔|url=http://www.allthatstar.com/tw/view.php?action=5211&postID=317113|title=羅PD新綜定名《尹食堂》 播出日期確定|publisher=全星網|date=2017-03-03}}{{zh-tw}}</ref><ref>{{cite news|author=Jessica|url=http://kpopn.niusnews.com/=K2szqe72|title=羅PD 新綜藝《尹食堂》將在3/24首播|publisher=Kpopn|date=2017-03-03}}{{zh-tw}}</ref>。3月7日演員和製作團隊完成拍攝日程經由[[仁川機場|仁川機場]]返抵韓國<ref>{{cite news|author=Laura|url=https://www.koreastardaily.com/tc/news/91976|title=《尹食堂》李瑞鎮、鄭有美、尹汝貞、申久完成首次拍攝 24日首播|publisher=韓星網|date=2017-03-08}}{{zh-tw}}</ref>。節目於2017年3月24日在韓國首播。

後續新聞媒體追蹤，食堂是借用當地真正有在營運的餐廳，且在節目拍攝中途早已拆除，同時也因而於第三集時移動餐廳的位置<ref>{{cite news|author=강예나|url=http://www.dispatch.co.kr/705547|title=“윤식당이 사라졌다”…발리에서 생긴, 철거|date=2017-03-28|publisher=Dispatch|language=ko}}</ref>。

== 尹食堂料理食譜與教學 ==
* 尹食堂特製烤肉醬（10人份）：洋蔥1顆，蘋果或梨子2/3顆，胡椒1茶匙（1t），醬油2碗，砂糖1匙，湯水2碗，蒜92克；混合後以榨汁果汁機打勻。
* 基本烤肉：鍋內倒入一些油，先放胡蘿蔔、洋蔥、蔥、洋菇、金針菇等蔬菜，後放入肉片一起混合炒熟後，再放入特製烤肉醬再炒一次。
* 烤肉飯：將基本烤肉放入盛滿白飯的盤子後，再放入些許洋白菜與香菜（可依自身喜好煎一顆蛋放於其上）。
* 烤肉粉條：粉條剪半後，先以冷水泡一小時。後續炒基本烤肉的過程中，再放入一些粉條一起炒即可。盛盤時再加入洋白菜與香菜。
* 烤肉漢堡：法國長條麵包以麵包刀對半切，放入已經一起炒過的基本烤肉與起士。最後於漢堡上放入生菜與香菜，並淋上一點松露油。
*雞蛋泡麵：泡麵中加入蛋。
*水餃泡麵：泡麵中加入水餃。
*雞蛋水餃泡麵：泡麵中加入蛋與水餃。
*油炸餃子：搭配醋加醬油的調料。
*炸雞Crispy Chicken：雞肉塊切塊，當地炸雞粉加水成為糊狀，將黏住糊狀的雞塊再裹上炸雞粉(拌到糊膏狀)進油鍋炸，9分鐘後呈現金黃即可食用，搭配辣味番茄醬的調料。
*蔥餅


== 尹食堂員工餐食譜與教學 ==
* 拌蛤蚌醬：辣椒切塊剁碎後與香油混合，再加入蛤蚌醬拌勻即可。
* 韓式泡菜湯：切適當大小的肉至鍋中，加入香油後翻炒，接著鍋內放入泡菜，最後倒入水熬煮一會兒。
* 李常務牌包飯醬：大醬、香油、辣椒醬等材料混合均勻。
*辣炒牛肉：先將牛肉單獨炒熟，接著將泡菜與辣椒等與前述的牛肉混合後再次炒熟<ref group =註>由於當地宗教信仰之故，而人民不食用豬肉，導致超市亦不販售豬肉，因此成員將原本韓式辣炒豬肉換成牛肉</ref>。
*蕎麥冷麵：先將蕎麥麵煮熟，煮熟後立刻投入冷水中進行冰鎮，冰鎮後裝盤，並於盤中放入冰塊；同時將蘿蔔打成蘿蔔泥，配著蕎麥冷面一起食用。
*喜麵
