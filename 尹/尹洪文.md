'''尹洪文'''（{{bd|1957年|11月||}}），[[山东|山东]][[临朐|临朐]]人，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
尹洪文曾任[[中国人民解放军南京军区|中国人民解放军南京军区]]政治部组织部部长、[[中国人民解放军陆军第一集团军|中国人民解放军陆军第一集团军]]步兵第一师政委、[[中国人民解放军总政治部|中国人民解放军总政治部]][[中国人民解放军三一一基地|三一一基地]]政委。2014年8月至2015年3月之间，调任中国人民解放军总政治部秘书长。2016年1月中央军委机关调整组建后，尹洪文少将担任[[中央军事委员会政治工作部办公厅|中央军委政治工作部办公厅]]主任。2017年1月，转任[[中央军委政治工作部|中央军委政治工作部]]主任助理<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1752739|title = 尹洪文少将已担任中央军委政治工作部主任助理 |author =  |date = 2017-08-04 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web|url =http://news.hexun.com/2015-04-12/174881368.html|title =尹洪文少将出任总政治部秘书长 李凤山不再担任|author =|date =2015-04-12|publisher =和讯|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20151218190929/http://news.hexun.com/2015-04-12/174881368.html|archivedate =2015-12-18}}</ref>。2017年8月，担任[[东部战区|东部战区]]副政委兼政治工作部主任<ref>{{cite web |title=东部战区领导尹洪文职务明确：副政委兼政治工作部主任 |url=https://www.thepaper.cn/newsDetail_forward_2348729 |website=澎湃新闻 |accessdate=2018-08-15}}</ref><ref>{{cite web|title=尹洪文少将晋升副战区级 曾任政治工作部主任助理|url=http://news.sina.com.cn/o/2017-12-10/doc-ifypnqvn2366105.shtml|website=新浪|accessdate=2017-12-10}}</ref>。
