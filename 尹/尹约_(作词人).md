'''尹约'''，[[中国大陆|中国大陆]]作词人、制作人、[[高晓松|高晓松]]主持的知名脱口秀节目《晓说》的主编<ref>{{cite web |title=她是高晓松《晓说》的主编，也是华语音乐的女性力量 |url=https://cj.sina.com.cn/articles/view/1765373140/693974d401900efh5 |website=新浪网 |publisher=福布斯中文网 |accessdate=2019年2月18日}}</ref>。2017年11月6日，尹约首次担任主要制作人为[[周深|周深]]制作的首张个人专辑《深的深》发行。2018年3月26日，凭借这张专辑的主打歌《蓝色降落伞》，尹约荣获了第25届[[东方风云榜|东方风云榜]]最佳作词。代表作《默》、《蓝色降落伞》、《大魚》、《风清扬》、《玫瑰与小鹿》等。


尹约先后为[[那英|那英]]、[[周深|周深]]、[[谭维维|谭维维]]、[[梁博|梁博]]、[[华晨宇|华晨宇]]、[[老狼|老狼]]、[[王菲|王菲]]、[[马云|马云]]、[[王凯|王凯]]、[[范冰冰|范冰冰]]、[[高云翔|高云翔]]、[[萧敬腾|萧敬腾]]、[[萨顶顶|萨顶顶]]、[[吉克隽逸|吉克隽逸]]、[[黄龄|黄龄]]、[[李治廷|李治廷]]、[[曲婉婷|曲婉婷]]、[[黄晓明|黄晓明]]、[[张信哲|张信哲]]、[[金志文|金志文]]、[[孙楠|孙楠]]等知名歌手和演员演唱的歌曲写过歌词<ref>{{cite web |title=她是高晓松的搭档，为王菲周深填词，一出现就惊诧观众太漂亮了 |url=https://new.qq.com/omn/20181007/20181007A0W5PN.html?pc |publisher=腾讯网 |accessdate=2018年10月7日}}</ref>。魔镜歌词网专门收录了尹约所作的歌词，截至2019年4月18日共32首<ref>{{cite web |title=【 尹约 】 【 歌词 】 共有 32笔相关歌词 |url=https://mojim.com/%E5%B0%B9%E7%BA%A6.html?g4 |website=魔镜歌词网 |accessdate=截至2019年4月18日}}</ref>。中国曲谱网也收录了尹约作词的部分歌曲的曲谱，并附有曲谱和歌词<ref>{{cite web |title=尹约作词的曲谱列表 |url=http://www.qupu123.com/author/128517 |website=中国曲谱网}}</ref>。


2019年03月26日，太合音乐集团正式宣布作词人尹约加盟旗下大石音乐版权。太合音乐集团总裁暨首席执行官徐毅表示：“尹约是近年业界炙手可热的创作新贵，作品成绩斐然，感谢尹约对于大石音乐版权的信任。”<ref>{{cite web |title=尹约加盟大石音乐版权助力太合音乐打造华语音乐流行词作 |url=http://ent.people.com.cn/n1/2019/0326/c1012-30996664.html |publisher=人民网娱乐频道 |accessdate=2019年03月26日}}</ref>

== 制作的专辑 ==

2017年11月6日，尹约首次担任制作人、高晓松监制、并与钱雷共同创作，用了三年时间精心打磨的周深演唱的首张个人专辑《深的深》发行<ref>{{cite web |title=高晓松领衔“高家班”集体亮相 为周深新专辑站台 |url=http://www.xinhuanet.com/ent/2017-11/07/c_1121918076.htm |publisher=新华网 |accessdate=2017年11月7日}}</ref>。专辑共收录了十首歌，其中《大鱼》、《玫瑰与小鹿》和《蓝色降落伞》分别获得了包括第24届东方风云榜（十大金曲《大鱼》）和2018全球华人歌曲排行榜（十大金曲《蓝色降落伞》）在内的多项大奖<ref>{{cite web |title=华语乐坛最高水准的舞台 2018华人歌曲音乐盛典创造经典 |url=http://ent.huanqiu.com/yuleyaowen/2018-08/12876745.html?agt=312 |publisher=环球网娱乐 |accessdate=2018年8月30日}}</ref>。尹约更是于2018年3月26日，凭借这张专辑的主打歌《蓝色降落伞》荣获第25届[[东方风云榜|东方风云榜]]最佳作词<ref>{{cite web |title=高晓松微博祝贺周深尹约获奖，美女制作人终亮相，这个细节被吐槽 |url=https://kuaibao.qq.com/s/20180327A1E36D00?refer=spider |website=天天快报 |accessdate=2018年3月27日}}</ref>。
