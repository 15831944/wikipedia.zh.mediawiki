'''荻伏站'''（{{jpn|j=荻伏駅|hg=おぎふしえき|rm=Ogifushi eki|f=y}}）位於[[北海道|北海道]][[浦河郡|浦河郡]][[浦河町|浦河町]]荻伏町元浦河，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[日高本線|日高本線]]上的站。電報略號是'''{{lang|ja|オキ}}'''。

名稱源自[[阿伊努語|阿伊努語]]的「o-ni-us-i」，意思為下游處有樹木生長的河流<ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/2130P.pdf|title=アイヌ語地名リスト|accessdate=2017-08-16|author=北海道庁|date=|publisher=北海道庁}}</ref>。

== 車站構造 ==
[[側式月台|側式月台]]1面1線的地面站。沒有站房，只有一座由[[守車|守車]]改造而來的候車室。

== 車站周邊 ==
離荻伏市區有一段距離。
* [[北海道道348號野深荻伏停車場線|北海道道348號野深荻伏停車場線]]、[[北海道道384號荻伏停車場線|北海道道384號荻伏停車場線]]
* 浦河町公所荻伏支所
* 浦河警察署荻伏派出所
* 荻伏郵局
* 日高東農業協同組合（JA日高東）荻伏事業所
* 日高中央漁業協同組合荻伏支所
* JR北海道巴士「荻伏站前」站牌
* 浦河町立荻伏中學

== 歷史 ==
* 1935年（[[昭和|昭和]]10年）10月24日 - 設立。
* 1977年（昭和52年）2月1日 - 廢止貨物和行李處理的業務。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為北海道旅客鐵道（JR北海道）轄下的車站。
* 2011年（[[平成|平成]]23年）5月 - 終止簡易委託。
