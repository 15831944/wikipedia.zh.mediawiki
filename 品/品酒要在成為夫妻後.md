== 故事簡介 ==
擔任公司主任的水澤千里能力出眾而不多話，受到身邊人們的尊敬。千里回到家後喜歡喝丈夫調製的雞尾酒，喝醉的她會展現出與工作時截然不同的模樣。

== 登場人物 ==
=== 主要人物 ===
; {{nihongo|水澤千里|水沢 千里（みずさわ ちさと）|聲：[[喜多村英梨|喜多村英梨]]<ref name="character">{{cite web|url=http://www.osakefufu-anime.jp/character/index.html|script-title=ja:キャラクタ|work=電視動畫「品酒要在成為夫妻後」官方網站|accessdate=2017-10-04|language=ja}}</ref>}}
: 生日：11月22日／年齡：28歲／身高：159cm／體重：52kg／血型：A型／出身：[[長野縣|長野縣]]
: 本作主人公。愛稱{{nihongo|「千醬」|「ちーちゃん」}}（僅限壮良）。
: 留學歸來後在一家公司擔任主任。不善於表達情感，但對待工作十分沉默嚴肅，深受信賴。喜歡喝酒，有著容易在短時間內醉酒的體質。在喝醉後常常會不經意間解開馬尾辮、摘下眼鏡並感嘆{{nihongo|「幸福」|しふく}}。知道自己酒後的情況所以一直忍著不在下屬或同事面前喝酒。
: 非常喜歡壮良。家務全由丈夫壯良負責。喜歡收集「檸檬男」形象的物品。
; {{nihongo|水澤壮良|水沢 壮良（みずさわ そら）|聲：[[市來光弘|市來光弘]]{{r|character}}}}
: 生日：3月10日／身高：172cm／體重：62kg／血型：O型／出身：[[東京都|東京都]]
: 千里的丈夫，暱稱{{nihongo|「Sora」|ソラ}}。家具設計師。性格溫柔。以抱石和飛鏢為愛好。
: 在母子家庭中長大。因爲母親愛喝酒的緣故，在20歲時走上調酒師的道路，曾在「No Rest」酒吧工作，對鷄尾酒調配和開胃菜配方非常熟悉，現在偶爾也會在閑時到「No Rest」酒吧做幫手。
: 對千里十分溫柔，負責家裡所有家務，也經常在千里回家時調酒給千里喝，是位十分理想的丈夫。

=== 同事 ===
; {{nihongo|白石結|白石 結（しらいし ゆい）|聲：[[朝井彩加|朝井彩加]]{{r|character}}}}
: 生日：2月20日／身高：160cm／體重：48kg／血型：B型／出身：[[群馬縣|群馬縣]]
: 千里的同事。有兩個姐妹。工作非常投入，性格嚴肅，總會盡量保持安靜，討厭嘈雜的地方。
: 將千里視作競爭對手但也非常喜歡和憧憬千里。喝醉後會表現出少女的一面。與丈夫正樹約定在喝醉後將她帶回家。
; {{nihongo|櫻井小春|桜井 小春（さくらい こはる）|聲：仲田亞里沙{{r|character}}}}
: 生日：7月2日／身高：156cm／體重：51kg／血型：A型／出身：[[北海道|北海道]]
: 千里的属下。性格活潑，處事認真，经常得到千里的帮助。決心臺詞為「幸福」。即便已經工作，但仍然住在家裏，尚未完全獨立。
: 喜歡到處探訪酒館，但酒品不太好，又容易忍不住喝醉。相親時因爲喝得爛醉如泥所以被攪黃。學生時代的綽號是「無櫻無春」。在和青年茅崎見面後逐漸縮短與他的距離。
; {{nihongo|楓|カエデ}}
: 千里的前輩職員。被小春稱呼為{{nihongo|「姐姐」|ねーさん}}。以有偉大理想爲由而不願結婚。喜歡喝香檳。
; {{nihongo|末永|末永（すえなが）}}
: 千里的同事，身材矮小。平日沉默寡言，但打游戲時會變得毒舌，常因在楓面前口無遮攔受到楓的懲罰。不能喝酒。厭倦戀愛與婚姻。
; {{nihongo|茅崎|茅ヶ崎（ちがさき）}}
: 千里就職公司的新職員。常和小春在一起，了解到她意想不到的一面而對其有好感。有兩個姐妹。曾經見過結婚之前的千里和壮良以及學生時代的小春，但本人並不記得。
: 其家族有經營一家叫{{nihongo|「茅崎」|チガサキ}}的小酒館，工作回家後會進酒館裡幫忙，對鷄尾酒有一定瞭解。

; {{nihongo|科長|課長（かちょう）|聲：[[石塚運昇|石塚運昇]]}}
: 千里就職公司所在部門的經理。一位灰白頭髮的老年男性。非常瞭解鷄尾酒。有一個名叫{{nihongo|「佳苗」|カナエ}}的孫女。

=== 親屬 ===
; {{nihongo|白石正樹|白石 正樹（しらいし まさき）|聲：[[赤羽根健治|赤羽根健治]]}}
: 白石結的丈夫。公司職員。戴眼鏡，表情嚴肅。與白石結同年，經常和白石結吵架，但實際上兩人相處得很好。在一次偶然的幫助中與壮良熟識。
; {{nihongo|山賀|山賀（やまが）}}
: 千里的父親。退休人員。一名酒鬼，常常独自一人饮酒，但平日里沉默寡言，容易害羞。據認為並不贊同千里與壮良的婚姻。口癖為「至福」。
; 千里的母親
: 法日混血。性格開朗。與女兒一樣喜歡「檸檬男」。
; 壮良的母親
: 喜歡以波波頭為髮型的中年婦女。行爲古怪，非常奔放。

=== 好友 ===
; {{nihongo|瓦妮莎|ヴァネッサ|Vanessa}}
: 法國人。千里在大學時代的好友。會説一些帶關西腔的日語。

; {{nihongo|輕飄美<!--暫譯名-->|ふわ美（ふわ び）}}
: 小春在大學時代的好友，同時也是她在聯誼會上的同伴。

; {{nihongo|神光子<!--暫譯名-->|オカル子（オカル こ）}}
: 小春的好友，本名{{nihongo|'''薰子'''|薫子（かおるこ）}}。黑髮女性，職業為命理師。

=== 其他 ===
; {{nihongo|Master|マスター|聲：{{tsl|ja|坂巻学|坂卷學}}}}
: 「No Rest」酒吧的老闆，也是壮良的師傅。曾告誡經常光顧酒吧的小春不要自暴自棄，要展現出年长、经验丰富的一面。

; {{nihongo|檸檬男|レモ男|聲：{{tsl|ja|山口翔平}}}}
: 某啤酒廠製造的吉祥物。有許多相關的衍生物品（如玩偶、茶杯、日曆等）。雖然已經不再具備其產品公關的作用，但直到現在仍然在年輕人之間有著相當高的崇拜熱度。
