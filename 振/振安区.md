'''振安区'''是[[辽宁省|辽宁省]][[丹东市|丹东市]]下辖的一个市辖区。面积659.73平方千米，人口17.38万。邮政编码118001。区人民政府驻珍珠街250号。

== 行政区划 ==
下辖4个[[街道办事处|街道办事处]]、5个[[镇_(中华人民共和国)|镇]]<ref>{{cite web |url=http://dm.dandong.gov.cn/qhsc.htm |title=存档副本 |accessdate=2009-11-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20091127085859/http://dm.dandong.gov.cn/qhsc.htm |archivedate=2009-11-27 }} 2009年丹东市行政区划统计表</ref>：
* 街道办事处：鸭绿江街道、珍珠街道、太平湾街道、金矿街道。
* 镇：五龙背镇、楼房镇、汤山城镇、同兴镇、九连城镇。
