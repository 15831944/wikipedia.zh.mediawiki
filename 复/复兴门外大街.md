'''复兴门外大街'''是位于中国[[北京市|北京市]][[西城区|西城区]]西部的一条大街，在[[长安街|长安街]]西延长线上。因位于[[北京内城|北京内城]][[复兴门_(北京)|复兴门]]外而得名<ref name=bj>《北京地名典》，北京：[[中国文联出版社|中国文联出版社]]，2008年，第207页</ref>。

==历史==
复兴门外大街东起[[复兴门内大街|复兴门内大街]]西端的[[复兴门桥|复兴门桥]]，西到[[三里河路|三里河路]]南端的[[木樨地桥|木樨地桥]]。1940年，[[北京内城|北京内城]]西城墙南端增辟豁口，称长安门，并自此向西修路。1946年长安门改称[[复兴门_(北京)|复兴门]]。1949年，这条东西向的道路也因此而定名为复兴门外大街。此名沿用至今。<ref name=bj/>

复兴门外大街是[[长安街|长安街]]的延长线，东接[[复兴门内大街|复兴门内大街]]，西接[[复兴路_(北京)|复兴路]]，沿途和[[南礼士路|南礼士路]]、[[西便门外大街|西便门外大街]]、[[二七剧场路|二七剧场路]]、[[真武庙路|真武庙路]]、[[三里河东路|三里河东路]]、[[白云路|白云路]]等道路相交叉。

==建筑==
复兴门外大街上有许多公共建筑、居民小区。由于此处位于北京旧城以外，故大街两侧均为新建建筑，没有古建筑。

==雕塑==
*《'''南极石'''》：位于复兴门外大街北侧，[[国家海洋局|国家海洋局]]楼前广场。1998年12月，由首都城市雕塑艺术委员会、中共北京市委宣传部、全国城市雕塑建设指导委员会共同组织的“新世纪的希望”城市雕塑设计方案竞赛开始向全国征稿，征得稿件750多件，邀请入选的120多件做成立体方案，并于1999年4月2日起在[[北京国际会议中心|北京国际会议中心]]举办“新世纪的希望”城市雕塑方案展览，由建筑、规划、园林、雕塑专家及长安街沿线单位讨论。经评审及优选，推荐其中8件作品在长安街设置，1999年9月中旬全部建成，以庆祝中华人民共和国成立五十周年。其中之一是《南极石》，这是中国第三次[[南极|南极]]考察队1987年采自[[中国南极长城站|中国南极长城站]]的南极石。象征中国科技工作者的奋斗成果、任重道远的征程和开拓前进的恒心<ref>{{Cite web|url = http://www.bjghw.gov.cn/web/static/articles/catalog_66/article_558/558.html|title = 六、城市雕塑|author = |date = 2002-07-05|publisher = 北京市规划和国土资源管理委员会（城乡规划）|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180217202932/http://www.bjghw.gov.cn/web/static/articles/catalog_66/article_558/558.html|archive-date = 2018-02-17|dead-url = yes}}</ref>。
*《'''中化之帆'''》：位于复兴门外大街南侧，[[中化大厦|中化大厦]]门前，1995年建成，由[[隋建国|隋建国]]创作。
