'''复兴门内大街'''是一条位于中国[[北京市|北京市]][[西城区|西城区]]中部的大街。该大街东起[[西单北大街|西单北大街]]南口，西端在[[复兴门桥|复兴门桥]]同[[复兴门外大街|复兴门外大街]]衔接。该大街因位于[[北京内城|北京内城]][[复兴门_(北京)|复兴门]]内而得名。<ref name=bjdm>王彬、徐秀珊主编，北京地名典，北京：中国文联出版社，2008年，第214页</ref>

==历史==
民国二十八年（1939年），日本占领时期，在邱祖胡同、卧佛寺街西口附近开辟城门，初名长安门，未建门楼，仅为豁口。[[抗日战争|抗日战争]]胜利后，长安门更名为复兴门，寓意国家复兴。<ref name=bjdm/>

1956年至1957年，报子胡同、旧刑部街、邱祖胡同、卧佛寺街被拆除，辟建大街。该大街因在复兴门内，故名复兴门内大街。<ref name=bjdm/>

如今复兴门内大街两侧原有的四合院大部分均已拆毁，改建为大楼。仅存[[北京三味书屋|三味书屋]]、[[马连良故居|马连良故居]]等，位于大街南侧。

原门牌号复兴门内大街48号（原西单报子街76号）是[[中华工程师学会|中华工程师学会]]旧址。民国五年（1916年），中华工程师学会总会由[[武汉|武汉]]迁至[[北京|北京]]。民国七年（1918年），中华工程师学会总会迁入[[詹天佑|詹天佑]]捐赠的西单报子街76号会址。民国20年（1931年），中华工程师学会与中国留美工程界人士成立的[[中国工程学会|中国工程学会]]合并，在[[南京|南京]]举行的联合年会上宣布“[[中国工程师学会|中国工程师学会]]”成立，以民国元年作为本学会创始年，设总会於[[南京|南京]]。民国八年（1919年）4月，[[詹天佑|詹天佑]]病逝。民国十一年（1922年）10月29日，在报子街76号举行了詹天佑铜像揭幕典礼。[[中华人民共和国|中华人民共和国]]时期，[[中华工程师学会|中华工程师学会]]旧址仍保存有部分建筑，成为民居。<ref name=bjdm/>

== 雕塑 ==
复兴门内大街两侧有多座雕塑：
* 《'''树'''》：位于[[复兴门桥|复兴门桥]]东南方向绿地内，复兴门内大街南侧。是1999年“新世纪的希望”城市雕塑设计方案竞赛选出的在长安街设置的8件作品之一。见[[复兴门桥|复兴门桥]]条目。
* 《'''[[马踏飞燕|马踏飞燕]]'''》：位于[[复兴门桥|复兴门桥]]东北方向绿地内，复兴门内大街北侧，[[北京百盛购物中心|北京百盛购物中心]]（原为[[中国工艺美术馆|中国工艺美术馆]]）正南。见[[复兴门桥|复兴门桥]]条目。
*《'''龙'''》：位于[[北京国际金融大厦|北京国际金融大厦]]前西侧绿地中。1998年12月，由首都城市雕塑艺术委员会、中共北京市委宣传部、全国城市雕塑建 设指导委员会共同组织的“新世纪的希望”城市雕塑设计方案竞赛开始向全国征稿，征得稿件750多件，邀请入选的120多件做成立体方案，并于1999年4月2日起在[[北京国际会议中心|北京国际会议中心]]举办“新世纪的希望”城市雕塑方案展览，由建筑、规划、园林、雕塑专家及长安街沿线单位讨论。经评审及优选，推荐其中8件作品在长安街设置，1999年9月中旬全部建成，以庆祝[[中华人民共和国|中华人民共和国]]成立五十周年。其中之一是《龙》。高5.8米。由北京敦煌艺术有限公司雕塑家[[耿铁群|耿铁群]]设计，北京市金鼎雕塑艺术有限公司制作。借鉴了[[中国书法|中国书法]][[草书|草书]]“[[龙|龙]]”字，使平面艺术立体化<ref>{{cite web|url = http://www.bjghw.gov.cn/web/static/articles/catalog_66/article_558/558.html|title = 六、城市雕塑|author = |date = 2002-07-05|publisher = 北京市规划和国土资源管理委员会（城乡规划）|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180217202932/http://www.bjghw.gov.cn/web/static/articles/catalog_66/article_558/558.html|archive-date = 2018-02-17|dead-url = yes}}</ref><ref>{{Cite web |url = http://www.bjghw.gov.cn/web/static/articles/catalog_66/article_553/553.html |title = 四、绿化美化（上） |author =  |date = 2002-07-05 |publisher = 北京市规划和国土资源管理委员会（城乡规划） |language = zh |accessdate =  |quote =  |archive-url = https://web.archive.org/web/20180217202917/http://www.bjghw.gov.cn/web/static/articles/catalog_66/article_553/553.html |archive-date = 2018-02-17 |dead-url = yes }}</ref>。
