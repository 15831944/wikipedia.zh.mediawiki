'''複分解反应'''又稱'''雙置換反應'''，是由两种[[化合物|化合物]]，通過互相交换成分並生成两种新化合物的[[化学反应|反应]]，模式为AB+CD→AD+CB。必发生在[[水溶液|水溶液]]中，它是基本类型的[[化学反应|化学反应]]之一。複分解都不是[[氧化还原反应|氧化还原反应]]（有些反应是複分解产物再发生氧化还原，而不是複分解的结果）。

== 无机複分解反应 ==
溶液中複分解发生条件：可从设想的生成物而定，如生成物有[[气体|气体]]、[[沉淀|沉淀]]、[[弱电解质|弱电解质]]、配（络）合物生成时，则複分解反应一般能进行。即有产物脱离体系时，复分解反应一般能进行。

=== 離子反應 ===

例1：[[HCl|HCl]](''aq'') + [[KOH|KOH]](''aq'') → [[KCl|KCl]](''aq'') + [[H2O|H<sub>2</sub>O]](''ℓ'')

H<sup>+</sup>與OH<sup>−</sup>結合生成弱電解質水，所以反應發生

H<sup>+</sup> + OH<sup>−</sup> → H<sub>2</sub>O ([[H3O+|H<sub>3</sub>O<sup>+</sup>]] + OH<sup>−</sup> → 2 H<sub>2</sub>O)

K<sup>+</sup>與Cl<sup>−</sup>留在溶液中，所以說生成了KCl與H<sub>2</sub>O

例2：[[AgNO3|AgNO<sub>3</sub>]](''aq'') + [[HCl|HCl]](''aq'') → [[AgCl|AgCl]](''s'')↓ + [[HNO3|HNO<sub>3</sub>]](''aq'')

Ag<sup>+</sup>與Cl<sup>−</sup>結合生成的AgCl几乎无法溶于水中，以固体形态存在，所以反應發生

<chem>Ag+ + Cl- -> AgCl v</chem>

=== 氣體生成 ===

例1：<chem>{H3PO4({\it aq})} + 3NaCl({\it s}) ->[\Delta] 3HCl({\it g}) \uparrow + Na3PO4({\it aq})</chem> 

加熱，溫度升高，沸點低的HCl氣體大部分脫離反應體系，反應發生。

例2：<chem>{CH3COONa({\it s})} + NaOH({\it s}) ->[\Delta] {CH4({\it g})} + Na2CO3({\it s})</chem>

同上

例3：[[CaCO3|CaCO<sub>3</sub>]](''s'') + 2 [[HCl|HCl]](''aq'') → [[CaCl2|CaCl<sub>2</sub>]](''aq'') + [[H2CO3|H<sub>2</sub>CO<sub>3</sub>]](''aq'')

首先通過離子反應生成弱電解質H<sub>2</sub>CO<sub>3</sub>（強酸制弱酸，但有例外），由于H<sub>2</sub>CO<sub>3</sub>极不稳定，分解產生CO<sub>2</sub>氣體：

[[H2CO3|H<sub>2</sub>CO<sub>3</sub>]](''aq'') → [[H2O|H<sub>2</sub>O]](''ℓ'') + [[CO2|CO<sub>2</sub>]](''g'')↑

== 烯烃複分解反应 ==
{{main|烯烃複分解反应}}
透過研究[[金屬卡賓|金屬卡賓]]在[[有機化合物|有機化合物]]生成中所扮演的角式而優化[[有機化合物|有機化合物]]的生產過程。由於生產過程中讓[[金屬|金屬]]擔當[[催化劑|催化劑]]的角色，使這些有機化合物無需要再透過利用[[卤化|卤化]]過程來達至產品的生成，從而使有機化合物的生產過程可以更[[環保|環保]]。[[伊夫·肖萬|伊夫·肖萬]]、[[羅伯特·格拉布|羅伯特·格拉布]]及[[理查德·施罗克|理查德·施罗克]]憑這個研究獲得2005年度[[諾貝爾化學獎|諾貝爾化學獎]]。
