'''复兴号CR300AF型电力动车组'''，是[[中国高速铁路|中国高速铁路]][[复兴号电力动车组|复兴号]]的车型之一，是[[中国铁路总公司|中国铁路总公司]]向[[中车青岛四方机车车辆|中车青岛四方机车车辆]]订购的[[中国高速铁路|高速铁路]][[电力动车组|动车组]]。中国铁路总公司将所有由[[中国中车|中国中车]]自主研发，完全具有自主知识产权，营运于中国铁路客运专线及高速铁路的电力动车组均命名为「'''[[复兴号电力动车组|复兴号]]'''」<ref>{{cite web|url=http://www.southcn.com/nfdaily/nis-soft/wwwroot/site1/nfrb/html/2017-06/27/content_7648077.htm|title=“复兴号”来了 中国标准动车组时代来了|accessdate=2017-06-29|date=2017-06-27|language=zh-cn|quote=“‘复兴号’是基于中国自主研发的高速动车组设计制造平台，是中国具有完全知识产权的新一代高速列车。”中国铁路总公司总经理陆东福说。|author1=新华社|website=南方新闻网|location=广州}}</ref>。

== 概要 ==
CR300系列复兴号动车组（250km/h级）是自CR400系列（350km/h级）、CR200J系列（160km/h级）之后，复兴号动车组的第三个级别。

2018年初，CR300系列复兴号动车组开始设计评审<ref>{{cite web |author1=徽高铁族 |title=关于西成高铁列车统一更换380系及后续CR300列车的分析和展望 |url=http://sh.qihoo.com/pc/92fd16144282f459d?sign=360_e39369d1}}</ref>，2018年9月末，CR300AF-0001样车制造完成下线。10月17日，0001样车无火回送至[[中国国家铁道试验中心（东郊分院）|北京环铁试验中心]]准备进行测试。12月7日，0001号车回送至[[成都站|成都站]]。12月13日，0001号车开始在正处于联调联试期间的[[成贵客运专线|成贵客运专线]][[乐山站|乐山]]至[[兴文站|兴文]]段进行测试运行。

==車輛構造==
===外观===
CR300AF复兴号动车组列车由[[中车青岛四方机车车辆股份有限公司|中车青岛四方机车车辆股份有限公司]]研发生产，采用4动4拖的8节编组。CR300AF暂时采用了类似CR400AF型动车组的红色色带外观配色，并采用了全新的头型，车鼻长度较CR400AF有所缩短。由于不设置商务座席，因此CR400AF驾驶室正面的原有的两个副窗被取消，侧窗的位置也相应提高，且由狭长的梯形窗改为菱形窗。在后续制造的车辆上，原有银灰色底色被改为天蓝色。

===动力配置===
CR300AF采用8辆编组，4动4拖的统一动力配置形式，由2个基本动力单元组成，其中02车、04车、05车、07车为动车，01、03、06、08车为拖车。

===主要设备===
====集电装置及供电系统====
CR300AF型在3号和6号车厢设置[[受电弓|受电弓]]，在正常运行时使用一个受电弓，重联运行则使用两个受电弓。

== 故障 ==
*2018年10月14日，刚出厂的CR300AF-0001号车经由[[京沪铁路|京沪铁路]]运行无火回送至[[北京环铁|北京环铁]]过程中，6号车顶安装的高压电气舱保护板在至少济南境内向上掀起并继续运行近500公里至北京境内，导致北京铁路枢纽内[[北京南站|北京南站]]至[[北京站|北京站]]区间接触网跳闸，造成[[京沪铁路|京沪铁路]]上行方向列车大面积晚点。
