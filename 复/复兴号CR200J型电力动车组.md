'''CR200J型电力动车组'''是[[中国铁路|中国铁路]]的一款时速160公里动力集中型电力动车组，为[[复兴号电力动车组|复兴号电力动车组]]系列当中的首款普速动车组，也是中国铁路时隔多年后再度采用的[[動力集中式列車|动力集中]]编组形式的[[动车组|动车组]]。该型动车组由[[中国铁路总公司|中国铁路总公司]]牵头，并由[[中国中车|中国中车]]旗下六家公司联合设计生产。

==发展历史==
[[File:CR200J-3_002@BCR_(20170927152741).jpg|thumb]]測試中的CR200J-3002號动力车（2017年9月）]]
[[File:FXD3-J0013_near_Chongqingbei_Railway_Station.jpg|thumb]]北场]]
時速160公里CR200J型动力集中动车组是在[[中国铁路总公司|中国铁路总公司]]牽頭推進“[[复兴号电力动车组|復興號]]”品牌戰略的背景下，為滿足普速鐵路開行動車組的需要，秉承“先進、可靠、成熟、經濟、必須”的方針，借鉴既有中國鐵路[[动力分散式列车|動力分散]]動車組發展的成功經驗，利用既有機車、客車的技術平臺及運用檢修資源，在滿足標準化、系列化、安全性和可靠性的基礎上，由[[中國中車|中國中車]]旗下[[中车大连机车车辆有限公司|中车大连机车车辆有限公司]]、[[中车株洲电力机车|中车株洲电力机车有限公司]]、[[中车南京浦镇车辆有限公司|中车南京浦镇车辆有限公司]]和[[中车唐山机车车辆有限公司|中车唐山机车车辆有限公司]]与相關企業及高校聯合設計製造的新型普速動車組。

2015年8月28日，[[中国铁路总公司|中国铁路总公司]]运输局以会议纪要形式下发通知，强调：“为进一步发挥既有线旅客列车开行效益，提高运输效率，铁路总公司组织相关单位进行技术研讨，按照机辆一体化的思路，研发时速160公里动车组。”《通知》明确了该型动车组的动力方式，编组构成，运用范围和检修整备方式，要求各路局结合客运市场开发需要，研究测算“时速160公里动力集中型动车组”组数（按照18辆和9辆分别测算）和每组编组内容，提出开行区段和对数的具体建议。

2017年4月27日，由中车唐山生产的首列样车（4001拖车组）正式下线。

2017年6月尾至7月间，由中车浦镇生产的4002拖车组由配属上海局集团的[[东风11型内燃机车|东风11型内燃机车]]以[[车辆运送#甲种运送/无火回送|甲种运送]]方式，送往[[北京环行铁路|北京环行铁路]]进行测试。

2017年8月17日，两列动车组在北京环行铁路进行首次动态测试。

2017年12月17日至2018年1月4日期间，部分动车组暂时存放于[[中国铁路成都局集团有限公司|成都局集团]]，先后在[[成渝铁路|成渝铁路]]、[[兰渝铁路|兰渝铁路]]及当时尚未启用的[[渝贵铁路|渝贵铁路]]进行动态测试。<ref>{{cite web|url=http://www.rw-tk.com/newsitem/278086892|title=CR200J动力集中型动车组，已经到成都铁路局开展实验-北京瑞威铁科高铁技术有限公司 江西瑞威铁科铁路工程有限公司|website=www.rw-tk.com}}</ref>

2018年初，CR200J-4001动力车+1001车组及CR200J-3001动力车+4002车組，曾存放于[[中国铁路乌鲁木齐局集团有限公司|乌鲁木齐局集团]]乌鲁木齐动车所，并于[[兰新高铁|兰新高铁]]进行动态测试。

2018年3月底，CR200J-3001动力车+4002车组再前往[[兰渝铁路|兰渝铁路]]，于兰州至四川广元间进行动态测试，并存放于兰州西动车所<ref>{{cite web|url=http://www.gaoloumi.com/forum.php?mod=viewthread&tid=3118622|title=高楼迷论坛-首列CR200J动力集中型动车组抵达兰州西站开展实验|website=gaoloumi.com}}</ref>，直至同年8月5日結束。

2018年8月，CR200J-1004动力车+4003车组及CR200J-1005动力车+4004车组前往[[滨洲铁路|滨洲铁路]]及[[哈佳铁路|哈佳铁路]]进行动态测试，是CR200J首次进入东北地区。

2018年9月10日，一列由10輛唐山制拖车（700101~700110）、8辆浦镇制拖车（700201~700208）、株机制动力车（CR200J-1002）及大连制动力车（CR200J-3002）组成的CR200J推拉式列车编组，由北京环铁以甲种运送方式，调往成都进行测试，并由配属北京机务段的[[和谐3D型电力机车|和谐3D型电力机车]]第0077号机车担当本务机车<ref>{{cite web|url=http://www.trainnets.com/archives/31207|title=非人狂想屋 - 你的火车发源地 » HXD3D牵引卧铺动集前往成都|website=trainnets.com}}</ref>。

2018年11月28日，CR200J型电力动车组获得由[[国家铁路局|国家铁路局]]颁发的型号合格证和制造许可证，取得批量生产条件和商业运营资格<ref name=cb/>。

2018年12月14日，[[HXD1B|HXD1B]]-0516牵引2辆CR200J型电力动车组量产动力车FXD1-J0006、0007自株洲出发，前往浦镇与配套拖车进行编组<ref>{{cite web |author1=吕杰琛 |title=HXD1B牵引FXD1-J北上 |url=http://www.trainnets.com/archives/32706 |website=非人狂想屋}}</ref>，并在12月18日与浦镇生产的6001车组（满编组16节）实现编组。动力车型号已由[[HXD1G|HXD1G]]型更名为FXD1-J型。

2018年12月29日，FXD3-J0010及0011機車以甲種運送方式，由[[ND5型柴油機車|ND5型柴油機車]]送往北京，交付北京局集團北京車輛段。

2019年1月5日，随着全国铁路实行新的运行图，CR200J型电力动车组正式上线载客运营。<ref>{{cite web |author1=新华网-张晨霖 |title=探访时速160公里动力集中动车组“绿巨人” |url=http://www.dzwww.com/xinwen/guoneixinwen/201901/t20190105_18259146.htm}}</ref>值得一提的是，正式载客运营后的CR200J拖车车身喷涂的拖车编号均已将“25T”字样消去，仅保留车厢席别代码。但由于部分执行卧铺动车组的CR200J车底内部异味严重，相关列车直至1月23日才正式开行<ref>{{cite news|title=时速160公里复兴号车厢甲醛超标，列车停开中车着手解决|url=https://www.jiemian.com/article/2790190.html|publisher=界面新闻|date=2019-01-14}}</ref>。

==CR200J型动车组基本组成及技术特性==
===編組形式===
該型動車組有長編組，短編組和靈活編組3種编组形式。短編組爲1動8拖，長編組為2動16拖。在短編組中的一端設置無動力控制座車，而長編組兩端均爲動力車。在席别佈局中，短編組全部爲座車，分別爲一等座車和二等座車；而長編組設置二等座車，一等臥車，二等臥車三種席別。

====CR200J（短编组）====
短编组的CR200J型为1动8拖[[動力集中式列車|動力集中型动车组]]，由1辆带驾驶室的KZ25T型控制拖车、7节不带驾驶室的中间车和1辆[[复兴1型电力机车|FXD1-J]]或[[复兴3型电力机车|FXD3-J]]型动力车组成。动力车可在列车前部牵引运行，也可在列车尾端推行，这類似[[中国铁路|中国铁路]]1990年代末的“[[DDJ1型電力動車組|大白鲨]]”、“[[DJJ1型电力动车组|蓝箭]]”等推挽式电力动车组的设计。短编组设计了重联运行模式，可采用动车(Mc)与控制拖车(Tc)重联（Mc1+7T+Tc1+Mc2+7T+Tc2）或控制拖车与控制拖车重联（Mc1+7T+Tc1+Tc2+7T+Mc2）两种方式运用，但不允许使用动力车与动力车重联（Tc1+7T+Mc1+Mc2+7T+Tc2）的方式运用。在短编组CR200J型动车组的席别布局中，除控制车为一等座车外，其余均为二等座车。

====CR200J（长编组）====
长编组的CR200J型动力集中电力动车组设计为2动16拖的编组，由16节中间车成的拖车组与两台分别位于两端的[[复兴1型电力机车|FXD1-J]]或[[复兴3型电力机车|FXD3-J]]型动力车组成。动力车分列编组的前后两端进行推挽运行，类似于[[中国铁路DJJ2型电力动车组|中国铁路DJJ2型电力动车组]](中华之星)及[[和谐长城号|和谐长城号]]的设计。与短编组的CR200J不同，长编组不设带有驾驶室的拖车，由于编组长度已接近站台有效长，故无法重联运行。
====CR200J（灵活编组）====
CR200J型动车组可根据运输的实际需要，在出厂前进行编组调整，除动力车外，诸如餐车、一（二）等座车、一（二）等卧车均可根据客流、开行线路等因素在9节至16节的范围内进行增减，但编组增减的必须在出厂前完成，且须录入铁路总公司的数据库方可上线运营。这一固定编组的管理方式与其他复兴号、和谐号动车组基本一致。CR200J型動車組在運用中按照既有高速動車組的各級檢修標準，並根據相關里程周期及時間周期進行檢修作業。<ref>{{cite web|url=https://www.bilibili.com/video/av14184572|title=中車唐山機車車輛CR200J動車組宣傳短片|website=bilibili.com}}</ref> <ref name=cb>{{cite web|url=http://www.cb.com.cn/index/show/zj/cv/cv13434591269|title=全路普客更新节点来临 时速160公里复兴号获“准生证”|publisher=中国经营网}}</ref>。

===牽引系統===
該型動車組動力車使用交流-直流-交流電傳動方式，這與其他中国铁路运用中的高速動車組和交流传动机车一致；其中牽引系統採用了{{来源请求|6.5kV IGBT，而列車亦採雙電路設計，分別爲DC600V供電，及牽引系統供電）}}。

===制動系統===
CR200J型動力集中動車組拖車還在既有25T型提速客車的基礎上升級了基礎制動裝置，采用整體單元缸夾鉗，粉末冶金閘片和鑄鋼制動盤，避免了既有普速客車在准高速運用中基礎制動裝置長期存在的熱負荷不均，也避免了該問題所導致的冒烟、磨耗嚴重、制動盤熱裂紋等故障，延長了閘片和制動盤更換周期，降低了運用成本。

===列车网络===
與[[25T|25T型客車]]不同的是，該動車組采用了双路实时在线Lonworks[[列車网络|列車网络]]，两路自動選擇、互爲冗餘。參考了動車組列車網絡的設計，將網關，代理節點集成到列車控制單元VCU中，提高了系統可靠性，并設置了軸溫報警、車門控制、烟火報警、制動系統四條安全環路。网络采用了可靠的硬綫控制，防止由於通信失敗導致的關鍵控制失效，提升了安全標準。该网络基本实现了高速動車組級別的遠程檢測控制功能。

=== 互通性 ===
該型動車組在兩端頭車设置了可与其他普速機車車輛通用的[[詹式車鈎|詹式車鈎]]，保證了該車型與其他机车车辆进行连挂、救援的互通性，而該動車組中間車則沿用了既有25T型客車所采用[[密接式车钩|密接式車鈎]]。

===外观设计===

==== 涂装 ====
[[File:400101@BCR_(20170927153007).jpg|缩略图]]
CR200J型动车组涂装以绿色(RAL 160 40 70)为底色，采用信号黑色(RAL 9004)涂装车窗边框及盲窗带，并配有国标淡黄色(Y06)色带，在动力车的端部标有“CR”标志，并在中部上方涂有[[魏碑|魏碑]]“复兴号”字样。动力车前部灯外轮廓及司机室盲窗带为石墨灰色（RAL 7024），附有暗红色[[中华人民共和国铁路路徽|中国铁路路徽]]，走行部、排障器及裙板为玄武岩灰(RAL 7012)，动力车侧面则有类似于箭头的淡黄色(Y06)装饰线条。这些设计元素均与动力分散型的[[CR400AF|CR400AF]]动车组相仿。在试验过程中，曾采用白色底色与蓝或橘红色带相搭配的涂装样式。而量产版的动车组均改以官方称为国槐绿 <ref>色号为#2AA332，偏向鲜艳的草绿色，官方称之为“国槐绿”。</ref>为底色，搭配黄色条带<ref>{{cite news|title=探营复兴号新型动车组：色彩谱系更丰富，超长列车拟下月投用|url=https://www.thepaper.cn/newsDetail_forward_2769125|publisher=澎湃新闻|date=2018-12-24}}</ref>作为装饰。

==== 外部设施 ====
[[File:CR200J_Klimaanlage.jpg|缩略图]]
在实验过程中，CR200J型动车组拖车顶部的单元式空调机组曾采用与国产25T型客车室同样的布置，高于车体顶部轮廓。量产版拖车对该处进行了设计优化，在空调机组与车体间的空隙中安装了螺栓固定的斜面导流板，以达到减小动车组整车空气阻力的目的。

CR200J型动车组一、二等卧铺拖车设置一对车门，而一、二等座车则设置两对车门。车门旁設有 LED 旅客信息显示器，显示车次、车厢号及发到站信息。

===內部設計及服务设施===
[[File:Interiors_of_CR200J_first_class_coach_20190218.jpg|thumb]]
該型動車組在内部裝飾設計中，同样参考了既有“[[复兴号电力动车组|复兴号]]”高速動車組的設計元素。与25T型客车相比，每车仍保留三对可半幅打开的通风窗，但各车窗边缘均改为略向外突出的水平窗台，可以放置较小的物品。车厢侧壁上为每排席位增设了220V 50Hz及USB电源插座，同时车内为旅客提供无线网络，使旅客乘车体验有所改善。

车厢两端均设有[[真空廁所|真空集便厕所]]。车厢连接处安装有向两侧打开的手动滑动门；而客室与通道间则采用与25T型客车相同的单侧电动按钮式滑动门相隔，门上方設有發光二極體（LED）旅客列車資訊顯示板，可顯示列車沿途停靠站、铁路宣传资讯、列車到站提示、抵站的室外温度以及列車的時速等資訊。另外，各车走廊顶部均安装有[[摄像头|监控摄像头]]，以加强保安。

二等座席位采用3+2的布局，一等座则为2+2布置，中间为过道。座椅靠背角度可调整，但不可旋转，所有乘客均面向车厢中心，排布方式{{来源请求|与大部分欧洲铁路客车及中国铁路软座车相仿}}。车厢中心面对面座椅间设有固定式茶桌，而其余旅客则可使用座椅后的折叠小桌板。

==運用狀況==
该動車組可在约10万公里的既有电气化铁路线網中开行，從豐富客運產品的角度看，该型動車組既可用於开行長途“夕发朝至”列车，也可开行中长途列车和城际短途列车<ref name=cb/>。2016年一名由[[中国铁路总公司|中国铁路总公司]]运输局营运部的一名工程人员发表的文章则指出，160km/h等级的动力集中型动车组造价和运营成本与[[中国铁路25T型客车|普通25T型客车]]不大，但又远低于200至250km/h等级的动车组列车；同时动集编组固定，回送效率、检修效率均高过普速列车；车内客运人员配置数量远低于普速（短编3人，长编5人），降低人力成本；此外，作者认为通过在[[既有线|既有线]]开行此类列车、并按照[[D字头列车|D字头列车]]或[[C字头列车|C字头列车]]标准收取票价，可为铁路部门提高经营效益<ref name="efficiency">{{cite journal|author=孟凡峰|title=160 km/h动力集中型动车组对构建完整客运产品体系作用研究|work=中国铁路|year=2016|doi=10.3969/j.issn.1001-683X.2016.10.009}}</ref>。

按照[[中國鐵路總公司|中國鐵路總公司]]於2018年初的計劃，本型動車組將率先由[[北京局集團|北京局集團]]、[[上海局集團|上海局集團]]、[[武漢局集團|武漢局集團]]及[[昆明局集團|昆明局集團]]使用，並擬配屬北京西、上海南、[[漢口動車所|漢口]]及昆明西车辆段，而相應车辆段亦須為此進行改建工程。<ref>{{cite news|title=最高时速160公里！“绿巨人”动车组在杭起跑|url=https://appm.hangzhou.com.cn/article_pc.php?id=260717|publisher=杭州网|date=2019-01-23}}</ref>。

2018年11月30日，中国铁路总公司正式发布时速160公里动力集中动车组采购公示，购置10列9节编组（1动8拖短编）、18列18节编组（2动16拖长编），<ref>{{cite web |author1=中国铁路投资有限公司 |title=时速160公里动力集中动车组采购项目采购方式公示 |url=http://www.cric-china.com.cn/zbdlyw/cgfs/201811/t20181130_1820.html |website=中国铁路投资有限公司网站}}</ref>接替现有[[京沪铁路|京沪铁路]]上的动卧及部分夕发朝至的特快列车。

==列车编组==
实际编组与本列表可能存在一定出入。
;车组号段分配
*唐山公司制造的短编组动车组号段为1001~1999，长编组为5001~5999；
*浦镇公司制造的短编组动车组号段为2001~2999，长编组为6001~6999
*长客股份制造的短编组动车组号段为3001~3999，长编组为7001~7999
*四方股份制造的短编组动车组号段为4001~4999，长编组为8001~8999
*机车与拖车组合相对固定，无特殊情况不进行拆分运行，整列车组号跟随拖车车组；
*动车仅喷涂“FXD”字头车号，整列编组的车号以机械间所在的9号车厢上所示车号为准。
;拖车型号
* ZY<sub>25T</sub>：一等座車（First Class Coach）；
*KZ<sub>25TA/B</sub> : 一等座车（First Class Coach)； *25TA为唐山平台控制车，25TB为浦镇平台控制车。
*ZE<sub>25T</sub>：二等座車（Second Class Coach）；
* ZEC<sub>25T</sub>：二等座車／餐車（Second Class／Dining  Coach）；
* WY<sub>25T</sub>：一等卧车（First Class Sleeper Coach）；
* WE<sub>25T</sub>：二等卧车（Second Class Sleeper Coach）。

;英文字母意思
* Z：Zuo（[[汉语拼音|拼音]]），座，座车；
* Y：Yi（[[汉语拼音|拼音]]），一，一等；
* E：Er（[[汉语拼音|拼音]]），二，二等；
* C／CA：Can（[[汉语拼音|拼音]]），餐，餐车；
* W：Wo（[[汉语拼音|拼音]]），卧，卧车。

== 争议 ==
===变相涨价===
2019年1月5日起，CR200J型取代部分机辆模式普速列车，且列车等级由“T、Z”等升级为“D”字头。而以上线运营的时速160公里复兴号列车车次看，其票价相对于之前的K/T/Z普通旅客列车票价涨幅较为明显，但从运行时间压缩上看，前者的优势并不明显。被批评有变相涨价之嫌。<ref>{{cite web |title=时速160公里普速列车票价普遍上浮 |url=https://finance.sina.cn/2019-01-10/detail-ihqhqcis4976001.d.html?vt=4&cid=76675 |website=新浪财经 |date=2019-01-10}}</ref>

===车内异味问题===
2019年初，多位中国铁路系统内部权威人士称，中国铁路总公司属下至少两个铁路局集团公司在近期发现新造时速160公里复兴号动力集中型动车组车厢空气甲醛残留超标（尤以中车唐山生产的列车为甚），暂停停止该车使用。中国铁路北京局集团有限公司官方微博发布回应新车内部气味较重，需充分散发至符合运营条件后再予以上线。<ref>{{cite web |title=中铁北京局公司：“复兴号甲醛超标”严重失实，新车需散气味 |url=https://new.qq.com/omn/20190114/20190114A1CR0C.html |website=腾讯新闻}}</ref>

===量产版涂装争议===
{{main|绿皮车}}
有观点认为量产版的列车将颜色改为较为亮眼的绿色涂装后不符合审美，影响美观。并且因为该涂装让人联想起了2014年颇具争议的“刷绿”事件，所以这款涂装一直以来都受到了大量诟病。也因此得到了“垃圾桶”<ref>{{cite web |title=因量产车进行喷涂工艺时，车身颜色与旁边的绿色垃圾桶极为相似，故名 |url=http://wx3.sinaimg.cn/large/ae6f68fdly1fwwywrn14qj20qo0f0406.jpg}}</ref>，“菜青虫”等称号。<ref name="知乎 2018">{{cite web | title=如何评价 CR200J 型动力集中式动车组列车的绿色涂装？ | website=知乎 | date=2018-11-05 | url=https://www.zhihu.com/question/301349624 | language=zh | access-date=2019-04-06}}</ref>
