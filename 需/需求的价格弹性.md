'''需求价格弹性'''（'''Price elasticity of demand'''），在[[经济学|经济学]]中一般用来衡量需求的数量随商品价格的变动而变化的[[弹性_(经济学)|弹性]]。通常来说，因为商品价格的下跌会导致需求数量的增加，反之商品价格的上升会减少需求；所以一般情况下价格与需求成反比，需求的价格弹性系数为负数。分析师通常知道，但经常会省略负号，有时会引起误会。实际上，仅有如[[韦伯伦商品|韦伯伦商品]]和[[吉芬商品|吉芬商品]]等不符合[[需求定律|需求定律]]的特殊商品的需求价格弹性会是正数。 通常来讲，当需求价格弹性绝对值小于1时，这个商品可以称为缺乏弹性；即：当商品价格产生变动时，对需求的影响较小。反之当需求价格弹性的绝对值大于1时，这个商品可以称为富有弹性。需求彈性為0时，則代表完全無彈性，即此商品存在着与价格无关的刚性需求。需求弹性为无限大时，则代表该商品具有完全弹性，即使价格产生轻微变动也会使得需求全部消失。

当需求价格弹性恰好为1时，称此商品拥有单位弹性，此时可以达到利润最大化。因此商品的需求价格弹性，可以反应该商品的[[租税负担率|租税负担率]]。某件商品的需求价格弹性可以通过[[市场调研|市场调研]]以及[[联合分析|联合分析]]得出。



假设<math>Q</math>为某个商品的需求，<math>P</math>为该商品的价格，则计算需求的价格弹性<math>E_d</math>为，

:<math>E_d=\frac{\Delta Q / Q}{\Delta P / P}=\frac{\Delta Q}{\Delta P}\cdot\frac{P}{Q}</math>

使用微积分：
:<math>E_d=\frac{d Q}{d P}\cdot\frac{P}{Q}</math>

== 不同的弹性程度 ==
对于商品的不同弹性程度，我们有不同的称呼

# 當<math>E_d=0</math>：'''完全无弹性'''（perfectly inelastic）
# 當<math>0<|E_d|<1</math>：'''缺乏弹性'''、'''不富彈性'''，或'''無彈性'''（inelastic）
# 當<math>|E_d|=1</math>：'''单位弹性'''、'''單一彈性'''，或'''恆一彈性'''（unitarily elastic）
# 當<math>1<|E_d|<\infty</math>：'''富有弹性'''（elastic）
# 當<math>E_d=\infty</math>：'''完全弹性'''或'''完全有彈性'''（perfectly elastic）

== 影響需求彈性的因素 ==
* '''替代品的多寡
一件商品的替代品愈多，該商品的需求彈性就愈大。<ref name="經濟學9789574184798">{{cite book|author=[[張清溪|張清溪]]、[[許嘉棟|許嘉棟]]、[[劉鶯釧|劉鶯釧]]、[[吳聰敏|吳聰敏]]|title=《經濟學》|year=2011|publisher=雙葉書廊|isbn=9789574184798|pages=第60頁}}</ref>以飲品市場為例，有[[百事可樂|百事可樂]]，有[[可口可樂|可口可樂]]。對可口可樂的需求彈性自然高。
* '''該物消費支出佔總所得的比例'''
報紙非必需品，但是因為只售1元，佔消費者的支出比例不大，需求彈性便低。
* '''物品耐用性'''
當某種耐用、耐久的物品出現漲價，消費者將會將手中已有的商品延遲丟棄湊合使用，延遲購買新品，所以此類商品彈性較高。
* '''必需品或奢侈品'''
** 一般情况下，[[必需品|必需品]]由于生活中不可缺少的，小量的价格变动不会引起需求急剧变动，因此被认为是缺乏弹性的，而[[奢侈品|奢侈品]]则是富有弹性的。<ref name="經濟學9789574184798"/>
** 特例：理性消费者对商品的需求数量随着其价格的上升而上升也是可能的。最典型的两类商品为[[吉芬商品|吉芬商品]]和[[韋伯倫商品|韋伯倫商品]]。
* '''商品的定義範圍'''
该商品的销售区域，使用区域等区域性或者其他特性。以区域性而言，房地产就是典型的区域性商品，每个不同区域的房地产都有不同的价格和不同的需求量。
* '''時間的長短'''
隨著時間的長度增加，商品會愈來愈有彈性。<ref name="經濟學9789574184798"/>

例如：當汽油的價格上漲時，汽油的需求量只會在頭幾個月微幅減少。不過，隨著時間的增加，人們會買更省油的汽車、改搭大眾運輸工具、或搬到離上班地點較近的區域居住。在幾年之內，汽油的需求量會明顯減少。
