'''禾雀'''（[[學名|學名]]：''{{lang|la|Lonchura oryzivora}}''）又名'''-{zh-cn:文鳥;zh-tw:禾雀;}-'''、'''爪哇禾雀'''、'''黑芙蓉'''、'''白芙蓉'''、'''銀芙蓉'''、'''淺黃褐芙蓉'''、'''爪哇雀'''，為雀形目[[梅花雀科|梅花雀科]][[禾雀屬|禾雀屬]]的[[鳥類|鳥類]]，分布於印尼，一般生活于低地草原或具草甸、灌丛的空旷林地、也见于耕地、花园、城郊的村镇以及活动于稻田或甘蔗田、树林或灌木丛中。该物种的模式产地在[[亚洲|亚洲]]及[[埃塞俄比亚|埃塞俄比亚]]<ref name='id4092'/>。

== 分佈地域 ==
原分佈於[[印尼|印尼]]的[[爪哇|爪哇]]、[[峇里|峇里]]及[[巴韋安島|巴韋安島]]，現已引入到其他地方。可惜在原產地已[[瀕臨滅絕|瀕臨滅絕]]。

== 特徵 ==
體長13至14公分、體重24至30公克。原生地為[[印尼|印尼]]的[[爪哇|爪哇]]與[[峇里島|峇里島]]地區，是知名的觀賞鳥類。棲息于平原地帶，主要食物為[[禾本科|禾本科]]植物的[[種子|種子]]。儘管禾雀在全世界被養殖作為[[寵鳥|寵鳥]]，但禾雀在原生地的野生族群因環境的人為限制而極需保護。

== 禾雀與[[鳥類貿易|鳥類貿易]] ==
禾雀被飼養作寵物鳥已有幾個世紀的歷史，例如[[日本|日本]]於[[江户時代|江户時代]]已從[[中國|中國]]引入禾雀作[[寵物|寵物]]。

== 禾雀與[[中國文化|中國文化]] ==
據說在400年前已有[[中國人|中國人]]飼養禾雀。因此禾雀經常被用作中國占卜術中'''靈雀占卜'''的「媒鳥」。[[相士|相士]]首先把禾雀從小盒中放出來，然後禾雀會在占卜用的签筒抽一支签，相士此後便會根據签文向客人解釋签文。這種占卜術在[[香港|香港]][[廟街_(香港)|廟街]]极為盛行。

== 威脅 ==
由於棲息地開發，以及經常被設陷阱捕捉作寵物[[鳥類貿易|鳥類貿易]]，野生爪哇禾雀在原生地數目下降。

== 保護 ==
《[[瀕危野生動植物種國際貿易公約|瀕危野生動植物種國際貿易公約]]》（CITES）附錄Ⅱ品種以及[[世界自然保育聯盟|世界自然保育聯盟]]([[IUCN|IUCN]])中的易危物種。
