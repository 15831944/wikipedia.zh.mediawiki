'''中国佤族司岗里摸你黑狂欢节'''是[[云南省|云南省]][[沧源佤族自治县|沧源佤族自治县]]的一个大型旅游文化节庆活动，由沧源县委和县政府主办，承办方为云南庆典文化传播有限公司<ref name="2009-08-10"></ref>。2004年举办第一届狂欢节，此后每年的5月1日前后都会举办活动<ref>{{cite web |url=http://www.mzhb.com/zzdf/%E6%91%B8%E4%BD%A0%E9%BB%91%E7%8B%82%E6%AC%A2%E8%8A%82/ |title=“摸你黑”狂欢节 |author1=滕俊 |author2=黄若天 |date=2014-03-31 |website=www.mzhb.com |publisher=民族画报 |access-date=2017-12-09 |quote=}}</ref>，到2018年已举办14届<ref>{{cite web |url=http://yn.sina.com.cn/news/b/2018-04-30/detail-ifzvpatr4458405.shtml |title=2018佤族司岗里摸你黑狂欢节在云南沧源启幕 |author= |date=2018-04-30 |website=yn.sina.com.cn |publisher=沧源县委宣传部 |access-date=2018-05-06 |quote=}}</ref>。

==活动==

“摸你黑”是[[佤族|佤族]]的一种欢庆活动，使用称作“娘布洛”的天然涂料相互涂抹，以示祝福平安健康<ref>{{cite web |url=http://cul.china.com.cn/minsu/2011-05/03/content_4173497.htm |title=佤族"摸你黑"狂欢节 |author=秦晴 |date=2011-05-03 |website=cul.china.com.cn |publisher=新华网 |access-date=2017-12-09 |quote=}}</ref>。“摸你黑”取意于佤族民间用锅底灰、牛血、泥土涂抹在额头上以驱邪祈福求平安的习俗<ref name="2009-08-26"></ref>。“摸你黑”一词是从佤语音译而来，意为“就是这样”<ref name="2017-04-29">{{cite web |url=http://travel.sina.com.cn/domestic/news/2017-04-29/detail-ifyetwtf8996640.shtml |title=2017中国佤族司岗里摸你黑狂欢节今日启幕 |last= |first= |date=2017-04-29 |website=travel.sina.com.cn |publisher=新浪旅游 |access-date=2017-12-10 |quote=}}</ref>，引申为“这儿是我们追求的、我们所期待的，坚持下去吧，坚持到永久”<ref name="2009-08-26">{{cite web |url=http://www.china.com.cn/culture/aboutchina/wazu/2009-08/26/content_18407688.htm |title=佤族“摸你黑狂欢节”(组图) |last= |first= |date=2009-08-26 |website=www.china.com.cn |publisher=中国网综合消息 |access-date=2017-12-09 |quote=}}</ref>。除了“摸你黑”之外，狂欢节中也有[[斗牛|斗牛]]、[[木鼓舞|木鼓舞]]、[[甩发舞|甩发舞]]等佤族传统节目表演<ref name="2009-08-10"></ref>。2012年的沧源摸你黑狂欢节，为了寻找[[台湾原住民族|台湾原住民]]和云南佤族的共通之处，[[台湾|台湾]][[阿美族|阿美族]]、[[布农族|布农族]]等4个民族24名原住民组成参访团联谊赴沧源参加活动<ref>{{cite web |url=http://www.taiwan.cn/local/yunnan/dongtaixinwen/201205/t20120502_2517164.htm |title=云南佤族联谊台湾原住民 共度“摸你黑”狂欢节 |author1=胡远航 |author2=顾一航 |first= |date=2012-05-02 |website=www.taiwan.cn |publisher=中国新闻网 |access-date=2017-12-09 |quote=}}</ref>。

==影响和荣誉==

摸你黑狂欢节被评为云南省最具魅力和影响力的“十大民族节庆”之一，促进了沧源县旅游产业的发展<ref name="2009-08-26"></ref>，2015年沧源摸你黑狂欢节三天内接待了超过12万名游客<ref>{{cite web |url=http://yn.news.163.com/15/0505/15/AOS4EEC903230LLH.html |title=沧源摸你黑狂欢节接待游客破12万 |last= |first= |date=2015-05-05 |website=yn.news.163.com |publisher=春城晚报 |access-date=2017-12-09 |quote=}}</ref>。在2009年举办的“第五届中国国际会展文化节”中，摸你黑狂欢节获得“2008年至2009年度中国十大魅力节庆活动”荣誉并获得中国会展界最高奖“金海豚奖”<ref name="2009-08-10">{{cite web |url=http://cpc.people.com.cn/GB/165240/166030/166031/166044/9852637.html |title=摸你黑狂欢节——云南民族文化节庆的骄傲 |author=胡馨予 |date=2009-08-10 |website=cpc.people.com.cn |publisher=人民网 |access-date=2017-12-09 |quote=}}</ref>。
