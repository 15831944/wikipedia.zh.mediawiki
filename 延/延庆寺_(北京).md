'''延庆寺'''，位于[[北京市|北京市]][[海淀区|海淀区]][[万寿寺路|万寿寺路]]，是[[汉传佛教|汉传佛教]]寺院。

==简介==
延庆寺位于[[万寿寺_(北京)|万寿寺]]东侧，[[南长河|长河]]北岸，靠近[[广源闸|广源闸]]。万寿寺和延庆寺门前有[[清朝|清朝]]建的万寿寺码头。[[清朝皇帝|清朝皇帝]]、皇后、太后等自[[北京故宫|紫禁城]]至[[颐和园|颐和园]]往返途中在万寿寺码头下舟到万寿寺上香时，延庆寺是一般官员和服侍人员用膳和休息的场所。延庆寺还是长河两岸民间庙会的举办地。据传延庆寺初建时曾是万寿寺的一部分，但究竟是初建时两寺便已分开，还是后来才分开，现已难查考<ref name=bjrb>{{Cite web |url =http://bjrb.bjd.com.cn/html/2017-08/24/content_169556.htm  |title = 延庆寺：长河的重要节点 |author =  |date =2017-08-24  |publisher = 北京日报 |language =  |accessdate =  |quote =  }}</ref><ref name=bw>{{Cite web |url =  http://www.takefoto.cn/viewnews-1327708.html|title = 在延庆寺住了45年的老居民 没想到家门口成了景点 |author =  |date = 2017-11-17  |publisher =北晚新视觉  |language =  |accessdate =  |quote =  }}</ref>。

延庆寺内的主要建筑有正殿和东西偏殿，原先寺内还有[[明朝|明朝]][[天顺|天顺]]六年（1462年）重修碑，[[嘉靖|嘉靖]]四十一年（1562年）[[文林郎|文林郎]]知[[肥城县|肥城县]][[万鹏程|万鹏程]]撰书《创建延庆寺山门记》，清朝[[顺治|顺治]]十二年（1655年）文林郎知肥城县县事[[袁沛|袁沛]]主持、[[平阴县|平阴县]][[廪生|廪生]]程步云撰书《重修延庆寺正殿山门记》等碑石<ref name=bjrb/>。

《北京寺庙历史资料》载：“延庆寺，坐落西郊三分署万寿寺街19号，建于清乾隆三十一年，属私庙。本庙面积二十二亩零八分，房屋六十一间，附属茔地二十六亩，田地三十五亩，土房五间，管理及使用状况为住持管理。除佛殿外，分禅舍及租户居住。庙内法物有[[释迦牟尼|释迦佛]]一尊，[[阿难|阿难]][[迦叶|迦叶]]二尊，[[罗汉|罗汉]]十八尊，均泥塑。大铜磬一口，铁五供两堂，三经一部，石碑三座。另有白果、槐、楸、柏等树木十七株。茔地柏树六十二株，槐树四株，马尾松九株，榆树四株，杨树两株。”延庆寺东南角原先还有一口古井，俗称大寺井，今已无存<ref name=bjrb/><ref name=bw/>。

1967年后，陆续有[[中国人民解放军总后勤部|中国人民解放军总后勤部]]和[[中国人民解放军总政治部|总政治部]]的军人及家属搬进延庆寺居住，截至2017年延庆寺内已有居民二百余户，整座寺庙及周边沦为[[棚户区|棚户区]]<ref name=bjrb/>。

延庆寺是[[北京市海淀区文物普查登记项目|北京市海淀区文物普查登记项目]]。2012年1月北京市海淀区文化委员会立文物普查登记项目保护牌<ref>{{Cite web |url = http://whw.bjhd.gov.cn/whyc/hdww/wwgk/ |title = 文物概况 |author =  |date =  |publisher = 北京市海淀区文化委员会 |language =  |accessdate = 2018-08-08 |quote =  |archive-url = https://web.archive.org/web/20141230103931/http://whw.bjhd.gov.cn/whyc/hdww/wwgk/ |archive-date = 2014-12-30 |dead-url = yes }}</ref>。2018年3月，《北京市大运河文化带保护建设规划》及《北京市大运河文化带保护建设五年行动计划（2018年-2022年）》经[[中共北京市委|中共北京市委]]常委会审议通过，同年延庆寺周边棚户区改造方案已由[[紫竹院街道|紫竹院街道]]上报北京市海淀区人民政府<ref>{{Cite web |url = https://www.thepaper.cn/newsDetail_forward_2328952 |title = 大运河文化带保护建设加快推进，北京将统一运河文化遗产标识 |author =  |date = 2018-08-08 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://bjwb.bjd.com.cn/html/2017-09/05/content_172870.htm  |title =紫竹院延庆寺周边将启动棚改  |author =  |date =2017-09-05  |publisher = 北京晚报 |language =  |accessdate =  |quote =  }}</ref>。
