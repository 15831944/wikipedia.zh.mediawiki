'''延安南泥湾机场'''（{{airport codes|ENY|ZLYA}}）是一座位于[[中国|中国]][[陝西省|陝西省]][[延安市|延安市]]的军民合用[[机场|机场]]，机场等级为4C级<ref>{{cite web|url=http://www.cwag-ya.com/s_index.asp?node_id=55|title=延安机场简介|publisher=延安机场|date=|accessdate=2014-02-01|deadurl=yes|archiveurl=https://web.archive.org/web/20140202221557/http://www.cwag-ya.com/s_index.asp?node_id=55|archivedate=2014-02-02}}</ref>。延安南泥湾机场是陕西省和延安市确定的“重点项目”，在2013年开工<ref>{{cite web |author1=李玉素 |title=延安南泥湾机场试飞成功 预计10月正式通航 |url=http://www.chinanews.com/sh/2018/09-11/8624416.shtml |website=中国新闻网 |accessdate=2019-01-06}}</ref><ref>{{cite web |title=祝福祖国，携手奋斗创造新荣光——从习近平主席发表二〇一九年新年贺词看奋斗姿态 |url=http://cn.chinadaily.com.cn/a/201901/02/WS5c2c1fbda3106072a903394e.html |website=中国日报 |accessdate=2019-01-06}}</ref>。延安南泥湾机场于2018年9月完成试航，在2018年11月正式开通<ref>{{cite web |author1=周家琪 |author2=袁勃 |title=延安南泥湾机场通航 |url=http://politics.people.com.cn/n1/2018/1111/c1001-30393373.html |website=人民网 |accessdate=2019-01-06}}</ref>，是[[中华人民共和国机场运营统计列表|陕西省客运吞吐量第三大机场]]。
