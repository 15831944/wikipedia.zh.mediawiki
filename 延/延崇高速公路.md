'''延崇高速公路'''，即{{China Expwy Name|S3801|P=Beijing}}延庆-崇礼段，是在[[北京市|北京市]]及[[河北省|河北省]][[张家口市|张家口市]]境内的一条高速公路，南起北京[[延庆区|延庆区]][[兴延高速|兴延高速]]营城子立交北侧，北至河北[[崇礼区|崇礼区]]棋盘梁村北，全长81.691千米；设连接线（一级公路）两条：太子城互通连接线1.18km（延崇高速棋盘梁村东-太子城村南冬奥村），赤城南互通连接线10.575km（延崇高速赤城上虎村南-赤城雕鹗村连接京北一级公路）；设延伸线17.077km，起自延崇高速棋盘梁村北，止于张承高速白旗枢纽；设赤城支线一条15.147km（延崇高速赤城韩庄村枢纽- 赤城城区{{中国国道名|239}}与{{中国国道名|112}}交叉处）。

延崇高速公路及其支线、连接线、延伸线均为双向四车道，延庆区平原部分路段预留双向六车道，河北段主线设计速度100km/h，北京段主线、赤城支线、延伸线为80km/h，连接线为60km/h。<ref>{{cite web |author1=张家口环保局 |title=延崇高速河北段环评 |url=http://www.zjkepb.gov.cn/upload/files/2017/3/2212941693.rar}}</ref>

这条高速公路是2019年北京世园会的保障项目<ref>{{cite web |author1=北京日报 |title=保障2019年北京世园会 15项工程将于2018年内完工 |url=http://www.beijing.gov.cn/bmfw/jtcx/ggts/t1527705.htm}}</ref>和2022年在北京市区、延庆区及张家口市召开的[[2022年冬季奥林匹克运动会|冬季奥林匹克运动会]]的配套项目<ref>{{cite web |author1=张家口新闻网 |title=2022年冬奥会重大交通保障项目—— 延崇高速河北段首座特大桥贯通 |url=http://www.zjknews.com/news/nengyuan/201807/05/214153.html}}</ref>，也是是京津冀一体化西北高速通道之一<ref>{{cite web |author1=河北新闻网 |title=延崇高速河北段施工正酣 |url=http://hb.jjj.qq.com/a/20180628/006292_1.htm}}</ref>。该高速北京平原段（营城子-G110）2019年1月1日12时通车，主线其余路段计划于2019年通车，延伸线预计2021年通车<ref>{{Cite web|url=http://ztbzx.hbsjtt.gov.cn/Site/details_news.aspx?NewsGuid=aae47a21-ee0c-40d0-bc92-0436ab2fcef2&type=gg|title=延庆至崇礼高速公路河北段延伸工程施工招标公告|accessdate=2018-11-23|author=河北交通招标投标网|date=|work=ztbzx.hbsjtt.gov.cn|publisher=}}</ref>。

根据《京津冀跨省市省级高速公路命名和编号规则》的要求，原[[兴延高速|兴延高速]]以及[[延崇高速|延崇高速]]共同组成[[北京|北京]]-[[崇礼|崇礼]]高速公路，简称[[京礼高速|京礼高速]]。<ref>{{cite web |author1=新华社 |title=兴延高速、延崇高速平原段将于明年元旦通车 |url=https://baijiahao.baidu.com/s?id=1621081807168599186}}</ref>在实际的道路导向信息中也仅保留京礼高速的命名。

由于该高速在[[赤城县|赤城县]]金家庄隧道进口至[[崇礼区|崇礼区]]棋盘梁隧道出口大约7公里的长度内需要克服250米左右的高差，为降低纵坡保证车辆安全行驶，这一段工程被设计为由1个半径为860米的螺旋线的隧道，1个普通的直隧道以及二者间设置的大桥组成。2018年4月20日，经世界纪录认证英国总部认证官现场认证、授牌，金家庄特长螺旋隧道被认证为在建世界最长高速公路螺旋隧道。该隧道于2017年8月开工建设，计划于2019年2月底实现贯通。<ref>{{cite web |author1=长城网 |title=延崇高速金家庄特长螺旋隧道获在建世界最长高速公路螺旋隧道认证 |url=http://news.ifeng.com/a/20180425/57864570_0.shtml |accessdate=2018/4/25}}</ref>
