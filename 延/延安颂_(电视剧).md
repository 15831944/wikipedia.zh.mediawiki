《'''延安颂'''》，2003年上映的中国大陆电视剧，导演[[宋业明|宋业明]]、[[董亚春|董亚春]]，主演[[唐国强|唐国强]]、[[刘劲|刘劲]]、[[郑强|郑强]]、[[王伍福|王伍福]]、[[郭连文|郭连文]]。<ref>{{cite web |url=http://www.people.com.cn/GB/yule/8222/31082/31085/2258948.html |title=《延安颂》演职员表 |accessdate= |publisher=人民网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://www.people.com.cn/GB/yule/8222/31082/31083/2259003.html |title= 《延安颂》导演阐述  寻找延安魂|accessdate= |publisher=人民网 |work= |language =zh-cn}}</ref>

==剧情==
该剧讲述从1935年到1945年，即延安时期[[中国工农红军|中国工农红军]]由弱变强、由小变大，在[[国共内战|国共内战]]、[[抗日战争|抗日战争]]中的历史故事。<ref>{{cite web |url= http://ent.sina.com.cn/v/2003-11-28/1000242069.html|title= 资料：四十集电视剧《延安颂》故事梗概|accessdate=2003年11月28日 |publisher=新浪网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://ent.hunantv.com/e/h/20080720/23510_2.html |title=四十集电视剧《延安颂》 |accessdate=2008 |publisher= 金鹰网|work= |language =zh-cn}}</ref>

==花絮==
该剧是“纪念毛泽东同志诞辰110周年”的献礼剧。<ref name=" bull" /><ref>{{cite web |url=http://news.sina.com.cn/o/2003-12-13/06091329776s.shtml |title=电视剧《延安颂》12月19日央视一套播出 |accessdate=2003年12月13日 |publisher= 新浪网|work= |language =zh-cn}}</ref><ref>{{cite web |url=http://www.people.com.cn/GB/yule/8222/31082/31083/2259006.html |title=《延安颂》编剧 感言延安颂 |accessdate= |publisher=人民网 |work= |language =zh-cn}}</ref>而唐国强是第五次出演毛泽东。<ref name=" bull" >{{cite web |url=http://ent.163.com/ent_2003/editor/tv/tvstar/031226/031226_211097.html |title=《延安颂》热播唐国强要五演毛泽东 |accessdate=2003年12月26日 |publisher=网易 |work= |language =zh-cn}}</ref>

该剧由电视剧《[[长征_(电视剧)|长征]]》的原班人马拍摄。<ref>{{cite web |url=http://www.dzwww.com/xinwen/shandong/sdxwzt/mzdysd/zxbd1/t20031216_363787.htm |title=电视剧《延安颂》澄清红色历史 |accessdate=2006 |publisher=大众日报 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://news.sina.com.cn/o/2003-12-19/15001382919s.shtml |title= 《延安颂》三看点|accessdate= 2003|publisher= 新浪网|work= |language =zh-cn}}</ref>

该剧2003年在[[人民大会堂|人民大会堂]]首映。<ref>{{cite web |url=http://www.gmw.cn/01gmrb/2003-12/13/05-0D8098981E40DD4248256DFB00003818.htm |title=40集电视连续剧《延安颂》首映 |accessdate=2003 |publisher=光明网 |work= |language =zh-cn}}</ref>

该剧有[[黄克功|黄克功]]打死女大学生被枪毙的故事情节。<ref>{{cite web |url=http://gsrb.gansudaily.com.cn/system/2004/01/05/000098357.shtml |title= 电视剧《延安颂》中的几个情节|accessdate=2004 |publisher=甘肃日报 |work= |language =zh-cn}}</ref>

该剧中澄清了[[毛泽东|毛泽东]]、[[贺子珍|贺子珍]]、[[江青|江青]]三者之间的关系。<ref>{{cite web |url=http://ent.163.com/ent_2003/editor/tv/tvstar/040119/040119_215897.html |title=《延安颂》编剧王朝柱澄清两大误传 |accessdate=2004 |publisher=网易 |work= |language =zh-cn}}</ref>

该剧获得观众的认可和多项大奖。<ref>{{cite web |url=http://news.sina.com.cn/c/2004-02-09/09441752445s.shtml |title= 《延安颂》的启示|accessdate=2004年2月9日 |publisher=光明网 |work= |language =zh-cn}}</ref>

该剧从[[山西省|山西省]][[临县|临县]][[碛口|碛口]]开机拍摄，在山西省临县、[[陕西省|陕西省]][[延安市|延安市]]等地取景。<ref>{{cite web |url=http://www.people.com.cn/GB/wenyu/64/128/20030307/938709.html |title= 40集电视剧《延安颂》在山西临县开拍|accessdate= 2003|publisher= 人民网|work= |language =zh-cn}}</ref>
