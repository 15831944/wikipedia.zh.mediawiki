'''延安东路外滩站'''位于[[上海|上海]][[黄浦区|黄浦区]][[延安东路_(上海)|延安东路]]的道路北侧，[[中山东一路_(上海)|中山东一路]]口，为[[延安路中运量公交系统工程|延安路中运量公交系统工程]]的地面[[岛式月台|岛式车站]]，也是[[上海公交71路|上海公交71路]]的下行终点站。

此站点原为[[上海公交42路|42路]]的终点站，为修建该站点，42路被缩线到龙门路淮海中路。<ref>{{cite web|author1=张欣平|title=上海延安路通道公交线网优化方案征询公众意见-延安路,运量,公交,线路,征询,优化,换乘,高效率,运营,通道,-上海频道-东方网|url=http://sh.eastday.com/m/20161117/u1ai10059732.html|website=东方网|publisher=新民晚报社|accessdate=2017-07-09|date=2016-11-17}}</ref>

== 车站構造 ==
[[File:The_Bund,_Yan'an_Road_(E)_Station,_Yan'an_Road_Medium_Transit_Bus_System.jpg|thumb]]
延安东路外滩站为[[岛式月台|岛式月台]]，1面1线地面车站，站台设有半高屏蔽门，两侧设有无障碍斜坡，开工于2016年6月25日<ref name="kg">{{cite web | title = 延安路中运量公交系统工程开工 | publisher = 上海市交通委员会 | date = 2016-06-27 | url = http://www.jt.sh.cn/jtw/xwzx/n218/u1ai51190.html | accessdate = 2016-11-30 | language = zh | archive-url = https://web.archive.org/web/20161010223253/http://www.jt.sh.cn/jtw/xwzx/n218/u1ai51190.html | archive-date = 2016-10-10 | dead-url = yes }}</ref>，于2017年2月1日清晨正式投入使用<ref name="syy">{{cite news|title=上海延安路中运量公交开通试运营|url=http://www.thepaper.cn/newsDetail_forward_1610252|publisher=澎湃新闻|date=2017-02-01}}</ref><ref>{{cite web|title=《延安路中运量系统公交线路优化调整计划》网上征询市民意见的公告|url=http://www.jt.sh.cn/jtw/gsgg/n39/u1ai52106.html|website=上海交通网站|publisher=上海市交通委员会|access-date=2017-07-09|archive-url=https://web.archive.org/web/20170728202604/http://www.jt.sh.cn/jtw/gsgg/n39/u1ai52106.html|archive-date=2017-07-28|dead-url=yes}}</ref>。

== 周边设施 ==
* 站点北面：[[上海电信博物馆|上海电信博物馆]]
* 站点东面：[[外滩|外滩]]（[[中山东一路|中山东一路]]）、[[东金线轮渡|金陵东路轮渡站]]
* 站点南面：[[十六铺|十六铺]]

== 公交换乘 ==
附近有多个常规公交站点，并有一个轮渡站：
* 延安西路中山东一路([[上海公交1250路|原71路]]终点站)：311路夜宵、316路夜宵、320路夜宵；
* 中山东二路金陵东路：33路、55路、65路、147路、305路夜宵、317路夜宵、576路、868路；
* 中山东一路广东路：123路、123路区间、135路、307路夜宵、317路夜宵、[[上海公交576路|576路（含区间车）]]、868路；
* 金陵东路轮渡站：[[东金线轮渡|东金线]]

== 车站出口 ==
这个车站有两个出入口。
* 东出入口位于靠近调度室一侧，通过人行横道连接延安东路北侧人行道。
* 西出入口位于远离调度室一侧，通过人行横道连接延安东路北侧人行道。
