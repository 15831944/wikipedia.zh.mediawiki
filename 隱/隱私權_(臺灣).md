[[司法院大法官|司法院大法官]]於釋字第603號解釋，明白承認[[隱私權|隱私權]]受憲法所保障，隱私權雖非憲法明文列舉之[[權利|權利]]，惟基於[[人性尊嚴|人性尊嚴]]與個人主體性之維護及人格發展之完整，隱私權乃為不可或缺之基本權利，而受憲法第二十二條所保障。

至於隱私權的[[保護範圍|保護範圍]]為何? 依釋字603號解釋，可分為「[[空間隱私|空間隱私]]」與「[[私密隱私|私密隱私]]」兩部分。所謂空間隱私，係指「保障個人生活私密領域免於他人侵擾及[[個人資料|個人資料]]之自主控制」，所謂私密隱私，係指「保障人民決定是否揭露其個人資料、及在何種範圍內、於何時、以何種方式、向何人揭露之決定權，並保障人民對其個人資料之使用有知悉與[[控制權|控制權]]及資料記載錯誤之[[更正權|更正權]]。」。

== 歷史 ==

大法官於釋字第293號解釋首次提及隱私權，認為[[銀行法|銀行法]]第四十八條銀行對於顧客資料應保守秘密的規定，係為維護人民之隱私權。惟此處所指係為「法律上的隱私權」，而非憲法上具[[基本權|基本權]]性質的隱私權。

之後大法官於解釋中片斷提到「隱私」一詞，但並未深入論述。直到釋字第585號解釋，大法官方明白承認隱私權受憲法所保障，隱私權雖非憲法明文列舉之權利，惟基於人性尊嚴與個人主體性之維護及人格發展之完整，並為保障個人生活秘密空間免於他人侵擾及個人資料之自主控制，隱私權乃為不可或缺之基本權利，而受憲法第二十二條所保障。

== 定義 ==
 
依據釋字603號解釋，所謂[[空間隱私|空間隱私]]，意指私生活不受干擾，即個人得自主決定是否獨處，保有自我內在空間；[[資訊隱私|資訊隱私]]，係指就個人自主控制個人資料之資訊隱私權而言，乃保障人民決定是否揭露其個人資料、及在何種範圍內、於何時、以何種方式、向何人揭露之決定權，並保障人民對其個人資料之使用有知悉與控制權及資料記載錯誤之更正權。

== 憲法基礎 ==

[[中華民國憲法|中華民國憲法]]第22條：「凡人民之其他自由及權利，不妨害社會秩序公共利益者，均受憲法之保障。」；[[中華民國憲法|中華民國憲法]]第12條：「人民有[[秘密通訊|秘密通訊]]之[[自由|自由]]。」

== 保護範圍 ==

「人」的保護範圍

隱私權的主體為「個人」，就[[法人|法人]]而言，原則上認定為[[財產權|財產權]]（譬如[[商標|商標]]），但亦有擴大及於[[法人|法人]]團體的趨勢。針對[[外國人|外國人]]是否受到隱私權的保護，學者[http://mypaper.pchome.com.tw/news/liaobruce/3/1252147153/20051008145500/ 廖元豪]表示，大法官既然從人性尊嚴與人格發展導出「按捺指紋涉及隱私權」，此種權利屬於普世[[人權|人權]]。或有人認為「[[邊境管制|邊境管制]]」是極重要的政府利益，也必須是[[立法者|立法者]]明定。

「事物」的保護範圍
 
於釋字603號解釋中，指出[[指紋|指紋]]係個人身體之生物特徵，具備高度人別辨識功能之一種個人資訊，使指紋形成得以監控個人之敏感性資訊。  
 
== 侵害 ==
 
凡違背空間隱私或資訊隱私者，即構成對隱私權的侵害。
 
== 侵害的合憲性 ==

* 強制按捺指紋（釋字第603號解釋）

針對強制按捺指紋的合憲性，採取「[[中度標準|中度標準]]」，必須：
# 指紋乃重要之個人資訊，個人對其指紋資訊之自主控制，受「[[資訊隱私權|資訊隱私權]]」之保障。
# 國家基於「特定重大公益之目的」而有建立資料庫儲存之必要者，應以法律明定蒐集之目的，
# 手段與目的間具有「密切」關聯性，並應明文禁止「法定目的外之使用」。
# 應配合科技發展，運用足以確保資訊正確及安全之方式，並對指紋檔案採取必要之防護。

 
* [[監聽|監聽]]（釋字第631號解釋）

衡酌「保障人民秘密通訊自由不受非法侵害」及「確保國家安全、維護社會秩序」之[[利益衝突|利益衝突]]，除應有法律依據外，限制之要件應具體、明確，不得逾越必要之範圍，所踐行之[[程序|程序]]並應合理、正當。
 

* 基本人權衝突--隱私權與[[新聞自由|新聞自由]]（釋字第509號解釋）

為兼顧對個人名譽、隱私及公共利益之保護，法律得對[[言論自由|言論自由]]依其傳播方式為合理之限制。依其所提[[證據|證據]]資料，認為行為人有相當理由確信其為真實者，即不能以[[誹謗|誹謗]]罪之刑責相繩。
 

* [[核心領域|核心領域]]

維護人性尊嚴與尊重人格自由發展，乃自由[[民主憲政|民主憲政]]秩序之核心價值。基於人性尊嚴與個人主體性之維護及人格發展之完整，隱私權乃為不可或缺之基本權利。
