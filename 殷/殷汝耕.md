'''殷汝耕'''（{{bd|1883年||1947年|12月1日|catIdx=Y殷}}），[[浙江|浙江]][[温州|温州]][[平阳县|平陽]]人（今属苍南），字亦農。中國的財稅官僚與近代政治人物，並曾出任日本扶植的[[冀東防共自治政府|冀東防共自治政府]]要職。

== 早年 ==
殷汝耕於1883年出生，光绪30年12月(1904)官费留學日本(21岁)，宣统元年(1909)入第七高等工科,後加入[[中國同盟會|同盟會]]，曾追隨[[黃興|黃興]]參加[[辛亥革命|辛亥革命]]。1913年[[二次革命|二次革命]]失敗以後再赴[[日本|日本]]留學。1919年從[[早稻田大學|早稻田大學]]畢業回國，進入[[北洋政府|北洋政府]]財政部裡擔任司長，與[[殷同|殷同]]、[[袁良|袁良]]（前北平市長）、[[程克|程克]]（前天津市長）並稱日本通四巨頭。殷汝耕也用妻姓，取了个日本名叫井上耕二。

== 漢奸 ==
1935年，日本與中國在華北的衝突增加。11月24日，在[[土肥原賢二|土肥原賢二]]慫恿下，冀東「非武裝區」專員殷汝耕宣佈「自治」，成立「冀東防共自治委員會」。<ref name="民國史第八卷，第378頁">[[李新_(1918年)|李新]]總主編，[[中國社會科學院|中國社會科學院]]近代史研究所中華民國史研究室編，周天度、鄭則民、齊福霖、李義彬等著，《[[中華民國史|中華民國史]]》第八卷，[[北京|北京]]：[[中華書局|中華書局]]，2011年7月，第378頁</ref>殷汝耕出任冀東防共自治政府委員長，宣佈脫離[[國民政府|國民政府]]，並且將[[通縣|通縣]]、[[臨榆|臨榆]]、[[撫寧|撫寧]]、[[昌黎|昌黎]]、[[盧龍|盧龍]]、[[遷安|遷安]]、[[灤縣|灤縣]]、[[樂亭|樂亭]]、[[豐潤|豐潤]]等二十二縣納入其轄下，因此被國民政府通緝。冀東防共自治政府殷汝耕為行政長官，以[[池宗墨|池宗墨]]任祕書長，下設民政、財政、外交等五廳。國民政府成立[[冀察政務委員會|冀察政務委員會]]與殷汝耕等人進行對抗。

1937年3月2日，殷汝耕飛長春面見[[板垣征四郎|板垣征四郎]]，商談擴大偽組織事宜<ref name="民" >{{cite book |editors=[[李新_(1918年)|李新]]總主編，中國社會科學院近代史研究所中華民國史研究室，韓信夫、姜克夫主編 |title=《中華民國史大事記》 |location=北京 |publisher=[[中華書局|中華書局]] |year=2011 |isbn=9787101079982}}</ref>{{rp|5377}}。6月17日，殷汝耕所屬教導隊在冀東玉田縣城北強繳民間自衛槍械，激起人民反抗，參加反抗者共21村<ref name=民 />{{rp|5447}}。6月21日，殷汝耕由通縣至天津，往見日本[[華北駐屯軍|華北駐屯軍]]司令[[田代皖一郎|田代皖一郎]]，議定擴充「保安隊」實力，並重新分配防區<ref name=民 />{{rp|5450}}。殷汝耕後來在1937年7月29日發生的[[通州事件|通州事件]]中失去了日方的信賴。

1938年2月1日冀東防共自治政府被併入[[王克敏|王克敏]]的[[中华民国临时政府_(1937–1940)|中華民國臨時政府]]。[[抗戰|抗戰]]勝利後，殷汝耕被[[重慶|重慶]]國民政府以[[漢奸|漢奸]]的罪名逮捕，入押在南京[[老虎桥监狱|老虎桥监狱]]，在狱中手抄《[[金刚经|金刚经]]》。

== 身亡 ==
1947年被判處[[死刑|死刑]]。12月1日上午，前冀東自治政府主席殷汝耕在南京老虎桥監獄刑場被枪决<ref name=民 />{{rp|8464}}。臨刑前寫[[汪精衛|汪精衛]]〈飛花〉詩一首：“春來春去有定時，花開花落無盡期，人生代謝亦如此，殺身成仁何所辭。”<ref>汪精衛：《雙照樓集》</ref>，执行庭设在南京可容万人的朝天宫大殿，这天殿内殿外挤满观看的市民。

== 家族 ==
殷汝耕有一兄長[[殷汝驪|殷汝驪]]，為國民政府前財政部次長。汝骊厌恶汝耕，曾对[[陈铭枢|陈铭枢]]说：“此弟品质极坏，只要有利可图，他就能卖友，甚至会出卖民族。”殷汝驪之子[[殷之浩|殷之浩]]為[[大陸工程公司|大陸工程公司]]創辦人，殷之浩之女[[殷琪|殷琪]]曾為[[台灣高速鐵路股份有限公司|台灣高鐵]]董事長。
