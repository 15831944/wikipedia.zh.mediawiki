'''殷'''是[[中国|中国]]唐宋之交的[[五代十国|五代十国]]时期一个短命的王国。

== 叛闽 ==
[[唐朝|唐朝]]灭亡后，[[闽_(十国)|闽国]]于[[开平_(后梁)|开平]]三年（909年）建立。但在开国者[[王审知|王审知]]于[[同光|同光]]三年（925年）去世后，诸子争执不休，导致了[[永隆_(闽)|永隆]]五年（943年），王审知的儿子之一、当时皇帝闽景宗[[王延羲|王延羲]]之弟富沙王王延政叛乱，在闽国西北部割据出独立的殷国。

== 领土范围 ==
殷国很小，占有在今[[福建省|福建省]]北部和[[浙江省|浙江省]]南部的一块地区。北邻[[吴越国|吴越国]]，东、南邻闽国，西邻[[南唐|南唐]]。

殷国实际国土仅[[建州|建州]]、[[镛州|镛州]]（王延政升[[将乐县|将乐县]]为镛州，南唐灭闽后废州为县）、[[镡州|镡州]]三州（南唐灭闽后改为[[剑州|剑州]]，[[龙津县|龙津县]]为[[剑浦县|剑浦县]]，又新设延平、富沙二县，北宋时又改为南剑州）五县（将乐、昭武、建阳、建安、浦城，加龙津则为六县），故王延政也被时人讥为“五县天子”。

== 作为独立实体的灭亡 ==
[[天德_(王延政)|天德]]二年（944年），王延政的敌人闽景宗被弑，其将[[朱文进|朱文进]]称闽国皇帝。次年，朱文进也被弑，其军转而效忠王延政，尊其为闽国皇帝，请他还都闽都[[福州|长乐]]。王延政称闽国皇帝，结束了作为独立政权存在的殷国，但没有还都长乐，而是留在大本营建州。同年，南唐围建州，迫使他投降。<ref name=ZZTJ284>[[:s:資治通鑑/卷284|《资治通鉴》卷二百八十四]]</ref><ref name=ZZTJ285>[[:s:資治通鑑/卷285|《资治通鉴》卷二百八十五]]</ref>
