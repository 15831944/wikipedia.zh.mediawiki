'''殷国洪'''（{{bd|1911年||2017年|10月27日}}），[[湖北省|湖北省]]黄安县（今[[红安县|红安县]]）人，[[中国人民解放军开国少将|中国人民解放军开国少将]]。

==生平==
1932年加入[[中国共产主义青年团|中国共产主义青年团]]。同年参加[[中国工农红军|中国工农红军]]。1934年转入[[中国共产党|中国共产党]]。曾任[[红四方面军|红四方面军]]连指导员、团特派员。参加了[[鄂豫皖苏区|鄂豫皖苏区]]、[[川陕苏区|川陕苏区]]反“围剿”以及[[长征|长征]]。是[[西路军|西路军]]的幸存者，西路军失败后被俘，后趁机脱逃，找到了中共党组织<ref name=hxjww>{{Cite web |url = http://www.huaxia.com/lasd/hxrwk/ddrw/dl/2003/11/500536.html |title =殷国洪  |author =  |date = 2003-11-24  |publisher = 华夏经纬网 |language =  |accessdate =  |quote =  }}</ref><ref name=pp>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1841967  |title = 106岁开国将军殷国洪逝世，“开国将帅”仅23人健在 |author =  |date =  2017-10-29|publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

在[[抗日战争|抗日战争]]中，历任[[八路军|八路军]][[129师|一二九师]]东进纵队三团特派员、冀南军区九旅政治部保卫科科长，第三军分区政治部保卫科科长。参加了[[响堂铺战斗|响堂铺战斗]]、[[百团大战|百团大战]]。在[[第二次国共内战|第二次国共内战]]中，历任[[晋冀鲁豫军区|晋冀鲁豫军区]]二纵政治部保卫部部长，冀南区政治部保卫部部长，晋冀鲁豫军区十纵政治部保卫部部长，桐柏军区第一军分区副政委，[[湖北省军区|湖北军区]]组织部部长。参加了[[淮海战役|淮海战役]]等战役战斗<ref name=hxjww/><ref name=pp/>。

[[中华人民共和国|中华人民共和国]]成立后，曾任[[中国人民解放军海军青岛基地|海军青岛基地]]干部部部长。1959年毕业于中国人民解放军政治学院，后历任海军炮兵学校政治委员，海军高级专科学校政治委员<ref name=hxjww/><ref name=pp/>。后来海军高级专科学校演变为[[中国人民解放军海军航空工程学院|海军航空工程学院]]。1984年，殷国洪[[离休|离休]]。12年后全家迁居[[青岛|青岛]]，由[[中国人民解放军北部战区海军航空兵|海军北海舰队航空兵]]代管。在代管单位干休所，殷国洪卧床十余年，长期住在青岛[[中国人民解放军第四〇一医院|中国人民解放军第四〇一医院]]<ref name=pp/>。

1955年被授予大校军衔，1961年晋升为少将军衔。曾获二级[[八一勋章_(1955年)|八一勋章]]、二级[[独立自由勋章|独立自由勋章]]、二级[[解放勋章|解放勋章]]，1988年7月被中央军委授予中国人民解放军一级[[红星功勋荣誉章|红星功勋荣誉章]]。是[[第五届全国政协委员|第五届全国政协委员]]<ref name=pp/>。

2017年10月27日，殷国洪在青岛病逝，享年106岁<ref>{{Cite web|url=http://cpc.people.com.cn/n1/2018/0111/c87393-29757651.html|title=殷国洪同志逝世--新闻报道-人民网|accessdate=2018-01-16|last=1010|work=cpc.people.com.cn}}</ref>。殷国洪是最后一位逝世的红安县籍以及湖北籍的开国将军<ref name=pp/>。
