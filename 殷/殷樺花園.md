'''殷樺花園'''（{{lang-en|'''Blessings Garden'''}}）是[[麗新集團|麗新集團]]和[[新世界發展有限公司|新世界發展有限公司]]發展的一個住宅項目。一期位於[[羅便臣道|羅便臣道]]95號，二期位於[[干德道|干德道]]56號。<ref>{{cite web |url=http://hk.centadata.com/ptest.aspx?type=2&code=SKEBGPEXPE&info=tr&code2=&page=0 |title=殷樺花園的成交記錄 中原數據 |website=中原地產}}</ref>

==簡介==
殷樺花園位處[[香港|香港]][[西半山|西半山]]，鄰近[[雍景台|雍景台]]。有很多名人入住，如藝員[[薛家燕|薛家燕]]。該住宅有基本設施及充足的休閒空間，室外[[游泳池|游泳池]]、多用途[[運動場|運動場]]和平台花園等。大部份單位面積約1105平方呎，有3間睡房，2間浴室。由[[富城物業管理有限公司|富城物業管理有限公司]]管理，[[G4S|G4S]]負責24小時保安。<ref>{{cite web |url=http://data.28hse.com/datarecord384.html |title=殷樺花園屋苑資料及成交數據 |website=data.28hse.com 香港屋網}}</ref>

==銷售及租務情況==
殷樺花園交投暢旺，單月累錄4宗買賣個案。如三房單位建築面積1,106方呎，由用家出價1,237.5萬元承接，每呎售價折合11,189元。賣方折算15年間樓價累漲80%。
