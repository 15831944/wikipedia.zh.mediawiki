'''漩门湾大桥'''又称玉环漩门湾大桥，位于[[中华人民共和国|中国]][[浙江省|浙江省]][[台州市|台州市]][[玉环市|玉环市]]。是中国首座[[月牙|月牙形]]单塔空间索面[[钢|钢]][[混凝土|混]]组合梁斜拉桥。<ref>{{Cite web|url=http://news.sina.com.cn/o/2017-11-06/doc-ifynmvuq8879978.shtml|title=[浙江]国内首座独塔“月环”型斜拉钢桥——玉环漩门湾大桥开工建设|accessdate=2018-04-08|author=|date=2017-11-06|work=|publisher=新浪新闻|deadurl=no|archiveurl=https://web.archive.org/web/20180408145830/http://news.sina.com.cn/o/2017-11-06/doc-ifynmvuq8879978.shtml|archivedate=2018-04-08}}</ref>

== 规划 ==
2017年1月20日，漩门湾大桥及接线工程初步设计审查会召开，会议审查通过了工程初步设计。<ref>{{Cite web|url=http://yhnews.zjol.com.cn/yuhuan/system/2017/01/22/021017145.shtml|title=玉环漩门湾大桥及接线工程初步设计顺利通过审查|accessdate=2018-04-08|author=|date=2017-01-22|work=|publisher=玉环新闻网|deadurl=no|archiveurl=https://web.archive.org/web/20180408145829/http://yhnews.zjol.com.cn/yuhuan/system/2017/01/22/021017145.shtml|archivedate=2018-04-08}}</ref>

2017年7月31日，漩门湾大桥及接线工程施工图审查会召开，会议审查通过了工程施工图。<ref>{{Cite web|url=http://yhnews.zjol.com.cn/yuhuan/system/2017/08/02/030292635.shtml|title=漩门湾大桥及接线工程施工图通过审查|accessdate=2018-04-08|author=|date=2017-08-02|work=|publisher=玉环新闻网|deadurl=no|archiveurl=https://web.archive.org/web/20180408145828/http://yhnews.zjol.com.cn/yuhuan/system/2017/08/02/030292635.shtml|archivedate=2018-04-08}}</ref>

== 建设 ==
2017年10月12日，玉环市重大项目集中开工暨漩门湾大桥及接线项目开工仪式，在漩门湾大桥及接线项目工地隆重举行，这标志着工程正式开工建设。<ref>{{Cite web|url=http://yhnews.zjol.com.cn/yuhuan/system/2017/10/12/030444705.shtml|title=玉环举行重大项目集中开工仪式 9个重大项目总投资120亿元|accessdate=2018-04-08|author=|date=2017-10-12|work=|publisher=玉环新闻网|deadurl=no|archiveurl=https://web.archive.org/web/20180408145830/http://yhnews.zjol.com.cn/yuhuan/system/2017/10/12/030444705.shtml|archivedate=2018-04-08}}</ref>

2018年3月29日，漩门湾大桥施工栈桥开工建设，这标志着工程进入实质性施工阶段。<ref>{{Cite web|url=http://xxgk.yuhuan.gov.cn/yh016/04/06/201804/t20180403_607592.htm|title=玉环漩门湾大桥进入实质性建设阶段|accessdate=2018-04-08|author=|date=|work=|publisher=玉环市人民政府信息公开专栏|deadurl=no|archiveurl=https://web.archive.org/web/20180408145831/http://xxgk.yuhuan.gov.cn/yh016/04/06/201804/t20180403_607592.htm|archivedate=2018-04-08}}</ref>

== 融资 ==
2018年3月7日，玉环市漩门湾大桥及接线工程PPP投资协议签约仪式，在玉环市交通运输局举行。王庆飞、马建勇分别代表玉环市交通运输局、[[中交二公局|中交二公局]]签署了《漩门湾大桥PPP投资协议》。<ref>{{Cite web|url=http://www.shb2ccccc.com/show.aspx?did=2876|title=漩门湾大桥及接线工程PPP投资协议顺利签约|accessdate=2018-04-08|author=|date=2018-03-09|work=|publisher=中交二公局第二工程有限公司|deadurl=no|archiveurl=https://web.archive.org/web/20180408145829/http://www.shb2ccccc.com/show.aspx?did=2876|archivedate=2018-04-08}}</ref>

== 影响 ==
当玉环漩门湾大桥建成通车后，将会爆破拆除原先于1977年建设完成并承担临时道路通行功能的漩门大坝。[[玉环岛|玉环岛]]也将从半岛恢复至原本的海岛形态。<ref>{{Cite web|url=http://www.taizhou.com.cn/news/2017-10/13/content_3717859.htm|title=撤坝建桥，玉环跨进环漩门湾时代|accessdate=2018-04-08|author=|date=2017-10-13|work=中国台州网|publisher=|deadurl=no|archiveurl=https://web.archive.org/web/20180408145828/http://www.taizhou.com.cn/news/2017-10/13/content_3717859.htm|archivedate=2018-04-08}}</ref>这会使得乐清湾与漩门湾之间的水系得以连通，待开启漩门二期和三期工程的水闸，进行水体交换后，会使[[乐清湾|乐清湾]]的水质有所改善。<ref>{{Cite web|url=http://www.sohu.com/a/207600739_183282|title=玉环漩门湾将撤坝设桥 乐清湾水质有望好转|accessdate=2018-04-08|author=|date=2017-11-30|work=|publisher=搜狐|deadurl=no|archiveurl=https://web.archive.org/web/20180408145829/http://www.sohu.com/a/207600739_183282|archivedate=2018-04-08}}</ref>

*支线路基宽度：36.5m
