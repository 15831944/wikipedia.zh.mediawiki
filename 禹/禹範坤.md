'''禹範坤'''（{{lang|ko|'''우범곤'''}}，{{bd|1955年|2月24日|1982年|4月27日|catIdx=Woo}}），已故[[韓國|韓國]][[韓國警察|警察]]，是[[韓國|韓國]][[現代史|現代史]]上已知最重大的[[纵欲杀手|纵欲杀手]]案例，在該國[[慶尚南道|慶尚南道]][[宜寧郡|宜寧郡]]殺死了包括自己在內共57人（有其他說法為55、56或61人），另有35人受傷。

== 背景 ==
禹範坤是[[韓國海軍陸戰隊|韓國海軍陸戰隊]]退伍[[軍人|軍人]]，於1978年退伍在[[漢城警察廳|漢城警察廳]]擔任[[警員|警員]]，之後調往宜寧郡的偏遠村落。除了因為仕途受到挫折，亦因為經濟原因及準備與同居兩個月的女朋友結婚，而承受了巨大[[壓力|壓力]]。

== 犯案 ==
1982年4月26日當日，禹範坤是因為同居女朋友打了停在禹範坤胸上的蒼蠅，禹範坤與女朋友口角。他盛怒下離家，到了警察武器庫，飲了很多[[威士忌|威士忌]]。晚上9時30分，他飲得爛醉後，拿了兩支[[卡賓槍|卡賓槍]]、180發實彈和7枚[[手榴彈|手榴彈]]，首先闖到電話局殺死了3名[[接線生|接線生]]，中斷對外通訊。當時正是晚飯時間。他憑著警察的身份，進入了一間又一間房子，然後槍殺或者以[[手榴彈|手榴彈]]炸死居民，就此經過了8小時，直到27日凌晨。禹範坤在一個村殺害一定數目的人後，就到另外一個村莊繼續殺人。他一共在5個村莊殺人。

== 自殺 ==
第二日，韩国武裝警察將禹範坤趕到山上。4月27日凌晨5時30分，此時走進絕路的禹範坤將兩枚手榴彈綁在自己身上，抱住3個人質同歸於盡。

== 影響 ==
在這一起現職警察大量殺人案中，因為警察署署長遊玩不在現場而延遲出動，使得韓國國民強烈要求[[韓國總統|韓國總統]][[全斗煥|全斗煥]]和[[韓國國務總理|國務總理]]辭職。最後內務部部長{{Link-ko|徐廷和|蘇仲華 (1933年)}}辭職，由[[盧泰愚|盧泰愚]]繼任。
