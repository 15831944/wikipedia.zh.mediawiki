'''磁山文化'''，是[[中国|中国]][[华北地区|华北地区]]的早期[[新石器时代|新石器文化]]，得名于1933年在[[河北省|河北省]][[武安|武安县]]发现的[[磁山遗址|磁山遗址]]，年代约为距今8000年-7000年前，主要分布在太行山东麓、燕山山麓以南、黄河以北的低山丘陵和山前平原地区，即河北省<ref>董廣輝、劉峰文、楊誼時、王琳、陳發虎《黃河流域新石器文化的空間擴張及其影響因素》</ref>。

磁山文化下接[[仰韶文化|仰韶文化]]、龙山文化，组成[[中原|中原]]先民一个完整的發展链条<ref>{{Cite web|url=https://archive.fo/6ELHo|title=河北武安建成磁山文化博物馆 |accessdate=|deadurl=yes|archiveurl=http://www.people.com.cn/GB/wenhua/22219/2566855.html|archivedate=2018-7-1}}</ref>
== 遺址 ==
磁山文化主要分布在[[河北|河北]]南部和[[河南|河南]]北部，遗址比较密集的地区在武安县境内的[[洺河|洺河]]流域。除磁山遗址外，重要遗址还包括河北保定[[容城|容城]]坡上遗址、河南[[淇县|淇县]]花窝遗址等。
== 生活 ==
居民从事原始[[农业|农业]]，农作物以[[粟|粟]]为主。考古人员发现了较多的[[石镰|石镰]]、石铲、石刀、[[石斧|石斧]]，说明此时的农业生产已经进入“锄耕农业”阶段。磁山和[[花窝遗址|花窝遗址]]出土的[[猪|猪]]和[[狗|狗]]的骨骼为原始[[家畜|家畜]]饲养提供了证据。各遗址普遍出土的骨簇、鱼镖等狩猎工具说明[[渔猎|渔猎]]也是居民经济生活的重要辅助。制陶业较原始，处于手制阶段，火候不高，前期多夹砂[[褐陶|褐陶]]，纹饰有绳纹、编织纹、篦纹等，器形以陶盂和支座为主，亦有罐、杯、盘等，后期夹砂[[红陶|红陶]]和细泥红陶逐渐增多，三足钵数量较多，出现了与圆足罐等。[[石器|石器]]多为磨制，亦有打制。住房为圆形或椭圆形半地穴式建筑，早期无门道，后期出现阶梯式门道。储藏东西的窖穴发现较多，内有粮食堆积。

在磁山遗址中，发现80多个储粟窖穴，共约十万多斤<ref>[http://www.cma.gov.cn/kppd/kppdsytj/201701/t20170124_389936.html 磁山先民如何制历划节]</ref>。

磁山文化有三项“世界之最”，最早由人工培植粟类；最早饲养家鸡；最早種植核桃<ref>[http://www.ward.gov.cn/news_view.asp?newsid=2376 磁山文化遗址]</ref><ref>漢族上古文化與政權</ref>。
== 研究 ==
磁山文化与[[裴李岗文化|裴李岗文化]]关系密切，有人提出了“'''裴李岗·磁山文化'''”，将它们视为同一种文化类型。<ref>严文明：《黄河流域新石器时代早期文化的新发现》，《考古》1979年第5期，462-467页</ref>
也有学者认为它们是有一些共同特征的两种不同文化类型。<ref>李友谋等：《试论裴李岗文化》，《考古》1979年第4期，347-352页</ref>总体看来，裴李岗文化和磁山文化后期比较接近。
