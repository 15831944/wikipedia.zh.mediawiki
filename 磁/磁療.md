'''磁療'''（magnetotherapy），即所謂的磁場治療，是指涉及"磁場"，"磁力"等概念的[[偽科學|偽科學]]性質的[[替代医学|替代医学]]。例如宣稱[[磁力|磁力]]或[[磁場|磁場]]對人體有正面效果，可利用磁的影響达到身心保養或疾病治療的效果。

== 歷史 ==
在前2世紀的《[[神農本草經|神農本草經]]》有記載“磁石味艱，至周痹風濕，肢節腫痛不可持物”食用天然磁石以治療疾病。

== 食用安全性 ==

磁石攝取過量會導致[[鐵中毒|鐵中毒]]

== 療效 ==
磁疗产品的理论依据不见于任何一本医学院教科书中，磁疗产品不是医学研究的成果，其原理也不受科学理论支持。因此有人認為磁疗只是骗钱的东西<ref>[http://www.xys.org/xys/ebooks/others/science/dajia9/ciliao.txt 所有磁疗产品都是骗人的][[新语丝|新语丝]]网站，2008-11-10</ref>。科普作家[[方舟子|方舟子]]认为磁疗只是一种概念炒作，实际是一种[[伪科学|伪科学]]。<ref>[http://www.xys.org/xys/ebooks/others/science/dajia10/315d.txt 3·15晚会曝光好记忆学习枕 天津仍在出售]天津日报，2009-03-16，新语丝网站</ref>[[美国食品药品管理局|美国食品药品管理局]]没有批准任何磁疗产品用于医疗，同时也禁止厂家宣传磁疗产品具有医疗、保健作用。<ref>[http://www.xys.org/xys/netters/Fang-Zhouzi/science/magnetic.txt 磁疗真的能治病吗？]方舟子，《科学世界》2009.5.，2009-4-27造访</ref>
