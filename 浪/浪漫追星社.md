《'''浪漫追星社'''》（{{lang-ja|'''宙のまにまに'''}}）是日本漫畫家[[柏原麻實|柏原麻實]]所作的校園戀愛漫畫。漫畫於[[講談社|講談社]]的《[[月刊Afternoon|月刊Afternoon]]》連載。[[單行本|單行本]]全10卷。一般簡稱「{{lang|ja|宙まに}}」或「{{lang|ja|まにまに}}」。

==劇情簡介==
主角大八木朔，搬回童年時生活過的小鎮，並且希望能可以有個沉穩而又平靜的生活。但是，卻遇上了幼時玩伴——明野美星，並把他硬拉進天文部，於是天文部一段浪漫觀星故事就這樣開始了。

==登場人物==
=== 蒼榮高中 ===
==== 天文社 ====
;{{nihongo|大八木 朔|大八木 朔（おおやぎ さく）||[[聲優|聲]]：[[前野智昭|前野智昭]]（幼年：[[早見沙織|早見沙織]]）}}
:本作品的主角，苍荣高中一年级生。父親隻身在外地工作， 因此跟母親兩人在小杉野市生活。長相斯文，戴著一副眼鏡，個性溫和有禮。文學、讀書是他的嗜好。他跟美星是一對[[青梅竹馬|青梅竹馬]]，暱稱美星為「小美（{{lang|ja|みーちゃん}}）」。在幼時就常被美星拖到戶外活動，某次夜晚因為試圖救從樹上摔落的美星而造成手臂[[骨折|骨折]]。之後由於美星都不來探望他而對她相當不滿；直到重回原地上高中後才知道原來當時美星也受傷而消除誤會，重修舊好。因為美星時常跑來班上找他，因此被同學誤會兩人是在「交往」。「朔」一名取自日本作家[[荻原朔太郎|荻原朔太郎]]，因父母為其愛好者。
;{{nihongo|明野 美星|明野 美星（あけの みほし）||聲：[[伊藤加奈惠|伊藤加奈惠]]}}
:本作品女主角，苍荣高中二年级生。非常喜歡星星，為了能招收天文社成員，在學校頂樓灑星星宣傳，常常因關於星星的事情而出現暴走的情況。個性天真單純。對朔非常執著，暱稱朔為「小朔（{{lang|ja|朔ちゃん}}）」。把學生會長－小文視為敵人。對星空異常熟悉，被稱為人肉[[旋轉星圖|星圖]]。當升上三年級後，成為了天文社社長。名稱疑似取自星空保育區[[岡山縣|岡山縣]]西南部的美星町（2005年併入[[井原市|井原市]]）。
;{{nihongo|矢来 小夜|矢来 小夜（やらい さよ）||聲：[[早見沙織|早見沙織]]}}
:苍荣高中二年级生，美星的同學，小學時的好朋友。受到美星的影響，也喜歡星星。
;{{nihongo|蒔田 姬|蒔田 姫（まきた ひめ）||聲：[[戶松遥|戶松遥]]}}
:苍荣高中一年级生，朔的同班同學。對朔有強烈的好感。頭髪會自然捲起來，在小學時被大家稱為「爆炸頭」，但到高中已經擺脫這個綽號。視美星為情敵並稱她為「老女人」。
;{{nihongo|江戶川 正志|江戸川 正志（えどがわ まさし）||聲：[[高木礼子|高木礼子]]}}
:苍荣高中一年级生，朔的同班同學，參加攝影社，受到小夜的美貌所吸引，所以幾乎常常出現天文部的活動中。
;{{nihongo|路萬 健康|路万 健康（（ろま たけやす）||聲：[[間島淳司|間島淳司]]}}

==主要舞台==
;{{nihongo|蒼榮高中|蒼栄高校（そうえいこうこう）||}}
:朔和美星一起就讀的高中。有很多積極的學生，且文化祭非常令人興奮。
;{{nihongo|小杉野市兒童天文科學館|小杉野市こども宇宙科学館（こすぎのしこどもうちゅうかがくかん）||}}
:在天文館，由晴子帶大家解説天文館內。館內設置了20釐米反射式的天文[[望遠鏡|望遠鏡]]。
:[[埼玉市|埼玉市]]青少年科學館為範本。
;{{nihongo|蒼流荘|蒼流荘（そうりゅうそう）||}}
:蒼榮高中所有的合宿所，位在湖泊的東北方，養老兼具時尚度假風建築。
:文藝部也在此合宿。
;{{nihongo|蒼流館|蒼流館（そうりゅうかん）||}}
:蒼榮高中擁有的合宿所。位於蒼流荘的旁邊。

==科學考察==
作者柏原麻實本人也是天文愛好者<ref>http://mmk.typepad.jp/1999/2009/07/post-a122.html </ref>。本作出現的天文知識和天象，有高度的正確性，在[[ACG|ACG]]作品中實屬罕見。例如所有[[星座|星座]]都是按真實繪製的，而且位置準確。[[銀河|銀河]]華麗地重現等等。

出現的星座與季節吻合。OP中，當眾人身穿夏裝，[[分鏡|分鏡]]出現的是[[天蠍座|天蠍座]]等夏季星座，還有夏季銀河。當眾人穿上厚衣，他們看到的是[[獵戶座|獵戶座]]和[[大犬座|大犬座]]等冬季星座。

幾乎每一話都有天文教學情節，例如第一話就教了用雙筒望遠鏡投影太陽影像，第二話有辨認春季星座的情節，第三話的天文問答比賽就出現了[[仙后座|仙后座]]的星座連線，到了第四話就教了天文攝影的基礎方法[[固定攝影|固定攝影]]<ref>{{cite web |url=http://www.lcsd.gov.hk/CE/Museum/Space/StarShine/AstroPhoto/c_starshine_astrophoto2.htm |title=存档副本 |accessdate=2009-07-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20090802071523/http://www.lcsd.gov.hk/CE/Museum/Space/StarShine/AstroPhoto/c_starshine_astrophoto2.htm |archivedate=2009-08-02 }}</ref>。這些全都是現實中天文愛好者會用到的技巧，絕非虛構。

另外，在[[手電筒|手電筒]]前套上紅布，這些都是真實存在的觀星禮儀<ref>{{cite web |url=http://www.lcsd.gov.hk/CE/Museum/Space/StarShine/Stargazing/c_starshine_stargazing_manner.htm |title=存档副本 |accessdate=2009-07-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20090725012638/http://www.lcsd.gov.hk/CE/Museum/Space/StarShine/Stargazing/c_starshine_stargazing_manner.htm |archivedate=2009-07-25 }}</ref>。到郊外觀星要慎防蚊蟲，觀星期間遇上壞天氣，這些全都是經驗之談。

第三話美星因為看不到星，導致情緒低落，都是[[天文愛好者|天文愛好者]]的通病。

===天文部器材===
本作出現的器材，全都是根據實物繪製，是現實中不少[[天文愛好者|天文愛好者]]的標準裝備。
*雙筒望遠鏡: VIXEN Ascot ZR7×50WP（7倍50mm口徑，這種規格的雙筒望遠鏡，曾被廣泛認為是入門必備的器材，觀星三寶之一。）
*[[旋轉星圖|旋轉星圖]]: 在日本被稱為「星座早見」<ref>{{cite web |author1=渡辺教具製作所 製 |title=「星座早見」 |url=http://www.astroarts.co.jp/shop/showcase/ct-wata1/index-j.shtml |publisher=AstroArts |accessdate=2018-10-15}}</ref>，另一入門必備的器材，觀星三寶之二。
*紅光手電筒: 臨時套上紅色布料，觀星必備的器材，觀星三寶之三。 (由於業餘天文在亞洲並不普及，很多最基本的器材都無法買到，只可以通過自製獲得。)
*星圖: 標準星圖，實物為[[地人書館|地人書館]]出版的的《標準星圖2000》<ref>{{cite web |author1=Deep-Sky Wonders |title=趣味の天文/参考文献 |url=http://www.geocities.jp/okita_tenmon/bibliography/bibliography.html |publisher=地人書館 |accessdate=2018-10-15}}</ref>，在日本天文愛好者間廣泛流傳。
*照相機: 「Nihon」 FM3A[[單鏡反光機|單鏡反光機]]，實際上是從攝影部借來的。[[影射|影射]][[尼康|尼康]]（Nikon）的同型號相機。

===贊助商、協力===
*[[VIXEN|VIXEN]]：現實中存在的望遠鏡生產商，在日本國內市場佔有率達60%。天文部的器材全都是VIXEN產品，例如美星手中的雙筒望遠鏡，就是現實中真實存在的產品，型號為Ascot ZR7×50WP<ref>[http://www.vixen.co.jp/product/bino/ascot/156207.htm Ascot ZR7×50WP 双眼鏡・単眼鏡]</ref>。
*《{{lang|ja|月刊星ナビ}}》：現實中真存在的天文雜誌，天文部的讀物，幾乎每一話都會出現這本雜誌<ref>{{cite web |author1=AstroArts |title=月刊星ナビ |url=http://www.astroarts.co.jp/hoshinavi/index-j.shtml |accessdate=2018-10-15}}</ref>。而現實中的《{{lang|ja|月刊星ナビ}}》曾為這部作品的改編動畫宣傳，例如在2009年7月7日的專稿。<ref>{{cite web |author1=AstroArts |title=アニメ『宙のまにまに』　七夕にオンエア |url=http://www.astroarts.co.jp/news/2009/07/07soramani/index-j.shtml |accessdate=2018-10-15}}</ref>
