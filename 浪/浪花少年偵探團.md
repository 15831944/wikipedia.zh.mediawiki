《'''浪花少年偵探團'''》是[[日本|日本]]作家[[東野圭吾|東野圭吾]]的[[長篇小說|長篇]][[推理小說|推理小說]]。1988年12月[[講談社|講談社]]發行了[[單行本|單行本]]，1991年11月發行[[文庫版|文庫版]]，2011年12月發行了[[文庫本|文庫本]]新裝版。

續集《'''浪花少年偵探團2'''》也於1993年12月發售[[單行本|單行本]]，1996年發行[[文庫本|文庫本]]改名{{nihongo|《'''再見了，忍老師 浪花少年偵探團獨立篇'''》|しのぶセンセにサヨナラ 浪花少年探偵団独立編|romaji=Shinobu Senseni Sayonara 　Naniwa Shounen Tantei Dan Dokuritsu Hen}}，2011年發行[[文庫本|文庫本]]新封面時縮短書名為{{nihongo|《'''再見了，忍老師'''》|しのぶセンセにサヨナラ}}。

故事講述被稱為「浪花少年偵探團」的小學生們和他們的班主任老師：竹內忍的故事。以本作為原作的電視劇於2000年在[[NHK|NHK]]教育頻道的「連續劇愛之詩」時段播放，由[[山田瑪麗亞|山田瑪麗亞]]主演。2012年7月由[[多部未華子|多部未華子]]主演的新版電視劇將於[[TBS系列|TBS系列]]的「Panasonic Drama Theater」時段播放。

; {{nihongo|竹內 忍|{{ruby|竹內|たけうち}} {{ruby|忍|しのぶ}}|Takeuchi Shinobu}}
: 生野區·大路小學教師，6年5班級任導師。短期大學畢業後，為了童年的夢想~成為老師進入大路小學赴任。
: 家庭組成有職任家電製造廠廠長的父親：茂三、母親：妙子和OL的妹妹在。

; {{nihongo|田中 鐵平|{{ruby|田中|たなか}} {{ruby|鉄平|てっぺい}}|Tanaka Teppei}}
: 大路小學6年5班→國中2年級的學生。到處炫耀擁有[[FC游戏机|紅白機]]，向朋友自誇耀對遊戲攻略之能力。

; {{nihongo|原田 郁夫|{{ruby|原田|はらだ}} {{ruby|郁夫|いくお}}|Harada Ikuo}}
: 大路小學6年5班→國中2年級的學生。鐵平的好友，總是與鐵平一起行動。

; {{nihongo|畑中 弘|{{ruby|畑中|はたなか}} {{ruby|弘|ひろし}}|Hatanaka Hiroshi}}
: 大路小學6年5班→國中2年級的學生。常常與鐵平·原田一起行動。

; {{nihongo|新藤|{{ruby|新藤|しんどう}}|Shindou}}
: 大阪府捜査一課刑事警察。初次登場時是明年將成為30歲這樣的年齡。被那個鐵平他們戲稱「鰣魚」。

; {{nihongo|漆崎|{{ruby|漆崎|うるしさき}}|Urushisaki}}
: 大阪府捜査一課刑事警察，與夥伴新藤組成優良的搜查組合。雖然總是被鐵平他們揶揄，但是身為刑警的洞察力頗高，會站在不同的角度思考，也經常接近案件的真相。

; {{nihongo|本間 義彦|{{ruby|本間|ほんま}} {{ruby|義彦|よしひこ}}|Honma Yoshihiko}}
: 任職於豐中市產業機械的大製造廠的子公司「K工業」，幹部候選的傑出人物。初次登場時是28歲。

; {{nihongo|竹內 妙子|{{ruby|竹內|たけうち}} {{ruby|妙子|たえこ}}|Takeuchi Taeko}}
: 竹內忍的母親。親切、多嘴多舌的性格。

; {{nihongo|村井|{{ruby|村井|うるしさき}}|Urushisaki}}
: 大阪府捜査一課警部。新藤、漆崎的上司，禿頭。

; {{nihongo|中田|{{ruby|中田|うるしさき}}|Urushisaki}}
: 大路小學的副校長。被學生們譏取「ゼロジュウ」這個綽號
