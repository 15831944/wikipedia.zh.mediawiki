'''殺青'''原本指爲便於着墨書寫，去除青竹表面光滑的青皮的工序，從而製成[[竹簡|竹簡]]，後來引申爲「完工」的意思，比如文學作品完稿，或者電影製作完成。

== 製簡 ==
用火炙烤竹簡的表面，使油脂浮出表面，因爲像流汗一樣，所以後來以「汗青」來代表竹簡。然後將表面的油和青皮刮取，即製成竹簡。

== 製茶 ==
「殺青」是[[製茶|製茶]]工藝的重要工序之一，在[[綠茶|綠茶]]、[[黃茶|黃茶]]及[[黑茶|黑茶]]的製程中都有，[[紅茶|紅茶]]及[[白茶|白茶]]則無，[[青茶|青茶]]的[[炒青|炒青]]工序則部份功能相同。

從製茶工艺角度看，殺青是以高溫令[[茶葉|茶葉]]的[[氧化酶|氧化酶]]停止作用，免除酵素發酵茶叶的[[茶多酚|茶多酚]]，使茶葉保持茶多酚的应有的色素。使茶葉水分消散、變軟，便于揉捻成形。不同的製茶工艺可以製出不同品味的茗茶。綠茶因此而得名。但黃茶與黑茶的殺青，則在停止茶葉的酵素作用後，添加人工或天然的菌種來進行發酵，產生特定的風味。青茶的[[炒青|炒青]]，則具有輕微發酵與停止發酵兩種效果。

殺青有乾熱法及濕熱法；前者導熱方式為金屬導熱、空氣導熱，後者為蒸氣導熱。金屬者稱為'''釜炒'''，[[五雜俎|五雜俎]]「揉而焙之」謂'''[[炒青|炒青]]'''，可能是明朝才開始的<ref>{{Cite web|url=http://www.books.com.tw/products/0010724824|title=擺一桌絕妙的宋朝茶席|accessdate=2018-08-10|work=博客來}}</ref>。陽光照射、空氣者為'''烘青'''（'''曬青'''），蒸氣者為'''蒸青'''。[[中國|中國]]自古已有此三法，宋朝[[點茶|點茶]]採用蒸青，現今主流為釜炒，少數特色茶用蒸青；[[日本|日本]]主流為蒸青，少數用[[釜|釜]]炒。工業化生產的綠茶，大多以熱風機作空氣導熱殺青。

值得指出的是，高级茗茶，都是经过多道的特殊杀青工艺，杀掉茶多酚的某些活性素，或者是保留某些活性素，并进行比例控制而得到的高级品茗。

== 影劇 ==
電影或電視劇的「殺青」指拍攝工作完畢，最後一幕拍攝的片段又稱「殺青戲」。在有些地方，拍完殺青戲後會由工作人員或[[製作人|製作人]]向[[演員|演員]]及[[導演|導演]]等人獻花以感謝拍攝期間的辛勞。拍攝完成之後即進入[[後製|後製]]階段。
