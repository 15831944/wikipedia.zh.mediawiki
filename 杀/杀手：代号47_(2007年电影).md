《'''杀手：代号47'''》（{{lang-en|'''''Hitman'''''}}）是一部2007年法國、美國、英國合拍的動作[[驚悚|驚悚]]電影，由{{link-en|泽维尔·吉恩斯|Xavier Gens}}執導，改编自热门同名游戏《[[刺客任務|刺客任務]]》系列。主演是[[蒂莫西·奥利芬特|蒂莫西·奥利芬特]]、[[欧嘉·柯瑞兰寇|欧嘉·柯瑞兰寇]]和[[多格雷·斯科特|多格雷·斯科特]]。雖然評價不好，但票房成功。

其續集被取消，但是[[二十世紀福斯|二十世紀福斯]]宣佈開發重啟版電影，命名為《[[刺客特攻47|刺客特攻47]]》（''Hitman: Agent 47''），由[[羅拔·費特|羅拔·費特]]飾演殺-{}-手47，於2015年8月21日上映。<ref>{{cite news |language = en |author = Mike Fleming Jr |url = http://deadline.com/2013/02/fox-reloading-hitman-with-paul-walker-as-bald-barcoded-assassin-agent-47-422594/ |title = Fox Reloading ‘Hitman’ With Paul Walker As Bald Barcoded Assassin Agent 47 |publisher = Deadline Hollywood |date = 2013-02-05}}</ref>

== 剧情 ==
讲述[[殺手47|殺-{}-手47]]为了追寻自己的本来面目和身世而和蘇俄KGB斗智斗勇的故事。<ref>{{cite web |url= http://www.pcpop.com/doc/0/267/267097.shtml|title= 杀手:代号47 出炉 本周五部电影推荐|accessdate= 2008年1月28日 |publisher=泡泡网 |work= |language =zh-cn}}</ref>

== 花絮 ==
蒂莫西·奥利芬特扮演的代号47剃成光头，显示出阴柔与冷峻的气质，符合游戏中的原版人物特质。<ref>{{cite web |url=http://www.pcpop.com/doc/0/582/582516_6.shtml |title=《生化4》首映 游戏改编电影谁最给力 |accessdate=2010年10月12日 |publisher=泡泡网 |work= |language =zh-cn}}</ref>
