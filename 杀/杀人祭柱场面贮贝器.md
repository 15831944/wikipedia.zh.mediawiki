'''杀人祭柱场面贮贝器'''，中国[[西汉|西汉]]时期文物，[[禁止出境展览文物|禁止出境展览文物]]<ref name=cnnew>{{cite web |title=94件一级文物列入"第三批禁止出境展览文物目录" |url=http://www.chinanews.com/cul/2013/08-19/5180173.shtml |website=中国新闻网 |accessdate=2018-07-14 |archive-url=https://web.archive.org/web/20180714164315/http://www.chinanews.com/cul/2013/08-19/5180173.shtml |archive-date=2018-07-14 |dead-url=no }}</ref>，1955年于[[云南|云南]][[晋宁|晋宁]][[石寨山古墓群|石寨山]]甲区一号墓出土<ref name="孙太初1956">{{cite journal |author1=孙太初 |title=云南晋宁石寨山古遗址及墓葬 |journal=考古学报 |date=1956 |issue=1 |pages=43-63}}</ref><ref name="李来玉">{{cite web |author1=李来玉 |title=杀人祭柱场面贮贝器——禁止出国（境）展览文物 |url=http://www.kaogu.cn/cn/kaoguyuandi/kaogubaike/2015/0618/50605.html |website=中国考古网}}</ref>。器身呈四耳[[铜鼓|铜鼓]]形，器盖上铸刻有古滇国杀人祭柱的祭祀场面。出土时器身稍有残缺，器内盛有[[贝币|贝币]]<ref name="孙太初1956" />。现藏于[[云南省博物馆|云南省博物馆]]<ref name=cnnew/>。

== 出土 ==
[[云南省博物馆|云南省博物馆]]从1950年代开始在该省全省范围内进行的科学考古发掘、试掘、调查<ref>{{cite web |title=云南省博物馆 |url=http://www.china.com.cn/book/zhuanti/1bowuguan/2008-11/11/content_16746569.htm |website=中国网 |accessdate=2018-07-14 |archive-url=https://web.archive.org/web/20180714135242/http://www.china.com.cn/book/zhuanti/1bowuguan/2008-11/11/content_16746569.htm |archive-date=2018-07-14 |dead-url=no }}</ref>。1955年，在[[云南省|云南省]][[昆明市|昆明市]][[晋宁县|晋宁县]]的石寨山西面坡上发掘了新石器时代遗址一处，获得大量陶器和石器。1955年至1960年间，先后四次在石寨山山麓发掘了[[春秋战国|春秋战国]]至[[东汉|东汉]]时期的墓葬48座，出土文物四千多件<ref>{{cite web |author1=白寿彝主编 |title=第八节 汉代少数族地区的考古资料 |website=中国通史}}</ref>。杀人祭柱场面贮贝器于1955年出土<ref name="李来玉" />，在发掘报告《云南晋宁石寨山古遗址及墓葬》中被称为“鼓形四耳器”<ref name="孙太初1956" />{{rp|55}}。2013年8月19日，杀人祭柱场面贮贝器被[[国家文物局|国家文物局]]收录到公布的《第三批禁止出境展览文物目录》中，同样在[[石寨山古墓群|石寨山古墓群]]出土的[[诅盟场面贮贝器|诅盟场面贮贝器]]和[[滇王之印|“滇王之印”金印]]也是禁止出境展览文物<ref name=cnnew/>。

== 形制 ==
[[贮贝器|贮贝器]]是[[滇国|滇国]]特有的贮放[[贝币|贝币]]的青铜器，器上铸造出姿态各异的群像，具体而生动地表现了祭祀、战争、狩猎、祈年、播种等情形，是研究滇国社会历史的重要资料<ref>{{cite web |author1=陈成军 |title=古滇国的外汇 |url=http://www.chnmuseum.cn/tabid/438/InfoID/78565/frtid/285/Default.aspx |website=中国国家博物馆 |accessdate=2018-07-14 }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

杀人祭柱场面贮贝器呈[[铜鼓|铜鼓]]形，通高38厘米，有底有盖，盖直径30厘米。贮贝器腰部有用阴线浅刻的狩猎图案，为八个裸体人物，胯下系宽带，上束至腰际，手持兵器作追逐野猪兽状。器盖上表现的是滇国的一次祭祀活动，共铸有各种人物52人，猪1头，狗1条。器盖两侧边沿各置一小铜鼓，中央立有一根高9厘米的圆柱，圆柱中段有两条[[蛇|蛇]]盘绕，顶端有一头[[老虎|老虎]]，底座横卧一条[[鳄鱼|鳄鱼]]。圆柱的右侧竖立一牌，牌上有一名双臂反缚的裸体男子，其发辫系于牌后。牌右前方坐一个左足锁于木枷中的人，一个反缚双手跪于地上的裸体人，三人应为用来祭祀的[[牺牲|牺牲]]。铜柱后方是四排齐整列坐的妇女，每排三至四人，前两列妇女中立一名肩披巾，双手抚于胸前，作观望状的男子。列坐者的左侧另有四名妇女，膝前放置有盛放鱼、肉等物的篮子，四名妇女前有一名伫立观望状的男子。另有一个位乘坐四人[[肩舆|肩舆]]的贵族妇女，可能是祭祀仪式的主祭人。肩舆侧面有两名女子，后方跟随一名男子，应当是主祭人的随从。在肩舆者附近另有受刑者、执斧行刑者若干人。<ref name="孙太初1956" /><ref name="李来玉" /><ref name="翟路">{{cite web |author1=翟路 |title=西汉　杀人祭柱场面铜贮贝器 |url=http://www.gg-art.com/profesional/index.php?detailid=977 |website=中华博物}}</ref>

对于器盖雕铸内容的[[宗教|宗教]]含义，学界有多种不同解读。[[孙太初|孙太初]]在发掘报告中推测其可能是祭祀仪式，而中央立柱可能是信仰[[图腾|图腾]]<ref name="孙太初1956" />。[[冯汉骥|冯汉骥]]则认为器盖所表现者是“报祭”，也就是“丰收祭”或“收获祭”，柱后列坐女子筐中或手持的物品，为新收获的农作物，铜柱可能是代表“[[社_(宗教)|社]]”所在的地方或者是“社”的象征<ref name="冯汉骥1963">{{cite journal |author1=冯汉骥 |title=云南晋宁石寨山出土铜器研究——若干主要人物活动图像试释 |journal=考古 |date=1963 |issue=6 |pages=319-329}}</ref>。[[方国瑜|方国瑜]]同意冯说，也认为器盖图像所表现者的是祭铜柱，属于一种与农事有关的丰收祭典<ref name="彝族史稿">{{cite book |author1=方国瑜 |title=彝族史稿 |date=1984-03 |publisher=四川民族出版社 |pages=78-79 |edition=第1版}}</ref>。张增祺在其《长江上游璀灿的滇国青铜文化》一文中认为，该贮贝器上所铸“杀人祭铜柱”场面的情景“与越人视铜柱为神物的习俗大致相同”，体现了滇人受古代越人崇拜铜柱的影响，视铜柱为神物风俗<ref name="张增祺1994">{{cite journal |author1=张增祺 |title=长江上游璀灿的滇国青铜文化 |journal=东南文化 |date=1994 |issue=4 |pages=23-36}}</ref>。易学钟则认为盖上的整个场面和人群活动“与《[[儀禮|仪礼]]·聘礼》及其‘三礼’记载的‘聘享’和‘夫人之聘享’祭仪相合”，赞同被缚者是“祭祀的牺牲”，却认为这是“刑于市”或“戮于市”的再现<ref name="易学钟1988">{{cite journal |author1=易学钟 |title=晋宁石寨山1号墓贮贝器上人物雕像考释 |journal=考古学报 |date=1988 |issue=1 |pages=37-49}}</ref>。<ref name="邓沛2006">{{cite journal |author1=邓沛 |title=试析云南少数民族"祭柱"习俗的宗教含义 |journal=青海师专学报 |date=2006 |volume=26 |issue=1 |pages=57-58}}</ref>
