'''純貞孝皇后'''尹氏（{{kor|k=순정효황후|hanja=純貞孝皇后}}，{{bd|1894年|9月19日|1966年|2月3日}}），為[[大韓帝國|大韓帝國]]第二任[[皇帝|皇帝]][[朝鮮純宗|純宗]]繼-{后}-，又称'''尹妃'''（{{kor|k=윤비|hanja=尹妃}}）。[[本贯|本贯]][[海平尹氏|海平尹氏]]，1894年9月19日（旧历[[开国_(朝鲜高宗)|开国]]五百三年甲午八月二十日）出生于[[朝鲜|朝鲜]][[京畿道|京畿道]][[楊平郡|杨根郡]]西宗面汶湖里。她的父亲是[[海丰|海丰]][[府院君|府院君]][[尹泽荣|尹泽荣]]，母亲是庆兴府夫人俞氏，她还是曾任[[大韩帝国内阁总理大臣|大韩帝国议政大臣]]的[[尹容善|尹容善]]的曾孙女。

== 生平 ==
=== 拣择储妃 ===

1904年11月，当时[[大韩帝国|大韩帝国]][[皇太子妃|皇太子妃]]闵氏（后追封[[纯明孝皇后|纯明孝皇后]]）去世，在为她服完丧事以后，另立[[太子妃|太子妃]]提上了[[大韩帝国|大韩帝国]]皇室的议事日程。1906年3月开始决定拣择新任太子妃，时年13岁<small>（虚岁）</small>的尹氏参选。1906年（光武十年）7月4日举行初拣择，同年9月22日行再拣择，[[尹泽荣|尹泽荣]]之女、[[沈钟灿|沈钟灿]]之女和[[成健镐|成健镐]]之女入选。此后过了3个月，即12月31日才举行三拣择，最终尹氏被拣择为[[皇太子妃|皇太子妃]]。这次拣择仪式空前持久，拣择日期反复推迟。1907年1月24日，[[皇太子|皇太子]][[李坧|李坧]]与尹氏行嘉礼于安国洞别宫，同日还在[[德寿宫|庆运宫]]举行了册妃礼和朝见礼。<ref>尹氏拣择及嘉礼过程参见《光武十年十二月嘉礼都监仪轨》（奎13183-1-2）。</ref>就这样，尹氏正式成为了[[大韩帝国|大韩帝国]]'''东宫继妃'''。尹氏的[[太子妃|太子妃]]册封的背后其实隐藏着政治意图，当时许多史料记载表明，她父亲[[尹泽荣|尹泽荣]]的推动和高宗[[纯献皇贵妃|纯献皇贵妃]]严氏的点头是尹氏能成为太子妃的主要原因（尹氏曾祖父[[尹容善|尹容善]]是[[纯献皇贵妃|严贵妃]]心腹大臣）。<ref>黄玹《梅泉野录》，第404页：“十二月，聘皇太子妃尹氏，秘书丞泽荣之女也，时年十四。预选者多，而上以泽荣为[[尹斗寿|斗寿]]之后，门地赫舄，不嫌其年幼而拣定焉。即外议藉藉以泽荣纳巨赂于[[纯献皇贵妃|严妃]]，得其斡旋之力也。”</ref><ref>郑乔《大韩季年史》下，第223页：“十二月，立太子妃尹氏。妃，前议政容善孙泽荣之女，泽荣兄德荣贪富贵，恳于严贵妃而被选。”</ref>而[[日本|日本]]的[[韩国统监府|韩国统监府]]的官方资料更是明确指出：

{{Quote|尹泽荣系尹容善之孙，为人奸佞，交际巧妙，善解人意，依其祖父之前导，开始结纳庆善宫（严妃），先参与升妃运动，后又参与升-{后}-运动，颇运密计，获得了严妃很深的信任。后来他的女儿（指纯贞孝皇后尹氏）被封为皇后，实成于严妃之主张，以酬其劳，且益亲密其关系，故有此为（指册封尹氏）。<ref>《统监府文书》卷8，{{lang|ja|韓官人ノ經歷一般}}，第218页。</ref>}}

=== 大韩皇后 ===

1907年7月，[[朝鲜高宗|高宗李熙]]退位，皇太子[[李坧|李坧]]即[[皇帝|皇帝]]位，是为[[朝鲜纯宗|纯宗]]，[[皇太子妃|皇太子妃]]尹氏随之成为[[皇后|皇后]]。她便成为继[[明成皇后|明成皇后]]闵氏的新一任中殿，同时也是[[大韩帝国|大韩帝国]]唯一一位生前做过的[[皇后|皇后]]的人。由于纯宗没有生育能力，尹氏自然无法为其诞下子女，因此在1907年（隆熙元年）9月立[[纯献皇贵妃|严贵妃]]之子英亲王[[李垠|李垠]]为[[皇太子|皇太子]]。同年11月，尹氏随[[朝鲜纯宗|纯宗皇帝]]由[[德寿宫|庆运宫]]搬往[[昌德宫|昌德宫]]居住。尹氏做[[皇后|皇后]]期间，曾入女学，并设置女侍讲，她还曾亲自下旨振兴女子教育。<ref>[http://sillok.history.go.kr/inspection/insp_king.jsp?id=wzb_10105020_002&tabid=w 《纯宗实录》卷2，隆熙二年5月20日条。]</ref>后来尹皇后还在宫中学习[[日语|日语]]。<ref>[http://gonews.kinds.or.kr/OLD_NEWS_IMG3/HSS/HSS19100427u00_02.pdf 《皇城新闻》，隆熙四年4月27日，“{{lang|ko|皇后陛下의 語敎師}}”。] {{webarchive|url=https://web.archive.org/web/20160306101646/http://gonews.kinds.or.kr/OLD_NEWS_IMG3/HSS/HSS19100427u00_02.pdf |date=2016-03-06 }}</ref><ref>[http://gonews.kinds.or.kr/OLD_NEWS_IMG3/HSS/HSS19100630u00_02.pdf 《皇城新闻》，隆熙四年5月24日，“{{lang|ko|語學日進}}”。] {{webarchive|url=https://web.archive.org/web/20160306150210/http://gonews.kinds.or.kr/OLD_NEWS_IMG3/HSS/HSS19100630u00_02.pdf |date=2016-03-06 }}</ref>尽管如此，尹氏入宫后必须循规蹈矩，而且和[[朝鲜纯宗|纯宗]]没有正常的家庭生活，其苦恼可想而知。据说，她进入[[昌德宫|昌德宫]]以后20多年间从未迈出宫门一步，一年之中只有一两次在昌德宫花园里散步而已。后来[[李垠|李垠]]之妻[[李方子|李方子]]在写回忆录时称“这真是一种囚徒的生活”。<ref name="{{cite web |url= http://news.china.com/zh_cn/history/photos/11025921/20050531/12359948_9.html|title= 朝鲜绝唱：弱国灭亡和末代王室的命运(图集)下(10)|publisher=中华网 |work= |language =zh}}">{{cite web |url= http://news.china.com/zh_cn/history/photos/11025921/20050531/12359948_9.html|title= 朝鲜绝唱：弱国灭亡和末代王室的命运(图集)下(10)|publisher=中华网 |work= |language =zh}}</ref>

1910年8月22日，[[日本|日本]]强迫[[大韩帝国|大韩帝国]]签订《[[日韩合并条约|日韩合并条约]]》，正式将[[朝鲜半岛|朝鲜半岛]]变为[[日本|日本]]的[[殖民地|殖民地]]。8月29日，[[朝鲜纯宗|纯宗]]颁布《让与诏书》，《[[日韩合并条约|日韩合并条约]]》生效，[[大韩帝国|大韩帝国]]灭亡。据说在[[朝鲜纯宗|纯宗]]颁布诏书前，尹皇后曾躲在屏风后面听到[[李完用|李完用]]、[[尹德荣|尹德荣]]与纯宗商讨退位事宜，她便出来阻止并把[[玉玺|玉玺]]藏起来，后来玉玺还是被她的伯父[[尹德荣|尹德荣]]强行夺走。<ref name="{{cite web |url= http://inews.ewha.ac.kr/news/articleView.html?idxno=17455|title= 女性与权力(6)大韩帝国的末代皇后——纯贞孝皇后|publisher=梨大学报 |work= |language =ko}}">{{cite web |url= http://inews.ewha.ac.kr/news/articleView.html?idxno=17455|title= 女性与权力(6)大韩帝国的末代皇后——纯贞孝皇后|publisher=梨大学报 |work= |language =ko}}</ref>[[大韩帝国|大韩帝国]]灭亡后，[[韩国皇室|韩国皇室]]被列入[[日本皇室|日本皇室]]中，尹皇后降格为'''李王妃'''，因此她又被称为“'''尹妃'''”。

=== 殖民时期 ===
[[File:Imperial_family_of_Korea_02.jpg|thumb]]
1926年[[朝鲜纯宗|纯宗]]逝世，尹妃按制守丧。直到葬仪结束，她不进食，不梳洗，整日呆在寝宫抽泣。1926年4月27日[[李垠|李垠]]依制继位为“李王”，尹妃的名号升为尹大妃。纯宗病逝后一个月，尹大妃拒绝[[李垠|李垠]]等人的劝说，立即从她所居的[[大造殿|大造殿]]搬出，住进[[昌德宫|昌德宫]][[乐善斋|乐善斋]]。她说，新王践祚必须住进王宫，这是几百年的礼法，也是王室的尊严。从此尹氏成为了[[李王家|李王家]]的最高长辈。[[朝鲜纯宗|纯宗]]去世这年，尹大妃33岁<small>（虚岁）</small>。在此后漫长的寂寞岁月里，她守护着[[乐善斋|乐善斋]]，同老年宫女们闲谈一些无关紧要的琐事，便成为她打发时日的主要方式。<ref name="{{cite web |url= http://news.china.com/zh_cn/history/photos/11025921/20050531/12359948_9.html|title= 朝鲜绝唱：弱国灭亡和末代王室的命运(图集)下(10)|publisher=中华网 |work= |language =zh}}"></ref>

=== 光复以后 ===

1945年[[日本投降|日本投降]]以后，[[朝鲜半岛|朝鲜半岛]]光复，[[李承晚|李承晚]]等[[韩国|韩国]]政府领导人对待以尹大妃为首的旧皇室极为冷淡，尹氏继续在[[乐善斋|乐善斋]]艰苦度日。1950年[[朝鲜战争|朝鲜战争]]爆发，[[朝鲜人民军|朝鲜人民军]]打进[[漢城|漢城]]后，尹氏被从乐善斋带出，关进了拘留所。据说在面对[[人民军|人民军]]战士时，尹氏曾大声呵斥道：“这里是国母居住的地方！”这个故事在[[韩国|韩国]]传为佳话。<ref name="{{cite web |url= http://inews.ewha.ac.kr/news/articleView.html?idxno=17455|title= 女性与权力(6)大韩帝国的末代皇后——纯贞孝皇后|publisher=梨大学报 |work= |language =ko}}"></ref>五天以后，她被释放，但乐善斋的旧物已荡然无存。她无处栖身，在[[昌德宫|昌德宫]]门前大街上徘徊。后来被人发现，将其送往[[兴宣大院君|兴宣大院君]]故居[[云岘宫|云岘宫]]暂时安身。[[联合国军|联合国军]]收复[[首尔|首尔]]后，尹氏再次流浪街头，后来随美军到[[釜山|釜山]]避难，1952年回到了乐善斋。但是，1953年停战以后，以“国父”自居的韩国总统[[李承晚|李承晚]]不希望旧皇室来分他的威望，于是在1954年颁布了《[[旧王室财产处置法|旧王室财产处置法]]》，[[韩国政府|韩国政府]]依据该法将[[昌德宫|昌德宫]]、[[乐善斋|乐善斋]]收为国有，尹氏被撵出宫廷，再次流离失所。她被人送到了[[首尔|首尔]]郊区[[贞陵|贞陵]]的在一家农舍（修仁斋）里居住。<ref name="{{cite web |url= http://inews.ewha.ac.kr/news/articleView.html?idxno=17455|title= 女性与权力(6)大韩帝国的末代皇后——纯贞孝皇后|publisher=梨大学报 |work= |language =ko}}"></ref><ref name="{{cite web |url= http://news.china.com/zh_cn/history/photos/11025921/20050531/12359948_10.html|title= 朝鲜绝唱：弱国灭亡和末代王室的命运(图集)下(11)|publisher=中华网 |work= |language =zh}}">{{cite web |url= http://news.china.com/zh_cn/history/photos/11025921/20050531/12359948_10.html|title= 朝鲜绝唱：弱国灭亡和末代王室的命运(图集)下(11)|publisher=中华网 |work= |language =zh}}</ref>

1960年4月，一向对[[朝鲜王朝|朝鲜王朝]][[王室|王室]]持敌视态度的[[李承晚|李承晚]]倒台，流亡[[夏威夷|夏威夷]]。5月4日，业已67岁的尹大妃被接回[[首尔|首尔]]，重新住进了[[乐善斋|乐善斋]]。新上台的[[民主黨_(大韓民國1955年)|韩国民主党]]政权一位名叫[[吴在璟|吴在璟]]（오재경）的干练官员被任命为“[[旧王室事务总局|旧王室事务总局]]”局长。此人工作很认真，据说连一本书也不放过，皆登录在册。经他追查发现，旧王室庞大的动产和不动产中，被韩国当权官吏侵吞变卖的状况触目惊心。他指示手下人把这一切都整理出来。刚想下一步如何处理时，他的办公室被人放了一把火，所有账目和取证材料被焚烧殆尽。[[吴在璟|吴在璟]]知道有人要坚决阻止他查下去，于是便在1961年挂冠而去。<ref name="{{cite web |url= http://news.china.com/zh_cn/history/photos/11025921/20050531/12359948_10.html|title= 朝鲜绝唱：弱国灭亡和末代王室的命运(图集)下(11)|publisher=中华网 |work= |language =zh}}"></ref>

旧王室的财产被人侵吞，尹氏作为王室最高长辈已无能为力。而且，从法理上说这些财产皆已属国家所有，尹大妃的生活是按《旧王室财产处置法》规定，国家每月拨付给她50万元生活费维持的。后来上台的韩国新任领导人[[朴正熙|朴正熙]]善待旧皇室，尹氏生活才有所改观。<ref name="{{cite web |url= http://news.china.com/zh_cn/history/photos/11025921/20050531/12359948_10.html|title= 朝鲜绝唱：弱国灭亡和末代王室的命运(图集)下(11)|publisher=中华网 |work= |language =zh}}"></ref>

1966年2月3日，尹氏於[[昌德宮|昌德宮]][[乐善斋|乐善斋]]病逝，得年73歲<small>（虚岁）</small>，“[[全州李氏大同宗约院|全州李氏大同宗约院]]”为其上[[諡號|諡號]]為'''獻儀慈仁純貞孝皇后'''，葬[[裕陵|裕陵]]。

== 信仰 ==

纯贞孝皇后尹氏晚年皈依[[佛教|佛教]]，取[[法号|法号]]“'''大地月'''”。
