'''純潔鈍綏蟎'''（''Amblyseius bellatulus''）為[[植綏蟎科|植綏蟎科]]{{Le|鈍綏蟎屬|Amblyseius}}下的一個種。

== 物種描述 ==
雌蟲背板（Dorsal shield）光滑，外側有網狀紋路。背毛（setae）19對，柔順，除z4、z5有些微鋸齒。背板上可見七對孔（solenostomes），[[氣門溝|氣門溝]]（Peritreme）於延伸逾j1位置。胸板有三對綱毛，毛旁有小孔。腹肛板成五角形，上有三對肛前毛，每對毛旁皆有小孔。[[螯肢|螯肢]]上動趾十齒，定趾三齒。{{Le|受精囊|Spermatheca}}盞成漏斗狀。第四對足有三對刺毛（macrosetae），第二對足膝節有七條綱毛<ref name=":0">{{Cite journal|title=Amblyseius bellatulus Tseng (Acari: Phytoseiidae): neotype designation with first description of a male|url=http://www1.montpellier.inra.fr/CBGP/acarologia/article.php?id=4157|last=Liao|first=J.-R.|last2=Ho|first2=C.-C.|journal=Acarologia|issue=2|doi=10.1051/acarologia/20164157|volume=57|pages=323–335|language=en|issn=0044-586X|last3=Ko|first3=C.-C.}}</ref>。

雄蟲腹肛板有三對肛前毛，[[擔精趾|擔精趾]]（Spermatodactyl）成U狀<ref name=":0" />。

== 分布及棲地 ==
本種目前僅知分布於[[台灣|台灣]]<ref name=":0" />。

== 發現過程 ==
本種最早由台灣蟎類學家[[曾義雄|曾義雄]]發現，並將[[正模式標本|正模式標本]]存放於標準檢驗局台南分局，然而模式標本之後卻遺失。後於2017年由廖治榮重新指定[[新模式標本|新模式標本]]<ref name=":0" />。
