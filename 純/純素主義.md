'''純蔬食主義'''（{{lang-en|Veganism}}）是一種哲學，是一種生活方式，志在盡可能排除對動物的所有剝削虐待，不論形式，不論目的，包括食用、做衫等等。

== 词语释意 ==
[[File:Greek_grocery_vegetables.JPG|thumb]]
'''纯蔬食主义者'''的英语单词 vegan 由 vegetarian（'''[[蔬食主义|蔬食主义者]]'''）衍生而成。1944年，Elsie Shrigley 和 Donald Watson 创立了[[完全蔬食協會_(英國)|英國纯蔬食協會]]，而当时仅有的蔬食者词汇 vegetarianism，尚不排除[[乳制品|乳制品]]的食用，故纯蔬食者 vegan 这个词诞生了。这个词由 '''veg'''etari'''an''' 开头的三个字母 veg 加上末尾的两个字母 an 组成，这样表示 vegan 源自于 vegetarian，但表述地更加明确。有了 vegan 这个词，就可以将严格避免所有动物制品的人，跟仅仅不吃肉的人区别开来。有些纯蔬食者不喜欢将 vegan 这个词用作名词，认为其应该作为形容词。比如，他们认为 "he is a vegan" 就表达不对，而应该是 "he is vegan"。

== 动物产品 ==
动物产品包含所有形式的[[肉|肉]]、[[禽|禽]]、[[鱼|鱼]]、[[贝|贝]]、[[蛋|蛋]]、[[乳制品|乳制品]]、[[毛皮|毛皮]]、[[皮革|皮革]]、[[毛料|毛料]]、[[丝|丝]]，和副产品例如[[明胶|明胶]]、[[凝乳|凝乳]]、[[乳清|乳清]]和类似物。纯蔬食社团和大多数 Vegan 在他们的定义中也包含[[昆虫|昆虫]]产品例如[[蜂蜜|蜂蜜]]。关于规定什么是动物产品点有一些争论。一些純蔬食者避免使用骨碳过滤的[[蔗糖|蔗糖]]，而一些不会食用[[蛋青|蛋青]]、动物[[血|血]]（现在极为少见），或[[鱼胶|鱼胶]]（甚至不会出现在最终产品中）过滤的[[啤酒|啤酒]]或[[葡萄酒|葡萄酒]]。

更进一步，如果那个[[锅|锅]]曾经烹调过肉类，某些纯蔬食者不会食用那个锅做的食物；而普通純蔬食者認為只要食物沒有肉类、鱼类、蛋类和乳类已經足夠。

對於是否應該殺死[[蒼蠅|蒼蠅]]和[[蚊子|蚊子]]，純蔬食者們則在為此而争论。<ref>[https://metro.co.uk/2017/12/27/really-vegan-willingly-kill-mosquitoes-fleas-flies-7186562/ Are you really a vegan if you willingly kill mosquitoes, fleas and flies?]</ref>

== 动机 ==
纯蔬食者定位他们的主要动机为减少动物的[[痛苦|痛苦]]的概念，来源于[[功利主义|功利主义]][[哲学|哲学]]，被例如[[杰里米·边沁|杰里米·边沁]]和[[彼得·辛格|彼得·辛格]]等作者表达为，纯蔬食主义是[[信仰|信仰]]，[[人类|人类]]有[[道德|道德]]义务避免引起任何其他生物苦难。动物与人类一样有固有的[[权利|权利]]过尽可能免于痛苦的生活。因此纯蔬食者不仅避免食用肉类和乳制品，也避免使用任何生产过程中包含动物苦难的产品。根据某人的信奉级别，这可能包括不使用某些[[药物|药物]]，因为它们为安全原因在动物身上试用。一些抱有强硬态度的则避免购买明胶制成的电影胶片而购买数字电影代替。虽然总有纯蔬食者社团为此类问题争论。纯蔬食主义的全部目的是最大可能范围的减少动物的痛苦。为此原因，许多纯蔬食者也是动物权利运动的支持者。

一个发表于时代杂志2002年7月7日的时代/CNN[[民意测验|民意测验]]发现，4%的[[美国|美国]]人认为自己是蔬食主义者，并且自认为蔬食主义者中的5%认为自己是纯蔬食者。这个小样本测验可以指出千分之二的美国人，即大约50万美国人是纯蔬食者。一个2000年的民意测验显示接近0.9%美国人可能是纯蔬食者，总人数大概为250万。在[[英国|英国]]，研究显示在2001年0.4%人口，大约250,000人是纯蔬食者。

'''饮食纯蔬食主义'''（{{lang-en|Dietary veganism}}）是指那些因为[[健康|健康]]原因（例如[[胆固醇|胆固醇]]）吃没有动物产品的食物，而不是由于同情动物。他们因为健康原因避免食用动物产品，但是会买新的动物皮革[[皮鞋|皮鞋]]。

'''道德纯蔬食主义'''（{{lang-en|Ethical veganism}}）的动机仅仅只是为了帮助动物。

纯蔬食主义哲学与梵语文化中'''[[不害|不害]]'''（[[IAST|IAST]]：Ahiṃsā）的观念关系很近。不害代表了不[[杀害|杀害]]和不伤害。不害理念被[[圣雄甘地|圣雄甘地]]教导给他的追随者。美国纯蔬食者社团网站说: “这不只是被动顺从，而是主动面对日常生活的困境和抉择的方法。在西方世界，我们称之为动态不害。”不害理念同时尊重[[生命|生命]]，尊重[[思想|思想]]、[[言语|言语]]和[[行为|行为]]的正直，为人类自然和万物服务。

== 可能的问题 ==
根据美国饮食协会关于蔬食的建议书，纯蔬食者食用各种各样的食物有充足的营养来源，但是纯蔬食者应该注意[[维生素B12|维生素B12]]的摄入。有很多迹象显示假若消耗足够多的包含B12的食物获得足够水平的B12是不困难的，这包括许多类型的[[豆奶|豆奶]]、[[酵母|酵母]]提取物、早餐谷类，和肉类类似物。纯蔬食者通常比非純蔬食者有更低的[[钙|钙]]摄入，但是也有更低的钙需求。纯蔬食者和其他蔬食主义者也应该注意[[Ω-3脂肪酸|Ω-3脂肪酸]]的摄入。现代国家的大多数人都没有得到足够的这种营养，所以这不是一个纯蔬食者食物的本质问题，但是鱼类比植物被发现含有更多长链Ω-3脂肪酸。所以纯蔬食者应该在他们的食物中包含足够的Ω-3脂肪酸来源，例如亚麻籽、胡桃和暗绿蔬菜。

== 益处 ==

纯蔬食可以降低很多[[健康|健康]]问题的风险，包括[[心血管疾病|心血管疾病]]、[[肥胖|肥胖]]、[[糖尿病|糖尿病]]、[[哮喘|哮喘]]、[[高血压|高血压]]和[[癌症|癌症]]。纯蔬食也有益于环境，同时可能改善[[第三世界|第三世界]]国家或其他国家的低收入人群的处境，通过释放更多的食物供人类消费。
