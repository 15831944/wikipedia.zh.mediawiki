==人名簡述==
史书没有记载[[宣太后|芈八子]]（秦宣太后）的本名和早年事迹，“芈月”一名，来自[[段熙仲|段熙仲]]教授对[[秦始皇兵马俑|秦始皇兵马俑]]身上的“脾”字的解读，段熙仲认为这是两个独体字，即左边的为“月”，右边则是“芈”。陈景元进一步解读，认为“月芈”即是秦宣太后的本名<ref>{{Cite web |url=http://news.ifeng.com/a/20151210/46600068_0.shtml |title=兵马俑是为芈月修造的？ 别笑，专家研究几十年 |accessdate=2015-12-21 |author= |date=2015-12-10|work= |publisher=凤凰网，来源：重庆晚报|language=Zh-hans }}</ref>。有报道称<ref>{{Cite web |url=http://xm.ifeng.com/baoliao/detail_2015_12/04/4626748_0.shtml|title=探访芈月墓 周围现神秘圆坑究竟是做什么的？|accessdate=2015-12-21 |author= |date=2015-12-04|work= |publisher=凤凰网，来源：观察者网综合|language=Zh-hans }}</ref>，除秦始皇兵马俑残字的说法外，“芈月”亦有来自[[阿房宫|阿房宫]]筒瓦上秦惠文王妃子“芈月”合体陶文的说法，同时有报道否认这两处来源的说法<ref>{{Cite web|url=http://difang.kaiwind.com/shaanxi/sxwswg/201512/04/t20151204_3205202.shtml|title=兵马俑、阿房宫的主人是芈月吗?|accessdate=2015-12-21|author=作者：宋丽娜|date=2015-12-04|work=|publisher=凯风网，来源：阳光报|language=Zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20151222101345/http://difang.kaiwind.com/shaanxi/sxwswg/201512/04/t20151204_3205202.shtml|archivedate=2015-12-22}}</ref>。

== 剧情概述 ==
講述中國歷史上第一位太后（[[秦宣太后|秦宣太后]]）极为曲折传奇的人生故事。

戰國時期，羋月（[[孫儷|孫儷]] 飾）是楚威王最寵愛的小公主，但在楚威王（[[趙文瑄|趙文瑄]] 飾）出戰之時地位一落千丈，母親向氏（[[孫茜|孫茜]] 飾）被楚威-{后}-（[[姜宏波|姜宏波]] 飾）逐出宮，多年後返回宮中以命復仇，養母莒姬（[[蔣欣|蔣欣]] 飾）也為了保護兩姐弟而自盡。羋月和羋戎（[[李泓良|李泓良]] 飾）處處受到楚威-{后}-的逼迫及同父異母的姐姐羋茵（[[徐梵溪|徐梵溪]] 飾）陷害。

羋月與楚公子黃歇（[[黃軒|黃軒]] 飾）青梅竹馬，真心相愛，為了可以與黃歇順利私奔，故自願作為嫡公主羋姝（[[劉濤|劉濤]] 飾）的陪嫁媵侍遠嫁秦國，在往秦國的路上羋月、羋姝互相扶持，中間楚國禮車在往秦國路上被義渠王翟驪（[[高雲翔|高雲翔]] 飾）帶領的軍隊強劫，黃歇因為要救羋月，而跌落山谷生死未詳，心灰意冷的羋月，為了要找出幕後主使而陪羋姝進入秦宮，羋姝當上了秦國的王后，羋月因魏夫人（[[馬蘇|馬蘇]] 飾）陷害，將其同母異父的弟弟魏冉（[[張鈞涵|張鈞涵]] 飾）綁架，不得已求助秦王成為寵妃。原本的姐妹之情在羋月生下兒子嬴稷（[[朱一龍|朱一龍]] 飾）後漸漸分裂。諸子爭位，秦王嬴駟（[[方中信|方中信]] 飾）抱憾而亡。羋月和兒子被發配到遙遠的燕國。不料秦武王嬴蕩（[[巴圖|巴圖]] 飾）舉鼎而亡，秦國大亂。羋月借義渠軍力回到秦國，平定了秦國內亂。羋月兒子嬴稷登基為王，史稱秦昭襄王。羋月當上了史上第一個太后，史稱秦宣太后。
