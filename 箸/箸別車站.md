'''箸別站'''（{{jpn|j=箸別駅|hg=はしべつえき|rm=Hashibetsu eki|f=y}}）是位於[[北海道|北海道]][[增毛郡|增毛郡]][[增毛町|增毛町]]箸別，隸屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[留萌本線|留萌本線]]，已廢除的[[鐵路車站|鐵路車站]]。電報略號為'''ハヘ'''。

名稱源自[[阿伊努語|阿伊努語]]的「pas-pet」（煤炭的河流）或「has-pet」（有灌木的河流）<ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/101110P.pdf|title=アイヌ語地名リスト|accessdate=2017-08-17|author=北海道庁|date=|publisher=北海道庁}}</ref>。

== 歷史 ==
* 1963年（昭和38年）12月1日 - [[日本國有鐵道|日本國有鐵道]]留萌本線[[舍熊車站|舍熊車站]]至[[增毛車站|增毛車站]]之間，開設'''箸別臨時車站'''（{{lang|ja|仮乗降場}}）。只處理旅客業務。
* 1987年（昭和62年）4月1日 - 由於[[國鐵分割民營化|國鐵分割民營化]]，車站由[[北海道旅客鐵道|JR北海道]]管理，並升格為'''箸別站'''。
* 1990年（平成2年）3月10日 - 制定營業距離。
* 2016年（平成28年）12月5日 - 由於[[留萌車站|留萌車站]]至增毛站路段被廢除，本站正式廢站<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2016/160428-3.pdf |language=ja |title=留萌線（留萌・増毛間）の鉄道事業廃止届の提出について |publisher=[[北海道旅客鐵道|JR北海道]] |date=2016-04-28 |accessdate=2017-06-13 |archive-url=https://web.archive.org/web/20160429052355/http://www.jrhokkaido.co.jp/press/2016/160428-3.pdf |archive-date=2016-04-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20160429052355/http://www.jrhokkaido.co.jp/press/2016/160428-3.pdf |archivedate=2016-04-29 }}</ref><ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2016/160628-1.pdf |language=ja |title=留萌線（留萌・増毛間）の廃止日繰上げの届出について |publisher=[[北海道旅客鐵道|JR北海道]] |date=2016-06-28 |accessdate=2017-06-13 |archive-url=https://web.archive.org/web/20160701083826/http://www.jrhokkaido.co.jp/press/2016/160628-1.pdf |archive-date=2016-07-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20160701083826/http://www.jrhokkaido.co.jp/press/2016/160628-1.pdf |archivedate=2016-07-01 }}</ref>。

== 車站構造 ==
廢站時為1面1線的地面車站。木製的[[側式月台|側式月台]]位於路軌的北方（增毛方向的右方）。沒有設置[[轉轍器|轉轍器]]的棒線車站。

由成為臨時車站開始已經是無人車站，月台的東方設有候車室。候車室外觀與朱文別站及阿分站相似，但內部有使用混凝土進行加工。沒有廁所，深川方向的斜路連接著車站外的道路。

== 載客量 ==
* 1992年的每天載客量為8人，每天上下車乘客量為16人。
* 2011年至2015年度平均每天載客量少於1人<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |language=ja |title=平成26年度 線区別の収支状況等について |publisher=JR北海道 |date=2016-02-10 |accessdate=2017-06-21 |archive-url=https://web.archive.org/web/20170407133949/http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |archive-date=2017-04-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20170407133949/http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |archivedate=2017-04-07 }}</ref>。
