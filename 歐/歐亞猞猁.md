'''歐亞猞猁'''（[[学名|学名]]：''{{lang|la|Lynx lynx}}''; [[汉语拼音|汉语拼音]]：Ōu-Yà shēlì，[[注音符號|注音符號]]：ㄡ ㄧㄚˋ ㄕㄜˋ ㄌㄧˋ） 是一种产于[[歐洲|歐洲]]和[[西伯利亞|西伯利亞]]森林的中型[[猫科|猫科]]动物，一般直接称猞猁。

== 形态 ==
猞猁通常重达40公斤左右，体长85-105厘米，尾长20-31厘米。体毛为灰色或红灰色，常具有黑斑。四肢粗长；耳直立，尖端有黑色毛丛；尾端黑色。

== 习性 ==
猞猁栖息于多岩石的森林中，主要在夜间活动，成年猞猁为独居动物。猞猁的叫声极少能够被觉察到。猞猁以[[野兔|野兔]]、[[啮齿类|啮齿类]]、[[狐|狐]]、[[麅|麅]]甚至[[驯鹿|驯鹿]]为食。

== 保护现状 ==
[[File:Lynx_lynx-4.JPG|thumb]]
猞猁在歐洲一度非常常见，但20世纪中以后在[[中歐|中歐]]和[[西歐|西歐]]已经越来越少能够见到，其数量有急剧下降的趋势。

== 亞种 ==
* ''Lynx lynx lynx''：[[北歐|北歐]]、中歐、[[东歐|东歐]]以及西伯利亞西部；
* ''Lynx lynx dinniki''：[[高加索|高加索]]山脉；
* ''Lynx lynx isabellinus''：[[中亞|中亞]]地区；
* ''Lynx lynx koslowi''：西伯利亞中部；
* ''Lynx lynx sardiniae''：[[撒丁岛|撒丁岛]]，已绝灭；
* ''Lynx lynx stroganovi''：俄罗斯远东地区；
* ''Lynx lynx wrangelli''：西伯利亞东部；
* ''Lynx lynx martinoi''：[[巴尔干|巴尔干]]半岛；
