《'''歐洲'''》，或称《'''歐羅巴'''》（[[阿爾巴尼亞語|阿爾巴尼亞語]]：'''Evropa'''；[[塞爾維亞語|塞爾維亞語]]：'''{{lang|sr|Европа}}'''），是[[科索沃共和國|科索沃共和國]]的[[國歌|國歌]]<ref>[http://www.novinite.com/view_news.php?id=94057 "Kosovo Parliament Adopts National Hymn"] ''novinite.com'' 2008-06-11 Link accessed 11/06/08</ref>由著名作曲家[[赫迪·门迪伊奇|赫迪·门迪伊奇]]（Mendi Mengjiqi）作曲，目前樂曲尚未填詞。<ref>[http://en.rian.ru/world/20080611/110143221.html "Kosovo parliament approves new national anthem"] ''rian.ru'' 2008-06-11 Link accessed 11/06/08</ref><ref>[http://news.bbc.co.uk/1/hi/world/europe/7447583.stm "Kosovo MPs choose national anthem"] ''bbc.co.uk'' 2008-06-11 Link accessed 11/06/08</ref>因其很好地體現了科索沃的多民族元素在2008年6月11日被科索沃議會選定為國歌。<ref>[http://www.redorbit.com/news/entertainment/1426772/kosovo_gets_national_anthem/ "Kosovo Gets National Anthem"] ''redorbit.com'' 2008-06-11 Link accessed 11/06/08</ref>

== 國歌遴選 ==
[[科索沃獨立|科索沃獨立]]后，國歌尚未確定。之前科索沃曾經有多個國歌提議，包括《[[歡樂頌|歡樂頌]]》<ref>{{cite web |url = http://www.usatoday.com/news/world/2008-02-17-kosovo-independence_N.htm |title = Kosovo declares independence - USATODAY.com |publisher = USA Today |date =2008-02-17 |accessdate =2008-03-12 }}</ref>、《团结在我们的国旗的周围》（[[阿爾巴尼亞|阿爾巴尼亞]]國歌）<ref>{{cite web |url = http://news.bbc.co.uk/2/hi/europe/6718105.stm |title = BBC NEWS |publisher =BBC |date =2007-06-04 |accessdate = 2008-03-11 }}</ref>和《Kur ka ra kushtrimi n'Kosovë》<ref>{{cite web |url = http://www.trepca.net/2002/12/021223-himni.kosoves-kur.ka.ra.kushtrimi.nKosove.htm |title =Himni i Kosovës |publisher = Trepca.net |date = 2002-12-23 |accessdate = 2008-03-16 }}</ref>。 

2008年3月12日，[[科索沃議會|科索沃議會]]宣布組成由總理[[哈辛·塔奇|哈辛·塔奇]]親自領導的國歌遴選工作組負責挑選國歌<ref>{{cite web |url = http://news.bbc.co.uk/2/hi/europe/7292240.stm |title=BBC NEWS |publisher=BBC |date=2008-03-12 |accessdate=2008-03-12 }}</ref>。國歌的要求包括：各國旋律或者音樂片段不得出自其他作品，長度不得短于30秒或者長于1分鐘，應當反映出多民族的因素等。最終勝出者可獲者10,000[[歐元|歐元]]的獎金<ref name="assembly">[http://www.b92.net/eng/news/politics-article.php?yyyy=2008&mm=06&dd=11&nav_id=51011 "Assembly approves Kosovo anthem"] {{webarchive|url=https://web.archive.org/web/20110215123402/http://www.b92.net/eng/news/politics-article.php?yyyy=2008&mm=06&dd=11&nav_id=51011 |date=2011-02-15 }} ''b92.net'' 2008-06-11 Link accessed 11/06/08</ref>。 

6月11日，科索沃議會以72票贊成、15票反對和5票棄權，通過《歐羅巴洲》為科索沃國歌<ref name="assembly" />。
