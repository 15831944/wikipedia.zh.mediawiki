'''歐洲柏大戟'''（''Euphorbia cyparissias''）是[[大戟屬|大戟屬]]的[[植物|植物]]，生長於[[歐洲|歐洲]]及於1860年代引入到[[北美洲|北美洲]]作為[[觀賞植物|觀賞植物]]。

== 分佈 ==
歐洲柏大戟生長在沙丘、岸邊及草原。於北美洲，它們被種植在路邊乾旱及多石的[[土壤|土壤]]、草地及草坪中。它們較為喜歡開放的地區生長，而非[[森林|森林]]或田地。

== 特性 ==
[[File:Euphorbia_cyparissias_bloemen.jpg|thumb]]
[[File:Euphorbia_cyparissias_bgiu.jpg|thumb]]
歐洲柏大戟以[[種子|種子]]及地下的[[根|根]]來繁殖。它們一般高10-80厘米。其[[苞葉|苞葉]]像[[花瓣|花瓣]]的，呈綠黃色，於5月至8月成熟，會變成紫色或紅色。歐洲柏大戟的[[果實|果實]]在成熟後會爆開，將種子散播達至16呎遠的地方。歐洲柏大戟亦會從側根繁殖，可以散佈得很快及很密。它的[[葉子|葉子]]很特別，細小及幼長，長2-4厘米及闊1-2毫米。

== 毒性 ==
歐洲柏大戟若被折斷，會分泌出一種乳狀的樹液。傳說這種樹液可以治癒[[疣|疣]]。有指這些樹液是有毒的，並會刺激[[皮膚|皮膚]]。樹液亦可能會刺激[[眼睛|眼睛]]及胃腸管。
  
歐洲柏大戟在很多地方，如[[美國|美國]][[科羅拉多州|科羅拉多州]]，都被指是有害的植物，會影響其他當地植物的生長。它們相信是對[[牛|牛]]及[[馬|馬]]有害，但對[[羊|羊]]則沒有害。
