'''歐豪年'''{{BD|1935年||||}}，[[嶺南畫派|嶺南畫派]]畫家，创作题材与形式丰富多样。<ref>{{cite web |title=欧豪年：还携笔墨向高丘 |url=http://art.people.com.cn/n1/2018/0316/c226026-29871821.html |website=人民网 |accessdate=2018-08-16}}</ref>

==生平==
歐豪年1935年出生於廣東，17歲師事從趙少昂，畫派屬於[[嶺南畫派|嶺南畫派]]<ref name="王儒哲">{{Cite news|url=http://www.chinatimes.com/newspapers/20140926002979-260113|title=豪氣萬千歐豪年80水墨特展|author=王儒哲|language=zh-tw|publisher=《中國時報》|date=2014-09-26|accessdate=2018-08-14}}</ref>。1950年，全家移居[[香港|香港]]，1970年赴[[台湾|台湾]]执教[[中国文化大学|中国文化大学]]并定居台湾。<ref>{{cite web |title=“鹏翼展天宽——欧豪年书画展”在我馆开幕 |url=http://www.chnmuseum.cn/tabid/1834/InfoID/94653/frtid/92/Default.aspx |website=中国国家博物馆 |accessdate=2018-08-16}}</ref>1993年获法国巴黎大宫美术馆特别奖誉，为第一个获此奖的中国画家。<ref>{{cite web |title=台湾著名画家欧豪年：曾获巴黎大宫博物馆特别奖 |url=http://jx.people.com.cn/n/2015/0120/c355202-23621447.html |website=人民网 |accessdate=2018-08-16}}</ref>
2016年6月获[[國立東華大学|國立東華大学]]荣誉文学博士学位<ref>{{cite web |title=歐豪年教授獲頒東華大學名譽文學博士學位
 |url=https://www.ndhu.edu.tw/files/14-1000-100149,r2390-1.php?Lang=zh-tw |website= 國立東華大學|accessdate=2018-08-19}}</ref>，同年11月获[[香港中文大学|香港中文大学]]荣誉文学博士学位。<ref>{{cite web |title=香港中文大学颁授荣誉博士学位予三位杰出人士 |url=https://www.cpr.cuhk.edu.hk/sc/press_detail.php?id=2401&t=%E9%A6%99%E6%B8%AF%E4%B8%AD%E6%96%87%E5%A4%A7%E5%AD%A6%E9%A2%81%E6%8E%88%E8%8D%A3%E8%AA%89%E5%8D%9A%E5%A3%AB%E5%AD%A6%E4%BD%8D%E4%BA%88%E4%B8%89%E4%BD%8D%E6%9D%B0%E5%87%BA%E4%BA%BA%E5%A3%AB&id=2401&t=%E9%A6%99%E6%B8%AF%E4%B8%AD%E6%96%87%E5%A4%A7%E5%AD%A6%E9%A2%81%E6%8E%88%E8%8D%A3%E8%AA%89%E5%8D%9A%E5%A3%AB%E5%AD%A6%E4%BD%8D%E4%BA%88%E4%B8%89%E4%BD%8D%E6%9D%B0%E5%87%BA%E4%BA%BA%E5%A3%AB |website=香港中文大学传讯及公共关系处 |accessdate=2018-08-16}}</ref>
