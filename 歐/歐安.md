'''歐安'''，遊戲代號'''Ainio.X.L'''，[[現場直播|遊戲實況主]]、[[電子競技|電子競技]]、《[[英雄聯盟|英雄聯盟]]》前中國大陸女子戰隊職業選手。現任[[Twitch|Twitch]]遊戲直播實況主、無經紀合約。出生於[[臺灣|臺灣]][[台南市|台南市]][[安平區|安平區]]，現居於[[臺灣|臺灣]][[新北市|新北市]][[新店區|新店區]]。

== 生平 ==
出生於單親家庭，有一對相差十歲的兄姊，國中畢業離家，在2014年憑藉競技鬥塔類型的[[英雄聯盟|英雄聯盟]]，踏入了電競圈。

== 過往經歷 ==
* 2016-02-01 《[[iG電子競技俱樂部|iG電子競技俱樂部]]》、《VG電子競技俱樂部》、《DR電子競技俱樂部》擔任英雄聯盟女子職業選手
* 2016-05-01 北京電影學院學生，參與電競題材「電競高校」女配角
* 2017-01-20 《GGirls team》女子實況團體、開始Twitch直播<ref>{{Cite web |url =https://tw.appledaily.com/new/realtime/20170122/1041002/|title =實況女團G Girls成立 |author =|date =2017年01月22日 |publisher =|language =zh-tw }}</ref>
* 2017年加入 [[ahq_e-Sports_Club|ahq e-Sports Club]] 成為旗下傳說對決戰隊隊經理<ref>{{Cite web |url =https://gnn.gamer.com.tw/2/151292.html|title =ahq 戰隊宣布加入三位隊經理 |author =|date =2017年08月17日 |publisher =|language =zh-tw }}</ref>
* 2018 由[[中華民國電子競技運動協會|中華民國電子競技運動協會]]負責的[[國際電子競技聯合會|國際電子競技聯合會]]《iESF世界電競錦標賽》擔任中華台北隊領隊經理。<ref>{{Cite web |url =https://gnn.gamer.com.tw/8/170888.html|title =蔡英文總統出席 2018 IESF 世界電競錦標賽 |author =|date =2018年11月11日 |publisher =|language =zh-tw }}</ref>
