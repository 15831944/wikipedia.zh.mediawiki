'''歐崇敬'''{{BD|1967年|3月7日|||}}，出生於[[台南市|台南市]]<ref>{{cite web |url=http://bulletin.cec.gov.tw/CECData/004/101/062/1.pdf |title=第8屆立法委員選舉選舉公報 |publisher=臺南市選舉委員會 |accessdate=2015-08-23 |language=zh-tw |deadurl=yes |archiveurl=https://web.archive.org/web/20141102124149/http://bulletin.cec.gov.tw/CECData/004/101/062/1.pdf |archivedate=2014-11-02 }}</ref>，現任[[中州科技大學|中州科技大學]]講座教授{{r|introduction}}、民權時報總編輯、中華兩岸事務交流協會副會長，曾任教于[[佛光大学|佛光大学]]、[[南华大学|南华大学]]、[[环球科技大学|环球科技大学]]，也从事过商业杂志总编、房地产杂志总监、财经台主持人兼顾问、文创、音乐教学等多种工作。

欧崇敬积极参与政治活動，从2008年起，三次争取參與选举。2011年受國民黨徵招參選台南市立委；2015年，欧崇敬由親民黨提名参选新竹市選區立法委員。
<br>

==经历==
欧崇敬自辅仁大学日语系获学士学位，政治大学获法学硕士，[[中国文化大学|中国文化大学]]哲学所获博士学位。他从2000年到2004年在[[佛光人文社会学院|佛光人文社会学院]]任助理教授，2005到2009年在[[南华大学|南华大学]]任副教授，2009年到2015年在[[环球科技大学|环球科技大学]]任副教授，从2015年开始在[[中州科技大学|中州科技大学]]任副校长，2017年8月獲聘為該校講座教授。其研究与专长包括中国哲学史、应用伦理学、未来与趋势、创意、管理策略、谈判策略等<ref>{{cite web |url=http://www.ccut.edu.tw/adminSection/front/showContent.asp?m_id=2&site_id=ocj |title=學歷}}</ref>。

欧崇敬还从事过商业杂志总编、房地產雜誌總監、財經台主持人兼顧問、文創、音樂教學等多樣化工作。欧崇敬称“從求學到就入不同職場，他一路上都不時檢視自己，因應趨勢做定位的調整”{{r|1111ad}}。

==从政经历==
欧崇敬从2008年起，曾经三次参加选举或者争取参加选举。歐於17岁求學時加入中國國民黨。2008年歐在大學執教時，3月22日在民主進步黨敗於總統選舉日，加入民进党，表態争取2009年[[嘉义市|嘉义市]]市长选举提名人资格。歐称与民进党政策會副執行長[[沈發惠|沈發惠]]是小學同學，理念與民進黨比較相近。但后来民进党不分区立法委员[[涂醒哲|涂醒哲]]愿意参选嘉义市长，欧表态支持<ref>{{cite news|url=http://hk.crntt.com/doc/1008/5/4/6/100854676.html?coluid=7&kindid=0&docid=100854676|title=綠委涂醒哲：願配合參選嘉義市長|date= 2009-01-11|publisher=中国评论网}}</ref>。

2011年，欧崇敬接受中國國民黨的邀請，至国民党艰困的台南市第一选区参选立法委员,

2015年，欧崇敬退出国民党，与[[亲民党|亲民党]]接触。他原打算参选臺北市第七選區立委，挑戰現任立法委員[[費鴻泰|費鴻泰]]<ref>{{cite news|url=http://udn.com/news/story/7741/1027208|title=立委選戰 國親整合破局、泛藍分裂|publisher=联合新闻网|author=周志豪、林庭瑤、陳信利、蔡維斌|date=2015-07-01|accessdate=2015-08-22|language=zh-tw|archive-url=https://web.archive.org/web/20150819000606/http://udn.com/news/story/7741/1027208|archive-date=2015-08-19|dead-url=yes}}</ref>，一度考慮由親民黨提名參選嘉義市立委、最後選擇於新竹市參選區域立委。

==家族==
歐崇敬的父親家族從台南搬到高雄地區，歐崇敬出生於[[臺南市|臺南市]]。

歐崇敬的母親則是台南[[佳里|佳里]]漚汪地區人，為「北門七子」中[[林芳年|林芳年]]的女兒。歐崇敬外叔公[[林金莖|林金莖]]，曾任[[中華民國駐日本大使列表|台灣駐日代表]]。
