'''歐亞板塊'''（[[英文|英文]]：'''Eurasian Plate'''）為包括大部份[[歐亞大陸|歐亞]]地區的[[板塊|板塊]]，但不含括[[南亞|南亞]]的[[印度半島|印度半島]]（[[印度次大陸|印度次大陸]]）、[[西南亞|西南亞]]的[[阿拉伯半島|阿拉伯半島]]（阿拉伯次大陸）以及東[[西伯利亞|西伯利亞]]的[[上揚斯克山脈|上揚斯克山脈]]以西的地區。
此外，歐亞大陸向西延伸至[[大西洋中洋脊|大西洋中洋脊]]（中大西洋海嶺；{{lang|en|Mid-Atlantic Ridge}}）。

== 板块边界 ==
歐亞板塊東至北緣與[[北美洲板塊|北美洲板塊]]和[[太平洋板块|太平洋板块]]交界，形成[[維科揚斯克山脈|維科揚斯克山脈]]及[[日本島弧|日本島弧]]，東至南緣與[[太平洋板块|太平洋板块]]的[[菲律賓海板塊|菲律賓海板塊]]交界，形成[[琉球島弧|琉球島弧]]及[[呂宋島弧|呂宋島弧]]，並可能存在[[鄂霍次克板塊|鄂霍次克板塊]]（{{lang|en|Okhotsk Plate}}）及[[黑龍江陸板塊|黑龍江陸板塊]]（阿穆爾大陸板塊；{{lang|en|Amurian Plate}}）並與之交界。
南至西緣則與[[非洲板塊|非洲板塊]]交界，分界線為[[地中海|地中海]]，交界之中有[[阿拉伯板塊|阿拉伯板塊]]，分界線為[[札格洛斯山脈|札格洛斯山脈]]，東則為[[印度-澳洲板塊|印澳大陸板塊]]（或分為[[印度板塊|印度板塊]]及[[澳洲板塊|澳洲板塊]]），分界線形成[[喜馬拉雅山脈|喜馬拉雅山脈]]。西緣與北美大陸板塊的[[分界線|分界線]]形成大西洋[[中洋脊|中洋脊]]的北端。

== 大陸漂移學說 ==
{{Main|大陸飄移學說}}
[[大陸漂移學說|大陸漂移學說]]是由德國科學家[[韋格納|韋格納]]提出的。歐亞板塊與北美大陸板塊、非洲大陸板塊的陸地連結形成的[[勞亞大陸|勞亞大陸]]與南美大陸板塊、非洲大陸板塊、印澳大陸板塊、[[南極洲板塊|南極洲板塊]]的陸地所連結形成的[[岡瓦納大陸|岡瓦納大陸]]在2億年前形成[[盤古大陸|盤古大陸]]。而後又分裂移動形成現今的七大洲四大洋。

== 火山與地震 ==
歐亞板塊和週遭板塊相互的[[板塊運動|板塊運動]]形成許多的[[构造地貌|构造地貌]]，如喜馬拉雅山脈、[[大西洋中洋脊|大西洋中洋脊]]、[[西太平洋海沟岛弧链|西太平洋海沟岛弧链]]、[[巽他海溝|巽他海溝]]。這些交界處形成[[地震帶|地震帶]]，其中的[[熱點_(地質學)|熱點]]並形成[[火山|火山]]。主要火山地震带有[[环太平洋火山地震带|环太平洋火山地震带]]和[[地中海－喜马拉雅火山地震带|地中海－喜马拉雅火山地震带]]。

著名的火山爆發：
* 79年8月24日意大利亞平寧半島[[維蘇威火山|維蘇威火山]]爆發，將山腳下的[[庞培城|龐貝]]古城掩埋。
* 1883年8月27日印尼拉卡塔島[[喀拉喀托火山|喀拉喀托火山]]爆發，火山的爆炸聲遠達3,541[[公里|公里]]外的澳大利亞、爪哇和蘇門答臘附近，並引發海嘯，海浪最高達37公尺，造成3萬6千餘人死亡，是人類記載火山爆發史上最強烈的一次。
* 2004年12月14日意大利西西里島[[埃特纳火山|埃特纳火山]]爆發。

著名的地震災難：
* 1995年1月17日日本[[阪神大地震|阪神大地震]]，M<sub>L</sub>7.2[[日本氣象廳|日本氣象廳]]為M<sub>J</sub>7.3[[美國地質調查所|美國地質調查所]]為M<sub>W</sub>6.9，造成5千5百餘人死亡。
* 1999年9月21日臺灣[[集集地震|集集大地震]]，M<sub>L</sub>7.3[[美國地質調查所|美國地質調查所]]為M<sub>W</sub>7.6，造成2千4百餘人死亡。
* 2004年12月26日印尼[[蘇門達臘|蘇門達臘]]外海大地震，M<sub>W</sub>9.3[[美國地質調查所|美國地質調查所]]為M<sub>W</sub>9.1，引發[[2004年印度洋大地震|南亞海嘯]]，造成29萬2千餘人死亡。
* 2005年10月8日南亞印巴交界的[[2005年克什米尔大地震|喀什米爾大地震]]，[[美國地質調查所|美國地質調查所]]為M<sub>W</sub>7.6，造成3萬餘人死亡。
* 2008年5月12日[[四川汶川大地震|四川汶川大地震]]，為M<sub>S</sub>8.0[[美國地質調查所|美國地質調查所]]為M<sub>W</sub>7.9，造成超过8萬人死亡。
* 2011年3月11日[[2011年日本東北地方太平洋近海地震|2011年日本東北地方太平洋近海地震]]為M<sub>W</sub>9.0，造成超過1萬人死亡。
* 2017年8月8日[[2017年九寨溝地震|九寨溝地震]]
