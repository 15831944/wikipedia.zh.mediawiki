'''歐拉恆等式'''是指下列的[[恆等式|關係式]]：

: <math>e^{i \pi}+1=\,</math>{{#invoke:Complex Number/Calculate|calculate|

    e^(i*pi)+1

|class=cmath|use math=yes}}

其中<math>e\,</math>是[[e_(數學常數)|自然對數的底]]，<math>i \,</math>是[[虛數|虛數]]單位，<math>\pi \,</math>是[[圓周率|圓周率]]。

這條恆等式第一次出現於1748年瑞士數學、物理學家[[萊昂哈德·歐拉|萊昂哈德·歐拉]]（{{lang|de|Leonhard Euler}}）在[[洛桑|洛桑]]出版的書<math>Introductio \,</math>。這是[[複分析|複分析]]的[[歐拉公式|歐拉公式]]的特殊情況。

美國物理學家[[理查德·費曼|理查德·費曼]]（{{lang|en|Richard Phillips Feynman}}）稱這恆等式為「數學最奇妙的公式」，因為它把5個最基本的[[數學常數|數學常數]]簡潔地連繫起來。

== 證明 ==
: <math>e^{ix} = \cos x + i \sin x \,\!</math>（[[歐拉公式|歐拉公式]]）

: <math>e^{i \pi}=\cos \pi+ i \sin \pi\,</math>（代入<math>x=\pi \,</math>）

: <math>e^{i \pi}=\,</math>{{#invoke:Complex Number/Calculate|calculate|
   exp(1)^(i*pi)
|class=cmath|use math=yes}}（因<math>\cos \pi = -1  \, \! </math>和<math>\sin \pi = 0\,\!</math>）

: <math>e^{i \pi}+1=\,</math>{{#invoke:Complex Number/Calculate|calculate|
   exp(1)^(i*pi)+1
|class=cmath|use math=yes}}
