'''歐文菊石'''（[[學名|學名]]：{{lang|la|'''''Owenites'''''}}），又名'''歐文氏菊石'''，是生存在[[三疊紀|三疊紀]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]，是三疊紀初期菊石的代表屬之一。其外殼渾圓，呈白色，有著波浪狀的縫合線。<ref name="Owenites">{{Cite web |url = http://liusir.pixnet.net/blog/post/248633045-owenites-%E6%AD%90%E6%96%87%E8%8F%8A%E7%9F%B3%E7%BE%A4%E9%AB%94-%28%E5%B8%9D%E6%B1%B6%29 |title = Owenites 歐文菊石群體 (帝汶) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-13 }}</ref>其[[化石|化石]]被發現於[[中國|中國]]、[[美國|美國]]、[[俄羅斯|俄羅斯]]、[[阿曼|阿曼]]、[[越南|越南]]及[[印度|印度]]等地。<ref name="Fossilworks" />
