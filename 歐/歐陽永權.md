'''歐陽永權'''（'''Andrew Wing-Kuen AU-YEUNG'''），[[香港製造網絡電視|香港製造網絡電視]]（MIHK TV）創辦人。香港出生，13歲到加拿大讀中學，之後入讀[[英屬哥倫比亞大學|英屬哥倫比亞大學]]，主修[[細胞生物學|細胞生物學]]。大學畢業後，在[[溫哥華|溫哥華]]當地一間實驗室，從事測試污水標本的工作。由於不習慣實驗室的沉悶工作，所以做了兩個月便離職。<ref name="sun120520">{{cite news|url=http://the-sun.on.cc/cnt/lifestyle/20120520/00481_001.html |title=歐陽永權傳承集體回憶|publisher=太陽報 |date=2012-05-20}}</ref>

1993年，他返回香港投身廣告界，在廣告界工作十多年，曾為不少著名品牌（包括：Adidas、Nokia等）創作廣告，亦創作出多個繪炙人口的廣告，包括：CTI「殺價超人」<ref>{{cite web |url=https://www.youtube.com/watch?v=H3OZMxt419M |title=CTI (殺價超人)1999}}</ref>、新鴻基地產「三隻白白豬」（Property Street）<ref>{{cite web |url=https://www.youtube.com/watch?v=3GWGrIGYVlE |title=[AD] Propertystreet - 三隻白白豬 }}</ref>等，其中不乏得獎之作。<ref name="hkgj">{{cite web |url=http://www.hkgoodjobs.com/articles.php?p=245&cat_id=1 |title=舊事物‧新設計 - 歐陽永權 |publisher=好工}}</ref><ref name="sun120520" />

2005年，在廣告界工作13年後，轉投出口製造業，自立門戶開了一間中港貿易公司，做了4年中港商人。<ref name="sun120520" />

2009年，他重投創作行列，夥拍舊戰友成立 mall852.com 港式精品公司，專門設計及售賣以香港文化為主題的產品，充滿地道特色和港式幽默，例如：「長貧牌五餐肉」萬字夾、「百家樂庄閒贏」手機套、「懷舊信箱」斜孭袋、魚蝦蟹沙灘巾、克力架餅乾紙巾盒、沙屎紙巾盒、荔園立體拼圖等。旗下品牌包括：「香港仔852」及「陀地852」。<ref name="ap160922-9bae">{{cite news|url=https://hk.lifestyle.appledaily.com/jfdigital/magazine/article/20160922/2_17768745/%E7%B6%B2%E7%B5%A1%E9%9B%BB%E8%A6%96%E5%8F%B0%E5%89%B5%E8%BE%A6%E4%BA%BA-%E5%B0%88%E7%82%AE%E8%A3%BD%E6%9C%AC%E5%9C%9F%E5%91%B3%E7%AF%80%E7%9B%AE |title=網絡電視台創辦人　專炮製本土味節目 |publisher=蘋果日報 |date=2016-09-22 }}</ref>

2013年，他投身網絡電視，創立「低俗頻道」，以通俗的廣東話暢談香港的地道話題，例如，《大丸有落》內容介紹香港的舊事，包括靈異故事、國術館、跌打館等。<ref name="ap160922-9bae" />

2015年，他創立[[香港製造網絡電視|香港製造網絡電視]]（MIHK TV）網絡電視台，拼入「低俗頻道」。該台以收費網上節目為主，主打製作有香港本土特色的節目。總部位於觀塘工業大廈內，共推出八條頻道，包括香港美食、本地文化、藝術、電影、奇案、靈異故事故及傳統習俗等，每星期播出超過50個節目。歐陽永權參與主持的節目包括：《大丸有落》、《的士判官》、《藍皮書》等。<ref name="ap160922-9bae" /><ref>{{cite news|url=https://www.hk01.com/%E8%81%B7%E5%A0%B4%E5%89%B5%E6%A5%AD/39563/-%E5%89%B5%E6%A5%AD-%E7%B6%B2%E4%B8%8A%E9%9B%BB%E8%A6%96%E5%8F%B0%E5%89%B5%E8%BE%A6%E4%BA%BA-%E8%BE%A6%E7%B6%B2%E5%AA%92-%E6%9C%80%E9%87%8D%E8%A6%81%E7%9A%84%E6%98%AF%E5%85%A7%E5%AE%B9-%E8%A6%81%E6%9C%89%E6%96%B0%E8%A7%80%E9%BB%9E|title=【創業】網上電視台創辦人：辦網媒，最重要的是內容，要有新觀點|publisher=香港01|date=2016-08-28}}</ref>
