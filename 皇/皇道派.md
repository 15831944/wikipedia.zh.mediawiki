{{Japanese|japanese = 皇道派|kana = こうどうは|romaji = Kōdōha}}
'''皇道派'''，是[[日本帝国陸軍|日本帝国陸軍]]内部存在的一個政治派系，其主要受[[北一輝|北一輝]]等的思想影響，主张尊皇、在[[天皇|天皇]]親政下對日本進行國家改造（其稱之[[昭和維新|昭和維新]]）为目标，以暴力革命打倒日本国内资本财阀，並視[[蘇聯|蘇聯]]為國家敵人。皇道派中心人物包括被奉为首领人物的[[荒木貞夫|荒木貞夫]]、[[真崎甚三郎|真崎甚三郎]]與核心人物[[小畑敏四郎|小畑敏四郎]]、[[鈴木率道|鈴木率道]]、[[满井佐吉|满井佐吉]]等，其勢力與以[[永田鐵山|永田鐵山]]、[[東條英機|東條英機]]和[[池田纯久|池田纯久]]等人為核心的[[統制派|統制派]]相對立。值得注意的是，兩派系皆無明確的組織或統一的行動<ref name='D-67'>{{Harnvb|林珠雪|1986|p=67}}</ref>。

== 溯源 ==
皇道派的出現原因可追溯回[[明治時期|明治時期]]，當時的日本陸軍長期為[[长州藩|长州藩]]派閥為多，要職皆由其出身者所把持，時間一久便出現反長州藩閥的派系。長州派將領[[山縣有朋|山縣有朋]]死後，其宿敵[[上原勇作|上原勇作]]便聯合其同志[[福田雅太郎|福田雅太郎]]、荒木貞夫、[[武藤信義|武藤信義]]、[[村岡長太郎|村岡長太郎]]、真崎甚三郎與小畑敏四郎等人公開與山縣繼任者[[田中義一|田中義一]]挑戰，為參謀總長和陸軍大臣人選問題展開論戰。1923年，[[清浦奎吾|清浦奎吾]]組閣，上原一系推舉福田任陸軍大臣人選，但內閣已採納長州派[[宇垣一成|宇垣一成]]為其人選，而參謀總長一職亦受宇垣所組，兩方的對立遂成正面衝突<ref name='伊藤-311-312'>{{Harnvb|伊藤正德|1958|p=311-312}}</ref>。

==主张==
當時的日本正處於[[脫亞入歐|脫亞入歐]]、邁向現代化的歷程，因此出現了許多[[資本主義|資本主義]]和[[政黨政治|政黨政治]]出現的社會矛盾。1927年起的[[昭和金融恐慌|昭和金融恐慌]]，更使國民經濟雪上加霜、步向兩極化，大批勞工失業、農村蕭條，而金融業和工業領導人則因為其彼此利益關係緊密而越漸富有，因此[[社會主義|社會主義]]、[[共產主義|共產主義]]和[[無政府主義|無政府主義]]的組織開始抬頭。無論是上原還是宇垣一派人，兩方都欲對國家進行改革，但主张採取的手段不同。宇垣一派主張依從憲法，在議會政治下執行政策；而上原一派形成了荒木貞夫和真崎甚三郎為首领的皇道派，否定政黨政治，認為宇垣一派與政黨勾結，國家改造只能在天皇的帶領下進行，主张以[[軍國主義|軍國主義]]代替[[议会制|议会制]]，并且痛恨宇垣推行的裁軍政策（裁減四個師团、約九萬人的兵力，用於建設軍隊現代化裝備）。

除了軍中高級將領外，皇道派的另一重大支持者為下层青年军官。裁军使得青年军官们的晋升推迟，待遇不如以前，从而不满。经济不景气造成的农村的悲惨状况，也给了农村出身的青年军官们危机感。當時日本激進思想家[[北一輝|北一輝]]曾著《國家改革法案大綱》，將[[帝國主義|帝國主義]]與[[国家社會主義|国家社會主義]]混杂結合，被皇道派份子視為思想依據，尤其是易受影響、富有理想和抱負的年輕人完全就被北一輝的思想所說服，他們不是加入軍隊就是参加具有追随此思想的政治團體，如[[血盟團|血盟團]]和[[櫻會|櫻會]]。北一輝還號召日本人起來「清除天皇身邊的障礙」（此觀點又被稱之[[清君側|清君側]]），让軍隊能與天皇直接溝通，在後者帶領下進行改革，故此此派别後來被稱之皇道派或國體原理派。

除了對內改革作法不同外，對於國際戰略的觀點，两派亦有相當分歧。1931年9月18日發生[[九一八事件|九一八事件]]，[[關東軍|關東軍]]在其參謀軍官[[石原莞爾|石原莞爾]]和[[坂垣征四郎|坂垣征四郎]]設計下，出兵佔領了比日本本土大得多的[[滿洲地區|滿洲]]，继而建立了「[[滿洲國|滿洲國]]」，皇道派视[[苏联|苏联]]为日本主要敌人，所以應協助建設[[滿洲|滿洲]]而非武力侵略，是为了使其成為[[亞洲|亞洲]]對抗[[共產主義|共產主義]]的堡壘。皇道派将統制派主张的向[[中國|中國]]擴張視為愚蠢之舉<ref name='T-10'>{{Harnvb|Toland|1970|p=10}}</ref>。他们认为，攻击中国會與[[蒋中正|蒋中正]]的[[国民革命军|国民革命军]]两败俱伤，以及制造和英美的矛盾，最後只會讓日本真正的宿敌苏联以及中国共产党漁翁得利。

== 活動 ==
在1931年12月荒木贞夫担任犬养毅内阁的陆军大臣，利用人事安排职缺的便利，形成了陆军省陆军大臣荒木贞夫以及参谋本部内参谋次长真崎甚三郎、第3部长小畑敏四郎、作战课长铃木率道的所谓皇道派核心。皇道派以武力发动国家创新的主张，并在围绕陆军领导权和未来对苏联对中国战争准备上，与统制派产生对立。

日本进入1930年代開始，相继发生幾次未遂政變，如[[三月事件_(日本)|三月事件]]、[[十月事件_(日本)|十月事件]]、[[五一五事件|五一五事件]]等，还有數位政府要員接连被刺殺，如[[大藏大臣|大藏大臣]][[井上准之助|井上准之助]]、[[內閣總理大臣|內閣總理大臣]][[犬養毅|犬養毅]] ( 五一五事件。1932。海軍 + 統制派發動 )、陆军省军务局长永田鐵山（ [[相泽事件|相泽事件]]‧ 1935。皇道派發動 ）等。其中以五一五事件來說，許多民眾對刺殺兇手表示莫大的同情，在被告辯駁，宣稱其僅是「為祖國敲起警鐘」後，公眾竟集体鼓掌，事後甚至有11萬封血書請求審判機關寬恕兇手，另還有9位青年願意替兇手服刑，為表真誠，他們每個人還各切下一部小指頭，泡在酒精裡送來<ref name='T-31'>{{Harnvb|Toland|1970|p=31}}</ref>。最終兇手裡沒有人被判處死刑，並都在幾年後獲釋，這也助長他們武力“维新”行之有理的想法，助長其行動，直至引發為[[二二六事件|二二六事件]] ( 1936。皇道派發動 )。

== 衰落 ==
{{Main|二二六事件}}
1934年1月荒木贞夫辞任陆军大臣，有望接任的参谋次长真崎甚三郎在参謀总長[[闲院宫载仁亲王|闲院宫载仁亲王]]的反对下，改任陸軍教育总监，支持统制派的[[林铣十郎|林铣十郎]]接任陆军大臣，通过人事调整压制陆军中央的皇道派势力。皇道派和统制派的矛盾开始激化。1935年，在倾向皇道派军官中享有很高威望的真崎甚三郎被迫辞去教育总監职务，接着[[相泽事件|相泽事件]]中統制派領袖永田铁山被刺杀。最終，1936年2月26日發生在首都[[東京|東京]]的二二六事件是皇道派衰落的關鍵。

由以国体原理派自居的香田清真、安藤輝三、栗原安秀等為首的下层青年軍官以一千餘名士兵，發動政變行動，不僅占据了[[總理大臣|總理大臣]]府邸，还殺害了[[大藏大臣|大藏大臣]][[高橋是清|高橋是清]]、[[內大臣|內大臣]][[齋藤實|齋藤實]]、教育總監[[渡邊錠太郎|渡邊錠太郎]]等高級官員，還佔領了陆军省、参谋本部、警视厅等東京數處據點與軍事機關，強迫高級將領支持他們，並要求任命皇道派軍官擔任要職。事件發生後，很長一段時間軍方無所作為，海軍將領擔憂海陸軍爆發[[軍種對立|軍種衝突]]<ref name='T-27'>{{Harnvb|Toland|1970|p=27}}</ref>、陸軍各將領對彼此派系不明、不敢隨便求援，最終引來[[天皇|天皇]][[裕仁|裕仁]]的干預（裕仁天皇反感皇道派主张排斥[[天皇机关说|天皇机关说]]的政策）：「如果陸軍不能鎮壓叛軍，我將親自勸阻他們。」<ref name='T-31'>{{Harnvb|Toland|1970|p=31}}</ref>。最終在空投傳單規勸、大軍包圍下，叛軍繳械投降。

與往日[[五一五事件|五一五事件]]不同的是，經過4天場面混亂的叛亂後，輿論不再支持暴力手段，而是要求恢復秩序，也正因如此，13名叛亂軍官、4名文官和北一輝本人被判處死刑<ref name='T-34'>{{Harnvb|Toland|1970|p=34}}</ref>，皇道派軍官遭到大規模的清洗，從此皇道派再也無興起，相對地，統制派坐大並控制內閣握有內閣人選廢立權，此後日本進入了軍部獨裁時期，最终統制派首領[[東條英機|東條英機]]建立了极权内阁。
