《'''皇后之死'''》是[[台灣|台灣]][[作家|作家]][[柏楊|柏楊]]著作的歷史叢書，包括《'''姑蘇響鞋'''》、《'''溫柔鄉'''》與《'''長髮披面'''》三本<ref>{{Cite web |url=http://boyang.nutn.edu.tw/archive/boyang/writer_history_queen.html |title= 皇后之死系列|accessdate=2014-03-01|publisher=柏楊文物館 }}</ref>。柏楊原本於1979年6月起開始在《[[台灣時報|台灣時報]]》的「湖濱讀史札記」專欄連載這一系列歷史相關著作<ref>{{Cite web |url=http://www.tpocl.com/content/writerTimeline.aspx?n=C0276 |title=柏楊 |accessdate=2014-03-01|publisher=臺北市政府文化局 |date=2013-12-13 }}</ref>，主要記述由上古至東漢結束共三十九位死於非命-{后}-妃的故事<ref>{{Cite web |url=http://www.21shiju.com/599.html |title= （周末读书）皇后之死：历史与虚构，哪个更毁三观？|accessdate=2014-03-01|publisher=识局网 |date=2013-12-31 }}</ref>。

柏楊認為皇后是世界上最危險的職業，一旦皇后失勢，自己身死不說，甚至禍連全家。他並且由這些-{后}-妃的一生，也解讀出當時的政治權力關係，更打破了一般人對以往認知中神聖君主的刻板印象。而在這一系列裡所記載的-{后}-妃，無論是何種死法，都很肯定的是權力爭鬥下的犧牲者。<ref>{{Cite web |url=http://book.ce.cn/read/history/hhzs/01/200608/03/t20060803_8001284.shtml |title=柏楊眼中死於非命的皇后們  |accessdate=2014-03-01|publisher=中國經濟網 |date=2006-08-03 }}</ref><ref>{{Cite web |url=http://www.xtrb.cn/epaper/ncwb/html/2012-12/26/content_375951.htm |title=《皇后之死》  |accessdate=2014-03-01|publisher=牛城晚报 |date=2012-12-26}}</ref><ref>{{Cite web |url=http://ent.sina.com.cn/x/2006-05-26/06241097506.html |title=柏杨出书细数皇后是最危险的职业 |accessdate=2014-03-01|publisher=重庆商报 |date=2006-05-26}}</ref><ref>{{Cite web |url=http://book.ce.cn/read/history/hhzs/yw/200608/03/t20060803_8001187.shtml |title=皇后之死  |accessdate=2014-03-01|publisher=中国经济网 |date=2006-08-03}}</ref>
