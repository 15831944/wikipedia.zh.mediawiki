'''皇后區'''（{{lang-en|'''Queens'''}}），又译作'''昆斯区'''，是[[美国|美国]][[纽约市|纽约市]]的五個[[行政區_(紐約市)|行政區]]之一，也是其中面積最大、人口第二多的行政區。其範圍和[[紐約州|紐約州]]'''皇后郡'''（{{lang-en|'''Queens County'''}}）相同。

皇后區位於[[長島_(紐約)|長島]]的西半部，紐約市主要的三個機場中，兩個就坐落於皇后區，分別是[[約翰·菲茨傑拉德·甘迺迪國際機場|甘迺迪國際機場]]和[[拉瓜地亞機場|拉瓜地亞機場]]。這些機場在世界上是數一數二的忙碌，讓皇后區的上空成為全國最擁塞的空域。皇后区的[[法拉盛草原可乐娜公园|法拉盛草原可乐娜公园]]是[[1939年世界博覽會|1939年世界博覽會]]和[[1964年世界博覽會|1964年世界博覽會]]的举办场地，公園內有{{link-en|大地球仪|Unisphere}}、[[美國网球公開賽|美國网球公開賽]]的主場館[[美國網球協會比莉·珍·金網球中心|美國網球協會比莉·珍·金網球中心]]和[[紐約大都會|紐約大都會]]棒球隊的主場[[花旗球場|花旗球場]]。

== 歷史 ==
早於[[美國獨立戰爭|美國獨立戰爭]]時，紐約皇后區曾經在一些戰役中發揮過次要作用。但直到十九世紀末，皇后區仍是以鄉郊為主。

1898年元旦正式改為區，謂之皇后區。後來道路網和[[紐約地鐵|紐約地鐵]]之伸延，使皇后區的人口不斷上升，由1920年大約469,042人，增加至1930年的1,079,129人<ref>
{{cite web |url=http://www.census.gov/population/www/documentation/twps0027.html |title=US Census figures for Queens 1900–1990}}</ref>。1939年，拉瓜地亞機場啟用；1948年7月1日紐約國際機場落成，為甘迺迪國際機場前身。

== 族裔及社區 ==
本區以[[人口|人口]]多元著稱，除[[白人|白人]]約佔半數以外，[[亞洲|亞洲]]裔、[[拉丁美洲|拉丁美洲]]裔、[[非裔美國人|非洲裔]]人口所佔比例，均高於全國平均值。白人人口當中，有相當大部分是[[猶太人|猶太人]]，以及20世紀以後來自[[歐洲|歐洲]]的新[[移民|移民]]；亞裔人口當中，又以[[印度|印度]]裔、[[海外華人|華裔]]、[[朝鮮族|韓裔]]三者為大宗。

== 交通 ==
皇后區是紐約市五個行政區當中面積最大的一區；[[甘迺迪國際機場|甘迺迪國際機場]]以及[[拉瓜地亞機場|拉瓜地亞機場]]均位在此區。另外[[長島鐵路|長島鐵路]]亦在皇后區設有[[鐵路|鐵路]]總站。搭乘终点为[[法拉盛|法拉盛]]的[[纽约地铁|纽约地铁]][[紐約地鐵7號線|7號線]]可以抵达皇后区各地。

== 產業 ==
有一些大公司在此區設立總部，例如時裝品牌[[宝路华_(手表)|宝路华]]、飲料公司[[Glacéau|Glacéau]]、航空公司[[捷蓝航空|捷蓝航空]]、鋼琴品牌[[史坦威公司|史坦威公司]]等。
