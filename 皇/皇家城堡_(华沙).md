'''皇家城堡'''（{{lang-pl|Zamek Królewski w Warszawie}}）是位於[[波蘭|波蘭]]首都[[華沙|華沙]]的[[宮殿|宮殿]]建築，从16世纪末到[[瓜分波兰|瓜分波兰]]200年间波兰君主的正式驻地，坐落於[[华沙老城|华沙老城]]入口处的[[城堡广场_(华沙)|城堡广场]]（Plac Zamkowy）。

在历史上，皇家城堡曾一再受到瑞典、[[勃兰登堡|勃兰登堡]]、德国、俄国军队的破坏和掠夺<ref name="dziedzictwo">{{pl icon}} {{cite web  |author = |url = http://www.dziedzictwo.pl/sources/muzea/waw/zamek-krolewski/zamekkw.html |title = Zamek Królewski w Warszawie (The Royal Castle in Warsaw)  |work = www.dziedzictwo.pl |publisher =  |pages =  |page =  |date =  |accessdate = 2008-07-18}}</ref>。

1791年，欧洲第一部现代宪法與世界上第二古老的国家宪法——[[五三宪法|五三宪法]]，由[[瑟姆|下议院]]在此起草<ref>{{pl icon}} {{cite web  |author = |url = http://www.zamek-krolewski.pl/?page=1080 |title = Sale Sejmowe |work = www.zamek-krolewski.pl |publisher =  |pages =  |page =  |date =  |accessdate = 2008-07-18}}</ref>。在19世纪，1830年11月起义失败后，皇家城堡被[[沙皇|沙皇]]用作一个行政中心。从1926年到第二次世界大战，这座宫殿用作波兰总统伊格纳齐·莫希齐茨基的总统府。经过第二次世界大战的{{link-en|拆毀華沙|Planned destruction of Warsaw|破坏}}后，皇家城堡又被重建起来。

目前，皇家城堡被列为国家历史古迹，开辟为国家博物馆<ref>{{pl icon}} {{cite web  |author = |url = http://www.zamek-krolewski.pl/?page=1003 |title = Zamek Królewski w Warszawie - Pomnik Historii i Kultury Narodowej |work = www.zamek-krolewski.pl |publisher =  |pages =  |page =  |date =  |accessdate = 2008-07-22}}</ref>。
