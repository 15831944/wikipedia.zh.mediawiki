'''皇后'''，在[[中國|中國]]史籍中常簡稱為-{'''后'''}-，是[[世界歷史|世界歷史]]上[[帝國|帝國]]最高統治者——[[皇帝|皇帝]][[嫡妻|正配]]的稱號。目前唯一仍有皇后存在的[[國家|國家]]及[[皇室|皇室]]是[[日本|日本]][[日本皇室|皇室]]，為[[明仁天皇|明仁天皇]]（亦是现存唯一的[[皇帝|皇帝]]）的皇后[[美智子|美智子]]，其他國家君主的正配一般只稱為“[[王后|王后]]”。

== 概述 ==
如前文所述，只有帝國最高統治者的正配才能被稱為、冊立為皇后，如[[東亞|東亞]][[漢字文化圈|漢字文化圈]]內的[[皇帝|皇帝]]、[[天皇|天皇]]，或[[歐洲|歐洲]]的[[俄羅斯帝國|俄羅斯帝國]][[沙皇|沙皇]]、[[神聖羅馬帝國|神聖羅馬帝國]]皇帝、[[中東|中東]]及[[北非|北非]]的[[萬王之王|萬王之王]]等君主的正配，因此，皇后的級別與使用和君王的稱號息息相關，只有在君王稱號為皇帝（包括天皇、沙皇、萬王之王等，下文以皇帝簡略之）等等時，其正配才能稱皇后；若僅為[[王|王]]、[[國王|國王]]者，則其正配只能稱[[王后|王后]]、甚至再次等的[[王妃|王妃]]。

皇后的冊立與人選，古來都是帝國的大事，尤其在中國，皇后的廢立有時常關係著政治鬥爭甚至[[外戚|皇后家族]]的存亡。原則上，皇帝是帝國的最高統治者，而皇后則是皇帝所有妻妾之首、內宮之長，有權力統御管理所有的[[皇|皇]]族與[[貴族|貴族]]女性，甚至是帝國內母儀及婦女風範的表率。

== 东亚 ==

=== 中國 ===
皇后的-{后}-，最早在中國的[[夏朝|夏朝]]，是君主生前的稱號，如-{[[后羿|后羿]]}-、-{[[后稷|后稷]]}-等等，君主在死後才改稱為[[帝|帝]]；至於君王正配則稱為[[妃|妃]]，[[嬪|嬪]]則是在[[周朝|周朝]]才開始使用的漢字。[[商朝|商朝]]開始，君主生前稱王，死後稱帝，原本的'''-{后}-'''則轉降為君王正配的稱號，稱'''王后'''，而妃也隨之降為君王[[側室|側室]]的稱號。

現代為人所熟悉與使用的皇后一號，則是在[[秦始皇|秦始皇]]確立了'''皇帝'''一詞作為國家最高統治者的稱號之後才出現，因為有了皇帝的出現，將原本的王后升格為級別更高的皇后，成為帝國內唯一能與皇帝平起平坐的地位與角色。現存的史料文獻無法確定秦始皇是否有冊立皇后，而秦二世临死前哀求咸阳县令阎乐说：“愿与妻子为黔首，比诸公子。”[[秦二世皇后|秦二世皇后]]除此之外别无记载。因此[[中國歷史|中國歷史]]上的第一位正式記載的皇后是[[漢高祖|漢高祖]][[劉邦|劉邦]]的妻子[[呂雉|呂雉]]。

==== 別稱 ====
*'''[[椒房|椒房]]'''：[[漢朝|漢朝]]皇后的宮殿多以'''椒'''塗壁，用以取暖避邪，也有“多子”之意，因此椒房可別稱皇后、或用以代稱皇后寢宮。
**漢朝哀帝寵愛美男子[[董賢|董賢]]，將董賢之妹冊封為[[董昭儀|董昭儀]]，並將之稱為'''次皇后'''，又在宮中賜董昭儀宮室名為'''椒風舍'''，和皇后所居之椒房相似。
*'''中宮'''、'''正宮'''：古代皇后所居的寢宮多位於後宮正中央，因此皇后寢宮可稱為'''正宮'''或'''中宮'''，<span style="font-size:smaller;">詳細解釋請參閱[[中宮|中宮]]</span>。
*'''元-{后}-'''、'''元嫡'''：皇帝的元配皇后可稱元-{后}-，以和繼位皇后（繼后）有所區別。
*'''梓童'''：皇帝對皇后的稱呼，如現今的'''老婆'''。
*'''慈壺'''、'''慈闈'''：對皇后或皇帝生母（多為[[皇太后|皇太后]]）的尊稱，或是用以代稱這兩者的寢宮。
*'''天下母'''、'''天地母'''：皆為對皇后的敬稱。
*'''坤極'''、'''坤位'''：皇后的別稱。
*'''聖人'''：宋朝皇后尊稱。

==== 册立 ====
[[File:“皇后之玺”玉印上部,_2015-09-22.jpg|thumb]]，被认为是西汉第一位皇后[[吕雉|吕雉]]的信物。]]
皇后的產生需經過皇帝冊立。冊立皇后是一件非常重要的國家事件，要詔告天下，普天同慶，同時還有一項正式隆重的立-{后}-儀式。除了正式的[[诏书|诏书]]（亦或以[[金册|金册]]形式）外，颁发[[印章|印章]]做为[[信物|信物]]。

若是皇帝在登基前以有[[嫡妻|正室]]，則通常不另擇人選，直接將正室納入[[後宮|後宮]]，並冊立為皇后，但也有皇帝是先將正室納入後宮，冊封為[[嬪妃|嬪妃]]，待日後在進行擢昇為皇后，如[[汉宣帝|汉宣帝]]的妻子[[许平君|许平君]]先被封为[[婕妤|婕妤]]，后立为皇后。不是所有正室都能被立为皇后，如[[曹丕|曹丕]]的妻子[[文昭皇后|甄氏]]。另一些得不到皇后之位的正室则会被册封为妃嫔，如[[唐宪宗|唐宪宗]]的发妻[[懿安皇后|郭氏]]，[[李存勗|后唐莊宗]]的发妻[[韓淑妃_(後唐莊宗)|韩氏]]。

==== 數量 ====

中國歷代皇帝在原則上，都保持著同時期只有一名皇后在位，但中國歷史上仍有部分由少數民族所建立的政權，其君主打破了一帝一-{后}-的規範，從而有一帝多-{后}-的局面，比較有名的是匈奴人所建立的前趙，其第二任皇帝劉聰，一生親自冊立過九位皇后，其中有多位皇后是同時存在；而鮮卑人所建立的北周，第四任皇帝宇文贇，先後冊立了共五位皇后，同一時間五-{后}-並立。另外，元朝的多位皇帝也同時有數位皇后並存，而由漢族建立之北齊，其第五任皇帝後主高緯也曾左右皇后並立，這也是漢族建立之政權中，唯一同時立-{后}-多於一人者。

==== 宫廷生活 ====
皇后在禮儀上與皇帝平等，出同車、入同座。在元旦、皇帝以及本人生日要接受百官的朝賀。皇后擁有自己的官署（如[[漢朝|漢朝]]的[[皇后三卿|皇后三卿]]），負責管理後宮，理論上皇帝的所有[[嬪御|嬪御]]、後宮的[[宮女|宮女]]、[[女官|女官]]等，都是她的臣屬。

當一名女子為皇后時沒有[[徽號|徽號]]（除去[[孝哀悊皇后|懿安皇后]]的情形），只有在皇后升為[[皇太后|皇太后]]、[[太皇太后|太皇太后]]時才有徽號，或是她在當[[嬪妃|嬪妃]]的時期才會有徽號。

==== 尊封 ====

皇帝去世（稱[[大行皇帝|大行皇帝]]）的話，皇后一般獲嗣皇帝尊為-{[[皇太后|皇太后]]}-，並同時恭上皇太后徽號，但嗣皇帝並不總是皇后親生子，有可能是前任皇后或[[妃嫔|妃嫔]]所生的儿子以及大行皇帝的兄弟。

若嗣皇帝为妃嫔所生的[[庶子|庶子]]，会将皇后和生母一并尊为皇太后，有的朝代只将皇后尊为皇太后，而皇帝生母得不到尊封或仅尊封為[[太妃|太妃]]的。如果生母早逝，皇帝通常会追尊她为皇后或皇太后甚至[[太皇太后|太皇太后]]，例如[[南明|南明]][[弘光帝|弘光帝]]追尊祖母[[鄭貴妃|鄭貴妃]]為太皇太后。

若嗣皇帝與去世的皇帝[[輩分|輩分]]相當，甚至是大行皇帝的尊長的時候，大行皇帝的皇后就只能稱○○皇后（○○為大行皇帝的最後一個年號，或是由繼任皇帝上徽號給該皇后，如[[明|明]]朝[[孝哀悊皇后|懿安皇后]]）。

===越南===

独立前的[[越南|越南]]君主[[李南帝|李南帝]]、[[梅黑帝|梅黑帝]]没有关于他们皇后的记载。越南的皇帝一般一次可以设置多个皇后。[[丁朝|丁朝]]的[[丁先皇|丁先皇]]立了5个皇后，[[前黎朝|前黎朝]]的[[黎大行|黎大行]]立了5个皇后，[[李太祖|李太祖]]立了3个皇后，[[李太宗|李太宗]]立了7个皇后。不过，在[[越南陳朝|越南陳朝]]、[[后黎朝|后黎朝]]刚建立时只立一个皇后。

[[阮朝|阮朝]]不设皇后，最高是[[皇貴妃|皇貴妃]]。当他的儿子成为皇帝后，她被尊为[[皇太后|皇太后]]，死后得到皇后谥号（除[[南芳皇后|南芳皇后]]例外）。

===日本===

[[日本天皇|日本天皇]]之正妻也和中國一般稱之為皇后，而自[[平安時代|平安時代]]的[[一條天皇|一條天皇]]以後，因權臣使然，開了'''一帝二-{后}-'''的先例，而自一條天皇以後的歷代天皇，在冊立皇后時大多先冊為[[中宮|中宮]]，之後立第二位皇后時，便將此前的中宮升為名義上地位較高的皇后，而新冊的第二位皇后仍稱中宮。 <small>（詳細解釋請參閱 '''[[中宮|中宮]]'''）</small>

自一條天皇首開兩-{后}-的先河後，後世的[[後冷泉天皇|後冷泉天皇]]，甚至'''一帝三-{后}-'''，此時除去前兩位稱之為皇后與中宮之外，第三位稱之為[[皇太后|皇太后]]。而自平安時代至[[鐮倉時代|鐮倉時代]]初期，皇后成為一種榮銜，不時用以封給天皇的姑母、叔母、准母與同母未婚姐妹，且受封者不需與天皇有夫妻之實。

由於冊立皇后或中宮，都需要大量的金錢來操辦冊封典禮，且皇后或中宮的生活開銷及歲奉，也都是可觀的數目，而皇室自[[室町時代|室町時代]]以後逐漸式微、財政窮困，很多天皇的登基儀式——[[大嘗祭|大嘗祭]]，都是在極為困窘的狀況下舉行，因此多位天皇都不再冊立中宮或皇后，直至[[江戶幕府|江戶幕府]]二代將軍[[德川秀忠|德川秀忠]]之女[[德川和子|德川和子]]、成為[[後水尾天皇|後水尾天皇]]皇后以後，為維持德川和子在宮內體面的生活，幕府增加了每年對朝廷的歲貢，但仍有幾位天皇只設[[女御|女御]]、不設皇后或中宮，主因還是皇室財政匱乏。

日本自[[二戰|二戰]]之後取消了[[女院|女院]]制，而出現了首位以皇后之號為[[諡號|諡號]]的皇后，為第123代[[大正天皇|大正天皇]]之皇后[[貞明皇后|貞明皇后]]。目前在位的日本皇后是第125代今上天皇[[明仁|明仁]]的皇后[[美智子|美智子]]，首開先例地以平民之身成為皇后。

===朝鮮===

[[朝鮮王朝|朝鮮王朝]]為中國之[[藩國|藩國]]，因此其[[君主|君主]]只能稱[[國王|國王]]而非[[皇帝|皇帝]]，相對的，國王之正妻也只能稱'''王妃'''而不是皇后。

國王正妻稱王妃，如下列：
*當朝國王正妻：王妃
*前任國王正妻：[[王大妃|王大妃]]
*前前任國王正妻：[[大王大妃|大王大妃]]
但若是宮中有三代以上前代王妃共存時；
*當朝國王正妻：王妃
*前任國王正妻：大妃
*前前任國王正妻：[[王大妃|王大妃]]
*前前前任國王正妻：大王大妃

朝鮮王妃對於管理後宮有很高的決定權，通常國王或王大妃亦不得干涉，而後宮[[嬪御|嬪御]]的[[徽號|徽號]]和[[牒紙|牒紙]]也是由王妃擬定、頒下。王妃死後以'''王后'''之名為諡號。
最後一位生前以王妃之名登位的國王正室是高宗的王妃——明成皇后，[[大韓帝國|大韓帝國]]成立以後，開國皇帝[[朝鮮高宗|高宗]]，追諡了部分先祖為[[皇帝|皇帝]]及'''皇后'''，而大韓帝國獨立期間，只有一位皇后，為[[朝鮮純宗|純宗]]的继室[[純貞孝皇后|純貞孝皇后]]尹氏。

== 欧洲 ==

欧洲国家的[[语言|语言]]中，皇后与[[女皇帝|女皇帝]]是同一个单词。例如[[英语|英语]]中为了区别两者，在empress加上consort（王配）表示是男皇帝的配偶，即皇后（empress consort）。加上regnant（佔统治地位的）表示女皇帝（empress regnant）。[[法语|法语]]中，则是在“皇帝”（empereur）一词前加femme（[[女性|女性]]）以示区别。

== 廢后、再婚 ==

{{main|废后}}
歷史上有許多皇后被廢。在[[一夫多妻|一夫多妻]]國家中，因為某些原因，如君主對皇后感情淡薄、皇后無子、皇后族屬犯罪、皇后本人犯罪等等原因，被廢的皇后或授予較低級別的位號、或軟禁在宮中而沒有任何稱號、或[[出家|出家]]（實際上不能在宮外居住，大多在宮中落髮修行），也有被賜死的例子。通常废后没有再次结婚的自由。但在国家动乱时，废后会因种种原因而再婚。[[东晋|东晋]][[南北朝|南北朝]]时，时局动荡，皇后和[[妃嫔|妃嫔]]再婚并不罕见。[[羊献容|羊献容]]是[[晋惠帝|晋惠帝]]的第二任皇后，后成为[[前赵|前赵]][[刘曜|末帝刘曜]]的皇后。也有的废后因政局动荡等原因重新获得皇后地位，如[[北宋|北宋]][[宋哲宗|宋哲宗]]废后[[孟皇后|孟氏]]曾被小叔子宋徽宗恢复皇后地位，后又因政局变动被再废，但在[[南宋|南宋]]初年[[金朝|金朝]]南侵之际，又作为[[宋高宗|宋高宗]]的伯母被尊为皇太后。

有的皇后则并非在丈夫生前被废，而是在成为皇太后之后，因政治斗争失势等原因被废。

在[[一夫一妻|一夫一妻]]制國家，也有一些皇后或王后被君主強制[[離婚|離婚]]或於死後褫奪皇后或王后銜頭。
