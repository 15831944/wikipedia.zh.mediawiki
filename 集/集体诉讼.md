'''集体诉讼'''（class action、class suit、representative action、集團訴訟），也称'''代表人诉讼'''，指的是[[訴訟|訴訟]]各方中有一方是一群人，他们会派一名代表人代表众多当事人来参与解决矛盾。<ref>{{cite web |author1=中国证券投资者保护网 |title=什么是集体诉讼-新华网 |url=http://www.xinhuanet.com/finance/2016-05/12/c_128978468.htm |website=www.xinhuanet.com |accessdate=2018-06-23 |archiveurl=https://archive.today/20180623030214/http://www.xinhuanet.com/finance/2016-05/12/c_128978468.htm |archivedate=2018-06-23 |language=zh-cn |date=2016-05-12 |dead-url=no }}</ref>

== 历史 ==
集体诉讼最早有据可考的是出现在13世纪的[[英格兰|英格兰]]，多是因为一项工事影响了一群人，而处理这些案件又受时间、交通条件限制。后来18世纪时英格兰法律变化巨大，集体案件变得难以处理，国会因此通过了一些法律，使得1850年后集体诉讼几乎销声匿迹。但美国在[[美国最高法院|美国最高法院]]法官[[约瑟夫·斯多利|约瑟夫·斯多利]]的影响下将集体诉讼的传统沿袭了下来，并流传至今。<ref>{{cite web |title=History of Class Action Lawsuits  {{!}}  Class Action Lawsuits Center |url=https://classactionlawsuitcenter.com/history-of-class-action-lawsuits/ |website=classactionlawsuitcenter.com |accessdate=2018-06-23 |archiveurl=https://archive.today/20150524215722/http://classactionlawsuitcenter.com/history-of-class-action-lawsuits/ |archivedate=2015-05-24 |language=en-us |date=2015-05-24 |dead-url=no }}</ref>

集体诉讼源如今仍然主要是美国在进行，但加拿大和一些使用[[欧陆法系|欧陆法系]]的欧洲国家近年作出过一定的变更，使得消费者组织能够代表消费者进行诉讼。
