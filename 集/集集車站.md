'''集集車站'''位於[[台灣|台灣]][[南投縣|南投縣]][[集集鎮|集集鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[集集線|集集線]]的[[鐵路車站|鐵路車站]]，不僅是集集鎮民的對外門戶，也是全線最多人造訪的車站，為南投縣著名觀光旅遊勝地，設有集集鐵路文物博覽館。

== 車站構造 ==
*[[島式月臺|島式月臺]]一座，曾辦理列車交會，現月台A側已無列車停靠。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#簡易站|簡易站]]（由[[南投縣|南投縣]][[集集鎮|集集鎮]]公所管理，並代售車票），行駛車輛為DR1000型[[柴油|柴油]]客車(冷氣柴客)，等級為[[臺鐵區間車|區間車]]。
*是台鐵少數仍有販售[[埃德蒙森式鐵路車票|名片式車票]]的車站（票面上印有[[集集鎮|集集鎮]]公所代售），但無訂票取票及代售他站各級列車車票之業務。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
[[File:集集線車票.jpg|thumb]]
*1922年1月14日：設「集集驛」。
*1930年2月2日：由縱貫線舊[[楊梅車站|楊梅車站]]拆遷而來的木造站房完工。
*1991年9月1日：降為招呼站，指定由[[二水車站|二水站]]管理。
*1996年8月3日：由[[集集鎮|集集鎮]]公所代售車票，站等升為簡易站。
*1997年12月:站外火車頭建立
*1999年9月21日：站房因[[921大地震|921大地震]]而嚴重傾斜。
*2001年2月7日：站房依原貌重建完工。
*2015年6月30日：啟用多卡通刷卡機。

==其他==
*集集車站是[[公共電視台|公視]]於2009年時拍攝、2010年播映，以火車站長的故事為主題的[[電視電影|電視電影]]《[[記得我們愛過|記得我們愛過]]》的主要拍攝場景。
