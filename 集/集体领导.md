'''集体领导'''（英语：Collective Leadership），被一些政治派别認定“一種[[寡頭統治|寡頭統治]]的形式”，通常存在於[[共產黨|共產黨]]管理的[[社會主義國家|社會主義國家]]裡。不過，“集體領導”這個詞語本身的主旨是指：將某些“權能”從個人（比如，黨主席）分配到一個集體中由多人共同決策的意思。

以[[中華人民共和國政治|中華人民共和國政治]]為例，“中央權力”被分散到[[中國共產黨中央政治局|中共中央政治局]]（[[中國共產黨中央政治局常務委員會|常委會]]）、[[中國共產黨中央委員會總書記|總書記]]、[[中國共產黨中央書記處|中央書記處]]，同時仍然保留一位最終決策者（[[中華人民共和國最高領導人|最高領導人]]）做統籌；在[[越南|越南]]，權力較為分散，由[[越南共產黨中央委員會總書記|黨總書記]]、[[越南社會主義共和國主席|國家主席]]、[[越南總理|政府總理]]以及包括[[越南共產黨中央政治局|越共中央政治局]]、[[越南共產黨中央書記處|中央書記處]]和中央委員會等在內的合議機構分享。在别的領域的“集體領導”的例子就不一一列舉，比如，[[董事會|董事會]]。

== 制度比較 ==
集體領導採用[[委員會制|委員會制]]，在集体领导下可能仍设有[[领袖|领袖]]，但没有最终决定权，一般只作为会议的召集人和主持人。集体领导被认为是一种阻止个人[[独裁|独裁]]的领导方式，但也可能造成决策上的效率降低，權責歸屬不明確，在非常时期一般不会采用。

集體領導是一種介于[[民主|民主]]和[[独裁|独裁]]之间的制度。在集體領導中，一方面因为实行小范围内的民主，可以避免独裁；另一方面因為缺少外部的權力制衡，容易形成均質的小團體，表现[[集權|集權]]與[[專制|專制]]。

== 实际运用 ==

=== 中华人民共和国 ===
{{参见|中国共产党集体领导制度|党和国家领导人}}
现在，[[中华人民共和国政府|中华人民共和国政府]]的领导机构权力集中在[[中国共产党中央政治局常务委员会|中国共产党中央政治局常务委员会]]，中央政治局常委现在有7位，由[[中共中央总书记|中共中央总书记]]作为首脑<ref>{{cite web |url=http://www.wantchinatimes.com/news-subclass-cnt.aspx?id=20121018000019&cid=1101 |title=存档副本 |accessdate=2013-02-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20130115173111/http://www.wantchinatimes.com/news-subclass-cnt.aspx?id=20121018000019&cid=1101 |archivedate=2013-01-15 }} "New Politburo Standing Committee decided: Mingjing News". ''Want China Times''. 18 October 2012. Retrieved 2 January 2013.</ref>。

在[[毛泽东|毛泽东]]统治前期，[[中国共产党|中国共产党]]内实施集体领导，由[[中共中央书记处|中共中央书记处]]五大书记领导，毛泽东担任[[中国共产党中央委员会主席|中央委员会主席]]。在[[中共八大|中共八大]]上，重新设立中央政治局常委会，除了党主席外，还加设[[中国共产党中央委员会副主席|副主席]]和[[中国共产党中央书记处总书记|总书记]]（书记处负责人）的职务。不过到了[[文化大革命|文化大革命]]时期，原来的集体领导制度崩溃，曾经担任党副主席兼国家主席的[[刘少奇|刘少奇]]和中央书记处负责人的[[邓小平|邓小平]]等人相继被打倒，毛泽东集大权于一身，甚至被[[神化|神化]]。

1982年[[中共十二大|中共十二大]]上，决定党内不再设置主席和副主席的职务，以[[中国共产党中央委员会总书记|中央委员会总书记]]作为最高负责人，重新确立集体领导制度。在1980年代，[[胡耀邦|胡耀邦]]和[[赵紫阳|赵紫阳]]先后担任总书记的职务，不过当时中共中央仍受制于以[[邓小平|邓小平]]为首的[[八大元老|八大元老]]和[[中国共产党中央顾问委员会|中央顾问委员会]]。

2017年[[中国共产党第十九次全国代表大会|中共十九大]]之后，[[中国共产党中央政治局|中共中央政治局]]全体委员和常委都要向[[中国共产党中央委员会总书记|中共中央总书记]]述职，“总书记绝对领导制”取代原来的集体领导制，确保总书记掌握绝对权力。<ref name="总书记独裁制">{{cite web |url = http://www.xinhuanet.com/politics/2018-03/21/c_1122569929.htm|title=中央政治局同志向党中央和习近平总书记述职|publisher=新华网|date=2017-10-27|accessdate=2018-03-21}}</ref>

=== 苏联 ===
{{main|最高苏維埃}}
[[苏联共产党|苏联共产党]]和[[苏联政府|苏联政府]]都采取集体领导的形式，党内最初不设最高领导人职务，国家也没用单独的[[国家元首|国家元首]]，是以[[最高苏维埃主席团|最高苏维埃主席团]]主席代表集体元首。不过[[斯大林|斯大林]]在[[列宁|列宁]]死后通过一系列的政治斗争取得了最高权力，实施独裁统治，他担任的[[苏共中央总书记|苏共中央总书记]]职务变成苏联的党和国家最高领导人职务。在[[勃列日涅夫|勃列日涅夫]]统治時期，总书记不再兼任部长会议主席，苏联党和国家的权力集中在[[苏共中央总书记|苏共中央总书记]]（最高领导人）、[[苏联最高苏维埃主席团|苏联最高苏维埃主席团]]主席（国家元首）和[[苏联部长会议主席|苏联部长会议主席]]（政府首脑）手里，又被称为“[[三驾马车|三驾马车]]”，不过总书记拥有最终决策权。

=== 越南 ===
{{main|越南政治}}
[[越南共产党|越南共产党]]在九大以后不设“[[越南共产党中央政治局常务委员会|中央政治局常委]]”的职务，[[越共中央政治局|越共中央政治局]]的权力主要集中在[[越共中央总书记|越共中央总书记]]、[[越南国家主席|越南国家主席]]和[[越南政府总理|越南政府总理]]手中，也被外界称为“[[三驾马车|三驾马车]]”。[[越共中央书记处|越共中央书记处]]常务书记和[[越南国会主席|越南国会主席]]也曾经是政治局常委会的一员。2018年，越共中央总书记[[阮富仲|阮富仲]]兼任国家主席一职。
===朝鲜===
{{main|朝鲜政治}}
[[朝鲜劳动党|朝鲜劳动党]]在党的章程和[[朝鲜宪法|朝鲜宪法]]层面上表现出集体领导的特点。但事实上[[朝鲜劳动党委员长|朝鲜劳动党委员长]]兼任[[朝鲜劳动党中央军事委员会委员长|朝鲜劳动党中央军事委员会委员长]]、[[朝鮮人民軍最高司令官|朝鮮人民軍最高司令官]]，为[[朝鲜人民军|武装力量]]最高统帅。体现了该政党对[[武装力量|武装力量]]拥有绝对的控制权。而除国家最高权力机构[[朝鲜国务委员会|朝鲜国务委员会]]外，作为[[朝鲜劳动党中央委员会|朝鲜劳动党中央委员会]]领导机构的[[朝鲜劳动党中央政治局|中央政治局]]是北韩事实上的最高权力机关，其中[[朝鲜劳动党中央政治局常务委员会|中央政治局常务委员会]]又是该机构中主要的意思决策机关。常务委员会由政治局常委组成。现今作为执政党劳动党的核心职务和人物分别是担任[[朝鲜劳动党委员长|朝鲜劳动党委员长]]作为[[朝鲜最高领导人|朝鲜最高领导人]]的[[金正恩|金正恩]]；作为朝鲜国务委员会第一副委员长、朝鲜劳动党副委员长并担任[[最高人民会议常任委员会委员长|最高人民会议常任委员会委员长]]的[[崔龙海|崔龙海]]；和作为[[朝鲜劳动党中央政治局常委|朝鲜劳动党中央政治局常委]]、[[朝鲜内阁总理|朝鲜内阁总理]]的[[金才龙|金才龙]]。
