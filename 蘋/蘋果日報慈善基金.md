'''蘋果日報慈善基金'''（{{lang-en|'''Apple Daily Charitable Foundation'''}}，[[縮寫|縮寫]]：'''蘋果基金'''）是香港一個[[慈善基金|慈善基金]]，由[[壹傳媒有限公司|壹傳媒有限公司]]於1995年成立。這基金透過《[[蘋果日報_(香港)|香港蘋果日報]]》籌集讀者的捐款，除協助老弱傷殘人士外，每年亦透過撥款去資助香港社會服務機構，推行各類型的服務計劃及活動<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/news/art/20080125/10682641 | title=150社福代表參與蘋果基金簡介 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date= 2008-01-25|accessdate=2016-01-20}}</ref>。

== 願景 ==
藉著與時並進和優質的服務，令社會的弱勢社群能感受到愛和關懷，生活亦得以改善，生命顯得有尊嚴。

== 使命 ==
* 這基金轄下有不同專戶，透過熱心人士的捐款，幫助有不同需要或遭遇突變的社群度過困境<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/realtime/breaking/20151107/54400025 | title=【蘋果慈善行】一分鐘認識蘋果基金 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date= 2015-11-07|accessdate=2016-01-20}}</ref>。
* 透過撥款資助慈善團體及社會服務機構舉辦各類型服務，填補現有服務的不足以及滿足不同社群的需要<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/news/art/20150201/19024713 | title=蘋果基金惠澤逾四萬人次  |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date= 2015-02-01|accessdate=2016-01-20}}</ref>。

== 主題曲 ==
此基金的信念為「生命因你變得好」。

這基金受助人之一席林遭癌病纏身，仍積極向著音樂夢進發，離世前是獨立樂隊GalPlay(膠玩音樂)的鼓手。他離世後，其餘樂隊成員忍著悲痛，為這基金成立20周年，創作一首勵志主題曲《生命因你變得好》，該主題曲會在這基金舉辦各大活動時播放，亦會用作長期「會歌」。GalPlay主音Cory希望給予溫暖和帶有正能量的感覺，融合這基金致力協助弱勢社群的願景，和配合這基金的信念<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/news/art/20150714/19219341 | title=【蘋果基金】獨立樂隊唱作 蘋果基金主題曲《生命因你變得好》|publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date= 2015-07-14|accessdate=2016-01-20}}</ref>。

由於主題曲推出後得到積極的回響，故這基金同工、義工隊、一群受助人及慈善機構「[[龍耳|龍耳]]」一同參與《生命因你變得好》的演出，以口語及手語一同合唱<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/news/art/20151214/19411632 | title= 【蘋果基金】迴響熱烈 主題曲加推一人一句傳承「愛」|publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date= 2015-12-14|accessdate=2016-01-20}}</ref>。

== 受惠人數 ==
這基金已成立超過20年，總撥款額已超過三億、共超過六十萬人受惠<ref>{{cite web |language=zh-hk |url=http://hk.apple.nextmedia.com/realtime/news/20151209/54518925 | title= 蘋果日報慈善基金20周年 總撥款逾3億 60萬人受惠 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date= 2015-12-09|accessdate=2016-01-20}}</ref>。
