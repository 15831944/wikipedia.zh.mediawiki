'''暎嬪李氏'''（{{lang-ko|'''영빈 이씨'''}}）（{{bd|1696年|8月15日（[[七月十八|七月十八]]）|1764年|8月23日（[[七月廿六|七月廿六]]）}}），[[朝鮮英祖|朝鮮英祖]]的後宮[[妃嬪|嬪御]]，[[朝鮮正祖|正祖]]的親祖母。[[本貫|本貫]][[全義|全義]]，贈贊成[[李榆蕃|李榆蕃]]之女。
 
== 生平 ==
李氏生於[[朝鮮肅宗|朝鮮肅宗]]二十二年（[[丙子|丙子]]年）七月十八日。[[辛巳|辛巳]]年入宮為宮女，時年六歲（虛歲）<ref>《暎嬪行狀》……辛巳肅廟命選宮人，嬪應選時年甫六歲。……</ref>。

英祖即位後李氏才獲得英祖的寵愛，當時她將近30歲。英祖二年（1726年）十一月，懷有身孕的李氏受封為[[淑儀|淑儀]]，可能因為她與英祖同姓的緣故，一開始封為淑儀引起大臣上疏規諫<ref>{{Cite web|url=http://sillok.history.go.kr/id/kua_10211016_002|title=《英祖大王實錄》10卷，英祖2年（西元1726年 / 丙午年 / 清雍正4年）11月16日（甲辰）2번째기사|accessdate=2017-09-30|author=|date=|work=sillok.history.go.kr|publisher=|language=ko}}</ref><ref>{{Cite web|url=http://sillok.history.go.kr/id/kua_10302006_001|title=《英祖大王實錄》11卷，英祖3年（西元1727年 / 丁未年 / 清雍正5年）2月6日（癸亥）1번째기사|accessdate=2017-09-30|author=|date=|work=sillok.history.go.kr|publisher=|language=ko}}</ref>，但仍順利冊封，並在隔年四月產下後來的[[和平翁主|和平翁主]]；四年（1728年）陞為[[贵人|贵人]]，六年（1730年）晉升為'''暎嬪'''。可惜的是她連生5名[[女兒|女兒]]，[[靖嬪李氏|靖嬪李氏]]則先在肅宗四十五年生下[[兒子|兒子]]敬義君[[李緈|李緈]]，英祖即位後冊封為世子，是為[[孝章世子|孝章世子]]。孝章世子卻在英祖四年夭折，年僅9歲，之後有7年的時間都沒有[[王子|王子]]誕生，使得英祖著急萬分。終於在英祖十一年（1735年），暎嬪李氏生下一子[[莊獻世子|李愃]]，英祖自是高興萬分，立刻冊封為[[世子|世子]]，即[[莊獻世子|莊獻世子]]。 

暎嬪李氏共為英祖生下1子6女，但有3名[[翁主|翁主]]未取名即夭折，順利長大成人的3位翁主分別是[[和平翁主|和平翁主]]、[[和協翁主|和協翁主]]與[[和緩翁主|和緩翁主]]。她最大的痛楚應是在晚年默许，并且眼睜睜看著莊獻世子被英祖活活餓死。

莊獻世子死後2年，她於英祖四十年[[七月廿六|七月廿六]]（1764年）過世，英祖相當悲傷<ref>{{Cite web|url=http://sillok.history.go.kr/id/kua_14007026_001|title=《英祖大王實錄》104卷，英祖40年（西元1764年 / 甲申年 / 清乾隆29年）7月26日（丙子）1번째기사|accessdate=2017-09-30|author=|date=|work=sillok.history.go.kr|publisher=|language=ko}}</ref>，賜[[宮號|宮號]]'''義烈'''，正祖時改為'''宣禧宮'''，高宗時賜[[諡號|諡號]]'''昭裕'''，全稱'''宣禧宮昭裕暎嬪'''。宣禧宮正是現在的[[七宮|七宮]]之一，暎嬪的牌位也供奉在這裡。
