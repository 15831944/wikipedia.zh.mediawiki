《'''汕頭都市報'''》（簡稱“'''汕都'''”）是[[中华人民共和国|中国]][[廣東省|廣東省]][[汕頭市|汕頭市]]的一份综合性[[日报|日报]]，于1999年3月15日创刊，为[[汕頭經濟特區報社|汕頭經濟特區報社]]旗下的一份[[報刊|報刊]]，國內統一刊號為CN44-0183，發行範圍一度覆盖[[潮汕地區|潮汕地區]]，輻射[[粵東|粵東]]、[[閩南|閩南]]及[[贛南|贛南]]部分地區，是粵東首份面向市民的都市類報刊<ref>{{cite web |url = http://www.dahuawang.com/bsxx/bsintro.htm |title= 汕头经济特区报社简介 |publisher=大华网 |accessdate = 2018-01-11}}</ref>。報紙版面大小四开，工作日發行版面24版，周末及節假日發行16版。

2017年12月31日，《汕頭都市報》在其頭版發表“休刊轉型致讀者”一文，正式宣佈從2018年1月1日起休刊<ref name="cp">{{cite news |url = http://dsb.dahuawang.com/html/2017-12/31/content_872042.htm |title= 再见，为了更好的相遇—汕头都市报休刊转型致读者 |publisher=汕头都市报 |date = 2017-12-31|accessdate = 2018-01-11}}</ref>。汕頭經濟特區報社旗下的另一報刊《[[汕頭特區晚報|汕頭特區晚報]]》也於同日休刊。两報休刊後，原有品牌版面、著名栏目将并入《[[汕頭日报|汕頭日报]]》，其[[微博|微博]]和[[微信#公眾號|微信公眾號]]作為汕頭經濟特區報社旗下全媒體繼續運營<ref name="fh">{{cite news |url = http://news.ifeng.com/a/20171231/54700289_0.shtml |title= 
汕头两家都市报明起停办 |publisher=凤凰网 |date = 2017-12-31 |accessdate = 2018-01-11}}</ref>。

==概況==

前身為汕頭市總工會主辦的《[[特區工報|特區工報]]》。1998年12月31日，《特區工報》刊登休刊啟事，於1999年1月1日起休刊兩個月。在隨後發生的汕頭報業整合中，《特區工報》的出版單位特區工報社與汕頭日報社、汕頭特區晚報社聯合，於1999年3月1日重組成立汕頭經濟特區報社，《汕頭都市報》計劃創刊，原《特區工報》員工大部分由《汕頭都市報》接收<ref>{{cite web |url = http://blog.sina.com.cn/s/blog_512ce01e0101krau.html |title= （第五章 特区风雨）四、下岗与重组 |author=王亚洲 |publisher=新浪博客 |date = 2013-05-11 |accessdate = 2018-01-11}}</ref>。經過兩個星期的籌備，3月15日，《汕頭都市報》正式發行。

《汕頭都市報》以“辦市民最喜愛的報紙”為宗旨，新聞報道立足於本地民生，先後設置民生欄目“都市侃吧”、“社區新聞”、“愛心連線”及訪談類欄目“曉顰專訪”<ref>{{cite news |url = http://dsb.dahuawang.com/html/2017-12/31/content_872074.htm |title= 历时十六载，讲述一个个都市“绝对隐私”——那一场“风花雪月”的故事 |author=李晓颦 |publisher=汕头都市报 |date = 2017-12-31 |accessdate = 2018-01-11}}</ref>等。

2017年末，汕頭經濟特區報社啟動改革，發展新媒體，將旗下的《汕頭日報》、《汕頭特區晚報》及《汕頭都市報》合而為一，《汕頭都市報》與《汕頭特區晚報》於2018年1月1日起休刊，累計發行6860期<ref name="cp"/>。休刊後，《汕頭都市報》官方微博和微信公眾號繼續運營<ref name="fh"/>。
