'''汕尾中學'''，是汕尾市一所擁有[[初級中學|初級中學]]和[[高級中學|高級中學]]混合型的公立中學，和另外两所所中學[[汕尾市新城中學|新城中學]]和[[林偉華中學|林偉華中學]]並列為城區三所重點中學之一。

== 歷史 ==
'''汕尾中學'''創辦於一九四二年，歷經坎白中學、海豐縣第二中學、汕尾中學三個發展時期。<ref>{{Cite web |url=http://www.swzx.org/Article/ShowArticle.asp?ArticleID=43 |title={title} |access-date=2012-07-22 |archive-url=https://web.archive.org/web/20140513065649/http://www.swzx.org/Article/ShowArticle.asp?ArticleID=43 |archive-date=2014-05-13 |dead-url=yes }}</ref>1949年10月17日，汕尾市政權易幟，新政府接管了[[海豐縣|海豐縣]]私立坎白初級中學與汕尾作嘉學校，始創海豐第二初級中學，後又改名為海豐汕尾中學、汕尾市城區汕尾中學。是海陸豐地區辦學最早的學校之一。

== 校園概況 ==
=== 基本概況 ===
學校占地面積37000平方米，建築面積31300平方米，綠化覆蓋率達76%；現有56個教學班，學生總數近4000人。<ref>{{Cite web |url=http://www.swzx.org/Article/ShowArticle.asp?ArticleID=371 |title={title} |access-date=2012-07-22 |archive-url=https://web.archive.org/web/20140512220408/http://www.swzx.org/Article/ShowArticle.asp?ArticleID=371 |archive-date=2014-05-12 |dead-url=yes }}</ref>　
=== 設施建設 ===
教室56間，實驗室5間，學生社團活動室2間，多媒體功能廳1間，教學平臺近30個，語言實驗室1間，電子閱覽室1間；圖書館1座，圖書閱覽室1間；行政辦公樓和學生宿舍樓各1幢，教師宿舍樓4幢共120套；擁有體育場15000平方米。<ref>{{Cite web |url=http://www.swzx.org/Article/ShowArticle.asp?ArticleID=371 |title={title} |access-date=2012-07-22 |archive-url=https://web.archive.org/web/20140512220408/http://www.swzx.org/Article/ShowArticle.asp?ArticleID=371 |archive-date=2014-05-12 |dead-url=yes }}</ref>　

== 榮譽 ==
學校為正科級建制，屬區重點中學。2003年8月被汕尾市教育局評定為“市一級學校”，並成為廣東省首批“現代教育技術實驗學校”。市“普通高考優秀學校”、市“青少年科技創新示範學校”，“汕尾市職業技術學院實習基地”等
