'''汕头站'''位于[[中国|中国]][[广东省|广东省]][[汕头市|汕头市]][[龙湖区|龙湖区]]，是[[广梅汕铁路|广梅汕铁路]]（畲龙铁路）、[[杭深线|杭深线]]汕头[[联络线|联络线]]车站。车站现为[[中国铁路广州局集团有限公司|中国铁路广州局集团有限公司]]惠州车务段管辖的[[一等站_(中铁)|一等站]]<ref name="22year">{{Cite web |url=http://news.163.com/17/0908/06/CTPR9HBM00018AOP.html |title=拥抱高铁时代 22载汕头火车站升级 |access-date=2018-12-09}}</ref>，办理客运业务。

==历史==
汕头站作为[[广梅汕铁路|广梅汕铁路]]的终点站于于1995年12月28日启用，2000年起车站到发客流不断下降<ref name="22year"/>。

2007年8月，连接汕头站和杭深线[[潮汕站|潮汕站]]的工程出炉，2013年9月6日被批准建设<ref>[http://news.nfdaily.cn/content/2013-12/18/content_87539217.htm 厦深铁路联络线明年动工 2年后开进汕头市区]</ref><ref>{{Cite web |url=https://mp.weixin.qq.com/s/WpN6Yee60YWHAhp4d9nKDQ|title=汕头人“喜提”动车刷爆网络！“动车入汕”从蓝图到现实，原来经历了这么多……|access-date=2019-01-01}}</ref>。工程于2015年5月10日正式开工<ref>{{Cite web |url=http://cnews.chinadaily.com.cn/2015-05/14/content_20716514.htm |title=厦深铁路汕头联络线打下第一桩|access-date=2018-12-09}}</ref>，为此汕头站进行了电气化及站台改造。联络线于2019年1月5日正式通车并开行始发至广州东、深圳的动车组列车<ref>{{Cite web |url=http://news.xkb.com.cn/guangzhou/2018/1231/526524.html|title=广州至汕头动车今日开通|access-date=2019-01-01}}</ref><ref>{{Cite web |url=https://tech.sina.com.cn/roll/2019-01-01/doc-ihqhqcis2061405.shtml|title=首趟动车开进汕头市区|access-date=2019-01-01}}</ref><ref name="22year"/>。

==始发车次==
汕头站是[[广梅汕铁路|广梅汕铁路]]的车站。每日办理旅客列车36列，其中始发列车17列<ref>{{Cite web |url=https://huoche.8684.cn/k_658|title=汕头火车时刻表|access-date=2019-01-01}}</ref>。當中一對特快終到[[廣州東站|廣州東站]]，一對快速经过[[梅州站|梅州站]]終到[[武昌站|武昌站]]；2014年7月1日調整運行圖，新增往来[[重慶北站|重慶北站]]之快速列車一對；2019年1月5日調整運行圖，新增往来[[广州东站|广州东站]]之动车组列車7.5對，往来[[深圳北站|深圳北站]]之动车组列車6.5對<ref>注：指来比往多一列</ref>。

春運、暑運期間，本站增開往武昌及九江之臨客列車。
