'''火斑鸠'''（[[學名|學名]]：''Streptopelia tranquebarica''），'''-{zh-cn:红鸠;zh-tw:火斑鳩;}-'''、'''紅斑鳩'''、'''斑鴿'''、'''红鴿鵻'''、'''火鸪鹪'''，棲息於[[亞洲|亞洲]]南部[[熱帶|熱帶]]區，為一種小型鳩鴿，常見於[[平原|平原]]、草地成群覓食。分布于印度、尼泊尔、不丹、孟加拉、缅甸、中南半岛、台灣、菲律宾以及[[中国大陆|中国大陆]]的自华北以南各地、西抵四川、[[西藏|西藏]]、长江以北各地等地，主要栖息于开阔田野以及村庄附近。该物种的模式产地在印度<ref name='id2054'/>。

== 体形 ==
[[File:Eurasian_Collared_Dove_(Female)_I_IMG_9674.jpg|thumb]][[哈里亞納邦|哈里亞納邦]][[費爾達巴德縣|費爾達巴德縣]][[荷達|荷達]]]]
成年紅鳩体长约23 cm，是[[鸠鸽科|鸠鸽科]]中体形较小的一种。成年雄鸟颈部为青灰色，颈后有黑色颈环，翅膀、胸腹和肩背为红褐色，胸腹部羽毛颜色浅於肩背部。成年雌鸟全身灰褐色，与[[灰斑鸠|灰斑鸠]]类似，且体形比雄鸟略小，腿部为深褐色。

== 分布 ==
紅鳩繁殖於[[中国|中国]]中部和[[中亚|中亚]]東部，是一种[[留鸟|留鸟]]，不过冬季有时也会群体性[[迁徙|迁徙]]到[[印度|印度]]、[[东南亚|东南亚]]和[[台湾|台湾]]。

== 亚种 ==
