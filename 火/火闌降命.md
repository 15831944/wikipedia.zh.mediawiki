'''火闌降命'''（{{jpn|j = 火闌降命|hg = ほのすそり<small>（或ほのすせり、ほすそり、ほすせり、ほのたけそり）</small> の みこと|rm = Ponotsutsoli no mikoto|kyu = |hgt = }}，'''火酢芹命'''、'''火闌命'''、火須勢理命）是[[日本|日本]]傳説中出現的人物，也是[[隼人|隼人]]等祖。又名'''火進命'''（{{jpn|j = 火進命|hg = ほのすすみ の みこと|rm = Ponotsutsumi no mikoto|kyu = |hgt = }}）、火照命（{{jpn|j = 火照命|hg = ほのてらす<small>（或ほのてる、ほのてり）</small> の みこと|rm = Ponotelatsu no mikoto|kyu = |hgt = }}）、'''海幸彥'''（{{jpn|j = 海幸彦|hg = うみのさちひこ/うみさちひこ|rm = Uminotsatipiko|kyu = 海幸彥|hgt = }}，海佐知毘古）。

== 名稱 ==
由於該人物在熊熊烈火中誕生，人名中的「火闌降（{{lang|ja|ほのすそり/ほのすせり/ほすそり/ほすせり/たけそり}}）」乃是形容如此景況。此外，「闌降（{{lang|ja|すそり/すせり/たけそり}}）」和[[須勢理姬神|須勢理姬神]]一樣具有「向前進」的意義。

== 事績 ==
=== 誕生 ===
==== 日本書紀之記載 ====
據《日本書紀》卷第二，[[天津彥彥火瓊瓊杵尊|天津彥彥火瓊瓊杵尊]]到達[[吾田長屋笠狹之碕|吾田長屋笠狹之碕]]時，遇到一名自號[[鹽土老翁|事勝國勝長狹]]之人，此人請天孫在該地任意遊之，又遇到一美人名喚[[鹿葦津姬|鹿葦津姬]]，亦名神吾田津姬、木花開耶姬。天孫得知此美人乃[[大山祇神|大山祇神]]之女，故[[人类性行为|臨幸]]之，而一夜有妊娠。天津彥彥火瓊瓊杵尊不相信鹿葦津姬一夜之間便有身孕，質疑祂與他人私通，後者忿恨之下自築一無戶室，並入居其內。祂發誓說：「妾所娠，若非天孫之胤，必當焦滅。如實天孫之胤，火不能害。」遂放火燒室，頭一個產下'''火闌降命'''，為[[隼人|隼人]]等之始祖；次生[[火折尊|彥火火出見尊]]；再生[[火明命|火明命]]，乃{{link-ja|尾張氏|尾張氏|尾張連}}等始祖。

接著第二種說法裡，[[神吾田鹿葦津姬|神吾田鹿葦津姬]]縱火燒房，火苗初點燃時產下'''火酢芹命'''，火盛時生[[火明命|火明命]]、再生[[火折尊|彥火火出見尊]]，又號火折尊。

第三種說法裡，[[神吾田鹿葦津姬|神吾田鹿葦津姬]]縱火燒房時先產下[[火明命|火明命]]、次生'''火進命'''，又號'''火酢芹命'''、復生[[火折尊|火折彥火火出見尊]]。神吾田鹿葦津姬以[[竹刀|竹刀]]割斷[[臍帶|臍帶]]，所丟棄的竹刀化成竹林，故地名曰[[竹屋|竹屋]]。後來神吾田鹿葦津姬以[[占卜|占卜]]方式決定哪座稻田出產之稻米獻給天神，名之為「狹名田」。所生之稻穗釀製天甜酒，所生之渟浪田稻煮成飯。

第五種說法提到天孫臨幸幸大山祇神之女[[吾田鹿葦津姬|吾田鹿葦津姬]]，一夜有身，遂生四子。吾田鹿葦津姬抱著四子晉見天孫，卻遭質疑與人私通，否則為何一夜有身孕。憤怒的吾田鹿葦津姬做無戶室，入居其內並發誓說：「妾所娠若非天神之胤者必亡。是若天神之胤者無所害。」遂放火燒屋，火燒剛開始時有一子說：「吾是天神之子，名[[火明命|火明命]]。吾父何處坐耶？」；火勢旺盛時另一子開口：「吾是天神之子，名'''火進命'''。吾父及兄何處在耶？」；火勢衰退時又一子出聲：「吾是天神之子，名[[火折尊|火折尊]]。吾父及兄等何處在耶？」；避火熱時最後一子說：「吾是天神之子，名[[火折尊|彥火火出見尊]]。吾父及兄等何處在耶？」最後吾田鹿葦津姬自灰燼中走出來，說道：「妾所生兒及妾身，自當火難，無所少損。天孫豈見之乎？」復對四子曰：「我知本是吾兒。但一夜而有身。慮有疑者，欲使衆人皆知是吾兒，并亦天神能令一夜有娠。亦欲明汝有靈異之威，子等復有超倫之氣。故有前日之嘲辭也。」。

第六種說法提及天孫[[天津彥根火瓊瓊杵根尊|天津彥根火瓊瓊杵根尊]]降至[[葦原中國|葦原中國]]，到[[吾田笠狹之御碕|吾田笠狹之御碕]]，遂登長屋之竹島。遇到[[鹽土老翁|事勝國勝長狹]]時詢問：「其於秀起浪穗之上，起八尋殿而手玉玲瓏織紝之少女者，是誰之子女耶？」長狹回答是：「[[大山祇神|大山祇神]]之女等。大號[[磐長姬|磐長姬]]，少號[[木花開耶姬|木花開耶姬]]、亦號豐吾田津姬，云云。」天孫因而臨幸豐吾田津姬，一夜有身，引起天孫懷疑。豐吾田津姬遂生'''火酢芹命'''、火折尊（亦號彥火火出見尊）等二子，並發誓這是天孫之後嗣。

其次第八種說法謂[[天忍穗耳尊|正哉吾勝勝速日天忍穗耳尊]]娶[[高皇產靈尊|高皇產靈尊]]之女[[栲幡千千姬命|天萬栲幡千幡姬]]，產下[[天火明命|天照國照彥火明命]]，是尾張連等遠祖；次生[[瓊瓊杵尊|天饒石國饒石天津彥火瓊瓊杵尊]]。後者娶[[大山祇神|大山祇神]]之女[[木花開耶姬|木花開耶姬命]]爲妃，生兒'''火酢芹命'''、[[火折尊|彥火火出見尊]]。

==== 古事記之記載 ====
根據《古事記》上卷記載，火照命是[[瓊瓊杵尊|邇邇藝命]]與[[木花開耶姬|神阿多都比賣]]（亦名謂木花之佐久夜毘賣）的兒子。邇邇藝命臨幸木花開耶姬而一夜有孕，後者對前者說：「妾妊身，今臨產時。是天神之御子，私不可產故請。」前者卻回答：「佐久夜毘賣，一宿哉妊。是非我子。必[[國神|國神]]之子。」後者接著說：「吾妊之子，若國神之子者，產不幸。若[[天神_(日本神話)|天神]]之御子者幸。」因此木花之佐久夜毘賣做了一座沒有門戶的八尋殿，走入其內並以黏土封閉。臨盆之際以火燃燒此殿，正當烈火熊熊燃燒時產下之子，名曰'''火照命'''（{{lang|ja|ほのてらす の みこと/ほのてる の みこと/ほのてり の みこと}}）；次生'''火須勢理命'''（{{lang|ja|ほのすせり の みこと}}）；復生[[火折尊|火遠理命]]（{{lang|ja|ほをり の みこと}}），又名天津日高日子穗穗手見命。一同在火光中降生，因此名字中都有「火」字。

==== 古事記之記載 ====
火照命（海幸彥）日日以釣魚為生，而他的兄弟[[火折尊|火遠理命（山幸彥）]]則以在山間打獵為生。後來雙方互換了工具（山海易幸），但山幸彥不慎將魚鉤遺失在海中。海幸彥十分生氣，刁難他，要求他找回魚鉤。山幸彥在[[鹽土老翁|鹽椎神]]的幫助下找到了魚鉤，並將魚鉤變成了受詛咒的魚鉤。海幸彥無法打到魚，進攻山幸彥。最終被山幸彥收服，子孫世世代代為山幸彥的後代看家護院。火照命者[[隼人|隼人]][[吾田氏|阿多氏]]之祖也。
