{{vgname|火影忍者 究极忍者风暴|ja=NARUTO－ナルト－ ナルティメットストーム|en=Naruto: Ultimate Ninja Storm}}为《{{tsl|en|Naruto: Ultimate Ninja|火影忍者 究极忍者风暴系列}}》的第一作，是一款[[格鬥遊戲|格鬥遊戲]]，由{{tsl|en|CyberConnect2|CyberConnect2}}开发，[[万代南梦宫娱乐|万代南梦宫娱乐]]於2008年11月发行。该游戏基于[[岸本齐史|岸本齐史]]原作、著名[[日本漫画|日本漫画]]和[[日本动画|动画]]系列《[[火影忍者|火影忍者]]》的剧情制作。

2010年10月，续作《{{tsl|en|Naruto Shippuden: Ultimate Ninja Storm 2|火影忍者疾风传 究极忍者风暴2}}》在日本、北美和欧洲的[[PlayStation_3|PlayStation 3]]和[[Xbox_360|Xbox 360]]平台上发行。<ref>{{cite web |url=http://www.ign.com/articles/2009/12/18/new-naruto-for-ps3-and-xbox-360 |title=New Naruto for PS3 and Xbox 360 |publisher=[[IGN|IGN]] |author=IGN Staff |date=2009-12-18 |accessdate=2015-07-24}}</ref>随后，续作《{{tsl|en|Naruto Shippuden: Ultimate Ninja Storm 3|火影忍者疾风传 究极忍者风暴3}}》和《[[火影忍者疾風傳_終極風暴4|火影忍者疾风传 究极忍者风暴4]]》分别在2013年<ref>{{cite web|last1=Staff|first1=I. G. N.|title=Naruto Shippuden Ultimate Ninja: Storm 2 Demo Now Available|url=http://www.ign.com/articles/2010/08/25/naruto-shippuden-ultimate-ninja-storm-2-demo-now-available|website=IGN|accessdate=2017-05-13|date=2010-08-25}}</ref>和2016年<ref name="TrueAchievements">{{cite web|url=http://www.trueachievements.com/n21178/naruto-shippuden-uns4-delayed.htm |title=Naruto Shippuden: UNS4 Delayed |publisher=True Achievements |date=2015-08-07 |accessdate=2015-08-27}}</ref>发行。

== 玩法与内容 ==

之前发行的游戏《究极忍者》系列中，许多角色在《火影忍者 究极忍者风暴》也保留了下来。然而，《究极忍者风暴》的游戏环境是[[3D|3D]]的，和前作《究极忍者》的2D游戏环境相反。<ref name="Official Ultimate Ninja Storm Web site">{{cite web |url=http://www.ultimateninjastorm.com/ |title=Naruto: Ultimate Ninja Storm Official Web site |publisher=[[万代南梦宫娱乐|Namco Bandai]] |accessdate=2008-07-15}}</ref>

此外，游戏中还加入了“觉醒模式”，当玩家在战斗中失去了一定的生命值后就会激活该觉醒模式，激活该模式所需要失去的生命值取决于模式的强大程度。觉醒完成后。角色的能力、速度、攻击力都将翻上一番。游戏中某些角色在觉醒后的移动方式和觉醒前完全不同。玩家还可利用预备好的物资对敌人造成伤害、激活状态等。玩家还可以自定义角色可使用的忍术，并指定两个辅助角色在对抗中支援自己。此外，游戏还有“合击忍术”模式，需要两位玩家同时发动特殊攻击。<ref name="Preview 2">{{cite web |url=http://www.ign.com/articles/2008/06/30/naruto-ultimate-ninja-storm-hands-on |title=Naruto: Ultimate Ninja Storm Hands-on |publisher=IGN |last=Clements |first=Ryan |date=2008-06-03 |accessdate=2015-07-24}}</ref>此外，每个角色都有自己的“究极忍术”，能为对手造成极大伤害。

在游戏中进行战斗时，玩家操控战斗角色的技术也会影响到对决结束时的奖励加成（例如玩家以战斗角色的“'''究极忍术'''”解决对手完成对局时奖励结算会加成）。总之玩家若能以最少时间、最多[[生命值|生命值]]剩余结束对局，游戏评价就越高（由低到高分别为DCBAS）奖励结算也会随之增加，这样的设计不仅可以激起玩家们的[[挑战心理|挑战心理]]，很大程度上也增加了游戏的[[可玩性|可玩性]]。

游戏包含25个可玩角色，全部可在战斗中用作辅助角色。<ref name="Official Ultimate Ninja Storm Web site" />游戏后来右追加了十个额外的纯辅助角色，可在游戏发行后五月提供的[[追加下载内容|追加下载内容]]中免费解锁。<ref>{{cite web |url=http://www.gaming-age.com/news/2008/10/29-51 |title=Namco Bandai announces free downloadable content for Naruto: Ultimate Ninja Storm |publisher=Gaming Age |last=Cordeira |first=Jim |date=2008-10-29 |archiveurl=https://web.archive.org/web/20081102205151/http://www.gaming-age.com/news/2008/10/29-51 |archivedate=2008-11-02 |accessdate=2015-07-24}}</ref>此外，追加下载内容还增加了不同的服装、新任务等等。<ref name="Preview 1">{{cite web |url=http://www.ign.com/articles/2008/04/17/naruto-ultimate-ninja-storm-preview |title=Naruto: Ultimate Ninja Storm Preview |publisher=IGN |last=Clements |first=Ryan |date=2008-04-16 |accessdate=2015-07-24}}</ref>

游戏内容大约覆盖了一直到动画剧情135集的位置。<ref name="radar interview" />此外，玩家可在任务期间探索作为任务中转站的{{tsl|en|Hidden Leaf Village|木叶村}}（也是火影忍者原作故事的主要发生地点）。<ref name="Preview 1" />

== 制作 ==

[[File:Screenshot-Naruto_Ultimate_Ninja_Storm_1.png|thumb]]，看起来很接近电视动画的效果。（图为中忍考试中，[[日向宁次|日向宁次]]对战[[日向雏田|日向雏田]]的场景。）]]

游戏图像均经过[[卡通渲染|卡通渲染]]，“将会打破动画与游戏之间的壁障”。<ref name="Release">{{cite web |url=http://www.namcobandaigames.com/games/narutoultimateninjastorm/ |title=Naruto Ultimate Ninja Storm - Namco Bandai Games America Inc |publisher=Namco Bandai |accessdate=2008-08-07}}</ref>游戏主创人员之一的松山浩评论说，制作人员一直在尝试着移除游戏与动画之间的边界。他们一直在试着让玩家感觉，自己实际上是在观看动画，而不是在玩游戏。游戏的核心概念是一对一战斗——虽然游戏的故事剧情基于火影忍者动画前135集，开发人员还是基于故事核心场景制作了多个一对一的战斗。<ref name="radar interview">{{cite web |url=http://www.gamesradar.com/naruto-ultimate-ninja-storm-2/ |title=Naruto: Ultimate Ninja Storm (Interview) |publisher=[[GamesRadar|GamesRadar]] |last=Antista |first=Chris |date=2008-04-18 |accessdate=2015-07-24}}</ref>

== 发行 ==

游戏最开始在2007年9月以“火影忍者PS3计划”的名义公布。<ref>{{cite web|title=次世代火影現身！PS3《火影忍者》最新作首部宣傳影片釋出|url=https://gnn.gamer.com.tw/5/28915.html|website=巴哈姆特電玩資訊站|accessdate=2017-05-13}}</ref>万代南梦宫宣布，游戏名称将会以“火影忍者：究极忍者”这几个字开头，并向广大粉丝征求意见，需要在此基础之上再加入一个、两个词构成的后缀。<ref>{{cite web |url=http://www.hokagesroom.com/blog/view/the-hokages-scroll/help-name-the-naruto-ps3-game/2008/02/25/ |title=The Hokage's Scroll: Help Name the Naruto PS3 Game! |publisher=hokagesroom |author=the_hokage |date=2008-02-15 |archiveurl=https://web.archive.org/web/20081201181512/http://www.hokagesroom.com/blog/view/the-hokages-scroll/help-name-the-naruto-ps3-game/2008/02/25/ |archivedate=2008-12-01 |deadurl=yes |accessdate=2015-07-24}}</ref>2008年4月，官方正式敲定名称为“火影忍者 究极忍者风暴”。此外，官方还允许粉丝们从六个不同的游戏封面候选中选一个，在最终发行的时候使用。<ref>{{cite web |url=http://www.engadget.com/2008/04/05/naruto-ps3-game-named-needs-box-art/ |title=Naruto PS3 game named, needs box art |publisher=[[Engadget|Engadget]] ([[Joystiq|Joystiq]]) |last=Yoon |first=Andrew |date=2008-04-05 |accessdate=2015-07-24}}</ref>

2008年7月17日，索尼[[PlayStation_Network|PlayStation Network]]上放出了一个可玩的游戏演示，演示中只有一个游戏场景，以及[[漩渦鳴人|漩渦鳴人]]和[[旗木卡卡西|旗木卡卡西]]两个可选角色。<ref>{{cite web |url=http://www.gamespot.com/articles/naruto-ultimate-ninja-storm-hands-on-ps3-demo-graphics-and-gameplay/1100-6193129/ |title=Naruto Ultimate Ninja Storm Hands-On - PS3 Demo, Graphics, and GamePlay |publisher=[[GameSpot|GameSpot]] |last=Torres |first=Ricardo |date=2008-06-27 |accessdate=2015-07-24}}</ref>

对于在指定零售商处预定的玩家，《究极忍者风暴》还有限定版。它增加了金属书架、两个限定版《火影忍者》卡牌游戏的卡牌、音轨的CD，以及带有独立编号的[[漩渦鳴人|漩渦鳴人]]和[[宇智波佐助|宇智波佐助]]的“Laser Cel”。<ref>{{cite web |url=http://www.ign.com/articles/2008/09/18/namco-bandai-games-america-reveals-naruto-ultimate-ninja-storm-limited-edition |title=Namco Bandai Games America Reveals Naruto: Ultimate Ninja Storm Limited Edition |publisher=IGN |author=IGN staff |date=2008-09-18 |accessdate=2015-07-24}}</ref>2008年11月7日，游戏在欧洲发行。11月20日，游戏在澳大利亚发行。日版在2009年1月15日以“{{nihongo|''Naruto: Narutimate Storm''|ＮＡＲＵＴＯ－ナルト－ナルティメットストーム|Naruto: Narutimetto Sutōmu}}”的名称发行。

游戏音乐被收录在一张光盘中，包含18个音轨，由[[福田考代|福田考代]]创作。

总体来看，《火影忍者 究极忍者风暴》收獲了正面的评价，在[[GameRankings|GameRankings]]和[[Metacritic|Metacritic]]两个网站上平均评价分数为75%。<ref name="gamerankings">{{cite web |url=http://www.gamerankings.com/ps3/943434-naruto-ultimate-ninja-storm/index.html |title=Naruto: Ultimate Ninja Storm for PlayStation 3 |publisher=[[GameRankings|GameRankings]] |accessdate=2010-06-25}}</ref><ref name="metacritic">{{cite web |url=http://www.metacritic.com/game/playstation-3/naruto-ultimate-ninja-storm |title=Naruto: Ultimate Ninja Storm for PlayStation 3 |publisher=[[Metacritic|Metacritic]] |accessdate=2015-07-24}}</ref>

[[IGN|IGN]]打分为8.4份（满分10分），称赞其拥有“非凡的画质”，任务与任务之间也有不少可以探索的事情。评价还提到，游戏中充满了一味的普通攻击，可能会让硬核玩家感到不快，但是快速的动作、查克拉系统却又让人-{欲}-罢不能。<ref name="IGNR" />[[GameSpot|GameSpot]]打出了7.5分（满分10分），称游戏不差，但也不是特别好。<ref name="GSR">{{cite web |url=http://www.gamespot.com/reviews/naruto-ultimate-ninja-storm-review/1900-6200864/ |title=Naruto: Ultimate Ninja Storm Review |publisher=GameSpot |last=Calvert |first=Justin |date=2008-11-10 |accessdate=2015-07-24}}</ref>[[1UP.com|1UP.com]]给予其B−的评价，称赞游戏画面很棒，但是游戏系统有些地方还是有点问题。<ref name="1UPR">{{cite web |url=http://www.1up.com/reviews/naruto-ultimate-ninja-storm |title=Naruto: Ultimate Ninja Storm Review |publisher=[[1UP.com|1UP.com]] |last=Li |first=Richard |date=2008-11-06 |accessdate=2015-07-24}}</ref>
