'''火箭万源站'''位于[[北京市|北京市]][[丰台区|丰台区]][[东高地街道|东高地街道]]，是[[北京地铁8号线|北京地铁8号线]]的一座车站。车站于2018年12月30日随8号线[[珠市口站|珠市口]]至[[瀛海站|瀛海]]段开通而投入使用<ref>{{cite news|url=http://www.bj.xinhuanet.com/bjyw/2018-12/30/c_1123927578.htm|title=两条地铁新线开通试运营|publisher=[[北京日报|北京日报]]|date=2018-12-30|accessdate=2018-12-30}}</ref>。

== 位置 ==
火箭万源站位于[[北京四环路|南四环]]外，[[南大红门路|南大红门路]]（[[104国道|104]]、[[105国道|105国道]]，南北向）和[[万源南路|万源南路]]（由此向东）交汇处，呈南北走向布置<ref>{{cite web|url=http://ghgtw.beijing.gov.cn/searches/detailForDiMingGongGao?id=8045|title=火箭万源站|date=2018-07-17|accessdate=2018-10-22|publisher=北京市规划和国土资源管理委员会}}</ref>。车站以东为万源庄，以西为[[中国运载火箭技术研究院|中国运载火箭技术研究院]]，西南为六营门。

== 命名 ==
本站规划建设阶段曾名'''六营门站'''。在2018年5月[[北京市规划和国土资源管理委员会|北京市规划和国土资源管理委员会]]对车站名称的公示中，本站提供了两个备选站名，其中一个是“六营门站”，另一个是“火箭万源站”，公示结束后决定采用“火箭万源站”。“火箭万源站”的名称来自于“火箭院”（即指中国运载火箭技术研究院）和万源庄，并取中国航天事业发祥地之意。<ref>{{cite news|url=http://www.bjnews.com.cn/news/2018/05/29/488804.html|title=轨道交通8号线6号线发布15座新车站命名预案|work=新京报|date=2018-05-29|accessdate=2018-10-22|author=邓琦, 余华尊}}</ref><ref>{{cite news|url=http://news.sina.com.cn/o/2018-06-21/doc-ihefphqk5756088.shtml|title=8号线六营门站采信为火箭万源站|work=北京青年报|date=2018-06-21|author=李天际|accessdate=2018-10-22}}</ref>

== 结构 ==
火箭万源站为地下二层车站，上层为站厅层，下层为[[岛式站台|岛式站台]]。<ref>{{cite news|url=http://www.xinhuanet.com/local/2018-03/15/c_1122540033.htm|title= 北京8号线六营门站主体施工完成 抬头能望“蓝色星空” |date=2018-03-15|accessdate=2018-10-22|work=北京青年报|author=李天际}}</ref>

由于靠近运载火箭技术研究院，本站的装饰以星空为主题。<ref>{{cite news|url=http://bjcb.morningpost.com.cn/html/2018-09/07/content_501039.htm|title= 8号线珠市口至瀛海站年底开通|date=2018-09-07|accessdate=2018-10-22|work=北京晨报|author=曹晶瑞}}</ref>
