{{Nihongo title|火箭人|ロケットマン}}是[[加藤元浩|加藤元浩]]在《[[月刊少年Magazine|月刊少年Magazine]]》雜誌（[[講談社|講談社]]）上連載的少年推理冒險漫畫作品。

中文版書名譯名為'''火箭人'''，由[[東立出版社|東立出版社]]代理出版。

== 簡介 ==
該作品於2001年11月至2005年1月期間在《月刊少年Magazine》上連載，單行本全10冊。

故事由'''水無葉'''和'''長月彌生'''兩位角色的觀點展開：描述兩人所遭遇的不可思議神秘事件，並由男主角水無葉負責解決。

與加藤元浩的代表作《[[神通小偵探|神通小偵探]]》一樣，內容多採一回完結的形式。 （由於原刊載雜誌為月刊，故每回連載均有約50頁的份量。）

== 登場人物 ==
; 水無葉（13嵗）
: 主角。在故事中擔任解題偵探的角色。普通中学生，對自己小時候的記憶空白。
: 因為其祖父是醫生，所以有豐富的醫學知識。
; 長月弥生（13嵗）
: 女主角。在故事中擔任偵探的助手。
: 水無葉的的同班同學，是少數能理解他的人。是個個性活潑開朗的女孩，很受男生歡迎。
; R
: 屬於間諜組織的超級情報員。
