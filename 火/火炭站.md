'''火炭站'''（{{lang-en|'''Fo Tan Station'''}}）是[[港鐵|港鐵]][[東鐵綫|東鐵綫]]的鐵路車站，位於[[香港|香港]][[新界|新界]][[沙田區|沙田區]][[火炭|火炭]]，於[[九廣鐵路|九廣鐵路]]－英段電氣化後才開設，車站取名於附近的[[火炭|火炭]]村，是一個地面車站。車站東邊為[[火炭鐵路大樓|火炭鐵路大樓]]、[[港鐵車廠#何東樓車廠|何東樓車廠]]，其上蓋發展為多個私人住宅發展項目，包括[[御龍山|御龍山]]、[[銀禧花園|銀禧花園]]和[[駿景園|駿景園]]。

本站與[[馬場站_(香港)|馬場站]]大致平行，列車若停靠本站就'''無法'''停靠馬場站。

===大堂===
{{ multiple image | direction = vertical | align = right | width = 220
| image1 = Fo Tan Station South Concourse 201801.jpg | caption1 = 南面大堂
}}
車站有兩個大堂，分別為連接A、B出入口的大堂（南）與連接C、D出入口的大堂（北）。兩個大堂之已付費區於2015年12月12日連通，在此以前只可透過車站外的樂景街步行來往。

====商店及自助服務====
火炭站南、北2個大堂現時分別設有1間[[便利商店|便利店]]和1間[[麵包糕餅店|麵包糕餅店]]，另外亦設有自助服務包括[[自動櫃員機|自動櫃員機]]及[[自動販賣機|自動售賣機]]<ref name="shop"/>。

===月台===
{{ multiple image | direction = vertical | align = right | width = 220
| image1 = Fo Tan Station 2017 08 part1.jpg | caption1 = 備用中央路軌
}}
車站設有4個月台，採用兩個[[島式月台|島式月台]]排列，而2號及3號月台共用同一條路軌。火炭站空間有限，狹窄的月台島只能容納一條單向[[電動扶梯|扶手電梯]]，<ref>[http://mtr.hk365day.com/images/ERL/ss/FOT.png MTR之今昔]</ref>當有城際客運列車通過火炭站，北行列車會於1號月台待避，而南行列車會於4號月台待避，但現時已甚少作出此安排，絕大多數列車於沙田站進行待避。惟當有北行列車需要進入何東樓車廠或馬場站，停靠火炭站的南行列車將會使用3號月台上落客。而於火炭站開出的南行特別班次列車也會使用4號月台。

==鄰接車站==
火炭站是東鐵綫的中途站，乘客可在此乘坐往[[紅磡站|紅磡]]或[[羅湖站_(香港)|羅湖]]／[[落馬洲站|落馬洲]]列車，往返[[九龍|九龍]]及[[新界|新界]]等地。
{{東鐵綫}}

==利用狀況==
火炭站西北邊為火炭工貿區，另一邊為車廠上蓋住宅群，除了上下班時間，其餘時間人流屬一般。該站也鄰近[[香港專業教育學院沙田分校|香港專業教育學院沙田分校]]，全日均有師生利用本站上學和上班。

在[[沙田馬場|沙田馬場]]舉行賽事時，由於不是每班東鐵綫列車都會停靠[[馬場站_(香港)|馬場站]]，而停靠馬場站的班次也不是十分頻密，而且車費較高，故部分前往沙田馬場的乘客會在火炭站下車步行前往，也為本站帶來一些額外客流量。

==歷史==
===建設及啟用===
火炭站的由來可追溯至1967年《香港集體運輸研究》中提及的[[禾寮坑|禾寮坑]]站，該站計劃為[[香港地鐵|香港地鐵]][[東九龍綫_(1970年方案)#沙田綫（1967年）|沙田綫]]的終點站及與[[東鐵綫|九廣鐵路（英段）]]的[[換乘站|轉乘站]]。但沙田線計劃最終擱置。後來，因應火炭工業區的發展，九廣鐵路改為在禾寮坑以南之火炭（火炭原名「河灘」，[[客家|客家]]村落）建設車站，於1985年2月15日九廣鐵路（英段）電氣化之後才啟用。

===增建大堂===
啟用初期車站只有南面大堂及A、B出入口，後來為方便乘客來往駿景園、銀禧花園及沙田馬場及坳背灣街，九鐵於1990年代中耗資9000萬港元，增建了北面大堂及C、D出入口<ref>{{cite web|url=http://www.districtcouncils.gov.hk/archive/st_d/pdf/ttc/2011/ST_TTC_2011_068_TC.pdf|title=文件TT68/2011|publisher=沙田區議會|date=2011年9月6日|accessdate=2012年2月19日|language=zh-hant}}</ref>。新大堂的面積約為1,000平方米，設有售票處（客務中心）、4條扶手電梯、10部出入閘機、3部售票機、八達通增值機及4間商店。九鐵於2000年1月為北面大堂舉行啟用典禮，並由時任沙田政務專員黃美蓮、[[沙田區議會|沙田區議會]]主席[[韋國洪|韋國洪]]、東鐵副總監李殷泰及東鐵工程總經理區柏年主持剪綵及揭幕儀式。經擴建後，九鐵估計火炭站每日可處理9萬6千人次的乘客。<ref>《火炭車站新大堂正式啟用》(東鐵傳真，2000年2月，第86期第三頁，九廣鐵路公司)</ref>

===東鐵綫月台翻新工程===
兩鐵合併後，港鐵為東鐵綫各月台進行低廉的翻新工程，以美化殘舊的月台。工程包括重新為各車站配上主題色，並加入代表花卉，為月台柱樑翻上月台配色的油漆，貼上印有車站主題花的圖案、車站周圍景物、站名書法字的牆紙，加設及更新直立站牌、路線圖、指示牌、告示等等。該站配色為橙色，主題花為木棉，街景為火炭工業區中心。

===翻新車站===
[[File:Fo_Tan_Station_Concourse_new_access_201801.jpg|220px]]
港鐵於2013年7月起斥資1.1億元翻新車站，現有兩個獨立大堂之間，將會加建大堂連接，令兩個原本分開的已付費區合而為一，工程於2015年12月12日完成後，經擴大的大堂為乘客提供更方便和舒適的乘車環境，大堂上方會設置一個大型天窗，引入自然光，減低耗電。為配合工程，火炭站需輪流封閉一條路軌，以便加建臨時工作-{台}-，每日三班由火炭站開出的特別班次會改由沙田開出，港鐵指會把工程對乘客的影響減到最低。<ref>《[http://www.mtr.com.hk/eng/corporate/file_rep/PR-13-058-C.pdf 港鐵火炭站車站美化工程 車站大堂二合一]》(港鐵公司新聞稿058/13號，2013年7月9日)</ref>

==昔日事故==
2011年2月20日（星期日）約上午8時，一列東鐵綫列車由[[大學站_(香港)|大學站]]前往[[紅磡站|紅磡站]]列車，駛入火炭站2月台前，疑因跳掣出現電力故障，部份車廂停電，車頂冒出火花，集電弓部份零件遭燒熔，消防員一度到場戒備。由於正值上班非繁忙時間，車廂滿載乘客，有乘客指，當時有廣播叫乘客向車頭方向行落車，港鐵安排特別車接載乘客，肇事列車其後被移至馬場站1號月台搶修。事發後，東鐵綫南行列車，駛離大學站後要繞往[[火炭站_(香港)|火炭站]]，前往馬場站的乘客則要在[[沙田站|沙田站]]1號月台，轉乘北行列車前往，直至8時35分恢復正常。

2015年9月29日晚上約6時25分，火炭站近D出口一部扶手電梯，突然聽見「轟」一聲，電梯即告煞停。起初以為有乘客誤觸緊急掣，豈料電梯再往後溜，幸速度與平時上行相若，乘客能及時抓緊扶手站穩，避免「人肉骨牌」意外，最終電梯溜後十多米始停下。[[機電工程署|機電工程署]]證實曾派員到現場翻查[[閉路電視|閉路電視]]，確定有關扶手電梯確一度停頓，並轉向下行約廿秒後再停頓，將就事件進行詳細調查，更指署方對港鐵未確定事故原因便恢復涉事扶手電梯運作感失望。<ref>[http://orientaldaily.on.cc/cnt/news/20151001/00176_073.html 上行電梯突溜後 火炭站險人疊人 《東方日報》 2015年10月1日]</ref>
