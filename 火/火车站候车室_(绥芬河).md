'''火车站候车室'''<ref>{{Cite web |url=http://www.sohu.com/a/259497231_166414 |title=开着汽车，沿着黑龙江中东铁路去旅行|accessdate=2018-10-31 |author=|date=2018-10-15 |work= |publisher=搜狐网，摘自《汽车自驾游》杂志2018年10月|language=Zh-hans }}</ref>或称'''绥芬河老火车站候车室'''<ref name="a1">{{Cite web |url=http://aixin.dpcm.cn/Article/TypeArticle.asp?ModeID=1&ID=3984 |title=绥芬河老建筑的历史印记（二）|accessdate=2018-11-06 |author=|date=2017-02-16 |work= |publisher=大鹏新闻网|language=Zh-hans }}</ref>，位于[[黑龙江省|黑龙江省]][[绥芬河市|绥芬河市]][[绥芬河镇|绥芬河镇]]站前路38号<ref name="a2">{{Cite web |url=http://travel.sina.com.cn/china/2014-04-04/1909256231.shtml |title=那些历史遗留下的痕迹 寻绥芬河百年老建筑|accessdate=2018-10-31 |author=|date=2014-04-03 |work= |publisher=新浪网|language=Zh-hans }}</ref>，是绥芬河老火车站的候车室。[[中东铁路建筑群|中东铁路建筑群]]扩展项目之一。

候车室为[[俄罗斯式建筑|俄罗斯式]]两层楼房，南北走向，西通铁路站台，东门临街<ref name="a1"/>。约40米长，依山而建。楼房建于1902年，建筑面积1342平方米<ref name="a2"/>。绥芬河老站建立成时为[[中东铁路|中东铁路]]的九个二等站之一。与其它车站类似，老站采用定型化设计<ref name="b1"/>。候车室建筑结构采用钢铁拱梁，是其在中东铁路现存建筑中最为特别之处，它是唯一的室内大跨度钢梁屋架站舍。曾在1929年的[[中东路事件|中东路事件]]中受毁，后修复为现状<ref name="a2"/>，仍保留原有建筑风格。西侧正门上方有阳文汉字“绥芬河”，下方对应俄文“鲍库拉尼奇那亚”<ref name="a3"/>。由于年代久远，现建筑地面远低于室外路面<ref name="a1"/><ref name="a2"/>。

建筑外墙具有[[新艺术运动|新艺术运动]]建筑风格，强调垂直划分，采用壁柱装饰，[[女儿墙|女儿墙]]高低分布。[[山墙|山墙]]、窗户使用曲线构图<ref name="b1">{{cite book |author=刘孛、刘大平 |date=2015-07-31 |title=2010年中国近代建筑史国际研讨会论文集 |url=http://www.wanfangdata.com.cn/details/detail.do?_type=conference&id=8511508 |dead-url= |location= |publisher=中国建筑学会、清华大学 |page=390-396,678|isbn= |archive-url= |accessdate=2018-10-31 |chapter=《中东铁路沿线站舍建筑特色研究》|archive-date=|language = Zh-hans}}</ref>。窗楣则采用俄式图形装饰。现室内大门、内饰、浮雕仍是百年前的建筑原物<ref name="a3">{{Cite web |url=http://news.ifeng.com/a/20150825/44509531_0.shtml |title=百年绥芬河火车站 中东铁路的记忆（组图）|accessdate=2018-10-31 |author=记者：刘振山|date=2015-08-25 |work= |publisher=凤凰网，来源：长城网|language=Zh-hans }}</ref>。2015年时，外墙颜色为蓝白相间<ref name="a2"/>。2017年，澎湃新闻报道时，外墙颜色已改为黄白相间<ref name="a4">{{Cite web |url=https://www.thepaper.cn/newsDetail_forward_1793091 |title=探访中东铁路⑥/牡丹江：弯弯曲曲的历史景观|accessdate=2018-11-06 |author=特约撰稿：马特|date=2017-09-22 |work= |publisher=澎湃新闻|language=Zh-hans }}</ref>。

1992年，在老站南侧新建国际旅客候车室<ref name="a1"/>。此候车室一直做为老站的国内旅客候车室<ref name="a2"/>，至2015年12月，老站因[[绥芬河站|绥芬河新站]]启用而停用。1999年1月10日，[[黑龙江省人民政府|黑龙江省人民政府]]以'''绥芬河火车站候车室'''之名列入[[第四批黑龙江省文物保护单位|第四批黑龙江省文物保护单位]]。2013年，做为[[中东铁路建筑群|中东铁路建筑群]]项目，以'''火车站候车室'''之名列入[[第七批全国重点文物保护单位|第七批全国重点文物保护单位]]（[[第七批全国重点文物保护单位中的近现代重要史迹及代表性建筑|近现代重要史迹及代表性建筑]]）。
