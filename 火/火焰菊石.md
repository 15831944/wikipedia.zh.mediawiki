'''火焰菊石'''（[[學名|學名]]：{{lang|la|'''''Metalegoceras'''''}}），又名'''伴臥菊石'''，是生存於[[二疊紀|二疊紀]]的一[[屬|屬]][[菊石|菊石]]。其外殼具有火焰狀的縫合線，因而得名。火焰狀縫合線的特徵為：腹側具有兩岔狀的外葉；側面則具有兩個火焰狀的側葉。<ref name="Metalegoceras">{{Cite web |url = http://liusir.pixnet.net/blog/post/248637173-metalegoceras-%E7%81%AB%E7%84%B0%E8%8F%8A%E7%9F%B3-%28%E5%B8%9D%E6%B1%B6%29 |title = Metalegoceras 火焰菊石 (帝汶) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-13 }}</ref>其[[化石|化石]]被發現於[[中亞|中亞]]、[[俄羅斯|俄羅斯]]、[[北美|北美]]、[[澳大利亞|澳大利亞]]、[[印度尼西亞|印度尼西亞]]、[[中國|中國]]及[[帝汶島|帝汶島]]等地。<ref name="Fossilworks" />
