'''火焰雞'''（{{lang-ja|'''バシャーモ'''}}，{{lang-en|'''Blaziken'''}}），是[[精靈寶可夢|精靈寶可夢]]系列登場的{{NUMBEROFPOKEMON}}種虛構[[角色|角色]]（[[怪物|怪獸]]）中的一種。

== 特徵 ==
火焰雞為火屬性的猛火寶可夢。額頭上有著V字型的雞冠，主要顏色為紅色，胸口與腳部有一部份為黃色，羽毛數年會脫落一次，然後長出新的羽毛。

手的[[爪|爪]]子相當尖銳，對戰時能夠從手腕發出[[火焰|火焰]]，並有能跳過30層樓的腿力，對格鬥招式相當擅長。

== 遊戲中的火焰雞 ==
為[[神奇寶貝_紅寶石·藍寶石|紅寶石、藍寶石、綠寶石版]]中最初拿到的寶可夢[[火稚雞|火稚雞]]的最終進化型，由[[力壯雞|力壯雞]]在36級時進化，特性為猛火。沒有野生種。豐緣聯盟編號為6，全國圖鑑編號為257。由XY版開始，可用超級石超級進化成超級火焰雞。

== 動畫中的火焰雞 ==
*城都聯盟白銀大會上，小智的勁敵葉越所持有的其中一隻寶可夢，有著相當強大的力量，在與小智交手時，打敗了小智的[[噴火龍|噴火龍]]。
*在《[[神奇寶貝超世代|神奇寶貝超世代]]》第190話中，小遙的[[力壯雞|力壯雞]]為了不讓寶可夢被火箭隊搶走而努力戰鬥，並進化成火焰雞。在寶可夢華麗大賽梣木大會上與小智的[[蜥蜴王|蜥蜴王]]打成平手。
*在《[[神奇寶貝鑽石&珍珠|神奇寶貝鑽石&珍珠]]》第75話中，為了能在七星級七星飯店用餐，小遙的火焰雞與小光的[[波加曼|波加曼]]合作打雙打對戰，後贏得勝利。

== 電影中的火焰雞 ==
《[[神奇寶貝劇場版：裂空的訪問者_代歐奇希斯|裂空的訪問者 代歐奇希斯]]》中，訓練家阿龍所使用的寶可夢。
