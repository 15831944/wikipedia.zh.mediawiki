'''火鍋'''是以水或湯導熱锅来涮煮食物的烹調方式。其特色為邊煮邊吃，由于鍋本身具有保溫效果，因此吃的時候食物仍熱氣騰騰。主要在東亞地區盛行。

典型的火鍋食材，包括： 火鍋料、海鮮、泡麵、肉片、豆腐、香菇、丸子、冬粉、蔬菜等等，將其放入煮開的鍋底{{efn|有些地方稱之為“湯底”或“湯頭”。}}燙熟後食用。有些吃法還會蘸上調味料{{efn|依地方習慣又稱為小料、醬料、沾醬。}}。

== 歷史 ==
火鍋的起源至今尚無定論。但最初人類的飲食文化便有以火燒鍋，以水導熱，煮熟食物的烹調方法。

[[商|商]][[周|周]]時期有一種容器[[鼎|鼎]]<ref>徐文苑：《中国饮食文化概论》。北京：清华大学出版社，2005年：51-52頁。ISBN 978-7-81082-370-8</ref> 在進行祭祀或慶典時，就要「鳴鐘列鼎」，將[[牛|牛]]、[[羊肉|羊肉]]等食材通通都放入鼎中，然後在底部生火把食物煮熟後再分食，是火鍋的雛形。

在[[秦|秦]][[漢|漢]]時期，把[[雞肉|雞肉]]、[[豬肉|豬肉]]等放在沸水中稍煮一下，稱之為「濯」，[[馬王堆|馬王堆]]漢墓遣策上就有「濯雞」及「濯豚」等文字記載,這和今天火鍋吃法簡直如出一轍。

[[三國|三國]]時期，[[曹丕|曹丕]]將銅鍋分為五格，名為[[五熟釜|五熟釜]]，可用不同的湯料涮煮不同的食物，與現在的鴛鴦火鍋有異曲同工之妙。

到了[[南北朝|南北朝]]时期，出現易於熟食的[[銅爨|銅爨]]，是當時居住在[[重庆|重庆]]、[[湖南|湖南]]一帶的[[獠|獠]]人所使用的一種銅火鍋。

到了[[唐朝|唐朝]],已經有銅製的暖鍋。1984年在[[內蒙古|內蒙古]][[赤峰市|赤峰市]][[敖漢旗|敖漢旗]]出土了一幅[[契丹|契丹]]人在[[穹廬|穹廬]]中圍著火鍋席地而坐的壁畫，據專家考證，這幅畫正畫的是[[遼代|遼代]]初期契丹人吃涮羊肉的情景。

[[南宋|南宋]][[林洪|林洪]]在《山家清供》一書中記載了火锅。它的吃法是先將兔肉切薄片，用[[料酒|料酒]]、[[醬油|醬油]]和[[花椒|花椒]]稍腌，然後将湯燒沸將肉片汆熟，蘸[[佐料|佐料]]而食<ref>林洪《山家清供》：“师云：山间只用薄批，酒酱、椒料活之。以风炉安桌上，用水半铫，候汤响一杯后，各分以箸，令自夹入汤摆熟，啖之，乃随意各以汁供。”</ref>。

[[元代|元代]]时，火鍋大為興盛，其中的「生爨羊」即是現今涮羊肉的前身。[[明代|明代]]官食食書《宋氏養生部》中，也有「生爨羊」、「熟爨牛」等火鍋菜式。

[[清朝|清朝]][[乾隆|乾隆]]年間，火鍋不僅在民間盛行，连皇帝也喜愛火鍋，而且成了宮廷菜肴。乾隆曾经在[[乾清宮|乾清宮]]辦了530桌火鍋席以宴請宗室；[[嘉慶|嘉慶]]皇帝登基時曾經舉辦[[千叟宴|千叟宴]]，動用[[銀|銀]]質、[[錫|錫]]質和[[銅|銅]]質的火鍋1,550個。

目前全中國共有60萬家火鍋店，已經成為中國飲食行業的一大支柱，每年收入逾5000億，可見其普及程度非同小可。<ref>[https://hk.finance.appledaily.com/finance/realtime/article/20190118/59154417"【火鍋保衞戰】中國有60萬家火鍋店　年收入逾5000億元"]</ref>

==種類==

火鍋基本上可以分為三大類別：
# 鍋內無料或料不多，而以涮生食材為主，其中又分兩種:1)湯底清淡，沾料占重要角色，以涮羊肉及打甂爐為代表；2)湯底濃厚，如麻辣火鍋。
# 鍋內食材已熟，如魚頭火鍋、羊肉爐等，爐火只是用來保溫及汆燙青菜。
# 鍋內食材全都先煮熟，連青菜也無需再汆燙，爐火完全是用來保溫的，如[[燒酒雞|燒酒雞]]、[[薑母鴨|薑母鴨]]、[[復興鍋|復興鍋]] 等；這種火鍋也稱為“煲”。

== 各地火鍋 ==

=== 中國北方 ===

====北京====

* [[涮羊肉|涮羊肉]]
*: 相傳由[[蒙古|蒙古]]軍隊发明，所以又稱為「蒙古火鍋」<ref>Frederick J. Simoons, "Food in China: A Cultural and Historical Inquiry", page 47. CRC Press, 1991. ISBN 978-0-8493-8804-0</ref>。吃法是將[[羊肉|羊肉]]、蔬菜、豆腐等食材用清湯涮熟後，沾芝麻酱等調料食用。由於羊肉加熱易熟，所以用開水短時間的一涮即可。

====东北====

* 殺豬菜(似酸菜白肉鍋)
*: 主材料為酸[[白菜|白菜]]與[[豬|豬]]的[[五花肉|五花肉]]，是[[滿洲|滿洲]]地區民眾的家庭傳統菜餚，主要流行於[[中國東北|中國東北]]，後傳入北京。做法是使用生炭火的銅鍋，將酸白菜與高湯煮成鍋底，白肉下锅涮熟，佐麻油、生抽、蒜泥等食用<ref>关云德。满族的火锅。满族文学，2003年第4期：64頁。</ref><ref>徐文苑。中国饮食文化概论。北京：清华大学出版社，2005年：第10頁。ISBN 978-7-81082-370-8。</ref>。

====河南====

* [[红焖羊肉|红焖羊肉]]
*: 属于[[河南|河南]][[新乡|新乡]]的地方名吃，以事先炒過的[[羊肉|羊肉]]再加入二十餘種食材與藥材料焖煮成鍋底。吃法是先吃锅内羊肉，然后放入其他食材加上剩下的湯汁涮食。在北方其它地区也有红焖狗肉、红焖兔肉、红焖鸽肉等變化。

==== 山西 ====

* 晋中火锅
*: 使用木炭铜锅，将食材与佐料、高汤放至锅内，煮熟后一边加热一边从中捞取食用，没有涮的吃法。

==== 湖南 ====
* 臘味火鍋
*: 將[[臘味|臘味]]過水過油後，加入料酒,[[大蒜|大蒜]]、[[薑|薑]]片等一起煸炒，再放入辣油鍋中煮成為鍋底。
* [[湘西|湘西]]狗肉火锅
湘西狗肉火锅的吃法是將[[狗肉|狗肉]]及中藥材一起燉煮成鍋底。

==== 贵州 ====

* 苗族火鍋
*: 贵州麻辣酸汤鸳鸯鍋
* 贵州酸汤鱼火锅，锅底采用西红柿和酸萝卜调出酸味，将鲤鱼或者鲶鱼煮熟后调其他食材。蘸料常用胡辣椒。
* 花江狗肉火锅
*: 源于贵州省[[关岭县|关岭县]][[花江镇|花江镇]]，选用当地良种狗，以黄狗为上品，黑狗次之。加工时，将肥壮带皮狗肉刮洗烧制干净，以文火清炖，将肉炖至熟而不烂，香而不膻、油而不腻、嫩软可嚼又抿牙耐品即可，配料选用黔中十多种特产药料香料煎制，色香味俱佳<ref>{{cite web |url=https://www.guizhouba.net/6317.html |accessdate=2018-12-05}}</ref>

==== 重庆 ====

===== 重庆麻辣火锅 =====
*: [[麻辣火锅|麻辣火锅]]目前是中國最普遍的火鍋種類。重庆麻辣火锅的主要菜品与北方火锅差异非常大，主要以毛肚、黄喉、鸭肠、鸡杂、猪脑、猪腰花、牛羊肉、黄鳝等为主。重庆火锅的调味是由香油、蒜泥、香辛料和耗油等混合调和而成，称为“油碟”，若不加油调和则称为“干碟”，把菜在里面涮再吃能有降火的作用。重慶火鍋以麻辣鲜香著称，以牛骨、豬棒骨熬汤為主要原料，辣椒则使用较四川辣椒更辣的贵州朝天椒，屬醬香味。

===== 重庆干锅 =====
*: [[干锅|干锅]]是重庆火锅的一个分支，使用100%的油取代火锅中的水。起源于重庆市区的袁家岗地区，在制法上与传统火锅不同之处在于锅底没有加水而直接用红油煮菜，比火锅温度更高，有炒菜的焦香，不同于一般火锅的煮食方式。但通常重庆市民认为干锅较火锅容易“上火”。干锅在2004年后在重庆开始传播，2005年后传向东部各地和广东等地。

===== 重庆鴛鴦鍋 =====
*: [[鸳鸯锅|鸳鸯锅]]起源于重庆。特色是將兩種湯底結合於一鍋，鍋中有一隔板或另置內鍋，將鍋底分為紅湯與白湯、沙嗲與清汤，可以同時滿足吃辣與不吃辣的人。因最早发明的店家使用的中间隔板上有鸳鸯图案而得名<ref>[http://shipin.people.com.cn/GB/107080/14026137.html 灵感来自两江交汇处 最早的鸳鸯火锅上真有一对鸳鸯]</ref>。

==== 四川 ====
* 成都火鍋
*: 鍋底以雞、魚為主要原料，辣椒使用四川二荆条辣椒，口感温和。

==== 廣東 ====
* 打甂爐
*: 俗寫[[打邊爐|打邊爐]]，[[嶺南|嶺南]]地區人士把吃火鍋讀成「打邊爐」，根據元朝[[呂誠|呂誠]]《來鶴亭集．南海口號六首．其五》：「炎方物色異東吳，桂蠹椰漿代酪奴。十月煖寒開小閣，張燈團坐打邊爐。」，是[[粵語|粵語]]對火鍋的稱呼。常用的湯底有雞湯、海鮮湯及[[沙嗲|沙嗲]]湯，部份食肆更會選用廣東[[老火湯|老火湯]]。兩湯同吃稱為鴛鴦鍋，燙熟食物後沾醬油食用。其中港式火鍋的鍋底並以配料眾多為特式。[[牛肉|牛肉]]、[[鯇魚|鯇魚]]片、[[鰻|鱔]]片、[[象拔蚌|象拔蚌]]、[[生蠔|生蠔]]、[[魚漿|魚滑]]、蝦滑、[[三文魚|三文魚]]，[[鵝肝|鵝肝]]、各類肉丸、香腸、各豆類製品、各種[[蔬菜|蔬菜]]菌類、餃子麵食等，均是熱門配料。
* 醉雞鍋
*: 把半隻至一隻的生雞連同[[花雕酒|花雕酒]]以及[[黨參|黨參]]、[[當歸|當歸]]、[[北蓍|北芪]]、[[枸杞|枸杞]]等[[中藥|中藥]]材作為鍋底，流行於[[香港|香港]]及[[廣東|廣東]]各地。
* 粥底火鍋
*: 用大米熬成的粥來做火鍋湯底，起源於廣東[[順德|順德]]，普遍以[[白粥|白粥]]為湯底，但也有食肆會選用有味粥為湯底。一般粥底鍋會選用各式材料。香港食肆提供的粥底火鍋，只會供鮮牛、鮮雞肉及內臟等食材，而不會供應肥牛肉、海鮮等。
* 豬骨煲
*: 流行於[[澳門|澳門]]，實際由廣東[[順德|順德]]原創再傳入澳門等地。流傳到[[深圳|深圳]]和[[香港|香港]]後有不同的變化。深圳的豬骨煲發展為湯類煲仔小菜，以吃豬骨邊肉、吸食[[骨髓|骨髓]]為主；[[香港|香港]]的豬骨煲較接近日式的豬骨濃湯，豬骨被烹調成呈乳白色的火鍋湯底。豬骨煲的作法是用豬大骨與其它材料一起熬煮，再加上白胡椒和粗鹽調味而成奶白色的火鍋鍋底。
* 重慶雞煲
*: 由[[重慶雞公煲|重慶雞公煲]]變化而成，已煮熟的辣雞以瓦煲盛載，吃完才加入豬骨湯底打甂爐，食材包括肥牛、肉丸、餃子等。<ref>[http://orientaldaily.on.cc/cnt/lifestyle/20111118/00296_004.html 今年最Hit！火辣重慶雞煲]《東方日報》2011年11月18日</ref>
* 魚頭火鍋
*: 源於[[沙鍋魚頭|沙鍋魚頭]]，以事先煎過的[[鰱魚|鰱魚]]頭或[[草魚|草魚]]頭再加入高湯與材料煮成鍋底。

*: 以[[鰱魚|鰱魚]]或[[鯽魚|鯽魚]]為主食材，配上[[昆明|昆明]]特有的糊辣味。

=== 台湾 === 
* [[燒酒雞|燒酒雞]]
*: 材料是[[雞肉|雞肉]]、[[米酒|米酒]]、[[當歸|當歸]]、[[枸杞|枸杞]]、[[黃耆|黃耆]]、[[川芎|川芎]]、[[甘草|甘草]]、[[薑|薑]]和[[冰糖|冰糖]]。能補血、活血、行氣、健胃、增強身體抵抗，促進新陳代謝、消化吸收及血液循環，是寒冷冬天活血暖身、舒筋養骨、溫補虛寒的常見藥膳。不過台灣家庭一般是先在廚房煮好再端上餐桌，嚴格講並不是邊煮邊吃的火鍋。
* [[羊肉爐|羊肉爐]]
*: 將羊大骨、汆燙過的[[羊肉|羊肉塊]]及各種中藥材一起熬煮成鍋底，用[[豆腐乳|豆腐乳]]或辣[[豆瓣醬|豆瓣醬]]做為沾醬。
* [[薑母鴨|薑母鴨]]
*: 將鴨肉與老薑及麻油拌炒，再加[[米酒|米酒]]及水燉煮。一般都使用豆腐乳做為沾醬。
* 台式[[麻辣鍋|麻辣鍋]]
*: 以大骨高湯加上[[辣椒|辣椒]]、[[花椒|花椒]]、醪糟和許多辛香料調製成麻辣鍋底，然後再加入鴨血、肥腸和豆腐等，味道厚重，油厚濃腴，既麻且辣，十分適合冬日食用，以袪除風寒。
* 台式[[涮涮鍋|涮涮鍋]]
*: 源自[[日本|日本]]，在[[臺灣|臺灣]]經過修改，成為台灣最普遍的小型火鍋種類。特色為生食材放進鍋裡燙至剛熟就撈起來吃。<ref>[http://news.xinhuanet.com/yzyd/gangao/20120420/c_111814462.htm 用美牛排涮火锅 台式涮涮锅也推“吃到饱” ]</ref>
* [[臭豆腐|臭豆腐]]火鍋
*: 一般稱為[[臭臭鍋|臭臭鍋]]，流行於台灣，是以火鍋高湯加入少許配料為湯底，之後放入臭豆腐乾。
* [[阿美族|阿美族]]石頭火鍋
*: 以[[檳榔|檳榔]]葉柄的葉梢摺疊成盒狀，盛滿水後將魚蝦和鹽放入，再逐一放入燒燙的石頭 ，直至湯水沸騰後放入野菜汆燙。
* 火烤兩吃火鍋
*: 火鍋加上烤肉兩種方式吃法。

== 燃料及爐具 ==

*[[木炭|木炭]]
*: 具有火力強、加熱快和保溫時間長等優點；但是炭灰容易污染空氣，也容易產生[[一氧化碳|一氧化碳]]。<ref name="火锅酒店餐厅空气卫生质量评价">王长辉、贾立冰、邢维华。火锅酒店餐厅空气卫生质量评价。山西预防医学，1998年7卷2期：123-124頁。</ref>
*[[煤气|煤气]]
*: 火候容易控制且較方便，但是因為煤氣保管不易，而且是有毒氣體，容易產生一氧化碳，因此比較危險。<ref name="火锅店空气卫生质量监测评价">白喜顺、吴怀玉等。火锅店空气卫生质量监测评价。中国卫生检验杂志，2001年11卷5期：613頁。</ref>*[[天然气|天然气]]：在重庆火锅店中最常使用，安全且卫生无烟，但需要通風良好，不然有一氧化碳中毒的危险。在建设店面时需要与现在楼层地面地砖下埋布管道。
*[[固體燃料|固體燃料]]
*: 優點是方便與乾淨，不會產生粉塵與炭灰，也容易保管。但是火侯不易控制，而且目前固體燃料的製作沒有規範標準，原料不一，因此燃燒時可能會產生有害氣體，並且刺鼻。<ref name="火锅酒店餐厅空气卫生质量评价"/>
*[[電磁爐|電磁爐]]
*: 電力爐具，與其它熱源比較起來更為乾淨<ref name="火锅店空气卫生质量监测评价"/>、方便、安全，以溫度設定進行控制，但是成本較高，因為以電磁場產生的渦流令金屬產生熱力，因此須要配合[[不銹鋼|不銹鋼]]鍋具使用。電磁爐可以嵌入桌子之中並藏於枱布之下，看起來比較美觀。
*[[電陶爐|電陶爐]]
*: 電力爐具，外觀與[[電磁爐|電磁爐]]相似，但不是以電磁場產生的渦流令金屬鍋具產生熱力，而是利用發熱管令爐面產生高溫，對鍋具種類限制較少，可不使用金屬鍋具，但須注意爐面的高溫。

== 食用火锅应注意事项 ==

* 保持空气流通
*: 火锅的火源是木炭时，在木炭燃烧不完全时会产生一氧化碳，会导致一氧化碳中毒<ref>陈楠。火锅餐厅内一氧化碳浓度对健康的影响及其对策。皖南医学院学报，2002年第21卷第2期：152-153頁。</ref>；而即便使用电磁炉或瓦斯炉，不流通的高温、高湿空气也容易导致[[细菌|细菌]]的滋生与疾病的传染。
* 勿过度饮用火锅的汤汁
*: 火锅的汤汁會大量代謝出[[尿酸|尿酸]]的[[嘌呤|嘌呤]]，严重时甚至会引发[[痛风|痛风]]，对健康是有碍的。所以吃火锅时应该少喝汤，多喝水。<ref>田双宏。常吃火锅谨防痛风。医药与保健，2001年第11期：55頁。</ref><ref>戴在松。寒冬火鍋旺季 痛風病患慎選食物。嘉義基督教醫院公告，2007年1月8日。</ref>
* 过于辛辣和肥厚油腻食品的火锅，会使脾胃消化功能失调，内生燥热湿邪，由此而诱发[[痔疮|痔疮]]，出现[[便秘|便秘]]、[[便血|便血]]等症狀。辛辣刺激还会出现咽喉肿痛、红斑、瘙痒、接触性[[皮膚炎|皮膚炎]]，诱发牙龈肿痛和[[口腔溃疡|口腔溃疡]]，及腹胀、腹痛、腹泻等胃肠道症状，此即「火锅综合征」。<ref>陆基宗。妙招防「火锅综合征」。家庭医学，2006年第1期：48頁。</ref>
* 吃火鍋時多是用筷子夾取用生肉、生海鮮、生菜等邊涮邊吃，所以最好完全煮熟再吃，必免吃進[[微生物|微生物]]或[[寄生蟲|寄生蟲]]。此外，夾生食材和盛放生食材的盤子應與夾入口的筷子和盛熟食的碟子分開，不要生熟混用。<ref name="吃火锅的讲究">陈新。吃火锅的讲究。保健医苑，2004年第1期：30頁。</ref>
* 口腔與食道的黏膜一般只能耐受50℃，而火鍋湯汁的溫度可高達120℃。因此剛燙好的食材最好先冷卻一下後再吃，否則會燙傷口腔或食道。如果太頻繁吃過熱的食物，嚴重者會導致[[食道癌|食道癌]]病變。<ref name="吃火锅的讲究" />
* 有些火锅店利用制造[[鸦片|鸦片]]的原料植物[[罂粟|罂粟]]的壳或粉去调味做底煮出来的火锅，長期食用可能会造成类似鸦片成瘾的症状。<ref>中国中央电视台，杭州：[http://www.cctv.com/program/qqzxb/20061227/103342.shtml 市场惊现“罂粟粉” 吃火锅要格外当心]。2006年12月27日更新。{{zh-cn}}</ref>
