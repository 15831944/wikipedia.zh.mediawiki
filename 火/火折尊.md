'''彥火火出見尊'''（{{jpn|j = 彦火火出見尊|hg = ひこほほてみ<small>（或ひこほほいてみ）</small> の みこと|rm = Pikopopotemi no mikoto|kyu = 彥火火出見尊|hgt = }}，'''彥火火出見命'''、日子穗穗手見命），一般稱作'''火折尊'''（{{jpn|j = 火折尊|hg = ほのさき<small>（或ほさき、ほのさをり、ほのをり、ほをり）</small> の みこと|rm = Ponotsaki no mikoto/Powoli no mikoto|kyu = |hgt = }}，'''火析尊'''、火遠理命），是[[日本|日本]]傳説中的[[葦原中國|葦原中國]][[治天|治天]]，也是[[地神五代|地神五代]]之一，也是[[神武天皇|神武天皇]]（日本第1代[[天皇|天皇]]）的祖父。又名'''彥火々尊'''（{{lang|ja|ひこほほ の みこと}}）、'''山幸彥'''（{{jpn|j = 山幸彦|hg = やまのさちひこ|rm = Yamanotsatipiko|kyu = 山幸彥|hgt = }}，山佐知毘古）、天津日高日子穗穗手見命、虛空津日高等。

== 事績 ==
{{Main|山幸彥與海幸彥}}
彥火火出見尊擅長在山中打獵，火闌降命擅長釣魚。後來兄弟二人互相交換了工具（山海易幸），但皆一無所獲，彥火火出見尊換得的魚鉤還被魚搶去。火闌降命不理會彥火火出見尊以自己佩帶的[[刀|横刀]]製造五百或一千個魚鉤賠償的提議，執意要求找回原魚鉤。彥火火出見尊遂跑到海邊哭泣，他的行為感動了海神[[鹽土老翁|鹽土老翁]]，替彥火火出見尊找回了魚鉤並對魚鉤進行詛咒，最終迫使火闌降命向自己稱臣。

彥火火出見尊後來在[[高千穗宮|高千穗宮]][[駕崩|崩]]，年580歲，葬於{{link-ja|高屋山上陵}}。

=== 父母 ===
* 父：[[天津彥彥火瓊瓊杵尊|天津彥彥火瓊瓊杵尊]]（天彥尊） - [[正哉吾勝勝速日天忍穗耳尊|正哉吾勝勝速日天忍穗耳尊]]和[[栲幡千千姬|栲幡千千姬]]之子。[[天照大神|天照大神]]・[[高皇產靈尊|高皇產靈尊]]之孫。
* 母：[[鹿葦津姬|鹿葦津姬]]（神吾田津姬、木花開耶姬、木花開姬、吾田津姬、吾田姬） - [[大山祇神|大山祇神]]之女。

彥火火出見尊，是天津彥彥火瓊瓊杵尊的第二子也。

=== 同母兄弟姉妹 ===
* 兄：[[火闌降命|火闌降命]]（火照命、海幸彥） - [[隼人|隼人]]等祖。
* 弟：[[火明命|火明命]] - {{link-ja|尾張氏}}等祖。

=== 配偶者・子女 ===
* 妻：[[豐玉姬|豐玉姬]] - [[海童|海童]]之大女。
** 第一皇子：'''[[彥波瀲武鸕鶿草葺不合尊|彥波瀲武鸕鶿草葺不合尊]]'''（彥波瀲尊、彥澰尊） - [[神武天皇|神武天皇]]（日本第1代[[天皇|天皇]]）的父親。
* 妻：[[玉依姬|玉依姬]] - 海童之小女。豐玉姬之妹。
** 子：[[武位起命|武位起命]] - {{link-ja|大和國造|大和国造}}等祖。
