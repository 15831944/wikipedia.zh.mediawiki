'''火山兔'''（''Romerolagus diazi''），又名'''墨西哥兔'''，是生活在[[墨西哥|墨西哥]]山區一種細小的[[兔|兔]]。牠們是[[世界|世界]]第二最細小的兔，僅次於[[侏兔|侏兔]]。牠們的[[耳朵|耳朵]]很圓，腳短，毛短而厚。牠們以小群生活，成員約有2-5名。牠們會發出很高音的[[聲音|聲音]]來警告其他兔子。牠們是夜間活動的。牠們重390-600克。於1969年統計，牠們野外只有1000-1200隻。

== 棲息地 ==
火山兔分佈在[[墨西哥|墨西哥]]，只限於[[伊斯塔西瓦特爾火山|伊斯塔西瓦特爾火山]]、[[佩拉多火山|佩拉多火山]]、[[波波卡特佩特火山|波波卡特佩特火山]]及[[特拉洛克|特拉洛克]]火山。牠們一般棲息在[[海拔|海拔]]2800-4250米的[[針葉林|針葉林]]及岩石地帶。

== 食性 ==
火山兔吃薩卡通草的[[葉子|葉子]]及[[赤楊|赤楊]]的樹皮等。在雨季，牠們會吃[[麥|麥]]及[[穀類|穀類]]等農作物。

== 保育狀況 ==
火山兔的衰落是因失去棲息地及獵殺所致。<ref name=iucn/>牠們的棲息地受到[[山火|山火]]、過度放牧、城市及農業發展及伐林等破壞。<ref name=iucn/>故此，已有計劃保護牠們的棲息地，以保護牠們。另外，在一些[[動物園|動物園]]亦已有培育的計劃。
