'''于右任墓'''，是[[中華民國監察院|中華民國監察院]]前院長[[于右任|于右任]]的土葬墓園，位於[[淡水區|淡水區]][[陽明山國家公園|陽明山國家公園]]內，[[巴拉卡公路|巴拉卡公路]]旁。<ref>{{cite web |url = https://nchdb.boch.gov.tw/assets/advanceSearch/monument/20120424000001 |title = 于右任墓 |work = 文化部文化資產局 |publisher = }}</ref>於2012年4月由[[新北市文化局|新北市文化局]]指定為[[新北市|新北市]]市定古蹟。<Ref>{{cite web |url = http://tw.news.yahoo.com/%E4%BA%8E%E5%8F%B3%E4%BB%BB%E5%A2%93%E5%9C%92-%E7%8D%B2%E5%AF%A9%E5%AE%9A%E6%96%B0%E5%8C%97%E5%B8%82%E5%8F%A4%E8%B9%9F-213000007.html | title= 于右任墓園 獲審定新北市古蹟 | last = 慶正 |publisher= 旺報 | date=2012-05-02 |accessdate = 2012-09-11}}</Ref>

==歷史沿革==
[[于右任|于右任]]於1964年在其[[監察院|監察院]]院長任內病逝，總統[[蔣中正|蔣中正]]安排於陽明山區下葬。墓園形制採近代大型墓園規格，內有[[蔣中正|蔣中正]]、[[嚴家淦|嚴家淦]]聯名褒揚令、[[孫科|孫科]]題字的「仰止亭」，以及石匾、石獅、石坊、牌樓等。其內的碑刻書法皆為上乘之作。因管理單位權責不清楚，時隔多年少有維護，多有荒廢。2012年新北市政府邀專家學者會勘，認定右任先生成就卓著、墓園形制恢弘，決議列為市定古蹟。
