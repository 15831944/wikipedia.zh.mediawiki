'''-{于}-右任'''（{{bd|1879年|4月11日|1964年|11月10日|catIdx=Y}}），[[陕西|陕西]][[三原|三原]]人，祖籍[[涇陽|涇陽]]。原名'''伯循'''，字'''誘人'''，爾後以“'''誘人'''”諧音“右任”為名；别署“'''髾心'''”、“'''髯翁'''”，晚年自號“'''太平老人'''”。[[中華民國|中華民國]]開國元勛之一。-{于}-右任早年係[[中國同盟會|中國同盟會]]成員，[[中華民國|民國]]成立之後長年在[[中華民國政府|政府]]擔任高級官員，尤其擔任[[監察院|監察院]][[監察院院長|院長]]長達34年，是歷史上在任最久的[[五院|五院]]院長。同時也是[[中國|中國]]近代知名的[[書法家|書法家]]。-{于}-右任長髯飄飄，是其一大特徵。

== 早年經歷 ==
-{于}-右任是清朝[[光绪|光绪]]年间[[举人|举人]]，1904年因刊印《半哭半笑樓詩草》譏諷時政被[[三原县|三原]][[县令|县令]]德锐和[[陕甘总督|陕甘总督]][[升允|升允]]举发，遭[[清廷|清廷]][[通緝|通緝]]，[[流亡|流亡]][[上海|上海]]，遂進入[[震旦公學|震旦公學]]。[[震旦学院|震旦学院]]肄业。同年4月到达[[日本|日本]]，加入[[光复会|光复会]]和[[同盟会|同盟会]]。

== 报人生涯 ==
-{于}-右任的報刊活動主要在1907年到1912年[[中華民國|民國]]成立前5年當中。其中與1909年開始連續創辦三份報紙：《民呼日報》、《民籲日報》、《[[民立報|民立報]]》被稱為‘[[豎三民|豎三民]]’。

1907年，於上海四馬路群學社書店創辦大型日報《[[神州日報|神州日報]]》，雖未聲明是革命派言論機關，但實際上革命傾向非常明顯。創刊後僅80天，因鄰居失火，殃及報社，編輯、印刷、營業部均被燒毀，于右任無力恢復，自行辭退。離開後經幾個月的籌備，1909年5月15日創辦《民呼日報》，由於負面報導很快遭到清廷控告，被逐出租界，被迫停刊。停刊兩月後，-{於}-右任在[[上海法租界|法租界]]創辦《民籲日報》。很快由於報導時任[[日本|日本]][[日本首相|首相]][[伊藤博文|伊藤博文]]遭[[朝鮮半島|朝鮮]]人[[安重根|安重根]]刺殺事件傾向性明顯遭到日方抗議，租界當局於1909年11月查封了《民籲日報》。後在[[沈縵雲|沈縵雲]]資助下與1910年10月創辦《民立報》，積極宣傳民主革命。

1913年[[二次革命|二次革命]]失败后，-{于}-右任逃亡日本，《民立报》停刊。

== 仕途 ==
[[File:Yu_Youren's_voting_20090420.jpg|thumb]][[國民政府|國民政府]][[监察院|监察院]][[監察院|院长]]-{于}-右任投票]]
[[中華民國|中華民國]]成立後，任[[中華民國臨時政府_(南京)|临时政府]][[中華民國交通部|交通部]]次长。1917年响应[[孙中山|孙中山]][[护法运动|护法运动]]，成立陕西靖国军，任驻陕[[总司令|总司令]]。1922年创办[[上海大学|上海大学]]。1924年出席[[国民党一大|国民党一大]]。年底陪同[[孙中山|孙中山]]北上赴京。

1926年，代表[[中國國民黨|中國國民黨]]赴[[蘇聯|蘇聯]]促[[馮玉祥|馮玉祥]]速回國參加[[國民革命軍北伐|國民革命軍北伐]]。馮玉祥由蘇俄回國，督率所部橫越朔漠，經過[[寧夏|寧夏]]，進入[[甘肅|甘肅]]，以[[孫良誠|孫良誠]]為援陝總指揮<ref name="細說西安一"/>{{rp|8}}。-{于}-右任先由五原起行，兼程回陝西收集國民二軍及國民三軍餘部，組成聯軍，率領7個師大軍，東進援救陝西<ref name="細說西安一">{{cite book|author=王禹廷|chapter=〈一、楊虎城一頁滄桑史〉|title =《細說西安事變》|location=臺北|publisher=[[傳記文學|傳記文學]]出版社|date=1989-10-01}}</ref>{{rp|8}}。

[[國民政府|國民政府]][[北伐|北伐]]統一全國后，-{于}-右任歷任审计院、[[监察院|监察院]]院长，是国民政府重要决策人物，前後共任監察院院長34年。

1932年秋，籌備建設國立西北農林專科學校（今[[西北農林科技大學|西北農林科技大學]])，並就任首任校長。

== 晚年經歷 ==
1948年4月19日，在當年3月的[[1948年中華民國總統選舉|總統選舉]]過後，曾参选副总统選舉，然而根據當時的國民大會秘書處所公布副總統候選人簽署代表人數：于右任獲得512票的簽署支持，僅次於[[孫科|孫科]]。在4月20日下午的首輪投票中，于右任只獲得了493票，卻在同時競選的六人中排名第四，而無法進入第二輪投票。

1948年6月，-{于}-右任當選為監察院院長，[[劉哲|劉哲]]當選為副院長<ref name="年表">{{cite book|author=[[陳布雷|陳-{布}-雷]]等編著|title=《蔣介石先生年表》|date=1978-06-01|publisher=[[傳記文學|傳記文學]]出版社|location=台北|page=55}}</ref>。

1949年，随[[中華民國政府|中華民國政府]]遷居[[台灣|台灣]][[台北市|台北市]]青田街官邸。-{于}-右任寓居臺灣十五年，台灣書法風氣首次出現碑學壓倒館閣帖的趨勢。

== 逝世 ==
1962年1月12日，-{于}-右任日記寫道：「我百年後，願葬於[[玉山|玉山]]或[[阿里山山脈|阿里山]]樹木多的高處，可以時時望大陸（旁邊自注：山要高者，樹要大者）。遠遠是何鄉，是我之故鄉。我之故鄉是中國大陸<ref>{{cite web|url=http://big5.huaxia.com/la/jwgc/2008/00796633.html|title=-{于}-右任的晚年生活}}</ref>。」

1月24日，-{于}-右任於日記中寫下《思鄉歌》：“葬我於高山之上兮，望我大陆；[[中国大陆|大陆]]不可见兮，只有痛哭。葬我於高山之上兮，望我故乡；故乡不可见兮，永不能忘。天苍苍，海茫茫，山之上，有国殇。不得大陆，不能回乡，大陆乎，何日光复<ref>{{cite book|author=古遠清|title=《台灣文壇的「實況轉播」: 一位大陸學者眼中的台灣文壇》 |url=http://books.google.com.hk/books?id=gM55erEM_v4C&pg=PA101&lpg=PA101&dq=%E4%B8%8D%E5%BE%97%E5%A4%A7%E9%99%B8%EF%BC%8C%E4%B8%8D%E8%83%BD%E5%9B%9E%E9%84%89%EF%BC%8C%E5%A4%A7%E9%99%B8%E4%B9%8E%EF%BC%8C%E4%BD%95%E6%97%A5%E5%85%89%E5%BE%A9%EF%BC%9F&source=bl&ots=p_LwF7k8U0&sig=MwhYmmo5PyCIJCEuKMiQweFjorc&hl=zh-TW&sa=X&ei=PY6aU4fkB4HnkAXw34CgBg&ved=0CCoQ6AEwAg#v=onepage&q=%E4%B8%8D%E5%BE%97%E5%A4%A7%E9%99%B8%EF%BC%8C%E4%B8%8D%E8%83%BD%E5%9B%9E%E9%84%89%EF%BC%8C%E5%A4%A7%E9%99%B8%E4%B9%8E%EF%BC%8C%E4%BD%95%E6%97%A5%E5%85%89%E5%BE%A9%EF%BC%9F&f=false|date=2013-07-01|publisher=秀威出版|location=台北|isbn=9789863261292|page=101}}</ref>？”

1964年8月12日，于右任因病於臺北榮民總醫院住院治療，後於11月10日20:08病逝{{r|中1}}，墓位於[[台灣|台灣]][[陽明山國家公園|陽明山國家公園]][[巴拉卡公路|巴拉卡公路]]上。

-{于}-右任去世後，[[中華民國政府|中華民國政府]]在[[玉山|玉山]]主峰樹立其雕像，計3公尺高，本意在補足4,000公尺高度（玉山當時測量數據為海拔3,997米），並有尊重-{于}-右任遙望大陸遺願「葬我於高山兮，望我大陸」之意。唯銅像1995年、1996年[[玉山于右任銅像破壞事件|二次遭到破壞]]；其中參與1995年11月1日第一次「砍頭」的「元兇」[[葉博文|葉博文]]（前[[建國廣場|建國廣場]]成員、前[[台北二二八紀念館|台北二二八紀念館]]館長、[[核四公投促進會|核四公投促進會]]執行長）在追訴期過後的2007年2月12日首次公開承認破壞行為並引以為榮，而另二位破壞者為[[林長安|林長安]]（前[[綠色和平電台|綠色和平電台]]主持人）與[[陳朝順|陳朝順]]（[[綠黨_(臺灣)|綠黨]]2002年[[台北市|台北市]]第六選舉區[[台北市議員|市議員]]候選人），三人破壞的理由為「還我玉山原貌」；在首次被「斷頭」經修復後不久，在1996年5月初被整個拆除、丟入山谷，不知所終，這次破壞迄今無人出面承認說明<ref>{{cite news|url=http://www.libertytimes.com.tw/2007/new/feb/13/today-fo5.htm|title=玉山-{于}-右任像斷頭 元凶說始末|work=《[[自由時報|自由時報]]》|date=2007-02-13|deadurl=yes|archiveurl=https://web.archive.org/web/20120707081835/http://www.libertytimes.com.tw/2007/new/feb/13/today-fo5.htm|archivedate=2012-07-07}}</ref>。

台北市[[仁愛路_(台北市)|仁愛路]]與[[敦化南路|敦化南路]]交叉處的仁愛圓環設有-{于}-右任銅像，後來在[[陳水扁|陳水扁]]市長任內藉[[臺北燈節|元宵節燈會]]佈置之便破壞銅像基座，並以此為由遷移至[[國父紀念館|國父紀念館]]<ref>{{cite web|url=http://lh4.ggpht.com/_0tibDhJQ0Og/TJtchtpirLI/AAAAAAAATNA/shh0dkMi2-M/s800/yu.jpg|title=元宵燈架惡意破壞銅像基座圖}}</ref>。

位在[[基隆市|基隆市]]的[[崇右技術學院|崇右技術學院]]為-{于}-右任所提議創設，並於1964年開始招生，學校名稱有崇敬-{于}-右任之意涵。

2012年，位於陽明山的[[于右任墓|于右任墓]]被定為新北市三級古蹟。

2014年3月，臺灣爆發[[佔領國會事件|佔領國會事件]]，-{于}-右任題寫的“立法院”匾額被[[示威|示威]]群眾拆下，台北市警局中正一分局20日表示，匾額已取回，先暫存分局內<ref>{{YouTube|93Yuk3D-DNw|20140319 立法院中山南路匾額被拆}}</ref>。

2014年，于右任逝世50周年紀念大會12月17日在臺北舉行，兩岸逾百位人士出席。[[中華民國副總統|中華民國副總統]][[吳敦義|吳敦義]]出席，並認為：于右任先生參與革命有成，在任[[監察院|監察院]]長期間端肅政風，更在書法上取得很大成就。

為紀念于右任逝世50周年，其祖籍陝西涇陽和出生地三原近日都舉行了紀念儀式和家祭。于右任的侄孫女于媛與中國大陸鄉親60余人近日還前往位於臺北陽明山的于右任墓園，以古禮獻上了來自家鄉的泥土、水和五穀等祭品。

== 名言 ==
《亡国三恶因》：
# 民穷财尽，社会破产，国家破产。国有金，吝不与人，为他人藏。此其一。
# 善不能举，恶不能退，利不能兴，害不能除。化善而作贪，使学而为盗。此其二。
# 宫中、府中、梦中，此哭中、彼笑中，外人窥伺中、霄小拨弄中，国际侦探金钱运动中，一举一动，一黜一陟，堕其术中。此其三。

== 著作和成就 ==
-{于}-右任精[[书法|书法]]，尤擅[[草书|草书]]，首創“[[標準草書|標準草書]]”，被誉为“当代草圣”、“近代書聖”。-{于}-右任被列為「民國四大[[书法家|书法家]]」<ref>[https://www.npm.gov.tw/Article.aspx?sNo=04009391 自然生姿態－于右任書法特展]，[[國立故宮博物院|國立故宮博物院]] </ref>，其为[[楷书|楷书]][[谭延闿|谭延闿]]、[[草书|草书]]-{于}-右任、[[隶书|隶书]][[胡汉民|胡汉民]]、[[篆书|篆书]][[吴稚晖|吴稚晖]]。

-{于}-右任1932年在上海創辦標準草書社，以易識、易寫、準確、美麗為原則，整理、研究與推廣草書，整理成系统的草书代表符号，集字編成《標準草書千字文》（1936年由[[上海|上海]]文正楷印书局初版），影響深遠，至今仍在重印。著作《右任詩存》、《右任文存》、《右任墨存》、《標準草書》、《牧羊兒的自述》（1953）等。

== 軼事 ==
-{于}-右任擔任監察院長時看到員工隨處小便，遂提筆寫下「不可隨處小便」公告張貼，因墨寶珍貴被人偷去收藏；又因-{于}-的標準草書字字可分離，後人便重新組裝成「小處不可隨便」<ref>{{cite web|title=不可隨處小便 -{于}-右任墨寶重現|url=https://tw.news.yahoo.com/%E4%B8%8D%E5%8F%AF%E9%9A%A8%E8%99%95%E5%B0%8F%E4%BE%BF-%E4%BA%8E%E5%8F%B3%E4%BB%BB%E5%A2%A8%E5%AF%B6%E9%87%8D%E7%8F%BE-052227416.html|author=記者葉素萍電|work=[[中央社|中央社]]|location=台北|date=2011-10-01}}</ref>。

== 子女 ==
長子，[[望德|-{于}-望德]]（1910－？），早年赴[[英國|英國]]留學，先入[[愛丁堡大學|愛丁堡大學]]，獲法學士學位。後入[[倫敦大學|倫敦大學]]，獲[[政治學|政治學]][[博士|博士]]學位。期間曾任國民黨英國愛丁堡支部書記。返國後歷任[[重慶大學|重慶大學]]、[[中央大學|中央大學]][[教授|教授]]。國民政府[[行政院憲政計劃委員會|行政院憲政計劃委員會]]法制組專門委員、經濟會議政務組副主任、國民政府[[行政院|行政院]][[參事|參事]]。後在國民黨中央訓練團黨政訓練班十三期和[[國防研究院|國防研究院]]受訓。1945年當選爲國民黨第六屆中央候補執行委員。1947年5月，任駐[[哥倫比亞|哥倫比亞]]全權公使；6月兼駐[[委內瑞拉|委內瑞拉]]全權公使；7月兼駐[[厄瓜多爾|厄瓜多爾]]全權公使。1948年當選爲行憲國民大會代表。1954年11月，任駐[[巴拿馬|巴拿馬]]大使。1955年兼任駐[[洪都拉斯|洪都拉斯]]公使。1956年卸任，從巴拿馬返回台灣，任[[外交部|外交部]]顧問、行政院顧問、[[外交領使人員講習所|外交領使人員講習所]]顧問、[[中國文化學院|中國文化學院]]政治研究所所長、國防研究院首席講座等。1984年-{于}-望德捐贈「南宋影青印花水草紋蓋盒」給[[國立故宮博物院|國立故宮博物院]]。

次子，[[于彭|-{于}-彭]]（1916年9月9日－），字仲芩，[[金陵大學|金陵大學]]肄業後，於英國愛丁堡大學獲碩士學位，後在[[哥倫比亞大學|哥倫比亞大學]]任國際關係研究員。曾任中華民國外交部美洲司專門委員。1956年3月，任中華民國駐秘魯大使館參事。1962年10月，任駐牙買加大使館代辦。1967年2月起，先後任外交部情報司副司長、簡任秘書兼[[國外經濟文化事務委員會|國外經濟文化事務委員會]]執行秘書、[[研究設計委員會|研究設計委員會]]委員兼執行秘書、行政院參議等職。1971年7月，調任中華民國駐[[菲律賓|菲律賓]]大使館公使。1976年1月，任外交部領事事務處處長。1980年1月起，擔任中華民國駐宏都拉斯大使。2016年-{于}-彭捐贈所藏18件其父書法珍品給[[國立故宮博物院|國立故宮博物院]]<ref>[http://www.cna.com.tw/news/firstnews/201712140149-1.aspx 于右任18件傳世寶 于彭全都捐給故宮]</ref>，2017年6月國立故宮博物院展出3個月的「自然生姿態－-{于}-右任書法特展」<ref>[http://theme.npm.edu.tw/exh106/YuYujensCalligraphy/ch/index.html 自然生姿態—于右任書法特展_展覽概述]</ref>，當中主打選件就為-{于}-彭捐贈之精品。

四女于念慈，其夫[[張澄基|張澄基]]。
