'''于冬'''（{{bd|1971年|1月8月|||Y}}）<ref>[《http://ent.sina.com.cn/m/c/2014-08-07/01064187581.shtml 于冬前妻被免职 曝男方对孩子不闻不问》“于冬的生日和儿子的生日是同一天（1月8日）。”]</ref>，北京人，[[北京电影学院|北京电影学院]]管理系毕业，[[长江商学院|长江商学院]][[工商管理|工商管理]]硕士学位学历，现任[[博纳影业|博纳影业]]集团总裁。2018年接受訪問時表示之後可能把[[博纳影业|博纳影业]]讓給職業經理人來做，而自己轉變為教師，因為是自己的理想，希望可以和學生分享職業的經驗和態度<ref>[https://bka.mpweekly.com/interview/%E6%B1%AA%E6%9B%BC%E7%8E%B2-%E6%9B%BC%E9%8F%A1%E9%A0%AD/20180509-112695 【于冬專訪1】三萬元白手興家　成就首間中國上市影視公司橫掃金像獎]</ref>。

== 创业简历 ==
*北影毕业后，1994-1999年间，他在[[北京电影制片厂|北京电影制片厂]]工作；
*1999-2000年，他在[[中国电影集团公司|中国电影集团公司]]从事负责国产影片的发行工作；
*1999年，他同几个事业上的伙伴创立了北京博纳文化交流有限公司，从事影片发行工作。第一部发行的作品为《[[说出你的秘密|说出你的秘密]]》，此后陆续参与承担了《[[我的兄弟姐妹|我的兄弟姐妹]]》、《[[天脉传奇|天脉传奇]]》等作品的成功发行；
*2002年6月，由于他所创办的博纳公司对国产影片发行做出了突出贡献，所以博纳成为第一家获得电影发行经营许可证的民营公司；
*2003年下半年，他向电影制作领域迈进走了关键一步：他领导的博纳公司与中国保利华亿传媒集团合作，双方重组为北京保利博纳有限公司，从事影片制作和发行；
*2004年，他建立了为电影娱乐品牌广告行销的广告公司；
*2005年，他又成立了演艺经纪公司和国际影院投资公司；
*2010年12月9日，博纳影业总裁于冬敲响[[纳斯达克|纳斯达克]]的开市钟，使得博纳影业成为中国第一家在美国成功上市的影视公司<ref>[http://ent.sina.com.cn/m/2010-12-09/23013172944.shtml 博纳影业登陆纳斯达克创纪录 于冬热泪盈眶(图)]</ref>；

== 个人生活 ==
于冬的前妻[[臧黎璐|臧黎璐]]（1977年-）是[[南京|南京]]人，比于冬小6岁，是博纳公司的联合创始人之一。2001年两人结婚，2002年7月25日，女儿出生；2007年1月8日，儿子出生。2010年9月两人协议离婚，两个孩子归女方抚养，男方支付女方数千万钱款。2011年，[[金巧巧|金巧巧]] （1975年4月21日-）与博纳影业集团总裁于冬相恋，2013年8月9日为其生下女儿。

== 荣誉 ==
*2006年11月被《[[好莱坞报道|好莱坞报道]]》评为“最具影响力的亚洲电影制作人”
*2008年获得“中国创意企业年度杰出人物”<ref>[http://www.bonafilm.cn/management-team.html 博纳电影集团官网资料] {{webarchive|url=https://web.archive.org/web/20101217073529/http://www.bonafilm.cn/management-team.html |date=2010-12-17 }}</ref>
*2012年被[[第十二届华语电影传媒大奖|第十二届华语电影传媒大奖]]授予“百家传媒年度致敬电影人”

== 趣事 ==
*他当初报考电影学院并非出于个人兴趣而是觉得电影好玩。不过在电影学院看了两千多部电影后，使他对电影产生了浓厚的兴趣，于是毕业后便到北影厂做发行工作；
*在电影学院读书时每逢大小晚会他都会说一段历史方面的[[评书|评书]]，这源于他从小对历史、典故、戏曲以及成语故事的爱好与熟知，最佩服的历史人物是[[关云长|关云长]]和[[韩信|韩信]]；
*他个人发展的人生目标是要做内地的[[江志强|江志强]]；
*他最开心的事情就是当票房大卖后和那些影片的导演主创们一起喝酒；
*他的另外一个职业理想就是要当北影管理系的教授，因为他想把他走过的路和做的事情讲给年轻的学生们，以便能让他们以后在人生的事业上少走弯路<ref>[http://famous.bjnews.com.cn/2010/0331/20.shtml 于冬：科班出身的民营老板] {{webarchive|url=https://web.archive.org/web/20100524164548/http://famous.bjnews.com.cn/2010/0331/20.shtml |date=2010-05-24 }}</ref>。
