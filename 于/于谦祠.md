'''于谦祠'''是位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[西裱褙胡同|西裱褙胡同]]23号的一座纪念[[明朝|明朝]][[于谦|于谦]]的祠堂。<ref name=xhw>[http://news.xinhuanet.com/ziliao/2009-08/20/content_11916345.htm 西裱褙胡同23号——于谦祠，新华网，2009-08-20]</ref>

==历史==
[[西裱褙胡同|西裱褙胡同]]原是[[崇文门内大街|崇文门内大街]]路东自北向南数第二条胡同，为东西走向，南与[[麻线胡同|麻线胡同]]相连。1993年，[[建国门内大街|建国门内大街]]南侧建成了[[北京邮政枢纽|北京邮政枢纽]]，西裱褙胡同被占用一段，胡同尚存约500米。后来，[[建国门内大街|建国门内大街]]南侧继续改造，西裱褙胡同最终被全部毁灭。于谦祠位于西裱褙胡同23号，原在西裱褙胡同中段路北，如今则位于[[建国门内大街|建国门内大街]]南侧。<ref name=xhw/>

《帝京景物略》载：“崇文门内东半里，有祠曰忠节，祀少保、[[兵部尚书|兵部尚书]][[于谦|于公谦]]也。公一臂一肩，定[[正统|正统]][[己巳之变|己巳之变]]。其被刑西市也，为[[天顺_(明朝)|天顺]]元年。九年复官，为[[成化|成化]]二年。又二十三年，赐谥肃愍，为[[弘治|弘治]]三年。又一百一年，改谥忠肃，为[[万历|万历]]十八年。凡百有三十三年而定论。祠三楹，祀公塑像，岁春秋，遣太常寺官致祭。”《春明梦余录》载：“于少保祠额曰忠节，在[[崇文门|崇文门]]内东裱褙巷，公故赐宅也。”据此，于谦祠实为俗称，该祠正名原为“忠节祠”，始建于[[明朝|明朝]][[成化|成化]]二年（1466年），原为于谦故居。<ref name=xhw/><ref>《东城区志》P12</ref>

[[清朝|清朝]][[顺治|顺治]]元年（1644年），清朝入主中原，于谦祠祠废像毁。清朝[[光绪|光绪]]十五年（1889年）在原址重建。《[[天咫偶闻|天咫偶闻]]》载：“于忠肃祠，在裱褙胡同。芜废已久，近始重修。[[浙江|浙]]人逢春秋闱，居为试馆。”这说明于谦祠在清朝末年曾充当[[浙江|浙江]]举子的[[会馆|会馆]]。光绪二十六年四月，[[义和团|义和团]]进入北京，在此设坛口教习拳艺<ref>《东城区志》P18</ref>。后来，于谦祠成了居民杂居的大杂院。1976年[[唐山大地震|唐山大地震]]波及北京，于谦祠内的奎星阁被震毁，随后奎星阁遭到拆除。<ref name=xhw/>

1984年，于谦祠被公布为[[北京市文物保护单位|北京市文物保护单位]]。2004年，西裱褙胡同被拆除，该胡同内多处历史遗迹除于谦祠被保留外，戏剧理论家[[齐如山|齐如山]]故居、“汉园三诗人”之一的[[何其芳|何其芳]]故居均被拆除<ref>{{Cite web |url = http://finance.ifeng.com/news/industry/20110530/4087081.shtml |title =谁动了公民的合法私产  |author =  |date = 2011-05-30 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。2000年代，于谦祠获得重修，其内居民也被迁出。如今，于谦祠东、西、南三面都是高层楼房，永久处于阴暗之中，仅北面临[[建国门内大街|建国门内大街]]南侧绿地。于谦祠东侧及南侧为[[英大国际大厦|英大国际大厦]]，西侧为[[新闻大厦酒店|新闻大厦酒店]]及[[华夏银行|华夏银行总行]]<ref name=xhw/>。

西裱褙胡同23号、甲23号原为一个院落，23号（东院）保存较好，被定为北京市文物保护单位“于谦祠”，而甲23号（西院）已不完整且残破，未被认定为文物保护单位，但东城区文化委员会一直以文物标准管理保护甲23号，定期开展安全巡视检查。这两处院落都在2008年修缮，经有关部门批准，两院产权划归东城区文化委员会。此后，东城区文化委员会分别与[[中国农业银行|中国农业银行]]和鲁能英大集团签署西裱褙胡同23号及甲23号的使用协议，约定院内仅可从事文化交流、文物宣传、企业文化活动。协议中规定西院不得从事商业经营活动，不得悬挂牌匾、不得向第三方转租等，东院则将建立于谦祠纪念馆，2014年已经完成展陈设计<ref>{{Cite web |url =http://bj.people.com.cn/n/2014/0824/c82846-22085333.html  |title = 北京于谦祠布展完成后将开放 官方否认祠边开餐馆 |author =  |date = 2014-08-24 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。

2017年5月23日，东城区官德教育基地在[[文天祥祠|文天祥祠]]、[[袁崇焕祠|袁崇焕祠]]、于谦祠挂牌，东城区还下发通知要求全区各单位参观学习。其中，于谦祠增设“于谦官德展”，并在现有“北京保卫战”的内容基础上扩充<ref>{{Cite web |url = http://bj.people.com.cn/n2/2017/0615/c82840-30329538.html |title = 文保单位文天祥祠、袁崇焕祠和于谦祠变身官德教育基地|author =  |date = 2017-06-15 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref>。自2017年6月起，于谦祠免费向游客开放参观<ref name=qqxun>{{Cite web |url =  http://op.inews.qq.com/m/20171030B01GF400|title = 丞相祠堂何处寻——凭吊北京于谦祠 |author =  |date =2017-10-30  |publisher = 腾讯 |language =  |accessdate =  |quote =  }}{{不可靠来源}}</ref>。

==建筑==
于谦祠坐北朝南，现存主要建筑为[[光绪|光绪]]十五年（1889年）重建，占地面积约1300平方米。于谦祠正院为二进宅院，带东、西跨院。 院内房屋均为青砖到顶的清水墙体，清水脊合瓦屋面。主要建筑有<ref name=xhw/><ref>[http://www.bjdclib.com/subdb/laneculture/historybuilding/200906/t20090615_2132.html 于谦祠，东华流韵，于2013-04-23查阅]</ref>：
*大门：在于谦祠的“[[巽|巽]]”位（东南角）建有[[广亮大门|广亮大门]]，门楣正中悬挂黑底金字“于忠肃公祠”木匾，四个门簪上有蓝底金字“吉祥如意”，门内悬挂黑底金字“丹心抗节”木匾<ref name=qqxun/>，大门西侧有倒座南房五间。
*奎光楼：正对大门，为中式两底一顶的三间二层小楼。一层上方原悬挂“热血千秋”木匾，两侧对联原为：“帝念有功群小谗谋冤太惨，公真不朽故居歆记地犹灵。”2017年开放后，一层门外抱柱联是黑底金字“中流砥柱独挽朱明残祚，庙容永奂长赢史笔芳名”，一层分前后两间展室，堂屋为前展室，正中供奉于谦画像，上方悬挂白底黑字“热血千秋”木匾，于谦画像两侧白底黑字木对联“丹心托月，赤手擎天”，画像前有摆放五供的供桌以及于谦的牌位，外侧抱柱联是[[林则徐|林则徐]]的名联“公论久而后定，何处更得此人”。四壁图文展览分“钱塘才子 志存高远”和“两袖清风 勤政为民”两个主题介绍于谦生平。该展室还有镇河铁犀仿制品。前展室后面为后展室，正中是《石灰吟》的碑刻，右侧悬挂于谦大事年表，图文展览分“铁骨铮铮，清风传世”和“后世缅怀，光耀万代”两主题，壁上还展出《咏煤炭》、《三月初三日出使》、《夏日过太行》、《悼内》四首诗。奎光楼的二层原为“奎星阁”，乃[[科举|科举]]应试[[举子|举子]]求“[[魁星|魁星]]点斗，金榜题名”之所，2017年开放后也辟为展室，展室正中是北京城防地图，分“保卫京师 击退瓦剌”和“夺门之变 忠臣遇害”两个主题，展览有明代火枪和于谦书法《题公中塔图赞》等实物<ref name=qqxun/>。
*外院：由大门西侧的倒座南房五间与其北侧的五间过厅组成。
*里院：正房五间，东、西厢房各两间。正房五间为享殿，内供于忠肃公塑像。2017年开放后，外院、里院部分房屋被[[中国农业银行|中国农业银行]]用于文化展<ref name=qqxun/>。
*东跨院：现已无存。
*西跨院：西裱褙胡同甲23号
