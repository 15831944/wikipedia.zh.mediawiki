'''于丹'''（{{bd|1965年|catIdx=Y于}}）是中國古代文學碩士、影視學博士，[[北京師範大學|北京師範大學]]藝術與傳媒學院院長助理、影視傳媒系系主任、碩士生導師。曾就读于[[北京四中|北京四中]]，本科就读于[[北京职业技术师范学院|北京职业技术师范学院]]（现[[北京联合大学|北京联合大学]]），硕士博士就读于[[北京师范大学|北京师范大学]]。現為[[中國電視藝術家協會|中國電視藝術家協會]]會員，中國視協高校藝術委員會秘書長、中國視協理論研究會特邀研究員、[[中國中央電視臺|中國中央電視臺]]研究處客座研究員、中國新聞研究會、[[中國廣播電視學會|中國廣播電視學會]]主持人研究會、中國廣播電視學會法制節目委員會常務評委、澳大利亞[[新聞集團|新聞集團]]首席顧問。現任[[北京電視臺|北京電視臺]]首席策劃顧問，並在[[北京师范大學|北京师范大學]]廣播學院電視學研究生班固定授課。其父为[[中华书局|中华书局]]副总经理[[于廉|于廉]]。<ref>{{Cite web |url=http://www.sznews.com/reading/content/2007-03/14/content_937823.htm# |title=网友称于丹成名靠父亲 中华书局出面澄清 |access-date=2009-09-28 |archive-url=https://web.archive.org/web/20150924120952/http://www.sznews.com/reading/content/2007-03/14/content_937823.htm# |archive-date=2015-09-24 |dead-url=yes }}</ref>

于丹在2006年[[十一長假|十一長假]]期間在《[[百家講壇|百家講壇]]》連續七天解讀自己对于《[[論語|論語]]》的诠释，大受好評。不過，于丹的講解也引發不少爭議，許多人認為她的講解錯漏百出<ref>[http://culture.people.com.cn/BIG5/n/2013/0310/c1013-20738761-4.html]</ref>。2007年[[春節|春節]]，于丹又從年初一到年初十連講10天的《〈[[莊子|莊子]]〉心得》；同年[[十一黃金周|十一黃金周]]，于丹又连讲7天的《[[遊园惊梦|遊园惊梦]]·崑曲之美》<ref>參見白先勇《青春版牡丹亭》</ref>。2008春節，于丹卷土重來，以孝敬、智慧、誠信、學習、治世等方面為題，在《百家講壇》上再講《論語》，被評為「一分鐘錯一次」<ref>[http://book.people.com.cn/BIG5/69362/6873087.html]</ref>。

== 生平 ==
1988年開始從事影視創作及研究工作，曾擔任大型專題片撰稿人，包括：
* 《[[在共和國史冊上|在共和國史冊上]]》（中央電視臺，1989年）
* 《[[太陽照常升起_（央视专题片）|太陽照常升起]]》（中央電視臺，1992年）
* 《[[香港滄桑|香港滄桑]]》（中央電視臺，1996年，獲得1997年中國電視專題片優秀獎）

1990年以來，為[[中央電視臺|中央電視臺]]擔任撰稿人，曾獲1994年、1996年星光獎最佳撰稿獎，欄目包括：
* 《[[正大綜藝|正大綜藝]]》
* 《中國報-{}-道》
* 《環球》

1999年以來，擔任中央電視臺欄目策劃，欄目包括：
* 《[[東方時空|東方時空]]》
* 《社會經緯》
* 《[[今日說法|今日說法]]》
* 《精品賞析》
* 《電視你我他》
* 《[[新聞調查|新聞調查]]》
* 《[[对话_(电视节目)|對話]]》
* 《[[藝術人生|藝術人生]]》

同時擔任中央電視臺《城市之間》欄目策劃、導演，中央電視臺教科文頻道策劃顧問，光線傳播中心《中國娛樂報導》欄目總策劃。並擔任1993年中央電視臺青少部春節晚會《紅紅火火迎大年》的策劃人與撰稿人。

曾先後為[[遼寧|遼寧]]、[[河北|河北]]、[[浙江|浙江]]、[[安徽|安徽]]、[[貴州|貴州]]、[[湖南|湖南]]、[[湖北|湖北]]、[[江西|江西]]、[[河南|河南]]、[[山東|山東]]等20多家省級電視臺及為[[青島|青島]]、[[大連|大連]]、[[瀋陽|瀋陽]]、[[長沙|長沙]]、[[武漢|武漢]]等20多家市級電視臺進行業務培訓。

2012年11月17日晚，[[北京大学|北京大学]]剧院邀请[[昆曲|昆曲]]大师[[张洵澎|张洵澎]]、[[岳美缇|岳美缇]]、[[汪世瑜|汪世瑜]]、[[张继青|张继青]]、[[石小梅|石小梅]]、[[王芳|王芳]]演唱昆曲，主办方[[皇家粮仓|皇家粮仓]]负责人王翔邀请于丹作总结，观众认为“[[心灵鸡汤|心灵鸡汤]]亵渎国宝”，结果于丹被观众轰下台。<ref>{{cite web |url=http://wei.sohu.com/20121119/n357997236.shtml?pvid=tc_news&a=&b=%E5%8C%97%E5%A4%A7%E5%A5%B3%E7%94%9F%E8%B0%88%E4%B8%BA%E4%BD%95%E7%8B%82%E5%98%98%E4%BA%8E%E4%B8%B9%EF%BC%9A%E5%BF%83%E7%81%B5%E9%B8%A1%E6%B1%A4%E4%BA%B5%E6%B8%8E%E5%9B%BD%E5%AE%9D(%E5%9B%BE) |title=北大女生谈为何“嘘”于丹：心灵鸡汤亵渎国宝|accessdate=2012年11月19日 |publisher=搜狐网 |archiveurl=http://wei.sohu.com/20121119/n357997236.shtml?pvid=tc_news&a=&b=%E5%8C%97%E5%A4%A7%E5%A5%B3%E7%94%9F%E8%B0%88%E4%B8%BA%E4%BD%95%E7%8B%82%E5%98%98%E4%BA%8E%E4%B8%B9%EF%BC%9A%E5%BF%83%E7%81%B5%E9%B8%A1%E6%B1%A4%E4%BA%B5%E6%B8%8E%E5%9B%BD%E5%AE%9D(%E5%9B%BE) |archivedate=2012年11月19日 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://www.chinese.rfi.fr/%E4%B8%AD%E5%9B%BD/20121118-%E4%BA%8E%E4%B8%B9%E8%B5%B4%E5%8C%97%E4%BA%AC%E5%A4%A7%E5%AD%A6%E8%AE%B2%E6%98%86%E6%9B%B2%E5%BC%95%E5%98%98%E5%A3%B0%E4%B8%80%E7%89%87%E8%A2%AB%E8%BD%B0%E4%B8%8B%E5%8F%B0 |title= 于丹赴北京大学讲昆曲引嘘声一片被轰下台|accessdate=发表日期 2012年 11月 18日 - 更新日期  |publisher=法广中文网 |archiveurl=http://www.chinese.rfi.fr/%E4%B8%AD%E5%9B%BD/20121118-%E4%BA%8E%E4%B8%B9%E8%B5%B4%E5%8C%97%E4%BA%AC%E5%A4%A7%E5%AD%A6%E8%AE%B2%E6%98%86%E6%9B%B2%E5%BC%95%E5%98%98%E5%A3%B0%E4%B8%80%E7%89%87%E8%A2%AB%E8%BD%B0%E4%B8%8B%E5%8F%B0 |archivedate=2012年11月19日 |work= |language =zh-cn}}</ref>

== 學術成就 ==
1995年開始從事影視教學與研究工作，曾獲1996年度北京市優秀教學獎，2000年度北京市高校青年教師基本功比賽一等獎第一名，2001年度中國寶鋼教育基金優秀教師獎，曾在《中國社會科學》、《現代傳播》等國家級刊物上發表多篇文章。

== 争议 ==

由于于丹所讲解都使用白话文，且语言活靈活現，使以前许多没有直接接触《论语》、《庄子》的人从中受益。當代哲学家[[李泽厚|李泽厚]]也在采访中说：“于丹是精英和平民之间的桥梁”，<ref>[http://bjyouth.ynet.com/view.jsp?oid=19632800&pageno=2 李泽厚：于丹是精英和平民之间的桥梁] {{webarchive|url=https://archive.is/20120707232822/http://bjyouth.ynet.com/view.jsp?oid=19632800&pageno=2 |date=2012-07-07 }}</ref>对于普及儒家文化有较大贡献。

正由于于丹讲述自己的心得，导致其演讲中加入了许多个人的感受和体会，其中一些被学者认为是过于牵强附会。

于丹的走紅一直帶著爭議。2007年3月2日，《[[天涯論壇|天涯論壇]]》出現了一篇題為《我們為什麼要將反對于丹之流進行到底》的帖子，發起人為[[中山大學|中山大學]]博士徐晉如，文中言辭激烈，甚至要求于丹從《百家講壇》中下課，並向電視觀眾道歉。在帖子的結尾，包括中山大學博士生劉根勤、[[清华大学_(北京)|清華大學]]博士生王曉峰、[[暨南大學|暨南大學]]博士生周韜、中山大學副教授朱崇科以及[[北京師範大學|北京師範大學]]的學生楊暘等人聯署，聯名表態稱“要將于丹抵制到底”。
