'''于永水'''（{{bd|？|||}}）籍贯不详，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员。

==生平==
1972年12月参加工作，1975年3月加入中国共产党，大专学历。1981年9月，到[[国务院机关事务管理局|国务院机关事务管理局]]工作，历任中勤服务开发中心副总经理，机关服务中心副主任兼接待处处长，办公室副主任，西山服务局局长。2009年12月，任国务院机关事务管理局党组成员，西山服务局局长。2010年3月，任局党组成员，驻京办事处管理司司长。2011年5月，任局党组成员，机关党委书记。2014年2月，任[[国家机关事务管理局|国家机关事务管理局]]国副局长、党组成员、机关党委书记<ref>{{Cite web |url =http://renshi.people.com.cn/n/2014/0313/c139617-24622375-2.html |title =于永水、陈建明任国管局副局长 高翔、鉴保卫不再担任  |author =  |date =2014-03-13  |publisher =中国共产党新闻网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.ce.cn/xwzx/gnsz/gdxw/201403/12/t20140312_2470201.shtml|title =于永水、陈建明任国管局副局长 高翔、鉴保卫不再担任  |author =  |date =  2014-03-12|publisher =中国经济网  |language =  |accessdate =  |quote =  }}</ref>。2016年5月，于永水不再担任副局长、党组成员职务<ref>{{Cite web |url =http://district.ce.cn/newarea/sddy/201605/17/t20160517_11689846.shtml  |title =赵峰涛任国管局副局长 于永水、王卫东职务调整(简历)  |author =  2016-05-17|date =中国经济网  |publisher =  |language =  |accessdate =  |quote =  }}</ref>。
