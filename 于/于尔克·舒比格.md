'''于尔克·舒比格'''（{{lang-de|Jürg Schubiger}}，{{bd|1936年|10月14日|2014年|9月15日}}）是[[瑞士|瑞士]]儿童文学作家和[[心理治疗师|心理治疗师]]，2008年[[汉斯·克里斯蒂安·安徒生奖|国际安徒生作家奖]]得主<ref name=andersen>
{{cite web|url=http://www.ibby.org/index.php?id=273 |title=Hans Christian Andersen Awards|publisher=[[国际儿童图书评议会|International Board on Books for Young People]] (IBBY)|accessdate=2013-08-01}}</ref>。
==生平==
1936年生于瑞士苏黎世，1940年全家搬到在[[温特图尔|温特图尔]]<ref>{{cite web |title=Jürg Schubiger |url=http://www.literaturport.de/Juerg.Schubiger/ |publisher=Literarisches Colloquium Berlin |accessdate=2019-04-07 |language=de |date=2015}}</ref>。后曾在[[苏黎世大学|苏黎世大学]]学习日耳曼语言文学、心理学、哲学<ref>{{cite web|title=Jürg Schubiger|url=http://www.goethe.de/kue/lit/prj/kju/aut/sz/sc/enindex.htm|work=Goethe Institut|accessdate=2013-11-03}}</ref>，毕业论文是对[[弗朗茨·卡夫卡|弗朗茨·卡夫卡]]的研究。后来从事过多种职业，在[[科西嘉岛|科西嘉岛]]做过包装工、伐木工、[[园艺|园艺]]工人，也曾做过[[编辑|编辑]]<ref>{{cite web |title=于尔克·舒比格：一种胡说八道的童真 |url=http://news.163.com/11/0419/04/71VO3FNL00014AED.html |publisher=网易新闻 |accessdate=2019-04-07 |language=zh-cn |date=2011-04-19}}</ref>。1979年搬到苏黎世，开始了心理治疗师职业生涯，同时也为儿童和成人写书。2014年9月15日在苏黎世病逝，享年77岁<ref>{{cite web |author1=Michael Luisier |url=https://www.srf.ch/kultur/literatur/juerg-schubiger-ein-im-besten-sinne-kindlicher-schriftsteller |title=Jürg Schubiger, ein im besten Sinne kindlicher Schriftsteller|publisher=[[瑞士德语广播电视|SRF]] |accessdate=2019-04-07 |language=de |date=2014-09-16}}</ref>。
==获奖情况==
*1996年：《当世界年纪还小的时候》获德国青少年文学奖、瑞士青少年文学奖。
*2008年：《白熊和黑熊》获德意志青少年文学提名奖。
*2008年：国际安徒生作家奖。
