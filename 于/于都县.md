'''于都县'''（旧名'''{{ruby-py|雩|-{zh-cn:yú;zh-tw:ㄩˊ;zh-hk:jyu4}-}}都'''）是[[中国|中国]][[江西省|江西省]][[赣州市|赣州市]]所辖的一个县。总面积为2893平方公里，辖9镇14乡357个行政村。2003年人口为90.2万。

== 历史沿革 ==
西汉高祖六年（前201年）置县。1957年由原名“雩都”更名为“'''-{于都}-'''”<ref>{{cite web |url=http://www.xzqh.org/QUHUA/36jx/0731yd.htm |title=存档副本 |accessdate=2009-02-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20090202175412/http://www.xzqh.org/quhua/36jx/0731yd.htm |archivedate=2009-02-02 }}</ref>。

* 14个乡：[[宽田乡|宽田乡]]、[[沙心乡|沙心乡]] 、[[仙下乡|仙下乡]]、[[马安乡|马安乡]]、[[车溪乡|车溪乡]]、[[桥头乡|桥头乡]]、[[利村乡|利村乡]]、[[葛坳乡|葛坳乡]]、[[黄麟乡|黄麟乡]]、[[靖石乡|靖石乡]]、[[小溪乡|小溪乡]]、[[段屋乡|段屋乡]]、[[罗江乡|罗江乡]]、[[新陂乡|新陂乡]]。

== 地理 ==
[[File:A_photo_of_yudu_city_from_above.jpg|thumb]]
因地处东经115°11'～115°49'，北纬25°35'8"～26°20'53"之间，属[[亚热带季风气候|亚热带季风气候]]，土壤为[[红壤|红壤]]，质地粘重，雨量充沛，年均降水量1507mm，年日照时数1621.9小时，年均无霜期305天。境内有贡水、梅江、濂江等河流，其中贡水在境内的长度为66公里，河流总长1283公里，江河水面约16.2万亩，水力蕴藏量38.7万千瓦。一月和七月平均气温约为8.2℃和29.7℃，极端气温为39.9℃和－8℃，夏季闷热，冬季干冷，有寒潮、台风、高温和冰雹等灾害性天气影响。

== 人口 ==
2003年人口为90.2万。以[[客家人|客家人]]为主。

== 经济 ==
2008年国民生产总值为64.4亿元。其中第一产业13.68亿元；第二产业29.96亿元；第三产业20.76亿元。<ref>{{cite web |url=http://www.yudu.gov.cn/ydgk/zytc/ |title=存档副本 |accessdate=2010-02-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20091019053804/http://www.yudu.gov.cn/ydgk/zytc/ |archivedate=2009-10-19 }}</ref>

== 自然资源 ==
[[钨|钨]]、[[铅|铅]]、[[碳石|碳石]]、[[锌|锌]]、[[稀土|稀土]]、[[石灰石|石灰石]]、[[锡|锡]]、[[铁|铁]]、[[锰|锰]]、[[金|金]]、[[银|银]]、[[铜|铜]]等28种，尤其钨储量极丰富，钨精矿年产量达2997.26吨。
