'''-{于}-非-{闇}-'''（{{bd|1889年|4月21日|1959年|7月3日|catIdx=}}），[[满族|满族]]，原名-{于}-照，字非廠，别署'''非-{闇}-'''，又号'''闲人'''。生於[[北京|北京]]，是[[中华民国|中华民国]]和[[中华人民共和国|中华人民共和国]]工笔花鸟画家、书法家<ref name=gj>{{Cite web |url = http://news.cri.cn/gb/9223/2005/12/09/109@814746.htm |title =于非闇工笔花鸟  |author =  |date = 2005-12-09 |publisher = 国际在线 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
于非闇生于北京。幼读[[私塾|私塾]]。1908年进入满蒙高等学堂学习。1912年进入北京师范学校学习。1913年任教于北京市第二小学，同时随一位王姓民间画师学画。后来在《北京晨报》艺圃美术周刊当记者。1930年代任[[古物陈列所|古物陈列所]]国画研究馆导师，并任教于北京师范学校、京华美专、华北大学、[[国立北平艺术专科学校|北平艺专]]。1949年起，历任北京中国画研究会副会长、[[中央美术学院|中央美术学院]]民族美术研究所研究员、北京市文联常务理事、[[北京画院|北京中国画院]]副院长等职务<ref name=gj/>。

于非闇自幼受到家庭和周边环境影响，喜爱书画、诗文，种植花草，钓鱼养鸟，直到40余岁时才将精力集中到工笔花鸟画上。从[[白描|白描]]入手，先学的是[[赵孟坚|赵子固]]、[[陈老莲|陈老莲]]的双勾花卉，随后上溯[[宋朝|宋朝]]、[[五代|五代]]，从[[赵佶|赵佶]][[瘦金体|瘦金体]]书法中悟笔致。1936年在[[北京中山公园|中山公园]]第一次举办个人画展。他为[[院体画|院体画]]注入了新活力，通过对花鸟的观察写生，对民间绘画、[[缂丝|缂丝]]、[[刺绣|刺绣]]的借鉴，使其艺术不断获得新生命。1940年代后，风格更加成熟，有“南陈北-{于}-”（陈指[[陈之佛|陈之佛]]）之称。他对当代工笔花鸟画产生了重要影响<ref name=gj/>。
