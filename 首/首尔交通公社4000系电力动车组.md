'''首尔交通公社4000系电力动车组'''（{{kor|諺=서울교통공사 4000호대 전동차|漢=서울交通公社 4000號帶 電動車}}）是[[首尔地铁|首尔地铁]]的一款[[通勤型电力动车组|通勤型电力动车组]]，在[[首都圈电铁4号线|首都圈电铁4号线]]运营。

== 概况 ==
4000系电力动车组是[[首都圈电铁4号线|首都圈电铁4号线]]的第2款列车，分为直流用和交直两用两种车辆，由[[大宇重工业|大宇重工业]]和[[现代Mobis|现代Mobis]]负责生产。运行于[[首都圈电铁4号线|首都圈电铁4号线]]的[[首尔地铁4号线|首尔地铁4号线]]区间，亦包括[[韩国铁道公社|韩国铁道公社]]营运的[[果川线|果川线]]和[[安山线|安山线]]。在1993年正式使用，取代了只能运行于[[首尔地铁4号线|首尔地铁4号线]]的[[首尔地铁3000系斩波控制电力动车组|3000系斩波控制]]列车。列车客室车门为氣動内藏门，每节车厢共4对车门，列车在两端设有[[紧急出口|紧急出口]]，目前均为10辆编组，配属于[[仓洞车辆基地|仓洞车辆基地]]。

== 列车特点 ==
4000系列车分为47组，分别为编号401-426、451-471。其中401-426为直流用车辆，451-471为交直两用车辆。401-409、451-463由[[现代Mobis|现代Mobis]]负责制造（注：另外还负责制造4064、4964、4065、4965、4066、4966、4067、4967、4068、4968车厢），410-426、464-471由[[大宇重工业|大宇重工业]]负责制造（注：另外还负责制造4556、4557、4558、4559、4560、4561、4661、4562、4662、4563、4663、4763车厢）。其中除451、466、470-471号使用的控制装置为[[IGBT|IGBT]]-[[VVVF|VVVF]]之外，其他列车的控制装置均为[[可关断晶闸管|GTO]]-[[VVVF|VVVF]]。

== 列车构造 ==
4000系列车为不锈钢车体，内部拥有LED显示屏。在2003年2月18日[[大邱地铁纵火案|大邱地铁纵火案]]发生后，列车材料全部改为了防火材料。另外，列车的电源切换装置在2008年改为自动切换。值得一提的是，本系列的部分列车在[[首都圈电铁3号线|首都圈电铁3号线]]也有服役，型号为[[韩国铁道3000系电力动车组|韩国铁道3000系电力动车组]]，只不过在外涂装上有所不同。
