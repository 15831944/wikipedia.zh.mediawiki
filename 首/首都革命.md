== 背景 ==
1925年10月20日，中国共产党中央发表《对反奉战争宣言》<ref>{{Cite web|url=https://www.marxists.org/chinese/reference-books/ccp-1921-1949/01/114.htm|title=中国共产党中国共产主义青年团对反奉战争宣言（一九二五年十月二十日）|accessdate=2018-11-18|work=www.marxists.org}}</ref>，号召民众反對臨時政府。11月下旬，冯玉祥国民军攻占北京。[[中共北方区执行委员会|中共北方区委]]和[[中國國民黨|国民党]]左派人士决定趁此有利时机，合作发动一场暴力推翻段政府、建立国民政府的斗争。

== 經過 ==
兩黨在[[中华民国|中华民国]][[北洋政府|北洋政府]][[首都|首都]][[北京|北京]]，发动的一场反政府运动。其时[[段祺瑞|段祺瑞]]任[[中華民國臨時政府_(段祺瑞)|中华民国临时执政]]。运动的领导人是国民党北京市党部负责人兼[[中共北方区委|中共北方区委]]负责人的[[李大钊|李大钊]]，在京国民党员和共产党员共同参与，包括[[朱家驊|朱家驊]]和[[陈豹隐|陳啟修]]<ref>{{Cite book|url=https://books.google.com.hk/books?id=YStjDwAAQBAJ&pg=PT232&lpg=PT232&dq=%E9%A6%96%E9%83%BD%E9%9D%A9%E5%91%BD+1925&source=bl&ots=c0AXmTlH0E&sig=8rIyBmIgLYtyQxCg58WMfPQPzEI&hl=en&sa=X&ved=2ahUKEwjF1Oncjd7eAhWE-2EKHVh1AVgQ6AEwA3oECAUQAQ#v=onepage&q=%E9%A6%96%E9%83%BD%E9%9D%A9%E5%91%BD%201925&f=false|title=《內幕》|last=《內幕》編輯部|first=|date=2018-06-11|publisher=內幕雜誌社|year=|isbn=9781940063607|location=|pages=|language=zh-TW|chapter=第76期: 誰可以制約習近平 政治鬥爭野火未盡}}</ref>。計劃原本联络[[冯玉祥|冯玉祥]]的国民军配合武装起义，但是国民军临时改变主意。因此最後转为一场游行示威行动。示威中还发生火烧发表反苏言论的《[[晨報_(北京)|晨报]]》馆。

== 事後 ==
首都革命結束以後，李大釗與段祺瑞進行談判但不成功，一年後被殺<ref>{{Cite book|url=https://books.google.com.hk/books?id=7Vs7DwAAQBAJ&pg=PT341&lpg=PT341&dq=%E9%A6%96%E9%83%BD%E9%9D%A9%E5%91%BD+1925&source=bl&ots=cn9-VJ0bDi&sig=93yVa86ERk5xLtVgQmfAMtYX0yY&hl=en&sa=X&ved=2ahUKEwjF1Oncjd7eAhWE-2EKHVh1AVgQ6AEwAnoECAcQAQ#v=onepage&q=%E9%A6%96%E9%83%BD%E9%9D%A9%E5%91%BD%201925&f=false|title=趣聞河北|last=聶樹鋒/高建華編著|first=|date=2018-01-09|publisher=崧博出版事業有限公司|year=|isbn=|location=|pages=|language=zh-TW|chapter=}}</ref>。
