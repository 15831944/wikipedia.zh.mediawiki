'''首南车辆段'''，曾称'''姜山车辆段'''，是[[宁波轨道交通|宁波轨道交通]]的一座车辆段，服务于[[宁波轨道交通3号线|宁波轨道交通3号线]]。首南庄车辆段位于[[鄞州区|鄞州区]][[首南街道|首南街道]]<ref name="plan1">{{Cite web |url=http://nb8185.cnnb.com.cn/plus/view.php?aid=701105&mobile=no |title=3号线一期15个车站拟命名方案出炉 |accessdate=2018-09-06 |author=傅钟中 |date=2015-02-27 |publisher=宁波网络民生服务平台}}</ref>，地处[[宁波绕城高速公路|宁波绕城高速公路]]、宁姜公路、高阳路以及明光路，接轨于3号线一期线路南端终点[[高塘桥站|高塘桥站]]。<ref name="bidding1">{{Cite web |url=http://www.nbmetro.com/tender.php?info/673 |title=宁波市轨道交通3号线一期工程首南（姜山）车辆段（施工）招标公告 |accessdate=2018-09-06 |author=宁波轨道交通建设分公司 |date=2015-06-10 |publisher=宁波轨道交通}}</ref>

车辆段于2016年3月21日开工建设。<ref name="construction1">{{Cite web |url=http://www.cr11g.com.cn/article_show.aspx?id=19271 |title=何义斌董事长参加宁波轨道交通3号线首南车辆段项目开工动员大会 |accessdate=2018-09-06 |author=中铁十一局集团六公司 |date=2016-03-23 |publisher=中铁十一局集团|deadurl=yes|archiveurl=https://web.archive.org/web/20160820003150/http://www.cr11g.com.cn/article_show.aspx?id=19271|archivedate=2016-08-20}}</ref>

== 结构与功能 ==
车辆段规划用地面积约15.87公顷，房屋总建筑面积约6.37万平方米，设有运用库、检修库、综合楼、运转综合楼、物资总库、材料棚、调机工程车库、混合变电所、洗车库、动调试验间、杂品库、公安派出所及门卫等14个建筑单体。<ref name="construction1"/><ref name="bidding2">{{Cite web |url=http://www.nbdpc.gov.cn/art/2015/7/20/art_816_137550.html |title=宁波市轨道交通3号线一期工程首南（姜山）车辆段（施工）中标公示 |accessdate=2018-09-06 |author=宁波市发改委 |date=2015-07-20 |publisher=宁波市发改委}}</ref>
