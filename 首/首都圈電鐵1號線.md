{{首都圈電鐵1號線RDT}}
'''首都圈電鐵1號線'''（{{kor|諺=수도권 전철 1호선|漢=首都圈 電鐵 1號線|rm=Sudogwon jeoncheol ilhoseon|f=y}}）是一條連結[[京畿道|京畿道]][[東豆川市|東豆川市]][[逍遙山站|逍遙山站]]與[[仁川廣域市|仁川廣域市]][[中區_(仁川)|中區]][[仁川站_(仁川)|仁川站]]、[[忠清南道|忠清南道]][[天安市|天安市]][[天安站|天安站]]、[[忠清南道|忠清南道]][[牙山市|牙山市]][[新昌站_(韓國)|新昌站]]、[[京畿道|京畿道]][[光明市|光明市]][[光明站_(京畿道)|光明站]]與[[京畿道|京畿道]][[烏山市|烏山市]][[西東灘站|西東灘站]]等之間[[首都圈電鐵|首都圈電鐵]][[運行系統|運行系統]]。

此線由[[韓國鐵道公社|韓國鐵道公社]]與[[首爾交通公社|首爾交通公社]]共同營運。[[京元線|京元線]]、[[首爾地鐵1號線|首爾地鐵1號線]]、[[京釜線|京釜線]]、[[京仁線|京仁線]]、[[京釜高速線|京釜高速線]]、[[餅店基地線|餅店基地線]]、[[長項線|長項線]]等的各條路線交叉運行，首都圈電鐵1號線的名稱是在這些路線上運行的列車的統稱。導覽等使用的路線顏色是[[藍色|藍色]]。現在是[[首都圈電鐵|首都圈電鐵]]路線當中最長的路線。

== 京元線 ==
{{main|京元線}}
[[京元線|京元線]]在首都圈電鐵1號線指的是原來的京元線在韓國已電鐵化的路段。最初從[[清涼里站|清涼里站]]到[[議政府站|議政府站]]，後來在2000年代中期的電鐵化，使路段北延至[[逍遙山站|逍遙山站]]。沿本線一直繼續北上，可直達[[白馬高地站|白馬高地]]。

== 首爾地鐵1號線 ==
{{main|首爾地鐵1號線}}
[[首爾地鐵1號線|首爾地鐵1號線]]是這條線的核心部分，亦是[[首爾地鐵|首爾地鐵]]的第一條線，为全部地下区间，电气化方式为直流1500V。後來一度大幅擴充至原有的鐵路，既[[议政府站|议政府站]]到[[九老站|九老站]]部分。現時一般單指路線的核心部分，即由[[清涼里站|清涼里站]]到[[首爾站|首爾站]]的一段。

== 京仁線 ==
{{main|京仁線}}
[[京仁線|京仁線]]是指首都圈電鐵1號線連接[[仁川|仁川]]的路段，從首爾經過[[京畿道|京畿道]]的[[富平区|富平区]]到達仁川，並可轉乘[[仁川地鐵|仁川地鐵]]。

== 京釜線 ==
{{main|京釜線}}

首都圈電鐵1號線的京釜線路段，專指[[京釜線|京釜線]]已經電鐵化的路段，即由[[龍山站_(首爾)|龍山站]]至[[天安站|天安站]]的路段。沿本線一直繼續南下，可直達[[釜山站|釜山]]、[[順天站_(全羅南道)|順天]]、[[晉州站_(慶尚南道)|晉州]]、[[浦項站|浦項]]、[[木浦站|木浦]]、[[麗水EXPO站|麗水EXPO]]及[[光州站|光州]]。

== 長項線 ==
{{main|長項線}}
[[長項線|長項線]]可以視為[[京釜線|京釜線]]的一條支線，由[[天安站|天安站]]起始，至[[新昌站_(韓國)|新昌站]]止。沿本線一直繼續南下，可直達[[益山站|益山]]。

== 饼店基地線 ==
{{main|饼店基地線}}
[[饼店基地線|饼店基地線]]是京釜線的一條支線，由[[饼店站|饼店站]]起始，至[[西东滩站|西东滩站]]止。

== 京釜高速线 ==
{{main|京釜高速线}}
[[京釜高速线|京釜高速线]]是[[韩国高速铁道|韩国高速铁道]]的核心线路，由[[衿川区厅站|衿川区厅站]]起始，至[[光明站_(京畿道)|光明站]]止。沿本線一直繼續南下，可直達[[釜山站|釜山]]、[[晉州站_(慶尚南道)|晉州]]、[[浦項站|浦項]]、[[木浦站|木浦]]、[[麗水EXPO站|麗水EXPO]]及[[光州站|光州]]。

== 運行模式 ==

=== 急行列車 ===
*龍山 - 天安（京釜A急行）
**平日南北行開12對列車，周六及假期則開14對列車。另外餅店 - 龍山間亦開行兩對列車。
***停靠車站：龍山 - 加山數碼團地各停、安養、水原、餅店、烏山、西井里、平澤、成歡、斗井、天安。
**2013年7月起，周六及假期上午開出一班天安往龍山、黃昏由龍山返回天安的列車。
***停靠車站：龍山 - 加山數碼團地各停、水原、平澤、斗井、天安。

*首爾站（地上） - 天安（京釜B急行）
**只在平日運行。早上尖峰時間北行開出3班車、南行1班；下午尖峰時間南行開出兩班車。
***首爾站、永登浦、衿川區廳、安養、軍浦、義王、成均館大學、水原、餅店、烏山、西井里、平澤、成歡、斗井、天安。

*永登浦 - 餅店（京釜C急行）
**因應首爾廣域公車禁止乘客站立乘車而實施之替代服務。只在平日運行，早上尖峰時間北行開出兩班車、下午尖峰時間南行開出兩班車，為京釜電鐵線急行列車當中停站最少的列車。
