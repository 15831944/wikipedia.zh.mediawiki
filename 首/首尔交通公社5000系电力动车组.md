'''首尔交通公社5000系电力动车组'''（{{kor|諺=서울교통공사 5000호대 전동차|漢=서울交通公社 5000號帶 電動車}}）是[[首尔特別市都市铁道公社|首尔特別市都市铁道公社]]的一款[[通勤型电力动车组|通勤型电力动车组]]，在[[首尔地铁5号线|首尔地铁5号线]]运营。

== 概况 ==
5000系电力动车组是[[首尔特別市都市铁道公社|首尔特別市都市铁道公社]]的第1款列车，是直流用车辆，由[[现代Mobis|现代Mobis]]、负责生产。列车在1995年开始正式使用，目前运行于[[首尔地铁5号线|首尔地铁5号线]]，配属于[[高德车辆基地|高德车辆基地]]和[[傍花车辆基地|傍花车辆基地]]。列车客室车门为内藏门，每节车厢共4对车门，列车在两端设有[[紧急出口|紧急出口]]，均为8辆编组。

== 使用情况 ==
5000系电力动车组一共为80组，分为三批：
===第1代===
第1代为501-576，由現代Mobis在[[韓國|韓國]]的厂房生產。除502編組外，控制方式为[[VVVF|VVVF]]-[[可关断晶闸管|GTO]]。目前，列车的LCD显示屏已全部安装。
====第1批====
501-546編組，由現代Mobis在[[韓國|韓國]]的厂房生產。

====第2批====
547-576編組，由[[現代Mobis|現代Mobis]]在[[韓國|韓國]]的厂房生產，與第1批設計沒有分別。

===第2代===
第2代为577-580，由[[現代Rotem|現代Rotem]]生產。

====第3批====
本批次列車由[[現代Rotem|現代Rotem]](577-580)
生產，預計於2019年投入服務。

== 运送过程 ==
第1批列车从位于[[庆尚南道|庆尚南道]]的[[新昌原站|新昌原站]]出发，通过[[京釜线|京釜线]]进入[[果川线|果川线]]、[[首尔地铁4号线|首尔地铁4号线]]后，从[[忠武路站|忠武路站]]附近的联络线进入[[首尔地铁3号线|首尔地铁3号线]]，到达[[水西车辆基地|水西车辆基地]]，最后再通过卡车运至[[高德车辆基地|高德车辆基地]]。第2批列车从位于[[庆尚南道|庆尚南道]]的[[新昌原站|新昌原站]]出发，通过[[京釜线|京釜线]]进入[[首尔地铁1号线|首尔地铁1号线]]后，从[[新设洞站|新设洞站]]附近的联络线进入[[首尔地铁2号线|首尔地铁2号线]]，到达[[新亭车辆基地|新亭车辆基地]]，最后再通过卡车运至[[傍花车辆基地|傍花车辆基地]]。第3批列车从位於[[大田廣域市|大田廣域市]]的[[大田站|大田站]]出发，通过[[京釜线|京釜线]]进入[[首尔地铁1号线|首尔地铁1号线]]后，从[[新设洞站|新设洞站]]附近的联络线进入[[首尔地铁2号线|首尔地铁2号线]]，到达[[新亭车辆基地|新亭车辆基地]]，最后再通过卡车运至[[高德车辆基地|高德车辆基地]]。

==列车改造==
在2003年2月18日[[大邱地铁纵火案|大邱地铁纵火案]]发生后，列车的材料全部改为了防火材料，并且加装了火灾报警系统。列车改造由[[SLS重工业负责|SLS重工业负责]]。2012年2月，502号列车开始改为使用[[VVVF|VVVF]]-[[IGBT|IGBT]]，取代了现有的[[ABB|ABB]]制造的[[VVVF|VVVF]]-[[可关断晶闸管|GTO]]<ref>[http://news.heraldcorp.com/view.php?ud=20110621000297&md=20120422131812_AI 다원시스, 서울도시철도공사 전동차용 인버터 장치 개량 사업 낙찰 ] , 《헤럴드경제》, 2011-06-21</ref>。

== 列车当前配属 ==
*高德车辆基地：501-538、577-580
*傍花车辆基地：539-576

== 批评 ==
由于5000系第1代列车运行时产生了严重的蛇形运动，故需要加装平衡装置，但是却导致了严重的噪音，尤其在[[东大门历史文化公园站|东大门历史文化公园]]与[[钟路3街站|钟路3街]]之间最为严重。尽管[[首尔特別市都市铁道公社|首尔特別市都市铁道公社]]已经安装了减噪设备，但是效果欠佳。
