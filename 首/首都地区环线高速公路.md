'''首都地区环线高速公路'''又称'''北京大外环高速公路'''，民间俗称'''北京七环路'''。[[中国国家高速公路网|中国国家高速公路网]]编号为'''G95'''，途径[[河北省|河北省]]的[[张家口|张家口]]、[[涿州|涿州]]、[[廊坊|廊坊]]、[[承德|承德]]，以及[[北京市|北京市]]的[[大兴区|大兴区]]、[[通州区|通州区]]和[[平谷区|平谷区]]等地<ref>{{cite web|url=http://news.qq.com/a/20130605/014572.htm|title=北京“七环路”2015年通车 全长约940公里|author=刘彬|date=2015-04-16|publisher=燕赵都市报|location=北京市}}</ref>。2016年12月6日全线贯通，全长约1000公里<ref name=sina201612>{{cite web|url=http://news.sina.com.cn/2016-12-07/doc-ifxyiayr9379636.shtml|editor=郑学友|title=北京七环正式贯通全长1000公里|date=2016-12-07|publisher=北京商报}}</ref>。

首都环线高速由[[密涿高速|密涿高速]]（含[[廊涿高速公路|廊涿高速公路]]）、[[张涿高速|张涿高速]]、[[张承高速|张承高速]]、[[承平高速|承平高速]]等组成。其中北京市境内包括[[密云|密云]]至[[涿州|涿州]]高速北京段、[[承德|承德]]至[[平谷|平谷]]高速北京段，约38公里。天津境内约38公里，河北境内约924公里<ref name=sina201612 />。

首都环线高速是京津冀区域交通一体化规划中的“2环”中的内环，其中河北段預計2016年全线建成通车<ref>{{cite web|url=http://hebei.sina.com.cn/news/m/2015-10-19/detail-ifxivsce6913983.shtml|title=北京大外环河北段或明年建成 北戴河机场将通航|publisher=中国政府网|location=北京市}}</ref>，届时[[河北省|河北省]]有11条[[高速公路|高速公路]]可到达[[北京市|北京市]]，可推进京津冀交通一体化。

== 分段介绍 ==
首都地区环线高速公路由河北省和北京市范围内若干条（段）高速公路组成，以张家口起顺时针方向分别为{{China Expwy Name|S010|P=Hebei}}胶泥湾至太师庄段、{{China Expwy Name|G6}}太师庄至沙岭子段、张承高速、{{China Expwy Name|G45}}大广高速单塔子至滦河镇段、承德西环高速滦河镇至大栅子段、{{China Expwy Name|G25}}大栅子至李家营段、承平高速、{{China Expwy Name|G95}}廊坊北三县段、通州-大兴段、{{China Expwy Name|S24|P=Hebei}}廊坊西至冀京界段、{{China Expwy Name|S24|P=Hebei}}涿州至廊坊西段、张涿高速。

其中承平高速因施工难度较大推后实施，河北省和北京市方面为早日实现首都环线高速高速成环，暂采用{{China Expwy Name|G25}}李家营至京哈高速段、{{China Expwy Name|G1}}长深互通至大沙务互通段作为替代线路。2018年底{{China Expwy Name|G0121}}三河至遵化段通车后，替代线路将改经京秦高速。

{{China Expwy Name|G95}}通州至大兴段于2018年8月20日通车，标志着首都环线高速高速替代线路正式成环。<ref>{{cite web |author1=海外网 |title=首都环线高速月底贯通 "北京七环"正式闭合成"环" |url=http://mil.news.sina.com.cn/2018-06-21/doc-ihefphqk7918865.shtml |website=新浪军事 |accessdate=2018-06-23}}</ref><ref>{{cite web |author1=新京报 |title=“北京大外环”今起全线通车|url=http://www.bjnews.com.cn/news/2018/08/20/500188.html |website=新京报网 |accessdate=2018-08-20}}</ref>
== 未来规划 ==
首都环线高速唐山至廊坊段（又名唐廊高速、唐涞高速）{{fact|作为今后{{China Expwy Name|G95}}的正式线路}}正在规划建设中，建成后首都环线高速将完全绕开北京市域，为北京今后发展留出空间。
