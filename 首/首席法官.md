'''首席法官'''是大部份地區中（特別是[[普通法|普通法]]地區）高級[[法院|法院]]最高級[[法官|法官]]的稱呼。首席法官通常也是該法院的首長。英文「{{lang|en|Chief Justice}}」和「{{lang|en|Chief Judge}}」意思相似但用法不同；前者通常指[[最高法院|最高法院]]最高級法官，譯為「首席大法官」；後者指其他法院最高級法官，譯為「首席法官」。

== 中華民國 ==
{{main|司法院大法官}}
司法院大法官，是[[中華民國政府|中華民國政府]]依照《[[中華民國憲法|中華民國憲法]]》於最高司法機關——[[司法院|司法院]]中所設置，具有解釋憲法、統一解釋法律及命令之職務、審理政黨違憲解散案及[[中華民國總統|總統]]、[[中華民國副總統|副總統]]彈劾案權限之常設「機關」。

== 中华人民共和国 ==
{{main|中华人民共和国首席大法官}}
2001年修订的《中华人民共和国法官法》第十八条规定，“法官的级别分为十二级。”，“最高人民法院院长为首席大法官，二至十二级法官分为大法官、高级法官、法官”。根据该法，[[中华人民共和国|中华人民共和国]]的[[中华人民共和国首席大法官|首席大法官]]暨最高审判机关[[最高人民法院|最高人民法院]]的[[中华人民共和国最高人民法院院长|院长]]。

== 香港 ==
{{main|香港首席大法官}}
[[香港特別行政區|香港特別行政區]]的[[香港區域法院|區域法院]]、[[香港高等法院|高等法院]]及[[香港終審法院|終審法院]]各自設有首席法官。<ref>http://www.yearbook.gov.hk/2007/tc/pdf/C02.pdf</ref>終審法院首席法官的英文職銜為{{lang|en|Chief Justice}}"（過去譯為「首席[[按察司|按察司]]」或「首席大法官」），亦是[[香港司法機構|香港司法機構]]首長；其他法院的首席法官則為"{{lang|en|Chief Judge}}"。

== 美国 ==
{{main|美国首席大法官}}
[[美国|美国]]首席大法官（Chief Justice of the United States）是[[美国联邦政府|美国联邦政府]]司法部门的领袖并主管[[美国最高法院|美国最高法院]]。首席大法官是美国最高司法官员，领导最高法院的事务並在弹劾[[美国总统|美国总统]]时主持[[美国参议院|参议院]]。
