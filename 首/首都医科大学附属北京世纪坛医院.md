'''首都医科大学附属北京世纪坛医院'''，位于[[北京市|北京市]][[海淀区|海淀区]]羊坊店铁医路10号，是[[首都医科大学|首都医科大学]]附属的一所[[三级甲等医院|三级甲等]]综合医院，同时也是'''首都医科大学肿瘤医学院'''，'''北京大学第九临床医学院'''<ref>{{cite web |url = http://www.bjgrh.com.cn/show.asp?NewsID=2 |title = 首都医科大学附属北京世纪坛医院简介 |publisher = 首都医科大学附属北京世纪坛医院 |accessdate = 2013-08-17 |archive-url = https://web.archive.org/web/20140715003521/http://www.bjgrh.com.cn/show.asp?NewsID=2 |archive-date = 2014-07-15 |dead-url = yes }}</ref><ref>{{cite web |url = http://www.bjgrh.com.cn/show.asp?NewsID=23 |title=首都医科大学附属北京世纪坛医院执业信息 |publisher=首都医科大学附属北京世纪坛医院 |accessdate=2013-08-17}}</ref>。

==简介==
北京世纪坛医院毗邻北京[[中华世纪坛|中华世纪坛]]，位于[[北京西客站|北京西客站]]北侧。其前身为创建于1915年4月的“'''[[京汉铁路|京汉铁路]]医院'''”，[[中华人民共和国|中华人民共和国]]成立后相继名为“'''铁道部铁路总医院'''”、“'''北京铁道医学院附属医院'''”、“'''铁道部北京铁路总医院'''”。2004年6月11日由[[中华人民共和国铁道部|中华人民共和国铁道部]]移交[[北京市人民政府|北京市人民政府]]，由[[北京市卫生局|北京市卫生局]]管理。2005年1月1日，正式更名为“'''北京世纪坛医院'''”。2011年1月1日，更名为“'''首都医科大学附属北京世纪坛医院'''”<ref>{{cite web |url = http://www.bjgrh.com.cn/show.asp?NewsID=1 |title=穿越风雨，走过岁月！ |publisher=首都医科大学附属北京世纪坛医院 |accessdate=2013-08-12}}</ref>。

北京世纪坛医院1989年通过全国首批[[三级甲等医院|三级甲等医院]]评审。曾被[[中华人民共和国卫生部|中华人民共和国卫生部]]授予“全国百佳医院”、[[世界卫生组织|世界卫生组织]]授予“爱婴医院”等称号。
