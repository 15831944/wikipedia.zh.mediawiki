'''首都高速神奈川7號橫濱北線'''（{{lang|ja|首都高速神奈川7号横浜北線、しゅとこうそくかながわ7ごうよこはまきたせん}}）是從[[日本|日本]][[神奈川縣|神奈川縣]][[横濱市|横濱市]][[都筑區|都筑區]]到[[鶴見區_(橫濱市)|鶴見區]]的[[首都高速道路|首都高速道路]]路線。本道路同時為[[橫濱環狀道路|橫濱環狀道路]]的一部份。事業計畫名為'''橫濱市道高速橫濱環狀北線'''。通稱「'''北線'''」（{{lang|ja|きたせん、KITASEN}}）<ref>{{Cite web|url=http://shin-yoko.net/2017/02/01/kitasen2017/|title=高速「きたせん」の開通は3/18（土）16時、一般開放イベントは後日発表|accessdate=2017-03-25|publisher=新横浜新聞（しんよこ新聞）}}</ref>。

== 路線資料 ==
* 都市計畫道路名 : 横濱國際港都建設計畫道路1、4、6號高速橫濱環狀北線
* 路線名 : 高速神奈川7號橫濱北線
** 路線編號 : K7
* 起點 : [[神奈川縣|神奈川縣]][[橫濱市|橫濱市]][[都筑區|都筑區]]川向町（第三京濱道路 橫濱港北系統交流道）
* 終點 : 神奈川縣横濱市[[鶴見區_(橫濱市)|鶴見區]]生麥二丁目（首都高速神奈川1號横羽線 生麥系統交流道）
* [[車道|車道]]數 : 4車道（單側2車道）
* 設計速度 : 60[[km/h|km/h]]
* 通車時間 : [[2017年|2017年]][[3月18日|3月18日]]
== 概要 ==
[[File:Kitasen_tunnel_-01.jpg|thumb]]
[[File:横浜環状北線_新横浜.JPG|thumb]]
以解除連接沿岸部與内陸部幹線道路較少之橫濱市北部的[[交通堵塞|交通堵塞]]為目的。

為減少對周邊環境的影響，全體結構約7成是隧道。

作為[[#相關街路|相關街路]]，於岸谷生麥出入口連接，串聯[[國道1號_(日本)|國道1號]]（[[第二京濱國道|第二京濱]]）與[[東京都道、神奈川縣道6號東京大師橫濱線|東京大師橫濱線]]（產業道路）的[[都市計畫|都市計畫道路]]岸谷生麥線已整備。

== 歷史 ==
* [[2000年|2000年]]（[[平成|平成]]12年）[[7月14日|7月14日]] : 都市計畫決定。
* [[2001年|2001年]]（平成13年）度 : [[首都高速道路公團|首都高速道路公團]]著手本事業。
* [[2005年|2005年]]（平成17年）[[10月1日|10月1日]] : 隨著[[道路關係四公團|道路關係四公團]]民營化，事業主體由首都高速道路公團變更為首都高速道路株式會社。
* [[2011年|2011年]]（平成23年）[[3月15日|3月15日]] : 都市計畫變更決定（追加[[港北交流道|港北出入口]]、與[[橫濱環狀北西線|橫濱環狀北西線]]連接）<ref>{{Cite web|url=http://www.pref.kanagawa.jp/cnt/f5901/p128723.html|title=横浜国際港都建設計画道路の変更（1・4・8号高速横浜環状北西線の新規追加及び1・4・6号高速横浜環状北線の変更）について|date=2011-03-15|accessdate=2011-11-24|publisher=神奈川県|deadurl=yes|archiveurl=https://web.archive.org/web/20111104222337/http://www.pref.kanagawa.jp/cnt/f5901/p128723.html|archivedate=2011-11-04}}</ref>。
* [[2016年|2016年]]（平成28年）[[5月11日|5月11日]] : 發表路線名「'''高速神奈川7號橫濱北線'''」與各出入口、系統交流道的名稱<ref>{{Cite web|url=http://www.shutoko.jp/ss/kitasen/yokokan/news/h28_news_0511.html |title=路線や出入口等の名称が決まりました |date=2016-05-11 |publisher=首都高速道路株式会社 |accessdate=2016-05-21}}</ref>。
* [[2017年|2017年]]（平成29年）[[3月18日|3月18日]] : '''全線通車'''（馬場出入口除外）<ref>{{Cite web|url=http://www.shutoko.co.jp/company/press/h28/data/02/01_kitasen/|title=横浜北線（横羽線〜第三京浜）が2017年3月18日（土）16時に開通します|date=2017-02-01|accessdate=2017-02-01|publisher=首都高速道路株式会社}}</ref>。
