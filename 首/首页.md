'''首页'''或'''-{主}-页'''一般指[[網站|网站]]或[[网页浏览器|网页浏览器]]的初始[[網頁|网页]]，有时也被称作'''起始页'''。

== 建立目的 ==
首页一般是用户通过[[搜索引擎|搜索引擎]]访问一个网站时所看到的首个页面，用于吸引访问者的注意，通常也起到{{link-en|登陆页|landing page}}的作用。
<ref>
{{cite web |url=http://www.smartinsights.com/conversion-optimisation/landing-page-optimisation/home-page-as-landing-page/ |title=Home Page as Landing Page examples |publisher=smartinsights.com |accessdate=4 September 2014 |author=Dave Chaffey}}
</ref>
<ref name=wdi76>
{{cite book |last=Campbell |first=Jennifer |title=Web Design: Introductory |url=http://books.google.com/books?id=5IDAAgAAQBAJ&pg=PA76 |year=2014 |publisher=Cengage Learning |isbn=978-1-305-17627-0 |page=76}}
</ref>
为使用户能访问网站的其他页面，首页会提供网站的重要页面及新文章的链接，并且常常有一个搜索框。
<ref name=wdi76/>
<ref name=nn>
{{cite web |url=http://www.nngroup.com/articles/top-ten-guidelines-for-homepage-usability/ |title=Top 10 Guidelines for Homepage Usability |author=Jakob Nielsen |publisher=nngroup.com |date=12 May 2002 |accessdate=4 September 2014}}
</ref>
例如，新闻网站可能会有一个用于反映最新最热的新闻的[[動態網頁|动态首页]]，它呈现热门新闻的提要和第一段，并提供通往全文的链接。
<ref>
{{cite book |last=Kalbach |first=James |title=Designing Web Navigation |url=http://books.google.com/books?id=LC0cG-vFJbcC&pg=PA106 |year=2007 |publisher=O'Reilly Media |isbn=978-0-596-55378-4 |page=106}}
</ref>

尽管大多数网站只有唯一的首页，但其实一个网站可以有多个首页。
<ref>
{{cite book |last=Schwerdtfeger |first=Patrick |title=Webify Your Business, Internet Marketing Secrets for the Self-Employed |url=http://books.google.com/books?id=KHNNAgAAQBAJ&pg=PA151 |year=2009 |publisher=Lulu |isbn=978-0-557-04901-1 |page=151}}
</ref>
例如，[[维基百科|维基百科]]的首页除<code>wikipedia.org</code>、<code>www.wikipedia.org</code>之外，还有<code>en.wikipedia.org</code>和<code>zh.wikipedia.org</code>等。

=== 网站结构 ===
虽然有的网站只有单独的一个页面，但绝大多数网页包含首页与内容页。
<ref name=wdi94>
{{cite book |last=Campbell |first=Jennifer |title=Web Design: Introductory |url=http://books.google.com/books?id=5IDAAgAAQBAJ&pg=PA76 |year=2014 |publisher=Cengage Learning |isbn=978-1-305-17627-0 |page=94}}
</ref>
首页的[[统一资源定位符|URL地址]]通常类似于<code><nowiki>http://www.example.com/index.html</nowiki></code>或<code><nowiki>http://www.example.com/default.html</nowiki></code>。其中，<code>www</code> 为网站的[[子域名|三级域名]]、<code>example</code> 为网站的[[二级域|二级域名]]、<code>com</code> 为网站的[[頂級域|顶级域名]]。
<ref name=url>
{{cite book |last=Boyce |first=Jim |title=Windows 7 Bible |url=http://books.google.com/books?id=waCQXe7tuAoC&pg=PA318 |year=2011 |publisher=John Wiley & Sons |isbn=978-1-118-08127-3 |page=318}}
</ref>
但如果用户省略了其中的 <code>index.html</code> 或 <code>default.html</code> ，[[服务器|服务器]]还是会将他们带入相应网页，
<ref name=url/>
比如<code><nowiki>http://www.example.com/</nowiki></code>和<code><nowiki>http://example.com/index.html</nowiki></code>就都通往<code>[[example.com|example.com]]</code>的首页。这是因为<code>{{link-en|网页服务器目录索引|Webserver directory index|index.html}}</code>被保持在服务器目录的最高级别。

如果一个网站的<code>index.html</code>没有被创建，在其安全设置允许的情况下，许多[[網頁伺服器|网页服务器]]会默认地展示一个由网站目录定位的文件列表，
<ref name=dir>
{{cite web |url=https://support.tigertech.net/directory-index |title=How do I enable directory listings for a folder on my Web site? |publisher=tigertech.net |accessdate=4 September 2014}}
</ref>
通常包含着到各个文件的[[超連結|超链接]]和简单的[[檔案分享|文件共享]]功能，但不会有单独的索引文件。

=== 瀏覽器主頁 ===
首页有时也指用户打开[[网页浏览器|网页浏览器]]后所进入的第一个网页，即'''起始页'''。起始页既可以是网站，也可以是各种功能页面——例如，它可以展示用户经常去的网站的缩略图。多数网页可被设为起始页，某些网站，例如 [[Msn#Msn.com|Msn.com]]、{{link-en|My Yahoo!}} 和 [[iGoogle|iGoogle]]（现已终止服务）的设计就是为了被设置为初始页，<ref>{{cite news |url=http://www.theguardian.com/technology/askjack/2013/nov/07/igoogle-alternatives |title=iGoogle: what are the best alternatives? |author=Jack Schofield |date=7 November 2013 |accessdate=4 September 2014 |work=The Guardian}}</ref>通过它们，可以发[[Webmail|Webmail]]、看天气预报等。

== 其它 ==
[[个人网站|个人主页]]也常被称作首页，尽管它们可能包含很多页面。
<ref>
{{cite book |last1=Crowder |first1=Phillip |last2=Crowder |first2=David A. |title=Creating Web Sites Bible |url=http://books.google.com/books?id=6jR9D2mOimQC&pg=PA16 |year=2008 |publisher=John Wiley & Sons |isbn=978-0-470-37259-3 |page=16}}
</ref>

在德语中，“首页”常作为“网站”的同义词。
<ref>
{{cite web |url=http://www.duden.de/node/733426/revisions/1647189/view |title=„Homepage“ auf Duden online |author=Dudenredaktion (o. J.) |publisher=Dudenverlag |language = de  |accessdate=13 June 2017 }}
</ref>

在浏览器环境外，首页一词也常被使用，比如，它常指移动设备如[[移动电话|手机]]的[[主界面|主界面]]。
