'''首都圈電鐵3號線'''（{{kor|諺=수도권 전철 3호선|漢=首都圈 電鐵 3號線|羅=Sudogwon jeoncheol samhoseon|f=y}}）是一條連結[[大韓民國|大韓民國]][[京畿道|京畿道]][[高陽市|高陽市]]{{link-ko|一山西區|일산서구}}[[大化站|大化站]]與[[首爾特別市|首爾特別市]][[松坡區|松坡區]][[梧琴站|梧琴站]]，屬於[[首都圈電鐵|首都圈電鐵]]的[[運行系統|運行系統]]。3號線原來只意味着[[首爾地鐵3號線|首爾地鐵3號線]]，現在則包括了與[[首爾地鐵3號線|首爾地鐵3號線]][[直通運行|直通運行]]的[[一山線|一山線]]。導覽使用的顏色是[[橙色|朱黃色]]。

1985年10月18日[[首爾地鐵3號線|首爾地鐵3號線]][[舊把撥站|舊把撥]]至[[良才站|良才]]路段開通，開始營運，1993年10月30日[[首爾地下鐵2期|2期地下鐵]]構成的[[紙杻站|紙杻站]]與[[良才站|良才]]至[[水西站_(首爾)|水西]]路段開通，1996年1月30日以[[紙杻站|紙杻站]]為起點建設的[[一山線|一山線]]完工，[[首爾地鐵3號線|3號線]]至[[一山線|一山線]]即現在的運行系統終於成立。2010年2月18日[[水西站_(首爾)|水西]]至[[梧琴站|梧琴]]路段開通，運行路段因此延長。

== 首爾地鐵3號線 ==
{{main|首爾地鐵3號線}}
[[首爾地鐵3號線|首爾地鐵3號線]]是這條線的核心部分，即由[[首尔|首尔]][[松坡区|松坡区]]的[[梧琴站|梧琴站]]到[[京畿道|京畿道]][[高阳市|高阳市]][[德阳区|德阳区]]的[[纸杻站|纸杻站]]的一段。

== 一山线 ==
{{main|一山线}}
[[一山线|一山线]]是指首都圈電鐵3號線在[[京畿道|京畿道]][[高阳市|高阳市]]的路段，從[[高阳市|高阳市]][[德阳区|德阳区]]的[[纸杻站|纸杻站]]到[[一山西区|一山西区]]的[[大化站|大化站]]。
