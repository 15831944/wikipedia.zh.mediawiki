'''忠正路站'''（{{kor|諺=충정로역|漢=忠正路驛|羅=Chungjeongno yeok|f=y}}）位於[[韓國|韓國]][[首爾特別市|首爾特別市]][[西大門區|西大門區]]忠正路3街，是[[首爾地鐵2號線|首爾地鐵2號線]]與[[首爾地鐵5號線|首爾地鐵5號線]]的[[轉乘站|轉乘站]]。站名來自車站本身位於忠正路，5號線位於{{link-ko|忠正路|충정로}}（鐘路方向），2號線位於{{link-ko|西小門路|서소문로}}（市廳方向）。副站名是「京畿大入口」（{{lang|ko|경기대입구}}）<ref>[[首爾交通公社|首爾交通公社]]、[[首爾地鐵9號線株式會社|首爾地鐵9號線株式會社]]所屬的所有車站括號內書寫的部分都是首爾市告示的官方站名。與[[韓國鐵道公社|Korail]]的副站名制度不同。</ref>，以鄰近的[[京畿大學|京畿大學]]命名。與[[麻浦區|麻浦區]]也很接近。另外在[[首爾地鐵5號線|首爾地鐵5號線]]，此站至[[新金湖站|新金湖站]]為止都是連續的[[島式月台|島式月台]]。

== 車站結構 ==
兩條路線都設有曲線結構的[[島式月台|島式月台]]，出口共10個。兩條路線都設有[[月台幕門|月台幕門]]。

== 车站周边 ==
*[[京畿大学|京畿大学]]首尔校园
*[[海洋水產部|海洋水產部]]（向[[東亚日報|東亚日報]]租用的大厦）
*江北三星医院（[[西大門站|西大門站]]比本站更接近）
*[[韓国经济新聞|韓国经济新聞]]社（[[西大門站|西大門站]]比本站更接近）
*[[法國駐韓大使館|法國駐韓大使館]]（使用2号線側出口）
*[[東亚日報|東亚日報]]社忠正路社屋（和海洋水産部共用同一座建筑）

== 历史 ==
* 1983年6月30日：站名決定為'''忠正路站'''<ref>[http://www2.seoul.go.kr/snews/data/CN_MST/1983-972.pdf#page=7&view=FitV&scrollbar=1&toolbar=1&statusbar=1&messages=1&navpanes=1 서울특별시고시 제326호], 1983년 6월 30일.</ref>。
* 1984年5月22日：[[首爾地鐵2號線|首爾地鐵2號線]][[首爾大入口站|首爾大入口]]－[[乙支路入口站|乙支路入口]]路段開通的同時開始營業（出口6個）。
* 1996年12月30日：[[首爾地鐵5號線|首爾地鐵5號線]]開通，成為[[轉乘站|轉乘站]]（出口擴建至10個）。
