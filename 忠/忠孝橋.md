'''忠孝橋'''是位於[[臺灣|臺灣]]北部河川[[淡水河|淡水河]]的[[橋樑|橋樑]]，啟用於1982年5月1日，為[[臺北盆地|臺北盆地]]內重要的聯絡通道之一，同時劃入[[台1線|台1線]]路段。全橋以[[新北市|新北市]][[三重區|三重區]]重安街為起點，跨越淡水河河道後，連接至[[臺北市|臺北市]][[大同區_(臺北市)|大同區]]、[[萬華區|萬華區]]的[[忠孝西路|忠孝西路]]與西寧南北路口。主橋全長519公尺{{r|大台北縣市聯外橋梁伸縮縫維修實務之探討}}、寬31.5[[公尺|公尺]]，設有6線快車道，2線慢車道與[[人行道|人行道]]{{r|大台北縣市聯外橋梁伸縮縫維修實務之探討}}。

忠孝橋是距離[[臺北車站|臺北車站]]最近的淡水河橋梁，東向（臺北市區端）銜接[[環河南北快速道路|環河南北快速道路]]、[[市民大道高架道路|市民大道高架道路]]以及[[臺北站前商圈|臺北站前商圈]]，西向（三重端）則銜接中山路高架道路、[[新北大道|新北大道]]與臺一線高架道路。東向原本還連結1978年10月10日通車的北門高架道路（又稱北門高架橋）<ref>[http://nco.gov.taipei/public/Data/612617101071.ods 車行陸橋一覽表 - 臺北市政府工務局新建工程處]</ref>，該道路跨越當時還以平面方式行經於此的[[縱貫線_(北段)|縱貫線]]鐵路，並兼具忠孝橋的[[引道|引道]]功能（一般稱忠孝橋引道），紓解周邊因[[平交道|平交道]]而造成的[[壅塞|壅塞]]現象<ref>[http://cwhung.blogspot.tw/2016/02/blog-post.html 飛行場の測候所: 死了都要幫它正名的「北門高架橋」再會啦！]</ref><ref>{{Cite web |url=http://reading.gov.taipei/ct.asp?xItem=166051857&CtNode=87680&mp=100021 |title=閱讀臺北主題網 ─ 拆除忠孝橋引道／北門再現古城風華 |access-date=2017-04-04 |archive-url=https://web.archive.org/web/20170404131018/http://reading.gov.taipei/ct.asp?xItem=166051857&CtNode=87680&mp=100021 |archive-date=2017-04-04 |dead-url=yes }}</ref>；由於北門高架道路阻礙[[臺北府城北門|臺北府城北門]]景觀甚大，在跨越的縱貫線鐵路於1989年的[[臺北鐵路地下化專案|臺北鐵路地下化專案]]完成後全面地下化，[[臺北市政府|臺北市政府]]最終在2016年春節期間將其全面拆除，忠孝橋與臺北車站間則改以619公尺長的平面道路連接。

== 歷史 ==
*[[1977年|1977年]][[7月31日|7月31日]]，[[颱風薇拉_(1977年)|強烈颱風薇拉]]侵襲北台灣，造成正在興建中的北門高架橋延平北路下行引道工程鋼架被吹落壓毀車輛。
*1978年，台北市區端626公尺路段由[[台北市政府|台北市政府]]動工興建。
*1979年，三重端519公尺路段由[[台灣省公路局|台灣省公路局]]動工興建。
*1982年5月1日，完工通車{{R|中央日報}}。當時橋梁為[[收費道路|收費道路]]，在三重端設有[[收費站|收費站]]，使用車輛必須繳交{{link-en|道路收費|Road pricing|過路費}}。
*1993年11月1日，凌晨起停止徵收過路費，同時拆除橋上收費站，減輕了尖峰時段嚴重的塞車問題。
*1993年，北門高架橋延平北路下行引道拆除；1995年，北門高架橋延平南路下行引道拆除；2000年，北門高架橋塔城街上行引道拆除。<ref>https://www.facebook.com/zhangzhesheng/photos/a.10153089427279531.1073742074.124164094530/10153555040009531</ref>{{better source}}
*2016年2月7日至2月13日，台北市政府拆除北門高架橋，[[臺北府城北門|臺北府城北門]]被高架橋遮蔽達39年的景觀問題解決。<ref> {{Cite web |url=http://news.ltn.com.tw/news/politics/breakingnews/1591186 |title=忠孝橋引橋拆除 柯文哲：8天一定要拆完 |date= 2016年2月1日 |publisher= 自由時報  |language= zh-tw }}</ref><ref> {{Cite web |url= http://news.ltn.com.tw/news/life/breakingnews/1597733 |title=忠孝橋引橋走入歷史 凌晨四時正式拆除 |date= 2016年2月7日 |publisher= 自由時報  |language= zh-tw }}</ref>。

== 設計 ==
忠孝橋由橋樑結構工程師[[林同棪|林同棪]]設計，忠孝橋上結構呈T字形{{r|廖耀宇}}，主橋長519公尺，寬31.5公尺，往返各3線快車道、1線機車道及1線人行道。其上構為每跨11支的預力混凝土Lin-T梁，下構在牆式墩柱兩端向四角斜伸出四支柱子呈Y型，末端串以短梁構成框架，延長跨距以減少落墩數。橋墩呈Y字形，剛好與「同棪」第一個英文字母同，巧妙地將設計者 T.Y. LIN（林同棪）的名字嵌入設計中{{r|廖耀宇}}。
