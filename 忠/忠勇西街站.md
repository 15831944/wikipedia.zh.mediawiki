'''忠勇西街站'''是[[桃園捷運|桃園捷運]][[桃園捷運綠線|綠線]]（興建中）的一座車站，位於[[桃園市|桃園市]][[八德區|八德區]]，預計2025年通車。<ref>{{cite web|url=https://udn.com/news/story/7266/3416237|title=桃園捷運綠線兩段動工時間定了 等待10年綠捷啟動|first=|last=聯合新聞網|website=聯合新聞網}}</ref>

== 車站概要 ==
車站位於八德區介壽路一段與忠勇西街路口附近，車站代碼為'''G5'''

== 車站結構 ==
預定設地下二層島式月台車站

== 利用狀況 ==
尚未通車，興建中

== 車站周邊 ==
* 桃園市大勇國小
* 東培工業股份有限公司(桃園廠)
