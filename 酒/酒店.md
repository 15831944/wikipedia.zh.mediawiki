'''酒店'''，又稱'''旅舍'''、'''賓館'''，是提供安全、舒適，讓使用者得到短期的[[休息|休息]]或[[睡眠|睡眠]]空間之商业机构。

旅館產業是服務業，販售主要的是無形服務，不是有形的製品。<ref>{{cite book |author1=許秉翔 |author2=古清南 |author3=蔣家皓 |author4=許興家 |author5=吳則雄 |title=旅館管理|date=2012 |publisher=華都文化事業有限公司 |isbn=9789866090226 |url=http://www.airitibooks.com/pdfViewer/index.aspx?PublicationID=P20131218042&IsPreview=true&GoToPage=0}}</ref>

現在的飯店，除主要為[[遊客|遊客]]提供[[住宿|住宿]]服務外，亦提供生活的服务及设施（寢前服務）、[[餐饮|餐饮]]、[[游戏|游戏]]、[[娛樂|娛樂]]、[[购物|购物]]、商務中心、[[宴會|宴會]]及[[會議|會議]]等设施，部分酒店亦會提供增值服務，如免費手機使用、房車接送等。

与[[餐馆|餐馆]]类似，最早的旅舍存在于各地古代政府建立的[[驿站|驿站]]体系中。

== 服務及設施 ==
酒店房間的基本設備包括[[床|床]]、[[衣櫃|衣櫃]]、[[沙發|沙發]]、{{le|茶几|Coffee_table|-{茶几}-}}、一般亦設有[[電話|電話]]、[[電視|電視]]、[[鬧鐘|鬧鐘]]；並有[[冰箱|冰箱]]，內有各種[[飲料|飲料]]、[[酒|酒類]]及[[小食|小食]]，全部需额外收費（有酒店例外），在結帳時繳付。房間亦備有[[保溫瓶|熱水瓶]]（或[[电热水壶|电热水壶]]）、[[杯子|水杯]]及[[即溶咖啡|即溶咖啡]]、[[袋茶|茶包]]、[[奶精|奶精]]、[[食糖|砂糖]]等供住客免费使用。除了十分便宜的酒店外，房內通常都設有獨立[[浴室|浴室]]，內有[[抽水馬桶|坐廁]]、[[浴缸|浴缸]]及[[淋浴|淋浴]]設備。現在越來越多飯店設有[[Wi-Fi|WIFI]]及[[互联网|互联网]]接駁服務，採免費或需額外收費。

== 度假村 ==
[[度假村|度假村]]（resort），是指一個用作休閒及娛樂用途的建築群，作為吸引遊客遊覽及停留之旅遊景點。度假村通常由一間獨公司營運，亦有集團合營例子。為了讓遊客充份享受假期，度假村內通常設有多項設施以應付客人的一切需要，包括餐飲、酒店、體育設施、娛樂、購物以及[[賭場|賭場]]等等。

== 等級 ==
{{main|星級}}
酒店的價格及質素，通常由其提供的服務所決定。在20世紀末開始，由於全球旅客人數大幅上升，酒店的服務水準亦有不少改善，世界各國一般都以[[星級|酒店等级标准]]（由一星至五星）為酒店[[星級|星級]]。[[阿拉伯联合酋长国|阿拉伯联合酋长国]]第二大城市[[迪拜|迪拜]]的[[阿拉伯塔|阿拉伯塔]]自稱為世界唯一的七星级酒店，提供各式頂級服務、設備與國際精品品牌用品（按酒店等級標準則為五星級）。

五星級酒店是最高等級的酒店，一般包含的附屬設施也比較多，例如[[泳池|泳池]]、[[水療|水療]]中心及高級餐廳等。五星級酒店的服務水平被稱為最高級的服務，因此「五星級服務」這個詞經常出現在生活裡面，指最高級的享受和生活水平等。

== 著名酒店 ==
世界各國的著名酒店都透過其獨特傳統，舉行特別活動或重要人物入住而著名。例如[[德国|德国]][[波茨坦|波茨坦]]的[[塞琪琳霍夫宫|塞琪琳霍夫宫]]（Schloss Cecilienhof），就因為在1945年、[[第二次世界大战|第二次世界大战]]期間成為[[温斯顿·丘吉尔|邱吉爾]]、[[哈里·S·杜鲁门|杜魯門]]及[[斯大林|斯大林]]三位[[同盟國_(第二次世界大戰)|盟軍]][[元帅|統帥]]舉行[[波茨坦會議|波茨坦會議]]的地點而馳名。其他酒店則可能由特別的菜式或飲品而廣為人所知，如[[美国|美国]][[纽约|纽约]]的[[华道夫-阿斯多里亚酒店|華爾道夫酒店]]（[[華爾道夫沙拉|華爾道夫沙拉]]）與[[墨西哥|墨西哥]][[蒂華納|蒂華納]]的[[凱撒飯店|凱撒飯店]]（[[凱撒沙拉|凱撒沙拉]]）；[[新加坡|新加坡]][[萊佛士酒店|萊佛士酒店]]則發明了[[新加坡司令|新加坡司令]]。[[奥地利|奥地利]][[維也納|維也納]]的[[薩赫酒店|薩赫酒店]]則是[[薩赫蛋糕|薩赫蛋糕]]的發源地。

以豪華及獨特傳統而聞名的酒店還有[[香港|香港]][[尖沙咀|尖沙咀]]的[[半島酒店|半島-{酒店}-]]、[[中環|中環]][[香港文華東方酒店|香港文華東方酒店]]、[[新加坡香格里拉大酒店|新加坡香格里拉大酒店]]、[[印度|印度]]{{tsl|en|The Imperial, New Delhi|新德里帝國飯店}}、[[孟買|孟買]]的[[泰姬瑪哈酒店|泰姬瑪哈酒店]]、[[奧地利|奧地利]][[維也納|維也納]]的[[帝國酒店|帝國酒店]]、[[南韓|南韓]][[首爾|首爾]]的[[新羅酒店|新羅酒店]]、[[愛爾蘭|愛爾蘭]][[都柏林|都柏林]]的{{tsl|en|Shelbourne Hotel|舒爾本酒店}}及[[阿布達比|阿布達比]]的{{tsl|en|Emirates Palace|酋長皇宮}}、[[杜拜|杜拜]]的[[阿拉伯塔|阿拉伯塔]]亦以豪華馳名。

另一些酒店則透過[[流行文化|流行文化]]進入大家的腦海，例如[[英国|英国]][[伦敦|伦敦]]的[[伦敦丽兹酒店|麗茲酒店]]（Ritz Hotel）及美国紐約的[[切爾西旅館|切爾西旅館]]，是因在賣座流行歌曲中被提及而聞名，兩家酒店本身亦有不少名人光顧。其他例子則有[[加利福尼亚州|加州]]的[[比華利山飯店|比華利山飯店]]（Beverly Hills Hotel）及[[馬蒙特酒店|馬蒙特酒店]]（Chateau Marmont）。此外，[[香港麗思卡爾頓酒店|香港麗思卡爾頓酒店]]為現時世界上最高的酒店，位於102樓至118樓。

[[台灣|台灣]]國際著名飯店，多為台灣本土企業與國際飯店集團經營合作方式，如[[台北|台北]]喜來登飯店為「寒舍餐旅管理顧問股份有限公司」經營、台北W飯店為太子建設「時代國際飯店股份有限公司」經營，並非由國際飯店集團母公司直接經營。也有台灣本土企業集團自行經營，如[[長榮桂冠酒店|長榮桂冠連鎖酒店]]為「[[長榮集團|長榮集團]]」經營；[[圓山大飯店|圓山大飯店]]由[[蔣宋美齡|蔣宋美齡]]創辦、[[中華民國交通部|中華民國交通部]]管轄的財團法人台灣敦睦聯誼會所經營，是唯一帶有官方色彩的大型飯店。

== 房租 ==
酒店通常是[[出租房|出租房]]類的[[短期租約|短期租約]]。租房前，[[顧客|顧客]]多數喜歡[[格價|格價]]。已訂出的房間，酒店保證不會再[[出租|轉租]]給第三者；而客人在未事先通知的情况下到時不入住则稱為「No Show」，酒店有權按房租收費。不少酒店可使用多种[[信用卡|信用卡]]结账。

=== 房租计算时间 ===
按照国际惯例，房租計算以中午12時為界，過時住客不交房（退房）则會按照超過時間另計費用。这一国际惯例来源于欧洲。19世纪末，欧洲宾馆数量稀少、人力价格高昂，为了节约成本，酒店经营者自发形成了统一的退房时间，并在统一的时间内聘请清洁工清扫房间，逐步演变为商业惯例。

== 名称 ==
===饭-{}-店===
把现代意义上的豪华的接待客人住宿的商业性质经营机构称为“饭-{}-店”，至少可以追溯到1900年在[[北京|北京]][[东交民巷|东交民巷]]附近外资创办的[[六国饭店_(北京)|六国饭-{}-店]]。這是中国第一家、也是直至1920年代中国最知名最重要的现代意义的豪华旅店之一。1846年在[[上海租界|上海租界]]成立的[[浦江饭店|浦江饭店]]應是[[中國|中國]]大陸現存最早的高級飯店，老上海灘知名地標建築[[国际饭店|国際饭-{}-店]]，亦以飯店為名。

在部分中國大陸省份（如河北、江蘇、浙江等）常使用「饭-{}-店」的稱呼方式，如[[北京飯店|北京饭-{}-店]]、[[浦江饭店|禮查飯-{}-店]]、[[和平饭店_(上海)|和平饭-{}-店]]。在中文用法上「饭-{}-店」這名稱的使用，與[[餐馆|飯館]]、飯-{}-莊相似{{NoteTag|{{cite book |title = [[儒林外史|儒林外史]] |chapter = 第三十三回 |quote = 他住在飯-{}-店裡，日日來問，不知為甚事。}}}}。但提供住宿。

===-{酒店}-===
古代中國實行榷酤政策，-{酒店}-通常是特許行業，也稱為「旗亭」。
在台灣，「-{酒店}-」通常指[[特種行業|特種行業]]場所，常有穿性感衣物的女郎陪酒，例如[[金錢豹_(酒店)|金錢豹]]，日治台灣指有日本藝妓或台灣藝旦服務，並有水酒餐飲的地方如江山樓、東薈芳、春風樓、蓬萊閣。
另外也有些飯店是使用「-{酒店}-」名稱，例如[[長榮桂冠酒店|長榮桂冠酒店]]還有[[豪景大酒店|豪景大酒店]]等，在國外「-{酒店}-」等同於「-{飯店}-」，只是名稱使用上不同。

=== 賓館 ===
從「[[迎賓館|迎賓館]]」的用法而來，有時被使用在較低费用的旅馆场合，但也非絕對，中國部分高級酒店亦稱為賓館，如[[广州市|廣州]]的[[广州宾馆|广州宾馆]]、[[广东迎宾馆|广东迎宾馆]]、[[广东胜利宾馆|广东胜利宾馆]]（原沙面维多利亚酒店）、[[白天鹅宾馆|白天鹅宾馆]]。但在台灣，多數賓館是給情侶過夜，容易聯想到[[愛情賓館|愛情賓館]]。

=== 時租酒店 ===
在各地有一些酒店提供時租服務，供情人短聚之用。這些時租酒店會提供浪漫的環境及情趣設施（如水床），以滿足顧客需要。

香港較著名的時租酒店有[[理想酒店|理想酒店]]、維多利亞酒店。
