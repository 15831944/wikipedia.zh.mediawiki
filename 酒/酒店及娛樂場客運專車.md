'''酒店及娛樂場客運專車'''，俗稱[[發財巴|發財巴]]、[[發財車|發財車]]，是[[澳門|澳門]]各間娛樂場（賭場）提供的巴士接載服務的一個官方統稱<ref>[http://www.macaodaily.com/html/2010-08/15/content_498434.htm 澳門日報，A01（澳聞），交局指暫無例規管發財巴，2010年8月15日] {{webarchive|url=https://web.archive.org/web/20100820074317/http://www.macaodaily.com/html/2010-08/15/content_498434.htm |date=2010年8月20日 }}</ref>。發財巴並不是公共交通工具，是酒店或娛樂場提供予客人的免費交通，但其角色與公共交通無異，因其運作模式貼近公共巴士服務，其行駛路線甚至比現有兩間公共巴士公司提供的線路更多，更全面。

== 「發財巴」名字的由來 ==
發財巴是賭場商業競爭的一種手段，賭場為求招攬更多客源，都在各口岸設立發財巴站點，免費接載遊客往來口岸至賭場酒店，務求爭奪最多客源，更會向客人派發免費饮品、[[籌碼|籌碼]]代金券等，甚至有賭場舉行「百萬元抽獎」、對其下穿梭巴士乘客送贈[[叉燒飯|叉燒飯]]等<ref>[http://www.chinareviewagency.net/doc/1002/0/1/0/100201037.html?coluid=0&kindid=0&docid=100201037 中國評論新聞網，澳門賭場肉摶　何鴻燊送叉燒飯搶客，2006年9月2日]</ref><ref>[http://paper.wenweipo.com/2006/09/03/HK0609030021.htm　  免費巴士招徠澳門續增賭場 爭客戰升級，2006年9月3日]</ref>。這種招客方式，最終被坊間及媒體冠以「發財巴」的稱呼<ref>[http://www.macaodaily.com/html/2010-08/15/content_498339.htm 澳門日報，B07版，規管發財巴疏交通利社區，2010年8月15日] {{webarchive|url=https://web.archive.org/web/20100820074236/http://www.macaodaily.com/html/2010-08/15/content_498339.htm |date=2010年8月20日 }}</ref>。

== 歷史沿革 ==
以往澳門的酒店及娛樂場客運專車，主要是酒店提供予住客使用的交通工具，一般居民或非酒店旅客是不可乘搭，數量亦不多。

至2004年5月，[[金沙娛樂場|金沙娛樂場]]開業，為方便[[中國大陸|中國大陸]]及[[香港|香港]]訪澳遊客可以直接由出入境口岸乘車直達[[金沙娛樂場|金沙]]，[[金沙娛樂場|金沙]]提供由[[關閘|關閘]]及[[外港客運碼頭|外港客運碼頭]]兩個口岸直達[[金沙娛樂場|金沙]]的免費巴士服務，接載客人由口岸直達賭場，及後多間賭場爭相。自[[金沙娛樂場|金沙]]開設發財巴服務後，其他賭場均陸續仿傚，提供發財巴路線。

發財巴競爭激烈，2005年1月，[[希臘神話娛樂場|希臘神話娛樂場]]更開通免費穿梭巴士直接往來[[廣東省|廣東省]]10多個城市；澳博向乘客送贈「懷舊[[叉燒飯|叉燒飯]]」；金沙更舉行大抽獎，並在車身貼上「搭金沙車，隨時袋走一百萬獎金」的廣告；有媒體形容發財巴成為澳門的一道新風景線<ref>{{cite news| url=http://paper.wenweipo.com/2006/09/03/HK0609030021.htm| newspaper=[[文匯報_(香港)|文匯報]]| title=大送叉燒飯 免費巴士招徠澳門續增賭場 爭客戰升級| date=2006-09-03}}</ref><ref>{{cite web| url=http://www.i-busnet.com/macau/photo/super/200607.html| publisher=巴士資訊網(澳門)| title=2006年7月份勁相：金沙娛樂場專車| deadurl=yes| archiveurl=https://web.archive.org/web/20101008162101/http://i-busnet.com/macau/photo/super/200607.html| archivedate=2010-10-08}}</ref>。 在隨後數年，基本上只要有一間新賭場開業，都會提供發財巴路線。

直至2010年，基本上全澳賭場都有提供發財巴服務，提供由一條，甚至數十條路線不等，各出入境口岸、酒店、旅遊點及市區的都設有發財巴站點。部分路線更加是全日通宵行駛。

發財巴現時實際車輛數量、載客人數及路線都難以統計，但據[[澳門日報|澳門日報]]報道，威尼斯人開幕首數天，單日便出動達超過350班次發財巴接載旅客，接載過萬人次，比起澳門兩間巴士公司提供的班次不相伯仲<ref>{{cite news| newspaper =[[澳門日報|澳門日報]]| title=新賭場引龐大人流 壓力倍增 公交超負荷 搭車更艱難| date=2007-08-31| url=http://hk.crntt.com/doc/1004/3/9/9/100439917_2.html?coluid=7&kindid=0&docid=100439917| publisher=中國評論通訊社}}</ref>。

不過隨著澳門賭業於2014年起走下坡，至2016年4月止賭業收入已連續下跌22個月，加上澳門多個大型賭場牌照即將在2022年前後到期，並於2016年進行中期檢討是否收回賭牌，結果未出前，賭業更現蕭條，由2016年初起，多間大型賭場都要求賭客一定要消費才可以乘搭由賭場開出的「發財車」<ref>{{cite news| url=http://nextplus.nextmedia.com/news/latest/20160411/384723| title=悲！輸清仲要行去碼頭　澳門賭場冇免費巴士坐| newspaper=壹週Plus| date=2016-04-11}}</ref>。

== 正面影響 ==
*發財巴免費乘搭、班次頻密、點到點亦節省旅途時間，故可彌補公共交通不足的缺憾，為居民及遊客提供另外的交通選擇<ref>澳門日報，B02版（澳聞），唔使等唔使迫 免費發財巴方便快捷，2007年7月12日</ref>。

== 負面影響 ==
*各賭場為求爭客，紛紛增加發財巴路線到各口岸，然而，澳門道路設施嚴重不足，大量旅遊巴在街上行走，令路面交通更為擠塞。同時，各口岸附近為旅遊巴提供的停車位亦因過量的發財巴而不敷應用<ref>澳門日報，B10（澳聞），新聞特搜：發財巴增速度快 不加規管恐坐大，2007年10月21日</ref><ref>澳門日報，A07（澳聞），新聞特搜：遏發財巴有得失 難解公交燃眉急，2007年10月22日</ref>。
*發財巴不斷增加路線及車輛，需要大量聘請司機，人資短缺令賭場增加司機薪金；由於賭場是靠博財收益去支付發財巴開支，公共巴士公司財力上根本不可能與賭場相比，令大量巴士司機轉職駕駛薪金較高的發財巴。但因此又造成公共巴士公司司機不足，被迫縮減公共巴士班次，巴士司機又毋須擔心沒有工作，以致服務態度下降，居民候車時間增加、巴士內長期擠擁。更曾有巴士公司負責人指：「巴士數量比巴士司機還要多」的奇怪現象<ref>澳門日報，B07（澳聞）新聞小語，夏耘，總結經驗 迎接未來挑戰，2007年12月23日</ref>。

== 爭議 ==
*發財巴是賭場商業競爭的一種手段，為求吸引乘客上車，故以派發優惠券等作招徠，但因各賭場之間惡性競爭，轉為向乘客派發籌碼代金券利誘客人。由於沒有嚴格過濾派發對象，籌碼代金券派發到居民的手中，賭博風氣因而逐漸在社區蔓延，增加居民染上賭癮及成為病態賭徒的風險<ref name="ReferenceA">澳門日報，C01（澳聞），政策過份傾斜影響民生 李從正促完善規管發財巴營運調控博彩業，2007年10月22日</ref><ref>[http://www.macaodaily.com/html/2010-08/15/content_498432.htm 澳門日報，A01版，發財巴旺區拉客惹關注，2010年8月15日] {{webarchive|url=https://web.archive.org/web/20100818010933/http://www.macaodaily.com/html/2010-08/15/content_498432.htm |date=2010年8月18日 }}</ref>。

*由於澳門政府政策向博彩業傾斜，提供多項免稅優惠，當中賭場購買車輛是免收行車稅，但免稅政策受坊間質疑<ref name="ReferenceA"/>。

*大量發財巴在路面行駛，同時，在關閘的旅遊巴士站，大量發財巴同時啟動車輛，造成的空氣污染及噪音，影響居民<ref>澳門日報，C01（澳聞），發財巴爭客阻路違規變本加厲 黃耀球倡劃位徵費執罰，2007年10月22日</ref>。

*發財巴雖然是一種交通工具，但作為監管交通的交通事務局，因當前沒有專屬法規規管其運作，引至賭場巴士有增無減，政府亦不能阻止。<ref>[http://www.macaodaily.com/html/2010-08/15/content_498434.htm 澳門日報，A01版，交局指暫無例規管發財巴，2010年8月15日] {{webarchive|url=https://web.archive.org/web/20100820074317/http://www.macaodaily.com/html/2010-08/15/content_498434.htm |date=2010年8月20日 }}</ref>

*個別酒店為求搶先一步，租用內地跨境巴士做發財巴，讓旅客可在內地上車直達賭場，增加口岸車輛通關壓力。同時，跨境巴士由內地人擔任司機，外地客運企業變相經營澳門客運服務，令客運市場產生混亂，尤其內地人收入與澳門不同，燃油等營運成本亦較澳門低，這對澳門的士、巴士和旅遊巴都是處於不公平競爭狀態<ref name="ReferenceA"/>。
