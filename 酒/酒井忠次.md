'''酒井忠次'''（{{bd|1527年||1596年|12月17日|catIdx=Sakai Tadatsugu}}）是[[日本戰國時代|日本戰國時代]]中後期名將，[[德川家|德川家]]家臣。與[[石川數正|石川數正]]被人稱為'''德川二之重臣'''、[[德川四天王|德川四天王]]和[[德川十六神將|德川十六神將]]之首。他是主公松平元康的姑丈，正室為松平元康祖父[[松平清康|松平清康]]之女(也就是[[松平元康|松平元康]]之姑、[[松平廣忠|松平廣忠]]之妹)。

值得一提的是，他是唯一非德川三傑的德川四天王。

== 生平 ==
[[大永|大永]]7年（1527年）在三河[[額田郡|額田郡]][[井田城|井田城]]出生，[[松平氏|松平氏]][[譜代|譜代]]家臣[[酒井忠親|酒井忠親]]的次男。元服後仕於[[德川家康|德川家康]]的父親[[松平廣忠|松平廣忠]]，被稱為'''酒井小五郎'''、'''小平次'''、'''左衛門尉'''。

松平元康成為[[今川義元|今川義元]]的人質而前往[[駿府館|駿府館]]時，為同行家臣中年紀最大的人（23歳）。之後仕於松平元康（德川家康），弘治年間初期移往福谷城。弘治2年（1556年），柴田勝家率2000騎攻擊福谷城，忠次出城應戰，在激烈戰鬥後，將勝家擊退。

[[永祿|永祿]]3年（1560年）5月的[[桶狹間之戰|桶狹間之戰]]後，元康原本尚未打算從今川家臣屬中獨立，不僅不斷出兵騷擾織田氏的領土並攻打織田氏的城寨，同時頻頻上書給義元之子今川氏真，討伐織田信長，為今川義元報仇雪恥。然而今川氏真的無能使得駿河國內的政情越加紊亂，不少今川氏的老臣紛紛出走或逃離，使得元康終於意識到今川家確定走向敗亡的路途。但對於在[[今川義元|今川義元]]死後松平家為此陷入意見分歧的局面，忠次的兄長[[酒井忠尚|酒井忠尚]]力主親今川，與今川家繼續同盟，並怒喝道：「我三河松平家與尾張織田家代代交惡敵對，豈可與之結盟？」正當元康為此感到不知所措時(元康受今川家當主今川義元提拔培養之恩、元康與織田家當主織田信長是少時玩伴)，忠次反駁道：「主公，貓（今川氏真）與虎（織田信長），哪一個才是我們應該結盟的對象，是非常值得考慮的問題。」並且力勸元康倒向織田家，最後元康也接纳忠次的意見，同時由於得回[[岡崎城|岡崎城]]，元康更加有決心獨力，並於永禄5年（1562年）與織田信长在[[清洲城|清洲城]]結盟「[[清洲同盟|清洲同盟]]」。

永祿6年（1563年）由於主公家康脫離今川家臣屬同盟關係，但先得把三河统一及整頓領內諸多政務，因此家康開始進行一系列的改革，但却也因此引起寺社及三河豪族的不满[[三河一向一揆|三河一向一揆]]中，'''酒井忠尚'''等多數[[酒井氏|酒井氏]]其中還包括了[[本多正信|本多正信]]及[[夏目吉信|夏目吉信]]加入一揆軍，而忠次則跟隨家康平亂。永祿7年（1564年）攻擊[[吉田城|吉田城]]立下戰功，守將小原鎮實撤退，無血開城，戰後成為吉田城城主。自此忠次成為[[旗本先手役|旗本先手役]]和「東三河旗頭」統領三河東部的松平家國人（西三河為[[石川數正|石川數正]]）。

永祿12年（1569年）末，[[甲斐|甲斐]]的[[武田信玄|武田信玄]]開始侵入[[今川氏真|今川氏真]]的領國[[駿河國|駿河國]]，[[德川氏|德川氏]]與[[武田氏|武田氏]]達成協議瓜分[[今川氏|今川氏]]領國，由忠次負責與武田方進行交涉。

之後參加[[元龜|元龜]]元年（1570年）的[[姉川之戰|姉川之戰]]布陣於姉川河川沿岸，與小笠原信興的部隊一同突入朝倉軍，元龜3年（1572年）的[[三方原之戰|三方原之戰]]擔任右翼部隊，擊破[[小山田信茂|小山田信茂]]隊。[[天正|天正]]3年（1575年）的[[長篠之戰|長篠之戰]]並立下戰功。特別在長篠之戰中率領分遣隊強襲[[武田勝賴|武田勝賴]]背後的戰略要地鳶巢山砦，攻陷鳶巢山砦並討取勝賴的叔父[[河窪信實|河窪信實]]以阻止其救援[[長篠城|長篠城]]，並掃除有海村的武田支軍而立下大功。戰後被信長讚賞「望塵莫及」（{{lang|ja|背に目を持つごとし}}）。

深受家康信賴的忠次在天正7年（1579年）幫[[德姬|織田五德姬]]發送寫滿[[松平信康|松平信康]]與[[築山殿|築山殿]]壞話的信書回到織田家，暴怒的信長質問事件真偽時，忠次與[[大久保忠世|大久保忠世]]一同擔任使者前往安土城向信長解釋，但忠次老實承認信件內文千真萬確，完全沒幫信康說情美言，導致信康遭下令切腹。此舉讓家康對忠次懷恨在心，也讓天下統一後酒井家待遇羞辱性的低下。

以後仍然作為重臣仕於家康，天正10年（1582年）6月2日[[本能寺之變|本能寺之變]]發生後，返回[[岡崎城|岡崎城]]的家康計劃討伐[[明智光秀|明智光秀]]之際成為先陣。家康在信長横死後掌握了成為空白地帶的武田遺領[[甲斐|甲斐]]和[[信濃|信濃]]（[[天正壬午之亂|天正壬午之亂]]），同年6月27日忠次被派遣到信濃，對信濃的[[國人眾|國人眾]]採取懷柔政策（『[[家忠日記|家忠日記]]』）。忠次經由奥三河[[伊那|伊那]]向信濃侵攻，但對於[[諏訪賴忠|諏訪賴忠]]和[[小笠原貞慶|小笠原貞慶]]等人的離反策略以失敗告終。

天正12年（1584年）在[[小牧、長久手之戰|小牧、長久手之戰]]中出陣，在羽黑之戰擊退[[森長可|森長可]]，以家康為主的戰鬥全部有份参與。

天正13年（1585年），在與忠次同為家康宿老的[[石川數正|石川數正]]出奔後成為家康場的第一重臣，天正14年（1586年）10月24日成為家中最高位的[[從四位下|從四位下]][[左衛門督|左衛門督]]。天正16年（1588年）10月，把家督之位讓給長男[[酒井家次|家次]]後隱居。但是之後在[[京都|京都]]被[[豐臣秀吉|豐臣秀吉]]賜予京都櫻井的屋敷、侍女和在京費用1千石。之後出家號為「一智」

天正18年（1590年）家康被移封至[[關東|關東]]，因為忠次已經隱居，所以家次被封[[下總|下總]][[臼井|臼井]]3萬石。

[[慶長|慶長]]元年（1596年）10月28日在京都櫻井屋敷死去。享年70歲。墓所在[[知恩院|知恩院]]的塔頭[[先求院|先求院]]。墓在知恩院山腹的墓地內。

== 人物‧逸話 ==
[[file:Sakai_Tadatsugu's_drum.jpg|thumb]]畫）]]
*在[[三方原之戰|三方原之戰]]中，德川軍大敗潰散，人人驚慌失措，回到[[濱松城|濱松城]]的忠次為了鼓舞己方的士氣而擊打[[太鼓|太鼓]]，當時相當不安的士兵聽到太鼓聲後漸漸平復下來。這個逸話出自[[新歌舞伎十八番|新歌舞伎十八番]]的其中一編「太鼓音智勇三略」，通稱「酒井之太鼓」（{{lang|ja|酒井の太鼓}}），聽到鼓聲的武田軍認為可能有著什麼奇策，於是全軍後退。但是這個逸話真偽不明。

*一說在三方原合戰時，家康被武田軍威勢嚇得失禁，而見此狀的忠次則大笑不已。

*[[元龜|元龜]]4年（1573年）正月，武田家派人送來「松木既枯，則竹斷之日亦不遠矣」（{{lang|ja|松枯れで竹類なき明日かな}}）的詩句，隱喻家康將要敗亡（家康幼時名為竹千代）。德川家上下無不暴怒，而忠次則提筆將詩句改為「松木既枯，則武田頭斷之日亦不遠矣」（{{lang|ja|松枯れで武田首なき明日かな}}）并遣人送回武田家（日語中，竹和武田的讀音分別是「たけ」和「たけだ」，兩者只差一音）。一說自此之後便有了在正月將門前竹飾斜著劈開的習俗。

*忠次的愛槍為「甕通槍」，有穿過甕殺死敵人的逸話。

*很擅長跳一種名為「海老掬」（{{lang|ja|海老すくい}}）的舞蹈，曾在重臣和諸將面前表演助興。[[天正_(日本)|天正]]14年（1586年），家康為了與[[北條氏政|北條氏政]]結成同盟而前往[[伊豆三島|伊豆三島]]時，忠次在酒宴上表演。

*嫡男家次被封臼井3萬石時，相比起[[德川四天王|德川四天王]]其餘三人為10萬石的規模顯得十分寒酸。為此忠次請求家康增加兒子封賞，卻被家康反諷「你有善待我的兒子嗎？」（{{lang|ja|お前も我が子が可愛いか}}），忠次無言以對黯然退下。

*在[[長篠之戰|長篠之戰]]中，忠次在信長的本營[[極樂寺山|極樂寺山]]進行軍議時献策，最初信長叱責「像這樣的小技倆是沒用的」（{{lang|ja|そのような小細工は用いるにあらず}}）。但是在軍議結束，諸將退下後，忠次被信長秘密召喚，並以親和的態度說「之前的叱責是因為恐怕會把計劃洩露，現在立即出擊吧」（{{lang|ja|先に叱責せしは間者に事が漏洩せんことを恐るるがため、すぐに出立せよ}}），於是忠次率領3千士兵，再加上[[金森長近|金森長近]]等鉄炮隊5百人，一同前往奇襲鳶巢山砦。（『[[常山紀談|常山紀談]]』）

== 系譜 ==
[[酒井氏|酒井氏]]的祖先和[[松平氏|松平氏]]的祖先是親族和兄弟（婚姻關係亦被稱作兄弟），更是被稱為安城譜代的松平家中最古老的[[宿老|宿老]]，忠次亦與松平氏有著相當深厚的血緣關係。即是忠次的正室是家康的祖父[[松平清康|松平清康]]和夫人[[華陽院|於富之方]]所生的[[碓井姬|碓井姬]]。還有因為於富之方成為松平清康正室之前是[[水野忠政|水野忠政]]的正室，家康的母親[[傳通院|於大之方]]的實母，忠次是家康父母雙方的妹妹的丈夫，義理上是家康的叔父。

與碓井姬之間生有[[酒井家次|家次]]、[[本多康俊|本多康俊]]，特別是繼任家督的家次由[[下總|下總]][[臼井藩|臼井藩]]3萬石成為[[越後|越後]][[高田藩|高田藩]]10萬石，子孫在最終變為[[出羽|出羽]][[庄内藩|庄内藩]]17萬石和[[譜代|譜代]]而有著相當份量的家族。其他庶流在出羽中佔有3個藩。還有繼承[[本多氏|伊奈本多氏]]的康俊流系一直持續到[[幕末|幕末]]為止。
