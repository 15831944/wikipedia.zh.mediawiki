'''釋奠佾舞'''簡稱佾舞，又稱丁祭佾舞、祭祀大成至聖先師[[孔子|孔子]]之佾舞。依不同編制又分有'''六佾舞'''和'''[[八佾舞|八佾舞]]'''，是[[祭孔|釋奠典禮]]的祭禮中所[[表演|表演]]的[[舞蹈|舞蹈]], 是[[東亞|東亞]]世界重要的[[非物質文化遺產|非物質文化遺產]]。

== 歷史 ==
中國[[南朝|南朝]][[齊武帝|齊武帝]]永明三年（西元485年），對於釋奠已有記載：「元嘉立學、裴松之儀、應舞六佾，以郊樂未具，故從權奏登歌，今金石已備，宜設軒縣之樂，六佾之舞，[[牲牢|牲牢]]器用，悉依[[上公|上公]]」，六佾即六佾舞，是中國古代[[諸侯|諸侯]]用舞的編輯，有六行六列，共三十六人。軒縣就是軒懸，是中國古代[[諸侯|諸侯]]樂隊的編制。這是歷史上樂舞並用祭孔儀式的開始，之後各朝代祭孔的規模和樂舞均有不同。

[[北魏|北魏]][[孝文帝|孝文帝]]太和十六年（492年），孝文帝時六佾舞中增加了「三獻禮」的項目，三獻禮即初獻、亞獻、終獻。[[唐朝|唐朝]][[唐玄宗|玄宗]]開元二十八年（740年）的祭孔大祀，是採用六十四人的八佾舞。{{sfn|陳昭瑛|2008|pp=74-75}}

佾舞源自宗廟宮庭雅樂舞，舞者稱為「佾生」，佾是隊伍的行列，分祭天子、公侯、大夫、士，又有八佾、六佾、四佾、二佾之分。與宮庭樂舞相同，釋奠佾舞也有文舞和武舞。跳文舞時右手執羽，常用雉尾，左手執籥，即短笛形的竹管，分別有立容、立聲之意。每個動作皆代表一個字，一節樂曲一組動作。武舞又稱干舞。武舞持幹（盾牌）戚（斧鉞）而舞。

唐朝[[貞觀|貞觀]]年間的祭孔佾舞為文舞、武舞並用，唐玄宗開元年間曾用八佾。[[宋朝|宋朝]]僅用文舞，表示謙遜禮讓之義。[[明朝|明朝]]文舞武舞並用，採六佾，文舞佾生右手執三羽。清[[順治|順治]]年間，以京師[[國子監|國子監]]為[[太學|太學]]，並設立孔廟，春秋兩季派遣大學士致祭，諡孔子為「大成至聖文宣先師」，後改諡孔子為「至聖先師孔子」，而各省府州縣比照太學孔廟祭典，當地總督、巡撫、學政皆須率領所屬官員祭致。{{sfn|高明士|2007|pp=174-175}}

釋奠佾舞共有三成，各三個樂章，每個樂章由四言八句（32字）的詩詞組成，總共有96個動作。佾舞自漢朝即有，但現今所存的佾舞舞譜只有明、清兩代。明朝的舞譜又有前期的《南雍志》和後期的《頖宮禮樂疏》，[[台灣|台灣]]在滿清時期所用的佾舞是依乾隆年間的《重修臺灣縣志》和《重修鳳山縣志》中的文字譜。

== 服式 ==
目前尊行的釋奠禮服以[[民國18年|民國18年]]（1929年）公布《服制條例》所規定中式禮服為主，服飾擷取歷代之精華，彙為今制<ref>[http://lis.ly.gov.tw/lghtml/lawstat/version2/04904/0490418041600.htm 服制條例(04904)]</ref>。《服制條例》今已廢止，目前台灣依舊慣成俗。原臺灣各地文廟全遵循清代釋奠禮制，現今惟台南祭孔儀式尊崇源自本地「雅樂十三音」儀軌，因此保有全體司禮人員均著清代禮服之俗。

=== 國家禮服 ===
國服以近代滿式服飾為基礎，依照祭祀司禮層級區分：正獻官、分獻官、陪祭官、糾儀官與[[衍聖公|奉祀官]]等，需以[[國服|國家禮服]]為禮服。

=== 漢式禮服 ===
漢服以傳統[[漢服|漢服]]為基礎，依照祭祀司禮層級區分：通贊、引贊、樂長、樂生、禮生與佾生（舞生），需以漢式禮服為禮服。

== 雅樂 ==
現行的祭孔雅樂存在清制（清光緒十七年）與明制（民國六十七年復制）<ref>{{cite web|title=中華民國文化部：文廟釋奠樂|url=http://rimh.ncfta.gov.tw/retrospective/page04/0303.swf|deadurl=yes|archiveurl=https://web.archive.org/web/20140222160124/http://rimh.ncfta.gov.tw/retrospective/page04/0303.swf|archivedate=2014-02-22}}</ref>兩種系統。原臺灣各地皆遵循清代十三腔聖樂，現僅存臺南一地。

=== 大成樂章 ===
「大成樂章」又名「大晟樂」。其演奏速度較緩，風格莊嚴肅穆。目前除台南孔廟外，全國均採用明制「大成樂章」，其中以[[台北孔廟|台北孔廟]]最具代表性。大成樂章前身為十一世紀[[宋朝|宋王朝]]的「大晟樂」，明代時整編成「大成釋奠樂譜」。

=== 雅樂十三音 ===
全國僅台南文廟仍尊行「雅樂十三音」，又名「聖樂」。其發源自古府城禮樂局（今以成書院）。「雅樂十三音」演奏速度較輕快、舒暢，是目前臺灣保存、演奏歷史最悠久的雅正之樂。臺南市大成至聖先師釋奠典禮已列入[[文化資產保存法|國家文化資產]][[中華民國民俗及有關文物|民俗及有關文物]]<ref>{{cite web|title=文化部文化資產局：民俗及有關文物|url=http://www.boch.gov.tw/boch/frontsite/cultureassets/caseBasicInfoAction.do?method=doViewCaseBasicInfo&caseId=DF10008000004&version=1&assetsClassifyId=5.1&siteId=101&menuId=302|deadurl=yes|archiveurl=https://web.archive.org/web/20140222052828/http://www.boch.gov.tw/boch/frontsite/cultureassets/caseBasicInfoAction.do?method=doViewCaseBasicInfo&caseId=DF10008000004&version=1&assetsClassifyId=5.1&siteId=101&menuId=302|archivedate=2014-02-22}}</ref><ref>{{cite web|title=臺南市文化資產管理處|url=http://tmach-culture.tainan.gov.tw/actionmediashow.aspx?NSNBR=39|deadurl=yes|archiveurl=https://web.archive.org/web/20140221221525/http://tmach-culture.tainan.gov.tw/actionmediashow.aspx?NSNBR=39|archivedate=2014-02-21}}</ref>。

== 祭祀儀典 ==
釋奠儀式分為三十七個程序，主要祭儀共分：排班就位、迎神、初獻禮、亞獻禮、終獻禮、飲福受胙、送神、望燎等。在釋奠典禮前，先進行祭祀至聖先師五代世祖的「崇聖祠祭五王」的儀式。<ref>http://tw.news.yahoo.com/%E5%8F%B0%E5%8D%97%E7%A5%AD%E5%AD%94-%E8%A9%95%E4%BC%B0%E7%99%BB%E9%8C%84%E5%9C%8B%E5%AE%9A%E6%B0%91%E4%BF%97-150600817.html</ref>。釋奠典禮前的預演則稱為「釋奠習儀」。<ref>http://www.wretch.cc/blog/cherry5251/26255777</ref>

== 台灣各地釋奠佾舞 ==
[[台南孔廟|台南文廟]]始建於[[明鄭時期|明鄭時期]][[永曆_(南明)|永曆]]19年（1665年）由[[鄭經|鄭經]]於[[臺南|臺南]]興建{{sfn|臺北市孔廟管理委員會|1979|p=1}}，直到清[[康熙|康熙]]年間，才有正式的釋奠儀式。當時是採用明朝制定的釋奠樂舞。日本統治時期的樂舞儀式因當時文化而有所變革。[[台灣戰後時期|民國時期]]以後的釋奠禮樂是採清制，佾舞是沿用清制聖門樂誌上的舞譜，手執單羽。目前由忠義國小學生擔任，保持傳統，皆由男學生中挑選。

[[彰化孔廟|彰化孔廟]]的佾生由彰化中山國小、民生國小、南郭國小、平和國小輪流擔任，佾長由佾舞老師擔任，佾生皆由男學生中挑選，直到1995-1996年左右，才開始選用女學生擔任佾生。

[[臺中孔廟|臺中孔廟]]採八佾舞，佾生由臺中力行國小學生擔任。

[[嘉義孔廟|嘉義孔廟]]於1963年落成，1997年的祭孔大典中的佾生，是由蘭潭國中的學生中挑選，指導老師為鄭月娥、葉素貞及林麗珠，採明制樂譜服制，清制舞譜。服裝的頭飾是明朝朱載堉《六代小舞譜》和《靈星小舞譜》的童子舞型，戴明式假髮，插白色骨簪束紅緞帶，身穿黑色金邊黃綢袍，繫綠綢腰帶，下身為白綢褲，黑靴。

[[高雄孔廟|高雄孔廟]]採八佾舞，佾生由高雄舊城國小學生擔任。

[[桃園孔廟|桃園孔廟]]採六佾舞，佾生由桃園東門國小學生擔任。

[[臺北孔子廟|臺北孔子廟]]，[[台灣日治時期|日治時代]]由[[辜顯榮|辜顯榮]]、[[陳培根|陳培根]]等捐地興建。{{sfn|臺北市孔廟管理委員會|1979|p=1}}在1967年前的佾舞在[台灣日治時期|日治時期]]改變服飾，1931－1932年（昭和6－7年）的佾生是著白衣繫紅帶，執雉羽。到了1934年－1936年（昭和9年－11年）佾生是著白長袍黑短掛，到了1943年，日本對台灣的統治進入皇民化時期，此期的佾生服裝也稍有不同。1946年－1967年共22年間，和1968年後，佾生的服裝也有兩個不同的變革。後期至今改用[[明朝|明朝]]服飾，佾生為黃袍墨綠腰帶黑靴，節生為綠袍墨綠腰帶黑靴。長期由大龍國小學生擔任，男女生皆可，採六六三十六的佾舞編制。由佾舞老師躬身教導後學生須自行記憶所有舞步。民國60年（1971年），辜、陳後裔將臺北孔廟捐獻臺北市政府，政府接管後，民國61年（1972年）成立臺北市孔廟管理委員會，專責辦理臺北孔廟相關事務。{{sfn|臺北市孔廟管理委員會|1979|p=1}}

[宜蘭孔廟]採六佾舞，佾生由宜蘭力行國小學生擔任。

== 著名的佾舞師 ==
*莊汶湞，擔任臺北市孔廟佾舞教師十餘年，曾任教於大龍國小，已於2014年7月退休。
*楊承家（1923年2月6日－），是彰化孔廟的佾舞老師，國小時曾擔任佾生，1946年開始教導佾舞。
*盧川，曾任臺北市孔廟佾舞教師38年，已逝世十餘年。
