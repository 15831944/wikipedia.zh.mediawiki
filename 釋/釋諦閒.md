'''釋諦閒'''（{{bd|1858年||1932年|}}），法名古虛，字諦閒，號卓三，俗姓[[朱姓|朱]]，[[浙江|浙江]]省[[黃巖|黃巖]]縣（今[[中國|中國]][[浙江省|浙江省]][[台州市|台州市]][[黃巖區|黃巖區]]）人<ref>[http://dev.dila.edu.tw/authority/person/?fromInner=A001825 法鼓佛教學院－人名規範檢索－諦閒]</ref>。諦閒法師為中國清末民初高僧，平生著述豐碩，弘法演教，誨人不倦。他重振天台法脈，為[[天台宗|天台宗]]第四十三代教觀總持，對中國近代佛教發展深具貢獻。

== 生平 ==

=== 早年生活 ===
諦閒法師生於[[清朝|清]][[咸豐_(年號)|咸豐]]八年（1858年）正月初六，父親為度潤公，母親為王氏，在家中排行老三。資賦聰慧，九歲時入[[私塾|私塾]]就學，十五歲時因父親病故而輟學<ref name="年譜">{{cite web |author1=釋寶鏡 |title=諦閑大師遺集 第五編 諦公老法師年譜|url=http://www.dachengyufosi.org/e/action/ShowInfo.php?classid=838&id=61318 |publisher=大乘玉佛寺 |accessdate=2018-09-01}}</ref>。十六歲聽從母命隨舅父學醫，一日有一中年人前來求診，平日身體健壯，只因微疾，便致一病不起。法師問舅父：「藥能醫命嗎？」舅父答：「藥只醫病，安能醫命！」於是法師心中萌生出世之念，轉而尋求醫命之學。十八歲娶妻後，於黃巖城北門，開設[[中藥|中藥]]店，兼為人診病開方，以仁慈心，常為貧苦無依者義診施藥<ref>{{cite web |title=天台宗四十三世～諦閑大師 |url=http://www.budaedu.org/doctrin/t15.php |publisher=佛陀教育基金會}}</ref>。二十歲時，妻母皆亡，法師深覺世世無常，人命脆弱，厭離家獄束縛之苦，有感而發：「藥雖多、醫雖良，但療身而不療心。治病而不治命，何況業火炎炎，境風浩浩，非世法所能濟於萬一。」於是決意發心[[出家|出家]]<ref name="年譜"/>。
