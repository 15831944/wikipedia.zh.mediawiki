'''釋心保'''（1964年－），俗姓'''王'''<ref>{{cite web |url=http://www.uwest.edu/site/index.php?option=com_content&view=article&id=386&Itemid=425 |title=存档副本 |accessdate=2013-03-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20150919124620/http://www.uwest.edu/site/index.php?option=com_content&view=article&id=386&Itemid=425 |archivedate=2015-09-19 }}</ref>，法號'''慧濟'''，字'''心保'''，出生於[[中華民國|台灣]][[台中縣|台中縣]]。為現任[[佛光山|佛光山]]宗務委員會（簡稱：佛光山宗委會）宗長及佛光山現任住持。<ref>{{cite web |url=http://news.chinatimes.com/society/11050301/112013031300138.html |title=存档副本 |accessdate=2013-03-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20130316040342/http://news.chinatimes.com/society/11050301/112013031300138.html |archivedate=2013-03-16 }}</ref>心保和尚於2013年繼[[心培和尚|心培和尚]]之後成為佛光山的第九任住持、臨濟正宗第四十九代傳人。1986年來佛光山禮[[星雲大師|星雲大師]]披剃出家，1988年于美國加州[[西来寺_(加利福尼亚)|西來寺]]受三壇大戒，2004年擔任西來寺住持，2012年推選為佛光山宗委會主席暨佛光山第九任住持。<ref>[http://www.cna.com.tw/News/aLOC/201303120263-1.aspx]</ref>
