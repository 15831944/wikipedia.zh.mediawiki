'''證嚴法師'''（{{bd|1937年|5月14日|||catIdx=S釋}}），俗名'''王錦雲'''，[[法名|法名]]'''證嚴'''，字'''慧璋'''，出家前自號'''靜思'''，生於[[日治台灣|日治臺灣]][[臺中州|臺中州]][[大甲郡|大甲郡]][[清水街|清水街]]（今[[臺灣|臺灣]][[臺中市|臺中市]][[清水區_(臺灣)|清水區]]），臺灣佛教[[比丘尼|比丘尼]]，[[慈濟功德會|慈濟功德會]]的創辦者，其信徒多稱其為'''證嚴上人'''或'''師公上人'''。1963年於[[印順法師|印順法師]]門下出家，後返回[[花蓮縣|花蓮縣]][[秀林鄉|秀林鄉]]佳民村與花蓮縣新城鄉康樂村交界處的普明寺苦修，被村民稱為「農場師父」。

1966年於[[花蓮縣|花蓮縣]]創立佛教克難[[慈濟功德會|慈濟功德會]]，為慈濟基金會之前身。1986年創建花蓮慈濟醫院。1989年再創設慈濟護理專科學校。1991年，獲得[[麥格賽賽獎|麥格賽賽獎]]社會領袖獎。2003年，總統[[陳水扁|陳水扁]]頒贈二等[[景星勳章|景星勳章]]。2011年，獲[[時代雜誌|時代雜誌]]（TIME）選為2011年年度[[全球最具影響力|全球百大最具影響力]]人物之一<ref>{{cite web| url =http://www.time.com/time/specials/packages/article/0,28804,2066367_2066369_2066393,00.html|title=The 2011 TIME 100 - Dharma Master Cheng Yen|last=  |first=  |coauthors=|date=2011-04-21| publisher=Time|accessdate=2011-05-05}}</ref><ref>{{cite news | title = 證嚴法師獲選百大人物　《時代》：她已是聖人| date = 2011-04-22 | url =  http://www.nownews.com/2011/04/22/91-2706816.htm#ixzz1LRnCK7CL | work = NOWnews 今日新聞網 | accessdate = 2011-05-05}}</ref>。2012年，[[馬來西亞|馬來西亞]][[槟城|檳城州]]元首授予她拿督斯里榮譽勳銜。

== 生平 ==
釋證嚴在臺中清水出生，其弟[[王端正|王端正]]。自幼過繼給叔父家當長女，隨其遷居臺中豐原。其繼父經營劇院，往來豐原、清水、潭子等城市之間，家境小康。

1958年，21歲時，繼父因[[腦溢血|腦溢血]]過世。王錦雲後至[[豐原寺|豐原寺]]拜拜，當地住持[[妙廣法師|妙廣法師]]贈其「[[解結科儀|解結科儀]]」。其為父做佛事前往[[慈雲寺|慈雲寺]]拜《[[梁皇寶懺|梁皇寶懺]]》，體會到經文中因緣果報之業力以及人生無常之道理。同時期，一位寺院信徒將家中《[[法華經|法華經]]》當廢紙售出，她請購回家，開始研讀《法華經》。此一經書至今仍置於靜思精舍。

她時常往來寺院，萌生出家之意，經當地慈雲寺出家師父引薦，前往[[新北市|新北市]][[汐止區|汐止]]靜修院，準備[[出家|出家]]，三天後被母親尋回。王錦雲隨著慈雲寺[[修道法師|修道法師]]再度離家，由臺中前往[[高雄|高雄]]，再轉[[臺東縣|臺東縣]][[鹿野鄉|鹿野鄉]]王母廟。幾經波折，在[[臺東縣|臺東縣]][[鹿野鄉|鹿野鄉]]王母廟之後，先後移居[[知本|知本]]清覺寺、[[玉里鎮|玉里]]玉泉寺、花蓮東淨寺、臺東佛教蓮社。後寄住在臺東佛教蓮社時，至信徒家中做客，在其府中見日文版《[[法華經大講座|法華經大講座]]》，才發現法華三部中的《[[無量義經|無量義經]]》及《[[觀普賢經|觀普賢經]]》。閱讀無量義經時，看見「靜寂清澄，志玄虛漠，守之不動，億百千劫。無量法門，悉現在前，得大智慧，通達諸法。」三十二字，對無量義經產生濃厚興趣。後回到花蓮，經許聰敏居士引薦<ref>[http://enlight.lib.ntu.edu.tw/FULLTEXT/JR-BJ010/bj010209137.pdf]</ref>，王錦雲和修道法師前往花蓮縣[[秀林鄉|秀林鄉]]佳民村普明寺參加慶典，並於普明寺對面的康樂村（俗稱農場）弘法。

1962年年底，修道法師返回豐原，她留在普明寺，自行落髮，居士許聰敏為她取[[法名|法名]]「修參」。1963年，至臺北[[臨濟護國禪寺|臨濟護國禪寺]]，希望受具足戒，但因為沒有正式出家，因此無法報名。王錦雲至[[慧日講堂|慧日講堂]]購買《太虛大師全書》，遇見印順法師，請求出家，以獲得報名資格。因開壇傳戒時間緊迫，[[印順法師|印順長老]]開示：「我們[[因緣|因緣]]很特別，我看時間來不及了；但既然出家了，妳要時時刻刻為佛教、為[[眾生|眾生]]啊！」。印順長老簡單行皈依禮，為她完成出家儀式，成為她的出家[[和尚|依止師]]，為其取法名叫『證嚴』，字慧璋。她隨後報名[[三壇大戒|三壇大戒]]，受[[比丘尼|比丘尼]][[具足戒|具足戒]]。

== 出家 ==
釋證嚴受具足戒後，便返回花蓮。許聰敏居士在[[普明寺|普明寺]]後搭建一小木屋，證嚴便在其中持誦抄寫《[[法華經|法華經]]》，後因颱風倒塌，又移往花蓮慈善寺講《[[地藏經|地藏經]]》四個月，在講經期間有四位弟子皈依法師座下，後又至[[基隆|基隆]]海會寺[[結夏安居|結夏安居]]三個月；直至1964年秋，方才回到普明寺，和德慈、德昭、德融、德恩四位弟子在附近之地藏廟旁之小木屋安居、[[修行|修行]]。

1966年，29歲的釋證嚴前往花蓮[[鳳林鎮_(臺灣)|鳳林鎮]]某私人醫院探望弟子德融的父親，見地上一灘血跡，聽人告知，為[[豐濱鄉|豐濱]]一名[[臺灣原住民|原住民]]婦女[[小產|小產]]，卻因缺少開刀所需[[保證金|保證金]]新台幣八千元而無法開刀，只得又帶回部落。此即「[[一灘血|一灘血]]」事件。後來證嚴法師又遇三位修女，雙方交流後對證嚴法師慈善工作推廣產生了很大影響，這就是慈濟緣起的「三修女」來訪。有紀錄的三位修女來訪有兩次，一次是三位穿著修女會衣的[[花蓮縣私立海星高級中學|海星中學]]老師貝蒂(並非[[亞美尼亞|亞美尼亞]]童星 伊莉莎白·丹尼揚)、[[黃雪文|黃雪文]]和[[高凌霞|高凌霞]][[修女|修女]]因為郊遊路過普明寺；不期而遇的情況下，比利時籍的貝蒂修女在談話中對說：「佛教信徒這麼多、力量這麼大，如果做社會服務的工作，影響會更大。」證嚴法師大受震撼。<ref>[http://www.unjinkr.url.tw/t_2.htm 記三位和證嚴法師交談的海星中學修女：貝蒂、黃雪文和高凌霞]</ref>一次是[[張金菊|張金菊]]修女、湯修女、黃修女路過，看到證嚴法師在種菜，張金菊修女憶起當時修女一行人向證嚴法師表示：「你們為什麼不為老百姓去付出你們的愛？」證嚴法師聽聞後，決心開始慈善工作。<ref>[https://www.youtube.com/watch?v=357f03_1w_8 【回眸來時路】20150110 - 借宿到起家]</ref>。同年，[[印順法師|印順長老]]希望證嚴法師住持[[嘉義市|嘉義]]妙雲蘭若；但在其信徒挽留之下，證嚴決定留在花蓮開展社會救濟工作。

=== 慈濟功德會 ===
功德會創立初期，與幾位信徒在普明寺做手工，發放救濟品。後慈濟開始發展，普明寺的空間漸漸地越來越無法容納參加法會的信眾，釋證嚴的母親前來援助，購下一甲五分的地，並建起一間大殿，釋證嚴稱其為[[靜思精舍|靜思精舍]]。由1968年起，三十一歲的釋證嚴便逐漸興建精舍、廂房、[[寮房|寮房]]、辦公室等，歷經十多次增建，規模已十分龐大。

功德會初期，證嚴在當地會員的協助規劃之下，發行「慈濟月刊」，開始建立「慈濟委員」、「慈誠」制度。

===  創立醫院 ===
1966年，「[[一灘血|一灘血]]」事件與「三修女」的來訪，成為了推動[[醫療志業|醫療志業]]的助力。1978年，釋證嚴罹患[[心絞痛|心絞痛]]，認為「人命在呼吸間」，認為慈濟功德會沒有長久支持的資源。2001年4月，慈濟基金會三十五週年慶祝會上，李滿妹透露「一攤血」事件之事發地點為[[莊汝貴|莊汝貴]]診所。莊汝貴的子女不甘父親名譽受損，告上法庭，認為「一灘血」事件是釋證嚴虛構、捏造的。經過為期二年的官司，2003年9月，[[花蓮地方法院|花蓮地方法院]]判決釋證嚴違法，必須賠償新台幣101萬元給莊汝貴，證嚴法師決定不上訴，此案暫告落幕。

1979年，釋證嚴舉辦多次[[義診|義診]]，也因十多年來的慈善志業，發現「貧病相依」、「因病而貧」，認為臺灣東部（[[花蓮縣|花]][[臺東縣|東]]）地區長期缺乏醫療資源，下令在慈濟委員聯誼月會上，發起建立「佛教慈濟綜合醫院」。

發佈消息後，然因花費太高，起初不得眾人支持。釋證嚴於各地奔波募款，受到一些社會人士及政府單位支持，歷經當時[[林洋港|林洋港]]、[[宋長志|宋長志]]、[[李登輝|李登輝]]、[[吳水雲|吳水雲]]縣長等人協助，解決用地問題。1984年4月24日，於花蓮市區原花蓮農工牧場土地動工。1986年8月17日正式完工，隔年又展開第二期工程。

釋證嚴後聘請前故[[臺大醫院|臺大醫院]]副院長[[杜詩綿|杜詩綿]]、原臺大醫院復健科主任[[曾文賓|曾文賓]]負責籌劃工作，並相繼出任[[花蓮市|花蓮]][[慈濟醫院|慈濟醫院]]院長。

=== 教育、人文 ===
醫院建院完成後，醫院有硬體而無軟體──即醫療人員，便逐步推展教育工作，建立'''慈濟護專'''、'''慈濟醫學院'''。而護專、醫學院後來相繼改名為[[慈濟技術學院|慈濟技術學院]]（2015年改制為[[慈濟科技大學|慈濟科技大學]]）、[[慈濟大學|慈濟大學]]。而除臺灣本土社會救濟外，進而從事[[資源回收|資源回收]]的環保工作，推展[[中國大陸|中國大陸]]賑災、國際賑災等工作。於海外建立慈濟人據點、分會，要求慈濟人能懷著「取之於當地，用之於當地」、「頭頂別人的天，就要回饋當地」等理念，在全球各地推動四大志業八大法印，也得到許多當地人的肯定。{{請求來源}}

1991年中國大陸華中、華東水患，釋證嚴認為災情嚴重，便發起街頭募款、勘災、救災等工作。

== 成立「慈濟宗」 ==
2006年12月16日，釋證嚴正式宣告「[[慈濟宗|慈濟宗]]」立宗<ref name="宗1">{{cite web|url=http://www.tzuchi.org.tw/community/index.php?option=com_content&view=article&id=6530:4A9B922CBE87D9424825775B00352F1E&catid=56:you-your-family&Itemid=201|title=志工做足功課　導讀書籍談「慈濟宗」|publisher=慈濟全球資訊網|archiveurl=https://archive.is/20170130123450/http://www.tzuchi.org.tw/community/index.php?option=com_content&view=article&id=6530:4A9B922CBE87D9424825775B00352F1E&catid=56:you-your-family&Itemid=201|archivedate=2017年1月30日|deadurl=yes}}</ref>，「靜思法脈勤行道」、「慈濟宗門人間路 」。

*釋證嚴曾表示：「曾經有人問我，慈濟既不是禪宗，也不是淨土宗，屬於佛教何種宗派？現在我告訴大家，我們是『慈濟宗』，以人與人之間為道場，修行無量法門。」<ref name = "宗4"></ref>

*現代的[[慈濟大藏經|慈濟大藏經]]：《[[靜思語|靜思語]]》，是釋證嚴由《[[法華經|法華經]]》第一部《無量義經》節選釋義。<ref name = "宗1"></ref>

*慈濟宗「四法四門」<ref name = "宗2"></ref>：
# 第一道門是總持門（合心）：「總一切法，持一切善」。
# 第二道門是和合門（和氣）：「和聖賢心，合菩薩道」。
# 第三道門是關懷門（互愛）：「內觀自在心，懷抱眾生苦」。
# 第四道門是力行門（協力）：「力持諸善法，行遍人間道」。

*釋證嚴「普天三無」：「普天之下沒有我不愛的人；普天之下沒有我不信任的人；普天之下沒有我不原諒的人。」<ref name="宗2">{{cite web|url=http://www.tzuchi.org.tw/index.php?option=com_content&view=article&id=1939%3A2009-10-30-06-01-21&catid=81%3Atzuchi-about&Itemid=198&lang=en|title=靜思法脈勤行道 慈濟宗門人間路|publisher=慈濟全球資訊網|archiveurl=https://archive.is/20170130124559/http://www.tzuchi.org.tw/index.php?option=com_content&view=article&id=1939:2009-10-30-06-01-21&catid=81:tzuchi-about&Itemid=198&lang=en|archivedate=2017年1月30日|deadurl=yes}}</ref><br />

*慈濟宗「四大志業」<ref name = "宗3"></ref>：
# 慈善
# 醫療（如慈濟醫院）
# 教育（如慈濟大學、慈濟技術學院、慈濟小學、慈濟中學）
# 人文（慈濟人文志業中心、大愛電視、經典雜誌、檀施會、慈濟月刊、外語期刊）

*慈濟宗「八大法印」<ref name="宗3">{{cite web|url=http://www.tzuchi.org.tw/index.php?option=com_content&view=article&id=159%3Aintroducing-tzu-chi&catid=81%3Atzuchi-about&Itemid=198&lang=zh|title=慈濟基金會簡介|publisher=慈濟全球資訊網|archiveurl=https://archive.is/20170130124403/http://www.tzuchi.org.tw/index.php?option=com_content&view=article&id=159:introducing-tzu-chi&catid=81:tzuchi-about&Itemid=198&lang=zh|archivedate=2017年1月30日|deadurl=yes}}</ref>即「四大志業」加上：
# 國際賑災（如援助川緬）
# 骨髓捐贈（慈濟骨髓資料庫）
# 環保（如慈濟環保教育站、大愛感恩科技公司）
# 社區志工（慈濟各分支會所）

*「慈濟宗」實踐方向：<ref name = "宗6"></ref>
# 靜思法脈大愛開啟慈濟宗門

== 獎項與榮譽 ==
