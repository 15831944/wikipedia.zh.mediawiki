'''釋若慧'''（[[英文|英文]]名：{{lang|en|'''Dharma Master Ruo Hui'''}}，{{bd|1956年|10月25日|catIdx=G}}），俗名陳素卿，[[台灣|台灣]][[基隆市|基隆市]]人，臺灣大學文學院哲學研究所碩士畢業，現任[[美國|美國]][[舊金山|舊金山]][[華藏寺|華藏寺]][[住持|住持]]。

== 生平==

若慧法師，生於信佛之家，自幼濡染佛法，長養向道之志。1979年，法師於年華正盛之際，發出世之心，禮千佛山菩提寺[[釋白雲|白雲禪師]]座下出家<ref>{{cite web|url=http://www.chiefsun.org.tw/zh-tw/2_zen_master/1_edit.php |language=zh-hant |title=白雲老禪師略傳|publisher=千佛山全球資訊網|accessdate=2018-09-17 }}</ref>，同年受戒於[[新竹市|新竹]]翠壁岩寺。法師學佛不拘一門，而能深契[[佛教|佛教]]經藏義理，通習各教派理論。法師好學不倦，出家數年後復進入[[國立臺灣大學哲學系|臺灣大學文學院哲學]]研究所深造<ref>{{cite web |title=臺大哲學系-論文期刊-碩士論文|url=http://www.philo.ntu.edu.tw/thesis/graduate.php |publisher=臺灣大學文學院哲學系|accessdate=2018-11-07}}</ref>，於研究所其間，紮下深厚的經論基礎，1990年，以《[[金剛般若波羅蜜經|金剛般若波羅蜜經]]空義的研究》為主題<ref>{{cite web |author1=陳素卿|title=金剛般若經空義的研究|url=http://www.gaya.org.tw/library/thesis/index.asp?sql_form=+WHERE+%E6%A0%A1%E9%99%A2%E5%90%8D%E7%A8%B1%3D%27%E8%87%BA%E7%81%A3%E5%A4%A7%E5%AD%B8%27&ScrollAction_form=73&page_rec_form=20&abstract=&submit_form=%E8%A9%B3%E7%B4%B0 |publisher=佛教相關博碩士論文提要檢索系統|accessdate=2018-11-07}}</ref>，完成碩士論文。

千佛山於1982年成立「千佛山女子佛學院」<ref>{{cite web|url=http://www.chiefsun.org.tw/zh-tw/3_family/about-detail.php?Md=2&Id=30 |language=zh-hant |title=關於菩提寺|publisher=千佛山菩提寺 |accessdate=2018-09-18}}</ref>，法師先後擔任佛學院教務長<ref>{{cite web |title=民國70年~76年|url=http://www.chiefsun.org.tw/zh-tw/1_about/2_history.php?MID=10 |publisher=千佛山全球資訊網|accessdate=2018-10-30}}</ref>、院長等職，為奠定佛學院管理基礎及僧俗教育而奉獻。

1996年，若慧法師赴美參訪深造，數年後，法師以律己德行及言傳身教，於2004年舊金山華藏寺開寺，榮任該寺首任監院及副住持<ref>{{Cite web |url = http://www.epochtimes.com/b5/5/5/20/n927896.htm |title = 舊金山華藏寺十二位長老多來自台灣|date = 2005-05-20 |publisher = 大紀元}}</ref>。2015年10月05日，升任華藏寺住持，於大雄寶殿舉行隆重的交接儀式，帶領全寺僧眾、居士，奉行[[釋迦牟尼佛|釋迦牟尼佛]]正宗教法，以利益眾生、普渡有情為己任<ref>{{cite web|url=http://idn-news.com/news/news_content.php?catid=4&catsid=2&catdid=0&artid=20151012guisin001 |language=zh-hant |title=釋若慧法師接任華藏寺住持|publisher=自立晚報| date=2015-10-12 |accessdate=2018-09-17 }}</ref>。法師多年來著重教育工作，為佛門培育出許多優秀弘法與寺院管理人才<ref>{{cite web|url=http://www.huazangsi.org/aboutus.php?id=3 |language=zh-hant |title=華藏寺住持若慧法師|publisher=華藏寺官方網站  |accessdate=2018-09-18 }}</ref>。
