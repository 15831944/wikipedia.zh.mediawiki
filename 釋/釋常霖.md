'''葉青霖'''（{{lang-en|Alain Yip}}，{{bd|1955年|5月19日|}}），曾是[[香港|香港]]專業[[攝影師|攝影師]]，擅長拍攝人像，於[[台灣|台灣]]果如法師座下[[出家|出家]]，法號'''釋常霖'''<ref>{{cite web|url=http://ccc198.blogspot.hk/p/blog-page.html |title=「常降甘霖」|last=  |first=  |coauthors=|date=|publisher=祖師禪林網站|accessdate=2012-11-01}}</ref>。葉青霖早年曾替藝人拍攝寫真集，亦曾推出過錄影帶和光碟教授攝影技巧。後來因為接觸[[佛教|佛教]]而與佛結緣，在2010年正式出家。

==背景==
葉青霖是家中長子，總共有五兄弟姊妹<ref>{{cite book |author1=釋常霖 |title=菩提樹下的野孩子 |date=2017 |publisher=台灣土狗 |page=39 |quote=因為小時候家境窮沒錢買玩具，五兄弟姐妹就聚在家中僅有的一張床上，看我用簿紙一邊畫公仔漫畫一邊講故事，大家其樂洋洋。|url=}}</ref>，早年在[[香港島|香港島]][[西營盤|西營盤]]長大<ref>{{cite book |author1=釋常霖 |title=菩提樹下的野孩子 |date=2017 |publisher=台灣土狗 |page=11 |quote=小時候跟祖母住在西營盤區的木造頂樓加蓋屋（香港稱做天台木屋），那時候窮到什麼地步呢？|url=}}</ref>。他的其中一名胞弟為國際知名的電影美術指導[[葉錦添|葉錦添]]<ref>{{cite book |author1=釋常霖 |title=菩提樹下的野孩子 |date=2017 |publisher=台灣土狗 |page=39 |quote=弟弟葉錦添憑「臥虎藏龍」電影獲得2001年奧斯卡最佳美術指導金像獎。|url=}}</ref>，父親是[[澳門|澳門]]葡京賭場的員工，母親早逝<ref>{{cite book |author1=釋常霖 |title=菩提樹下的野孩子 |date=2017 |publisher=台灣土狗 |page=39 |quote=父親在澳門葡京賭場工作，壓力很大，因為母親逝世得早，他怕在家中無聊，一直都不肯退休。|url=}}</ref>。葉青霖畢業於同區一所私立、已停辦的小學。他在校時的成績不好，而且有閱讀障礙<ref>{{cite book |author1=釋常霖 |title=菩提樹下的野孩子 |date=2017 |publisher=台灣土狗 |page=22 |quote=這裡給自己做過的壞事定成「七宗罪」，包括逃學、說髒話、離家出走、留級、吸煙、打架、毀壞公物等。......加上有閱讀障礙，成績又不是很好，所以很孤單，於是就逃學。|url=}}</ref>。升讀[[金文泰中學|金文泰中學]]之後<ref>{{web reference|URL=http://std.stheadline.com/yesterday/edu/octopus/1112go05.html|title=金文泰創校90載 連串活動賀校慶|work=星島日報|year=2015年|date=11月12日}}</ref>曾在中二時留級一年<ref>{{cite book |author1=釋常霖 |title=菩提樹下的野孩子 |date=2017 |publisher=台灣土狗 |page=24 |quote=十三歲讀中二時留級了，成績表上只有三科及格：體育、音樂、美術，其他都是紅色的。|url=}}</ref>，而在[[香港中學會考|香港中學會考]]中，他取得八科中有四個E等、兩個D等、兩個C等的成績<ref>{{cite book |author1=釋常霖 |title=菩提樹下的野孩子 |date=2017 |publisher=台灣土狗 |page=30 |quote=以我過去的成績而論，對當時的中學會考成績是相當滿意的：八科中有四個E等、兩個D等、兩個C等，那時我高興極了。|url=}}</ref>，但因為根本不想升讀中六，結果在1974年畢業於[[金文泰中學|金文泰中學]]（是該校第49屆中五畢業生）後入讀[[香港理工學院|香港理工學院]]設計系，並在1977年取得高級文憑<ref name="葉青霖 - 見證香港藝術：多元感官及媒體藝術評賞與創作教學計劃">[http://www.hksea.org.hk/artwitness/artistteachingplan19.html 葉青霖 - 見證香港藝術：多元感官及媒體藝術評賞與創作教學計劃，2006年8月]</ref>。

他在1979年創辦[[現代經典|現代經典]]影室，拍攝人像及時裝[[廣告|廣告]]。1986年則開始[[婚紗|婚紗]]攝影業務<ref>{{web reference|URL=http://eastweek.my-magazine.me/main/42917|title=我不寂寞！廖安麗|work=《東周刊》第613期 Book B 【母心肝】|year=2015年|date=5月27日}}</ref>，曾為藝人拍攝多本[[寫真集|寫真集]]和推出葉青霖寫真實錄錄影帶（1990年）與葉青霖數碼寫真劇場[[DVD|DVD]]（2013年）。葉青霖於1986年發起成立[[香港專業攝影師公會|香港專業攝影師公會]]<ref>{{web reference|URL=http://photo.popart.hk/newweb/archives/135893|title=【香港攝影30年】葉青霖談HKIPP成立，攝影雜誌訪問專業攝影師公會創會副會長|work=攝影雜誌|year=2017年|date=2月10日}}</ref>（1987年初才完成註冊手續），現為創會及榮譽會員<ref>{{web reference|URL=https://www.master-insight.com/%E5%B8%B8%E9%9C%96%E6%B3%95%E5%B8%AB%EF%BC%9A%E6%94%9D%E5%BD%B1%E7%9A%84%E6%9C%80%E9%AB%98%E6%8A%80%E8%A1%93-%E6%98%AF%E3%80%8C%E5%BF%98%E8%A8%98%E3%80%8D%E9%82%A3%E4%BA%9B%E6%8A%80%E8%A1%93/|title=常霖法師：攝影的最高技術 是「忘記」那些技術 失物認領──存在的不存在（之一）|work=灼見名家|year=2016年|date=11月20日}}</ref>。同年娶香港演員[[廖安麗|廖安麗]]為妻。二人的長女葉懿德和次女葉懿恩<ref>{{web reference|URL=https://hk.entertainment.appledaily.com/entertainment/daily/article/20140817/18834621|title=話你知：花旦去哪兒？|work=香港蘋果日報|year=2014年|date=8月17日}}</ref>分別於1989年以及1990年出生。他之後在1992年舉家移民[[澳洲|澳洲]]至1999年回流香港。<ref>{{web reference|URL=http://paper.wenweipo.com/2007/06/04/FC0706040001.htm|title=移民紛回流 香港才是家|work=香港文匯報|year=2007年|date=6月4日}}</ref>葉青霖曾於2005年和2006年為[[電視廣播有限公司|TVB]]拍攝旅遊攝影節目《[[江山如畫|江山如畫]]》，到[[中國|中國]][[張家界|張家界]]、[[武夷山|武夷山]]、[[雁蕩山|雁蕩山]]、[[肇慶|肇慶]]等地教授風景及人像攝影技巧<ref name="攝影師 葉青霖 半生‧緣">[http://ubeat.com.cuhk.edu.hk/ubeat_past/080585/yip.html 香港中文大學《大學線》〈攝影師 葉青霖 半生‧緣〉，2008年]</ref>。此外，他曾於2003年[[SARS事件|SARS事件]]之後，在香港義務工作發展局成立一支攝影義工隊<ref>{{cite book |author1=釋常霖 |title=菩提樹下的野孩子 |date=2017 |publisher=台灣土狗 |page=23 |quote=因為自己在出家前是一個專業攝影師，曾經在香港義務工作發展局成立了一支攝影義工隊。|url=}}</ref>。

葉青霖在1990年至2008年加入[[聖約翰救傷隊|聖約翰救傷隊]]，並於1997年獲[[港督|港督]]頒授聖約翰五級勳銜<ref>{{web reference|URL=http://www.etnet.com.hk/mobile/tc/lifestyle/article.php?id=14921&catalias=whatson&secalias=leisure|title=常霖法師：「假如世界未終結」我們該怎樣面對？|work=經濟通|year=2012年|date=12月21日}}</ref>。

== 出家經過 ==
葉青霖於1994年父親過身後接觸[[佛教|佛教]]<ref>{{cite book |author1=釋常霖 |title=菩提樹下的野孩子 |date=2017 |publisher=台灣土狗 |page=39 |quote=因為父親的病，讓我接觸到佛法，找到之前所有問題的答案，是我覺得可以追尋下去，離苦得樂的方法。|url=}}</ref>，1997年曾於[[澳洲|澳洲]][[佛光山|佛光山]][[南天寺|南天寺]]短期出家。2008年5月19日決定放下一切名利，2009年正式前往台灣準備出家，並將與廖安麗合資的婚紗攝影公司出售<ref>{{web reference|URL=https://hk.lifestyle.appledaily.com/nextplus/magazine/article/20110120/2_14882104/%E6%8A%B9%E7%99%BD-%E5%BB%96%E5%AE%89%E9%BA%97|title=抹白 廖安麗|work=香港蘋果日報|year=2011年|date=1月20日}}</ref>。2010年5月16日為[[佛誕|佛誕]]。葉青霖在[[台北|台北]][[中和區|中和區]][[玉佛寺|玉佛寺]]剃度[[出家|出家]]，與妻子及家人舉行「辭親」儀式<ref>{{cite news| title =獨家專訪 丈夫葉青霖出家 廖安麗的內心掙扎| date =第2171期| url =http://www2.mingpaoweekly.com/contents/?id=6834| work =明報周刊| quote =| accessdate =2012-11-01| deadurl =yes| archiveurl =https://web.archive.org/web/20160306061724/http://mingpaoweekly.com/contents?id=6834| archivedate =2016-03-06}}{{deadlink}}</ref>。
