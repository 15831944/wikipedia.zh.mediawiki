'''釋志開'''（{{bd|1911年||1979年||}}），[[江苏省|江苏省]][[海安縣|海安縣]]人，曾出任[[南京市|南京市]][[栖霞寺|栖霞寺]]主持，為[[佛光山|佛光山]][[星雲法師|星雲法師]]的入門師父。<ref name=" web100" >{{cite web |url=http://fgsarts.webgo.com.tw/images/water/water2-4.htm|title=師恩浩瀚 |accessdate=|work=|publisher=星云大师弘法50周年纪念专辑|language=zh-hant}}</ref>

== 简介 ==
釋志開曾与[[大本|大本]]、[[觉民|觉民]]等法师创建了[[栖霞寺|栖霞寺]]律学院。<ref name=" web100" />中期收徒[[釋星雲|釋星雲]]。釋星雲在17岁时染上[[疟疾|疟疾]]，折腾半个多月，差不多奄奄一息。釋志開在那个物质匮乏的年代送了徒弟釋星雲半碗[[咸菜|咸菜]]，拯救了釋星雲的性命。釋星雲立誓：“师父，弟子有幸跟随您出家修行，我将来一定弘扬佛法，普度众生，絕不辜负师父的一片慈悲！”<ref name=" webxinlang1" >{{cite web |url=http://vip.book.sina.com.cn/book/chapter_149716_128165.html|title=半碗咸菜 |accessdate=2011年9月18日|work=|publisher=新浪网|language=zh-hans}}</ref>1937年，[[日本军队|日本军队]]攻佔[[南京城|南京城]]，{{fact|釋志開带领僧众救难民，并为[[中国军队|中国军队]]筹集粮食。}}

于[[中國大陸|中國大陸]][[文革|文革]]期间過世，安葬于[[江苏|江苏]][[海安|海安]]。<ref name=" web100" />
