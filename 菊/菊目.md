'''菊目'''（[[学名|学名]]：'''Asterales'''）是[[被子植物|被子植物]][[雙子葉植物|雙子葉植物]]下的一個[[目|目]]。有11個[[科|科]]，分佈於全世界，[[草本|草本]]，稀為[[喬木|喬木]]或[[灌木|灌木]]。[[菊科|菊科]]是本目的第一大科，約有25000多種，[[桔梗科|桔梗科]]是第二大科，約有2000多種，其餘的九個科合計少於500種。菊科和桔梗科這二個科分佈於全世界，主要分佈於北半球，少部份分佈於[[澳洲|澳洲]]及其附近地區或[[南美洲|南美洲]]。

== 演化與生物地理 ==
菊目可能是起源於[[白堊紀|白堊紀]]的岡瓦納古陸，約在現今的澳洲與[[亞洲|亞洲]]地區。雖然現存的種類大多數是草本植物，據調查研究顯示，本目的共同祖先可能是木本植物。菊目的化石證據很稀少，而且是屬於較近代的化石，所以要推測菊目出現的確切年代是很困難的。本目已知的化石，有漸新世時期菊科和草海桐科的花粉化石，及漸新世與中新世時期睡菜科和桔梗科的[[種子|種子]]化石。

== 經濟用途 ==
菊科的一些種類被栽培來做為[[食品|食物]]，例如：[[向日葵|向日葵]]或[[菊苣|菊苣]]。也有一些種類可以做[[調味料|調味料]]與藥草。菊科與桔梗科的许多品种可以做觀賞植物，例如：[[六倍利|六倍利]]、[[蓝眼菊|蓝目菊]]和[[玛格丽特|玛格丽特]]等等。

== 分类 ==
早期的分類系統（Cronquist system）在菊目下只有菊科一個科。[[APG系统|APG系统]]（截至2016年的[[APG_IV|APG IV]]）中菊类有以下11个科。：{{r|APG4}}{{r|APG3科属}}
* [[岛海桐科|岛海桐科]]（Alseuosmiaceae）
* [[雪叶木科|雪叶木科]]（Argophyllaceae）
* [[菊科|菊科]]（Asteraceae）
* [[萼角花科|萼角花科]]（Calyceraceae）
* [[桔梗科|桔梗科]]（Campanulaceae）
* [[草海桐科|草海桐科]]（Goodeniaceae）
* [[睡菜科|睡菜科]]（Menyanthaceae）
* [[五膜草科|五膜草科]]（Pentaphragmaceae）
* [[石冬青科|石冬青科]]（Phellinaceae）
* [[卢梭木科|卢梭木科]]（Rousseaceae）
