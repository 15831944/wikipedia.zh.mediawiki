'''穗高岳'''位於[[日本|日本]]，是[[日本百名山|日本百名山]]的其中之一，海拔3,190公尺，是[[飛驒山脈|飛驒山脈]]的最高峰，也是日本第三高峰。穗高岳附近有[[新穗高溫泉|新穗高溫泉]]。

== 概要 ==
奥穂高岳，涸澤岳，北穂高岳，前穂高岳，西穂高岳這些山峰合起來稱爲'''穂高連峰'''。除了前穂高岳以外主要都在[[長野縣|長野縣]][[松本市|松本市]]和[[岐阜縣|岐阜縣]][[高山市|高山市]]的交界上。主峰是奥穂高岳，是[[長野縣|長野縣]]（[[信濃国|信州]]）和[[岐阜縣|岐阜縣]]（[[飛騨国|飛騨]]）的最高峰。也被稱爲「北阿爾卑斯的盟主」。

北邊通過大山嶺，[[南岳|南岳]]，[[大食岳|大食岳]]的峻険道路可以到[[槍岳|槍岳]]。途中涸澤岳到南岳的山脊上飛騨一側是和谷川一的倉倉澤齊名的陡峭岩壁。南面從奥穂高岳到西穂高岳然後通過山脊可以到[[燒岳|燒岳]]。從奥穂高岳經過吊尾根還可以到前穂高岳，從這裡的圓形大斜坡下去就是[[上高地|上高地]]的[[河童橋|河童橋]]。從[[岐阜縣|岐阜縣]]一側的[[穂高岳山荘|穂高岳山荘]]可以下到白出澤，或者從西穂高岳乘坐[[纜車|纜車]]可以下到[[新穂高温泉|新穂高温泉]]。

穂高岳和[[剱岳|剱岳]]，[[谷川岳|谷川岳]]並稱爲[[日本三大岩場|日本三大岩場]]。特別是涸澤岳到南岳的飛騨一側。前穂高岳的東側，奥又白谷的上部也是陡峭的岩壁（也叫作前穂東壁）。[[井上靖|井上靖]]的[[小説|小説]]『[[冰壁|冰壁]]』中魚津和小坂就是這裡摔死的。
一般將[[涸澤|涸澤]]作爲攀登穂高岳的基地，這裡是奥穂高岳和前穂高岳之間的U字型山谷，但是即使是夏季仍然有積雪，所以要做好防寒準備。

== 穂高的山峰 ==
; 奥穂高岳（おくほたかだけ）
: 穂高連峰的主峰,標高3,190公尺。僅次於[[富士山|富士山]]與[[北岳|北岳]]，是日本第三高山峰。長野縣和岐阜縣的最高峰。[[穂高神社|穂高神社]]的主祭神「穂高見神」降臨的地方，建有嶺宮。
; 涸澤岳（からさわだけ）
: 白出峠（[[穂高岳山荘|穂高岳山荘]]）和北穂高岳之間的山峰。[[三角点|三角点]]標高為3,103公尺，最高點為3,110公尺，僅次於奥穂高岳。
; 北穂高岳（きたほたかだけ）
: 穂高連峰北端的山峰。西面有日本屈指可數的陡峭岩壁[[瀑谷|瀑谷]]。山頂付近有[[北穂高小屋|北穂高小屋]]。標高3,106公尺。
; 前穂高岳（まえほたかだけ）
: 奥穂高岳和吊尾根之間的山峰。標高3,090公尺。北方延伸到北尾根，西面奥又白方面是陡峭岩壁。北尾根突出的山峰叫做'''屏風頭'''（標高2,570公尺），東北面有高度差達1,000公尺的岩壁[[屏風岩|屏風岩]]。
; 明神岳（みょうじんだけ）
: 前穂高岳向南方延伸的山脊上的山峰。有數條凳山道在這裡交匯。標高2,931公尺。
; 西穂高岳（にしほたかだけ）
: 奥穂高岳向南西延伸到痩尾根的部分，前後都有其他的岩峰相連，所以比較難以區別。標高2,909公尺。
; [[間岳|間岳]]（あいのだけ）
: 奥穂高岳和西穂高岳山脊上的山峰。標高2,907公尺。
; 蒲田富士（がまだふじ）
: 涸澤岳的西峰。冬期一般從這裡攀登奥穂高岳。標高2,742公尺。

== 登山 ==
穂高連峰雖然是日本第三高峰，但是攀登的危險程度遠遠要高於攀登[[富士山|富士山]]。自古以來有數不盡的登山愛好者在這裡成了不歸人。因此沒有足夠的準備和一定的登山知識不要小看自然而輕裝登山。也正因爲如此穂高連峰吸引了日本囯内外衆多登山愛好者慕名而來。

穂高連峰在五月黃金周的残雪期也可以攀登，下面所列一般路綫是在無[[雪|雪]]期的常用路綫。有[[積雪|積雪]]的時候非常危險，是無法通行的。

=== 一般路綫 ===
; 奥穂高岳
: [[上高地|上高地]] ― [[明神池|明神池]] ― [[徳澤|徳澤]] ― [[横尾|横尾]] ― [[涸澤|涸澤]] ― 奥穂高岳
: 上高地 ― 明神池 ― 徳澤 ― 環行走廊（屏風耳） ― 涸澤 ― 奥穂高岳
; 北穂高岳
: 上高地 ― 明神池 ― 徳澤 ― 横尾 ― 涸澤 ― 北穂高岳
; 西穂独標
: 新穂高温泉 ― 新穂高纜車 ― [[西穂山荘|西穂山荘]] ― 丸山 ― 西穂独標
: 上高地 ― 中尾根 ― 西穂山荘 ― 丸山 ― 西穂独標

從徳澤經過環行走廊到涸澤的路綫，過了屏風耳北側是個斜面，這裡即使8月路上也有残雪，設置有鉄鎖和纜繩，是道路狹窄的危險地帶。

=== 強健者路綫 ===
; 前穂高岳，奥穂高岳
: 上高地 ― [[岳澤山小屋|岳澤山小屋]] ― [[紀美子平|紀美子平]] ― 前穂高岳 ― 奥穂高岳
: 新穂高温泉 ― [[穂高平小屋|穂高平小屋]] ― 白出澤 ― 奥穂高岳

岳澤山小屋到紀美子平這段（[[重太郎新道|重太郎新道]]）是日本屈指陡峭山路，而且岳澤山小屋因爲雪崩而廢棄，所以直到山頂都沒有可休息的場所，所以一般攀登前穂高岳和奥穂高岳不是走這條路綫，而是從上高地從北面繞經涸澤先到奥穂高岳然後到前穂高岳。

=== 專業者路綫 ===
; 槍穂高縦走
: [[槍岳|槍岳]] ― [[大食岳|大食岳]] ― [[中岳|中岳]] ― [[南岳|南岳]] ― 大山嶺 ― 北穂高岳 ― 涸澤岳 ― 奥穂高岳 ― 前峰 ― 間岳 ― 西穂高岳 ― 西穂独標

上述，各山峰有縦走路連接，其中有一般人也能攀登的，涸澤岳北面如概要所記述飛騨一側瀑谷是很陡峭的。想爬北穂高岳，奥穂高岳兩山的情況還是不要走山脊，先下到涸澤然後再爬另一座山比較安全容易一些。奥穂高岳到西穂独標之間的前衛峰，天狗頭，間岳是難關。

此外，從奥穂高岳經過前穂高岳，吊尾根，岳澤從重太郎新道下山雖然不算很難，但是這樣走的人不多，還是從涸澤繞行更加容易一些。
