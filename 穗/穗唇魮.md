'''長身穗唇魮'''（[[学名|学名]]：'''''Crossocheilus oblongus'''''，Crossocheilus siamensis是舊學名，穗唇魮是舊正名），又稱'''長橢圓鯉'''、'''暹羅角魚'''、'''暹羅穗唇魮'''，為[[輻鰭魚綱|輻鰭魚綱]][[鯉形目|鯉形目]][[鯉科|鯉科]]的其中一個[[種|種]]。

== 分布 ==
本魚分布於[[泰國|泰國]]、[[緬甸|緬甸]]、[[柬埔寨|柬埔寨]]、[[寮國|寮國]]、[[馬來西亞|馬來西亞]]、[[印尼|印尼]]、[[新加坡|新加坡]]的溪流中。

== 特徵 ==
本魚體呈魚雷形，被明顯的色彩帶所劃分。頂部的色彩為深綠色，中間隔一條淺黃色條紋，緊挨其下面有一條黑色條紋橫貫全身，腹部為銀黃色。所有鰭都有黃色斑紋，背鰭基部呈黑色，口下位。體長可達16公分。

== 生態 ==
本魚棲息在淡水的溪流中，性情溫和，屬草食性，以嘴用來刮食岩石上的[[水草|水草]]或[[藻類|藻類]]，雄魚在繁殖期會長出頭瘤。

== 經濟利用 ==
為觀賞性魚類。
