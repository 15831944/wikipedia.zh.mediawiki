），[[化学符号|化学符号]]：'''W'''（{{lang-de|'''W'''olfram}}），是一種[[化学元素|化学元素]]，[[原子序数|原子序数]]是74，是非常硬、钢灰色至白色的[[过渡金属|过渡金属]]。含有钨的矿物有[[黑钨矿|黑钨矿]]和[[白钨矿|白钨矿]]等。钨的物理特征非常强，尤其是[[熔点|熔点]]非常高，是所有非[[合金|合金]]金属中最高的。纯钨主要用在电器和电子设备，它的许多[[化合物|化合物]]和合金也有很多其它用途（最常见的有鎢絲[[灯泡|灯泡]]、[[X射线管|X射线管]]以及[[高温合金|高温合金]]）。

鎢的最穩定的五種同位素都有輕微的[[放射性|放射性]]。

==字源==
瑞典化學家最早由[[白鎢礦|白鎢礦]]中分離出鎢酸，因此根據[[白鎢礦|白鎢礦]]這種礦石的瑞典古名，將這種元素被稱為{{lang-sv|tungsten}}（這個字可被分解為{{lang|sv|tung sten}}，字面意義為重石）。在[[英文|英文]]、[[法文|法文]]等語言中，都使用這個名稱。但因為{{lang|sv|tungsten}}在瑞典文中也是白鎢礦的名稱，為了避免混淆，瑞典採用{{lang|sv|volfram}}作為元素的名稱。

在歐洲其他國家，主要以[[德文|德文]]及各[[斯拉夫语族|斯拉夫語]]為代表，則使用{{lang-de|wolfram}}或{{lang|de|volfram}}，在[[北歐五國|北歐五國]]也使用這個名稱。這個名稱來自[[黑钨矿|黑钨矿]]（Wolframite）這個礦石的名字<ref name="sweetums">{{cite web

== 主要特征 ==
纯钨是钢灰色至锡白色的坚硬金属，非常纯的钨可以[[拉锯|拉锯]]锯开（纯钨很脆，不易加工）。钨的加工方法有[[锻造|锻造]]、拉伸和冲击。在所有金属中，钨的熔点最高（3415[[°C|℃]]）[[蒸汽压|蒸汽压]]最低，在[[抗张强度|抗张强度]]最高（1650℃时）。钨的防腐性能非常好，大多数[[无机酸|无机酸]]对其的侵蚀都很小。在空气裡它的表面會形成一层保护性氧化物，但是在高温下会完全氧化。在[[钢|钢]]加入少量钨，可以大大增高钢的硬度。

== 应用 ==
钨的应用非常广泛，最常见的是[[碳化钨|碳化钨]]（W<sub></sub>C）硬质合金。这样的硬质合金用在金属加工、采矿、采油和建筑工业中作为耐用金属。此外在电灯泡和[[真空管|真空管]]中钨丝的应用也很广。钨还常用作[[电极|电极]]。钨可以拉成很细的丝，而且熔点非常高。它的其它应用包括：
* 由于钨的熔点非常高，所以常用於航空和高温環境，例如电子、加热和焊接(E.G. [[钨极气体保护电弧焊|钨极气体保护电弧焊]])。
* 钨非常坚硬，非常紧密，因此制作[[重金属|重金属]]合金非常理想，这样的合金用在装甲、[[散热片|散热片]]和高密度的应用上例如压重物、平衡重物、船和飞机的压重物等。
* 由于钨非常紧密，[[飞镖|飞镖]]往往含80%至97%的钨。
* [[高速钢|高速钢]]含钨，有时含18%的钨。
* 制造[[涡轮机|涡轮机]]片、耐用部分和保护层的高温合金含钨（[[哈氏合金|哈氏合金]]、[[钨铬钴合金|钨铬钴合金]]等）。
* 在[[子弹|子弹]]中使用钨来取代[[铅|铅]]。
* 钨的化合物被用作[[催化剂|催化剂]]、无机颜色。二硫化钨是高温[[润滑剂|润滑剂]]，它在500 °C依然稳定。
* 由于钨的漲性和[[硅酸硼玻璃|硅酸硼玻璃]]类似，所以人們用它進行玻璃/金属密封
* 钨与[[镍|镍]]、[[铁|铁]]和[[钴|钴]]的合金被用来制作重合金，这样的重合金用在[[动能弹|动能弹]]中取代[[贫铀|贫铀]]。
* 在[[集成电路|集成电路]]中钨是前路之间的连接物。在二氧化硅绝缘体中侵蚀接触孔，注入钨，磨平来连接三极管。典型的接触孔可以小到65纳米。
* 碳化钨是最硬的物质之一，被用在机器工具和磨料中。碳化钨是磨具和转具中最常见的材料，往往也是最好的材料。
* 在放射医学中钨是屏蔽物质。运输[[氟脱氧葡萄糖|氟脱氧葡萄糖]]一般用钨容器，因為氟脱氧葡萄糖中的高能量[[氟-18|氟-18]]令铅容器无法使用。

其它：氧化钨被用在[[陶瓷|陶瓷]]釉中，[[钙|钙]]或[[镁|镁]]钨常用在荧光粉中。在[[核物理|核物理]]和[[核医学|核医学]]中钨晶体被用作[[闪烁探测器|闪烁探测器]]。钨被用作[[X射线|X射线]]目标和在电子炉中作为加热器。含钨的盐被用在化学和皮革工业中。青铜色的氧化钨被用在绘画中。由于它的[[低敏感性|低敏感性]]碳化钨被用作首饰，此外由于它非常硬它不会像其它擦光的金属被划痕。有些乐器的铉使用钨丝。

== 历史 ==
1781年，瑞典化學家[[卡尔·威廉·舍勒|卡尔·威廉·舍勒]]發現，使用[[白钨矿|白钨矿]]，可以制作出一种新的[[酸|酸]]，即[[钨酸|钨酸]]。當時[[卡尔·威廉·舍勒|卡尔·威廉·舍勒]]與其友人[[托尔贝恩·贝里曼|托尔贝恩·贝里曼]]皆相信在鎢酸中一定可以進一步分解出一種新的化學元素。1783年[[胡塞·德卢亚尔|胡塞·德卢亚尔]]和[[浮士图·德卢亚尔|浮士图·德卢亚尔]]兄弟发现从黑钨矿可以获得同样的酸。同年他们使用[[碳|碳]]还原钨酸获得了钨，因此他们被公认为钨的发现者<ref>{{cite web |url=http://www.itia.info/FileLib/ITIA_Newsletter_June05.pdf |title=存档副本 |accessdate=2008-01-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20080307024359/http://www.itia.info/FileLib/ITIA_Newsletter_June05.pdf |archivedate=2008-03-07 }} </ref><ref>{{cite web |url=http://www.itia.info/FileLib/ITIA_Newsletter_December05.pdf |title=存档副本 |accessdate=2008-01-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20080307024340/http://www.itia.info/FileLib/ITIA_Newsletter_December05.pdf |archivedate=2008-03-07 }} </ref>。

== 生理作用 ==
[[还原酶|还原酶]]使用钨[[蝶呤|蝶呤]]。

虽然有人怀疑钨会导致[[白血病|白血病]]，但是至今为止缺乏有说服力的证明。

== 来源 ==
[[黑钨矿|黑钨矿]]、[[白钨矿|白钨矿]]、[[钨铁矿|钨铁矿]]等矿物含钨。重要的钨矿位于[[玻利维亚|玻利维亚]]、[[美国|美国]][[加利福尼亚州|加利福尼亚州]]和[[科羅拉多州|科羅拉多州]]、[[中国|中国]]、[[葡萄牙|葡萄牙]]、[[俄罗斯|俄罗斯]]以及[[韩国|韩国]]。中国出产全世界钨的75%。通过使用碳还原钨的氧化物获得纯的金属。

全世界钨的贮藏总量估计为700万吨，其中约30%是黑钨矿，70%是白钨矿。但是目前大多数这些矿藏无法经济性地开采。按照目前的消耗量这些矿藏只够使用约140年。另一个获得钨的方法是回收。回收的钨比钨矿含量高，事实上非常有利润。

== 化合物 ==
钨最常见的[[氧化态|氧化态]]是+6价，但它也有-1至+6之间的氧化状态<ref>{{cite book | title=The Elements | first=John | last=Emsley | year=2000 | edition=3rd edition }}</ref>。最常见的氧化物是黄色的[[三氧化钨|三氧化钨]]，WO<sub>3</sub>，它可以在碱性的水中溶化形成WO<sub>4</sub><sup>2−</sup>。

== 同位素 ==
{{Main|钨的同位素}}

自然界里出现的有五种钨的同位素，其中<sup>182</sup>W、<sup>183</sup>W、<sup>184</sup>W、<sup>186</sup>W是[[稳定同位素|稳定同位素]]，<sup>180</sup>W的[[半衰期|半衰期]]非常长，可以被看作是[[稳定同位素|稳定同位素]]。所有这些同位素理论上均可以通过[[α衰变|α衰变]]蜕化为[[铪|铪]]，但是至今为止只有<sup>180</sup>W能够被观测到自然衰变，其半衰期为1.8×10<sup>18</sup>年，其它同位素没有被观测到自然衰变，估计的半衰期下界：<sup>182</sup>W, T<sub>1/2</sub> > 1.7×10<sup>20</sup> 年，<sup>183</sup>W, T<sub>1/2</sub> > 8×10<sup>19</sup> 年， <sup>184</sup>W, T<sub>1/2</sub> > 1.8×10<sup>20</sup> 年， <sup>186</sup>W, T<sub>1/2</sub> > 4.1×10<sup>18</sup> 年<ref>National Nuclear Data Center table of nuclides, http://www.nndc.bnl.gov/chart/</ref>。平均每年在一克<sup>180</sup>W中发生两次α衰变。

钨目前有27种人造[[放射性同位素|放射性同位素]]，其中最稳定的是<sup>181</sup>W，其半衰期为121.2天，<sup>185</sup>W的半衰期为75.1天，<sup>188</sup>W的半衰期为69.4天，<sup>178</sup>W的半衰期为21.6天。其它放射性同位素的半衰期均在24小时以下，其中大多数少于8分钟。
