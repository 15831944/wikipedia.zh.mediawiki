'''鼻毛真拳'''（{{lang-ja|ボボボーボ・ボーボボ}}）是由[[日本|日本]][[漫畫家|漫畫家]][[澤井啟夫|澤井啟夫]]所畫的[[搞笑漫畫|搞笑漫畫]]，2000年時以短篇形式刊載在《[[赤丸Jump|赤丸Jump]]》，之後連載於《[[週刊少年Jump|週刊少年Jump]]》自2001年12號連載至2007年31號，後期改名為「真說・鼻毛真拳」，單行本全28卷，以《週刊少年Jump》的搞笑漫畫而言屬於長期連載。本作另有改編成動畫、遊戲、卡片遊戲等作品。

==概要==
西元300X年，世界已經被'''圓頂帝國'''統治。圓頂帝國的帝王組織了'''拔毛隊'''，對世界各地的人進行毛髮狩獵。為了捍衛人類毛髮的自由與和平，使用'''鼻毛真拳'''的高手'''阿母'''與他們展開了戰鬥。

本作品的特徵在於毫無邏輯的故事與荒謬無理的戰鬥，戰鬥的招式異想天開、完全不能用一般常識理解，而且同伴加入主角方的理由相當隨性，完全沒有沉重的劇情，而是全以搞笑為優先。舞台設定為近未來，但並不強調高科技，反而是充斥著各種稀奇古怪的生物。

本作品也與許多其它的作品進行合作，包括《[[遊戲王|遊戲王]]》、《[[死亡筆記本|死亡筆記本]]》、《[[烏龍派出所|烏龍派出所]]》等；並[[戲仿|戲仿]]同雜誌的其他作品如《[[七龍珠|七龍珠]]》、《[[北斗神拳|北斗神拳]]》、《[[魁！！男塾|魁！！男塾]]》、《[[JoJo的奇妙冒險|JoJo的奇妙冒險]]》等。

==登場人物==
;阿母 ({{lang-ja|ボボボーボ・ボーボボ}} )
:{{配音2|JP|[[子安武人|子安武人]]／[[木內レイコ|木內レイコ]]（小時候）}}
:頂著金色[[爆炸頭|爆炸頭]]的壯碩男性，27歲，鼻毛真拳的高手，第七代鼻毛真拳傳承者。
:擅長使用鼻毛真拳，也會使用腋毛真拳、面紙真拳等，並且能夠跟其他主要人物進行融合。此外還能使用其它漫畫角色的招式，例如[[龜派氣功|龜派氣功]]、[[日向寧次|八卦六十四掌]]等。

;小美 ({{lang-ja|ビュティ}} )
:{{配音2|JP|[[野中藍|野中藍]]}}
:14歲，粉紅短髮的少女，女主角。因為某種理由和主角同行，是本作品重要的[[吐槽役|吐槽役]]。
:是主要角色中唯一一位非戰鬥員，非常討厭黃色笑話，作中也較少描繪其吐血的畫面。
:原本的設計為男性，但責任編輯希望改為女性，雖然作者並不擅長畫女性角色，但作者認為「如果不修改的話可能無法連載」於是便「答應」了。

;首領巴其 ({{lang-ja|首領パッチ}} ) 
:{{配音2|JP|[[小野坂昌也|小野坂昌也]]}}
:橘色太陽外型的神祕生物，自稱主角或女主角。極度自我中心、擅長忌妒。

;白爛王 ({{lang-ja|ヘッポコ丸}} )
:{{配音2|JP|[[進藤尚美|進藤尚美]]}}
:16歲，白色刺蝟頭少年，使用臭屁真拳。
:出身地為原毛之王國中的〈放屁城市〉，在被四天王之一的'''軍艦'''入侵並摧毀時倖存下來。
:似乎對小美有好感。

;濱綺淋 ({{lang-ja|ソフトン}} )
:{{配音2|JP|[[綠川光|綠川光]]}}
:[[大便|大便]]頭的神秘男性，原作是咖啡色、動畫改成粉紅色。25歲。霜淇淋店店長，真實身分是小美的哥哥。
:使用[[巴比倫|巴比倫]]真拳，在與J一戰之後獲得了J的黑太陽之力，進化為黑太陽巴比倫真拳。

;陽頂天 ({{lang-ja|ところ天の助}} )
:{{配音2|JP|[[園部啟一|園部啟一]]}}
:一塊藍色[[洋菜凍|洋菜凍]]，但其中5%是果凍，常帶著一條「怪」字手帕。使用滑溜真拳。
:原本是僅登場一次的角色，但因為人氣非常高而成為主角群之一。

;破天荒 ({{lang-ja|破天荒}} )
:{{配音2|JP|[[岸尾大輔|岸尾大輔]]}}
:與阿母同樣是毛之王國的倖存者，24歲。
:使用鑰匙真拳，因為作者表示他是故事中的關鍵人物。

;田樂 ({{lang-ja|田楽マン}} )
:{{配音2|JP|[[金田朋子|金田朋子]]}}
:白色人型的神祕生物。

;魚雷女郎 ({{lang-ja|魚雷ガール}} )
:{{配音2|JP|[[西川宏美|西川宏美]]}}
:魚雷外型生物。

;鼻毛王（{{lang-ja|KING鼻毛}}）
:{{配音2|JP|[[竹本英史|竹本英史]]}}
:封印於阿母的鼻子裡面的鼻毛之神，擅長使用鞭毛神拳。
:外表顯眼且性格經常表現出豪邁的樣子，說話也很輕浮，習慣以「N（ん）――――」為口頭禪。在大戰結束的300年後從長眠中甦醒，幫助遇到困難的小美，可是稍後在阿母說了「這是什麼鼻毛？真礙事！」之後直接被拔除了。

===反派角色===
;滑溜滑溜四世（{{lang-ja|ツル·ツルリーナ4世}}）
:{{配音2|JP|[[松野太紀|松野太紀]]}}
:圓頂帝國的現任國王，最初登場時為剛新婚後不久（妻子是真子）。戰鬥實力不明（不過有曾經說出「自己已經預先儲存了大量的宇宙能量」的宣言），為本作最大反派角色。

;真子（{{lang-ja|マコちゃん}}）
:滑溜滑溜四世的愛妻。

;奧客多帕斯卡耳（{{lang-ja|オクトパスカル}}）
:{{配音2|JP|岸尾大輔}}
: 滑溜滑溜四世的侍衛。外型為[[德國香腸|德國香腸]]。
: 曾經被三世用封箱膠帶給封印了一陣子。

;軍艦（{{lang-ja|軍艦}}）
:{{配音2|JP|[[大友龍三郎|大友龍三郎]]／[[吉田小南美|吉田小南美]]（小時候）}}
:以巨大的飛機頭為特色的男人，圓頂帝國四天王之一。後期用了「邪惡力量」把鼻毛真拳加以改造後變成「我流鼻毛真拳」的持有者。
:小時候曾經與阿母一起學習鼻毛真拳，但因為後來在沒有任何有毛人類存在的王國中因為找不到繼承者，所以非常痛恨該王國。
:過去曾經摧毀過白爛王的家鄉〈放屁城市〉，偶而會試圖追捕身為該城市被摧毀時倖存下來的白爛王。

;普魯普（{{lang-ja|プルプー}}）
:{{配音2|JP|高戶靖廣}}
:圓頂帝國四天王之一，登場時經常搭乘不同的交通工具。
:個性凶暴殘忍，擁有比軍艦還要高的戰鬥能力，口頭禪是-{「}-死刑（動畫版改成"消失吧"）-{」}-，在模仿[[麥克·傑克森|麥克·傑克森]]時動作很微妙。在行穴村意外跟阿母對峙、然後在跟'''彈珠汽水'''、'''禁止吸菸'''兩名手下聯手突擊時被首領八奇隱藏的實力給瞬間秒殺。
:失敗之後被OVER給懲罰，拔光身上所有的毛。
:漫畫-{「}-真說篇-{」}-完結篇於圓頂禿頭同窗會登場，同時是四天王中唯一沒有在武鬥大會上登場的。

;哈雷克拉尼（{{lang-ja|ハレクラニ}}）
:{{配音2|JP|[[三木眞一郎|三木眞一郎]]}}
:圓頂帝國四天王之一，自稱是四天王中最強的男人。
:特徵綠髮藍眼，使用可操縱金錢的豪華真拳，有者穩重且冷靜的性格。哈雷路亞遊樂園的負責人，為只要遊客身上沒有金錢就會趕出去的金錢死者。
:喜歡的國家是[[夏威夷|夏威夷]]和高級牛排，持有的所有資產無限，在當上四天王之前正職是銀行職員。
:小時候因為貧窮還有當時本人對於金錢異常的執著下創造出自創的豪華真拳，在放出阿母使其戰鬥的過程中因為自己理解了世上有比金錢更重要的東西然後於阿母的鼻毛真拳超絕奥義「ワンダフル鼻毛7DAYS」招式之下敗北。
:漫畫-{「}-真說篇-{」}-完結篇於圓頂禿頭同窗會登場，以前有認識OVER還有魚雷女，在【裡·圓頂帝國篇】遇到魚雷女的時候曾推薦來個下午茶時間。
:名字來自於真實世界中在夏威夷開設的同名旅館。

==特殊用語==
;真拳 ({{lang-ja|真拳}}) 
:真拳指的是象徵該人物的某種特殊能力，基本上、一個人物只會有一種真拳。

;毛髮狩獵 ({{lang-ja|毛狩り}}) 
:為了炫耀自己的權力，由圓頂帝國國王滑溜滑溜四世所下令、拔除他人新鮮毛髮的行動。

;澀谷區大型百貨公司YOKOSEYO（{{lang-ja|渋谷区大型デパートヨコセヨ}}）
:本作中登場的大型百貨公司，一般都稱為「YOKOSEYO百貨」。於本作中做為〈新·毛之王國〉坊間流傳的「聖鼻毛領域」真實存在的地方，同時動畫版也有專屬背景音樂。

;真拳狩獵（{{lang-ja|真拳狩り}}）
:主要是進行在特定的場所給強行擄走真拳的持有者行動，機加與四世的父親滑溜滑溜三世等都有親自執行過該行動。

;毛球（{{lang-ja|毛玉}}）
:只有過去原毛之王國被摧毀後活下來的倖存者所持有，存在於體內的一種毛狀球體。一旦球體被破壞持有者就會立刻死亡，但如果只是從體內被取出來該對象則不會立刻死亡，除了閃光俠（ビービビ）的情形例外（在當毛球被取出來時就已經死亡）。
:不過漫畫第2部真說篇第72話時原本被取出來的毛球回歸體內，使閃光俠復活。

==電視動畫==
自2003年11月8日起至2005年10月29日止於[[朝日電視台|朝日電視台]]播放，全76話。台灣自2006年暑假期間由[[台視|台視]]引進並於晚間時段播放，{{fact|但在播放不久後便遭到家長投訴並很快地停播}}，後來於2013年5月中自[[衛視電影台|衛視電影台]]晚間時段再次重播。
