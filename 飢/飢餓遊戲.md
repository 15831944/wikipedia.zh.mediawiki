《'''飢餓遊戲'''》（{{lang-en|''The Hunger Games''}}）是一本[[美國|美國]]青少年冒險科幻小說，為一本[[反烏托邦|反烏托邦]]的小說。為小說家[[蘇珊·柯林斯|蘇珊·柯林斯]]創作。《[[飢餓遊戲三部曲|飢餓遊戲三部曲]]》中的首部曲。內容是講述[[未來|未來]]的[[北美洲|北美洲]]虛構國家「施惠國」，主角凱妮絲·艾佛丁因為頂替妹妹而參加了一個由都城政府舉辦的[[真人秀|真人秀]]，名為「飢餓遊戲」，遊戲參加者有24人，他們要用任何方法殺死對方，剩下的一位參賽者，便是遊戲的優勝者。

== 故事 ==
第七十四屆飢餓遊戲即將開始。飢餓遊戲是至死方休的無情決鬥。

在這塊過去曾被稱為北美洲的廢墟大陸上，有個新建立的國家「施惠國」，該國內有著羅列四周的十二個[[行政區|行政區]]，以及一座被行政區圍繞的富饒都城。至高無上且專制殘酷的都城，每年逼迫十二行政區交出12歲至18歲的少男少女各一名，將之投放在一年一度的「飢餓遊戲」當中，並以[[電視|電視]]實況轉播，藉由此恐怖的手段來維持其威權統治以及國家秩序。

十六歲的凱妮絲與寡母以及妹妹櫻草花同住，當她自願代替在「抽籤日」被抽中的妹妹參加此獵殺遊戲時，她早已認命赴死。但是曾經在死亡邊緣掙扎過的她，卻也養成了強悍的求生野性。凱妮絲在無意間成為了遊戲的有利競爭者。在通往生存的苦戰之路上，她勢必將面臨重重艱難的抉擇，權衡生命、人性以及愛之間，何者才真正寶貴。

== 本集主要角色 ==
* [[凱妮絲·艾佛丁|凱妮絲·艾佛丁]]（Katniss Everdeen）- 故事的女主角，故事由她作視點以第一人稱寫出。參加遊戲前和蓋爾是青梅竹馬。有一頭長黑直髮，橄欖膚色，一雙灰色眼睛。第七十四屆和七十五屆遊戲第十二區的女貢品。在七十四屆抽籤日自願取代被抽中的最心愛的妹妹櫻草花·艾佛丁成為貢品。在被選中為貢品後，因設計師給她的特殊打扮，造成轟動，被大家稱為「燃燒的女孩」。跟比德在遊戲中演出悲劇戀人的戲碼。對比德和蓋爾兩人都有特殊感情，但不知道自己喜歡的是誰。和蓋爾接吻過兩次，卻每每在攝影機前和比德接吻，即使知道這會傷害蓋爾，但只是為了應付史諾和媒體。比德曾用兩條麵包救她一命，自此她就默默記下比德的事。在書裡的第三集，帶領第十三區的反抗軍和蓋爾、比德、芬尼克對反抗都城。成功後回到十二區和比德先後有了一女一男兩個小孩。

* 比德·梅爾拉克（Peeta Mellark）- 故事的男主角。有金色頭髮與一雙藍眼睛，家中開麵包店，屬於商人階級。會非常高明的偽裝術。第七十四屆和第七十五屆遊戲中第十二區的男貢品。在第七十四屆的遊戲中跟專業貢品結盟，但從中不停誤導他們有關凱妮絲的事，並在專業貢品手上救了她一命，但被專業貢品卡圖砍傷大腿，凱妮絲找到瀕臨死亡的比德後趁比德熟睡時冒險去豐饒角，帶回治療比德的藥，救了他一命。

* 蓋爾·霍桑（Gale Hawthorne）- 十八歲，故事的第二男主角。凱妮絲的打獵夥伴兼青梅竹馬。有一頭黑直頭髮，橄欖膚色，一雙灰色眼睛，長相十分帥氣。個性十分頑強，非常痛恨都城的人，常在跟凱妮絲一起打獵時咒罵他們，深愛著凱妮絲。

* 櫻草花·艾佛丁（Primrose Everdeen）- 十二歲。凱妮絲的妹妹，亦是她最心愛的人。跟凱妮絲不同，她有一頭金色的頭髮和藍色眼睛，遺傳自出身於商人階級的媽媽。第七十四屆遊戲中，僅有一個籤條的她不幸被抽中，但凱妮絲自願代替她參加遊戲。

* 黑密契·阿勃納西（Haymitch Abernathy）- 唯一還活著，第十二區僅有曾經贏得遊戲的兩位冠軍之一，擔任凱妮絲和比德的遊戲導師。

* 秦納（Cinna）- 第七十四屆遊戲中，都城分派給第十二區的造型設計師。爲凱妮絲朋友。

* [[史諾總統|史諾總統]]（President Coriolanus Snow）- 都城總統，因第七十四屆遊戲的結果，大傷元神，非常痛恨凱妮絲。

* 艾菲·純克特（Effie Trinket）- 第七十四屆遊戲中，擔任十二區抽籤人。

== 改編 ==
{{see also|飢餓遊戲 (電影)}}
飢餓遊戲於2012年3月24日被改編成[[電影|電影]]，由[[珍妮佛·勞倫斯|珍妮佛·勞倫斯]]、[[喬許·哈卻森|喬許·哈卻森]]和[[連恩·漢斯沃|連恩·漢斯沃]]飾演凱妮絲、比德與蓋爾。本片獲得空前成功，成為[[美國|美國]]史上的非續集電影開畫冠軍。
