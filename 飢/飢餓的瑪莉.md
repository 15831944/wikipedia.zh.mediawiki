《'''飢餓的瑪莉'''》 （{{lang|ja|腹ペコのマリー}}） 是由[[日本漫畫家|日本漫畫家]][[田村隆平|田村隆平]]於《[[週刊少年Jump|週刊少年Jump]]》2017年13號46號連載的漫畫。<ref>{{cite web|title=「べるぜバブ」の田村隆平がジャンプ帰還、新連載はバトルラブコメ|url=http://natalie.mu/comic/news/222417|website=ナタリー|language=ja|date=2017年2月27日}}</ref><ref>{{cite web|title=ジャンプ13号より『べるぜバブ』田村隆平の新連載スタート！　「この1話だけでわかる… 絶対面白い」とファン大興奮|url=https://ddnavi.com/news/356605/|website=ダ・ヴィンチニュース|language=ja|date=2017年3月3日}}</ref>台灣方面《[[寶島少年|寶島少年]]》新連載第三彈於2017年第19期開始連載。作品已出版全4冊，已完結。

== 故事大綱 ==
出生在一日本少有的傳統[[道教|道教]]家庭的'''美女木大河'''，天生擁有怪異的思想。住居隔壁是一間名叫鷺宮教會的教堂，教堂主人的女兒'''鷺宮安娜'''，是個有點腹黑的美少女學生。在教堂中有個[[幽靈|幽靈]]，只有安娜可以看見，那個女幽靈名字叫做'''瑪莉·泰瑞絲'''。為了讓瑪莉復活，安娜只好把大河拖下水，讓大河成為瑪莉的憑依.........

== 主要人物 ==
; {{nihongo|美女木大河|美女木 タイガ／びじょぎ タイガ}}
: 本作男主角。天生有著奇怪的思想，也有異於常人的靈感能力，不過不能看見鬼魂。但是打架能力高超，功課是一般水準。家庭是日本少見的[[道教|道教]]家庭，住家大門口是有如中國傳統[[廟宇|廟宇]]的外型，主供奉神為[[呂洞賓|呂洞賓]]。
: 暗戀鷺宮安娜，其實與安娜是互相喜歡。不過總是被安娜等人給拖下水。
: 是瑪莉的憑依，所以常常會穿上女裝。被瑪莉[[憑依|憑依]]時，自身的靈魂會被封印在狗布偶內。其實就是當初將瑪莉從革命者手中救下來帶至日本的美女木家的祖先轉世，只是本人沒有這份記憶。
:* 家庭組成：祖母、父親(未出現，僅口頭得知)、叔叔(豬肉味繒)、弟弟。

; {{nihongo|鷺宮安娜|鷺宮 アンナ／さぎのみや アンナ}}
: 本作女主角之一，是個身材纖細、有著略為平胸的貧乳美少女，有點腹黑。能看見幽靈，所以能看見瑪莉泰瑞絲的幽魂。為了幫助瑪莉復活，因此把大河拖下水。
: 喜歡大河，但是因為害羞所以不敢說出來，被瑪莉提點之後才正視自己的心意。
: 沒有打架能力，但是功課算是前段班的名列前矛。
:* 家庭組成：父親(鷺宮教會的住持，有著惡魔髮型)、母親(已過世)

; {{nihongo|瑪莉·泰瑞絲|マリー・テレーズ・シャルロット}}
: 本作女主角之一，是個鬼魂，最後靠著精靈羅潔成功在現世復活，不過身材縮水成小學生。有著貴族血統([[法國|法國]][[波旁王朝|波旁王朝]]之末代公主)，在[[法國大革命|法國大革命]]前夕被美女木家的祖先帶往日本，不過因此病弱而死，靈魂經常徘徊在現鷺宮教堂內。
: 是個大胃王，被安娜以兩支燒雞腿加上大河為召喚成功地在現世出現。由於是十八世紀的法國人，對於現代世界略為不熟悉。
:* 家庭組成：父親(法王[[路易十六|路易十六]])、母親([[瑪麗·安東妮|瑪麗·安東妮]])(備註：以上兩人在[[法國大革命|法國大革命]]時被[[雅各賓派|雅各賓派]]以[[斷頭台|斷頭台]]處死)、弟弟([[路易十七|路易十七]])(再註：以上除其母瑪麗·安東妮之外，剩下的家人皆未出現，皆是無名氏。)
