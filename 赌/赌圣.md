{{engname|《賭聖》|''All for the Winner''}}，1990年[[香港電影|香港電影]]，是[[周星馳|周星馳]]在電視劇《[[蓋世豪俠|蓋世豪俠]]》成名之後第一部電影代表作，亦是香港影史上首部票房突破四千萬的電影，從此[[周星馳|周星馳]]成為喜劇巨星、票房靈丹。

== 背景 ==
香港赌片唯一大陸人主角的低成本電影，沒想到只上映28天，票房高達41,326,156[[港元|港元]]，大破香港影史紀錄，從此「星爺」之號不脛而走。

本片的唯一投資人[[吳思遠|吳思遠]]在看過本片樣片之後，對周星馳很認真地說了一句話：「《賭聖》上映之後，你就不再是當紅小生，而是香港最紅的人。」

== 劇情 ==
左颂星（周星馳飾）乃身懷[[特異功能|特異功能]]的大陸仔，又有個稱號叫做「廣東拳王」（據周星馳母親所說：周星馳小時候經常練習打沙袋，夢想當拳王。而星媽正是來自[[廣東|廣東]]，所以[[周星馳|周星馳]]自稱是「廣東拳王」），申請來港與三叔（[[吳孟達|吳孟達]]飾）團聚。三叔後來發現他的特異功能，可以透視也可以搓[[彩票|馬票]]……等。讓財迷心竅的三叔認為如用在賭博方面不失為一條發財之路。三叔教導星要以賭神為[[偶像|偶像]]，將賭發揚光大成為新一代的賭聖。

兩叔侄開始走[[江湖|江湖]]，引起[[台灣|台灣]]賭王陳松與[[香港|香港]]賭王洪光（[[秦沛|秦沛]]飾）的注意，雙方欲爭奪星為其手下代表出席世界賭王大賽。在陳松的積極邀約之下，再加上此時出現一個奇女子「綺夢」（[[張敏|張敏]]飾）（亦為陳松埋伏在洪光之處的臥底）深深地吸引著阿星，令阿星「魂縈夢牽」，於是星與叔遂代表陳松參加[[世界|世界]]賭王大賽。

洪光得知星與叔投靠陳松，心知不妙於是[[綁架|綁架]]了綺夢。此舉令阿星魂不守舍，為了不影響阿星接下來在大賽中的表現，於是三叔想了一妙計找了大笑姑婆（[[吳君如|吳君如]]飾）假扮綺夢，結果誤打誤撞地獲得了阿星的注意，於是得以苟延殘喘地在世界賭王大賽挺進最後進入了決賽！但靠著腋下的一顆痣始終無法矇混太久，最後阿星不得不接受綺夢失蹤的事實，悵然若失地參加了最後的決賽。

為了幫助阿萍的小孩籌措[[開刀|開刀]]醫藥費，阿星的功力尚未恢復過來，面對強大非贏不可的壓力，一無所有的阿星一度想要逃避。最後阿星只好想出奇招：逼洪光一次定生死，純粹比[[運氣|運氣]]。在比賽完結前的倒數時刻，阿星又看見了他朝思暮想的綺夢，於是他又可以開始「搓牌」了……

== 軼事 ==
*片中[[世界|世界]]賭王大賽中，阿星所代表的[[中華民國國旗|中華民國國旗]]內的白日應該要有十二道光芒尖鋒，在片中的國旗只有九道而已。

*周星馳曾透露本片是其第一部收取百萬片酬的電影。
