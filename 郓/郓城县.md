'''郓城县'''位于中国[[山东省|山东省]]西南部，是[[菏泽市|菏泽市]]下辖的一个县。

== 地理 ==
郓城东邻[[梁山县|梁山县]]、[[嘉祥县|嘉祥县]]，西接[[鄄城县|鄄城县]]，南连[[巨野县|巨野县]]、[[牡丹区|牡丹区]]，北隔黄河与[[河南省|河南省]][[台前县|台前县]]、[[范县|范县]]相望。南北长44千米，东西宽36千米，位于[[济南|济南]]西南方向150千米处。

== 历史 ==
[[春秋|春秋]]时期，[[鲁成公|鲁成公]]四年冬（前587年），鲁国为加强防御，筑城名郓。[[隋朝|隋朝]][[开皇|开皇]]十年（590年）于[[万安县_(隋朝)|万安县]]置[[郓州|郓州]]；开皇十八年（598年）改万安县为郓城县。

== 行政区划 ==
郓城县辖21个（乡）镇：
*镇：[[郓城镇|郓城镇]]、[[黄安镇|黄安镇]]、[[杨庄集镇|杨庄集镇]]、[[侯咽集镇|侯咽集镇]]、[[武安镇|武安镇]]、[[郭屯镇|郭屯镇]]、[[丁里长镇|丁里长镇]]、[[玉皇庙镇|玉皇庙镇]]、[[程屯镇|程屯镇]]、[[随官屯镇|随官屯镇]]、[[张营镇|张营镇]]、[[潘渡镇|潘渡镇]]；
*乡：[[双桥乡|双桥乡]]、[[唐庙乡|唐庙乡]]、[[南赵楼乡|南赵楼乡]]、[[黄堆集乡|黄堆集乡]]、[[黄集乡|黄集乡]]、[[李集乡|李集乡]]、[[张集乡|张集乡]]、[[水堡乡|水堡乡]]、[[陈坡乡|陈坡乡]]。

== 经济 ==
[[黄河|黄河]]流经郓城28千米，水资源充沛，年可引黄河水2.8亿立方米。农业基础较好，是全国商品粮、优质棉、鲁西黄牛、小尾寒羊、青山羊基地县和“平原绿化先进县”。

== 资源 ==
煤炭资源丰富，是中国规划的“13个大型能源基地”之一，境内优质煤地质储量30.81亿吨，煤田面积646.2平方公里。近年正在努力打造“新兴煤电新城基地”。

== 名人 == 
郓城是《[[水浒|水浒]]》故事的发祥地，素有“梁山一百单八将，七十二名在郓城”之说，义军领袖[[宋江|宋江]]、[[晁盖|晁盖]]、[[吴用|吴用]]都是郓城人，与水浒故事有关的遗址遗迹较多，有智取生辰纲的黄泥岗，宋江杀惜的乌龙院，梦得天书的九天玄女庙，宋江故里的宋宅、宋坑、宋井等。

[[中華人民共和國|中華人民共和國]]第五代最高領導人[[習近平|習近平]]夫人，中国当代著名歌唱家[[彭丽媛|彭丽媛]]也是郓城人。
