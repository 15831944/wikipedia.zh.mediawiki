《'''法学评论'''》是教育部主管、[[武汉大学法学院|武汉大学法学院]]主办的[[法学期刊|法学期刊]]，创刊于1980年。主要栏目有评论、专论、争鸣、法律时务、立法研究、外国法制、欧盟法论坛和立法建议，是中国中文类核心期刊之一。<ref>{{cite web |url=http://www.xsqk.org/qikan/about1290.html |language= |title=法学评论 |publisher=学术期刊 |date= |accessdate=2012年11月 |deadurl=yes |archiveurl=https://web.archive.org/web/20130119194514/http://xsqk.org/qikan/about1290.html |archivedate=2013-01-19 }}</ref>

== 简介 ==
原名《法学研究资料》，1983年经文化部批准改名《法学评论》，[[彭真|彭真]]题写刊名。初期[[韩德培|韩德培]]、[[马克昌|马克昌]]教授负责编辑出版工作。1986年马克昌任主编。1993年至今[[黄进_(法学家)|黄进]]任主编<ref>{{Cite web |url=http://fxy.whu.edu.cn/html/3/publication/2010/0902/159.html |title=《法学评论》 |access-date=2012-11-11 |archive-url=https://web.archive.org/web/20121031091953/http://fxy.whu.edu.cn/html/3/publication/2010/0902/159.html |archive-date=2012-10-31 |dead-url=yes }}</ref>。
