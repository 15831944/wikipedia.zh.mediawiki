'''法国国际广播电台'''（{{lang-fr|'''R'''adio '''F'''rance '''I'''nternationale, '''RFI'''}}）简称'''法广'''，前身是1931年成立的[[法国殖民地|法国殖民地]]电台，当时是作为[[法国广播电台|法国广播电台]]的一部分，为[[法属赤道非洲|法属赤道非洲]]而服务的[[传媒|传媒]]。1938年更名为“巴黎－世界电台”。1975年，巴黎－世界电台更名为法国国际广播电台，隶属[[法国广播集团|法国广播集团]]（Groupe Radio France）。1986年，[[法国国会|法国国会]]通过一个新的[[法律|法律]]，允许法广独立于法国广播之外运营，随后发展成为了一个国际广播集团。目前该集团旗下有数间电台，包括法国国际广播电台、[[蒙特卡洛中东电台|蒙特卡洛中东电台]]（RMC Moyen-Orient）、[[巴黎－里斯本电台|巴黎－里斯本电台]]（Paris Lisbonne）、[[法广－索菲亚电台|法广－索菲亚电台]]（RFI Sofia）以及设在[[罗马尼亚|罗马尼亚]]的[[法广－三角洲|法广－三角洲]]电台（Delta RFI）。

法广是在[[法国外交部|法国外交部]]的预算和赞助下运营，它用多种[[语言|语言]]向全世界广播，其中包括[[法语|法语]]、[[英语|英语]]、[[法语（非洲地区）|法语（非洲地区）]]、[[斯瓦希里语|斯瓦希里语]]、[[豪萨语|豪萨语]]、[[罗马尼亚语|罗马尼亚语]]、[[波斯语|波斯语]]、[[西班牙语|西班牙语]]、[[葡萄牙语|葡萄牙语]]、[[葡萄牙语（巴西）|葡萄牙语（巴西）]]、[[俄语|俄语]]、[[柬埔寨语|柬埔寨语]]、[[越南语|越南语]]、[[华语|华语]]。其中[[德语|德语]]、[[阿尔巴尼亚语|阿尔巴尼亚语]]、[[波兰语|波兰语]]、[[阿拉伯语|阿拉伯语]]、[[保加利亚语|保加利亚语]]广播已于2009年停播，[[老挝语|老挝语]]节目于2010年2月1日停播。法广的英语广播是每天五个半小时。

== 华语广播 ==
法国国际广播电台华语节目于1989年6月8日开播。节目从最初的每天30分钟，增加到目前的每天三小时，通过网络直播、卫星、播客等方式辐射全球及巴黎大区听众。每个时段播音的前15分钟为新闻报道。法广华语节目通过全方位报道国际时事、中国动态以及法国与欧洲大事件，兼顾法国及世界舆论对全球时事的分析与评论，力求在准确、快速报道世界动向的同时，以较多角度和较宽视野分析世界大事，追寻突发事件背后的来龙去脉。

法广华语广播节目，由各位记者独立主持制作。从政治、经济、社会、地缘、军事、思潮、文化、环保、艺术、科技、旅游等各个侧面深入报道和探讨日益成为整体的当今世界。观照当今世界全球化的大趋势，反映人类社会文明发展的多样性。主要节目： 国际要闻、要闻解说、法国报纸摘要（周一至周五）/法国舆论看中国（周末）、香港传真（周六）、台北一周（周日）、当今世界。专题节目分别是《走进法兰西》、《法中交往》、《今日欧洲》、《欧亚论坛》、《热门话题》、《巴黎华人》、《关注中国》、《中华世界》、《今日亚洲》、《思潮与政见》、《书香世界》、《周末文摘》、《大众传媒》、《环境与健康》、《科技之窗》、《法国舆论看中国》、《沧海桑田》、《人与社会》、《公民广场》、《妇女与家庭》、《人口与未来》、《法律生活》、《国际纵横》、《焦点评述》、《人物特写》、《东西南北》、《文化与艺术》、《艺文生活》、《旅游消闲》、《体育热线》、《明镜书刊》（周日）。

目前法广中文部有常设记者16名，非常设记者8名，分布世界各地的特约记者15名。另有技术制作人员6名，行政秘书1名，中文部主任为旅法华人陈彦。

== 法国国际广播集团 ==
法国国际广播集团（Groupe Radio France Internationale）是法国唯一面向全球播音的公营广播集团。就2004年的预算来讲，来自法国外交部的资金为七千一百四十二万欧元，来自视听税收的资金为五千三百万欧元，自筹资金为四百一十三万欧元。法国国际广播集团通过新闻节目及时事专题节目迅速、快捷地向全球播报发生在世界各个角落的政治、经济、体育、社会及文化活动。法国国际广播集团的现总裁: 安托万·施瓦茨 (Antoine SCHWARZ)。

== 被防火长城屏蔽 ==
法国国际广播电台中文网及官方网站被[[防火长城|防火长城]]完全屏蔽。

== 争议 ==
2008年12月5日，中共官方媒体《[[环球时报|环球时报]]》以《法媒：温州老人＂施粥摊＂寒风中暖人心》转载法广中文网《上海视窗》专题同名文章，但篡改文章来源为“法国之声”。法国并不存在名为“法国之声”的媒体。另外，法广指《环球时报》刊登的《法媒煽动欧洲与中国保持距离》文章大量抄袭法广报道，也没有注明出处，并加上贬损和诱导性用语，已构成[[侵权行为|侵权行为]]。随後法广发表声明，谴责《环球时报》不注明真实出处。最後，第二篇文章在《环球时报》官网被删除<ref>{{cite web|url=http://www1.rfi.fr/actucn/articles/108/article_10882.asp|language=zh|title=法广声明|publisher=法广| date=2008-07-12|accessdate=2015-08-11}}</ref>。

《环球时报》2011年7月指，在和田爆炸案中，法广使用了一张[[2009年乌鲁木齐骚乱|2009年乌鲁木齐骚乱]]期间的照片，标题就指为和田爆炸现场<ref>{{cite news|title=扭曲中国反恐，西方有“嗜血般的兴奋”|url=http://opinion.huanqiu.com/picture/2011-07/1838107.html|language=zh-cn|publisher=[[环球网|环球网]]|date=2011-07-21|archiveurl=https://web.archive.org/web/20111014075432/http://opinion.huanqiu.com/picture/2011-07/1838107.html|archivedate=2011-08-13}}</ref>。
