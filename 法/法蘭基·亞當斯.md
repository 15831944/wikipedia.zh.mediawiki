'''法蘭基·亞當斯'''（'''Frankie Adams'''，{{bd|1994年|1月3日||}}）是一名新西蘭女演員和[[薩摩亞族|薩摩亞族]]業餘拳擊手。她身高有180厘米（5英尺11英寸）。<ref name="Blastr 14 April 2016">{{Cite web |url = http://www.blastr.com/2016-4-14/exclusive-expanses-ty-franck-daniel-abraham-reveal-major-cast-addition-season-2 |title = Exclusive: The Expanse's Ty Franck & Daniel Abraham reveal a major cast addition for Season 2 |author = Cher Martinetti |date = 2016-04-14 |publisher = Blastr |language = en}}</ref>

== 演出 ==
亞當斯的第一個角色是在16歳時，於肥皂劇《[[肖特蘭街|肖特蘭街]]》中飾演{{link-en|Ula Levi|Ula Levi}}。她也在電視劇《{{link-en|温特沃斯|Wentworth (TV series)}}》及電影《''1000 Ropes''》中演出。<ref>{{Cite web |url = http://www.radionz.co.nz/news/national/301603/kiwi-actor-frankie-adams-cast-in-us-space-drama |title = Kiwi actor Frankie Adams cast in US space drama |date = 2016-04-15 |publisher = Radio New Zealand |language = en}}</ref>

2016年，她被揀選在美國科幻電視劇《[[太空無垠|太空無垠]]》第二季中扮演士兵Bobbie Draper。<ref name="Blastr 14 April 2016" /><ref>{{Cite web |url = http://deadline.com/2016/04/the-expanse-frankie-adams-bobbie-draper-1201738056/ |title = ‘The Expanse’ Casts Frankie Adams As Bobbie Draper |author = Denise Petski |date = 2016-04-14 |publisher = Deadline Hollywood |language = en}}</ref>
