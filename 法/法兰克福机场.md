'''法兰克福机场'''（{{lang-de|'''Flughafen Frankfurt am Main'''或者'''Rhein-Main-Flughafen'''}}；{{Airport codes|FRA|EDDF}}），一般也稱作「'''萊茵-美茵國際機場'''」，或「'''法蘭克福國際機場'''」，是位于德国黑森州法兰克福的一座民用機場。位於法蘭克福市中心南方約12公里的法蘭克福國際機場是由[[Fraport公司|Fraport公司]]所經營，除了是歐洲最重要的客運航空樞紐之外，它也是歐洲境內貨運航班量排行第二的集散點。在1947年到2005年這段期間，法蘭克福机场的南半部一直都是[[美國空軍|美國空軍]]重要的空运基地——[[莱茵-美因空军基地|莱茵-美因空军基地]]——的所在地，但該基地已在2005年底正式交還給德國，並由Fraport接手經營管理，而原空軍基地的任務則轉移由[[拉姆施泰因空軍基地|拉姆施泰因]]（Ramstein）與[[斯潘達勒姆空軍基地|斯潘達勒姆]]（Spangdahlem）兩基地承接。

法兰克福国际机场是德国的[[國家航空公司|國家航空公司]]——[[德国汉莎航空公司|德国汉莎航空公司]]的主場基地與主要[[樞紐機場|樞紐機場]]。由于法兰克福的容量有限，因此除了法蘭克福外，汉莎航空公司又另行在[[慕尼黑|慕尼黑]]的[[慕尼黑国际机场|慕尼黑国际机场]]設置了第二樞紐，共同負擔該公司龐大的班機接駁量。

== 排名 ==
法兰克福国际机场比[[伦敦|伦敦]]的[[希思罗国际机场|希思罗国际机场]]提供更多的飞行目的地，但按乘客流量来算，法兰克福国际机场在欧洲位列第三名，全球第八，排在[[伦敦|伦敦]]的[[希思罗国际机场|希思罗国际机场]]和[[巴黎|巴黎]]的[[夏尔·戴高乐国际机场|夏尔·戴高乐国际机场]]之后。

* 2007年法兰克福机场客运量为54,161,856人次 <ref>http://www.aci.aero/cda/aci/display/main/aci_content.jsp?zn=aci&cp=1-5-54_9_2__</ref>，[[希思罗机场|希思罗机场]]为68,068,554人次，[[夏尔·戴高乐机场|夏尔·戴高乐机场]]为59,919,383人次。 
* 法兰克福机场以477475架次的起降量位列欧洲第二<ref>http://www.aci.aero/cda/aci/display/main/aci_content.jsp?zn=aci&cp=1-5-54-57_9_2__</ref>，排名在夏尔·戴高乐机场（525660架次）和希思罗机场（475999架次）之间。
* 按货运量来算，法兰克福机场依然是第二名（1838894吨）<ref>http://www.aci.aero/cda/aci/display/main/aci_content.jsp?zn=aci&cp=1-5-54-190_9_2__</ref>，仅次于夏尔·戴高乐机场（1876900吨），位列希思罗机场之前（1412033吨）。

== 历史 ==
1936年莱茵-美茵空军基地启用，是德国[[第二次世界大战|第二次世界大战]]时期第二大机场（仅次于[[柏林|柏林]]的[[腾珀尔霍夫机场|腾珀尔霍夫机场]]）。战后，它成为柏林空中运输的主要基地。

在1972年机场新的航站楼（即现在的1号航站楼）投入使用之前，它都不是主要的国际航班集散中心。

在正常運作情況下，靠外側的兩條平行跑道（07L/25R和07R/25L）用來降落，而中央跑道（07C/25C）和西跑道（18）用來起飛。三條平行跑道因皆可雙向使用而各有兩個跑道號碼標示，但西跑道卻由於只能單向使用而只有一個跑道號碼標示。

三條跑道（南、北和西）在2010年處理了464,432架次，亦即每小時83架次。在西北跑道於2011年十月啟用後,機場每小時應可處理126架次。預計到2020年時起降架次數將可增加到700,000。第四條跑道的啟用使得法蘭克福機場首次能夠進行獨立平行降落，這是因為西北與北跑道間的距離為1,400公尺(4,593 ft)。在原來的北與南跑道這是不可行的，因為它們間的安全距離低於國際民航組織的要求。

== 航空事故 ==
1969年，一架[[阿富汗航空公司|阿富汗航空公司]]701航班的[[波音727|波音727]]飞机在从法兰克福国际机场飞往[[伦敦盖特威克机场|伦敦盖特威克机场]]的途中坠毁，机上66名乘客中的50名和2名地面人员遇难。

1983年5月22日在莱茵-美因空军基地的一场[[航空展|航空展]]中，一架[[加拿大皇家空军|加拿大皇家空军]](RCAF)的[[F-104|F-104]]战斗机附毁在附近的街道上，击中一辆汽车，车上的乘客全部遇难。飞行员弹射出舱。

美国[[泛美航空103號班機|泛美航空103號班機]]在蘇格蘭上空爆炸。該班機从法兰克福国际机场起飞，大约有一半的人和行李在[[倫敦希斯路机场|倫敦希斯路机场]]换机。

== 建筑物及其功能 ==
法兰克福国际机场有两座航站楼，由走廊连接，也可以使用旅客运送工具和[[巴士|巴士]]。

=== 1号航站楼 ===
[[File:AirportFrankfurt_terminal1.jpg|thumb]]
1号航站楼在1972年3月14日投入使用。在当时这是一个现代风格的设计，银亮的内部和起伏的外墙。它被分成三个大厅，主要由[[德国汉莎航空|德国汉莎航空]]和它的[[星空聯盟|星空聯盟]]伙伴使用。
=== 2号航站楼 ===
[[File:AirportFrankfurt_terminal2.jpg|thumb]]
2号航站楼启用于1994年10月24日。从正面看，它被设计成类似于一座古典[[火车站|火车站]]的样子。它被分成二个大厅。

=== 其它建筑 ===
法兰克福有南北两个货物站以及一个单独的通用航站楼在机场的南端。在1号航站楼附近还有一家[[喜来登|喜来登]]酒店。

===後續增建計畫===
法兰克福国际机场除了原本已經著手進行改裝以便符合[[空中巴士A380|空中巴士A380]]泊靠所需的1B航廈之外，也開始A Plus航廈的修築以便對應未來越來越多的A380停泊需求。至於在回收了莱茵-美因空军基地之後所獲得的舊軍機場用地上，當局計畫修築一個可以提供75個停機位的三號航站大廈，以大幅提昇機場的胃納能力。由於距離原主航廈較遠，三號航廈與主建築之間會以Sky Line接駁電聯車串連。

== 地面交通 ==

法蘭克福機場內外均設有火車站，分別是「區域列車站」（Regionalbahnhof）以及「長途列車站」（Fernbahnhof）。

*區域列車站（第1、2、3月台）：位在第一航站大廈地下，提供城市快鐵列車（[[S-Bahn|S-Bahn]]）8號線、9號線的服務，乘客一般由此前往法蘭克福市區，平日每15分鐘開出一班，收費約4歐元，前往市區的行車時間約為12分鐘。往反方向搭乘則可抵達鄰近城市[[萊茵蘭-普法茲|萊茵蘭-普法茲]]首府[[美茵茲|美茵茲]]，以及[[黑森邦|黑森邦]]首府[[威斯巴登|威斯巴登]]。除了城市快鐵（[[S-Bahn|S-Bahn]]），區域列車站也提供區間快車（RE）和區間車（RB）服務，開往附近路程三小時以內的城鎮。例如[[哈瑙|哈瑙]]、[[科布林茲|科布林茲]]，以及[[薩爾蘭|薩爾蘭]]首府[[薩爾布魯根|薩爾布魯根]]。

*長途列車站（第4、5、6、7月台）：位於第一航站大廈外側，是一座銀白色的獨立長形建築物，現代感十足。乘客可以在此轉搭城際高速列車（[[ICE列车|ICE]]）或一般城際快車（IC），前往[[科隆|科隆]]、[[波昂|波昂]]、[[杜塞道夫|杜塞道夫]]、[[漢堡|漢堡]]、[[漢諾威|漢諾威]]、[[慕尼黑|慕尼黑]]、[[紐倫堡|紐倫堡]]、[[斯圖加特|斯圖加特]]、[[萊比錫|萊比錫]]等等距離較遠的德國城市。除了連通國內，長途列車站也提供國際列車服務，開往[[布魯塞爾|布魯塞爾]]、[[阿姆斯特丹|阿姆斯特丹]]、[[巴塞爾|巴塞爾]]、[[維也納|維也納]]等外國城市。有時，抵達機場的旅客也可以在此搭乘來自各地的長途列車，進入法蘭克福市區。

[[德國鐵路|德國鐵路]]與[[漢莎航空|漢莎航空]]、[[美国航空|美国航空]]及[[阿聯酋航空|阿聯酋航空]]聯營機場鐵路服務（AirRail Service）。

多家巴士公司也提供前往法蘭克福國際機場的巴士服務。

機場也接近A3與A5高速公路；乘的士前往法蘭克福市區，費用約為40[[歐元|歐元]]。

== 機場建設爭議 ==
法蘭克福機場距離市中心僅僅12分鐘車程，擁有兩座鐵路車站，交通極其便利。然而距離市區太近，既是優勢也是劣勢。隨著萊茵-美茵都會區的經濟發展，和法蘭克福國際金融與會展之都的交通需求，機場當局已經增闢西北跑道，而增建第三航廈也早已在腹案之中。不過機場所帶來的噪音與各種管制，讓萊茵-美茵都會區部份處在飛機航道上的縣市，尤其是黑森邦大蓋勞縣以及萊茵邦[[美茵茨|美茵茨]]市，怨聲載道。自增建第二航廈及西北跑道的年代起，部分居民與政黨即要求停止擴建機場，至今是否增建第三航廈仍然爭議重重，成為每次地方選舉的重要議題。<ref name=綠黨黑森邦黨部"">{{cite web|url=http://www.gruene-hessen.de/themen/frankfurter-flughafen/|title=綠黨黑森邦黨部：法蘭克福機場議題專區 |accessdate=2015-04-20}}</ref><ref name=黑森邦大蓋勞縣"">{{cite web|url=http://www.gg-online.de/html/v_ausfluege_flughafen_proteste.htm|title=黑森邦大蓋勞縣入口網：法蘭克福機場抗爭|accessdate=2015-04-20}}</ref><ref name=萊茵邦美茵茨市"">{{cite web|url=http://www.gg-online.de/html/v_ausfluege_flughafen_proteste.htm|title=萊茵邦美茵茨市自發性公民團體：不要機場|accessdate=2015-04-20}}</ref>
