'''法魯克一世'''（{{lang-ar|فاروق الأول}}；[[拉丁文|拉丁文]]转写：''Fārūq al-Awwal''，{{bd|1920年|2月11日|1965年|3月18日|catIdx=Farouk}}），全名'''穆罕默德·法魯克'''（''Muhammad Fārūq''），[[埃及|埃及]]国王，1936年至1952年在位，是[[穆罕默德·阿里王朝|穆罕默德·阿里王朝]]的第十任统治者。

== 早年生活 ==

法魯克於1920年2月11日出生於埃及[[亚历山大港|亚历山大港]][[蒂恩角宮|蒂恩角宮]]（又譯埃及冬宮），奥斯曼帝国埃及省督[[穆罕默德·阿里帕夏|穆罕默德·阿里帕夏]]的玄孙，[[穆罕默德·阿里王朝君主列表|埃及苏丹]][[福阿德一世|福阿德一世]]与王后娜兹莉的长子，在[[福阿德一世|福阿德一世]]所有儿子中排行第二。福阿德一世原正為無繼承人之事煩惱，法魯克的出生使他極為高興，當日大擺酒宴，慶祝三日。法魯克刚出生便成为苏丹世子。1922年福阿德称埃及国王，法鲁克成为埃及王储，被封为“赛义德亲王”。

法魯克於五歲開始接受教育，不過他有[[學習障礙|學習障礙]]，所以毫無進步。但法魯克在父親的栽培下，意外練得一手好字。盼望能有所作為的父亲決定將法魯克送去[[英國|英國]]深造，但他不學無術，成日與[[貴族|貴族]]子弟以[[賭博|賭博]]及[[打獵|打獵]]當作消遣。除此之外法魯克生性好色，常與少女廝混，至少與十幾位女性有過關係。

1936年5月，法魯克因父親去世而繼承王位，年僅十六歲便坐擁1億美元的財產、200輛汽車和7.5萬英畝肥沃土地。直到1937年7月26日法魯克正式達到法定年齡順勢登基。

== 在位生活 == 
繼承王位後的法魯克於1938年同貴族小姐法里達（''Farida Favzira''）成婚，不過婚後仍成日勾引婦女，公開主張人在愛情上不應受到束縛，應當允許有情婦存在。其情婦不勝枚舉，如女演員及知名女才子，1948年法魯克與法里達離婚。除此之外法魯克愛好美食，一頓早餐可食用十二顆雞蛋，午餐亦可使食用40隻[[鵪鶉|鵪鶉]]，一天可飲用高達三十瓶的[[啤酒|啤酒]]，時常將自己關進黑暗房間裡不斷食用[[巧克力|巧克力]]，體重直線上升，增至300磅。任內仍維持其嗜賭本性，曾於7小時內輸掉15萬美元。

法魯克還有一個相當知名的怪癖，法魯克於[[監獄|監獄]]裡同一位老[[扒手|扒手]]拜師學藝，學了一身竊盜功夫，為感謝這名扒手，竟破天荒的使其免釋出獄。法魯克學成後便縱橫宮內各種舞會及晚會，遊走貴賓之間行竊，並將得手之物當作個人收藏。其最為人所知之事跡是竊取了[[英國|英國]][[首相|首相]][[邱吉爾|邱吉爾]]的懷錶，最後因英國政府嚴正抗議法魯克才將懷錶交還邱吉爾。法魯克也曾於1944年[[伊朗|伊朗]][[國王|國王]]送葬隊經過[[埃及|埃及]]時從國王的屍體身上偷走了陪葬物，包括寶劍、寶帶和勳章。知曉內情者則戲稱法魯克為「[[開羅|開羅]]小偷」。

法魯克於1948年在珠寶店結識了一位十六歲的女孩[[納里曼_(埃及王后)|納里曼]]（''Narriman''），立刻對年輕貌美的她進行熱烈追求。納里曼父母只好讓她與未婚夫扎基·哈謝姆解除婚約，於1951年5月6日與31歲的法魯克成婚。法魯克為納里曼準備了釘上兩萬多顆[[鑽石|鑽石]]的新娘禮服，數千美元的女裝，以及十三個禮拜的[[歐洲|歐洲]]豪華旅行。

法魯克其實也有他的功績。法魯克十分重視教育，任內建立了不少[[學校|學校]]，但成效不大，未能獲得群眾肯定，甚至有朝臣暗中批評他治國無方。

1952年，[[納賽爾|納賽爾]]領導的[[自由軍官組織|自由軍官組織]]在亳無預警之下包圍了法魯克一世出生地[[蒂恩角宮|蒂恩角宮]]（亦被稱為埃及冬宮），法魯克一世被迫在7月26日簽字退位。法魯克在被允許的情況下乘坐皇家遊艇[[馬赫魯薩號|馬赫魯薩號]]流亡[[希臘|希臘]]及[[義大利|義大利]]，他在船上的38箱[[香檳|香檳]]箱中藏了無數金銀財寶，但其他擁有之物都遭到[[革命|革命]]者奪下。人們在他的寓所內除了發現其個人收藏的寶物和[[贓物|贓物]]外，也發現了大量的[[色情|色情]]讀物。

== 退位生活 ==

法魯克於1952年宣佈王位由年僅六個月的兒子[[福阿德二世|福阿德二世]]繼任，自己則流亡義大利。

1953年6月18日，埃及軍事政變當局正式宣佈廢除[[君主制|君主制]]。建立[[共和制|共和制]]。

王后納里曼則在1954年與法魯克辦理離婚，心灰意冷的法魯克在[[羅馬|羅馬]]路上四處找人求婚，但其魅力及權力早已不如當年。
1965年3月18日，法魯-{克}-死於義大利[[羅馬|羅馬]]的聖卡米洛醫院，終年四十五歲，葬於開羅。其死後眾說紛紜，{{fact|有說他生前的最後一頓飯據說共吃了12隻大[[龍蝦|龍蝦]]、10顆[[牡蠣|牡蠣]]、8條魚、5碗炒飯，此外還有數不清的[[奶酪|奶酪]]、[[果醬|果醬]]、大餅、[[豆類|豆類]]、[[蔬菜|蔬菜]]和[[水果|水果]]，之後他躺在沙發上奄奄一息，不久便死去。|來源請求}} 而陰謀論者則認為，法魯克是被埃及[[秘密警察|秘密警察]]暗殺。{{fact|亦有說，統計法魯克生前所吃的食物數量，足以供應一整個[[軍隊|軍隊]]。|來源請求}}

== 子女 ==
[[File:Farouk_family_official.jpg|thumb]]
法魯克於1938年1月20日与法里達·法夫齐拉（Farida Favzira，1921年9月5日—1988年10月15日）成婚，1948年11月17日离婚，婚后有三个女儿：
*长女'''法里亚尔'''（''Farial''），1938年11月17日生于[[亚历山大港|亚历山大港]]，1966年在[[伦敦|伦敦]]与[[瑞士|瑞士]]人让-皮埃尔·佩尔腾（''Jean-Pierre Perreten ''）结婚，有一女。2009年11月29日在瑞士[[日内瓦|日内瓦]]去世。
*次女'''法伊齐亚'''（''Faizia''），1940年4月7日生于开罗，2005年1月27日在瑞士去世。
*三女'''法狄亚'''（''Fadia''），1943年12月15日生于开罗，1965年在伦敦与瑞士人彼得·亚历克谢耶维奇·奥尔洛夫结婚，有二子。
1951年5月6日法魯克与納里曼·萨蒂克（Nariman Sadeq，1934年10月31日—2005年2月16日）结婚，有一子：
*长子'''[[福阿德二世|艾哈迈德·福阿德]]'''（''Ahmad Fu'ad''），1952年至1953年为埃及国王。
