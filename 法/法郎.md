'''法郎'''（{{lang-fr|'''Franc'''}}，簡寫：'''{{Unicode|₣}}'''）是[[貨幣|貨幣]]的一種。可以指以下貨幣：
*[[法國法郎|法國法郎]]，已停止流通
*[[比利時法郎|比利時法郎]]，已停止流通
*[[盧森堡法郎|盧森堡法郎]]，已停止流通
*[[摩納哥法郎|摩納哥法郎]]，已停止流通
*[[瑞士法郎|瑞士法郎]]
*[[中非法郎|中非法郎]]（於[[赤道几內亞|赤道几內亞]]、[[剛果共和國|剛果共和國]]、[[加蓬|加蓬]]、[[喀麥隆|喀麥隆]]、[[乍得|乍得]]及[[中非共和國|中非共和國]]使用）
*[[西非法郎|西非法郎]]（於[[貝寧|貝寧]]、[[布基納法索|布基納法索]]、[[象牙海岸|象牙海岸]]、[[幾內亞比紹|幾內亞比紹]]、[[馬里|馬里]]、[[尼日爾|尼日爾]]、[[塞內加爾|塞內加爾]]及[[多哥|多哥]]使用）
*[[太平洋法郎|太平洋法郎]]（[[法属波利尼西亚|法属波利尼西亚]]、[[新喀里多尼亚|新喀里多尼亚]]和[[瓦利斯和富图纳|瓦利斯和富图纳]]使用）
*[[葛摩法郎|葛摩法郎]]
*[[马达加斯加阿里亚里|馬達加斯加法郎]]，已停止流通

== 法國法郎 ==
{{main|法国法郎}}

最早在1360年法國國王[[約翰二世|約翰二世]]鑄造叫法郎的硬幣，[[法国大革命|法国大革命]]后推行公制，1795年，法郎開始作爲标准貨幣在[[法國|法國]]流通，取代原有的[[里弗尔|里弗尔]]。

2002年1月1日[[歐元|歐元]]發行之後，法郎已逐漸的停止流通。

除了在法国使用，法国法郎也曾是[[摩纳哥|摩纳哥]]和[[安道尔|安道尔]]的流通货币。现在，这两个国家也都已转用欧元。

== 瑞士法郎 ==
{{main|瑞士法郎}}
除了法国之外，原[[拉丁货币同盟|拉丁货币同盟]]中的瑞士和比利时均使用法郎作为货币单位。[[列支敦士登|列支敦士登]]使用瑞士法郎。

== 其他國家的法郎 ==
[[卢森堡|卢森堡]]独立后也使用法郎（比利时和卢森堡现改用[[欧元|欧元]]）。

下列曾经属于法国和比利时殖民地的國家，其官方貨幣也叫法郎：[[几内亚|几内亚]]、[[吉布提|吉布提]]、[[卢旺达|卢旺达]]、[[布隆迪|布隆迪]]、[[刚果|刚果]]、[[科摩罗|科摩罗]]。

另外，[[贝宁|贝宁]]、[[布基纳法索|布基纳法索]]、[[科特迪瓦|科特迪瓦]]、[[几内亚比绍|几内亚比绍]]、[[马里|马里]]、[[尼日尔|尼日尔]]、[[塞内加尔|塞内加尔]]和[[多哥|多哥]]8国统一使用[[西非法郎|西非法郎]]，[[喀麥隆|喀麥隆]]、[[中非共和國|中非]]、[[查德|查德]]、[[刚果共和国|刚果共和国]]、[[赤道几内亚|赤道几内亚]]和[[加彭|加彭]]6国統一使用[[中非法郎|中非法郎]]，两者等值但互不流通，常合称非洲法郎（“非洲法郎区”共15国，除上述14国外还包括[[科摩罗|科摩罗]]）。

[[马达加斯加|马达加斯加]]原来的法定货币是[[马达加斯加法郎|马达加斯加法郎]]，2005年1月1日起，转用[[阿里亞里|阿里亞里]]（Malagasy ariary）。
