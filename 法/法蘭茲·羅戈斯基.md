'''法蘭茲·羅戈斯基'''（[[德語|德語]]：'''Franz Rogowski''' ，{{bd|1986年|2月2日||}}），是一位[[德國|德國]]的男演員。

== 生平 ==
法蘭茲·羅戈斯基小時候曾進行唇裂手術，以致他的臉上留下疤痕並影響他的口齒發音；他在16歲時與母親衝突後就搬離家裡，中間曾前往瑞士小丑學校學習技法一年，他後來搬到柏林後以擔任舞者與街頭音樂家維生。<ref>{{cite web |url=https://www.nytimes.com/2018/05/02/movies/franz-rogowski.html |title=An Unlikely Sex Symbol, Poised for a Breakthrough |accessdate=2019-03-20 | work=[[紐約時報|紐約時報]]}}</ref>

2015年，法蘭茲·羅戈斯基因演出一鏡到底的《[[維多莉亞_(2015年電影)|維多莉亞]]》而受到矚目<ref>{{cite web |url=http://www.hollywoodreporter.com/news/berlin-is-victoria-germanys-answer-771095 |title=Berlin: Is 'Victoria' Germany's Answer to 'Birdman'? |accessdate=2015-02-17 | work=[[The_Hollywood_Reporter|The Hollywood Reporter]]}}</ref>；2017年，他參與演出[[麥可·漢內克|麥可·漢內克]]的法語片《{{link-en|完美結局 (電影)|Happy End (2017 film)|完美結局}}》<ref>{{cite web |url=https://www.festival-cannes.com/en/films/happy-end-1 | title= In Competition – Feature Films: Happy End |work= [[Cannes_Film_Festival|Cannes Film Festival]] |accessdate=2019-02-28}}</ref>
。2018年，[[第68屆柏林影展|第68屆柏林影展]]正式競賽單元中便有兩部德國電影由他主演，分別是《{{link-de|過境 (2018年電影)|Transit (2018)|過境}}》與《{{link-en|賣場華爾滋|In the Aisles}}》，他並憑藉後者獲得[[德國電影獎|德國電影獎最佳男主角獎]]。
