'''法屬圭-{}-亞那'''（{{lang-fr|'''Guyane française'''}}），正式名称是'''圭-{}-亚那'''（{{lang-fr|'''Guyane'''}}），是[[法国|法国]]的一个海外大区和{{link-fr|单一领土集体|Collectivité territoriale unique}}，下辖一个[[海外省|海外省]]，即圭-{}-亚那省。法属圭亚那也是[[欧盟|欧盟]]的一部分；位于[[南美洲|南美洲]]北部[[大西洋|大西洋]]边，与[[巴西|巴西]]和[[苏里南|苏里南]]交界。地理位置为[[纬度|北纬]]4°0-{’}-和[[经度|西经]]53°0-{’}-。

== 地理 ==
法属圭-{}-亞那面积83,534平方公里，与[[奥地利|奥地利]]相當。北濱大西洋，海岸线长378公里。东、南面与巴西为邻，边界长673公里，西面是苏里南，边界长510公里。

法属圭-{}-亞那南部位于横跨南美洲北部的[[圭亞那高原|圭-{}-亞那高地]]前，最高点高850-{zh-hans:米; zh-hant:公尺;}-。重要河流有[[马罗尼河|马罗尼河]]（它也是与苏里南的边界）、[[西纳马利河|西纳马利河]]和[[奥亚波克河|奥亚波克河]]（与巴西的边界）等。

法属圭-{}-亞那境內98%是热带雨林，森林覆盖率90%，是法国和欧洲联盟的大片森林區。大多数人口集中在北部海岸，大城市也集中於此。

法属圭-{}-亞那的气候属[[热带雨林气候|热带雨林气候]]，年降水量3500毫米。全年温度在28℃左右。从8月到12月是旱期，其它时间是雨期。平均湿度比较高，终年在80%到90%左右。法属圭-{}-亞那不受[[加勒比海|加勒比海]][[飓风|飓风]]的袭击。

== 历史 ==
法属圭-{}-亞那的原来的居民是加勒比人和[[阿拉瓦克人|阿拉瓦克人]]。

1498年，[[克里斯托弗·哥伦布|克里斯托弗·哥伦布]]来到圭-{}-亞那的海岸。一百多年后[[荷兰|荷兰]]人开始在这里殖民。

1604年，[[法国|法国]]开始侵入，建立居民点。后[[英国|英国]]、荷兰、法国和[[葡萄牙|葡萄牙]]相互争夺此地，直到1816年最后归属法国。

1946年，法国宣布法属圭-{}-亞那为法国的“[[海外省|海外省]]”。从此以后，许多土著号召自治，但支持从法国独立之支持度只有5%，部分因为[[法国政府|法国政府]]提供更多的津贴。

从1947年开始，法属圭-{}-亞那拥有有限的自治权。从此该地区在[[法国国民议会|法国国民议会]]和[[法国参议院|法国参议院]]各有两个议席。

1968年，欧盟在[[库鲁|库鲁]]建立了一个[[圭亞那太空中心|圭亞那太空中心]]，[[阿丽亚娜火箭|阿丽亚娜火箭]]在这个发射场发射。在库鲁，「欧洲太空计划」使法属圭-{}-亞那的这个角落变成了现代的世界，并吸引了一些移居到此工作的人力。

1977年，成为法国的一个大区。

== 政治 ==
如同法国其它省份一样，法屬圭亞那也参加法国的[[法國國民議會|国民议会]]和省级立法机构。在法国国民议会和参议院中法屬圭亞那各拥有两个议席。

所有法国法律在法屬圭亞那均被视为有效，但法国宪法第73条规定地方亦有权按地方特点对当地法律进行一定的修改。

从1982年开始法屬圭亞那是法国的一个大区和省。它分为两个区，19个乡和22个县。

法屬圭亞那設省政府及大區政府，政府首腦為政府委員會主席。各省選舉產生省議會和省政府。省主席為法屬圭亞那省份的領導人。法屬圭亞那省政府主席兼任大區政府主席。

每隔六年法属圭-{}-亞那进行省议会（19席）和大区议会（31席）的改选，从2010年3月26日起大区主席是[[鲁道夫·亚历山大|鲁道夫·亚历山大]]（[[w:Rodolphe_Alexandre|Rodolphe Alexandre]]）。

法属圭-{}-亞那是[[欧洲联盟|欧洲联盟]]的一部分，因此它与[[巴西|巴西]]及[[苏里南|苏里南]]的边界是[[欧洲联盟|欧洲联盟]]極西的边界，但不屬於[[申根區|申根區]]的地區（所有[[法國海外部分|法國海外部分]]均不歸入申根區）。

2016年1月1日，第一届[[法属圭亚那议会|法属圭亚那议会]]正式成立，取代此前的[[圭亚那大区委员会|圭亚那大区委员会]]和圭亚那总委员会，作为法属圭亚那新的立法机关。

== 经济 ==

[[File:Ile_Royale_musée_du_bagne.jpg|缩略图]]

林业是法屬圭亞那的一个重要经济行业，农业只集中在海岸地区，此外捕虾业和金矿是值得一提的产业。

最重要的贸易对象是法国本土、[[特立尼达和多巴哥|特立尼达和多巴哥]]和[[意大利|意大利]]。主要出口品有[[鱼|鱼]]、[[米|米]]和[[金|金]]，主要进口品有[[机器|机器]]和[[汽车|汽车]]。

库鲁发射场目前成为内地的一个经济和旅游动力。

近年来，日益猖獗的[[巴西|巴西]]非法淘金人，导致严重的生态环境破坏：他们使用[[汞|汞]]等有害物质来清理和加工黄金。

== 语言 ==
圭-{}-亞那原住民的语言有六种，分成三种语系：

# [[瓦扬皮语|瓦扬皮语]]（wayampi）、[[埃梅里用永语|埃梅里用永语]]（émérillon）属于[[图皮-瓜拉尼语系|图皮-瓜拉尼语系]]（Tupi-Guarani）；
# [[阿拉瓦克语系|阿拉瓦克语系]]（arawak）、[[巴里库尔语|巴里库尔语]]（palikour）属于[[阿拉瓦语系|阿拉瓦语系]]；
# [[卡里纳语|卡里纳语]]（kalihna）、[[瓦亚纳语|瓦亚纳语]]（wayana）属于[[加勒比语系|加勒比语系]]。

除此之外，还有一些基于[[法文|法文]]和[[英文|英文]]的[[克里奥尔语|克里奥尔语]]言。

== 其它 ==
糖和雨林木材成为了殖民地的经济支柱。奴隶们从非洲带了种植糖园的工作，尽管他们的成功被本地的印地安人的敌视和热带的疾病所限制。且在1848年的奴隶制度废除以后，种植园的产量，从来无法配合法国的加勒比海殖民地，本地的工业几乎崩溃了。在大约相同的时间，法国决定减少监狱的费用，将受刑人移民至圭-{}-亞那，贡献殖民地的发展。约有70,000罪犯，包括阿尔弗雷德·德雷福斯和亨利·帕皮隆·查利尔尼（Herry Papillon Charriere），在1852年和1939年之间到达。刚开始，许多流放的罪犯，熬过了恶劣的环境，大部分都死于疟疾和发烧。在第二次世界大战以后，圭-{}-亞那仍然是罪犯的殖民地。

=== 发射场 ===
[[欧洲空间局|欧洲空间局]]从[[库鲁发射场|库鲁发射场]]使用[[阿丽亚娜火箭|阿丽亚娜火箭]]发射[[人造卫星|人造卫星]]。

=== 军用 ===
法国在罗尚波有一个军事基地，[[法国外籍兵团|外籍兵团]]亦設有森林基地，并在这里驻扎和培训上百士兵，同時擔任太空中心的守備任務。2004年3月法国的外籍兵团就是从这里出发进驻[[海地|海地]]。
