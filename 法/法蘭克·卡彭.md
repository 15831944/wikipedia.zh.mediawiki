'''法蘭克·卡彭'''（'''Frank Capone'''，{{bd|1895年|7月16日|1924年|4月1日}}）是一名芝加哥黑幫，他參與了由他弟弟[[艾爾·卡彭|艾爾·卡彭]]的犯罪組織企圖接管[[伊利諾州|伊利諾州]]{{link-en|西塞羅 (伊利諾州)|Cicero, Illinois|西塞羅}}的行動。他有一個哥哥[[雷夫·卡彭|雷夫·卡彭]]，與艾爾和法蘭克一起工作。

== 兩名兄弟 ==
本名薩瓦托·卡彭（Salvatore Capone），法蘭克是加布里埃萊·卡彭（Gabriele Capone，1865年-1920年）的兒子，加布里埃萊是一名意大利移民，於1894年在紐約市與他的妻子特雷西·納萊奧拉（Teresa Raiola，1867年-1952年）定居。除了艾爾之外，法蘭克還有另一名成為黑幫的兄弟[[雷夫·卡彭|雷夫·卡彭]]。在紐約長大，法蘭克和艾爾與黑幫[[強尼·托里奧|強尼·托里奧]]都加入了[[五點幫|五點幫]]。到1918年，托里奧已經搬到芝加哥幫助一名親戚保住他的非法勾當，托里奧不久需要艾爾，後來法蘭克便加入他了。

到1920年，托里奧已經負責管理[[芝加哥犯罪集團|南邊幫]]，而[[禁酒令|禁酒令]]時代已經開始了。隨著幫派的權力和財富的增加，艾爾和法蘭克的權力財富也增加了。

== 接管西塞羅 ==
1923年，芝加哥選民推選了新的市長{{link-en|威廉·艾米特·丹佛|William Emmett Dever|威廉·丹佛}}，他接下來鎮壓托里奧、卡彭兄弟和他們的[[芝加哥犯罪集團|南邊幫]]。作為回應，托里奧任命艾爾在芝加哥郊區的{{link-en|西塞羅 (伊利諾州)|Cicero, Illinois|西塞羅}}創建[[地下酒吧|地下酒吧]]、[[妓院|妓院]]和非法賭博場所。一年之內，卡彭已將西塞羅市經理Joseph Z. Klenha和鎮上的委員放在幫派工資單上。法蘭克·卡彭的工作是代表幫派與西塞羅鎮議會進行交易。法蘭克與他的弟弟艾爾相比下較溫和，展現了一名值得尊敬的商人形象，總是穿著整潔的西裝。<ref name="crime">{{Cite web |url = http://www.crimemagazine.com/brothers-capone |title = The Brothers Capone |date = 2009-10-15 |publisher = Crime Magazine |language = en |access-date = 2017-06-05 |archive-url = https://web.archive.org/web/20171025081303/http://www.crimemagazine.com/brothers-capone |archive-date = 2017-10-25 |dead-url = yes }}</ref>

在1924年4月1日的[[初選|初選]]中，[[民主黨_(美國)|民主黨]]政客對[[共和黨|共和黨]]員Klenha和他的伙伴發起了嚴肅的選舉挑戰。為了保護幫派對西塞羅的政治控制，法蘭克在城市裡發動了一波恐怖活動的浪潮。他派南邊幫的成員前往投票站，用[[衝鋒鎗|衝鋒鎗]]和[[削短型霰彈槍|削短型霰彈槍]]確保當地居民「投右票」。不合作的選民會遭到毆打並被阻止投票。法蘭克在對手的競選總部進行了攻擊，亂翻他的辦公室，並毆打幾名競選人員。一名競選人員被射中兩條腿，另有八名競選人員被扣留，當選舉日結束時則被釋放。<ref name="crime" />

== 逝世與警察增援 ==
隨著選舉日動亂的過程，憤慨的西塞羅公民向庫克縣的[[縣法官|縣法官]]傑瑞其（Edmund J. Jareki）[[請願|請願]]。[[芝加哥警察局|芝加哥警察局]]（CPD）派出70名便衣警察到西塞羅維持投票秩序，而Jareki則以副警長的身份向他們發誓。CPD警察直到下午才到達西塞羅的街頭，到那時他們對選舉有關的暴力只有少微影響。

大約在黃昏時，由警佐古薩克（William Cusack）率領的探員小隊發現法蘭克·卡彭、{{link-en|查理·腓瑟提|Charles Fischetti}}，以及一名他們無法認出的身材矮小的男子後，在西塞羅大街和二十二街的投票站將車輛慢慢停下。這些探員走出他們的車，並在槍擊莫名其妙地爆發時開始走過去。後來幾名證人聲稱，黑幫從未開火。在晚些時候的調查中，警方聲稱法蘭克·卡彭首先開槍；他們製造了一支有缺少三發子彈的手槍，他們發誓他用過的。據說法蘭克以為這些[[便衣警察|便衣警察]]是敵對黑幫。在任何一種情況下，卡彭在隨後的混戰中被警長麥格林（Phillip J. McGlynn）致命地射殺。腓瑟提則衝過一個附近的空地，一到警察趕上他，只能扔掉他的武器並投降。第三名黑幫分子跑到南方，各自開火，並成功逃跑了。後來有一個城市神話漸漸變成說這個人只不過是艾爾·卡彭。事實上，第三名槍手後來被非常確定為David Hedlin，警方也打傷了他。<ref>Schoenberg, Robert, ''Mr. Capone'', New York: William Morrow & Co., 1992, pgs. 98-99.</ref> 

在這天結束時，卡彭的候選人Klenha勝出了。

== 葬禮 ==
法蘭克去世後，芝加哥報紙上充滿了讚美或譴責CPD的文章。後來，[[驗屍官|驗屍官]]的調查結果，確定法蘭克一直拒捕，射殺他是情有可原的。

1924年4月4日，法蘭克·卡彭舉行了奢華的葬禮，鍍銀的[[棺材|棺材]]，飾以價值2萬美元的花朵，車隊超過150輛汽車。諷刺的是，艾爾是從他的北邊幫對手[[狄恩·歐班寧|狄恩·歐班寧]]的店鋪購買花朵的。法蘭克被埋葬在芝加哥外面的{{link-en|卡梅爾山墓園|Mount Carmel Cemetery (Hillside, Illinois)}}。<ref>{{Cite web |url = https://www.findagrave.com/cgi-bin/fg.cgi?page=gr&GRid=7248998 |title = Salvatore Capone |publisher = Find A Grave |language = en}}</ref>《[[芝加哥論壇報|芝加哥論壇報]]》報導說，這儀式適用於「一個適當的紳士」上。由於尊敬他死去的哥哥，艾爾·卡彭在葬禮期間關閉了西塞羅的[[賭場|賭場]]和[[地下酒吧|地下酒吧]]兩個小時。
