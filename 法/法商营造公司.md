'''法商营造公司'''是一家在20世纪30年代活跃于[[上海|上海]]的建筑设计事务所，由法国工程师Rene Minutti于1930年在上海创立。Rene Minutti于1887年在瑞士日内瓦出生，就读于苏黎世工学院，1920年抵达上海<ref name="建筑史">{{Cite book |title = 上海百年建筑史(1840-1949)  |publisher = 同济大学出版社 |author = 伍江 |pages = 第137页 | ISBN = 7-560-83895-2 | accessdate = 2014-01-09 | url = }}</ref><ref name="地方志">[http://www.shtong.gov.cn/newsite/node2/node4/node2249/huangpu/node34875/node34905/node62401/userobject1ai19776.html 区县志 >> 区志 >> 黄浦区志 >> 第十编房屋建设 >> 第四章建筑设计、施工、管理 >> 第一节　设计] {{webarchive|url=https://web.archive.org/web/20131230233602/http://www.shtong.gov.cn/newsite/node2/node4/node2249/huangpu/node34875/node34905/node62401/userobject1ai19776.html |date=2013-12-30 }} 上海市地方志办公室.</ref>。
