《'''法海你不懂爱'''》，著名歌手[[龚琳娜|龚琳娜]]的歌曲，2012年12月31日，该歌曲首次亮相于[[湖南卫视|湖南卫视]]的跨年晚会。原是用以批判《[[白蛇传|白蛇传]]》中拆散情人的和尚[[法海_(白蛇傳)|法海]]，视频流传后引起网友們[[吐槽|吐槽]]，同时由于[[中国历史|中国历史]]上[[法海|法海]]真有其人，是[[唐朝|唐朝]]的禪僧[[金山法海|金山法海]]，佛教徒大為抗议，以及网友对文艺创作中娱乐底线的讨论，导致“[[法海事件|法海事件]]”。

== 概述 ==
该歌曲作者是龚琳娜的丈夫[[老锣|老锣]]，灵感来源自[[中国|中国]]古代[[神话|神话]]传说《[[白蛇传|白蛇传]]》[[许仙|许仙]]和[[白素贞|白素贞]]的爱情故事，由于农历2013年是蛇年，故而写作了这首歌。<ref>{{cite web|url=http://www.ln.xinhuanet.com/newscenter/2012-12/29/c_114200756.htm|title=龚琳娜推新作《法海你不懂爱》|accessdate=2012-12-29|publisher=新华网|archiveurl=https://web.archive.org/web/20160304194501/http://www.ln.xinhuanet.com/newscenter/2012-12/29/c_114200756.htm|archivedate=2016年3月4日|work=|language=zh-cn|deadurl=yes}}</ref>

该歌曲在湖南卫视播出之后，爆红网络，成为继《[[忐忑_(龚琳娜)|忐忑]]》之后的龚琳娜第二首神曲。<ref>{{cite web |url=http://news.cnhubei.com/xw/yl/201301/t2398071.shtml |title=法海你不懂爱成龚琳娜新神曲 网上爆红成热点 |accessdate=2013-01-01 |publisher= 荆楚网|archiveurl=http://news.cnhubei.com/xw/yl/201301/t2398071.shtml |archivedate=2013年1月1日 |work= |language=zh-cn}}</ref><ref>{{cite web |url= http://www.dzwww.com/xinwen/xinwenzhuanti/2008/ggkf30zn/201301/t20130101_8047965.htm|title= 龚琳娜新神曲法海你不懂爱 雷死人不偿命|accessdate=2013年1月1日 |publisher= 大众网|archiveurl=http://www.dzwww.com/xinwen/xinwenzhuanti/2008/ggkf30zn/201301/t20130101_8047965.htm |archivedate=2013年1月1日 |work= |language=zh-cn}}</ref>

从2013年元月中旬开始，凤凰网华人佛教频道发表了佛教界法师抗议龚琳娜歌曲以及学者关于法海事件的一系列文章，上千万网友参与评论。<ref>{{cite web |url=http://fo.ifeng.com/special/wendaotj1 |title=法海事件专辑1：别拿信仰开玩笑 不是道歉那点事 |accessdate=2013-03-17 |publisher= 凤凰网|archiveurl= |archivedate=  |work= |language=zh-cn}}</ref><ref>{{cite web |url= http://fo.ifeng.com/special/ganfashengsjb/ |title= 法海事件专辑2：敢发生说话是历史的进步|accessdate=2013年3月17日 |publisher= 凤凰网|archiveurl= |archivedate=  |work= |language=zh-cn}}</ref>
