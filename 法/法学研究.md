《'''法学研究'''》是由[[中国社会科学院|中国社会科学院]]法学研究所主办、法学研究编辑部编辑、法学研究杂志社出版的双月期刊，创办于1954年。为[[中国|中国]]最具影响力的[[法学|法学]]研究[[杂志|杂志]]之一<ref name="研究">{{cite web|url=http://www.baywatch.cn/a/qikandaohang/faxueqikan/20111118/1032.html | language= | title=法学研究 |publisher=中文期刊网 |date= |accessdate=2012年11月 }}</ref>。

== 历史 ==
《法学研究》的前身是中国政法学会1954年创刊、1957年停办的《'''政法研究'''》。1979年复刊。

== 主要专栏 ==
#法学专论，刊载研究[[中国法治|中国法治]]建设重要理论与实践问题的文章；
#法条释评，刊载研究[[中国法律|中国现行法律]]规定的文章；
#判解研究，刊载研究[[中国司法|中国司法]]解释和重大疑难案件判决的文章；
#书评，刊载对[[学术著作|学术著作]]的研究性评论<ref name= "研究"/>。

== 主编 ==

历任主编有[[金默生|金默生]]、[[孙亚明|孙亚明]]、[[张尚鷟|张尚鷟]]、[[吴大英|吴大英]]、[[李步云|李步云]]、[[王保树|王保树]]。

现任主编为著名民法学家[[梁慧星|梁慧星]]，副主编有[[张广兴|张广兴]]、[[张志铭|张志铭]]，责任编辑有张少瑜、庄立、谢海定、冉昊、范亚峰。

== 荣誉 ==
1999年获[[中国社会科学院|中国社会科学院]]首届优秀期刊奖、第二届全国百种重点社科期刊奖；2002年获中国社会科学院第二届优秀期刊奖。

== 参见 ==
*《[[环球法律评论|环球法律评论]]》，中国社会科学院法学研究所主办的另一双月刊，创刊于1979年，主要比较研究各国法律。
