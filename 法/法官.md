'''法官'''（[[臺灣|台湾]]前称[[推事|推事]]，[[法文|法文]]: '''Juge'''， [[英文|英文]]：'''Judge'''， [[西班牙文|西班牙文]]: '''Juez'''， [[德文|德文]]: '''Richter'''）是司法机构中审判人员的通称，[[司法|司法]]权的执行者。在不同[[法系|法系]]中法官的角色不尽相同，要求基本上都是不偏不倚、不受他人影响或掣肘、大公无私地根据[[法律|法律]]判案。

== 称谓 ==
{{维基文库|中华人民共和国法官法}}
根据《中华人民共和国法官法》，法官是依法行使国家审判权的审判人员，包括最高人民法院、地方各级人民法院和军事法院等专门人民法院的院长、副院长、审判委员会委员、庭长、副庭长、审判员和助理审判员。

在[[香港法院|香港法院]]內，於[[香港主權移交|香港主權移交]]後，稱呼法官需要在法官後加上[[閣下|閣下]]（The Honourable）二字，即法官閣下，以示尊敬。在[[香港殖民地時期|香港殖民地時期]]，則稱為[[勳爵#.E6.B3.95.E5.AE.98|大人]]，即法官大人。另見[[按察司|按察司]]。

在[[臺灣|臺灣]]，法官在法庭內的尊稱為'''庭上'''。在[[合議庭|合議庭]]，有審判長、受命法官、陪席法官之區別。

== 历史 ==
[[中国歷史|中国歷史]]傳說，古代[[帝|帝]][[尧|尧]]时代中有一位公正的法官[[皋陶|皋陶]]，被奉为中国司法鼻祖。他有一只[[獬豸|獬豸]]，在难以判决之时根据碰触可以决定谁有罪。

[[中国歷史|中国歷史]]傳說，古代宋仁宗时代中有一位公正的法官[[包青天|包青天]]被奉为中国「司法之神」。

== 在司法体系中的角色 ==
在[[大陸法系|大陸法系]]国家中，法官通常是和[[律師|律師]]分開訓練的；在[[訴訟|訴訟]]中會主動地調查案件、盤問證人和當事人，質疑證供的可信性，是法律的守護人，主要目的在發現事件的真相。

在[[普通法系|普通法系]]国家中，法官和[[律师|律师]]都是接受[[法学院|法学院]]的训练，法官一般是从经验丰富的[[律师|律师]]当中挑选，有时也从[[法学院|法学院]]的资深[[教授|教授]]当中挑选，而且一般必须得到[[議會|議會]]、立法机构的批准。在法庭审判中他不會主動地調查案件或盤問[[證人|證人]]和當事人；他的角色尤如球賽中的[[球證|球證]]，確保控辯雙方有平等的發言機會，並在聆聽雙方的理據後裁決。由於採用判例法，法官不只是應用法律，其判決某程度會延伸了法律，普遍被認為是德高望重才能擔當的職位。

[[普通法|普通法]]制度中法官的角色和古代[[中國|中國]]當人們遇上糾紛時尋求作評理的長者非常相似。他不會主動調查事件的始末，而是聆聽雙方的理據，並依當地的風俗習慣作他認為最符合大眾利益的判決。在原始普通法中並沒有成文的[[法律|法律]]，法官只是作他認為最佳的判決，故法官作的判決可視為「創造」了法律，而非只是應用法律。

== 香港 ==

2013年11月26日，[[立法會司法及法律事務委員會|立法會司法及法律事務委員會]]討論全體法官及司法人員於2013年至2014年年度加薪3.15%的建議，涉及約1千萬港元。身兼[[大律師|大律師]]的[[立法會議員|立法會議員]][[郭榮鏗|郭榮鏗]]認為，法官的工作量大而且支援不足，最終令到人才卻步，批評[[司法人員薪俸及服務條件常務委員會|司法人員薪俸及服務條件常務委員會]]的加薪建議脫離現實。立法會議員[[黃毓民|黃毓民]]則批評，不少法官皆為臨時委任，加上薪酬遠低於私人執業律師，故此難以吸納精英投身職業。另外，立法會議員[[劉慧卿|劉慧卿]]則認為法官人手不足以至拖延審訊時間，對當事人並不公道。[[行政署|行政署]]承認[[高等法院|高等法院]]原訟庭人手不足，2012年一般民事案件由申請排期至聆訊平均需時244日，遠超於180日標準，惟強調僅為短期人手調配出現問題，而非編制人手不足<ref>[http://orientaldaily.on.cc/cnt/news/20131127/00176_033.html 法官人手不足 拖延審訊] 《東方日報》 2013年11月27日</ref>。

== 特徵 ==
* 鑒於法官代表司法權獨立審判的重要性，現今世界上多數憲政國家會將有關法官的職權行使，薪俸、地位的保障以及任免限制直接規範在[[憲法|憲法]]內，尤其是積極避免審判涉入政治，或是政治干涉審判，以昭公信。
* 在一些國家，對於法官的裝扮有著特殊要求，以表示法官在法庭上地位的特殊性，及協助審判庭上的人員容易做出身分上的辨別。
例如：[[中華民國|中華民國]]的法官開庭時需穿著黑色鑲藍邊[[法袍|法袍]]。
* 法官執行審判權時，不因其判決有瑕疵而遭受指控，而且其判決不能由法官自行任意改判；因判決而遭受不利益之人，僅能透過上訴尋求救濟。
