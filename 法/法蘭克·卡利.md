'''弗朗西斯科·保洛·奧古斯托·卡利'''（{{lang-it|'''Francesco Paolo Augusto Calì'''}}，{{bd|1965年|3月26日|2019年|3月13日|}}），被稱為「法蘭克」（Frank）或「法蘭基小子」（Franky Boy），是{{link-en|甘比諾犯罪家族|Gambino crime family}}的美國犯罪老大。執法部門認為卡利是甘比諾的「[[西西里島黑手黨|西西里黑幫]]大使」，並將他與[[帕勒莫|帕勒莫]]的{{link-en|因澤里洛黑手黨大家族|Inzerillo Mafia clan}}聯繫起來。根據美國助理檢察官Joseph Lipton所說，「卡利被意大利的有組織犯罪成員視為一個有影響力和有權力的人。」<ref>{{Cite web |url = https://www.nydailynews.com/news/crime/reputed-gambino-crime-capo-house-bought-mob-cash-article-1.310000 |title = Reputed Gambino crime capo's house bought with mob cash? |author = John Marzulli |date = 2008-02-20 |publisher = New York Daily News |language = en}}</ref>卡利於2019年3月13日在[[史泰登島|史泰登島]]的家外被槍殺。<ref>{{Cite web |url = https://www.nydailynews.com/new-york/nyc-crime/ny-metro-staten-island-mob-boss-shooting-20190313-story.html |title = Gambino crime family boss Frank Cali shot and killed outside Staten Island home: sources |author = Kerry Burke, John Annese |date = 2019-03-13 |publisher = New York Daily News |language = en}}</ref>

== 早年 ==
法蘭克·卡利出生於[[紐約市|紐約市]]，<ref name=Repubblica>{{Cite web |url = http://www.repubblica.it/2008/01/sezioni/cronaca/operazione-palermo-ny/franky-boy-english/franky-boy-english.html |title = Franky Boy, the invisible boss
who wanted to have Palermo back |publisher = La Repubblica |language = en}}</ref>父母是Augusto和Agata Cesare，兩人都是[[西西里島|西西里島]][[帕勒莫|帕勒莫]]的當地人。他的父親在帕勒莫經營一間家居用品商店，還有在[[布魯克林|布魯克林]]的{{link-en|本森赫斯特|Bensonhurst, Brooklyn}}經營一間影視店。他在美國有乾淨的警察記錄，儘管他在{{link-en|披薩店販毒案|Pizza Connection Trial}}的調查中被提及到，當時警察發現他是與西西里黑手黨首領{{link-en|Gaetano Badalamenti|Gaetano Badalamenti}}結盟的Domenico Adamita的搭檔。<ref name=Repubblica/>

法蘭克·卡利是西西里黑幫{{link-en|約翰·甘比諾|John Gambino}}的侄子，與{{link-en|薩瓦托·因澤里洛|Salvatore Inzerillo}}領導的強大西西里黑手黨家族有密切的關係。卡利也是{{link-en|博納諾犯罪家族|Bonanno crime family}}黑幫{{link-en|Giovanni Bonventre|Giovanni Bonventre}}和{{link-en|Vito Bonventre|Vito Bonventre}}的外甥。

作為一名年輕人，卡利與甘比諾黑幫{{link-en|Jackie D'Amico|Jackie D'Amico}}關係密切，D'Amico是甘比諾老大[[約翰·高蒂|約翰·高蒂]]的中尉，高蒂在布魯克林的第18大道上操縱一群手下。1997年1月，[[聯邦調查局|聯邦調查局]]（FBI）向意大利當局報告說，卡利已經被「合併」到甘比諾家族。<ref name=Repubblica/>當D'Amico成為代理老大時，卡利被晉升為代理頭目（acting capo）。卡利在布魯克林經營了幾間進出口公司，包括位於布魯克林{{link-en|漢密爾頓堡|Fort Hamilton}}的馬戲團水果批發公司（Circus Fruits Wholesale）。<ref name=Repubblica/>

== 逝世 ==
卡利於2019年3月13日大約晚上9點20分在他的家門前發生暴力事件後被送往{{link-en|史泰登島大學醫院|Staten Island University Hospital}}，證實死亡，享年53歲。監視錄像顯示一輛輕型貨車撞上他停放的凱迪拉克車，隨後司機和卡利之間發生爭執，然後開槍。卡利試圖用他的車當作盾牌來躲避殺手。他被殺手用9mm手槍的子彈擊中了十次。<ref>{{Cite web |url = https://edition.cnn.com/2019/03/16/us/gambino-crime-family-killing-arrest/index.html |title = Reputed crime boss' killing doesn't appear to be mob-related, source says |author = Ralph Ellis, Brynn Gingras |date = 2019-03-17 |publisher = CNN |language = en}}</ref>

2019年3月16日，一名24歲男子在[[新澤西州|新澤西州]]的布里克鎮被[[紐約市警察局|紐約市警察局]]和[[美國法警|美國法警]]逮捕，並將在史泰登島面臨謀殺指控，他的輕型貨車在史泰登島被回收。<ref>{{Cite web |url = https://www.usatoday.com/story/news/nation/2019/03/16/frank-cali-killing-man-in-custody-after-gambino-crime-boss-shot-nypd/3186410002/ |title = NYPD: Man arrested in death of Frank Cali, reputed Gambino crime family boss |author = Stacey Barchenger, Anthony Zurita |date = 2019-03-16 |publisher = USA Today |language = en}}</ref>據報導槍手是24歲的Anthony Comello。<ref>{{Cite web |url = https://www.cbc.ca/news/world/gambino-mob-boss-shooting-arrest-1.5059821 |title = Arrest made in shooting death of reputed Gambino family mob boss |date = 2019-03-17 |publisher = CBC |language = en}}</ref>關於動機以及是否涉及其他方面的調查仍在繼續。
