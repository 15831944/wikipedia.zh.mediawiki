《'''法蘭德斯棋盤'''》（[[西班牙語|西班牙語]]：{{lang|es|''La tabla de Flandes''}}）是一部由[[西班牙人|西班牙]]作家[[阿圖洛·貝雷茲-雷維特|阿圖洛·貝雷茲-雷維特]]所著的小說，初版發行於1990年。小說結合歷史、懸疑、犯罪等元素，[[藝術|藝術]]和[[西洋棋|西洋棋]]是此書的兩大主題，故事圍繞一個隱藏於15世紀古畫中的祕密展開。[[中華民國|臺灣]][[麥田出版|麥田出版]]於1995年10月1日將小說首次譯介至華語世界，譯名《'''步步殺機'''》，由吳佳綺翻譯<ref>{{cite web |url=https://www.kingstone.com.tw/book/book_page.asp?kmcode=2018780003089 |title=步步殺機 |work=[[金石堂書店|金石堂書店]] |accessdate=2017-06-28 |language=zh-TW}}</ref>；漫遊者文化於2007年10月15日推出小說的新譯本《'''-{法}-蘭德斯棋盤'''》，由陳慧瑛翻譯<ref>{{cite web |url=http://www.books.com.tw/products/0010383437 |title=法蘭德斯棋盤 |work=[[博客來|博客來]] |accessdate=2017-06-28 |language=zh-TW}}</ref>。簡體中文版由重慶出版社首次發行於2006年，沿用吳佳綺譯本<ref>{{cite web |url=https://book.douban.com/subject/1909942/ |title=步步殺機 (簡體版) |work=[[豆瓣|豆瓣]] |accessdate=2017-06-28 |language=zh-CN}}</ref>；新版簡體本由南海出版公司發行於2014年7月1日，沿用陳慧瑛譯本，但將標題改為《'''佛蘭德斯棋盤'''》<ref>{{cite web |url=http://www.books.com.tw/products/CN11139871 |title=佛蘭德斯棋盤 (簡體書) |work=[[博客來|博客來]] |accessdate=2017-06-28 |language=zh-TW}}</ref>。

== 內容概述 ==
胡莉雅是一名居住在[[馬德里|馬德里]]的{{tsl|en|Restoration (cultural heritage)|藝術品修復|古畫修復專家}}，她受人所託修復[[早期尼德蘭繪畫|法蘭德斯畫派]]畫師彼得·范·赫斯的畫作《對弈》，這幅畫曾被用於小說的封面<ref>{{cite web |url=http://www.perezreverte.com/libro/144/la-tabla-de-flandes/ |title=La tabla de Flandes |work=[[阿圖洛·貝雷茲-雷維特|阿圖洛]]的個人官方網站 |accessdate=2017-06-28 |language=es}}</ref>。她發現畫作顏料底層用[[拉丁文|拉丁文]]寫着「誰殺了騎士？」（{{lang|la|Quis necavit equitem?}}），胡莉雅在與她情同父女的[[男同性戀者|同性戀]]古董商西瑟和{{tsl|en|Chess master|西洋棋師}}穆紐茲的幫助下探查這起延續500年的謀殺案之謎。隨着探查的深入，胡莉雅發現自己被捲入不明危機，而協助她的朋友也被一一謀殺。
