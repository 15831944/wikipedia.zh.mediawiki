'''大区'''（{{lang-fr|région}}）是[[法国行政区划|法国行政区划]]的[[一级行政区|第一级单位]]，下分为[[省_(法国)|省]]。法国共有18个大区，其中13个位于法国本土（其中[[科西嘉|科西嘉]]地位较特殊，称为“领土集体”''collectivité territoriale''），其余5个则位于海外。

一些[[法语系国家|法语系国家]]，如[[马里|马里]]，一级行政区也称作'''大区'''（{{lang-fr|région}}）。

== 作用 ==
法国是一个[[单一制|单一制]]国家，大区并不是一个[[法律|法律]]或规章意义上的自治体。不过，它从政府接受一部分国家[[税收|税收]]，拥有相应的[[预算|预算]]，他的职责就是将这些资源分配使用到不同的领域。

大区最主要的功用是在建设和财政上支持法国的[[高级中学|高级中学]]。除了这个法律职责之外，它的预算还参与到社会各个领域中，如[[基础设施|基础设施]]、[[教育|教育]]、[[公共交通|公共交通]]、[[科学研究|科学研究]]、[[企业|企业]]援助等。

在法国，一直以来对于是否授予大区以法律[[自治|自治权]]存在着激烈的争议；也有提议将[[省委员会_(法国)|省议会]]取消，而将它们并入[[大区委员会_(法国)|大区议会]]，并将[[省_(法国)|省]]作为大区的附属行政级别。不过目前并无后文。

== 行政 ==
自从1982年颁布第一个[[权力分散|地方分权法案]]起，大区拥有行政区域的地位，并且各由一个每6年直接[[普选权|普选]]产生的[[大区委员会_(法国)|大区议会]]管理。

[[科西嘉|科西嘉]]大区由地方[[议会|议会]]管理。

每一个大区同样有一个[[政府|政府]]任命的（大区的）主席，他的职责是在各省中代表政府。

== 历史 ==
在1789年[[法国大革命|法国大革命]]之前，[[法兰西|法兰西]][[王国|王国]]曾经在封建[[领地|领地]]的基础上分为各个[[行省_(法国)|行省]]，其中不少大小大致相当于现在的大区。1789年，这些行省被撤销，取而代之为83个省。

1917年8月25日发布了一份由大区理论所启发的商业部行政通告，随后于1919年4月5日，部门签发了法令，设立一些“区域经济联合体”，亦称“克雷芒（大）区”（région Clémentel），其中第一个是东部区域（南希），它包含了现在的[[洛林|洛林]]和[[香槟|香槟]]。这些“经济区”在自愿的前提下，于法国本土集合了各个商会，它们能够自由地依附于自己所选的区域，也可以自由地变更选择，因此最初设计的17个大区很快就成为了21个。

1919年9月，旅游事业联合会基于这个模式构造了19个“旅游区域”，划分主要基于[[地理|地理]]、[[民族|民族]]、[[历史|历史]]和[[旅游|旅游]]的标准，并不限于各省的界限。

伴随着这些运动，最早在1915年，随后于1920年（Hennessy法案）和1921年（Charles Rebel法案，Millerand-Marraud-Doumer法律草案）分别提出了行政权力下放的议案，也包括了建立大区宪法和成立大区议会的想法。但这些项目都没有付诸实施。

自第一次世界大战始，交通的发展、城市构造的转变以及区域理念的强化促使更多的人思考成立比省更大级别的行政区划的可能性。讽刺的是，这个[[法兰西第三共和国|第三共和国]]的想法在某种程度上于[[维希政府|维希政府]]治下实现过，当时[[贝当|贝当元帅]]签署了1941年4月19日法，将一些省重新组织成了（法国历史上的）[[行省_(法国)|行省]]。这种组织形式在1945年贝当政权倒台后就废止了。

[[戴高乐|戴高乐]]将军解放法國後于1944年1月10日发布命令，着手解放后法國领土的行政组织。他建立了一些行政区域，但在1946年他下台之后被放弃。

[[法兰西第四共和国|第四共和]]治下的领土布置同样考虑到建立超越省级的行政组织。1956年10月28日颁布了一项法令，规定了22个“计划中的大区”，其界限由计划的行政总专员让·韦尔若（Jean Vergeot）划定。这些大区仅为行政公器，不作它用。尽管他们的划定完全基于技术上的考量，但还是在很多区域与旧行省吻合。

戴高乐将军在1958年重新上台執政後，1964年，大区主席们在“区域行为的界限”框架下被任命。1969年，一项旨在扩大大区权限的[[公投|全民公决]]的失败导致了戴高乐再次退位。这次拒绝或许解释了为何创建大区委员会的1972年7月5日法律颁布的时候，对于大区的权限施加了如此多的限制，赋予如此少的权力。然而它们被正式命名为了“大区”（région），这个称谓为后来的1982年《权力下放法案》所用。

1982年3月2日法律规定了大区议会的选举为在省的范围内，以六年为任期的直接普选。第一次选举于1986年3月16日举行。大区成为了和'''省'''及'''市镇'''同一个框架下的行政区域。

最后，法国政府于2004年3月推出一项备受争议的计划，旨在将一部分非教育领域的成员转移至大区一级。反对意见指出，大区并不具有和这一功能相匹配的预算，并且这样的措施会加剧区域间的不平等。

目前的区域划分是源于1950年的法国行政区域整合，及更早的一些计划。它一直处于争论之中。比如对于[[大西洋卢瓦尔省|大西洋卢瓦尔省]]，一直有很强烈的呼声将其并入[[布列塔尼|布列塔尼]]大区；将历史上的[[諾曼底|諾曼底]]拆分为两个大区（[[上诺曼底|上诺曼底]]和[[下诺曼底|下诺曼底]]）也同样具有疑义，不少人鼓吹将它们合并。

2014年法国总统奥朗德推出大区合并计划，本土22个大区将合并缩减为13个大区，获得议会通过。

2016年1月1日，法國實施將本土現有22個大區合併為13個。

== 2016年行政区划改革 ==
{{France Regions Labelled Map}}

2016年1月1日，法国将本土现有22个大区合并为13个。（以下各大區參右圖代號）

其调整为：

; 整体保留
* [[法兰西岛|法兰西岛]](12)
* [[中央-盧瓦爾河谷大區|中央-羅亞爾河谷]](7)
* [[卢瓦尔河地区大区|卢瓦尔河地区]](18)
* [[布列塔尼大區|布列塔尼]](6)
* [[普罗旺斯-阿尔卑斯-蓝色海岸|普罗旺斯-阿尔卑斯-蓝色海岸]](21)
* [[科西嘉领土集体|科西嘉]](9)

; 两区合并
* [[北部-加来海峡|北部-加来海峡]](17) + [[皮卡第|皮卡第]](19) = [[上法蘭西大區|上法蘭西]]
* [[上诺曼底|上]](11)、 [[下诺曼底|下]][[诺曼底|诺曼底]](4) = [[諾曼第大區|諾曼第]]
* [[勃艮第|勃艮第]](5) + [[弗朗什-孔泰|弗朗什-孔泰]](10) = [[勃艮第-弗朗什-孔泰大區|布根地-法蘭琪-康堤]]
* [[罗讷-阿尔卑斯|罗讷-阿尔卑斯]](22) + [[奥弗涅|奥弗涅]](3) = [[奧弗涅-羅納-阿爾卑斯大區|奧文尼-隆-阿爾卑斯]]
* [[南部-比利牛斯|南部-比利牛斯]](16) + [[朗格多克-鲁西永|朗格多克-鲁西永]](13) = [[奧克西塔尼大區|奧克西塔尼]]

; 三区合并
* [[香槟-阿登|香槟-阿登]](8) + [[洛林|洛林]](15) + [[阿尔萨斯|阿尔萨斯]](1) = [[大東部大區|大東部大區]]
* [[普瓦图-夏朗德|普瓦图-夏朗德]](20) + [[利穆赞|利穆赞]](14) + [[阿基坦|阿基坦]](2) = [[新阿基坦大區|新阿基坦大區]]
