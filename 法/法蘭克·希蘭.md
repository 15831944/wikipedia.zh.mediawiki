'''法蘭克·「愛爾蘭人」·希蘭'''（{{lang-en|'''Frank "The Irishman" Sheeran'''}}，1920年10月25日 - 2003年12月14日）是一名勞工工會官員，被指控與{{link-en|布法利諾犯罪家族|Bufalino crime family}}有聯繫。

希蘭的身份是{{link-en|貨車司機國際兄弟會|International Brotherhood of Teamsters}}的高級官員，是在工會腐敗中受到有組織犯罪集團影響的主要人物。在他去世前不久，希蘭也承認殺死貨車司機的領導人[[吉米·霍法|吉米·霍法]]。作者Charles Brandt在非虛構著作《聽說你刷房子了》（''I Heard You Paint Houses''，2004年）中詳述了希蘭告訴他關於他聲稱謀殺霍法的事件。<ref name=paint>{{cite book

== 早年 ==
法蘭克·希蘭於1920年出生在[[新澤西州|新澤西州]]的[[肯頓_(新澤西州)|肯頓]]，父親是來自[[費城|費城]]的房屋油漆工Thomas Francis Sheeran Jr.，母親是Mary Agnes Hanson。他長大於[[賓夕凡尼亞州|賓夕凡尼亞州]]的{{link-en|達比 (賓夕凡尼亞州)|Darby, Pennsylvania|達比}}<ref>{{Cite book | last = English | first = T. J. | title = ''Paddy whacked: the untold story of the Irish American gangster'' | publisher = HarperCollins | year = 2005 | page = 438 | url = https://books.google.com/books?id=uvDN9yBVBF8C&lpg=PA438&ots=FfnoXLHf1g&dq=%22frank%20sheeran%22%20darby&pg=PA438#v=onepage&q&f=false | isbn = 978-0-06-059002-4}}</ref>，費城郊外的一個小型工人階級自治市。他的家人是[[愛爾蘭裔美國人|愛爾蘭]]和{{link-en|瑞典裔美國人|Swedish Americans|瑞典}}血統。在[[第二次世界大戰|第二次世界大戰]]期間於軍隊中服役時，他成長到成年人身高有6尺4寸（1.93米）。

== 第二次世界大戰 ==
法蘭克·希蘭在1941年8月入伍[[美國陸軍|美國陸軍]]，在[[密西西比州|密西西比州]][[比洛克西_(密西西比州)|比洛克西]]附近進行[[新兵訓練|新兵訓練]]，並被指派到[[憲兵|憲兵]]。在[[珍珠港事件|偷襲珍珠港事件]]後，他自願在[[喬治亞州|喬治亞州]][[班寧堡|班寧堡]]的空降兵軍隊中參加訓練。在他的肩膀脫臼後，他被轉移到被稱為「雷鳥」（The Thunderbirds）的{{link-en|第45步兵師|45th Infantry Division (United States)}}。<ref>Brandt (2004), page 39.</ref>1943年7月14日，希蘭起航前住北非。

=== 戰鬥職責 ===
希蘭服役411天的戰鬥職責。<ref>Brandt (2004), page 38.</ref>他的第一個戰鬥經驗是在[[意大利攻防戰_(二戰)|意大利攻防戰]]期間；包括[[西西里島戰役|西西里島戰役]]、[[入侵義大利#薩萊諾登陸|薩萊諾登陸]]、[[鵝卵石行動#戰鬥過程|安齊奧戰役]]和[[卡西諾戰役|卡西諾戰役]]。<ref>Brandt (2004), pages 38-50.</ref>然後他在[[龍騎兵行動|法國南部登陸]]、[[突出部之役|突出部之役]]和[[中歐會戰|德國入侵]]中服役。<ref name="Brandt 2004, page 51">Brandt (2004), page 51.</ref>

然而，希蘭補充道：「總而言之，我有50天在[[逃兵|擅離職守]]（AWOL）之下糊塗 - 沒有公假下而缺席，大多花費在喝[[紅葡萄酒|紅葡萄酒]]和追求意大利、法國和德國女人。但是，當我的人馬回到前線時我未有擅離職守。如果當你的小隊回到戰鬥而你是擅離職守的話，你可以繼續下去，因為你自己的軍官會殺掉你，他們甚至不必說是德國人殺的。」<ref>Brandt (2004), 40.</ref>

==== 戰爭犯罪 ====
希蘭後來回憶起他的戰爭服役，當時他第一次開始冷酷無情地取人性命。希蘭聲稱參與了許多屠殺和德國戰俘的即時處決，違反了[[海牙公約|1899年和1907年海牙公約]]以及{{link-en|日內瓦公約 (1929年)|Geneva Convention (1929)|1929年日內瓦戰俘公約}}。

=== 退伍 ===
列兵法蘭克·希蘭於1945年10月24日從美國陸軍中退伍。他後來回憶說，這是「我25歲生日前的一天。」

== 犯罪生涯 ==
他離開軍隊後，他成了一名貨車司機，但透過犯罪賺取額外的錢，包括受雇殺人。由於他在犯罪上的精明果斷，他成為[[美國黑手黨|黑手黨]]老大{{link-en|羅素·布法利諾|Russell Bufalino}}和{{link-en|安吉洛·布魯諾|Angelo Bruno}}的親密伙伴。布法利諾是{{link-en|布法利諾犯罪家族|Bufalino crime family}}的首領，在他的一生中擔任希蘭的導師。

== 卡車司機工會 ==
布法利諾向{{link-en|貨車司機國際兄弟會|International Brotherhood of Teamsters|國際}}會長[[吉米·霍法|吉米·霍法]]提供了希蘭。後來霍法成為一名親密的朋友，利用希蘭作為打手，包括暗殺不聽話的工會成員和威脅到貨車司機地盤的敵對工會成員。

== 逝世 ==
希蘭在2003年12月14日死於癌症，在[[費城|費城]]附近的[[養老院|養老院]]中去世。

== 有爭議的指控 ==
希蘭聲稱他參與了在[[豬灣事件|豬灣事件]]的反[[菲德爾·卡斯特羅|菲德爾·卡斯特羅]]部隊的供給協助，並且說他對[[甘迺迪遇刺案|美國總統甘迺迪遇刺案]]了如指掌。根據希蘭的說法，吉米·霍法希望甘迺迪去死，因為甘迺迪的弟弟、美國司法部長[[羅拔·法蘭西斯·甘迺迪|博比·甘迺迪]]不斷騷擾他。據希蘭說，暗殺甘迺迪總統是一次黑手黨的襲擊，他並沒有積極參與這個陰謀，但是經由{{link-en|大衛·菲利|David Ferrie}}向被指控的刺客運送三支步槍。

== 流行文化 ==
[[馬田·史高西斯|馬田·史高西斯]]執導了一部電影關於希蘭的一生和他涉嫌參與殺害吉米·霍法，片名為《[[愛爾蘭殺手|愛爾蘭殺手]]》。{{link-en|史蒂芬·澤里安|Steven Zaillian}}編劇，[[羅拔·迪尼路|羅拔·迪尼路]]飾演希蘭。其他演員有[[阿爾·柏仙奴|阿爾·柏仙奴]]、[[喬·佩西|喬·佩西]]和[[夏菲·基圖|夏菲·基圖]]。<ref>{{Cite web |url = http://www.imdb.com/title/tt1302006/ |title = The Irishman |publisher = IMDb |language = en}}</ref>本片預計在2019年上映。
