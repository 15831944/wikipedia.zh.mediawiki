'''法務部廉政署'''（簡稱'''廉政署'''）是2011年7月20日由[[中華民國法務部|中華民國法務部]]政風司改制轉編的肅貪廉政機關，構想源於[[香港|香港]][[廉政公署_(香港)|廉政公署]]及[[新加坡|新加坡]][[貪汙調查局|貪汙調查局]]，目的在防貪和反貪，在[[陳水扁|陳水扁]]與[[馬英九|馬英九]]兩位[[中華民國總統|中華民國總統]]的競選政見都有提出<ref>[http://www.moj.gov.tw/ct.asp?xItem=28316&ctNode=28004&mp=001 成立廉政署的構想？]</ref>。廉政署也是全國中央和地方機關政風機構的業務監督及指導機關，並基於「一條鞭」體系掌握全國政風機構人員之派免、調動、陞遷、獎懲及考績。

== 沿革 ==
*1987年7月，103位[[立法委員|立法委員]]共同連署，提案在[[行政院|行政院]]下成立「反貪污局」。
*2000年，法務部部長[[陳定南|陳定南]]上任後宣示推動成立「法務部廉政署」，修定《法務部組織法》及研擬《法務部廉政署組織條例》草案，但最終並未獲共識而緩議。
*2005年10月17日，第63次行政院政策協調會報。2005年10月26日，[[行政院會議|行政院第2963次院會]]通過相關組織法草案，送請[[立法院|立法院]]審議；惟至2007年12月21日立法院第6屆第6會期結束，未付委審查。因立法院屆期不續審，相關組織法草案行政院於2008年2月13日第3079次院會決議通過，函送立法院審議（第7屆）。
*2008年2月13日，行政院第3079次院會決議通過，函送第7屆立法院審議。2008年5月，法務部部長[[王清峰|王清峰]]成立專案小組，就原規劃之「法務部廉政局」重新研擬成立之必要。
*2009年5月22日，法務部提出〈我國設立廉政專責機關評估報告〉陳報行政院，建議規劃設立統籌防貪、肅貪、反貪權責之廉政專責機關。7月27日，「法務部組織調整規劃分組會議」決議，整併'''政風司'''與'''中部辦公室'''，成立「法務部廉政署」，並且納編各大小單位政風科/課/組/室。12月4日，法務部函請行政院准允撤回《法務部廉政局組織法》草案及《法務部組織法》第9條之1、第29條修正草案，經立法院在2010年1月8日「台立院議字第0990700163號函」同意撤回。
*2010年12月9日，《法務部廉政署組織法》草案於立法院第七屆第六會期[[立法院司法及法制委員會|司法及法制委員會]]第16次全體委員會議完成初審。
*2011年4月1日，立法院第七屆第七會期第七次會議逐條表決完成《法務部廉政署組織法》三讀立法程序。4月20日，總統[[馬英九|馬英九]]公布《法務部廉政署組織法》；7月20日，法務部廉政署正式成立。
*2012年1月，原公務員政風職系更名為廉政職系。
*2014年5月，揭發[[遠雄集團|遠雄集團]]董事長[[趙藤雄|趙藤雄]]、時任[[桃園縣政府|桃園縣副縣長]][[葉世文|葉世文]]涉[[合宜住宅弊案|合宜住宅弊案]]。
*2015年7月，揭發前[[新北市政府|新北市副市長]][[許志堅|許志堅]]涉樂揚建設及寶興開發分別於板橋及新店之都更弊案。<ref>[https://www.nextmag.com.tw/realtimenews/news/23490208 檢廉先鎖定朱立倫愛將 才槓上開花活抓葉世文]，壹週刊</ref>
*2015年10月，揭發[[內政部營建署|內政部營建署]]主任秘書[[洪嘉宏|洪嘉宏]]涉七件眷改案向遠雄等三家建商索賄1.2億元。<ref>[http://news.ltn.com.tw/news/society/breakingnews/1492542 遠雄去年標得4案 疑行賄洪嘉宏取得]，自由時報</ref><ref>[http://news.ltn.com.tw/news/politics/breakingnews/2578589 收受遠雄賄賂 營建署前主秘洪嘉宏被彈劾]，自由時報</ref>

*2017年7月，揭發[[台灣自來水公司|台灣自來水公司]]第6區管理處工務人員涉收賄及接受招待案。<ref>[http://news.ltn.com.tw/news/society/breakingnews/2606606 水公司六區處爆官商勾結弊案 廉政署搜查傳訊]，自由時報</ref>
*2017年11月，查扣[[高雄市政府#局|高雄市政府海洋局]]涉[[慶富造船|慶富造船]]興達港標案的錄音檔。<ref>[https://www.mirrormedia.mg/story/20171128inv018/?utm_source=facebook&utm_medium=mmpage 內鬼長期偷錄音　前海洋局長涉貪情節升高]，鏡傳媒</ref>
*2018年1月，揭發[[北埔鄉|北埔鄉]]長[[姜良明|姜良明]]與劉姓秘書涉圖利廠商案。<ref>[http://news.ltn.com.tw/news/society/breakingnews/2306854 北埔鄉長涉收賄 廉政署布線1年逮人]，自由時報</ref>
*2018年2月，揭發署內陳姓廉政專員涉索賄案。

== 爭議 ==
廉政署與[[法務部調查局|法務部調查局]]關於公務員涉犯貪瀆犯罪之偵查業務重疊，雖然法務部聲稱「交叉火網、分進合擊」<ref>{{Cite web|url=http://www.moj.gov.tw/ct.asp?xItem=259578&ctNode=27518|title=新聞發布|accessdate=2017-01-15|date=2008-05-01|last=法務部全球資訊網|work=www.moj.gov.tw}}</ref>，但在實際運行上，兩機關不時因辦案搶功、踩線起心結。<ref>[http://www.chinatimes.com/newspapers/20130323000382-260102 廉政署調查局互鬥 法部再打圓場]</ref>

== 宣導標語 ==
