'''法国蝸牛'''（{{lang-fr|'''Escargots'''}}，{{IPA-all|ɛs.kaʁ.ɡɔ}}），是一種可食用的蝸牛。一般會用來作[[法国菜|法国菜]]的[[頭盤|頭盤]]菜。

== 可食用的品種 ==
在世界上，並非所有的蝸牛都可以食用，不过其中还是有很多是可食的；即使在可食用的品種裡，不同品種的味道及口感各異，而其中有兩種品種有法國原生，成為了法國菜所用的蝸牛品種。这两种蜗牛中有一种[[散大蜗牛|散大蜗牛]]（''Helix aspersa''，{{lang-fr|Petit Gris}}），在全球的[[温带|温带]]气候地区都普遍繁殖。

== 烹調與食用 ==
由於蝸牛日常以泥土、[[腐殖質|腐殖質]]及各種不同品種的樹葉作食糧，蝸牛胃部裡的食物殘餘可能對人體有害。因此，在烹調之前，一定要先讓蝸牛禁食，讓蝸牛排出所有殘餘的食物，才可以把它們的消化系統切除，以準備食用。由於這項處理過程需時多天，一般急凍的蝸牛都已清理過。在美國及歐洲，亦有蝸牛農場專門培養烹調用的蝸牛。這些蝸牛的食糧都是研磨後的穀物，使食客可以放心食用。

== 營養價值==
法国蜗牛是高蛋白低脂肪的食物，大约含有15%蛋白质，2.4%脂肪以及大约80%的水分。<ref>[http://www.dietaryfiberfood.com/snail-escargot.php dietaryfiberfood.com]</ref>

== 做法 ==
[[File:Escargotbordeaux.jpg|thumb]]
一般來說，蝸牛在烹調前都已把它的殼和內臟移除。最常見的烹調方式是與[[大蒜|大蒜]]和[[黄油|黄油]]一起烹煮，然後把煮好的蝸牛肉和汁液放進殼裡。蝸牛一般都會用有小凹坑的碟來上菜，以防止蝸牛在碟上滾動。法国人也發明了一种专门吃蜗牛的[[叉子|叉子]]和[[钳子|钳子]]。鉗子的外貌像女性的[[睫毛鉗|睫毛鉗]]，而叉子則是一條很細的兩指叉。

除了用大蒜和黄油一起煮，亦有模仿海鮮的煮法，把蝸牛肉與[[马铃薯泥|薯蓉]]及[[烟肉|烟肉]]一起烤焗。
