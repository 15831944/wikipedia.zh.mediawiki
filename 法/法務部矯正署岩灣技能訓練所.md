'''法務部矯正署岩灣技能訓練所'''，是一座隸屬於[[中華民國法務部|中華民國法務部]][[矯正署|矯正署]]旗下的[[矯正機構|矯正機構]]，該機構坐落於[[臺灣省|臺灣省]][[臺東縣|臺東縣]][[臺東市|臺東市]][[卑南溪|卑南溪]]谷最南端之[[沖積平原|沖積平原]]上，當前收容人數（2018年10月18日數據）799名。<ref name=" 法務部矯正署岩灣技能訓練所">{{cite web |url=http://www.ywv.moj.gov.tw/mp071.html |title= 法務部矯正署岩灣技能訓練所 | accessdate=2015-02-11}}</ref>

=== 沿革 ===
* 岩灣技能訓練所最初是'''「臺灣省遊民習藝所」'''。後因[[228事件|228事件]]後涉案情節較輕者接訓人數過多，[[臺灣警備總司令部|臺灣警備總司令部]]於1947年（民國36年）6月下旬將臺灣省遊民習藝所改制為'''「東部勞動訓導營」'''。<ref name=" 中華民國後備憲兵論壇 臺灣警備總司令部 職業訓導處">{{cite web |url=http://mp.rocmp.org/kind/lagger/ |title= 中華民國後備憲兵論壇 臺灣警備總司令部 職業訓導處 | accessdate=2015-02-11}}</ref>
* 在1952年（民國41年）5月1日，[[臺灣省保安司令部|臺灣省保安司令部]]依據同年4月30日公佈的《臺灣省戒嚴時期取締流氓辦法》，將東部勞動訓導營改制並成立'''臺灣警備總司令部職業訓導第二總隊'''。<ref name=" 中華民國後備憲兵論壇 臺灣警備總司令部 職業訓導處"/>
* 在臺灣解除[[戒嚴|戒嚴]]後，中央政府配合[[動員戡亂時期|動員戡亂時期]]終止，裁撤臺灣警備總司令部；並於1992年（民國81年）7月1日奉[[行政院|行政院]]核定，將臺灣警備總司令部職業訓導第二總隊撥交給法務部接管，更名為'''「臺灣岩灣技能訓練所」'''。<ref name=" 中華民國後備憲兵論壇 臺灣警備總司令部 職業訓導處"/>
* 在法務部接管臺灣岩灣技能訓練所後，於1994年（民國83年）3月1日奉示附設'''「臺灣臺東監獄岩灣分監」'''。爾後又展開一連串的新建、改建工程，第一期工程於1995年（民國84年）6月30日竣工，1995年（民國84年）8月1日正式啟用收容受感訓處分人。 <ref name=" 法務部矯正署岩灣技能訓練所"/><ref name=" 中華民國後備憲兵論壇 臺灣警備總司令部 職業訓導處"/>
* 2009年（民國98年）1月23日，《[[檢肅流氓條例|檢肅流氓條例]]》廢止，岩灣技能訓練所釋放所有感訓處分人。<ref name=" 法務部矯正署岩灣技能訓練所"/><ref name=" 中華民國後備憲兵論壇 臺灣警備總司令部 職業訓導處"/>
* 2011年（民國100年）1月1日法務部矯正署成立，改隸為'''「法務部矯正署岩灣技能訓練所」'''。<ref name=" 法務部矯正署岩灣技能訓練所"/><ref name=" 中華民國後備憲兵論壇 臺灣警備總司令部 職業訓導處"/>

== 設施 ==
岩灣技能訓練所總佔地面積11.6761公頃，接收臺灣警備總司令部所遺老舊房舍後即積極籌劃改建，第一期新建、改建工程於1995年（民國84年）6月30日竣工。<ref name=" 法務部矯正署岩灣技能訓練所"/>

岩灣技能訓練所共劃分為行政辦公區、戒護區及技訓區三大部分，其中戒護區為注重高度安全性，乃突破傳統格局將戒護區建築採用"◇型"設計。<ref name=" 法務部矯正署岩灣技能訓練所"/>

== 人員配置 ==
當前岩灣技能訓練所員額編制，截止106年12月8日為止共158名，其中共分為職員151人、駕駛3人、技工1人、工友3人。<ref name=" 法務部矯正署岩灣技能訓練所"/>

另設「所務委員會」，由所長、秘書及各科室主管人員組成，目的在於處理有關收容人之處遇、假釋、聲請免除感訓處分或停止執行等重大事項。<ref name=" 法務部矯正署岩灣技能訓練所"/>

== 歷史事件 ==

{{main|岩灣事件}}

又稱'''岩灣監獄暴動事件'''，'''岩灣絕食事件'''，於1987年（民國76年）11月29日於台東岩灣監獄爆發的監獄暴動。在[[一清專案|一清專案]]後，台灣許多[[幫派|幫派]]份子及[[角頭|角頭]]被送到岩灣監獄監禁。在警備總部接管監獄管理後，因管理不當，數名囚犯發起絕食抗議，警備總部強勢介入，造成囚犯暴動。台灣政府對外宣稱，是因為取締監獄內賭博活動才造起這起事件。
