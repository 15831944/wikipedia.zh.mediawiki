《'''法医秦明'''》改编自[[秦明_(法医)|秦明]]创作的同名系列小说《[[法医秦明_(小说)|法医秦明]]》之第十一根手指和清道夫。2016年4月11日开机拍摄<ref>{{Cite web |url=http://ent.ifeng.com/a/20160413/42604603_0.shtml |title=《法医秦明》开机 李现张若昀首次合作成生死拍档|accessdate=2017-05-09 |author= |date=2016-04-13 |work=责任编辑：莫海琼 PK034 |publisher=凤凰网|language=Zh-hans }}</ref>。拍摄地在[[辽宁省|辽宁省]][[大连市|大连市]]<ref name="a1">{{Cite web |url=http://ln.qq.com/a/20160529/006928.htm |title=《法医秦明》旅顺热拍 |accessdate=2017-05-09 |author=记者：张明春 |date=2016-05-29 |work= |publisher=腾讯网，来源：大连晚报|language=Zh-hans }}</ref>。10月13日起，以每周四播出两集的方式在[[搜狐网|搜狐网]]播出<ref name="a2">{{Cite web |url=http://yule.sohu.com/20161013/n470162094.shtml |title=《法医秦明》首播 钱波徐昂再度合作默契足 |accessdate=2017-05-09 |author= |date=2016-10-13 |work= |publisher=搜狐网|language=Zh-hans }}</ref>。

此剧以法医[[尸检|尸检]]、刑侦破案为题材，出现各类尸体的“重品味”画面，因此有“减肥利器”、“影响食欲”的评价，小说原作者秦明则回应：“其实当看惯了恶心的东西，任何可以吃的都是美食了。这就是我发胖的原因。”<ref>{{Cite web |url=http://mt.sohu.com/20161103/n472162567.shtml |title=据说《法医秦明》是减肥利器？ |accessdate=2017-05-10 |author= |date=2016-11-03 |work= |publisher=|language=Zh-hans }}</ref>

截止至2016年12月，该剧网络播放量已破12亿，成为2016年度热播现象级网剧<ref>[http://www.chinanews.com/yl/2016/12-10/8089948.shtml 《法医秦明》点击破12亿 张若昀：每天会刷10遍]</ref>。
