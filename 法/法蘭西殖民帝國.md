'''法蘭西殖民帝國'''是指[[法國|法國]]在17世紀至1960年代控制的諸多[[殖民地|殖民地]]。在19至20世紀初，它成為了僅次[[大英帝國|大英帝國]]的第二大[[殖民帝國|殖民帝國]]。在1919至1939年这个巅峰时期，它横跨了1234.7万平方公里的土地。如果把法国本土也计算在内的话，它的面积就达到1289.8万平方公里，占世界土地总面积的8.6%。這些殖民地在20世紀陸續獨立後，到了今天只保留了分佈在北[[大西洋|大西洋]]、[[加勒比海|加勒比海]]、[[印度洋|印度洋]]、南[[太平洋|太平洋]]及北太平洋中百多個不同的[[島嶼|島嶼]]及[[群島|群島]]，以及[[法属南方和南极洲领地|法属南方和南极洲领地]]，總面積只有12.315万平方公里，只是1939年所擁有土地面積的1%，其總人口只有154,300人。

== 法蘭西第一殖民帝國 ==
在16世紀，隨著航海家[[乔万尼·达·韦拉扎诺|韦拉扎诺]]及[[卡蒂亚|卡蒂亚]]開拓了航線後，法國漁民便開始在[[紐芬蘭|紐芬蘭]]一帶航行，這樣就揭開了法蘭西殖民地擴張的序幕。但[[西班牙|西班牙]]在[[中南美洲|中南美洲]]一帶的壟斷，以及法國在16世紀晚期[[法国宗教战争|法国宗教战争]]中失利所造成的內亂，都使得法國有必要建立殖民地帝國。法國在早期曾經企圖以[[巴西|巴西]]作為殖民地，並在1555年及1612年入侵[[里約熱內盧|里約熱內盧]]及[[聖路易|聖路易]]兩地，但由於[[葡萄牙|葡萄牙]]和西班牙的對法國的戒備及阻止，使得法國在兩地及北美[[佛羅里達|佛羅里達]]的侵略均告失敗。

在1605年7月27日，當法國在[[新斯科舍|新斯科舍]]建立起[[皇家港|皇家港]]開始，就揭開了法蘭西第一殖民帝國的序幕。數年後，法國探險家[[山姆·德·尚普蘭|山姆·德·尚普蘭]]建立起[[魁北克城|魁北克城]]，而它亦成為了一個以皮毛貿易為主的殖民地，稱為[[新法蘭西|新法蘭西]]（或稱為[[加拿大|加拿大]]）。其後，法國更把其勢力擴展至[[圣劳伦斯河|圣劳伦斯河]]河谷一帶。隨著勢力的擴展，法國都把這些新的疆土作為貿易之用。但法國在本土發展上有更大的野心，而對殖民地的發展的興趣則不大，因此後來的加拿大發展並不如英國在北美的發展，更在1713年的[[烏得勒支和約|烏得勒支和約]]中失去了[[阿卡迪亞|阿卡迪亞]]的土地。但法國在1699年聲稱自己在北美的領土不斷的深入擴大，遠至北美的[[路易斯安娜|路易斯安娜]]及[[密西西比河|密西西比河]]。此外，它更在[[美洲|美洲]]從事[[非洲|非洲]][[奴隸|奴隸]]的交易。

法國的殖民地擴張不只在[[新大陸|新大陸]]進行，更在其他地方發展。如1624年法國就在[[西非|西非]]的[[塞內加爾|塞內加爾]]岸邊建立第一個貿易站。此外，[[法國東印度公司|法國東印度公司]]亦在東方建立起貿易商旅，如[[孟加拉|孟加拉]]的[[金德訥格爾|金德訥格爾]]、[[印度|印度]]的[[本地治里|本地治里]]等。

== 英法殖民地衝突（1744年至1815年） ==
18世紀中期，一連串的殖民地衝突在法國及[[大不列顛王國|英國]]之間展開，最後更險致法蘭西殖民帝國的瓦解。這一連串的衝突就是從[[奧地利王位繼承戰爭|奧地利王位繼承戰爭]]（1744年-1748年）、[[七年戰爭|七年戰爭]]（1756年-1763年）、[[美國獨立戰爭|美國獨立戰爭]]（1778年-1783年）、[[法國大革命|法國大革命]]（1793年-1802年）及[[拿破崙戰爭|拿破崙戰爭]]（1802年-1815年）中衍生出來。

雖然法國在奧地利王位繼承戰爭失利，被迫割讓在印度的殖民地，並在七年戰爭中失去了新大陸的大部份殖民地，但它在美國獨立戰爭期間所作出的調停卻使它可以收復一些在美洲的殖民地。此外，當1789年法國大革命爆發後，[[拿破崙|拿破崙]]便逐步奪取政權，最終成為[[法蘭西第一帝國|法蘭西第一帝國]]皇帝。這使得法國開始走上了對外侵略之路，並開始建立殖民地。

== 法蘭西第二殖民帝國 ==
拿破崙戰爭結束後，很多法國殖民地都被劃為[[英國|英國]]所有，特別是[[南美洲|南美洲]]北岸的[[法屬圭亞那|法屬圭亞那]]，以及在西非塞內加爾所建立的貿易站。因此，法國需要再次擴張殖民地，因而揭開[[法蘭西第二殖民帝國|法蘭西第二殖民帝國]]的序幕。

法蘭西第二殖民帝國的建立，始於1830年法國對[[阿爾及利亞|阿爾及利亞]]的侵略。雖然法國在法皇[[拿破崙三世|拿破崙三世]]在位期間，曾出兵入侵[[墨西哥|墨西哥]]，企圖對墨西哥以保護國之名進行殖民統治，但最後失敗落空，使得法國被迫放棄了對墨西哥的企圖。但法國在亞洲的[[印支半島|印支半島]]殖民計劃較為順利，先佔領越南南部[[交趾支那|交趾支那]]，以及取得柬埔寨的保護權。

[[普法戰爭|普法戰爭]]後，法國[[第三共和国|第三共和国]]取代[[拿破崙三世|拿破崙三世]]的統治，它成功的將法國的殖民帝國大大擴大，主要集中在非洲的擴張。在[[北非|北非]]，以早已成為法國殖民地的阿爾及利亞為基礎，向東把[[突尼西亞|突尼西亞]]、[[利比亞|利比亞]]一帶建立勢力範圍，後來更將[[摩洛哥|摩洛哥]]變為自己的保護國。而在[[中非|中非]]及[[西非|西非]]，法國逐漸把[[茅利塔尼亞|茅利塔尼亞]]、塞內加爾、[[几內亚|几內亚]]、[[馬里|馬里]]、[[科特迪瓦|科特迪瓦]]、[[貝南|貝南]]、[[尼日尔|尼日尔]]、[[查德|查德]]、[[中非共和国|中非共和国]]及[[刚果共和国|刚果共和国]]等地納入自己的控制下，在東非則佔據[[吉布提|吉布提]]一帶[[东非|东非]]的狹長海岸。在東南亞，中法战争後，法國在印支半島的殖民統治進一步得到確保，將整個[[越南|越南]]、[[柬埔寨|柬埔寨]]及[[老撾|老撾]]變成法國殖民地，建立起[[法屬印度支那|法屬印度支那]]，更向中國租借[[廣州灣|廣州灣]]。與此同時，法國更在南太平洋建立起殖民地，包括[[新喀里多尼亞|新喀里多尼亞]]、[[法屬玻里尼西亞|法屬玻里尼西亞]]一帶的群島以及[[新海布里地群島|新海布里地群島]]。

這個龐大的殖民帝國在[[第一次世界大戰|第一次世界大戰]]後達到顛峰。它接收了前[[奧斯曼土耳其帝國|奧斯曼土耳其帝國]]的領土[[黎巴嫩|黎巴嫩]]及[[敘利亞|敘利亞]]，以及[[德意志帝國|德國]]擁有的[[喀麦隆|喀麦隆]]及[[多哥|多哥]]，使得帝國版圖再次增加，其後更在[[西非|西非]]建立了法屬西非，直至[[第二次世界大戰|第二次世界大戰]]結束。

== 帝國的瓦解 ==
第二次世界大戰期間，法蘭西殖民帝國逐漸分解，當中大部份都為他國所佔領（如[[日本|日本]]佔領[[印度支那|印度支那]]，英國佔據了[[敍利亞|敍利亞]]、[[黎巴嫩|黎巴嫩]]及[[馬達加斯加|馬達加斯加]]，英美聯合佔有[[摩洛哥|摩洛哥]]及阿爾及利亞，德國搶奪了突尼西亞）。但不久後，由[[戴高樂|戴高樂]]重新控制這些地方。

第二次世界大戰結束後，儘管法國是戰勝國，然而這無法掩飾法國國力衰退的事實，而在法國各殖民地興起的[[非殖民地化運動|非殖民地化運動]]浪潮影響，紛紛走向獨立建國之路，這使得法國的殖民帝國面臨嚴重挑戰，雖然法國試圖嘗試繼續維持殖民地的統治，但各地反抗獨立力量，已非法國所能對付，如[[越南|越南]]在[[胡志明|胡志明]]的領導下經過9年[[越南抗法战争|戰爭]]，擺脫法國的統治，其後經過20年[[越戰|越戰]]，最終建立[[越南社會主義共和國|越南社會主義共和國]]；而北非的[[阿爾及利亞|阿爾及利亞]]在經過8年的反抗後，取得獨立，法国也在二战结束后放弃其在中国的租借地和治外法权。戴高樂1958年再次上台後，容許通過全民公決實現殖民地獨立，受到非殖民地化運動影響的法屬非洲也有很多從法國殖民地中獨立，而這些殖民地也得以立於萬國之林。至1980年，法国主要的殖民地全部脱离法国独立。

直至今天，法國只剩下在南太平洋的一些海外屬地，[[法屬圭亞那|法屬圭亞那]]以及一部份島嶼。
