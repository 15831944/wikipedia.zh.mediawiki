{{法国文化}}

[[File:Eugène_Delacroix_-_La_liberté_guidant_le_peuple.jpg|thumb]][[歐仁·德拉克羅瓦|歐仁·德拉克羅瓦]]為紀念1830年法國[[七月革命|七月革命]]的經典作品《[[自由引導人民|自由引導人民]]》]]

'''法國文化'''是指[[法國|法國]]與[[法國人|法國人]]創造的[[文化|文化]]。法國是一個充滿文化、[[藝術|藝術]]的國家，而在法國文化中，他們的[[文學|文學]]、[[電影|電影]]、[[繪畫|繪畫]]、[[建築|建築]]等都使人再三回味。幾百年來，法國都一直是[[歐洲|歐洲]]乃至全世界的文化中心之一。

法國對文化嚴肅以待，作家、藝術家、服飾設計師以及知識份子等，在法國社會上都有崇高的地位，法國政府於1960年代撥出了巨款以成立由地方文化中心的龐大[[網路|網路]]。[[實驗|實驗]]性藝術和設計也往往可得到補助。法國人對自己的電影傳統為榮，堅決不屈服於[[好萊塢|好萊塢]]之下，在其他方面，音樂界都是以法語為主，以保護原有的主義氣息，而[[紅磨坊|紅磨坊]]亦從1889年起成為當地文化一部分，並以[[康康舞|康康舞]]著名。前衛的藝術、文學及現代建築在法國都得到強烈的支持。

==文學==
{{main|法國文學}}

最早的法语文学可以追溯到11世纪。作者不明的《[[罗兰之歌|罗兰之歌]]》便是法国最早的[[史诗|史诗]]。形式相似的史诗还包括[[英格兰|英格兰]]的《[[贝奥武夫|贝奥武夫]]》、日尔曼人的《[[尼伯龙根之歌|尼伯龙根之歌]]》等。这些史诗多半歌颂伟大的君主。比如《罗兰之歌》便是歌颂著名的法兰克国王[[查理曼|查理曼]]（742年－814年）的。此外中世纪法国还盛行[[骑士传奇|骑士传奇]]，这些传奇无不歌颂骑士精神，弘扬典雅的爱情。这一时期的法国最重要的作家是十二世纪的[[克里蒂安·德·特鲁瓦|克里蒂安·德·特鲁瓦]]（Chretien de Troyes），他是法国中世纪最著名的叙事诗人。此外，在南部[[普罗旺斯|普罗旺斯]]地区大行其道的行吟诗也多半以骑士与贵妇之间的典雅爱情为主题，这些行吟诗多半具有丰富的想象力，风格却矫饰造作，文学价值不高。直到中世纪快要结束的时候，法国才出现了第一位真正的伟大抒情诗人[[維永|弗朗索瓦·维永]]（1431年－1465年）。

[[File:Jean-Jacques_Rousseau_(painted_portrait).jpg|thumb]]

18世纪上半期的[[皮埃尔·德马里沃|皮埃尔·德马里沃]]（1688年－1763年）和18世纪下半期的[[博马舍|比埃尔·博马舍]]（1732年－1799年）继承了古典主义喜剧创作的传统。此外仍有一些作家从事着和理念宣传无关的纯文学创作，出现了一些[[小说|小说]]，包括[[阿兰·勒内·勒萨日|阿兰·勒内·勒萨日]]（1668年－1747年）的《吉尔·布拉斯》和[[拉比·普莱伏|拉比·普莱伏]]（1697年－1743年）的《曼侬·雷斯考》。

18世纪法国最重要的作家是生于瑞士的[[让-雅克·卢梭|让-雅克·卢梭]]（1712年－1778年）。他的作品宣扬人类天然的美德和本能的合理性，攻击社会文明的腐朽。他渴望生存在一个崇尚知识的世界，他本人是个教育的革新者、革命思想的启蒙者。在文学上，他则被认为是浪漫主义的先驱。在整个十八世纪，卢梭的影响力超过其他任何一位文学家。

[[浪漫主义|浪漫主义]]对诗体悲剧的捍卫揭开了这场运动的真正帷幕。[[维克多·雨果|维克多·雨果]]的《[[欧那尼|欧那尼]]》於1830年上映象征着浪漫主义戏剧的最终胜利。布瓦洛所制定的“三一律”被正式废除，诗体悲剧的韵律限制也得到了解放。雨果（1802年－1885年）是浪漫主义运动中最杰出的天才和当之无愧的精神领袖。他一生著述丰富，在诗歌、戏剧、小说等领域均有很高建树，其抒情诗创作的地位被认为至高无上。

[[现实主义|现实主义]]承接浪漫主义而出现，秉承真实描绘现实生活的原则。现实主义的诞生在某种程度上是对浪漫主义矫枉过正的一种调和，但究其实质，却是科学技术不断发展和工业化进程不断加剧的产物。



现实主义出现以后，真实的描绘现实生活成了文学创作的主流。[[司汤达|司汤达]]（1783年－1842年）是法国现实主义的真正先驱，小说《[[红与黑|红与黑]]》至今仍有广泛的读者。[[巴尔扎克|巴尔扎克]]（1799年－1850年）的小说创作是法国现实主义文学最杰出的代表。他的《[[人间喜剧|人间喜剧]]》是一部丰伟的文学大厦，不亚于一部关于19世纪法国社会风俗的编年史。稍后的现实主义作家中，[[福楼拜|福楼拜]](1821年－1880年)的《[[包法利夫人|包法利夫人]]》和[[莫泊桑|莫泊桑]](1850年-1893年)的短篇小说也脍炙人口。除小说以外，小仲马和奥杰尔的戏剧创作也遵循着现实主义原则。

这种追求真实、科学的现实主义原则甚至也被运用到诗歌创作中，著名的“高蹈派”诗人团体就是其中杰出的代表，其中又尤以[[勒贡特·德·列尔|勒贡特·德·列尔]]（1818年－1894年）成就最高。

1920年代，法国出现[[超现实主义|超现实主义]]运动，[[保尔·艾吕雅|保尔·艾吕雅]]在这一领域做出了很多可贵的尝试。
1940年代[[存在主义|存在主义]]文学的代表人物[[保罗·萨特|保罗·萨特]]和[[阿尔贝·加缪|阿尔贝·加缪]]也许是20世纪最负盛名的法语作家。[[西蒙娜·德·波伏娃|西蒙娜·德·波伏娃]]则是女权主义文学的先驱之一。

1950年代法国成为[[新小说|新小说]]运动的策源地。当代法语文学产生了一大批成就斐然的大师，却鲜有哪个人能够独领风骚。法语文学在小说、戏剧、诗歌以及文学评论等各个领域都有所发展。姑且不论现代法语文学的成就及其发展趋向，至少到目前为止，法语文学的发展仍然是充满活力、永不疲倦的。

== 戲劇 ==
{{main|法國戲劇}}

法國三大古典[[戲劇|戲劇]]家，[[皮埃尔·高乃依|高乃伊]]、[[拉辛|拉辛]]及[[莫里哀|莫里哀]]都活躍於17世紀，莫里哀的[[喜劇|喜劇]]譏諷人性的弱點，而高乃伊、拉辛就擅長[[詩|詩]]體[[悲劇|悲劇]]。18世紀出現了兩位浪漫喜劇大師，[[博馬舍|博馬舍]]和[[馬里沃|馬里沃]]，博馬舍的「[[塞維爾理髮師|塞維爾理髮師]]」和「[[費加洛婚禮|費加洛婚禮]]」在後來都度改編為[[歌劇|歌劇]]。19世紀最膾炙人口的戲劇都出自[[維克多·雨果|維克多·雨果]]。20世紀的安奧爾專寫文雅哲學喜劇。

== 建築 ==
{{main|法國建築}}

法國保留了不少[[中世紀|中世紀]]時的建築，其中有仿[[羅馬|羅馬]]式的小教堂和哥德式的宏偉聖殿。法國撼動人心的建築物包括[[羅浮宮#金字塔|羅浮宮金字塔]]、[[凱旋門|凱旋門]]以及[[艾菲爾鐵塔|艾菲爾鐵塔]]等。

== 宗教 ==
{{main|法國宗教}}

法國主要宗教是[[天主教|天主教]]，其次是[[基督新教|基督新教]]、[[伊斯蘭教|伊斯蘭教]]、[[猶太教|猶太教]]和[[佛教|佛教]]。

根據CSA 2006-2007年度民意調查<ref>Sondage CSA ，Sondage publié en janvier 2007 dans Le Monde des religions</ref>，其比例如下：
* [[天主教|天主教]]：佔總人口51%。
* 無宗教：約占總人口31%。
* [[伊斯蘭教|伊斯蘭教]]：佔總人口約4%（包括所有[[穆斯林|穆斯林]]移民则可能高达全国人口的7%）。在法國監獄服刑的囚犯估計有約70%是穆斯林。<ref>{{Cite news |url=http://www.telegraph.co.uk/news/worldnews/europe/france/11352268/What-is-going-wrong-in-Frances-prisons.html |title=What is going wrong in France's prisons? |publisher=The Telegraph |date=2015-01-17 |quote=France's prison population is estimated to be 70 per cent Muslim}}</ref>
* [[基督新教|基督新教]]：佔總人口3%。
* [[猶太教|猶太教]]：佔總人口約1%。
* [[佛教|佛教]]：根據法國佛教徒協會，法國約有100萬佛教徒，其中主要是亞洲裔。法國是最多佛教徒的歐洲國家。

== 語言 ==
{{main|法语}}
法国官方[[語言|語言]]為[[法語|法語]]；然而，最近[[法国政府|法国政府]]開始鼓勵[[學校|學校]]和[[法国政府机构|政府機構]]使用一些[[法国方言|地方語言]]，例如：[[巴斯克語|巴斯克語]]、[[布列塔尼語|布列塔尼語]]、[[阿爾薩斯語|阿爾薩斯語]]<small>（屬[[德語|德語]]）</small>、[[弗拉芒語|弗拉芒語]]<small>（屬[[荷蘭語|荷蘭語]]）</small>、[[科西嘉語|科西嘉語]]，[[朗格多克語|朗格多克語]]， [[普羅旺斯語|普羅旺斯語]] 等；一些[[學校|學校]]開始教授當地[[方言|方言]]，但是[[法語|法語]]依然是全國官方語言。[[阿爾薩斯人|阿爾薩斯人]][[英語|英語]]正在成為法国第二[[語言|語言]]。{{Citation needed|time=2007-09-08T09:30:23Z}}[[布列塔尼人|布列塔尼人]]中的一些[[農村|農村]][[居民|居民]]以[[布列塔尼語|布列塔尼語]]為[[口語|口語]]。[[科西嘉人|科西嘉人]]日常生活中亦常用當地的兩種[[方言|方言]]：一種與[[義大利|義大利]][[托斯卡納|托斯卡納]][[方言|方言]]相近，另一種與[[撒丁島|撒丁島]]北部[[方言|方言]]相近。

==法國料理==
{{main|法國料理}}

[[图像:Foie_gras_et_sa_gelée_modified.jpg|thumb]]是法国西南部的传统名菜]]
[[File:Austriazismus_03.jpg|缩略图]]
法國料理是[[西方|西方]]世界最具影响和最具特色的[[烹饪|烹饪]]系统之一，[[法国|法国]][[君主|君主]]具有较强的王权，在[[路易十四|路易十四]]时达到顶峰，宫廷奢华风气在[[饮食|饮食]]上十分讲究，各种烹饪方法通过效仿的[[贵族|贵族]]流入民间。法国烹饪重视烹饪方法和就餐礼仪，法国盛产[[葡萄酒|葡萄酒]]（[[法语|法语]]：{{lang|fr|vin}}）和[[奶酪|奶酪]]（{{lang|fr|fromage}}），成为法国烹饪必不可少的[[调味料|调料]]，法国各地大约出产450多种不同风味的奶酪，每一种奶酪以最先发明其做法的[[村镇|村镇]]名命名，有的村镇如{{lang|fr|Camembert}}在任何普通[[地图|地图]]中都找不到，但在任何一本[[法语|法语]][[词典|词典]]甚至[[英语|英语]]词典中都能找到这个代表奶酪名称的词。

法国烹饪在各地区根据其特产也具有不同的特色，南方沿海多使用[[橄榄油|橄榄油]]，南方和西方沿海多[[水产|水产]]，北方和东方则多[[肉|肉食]]，总的说法国人不太喜欢[[素食|素食]]。

法国比较具有特色的食品有：[[青蛙腿|青蛙腿]]（{{lang|fr|grenouille}}）、炖鸡（{{lang|fr|coq au vin}}）、[[法国蜗牛|法国蜗牛]]（{{lang|fr|escargot}}）；主食主要是[[面包|面包]]，具有法国特色的面包有[[羊角面包|牛角包]]（{{lang|fr|croissant}}）和[[法国面包|棍式面包]]（{{lang|fr|baguette}}）。

== 其他 ==
法国[[香水|香水]]及[[化妆品|化妆品]]业举世闻名，它和法国[[时装|时装]]、法国[[葡萄酒|葡萄酒]]并列为法国三大精品产业，都是法国人的骄傲。
