'''法国新闻社'''（{{lang-fr|L'Agence France-Presse}}），简称'''法新社'''（{{lang|fr|AFP}}），是全球第一家[[通讯社|通讯社]]，同时也是[[法国|法国]]最大的通讯社，世界第三大通讯社，在[[美联社|美联社]]和[[路透社|路透社]]之后。

法新社总部设在[[巴黎|巴黎]]，在约110个国家设有办事处，它通过[[法语|法语]]、[[英语|英语]]、[[阿拉伯语|阿拉伯语]]、[[西班牙语|西班牙语]]、[[德语|德语]]和[[葡萄牙语|葡萄牙语]]向全世界发布消息。

== 历史 ==
法新社前身為1835年立的[[哈瓦斯社|哈瓦斯社]]，最初是在[[巴黎|巴黎]]承担[[广告|广告]]的翻译代理业务，当时只有[[查尔斯·路易斯·哈瓦斯|查尔斯·路易斯·哈瓦斯]]和他的两名员工[[保罗·路透|保罗·路透]]和[[貝納德·沃尔夫|貝納德·沃尔夫]]，后来两名员工都成了他的竞争对手，分别在[[伦敦|伦敦]]和[[柏林|柏林]]成立了[[路透社|路透社]]和[[沃尔夫社|沃尔夫社]]。为了减少间接费用和开发利润丰厚的广告方面的业务，哈瓦斯的儿子成功地于1852年分别和[[路透社|路透社]]和[[沃尔夫社|沃尔夫社]]签署协议，让每一个[[通讯社|通讯社]]有独家新闻。这种安排一直持续到1930年，当时的无线短波改善和降低了通讯费用。为了帮助哈瓦斯扩大范围，为了应对复杂的、紧张的国际局势，法国政府资助了高达47％的投资。1940年，当[[德国军队|德国军队]]占领法国时，更名为'''法国新闻办公室'''（FIO）；哈瓦斯只保留私营广告公司名称。

1944年8月20日，[[盟军|盟军]]占领巴黎，一群记者在FIO报道法国抵抗运动，并首次发表了盟军如何攻下每一座城市。法新社作为一个[[国有企业|国有企业]]，战后，法新社专门向[[发展中国家|发展中国家]]派遣记者，其中最先报道的西方人物是[[斯大林|斯大林]]。1953年3月6日，法新社渴望摆脱其半官方的地位。1957年1月10日，[[法国议会|法国议会]]通过了一项法律，确立法新社的独立性。从那时起，法新社来自政府部门的投资逐年下降。

== 目前状态 ==
[[File:Agence_France_Presse_à_Paris.jpg|thumb]]
[[File:AFP_Paris_dsc04592.jpg|thumb]]

目前，法新社拥有来自81个国家和地区的2000多名雇员，其中900人左右在国外工作，及覆盖全球165个国家和地区的110个办事处，它们分设在五个大区：
# 北美区（9个），总部[[华盛顿哥伦比亚特区|华盛顿]]；
# 拉美区（21个），总部[[蒙德维的亚|蒙德维的亚]]（[[乌拉圭|乌拉圭]]首都）；
# 亚太区（25个），总部[[香港|香港]]；
# 非欧区（52个），其中[[欧洲|欧洲]]36个，[[非洲|非洲]]16个，总部[[巴黎|巴黎]]；
# 中东区（9个），总部[[尼科西亚|尼科西亚]]（[[塞浦路斯|塞浦路斯]]首都）

法新社是一个由1957年特别法 ，特许授权的非營利性的自治公共组织。它是商业化的操作，并且独立于法国政府，它由一位首席执行官和15位董事组成的董事会管理。
* 8位法国出版社的代表，
* 2位法新社的代表，
* 2位广播和电视公共服务部门的代表，
* 3位来自向通讯社捐助的公共服务部门的代表（一位是由总理指定的，另一位是由财政部长，还有一位是外交部长）。

董事会三年选举一次首席执行官。

法新社还有一个上级的部门管理，以确保通讯社的是按照它的法令运作的，它的命令是独立和中立的。

法新社的主要客户是法国政府，为法新社的各种服务提供捐赠。实际上，这些捐赠相当多地做为法新社的补助金，法令规定：法新社禁止接受直接的政府补助。

== 亞太業務 ==
=== 中國大陸 ===
法新社在[[中國大陸|中國大陸]]的業務，按照中國媒體的習慣，分為文字和新聞圖片兩部分。文字部分由法新社駐中國[[北京|北京]]的辦事處和[[香港|香港]]的亞太區辦事處負責經營。其中國區客戶主管為吳紹華先生。

法新社的新聞圖片部分，長期交由中國國家通訊社[[新華社|新華社]]代理銷售，並從中收取代理費用。新华社于2006年也规定，所有外国通讯社在中国大陆销售新闻，必须<!--“应当”，但不这样做是“不得”-->由新华社批准及按照新华社指定选择代理机构，否则违法。<ref>[http://news.sina.com.cn/c/2006-09-11/12259990453s.shtml 外国通讯社在中国境内发布新闻应经新华社批准并指定机构代理]，新浪网</ref>

與此同時，法新社將其新聞圖片銷售給雜誌、期刊、出版類媒體用戶的權益交給中國[[上海|上海]]一家圖片公司“東方ic”（又名[[中國圖片庫|中國圖片庫]]、ImagineChina、icpress、ic傳媒等）代理。該公司掛名於[[中共上海市委|中共上海市委]]宣傳部、[[上海市人民政府|上海市人民政府]]新聞辦公室實際擁有的政府網站《[[東方網|東方網]]》之下，主要經營國際通訊社在中國大陸的新聞圖片代理銷售業務，如[[美聯社|美聯社]]、[[歐洲新聞圖片社|歐洲新聞圖片社]]（EPA）、[[瑪格南圖片社|瑪格南圖片社]]等均由該公司負責銷售。

=== 台灣及香港 ===
法新社網際網路中文版新聞經法新社授權由[[中央通訊社|中央通訊社]]獨家代銷。目前[[Google新聞|Google新聞]]（包括中國版）、[[MSN|MSN]]台灣、[[雅虎香港|雅虎香港]]和[[雅虎奇摩|雅虎奇摩]]新聞中的法新社中文新聞均由中央社提供。

香港《[[南華早報|南華早報]]》常採用法新社的報導。
