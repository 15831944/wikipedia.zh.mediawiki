'''法医秦明'''，是中国[[法医|法医]][[秦明_(法医)|秦明]]创作的犯罪推理悬疑[[系列小说|系列小说]]。由《尸语者》、《无声的证词》、《第十一根手指》、《清道夫》、《幸存者》五本小说组成。

作者秦明为就职于[[安徽省公安厅|安徽省公安厅]]物证鉴定中心的专职法医，以工作案例创作小说，虚构时间、地点和人物。小说主角秦明使用了作者本人名字，[[痕检|痕检]]员林涛原型为作者同事王林涛，秦明妻子铃铛来自作者妻子的网名“铃铛姐姐”。2012年9月，第一部《尸语者》出版<ref name="a1">{{Cite web |url=http://www.hqrw.com.cn/2015/0507/25173.shtml |title=秦明：法医要鬼手佛心——他根据工作经历创作的作品成为最畅销的悬疑小说 |accessdate=2017-05-09 |author=记者：李雪|date=2015-05-07 |work=责任编辑：刘馨瑶 |publisher=环球人物网|language=Zh-hans }}</ref>。2015年<ref>{{Cite web |url=http://zj.people.com.cn/n/2015/1102/c228592-26996547-4.html |title=首届网络文学双年奖颁奖典礼宁波举行 网络小说《将夜》获金奖 |accessdate=2017-05-09 |author= |date=2015-11-02 |work= |publisher=人民网-浙江频道|language=Zh-hans }}</ref>，《第十一根手指》获第一届网络文学双年奖优秀奖。2016年10月的报道，称秦明正在创作法医秦明系列第六本小说<ref>{{Cite web |url=http://tj.ifeng.com/a/20161031/5104476_1.shtml |title=秦明：要做一个能写很久的作者 |accessdate=2017-05-09 |author= |date=2016-10-31 |work=责任编辑：吴兴 |publisher=凤凰网，来源：每日新报|language=Zh-hans }}</ref>。同年，[[网络剧|网络剧]]《[[法医秦明_(网络剧)|法医秦明]]》播出。2018年制作播出续集——《[[法医秦明II|法医秦明II]]》，以及《[[法医秦明之幸存者|法医秦明之幸存者]]》。2019年4月，上映电影《[[生死语者·秦明|生死语者·秦明]]》。
