'''马季'''（{{bd|1934年|8月2日|2006年|12月20日|catIdx=M马}}），原名'''馬樹槐'''，[[中国|中国]][[相声演员师承关系|第七代]][[相声|相声]]演员。

==生平==
马季[[籍贯|祖籍]][[中国|中国]][[天津|天津]][[宝坻|宝坻]]，出生于[[北京|北京]]。年轻时是上海宏德织造厂的学徒。1953年，考入[[新华书店|新华书店]]华北发行所做售货员。1956年参加全国职工业余曲艺汇演，被[[侯宝林|侯宝林]]、[[刘宝瑞|刘宝瑞]]分别看中，加入[[中央广播文工团|中央广播文工团]]说唱团为专业相声演员，师从[[侯宝林|侯宝林]]、[[刘宝瑞|刘宝瑞]]、[[郭全宝|郭全宝]]、[[郭启儒|郭启儒]]，责任老师为侯宝林。因为当时[[匈牙利|匈牙利]]电影《{{link-hu|牧鹅少年马季 (1949年电影)|Ludas Matyi (film, 1949)|牧鹅少年马季}}》<ref>[http://www.imdb.com/title/tt0041608/ 牧鹅少年马季]{{英语}}</ref>（根据匈牙利作家{{link-hu|米哈伊·法泽考|Fazekas Mihály}}{{link-en|牧鹅少年马季|Mattie the Goose-boy|长诗}}改编）在中国放映，馬樹槐取艺名马季<ref name="hou">{{cite news|title=马季首次披露：我与侯宝林的恩怨是怎样结下的|url=http://culture.people.com.cn/GB/40462/40463/4087788.html|work=[[人物_(人民出版社)|人物]]|publisher=[[人民网|人民网]]|date=2006-02-09|accessdate=2019-03-08}}</ref>。

由于马季被领导安排和侯宝林、郭全宝、刘宝瑞、郭启儒等大师学艺，侯宝林为责任老师。因为同时有多个师傅，在师徒关系上“很难处”<ref name="hou"/>。之后在[[文革|文革]]中马季和侯宝林都被批斗，民间普遍传说马季在一次批斗会上打了侯宝林。文革之后多次有人求证打人事件是否属实，师徒二人都没有正面回答。《[[南方周末|南方周末]]》记者在对侯宝林三子[[侯耀文|侯耀文]]的采访中提到此传言，又说传言中侯宝林曾经删除过其传记《毛主席听我说相声》前言中讲述此事的内容，侯耀文则回应“不要听底下瞎说八道”，并表示侯宝林在相声专业上一直最喜欢马季，同时评价侯宝林能够公正明确区分专业与个人恩怨<ref>{{cite news|author=夏榆|title=侯耀文：没有“龙”哪有“首”|url=http://www.infzm.com/content/9937|publisher=[[南方周末|南方周末]]|date=2007-12-18|accessdate=2019-03-08}}</ref>。[[郭德纲|郭德纲]]也曾求证于侯宝林次子[[侯耀华|侯耀华]]以及侯耀文，他们均表示绝无此事<ref>[http://www.youtube.com/watch?v=5Xp88M7qMpI 今夜有戏：向天堂的喜剧大师致敬，郭德纲、于谦等 32分40秒-33分15秒处 2011年4月5日]</ref>。侯宝林临终前，给马季留下遗言：“马季，相声的即兴发挥千万不能丢掉啊，它是咱们相声的主要技巧。”<ref name="hou"/><ref>{{cite news|author=蒯乐昊|title=马季：未完成的功课
|url=http://news.sina.com.cn/c/cul/2007-01-05/121111961078.shtml|work=[[南方人物周刊|南方人物周刊]]|publisher=[[新浪网|新浪网]]|date=2007-01-05|accessdate=2019-03-08}}</ref>

马季是中国“歌颂型”相声的开创者之一。他创作演出了很多歌颂[[中华人民共和国|中华人民共和国]]建设成就和英雄人物的作品，其中以《登山英雄赞》、《画像》和《找舅舅》等节目影响为大。因而也是经常出国的相声演员之一。在[[文革|文革]]中受到冲击，文革之后表演日趋成熟。1985年当选“[[十大笑星|十大-{笑}-星]]”之首。一直演出到20世纪90年代，后因不满[[中央电视台|中央电视台]]按照[[新闻|新闻]]的标准审查相声，中止与后者合作。在中国大陆，一些人把“惧内”叫做“[[气管炎|气管炎]]”（音同“妻管严”，即妻子管得太严）就是来自于马季的相声。

马季于北京时间2006年12月20日早上10时25分因[[心脏病|心脏病]]突发于[[北京|北京]]天通苑五區的家中逝世<ref>[http://news.xinhuanet.com/ent/2006-12/20/content_5512288.htm 中国著名相声表演艺术家马季在家因心脏病突发病逝] {{webarchive|url=https://web.archive.org/web/20080220071527/http://news.xinhuanet.com/ent/2006-12/20/content_5512288.htm |date=2008-02-20 }}</ref>。

== 家庭 ==

== 评价 ==
*[[陈佩斯|陈佩斯]]：“马季先生是所有文艺工作者的楷模，他对艺术追求的勤奋、对事业的执着值得大家一生去学习。”<ref>{{cite web |url=http://ent.qq.com/a/20061220/000328.htm |title=陈佩斯痛悼马季先生：他是文艺工作者的楷模 |accessdate=2006年12月20日 |work=|publisher=腾讯网 |language= zh-hans}}</ref>
*[[奇志|奇志]]：“他是勤奋和硕果累累的一身，是建国以来新相声的里程碑。[[侯宝林|侯宝林]]是收集继承传统的相声艺术。而马季的相声是走进新时代的，是与时俱进的。”<ref>{{cite web |url=http://ent.qq.com/a/20061220/000348.htm |title=奇志几天前见过马季 二人飞机上亲密交谈两小时 |accessdate= 2006年12月20日|work=|publisher=腾讯网 |language= zh-hans}}</ref>
*[[郭德纲|郭德纲]]：“我们要永远承认马季先生是我们的相声大师，他创造、表演了很多歌颂型的节目，给相声开出了一条路来，而且他在这方面做得非常的成功……他不仅新活表演的好，老活也表演地非常的好。这是真正意义上的相声大家。”<ref name="konya">[http://www.youtube.com/watch?v=5Xp88M7qMpI 今夜有戏：向天堂的喜剧大师致敬，郭德纲、于谦等 32分钟8秒开始 2011年4月5日]</ref>
