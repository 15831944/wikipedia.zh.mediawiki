'''马德望皇家火车站'''是指位于[[柬埔寨|柬埔寨]][[马德望省|马德望省]]省会[[马德望|马德望]]的火车站。该火车站于1953年开始运营，直至2009年关停。2018年7月，该火车站重新恢复运营。

== 建筑 ==
在2009年停止运营后，马德望皇家火车站的{{link-en|站房|Station building}}重新进行了外墙粉刷工作，时至今日保存完好。火车站入口处的一个钟表从一个未知的日子起停滞在8时02分。2018年1月起，马德望站站内铁轨被陆续替换。除站房外，马德望站还有一个废弃仓库和机车车辆的残骸。虽然这些设施一直处于未维护的状态，但其中一些建筑一直被用于生活，储存物资或处理部分业务<ref name="BRS">{{Cite web |url=https://battambangtraveler.com/%E1%9E%81%E1%9F%92%E1%9E%89%E1%9E%BB%E1%9F%86%E1%9E%9F%E1%9F%92%E1%9E%9A%E1%9E%9B%E1%9E%B6%E1%9E%89%E1%9F%8B%E1%9E%94%E1%9E%B6%E1%9E%8F%E1%9F%8B%E1%9E%8A%E1%9F%86%E1%9E%94%E1%9E%84-copy-right-battambang-traveller-group-%E1%9E%81%E1%9F%92%E1%9E%89%E1%9E%BB%E1%9F%86%E1%9E%9F%E1%9F%92%E1%9E%9A%E1%9E%9B%E1%9E%B6%E1%9E%89%E1%9F%8B%E1%9E%94%E1%9E%B6%E1%9E%8F%E1%9F%8B%E1%9E%8A%E1%9F%86%E1%9E%94%E1%9E%84/battambang-railway-station |title=Battambang Railway Station |date=2016-11-02 |access-date=2018-11-10 |language=en |publisher=Battambang Traverler |archive-url=https://web.archive.org/web/20171117002811/https://battambangtraveler.com/%E1%9E%81%E1%9F%92%E1%9E%89%E1%9E%BB%E1%9F%86%E1%9E%9F%E1%9F%92%E1%9E%9A%E1%9E%9B%E1%9E%B6%E1%9E%89%E1%9F%8B%E1%9E%94%E1%9E%B6%E1%9E%8F%E1%9F%8B%E1%9E%8A%E1%9F%86%E1%9E%94%E1%9E%84-copy-right-battambang-traveller-group-%E1%9E%81%E1%9F%92%E1%9E%89%E1%9E%BB%E1%9F%86%E1%9E%9F%E1%9F%92%E1%9E%9A%E1%9E%9B%E1%9E%B6%E1%9E%89%E1%9F%8B%E1%9E%94%E1%9E%B6%E1%9E%8F%E1%9F%8B%E1%9E%8A%E1%9F%86%E1%9E%94%E1%9E%84/battambang-railway-station# |archive-date=2017-11-17 |dead-url=yes }}</ref>。

== 线路 ==
马德望皇家火车站内的线路标准为1000毫米（即[[米轨|米轨]]）。[[第二次世界大战|第二次世界大战]]时，马德望被泰国占领，该线路即在该期间由泰国人完成的。在[[柬埔寨内战|柬埔寨内战]]期间，该线路遭到大炮的猛烈攻击，[[赤色高棉|赤色高棉]]破坏了铁轨的多处地段。20世纪80年代，有限的铁路服务被恢复。恢复的路段为[[马德望|马德望]]至[[金边|金边]]区间，单程用时为14小时。2009年，由于铁路条件过差，铁路服务被完全取消。2018年7月，马德望皇家火车站和相关线路重新开通。现时，到[[泰国|泰国]]边境的列车每隔一天运行一次，以满足从[[金边|金边]]到[[曼谷|曼谷]]的运输服务计划<ref name="BRS"/>。
