'''马蒂亚斯·奥克尔特'''（{{lang-de|Matthias Ockert}}，{{bd|1970年}}）是一名德国的[[作曲家|作曲家]]兼[[吉他手|吉他手]]。

==简历==
马蒂亚瑟·奥克尔特生于[[博登湖县|博登湖县]]。在他学习音乐之前，他在[[柏林工业大学|柏林工业大学]]进行建筑专业本硕连读的学习，并且在1998年毕业<ref>{{cite web |url=http://www.operare.de/08/Teilnehmer_08/Ockert.htm |title=Matthias Ockert |website=operare.de |access-date=July 25, 2017}}</ref>。在此期间，他亦师从卡罗·尹德希斯（Carlo Inderhees）学习作曲与乐理。在此之后，他继续在{{Link-en|柏林音乐大学|Hochschule für Musik "Hanns Eisler"}}与{{Link-en|卡尔斯鲁厄音乐大学|Hochschule für Musik Karlsruhe}}师从沃夫冈·里姆（Wolfgang Rihm）、桑迪普·巴格瓦蒂（Sandeep Bhagwati）和汉斯皮特·居布尔茨（Hanspeter Kyburz）完成了作曲专业本硕连读的学习<ref name="klangfaecher">{{cite web |url=http://www.klangfaecher.de/klangfaecher/kollegium-ockert.php| title=Matthias Ockert - Gitarrist / Lehrer für Gitarre, E-Gitarre |website=klangfaecher.de |access-date=July 25, 2017}}</ref>。他亦定期去[[纽约|纽约]]，师从爵士吉他手{{Link-en|阿特利亚·佐勒|Attila Zoller}}和{{Link-en|比尔·康纳斯|Bill Connors}}学习爵士吉他技巧<ref>{{cite web |url=http://www.matthiasockert.de/bio/page4/| title=English Bio |website=matthiasockert.de |access-date=July 25, 2017}}</ref>。

奥克尔特从1995年起开始进行吉他教学的工作。从2001年起，他作为一名吉他教师受雇于[[卡尔斯鲁厄|卡尔斯鲁厄]]的Klangfächer音乐学校<ref name="klangfaecher"/>。
