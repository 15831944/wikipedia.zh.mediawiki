'''马克·派特森'''（{{lang-en|'''Mark Paterson'''}}），英国[[音频工程师|音频工程师]]。<ref name="goldcrestfilms">{{cite web |url=http://www.goldcrestfilms.com/staff_profile/mark-paterson |title=Mark Paterson |accessdate=2013-03-08 |work=goldcrestfilms |deadurl=yes |archiveurl=https://archive.is/20130411020823/http://www.goldcrestfilms.com/staff_profile/mark-paterson |archivedate=2013-04-11 }}</ref><ref name="postmagazine">{{cite web |url=http://www.postmagazine.com/Publications/Post-Magazine/2012/December-1-2012/The-Sound-of-Les-Miserables.aspx |title=The Sound of 'Les Miserables' |accessdate=2013-03-08 |work=postmagazine}}</ref> 他因[[悲惨世界_(2012年电影)|悲惨世界]]赢得了[[奥斯卡最佳音响效果奖|奥斯卡最佳音响效果奖]]。<ref name="Oscars">{{cite web |url=https://www.bbc.co.uk/news/entertainment-arts-20973004 |title=Oscars 2013: Full list of winners |accessdate=2013-03-08|work=BBC News}}</ref>自2004年起，派特森参与制作过60多部电影。
