'''马陆站'''位于[[上海|上海]][[嘉定区|嘉定区]][[马陆镇|马陆镇]]，为[[上海轨道交通11号线|上海轨道交通11号线]]一期的高架[[侧式车站|侧式站台]]，[[上海轨道交通11号线|11号线]]站台位于第二层。于2009年12月31日启用。

== 车站设计 ==
== 公交换乘 ==
马陆1路、马陆2路

== 车站出口 ==
*1号口：站体东北侧，近崇蛟路阿克苏南路
*2号口：站体东南侧，近亚刚路阿克苏南路
*3号口：站体西南侧，近康丰路亚刚路
*4号口：站体西北侧，近崇蛟路康丰路
