'''马向东'''（{{bd|1953年|5月31日|2001年|12月19日|catIdx=Ma,马}}），[[中国|中国]][[辽宁|辽宁]][[沈阳|沈阳]]人，[[回族|回族]]，曾任[[沈阳市|沈阳市]][[常务副市长|常务副市长]]，后因[[贿賂|贿賂]]被判处[[死刑|死刑]]（贪污罪，判处无期徒刑，剥夺政治权利终身；犯受贿罪，判处死刑，并处没收个人全部财产；犯挪用公款罪，判处有期徒刑八年；犯[[巨额财产来源不明罪|巨额财产来源不明罪]]，判处有期徒刑五年。决定执行死刑，剥夺政治权利终身，并处没收个人全部财产）。

== 经历 ==
马向东于1983年出任[[中国共青团|中国共青团]]沈阳市委书记，1984年调任沈阳市商业局局长。1993年，马向东在竞选沈阳市副市长时拉选票，导致竞选对手[[张鸣岐_(1945年)|张鸣岐]]退出选举<ref>{{cite news|url=http://news.eastday.com/epublish/gb/paper148/20010129/class014800020/hwz299737.htm|title=副市长折戟东方"蒙特卡罗"|publisher=东方网|date=2001-01-29|accessdate=2018-07-14}}</ref><ref name="untergang">{{cite journal|author=玄洪云 ,赵辉|title=赌市长的毁灭之路——沈阳市原常务副市长马向东的堕落轨迹|journal=中国监察|publisher=中国监察杂志社|issn=2096-0948|year=2001|issue=24|pages=38-44}}</ref>。

1999年7月，因在[[澳门|澳门]]赌场3天输掉公款上千万元被[[双规|双规]]，10月10日被逮捕。2001年10月10日，马向东被[[南京市中级人民法院|南京市中级人民法院]]判处死刑<ref name="untergang"/><ref>{{cite news|url=http://www.chinanews.com/2001-10-10/26/129136.html|title=沈阳慕马大案宣判 16名巨蠹被依法严惩|publisher=[[中国新闻社|中国新闻社]]|date=2001-10-10|accessdate=2018-07-14}}</ref>，12月19日在[[南京|南京]]被执行[[注射死刑|注射死刑]]，成为江苏省首例注射死刑的被执行人<ref>{{cite news|url=http://news.southcn.com/china/china07/200112191058.htm|title=沈阳市原副市长马向东等今日伏法|publisher=南方网|date=2001-12-19|accessdate=2018-07-14}}</ref><ref>{{cite news|url=http://news.sina.com.cn/c/2001-12-21/425589.html|title=沈阳巨贪马向东被注射死亡 系江苏首例注射死刑犯|publisher=[[南方都市报|南方都市报]]|date=2001-12-21|accessdate=2018-07-14}}</ref>。

== 个人生活 ==
马向东热爱读书，经常随身携带三本书以便学习和研究：《[[赌术精选|赌术精选]]》、《[[赌博游戏技巧分享|赌博游戏技巧分享]]》、《[[赌术实战108招|赌术实战108招]]》。<ref>{{cite web |url=http://news.qq.com/a/20120319/000520.htm?pgv_ref=aio2012&ptlang=2052 |title= 人民日报评贪官读书：胡长清爱看《肉蒲团》|accessdate= 2012年|work= |publisher=腾讯网 |language =zh-cn}}</ref><ref>{{cite web |url= http://china.rednet.cn/c/2012/03/19/2553224.htm|title= 人民日报评贪官读书 胡长清爱看肉蒲团金瓶梅|accessdate= 2012年|work= |publisher=中国网 |language =zh-cn}}</ref>
