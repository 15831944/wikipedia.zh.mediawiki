{{Vgname|马力欧卡丁车DS|ja=マリオカートDS|en=Mario Kart DS|cn=马力欧卡丁车DS|tw=瑪利歐賽車DS|hk=瑪利歐賽車DS}}是由[[任天堂情報開發本部|任天堂情報開發本部]]開發的一款[[任天堂DS|任天堂DS]]平台[[賽車遊戲|賽車遊戲]]。本作是[[瑪利歐賽車系列|瑪利歐賽車系列]]的第五作，也是第一款可以通过[[任天堂|任天堂]]官方架设的免费网络平台[[任天堂Wi-Fi连接|任天堂Wi-Fi连接]]进行网络联机对战的游戏。《瑪利歐賽車DS》于2005年11月14日在北美發售，2005年11月25日在欧洲发售，2005年12月8日在日本发售，2007年4月5日在南韩发售。如同系列其他作品，游戏采用[[马里奥系列|马里奥系列]]的角色，在马里奥系列主题赛道上驾驶[[卡丁车|卡丁车]]竞速。

《马力欧卡丁车DS》获得好评，在[[Metacritic|Metacritic]]上的汇总得分为91%。称赞围绕游戏的图形和游戏性，批评之处主要为单调的单人模式。游戏获得[[GameSpot|GameSpot]]、[[IGN|IGN]]和《[[G-Phoria|G-Phoria]]》颁发的数项奖项。游戏在2013年时第三畅销的任天堂DS游戏，到2014年6月时全球销量2356万。

== 玩法 ==

[[File:Mario_Kart_DS_screenshot.png|left]]正在8型赛道上比赛，在他前面的是道具箱；左下角是当前排名，右上是当前圈数。<br>下屏幕：左侧显示所有赛手的当前排名，右上角是圈速，主画面是赛道的俯视图。]]
《马力欧卡丁车DS》是一款竞速游戏，玩家从12名[[瑪利歐賽車系列#角色|角色]]中选择其一，驾驶[[卡丁车|卡丁车]]和其他赛手对战，每名角色有三辆卡丁车。在竞速或对战模式，[[任天堂DS|任天堂DS]]的上屏幕以第三人称视角显示玩家赛车， 下[[触屏|触屏]]显示赛道当前排位、各赛手所持道具以及赛道地图。下屏幕有两种模式可以切换，一种可以总览赛道全框，另一种会俯瞰玩家赛车周围的情况，包括 临近赛手、赛道障碍、道具盒和来临的攻击。每条赛道都有设定有诸多道具盒，角色驾驶穿过就可以随机获得道具，使用道具有助于玩家领先对手。一类道具会攻击或阻滞其他赛手，另一类道具则会为玩家赛车加速<ref name="manual">{{cite book |title=Mario Kart DS instruction manual |publisher=Nintendo |date=2005-11-14}}</ref>。

=== 游戏模式 ===

游戏有五种单人模式：大奖赛、时间挑战、对战、决斗和任务。大奖赛和对战模式需要玩家从50 cc、100 cc、150 cc和150 cc[[镜像_(几何)|镜像]]中选择引擎等级。等级表示[[难度水平|难度水平]]——引擎等级越高，卡丁车行驶越快。另外150 cc镜像模式是150 cc赛道的水平翻转，需要解锁。在大奖赛模式中，玩家和7名[[非玩家角色|电脑控制]]赛手跑完一系列预定赛道。之前马里奥赛车游戏只有四种杯赛，本作增加到八种杯赛：蘑菇、花、星星、特别、贝壳、香蕉、叶子和闪电，后四种杯赛完全使用系列前作的赛道。每个杯赛有4赛道，合起来赛道数为32。当杯赛完成就会显示等级，等级从E到A再到1、2、3星。如果《[[马里奥赛车_双重冲击|双重冲击]]》，玩家不同名次会得到不同积分。在时间练习测试，玩家需要用1到3个蘑菇尽可能快完成赛道。最快的时间会保存影子，玩家之后游戏时会出现影子副本作为对照。在对战模式，玩家和所选的电脑对手对战。这模式可以是个人赛或团队赛，团队赛会分为红队和蓝队。决斗模式有气球大战和收集闪耀两个模式，都支持玩家个人或团队对战。在气球大战中，玩家必须攻击对手爆掉其气球，或加速向其他赛车夺取气球。每名赛手开始有一个气球。这些可以重新充气至多4次。多人模式中，人类玩家失去全部气球时就会退出游戏变成幽灵。幽灵玩家只能放置道具盒。如果是单人模式，则人类角色气球全部被爆时比赛结束，并以这时的状况作为游戏结果。在第二个模式收集闪耀中，玩家必须收集闪耀精灵（来自《[[超级马里奥阳光|超级马里奥阳光]]》）。玩家可以攻击其他赛手夺走其闪耀精灵，同时闪耀精灵最少者会在游戏结束时淘汰。在单人模式中，如果人类角色被淘汰，则游戏结束且结果随机生成<ref name="manual" />。

任务模式要求玩家完成任务，任务从收集硬币到攻击敌人不尽相同。在任务中玩家控制指定角色。任务有7个等级，每个下面有8个任务。在完成任务后玩家会得到分级，范围从E到A再到1、2、3星。在8个任务完成后就会完成此等级，在完成[[头目_(游戏)|头目]]任务后才能进入下一级。前六个任务完成且所有任务至少1星时，第7等级才会解锁。

游戏还支持多人模式，8名玩家赛手可使用[[DS下载游玩|DS下载游玩]]功能或多卡[[无线局域网|无线局域网]]连接。在2014年5月20日前，《马力欧卡丁车DS》还通过[[任天堂Wi-Fi连接|任天堂Wi-Fi连接]]，支持至多四人同时在线游戏<ref>http://www.gamespot.com/articles/nintendo-killing-online-support-for-mario-kart-wii-super-smash-bros-brawl-in-may/1100-6417983/</ref>。在线游戏时，参与者只能互相竞速，在线连接不支持决斗模式<ref name="manual" />。{{clear}}

== 开发 ==

任天堂在2005年5月11日首度公开[[任天堂DS|任天堂DS]]马里奥赛车作品发售计划，同时公布了一些游戏性视频镜头<ref>{{cite web |accessdate=2009-01-08 |url=http://ds.ign.com/articles/513/513232p1.html |title=E3 2004: Mario Kart on DS |publisher=IGN |date=2004-05-11 |author=Harris, Craig }}</ref>。公司在2005年[[游戏开发者大会|游戏开发者大会]]首次公开提供游戏试玩，并展示了游戏无线功能<ref name="ign">{{cite web |accessdate=2009-01-08 |url=http://ds.ign.com/articles/595/595106p1.html |title=GDC 2005: Mario Kart Impressions |publisher=IGN |date=2005-03-10 |author=Harris, Craig }}</ref>。《马里奥赛车DS》由2005年《[[任天堂|任天堂]]》制作人[[绀野秀树|绀野秀树]]制作<ref name="ign">{{cite web |accessdate=2009-01-08 |url=http://ds.ign.com/articles/617/617391p1.html |title=E3 2005: Mario Kart DS Interview |publisher=IGN |date=2005-05-19 |author=Harris, Craig }}</ref>。游戏以恒60[[帧每秒|帧每秒]]显示全3D角色和环境<ref name="ign" />。

《马力欧卡丁车DS》是首部支持在线游戏的马里奥赛车的作品。绀野称虽然《马里奥赛车DS》和[[光环系列|光环系列]]都支持在线游戏，但使用光环在线功能多为[[硬核玩家|硬核玩家]]。对于《马力欧卡丁车》，绀野“希望所有人都上线，而这正是实现这一点的正确技术和时间”。每部马里奥赛车都承袭传统且引入新机制，《马力欧卡丁车DS》是首部以一卡带支持8玩家同时游戏的作品。游戏还引入无需多人参与的单人对战模式。作为任天堂DS上的首部马里奥赛车，开发者尝试多种利用设备下[[触屏|触屏]]的方式。他们考虑过让玩家将道具投掷到赛道任意地方，而非只能扔在卡丁车后。然而他们发现这回让玩家太过分心，在赛车时难以控制放置地点<ref name="ign" />。

在《马力欧卡丁车DS》中，卡丁车可以在其他卡丁车后面[[漂移|漂移]]获得瞬间加速，这一特性也在之前1996年的《[[马力欧卡丁车64|马力欧卡丁车64]]》中出现。《马力欧卡丁车DS》更加重此功能，并再度于赛车漂移时使用视觉提示。绀野在采访中表示，游戏收录了前作的赛道，玩过[[超级任天堂|超级任天堂]]初代《[[超级马力欧卡丁车|超级马力欧卡丁车]]》的玩家会更为熟悉《DS》<ref>{{cite web | accessdate = 2012-02-18 |url=http://www.cubed3.com/news/4200 |title=Nintendo Interview | Hideki Konno On All Things Mario Kart |work=Cubed3 | date = 2012-08-20 |author=Riley, Adam}}</ref>。

游戏原声由《[[马里奥赛车_双重冲击|双重冲击]]》作曲者田中忍谱写<ref>[http://www.giantbomb.com/shinobu-tanaka/3040-61622/ Shinobu Tanaka Profile].GaintBomb.[2015-01-15].</ref>，配音员有[[户高一生|户高一生]]和[[查尔斯·马蒂内|查尔斯·马蒂内]]等。

《马里奥赛车DS》由[[任天堂|任天堂]]在[[任天堂DS|任天堂DS]]平台发行，北美版于2005年11月14日、澳洲版于2005年11月17日、欧洲版于2005年11月25日、日本于2005年12月8日面世<ref>{{cite web |accessdate=2009-01-08 |url=http://www.gamespot.com/ds/driving/mariokartds/similar.html?mode=versions |title=Release Summary |publisher=GameSpot }}</ref>。任天堂随后宣布在2005年11月28日发售红色任天堂DS同捆《马里奥赛车DS》，并附带“方格旗腕带和赛车风格贴花来定制红色掌机”<ref>{{cite web |accessdate=2009-01-08 |url=http://ds.ign.com/articles/658/658077p1.html |title=Nintendo Reveals Red-Hot DS Bundle |publisher=IGN |date=2005-10-12 |author=Adams, David }}</ref>。

游戏获得媒体称赞，在[[Metacritic|Metacritic]]上汇总得分为91%<ref name="metacritic" />，日本杂志《[[Fami通|Fami通]]》为游戏打出36/40<ref name="fami" />。评论集中称赞游戏的图形和游戏性，并批评重复的单人模式。《马里奥赛车DS》获得[[GameSpot|GameSpot]]和[[IGN|IGN]]的编辑推荐奖<ref name="gamespot" /><ref name="ign" />。游戏获得GameSpot的最佳多人游戏<ref>{{cite web |accessdate=2009-01-08 |url=http://www.gamespot.com/pages/features/bestof2005/index.php?day=2&page=17 |title=Best Multiplayer Game |publisher=GameSpot }}</ref>、最佳驾驶游戏和最佳DS游戏提名<ref>{{cite web |accessdate=2009-01-08 |url=http://www.gamespot.com/pages/features/bestof2005/index.php?day=4&page=6 |title=Best Driving Game |publisher=GameSpot }}</ref>，并摘取最后一项奖项<ref>{{cite web |accessdate=2009-01-08 |url=http://www.gamespot.com/pages/features/bestof2005/index.php?day=5&page=2 |title=Best DS Game |publisher=GameSpot }}</ref>。游戏获得《[[G-Phoria|G-Phoria]]》的最佳掌机游戏奖<ref>{{cite web |accessdate=2009-01-08 |url=http://www.gamespot.com/ds/driving/mariokartds/news.html?sid=6155587 |title=G4 announces G-phoria winners, books Jack Thompson |publisher=GameSpot |date=2006-08-10 |author=Thorsen, Tor }}</ref>。IGN授予游戏2005年度最佳竞速/驾驶游戏奖<ref>{{cite web |accessdate=2009-01-08 |url=http://bestof.ign.com/2005/ds/21.html |title=Game of the Year |publisher=IGN }}</ref>。

几家评论称赞游戏没有辜负前作创下的标准。[[GameSpy|GameSpy]]认为，虽然游戏在线部分恼人，但单人模式和本地无线模式远补偿了此项<ref name="gamespy" />。任天堂世界报道称“以往马里奥赛车游戏的最佳功能又回来了”，并和《马里奥赛车DS》的新功能配合，铸成了“有史以来印象最深刻的任天堂DS游戏，也是马里奥赛车系列的最佳游戏”<ref name="nwr">{{cite web |accessdate=2009-01-08 |url=http://www.nintendoworldreport.com/reviewArt.cfm?CFID=92669d9c-07c5-49b6-9881-f1dba26b49f3&CFTOKEN=0&artid=4450 |title=Mario Kart DS |publisher=Nintendo World Report |date=2005-11-19 |author=Sklens, Mike }}</ref>。持同样观点的《[[X-Play|X-Play]]》表示，游戏突破一切期待，成为“全版本——掌机还是其它——卡丁车竞速游戏里最棒的”<ref name="xplay">{{cite web |accessdate=2009-01-08 |url=http://www.g4tv.com/xplay/reviews/211/Mario_Kart_DS.html |title=Mario Kart DS |work=X-Play }}</ref>。[[GameZone|GameZone]]确信《马里奥赛车DS》用创造性的赛道和“出色的”多人模式和比其他同类型道具类竞速游戏更高的重复可玩性的价值“没有令这个系列失望”<ref name="gamezone">{{cite web |accessdate=2009-01-09 |url=http://nds.gamezone.com/gzreviews/r24080.htm |title=Mario Kart DS Review |publisher=GameZone |date=2005-12-12 |author=Bedigian, Louis |deadurl=yes |archiveurl=https://web.archive.org/web/20081227070106/http://nds.gamezone.com/gzreviews/r24080.htm |archivedate=2008-12-27 }} </ref>。[[CBS新闻|CBS新闻]]的亚历杭德罗·K·布朗称赞游戏以独特方法使用了任天堂DS特性，如麦克风和无线连接<ref>{{cite web |accessdate=2009-02-03 |url=http://www.cbsnews.com/stories/2005/11/18/tech/gamecore/main1058578.shtml |title=Mario Kart DS Dominates |publisher=CBS News |date=2005-11-18 |author=Vitka, William }}</ref>。在[[GamesRadar|GamesRadar]]的一篇文章中，《马里奥赛车DS》击败《[[口袋妖怪_黑·白|口袋妖怪 黑·白]]》（第二名）和《[[侠盗猎车手：血战唐人街|侠盗猎车手：血战唐人街]]》（第三名），成为史上最佳任天堂DS游戏<ref>{{cite web | accessdate = 2013-02-03 | url=http://www.gamesradar.com/best-ds-games/ | title=Best DS Games | publisher=[[GamesRadar|GamesRadar]] | author=GamesRadar staff | date = 2012-09-15}}</ref>。

IGN称赞《马里奥赛车DS》的游戏性和设计深度，并称难以想象任天堂如何做出胜过本作的马里奥赛车游戏<ref name="ign" />。GameSpot称游戏是马里奥赛车系列的“显著进步”，部分原因是系列首次采用联机游戏<ref name="gamespot" />。[[Game_Revolution|Game Revolution]]称游戏用单人和多人模式“完成职责”<ref name="gamerev">{{cite web |accessdate=2009-01-08 |url=http://www.gamerevolution.com/review/ds/mario_kart_ds |title=Mario Kart DS - DS Review |publisher=Game Revolution |date=2005-11-15 |author=Silverman, Ben |deadurl=yes |archiveurl=https://web.archive.org/web/20080512161108/http://www.gamerevolution.com/review/ds/mario_kart_ds |archivedate=2008-05-12 }}</ref>。[[1UP.com|1UP.com]]称游戏“出奇抓人的包装”是“和史上所有主机作品比肩的携带式竞速游戏”<ref name="1up" />。电子游戏杂志《[[GamePro|GamePro]]》对游戏多样的赛手、赛道和模式以及多人选项表示称赞，称本作是任天堂爱好者的“必玩游戏”及任天堂DS机主的必备游戏<ref name="gamepro">{{cite web |accessdate=2009-01-08 |url=http://www.gamepro.com/article/reviews/49814/mario-kart-ds/ |title=Mario Kart DS |work=GamePro |date=2005-11-14 |author=Burner, Rice |archiveurl=https://web.archive.org/web/20080914153322/http://www.gamepro.com/article/reviews/49814/mario-kart-ds/ |archivedate=2008-09-14 |deadurl=yes }}</ref>。《[[电脑与电子游戏|电脑与电子游戏]]》称《马里奥赛车DS》虽有少量图形有瑕疵，却是“最齐全”的马里奥赛车游戏<ref name="cvg">{{cite web |accessdate=2009-01-08 |url=http://www.computerandvideogames.com/article.php?id=130044 |title=Mario Kart DS |work=Computer and Video Games |date=2005-11-24 }}</ref>。[[Eurogamer|Eurogamer]]对游戏多人模式表示满意，称游戏“真正实现于他人同乐”<ref name="eurogamer" />。中国大陆杂志《[[游戏机实用技术|游戏机实用技术]]》观点相同，但称游戏机十字键较浅阻碍了部分技巧的使用<ref name="UCG" />。英国出版物《[[GamesTM|GamesTM]]》批评游戏“除马里奥赛车概念文饰外无他”<ref name="gamestm">{{cite journal |title=Mario Kart DS review |journal=GamesTM | date = 2006-05 |page=99 }}</ref>，电子游戏网站Nintendophiles对“单调的”多人模式和“简单的电脑玩家”表示失望<ref name="nintendophiles">{{cite web |accessdate=2009-01-08 |url=http://www.n-philes.com/reviews/165/mario-kart-ds/ |title=Mario Kart DS (DSN) |publisher=Nintendophiles |date=2005-11-28 |author=Biersdorf, Dan |deadurl=yes |archiveurl=https://web.archive.org/web/20080620230039/http://www.n-philes.com/reviews/165/mario-kart-ds/ |archivedate=2008-06-20 }}</ref>。
