== 历史 ==
这座博物馆的建筑始建于1928年。它最初是用作人工电话交换机的办公室。在1938年，它被升级成为机械式电话交换机的办公室。<ref name="muziumtelekom.com.my">{{Cite web|url=http://www.muziumtelekom.com.my/aboutus.html|title=Muzium Telekom|publisher=Telekom Museum|accessdate=6 December 2016}}</ref>

在1980年代早期， [[马来西亚电讯公司|马来西亚电讯]]打算在吉隆坡寻找一个空间来容纳他们的办公室，以便他们能存放公司的电信设备。 在1984年，[[马来西亚电讯公司|马来西亚电讯]]确认了该建筑物是合适的地点并计划扩建。 然而，该计划因时任首相[[马哈迪·莫哈末|马哈迪·莫哈末]]在1985年提议将该建筑物视作为国家遗产予以保护而取消。 马来西亚电讯随后决定将该建筑转变成一座耗资马币1000万的博物馆。 博物馆的建造工作于1989年至1992年完成，而博物馆内的收藏品于1993 - 1994年开始收集并准备展示。 在1994年6月3日，该博物馆正式向公众开放。 <ref>{{Cite web|url=http://www.muziumtelekom.com.my/|title=History of Museum Telekom|publisher=Telekom Museum|accessdate=6 December 2016}}</ref> 

== 建筑 ==
博物馆的建筑拥有新古典希腊建筑风格。 <ref name="muziumtelekom.com.my">{{Cite web|url=http://www.muziumtelekom.com.my/aboutus.html|title=Muzium Telekom|publisher=Telekom Museum|accessdate=6 December 2016}}</ref> 

== 开放时间 ==
除公众假期外，该博物馆于每日上午九时至下午五时向公众开放。

== 交通 ==
该博物馆位于[[占美清真寺站|占美清真寺站]]以东，步行即可到达。
