'''马来语克里奥尔语'''是以[[马来语|马来语]]为基础的[[克里奥尔语|克里奥尔语]]。

== 泰國 ==
* {{tsl|ms|Bahasa Melayu Kedah|吉打話}}：主要分布于[[沙敦府|沙敦府]]。
== 馬來西亞 ==
* {{tsl|ms|Bahasa Melayu Kedah|吉打話}}：主要分布于[[吉打|吉打]]、[[檳城|檳城]]、[[霹靂州|霹靂州]]和[[玻璃市|玻璃市]]。
* 峇峇話：主要分布于[[馬來西亞|馬來西亞]]、[[新加坡|新加坡]]、[[印尼|印尼]]，为[[峇峇娘惹|海峡华人]]的传统语言之一，混合[[爪哇语|爪哇语]]、[[閩南話|閩南話]]，如今几乎失传。

== 印尼 ==
* {{tsl|id|Bahasa Betawi|Betawi}}：主要分布于[[雅加達|雅加達]](雅加達以前称为Betawi)，混合[[爪哇语|爪哇语]]、[[巽他语|巽他语]]、[[漢语|漢语]]、[[葡萄牙语|葡萄牙语]]、[[荷蘭语|荷蘭语]]、[[巴厘语|巴厘语]]，使用人口约270萬人（1993年统计）。
* {{tsl|id|Bahasa Manado|萬鴉老话}}：主要分布于[[萬鴉老|萬鴉老]]，受到德那第话、[[荷兰语|荷兰语]]、{{tsl|id|Suku Minahasa|Minahasa}}话以及少许的[[西班牙语|西班牙语]]、[[葡萄牙语|葡萄牙语]]的影響。
* 德那第话：主要分布于[[德那第島|德那第島]]、[[蒂多雷島|蒂多雷島]]、[[哈馬黑拉島|哈馬黑拉島]]、[[北馬魯古省|北馬魯古省]]、[[蘇拉群島|蘇拉群島]]。
* 巴占话：主要分布于[[巴占群岛|巴占群岛]]，有別於安汶话和德那第话。
* {{tsl|en|Larantuka Malay|Larantuka話}}：主要分布于{{tsl|id|Larantuka, Flores Timur|Larantuka}}区，受[[葡萄牙语|葡萄牙语]]影響。
* 古邦话：主要分布于[[古邦市|古邦市]]，混合[[荷兰语|荷兰语]]、[[葡萄牙语|葡萄牙语]]和一点的当地土话。
* [[安汶话|安汶话]]：主要分布于[[安汶島|安汶島]]，混合[[荷兰语|荷兰语]]、[[葡萄牙语|葡萄牙语]]和一点的当地土话。
* 班达话：主要分布于[[班达群岛|班达群岛]]，有別於安汶话。

== 斯里蘭卡 ==
* {{tsl|en|Sri Lankan Malay language|斯里兰卡马来语}}：混合[[僧伽羅语|僧伽羅语]]、[[泰米爾语|泰米爾语]]，为[[斯里兰卡马来人|斯里兰卡马来人]]的传统语言之一。
