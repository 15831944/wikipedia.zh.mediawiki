'''马修·斯科特·加尔萨'''（{{lang-en|Matthew Scott Garza}}，{{bd|1983年|11月26日|||Garza}}），出生于[[加利福尼亚州|加利福尼亚州]][[塞尔玛|塞尔玛]]，[[美国职棒大联盟|美国职棒大联盟]][[密爾瓦基釀酒人|密爾瓦基釀酒人]]的[[投手|投手]]。

==职业生涯==
===明尼苏达双城===
加尔萨在2005年的选秀大会上第一轮第25顺位被[[明尼苏达双城|明尼苏达双城]]挑中。2006年他在[[美国棒球|美国棒球]]杂志评选的双城队未来之星中名列第七<ref>{{cite web|title = "Top 10 Prospects: Minnesota Twins"|date = 2006-02-01|publisher = 美国棒球|url = http://www.baseballamerica.com/today/features/05top10s/twins.html}}</ref>。这个赛季加尔萨从高阶1A的[[麦尔兹堡奇迹|麦尔兹堡奇迹]]起步，之后升上2A的[[新不列巅石猫|新不列巅石猫]]和3A的[[罗彻斯特红翼|罗彻斯特红翼]]，他在三个等级的小联盟球队中总共投了135又2/3局，拿到了14胜4败、防御率1.99、三振154次的成绩。靠着优异的表现，加尔萨分别入选了1A[[佛罗里达州联盟|佛罗里达州联盟]]和2A[[东方联盟|东方联盟]]的明星赛，还获得了今日美国年度最佳小联盟球员的荣誉，这个奖项连续第二年被双城队的球员获得（前一年是加尔萨的队友[[弗朗西斯科·利瑞安诺|弗朗西斯科·利瑞安诺]]夺得）<ref>{{cite web|title = "Alexi Casilla and Matt Garza named 2006 Twins' Minor League Player and Pitcher of the Year"|date = 2006-11-01|publisher = MLB.com|url = http://minnesota.twins.mlb.com/news/press_releases/press_release.jsp?ymd=20061101&content_id=1728954&vkey=pr_min&fext=.jsp&c_id=min}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

2006年8月8日，双城队把加尔萨升上大联盟，顶替受伤的利瑞安诺<ref>{{cite web|title = "Notes: Early start for Garza"|date = 2006-08-08|publisher = MLB.com|url = http://minnesota.twins.mlb.com/news/article.jsp?ymd=20060808&content_id=1600111&vkey=news_min&fext=.jsp&c_id=min|deadurl = yes|archiveurl = https://web.archive.org/web/20110616235856/http://minnesota.twins.mlb.com/news/article.jsp?ymd=20060808&content_id=1600111&vkey=news_min&fext=.jsp&c_id=min|archivedate = 2011-06-16}}</ref>。加尔萨是少数仅在小联盟呆了一个赛季就升上大联盟的球员。8月11日，加尔萨在大联盟初登板，但表现并不理想，在对[[多伦多蓝鸟|多伦多蓝鸟]]的比赛中他仅投了2又2/3局就丢了7分[[自责分|自责分]]<ref>{{cite web|title = "Garza hit hard in debut vs. Jays"|date = 2006-08-12|publisher = MLB.com|url = http://minnesota.twins.mlb.com/news/gameday_recap.jsp?ymd=20060811&content_id=1604843&vkey=recap&fext=.jsp&c_id=min}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。8月23日加尔萨在对[[巴尔的摩金莺|巴尔的摩金莺]]的比赛中主投6局失1分非自责分，拿下了他在大联盟的第一胜<ref>{{cite web|title = "Garza notches first MLB win"|date = 2006-08-23|publisher = MLB.com|url = http://minnesota.twins.mlb.com/news/gameday_recap.jsp?ymd=20060823&content_id=1625722&vkey=recap&fext=.jsp&c_id=min}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

进入2007年，加尔萨在[[美国棒球|美国棒球]]杂志评选的未来之星中排名双城球团第一<ref>{{cite web|title = "Top 10 Prospects: Minnesota Twins"|date = 2006-11-30|publisher = 美国棒球|url = http://www.baseballamerica.com/today/prospects/features/262912.html}}</ref>，并在全联盟排名第21<ref>{{cite web|title = "2007 Top 100 Prospects"|date = 2007-02-28|publisher = 美国棒球|url = http://www.baseballamerica.com/today/prospects/features/263445.html}}</ref>。不过加尔萨并未在开季就赢得在大联盟的位置，因为球团希望他能多增加变化球，而不是仅仅依赖他的快速球。加尔萨因此从3A起步。6月28日加尔萨被叫上大联盟<ref>{{cite web|title = "Twins call up Garza from Triple-A"|date = 2007-06-28|publisher = MLB.com|url = http://minnesota.twins.mlb.com/news/article.jsp?ymd=20070628&content_id=2054634&vkey=news_min&fext=.jsp&c_id=min|deadurl = yes|archiveurl = https://web.archive.org/web/20070821203557/http://minnesota.twins.mlb.com/news/article.jsp?ymd=20070628&content_id=2054634&vkey=news_min&fext=.jsp&c_id=min|archivedate = 2007年8月21日|df = }}</ref>。他还入选了当年的全明星未来之星赛，但由于升上了大联盟，因此他没有参加<ref>{{cite web|title = "Futures' stars set to align today"|date = 2007-07-07|publisher = MLB.com|url = http://pittsburgh.pirates.mlb.com/news/article.jsp?ymd=20070707&content_id=2072922&vkey=allstar2007&fext=.jsp}}</ref>。这个赛季加尔萨在大联盟上场16次，拿下了5胜7败、防御率3.69的成绩。

===坦帕湾光芒===
2007年11月28日，双城队把加尔萨和[[傑森·巴列特|傑森·巴列特]]、[[爱德华多·莫兰|爱德华多·莫兰]]（Eduardo Morlan）打包送到[[坦帕湾光芒|坦帕湾光芒]]，交换[[戴勒蒙·杨|戴勒蒙·杨]]（Delmon Young）、[[杰森·普里迪|杰森·普里迪]]（Jason Pridie）和[[布伦丹·哈里斯|布伦丹·哈里斯]]（Brendan Harris）。2008年加尔萨在光芒队表现出色，常规赛投出了11胜9败、防御率3.70的成绩。在[[美国联盟冠军赛|美国联盟冠军赛]]中，加尔萨先发两场均拿下胜投，防御率仅为1.38，帮助球队击败[[波士顿红袜|波士顿红袜]]进入到[[世界大赛|世界大赛]]，他也获得了美联冠军赛最有价值球员的荣誉。

2010年7月26日，加尔萨於主場面對[[底特律老虎|底特律老虎]]，投出大聯盟生涯第一個也是光芒隊史第一個[[無安打比賽|無安打比賽]]，這場比賽他只讓一個打者靠保送上壘，但下一個打者擊出雙殺打，所以總共是27上27下，如果沒有投出這一個保送，就有機會成為[[完全比賽|完全比賽]]<ref>[http://tampabay.rays.mlb.com/news/article.jsp?ymd=20100726&content_id=12654834&vkey=news_tb&fext=.jsp&c_id=tb Garza takes place in history, no-hits Tigers] {{Wayback|url=http://tampabay.rays.mlb.com/news/article.jsp?ymd=20100726&content_id=12654834&vkey=news_tb&fext=.jsp&c_id=tb |date=20120204032358 }} MLB.com July 26, 2010</ref>。

===芝加哥小熊===
2011年1月7日，加尔萨經由交易來到[[芝加哥小熊|芝加哥小熊]]，總計有8個人經由交易轉隊。[[坦帕灣光芒|坦帕灣光芒]]獲得Sam Fuld、[[克理斯·阿契|克理斯·阿契]]、李學周（Hak-Ju Lee）、Brandon Guyer和Robinson Chirinos，小熊獲得加尔萨、Fernando Perez及Zachary Rosscup<ref>[http://sports.espn.go.com/chicago/mlb/news/story?id=5998431 Source: Cubs agree to Garza deal | ESPN]</ref><ref>{{cite web |url=http://chicago.cubs.mlb.com/news/article.jsp?ymd=20110105&content_id=16392484&vkey=news_chc&c_id=chc |title=存档副本 |accessdate=2011-01-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20110108172706/http://chicago.cubs.mlb.com/news/article.jsp?ymd=20110105&content_id=16392484&vkey=news_chc&c_id=chc |archivedate=2011-01-08 }}</ref>。

===德州遊騎兵===
2013年7月22日被交易到遊騎兵隊，小熊隊獲得Mike Olt、Justin Grimm與C. J. Edwards。

===密爾瓦基釀酒人===
稍早與釀酒人達成4年5200萬美金的協議，這名今年季末也蠻受矚目的右投手，由於去年季中被交易到遊騎兵，不在提Qualifying Offer的資格範圍內，所以與他簽約的新球團並不用付出隔年的首輪選秀權，讓他增添了一些優勢，隨著田中落腳處底定，Garza也在隔天跟釀酒人簽下四年合約，重回國聯中區。

==主要球路==
加尔萨的四缝线快速球在93—96英里/小时之间，偶尔可以达到98英里/小时，而二缝线快速球则在90—94英里/小时之间。他主要的变化球武器是曲球，在72—77英里/小时之间。另外他的滑球在85英里/小时左右，他还常用变速球来欺骗打者<ref>{{Cite web |url=http://www.saberscouting.com/2008/03/20/matt-garza-video-mechanical-breakdown/ |title=Mechanical Analysis: Matt Garza | SaberScouting |access-date=2008-05-29 |archive-url=https://web.archive.org/web/20080609014945/http://www.saberscouting.com/2008/03/20/matt-garza-video-mechanical-breakdown/ |archive-date=2008-06-09 |dead-url=yes }}</ref>。

光芒队的投手教练[[吉姆·希基|吉姆·希基]]（Jim Hickey）称赞加尔萨的资质是全队中最好的<ref>{{cite web | url = http://sportsillustrated.cnn.com/2008/writers/john_donovan/10/25/game3.garza/index.html?eref=sircrc | title = Grown-up Matt Garza has become the Rays' not-so-secret weapon | author = John Donovan | publisher = SI.com | date = 2008-10-25 | accessdate = 2008-12-15}}</ref>。

==私人生活==
加尔萨的妻子是他的高中同学Serina。他们的第一个孩子马修是在他们刚高中毕业没几天后出生的（2002年）。他们还有一个女儿Sierra（2005年出生）。
