== 马里奥高尔夫 ==
{{Vgsname|马里奥高尔夫|ja=マリオゴルフ|en=Mario Golf}}是以[[任天堂|任天堂]]公司知名游戏角色[[马里奥_(角色)|马里奥]]为主题开发的[[高尔夫|高尔夫]]游戏。由[[Camelot_Software_Planning|Camelot Software Planning]]制作，任天堂发行。目前系列总共有七款作品，分别为[[红白机|FC]]平台的《[[高尔夫_(游戏)|高尔夫]]》和《[[马里奥公开赛高尔夫|马里奥公开赛高尔夫]]》、[[任天堂64|任天堂64]]的《[[马里奥高尔夫64|马里奥高尔夫64]]》、[[Game_Boy_Color|Game Boy Color]]掌机的《[[移动高尔夫|移动高尔夫]]》、[[任天堂GameCube|任天堂GameCube]]的《[[马里奥高尔夫_家庭巡回|马里奥高尔夫 家庭巡回]]》、[[Game_Boy_Advance|Game Boy Advance]]的《[[马里奥高尔夫_GBA巡回|马里奥高尔夫 GBA巡回]]》、[[任天堂3DS|任天堂3DS]]《[[马里奥高尔夫_世界巡回|马里奥高尔夫 世界巡回]]》。最新作为任天堂3DS平台上的《马里奥高尔夫 世界巡回》。

== 马里奥和索尼克在奥林匹克运动会 ==
{{main|瑪利歐和音速小子在奧林匹克運動會系列}}

'''[[瑪利歐和音速小子在奧林匹克運動會系列|瑪利歐和音速小子在奧林匹克運動會系列]]'''是由日本{{link-en|世嘉运动研发组|SEGA Sports R&D}}于2007年首发的[[奥林匹克运动会|奥林匹克运动会]]在[[Wii|Wii]]以及[[任天堂DS|任天堂DS]]平台官方游戏。本系列的最大看点是取消了以往官方授权游戏中的真人运动员作为游戏主角的设定，继而改为由世嘉吉祥物之称的[[索尼克|索尼克]]游戏系列人物和业内巨头[[任天堂|任天堂]]旗下的[[马里奥系列|马里奥游戏系列]]人物这对梦幻组合担当。
