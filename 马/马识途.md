'''马识途'''（{{bd|1915年|1月14日|||catidx=M}}），原名'''马千木'''，曾用名'''马千禾'''，[[四川省|四川省]][[忠县|忠县]]（今[[重庆市|重庆市]]忠县）人，[[中共|中共]]退休官员，现代著名[[作家|作家]]，曾与[[巴金|巴金]]、[[张秀熟|张秀熟]]、[[沙汀|沙汀]]、[[艾芜|艾芜]]并称“蜀中五老”<ref>{{cite web|url=http://epaper.gmw.cn/zhdsb/html/2010-12/08/nw.D110000zhdsb_20101208_1-03.htm|title=川籍老作家·马识途|publisher=中华读书报|accessdate=2011-01-29}}</ref>。

== 生平 ==
马识途出生于四川省忠县[[石宝乡|石宝乡]]坪山村马家大院，在家中排行第五<ref>{{cite news|url=http://cq.cqnews.net/wtxw/2010-12/25/content_5530343.htm|title=马识途：大作家爱编打油诗|publisher=华龙网|date=2010-10-25|accessdate=2011-01-29|deadurl=yes|archiveurl=https://web.archive.org/web/20101228164855/http://cq.cqnews.net/wtxw/2010-12/25/content_5530343.htm|archivedate=2010-12-28}}</ref>。先后在忠县、[[北京|北京]]、[[上海|上海]]读小学、中学<ref name="中国文学家辞典">{{Cite book |title=中国文学家辞典·现代第一分册 |publisher=四川人民出版社 |year=1979 |unified=17118·16 |pages=21-22}}</ref>。1936年7月，马识途考入[[国立中央大学_(南京)|国立中央大学]][[东南大学|工学院]][[南京工业大学|化学工程系]]<ref name = "mashitu">{{cite news|url=http://news.ifeng.com/gundong/detail_2011_01/13/4262435_0.shtml|title=《夜谭十记》浓缩苦难与重生 马识途:子弹飞40年|publisher=凤凰网|date=2011-01-13|accessdate=2011-01-29}}</ref>，在校期间秘密加入了[[中国共产党|中国共产党]]地下党领导的南京学联小组。这一时期，他在地下活动中为了避免自己被暴露，曾使用“马千禾”的名字<ref>{{cite news|url=http://www.luobinghui.com/ld/wt/200510/5992.html|title=博学多才马识途|publisher=国立西南联合大学纪念网站|accessdate=2011-01-30|deadurl=yes|archiveurl=https://web.archive.org/web/20131216212131/http://www.luobinghui.com/ld/wt/200510/5992.html|archivedate=2013-12-16}}</ref>。1937年冬入鄂豫皖边区党训班学习<ref name="中国文学家辞典"/>。1938年3月加入中国共产党。在入党志愿书上，他将他的原名“马千木”改为“马识途”，用以表示自己“终于找到了正确的革命道路”<ref name = "mashitu"/>。后任武汉汉口职工区委委员，同时和[[胡绳|胡绳]]共同主办《大众报》，又任鄂北特委委员、枣阳县委委员，1939年调任鄂西特委书记<ref name="中国文学家辞典"/>。1940年，鄂西地下党组织遭到破坏。1941年[[皖南事变|皖南事变]]后到重庆中共南方局，不久到昆明考入[[西南联合大学|西南联合大学]]中文系<ref name = "mashitu"/>，1945年毕业。[[一·二一事件|“一二·一”事件]]后调任滇南工委。1946年调[[成都|成都]]任川康特委副书记。1949年后，历任川西区党委组织部副部长、四川省建设厅厅长、建委主任、[[中国科学院|中国科学院]]西南分院党组书记、中共中央西南局宣传部副部长、科委副主任等职。1962年加入[[中国作家协会|中国作家协会]]。1972年任四川省委宣传部副部长，1978年任中国科学院成都分院副院长、党组副书记。<ref name="中国文学家辞典"/>

== 文学创作 ==
由于长期从事地下工作，马识途与社会各阶层的人都有交往，通过这些交往他了解了民间许多奇闻异事。“他们给我摆<ref>摆：在[[四川话|四川话]]表示“谈论”。</ref> 了许多我闻所未闻、千奇百怪的[[龙门阵|龙门阵]]<ref>龙门阵：四川话里表示“闲聊”。</ref>，尤其是我接触的一些小科员，他们平时没有什么娱乐和消遣，只好三五结伙到人家里去坐冷板凳，喝冷茶，扯乱谭，摆龙门阵，自寻其乐，在他们结成的冷板凳会上，我听到了难以想象的奇闻轶事。我才深知那个社会是多么乖谬绝伦，荒唐可笑；人民的生活是多么困苦无状而又丰富多彩。”<ref name="mashitu2">{{cite news|url=http://news.xinhuanet.com/book/2010-12/21/c_12902258.htm|title=创作历时40年 马识途：“龙门阵”摆出一部巨著|publisher=新华网|date=2010-12-21|accessdate=2011-01-29|deadurl=yes|archiveurl=https://web.archive.org/web/20101227000138/http://news.xinhuanet.com/book/2010-12/21/c_12902258.htm|archivedate=2010-12-27}}</ref> 马识途被这些奇闻异事感染，产生了强烈的创作冲动。他计划选取其中十个最典型的故事来创作《[[夜谭十记|夜谭十记]]》。《夜谭十记》的书稿曾经三次被损毁，第一次是因为工作调往[[成都|成都]]前为防止[[国民党|国民党]]情报人员发现，第二次是他在成都的家被情报人员查抄，第三次是因为[[文化大革命|文化大革命]]。最终历经接近40年，《夜谭十记》终于在1982年完稿<ref name = "mashitu2"/>。2010年，其《夜谭十记》中的《盗官记》被导演[[姜文|姜文]]改编为电影《[[让子弹飞|让子弹飞]]》<ref name = "mashitu"/>，颇受观众欢迎。

除了小说，马识途在纪实文学、散文、诗词、随笔、游记、杂文等领域均有造诣<ref name = "mashitu3">{{cite web|url=http://www.cdhtlib.org/xstj_show.php?id=31|title=马识途文集|publisher=成都高新区文化指导服务中心|accessdate=2011-01-29}}</ref>。《夜谭十记》以外，马识途较为著名的作品还包括《[[清江壮歌|清江壮歌]]》、《[[沧桑十年|沧桑十年]]》、《在地下》、《找红军》、《马识途讽刺小说集》等。他被誉为“在巴蜀现当代文学史上具有承先启后的地位和作用，他是继[[郭沫若|郭沫若]]、巴金、[[何其芳|何其芳]]之后最具影响的作家”<ref>{{cite book|isbn=9787541123504|title=马识途文集1: 清江壮歌|publisher=四川文艺出版社|pages=作者简介}}</ref>。

另外，马识途在书法方面也有一定的功底，曾出版有《马识途书法集》<ref name = "mashitu3"/>。
