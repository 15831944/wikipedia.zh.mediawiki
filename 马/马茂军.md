'''马茂军'''（{{bd|1966年|4月|}}），[[安徽|安徽]][[滁州|滁州]]人，[[中央电视台|中央电视台]]《[[百家讲坛|百家讲坛]]》主讲嘉宾，现任[[华南师范大学|华南师范大学]]文学院教授、博士生导师。

== 生平简介 ==
1987年，毕业于[[安徽师范大学|安徽师范大学]]文学院，毕业后在滁州市乌衣中学任教两年。后考取[[陕西师范大学|陕西师范大学]]研究生，于1995年博士毕业，师从著名文艺理论家、古典文学专家[[霍松林|霍松林]]教授。1995年至今，任教于[[华南师范大学|华南师范大学]]，2004年评为教授。2016年11月，做客央视《百家讲坛》，主讲《大宋名相赵普》。<ref>{{cite news|url=http://www.chinawriter.com.cn/2015/2015-04-28/240782.html|title=马茂军： 读书做学问要有大格局，才能有创新|date=2015-04-28|work=信息时报|publisher=中国作家网}}</ref><ref>{{cite news|url=http://mini.eastday.com/a/161116121015828.html|title=滁州籍大学教授马茂军做客央视 主讲《百家讲坛》|date=2016-11-16|work=安徽网|publisher=东方头条}}</ref>。

主要学术兼职有：中国古代散文学会副会长、中国周必大研究会副会长。

== 主要著作 ==
* 《北宋儒学与文学》，暨南大学出版社，1999年8月出版
* 《宋代散文史论》，中华书局，2008年4月出版
* 《中国古代散文思想史》，人民出版社，2011年6月出版
* 《宋代文章学》，社会科学文献出版社，2016年2月出版
* 《大宋名相赵普》，中国财政经济出版社，2017年7月出版
