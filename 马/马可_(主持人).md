'''马可'''{{BD|1978年|2月20日|||catIdx= M马}}出生于[[湖南省|湖南省]][[株洲市|株洲市]]，毕业于湖南兵器工业职工大学（现[[湖南科技工业职业技术学院|湖南科技工业职业技术学院]]），为[[湖南电视台|湖南电视台]]著名节目主持人。<ref name="lion " >{{cite web |url=http://data.ent.sina.com.cn/star/stardetail.php?id=5834&detail=intro |title=马可 |accessdate= |publisher=新浪网 |work= |language =zh-cn}}</ref>

== 简介 ==
1978年，马可出生于[[湖南省|湖南省]][[株洲市|株洲市]]，父亲是株洲电力厂的工人，他的小学和初中都是在电力厂子弟公学校度过。<ref name="lion " />
1994年，马可考上了湖南兵器职业工业大学，大学毕业后，他被分配到父亲所在的电力厂工作，业余时间在[[株洲电视台|株洲电视台]]做兼职。<ref name="lion " />
1998年，马可到[[湖南生活频道|湖南生活频道]]做主持人，当时的工资为600元一月，他只好住在好友尹伟家。<ref name="lion " />

== 个人生活 ==
马可的妻子叫郭晓华，2010年1月28日，他们的女儿蕾蕾诞生。<ref>{{cite web |url=http://ent.people.com.cn/GB/10924993.html |title=湖南卫视主持马可喜添千金 高呼“有女万事足” |accessdate=2010年2月4日 |publisher=人民网 |work= |language =zh-cn}}</ref>
