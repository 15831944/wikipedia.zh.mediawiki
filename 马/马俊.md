'''马俊'''（Ma Jun,{{Bd|1963年||||catIdx=Jamyang}}），东乡族，祖籍[[甘肃省|甘肃省]][[东乡县|东乡县]]，[[青海省|青海省]][[海东市|海东市]][[民和回族土族自治县|民和回族土族自治县]]人，[[花儿|花儿]]演唱家。“青海省花儿艺术团”团长，<ref>{{cite web |url= https://www.v4.cc/News-3087574.html |title= 走近青海“花儿王子”马俊，了解花儿人背后的故事|accessdate= 2016年12月18日 |publisher= 微众圈网|archiveurl=https://web.archive.org/web/20180304061432/https://www.v4.cc/News-3087574.html |archivedate=2018-03-04|work= |language =zh-hans}}</ref>国家二级演员。<ref>{{cite web |url= http://news.lzu.edu.cn/c/201712/46794.html |title= 关于观看兰州大学西北花儿传承与实践基地“花儿名家”示范演唱教学音乐会的通知|accessdate= 2017年12月15日 |publisher= 兰州大学新闻网||work= |language =zh-hans}}</ref>青海省音乐家协会副主席。<ref>{{cite web |url= http://epaper.tibet3.com/qhrb/html/2013-11/29/content_69846.htm |title= 省文联六个协会完成换届|accessdate= 2013年11月29日 |publisher= 青海日报|work= |language =zh-hans}}</ref>青海省民间文艺家协会会员，青海省演艺集团花儿艺术团团长。<ref>{{cite web |url= http://www.qhddsh.com/system/2017/04/13/012286469.shtml |title= 曲浓弦和赋新歌 花儿贤孝唱心声——《总书记视察到青海》受好评|accessdate= 2017年4月13日 |publisher=  青海党的生活网|work= |language =zh-hans}}</ref>1998年被中宣部、文化部等授予“文化科技卫生三下乡先进个人”称号。<ref>{{cite web |url= http://rmrbw.xyz/read.php?tid=1159067&fpage=6 |title= 全国文化科技卫生“三下乡”活动先进集体、先进个人名单|accessdate= 1998年1月8日 |publisher= 人民日报||archiveurl=https://web.archive.org/web/20180304060046/http://rmrbw.xyz/read.php?tid=1159067&fpage=6 |archivedate=2018-03-04|work= |language =zh-hans}}</ref>“花儿王”朱仲禄先生得意门生，被人们尊称为“花儿王子”。

==经历== 
1986年，马俊在民和回族土族自治县民歌比赛中崭露头角、力拔头等，同年被青海省民族歌舞剧团破格吸收为独唱演员。随后参加由文化部和广电部举办的“全国第二届民族民间音乐舞蹈比赛”，以一曲《尕妹是才开的牡丹》荣获声乐二等奖。1989年，在“上海至巴黎世界歌唱比赛国内选赛”上，以《麻青稞》赢得“优秀奖”。<ref>{{cite web |url= http://www.cctv.com/folklore/20040722/102477.shtml |title= 花儿王子马俊：花儿吐芳 香飘河湟|accessdate= 2004年7月22日 |publisher= 央视国际||archiveurl=https://web.archive.org/web/20180304073146/http://www.cctv.com/folklore/20040722/102477.shtml |archivedate=2018-03-04|work= |language =zh-hans}}</ref>黄荣恩从乡下的一次选拔赛，将马俊推荐到省城，并在日后为其谱曲作词。冶进元和朱仲禄分别在作词方面和演唱方面给了他非常大的帮助。<ref>{{cite web |url= http://www.people.com.cn/n1/2017/0802/c347407-29444219.html |title= 2017年阿斯塔纳世博会肖像：“花儿王子”唱响中国馆|accessdate= 2017年8月02日 |publisher= 人民网|archiveurl=https://web.archive.org/web/20180304080050/http://www.people.com.cn/n1/2017/0802/c347407-29444219.html |archivedate=2018-03-04|work= |language =zh-hans}}</ref><br />1993年担任青海省演艺集团花儿艺术团团长，<ref>{{cite web |url= https://www.v4.cc/News-3087574.html |title= 走近青海“花儿王子”马俊，了解花儿人背后的故事|accessdate= 2016年12月18日 |publisher= 微众圈网|archiveurl=https://web.archive.org/web/20180304061432/https://www.v4.cc/News-3087574.html |archivedate=2018-03-04|work= |language =zh-hans}}</ref>2013年11月当选为青海省音乐家协会副主席。<ref>{{cite web |url= http://epaper.tibet3.com/qhrb/html/2013-11/29/content_69846.htm |title= 省文联六个协会完成换届|accessdate= 2013年11月29日 |publisher= 青海日报|work= |language =zh-hans}}</ref>
