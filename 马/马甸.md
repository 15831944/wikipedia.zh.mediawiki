==简介==
马甸[[元朝|元朝]]时在[[元大都|大都]]城内。大都仿[[金中都|金中都]]，将街巷以“坊”命名，全大都城分为50坊。其中在健德门内，东为善俗坊，西为可封坊。如今的马甸地区元朝时属于可封坊。明朝时，北城墙向南移5里，到达如今的[[德胜门|德胜门]]、[[安定门|安定门]]一线。原先的大都可封坊变成了郊外。明廷经常与[[蒙古|蒙古]]开展茶马交易，此处成为季节性贩马的集散地。[[清朝|清朝]][[康熙|康熙]]时期，此处形成村落。[[乾隆|乾隆]]时期，自蒙古进贡的马匹全都圈养在该地，供[[上驷院|上驷院]]官员从中挑选，挑选之后剩下的马匹大部分就地出售。这里形成了许多“马店”，是一种为马匹买卖者服务的旅店，人住店内客房，马入旅店的马圈中。[[嘉庆|嘉庆]]、[[道光|道光]]时期，部分马店南迁到关厢一带，此处变成了贩羊业的中心，羊存栏总数万余头，但是这里的地名仍然称为“马店”<ref name=ying/>。[[中华民国|中华民国]]时期，“店”字衍为“甸”（加[[儿化音|儿化音]]），称为“马甸”<ref name=ma/>。“甸”指都城郊外，比如海淀也写作“海甸”。当时这里尚有马店一、二十家<ref name=ying>{{Cite web|url =http://www.beijing.gov.cn/rwbj/lsmc/htcq/shy/t705309.htm|title =影像四合院——马甸的变迁|author =|date =|publisher =首都之窗|language =|accessdate =2017-01-23|quote =|deadurl =yes|archiveurl =https://archive.is/20140113061756/http://www.beijing.gov.cn/rwbj/lsmc/htcq/shy/t705309.htm|archivedate =2014-01-13}}</ref>。马甸南村在清朝是京营[[正黄旗|正黄旗]]骑射校场，并且设有管考[[武举|武举]]的官厅<ref name=ma>{{Cite web|url =http://bjrb.bjd.com.cn/html/2014-03/04/content_157347.htm|title =马与胡同不了情|author =朱小平|date =2014-03-04|publisher =北京日报|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170202020255/http://bjrb.bjd.com.cn/html/2014-03/04/content_157347.htm|archivedate =2017-02-02}}</ref>。

从马甸向东不远便是[[六铺炕|六铺炕]]<ref name=ma/>。海淀区[[花园路街道_(北京市)|花园路街道]]和[[西城区|西城区]][[德胜街道_(北京市)|德胜街道]]在马甸交界。自1990年代起，马甸这一地名逐渐演变为泛指跨[[西城区|西城区]]、海淀区、[[朝阳区_(北京市)|朝阳区]]的一大片地区。因马甸而得名的有：[[马甸桥|马甸桥]]、[[马甸清真寺|马甸清真寺]]、马甸小学（今名[[北京市海淀区民族小学|北京市海淀区民族小学]]）、马甸中学（后名北京市陶行知中学，2015年并入[[北方交通大学附属中学|北方交通大学附属中学]]）、马甸官厅、[[马甸路|马甸路]]、[[马甸西路|马甸西路]]、[[马甸南路|马甸南路]]、马甸西村、马甸南村、[[马甸公园|马甸公园]]、马甸月季园、马甸玉兰园等等<ref name=ying/>。

马甸是[[回民|回民]]传统聚居区，有[[马甸清真寺|马甸清真寺]]<ref name=wang/>{{Rp|115}}。还有[[北京市海淀区民族小学|北京市海淀区民族小学]]<ref>{{Cite web |url =http://hdminzuxx.bjedu.cn  |title =北京市海淀区民族小学  |author =  |date =  |publisher = 北京市海淀区民族小学 |language =  |accessdate = 2017-01-23 |quote =  }}</ref>（原名“马甸小学”，2004年更名为北京市海淀区民族小学<ref>{{Cite web |url = http://learning.sohu.com/20130513/n375761945.shtml |title =北京市海淀区特色校推荐：民族小学  |author =  |date = 2013-05-13 |publisher =  搜狐|language =  |accessdate =  |quote =  }}</ref>）。
