== 生平 ==
1959年5月4日，马星耀出生于[[北京市|北京市]][[平谷区|平谷区]]。<ref name=" laxww" /><ref name=" lhws"/>

1978年，马星耀[[高中|高中]]毕业后在[[平谷区|平谷区]]插队。<ref name=" laxww" /><ref name=" lhws"/>

1980年，马星耀到北京帆布厂工作。<ref name=" laxww" /><ref name=" lhws"/>

1981年，马星耀进入[[北京市戏曲学校|北京市戏曲学校]]读书，与[[宋丹丹|宋丹丹]]，[[王姬|王姬]]，[[梁冠华|梁冠华]]，[[王长立|王长立]]是同班同学。<ref name=" laxww" /><ref name=" lhws"/><ref name="sdd ">{{cite web |url=http://roll.sohu.com/20120327/n338973203.shtml |title= “黄胖子”马星耀去世|accessdate= 2012年|work= |publisher=搜狐网 |language =zh-cn}}</ref>

1983年，马星耀分配到[[北京人民艺术剧院|北京人民艺术剧院]]担任话剧演员。<ref name=" laxww" /><ref name=" lhws"/>

2012年3月25日下午，马星耀在[[北京市|北京市]][[北京航天总医院|北京航天总医院]]因癌症病逝。<ref name=" mxys" /><ref name=" lhws"/>

== 个人生活 ==
日常生活中，马星耀性格宽容幽默，助人为乐。<ref name=" mxys" /><ref name=" lhws"/> 
马星耀的生活作息时间很有规律，晚上一般喝点小酒就睡觉。<ref name=" mxys" /><ref name=" lhws"/>
