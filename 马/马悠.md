'''马悠'''（{{lang-de|Josef Margraf}}，{{bd|1953年|4月3日|2010年|1月26日}}），是一名[[德国|德国]][[生态学家|生态学家]]，一生致力于保护[[东南亚|东南亚]]和[[云南|云南]][[西双版纳|西双版纳]]的[[热带雨林|热带雨林]]。在云南的13年间，成立了西双版纳天籽生物多样性开发中心，致力于当地热带雨林的修复和再造工作。他与妻子[[李旻果|李旻果]]育有两个女儿李林妲和李宛妲，并有一养女王艺洁<ref>{{cite web |title=揭秘快女王艺洁两个干妹妹身世 |url=http://ent.163.com/11/0818/10/7BNV167M00032DGD.html |website=网易娱乐 |accessdate=2018-10-29}}</ref>。他的家人在他死后依然坚持着热带雨林的保护工作。<ref name="YC">{{cite journal |author1=王大骐 |title=马悠的遗产 |journal=南方人物周刊 |date=2010-06-18}}</ref>

==生平==
马悠出生在[[西德|西德]][[慕尼黑|慕尼黑]]的一个小镇，是贵族大公的后裔，父亲是一名[[纳粹|纳粹]]分子。他之後在[[蓋梅爾斯海姆|蓋梅爾斯海姆]]長大，並後來在[[艾希斯特|艾希斯特]]的[[文法學校|文法學校]]就讀。1974年至1979年间，馬悠在[[斯图加特|斯图加特]]的{{link-de|霍恩海姆大学|Universität Hohenheim}}学习动物学<ref>{{cite web |url=https://www.researchgate.net/publication/234006846_Okologische_Studien_der_temporaren_Flachseen_auf_der_%27Giara_di_Gesturi%27_in_Sardinien |title=Über die Ökologie von temporären Süßwasserflachseen auf der Giara die Gesturi in Sardinien |access-date=2018-11-04 |archive-url=https://web.archive.org/web/20180709192122/https://www.researchgate.net/publication/234006846_Okologische_Studien_der_temporaren_Flachseen_auf_der_'Giara_di_Gesturi'_in_Sardinien |archive-date=2018-07-09 |dead-url=no }}</ref>，于1985年获得了生态学的硕士和热带农业学的博士学位，随后创办了以自己名字命名的科学书籍出版公司。 <ref name="YC"/>

从1989年开始，他在菲律宾创建了群落式雨林再造模式，被称为“雨林再造之父”，改写了菲律宾的国策和林学院教材，并于1997年获菲律宾政府总统奖。当年菲律宾的[[萊特島|萊特島]]雨林再造经过近20年的建设，得到欧盟的最佳评估，成为生物多样性雨林种植模式的样板。1997年，马悠受德国政府的委派来到中国，担任中德政府间合作的“西双版纳热带雨林恢复和保护项目”专家组组长达6年，还专门为欧盟设计了援华项目《中国西部生物多样性保护项目》，资金总额高达5100万欧元。 <ref name="YC"/>

1999年，马悠和李旻果在一次秘鲁大使招待晚宴上相识并于2000年结婚。婚后马悠和李旻果来到西双版纳购买下15亩橡胶林地，再把橡胶树砍掉，开始按自然生态的模式重建雨林，种下了300多种植物。二人的女儿林妲于2001年8月出生，二女儿宛妲于2003年6月出生。2004年，马悠夫妇成立了天籽生物多样性开发中心。<ref name="HX">{{cite web |title=李旻果再造雨林 |url=http://news.hexun.com/2014-08-07/167317511.html |website=和讯新闻 |accessdate=2018-11-04}}</ref>2005年，李旻果和马悠从西双版纳[[勐海县|勐海县]]老班章村民手里承包了6666.6亩土地，成立了中国首个民间生物多样性保护区——“天籽老班章生物多样性茶园保护区”；马悠为此卖掉了德国故乡的财产和自己的保险。<ref name="HX"/><ref name="YC"/>

2010年1月，马悠因心脏病突发而在[[西雙版納傣族自治州|西雙版納傣族自治州]][[景洪市|景洪市]]內去世，享年56歲。<ref>{{cite web |title=坚守的力量 |url=http://xinwen.china-flower.com/index.php?m=wap&siteid=1&a=show&catid=108&typeid=7&id=182703 |website=中国花卉报 |accessdate=2018-11-03}}</ref>
