'''马克思学说研究会'''1920年3月在[[北京大学|北京大学]]创立，是中国“最早研究与宣传马克思主义的理论团体”，对[[马克思主义|马克思主义]]在中国的传播有开拓性贡献。{{r|a|page=93}}

== 称谓 ==
因对[[卡尔·马克思|卡尔·马克思]]的译名不同，有“马克斯学说研究会”、“马克思学说研究会”、“马克司学说研究会”三种称呼。研究会会员陈仲瑜回忆说：“为了避免反动政府的干扰，不用‘主义’而用‘学说’二字，可说是策略。”研究会启事多数情况下只称“马克思学说研究会”，有时称“北京马克思学说研究会”。{{r|a|page=98}}

== 历史 ==
1918年末，[[李大钊|李大钊]]组织了一个马克思主义研究团体，当时称为“马尔格士学说研究会”（或“马尔克斯学说研究会”）。{{r|a|page=96}} 到1919年12月被[[社会主义研究会|社会主义研究会]]取代，一般认为和后来的马克思学说研究会无实际关联。{{r|a|page=97}}

1920年3月马克思学说研究会成立，1921年11月17日《北大日刊》刊载《发起马克斯学说研究会启事》，公开宣布研究会的存在。{{r|a|page=97}} 

最初发起人一共十九人：[[高崇焕|高崇焕]]、[[王有德_(1897年)|王有德]]、[[邓中夏|邓中夏]]、[[吴汝明|吴汝明]]、[[罗章龙|罗章龙]]、[[黄绍谷|黄绍谷]]、[[王复生|王复生]]、[[黄日葵|黄日葵]]、[[李骏|李骏]]、[[杨人杞|杨人杞]]、[[李梅羹|李梅羹]]、[[吴溶沧|吴溶沧]]、[[刘仁静|刘仁静]]、[[范鸿劫|范鸿劫]]、[[宋天放|宋天放]]、[[高尚德|高尚德]]、[[何孟雄|何孟雄]]、[[朱务善|朱务善]]、[[范齐韩|范齐韩]]<ref>{{Cite book|title=百年中国记忆 文史资料百部经典文库 回忆五四运动|last=全国政协文史和学习委员会|first=|publisher=北京：中国文史出版社|year=2017.01|isbn=978-7-5034-8068-3|location=|pages=45}}</ref>。研究会的图书室取名“亢慕义斋”，意即“共产主义小室”。{{r|a|page=99}} 多数学者认为研究会存在到1926年，少数学者认为研究会于1925年解散。{{r|a|page=100}}

== 评价 ==
* 多数学者认为：研究会是“中国最早学习研究马克思主义的团体”。{{r|a|page=99}}
* [[李丹阳|李丹阳]]认为，研究会是中国共产主义组织的源头。[[张申府|张申府]]（研究会发起人之一）评价说，研究会为中国共产党打下理论、组织基础。{{r|a|page=100}}
* [[史华慈|史华慈]]质疑研究会的活动仅仅只是研究马克思学说，而非宣传马克思主义。{{r|a|page=100}}
