'''马里共和国'''（{{lang-fr|République du Mali}}） 通稱'''馬利'''，是位於[[西非|西非]]的[[内陆国家|内陆国家]]，向北与[[阿尔及利亚|阿尔及利亚]]、向东与[[尼日尔|尼日尔]]、向南与[[布基纳法索|布基纳法索]]和[[科特迪瓦|科特迪瓦]]、向西南与[[几内亚|几内亚]]、向西与[[毛里塔尼亚|毛里塔尼亚]]和[[塞内加尔|塞内加尔]]接壤，是西非面积第二大的国家。它的北部边界在[[撒哈拉沙漠|撒哈拉沙漠]]的中心，大多数人集中在南部，[[尼日尔河|尼日尔河]]和[[塞内加尔河|塞内加尔河]]源于这裡。马里过去也被称为[[法属苏丹|法属苏丹]]，它的名字来源于[[马里帝国|马里帝国]]。

== 國名 ==
[[法語|法語]]名稱為「'''''République du Mali'''''」，簡稱「'''''Mali'''''」；英語名稱為「'''''Republic of Mali'''''」，簡稱「'''''Mali'''''」。華語名稱，[[台灣|台灣]]稱為「'''馬-{}-利共和國'''」，簡稱「'''馬-{}-利'''」；[[中國大陸|中國大陸]]稱為「'''馬-{}-里共和國'''」，簡稱「'''馬-{}-里'''」，[[新加坡|新]]、[[馬來西亞|馬]]、[[香港|港]]、[[澳門|澳]]與中國大陆相同。

== 历史 ==
{{main|馬利歷史}}
历史上在[[萨赫勒|萨赫勒]]地带（包括今天的马里）有过一系列帝国，其中包括[[加纳帝国|加纳帝国]]、[[马里帝国|马里帝国]]和[[桑海帝国|桑海帝国]]等。[[廷巴克图|廷巴克图]]是这些帝国在[[跨撒哈拉贸易|跨撒哈拉贸易]]中的一个关键城市和学术中心。1591年桑海帝国受[[摩洛哥|摩洛哥]]入侵不断虚弱下来。

从1880年开始[[法国|法国]]入侵马里。法国殖民地法属苏丹还包括马里周边的国家。1959年初，马里与[[塞内加尔|塞内加尔]]组成[[马里联邦|马里联邦]]。1960年6月20日马里联邦从法国独立，数月后塞内加尔退出联邦。同年9月22日在[[莫迪博·凯塔|莫迪博·凯塔]]领导下马里共和国退出[[法兰西共同体|法兰西共同体]]。

从其独立到1991年一系列独裁者统治马里。1991年的反政府抗议导致政变、一个过渡政府和一部新宪法。1992年[[阿尔法·乌马尔·科纳雷|阿尔法·乌马尔·科纳雷]]在马里第一次多党派、民主选举中当选为总统，1997年他再次当选。他加强政治和经济的改革和打击贪污。

2002年，[[阿马杜·图马尼·杜尔|阿马杜·图马尼·杜尔]]当任马里总统，杜尔在1991年的民主运动中是一个关键人物。2007年4月29日，举行总统选举。<ref>张崇防，韦巍，[http://news.xinhuanet.com/world/2007-04/29/content_6045359.htm 马里举行总统选举]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}，新华网</ref>杜尔连任。<ref>张崇防，韦巍，[http://news.xinhuanet.com/world/2007-05/04/content_6056181.htm 马里现任总统杜尔成功连任]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}，新华网</ref>

2012年1月，[[圖瓦雷克人|圖瓦雷克人]]在北方發動武裝叛亂，[[马里北部冲突|马里北部冲突]]開始。2012年3月21日，一批士兵在首都巴馬科发动政变，推翻杜尔，宣布中止宪法、解散国家机构。在国际社会压力下，政变军人4月6日与[[西非国家经济共同体|西非国家经济共同体]]签署协议，同意交权，由国民议会议长出任临时总统。4月8日杜尔辞职。<ref>[http://news.163.com/12/0410/02/7UMRNM3B00014AED.html 马里总统宣布辞职]，网易新闻</ref>在南方局勢不明的時候，圖阿雷格分離主義者在伊斯蘭激進分子幫助下趁機控制了馬里北部地區。稍後強硬派伊斯蘭激進分子撃敗圖阿雷格分離主義者，控制了馬里北部。2013年1月，伊斯蘭反政府武裝向南方進軍，法國出兵協助馬里政府作戰，迅速收復北方所有主要城市。

== 政治 ==
《马里共和国宪法》规定马里为一个[[多党制|多党制]][[民主|民主]][[共和国|共和国]]，唯独基于[[民族|民族]]、[[宗教信仰|宗教信仰]]、[[地区|地区]]或性别而成立的党派被禁止。国家[[立法机关|立法机关]]是[[国民议会_(马里)|国民议会]]，目前国会有147名成员。国会议员按各地区人口数目制定。政府执政期为5年。

== 地理 ==
[[File:Mali_sat.png|right]]
马里大部分为海拔约300米的[[台地|台地]]，最东部和西部有一些砂岩低山和陡崖，最高点海拔833米；北部属于[[撒哈拉沙漠|撒哈拉沙漠]]，中、南部为热带半沙漠和草原。

马里约一半的面积是[[沙漠|沙漠]]或半沙漠。北部从[[阿尔及利亚|阿尔及利亚]]伸入的[[阿哈加尔山脉|阿哈加尔山脉]]形成一个高地。南部和中部是尼日尔河宽广的低地。

马里的气候从热带温暖的气候一直到撒哈拉沙漠的沙漠气候。南部年降雨量可达1000毫米，北部有些地方终年不雨。南部是潮湿的草原，向北演变成灌木草原、半沙漠和沙漠。

== 居民 ==
马里有约20个不同的民族，其中人数最多的是[[班巴拉人|班巴拉人]]。这些不同的民族的语言和文化各不相同，但是和平共处。东北部苏丹地区的居民很早就与北非的穆斯林交往，大多数信仰[[伊斯兰教|伊斯兰教]]。同时国内也有许多非洲传统的信仰。马里非常重视它多种的文化历史。

马里人约只有三分之二的人口有干净的水源。平均期望寿命为58歲。人均[[国民收入|国民收入]]只有192欧元。

官方语言是[[法语|法语]]，但许多马里人将法语看作外语，马里有许多民族语言，大多数马里人懂多种民族语言。在教育能力上，根據2010年時的統計超過15歲的人口中僅有31.1%可以讀寫，受過教育的馬利人還是以男性居多，佔了男性人口比例的43.4%，相比之下女性僅有20.3%{{R|WorldFactBook}}。

在马里北部过去经常发生[[圖瓦雷克人|圖瓦雷克人]]的独立运动。2012年4月6日，[[阿扎瓦德民族解放运动|阿扎瓦德民族解放运动]]声称完全控制了其主张的国土，并宣布[[阿扎瓦德|阿扎瓦德]]独立<ref>{{cite news|url=http://www.mnlamov.net/component/content/article/169-declaration-dindependance-de-lazawad.html|title=DÉCLARATION D’INDÉPENDANCE DE L'AZAWAD|date=2012-04-06|accessdate=2012-04-07|deadurl=yes|archiveurl=https://www.webcitation.org/6BTul59wU?url=http://www.mnlamov.net/component/content/article/169-declaration-dindependance-de-lazawad.html|archivedate=2012-10-17}}</ref>。
人口分布要受到自然环境的影响，对于马里来讲，亦是如此。
从地形来看，马里北部为广大沙漠，中部和南部为平原和台地。
从气候来看，马里由北向南分为热带沙漠、热带草原和热带雨林3种气候区域。
也就是说马里人口多分布在南部，从自然环境来讲，这里相对降水丰富一些。

== 宗教 ==
90%的人信仰伊斯兰教，5%的人信仰[[拜物教|拜物教]]，5%的人信仰[[基督教|基督教]]。其他伊斯兰教国家对马里的经济有一定的支持。

== 经济 ==
縱使馬里在近十多年間，每一年都擁有著5%以上的GDP增幅，不過馬里目前仍然屬於[[最不發達國家|最不發達國家]]之一，2018年馬里的人均收入(國際匯率)只有約900美元。該國經濟支柱為農業，較少工業。在農業上，农用地佔了全國面積的2%，同时80%的劳动力在农业工作。在尼日尔河和塞内加尔河流域以及在南方多雨地区农业用地非常密集。种植物有[[花生|花生]]、[[玉米|玉米]]、[[高粱|高粱]]和[[棉花|棉花]]等。

北方由于干旱收获不能获得保证，但由于人口增加所造成的压力到年降雨量200毫米的地方也有很多农田，主要作物是[[小麦|小麦]]和绿色饲料。

近年，馬里開始出口水果到西歐製作果汁，諸如：[[芒果|芒果]]、[[薑|薑]]、[[洛神花|洛神花]]等<ref>[http://programme.tvb.com/foodandtravel/globaldrin必k/episode/1/#page-1 《全球飲勝》第一集]，[[高清翡翠台|高清翡翠台]]，2012-9-23 17:30~18:30</ref>。
