'''马拉维共和国'''（[[齊切瓦語|齊切瓦語]]：'''{{lang|ny|Dziko la Malaŵi}}'''；[[英語|英語]]：'''{{lang|en|Republic of Malawi}}'''），簡稱'''馬拉威'''，是一個位於[[非洲|非洲]]東南部的[[內陸國家|內陸國家]]，鄰接[[贊比亞|贊比亞]]、[[莫桑比克|莫桑比克]]及[[坦桑尼亞|坦桑尼亞]]。國土位於[[緯度|南緯]]9°45'至17°16'、[[經度|東經]]32°35'-35°24'之間。

其[[首都|首都]][[利隆圭|利隆圭]]位於马拉维的中部。

== 历史 ==
{{main|马拉维历史}}
[[十九世纪|十九世纪]]中叶，[[葡萄牙|葡萄牙]]、[[英国|英国]]殖民者相继侵入；1891年沦为英国保护国；1904年英国政府直接管辖，名[[尼亚萨兰|尼亚萨兰]]；1953年成为[[中非联邦|中非联邦]]一部分；1964年7月6日独立，改名馬拉威。

== 地理 ==
{{main|马拉维地理}}
{{Image|zh-hans=Malawei-ditu-zh.png|zh-hant=Malawei-ditu-zh-tw.png|left|thumb|175px|马拉维地图}}
马拉维位于非洲东南部，南北有[[東非大裂谷|東非大裂谷]]贯穿，谷底就是[[马拉维湖|马拉维湖]]，是非洲第三大[[湖|湖]]，占马拉维总面积的20％。[[希雷河|希雷河]]出自马拉维湖南端，流经马拉维南部地区，最后注入400公里以南的[[赞比西河|赞比西河]]。裂谷的东西面为海拔达900-1200米的高原。北部地区的[[尼卡高原|尼卡高原]]高达2600米。南部的[[希雷高地|希雷高地]]平均海拔600－1600米，不过也有[[松巴高原|松巴高原]]高2130米，和[[木蘭吉山|木蘭吉山]]达3050米。希雷河下游地势较低，海拔只有60－90米。

马拉维是非洲[[撒哈拉沙漠|撒哈拉沙漠]]以南人口密度最高的国家之一，首府[[利隆圭|利隆圭]]人口为400,000，最大城市[[布兰太尔_(马拉维)|布兰太尔]]人口在1998年达到500,000。

马拉维湖因其长365[[英里|英里]]、宽52英里而俗称“日历湖”。

马拉维设有5座[[国家公园|国家公园]]：

* [[卡松古国家公园|卡松古国家公园]]（{{lang|en|Kasungu}}）
* [[令圭国家公园|令圭国家公园]]（{{lang|en|Lengwe}}）

=== 氣候 ===
马拉维差不多整個國家都處於高山之上，所以大部份地區的氣候垂直差异显著，为[[热带干湿季气候|热带干湿季气候]]区的[[高地气候|高地气候]]，普遍來說高山平原的氣溫十分溫和和穩定，但冬天的氣溫可以低至[[攝氏|攝氏]]4度。而[[盆地|盆地]]的氣溫較為悶熱，最熱的時候可以攝氏39度，而每年的降雨量也因地理環境而有所差別，平均由600mm至3000mm不等。通常雨季由11月開始至5月，而5月至11月的降雨量很低。馬拉威因處於[[高原|高原]]地帶，所以極少受到[[飓風|飓風]]、[[洪水|洪水]]等威脅。

== 經濟 ==
{{main|馬拉维經濟}}
{{seealso|马拉维糧食危機}}
马拉维曾被[[聯合國|聯合國]]評為世界上最不發達的國家之一，全國约有55%的人生活在貧窮線以下，[[人均国民生產總值|人均国民生產總值]]只有600[[美元|美元]]，是全世界最低收入的一群。本国政府長期舉債度日，國債佔[[國民生產總值|國民生產總值]]的228.3%。因受[[愛滋病|愛滋病]]及高生育率影響，馬拉威劳动力人口比例過低，大概只有50%的人处于15-64歲之間。加上人民的教育程度很低，國內低技術勞工過剩，但同時嚴重缺乏高技術人才。

该国經濟長期倚賴于農產品出口，农业人口超過90%，因而國家經濟很容易受農產品的國際價格影響。農民主要种植烟草、棉花、玉米等，是非洲主要烟草生产国之一，烟草出口占国家外汇收入70%。如果農產品收成時候價格回落，農民只能以低價出售。新總統最近推出多項措施改善國家經濟，包括發展[[市場經濟|市場經濟]]、反貪污及加強[[國民教育|國民教育]]。

马拉维的人口大約有1840萬人（2017年九月），超過80%的人口居住在未發展的農村。马拉维因受[[艾滋病|艾滋病]]影響，人口大幅下降，而平均壽命也因此下降。根據2003年的資料，成人的艾滋病感染率達14.2%，即差不多有90萬名[[艾滋病|艾滋病]]患者，而馬拉维人的平均壽命只有36.97歲。马拉维的生育率頗高，平均每名已婚婦女會誕下5.98名嬰孩，但艾滋病令马拉维的嬰兒夭折率達10%，即每1000名新生嬰兒就有103.22名嬰兒死亡。

== 文化 ==
马拉维的官方語言是[[齊切瓦語|齊切瓦語]]及[[英語|英語]]。大部份馬拉威人信奉[[天主教|天主教]]，另外大概有10%的人信奉[[回教|伊斯兰教]]。馬拉威大約有40%的人是文盲，當中有一半以上的文盲是婦女。

*語言：[[齐切瓦语|齐切瓦语]]57.2%、Chinyanja 12.8%、Chiyao 10.1%、Chitumbuka 9.5%、Chisena 2.7%、Chilomwe 2.4%、Chitonga 1.7%、3.6%其他（CIA factbook）
*宗教信仰：79.9%[[天主教|天主教]]、12.8%[[回教|回教]]、3%其他、4.3%沒有宗教信仰
*文盲比率：男：23.9%,女：50.2%

=== 教育 ===
政府提供免費的小學教育。在[[高等教育|高等教育]]方面，國內設有兩所主要的大學分別為[[馬拉威大學|馬拉威大學]]及[[姆祖祖大学|姆祖祖大学]]。其中以马拉维大學的歷史最為悠久且最具規模，此外另有“馬拉威管理學院”（{{lang|en|Malawi Institute of Management}}）及“馬拉威企業發展學院”（{{lang|en|Malawi Entrepreneurial Development Institute}}）均有提供管理人才的訓練課程。

== 政治 ==
马拉维是眾非洲國家中其中一個政局最穩定的國家，也是少數沒有受[[內戰|內戰]]影響的非洲國家，該國於1966年7月6日改制共和，為[[總統制|總統制]]。[[馬拉威總統列表|馬拉威總統]]為國家元首、最高行政首長及軍事統帥，以選舉方式產生，任期五年，每五年舉行一次普選。總統下設一至兩位[[副總統|副總統]]，內閣閣員由總統直接任免。[[國會|國會]]採[[一院制|一院制]]，稱「國民議會」。地方政府則分設北、中、南，全國共有26個縣。

2012年4月5日，时任总统[[宾古·瓦·穆塔里卡|宾古·瓦·穆塔里卡]]因突发心脏病去世。6日晚政府称副总统[[乔伊斯·班达|乔伊斯·班达]]因脱离执政党，不具备接任总统资格。但在7日，乔伊斯·班达宣誓就任，成为马拉维历史上首位女总统。2013年10月，马政坛曝出公职人员贪污公款的“现金门”丑闻，欧盟、挪威、国际货币基金组织宣布暂停对马预算援助，至今尚未恢复。2014年總統選舉中班達被彼得·穆塔里卡擊敗，穆塔里卡成為第5任馬拉威總統。

== 外交 ==
1966年7月11日，马拉维曾同[[中华民国|中华民国]]建交，但2007年12月28日该国转而与[[中华人民共和国|中华人民共和国]]建立外交关系。

== 交通 ==
* 公路：連絡鄰國的主要道路有：往[[莫三比克|莫三比克]]的[[貝拉|貝拉]]、[[南非|南非]]的[[德班|德班]]；往[[坦桑尼亞|坦桑尼亞]]的[[三蘭港|三蘭港]]；國內客貨運班車主要往來於[[布兰太尔|布兰太尔]]和[[里朗威|里朗威]]之間。
* 空運 
** 航空公司：[[馬拉威航空|馬拉威航空]]公司提供區域及國際航班，可前往之目的地包括[[奈洛比|奈洛比]]、[[哈拉雷|哈拉雷]]、[[路沙卡|路沙卡]]、[[布兰太尔|布蘭岱]]和[[約翰尼斯堡|約翰尼斯堡]]等等；歐洲及非洲航空公司在里朗威國際機場每天都有班機起降，包括[[英国航空|英國]]、[[肯尼亚航空|肯亞]]、[[荷兰皇家航空|KLM]]、[[埃塞俄比亚航空|衣索匹亞]]、[[南非航空公司|南非]]、辛巴威和尚比亞等航空公司。
** 機場：目前有首都[[里朗威|里朗威]]和[[布兰太尔|布兰太尔]]兩座大型國際機場，均設有大規模的報稅和冷藏倉庫可供使用。
* 鐵路：除了國內鐵路線外，還有國際鐵路運輸通往莫三比克的[[納卡拉|納卡拉]]港（Nacala）。
**2006年7月4日：原由[[台灣鐵路管理局|台灣鐵路管理局]]所有的四輛[[台鐵R20型柴電機車|R20型柴電機車]]（編號：R56、R57、R58、R59）贈送給馬拉威，但在台灣和馬拉威斷交後維修技術支援中斷，現只剩R58仍可使用。
* 電信：電話可以國際直撥至世界主要的城市，傳真、電子郵件及資料都可以從馬拉威傳送至世界各地。在境內的主要城市都可使用行動電話。
