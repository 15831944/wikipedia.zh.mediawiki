'''马丁·布赖恩·马尔罗尼'''（'''{{lang|en|Martin Brian Mulroney}}'''）（{{bd|1939年|3月20日|catIdx=M}}），是第18任[[加拿大总理|加拿大总理]]。他的任期从1984年9月17日到1993年6月25日。

== 簡歷 ==
马尔罗尼出身于[[魁北克省|魁北克省]][[贝科莫|贝科莫]]一个[[爱尔兰|爱尔兰]]裔家庭，青年时期曾就读于[[:en:St._Francis_Xavier_University|圣弗朗西斯·格扎维埃大学]]。在那里他参加了保守党，并与当时的保守党领袖[[约翰·迪芬贝克|约翰·迪芬贝克]]建立了联系。大学毕业后，马尔罗尼进入魁北克城的[[拉瓦尔大学|拉瓦尔大学]]法学院，被迫重操[[法语|法语]]，因而熟练掌握了加拿大的两种主要语言。他曾任魁北克北岸与拉布拉多铁路公司董事和总经理、联合省保险公司董事，并在[[加拿大帝国商业银行|加拿大帝国商业银行]]、[[汉纳煤炭公司|汉纳煤炭公司]]、[[标准广播公司|标准广播公司]]等企业任职。1965年马尔罗尼成为[[蒙特利尔|蒙特利尔]]市一位颇负盛名的律师。1973年他同比他小13岁，出身于[[南斯拉夫|南斯拉夫]]移民家庭的米拉结婚，婚后生育了3个子女。

1977年后，马尔罗尼历任[[:en:Iron_Ore_Company_of_Canada|加拿大铁矿公司]]总裁、董事兼总经理，加拿大帝国商业银行及其他12家公司董事等职。

== 總理 ==
1983年马尔罗尼复出从政，并当选为进步保守党新领袖，并通过补缺入选众议院。

在1984年9月的大选中，进步保守党获得211个议会席位，达到加拿大國會历史上的空前絕對多数，取得超過三分之二的席次，自由黨議席大減至40店，結束自由黨長達二十年的執政。1984年9月17日，45岁的马尔罗尼成为加拿大第18任总理。新的保守黨政府强调民族和解，承认魁北克省的特殊性，于1987年6月与10省政府达成有关承认魁北克省特殊社会地位的[[:en:Meech_Lake_Accord|米奇湖修宪协议]]。1988年1月2日马尔罗尼总理和[[羅納德·里根|羅納德·里根]]总统分别签署加、美自由贸易协定，後更支持[[北美自由貿易協議|北美自由貿易協議]]。

1988年10月1日马尔罗尼宣布解散议会。11月21日加拿大举行第三十四届联邦大选，马尔罗尼成功连任总理，但保守黨總席次下降至169席。

== 修宪受挫及下台 ==
1990年6月米奇湖修宪协议因未能在规定的3年内获得全部10省议会的批准而宣告夭折。魁北克省[[法裔加拿大人|法裔]]的独立情绪随之再度高涨。马尔罗尼与10省2地区和土著人代表经过马拉松式的艰苦谈判，终于在1992年8月达成一项新的修宪协议。为保证该协议在联邦和各省议会顺利通过，马尔罗尼决定以全国公民投票方式对该决议进行表决，结果因55％的选民投票反对而遭否决。这是继1990年米奇湖协议夭折后，马尔罗尼政府在修宪问题上再次受挫。

随后的民意测验结果表明，马尔罗尼的选民支持率降至12％，远远落在自由党领袖的后面，保守党的选民支持率也只有19％。为使保守党在1993年秋天的大选中获胜，马尔罗尼于1993年2月宣布辞职。6月，加拿大国防部长[[金·坎贝尔|金·坎贝尔]]在执政的保守党领袖选举的第二轮投票中获胜，当选为该党的新领袖，并接替前领袖马尔罗尼出任政府总理，成为加拿大第一位担当此任的女性。9月8日，坎贝尔宣布解散[[加拿大國會|國會]]，定于10月25日举行联邦大选。选举结果自由党大获全胜。
