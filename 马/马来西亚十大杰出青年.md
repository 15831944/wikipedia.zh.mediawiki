'''马来西亚十大杰出青年'''奖（简称十大杰青）是由马来西亚[[国际青年商会|国际青年商会]]举办的一项评选活动<ref>{{cite web |title=十大杰青奖勉励幕后功臣提名人陪得奖者领奖 |url=http://www.enanyang.my/news/20180331/%E5%8D%81%E5%A4%A7%E6%9D%B0%E9%9D%92%E5%A5%96%E5%8B%89%E5%8A%B1%E5%B9%95%E5%90%8E%E5%8A%9F%E8%87%A3br-%E6%8F%90%E5%90%8D%E4%BA%BA%E9%99%AA%E5%BE%97%E5%A5%96%E8%80%85%E9%A2%86%E5%A5%96/ |publisher=e南洋商报 |accessdate=2018-08-15}}</ref><ref>{{cite web |title=十大杰青奖40强．谢佳见郭美君入围 |url=http://www.sinchew.com.my/node/1670673 |website=星洲网 |accessdate=2018-08-15}}</ref><ref>{{cite web |title=2017年马十大杰出青年奖 公开提名角逐杰出青年荣誉 |url=https://www.intimes.com.my/index.php/2013-09-08-02-08-40/item/45285-2017 |accessdate=2018-08-15}}</ref>，旨在每年选出十名18-40岁的获奖者。举办方称活动的宗旨是“表扬在专业工作领域上有卓越表现及对社会做出贡献的18-40岁青年，使他们成为青年人的学习楷模”。该活动自1985年起成为一项年度活动。获奖的杰青将会代表马来西亚参与{{link-en|世界十大杰出青年奖|Ten Outstanding Young Persons of the World}}的评选。
