'''马璧'''（{{bd|1912年|4月4日|1985年|10月8日}}），字'''光奎'''、号'''默炉'''，[[湖南|湖南]][[湘潭|湘潭]]人，中国[[政治家|政治家]]、[[外交家|外交家]]、[[哲学家|哲学家]]、[[诗人|诗人]]。

==生平==
1912年4月4日，马璧出生于[[湘潭县|湘潭县]][[茶恩寺镇|茶恩寺镇]]一个普普通通的农民家庭。1932年，马璧在小学教书。1933年，马璧开始写作生涯。在[[抗日战争|抗日战争]]期间，1942年马璧任湖南省教育厅专员；后曾任湖南省民政厅视察。马璧在[[中国|中国]][[文学|文学]]、[[哲学|哲学]]、[[诗词|诗词]]方面造诣颇深，而且在[[美术|美术]]、外文方面也有较高水平，曾拜[[齐白石|齐白石]]为师学画。1946年，马璧任[[中华民国国防部|国防部]]新闻局专员。同年，获聘为中国国民党中央文化运动委员会美术委员会为委员。1948年，被[[程潜|程潜]]任用为[[长沙绥靖公署|长沙绥靖公署]]政工处专员、主任秘书。<ref name=" 湖湘人物：马璧">{{cite web |url=http://www.library.hn.cn/hxrw/xdrw/xjlwrs/MB.HTM|title=湖湘人物：马璧|accessdate=|work=|publisher=湖南图书馆官方网|language=zh-hans}}</ref>

1949年，马璧去[[香港|香港]]静观时局。1951年，马璧赴[[台湾|台湾]]后，历任[[总政治作战部|总政治作战部]]新中国出版社编辑主任、特约研究委员，政工干校（后1970改为政治作戰學校,2006改为國防大學政治作戰學院）系主任，还曾任[[中国文化学院|中国文化学院]]、[[天主教辅仁大学|辅仁学院]]等6所高等院校教授。1977年自政治作戰學校退休后，又任[[台湾警备总司令部|台湾警备总司令部]]顾问、总政治部特约研究员。<ref name="kmt">{{cite book|title=中国国民党百年人物全书|year=2005|publisher=团结出版社}}</ref><ref name=" 湖湘人物：马璧"/>

1981年9月，[[全国人大常委会委员长|全国人大常委会委员长]][[叶剑英|叶剑英]]向台湾当局发出九点建议。1981年11月3日，马璧叛逃到大陆。此后出任[[厦门大学|厦门大学]]台湾研究所研究员、[[孙中山|孙中山]]研究会理事。1981年11月20日，获增补为[[第五届全国政协委员|第五届全国政协委员]]，1982年12月23日获增补为[[第五届全国政协常委|第五届全国政协常委]]。1983年，当选为[[第六届全国人大代表|第六届全国人大代表]]、全国人大常委会委员。1983年，参加[[中国国民党革命委员会|中国国民党革命委员会]]，任中国国民党革命委员会中央常务委员。<ref name=shq/><ref name="kmt" />1982年12月，马璧受到[[邓小平|邓小平]]的接见。<ref>{{cite web |url=http://agzy.youth.cn/mzh/jng/dxp/yrxm/201006/t20100629_1272745.html|title=1982年12月，邓小平会见原国民党军官马璧教授|accessdate= 2010-06-29 |work=|publisher=孙中山纪念馆|language= zh-hans}}</ref>

1983年4月，马璧秘密进入[[香港|香港]]，凭着政治社交才能，圆满完成了港行计划。<ref name=" 湖湘人物：马璧"/>晚年，他为[[湘潭|湘潭]]筹建[[齐白石故居_(湘潭)|齐白石纪念馆]]费尽心力，曾为此抱病上书[[中共中央宣传部|中共中央宣传部]]。在[[全国人大常委会|全国人大常委会]]任职期间，提出了著名的“有才便有能，多才便多能，无才便无能”的选拔人才观点。<ref name=" 湖湘人物：马璧"/>

1985年10月8日，马璧病逝于[[湖南|湖南]][[长沙|长沙]]，享年73岁。骨灰安葬于[[北京|北京]][[八宝山革命公墓|八宝山革命公墓]]。[[胡耀邦|胡耀邦]]、[[邓小平|邓小平]]、[[赵紫阳|赵紫阳]]、[[彭真|彭真]]、[[邓颖超|邓颖超]]、[[万里|万里]]、[[习仲勋|习仲勋]]、[[杨尚昆|杨尚昆]]、[[胡启立|胡启立]]、[[陈丕显|陈丕显]]、[[王震|王震]]、[[刘澜涛|刘澜涛]]、[[程子华|程子华]]、[[彭冲|彭冲]]、[[王任重|王任重]]、[[朱学范|朱学范]]、[[周谷城|周谷城]]、[[荣毅仁|荣毅仁]]、[[杨静仁|杨静仁]]、[[钱昌照|钱昌照]]、[[费孝通|费孝通]]、[[屈武|屈武]]等人以及全国人大常委会、全国政协、[[中共中央统战部|中共中央统战部]]、民革中央、[[台盟|台盟]]、台联和[[湖南省|湖南省]]、[[长沙市|长沙市]]、[[湘潭县|湘潭县]]的党政领导机关送了花圈。<ref name=" 湖湘人物：马璧"/>

官方悼词评价马璧说：“马璧在[[全国政协|全国政协]]和[[全国人民代表大会常务委员会|全国人民代表大会常务委员会]]工作期间，对国内四化建设、人民教育、对外宣传工作提出了许多积极的建议。他孜孜不倦地劝慰在[[台湾|台湾]]的朋友、同事和学生不妨来[[大陆|大陆]]走走看看，亲自了解体会，为祖国统一大业立功。直到病重之际，他仍然深切怀念在[[台湾|台湾]]的故旧好友，盼望祖国统一大业早日实现。”<ref name=shq>{{cite web |url=http://www.waheaven.com/memorial/retrospection/view.aspx?number=17961&articleid=27864|title=1985年10月15日 胡耀邦为马璧骨灰安放仪式送花圈|accessdate=2010-10-06 |work=|publisher= 天堂纪念网|language=zh-hans}}</ref>
