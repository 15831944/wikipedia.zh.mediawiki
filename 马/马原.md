'''马原'''{{BD|1953年||||catIdx= M马原}}出生于[[辽宁省|辽宁省]][[锦州|锦州]]，毕业于[[辽宁大学|辽宁大学]]，著名的[[画家|画家]]、[[小说家|小说家]]，“马原、[[余华|余华]]、[[苏童|苏童]]、[[格非|格非]]、[[洪峰_(作家)|洪峰]]”被誉为[[先锋派|先锋派]]五虎将之一。<ref name=" cow" >{{cite web |url=http://news.163.com/10/0510/06/66A772TA00014AED.html |title= 马原：小说已死|accessdate=2010-05-10 |publisher=网易 |work= |language =zh-cn}}</ref><ref name=" calf" >{{cite web |url= http://culture.ifeng.com/gundong/detail_2012_06/03/15008063_0.shtml?_from_ralated|title=马原，小说用嘴说 |accessdate= 2012年6月3日 |publisher= 凤凰网|work= |language =zh-cn}}</ref>

== 生平 ==
1953年，马原出生在[[辽宁省|辽宁省]][[锦州|锦州]]，早年当过[[农民|农民]]和钳工。<ref name=" cow" />

1982年，马原在[[辽宁大学|辽宁大学]]的中文系毕业，随后用[[记者|记者]]和[[编辑|编辑]]的身份远赴[[西藏自治区|西藏自治区]]，担任西藏电视台的记者。<ref name=" cow" /><ref name=" calf" />

马原从西藏回到辽宁省之后，出任辽宁省作家协会的会员。<ref name=" cow" />

2000年，马原分配到[[同济大学|同济大学]]教书。<ref name=" cow" />

== 风格 ==
马原的小说写法是“叙述圈套”，“以形式为内容”的中国小说新写法。<ref name=" calf" /><ref>{{cite web |url=http://culture.ifeng.com/whrd/detail_2012_03/22/13372077_0.shtml |title=马原《牛鬼蛇神》描写大串联：我从没得到官方认同 |accessdate=2012年3月22日 |publisher= 凤凰网|work= |language =zh-cn}}</ref>
