'''马正其'''（{{bd|1959年|3月||catIdx=}}），[[重庆|重庆]][[永川|永川]]人，1976年10月参加工作，1977年4月加入中国共产党，[[中央党校|中央党校]][[研究生|研究生]]学历，现任[[国家工商行政管理总局|国家工商行政管理总局]]党组成员、副局长，分管办公厅、消费者权益保护局、食品流通监督管理司。<ref>{{cite web| url=http://www.saic.gov.cn/zzjg/zjld/mzq/zggz/| title=副局长马正其主管工作| publisher=国家工商行政管理总局网站| accessdate=2013-07-29| deadurl=yes| archiveurl=https://web.archive.org/web/20130806075817/http://www.saic.gov.cn/zzjg/zjld/mzq/zggz/| archivedate=2013-08-06}}</ref>曾任第十届、十一届全国人大代表，中共重庆市一次、二次、三次党代会代表，重庆市二届、三届人大代表。[[中国共产党第十八届中央委员会|中国共产党第十八届中央委员会]]候补委员，

== 学历背景 ==
1984年9月至1986年6月，时任四川省永川县临江区委书记的马正其在[[重庆师范专科学校|重庆师范专科学校]]（后并入[[重庆师范大学|重庆师范大学]]）中文系干修班学习。<ref name="人民网马正其同志简历">{{cite web | url=http://cpc.people.com.cn/GB/64162/123659/123804/7330792.html | title=人民网马正其同志简历 | accessdate=2012-09-18 | newspaper=人民网}}</ref>1993年9至1995年12月，时任四川省重庆市双桥区委副书记、区长的马正其在[[四川省委党校|四川省委党校]]经济管理专业接受本科层次的函授学习。<ref name="人民网马正其同志简历" />1998年9月至2001年7月，时任重庆市政府副秘书长，市政府办公厅主任的马正其在[[中央党校|中央党校]]法学理论专业在职研究生班学习。

== 工作履历 ==
根据官方简历，1976年10月马正其参加工作，初任四川省永川县永安公社半脱产干部。早年曾先后担任四川省永川县松溉乡党委书记、永川县[[临江区|临江区]]区委书记等职务。1992年，担任四川省重庆市[[双桥区|双桥区]]委副书记、区长。1994年，任四川省[[荣昌县|荣昌县]]委书记。1997年，任重庆市政府副秘书长。2002年，任重庆市委常委、[[万州区|万州区]]委书记。2007年，升任[[重庆市|重庆市]]副市长、重庆市政府党组成员。2008年起，任重庆市政府党组副书记。2011年起，兼任重庆行政学院院长。<ref name="人民网马正其同志简历" />2013年4月，马正其调任国家工商行政管理总局党组成员、副局长。<ref>{{cite web| url=http://www.saic.gov.cn/zzjg/zjld/mzq/grjl/| title=副局长马正其个人简历| publisher=国家工商行政管理总局网站| accessdate=2013-07-29| deadurl=yes| archiveurl=https://web.archive.org/web/20130806065230/http://www.saic.gov.cn/zzjg/zjld/mzq/grjl/| archivedate=2013-08-06}}</ref>2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。2018年3月，任新组建的[[国家市场监督管理总局|国家市场监督管理总局]]副局长<ref>{{cite web |accessdate=2018-04-05 |url = http://www.gov.cn/guowuyuan/2018-04/04/content_5279818.htm |title=国务院任免国家工作人员（4月4日） |publisher=中国政府网}}</ref>。

== 遭举报事件 ==
2013年7月29日，《[[新快报|新快报]]》记者[[刘虎|刘虎]]通过其在[[新浪微博|新浪微博]]实名认证的账号实名举报国家工商总局副局长马正其嫌渎职犯罪，这是继[[罗昌平|罗昌平]]举报[[刘铁男|刘铁男]]、王文志举报[[宋林|宋林]]后的第三起中国大陆记者实名举报省部级高官的事件。<ref>{{cite web | url=http://news.qq.com/a/20130729/012162.htm | title=记者举报工商总局副局长马正其涉嫌渎职犯罪 | publisher=腾讯新闻 | accessdate=2013-07-29}}</ref>对此，国家工商总局办公厅副主任接受采访时表示，已经知晓刘虎举报马正其一事，但是否认相关指控并表示与事实严重不符。<ref>{{cite web| url=http://news.cnfol.com/130729/101,1277,15639009,00.shtml| title=记者实名举报工商总局副局长马正其嫌渎职犯罪| accessdate=2013-07-29}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>8月23日中午，刘虎在其位于重庆市渝北区的家中，被北京警方和当地社区民警一同带走，刘虎被带走时带有手铐。刘虎在相关文件上签字，对方证件显示其为北京市公安局。<ref>{{cite web | url=http://finance.sina.com.cn/china/20130823/151116545046.shtml | title=新快报记者被北京警方带走 曾举报工商总局副局长 | accessdate=2013-08-23}}</ref>
