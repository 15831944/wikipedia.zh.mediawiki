'''-{zh-tw:馬爾他;zh-cn:马耳他}-共和國'''（[[馬耳他语|-{zh-tw:馬爾他;zh-cn:马耳他}-语]]：'''Repubblika ta' Malta'''，[[英語|英語]]：'''Republic of Malta'''，[[阿拉伯語|阿拉伯語]]拼寫：'''جمهورية مالطة'''），通稱'''-{zh-tw:馬爾他;zh-cn:马耳他}-'''，位於[[南歐|南歐]]的[[共和制|共和制]]的一個[[微型國家|微型國家]]，首都[[瓦萊塔|瓦萊塔]]。是一個位於[[地中海|地中海]]中心的[[島國|島國]]， 由马耳他岛、戈佐岛、科米诺岛等几个岛屿组成。其中科米诺岛为鸟类保护区和自然保护区。素有“地中海心脏”之称，誉为“欧洲后花园”。[[官方語言|官方語言]]為[[馬耳他语|-{zh-tw:馬爾他;zh-cn:马耳他}-语]]和[[英語|英語]]。国内教学基本以英语为主。-{zh-tw:馬爾他;zh-cn:马耳他}-亦是[[英联邦|英联邦]]、欧盟成员国、申根区国家。
马耳他首都瓦莱塔是2018年欧洲文化中心。
-{zh-tw:馬爾他;zh-cn:马耳他}-经济以[[服务业|服务业]]和[[金融业|金融业]]为主，[[旅游业|旅游业]]是-{zh-tw:馬爾他;zh-cn:马耳他}-主要的[[外汇|外汇]]来源。另外，-{zh-tw:馬爾他;zh-cn:马耳他}-[[社会保障|社会保障]]体系较为完备，实行免费教育，免费医疗及退休保险制。这里也是多部如《特洛伊》、《角斗士》、《海岸情深》、《大力水手》、《权力的游戏》等多部电影、电视剧的取景地。

== 國名 ==
正式名稱-{zh-tw:馬爾他;zh-cn:马耳他}-語為{{lang|mt|Repubblika ta' Malta}}，通稱{{lang|mt|Malta}}。英語為{{lang|en|Republic of Malta}}，通稱{{lang|en|Malta}}。

中文表記為'''-{zh-tw:馬爾他;zh-cn:马耳他}-共和國'''，通稱'''-{zh-tw:馬爾他;zh-cn:马耳他}-'''。

“-{zh-tw:馬爾他;zh-cn:马耳他}-”一词的起源无从考究。通常的词源学说认为其来自希腊语{{lang|gk|μέλι}}（音meli），意为“蜜”。希腊人称此岛为{{lang|gk|Μελίτη}}（音Melite），指“蜜”或“甘甜”，或因-{zh-tw:馬爾他;zh-cn:马耳他}-出產一种品味獨特的蜂蜜，故得昵称“蜜地”。希腊在前700年即对-{zh-tw:馬爾他;zh-cn:马耳他}-岛富有影响，使用希腊语的東羅馬帝国在公元395年至公元870年间亦统治-{zh-tw:馬爾他;zh-cn:马耳他}-。

另一说“-{zh-tw:馬爾他;zh-cn:马耳他}-”来自腓尼基语“Maleth”，意為「避风港」。

== 历史 ==
{{main|馬爾他歷史|l1=-{zh-tw:馬爾他;zh-cn:马耳他}-歷史}}
在公元前10至前8世纪，[[腓尼基人|腓尼基人]]到此地定居，其後-{zh-tw:馬爾他;zh-cn:马耳他}-被[[希腊|希腊]]人占领，公元前4世纪又被[[迦太基|迦太基]]占领。公元前218年始被[[古罗马|古罗马]]统治。在[[迦太基|迦太基]]、[[羅馬共和國|羅馬共和國]]時代，因地中海貿易而繁榮。

9世纪起-{zh-tw:馬爾他;zh-cn:马耳他}-相繼受[[拜占庭帝国|拜占庭帝国]]、[[阿拉伯帝国|阿拉伯帝国]]、[[诺曼人|诺曼人]]统治。1192年西西里国王[[坦克雷迪|坦克雷迪]]封海军将领马加里托为首任[[马耳他伯爵|马耳他伯爵]]。

1523年，[[圣约翰骑士团|圣约翰骑士团]]自[[愛琴海|愛琴海]]的[[罗得岛|罗得岛]]移此，并获[[神圣罗马帝国|神圣罗马帝国]]皇帝[[查理五世_(神圣罗马帝国)|查理五世]]的承认。該支骑士团后更名[[馬爾他骑士团|-{zh-tw:馬爾他;zh-cn:马耳他}-骑士团]]。1565年的[[馬爾他大圍攻|-{zh-tw:馬爾他;zh-cn:马耳他}-大圍攻]]中，骑士团击败[[奥斯曼帝国|奥斯曼帝国]]軍隊，此战是-{zh-tw:馬爾他;zh-cn:马耳他}-史上最重大事件，成功阻止鄂圖曼帝國向西歐擴張。

1798年，[[拿破仑|拿破仑]]率[[法国|法国]]军队逐出骑士团，[[馬爾他騎士團|-{zh-tw:馬爾他;zh-cn:马耳他}-騎士團]]如今成為總部位於[[羅馬|羅馬]]的特殊政治實體。1800年[[英国|英国]]占领-{zh-tw:馬爾他;zh-cn:马耳他}-，1814年正式成为大英帝國殖民地。

1964年9月21日宣佈独立，但仍为[[英联邦|英联邦]]成员国。1974年成为-{zh-tw:馬爾他;zh-cn:马耳他}-共和国，2004年加入[[欧盟|欧盟]]。

== 政治 ==
[[File:Malta_Valletta_BW_2011-10-07_10-41-05.JPG|thumb]]
1964年7月21日颁布的独立宪法规定，-{zh-tw:馬爾他;zh-cn:马耳他}-为[[君主立宪制|君主立宪制]]政体，英国女王为-{zh-tw:馬爾他;zh-cn:马耳他}-国家元首。1974年12月13日修改宪法，-{zh-tw:馬爾他;zh-cn:马耳他}-成为[[共和国|共和国]]，总统为[[国家元首|国家元首]]，由[[议会|议会]]选举产生，任期五年。议会为[[一院制|一院制]]，称众议院。

== 地理 ==
[[File:Malta-CIA_WFB_Map.png|thumb]]
[[File:Ostansicht_Vallettas.jpg|thumb]]
-{zh-tw:馬爾他;zh-cn:马耳他}-位于[[地中海|地中海]]中部，面积316平方公里，北距[[西西里岛|西西里岛]]90公里。全国由[[馬耳他岛|-{zh-tw:馬爾他;zh-cn:马耳他}-岛]]、[[戈佐岛|戈佐岛]]（Gozo）、[[科米诺岛|科米诺岛]]（Comino）、科米诺托和菲尔夫拉岛五个小岛组成，其中只有-{zh-tw:馬爾他;zh-cn:马耳他}-岛、戈佐、科米诺三岛有人居住。面积最大的-{zh-tw:馬爾他;zh-cn:马耳他}-岛，约245平方公里。-{zh-tw:馬爾他;zh-cn:马耳他}-岛地势西高东低，以丘陵为主，缺乏淡水。

-{zh-tw:馬爾他;zh-cn:马耳他}-分为六个统计区：南港：面积26km²，人口83,234。北部：面积74km²，人口44,852。北港：面积24km²，人口118,409。东南：面积50km²，人口50,650。西部：面积73km²，人口51,961。[[戈佐-科米诺|戈佐-科米诺]]：面积69km²，人口29,026。

== 经济 ==
[[File:Former_Arriva_London_Bus_for_Malta.jpg|thumb]]
[[File:Map_of_Malta_2.png|thumb]]
-{zh-tw:馬爾他;zh-cn:马耳他}-[[自然资源|自然资源]][[贫乏|贫乏]]，矿产资源仅有[[石灰岩|石灰岩]]，又因为土层浅薄，缺乏灌溉水源，-{zh-tw:馬爾他;zh-cn:马耳他}-农业不发达，农副产品尚不能自给，[[粮食|粮食]]、[[肉禽|肉禽]]等主要依赖进口。-{zh-tw:馬爾他;zh-cn:马耳他}-[[经济|经济]]依靠[[加工制造业|加工制造业]]（主要是[[电子|电子]]、纺织）、旅游业、国际贸易。-{zh-tw:馬爾他;zh-cn:马耳他}-在2004年5月1日成為[[歐盟|歐盟]]成員之一，并於2008年以[[歐元|歐元]]為貨幣。

== 人口和語言 ==
-{zh-tw:馬爾他;zh-cn:马耳他}-是世界上人口最稠密的国家之一，每平方千米约1250人。主要是-{zh-tw:馬爾他;zh-cn:马耳他}-人，占总人口的九成，其余为[[阿拉伯人|阿拉伯人]]、[[意大利人|意大利人]]、[[英国人|英国人]]等。官方语言为[[馬耳他语|-{zh-tw:馬爾他;zh-cn:马耳他}-语]]和[[英语|英语]]，通用[[意大利语|意大利语]]。欧盟民调处的报告显示，100％的人口能讲-{zh-tw:馬爾他;zh-cn:马耳他}-语，88％的人口能讲英语，66％意大利语，17％法语。基本上都是以说英语为主。

== 宗教 ==
[[天主教|天主教]]为国教，少数人信奉基督教新教和希腊东正教。

== 文化 ==
[[File:Malta_Pastizzi.JPG|thumb]]
[[File:Malta_ftira.JPG|thumb]]
-{zh-tw:馬爾他;zh-cn:马耳他}-文化是数世纪来不同文化在[[馬耳他岛|-{zh-tw:馬爾他;zh-cn:马耳他}-岛]]上相互接触融合的产物，包括邻近的[[地中海|地中海]]地区的文化以及1964年-{zh-tw:馬爾他;zh-cn:马耳他}-独立前曾长期统治管理该地的诸多国家的文化。

=== 饮食 ===
马耳他菜是几个世纪以来岛上居民与来马耳他定居的外来者在烹调习惯上互相影响的结果，这种结合使马耳他菜混合了地中海周围各地的饮食口味。尽管许多菜式是岛上特有的，然而一部分很受欢迎的菜式的做法却是源自[[西西里岛|西西里岛]]、[[意大利|意大利]]南部以及[[中东地区|中东地区]]。被当地人称为''ftira biż-żejt''、''ġbejniet''和''pastizzi''的食物便是很好的例子。

=== 教育 ===
[[File:Uni-20malta.jpg|thumb]]]]
1946年-{zh-tw:馬爾他;zh-cn:马耳他}-開始普及小學義務教育，1971年16歲以下的中學教育也納入義務教育之中。除去免費的公立學校外，-{zh-tw:馬爾他;zh-cn:马耳他}-的[[天主教會|天主教會]]也提供免費的教育服務。至2006年所有公立學校都統一合併成了各類幼兒園、小學和中學一體的學校。<ref>{{cite web|url=http://www.aboutmalta.com/grazio/education.html|title=Education in Malta|accessdate=12 October 2007}}</ref>

-{zh-tw:馬爾他;zh-cn:马耳他}-採用[[英國教育|英式教育]]模式，-{zh-tw:馬爾他;zh-cn:马耳他}-語和英語在小學和中學授課時都有可能使用到。所有11歲的學生都要參加中學考試，進入教會或公立學校讀書，16歲時參加稱為SEC [[O-level|O-level]]的考試。在此之後學生可以進入[[第六級學院|第六級學院]]讀書，或者升入高中讀書，此後申請大學。-{zh-tw:馬爾他;zh-cn:马耳他}-成人識字率為99.5%。<ref>{{cite web|url=http://www.indexmundi.com/facts/malta/literacy-rate |title=Malta – Literacy rate |publisher=Indexmundi.com |accessdate=20 October 2013}}</ref>

[[馬耳他大學|-{zh-tw:馬爾他;zh-cn:马耳他}-大學]]成立於1768年，是該國最高學府。像該國的私立學校一樣，-{zh-tw:馬爾他;zh-cn:马耳他}-大學主要採用英語授課。此外，在中學選擇外語課程時，意大利語和法語最受歡迎。<ref>[http://www.nso.gov.mt/statdoc/document_file.aspx?id=580 Foreign Language Learning; National Statistics Office – Malta; 1 September 2004]</ref>

=== 媒体 ===
==== 报纸 ====
马耳他全国共有11家报纸，包括《马耳他时报》、《马耳他独立报》、《地平线》、《今日马耳他》、《祖国报》、《火炬报》（It-Torċa）等，大部分报刊以马耳他文为主，其中《马耳他时报》是由马耳他联盟报业公司出版的英文报刊，而《火炬报》、《地平线》则是马耳他工人总工会下属的联盟出版社发行，其中后者发行量在全国位居第一。

==== 广播电台 ====
马耳他一共有17家广播电台，且全部使用[[FM|FM]]频率播音，其中马耳他广播电台一台同时使用[[中波|中波]]999kHz播出，也是马耳他唯一的中波频率。马耳他广播电台开播于1935年11月11日，是马耳他第一家广播电台，隶属于马耳他公共广播服务公司。

此外，马耳他大学也有开办自己的广播电台，名为“Campus FM”，频率是103.7。

==== 电视台 ====
马耳他的电视事业始于1959年，目前马耳他拥有八套地面电视频道：TVM、TVM2、Parliament TV、NET Television、One、F Living、Smash Television、Xejk。

=== 特別活動 ===
微型飛機俱樂部在最近十年把此航空運動（微型飛機）<ref>[http://www.islandmicrolightclub.com Island Microlight Club - Malta]</ref>推展到島上，並且在十年內總共有22架微型飛機從-{zh-tw:馬爾他;zh-cn:马耳他}-國際機場起飛。
