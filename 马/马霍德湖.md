'''马霍德湖'''（{{lang-ur|{{nq|مہوڈنڈ جھیل}}}}; {{lang-ps|د ماهو ډنډ}}，意译"鱼类之湖"）是一个位于上{{le|乌索山谷|Usho}}的湖泊，距离[[巴基斯坦|巴基斯坦]][[开伯尔-普赫图赫瓦省|开伯尔-普赫图赫瓦省]][[斯瓦特县|斯瓦特县]]{{le|卡拉姆山谷|Kalam Valley}}约40公里。该湖泊可通过[[四轮驱动|四轮驱动]]车辆进入，通常用于[[钓鱼|钓鱼]]和[[划船|划船]]。<ref>{{cite web|title=Mahodand Lake Intoduction|url=https://www.flickr.com/search/?q=mahodand+lake|website=www.flickr.com}}</ref>

==地理==
[[File:Beautiful_Maodant_Kallam_Valley_KPK_Pakistan.jpg|thumb]]
马霍德湖位于[[兴都库什山脉|兴都库什山脉]]的山麓，海拔2865米（9400英尺），周围有[[草甸|草甸]]、[[山脉|山脉]]和茂密的[[森林|森林]]。<ref>{{cite web|title=Located at Foothills of Hindu Kush Peaks|url=https://www.findmyadventure.pk/|website=www.findmyadventure.pk|accessdate=2018-05-20}}</ref>马霍德湖水源由兴都库什山脉融化的冰川和泉水提供，并形成了[[斯瓦特河|斯瓦特河]]左岸的主要支流乌舒河（Ushu Khwar）。<ref>{{cite web|title=Geography of Lake|url=https://www.discoverworld.com/Pakistan/Khyber-Pakhtunkhwa/Swat-District/Mahodand-Lake|website=www.discoverworld.com|accessdate=2018-05-20}}</ref>

==动植物==
[[File:Mahudand_Lake_01.jpg|thumb]]
在冬季，马霍德湖湖面结冰并被大雪覆盖。在夏季，湖盆周围是一大片高山花卉，如{{le|水杨梅属|Geum}}、[[绿绒蒿属|绿绒蒿属]]、[[委陵菜属|委陵菜属]]和[[龙胆属|龙胆属]]。除此之外，湖四周环绕着不同种类的[[松树|松树]]，这些松树是野生鸟类的栖息地。湖中有大量鳟鱼，它们为垂钓活动提供了理想的机会，但只有持有相关执照的垂钓者才允许捕捞。<ref>{{Cite web |url = http://www.valleyswat.net/tourism/mahodand.html |title = Mahodand (Fish Lake) |language = en |accessdate = 2019-03-25 }}</ref>

==钓鱼和露营==
[[File:LRM_EXPORT_20180503_183004.jpg|thumb]]
游客常常在湖边[[钓鱼|钓鱼]]和[[露营|露营]]。马霍德湖的湖水分成一系列大小河流，这些河流曾经聚集着由{{le|斯瓦特土邦|Swat (princely state)}}统治者引进的棕色及彩虹[[鳟鱼|鳟鱼]]。当地人为了日常收入而过度捕鱼，并且非法使用诸如电击、[[硝酸甘油炸药|硝酸甘油炸药]]和大渔网等捕鱼方法，这使得鱼类数量大大减少。<ref>{{Cite web |url = http://www.dailymotion.com/video/x2r2inm_mahodand-lake_lifestyle |title = mahodand lake |website=dailymotion |accessdate = 2019-03-25 }}</ref><ref>{{Cite web |url = https://etdip.com/ |title = Area Rug Tips |date = 2018-09-21 |language = en |accessdate = 2019-03-21 }}</ref>
