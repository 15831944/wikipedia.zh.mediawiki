'''马德勒撞击坑'''（Mädler）是[[火星|火星]]上一座位于东经2.7°、南纬10.7°处，以[[德国|德国]][[天文学家|天文学家]][[约翰·海因里希·冯·马德勒|约翰·海因里希·冯·马德勒]]命名的[[陨石坑|陨石坑]]，该名称于1973年被[[国际天文学联合会|国际天文学联合会]]正式接受<ref name=gazetteer>{{cite web
 |url         = http://planetarynames.wr.usgs.gov/Feature/4092
 |title       = Mädler
 |work        = Gazetteer of Planetary Nomenclature
 |publisher   = International Astronomical Union (IAU) Working Group for Planetary System Nomenclature (WGPSN)
 |date        = 2010-11-17
 |archiveurl  = https://archive.is/20160708203956/http://planetarynames.wr.usgs.gov/Feature/4092
 |archivedate = 2016-07-08
 |deadurl     = yes
}}</ref>。 

马德勒和合作者[[威廉·沃尔夫·比尔|威廉·比尔]]在19世纪30年代早期绘制了第一幅相当好的火星地图。绘制时他们选择了一个特定的特征为作为火星图的[[本初子午线|本初子午线]]。他们的选择是加强时，当[[乔凡尼·斯基亚帕雷利|乔凡尼·斯基亚帕雷利]]在他更著名1877年火星图中也使用了这一位置时。该特征后来被称为子午线湾（中央湾），但随着2004年[[美国宇航局|美国宇航局]][[机遇号|机遇号]]探测器的火星登陆，也许[[子午线高原|子午线高原]]更为人所知。 

马德勒撞击坑位于[[子午线高原|子午线高原]]南部，靠近本初子午线，往东10°为[[比尔撞击坑|比尔撞击坑]]，[[斯基亚帕雷利撞击坑_(火星)|斯基亚帕雷利撞击坑]]也位于该区域。
