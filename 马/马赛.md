'''马赛'''（[[法语|法语]]：''{{lang|fr|Marseille}}''； [[加斯科方言|加斯科]]：''Marselha''）是[[法国|法国]]第二大城市和第三大都會區（{{lang|fr|aire urbaine}}）。在2011年的人口普查中，有1,720,941名居民。它位于[[地中海|地中海]]沿岸，原属于[[普罗旺斯省|普罗旺斯省]]（{{lang|fr|Provence}}）。它是法国最大的商业[[港口|港口]]，也是地中海最大的商业港口。

马赛是[[普罗旺斯-阿尔卑斯-蓝色海岸|普罗旺斯-阿尔卑斯-蓝色海岸]][[大区|大区]]的首府，也是[[罗讷河口省|罗讷河口省]]的首府。

== 历史 ==
马赛是由来自[[福西亚|福西亚]]（[[:en:Phocaea|Phocaea]]）的古希腊人于公元前600年作为一个贸易港而興建的，原取名为馬西利亞（希臘文：''Μασσαλία'' 英語：''Massalia''；参见[[各地傳統的希臘化古稱列表|各地傳統的希臘化古稱列表]]）。它被[[凯尔特人|凯尔特人]]蹂躏过，然后又被[[古罗马|古罗马]]征服。在[[古罗马|古罗马]]时代，它被称为''Massilia''。在中世紀時，這裡的商業繁華，[[威尼斯|威尼斯共和國]]的商人也會到這裡做生意。

1348年[[黑死病|黑死病]]袭击马赛，2万5千市民死了1万5千。

1830年法国殖民[[阿尔及利亚|阿尔及利亚]]和1869年[[苏伊士运河|苏伊士运河]]开通之后，作为商业港口的马赛，获得了更多的发展机会。

1934年，[[亞歷山大一世_(南斯拉夫)|南斯拉夫王国亚历山大一世]]抵达港口并会见了法国外交部长[[路易·巴尔杜|路易·巴尔杜]]。他在那里被{{link-en|弗拉多·切爾諾澤姆斯基|Vlado Chernozemski}}刺殺身亡，刺杀者对亚历山大拒绝承认[[克罗地亚|克罗地亚]]为一个独立国家而感到愤怒。

[[二战|二战]]时期，1942年盟軍佔領北非，納粹德國及意大利进駐[[維希法國|維希法國]]控制港口以維持地中海勢力，马赛被[[意大利王國|義大利]]軍隊佔领，直至意大利法西斯倒台。

== 经济 ==
马赛是法國最大的海港，亦為地中海地區的重要港口。由於周邊地區農業發達，整個城市經營的農產品也多種多樣。

== 行政管理 ==
马赛被分为16个市政专区（arrondissement），然后又被分为111个街区（quartier）。这些专区每两个组成一个小区（secteur），8个小区拥有一个议会和一座市政大厅，就像[[巴黎|巴黎]]和[[里昂|里昂]]的专区一样。

市政选举按小区进行。每一个小区选举自己的议员（共计303个），其中的三分之一为市议员。

== 文化 ==
法国[[国歌|国歌]]“[[马赛曲|马赛曲]]”正是以马赛义勇军命名的。

广泛传播的[[塔罗牌|塔罗牌]]来自马赛，它也被称为马赛塔罗牌。

== 人口统计 ==
近25%的马赛人口为北非血统，大多为[[阿尔及利亚|阿尔及利亚]]人和[[突尼斯|突尼斯]]人。据人口统计学家估计，在不久的将来，若再不斷收容北非移民，马赛将成为欧洲第一个穆斯林人口占多数的城市。此外，它的犹太人社区是欧洲第三大的。

== 交通 ==
馬賽市内交通发达，[[城市軌道交通系統|城市軌道交通系統]]是採用[[膠輪路軌系統|膠輪路軌系統]]，道路则拥有[[无轨电车|无轨电车]]及[[公共汽车|公共汽车]]。马赛的机场为[[马赛普罗旺斯机场|马赛普罗旺斯机场]]，它位于[[马里尼亚讷|马里尼亚讷]]。藉著[[TGV|TGV]]可以前往法國其他地區，更可以前往[[倫敦|倫敦]]。

== 其它 ==
[[File:Madonna_and_Child_Marseille.jpg|thumb]]的圣母圣婴像]]
这座城市主要的[[足球俱乐部|足球俱乐部]]是[[马赛足球俱乐部|马赛奥林匹克足球俱乐部]]（Olympique de Marseille），于1899年成立，曾夺得8届法国甲级联赛冠军及10次法国杯。也是1993年[[欧洲冠军杯|欧洲冠军杯]]的冠军得主，但是二十世纪九十年代由当时的所有者[[贝尔纳·塔皮|贝尔纳·塔皮]]（Bernard Tapie）犯下的足球丑闻令它染上了污点。马赛成功于 07/08 赛季重返欧洲冠军杯赛场。上个赛季（09/10赛季），马赛再次夺回联赛冠军的宝座。
