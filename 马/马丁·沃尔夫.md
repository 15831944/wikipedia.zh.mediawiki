'''马丁·沃尔夫'''（{{lang-en|Martin Wolf}}, {{bd|1946年|3月16日|}}）是一位英国经济评论家和记者。英国《[[金融时报|金融时报]]》的副主编及首席经济评论员。<ref>[http://www.ftchinese.com/column/007000012 关于马丁·沃尔夫]，《金融时报》中文网官网。</ref>

==生平==

本科毕业后进入[[牛津大学纳菲尔德学院|牛津大学纳菲尔德学院]]，1971年拿到经济学硕士学位。他表示自己不想读博士，因为不想成为学者。<ref name = "wolfBio"/>毕业后进入[[世界银行|世界银行]]的青年学者项目，1974年成为一名高级经济学家。他对[[罗伯特·麦克纳马拉|罗伯特·麦克纳马拉]]执掌下的[[世界银行|世界银行]]颇为失望，观点也转向右倾和自由市场模式。<ref name = "CalloftheWolf"/><ref name = "wolfBio"/>

1981年离开[[世界银行|世界银行]]在伦敦一家''贸易政策研究中心''担任主管，1987年加入英国《[[金融时报|金融时报]]》，1990年成为助理编辑，1996年成为首席经济评论员。2000年代末之前，他是[[全球化|全球化]]和[[自由市场|自由市场]]的颇具影响力支持者。
