'''马约特大區'''（{{lang-fr|Mayotte}}；[[斯馬奧萊語|斯馬奧萊語]]：''Maore''；[[馬拉加斯語|馬拉加斯語]]：''Mahori''） 通稱'''馬約特'''，位于[[莫桑比克海峡|莫桑比克海峡]]，与[[大科摩罗岛|大科摩罗岛]]、[[昂儒昂岛|昂儒昂岛]]、[[莫埃利岛|莫埃利岛]]共同组成[[科摩罗群岛|科摩罗群岛]]。面积为374平方公里，经济以农业为主，主要生产香子兰等香料，是法国的海外大区，下辖一个省，即马约特省。马约特岛包括大陆地岛、小陆地岛以及周围一些小岛，首府和最大城市[[马穆楚|马穆楚]]。

2009年3月29日，进行了一次关于在2011年成为法国海外省的公民投票<ref>{{fr icon}} {{cite web|url=http://www.lefigaro.fr/lefigaromagazine/2009/03/14/01006-20090314ARTFIG00183--enquete-sur-le-futur-101-e-departement-.php| title=ENQUETE SUR LE FUTUR 101e DEPARTEMENT}}</ref>，结果以95.2%赞成而通过<ref>{{sv icon}}{{cite web|url=http://dn.se/nyheter/varlden/mayottier-vill-vara-fransman-1.833151|title=Mayottier vill vara fransmän|deadurl=yes|archiveurl=https://web.archive.org/web/20090331021920/http://www.dn.se/nyheter/varlden/mayottier-vill-vara-fransman-1.833151|archivedate=2009-03-31}}</ref>。

== 地理 ==
[[File:Cn-map.png|left]]
主要岛屿包括大陆地岛（Grande-Terre）和小陆地岛（La Petite-Terre），地质学上来说，大陆地岛是科摩罗地区最古老的岛屿，有39公里长，22公里宽，最高点为海拔660米高的贝拿勒斯山（''Mont Bénara''）。由于是火山岩构成的岛屿，一些地区的土地特别的肥沃。珊瑚礁围绕着一些岛屿起到了保护船只和栖息鱼群的作用。 

[[藻德济|藻德济]]是1977年前馬約特的首都，位于小陆地岛上，这座岛屿有着10公里长，是大陆地岛周边几个零星小岛中最大的一个。马约特是独立的印度洋委员会成员。

== 政治 ==
马约特的政治框架为法国海外领地的议会民主制，马约特省委員會主席（President of the General Council）是岛内的政府首脑，实行多党制，政府拥有最大的行政权力。

在法国国民议会中马约特拥有一个代表资格以及两个法国参议院议员席位。

== 历史和现今地位 ==
马约特的地位与一个省相当，拥有一个省委员会。根据2001年7月11日法律，它确切上应为“省级行政区域”（collectivité départementale），并且有权利在2010年选择成为全权的法国省。马约特下辖17个市镇，组成同样数量的乡。
=== 历史 ===
在1500年，马约特岛（阿拉伯语: جزيرة الموت）成为了伊斯兰教的君主领地。

1503年，马约特岛被葡萄牙探险者发现。但未侵占
。
16世纪起[[梅里纳人|马尔加什人]]开始统治马约特岛和科摩罗，直到1843年割讓給[[法國|法國]]。1974年和1976年，马约特岛的两次公民投票分別以63.8%和99.4%表決維持與法國的关系而不獨立。科摩罗则宣称拥有马约特岛的主权。在1976年[[聯合國安全理事會|联合国安理会]]一份关于承認科摩罗對马约特主權的決議草案中，雖獲14个會員國支持，但最终被身為[[联合国安全理事会常任理事国|安理会常任理事国]]的法國一票否决。

岛屿的防務完全由法国政府承担，[[法国军队|法国军队]]在马约特岛设有一支駐軍，即[[法國外籍兵團马约特特遣队|法國外籍兵團马约特特遣队]]。

== 经济 ==
马约特的官方货币为[[欧元|欧元]]。

据[[INSEE|INSEE]]的评估，2001年马约特的[[国内生产总值|GDP]]总计为6.1亿欧元 (根据2001年的汇率约等于5.47亿美元；根据2008年的汇率约等于9.03亿美元)。<ref name=GDP>{{fr icon}} {{cite web| url=http://www.insee.fr/fr/insee_regions/reunion/zoom/mayotte/publications/tem/tem2007/TEM2007_14.3.pdf| title=Résultats économiques| author=[[INSEE|INSEE]]| accessdate=2008-01-16}}</ref> 同一时期的人均国内生产总值为3,960欧元 (2001年时期的3,550美元; 2008年时期的5,859美元)，<ref name=GDP />这个数据比同期的[[科摩罗|科摩罗]]高出了9倍之多，但也只是临近法国海外省[[留尼汪|留尼汪]]GDP的三分之一和法国本土的16%。<ref name=GDP />

== 交通 ==
马约特的铁路全长10.5公里，有着总长度为93公里的[[公路|公路]]，其中72公里为铺装公路。马约特岛内没有水道，最大港口是[[藻德济|藻德济]]，并有着一座飞机场。
