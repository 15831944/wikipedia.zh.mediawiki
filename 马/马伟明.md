'''马伟明'''（{{bd|1960年|4月6日|}}），[[江苏省|江苏]][[扬中市|扬中]]人，博士学历，[[中华人民共和国|中华人民共和国]]舰船动力与[[电气工程|电气工程]]专家，[[中国工程院|中国工程院]]机械与运载工程学部院士，[[中国人民解放军少将|中国人民解放军海军少将]]军衔。[[中共十六大|中共十六大]]代表、[[中国共产党第十八届中央委员会候补委员列表|中共第十八]]、[[中国共产党第十九届中央委员会候补委员列表|十九届中央候补委员]]，第九、十一、十二、十三届[[全国人大代表|全国人大代表]]<ref>{{cite news |title = 全国人大代表信息-马伟明 |url = http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=122723 |accessdate = 2013-04-05 |website = 全国人大网 }}</ref>。

== 生平 ==
1978年考入[[中国人民解放军海军工程大学|中国人民解放军海军工程大学]]电气工程专业，先后获得工学学士学位和硕士学位，[[清华大学|清华大学]][[电机|电机]]专业工学博士毕业。1985年加入[[中国共产党|中国共产党]]。曾经是最年轻的[[中国工程院院士|中国工程院院士]]。马伟明是[[中国人民解放军海军工程大学|中国人民解放军海军工程大学]]教授、[[教授|博士生导师]]、电力电子技术研究所所长、舰船综合电力技术国防科技重点实验室主任。

34岁破格晋升教授，38岁成为博士生导师，41岁当选中国工程院院士，42岁晋升海军少将军衔。曾荣膺[[国家科技进步一等奖|国家科技进步一等奖]]、“[[中国发明创业奖|中国发明创业奖]]”特等奖、“军队专业技术重大贡献奖”、“优秀科技青年创业奖”、“求是杰出青年实用工程奖”、“[[何梁何利奖|何梁何利奖]]”；获授全国十大“杰出专业技术人才”、“[[国家有突出贡献的中青年专家|国家有突出贡献的中青年专家]]”和“当代[[发明家|发明家]]”[[荣誉称号|荣誉称号]]<ref>{{cite web |url = http://news.qq.com/a/20111227/000529.htm |title=“国宝级”技术少将马伟明：两获军委一等功 |publisher=腾讯网 |accessdate=2011-12-27 }}</ref>。2016年6月，当选[[中国科学技术协会|中国科协]]第九届全国委员会副主席<ref>{{cite web |url = http://news.qq.com/a/20160603/024133.htm |title=电磁弹射专家马伟明当选中国科协全委会副主席 |publisher=腾讯网 |accessdate=2016-06-03 }}</ref>。

2017年7月，中央军委主席[[习近平|习近平]]签署命令，授予马伟明等10位同志“[[八一勋章_(2017年)|八一勋章]]”<ref>{{cite web |url = http://news.ifeng.com/a/20170728/51522244_0.shtml |title = 景海鹏、程开甲、王刚等10人被授予“八一勋章” |author =  |date = 2017-07-28 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。2017年7月28日，中央军委颁授“八一勋章”和授予荣誉称号仪式在北京[[八一大楼|八一大楼]]举行，中共中央总书记、国家主席、中央军委主席习近平向马伟明等10位“八一勋章”获得者颁授勋章和证书。这是“八一勋章”首次颁授。马伟明同时获授“心系强军、锐意创新的科研先锋”荣誉称号<ref>{{cite web |url = http://news.ifeng.com/a/20170728/51524452_0.shtml |title = 习近平向“八一勋章”获得者颁授勋章和证书 |author =  |date = 2017-07-28  |website = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

马伟明长期致力于独立发供电系统研究，创建并发展十二相发电机整流供电系统的基础理论体系，攻克系统中稳定性预测、固有振荡抑制、复合故障诊断、短路保护等国内外长期未解决的关键技术难题（德国有关公司购买其稳定装置专利），研制出具有国际先进水平的十二相发电机整流供电系统，已装备应用，获2000年度国家科技进步一等奖（第一名）。提出三相/十二相双绕组电力集成新原理，据此研制出艇用交直流混合发供电系统。在独立电力系统电磁传导干扰预测理论和抑制技术上取得突破，完成了运载工具不同供电系统电磁兼容研制<ref>{{cite web |url = http://www.guancha.cn/military-affairs/2014_01_25_202230.shtml |title = 马伟明院士获奖感言批露中国海军全电推进、电磁弹射器研制 |website =观察家网 |accessdate=2014-01-25 }}</ref>。他领导了中国海军航母全电推进和[[电磁弹射器|电磁弹射器]]研制项目，有“中国电磁弹射技术之父”称誉<ref>{{cite web |publisher=大公网 |accessdate=2016-10-19 |url = http://news.takungpao.com/mainland/focus/2015-10/3219916.html |title=国产航母电磁弹射器曝光 歼15或用新技术升空 }}</ref>。
