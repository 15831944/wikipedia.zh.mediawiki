'''马蔚华'''（1949年6月生），[[辽宁|辽宁]][[锦州|锦州]]人，高级经济师，自1999年3月至2013年5月担任[[招商银行|招商银行]]董事、行长，兼任[[吉林大学|吉林大学]]董事会董事<ref name="人物">{{cite web |url=http://news.jlu.edu.cn/new/?mod=article&act=show&id=8167 |language= |title=吉林大学董事会董事马蔚华、李书福分别当选中国经济十年商业领袖、2009CCTV中国经济年度人物 |publisher=吉林大学 |date=2009-12-25 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20150428010927/http://news.jlu.edu.cn/new/?mod=article |archivedate=2015-04-28 }}</ref>、[[中国金融学会|中国金融学会]]常务理事、[[中国企业家协会|中国企业家协会]]副会长。<ref name="简介">{{cite web |url=http://www.mingren.biz/html/8279.html |language= |title=马蔚华 |publisher=名人网 |date=2008年5月1日 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20130101134246/http://www.mingren.biz/html/8279.html |archivedate=2013年1月1日 }}</ref>

== 生平 ==
1982年毕业于[[吉林大学|吉林大学]]经济系，1986年获得吉林大学[[经济学|经济学]][[硕士|硕士]]，1999年获[[西南财经大学|西南财经大学]]经济学[[博士|博士]]，并获得美国[[南加州大学|南加州大学]][[荣誉博士|荣誉博士]]。<ref name= "简介"/>

1988年至1998年，在[[中国人民银行|中国人民银行]]工作。

== 荣誉 ==
*2009年CCTV中国经济年度人物颁奖盛典：“中国经济十年商业领袖”<ref name= "人物"/>。
