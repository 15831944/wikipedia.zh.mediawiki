'''馬建忠'''（{{bd|1845年|2月9日|1900年|8月14日|catIdx=M马}}），別名'''-{乾}-'''，[[學名|學名]]'''馬斯才'''，[[表字|字]]'''眉叔'''，[[江苏|江苏]][[丹徒|丹徒]]（今[[镇江|镇江]]）人，[[中国|中国]][[清朝|清末]][[洋務派|洋務派]]重要官員、维新思想家、[[外交家|外交家]]、[[語言學家|語言學家]]。其所著《[[马氏文通|文通]]》是第一部中国人编写的全面系统的[[汉语语法|汉语语法]]著作。

==生平== 

===早年===
马建忠于1845年2月9日（清[[道光|道光]]十四年）生于一个[[天主教|天主教]]家庭。{{fact|是《[[文献通考|文献通考]]》作者[[马端临|马端临]]第二十世孙}}。父亲马岳熊，兄弟中排行老幺，有四兄一姊。二哥[[馬建勛|馬建勛]]早年受[[曾國荃|曾國荃]]拔擢，入[[李鸿章|李鸿章]]幕府，司[[淮軍|淮軍]][[糧台|糧台]]。四哥[[馬相伯|馬相伯]]是[[震旦大學|震旦大學]]、[[復旦大學|復旦大學]]的創辦人。

马建忠从小學曉中國傳統經史，五歲就諗考[[科舉|科舉]]。1853年（[[咸丰|咸丰]]三年），[[太平军|太平军]]攻入[[南京|南京]]，马家遂搬去[[上海|上海]]躲避战乱。马建忠与四哥就讀中西学并重的[[天主教|天主教]][[耶穌會|耶穌會]][[上海市徐汇中学|徐匯公學]]，学习法文和拉丁文等课程，同时准备科举，后与其四哥同为该校首届毕业生<ref>天主教徐匯高級中學校史 {{cite web |url=http://www.sish.tpc.edu.tw/hstry.htm |title=存档副本 |accessdate=2007-06-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20080804064552/http://www.sish.tpc.edu.tw/hstry.htm |archivedate=2008-08-04 }} </ref>。受1854年至1860年[[第二次鴉片戰爭|第二次鴉片戰爭]]的影响，他“决然舍其所学，而学所谓洋务者”<ref>《適可齋記言記行》</ref>，进入耶穌會初學院做[[修士|修士]]，继续學习[[拉丁文|拉丁文]]、[[法文|法文]]、[[英文|英文]]和[[希臘文|希臘文]]等。经过十余年的刻苦努力，成了一位“善古文辞，尤精欧文，英、法现行文字以至希腊、拉丁古文，无不兼通”<ref>《清史稿》卷446，本传</ref>的学贯中西的新式人才。

===留学===
1870年（[[同治|同治]]九年），经二哥馬建勛引荐，他也成为李鴻章的幕僚，随办洋务。因为熟悉西洋文化和语言，他受到李鴻章的赏识。1876年（[[光绪|光绪]]二年），他以[[郎中_(官位)|郎中]]资格被李鴻章派往[[法國|法國]]学习[[国际法|国际法]]，同时兼任中國駐法[[公使|公使]][[郭嵩燾|郭嵩燾]]的翻譯。1877年（光绪三年），他通过了[[巴黎考試院|巴黎考試院]]的[[文科|文科]]和[[理科|理科]]考試，成为第一个取得{{link-fr|法国高中会考毕业证书|Baccalauréat en France}}的中国人。1878年（光绪四年）至1880年，他進入[[巴黎自由政治学堂|巴黎自由政治学堂]]（[[巴黎政治大学|巴黎政治大学]]前身）全日制學習[[公法|公法]]。

===事業===
1880年（光绪六年），马建忠回到[[天津|天津]]，重新在李鸿章幕下办理洋务。翌年，奉李鸿章之命赴[[英属印度|英属印度]]，与[[印度總督|印度總督]][[乔治·罗宾逊_(里蓬侯爵)|里蓬]]交涉[[鴉片|鴉片]]专卖及稅收问题。1882年（光緒八年），李鴻章又派他去[[朝鮮|朝鮮]]，协助朝鲜政府与英、美、德簽订商約。同年朝鮮發生[[壬午兵变|壬午兵变]]，[[明成皇后|闵妃]]请求清朝出兵，直隶总督[[张树声|张树声]]和李鸿章派[[淮军|淮军]]将领[[吴长庆|吴长庆]]平乱，马建忠作為长庆的幕僚同去，马建忠设下擒賊先擒王之计，派[[袁世凱|袁世凱]]抓捕了[[雲峴君|雲峴君]]，押回[[中國|中國]][[軟禁|軟禁]]，平定此事。同年10月18日，李鴻章派馬建忠赴天津与法國公使[[寶海|寶海]]达成[[越事草案|越事草案]]三條，即中國撤兵、通商、中法分巡[[紅河|紅河]]南北。

1884年（光绪十年），马建忠加入[[唐廷樞|唐廷樞]]主持的[[輪船招商局|輪船招商局]]。[[中法战争|中法战争]]后，他升任輪船招商局[[會辦|會辦]]。1890年（光绪十六年），马建忠开始撰写《富民说》，主张发展对外贸易、扶持民间工商业等措施以富民强国，并将《富民说》上呈李鸿章。不久，受李鸿章委派担任[[上海机器织布局|上海机器织布局]][[总办|总办]]，旋因资金周转等问题去职回籍。1895年（光绪廿一年），[[甲午战争|甲午战争]]失败后，他应李鸿章之邀到北京，襄助李鸿章赴[[日本|日本]][[马关|马关]]议和。翌年，马建忠与上海《[[时务报|时务报]]》主笔[[梁启超|梁启超]]相识，并出版了《[[適可齋記言記行|適可齋記言記行]]》。

此后，马建忠一直埋头整理《[[馬氏文通|文通]]》一书<ref>有学者认为，该书编写也得到了其兄[[马相伯|马相伯]]的帮助，参见 《马氏文通》的作者到底是谁 {{cite web |url=http://www.yaoxiaoping.org/news/news_detail.asp?id=84 |title=存档副本 |accessdate=2007-09-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20070929081352/http://www.yaoxiaoping.org/news/news_detail.asp?id=84 |archivedate=2007-09-29 }} </ref>，于1898年（光绪廿四年）出版。该书以西方语文的[[语法|语法]]为本，对照从古书中精选的例句，研究[[古代汉语|古汉语]]的语法规律，创建了一套[[汉语|汉语]]的[[汉语语法|语法]]体系，是奠定汉语[[语法学|语法学]]基础的开山之作，对后世汉语语法研究产生了巨大影响。1931年，语言学家[[杨树达|杨树达]]著成《马氏文通刊误》由商务印书馆印行，将《文通》补完增订。

1900年（光绪廿六年），馬建忠再度应李鸿章之召，任上海[[行轅|行轅]][[襄理|襄理]][[機要|機要]]。8月14日，因赶译长篇急电而猝然去世，终年55岁。

==思想==
馬建忠主張废除[[厘金|釐金]]，收回[[关税|关税]]主权，發展对外贸易，扶持民營[[工商業|工商業]]，致力推行洋務，并称许西方[[议会制度|议会制度]]。同时建議開设翻譯書院，提倡國人多學洋文，汲取外國科學文化知識。
