'''马库斯·莫里斯'''（{{lang-en|'''Marcus Morris'''}}，{{bd|1989年|9月2日|catIdx=M}}），生于[[美国|美国]][[宾夕法尼亚州|宾夕法尼亚州]][[费城|费城]]，美国职业篮球运动员，司职[[前锋|前锋]]，目前效力于[[波士頓塞爾提克|波士頓塞爾提克]]队。其双胞胎哥哥[[马基夫·莫里斯|马基夫·莫里斯]]也是一名NBA球员，效力于巫師队。

== 大学时期 ==
2008年，马库斯·莫里斯同哥哥马基夫·莫里斯一起进入堪萨斯大学，在大一赛季他可以得到7.4分和4.7个篮板，篮板数排在大十二联盟大一新生中的第一位。第二年他能得到12.8分6.1个篮板，56.1%的命中率也排在大十二联盟球员中的第一位，帮助堪萨斯大学获得大十二联盟锦标赛的冠军。2010年3月18日，[[NCAA|NCAA]]第一轮对阵[[理海大学|理海大学]]，15投12中砍下26分10个篮板帮助球队获胜，也入选了大十二联盟最佳阵容第二队。

2010-11赛季，升入大三的莫里斯成为球队得分王，场均17.2分7.6个篮板，命中率达到57%。2011年1月12日面对[[爱荷华州立大学|爱荷华州立大学]]得到大学时代最高分33分。堪萨斯大学以分区锦标赛冠军杀进NCAA精英八强赛，虽然莫里斯得到20分16个篮板，依旧不敌[[弗吉尼亚联邦大学|弗吉尼亚联邦大学]]无缘NCAA四强。但个人表现抢眼，令他入选了全美最佳阵容第二队和约翰-伍登奖全美最佳阵容。赛季结束后莫里斯兄弟宣布参加NBA选秀。

== 职业生涯 ==
2011年6月23日，在哥哥马基夫·莫里斯被[[菲尼克斯太阳|菲尼克斯太阳]]选中之后，马库斯·莫里斯也被休斯顿火箭以第一轮14顺位选中。新秀赛季，莫里斯并没有得到太多的机会，只以替补身份出场了16次，大多时间都被火箭队下放至[[NBDL|NBDL]]锻炼。

2012-13赛季，球队主力大前锋[[帕特里克·帕特森|帕特里克·帕特森]]开赛后受伤，这让莫里斯成为了球队先发，但在帕特森伤愈后再次成为替补。2013年2月，被交易至[[菲尼克斯太阳|菲尼克斯太阳]]。

== 技术特点 ==
莫里斯有着出色的攻击技巧，在高位可以投射，也擅长面筐进攻。但作为一名内线球员，他身高和爆发力都有欠缺，篮板和盖帽能力也不够。
