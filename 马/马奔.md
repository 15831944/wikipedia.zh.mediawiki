'''马奔'''（{{bd|1958年|11月||}}）[[陕西|陕西]][[黄陵县|黄陵]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员。

==生平==
1978年3月参加工作，1984年12月加入[[中国共产党|中国共产党]]，[[西北工业大学|西北工业大学]]在职研究生学历，工学博士。1978年3月到1981年4月，在西北国棉五厂工作。1981年4月到1983年7月，任[[陕西省|陕西省]][[西安市|西安市]][[灞桥|灞桥]]公安分局十里铺派出所民警。1983年7月起，历任西安市公安局机关团委干部、书记，西安市公安局团委书记（副处级）。1994年12月起，历任西安市国家安全局副处长、处长，副局长、党委副书记，局长、党委书记（其间，2001年9月到2006年11月在西北工业大学九系系统管理专业学习，2006年11月获得工学博士学位）。2008年3月起，历任[[陕西省纪委|陕西省纪委]]秘书长、[[陕西省监察厅|陕西省监察厅]]副厅长、陕西省纪委副书记。2014年4月到2014年12月，任[[中共中央直属机关事务管理局|中共中央直属机关事务管理局]]局长。2014年12月到2017年3月，任中共中央直属机关事务管理局局长、机关党委书记。2017年3月，任[[中央纪委|中央纪委]]驻[[国家卫生和计划生育委员会|国家卫生和计划生育委员会]]纪检组组长、国家卫生和计划生育委员会党组成员<ref>{{Cite web |url =  http://renshi.people.com.cn/n1/2017/0427/c139617-29239444.html|title = 马奔任中纪委驻国家卫计委纪检组组长 |author =  |date = 2017-04-27 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref><ref name=maben>{{Cite web |url = http://china.caixin.com/2017-04-25/101082843.html |title =  中直管理局局长马奔任中央纪委驻国家卫计委纪检组长|author =  |date =2017-04-25  |publisher =财新网  |language =  |accessdate =  |quote =  }}</ref>。
