'''马西米连诺一世'''（{{lang-es|Maximiliano I}}，{{bd|1832年|7月6日|1867年|6月19日|Maximiliano I}}），全名'''斐迪南·马克西米连·约瑟夫·冯·哈布斯堡-洛林'''（{{lang-de|Ferdinand Maximilian Josef von Habsburg-Lothringen}}），[[奥地利|奥地利]][[哈布斯堡-洛林皇朝|哈布斯堡王朝]]成员，曾任[[倫巴第-威尼西亞王國|倫巴第-威尼西亞王國]]总督，1864年4月10日在[[法国|法国]][[皇帝|皇帝]][[拿破仑三世|拿破仑三世]]的怂恿下，接受了[[墨西哥|墨西哥]]皇位，称[[墨西哥君主列表|墨西哥皇帝]]马西米连诺一世。

== 早年生活 ==
马克西米连生于[[维也纳|维也纳]]的[[美泉宫|美泉宫]]，是[[奥地利|奥地利]]的[[弗兰茨·卡尔|弗兰茨·卡尔]]大公与[[巴伐利亚|巴伐利亚]][[维特尔斯巴赫家族|维特尔斯巴赫家族]]的公主[[苏菲皇太后_(奥匈帝国)|苏菲]]的次子，[[奥地利君主列表|奥地利皇帝]][[弗兰茨·约瑟夫一世|弗兰茨·约瑟夫一世]]之弟。

马克西米连小时候非常聪明，显示出对[[艺术|艺术]]浓厚的兴趣，对[[科学|科学]]特别是[[植物学|植物学]]也相当感兴趣。他后来加入了[[海军|海军]]，由于他对这项事业的热忱，以及他在[[的里雅斯特|的里雅斯特]]军港的建设和[[意大利|意大利]]战争中的重要作用，他被迅速提升为海军总司令。他是个有名的[[自由主义|自由主义]]者。1857年2月，他被任命为[[倫巴第-威尼西亞王國|倫巴第-威尼西亞王國]]总督。

马克西米连在1857年7月21日与[[比利时|比利时]]国王[[利奥波德一世_(比利时)|利奥波德一世]]之女[[卡洛塔皇后_(墨西哥)|夏洛特]]公主结婚。婚后他们居住在[[米兰|米兰]]，直到1859年[[奥地利皇帝|奥地利皇帝]][[弗兰茨·约瑟夫一世|弗兰茨·约瑟夫一世]]解除了他的总督职务。因为皇帝对他在[[意大利|意大利]]实施[[自由主义|自由主义]]政策很是不满。不久，[[哈布斯堡王朝|哈布斯堡王朝]]失去了在[[意大利|意大利]]的领地。马克西米连于是在[[的里雅斯特|的里雅斯特]]的米拉马尔城堡里享受私人生活。

== 墨西哥王位的诱惑 ==
1859年，[[墨西哥|墨西哥]]的[[君主立宪|君主立宪]]派想在[[欧洲|欧洲]]的[[天主教|天主教]]王室中挑选一位王子作为[[墨西哥|墨西哥]]的君主，他们找到了马克西米连。马克西米连起初没有接受这个计划，而是去[[巴西|巴西]]探险研究[[南美洲|南美洲]]的[[热带雨林|热带雨林]]的植物。然而在1862年，[[法国|法国]]以“索债”为名，联合[[英国|英国]]、[[西班牙|西班牙]][[法國武裝干涉墨西哥|入侵墨西哥]]。墨西哥总统[[胡亚雷斯|胡亚雷斯]]不敌，被迫退入山区，[[墨西哥城|墨西哥城]]被法军占领。由于法皇[[拿破仑三世|拿破仑三世]]鼓动和马克西米连妻子夏洛特（后来的''卡洛塔皇后''）想做皇后的欲望，及一次没有法律效果的的[[公民投票|公民投票]]（从墨西哥传来的消息表明：胡亚雷斯在国内仍很有影响，[[君主制|君主制]]的政治力量只集中在墨西哥城和周边小部分地区；正在经历[[南北战争|南北战争]]的[[美国总统|美国总统]][[林肯|林肯]]也不同意墨西哥成为[[君主制|君主制]]国家），马克西米连于1863年同意接受[[墨西哥君主列表|墨西哥皇位]]。这个决定遭到了[[哈布斯堡家族|哈布斯堡家族]]大部分成员的反对，使他和他的后代失去了对[[奥地利|奥地利]]皇位的继承权，这件事马克西米连是在上船前往墨西哥前夕被告知的。

1864年5月，在航向墨西哥的船上，他撰写了法院礼仪指南，而不是阅读与墨西哥相关的书籍。

== 墨西哥皇帝 ==
[[File:Maximilian_of_Mexico_Winterhalter.jpg|thumb]]绘]]
1864年5月28日，皇帝在[[韦拉克鲁斯|韦拉克鲁斯]]登陆。他发现自己被卷入了法国军队与墨西哥[[共和主义|共和主义]]者的战斗中，他的统治没有得到前总统[[胡亚雷斯|胡亚雷斯]]的承认。

皇帝和皇后将他们的宫殿设在[[墨西哥城|墨西哥城]]郊外的Chapultepec山上，皇帝开辟了一条穿过市区从Chapultepec到市中心的马路，并命名为''皇帝大道''，也就是现在著名的''改革大道''。

由于皇帝与皇后没有子嗣，他们收养前[[墨西哥君主列表|墨西哥皇帝]][[阿古斯汀一世_(墨西哥)|阿古斯汀·德·伊图尔维德]]的孙子[[阿古斯汀三世_(墨西哥)|小阿古斯汀]]和[[萨尔瓦多亲王_(墨西哥)|萨尔瓦多]]为养子，授予他们''伊图尔维德亲王殿下''的头衔，以[[阿古斯汀三世_(墨西哥)|小阿古斯汀]]为[[墨西哥君主列表|墨西哥皇位]]继承人。

皇帝继续保持了胡亚雷斯提出的[[土地改革|土地改革]]、[[宗教宽容|宗教宽容]]、扩大选举权等[[自由主义|自由主义]]政策。皇帝曾试图赦免胡亚雷斯，前提是胡亚雷斯承认他的皇位，但遭到了胡亚雷斯的拒绝。

1866年，[[拿破仑三世|拿破仑三世]]因国内的抵抗和美国的反对开始逐步从墨西哥撤军。法军的撤离对墨西哥皇室是个巨大的触动。[[卡洛塔皇后_(墨西哥)|卡洛塔皇后]]返回欧洲，到处游说，反对法国撤军，在[[巴黎|巴黎]]、[[维也纳|维也纳]]、[[罗马|罗马]]寻求援助。她的努力失败了，并且从未回到墨西哥。而皇帝拒绝离开他的追随者。1867年2月，皇帝退守[[克雷塔罗|克雷塔罗]]，并以之为[[首都|首都]]。围困在此数周後，在5月11日试图突破封锁线时被捕。他被墨西哥的[[军事法庭|军事法庭]]以颠覆墨西哥共和国的罪名判处枪决。

得知此消息后，欧洲各国[[君主|君主]]、美国总统[[安德鲁·约翰逊|安德鲁·约翰逊]]，甚至[[維克多·雨果|維克多·雨果]]、[[加里波第|加里波第]]等[[民主人士|民主人士]]都致信胡亚雷斯，要求其不要枪决皇帝，均遭到胡亚雷斯的反对。

== 枪决马西米连诺皇帝 ==
[[File:Edouard_Manet_022.jpg|thumb]]

1867年6月19日星期三凌晨三点，行刑队押解皇帝和他手下的两位将军[[米格尔·米拉蒙|米格尔·米拉蒙]]和[[托马斯·梅西拉|托马斯·梅西拉]]去执行死刑，皇帝身穿黑色制服，胸前佩带着[[哈布斯堡家族|哈布斯堡家族]]的[[金羊毛勋章|金羊毛勋章]]。[[神父|神父]]为皇帝主持[[弥撒|弥撒]]之后，被押赴刑场[[克雷塔罗|克雷塔罗]]的克罗什高地。他把他的帽子给了他的[[匈牙利|匈牙利]]厨师，并对他说：“把这个交给我的母后。请转告她，我在临终前仍然思念着她。”皇帝与行刑队成员一一握手，然后站到离他们几米远的矮墙前。“我愿宽恕天下人”，皇帝大声说道，“但愿天下人能宽恕我，为了国家的利益，我甘洒热血，墨西哥万岁！独立万岁！”行刑队员举起了指挥刀。“孩子们瞄准了，瞄准正前方！”皇帝一边叫喊，一边指着胸膛。枪声响起，皇帝应声倒下。有人听到他低语：“灾难！（Hombre!）”

[[File:Maximilian.von.Mexico.jpg|thumb]]
约十天之后，消息传到[[欧洲|欧洲]]，[[哈布斯堡家族|哈布斯堡家族]]沉浸在一片悲恸之中。皇帝的尸体经过数月才运回[[奥地利|奥地利]]。次年2月，[[奥匈帝国|奥匈帝国]]为其举行了[[国葬|国葬]]，最终安葬在[[维也纳|维也纳]]的皇家墓室。
