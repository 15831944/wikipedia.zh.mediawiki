'''松野觉'''（{{bd|1918年||1944年|3月5日}}），[[日本|日本]][[广岛县|广岛县]]人，1940年1月随[[侵华日军|侵华日军]]丸山旅团到中国作战，1941年被俘后参加[[新四军|新四军]]，后任日本人反战同盟苏中支部宣传委员，1944年3月5日牺牲于[[车桥战役|车桥战役]]。

== 生平 ==
1918年，松野觉生于日本广岛，1940年1月被征入侵华日军丸山旅团到中国作战。1941年12月8日，松野觉在江苏如东县双灰山战斗中被[[新四军第1师|新四军1师]]3旅8团俘虏。被俘之初，松野觉态度曾十分坚决，但和新四军部队共同生活几个月后，思想发生转变，逐步认识到日本帝国主义发动战争不仅给中国人民带来灾难，也给日本人民带来痛苦，认清了侵华战争的罪恶，特地退还了发放给他的特殊津贴，并要求加入新四军。鉴于松野觉的表现，他参加新四军的要求得到了批准。1942年秋，[[日本人反战同盟|日本人反战同盟]]成立苏中支部，松野觉当选为支部宣传委员。1943年秋，[[日本人民解放联盟|日本人民解放联盟]]苏中支部在苏北成立，松野觉等7位反战同盟成员宣誓加入[[共产党|共产党]]<ref name="changmian">{{cite web |title=长眠在江苏车桥的“日本人反战同盟”成员松野觉 |url=http://www.zytzb.gov.cn/tzb2010/s1488/201505/73dae80ebf9a4e8895fe295a630c0165.shtml |website=人民政协报 |accessdate=2018-07-30 |archive-url=https://web.archive.org/web/20180730151615/http://www.zytzb.gov.cn/tzb2010/s1488/201505/73dae80ebf9a4e8895fe295a630c0165.shtml |archive-date=2018-07-30 |dead-url=no }}</ref>。

1944年，新四军对日伪展开军事攻势，反战同盟积极配合进行反战宣传，松野觉多次到前线散发传单、劝说日本士兵。3月，新四军第1师发起[[车桥战役|车桥战役]]，松野觉等人随队参战。战斗打响后，他紧跟突击队后向日军喊话。3月5日凌晨，日军已只剩下最后一座堡垒未被击破，面对松野觉的喊话，堡垒中的日本兵并未停止射击，松野觉拿枪参加了战斗。在命中两名日本士兵后{{efn|也有资料并未提及松野觉曾拿枪参战，仅说其在喊话时中弹牺牲<ref name="changmian"/>。}}，松野觉被敌人打中头部，当场牺牲，时年仅26岁<ref>{{cite web |author1=陈劲松 |title=抗战中的“在华日人反战同盟”（下） |url=http://www.people.com.cn/GB/paper39/14801/1312972.html |website=人民网 |accessdate=2018-07-30 |archive-url=https://web.archive.org/web/20180730151615/http://www.people.com.cn/GB/paper39/14801/1312972.html |archive-date=2018-07-30 |dead-url=no }}</ref>。

== 纪念 ==
松野觉牺牲后，新四军专门为他召开了追悼会，《苏中报》出专刊对他在抗日战争中作出的贡献给予了高度评价。时任新四军第一师副师长[[叶飞|叶飞]]赞扬说：“松野觉的战场喊话是真理的声音，震撼了那些远离故国的日本士兵的心弦<ref>{{cite web |author1=曹晋杰 |title=从战俘到优秀革命者——记文化战士松野觉 |url=http://dangshi.people.com.cn/n/2015/0916/c85037-27593643.html |website=中国共产党新闻网 |accessdate=2018-07-30}}</ref>。”1944年3月25日至5月20日，《[[解放日报|解放日报]]》多次介绍松野觉烈士的事迹。6月22日，时任中共中央军委参谋长[[叶剑英|叶剑英]]面对中外记者参观团高度赞扬松野觉的国际主义精神<ref>{{cite web|url=http://www.djnb.cn/journal_articles/view/817|title=车桥战役和日本战友|accessdate=2018-02-22|author=丁星|date=2011-09|publisher=大江南北杂志|archive-url=https://web.archive.org/web/20180223050835/http://www.djnb.cn/journal_articles/view/817|archive-date=2018-02-23|dead-url=no}}</ref>。

[[中华人民共和国民政部|中华人民共和国民政部]]于2015年8月24日公布第二批《[[著名抗日英烈和英雄群体名录|著名抗日英烈和英雄群体名录]]》，松野觉的名字被收录其中<ref>{{cite web|url=http://politics.people.com.cn/n/2015/0825/c1001-27512283.html|title=民政部公布第二批600名著名抗日英烈和英雄群体名录|date=2015-08-24|publisher=中華人民共和國民政部|access-date=2016-07-21|archive-url=https://web.archive.org/web/20151017224402/http://politics.people.com.cn/n/2015/0825/c1001-27512283.html|archive-date=2015-10-17|dead-url=no}}</ref>。
