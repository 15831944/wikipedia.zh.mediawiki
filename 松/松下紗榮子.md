'''松下紗榮子'''（{{lang-ja|松下紗栄子}}，{{bd|1990年|9月30日}}），[[日本|日本]]的[[AV女優|AV女優]]，[[北海道|北海道]]出身<ref name="T-Powers"/>。目前為「[[Attackers|Attackers]]」專屬女優<ref name="Attackers">{{Cite web |url=https://www.attackers.net/actress/detail/1030550/ |title=女優情報：松下紗栄子 |publisher=[[Attackers|アタッカーズ]] |accessdate=2018-09-16|language=ja}}</ref>。所屬於「[[T-POWERS|T-POWERS]]」事務所<ref name="T-Powers"/>。

== 人物 ==
* 興趣是[[閱讀|閱讀]]跟[[日本茶道|茶道]]<ref name="T-Powers"/>。

== 簡歷 ==
2015年5月，以「副職AV女優」的名號成為日本[[AV片商|AV片商]]「[[Soft_On_Demand|SOD]]」的専屬女優[[AV|AV]][[出道|出道]]，[[專業人士|本職]]是擔任[[禮儀|禮儀]][[講師|講師]]以及大型[[航空公司|航空公司]]前[[空服員|空服員]]<ref>{{cite web|url=http://www.playno1.com/article-16479-1.html#.W55EHegzaUk |title=美麗空姐來兼差! - 松下紗栄子 |publisher=PLAYNO.1玩樂達人 |date=2015-06-09}}</ref>。

2016年5月，成為「[[Attackers|Attackers]]」的專屬AV女優。2017年7月跟9月短暫的在「{{link-ja|溜池ゴロー|溜池ゴロー}}」發布作品<ref>{{cite web|url=http://www.playno1.com/article-23608-1.html#.W55HqugzaUk |title=太雞巴的下場！松下紗栄子慘遭無套中出！ |publisher=PLAYNO.1玩樂達人 |date=2017-06-14}}</ref>。

2018年10月12日，日本[[電子商務|電子商務]]網站《FANZA》（原《[[DMM.com|DMM.R18]]》）公布網站統計資料<ref>{{cite web|url=https://kai-you.net/article/58273 |title=FANZA「性に関する統計調査」が話題　世界で高まる「熟女」の需要 |publisher={{link-ja|KAI-YOU|KAI-YOU}} |date=2018-10-12 |language=ja}}</ref>，松下紗榮子在年齡層45歲以上觀眾的「最受歡迎AV女優」排行榜都獲得第1名<ref>{{cite web|url=http://www.playno1.com/article-28236-1.html#.XA5Y32gzaUk |title=45歲以上影迷的最愛：松下紗栄子 |publisher=PLAYNO.1玩樂達人 |date=2018-10-16}}</ref><ref>{{Cite news|url=https://www.setn.com/News.aspx?NewsID=443742 |title=這AV女優通殺這個年齡層　靠的就是這些關鍵 |publisher=[[三立新聞|三立新聞]]|date=2018-10-18}}</ref>。

2019年2月，個人在片商「BeFree」發行的作品《彼女が四日間家族旅行で不在の間、彼女のお姉さんと夢中で中出ししまくった》，拿下《FANZA》網站DVD銷售排行榜以及女優累積銷售排行榜的雙料冠軍<ref>{{Cite news|url=https://www.setn.com/News.aspx?NewsID=495376&From=Search&Key=%E6%9D%BE%E4%B8%8B%E7%B4%97%E6%A0%84%E5%AD%90 |title=新作品竟送「內褲」！她幹掉高橋聖子登基成AV女皇 |publisher=[[三立新聞|三立新聞]]|date=2019-02-04}}</ref>。
