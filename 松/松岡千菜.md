== 人物・簡歷 ==
松岡千菜於2014年7月28日發售的《[[週刊Playboy|週刊 Playboy]]》裡，以「史上最強素人Nude」—— [[寫真偶像|寫真偶像]]「'''<ruby><rb>千菜</rb><rp>(</rp><rt>ちな</rt><rp>)</rp></ruby>（China）'''」身份出道。同年12月，她正式以「[[SODクリエイト|SOD Create CO.,LTD.]] 旗下AV女優」身份出道<ref name="wpb"/>。

2015年1月，她獲[[埼玉電視台|埼玉電視台]]起用為[[真人實境秀|真人實境節目]]《{{link-ja|Beach9|ビ〜チ9}}》助手。同年6月，她於「{{link-ja|AV OPEN 2015|AV OPEN〜あなたが決める!セルアダルトビデオ日本一決定戦〜}}」活動內與[[佐倉絆|佐倉絆]]和[[天使萌|天使萌]]一同獲選為打氣女郎。

2016年3月3日，松岡千菜在《[[成人廣播獎|成人廣播獎]]2016》中奪得「'''新人女優獎'''」及「'''FLASH獎'''」<ref>{{cite web |url=http://nikkan-spa.jp/1066590 |title=“史上初の4冠”ナンバーワンAV女優が決定！「スカパー！アダルト放送大賞2016」写真レポ |newspaper=日刊SPA! |date=2016-03-04 |accessdate=2016-03-05}}</ref>。同年6月1日，她於自己[[Twitter|Twitter]]中表示遇上車禍，以致全身受傷，肇事司機亦不顧而去<ref>{{cite web |url=https://twitter.com/china_0416/status/737911767418953729 |title=松岡ちな Lv22さんのツイート: "実は昨日夜中にコンビニに行ってたら追突事故に巻き込まれてしまい、全身傷だらけになってしまって1日入院しました💦 普通の生活も痛いけど出来るし命に別条はないけど結構大きな傷ができてしまって怪我の具合と体調見て今後のお仕事の話し合いを今日まみー含めたBOSSとしてきます。"|date=2016-06-01 |accessdate=2016-06-01}}</ref><ref>{{cite web |url=http://www.narinari.com/Nd/20160637983.html |title=松岡ちながひき逃げに遭い全身けが「ながらスマホは危ないと再認識」。|newspaper=Narinari.com |date=2016-06-05 |accessdate=2016-06-05}}</ref>。而7月19日，她也在自己的 [[Twitter|Twitter]] 中透露内臟出現問題，並已入院治療<ref>{{cite web |url=https://twitter.com/china_0416/status/755244350209036288 |title=実は事故の後に休養貰いに実家に帰ったんですが、帰ってすぐある内臓系の病気になって入院してます。|date=2016-07-19 |accessdate=2016-07-20}}</ref>。至8月15日就以四頁紙書信向 SOD Create CO.,LTD. 請辭<ref>{{cite web |url=https://twitter.com/SOD_magazine/status/765096315780866048 |title=5月末から交通事故のため休養していた松岡ちなちゃんからSODに手紙が届きました。そこには、引退を表明する内容が綴られていました…。編集部はちなちゃんの今の声を聞くべくインタビューを敢行し、手紙全文と共に18日発売号に掲載いたします。|date=2016-08-15 |accessdate=2016-07-16}}</ref><ref>{{cite web |url=https://twitter.com/china_0416/status/766031880567201792 |title=松岡ちなは引退することに致しました。今までありがとうございました。|date=2016-08-18 |accessdate=2016-08-18}}</ref>。
