'''松平忠輝'''（1592年－1683年）是[[安土桃山時代|安土桃山時代]]至[[江戶時代|江戶時代]]中期的[[大名|大名]]。父親是[[德川家康|德川家康]]。母親是側室[[茶阿局|茶阿局]]。

== 生平 ==
=== 出生 ===
在[[天正_(日本)|天正]]20年（1592年）1月4日於[[江戶城|江戶城]]出生，家中六男。幼名是'''辰千代'''。

因為生母[[茶阿局|茶阿局]]身分低微的關係，家康在辰千代剛誕生時就不喜歡他，在捨棄他時令側近[[本多正信|本多正信]]拾回（當時有這種風習：為了祈願平安養大孩子而把孩子放在寺門前，然後由家臣拾回並養育），並尋求養育人，於是由3萬5千石的[[下野|下野]]栃木（長沼）城城主[[大名|大名]][[皆川廣照|皆川廣照]]養育。

家康與忠輝第一次見面是在[[慶長|慶長]]3年（1598年），不過此時家康亦相當討厭忠輝（後述）。

=== 繼承長澤松平氏 ===
[[File:Keep_Tower_of_Takada_Castle_20121202,_003.jpg|thumb]][[上越市|上越市]]）]]
慶長4年（1599年）1月，因為弟弟[[松平松千代|松千代]]死去，於是忠輝繼承[[長澤松平家|長澤松平家]]的[[家督|家督]]，被賜予[[武藏國|武藏國]]深谷1萬石。在慶長7年（1602年）12月被加增移封至[[下總國|下總國]]佐倉5萬石並[[元服|元服]]而改名為'''上總介忠輝'''。

慶長8年（1603年）2月，被加增移封至[[信濃國|信濃國]][[川中島|川中島]]12萬石（與上次移封只相差40天），於是姐夫[[花井吉成|花井吉成]]以[[家老|家老]]的身份輔佐忠輝。慶長10年（1605年），[[德川秀忠|秀忠]]在就任[[豐臣秀賴|豐臣秀賴]]的[[右大臣|右大臣]]時，家康下令忠輝前往[[大坂|大坂]]與秀賴會面。慶長11年（1606年），與[[伊達政宗|伊達政宗]]的長女[[五郎八姬|五郎八姬]]結婚。慶長14年（1609年），重臣[[皆川廣照|皆川廣照]]等人引起[[御家騷動|御家騷動]]而失勢。

慶長15年（1610年），被任命為[[越後|越後]][[高田藩|高田藩]]藩主（[[福島城_(越後国)|福島城]]城主，後述），此時加上川中島12萬石，合計擁有75萬石。領有越後的忠輝當初是[[堀氏|堀氏]]所築的福島城城主，不過在慶長19年（1614年）築起[[高田城|高田城]]，於是移到該處。高田城因為幕命的關係，由忠輝的義父[[伊達政宗|伊達政宗]]為首等13家[[大名|大名]]幫助築造。

=== 改易、流放 ===
不過此時並沒有減少與父親[[德川家康|家康]]的距離，在[[慶長|慶長]]19年（1614年）的[[大坂冬之陣|大坂冬之陣]]中被任命擔任留守居役。剛毅的忠輝對這個命令相當不滿，不過最後都有遵從。在慶長20年（1615年）的[[大坂夏之陣|大坂夏之陣]]中向[[大坂|大坂]]出陣。

[[元和_(後水尾天皇)|元和]]2年（1616年）4月，家康死去。家康在此時令[[德川秀忠|秀忠]]、[[德川義直|義直]]、[[德川賴宣|賴宣]]、[[德川賴房|賴房]]等人前往見最後一面，不過沒有召喚忠輝。希望拜謁家康的忠輝隻身前往[[駿府|駿府]]，不過沒有被允許會見家康的最後一面。『[[德川實紀|德川實紀]]』中記載「忠輝立即出發到駿府探望家康，此時隨侍在家康身旁的有許多宿老政要，但家康因為不喜歡忠輝，所以獨獨不准忠輝入城，因此無法實現探望之願。少將（忠輝）手足無措只好暫居城下的禪寺，希望能等到入城許可。最後忠輝在無法進城謝罪的情況下家康便駕崩……」（{{lang|ja|忠輝、いそぎ発途して駿府へ参られ、宿老もて御気しき伺はれしに。家康は以の外の御いかりにて。城中へも入るべからざる旨仰下され。御対面も叶はざれば。少将せんかたなく御城下の禅寺に寓居して。御気のひまを伺ひて。謝し奉られんとする内に薨去……}}）。

[[天和_(日本)|元和]]2年（1616年）7月6日，忠輝被兄長[[德川秀忠|秀忠]]命令改易並流放至[[伊勢國|伊勢國]]朝熊。忠輝的母親[[茶阿局|茶阿局]]向家康的側室[[阿茶局|阿茶局]]求助，但是最終都失敗。在元和4年（1618年）和[[寬永|寬永]]3年（1626年）分別被流放至[[飛驒國|飛驒國]]高山和[[信濃國|信濃國]]諏訪。

[[Image:Grave_of_Matsudaira_Tadateru.jpg|thumb]][[諏訪市|諏訪市]]的[[貞松院|貞松院]]）]]
天和3年（1683年）7月3日，在諏訪[[高島城|高島城]]（南之丸）被幽禁期間死去。享年92歳。

兒子[[松平德松_(江戶時代中期)|德松]]（母親是竹之局）在忠輝流放之際沒有被允許同行，更被[[岩槻藩|岩槻藩]]藩主[[阿部重次|阿部重次]]軟禁並冷遇，在寬永9年（1632年）於住居放火並自殺（享年18歲）。

一說為與家康的關係被隱藏，關於此事有[[乃可勢|野風之笛]]的逸話。這支笛子由[[織田信長|織田信長]]傳至[[豐臣秀吉|豐臣秀吉]]，再傳到家康手上，被認為是天下人的象徴，家康通過茶阿局把此笛傳至忠輝。現今此笛保存在[[長野縣|長野縣]][[諏訪市|諏訪市]]的[[貞松院|貞松院]]。

== 赦免 ==
德川宗家的赦免是在忠輝死去300年後的[[昭和|昭和]]59年（1984年）。忠輝的[[菩提寺|菩提寺]]貞松院的住職[[山田和雄|山田和雄]]建議在第300次死忌中赦免忠輝，並向德川宗家第18代當主[[德川恒孝|德川恒孝]]提出請求而實現。於是在7月3日因為恒孝而被赦免，向佛前奉告則是在貞松院的檀信徒在3年後的昭和62年（1987年）10月24日進行。在10月24日的[[法要|法要]]中有[[仙台|仙台]]伊達家當主的妹妹和諏訪家當主、當時家臣的子孫等大約400人參加，由恒孝讀出赦免狀。

一說原因是因為受到[[隆慶一郎|隆慶一郎]]的小説『{{lang|ja|捨て童子・松平忠輝}}』刺激而造就了這次赦免，不過此作在『[[信濃毎日新聞|信濃毎日新聞]]』等連載是在昭和62年（1987年）5月22日開始，德川恒孝的赦免狀比連載早很多。

== 被父親討厭的理由 ==
與次兄[[結城秀康|結城秀康]]同樣，忠輝由出生開始就被父親討厭。理由可能是忠輝的母親和秀康的母親同様身分低下，不過有許多記錄顯示這是跟忠輝的容貌有關。『[[藩翰譜|藩翰譜]]』記載「世間傳說在介殿（忠輝）出生時，德川殿（家康）看到（忠輝）膚色極黑、大力張眼而感到恐怖並感到憎惡，於是說出想捨棄」（{{lang|ja|世に伝ふるは介殿生れ給ひし時、徳川殿御覧じけるに色きわめて黒く、まじなりさかさまに裂けて恐しげなれば憎ませ給ひて捨てよと仰せあり}}）。即是家康以剛出生的忠輝容顔醜惡為理由而將其捨掉。『藩翰譜』記載慶長3年，在忠輝7歳時，與忠輝會面的家康說「真是可怕的樣貌，與三郎（[[松平信康|松平信康]]）年幼時無異」（{{lang|ja|恐ろしき面魂かな、三郎が幼かりし時に違ふところなかりけり}}）。亦有其他史料記述此年的事情，家康看到忠輝後說「面貌怪異，與三郎的幼顏相似」（{{lang|ja|面貌怪異、三郎ノ稚顔ニ似タリ}}）。家康在看見忠輝時就像見到信康一樣。與同樣遭到改易的[[松平忠直|松平忠直]]等人同様，忠輝亦有粗暴的一面。

由於忠輝順利地出世，最終成為75萬石的太守，因此許多說法指這是因為家康想補償給忠輝。不過在慶長11年（1606年）擔任[[川中島|川中島]]12萬石太守的時期，弟弟[[德川義直|義直]]（7歳）被賜予[[甲斐|甲斐]]府中25萬石，[[德川賴宣|賴宣]]（5歳）被賜予[[常陸|常陸]][[水戶藩|水戶藩]]25萬石，[[德川賴房|賴房]]（4歳）被賜予[[常陸|常陸]][[下妻藩|下妻藩]]10萬石。亦有家康之所以賜予忠輝領地是因為[[伊達政宗|伊達政宗]]和[[茶阿局|茶阿局]]等人的關係。而且弟弟[[松平松千代|松千代]]（同父同母）亦在年幼時在形式上繼承[[長澤松平氏|長澤松平氏]]並被賜予[[武藏|武藏]][[深谷藩|深谷藩]]，而同時期的忠輝卻沒有被賜予特別的獎賞。弟弟松千代在夭折後，身為兄長的忠輝繼承松千代的領地，這時情況才有好轉。

== 被改易的理由 ==
在家康死後的[[天和_(日本)|元和]]2年（1616年）7月6日，忠輝被兄長[[德川秀忠|秀忠]]命令改易，理由有多種說法：
*在[[大坂夏之陣|大坂夏之陣]]時，被任命為從[[大和|大和]]攻入[[大坂|大坂]]的總大將，最後卻遲到。
*忠輝軍向大坂進軍途中，把在[[近江|近江]]守山超越過軍列的[[長坂信時|長坂信時]]等人（秀忠的直屬[[旗本|旗本]]）斬殺（不過在當時的軍法中，於戰爭中越過友軍前進是極之無禮的事，而忠輝的做法是所謂的「[[切捨御免|切捨御免]]」，因此這樣的處置屬於合法）。
*為了向[[朝廷|朝廷]]上奏在大坂夏之陣的勝利，家康命令忠輝一同前往參見。但是忠輝卻以生病為理由來拒絕同行，而且還前往[[嵯峨野|嵯峨野]]並在[[桂川_(淀川水系)|桂川]]乘舟遊玩。

以上就是秀忠命令改易的表面理由。不過實際上亦可能是有以下的理由：
*與[[基督教|基督教]]有親近的關係。
*忠輝與岳父[[伊達政宗|伊達政宗]]、以及[[幕府|幕府]]內部兼任[[奉行|奉行]]職而有莫大財力和權力的[[大久保長安|大久保長安]]相當親近，所以秀忠很早就對忠輝非常警戒和厭惡。

== 人物 ==
*雖然已經敘任[[從四位下|從四位下]][[左近衛權少將|左近衛權少將]]，不過一生自稱為'''[[上總介|上總介]]'''。因此在部份史書中，在忠輝成為少將後，亦被記錄為'''上總介'''。
*顯示出與海外交易的興趣，愛好武術的同時，亦是精通[[茶道|茶道]]、[[繪畫|繪畫]]、藥學的文化人。
