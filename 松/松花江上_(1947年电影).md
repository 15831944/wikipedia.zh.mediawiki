'''松花江上'''，1947年中国电影，由[[金山_(演员)|金山]]导演，[[张瑞芳|张瑞芳]]、[[浦克|浦克]]、[[朱文顺|朱文顺]]主演。

== 拍摄背景 ==
1945年8月，随着[[日本投降|日本的无条件投降]]，[[满洲国|满洲国]]覆灭，大型制片厂[[满洲映画协会|满洲映画协会]]也宣告垮台，遗留下来一大批摄影及整套制片生产设备，成为国共两党争夺的目标<ref>{{cite journal |author1=邵功游 |title=创业“东影”的难忘往事 |journal=大众电影 |date=2002 |issue=24}}</ref>。中共接收满映后将部分设备迁出，1946年，国民政府委派金山到长春接收满映，筹建[[长春电影制片厂|长春电影制片厂]]，并出任厂长<ref>{{cite web |title=电影演员金山照片帅气 赵默与张瑞芳婚姻危急遇小三果断 |url=https://www.520730.com/yulequan/mingxing/562595.html |website=巴陵时尚网 |date=2016-10-10}}</ref>。1947年，金山利用接收的器材设备编导、拍摄了影片《松花江上》<ref>{{cite web |title=金山_沅陵当代人物专题 |url=http://ren.bytravel.cn/history/9/jinshan3355.html |website=博雅人物网}}</ref><ref>{{cite web |author1=夏衍 |title=夏衍同志在金山同志追悼会上致悼词 |url=http://www.lhlyy.com/xgwz/10532.html |website=乐遥园 |date=2017-10-15}}</ref>。
