'''松永真穗'''（{{lang-ja|まつなが まほ}}，{{bd|1993年|1月23日}}）是[[日本|日本]]的[[声優|女性声優]]、[[偶像|偶像]]。前[[Style_Cube|Style Cube]]所屬，前[[StylipS|StylipS]]、[[HAPPY!_STYLE|HAPPY!STYLE Rookies]]的成員。

== 略歴 ==
; 2008年
* 12月23日，『HAPPY! STYLE Xmas Special Talk LIVE "Tiny Xmas Fantasy"』首次披露<ref>{{Cite web |author=松永真穂|title=X'mas〜いぶ〜 | url=http://ameblo.jp/matsunaga-maho/entry-10418435189.html | publisher=アメーバブログ | date=2009-12-24 | accessdate=2011-05-22}}</ref>。
; 2011年
* 3月，[[高級中學|高中]]畢業<ref>{{Cite web |author=松永真穂|title=涙の向こうは| url=http://ameblo.jp/matsunaga-maho/entry-10827458875.html | publisher=[[アメーバブログ|アメーバブログ]] | date=2011-03-11 | accessdate=2011-05-22}}</ref>。
* 4月，進入[[大学|大学]]<ref>{{Cite web |author=松永真穂|title=え、いま何度ですか？| url=http://ameblo.jp/matsunaga-maho/entry-10861336641.html | publisher=アメーバブログ | date=2011-04-14 | accessdate=2011-05-22}}</ref>。
* 12月，作為[[StylipS|StylipS]]的成員之一活動開始<ref>{{Cite web | date=2011-12-01 | url=http://www.ota-suke.jp/news/59666 | title=新星声優ユニットのStylipSが始動、メッセージ動画を公開（おた☆スケ） | publisher=株式会社にゅーあきば | accessdate=2012-01-31}}</ref>。
'''2013年'''
* 4月13日、發表[[水面下的天空|水面下的天空]]的新團體的迷你專輯『5色』、擔任了做詞<ref>{{Cite web|url=http://ameblo.jp/matsunaga-maho/entry-11510201496.html|title=ぎるてぃ|work=松永真穂オフィシャルブログ「sweetすうぃーちゅ」Powered by Ameba|author=松永真穂|date=2013-04-12|accessdate=2013-04-16}}</ref>。
* 9月28日、最初的演唱會是白天跟晚上兩場。
* 10月10日、發表了[[PlayStation_3|PS3]]專用遊戲『[[妖精劍士f|妖精劍士f]]』的主題曲「Resonant World」以松永真穗（from StylipS）的名義歌唱<ref>{{Cite web|url=http://www.compileheart.com/fairyfencer_f/#/sound|title=SOUND|work=フェアリーフェンサー エフ|accessdate=2013-06-30}}</ref>
'''2016年'''
* 5月23日，發布於同年5月20日退出事務所Style Cube，並從Stylips畢業的消息。同時從聲優界引退，之後將以DJ做為主要的活動<ref>{{Cite web | date=2016-05-23 | url=http://stylips.jp/info/pg37.html | title=松永真穂卒業のお知らせ| publisher=StylipS Official Site | accessdate=2016-05-23}}</ref>。

== 特徴 ==
興趣是[[競技啦啦隊|啦啦隊]]、[[指揮棒|指揮棒]]、[[料理|料理]]、[[卡拉OK|卡拉OK]]。

[[視力|視力]]是2.0<ref>{{Cite web |author=松永真穂|title=視力| url=http://ameblo.jp/matsunaga-maho/entry-10837060939.html | publisher=アメーバブログ | date=2011-03-21 | accessdate=2011-05-22}}</ref>。對於自己圓圓的臉有看起來像是[[小朋友|小朋友]]的自覺<ref>{{Cite web |author=松永真穂|title=最近の機械の発達具合| url=http://ameblo.jp/matsunaga-maho/entry-10631519585.html | publisher=アメーバブログ | date=2010-08-27 | accessdate=2011-05-22}}</ref>。喜歡磨牙。除了門牙之外其他的都還是乳牙<ref>松永真穂の乙女★ろっくPodcastおまけ放送 2014年2月17日</ref>。

對粉絲用「比拉」來稱呼<ref group="注">松永真穂の乙女★ろっくのおまけ放送で決定。</ref>。

曾經說過對於同一間事務所的[[三澤紗千香|三澤紗千香]]「與其說是好朋友不如說是競爭對手」<ref>{{Cite web|url=http://www.koepota.jp/news/2013/02/08/0303.html|title=三澤紗千香、松永真穂が20歳の抱負を語る！『Birthday LIVE 2013』公演後インタビュー！|work=こえぽた|date=2013-02-08|accessdate=2013-09-21}}</ref>。
