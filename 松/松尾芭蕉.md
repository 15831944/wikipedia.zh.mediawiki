'''松尾芭蕉'''{{noteTag|1= 雖然'''松尾'''是這位日本詩聖的本姓，但一般均以'''芭蕉'''直接稱呼他而不用姓氏，如芭蕉翁。}} （生於[[寬永|寬永]]21年（1644年），卒於[[元禄|元禄]]7年10月12日（1694年11月28日）），是[[日本|日本]][[江戶時代|江戶時代]]前期的一位{{link-ja|俳谐师|俳諧師}}的署名。他公认的功绩是把[[俳句|俳句]]形式推向顶峰，但是在他生活的时代，芭蕉以作为俳谐连歌（由一组诗人创作的半喜剧链接诗）诗人而著称，被誉为日本“俳圣”。<ref name=" xsk" >{{cite web |url=http://www.jnocnews.jp/news/show.aspx?id=28274 |title=俳圣——松尾芭蕉 |accessdate= 2009年|work=|publisher=日本新华侨报网 |language=zh-hans}}</ref>{{noteTag|1=在19世纪，[[连歌|连歌]]的开始一节（称为[[和歌|和歌]]）发展成独立的诗体，称为{{link-ja|俳谐|俳諧}}。[[明治时代|明治时代]]的诗人[[正冈子规|正冈子规]]首先称其为[[俳句|俳句]]。}}

松尾芭蕉在年輕時就開始寫詩，在進入[[江戶|江戶]]後很快的成為全日本的知名人物。他曾以教書維生，但後來丟棄了在文學圈中的社交及都會生活，開始在日本各地漫遊，為了獲得靈感，甚至曾前往北方的荒野。他的作品也受到他第一手的生活經驗所影響，多半藉由一些簡單的事物來表達其感受。

== 名字 ==
松尾芭蕉出生於[[寬永|寬永]]21年（西元1644年），幼名'''金作'''，後又以'''宗房'''（{{lang|ja|むねふさ}}，[[寬文|寬文]]7年（西元1667年））為名。<ref name=" xsk" />在他一生曾用了許多筆名，如早期取的[[俳號|俳號]]'''桃青'''（[[延宝|延宝]]3年（西元1675年）），用以表示尊崇[[唐朝|唐朝]]詩仙[[李白|李白]]，「桃青」與「李白」是[[對偶|對偶]]，前者指的是未成熟的桃子或青色桃子，而後者則為白色李子。

松尾芭蕉的弟子李下於[[延寶|延寶]]9年（1681年）送了一株[[芭蕉|芭蕉]]樹給他，並種在他當時隱居的庭園內（後來因此命名為芭蕉庵）<!--<ref>[http://www.ese.yamanashi.ac.jp/~itoyo/basho/haikusyu/basewo.htm ]-->。不久後，他在天和2年（西元1682年）開始使用'''芭蕉'''為俳號。意指氣候太冷以致無法生出果實，此外他有意使用這個筆名喚回詩作無用的想法，或企圖影響詩無用論。松尾芭蕉的許多筆名很少以[[羅馬拼音|羅馬拼音]]翻譯，除了[[日語羅馬字|日本式]]拼音'''Matuo Basyou'''外，還有以羅馬拼音對應的'''Matuwo Baseu'''拼法用來作英文生平介紹時使用。

== 生平 ==
[[File:Haiseiden.jpg|left]]]]
他出生于[[伊贺国|伊贺国]]的上野（今[[三重县|三重县]][[伊贺市|伊贺市]]），靠近[[京都|京都]]。他是低级武士的儿子，一开始是当地领主{{link-ja|藤堂良忠|蝉吟}}的侍童，藤堂良忠只比他大两岁。他们都喜欢写[[俳句|俳句]]，松尾芭蕉第一次出名是在1662年。在1664年在京都出版了他的第一部俳谐诗集。大约在这段时间，他起了武士的名“宗房”。在1666年，他的主人良忠逝世，松尾芭蕉选择离开这个家而不是留下来服侍新的主人。<ref name=" xsk" />他的父亲于1656年逝世。

传统上认为在接下来的6年中，他至少有一部分时间生活在京都；在这段时间他出版了几部[[诗选|诗选]]。1672年他迁往[[江户|江户]]（现在的[[东京|东京]]）。他继续写作，到1676年他被公认为俳句大师，出版自己的文学小册子并且给诗歌比赛做裁判。他接受了一批追随的弟子，这些弟子在1680年为他建了第一間芭蕉庵。

芭蕉发现他的成功是不能令人满意的，而转向了[[禅宗|禅宗]]（[[沉思|沉思]]）以求慰藉。1682年的冬天芭蕉庵被焚毁，他的母亲于1683年早些时候去世。1683年的冬天他的弟子为他建了第二間芭蕉庵，但是他还是不满意。1684年的秋天他开始他后来称之为《曠野紀行》或《饱经风霜的骷髅之紀录》（{{link-ja|野ざらし紀行|野ざらし紀行}}<!--；原作見[http://www.ese.yamanashi.ac.jp/~itoyo/basho/nozarasi/nozara01.htm]-->）的旅行；在这次旅行结束时他创作了《散文之旅》和一些诗歌。这次旅途他从江户到富士山，到伊勢，上野和京都，1685年夏天回到江户。

由於松尾芭蕉在旅程中展現的快速步伐，有些人認為他可能當過[[忍者|忍者]]。而他的長程旅途同時也讓他得以觀察列國的狀況和得知最新的消息，包括獲得[[德川幕府|德川幕府]]的相關情報。當時的德川幕府本身也派遣許多忍者進行查探消息的工作。松尾芭蕉的出生地就在[[伊賀國|伊賀國]]的上野，而伊賀就是眾多忍者的故里。早期他也可能曾作過藤堂良忠的隨從侍衛。然而，少數的文學領域的學者卻嚴正暗示松尾芭蕉其實是德川幕府的間諜。
[[File:Basho_in_Ogaki.JPG|thumb]]”終点站——大垣市的芭蕉像]]

旅程中看似成功消除了他心中的惡性，在接下來幾年的寫作也描述出松尾芭蕉對友好與自然的平靜愛好。他曾在貞亨4年（1687年）仲秋為了欣賞中秋[[滿月|滿月]]特地到[[鹿島神宮|鹿島神宮]]（位於[[鹿嶋市|鹿嶋市]]一遊。也因此寫了篇遊記提作《[[鹿島紀行|鹿島紀行]]》；原作見[http://www.ese.yamanashi.ac.jp/~itoyo/basho/kashimamoude/zenbun.htm]）。

同年10月在舉行如同達官顯要般的餞別宴會後，松尾芭蕉開始向另一階段的長途旅程出發。
旅程經過了上野、[[大坂|大坂]]、[[須磨區|須磨]]、{{link-ja|明石|明石}}、京都、[[名古屋|名古屋]]、日本中部地方山區。此外並前往更科里欣賞姨捨山（今[[長野縣|長野縣]][[千曲市|千曲市]]的冠著山）的中秋滿月。這段從上野到明石的旅程詳細敍述在《笈之小文》或《笈中小札》（原作見[http://www.ese.yamanashi.ac.jp/~itoyo/basho/oinokobumi/oino01.htm]）中。而在這部作品中他表現出以俳諧作為主要作品型式的信念。

另一部作品《更科紀行》（原作見[http://www.ese.yamanashi.ac.jp/~itoyo/basho/sarashina/sarasina.htm]）則描敍了賞月的另一段行程。

在1689年的春末，他開始了一段更具有挑戰性的[[本州島|本州島]]北部之旅。这次的行程包括[[日光市|日光]][[东照宫|东照宫]]、[[宫城县|宫城县]]的[[松岛|松岛]]、{{link-ja|象泻|象潟}}和[[石川县|石川县]]的[[金泽|金泽]]，最后路过[[佐渡岛|佐渡岛]]。且他做出了一本旅遊日誌——《[[奧之細道|奧之細道]]》或《奧州小道》，內容主要以[[侘寂|寂]]為思想概念以及人對自然的歸屬感<ref>{{cite news |author=有紀 |title=尋訪松尾芭蕉～奧之細道 |url=http://www.epochtimes.com/b5/5/9/23/n1061479.htm |work= |publisher=大紀元時報 |date=2005-09-23 |accessdate=2014-02-06}}</ref>。另外兩本書也給了他靈感，分別是《[[七日事記|七日事記]]》或《七天的紀錄》（『{{lang|ja|記紀がき 七日ぐさ}}』）和 《[[山中經|山中經]]》。

自1689年的秋天，芭蕉以兩年的時間拜訪朋友及到京都和[[琵琶湖|琵琶湖]]做短暫旅行。在这段时间，他完成了由他的几位学生编辑完成的俳谐集《[[猿蓑|猿蓑]]》（原作見[http://www.ese.yamanashi.ac.jp/~itoyo/basho/shitibusyu/sarumino.htm]），表现了他在北方之旅达成的美学观点。

元祿4年（1691年）冬，松尾芭蕉返回江戶並搬進第三間弟子為其搭建的芭蕉庵中。然而他卻不再維持獨居，而與兩個健康狀況不佳的人同住，一個是外甥桃印（[http://www.ese.yamanashi.ac.jp/~itoyo/basho/whoswho/tohin.htm]），另一個是至愛壽貞尼（[http://www.ese.yamanashi.ac.jp/~itoyo/basho/whoswho/jutei.htm]）。同時期來訪者絡繹不絕讓芭蕉庵門庭若市。松尾芭蕉曾在一封信中抱怨這個光景害他心無寧日。

元祿6年（1693年）秋，曾有長達一個月之久的時間松尾芭蕉拒絕接見任何人，如此閉關清心後才恢復心平氣和。沒有物欲毫無羈絆的生活信條讓他得以過著超越痛苦的隱居生活。

元祿7年（1694年）夏，松尾芭蕉最後一次離開京都，在到大坂前曾在上野和京都停留一段時間。最後因腹疾死於大坂，享年51歲，臨終前留下了最後一句俳句：

:旅途罹病，荒原驰骋梦魂萦{{noteTag|1=该俳句的意思是：行旅中病了，夢在枯槁的荒野上廻盪。原文：旅に病で、夢は枯野をかけ廻る}} <ref name=" xsk" />

== 事業 ==
[[File:Basho_by_Hokusai-small.jpg|thumb]]画]]

松尾芭蕉將一般輕鬆詼諧的喜劇詩句提昇為正式形式的詩體──俳句，並在詩作中灌輸了[[禪宗|禪]]的意境。事實上他多數俳句的前三行為長[[連歌|連歌]]（一些評論家認為此為其最佳創作）組成而不是各別單獨的作品。但這些詩句本身曾被多次選集並出版，而且他的詩作往往也是後期詩人如[[小林一茶|小林一茶]]和[[正岡子規|正岡子規]]等的最佳靈感來源。最有名的俳句之一要算松尾芭蕉在後江戶時期的作品──{{lang|ja|『たわらぼ』}}中的詠嘆詩句「松島啊，松島呀松島」（原文為「''{{lang|ja|松島やああ松島や松島や}}''」）頌揚[[松島|松島]]灣的無語之美。（但近代論者普遍以為此句並非芭蕉所寫，實為後世的狂歌師田原坊所作）此外，松尾芭蕉尤偏好在[[農曆|農曆]]十月十二以[[季語|季語]]「時雨」（意指秋之寒雨）來寫作。

1686年，創作了蛙的發句：
{{Quotation|古池や蛙飛びこむ水の音（ふるいけや かはづとびこむ みずのおと）　『蛙合』}}

松尾芭蕉一生遊遍了相當廣大的地區，而這些遊歷也充分反映在他的作品上。像他在元祿7年（西元1694年）完成的最佳傑作——《[[奧之細道|奧之細道]]》或《奧州小道》（原作見[https://web.archive.org/web/20050701020315/http://www.ese.yamanashi.ac.jp/~itoyo/basho/okunohosomichi/okuindex.htm]），就是一個很好的例子。在這部作品集中，松尾芭蕉以他最聞名的俳句為所經過的景色留下了引人入勝的翩翩詩-{采}-。
