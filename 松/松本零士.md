'''松本零士'''（{{lang-ja|まつもと れいじ}}，{{bd|1938年|1月25日|catIdx=Reiji, Matsumoto}}），原名'''松本晟'''，是一位著名的[[日本|日本]][[漫画家|漫画家]]、動畫家。

== 生平 ==
1938年1月，松本零士出生于[[福冈|福冈]]。1954年15岁时发表了漫画处女作《蜜蜂的冒险》，获得第一届“新人王”奖。高中毕业后，他立志要当个[[漫画家|漫画家]]，并到东京开始画[[少女漫画|少女漫画]]。1960年左右，又向[[少年漫画|少年漫画]]挺进，绘制了《四次元漫画系列》。松本零士还对[[天文学|天文学]]、[[考古学|考古学]]、[[武器|武器]]、[[軍事史|軍事史]]等方面有研究，对其创作有很大影响。

他於2005年担任[[日本宇宙少年团|日本宇宙少年团]]理事长。

2018年 99.9:刑事專業律師第二季第7集客串

== 作品 ==
松本零士代表作有《[[宇宙战舰大和号|宇宙战舰大和号]]》、《[[银河铁道999|银河铁道999]]》、《[[宇宙海贼哈洛克船长|宇宙海贼哈洛克船长]]》、《[[無限軌道SSX|無限軌道SSX]]》、《[[千年女王|千年女王]]》、《[[银河铁道物語|银河铁道物語]]》、《[[尼貝龍根的戒指|尼貝龍根的戒指]]》、《[[战地启示录|战地启示录]]》（{{lang|ja|[[w:ja:戦場まんがシリーズ|戦場まんがシリーズ]]}}）等。

== 家族 ==
松本零士妻子是漫畫家[[牧美也子|牧美也子]]（《[[惡女聖書|惡女聖書]]》作者）。
