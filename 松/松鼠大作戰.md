{{Vgname|松鼠大作戰|ja=チップとデールの大作戦|en=Chip 'n Dale Rescue Rangers|又译「'''松鼠大战'''」}}是一款[[平台遊戲|平台遊戲]]，由[[卡普空|卡普空]]根據[[迪士尼|迪士尼]]動畫《[[救難小福星|救難小福星]]》所改編。<ref>{{cite web|author= |url=http://www.gamefaqs.com/nes/587565-chip-n-dale-rescue-rangers/data |title=Chip 'n Dale: Rescue Rangers Release Information for NES |publisher=GameFAQs |date= |accessdate=2011-07-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20100720223756/http://www.gamefaqs.com/nes/587565-chip-n-dale-rescue-rangers/data |archivedate=2010-07-20 }}</ref>該遊戲最早只在[[FC遊戲機|FC遊戲機]]平台發售，但到1991年年底，卡普空也將這款遊戲移植到{{Link-en|PlayChoice-10|PlayChoice-10}}。該遊戲在全球共計售出120萬份。

== 劇情 ==
《松鼠大作戰》的主要任務是玩家操作角色去拯救一隻名為“曼迪”的小母貓。玩家可以分別選擇奇奇或蒂蒂其中角色來進入遊戲，玩家將要穿過各種機關和機器狗組成的小關卡，最後在最終關卡打敗肥貓。

== 續作 ==
1993年12月，卡普空為本款遊戲推出了續作《[[松鼠大作戰2|松鼠大作戰2]]》，這款遊戲依舊是[[FC遊戲機|FC遊戲機]]平台。<ref>{{cite web|author= |url=http://www.gamefaqs.com/nes/587566-chip-n-dale-rescue-rangers-2/data |title=Chip 'n Dale: Rescue Rangers 2 Release Information for NES |publisher=GameFAQs |date= |accessdate=2011-07-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20110515154436/http://www.gamefaqs.com/nes/587566-chip-n-dale-rescue-rangers-2/data |archivedate=2011-05-15 }}</ref>在《松鼠大作戰2》中，卡普空為雙人協作模式增加了許多額外關卡，例如在某些關卡里，遊戲人物必須利用自己的搭檔作為武器，拋擲同伴來獲得獎勵或攻擊敵人。

== 小知识 ==
游戏中的两只动物实际上并非松鼠（squirrel），而是花栗鼠（chipmunk）。
