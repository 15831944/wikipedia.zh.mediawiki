'''松平信康'''（{{bd|1559年|4月13日|1579年|10月5日}}）是[[日本戰國時代|日本戰國時代]]至[[安土桃山時代|安土桃山時代]]武將。父親是[[松平元康|松平元康]]（[[德川家康|德川家康]]）。母親是[[關口親永|關口氏廣]]的女兒[[築山殿|築山殿]]（[[今川義元|今川義元]]的外甥女）。妻子是[[織田信長|織田信長]]長女[[織田五德|德姬]]，生有[[登久姬|登久姬]]（[[小笠原秀政|小笠原秀政]]正室）與[[熊姬|熊姬]]（[[本多忠政|本多忠政]]室）。因為擔任[[松平氏|松平宗家]]的居城[[岡崎城|岡崎城]]的城主，因此與祖父[[松平廣忠|松平廣忠]]同被稱為'''岡崎三郎'''。別稱'''次郎三郎'''。

不少史書記載為'''德川信康'''，理由是進入[[江戶時代|江戶時代]]後，[[江戶幕府|江戶幕府]]許可將軍家和[[御三家|御三家]]都以「德川」為姓，以後表記再被統一，而信康則以德川家康的嫡長男身份而稱「德川信康」。

== 生平 ==
在[[永祿|永祿]]2年（[[1559年|1559年]]）3月6日於[[駿府城|駿府城]]出生，家中嫡長子。作為[[今川氏|今川氏]]的人質，幼年在駿府度過，不過在[[桶狹間之戰|桶狹間之戰]]後，德川軍以俘虜[[鵜殿氏長|鵜殿氏長]]和[[鵜殿氏次|鵜殿氏次]]交換，竹千代前往[[岡崎城|岡崎城]]居住。

永祿5年（[[1562年|1562年]]），家康和[[織田信長|織田信長]]的[[清洲同盟|清洲同盟]]成立。在永祿10年（[[1567年|1567年]]）5月與信長的女兒[[五德姬|五德姬]]結婚，兩人都是9歲並以夫婦的形式在[[岡崎城|岡崎城]]生活。同年6月家康把居城移往[[濱松城|濱松城]]，把岡崎城讓給竹千代。在7月元服時，從信長接受[[偏諱|偏諱]]「信」字並改名為'''信康'''。[[元龜|元龜]]元年（[[1570年|1570年]]）正式成為岡崎城的城主。

信康自小就勇猛果敢，初陣在[[天正_(日本)|天正]]元年（[[1573年|1573年]]）完成。

在天正3年（[[1575年|1575年]]）的[[長篠之戰|長篠之戰]]中以大將身份參戰，之後在與[[武田氏|武田氏]]的戰鬥中獲得軍功，在戰鬥中受到注目。特別在天正5年（[[1577年|1577年]]）8月的[[遠江横須賀之戰|遠江横須賀之戰]]中的退卻戰中擔任[[殿軍|殿軍]]，令武田軍不能越過[[大井川|大井川]]。率領[[岡崎眾|岡崎眾]]輔助家康。

天正7年（[[1579年|1579年]]）8月3日，家康到訪岡崎城，翌日信康離開岡崎城並被移至[[大濱城|大濱城]]。之後信康被移至[[遠江|遠江]]的[[堀江城|堀江城]]、[[二俁城|二俁城]]，在9月15日被家康命令[[切腹|切腹]]。享年21歲。

信康切腹的事件在《[[三河物語|三河物語]]》中有詳細的記述。根據此書記載，信長的女兒[[五德姬|五德姬]]和跟[[今川氏|今川氏]]有關係的[[築山殿|築山殿]]不和，與信康的關係都相當差，天正7年（1579年），五德向父親[[织田信长|信長]]寫了12項事情的信書，並拜託德川家重臣[[酒井忠次|酒井忠次]]作為使者送信給信長。信中包括了自己與信康不和及築山殿與[[武田勝賴|武田勝賴]]内通的事情。信長質問使者忠次，忠次完全沒有庇護信康並確認書信所寫的是事實。結果，信長要求家康讓信康切腹。

在德川家中反對把信康[[賜死|賜死]]的人有很多，甚至有家臣主張與信長[[斷交|斷交]]。信康的傅役[[平岩親吉|平岩親吉]]想負上責任，要求把自己的首級送給信長。但是家康判斷形勢後，認為這些小手段不能平息信長的怒火，於是決定處決信康。8月29日，首先是築山殿被護送前往[[二俣城|二俣城]]（守將是[[大久保忠世|大久保忠世]]）途中，在[[佐鳴湖|佐鳴湖]]湖畔被德川家家臣[[岡本時仲|岡本時仲]]和[[野中重政|野中重政]]殺死。之後的9月15日，事件發生以來一直被幽禁在二俣城的信康被命令切腹。[[介錯|介錯]]是[[服部正成|服部正成]]，但是正成悲痛萬分，無法把主人[[斬首|斬首]]，最後由負責檢死的[[天方道綱|天方道綱]][[介錯|介錯]]。享年21歲。

關於這次切腹事件的原由有諸多說法，包括父子不和、家中派系鬥爭、與家臣團對立等。

== 墓所、祭祀 ==
信康死後，家康建立的信康廟所[[清瀧寺_(濱松市)|清瀧寺]]，現今在該寺域中有信康的廟。在祭祀首塚的[[若宮八幡宮_(岡崎市)|若宮八幡宮]]中，信康是祭神，而與信康有深厚關係的人亦建立了數個寺院。

*[[西念寺_(新宿區)|西念寺]]（[[新宿區|新宿區]]）岡崎三郎信康供養塔－由[[服部正成|服部正成]]建立。
*[[隆岩寺_(古河市)|隆岩寺]]（[[古河市|古河市]]）－由女婿[[小笠原秀政|小笠原秀政]]建立
*[[萬松院_(小田原市)|萬松院]]（[[小田原市|小田原市]]）－由[[大久保忠隣|大久保忠隣]]建立
*在[[江淨寺|江淨寺]]（[[靜岡市|靜岡市]]）的供養塔－由[[榊原清政|榊原清政]]建立
*在高野山[[金剛峯寺|金剛峯寺]]（[[高野町|高野町]]）的岡崎三郎信康墓所－[[平岩親吉|平岩親吉]]建立

== 人物 ==
* 家康對信康死非常悲傷，[[關原之戰|關原之戰]]時三男秀忠遲到未能參戰時，家康說了「信康還在的話一定不會如此」（{{lang|ja|信康がいればこんな思いをしなくて済んだ}}）。
* [[酒井忠次|酒井忠次]]因為嫡男[[酒井家次|家次]]的所領太少，於是向家康提出不滿，反被家康諷刺「你也會疼愛自己的兒子啊？」（{{lang|ja|お前も我が子が可愛いか}}）。還有一次，與忠次父子一起觀賞[[幸若舞|幸若舞]]時，看到為了主公而獻出自己兒子首級的場面時落淚，家康說「你們看看啊」（{{lang|ja|両者あれを見ろ}}），當場令忠次父子非常恐懼。
* 對於被冷遇的異母弟[[結城秀康|結城秀康]]表示憐憫，與父親家康不同，顯示出重感情的一面。
* 在[[切腹|切腹]]後有[[殉死|殉死]]者（在[[濱松市|濱松市]]的[[清瀧寺|清瀧寺]]中有殉死家臣[[吉良於初|吉良於初]]（初之丞）的墳墓）出現，因此應該頗有人望。
* 在[[大久保忠教|大久保忠教]]的《[[三河物語|三河物語]]》中形容，信康說的總是戰爭話題，經常去騎馬和獵鷹，被描繪為典型的武者。
*《[[德川實紀|德川實紀]]》中如此評價信康「東照公（家康）的公子們中，'''岡崎三郎君'''（信康）、越前黄門（[[結城秀康|結城秀康]]）、薩摩中將（[[松平忠吉|松平忠吉]]）等人，都繼承了父親勇武的稟性，他們的軍事雄略在世間中都是十分傑出」（{{lang|ja|東照公の公達あまたおはしましける中に、岡崎三郎君はじめ、越前黄門、薩摩中将等は、おづれも父君の神武の御性を稟させられ。御武功雄略おおしく世にいちじるしかりし中に}}）。

在《三河物語》中記述「沒有這樣的殿下（信康）」（{{lang|ja|これほどの殿はなし}}）「上下萬民都發出聲音，沒有不悲傷的」（{{lang|ja|上下万民、声を引て、悲しまざるはなし}}）。信康是武勇優秀的人，在與武田軍對陣時擔任[[殿軍|殿軍]]並寸步不讓，家康亦對此驚嘆並說「真是勇將。即使勝賴引十萬士兵來對陣都無需驚慌」（{{lang|ja|まことの勇将なり。勝頼たとえ十万の兵をもって対陣すとも恐るるに足らず}}）（《[[大川三志|大川三志]]》）。而且還在劣勢時向家康進言與勝賴決戰（《松平物語》）。如此勇猛的信康亦有溫柔的一面，在知道異母弟弟秀康被家康討厭，在邀請家康前來岡崎城後，把秀康抱到家康的膝上，令秀康被家康承認為兒子（《[[貞享松平越前守書上|貞享松平越前守書上]]》）。

== 信康切腹之謎 ==
出自《[[三河物語|三河物語]]》有一說法為：信長見信康儀表不凡，唯恐自家地位不保，故意捏造信函逼迫家康除掉嫡長子。但此說漸不被採信，因為在另一本史書《[[松平記|松平記]]》中則是提及：「讓其（信康）自殺之事，在天正七年八月朔日向信長報告了。若信長也因此而震怒的話，則任由他的意思來決定了。」這就是完全相反的記載了，將[[織田信長|織田信長]]由主導者，改為被通知者的角色。

此外所謂家中權力爭奪說，則是《三河物語》強調了[[酒井忠次|酒井忠次]]在信康之死的責任問題，歸咎給[[酒井忠次|酒井忠次]]未有效替信康辯護或加罪於信康，但在流傳下來的《信光明寺文書》則收錄了家康寫與[[織田信長|織田信長]]側近[[堀秀政|堀秀政]]的書信中表明：「此次派左衛門督（忠次）（向信長）報告之事，得到信長的懇切回應，實不勝感謝，而有關三郎不自重之事，已在去（八月）四日把他趕出岡崎城了。」

可見在信康之死一事中，真正起作用的主導者還是[[德川家康|德川家康]]本人。至於何以德川家的父子相爭會演變至不死不休的局面，則有兩個不同說法，其一為武田內應說，此乃延續過去[[織田信長|織田信長]]主導論的理由，雖然當時德川家中確有零星與武田家內通情事，但就信康本人來說並無實據。

另外就是比較陰謀論的家康追放說，由於德川家長期支援織田家的戰事，對三河造成很大的經濟壓力。因此以[[石川數正|石川數正]]為首的三河派的重臣打算擁立信康繼任，才招致[[德川家康|德川家康]]反撲。此說亦無實證，只是從事後三河重臣大量變動進行推測。整體而言，信康之死的起因仍是未知數。
