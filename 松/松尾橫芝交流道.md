'''松尾橫芝交流道'''（{{lang|ja|まつおよこしばインターチェンジ}}）是[[日本|日本]][[千葉縣|千葉縣]][[山武市|山武市]]內屬於[[千葉東金道路|千葉東金道路]]、[[首都圈中央聯絡自動車道|首都圈中央聯絡自動車道]]（東金交流道松尾横芝交流道間與千葉東金道路共線）與[[銚子聯絡道路|銚子聯絡道路]]之[[交流道|交流道]]。

== 連接道路 ==
=== 直接道路 ===
* [[千葉縣道62號成田松尾線|千葉縣道62號成田松尾線]]

== 歷史 ==
*1998年3月30日 - [[千葉東金道路|千葉東金道路]][[東金IC|東金IC]]/JCT至松尾橫芝IC/JCT開通，此交流道也同時啟用。
*2006年3月25日 - [[銚子聯絡道路|銚子聯絡道路]]松尾橫芝IC至[[橫芝光IC|橫芝光IC]]開通。

== 周邊景點 ==
