'''松原站'''（{{jpn|j=松原駅|hg=まつばらえき|rm=Matsubara eki|f=y}}）是一個位於[[東京都|東京都]][[世田谷區|世田谷區]][[松原_(世田谷區)|松原]]四丁目，屬於[[東京急行電鐵|東京急行電鐵]][[世田谷線|世田谷線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''SG09'''。

此站位於世田谷區[[赤堤|赤堤]]二、三、四丁目、松原四丁目的邊界。

== 歷史 ==
* 1949年（[[昭和|昭和]]24年）9月1日 - 近山下的六所神社前站搬遷，設立玉電松原站。但是實際上與近下高井戶的七軒町站統合。
* 1969年（昭和44年）5月11日 - 改稱松原站。

== 車站構造 ==
[[對向式月台|對向式月台]]2面2線的[[地面車站|地面車站]]。平日早上[[尖峰時段|尖峰時段]]的下行月台設有職員，負責上下整理（不接受車費）。

== 利用狀況 ==
