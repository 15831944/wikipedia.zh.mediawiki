'''松山區'''是[[中華民國|中華民國]][[臺北市|臺北市]]轄下的[[市轄區|行政區]]，位於[[臺灣|臺灣]]北部，臺北市中心偏東地帶。區內有許多大型[[百貨公司|百貨公司]]及[[購物中心|購物中心]]、夜店，[[台北小巨蛋|臺北小巨蛋]]及[[臺北松山機場|松山機場]]也位於此區。區內的[[饒河街夜市|饒河街夜市]]則為臺灣著名的[[觀光|觀光]][[夜市|夜市]]之一。而區內的[[敦化北路|敦化北路]]及[[南京東路_(臺北市)|南京東路]]一帶是臺北重要的金融商圈，不少國內外知名的企業及[[金融機構|金融機構]]均在此雲集，而有「臺北[[華爾街|華爾街]]」之稱。此區也有臺北市最早的[[都市計劃|都市計劃]]開發區[[民生社區_(臺北市)|民生社區]]，由於該區域內[[建蔽率|建蔽率]]低，[[容積率|容積率]]嚴格限制，而且處處[[公園|公園]]，文教設施完善，而成為臺北市區傳統的優質[[住宅區|住宅區]]之一，2010年臺北市舉辦[[臺北國際花卉博覽會|臺北國際花卉博覽會]]而制定區花，松山區為[[朱槿|朱槿]]，[[臺北市政府民政局|臺北市政府民政局]]也特別製作區花版門牌。

== 歷史 ==
*昔日為[[凱達格蘭族|凱達格蘭族]]-{猫里}-[[錫口|錫口]]社（[[羅馬拼音|羅馬拼音]]：Malotsigauan）之地，亦音譯作'''麻里折口'''、'''麻里即吼'''、'''毛里即錫口'''。
*1815年（清嘉慶20年）時刪去-{猫里}-，改稱錫口。
*1920年（大正9年），全臺地名重整，錫口改為日式地名「松山」，設「-{[[松山庄|松山庄]]}-」，隸屬[[臺北州|臺北州]][[七星郡|七星郡]]。
*1938年（昭和13年），-{[[松山庄|松山庄]]}-併入臺北市。
*1945年（民國34年；昭和20年）10月25日，中華民國接管臺灣，松山-{庄}-改置松山區。
*1990年（民國79年）3月23日，臺北市行政區重劃，劃出南半部設立[[信義區_(臺北市)|信義區]]。

*由來和英譯
「松山」的由來一說是因為當地風景和[[四國|四國]]的[[松山市|松山]]相似，另一說是因為附近山丘多松樹。松山原為日式地名，其發音為[[日語|日語]]之「Matsuyama」，在[[台灣日治時期|日治時期]]對外音譯均採用之，如松山飛行場稱作「Matsuyama Airdrome」。戰後雖沿用該名，但對外音譯則改為[[華語|華語]]發音，而成為「Sungshan」（[[威妥瑪拼音|威妥瑪拼音]]），2003年後改為 「Songshan」（[[漢語拼音|漢語拼音]]）。但一般來說，松科植物在北緯40-60度間，或高海拔的地方較為常見。松山位於臺北盆地，既不是山，也不可能有松樹。但臺灣真的有一座"松山"，位於阿里山山脈的中間，海拔2500公尺。因此也有可能只是當時負責改名的人來自日本某個縣的松山也不一定。（大灣大員福爾摩沙 翁佳音 曹銘宗）

== 地理位置 ==
'''東界'''：以三張犁截水圳、[[玉成橋|玉成橋]]、[[松山橋|松山橋]]鄰接[[南港區|南港區]]。<br>
'''西界'''：以復興北路、南路及[[松山機場|松山機場]]西側圍牆鄰接[[中山區_(臺北市)|中山區]]。<br>
'''南界'''：市民大道四段（復興南路至光復南路），鄰接[[大安區_(臺北市)|大安區]]。市民大道五段至[[松山車站|松山車站]]，鄰接[[信義區_(臺北市)|信義區]]。<br>
'''北界'''：以基隆河三張犁截水圳出口至港墘路口堤外河道，鄰接[[內湖區|內湖區]]。港墘路口堤外基隆河道、七號水門至機場北邊圍牆鄰接[[中山區_(臺北市)|中山區]]。
