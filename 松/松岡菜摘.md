'''松岡菜摘'''（{{lang-ja|松岡 菜摘}}，{{bd|1996年|8月8日||}}）是[[日本|日本]][[日本偶像|偶像藝人]]，為女子偶像團體[[HKT48|HKT48]] Team H成員、並擔任隊長（第2任）。[[福岡縣|福岡縣]]出身，所屬[[經紀公司|經紀公司]]為[[AKS|AKS]]。

== 經歷 ==
; 2011年
* 7月10日，通過HKT48的一期生選秀<ref group="註">共有23名通過徵選，其中3名在公佈前自行辭退。</ref>。
* 10月23日，於[[西武巨蛋|西武巨蛋]]舉辦的《[[飛翔入手|飛翔入手]]》發售紀念全國[[握手會|握手會]]活動「AKB祭 powered by AKB48 [[神TV|神TV]]」中首度公開在大眾面前亮相<ref>{{cite web|url=http://www.oricon.co.jp/news/music/2002974/full/ | title=<span lang="ja">HKT48、西武ドームで初代21人お披露目　AKBグループ最年少の平均年齢13.8歳</span>|publisher=[[oricon|oricon]] |language=ja |date=2011-10-23|accessdate=2014-02-05}}</ref>。
* 11月26日，於HKT48劇場{{link-ja|柿落|こけら落し}}<ref group="註">「柿落」（{{lang|ja|こけら落し}}）是日本的表演藝術用語，意指一個[[劇場_(建築)|劇場]]落成之後的首次公開演出。</ref>公演中實現劇場出道。

; 2012年
* 2月5日，與HKT48的隊員[[村重杏奈|村重杏奈]]與[[宮脇咲良|宮脇咲良]]一起加入於2010年6月5日在幕張展覽館的活動中成立的「波羅麵包同盟<ref group="註">原文：{{lang|ja|メロンパン同盟。}}</ref>，」此團體由[[松井玲奈|松井玲奈]]擔任會長，[[島崎遙香|島崎遙香]]擔任副會長，成員另有AKB48的[[市川美織|市川美織]]、NMB48的[[福本愛菜|福本愛菜]]、SKE48的[[磯原杏華|磯原杏華]]<ref>{{cite web|url=http://ameblo.jp/ske48official/entry-11156567311.html |title=れな(増え・ω・たよ) |publisher=SKE48官方Blog |language=ja |date=2012-02-05 |accessdate=2014-03-11}}</ref>。
* 3月4日，Team H結成，從研究生升格為正式成員，為16位創團成員之一。
* 4月22日，参加AKB48「[[「為了誰」計劃|『為了誰』計劃]]」的活動，與AKB48的6名成員<ref group="註">6名AKB48的成員分別是[[片山陽加|片山陽加]]、[[篠田麻里子|篠田麻里子]]、[[前田敦子|前田敦子]]、[[前田亜美|前田亜美]]、[[加藤玲奈|加藤玲奈]]與[[高橋朱里|高橋朱里]]。</ref>共同訪問因[[東日本大震災|311地震]]而受創的[[岩手縣|岩手縣]][[釜石市|釜石市]]。

; 2014年
* 1月11日，於大分縣{{link-ja|iichiko剧场|大分県立総合文化センター#グランシアタ}}“[[HKT48九州7县巡演～和可爱的孩子们一起旅行～|HKT48九州7县巡演～和可爱的孩子们一起旅行～]]”的首日夜公演中，發表了被稱為「轉班<ref group="註">原文：{{lang|ja|クラス替え。}}</ref>」的隊伍重组消息，宣布留任Team H<ref>{{cite web|url=http://natalie.mu/music/news/107576 |script-title=ja:HKT48ツアー初日に“新クラス”と研究生17名昇格を発表 |publisher=[[Natalie_(网站)|Natalie]] |language=ja |date=2014-01-12 |accessdate=2014-02-05}}</ref>。
* 2月24日，於{{link-ja|Zepp DiverCity Tokyo|Zepp}}舉辦的「[[AKB48集團大組閣祭|AKB48集團大組閣祭]]」中，宣布所屬隊伍除維持轉班時發表的留任Team H，並成為Team H副隊長<ref>{{cite web|url=http://www.akb48.co.jp/page/daisokaku/list.php?after=1&group=akb48 |script-title=ja:AKB48グループ大組閣祭り |publisher=AKB48官方網站 |language=ja |date=2014-02-24 |accessdate=2014-02-26}}</ref>。
* 6月7日，於「[[AKB48第37張單曲選拔總選舉|AKB48第37張單曲選拔總選舉]]」中獲得12,569票，得到第64名，首次進入圈內並成為Future Girls的一員<ref>{{cite web|url=http://ameblo.jp/akihabara48/entry-11873269522.html |title=「AKB48 37thシングル 選抜総選挙」開票結果 |publisher=AKB48官方Blog |language=ja |date=2014-06-07|accessdate=2014-06-25}}</ref>。

; 2015年
* 6月6日，於「[[AKB48第41張單曲選拔總選舉|AKB48第41張單曲選拔總選舉]]」中以17,387票獲得第51名，再度成為Future Girls。
* 9月24日，首部個人寫真集《追伸》發行。

; 2016年
* 6月18日，於「[[AKB48第45張單曲選拔總選舉|AKB48第45張單曲選拔總選舉]]」中以20,980票獲得第46名，成為Next Girls。
*7月11日，於「ＨＫＴ４８夏のホールツアー２０１６～ＨＫＴがＡＫＢ４８グループを離脫？國民投票コンサート～」中，被將畢業的Team H隊長穴井千尋指名為新一任Team H隊長。
*8月1日，正式成為新一任Team H隊長。

; 2017年
* 6月17日，於「[[AKB48第49張單曲選拔總選舉|AKB48第49張單曲選拔總選舉]]」中以19,372票獲得第59名，成為Future Girls。

; 2018年
* 6月16日，於「[[AKB48第53張單曲選拔總選舉|AKB48第53張單曲選拔總選舉]]」中以22,050票獲得第55名，成為Future Girls。

== 人物 ==
* 在HKT劇場中使用的Catch phrase是「內臟～『想吃』（由觀眾與成員一起喊）。最喜歡吃內臟的暱稱『Natsu』的松岡菜摘<ref group="註">原文：{{lang|ja|ホルモンがー、『食べたーい!!』。ホルモンが大好きなの”なつ”こと松岡菜摘です。}}</ref>。」

* 如同Catch phrase所說，喜歡吃內臟類的食物，例如九州的牛內臟鍋，但討厭吃豆類與海鮮類的食物<ref name="2012-01-12部屋之子 ">{{cite web|url=https://plus.google.com/111799140715858863784/posts/ML7RHq3eWoA |title=本人記事 |publisher=HKT48部屋之子 in Google+ |language=ja |date=2012-01-12 |accessdate=2014-07-01}}</ref>。

* 迷人之處則是眼睫毛<ref>HKT48官方網站的個人介紹頁面</ref>。

* 家中成員除父母外，還有一位比松岡年長四歲的姐姐<ref>{{cite web|url=https://plus.google.com/107241677527739013868/posts/Q4tmDG4syPf |title=本人記事 |publisher=Google+ |language=ja |date=2012-08-04|accessdate=2014-07-01}}</ref><ref>{{cite web|url=https://plus.google.com/107241677527739013868/posts/fTJnn9w4HZj |title=本人記事 |publisher=Google+ |language=ja |date=2012-07-22 |accessdate=2014-07-01}}</ref>。

* 家中並未飼養寵物，此外松岡也不喜歡動物<ref name="2012-01-12部屋之子 "/><ref>{{cite web|url=https://plus.google.com/107241677527739013868/posts/NLgr9tDtdJS |title=本人記事 |publisher=Google+ |language=ja |date=2012-06-08 |accessdate=2014-07-01}}</ref>。

* 與前SKE48的磯原杏華同年同月同日生（1996年8月8日）。
