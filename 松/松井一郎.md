'''松井一郎'''（{{bd|1964年|1月21日|||Matsui Ichirō}}），[[日本|日本]][[政治家|政治家]]，[[大阪府|大阪府]]出身。現任[[大阪府知事|大阪府知事]]（民選第18代，2011年至今）。國政政黨[[日本維新會|日本維新會]]幹事長，地域政黨[[大阪維新會|大阪維新會]]幹事長。2003年至2011年連任三屆大阪府議員。父親是曾任大阪府議會議長的[[松井良夫|松井良夫]]。

== 經歷 ==
童年在[[大阪府|大阪府]][[八尾市|八尾市]]度過。高中就讀於[[福岡工業大學附属城東高等學校|福岡工業大學附属高等學校]]，1986年3月大學畢業於[[福岡工業大學|福岡工業大學]]工学部電氣工學科。之後在[[關西電力|關西電力]]集團的[[Kinden|Kinden]]工作。2003年4月13日參加[[第15回統一地方選舉|第15回統一地方選舉]]的大阪府議會議員選挙，出戰八尾市選舉區，獲得[[自由民主黨_(日本)|自由民主黨]]的提名，順利當選，之後連續當選三屆。

府議會議員期間，松井擔任自民黨大阪府議團的政策調查會長，因為WTC府廳轉移問題和黨內主流意見不同而成立「自民黨維新會」。

2010年加入[[橋下徹|橋下徹]]擔任代表的地域政黨[[大阪維新會|大阪維新會]]，就任幹事長。2012年大阪維新會成立國政政黨[[日本維新會|日本維新會]]，松井又擔任幹事長。日本維新會於2014年5月，因[[石原慎太郎|石原慎太郎]]理念不同而退黨，與[[日本连结党|日本连结党]]合併為[[維新黨|維新黨]]。

2015年8月27日，因維新黨幹事長[[柿澤未途|柿澤未途]]公開支持民主黨、共產黨、社民黨及生活黨聯合推薦的無黨籍[[山形市|山形市]]市長候選人[[梅津庸成|梅津庸成]]引起爭議，松井一郎與大阪市長橋下徹宣布退出維新黨，專心於大阪的地方政治<ref>{{cite news|url=http://www.cna.com.tw/news/aopl/201508270389-1.aspx|title=日維新黨內訌 橋下徹退黨|publisher=中央通訊社|date=2015-08-27|accessdate=2015-08-29}}</ref>。

== 大阪府知事 ==
2011年10月22日，時任[[大阪府知事列表|大阪府知事]]的橋下徹因為和大阪市長[[平松邦夫|平松邦夫]]就[[大阪都構想|大阪都構想]]等問題意見不同而提出辭職，決定轉戰大阪市挑戰平松市長。大阪都知事選舉和大阪市長選舉在同年11月27日舉行，被稱為「大阪雙選」。<ref>{{Cite news|url=http://www.asahi.com/national/update/1022/OSK201110210203.html |script-title=ja:橋下知事が辞職願 大阪市長選に出馬表明、ダブル選へ |date=2011-10-22 |newspaper=朝日新聞 |archiveurl=https://archive.is/20120718172950/http://www.asahi.com/national/update/1022/OSK201110210203.html |archivedate=2012-07-18 |deadurl=yes }}</ref>。10月23日，松井和橋下舉行共同會見，決定橋下參加大阪市長選舉，松井走上前台參加大阪府知事。<ref>{{Cite news|url=http://www.yomiuri.co.jp/election/local/news/20111023-OYT1T00616.htm |script-title=ja:行政と渡り合えるのは松井氏だけ…橋下氏が擁立 |date=2011-10-23 |newspaper=讀賣新聞 |archiveurl=https://web.archive.org/web/20111024064643/http://www.yomiuri.co.jp/election/local/news/20111023-OYT1T00616.htm |archivedate=2011-10-24 |deadurl=yes }}</ref>11月27日舉行的[[2011年大阪府知事選舉|大阪府知事選舉]]，以大比例優勢擊敗了[[民主黨_(日本)|民主黨]]和自民黨支援的前[[池田市|池田市]]長[[倉田薰|倉田薰]]，當選第18代大阪府知事。橋下徹也成功擊敗平松當選大阪市長。大阪維新會在雙選中大獲全勝<ref>{{Cite news|url=http://www.jiji.com/jc/c?g=pol_30&k=2011112800323|script-title=ja:橋下、松井両氏に当選証書＝本人欠席、代理人受け取る－大阪ダブル選|date=2011-11-28|accessdate=2011-11-28|newspaper=時事通信}}</ref>。

2019年3月24日，因宣佈參選[[大阪市長|大阪市長]]而辭去府知事職務<ref>{{Cite web|title=大阪知事、市長辞職 入れ替えダブル選出馬を正式表明|url=https://www.sankei.com/west/news/190308/wst1903080030-n1.html|work=産経ニュース|date=2019-03-08|accessdate=2019-03-26|language=ja|first=SANKEI DIGITAL|last=INC}}</ref>。4月7日，在大阪市長選舉中，松井一郎成功當選第21任民選大阪市長。

== 批評 ==
父親[[松井良夫|松井良夫]]是日本戰後[[右翼|右翼]]政治活動家，與[[指定暴力團|暴力團]]關係緊密的[[笹川良一|笹川良一]]的舍弟。松井一郎也被認為是大阪維新會的實際操控者，是大阪府重建計劃的第一策劃人。他擔任大阪維新會的幹事長，經常不顧反對意見在府議會對法案實行[[強行表決|強行表決]]，被批判是強權政治。
