'''松岭区'''是[[中华人民共和国|中华人民共和国]][[黑龙江省|黑龙江省]][[大兴安岭地区|大兴安岭地区]]下辖的一个县级区，位于[[内蒙古自治区|内蒙古自治区]][[呼伦贝尔市|呼伦贝尔市]][[鄂伦春自治旗|鄂伦春自治旗]]境内，隶属于[[国家林业局|国家林业局]]大兴安岭林业集团公司的松岭林业局，属于政企合一体制。松岭林业局设立于1965年，1970年设立松岭区建制。

== 地理 ==
松岭区位于[[大兴安岭|大兴安岭]][[伊勒呼里山|伊勒呼里山]]东南部余脉，境内水系为[[嫩江|嫩江]]水系，气候属于大陆性寒[[温带季风气候|温带季风气候]]。

== 管辖范围 ==
=== 行政区划 ===
下辖3个镇：[[小扬气镇|小扬气镇]]、[[古源镇|古源镇]]、[[劲松镇|劲松镇]]。

由于政府驻地在小扬气镇，所以松岭区主要铁路车站为小扬气站。

=== 林业区划 ===
松岭林业局（驻小扬气镇）辖5个林场：即[[绿水林场|绿水林场]]、[[古源林场|古源林场]]、[[大扬气林场|大扬气林场]]、[[壮志林场|壮志林场]]、[[新天林场|新天林场]]。

=== 行政区的特殊地位 ===
松岭区地理位置上在[[内蒙古自治区|内蒙古自治区]][[鄂伦春自治旗|鄂伦春自治旗]]境内，但实际在行政上隶属于[[黑龙江省|黑龙江省]]。

== 人口 ==
区内有[[汉族|汉族]]、[[满族|满族]]、[[蒙古族|蒙古族]]、[[达斡尔族|达斡尔族]]、[[回族|回族]]、[[朝鲜族|朝鲜族]]、[[鄂温克族|鄂温克族]]、[[锡伯族|锡伯族]]、[[羌族|羌族]]、[[俄罗斯族|俄罗斯族]]等11个民族，总人口为3.52万人。

== 经济 ==
区内之前主要以林木材采伐业为主，其后实施的[[国家天然林资源保护工程|国家天然林资源保护工程]]使其从主要的森林采伐工作转变为“封山育林”的林木种植保护工作和林业附产业为主的经济主体。其辖属的松岭林业局是全国大型木材生产基地，下辖5个主伐林场，以及苗圃、胶合板厂、卫生筷子厂、雪条棒厂、牙签厂、地板块厂、锯材加工厂、筑路工程处、机制木炭厂、建筑工程处、酒厂、煤炭公司、黄金公司等生产经营单位。

== 历史 ==
1956年春季，建立绿水林场。

1964年，开始筹建大扬气林场。

1965年，松岭林业局成立。

1965年2月，开发建设大兴安岭第一个大会战所在地--古源林场。

1970年，设松岭林业局为松岭区建制，形成政企合一体制。

1974年，设立松岭镇。

1981年，地名普查时，把松岭镇按照车站名称改为小扬气镇。
