'''松平康重'''（1568年－1640年8月14日）是[[安土桃山時代|安土桃山時代]]至[[江戶時代|江戶時代]]前期武將、[[大名|大名]]。[[松井松平家|松井松平家]]第2代。父親是[[松平康親|松平康親]]。

== 生平 ==
在[[永祿|永祿]]11年（1568年）出生，家中長男。父親康親是[[駿河國|駿河國]][[三枚橋城|三枚橋城]]城主。在[[天正_(日本)|天正]]11年（1583年）3月16日[[元服|元服]]之際被家康授予[[偏諱|偏諱]]「康」字而改名為'''康次'''，後來改為'''康重'''。

在家康於[[東海地方|東海地方]]的天正11年（1583年）至天正18年（1590年）繼承父親康親的領地並擔任駿河國[[沼津城|沼津城]]的守備役，對[[後北條氏|後北條氏]]對峙約8年時間。

在[[小田原征伐|小田原征伐]]後的天正18年8月，家康被移封至[[關東地方|關東地方]]，此時被賜予[[武藏|武藏]]騎西2萬石。在[[文祿|文祿]]4年（1595年）被賜予[[豐臣氏|豐臣姓]]。

在[[慶長|慶長]]6年（1601年）被加增[[常陸|常陸]]笠間3萬石。在慶長13年（1608年）被加增移封至[[丹波國|丹波國]][[篠山藩|篠山藩]]5萬石。（『[[寬政重修諸家譜|寬政重修諸家譜]]』）

在[[元和|元和]]5年（1619年）被移封至[[大坂|大坂]]平野南方的要衝[[和泉|和泉]]岸和田（『寬政重修諸家譜』）。在[[寬永|寬永]]11年（1634年）敘任[[從四位下|從四位下]]。在寬永17年（1640年）6月27日死去。享年73歲。[[家督|家督]]由次男[[松平康映|康映]]繼承（『寬政重修諸家譜』）。

== 私生子說 ==
康重被當成[[松平康親|松平康親]]的兒子，但是亦有說法指其實是[[德川家康|德川家康]]的私生子。生母是家康的侍女，懷著家康的兒子時嫁給康親。後來子孫亦使用家康的「康」為[[通字|通字]]。（[[川口素生|川口素生]]著『{{lang|ja|戦国武将、逸話の真実と謎}}』，[[学習研究社|学習研究社]]，2007年）
