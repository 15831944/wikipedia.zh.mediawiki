'''瀆山大玉海'''，是中国[[北京市|北京]]的一塊著名巨型[[石器|石器]]。[[玉海|玉海]]一般用作盛酒的大型器皿。它美其名為粗獷，但其材質獨山玉事實上是一種無法拋光的劣質長石石材。相對傳統玉器而言，其作工極為粗糙。瀆山大玉海原為[[北宋|北宋]][[宋徽宗|宋徽宗]]的[[花石綱|花石綱]]，為[[汴梁|汴梁]][[艮岳|艮岳]]採辦無數石材中其中一塊。及後在金人佔領汴京城之後被移至[[北海公園_(北京)|北海]]。瀆山大玉海現時的外貌黃黑，又有裂縫。之所以會失去玉的光彩可歸咎於保存不當所致。{{noteTag|見王廷洽《中華國寶之謎》（2005年由黄山書社出版）第39頁。}}

== 歷史 ==
渎山大玉海是[[蒙古帝国|蒙古帝国]]君主[[忽必烈|忽必烈]]的玉器。[[至元_(忽必烈)|至元]]二年（1265年），瀆山大玉海的雕刻工序完畢，並被置於[[廣寒宮|廣寒宮]]{{noteTag|見明朝宋濂所編的《元史》卷六〈本紀第六‧世祖三〉的原文記載：「己丑，瀆山大玉海成，敕置廣寒殿。」。}}，即現今[[北海公園|北海公園]]白塔處。。在[[明朝|明代]]至[[清朝|清代]]初年，玉海被移至[[西華門|西華門]]外的[[真武庙_(北京玉钵胡同)|真武廟]]。廟內的道人曾用它作為菜甕醃菜。<ref>{{web reference|URL=http://bjwb.bjd.com.cn/html/2017-08/14/content_165707.htm|title=團城的兩件國寶從何而来|work=北京晚報|year=2017年|date=8月14日}}</ref>至[[乾隆|乾隆]]十年（1745年），瀆山大玉海被重新發現，[[乾隆帝|乾隆帝]]以千金易得，並置它於承光殿（[[北海公園_(北京)|北海]][[團城|團城]]）<ref name="中國玉器藝術 瀆山大玉海">[https://www.chiculture.net/0502/html/d14/0502d14.html 中國文化研究院 燦爛的中國文明〈中國玉器藝術 瀆山大玉海〉]</ref>。乾隆十四年（1749年），乾隆帝建造玉甕亭，將瀆山大玉海陳設在亭之內，再配以漢白玉雕花石座作襯托。他又命四十名翰林學士各賦詩一首，刻於亭柱之上<ref name="瀆山大玉海簡介大全">[http://trip.elong.com/dushandayuhai/jianjie/ 藝龍旅遊指南〈瀆山大玉海簡介大全〉]</ref>。玉海自雕刻工序完成以來經歷過合共四次刮苔修整，才能恢復本來的面貌。1988年，在[[法源寺_(北京)|北京法源寺]]内发现原配底座<ref>{{cite web |url = http://epaper.01ny.cn/nywb/html/2011-09/14/content_902617.htm |title = 失而复得四次修复 |publisher=南阳晚报—南阳新闻网|author=记者：王金榜|date=2011年9月14日 |accessdate=2012-10-02 |language = zh-Hans}}</ref>。

== 玉器資料 ==
瀆山大玉海高70厘米{{noteTag|資料出自徐湘人、晏京山《中國文物與考古: 靜態的歷史見證》（1995年由湖北人民出版社出版）第87頁。}}，內部空無一物。而口徑則為135至182厘米，其最大週長達493厘米，膛深55厘米，重達3500公斤。{{noteTag|資料出自田書和《北京市區主要街道景觀導覽》（2009年由旅遊教育出版社出版）第120頁「第九章 朝阜大街 -- 朝陽門至阜城門（沿線主要公交線路：846路）」。}}工匠隨形設計和加工玉海，一方面量材取料。另一方面，也因材施藝。玉海的玉身碾琢了隱起的波濤、海龍、海馬、海羊、海豚、海犀、海蛙、海螺、海魚、海鹿等共十三種海中瑞獸。當中的雕刻栩栩如生，且氣勢宏偉。而造型就顯得敦重古樸，粗獷豪放。瀆山大玉海可以貯三十石酒，在[[元朝|元朝]]時是最大的帝王玉。直至現在，玉海仍然是一件膾炙人口的國寶。<ref name="中國玉器藝術 瀆山大玉海" />其雕刻工藝體現了宋元兩代的高水平琢玉技術。{{noteTag|見周南泉《中國玉器》（2008年由中央編譯出版社出版）第253及303頁。}}

== 玉料来源 ==
乾隆帝认为玉料就來自瀆山（今[[重庆市|重庆市]][[巫山|巫山]]）的[[蜀玉|蜀玉]]。另一种说法是[[新疆维吾尔自治区|新疆]][[和田玉|和田玉]]或[[墨玉|墨玉]]。近年来有人提出，玉料为[[河南省|河南]][[独山玉|独山玉]]。<ref>{{cite web |url = http://epaper.01ny.cn/nywb/html/2011-09/14/content_902620.htm |title = 质地是独山玉 |publisher=南阳晚报—南阳新闻网 |author=记者：王金榜 |date=2011年9月14日 |accessdate=2012-10-02 |language = zh-Hans}}</ref>
