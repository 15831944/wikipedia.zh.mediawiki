'''定海军政府'''，是[[第一次鸦片战争|第一次鸦片战争]]中1841年至1846年[[英国|英国]]占领[[清朝|清朝]][[浙江省|浙江省]][[定海县|定海县]]期间设立的临时政权。

==简介==
[[第一次鸦片战争|第一次鸦片战争]]中，早在1840年7月5日至6日[[第一次定海之战|第一次定海之战]]后，[[英军|英军]]第一次占领定海县，直到7个月后的1841年2月24日英军才全部撤出定海县。在此期间，英军在[[舟山群岛|舟山群岛]]实行占领区统治，并成立了殖民当局——“'''巡理府'''”，宣布舟山群岛全部民政、财政、司法管理均由英方执行，岛上居民需向英国殖民者缴税，接受英国法律管辖，英国官员有权“驱逐任何人员”。而且强行将定海划为自由贸易港，“凡各国船只，俱得出入买卖”。英国殖民当局在岛上擅自抽税，先后设立复书院、苦老院、育婴堂、养济院等宗教机构<ref>{{Cite web|url = http://study.ccln.gov.cn/fenke/lishixue/lsjpwz/lszgs/53743.shtml|title = 鸦片战争时期英国对华侵占地的目标转换|author = 郭卫东|date = |publisher = 民国档案1997(2):86-95|language = |accessdate = |quote = }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1841年9月26日[[第二次定海之战|第二次定海之战]]后，英军第二次占领定海县，这次占领为期5年，直到1846年4月29日中英签订《[[英军退还舟山条约|英军退还舟山条约]]》后，1846年7月25日英军才退出定海县。在占领期间，1841年10月6日[[璞鼎查|璞鼎查]]在定海设“'''军政府'''”，由英军上尉丹尼士（Capt Dennis）担任“军政长官”<ref>{{Cite book|url =http://www.cnbsz.org.cn/nbsz1995/szbook2010/Default.aspx?CategoryId=912  |title =宁波市志（下）|author =  宁波市地方志编纂委员会编|date =1995年 |page=2028 |publisher =  中华书局|language =  |accessdate =  |quote =第三十五卷 军事 第一章 战事 第二节 抗击外侮  }}</ref>。

在第二次占领定海县期间，英国殖民者注重研究当地语言文化。1846年5月，应军方要求，英军中担任首席会计（Head conicopoly）的仙尼华四（Streenevassa，全名P. Streenevassa Pillay，[[印度|印度]][[马德拉斯|马德拉斯]]（今[[泰米尔纳德邦|泰米尔纳德邦]][[金奈|金奈市]]）高[[种姓|种姓]]的学者）编写出版了一本宁波方言工具书《英华仙尼华四杂字文》，封面印有“大清[[道光|道光]]二十六年仲夏，大英一千八百四十六年五月”，“定海舟山南门外太保庙内刊，宁郡东壁斋刻”字样，这是[[吴语|吴语]]地区出现的第一本方言英汉词汇集，也是中国首部[[中文|中文]]、[[英文|英文]]、[[泰卢固文|泰卢固文]]三种文字对照词汇集<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1707893  |title =《英华仙尼华四杂字文》，英军占领舟山的书证  |author = 徐春伟 |date =2017-06-21  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。
