[[File:O&M_instructor.jpg|thumb]]'''定向行動訓練師'''（{{lang-en|Orientation and Mobility Specialist / Orientation and Mobility Instructor}}，縮寫:O&M Specialist / Instructor），係指接受過專業的[[定向行動|定向行動]]師資培訓結訓或檢定認證通過者。<br />
參考美國定向行動訓練師認證方式，於1990年代各州均已有訂定培訓及認證規定<ref>{{cite web |title=Certificate in Orientation and Mobility |url=http://ecps.educ.ubc.ca/special-education/certificate-in-orientation-and-mobility/}}</ref>，足見定向行動訓練師需要一定時間的培訓方能執行教學工作。<br />
[[台灣|台灣]]的定向行動訓練師資培訓，是由[[台師大|台師大]][http://www.spe.ntnu.edu.tw/people/bio.php?PID=20 張千惠]和[[全國教師工會總聯合會|全國教師工會]]謝曼莉等人結合國內視障機構倡議，[[中華民國內政部|中華民國內政部]]訂定[[定向行動|定向行動]]訓練師之受訓內容及規定受訓時數含實習為650小時以上，作為受訓單位之參考依據。爾後[[行政院|行政院]][[勞工委員會|勞工委員會]]也頒定技術士技能檢定定向行動訓練職類單一級規範，作為台灣定向行動訓練師之認證管道。<ref>{{cite web |title=技術士技能檢定定向行動訓練職類單一級規範 |url=http://www.rootlaw.com.tw/LawContent.aspx?LawID=A040290091046900-0990226}}</ref>

== 定向行動訓練師的角色與功能 ==
'''定向行動訓練師'''，或稱'''定向行動訓練專業人員'''，其功能在協助視覺障礙者透過[[定向行動|定向行動]]訓練，能習得或重建其移動能力，增進視障者[[獨立|獨立]][[自主|自主]]能力，促使其融入[[社會|社會]]生活。<br />
定向行動訓練師服務的對象以視覺障礙者為主，或視覺多重障礙者也可能為接受服務的對象。視覺障礙者又稱[[視障人士|視障人士]]，依視覺狀況區分為全盲者（Blind）及低視能者（Low-Vision），故[[定向行動|定向行動]]訓練並非只有全盲者才需要，低視能者也需要。<br />
定向行動訓練師指導的內容包括:
*指導視障者運用感官知覺蒐集、分析、判斷、選擇、修正環境資訊，感官知覺有可能包括運用剩餘視覺。
*指導視障者或其家人、朋友運用[[人導法|人導法]]，使引導者能以安全有效方式提供協助。
*指導視障者使用[[徒手法|徒手法]]（或稱上下身保護法），能於室內熟悉環境以徒手方式自我保護，或搭配手杖技巧於空間中安全移行。
*指導視障者使用[[白手杖|白手杖]]，運用合宜技巧探測路況、閃避障礙物、處理高低落差、區辨路面材質...等等，白手杖也作為視覺障礙者標示其有視覺方面的特殊需求之功能。
*協助視障者建立[[心理地圖|心理地圖]]。
*指導視障者[[道路|道路]][[交通|交通]]系統，包括巷口/路口穿越。
*指導視障者搭乘[[大眾運輸|大眾運輸]]工具。
*指導視障者與他人[[互動|互動]]之技巧，包括尋求協助之技巧及策略。
*指導視障者運用與行動相關之[[輔具|輔具]]，以使行動更為安全有效率。<ref>{{cite web |title=Orientation and Mobility Instructors: Resources from the American Foundation for the Blind |url=http://www.afb.org/info/programs-and-services/professional-development/orientation-and-mobility/123}}</ref><ref>{{cite web |author1=張千惠 |title=定向行動老師之角色與應負職責>  定向行動老師之角色與應負職責 |url=https://sites.google.com/a/gapps.ntnu.edu.tw/sofchang/otherresources/mr}}</ref><ref>{{cite web |author1=張千惠 |title=定向行動技能檢核表 |url=https://sites.google.com/a/gapps.ntnu.edu.tw/sofchang/otherresources/mss}}</ref>
