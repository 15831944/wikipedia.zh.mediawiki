'''定向行動'''（Orientation and Mobility，縮寫 O&M），是一門有關如何協助[[視覺障礙|視覺障礙]]者培養或重建其[[移動|移動]]能力之專業，其功能在於訓練視覺障礙者透過[[感官|感官]][[知覺|知覺]]蒐集、判斷、選擇[[環境|環境]][[資訊|資訊]]，知道自身[[位置|位置]]並計畫如何運用合適[[輔具|輔具]]及[[策略|策略]]，以便安全、有效率地移動到目的地。

定向行動最常被連結於教導視障者使用[[白手杖|白手杖]]。實際上，定向行動指導[[白手杖|白手杖]]僅是其中一環，其他包括感官知覺訓練、建立對環境的[[心理地圖|心理地圖]]、[[人導法|人導法]]、[[徒手法|徒手法]]、[[手杖技巧|手杖技巧]]、路口/巷口定位及安全穿越、搭乘大眾交通運輸工具、尋求協助及與他人互動技巧等等。<ref>{{cite web |title=Orientation and Mobility Instructors: Resources from the American Foundation for the Blind |url=http://www.afb.org/info/programs-and-services/professional-development/orientation-and-mobility/123}}</ref>

== 定向行動之於視覺障礙者之重要性 ==
無論是先天或是後天的視覺障礙者，如何安全有效的移動是首先要克服的難題。
視障者學習定向行動相關概念及技能，於下列各方面均有助益: <ref>{{cite book |author1=Rev. Tomas J. Carroll |title=Ying jie shi mang mang shi jie : mang de yi yi . ying xiang ji mian dui |publisher=Ya ge |isbn=957-8763-57-3 |edition=Chu ban}}</ref>

* [[心理|心理]]安全感及提升[[自信|自信]]
* 身體功能及[[健康|健康]]維持與促進
* 溝通及[[社交|社交]]技巧之維持與促進
* [[經濟|經濟]]能力之獲得或回復
* 視障者提升獨立自主能力，減少依賴，同時減輕家人負擔及社會[[成本|成本]]支出
* 視障者提升獨立自主能力，增加主動性，積極參與社會，履行[[公民|公民]]義務

== 定向行動教學執行者 ==
定向行動是由取得合格結訓證書或證照的[[定向行動訓練師|定向行動訓練師]]（英語:O&M Specialist / O&M Instructor）執行。<ref>{{cite web |author1=張千惠 |title=定向行動老師之角色與應負職責 |url=https://sites.google.com/a/gapps.ntnu.edu.tw/sofchang/otherresources/mr}}</ref>

== 功能性視覺評量 ==

此評估是由專業[[定向行動訓練師|定向行動訓練師]]施測，目的是瞭解視障者如何運用現有之視覺能力於日常生活中，有別於醫療人員之[[視力檢查|視力檢查]]及視覺訓練，強調的是日常生活中視覺運用之功能性。[[定向行動訓練師|定向行動訓練師]]依據評估結果與視障者共同討論，訂定適切教學計畫及實施適性個別化教學。<ref>{{cite web |author1=張千惠 |title=功能性視覺評估|url=https://sites.google.com/a/gapps.ntnu.edu.tw/sofchang/otherresources/fva}}</ref>
