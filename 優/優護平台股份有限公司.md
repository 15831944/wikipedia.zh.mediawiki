'''優護平台股份有限公司'''（Ucarer Inc.）<ref>{{cite web|url=https://aws.amazon.com/tw/solutions/case-studies/ucarer/|title=UCarer Case Study|agency=AWS|language=en}}</ref>為台灣一家照護服務媒合平台公司，創立於2015年，總部位於台北市。共同創辦人為陳宏益<ref>{{cite news|url=http://host.cc.ntu.edu.tw/sec/schinfo/epaper/article.asp?num=1275&sn=14635|title=生醫電資所陳宏益創辦臺灣首家短期照護媒合平臺上線|date=2016-08-31|work=台大校訊第1275期|agency=台大校訊|language=zh-tw}}</ref>、劉詩瀚。陳宏益目前擔任公司董事長，劉詩瀚為總經理<ref>{{cite news|url=https://innoservice.org/11143/專訪-優照護總經理劉詩瀚/|title=[專訪] 優照護總經理劉詩瀚|last=熊|first=耿得|date=2016-09-27|work=|agency=服務創新電子報|language=zh-tw}}</ref>。

公司以開發[[流動應用程式|行動應用程式]]連結[[居家服務員|照顧服務員]]和照顧需求者，提供照顧服務業的[[分享型经济|分享型經濟]]服務。以電子商務的媒合模式，藉由App解決尋找居家照護人員的不便，讓有照護需求的使用者透過網際網路，找到符合條件的照顧者，並預約服務。不僅雙方資訊透明，還可即時溝通，縮短了流程並避免許多問題的發生。<ref>{{cite web

== 歷史 ==
優照護的行動應用程式在2016年於台灣地區推出，支援[[iOS|iOS]]和[[Android|Android]]系統的智慧型手機，提供居家照顧服務員的媒合服務。<ref>{{cite news|url=https://meet.bnext.com.tw/articles/view/39051|title=[Meet創業之星]短期居家照護不用愁，「優照護」幫你迅速找到專業照護人員！|last=蘇|first=怡綾|date=2016-11-12|work=Meet|agency=數位時代|language=zh-tw}}</ref><ref>{{cite news|url=http://npost.tw/archives/38216|title=優質線上媒合平臺，居家照顧彈性多樣又專業／專訪「優照護」社會企業|last=黃|first=愉婷|date=2017-10-25|work=|agency=NPOst公益交流站|language=zh-tw}}</ref>

== 功能 ==

* '''問問：'''需求者與照顧者可透過「問問」通訊功能，溝通受照顧者的狀況，並讓照顧者評估是否合適接案，可大幅減少資訊落差造成的困擾，提升照護的服務品質。<ref name=":0">{{Cite news|url=https://health.ettoday.net/news/939623|title=不怕工作沒時間、長照難申請！「優照護」提供短期預約 {{!}} ETtoday健康雲|last=ETtoday新聞雲|newspaper=ETtoday健康雲|accessdate=2018-08-15|language=zh-TW}}</ref>

* '''客戶評價：'''來自電子商務的評價機制，可提供客戶客觀的服務評鑑，並透過評價優劣的排序，協助優質人員獲得額外的收入。

* '''GPS定位打卡：'''照護服務人員到居家服務時，常遭遇簽到簽退是否準時的爭議，過往大都雙方各執一詞，無法判斷實際狀況，「優照護」採用[[全球定位系统|GPS]]衛星定位輔助簽到簽退，記錄實際服務的時間地點，減少這類問題產生的爭議。<ref name=":0" />

* '''數位工作排程：'''短期居家照護需求者，有不同時間、地點、對象等狀況，個案管理常造成傳統照護業者的龐大行政負擔，「優照護」提供數位排程管理機制，大幅減輕人工作業的負擔。
