'''池上茜'''（{{lang-ja|池上 茜}}，{{bd||9月1日}}）是[[日本|日本]]女性[[插畫家|插畫家]]及遊戲[[原畫家|原畫家]]<ref>{{cite web | title=プロフィール| language = ja | url =http://akanemakesrevolution.blog121.fc2.com/| accessdate =2012-11-16 }}</ref>。

== 人物 ==
*小時候喜歡《[[Nakayoshi|Nakayoshi]]》及《[[Ribon|Ribon]]》等少女漫畫雜誌，高中時，得到某雜誌的漫畫獎。之後有雜誌社的工作邀約，因而思考以職業漫畫家出道，而報考相關專門學校。以音響類的學科入學，但因挫折感而輟學。

*創立「A・M・R」（Akane Makes Revolution）同人社團。<ref>{{cite web | title=AMR| language = ja | url =http://akanemakesrevolution.blog121.fc2.com/| accessdate =2012-11-16 }}</ref>於2000年成為『{{lang|ja|Liebe 〜想い出の贈り物〜}}』成人遊戲的原畫家。<ref>{{cite web| title=カスタムソフトウェア(日本境外屏蔽)| language=ja| url=http://www.custom-jp.net/software/| accessdate=2012-11-17| deadurl=yes| archiveurl=https://web.archive.org/web/20120117124524/http://www.custom-jp.net/software/| archivedate=2012-01-17}}</ref>

*興趣是音樂鑑賞，喜歡TMN、access、T.M.Revolution等音樂團體。
