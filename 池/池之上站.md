'''池之上站'''<ref>{{cite web|url=https://www.keio.co.jp/chinese_t/railwaymap/map.pdf|title=京王電鐵路線圖|publisher=京王グループ|accessdate=2017-05-28}}</ref>（{{jpn|j=池ノ上駅|hg=いけのうええき|rm=Ikenoue eki|f=y}}）是一個位於[[東京都|東京都]][[世田谷區|世田谷區]][[代澤|代澤]]二丁目，屬於[[京王電鐵|京王電鐵]][[井之頭線|井之頭線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''IN04'''。


== 歷史 ==
* 1933年（[[昭和|昭和]]8年）8月1日 - 隸屬於帝都電鐵的車站開始營運。
* 1940年（昭和15年）5月1日 - 與小田原急行鐵道合併，成為該公司帝都線的車站。
* 1942年（昭和17年）5月1日 - [[小田急電鐵|小田急電鐵]]與[[東京急行電鐵|東京急行電鐵]]（大東急）合併。
* 1948年（昭和23年）6月1日 - 從東急分離，成立京王帝都電鐵，成為該公司井之頭線的車站。

== 車站構造 ==
配置有1面2線[[島式月台|島式月台]]、[[跨站式站房|跨站式站房]]的[[地面車站|地面車站]]。2006年3月進行車站改善工程，同時也改將車站提升為無障礙空間。雖然有2座出入口，但由於車站的改善工程中空間受到挪用，設置了中央通道與月台、南北側出入口共3座[[電梯|電梯]]的關係，造成北側出入口的位置僅有電梯而無一般階梯可以使用。

廁所原設於月台上，在站房工程完工後改設立於2樓的車站閘口內，並基於[[通用設計|通用設計]]的一環，同時也設有無障礙洗手間。

== 車站周邊 ==
[[File:IkenoueShotengai.jpg|thumb]]
車站出入口面對{{link-ja|東京都道420號鮫洲大山線|東京都道420号鮫洲大山線}}。該道路與車站西側的鐵道呈平面交叉，但因路幅僅5-6m的程度，沒有足夠的空間形成站前廣場。該道路周邊有商業區、[[安哥拉|安哥拉]]駐日大使館，以及池之上站前[[郵便局|郵便局]]。至下北澤站步行約8分左右。周邊一帶是閑靜的住宅區。

附近有{{link-ja|松蔭中學校、高等學校 (東京都)|松蔭中学校・高等学校 (東京都)|松蔭中學校、高等學校}}之外，南側約步行20分鐘的距離有{{link-ja|駒場學園高等學校|駒場学園高等学校}}，在此站不乏見到通學學生。
