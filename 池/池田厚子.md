'''池田厚子'''（{{bd|1931年|3月7日||catIdx=Ikeda, Atsuko}}）是前[[日本皇室|日本皇室]]成員，[[昭和天皇|昭和天皇]]與[[香淳皇后|香淳皇后]]的四女兒，原稱號為'''順宮厚子內親王'''，後與舊[[岡山藩|岡山藩]]主池田家第16代家主{{link-ja|池田隆政|池田隆政}}結婚後脫離皇室身份。1988年至2017年間曾任神宮祭主，獲勳勳一等[[寶冠章|寶冠章]]。她有三姊（[[東久邇成子|東久邇成子]]、[[久宮祐子內親王|久宮祐子內親王]]、[[鷹司和子|鷹司和子]]）、兩弟（[[明仁天皇|明仁天皇]]、[[常陸宮正仁親王|常陸宮正仁親王]]）和一妹（[[島津貴子|島津貴子]]）。徽印為菊櫻<ref>1952年（昭和27年）9月3日 [[朝日新聞|朝日新聞]]「お支度もすべて簡素に 菊桜を御移植」<br />「菊櫻」為[[山櫻|山櫻]]的一種</ref>。

== 生平 ==
1931年3月7日誕生。1936年起，離開雙親到吳竹寮與姊妹一起接受教育。於{{link-ja|學習院初等科|学習院初等科}}、女子學習院（在學時女子學習院改制成{{link-ja|學習院女子中等科|学習院女子中・高等科}}、{{link-ja|學習院女子高等科|学習院女子中・高等科}}）完成學業後，進入[[學習院女子短期大學|學習院女子短期大學]]文科就讀，二年級時轉往家庭生活科並順利畢業。

1951年5月2日，拜訪[[中國、四國地方|中國、四國地方]]之際在[[岡山市|岡山市]][[後樂園|後樂園]]的[[懷石料理|懷石料理]]餐廳{{link-ja|荒手茶寮|荒手茶寮}}和{{link-ja|池田隆政|池田隆政}}相遇。池田隆政是[[備前國|備前]][[岡山藩|岡山藩]]主的後裔，父親{{link-ja|池田宣政 (侯爵)|池田宣政 (侯爵)|池田宣政}}為舊[[華族_(日本)|華族]]的前侯爵。此後，6月底在正式的會面場合訂婚，由{{link-ja|松平康昌|松平康昌}}作媒。

同年5月17日[[貞明皇后|貞明皇后]][[駕崩|駕崩]]，經過第一期的50日服喪期間，7月10日[[昭和天皇|昭和天皇]]命令[[宮內廳|宮內廳]]長官{{link-ja|田島道治|田島道治}}發表訂婚的消息。按照1947年廢止的{{link-ja|皇室服喪令|皇室服喪令}}中規定厚子內親王應服喪150日，且這次訂婚經由昭和天皇等人而非{{link-ja|皇族會議|皇族会議}}決定，這兩點與過往相比具有相當不同之處。

直到皇室全員服喪完畢，1952年10月10日和池田隆政結婚，也正式脫離皇室身份。內親王遠嫁[[岡山縣|岡山縣]]的農場主人在過去也是罕見的例子。1953年2月，夫妻創辦有限公司池田產業。1960年5月{{link-ja|池田動物園|池田動物園}}成立，由池田隆政就任園長。

1964年4月29日，獲頒勳一等[[寶冠章|寶冠章]]，並擔任[[日本赤十字社|日本赤十字社]]岡山縣支部有功會名譽會長。

1965年，罹患[[敗血症|敗血症]]，於{{link-ja|岡山大學醫院|岡山大学病院|岡山大學醫學部附屬醫院}}接受長期治療。掛念厚子病情的昭和天皇夫婦屢次前往岡山探病，厚子也曾於生病之際為昭和天皇夫婦、[[明仁天皇|皇太子明仁親王]]夫婦當起在地導遊<ref>[http://ikedazoo.com/html/rekishi.html 池田動物園をおうえんする会＞池田動物園の誕生]</ref>。

1988年，接替身體因素退任的姊姊[[鷹司和子|鷹司和子]]擔任[[伊勢神宮|伊勢神宮]]祭主。2012年4月26日，由姪女[[黑田清子|黑田清子]]擔任臨時祭主<ref>{{Cite web |date=2012-05-08 |url=http://www.chunichi.co.jp/article/mie/20120508/CK2012050802000027.html |title=中日新聞臨時神宮祭主に黒田清子さん三重（CHUNICHI Web） |accessdate=2012-06-06}}<br>{{Cite web |date=2012-05-08 |url=http://www.j-cast.com/2012/05/08131411.html?p=all |title=天皇家長女、黒田清子さんが就任　「伊勢神宮の臨時祭主」 どんな役職なのか J-CASTニュース |accessdate=2012-06-06}}</ref>。同年7月，丈夫池田隆政過世。

2017年6月19日，傳出卸任伊勢神宮祭主的消息，職務正式由黑田清子接手<ref>[https://this.kiji.is/249798616961648120 伊勢神宮祭主に黒田清子さん]共同通信(2017年6月20日), 2017年6月20日閲覧。</ref>。
