'''池莉'''（{{bd|1957年|catIdx=C池}}），[[湖北|湖北]][[仙桃|仙桃]]人，1986年毕业于[[武汉大学|武汉大学]]中文系，[[中国|中国]]当代著名女作家，[[中国作家协会|中国作家协会]]会员，现任武汉市文联主席。
== 作品一览 ==
池莉创作了大量的小说。其中著名的有：
*《[[生活秀|生活秀]]》：[[武汉|武汉]][[吉庆街|吉庆街]]上小饭馆主人的故事。同名电影由[[陶红|陶红]]、[[陶泽如|陶泽如]]主演。
*《[[来来往往|来来往往]]》：武汉发生的婚外情的故事。同名电视连续剧[[濮存昕|濮存昕]]、[[吕丽萍|吕丽萍]]、[[许晴|许晴]]、[[李小冉|李小冉]]主演。
*《[[不谈爱情|不谈爱情]]》和《[[太阳出世|太阳出世]]》：改变成电视连续剧《[[不谈爱情|不谈爱情]]》。两个截然不同的家庭对待自己儿女的恋爱采取截然不同的态度。
*《[[小姐你早|小姐你早]]》：改编成二十集同名电视连续剧。
*《[[有了快感你就喊|有了快感你就喊]]》：改编为电视剧《[[幸福来了你就喊|幸福来了你就喊]]》。

2009年3月28日，於[[香港大学|香港大学]]召開中國文化研究會「第一屆池莉小說研討會」，召集人為[[伍懷璞|伍懷璞]]教授、[[李思齊|李思齊]]教授、[[黎活仁|黎活仁]]博士和[[余境熹|余境熹]]先生。

== 评价 ==
中国当代著名作家[[王蒙|王蒙]]评价池莉说：“池莉的写实风格在读者中大受欢迎，据说她的不少小说可以卖到几十万册，这是我辈不能望其项背的。”<ref>{{cite web |url=http://www.gmw.cn/01wzb/2001-06/03/05-B8674F610035B20448256A600000DBB8.htm|title=《王蒙点评当今诸作家》|accessdate=2011年10月20日|work=常江|publisher=文摘报|language= zh-hans}}</ref>
