'''池上車站'''位於[[台灣|台灣]][[臺東縣|臺東縣]][[池上鄉|池上鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。知名的[[池上飯包|池上便當]]即起源於此站，本站站房已配合「花東新車站運動」計畫改建完工。

== 車站構造 ==
*[[島式月台|島式月台]]、[[侧式月台|侧式月台]]各一座（[[侧式月台|侧式月台]]暫不使用）。
*車站股道有7股線，辦理客貨運。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]。
*停靠狀態：
**[[普悠瑪列車|普悠瑪號]]
***停靠南下：406、412、438、448次(週日停駛) 、6448次 （週日行駛）
***停靠北上：401、411、425次(週日停駛)、6425次 （週日行駛）
**[[太魯閣列車|太魯閣號]]
***停靠南下：402次
***停靠北上：417次（週日停駛）、6417次（週日行駛）。
**[[自強號列車|自強號]]
***不停靠順行：324次、484次（週五行駛）、逆行：301次、6443次（週日行駛）。
**[[莒光號列車|莒光號]]：皆停靠。
**[[臺鐵區間車|區間車]]：皆停靠。
**[[觀光列車|觀光團體列車]]：皆停靠。
*對號列車停靠率：64.0%；所有車種停靠率：71.0%。
*站內設有「鐵路之旅－小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1924年9月1日：開始設站，同日起池上至里壠間完工通車。此地剛好位於「[[大坡池|大陂池]]」之上方，故稱為「池上」。
*2016年6月28日：啟用多卡通刷卡機。
