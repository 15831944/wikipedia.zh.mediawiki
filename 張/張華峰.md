'''張華峰'''{{SBSHK}}{{JP}}（{{lang-en|'''Christopher Cheung Wah-fung'''}}，{{bd|1952年|5月2日}}），出生於[[福建|福建]][[晉江|晉江]]，[[全國政協委員|全國政協委員]]，恒豐證券集團創辦人董事長及行政總裁，現任香港金融服務界立法會議員。

== 背景 ==
本為菲律賓華僑，1960年代由菲律賓遷居香港。曾經就讀左派學校[[培僑中學|培僑中學]]，
現時為[[通達集團|通達集團]]控股有限公司及[[閩港控股|閩港控股]]有限公司獨立非執行董事。

1980年創立恒豐證券，成為華資證券行之一。

參與[[2004年香港立法會選舉|2004年香港立法會選舉]](金融服務界)以總票數92票敗於[[詹培忠|詹培忠]]。

參與[[2012年香港立法會選舉|2012年香港立法會選舉]](金融服務界)以總票數208票成功當選。

2012年10月7日，與六名建制派立法會議員成立「[[香港經濟民生聯盟|香港經濟民生聯盟]]」，張擔任副主席<ref>[http://www.881903.com/Page/ZH-TW/newsdetail.aspx?ItemId=564124&csid=261_341 七名建制派立法會議員組成香港經濟民生聯盟]</ref>。

== 爭議 ==
2013年7月，經民聯議員張華峰要求[[梁振英|梁振英]]回應政府可曾就「占中」進行政經風險評估，有否評估香港警力是否足以應付占中行動，「一旦占領中環發生混亂和衝突，有沒有考慮過在必要時向中央政府提出要求，出動駐港部隊協助香港社會秩序？有否評估這方面的可行性？」

此舉被認為是邀請中央政府干預香港內部事務，語帶恐嚇意味，被認為有機會當上引清兵入關的[[吳三桂|吳三桂]]。<ref>{{cite web |url= http://www.chinese.rfi.fr/%E4%B8%AD%E5%9B%BD/20130712-%E9%A6%99%E6%B8%AF%E5%90%B4%E4%B8%89%E6%A1%82%EF%BC%9A%E5%BC%A0%E5%8D%8E%E5%B3%B0%E6%8F%90%E7%94%A8%E8%A7%A3%E6%94%BE%E5%86%9B%E5%A4%84%E7%90%86%E5%8D%A0%E9%A2%86%E4%B8%AD%E7%8E%AF|title= 香港吴三桂：张华峰提用解放军处理占领中环 |accessdate=2013年7月12日 |publisher=法広中文網 |work= |language=zh-cn}}</ref>

2016年8月31日，香港電台播出2016年立法會選舉金融服務界選舉論壇，主持問三位候選人一條假設性問題：「如果我係丁蟹」。第一個回答問題的候選人就是張華峰，他的回答如下:「如果我係丁蟹，我就要了解佢嘅業務性質，了解佢工作範疇，了解佢嘅專長。知道佢嘅專長，至可以引導佢發揮專業，為社會服務，我希望對丁蟹呢個人，有多啲認識，我暫時對佢認識唔深，希望有機會睇多啲關於佢嘅新聞，喺佢度學到佢嘅經驗、專長。」其後主持跟進再問張華峰，信不信有「丁蟹效應」。張華峰答：「丁蟹效應，有好有唔好，好嘅方面，大家要學習；唔好嘅，大家要改變，呢個就係我對人對事嘅睇法。」

張華峰自2012年出任香港立法會功能界別金融服務界議員，卻完全不認識香港的經濟學術語「丁蟹效應」，令人訝異。

== 公職 ==
全國政協委員、[[香港證券學會|香港證券學會]]會長、[[香港中華總商會|香港中華總商會]]選任會董、[[中華海外聯誼會|中華海外聯誼會]]理事、[[香港友好協進會|香港友好協進會]]董事、[[廣東省工商聯合會|廣東省工商聯合會]]常務委員、[[香港立法會|香港立法會]]議員，以及[[香港福建社團聯會|香港福建社團聯會]]永遠名譽會長。

==馬主生涯==
張華峰是[[香港賽馬會|香港賽馬會]]的會員，自1980年代開始成為馬主，名下馬匹包括「鷹揚」、「高湖福將」、「一世夠運」、「大丈夫」、「創新高」、「再創新高」。

==軼事==
*張華峰在不同場合的發言中，誤稱自己的政黨[[經民聯|經民聯]]為[[民建聯|民建聯]]。事實上，張華峰的確曾是民建聯成員，不過已經退黨。<ref>[https://news.mingpao.com/pns/dailynews/web_tc/article/20160721/s00002/1469037475304 張華峰又講錯 經民聯變民建聯]</ref>
