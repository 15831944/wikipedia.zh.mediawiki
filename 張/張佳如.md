'''張佳如'''（{{bd|1985年|11月22日|catIdx=C張}}）是一名[[臺灣|臺灣]][[新聞主播|新聞主播]]，1985年出生於台灣台北市，畢業於[[國立政治大學|國立政治大學]]。現任[[東森新聞台|東森新聞台]]《HELLO！台灣》[[主播|主播]]，曾參與[[2016年高雄美濃地震|2016年高雄美濃地震]]等事件的報導。張佳如與藝人[[大S|大S]]、[[小S|小S]]是[[表姐|表姐妹]]關係<ref>{{cite web |title=甜美主播張佳如遇粉絲幫結帳 決定這樣回報 |url=https://stars.udn.com/star/story/10089/3032074 |website=udn |accessdate=2018-12-17}}</ref><ref>{{cite web |title=救災現場圍觀 主播怒斥：這不是嘉年華！ |url=http://ent.ltn.com.tw/news/breakingnews/1599360 |website=自由電子報 |accessdate=2018-12-17}}</ref><ref>{{cite web |title=主播張佳如與表姐小S聚餐 笑容甜美氣質清新 |url=http://ent.ifeng.com/a/20151214/42544164_0.shtml#p=1 |website=凤凰网 |accessdate=2018-12-17}}</ref>。
