'''張敏'''（{{lang-en|'''Sharla Cheung Man'''}}，{{bd|1968年|2月7日|catIdx=C張}}），出生於[[上海|上海]]，著名[[香港|香港]]演員，出道25年，拍過近70部影視劇，因常在[[周星馳|周星馳]]的電影裡演出而為人所熟悉<ref name="HKCinebio">{{cite web |author=Lanuque , Arnaud |title=Sharla Cheung Man|publisher=[[Hong_Kong_Cinemagic|Hong Kong Cinemagic]]|url=http://www.hkcinemagic.com/fr/people.asp?id=201|language=fr|accessdate=2008-03-24}}</ref>。

== 生平 ==
張敏出生於[[上海|上海]]，幼年時移居[[香港|香港]]。1985年[[亞洲電視|亞洲電視]]藝員訓練班出身，與[[黎明|黎明]]為同班同學。1987年被[[王晶_(導演)|王晶]]导演發掘<ref name="HKCinebio"/>，加盟了[[永盛電影公司|永盛電影公司]]。她出道不久即成為了香港90年代炙手可熱、拍片最多的女星，諸多[[武俠電影|武俠電影]]都有張敏的身影。她和當時眾明星幾乎都合作過，尤其以與[[周星馳|周星馳]]的合作最廣為人知，例如《[[賭聖|賭聖]]》、《[[逃學威龍|逃學威龍]]》系列、《[[鹿鼎記|鹿鼎記]]》、《[[新精武門|新精武門]]》、《[[摩登武聖|摩登武聖]]》、《[[武狀元蘇乞兒|武狀元蘇乞兒]]》、《[[九品芝麻官|九品芝麻官]]》等等。1991年与[[劉德華|劉德華]]合作的《[[與龍共舞|與龍共舞]]》让她获得[[1992年香港電影金像獎|第11届香港電影金像獎]]「最佳女主角」提名。

1995年拍完《十兄弟》後，張敏即淡出演藝圈，從事服裝、美容、销售等生意。

1997年曾被遊說復出拍電視劇《東方母親》和《戰國紅顏西施》，拍完後又消聲匿跡。

2003年6月張敏短暫復出，將演藝經紀合約簽在中国的九州亞華公司，拍攝了兩部連續劇和一些廣告後又消失螢光幕前。

2013年傳出張敏低調下嫁前經濟人[[劉永輝|劉永輝]]，劉是[[北京|北京]]人兩人婚後定居[[北京|北京]]。然而2017年張敏否認澄清「我從來沒有結過婚」表示一切都謠言。

2017年在台灣拍攝信義微風百貨公司DM被認為想復出，外傳因離婚與事業崩盤才復出，但之後仍無戲劇作品，張敏否認離婚一事認為從未結婚何來離婚，並承認目前與一小10多歲男子交往。<ref>[http://www.chinatimes.com/realtimenews/20170305001787-260404 張敏首談戀情]</ref>

== 軼聞 ==
張敏是90年代唯一全無緋聞的一線女星，外傳當時張敏的男友或後台是香港演藝圈老大兼黑社會大哥[[向華勝|向華勝]]而無人敢沾染。
