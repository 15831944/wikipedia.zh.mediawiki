'''張星建'''（{{Bd|1905年|10月23日|1949年|1月20日}}<ref name=":1">{{Cite journal|title=台灣文學與中央書局|author=巫永福|url=|journal=[[台灣文藝_(吳濁流)|台灣文藝]]|issue=111|doi=|others=|year=1988|volume=|page=|pmid=}}</ref>），[[筆名|筆名]]'''掃雲'''，是一位出身臺中[[龍井區|龍井]]的[[文化|文化]]運動者。

== 生平 ==

=== 早年生活 ===
[[明治|明治]]三十八年（1905年）10月23日，張星建[[出生|出生]]於[[母親|母親]]林氏位在[[臺中州|臺中州]]大甲郡[[龍井庄|龍井庄]]<ref>{{Cite book|title=臺灣人士鑑|last=|first=|publisher=[[興南新聞|興南新聞]]社|year=1943|isbn=|location=|pages=}}</ref>的[[娘家|娘家]]（一說於[[臺中市_(州轄市)|臺中市]]楠町<ref name=":1" />），並成為他父母第一個誕生的[[子女|孩子]]；當時，母親的[[家族|家族]]在地方上具有聲望，並有多位親族考取[[秀才|秀才]]或[[醫生|醫生]]。張星建的[[祖父|祖父]]也曾因[[商人|經商]]致富，但在傳承至他[[父親|父]]輩時日漸衰退。<ref name=":0">{{Cite journal|title=美術舞臺上的燈光師─論日治時期張星建在臺灣美術中扮演的角色與貢獻|author=林振莖|url=https://www.ntmofa.gov.tw/ntmofapublish_1047_120.html|journal=臺灣美術|issue=86|doi=|others=|year=2011|volume=|page=88-112|pmid=|deadurl=yes|archiveurl=https://web.archive.org/web/20171107014154/https://www.ntmofa.gov.tw/ntmofapublish_1047_120.html|archivedate=2017年11月7日|df=}}</ref>

十八歲時，張星建考入[[台灣日治時期高等教育機構#.E5.95.86.E6.A5.AD.E5.B0.88.E9.96.80.E5.AD.B8.E6.A0.A1|臺灣總督府商業專門學校]]就讀，並在[[預科|預科]]修習完畢後入[[本科教育|本科]]就讀。不久，他因患病決定退學，並自此沒有再繼續升學。隨後，他隨父親進入乳製品工廠任職。<ref name=":0" />

[[大正|大正]]十五年（1926年），張星建進入千代田里太郎位在臺中市的律師事務所工作。1928年，張星建改於剛成立不久的[[中央書局|中央書局]]擔任其營業部主任<ref>{{Cite news|url=http://www.chinatimes.com/newspapers/20150120000465-260115|title=沉寂17年 中央書局回來了|author=|date=2015-03-19|work=|newspaper=中時電子報|accessdate=2017-11-06|deadurl=yes|archiveurl=https://archive.is/20150319085508/http://www.chinatimes.com/newspapers/20150120000465-260115|archivedate=2015-03-19}}</ref>；此後，張星建開始在每年[[暑假|暑假]]帶領書局於當地舉辦[[西洋畫|洋畫]]講習會，甚至每年舉辦一次以上免費公開繪畫展覽會。<ref name=":0" /><ref name=":1" />

=== 文藝生涯 ===
1931年，張星建與[[賴和|賴和]]、[[葉榮鐘|葉榮鐘]]等發起設立「臺灣文藝作家協會」；同時，他經常參與「中州[[俱樂部|俱樂部]]」所舉辦之文藝活動。<ref name=":0" /><ref name=":2">{{Cite journal|title=張星建及其文藝之道──以《南音》、《臺灣文藝》為考察中心|author=王文仁|url=|journal=東吳中文學報|issue=二十三|doi=|others=|year=2012|volume=|page=327-352|pmid=}}</ref>

1932年5月，張星建接任《[[南音_(雜誌)|南音]]》雜誌[[分銷|發行]]人兼[[編輯|編輯]]職務。<ref name=":0" />

1934年，張星建與[[張深切|張深切]]、賴明弘等於臺中西湖咖啡館發起全島文藝作家聚會，舉行「第一回全臺文藝大會」， 並決定成立[[臺灣文藝聯盟|臺灣文藝聯盟]]（簡稱「文聯」）且親自擔任該組織之《[[台灣文藝_(台灣文藝聯盟)|臺灣文藝]]》雜誌的發行人兼編輯人。同年，張星建也義務協助[[郭雪湖|郭雪湖]]籌辦其在[[臺中州立圖書館|臺中州立圖書館]]的個展，並在展前與其一同前往拜訪邀請[[林獻堂|林獻堂]]蒞臨觀展。<ref name=":0" /><ref name=":2" />

1930年代初，張星建曾參與於由[[陳炘|陳炘]]與[[宮原武熊|宮原武熊]]等人士所共同倡議成立的「[[大東亞共榮圈|大東亞共榮]]協會」。<ref name=":0" />

日治末期，張星建曾擔任[[皇民奉公會|皇民奉公會]]臺中州支部奉公委員及臺中州地主增產協力會委員；同時，他也與陳炘、顏春福、[[楊逵|楊逵]]等於臺中組成「臺中藝能奉公會」，演出[[閩南語|閩南語]]舞台劇《怒吼吧！中國》，又兼任《臺灣文學》雜誌編輯委員。<ref name=":0" />

=== 捲入亂事 ===
[[戰後台灣|戰後]]初期，張星建曾加入臺中地區[[歡迎國民政府籌備會|歡迎國民政府籌備會]]並擔任常任委員，負責「庶務」及「右連絡部」工作。[[陳儀|陳儀]]就任[[臺灣省行政長官公署|臺灣省行政長官公署]]行政長官後，張星建更被指派為[[三民主義青年團|三民主義青年團]]（簡稱「三青團」）[[臺中市_(省轄市)|臺中市]]分團長。不久，[[二二八事件|二二八事件]]發生，張星建團下區隊長林連城及賴耿松被逮捕，[[林連宗|林連宗]]及童炳輝則陸續[[失蹤|失蹤]]；同時，張星建加入臺中市時局[[二二八事件處理委員會|處理委員會]]，並在[[國民革命軍|國民革命軍]]進入當地[[清鄉|清鄉]]時逃亡[[南投縣|南投]]山區藏匿數月。<ref name=":0" />

1949年1月21日，張星建死亡並陳屍於[[中區_(臺中市)#.E6.AD.B7.E5.8F.B2|臺中市大正橋]]下<ref>{{Cite news|url=|title=大正橋下發生離奇殺案 張星建無辜慘死|author=|date=1949-01-22|work=[[臺灣民聲日報|臺灣民聲日報]]|accessdate=}}</ref>。<ref name=":0" /><ref name=":1" />

== 家庭 ==
張星建育有張惠炘等子女。<ref name=":0" />

== 評價 ==
* 小說家[[呂赫若|呂赫若]]認為張星建是「臺灣文化界的綠洲」。<ref name=":0" />
* 作家[[巫永福|巫永福]]稱他為「臺中市的[[甘草|甘草]]」及「{{Link-ja|せわやく|世話人}}（世話役）」。<ref name=":0" />
* 藝術創作者[[王水河|王水河]]稱他為「臺中的文化市長」 。<ref name=":0" />

== 軼事 ==
* 張星建曾被張深切取[[綽號|綽號]]為「萬善堂」。<ref name=":0" />
