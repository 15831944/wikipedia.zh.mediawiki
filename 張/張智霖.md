'''張智霖'''（英文名：'''Julian Cheung Chi Lam'''；{{bd|1971年|8月27日|catIdx=C張}}），[[香港|香港]][[艺人|艺人]]，現為[[HMV數碼中國集團|中國3D數碼娛樂有限公司]]旗下藝人、[[星娱乐|星娛樂]]旗下歌手。其妻為1990年[[香港小姐|香港小姐]]冠軍[[袁詠儀|袁詠儀]]。

== 簡歷 ==
張智霖生于[[香港|香港]]、籍貫[[广东省|廣東]][[南海|南海]]，亦是[[澳籍华人|澳籍華人]]。舅父[[曹查理|曹查理]]為香港知名演員，而母親[[曹珍妮|曹珍妮]]則是1962年香港[[工展小姐|工展小姐]]冠軍，其外公是一名[[軍人|軍人]]<ref>{{cite news|title=張智霖媽媽曾是選美冠軍 一家子帥哥美女 |url=http://news.xinhuanet.com/gangao/2013-09/18/c_125408587.htm |language=中文 |publisher=新華網 |date=2013年9月18日 }}</ref>。他年少时就因表姐[[上山诗钠|上山诗钠]]的关系与圈中一些知名艺人相识，連[[张学友|张学友]]也曾說過收他為弟子。

張智霖在小学時期曾就讀于香港[[番禺會所華仁小學|番禺會所華仁小學]]，但因太沉迷踢足球導致他無心向學，結果未能直升原校被調到[[嶺南中學|嶺南中學]]。

1984年，13歲時父母离异，並於15歲中四時与父亲一同移民到[[澳洲|澳洲]]，就讀於[[悉尼|悉尼]]的Pendle Hill High School<ref>{{cite web |title=Chi Lam’s Biography |trans_title=智霖小傳 |author=Kelly Gu |website=Chilam.com |publisher=C&L |url=http://www.chilam.com/english/biography/ |date=2001-06-15 |accessdate=2013-08-31 |language=en}}</ref>。1990年，張智霖跟父親回到香港。

1991年11月，他與[[許秋怡|許秋怡]]推出粵語合唱專輯《[[現代愛情故事|現代愛情故事]]》（其中同名主打歌曲《現代愛情故事》改编翻唱自台灣[[華納唱片_(台灣)|飛碟唱片]]著名組合[[小虎隊_(台灣)|小虎隊]]的國語作品《叫你一聲My Love》）而成名，唱片達雙白金銷量，碟內歌曲成為熱門合唱K歌。更與拍檔許秋怡於翌年1月榮獲[[商業電台|商業電台]][[1991年度叱咤樂壇流行榜頒獎典禮|1991年度叱咤樂壇流行榜頒獎典禮]]「叱咤樂壇新力軍組合金獎」。

1992年下旬，張智霖正式以獨立身份於香港樂壇發展，10月推出首張個人大碟《[[逗我開心吧|逗我開心吧]]》，開始歌唱事業，更為他奪得新人獎——[[商業電台|商業電台]][[1992年度叱咤樂壇流行榜頒獎典禮得獎名單|1992年度叱咤樂壇流行榜頒獎典禮]]「叱咤樂壇新力軍男歌手銀獎」及[[TVB|TVB]][[十大勁歌金曲頒獎典禮|十大勁歌金曲頒獎典禮]]「最受歡迎新人獎（男歌星）銅獎」。

1993年9月，推出第二張唱片《[[如此這般想你|如此這般想你]]》，同時獲得[[新城電台|新城電台]][[新城勁爆家族音樂大賞|新城勁爆家族音樂大賞]]「勁爆家族大躍進男歌手獎」  。1994年，張智霖轉會至[[新力唱片|新力唱片]]。

其後，他於1994年至1997年期間推出多張唱片專輯均有不俗反應，當中《[[Chilam|Chilam]]》（1994年10月）、《天地男兒超級精選》（1996年3月）、《我也喜歡你》（1997年3月）更達白金銷量，《我也喜歡你》更是他首次奪得四台冠軍歌曲，是當時流行品牌的電視廣告歌曲，熱播一時。

由於忙於電影發展，1998年至1999年張智霖的歌唱事業進入停滯階段，1999年4月推出的專輯《孩子先生》反應未如預期。

2000年，由於參演TVB電視劇《[[十月初五的月光|十月初五的月光]]》飾「文初（初哥哥）」一角而大受歡迎，獲得「本年度我最喜愛的電視角色」及「本年度我最喜愛的拍檔（戲劇組）」成為他代表作之一。劇中主題曲《祝君好》熱播全城，使張智霖人氣大幅攀升。2000年，更推出暌違兩年多的全新專輯《十指緊扣》，獲得不俗銷量。

2003年，推出《愛與夢 新曲+精選》，主打歌《相愛無夢》為電視劇《西關大少》主題曲，亦流行一時。其後數年，暫時推出音樂作品專注電影發展。

2009年復出香港樂壇并簽約唱片公司[[Neway_Star|Neway Star]]，新歌《妳太善良》派台後大獲好評，深受樂迷喜愛，其後推出新碟《I Am Chilam》。同年底拍摄[[香港|香港]][[TVB|TVB]][[电视剧|电视剧]]《[[魚躍在花見|魚躍在花見]]》，該劇於2011年2月播映，成為全年收視季軍。

2011年3月26至27日，於[[香港體育館|香港體育館]]舉行“我係外星人”，舉行入行20年來首個個人演唱會，演唱會CD達白金銷量。

2013年，憑藉TVB電視劇《[[衝上雲霄II|衝上雲霄II]]》“Cool魔”顧夏陽一角而再度爆紅，因而人氣急升，並成為城中熱話，亦為其新代表作品，其後更奪得[[TVB_馬來西亞星光薈萃頒獎典禮2013|TVB 馬來西亞星光薈萃頒獎典禮2013]]「最愛TVB電視角色」及「最喜愛TVB男主角」，成為雙料大馬視帝。於[[萬千星輝頒獎典禮2013|萬千星輝頒獎典禮2013]]獲得全民選出的「最受歡迎電視男角色」，可惜大熱倒灶失落「最佳男主角」。同年9月宣布簽約[[萧定一|萧定一]]旗下公司，唱片公司轉往[[星娛樂|星娛樂]]。

2014年7月6日至8日，於香港紅磡體育館舉行「Crazy Hours 巡迴演唱會」第一站，舉行第二個個人演唱會。其後於2014年8月29日[[馬來西亞|馬來西亞]]，2015年7月4日[[澳門|澳門]]，2016年1月16日[[廣州|廣州]]，2016年5月27日[[深圳|深圳]]，2016年8月27日[[上海|上海]]先後巡迴，其中上海場更是於張智霖生日當天舉行，票價827元的入場票亦有特別封套。

2018年12月中，隨着[[HMV|HMV]]清盤結業，張智霖以及其他相關歌手於HMV的合約亦相告結束。<ref>{{cite web |author1=梁思浩 |title=思浩大談蕭若元公子蕭定一HMV清盤張智霖大受影響！大爆某女藝人呃圈中人出Show內幕！(大家真瘋Show) |url=https://www.youtube.com/watch?v=-TWXek5PsCw&t=43s}}</ref>

== 家庭生活 ==
張智霖於2001年2月8日與[[袁詠儀|袁詠儀]]於[[美國|美國]]註冊結婚<ref>[http://news.21cn.com/caiji/roll1/2012/10/22/13333180.shtml 袁咏仪晒爱子游玩照 魔童骑马耍宝卖萌] {{Wayback|url=http://news.21cn.com/caiji/roll1/2012/10/22/13333180.shtml |date=20130719030959 }}</ref>，2006年11月12日袁詠儀在香港[[養和醫院|養和醫院]]剖腹誕下8磅男嬰，取名張慕童（乳名：魔童），英文名為「Morton」。

== 政治活動 ==
張智霖、[[袁詠儀|袁詠儀]]、[[區瑞強|區瑞強]]和「肥媽」[[Maria_Cordero|Maria Cordero]]在[[2018年11月香港立法會九龍西地方選區補選|2018年11月香港立法會九龍西地方選區補選]]為獲[[民建聯|民建聯]]、[[經民聯|經民聯]]等支持的[[建制派|建制派]]候選人[[陳凱欣|陳凱欣]]站台。<ref> {{cite web |language=zh-hk |url=https://hk.news.appledaily.com/local/realtime/article/20181126/58958499 | title=【九西補選】張智霖袁詠儀都撐陳凱欣 網民：罷買FANCL  |publisher=蘋果日報 |date=2018-11-26 |accessdate=2018-11-28}}</ref>

== 軼聞 ==
張智霖是[[香港賽馬會|香港賽馬會]]會員，也是2012至2013年度馬季自購新馬進口許可證中簽會員（與陳汝安、[[郭富城|郭富城]]、[[周文曦|周文曦]]合夥擁有），馬名初為「愛河」<ref>[http://news.hotpot.hk/fruit/art_main.php?iss_id=20110608&sec_id=462&art_id=15321950 馮仁昭四圍超：城城再做馬主　帶挈張智霖]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://hk.news.yahoo.com/忠告馬迷勿-身-城城納新寵即墮愛河-223000553.html 忠告馬迷勿瞓身 城城納新寵即墮愛河 ]</ref>，但其後更名「愛將」<ref>[http://www.hkjc.com/chinese/corporate/racing_news_item.asp?in_file=/chinese/news/2013-03/news_2013032101556.html 馬匹易名 - P288及P308]</ref>。「愛將」由[[方嘉柏|方嘉柏]]訓練，在港出賽六次取得四冠兩亞，累積贏得獎金2,742,000港元，已於2014年10月初宣布退役。
