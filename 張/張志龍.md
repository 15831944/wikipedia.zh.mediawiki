'''張志龍'''（英文名：Richard Chang，{{bd|1964年|10月23日|||}}），台灣企業家、作家，出生於桃園。

2010年，與周杰倫等人共同發起「擁抱絲路」活動。這個史上最長里程的超級馬拉松，不但開展台灣海峽兩岸共同舉辦大型國際性活動的先河，也突破中西亞國家外交和簽證的困難，讓國際團隊得以長跑和沿途宿營的方式，成功地穿越六個國家，完成一百五十天、 一萬公里的絲路長征<ref>{{Cite web |url = http://www.ettoday.net/news/20120622/64061.htm |title = 《擁抱絲路》林義傑長征1萬公里、150天全紀錄 |date = 2012年6月22日  |publisher = ETtoday  }}</ref><ref>{{Cite web |url =http://ec.cpu.edu.tw/files/13-1028-24525-1.php?Lang=zh-tw |title = 「中央警察大學」於104年11月份「中央警察大學」月會 |date = 2015-11-17   }}</ref><ref>{{Cite web |url = http://inpo.pixnet.net/blog/post/46241100 |title = 【逆風名人講堂】《擁抱絲路》圓夢者張志龍：相信自己做得 |date = 2012年7月30日  }}</ref>。
。

張志龍現為敦南藝術講堂負責人。

==經歷==
*1988-1989年，東盟纖維（股）公司 業務工程師
*1989-1991年，台灣[[聯合利華|聯合利華]]（股）公司 產品副理
*1991-2003年，台灣[[雀巢|雀巢]]（股）公司 行銷處長
*2003-2005年，Goldmund Consulting Co.,Ltd Managing Director
*2005-2010年，欣技資訊（股）公司 執行副總經理
*2010年迄今，中華地球探索保護協會共同發起人
*2012-2018年，鉅賦福斯汽車執行長。
*2018年迄今，敦南藝術講堂負責人。
