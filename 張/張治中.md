'''張治中'''{{BD|1890年|10月27日|1969年|4月10日}}，[[表字|字]]'''文白'''，[[安徽省|安徽省]][[巢县|巢县]]（今[[巢湖市|巢湖市]]）人，原为[[中國國民黨|中國國民黨]]黨员，與[[政學系|政學系]]關係密切，[[國民革命军|國民革命军]][[二級上將|二級上將]]，曾任[[湖南省政府|湖南省政府]]主席、[[新疆省政府|新疆省政府]]主席，曾主導参與多次[[國共和談|國共和談]]。1947年在原籍[[安徽省_(中华民国)|安徽省]][[巢縣|巢縣]]當選為[[第一屆國民大會代表|第一屆國民大會代表]]。

1949年6月宣布脫離中國國民黨，后加入[[中國國民黨革命委員會|中國國民黨革命委員會]]，任民革中央副主席。中華人民共和國成立後，历任[[中央人民政府委员|中央人民政府委员]]、[[西北军政委员会|西北军政委员会]]副主席、[[中华人民共和国国防委员会|中华人民共和国国防委员会]]副主席，于[[第三届全国人民代表大会常务委员会|第三届全国人大常委会]][[全国人民代表大会常务委员会副委员长|副委员长]]任上病逝。

== 早年經歷 ==
張治中1916年畢業於[[保定軍校|保定軍校]]3期步科，與[[何鍵|何鍵]]、[[夏威|夏威]]、[[葉琪|葉琪]]、[[白崇禧|白崇禧]]等皆同期畢業。之後曾參加護法，並在滇軍、桂軍、川軍中任連長、營長。

== 中年經歷 ==
1924年，張任校長的廣西桂軍軍校結束，[[蔣介石|蔣介石]]調張到[[黃埔陸軍軍官學校|黃埔陸軍軍官學校]]，任上校代理入伍生總隊隊長。不久，國民革命軍成立，張任第二師參謀長。黃埔軍校後來成立軍官團，由張任團長。1926年[[北伐|北伐]]開始後，設立學兵團及黃埔軍校武漢分校，張任學兵團長及分校教育長。1927年蔣介石清黨以後，張治中從武漢到南京；蔣介石下野後，張出國考察。

1928年[[黃埔軍校|黃埔軍校]]遷[[南京|南京]]，改稱中央陸軍軍官學校。張治中由1928年起一路出任陸軍軍官學校教育長，直至1937年。在這期間張亦曾多次為蔣介石帶兵。1928年[[馮玉祥|馮玉祥]]反蔣，張任武漢行營主任；[[中原大戰|中原大戰]]時，張任教導第二師師長，率部進攻[[開封|開封]]馮玉祥部。1933年[[閩變|閩變]]，張任第四路軍指揮平亂。與其他國民黨將領不同，每次戰事完結後張治中都會主動交出兵權，回到軍校。

== 中國抗日戰爭 ==
1932年上海爆發「[[一二八事變|一二八事變]]」，張臨時被任為第五軍軍長，指揮國軍中的中央精銳，助[[十九路軍|十九路軍]]守衞上海。中日簽署停戰後，張立即投入京滬防守的計劃工作。1936年，正式出任京滬警備司令。1937年4月6日，中央陸軍軍官學校教育長張治中辭職照准，校長蔣介石派[[陳繼承|陳繼承]]接任<ref name="民">{{cite book |editors=[[李新_(1918年)|李新]]總主編，中國社會科學院近代史研究所中華民國史研究室，韓信夫、姜克夫主編 |title=《中華民國史大事記》 |location=北京 |publisher=[[中華書局|中華書局]] |year=2011 |isbn=9787101079982}}</ref>{{rp|5399}}。[[抗戰|抗戰]]開始後，張兼任第九集團軍司令。8月13日，中日在上海爆發衝突引發[[淞滬會戰|淞滬會戰]]。張治中指揮國民革命軍的中央系部隊在上海以攻勢對抗日軍。8月17日，張密電蔣及[[何應欽|何應欽]]：
{{quote|
「特急。南京委員長蔣、部長何：……本軍……按預定部署全部開始總攻擊，最初目的原求遇隙突入，不在攻堅，但因每一通路，皆為敵軍堅固障礙物阻塞，並以戰車為活動堡壘，終至不得不對各點目標施行強攻。其各部激戰鬥終日之狀況如次：1. 八十八師以主力由北分向日本墳山、八字橋、法學院、虹口公園攻擊，往返爭奪，傷亡甚重，僅法學院一處，已犧牲近一營之眾。……2. 八十七師……對滬江大學、公大紗廠及引翔港鎮方面，則激戰終日，尚未得手……3. 本日我炮兵……因目標堅固，未得預期成果……4. 本日未聞敵兵艦炮聲。5. 職……見空軍動作亦甚敏捷勇敢。但敵高射槍炮似有電氣裝置，所構成之威力圈彈發如雨，其聲之密勝年夜爆竹，我炮兵曾同時向其制壓。本日……官兵傷亡數另呈。職張治中叩<ref name="抗日戰爭圖誌">楊克林、曹紅著：《中國抗日戰爭圖誌》，廣州：廣東旅游出版社，1995年1月，ISBN 978-7-80521-541-9</ref>{{rp|292}}。……」
}}

之後，國軍被迫從前線後撤，張改任[[湖南省政府|湖南省政府]]主席兼省保安司令。蔣介石對張治中的淞滬戰場上表現十分不滿，初期蒋介石计划用十天消灭日本海军特别陆战队，當時日本海军特别陆战队僅數千人，蒋介石交待張治中指挥数万德械装备的精锐军队前去围歼，中日双方兵力十比一，但張以四個師圍殲日軍還是没有拿下来，蔣氣得在日記寫著：“撤張文白”，又責備[[何應欽|何應欽]]無識人之明，即以[[朱紹良|朱紹良]]代替張治中<ref>李玉等：《重探抗戰史一：從抗日大戰略的形成到武漢會戰1931-1938》</ref>。

1938年11月，日軍進犯湖南一帶。當時中國的[[焦土政策|焦土政策]]規定日军进入城郊30华里以内时，即实施之，破壞任何可能對敵人有用的東西。但因訊息傳遞錯誤，和[[長沙|長沙]]民兵自衛隊自作主張，在日軍進攻長沙前即告焚城，11月12日上午，日军向[[岳阳|岳阳]]以南离长沙还有250里的新墙河进犯时，译电员竟将前方电讯漏一「墙」字，致将「[[新墙河|新墙河]]」变成离长沙仅有12华里的「[[新河|新河]]」。首先得知此訊息的民兵自衛隊在未接到指令的情況下，即擅自放火（即[[文夕大火|文夕大火]]），造成嚴重的人命財產損失<ref>[http://history.news.163.com/09/0617/16/5C19PIJM00011247.html 亲历长沙大火：译电员漏字酿惨祸]</ref>。張治中因而被革職，但在部下[[酆悌|酆悌]]等3人的頂罪以及蔣中正的介入下，張治中並未遭到槍斃，僅獲留任負責善後。但是由部下頂罪的行為使得張治中失去了軍中威信，也不可能繼續從事實質軍職。事實上[[戴笠|戴笠]]、[[陳誠|陳誠]]都提及張治中“不切實際”，“尤其在抗戰軍事上，而湖南除出兵外，其他毫無成績可言<ref>陳誠，〈此次長沙大火不知委座合已不令軍隊執行而令地方政府準備〉</ref>。”

1939年2月，張到[[重慶|重慶]]，任[[軍事委員會委員長侍從室|軍事委員會委員長侍從室]]第一處主任，負責軍事機要，成為蔣介石身旁的重要助手。1940年11月26日，張治中離開重慶，前赴江南九省視察，旨在推動各省加速組織三青團<ref name=民/>{{rp|6420}}。

== 重慶談判 ==
[[File:赫尔利.jpg|thumb]]
1941年「[[皖南事變|皖南事變]]」後，張治中即代表中國國民黨與中國共產黨談判。之後國共雙方的談判一直持續到抗戰結束，張治中多次代表中國國民黨。1945年日本投降，[[毛澤東|毛澤東]]首次飛重慶與中國國民黨會談。張治中與美國特使[[赫爾利|赫爾利]]一同飛到[[延安|延安]]接[[毛澤東|毛澤東]]，在[[重慶|重慶]]代表中國國民黨參與會談九次。國共雙方簽署「[[雙十協定|雙十協定]]」後，張再隨機送毛澤東回延安。

之後美國特使[[乔治·卡特莱特·马歇尔|馬歇爾]]於1946年初來華，並與張治中、[[周恩來|周恩來]]組成三人小組，負責國共雙方軍事整編。此間，張治中經常向馬歇爾抱怨國民政府，為中國共產黨宣傳。3月27日，国民政府代表张治中、中国共产党代表周恩来和美国代表吉伦於重慶怡园正式签订关于派遣執行小組前往[[东北停战协议|東北調處停止衝突協定]]<ref name=民/>{{rp|8019}}。3月28日，國民政府特派張治中為軍事委員會委員長西北行營主任<ref name=民/>{{rp|8019}}。

1947年1月1日，[[國民政府主席|國民政府主席]]西北行轅主任兼[[新疆省|新疆省]]政府主席張治中奉召專機抵南京，1月2日見蔣介石，談打破國共僵局問題，在座者有陳誠、白崇禧、[[陳立夫|陳立夫]]、[[張厲生|張厲生]]、[[雷震|雷震]]等<ref name=民 />{{rp|8254}}。1月16日，蔣決定派張治中攜和談新方案到延安商談和平；和談方案經由[[司徒雷登|司徒雷登]]轉交中共在南京代表王炳南，計有四項：就地停戰、整編軍隊、恢復交通、解決地方政權等<ref name=民 />{{rp|8267}}。

張治中赴延安，商談和平統一，中國共產黨拒絕接受<ref name="年表"/>{{rp|50}}。2月1日，蔣在黃埔路官邸邀宴-{于}-右任、孫科、[[戴傳賢|戴傳賢]]、[[邵力子|邵力子]]、[[王寵惠|王寵惠]]、[[吳鼎昌|吳鼎昌]]、[[張厲生|張厲生]]、[[陳立夫|陳立夫]]、張治中等，就[[行憲|行憲]]準備工作交換意見<ref name=民 />{{rp|8277}}。3月3日上午7時，中共代表董必武自上海抵達南京，車抵下關，遭中國國民黨特務囚禁1小時；董必武回中共辦事處後，即往訪張治中，詢問政府有無致中共人員撤離南京、上海、重慶三地之正式公函，及政府是否表示決心用軍事方式解決國共問題<ref name=民 />{{rp|8302}}。

1948年8月，蔣以張治中為西北軍政長官<ref name="年表"/>{{rp|55-56}}。8月12日，蔣特派西北行轅主任張治中為西北[[軍政長官公署|軍政長官公署]]軍政長官，[[馮治安|馮治安]]、[[劉汝明|劉汝明]]、[[杜聿明|杜聿明]]為[[徐州剿匪總司令部|徐州剿匪總司令部]]副總司令<ref name=民 />{{rp|8657}}。下半年，中國國民黨在軍事、政治及經濟上遭受嚴重打擊。

== 新疆親共 ==
[[File:迪化各界慶祝和平大會盛況：張治中致辭.jpg|thumb]]
1944年，在苏联指使下，新疆发生“[[三区革命|三区革命]]”，[[第二東突厥斯坦共和國|东突厥斯坦共和国]]成立。张治中奉国民政府之令前往新疆谈判，在谈判中，张治中同意新疆维族自治，选举[[新疆省参议会|新疆省参议会]]。在国民政府的诚意之下，[[新疆独立运动|新疆独立运动]]人士放弃原先主张，接受国府条件，推选自己的地方议会，施行[[地方自治|地方自治]]。

與中國共產黨談判同時，張治中亦負責談判統一[[新疆|新疆]]的工作。1944年到1946年6月，張治中任全權代表多次赴[[迪化|迪化]]<small>（今[[烏魯木齊|烏魯木齊]]）</small>，在[[蘇聯|蘇聯]]領事調停下，与[[伊宁事变|伊宁事变]]领导人談判解決新疆問題。1945年9月13日，蔣介石召派張治中赴新疆處理「[[伊寧事變|伊寧事變]]」，是日張治中由重慶飛抵迪化<ref name=民/>{{rp|7826}}。9月14日，中國各戰區、方面軍自是日起開始解決除日軍武裝，至次年2月13日止繳械基本結束，總計俘虜日軍128.3206萬人；張治中在迪化會見蘇聯駐迪化代總領事葉謝也夫，徵詢對「伊寧事變」意見<ref name=民/>{{rp|7827}}。9月16日，張治中晉謁蔣介石，報告解決新疆伊寧事變之途徑，建議請蘇聯政府調解伊寧事變<ref name=民/>{{rp|7830}}。9月20日，國民政府外交部向蔣介石請示後答覆蘇聯駐華大使彼得羅夫，對蘇聯政府願意協助伊寧事變表示感謝，並請蘇駐伊領事代為通知與介紹伊寧領事代為通知與介伊寧事代表到迪化晉謁張治中，商洽進行和平解決辦法<ref name=民 />{{rp|7835-7836}}。1946年3月，张治中出任[[国民政府軍事委員會|国民政府軍事委員會]]委員長[[西北行轅|西北行轅]]主任。5月，蔣介石任張治中為[[新疆省政府|新疆省政府]]主席<ref name="年表">[[陳布雷|陳-{布}-雷]]等編著：《蔣介石先生年表》，台北：[[傳記文學|傳記文學]]出版社，1978年6月1日</ref>{{rp|48}}。6月6日，「[[伊寧事變|伊寧事件]]」最後協議在迪化簽字，張治中下令釋放全新疆之政治犯<ref name=民/>{{rp|8070}}。

3月25日，新疆省政府主席張治中發表告民眾書，重申徹底實行和平條款和施政綱領，民族平等團結<ref name=民/>{{rp|8321}}。

5月，[[蒙古人民共和国|蒙古人民共和国]]军队[[北塔山事件|进攻北塔山]]，张治中再度前往新疆处理此事。7月8日，[[中華民國外交部|外交部]]就外蒙軍隊入侵新疆制造北塔山事件，致蘇聯駐華使館照會，提出抗議<ref name=民/>{{rp|8380}}。

張治中在新疆任期内，已逐漸親共，新疆省政府所任用的重要人員盡是中國共產黨，如以[[屈武|屈武]]任[[廸化市|廸化市]][[市長|市長]]，[[劉孟蒓|劉孟蒓]]任新疆省政府秘書長。因此當時有人認爲張治中為中國共產黨長期之間諜。[[郭寄嶠|郭寄嶠]]、[[上官業佑|上官業佑]]等，均提交中央舉報，蔣介石亦密諭他人，注意張治中的行動<ref>[[龔德柏|龔德柏]]口述：〈張治中的親共與利共〉</ref>。1948年6月25日，蔣在西安召開軍事會議，參謀總長顧祝同、西北行轅主任張治中、副主任[[馬步芳|馬步芳]]、[[馬鴻賓|馬鴻賓]]至西安出席<ref name=民/>{{rp|8625}}。8月12日，蔣特派西北行轅主任張治中為[[西北軍政長官公署|西北軍政長官公署]]軍政長官<ref name=民/>{{rp|8657}}。

== 北平和談 ==
1949年1月，蔣介石下野，[[李宗仁|李宗仁]]代總統立即展開與共產黨和談。1月4日，南京報紙報導張治中向美國及蘇聯兩國大使館接洽，請安排與中共談判<ref name="錄" >{{cite book |year=2011 |title=《1949大流亡：美國外交檔案室密錄》 |editor=王景弘編譯 |location=台北市 |publisher=玉山社出版 |isbn=978-986-294-000-6 }}</ref>{{rp|21}}。2月1日，西北軍政長官張治中下午抵達[[蘭州|蘭州]]，對記者談話稱：和平並沒有絕望；自己為政府派定和平代表，願為和平而努力<ref name=民 />{{rp|8800}}。

3月3日，張治中攜李宗仁函，商承蔣介石對「和平的條件和限度之意見」；並謂：「中共雖已微開和平之門，但前途困難重重<ref name="寧靜">[[蔣經國|蔣經國]]：〈危急存亡之秋〉，刊《風雨中的寧靜》，[[台北|台北]]：[[正中書局|正中書局]]，1988年</ref>{{rp|164}}。」3月28日，張治中以其將赴[[北平|北平]]，特電呈蔣介石，要前往[[溪口鎮_(宁波市)|溪口]]，「當面報告政府所定的和談腹案」<ref name="寧靜"/>{{rp|173}}。蔣介石說：「他來不來無所謂<ref name="寧靜"/>{{rp|174}}。」3月29日，張治中往奉化溪口謁蔣介石，就和談腹案向蔣請示；蔣向張治中表示：「為倡導和平，本人業已引退，因之對政治問題，不便表示意見。今後和談大計，應由李代總統與何院長負責主持。本人甚願以在野身份，盡力支持李代總統使和平早日實現<ref name=民 />{{rp|8854}}。」蔣介石對他非常冷淡，「只邀他遊覽溪口附近的風景」<ref name="寧靜"/>{{rp|174}}。

3月30日，張治中離開溪口<ref name="寧靜"/>{{rp|174}}。張治中偕[[屈武|屈武]]回南京，將蔣之表示寫成新聞稿發表，用以壓攝中國國民黨內派系；並與[[邵力子|邵力子]]參加立法院秘密會議，報告溪口之行，博得贊同<ref name=民 />{{rp|8854}}。行政院臨時政務會議，通過加派[[劉斐|劉斐]]為和談代表，並指派張治中為首席代表<ref name=民 />{{rp|8855}}。

李宗仁派出親共人士如張治中、邵力子、[[章士钊|章士钊]]、[[黄绍竑|黄绍竑]]、[[李蒸|李蒸]]、刘斐（可能是中共地下党。然而，[[俞大維|俞大維]]堅信其『牆頭草』而已，稱其為『中國共產黨』黨員，則略嫌誇大<ref name="俞大維傳-李"/>。）等6人为代表的和谈代表团，張治中任代表團長，於4月1日到[[北平|北平]]進行[[北平和谈|和談]]。4月13日，張治中由北平電李宗仁與[[何應欽|何應欽]]，謂[[周恩來|周恩來]]於今日面交「國內和平協定」一件，「內分八條，二十四款」<ref name="寧靜"/>{{rp|182}}。張又上蔣介石一電報，謂到北平後，中國共產黨言論態度，意在逼降，並勸蔣「毅然放下一切」<ref name="寧靜"/>{{rp|182}}。4月20日，[[行政院院長|行政院院長]][[何應欽|何應欽]]對中國共產黨所提八條二十四款，斷然予以拒絕<ref name="寧靜"/>{{rp|186}}。4月23日，和談代表邵力子、張治中等投向中國共產黨<ref name="寧靜"/>{{rp|189}}。張沒有回到南方，而是留在北平，於6月宣佈脫離中國國民黨<ref name="俞大維傳-李">李元平：《[[俞大維|俞大維]]傳》台北：[[臺灣日報|臺灣日報]]社，1992-01-05;{{zh-tw}}，[http://books.google.com.tw/books/about/%E4%BF%9E%E5%A4%A7%E7%B6%AD%E5%82%B3.html?id=PGHemgEACAAJ&redir_esc=y 01]，[http://books.google.com.tw/books/about/%E4%BF%9E%E5%A4%A7%E7%B6%AD%E5%82%B3.html?id=VqxoAAAAIAAJ&redir_esc=y 02]，[http://www.lib.hcu.edu.tw/Webpac2/store.dll/?ID=24348&T=0 增訂版] {{webarchive|url=https://web.archive.org/web/20140221191424/http://www.lib.hcu.edu.tw/Webpac2/store.dll/?ID=24348&T=0 |date=2014-02-21 }}</ref>{{rp|392}}。

== 晚年經歷 ==
[[File:Zhou_Enlai,_Zhang_Zhizhong,_Fu_Zuoyi,_Qu_Wu_in_1962.jpg|thumb]]邀请[[傅作义|傅作义]]、张治中、[[屈武|屈武]]商谈对台湾工作问题。这是会后合影。自左至右：张治中、周恩来、傅作义、屈武]]
1949年[[中華人民共和國|中華人民共和國]]成立後，張治中先後曾任[[西北軍政委員會|西北軍政委員會]]副主席，[[全国人大常委会副委员长|全国人大常委会副委员长]]，[[民革|民革]]中央副主席、和平解放台灣工作委員會主任等職。張在[[文化大革命|文化大革命]]中被抄家。在[[周恩来|周恩来]]的提议下，[[毛泽东|毛泽东]]批准了一个“[[一份应予保护的干部名单|一份应予保护的干部名单]]”，主要包括高级民主人士，宋庆龄被列为第一位。完整名单如下：[[宋庆龄|宋庆龄]]、[[郭沫若|郭沫若]]、[[章士钊|章士钊]]、[[程潜|程潜]]、[[何香凝|何香凝]]、[[傅作义|傅作义]]、'''张治中'''、[[邵力子|邵力子]]、[[蒋光鼐|蒋光鼐]]、[[蔡廷锴|蔡廷锴]]、[[沙千里|沙千里]]、[[张奚若|张奚若]]、[[李宗仁|李宗仁]]。才倖免了直接人身迫害。1969年4月逝于[[北京|北京]]，[[火化|火化]]后葬于[[八宝山革命公墓|八宝山革命公墓]]。

==家庭成員==
張治中的後代有二個兒子和四個女兒:
* 長女：[[張素我|張素我]]曾是[[全國政協常委|全國政協常委]]和[[全國婦女聯合會|全國婦女聯合會]]副主席。已於2011年12月初去世，享年96歲。
* 長子：張一真美國留學，結婚後住[[上海|上海]]後隨岳父和太太遷居[[台灣|台灣]]，直至他母親去世才回到[[北京|北京]]奔喪，後移居美國，於2014年1月在家中安詳過世，時年94歲。
* 次女：張素央曾在美國留學，在中國[[國家體委|國家體委]]工作多年，曾是中國技巧協會主席和國際技巧聯合會的執行委員，曾幾次收到體委和國際技聯的表彰。
* 三女：張素初曾在北京外文出版社工作，後來到美國繼續學習並獲得了[[哈佛大學|哈佛大學]]碩士學位，後在[[美國政府|美國政府]]無家可歸辦公室工作。 2011年1月因病去世，時年85歲，夫婿為資深報人[[趙景倫|趙景倫]]<ref>壹讀(2015-01-29)，《資深報人趙景倫去世》 https://read01.com/J26oAa.html</ref>。
* 次子：張一純曾是北京市政協副秘書長，退休後在北京市港澳台辦公室工作。2015年10月去世，約85歲。
* 四女：張素久是張家最小的，比較活躍。現住美國[[加州|加州]]，組織[[美籍華人|美籍華人]]一些公益事業活動，經常參加[[國務院僑辦|國務院僑辦]]所組織一些活動。

== 評價 ==
張治中在長沙人印象中評價極差<ref>1958年12月号的《[[人民文学|人民文学]]》发表郭沫若的《洪波曲──抗日战争回忆录》，第15章长沙大火的描述：“……放火烧长沙，是张治中、[[潘公展|潘公展]]这一竿子人的大功德。他们是想建立一次奇勋，摹仿库图索夫的火烧莫斯科，来它一个火烧长沙市。”</ref>，因此有人做了一副對聯和匾額譏諷他，流傳在民間很廣：
上聯是「'''治'''績云何，兩大方案一把火（指[[文夕大火|文夕大火]]）」；下聯是「'''中'''心何忍，三顆人頭萬古冤」；橫批則為「'''張'''惶失措」，第一個字連起來正好是張治中。

因张治中脱离国民党加入[[中华人民共和国|中华人民共和国]]阵营，中華民國在戡亂時期称其为“张逆治中”，然而张至死亦未加入[[共产党|共产党]]。但是中國共產黨則認為张治中是少数未与共产党兵戎相见的国民党高级将领之一，人称“和平将军”，为[[解放軍|解放軍]]進駐新疆作出巨大贡献，积极参与了中华人民共和国成立工作，在张治中等人的建议下五星红旗成为[[中华人民共和国国旗|中华人民共和国国旗]]。
