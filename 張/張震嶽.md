'''張震嶽'''（[[阿美語|阿美語]]：{{lang|ami|'''Ayal Komod'''}}，[[音譯|音譯]]：'''海雅·谷慕'''，{{bd|1974年|5月2日|catIdx=Zhang張}}），生於[[台灣|台灣]][[宜蘭縣|宜蘭縣]][[蘇澳鎮_(臺灣)|蘇澳鎮南方澳]]，[[阿美族|阿美族]]人，[[流行音樂|流行]][[搖滾樂|搖滾]][[創作歌手|創作歌手]]，自1993年至今共發行過多張音樂專輯，創作出的經典歌曲有《愛我別走》、《乾妹妹》、《自由》、《分手吧》、《思念是一種病》、《破吉他》等，也曾參與[[電影|電影]]演出。2014年憑《我是海雅谷慕》專輯獲得[[第25屆金曲獎|第25屆金曲獎]]『最佳國語專輯獎』。<ref>[http://www.cna.com.tw/news/firstnews/201406285003-1.aspx 2014金曲獎完整得獎名單]，中央社，2014年6月28日</ref><ref>[http://www.cna.com.tw/news/firstnews/201406285013-1.aspx 金曲最佳國語專輯 張震嶽鍍金]，中央社，2014年6月28日</ref>

== 簡歷 ==

{{Inappropriate tone|章节|time=2013-08-10T09:53:27+00:00}}


*年幼時參加[[教會|教會]]唱詩班的經歷，使張震嶽開始對音樂產生興趣<ref name="基督時報">{{Cite web |url=http://www.christiantimes.cn/news/201211/30/8757.html |title=「坏小子」歌手张震岳：从教会唱诗班走向乐坛 |accessdate=2013-06-14 |work= |publisher=基督時報 |date=2012-11-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20130825002656/http://www.christiantimes.cn/news/201211/30/8757.html |archivedate=2013-08-25 }}</ref>，他在國中時開始學習彈奏[[吉他|吉他]]，並逐漸接觸[[搖滾樂|搖滾樂]]，在參加[[木船民謠歌唱大賽|木船民謠歌唱大賽]]的過程中與[[滾石唱片|滾石唱片]]簽下合約。<ref name="晴報">{{Cite web |url=http://hk.news.yahoo.com/人物-張震嶽-我要唱我的人生-224527038.html |title=人物 — 張震嶽：我要唱我的人生 |accessdate=2013-06-14 |work= |publisher=晴報 |date=2013-05-22 }}</ref>就讀高中的張震嶽，在轉學來到台北之後於1993年12月發行第一張專輯《[[就是喜歡你_(張震嶽專輯)|就是喜歡你]]》，約半年後（1994年6月23日）發行第二張專輯《[[花開了沒有|花開了沒有]]》。

*畢業后張震嶽應徵[[中華民國徵兵規則|入伍]]，隸屬於[[中華民國海軍陸戰隊|海軍陸戰隊]]。張震嶽在軍中[[藝工隊|藝工隊]]結識了學長阿蒙（吳蒙惠），兩人決定在退伍後組成樂團「[[Free_Night|Free Night]]」（Free 9），由阿蒙擔任[[貝斯|貝斯]]手兼任樂團團長，並找到曾和阿蒙合組「High Club」樂團的[[鼓|鼓手]]阿萍（潘瑞萍）以及吉他手阿德（宋人德）。Free Night成立後開始在高雄各餐廳、酒吧表演，一年後回到台北，在寄給唱片公司一卷[[Demo|試聽帶]]後，於1997年10月以「張震嶽 + Free Night」的組合發行了專輯《[[這個下午很無聊|這個下午很無聊]]》。

*之後張震嶽也曾以「DJ Orange」為名在2001年4月和7月發行了兩張[[電子音樂|電子音樂]]專輯，分別為《[[Orange_(張震嶽專輯)|ORANGE]]》和《[[Orange_2|ORANGE2]]》。

*在2003年12月19日張震嶽、Free Night前往[[美國|美國]][[好萊塢|好萊塢]]大型連鎖音樂表演場所「{{link-en|House of Blues|House of Blues}}」演出，約十個月後（2004年10月）他開始進行北美洲的巡迴演唱之旅「Kill Kitty Tour」，前往包括[[舊金山|舊金山]]、[[洛杉磯|洛杉磯]]、[[溫哥華|溫哥華]]、[[費城|費城]]、[[華盛頓|華盛頓]]、[[紐約|紐約]]、[[波士頓|波士頓]]、[[芝加哥|芝加哥]]、[[休士頓|休士頓]]及[[好萊塢|好萊塢]]的十大城市。

*2008年8月，張震嶽和[[羅大佑|羅大佑]]、[[李宗盛|李宗盛]]、[[周華健|周華健]]組成樂團[[縱貫線_(樂團)|縱貫線]]，張震嶽擔任鼓手。縱貫線展開為期一年共54場的世界巡迴演唱會的壯舉後逐漸清晰成形，和大哥們一起同進同出的「世界級天堂路」，讓阿嶽各方面急速地茁壯成長，音樂能量的醖釀更上一層更加穩健。<ref name="鳳凰網">{{Cite web |url=http://ent.ifeng.com/music/hk/detail_2009_09/29/129167_0.shtml |title=张震岳并非纵贯线最初人选 加盟搞定90后歌迷 |accessdate=2013-06-14 |work= |publisher=凤凰网 |date=2009-09-29 }}</ref>

*2013年6月起，張震嶽展開「艷陽天世界巡迴演唱會」，首站香港紅磡，阿嶽和他所精選相惜的音樂同伴「艷陽天樂團aka保力達大樂隊」站上海梅賽德斯奔馳文化中心。

*2013年7月5日，張震嶽從都市回歸到大自然，集聚這些年對生命與環境細細反芻的音樂力量推出「[[我是海雅谷慕|我是海雅谷慕]]」專輯；海雅谷慕（Ayal Komod）是張震嶽的阿美族名字，試重新喚回眾人對大地的尊重、以行動力實踐對環境問題的正視，再次以敏銳的觀察反映著「真實世界」的面貌。

*2015年7月10日，張震嶽與華語饒舌指標性歌手[[MC_HotDog|MC HotDog]]，再加上華語音樂圈嘻哈新人記錄王[[頑童MJ116|頑童MJ116]]，在[[北京|北京]]召開新聞發佈會，宣佈「[[兄弟本色|兄弟本色]]日落黑趴世界巡迴演唱會」正式展開，並宣告「兄弟本色」時代正式降臨。

*2016年5月12日，[[民主進步黨|民主進步黨]][[立法委員|立法委員]][[陳亭妃|陳亭妃]]在[[立法院|立法院]][[質詢|質詢]]時說，本年[[母親節|母親節]]，[[中華民國國防部|中華民國國防部]]在[[facebook|facebook]]貼出[[海軍陸戰隊兩棲偵搜大隊|海軍陸戰隊兩棲偵搜大隊]]學員被操到流鼻血、還要綁著手游泳的照片，可能會影響[[中華民國募兵制|募兵]]成效。同月13日，張震嶽在facebook對其發表，兩棲訓練對不理解的人來說是近乎虐待的殘忍，但他覺得磨練的背後是挑戰超越自我、是驕傲和榮譽、是對國家的奉獻，畢竟戰爭發生時軍人必須犧牲性命保衛人民，但這種價值觀和思想如今已蕩然無存；一個小兵能完成大我、保護人民，這種[[無私|無私]]精神不該淪為政治口水<ref>{{cite news|url=http://www.chinatimes.com/realtimenews/20160513005378-260404|title=陳亭妃嫌蛙人流血照影響募兵 張震嶽：不懂榮譽的人閉嘴|author=施品瑜|date=2016-05-13|accessdate=2016-11-18|publisher=[[中時電子報|中時電子報]]|page=}}</ref>。

*2017年，擔任〈[[中國有嘻哈|中國有嘻哈]]〉節目製作人兼評審。

*2018年，擔任《[[中國新說唱|中國新說唱]]》明星製作人兼評審。
