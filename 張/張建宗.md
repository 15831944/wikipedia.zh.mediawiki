'''張建宗'''{{大紫荊|nocat}}{{GBSHK}}{{JP}}（{{lang-en|Matthew Cheung Kin-chung}}，{{bd|1950年|11月|catIdx=K建宗}}）<ref>〈張常秘不敵歲月無得留低〉。《明報》。2007年1月8日。</ref>，現任[[香港特別行政區|香港特別行政區]][[政務司司長|政務司司長]]，並曾任[[勞工及福利局局長|勞工及福利局局長]]。他擔任勞工及福利局局長十年，之前亦曾任[[勞工處處長|勞工處處長]]，是[[香港回歸|香港回歸]]以來處理勞工權益事務最久的政府官員及局長。

== 早年經歷 ==
張建宗祖籍[[廣東|廣東]][[惠州|惠州]]<ref name="tkp"/>，曾就讀香港[[高主教書院|高主教書院]]以及[[天主教新民書院|天主教新民書院]]，後於1972年在[[香港大學|香港大學]]歷史系畢業，於同年7月加入政府新聞主任職系，其後於1979年9月轉職[[政務職系|政務職系]]。他先後於多個[[香港政府部門|香港政府部門]]服務，包括[[財政科|財政科]]、[[民政署|民政署]]、[[政務總署|政務總署]]、[[總督府|總督府]]、[[工業署|工業署]]、[[貿易署|貿易署]]、[[財政司辦公室|財政司辦公室]]、[[司法機構|司法機構]]及[[中央政策組|中央政策組]]。<ref>[http://hk.on.cc/hk/bkn/cnt/news/20170116/bkn-20170116142857875-0116_00822_001.html 張建宗新聞官出身 官場打滾逾40年]，《東方日報》 2017年1月16日</ref>他於2004年9月晉升為首長級甲一級政務官。

1996年3月至1999年1月，擔任副教育統籌司（今[[教育統籌局|教育統籌局]]副局長），於1999年1月至2000年6月擔任勞工處處長，於2000年6月至2002年6月擔任[[教育署署長|教育署署長]]，於2002年7月至2007年3月擔任[[經濟發展及勞工局|經濟發展及勞工局]][[常任秘書長|常任秘書長]]（勞工）。2007年3月，張建宗達至退休年齡，工會中人希望他留任，[[工聯會|工聯會]][[香港立法會議員列表|立法會議員]][[陳婉嫻|陳婉嫻]]更在答問大會上質詢時任[[香港行政長官|香港行政長官]][[曾蔭權|曾蔭權]]，希望留任張建宗，惟不得要領。

== 勞工及福利局局長 ==
[[File:Matthew_Cheung_Kin-chung.JPG|thumb]]
2007年6月，張建宗獲曾蔭權提名為勞工及福利局局長。

2012年6月，張建宗獲得新任行政長官[[梁振英|梁振英]]邀請留任。

任內，他處理過[[2007年香港紮鐵工人大罷工|2007年香港紮鐵工人大罷工]]、[[僱員再培訓徵款|僱員再培訓徵款（外傭稅）]]及[[2013年葵青貨櫃碼頭工潮|2013年葵青貨櫃碼頭工潮]]等事件。他執行[[長者生活津貼|長者生活津貼]]的長者退休保障政策備受爭議。

== 政務司司長 ==
2017年1月16日，因應[[林鄭月娥|林鄭月娥]]辭任[[政務司司長|政務司司長]]參加[[2017年香港特別行政區行政長官選舉|2017年行政長官選舉]]，張建宗得到時任行政長官[[梁振英|梁振英]]提名，並獲[[中華人民共和國國務院|國務院]]任命為政務司司長。
