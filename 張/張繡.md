'''張繡'''（{{bd||?|207年|catIdx=Z張}}），是[[東漢|東漢]]末年的武將和割據軍閥之一，[[武威郡|武威郡]][[袓厲縣|袓厲縣]]（今[[甘肅省|甘肅省]][[白銀市|白銀市]][[靖遠縣|靖遠縣]]西南）人，[[董卓|董卓]]手下將領[[張濟_(東漢)|張濟]]的族子。

== 生平 ==
185年，[[韓遂|韓遂]]與[[邊章|邊章]]在西涼起兵造反，袓厲縣長[[劉雋_(東漢)|劉雋]]與[[麹胜|麹胜]]作戰被殺。當時張繡為縣吏，尋找機會殺了麴勝，為劉雋報仇，郡内的百姓都認爲他很講道義。此後跟隨族伯[[張濟|張濟]]為[[董卓|董卓]]效力，建立軍功，遷為建忠將軍，封'''宣威侯'''。董卓死後，張濟於197年因為進攻荆州牧[[劉表|劉表]]鎮守的[[南陽郡|南陽]]，中流矢戰死，不過張繡卻得到劉表的接納，接管了張濟的軍隊，佔據[[宛城|宛城]]，與[[劉表|劉表]]聯合。

此後，[[曹操|曹操]]率軍[[宛城之戰|攻打宛城]]，張繡舉眾投降。由於曹操好色，染指於張繡的伯母——張濟之妻，將張濟之妻納為妾室，張繡恨之。曹操聽聞張繡心懷不滿，也密謀誅除張繡。張繡有一勇將[[胡車兒|胡車兒]]，曹操愛其驍健，於是賞賜金錢給胡車兒，張繡懷疑這是曹操收買叛徒的手段，遂決定先下手為強，襲擊曹操軍并擊斃曹操長子[[曹昂|曹昂]]、侄[[曹安民|曹安民]]与大將[[典韋|典韋]]，將曹操暫時驅逐出宛城地區，並於此後與曹操持續對抗。（註：[[丘遲|丘遲]][[s:與陳伯之書|與陳伯之書]]：張繡剚刃於愛子…，[[曹操|魏君]]視之若舊。）<ref>《[[三國志|三國志]]·魏志·武帝紀》載：「[[建安|建安]]二年，公（曹操）到[[宛城|宛]]。張繡降，既而悔之，復反。公與戰，軍敗，為流矢所中。長子[[曹昂|昂]]、弟子[[曹安民|安民]]遇害。」</ref>

不久[[袁紹|袁紹]]與曹操戰於[[官渡|官渡]]，袁紹派人到張繡處，要與張繡結盟，謀士[[賈詡|賈詡]]嚴詞反對，以三計建議張繡降曹，并将女兒嫁與曹操之子[[曹均|曹均]]以結為[[親家|親家]]。200年，[[官渡之戰|官渡之戰]]後，曹操封賞部下，張繡力戰有功，任破羌將軍。此後他跟隨曹操在[[南皮|南皮]]擊敗[[袁譚|袁譚]]，再授予食邑凡2000戶。當時因天下戶口減耗，諸將未有滿千戶者，所以張繡的封戶算是特别多的。

《[[三國志|三國志]]》記載，207年，張繡隨曹操出征[[烏丸|烏丸]]，在行軍至[[柳城|柳城]]途中病死，[[諡|諡]]'''定侯'''。不过[[三國志注|裴注]]引《[[魏略|魏略]]》則記載了另一說法，張繡被[[曹丕|曹丕]]責備以前殺死了哥哥[[曹昂|曹昂]]的事，认为曹家並未忘記昔日恩怨，心不自安，於是[[自殺|自殺]]。<ref>《[[魏略|魏略]]》曰：[[曹丕|五官將]]數因請會，發怒曰：「君殺吾兄，何忍持面視人邪！」繡心不自安，乃自殺。</ref>

張繡兒子[[張泉|張泉]]襲爵，封[[長樂|長樂]][[衛尉|衛尉]]。可是後來卻捲進了[[魏諷|魏諷]]的謀反事件，被視為同黨誅戮。

== 民間藝術 ==
在很多後代評書評話中，張繡的形象與《三國志》甚至《[[三國演義|三國演義]]》都有很大的差别。他被稱為「北地槍王」、「宛城侯」，成為槍法名將，不過最終在[[長坂坡|長坂坡]]死在同樣使長槍的[[趙雲|趙雲]]手下。<ref>{{cite book|title=長坂坡|author=[[張國良|張國良]]|year=1984|publisher=上海文藝出版社}}</ref>

== 评价 ==
*[[陈寿|陈寿]]：“燕、绣、鲁舍群盗，列功臣，去危亡，保宗祀，则于彼为愈焉。”（《三国志·魏书·二公孙陶四张传第八》）
*[[贾诩|贾诩]]：“将军（指张绣）虽'''善用兵'''，非曹公敌也。军虽新退，曹公必自断后；追兵虽精，将既不敌，彼士亦锐，故知必败。曹公攻将军无失策，力未尽而退，必国内有故；已破将军，必轻军速进，纵留诸将断后，'''诸将虽勇，亦非将军敌'''，故虽用败兵而战必胜也。”（《三国志·贾诩传》）
