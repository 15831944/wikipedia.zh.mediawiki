'''張果'''，（596年-735年）[[唐代|唐代]]一位精通[[服气|服气]]，修炼[[内丹|内丹]]的[[道士|道士]]，中唐以后，逐渐被神化，后来成为民间神话中的[[八仙|八仙]]之一，被称为'''张果老'''，据《[[明皇杂录|明皇杂录]]》記载，張果老是婚生子。他母亲生他時85岁，而父亲已經365岁了，父亲活到了407岁，而母亲活到了157岁。

== 生平 ==
据记载，張果老是[[唐朝|唐朝]]（618年-907年）人；他的故事最早见于《[[明皇杂录|明皇杂录]]》卷下：“张果者，隐于恒州条山。常往来汾晋间，时人传有长年秘术，……唐太宗、高宗屡征之不起，则天召之出山，佯死于妒女庙前。……后有人于恒州山中复见之。” ；生平又见于《[[大唐新语|大唐新语]]》卷10，《[[新唐书|新唐书]]·张果传》乃至於《[[资治通鉴|资治通鉴]]》卷214开元二十二年載“方士张果自言有神仙术……”，基本均取材于此。

[[武则天|武则天]]曾谴使欲召见之，即佯死；后人又见到他居于[[恒州|恒州]]山中，常倒骑白驴，日行数万里，休息时候又将毛驴折叠藏于巾箱；他曾被[[唐玄宗|唐玄宗]]召见至京师，演出种种法术，授以[[光禄大夫|光禄大夫]]，赐号'''通玄先生'''。<ref>《旧唐书》卷8《玄宗本纪上》本年二月：“征恒州张果先生，授银青光禄大夫，号曰通玄先生。”</ref>

張果声称自己非常长寿，时间大约在[[唐高宗|唐高宗]]到唐玄宗时期<ref>《[[新唐書|新唐書]]‧方技傳》：“張果者，晦鄉里世系以自神，隱中條山，往來汾、晉間，世傳數百歲人。”</ref>。《[[通鉴|通鉴]]》更記載张果自稱[[尧|尧]]时为[[侍中|侍中]]，到唐玄宗時已三千多岁。<ref>《[[资治通鉴|资治通鉴]]》卷214开元二十二年載“方士张果自言有神仙术，诳人云尧时为侍中，唐玄宗已三千多岁；多往来恒山中，则天以来，屡征不至。”</ref>

== 主要思想 ==
張果内外丹兼修，但以内丹修炼为主。他分[[内丹|内丹]]为三品，以[[九转大还丹|九转大还丹]]为上品。并且总结出九转大还丹有[[九要|九要]]，依此而行，可以得道。此说与[[司马承祯|司马承祯]]的[[坐忘|坐忘]]相通。此外他提出[[性命双修|性命双修]]，被认为是宋元内丹学的先驱。

== 神话形象 ==
[[File:Audience_by_Emperor_Tang_Xuanzong.jpg|left]] 他的典型形象是倒骑一头白驴。据说这头白驴，日行千里，不用吃也不用喝。休息时把它折叠起来，要乘坐时对它吹口气，又变成了驴子。


关于张果老的出生时间没有明确记载，如果硬要说有记载，也都是些传说，明显不可信，但张果老作为历史上确实存在的人物不可能活到数百上千岁，更不可能与天同寿，那么张果老到底出生在什么时间呢？根据《唐玄宗敕封仙人张果记》记载：“仙翁张姓果名，[[邢州|邢州]][[广宗|广宗]]人也。”邢州广宗，在今天的河北广宗、威县一带，《[[史记|史记]]》记载其为沙丘地，[[西汉|西汉]]后期建广宗国始用[[广宗|广宗]]之名，后历经[[广宗国|广宗国]]、[[广宗郡|广宗郡]]、[[广宗县|广宗县]]，到601年为避隋太子杨广之“广”字之讳，乃更名为宗城县。由此可见，张果老应当在隋炀帝杨广正式被立为太子之前出生才会在自己仙逝后自己的弟子或是其他守观之人用自己出生时的故乡之名来记载仙人，所以张果老当在杨广被立为太子之前（广宗更名为[[宗城|宗城]]之前）出生，即最迟在公元601年出生。能明确记着自己是[[邢州|邢州]][[广宗|广宗]]人并且对广宗有着深厚的感情（通过其去世后碑文记载用出生时的故乡之名来记载可以看出），当为广宗更名时已经五六岁或是更大，如此推断张果老当在公元六世纪末出生，而邢州之名是在开皇十六年（公元596年）由[[襄国郡|襄国郡]]改名[[邢州|邢州]]的，在公元607年又复改[[襄国郡|襄国郡]]，由此得知，张果老出生年限当在公元596年至公元601年之间。

而张果老去世的时间则是一个比较明确的话题，大多人都认为是在[[开元|开元]]年间去世的，而《唐玄宗敕封仙人张果记》文中也有记载到“开元二十三年......玄宗......遣通士舍人裴晤驰驲迎之，肩舆入宫，果试仙术”。可见开元二十三年张果老还在世，但细读该碑文会发现：该碑文是[[开元|开元]]二十三年篆刻的，而记载之事也发生在开元二十三年，难道张果老还在世就已经被石碑列文记载了？由此推断，张果老仙逝之时间当在玄宗“遣通士舍人裴晤驰驲迎之”之后碑文记载之前，即当年就仙逝，时间是[[开元|开元]]二十三年（公元735年）。

这样一来我们大概可以推断，张果老的岁数，从公元596年至公元601年之间到公元735年张果老大概活了135岁到139岁）之间，很巧在《顺德府志》就记载的张果老为139岁左右，所以我们大抵可以得出张果老的大约岁数。
