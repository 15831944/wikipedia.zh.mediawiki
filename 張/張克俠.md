'''張克俠'''（{{bd|1900年|10月7日|1984年|7月7日|catIdx=Z}}），原名'''張樹棠'''，[[河北|河北]][[獻縣|獻縣]]侯陵屯村人，[[西北军|西北军]]将领，[[中国共产党|中国共产党]]党员，[[中华人民共和国林业部|中华人民共和国林业部]]副部长。

== 生平 ==
青少年时期，张克侠在北京汇文小学、[[北京汇文中学|北京汇文中学]]学习。民国4年（1915年），为抗议[[袁世凯|袁世凯]]接受“[[二十一条|二十一条]]”，张克侠投笔从戎，考入北京清河陆军军官预备学校<ref name=shht>{{Cite web |url =http://www.shtong.gov.cn/newsite/node2/node2245/node75342/node75355/node75363/node75367/userobject1ai90386.html  |title = 上海军事志 第九编人物 一、人物传 张克侠 |author =  |date =  |publisher = 上海通 |language =  |accessdate = 2018-03-19 |quote =  }}</ref>。

1918年，张克侠与[[通州区_(北京市)|通县]]姑娘李德璞（后改名李英）结婚。李德璞的姐姐[[李德全|李德全]]后与[[冯玉祥|冯玉祥]]结婚（冯玉祥前妻刘德贞1923年病逝，1924年冯玉祥续娶李德全），张克侠与[[西北军|西北军]]领袖冯玉祥成为[[连襟|连襟]]<ref name=shijiu>{{Cite web |url = http://news.163.com/15/0526/14/AQI2GC4S00014Q4P.html |title = 张克侠：潜伏敌营19载 |author =  |date = 2015-05-26 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。

1922年，张克侠自[[保定陸軍軍官學校|保定陸軍軍官學校]]畢業，後加入冯玉祥的西北军。1924年1月抵达[[广州|广州]]，先后在[[孙中山|孙中山]]大本营军政部任少校科员兼陆军讲武学校教育副官及队长。其间接触了许多[[中国共产党|中国共产党]]党员，受到[[共产主义|共产主义]]思想启蒙，并致信冯玉祥宣传革命道理。1925年，[[国民革命军|国民革命军]]组建，以陆军讲武学校毕业学员为骨干编成两个营，张克侠任第一营营长<ref name=shht/><ref name=shishi/>。1926年9月，冯玉祥的西北军[[五原誓师|五原誓师]]响应[[北伐战争|北伐]]，张克侠返回西北军任学兵团团副<ref name=shht/><ref name=shijiu/>。

1927年春，在冯玉祥夫人李德全支持下，张克侠赴[[苏联|苏联]][[莫斯科中山大学|莫斯科中山大学]]学习。1928年秋毕业，1929年回国<ref name=shht/><ref name=shishi/>。在校期间，张克侠向中共党组织递交过入党申请书，但组织上还没批准他入党，[[大革命|大革命]]便失败，张克侠等人被校方送回国内。回国前，党组织派人与张克侠谈话称，目前中国革命正受严重威胁，[[白色恐怖|白色恐怖]]下大批中共党员和革命群众被杀，张克侠能在此刻提出入党申请实属难能可贵，还表示尽管张克侠未能在[[莫斯科|莫斯科]]入党，但和张克侠一道回国的张振亚（[[张存实|张存实]]）会向国内的中共党组织汇报张克侠的情况，希望张克侠回国后继续争取入党<ref name=shijiu/>。

1929年7月，张克侠在[[上海|上海]]经张存实、[[李翔梧|李翔梧]]介绍加入中国共产党，在[[周恩来|周恩来]]直接领导下工作<ref name=shht/><ref>{{cite journal |author=陆昇 |year=2012 |title=潜伏：红色特工张克侠 |journal=同舟共进 |issue=7 |pages=28-32 |url=http://dangshi.people.com.cn/n/2012/0716/c85039-18523930.html |accessdate= }}</ref>。后来遵照中共党组织的指示返回西北军，任[[张自忠|张自忠]]师的参谋长。1931年[[九一八事变|九一八事变]]后，考入[[南京|南京]][[國防大學_(中華民國)|陆军大学]]。1933年5月，冯玉祥与中国共产党合作，在[[张家口|张家口]]组织[[察哈尔民众抗日同盟军|察绥民众抗日同盟军]]，张克侠利用陆军大学放暑假的机会来到张家口，任同盟军高级参谋、干部学校校长。后来抗日同盟军在[[日本|日本]]和[[蒋介石|蒋介石]]的内外夹击下失败，张克侠回到南京陆军大学<ref name=shht/><ref name=shishi/>。

1935年底，张克侠从陆军大学毕业回[[国民革命军第二十九军|国民革命军第二十九军]]三十八师任师长。1936年，调任二十九军副参谋长兼三十八师参谋长。其间他与[[王世英|王世英]]、[[南汉宸|南汉宸]]、[[彭雪枫|彭雪枫]]、[[杨秀峰|杨秀峰]]、[[刘清扬|刘清扬]]、[[萧明|萧明]]等中共党员联系或合作，并将中国共产党提出的[[抗日民族统一战线|抗日民族统一战线]]作为他在二十九军的工作原则，进行抗日准备<ref name=shijiu/><ref name=shht/>。张克侠按照[[中共中央|中共中央]]要二十九军不要妥协、积极对日作战、发动群众、支援抗战的指示精神，提出了一个集结兵力、主动攻击的作战方案，计划在[[日军|日军]]增援前，以二十九军优势兵力一举消灭驻[[华北地区|华北地区]]的2万日军。张克侠将该作战计划交萧明报送中共领导，[[刘少奇|刘少奇]]亲自批示同意了该作战计划。但后来该计划未能实现<ref name=shijiu/>。

1937年7月[[七七事变|七七事变]]爆发后，[[中共中央|中共中央]]发出《为日军进攻[[卢沟桥|卢沟桥]]通电》，要求[[宋哲元|宋哲元]]动员全部二十九军开赴前线应敌。但当时宋哲元正在[[山东|山东]][[乐陵|乐陵]]家乡休养，二十九军各部仍旧分驻各地，形势急迫。[[邓哲熙|邓哲熙]]、[[赵登禹|赵登禹]]、张克侠乘飞机紧急接宋哲元回[[北平|北平]]主持大计。张克侠报告[[何应钦|何应钦]]请宋哲元回北平备战，但宋哲元犹豫不决。张克侠劝说宋哲元称，现已到民族存亡关头，不战将成民族罪人，战而不胜虽败犹荣，并表示目前我军占优势，可在日军增援前抓住战机获胜。张克侠还连夜拟出一份作战计划，与上次给中共党组织的计划大致相同。次日张克侠将该计划送交宋哲元<ref name=shijiu/>。

宋哲元回到北平后，同[[日本|日本]]代表谈判交涉，同时暂停军事准备。但日军趁机不断向华北增兵。张克侠多次建议宋哲元把[[南苑|南苑]]的二十九军军部及部队移往便于指挥的地方。然而宋哲元担心引起日本怀疑，不同意调动军队和转移军部。7月27日傍晚，日军逼近南苑即将发动进攻，宋哲元匆忙下令二十九军军部移往北平城内的[[中南海|中南海]][[怀仁堂|怀仁堂]]，命赵登禹到南苑指挥部队。当晚，张克侠向赵登禹介绍了南苑部队情况，随后张克侠撤至北平城内。二十九军在南苑因准备不足、仓促应战而损失惨重，[[佟麟阁|佟麟阁]]、赵登禹阵亡<ref name=shijiu/>。日军占领北平前夕，张克侠得知日军即将不战入城的情报后，及时通知[[刘清扬|刘清扬]]、[[張友漁_(法學家)|张友渔]]、[[杨秀峰|杨秀峰]]等人，使一批抗日骨干安全转移<ref name=shht/>。

1937年7月29日，汉奸[[江朝宗|江朝宗]]主持的“北平地方维持会”成立。[[张自忠|张自忠]]躲进[[东交民巷|东交民巷]]使馆区，后逃到[[天津|天津]]乘船赴山东。张克侠先在北平隐蔽，其间得知[[八一三事变|八一三淞沪抗战]]消息后，盼望南下到前线杀敌。张克侠托在天津经商的表弟房兆梁想办法，房兆梁寄来许多做生意的证明，8月21日，张克侠和李连山乘火车赴天津，在[[天津站|天津站]]正逢日方搜查，房兆梁将张克侠送到[[天津英租界|天津英租界]]一位亲戚家里。数日后，他们乘[[英国|英国]]轮船到[[烟台|烟台]]，转赴[[济南|济南]]<ref name=shijiu/>。

此后，张克侠辗转寻找到冯玉祥，历任冯玉祥的[[抗日战争第六战区|第六战区]]司令部高级参谋、副参谋长等职。1938年随冯玉祥到[[武汉|武汉]]。在武汉期间，与八路军驻汉口办事处取得联系，听取了[[叶剑英|叶剑英]]等领导的指示，并按照中共党组织的指示在国民革命军中开展抗日工作。1938年，张克侠任国民革命军第五十九军参谋长，后任第三十三集团军参谋长、中将副总司令等职，转战[[山东|山东]]、[[河南|河南]]、[[湖北|湖北]]。其间创办了自忠中学，宣传抗日思想<ref name=shht/><ref name=shijiu/><ref name=shishi/>。在[[台儿庄战役|台儿庄战役]]中，张克侠协助张自忠指挥五十九军痛击日军[[板垣师团|板垣师团]]，声名鹊起。[[抗日战争|抗日战争]]胜利后，[[蒋介石|蒋介石]]颁发一批佩剑，称“[[中正剑|中正剑]]”，张克侠获赠一柄，故之后人称张克侠为“佩剑将军”<ref name=shijiu/>。

1945年抗日战争胜利后，国民政府在蒋介石领导下作[[第二次国共内战|内战]]准备，第三十三集团军改为第三绥靖区，张克侠任中将副司令长官。张克侠在第三十三集团军上层人士中开展反内战、反独裁活动，并在策划[[郝鹏举|郝鹏举]]部起义前，亲自带郝鹏举面见[[陈毅|陈毅]]司令员。1946年夏，在蒋介石催逼下，冯王祥被迫出国“考察”，张克侠借机到南京送行，经联系后受到周恩来接见，张克侠表示“……只要党下令，我可以保证随时起义”<ref name=shht/>。在第三绥靖区中将副司令长官任内，他还获取了[[徐州剿总|徐州剿总]]各类机密情报，派人密送[[中国人民解放军华东野战军|中国人民解放军华东野战军]]司令部<ref name=shijiu/>。

1948年10月中旬，在[[淮海战役|淮海战役]]（国军称徐蚌会战）中，中国人民解放军华东野战军派代表到[[徐州|徐州]]，向张克侠传达陈毅等首长的指示，定于11月8日解放军发起进攻时，争取[[中华民国国军|国军]]两个军起义。但就在起义前夕，第三绥靖区司令长官[[冯治安|冯治安]]对张克侠产生怀疑，指令张克侠住在徐州，禁止他去第三绥靖区总部所在地[[贾汪镇|贾汪]]。但张克侠摆脱了参谋长、[[军统|军统]]特务[[陈继淹|陈继淹]]的监视，及时赶到贾汪，与[[何基沣|何基沣]]一同在1948年11月8日率国军五十九军两个师、七十七军一个半师共23000多名官兵，在贾汪、[[台儿庄|台儿庄]]一带防地发动[[贾汪起义|贾汪起义]]，从而开放了台儿庄运河上的通道，使徐州东北门敞开，解放军得以直取徐州，并切断[[黄百韬|黄百韬]]部退路，对取得[[淮海战役|淮海战役]]的胜利发挥了重要作用，受到[[毛泽东|毛泽东]]主席、[[朱德|朱德]]总司令嘉勉<ref name=shht/>。

起义部队后与中国人民解放军渤海纵队合并，组成中国人民解放军第三十三军，张克侠任军长，参加了[[渡江战役|渡江战役]]、郎广战役、[[上海战役|上海战役]]。在上海战役中，张克侠率部攻打[[宝山|宝山]]、[[吴淞|吴淞]]。攻克上海后，张克侠兼任中国人民解放军淞沪警备司令部参谋长，任内在上海的军事接管、警备区域划分、收集国民党军散兵，剿匪镇反、解放沿海岛屿、粉碎国民党对上海的封锁与空袭，恢复上海社会治安，保障人民生活，发展生产等方面做了许多工作<ref name=shht/><ref name=shishi/>。

1950年3月7日，[[中共中央组织部|中共中央组织部]]作出《关于张克侠党籍问题的决定》，决定称：“我们认为张克侠同志虽然在国民党军队工作，但1929年入党以来，一贯与党保持联系，设法为党工作，并有成绩，故其全部党籍，应予承认。”<ref name=shijiu/>

1950年6月，张克侠转业到地方。历任[[华东军政委员会|华东军政委员会]]委员、农村部部长，[[华东行政委员会|华东行政委员会]]森林工业管理局局长。1954年调任[[中华人民共和国林业部|中华人民共和国林业部]]副部长、党组成员，后兼任[[中国林业科学研究院|中国林业科学研究院]]院长、分党组书记等职，在开展大规模植树造林，有计划采伐和节约木材，建立健全林业科研体系和发展科研事业等方面作出了贡献。1955年获授一级[[解放勋章|解放勋章]]。他是[[第四届全国人大代表|第四届全国人大代表]]，第五届全国政协常务委员<ref name=shht/><ref name=shishi/>。

“[[文化大革命|文化大革命]]”期间，张克侠遭到迫害，身心受严重摧残，其亲属子女也受牵连。文革初期，在全国全面夺权的风潮中，张克侠坚持没有中央指示绝不签字交权。[[粉碎四人帮|粉碎四人帮]]后，张克侠拥护[[中共十一届三中全会|中共十一届三中全会]]以来的路线、方针、政策，尽管身体虚弱仍然积极工作，撰写革命回忆录。他多次嘱咐家属，并留有遗嘱：身后不开追悼会，不留骨灰<ref name=shht/><ref name=shishi/>。

1984年7月7日，张克侠在北京病逝，享年84岁。7月18日，在[[北京八宝山革命公墓|北京八宝山革命公墓]]礼堂举行了遗体告别仪式<ref name=shht/><ref name=shishi>中国共产党优秀党员忠诚的共产主义战士张克侠逝世，人民日报1984年7月19日</ref>。
