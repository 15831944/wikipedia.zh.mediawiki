'''張彥博'''（{{lang|en|'''Bob Cheung Yin Pok'''}}，{{bd|1980年|10月5日||}}），[[香港|香港]]男歌手及演员。前樂隊「赤子」成員<ref name="sh">[http://roll.sohu.com/20130702/n380505293.shtml 快乐男声张彦博个人资料详细介绍]</ref>，現為[[無綫電視藝員列表|無綫電視基本藝人合約]]<ref>[http://artiste.tvb.com/main/artistedetail/cheungyinpok 張彥博 Cheung Yin Pok - TVB藝人資料 - tvb.com]</ref>兼[[邵氏兄弟|邵氏兄弟]]藝員<ref>[http://www.shawbrothers.hk/artist/pkUEhNXlMOMMkigGmeSo4 張彥博 Cheung Yin Pok - 邵氏兄弟 Shaw Brothers]</ref>。

== 經歷 ==
張彥博曾就讀[[北角衛理小學|北角衛理小學]]、[[衛理中學|衛理中學]]和[[北角協同中學|北角協同中學]]，後來畢業於[[香港專業教育學院|香港專業教育學院]]。其父親自1981年起在衛理中學任職體育科教師至今。他涉足[[娛樂圈|娛樂圈]]前曾當過電腦公司職員<ref>[http://www.orientalsunday.hk/%e6%9c%80%e6%96%b0%e5%a8%9b%e8%81%9e/%e4%b8%89%e9%a4%90%e9%a3%9f%e5%8d%b3%e9%a3%9f%e9%ba%b5-%e5%bc%b5%e5%bd%a5%e5%8d%9a%e5%8d%81%e5%b9%b4%e8%be%9b%e9%85%b8%e5%8f%b2/ 三餐食即食麵 張彥博十年辛酸史]，東方新地，2016年7月26日</ref>、[[茶餐廳|茶餐廳]]侍應、救生員及從事派傳單等工作<ref>[http://orientaldaily.on.cc/cnt/entertainment/20170104/00282_025.html 張彥博捱足16年窮到燶 拍拖驚累人]，[[東方日報_(香港)|東方日報]]，2017年1月4日</ref>。

2005年，張彥博與[[馮嘉輝|馮嘉輝]]、[[伍德耀|伍德耀]]組成[[男子音樂組合|男子音樂組合]][[赤子_(樂隊)|赤子樂隊]]（Akago）<ref>[http://hk.apple.nextmedia.com/entertainment/art/20050715/5052065 三小伙子闖樂壇賣青春活力 赤子玩水見紅好兆頭。]，[[蘋果日報_(香港)|蘋果日報]]，2005年7月15日</ref>，但只維持了四年便因約滿而宣佈解散。及後與另一歌唱組合[[Boy'z|Boy'z]]成員[[張致恆|張致恆]]合作推動基督教音樂事工<ref>[http://www.grateful-heart.org/WonderfulGrace/content.php?article=2349&page=2 張致恆 張彥博 媽媽的大細路（第29期）]，《奇妙恩典》網上版，2010年</ref>，並創作和演唱過些基督教歌曲，且跟多位相同信仰的藝人合作，包括[[王祖藍|王祖藍]]、[[泳兒|泳兒]]、[[鍾舒漫|鍾舒漫]]、[[馮允謙|馮允謙]]等，兩人亦同時在2011年[[受洗|受洗]]成為基督徒<ref>[http://www.gospeltimes.cn/news/16466/%E8%89%BA%E4%BA%BA%E5%BC%A0%E8%87%B4%E6%81%92%E4%B8%8E%E5%BC%A0%E5%BD%A6%E5%8D%9A%E5%8F%97%E6%B4%97%E4%BF%A1%E4%B8%BB%20%20%E6%9C%89%E6%84%8F%E6%84%BF%E7%BB%A7%E7%BB%AD%E6%9C%8D%E4%BE%8D%E6%95%99%E4%BC%9A 艺人张致恒与张彦博受洗信主 有意愿继续服侍教会。]，福音時報，2011年4月1日</ref><ref>[http://the-sun.on.cc/cnt/entertainment/20110328/00470_040.html 張致恒悔改：傷過好多女。]，[[太陽報_(香港)|太陽報]]，2011年3月28日</ref>。

2012年，張彥博參加[[上海|上海]][[東方衛視|東方衛視]]音樂文化交流節目《[[聲動亞洲|聲動亞洲]]》<ref>[http://www.hunantv.com/c/20120801/1149255081.html 李克勤声动唱《红日》 与港版“小猪”再现经典]，金鷹網，2012年8月1日</ref>，惟只能晉級至第二輪比賽便遭淘汰。2013年，他再參與《[[2013快樂男聲|2013快樂男聲]]》，也順利進入[[長沙|長沙]]唱區10強<ref>{{cite web|url=http://www.mgtv.com/c/20130702/1919474900.html | title=快男多鬼才火星哥走红 港仔張彦博獲王祖藍力挺 |publisher=金鷹網 |date=2013年7月2日}}</ref>，然而當進行全國比賽時，卻不幸在第一關被淘汰。雖然兩次比賽都未能為他帶來獎項，不過由於連續兩年參加內地歌唱比賽，故能累積一定人氣，亦因此獲取了許多中國內地演出機會如[[舞台劇|舞台劇]]及[[電影|電影]]。

2013年，張彥博加入[[無線電視藝員訓練班|無綫電視第27期藝員訓練班]]，2014年8月畢業後正式投入演員工作，2016年，他憑無綫電視劇《[[一屋老友記|一屋老友記]]》內廢青「寶怡（細寶、David）」一角，形象深入民心而備受注目<ref>{{cite web|url=http://hk.apple.nextmedia.com/enews/realtime/20160711/55342686 |title=張彥博35歲娃娃臉 「廢青」有出頭天 |publisher=蘋果日報 即時娛樂 |date=2016年7月11日}}</ref><ref>{{cite web|url=http://hk.on.cc/hk/bkn/cnt/entertainment/20160707/bkn-20160707155407748-0707_00862_001.html |title=娛樂圈如長跑賽　35歲張彥博唔怕起步遲 |publisher=[[東方日報_(香港)|東網]] |date=2016年7月7日}}</ref>。另外，他在劇中與[[劉佩玥|劉佩玥]]所飾演的「呂星晨」，合唱插曲《你認真嗎？》也備受讚賞，獲得正評<ref>{{cite web|url=http://ent.fanpiece.com/purestalker/%e6%98%9f%e6%99%a8%e7%b4%b0%e5%af%b6%e6%83%85%e4%be%b6%e6%aa%94%e4%b8%80%e9%b3%b4%e9%a9%9a%e4%ba%ba-%e5%90%84%e5%9c%8b%e8%a7%80%e7%9c%be%e5%8b%81buy-c1238476.html |title=星晨細寶情侶檔一鳴驚人，各國觀眾勁 buy |publisher=FanPiece 大娛樂家 |date=2016年8月1日}}</ref>。同年，他首度獲提名《[[TVB_馬來西亞星光薈萃頒獎典禮2016|TVB 馬來西亞星光薈萃頒獎典禮2016]]》「最喜愛 TVB 飛躍進步男藝員」、「2016年紅爆網絡新鮮人大獎」，以及《[[萬千星輝頒獎典禮2016|萬千星輝頒獎典禮2016]]》「飛躍進步男藝員」。

2018年11月，張彥博接受報章訪問時透露，在拍攝電視劇《[[跳躍生命線|跳躍生命線]]》期間不慎弄傷頸椎，須休息大約四個月和利用大量物理治療才得以康復<ref>{{Cite web |url=https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/254262/%E8%B7%B3%E8%BA%8D%E7%94%9F%E5%91%BD%E7%B7%9A-%E5%BC%B5%E5%BD%A5%E5%8D%9A%E6%AE%89%E8%81%B7%E6%88%B2%E5%90%8A%E5%A8%81%E4%B9%9F-%E9%A0%B8%E6%A4%8E%E9%87%8D%E5%89%B5-%E5%9B%9B%E5%80%8B%E6%9C%88%E5%85%88%E5%A5%BD%E8%BF%94 |title=【跳躍生命線】張彥博殉職戲吊威也　頸椎重創：四個月先好返 |author=鄺鈺瑩 |date=2018年11月2日 |publisher=[[香港01|香港01]] }}</ref>，又憑該劇角色「呂劍雄（雄姐）」受到觀眾注視<ref>{{Cite web |url=https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/253502/%E8%B7%B3%E8%BA%8D%E7%94%9F%E5%91%BD%E7%B7%9A-%E5%B0%88%E8%A8%AA-%E5%BC%B5%E5%BD%A5%E5%8D%9A%E7%94%B1%E7%B4%B0%E5%AF%B6%E5%88%B0%E9%9B%84%E5%A7%90%E5%AD%B8%E6%9C%83%E9%96%8B%E7%AB%85-%E5%94%94%E6%83%B3%E5%86%8D%E9%99%90%E5%88%B6%E4%BD%8F |title=【跳躍生命線．專訪】張彥博由細寶到雄姐學會開竅：唔想再限制住 |author=鄺鈺瑩 |date=2018年11月1日 |publisher=香港01 }}</ref><ref>{{Cite web |url=https://www.am730.com.hk/news/%E5%A8%9B%E6%A8%82/%E3%80%90%E9%9B%84%E5%A7%90%E6%AE%89%E8%81%B7%E3%80%91%E7%B6%B2%E6%B0%91%E5%93%AD%E5%B4%A9-%E5%BC%B5%E5%BD%A5%E5%8D%9A%E7%8D%B2%E9%A0%92%E8%8B%B1%E5%8B%87%E7%8D%8E-148423 |title=【雄姐殉職】網民哭崩 張彥博獲頒英勇獎 |date=2018年11月4日 |publisher=[[am730|am730]] }}</ref>，獲提名《[[萬千星輝頒獎典禮2018|萬千星輝頒獎典禮2018]]》「最佳男配角」且入圍最後五強；同年12月9日於[[九龍灣國際展貿中心|九龍灣國際展貿中心匯星]]開辦入行13年首個演唱會《GOOD DAY．張彥博演唱會2018》<ref>{{Cite web |url=https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/262680/%E8%B7%B3%E8%BA%8D%E7%94%9F%E5%91%BD%E7%B7%9A-%E9%9B%84%E5%A7%90%E7%88%86%E7%B4%85%E8%B4%8A%E5%8A%A9%E5%95%86%E7%9D%87%E4%B8%AD-%E5%BC%B5%E5%BD%A5%E5%8D%9A%E8%8B%A6%E7%AD%8913%E5%B9%B4%E7%B5%82%E5%9C%93%E5%80%8B%E5%94%B1%E5%A4%A2 |title=【跳躍生命線】雄姐爆紅贊助商睇中　張彥博苦等13年終圓個唱夢 |author=吳子生 |date=2018年11月23日 |publisher=香港01 }}</ref>。
