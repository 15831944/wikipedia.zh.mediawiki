'''張俊彥'''（{{bd|1937年|10月12日|2018年|10月12日|catIdx=C張俊彥}}），台湾電子學者，[[中央研究院院士|中央研究院院士]]，曾出任[[國立交通大學|國立交通大學]]校長。

== 生平 ==
其父[[張木火|張木火]]，其母鄭淑玉。其父曾為麻豆初中教師，因[[二二八事件|二二八事件]]而被解職。1950年，在[[麻豆事件|麻豆事件]]中，以意圖顛覆政府罪名遭逮捕，判死刑。<ref>[https://newtalk.tw/news/view/2018-10-04/147701 促轉系列故事2》政治犯最長刑期 麻豆案24人總計判230年（見國防部39年8月29日法簽字第301號簽呈圖檔）]</ref><ref>[https://culture.tainan.gov.tw/form/index-1.php?m2=243&id=1369 臺灣歷史辭典〈麻豆案〉辭條（臺南市政府文化局轉載版）]</ref>

張俊彥1960年畢業於[[國立成功大學|國立成功大學]]電機系，並分別於1962年、1970年獲得[[國立交通大學|國立交通大學]]電子研究所碩士、博士，也是[[臺灣|臺灣]]教育史上第一位本土工學博士。張俊彥原有意出國留學，但因其父張木火涉入麻豆事件，張俊彥被[[警總|警總]]列入黑名單，無法出國留學，因此只能在台灣取得博士。<ref>[http://news.ltn.com.tw/news/focus/paper/220993 張俊彥 半導體之父 二二八受難者家屬]</ref><ref>[http://news.tvbs.com.tw/entry/173969 父母名建木淑館 張俊彥落淚澄清]</ref>其專長為[[半導體元件|半導體元件]]、[[半導體|半導體]]物理及[[:en:VLSI|VLSI]]技術。

2018年10月12日在抗癌兩年後辭世。

== 專業貢獻 ==

張俊彥自1963年起即開始指導學生，並於1964年與[[張瑞夫|張瑞夫]]、[[郭雙發|郭雙發]]共同建立[[台灣|台灣]]首座[[半導體|半導體]]研究中心，培養[[台灣|台灣]]眾多電子資訊產業人才。張俊彥個人自1964年研發矽[[電晶體|電晶體]]開始，就屢屢領導[[台灣|台灣]]電子、半導體研究的發展，並於1966年開始研究[[積體電路|積體電路]]、[[砷化鎵|砷化鎵]](1970年)、[[非晶矽|非晶矽]](1978年)。而他為[[行政院國科會|行政院國科會]]建立的[[國家毫微米元件實驗室|國家毫微米元件實驗室]]，也為該領域世界級的實驗室。

張俊彥曾經擔任交大電子研究所教授、電子物理系主任、成功大學電機系主任、[[美國|美國]][[貝爾實驗室|貝爾實驗室]]VLSI Group高級研究員，交大工學院院長、電機資訊學院院長、電子與資訊研究中心及校長等職務。除此之外，張俊彥也是政府科技政策智囊，曾經擔任[[行政院|行政院]]科技顧問及[[總統府國策顧問|總統府國策顧問]]。

==評價==
一般而言，大部分交大校友與學生皆非常感念張俊彥對交大的貢獻。由於張俊彥長年擔任企業顧問，所得之顧問費用幾乎全投入交大校務基金，超過新台幣一億餘元。在校務基金吃緊的情況下，的確對交大的發展居功厥偉。

張俊彥為人風趣嚴謹，學術上，除了學術上有極高的成就外，卸任後已在交大退休但仍執意指導學生進行研究，深獲學生愛戴。在校內活動上，張俊彥時常上台與同學同樂，純樸風趣的形象，常感動學生。所以一般來說，在交大學生的心目中張俊彥有相當高的地位。後來張俊彥遇到新聞事件質疑，常會有大批學生在網路上聲援支持，可見其在交大學子心中形象。惟張俊彥擔任校長期間仍有部分教授持負面看法，在許多校務的議題上無法支持校方政策，認為校方決定事務常過於草率。更有部分教授認為張俊彥時代開始，交大的發展過度偏重電機資訊科系。<ref>[http://blog.udn.com/superbird/1866911 「獨尊電資 鄙視傳產」 價值扭曲的大學校園]</ref><ref>[http://blog.libertytimes.com.tw/superbird/2010/03/02/51222 校長續任爭議 校友憂心交大走下坡]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

在校長任內，交大與[[臺南縣|臺南縣]]政府簽定「設置國立交通大學台南校區協議書」，由[[台南縣|台南縣]]政府無償供地予交大興建分校，並於2008年動工，翌年招生上課。
目前張前校長也正積極規劃交大台南校區旁邊的區域，未來中研院以及一些企業將會進駐在台南校區這區塊，形成產業學合作而讓交大台南校區更加繁榮。

==木淑館風波==
2006年交大傳出要將北大門旁的機車C棚與部分矩陣林拆除，將利用張俊彥爭取來的企業捐款改建為第三招待所「木淑館」，以做成國際學者與外賓來校訪問時的居所，曾引發一番討論。外界質疑為何張俊彥可用父母名字取名「木」「淑」作為招待所命名，校方回應是由於張俊彥校長捐出新台幣兩千五百萬元作為建造經費，滿足「璞玉計畫捐款感謝辦法」命名建築物的條件，故請張俊彥校長命名。但有部分教授質疑，兩千五百萬元為企業捐款，是否可作為張俊彥本人名義捐助交大。但校方舉出，此款原為企業給張俊彥校長的兩千五百萬元顧問費，是依照張俊彥本人意願改捐給交大，其實仍為張俊彥校長之捐款。校方僅依照2004年校務規劃委員會將此款撥為交大興建招待所之用，以解決學者來交大交流的居住問題，並無不妥。目前該館已經完工，但未用木淑館稱呼，設牌為第三招待所。<ref>[ 姚仁喜是否選擇無奈 交大木淑館即將完工]</ref><ref>[http://chadesign.blogspot.com/2007/09/blog-post.html 案例：交大木淑館]</ref><ref>[ 交大：第三招待所木淑館並無公私不分]</ref>

== 考試院長提名風波 ==
2008年6月，總統[[馬英九|馬英九]]提名張俊彥任[[考試院|考試院]]院長。但這項提名受到眾多國民黨籍的立法委員的反對，並對張提名一連串的指控和批評。其中台灣第二大證券商寶來集團董事長[[白文正|白文正]]被《[[壹周刊|壹周刊]]》影射通過金錢交易收買張俊彥獲得國立交通大學榮譽博士學位。7月2日，白文正在家鄉[[澎湖縣|澎湖縣]][[自殺|自殺]]以表清白。

白文正的自殺直接導致張俊彥宣布退出考試院長的提名，張在記者會上情緒激動地表示「對扭曲人性的媒體與政治感到痛心」，他也「見識到極少數媒體操弄大眾，把一位創造台灣奇蹟的英雄逼上絕路。」 <ref>[http://news.bbc.co.uk/chinese/trad/hi/newsid_7490000/newsid_7491600/7491658.stm 張俊彥退出台灣考試院長提名] 英國廣播公司</ref>
