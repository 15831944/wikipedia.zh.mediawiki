'''張曉明'''（{{bd|1963年|9月}}），[[江蘇|江蘇]][[泰興|泰興]]人。1986年7月参加工作，[[中國共產黨|中國共產黨]]黨員、[[中國共產黨第十八屆中央委員會候補委員列表|中共第十八屆中央候補委員]]、[[中国共产党第十九届中央委员会委员列表|第十九届中央委员]]。[[中国人民大学法学院|中国人民大学法学院]]刑法专业毕业，[[研究生|研究生]]学历，法学硕士，曾任[[廖晖|廖晖]]秘书。現任[[中共中央港澳工作協調小組|中共中央港澳工作協調小組]]成員和[[国务院港澳事务办公室|国务院港澳事务办公室]]主任兼[[党组书记|党组书记]]<ref>[http://news.qq.com/a/20170922/045963.htm 张晓明任国务院港澳办主任 王玉普任国家安监总局局长]，腾讯网，2017年9月22日</ref>。

== 生平 ==
1984年畢業於[[西南政法學院|西南政法學院]]法律系，並入讀[[中國人民大學|中國人民大學]]，師從刑法名家[[高銘暄|高銘暄]]<ref>{{cite web |url = http://www.singtao.com/yesterday/chi/1114eo02.html
|title=彭王任中委張曉明晉候補委員 |publisher=星島日報 |accessdate=2013-03-17|date=2012-11-14}}</ref>。高銘暄則師從前蘇聯著名刑法學家貝斯特洛娃教授、達瑪亨教授、尼古拉耶夫教授和柯爾金教授<ref>{{cite web|url=http://law.ruc.edu.cn/introduction/shizi3.asp?No=73|title=高銘暄|publisher=中國人民大學|accessdate=2013-03-17|deadurl=yes|archiveurl=https://web.archive.org/web/20130724062003/http://law.ruc.edu.cn/introduction/shizi3.asp?No=73|archivedate=2013-07-24}}</ref>。

1986年，張曉明法學院刑法專業[[研究生|研究生]][[畢業|畢業]]，获[[法學碩士|法學碩士]]学位。同年畢業生有黃京平教授。<ref>{{cite web|url=http://law.ruc.edu.cn/introduction/shizi3.asp?No=72|title=黃京平|publisher=中國人民大學|accessdate=2013-03-17|deadurl=yes|archiveurl=https://web.archive.org/web/20130724054738/http://law.ruc.edu.cn/introduction/shizi3.asp?No=72|archivedate=2013-07-24}}</ref>

1986年起在[[國務院港澳事務辦公室|國務院港澳事務辦公室]]工作，長期擔任原港澳辦主任[[廖暉|廖暉]]的秘書。

張曉明曾參加[[中英聯合聯絡小組|中英聯合聯絡小組]]和[[香港特別行政區籌備委員會|香港特別行政區籌備委員會]]預備委員會秘書處工作。曾任[[國務院港澳事務辦公室|國務院港澳事務辦公室]]政策研究司司長、綜合司司長、辦公室副主任。

2006年2月28日被任命為[[全國人大常委會|全國人大常委會]][[香港特別行政區基本法委員會|香港特別行政區基本法委員會]]委員。2008年3月當選[[第十一屆全國人民代表大會代表名單|第十一屆全國人民代表大會代表]]。<ref>{{cite web|url=http://www.swupl.edu.cn/xiaoyh/xin/news.asp?id=1315|title=张晓明校友任中央驻港联络办主任|publisher=西南政法學院|date=2013-01-08|accessdate=2013-03-16|deadurl=yes|archiveurl=https://web.archive.org/web/20150323152546/http://www.swupl.edu.cn/xiaoyh/xin/news.asp?id=1315|archivedate=2015-03-23}}</ref>

2012年11月的[[中國共產黨第十八次全國代表大會|中共十八大]]上，當選[[中國共產黨第十八屆中央委員會候補委員列表|第十八屆中央委員會候補委員]]。2017年10月的[[中國共產黨第十九次全國代表大會|中共十九大]]上，當選[[中國共產黨第十九屆中央委員會委員列表|第十九屆中央委員會委員]]。

== 主管港澳事務 ==
2012年12月18日，張曉明出任[[中央人民政府駐香港特別行政區聯絡辦公室|中央人民政府駐香港特別行政區聯絡辦公室]]主任一職，進入[[中共中央港澳工作協調小組|中共中央港澳工作協調小組]]。

其中談及中聯辦的工作方向時他表示，會客觀、全面及準確地向中央反映香港情況，不存在中央收緊對港政策問題，也主動否認「[[西環|西環]]治港」，強調「西環要盡職」，履行中央賦予的權力，做應該做的事。他強調撤換行政長官[[梁振英|梁振英]]之說是「無稽之談，可以休矣」。在《[[中华人民共和国香港特别行政区基本法|香港基本法]]》[[香港特别行政区基本法第二十三条|第二十三條立法]]問題上，張曉明重申完成二十三條立法是香港應盡的憲制義務，由特區政府按實際情況決定，他認為，二十三條並非「提都不能提的禁區」。<ref>{{cite web |url = http://orientaldaily.on.cc/cnt/news/20130111/00176_031.html |title=張曉明：西環不治港 |publisher=東方日報 |date=2013-01-11}}</ref><ref>{{cite web|url=http://www.singpao.com/xw/gat/201301/t20130111_413098.html|title=張曉明：西環不治港 不存在中央收緊對港政策|publisher=成報|date=2013-01-11}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

担任[[中央人民政府驻香港特别行政区联络办公室|中央人民政府驻香港特别行政区联络办公室]]（中联办）主任期间，被指高度影響及介入香港的內部事務及被[[泛民主派|泛民主派]]認為有違反《[[香港基本法|香港基本法]]》第22條及《[[中英聯合聲明|中英聯合聲明]]》之嫌。[[2016年香港立法會選舉|2016年香港立法會選舉]]，他表明不容主張[[香港獨立|香港獨立]]的人士參選立法會，對港獨零容忍，導致有數名人士無法參選。

2017年7月，張曉明在[[香港特別行政區成立二十周年紀念|香港回歸20周年]]暨中聯辦成立70周年慶祝大會發言：「[[習近平|習近平]][[總書記|總書記]]在[[習近平總書記視察香港|視察香港]]期間發表的系列重要講話高瞻遠矚，為繼續推進“[[一國兩制|一國兩制]]”在香港的實踐指路引航，是我們繼續砥礪前行的科學指南」。張曉明又表示中聯辦的首要任務，就是認真學習、深刻領會習近平總書記的有關[[習近平總書記重要講話精神|重要講話精神]]，不折不扣地落實總書記的希望和要求。<ref>{{cite web|url=http://news.takungpao.com.hk/paper/q/2017/0722/3475530.html|title=不折不扣落實總書記要求|publisher=大公報|date=2017-07-22}}</ref>

2017年9月22日，接替[[王光亞|王光亞]]出任為[[國務院港澳事務辦公室|國務院港澳事務辦公室]]主任<ref>{{cite news|title=張曉明任國務院港澳辦主任，王玉普任國家安監總局局長|url=http://www.thepaper.cn/newsDetail_forward_1803010|newspaper=凤凰网|date=2017-09-22}}</ref>。

2017年11月8日，港澳辦主任張曉明帶領一眾幹部舉行[[中華人民共和國憲法|憲法]]宣誓儀式，張曉明在講話中提到：「習總書記強調好幹部的標準是忠誠、乾淨、擔當，忠誠是排在第一位的」，「[[習近平|習近平]]總書記在[[中國共產黨第十九次全國代表大會|十九大]]報告中指出，我們既要政治過硬，也要本領高強。」，「用[[習近平新時代中國特色社會主義思想|習近平新時代中國特色社會主義思想]]武裝頭腦」。<ref>{{cite web|url=https://news.mingpao.com/pns/dailynews/web_tc/article/20171110/s00002/1510250337410|title=港澳辦搞宣誓儀式 張曉明三提總書記|publisher=明報|date=2017-11-10}}</ref>

== 爭議 ==
=== 活著論 ===
2014年9月11日，在回答北京是否允許[[泛民主派|泛民主派]]人士擔任[[香港行政長官|香港行政長官]]時發表言論：“讓你活著就足以顯示國家的文明和包容”。<ref>{{cite web |title = China asserts paternal rights over Hong Kong in democracy clash |url = http://www.reuters.com/article/2014/09/11/us-china-hongkong-insight-idUSKBN0H600120140911|website=Reuters|accessdate=2015-06-29|date=2014-09-11}}</ref>對此他表示，個別媒體引用他的「泛民能活著，足證國家包容」的說法，是用意惡毒、斷章取義。他說，雙方會面事先已有約定，就是四場會面的交談都是閉門進行，互不引述對方講話。他承認曾在對話中使用「還活著」一詞，但他的原話是在一國兩制下，宣稱以推翻[[一黨專政|一黨專政]]為宗旨的[[支聯會|支聯會]]，現在還能舉行各種活動，支聯會成員還可以當選立法會議員，本身就體現了一種很大的政治包容。他指1997年以前，有人擔心回歸後無法在香港待下去，但他反問，說這種話的人現在不是還活著，而且活得好好嗎？<ref>{{cite web |title = 張曉明試圖澄清「活著論」繼續引發爭議 |url = http://www.bbc.co.uk/zhongwen/trad/china/2014/09/140913_china_hk_media_zhangxiaoming|publisher=BBC|accessdate=2015-06-29 |date=2014-09-13}}</ref>

=== 行政長官超然論 ===
2015年9月11日，中聯辦主任張曉明出席《基本法》頒布25周年研討會時就香港政制表示，草擬基本法時的主導思想，是以特區政府作為核心的主要體制，「行政長官具有超然於行政、立法、司法三個機關之上的特殊法律地位」。張曉明於研討會發表致謝辭說，香港於回歸前及回歸後，也沒有實行過任何三權分立的政治體制。他說，「作為領導特區政府的行政長官，他所具有的雙首長身分和雙負責制的責任，使得行政長官具有超然於行政、立法、司法的三個機關之上的特殊法律地位，處於特別行政區權力運行的核心位置，在中央政府之下，特別行政區的三權之上起着連結的樞紐作用」。<ref>{{cite web|title = 張曉明「行政長官具有超然於行政、立法、司法三個機關之上的特殊法律地位」|url = http://news.mingpao.com/pns/%E5%BC%B5%E6%9B%89%E6%98%8E%EF%B9%95%E7%89%B9%E9%A6%96%E8%B6%85%E7%84%B6%E4%B8%89%E6%AC%8A%E4%B9%8B%E4%B8%8A%20%E6%9D%8E%E6%9F%B1%E9%8A%98%EF%BC%9A%E6%B8%AF%E8%A1%8C%E6%99%AE%E9%80%9A%E6%B3%95%20%E5%B1%AC%E4%B8%89%E6%AC%8A%E5%88%86%E7%AB%8B/web_tc/article/20150913/s00001/1442080836494|publisher =  明報|date=2015-09-13}}</ref>

有關言論引發巨大爭議，多位政界人士對此作出評論。9月14日，[[香港大律師公會|香港大律師公會]]發表聲明，對張曉明的演講內容的討論表示深切關注。聲明提到，基本法清晰界定了三權的關係和權責。大律師公會堅信「權力分立」的法律概念在基本法底下得到充分的落實。張曉明「超然於三個機關之上」的描述，難免被解讀為特首「凌駕於三權之上」，引起公眾憂慮，大律師公會對此深表遺憾。大律師公會又促請張曉明及律政司司長早日作出澄清，「消除香港市民以及國際社會對香港在一國兩制下秉行法治的疑慮」。<ref>{{cite web|title=香港大律師公會就張曉明主任在九月十二日《香港特別行政區基本法》頌布25周年研討會的演講內容的聲明|url=http://www.hkba.org/whatsnew/press-release/20150914%20-%20Press%20Statement%20of%20HKBA%20-%20Chinese.pdf|publisher=香港大律師公會|accessdate=2015-10-10|deadurl=yes|archiveurl=https://web.archive.org/web/20160306062916/http://www.hkba.org/whatsnew/press-release/20150914%20-%20Press%20Statement%20of%20HKBA%20-%20Chinese.pdf|archivedate=2016-03-06}}</ref>

同日，23名泛民立法會議員以[[泛民會議|泛民會議]]名義發表聯合聲明，引用基本法三項條文，說明特首權力受多重制衡，因此超然一說是無中生有。泛民會議表示希望張曉明能「尊重『一國兩制』及《基本法》，勿再發表損害香港核心價值的言論」，又促請梁振英「公開重申會遵守本地法律，並按基本法條文接受立法與司法的制衡」。<ref>{{cite web|title=泛民會議就中聯辦主任張曉明「特首超然論」的聲明|url=http://www.ipkinyuen.org.hk/file_7nevts/newEvts_20150914.htm|publisher=立法會葉建源議員辦事處|accessdate=2015-10-10}}</ref>9月15日，[[梁振英|梁振英]]在政府總部會見記者時，評論了張曉明的演說內容，指張曉明在演說中並無使用「凌駕」字眼，有人沒在現場聽而作出批評，出現斷章取義或誤導的情況。<ref>{{cite news|title=為張曉明超然論辯解 梁振英斥外界斷章取義|url=http://www.am730.com.hk/article-281467|newspaper=AM730|date=2015-09-16|accessdate=2015-10-10}}</ref><ref>{{cite news|title=梁振英：張曉明「香港特首超然」論遭曲解 |url=http://www.bbc.com/zhongwen/trad/china/2015/09/150915_hongkong_two_system|newspaper=BBC|date=2015-09-15|accessdate=2015-10-10}}</ref>9月16日，梁振英又對傳媒表示行政長官地位「確實是超然」。<ref>{{cite news|title=香港特首梁振英：特首地位「確實是超然」|url=http://www.bbc.com/zhongwen/trad/china/2015/09/150916_hongkong_zhang_xiaoming_responses|newspaper=BBC|date=2015-09-16|accessdate=2015-10-10}}</ref>9月20日，[[香港律師會|香港律師會]]就張曉明的言論發表聲明，重申兩個基本原則：「司法獨立對法治的重要」和「香港司法機構的專業及其制度健全」。<ref>{{cite web|title=香港律師會的聲明|url=http://www.hklawsoc.org.hk/pub_c/news/press/20150920.asp|publisher=香港律師會|accessdate=2015-10-10}}</ref>
