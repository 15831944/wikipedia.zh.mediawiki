'''張若凡'''（{{bd|1994年|6月25日|||}}），[[台灣|台灣]]新聲代[[女歌手|女歌手]] ，出生於台灣[[高雄市|高雄市]] ，成長于台灣[[台中市|台中市]] ，畢業於樹德科技大學表演藝術系。2015年參與《[[Super_Star_我要当歌手2|Super Star 我要当歌手2]] 》，擔任[[梁文音|梁文音]]戰隊隊員。2018年以[[浪Live|浪Live]] 直播主的身份報名參加台灣大型選秀节目《[[聲林之王|聲林之王]]》，最終獲得第六名。<ref>{{Cite web |url =https://star.ettoday.net/news/1284799|title =《聲林》張若凡飆唱「坦白」面具後哭了：不可以再懦弱|author = 許逸群}}</ref><ref>{{Cite web |url =https://star.ettoday.net/news/1365624|title =《聲林》總冠軍出爐！李友廷「119比114」險勝林孟辰|author = 許逸群}}</ref>若凡受邀參與由聲林之王攜手直播平台《浪Live》舉辦“聲林直播會外賽”，最終拿下第二名，獲得發行數位單曲的機會。<ref>{{Cite web |url =https://www.ettoday.net/news/20181207/1325298.htm|title =《浪Live》祭出百萬獎勵！替聲林選手圓發片夢}}</ref>2019年2月27日在各大數位平台發行重新詮釋的單曲《走了嗎》，短短幾天內奪下KKBOX新歌即時榜、華語日榜、單曲榜冠亞軍。<ref>{{Cite web |url =https://www.kkbox.com/tw/tc/column/showbiz-0-7246-1.html|title =張若凡〈走了嗎〉橫掃新歌榜冠軍！《聲林》成名曲改編後更催淚}}</ref>
