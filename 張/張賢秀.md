'''張賢秀'''（{{lang|ko|장현수}}，{{bd|1991年|9月28日|||Jang, Hyun-Soo}}），[[韩国|韩国]]足球运动员，现效力于[[日職聯|日職聯]]球队[[FC東京|FC東京]]。

== 俱乐部生涯 ==
张贤秀2010年就读于韩国[[延世大学|延世大学]]，但没有毕业就被J联赛球队相中，2012年加入[[日本职业足球联赛|日本职业足球联赛]]球队[[FC东京|FC东京]]。<ref>[http://www.fctokyo.co.jp/home/index.phtml?cont=item&item=12020 チャン ヒョンス選手 来季新加入決定のお知らせ] {{webarchive|url=https://web.archive.org/web/20120715052813/http://www.fctokyo.co.jp/home/index.phtml?cont=item&item=12020 |date=2012-07-15 }}{{ja icon}}</ref>由于球队从[[磐田喜悦|磐田喜悦]]引进的后防主将[[加賀健一|加賀健一]]受伤，张贤秀在第一个职业赛季就获得很多出场机会。<ref name="sponavi">[http://sportsnavi.yahoo.co.jp/soccer/jleague/headlines/20120408-00000001-spnavi-socc.html FC東京のチャン・ヒョンス「Jリーグでのプレーを夢みてきた」]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}{{ja icon}}</ref>2012年3月6日，他在[[2012年亚足联冠军联赛|2012年亚足联冠军联赛]]小组赛第一轮FC东京客场对阵[[布里斯班狮吼足球俱乐部|布里斯班狮吼]]的比赛中出场，上演职业生涯首秀。4月8日，他在联赛第五轮和[[川崎前锋|川崎前锋]]的比赛中首次在联赛亮相。

2014年1月，[[中国超级足球联赛|中国超级足球联赛]]球队[[广州富力|广州富力]]在官方微博率先宣布张贤秀加盟球队。2月初，广州富力和FC东京的官方网站分别确认了张贤秀转会的消息。

== 国家队生涯 ==
张贤秀曾经代表[[韩国20岁以下国家足球队|韩国U-20国家队]]参加[[2011年U20世界盃足球賽|2011年U20世界盃足球賽]]，帮助球队进入十六强。之后被[[洪明甫|洪明甫]]招入韩国国奥队备战[[2012年夏季奥林匹克运动会足球比赛|2012年夏季奥林匹克运动会足球比赛]]，本来计划让他作为[[洪正好|洪正好]]和[[金英權|金英權]]的替补，担任球队的替补中卫，之后由于洪正好受伤无法参加比赛，张贤秀被委以主力的位置。<ref>[http://sports.media.daum.net/soccer/news/a_match/breaking/view.html?cateid=1172&newsid=20120604165907406 장현수, “홍정호 빈자리 메우겠다”]{{Ko icon}}</ref>但在报名参赛后不久，张贤秀也因为受伤无法参加伦敦奥运会比赛，最终被[[金基熙|金基熙]]所替代。

2013年1月，张贤秀首次入选[[韩国国家足球队|韩国国家足球队]]集训名单。<ref>[http://www.chunichi.co.jp/chuspo/article/fctokyo/news/201301/CK2013012202000158.html ヒョンスが韓国A代表] {{webarchive|url=https://web.archive.org/web/20131112134117/http://www.chunichi.co.jp/chuspo/article/fctokyo/news/201301/CK2013012202000158.html |date=2013-11-12 }} 東京中日スポーツ (2013年1月22日)</ref>同年6月18日，他在[[2014年世界盃外圍賽亞洲區第四輪|2014年世界盃外圍賽亞洲區第四輪]]最后一场比赛韩国主场0比1不敌[[伊朗国家足球队|伊朗]]的比赛出场，首次在国际A级比赛亮相。

2014年，张贤秀代表[[韩国23岁以下国家足球队|韩国U-23国家队]]参加了[[2014年亚洲运动会足球比赛|2014年亚运会]]，并在比赛中夺得金牌，因此获得免服兵役的资格<ref>{{Cite web|url=https://people.search.naver.com/search.naver?where=nexearch&query=%EC%9E%A5%ED%98%84%EC%88%98&sm=tab_etc&ie=utf8&key=PeopleService&os=189860|title=장현수, 네이버 인물검색|last=|first=|date=|website=네이버(Naver)|archive-url=|archive-date=|dead-url=|access-date=}}</ref>。

2018年夏天，他為韩国国家队出戰[[2018年世界杯足球赛|2018年世界杯足球赛]]。

张贤秀虽因获得亚运金牌而无需服兵役，但仍需进行相关军事训练和社区服务。由于张贤秀伪造社区服务证明，2018年11月1日，[[韩国足球协会|韩国足球协会]]宣布对张贤秀处以永久开除出国家队的处罚，同时罚款3000万韩元<ref>{{cite web |author1=一拳超人 |title=前富力外援免兵役造假被韩国队禁赛 所属球队警告 |url=http://sports.sina.com.cn/china/national/2018-10-30/doc-ifxeuwws9473748.shtml |website=新浪 |accessdate=2018-11-01}}</ref><ref>{{cite web |author1=一拳超人 |title=韩国队史上最严厉处罚诞生 前富力外援被永久除名 |url=http://sports.sina.com.cn/china/national/2018-11-01/doc-ifxeuwwt0251972.shtml |website=新浪 |accessdate=2018-11-01}}</ref>。
