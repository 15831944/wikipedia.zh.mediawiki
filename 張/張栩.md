'''張栩'''{{BD|1980年|1月20日||}}是[[中華民國|台灣]][[台北市|台北市]]人，台灣旅日圍棋職業棋士，隸屬[[日本棋院|日本棋院]]；與[[羽根直樹|羽根直樹]]、[[山下敬吾|山下敬吾]]、[[高尾紳路|高尾紳路]]並稱日本棋界“若手四天王”或“平成四天王”。

於2009年曾同時擁有日本棋界[[名人戰_(日本圍棋)|名人]]、[[王座戰_(日本圍棋)|王座]]、[[天元戰_(日本圍棋)|天元]]、[[碁聖戰|碁聖]]、[[十段|十段]]五個頭銜，為日本職業棋壇自1977年創立七大頭銜以來，首度出現同時擁有五項頭銜的棋士。

2010年2月奪下[[棋聖戰|棋聖]]頭銜後，成為日本棋院繼[[趙治勳|趙治勳]]之後第二位取得大滿貫（七大頭銜均曾獲得）的棋士。

2018年11月，張栩從[[井山裕太|井山裕太]]手上奪去[[名人戰_(日本圍棋)|名人]]頭銜，時隔五年後重返頭銜者行列，也是時隔十期後重返[[名人戰_(日本圍棋)|名人]]位。

==簡歷==
六歲半起開始習棋，於1990年赴日加入日本棋院，由[[林海峰_(圍棋)|林海峰]]九段收為內弟子，同門師兄弟有[[林子淵|林子淵]]、[[林漢傑|林漢傑]]，於1994年成為職業棋士。

2003年起，陸續獲得本因坊、王座、名人之頭銜，成為最年輕同時獲得「名人」、「本因坊」頭銜者，2008年更成為史上第四人同時獲得名人、天元、王座、碁聖頭銜的棋士，隔年四月再取得十段頭銜，成為1977年開始實施現行七大頭銜賽制後，第一位同時擁有五項頭銜的棋士。

2004年，與同為職業棋士的[[小林泉美|小林泉美]]結婚，當時兩人分別擁有「本因坊」與「女流本因坊」之頭銜，因此也被稱為「本因坊夫妻」。兩人婚後陸續於2006年3月、2009年11月產下長女及次女。

2005年至2007上半年為張栩國際賽成績最好的時期<ref> [http://sports.geocities.jp/mamumamu0413/world/column004.html 張栩九段の国際戦成績] {{webarchive|url=https://web.archive.org/web/20150209160247/http://sports.geocities.jp/mamumamu0413/world/column004.html |date=2015-02-09 }}。</ref>，2005年張栩斬獲LG杯、亞洲杯電視快棋賽二冠；2007年進入豐田杯世界王座戰決賽，惜以1:2負於[[李世乭|李世乭]]。2007下半年後張栩再無突出的成績，卻在日本國內的賽事逐步邁向了頂峰。

2010年2月取得棋聖頭銜後，成繼趙治勳後，日本圍棋史上第二位，也是最年輕（30歲1個月）先後獲得七大棋戰頭銜者。

2015年，因為戰績無法突破，陷入低潮，張栩舉家搬回台灣生活，並侍奉自10歲赴日即長年分別生活的雙親、讓在日本出生長大的2個女兒學習中文及閩南語，日籍妻子也和女兒們一起學習中文，且全家因此機緣而重新體驗了台灣當地的文化。但張栩仍會至日本參加比賽。

2016年，張栩獲得NHK盃後，宣布重返日本生活。

2018年7月19日，張栩以7战全勝戰績獲得日本第43期圍棋名人戰挑戰者資格，挑戰的在位名人是曾奪取自己多個頭銜的井山裕太。這也是自2013年第61期王座戰以来，張栩首次殺入日本圍棋傳統七大頭銜戰的決賽。

2018年11月2日，張栩在第43期名人戰決勝局當中執白4目半擊敗井山裕太，並以總比分4:3挑戰成功，從井山裕太手中奪回名人頭銜。此次獲勝是張栩時隔五年首次奪取七大頭銜戰的頭銜，也是張栩時隔九年重新奪回名人頭銜。此次名人戰張栩一度以總比分1:3落後於井山裕太，但張栩最終連扳三城，且多次上演驚天逆轉，在奪得自己重返日本生活後的首個頭銜，也強勢宣告自己職業生涯的復甦。<ref>{{cite news|title=張栩王子復仇記 逆轉勝擊敗井山裕太重奪名人|url=https://www.cna.com.tw/news/firstnews/201811025006.aspx|publisher=中央社|date=2018-11-02|accessdate=2018-11-06}}</ref>

==棋風==
張栩的棋風薄而銳利，行棋速度感甚快，局部[[手筋|手筋]]甚強，於勝負處、[[官子|官子]]都堪稱一絕，不懼怕戰鬥為主的韓流棋風，但對於厚實棋風的[[高尾紳路|高尾紳路]]卻稍感棘手，兩人對局往往難分優劣。2010年以後，新生代的[[井山裕太|井山裕太]]以凌厲但厚實的棋風讓張栩吃盡苦頭，同時也一一奪去張栩的頭銜。

==出賽紀錄==
; 2001年
* 取得第56期[[本因坊戰|本因坊戰]]挑戰權，為日本棋院史上最年少者的本因坊挑戰者，但最終以3：4不敵[[王銘琬|王銘琬]]本因坊。

; 2002年　
* 獲得[[NHK盃|NHK盃]]、[[新人王戰_(日本圍棋)|新人王]]優勝。
* 年度戰績70勝14敗，創日本史上年度最多勝紀錄。

; 2003年　頭銜：本因坊、王座
* 擊敗[[加藤劍正|加藤正夫]][[本因坊|本因坊]]，首次獲得本因坊頭銜，並因此晉升九段。
* 擊敗[[王銘琬|王銘琬]][[王座|王座]]，首次獲得王座頭銜。

; 2004年　頭銜：名人、本因坊、王座
* 先後在本因坊戰及名人戰擊敗[[依田紀基|依田紀基]]名人，完成本因坊二連霸，並首次獲得[[名人|名人]]頭銜，為史上第五位獲得名人‧本因坊頭銜的棋士。
* 於年底王座戰中擊敗[[山下敬吾|山下敬吾]][[天元|天元]]，連霸王座頭銜。

; 2005年　頭銜：名人、王座
* 獲得NHK杯、NEC杯、LG杯世界棋王戰冠軍、亞洲電視快棋冠軍。
* 於本因坊戰中敗給[[高尾紳路|高尾紳路]]，失去本因坊頭銜。
* 繼續保有名人、王座頭銜。

; 2006年　頭銜：碁聖
* 擊敗依田紀基[[碁聖|碁聖]]，首獲碁聖頭銜。
* 獲[[龍星戰|龍星戰]]、[[阿含桐山杯|阿含桐山杯]]冠軍。
* 失去名人及王座頭銜。

; 2007年　頭銜：名人、碁聖
* 獲NEC杯、龍星戰冠軍。
* 衛冕碁聖頭銜。
* 奪回名人頭銜。

; 2008年　頭銜：名人、天元、王座、碁聖
* 獲NHK杯、阿含桐山杯冠軍。
* 衛冕碁聖、名人頭銜。
* 奪取天元、王座頭銜。成為史上第四人同時獲得名人、天元、王座、碁聖頭銜的棋士。

; 2009年　頭銜：十段、王座、碁聖
* 4月：擊敗高尾紳路[[十段|十段]]，首獲十段頭銜，成為日本職業棋壇自1977年實施現行七大頭銜制以來，首位同時擁有五項頭銜（名人、十段、天元、王座、碁聖）的棋士。
* 7月：衛冕碁聖（4連霸）。
* 10月：負於挑戰者[[井山裕太|井山裕太]]八段而失去名人頭銜。
* 11月：以3：0擊敗山田規三生九段，衛冕王座頭銜。
* 12月：負於挑戰者山下敬吾棋聖，失去天元頭銜。

; 2010年　頭銜：棋聖、十段、王座
* 以4:1戰績擊敗山下敬吾棋聖，首獲[[棋聖|棋聖]]頭銜，成為繼[[趙治勳|趙治勳]]後，日本圍棋史上第二位同時也是最年輕（30歲1個月）先後獲得七大棋戰頭銜者。
* 4月：擊敗山下敬吾天元，衛冕十段頭銜。
* 8月：2：3不敵[[坂井秀至|坂井秀至]]七段，失去碁聖頭銜，也喪失因五連霸獲得名譽碁聖的機會。
* 11月：以3：0擊敗山田規三生九段，第二次達成王座三連霸。代表台灣出征廣州亞運，與謝依旻女流名人本因坊配對混雙比賽。

; 2011年　頭銜：棋聖、王座
*3月：擊敗山下道吾本因坊，獲得NEC杯（生平第三次）。同月稍後又以4：2擊敗井山名人，完成棋聖二連霸。（同日發生[[2011年日本东北地方太平洋外海地震|日本东北地方太平洋外海地震]]，兩人對局一度中斷。）
*4月：2：3不敵井山名人，失去十段頭銜。
*11月：3：0擊敗羽根直樹碁聖，達成王座四連霸。

; 2012年　頭銜：棋聖
*3月：4：3擊敗高尾紳路，達成棋聖三連霸。
*10月：擊敗同門林子淵七段，第四度奪取阿含桐山杯。
*11月：3：0不敵井山本因坊，痛失王座，也喪失因五連霸獲得名譽王座的機會。

; 2013年
*3月：不敵井山五冠，失去棋聖頭銜。自2003年來首度無冠，降格為張栩九段。
*12月: 1 : 3挑戰王座失敗

; 2014年
*12月：打入春蘭盃八強，敗於古力九段。

; 2014年
*與妻女返台定居
; 2016年
*3月：擊敗寺山怜，獲NHK盃，四年來第一次重新奪冠。

; 2018年　頭銜：名人
*7月：以全胜战绩获得第43期名人战挑战者资格，时隔四年半重返头衔战决赛。
*11月：在第43期名人戰中以總比分4:3擊敗在位名人井山裕太，重新奪回九年前失去的名人頭銜。這也是張栩時隔五年後重新奪得日本圍棋七大頭銜。

===紀錄===
* 2002年度戰績70勝（14敗）史上單一年份最多勝紀錄。
* 獎金排名第一之年度：2003年-2005年、2007年-2010年
* 2009年4月，獲得十段頭銜，成為史上第一位同時身兼小四喜（十段、天元、王座、碁聖）的第一人。
* 2009年12月17日，張栩在第65期本因坊戰循環圈賽執黑3目半勝武宮正樹，這一勝為他締造了史上第36人通算700勝的職業生涯里程碑，達成時戰績為700勝254負2和1無勝負，並刷新三項紀錄成為史上第一：最年少達成（29歲10個月）、入段後最短期間（15年8個月）、達成時勝率最高（73.38%）<ref>{{cite web | language = ja | publisher = 日本棋院 | title = 張栩十段が通算700勝達成【史上最年少・史上最短期間・史上最高勝率】 | url = http://www.nihonkiin.or.jp/match/2009/12/700_7.html | author =  | date = 2009-12-17 | accessdate = 2010-11-15 | deadurl = yes | archiveurl = https://web.archive.org/web/20100208214514/http://www.nihonkiin.or.jp/match/2009/12/700_7.html | archivedate = 2010-02-08 }}</ref>。
* 2010年獲得棋聖頭銜，完成七大棋戰大滿貫（史上第二人，史上最年少，但隨著井山於2013年取得棋聖頭銜後，最年少記錄已被打破）。
* 2017年1月19日，張栩在日本第43期碁聖戰預選賽上執黑109手中盤擊敗女棋手金豔四段，從而達成職業生涯正式比賽900勝。達成900勝時的總戰績為900勝388負（2和棋1無勝負），勝率為69.9%，排名日本棋院史上第二，僅次於高尾紳路（900勝時勝率70%） 。另外他36歲11個月的年齡達成900勝，創造了日本棋院史上900勝最年少紀錄。經歷了22年9個月達成900勝，排名日本棋院史上第二位，第一位是山下敬吾的22年7個月。

==家庭==
* 父親是圍棋教育家[[張遠錫|張遠錫]]，[[中環|中環]]董事長[[翁明顯|翁明顯]]為乾爹，前[[國立清華大學|國立清華大學]]校長[[沈君山|沈君山]]則是義父。
* 夫人[[小林泉美|小林泉美]]六段是日本圍棋六超之一的[[小林光一|小林光一]]九段的女兒，岳母[[小林禮子|小林禮子]]是[[木谷道場|木谷道場]]創始人[[木谷實|木谷實]]九段的女兒。生有兩女：張心澄、張心治。
* 張栩為瀨越憲作九段一門第三代佼佼者，師父為林海峰九段，以內弟子身份受教。太師父為昭和棋聖吳清源九段。就輩份來說，應該稱曹薰鉉九段為太師叔，李昌鎬九段為師叔。

==逸話==
*張栩、[[羽根直樹|羽根直樹]]、山下敬吾、高尾紳路為日本棋院新生代之優秀棋士，有「若手四天王」或「平成四天王」之稱。
*對於張栩姓名日語發音方面，日本中國文學家兼圍棋迷[[高島俊男|高島俊男]]指，「栩」的正確日語發音應為「{{lang|ja|く}}」（ku），而非「{{lang|ja|う}}」（u），但張栩本人則說，由於其師林海峰指出，如果以正音唸張栩的日語名字，會變成Choku，與「粉筆」的日語音譯相同，造成混淆，故因此惟有以「Cho U」來發音。
*張栩以官子能力強著稱，據說他小時候在練習官子的時候，是擺到一盤棋的最後一手，然後從最後一手往回覆盤。 <ref>{{cite web|title=第60期王座大盤解說23周俊勳+林至涵|url=http://www.youtube.com/watch?v=TcClb0-cEDI}}</ref>
