'''張誠'''（{{bd|1960年|6月2日}}），[[中華民國陸軍|中華民國陸軍]][[上校|上校]]，[[臺灣|臺灣]][[外省|外省]][[客家人|客家人]]，也是台灣的火箭科學家、監獄教誨師、[[國立中央大學|國立中央大學]][[教授|教授]]與政治人物，曾是[[國軍|國軍]][[雄風三型反艦飛彈|雄三飛彈]]專案總工程師。[[上校|上校]]28年退伍後曾代表民國黨參選立委。現任[[國立中央大學|國立中央大學]]企業管理學系兼任助理教授、[[中華民國|中華民國]]解癮戒毒協會資深顧問。畢業於[[美國|美國]][[伊利諾大學|伊利諾大學]]電腦博士、[[國立清華大學|國立清華大學]]計算機研究所碩士、[[中正理工學院|中正理工學院]]數學系學士。

== 家庭 ==
張誠的父親為[[福建省|福建省]][[長汀縣|長汀縣]]的[[客家人|客家人]]，母親則是出生於[[日治時期|日治時期]]的[[臺灣|臺灣]][[本省人|本省人]]。

==早年生涯 ==
2011臺北世界設計大展中以「天盾」、「心靈的天堂」<ref>[https://www.youtube.com/watch?v=cj0QmWuXzjk 張誠博士在2011臺北世界設計大展]</ref>，是唯一一位以台灣科學家身分受邀參展。

曾在[[美國太空總署|美國太空總署]]（NASA）跟隨[[丁肇中|丁肇中]]博士（諾貝爾物理學獎得主）進行太空研究計畫的臺裔科技[[軍官|軍官]]，回國後擔任[[中華民國國防部|中華民國國防部]][[軍備局|軍備局]][[中山科學研究院|中山科學研究院]]「雄風計畫」（[[雄風三型反艦飛彈|雄風三型反艦飛彈]]）的專案總工程師。

=== 毒品防治 ===
{{Refimprove section|time=2018-09-27T13:54:53+00:00}}
有感於[[毒品|毒品]]問題對國家社會造成的危害，張誠退休後積極投入毒品防治工作。

==== 解癮戒毒 ====
張誠於2015年加入中華民國解癮戒毒協會，進入監獄擔任教誨師，為毒品受刑人辦理解癮戒毒課程，以期解除身癮和心癮，出獄後能回歸正常生活，

擔任台北地檢署榮譽觀護人，以每週定期晤談的方式，陪伴200餘位吸毒個案走出毒品陰霾，重拾人生。

==== 預防宣導 ====
2016年6月20日，中華民國解癮戒毒協會在桃園市龍潭區中興社區發展協會對社區年長老人三十餘人進行反毒分享巡迴講座，本梯次由解癮戒毒協會資深顧問，也是台北地檢署榮譽觀護人張誠博士對阿公阿嬤們進行反毒宣講。<ref>[http://www.sinatimes.tw/?tag=%E5%8F%8D%E6%AF%92%EF%BC%8C%E5%BC%B5%E8%AA%A0%EF%BC%8C%E8%A7%A3%E7%99%AE%E6%88%92%E6%AF%92%EF%BC%8C%E6%AF%92%E5%93%81 古美珠.我的家鄉 真善美、沒有毒品-兩岸時報總社-2016-6-20]</ref>

2017年，解癮戒毒協會理事張誠受邀至台北及桃園十餘所中學及多個社區進行反毒宣導，介紹毒品防治觀念，並透過個案親身分享，期望讓學子遠離毒品危害。

==== 反毒情境桌遊 ====
張誠認為反毒教育是毒品防治工作的源頭，只要無人吸毒，自不必緝毒與戒毒。然而政府一直以來的宣導方式多以辨識毒品為主，然毒品種類及偽裝難以預料，成效頗為有限。故開始構思有否其他方式能有效達成預防宣導之目的。

在輔導吸毒個案的過程中，張誠發現每位個案染上毒癮的過程雖有不同，但背後原因卻可歸納出幾項共通點。於是決定將實際案例的情境融入遊戲中，製作「情境體驗」式的反毒桌遊。

2018年7月21日，張誠博士創意企劃團隊於平鎮圖書館舉辦「《[[遠離毒品陷阱|遠離毒品陷阱]]》反毒情境桌遊首發會」，歷時一年多研發，台灣首款以「情境模擬」式反毒教育桌遊，正式亮相。

透過情境，玩家可觀察自己的思考模式，暸解自己可能染毒的原因，進而自我警愓，真正遠離毒品陷阱。首發會後，獲得廣大迴響，兩個月內有500人以上參與體驗，其中更有數十人報名培訓，成為首批種子教師。

校園推廣：107學年度第一學期，桃園市內所有高中職將以此桌遊融入反毒宣導。[[國立中央大學|中央大學]]並開設服務學習課程，提供450位學生接受「反毒大使」培訓，培訓完成後將進入各級學校，推動全新的毒品防治教育浪潮。

=== 科技教育 ===
==== 亞洲．矽谷論壇 ====
2017年1月12日，張誠與其創意企劃團隊於桃園市政府舉辦「亞洲．矽谷論壇」，由未來國防、城市交通與情緒控管三個面向，探討「[[工業4.0|工業4.0]]」下的科技趨勢將如何改變人們的生活。

==== 運算思維聯盟 ====
有鑑於108學年度課綱中，程式教育將向下延伸至國小階段，其中可能包含的隱憂，張誠於2018年4月發起「桃園市運算思維聯盟」，倡導「學程式前，先學思維」，為程式教育作好奠基。

其所指「運算思維」是以程式邏輯為基礎的問題處理模式，分為「問題折解」、「抽象歸納」、「尋求規則」、「設計解法」四大步驟。

2018年5月起，展開「小小科學家推動計劃」，以智高積木和海霸桌遊為教材，設計「動力能源」及「移動控制」兩項主題，辦理適合小學三至六年級學生參與的科學體驗營。由講師和助教義工帶領學童實際操作，培養觀察、合作及解決問題的能力。<ref>[https://www.ettoday.net/news/20180523/1175133.htm 桃市舉辦「2018動力能源科學體驗營」，5/27續辦第二場-ETtoday新聞網-2018-5-23]</ref>

營隊中學生實作的同時，家長也會參與「運算思維面面觀」講座，由張誠主講，介紹運算思維的概念及教育趨勢，讓教育落實到「整個家庭」。營隊活動廣受好評，不僅受到學校及家長會支持，亦有多位家長參與後一起投入義工行列，「聯盟」逐漸發展茁壯。

=== 共享經濟 ===
{{Refimprove section|time=2018-09-27T13:52:57+00:00}}
==== 社團＆讀書會 ====
2016年中旬起，張誠的學生及一同從事公益活動的義工們，原本只是彼此間興趣交流，逐漸形成各式社團，包含英文學習、日文歌唱、攝影、桌遊、蠟筆板畫、花藝、植物[[嫁接|嫁接]]、籃球、髮夾手作、皮件製作…等，隨著參與同好愈來愈多，各社團也開始舉辦小型社區活動與民眾交流。

2016下半年至2017年，張誠義工團隊於桃園市平鎮區各鄰里舉辦多場讀書會，主題包括健康飲食、運動養生、血管保健、自我探索、情緒管理…等，以一次分享一本好書的方式，推廣閱讀風氣。

隨著社團活動及讀書會的廣受好評，逐漸成為後來「就是市集」的契機。

==== 好書交換-推廣閱讀 ====
「大量的閱讀能帶來良好的思維」是張誠對提倡閱讀的信念。從讀書會開始，張誠發現不論老少，其實很多人曾經都是願意閱讀、喜歡閱讀的，而民眾也認同閱讀的重要性。於是開始構思創立一個平台，讓更多人能一起交流，找回閱讀風氣。每個人會買的書對自己都曾深俱價值，也許現在不會再讀了，何不把這些好書拿來和別人「交換」？

2017年3月起，張誠帶領推動博愛關懷協會義工群，發起「你無用，我良品」二手好書交換活動，利用假日於公園、里民活動中心等地設攤收書換書。

至2018年9月，平鎮地區已有十餘處地點，每週末固定時間擺設二手書交換攤位，時間一到，男女老少紛至遝來，共享閱讀樂趣。「週末一起來看書」的風氣已逐漸成為在地生活的一環。

==== 「All hands 就是市集」 ====
2017年4月23日，「All handa 就是市集」第一次大型活動於平鎮湧光里集會所登場。以好書交換為主軸，結合了包含繪畫體驗、手作DIY、語文學習、花藝美學…等十餘項社團攤位，兩小時的活動，近兩百位當地民眾一同參與。

2017年12月24日，張誠帶領義工團隊 於平鎮文化公園舉辦「公益書香狂音樂-平鎮人的聖誕夜」活動，除了好書交換及文創社團，更全程加入了在地音樂舞蹈團體登台演出，包含國小音樂班，音樂教室、肚皮舞演出、地方樂團等，不一樣的聖誕夜，難得的文化饗宴。

2018年1月1日，於平鎮義民廟舉辦「全家一起來升旗-平鎮人的元旦升旗趴」活動。寒風中，親子齊唱國歌升旗，抖擻精神；文創氛圍裡，和樂溫馨。

2018年2月11日，於平鎮三崇宮廣場舉辦「平鎮人的旺旺嘉年華」，除了就是市集原有的書香藝文元素，此後也開始與桃園市政府防疫處合作推動狗狗認養，推廣愛護動物理念。

2018年4月29日，於平鎮福星宮廣場舉辦「青銀共煮-惜食、福利、平鎮人」活動，從此市集又加入推廣惜食愛物，世代情感交流的理念，此次活動更獲得平鎮家樂福響應，免費提供即期食材及義工人力協辦活動。

2018年8月12日，張誠率領義工團隊於平鎮新勢公園舉辦「青銀共煮x就是市集」活動，除了原有的好書交換、文創攤位、寵物認養、青銀共煮、在地團體表演外，更加入了運算思維、反毒桌遊、兒童科普等內容。還邀請到世界皮舞冠軍-[[郭淑貞|郭淑貞]]老師及東安國小獨輪車隊到場表演。四個小時的活動，吸引近兩千餘人次參與，儼然是一場嘉年華盛會。<ref>[http://www.epochtimes.com/b5/18/8/13/n10634055.htm 徐乃義.大量的閱讀創造良好的思維 串聯惜食與共享-大紀元-2018-08-13]</ref>

== 發言與主張 ==
* 2016年7月2日，前中山科學研究院雄風計劃上校總工程師張誠，接受中評社訪問表示，這次海軍誤射雄三飛彈打穿漁船，絕對是軍紀與士氣問題。另外，他也強調，台灣武器研發能力，做了很多的事情，比外界想像還棒。<ref>[http://www.chinatimes.com/realtimenews/20160702006339-260417 丁世傑.前雄三工程師張誠：台灣研發能力比外界想像棒-中時電子報2016-07-02]</ref>
* 2017年9月11日，桃園市政府下午與中央大學舉行“科技反毒　智慧解癮”合作記者會，參與校園反毒宣導的張誠也出席，會後接受中評社訪問。有關“教育部”高中課綱審議引發文言文與白話文比例爭議，他認為最核心還是“禮義廉恥”這四個字，台灣什麼都不缺，特別是不欠缺衝突，因此文白不是重點，如果連禮義廉恥都認為“那是中國的”，問題才嚴重。<ref>[http://hk.crntt.com/crn-webapp/touch/detail.jsp?coluid=7&kindid=0&docid=104808425 黃文杰.民國黨張誠：如果連禮義廉恥都切割就嚴重了-中評社-2017-09-11]</ref>

* 2018年3月20日，中國大陸社群網上近日流傳多張圖片，顯示停泊於湖北武昌造船廠內的072Ⅲ型登陸艦「海洋山」號（舷號936），艦艏安裝了疑似電磁砲的武器。何謂電磁砲？電磁砲在軍武科技中的定位？對台海的武力平衡是否具有突破性的威脅？國軍又該如何因應？<ref>[https://www.ettoday.net/news/20180320/1130357.htm?feature=profile_066&tab_id=904 領先全球？解放軍的電磁砲疑上了日軍艦《ETtoday新聞雲》-2018-03-20]</ref>

* 2018年4月9日，當潛艦國造遇到美國行銷核准<ref>[https://www.ettoday.net/news/20180409/1146415.htm?feature=profile_066&tab_id=904 當潛艦國造遇到美國行銷核准《ETtoday新聞雲》-2018-04-9]</ref>

* 2017年5月5日 草根影響力文教基金會針對毒品議題進行民意調查，並進入軍中、校園等場域詢問民眾看法，解癮戒毒協會資深顧問張誠指出，吸毒有分「身癮」跟「心癮」，身癮雖透過隔離可能慢慢減弱強度，但心癮難解，目前對勒戒者採用傳統的靜坐療法，發現可以讓有毒癮的人心中慢慢鬆掉，正在實驗推廣。<ref>[http://news.ltn.com.tw/news/society/breakingnews/2057806 吳柏軒.毒品問題多嚴重？ 民調：近9％民眾身邊有人濫用-2017-05-05]</ref>

* 2018年6月19日 工業合作的原始精神，是政府向國外大規模採購時，在合約中要求賣方，必須依採購案總金額的某一特定比例，向我國的供應商採購部分的零件，將對外採購的金額部分回流到台灣的產業，也將台灣產業融入全世界軍火市場的供應鏈中，以扶持國內產業發展。<ref>[https://www.ettoday.net/news/20180619/1193969.htm?feature=profile_066&tab_id=904 工業合作：台灣進入全球軍武供應鏈的任意門 《ETtoday新聞雲》2018-06-19]</ref>

* 2018年7月3日 國軍一方面強調裝備性能提升、國防科技自主，另一方面，有了美國的大力支持，台灣是否就可以高枕無憂？「國防部長們」是否查覺到國軍在精神戰力、訓練、編現比、後備動員等已經呈現的窘況？<ref>[https://www.ettoday.net/news/20180703/1203914.htm?feature=profile_066&tab_id=904 美軍參與台灣軍事演習，國防部長看不到的真相？《ETtoday新聞雲》2018-07-3]</ref>

* 2018年8月16日，雄三飛彈前總工程師、民國黨軍事國防顧問張誠接受中評社訪問表示，台美關係升溫，因為台灣戰略位置，他以“剪刀”、“石頭”、“布”比喻中美台，中美貿易大戰就是剪刀對石頭，美方用“國防授權法”的石頭阻擋中國大陸的剪刀，但是台灣戰略位置會改變，應該發揮“布”的功能，把石頭包住。<ref>[http://www.crntt.tw/doc/1051/5/8/7/105158785.html?coluid=253&kindid=14951&docid=105158785 黃文傑.雄三飛彈前總工程師張誠：國防授權法模糊-中評社-2018-08-16]</ref>

* 2018年10月11日，張誠博士於臉書進行直播，談論觀塘電廠及能源議題。他表示藻礁屬於全人類，不屬於中華民國。藻礁之於地球如同腎臟之於人類，沒有千年藻礁將失去健康的地球。而此事件揭示人民對國家的不信任感已崩盤，這種情況比藻礁事件更嚴重。<ref>[https://www.facebook.com/PhDChang/videos/274237103202407/ 桃園觀塘電廠議題發言精華 2018-10-11]</ref>

* 2018年10月，桃園市平鎮區市議員候選人張誠受南桃園新聞採訪，表示教育才是反毒根本，而目前政府反毒宣導著重於認識及辨識毒品，卻未教導青少年可能於何種情況下染毒，缺乏反毒情境教育。於是他設計出反毒桌遊，透過模擬的方式，讓青少年於遊戲中檢視自己的思考模式，進而遠離毒品陷阱。<ref>[https://www.facebook.com/PhDChang/videos/397421180795572/?__xts__%5B0%5D=68.ARDFdHXSC6VCptK46pMY9dSy5kXydazOvfSN0DtqH652JzEmX-C2FOL90_UmI72d8Jq2KAUuss6a0cP2Emum99ZsrQCqx1EpvmMAXSnZGtL_6T3HdGZZnM7toNKLGbOOdhajSxClGhxDHf5jQkT3FNM9HraDijXLBbfcrJli0o6niwqAnZmHKAi6cmMIW51mXqCdblShc6CcU94n6igy5GHhLMuOJuej8ps_lYlw4_lrphxyeve9ohMSxvArNGZu7XEmBDxooJcMK0dQXdLBEze7AsVRfftSL1I4QKva_CobdwqsQhwb8q-AGI6F12565zYLX0qW07EffzwdgJCFMCPckjuRs1xSN3pXjuI&__tn__=H-R 關心毒品議題 候選人張誠設計反毒桌遊-南桃園新聞採訪-2018-11-04]</ref>

<ref>[https://video.udn.com/news/404364 民國黨黨主席徐欣瑩今天晚上赴桃園平鎮義民廟，幫黨籍立委候選人張誠站台，她說張是國防科技博士，是好人材，應該被民眾送進立法院。]</ref><ref>[http://news.ltn.com.tw/news/local/paper/936302 為張誠站台 徐欣瑩：再創經濟奇蹟]</ref>

* 2018年10月10日，適逢107年國慶，張誠帶領義工團隊於平鎮區各地發放國旗，並於競選總部舉辦國家慶生會，上百名支持民眾齊聚現場。他表示感激國家栽培之恩，從擔任中科院雄三飛彈總工程師退伍後，希望仍能對國家社會有所貢獻。從擔任台北地檢署榮譽觀護人起，到推動各項社會公益等持續努力；而現在台灣人民，普遍對政府不信任，他呼籲民眾把愛國熱情拿出來，沒有國哪有家，全民一起愛國家、愛國旗、愛國歌。<ref>[http://news.tnn.tw/news.html?c=1&id=130222 上百人擠爆競總　張誠辦國慶會盛況-TNN台灣地方新聞網-2018-10-11]</ref>
