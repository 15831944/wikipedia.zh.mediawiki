'''張以仁'''，[[銘傳大學|銘傳大學]]傳播管理研究所碩士畢業，2008年加入[[中視新聞|中視新聞]]，主跑社會線，2017年6月23日至今是《台灣新聞眼》節目主持人

== 學歷 ==
*中國文化大學大眾傳播系 學士
*銘傳大學傳播管理研究所 碩士
== 經歷 ==
*2008年：加入[[中視新聞|中視新聞]]主跑社會線記者
*2009年：八八風災採訪
*2010年：國道走山採訪
*2011年：日本311大地震採訪
*2012年至今：新聞播報工作
*2017年3月20日至2017年12月29日與[[盧秀芳|盧秀芳]]主持[[了解與互信兩岸一旺|了解與互信兩岸一旺]]
*2017年6月23日至今~主持中視新聞台[[台灣新聞眼|台灣新聞眼]]
*2017年7月17日至2017年9月25日擔任[[中視1800大飆新聞|中視1800大飆新聞]] 社會大現場特報主播
*2017年8月19日至2017年8月30日[[世界大學運動會|世大運]]看中視特報主播
*2017年:採訪川普狂人亞洲行全球前途解碼系列特別報導
*2017年:採訪大陸丹東系列特別報導
*2018年:採訪和平新開始直擊兩韓高峰會特別報導
*2018年:採訪狂人聚星川金拚核平系列特別報導
*2018年:擔任雙十國慶大典戶外主播
*2018年12月1日至2019年2月23日主持中視[[夢想大學堂|夢想大學堂]](第四季)

== 現任 ==
