'''張寶華'''（{{lang-en|Sharon CHEUNG Po-Wah}}<ref>{{Cite news|url=http://www.com.cuhk.edu.hk/ubeat_past/071180/award/Untitled-16.html|title=張寶華:感覺從沒有離開過新聞界|author=周曉暘|date=|work=新闻与传播学院|accessdate=2017-01-28}}</ref>）<ref>[https://theinitium.com/article/20160817-mainland-sharoncheung/ 十六年前那次採訪，讓她成了「膜蛤教主」｜大陸｜端傳媒]</ref>，曾改名'''張洺華'''<ref name="a">{{Cite web|url=http://eastweek.my-magazine.me/?aid=34471|title=《東周刊656期》Too smart, sometimes cautious... 張寶華|date=2014-06-25|accessdate=2015-03-26}}</ref>，籍貫[[上海|上海]]，出生于[[香港|香港]]，曾任電視新聞記者，因樣貌標緻，被称为“新聞之花”、“寶娃娃”。現为一介傳媒CEO，曾任上市公司管理層、電台節目主持人及專欄作家。访问过许多名人，包括[[江澤民|江澤民]]、[[金庸|金庸]]、[[羅傑承|羅傑承]]、[[劉德華|劉德華]]、[[鄭秀文|鄭秀文]]、[[黎智英|黎智英]]、[[唐英年|唐英年]]、[[梁振英|梁振英]]、[[陳可辛|陳可辛]]、[[王維基|王維基]]等。曾為[[信報財經新聞|信報]]、[[經濟日報|經濟日報]]、[[蘋果日報|蘋果日報]]、[[星島日報|星島日報]]、[[東周刊|東周刊]]及[[明報|明報]]等多份報章雜誌撰寫專欄，亦有多部著述。現時是[[香港足球總會|香港足球總會]]市務及傳訊委員會成員<ref>[http://www.hkfa.com/ch/load_page/5/ 香港足總網頁] {{Wayback|url=http://www.hkfa.com/ch/load_page/5/ |date=20160628212248 }} 香港足總網頁</ref>。

== 個人簡歷 ==
張寶華小學畢業於[[聖公會聖米迦勒小學|聖公會聖米迦勒小學]]，中學畢業於[[聖馬可中學|聖馬可中學]]。<ref>{{web reference|URL=http://eastweek.my-magazine.me/main/2799|title=走過的青葱歲月|work=東周刊 第310期 Book B 【名人講場】|year=2009年|date=8月5日}}</ref>1995年畢業於[[香港中文大學|香港中文大學]]新聞及傳播學院。畢業後曾在[[電視廣播有限公司|無綫電視]][[無綫新聞|新聞及資訊部]]當實習記者，後轉職[[南華早報|南華早報]]，任政治版记者。<ref name="cp3002">{{cite web |url=https://www.cp1897.com.hk/product_info.php?BookId=9789882003002 |title=《Libeal系列——生命中該有的（第二版）》作者簡介|}}</ref>

1998年加入[[香港有線電視|有線電視]][[有線新聞|新聞部]]，至2005年期间任中国组高级记者，專責報道兩岸政治新聞。<ref>{{Cite web|title=Good Journalism is a Tough Job|url=http://varsity.com.cuhk.edu.hk/varsity/0405/people_sharon.htm|work=varsity.com.cuhk.edu.hk|accessdate=2019-02-17|language=en|month=4|year=2004|author=|date=|publisher=}}</ref>

2000年10月27日，[[董建華|董建華]]赴北京述職時，張寶華反複追問當時的國家主席[[江澤民|江澤民]]是否「欽點」董建華連任[[香港特別行政區行政長官|香港特別行政區行政長官]]，江澤民大發雷霆，並公開以「too simple，sometimes naive.」(“太簡單，有時很幼稚”，即“很傻很天真”的較早版本)斥責，使她名噪一時。詳見[[#与膜蛤文化的关系|下文]]。

她曾任[[香港記者協會|香港記者協會]]執行幹事及[[香港大學|香港大學]]校外進修學院新聞系講師。

2003年獲香港中文大學新聞學院頒發「傑出表現獎」。

2005年獲選為[[路透社|路透社]]基金會研究學人。同年10月，考獲「志奮領留英獎學金」，赴[[英國|英國]][[牛津大學|牛津大學]]進修新聞，一年後回港。

2007年6月1日起，張寶華成為[[豐德麗控股有限公司|豐德麗控股有限公司]]的高級副總裁，專注業務發展。

2008年[[汶川大地震|汶川大地震]]發生後，張寶華跟[[劉德華|劉德華]]分別當上[[演藝界512關愛行動|演藝界512關愛行動]]的發言人跟召集人，為災民籌款；其後有關工作轉交[[香港神學院|香港神學院]]負責。<ref name="cp3002" />

2009年3月，正當香港[[豬流感|豬流感]]及經濟低迷時，張寶華推出香港首個免費正版電影網站goyeah.com，網站的收入來源主要是靠[[網上廣告|網上廣告]]。當時她跟傳媒說，香港的內容商走得太慢，給盜版商盜取了接近八成正收入，所以goyeah.com是一個自救的運動<ref>[http://hk.apple.nextmedia.com/news/art/20090421/12663336 張寶華：娛樂人行得太慢]</ref>。

2013年，張寶華任[[寰亞傳媒集團|寰亞傳媒集團]]高級副總裁。

2013年8月14日，張寶華接受[[陳志全|陳志全]]的網台節目《慢談風月》訪問，談到為何與[[蔡子強|蔡子強]]交惡？<ref>{{cite web |url=https://www.youtube.com/watch?v=EQsFh-BkLSo |title=張寶華為何與蔡子強交惡？〈慢談風月〉2013-08-14 |author= |date=2013-08-14 }}</ref>

2015年張寶華成立奖学金，每年資助一名香港在職記者到牛津升學，接受「路透社新聞研究中心」提供的專業新聞培訓，并希望獎學金日後能擴展到中國大陸。獎學金運作兩年，分別資助了一名明報、一名蘋果日報記者及一名南華早報記者到牛津大學升學<ref>[http://www.am730.com.hk/article-263345 牛津「獅子山精神獎學金」 張寶華資助港記者深造]</ref>。

2015年9月，張寶華正式擔任[[香港中文大學|香港中文大學]]新聞及傳播學院講師，並於2016年7月把學程結合出版了《BRAND U》，並打入了[[誠品書店|誠品書店]]的暢銷書榜。

2015年5月11日，張寶華於[[信報|信報]]撰文支持通過[[2016年及2017年香港政治制度改革|政改方案]]<ref>[http://www.post852.com/%E5%BC%B5%E5%AF%B6%E8%8F%AF%E6%94%AF%E6%8C%81%E8%A2%8B%E4%BD%8F%E5%85%88%E3%80%80%E3%80%8C%E8%AE%8A%EF%BC%8C%E5%A5%BD%E9%81%8E%E4%B8%8D%E8%AE%8A%E3%80%8D/ 張寶華支持袋住先　「變，好過不變」]</ref>。

2015年8月，張寶華開始擔任[[香港數碼電台|香港數碼電台]]時事節目主持，并主持[[清談節目|清談節目]]《最好的時光》。

2016年11月，張寶華舉行第一個個人油畫展，名為「色繪.女人世界」一眾名人嘉賓出席，畫展一共籌得25萬元，全數捐給慈善機構「惜食堂」。

== 逸事 ==
2010年4月，張寶華發表網誌，回應[[梁家權|梁家權]]《[[蔡子強|蔡子強]]不壞》一文<ref>{{cite news|url= |title=非佬正傳：蔡子強不壞 |publisher=星島日報 |author=梁家權 |date=2010-04-12 }}</ref>，她指出「那位講師曾經是我的朋友」，「那位講師沒人愛，不是他壞，而是他小家氣。」<ref>{{cite web |url=http://cheungpowah.blogspot.hk/2010/04/blog-post_14.html |title=男人不壞，女人不愛? |author=張寶華 }}</ref>

2013年8月14日，張寶華接受陳志全的網台節目《慢談風月》訪問，談到愛情生活，同時解釋為何與[[蔡子強|蔡子強]]交惡。<ref>{{cite web |url=https://www.youtube.com/watch?v=EQsFh-BkLSo |title=張寶華為何與蔡子強交惡？〈慢談風月〉2013-08-14 |author= |date=2013-08-14 }}</ref>

== 与膜蛤文化的关系 ==
{{seealso|膜蛤文化|江泽民怒斥香港女记者}}
2000年10月27日，[[香港行政長官|香港行政長官]][[董建華|董建華]]赴[[北京|北京]]述職時，張寶華询问[[中國共產黨中央委員會總書記|中共總書記]][[江澤民|江澤民]]是否支持董建华连任[[香港特別行政區行政長官|特首]]，在获得肯定的回答后继续追问江的表态是否有“钦定”的意味。江泽民不悅，并大声斥责張寶華“跑得比西方記者還快”、“问的问题 [[q:江泽民|too simple，sometimes naive.]] (太簡單，有時很幼稚)”，成为张宝华记者生涯中的一个重要事件。<ref>{{Cite web|url=http://www.com.cuhk.edu.hk/ubeat_past/071180/award/Untitled-16.html|title=張寶華:感覺從沒有離開過新聞界|accessdate=2016-05-17|work=www.com.cuhk.edu.hk}}</ref>

伴随2014年前後兴起的[[膜蛤文化|膜蛤文化]]，张宝华因为这段流传于互联网的视频再次成为大陆网络热点，张江两人的对话被网民大量引用、调侃。张宝华于2010年开通的[[新浪微博|新浪微博]]也随之获得了大量关注，并被称为“蛤丝”（意为江泽民的粉丝）口中的“[[教主|教主]]”。2016年1月，张宝华因參加活動到[[成都市|成都]]<ref>{{Cite web|url=http://vancouver.singtao.ca/400108/2016-01-16/post-kelly-online%EF%BC%9A%E6%88%90%E9%83%BDifs%E5%85%A9%E9%80%B1%E5%B9%B4%E6%99%9A%E5%AE%B4%25E3%2580%2580%E5%BC%B5%E5%AF%B6%E8%8F%AF%E5%96%9C%E7%8D%B2%E5%A4%A7%E7%8D%8E/?variant=zh-cn|title=Kelly Online：成都IFS两周年晚宴　张宝华喜获大奖|accessdate=2016-06-07|work=vancouver.singtao.ca}}</ref>，事先於微博召集粉絲聚會<ref>{{Cite web|url=http://www.weibo.com/1847856024/DbRsp1y36?type=comment#_rnd1465269354476|title=張寶華新浪微博|accessdate=2016-06-07|work=www.weibo.com}}</ref>，最終超過100人參加，其中大部分為“蛤丝”<ref>{{Cite web|url=http://chinadigitaltimes.net/chinese/2016/01/%E3%80%90%E9%BA%BB%E8%BE%A3%E6%80%BB%E5%B1%80%E3%80%91%E5%85%A8%E7%90%83%E9%A6%96%E5%B1%8A%E8%9B%A4%E4%B8%9D%E5%A4%A7%E4%BC%9A%E6%88%90%E9%83%BD%E4%B8%BE%E8%A1%8C/|title=【网通社】全球首届蛤丝大会成都举行|accessdate=2016-06-07|date=2016-01-13|work=中国数字时代|language=zh-CN}}</ref>，參加者甚至遠自日本而來。对于张宝华所发布的跑步照或自拍照等，“蛤丝”常以“跑得快”、“too young”、“是[[彭定康|彭定康]]说的”等来源于上述采访视频的固定句式予以回覆。

2017年11月，張寶華在重慶市舉行"brand u"簽名會，超過50位大學生參加。
