'''张荫棠'''（{{bd|1864年||1937年|7月7日|catIdx=Z张}}），[[表字|字]]朝弼，[[号|号]]少卿，又号憩伯，[[中国|中国]][[广东|广东]][[新会|新会]][[双水|泷水]]豪山乡人。曾以灼文为名中举。著名外交家，[[清朝|清朝]]最后一位驻[[美国|美国]]公使，也是[[中华民国|中华民国]]首位驻美国公使。

==生平==
早年以'''张灼文'''为名中举，在[[北京|北京]]候补。[[光绪|光绪]]十八年（1892年）捐官为内阁中书，次年考取海军衙门[[章京|章京]]，后任驻[[日本|日本]]参赞。光绪二十二年(1896年)，张出任中国驻美国使馆三等参赞兼驻[[旧金山|旧金山]]领事，不久调往[[西班牙|西班牙]]，任[[代办|代办]]。

光绪二十八年（1902年），[[英国|英国]]入侵[[西藏|西藏]]，张随同[[唐绍仪|唐绍仪]]赴[[印度|印度]]与英国谈判，结果失败。光绪三十二年(1906年)，再次前往印度，参加关于修订《[[拉萨条约|拉萨条约]]》的谈判。在谈判中，张上奏政府，认为西藏局势“岌岌可危，英人觊觎已非一日”，呼吁政府尽快重视西藏问题。同年4月14日，张被破格任命为[[駐藏大臣|驻藏帮办大臣]]，10月12日到任，迅速稳定了因[[第十三世達賴喇嘛|十三世达赖喇嘛]]外逃而混乱的[[拉萨|拉萨]]局势。在西藏期间，张兴办教育，进行改革，于1907年1月13日提出了《[[新治藏政策大纲|新治藏政策大纲]]》（《治藏刍议十九条》），主要内容有：

# 优待[[达赖|达赖]]、[[班禅|班禅]]，恢复藏王制，以汉官监督；
# 清查户口、租赋；
# 筹饷练兵，修筑交通，兴办教育；
# 振兴农工商业，开发矿产，革除苛政，废除[[乌拉差役|乌拉差役]]，设立[[银行|银行]]；
# 改良风俗，办理一切“[[新政|新政]]”；
# 设置西藏行部大臣、会办大臣等，分治外交、督练、财政、学务、盐茶、巡警、农务、工商、路矿等9局事务；

张还弹劾了原驻藏大臣[[有泰|有泰]]。这些措施激起了西藏农奴主们的恐慌和对抗，也引发了英国政府的抗议，新任驻藏大臣[[联豫|联豫]]也对张多有猜忌。张被迫于光绪三十三年（1907年）五月离开西藏，负责与英国谈判《[[修订藏印通商章程|修订藏印通商章程]]》。光绪三十四年（1908年），张回京后授外务部右参赞。

[[宣统|宣统]]元年（1909年）6月27日，张被任命为驻美国、[[墨西哥|墨西哥]]、[[秘鲁|秘鲁]]和[[古巴|古巴]]四国公使。在任期间他曾成功与墨西哥政府交涉，为在当地排华风潮中被打死的中国民工争取到310万[[墨西哥比索|墨西哥比索]]的赔偿。

1911年10月10日，[[辛亥革命|辛亥革命]]爆发，张支持革命运动，遂称病辞去公使职务，转而接受[[孙中山|孙中山]][[中华民国|中华民国]]政府的任命，担任驻美国外交代表。清政府虽然任命[[施肇基|施肇基]]继任驻美公使，但是施并未到任。1913年，因为[[西姆拉會議|西姆拉会议]]召开在即，民国政府任命[[夏偕复|夏偕复]]继任驻美公使，将张调回国内，拟派作中方代表与会，但因英国坚持拒绝未果。

张此后拒绝了[[袁世凯|袁世凯]]的邀请，赋闲在家，1937年7月7日，于[[七七事变|七七事变]]当天在北京去世。
