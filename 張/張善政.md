'''張善政'''（{{bd|1954年|6月24日||}}），[[中華民國|中華民國]]政治人物，[[美國|美國]][[康乃爾大學|康乃爾大學]]土木與環境工程學[[博士|博士]]。曾任[[行政院國家科學委員會|行政院國家科學委員會]][[國家高速網路與計算中心|國家高速電腦中心]]主任、[[Google|Google]]有限公司亞洲硬體營運總監、[[行政院|行政院]][[政務委員_(中華民國)|政務委員]]、[[中華民國科技部|科技部]]部長<ref name="科技部長cna20140226">{{Cite news |url=http://www.cna.com.tw/topic/popular/4324-1/201402260016-1.aspx |title=張善政出任首任科技部部長 |publisher=中央社 |date=2014-02-26 |access-date=2014-02-26 |archive-url=https://web.archive.org/web/20160306021454/http://www.cna.com.tw/topic/popular/4324-1/201402260016-1.aspx |archive-date=2016-03-06 |dead-url=yes }}</ref>、[[行政院副院長|行政院副院長]]以及[[行政院院長|行政院院長]]。

2014年12月8日，出任[[毛治國內閣|毛治國內閣]]的行政院副院長一職。<ref>[http://www.cna.com.tw/news/aipl/201412050194-1.aspx 張善政善溝通 務實前瞻利開創]，中央社，2014年12月5日</ref><ref>[http://www.cna.com.tw/news/firstnews/201412055010-1.aspx 政院公布內閣名單 閣員多留任]，中央社，2014年12月5日</ref><ref>[http://udn.com/NEWS/NATIONAL/NATS2/%E6%AF%9B%E5%85%A7%E9%96%A3%E5%BA%95%E5%AE%9A%20%E5%89%AF%E9%96%A3%E6%8F%86%E5%BC%B5%E5%96%84%E6%94%BF-9111671.shtml 毛內閣底定 副閣揆張善政]，聯合報，2014.12.05</ref>2016年1月18日，由於[[毛治國|毛治國]]提出辭呈，張善政以副院長身分代理其職務。2016年2月1日，張接任行政院院長<ref>[http://www.techbang.com/posts/41062-first-acer-google-took-over-as-executive-president-zhang-shanzheng-took-office-on-february-1 首位宏碁、Google人接任行政院長，張善政2月1日上任！]</ref>，他是[[馬英九政府|馬英九政府]]任內最後一個行政院長，也是首位以[[無黨籍|無黨籍]]正式上任的行政院長，5月20日卸任，任期約109天。<ref>在此之前，只有1963年同樣曾於行政院副院長任內代理行政院長的[[王雲五|王雲五]]為無黨籍。</ref><ref>[http://note.taable.com/post/164868/Hit-Fm%E8%81%AF%E6%92%AD%E7%B6%B2-%E5%91%A8%E7%8E%89%E8%94%BB%E3%80%8A%E8%94%BB%E8%94%BB%E6%97%A9/2b58--408--95T464198439503496037- 張善政臨危受命接閣揆 四個月任期「做好做滿」]，蔻蔻早餐</ref>

卸任閣揆後，張善政擔任了[[東吳大學_(臺灣)|東吳大學]]巨量資料管理學院唐松章校友講座教授兼榮譽院長、[[國立交通大學|國立交通大學]]電子與資訊中心客座講座教授兼亥客書院院長、[[中華大學|中華大學]]客座講座教授、[[國立臺北商業大學|國立臺北商業大學]]榮譽講座教授、[[聖約翰科技大學|聖約翰科技大學]]榮譽講座教授、[[國家生技醫療產業策進會|國家生技醫療產業策進會]]會長、智網聯盟總網會長、[[台灣大哥大|台灣大哥大]]董事兼基金會董事長<ref>[http://www.chinatimes.com/realtimenews/20160722002989-260401 張善政新動向 接台灣大哥大基金會董事長]</ref>、[[華邦電|華邦電]][[獨立董事|獨立董事]]及[[宏碁|宏碁]]獨立董事。

==早年==
張善政的父親張同文是[[天津|天津]]人，1948年到[[臺灣|臺灣]]，後於[[台灣中油股份有限公司|中油]]退休，母親是[[臺南|臺南]]人<ref>[http://udn.com/news/story/1/1692313-%E5%BC%B5%E5%96%84%E6%94%BF%EF%BC%9A%E6%88%91%E6%B2%92%E6%9C%89%E8%97%8D%E7%B6%A0-%E6%98%AF%E5%BC%B5%E7%99%BD%E7%B4%99 張善政：我沒有藍綠 是張白紙]，[[經濟日報|經濟日報]]，2016年5月13日</ref>，張則在[[臺北|臺北]]長大。<ref>[http://udn.com/news/story/1/1473985-%E6%86%82%E5%85%A9%E5%B2%B8-%E5%BC%B5%E5%96%84%E6%94%BF%EF%BC%9A%E9%81%B8%E5%BE%8C%E5%85%A9%E5%B2%B8%E7%86%B1%E7%B7%9A%E6%B2%92%E6%8E%A5%E4%B8%8A 憂兩岸 張善政：選後兩岸熱線沒接上]，聯合晚報，2016年1月29日</ref>他因父親認為念理工不要沾政治而沒有加入任何政黨。<ref>[http://news.ltn.com.tw/news/politics/breakingnews/1697427 未來是否參選？ 張善政預留伏筆]</ref>他在[[國立臺灣大學|國立臺灣大學]]土木工程學系工學學士畢業後，前往美國[[史丹佛大學|-{zh-cn:斯坦福大学; zh-sg:斯坦福大学; zh-tw:史丹佛大學; zh:史丹佛大學; zh-hk:史丹福大學; zh-mo:史丹福大學;}-]]進修土木與環境工程學，並於1977年獲得碩士學位。隨後於1981年，獲得美國[[康乃爾大學|-{zh-hans:康乃尔大学; zh-hk:康乃爾大學; zh-tw:康乃爾大學; zh:康乃爾大學;}-]]土木與環境工程學博士學位。<ref>[http://www.cna.com.tw/news/aipl/201402260110-1.aspx 雲端推手 張善政責任重大]，中央社，2014年2月26日</ref>博士論文《An Integrated Finite Element Nonlinear Shell Analysis System with Interactive Computer Graphics》。<ref>[https://dl.acm.org/citation.cfm?id=909978 An integrated finite-element nonlinear shell analysis system with interactive computer graphics], ACM Digital Library, 1981</ref>

在獲得[[康乃爾大學|康乃爾大學]]博士學位後，在1981年至1990年間，獲聘至母校[[國立臺灣大學|國立臺灣大學]]土木工程學系擔任講師，歷任講師、副教授，1983年就任教授。<ref name="ct20120201" />1984年獲頒[[中國青年救國團|中國青年救國團]]青年獎章。<ref>[http://www.storm.mg/article/240397 61年來第一次！總統、閣揆拒接見救國團青年獎章得獎者]，風傳媒，2017年3月28日</ref>1991年，行政院核准成立[[國家高速網路與計算中心|國家高速電腦中心]]{{NoteTag|1992年該中心進駐[[新竹科學園區|新竹科學園區]]，1993年正式對外運轉提供高速計算服務，主管機構為行政院[[國家科學委員會|國家科學委員會]]。}}；1991年至1997年，張善政為國家高速電腦中心主任。<ref>[https://www.youtube.com/watch?v=fQrTFR4iVaI 35年的挑戰與危機給我的淬鍊——張善政 前行政院長]</ref>1998年至2000年，張善政為行政院國家科學委員會企劃考核處處長。

張善政受[[宏碁|宏碁]]創辦人[[施振榮|施振榮]]邀請，曾參與籌設宏碁子公司「安碁資訊」。<ref name="ct20120201" />2000年至2010年，張善政為宏碁股份有限公司電子化事業群（電子化資訊管理中心，Acer eDC）副總經理。2010年至2012年，張善政擔任[[Google|Google]]有限公司亞洲硬體營運總監（[[Director|Director]] of Google's hardware operation in Asia），主管範圍為Google亞洲的資料中心營運，為資料中心營運業務亞洲最高主管，直接向美國總公司全球資料中心營運主管負責，任職期間新加坡與香港資料中心動土儀式，都是由他本人以未來資料中心的營運主管身份主持。<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20140611/414170/ 谷歌總監頭銜遭質疑　張善政：外界可檢視]</ref><ref>[https://www.inside.com.tw/2014/06/11/san-cheng-chang-google-job-title 張善政 Google 經歷膨風？翟本喬：低級抹黑]</ref>
