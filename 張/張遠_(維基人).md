'''張遠'''網名暱稱為「'''ffaarr'''」，是[[中文維基百科|中文維基百科]]的[[維基人|維基人]]，在[[臺灣|臺灣]]的[[批踢踢|批踢踢]]則有「'''哆啦王'''」、「'''叮噹王'''」等稱號。他來自[[中華民國|中華民國]][[臺北市|臺北市]]，是[[國立臺灣大學|國立臺灣大學]][[國立臺灣大學歷史學系|歷史學研究所]]博士，在[[德明財經科技大學|德明財經科技大學]]通識中心擔任兼任助理教授。其主要研究領域為[[清朝|清朝]]至近代的性別與通俗文化，著有《近代平津滬的城市京劇女演員1900－1937》、《清中期北京梨園花譜中的性別特質想像》等。

由於經常在批踢踢上回應與日本漫畫《[[哆啦A夢|哆啦A夢]]》有關的奇特問題，張遠成為批踢踢上的知名人物之一<ref name = "陳瑞霖 02">{{Cite web |url = https://technews.tw/2017/11/03/senior-ptter-says-things-about-bbs-ptt-offline/ |title = PTT 斷線 5 天的影響？看看這些資深鄉民怎麼說 |author = 陳瑞霖 |date = 2017年11月23日 |publisher = TechNews 科技新報 |language = Zh-hant |accessdate = 2018年2月15日 |quote =  }}</ref>。同時，他長期參與中文維基百科的運作，擔任[[管理員_(維基百科)|管理員]]與行政員的工作；也長期參與[[台灣維基媒體協會|台灣維基媒體協會]]，並在2018年出任理事長，致力於中文維基百科社群的發展。此外亦是[[Wikia|Wikia]]中文社群發展專員，以及是「哆啦A夢wiki」和「PTT鄉民百科」創辦人之一。他曾和「Lon」兩人合作推出《PTT鄉民大百科》一書，介紹批踢踢[[鄉民_(網路用語)|鄉民]]的流行用語。

== 早年經歷 ==
=== 學術成長 ===
[[File:國立臺灣大學文學院.jpg|thumb]]博士班畢業。]]

張遠來自[[中華民國|中華民國]][[臺北市|臺北市]]<ref name = "Ffaarr 02">{{Cite web |url = https://zh.wikipedia.org/w/index.php?title=User:Ffaarr&oldid=43683719 |title = 使用者:Ffaarr |author =  |date =  |publisher = [[中文維基百科|中文維基百科]] |language = Zh-hant |accessdate = 2018年2月15日 |quote =  }}</ref>，其父親是著名詩人、散文家暨學者[[張健_(詩人)|張健]]<ref name = "徐世澤、邱燮友、張健和徐德智 01">{{Cite book | author = 徐世澤、邱燮友、張健和徐德智 | title = 《並蒂詩風》 | location = 臺灣臺北 | publisher = 萬卷樓圖書股份有限公司 | date = 2011年 | pages = 第149頁至第150頁 | ISBN = 978-9577397331 | accessdate = 2018年2月15日 | url = https://books.google.com.tw/books?id=pWREDwAAQBAJ&printsec=frontcover&hl=Zh-hant#v=onepage&q&f=false | language = Zh-hant | quote =  }}</ref><ref name = "楊菁菁 01">{{Cite web |url = http://news.ltn.com.tw/news/supplement/paper/153438 |title = 最精鍊的語言─張健創作文物展 |author = 楊菁菁 |date = 2007年9月10日 |publisher = 《[[自由時報|自由時報]]》 |language = Zh-hant |accessdate = 2018年2月15日 |quote =  }}</ref>。

張遠從不識字的孩童時期，就開始翻閱[[日本|日本]][[藤子·F·不二雄|藤子·F·不二雄]]創作的著名漫畫《[[哆啦A夢|哆啦A夢]]》<ref name = "蕭歆諺 01">{{Cite web |url = https://www.gvm.com.tw/article.html?id=41666 |title = 哆啦A夢吃過多少個銅鑼燒？問他就知道 |author = 蕭歆諺 |date = 2017年12月26日 |publisher = 《[[遠見雜誌|遠見雜誌]]》 |language = Zh-hant |accessdate = 2018年2月15日 |quote =  }}</ref>。在他小時候，會拿著零用錢去蒐集盜版漫畫，不過小時候的他由於成績不錯，並未遇到如[[野比大雄|野比大雄]]般的課業問題，因此當時不太能夠體會大雄的心情<ref name = "蕭歆諺 01"/>。直到他讀到研究所以後，由於課業的困難度增加，他才發現自己和野比大雄間有著許多相似處，例如大雄經常檢討自己，這點和張遠的個性十分吻合<ref name = "蕭歆諺 01"/>。

張遠喜愛歷史，主修的學科為歷史學<ref name = "Ffaarr 02"/>，學習的研究內容較偏向社會文化史部分<ref name = "陳琡分 01">{{Cite web |url = http://okapi.books.com.tw/article/2472 |title = 《PTT鄉民大百科》Ffaarr+Lon：愛湊熱鬧是鄉民最大的特質 |author = 陳琡分 |date = 2013年10月31日 |publisher = 《[[博客來網路書店|博客來網路書店]]》 |language = Zh-hant |accessdate = 2018年2月15日 |quote =  }}</ref>；而在大學時期，他經常到[[光華商場|光華商場]]找尋漫畫<ref name = "蕭歆諺 01"/>。他的其他興趣還有中國音樂、地理、戲曲、登山、自行車、圍棋、漫畫、棒球比賽等，也曾學過[[嗩吶|嗩吶]]與[[竹笛|竹笛]]<ref name = "Ffaarr 02"/>。在從[[國立臺灣大學歷史學系|國立臺灣大學歷史學研究所]]博士班畢業後，他擔任[[國立清華大學|國立清華大學]]人文社會研究中心博士後研究員<ref name = "博客來網路書店 01">{{Cite web |url = http://www.books.com.tw/products/0010593877 |title = 中國最後的帝國：大清王朝 |author =  |date = 2013年 |publisher = 《[[博客來網路書店|博客來網路書店]]》 |language = Zh-hant |accessdate = 2018年2月15日 |quote =  }}</ref>，爾後轉任[[中國文化大學|中國文化大學]]歷史學系兼任助理教授<ref name = "中國歷史上的婦女與社會 01">{{Cite web |url = http://pub.tust.edu.tw/classics/gender/citizen/paper.html |title = 專題演講 |author =  |date = 2013年 |publisher = 中國歷史上的婦女與社會 |language = Zh-hant |accessdate = 2018年2月15日 |quote =  }}</ref>、[[德明財經科技大學|德明財經科技大學]]通識中心兼任助理教授<ref name = "蕭歆諺 01"/><ref name = "劉嫈楓 01">{{Cite web |url = https://www.taiwan-panorama.com/Articles/Details?Guid=8c961669-a132-4b96-ad55-f598f47e2b17&CatId=1 |title = 台灣最大BBS站 ——PTT鄉民傳奇 |author = 劉嫈楓 |date = 2014年8月 |publisher = 《台灣光華雜誌》 |language = Zh-hant |accessdate = 2018年2月15日 |quote =  }}</ref><ref name = "國立臺灣大學歷史學系 02">{{Cite web |url = https://epaper.ntu.edu.tw/view.php?listid=178&id=24743 |title = 臺大歷史系電子報 No.165 |author =  |date = 2016年6月22日 |publisher = [[國立臺灣大學歷史學系|國立臺灣大學歷史學系]] |language = Zh-hant |accessdate = 2018年2月15日 |quote =  }}</ref>。

=== 網路活躍 ===
[[File:Xinzhujuhui.jpg|thumb]]

張遠的主要研究範圍是中國[[清朝|清朝]]至近代的社會與文化領域，尤其特別關注性別與戲劇等通俗文化的相關議題<ref name = "Ffaarr 02"/><ref name = "蕭歆諺 01"/>。在過去，他曾經發表《近代平津滬的城市京劇女演員1900－1937》、《清中期北京梨園花譜中的性別特質想像》等著作，還和[[李仁淵|李仁淵]]共同翻譯美國歷史學家{{Link-en|威廉·T·羅維|William T. Rowe}}的著作《中國最後的帝國──大清王朝》（China's Last Empire: The Great Qing）<ref name = "國立臺灣大學歷史學系 02"/>。

另一方面，張遠還會研讀現代卡通人物等不同領域，他的書櫃蒐集有《哆啦A夢》20冊正版大全集，且他會不時隨手翻閱，每篇都已經看過上百回<ref name = "蕭歆諺 01"/>。由於他對於《哆啦A夢》的研究相當深入<ref name = "三立新聞台 01">{{Cite web |url = http://www.setn.com/News.aspx?NewsID=125224 |title = 大雄在漫畫中罰站過幾次？「哆啦王」神計算被推爆 |author =  |date = 2016年2月20日 |publisher = [[三立新聞台|三立新聞台]] |language = Zh-hant |accessdate = 2018年2月15日 |quote =  }}</ref>，所以只要花費2個小時就能翻完總共20冊的《哆啦A夢》大全集，甚至只看封面就知道該集有沒有出現[[銅鑼燒|銅鑼燒]]<ref name = "蕭歆諺 01"/>。另外，有網友曾建議他出版《哆啦A夢》百科全書，對此他則表示若有出版社希望發行，他確實是會考慮，不過也指出仍然有圖片著作權的問題需要處理<ref name = "三立新聞台 02"/>。

張遠的網名暱稱為「ffaarr」<ref name = "蕭歆諺 01"/>。1995年，架設在[[臺灣|臺灣]]學術網路上的[[BBS|BBS]]論壇[[批踢踢|批踢踢]]開始服務<ref name = "中時電子報 01">{{Cite web |url = http://hottopic.chinatimes.com/20151227002332-260809 |title = 「小夫」家到底多有錢？網友專業分析被推爆 |author =  |date = 2015年12月27日 |publisher = [[中時電子報|中時電子報]] |language = Zh-hant |accessdate = 2018年2月15日 |quote =  }}</ref>，不久之後，張遠便受到論壇「愛湊熱鬧」的核心特質影響，成為資深[[鄉民_(網路用語)|鄉民]]與重度使用者，有著超過15年的登入資歷<ref name = "陳琡分 01"/><ref name = "人間事特別企劃 01">{{Cite web |url = https://tw.lifestyle.appledaily.com/daily/20140204/35618115/ |title = 【人間事特別企劃】你是合格的 鄉民嗎? |author =  |date = 2014年2月4日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = Zh-hant |accessdate = 2018年2月15日 |quote =  }}</ref>，兩者幾乎是共同成長{{noteTag|批踢踢一開始的使用者大多是以學生族群為主，然而隨著規模日益壯大，會員人數已經達到百萬人（同一時間更可以容納10幾萬人上線），討論板也多達6,000多個<ref name = "小治 01">{{Cite web |url = https://www.techbang.com/posts/15836 |title = PTT 鄉民驚點語錄：跟隨流行不如創造流行 |author = 小治 |date = 2013年12月11日 |publisher = T客邦 |language = Zh-hant |accessdate = 2018年2月15日 |quote =  }}</ref>。這些鄉民來自國內外地區、遍及社會各個階層，當中包括許多善於爆料的[[深喉_(水門事件)|深喉]]，或是擅長[[人肉搜尋|人肉搜尋]]技巧的用戶，過去有非常大量的八卦消息都是自此首先流出，也導致許多記者總是在上面找尋新聞<ref name = "小治 01"/>。}}<ref name = "小治 01"/>。與此同時，張遠還是[[中文維基百科|中文維基百科]]的元老級人物<ref name = "蕭歆諺 01"/>，在國立臺灣大學維基研究社創社時，他便曾加入成為社員<ref name="台灣維基媒體協會 02">{{Cite web |url = https://upload.wikimedia.org/wikipedia/commons/0/02/台灣維基媒體協會會訊_105年10月號.pdf |title = 台灣維基媒體協會會訊（105年10月號） |author = 王則文和上官良治 |date = 2016年9月30日 |publisher = [[台灣維基媒體協會|台灣維基媒體協會]] |language = Zh-hant |accessdate = 2018年2月15日 |quote =  }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；而在平日念書、工作的忙碌之餘，他還會參與[[管理員_(維基百科)|管理員]]的工作<ref name = "陳琡分 01"/>。

=== 鄉民百科 ===

憑藉着在維基百科和「哆啦A夢wiki」上共同編輯的經驗，讓張遠認為有為批踢踢留下記錄的必要，而不能只放在資訊較不易保存和搜尋的BBS平臺上<ref name = "Ffaarr 01">{{Cite book | author = Ffaarr | title = 《PTT鄉民大百科》 | location = 臺灣臺北 | publisher = [[時報文化|時報文化]] | date = 2013年 | pages = 第3頁 | ISBN = 978-9571358253 | accessdate = 2018年2月15日 | url = https://books.google.com.tw/books?id=wapnAgAAQBAJ&printsec=frontcover&hl=Zh-hant#v=onepage&q&f=false | language = Zh-hant | quote =  }}</ref>。原本他想為此架設Wiki型態的網站，邀集有興趣的鄉民們共同撰寫<ref name = "陳琡分 01"/>。但經過考慮之後，他認為可能有不少內容容易產生多重說法，基於管理上的方便，決定在2012年1月開設「PTT鄉民百科」部落格<ref name = "蕭歆諺 01"/>、及[[Facebook|Facebook]]粉絲團「Ptt鄉民百科」<ref name = "人間事特別企劃 01"/> ，自己蒐集整理與發表相關人物、事件及用語<ref name = "陳琡分 01"/><ref name = "Ffaarr 01"/>，不過後來仍在Wikia成立「PTT鄉民百科」<ref name = "陳瑞霖 01"/>。

由於每天使用這些批踢踢的用語，張遠主要是以參與者的身分（而非研究者角度切入）撰寫「PTT鄉民百科」的內容，並在部落格中收集、整理其上的各種資料<ref name = "陳琡分 01"/><ref name = "小治 01"/>。他還嘗試從釋義、起源開始，解說各別語彙進一步的延伸意涵<ref name = "小治 01"/>。然而，儘管他試圖保存這20年來衍生出的最精華詞彙、鄉民文化及歷年重要事件<ref name = "劉嫈楓 01"/><ref name = "小治 01"/>，但即便是他自己早已經習慣使用的語言，查詢內容仍然相當耗費心力{{noteTag|由於批踢踢與外界間的密集交流，當中各種有趣的流行用語，有些是自己的鄉民獨創，並進一步影響網際網路及媒體<ref name = "小治 01"/>；有些則源自於外界，卻在論壇上開始流行，並且添加新的用法或文化內涵<ref name = "小治 01"/>。}}<ref name = "陳琡分 01"/>。

部分鄉民用語剛出現時，張遠亦在八卦版上，這些內容可以直接撰寫<ref name = "陳琡分 01"/>，或在過程中翻閱各版文章記錄{{noteTag|作為眾多鄉民之一的他<ref name = "小治 01"/><ref name = "Ffaarr 01"/>，也會在[[Google搜尋|Google搜尋]]上以「site:ptt.cc」技巧，加上年份逐一搜尋，藉此找到真正關鍵的事件發生時間點<ref name = "陳琡分 01"/>。}}<ref name = "Ffaarr 01"/>。相對地，因大量的批踢踢鄉民們會隨時提出不同的意見，他經常必須到處搜尋先前其他用戶細心整理後的[[懶人包|懶人包]]或相關資料<ref name = "陳琡分 01"/><ref name = "小治 01"/>。對於無法確定的說法，他則全部列出來，讓知道此事的熱心鄉民或讀者，對其中的內容提出補充或糾正<ref name = "陳琡分 01"/><ref name = "Ffaarr 01"/>。2013年，張遠還和鄉民資歷較短的「Lon」兩人合作，編目出版成《PTT鄉民大百科》一書<ref name = "劉嫈楓 01"/>，由後者負責為書中的用語繪製使用情境漫畫<ref name = "陳琡分 01"/>。

== 思想理念 ==
{{Quote box| width = 300px | align = left | quoted = ture | bgcolor = #FFFFF0 | salign = right | quote = 鄉民的面向很豐富，同一個鄉民在這個板很愛酸人、什麼都要戰，在另一個板卻可能很溫暖、很樂意助人。這些都是正義感的延伸表現。反正有什麼事都想管一管。 | source = 張遠，2013年10月31日<ref name = "陳琡分 01"/> }}

張遠認為自己最喜歡的角色是野比大雄，且對此選擇毫不猶豫，不過挑選最喜歡的一篇故事則較為困難<ref name = "蕭歆諺 01"/>。在熟讀1,300多篇《哆啦A夢》的故事後，他認為有200篇至300篇相當值得細講：在《Hello 宇宙人》裡討論事實真假的認定<ref name = "蕭歆諺 01"/>；在《像白百合花的女孩》裡討論野比大雄不願讓父親幻想破滅等<ref name = "蕭歆諺 01"/>。他更特別推薦短篇《鉛筆飛彈和自動報復雷達》：野比大雄陣營和[[剛田武|剛田武]]陣營互相準備飛彈，只要一方先發射、對方就馬上反擊，最終雙方感受到殺傷力過大，決定和平相處、一起打棒球<ref name = "蕭歆諺 01"/>。張遠認為這個看似朋友間日常爭執的劇情，已經隱約處理到[[冷戰|冷戰]]的[[核軍備競賽|核軍備議題]]<ref name = "蕭歆諺 01"/>。

另一方面，張遠認為批踢踢資訊豐富、具有文化創意<ref name = "Ffaarr 01"/>，鄉民在批踢踢上創造出、大量特色且有趣的語言，是當代很有意義的文化現象{{noteTag|鄉民們創造出如「塊陶」、「踹共」等新話語，經常藉由新聞媒體進一步推廣應用，因此這些詞彙的背後能代表著某種文化現象跟熱潮<ref name = "小治 01"/>。不過對於大部分人來說，當中許多僅僅是跟隨著流行才使用著，而不見得會理解其背後的概念<ref name = "小治 01"/>。}}，應當加以記錄整理{{noteTag|批踢踢最大的資產是每天發表在各個板上的龐大數量文章資訊，只要有某些新聞或事件成為大家討論關注的話題，其中的詞語或句子便常常會迅速流行，形成特殊的[[次文化|次文化]]，這也就是鄉民流行語的由來<ref name = "小治 01"/>。}}<ref name = "陳琡分 01"/><ref name = "小治 01"/>。而透過《PTT鄉民大百科》一書的推出，張遠期待可以讓曾經是鄉民的人重溫與補足記憶中的往事，他並且提到：「很多鄉民提到這本書時會說『都知道了幹嘛還要看』，但如果能讓多點人記得這些大家共同經歷過的事，還是很值得的。<ref name = "陳琡分 01"/>」

張遠還認為批踢踢的鄉民與一般[[網友|網友]]不太一樣，且和「Lon」一致同意，任何時期的鄉民最大共同特色便是愛湊熱鬧<ref name = "陳琡分 01"/>。他也認為儘管八卦版最為強勢，但批踢踢各板都有自己的文化、而非單一性質，且文化這個東西並非立即形成，且無法簡單取代，這些多樣性讓批踢踢被取代的可能性更低<ref name = 郭雅茵、金宸瑋和陳宜秀 01"/>；同樣地，他主張社群網站和論壇實際上屬於不同性質，前者的「社團」確實取代批踢部分功能（其上的班板、社團板沒落），但同樣作為論壇的[[Dcard|Dcard]]才是造成使用人數下滑的重要因素<ref name = 郭雅茵、金宸瑋和陳宜秀 01"/>。
