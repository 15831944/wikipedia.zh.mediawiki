'''張才'''（{{lang-en|Chang Tsai}}；{{bd|1916年||2014年|5月4日}}），臺灣攝影家。出生於臺北大稻埕，青年時期受兄長[[張維賢|張維賢]]影響赴日本研習攝影。與同時期的[[鄧南光|鄧南光]]、[[李鳴鵰|李鳴鵰]]並稱為「攝影三劍客」或「快門三劍客」，活躍於1930-1950年代。2014年入選IPA亞洲30位最具影響力攝影師。2014年5月4日逝於臺北自宅。

==生平簡介==
張才1916年出生於臺北大稻埕，幼年喪父，兄長[[張維賢|張維賢]]為臺灣新劇運動領袖。1934年，張才聽從[[張維賢|張維賢]]建議赴日本學習攝影，學習期間受到德國「新即物主義」（Neue Sachlichkeit）啟發，因此發展出客觀寫實的攝影美學，<ref>{{cite web |url=http://nrch.culture.tw/twpedia.aspx?id=2417 |title=〈張才〉|author=蕭永盛|date=2009-09-24 |website=【臺灣大百科全書】|publisher=文化部 }}</ref>青年時期被暱稱為「社會大學的畢業生」，作品反映人文精神、社會意識與文化現象等多重面向。<ref>{{cite book |author=張才|date=1994-11|title=《臺灣攝影家群像：張才》|location=臺北縣中和市 |publisher=躍昇文化  |isbn=978-957-630-319-7}}</ref><p>
1936年於大稻埕創立「影心寫場」。二次世界大戰爆發，與家人赴上海。1946年返臺，1948年於臺北[[中山堂|中山堂]]舉辦首次個展。1948 年新生報於淡水沙崙舉辦攝影比賽，張才獲第一名，[[鄧南光|鄧南光]]跟[[李鳴鵰|李鳴鵰]]同獲第二名，當時較年輕的攝影家[[黃則修|黃則修]]稱三人「三劍客」<ref>{{cite news |author=吳逸驊／文、余志偉／攝影|date=2016-05-04 |title=〈張照堂專訪──攝影書不僅是一本書〉|url=https://www.twreporter.org/a/photo-interview-chang-chao-tang|newspaper=【報導者】}}</ref>，此稱呼流傳開來，張才、鄧南光跟李鳴鵰遂常被合稱為「攝影三劍客」或「快門三劍客」<ref>{{cite web |url=http://www.insighttaiwandb.com.tw/Share/ShareDetail/S201511260007 |title=〈攝影三劍客之真實版〉 |author=簡永彬 |date=2015-11-26 |website=【意象・臺灣】|publisher=國藝會}}</ref>。<p>
1950年代，張才隨臺灣大學人類學教授進行田野調查拍攝原住民，同時因個人興趣拍攝臺灣宗教慶典與酬神歌仔戲，以鏡頭捕捉了原住民質樸的性格與生活、臺灣宗教慶典的熱鬧面向與戲臺下歌仔戲演員的心酸，為臺灣紀實攝影開先鋒。<ref>{{cite web |url=http://www.tfam.museum/Exhibition/Exhibition_page.aspx?id=344&ddlLang=zh-tw |title= 〈意象地圖－張才攝影紀念展〉|date= 2014-07-11 |website=【臺北市立美術館】 |publisher=臺北市立美術館  }}</ref><p>
2014年1月14日，IPA公布〈2014 亞洲最具影響力的 30 位攝影家〉，臺灣入選者為[[張照堂|張照堂]]、[[張乾琦|張乾琦]]、張才。<ref>{{cite web |url=http://invisiblephotographer.asia/2014/01/17/30influentialasiaphotographers/ |title=〈30 Most Influential Photographers in Asia〉 |date=2014-01-07|website=【IPA】|publisher=Invisible Photographer Asia  }}</ref>同年5月4日，張才於臺北自宅過世。<ref>#{{cite web |url=https://cyberisland.teldap.tw/graphyer/photo/znNl |title=〈張才〉  |website=【數位島嶼】|publisher=中央研究院數位文化中心 }}</ref>
張才曾於1950年代拍攝三峽宗廟慶典，留下珍貴歷史影像，2016年為張才百年誕辰，三峽成福國小攝影社舉辦「百年張才巧遇百年三峽」，向張才致敬。<ref>{{cite news |author=謝幸恩 |date=2016-03-14 |title=〈攝影大師張才百年誕辰 小小攝影師 向大師致敬〉 |url=http://www.chinatimes.com/newspapers/20160314000475-260107 |newspaper= 中國時報|location=新北 }}</ref><p>
