'''張穎沁'''（{{Lang-en|Pamela Cheung Wing Sum}}，{{bd|1990年|4月13日|catIdx=C張}}），香港女性模特兒、Youtuber及專欄作家。現為[[寰宇國際控股有限公司|寰宇國際控股有限公司]]旗下模特兒。

== 簡介 ==
張穎沁有兩名胞妹<ref>{{Cite web |url=https://unwire.hk/2015/03/30/notebook-hp-stream-13-pamela/notebook/ |title=贏在起跑線！Pamela × 小編 對談 實試小學生必備 Notebook HP Stream 13 |date=2015-03-30 |publisher=[[unwire.hk|unwire.hk]]}}</ref><ref name=PC>{{Cite journal |author=諾藍 |url=http://www.tlcomics.com/information.php?pkey=15 |title=【新少年 CoverGirl】冬日裡的暖流- Pamela |journal=[[新少年週刊|新少年週刊]] |year=2010 |issue=3 |pages=15}}</ref>，曾就讀[[民生書院|民生書院]]，其後分別取得了[[灣仔|灣仔]]CO1設計學校與[[香港知專設計學院|香港知專設計學院]]共兩個室內設計高級文憑<ref name=PC/>。她於2008年開始任職模特兒<ref>{{Cite web |url=http://the-sun.on.cc/cnt/entertainment/20090626/00470_067.html |title=『𡃁版恭子』張穎沁最叻長距離戀愛 |date=2009-06-26 |publisher=[[太陽報_(香港)|太陽報]]}}</ref>，曾參與演出[[香港迪士尼樂園|香港迪士尼樂園]]、[[香港郵政局|香港郵政局]]、[[HBO|HBO]] 等不同機構的電視及平面廣告；2014年跟交往九年的郭氏商人結婚，並於2016年11月1日在[[養和醫院|養和醫院]]剖腹誕下兒子 Adam<ref>{{Cite web |url=https://hk.on.cc/hk/bkn/cnt/entertainment/20161105/bkn-20161105102733900-1105_00862_001.html |title=張穎沁喜誕6.8磅 B 仔 |date=2016-11-05 |publisher=[[ON.CC|東網]]}}</ref><ref>{{Cite web |url=https://www.cosmopolitan.com.hk/beauty/1598c8e605090ff |title=新晉媽媽張穎沁 Pamela 為寶寶的第一個農曆年做最好的準備 |date=2017-01-12 |publisher=[[COSMOPOLITAN|COSMOPOLITAN]]（香港）}}</ref>。

2012年11月，張穎沁開始擔任 [[YouTuber|YouTuber]]，2017年則推出首本書籍《Young 媽日記》，分享成為母親的經歷<ref>{{Cite web |url=https://orientaldaily.on.cc/cnt/entertainment/20171215/mobile/odn-20171215-1215_00282_050.html |title=模媽張穎沁出書分享產後心得 |date=2017-12-15 |publisher=[[東方日報_(香港)|東方日報]]}}</ref><ref>{{Cite web |url=https://topick.hket.com/article/1973631/一年勁減40磅%E3%80%8090後潮媽張穎沁產後紮肚修身大法 |title=一年勁減40磅　90後潮媽張穎沁產後紮肚修身大法 |date=2017-12-28 |publisher=[[經濟日報|經濟日報]]}}</ref>；現時亦任職[[意見領袖|意見領袖]]，從事撰寫網上專欄工作<ref>{{Cite web |url=http://www.ohpama.com/178342/24parent/%E3%80%90follower-%E5%8B%81%E9%81%8E%E6%B8%AF%E5%A7%90%E5%90%8D%E6%A8%A1%E3%80%9190%E5%BE%8Ckol%E5%A5%B3%E7%9A%87pamela-%E6%95%99%E4%BD%A0%E7%94%9F%E5%AE%8Ckeep-%E4%BD%8F%E9%9D%9A/ |title=【Follower 勁過港姐名模】90後 KOL 女皇 Pamela 教你生完 Keep 住靚 |date=2017-04-29 |publisher=[[星島新聞集團|Oh! 爸媽]]}}</ref>。
