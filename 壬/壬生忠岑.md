'''壬生忠岑'''（{{jpn|j=壬生 忠岑|hg=みぶ の ただみね|rm=Mibu no Tadamine|f=h}}，約860年 - 約920年）是[[平安时代|平安时代]]前期的[[歌人|歌人]]，[[三十六歌仙|三十六歌仙]]中的一位。“{{lang|ja|岑}}”有時又寫作“{{lang|ja|峯}}”，這兩字在日語中[[同音词|同音]][[同义词|同義]]，都讀作“{{lang|ja|みね}}”。
[[File:Sanjūrokkasen-gaku_-_27_-_Kanō_Yasunobu_-_Mibu_no_Tadamine.jpg|right]]

== 經歷 ==
[[甲斐國|甲斐國]]的[[國造|國造]]壬生直家族<ref>{{cite book |author={{link-ja|太田亮|太田亮}}|year=1963|title=姓氏家系大辞典|series=第3巻|language=ja|location=東京|publisher=[[角川書店|角川書店]]|page=五八八七頁|isbn=9784040302300}}</ref>[[從五位|從五位]]下壬生安綱之子<ref name="asahi" /><ref>《古今和歌集目錄》</ref>也有木工允壬生忠衡之子的说法。<ref>{{cite book |author={{link-ja|宝賀寿男|宝賀寿男}}|year=1986|title=古代氏族系譜集成|language=ja|publisher=古代氏族研究会|isbn=}}，根据{{cite book |author={{link-ja|鈴木真年|鈴木眞年}}|year=1871|title=百家系図|language=ja}}</ref>古書《三十六人歌仙传》中记为“先祖不見”，認為是不清楚比較妥當。<ref>跟據迫徹郎</ref>其子壬生忠見也是三十六歌仙之一。

壬生忠岑雖然出身為下級武官，但是其作為歌人的水平是一流的，被選為《[[古今和歌集|古今和歌集]]》的編者之一。{{link-ja|家集|家集}}有《忠岑集》現存。

== 作品 ==
他的作品風格溫和多抒情，富機智。後人[[藤原公任|藤原公任]]著書《和歌九品》中“上品上”也就是最高水平是以忠岑的歌為例子的。忠岑著名的歌有：
{{quote|{{lang|ja|春立つといふばかりにやみ吉野の山も霞みてけさは見ゆらむ}}|壬生忠岑|[[拾遺和歌集|拾遺和歌集]]・巻頭歌}}
[[File:MET_DP276497.jpg|thumb]]畫。上面寫有“{{lang|ja|忠峯}}”和“{{lang|ja|春立つといふばかりにやみ吉野の山も霞みてけさは見ゆらむ}}”這首歌。]]
傳說將和歌分為十種的《和歌体十種》是忠岑所作，現在認為更可能是誤傳。
