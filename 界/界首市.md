'''界首市'''在[[中國|中國]][[安徽省|安徽省]]西北部，领接河南省，[[颍河|颍河]]横贯，是[[阜阳市|阜阳市]]下辖的一个县级市。2006年被安徽省列为12个享有省辖市部分经济社会管理权限的试点县市之一。
== 历史沿革 ==

[[西周|西周]]时期，境北部属[[陈国|陈国]]，南部属沈子国。

[[春秋时期|春秋]]时期，[[魯定公|鲁定公]]四年（前506年），[[蔡國|蔡]]灭沈子国，境南部自沈入蔡；[[鲁哀公|鲁哀公]]十七年（前478年）[[楚国|楚]]灭陈，境北部自陈入楚。

[[戰國|战国]]时期，境北部由楚入[[魏国_(战国)|魏]]，南部属楚。

[[秦朝|秦]]，始行郡县二级制，境北部属新阳县，南部属寝县，统辖于[[颍川郡|颍川郡]]。后分立[[陳郡|陈郡]]改属之。

[[西汉|西汉]][[刘邦|高帝]]六年（前201年），改新阳县为侯国，境北部属之，后复为县；[[汉宣帝|汉宣帝]][[地节|地节]]四年（前66年）境中部置阳城侯国，旋废；南部仍属寝县（一作县）。同辖于[[汝南郡|汝南郡]]。

[[王莽|王莽]]篡汉，[[始建国|始建国]]元年（公元9年），改汝南郡为汝土贲郡，改阳城为新安，改新阳为新明，改寝为闰治。

[[东汉|东汉]]始行州郡县三级制，境北部属新阳侯国；南部属固始侯国，后国废为县；中部属汝阴县。同辖于[[豫州刺史部|豫州]][[汝南郡|汝南郡]]。

[[三国|三国]][[曹魏|魏]]，境北部属新阳县，南部属[[固始县|固始县]]，同辖于[[豫州_(古代)|豫州]]汝南郡。

[[西晋|西晋]]，新阳县废。境北部属宋县，南部属固始县，同辖于豫州[[汝阴郡_(曹魏)|汝阴郡]]。[[晉惠帝|惠帝]]分汝阴郡置[[新蔡郡_(西晋)|新蔡郡]]，固始县改属之。

[[东晋|东晋]]、[[南北朝|南朝]][[刘宋|宋]]、[[北魏|北魏]]，境北部属汝阴郡宋县，南部属新蔡郡固始县，同辖于豫州。

[[东魏|东魏]]，境北部属北扬州汝阴郡宋县，南部属豫州新蔡郡固始县。

[[北齐|北齐]]、[[北周|北周]]，北部属[[颍州_(北魏)|颍州]][[陳留郡|陈留郡]][[陳留縣|陈留县]]，南部属豫州广宁郡褒信县。

[[隋朝|隋]][[开皇|开皇]]三年（583年），罢郡为州，境北部属颍州陈留县（后更名颍阳县），南部属沈州鱼同阳县。[[大业_(年号)|大业]]三年（607年）罢州为郡，北部属汝阴郡颍阳县，南部属[[淮阳郡_(隋朝)|淮阳郡]]鱼同阳县。

[[唐朝|唐]][[貞觀_(唐朝)|贞观]]元年（627年），废颍阳，并入汝阴。[[武德|武德]]初罢郡为州，[[天宝_(唐朝)|天宝]]元年（742年）罢州为郡，[[乾元_(唐朝)|乾元]]元年（758年）罢郡为州。境北部属汝阴县，南部属[[沈丘县|沈丘县]]，同隶属于颍州（汝阴郡）。

[[五代十国|五代]]，先后为[[后梁|梁]]、[[后唐|唐]]、[[后晋|晋]]、[[後漢|汉]]、[[后周|周]]领地，县境分属颍州汝阴县、沈丘县。

[[宋朝|宋]][[开宝|开宝]]六年（973年），分汝阴北5乡置万寿县。境北部属万寿县（后改[[太和县|泰和县]]），南部属沈丘县，同辖于[[京西北路|京西北路]]颍州（后改[[顺昌府|顺昌府]]）。始置界沟镇于现城区东北部，属万寿县。

[[金朝|金]]，境分属泰和县、沈丘县，同隶属于[[南京路_(金朝)|南京路]]颍州。

[[元朝|元]]，境分属泰和（后改太和）、沈丘县，同隶于[[河南江北等处行中书省|河南江北等处行中书省]]颍州。[[至元_(忽必烈)|至元]]三十年（1293年）改属[[汝宁府|汝宁府]]。

[[明朝|明]]，境北部属颍州太和县，南部属颍州，同隶属于南京[[凤阳府|凤阳府]]。城区西北部属河南布政使[[开封府|开封府]]陈州沈丘县。

[[清朝|清]][[雍正|雍正]]二年（1724年），境北部改属[[亳州_(古代)|亳州]]太和县，十三年改颍州为[[颍州府|颍州府]]，境北部改属安徽省颍州府太和县，境南部改属[[阜阳县|阜阳县]]，城区西北部属河南省陈州府沈丘县。

[[中華民國|民国]]初，县境大部属安徽省[[淮泗道|淮泗道]]阜阳县、太和县。民国21年（1932年），设阜阳专员公署，太和县、阜阳县皆属之。民国24年，析阜阳县西部置[[临泉县|临泉县]]，境南部遂划入临泉县。民国28年，城区西北部发展为皂庙镇，属河南省沈丘县，与界首镇、刘兴镇鼎立为界首三镇。民国36年10月，界首、刘兴、皂庙3镇合并建立界首市民主政府。是年11月，境北部属沈鹿淮县（界首县），12月，南部属泉阳县。界首市、界首县同属豫皖苏边区行署二分区，泉阳县属四分区。民国38年3月，豫皖苏边区行署建制撤销，境北部属太和县，中部（现城区）属界首市，南部属临泉县。同辖于[[皖北行政区|皖北行署]][[阜阳专区|阜阳专署]]。

[[中华人民共和国|中华人民共和国]]时期，1953年9月，撤界首市建界首县，划临泉县的洪庄、砖集、陶庙3区，太和县的光武区和税黄区的西部为界首县辖地。隶属安徽省阜阳专区。1958年11月，界首县与太和县合并为首太县，境南洪庄、砖集2区划入临泉县。翌年4月底，太和县、界首县分置，辖地仍沿原制，属安徽省阜阳专区，至1985年未变。1989年，撤县复市。

== 地理 ==
界首市位于安徽省西北部边陲。南与[[临泉县|临泉县]]和[[阜阳市|阜阳市]][[颍泉区|颍泉区]]接壤，东与[[太和县|太和县]]为邻，西北与河南省的[[沈丘县|沈丘]]、[[郸城县|郸城]]两县交界。

地处东经115°15′-115°32′，北纬30°0′-33°0′。地处淮北平原西北部。[[潁河|沙颍河]]横贯中部，[[泉河_(颍河)|泉河]]流经界首市。年降水量817毫米，年均气温14.7℃。地下蕴藏有含碘矿泉水。

== 名人 ==
*[[刘福通|刘福通]]（1320～1365）：元代颍州（今安徽界首）人，元末[[紅巾軍|红巾军]]首领。与[[韩山童|韩山童]]等长期利用[[白蓮教|白莲教]]在民间进行活动。[[至正|至正]]十一年（1351年）五月，乘元廷徵民夫修[[黄河|黄河]]之机聚众起事，攻占州。因以红巾为号，称红巾军。随后，攻占河南许多州县。十五年，立[[韩林儿|韩林儿]]（韩山童之子）为帝，称小明王，以亳州（今安徽亳州）为都，建国号[[宋_(韓林兒)|大宋]]，年号[[龍鳳_(韓林兒)|龙凤]]。他历任[[同平章事|平章]]、[[丞相|丞相]]等职，掌握韩宋的军政大权。元廷派兵镇压，亳州失陷，迁都安丰（今安徽[[寿县|寿县]]）。后分兵三路攻略[[陕西省|陕西]]、[[山东省|山东]]、[[河南省|河南]]、[[山西省|山西]]、[[河北省|河北]]等地，兵锋曾达[[元大都|大都]]附近的柳林（今[[北京市|北京]][[通州区_(北京市)|通州]]南）。十八年，他统兵攻占[[汴梁|汴梁]]（今河南[[开封市|开封]]），作为都城。次年，汴梁被元军攻破，退守安丰。二十三年，[[張士誠|张士诚]]军围攻安丰，他与韩林儿往[[滁州_(古代)|滁州]]（今安徽[[滁州市|滁州]]）依[[朱元璋|朱元璋]]。后被朱元璋命部将沉于江中溺死（一说在安丰阵亡）。
*卢山义（1920年2月～2007年10月）：界首人，中国工艺美术家协会会员、中国民间艺人、中国民间文化杰出传承人、中国工艺美术大师，为安徽省省最年长的国家级工艺美术大师。卢山义出生在素有“陶艺之乡”之称的安徽省界首市卢窑村一个艺术之家；祖上以手工艺品为生，其父为当地有名的戏剧刻画名匠。卢山义自幼就受父亲的艺术感染和熏陶，喜欢上民间酒坛、花瓶、脸盆上绘制的鸟语花香，并萌发出艺人的聪颖智慧；受到父亲的支持并教授其走上从艺人生。在他十五岁那年他父亲的去世使他承担起养家糊口的重担，并使得其技艺很快成熟起来。
*[[郭汝霖|郭汝霖]]（1920—2010），乳名铁良，安徽省界首人。[[中華民國總統府|中华民国总统府]]参军长、[[中華民國空軍司令|空军总司令]]、[[中國國民黨|国民党]]籍。空军机械学校、空军官校第14期、空军指挥参谋学校、三军联大、战争学院将官班毕业。曾任飞行员、分队长。去台后，历任中队长，副大队长，大队长，联队政战部主任，副联队长、联队长，空军总部作战训练署副署长，空军作战司令部副司令，空军总部组织人事署署长，空军官校校长，空军总部参谋长。1980年1月任警备总部副总司令。1981年7月任空军副总司令，12月任空军总司令，至1986年7月。并晋升空军二级上将。1986年7月任副参谋总长兼执行官，至1988年7月。1988年5月任总统府参军长。1989年12月任总统府国策顾问，1996年5月获续聘。1988年7月当选为国民党第十三届中央委员。1993年被聘为国民党第十四届中央评议委员。
　　

== 经济 ==
界首工业基础较好，品牌优势明显，发展潜力巨大。目前，全市已形成20多个行业，2000多个品种的工业生产体系，再生金属、再生塑料、医药、纺织制鞋、食品酿造、机械制造、渔具生产等支柱产业优势明显，培育出沙河特曲、金裕皖系列白酒、富硒康口服液等全国知名产品。

== 历代名镇 ==
[[光武镇|光武镇]]，旧称光武庙，在界首市北部，光武镇是一座悠久的文明古镇。在东汉以前叫赵路口，因[[汉光武帝|汉光武帝]]刘秀与王莽长期争战于此，留下了大量的典故和民间传说，镇因汉室苗裔建光武庙祭祀而得名，历时近2000年。光武镇位于皖西北豫皖两省三县交界处，距界首市区16公里，辖2个集镇，16个村委会，4个居委会，面积46.4平方公里，耕地4.5万亩，总人口5.6万人。其中建成区面积3.7平方公里，人口2.3万。自古以来就是中原地区的驿站和重镇，19世纪初至四十年代，商业繁荣，主要经营棉花、布匹、药材等，是当时中原地区重要的集散市场，一年一度的十一月冬季物资交流大会，吸引着京、广、汉等地客商大贾不辞千里之遥，来此交易。现已形成以废旧塑料为龙头的全国性回收加工市场之一。
