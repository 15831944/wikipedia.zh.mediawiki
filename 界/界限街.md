'''界限街'''（{{lang-en|Boundary Street}}），是[[香港|香港]][[九龍|九龍半島]]一條東西向街道，連接[[深水埗區|深水埗區]]與[[九龍城區|九龍城區]]。道路始於[[深水埗|深水埗]]南，中間穿越[[九龍塘|九龍塘]]南部和[[九龍仔|九龍仔]]，最後到達[[九龍城|九龍城]]並連接[[太子道|太子道西]]。而界限街本身亦與太子道西大致平行。

殖民地時代，清廷[[割讓|割讓]]予英國的土地以此線為界，此線以南為割讓地（[[九龍|九龍]]），以北為[[租借|租借]]地（[[新界|新界]]）。

== 歷史 ==
1860年，[[清朝|清朝]]政府與[[英國|英國]]簽訂《[[北京條約|北京條約]]》，將九龍半島南部一大半割讓給英國。而界限街就成為清朝與[[英屬香港|英屬香港]]的界線，英方稱為「界限線」（'''Boundary Line'''），規定開關時間為每天早上6時至日落之後，直至1898年英國政府根據《[[展拓香港界址專條|展拓香港界址專條]]》租借[[新九龍|新九龍]]及[[新界|新界]]為止。這條界限線亦被英方改稱為「舊邊境線」（'''Old Frontier Line'''）。

當時此界線並非正式道路，原為一座小山，及後由於耕地不足，小山被夷平。到了1934年，為配合[[九龍塘|九龍塘]]發展，以及處理租界內外的[[地稅|地稅]]問題，[[港英政府|香港政府]]便於此界線上興建一條道路，即是界限街<ref>《九龍街道命名考源》梁濤 著，第51頁，市政局出版，1993年</ref><ref>《香港歷史文化小百科16－趣談九龍街道》 爾東 著，第97-99頁，明報出版社，2004年11月，ISBN 962-8871-46-3</ref>。

== 現今意義 ==
現在，界限街仍然是九龍與新九龍的分界線。位於界限街以北地區的物業需向香港政府繳付[[地租|地租]]（以南之物業則需繳付[[地稅|地稅]]）。此外，[[港鐵|港鐵]][[東鐵綫|東鐵綫]]以西的一段界限街亦是[[油尖旺區|油尖旺區]]與[[深水埗區|深水埗區]]的分界線，該段界限街以北屬深水埗區，以南則屬油尖旺區。

== 現況 ==
界限街中段是[[九龍仔|九龍仔]]豪宅區，居住者大多是[[中產階層|中產階層]]，該處亦是不少名校如[[瑪利諾修院學校|瑪利諾修院學校]]、[[喇沙小學|喇沙小學]]的所在地；而西面就是深水埗及[[大角咀|大角咀]]，居住者以中下階層、新移民為主。
