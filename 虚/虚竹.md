'''虛竹'''，後改道號'''虛竹子'''，是[[金庸|金庸]]的武侠小说《[[天龙八部_(小说)|天龙八部]]》中的三位主角其一（其餘兩位是[[段誉_(天龙八部)|段譽]]以及[[喬峰|喬峰]]）。為[[金庸小說|金庸小說]]中武功最絕頂的高手之一。

虛竹子性格木讷老实，相貌丑陋，浓眉大眼，鼻孔上翻，双耳招风，嘴唇甚厚，又不善于词令。

== 生平 ==
自幼出家為少林寺小僧，二十二歲起始習武，二十四歲時因跟隨師父發放名帖下山。

誤打誤撞之下，與一眾少林高僧一起被[[星宿派|星宿派]]所擄。

後來因為救[[四大惡人|四大惡人]]之首「惡貫滿盈」[[段延慶|段延慶]]一命，無意間破解[[逍遙派|逍遙派]]「聾啞老人」[[蘇星河|蘇星河]]的珍瓏棋局（蘇星河師父無崖子所擺下的），[[逍遙派|逍遙派]]掌門人[[無崖子|無崖子]]遂收其為徒，化去其[[少林|少林]]內功，並將七十年逍遙派功力傳於虛竹，並傳授逍遙派第三代掌門之位。
  
後來無意間救了[[天山|天山]][[缥缈峰|缥缈峰]][[靈鷲宮|靈鷲宮]]尊主[[天山童姥|天山童姥]]一命，為了逃避[[天山童姥|天山童姥]]的死對頭師妹[[逍遙派|逍遙派]][[李秋水|李秋水]]的追殺，他倆決定躲藏在李秋水的大本營 [[西夏|西夏]]皇宮內的[[冰窖|冰窖]]中。

躲藏期間天山童姥硬逼虛竹破了[[葷戒|葷戒]]，但要他自願破戒卻始終未成，索性便在暗夜時分，將一名一絲不掛的妙齡女子[[夢姑|夢姑]]放在虛竹身旁，而虛竹終於破了[[淫戒|淫戒]]，當「[[天山折梅手|天山折梅手]]」傳授給虛竹後，為對付李秋水便打算一併把「[[天山六陽掌|天山六陽掌]]」教給他，不料虛竹不肯助其殺人，堅不肯學，甚至起意離開，為此，天山童姥用謀在虛竹身上種落「[[生死符|生死符]]」，再教他破解之道並解開生死符，又把「天山六陽掌」的功夫傾囊相授，因而在李秋水追入冰庫要殺天山童姥時，虛竹無意間就使出了第二招「陽春白雪」、第七招「陽關三疊」以及「陽歌天鈞」三招接下李秋水的攻擊，而天山童姥在與李秋水互比內力時，虛竹被夾於兩人中間，因此兩人的內力皆囤積於虛竹體內，而虛竹身上則有逍遙派三大高手二百餘年的深厚內力。

最後虛竹得到天山童姥、李秋水及無涯子三人深厚的內功，天山童姥亦傳其為靈鷲宮尊主，又在宮內和大[[遼|遼]]南院大王「北喬峰」[[蕭峰|蕭峰]]、[[大理|大理]][[皇子|皇子]][[段誉_(天龙八部)|段譽]]結拜為兄弟。

處理完靈鷲宮大小事務後，虛竹決定回[[少林寺|少林寺]]，遇上[[吐蕃|吐蕃]]大輪寺[[鳩摩智|鳩摩智]]明王前來挑戰少林眾僧，虛竹挺身而出，鳩摩智先使「般若掌」，被虛竹當即以[[逍遙派|逍遙派]]的「[[天山六陽掌|天山六陽掌]]」和一招「韋陀掌」中的「山門護法」將他掌力化去，鳩摩智袖里乾坤「托缽掌」擊向对方，虚竹闪避，鸠摩智早料到他闪避的方位，鳩摩智再出「大金剛拳」轟中虛竹肩頭，哪知虛竹有「北冥真氣」護體，鳩摩智先後使出「如影隨形腿」、「多羅指法」、「燃木刀法」、「大智無定指」、「去煩惱指」、「寂滅爪」和「因陀羅爪」，接連使出六七門少林神功，虛竹無從招架，連「韋陀掌」也使不上了，虛竹用卻只一門「[[羅漢拳|羅漢拳]]」中的「黑虎偷心」，來來去去便只一招，但這招「黑虎偷心」中所含的勁力卻竟不斷增強，鳩摩智使出「龍爪功」拿下虛竹，卻反被虛竹用[[逍遙派|逍遙派]]的「天山折梅手」反拿己腕，鳩摩智用右手不断出「火燄刀」擊向，虚竹挥「天山六陽掌」化解，鳩摩智突發勁使[[匕首|匕首]]，連「天山折梅手」也抓不實，而刺中虚竹，而[[靈鷲宮|靈鷲宮]][[梅剑|梅]][[兰剑|兰]][[菊剑|菊]][[竹剑|竹]]四剑突現身阻止，虛竹憑藉一身逍遙派武功，久鬥後大敗鳩摩智，拯救了少林之名譽，但仍被逐出少林，改名「虛竹子」。

在[[少室山|少室山]]武林大會中，蕭峰遭受[[姑蘇慕容|姑蘇慕容]]氏家主[[慕容復|慕容復]]、[[星宿派|星宿派]]創派師祖「星宿老怪」[[丁春秋|丁春秋]]和[[丐幫|丐幫]]幫主[[游坦之|莊聚賢]]圍攻的時候，虛竹引走「星宿老怪」[[丁春秋|丁春秋]]，虛竹使開「[[天山折梅手|天山折梅手]]」和「[[天山六陽掌|天山六陽掌]]」與丁春秋展開激烈交戰，兩人鬥得勢均力敵，難分難解，久鬥後虛竹以烈酒化成的「生死符」降服丁春秋，而後丁春秋被送入[[少林寺|少林寺]][[戒律院|戒律院]]感化，[[星宿派|星宿派]]則改投靈鷲宮門下，報了弒師（[[無崖子|無崖子]]）之仇。

後來，[[西夏|西夏]]國王招納[[駙馬|駙馬]]，虛竹原本只打算陪同義弟[[段誉_(天龙八部)|段譽]]應徵，卻發現原來[[西夏|西夏]]「銀川公主」[[李清露|李清露]]竟就是在冰窖中曾與他有過肌肤之亲的妙齡女子[[夢姑|夢姑]]，二人终成眷属。

而後為了拯救義兄[[蕭峰|蕭峰]]免於牢獄之災，率領靈鷲宮部眾與義弟[[段誉_(天龙八部)|段譽]]一行人遠赴[[大遼|大遼]]。

救出[[蕭峰|蕭峰]]後，蕭峰將[[丐幫|丐幫]]神功[[降龍十八掌|降龍十八掌]]、[[打狗棒法|打狗棒法]]傳給他，希望他能在下任幫主選出時，將這兩項神功傳給他。

[[蕭峰|蕭峰]]自盡後，與一行人返回靈鷲宮。

== 性格 ==
天真、戇直，自幼深受少林高僧薰陶，恪守佛家一切清規戒律，因此本性溫馴，並且常為他人設想，但卻堅守原則，所有違背其理念的行為均會拒絕。

即使後來被[[天山童姥|天山童姥]]誘騙而盡破佛家三戒，仍不改其本性。

婚前深深的愛著神秘的夢姑，發現她的身份既是西夏「銀川公主」李清露後，和她成婚以及對妻子感情專一。

== 身世 ==
天龍八部中三位男主角的身世均十分離奇，其中虛竹即為少林方丈[[玄慈|玄慈]]及[[四大惡人|四大惡人]]中的「無惡不作」[[葉二娘|葉二娘]]的兒子。

自幼被[[蕭峰|蕭峰]]之父[[蕭遠山|蕭遠山]]奪去，偷偷放回少林寺的菜圃，讓少林高僧以為他是一名棄嬰。

於武林大會上，被[[蕭遠山|蕭遠山]]當眾揭開身世，但是[[玄慈|玄慈]]方丈先處自己二百杖之刑，再自絕經脈，母親[[葉二娘|葉二娘]]也用匕首自盡，真正的成了孤兒。

== 武功 ==
虛竹原本為少林派弟子，學以「[[羅漢拳|羅漢拳]]」和「韋陀掌」以防身。

然而在意外破解了[[無涯子|無涯子]]所設的「珍瓏」棋局後，承得[[無涯子|無涯子]]七十年深厚的內力，而其中也含有無涯子本身所學的「[[北冥神功|北冥神功]]」以及「小無相功」。

而後虛竹為[[天山童姥|天山童姥]]所傳，習得了「[[逍遙派|逍遙派]]」兩大絕招「天山折梅手」以及「[[天山六陽掌|天山六陽掌]]」，另外也習得了[[天山童姥|天山童姥]]令人聞風喪膽的絕招「[[生死符|生死符]]」，是以中符後身體會百般難受，需得以「[[天山六陽掌|天山六陽掌]]」化解。而[[天山童姥|天山童姥]]在與[[李秋水|李秋水]]互比內力時，虛竹被夾於兩人中間，因此兩人的內力皆囤積於虛竹體內，而虛竹身上則有[[逍遙派|逍遙派]]三大高手深厚的內力。

在世紀新修版中，[[蕭峰|蕭峰]]自知無法返回大宋，因而將「降龍十八掌」、「[[打狗棒法|打狗棒法]]」傳給虛竹，希望他代為傳授給丐幫新任幫主；蕭峰將「降龍二十八掌」化繁為簡，成為日後出名的「[[降龍十八掌|降龍十八掌]]」。

;小無相功
威力强大，其主要特点是不着形相，无迹可寻，只要身具此功，再知道其他武功的招式，倚仗其威力无比，可以模仿别人的绝学甚至与原版极为近似，外行很难分辨。

;天山折梅手
共包括三路掌法，三路擒拿法，一共六路武功，掌法和擒拿手之中，含蘊有劍法、刀法、鞭法、槍法、抓法、斧法等等諸般兵刃的絕招，變化繁多，天下任何招數武功，都能自行化在這六路折梅手之中

;天山六陽掌
共分六式，每招名稱中均帶一「陽」字，故名「六陽掌」，輕靈飄逸，閒雅清雋，招招兇險，攻敵要害，威力極大。



;生死符 
[[金庸|金庸]]武學[[暗器|暗器]]之一，為[[逍遙派|逍遙派]]靈鷲宮[[天山童姥|天山童姥]]所使用的一種暗器，被植入者曾經說過中此招者「求生不能，求死不能」，而會受制於他人。生死符威力極強，如果被植入而未取出來，則必須定期服用「鎮癢丸」，否則麻癢難當，天龍八部小說裡中了生死符的人有三十六洞主、七十二島主、虛竹(在冰窖內被植入)、丁春秋(在少室山被虛竹植入)。發作之時，一日比一日厲害，其癢劇痛約81日後會逐步散退，但在81日後又會再增加。生死符算是武林中第一等的暗器，利用酒或水等液體逆運真氣，將剛陽之氣轉為陰柔，使掌心中發出來的真氣冷於寒冰數倍，手中液體自然凝結成薄薄的圓冰。生死符除了搭配天山六陽掌加上靈鷲宮醫典以便掌握到陰、陽、虛、實才可以化解體內的生死符外，沒有其他辦法可以取出。為天下第一等暗器。

;羅漢拳
[[少林寺|少林寺]]入門武功之一，是拜師後所學的第一門武功。

;韋陀掌
[[少林寺|少林寺]]入門武功之一，是拜師後所學的第二門武功。

;[[降龍十八掌|降龍十八掌]]
[[丐幫|丐幫]]兩大鎮幫絕技之一，招式脫胎至易經，並對應天上二十八[[星宿|星宿]]，是[[蕭峰|蕭峰]]繼承幫主後從師父[[汪劍通|汪劍通]]處習得，因為後十招過於繁瑣，經[[虛竹|虛竹]]和[[蕭峰|蕭峰]]刪除重整後刪為十八掌，並在蕭峰死後由虛竹代傳給下任幫主。

;[[打狗棒法|打狗棒法]]
丐幫鎮幫之寶，是丐幫主代代相傳的武功，輕動靈便，變化精微，招術奇妙，共有絆、劈、纏、戳、挑、引、封、轉八訣，棒影飛舞，神龍夭矯，如流星趕月。

數百年來，丐幫逢到危難關頭，幫主親自出馬，往往便仗這打狗棒法除奸殺敵，震懾群邪。在蕭峰死後由虛竹代傳給下任幫主。
