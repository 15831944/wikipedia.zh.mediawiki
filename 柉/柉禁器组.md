[[File:MET_DP219959.jpg|缩略图]]
'''柉禁器组'''，又称'''柉禁十三器'''{{efn|实际有各类器物14件，所谓“十三器”是未将斗单独计数}}<ref name="任雪莉" />、'''柉禁诸器'''<ref name="鉴古" />，1901年出土于[[陕西|陕西]][[宝鸡|宝鸡]][[斗鸡台_(宝鸡)|斗鸡台]]一座[[西周|西周]]贵族墓葬的一组青铜器，共14件。其中有一件青铜夔蝉纹[[禁|铜禁]]，是自有[[金石学|金石学]]之后首次出土，此前禁这一类器物只见于古书记载<ref name="雅昌艺术网">{{cite web |title=端方最重要的收藏—西周柉禁套器全貌 |url=https://news.artron.net/20161114/n884366.html |website=雅昌艺术网 |accessdate=2018-11-12 |language=zh-cn |date=2016-11-14}}</ref>。柉禁器组出土后辗转被清朝[[金石学家|金石学家]][[端方|端方]]收藏，并被端方收录于他编撰的《[[陶斋吉金录|陶斋吉金录]]》中。1924年，端方的后人将柉禁器组通过美国人[[福开森|福开森]]卖给美国纽约[[大都会艺术博物馆|大都会艺术博物馆]]，由大都会艺术博物馆收藏至今。

柉禁器组的14件青铜器分别是1件青铜尊和13件青铜酒器<ref name="Altar Set">{{cite web |title=Altar Set |url=https://www.metmuseum.org/art/collection/search/76974 |website=The Metropolitan Museum of Art}}</ref>，具体为柉禁、鼎卣一、鼎卣二、铜勺、鼎尊、父乙盉、妣己觚、子扫帚斝、父乙觯、牺形爵、祖癸角、父甲觯、雷纹觯、亚形妣己觯和铜斗。除柉禁、雷纹觯和铜斗之外，其余9件均带有铭文。

== 命名 ==
{{参见|柉禁}}
端方在《陶斋吉金录》将其所藏青铜禁命名为“柉禁”，但并未详述其命名的理由<ref name="陶斋吉金录">{{cite book |author1=端方 |title=陶斋吉金录 |date=1908 |edition=光绪三十四年石印本}}</ref>。据[[容庚|容庚]]推测，应是源自东汉《[[礼器碑|礼器碑]]》中“笾柉禁壶”的记载<ref name="任雪莉">{{Citeweb|title=宝鸡石鼓山西周早期贵族墓现柉禁器组 |url=http://collection.sina.com.cn/tqfx/20120926/173986574.shtml |date=2012-09-26 |author=任雪莉 |publisher=新浪收藏}}</ref>。“柉”原为树名，在《礼器碑》中借指杯或碗类容器，将青铜禁命名为“柉禁”并不十分恰当，但自端方以后，已成为该青铜禁约定俗成的称谓<ref name="鉴古">{{cite journal |author1=孔令伟 |author2=严海晏 |title=鉴古——端方旧藏柉禁的称谓及图像纪录方式 |journal=新美术 |date=2012 |issue=6 |pages=43-50}}</ref>。

== 出土收藏 ==
光绪辛丑秋（1901年秋），柉禁器组在[[陝西省_(清)|陕西]][[鳳翔府|凤翔府]][[寶雞縣|宝鸡县]]三十里[[斗鸡台_(宝鸡)|斗鸡台]]<ref name="陶斋吉金录" />（今[[陕西省|陕西省]][[宝鸡市|宝鸡市]][[金台区|金台区]]戴家湾一带<ref>{{cite web |title=陕西宝鸡戴家弯的来历 |url=http://www.huayiw.com/world/html/2014/1/15/688.html |website=华裔网}}</ref>）的一座[[西周|西周]]贵族墓葬<ref name="Altar Set" />中出土，为戴家湾村村民王奎在村北坡地取土时发现的，包括柉禁器组在内的共有30余件青铜器<ref>{{cite web |author1=邹新社 |title=闲谝戴家湾 |url=http://www.xinshaan.com/whpd/ss/201811/08/t20181108_6602352.shtml |website=新陕网}}</ref>。

柉禁器组出土后辗转被清朝金石学家端方收藏，端方是[[晚清|晚清时期]]的重臣，生于[[直隶省|直隶]]的一个[[满族|满洲]]贵族家庭，曾在1899年出任陕西按察使、布政使、并代理[[陝西巡撫|陕西巡抚]]。柉禁器组出土的1901年，端方已经调任[[湖北巡抚|湖北巡抚]]。1911年，四川总督[[赵尔丰|赵尔丰]]因四川局势失控被免职，清廷命时任川汉粤汉铁路督办大臣的端方[[署理|署理]]四川总督，率湖北[[新军|新军]]入川鎮壓革命黨人。1911年11月27日，新军哗变，端方和他的弟弟在兵变中被杀<ref name="雅昌艺术网" />。

端方死后，其在京城的房产被人纵火烧毁，家道逐渐衰落。1924年春，端方后人将柉禁器组通过时任[[中华民国政府|中华民国政府]]高等政治顾问及[[中国科学美术杂志|中国科学美术杂志]]编辑的美国人[[福开森|福开森]]卖给了[[纽约|纽约]][[大都会博物馆|大都会博物馆]]<ref name="任雪莉"/>。当时卖出的总共20件器物，除上述的14件外，另有6件青铜杓（[[匕|匕]]）<ref name="雅昌艺术网" />。在运往美国之前，福开森将这组器件逐一摹拓打印，据器之大小而定纸之长短，一套15幅图，辑成《斋旧藏古禁全器》，[[郑孝胥|郑孝胥]]题签，影印出版<ref name="宋路霞">{{cite web |author1=宋路霞 |title=百年收藏 |url=http://old.njmuseum.com/zh/book/xszt/shouchan/slxzbnsc.html |website=南京博物院}}</ref>。据福开森称6件铜杓是发现柉禁的人第二次转售给端方的<ref name="鉴古" />。

== 摆放方式 ==
端方《陶斋吉金录》中的柉禁全图使将所有的13件酒器全部放置在禁台上。华觉明在《端方柉禁诸器的工艺考察》一文中则称，柉禁诸器在器物形制、纹饰和铸造技法上看，应是晚商至西周早期的作品，根据因长期放置而在禁面形成的器物痕迹，原先放置在禁上的有鼎卣二、雷纹觯、鼎尊、父甲觯和鼎卣一5件；斝不能与禁、卣、尊等配伍；妣己觚、子扫帚斝、父乙觯、亚形妣己觯共4件圆足器未在禁面留下器痕，不应与柉禁放在一起；而角、爵为三足器，很难留下器痕，是否与柉禁为同一器组存疑<ref name="鉴古" />。

1933年，日本人[[梅原末治|梅原末治]]所写的《柉禁の考古学的考察》中使用了一张共20器的照片，增加了6件结为一束放置在尊中的铜杓（[[匕|匕]]）。由于柉禁器组基本已被认定为十四器，大都会博物馆并未将铜杓与禁同时展出。柉禁器组被运到大都会博物馆后，在布置展台时将斝和盉以及两件觯被取下，铜斗被横陈在尊的前面。后来有进行了调整，仅留了5件酒器在禁面上<ref name="鉴古" />。

== 器组 ==
端方在《陶斋吉金录》绘制了《柉禁全图》一张，器物分图十三张，并有每件器物的说明及铭文拓印图案。书中柉禁器组共计有各类青铜器14件，包括柉禁、鼎卣一、鼎卣二、铜勺、鼎尊、父乙盉等。
