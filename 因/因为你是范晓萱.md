'''《因为你是范晓萱》'''是由[[中国大陆|中国大陆]][[歌手|歌手]]兼电视艺人[[刘维|刘维]]创作并演唱的一首单曲。刘维凭借这首单曲成功入选《[[中国好歌曲|中国好歌曲]]》第三季的“[[范晓萱|范晓萱]]战队”，<ref>{{cite web |title=刘维《因为你是范晓萱》港台受热捧不断升温 |url=http://music.yule.sohu.com/20160202/n436633599.shtml |website=[[搜狐网|搜狐网]] |publisher=搜狐音乐 |accessdate=2018-07-01}}</ref><ref>{{cite web |author1=蔡宜芳 |title=有偶像一定懂！最強致敬曲《因為你是范曉萱》逼哭本尊|url=https://star.ettoday.net/news/641344 |publisher=[[ETtoday新聞雲|ETtoday新聞雲]] |accessdate=2018-07-01}}</ref> 同时该单曲的正式数字版在2016年6月8日发行。<ref name="sohu">{{cite web |title=刘维致敬精神领袖《因为你是范晓萱》全新上线 |url=http://ent.southcn.com/8/2016-06/08/content_149103544.htm |website=南方网 |publisher=搜狐音乐 |accessdate=2018-07-01}}</ref>

== 创作背景 ==

== 歌词花絮 ==
因为[[刘维|刘维]]所创作的这首《因为你是范晓萱》是向自己的偶像范晓萱致敬，故而歌词中引用了不少范晓萱的歌曲名或代表歌词。<ref name="sohu" /> 例如：“一起唱着You are my sunshine”中《You Are My Sunshine》就是范晓萱专辑《[[还有别的办法吗|还有别的办法吗]]》中的歌曲名；“看我对我说我不是失败的man”中《失败的MAN》则是《[[我要我們在一起_(專輯)|我要我们在一起]]》中的歌曲名；“Rain fall in my heart”中《Rain》是同名专辑《Rain》中的歌曲名；“想要见你一面，如果我先死了怎么办”中《如果我先死了怎么办》是专辑《[[绝世名伶|绝世名伶]]》中的歌曲名。而歌曲末段的歌词“哎哟、哎哟、哎哟、哎哟、哎哟，你说你说我们要不要在一起？”则是完全引用专辑《[[我要我們在一起_(專輯)|我要我们在一起]]》同名主打歌的歌词。

== 评价争议 ==
《因为你是范晓萱》在豆瓣音乐上基于59人评价获得4.8分（满分10分）的评价。<ref>{{cite web |title=因为你是范晓萱 |url=https://music.douban.com/subject/26813050/ |publisher=豆瓣音乐 |accessdate=2018-07-02}}</ref> 比较热门的评论认为这首歌比较“尴尬”或者“旋律太熟悉”。<ref>{{cite web |title=因为你是范晓萱 评论区 |url=https://music.douban.com/subject/26813050/comments/ |publisher=豆瓣音乐 |accessdate=2018-07-02}}</ref> 而也有媒体报道，这首歌的[[副歌|副歌]]旋律非常类似韩国歌手[[白智荣|白智荣]]的《{{Link-ko|感性 (白智荣专辑)|Sensibility|像中枪一样}}》；并进一步怀疑歌曲还有抄袭[[李圣杰|李圣杰]]的《关于你的歌》等其他歌曲。<ref>{{cite web |title=《好歌曲》刘维致敬范晓萱神曲爆红 网友指其疑似抄袭 |url=http://ent.ifeng.com/a/20160201/42571248_0.shtml |publisher=凤凰网 |accessdate=2018-07-02}}</ref>
