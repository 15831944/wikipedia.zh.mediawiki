《'''來吧Shampoo'''》（{{lang-ja|おいでシャンプー}}）是日本女子組合[[乃木坂46|乃木坂46]]的第2張[[單曲|單曲]]，於2012年5月2日由[[Sony_Music_Records|Sony Music Records]]發售。

== 概要 ==
與前作《[[窗簾圍繞|窗簾圍繞]]》相隔約2個月發售的單曲。單曲分為附有DVD的Type-A、Type-B、Type-C和只附帶CD的CD盤，共4種形式發售。本作跟前作一樣，個人[[音樂影片|PV]]繼續收錄在Type-A、Type-B和Type-C，每個版本收錄11個人。Type-A、Type-B和Type-C在初回期間均附有[[生寫真|生寫真]]一張（每個版本11種，隨機封入一種）和全國握手會應徵券。<ref>{{ja icon}}{{cite web|url=http://www.nogizaka46.com/discography/?d=SRCL-7966&k= |title=ディスコグラフィー｜乃木坂46公式サイト |publisher=[[乃木坂46|乃木坂46]] |date=2012 |accessdate=2012-04-28}}</ref>

== 歌曲 ==
A面曲《來吧Shampoo》是[[宏達國際電子|HTC Nippon]]「[[au_(行動電話)|au]]（[[KDDI|KDDI]]）[[HTC_J|HTC J ISW13HT]]」的廣告歌曲，乃木坂46亦成為這部手提電話的代言人。<ref>{{ja icon}}{{cite web|url=http://natalie.mu/music/news/68150 |title=乃木坂46、最新スマホ発表会に参加＆テレビCMにも登場 |publisher=[[Natalie_(網站)|Natalie]] |date=2012-04-20 |accessdate=2012-04-28}}</ref><ref>{{cite web|url=http://www.cnyes.com/life/content/20120914/kfmq761lslgr2.shtml |title=「乃木坂46」初次訪台　擔任HTC J產品大使 |publisher=鉅亨網 |date=2012-09-14 |accessdate=2012-12-13}}</ref>《{{lang|ja|偶然を言い訳にして}}》是[[愛知電視台|愛知電視台]]和[[東京電視系|東京電視系]][[綜藝節目|綜藝節目]]《[[乃木坂在哪裡？|乃木坂在哪裡？]]》的片尾歌。

== 發售前 ==
3月13日，[[乃木坂46|乃木坂46]]宣佈將於5月2日發售其第2張單曲，當時還未定標題。由於跟[[AKB48|AKB48]]成員[[指原莉乃|指原莉乃]]的首張單曲《[[就是喜歡妳|就是喜歡妳]]》於同一天開始發售，這次單曲發售被稱為身為AKB48官方對手的乃木坂46與指原之間的「直接對決」。<ref>{{ja icon}}{{cite web|url=http://www.oricon.co.jp/news/music/2008369/full/ |title=乃木坂46、新曲で“ライバル”さしことガチ対決！ |publisher=[[Oricon|Oricon]] |date=2012-03-13 |accessdate=2012-04-28}}</ref> 4月2日，宣佈單曲名稱是《來吧Shampoo》。<ref>{{ja icon}}{{cite web|url=http://www.nogizaka46.com/news/2012/04/46-nd.php |title=乃木坂46 ２nd シングル楽曲名決定！ |publisher=[[乃木坂46|乃木坂46]] |date=2012-04-02 |accessdate=2012-04-28}}</ref>

4月2日至4日，單曲封面照以及本作特典「擬似約會影像」的刪剪版在乃木坂46於[[Youtube|Youtube]]的官方頻道和{{link-ja|GyaO!|GyaO!}}等公開，每日公開11段影像。<ref>{{ja icon}}{{cite web|url=http://www.barks.jp/news/?id=1000078343 |title=乃木坂46、メンバー33人の“疑似デート映像”公開。膨らむ妄想は無限大か!? |publisher=BARKS |date=2012-04-04 |accessdate=2012-04-28}}</ref>

4月8日，《來吧Shampoo》的歌舞在其冠名節目《乃木坂在哪裡？》上首次在電視上披露。4月16日，[[音樂影片|音樂影片]]（MV）在Youtube和GyaO!等公開。<ref>{{ja icon}}{{cite web|url=http://natalie.mu/music/news/67673 |title=乃木坂46「おいでシャンプー」PVで大胆ダンス披露 |publisher=[[Natalie_(網站)|Natalie]] |date=2012-04-12 |accessdate=2012-04-28}}</ref> 當中在歌曲的間奏部分，舞蹈中有將長裙掀起，露出內裏短裙的動作。<ref>{{ja icon}}{{cite web|url=http://www.oricon.co.jp/news/music/2010100/full/ |title=乃木坂46vsさしこ、今度は“美脚”対決!?  |publisher=[[Oricon|Oricon]] |date=2012-04-12 |accessdate=2012-04-28}}</ref> 由於該舞蹈部份在網上引起爭議，在4月16日，官方網站宣佈將會作出修改。<ref>{{ja icon}}{{cite web|url=http://www.nogizaka46.com/news/2012/04/post-43.php |title=「おいでシャンプー」振り付け変更についてのお知らせ |publisher=[[乃木坂46|乃木坂46]] |date=2012-04-16 |accessdate=2012-04-28}}</ref><ref>{{ja icon}}{{cite web |url=http://www.sanspo.com/geino/news/20120417/ido12041705010000-n1.html |title=乃木坂４６“スカートめくり”振り付け変更 |publisher=SANSPO.COM |date=2012-04-17 |accessdate=2012-04-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20120420043123/http://www.sanspo.com/geino/news/20120417/ido12041705010000-n1.html |archivedate=2012-04-20 }}</ref>

== 反應 ==
單曲於發售首日以11.1萬張的銷量取得當日[[Oricon公信榜|Oricon公信榜]]日榜第1位，[[指原莉乃|指原莉乃]]的《[[就是喜歡妳|就是喜歡妳]]》則以5.4萬張取得第2位，即是首天[[乃木坂46|乃木坂46]]在「直接對決」中先贏一仗<ref name="oricondaily">{{ja icon}}{{cite web|url=http://www.oricon.co.jp/news/rankmusic/2011026/full/ |title=【オリコン】AKBさしこ、乃木坂46に初日敗れる　同門対決はまさかのダブルスコア |publisher=[[Oricon|Oricon]] |date=2012-05-02 |accessdate=2012-05-03}}</ref>，其後更以15.6萬獲得周榜首位。<ref name="oriconweekly">{{ja icon}}{{cite web|url=http://www.oricon.co.jp/news/confidence/2011204/full/ |title=【オリコン】乃木坂46、AKB指原との同門対決制す　今年初の新人首位  |publisher=[[Oricon|Oricon]] |date=2012-05-08 |accessdate=2012-05-08}}</ref>另外，首日銷量亦比自身首張單曲《[[窗簾圍繞|窗簾圍繞]]》的8.5萬張多了2.6萬張。<ref name="oricondaily" />
