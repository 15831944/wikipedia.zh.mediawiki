'''來來電器廣場'''（'''{{lang-en|Royal Electronics Square}}'''）目前為全澳最大型、貨品種類最多的[[電器|電器]]零售店，其分店的覆蓋率亦為最廣。於2008年10月2日開設的首間[[旗艦店|旗艦店]]<ref>[http://www.173jobs.com/display/company/1/238.html 來來電器廣場招聘人才]</ref>，位於[[慕拉士大馬路|慕拉士大馬路]]來來集團大廈二樓，總面積達25,000呎。

== 沿革 ==
原本只經營超級市場業務的[[來來集團|來來集團]]，於2008年起新增電器業務，開設來來電器廣場，主要銷售對象為[[澳門人|澳門人]]。<br> 來來電器廣場一直以革新電器零售服務為理念，為顧客提供更舒適的購物環境，更合乎潮流的電器產品以及更多元化的服務選擇。為了將其優質的服務帶給本澳每一位顧客，自首間旗艦店開業至今，已積極地將其業務開拓至本澳每一個角落，至今來來電器廣場已是全澳分店最多的電器零售商。


== 服務 ==
作為全澳最大的電器零售商，推出多項特別服務藉以帶動本澳電器業轉型。

===正貨保證===
來來電器廣場於店內貼出正貨保證標誌，強調所有貨品均為正貨，以提升顧客對該店的信心。<br>

===10天保障服務===
顧客購物後若發現貨品有問題，10天內可將貨品連同有關購物單據帶到相關分店更換。<ref group="註">需符合有關條件方可更換同款產品。</ref><br>

===送貨服務===
購買電器產品滿澳門幣500或以上均可獲得送貨服務。<br>

===落樓易===
顧客購買新電器後，可要求來來電器廣場棄置舊電器。<br>

===免息寶===
有別於市面上一般免息分期的現金價與分期價不一的情況，來來電器廣場所推出的免息分期計劃不向顧客收取任何附加費用，即現金價與分期價相同。為全澳首間且唯一一間有此服務的電器鋪。<br><ref group="註">需受有關條款限制。</ref><br>

===葡幣標價===
為方便顧客購物，來來電器廣場內全線貨品使用[[葡幣|葡幣]]標價，有別於其他電器鋪常用[[港幣|港幣]]標價。

== 繳費方式 ==
來來電器廣場全線分店均可使用[[現金|現金]]、[[VISA|VISA]]、[[MasterCard|MasterCard]]、支付寶及澳門通付款。<ref group="註">使用VISA、MasterCard、支付寶及澳門通付款無須支付任何額外費用。</ref><br>

== 優惠 ==
來來電器廣場除了每個月的優惠活動，節日更不定期推出促銷活動(如︰來來Jetso Day)，購買指定貨品可獲特別優惠，可參考官網每月優惠。

===來來Jetso Day===
來來Jetso Day({{lang-zh|來來著數日}})為來來集團的促銷日，通常一年舉行2至3次，促銷期間全店大量貨品均有折扣優惠<ref>{{cite news|publisher=[[澳門日報|澳門日報]]|date=2011年5月13日A4版|title=來來Jetso Day全版廣告|url=http://www.macaodaily.com/html/2011-05/13/node_22.htm|deadurl=yes|archiveurl=https://web.archive.org/web/20110516064103/http://www.macaodaily.com/html/2011-05/13/node_22.htm|archivedate=2011年5月16日}}</ref>。根據過往紀錄，來來Jetso Day多於來來電器廣場旗艦店(位於[[來來集團大廈|來來集團大廈]])連同[[來來超級市場|來來超級市場]]及[[來來家居|來來家居]]舉行。
