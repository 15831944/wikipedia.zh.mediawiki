{{日本作品|來自繽紛世界的明日|色づく世界の明日から}}是由[[日本動畫|日本動畫]]公司[[P.A.Works|P.A.Works]]製作、[[篠原俊哉|篠原俊哉]]執導的原創電視動畫作品。系列全13話，於2018年10月6日至12月29日首播。

该作是一部有著細膩作畫與情感演出的青春群像劇<ref name="Yoru">{{cite web |title=【試片】《來自繽紛世界的明日》P.A.WORKS 與篠原俊哉再次合作原創動畫 |url=https://gnn.gamer.com.tw/2/169362.html |website=巴哈姆特電玩資訊站}}</ref>。舞台设定在一个日常生活中有着些许魔法的奇妙世界，主角月白瞳美年幼时就失去了[[色觉|色觉]]、是个缺乏情感表现的17岁高中女生，有一天突然被她的祖母·月白琥珀用魔法送往60年前。在那里，瞳美遇到了男生葵唯翔而取回了失去的色觉，在與同樣是17歲的祖母以及社團的夥伴們的交流當中逐漸成長。

==故事簡介==
故事的舞台是日常之中殘留些微魔法的[[长崎市|长崎市]]。主角是魔法使一族的末裔、17歲的月白瞳美，是一个小时候就失去了[[色覺|色覺]]、對人閉上心扉并且討厭魔法的女孩。有一天，她的祖母、大魔法使月白琥珀使用魔法將她送往60年前-2018年去見17歲的自己。
魔法傳送結束，瞳美掉入男生葵唯翔的房間，然后從窗戶爬出，被与唯翔同一社团的川合胡桃和風野淺蔥看见。眾人都誤以為瞳美是唯翔的女朋友，之后带她到月白家的魔法屋，然而高中時代的琥珀正在英國留學。其後瞳美向唯翔討回掉落在其房內的傳家耳飾時，意外地通過唯翔的繪畫見到世界的色彩。月白家安排瞳美入讀南丘高中，瞳美到校初日，再度遇上攝影美術社一行人。為澄清周圍朋友的誤會，唯翔要求瞳美施展魔法自我證明，不會魔法的瞳美勉強施展魔法，場面氛圍尷尬。之后瞳美希望唯翔让她再看一次繪畫，不太喜欢给别人看的唯翔因为抱著歉意便答应了并表示期待再次見到她施展魔法。在社團招生活動中，虽然发生了意外，但瞳美最终还是加入了攝影美術社。

琥珀回到日本，得知瞳美状况後更加努力學習魔法。回到学校后，个性开朗的琥珀因瞳美而與攝影美術社成員結識並急速融入。社團夜景拍攝活動中，瞳美在琥珀鼓勵下告訴同社成員自己來自未來。活動後，琥珀以魔法社身份入社，故合併為魔法攝影美術社。
為鼓勵創作瓶頸中的唯翔，瞳美在琥珀的指导下成功製作出魔法星砂相贈。社團外出活動時，瞳美意外踏入唯翔畫作中發現畫面背後的陰鬱和金色之魚的死亡，瞳美的追问引發唯翔強烈的抵觸。之後唯翔答應瞳美繼續畫下去并给她看，金色之魚復活且四處遊走，瞳美的色覺也一度恢復。琥珀對瞳美偶爾恢復色覺的契機展開調查，猜想色覺的恢復可能與瞳美動情有關，于是讓唯翔和瞳美復原各式相處模式。琥珀的實驗後，两人開始以名字相称。瞳美對攝影愈加熱忱，向社長山吹將學習技巧，指導後的將卻突然告白，瞳美起初極力逃避答覆，之後回絕時提到自己可能已有在意的人。暗戀將的淺蔥开始迴避瞳美，但在瞳美努力下，兩人很快就和好。

琥珀決定要在[[文化祭|文化祭]]推出可以讓大家進入畫作遊玩的活動，於是眾社員指派唯翔作畫。想見唯翔畫作的瞳美，在琥珀推動下成功練好進入畫作的複雜魔法。與眾社員一起進入畫作的唯翔在深處發現幼時封閉的瞳美，出來後瞳美說起自己母親是月白家首個不會魔法的成員，並在自己會使用魔法後的某日突然離家出走的往事。唯翔極力勸慰自責中的瞳美，吐露往事後的瞳美心覺寬慰。
文化祭前開始出現瞳美突然消失的現象，琥珀判斷必須要在文化祭的最後一日送還未來。告訴眾社員情況後，大家協助撿拾發動時光魔法所需的大量星砂。瞳美因留戀當下而糾結，特别是与唯翔的牵绊。在最后的时间里，瞳美和大家一起度过了文化祭。
時光魔法发动期间，大家一一与瞳美道別，唯翔卻只進行了簡單的告別。之后魔法突然失控，瞳美跟前來救自己的唯翔落入了單獨的空間。兩人終於說出了心裡話，互相表達了感謝，感謝對方將自己從自我封閉拯救，並且彼此告白。之後單獨空間隨即解除，瞳美的色覺也完全恢復。

回到了60年後的瞳美，在这段經歷後，終於明白了自己也可以變得幸福，要和琥珀一起去尋找失蹤的媽媽。然後從琥珀那拿到了大家一起準備的相冊，還有唯翔所作的繪本。雖然繪本是小時候與媽媽經常讀的，但是現在才發現原來繪本的內容就是自己的故事。
之後回到學校，主动加入了60年後的攝影美術社，加入后成了魔法攝影美術社，開展了拥有色彩的人生。

该作企划的立案是基于[[P.A.Works|P.A.Works]]的动画制作人（制片人）山本辉「想要描绘出至今为止只能看见黑白的风景，在一瞬间被赋予色彩的画面」<ref name=part1>{{cite web |script-title=ja:「色づく世界の明日から」篠原俊哉監督の密かな試み「『時間』と『人』の関係性にも触れられるのでは」 |url=https://news.nicovideo.jp/watch/nw4564258 |website=ニコニコニュース |date=2019-01-05 |language=ja}}</ref>。山本辉在《[[来自风平浪静的明天|来自风平浪静的明天]]》曾与导演[[篠原俊哉|篠原俊哉]]共事，当时就很喜欢导演的作风，希望有朝一日能一起做片；于是在2016年4月把企划拿给了他看，篠原俊哉也表示「如果是这个的话，或许可以做出我想做的」，于是就开始正式筹备。山本辉与筱原俊哉在动画制作阵容尚未敲定的情况下花了大约一年时间来思考故事的大框架，之后与infinite的[[永谷敬之|永谷敬之]]和[[Good_Smile_Company|Good Smile Company]]的小倉充俊两位製作人（出品人）进行了企划。企划团队成员全是男性，因此希望能有一个女性视角，还要能描绘出导演所设想的青春期内心的细腻，于是山本辉在2017年8月左右邀请了参与过《[[月色真美|月色真美]]》《[[Orange橘色奇蹟|Orange橘色奇蹟]]》等动画的[[柿原優子|柿原優子]]担任劇本統籌，这才开始着手制作正式的剧本。<ref name=Newtype201810>{{cite journal |title=気持ちひとつで世界は変わる |journal=[[月刊Newtype|月刊Newtype]] |year=2018 |issue=10 |pages=40-41 |language=ja}}</ref>

=== 故事创作 ===
为了实现山本辉最初的企划创意，于是把主角设定为看不见颜色的女生，然后将「色彩」和自我肯定感结合，确定主题为「世界因一个心情而变化」，并且一开始就决定描写以恋爱为主轴的青春群像剧<ref name=Newtype201810 /><ref name=part1 />。围绕故事主轴线「看不见颜色的女生，遇到了一个男生，然后变得能看见颜色了」，大家一起出谋划策<ref name=Newtype201810 />。
导演篠原俊哉很喜欢《[[湯姆的午夜花園|湯姆的午夜花園]]》这类涉及到时间的儿童文学，因此设计成主人公前往过去和同岁的祖母一同度过时光。关于[[时间旅行|时间旅行]]的实现方式，选择了与作品所希望的柔和触感切合度较高的魔法；之后为了平衡魔法世界和现实世界，想出了「如果利用星砂任何人都可以使用魔法」这样的主意，并且将魔法能做到的限定在一些无关紧要的事情。但很多非常细小的世界观设定如：魔法如何传入日本、魔法使的不成文规定、魔法屋的运营，在成品中几乎没有触及，因为不想为了说明而说明。<ref name=part1 />

作为主人公瞳美的故事，导演认为需要有一个不被[[欺凌|欺凌]]的氛围、世界无论何时都准备着接受瞳美，将着眼点聚焦在瞳美自身心理上。由于瞳美内心非常细腻但不怎么表现自己的感情，在描写这个角色的时候特别重视「不能因为瞳美的这种性格而让观众讨厌」，关于这点山本辉和小倉充俊也反映了很多像「不能把话说到这个地步」「做这种事情不太好吧」这样的意见。<ref name=part1 />
为了使瞳美所处的状况更加容易被理解，从最初的阶段就将琥珀设定成和瞳美性格正相反的角色<ref name=part2 />。原本打算让琥珀在第6话或第7话登场，但是因为Fly设计的人物原案实在是太好了，于是就让她早点登场了；而且製作人永谷敬之也说「让她早点出场吧」<ref>{{cite web |script-title=ja:「色づく世界の明日から」篠原俊哉監督が更に語る。結末は「最初から決まっていて、全く揺らいでない」(2/9) |url=https://www.excite.co.jp/news/article/E1547509242055/?p=2 |website=エキサイトニュース |date=2019-01-15 |language=ja |access-date=2019-03-12 |archive-url=https://web.archive.org/web/20190203205438/https://www.excite.co.jp/news/article/E1547509242055/?p=2 |archive-date=2019-02-03 |dead-url=no }}</ref>。
在世界观设定基本确定和主角七人的雏形出现后，剧情创作和丰满角色方面主要是由柿原優子推进的，千草和胡桃的关系也是基于柿原的主意<ref name=part1 />。在剧中浅葱和瞳美成为情敌后关系变僵，原本导演等人还想让这段感情更纠结一些，但在柿原的建议下让她们很快地就和好了<ref name=part2 />。

故事结局是最初开始就决定了的，没有变过。但关于60年间魔法摄影美术部成员身上有什么改变，在团队中也存在分歧，导演篠原俊哉认为应该交给观众自行体会，把这些全部一一说明是没必要的；製作人永谷敬之说：「如果拿出『结局就交给观众吧』这样的企划书肯定是行不通的，从市场的角度看就会被否定。」他希望看到跨越60年的时间、两人的思念重合到一起的这一瞬间。<ref name=part2>{{cite web |script-title=ja:「色づく世界の明日から」篠原俊哉監督が更に語る。結末は「最初から決まっていて、全く揺らいでない」 |url=https://news.nicovideo.jp/watch/nw4666198 |website=ニコニコニュース |date=2019-01-15 |language=ja}}</ref>

=== 画面与角色设计 ===
邀请插畫家{{link-ja|Fly (插畫家)|フライ (イラストレーター)|Fly}}设计角色原案的最大原因是，山本辉认为Fly笔下的角色有一种梦幻和朦胧的感觉、但整体上还有透明感和清洁感，与瞳美的形象最为贴近<ref name=Newtype201810 />。结果Fly也不负所望，而且将瞳美设计为银发这一点与山本辉和篠原俊哉不谋而合<ref>{{cite web |script-title=ja:「色づく世界の明日から」篠原俊哉監督の密かな試み「『時間』と『人』の関係性にも触れられるのでは」(8/9) |url=https://www.excite.co.jp/news/article/E1546609694086/?p=8 |website=エキサイトニュース |date=2019-01-05 |language=ja}}</ref>。秋山有希再调整角色設計以适合动画制作，并且担任總作畫監督。
画面的表现和制作方面，色彩和镜头是非常重要的主题<ref name=part2 />。在做魔法星砂设定的时候，失去了色觉的主人公和星砂贩卖店内陈列的五颜六色的星砂形成强烈对比的画面浮现在导演脑中，篠原俊哉参考了山本辉收集的颜料店之类的照片后，确定了魔法屋的视觉图<ref name=part1 />。导演认为通过[[广角镜头|广角镜头]]难以表现出情绪，所以有意地加入更多[[遠攝鏡頭|遠攝鏡頭]]<ref name=part2 />。并且经常说「背景的质量是决定作品品质的重大因素」，也很重视画面的镜头感，注意景深、光的表现、空气感、透明感等；使用的色彩数也比P.A.Works之前的作品都要多，负责色彩设计的中野尚美下了不小功夫<ref name=Newtype201810 />。

===音樂===

== 反響 ==
《来自多彩世界的明天》的画面表现得到了评论家的普遍好评<ref>{{cite web |title=The Fall 2018 Anime Preview Guide - IRODUKU: The World in Colors |url=https://www.animenewsnetwork.com/preview-guide/2018/fall/iroduku-the-world-in-colors/.137739 |website=Anime News Network |date=2018-10-05 |language=en |access-date=2019-04-05 |archive-url=https://web.archive.org/web/20190417183358/https://www.animenewsnetwork.com/preview-guide/2018/fall/iroduku-the-world-in-colors/.137739 |archive-date=2019-04-17 |dead-url=no }}</ref>。細緻的場景與鮮艷豐富的色彩讓觀眾留下深刻的印象，niconico新闻的评论文章称：美到令人窒息的画面达到了治愈的效果，可以净化现代社会中疲惫的心灵<ref>{{cite web |script-title=ja:もはやテラピーの域！ 『色づく世界の明日から』の圧倒的クオリティで描かれる色彩美で心をデトックスしよう |url=https://news.nicovideo.jp/watch/nw4399895 |website=ニコニコニュース |language=ja}}</ref>。美麗的景像強烈的襯托出了看不見色彩的瞳美眼中灰白的世界，也讓第一話結尾、瞳美看見葵唯翔畫作上色彩時的衝擊得到優秀的展現<ref name="Yoru" />。同時，P.A.Works也維持了一貫優秀的人物作畫，細膩的動作與表情展現出許多台詞以外的情感<ref name="Yoru" />。故事方面，不少评论指出前期剧情发展缓慢，这是一些观众在动画完结之前弃剧的原因，但这种节奏有助于角色关系的发展以及故事的建立<ref>{{cite web |author=Rebecca Silverman |title=IRODUKU: The World in Colors ‒ Episodes 1-3 |url=https://www.animenewsnetwork.com/review/iroduku-the-world-in-colors/episodes-1-3/.138423 |website=Anime News Network |date=2018-10-21 |language=en |access-date=2019-04-05 |archive-url=https://web.archive.org/web/20181025194959/https://www.animenewsnetwork.com/review/iroduku-the-world-in-colors/episodes-1-3/.138423 |archive-date=2018-10-25 |dead-url=no }}</ref>，最后几集则变得越来越好<ref name=Gracie>{{cite web |author=Gracie |title=Final Impressions: IRODUKU The World in Colors |url=https://anitrendz.net/news/2019/01/07/final-impressions-iroduku-the-world-in-colors/ |website=Anime Trending News |date=2019-01-08 |language=en}}</ref>。
[[動畫新聞網|動畫新聞網]]（Anime News Network）的评论员Theron Martin评论称该作从未令人亢奋或表现强烈情感，但汇集了高品质的艺术表现形式，并辅以一个整洁的故事概念、发展良好和可信的角色，以及年度最佳开场之一<ref name=Best>{{cite web |title=The Best (and Worst) Anime of Fall 2018 |url=https://www.animenewsnetwork.com/feature/2019-01-04/the-best-anime-of-fall-2018/.141658 |website=Anime News Network |date=2019-01-04 |language=en |access-date=2019-04-05 |archive-url=https://web.archive.org/web/20190107223915/https://www.animenewsnetwork.com/feature/2019-01-04/the-best-anime-of-fall-2018/.141658 |archive-date=2019-01-07 |dead-url=no }}</ref>。Anime Trending News的Gracie指出：角色有点刻板定型，角色虽然有发展，但发展相对较小且相当可预测；角色之外则没有什么可批评的了，整体表现远远超出预期<ref name=Gracie />。動畫新聞網的Rebecca Silverman认为作品中过剩的象征意象处理得非常好，也留下了一些细节让观众寻找<ref name=Best />，故事的神秘性和不确定性是这个系列引人入胜的重要原因<ref>{{cite web |author1=Rebecca Silverman |title=IRODUKU: The World in Colors ‒ Episode 4 |url=https://www.animenewsnetwork.com/review/iroduku-the-world-in-colors/episode-4/.138727 |website=Anime News Network |date=2018-10-29 |language=en |access-date=2019-04-05 |archive-url=https://web.archive.org/web/20181029224542/https://www.animenewsnetwork.com/review/iroduku-the-world-in-colors/episode-4/.138727 |archive-date=2018-10-29 |dead-url=no }}</ref>。虽然故事并不是完美的，但它确实是一段美丽的旅程；正如俗话所说，有时候目标不是要达到顶峰，而是要享受过程<ref>{{cite web |author=Rebecca Silverman |title=Series Finale: IRODUKU: The World in Colors ‒ Episode 13 |url=https://www.animenewsnetwork.com/review/iroduku-the-world-in-colors/episode-13/.141433 |website=Anime News Network |date=2018-12-29 |language=en |access-date=2019-04-05 |archive-url=https://web.archive.org/web/20181229223351/https://www.animenewsnetwork.com/review/iroduku-the-world-in-colors/episode-13/.141433 |archive-date=2018-12-29 |dead-url=no }}</ref>。
日本小说家[[吉本芭娜娜|吉本芭娜娜]]认为动画的品质非常高，其中细节的刻画很赞、色彩也很漂亮、故事也用印象很好地传达出来，体现出了创作者对作品的爱<ref>{{cite web |title=评价很高！《多彩世界》得到日本小说家关注 |url=https://new.qq.com/cmsn/20190105/20190105005509.html |website=腾讯网 |access-date=2019-04-05 |archive-url=https://web.archive.org/web/20190105210106/https://new.qq.com/cmsn/20190105/20190105005509.html |archive-date=2019-01-05 |dead-url=no }}</ref>。

Anime! Anime!（アニメ！アニメ！）网站以「2018年秋季最佳动画」为题在2018年12月12日至12月19日进行了读者问卷调查，结果显示该作获得约7%的支持率、排名第3<ref>{{cite web |script-title=ja:2018年“秋アニメ”で一番良かったアニメ作品は？ 3位「色づく」、2位「ゾンサガ」、1位は… |url=https://animeanime.jp/article/2018/12/26/42335.html |website=アニメ！アニメ！ |date=2018-12-26 |language=ja |access-date=2019-03-12 |archive-url=https://web.archive.org/web/20190227072635/https://animeanime.jp/article/2018/12/26/42335.html |archive-date=2019-02-27 |dead-url=no }}</ref>。但在2018年12月28日[[niconico動畫|niconico動畫]]公布的「媒體總評2018秋番排行」中，并未入选前十<ref>{{cite web |title=在niconico秋番總選舉之前公開的是「媒體總評2018秋番排行」，本季霸權是喪屍偶像!! |url=https://www.gamebase.com.tw/news/topic/99135346/ |website=遊戲基地 |date=2018-12-28}}</ref>。
作品的现实舞台地[[长崎市|长崎市]]引起了观众们的注目，不少观众以[[朝聖_(通俗)|聖地巡礼]]的形式体验作品世界，[[长崎县|长崎县]]电影委员会（Film commission）也致力于相关地点的宣传<ref>{{cite web |script-title=ja:アニメ“聖地巡礼”ファン続々 長崎市舞台「色づく世界の明日から」 長崎南高もモデル、実在の「魔女カフェ」も |url=https://www.nishinippon.co.jp/nnp/nagasaki/article/494835/ |website=西日本新聞Web |date=2019-03-17 |language=ja |access-date=2019-04-05 |archive-url=https://web.archive.org/web/20190320223126/https://www.nishinippon.co.jp/nnp/nagasaki/article/494835/ |archive-date=2019-03-20 |dead-url=no }}</ref>。该作也被选为长崎县电影委员会支援动画，并公开了一段1分钟长的宣传动画<ref>{{Twitter status |user=NagasakiFC |statusid=1082935095084802048 |title=長崎県フィルムコミッション支援アニメ「色づく世界の明日から」 |name=長崎県フィルムコミッション |date=2019-01-09 |language=ja}}</ref>。
