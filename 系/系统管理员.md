'''系统管理员'''（System Administrator，簡稱'''SA'''）是负责管理计算机系统的人。其具体的含义因环境而异。

拥有庞大复杂的计算机系统的组织，通常按照专长分派计算机员工。在那里系统管理员负责[[维护|维护]]现有的[[计算机系统|计算机系统]]，因此其工作不同于：

*[[系统设计师|系统设计师]]（Systems design，SD）
*[[系统分析师|系统分析师]]（Systems analyst，SA）
*[[数据库管理员|数据库管理员]]（Database administrator，DBA）
*[[存储管理员|存储管理员]]（Storage Administrator）
*[[程序员|程序员]]（Programmer，PG）
*[[技术支持|技术支持]]（Technical support），或[[信息技术|信息技术]]（Information technology，IT）
*[[網路管理員|網路管理員]]（Network administrator）
*[[安全管理員|安全管理員]] ( Security administrator )
*[[網站管理員|網站管理員]]（Web administrator，Webmaster）

而在较小的企业和组织中，没有这么多不同的专家。系统管理员的称呼含义更加广泛 - 系统管理员就是那个了解计算机系统如何工作，并能处理有关问题的人。

== 系统管理员的技能 ==
*计算机基础理论
*计算机网络
*计算机操作系统

==系统管理相关领域==
*计算机房管理
*计算机服务器
*计算机操作系统
*计算机网络
*系统[[备份|备份]]和恢复
*个人计算机
*[[数据库|数据库]]
*系统和用户账号管理
*计算机安全策略执行
*计算机系统安装、维护、升级、报废、回收

== 系统管理员的工作范围 ==
*回答有关计算机系统的技术问题
*安装和配置新的硬件、软件
*进行[[操作系统|操作系统]]升级，更改配置
*执行数据备份
*负责系统的安全
*编写和更新系统配置文档
*调整和优化系统性能
*[[排错|排除故障]]
*连接其他系统
*安装和配置网络服务

== 系统管理员的知识和技能 ==
*计算机硬件和外部设备
*计算机网络
*操作系统
*网络服务程序
*应用程序
*系统管理专用工具
*[[项目管理|项目管理]]
*[[软件配置管理|配置管理]]
*编写程序
