{{日本作品|綻放花朵的飛行兵器|花咲けるエリアルフォース}}，為[[杉井光|杉井光]]撰寫，[[LLO|LLO]]繪製封面與插畫的一部[[輕小說|輕小說]]作品，日文[[文庫本|文庫本]]於2011年2月由[[GAGAGA文庫|GAGAGA文庫]]（[[小學館|小學館]]所屬）出版<ref>{{cite web|title=ライトノベル「花咲けるエリアルフォース」発売！|url=http://news.searchina.ne.jp/disp.cgi?y=2011&d=0222&f=national_0222_011.shtml}}</ref>，繁體中文版由臺灣[[尖端出版社|尖端出版社]]代理<ref>[http://item.jd.com/16045149.html 《綻放花朵的飛行兵器01》(杉井光)]，摘要书评试读 - 京东图书</ref><ref>[http://www.cp1897.com.hk/product_info.php?BookId=9789571048116 《綻放花朵的飛行兵器》]，商務印書館，出版社：尖端出版股份有限公司，出版日期：2012年8月，ISBN：9789571048116</ref><ref>[http://acg.gamer.com.tw/acgDetail.php?s=57233 綻放花朵的飛行兵器]，巴哈姆特</ref>。

== 概要 ==
杉井氏首次於[[GAGAGA文庫|GAGAGA文庫]]所刊載之作品，是其出道時期所撰文之其中一長篇作品，保留當中關於兵器驅動之設定所修改而成的作品。因文中內容所涉及之諸多有關議題之故，於文末後記表明了，此乃其寫過中「最危險的故事」。<ref>詳見第1卷後記，文庫本第327頁，尖端中譯本第287頁。</ref>

== 簡介 ==
因國家[[內戰|內戰]]動盪的緣故，成為戰爭[[孤兒|孤兒]]的主角'''仁川佑樹'''，因皇國軍戰事的節節敗退而不斷轉學遷徙。在距開戰2年後成為中二生的祐樹，與從天而降的少女──'''櫻子'''相遇了，也因為某種契機之故，被徵招成為特殊[[戰鬥機|戰鬥機]]的駕駛員。也因此展開了亡命蒼穹的軍旅之途<ref>[http://www.cite.com.tw/book?id=25424 綻放花朵的飛行兵器 杉井光(Hikaru Sugii)]，城邦讀書花園</ref>。

; {{nihongo|櫻子|桜子（さくらこ）}}

; {{nihongo|那須野 佳織|那須野 佳織（なすの かおリ）}}

: 相對於柚樹與櫻子，與班上的同學相處十分融洽，同儕間完全不會在意其軍人的身分。<ref>詳見第1卷，文庫本第118、119頁，尖端中譯本第100頁。</ref>
; {{nihongo|吉田 神樂|吉田 神楽（よしだ かぐら）}}

; {{nihongo|土佐 美鶴|土佐 美鶴（とさ みつる）}}
: 皇國空軍獨立第八航空軍團，即飛行[[航空母艦|航空母艦]]「三栖籏」的代理艦長，軍階為[[中校|中校]]。與櫻子年幼時便相識，軍部亦以此一原由，而令其擔負三栖籏的代理艦長一職。<ref>詳見第1卷，文庫本第195頁，尖端中譯本第169頁。</ref>也不知是否因之所故，對於軍隊的紀律十分藐視，時常衣裝不整，並如同標誌班地叼著[[香菸|香菸]]，有時也會帶著[[墨鏡|墨鏡]]。<ref>她叼著香菸，把軍服被在肩膀上，臉上畫著完美無瑕的妝，而且年輕到就算有人說她去年還是高中不良少女，我也會有一瞬間相信的地步。摘錄自第1卷，文庫本第48、49、50頁，尖端中譯本第37頁。</ref>
: 十分厭惡他人以[[軍階|軍階]]的方式稱呼之，如文中便要求佑樹直接以美鶴的方式叫喚，其後佑樹皆稱之為美鶴小姐（{{lang|ja|美鶴さんは}}）。<ref>詳見第1卷，文庫本第54、55頁，尖端中譯本第43頁。</ref>
: 寢室宛如人間煉獄般地凌亂，堆疊至無法再堆疊的文件、盡是菸蒂的菸灰缸，任意丟放的衣物與菸盒充斥著整個房間。<ref>詳見第1卷，文庫本第196、197頁，尖端中譯本第170、171頁。</ref>
; {{nihongo|栗園 珠美|栗園 珠美（くりぞの たまみ）}}
: 九段下中學，佑樹所屬班級的班導師。同時也是皇國空軍，教練航空團的訓練教官，軍階為[[上尉|上尉]]。個性好管閒事又不帶惡意，平時傻里傻氣的樣子，令祐樹感到難以應對相處。<ref>詳見第1卷，文庫本第101、102頁，尖端中譯本第85、86頁。</ref>

== 用語 ==
; {{nihongo|國立九段下中學|国立九段下中学校}}
: 作品中主角升上中學後第四次轉學所轉入的學校。<ref>詳見第1卷，文庫本第25頁，尖端中譯本第16頁。</ref>依據故事中所撰之敘述，應是以比鄰[[靖國神社|靖國神社]][[參道|參道]]、[[日本武道館|武道館]]與[[皇居|皇居]]的{{tsl|ja|千代田区立九段中等教育学校|千代田区立九段中等教育学校}}（原{{tsl|ja|東京都立九段高等学校|東京都立九段高等学校}}）為藍本，所虛構之學校。<ref>中學後門外面是一條不起眼的冷清車道，行道樹與灌木萌發著初夏的炫目綠葉。；「那是武道館啦。你是怎麼把那麼大的建築物看成涼亭的？還有那一帶都是皇居。」；摘錄自第1卷，文庫本第65、66頁，尖端中譯本第53頁。</ref>
: 制服的樣式因應部分學員中的軍人身分，所設計的樣式與皇國軍的軍服十分相似。<ref>為了讓他們在緊急狀況時能直接執行軍務，才把制服作成類似軍服的樣子。摘錄自第1卷，文庫本第60頁，尖端中譯本第48頁。</ref>另外，九段下中學的教職員中亦有半數同時具軍人的身份。<ref>詳見第1卷，文庫本第101、102頁，尖端中譯本第85頁。</ref>
; {{nihongo|皇國軍、民國軍|皇国軍、民国軍}}
: 不知何故日本以[[箱根|箱根]][[絕對防衛線|絕對防衛線]]為界分裂為皇國及民國兩大勢力展開內戰,史稱「極東動亂」。雙方均擁有完善的政府機制,因此互不承認,也不將此次內戰視為戰爭。作品中對民國的著墨不多,目前只知道民國的主要政黨是一個叫「民權黨」的政黨,皇國則是以天皇為國家元首，{{what|政體為絶對君主制（君主立憲制）}}，內閣及國會也漸次設置。
; {{nihongo|皇國軍章|皇国軍章}}
: 能投射出立體影像並加以操作的高機能軍用植入式存取器，如作品中的佑樹即在其左手指甲的皮膚植入此一[[晶片|晶片]]。<ref>第1卷，文庫本第52頁，尖端中譯本第41頁。</ref>軍章具有許多的功能，如連結資料庫，調閱他人的個人資料或其他紀錄、即時戰情資料、物價查詢、具軍人優惠價格的[[電子錢包|電子錢包]]、紀錄影音資料，甚至是確認友軍的生命反應等。<ref>詳見第1卷，文庫本第61、93、118、119、246、275、282、289頁，尖端中譯本第49、78、100、216、241、247、254頁。</ref>軍章間也可以相互交換彼此的資料。<ref>詳見地1卷，文庫本第68頁，尖端中譯本第82頁。</ref>
; {{nihongo|三栖籏|三栖籏（みすはた）}}
:飛天航空母艦,以受損的伍號機「清竹」作動力來源。 
; {{nihongo|特殊攻擊機‧櫻花|特殊攻擊機・桜花（おうか）}}
: 世上最強的航空戰鬥兵器,在空中散佈象徵淒美與滅亡的櫻花花瓣，一一擊落侵犯國土的敵機，僅有九架，以染井吉野櫻為材料製做出來，損毀時會自動再生。「櫻花機」顛覆了飛行的常識，除了需要詭異的飛行技巧，還需要找到適任者。
; {{nihongo|散華、滅卻|散華（バタナスハ）、滅卻（アブハヴア）}}
: 櫻花特殊攻擊機把機體散出去像雲一樣把大量敵機包圍其中,然後同時引爆
; {{nihongo|多腳砲臺|多脚砲台}}
: 依據作品中的描述，為造型類似蜘蛛，具登陸作戰能力，但獨立作戰之性能不足以突破箱根絕對防禦線的機動性作戰兵器。<ref>詳見第1卷，文庫本第198、217、218頁，尖端中譯本第172、189、190頁。另外，類似的兵器設定可參考條目：[[蘋果核戰|蘋果核戰]]中的同名用語：「'''{{lang|ja|多脚砲台}}'''」，或電視動畫《{{lang|ja|[[空·之·音|空·之·音]]}}》中的「'''[[建御雷|建御雷]]'''」。</ref>
; {{nihongo|靈璽簿|霊璽簿（れいじぼ）}}
: 即戰死者名簿，奉納於靖國神社中的'''靈璽簿奉安殿'''（{{lang|ja|霊璽簿奉安殿（れいじぼほうあんでん）}}）中。<ref>詳見第1卷，文庫本第72、73頁，尖端中譯本第59、60頁。</ref>
