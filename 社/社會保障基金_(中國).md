'''社會保障基金'''，簡稱'''社保基金'''，俗稱老百姓的「养命钱」，是[[基金|基金]]的一種。[[中國大陸|中國大陸]]的[[社會保障|社保]]基金開始出現於1980年代末，而在2002年已經形成數千億[[人民幣|人民幣]]的規模。目前全國社會保障基金理事會理事長是[[刘伟_(1961年)|刘伟]]。

== 概述 ==
全國社保基金在2006年3月已經在[[香港交易所|香港交易所]]開戶。外界投資者憧憬[[QDII|QDII]]的安排，有助社保基金北水南調，推動外間[[資本市場|資本市場]]的發展。<ref>http://news.xinhuanet.com/stock/2005-01/13/content_2454791.htm</ref><ref>http://insurance.money.hexun.com/3877_1830493A.shtml {{dead link}}</ref>

截至2009年12月31日，全國社保基金會管理的基金資產總額為7765億元，比上年增長38%，其中，基金權益6927億元，較上年增加2123億元。

2009年，全國社保基金投資收益849億元，投資收益率為16.1%，其中，已實現收益426億元，浮盈423億元。截至2011年底，全國社保基金成立9年來累計投資收益2448億元，年均投資收益率為9.75%。

全国社会保障基金理事会2013年年报显示，社保基金会管理的基金资产总额12415.64亿元。其中，社保基金会直接投资资产6697.74亿元，占比53.95％；委托投资资产5717.90亿元，占比46.05％。

2013年，全国社保基金基金权益投资收益额685.87亿元，投资收益率6.20％，超过同期2.6％的通货膨胀率。自成立以来，全国社保基金累计投资收益额4187.38亿元，年均投资收益率8.13％，超过同期2.46％的年均通货膨胀率。<ref>[http://www.ce.cn/xwzx/gnsz/gdxw/201406/30/t20140630_3063931.shtml 全国社保基金投资多元 年均收益率逾8％远超CPI]，中国经济网，2014年6月30日</ref>

截至2013年年末，中国基本养老保险基金累计结存31275亿元。目前养老保险基金由各级社保部门进行管理。按照现行的政策规定，养老基金只能存银行和买国债。在经济持续增长，人口老龄化趋势加快的形势下，基金的投资范围偏窄、运营效率偏低、贬值缩水的风险越来越凸显。

据人社保部公布的《2013年全国社会保险情况》，去年3万多亿养老保险基金中只有1920亿元是主动投资于债券、委托运营和协议存款，约占基金总规模的6％，其他资金均被动地存放在银行账户上。中国社科院世界社会保障中心主任郑秉文认为，通过这种投资策略，10多年来养老金收益率平均不到2％，扣除通货膨胀率，年均收益率为负数，处于贬值风险之中。<ref>[http://finance.ce.cn/rolling/201407/31/t20140731_3263376.shtml 我国养老保险基金“所有者缺位”亟须解决]，中国经济网，2014年7月31日</ref>

== 種類 ==
社會保障基金之下分為三部份：
# 社会保险基金：資金來自[[企业|企业]]及个人缴费，名堂是[[中國養老保險|养老保险]]、失业保险、医疗保险、工伤保险和生育保险。
# [[全国社会保障基金理事会|全国社会保障基金]]：資金來自中央财政拨款、国有股及股权、经国务院批准的其他集资及投资收益。
# 补充保障基金：資金來自企业及个人缴费补充等。
