'''社區網絡聯盟'''（[[英文|英文]]：'''Community Network Union'''），香港政治聯盟，2017年8月6日正式成立，主要做地區工作<ref name = 蘋果>{{cite_news|url=http://hk.apple.nextmedia.com/realtime/news/20170806/57048612|title=傘後團體組政治聯盟冀區會奪15席 戴耀廷：可取30席|work=《[[蘋果日報_(香港)|蘋果日報]]》|date=2017-08-06|accessdate=2017-08-06}}</ref>。

==成立==
[[File:社區網絡聯盟成立典禮六人合照.jpg|thumb]]、[[李晉希|李晉希]]、[[劉頴匡|劉頴匡]]、[[張文龍|張文龍]]、[[林錫添|林錫添]]、[[羅焯勇|羅焯勇]]於社區網絡聯盟成立典禮合照。]]
2017年8月6日，6個社區組織[[葵青連結動力|葵青連結動力]]、[[沙田社區網絡|沙田社區網絡]]、[[荃灣社區網絡|荃灣社區網絡]]、[[屯門社區網絡|屯門社區網絡]]、[[天水圍社區網絡|天水圍社區網絡]]、[[藍田社區網絡|藍田社區網絡]]正式成立社區網絡聯盟。<ref name = 蘋果/><ref>{{cite_news|url=http://tmhk.org/2017/08/%E5%85%AD%E5%82%98%E5%BE%8C%E5%9C%98%E9%AB%94%E6%88%90%E7%AB%8B%E3%80%8C%E7%A4%BE%E5%8D%80%E7%B6%B2%E7%B5%A1%E8%81%AF%E7%9B%9F%E3%80%8D-%E5%8A%8D%E6%8C%87%E4%BA%8C%E9%9B%B6%E4%B8%80%E4%B9%9D%E5%B9%B4/|title=六傘後團體成立「社區網絡聯盟」 劍指二零一九年區議會選舉|work=《[[TMHK|TMHK]]》|date=2017-08-06|accessdate=2017-08-06}}</ref>社區網絡聯盟發言人[[劉頴匡|劉頴匡]]說，其等不想受政黨束縛，所以就成立組織做社區工作。而其等都面對到資源分配不足，所以就成立一個平臺分享資源。<ref name = 明報>{{cite_news|url=https://news.mingpao.com/pns/dailynews/web_tc/article/20170807/s00002/1502041864757|title=傘後組織結盟 盼區會奪15席|work=《[[明報|明報]]》|date=2017-08-07|accessdate=2017-08-07}}</ref>

2018年2月9日，荃灣社區網絡和葵青連結動力退出社區網絡聯盟。黃丹晴也辭任社區網絡聯盟公共關係一職。2月10日，[[屯門社區網絡|屯門社區網絡]]退出社區網絡聯盟。
