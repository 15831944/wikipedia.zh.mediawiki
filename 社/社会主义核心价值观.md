'''社会主义核心价值观'''是[[中国共产党|中国共产党]]从2012年[[中国共产党第十八次全国代表大会|中国共产党第十八次全国代表大会]]起提倡的[[价值观|价值观]]。

== 历史 ==
2012年11月8日，[[中共中央总书记|中共中央总书记]][[胡锦涛|胡锦涛]]代表[[中国共产党第十七届中央委员会|中国共产党第十七届中央委员会]]向[[中国共产党第十八次全国代表大会|中国共产党第十八次全国代表大会]]所作的报告中首次提出了社会主义核心价值观的基本内容，共24个字<ref>{{cite news |url = http://theory.people.com.cn/n/2014/0212/c40531-24330538.html |title = 社会主义核心价值观基本内容 |accessdate=2014-11-13 |date=2014-02-12 |publisher=人民网 |language = zh-cn}}</ref>。

2013年12月23日，[[中共中央办公厅|中共中央办公厅]]印发的《{{到维基文库|关于培育和践行社会主义核心价值观的意见}}》将其分成3个层面<ref>{{cite news |url = http://www.chinadaily.com.cn/hqgj/jryw/2014-02-12/content_11195801.html |title = 社会主义核心价值观24字基本内容公布|accessdate=2014-11-13 | date=2014-02-12 |publisher=中国日报网 |language=zh-cn}}</ref>。

2014年2月12日《[[人民日报|人民日报]]》头版予以刊登<ref>{{cite web |url=http://paper.people.com.cn/rmrb/html/2014-02/12/nw.D110000renmrb_20140212_10-01.htm |title=社会主义核心价值观基本内容 |accessdate=2014-11-13 |date=2014-02-12 |work=人民日报 |publisher=人民网 |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20140712064034/http://paper.people.com.cn/rmrb/html/2014-02/12/nw.D110000renmrb_20140212_10-01.htm |archivedate=2014-07-12 }}
</ref>。
中共中央总书记[[习近平|习近平]]自上任始就多次提及社会主义核心价值观，强调[[社会主义|社会主义]]在中国社会的领导性、积极性与先进性。<ref>{{cite web |url = http://news.xinhuanet.com/politics/2014-03/26/c_126318353.htm|title = 习近平论社会主义核心价值观——十八大以来重要论述选编 |accessdate= 2014-05-30  |date= 2015-02-06 | publisher=人民网 |language=zh-cn }}</ref><ref>{{cite web |url = http://news.xinhuanet.com/politics/2014-03/26/c_126318353.htm |title = 习近平:从小积极培育和践行社会主义核心价值观 |accessdate= 2015-02-06 |date= 2014-05-30 | publisher=新华网 |language=zh-cn }}</ref>

== 内容 ==
* 国家层面的价值目标：'''富强、民主、文明、和谐'''
* 社会层面的价值取向：'''自由、平等、公正、法治'''
* 公民个人层面的价值准则：'''爱国、敬业、诚信、友善'''

== 宣传与考核 ==
[[File:社會主義核心價值觀宣傳海報.jpg|thumb]][[思明區|思明區]][[中華街道_(廈門市)|中華街道]]。]]
[[File:社会主义核心价值观在一家银行的电子显示屏上展示.jpg|thumb]]的[[新闻跑马灯|电子显示屏]]滚动展示社会主义核心价值观。]]
为了“培育和践行社会主义核心价值观”，[[中华人民共和国|中华人民共和国]]各级行政部门下达通知文件，要求各个[[学校|学校]]、[[医院|医院]]、[[企业|企业]]、[[网吧|网吧]]、商铺、[[购物中心|购物中心]]、[[公交车|公交车]]公司等单位通过宣传展板、横幅或电子显示屏等媒介展示社会主义核心价值观24字的全文<ref>{{cite web|title=中共崇信县委办公室   关于印发《崇信县社会主义核心价值观“24字 人知人晓工程”工作方案》的通知|url=http://www.chongxin.gansu.gov.cn/contents/105/93647.html|publisher=甘肃·崇信|archiveurl=https://web.archive.org/web/20141220124126/http://www.chongxin.gansu.gov.cn/contents/105/93647.html|archivedate=2014-12-20}}</ref>。目前，社会主义核心价值观的宣传已经遍布中国大陆各个城市的大街小巷。

根据[[中央文明办|中央文明办]]下属的“[[中国文明网|中国文明网]]”的报道，2017年广东肇庆市在城区的牌坊广场、东门广场上的两组时钟，以“社会主义核心价值观”内容为时间刻度，并评论称此举“是肇庆创建文明城市进程中的又一壮举”。<ref>{{cite news|title=肇庆价值观表盘走红 网友议“正确的价值观如时间般珍贵”|url=http://www.wenming.cn/dfcz/gd/201707/t20170707_4329927.shtml|accessdate=2017-07-08|publisher=[[中国文明网|中国文明网]]|date=2017-07-06}}</ref>另据媒体报道，此前这里时钟的表盘内容为“[[社会主义荣辱观|社会主义荣辱观]]”（“八荣八耻”）。<ref>{{cite news|title=广东现“社会主义价值观”时钟 遭讽|url=http://china.dwnews.com/news/2017-07-05/59823862.html|accessdate=2017-07-08|publisher=[[多维新闻|多维新闻]]|date=2017-07-05}}</ref>

此外，一些由中国共产党所主导的考核、评测活动，也将社会主义核心价值观建设情况列为考核的一部分。以第四届[[全国文明城市|全国文明城市]]评选测评为例，这一轮全国文明城市创建和以往有很大的不同<ref>{{cite news|author=李源|title=武汉全城动员冲刺全国文明城市 阮成发要求：全力以赴背水一战|url=http://news.cnhubei.com/xw/wuhan/201410/t3068844_4.shtml|accessdate=2015-01-01|newspaper=湖北日报|date=2014-10-15|quote=据悉，这一轮全国文明城市创建和以往有很大的不同，即把培育和践行社会主义核心价值观放在重中之重，贯穿在文明创建的各个方面，尤其是在测评体系中，从5个方面提出了16条标准，分值和比重都很大。}}</ref>，“培育和践行社会主义核心价值观”成为评价体系中权重最高的指标。测评时要求：
{{quote|
一是要求提供今年以来宣传报道统计数字，要抽查两个月报纸样报和电视截屏（不包含核心价值观的公益广告），也将在问卷调查中了解宣传普及的效果；

二是实地暗访查社会场景营造，随机察看大街小巷、建筑围挡、车站、公园、广场等地；

三是抽查学校，既要提供大中小学校宣传展示核心价值观的图片，也要实地抽查学校传播核心价值观的氛围。<ref>{{cite web|author=涂更新|title=关于第四届全国文明城市评选测评和文明单位推荐工作的说明|url=http://www.wenming.cn/wmcj_pd/cjdt/201411/t20141128_2318342.shtml|publisher=[[中央文明办|中央文明办]]|work=第四届全国文明城市、文明村镇、文明单位和未成年人思想道德建设工作先进城市测评评选工作视讯会|accessdate=2015-01-01}}</ref>}}

为了评选全国文明城市，[[武汉市|武汉市]]全力宣传、全城背诵、全城考核社会主义核心价值观内容<ref>{{cite news|author=黄俊如|title=为评“文明城市”武汉拼了，全城背诵24字价值观|url=http://www.thepaper.cn/newsDetail_forward_1289056|accessdate=2015-01-01|newspaper=澎湃新闻|date=2014-12-27}}</ref><ref>{{cite news|title=武汉要求市民背诵社会主义核心价值观，今天你背了吗|url=http://cq.takungpao.com/html_content/2014-12-29/74754.shtml|accessdate=2015-01-01|newspaper=[[大公网|大公网]]|date=2014-12-29|deadurl=yes|archiveurl=https://web.archive.org/web/20150101014921/http://cq.takungpao.com/html_content/2014-12-29/74754.shtml|archivedate=2015-01-01}}</ref>。
