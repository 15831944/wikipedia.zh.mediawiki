'''社会主义国家'''，就一般定义而言，就是实行[[社会主义|社会主义]]制度的国家，或是采用社会主义意识形态的[[国家|国家]]。[[政體|政體]]均為[[共和制|共和制]]。其中那些长期实行共产党[[一党制|一党执政]]，并信仰[[马克思列宁主义|马克思列宁主义]]及其衍生理论的国家又被称为“[[共产主义国家|共产主义国家]]”。在这些国家范围内和其他认可这一制度的国家内被称为社会主义国家，实行社会主义制度。共产主义国家可以存在一些合法的政党，但国家的[[宪法|宪法]]只会把执政权赋予[[共产党|共产党]]。因而，在这种情况下，国家与共产党的机构会密切地联系在一起。目前，世界上共有14个社会主义国家（参见[[社会主义国家列表|社会主义国家列表]]）；其中4个以共产党为法定执政党，即[[中华人民共和国|中华人民共和国]]、[[越南社会主义共和国|越南社会主义共和国]]、[[老挝人民民主共和国|老挝人民民主共和国]]和[[古巴共和国|古巴共和国]]。这些国家有的实行着高度集权的计划经济体制，有的则实行[[社会主义市场经济|社会主义市场经济]]体制。

虽然朝鲜民主主义人民共和国仍宣称该国为社会主义共和国，鉴于该国已经删除了[[共产主义|共产主义]]和[[马克思列宁主义|马克思列宁主义]]作为党和国家的指导思想和意识形态，因此在定義上不被列入狭义的社会主义国家中。而中国官方的部分人士已承认[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]不是社会主义国家，因为[[朝鲜劳动党|朝鲜劳动党]]早在1980年已放弃马列主义而将[[主体思想|主体思想]]确立为唯一指导思想。<ref>{{Cite web |url=http://www.nfcmag.com/article/1790.html |title=“被修正”的朝鲜劳动党 |access-date=2015-04-24 |archive-url=https://web.archive.org/web/20150509110858/http://www.nfcmag.com/article/1790.html |archive-date=2015-05-09 |dead-url=no }}</ref><ref>{{Cite web |url=http://military.cnr.cn/sdpl/201412/t20141201_516933807.html |title=王洪光中将:中国不存在“放弃朝鲜”的问题 |access-date=2018-08-24 |archive-url=https://web.archive.org/web/20180824070010/http://military.cnr.cn/sdpl/201412/t20141201_516933807.html |archive-date=2018-08-24 |dead-url=no }}</ref><ref>{{Cite web |url=http://net.blogchina.com/blog/article/3032842 |title=为何说朝鲜是对社会主义的侮辱? |access-date=2018-08-24 |archive-url=https://web.archive.org/web/20180824101745/http://net.blogchina.com/blog/article/3032842 |archive-date=2018-08-24 |dead-url=no }}</ref>

==对于其界定的争议==
===马克思主义者的观点===
由于对[[马克思主义|马克思主义]]的理解和政治目的不同，不同类型的马克思主义者对“何为社会主义国家”有完全不同的看法。对于所有反对[[斯大林主义|斯大林主义]]的马克思主义者而言，例如[[民主社会主义|民主社会主义]]者认为“没有民主就没有社会主义”<ref>威廉·李卜克内西:《不要任何妥协》</ref><ref name="Frankfurt">《法兰克福宣言》</ref>，[[托洛茨基主义|托洛茨基主义]]者认为“一国不能建成社会主义”并反对苏联的官僚体制<ref name="trotsky">托洛茨基《被背叛的革命》</ref>，部分[[西方马克思主义|西方马克思主义]]者例如[[赫伯特·马尔库塞|赫伯特·马尔库塞]]等人则认为苏联等国是把马克思主义扭曲成为专制统治辩护的工具，苏联是一个不民主的、官僚体制的国家而并非真正的社会主义。<ref>马尔库塞《苏联的马克思主义》</ref>在[[马克思主义|马克思主义]]语境下，社会主义取代[[资本主义|资本主义]]，正如同资本主义社会取代[[封建社会|封建社会]]一样；真正的社会主义只能由高度发达的资本主义社会才能孕育，它必须是比资本主义社会更民主更自由的社会形态，而[[苏联|苏联]]、[[中国|中国]]、[[朝鲜|朝鲜]]等国却反而是比当代资本主义更为落后的社会形态。很多[[反斯大林主义左翼|反斯大林主义左翼]]都拒绝承认所谓的“社会主义阵营国家”是真正的社会主义，而认为它们是官僚工人国家或者“[[国家资本主义|国家资本主义]]”甚至封建社会，因此人类历史上至今并未产生过真正的社会主义国家，而只凭一国也不可能完全建成[[科学社会主义|科学社会主义]]。“共产主义国家”这个提法也是自相矛盾的，因为在[[马克思主义|马克思主义]]的历史观上“[[共产主义社会|共产主义社会]]”裏是不存在[[国家机器|国家机器]]的。<ref>马克思《哥达纲领批判》</ref>

[[马克思|马克思]]和[[恩格斯|恩格斯]]本人并没有过[[一党专政|一党专政]]的观点，并且支持[[普选|普选]]和[[结社自由|结社自由]]。<ref>马克思《宪章派》</ref><ref>马克思马克思《路易波拿巴的雾月十八日》</ref><ref>恩格斯《关于工人阶级的政治行动》</ref>[[巴黎公社|巴黎公社]]是一个全民直接选举、[[多党制|多党制]]的政权。<ref>秦晖：《专政、民主与所谓“恩格斯转变”》 </ref>在今天的大部分国家，国内也都存在着多个社会主义性质的左翼政党。而根据[[列宁|列宁]]个人的思想，他实际上也更希望建立一个实行[[多党制|多党制]]的社会主义国家<ref>{{Cite web |url=http://m.kdnet.net/content-1-2370584.html |title=十月革命后俄国政局的变化 |access-date=2014-12-07 |archive-url=https://web.archive.org/web/20141211000150/http://m.kdnet.net/content-1-2370584.html |archive-date=2014-12-11 |dead-url=no }}</ref>；但列宁时期的[[苏联|苏联]]在内战中就已日益官僚化，[[斯大林|斯大林]]则建立起[[一党专政|一党专政]]体制，并且成为后来所有社会主义国家效仿的对象。[[罗莎·卢森堡|罗莎·卢森堡]]、[[卡尔·科尔施|卡尔·科尔施]]和[[列夫·托洛茨基|列夫·托洛茨基]]都曾批判过[[苏联|苏联]][[布尔什维克|布尔什维克]]的[[一党专政|一党专政]]模式。<ref>罗莎·卢森堡《论俄国革命》</ref><ref>卡尔·科尔施《关于“马克思主义和哲学”问题的现状——一个反批判》</ref><ref name="trotsky"/>[[社会主义|社会主义]][[民主|民主]]在理论上具有全民性，普选则包括了选举权和被选举权的普遍拥有。而在[[一党专政|一党专政]]（党政合一）的体制下，[[党和国家最高领导人|国家最高领导人]]和地方最高领导人只能是当地的共产党[[党委书记|党委书记]]，[[无党派|无党派]]人士和[[卫星党|卫星党]]人士无法参选，这就相当于被剥夺了被选举权，与人人平等的[[社会主义|社会主义]][[民主|民主]]是有一些矛盾的。而人民不能自由组党，也违背了结社自由。当然，[[无党派|无党派]]人士和[[卫星党|卫星党]]人士中也有人担任少数国家干部（如[[中华人民共和国卫生部|中华人民共和国卫生部]]前部长[[陈竺|陈竺]]），但基本上全部权力还是掌握在共产党手中。因此，中国、越南、老挝、古巴等[[社会主义阵营|社会主义阵营]]国家的政治制度不符合[[社会主义|社会主义]]的要求；而且，前两者的国内存在大量的私营经济，有许多本国的和外国的资本家在剥削工人，经济模式实质上也是[[资本主义|资本主义]]。

=== 西方国家的观点 ===
在西方国家，对于是否存在“社会主义国家”，或者说是否可以用“[[社会主义|社会主义]]”来形容国家，仍存在争议。在很多人看来，[[社会主义|社会主义]]只能用来形容经济或社会，但不能形容[[国家政权|国家政权]]，因此不存在所谓的“社会主义国家”。对于社会主义阵营国家，西方普遍认为虽然他们自称是“社会主义国家”，但它们不是真正的[[社会主义|社会主义]]，甚至不具有社会主义性质，原因同样主要在于对[[社会主义|社会主义]]的理解存在差异。相对而言，西方国家所说的[[社会主义|社会主义]]多指[[社会民主主义|社会民主主义]]和[[民主社会主义|民主社会主义]]，例如[[加拿大|加拿大]]、[[北欧|北欧]]五国等之类的福利国家和[[法国社会党|法国社会党]]、[[英国工党|英国工党]]等之类的[[社会党国际|社会党国际]]政党他们所主张的社会主义，以及[[拉丁美洲|拉丁美洲]]的一些由社会主义政党执政的多党制国家。<ref>{{Cite web |url=http://www.doc88.com/p-111694950319.html |title=欧洲遍地都是社会主义国家 |access-date=2018-08-24 |archive-url=https://web.archive.org/web/20180824070250/http://www.doc88.com/p-111694950319.html |archive-date=2018-08-24 |dead-url=no }}</ref><ref name="Lamei">{{Cite web |url=http://www.mofcom.gov.cn/aarticle/i/dxfw/nbgz/200711/20071105255747.html |title=拉美如何将社会主义理论变成拉美模式社会主义实践 |access-date=2018-08-24 |archive-url=https://web.archive.org/web/20080204085636/http://www.mofcom.gov.cn/aarticle/i/dxfw/nbgz/200711/20071105255747.html |archive-date=2008-02-04 |dead-url=no }}</ref><ref name="Frankfurt"/>

西方国家一般称[[苏联|苏联]]、[[中华人民共和国|中华人民共和国]]、[[越南|越南]]、[[古巴|古巴]]等由共产党执政的社会主义国家为“Communist state”，即“共產主義國家”{{NoteTag|在汉语语境里，对于“共产主义国家”一词的起源存在争议。一种说法认为“共产主义国家”是对外文的[[直译|直译]]，这个词来自于[[东方集团|东方集团]]，“Communist state”照字面完全直译就是“共产主义者的国家”，但实际名词翻译不可能如此繁琐，故这一名词在汉语中一般有这两种译法，并且使用都很普遍。而另一种说法則認為，“共产主义国家”这一汉字词汇起源于对共产主义的错误理解，马克思主义传入早期，不仅翻译的著作少，而且读过原著的人也不多，很多人对马克思主义及其主张的共产主义有错误的理解和宣传，只是一味的理解成“建立一个共产主义国家”。现今一般都认为共产主义国家这一名词从构词法及语义的角度而言确实存在一定问题。}}。也有些西方学者认为社会主义阵营国家实行的经济制度属于社会主义，故称之为[[苏式社会主义|苏式社会主义]]。<ref>[https://wenku.baidu.com/view/6631f835876fb84ae45c3b3567ec102de2bddf37.html 之9——苏式社会主义]</ref><ref>{{Cite web |url=http://www.docin.com/p-1591373178.html |title=Socialism with Chinese characteristics on the Soviet model of socialism with Network Analysis |access-date=2018-08-24 |archive-url=https://web.archive.org/web/20180824101853/http://www.docin.com/p-1591373178.html |archive-date=2018-08-24 |dead-url=no }}</ref>

另外，西方学者也对[[马克思主义|马克思主义]]和[[社会主义|社会主义]]阵营国家之间的具体关系进行了区分。虽然绝大多数社会主义阵营国家都声称以马克思主义为最高信仰，但有不少西方学者认为他们信仰的根本不是[[马克思主义|马克思主义]]，或者说不是真正的[[马克思主义|马克思主义]]，而是[[列宁主义|列宁主义]]，甚至是[[斯大林主义|斯大林主义]]，这也是[[苏式社会主义|苏式社会主义]]这一名称的由来。实际上，这也不是毫无根据的。以[[中华人民共和国|中华人民共和国]]为例，翻译的马克思的著作只有60多卷，且全由俄文版翻译，而公认最全的法文版有100多卷，因此各国对马克思主义的理解可能存在很大的偏差。

=== 社会主义阵营的观点 ===
社会主义阵营国家认为自己就是社会主义国家，且只承认那些和自己制度相类似的国家才是社会主义国家{{NoteTag|当时还有很多国家自称是和[[社会主义阵营|社会主义阵营]]国家一样的社会主义国家，但是不仅西方国家不承认他们是社会主义国家，甚至连社会主义阵营国家也不承认他们是社会主义国家。}}。社会主义国家的标志是实行[[生产资料|生产资料]][[公有制|公有制]]（但不一定只有生产资料公有制），而且理论上是由一个代表最广大人民根本利益真正意义上的[[共产党|共产党]]执政，并往往采用[[一党制|一党制]]（一些共产主义者认为此并非西方所称的“一党制”），因此西方又称之为“共產主義國家”（Communist state）。目前世界上还没有完全意义上的[[共产主义国家|共产主义国家]]，其中[[共产主义|共产主义]]是其最高目标，而[[社会主义|社会主义]]是其过渡阶段。因此除了[[民主柬埔寨|民主柬埔寨]]自称是共产主义国家外其余的都自称处在[[共产主义|共产主义]]的第一階段：[[社会主义|社会主义]]，他们通常自称是社会主义国家而非共产主义国家。不过在社会主义理论中，对这一时期国家社会形态定义为“社会主义社会”，故而认为社会主义是共产主义的前一阶段。其中中國还提出了[[社會主義初級階段|社會主義初級階段]]理論，[[中华人民共和国|中华人民共和国]]称自己目前处于“[[社会主义初级阶段|社会主义初级阶段]]”，这是一个独有的提法。另外，国名中出现“社会主义”字样的国家不一定是社会主义国家，没有出现“社会主义”字样的国家不一定不是社会主义国家。

共产党掌握政权主要像[[苏联|苏联]]、[[中国|中国]]等国那样通过暴力革命夺取政权，也有一些欧洲或者亚洲的社会主义国家在苏联或者中国的革命的影响下或者协助夺取政权，走上社会主义道路。社会主义国家建立后，大多都加入了[[社会主义阵营|社会主义阵营]]，也有一些国家因与[[苏联|苏联]]发生争执而退出了[[社会主义阵营|社会主义阵营]]，如[[狄托|狄托]]領導的[[南斯拉夫社会主义联邦共和国|南斯拉夫社会主义联邦共和国]]、[[毛澤東|毛澤東]]領導的[[中华人民共和国|中华人民共和国]]、[[霍查|霍查]]領導的[[阿尔巴尼亚社会主义人民共和国|阿尔巴尼亚社会主义人民共和国]]等。

1989年到1991年间，苏联和东欧发生了剧烈的政治动荡，[[共产党|共产党]]纷纷失去政权，[[苏联解体|苏联解体]]和[[东欧剧变|东欧剧变]]使得社会主义国家的数量急剧减少，令其余的社会主义国家产生了空前的危机，也令其余的社会主义国家开始了意识形态的分裂。以朝鲜为代表社会主义国家继续实行过去的方针政策，而以中国为代表社会主义国家开始进行改革。对于进行改革的社会主义国家，在很多西方人眼中已经不再属于社会主义国家。而像[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]这样的原强硬[[斯大林主义|斯大林主义]]国家也已经“[[修正主义_(马克思主义)|变质]]”（[[家族独裁|家族独裁]]）了。但鉴于对“社会主义国家”概念本身就很难给出一个绝对的定义，因此对于进行改革的社会主义国家是否还是“社会主义国家”现在也很难给出明确的结论。

===其他国家的观点===
二战后的民族独立浪潮中，除部分国家走上[[苏式社会主义|苏式社会主义]]的道路外，还有相当数量新独立的国家也曾宣布自己是某种特殊类型的“社会主义国家”，总共约有50多个。这些国家的“社会主义”往往具有[[民族主义|民族主义]]或[[宗教社会主义|宗教社会主义]]的色彩，如：“[[国大党|国大党]]社会主义”、“纲领社会主义”、“[[缅甸式社会主义|缅甸式社会主义]]”、“复兴社会主义”、“[[基督教社会主义|基督教社会主义]]”、“[[佛教社會主義|佛教社會主義]]”、“[[伊斯兰社会主义|伊斯兰社会主义]]”、“[[阿拉伯社会主义|阿拉伯社会主义]]”、“乌贾马社会主义”、“非洲社会主义”、“桑地诺社会主义”。

其中，例如[[圭亚那|圭亚那]]、[[印度|印度]]、[[葡萄牙|葡萄牙]]、[[坦桑尼亚|坦桑尼亚]]、[[斯里兰卡|斯里兰卡]]等民主国家的宪法中规定有社会主义，而[[缅甸|缅甸]]、[[利比亚|利比亚]]、[[伊拉克|伊拉克]]、[[津巴布韦|津巴布韦]]、[[叙利亚|叙利亚]]等非民主国家的宪法中也曾经规定或者目前仍规定有社会主义。总之，亚非拉地区尤其是拉丁美洲的部分国家政府经常喊着社会主义口号推行自己的模式，它们认为自己是“有本国特色的社会主义”。<ref name="Lamei"/>
