'''鲍伊'''（Bowie）是美国[[马里兰州|马里兰州]][[乔治王子县|乔治王子县]]的一个城市<ref>{{gnis|597104}}</ref>，位于{{Coord|38|57|53|N|76|44|40|W|type:city}}<ref name="GR1">{{cite web|url=http://www.census.gov/geo/www/gazetteer/gazette.html|publisher=[[United_States_Census_Bureau|United States Census Bureau]]|accessdate=2011-04-23|date=2011-02-12|title=US Gazetteer files: 2010, 2000, and 1990}}</ref>
，面积{{convert|18.51|sqmi|sqkm|2}}<ref name="Gazetteer files">{{cite web |title=US Gazetteer files 2010 |url=http://www.census.gov/geo/www/gazetteer/files/Gaz_places_national.txt |publisher=[[United_States_Census_Bureau|United States Census Bureau]] |accessdate=2013-01-25 |deadurl=yes |archiveurl=https://www.webcitation.org/64vfLAeJ2?url=http://www.census.gov/geo/www/gazetteer/files/Gaz_places_national.txt |archivedate=2012-01-24 |df= }}</ref>人口为54,727人（2010年）。

鲍伊已从一个铁路小站成长为乔治王子县最大的市镇，以及马里兰州第五大城市<ref>{{cite web |url=http://www.census.gov/popest/cities/tables/SUB-EST2006-04-24.csv |title=Annual Estimates of the Population for Incorporated Places in Maryland, Listed Alphabetically |publisher=United States Census Bureau |date=July 1, 2006 |access-date=August 29, 2015 |deadurl=yes |archiveurl=https://web.archive.org/web/20071105221658/http://www.census.gov/popest/cities/tables/SUB-EST2006-04-24.csv |archivedate=十一月 5, 2007 }} </ref>。2014年，CNN Money将其评为美国最佳居住地点第28名<ref>{{cite web |url=http://time.com/money/3312338/bowie-md-best-places-to-live |title=Best Places to Live 2014 |date=September 19, 2014 |access-date=September 22, 2014 |work=Money}}</ref>。
