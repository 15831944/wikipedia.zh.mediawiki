'''鲍晓毅''' 是一位光学物理学家，目前担任[[渥太华大学|渥太华大学]]物理教授。

==生平==
毕业于[[南开大学|南开大学]]光学学士和硕士，1987年毕业于[[中国科学院|中国科学院]]安徽光学精密机械研究所光学专业博士学位。之后先后在[[堪萨斯州立大学|堪萨斯州立大学]]、[[雷根斯堡大学|雷根斯堡大学]]、[[斯特拉斯克莱德大学|斯特拉斯克莱德大学]]和[[肯特大学|肯特大学]]做研究。1994年加入[[新不伦瑞克大学|新不伦瑞克大学]]担任助理教授，后升为教授。2000年加入[[渥太华大学|渥太华大学]]担任物理教授。已发表230余篇期刊论文和200余篇会议论文。<ref>[http://66199.net/html/xwpd/crgk/whlgdx/8413.html 聚焦人才强校战略科学家——姜晶教授、鲍晓毅教授]，东陆教育集团。</ref>

==荣誉==
她是[[光学学会|光学学会]]会士，[[加拿大皇家学会|加拿大皇家学会]]会士和[[国际光学工程师学会|国际光学工程师学会]]会士。<ref>{{Cite web |url=https://science.uottawa.ca/physics/people/bao-xiaoyi |title=Xiaoyi Bao |publisher=uottawa.ca |accessdate=December 18, 2016}}</ref><ref>{{Cite web |url=https://research.uottawa.ca/people/bao-xiaoyi |title=Xiaoyi Bao |publisher=uottawa.ca |accessdate=December 18, 2016}}</ref><ref>{{Cite web |url=http://www.chairs-chaires.gc.ca/chairholders-titulaires/profile-eng.aspx?profileId=576 |title=Xiaoyi Bao |publisher=gc.ca |accessdate=December 18, 2016}}</ref>
