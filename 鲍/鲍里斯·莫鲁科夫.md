'''鲍里斯·弗拉基米罗维奇·莫鲁科夫'''（{{lang-ru|Бори́с Влади́мирович Моруков}}，{{bd|1950年|10月1日|2015年|1月1日}}）是[[俄罗斯|俄罗斯]][[宇航员|宇航员]]和太空医生。
==早年==
1950年10月1日出生于莫斯科。1973年毕业于第二莫斯科国立医学研究所（现俄罗斯国立医学研究大学）。

1973年至1975年在[[俄羅斯科學院|俄羅斯科學院]][[俄罗斯科学院生物医学问题研究所|生物医学问题研究所]]担任实验室助理。1984年4月成为高级研究员。<ref name=bio>{{cite web |title=Моруков Борис. Биография |url=https://ria.ru/20150102/1041226839.html |publisher=[[俄罗斯新闻社|РИА Новости]] |accessdate=2019-04-12 |language=ru |date=2015-01-02}}</ref>

==宇航员生涯==
1989年成为宇航员候选人。在1990年10月至1992年2月间，在[[加加林宇航员培训中心|加加林宇航员培训中心]]接受训练，通过考试后获得测试宇航员资格。2000年9月8日至20日，他和其他6名宇航员乘坐[[亚特兰蒂斯号航天飞机|亚特兰蒂斯号航天飞机]]前往[[国际空间站|国际空间站]]，在太空累计停留了11天19小时10分。<ref name=bio/>
==退役后==
2006年12月，时任俄罗斯医学生物问题研究所副所长的鲍里斯·莫鲁科夫被任命为火星飞行地面模拟试验志愿者选拔委员会主席<ref>{{cite web |title=俄航天署署长在网络新闻发布会上透露，目前已经有21个国家的公民申请参加俄罗斯的火星飞行地面模拟试验 |url=http://sputniknews.cn/russia/2006120641597506/ |publisher=[[俄罗斯卫星通讯社|俄罗斯卫星通讯社]] |accessdate=2019-04-12 |date=2006-12-06}}</ref>。2010年成为[[火星-500|火星-500]]模拟项目总负责人<ref>{{cite web |title=257天模拟太空之旅结束 宇航员开始"登陆火星" |url=http://it.sohu.com/20110216/n279360852.shtml |publisher=[[搜狐|搜狐]]科技 |archiveurl=https://web.archive.org/web/20110221221117/http://it.sohu.com/20110216/n279360852.shtml |archivedate=2011-02-21 |date=2011-02-16}}</ref>。2015年1月1日晚在莫斯科逝世<ref>{{cite web |title=Моруков Борис Владимирович 1.10.1950. – 1.01.2015. |url=http://www.imbp.ru/WebPages/win1251/News/2015/news01-01-15.html |website=www.imbp.ru |accessdate=2019-04-12 |language=ru}}</ref>。
