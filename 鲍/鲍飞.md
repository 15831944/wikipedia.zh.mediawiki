'''鲍飞'''（{{bd|1992年|4月20日}}，绰号'''文森特'''），[[中国|中国]][[安徽|安徽]][[芜湖市|芜湖]]人。[[英雄联盟|英雄联盟]]玩家，主要进行游戏直播，因为使用英雄联盟中的英雄“德莱文”技术高超而闻名。

==名称由来==
鲍飞的英雄联盟中角色ID是“Vincent”，这个名称来源于《[[最终幻想系列|最终幻想]]》中的主角[[萨菲罗斯|萨菲罗斯]]的母亲露克蕾西亚的恋人{{link-en|文森特·瓦伦汀|Vincent Valentine}}。<ref>{{Cite web |url = http://lol.tuwan.com/108771/ |title = lol文森特是谁？怎么火的？ |author = 兔小睡 |date = 2015年5月28日 |publisher = 兔玩LOL |language = zh-cn }}</ref>

==人生经历==
鲍飞于1992年4月20日出生在安徽芜湖市，是中国大陆地区最早的一批[[英雄联盟|英雄联盟]]玩家。2013年被[[解说小漠|解说小漠]]看中，并做了一期“国服第一德莱文”视频，从而使得他迅速火了起来，<ref name="文森特介绍">{{Cite web|url=http://www.52pk.com/gl/6444706.shtml|title=《英雄联盟LOL》国服第一德莱文文森特介绍|author=FT|date=2015年6月12日|publisher=52PK|language=zh-cn}}</ref>后加入YY战队U4担任ADC位置，并在YSL2014中取得冠军。<ref>{{Cite web|url=http://games.enet.com.cn/a/2014/0428/A2014042800580309_0.shtml|title=YSL2014第一赛季赛况回顾 U4强势夺冠|author=eNet游戏|date=2014年4月28日|publisher=eNet游戏|language=zh-cn}}</ref>但到后来因利益分配出现分歧等原因与卡尔一起退出了U4。<ref>{{Cite web |url = http://lol.mingrent.com/bg/6373.html |title = U4战队退队门事件 U4战队出现利益问题 |author = 草帽王 |date = 2014年8月6日 |publisher = 游戏名人堂 |language = zh-cn |deadurl = yes |archiveurl = https://web.archive.org/web/20160809101820/http://lol.mingrent.com/bg/6373.html |archivedate = 2016年8月9日 |df =  }}</ref>

2014年，解说小漠又做了一期“国服第一德莱文”视频，使文森特成为了“德莱文”这个英雄的符号。<ref name="文森特介绍"/>2015年1月1日，鲍飞与斗鱼签约成为[[斗鱼TV|斗鱼TV]]主播。

2015年5月24日，鲍飞发出单方面与斗鱼直播解约声明，并发文声称斗鱼存在合同欺骗、拖欠工资。其后斗鱼平台声称已向鲍飞支付192万元人民币，并称是鲍飞突然中断联系，拒绝与斗鱼沟通，有开价更高的直播平台拉拢他。<ref>[http://www.youxiniao.com/news/chanye/2500884.html LOL文森特事件 LOL文森特与斗鱼闹翻的经过]</ref>2015年5月27日，鲍飞来到杭州，与[[战旗tv|战旗tv]]接触。其后战旗TV官方平台在微博称鲍飞加盟战旗TV<ref>[http://gameonline.yesky.com/72/66475572.shtml LOL主播文森特现身杭州 爆签约战旗TV]</ref>

==争议==
2015年2月，鲍飞与另一玩家节奏使用粉丝的账号到英雄联盟台湾服务器进行双人排位，多次出现11分钟到对面泉水区推基地，并获得15连胜。后被台服玩家举报，而英雄联盟台服官方以“非本人账号/代练”、“影响两岸关系”为由将鲍飞粉丝账号永久停封。<ref>[http://news.gao7.com/article/440891-20150212-1 LOL大神文森特携节奏血虐台服 因太屌被永久封号] {{webarchive|url=https://web.archive.org/web/20150530140853/http://news.gao7.com/article/440891-20150212-1 |date=2015-05-30 }}</ref><ref>[http://wangyou.pcgames.com.cn/485/4851520.html 台服玩家报警了！文森特血虐台服被封号]</ref>

2015年9月30日，有[[百度贴吧|百度贴吧]]网友公布有关斗鱼起诉主播后法院判定的裁定书，裁定书显示鲍飞被冻结银行存款1500万。 <ref>[http://gameonline.yesky.com/142/97810642.shtml 斗鱼起诉主播后续:播主5100万存款被冻结]</ref>当晚鲍飞发微博称合同是属于“卖身”合同，[[斗鱼TV|斗鱼]]是家无良公司。<ref>[http://www.taihainet.com/news/finance/hot/2015-10-13/1551492.html LOL文森特微博回应跳槽违约斗鱼事件 媒体评文森特“道德绑架”博同情？]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
