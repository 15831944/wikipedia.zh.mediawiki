'''泰緬孤軍的戰役'''是1950年國軍在[[雲南省_(中華民國)|雲南省]]的餘部，輾轉來到泰緬邊境，為求生存，並實現反共理想，先後與緬甸政府、中共，及泰共發生的戰役。<ref name="泰北孤軍簡史">{{Cite web |url = https://www.hsitai.world/northernthailand/泰北孤軍簡史/ |title = 泰北孤軍簡史 |accessdate = 2018/08/18  }}</ref>

==與緬甸政府的戰役==
===孟果大其力戰役===
{{main|孟果大其力戰役}}
1950年孤軍進入緬甸，6月緬軍以驅逐入侵者為由圍攻，反被孤軍擊退。

[[柏楊|柏楊]]的小說《[[異域|異域]]》，稱之為「第一次中緬大戰」。

===血戰薩爾溫江===
{{main|薩爾溫江戰役 (1953年)}}
1953年3月，緬軍得到中共的支持，進攻位於[[薩爾溫江|薩爾溫江]]流域的孤軍基地[[猛撒|猛撒]]，再度被孤軍擊退。

柏楊的小說《異域》，稱之為「第二次中緬大戰」。

緬甸戰敗後惱羞成怒，向聯合國控訴中華民國侵略，中華民國政府為平息爭端，將孤軍總指揮[[李彌|李彌]]將軍及部分孤軍撤往臺灣。政府指派[[柳元麟|柳元麟]]將軍解任總指揮，將孤軍改組為五個軍。不久，孤軍基地撤往[[江拉|江拉]]。

柏楊的小說《異域》，稱之為「勝利帶給我們撤退」。

===江拉基地保衛戰===
{{main|江拉基地保衛戰}}
1960年6月，緬軍與中共聯手，進攻位於江拉的孤軍基地，雙方激戰數月後，孤軍不敵，於1961年2月撤往[[泰國|泰]]、[[緬甸|緬]]、[[寮國|寮]]邊境。

緬甸再度向聯合國控訴中華民國侵略，中華民國政府將柳元麟，及部分一、二、四軍撤往臺灣。而三、五軍分別由[[李文煥|李文煥]]、[[段希文|段希文]]率領，撤往泰國。

==反攻大陸==
1951年4月，孤軍大舉進軍雲南，6月成功收復[[滄源|滄源]]等6個縣，8月共軍以優勢兵力反擊，孤軍不敵，撤回緬甸。<ref name="泰北孤軍—他們的故事"/>

柏楊的小說《異域》，稱之為「反攻雲南」。

==與泰國共產黨的戰役==
===帕蒙山地區之役===
1970至1975年間，孤軍應泰國政府請求，對泰北地區[[苗共|苗共]]（泰共的[[苗族|苗族]]成員）武裝發動6次圍剿，屢戰屢勝，使泰北局勢轉危而安。<ref name="泰北孤軍—他們的故事"/>

===考科考牙戰役===
1981年孤軍應泰國政府請求，圍剿考科考牙山區的苗共游擊隊，經過22天的戰役（泰國政府規定45天內完成任務），孤軍收復考科考牙山區，凱旋歸來。

此役孤軍一舉攻下泰共的大本營，重挫其勢力，泰共在1990年代初消亡，孤軍功不可沒。<ref>{{Cite web |url = https://www.hsitai.world/northernthailand/220考科考牙紀念日「泰北華人光榮的一天」/ |title = 2/20考科考牙紀念日「泰北華人光榮的一天」 |accessdate = 2018/08/19  }}</ref>
