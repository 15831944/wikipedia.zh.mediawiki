'''泰安車站'''位於[[台灣|台灣]][[臺中市|臺中市]][[后里區|后里區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺中線|臺中線]]的[[鐵路車站|鐵路車站]]，是台鐵第一個[[高架車站|高架車站]]。本站已於2017年9月完成月台無階化工程。

== 車站構造 ==
*高架車站，售票處位於一樓。
*[[電梯|電梯]]一座，供身障者使用。
*樓梯一座，由一樓[[剪票口|剪票口]]通往五樓月台。
*[[島式月台|島式月台]]一座，位於五層樓高，由於月台風大，設有[[候車室|候車室]]及欄杆。
*因三義至泰安間近11公里，興建本站時有預留[[側線|側線]]的空間。但因山線后里以北的區間車班次較少，列車可在前後兩站待避，所以側線並沒有鋪軌、也沒有裝設[[橫渡線|橫渡線]]，故本站無待避或交會列車的功能。

== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#簡易站|簡易站]]，由[[后里車站|后里車站]]派站員管理，僅有[[臺鐵區間車|區間車]]停靠。
**本站也停靠成追線區間車。
**本站接近三泰隧道。
**區間快車不停靠。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
**本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1998年9月23日：舊山線三義至后里路段最後一天行駛列車，深夜路線停用後從舊的泰安站遷至本站。
*1998年9月24日：隨新山線通車而啟用。
*2015年6月30日：啟用多卡通刷卡機。
