'''泰安市'''是[[中华人民共和国|中华人民共和国]][[山东省|山东省]]下辖的[[地级市|地级市]]，位于山东省中部，[[黄河|黄河]]之东。东南两面临济南市，东南与[[临沂市|临沂市]]相接，南与[[济宁市|济宁市]]毗邻，西与[[聊城市|聊城市]]及[[河南省|河南省]][[濮阳市|濮阳市]]隔黄河相望。全市总面积7,761平方公里，人口560.08万。泰安之名取“国泰民安”之意，因五岳之首[[泰山|泰山]]而驰名。主城区位于泰山南麓，山城一体，是[[国家历史文化名城|国家历史文化名城]]、[[国家卫生城市|国家卫生城市]]以及[[中国优秀旅游城市|中国优秀旅游城市]]。

== 历史 ==
泰安，因[[五岳|五岳]]之[[东岳|东岳]]──[[泰山|泰山]]而得名，古语云：“泰山安，则四海皆安”，寓“国泰民安”之意。[[秦代|秦代]]今境先属[[齐郡|齐郡]]，后属[[济北郡_(秦朝)|济北郡]]，并置[[博阳县|博阳县]]，为济北郡治。[[西汉|西汉]]初设[[泰山郡|泰山郡]]，改博阳县为[[博县|博县]]，为泰山郡治。[[元封_(西汉)|元封]]元年（前110年）置[[奉高县|奉高县]]，治今泰安市东，为泰山郡治。[[魏晋|魏晋]]因之。[[北魏|北魏]]年间泰山郡移治博县，改名[[博平县|博平县]]，治今市境东南。[[北齐|北齐]]改泰山郡为[[东平郡|东平郡]]，博平县复为博县，为郡治。

[[隋朝|隋]][[开皇|开皇]]初废东平郡，开皇十六年（596年）改博平县为[[汶阳县|汶阳县]]，后改[[博城县|博城县]]。[[唐朝|唐]][[武德|武德]]五年（622年），于博城县置[[东泰州|东泰州]]。[[贞观|贞观]]元年（627年）废东泰州，省[[梁父县|梁父县]]入博城县。[[乾封|乾封]]元年（666年）改博城县为[[乾封县|乾封县]]。[[总章|总章]]元年（668年），复改博城县。[[神龙|神龙]]元年（705年），复为乾封县，属[[兖州|兖州]]。[[北宋|北宋]][[开宝|开宝]]五年（972年），乾封县移治[[岱岳|岱岳]]。[[大中祥符|大中祥符]]元年（1008年）改乾封县为[[奉符县|奉符县]]，属[[袭庆府|袭庆府]]。

[[金朝|金]][[天会_(金朝)|天会]]十四年（1136年），于奉符县置[[泰安军|泰安军]]，“泰安”之名始此。清[[乾隆|乾隆]]《泰安府志》：“汉人称天下之安如泰山而维之，名盖取诸此。”《山东通志》曰：取“安益求安”之意。[[大定_(金朝)|大定]]二十二年（1182年）升泰安军为[[泰安州|泰安州]]，治奉符县。[[明朝|明]][[洪武|洪武]]初，州治奉符县省入泰安州，属[[济南府|济南府]]。[[清朝|清]][[雍正|雍正]]二年（1724年）升泰安为[[直隶州|直隶州]]，十三年升为[[泰安府|泰安府]]，并设[[泰安县|泰安县]]，为府治。

[[民国|民国]]二年（1913年）废泰安府，辖县分属岱北、岱南、济西三道，次年分別改[[济南道|济南道]]、[[济宁道|济宁道]]、[[东临道|东临道]]。1925年分属泰安、兖济等道。1928年废道制。1940年置[[泰西专区|泰西专区]]，属[[鲁西行政区|鲁西行政区]]；又置[[泰山专区|泰山专区]]。1941年析泰山专区南部置[[泰南专区|泰南专区]]，同属[[鲁中行政区|鲁中行政区]]；后泰西专区改属[[冀鲁豫行政区|冀鲁豫行政区]]。1942年2月，析泰安、宁阳两县置[[泰宁县|泰宁县]]，隶泰南专区。1945年将泰南专区并入[[沂蒙专区|沂蒙专区]]。1948年泰山、沂蒙、泰西三专区均属[[鲁中南行政区|鲁中南行政区]]。

[[中华人民共和国|中华人民共和国]]成立后，泰安、泰宁两县隶属泰山专区。1950年将泰山、泰西两专区合并为[[泰安专区|泰安专区]]。1952年9月，泰宁县改名[[徂阳县|徂阳县]]。1956年3月撤销。1958年6月，撤销泰安专区，析泰安县城区和郊区置泰山市；同年11月将泰安县和泰山市合并置泰安市，隶属[[济南市|济南市]]。1961年5月复置泰安专区。1963年5月，改泰安市为泰安县。1967年泰安专区改称[[泰安地区|泰安地区]]。1982年3月，泰安县改为县级泰安市，属泰安地区。1985年5月，撤销泰安地区，设立地级泰安市；原县级泰安市改设[[泰山区_(泰安市)|泰山区]]、[[郊区_(泰安市)|郊区]]；辖[[宁阳|宁阳]]、[[肥城|肥城]]、[[东平|东平]]三县，代管[[莱芜|莱芜]]、[[新泰|新泰]]二市；将[[汶上|汶上]]、[[泗水县|泗水]]两县划归[[济宁市|济宁市]]，[[平阴县|平阴县]]划归济南市。1992年11月，莱芜市升为地级市；同年肥城撤县改市。2000年4月，泰安市郊区更名为[[岱岳区|岱岳区]]。

== 地理 ==
地处东经116°02′至117°59′、北纬35°38′至36°28′之间，东西长约176.6千米，南北宽约93.5千米；位于[[大汶河|大汶河]]上游；城区位于泰山脚下，依山而建，山城一体。北距省会济南市66.8千米，南至三孔圣地曲阜市74.6千米，是山东省“一山、一水、一圣人”旅游热线的中点。泰安市最高点为[[玉皇顶_(泰山)|玉皇顶]]，海拔1532.7米。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为549.42万人<ref>{{cite book |author=泰安市统计局|title=《泰安市2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加了15.96万人。增长2.99%，年平均增长0.30%。其中，男性为274.20万人，占总人口的49.91%；女性为275.22万人，占总人口的50.09%，总人口性别比（以女性为100）为99.63。0－14岁的人口为 85.74万人，占15.60%，15－64岁的人口为411.06万人，占74.82%；65岁及以上的人口为52.62万人，占9.58%。

=== 民族 ===
以[[汉族|汉族]]为主。主要少数民族为[[回族|回族]]，泰安市回族人口据省内前列（少于[[德州市|德州市]]）。全市回族超过60000人，其中尤以泰山岱岳两区为多。市区内有[[清真寺|清真寺]]小区，并设立回民中学一所。

== 交通 ==
{{main|泰安交通}}
*铁路：[[泰山站_(泰安)|泰山站]]（始建于1909年，2000年前称泰安火车站）为[[京沪铁路|京沪铁路]]、[[辛泰铁路|辛泰铁路]]、[[泰肥铁路|泰肥铁路]]的交点，境内还有[[磁莱铁路|磁莱铁路]]，并计划建设[[泰聊铁路|泰聊铁路]]和[[济泰城际轨道|济泰城际轨道]]。业已通车的的[[京沪高铁|京沪高铁]]在境内开设高铁[[泰安站|泰安站]]。
*公路：[[104国道|104国道]]、[[103国道|103国道]]、[[莱泰高速公路|莱泰高速公路]]、[[京沪高速公路|京沪高速公路]]、[[京台高速公路|京台高速公路]]、[[济广高速公路|济广高速公路]]等公路交通网四通八达。在建的[[泰聊高速公路|泰聊高速公路]]，将构成未来[[青兰高速公路|青兰高速公路]]的一部分。

== 经济 ==
泰安是山东省重要的[[煤炭|煤炭]]、[[钢铁|钢铁]]基地，拥有[[汽车|汽车]]（[[中国重型汽车集团|中国重型汽车集团]]、青年莲花）、[[煤炭|煤炭]]、输变电设备、新材料、新能源、[[化学工业|化工]]、纺织服装、[[食品|食品]]等八大主导产业。农业基础稳固，是山东省重要的[[粮食|粮食]]产区之一，国家优质[[小麦|小麦]]商品粮生产基地、畜牧生产基地、蔬菜生产基地和花卉苗木生产基地。

== 物产 ==
*矿产资源丰富，是山东省重要的能源和原材料基地。已探明的矿产资源有58种，保有储量500亿吨。包括有[[石膏|石膏]]、天然[[硫|硫]]、[[煤|煤]]、[[石灰石|石灰石]]、[[石棉|石棉]]等，其中硫储量（31.9亿吨）居亚洲第一位。石膏储量在中国居首位。已开采的矿产有煤、[[铁|铁]]、[[铜|铜]]、石膏、[[耐火土|耐火土]]、[[钾长石|钾长石]]、[[石棉|石棉]]、[[钾盐|钾盐]]、[[岩盐|岩盐]]、石灰石、[[花岗石|花岗石]]、[[麦饭石|麦饭石]]、[[矿泉水|矿泉水]]等。
*特产有肥城[[桃|桃]]（佛桃）、泰山[[赤鳞鱼|赤鳞鱼]]、泰山[[赤灵芝|赤灵芝]]、泰山[[板栗|板栗]]、葛石[[大枣|大枣]]、河岔山[[鸭蛋|鸭蛋]]、莱芜[[黄姜|黄姜]]、汶河[[花生|花生]]、东平湖[[鲤鱼|鲤鱼]]、四大名药（[[何首乌|何首乌]]、[[紫草|紫草]]、[[黄精|黄精]]、[[四叶参|四叶参]]）。
