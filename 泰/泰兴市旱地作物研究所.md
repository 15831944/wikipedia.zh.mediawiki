'''泰兴市旱地作物研究所'''，原'''泰兴市农业科学研究所'''，是位于[[江苏省|江苏省]][[泰兴市|泰兴市]]的以小杂粮研究和技术服务为特色的县级农业科研单位<ref>{{Cite web |url=http://www.jaas.ac.cn/show-29-1530-1.html |title=常有宏副院长在泰州市旱地作物研究所调研|accessdate=2018-11-17 |author=jash |date=2014-08-05 |work= |publisher=江苏省农业科学院网站|language=Zh-hans }}</ref>。

2017年，研究所被列为第二批江苏现代农业科技综合示范基地。它与[[江苏省农业科学院|江苏省农业科学院]][[泰州农科所|泰州农科所]]、泰兴市农业技术推广中心为共建单位。对接的科研教学单位是[[扬州大学|扬州大学]]农学院。时任负责人、专家组长王全友。研究所拥有150亩土地，位于[[根思乡|根思乡]]宣泰路267号<ref>{{Cite web |url=http://www.jsagri.gov.cn/announce/files/662490.asp |title=关于公布第二批江苏现代农业科技综合示范基地名单的通知 苏农科〔2017〕3号|accessdate=2018-11-17 |author= |date=2017-03-16 |work= |publisher=江苏农业网|language=Zh-hans }}</ref>。所在区域以'''市农科所'''之名，列为泰兴市的一个[[类似乡级单位|类似乡级单位]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|32/12/83/401/000}}}}</ref>。
