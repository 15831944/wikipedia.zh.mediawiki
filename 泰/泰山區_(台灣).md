'''泰山區'''位於[[臺灣|臺灣]][[新北市|新北市]]西邊的[[區_(中華民國)|市轄區]]，前身為[[臺北縣|臺北縣]][[泰山鄉|泰山鄉]]，屬於[[臺北都會區|臺北都會區]]的[[衛星城市|衛星城市]]之一，緊鄰[[新莊區|新莊區]]、[[林口區|林口區]]、[[五股區|五股區]]。轄境32%為平地，其餘為山坡地與河川。

== 概述 ==
泰山舊名山腳，因座落於平頂山（今[[林口台地|林口台地]]）之山下，自古依地理環境而習稱之'''山腳'''，由於境內有大窠坑溪及貴子坑溪流經，具「勢高而近溪澗淡水」之「天泉水堀」地利，為早期移民化地為田的首選，而成為新莊平原早期開發地區，「北臺首學」明志書院位居本區，人文薈萃，文風鼎盛。復因境內三級古蹟「頂泰山巖」香火鼎盛、祖師靈驗、德澤遐爾、廟貌巍巍，登臨其境，台北盆地一覽無遺，因生「登泰山而小天下」之感，遂相傳成習改曰「泰山」。

新莊平原的拓墾，最早以泰山山腳一帶，而後向營盤、海山、五股、新莊中港厝、頭前、二重埔方面發展。因為新莊、頭前、二重埔所依傍約台北大湖是個鹹水湖，食用水和灌溉水取得不易。平頂山腳的泰山原野以當時的條件而言，取水最為便利。

== 歷史 ==
因座落於[[林口台地|林口台地]]（平頂山）之山下，泰山自古依地理環境而昔稱之'''山腳'''，昔多[[閩南|閩南]][[泉州府|泉州府]][[安溪縣|安溪縣]]移民開墾，[[臺灣清治時期|清治時期]]隸屬[[淡水廳|淡水廳]][[八里坌堡|八里坌堡]]，[[台灣日治時期|日治時期]]十二廳時代，新-{庄}-支廳下設有貴仔坑區，即為今泰山區之前身。區長役場則位於貴仔坑-{庄}-，今[[貴子里|貴子里]]。

[[大正|大正]]9年（1920年）地方制度改革，新-{庄}-區與貴仔坑區合併為[[新莊街|新莊街]]，屬[[新莊郡|新莊郡]]。[[二次大戰|二次大戰]]後，[[國民政府|國民政府]]改新莊街為新莊鎮，唯山腳居民皆感公務往返交通不便，遂有分治之議。

1950年3月1日，正式設[[鄉_(中華民國)|鄉]]，由新莊鎮劃出、成立泰山鄉。鄉名取自境內奉祀移民的鄉土神[[泰山顯應祖師|泰山顯應祖師]]的[[廟宇|廟宇]]，亦是本地信仰中心：「[[頂泰山巖|頂泰山巖]]」與「[[下泰山巖|下泰山巖]]」兩廟宇之「泰山」二字。

2010年12月25日，因[[新北市|臺北縣]]升格改制新北市，泰山鄉改為泰山區。泰山現因人口快速發展，復以捷運、道路通車、通往[[新莊區|新莊區]]、[[臺北市|臺北市]]、[[桃園市|桃園市]]之交通完備，日益繁榮。

== 行政區域 ==

*[[File:TW_CHW106.svg|市道]][[市道106號|市道106號]]（泰林路（橫窠雅路））：泰山主要的東西向道路，東接新莊區新泰路，西往新北市林口區（13.5K~20.3K）。
*[[File:TW_CHW107.svg|市道]][[市道107號|市道107號]]（明志路）：縱向貫穿泰山的最主要道路，亦為[[市道107號|107市道]]，北接五股區成泰路通往八里淡水，南接新莊區中正路，延路皆為商店，是泰山區的都會中心（4.8K~9.3K）。
*[[File:TW_CHW107a.svg|市道]][[市道107號#支線|市道107甲]]（新五路）：五股交流道聯外道路，北往五股，南往新莊。由楓江路與中港西路與泰山市區相連（3.0K~3.8K）。

*[[File:TW_THWtp72.png|30px]][[北72線|北72線]]（貴子路）：貴子地區主要道路，可通往[[輔仁大學|輔大]]後門與新莊、樹林地區，西起明志路三段，接[[新莊區|新莊區]]三泰路，東接[[中正路|中正路]]與新樹路。

*[[File:TW_THWtp117.png|30px]][[北117線|北117線]]（南林路）：起自南林科技園區（南林路），為昔跨縣鄉道桃北9-1線的改編，可連接[[File:TW_THWtp118.png|30px]][[北118線|北118線]]（壽山路）續行前往[[桃園市|桃園市]][[龜山區|龜山區]][[華亞科技園區|華亞科技園區]]，或下山前往[[新莊區|新莊區]][[丹鳳_(新北市)|丹鳳]]地區。
