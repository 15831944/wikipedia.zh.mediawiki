'''泰穆·普基'''（{{lang-fi|Teemu Pukki}}，{{bd|1990年|3月29日|catIdx=Pukki, Teemu}}），是一名[[芬兰|芬兰]][[前锋_(足球)|前锋]]，现效力于[[英冠|英冠]]球队[[諾里奇城足球俱樂部|諾域治]]。

== 俱乐部 ==
普基的职业足球生涯始于家乡球队[[科泰佩|科泰佩]]。<ref>{{cite web |url=http://www.fckooteepee.fi/?page=team&playerId=219&teamRef=7 |title=16 Teemu 'Teme' Pukki |publisher=FC KooTeePee |date= |accessdate=2011年11月12日 |language=fi }}</ref>2006年4月21日的[[芬兰足球超级联赛|芬超]]比赛，刚满16周岁的普基首次代表科泰佩出场，帮助球队在主场以2-1战胜劲旅[[HJK赫尔辛基|HJK赫尔辛基]]。<ref name="veikkausliiga">{{cite web |url=http://www.veikkausliiga.com/pelaajatilastot.asp?Nimi=Pukki+Teemu |title=VEIKKAUSLIIGAN PELAAJAT |publisher=[[芬兰足球超级联赛|芬超]]官网 |date= |accessdate=2011年11月12日 |language=fi }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>这个赛季他在联赛中共为球队出场5次，均为替补出场<ref name="veikkausliiga"/>，并被[[芬兰足协|芬兰足协]]评为年度最佳少年球员<ref name="vpp">{{cite web |url=http://www.palloliitto.fi/viestinta/?num=84015 |title=Vuoden parhaat, 2006 |publisher=[[芬兰足协|芬兰足协]] |date=2006年11月4日 |accessdate=2011年11月12日 |language=fi |deadurl=yes |archiveurl=https://web.archive.org/web/20070926230036/http://www.palloliitto.fi/viestinta/?num=84015 |archivedate=2007年9月26日 }}</ref>。2007年5月3日，科泰佩客场1-0战胜[[维京人足球俱乐部|维京人]]，普基攻入个人首粒芬超进球，也成为第一位在芬超取得进球的“90后”球员<ref>{{cite web |url=http://www.veikkausliiga.com/News.aspx?type=1&ID=24813 |title=Pukki ensimmäinen 90-lukulainen |publisher=[[芬兰足球超级联赛|芬超]]官网 |date=2007年5月5日 |accessdate=2011年11月12日 |language=fi |deadurl=yes |archiveurl=https://web.archive.org/web/20070930024101/http://www.veikkausliiga.com/News.aspx?type=1&ID=24813 |archivedate=2007年9月30日 }}</ref>。这个赛季他10次首发、14次替补出场，在24场比赛中打进3球。<ref name="veikkausliiga"/>

2008年1月，普基转会西班牙[[塞维利亚足球俱乐部|塞维利亚]]，与新俱乐部签了3年半合同。<ref name="hs">{{cite web |url=http://www.hs.fi/english/article/Football+prospect+Teemu+Pukki+chooses+Spain+and+Sevilla+FC/1135233297207 |title=Football prospect Teemu Pukki chooses Spain and Sevilla FC |publisher=赫尔辛基新闻 |date=2008年1月15日 |accessdate=2011年11月14日 |language=en }}</ref>同年6月，他帮助塞维利亚青年队在决赛上以2-0战胜巴塞罗那青年队，赢得西班牙青年国王杯冠军。<ref name="esp_cup">{{cite web |url=http://www.kymensanomat.fi/Online/2008/06/29/Pukin+Sevilla+Espanjan+cupmestariksi/200835479994/4 |title=Pukin Sevilla Espanjan cupmestariksi |publisher=屈米新闻 |date=2008年6月29日 |accessdate=2011年11月14日 |language=fi |deadurl=yes |archiveurl=https://web.archive.org/web/20081208103223/http://www.kymensanomat.fi/Online/2008/06/29/Pukin+Sevilla+Espanjan+cupmestariksi/200835479994/4 |archivedate=2008年12月8日 |df= }}</ref>在青年队度过一段时间后，他被提拔到俱乐部的[[预备队|预备队]]，即[[塞维利亚竞技|塞维利亚竞技]]。2008-09赛季[[西乙|西乙]]第6轮，普基首次代表塞维利亚竞技出场，并打进全场唯一进球，帮助球队战胜[[皇家社会足球俱乐部|皇家社会]]。<ref>{{cite web |archivedate=2011年10月5日 }}</ref>2008年12月7日，塞维利亚客场挑战[[皇家马德里|皇家马德里]]，普基首次进入一队18人名单，但并未出场。<ref>{{cite web|url=http://www.sevillafc.es/_www/actualidad.php?op=not&id=12653&leng=esp |title=PUKKI, GRAN }}</ref>2009年1月29日，塞维利亚主场对阵[[桑坦德竞技|桑坦德竞技]]，普基在第63分钟替下[[伊维卡·德拉古蒂诺维奇|伊维卡·德拉古蒂诺维奇]]，完成个人[[西甲|西甲]]处子秀。<ref>{{cite web|url=http://www.sevillafc.es/_www/temporada.php?op=cro&id=4049 |title=EL SEVILLA NUNCA |url=http://www.soccerway.com/matches/2009/01/25/spain/primera-division/sevilla-futbol-club/real-racing-club-de-santander/659824/ |title=Sevilla vs. Racing Santander  0 - 2 |publisher=soccerway.com |date= |accessdate=2011年11月14日 |language=en }}</ref>但这也是他唯一一次代表塞维利亚一队出场。

由于在西班牙上场机会太少，普基决定回到芬超。2010年8月，普基与[[HJK赫尔辛基|HJK赫尔辛基]]签约3年半。<ref>{{cite web |url=http://www.mtv3.fi/urheilu/futis/uutiset.shtml/2010/08/1178306/teemu-pukin-hjk-debyytti-meni-silmaa-pidellessa |title=Teemu Pukin HJK-debyytti meni silmää pidellessä |publisher=mtv3.fi |date=2010年8月30日 |accessdate=2011年11月14日 |language=fi }}</ref>10月份赛季结束前，普基共为赫尔辛基出场7次，打进两球，并随队获得联赛冠军。<ref name="veikkausliiga"/>2011赛季，普季表现出色，其中还在6月22日6-0大胜[[TPS图尔库|TPS图尔库]]的比赛中首次上演[[帽子戏法|帽子戏法]]。<ref>{{cite web |url=http://www.iltalehti.fi/jalkapallo/2011062213939066_vg.shtml |title=Joko odotat maajoukkuekutsua, Teemu Pukki? |first=Lauri |last=Markkanen |publisher=iltalehti |date=2011年6月22日 |accessdate=2011年11月14日 |language=fi }}</ref>8月18日，赫尔辛基在[[欧罗巴联赛|欧罗巴联赛]]附加赛首回合爆冷以2-0战胜[[沙尔克04|沙尔克04]]，两粒进球均来自普基。一周后，在[[盖尔森基兴|盖尔森基兴]]举行的次回合比赛，赫尔辛基大比分失利，但普基再次攻破沙尔克的球门。普基的优异表现引起了沙尔克的关注。8月31日转会窗关闭前，沙尔克宣布签下普基，合同至2014年6月底。<ref>{{cite web |url=http://www.schalke04.de/aktuell/news/einzelansicht/artikel/s04-verpflichtet-angreifer-teemu-pukki-von-hjk-helsinki.html |title=S04 verpflichtet Angreifer Teemu Pukki von HJK Helsinki |publisher=[[沙尔克04足球俱乐部|沙尔克04足球俱乐部]] |date=2011年8月31日 |accessdate=2011年11月14日 |language=de }}</ref>转会沙尔克之前，普基在2011赛季芬超联赛中出场18次，打进11球，助攻8球，在欧洲赛事中出场6次，打进5球。

2011年9月18日，普基首次亮相[[德甲|德甲]]联赛，在主场和[[拜仁慕尼黑|拜仁慕尼黑]]的比赛下半场替补出场。11月6日，德甲第12轮，普基在联赛中第一次作为首发球员出场，并且打进两球，帮助沙尔克在客场以2-2逼平[[汉诺威96|汉诺威96]]，同时还入选了《[[踢球者|踢球者]]》杂志等多家媒体评选的当轮最佳阵容。<ref>{{cite web |url=http://www.kicker.de/news/fussball/bundesliga/spieltag/1-bundesliga/2011-12/12/elf-des-tages-am-spieltag.html |title=1. Bundesliga: die Elf des Tags am 12. Spieltag der Saison 2011/12 |publisher=[[踢球者|踢球者]] |date= |accessdate=2011年11月14日 |language=de }}</ref><ref>{{cite web |url=http://sports.sohu.com/20111108/n324928635.shtml |title=德媒狂赞沙尔克21岁小将 未来帝星首秀轻松两球 |author=艾晚 |publisher=[[搜狐|搜狐]]体育 |date=2011年11月8日 |accessdate=2011年11月14日 |language= }}</ref>

== 国家队 ==
2009年2月4日，[[芬兰国家足球队|芬兰队]]与[[日本国家足球队|日本队]]在东京进行友谊赛，普基在第67分钟换下[[亚里·利特马宁|利特马宁]]，首次代表国家队出场。<ref>{{cite web |url=http://www.palloliitto.fi/maajoukkueet/miesten_a-maajoukkue/pelaajat/pukki_teemu/ |title=Pukki, Teemu |publisher=[[芬兰足协|芬兰足协]] |date= |accessdate=2011年11月14日 |language=fi |deadurl=yes |archiveurl=https://web.archive.org/web/20111026120434/http://www.palloliitto.fi/maajoukkueet/miesten_a-maajoukkue/pelaajat/pukki_teemu/ |archivedate=2011年10月26日 }}</ref><ref>{{cite web |url=http://www.palloliitto.fi/viestinta/?num=135657 |title=Huuhkajille selvä tappio Japanissa |publisher=[[芬兰足协|芬兰足协]] |date=2009年2月4日 |accessdate=2011年11月16日 |language=fi |deadurl=yes |archiveurl=https://web.archive.org/web/20120315023340/http://www.palloliitto.fi/viestinta/?num=135657 |archivedate=2012年3月15日 }}</ref>
