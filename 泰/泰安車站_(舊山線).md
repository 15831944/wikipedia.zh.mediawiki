'''泰安舊站'''是一個位在[[台灣|台灣]][[臺中市|臺中市]][[后里區|-{后里}-區]]、[[臺灣鐵路管理局|臺灣鐵路管理局]]（臺鐵）[[舊山線|舊山線]]沿線的已停用[[鐵路車站|鐵路車站]]。在臺鐵將山線的行車路線遷往[[臺中線|新線]]後，泰安舊站的站房與附近包括隧道在內的相關鐵路設施經整理後開放給一般民眾參觀，命名為'''泰安鐵道文化園區'''，是鄰近地區頗受歡迎的觀光景點，而車站站房也被臺中縣政府（今臺中市政府）指定為臺中縣縣定古蹟（今臺中市市定古蹟）。

== 營運狀況 ==
1998年9月24日舊山線停用後，由於路線切換至新線，本站停止使用，乘客需改至新設的[[泰安車站_(台中線)|泰安車站]]乘車。站房及站場轉為觀光用途，站內軌道與[[轉轍器|轉轍器]]仍維持廢站前的配置，但電車線已被拆除，舊山線軌道只到本站站場的南端為止，並未銜接到[[后里車站|后里站]]。

自2010年6月5日起，舊山線復駛觀光列車，12年來首次有列車進站。

因2018年台中市將主辦[[2018年臺中世界花卉博覽會|花卉博覽會]]，台中市政府於2017年5月提出舊山線泰安站至后里站的復駛計畫，將鋪回鐵軌並整修大安溪鐵橋。<ref>[http://times.hinet.net/news/20212379 中市推動后里舊山線復駛計畫 預估創造40億觀光年產值]</ref>

== 歷史 ==
*1910年12月1日：設「大安溪信號場」。<ref name=":0">{{cite book |author = 戴震宇 |authorlink =  |title = 《台灣的老火車站》 |publisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>
*1912年7月1日：升為車站。
*1920年：改為「大安」。<ref name=":0" />
*1935年：[[1935年新竹台中地震|新竹台中地震]]，原木造站房毀損，重新改建成混泥土建築，並在車站的後方設立地震復建紀念碑，此外，重建後的站房形式與[[造橋車站|造橋車站]]雷同。
*1954年3月1日：改為「泰安」。<ref name=":0" />
*1998年9月24日：停用。
*2010年6月5日：[[舊山線|舊山線]]復駛，首班車以[[CK124|CK124]]蒸氣火車行駛。
