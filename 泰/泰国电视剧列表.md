'''本表'''列出泰国各电视台制作、播出的电视剧。播出时间为泰国时间（比北京时间早1个小时）。

== 概述 ==
泰国电视剧播出时间分晚间档电视剧，黄金档电视剧和处境喜剧。泰国电视剧播出基本出自在泰国3台，5台，7台。

2013年
晚间档电视剧：
:           3台每天播出2部电视剧，播出时间周一至周五17：45，周一至周五18：30
:           7台每天播出2部电视剧，播出时间周一至周五18：00，周一至周五18：40
黄金档电视剧：
:           3台每周播出3部电视剧，播出时间分别为周一周二20：30，周三周四20：30，周五至周日20：15
:           7台每周播出3部电视剧，播出时间分别为周一周二20：25，周三周四20：25，周五至周日20：25
:           5台每周播出2部电视剧，播出时间分别为周一周二20：10，周三周四20：10
:           8台每周播出1部电视剧，播出时间分别为周一至周二20：30
处境喜剧：
:           3台每周播出3部电视剧，播出时间分别为周四23：15，周六23：15，周日10：15 
:           7台每周播出4部电视剧，播出时间分别为周六11：15，周六13：00，周日11：15，周日13：00
:           5台每周播出2部电视剧，播出时间分别为周六12：50，周六20：30
:           9台每周播出3部电视剧，播出时间分别为周六15：00，周六18：00，周日13：25
