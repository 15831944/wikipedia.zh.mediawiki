《'''泰国公主'''》是一幅由時任[[台灣師範大學|台灣師範大學]]美術系教師的画家[[孙多慈|孙多慈]]于1961年完成的布面[[油画|油画]]作品<ref>{{cite web |url=http://www.artnet.de/k%c3%bcnstler/sun-duoci/taiguogongzhu-q42umihFM0aPKL7tEZpIfg2 |title=Vergangene Auktion |last= |first= |date= |website= |publisher= |access-date= |quote=}}</ref>。

== 描述 ==
[[File:KITLV_10725_-_Kassian_Céphas_-_Studio_picture_of_a_young_woman_in_dance_pose_from_Yogyakarta_-_Around_1900.tif|thumb]]
画中人物跪坐在一个祭台前，两手合十拿着一朵[[缅栀花|缅栀花]]；身穿绿色抹胸，系有一条黄色腰带，下穿刺绣着金色图案的红底长裙。虽然题为泰国，但服装不像是[[泰国|泰国]]的传统服饰，而是[[印尼|印尼]][[爪哇族|爪哇族]]和[[巴厘族|巴厘族]]的传统服装{{link-en|坎本|Kemben}}。头上戴着两朵未开的花，巧合的是发型与后来多次访华的泰国公主[[诗琳通|诗琳通]]相似<ref>{{cite web |url=http://51lovehome.com/html/tslz/2010-09/07/content_3702779.htm |title=孙多慈1961年油画《泰国公主》（图） |last= |first= |date=2010-09-07 |website=艺术中国 |publisher= |access-date= |quote=}}</ref>。

== 轶闻 ==
20年后即1981年，泰国[[诗琳通公主|诗琳通公主]]首次访问中国，并写了一部题为《[[踏访龙的国土|踏访龙的国土]]》的游记。
