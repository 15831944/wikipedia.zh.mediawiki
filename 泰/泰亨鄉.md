== 歷史 ==
===泰亨文氏===
[[新界|新界]]文氏始祖[[文天瑞|文天瑞]]，為[[南宋|南宋]]抗[[元朝|元]][[丞相|丞相]][[文天祥|文天祥]]的堂弟，於南宋末年逃避元兵移居[[東莞|東莞]]<ref>[http://news.takungpao.com/paper/q/2014/0221/2291083.html 香港文氏始祖]</ref>。新界文氏原籍[[江西|江西]][[吉安|吉安]]，族人主要聚居於[[元朗區|元朗]][[新田_(香港)|新田]]及大埔泰亨。[[新田文氏|新田文氏]]源自天瑞三世孫[[文起東|文起東]]一系<ref>[https://mmis.hkpl.gov.hk/coverpage/-/coverpage/view?_coverpage_WAR_mmisportalportlet_ref=LPEC03&p_r_p_-1078056564_c=QF757YsWv58JCjtBMMIqooA96qE2Gb9d 口頭傳統和表現形式 - 宗族口述傳說（新田文氏）]</ref>，而泰亨文氏則源自文起東之弟[[文起南|文起南]]一系；兩地文氏同屬[[寶安文氏|寶安文氏]]七大房。

文起南之長孫文蔭為泰亨文氏之開基祖，約於1300年由[[屯門|屯門]]老虎坑上岸<ref>{{cite web |url=http://www.somanhing.com/gotowalk/clan/man/intro.pdf|title=新界文氏簡史}}</ref>。蔭公先居[[屏山|屏山]]
，後遷大埔[[陶子峴|陶子峴]]，以造碗為生；族人於[[泮涌|泮涌]]建立「文家莊」，養鴨務農。隨後因人口漸多而轉居泰亨，成為區內一大宗族。現今，泰亨文氏約一千人，分三大房<ref>[https://mmis.hkpl.gov.hk/coverpage/-/coverpage/view?_coverpage_WAR_mmisportalportlet_ref=LPEC03&p_r_p_-1078056564_c=QF757YsWv58JCjtBMMIqovVq%2FQJ1W7Gd 口頭傳統和表現形式 - 宗族口述傳說（泰亨文氏）]</ref>。

===太和市===
[[太和市|太和市]]，即[[大埔墟|大埔新墟]]，是泰亨文氏領導[[大埔七約|大埔七約]]建立的墟市，每月即初三、十三、廿三、初六、十六、廿六及初九、十九、廿九為墟期。太和市曾是[[新界東|新界東]]第一大墟市，建立目的為挑戰新界五大氏族之首[[香港新界鄧氏|鄧氏]]壟斷的[[大埔頭|大步頭墟]]生意<ref>[http://www.pof.org.hk/new/s244_content.php?id=76 無障礙好去處 大步墟市爭今昔] {{webarchive|url=https://web.archive.org/web/20150529074229/http://www.pof.org.hk/new/s244_content.php?id=76 |date=2015-05-29 }}</ref>。

[[嘉慶|嘉慶]]年間，[[龍躍頭|龍躍頭鄧氏]]在[[新界東|新界東]]影響力大不如前。相反，泰亨文氏堀起，擁有泮涌及[[粉嶺|粉嶺]]以南大片土地<ref>[https://books.google.com.hk/books?id=ZLrTBAAAQBAJ&pg=PA20&lpg=PA20&dq=%E7%B2%89%E5%B6%BA%E4%BB%A5%E5%8D%97+%E5%A4%AA%E5%9D%91&source=bl&ots=aCIRjkPQsg&sig=yBRkaz6fBerc-rsoZjeG1ec0nSI&hl=en&sa=X&ved=0CDgQ6AEwBGoVChMItfX72-vQxwIVio6UCh0GMwwR#v=onepage&q=%E7%B2%89%E5%B6%BA%E4%BB%A5%E5%8D%97%20%E5%A4%AA%E5%9D%91&f=false 香港新界侯族的建構 宗族組織與地方政治和民間宗教的關係]</ref>，並多次計劃在大埔開設商舖，但屢次遭[[大埔頭|大埔頭]]鄧氏阻攔。鄧氏兩度控告文氏，其後新安縣衙門宣布文氏不得「恃強」侵奪鄧氏在大埔的商業利益，判鄧氏勝訴。文氏族人[[文湛泉|文湛泉]]乃於1892年聯合附近七個非鄧氏的村落，組成「[[大埔七約|大埔七約]]」（泰亨約、[[林村|林村約]]、[[翕和|翕和約]]、[[沙羅洞|集和約]]、[[大埔尾_(香港)|樟樹灘約]]、[[汀角村|汀角約]]、[[粉嶺圍|粉嶺約]]），於「大步墟」隔河另設鄉鎮，翌年，定名為「太和市」。「太」為太坑，「和」為和平。

1898年，[[英國|英國]]租借新界後，大埔區因位於新界東中央而成為[[行政中心|行政中心]]。1913年，太和市旁的[[香港鐵路博物館|大埔墟火車站]]落成營運，舊有的大埔墟日漸衰落，太和市得地利，店戶日增，故習慣將太和市稱[[大埔墟|大埔墟]]，原有的大埔墟則稱大埔舊墟<ref>[http://www.oocities.org/hk/tphistory/history_market02.htm 大埔今昔]</ref>。

== 村落 ==
[[File:Hong_Lok_Yuen_overview.JPG|thumb]]位於[[康樂園|康樂園]]與[[粉嶺公路|粉嶺公路]]之間]]
泰亨鄉目前為大埔區[[人口|人口]]最多和[[面積|面積]]最大的[[單姓|單姓]][[香港圍村|原居鄉村]]，鄉村被[[粉嶺公路|粉嶺公路]]穿過<ref>{{Cite web |url=http://www.had.gov.hk/rre/chi/village_map/tp_tp_rc1518.html |title=大埔鄉事委員會 鄉村範圍圖 |access-date=2015-06-05 |archive-url=https://web.archive.org/web/20150605162148/http://www.had.gov.hk/rre/chi/village_map/tp_tp_rc1518.html |archive-date=2015-06-05 |dead-url=yes }}</ref>。文氏族人主要聚居於粉嶺公路以西的「兩圍一村」，當中「兩圍」是指[[中心圍|中心圍]]、[[灰沙圍|灰沙圍]]，「一村」則是[[祠堂村|祠堂村]]<ref>[http://www.careaction.org.hk/archive/issue15/taipo.htm 新春大埔古村探勝]</ref>，其他村落還有坳仔村和新圍。泰亨鄉亦為大埔區內最多[[學塾|學塾]]的鄉村，[[清代|清代]]建有「[[藝沅學校|藝沅學校]]」、「[[善慶書室|善慶書室]]」、「[[叢桂書室|叢桂書室]]」和「[[正倫書室|正倫書室]]」<ref>[http://www.oocities.org/hk/tphistory/history_education.htm 大埔今昔 大埔早期教育]</ref>。

== 名勝 ==
[[File:HK_TaiHang_100years_DoubleTrunkTree.jpg|thumb]] 
泰亨鄉文氏宗祠於2005年中秋完成修繕工程，宗祠的建材選料考究，當中石柱、石雕等在[[福建|福建]]訂製；而青磚、漢瓦在[[廣東|廣東]][[開平|開平]]採用人手鑄造；至於精美的神樓及案台在[[深圳|深圳]][[布吉|布吉]]及[[松崗|松崗]]訂造；檐口板雕花、神樓門神、屏風更是遠道到[[浙江|浙江]]邀請工藝師人手雕刻，祠內牌匾「浩然正氣」及「厚德參天」由國學家文懷沙親筆書寫。而同年亦建成文山公園，紀念[[文天祥|文天祥]]的史蹟，內有文天祥全身站像，而龔子亭是紀念泰亨鄉文化始祖蔭公（字龔子）而興建，園內尚有《[[正氣歌|正氣歌]]》碑刻、蓮池、曲徑等；其後更得政府當局協助，於宗祠及文山公園前築建牌樓，以增莊嚴。

天后宮於1728年（即雍正五年）建成，宮內現存雍正時所鑄造的古鐘及爐鼎；而毗連的文帝古廟則建於1884年（即光緒十年）。廟旁有兩棵[[細葉榕|細葉榕]]，樹幹相連形成連理樹，據說已有百年歷史<ref>[http://programme.tvb.com/variety/bigfuncanton/episode/25/ 千奇百趣省港澳第25集]</ref>。

藝沅堂內祀奉十六世祖文宗齋等三代祖先木主，是典型兩進式[[祠堂|祠堂]]，亦為[[香港三級歷史建築|香港三級歷史建築]]。由藝沅堂演變而來的泰亨公立學校於1962年創辦，可是隨著村校收生不足，終因政府殺校而停辦。

== 太平清醮 ==
[[File:HK_DaJinOpera_TaiHang_2015.jpg|thumb]]年太平清醮大戲棚]]
泰亨鄉每五年舉行一次[[香港太平清醮|太平清醮]]，未來一屆為2020年。

泰亨鄉文族於建村立業後，為酬謝神恩佑庇鄉人，訂下五年一屆太平清醮，成為俗例。文族認為建醮以示崇敬，祀神以表至誠，普施濟眾，惠澤幽靈，用意祈求風調雨順，合境康泰。
