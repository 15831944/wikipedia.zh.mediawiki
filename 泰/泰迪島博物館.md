'''泰迪島博物館'''（[[泰文|泰文]]：พิพิธภัณฑ์ ตุ๊กตาหมี พัทยา；[[英文|英文]]：Teddy Island Museum）位于[[泰國|泰國]]東部[[春武里府|春武里府]]。它是一間在2013年9月27日（[[星期五|星期五]]）開始營業，佔地超過0.25[[公頃|公頃]]的私人牟利[[博物館|博物館]]，其[[建築物|建築物]]之[[外觀|外觀]][[設計|設計]]為一艘40[[公尺|公尺]]長的[[龍舟|龍舟]]，[[展品|展品]]的[[主題|主題]]為[[玩具|玩具]][[熊|熊]]，可同時接待數千名[[觀光客|觀光客]][[參觀|參觀]]及[[拍照|拍照]]。

== 展品 ==
數千隻[[玩具|玩具]][[熊|熊]]被裝飾成12個主題的[[展覽館|展覽館]]：
* [[南美洲|南美洲]][[印加|印加]][[部落|部落]]（โซนชนเผ่าอินคา；Inca）；
* [[恐龍|恐龍]]（โซนไดโนเสาร์；Dinosour）；
* [[恐龍|恐龍]][[化石|化石]]（โซนฟอสซิล；Fossil）；
* [[非洲|非洲]]（โซนแอฟริกา；Africa）；
* [[泰國|泰國]]（โซนเมืองไทย；Thailand）；
* 海底世界（โซนใต้ท้องทะเล；Underwater）；
* [[愛斯基摩人|愛斯基摩人]]的[[文化|文化]]（โซนเอสกิโม；Eskimo）；
* [[聖誕節|聖誕節]]（โซนเมืองซานต้า；Santa）；
* [[太空|太空]]（โซนอวกาศ；Space）；
* [[神話|神話]]及[[傳說|傳說]]（โซนเทพนิยาย；Fair Tale）；
* 命名為『泰迪城』的[[中國|中國]][[牌坊|牌坊]]（โซนจีน；China）；
* [[歐洲|歐洲]]（โซนยุโรป；Europe）。

== 開放時間 ==
* 每天[[上午|上午]]9時至[[晚上|晚上]]10時，[[晚上|晚上]]8時30分停止入場。
* [[星期六|星期六]]、[[星期日|星期日]]有三場（下午2時；下午3時；下午4時）以[[泰迪熊|泰迪熊]]為主題的歌舞表演。
*  入場費：外國成人500泰銖；外國兒童300泰銖，身高90[[厘米|厘米]]以下的兒童，免費入場。（[[泰國|泰國]]國民成人250泰銖；[[泰國|泰國]]國民兒童：150泰銖。）
