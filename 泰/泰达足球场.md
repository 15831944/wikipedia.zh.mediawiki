'''泰达足球场'''是一座位于[[天津市|天津市]][[滨海新区|滨海新区]]的专业[[足球场|足球场]]，为[[中超|中超]]球队[[天津泰达足球俱乐部|天津泰达足球俱乐部]]全资拥有。球场建造、场地尺寸完全符合[[国际足联|国际足联]]标准，因此也符合奥林匹克和世界杯的规格标准。泰达足球场于2005年度[[中国建筑工程鲁班奖|中国建筑工程鲁班奖]]。<ref>[http://www.tj.xinhuanet.com/zhibo/2006-01/20/content_6089011.htm 2005年度中国建筑工程鲁班奖(国家优质工程)获奖名单公示] {{webarchive|url=https://web.archive.org/web/20060531031801/http://www.tj.xinhuanet.com/zhibo/2006-01/20/content_6089011.htm |date=2006-05-31 }}</ref>[[国际足联|国际足联]]体育场馆安全技术主任沃尔特·盖格2004年4月7日对天津泰达足球场进行了实地考察，他表示：“这座场馆在亚洲乃至世界都是第一流的，足球场从设计到施工都非常完美。”<ref>[http://news.enorth.com.cn/system/2004/04/06/000763733.shtml 接受FIFA官員考察 泰達球場被評為世界一流]</ref>在[[2015年天津港危化品仓库爆炸事故|2015年天津港危化品仓库爆炸事故]]中，泰达足球场受损严重，被迫关闭。

== 场馆结构 ==
泰达足球场按照国际足联标准建设，因而场内不设田径跑道，使得观众可以近距离观看比赛。足球场东西两侧看台为四层，南北两侧看台为二层，三层看台为贵宾包厢，席位总数可达34559个；看台下部设计有比赛附属用房、观众服务用房等附属空间。<ref>[http://www.tedapromenades.com.cn/zuqiuchang/about.asp 泰达足球场官方网站的场馆介绍] {{dead link}}</ref>

== 草坪 ==
泰达足球场为专用足球场，场地尺寸长为105米，宽为68米，场地四周设有深1.4米、宽1.8米的安全壕沟，比赛场地的表层设有加热系统，以防天气寒冷时造成草皮结冻，可承办所有洲际足球比赛，被认为是[[亚洲|亚洲]]乃至[[世界|世界]]第一流的足球场地<ref>[http://www.sportsonline.com.cn/GB/channel2/57/214/20040408/216677.html 国际足联官员来津考察 张吉龙领略“泰达速度”]</ref>，{{来源请求|是中国唯一通过FIFA认证的专业足球场}}。

== 场馆外观 ==

场馆成长方形，罩棚为弧形结构，外围有弧形的桅杆林立，类似太空舱的造型，具有现代感。

== 举办赛事 ==
泰达足球场落成后的首场比赛为2004年4月25日由天津泰达-[[澳大利亚|澳大利亚]][[南墨尔本足球俱乐部|南墨尔本]]的友谊赛，天津泰达以2比1获胜。<ref>[http://www.sportsonline.com.cn/GB/channel2/57/214/20040426/220246.html 人民网]</ref>
2004年5月15日，泰达足球场承办了[[2004年中国足球超级联赛|首届中国足球超级联赛]]的[[开幕式|开幕式]]，随后举行了首场正式比赛，由天津泰达对阵[[重庆力帆|重庆力帆]]。<ref>[http://sports.sohu.com/2004/05/15/02/news220140283.shtml 中国足球进入新的纪元 泰达大球场见证中超揭幕]</ref>

== 大型活动 ==
泰达球场落成典礼暨文艺晚会亦于2004年4月30日举行，[[周杰伦|周杰伦]]，[[S.H.E|S.H.E]]，[[徐怀钰|徐怀钰]]演艺明星受邀演出。<ref>[http://sh.tedaonline.com/sh/shhd/04/hd_4more.asp?Id=106 开发区商会活动内容] {{webarchive|url=https://web.archive.org/web/20110405213031/http://sh.tedaonline.com/sh/shhd/04/hd_4more.asp?Id=106 |date=2011-04-05 }}</ref>

== 关闭与修复 ==
在[[2015年天津港危化品仓库爆炸事故|2015年天津港危化品仓库爆炸事故]]中，球场受损严重，加之固有的地面沉降问题，被迫关闭。据《今体育》报道，在2017年底，修复计划已启动。<ref>[http://www.dongqiudi.com/archive/467833.html]，今体育：泰达球场修复计划开启。</ref>
