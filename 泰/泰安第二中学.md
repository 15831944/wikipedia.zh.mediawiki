'''泰安第二中学'''，是位于[[中华人民共和国|中华人民共和国]][[山东省|山东省]][[泰安市|泰安市]][[泰山区|泰山区]]虎山东路6号的一所中学，学制三年。于1950年后创立。整个校园占地160多亩。该校是一所省级规范的重点中学，至2019年已建校66周年，具有高级教师，特级教师教职人员职工。

== 学校活动 ==
学校于2013年举办了60周年校庆活动，为此学校准备了一场盛大的演出，并有请历年学校知名校友返校共庆。<ref>{{cite web |title=山东省泰安市第二中学新闻显示 |url=http://www.taez.com.cn/taez/news?action=show&f=1&newsId=1 |website=www.taez.com.cn |accessdate=2019-04-02}}</ref>

== 二中校友办 ==
[[File:Mmexport1554649380090.jpg|thumb]]
2014年初校友办正式成立，至今2年时间。目的在开发校友资源，弘扬二中文化，搭建校友平台，共谋学校发展。2年中，校友办在学校党委的正确领导下，扎扎实实努力做好校友寻访工作。2年中，先后拜访了泰安市、济南市、青岛市、北京市、辽宁省沈阳市与锦州市、黑龙江省哈尔滨市等地的校友。2年中也有各级、各届校友先后回访母校，他们看望恩师、汇报成绩，以各种方式回馈母校。

== 知名校友 ==
=== 范方平 ===
[[File:Mmexport1554649396515.jpg|thumb]]
1945年11月出生。1965年毕业我校，1969年毕业北京政法学院，曾任辽宁省人民法院院长，国家司法部副部长，第九届全国人民代表大会代表。

=== 张克智 ===
[[File:Mmexport1554649401205.jpg|thumb]]
1941年1月——2004年11月，1960年毕业于我校。曾任国务院副秘书长、鼓舞元机关党组副书记。

=== 高建一 ===
[[File:Mmexport1554649405611.jpg|thumb]]
1953年1月出生，空军少将。1969年毕业于我校。曾任中国人民解放军空军武汉基地政治委员，广州军区空军军事检查院检察长，现任广州军区政治部副主任。

=== 游铭钧 ===
[[File:Mmexport1554649410399.jpg|thumb]]
1956年毕业于我校。曾任北京景山学校校长兼党支书记，国家教育部基础教育司副司长和教育部课程教材研究中心主任。在担任北京景山学校校长期间，请邓小平同志为景山学校题词：教育要面向现代化，面向世界，面向未来。
