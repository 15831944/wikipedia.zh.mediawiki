[[泰王國|泰王國]]的行政區劃分為四個等級。其中[[一級行政區|一級行政區]]是「[[泰國的府|府]]」（{{lang|th|จังหวัด}} ''changwat''）。府的以下的次級行政區是「縣」<ref name="zhou">{{cite book|title=[[世界地名翻译大辞典|世界地名翻译大辞典]]|author=周定国|publisher=中国对外翻译出版公司|year=2008|isbn=978-7-5001-0753-8|page=1233|quote=府以下设置795个县（amphur）。县以下设区、村。曼谷为府级直辖市（metropolitan administration）；芭堤雅设为府级特别市（special administrative city）。}}</ref><ref name="waijiaobu">{{cite news|url=https://www.fmprc.gov.cn/web/gjhdq_676201/gj_676203/yz_676205/1206_676932/1206x0_676934/|title=泰国国家概况|author=中华人民共和国外交部|date=|quote=全国分中部、南部、东部、北部和东北部五个地区，共有77个府，府下设县、区、村。曼谷是唯一的府级直辖市。}}</ref><ref>{{cite news|url=http://hi.people.com.cn/n2/2017/0727/c234377-30538744.html|title=一中国女游客在泰国普吉坠海身亡|author=人民网|date=2017-07-27|quote=普吉府甲涂县}}</ref>（{{lang|th|อำเภอ}} ''amphoe''），又譯「大區」或「郡」。

如果在[[曼谷|曼谷]]的次級行政區則稱為''khet''（{{lang|th|เขต}}），中國大陸官方將其譯作「縣」，<ref>{{cite web|url=http://world.people.com.cn/n1/2018/0131/c1002-29797269.html|title=曼谷的唐人街，你知道多少|author=人民网 人民日报海外版|date=2018-01-31|quote=曼谷三攀他旺县}}</ref>也有文章將該詞譯為「區」。今天大曼谷下分50個縣，曼谷以外劃76個府，各府皆下分為若干個縣。

次級行政區「縣」之下的行政單位依次是「{{link-en|泰國三級行政區劃|Tambon|區}}」（{{lang|th|ตำบล}} ''tambon''）、「{{link-en|泰國四級行政區劃|Muban|村}}」（{{lang|th|หมู่บ้าน}} ''muban''）。<ref name="zhou"/><ref name="waijiaobu"/>

== 县的面積、人口 ==
泰国各县（又译郡）的面積與人口都有很大差別，最少為[[桐艾府|桐艾府]]的[[沽島縣|沽島縣]]（''Koh Kut''）僅2042人，最多為[[色功府縣|色功府縣]]接近45萬人。曼谷[[三攀他旺縣|三攀他旺縣]]面積最小僅1.4平方公里，而[[達府|達府]][[翁龐縣|翁龐縣]]（''Umphang''）面積達4325平方公里，她甚至大於很多個府，同時地處偏遠山區的翁龐也是全國人口密度最低的縣。

== 县的名字 ==
县的名字多數是獨一無二的，少數县因為音近，所以轉寫為拉丁字後的譯名竟然相同。1996年，為了紀念泰王[[拉瑪九世|普密蓬]]登基五十週年，「差隆帕傑」<!--（''Chaloem Phra Kiet''）-->名字（''Chaloem Phra Kiat''）被同時賜封給五個新縣，因為這個名字是對王室成員的敬稱，除此以外，別無例外。	

各县的長官稱為县令（郡令），他們由[[泰王國民政部|泰王國民政部]]委派，必須向本府的[[府尹|府尹]]彙報。

== 府治县（''Amphoe Mueang''） ==
各府的行政中心稱為府治县（又译府治郡、直辖县、大區），往往在一個小城市。包含這個城市的行政區域則稱為府治縣（例如：普吉府治縣、清邁府治縣等）。在1930年以前各府中心城市的名稱不盡同于府的名稱，但是隨著當年進行的行政制度改革，名稱都統一了。唯一的例外是[[大城府|大城府]]的府治縣稱為「聖城阿育跎耶府治縣」，而大城府全名則是「聖城阿育跎耶府」。

各府治县管轄下的城市往往是本府最大的，唯一例外是[[宋卡府|宋卡府]]的[[合艾市|合艾市]]原來是第二大城，由於興建了火車站和機場，較好的基礎設施使得泰－馬貿易迅速增長，合艾現在是宋卡府的第一大都市。

以下四個泰國的县名稱裡面雖然像府治縣一般名字裡有「{{lang|th|เมือง}}」（Mueang，[[勐|勐]]）這個詞，但她們並不是府治縣：
* [[孟汕|孟汕]]（''Mueang Chan''，在[[四色菊府|四色菊府]]，其府治縣為[[四色菊|四色菊]]） 
* [[孟盼|孟盼]]（''Mueang Pan''，在[[南邦府|南邦府]]，府治縣為[[南邦|南邦]]）
* [[孟雙|孟雙]]（''Mueang Suang''，在[[橫逸府|橫逸府]]，府治縣為[[橫逸|橫逸]]）
* [[孟揚|孟揚]]（''Mueang Yang''，在[[呵叻府|呵叻府]]，府治縣為[[那空拉差是瑪|那空拉差是瑪]]）

== 次級縣（''King Amphoe''） ==
在一些縣含有相對比較偏遠的地區，則劃分出次級縣（又譯次區、次郡、支郡）以方便本地群眾生活，次級縣雖然在級別上低于縣，但包括縣的幾乎全部功能。

行政上設立縣的標準是某地區至少包含5個區，人口達到3萬人。或者距離本郡行政署25公里以上，包含四個區，人口達到1.5萬人。

一旦次級縣達到獨立成縣的要求，一般被提昇為縣。有的時候新的行政分劃直接成為縣，而跳過次級縣的階段，如[[長島縣|長島縣]]在1903年至1988年僅是[[普吉府|普吉府]]的區，88年被直接提昇為縣。有的縣因為人口流失而被貶為次級縣，如[[沙敦府|沙敦府]]「''Thung Wa''」曾遭受很多人口流失至鄰近的區「''La Ngu''」，結果兩地的縣級別互換。另一個例子是前春蓬武里縣（''Chumphon Buri''），其較發達一部份獨立成次級縣後被提昇，而她則被貶為次級縣。

次級縣的長官稱為次級縣令（次郡令）。請注意次級縣中「King」僅僅是泰語音譯，同泰王並沒有直接的關係。

當前泰國總共有877個縣、81個次級縣，[[2007年5月|2007年5月]][[泰王國民政部|泰王國民政部]]決定將全國的次級縣都提昇為縣以方便行政管理。請看2007年8月24日王家檄文。

== 县的行政署 ==
每個县設有一間行政署，一般設立在本縣的人口中心。泰國公路的路標上的距離都是指到各縣行政署的距離。

縣行政署的主要功能為進行各類登記，如出生、死亡、婚姻登記等，這對于泰國人民的生活至關重要。
