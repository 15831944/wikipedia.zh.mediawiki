'''泰达街道'''{{NoteTag|有资料<ref>{{cite news |title=天津泰达街道办事处成立 |url=http://news.enorth.com.cn/system/2011/04/18/006383838.shtml |accessdate=2018-07-18 |work=人民网 |date=2011-04-18}}</ref><ref>{{cite web |title=泰达街道 |url=http://www.xzqh.org/html/show/tj/569.html |website=www.xzqh.org |publisher=行政区划网 |accessdate=2018-07-18}}</ref>显示2011年4月15日泰达街道已经成立，且辖区范围与2017年成立之泰达街道相同，现暂以[http://www.bh.gov.cn/jz/ 滨海新区人民政府街镇频道]的资料为准}}，是[[中华人民共和国|中华人民共和国]][[天津市|天津市]][[滨海新区|滨海新区]]下辖的一个[[乡镇|乡镇]]级行政单位，位于滨海新区中部，于2017年12月6日挂牌成立，首任街道党工委书记兼办事处主任由滨海新区副区长张国盛担任。<ref name="简介">{{cite web |title=泰达街街道简介 |url=http://www.bh.gov.cn/jz/channels/2398.html |publisher=天津市滨海新区人民政府 |accessdate=2018-07-18}}</ref><ref>{{cite web |title=泰达街道工委、办事处领导信息 |url=http://www.bh.gov.cn/contents/431/51023.html |publisher=天津市滨海新区人民政府 |accessdate=2018-07-19}}</ref>

泰达街道办事处由[[天津经济技术开发区|天津经济技术开发区]]（[[南港工业区|南港工业区]]）管委会所属社会管理职能部门整体剥离合并组成，负责承接开发区的社会管理与公共服务职能，同时加挂“天津市滨海新区人民政府泰达社会管理委员会”牌子，受滨海新区人民政府委托管理辖区教育、卫生等方面工作。<ref name="简介"/><ref>{{cite news |author1=天津开发区公寓管理中心 |title=泰达街工委、办事处、社会管理委员会今天正式挂牌 |url=http://www.sohu.com/a/208787983_100013522 |accessdate=2018-07-18 |work=搜狐财经 |date=2017-12-06}}</ref>

== 行政区划 ==
泰达街道辖区东至东海路，西至京山铁路，南至新港四号路，北至北塘镇，与[[天津经济技术开发区|天津经济技术开发区]]辖区范围相同。目前下辖11个社区居民委员会<ref name="简介"/><ref name="社区"/>{{noteTag|泰达街道成立前仍承担社会管理职能的开发区下辖23个社区<ref>{{cite web |title=2017年统计用区划代码和城乡划分代码：开发区 (天津市)|url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2017/12/01/16/120116400.html |publisher=[[中华人民共和国国家统计局|中华人民共和国国家统计局]]|year=2017 |accessdate=2018-07-18}}</ref>，包含泰达街道的所有社区，尚不清楚其他社区是否有调整}}：

翠亨社区、雅园社区、华纳社区、康翠社区、紫云社区、泰丰社区、芳林社区、时尚广场社区、福瑞社区、东海社区、盛清社区
