'''竇皇后'''（前205-前135年），[[中国|中国]][[西汉|西汉]]時期皇族女性，為[[汉文帝|汉文帝]][[皇后|皇后]]、[[漢景帝|漢景帝]]生母，出身[[清河郡|清河郡]][[观津|观津]]（今[[河北省|河北省]][[衡水市|衡水市]][[武邑縣|武邑縣]]審坡鎮）。儘管《[[史记|史记]]》和《[[汉书|汉书]]》均未提及她的名字，但東漢趙歧在《三輔決錄》有提到其名為“猗”<ref name="三輔決錄"/><ref name="趙歧傳">《[[後漢書|後漢書]]·卷六十四·吳延史盧趙列傳·趙歧》：「（趙）歧多所述作，著《孟子章句》、《三輔決錄》傳於時。」</ref>。另外，唐代[[司马贞|司马贞]]撰写的《[[史记索隐|史记索隐]]》中，提到[[西晋|西晋]][[皇甫谧|皇甫谧]]称窦皇后的名字为“猗房”<ref>史記索隱：「......[[皇甫谧|皇甫谧]]-{云}-名猗房......」[http://www.guoxue.com/shibu/24shi/shiji/sj_049.htm 司马贞注释《史记 外戚世家第十九》]</ref>。竇氏掌握政權橫跨景帝、武帝兩代，主張[[黃老之術|黃老之術]]，成為西漢初年[[道家|道家]]「[[無為而治|無為而治]]」的忠實執行者及維護者。雖然竇氏不如[[呂雉|呂雉]]、[[衛子夫|衛子夫]]般為人熟悉，但對西漢的政治影響深遠。

== 早年 ==
窦氏生年不详。吕雉為[[皇太后|皇太后]]時（前195年—前180年），窦氏以[[家人子|家人子]]的身份被选入宫服侍呂雉。之後，吕雉将一批宫女送出宫，赐予[[诸侯王|诸侯王]]，每王五名，窦氏就在其中。窦姬因家乡在清河郡，于是請求主管此事的[[宦官|宦官]]將她分去赵国。宦官答應，卻粗心大意，把竇氏排入前往[[代_(西汉)|代国]]的队伍中，竇氏不得已来到代国，卻得到了代王[[汉文帝|刘恒]]的寵幸。窦氏先生一女[[刘嫖|刘嫖]]，后生[[汉景帝|刘启]]、[[劉武_(梁王)|刘武]]等二子。

代王尚未入朝为帝时，[[代王王后|代王王后]]早已亡故。[[陈平_(西汉)|陳平]]、[[周勃|周勃]]等[[元老|元老]][[誅滅諸呂|誅滅諸呂]]，迎立代王劉恆為漢文帝後，代王王后所生的四子也相繼逝世。[[前元#汉文帝|前元]]元年（前179年），刘启以庶长子身分被冊立為[[皇太子|皇太子]]。同年三月，群臣请立皇后。文帝之母[[薄姬|薄太后]]說：“诸侯皆同姓，立太子母为皇后。”窦氏因而被冊立為皇后，女儿刘嫖封为馆陶公主。次年，幼子刘武被立为[[代王|代王]]，不久改迁[[梁国|梁国]]，改封为[[梁王|梁王]]。

== 家庭背景 ==
窦氏的父母早亡，安葬在[[观津|观津]]（現河北省重點文物保護單位：[[竇氏青山|竇氏青山]]墓）。薄太后下诏追封窦皇后的父亲为安成侯，母亲为安成夫人。并且在清河设置二百户的园邑，由长丞侍奉看守，一切按薄太后父親灵文侯之墓——靈文園的做法。

窦皇后有兄弟二人，兄長为[[窦长君|窦长君]]，弟弟为[[窦少君|窦少君]]，在竇氏四五岁时，因家境贫困，兄弟二人皆被掳走並[[人口贩卖|贩卖]]到外地，成為[[奴工|奴工]]，渺无音讯。後又被转卖了十几户人家，最後到了宜阳（今[[河南|河南]][[宜阳|宜阳]]）進山挖[[石炭|石炭]]。一天黄昏，山崖边有一百多人在睡觉，突然[[山崩|山崩]]，睡在崖边的人都压死了，只有[[竇少君|竇少君]]脱险逃生。不久，窦少君跟随主人到了[[京師|京師]]，听说新皇后姓窦，[[原籍|原籍]]在[[观津|观津]]，窦少君認為這就是他的姊姊。竇少君离家的时候虽然年纪幼小，却记得自己的[[籍贯|籍贯]]和姓氏，还隐约记得与姊姊一起採[[桑叶|桑叶]]並从树上摔下来的情景。他把这些事详细地写下来後，託人转交给窦皇后。窦皇后见到了这一封信後大驚，即刻將窦少君召来，并详细问了其他一些情况，姊弟就此相認。竇皇后还要窦少君回忆一些过去的情景，少君回忆道：「姊姊离我西去，我记得在[[驿站|驿站]]分别时，讨来米汤给我洗头，临走时又给我吃了饭才走的。」窦皇后泣不成声，緊握著窦少君的手。

窦皇后重赏两个兄弟，把他们安置在[[長安城|長安城]]居住。后来又为他俩请了有德行的长者对他们进行教育。由于这样，窦长君、窦少君兄弟俩后来成为谦让有礼的君子，不因地位显贵而盛气凌人。

竇氏成為皇太后時，[[漢景帝|漢景帝]]封其弟[[窦少君|窦少君]]为章武侯，其兄窦长君早逝，封他的兒子[[窦彭祖|窦彭祖]]为南皮侯；因平定[[七國之亂|七國之亂]]其堂侄[[窦婴|窦婴]]被封为魏其侯。

== 皇后生涯 ==
窦皇后因病而双目失明。同时，文帝有新宠[[慎夫人|慎夫人]]和[[尹姬|尹姬]]。其中慎夫人甚至可以与窦皇后、汉文帝同席而坐，直到[[袁盎|袁盎]]以「當年[[戚夫人|戚夫人]]得寵後被呂太后所虐殺」為由勸阻，才讓慎夫人坐於下方<ref>《[[史记|史记]] 卷一百一 袁盎晁错列传第四十一》上幸上林，皇后、慎夫人从。其在禁中，常同席坐。及坐，郎署长布席，袁盎引卻慎夫人坐。慎夫人怒，不肯坐。上亦怒，起，入禁中。盎因前说曰：“臣闻尊卑有序则上下和。今陛下既已立-{后}-，慎夫人乃妾，妾主岂可与同坐哉！適所以失尊卑矣。且陛下幸之，即厚赐之。陛下所以为慎夫人，適所以祸之。陛下独不见‘人彘’乎？”於是上乃说，召语慎夫人。慎夫人赐盎金五十斤。</ref>。但由于慎夫人與尹姬都没有儿子，对窦皇后和刘启的地位无法构成威胁。

== 皇太后时期 ==
汉文帝去世后，太子刘启即位，是为景帝。窦皇后被尊为[[皇太后|皇太后]]。窦太后尊崇[[道家|道家]]，因此汉景帝与窦氏宗族都必须学习[[黄老|黄老]]学说。由於竇太后的影响，在汉景帝在位期间与[[汉武帝|汉武帝]]早期，[[儒家|儒家]]学者都無法得到重用。

窦太后命景帝和窦姓[[外戚|外戚]]宗族得读[[老子|老子]]的《[[道德經|道德經]]》，并推尊其学说，因此她在世时“故诸博士具官待问，未有进者”（《史记·儒林传》）。

景帝时，她曾召[[博士|博士]][[辕固生|辕固生]]问他《[[道德經|道德經]]》是怎样的一部书，辕固生不识时务，猝然答道：“这不过是部平常人家读的书。”窦太后大怒道：“難道一定要讀『司空城旦书』（[[刑法|刑法]]之類的书）吗？”（讥讽[[儒教_(宗教)|儒教]]嚴苛，類似於[[司空|司空]]之類的[[法官|法官]]，[[城旦|城旦]]之類的[[刑罰|刑罰]]。）太后憤怒，命辕固生到猪圈裏殺猪。[[汉景帝|汉景帝]]见辕固生为一文弱书生，恐不敌猪，於是給他一把利刃，才让辕固生把猪刺死。

景帝在位十六年，始终未用[[儒生|儒生]]。通常认为[[文景之治|文景之治]]的主角是汉文帝与汉景帝。实际上，竇太后在[[文景之治|文景之治]]时代所起的作用也是举足轻重的。在汉景帝统治时期，汉景帝曾经想启用[[儒家|儒家]]学者，但都被其阻止。

竇太后极其溺爱自己的小儿子[[劉武_(梁王)|刘武]]。不但赏赐无数，而且还希望他能在景帝去世后继位。前154年，汉景帝曾在宴会上对梁王刘武表示，愿在自己死后传位与他，但遭到竇婴等朝臣的反对。尽管后来窦太后对景帝多次施加压力，但终未成功，刘武更是在景帝生前即去世。

== 太皇太后時期 ==
前141年，[[漢景帝|漢景帝]][[劉啟|劉啟]][[駕崩|駕崩]]，[[太子|太子]][[劉徹|劉徹]]即位，是為[[漢武帝|漢武帝]]，奉生母[[王皇后_(汉景帝)|皇后王氏]]為[[皇太后|皇太后]]，尊[[祖母|祖母]]竇太后為[[太皇太后|太皇太后]]。

竇太皇太后闻武帝好[[儒學|儒學]]，非常不以為然，常出面干预朝政。武帝也不便违忤祖母，所有朝廷政事，都随时向她请示。当时[[御史大夫|御史大夫]][[赵绾|赵绾]]和[[郎中令|郎中令]][[王臧|王臧]]，迎[[鲁國|鲁國]]耆儒[[申公|申公]]来朝，并建议仿古制，设[[明堂|明堂]]辟雍，改曆易服，行[[巡狩|巡狩]][[封禅|封禅]]等礼仪，还建议政事“不必事事请命太皇太后”。太皇太后听罢，怒不可遏，命武帝下令革去赵绾、王臧官职。至她去世前，武帝都不重用[[儒生|儒生]]，可见她在政治上的影响。

[[建元_(西漢)|建元]]六年（前135年），竇太皇[[太后|太后]]去世，遗[[诏|诏]]將[[東宮|東宮]]（[[長樂宮|長樂宮]]）的金钱财物全部赐给女儿[[刘嫖|刘嫖]]。她与[[汉文帝|汉文帝]]合葬在[[灞陵_(漢)|灞陵]]。

== 历史评价 ==
窦氏干预朝政，推崇[[道家|道家]]，自她之后，没有一位[[中国|中国]]的统治者能像她一样真正的以「[[黄老|黄老]]思想」来「[[无为而治|无为而治]]」。有人说其扮演了一个逆历史潮流而动的角色。但[[文景之治|文景之治]]政局稳定海晏河清，宽政待民，窦氏经历[[漢文帝|漢文帝]]、[[漢景帝|漢景帝]]二朝，史称“[[文景之治|文景之治]]”的盛世，与推行[[黄老|黄老]]之术的宽民政策有很大关系。后[[汉武帝|漢武]][[罢黜百家|罢黜百家]]，[[独尊儒术|独尊儒术]]，虽布威名于四方，却将[[汉朝|汉朝]]折腾得千疮百孔筋疲力尽。由此可见窦氏之功过不可一概而论。

== 陵墓被盗事件 ==
2001年10月，窦氏的陵墓被盗，大量陪葬[[陶俑|陶俑]]被盗墓贼出售，其中6件被偷运出境，并且于2002年3月20日在[[美國|美國]][[纽约|纽约]][[蘇富比|蘇富比]]拍卖。[[中华人民共和国国家文物局|中国国家文物局]]及[[中华人民共和国外交部|外交部]]闻讯后与美方进行了大量交涉，蘇富比才撤銷拍賣，并且于2003年6月将这6件陶俑归还中国。
