'''孝莊文皇后'''（{{lang-mnc|ᡥᡳᠶᠣᠣᡧᡠᠩᡤᠠ<br /> ᠠᠮᠪᠠᠯᡳᠩᡤᡡ<br /> ᡤᡝᠩᡤᡳᠶᡝᠨ<br /> ᡧᡠ<br /> ᡥᡡᠸᠠᠩᡥᡝᠣ|v=hiyoošungga ambalinggū genggiyen šu hūwangheo|a=hiyouxungga ambalinggv genggiyen xu hvwangheu}}；{{bd|1613年|3月28日|1688年|1月27日||catIdx=B,博}}），[[博尔济吉特氏|博尔济吉特氏]]（{{mong|m=ᠪᠣᠷᠵᠢᠭᠢᠨ|Ppp=Borǰigin}}，即[[孛兒只斤氏|孛兒只斤氏]]），名'''布木布泰'''（{{lang-mnc|ᠪᡠᠮᠪᡠᡨᠠᡳ|v=bumbutai|a=bumbutai}}），亦作'''本布泰'''。[[皇太极|清太宗皇太极]]之妃，[[孝端文皇后|孝端文皇后]]的[[姪女|姪女]]，[[顺治帝|顺治帝]]的生母。本为蒙古[[科尔沁部|科尔沁部]][[多罗贝勒|贝勒]][[寨桑_(人物)|寨桑]]和[[博禮|博禮]]之女。

== 生平 ==

[[File:《孝庄文皇后常服像》局部.jpg|thumb]]

布木布泰（本布泰）生于[[万历|万历]]四十一年[[二月初八|二月初八]]日（1613年3月28日）。后金[[天命_(后金)|天命]]十年（1625年）二月，在哥哥[[吴克善|吴克善]]的护送下，年仅十三岁的布木布泰嫁给大汗[[努尔哈赤|努尔哈赤]]第八子[[皇太极|皇太极]]为妻。皇太极亲迎至沈阳北冈，宴之。将至时，大汗努尔哈赤与诸贝勒、后妃等出迎十里，大宴入城。入城后又复设宴，完成婚礼<ref>《[[满洲实录|满洲实录]]·卷八》○天命十年正月至十一年八月……○二月.科爾沁宰桑貝勒子武克善台吉送其妹來與四王為妃.四王迎至瀋陽北岡宴之.將至.帝與諸王及后妃等出迎十里.大宴入城.復設宴.以禮成配.因其送婚遂優待之.賜以人口.金銀.蟒段.布帛.盔甲.銀器等物.令之還……</ref>。当时，后金贵族奉行[[一夫多妻多妾制|一夫多妻多妾制]]，诸妻皆可称[[福晋|福晋]]。此前，皇太极已经娶她的姑姑[[孝端文皇后|哲哲]]为福晋。

天聪八年（1634年) ，她的姊姊[[敏惠恭和元妃|海兰珠]]再嫁皇太极，形成姑侄三人同嫁一夫的婚姻。[[崇德|崇德]]元年（1636年），皇太极[[即位|称帝]]，姑姑哲哲由正宫大福金受封为国主福金，姐姐海兰珠由关雎宫福晋册封为东大福金，即宸妃。而布木布泰本人由'''西永福宫福晋'''（wargi hūturingga booi fujin）册封为'''西侧福金'''（wargi ashan i fujin）<ref>{{Cite web |url=http://www.sohu.com/a/167862409_610385 |title=条条大路通罗马：清代后宫的等级与来源 |accessdate=2019-03-14 |author= |date=2017-08-28 |work= |publisher=搜狐网|language=Zh-hans }}</ref>，汉文称'''莊妃'''。

崇德三年（1638年），布木布泰生下皇九子[[顺治帝|福临]]。崇德八年（1643年），丈夫皇太极逝世，生前未指定[[储君|太子]]，引起[[多爾袞|多爾袞]]、[[豪格|豪格]]等人对皇位的争夺。多方角逐之下，其子福临因布木布泰地位较高、自己年长{{efn|皇太极逝世时，除[[豪格|豪格]]外，尚有[[葉布舒|叶布舒]]、[[硕塞|硕塞]]、[[高塞|高塞]]、[[常舒|常舒]]、[[韬塞|韬塞]]、福临、[[博穆博果尔|博穆博果尔]]七子。在叶布舒、高塞、常舒、韬塞四人中，有三人年长于福临，但皆生母低微，无法越过福临、博穆博果尔继承皇位。而硕塞的生母[[福晋叶赫那拉氏|叶赫那拉氏]]则早被皇太极赐给大臣，博穆博果尔生母[[懿靖大貴妃|娜木鐘]]虽与布木布泰地位相当，但其年幼于福临。}}的缘故而被拥立为帝。

其子福临[[即位|即位]]后，年号[[顺治|顺治]]。时由多尔衮主政，布木布泰被稱為「'''皇母庄妃'''」或「'''圣母'''」。[[乾隆|乾隆]]朝为完善制度等原因而修改为崇德八年太宗文皇帝崩逝，世祖即位即与孝端文皇后两宫并尊。在顺治十七年，其子顺治帝要求[[清朝礼部|礼部]][[追封|追封]]乳母[[李氏_(顺治帝)|李氏]]的诏书<ref>《[[清实录|清实录]]·世祖实录·卷之一百四十三》（顺治十七年。庚子。十二月）○乙巳。谕礼部。乳母李氏、当朕诞毓之年。入宫抚哺。尽心奉侍。进食、必饥饱适宜。尚衣、必寒温应候。啼笑之间、曲意调和。期于中节。言动之际、相机善导。务合规程。诸凡襁褓殷勤、无不周详恳挚。睿王摄政时、皇太后与朕分宫而居。每经累月、方得一见。以致皇太后萦怀弥切。乳母竭尽心力。多方保护诱掖。皇太后惓念慈衷、赖以宽慰。即读书明理者、未必过是。此其贤德、今昔罕闻。乃一旦溘然长逝。深堪悯悼。追封恩恤、宜从优厚。尔部即详察典例、确议具奏。</ref>中，指多爾袞[[摄政|摄政]]时，“皇太后与朕分宫而居。每经累月、方得一见。以致皇太后萦怀弥切。乳母（李氏）竭尽心力。多方保护诱掖。皇太后惓念慈衷、赖以宽慰。”

多尔衮在顺治七年年底逝世。[[順治|順治]]八年正月，儿子顺治帝正式[[亲政|亲政]]。二月，为她始上[[尊号|尊号]]，是为'''昭圣慈寿皇太后'''；同年八月，儿子顺治帝与侄女[[额尔德尼布木巴|额尔德尼布木巴]]的[[清帝大婚|大婚]]礼成，为皇太后再加尊号，为'''昭圣慈寿恭简皇太后'''<ref>《[[清史稿|清史稿]]·卷五·本纪五》◎世祖本纪二……（八年春正月）庚申，上亲政，御殿受贺，大赦。诏曰：“朕躬亲大政……（二月）戊子，上昭圣慈寿皇太后尊号。己丑，大赦……（八月）戊午，册立科尔沁卓礼克图亲王吴克善女博尔济锦氏为皇后。壬戌，更定马步军经制。吏部尚书谭泰有罪，伏诛，籍其家。乙酉，大婚礼成，加上太后尊号为昭圣慈寿恭简皇太后。丙寅，御殿受贺，颁恩赦……</ref>。[[顺治|顺治]]十年七月十三日，順治帝奉[[皇太后|皇太后]]面谕：「予居深宫之中，不闻外事。近知雨潦为灾，房舍倾颓，田禾淹没，兵民困苦，深可悯恻，特发宫中节省银八万两，赈济满汉兵民。」顺治十一年二月二十五日，昭圣慈寿恭简皇太后聽聞某地方發生灾荒，下旨宫中节省费用并各项器皿，共银四万两。[[顺治|顺治]]十三年八月十二日，皇太后得悉畿辅近地连年荒歉，民生艰瘁。因此，下令宫中要节省银三万两，以赈济災民<ref>《[[清实录|清实录]]·顺治朝实录》：朕亲诣慈宁宫，朝见皇太后。奏知畿辅近地连年荒歉。今岁自夏徂秋，复苦䨙雨飞蝗，民生艰瘁。蒙皇太后慈谕:小民如此苦楚，深为可悯，所有宫中节省银三万两，即行发出，速加赈济。</ref>。

[[顺治|顺治]]十一年，追封皇太后的父亲寨桑为和硕忠亲王，母亲为和硕忠亲王[[贤妃|贤妃]]。[[順治|順治]]十三年十二月加尊号为'''昭圣慈寿恭简安懿章庆皇太后'''，宮廷亦稱其為'''圣母皇太后'''。[[顺治|顺治]]十八年（1661年），顺治帝死后，第三子[[康熙帝|玄烨]]即位（年号[[康熙|康熙]]），布木布泰被尊为[[太皇太后|太皇太后]]，徽號：昭聖[[太皇太后|太皇太后]]。[[康熙|康熙]]元年十月，加上尊号为'''昭圣慈寿恭简安懿章庆敦惠太皇太后'''。[[康熙|康熙]]四年九月为'''昭圣慈寿恭简安懿章庆敦惠温莊太皇太后'''。

[[康熙|康熙]]六年（1667年)，太皇太后曾主持写造泥金土伯特文《甘珠尔经》的工程。然而，[[鳌拜|鳌拜]]、[[遏必隆|遏必隆]]等辅政大臣势力尚存，故而这件事遭到以辅臣为首的朝臣阻挠。太皇太後力排众议、屡下懿旨<ref>《清代内务府档案》：太皇太后懿旨：尔等此番估奏，非欲令朕惊惧息意否？朕断然不退！著将朕岁得貂皮、绸缎、丝绒等项，估算后拿来造佛经。毋将此事奏闻于皇上。钦此。</ref>。许多外藩公主、[[格格|格格]]都响应太皇太后，主动参与並资助这项工程。

康熙六年十一月为'''昭圣慈寿恭简安懿章庆敦惠温-{zh:庄; zh-hans:庄; zh-hant:莊;}-康和太皇太后'''。[[康熙|康熙]]十五年正月为'''昭圣慈寿恭简安懿章庆敦惠温-{zh:庄; zh-hans:庄; zh-hant:莊;}-康和仁宣太皇太后'''。[[康熙|康熙]]二十年十二月为'''昭圣慈寿恭简安懿章庆敦惠温-{zh:庄; zh-hans:庄; zh-hant:莊;}-康和仁宣弘靖太皇太后'''。

康熙二十六年十二月二十五日（1688年1月27日），昭聖[[太皇太后|太皇太后]]崩逝於[[北京市|北京]][[故宫|紫禁城]][[慈寧宮|慈寧宮]]內，享壽七十五岁。病危时，布木布泰对康熙帝说：“太宗奉安久，不可为我轻动。况我心恋汝父子，当於孝陵近地安厝，我心始无憾。”于是康熙帝将布木布泰灵柩暂安于父亲顺治帝葬所[[清孝陵|孝陵]]附近。

康熙二十七年三月二十七日，康熙帝诣[[壽安宮|咸安宮]]请安原需经过[[慈宁宫|慈宁宫]]，但由于[[太皇太后|太皇太后]]过世不久，康熙帝看到慈宁宫都会伤心，所以改从启祥门行走<ref>《康熙朝實錄》：康熙二十七年三月二十七日，上谕：从前诣两宫请安，皆于起居注记档。今诣“宁寿宫”请安，朕因不忍过慈宁宫，故从启祥门行走。但此系宫禁之地，外官无由得知。此后每次请安著令太监传谕敦住，仍令起居注官记载，其不忍由隆宗门行走之故亦令谕侍郎库勒纳知之。</ref>。同年十月上[[諡號|諡號]]为'''孝莊仁宣诚宪恭懿翊天启圣文皇后'''。[[雍正帝|世宗]][[雍正|雍正]]元年（1723年）八月为'''孝莊仁宣诚宪恭懿至德翊天启圣文皇后'''。[[雍正|雍正]]三年（1725年）十二月，[[雍正帝|雍正帝]]在布木布泰灵柩暂安之处就地起陵，是为[[昭西陵|昭西陵]]。[[乾隆|乾隆]]元年（1736年）三月上諡號為'''孝莊仁宣诚宪恭懿至德纯徽翊天启圣文皇后'''。

== 评价 ==

孝莊文皇后先后辅佐前清三代帝王，对满族的入关，灭[[明帝國|明帝國]]和巩固对汉族的统治發揮极大的作用。又禮遇漢人，吸取金、蒙古之教訓，並對西方知識極為尊重，對康熙帝起極大之啟蒙作用。另外一方面，孝莊經常被歷史學家拿來跟滿清末年的[[慈禧太后|慈禧太后]]來相互比較。她的手段高明、才能出眾，但卻不戀棧權位，對少年康熙的謀策頗有指點，但從未越俎代庖；為清朝國力打下基礎。

=== 确保福临即位 ===

皇太极生前未指定继承人，遂出现争夺最高领导权的纷争。孝莊以灵活巧妙的手段，使[[福临|福临]]继位，避免[[多爾袞|多爾袞]]、[[豪格|豪格]]兩強相爭帶來的災害。

=== 下嫁多尔衮 ===
《中央日报--文史周刊》民國三十六年一月二十八日，《清初皇父摄政王多尔衮起居注跋》：「清季，宣统改元，内阁库垣圮。时家君方任阁读，奉朝命检库藏。既得顺治时太后下嫁皇父摄政王诏，摄政王致史可法、唐通、马科书稿等，遂以闻于朝，……明又于起居注档上，见有《皇父摄政王多尔衮起居注》一册，黄绫装背，面钤弘文院印。」

刘文兴之父錄有《皇父摄政王多尔衮起居注》副本，目前藏於四川师范学院图书馆，其內容亦可見故宫博物馆出版《文献丛书》第十五种之《多尔衮摄政日记》<ref>《文献丛书》：「《多尔衮摄政日记》为清内阁大库旧物，宣统间清理库档流落于外，后归宝应刘氏食旧德斋，原册起五月二十九日迄七月初九日而不纪年，兹因有闰六月，检勘历书及实录知为顺治二年事。原书初无名称。每日记事后均书记者衔名与清代起居注体例略同，故刘氏于其所录副册题曰《摄政王多尔衮起居注》。考清代起居注，康熙间始置馆，时尚无其制，今刘氏嘱由本院刊行爰改题曰《多尔衮摄政日记》。」</ref>其史料經常被後世史學家引用。

一些野史中记录了孝莊太后下嫁和多尔衮通姦之事。[[南明|南明]]反清之士張煌言曾作詩譏諷：「上壽觴為合卺樽，慈寧宮裏爛迎門，春官昨進新儀注，大禮恭逢太后婚。」諷刺給太后的壽酒變作了喜酒，不少稗官野史以此為據捕風捉影，然而真實度低。

清史學者[[孟森|孟森]]曾著有〈太后下嫁考實〉一文，認為下嫁一事乃子虛烏有，但並未完全解釋何以順治稱呼攝政王多爾袞為「叔父皇阿瑪」? [[胡適|胡適]]亦對此表示懷疑；學者陳捷先則認為從滿文的檔案來看，親屬稱謂必然置於人名之前，在人名後加上親屬稱謂可能是類似官職的美稱；學者鄭天挺則就蓋有[[順治|順治]]印璽文書來看，上面顯示為「叔父攝政父王」，既然已稱叔父(ecike)何以又重複阿瑪(ama)?可見父(ama)在此是一種對地位崇高之人的尊稱，而非真實親屬關係<ref>{{cite book|title=明清史|last=陳捷先|publisher=三民書局|isbn=978-957-14-3957-0|location=台北|page=194}}</ref>。

电视[[纪录片|纪录片]]《清宫秘档》中的观点认为，莊妃下嫁多尔衮不是事实。另如光緒亦曾稱慈禧太后為「皇爸爸」，可知此類稱謂僅為尊稱。亦有學者認為確有其事，認為[[夫兄弟婚|弟娶寡嫂]]的收繼婚形式，符合當時滿人風俗，後因受漢人文化影響才認為是羞恥之事並加以隱瞞，但此說僅是猜測，與可信史料不符，故學者仍多以正史記載，斥太后下嫁為假，是漢人為抹黑滿清而造的謠言。

== 艺术形象 ==

野史、小说或影视剧中多有设定孝莊文皇后的名字為大玉兒。她的兄弟[[索诺木_(科尔沁部)|索诺木]]之女、[[多爾袞|多爾袞]]之妻——[[敬孝義皇后|博爾濟吉特氏]]则演绛为小玉儿，亦或设定为她的族姐妹。而真实的本名布木布泰，则少见使用。{{or|部分漢人（尤其是較南方的漢人），在書籍上稱其為「大玉兒」，遂被後世的人延用。}}
