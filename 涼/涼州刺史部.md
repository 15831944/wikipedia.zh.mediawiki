'''涼州刺史部'''，原屬九州中的雍州地域<ref>《晉書卷十四·志第四·地理上·涼州》：「涼州。案禹貢雍州之西界，周衰，其地為狄。秦興美陽甘泉宮，本匈奴鑄金人祭天之處。匈奴既失甘泉，又使休屠、渾邪王等居涼州之地。二王後以地降漢，漢置張掖、酒泉、敦煌、武威郡。其後又置金城郡，謂之河西五郡。漢改周之雍州為涼州，蓋以地處西方，常寒涼也。」</ref>。始設於[[西漢|西漢]]，其行政長官為'''[[涼州|涼州]][[刺史|刺史]]'''，屬漢代十三[[刺史部|刺史部]]之一。因在[[中原|中原]]之西，故又稱'''西州'''。其管轄範圍在今[[甘肅省|甘肅省]]及[[寧夏|寧夏]]全境、[[青海|青海]]東北部、[[新疆|新疆]]東南部及[[內蒙古|內蒙古]][[阿拉善盟|阿拉善盟]]一帶。

== 西漢涼州刺史部 ==
西漢時，轄域約為今[[甘肅|甘肅]]、[[寧夏|寧夏]]、[[青海|青海]]三省區[[湟水|湟水]]流域，[[陝西省|陝西省]][[定邊|定邊]]、[[吳旗|吳旗]]、[[鳳縣|鳳縣]]、[[略陽|略陽]]等縣和[[內蒙古自治區|內蒙古自治區]][[額濟納旗|額濟納旗]]等地。

=== 郡級行政區 ===
秦代時在隴右一帶設置了隴西、北地2郡；[[西漢|西漢]][[元狩|元狩]]二年（前121年），[[漢武帝|漢武帝]]派驃騎大將軍[[霍去病|霍去病]]遠征河西，擊敗[[匈奴|匈奴]]，開闢了河西地，置酒泉郡。元鼎三年（前114年）置安定郡、天水郡。元鼎六年（前111年）置張掖郡、敦煌郡。[[元封|元封]]五年（前106年），分天下為十三州，各置一刺史，史稱「十三部刺史」。在今甘肅省置涼州刺史部，下轄隴西、天水、安定、酒泉、張掖、敦煌6郡；始元六年（前81年）置金城郡；地節三年（前67年）置武威郡；元始四年（公元4年）置西海郡。西漢末計轄9郡。

; [[隴西郡|隴西郡]]
:秦朝舊郡，秦末可考領縣為狄道、西、蘭干、略陽、上邽、冀、邸道、故道、臨洮、獂道、綿諸、襄武、阿陽、下辨、辨道、戎道、武都道、予道、薄道、成紀、枹罕21縣{{refTag|1=后曉榮《秦代政區地理》，第149-159頁。}}。
:漢時治狄道縣，繼承秦代領縣。西漢時概增置安故、首陽、大夏、羌道、白石等縣；元鼎三年（前114年），蘭干、略陽、冀、獂道、綿諸、阿陽、戎道、成紀等縣移屬天水郡；元鼎六年（前111年），故道、下辨道、武都道等縣移屬武都郡；始元六年（前81年），白石、枹罕2縣移屬金城郡。西漢末領狄道、上邽、氐道、予道、襄武、臨洮、西，安故、首陽、大夏、羌道11縣。

; [[天水郡|天水郡]]
:元鼎三年（前114年）分隴西郡蘭干、略陽、冀、獂道、綿諸、阿陽、戎道、成紀等縣置，治平襄縣。西漢末領平襄、街泉、戎邑道、望垣、罕幵、緜諸道、阿陽、略陽道、冀、勇士、成紀、清水、奉捷、隴、豲道、蘭干16縣。

; [[安定郡|安定郡]]
:元鼎三年（前114年）分[[北地郡|北地郡]]置，郡治在高平縣（今寧夏固原）。西漢末領高平、復累、安俾、撫夷、朝那、涇陽、[[臨涇|臨涇]]、鹵、烏氏、陰密、安定、參欒、三水、陰槃、安武、祖厲、爰得、眴卷、彭陽、鶉陰、月氏道21縣。

; [[酒泉郡|酒泉郡]]
:元狩二年（前121年）所置{{refTag|name=河西4郡|1=周振鶴《西漢政區地理》，第168頁。}}。治祿福縣，西漢末領祿福、表是、樂涫、天衣、玉門、會水、池頭、綏彌、乾齊9縣。

; [[張掖郡|張掖郡]]
:元鼎六年（前111年）分酒泉郡置{{refTag|name=河西4郡}}。治觻得縣，西漢末領觻得、昭武、刪丹、氐池、屋蘭、日勒、驪靬、番和、居延、顯美10縣。

; [[敦煌郡|敦煌郡]]
:元鼎六年（前111年）分酒泉郡置{{refTag|name=河西4郡}}。治敦煌縣，西漢末領敦煌、冥安、效穀、淵泉、廣至、龍勒6縣。

; [[武威郡|武威郡]]
:地節三年（前67年）置{{refTag|name=河西4郡}}。治故臧縣，西漢末領姑臧、張掖、武威、休屠、揟次、鸞烏、撲褱、媼圍、蒼松、宣威10縣。

; [[金城郡|金城郡]]
:始元六年（前81年）分天水郡榆中、金城2縣，隴西郡白石、枹罕2縣，張掖郡令居、枝陽2縣合置{{refTag|1=《漢書》卷7〈昭帝紀〉：「始元六年，以邊塞闊遠，取天水、隴西、張掖郡各二縣置金城郡。」}}{{refTag|1=周振鶴《西漢政區地理》，第173頁。}}之後逐步向西增置臨羌、浩亹2縣。[[神爵|神爵]]二年（前61年），置允街、破羌、河關3縣，並置'''金城屬國'''管理內附羌人。治允吾縣，西漢末領允吾、浩亹、令居、枝陽、金城、榆中、枹罕、白石、河關、破羌、允街、安夷、臨羌13縣。

; [[西海郡|西海郡]]
:[[元始|元始]]四年（4年）置，治[[龍夷城|龍夷城]]。領龍夷等5縣。{{refTag|1=《元和郡縣圖志》卷39〈隴右道上〉：「平帝元始四年，金城塞外羌獻魚鹽之地內屬，漢遂得西王母石室，以為西海郡，理龍夷城。」；《後漢書》卷87〈西羌傳〉：「至王莽輔政，欲燿威德，以懷遠為名，乃令譯諷旨諸羌，使共獻西海之地，初開以為郡，築五縣。」}}

=== 縣級行政區 ===
涼州地區在秦末可考縣數，計隴西郡21縣{{noteTag|后曉榮考証為：狄道、西縣、蘭干、略陽、上邽、冀縣、邸道、故道、臨洮、獂道、綿諸、襄武、阿陽、下辨、辨道、戎道、武都道、予道、薄道、成紀、枹罕等21縣{{refTag|1=后曉榮《秦代政區地理》，第149-159頁}}。}}。隨著西漢開擴河西，大量增置縣份。西漢末計轄101縣。

東漢時，轄域等同於西漢。[[漢獻帝|漢獻帝]][[建安|建安]]初年，分涼州河西的酒泉、張掖、敦煌、武威、張掖居延屬國5郡置[[雍州_(古代)|雍州刺史]]，轄域縮小至僅存今甘肅省黃河以東、寧夏南部的地區。建安十八年（213年）省涼州入雍州。治所隴縣（今[[甘肅省|甘肅省]][[張家川回族自治縣|張家川回族自治縣]]境内），直到[[漢獻帝|漢獻帝]]時，才遷治到天水郡冀縣（今[[甘肅省|甘肅省]][[甘谷縣|甘谷縣]]）。

=== 郡級行政區 ===
[[東漢|東漢]]時，復改[[雍州|雍州]]為[[涼州|涼州]]。增置[[張掖屬國|張掖屬國]]、[[張掖居延屬國|張掖居延屬國]]（即[[西海郡|西海郡]]），并領[[武都郡|武都郡]]。[[永和_(東漢)|永和]]年間，計有12郡國，98縣。[[漢獻帝|漢獻帝]]、[[漢獻帝|漢獻帝]]時期增置[[南安郡|南安郡]]（188年置）、[[永陽郡|永陽郡]]（193年置）[[新平郡|新平郡]]（194年置）。[[建安|建安]]初，分涼州河西的酒泉、張掖、敦煌、武威、張掖居延屬國5郡置[[雍州_(古代)|雍州刺史]]。建安十八年（213年）省涼州入雍州。

; [[隴西郡|隴西郡]]（25年 - 220年）
:郡治狄道縣，因羌亂，永初五年（111年）三月，郡治由狄道縣內徙到襄武縣，延光三年（124年）還治狄道縣。初領狄道、安故、氐道、首陽、大夏、襄武、臨洮、上邽、西、羌道、予道11縣。東漢初省予道縣{{noteTag|name=東漢初省縣|李曉杰據《後漢書·光武帝紀下》「（建武六年六月），并省四百餘縣」，認為該縣當於東漢初年省併無疑{{refTag|1=李曉杰《東漢政區地理》中的各郡沿革}}。}}。建武十二年（36年）省金城郡，其枹罕、白石、河關等13縣屬隴西郡<ref group="古">《續漢志》卷23〈郡國五〉：「枹罕，故屬金城；白石，故屬金城；河關，故屬金城。」</ref>，次年（37年）以允吾、浩亹、令居等10縣復置金城郡。永元元年（89年）前置鄣縣{{refTag|name=鄣縣}}。永初五年（111年）以前，上邽、西2縣移屬漢陽郡，羌道縣移屬武都郡{{refTag|name=隴西3縣|1=李曉杰《東漢政區地理》，第143頁。}}。建安十八年（213年）以後歸屬雍州管轄。建安末因亂而廢白石縣。東漢末，領狄道、安故、氐道、首陽、大夏、襄武、臨洮、鄣、枹罕、河關10縣。

; [[天水郡|天水郡]]（25年 - 74年）→[[漢陽郡|漢陽郡]]（74年－220年）
:本天水郡，永平十七年（74年）更名為漢陽郡<ref group="古">《後漢書》卷2〈顯宗明帝紀〉：「是歲（永平十七年），改天水為漢陽。」；《續漢志》卷23〈郡國五〉：「漢陽郡，武帝置，為天水，永平十七年更名。」</ref>，郡治冀縣。初領冀、望恒（漢書作望垣）、阿陽、略陽（《漢書》作略陽道）、勇士、成紀、隴、蘭干、平襄、豲道、街泉、清水、奉捷、罕幵、戎邑道、緜諸道16縣。東漢初省街泉、戎邑道、緜諸道、清水、奉捷、罕幵6縣。建武八年（32年）以前置顯親縣<ref group="古" name="顯親縣"></ref>。永初五年（111年）左右，隴西郡上邽、西2縣來隸<ref group="古">《續漢志》卷23〈郡國五〉：「上邽，故屬隴西；西，故屬隴西。」</ref>{{refTag|name=隴西3縣}}。中平五年（188年），豲道縣移屬南安郡。初平四年（193年）十二月，上邽縣移屬永陽郡。建安十八年（213年）以後歸屬雍州管轄。建安十九年（214年）南安郡、永陽郡廢，所屬豲道、新興、中陶、上邽、清水、罕幵、緜諸道7縣來隸。東漢末，領冀、望恒、阿陽、略陽、勇士、成紀、隴、蘭干、平襄、顯親、西、豲道、新興、中陶、上邽、清水、罕幵、緜諸道18縣。

; [[武都郡|武都郡]]（25年 - 219年）

; [[金城郡|金城郡]]（25年 - 220年）
:治允吾縣，因[[东汉与羌的战争|羌亂]]，永初四年（110年）內徙隴西郡襄武縣，元初五年（118年）復還。初領允吾、浩亹、令居（《漢書》作令皇）、枝陽、金城、榆中、允街、枹罕、白石、河關、臨羌、破羌、安夷13縣，建武十二年（36年）省金城郡入隴西郡；次年（37年）復置<ref group="古">《後漢書》卷1下〈世祖光武帝紀下〉：「建武十二年，省金城郡屬隴西；建武十三年，復置金城郡。」</ref>，領允吾、浩亹、令居、枝陽、金城、榆中、允街、臨羌、破羌、安夷10縣。建安十八年（213年）以前析金城郡臨羌、破羌、安夷3縣置西平郡；同年歸屬雍州管轄。東漢末，領允吾、浩亹、令居、枝陽、金城、榆中、允街7縣。

=== 縣級行政區 ===
東漢初涼州轄縣數，計隴西郡11縣、天水郡16縣、武都郡9縣、金城郡13縣、安定郡21縣、北地郡13縣、武威郡10縣、張掖郡10縣、酒泉郡9縣、敦煌郡6縣，合計118縣。東漢初省35縣（隴西郡1縣，天水郡6縣，武都郡3，安定郡11郡，北地郡馬13縣，酒泉郡1縣），新置鄣縣、顯親、延壽、新興、中陶、清水、罕幵、緜諸道8縣，復置富平、泥陽、弋居、廉、靈州5縣，綏彌縣改名安彌縣。

東漢末，廢枹罕、白石（以上2縣屬隴西）、陰槃（以上1縣屬安定）3縣。至建安十八年（213年）廢涼州為止，合計下轄16郡、101縣。
