'''《跳進來》'''為臺灣歌手[[張惠妹|張惠妹]]的一首單曲，此單曲收錄在其第十七張專輯《偏執面》之中，由台灣環球音樂在2014年7月2日發行，音樂錄影帶則於2014年7月28日釋出。同時，此首單曲亦是《[[敢死队3|浴血任務3]]》的中文宣傳曲<ref>{{cite web|url=https://movie.yam.com/Article/154647|title=天后a MEI跨刀力挺【浴血任務3】 預祝票房破千萬「美金」|publisher=yam蕃薯藤|access-date=2018-08-11|archive-url=https://web.archive.org/web/20180811064533/https://movie.yam.com/Article/154647|archive-date=2018-08-11|dead-url=no}}</ref><ref>{{cite web|url=https://www.nownews.com/news/20140805/1356554|title=張惠妹「跳進」動作片　力挺史特龍|author=翁新涵|publisher=nowNews|date=2014年8月5日|access-date=2018年8月11日|archive-url=https://web.archive.org/web/20180811063739/https://www.nownews.com/news/20140805/1356554|archive-date=2018年8月11日|dead-url=no}}</ref>。

== 創作背景 ==
此首單曲為葛仲珊與張惠妹首次合作，葛仲珊表示曾因壓力太大而花了半年時間創作，去了美國後回台便交出《跳進來》的Demo版<ref>{{cite web|url=https://star.ettoday.net/news/382943|title=MissKo受邀替張惠妹寫歌　拖了半年才交出|author=林政平|publisher=ET東森|date=2014年7月27日|access-date=2018年8月11日|archive-url=https://web.archive.org/web/20180811063312/https://star.ettoday.net/news/382943|archive-date=2018年8月11日|dead-url=no}}</ref>，亦表示觀察張惠妹愛聽快歌，且跳舞時喜歡抓頭髮，彷如掉進自己的世界，只花半小時就完成創作<ref>{{cite web|url=http://www.chinatimes.com/newspapers/20140806000677-260112|title=阿妹愛嗨歌 葛仲珊靈感〈跳進來〉|author=黃嘉平|publisher=中國時報|date=2014年8月6日|access-date=2018年8月11日|archive-url=https://web.archive.org/web/20180811064154/http://www.chinatimes.com/newspapers/20140806000677-260112|archive-date=2018年8月11日|dead-url=no}}</ref>。對於音樂要求很高的張惠妹廳完Demo版後，完全沒做任何修改，直接進錄音室進行錄唱，錄音過程還一次完成，當時工作人員聽了這首旋律簡單的歌曲都十分喜歡<ref>{{cite web|url=http://www.epochtimes.com/b5/14/7/27/n4210659.htm|title=阿妹與葛仲珊一拍即合 邀歌《跳進來》|publisher=大紀元|date=2014年7月27日|access-date=2018年8月11日|archive-url=https://web.archive.org/web/20180811063728/http://www.epochtimes.com/b5/14/7/27/n4210659.htm|archive-date=2018年8月11日|dead-url=no}}</ref>，為了符合《跳進來》的快節奏曲風，張惠妹還特別請了美國舞蹈老師設計了街舞風格的舞蹈，並推出示范教學影帶，讓喜愛跳舞的歌迷可以跟著一起跳<ref>{{cite web|url=http://culture.people.com.cn/BIG5/n/2014/0731/c22219-25375774.html|title=張惠妹推《跳進來》MV 首度與葛仲珊合作|publisher=人民網|date=2014年7月31日|access-date=2018年8月11日|archive-url=https://web.archive.org/web/20180811065020/http://culture.people.com.cn/BIG5/n/2014/0731/c22219-25375774.html|archive-date=2018年8月11日|dead-url=no}}</ref>。

== 紀錄與發行 ==
===紀錄===
張惠妹在《跳進來》發行後表示，希望於8月9日的生日時，在各音樂平台達到1000萬次點閱率<ref>{{cite web|url=https://www.nownews.com/news/20140727/1341500|title=跳起來下海秀舞姿　張惠妹猛奪夜店夯歌|author=林士傑|publisher=nowNews|date=2014年7月27日|access-date=2018年8月11日|archive-url=https://web.archive.org/web/20180811063928/https://www.nownews.com/news/20140727/1341500|archive-date=2018年8月11日|dead-url=no}}</ref>。為此，台灣方面由[[KKBOX|KKBOX]]於當日出動1000萬次會員邀請，並截至當日下午5時共有超過百萬用戶回響；中國方面由[[天天動聽|天天動聽]]及[[网易云音乐|网易云音乐]]分別布達900万笔推播及7000万封私讯，中國雙平台點閱率共計950万余次，綜合其他平台點閱率，超过1200万次<ref>{{cite web|url=http://ent.ce.cn/news/201408/12/t20140812_3337373.shtml|author=傅云鹏|title=张惠妹《跳进来》点击破1200万 感谢粉丝完成生日愿望|publisher=中國經濟網|date=2014年8月12日|access-date=2018年8月11日|archive-url=https://web.archive.org/web/20180811063356/http://ent.ce.cn/news/201408/12/t20140812_3337373.shtml|archive-date=2018年8月11日|dead-url=no}}</ref><ref>{{cite web|url=http://ent.ltn.com.tw/news/breakingnews/1077578|author=陳慧玲|title=張惠妹生日願望達成 「跳進來」點擊破1200萬次|publisher=自由時報|date=2014年8月10日|access-date=2018年8月11日|archive-url=https://web.archive.org/web/20180811064350/http://ent.ltn.com.tw/news/breakingnews/1077578|archive-date=2018年8月11日|dead-url=no}}</ref>。数位合作团队台湾滚石移动表示：「累积24小时回击率约12%，已经突破1200万次，其中台湾歌迷超过1成。」，張惠妹聞訊後則透過經紀人表示：「努力许久的专辑有了很好的回报，会给我更大的信心制作更好的作品」<ref>{{cite web|url=http://ent.qq.com/a/20140812/047873.htm|title=张惠妹新歌MV砸三百万 《偏执面》挑战暴力美学|publisher=騰訊娛樂|date=2014年8月12日|access-date=2018年8月11日|archive-url=https://web.archive.org/web/20180811064546/http://ent.qq.com/a/20140812/047873.htm|archive-date=2018年8月11日|dead-url=no}}</ref>。

== 迴響 ==
此曲發布後，成為台灣夜店熱門曲，張惠妹對此表示：「為了拍攝音樂錄影帶，還特別請美國舞蹈老師編排、敎跳街舞」<ref>{{cite web|url=http://ent.ltn.com.tw/news/breakingnews/1065785|author=陳慧玲|title=張惠妹跳街舞 夜店夯曲挑戰點擊破千萬次|publisher=自由時報|date=2014年7月27日|access-date=2018年8月12日|archive-url=https://web.archive.org/web/20180812053340/http://ent.ltn.com.tw/news/breakingnews/1065785|archive-date=2018年8月12日|dead-url=no}}</ref>，而《跳進來》也是專輯《偏執面》發行以來，反應最好的一首快歌，不管男女都會跟著輕快的節奏舞動身體，因此成為夜店期待值最高的人氣新歌曲<ref>{{cite web|url=http://www.chinatimes.com/realtimenews/20140727002818-260404|title=阿妹葛仲珊 激盪爆紅夜店歌|publisher=中國時報|date=2014年7月27日|access-date=2018年8月12日|archive-url=https://web.archive.org/web/20180812053617/http://www.chinatimes.com/realtimenews/20140727002818-260404|archive-date=2018年8月12日|dead-url=no}}</ref>，同時也掀起臺灣民眾的學舞潮<ref>{{cite web|url=https://news.tvbs.com.tw/entertainment/540857|author=盧冠妃|title=張惠妹MV首度唱跳　《跳進來》掀學舞潮|publisher=TVBS|date=2014年7月30日}}</ref>。
