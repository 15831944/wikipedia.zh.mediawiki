'''跳羚'''是跳羚属仅有的一种。

== 主要特征 ==
眼大似瞪羚，常被误认为[[瞪羚|瞪羚]]，但体形较大，肩高120厘米，体重40公斤。身上有一道黑条，常被误认为黑斑羚（即[[高角羚|高角羚]]）。体型纤瘦，跑得很快。除了身体侧部的黑斑外，其余的地方都是白色的。

== 习性 ==
天生善跑跳，时速可达94公里，仅次[[叉角羚|叉角羚]]和[[猎豹|猎豹]]。可以说是跳栏冠军，最高可跳3.5米，最远可跳10米。牠们就是利用这方式来應付牠们的头号天敌——猎豹，以跳躍閃避獵豹的攻擊或擾亂其視線。

== 食性 ==
以植物的枝叶为食，有时也吃茎。

== [[天敌|天敌]] ==
主要是[[猎豹|猎豹]]、[[花豹|花豹]]、和[[非洲野犬|非洲野犬]]。此外[[猎鹰|猎鹰]]、[[豺|豺]]也会把[[幼羚|幼羚]] 当作[[食物|食物]]。
