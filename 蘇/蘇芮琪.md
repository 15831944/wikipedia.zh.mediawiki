'''蘇芮琪'''<ref name="百度百科">{{cite web |title=苏芮琪 |url=https://baike.baidu.com/item/%E8%8B%8F%E8%8A%AE%E7%90%AA/22581707|website=百度百科 |date=2018-12-16}}</ref>（{{lang-en|'''Rui Qi-Su'''}}，{{bd|2000年|08月20日}}），中國女藝人，現為[[ETM活力時代|ETM活力時代]]旗下女團[[Chic_Chili|Chic Chili]]隊長，是隊內領舞及Rapper擔當。曾在2018年參加選秀綜藝節目[[創造101|創造101]]，最後成績第25名。

== 經歷 ==
蘇芮琪2015年進入[[ETM活力時代|ETM活力時代]]成為旗下練習生<ref name="ETM活力時代">{{cite web |title=etm活力时代 - ETMSkies |url=http://www.etmskies.com/Skies.asp |website=ETM活力時代官方網站 |date=2018-08-02 }}</ref>。2017年7月，擔任女子偶像組合[[ETM_OrangE|ETM OrangE]]首批小分隊隊長，11月18日發表首支廣告曲《咪估圈圈》<ref name="搜狐娛樂網">{{cite web |title=咪咕圈圈发布会—ETM家族OrangE小分队首演圈圈主题曲 |url=http://www.sohu.com/a/205675849_100022609 |website=搜狐娱乐 |date=2017-11-21}}</ref>。2018年2月26日，她又成為[[ETM氧氣少女計畫|ETM氧氣少女計畫]]的其中一員，同年4月21日參加[[騰訊視頻|騰訊視頻]]女子偶像團體養成競技類節目「[[創造101|創造101]]」<ref>{{Cite web|url=https://v.qq.com/detail/7/78483.html|title=创造101|date=2018-06-27|work=v.qq.com|language=zh-CN}}</ref>，最終止步於第25名；8月24日於[[成都|成都]]ETM公演星耀秀上宣布跟[[劉人語|劉人語]]組成雙人小分隊出道；9月30日則與[[劉人語|劉人語]]、[[羅奕佳|羅奕佳]]、[[張靜萱|張靜萱]]及[[吉利|吉利]]組成偶像女團[[Chic_Chili|Chic Chili]]，並在11月19日發行組合先行曲《Chic Chili》<ref name="环球网娱乐">{{cite web |title=Chic Chili女子演唱组合初舞台公开同名单曲燃炸 |url=http://ent.huanqiu.com/music/yinle-nedi/2018-11/13643936.html |website=[[环球网|环球网]]娱乐 |date=2018-11-26}}</ref>；12月2日受邀參加[[英雄聯盟|英雄聯盟]]音樂節暨頒獎典禮，並於在12月11日發行[[英雄聯盟|英雄聯盟]]同人曲《冰雪節》<ref name="新华网体育">{{cite web |title=2018英雄联盟音乐节暨颁奖之夜落幕，iG成大赢家 |url=http://www.sohu.com/a/205675849_100022609 |website=新华网体育 |date=2018-12-02}}</ref>。

=== 創造101時期 ===
蘇芮琪在第一期導師評級被分配為C班,第二期主題曲評級晉級為B班,因節目剪輯畫面幾乎遭刪減。第三期按分級選擇隊長時,沒有選擇實力較強的能力C位，而是選擇了舞蹈能力較弱的勤奮C位[[鹿小草|鹿小草]],主動擔任輔導隊員舞蹈的重任,第四期首次公演舞台演唱了歌曲《第一次說愛你》，側手翻驚艷公演現場。<ref name="网易娱乐">{{cite web |title=舞力少女苏芮琪勇挑重担 侧手翻惊艳公演现场 |url=http://ent.163.com/18/0510/16/DHF6MJPH00038793.html |website=[[网易|网易]]娱乐 |date=2018-05-10}}</ref>第一次順位發布位列第28名,成功晉級55強。第五期被分配至舞蹈組[[徐夢潔|徐夢潔]]戰隊。第六期公演中表演了以[[Taylor_Swift|Taylor Swift]]《Look What You Made Me Do》歌曲改編的木偶舞。第七期二次順位發布,攜手同公司練習生[[劉人語|劉人語]]、[[羅奕佳|羅奕佳]]頂住重重壓力，成功晉級36強,成為全場晉級人數最多的團隊。第八期公演中表演原創曲《Shiny》。第九期順位發布為第25名,最終無緣晉級總決賽。<ref name="新浪娱乐
">{{cite web |title=101决赛名单出炉!孟美岐第一王菊第二 陈芳语落选 |url=http://ent.sina.com.cn/music/zy/2018-06-11/doc-ihcufqih0516403.shtml |website=新浪娱乐
 |date=2018-06-11}}</ref>
