'''蘇南成'''（{{bd|1936年|1月29日{{r|大|page=}}|2014年|9月2日}}<ref name = "CNAobituary">{{cite news | title = 蘇南成病逝 疑心衰竭 | publisher = [[中央通訊社|中央通訊社]] | url = http://www.cna.com.tw/news/aipl/201409025010-1.aspx | date = 2014-09-02 | accessdate = 2014-09-03 | language = zh-tw}}</ref>），[[台灣|台灣]][[政治人物|政治人物]]，[[台南市|台南市]]人。有「'''蘇大頭'''」、「'''大頭成'''」的外號。曾任[[臺南市_(省轄市)|臺南市]][[台南市市長|市長]]、[[高雄市|高雄市]][[高雄市市長|市長]]，後任[[國民大會|國民大會]]代表、議長。

蘇南成以其個人魅力及獨特施政風格，成為在台灣地方自治史上一特立人物，在威權統治時期曾因爭取[[國民黨|國民黨]]提名未果，脫黨以[[黨外|黨外]]身分參與台南市長選舉，並順利當選台南市長，任內因大刀闊斧開辦市政，連任下屆台南市長，其施政也令當時行政院長[[蔣經國|蔣經國]]印象深刻，因而在[[蔣經國|蔣經國]]「吹台青」時期獲得提拔，出任官派（院轄市）高雄市長。卸任高雄市長後，轉任國民大會代表、並於1999年李登輝總統時代接任國民大會議長，但在議長任內因[[國民大會代表自肥案|國代延任案]]引起爭議而名噪一時。

== 早年生平 ==
蘇南成於[[昭和|昭和]]十一年（1936年）出生在[[臺南州|臺南州]][[臺南市_(州轄市)|臺南市]]，昭和十七年（1942年）入[[基督教|基督教]]太平境幼稚園，在[[台灣日治時期|日治時期]]完成幼稚園教育。{{r|絲|page=14}}父親蘇祖蔭在戰前曾為[[西裝|西裝]]師傅，{{r|絲|page=45}}[[二次大戰|二次大戰]]時被[[日軍|日軍]]徵做軍伕。後來蘇祖蔭逃兵，全家即避居[[永康區|永康]][[大灣_(臺南市永康區)|大灣]]鄉下。直到蘇南成小學四年級時，因[[日本投降|日本投降]]才回台南市，住在[[開山路|開山路]]。

蘇南成於1949年考上[[長榮中學|長榮中學]]，然而因父親生意失敗，{{r|絲|page=29}}家庭陷入困境，蘇南成曾擺攤賣[[油條|油條]]、[[豆腐|豆腐]]賺取學費。{{r|絲|page=21}}高中時直升長榮中學高中部。1952年加入[[中國國民黨|中國國民黨]]。{{r|絲|page=96}}

1955年，蘇南成考上[[省立成功大學|省立成功大學]]（今[[國立成功大學|國立成功大學]]）會計統計系，大二時曾到廣播電臺任播音員及家教，亦曾出國到[[菲律賓|菲律賓]]外海打撈沉船。{{r|絲|page=23}}1959年畢業於[[成功大學|成功大學]]會統系。

蘇南成大學畢業後服預官役入[[海軍陸戰隊|海軍陸戰隊]]一年，{{r|絲|page=38}}退伍後曾在[[光華女中|光華女中]]、[[國立臺南大學附屬高級中學|台南高農]]等學校教書，除此之外並在報社、旅行社等處兼差。{{r|絲|page=24}}1962年接辦《新生報》直到1971年，1972到1976年又接下《[[臺灣時報|臺灣時報]]》台南分社的發行工作。{{r|理|page=235}}

== 進入政界 ==
蘇南成在其自傳《絲瓜棚下》曾回憶自己高中時讀[[三民主義|三民主義]]時，「雙手顫抖的捧著書本，流下了淚水，我真像發現到一件寶貝似的」，因此從那時起認為唯有[[政治|政治]]才能樹立良好的社會制度。{{r|絲|page=23}}

早在第五屆[[臺南市_(省轄市)|臺南市]]議會時期，蘇南成即因不滿發生14名市議員因大林國宅興建案的集體辭職事件，而打算辭職參選市議員，{{r|絲|page=23}}但最後遭駁回辭職，未能參選。{{r|絲|page=24}}1964年第六屆台南市市議會選舉時，蘇南成參加市議員選舉，最後最高票落選，激起其再次參選。{{r|絲|page=24}}1968年第七屆市議員選舉，蘇南成由[[中國國民黨|中國國民黨]]提名，以4509票當選台南市議員，踏入政壇。

在議員期間，蘇南成曾受朋友鼓勵參選議長。然而在爭取國民黨地方黨部提名時，卻遭到要求給予每位黨籍議員三千元的要求，引起蘇南成反感。{{r|黨|page=163}}隨後蘇南成到[[臺中|臺中]]的國民黨台灣省黨部後發現，整個臺中的[[酒家|酒家]]、[[舞廳|舞廳]]、[[飯店|飯店]]，淪為全省各縣市議會想當正副議長參選人的招待所。至此蘇南成發現「我身處的這個黨，原來竟以墮落、腐敗到這種地步了」{{r|黨|page=163}}最後蘇南成當然並未獲得國民黨的提名，最後他把國民黨內提名的議長及副議長用來向每位議員[[賄選|賄選]]的三萬元捐出去，{{r|黨|page=164}}同時也埋下了日後退黨的因素。（當時正副議長為魏神助及李銅）

1972年第七屆[[1972年中華民國縣市長選舉|台南市長選舉]]國民黨提名時，蘇南成爭取黨內台南市長提名，但最後國民黨提名[[張麗堂|張麗堂]]，蘇氏脫黨以「自由人士」的無黨籍身份參加選舉，並於11月17日發表退黨聲明書宣告退黨。{{r|理|page=282}}在政見發表會上，蘇氏一再要求與張麗堂公開辯論國民黨提名方式，張雖答應，但因選舉監察小組未同意作罷。之後每場政見發表會都成為雙方針鋒相對場所。{{r|南|page=2}}最後蘇南成獲75,124票，敗給張麗堂。

1973年第八屆臺南市議員選舉，蘇南成以7千9百多票全市最高票當選，連任臺南市議員。{{r|理|page=277}}

== 臺南市長與高雄市長 ==
1977年，蘇南成捲土重來。再度以「[[黨外|黨外]]」身分參加[[1977年中華民國縣市長選舉#臺南市|第八屆臺南市長選舉]]。此次選舉蘇積極攻勢，與黨外[[臺灣省議員|省議員]]候選人[[蔡介雄|蔡介雄]]聯合造勢。{{r|臺|page=255}}在政策上以[[都市計劃|都市計劃]]為訴求。<ref>「蘇南成自認敢做敢當 願提高政治道德水準」，《[[中華日報|中華日報]]》，1977年11月2日</ref>由於蘇的言辭犀利，加上市府發生建設局長王欽誠詐賭案，對張選情不利。{{r|南|page=7}}最後在11月29日的投票中，蘇南成獲得131,504票，以一萬七千票的差距擊敗尋求連任的張麗堂，當選第八屆（省轄市）台南市長。

蘇南成上任後不久即宣佈實施夜間辦公，將市長服務時間擴大至每周一至周五晚間八時到十一時，<ref>「市長實施夜間辦公 議員自嘆無用武之地」，《中華日報》，1978年1月9日</ref>獲得時任[[行政院長|行政院長]][[蔣經國|蔣經國]]讚揚。<ref>「市長夜間辦公 蔣院長表讚揚」，《中華日報》，1978年1月22日</ref>任內設立馬上辦中心，並舉辦[[台灣區運動會|台灣區運動會]]等大型文化活動，獲得極高民意支持度。然而蘇南成在市長任內也表現與黨外保持距離，1979年[[美麗島事件|美麗島事件]]發生後，黨外人士要在中山公園（今[[台南公園|台南公園]]）辦活動，遭蘇斷然拒絕，並批美麗島人士為「暴力分子」，引起黨外勢力與其翻臉並公開批判。{{r|臺|page=255}}

雖然作風評價兩極，但蘇南成仍獲得台南市民支持。1981年[[1981年中華民國縣市長選舉|台南市長選舉]]中，在國民黨開放參選未提名的情況下，以壓倒性擊敗國民黨籍的[[王奕棋|王奕棋]]、[[林永發|林永發]]、[[林三和|林三和]]，以190,588票高票（得票率超過75%）順利連任第九屆台南市長，成為第一位連任成功的台南市長。之後蘇南成在當時[[中華民國總統|中華民國總統]][[蔣經國|蔣經國]]「欽點」下，恢復中國國民黨籍。1982年12月，蘇南成重回國民黨，並對臺南市黨部主委林武俊表示「今後在國民黨領導下，決定全心全力為黨國、為地方效力」。<ref>「蘇南成獲准恢復國民黨籍」，《[[中央日報|中央日報]]》，1982年12月30日</ref>
[[File:Tainan_China_Town_20090831.jpg|300px]]
1983年，蘇南成台南市長任內，贏得菲律賓[[麥格塞塞獎|麥格塞塞獎]]政府服務獎。

蘇南成任期內大刀闊斧，許多知名臺南市建築建設於此時。如[[臺南中國城|臺南中國城商業大樓]]、[[臺南市立殯儀館|臺南市立殯儀館]]、[[臺南市立文化中心|臺南市立文化中心]]等皆完成此時。

前市長[[辛文炳|辛文炳]]曾評價蘇市長任內，認為他是最具[[經濟學|經濟學]]頭腦的市長。蘇先編特別預算去向[[台灣省政府|台灣省政府]]借支，或向一般銀行[[貸款|貸款]]來建設，之後再償還。由於物價上漲關係，償還時幣值就變小了。蘇南成是以企業觀點來從事建設。在[[臺南市第五期市地重劃區|五期重劃區]]就借了三、四十億。{{r|臺|page=269}}事實上，五期重劃區案在第九屆議會時，以二十幾億的天文數字通過，曾引起議會疑慮，後經蘇南成以會計統計的專業說服，{{r|臺|page=87}}此案奠定了後來台南市行政中心的基礎。不過蘇氏提出高額預算也曾引起議會不滿，1982年5月，第十屆議會即因不滿蘇南成預算猛漲40%，引起代理議長楊家寶拒審，最後蘇南成動員市府連夜動工將預算調整在10%。{{r|臺|page=89}}

1985年5月，蘇南成在第二任台南市長任期屆滿前，獲官方轉派接替[[許水德|許水德]]擔任（直轄市）[[高雄市市長|高雄市市長]]一職，然而在高雄市長任內即因與地方勢力產生摩擦，造成施政上受阻。{{r|臺|page=256}}1988年還因市政大樓弊案遭到監察院介入調查，然而在彈劾時卻未通過。一度發生市議員[[陳光復|陳光復]]至監察院抗議，並以燒紙錢、輓聯、鮮花設置等形式為監察院「治喪」，反而遭到逮捕、以「偽造文書罪」遭判刑入獄的事件。<ref>[http://www.tpa.gov.tw/opencms/about/introduction/councilor/councilor00038.html 省諮議員陳光復簡介]，[[台灣省諮議會|台灣省諮議會]]</ref>1990年卸任。

== 國大代表議長與國代延任案 ==
{{Main|國代延任案}}
1992年，蘇南成轉任不分區[[國民大會|國民大會]]代表，並於1999年1月繼[[錢復|錢復]]之後接任國民大會議長，被稱為「大頭議長」。然而同年9月，形象不佳的國大代表欲推行[[國代延任案|國代延任案]]，自行表決延任，使國代及立委延任直到2002年6月30日，立委延任數月，而國代則延任兩年，引來外界批評為自肥。而身為議長的蘇南成在9月3日二讀時，不顧時任國民黨黨鞭[[陳明仁|陳明仁]]反對，強勢通過無記名投票。下午2時進行投票時，在第一次投票失敗後，當民進黨國代反映投票無效後，蘇南成不待國民黨回應，即逕自宣布重新投票，並包裹表決三院院長報告及國代自審預算案，依表決結果通過二讀。<ref>「延任案表決程序 蘇南成強力主導」，《[[中國時報|中國時報]]》，1999年9月4日，二版。</ref>9月4日，通過三讀。<ref>「延任案通過 三黨都不滿」，《中國時報》，1999年9月5日，頭版。</ref>

此案一過，包括時任總統兼國民黨主席[[李登輝|李登輝]]及副總統[[連戰|連戰]]、民進黨總統參選人[[陳水扁|陳水扁]]都反對通過國代延任案。<ref>「延任案通過 三黨都不滿」，《中國時報》，同上。</ref>1999年9月7日，國民黨召開考紀會，處分蘇南成。<ref>「處分蘇南成 國民黨今開考紀會」，《中國時報》，1999年9月7日。</ref>同日蘇南成向黨主席李登輝請辭國大議長以示負責。晚上8點，國民黨召開考紀會，正式開除其黨籍，並喪失不分區國大代表資格。<ref>「國民黨中央考紀會通過開除蘇南成黨籍」，《中國時報》，1999年9月8日，頭版。</ref>9月8日上午，蘇南成向國民大會請辭國大議長。

2001年5月10日，蘇南成接受[[中天新聞台|中天新聞台]]專訪，批評李登輝曾在國民黨中常會堅持將力主國代延任案通過的他予以「處理」，並稱李登輝一生中的最大遺憾應該是培養[[連戰|連戰]]。蘇南成在專訪中稱，當時政治情勢，陳水扁難以出線，連戰也無力應戰，在擔心[[宋楚瑜|宋楚瑜]]出頭的情況下，國民黨決定以「廢除國民大會」交換通過民進黨國大代表「最原始提案」的總統延任案；但他擔任國民大會議長後認為與其廢除國民大會，不如由國民大會來全盤改革政治體制，所以他才積極推動全面修憲；直到1999年8月20日上午，李登輝明確對他表示，由於美國出面干預，「總統延任案不可行」，整個計畫才無疾而終。<ref>「蘇南成爆內幕：兩國論為總統延任鋪路」，《聯合報》，2001年5月11日。</ref><ref>「執政告白／蘇南成爆內幕：兩國論為總統延任鋪路」，《[[東森新聞報|東森新聞報]]》，2001年5月11日。</ref>

== 重回地方 ==
2001年，蘇南成以無黨籍身分參選[[2001年中華民國縣市長選舉#臺南市|第十四屆台南市長選舉]]，最後僅獲47,133票，以14.36%得票率落選。2002年，第十五屆（省轄市）台南市議員選舉，蘇南成以3,146票吊車尾當選市議員。{{r|臺|page=256}}

離開國民黨後，蘇南成一度轉往本土陣營。2003年12月，在時任[[中華民國總統|總統]][[陳水扁|陳水扁]]拜訪下，蘇南成表態支持陳水扁連任，並稱「本土政權要繼續存在，希望大家支持本土色彩濃厚的陳水扁。」並批評過去6年國民黨把《[[中華民國憲法|憲法]]》搞得一塌糊塗。<ref>[http://news.tvbs.com.tw/entry/391238 總統親自造訪　蘇南成表態力挺]，[[TVBS|TVBS]]，2003年12月8日</ref>

2004年，時任台南市議員的蘇南成以無黨籍身份參加[[2004年中華民國立法委員選舉|立法委員選舉]]，以[[台灣正名運動|台灣正名]]、廢除[[考試院|考]][[監察院|監]]二院、首都[[遷都|遷都]]至南台灣為政見，<ref>[http://bulletin.cec.gov.tw/CECData/004/006/001-001/L06-023.pdf 第六屆立法委員選舉公報]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}，中央選舉委員會</ref>最後僅獲7,249票敗選。2005年，蘇南成獲陳水扁總統聘為[[總統府資政|總統府資政]]。

2010年[[2010年中華民國直轄市市長暨市議員選舉#高雄市|高雄市長選舉]]時，蘇南成表態支持脫離民主進步黨參選的[[楊秋興|楊秋興]]，<ref>[http://hk.crntt.com/doc/1015/0/9/1/101509172.html?coluid=7&kindid=0&docid=101509172 蘇南成：楊秋興能力比陳菊好太多]，中國評論新聞，2010年11月18日</ref>在11月20日的高縣三山造勢大會上，蘇南成稱票投[[黃昭順|黃昭順]]等於投給[[陳菊|陳菊]]。<ref>[http://news.ltn.com.tw/news/politics/paper/445584 楊秋興 秦楊跪叩力挺]，《[[自由時報|自由時報]]》，2010年11月21日</ref>2011年[[2012年中華民國立法委員選舉|第八屆立委選舉]]時，蘇南成表態支持[[邱毅|邱毅]]競選立委，並表示「邱毅應該是台灣正義代表，但是高雄人都不懂得珍惜」，「國民黨竟然沒把邱毅列為[[不分區立委|不分區]]，還讓邱毅這麼辛苦的選」，替邱毅抱屈說「老馬再度出風塵很累」。<ref>[http://www.twtimes.com.tw/index.php?page=news&nid=214497 蘇南成偕友人力挺邱毅]，《[[臺灣時報|臺灣時報]]》，2011年12月20日</ref>

== 逝世 ==
2013年7月，蘇南成傳出[[中風|中風]]病倒，後來病況曾一度好轉。然而2014年5月11日傳出蘇南成第四度中風昏迷不醒，醫院一度對家屬發出病危通知。6月11日早上經醫師評估進行[[氣切|氣切]]手術，<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20140611/414196/ 昏迷26日　蘇南成傳病危]，《[[蘋果日報|蘋果日報]]》，2014年6月11日</ref>傍晚已能自主呼吸。<ref>[http://www.chinatimes.com/realtimenews/20140611004602-260407 蘇南成中風昏迷 已能自主呼吸]，《[[中國時報|中國時報]]》，2014年6月11日</ref>

2014年9月2日，蘇南成在20時50分趁著尚未氣絕，由家人從醫院接回家中，理由是蘇南成疑因[[心臟衰竭|心臟衰竭]]病危，可能撐不過去了。醫院稍後接獲通知，蘇南成晚上21時10分在家中病逝，壽終正寢<ref name = "CNAobituary" />。

== 家庭 ==
蘇南成終身未婚，無子女，但曾與台南市長及高雄市長任內的市府顧問、[[菲律賓|菲律賓]]女華僑陳舒珊交往二十幾年，並對陳舒珊先前婚姻所生的一對兒女視如己出，兩人晚年因房地產所有權糾紛分手<ref>[http://news.ltn.com.tw/news/local/paper/810473 弔唁蘇南成 汪笨湖籲陳舒珊出面] 《[[自由時報|自由時報]]》，2014年9月4日</ref>。

== 其他 ==
* 蘇南成任台南市長時，成立馬上辦中心，一民眾表示，當年竹溪遇雨即氾濫，造成附近墓園被水淹沒，遠在宜蘭的兒女為此寫信給蘇南成懇請台南市政府協助改善，沒想到很快收到馬上辦中心回函，表示已經現場進行會勘，將爭取預算整治，一段時間後又收到台南市政府的回信，表示預算經費已編列，...何時動工...何時完工，讓家人大受感動，工程完工後又收到台南市政府報告完工的信函，親人大讚好市長，自此他們家族都投給蘇南成。
* 白色恐怖時期的許多政治犯大多在李天生的大榮鋼鐵廠工作，1984年該工廠倒閉，由於李天生的兒子打算結束工廠用盡各種方法刁難勞方，張大邦先生回憶，當時，大多數的員工生活都很苦，而公司要把人逼走規避資遣費，協調會時談判破裂，高雄市長許水德規避處理，把員工趕入地下室，於是員工只好霸佔廠房及辦公室抗爭，後來蘇南成接任高雄市長，他比較積極協調處理，工廠被法院拍賣由幸福水泥得標，蘇南成對幸福水泥的人說，這些員工很可憐，多少也給點意思，要不然這公司被員工霸佔著，沒辦法點交，也不是辦法。蘇南成說，這不是我的事情，但是我幫你們協商，後來幸福水泥不得不拿出一筆資遣費。

* 台灣威權統治時期，蘇南成忽而黨內，乎而黨外。在其重回[[國民黨|國民黨]]時，被黨外作家[[李敖|李敖]]批評為「台灣第一不要臉」、「蘇南成出身國民黨又出賣國民黨，投身[[黨外|黨外]]又出賣黨外，側身黑社會又出賣黑社會，十多年下來，又回頭重啃國民黨的一對隆乳，有奶就是娘的表演『浪子回頭』」、<ref>〈台灣第一不要臉〉，李敖，1983年1月20日，見《戰鬥是檢驗黨外的唯一標準》，李敖，臺北：獅谷出版，1983年11月10日初版</ref>「他的當選，象徵著國民黨的沒落、黨外的沒落、和現實主義的抬頭」。<ref>〈大頭仔回頭磕頭說從頭〉，李敖，1983年1月9日，見《戰鬥是檢驗黨外的唯一標準》，李敖，同前。然而蘇南成重入國民黨是在臺南市長連任後。</ref>
* 臺南市有位資深國民黨籍人士，曾經評論蘇南成「他以前罵國民黨，說要火燒市黨部、《[[中華日報|中華日報]]》等言論，凶狠到讓我心驚肉跳；後來他討好、歌頌國民黨的話，更肉麻到使我全身起雞皮疙瘩！」<ref>張業生，〈駱駝與狐狸的故事〉。收錄於許春風發行，《運河脈動－蔡介雄評傳》。臺南：嘉鴻出版社。1985年10月初版。頁6。</ref>
* 曾擔任多屆蔡介雄競選總幹事、民進黨台南市黨部主委張國堂曾批評蘇南成的「叛黨」只為了投懷送抱更親熱的姿態。並引述比喻蘇南成的投機特性「蘇南成如果單獨路過[[民生路_(台南市)|民生路]]旁『摸乳巷』碰見一名奄奄一息的老婦人倒在巷中，他可以視若無睹地橫跨過去；可是他經過[[中正路|中正路]][[永福路|永福路]]口，如果碰見一名學童走路不小心自己跌倒，那時他必定會煞有其事飛奔過去搶救甚至大喊『快叫救護車』」<ref>張國堂著，《打拼為台灣 30年來民主運動在府城－關懷與期盼》，1995年4月，頁C-18至C-21。</ref>
