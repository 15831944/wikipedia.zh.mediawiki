'''蘇啟誠'''（{{bd|1957年|8月1日|2018年|9月14日}}），[[台灣|台灣]][[嘉義縣|嘉義縣]][[竹崎鄉|竹崎鄉]]人，[[中華民國|中華民國]][[外交官|外交官]]，從事外交工作長達27年，2018年7月8日起接任[[臺北駐日本經濟文化代表處|臺北駐大阪經濟文化辦事處]]處長，同年9月14日被發現在日本[[自縊|自縊]]身亡，享壽61歲。

==生平==
蘇啟誠是[[東吳大學_(臺灣)|東吳大學]]日本語文學系[[學士|學士]]、[[中國文化大學|中國文化大學]]日本語文學研究所[[碩士|碩士]]。他在1985年赴[[日本|日本]][[大阪大學|大阪大學]]日本語學研究科求學，於1988年4月取得碩士學位<ref name="roc-taiwan.org-4758">{{cite web|url=https://www.roc-taiwan.org/jposa/post/4758.html|title=臺北駐大阪經濟文化辦事處-處長簡歷|publisher=中華民國外交部|accessdate=2018-09-14|archive-url=https://web.archive.org/web/20180914165528/https://www.roc-taiwan.org/jposa/post/4758.html|archive-date=2018-09-14|dead-url=no}}</ref><ref name = "認識蘇啟誠多年 蕭美琴：優秀認真的外交官_2018" /><ref name = "大阪辦事處長蘇啟誠輕生 外交部官員：他是好人_2018" /><ref name = "一位外交官之死》同事與朋友眼中的蘇啟誠：人緣極佳、沒有架子、顧念家鄉、積極為台灣發聲_2018" />。

1991年起進入[[中華民國外交部|外交部]]任職。2007年擔任[[臺北駐日本經濟文化代表處|臺北駐日本經濟文化代表處]]代表秘書官。曾任[[亞東關係協會|亞東關係協會]]副秘書長<ref>{{ cite web | title = 本部單位主管例行新聞說明會紀要---亞協 | url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=70BCE89F4594745D&sms=700DE7A3F880BAE6&s=1269C78D70DC51BB | publisher = 中華民國外交部 | author = 公眾外交協調會 | language = zh-tw | date = 2012-08-30 | access-date = 2018-09-15 | archive-url = https://web.archive.org/web/20180915122237/https://www.mofa.gov.tw/News_Content_M_2.aspx?n=70BCE89F4594745D&sms=700DE7A3F880BAE6&s=1269C78D70DC51BB | archive-date = 2018-09-15 | dead-url = no }}</ref>、外交部亞東太平洋司副司長<ref name = "本部單位主管例行新聞說明會紀要---亞非司、亞太司_2013"> {{ cite web | title = 本部單位主管例行新聞說明會紀要---亞非司、亞太司 | url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=70BCE89F4594745D&sms=700DE7A3F880BAE6&s=75BA1BECA4D22693 | publisher = 中華民國外交部 | author = 公眾外交協調會 | language = zh-tw | date = 2013-10-24 }} </ref>。2013年12月赴日本[[沖繩縣|沖繩縣]][[那霸市|那霸市]]任職臺北駐日經濟文化代表處那霸分處處長。<ref name = "認識蘇啟誠多年 蕭美琴：優秀認真的外交官_2018" /><ref name = "臺灣日本關係協會_2018"> {{ cite web | title = 臺灣日本關係協會 | url = https://www.mofa.gov.tw/aear/News_Content.aspx?n=5386B4719CE6FA82&s=D144DE5055599853 | publisher = 中華民國外交部 | language = zh-tw | accessdate = 2018-09-14 }} </ref><ref name = "駐那霸辦事處蘇啟誠處長拜會沖繩縣新任知事翁長雄志_2015"> {{ cite web | title = 駐那霸辦事處蘇啟誠處長拜會沖繩縣新任知事翁長雄志 | url = https://www.mofa.gov.tw/News_Content_M_2.aspx?n=BD3B450373053909&sms=02499EFF01F32DED&s=A352CCF3D5050F0F | publisher = 中華民國外交部 | author = 公眾外交協調會 | language = zh-tw | date = 2015-01-23 }} </ref>
