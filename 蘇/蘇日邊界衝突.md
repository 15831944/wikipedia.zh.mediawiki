{{蘇日邊界衝突}}

'''蘇日邊界衝突'''是指從1938年—1945年間，[[蘇聯|蘇聯]]與[[大日本帝國|日本]]之間的一系列邊界衝突。

在佔領[[滿洲國|滿洲國]]及[[日韓合併|朝鮮半島]]後，日本轉而有意奪取對蘇聯領土，[[日本皇軍|日本皇軍]]與[[蘇聯紅軍|蘇聯紅軍]]因而在[[中國東北|中國東北]]頻繁地爆發衝突。

== 張鼓峰事件 ==
{{main|張鼓峰事件}}

張鼓峰事件（1938年7月29日─1938年8月11日），是蘇聯聲稱[[滿洲國軍|滿洲國軍]]（日軍）入侵其領土。日本相信這次入侵原因是蘇聯誤解[[俄羅斯帝國|俄羅斯帝國]]和[[清朝|清朝]]簽訂的《[[北京條約|北京條約]]》規定的邊界劃分（及一系列關於邊界劃分的補充協議），加上邊界劃分標誌遭到竄改。

== 諾門罕戰役 ==
{{main|諾門罕戰役}}

'''諾門罕戰役'''，或稱'''哈拉哈河戰役'''，是[[第二次世界大戰|第二次世界大戰]]時[[日本|日本]]及[[蘇聯|蘇聯]]在[[遠東|遠東]]發生的一場戰爭。戰事在1939年在當時的滿洲與蒙古的邊界[[諾門罕|諾門罕]]發生。日、蘇雙方的軍隊分別代表[[滿洲國|滿洲國]]及[[蒙古人民共和國|蒙古人民共和國]]交戰，但日、苏雙方並沒有向對方正式宣戰。戰事以日本[[關東軍|關東軍]]战敗結束。日、苏雙方此後在二戰中一直維持和平狀態，直至1945年8月6日美軍在日本廣島投下[[原子彈|原子彈]]後，蘇聯在8月8日对日本宣戰並發動[[八月風暴|八月風暴]]行動為止<ref>{{cite news|title=鲜为人知日本陆军首败 被打得落花流水赔光老本|url=http://world.people.com.cn/GB/9938251.html|accessdate=2014-01-04|newspaper=人民网|date=2009-08-27|location=北京|language=中文}}</ref> <ref>{{cite web|title=1945年8月6日 美国在日本广岛投掷原子弹|url=http://www.people.com.cn/GB/historic/0806/2578.html|publisher=人民网|accessdate=2014-01-04|location=北京|language=中文}}</ref> 。

== 蘇日中立條約 ==
{{main|蘇日中立條約}}

由於日本在諾門罕戰敗，日本與蘇聯在1941年4月13日簽訂類似《[[德蘇互不侵犯條約|德蘇互不侵犯條約]]》的中立條約。

1941年下半年，當[[纳粹德国|纳粹德国]]入侵蘇聯後（[[巴巴羅薩行動|巴巴羅薩行動]]），日本曾考慮撕毀條約，但最終依然維持及繼續進侵[[東南亞|東南亞]]，很大程度由於諾門罕戰役的失敗，導致日本不敢聯同[[納粹德國|納粹德國]]攻打蘇聯，雖然日本與德國是[[三國同盟條約|三國同盟條約]]的成員，1945年4月5日，蘇聯單方面中止中立條約，三個月後，[[蘇聯入侵滿洲|蘇聯入侵滿洲]]，進攻在[[滿洲國|滿洲]]的日本[[關東軍|關東軍]]<ref>{{cite journal|author=黄力民|title=《远东战役史事新探》|journal=《二十一世纪》|issue=第 49 期|accessdate=2014-01-04|date=2006年4月}}</ref> 。

== 八月风暴行动 ==
{{main|八月风暴行动}}

'''八月風暴行動'''，或稱'''滿洲之戰'''、'''蘇日戰爭'''，由1945年8月8日開始。[[蘇聯|蘇聯]]在這場軍事行動中进攻[[大日本帝國|日本]]的[[傀儡政權|傀儡政權]][[滿洲國|滿洲國]]，同時进攻鄰近的[[蒙疆聯合自治政府|蒙疆]]，占领南[[庫頁島|庫頁島]]及[[千島群島|千島群島]]。根据美国的建议，苏联佔領了[[朝鮮半島|朝鮮半島]]北部、美国占领朝鲜半岛南部。這標誌著蘇聯自從1939年蒙古邊境戰爭後首次對日本帝國的軍事行動，在[[雅爾達會議|雅爾達會議]]上，它已經同意同盟國中止同日本的中立條約，在[[第二次世界大戰歐戰勝利紀念日|歐洲戰爭結束]]後三個月，加入[[太平洋戰爭|太平洋戰爭]]<ref>{{cite web |title=第二次世界大战的扩大 |url = http://www.pep.com.cn/gzls/js/tbjx/kb/jsys/9x_1/201008/t20100830_832956.htm |publisher=人教网|accessdate=2014-01-04|location=北京|language=中文|date=2008-09-09}}</ref> 。

蘇聯的軍事行動在1945年8月8日開始，即德國於5月8日投降的三個月後。另外，這是在8月6日[[廣島市原子彈爆炸|廣島市原子彈爆炸]]及8月9日[[長崎市原子彈爆炸|長崎市原子彈爆炸]]之間。

在政治上，衝突仍未結束，而且日本與俄羅斯至今還未签订二战后的和平条约，及兩國關係因[[南千島群島|南千島群島]]的领土争议而仍然存在緊張，而二战后美、蘇分别占领朝鮮，使[[朝鮮半島|朝鮮半島]]出現分裂，導致日後[[韓戰|韓戰]]的爆發<ref>{{cite news |title=纪念朝鲜战争爆发60周年 |url = http://www.huanqiu.com/zhuanti/history/kw60y/ |accessdate=2014-01-04|newspaper=环球网历史|location=北京|language=zh}}</ref> 。
