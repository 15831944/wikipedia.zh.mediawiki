'''蘇凌峰'''（'''Stanley So''' 1945—）為[[加拿大|加拿大]][[多元文化電視台|多元文化電視台]]（OMNI）的前任[[粵語|粵語]]全國新聞主播及高級編輯。移民[[加拿大|加拿大]]前為[[香港|香港]][[電視廣播有限公司|無綫電視]]著名新聞主播，為香港電視觀眾熟悉。

==生平==
蘇凌峰本名蘇子明，兩歲時隨家人從[[廣州|廣州]]遷居香港<ref name="壹周刊">{{Cite web |title=非常人語：上善若水蘇凌峰 |date=2012-08-16 |publisher=壹周刊 }}</ref>。他從柏立基教育學院（今[[香港教育大學|香港教育大學]]，教育學院前身）畢業後，曾分別在[[元朗|元朗]]某小學和[[廣悅堂魯班學校|廣悅堂魯班學校]]任教<ref name="壹周刊"/><ref name="加中時報">{{Cite web |url=http://cctimes.ca/portal/famous.do;jsessionid=EEDAFB6AC1A2E846CE67B42D8C5DA0CB?type=26&lang=big5 |title=蘇凌峰: 無心插柳 一路長紅 由為人師表轉做傳媒人的心路歷程 |accessdate=2012-07-28 |publisher=加中時報 }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>，其學生包括有「臺灣國民歌王」之稱的[[周華健|周華健]]、漫畫家[[黎達榮|黎達榮]]、[[商業電台|商業電台]]名主持[[阮小儀|阮小儀]]以及[[韋家雄|韋家雄]]與[[韋家輝|韋家輝]]兄弟。為了鍛煉口才，他報讀了[[香港商業電台|香港商業電台]]開辦的廣播訓練班，培訓完畢後獲[[佳藝電視|佳藝電視]]邀請試鏡，並獲聘為兼職電視新聞主播，後再轉為佳藝電視的全職新聞主播及記者<ref name="加中時報"/>。佳藝電視於1978年8月22日倒閉後，他轉職[[無綫新聞|無綫電視新聞部]]，於1980年代時經常主持無綫的《[[六點半新聞報道|六點半新聞報道]]》。其穩重親民的專業形象深入民心，深受觀眾歡迎，甚具星級魅力，因而同僚戲稱「明星蘇」或「蘇星」（名字諧音：蘇軾）；80年代與[[袁志偉_(記者)|袁志偉]]、[[葉雅媛|葉雅媛]]等被視為無綫新聞的台柱，風采一時無兩。蘇凌峰除了主播，亦曾拍檔[[李汶靜|李汶靜]]主持[[星期二檔案|星期二檔案]]多年。其後的星期X檔案轉由[[阮小清|阮小清]]旁述。

蘇凌峰於1990年6月28日最後一次於無綫電視播報新聞，並於該節新聞結束前，史無前例破格在螢光幕前10秒向全港觀眾道別。[[香港中文大學|香港中文大學]]一份學生刊物《[[大學線月刊|大學線月刊]]》1995年一次電話民意調查中，蘇凌峰在離職5年後仍然穩佔最受歡迎新聞報導員三甲位置。<ref>中文大學《大學線》1995年最受歡迎新聞報導員民意調查</ref>

他於1990年7月移民到[[多倫多|多倫多]]，同年9月成為加拿大中文電視的新聞主播<ref name="加中時報"/>。1993年他離開中文電視，轉至北美影城製作公司擔任節目制作<ref name="加中時報"/>。加拿大中文電視於1993年改組成[[新時代電視|新時代電視]]後，他於1995年加入該台，出任其多倫多支部的[[新時代電視新聞|新聞主播]]及總監<ref name="加中時報"/>。1998年他轉到多倫多CFMT電視台（於2002年改稱[[多元文化電視台|OMNI]]）擔任粵語新聞主播、高級編輯和公共事務科主任<ref name="加中時報"/>。曾經監製及主持近60集人物專訪《真情對話》，廣獲坊間好評。蘇凌峰以其新聞敏銳觸覺，捕捉社會各式人物的精采生命和背後故事，專訪人物包括現為[[香港大學|香港大學]]校長的[[徐立之|徐立之]]等。此外蘇凌峰間中主持[[多元二台|多元二台]]紀錄片節目，亦為[[加拿大國營頻道|CBC]]節目《{{link-en|加拿大：人民的歷史|Canada: A People's History}}》編輯粵語稿件及旁述。隨著OMNI於2011年增設全國新聞報道，蘇凌峰亦獲任為該台的粵語全國新聞主播。

2004年獲加拿大研商學社頒發「最佳電視新聞主播」一獎，肯定蘇凌峰多年於新聞傳媒界的專業及貢獻。<ref>2004年研科學社專業精神獎新聞稿</ref>

2006年，在無綫電視《[[友緣相聚|友緣相聚]]》節目中，主持[[沈殿霞|沈殿霞]]到訪多倫多，與蘇凌峰暢談生活及現時新聞報導的處理手法。另外，2007年，在無綫電視為慶祝其啟播40週年而製作的特備節目《[[新聞話事·人|新聞話事·人]]》中，主持[[伍晃榮|伍晃榮]]專訪回港省親的蘇凌峰，分享無綫時代點滴及新聞之事。

2012年7月26日，蘇凌峰在新聞報導的尾聲時宣布離開OMNI電視台，並感謝觀眾36年來的愛戴、支持、勉勵和督促。<ref>[http://www.facebook.com/photo.php?v=10151092842845540 蘇凌峰告別加拿大 OMNI NEWS的片斷]</ref><ref>{{YouTube|w2oe_TiNgFg|蘇凌峰退休了， 在OmniTV 的最後一天 !}}</ref>他接受報章訪問時稱會考慮日後會轉去學院教授傳理知識。<ref>{{Cite web |url=http://news.singtao.ca/toronto/2012-07-27/canada1343378317d3997941.html |title=蘇凌峰難捨第一手資訊 |accessdate=2012-07-28 |publisher=星島日報 |date=2012-07-27 }}</ref>

==個人生活==

蘇凌峰十分關心加國本地華裔青年成長，現擔任服務華裔青年機構Across U-hub董事會副主席。

蘇凌峰與夫人劉婉薇結婚30多年，育有二女<ref name="壹周刊"/>。
