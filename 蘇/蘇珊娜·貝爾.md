'''蘇珊娜．貝爾''' (Susanne Baer)

德國法律學者，2011年起任德國聯邦憲法法院法官，為德國聯邦憲法法院全院16位中在第一庭唯二的女法官之一，也是首位公開出櫃的女同志法官<ref>{{Cite news|url=https://www.dw.com/zh/%E5%A5%B3%E6%9D%83%E4%B8%BB%E4%B9%89%E8%80%85%E8%8B%8F%E7%8F%8A%E5%A8%9C%E8%B4%9D%E5%B0%94%E5%B0%B1%E4%BB%BB%E5%BE%B7%E5%9B%BD%E5%AE%AA%E6%B3%95%E6%B3%95%E5%AE%98/a-14752980|title=女權主義者蘇珊娜·貝爾就任德國憲法法官|author=德國之聲|date=2011年9月1日|work=德國之聲|accessdate=2019年3月31日}}</ref>。她於2010年冬季起擔任密西根大學法學院法律教授，也在德國柏林洪堡大學((Humboldt University of Berlin)法律學院，擔住公法及性別研究教授及學務處長。

== 求學歷程 ==

1964年2月16日出生於薩爾布呂根(Saarbrücken)，1983年至1988年間，蘇珊娜．貝爾就讀於柏林自由大學，攻讀德國法律及政治學。1993年在密西根大學法學院取得碩士(LL.M.)學位。2003年至2005年間，蘇珊娜．貝爾獲漢斯柏克勒基金會奬學金在法蘭克福歌德大學(Goethe University Frankfurt)攻讀博士學位，題目為：德國與美國職場性騷擾態樣之比較。

== 職業經歷 ==

2001/2002年間在畢爾菲大學(Bielefeld)及2001年在葉爾福(Erfurt)擔任大學公法訪問學者。2002年時蘇珊娜．貝爾推辭了畢爾菲大學(Bielefeld)提供的教職，但很快被任命為柏林洪堡大學的大學教職。在2005及2006年，她擔任洪堡大學副校長掌管學生事務及國際事務，同時是跨領域性別研究中心及性別權能中心主任。她主要研究的領域包含：社會文化法律研究、性別研究、反歧視法及比較憲法。

== 德國聯邦憲法法院法官 ==

2011年2月起由綠黨提名第二位出任德國聯邦憲法法院的法官 ，並且也是第一位擔聯邦憲法法院職務的女同志，並已有伴侶。由於她的學術背景及致力於性別平等，也被著名的德國作家費迪南‧馮‧席拉赫 (Ferdiand von Schirach)稱許：有蘇珊娜．貝爾這樣的女權人士成為聯邦憲法法院法官，是值得恭賀的事<ref>{{Cite book|title=可侵犯的尊嚴：一位德國律師對罪行的13個提問，Die Würde ist antastbar|last=馮‧席拉赫(von Schirach)|first=費迪南(Ferdiand )|publisher=先覺出版社|year=2016年|isbn=|location=台北市|pages=11}}</ref> 。

2014年10月27日蘇珊娜．貝爾到訪台灣<ref>{{Cite news|url=https://news.pts.org.tw/article/282459?NEENO=282459|title=德國首位同志大法官 貝爾訪台演講|author=李威撰、莊志成|date=2014年10月28日|work=公視新聞網|accessdate=2019年3月31日}}</ref> ，針對死刑<ref>{{Cite web|url=https://www.taedp.org.tw/story/2748|title=蘇珊娜‧貝爾：法官不是要取悅所有人，而是勇敢做出決定|accessdate=2019年3月31日|author=|date=2014年10月29日|publisher=廢除死刑推動聯盟}}</ref> 、同性婚姻以及性與法律分別發表演說，她支持婚姻平權要直接修改民法一步到位平等的保障同性婚姻<ref>{{Cite news|url=https://newtalk.tw/news/view/2014-10-27/52919|title=同婚法 德國女同志大法官傾向直接開放|author=林雨佑|date=2014.10.27|work=新頭殼newtalk|accessdate=2019年3月31日}}</ref> 。在世新性別所的活動中，她介紹德國性別相關法律的改革歷程及相關案例，並和NGO團體與現場來賓互動，分享同性伴侶的相關法律案例。
