[[File:Suva_from_harbour.jpg|thumb]]
'''蘇瓦'''（{{lang-fj|Suva}}）是[[斐濟|斐濟]]的[[首都|首都]]，也是斐濟最大的城市。蘇瓦位於[[維提島|維提島]]的東南部，屬於中部專區，人口約77,366人<ref>[http://www.statsfiji.gov.fj/FjFacts&Figs05.pdf 斐濟政府網]</ref>。1877年蘇瓦正式成為斐濟的首都。

== 歷史 ==
1874年斐濟成為[[英國|英國]]的殖民地，1877年，斐濟首都由[[列霧卡|列霧卡]]改為蘇瓦，正式成為斐濟的首都。

== 人口 ==
蘇瓦有約77,366人，主要說[[英語|英語]]，但[[斐濟語|斐濟語]]、[[印度斯坦語|印度斯坦語]]和其它印度語言在當地亦十分普及。
== 交通 ==
[[瑙索里國際機場|瑙索里國際機場]]是斐濟的第二大機場，位於蘇瓦的東北方約23公里，提供國內和國際航班。水路方面，蘇瓦是斐济最主要的港口，設有不少觀光船供遊客使用。<ref>[http://news.xinhuanet.com/ziliao/2007-03/06/content_5807759.htm 新華網] {{Wayback|url=http://news.xinhuanet.com/ziliao/2007-03/06/content_5807759.htm |date=20080827121022 }}</ref>
== 教育 ==
成立於1968年的[[南太平洋大學|南太平洋大學]]位於蘇瓦，由十二個太平洋國家共同擁有，是南太平洋國家中最高的學府，其校園分佈在南太平洋國家之中。
