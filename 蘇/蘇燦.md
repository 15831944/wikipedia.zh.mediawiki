'''蘇燦'''，生卒年不詳，[[清朝|清朝]]後期拳師，[[廣東|廣東]][[南海區|南海]]人，一說為湖南人<ref>{{cite web |url=http://blog.xuite.net/joekuo888/twblog/122435046-武俠小說《武壇二虎+》自序裡的廣東十虎 |title=武俠小說《武壇二虎》自序裡的廣東十虎 |author=joekuo999 |publisher=隨意窩 |date=2008-05-27 |language = |accessdate= }}</ref>，為「[[廣東十虎|廣東十虎]]」之一。

==生平==

蘇燦精於拳棍，放蕩不羈，淡薄名利，是「[[:廣東十虎|:廣東十虎]]」之一。曾與妹賣藝，流落廣東。又曾在[[廣州市|廣州市]]三聖社設教館授徒。<ref>{{cite web |url=https://www.cchere.com/article/492059 |title=當傳說死亡的時候——重提嶺南少林舊事 |author=吳昊 |publisher= |date= |language = |accessdate=2016-07-05 }}</ref>

相傳[[黃飛鴻|黃飛鴻]]向蘇燦學得[[醉拳|醉拳]]。<ref name="奇摩">{{cite web |url=https://tw.answers.yahoo.com/question/index?qid=20101109000016KK02759 |title=葉問、李小龍、黃飛鴻、蘇乞兒、霍元甲 的關係 |author=Draculea V |publisher=Yahoo奇摩知識+ |date=2010-11-09 |language = |accessdate= }}</ref>
