'''世界羽毛球混合团体锦标赛'''，通称'''蘇迪曼杯'''，自1989年举行首届比赛，每两年一届，是[[羽毛球|羽毛球]]三大世界性的团体賽之一，另两项是[[汤姆斯杯|汤姆斯杯]]和[[尤伯杯|尤伯杯]]。

==歷史==
[[迪克·蘇迪曼|蘇迪曼]]原本是[[印尼|印尼]]的「羽毛球之父」，他也是印尼羽毛球協會的创始人之一和[[国际羽毛球联合会|国际羽毛球联合会]]（現名羽毛球世界联合会）副主席，為羽毛球運動作出了莫大的貢獻。1988年，国际羽联决定举办混合团体锦标赛，并决定将奖杯命名为蘇迪曼杯。

首屆的蘇迪曼杯賽於[[雅加達|雅加達]]舉行，時間是1989年，該屆冠軍是主辦國的印尼。

==獎杯==

蘇迪曼杯的獎杯極其精美：羽毛球造型，杯身由純[[銀|銀]]所造成，並在杯身表面鍍上了純[[金|金]]，基座刻有印尼国家象征[[婆罗浮屠|婆罗浮屠]]圖案，有印尼人鐘愛羽毛球運動之意。

另外，獎杯的高度為80厘米、重12[[公斤|公斤]]，並寬50厘米。

==賽制==
===比賽方式===
蘇迪曼盃是由不同等級的比賽所組成，最終會決出賽事總排名。只有最高級別的比賽（第一級別）有資格爭奪蘇迪曼盃的冠軍寶座。除第一級別外，其餘級別都由八隊組成；每個級別內將分為兩個小組進行單循環對賽，然後再進行名次賽決出賽事總排名。比賽的最低級別或會因應參賽隊伍數目而有特殊的分組安排<ref name="REGULATIONS">{{cite web|title=REGULATIONS FOR THE SUDIRMAN CUP|url=http://www.bwfbadminton.org/file.aspx?id=550402&dl=1|publisher=BWF}}{{en}}</ref>。

第一級別中共有12隊，12隊先被分為四組，一組三隊進行單循環對賽，各組的首兩名出線八強，再透過抽籤決定八強對戰組合，再依序打四強產生最後兩強，最後決出冠軍。第一級別將不設名次賽及季軍賽<ref name="REGULATIONS"/>。

===比賽項目===
每場賽事都是以男子單打、女子單打、男子雙打、女子雙打和混合雙打五個項目所組成。在小组赛阶段，无论是否已经分出胜负，5盘比赛都要打完；而淘汰赛或者名次赛阶段將採用5盤3勝制定出勝負，只要某一方先赢下3盘，比赛就即時结束<ref name="REGULATIONS"/>。

===隊伍成員===
每個參賽協會會員（國家 / 地區）將負責制訂遴選參賽成員的標準和方法，然後將名單提交予世界羽聯。隊伍成員名單不能少於4人（兩男及兩女隊員），人數上限為20人。隊伍只能從提交的名單中挑選球員參加比賽<ref name="REGULATIONS"/>。

每位球員不得在同一場賽事中出戰超過兩場比賽。若隊伍到達場地後，有球員因受傷或患病，以致只有一男或一女隊員能夠參賽，則該隊員只能參加一場比賽，其餘兩場必須棄權。若隊伍因上述原因只餘下兩名球員能夠參賽（兩男、兩女或一男一女），該隊伍都必須棄權<ref name="REGULATIONS"/>。

===比賽順序===
每場賽事五個項目須依照以下7個可能的順序進行<ref name="REGULATIONS"/>：
#男子雙打 → 女子單打 → 男子單打 → 女子雙打 → 混合雙打
#男子單打 → 女子單打 → 男子雙打 → 女子雙打 → 混合雙打
#混合雙打 → 男子單打 → 男子雙打 → 女子單打 → 女子雙打
#混合雙打 → 男子單打 → 女子單打 → 男子雙打 → 女子雙打
#混合雙打 → 女子單打 → 男子單打 → 女子雙打 → 男子雙打
#女子單打 → 男子單打 → 女子雙打 → 男子雙打 → 混合雙打
#男子雙打 → 女子雙打 → 男子單打 → 女子單打 → 混合雙打

上列的順序1（男雙→女單→男單→女雙→混雙）為最優先選擇。然而，賽會會按照隊伍的實際報名情況，安排其他的順序進行比賽，以避免隊伍中一位球員連續兩場作賽，或一位球員出戰雙打項目先於單打項目。採用順序依次為順序2、順序3……如此類推。如上列7個順序都無法避免隊伍中一位球員連續兩場作賽，或一位球員出戰雙打項目先於單打項目，賽會則會選擇順序2（男單→女單→男雙→女雙→混雙）為比賽順序。賽會將確保任何需要參加兩場比賽的球員，在兩場比賽間有最少30分鐘的休息時間<ref name="REGULATIONS"/>。

==曾進入決賽的國家隊==
至今只有三支國家隊曾獲得蘇迪曼杯的冠軍，其中以中國國家隊的成績最為優異，共10次奪得冠軍；其次是韓國隊（4次冠軍）和印尼隊（1次冠軍）。丹麥隊曾兩度打進蘇迪曼杯的決賽，但都負於中國隊，未能奪冠，而日本隊則在[[2015年蘇迪曼盃|2015年蘇迪曼盃]]首次打進決賽，贏得亞軍<ref>{{cite news|title=（苏迪曼杯·全天综合）中国胜印尼第十一次进决赛　日本逆转韩国再次刷新历史|url=http://news.xinhuanet.com/sports/2015-05/17/c_127808816.htm|agency=新华社|date=2015-05-17|deadurl=yes|archiveurl=https://web.archive.org/web/20150518144822/http://news.xinhuanet.com/sports/2015-05/17/c_127808816.htm|archivedate=2015-05-18}}</ref>。
