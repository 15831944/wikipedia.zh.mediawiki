'''蘇澳新站'''位於[[台灣|台灣]][[宜蘭縣|宜蘭縣]][[蘇澳鎮_(臺灣)|蘇澳鎮]]，坐落於[[台9線|台9線]]旁邊，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[宜蘭線|宜蘭線]]、[[北迴線|北迴線]]的[[鐵路車站|鐵路車站]]。舊名'''南聖湖車站'''，是位在[[北迴線|北迴線]]的起點，與[[宜蘭線|宜蘭線]]的交會點（宜蘭線的終點是[[蘇澳車站|蘇澳車站]]）。

== 車站構造 ==
*[[島式月台|島式月台]]兩座。（第2月台是宜蘭線）
*[[跨站式站房_(台鐵)|跨站式站房]]，接送者可以直接開車上二樓送客。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#一等站|一等站]]。
*[[普悠瑪列車|普悠瑪號]]：全部不停靠。
*[[太魯閣列車|太魯閣號]]：停靠順行228次；逆行229/6229次。
*[[復興號列車|復興號]]：南下（逆行）683、689、北上（順行）684、688/6688次。
*除6691次、6692次外不停靠。
*[[自強號列車|自強號]]：順行停靠204、272、212、172次；逆行停靠407、175/6175、251/6251、249次。
*[[莒光號列車|莒光號]]：除[[觀光列車|觀光團體列車]]、南下（順行）616、624/6624次。北上（逆行）6615次外均有停靠。
*[[臺鐵區間快車|區間快車]]：皆停靠。
*4022次、4043次、4030[[臺鐵區間快車|區間快車]](終點)。
*[[臺鐵區間車|區間車]]：皆停靠。
*對號列車乘客前往武荖坑風景區之最近車站。
*站內設有[[新馬車站|新馬車站]]所屬的「鐵路之旅 - 小站巡禮紀念章」戳章，供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1968年4月15日：設「南新城車站」。
*1975年1月1日：改為「南聖湖車站」。
*1982年1月1日：改為「蘇澳新站」。
*2003年11月14日：[[跨站式站房_(台鐵)|跨站式站房]]落成啟用，接送者可以直接開車上二樓。
*2014年7月18日：由二等站升為一等站，與[[蘇澳車站|蘇澳車站]]對調級別。
*2015年6月30日：啟用多卡通刷卡機。

== 其他 ==
*由于2010年10月底的[[蘇花公路|蘇花公路]]大坍方，造成重大傷亡事件，使得來台旅遊的中國大陸旅客就以“鐵路加公路”的方式，來避開蘇花公路最難行的路段。此轉運方式讓位在宜蘭縣、距離蘇花公路山區路段最近的主要火車站蘇澳新站成為陸客的轉運車站<ref>[https://news.housefun.com.tw/news/article/898896109445.html 蘇澳新站擠　每天6千陸客轉乘《好房網》]</ref>。根據資料顯示，蘇澳新站原本旅客流量每天約一千人，蘇花公路坍方過後，每天約有80輛的遊覽車前來載客<ref>[http://www.chinareviewnews.com/crn-webapp/search/allDetail.jsp?id=101559010&sw=%E5%8F%B0%E6%B9%BE%E8%B5%B0%E4%BA%B2%EF%BC%9A%E8%8B%8F%E6%BE%B3%E6%96%B0%E7%AB%99%EF%BC%8C%E5%AE%9C%E5%85%B0%E9%99%86%E5%AE%A2%E5%AE%89%E5%85%A8%E8%BD%AC%E8%BF%90%E7%AB%99 台灣走親：蘇澳新站，宜蘭陸客安全轉運站]</ref>。而蘇澳新站的旅客量於2011年較前一年成長了44.23%<ref>{{cite web|url=http://www.ptt.cc/bbs/Railway/M.1335940294.A.83E.html|title=[分享] 民國100年臺鐵各站上下車人次|publisher=批踢踢實業坊}}</ref>。
