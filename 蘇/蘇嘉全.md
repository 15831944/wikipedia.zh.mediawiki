'''蘇嘉全'''{{BD|1956年|10月22日|catIdx=S蘇}}，[[中華民國|中華民國]][[政治人物|政治人物]]，[[屏東縣|屏東縣]][[長治鄉|長治鄉]]繁昌村馬卡道族人<ref>http://udn.com/news/story/1/1479941-國會龍頭選舉-蘇嘉全獲74票勝出</ref><ref>{{cite news|title =搏感情 蘇嘉全有馬卡道族血統|date =2011-09-22|publisher=中央社}}</ref>，現任[[中華民國|中華民國]][[立法院院長|立法院院長]]。

蘇嘉全自地方[[民意代表|民意代表]]、幕僚幹部（如「[[陳定南|陳定南]]省長選舉執行總幹事」）、地方行政官員、中央[[內閣|內閣]]閣員均有歷練，現任[[立法院院長|立法院院長]]（民主進步黨籍全國不分區）。此前亦曾任兩屆[[立法委員|立法委員]]、[[國大代表|國大代表]]等中央民代職位。黨職上擔任過兩任[[民主進步黨秘書長|民主進步黨秘書長]]，並曾擔任兩屆[[屏東縣縣長|屏東縣縣長]]，創下屏東縣20幾年來首次的縣長連任紀錄<ref name="今周刊">{{cite web |language=zh-tw |url=http://www.businesstoday.com.tw/v1/content.aspx?a=W20041124084&p=1 |title=選前陳水扁就欽點他入閣 |publisher=[[今周刊|今周刊]] |date=2004-11-24 |accessdate=2011-10-17 }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>；[[陳水扁政府|陳水扁政府]]執政後，其曾任[[中華民國內政部|內政部]]部長與[[行政院農業委員會|行政院農業委員會]]主任委員等[[中華民國政府|政府]]要職。2012年1月14日和時任[[民進黨主席|民進黨主席]][[蔡英文|蔡英文]]搭檔參選第13任[[中華民國副總統|副總統]]以609萬3587票、80萬票之差敗給[[中國國民黨|中國國民黨]]尋求連任的[[馬英九|馬英九]]及時任[[行政院院長|閣揆]]的[[吳敦義|吳敦義]]。

[[2016年中華民國立法委員選舉|2016年立委選舉]]中當選第九屆不分區立委，之後當選立法院院長<ref>[http://www.cna.com.tw/news/firstnews/201602015010-1.aspx 當選立院新龍頭 蘇嘉全得票過半寫新頁]，中央社，2016-02-01</ref>，成為中華民國史上首位非中國國民黨（民主進步黨籍）的立法院院長<ref>http://udn.com/news/story/1/1479941-國會龍頭選舉-蘇嘉全獲74票勝出</ref>。

== 早年經歷 ==
父親為[[蘇興旺|蘇興旺]]，從事販賣[[豬肉|豬肉]]攤販、母親為[[陳水蔭|陳水蔭]]，擔任傳統農家婦女，家中共8位子女，蘇嘉全排行老么，大姊為[[蘇秀美|蘇秀美]]、大哥為[[蘇嘉川|蘇嘉川]]、二哥為[[蘇嘉富|蘇嘉富]]、外甥為[[張世文|張世文]]、岳母為[[洪木蒞|洪木蒞]]、姪子為現任民進黨籍立法委員[[蘇震清|蘇震清]]。

蘇嘉全的童年是在屏東縣大武山腳下度過，就讀屏東[[繁華國小|繁華國小]]期間，喜歡看武俠小說踢足球和排球，並從屏東繁華國小及[[屏東縣立長治國中|屏東縣立長治國中]]念到[[屏東高中|臺灣省立屏東高中]]，考上[[國立臺灣海洋大學|國立臺灣海洋大學]]後北上求學{{r|今周刊}}。1979年，自國立台灣海洋大學水產製造系（今食品科學系）畢業後，開始在[[金門|金門]][[馬祖|馬祖]]地區擔任預備軍官、醫官，1981年，退役後擔任過[[凱勝塑膠|凱勝塑膠]]公司業務推銷員、凱勝塑膠業務經理、[[教師|教師]]{{r|今周刊}}。1984年，與屏東女警察[[洪恆珠|洪恆珠]]結婚。1989年，父親蘇興旺過世。

== 從政經歷 ==
1985年，蘇嘉全當時為了替哥哥蘇嘉川拿回新台幣3萬元保證金，硬著頭皮參選[[屏東縣議會|屏東縣議員]]，直到1986年，在[[黨外|黨外]]公政會的推薦下參選[[屏東縣議會|屏東縣議員]]，結果以最高票落選。同年，蘇嘉全參與[[民主進步黨|民主進步黨]]建黨，擔任屏東縣黨部主任委員，並當選[[國大代表|國大代表]]。1991年，出任民進黨國大黨團召集人，曾在國民大會掀桌抗議「[[萬年國代|萬年國代]]」（中央民代長期不改選）而被憲警抬出場外{{r|今周刊}}。

1993年，在屏東縣當選第二屆[[立法委員|立法委員]]，1995年獲連任當選第三屆[[立法委員|立法委員]]。1997年參選[[屏東縣長|屏東縣長]]，以過半選票當選第十三屆屏東縣長，擊敗[[中國國民黨|中國國民黨]]提名的[[曾永權|曾永權]]。擔任屏東縣長的期間，以「[[東港鎮_(臺灣)|東港]][[黑鮪魚|黑鮪魚]]」和「[[恆春鎮|恆春]]風鈴節」的方式行銷屏東縣，2001年再度以過半選票連任當選第十四屆屏東縣長一職，成為首位連任成功的非國民黨籍縣長。因蘇嘉全面色黝黑，加上推銷黑鮪魚有成，使其贏得「黑鮪魚」的綽號<ref>{{cite web |language=zh-tw |url=http://www.youtube.com/watch?v=F3PltzoxkVo |title=五都大明星》最黑的秘書長 黑鮪魚蘇嘉全 |publisher=中時電子報 |date=2010-10-01 |accessdate=2011-10-17}}</ref><ref>{{cite news |language =zh-tw |author =李欣芳 |url =http://news.ltn.com.tw/news/politics/paper/359307 |title=蘇嘉全︰小刀殺不了黑鮪魚 |publisher =[[自由時報|自由時報]] |date=2009-12-17 |accessdate =2011-10-17}}</ref><ref>{{cite news |language =zh-tw |author=葉素萍 |url =http://www.epochtimes.com/b5/11/9/9/n3368223.htm%E9%BB%91%E9%AE%AA%E9%AD%9A%E7%A0%B4%E6%B5%AA--%E8%98%87%E5%98%89%E5%85%A8%E4%B8%AD%E7%94%9F%E4%BB%A3%E5%87%BA%E9%A0%AD |title=黑鮪魚破浪 蘇嘉全中生代出頭 |publisher=大紀元 |date=2011-09-09 |accessdate=2011-10-17}}</ref>。

2004年4月，出任[[中華民國內政部#歷任部長|內政部長]]，在民進黨內也任職中央執行委員與中央評議委員。2006年1月，轉任[[農委會|農委會]]主委。2009年，[[中華民國|中華民國]][[2009年中華民國地方公職人員選舉|縣市三合一選舉]]，蘇嘉全擔任民進黨的選戰策略小組召集人。

2010年5月20日，蘇空降參選2010年[[臺中市_(直轄市)|大台中]][[臺中市市長|市長]]後，辭去[[民進黨秘書長|民進黨秘書長]]一職，但11月27日的[[五都選舉|五都選舉]]最終在不被看好的情況下小輸給國民黨提名、尋求連任的[[胡志強|胡志強]]。[[吳乃仁|吳乃仁]]於選後請辭祕書長一職，[[民進黨主席|主席]][[蔡英文|蔡英文]]於同年12月20日宣佈蘇嘉全重新接任秘書長，於同日的中執會後正式上任。

2011年9月9日，蘇嘉全接受[[蔡英文|蔡英文]]的正式邀請，成為[[2012年中華民國總統選舉|2012年總統大選]]的[[中華民國副總統|副總統]]候選人，隔年1月14日宣告落敗。

2015年11月11日，民進黨第9屆不分區立委名單出爐，蘇嘉全排名第9位，並於隔年1月16日當選。

2016年1月29日，與有意角逐[[立法院長|立法院長]]的[[柯建銘|柯建銘]]、[[陳明文|陳明文]]經過協調後，確定與[[蔡其昌|蔡其昌]]一同角逐立院正副龍頭寶座。2016年2月1日，以74票確認當選為第9屆立法院院長。

2018年7月18日至20日，立法院長蘇嘉全與立法委員[[邱志偉|邱志偉]]、[[尤美女|尤美女]]、[[蔡適應|蔡適應]]、[[林昶佐|林昶佐]]及多名企業界人士率團訪問[[英國|英國]]。獲得[[英國上議院|英國上議院]]議長[[諾曼·福勒|佛勒]]親自會晤，並與台英國會小組榮譽會長{{link-en|史蒂爾勳爵|David Steel}}、上議院副議長暨首相對台貿易特使{{link-en|福克納勳爵|Richard Faulkner, Baron Faulkner of Worcester}}及上議員[[諾曼·福斯特|諾曼·福斯特]]陪同會見。<ref>{{Cite web |url = http://www.storm.mg/article/469371 |author = 曾詩婷 |date = 2018年7月29日 |title = 國會外交！蘇嘉全率團訪歐洲 英國上議院議長親自會晤 |publisher = [[風傳媒|風傳媒]]}}</ref>並與[[英國自由民主黨|英國自由民主黨]]主席{{link-en|布林頓|Sal Brinton}}、多名議員及[[國際自由聯盟|國際自由聯盟]]（Liberal International）重要成員於全國自由俱樂部進行座談。布林頓表示欣見同屬國際自由聯盟的友黨民進黨成為執政黨，時值自由民主價值在歐洲逐漸退潮，期盼理念相近的自由主義政黨，共同致力推動所堅信的普世價值。<ref>{{Cite web |url = https://www.cna.com.tw/news/firstnews/201807280217.aspx |author = 戴雅真 |date = 2018年7月29日 |title = 蘇嘉全訪英 拜會英國上議院議長 |publisher =[[中央社|中央社]]}}</ref>

* '''佔領立院立場'''
:2016年11月1日，反[[一例一休|一例一休]]修法的青年闖進立法院，佔領民進黨立院黨團總召[[柯建銘|柯建銘]]的辦公室，隨後被警方驅離。立法院長蘇嘉全聞訊後指示草擬立院共同聲明譴責脫序、違法行為，希望各黨團一起連署，但因在野黨反對未通過。由於2014年[[太陽花學運|太陽花學運]]時蘇嘉全曾在臉書發文支持佔領立法院的學生，並呼籲「希望政府能正視民意，站在人民的立場」，因此他被網友質疑在野與執政時期的標準不一。<ref>[http://www.appledaily.com.tw/realtimenews/article/politics/20161102/980241/%E6%98%94%E6%8C%BA%E5%A4%AA%E9%99%BD%E8%8A%B1%E4%BB%8A%E8%AD%B4%E8%B2%AC%E5%AD%B8%E7%94%9F%E3%80%80%E8%98%87%E5%98%89%E5%85%A8%E8%87%89%E6%9B%B8%E9%81%AD%E7%B6%B2%E5%8F%8B%E9%85%B8%E7%88%86 昔挺太陽花今譴責學生　蘇嘉全臉書遭網友酸爆]</ref>

* '''水災期間在日本'''
:2018年8月28日，南台灣豪雨成災，各地災情陸續傳出，副總統陳建仁24日帶著全家人跑來金門度假，引發批評。立法院長蘇嘉全也被民眾目擊現身日本，被懷疑也是旅遊。[[立法院秘書長|立法院秘書長]][[林志嘉|林志嘉]]表示，蘇嘉全是受到日本政府邀請而前往訪問，妻子也同行；不過，因為擔心台灣災情，蘇嘉全提前在今日返國。<ref>[https://udn.com/news/story/6656/3334551 陳建仁水災嚴重旅遊挨罵 蘇嘉全也被直擊人在日本]</ref>
