'''招顯聰'''（{{lang-en|Billy Chiu}}；{{bd|1985年|6月18日}}），[[香港|香港]]民間[[社會運動|社運]]人士，越南裔香港人，持[[反中|反中]]立場，曾是[[香港獨立運動|香港獨立運動]]，[[社會民主連線|社會民主連線]]前成員和「[[我哋係香港人，唔係中國人。|我哋係香港人，唔係中國人。]]」前發言人，因為與另一發起人[[陳梓進|陳梓進]]的想法相違背而退出，後來另起爐灶，先後成立了[[香港人優先|香港人優先]]、[[香港人主場|香港人主場]]及[[歸英獨立聯盟|歸英獨立聯盟]]，提倡香港再次成為[[英國殖民地|英國殖民地]]，然後請求[[英國|英國]]批准[[香港獨立運動|香港獨立建國]]。

==生平==
招顯聰生於[[香港|香港]]，{{来源请求|父母为越南船民}}，曾經在[[尖沙咀|尖沙咀]]的[[酒吧|酒吧]]當[[酒保|酒保]]，因為在一次集會期間涉嫌搶奪警察配槍而被辭退<ref>[http://news.takungpao.com.hk/hkol/topnews/2014-01/2149433.html 揭「港人優先」淪扯線公仔]《大公報》2014年01月03日</ref>，現時報稱無業。

== 政治立場 ==
招顯聰持鮮明的[[反中|反中]]立場，並且支持[[香港獨立運動|香港獨立運動]]。招顯聰自稱為[[香港人|香港人]]，而非[[中國人|中國人]]，支持[[香港獨立|香港獨立]]，並且認為[[香港獨立|香港獨立]]後，毋須再依靠中國供應糧食，[[香港人|香港人]]可以種菜養禽畜自給自足，經濟上亦不再依賴旅遊和金融業，重新發展[[輕工業|輕工業]]，其言論被指想法幼稚<ref>[http://tc.kwokhy.com/m/NP_hkhl/view.php?link=/dailynews/content_hk/2013/06/28/243806.asp 本土派倡食物自給自足 經濟學者警告想法幼稚]《頭條日報》2013年06月28日</ref>。

部分[[香港報章|香港報章]]標籤他為「鼓吹香港獨立」、「煽動[[弱勢社群|弱勢社群]]搞亂香港」。<ref name="煽動">[http://news.takungpao.com.hk/hkol/politics/2013-08/1837025.html 「港獨」組織漸趨囂張 目的是煽動「分離」主義]《文匯報》2013年8月19日</ref><ref name="鼓吹">[http://paper.wenweipo.com/2014/01/02/HK1401020006.htm 特寫：「香港人優先」 骨幹大起底]《文匯報》2014年1月2日</ref><ref name="掌摑">[http://hk.apple.nextmedia.com/realtime/news/20140619/52603813 涉闖軍營裁決 招顯聰庭外被掌摑]《蘋果日報》2014年6月19日</ref>

== 社会運動 ==
2009年，招顯聰為[[社會民主連線|社會民主連線]]普通黨員。2012年，[[張漢賢|張漢賢]]邀請招顯聰加入[[臉書|臉書]]組織「[[我哋係香港人，唔係中國人|我哋係香港人，唔係中國人]]」成為行動組組長，帶領組織走向激進民主派。

2013年，招顯聰與張漢賢及[[陳柏豪|陳柏豪]]另外在[[facebook|facebook]]創立組織「[[香港人優先|香港人優先]]」及「[[香港人主場|香港人主場]]」，並擔任的發言人<ref>[http://eastweek.my-magazine.me/?aid=28003 隔牆有耳：集會內訌 社運人士拖累港視員工]《蘋果日報》2013年10月22日</ref>。

2014年，加入[[民間電台|民間電台]]主持節目《香港人優先》，節目後來改名為港藏同心。

2016年6月26日，招顯聰成立組織「香港歸英獨立聯盟」，並且舉行記者會，宣布計劃參選2016年香港立法會選舉，提出恢復英國主權作過渡，先歸英後獨立，並且尋求港人捐款。

2016年6月27日，招顯聰離開「歸英獨立聯盟」，組織人員宣稱主要原因是招顯聰與其他成員的見解不同，他也不會代表「歸英獨立聯盟」出選立法會。

== 街頭遇襲 ==
2013年2月，招顯聰在[[香港電台|香港電台]]電視部門外被[[愛護香港力量|愛護香港力量]]的支持者推倒受傷送院。

2014年6月19日，招顯聰在進入法庭受審前喊港獨口號，被一名陈姓男子掌摑即倒地不起。

== 被拘捕 ==
2010年12月，招顯聰、[[姜靈彰|姜靈彰]]在參與要求釋放[[趙連海|趙連海]]的遊行中觸犯「擾亂公共安寧」罪名<ref>[http://www.lsd.org.hk/index.php/event/detailFive/803 聲援姜靈彰、招顯聰︰示威無罪　政治檢控可恥] {{webarchive|url=https://web.archive.org/web/20140102200454/http://www.lsd.org.hk/index.php/event/detailFive/803 |date=2014-01-02 }}，[[社會民主連線|社會民主連線]]，2011年05月20日</ref>。

2013年8月，招顯聰在旺角集會撐[[林慧思|林慧思]]被警方指企圖搶槍，遭多名警員先按在地上，再合力抬走，期間招顯聰多次高呼「我冇搶槍」，又稱「警察屈人」，事後警方指是「一場誤會」將他釋放<ref>[http://m.rthk.hk/news/20130821/944197.htm 香港本土聯署曾偉雄促徹查招顯聰被捕事件] {{Wayback|url=http://m.rthk.hk/news/20130821/944197.htm |date=20140103144045 }}《香港電台》2013年8月21日</ref>。

2013年8月，招顯聰在特首[[梁振英|梁振英]]未到會場已被6名警察手拉手圍住。招顯聰多次表示當日並無衝擊警察防線，沒有越過封線，亦沒有越過鐵馬，但當日警方指他「兩次闖過鐵馬，及力圖反抗」<ref>[http://m.rthk.hk/news/20130821/944197.htm 香港本土聯署曾偉雄促徹查招顯聰被捕事件] {{Wayback|url=http://m.rthk.hk/news/20130821/944197.htm |date=20140103144045 }}《香港電台》2013年8月21日</ref>。

2014年10月17日，据「愛港行動」微博消息，招显聪当日早晨因襲擊一海外記者，在旺角被警方拘捕。<ref>[http://weibo.com/3490921680/BrWhyu1Vz “愛港行動”微博]</ref>
 
== 闖入駐港解放軍中環軍營 ==
2013年12月26日，招顯聰受張漢賢指導衝入[[中環|中環]]屬於軍事禁區的[[駐港解放軍|駐港解放軍]][[中環軍營|中環軍營]]被捕。部分人要求严惩擅闯军营者。<ref>[http://hk.apple.nextmedia.com/realtime/news/20131227/52033972 帶港英旗闖軍營 招顯聰：開槍都唔驚]《蘋果日報》2013年12月27日</ref><ref>[http://v.ifeng.com/news/mainland/2014001/01f4fc8d-7ccc-4007-8a05-50f2ca768177.shtml 香港各界要求严惩擅闯驻港军营肇事者]，[凤凰卫视，2014-01-02]</ref>

2014年1月1日，中區重案組在[[香港島|香港島]][[西灣河|西灣河]]拘捕招顯聰，及下午在[[長洲_(香港)|長洲]]拘捕[[張漢賢|張漢賢]]，他們涉嫌擅自闖入位於中環军营的解放军驻港部队总部，觸犯《[[公安條例_(香港)|公安條例]]》（第245章）第38條，在無許可證情況下進入香港驻军禁区<ref>[http://hk.apple.nextmedia.com/realtime/news/20140101/52047856 「香港人優先」闖解放軍總部警拘兩人]《蘋果日報》2014年1月1日</ref>。「香港人優先」四名成員被捕，但他們在[[社交網站|社交網站]][[Facebook|Facebook]]發表：「此仇我一定千倍奉還」。<ref>[http://news.takungpao.com.hk/hkol/politics/2014-01/2152423.html 「香港人優先」發爛渣 網上揚言「千倍奉還」]《大公報》2014年1月5日</ref>

2014年6月19日，4名港人擅闖解放军驻港部队军营案在[[東區裁判法院|東區裁判法院]]宣判，帶頭闖军营的「香港人優先」成員招顯聰罪名成立。裁判官指解放军持槍，三名被告未有顧及自身安危是非常不智，但考慮他們只為表達意見，並無作出破壞，被判入獄2星期、緩刑12個月<ref name="aigo">[http://news.ifeng.com/photo/hdsociety/detail_2014_06/20/36929913_0.shtml#p=1 闯驻港部队者喊港独口号被人扇脸画面] 鳳凰网</ref>，兩認罪的被告各被判罰款2千元<ref name="apple0620">{{cite news|title=招顯聰庭外遭狠摑 闖軍營判罪成　官：不顧自身安危不智|url=http://hk.apple.nextmedia.com/news/art/20140620/18768089|accessdate=2015-06-12|agency=《蘋果日報》(香港)|date=2014-06-20}}</ref>。同日，招顯聰與一名支持者在進入法院前喊港獨口號，被一名中年男性兩掌擊倒在地，在場警员立即拘捕施袭者，有傳聞這位施袭者是退休[[警署警长|警署警长]]<ref name="aigo"></ref>，警方列為普通襲擊案處理<ref name="apple0620"/>。招顯聰表示沒想到法院輕判，但仍會上訴。

== 新界東北案 ==
2014年6月[[反新界東北撥款示威|反新界東北撥款示威]]期間，包括招顯聰在內的13名示威者以竹枝及雨傘撬開[[立法會綜合大樓|立法會綜合大樓]]大門、阻礙關門等被捕。

2016年2月19日，他被裁定參與[[非法集結|非法集結]]罪成立，被判120小時[[社會服務令|社會服務令]]，後來律政司不滿刑期過輕，向[[香港高等法院|高等法院]]上訴法庭申請覆核刑期。

2017年8月15日，法庭裁定律政司覆核刑期得直，招顯聰被改判入獄13個月。<ref>{{cite web|title=【衝擊立會案】上訴庭准許加刑 12被告入獄13個月|url=http://hk.apple.nextmedia.com/realtime/news/20170815/57082514|publisher=蘋果日報|accessdate=2017-08-16|date=2017-08-15}}</ref>他涉[[藐視法庭|藐視法庭]]，拒絕出庭現訊，後在[[紅磡|紅磡]]被拘捕。<ref>[https://www.hk01.com/%E6%B8%AF%E8%81%9E/112286/-%E5%8F%8D%E6%9D%B1%E5%8C%97%E7%99%BC%E5%B1%95%E6%A1%88-%E6%8B%9B%E9%A1%AF%E8%81%B0%E7%BC%BA%E5%B8%AD%E5%87%BA%E5%BA%AD%E8%A2%AB%E9%80%9A%E7%B7%9D-%E7%B4%85%E7%A3%A1%E9%81%AD%E6%88%AA%E6%9F%A5%E8%A2%AB%E6%8D%95 【反東北發展案】招顯聰缺席出庭被通緝　紅磡遭截查被捕] 推廣大01 2017-08-16</ref><ref>[http://hk.on.cc/hk/bkn/cnt/news/20170815/bkn-20170815224751725-0815_00822_001.html 缺席反東北案遭通緝　招顯聰紅磡落網] 東方日報 2017-08-15</ref>

== 悼念恐怖分子言論 ==
2014年3月3日，招显聪在[[新浪微博|新浪微博]]上发表“为昆明四位牺牲的义士默哀”的言论，悼念在[[2014年昆明火车站暴力恐怖袭击事件|2014年昆明火车站暴力恐怖袭击事件]]中，被[[中华人民共和国政府|中华人民共和国政府]]及[[美国政府|美国政府]]定性为[[恐怖分子|恐怖分子]]的恐怖袭击者。受到网民的猛烈批评，直指他发表「反人类言论」。该条微博后被删除，但随后其又上载一张恐怖分子的照片并发表“蜡烛”表情。<ref>[http://bbs.tianya.cn/post-worldlook-1037876-1.shtml 港独分子招显聪在微博上针对昆明事件公然发表反人类言论（转）(转载)]，[[天涯社区|天涯社区]]，2014年3月5日</ref><ref>[http://mil.huanqiu.com/china/2014-05/4999784.html 闯驻港军营4被告3人认罪 首被告仍然否认控罪]，[[环球网|环球网]]，2014年5月20日</ref>
