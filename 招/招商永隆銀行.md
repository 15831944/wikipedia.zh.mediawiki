'''招商永隆銀行'''創立於1933年，是香港具悠久歷史的[[華資|華資]]銀行之一，素持「進展不忘穩健、服務必盡忠誠」之宗旨，為客戶提供全面銀行服務，包括[[存款|存款]]、[[貸款|貸款]]、私人銀行及財富管理、投資理財、證券、信用卡、網上銀行、銀團貸款、企業貸款、押匯、租購貸款、匯兌、保險代理、強制性公積金等。永隆銀行更透過全資附屬公司提供期貨經紀、保險經紀及一般保險承保、物業管理、信託、受託代管，以及資產管理等服務。

2018年10月1日，永隆銀行改名為招商永隆銀行。<ref>[https://www.cmbwinglungbank.com/wlb_corporate/hk/about-us/corporate-information/company-history.html 公司網站 2018-10-02]</ref>

== 歷史 ==
永隆於[[1933年|1933年]][[2月25日|2月25日]]在香港文咸東街37號開業，定名'''永隆銀號'''，資本港幣44,500[[港元|港元]]，經營找換、匯兌、存款、各埠來往、代客買賣股票、黃金及國內公債等業務，成績按年遞增。
 
[[1937年|1937年]]﹐為擴充營業，遷至香港[[皇后大道中|皇后大道中]]112號。
 
[[1941年|1941年]]，[[太平洋戰爭|太平洋戰爭]]爆發，[[香港淪陷|香港淪陷]]。永隆轉往[[澳門|澳門]]繼續經營銀業，及在[[廣西|廣西]][[柳州|柳州]]設立「永隆金號」。

[[1945年|1945年]]，戰事結束，回港復業。

[[1952年|1952年]]，經營[[押匯|押匯]]業務；同年舊總行行址重建落成。

[[1956年|1956年]]，註冊為有限公司。由於業務發展，職員增多，原有行址已不敷用，乃將預置的右鄰和店後舖位重建，擴大營業。

[[1960年|1960年]]，中文名稱由「永隆銀號有限公司」改稱「永隆銀行有限公司」。

[[1965年|1965年]]-為配合業務發展，將原總行行址作第3次重建。

[[1973年|1973年]]-位於[[德輔道中|德輔道中]]連貫[[干諾道中|干諾道中]]之總行新大廈落成，適值本行成立40周年。總行大廈樓高300呎，共25層，總面積145,000餘呎。是年改為公共有限公司。[[渣打銀行|渣打銀行]]加入為本行股東。

[[1975年|1975年]]，另一物業－[[旺角|旺角]][[彌敦道|彌敦道]]之[[銀行中心|銀行中心]]亦告落成，大廈樓高300呎，共25層，總面積245,000餘呎，同時設「旺角分行」於該廈。本行亦於是年開始使用電腦處理存款業務，初期主機附搭電腦公司，至[[1981年|1981年]]自行設立電腦中心，陸續將各項業務電腦化。

[[1980年|1980年]]年初，本行股份上市（上市編號096），將25%股份公開發售與社會人士，並在證券交易所掛牌買賣。

[[1982年|1982年]]，聯同另外4間華資銀行組成銀聯通寶有限公司，向客戶提供自動櫃員機服務。

[[1984年|1984年]]，在[[美國|美國]][[羅省|羅省]]開設首家海外分行。

[[1986年|1986年]]，全資附屬之永隆保險有限公司獲政府授權經營保險業務。

[[1987年|1987年]]，[[渣打銀行|渣打銀行]]因改變投資政策，退出為本行股東，將所持10%股權轉讓與伍氏家族控制之公司。同年成立康令有限公司（其後改名為永隆證券有限公司），向客戶提供證券投資服務。
 
[[1992年|1992年]]，[[新加坡發展銀行|新加坡發展銀行]]從伍氏家族控制之公司承讓百分之十本行股權，加入為股東。

[[1994年|1994年]]，在[[廣州市|廣州市]]設立代表處，為本行在國內首個業務據點。

[[1995年|1995年]]-開始經營[[信用卡|信用卡]]。

[[1996年|1996年]]-成立服務後勤中心，將各部及各分行多項業務工序集中在該中心處理。同年增設[[開曼群島|開曼群島]]分行，向客戶提供離岸銀行服務。

[[1998年|1998年]]-透過全資附屬之永隆財務有限公司經營租購貸款業務。同年推出「永隆網上銀行」，利用互聯網向客戶提供自動化服務。

[[1999年|1999年]]-在[[上海市|上海市]]設立代表處。同年本行聯同幾家同業組成銀聯信託有限公司，開拓強制性公積金服務。並推出電子證券交易系統「永隆電話買賣證券」及「永隆網上買賣證券」服務。
 
[[2000年|2000年]]-與同業組成銀和再保險有限公司及香港人壽保險有限公司，進一步拓展保險業務。

[[2004年|2004年]]-本行是內地與香港更緊密經貿關係安排落實後，首家獲中國銀行業監督管理委員會批准在內地設立分行的香港銀行。同年首家內地分行設於[[深圳|深圳]]，向客戶提供外匯存款、放款、匯款及押匯服務，開辦人民幣業務。主要股東伍絜宜有限公司購回星展銀行所持10%本行股權。

2008年，大股東[[伍宜孫家族|伍宜孫家族]]向[[招商銀行|招商銀行]]以193億港元出售所持有的53.12%永隆銀行權益。<ref>[http://appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20080531&sec_id=15307&subsec_id=15320&art_id=11173608 招行193億購永隆53%]</ref>伍氏家族[[伍步高|伍步高]]、伍步剛及伍步謙分別辭任公司[[董事長|董事長]]、副董事長及[[行政總裁|行政總裁]]職務，但續留任為[[非執行董事|非執行董事]]。<ref>[http://hk.news.yahoo.com/article/080930/9/8hff.html]{{dead link|date=November 2013}}</ref>永隆委任朱琦、[[馬蔚華|馬蔚華]]及張光華為新行政總裁、董事長及副董事長。

2009年，招商銀行完成強制性收購永隆銀行餘下股份後，永隆銀行撤銷上市地位。

2019年10月1日，永隆銀行正式更新名字為招商永隆銀行。

== 營業點 ==
招商永隆銀行現於香港、[[中國|中國]]內地、[[澳門|澳門]]及海外共有機構網點45家，員工逾1,900人。

== 財務狀況 ==
截至2016年6月30日，綜合資產總額為港幣2,379億元。永隆銀行於2008年成為[[招商銀行|招商銀行]]集團成員，並由民營上市公司轉為[[國有企業|國有企業]]。按資產總額計，招商銀行為中國第六大商業銀行，現已躋身全球百大銀行之列。
