'''莫斯科国立库塔芬法律大学'''（{{lang-ru|Московский государственный юридический университет имени О. Е. Кутафина}}，缩写'''{{lang|ru|МГЮА}}'''）是一所位于俄罗斯联邦首都[[莫斯科|莫斯科]]的国立国家预算高等教育机构。

== 学校沿革 ==
[[File:Moscow,_Sadovaya-Kudrinskaya_Street_9_by_night.jpg|右]]
该校创始于1931年，前身是当时在莫斯科成立的中央苏维埃函授法律培训班（函授法律大学）。1933年，更名为中央苏维埃法律函授学院，1937年成为全联盟函授法律学院，并于1990年改建为在莫斯科法律学院。1993年10月6日变更为莫斯科国立法学院（MSAL）。根据俄罗斯联邦教育科技部2012年10月12日通过的第812号文件，该校变更为莫斯科国立法律大学，2013年改名为国立国家预算高等教育机构莫斯科国立库塔芬法律大学。

== 学校历史 ==
[[File:Institut_d'Orenbourg_de_l'Acadèmie_de_droit_de_Moscou_O.I.Koutafine.JPG|右]]
1931年，莫斯科市政府考虑筹建莫斯科国立法律大学的前身——莫斯科苏维埃法律学院，开办了中央苏维埃法律函授培训班（苏维埃法律函授大学）。<br>

1937年，中央函授培训班改为全联盟法律函授学院（{{lang|ru|Всесоюзный юридический заочный институт}} ），该校成为全苏联最大的法律大学。<br>

1940年，全联盟函授法律学院（{{lang|ru|Всесоюзная заочная правовая академия}}）并入全联盟法律函授学院（{{lang|ru|Всесоюзный юридический заочный институт}} ）。<br>

1955年，在莫斯科开办夜校部。<br>

1987年，开办日校部。<br>

1990年，改为莫斯科法学院。<br>

1993年，改为莫斯科国立法学院。<br>

2008年12月23日，该校采用曾担任该校21年领导职务的苏联法学家库塔芬的名字命名以兹纪念<ref>[http://www.consultant.ru/online/base/?req=doc;base=EXP;n=438181 {{lang|ru|Указ Президента РФ от 23 декабря 2008 года № 1814 «Об увековечивании памяти О. Е. Кутафина}}»]</ref>。

== 师生状况 ==

目前，莫斯科国立法律大学有17 000名学生，27个部门，150 000毕业生<ref name="link1">{{Cite web |url=http://msal.ru/academy/index.html |title={{lang|ru|МГЮА сегодня}} |access-date=2018-08-04 |archive-url=https://web.archive.org/web/20120615085831/http://msal.ru/academy/index.html |archive-date=2012-06-15 |dead-url=yes }}</ref>. 此外，该校有：一名[[俄罗斯科学院|俄罗斯科学院]]院士，一名俄罗斯科学院通讯院士，100多位[[全博士学位|全博士]]（俄制学位）和300多位[[副博士学位|副博士]]（俄制学位，相当于中华人民共和国和中华民国的博士），14位功勋科学工作者、15位功勋律师、13位高等职业教育荣誉工作者、12位其他科学院的院士。

== 分校 ==

目前莫斯科国立法律大学有以下分校：
