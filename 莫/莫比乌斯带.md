'''莫比乌斯带'''（{{lang-de|Möbiusband}}），又譯'''梅比斯環'''、'''莫比乌斯环'''或'''麦比乌斯带'''，是一种只有一个面（表面）和一条边界的曲面，也是一种重要的[[拓扑学|拓扑学]]结构。它是由[[德国|德国]][[数学家|数学家]]、[[天文学家|天文学家]][[奥古斯特·费迪南德·莫比乌斯|莫比乌斯]]和[[利斯廷|约翰·李斯丁]]在1858年独立发现的。这个结构可以用一个纸带旋转半圈再把两端粘上之后轻而易举地制作出来。事实上有两种不同的莫比乌斯带镜像，他们相互对称。如果把纸带顺时针旋转再粘贴，就会形成一个[[手性|右手性]]的莫比乌斯带，反之亦類似。

莫比乌斯带本身具有很多奇妙的性质。如果从中间剪开一个莫比乌斯带，不会得到两个窄的带子，而是会形成一个把纸带的端头扭转了两次再结合的环（并不是梅比斯環），再把剛剛做出那個把纸带的端头扭转了两次再结合的环從中間剪開，則變成兩個環。如果你把带子的宽度分为三分，并沿着分割线剪开的话，会得到两个环，一个是窄一些的莫比乌斯带，另一个则是一个旋转了两次再结合的环。另外一个有趣的特性是将纸带旋转多次再粘贴末端而产生的。比如旋转三个半圈的带子再剪开后会形成一個[[三叶结|三叶结]]。剪开带子之后再进行旋转，然后重新粘贴则会变成数个[[Paradromic|Paradromic]]。

莫比乌斯带常被认为是无穷大符号「[[∞|∞]]」的创意来源，因为如果某个人站在一个巨大的莫比乌斯带的表面上沿着他能看到的“路”一直走下去，他就永远不会停下来。但是这是一个不真实的传闻，因为「∞」的發明比莫比乌斯帶還更要早。

== 几何学与拓扑学结构 ==
[[File:MobiusStrip-01.png|right]]
一个利用参数方程式创造出立体莫比乌斯带的方法：
:<math>\begin{align}
&x(u,v)=\left(1+\frac{v}{2}\cos\frac{u}{2}\right)\cos(u)\\
&y(u,v)=\left(1+\frac{v}{2}\cos\frac{u}{2}\right)\sin(u)\\
&z(u,v)=\frac{v}{2}\sin\frac{u}{2}\\
&(0 \leq u < 2\pi,-1 \leq v \leq 1)
\end{align}</math>
这个方程组可以创造一个边长为1半径为1的莫比乌斯带，所处位置为''x-y''面，中心为（0，0，0）。参数''u''在''v''从一个边移动到另一边的时候环绕整个带子。

如果用[[圆柱坐标系|圆柱坐标系]]（''r'',θ,''z''）表示的话，一个无边界的莫比乌斯带可以表示为：

:<math>\log(r)\sin\left(\frac{\theta}{2}\right)=z\cos\left(\frac{\theta}{2}\right).</math>

</div>
从拓扑学上来讲，莫比乌斯带可以定义为[[矩阵|矩阵]][0,1]×[0,1]，边由在0 ≤ ''x'' ≤ 1的时候（''x'',0）~（''1-x'',1）决定，如右图所示。

莫比乌斯带是一个二维的[[紧致流形|紧致流形]]（即一个有边界的[[面|面]]），可以嵌入到三维或更高维的流形中。它是一个[[不可定向流形|不可定向的]]的标准范例，可以看作'''R'''P<sup>2</sup> # '''R'''P<sup>2</sup>。同时也是数学上描绘[[纤维丛|纤维丛]]的例子之一。特别地，它是一个有一纤维[[单位区间|单位区间]]，''I'' = [0,1]的圆''S''<sup>1</sup>上的非平凡丛。仅从莫比乌斯带的边缘看去给出''S''<sup>1</sup>上一个非平凡的两个点（或'''Z'''<sub>2</sub>）的从。{{clear|right}}

== 有关的物体 ==
和莫比乌斯带非常近似的一个几何学物体叫做[[克莱因瓶|克莱因瓶]]。一个克莱因瓶可以用粘贴两个莫比乌斯带的方法制作出来。但是如果物体不进行自我交叉，这个步骤在[[三维空间|三维空间]]内是不可能完成的。

另外一个相近的结构是[[實射影平面|實射影平面]]。如果在實射影平面上有一个洞的话，从左侧看就会形成一个莫比乌斯带。或者把莫比乌斯带的边界进行有限定义，就会形成一个真投影屏面。更形象地说法是重建莫比乌斯带的边缘形成一个普通的环。有一种普遍的误解认为如果不进行平面的自我交叉就无法在三维空间内形成一个有普通环边缘的莫比乌斯带。事实上是可能的，方法是这样的：定义C为xy面上的单位圆，现在连接C上面的[[对跖点|对跖点]]，比如θ和θ + π。当θ在0到π/2之间运动的时候，在xy面上方做这条线的反余切，其他情况则在面下做反余切。

== 艺术和科技 ==
* 莫比乌斯带为很多艺术家提供了灵感，比如美术家[[莫里茨·科内利斯·埃舍尔|莫里茨·科内利斯·埃舍尔]]就是一个利用这个结构在他木刻画作品里面的人，最著名的就是莫比乌斯二代<ref>{{cite web|url=http://www.mcescher.com/Gallery/recogn-bmp/LW441.jpg|format=jpg|title=莫比乌斯二代|author=|website=mcescher.com|language=|date=|access-date=|archive-url=https://web.archive.org/web/20051029040038/http://www.mcescher.com/Gallery/recogn-bmp/LW441.jpg|archive-date=2005-10-29|dead-url=yes}}</ref>，图画中表现一些蚂蚁在莫比乌斯带上面前行。
* 它也经常出现在[[科幻小说|科幻小说]]里面，比如[[亞瑟·克拉克|亞瑟·克拉克]]的《黑暗之墙》{{when|time=2018-07-07T08:46:37+00:00}}。科幻小说常常想象我们的宇宙就是一个莫比乌斯带。由A.J.Deutsch创作的短篇小说{{when|time=2018-07-07T08:46:37+00:00}}《一个叫莫比乌斯的地铁站》<ref>{{cite web|url=http://math.cofc.edu/faculty/kasman/mathfict/mfview.php?callnumber=mf102|title=一个叫莫比乌斯的地铁站|author=A. I. Deutsch|website=cofc.edu|language=|date=|access-date=|deadurl=yes|archiveurl=https://web.archive.org/web/20050829111232/http://math.cofc.edu/faculty/kasman/mathfict/mfview.php?callnumber=mf102|archivedate=2005年8月29日|df=}}</ref>为[[波士顿|波士顿]]地铁站创造了一个新的行驶线路，整个线路按照莫比乌斯带方式扭曲，走入这个线路的火车都消失不见。另外一部小说《[[星际迷航：下一代|星际迷航：下一代]]》中也用到了莫比乌斯带空间的概念。
* 有一首小诗也描写了莫比乌斯带：

* 莫比乌斯带也被用于工业制造。{{citation needed|一种从莫比乌斯带得到灵感的[[传送带|传送带]]能使用更长的时间，因为可以更好的利用整个带子，或者用于制造磁带，可以承载双倍的信息量。|time=2018-07-07T08:46:37+00:00}}
* 有一座钢制的莫比乌斯带雕塑位于[[美国|美国]][[华盛顿|华盛顿]]的史密斯森林历史和技术博物馆。
* {{when|time=2018-07-07T08:46:37+00:00}}荷兰建筑师[[Ben_Van_Berkel|Ben Van Berkel]]以莫比乌斯带为创作模型设计了著名的莫比乌斯住宅。
* {{when|time=2018-07-07T08:46:37+00:00}}在日本漫畫《[[哆啦A夢|哆啦A夢]]》中，哆啦A夢有個道具的外觀就是莫比烏斯帶；在故事中，只要將這個環套在門把上，則外面的人進來之後，看到的依然是外面。
* {{when|time=2018-07-07T08:46:37+00:00}}在電玩遊戲「[[音速小子|音速小子]]－滑板流星故事」中最後一關魔王戰就是在莫比烏斯帶形狀的跑道上進行，如果不打敗魔王，就會一直在莫比烏斯帶上無限循環的跑下去。
* 1988年在日本上映的動畫電影[[機動戰士GUNDAM_逆襲的夏亞|機動戰士GUNDAM 逆襲的夏亞]]以莫比烏斯帶作為對[[命運|命運]]的隱喻：人類就好比行走在莫比烏斯帶上的螞蟻一般，永遠逃不出這個怪圈，不斷重覆著相同的錯誤，類同的悲劇也在不斷地上演。
* 電影《[[機動戰士GUNDAM_逆襲的夏亞|機動戰士GUNDAM 逆襲的夏亞]]》的主題歌BEYOND THE TIME（メビウスの宇宙を越えて）亦呼應了這個主題（日文「メビウス」就是Möbius的意思）。
* [[JoJo的奇妙冒险|JoJo的奇妙冒险]]第6部空條徐倫對c-moon一幕亦有於戰鬥中使用此結構。
* 美国微软公司出品的软件开发工具[[Visual_Studio|Visual Studio]] 2010版的[[logo|logo]]易被误认为是一条莫比乌斯带。<ref>{{cite web |url= https://www.xcode.me/tool/visual-studio-logo |title=visual studio logo 含义和版本 |author= |website=零度编程 |language=zh-cn |date=2017年7月9日 |access-date=2018年5月25日}}</ref>但其实并非如此，因为沿著该logo的表面，走無法像莫比乌斯带那样將全部表面都走遍。
* 2013年，韩国导演[[金基德|金基德]]的电影《[[莫比烏斯_(電影)|莫比烏斯]]》命名就取材于莫比乌斯环，象征人性周而复始的重复悲剧和错误。
* {{when|time=2018-07-07T08:46:37+00:00}}網路上流傳一部動畫影片，用莫比烏斯帶原理，來解釋巴哈所著的[[逆行卡農|逆行卡農]]作品。》{{cite web |url= http://strangepaths.com/canon-1-a-2/2009/01/18/en/ |title=J. S. Bach's crab canon animated on a Möbius strip |author= |website= |language=en |date= |access-date=}}</ref>
*《[[Bilibili拜年祭|Bilibili拜年祭]]2017》中作品《再一次》以莫比乌斯带原理构筑了困住木琳与海阅的古堡。

== 趣闻 ==
中文网络上曾流传有一些以年轻人和老禅师为主人公的涉及数学概念的[[冷笑话|冷笑话]]。其中一则来源于人人网用户黄雁捷的段子大致内容如下：“青年向禅师讨教，希望可以让他的女朋友没有缺点，只有优点。禅师微笑着，请青年为他找一张只有正面没有背面的纸。然后青年掏出了一个莫比乌斯环……”<ref>{{cite web |url= http://chuansong.me/n/1824694 |title=第一个拿莫比乌斯环去坑老禅师的人竟然是：费曼！ |author=Guokr42 (果壳网ID) |website=[[果壳网|果壳网]], 传送门 |language=zh-cn |date=2015年10月20日 |access-date=2018年5月25日}}</ref>

历史上确有相似的事情发生过。主人公是同样拥有传奇色彩的美国[[物理学家|物理学家]][[理查德·费曼|理查德·费曼]]和他当时的女朋友阿琳。少年费曼有一次与阿琳一同谈论[[勒内·笛卡尔|笛卡尔]]的哲学时，指出笛卡尔对于完美必定存在的论述是在[[偷换概念|偷换概念]]。阿琳感叹说也许就像哲学老师说的一样，任何事物都像纸张一样拥有不同的2个面。费曼则说这一说法本身也是值得权衡的，然后根据从《[[大英百科全书|大英百科全书]]》学到的知识，拿出一张纸，在女友面前现场制作了一个莫比乌斯纸环。阿琳非常惊喜，第2天把纸环带到了学校。当老师拿起一张纸又开始举例事物都有两面性时，她兴奋地举起了莫比乌斯纸环，令在场的师生们都为之惊讶。<ref>{{cite book |title=What do you care what the other people think? |trans_title=你干吗在乎别人这么想？——充满好奇心的费曼 |author=[[理查德·费曼|理查德·费曼]] |others=李沉简 (翻译), 徐杨 (翻译), 钱红林 (责任编辑)|publisher=中国社会科学出版社 |edition= |isbn=7-5004-2263-6 |section=第1部分“一个满是好奇心的人”第2节“你干吗在乎别人怎么想？” |pages=23-24 |language=zh-cn |year=1999}}</ref>不过费曼非常欣赏笛卡尔的科学贡献。<ref>{{Cite book |author=[[倫納德·姆沃迪瑙|里昂纳德·曼罗迪诺]] |others=陈雅云 (翻译), 周宏 (责任编辑), 辛艳 (特约编辑) |title=''Feynman's Rainbow: A Search For Beauty In Physics And In Life'' |trans_title=费曼的彩虹：物理大师的最后24堂课 |publisher=[[陕西师范大学出版社|陕西师范大学出版社]] |location=中国[[陕西|陕西]] |isbn=978-7-5613-3712-7 |edition=第1版 |section=封面内页 |language=zh-cn |origyear= |year=2007 |quote=}})</ref>
