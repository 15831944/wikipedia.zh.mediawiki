'''莫怀戚'''{{BD|1951年|6月3日|2014年|7月27日|catIdx= Mo莫}}，出生于[[重庆市|重庆市]]，中国当代作家，[[中国作家协会|中国作家协会]]会员、[[重庆师范大学|重庆师范大学]]教授。<ref name="cat " >{{cite web |url=http://edu.qq.com/a/20110302/000250.htm |title=专家简介：莫怀戚 |publisher= 腾讯网|accessdate=2011-03-24 |work= |language =zh-cn}}</ref>毕业于[[四川大学|四川大学]]中文系。

== 生平 ==
1951年出生于[[重庆市|重庆市]]。<ref name="cat " />1966年初中毕业，1969年上山下乡，作为知青到[[四川|四川]][[内江|内江]]插队当农民。<ref name="lion " />1972年担任重庆电影公司职员。<ref name="lion " >{{cite web |url=http://www.chinawriter.com.cn/zxhy/member/9318.shtml |title= 莫怀戚|publisher= 中国作家网|accessdate= |work= |language =zh-cn}}</ref>1977年恢复高考制度之后，毕业于[[四川大学|四川大学]]中文系。<ref name="cat " /><ref name="lion " />1995年加入[[中国作家协会|中国作家协会]]。<ref name="lion " />

2014年7月27日下午15点45分，因病在家中去世。<ref>http://mn.sina.com.cn/news/m/2014-07-27/213488649.html</ref>

== 风格 ==
莫怀戚的[[散文|散文]]精致唯美，韵味深厚。文字张弛有度，含蓄隽永，包含诗意，典雅庄重。<ref name="cat " />
