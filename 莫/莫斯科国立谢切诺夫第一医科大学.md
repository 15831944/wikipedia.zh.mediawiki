'''莫斯科国立谢切诺夫第一医科大学'''<ref name=studyinrussia>{{cite web |title=莫斯科国立谢切诺夫第一医科大学|url=https://studyinrussia.ru/cn/study-in-russia/universities/msmu/|publisher=Study In Russia俄罗斯联邦科学与高等教育部网站 |language=zh-cn}}</ref>（缩写'''MSMU'''，通称'''莫斯科医科大学'''或'''谢切诺夫大学'''），另译'''谢切诺夫莫斯科国立医科大学'''或'''谢切诺夫莫斯科第一国立医科大学'''，是俄罗斯国内历史最为悠久的[[醫學院|醫學院]]。<ref name=history>{{cite web |title=The University history|url=https://www.sechenov.ru/eng/about-msmu/history/|publisher=sechenov.ru |accessdate=2018-10-10}}</ref><ref name="ria">{{cite web |title=Московский государственный медицинский университет им И.М. Сеченова |url=https://ria.ru/spravka/20131118/977203153.html |work=[[俄罗斯新闻社|俄罗斯新闻社]] |accessdate=2018-10-10 |language=ru |date=2013-11-18}}</ref>

该大学成立于1758年，是{{le|帝国莫斯科大学|Imperial Moscow University}}的医学院，是俄罗斯第一所医学院。<ref name="ria"/>该机构与[[莫斯科国立大学|莫斯科国立大学]]分离并于1930年独立。该大学于1955年为纪念俄罗斯生理学家[[伊万·米哈伊洛维奇·谢切诺夫|伊万·米哈伊洛维奇·谢切诺夫]]而改名。<ref name=history/>
