'''莫应丰'''{{BD|1938年|1月19日|1989年|2月17日|catIdx=M莫}}，[[湖南人|湖南省]][[桃江县|桃江县]]人，毕业于湖北艺术学院，曾担任湖南省文联副主席、湖南省作家协会副主席、[[中国作家协会|中国作家协会]]理事，1982年曾因《将军吟》荣获第一届[[茅盾文学奖|茅盾文学奖]]。<ref name="fish" /><ref name="dang" >{{cite web |url=http://hn.rednet.cn/c/2013/11/27/3209892.htm |title=继往开来书写湖南文艺发展繁荣新华章 |accessdate=2013 |publisher=红网 |work= |language=zh-hans}}</ref><ref name="bee " >{{cite web |url=http://china.rednet.cn/c/2009/09/17/1826631.htm |title=中国首届茅盾文学奖获得者：莫应丰 |accessdate= 2009|publisher=红网 |work= |language=zh-hans}}</ref>

==生平==
1938年，莫应丰出生在[[湖南省|湖南省]][[益阳|益阳]]。
1956年，莫应丰考上湖北艺术学院音乐系。
1961年，莫应丰加入[[中国人民解放军|中国人民解放军]]，在广州军区空军文工团工作。<ref name="fish" >{{cite web |url=http://www.chinawriter.com.cn/xdzj/161.shtml |title=莫应丰 (1938.1.19—1989.2.17) |accessdate= |publisher=中国作家网 |work= |language=zh-hans}}</ref><ref name="bee " />
1970年，莫应丰从[[广州市|广州市]]调到湖南省[[长沙市|长沙市]]的群众文艺工作室担任文学组组长。
1972年，莫应丰开始发表作品。<ref name="fish" /><ref name="bee " />
1978年，莫应丰进入[[潇湘电影制片厂|潇湘电影制片厂]]担任编剧。<ref name="fish" /><ref name="bee " />
1989年2月，莫应丰因病去世。

==家庭==
莫应丰结婚2次。与前妻婚姻持续15年，有2个女儿：大女儿莫竹苇，小名贝贝，毕业于[[广州外贸学院|广州外贸学院]]；小女儿婴婴

第二任妻子欧阳慧玲。

==奖项==
*1982年，《将军吟》荣获第一届茅盾文学奖。
