'''莫建成'''，[[浙江省|浙江]][[嵊州市|嵊州]]人。男，汉族。已被[[双开|双开]]的[[中华人民共和国|中华人民共和国]]高级干部。1972年2月参加工作，1977年5月加入[[中国共产党|中国共产党]]（现已被开除）。[[内蒙古农业大学|内蒙古农牧学院]]农牧业经济管理系农业经济管理专业干部专修班毕业，[[中共中央党校|中共中央党校]]经济管理专业大学本科学历。[[中国共产党第十七次全国代表大会|中共十七大]]、[[中国共产党第十八次全国代表大会|十八大代表]]，[[中国共产党第十八届中央委员会候补委员列表|中共十八届中央委员会候补委员]]。2017年6月，当选[[中国共产党第十九次全国代表大会代表名单|中共十九大代表]]。[[中国共产党第十九次全国代表大会|中共十九大]]尚未召开，即在8月“落马”。

== 简历 ==
=== 内蒙古 ===
1972年2月-1974年3月，内蒙古自治区[[西卓子山街道|西卓子山]]水泥厂工人。1974年3月-1978年7月，内蒙古自治区西卓子山水泥厂青年农场知青、生产队长、副场长。1978年7月-1983年1月。

内蒙古自治区[[乌海市|乌海市]]知青办干事、劳动局技培科负责人。1983年1月-1983年9月，中共内蒙古自治区乌海市委组织部组织员。1983年9月-1985年7月，内蒙古农牧学院农牧业经济管理系农业经济管理专业干部专修班学习。

1985年7月-1985年9月，中共内蒙古自治区乌海市委组织部组织员。1985年9月-1990年10月，共青团内蒙古自治区乌海市委书记；1990年10月-1991年5月，内蒙古自治区乌海市政府秘书长；1991年5月-1991年11月，中共内蒙古自治区乌海市委秘书长；1991年11月-1993年10月，中共内蒙古自治区乌海市委常委、秘书长；（1989年8月-1992年06,在中央党校函授学院经济管理专业学习；1992年9月-1993年01在中央党校进修部进修二班学习）1993年10月-1995年7月，内蒙古自治区乌海市副市长；1995年7月-1996年6月，中共内蒙古自治区乌海市委常委、副市长；1996年6月-1998年2月，中共内蒙古自治区乌海市委副书记、副市长。

1998年2月-2000年2月，内蒙古自治区乡镇企业局副局长；2000年2月-2001年9月，中共内蒙古自治区[[通辽市|通辽市]]委副书记、副市长；2001年9月-2003年1月，中共内蒙古自治区通辽市委副书记、市长；2003年1月-2003年3月，中共内蒙古自治区通辽市委书记、市长；2003年3月-2004年12月，中共内蒙古自治区通辽市委书记。

2004年12月-2006年11月，[[中国共产党内蒙古自治区委员会|中共内蒙古自治区党委]]常委、宣传部部长；2006年11月-2010年4月，中共内蒙古自治区党委常委、包头市委书记。

=== 江西 ===
2010年4月-2011年5月，[[中国共产党江西省委员会|中共江西省委]]常委、组织部部长；2011年5月-2011年11月，中共江西省委常委、组织部部长、统战部部长；2011年11月-2013年7月，中共江西省委常委、组织部部长；2013年7月-2015年1月，中共江西省委常委月，[[江西省人民政府|省政府]]党组副书记、常务副省长；2015年1月-2015年9月，中共江西省委副书记、省政府党组副书记、常务副省长；2015年9月-2015年12月，中共江西省委副书记。

=== 中央 ===
2015年12月-2017年8月，[[中国共产党中央纪律检查委员会|中央纪委]]驻[[中华人民共和国财政部|财政部]]纪检组组长、财政部党组成员。

== 落马 ==
2017年8月27日，[[中央纪委监察部网站|中央纪委监察部网站]]宣布，莫建成涉嫌严重违纪，接受组织审查<ref>{{Cite news|url=http://www.ccdi.gov.cn/jlsc/zggb/jlsc_zggb/201708/t20170827_105492.html|title=中央纪委驻财政部纪检组组长、财政部党组成员莫建成接受组织审查|author=|date=2017-08-27|work=|accessdate=2017-08-27|agency=中央纪委监察部网站}}</ref>。这标志着已经位列中共十八届中央候补委员之首的莫建成已经彻底失去了递补为中央委员的机会。

2017年9月23日，[[中国共产党中央纪律检查委员会|中纪委]][[中华人民共和国监察部|监察部]]宣布，莫建成严重违反党的纪律，并涉嫌犯罪，决定给予其开除党籍、开除公职处分；收缴其违纪所得；将其涉嫌犯罪问题、线索及所涉款物移送司法机关依法处理。给予其开除党籍的处分，待召开中央委员会全体会议时予以追认<ref>{{cite web |url =http://www.ccdi.gov.cn/xwtt/201709/t20170923_107757.html|title=中国保险监督管理委员会原党委书记、主席项俊波，中央纪委驻财政部纪检组原组长、财政部原党组成员莫建成严重违纪被开除党籍和公职|accessdate=2017-09-23|date=2017-09-23|publisher=中央纪委监察部网站}}</ref>。这时距离宣布其接受组织调查不足一个月，创下中共十八大以来贪腐官员从落马到“双开”的最快纪录。

2019年1月23日，[[北京市第一中级人民法院|北京市第一中级人民法院]]一审公开宣判，对莫建成以受贿罪判处有期徒刑14年，并处罚金人民币400万元；扣押在案的赃款赃物及孳息予以没收，上缴国库。莫建成当庭表示服判，不上诉<ref>{{cite web |url =http://www.chinanews.com/gn/2019/01-23/8737282.shtml|title=中央纪委驻财政部纪检组原组长莫建成一审被判14年|accessdate=2019-01-23|date=2019-01-23|publisher=中国新闻网}}</ref>。
