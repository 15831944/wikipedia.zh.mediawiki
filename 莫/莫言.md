'''莫言'''（{{bd|1955年|2月17日|catIdx=G管}}{{efn|出生日期一作1956年3月5日<ref>{{cite web|author=彭子诚 陈敬|title=凤凰网资讯 > 历史 > 中国现代史 >作家莫言谈1976年9月9日：毛主席怎么能死呢？|url=http://news.ifeng.com/history/zhongguoxiandaishi/detail_2011_12/22/11491132_0.shtml|work=2011-12-22|publisher=凤凰网|accessdate=2015-04-16}}</ref>。}}）本名'''管谟业'''，[[山东省|山东省]][[高密市|高密市]]人，[[中国大陆|中国大陆]][[作家|作家]]，[[中国共产党|中国共产党]]黨員，[[中国作家协会|中国作家协会]]副主席。[[北京师范大学|北京师范大学]][[鲁迅文学院|鲁迅文学院]][[文艺学|文艺学]][[硕士|硕士]]，[[香港公開大學|香港公開大學]]榮譽[[文學|文學]][[博士|博士]]，[[华东师范大学|华东师范大学]]<ref>{{Cite web |url=http://news.qq.com/a/20120517/000749.htm |title=著名作家莫言受聘华东师范大学兼职教授 |publisher=腾讯新闻|accessdate=2012-05-17 |language=zh-cn }}</ref>、[[山东大学|山东大学]]兼职[[教授|教授]]<ref>{{Cite web|url=http://www.view.sdu.edu.cn/new/2012/1011/44613.html|title=山东大学研究生导师莫言教授荣获诺贝尔文学奖|publisher=山东大学|accessdate=2012-10-11|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20121016000121/http://www.view.sdu.edu.cn/new/2012/1011/44613.html|archivedate=2012-10-16}}</ref>，[[青岛科技大学|青岛科技大学]]客座[[教授|教授]]，[[汕头大学|汕头大学]]兼职教授<ref>{{Cite web |url=http://www.gdhed.edu.cn/main/www/124/2003-11/79148.html |title=著名作家莫言先生受聘为汕头大学兼职教授 |accessdate=2012-10-14 |work=广东省教育厅 |publisher= |date=2003-11-17 |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20130401051558/http://www.gdhed.edu.cn/main/www/124/2003-11/79148.html |archivedate=2013-04-01 }}</ref>，现为北京师范大学教授<ref>{{Cite web|url=http://www.bnu.edu.cn/bsdkx/50896.htm|title=莫言学长受聘我校教授|publisher=北京师范大学|accessdate=2013-01-22|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20130129173452/http://www.bnu.edu.cn/bsdkx/50896.htm|archivedate=2013-01-29}}</ref>。

1985年起，莫言受到[[魔幻现实主义|魔幻现实主义]]影响，创作出了一批带有先锋色彩的独特作品<ref>{{cite journal|title=媚俗：莫言近期小说创作的价值取向|year=2004|issue=1|page=31-32|author=孙玉双|journal=渤海大学学报}}</ref>，以大胆新奇的写作风格著称<ref>{{cite news|url=http://news.sina.com.cn/c/2012-10-11/190425339396.shtml|title=莫言获诺贝尔文学奖 写作风格以大胆著称|publisher=中国新闻网|date=2012-10-11|accessdate=2012-10-11}}</ref>。2011年8月，莫言创作的长篇小说《蛙》获得第8届[[茅盾文学奖|茅盾文学奖]]。2012年10月11日，莫言因為其“以[[幻觉现实主义|幻觉现实主义]]融合了[[民间故事|民间故事]]、[[历史|历史]]与[[当代|当代]]”<ref name="魔幻现实主义">{{cite web |url = http://article.yeeyan.org/view/245405/324909 |title=莫言到底是不是“魔幻现实主义融合了民间故事、历史与当代社会” |author=[[瑞典学院|瑞典学院]] |others=江烈农譯 |accessdate= 2012年10月14日 |publisher=[[译言网|译言网]] |work= |language = zh-cn}}</ref>而获得[[诺贝尔文学奖|诺贝尔文学奖]]，成为首位获得该奖的中华人民共和国籍作家{{efn|[[赛珍珠|赛珍珠]]获奖时为[[中华民国|中华民国]]籍。}}<ref>{{cite news|title=莫言成为首位获诺贝尔文学奖中国籍作家|url=http://news.163.com/12/1011/19/8DIC5OMV0001121M.html|accessdate=2015-03-29|date=2012-10-11}}</ref><ref name="Nobel Prize">{{cite web |title=The Nobel Prize in Literature 2012 Mo Yan |url = http://www.nobelprize.org/nobel_prizes/literature/laureates/2012/ |publisher=Nobel Media AB |accessdate=2012-10-11 |language=en}}</ref>。

==生平==
1955年2月17日，莫言出生于[[山东省|山东省]][[高密县|高密县]]大栏乡平安庄（今属[[高密市|高密市]][[夏庄镇_(高密市)|夏庄镇]]）{{efn|1= 在短篇小说《[[白狗秋千架|白狗秋千架]]》里他把家乡虚构为高密东北乡，并且此处成为他很多小说中的故事发生地。}} <ref name=nytimescn />，祖父管嵩峰，祖母戴氏，父亲管贻凡，大哥管谟贤，二哥管谟欣。童年时在家乡小学读书，5年级时因[[文化大革命|文化大革命]]辍学，在农村劳动长达10年，主要从事农业，种高粱、种棉花、放牛、割草。<ref>{{cite web|url=http://book.qq.com/a/20110820/000023.htm|title=第八届茅盾文学奖获奖者：莫言|accessdate=2011年8月20日|work=|publisher=腾讯网|language=zh-cn}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>1976年，莫言加入[[中國人民解放軍|中國人民解放軍]]，历任[[班长|班长]]、[[保密员|保密员]]、图书管理员、教员、干事等职。<ref name=rwjs/>

1981年，莫言在[[河北|河北]][[保定|保定]]的《莲池》第5期上发表了处女作短篇小说《春夜雨霏霏》<ref name=nytimescn />。其后又陆续发表了《枯河》、《秋水》、《民间音乐》等作品。1986年，在[[解放军艺术学院|解放军艺术学院]]文学系毕业。1988年[[中国人民解放军军衔|恢復軍銜制]]，莫言被授予上尉軍銜，<ref>[http://blog.sina.com.cn/s/blog_48670cb20102egtv.html 从80年代的《人民日报》回看作家莫言成名路]</ref>後晉升少校。1991年，在[[北京师范大学|北京师范大学]]鲁迅文学院创作研究生班获得文艺学硕士学位。<ref name=rwjs/>

1997年，莫言以长篇小说《[[丰乳肥臀|丰乳肥臀]]》夺得中国有史以来最高额的“[[大家文学奖|大家文学奖]]”，获得高达十万元[[人民币|人民币]]的奖金。同年脱离军界，转至《[[检察日报|检察日报]]》社工作，并为报社的影视部撰写连续剧剧本。<ref name=rwjs/>

2005年，莫言接受[[香港公开大学|香港公开大学]]授予的荣誉文学博士时曾透露，自己年幼时曾经受到一位被划为右派回乡劳动的中文系大学生影响，憧憬有一天也能写出一部作品，过上“腐败作家一天吃三顿[[饺子|饺子]]的生活”。他还说：“如果因为我敢于说实话而授予我荣誉文学博士，那么我觉得自己当之无愧。”。“莫言”是他开始创作时所起的笔名，却是为了提醒自己不要“放炮”说真话，告诫要少说话。<ref name=rwjs>{{cite web |url=http://www.bbc.co.uk/zhongwen/simp/chinese_news/2012/10/121011_profile_moyan.shtml|title=人物介绍： 诺贝尔文学奖得主莫言|accessdate=2012年10月11日|work=|publisher=BBC中文网|language=zh-cn}}</ref>

2007年10月，莫言从《[[检察日报|检察日报]]》社调到[[中华人民共和国文化部|文化部]][[中国艺术研究院|中国艺术研究院]]工作。其《[[检察日报|检察日报]]》社[[记者证|记者证]]于2012年10月18日被注销。<ref>{{cite web |url= http://culture.ifeng.com/whrd/detail_2012_12/29/20655165_0.shtml|title=莫言赶上“严打”记者证被注销 曾供职《检察日报》 |accessdate= 2012年12月29日|date=2012年12月29日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>

==创作风格==
{{expand section}}
莫言的文学作品，题材敏感、反思尖锐、风格独特、语言犀利、想象狂放、叙事磅礴。其作品出版后常常引发广泛的争议。<ref name=nytimescn>{{cite news|author=叶开|title=莫言用“残酷叙事”建立了一个隐秘的文学王国|url=http://cn.nytimes.com/article/culture-arts/2012/10/11/cc11moyan/|accessdate=2012-10-11|newspaper=纽约时报中文网|date=2012年10月11日|deadurl=yes|archiveurl=https://web.archive.org/web/20121013235431/http://cn.nytimes.com/article/culture-arts/2012/10/11/cc11moyan/|archivedate=2012年10月13日|df=}}</ref>德國漢學家顧彬認爲莫言的小说是“老掉牙的方式，重复地写刺激感官的男人和女人、性、犯罪等等。”。莫言曾说自己的写作是“日常生活中，我可以是孙子，懦夫，是可怜虫，但在写小说时，我是贼胆包天、色胆包天、狗胆包天。”<ref>{{cite web |url= http://news.ifeng.com/opinion/special/moyan/|title=莫言是叛徒吗？ |accessdate=2012年12月15日 |publisher=凤凰网 |archiveurl=http://news.ifeng.com/opinion/special/moyan/ |archivedate=2012年12月15日 |work= |language =zh-cn}}</ref>

莫言曾说：“80年代，我读到了[[川端康成|川端康成]]的小说《[[雪国|雪国]]》，其中的一句话‘一只黑色而壮硕的秋田狗蹲在那里的一块踏石上，久久地舔着温热的河水。’这让我想起了我家乡的一条大白狗，然后我写出了《白狗秋千架》，就是在这部作品中第一次出现‘高密东北乡’，从此，我就高举起了‘高密东北乡’的大旗，如同一个草莽英雄现世，创建了自己的文学王国。”<ref>{{cite web |url=http://news.sohu.com/20121012/n354765784.shtml |title=日媒盛赞莫言 认为其受《雪国》等日本小说影响 |accessdate=2012 |publisher=搜狐网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/x/2007-10-01/22401736618.shtml |title=莫言：“高密东北乡”源自《雪国》 |accessdate=2007 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web|url=http://book.sina.com.cn/news/a/2006-07-28/1139203376.shtml|title=莫言获奖“事出偶然” 创作灵感来自川端康成|accessdate=2006|publisher=新浪网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20141129022513/http://book.sina.com.cn/news/a/2006-07-28/1139203376.shtml|archivedate=2014-11-29}}</ref>

莫言在《唯一一个报信者》中写道：“一个作家，一辈子只能干一件事：把自己的血肉，连同自己的灵魂，转移到自己的作品中去……我在写作，早期是向外看，对罪恶的抨击多一些，更多想到的是外部强加的痛苦，想到自己怎么受社会的挤压和别人的伤害。慢慢就向内写了，写内心深处的恶，尽管没有释放出来。”<ref>{{cite journal | language =zh-hans | author =莫言 | title =《唯一一个报信者》| journal =《青年文摘》 | volume =第8期 | year = 2010年4月 | pages = 第37页 | issn = 1003-0565}}</ref>

莫言的作品带有强烈的社会责任感，语言风格汪洋恣肆，豪放不羁，兼用长句和短句，同时又有民间语言的粗野淳朴的原始风貌。<ref name="september" />

莫言早期最具有代表性的作品是《[[天堂蒜薹之歌|天堂蒜薹之歌]]》，这是根据改革开放初期一个真实事件改编而成，当时数千农民响应县政府的号召去种植[[蒜薹|蒜薹]]，导致蒜薹全部滞销，县政府却充耳不闻，酿成了当年震惊全国的“蒜薹事件”，莫言阅读新闻后，放下当时在写作计划中的《红高粱家族》，用35天写出了这部长篇小说，以“为民请命”。<ref name="september" >{{cite journal | language =zh-hans | author =吴洪群 | title =《中国的脊梁莫言》| journal =《[[老年人_(杂志)|老年人]]》 | volume =第1期 | year = 2013年 | pages =  | issn =1007-2616 }}</ref>
这部长篇小说体现了莫言作为作家的良知，反映出了中国大陆“老百姓”的弱势和生存艰辛，同时抨击了中国共产党领导的政府当局对“人”的漠视和生命尊严的践踏。<ref name="may" >{{cite book |language=zh-hans|author=曹元勇 |title=莫言《蛙》|year=2009 |publisher=上海文艺出版社 |location=上海市 |pages=344页 |isbn=978-7-5321-3676-6 }}</ref>

《十三步》被誉为“变貌记”，用“荒诞派”的艺术手法写出了中国知识分子的生活和生存状态。<ref name="may" />

《食草家族》仍旧沿用了“荒诞派”艺术手法，同时加入了“魔幻现实主义”。<ref name="may" />

《红树林》是莫言唯一一部背景离开家乡高密县东北乡的长篇小说。<ref name="may" />

《丰乳肥臀》最能体现莫言写作脱离意识形态的影响，该书从抗日战争、国共内战、朝鲜战争、大跃进、大饥荒、反右、文化大革命、一直写到改革开放，既赞美了中国共产党艰辛摸索开天辟地夺取政权的艰辛，也赞美了中国国民党在正面战场的抗日功劳，同时没有回避敏感的斗争老干部、知识分子的反右运动和文化大革命，也没有回避大饥荒饿死诸多人口的历史事实。<ref>{{cite book |language=zh-hans|author=莫言 |title=《丰乳肥臀》|year=2010 |publisher=北京十月文艺出版社 |location=北京市 |pages= |isbn=978-7-5302-1019-2 }}</ref>

==主要作品==

12月6日，莫言抵达[[斯德哥尔摩|斯德哥尔摩]]领奖，并出席了领奖前的新闻发布会<ref name="ppw">{{cite news |url=http://world.people.com.cn/n/2012/1207/c1002-19818182.html | title=莫言抵达瑞典领奖 淡定出席新闻发布会
 | work=人民网 | date=2012-12-06 |accessdate=2012-12-07}}</ref>。他在新闻发布会上表示获奖很高兴，并用“心如巨石，风吹不动”来形容自己的心情<ref name="ppw"/>。他认为“获奖是个人的事情”，但也希望自己获奖“对中国文学起到一个积极的推动作用”，希望把对他的热情“转移到对中国广大的作家身上去，也希望能从阅读莫言一个人的作品开始阅读更多人的作品”，也希望自己能够在获奖之后仍然写出优秀的作品。对于外界的批评，他认为“很多人用自己丰富的想象力塑造了另一个莫言，我就和大家一起来围观对‘莫言’的批评和表扬”<ref name="ppw"/>。

==颁奖机构颁奖词==
*[[冯牧文学奖|冯牧文学奖]]：“莫言以近20年持续不断的旺盛的文学写作，在海内外赢得了广泛声誉。虽然，他曾一度在创新道路上过犹不及，但他依然是新时期以来中国最有代表性的作家之一。他创作于80年代中期的‘红高粱’家族系列小说，对于新时期[[军旅文学|军旅文学]]的发展产生过深刻而积极的影响。《红高粱》以自由不羁的想象，汪洋恣肆的语言，奇异新颖的感觉，创造出了一个辉煌瑰丽的莫言小说世界。他用灵性激活历史，重写战争，张扬生命伟力，弘扬民族精神，直接影响了一批同他一样没有战争经历的青年军旅小说家写出了自己‘心中的战争’，使当代战争小说面貌为之一新。”<ref>{{cite web|url=http://www.cnread.net/cnread1/news/a02/086.htm|title=《第二届冯牧文学奖颁奖，莫言意外获得“军旅文学”奖》|accessdate=2001年2月26日|work=胡晓凡|publisher=江南时报|language=zh-cn}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

*法国[[儒尔·巴泰庸外国文学奖|儒尔·巴泰庸外国文学奖]]：“由中国杰出小说家莫言原创、优秀汉学家杜特莱翻译成法文的《酒国》，是一个空前绝后的实验性文体。其思想之大胆，情节之奇幻，人物之鬼魅，结构之新颖，都超出了法国乃至世界读者的阅读经验。这样的作品不可能被广泛阅读，但却会为刺激小说的生命力而持久地发挥效应。”<ref name="multiple1">{{cite book |language =zh-hans |author=莫言|title=《蛙》|date=2009-12-01|publisher=上海文艺出版社|page=342|isbn=9787532136766}}</ref>

*[[鼎钧双年文学奖|鼎钧双年文学奖]]：“从《透明的红萝卜》开始，莫言的创作一直保持了旺盛的生命力。究其根本，应该归诸于莫言的感觉方式有着深厚的地域和民间渊源。《檀香刑》是这样一个标志：民间渊源首次被放到文源论的高度来认识，也被有意识地作为对近二三十年中国小说创作宗从西方话语的大格局寻求超越和突破的手段加以运用；同时，作者关于民间渊源的视界进一步开拓，开始从抽象精神层面而转化到具体的语言形式层面，从个别意象的植入发展到整体文本的借鉴。[[义和团|义和团]]现象本身就是[[民间文化|民间文化]]所孕育所造就，是山东古老民间文化的一次狂欢。借这个题材来激活一种以民间文化为底蕴的小说叙述，使本事与形式之间的天衣无缝，形成了一种妙不可言的“回声”。民间[[戏曲|戏曲]]、[[说唱|说唱]]，既被移植到小说的语言风格中，也构成和参与了小说人物的精神世界。这种‘形式’与‘内容’的浑然一体，使得《檀香刑》比以往任何高扬‘民间性’的小说实践，走得更远，也更内在化。神奇化、暴力倾向，仍旧是莫言给人的突出印象。作者把他的这一奇特兴趣，用于表现或映衬一种桀骜不驯，一种野性，一种[[英雄主义|英雄主义]]气概。重要的不在于人们是否接受他的观点，而在于他的这种心理倾向已经铸成了鲜明的小说个性。这部小说尚须探讨的问题包括：一、其艺术表现中的某些粗疏之处多少伤及小说的整体肌理；二、其对人性品质的表现及所持价值观，在不同读者中间，可能不是没有异议的。”

*[[法兰西文学与艺术骑士勋章|法兰西文学与艺术骑士勋章]]：“您写作的长、短篇小说在法国广大读者中已经享有名望。您以有声有色的语言，对故乡山东省的情感、反映农村生活的笔调、富有历史感的叙述，将中国的生活片段描绘成了同情、暴力和幽默感融成一体的生动场面。您喜欢做叙述试验，但是，我想最引起读者兴趣的还是您对所有人物，无论是和您一样农民出身的还是所描写的干部，都能够以深入浅出的手法来处理。” 

*[[华语文学传媒大奖|华语文学传媒大奖]]·年度杰出成就奖：“莫言的写作一直是当代中国的重要象征之一。他通透的感觉、奇异的想象力、旺盛的创造精神，以及他对叙事艺术探索的持久热情，使他的小说成了当代文学变革旅途中的醒目界碑。他从故乡的原始经验出发，抵达的是中国人精神世界的隐秘腹地。他的笔下的欢乐和痛苦，说出的是他对民间中国的基本关怀，对大地和故土的深情感念。他的文字性格既天真又沧桑；他书写的事物既素朴又绚丽；他身上有压抑不住的狂欢精神，也有进入本土生活的坚定决心。这些品质都见证了他的复杂和广阔。从几年前的重要作品《檀香刑》到2003年度出版的《四十一炮》和《丰乳肥臀》，莫言依旧在寻求变化，依旧在创造独立而辉煌的生存景象，他的努力，极大地丰富了当代文学的整体面貌”。<ref name="multiple1_p280">{{cite book |language =zh-hans|author=莫言|title=《蛙》|date=2009-12-01|publisher=上海文艺出版社|page=280|isbn=9787532136766}}</ref> 

*[[诺尼诺国际文学奖|诺尼诺国际文学奖]]：“莫言的作品植根于古老深厚的文明，具有无限丰富而又科学严密的想象空间，其写作思维新颖独特，以激烈澎湃和柔情似水的语言，展现了中国这一广阔的文化熔炉在近现代史上经历的悲剧、战争，反映了一个时代充满爱、痛和团结的生活。”

*[[福冈亚洲文化奖|福冈亚洲文化奖]]：“莫言先生是当代中国文学的代表作家之一，他以独特的写实手法和丰富的想象力，描写了中国城市与农村的真实现状，作品被译成多种语言。莫言先生的作品引导亚洲走向未来，他不仅是当代中国文学的旗手，也是亚洲和世界文学的旗手。”<ref>{{cite web |url=http://news.sina.com.cn/c/2006-07-22/08009534932s.shtml|title=《莫言荣获福冈亚洲文化奖大奖这是继巴金之后我国第二位作家获此奖项》|accessdate=2006年7月22日|work=孙丽 周立 |publisher=新浪网|language=zh-cn}}</ref>

*[[茅盾文学奖|茅盾文学奖]]：“莫言的《蛙》以乡村医生别无选择的命运，折射着我们民族伟大生存斗争中经历的困难和考验。小说以多端视角呈现历史和现实的复杂苍茫，表达了对生命伦理的思考。叙述和戏剧多文本的结构方式建构宽阔的对话空间，从容自由、机智幽默，体现作者强大的叙事能力和执著的创新精神。”<ref>{{cite web |url=http://book.sina.com.cn/news/c/2011-09-20/1517291229.shtml|title=《茅盾文学奖颁奖词公布 对获奖作品给予好评》|accessdate=2011年9月20日|work=喵喵|publisher=新浪网|language=zh-cn}}</ref>

*[[诺贝尔文学奖|诺贝尔文学奖]]：“who with hallucinatory realism merges folk tales, history and the contemporary”（用理想、魔幻的现实主义，将民间故事、历史与现实融合起来的人。他创作中的世界令人联想起[[威廉·福克纳|威廉·福克纳]]和[[加夫列尔·加西亚·马尔克斯|加夫列尔·加西亚·马尔克斯]]作品的融合，同时又在中国传统文学和口头文学中寻找到一个出发点。<ref>{{cite web |url=http://news.eastday.com/c/20121014/u1a6918865.html |title=莫言曾多次反对当地政府修缮其旧居 称劳民伤财 |accessdate= 2012年10月14日|publisher= 东方网|work= |language =zh-cn}}</ref>）<ref name="Nobel Prize"/>
