'''莫俊鹏'''（{{bd|1958年||}}），[[辽宁|辽宁]][[海城|海城]]人，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
是[[中国共产党|中国共产党]]党员。[[中国人民解放军国防大学|中国人民解放军国防大学]]军事战略学专业毕业<ref name=zgrd/>。曾任[[中国人民解放军第二炮兵装备部|中国人民解放军第二炮兵装备部]]副部长、二炮第二十二基地（驻[[陕西|陕西]][[宝鸡|宝鸡]]）司令员、二炮副参谋长、二炮装备部部长等职<ref>{{cite web |url = http://news.takungpao.com/mainland/zgzq/2015-02/2916619.html |title = 二炮22基地司令员莫俊鹏少将改任二炮副参谋长 |publisher = 大公网 |accessdate = 2016-01-12}}</ref><ref>{{Cite web |url = http://mil.sohu.com/20150310/n409581007.shtml |title =莫俊鹏调任二炮党委常委、装备部长  |author =  |date = 2015-03-10 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

2015年12月31日，进入新成立的[[中国人民解放军火箭军|中国人民解放军火箭军]]首任领导班子，出任[[中国人民解放军火箭军装备部|中国人民解放军火箭军装备部]]部长<ref>{{cite web |url = http://news.sina.com.cn/c/nd/2016-01-04/doc-ifxncyar6302118.shtml |title = 火箭军首任11人领导班子曝光 1人由外单位调入 |publisher = 新浪 |accessdate = 2016-01-12}}</ref>。

2008年7月，晋升少将军衔。2013年，当选[[第十二届全国人民代表大会|第十二届全国人民代表大会]]解放军代表<ref name=zgrd>{{cite web |title = 全国人大代表信息-莫俊鹏 |url = http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=122930 |accessdate = 2013-04-05 |website = 中国人大网}}</ref>。
