'''莫绍夫采 (Mošovce)'''是[[斯洛伐克|斯洛伐克]]境内图列茨地区最大的村落。

== 历史 ==

许多历史建筑保存证明这个城市有770多年的历史。在[[国王|国王]][[安德烈二世|安德鲁二世]]的捐赠行为中首次提到. 原本Mošovce是由两个定居点组成:第一,''Machyuch'', 位于今天的''StarýRad'',第二是''TerraMoys'',现在该村仍用此名称，它曾占领地当今Vidrmoch. 即它的名字第二意义''Mojš''的土地。,这使人们相信，全村曾经属于某些''Mojš''先生，他的名字曾被一个缩写的[[斯拉夫人|斯拉夫]]名字''Mojtech''，类似的名称有''Vojtech''或''Mojmír''. 历代名村发生了许多变化，从''Mossovych'', ''Mosocz'', ''Mossowecz''到凤凰山庄，''Mayos Alio Nomine Mossovych''村落到''Oppidioum Mayus Sue Mosocz'',发展成''Mosocz Olim Mayus''的村落现在变成Mošovce. Mošovce分离的古代部分，前遗址''Chornukov''一直保存在现代''Čerňakov''形式.
Mošovce首先作为皇室部落被自由宣传发展，,从14世纪中叶的城市作为特权城[[镇|镇]]归属于Blatnica皇室[[城堡|城堡]]。它在1527年陷入了Révay家族手中，他们镇压城市特权将近四百年.   
过去, Mošovce Turiec是一个重要的工艺中心地区. 工艺经历了惊人传播，约有15个城市设有[[公会|公会]]活动; 皮靴商人和毛皮商人[[公会|公会]]存在时间最长。现在Mošovce被改建成一个有许多景点的重要旅游区。

== 景点 ==

其中最引人注目的是[[不可移动文物|古迹]]—[[洛可可|洛可可]][[古典|古典]]宅邸，18世纪后半期扩建的英格兰式[[公园|公园]]. 其他用地城市包括: JánKollár诞生地，具有宝贵古代前身的古新歌德[[天主教|天主]][[教堂|教堂]]，建于1784年的路德[[教会|教会]]，陵墓改建的工艺[[博物馆|博物馆]],从1800年至今的[[艺术|艺术]][[温室|温室]].

== 大自然 ==

Mošovce周围有很多独特风景. 一个复杂的历史园林和果树园，创造新颖的美学景观，这些风景林地扩大到Veľká Fatra[[山脉|山区]]. 这个山脉是[[斯洛伐克|斯洛伐克]]最吸引人的山脉. 美国海军和[[石灰岩|石灰]]质的奇异造型，以及在Blatnická附近的美丽大自然和Gaderská村落，吸引着来自世界各地的人.

== 文化传统 ==

Mošovce产生了许多重要人物. 最伟大的是Frico Kafenda(1883年至1963年),[[作曲家|作曲家]]。 Lacková安娜-Zora(1899年至1988年)、[[作家|作家]] 。Štefan Krčméry(1892年至1955年)、[[文学评论家|文学评论家]]、[[历史学家|历史学家]]、[[诗人|诗人]]。 Júr Tesák Mošovský、[[巴洛克艺术|巴洛克]][[剧作家|剧作家]]。 Miloslav Schmidt,创办了业余消防队。
但是，也许最重要的人是出生在Mošovce的斯拉夫[[诗人|诗人]]、[[哲学|哲学]][[信義宗|家路德]]Preacher, Ján Kollár(1793年至1852年),他的[[文学|文学]]影响广泛，他的诗剧''Slávy Dcera''至少深刻地影响两个国家。他的作品充满了当代爱国主义与[[民族|民族]]活动家的热情. 已被译成各种[[斯拉夫语族|斯拉夫]]以及非斯拉夫语言。
