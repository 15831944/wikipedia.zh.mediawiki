《'''琢玉诗词'''》（{{lang-th|หยกใสร่ายคำ}}），又译《'''琢玉吟诵'''》<ref>{{cite web |url=http://people.com.cn/GB/channel1/10/20001016/273230.html |title=通讯：中泰一家亲 |last= |first= |date=2000-10-16 |website=人民网 |publisher=孙伟 |access-date=2018-09-19 |quote=}}</ref>，是[[泰国|泰国]][[诗琳通公主|诗琳通公主]]翻译的[[泰语|泰语]]诗集，收录21位中国古代诗人的[[唐诗|唐诗]]和[[宋词|宋词]][[泰文|泰文]]译本34首<ref name="bclc">{{cite web |url=http://bclc-tu.com/yoksai53_page1.html |title=bclc-tu หยกใสร่ายคำ |last= |first= |date= |website=bclc-tu |publisher= |access-date=2018-09-19 |quote=}}</ref>。1998年5月，[[法政大学_(泰国)|泰国法政大学]]艺术系中文专业将诗琳通公主于1980年代学习[[汉语|汉语]]期间翻译的100多首诗词当中，精选出泰国读者熟悉的34首予以出版，将其命名为《琢玉诗词》<ref>{{cite web |url=http://www.sac.or.th/exhibition/princess/writings/yok-sai-rai-kum/ |title=
หยกใส ร่ายคำ (๒๕๔๑) |last= |first= |date= |website=sac |publisher= |access-date=2018-09-19 |quote=}}</ref>；直至2001年，《琢玉诗词》已经再版了5次<ref>{{cite web |url=http://www.people.com.cn/GB/guoji/25/96/20010817/537229.html |title=勤奋学贤  友谊使者——记泰国公主诗琳通 |last= |first= |date=2001-08-17 |website=人民网 |publisher=孙伟 |access-date=2018-09-19 |quote=}}</ref>。也是泰国第一本中国古典诗词译本<ref name="pbic">{{cite web |url=http://www.pbic.tu.ac.th/main/sites/default/files/16_Asscoc.Prof_.Zhang%20Gongchang_CH.pdf |title= 诗琳通公主与中国文化的泰国传播|last= |first= |date= |website= Pridi Banomyong International College |publisher= 张弓长、冯忠芳 |access-date= 2017-12-05 |quote=}}</ref>。

其中诗琳通公主在[[孟浩然|孟浩然]]的《[[春晓|春晓]]》之后创作并加入了三首[[五言绝句|五言诗]]，名为《[[读孟浩然诗感兴|读孟浩然诗感兴]]》（又名：《採桃花》）<ref>{{cite web |url=http://bclc-tu.com/yoksai47_page1.html |title=bclc-tu หยกใสร่ายคำ |last= |first= |date= |website=bclc-tu |publisher= |access-date= 2017-12-05 |quote=}}</ref>。
