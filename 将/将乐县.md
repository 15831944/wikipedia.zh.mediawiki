'''将乐县'''是[[福建省|福建省]][[三明市|三明市]]下辖的一个县。

将乐县人口为17万，有[[汉族|汉族]]、[[畲族|畲族]]、[[回族|回族]]、[[苗族|苗族]]、[[满族|满族]]、[[土家族|土家族]]、[[侗族|侗族]]、[[布依族|布依族]]、[[壮族|壮族]]、[[蒙古族|蒙古族]]、[[彝族|彝族]]、[[仡佬族|仡佬族]]、[[高山族|高山族]]、[[仫佬族|仫佬族]]、[[纳西族|纳西族]]、[[东乡族|东乡族]]等16个民族。

将乐县通行语言为将乐话。

== 历史 ==
将乐置县于[[三国|三国]][[东吴|东吴]][[永安_(東吳)|永安]]三年(公元260年)，属建安郡。[[五代十国|五代十国]][[闽国|闽国]][[天德_(王延政)|天德]]三年(公元945年)，将乐升为州并取名[[镛州|镛州]]。天德四年，废州为县。[[明|明]][[成化|成化]]六年(公元1470年)，将乐县改属延平府。1950年后，将乐先后归属福建省第二专区、南平专区(地区)、三明地区。1983年起，将乐为三明市所辖。

==行政区划==
辖6个镇、7个乡：
*镇：古镛镇、万安镇、高唐镇、白莲镇、黄潭镇、水南镇。
*乡：光明乡、漠源乡、南口乡、万全乡、安仁乡、大源乡、余坊乡。

== 交通 ==
[[福银高速公路|福银高速公路]]贯穿将乐县境，并设有将乐、万安两个互通口。

==旅游==
[[玉华洞|玉华洞]]为当地重要旅游景点，玉华洞景区为第四批[[国家重点风景名胜区|国家重点风景名胜区]]之一。

== 特产 ==
古镛镇的龙池砚为[[中国地理标志产品|中国地理标志产品]]<ref>[http://kjs.aqsiq.gov.cn/dlbzcpbhwz/bhml/200906/t20090622_118997.htm 《地理标志产品保护规定》实施以来批准的地理标志名录（2005年7月至2009年6月）] {{webarchive|url=https://web.archive.org/web/20110311120900/http://kjs.aqsiq.gov.cn/dlbzcpbhwz/bhml/200906/t20090622_118997.htm |date=2011-03-11 }}</ref>。

==文化==
将乐县的竹纸制作技艺被列入[[国家级非物质文化遗产名录|国家级非物质文化遗产名录]]。
