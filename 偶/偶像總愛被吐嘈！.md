== 簡介 ==
{{more plot|time=2015-12-22}}
被春天的溫暖陽光所吸引而在街上隨意閒晃的我，突然被人給綁架了。
咦？什麼？不不不，你們究竟要做什麼啊！？……
當我回過神來之後，發現自己不知為何待在廣播節目的錄音室裡頭。
就在我發著呆時，一名美少女踏進錄音室裡——
天…天啊！這不是我一直很崇拜的人氣偶像配音員音無圓嗎……
咦咦咦咦咦!!
結果，我們倆就這樣順勢開始了廣播節目的工作……但我的命運到底會何去何從!?
以怒濤之勢急遽展開＆熱鬧滾滾的喜劇，在此揭幕！

== 登場人物 ==
; {{nihongo|常村 良人|常村 良人（つねむら りょうと）}}
: 本作[[主角|主角]]。喜好[[日本動畫|動畫]]與[[漫畫|漫畫]]的16歲高中二年級生。平凡且沉默寡言，但卻喜愛[[吐槽|吐槽]]，在因緣巧合之下被誤以為是熱門輕小說《波伊爾・夏璐璐的法則》的原作者與太與作老師，而與其[[日本动画|動畫化]]作品中女主角的配音音無圓共同錄製該作品的網路廣播節目《廣播的法則》。
: 並且，因圓意外成為隔壁鄰居及在校園的相處之後，而得知其不為人知的真面目。
: 在相處之後而了解圓的強韌與柔弱之處，而開始變得喜歡上圓。
: 在圓30歲生日當天向圓求婚成功。
; {{nihongo|音無 圓|音無 まどか（おとなし まどか）}}
: 高中二年級，最初登場時為16歲，在第三卷時迎來17歲生日（6月6日）。<ref>曾在第4卷的彩頁被誤植成16歲，而後在（文庫本）第5卷的第86、87頁以吐槽的方式訂正。原句：「もしくは、音無まどかは永遠十六歳なのかもね。」</ref>
: 年幼時，因著名[[演員|演員]]'''音無源太'''之女的身分，而在劇團活躍，現為人氣廣播劇的聲優。
: 雖有著血親的光環，但也憑藉實力席列聲優之座。有著身為偶像的極高自尊心，完美的笑容總掛在臉上，但實際上的性格卻是意外地[[傲嬌|傲嬌]]。對於工作相關的事情十分地嚴厲，絕不允許任何的失敗。從相關的描述上毫無疑問地對良人抱持著好感。
: 誤飲酒醉後，意外地十分嬌羞且容易哭泣。喜愛的食物是{{link-ja|軟糖|グミ}}。
: 在30歲生日當天接受良人的求婚。
; {{nihongo|天乃川 時雨|天乃川 しぐれ（あまのがわ しぐれ）}}
: 聲優，16歲。與圓屬同事務所的聲優，與圓的關係亦敵亦友。或許是因為成長的家庭環境中，家人皆從事聲優的工作，言談舉止十分文雅。
: 清秀的玉女角色，喜愛魔法童話之類的故事，會把[[萝莉塔风尚|蘿莉塔]]或魔法[[Cosplay|Cosplay]]之類的服裝當成私服穿出外。
: 笑點非常低，常被良人與圓的舉止互動引發爆笑。言行中對良人一樣有好感，行徑顯見。
; {{nihongo|常村 愛好|常村 愛好（つねむら あこ）}}
: 良人的妹妹。13歲，就讀女子學校中學二年級。常配戴[[眼鏡|眼鏡]]，但卻不是因為視力不好。好奇心旺盛，常言及露骨的[[黃色笑話|性話題]]而令良人飽受困擾。以觀察良人與圓的互動關係為樂。
; {{nihongo|霜月 幸春|霜月 幸春（しもつき こはる）}}
: 13歲，中學2年級生。愛好的友人。有著不思而行的隨意性格，相當地[[天然呆|天然呆]]。同時也是極度的路痴。
: 大企業Moon Frost Company的獨生女。<!-- 在海外的小島上有著別墅，也擁有數臺[[加長型禮車|加長型禮車]]的[[痛車|痛車]]。 -->
; {{nihongo|芹澤 優花|芹沢 優花（せりざわ ゆか）}}
: 圓的[[經理人|經理人]]。<!-- 過去曾有過家庭，因成為音無源太的[[粉絲|粉絲]]而作為經理人踏入演藝圈。平常雖有著女性的一面，但也會因笨拙的料理能力而出現恐怖的性格。 -->
; {{nihongo|嘉保 芽愛璃|嘉保 芽愛璃（かほう めあり）}}
: 高校2年級，16歲。新聞部的部長。擅長無聲無息的動作模式，使被拍照的本人也不被發現。
: 有著京都腔<ref>關西腔的一種，詳見條目：[[近畿方言|近畿方言]]</ref>，無法自拔地喜愛男性身體的照片。
; {{nihongo|與太與作|与太与作（よたよさく）}}
: 輕小說《波伊爾・夏璐璐的法則》的作者。

== 用語 ==
; {{nihongo|波伊爾・夏璐璐的法則|ぼいる・しゃるるの法則}}
: 作品中的人氣輕小說。有關於「發熱少年波伊爾」（{{lang|ja|発熱少年ぼいるくん}}）和「微熱少女夏璐璐」（{{lang|ja|微熱少女しゃるるちゃん}}）的故事。圓在動畫版中擔任夏璐璐配音的角色。<ref>詳見文庫本第1卷第13、16頁。</ref>
; {{nihongo|廣播的法則|ラジオの法則}}
: 作品中伴隨著人氣輕小說《波伊爾・夏璐璐的法則》動畫版而開始的網路廣播節目。每週四更新放送。錄音時間為週六。
; {{nihongo|古里學園|古里學園（ふるさとがくえん）}}
: 良人他們所就讀的高中，每個學級有六個班。距離良人家步行約十五分鐘可達。<ref>詳見文庫本第1卷第91、94頁。</ref>
; {{nihongo|Moon Frost Company|ムーンフロストカンパニー}}
: 霜月幸春的父親所開設的公司，世界級的大企業。圓所獻聲的動畫片及廣播節目廣告的贊助商。
; {{nihongo|All Make World|オール・メイク・ワールド}}
: 《廣播的法則》的贊助商，《波伊爾・夏璐璐的法則》的出版社。
; {{nihongo|私立葛蕾絲女學院國中部|私立グレイス女學院中等部}}
: 良人的妹妹愛好與幸春所就讀的中學。有錢人家大小姐眾多的貴族學校。
