《'''-{zh-hans:偶像宣言;zh-hk:花漾明星;zh-tw:偶像宣言;}-'''》（{{lang|ja|きらりん☆レボリューション}}）又譯為《'''-{zh-hans:星梦天使;zh-hk:偶像宣言;zh-tw:星夢天使;}-'''》或《'''-{花漾明星}-KIRARIN'''》，是[[日本|日本]][[少女漫畫|少女]][[漫畫家|漫畫家]][[中原杏|中原杏]]所創作的[[漫畫|漫畫]]，以及依此改編的[[電視動畫|電視動畫]]與[[電子遊戲|遊戲]]。

== 概要 ==
本作品漫畫版於2004年2月開始在[[小學館|小學館]]的[[漫畫雜誌|漫畫雜誌]]《[[Ciao|Ciao]]》（{{lang|ja|[[:ja:ちゃお|ちゃお]]}}）3月號連載，漫畫單行本共14本。於2006年度（平成18年度）奪得第52回[[小學館漫畫賞|小學館漫畫賞]]<ref>小學館漫畫賞（原文：{{lang|ja|小学館漫画賞}}）是日本小學館對連載於自家漫畫雜誌所評選頒發出來的獎賞，第一回獎賞成立於1955年。可以參考下列網站：{{ja}}[http://www.shogakukan.co.jp/mangasho/ 小學館漫畫賞公佈網頁]、{{ja}}[http://comics.shogakukan.co.jp/mangasho/rist.html 小學館漫畫賞歷代受賞者庫存頁]  {{webarchive|url=https://www.webcitation.org/5msLIPidX?url=http://comics.shogakukan.co.jp/mangasho/rist.html |date=2010-01-18 }}</ref> 兒童部門獎。

動畫版《花漾明星KIRARIN》於2006年4月7日於日本[[東京電視台|東京電視台]]播映，結束於2009年3月27日，全153話。其他地區的電視台播放日期與進度，請參閱本頁最下方的外部連結。

== 故事簡介 ==
'''月島星璃'''是一位對感情一竅不通14歲的中學二年級生，有一天星璃與人氣偶像團體『SHIPS』的'''日渡星司'''意外相遇並對他一見鍾情。為了想接近星司而一心一意地想成為偶像的星璃做出屬於自己的'''-{偶像宣言}-'''。雖然最初覺得很不容易，但得到大家的支持而努力下去。後因工作不斷增加的關係，轉校到藝能學校，並認識更多朋友和勁敵。但在日後的工作相處之下，面對SHIPS另一成員風真宙人的幫助，竟發現自己也好像喜歡上了宙人。弄不清楚到底喜歡誰的星璃，在矛盾的心情下又該如何判斷自己的心意呢？面對了不同的相遇和離別的她又能不能成為NO.1旳偶像？

== 惡搞 ==
曾有網友將日本版第二片頭曲『バラライカ』MV改編，將MV中久住小春的聲音換成日本網路歌手いさじ的聲音並修改部分歌詞，尤其將原曲中的バラライカ換成了發音類似，出自[[日本|日本]][[漫畫|漫畫]][[瞎攪和的技術|瞎攪和的技術]]中阿部高和的著名台詞やらないか(中文翻譯為不做嗎?)，並放上了[[NICONICO|NICONICO]]造成了轟動，甚至改編後的版本還遠比久住小春唱的原始版本還有名，也因此被網友揶揄她是因[[NICONICO|NICONICO]]而導致的最可憐受害者。<ref>日本網友惡搞的影片版本[https://www.youtube.com/watch?v=9J65gbF8pdk バラライカ　いさじVer]</ref>
