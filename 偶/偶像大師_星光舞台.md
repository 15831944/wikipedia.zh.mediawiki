《'''偶像大師 星光舞台'''》（日語：{{lang|ja|アイドルマスター ステラステージ}}、''THE IDOLM@STER STELLA STAGE''），是一款由{{link-ja|萬代南夢宮工作室|バンダイナムコスタジオ|萬代南夢宮工作室}}開發的偶像養成遊戲。

遊戲為[[偶像大师系列|偶像大师系列]]作品之一，也是繼《[[偶像大師_白金星光|偶像大師 白金星光]]》之後，第二款在[[PlayStation_4|PlayStation 4]]平台上獨佔發行的系列作品。於[[2017年|2017年]][[12月21日|12月21日]]在[[日本|日本]]、[[大韩民国|韓國]]、[[臺灣|台灣]]、[[香港|香港]]四地同步發售<ref>{{Cite web |url = https://www.4gamer.net/games/395/G039553/20171118015/|script-title  = ja : 「アイドルマスター ステラステージ」のプレイレポートをあなただけにお届け。春香と一緒にレジェンドアイドルを目指してみた|author = 松本隆一|date = 2017-11-22|publisher = 4Gamer.net|work = PS4|language = ja|accessdate = 2019-04-09}}</ref><ref>{{Cite web|url = https://www.4gamers.com.tw/news/detail/33196/ps4-idolmaster-traditional-chinese-version-coming-at-december|title = TGS17｜PS4《偶像大師 星光舞台》繁中版12月底台日同步發售|author = 歪力|date = 2017-09-22|publisher = 4Gamers|work = 遊戲資訊|language = zh-tw|accessdate = 2019-04-09}}</ref><ref>{{Cite web |url = https://gnn.gamer.com.tw/6/155676.html|title = 【試玩】《偶像大師 星光舞台》繁體中文版搶先體驗 整理新作改進要素|author = 阿爾基諾|date = 2017-11-24|publisher = 巴哈姆特|work = GNN新聞|language = zh-tw|accessdate = 2019-04-09}}</ref>。

== 概要 ==
玩家在遊戲中將扮演「765事務所製作人」一役，培養事務所旗下的各位偶像。在本作中，玩家將以舉行由互為對手的765事務所和961事務所的兩位社長過去共同籌劃但最終擱淺的「星光舞台」（日語：{{lang|ja|ステラステージ}}，即''Stella Stage''）為目標。本作遊戲系統在沿襲《白金星光》遊戲系統基礎之上，還新增了「技能訓練」、「休息」等新系統<ref>{{Cite web|url = https://www.famitsu.com/news/201711/22146239.html|script-title=ja : 『アイドルマスター ステラステージ』編集部のガチPによる先行プレイリポート――如月千早の序盤のストーリーも独占公開|author = {{lang|ja|北埜トゥーン}}|date = 2017-11-22|publisher = {{lang|ja|ファミ通.com}}|language = ja|accessdate = 2019-04-09}}</ref>。

本作中的3D效果製作採用了MATCHLOCK股份有限公司（日語：{{lang|ja|マッチロック株式会社}}）的軟件BISHAMON<ref>{{Cite web |url = https://game.watch.impress.co.jp/docs/news/1099210.html|script-title  = ja : マッチロック、リアルタイム3Dエフェクト「BISHAMON」が「アイドルマスター ステラステージ」に採用されたことを発表|author = 今藤祐馬|date = 2017-12-27|publisher = GAME Watch|language = ja|accessdate = 2019-04-09}}</ref>。

== 遊戲系統 ==
在本作中，通過舉辦演唱會可獲得訓練點數（日語：{{lang|ja|コーチングポイント}}，即Coaching Point，簡稱CP），在類似桌遊棋盤的技能訓練板上使用訓練點數解鎖訓練方塊，偶像便可習得該方塊的技能，令各方面的能力得到提升。通過不斷解鎖已解鎖方塊的相鄰訓練方塊，還可以解鎖新樂曲、獲得偶像服飾等<ref>{{Cite web |url = http://dengekionline.com/elem/000/001/610/1610807/|script-title  = ja : 『アイマス ステラステージ』詩花のキャストが高橋李依さんに決定。新要素“コーチング”の詳細が判明|date = 2017-10-16|publisher = {{lang|ja|電撃オンライン}}|language = ja|accessdate = 2019-04-10}}</ref>。

偶像每次活動后積累的疲勞越多，演唱會氣氛將越來越難以高漲起來，此時可通過休息進行回復，休息時玩家所扮演的製作人將與偶像相約去動物園或游泳館等地遊玩消解疲勞。

本作可以繼承前作《白金星光》DLC創刊號到6號的樂曲、服裝、飾品等部分[[可下載內容|可下載內容]]。
