== 游戏系统 ==

游戏以增加剧场的知名度以及成为“顶级偶像”为目标。其主要游戏模式由节奏打击类型音乐游戏的“Live”和[[情境猜谜|情境猜谜]]类型的“工作”组成，通过这两种模式可以获得多种报酬。进行游戏需要“体力”，而体力会随着现实时间逐渐回复。

; Live
: Live是上方落下式的节奏打击游戏。分为单人Live和多人Live两种，前者需要手机竖屏操作，后者则和游戏大多数时间一样是横屏。每次Live的都需要五个偶像角色组队，不过单人Live时只有center会参演。
: 游戏中，屏幕下方有多个打击点，需要对应上方落下的不同音符来操作。最简单的圆形音符只需要在和打击点重叠时点击，除此之外还有滑条（需按住不放）、箭头（需要按照方向滑动）等。打击点的个数以及落下音符的数量/速度根据难度不同而变化，最多会有6个打击点。
:; Live服装
:: 进行Live或者观看音乐视频时，每个角色的服装可以选择。初期仅有“Shiny Trinity”服装，可以从Live的报酬和抽卡等获得新服装。 
; 工作
: 和Live不同，工作更接近于[[视觉小说|视觉小说]]游戏。需要和偶像进行对话来获取报酬。工作会随机出现不同类型的“Communication”，其中又是会有问答环节，回答正确可以获得更高奖励。
; 剧场
: 游戏的舞台。分为四个场景：入口，事务室，休息室和更衣室。在各个房间可以和偶像交流对话。其中部分对话可以和Live、工作一样，增加偶像的亲爱度。在不同的时间，也会有不同的场景出现（例如庆祝生日等）。另外，游戏中也可以查看偶像的Blog、短信交流等。

除此之外，玩家亦可通过Lounge和其他玩家好友进行聊天，或者互相赠送花架等。

== 活动 ==
目前为止，有以下几种活动定期举行。

; THEATER SHOW TIME☆
: 每天通过Live和工作获取不同的额外物品。

; MILLIONLIVE COLLECTION
: 在活动期间会掉落特殊的扭蛋硬币，可以抽取限定偶像以及物品。

; Platinum Star Theater
: 追加新乐曲（之前发售过的歌曲）的活动。活动期间通过进行Live和工作可以获取活动专用道具以及点数。使用道具可以进行活动歌曲的Live（同时获得更多活动点数）。随着点数增加可以解锁活动剧情以及入手道具和限定偶像。
: 另外，活动结束后，根据点数的排名、活动歌曲分数的排名可以而获得不同的奖励。活动乐曲会在结束后追加到Live的歌曲列表中。
; Platinum Star Tour
: 追加新乐曲（全新歌曲）的活动。从活动准备画面选取特定的Live和工作，从而获得专用点数和道具。道具可以用来进行活动乐曲的Live。
:和Platinum Star Theater一样，活动结束后，根据点数的排名、活动歌曲分数的排名可以而获得不同的奖励。活动乐曲会在结束后追加到Live的歌曲列表中。
