'''鄒泰'''，一作'''周泰'''，[[清末|清末]][[廣東|廣東]][[湛江|湛江]]武術家，外號「'''鐵頭'''」（'''周鐵頭'''），以[[八卦棍|八卦棍]]、[[軟綿掌|軟綿掌]]馳名，有「'''嶺南棍王'''」的稱號。<ref name="趣談中">{{cite web |url=http://lsw1230795.mysinablog.com/index.php?op=ViewArticle&articleId=1179109 |title=廣東十虎趣談 (中) |author=山中人 |publisher=山中札記 |date=2008-05-29 |language = |accessdate= }}</ref><ref>{{cite web |url=http://only3m.pixnet.net/blog/post/29644917 |title=廣東十虎 |author=only3m |publisher=痞客邦 |date=2009-10-26 |language = |accessdate= }}</ref>

按[[朱愚齋|朱愚齋]]所說，鄒泰是[[廣東十虎|廣東十虎]]之一。但依照[[我是山人|我是山人]]（陳勁）在武俠小說《[[武壇二虎|武壇二虎]]》之自序所指，「廣東十虎」中有[[黃飛鴻|黃飛鴻]]而無周泰。<ref name="十虎世榮">{{cite web |url=http://bbs.cantonese.asia/thread-891-1-1.html |title=廣東十虎 與 林世榮 |author=Kathy |publisher=粵語協會 |date=2005-05-16 |language = |accessdate= }}</ref><ref name="二虎自序裡">{{cite web |url=http://blog.xuite.net/joekuo888/twblog/122435046 |title=武俠小說《武壇二虎》自序裡的廣東十虎 |author=joekuo999 |publisher=隨意窩 |date=2008-05-27 |language = |accessdate= }}</ref><ref name="趣談上">{{cite web |url=http://lsw1230795.mysinablog.com/index.php?op=ViewArticle&articleId=1177287 |title=廣東十虎趣談 (上) |author=山中人 |publisher=山中札記 |date=2008-05-28 |language = |accessdate= }}</ref>
