'''况浩文'''（{{bd|1930年||2018年|1月28日}}），[[重庆|重庆]]人，[[中华人民共和国|中华人民共和国]]作家、官员。

==生平==
况浩文是[[中国共产党|中国共产党]]党员。大学文化。出身地主家庭。参加革命后，自[[西南人民革命大学|西南人民革命大学]]一期毕业，在[[西南军政委员会|西南军政委员会]]公安部五处（边防保卫处）以及[[西南军区|西南军区]]公安部队司令部侦察处工作。1955年因身体不好，转业到重庆市第二工业局工作，任党组秘书。1959年任重庆市化工局秘书副科长。[[文革|文革]]中遭受长期迫害。平反后历任重庆市外贸局局长兼[[党组|党组]]书记，重庆市外经委主任兼党组书记，重庆长发集团公司董事长。[[中共重庆市委|中共重庆市委]]第五、六届委员。高级经济师。1956年起开始文学创作。2003年加入[[中国作家协会|中国作家协会]]。他写的小说《[[一双绣花鞋|一双绣花鞋]]》1958年创作完稿后，在文革期间通过手抄本形式在中华人民共和国国内广为传抄。1979年，重庆文学杂志《红岩》复刊后，《一双绣花鞋》首次正式发表在《红岩》。2002年，[[重庆出版社|重庆出版社]]推出《一双绣花鞋》单行本。文革后，《一双绣花鞋》被改编为电影《雾都茫茫》，后来又两次被改编为电视连续剧《C—3计划》及《一双绣花鞋》<ref name=pp>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1972752  |title =  88岁著名作家、手抄本小说《一双绣花鞋》作者况浩文去世|author =  |date = 2018-01-28 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.chinawriter.com.cn/zxhy/member/1833.shtml|title =况浩文 (1930～)  |author =  |date =  |publisher = 中国作家网 |language =  |accessdate =2018-01-29  |quote =  }}</ref><ref>{{Cite web |url = http://news.163.com/11/0130/10/6RKUSR7C00014JB6_all.html |title =“文革地下文学第一书”灵感来自作者亲身经历  |author =  |date =2011-01-30  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2018年1月28日，况浩文在重庆病逝，享年88岁<ref name=pp/>。
