== 登場角色 ==
===主要人物<ref>{{Cite web |date=2017-07-21 |url=http://www.oricon.co.jp/news/2094474/full/|title=篠原涼子主演・秋の“月9”、『民衆の敵』にタイトル決定 出演者追加発表  |publisher=[[ORICON_NEWS|ORICON NEWS]] |accessdate=2017-07-24}}</ref> ===
*佐藤智子：[[篠原涼子|篠原涼子]]（幼年：{{link-ja|吉澤梨里花|吉泽梨里花}}）
:青叶市议会新进议员，隶属于教育委员会，不喜欢咖啡。因工作中被投诉未妥善处理而被辞职。<br>因为议员的高薪酬而参选,认为得到高薪，家人就会变得幸福。<br>参与市议会选举竞选时，原本落选但因矶部因病放弃参选而成功进入议会。后加入教育委员会，属于犬崎派。<br>後因成為青葉市市長與犬崎派反目。
*藤堂誠：[[高橋一生|高橋一生]]
:青叶市议会新进议员。日本政民党公认的议会选举候选人，以投票率第一当选。<br>出生於政治家族，曾祖父是原防卫大臣，父亲是日本政民党的泰斗，哥哥是众议院议员。大学政治学毕业后，曾在哥伦比亚大学留学。<br>私下以“渡边”为名。有不想當政治家的時候，有著其他人不知道的另一面。最終回和智子反目並參與國政。
*平田和美：[[石田百合子|石田百合子]]
:智子的朋友。千叶新闻社記者。喝醉时经常大笑。<br>隶属于政治新闻部，产假时人事调动至人事部。之后向人事部提交了“关于产假前后的女性职员的分配的调查报告书”，再次调配至文化部。<br>在记者活动的同时，一边收集信息一边给予智子建议。最終回重返政治部後轉職自由記者。

===青葉市議會===
*犬崎和久：[[古田新太|古田新太]]
:市議會議員。議會最大黨的代表，教育委员会的委员长。
*河原田晶子：[[余貴美子|余貴美子]]
:青葉市市長。女性充憧的對象,和犬崎對立。<br>对于议会半数为犬崎派的问题并不关心，首次进行了各派系议席随机分配的改革。<br>将犬崎比喻为沟鼠，反对犬崎的新港口计划。<br>後因犬崎和富田使詐被迫下台，後於最終回復職。
*小出未亞：[[前田敦子|前田敦子]]
:青叶市议会新进议员，与智子一同当选。<br>前寫真偶像，十分擅長SNS。原属于犬崎派，後於故事尾段退出全力支持智子。<br>与岡本遼为同一国中的学妹，相差2岁。
*岡本遼：[[千葉雄大|千葉雄大]]
:青叶市议会新进议员。共進党公认的议会选举候选人，受河原田推荐。<br>對不熟悉政治的智子略不滿意。
*園田龍太郎：{{link-ja|齋藤司 (搞笑藝人)|斎藤司 (お笑い芸人)|齋藤司}}
:青叶市议会新进议员，農民家族3兄弟的老大。為人沒有主見，會依賴他人。
*若宮寛：{{link-ja|若旦那|若旦那 (ミュージシャン)}}
:犬崎的秘書。
*前田康：{{link-ja|大澄賢也|大澄賢也}}
:市議會资深议员,犬崎派的2号人物。<br>後於最終回退出犬崎派。
*望月守：{{link-ja|細田善彦|細田善彦}}
:河原田的秘書，是[[同性戀|同性戀]]者。
*安部崇裕：[[久保田悠来|久保田悠来]]
:市議会事務局職員。
*富田恭一：{{link-jp|渡邊一惠|渡辺いっけい}}
:前福利部課長，後成為智子秘書。實為犬崎的心腹，河原田的下台和智子的醜聞也是他無中生有造成。<br>最終回因收到藤堂誠的巨款報酬而拍影片公開自己和犬崎的胡作非為還智子清白，化解了她的下台危機。
*青叶市議会議長：{{link-ja|川守田政人|川守田政人}}
