'''民族和解党'''是[[冈比亚|冈比亚]]的一个政党。它成立于1996年，由创始人{{le|哈马特·巴赫|Hamat Bah}}领导。

== 历史 ==
民族和解党是{{le|哈马特·巴赫|Hamat Bah}}在1996年成立的，作为他竞选总统的一部分。他在1996年的总统选举中获得第三名。2名该党成员在1997年议会选举中当选。在2001年的总统选举中，巴赫名列第三。民族和解党作为{{le|联盟2016|Coalition 2016}}的一部分参加2016年总统选举，该联盟宣布[[阿达马·巴罗|阿达马·巴罗]]为联盟的候选人，随后获胜。<ref>{{cite news|url=http://gainako.com/gambian-opposition-parties-set-select-single-candidate-december-polls-convention/ |title=Gambian Opposition Parties Set to Select a Single Candidate for December polls at a Convention |website=GAINAKO |date=30 October 2016}}</ref>

== 选举结果 ==
该政党的选举结果如下：
*在1997年议会选举中赢得48个席位中的2个
*在2002年议会选举中赢得48个席位中的1个
*在2007年议会选举中，没有赢得席位
*在2012年议会选举中赢得48个席位中的1个（加上中期选举中的1个席位）
*在2017年议会选举中赢得53个席位中的4个
