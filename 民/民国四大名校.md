'''民国四大名校'''一般指[[中华民国|中华民国]]大陆時期的四所著名高校，即[[國立中央大學|國立中央大學]]、[[國立浙江大學|國立浙江大學]]、[[國立西南聯合大學|國立西南聯合大學]]（[[國立北京大學|國立北京大學]]、[[國立清華大學|國立清華大學]]、[[南開大學|私立南開大學]]）和[[國立武漢大學|國立武漢大學]]<ref name="Chinese Education in the War">{{cite book |title=Chinese Education in the War  |location = Taipei |publisher = Kelly & Walsh; Ch'eng Weng Publishing Company |last = Freyn |first = Hubert |year=1940 |page =105}}</ref>。

==缘起与背景==
1937年，[[国立中央大学|国立中央大学]]、[[国立北京大学|国立北京大学]]、[[国立清华大学|国立清华大学]]、[[国立浙江大学|国立浙江大学]]、[[国立武汉大学|国立武汉大学]]五所中国顶尖学府在全国进行统一招生考试，简称五大名校联考。此即'''民国五大名校'''名称的来历（抗战时期清华、北大、南开合并为[[国立西南联合大学|国立西南联合大学]]）<ref name="民国名牌大学有哪些">{{cite web |url = http://cn.rocidea.com/roc-499.aspx |title=民国名牌大学有哪些 |accessdate=2012-03-21}}</ref>。

七八月间，各地试卷集中于南京中央大学校园评阅。时有日本飞机到[[南京|南京]]轰炸，阅卷工作受到极大的骚扰。特别是参加阅卷工作的教授的人身安全受到威胁。负责组织这次评阅试卷的国立中央大学校长[[罗家伦|罗家伦]]、国立浙江大学校长[[竺可桢|竺可桢]]及国立武汉大学的[[曾昭安|曾昭安]]等人安排阅卷完的教授们尽快离开南京。8月19日19时前后，日机飞临南京上空，日机的轰炸目标有中央大学图书馆、大礼堂和科学楼，图书馆周围房屋的窗户玻璃全被震碎，礼堂的后墙被炸毁，科学楼没有击中而炸毁了化学实验室，猛烈的轰炸使国立中央大学女生宿舍倒塌。由此拉开了迁校序幕。国难当头，国立北京大学、国立清华大学与[[南开大学|私立南开大学]]合并组建[[国立西南联合大学|国立西南联合大学]]，最后迁至[[云南|云南]][[昆明|昆明]]，国立中央大学迁至[[重庆|重庆]]，国立浙江大学迁至[[贵州|贵州]][[遵义|遵义]]，国立武汉大学迁至[[四川|四川]][[乐山|乐山]]。

==八年抗战==
五大名校抗战时坚持办学，为20世纪中华民族之光；他们在大西南的岁月是各自历史上最辉煌，最可歌可泣的时光。

==师资力量==
1934年，上述五大名校教授总数如下<ref name="中国大学的去向：基于民国大学史的观察">{{cite web |url = http://news.ifeng.com/history/special/minguodaxue/201002/0203_9438_1536273_1.shtml |title = 中国大学的去向：基于民国大学史的观察 |publisher=凤凰网 |accessdate = 2012-10-21}}</ref>：中央大学197人（副教授0人），清华大学87人（副教授16人），北京大学56人（副教授11人），浙江大学2人（副教授75人），武汉大学80人（副教授0人）。

==復校及其他==
1948年，英国[[牛津大学|牛津大学]]致函[[中华民国教育部|中华民国教育部]]，确认包括[[國立中央大學_(南京)|国立中央大学]]、[[北京大学|国立北京大学]]、[[清华大学|国立清华大学]]、[[浙江大学|国立浙江大学]]、[[国立武汉大学|国立武汉大学]]、[[私立南开大学|私立南开大学]]以及[[北京协和医学院|协和医学院]]的文理科学士毕业生成绩平均在八十分以上者，享有“牛津之高级生地位”（即今之大学四年级学生）<ref name="民国名牌大学有哪些" /><ref name="武大校史之信步杂谈">{{cite web |url = http://manage.future.org.cn/article/53646.htm |title=武大校史之信步杂谈 |accessdate=2012-03-21}}</ref><ref name="校名最讲究的八所大学">{{cite web |url = http://sc.sina.com.cn/edu/info/gaokao/2011-10-18/084114789_2.html |title=校名最讲究的八所大学 |accessdate=2012-03-21}}</ref>。

==社會評價==
這些學校學術水平之高，現在看來也嘆為觀止。如浙江大學被英國學者在《自然》雜誌​​上稱為“東方[[劍橋|劍橋]]”，就是因為有一批優秀教授做出了一流成果，在極其艱苦的日子裡，在《自然》、《科學》等頂尖學術刊物上發表了10餘篇高品質文章。1945年至1952年時期畢業的學生後來成為中國院士的學者最多也出自這幾所學校，如中科院院士前五名為[[清華大學|清大]]25人、[[國立浙江大學|浙大]]22人、[[國立中央大學|中大]] 21人、[[國立北京大學|北大]]14人、[[國立武漢大學|武大]]12人。

抗日戰爭結束後，1948年中華民國政府遴選了第一批[[中央研究院|中央研究院]]院士，其中北京大學、清華大學、浙江大學、中央大學入選教授最多，名列前四強。
