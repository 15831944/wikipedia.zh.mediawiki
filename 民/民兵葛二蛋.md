《'''民兵葛二蛋'''》，2012年上映的抗日战争题材电视剧，导演韦大军、林柯，主演[[黄渤|黄渤]]、[[高虎|高虎]]、[[童瑶|童瑶]]、[[阎娜|阎娜]]、[[汤晶媚|汤晶媚]]等。2012年9月8日在深圳电视台都市频道播出，同年12月12日在[[浙江卫视|浙江卫视]]、[[北京卫视|北京卫视]]、[[湖北卫视|湖北卫视]]、[[河南卫视|河南卫视]]上星。該劇獲[[2013国剧盛典|2013国剧盛典]] 《年度十佳電視劇榮譽》第十位。

== 剧情 ==
1943年，[[抗日战争|抗日战争]]已经接近尾声，小街镇塔湾村的葛二蛋和麦子两结义兄弟为了找[[日军|日军]]报仇雪恨，最终失散，葛二蛋目睹[[八路军|八路军]]抗日事迹而成了一名民兵，麦子因为抵挡不住金钱和高享受物质生活的诱惑而蜕变为日治地区伪军的士兵……

== 花絮 ==
黄渤的作品《[[泰囧|泰囧]]》和《民兵葛二蛋》在2012年岁末上映，都获得高收视率。<ref>{{cite web |url=http://lady.163.com/12/1226/08/8JKUQGVR00264M4F.html |title=《民兵葛二蛋》爆笑荧屏 黄渤：狗屎运都往我头上掉 |accessdate=2012年12月26日 |publisher= 网易|work= |language =zh-cn}}</ref>

演员王迅是黄渤工作室的成员，在该剧中颇为抢戏。<ref>{{cite web |url=http://ent.163.com/12/1226/03/8JKCKV2700032DGD.html |title=《民兵葛二蛋》浙江热播 王迅黄渤荧屏抢戏 |accessdate=2012-12-26  |publisher=网易 |work= |language =zh-cn}}</ref>

韩童生出演幽默的村长。<ref name="cat " >{{cite web |url=http://gb.cri.cn/27564/2012/12/26/1042s3971558.htm |title= 《民兵葛二蛋》村长笑点十足 韩童生接地气|accessdate=2012 |publisher= |work= |language =zh-cn}}</ref>

该剧没有“激情戏”戏份。<ref>{{cite web |url=http://et.21cn.com/gundong/etscroll/2012/12/26/14171872.shtml |title=汤晶媚《葛二蛋》热播 痴情良子无奈为爱献身 |accessdate=2012年12月26日 |publisher=21CN |work= |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20141012021218/http://et.21cn.com/gundong/etscroll/2012/12/26/14171872.shtml |archivedate=2014年10月12日 |df= }}</ref>

该剧中有黄渤和童瑶骑驴的搞笑镜头。<ref>{{cite web |url=http://lady.163.com/12/1225/13/8JIU2PFN00264M4F_all.html |title= 童瑶黄渤《民兵葛二蛋》上演骑驴搞笑戏码(图)（全文）|accessdate= 2012年12月25日|publisher=网易 |work= |language =zh-cn}}</ref>

袁满出演刘排长在该剧中暗恋童瑶出演的孟喜子，和黄渤一起上演“[[三角恋|三角恋]]”。<ref>{{cite web |url=http://et.21cn.com/tv/roll/2012/12/24/14145749.shtml |title=《民兵葛二蛋》飘红 袁满黄渤深陷纠结三角恋 |accessdate=2012-12-24 |publisher=21CN |work= |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20160304131925/http://et.21cn.com/tv/roll/2012/12/24/14145749.shtml |archivedate=2016年3月4日 |df= }}</ref>而王晴和韩童生则在该剧中上演“[[黄昏恋|黄昏恋]]”。<ref>{{cite web |url=http://ent.ifeng.com/zz/detail_2012_12/24/20472227_0.shtml |title=王晴屎蛋娘悲情动人 《民兵葛二蛋》出彩 |accessdate=2012 |publisher=凤凰网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://news.xinhuanet.com/ent/2012-12/21/c_124127071.htm |title=《民兵葛二蛋》热播 韩童生遭遇“黄昏恋” |accessdate=2012年12月21日 |publisher=新华网 |work= |language =zh-cn}}</ref>

演员浅野长英是中日混血儿，他表示不希望只出演日本兵。<ref>{{cite web |url=http://news.hexun.com/2012-12-16/149105611.html |title=日本军官专业户 《民兵葛二蛋》中吉田再出彩 中日混血演员接受专访 称好人坏人都要演出人性 浅野长英：不想只演日本兵 |accessdate=2012年12月16日 |publisher=和讯网 |work= |language =zh-cn}}</ref>
