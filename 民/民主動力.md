'''民主動力'''，[[香港|香港]][[泛民主派|泛民主派]]組織之一，於2002年成立，由多位關心香港民主政制發展的學者、專業人士及政黨成員組成的民間組織，為香港各民主黨派提供一個交流和合作的平台，多次協調民主派參加立法會及區議會選舉。此外，亦為香港民主發展相關議題進行學術研究，並積極參與和籌劃多項社會行動及公眾教育，致力推動香港的政制民主化改革 。<ref name=PfDfb>[https://www.facebook.com/pg/PowerForDemocracy/about/?ref=page_internal 民主動力 Power for Democracy]</ref>

[[2011年香港區議會選舉|2011年香港區議會選舉]]曾策劃和參與[[泛民區選聯盟|泛民區選聯盟]]。2013年3月21日，民主動力聯同其他香港泛民主派政黨及團體成立[[真普選聯盟|真普選聯盟]]。<ref >{{Cite web | title = 政情：民主動力 變爛攤子 | work = 東方日報 | date =  2014年11月16日| url = http://orientaldaily.on.cc/cnt/news/20141116/00176_102.html }}</ref>

2015年發起91名民選議員及增選委員聯署反對[[2016年及2017年香港政治制度改革|2016年及2017年香港政治制度改革]]。<ref name=>{{Cite web | title =  泛民示威：區會主席不代表我 | work = 明報| date = 2015-06-01 | url = http://life.mingpao.com/cfm/dailynews3b.cfm?File=20150601/nalga/gae1.txt |  }}</ref>

2018年1月14日，為決定[[2018年香港立法會補選|2018年香港立法會補選]]的九龍西和新界東人選，舉行民主派初選實體投票<ref>{{Cite web | title =  民主動力：半日逾萬人投票超預期 | work = 明報| date = 2018-01-14 | url = https://news.mingpao.com/ins/instantnews/web_tc/article/20180114/s00001/1515922450870 |  }}</ref>。
