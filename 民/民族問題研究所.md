'''民族問題研究所'''（{{lang-ko|민족문제연구소}}）是[[韩国|韩国]]的[[民间团体|民间团体]]，是韩国为继承追查、研究[[亲日派|亲日派]]者[[林鍾国|林鍾国]]<ref name="shibun-9">[http://japan.donga.com/srv/service.php3?biid=2004012826658 政府、「親日人名辞典」募金を許可(東亜日報記事)]に林鍾國の解説がある。</ref>的遗志而设立的机构。目前由[[任軒永|任軒永]]担任所长。

== 设立目标 ==
民族問題研究所因以下几点原因而设立：
# 研究韩国近现代史中有争议的话题、对韩日间的历史问题进行清算。
# 负责编纂《[[親日人名辞典|親日人名辞典]]》、对韩国国内的日本法西斯残余势力（即所谓的“日帝残滓”）进行清算。

参看研究所《民族问题解决的重点在于对曲折的民族历史进行清算》、《推行[[正确历史教育_(韩国)|正确历史教育]]、复兴民族正气的重要性》中的说明。（正文在外部链接中研究所的网站可以查看）

== 沿革 ==
* 1991年 反民族問題研究所设立。
* 1995年 改名为社团法人民族問題研究所。
* 2001年 由统一时代民族文化集团发起的[[親日人名辞典編纂委員会|親日人名辞典編纂委員会]]成立。
