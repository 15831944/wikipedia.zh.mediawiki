《'''甜蜜懲罰～我是看守專用寵物'''》（{{lang-ja|甘い懲罰〜私は看守専用ペット}}）是日本漫畫家いづみ翔的漫畫作品。於2016年6月3日以“「このままじゃ…イク…」看守の執拗な身体検査”為標題，在網站めちゃコミック上開始連載<ref>{{Cite web |author= いづみ翔|date= 2016-06-03|url= http://blog.goo.ne.jp/abundance8888/e/3014198bfcf2ef9bdbc29265a987feee|title= ブログ更新「新作配信始まりました」|work= いづみ翔ブログ|publisher= gooブログ|accessdate=2018-02-28}}</ref>。以單行本發行的同時將作品名字改爲『甜蜜懲罰～我是看守專用寵物』<ref name="anime">{{Cite web |date= 2018-02-13|url= http://kk1up.jp/archives/20180213_kanshu.html|title= 【アニメ】“ドS看守ブーム”巻き起こす『甘い懲罰』が待望のアニメ化　「通常版」と過激シーン収録「完全版」のW放送|work= 1up情報局|publisher= ヨシクラデザイン|accessdate=2018-02-28}}</ref>。改編電視動畫自2018年4月2日開始播放，為短動畫。

== 劇情簡介 ==
OL・早乙女陽菜因蒙受不白之冤，而慘遭入獄。在裡面等待她的是冷酷監獄長明神亞貴所施予的「以身體檢查為名的性支配」。
