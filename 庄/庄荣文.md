'''庄荣文'''（{{bd|1961年|2月}}），男，[[汉族|汉族]]，[[福建|福建]][[泉州|泉州]]人。1987年6月加入[[中国共产党|中国共产党]]。1989年毕业于[[河海大学|河海大学]]水力学与河流动力学专业。获得工学[[博士|博士]]学位，1992年获得高级工程师职称。

== 工作经历 ==
历任福建省池潭水电厂技术员；河海大学教师；福建省工程咨询总公司业务部工程师、高级工程师，业务部副主任、主任；厦门市海沧投资区建设局副局长（挂职）；福建省计划委员会工业处处长、能源处处长；福建省计划委员会副主任、党组成员；中国长江三峡工程开发总公司工程建设部副主任兼工程建设部机电安装项目部副主任，长江投资发展有限责任公司副总经理（挂职）；福建省发展改革委员会副主任、党组成员；福建省科技厅副厅长、厅长、党组书记。

2010年12月，任[[国务院侨务办公室|国务院侨务办公室]]经济科技司司长。

2014年6月，任国务院侨务办公室副主任、党组成员<ref>{{Cite web|url=http://politics.people.com.cn/n/2014/0620/c1001-25177000.html|title=庄荣文任国务院侨务办公室副主任(图/简历)--时政--人民网|accessdate=2018-07-15|last=392|work=politics.people.com.cn}}</ref>。

2015年8月，任[[中央网络安全和信息化委员会办公室|中央网信办]]、国家网信办副主任<ref>{{Cite web|url=http://news.cri.cn/gb/42071/2015/08/20/6351s5073431.htm|title=庄荣文任中央网信办、国家网信办副主任(图/简历)- 新闻 - 国际在线|accessdate=2018-07-15|last=gb.cri.cn|work=news.cri.cn}}</ref>。

2018年4月，任[[中央宣传部|中央宣传部]]副部长、[[国家新闻出版署|国家新闻出版署]]（国家版权局）署长（局长）<ref>{{Cite web|url=http://ah.people.com.cn/n2/2018/0419/c358314-31484465.html|title=庄荣文任中宣部副部长（图/简历）--安徽频道--人民网|accessdate=2018-07-15|last=L_104514|work=ah.people.com.cn}}</ref>。

2018年7月，任中宣部副部长、[[中央网络安全和信息化委员会办公室|中央网络安全和信息化委员会办公室]]主任、[[国务院新闻办公室|国务院新闻办公室]]副主任<ref>{{Cite web |url =http://politics.people.com.cn/n1/2018/0801/c1001-30182160.html|title =  庄荣文任中央网信办主任 徐麟不再担任|author =  |date =2018-08-01  |publisher =人民网 |language =  |accessdate =  |quote =  }}</ref>。

2018年8月，国务院任命其为[[国家互联网信息办公室|国家互联网信息办公室]]主任，并免去其[[国家版权局|国家版权局]]局长暨[[国家新闻出版署|国家新闻出版署]]署长职务。<ref>{{cite web |author1=张楷欣 |title=庄荣文任国家网信办主任 徐麟任国新办主任-中新网 |url=http://www.chinanews.com/gn/2018/08-21/8606032.shtml |website=中新网 |accessdate=2018-08-28}}</ref>
