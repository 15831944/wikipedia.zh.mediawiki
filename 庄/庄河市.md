'''庄河市'''是[[中华人民共和国|中国]][[辽宁省|辽宁省]][[大连|大连]]市下辖的一个县级市。位于[[辽宁|辽宁]]省东南部，[[黄海|黄海]]北岸，[[辽东半岛|辽东半岛]]的东侧。1992年9月21日撤[[县|县]]设[[县级市|市]]。与普兰店市、瓦房店市共称“北三市”。

== 概况 ==
[[明|明]]属辽东卫地，[[清|清]]属盖平县。直至清末，方置莊河厅，以境内河流莊河而得名。1913年改为'''庄河县'''，1992年撤县设市。目前隶属于[[副省级城市|副省级城市]][[大连|大连]]市。

全市处于[[黄海|黄海]]向[[千山|千山]]山脉的过渡地带，地势北高南低。年平均气温8℃，降水量890毫米。莊河的农业以[[渔业|渔业]]为主，[[粮食|粮食]]生产也颇重要。沿海[[滩涂|滩涂]]、[[湿地|湿地]]面积极大，洞礁颇多，盛产各种[[鱼|鱼]]、[[虾|虾]]、[[蟹|蟹]]、[[贝|贝]]等。其中珍贵贝类如文[[蛤|蛤]]、竹[[蛏|蛏]]等，更是销往世界。境内有仙人洞等风景点，海岸及附近海区曾为[[中日甲午战争|中日甲午战争]]的战场。为大连城市重要水源地。

== 行政区划 ==
庄河市辖有5个街道办事处、15个镇、4个乡、2个民族鄉。
*街道办事处：城关街道、新华街道、兴达街道、昌盛街道、明阳街道
*镇：[[青堆镇|青堆镇]]、[[徐岭镇|徐岭镇]]、[[黑岛镇|黑岛镇]]、[[栗子房镇|栗子房镇]]、[[大营镇_(庄河市)|大营镇]]、[[塔岭镇|塔岭镇]]、[[仙人洞镇|仙人洞镇]]、[[长岭镇_(庄河市)|长岭镇]]、[[荷花山镇|荷花山镇]]、[[城山镇_(庄河市)|城山镇]]、[[光明山镇|光明山镇]]、[[大郑镇|大郑镇]]、[[蓉花山镇|蓉花山镇]]、[[吴炉镇|吴炉镇]]、[[王家镇_(庄河市)|王家镇]]
*乡：[[兰店乡|兰店乡]]、[[步云山乡|步云山乡]]、[[石城乡_(庄河市)|石城乡]]、[[鞍子山乡|鞍子山乡]]

== 少数民族 ==
至2004年，庄河市有[[满族|满]]、[[蒙古族|蒙]]、[[回族|回]]、[[朝鲜族|朝鲜]]、[[锡伯族|锡伯]]、[[壮族|壮]]、[[苗族|苗]]、[[瑶族|瑶]]、[[黎族|黎]]、达尔、[[维吾尔族|维吾尔]]、[[傣族|傣]]、[[京族|京]]、[[高山族|高山]]、[[鄂伦春族|鄂伦春]]、[[鄂温克族|鄂温克]]等16个少数民族。少数民族总人口7.5万人，占全市人口总数的8.2%。其中满族、蒙古族、回族、朝鲜族居多，此4个少数民族的人口占全市少数民族总人口90%以上。满族人口近7万人，主要分布在太平岭、塔岭、桂云花、吴炉、青堆、仙人洞等乡镇；蒙古族人口200人左右，主要分布在大营、塔岭两镇；回族人口近1000人，主要分布在庄河市区和青堆、明阳两镇；朝鲜族人口300人左右，主要分布在桂云花乡。

== 交通 ==
丹大高速公路：连接大连与丹东（途径庄河）。大连、丹东等地的客运汽车也在此运行。

金庄铁路：连接金州站与庄河站的非电气化本地铁路。大连站与庄河站有一天一次往返的列车。
庄河到丹东方面：东港市前阳也有铁路建设的计划。起初为单轨非电气化铁路。之后改为多轨电气化铁路、客货共用，庄河西站作为货运站。其他的，例如庄河北站，庄河东站也在建设中。开通后将与大连-丹东的铁路相连。

庄河港：用于停泊到石城岛的货客渡轮，大连、丹东的货船及海产品的装卸。
