'''姜恩卓'''（{{lang-ko|'''강은탁'''}}，{{bd|1982年|8月16日}}），本名'''申瑟基'''（{{lang-ko|'''신슬기'''}}），[[韓國|韓國]][[男演員|男演員]]。2001年在安德烈‧金時裝秀（{{lang-ko|앙드레김 패션쇼}}）中以模特兒身份亮相。2006年在[[文化廣播_(韓國)|MBC]]電視劇《[[朱蒙_(電視劇)|朱蒙]]》中飾演燦秀一角，正式轉型為演員並出道。曾在數部電視劇如《[[純金的地|純金的地]]》、《[[狎鷗亭白夜|狎鷗亭白夜]]》中擔任主要角色<ref>{{cite web|last1=Bae|first1=Sun-Young|title=신예 강은탁 박하나, 임성한 호 탑승, '압구정 백야' 주연 확정|url=http://tenasia.hankyung.com/archives/316006|website=TenAsia|language=ko|accessdate=2017-06-26|date=2014-09-11}}</ref>。
==感情生活==
2018年因拍攝 [[韓國放送公社|KBS]]《[[愛到最後|愛到最後]]》與劇中飾演女主角的[[李英雅|李英雅]]相識，進而發展成戀人關係。<ref name="a1">{{Cite web |url=https://hk.entertainment.appledaily.com/enews/realtime/article/20181112/58903073 |title=【韓流】戲假情真！34歲李英雅認戀拍檔姜恩卓|accessdate=2018-11-13 |author= |date=2018-11-12 |publisher=蘋果日報 |language= }}</ref><ref name="a2">{{Cite web |url=http://ent.sina.com.cn/v/j/2018-11-12/doc-ihmutuea9385262.shtml |title=韩星姜恩卓李英雅恋情曝光 曾剧中饰演离婚夫妻|accessdate=2018-11-13 |author= |date=2018-11-12 |publisher=新浪娱乐 |language=Zh-hans }}</ref><br>
