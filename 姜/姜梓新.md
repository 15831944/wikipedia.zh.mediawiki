'''姜梓新'''（{{lang-en|'''Jiang Zi Xin'''}}，{{bd|1996年|3月11日|}}），是[[中国大陆|中国大陆]]女演员及歌手。現為[[吉翔影坊影视传媒|吉翔影坊影视传媒]]旗下藝人。

== 簡歷 ==
姜梓新4岁学习[[钢琴|钢琴]]，11岁考入[[中央音乐学院|中央音乐学院]]附属中等音乐学校，修畢音乐剧与影视表演专业科班課程。2016年，她參加《[[超級女聲|超級女聲]]》海選，演唱〈我等到花兒也謝了〉，結果落選<ref>
{{cite web |url = http://ent.ynet.com/2018/08/24/1381977t1254_3.html?h5_tuiguang |title = 《完美假期》：她凭《延禧攻略》圈粉无数，竟然参加过超级女声 |publisher = 北青網  |date = 2018年8月24日}}{{Dead link}}</ref>，同年5月則参加了[[古装剧|古装剧]]《[[如懿传|如懿传]]》试镜，雖然未能獲選，但已开始受到关注<ref>{{cite web |url = http://ent.sina.com.cn/v/m/2016-05-16/doc-ifxsenvn7200852.shtml |title = 《如懿传》导演晒试妆照 妹子甜美获赞 |publisher = [[新浪網|新浪網]]  |date = 2016年5月16日}}</ref>；7月又参與全时全景直播网络互动真人秀《[[完美假期|完美假期]]第二季》，并成为第十二位退房房者<ref>{{cite news |url = http://ent.163.com/16/0804/11/BTKE1U7B0003501E.html |title = 《完美假期》"氧气妹妹"姜梓新腹黑属性初露？ |publisher = 網易娛樂  |date = 2016年8月4日}}</ref>。

2017年，姜梓新参加清装宫廷剧《[[延禧攻略|延禧攻略]]》直播海选演员的活动且顺利进入复试阶段，还以优异表现直接獲得试戏機會，最後能夠主演皇后近身侍女「明玉」一角，備受觀眾注目<ref>{{cite news |url = https://bka.mpweekly.com/interview/%E5%A8%9B%E6%A8%82123/20180825-128566 |title = 【《延禧》明玉專訪】姜梓新佩服吳謹言充滿活力 |publisher = [[明報周刊|明報周刊]]  |date = 2018年8月25日}}</ref><ref>{{cite news |url = http://www.chinapress.com.my/20180827/%E5%A7%9C%E6%A2%93%E6%96%B0%E8%A2%AB%E3%80%8A%E5%A6%82%E6%87%BF%E5%82%B3%E3%80%8B%E6%B7%98%E6%B1%B0-%E7%B5%82%E6%86%91%E6%98%8E%E7%8E%89%E7%BF%BB%E8%BA%AB/ |title = 姜梓新被《如懿傳》淘汰終憑明玉翻身 |publisher = [[明報周刊|明報周刊]]  |date = 2018年8月27日}}</ref>；2018年亦參演古装剧《[[皓镧传|皓镧传]]》中要角「殷小春」。另外，由於其歌喉不俗<ref>{{cite news |url = https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/233300/%E5%BB%B6%E7%A6%A7%E6%94%BB%E7%95%A5-%E6%98%8E%E7%8E%89%E5%8E%9F%E4%BE%86%E4%BF%82%E9%9D%9A%E8%81%B2%E7%8E%8B-%E5%A7%9C%E6%A2%93%E6%96%B0%E5%94%B1-%E5%96%9C%E5%B8%96%E8%A1%97-%E5%A4%A7%E6%9B%AC%E5%BB%A3%E6%9D%B1%E8%A9%B1 |title = 【延禧攻略】明玉原來係靚聲王 　姜梓新唱《喜帖街》大曬廣東話 |publisher = [[香港01|香港01]]  |date = 2018年9月9日}}</ref><ref>{{cite web |url = https://www.marieclaire.com.tw/celebrity/news/38548/page_1 |title = 《延禧攻略》「明玉」姜梓新這麼會唱，聽完都有戀愛的感覺了，跪求出唱片~ |publisher = [[美麗佳人|美麗佳人]]  |date = 2018年9月7日}}</ref>，遂於2016年8月19日发行首張個人單曲《永不言败》，為内地第一部足球爱情题材网络大电影《[[球爱大越位|球爱大越位]]》的主题曲。
