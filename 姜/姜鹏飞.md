'''姜鹏飞'''（{{bd|1907年||1946年|9月11日|catIdx=J}}），原名姜凤飞，号龙涛，生于[[奉天府|奉天府]][[金州厅|金州厅]]（今[[辽宁省|辽宁省]][[大连市|大连市]][[金州区|金州区]]）<ref name="a1"/>，曾效命于[[伪满洲国|伪满洲国]]和[[汪精卫政权|汪精卫政权]]，[[抗日战争|抗日战争]]结束前夕，转投[[国民政府|国民政府]]，军衔为[[中华民国中将|中将]]，1946年8月，在[[中共政权|中共政权]]控制的[[哈尔滨|哈尔滨]]策划[[八二八暴动|八二八暴动]]，9月被中共政权处决。

== 生平 ==

1924年毕业于辽宁省立第五师范学校，后进入东三省陆军测量学校第二期、[[东北陆军讲武堂|东北陆军讲武堂]]第七期高等研究班和航空侦察班等军校学习。1931年毕业后，派往黑龙江第二独立旅[[程志远|程志远]]部，担任团长。1931年[[九一八事变|九一八事变]]后，随程志远投日。1935年，满洲国陆军大专毕业后，进入[[日本陆军大学|日本陆军大学]]受训，被日本[[裕仁天皇|裕仁天皇]]接见。1937年后，由驻黑龙江第三军调住沈阳训练学校教导队，历任新京陆军训练处中校教官、满洲国第七军管区（驻佳木斯）参谋长（少将）、独立第二旅旅长长、[[满洲国江上军|满洲国江上军]]司令。1940年，姜鹏飞被派往[[华北地区|华北地区]]。1942年后，历任伪华北绥靖总司令部一级少将、伪唐山行营主任。伪[[冀東行政公署|冀東行政公署]]成立后，兼任公署长官。是[[潘家峪惨案|潘家峪惨案]]的主谋<ref name="a1">{{Cite web |url=http://imharbin.com/post/16107 |title=国民党27军在哈尔滨覆灭 |accessdate=2017-04-11 |author=王树元 |date=2015-04-28 |work= |publisher=大话哈尔滨|language=Zh-hans }}</ref>。

[[抗日战争|抗日战争]]结束前夕，转投[[国民政府|国民政府]]。1945年7月，由[[蒋介石|蒋介石]]任命为冀东挺进军总指挥，9月，受命为[[国民革命军陆军新编第二十七军|陆军新编第27军]]军长。第27军与28军合并后，姜鹏飞受命为[[第十一战区|第十一战区]]中将高参，仍以第27军军长名义至东北组织“地下军”。1946年1月，受[[东北保安司令部|东北保安司令部]]司令官[[杜聿明|杜聿明]]委派，接受“相机占领哈尔滨”的任务<ref name="a1"/>。此时，[[中国共产党|中国共产党]]正与中华民国政府争夺东北地区的控制权，其部在长春被中共军队击败。2月，姜鹏飞潜入中共控制的哈尔滨。期间，姜鹏飞组织[[地方武装|地方武装]]、[[土匪|土匪]]、[[侵华日军|侵华日军]]残部试图在8月间攻占哈尔滨<ref name="a1"/>。他与[[李明信|李明信]]共同策划了[[八二八暴动|八二八暴动]]。但在8月26日，姜鹏飞被中共政权查获。8月28日，暴动失败。哈尔滨地方法院在9月9月组织公开审判，9月11日<ref>{{Cite web |url=http://rmfyb.chinacourt.org/paper/html/2016-06/29/content_113462.htm?div=2 |title=哈尔滨地方法院：人民法院“进城记” |accessdate=2017-04-11 |author=记者：乔文心 |date=2016-06-29|work= |publisher=人民法院报网站 |language=Zh-hans }}</ref>（或说9月10日<ref>{{Cite web |url=http://history.people.com.cn/n/2014/0808/c387503-25431078.html |title=战胜者与战败者 |accessdate=2017-04-11 |author= |date=2014-08-08 |work=责编：杨箫含、周斌 |publisher=人民网-文史频道 |language=Zh-hans }}</ref>），与李明信、[[崔大刚|崔大刚]]一同被[[枪决|枪决]]。
