'''姜立夫'''（{{bd|1890年|7月4日|1978年|2月3日|catIdx=J姜}}），譜名培珦，學名蔣佐，字立夫，以字行，[[浙江|浙江]][[平陽|平陽]]人，[[中國|中國]][[數學家|數學家]]。他的主要研究範疇為[[圓素幾何|圓素幾何]]與[[矩陣理論|矩陣理論]]。學生有[[江澤涵|江澤涵]]、[[陳省身|陳省身]]、[[劉晉年|劉晉年]]、[[孫本旺|孫本旺]]和[[申又棖|申又棖]]等。

== 生平 ==
1907年入讀新式學堂[[杭州府学堂|杭州府学堂]]。1910年[[清政府|清政府]]遊美學務處招考留學生。1911年8月赴[[美國|美國]]。1915年5月獲[[加州大學|加州大學]]理學士學位，不久後在[[哈佛大學|哈佛大學]]攻讀。1919年獲博士學位，其論文為''The Geometry of Non－Euclidean Line Sphere Transformatinon''（《非[[歐幾里得空間|歐幾里得空間]]直線球面變換法》）。10月，為了照顧侄兒而回國。

1920年，在[[南開大學|南開大學]]創辦算學系<ref>{{cite web |author1=张洪光 |title=姜立夫—“南开”元老、中国现代数学的先驱 |url=http://news.nankai.edu.cn/xs/system/2014/02/18/000166574.shtml |website=南开大学校史网 |accessdate=2018-10-08}}</ref>。最初算學系只有他一個老師，被說是「一人系」。他的教學十分嚴格。1934年曾到[[漢堡大學|漢堡大學]]和[[哥廷根大學|哥廷根大學]]進修。抗戰期間任教於[[西南聯大|西南聯大]]，兼為新數學學會主席。1940年起负责筹备数学所，1947年[[中央研究院数学所|中央研究院数学所]]成立，为第一任所长。1949年4月，將數學所圖書館收藏的全部圖書搬遷到台灣，[[四六事件|四六事件]]爆發後，1949年中，应[[陈序经|陈序经]]之邀任[[岭南大学_(广州)|嶺南大學]]數學-{系系}-主任。1950年代，嶺南大學併入中山大學後於[[中山大學|中山大學]]任教。他參與了《算学名词汇编》和《数学名词》此兩書的編輯。在中山大學期間主持了外國數學教科書的翻譯工作。

1978年因心力衰竭病逝于广州，享年87岁<ref>{{cite web |title=南开大学数学系创始人姜立夫 |url=http://www.kepu.net.cn/gb/basic/szsx/8/8_81/8_81_1028.htm |website=中国科普专栏 |accessdate=2018-10-08}}</ref>。

==家庭==
其子[[姜伯驹|姜伯驹]]是[[北京大学|北京大学]]数学系教授。
