'''姜斯宪'''（{{bd|1954年|9月||catIdx=J姜}}），[[江苏|江苏]][[江都|江都]]人。现任[[上海市人大常委会|上海市人大常委会]]副主任、[[上海交通大学|上海交通大学]]党委书记。

== 从政经历 ==
姜斯宪1972年参加工作；在[[文化大革命|文化大革命]]期间，曾在[[辽宁省|辽宁省]][[大连电机厂|大连电机厂]]当工人。1976年3月加入[[中国共产党|中国共产党]]。1978年，考入[[上海交通大学|上海交通大学]]机械工程系液压传动及控制专业学习；1982年本科毕业后，继续攻读硕士学位。1985年1月，获得工学硕士学位的姜斯宪当选上海交通大学[[共青团|共青团]]委书记，成为党工干部；此后，历任校党委宣传部副部长、部长，校党委副书记；1993年，晋升副教授；1994年，出任[[上海交通大学|上海交通大学]]副校长、党委副书记。

1994年11月，40岁的姜斯宪进入[[中共上海市委|中共上海市委]]工作，任市委组织部副部长；1995年12月至1998年8月，担任[[上海市|上海市]][[徐汇区|徐汇区]]副区长、代区长，区长。自1998年8月，先后担任[[中共上海市委|中共上海市委]]副秘书长、[[上海市人民政府|上海市人民政府]]副秘书长，市政府办公厅主任，市政府秘书长，是时任市长[[徐匡迪|徐匡迪]]的助手；2002年8月，升任上海市副市长；2003年2月，兼任[[浦東新区|浦東新区]]区委書記、区長；2004年4月，出任[[中共上海市委|中共上海市委]]常委、组织部部长，兼市国资委党委书记。

2006年10月，因两下属涉[[上海社保基金挪用案|上海社保案]]，外调[[海南省|海南省]]工作<ref>{{cite web |accessdate=2012-04-21 |url = http://www.zaobao.com/special/realtime/2006/10/061025_31.html |title = 两下属涉社保案　上海市委组织部长调职海南 |publisher=[[联合早报|联合早报]]}}</ref>，2006年11月任[[海南省人民政府|海南省人民政府]]副省长<ref>{{cite web |accessdate=2012-04-21 |url = http://news.xinhuanet.com/local/2006-11/05/content_5391744.htm |title =海南省人大任命姜斯宪为海南省副省长 |publisher=新华网}}</ref><ref>{{cite web |accessdate=2012-04-21 |url=http://society.dwnews.com/news/2006-10-25/2429543.html |title=姜斯宪外贬海南，中纪委追查其问题 |publisher=多维新闻网 |deadurl=yes |archiveurl=https://web.archive.org/web/20131230234847/http://society.dwnews.com/news/2006-10-25/2429543.html |archivedate=2013-12-30 }}</ref>；2007年6月起，兼任省委教育工委书记；2011年12月，兼任中共[[三亚|三亚]][[市委书记|市委书记]]<ref>{{cite web |accessdate=2012-04-21 |url = http://leaders.people.com.cn/GB/215059/215938/14072004.html |title=姜斯宪任三亚市委书记 |publisher=人民网}}</ref>；2012年4月，在[[中共海南省委|中共海南省委]]六届一次全会上，当选省委常委；同时，兼任海南省副省长（至2012年7月）、三亚市委书记<ref>{{cite web|accessdate=2013-12-30 |url = http://district.ce.cn/newarea/sddy/201205/02/t20120502_23288268.shtml |title=姜斯宪、马勇霞、谭本宏、孙新阳新晋海南省委常委|publisher=中国经济网}}</ref>。

2013年12月，姜斯宪被补选为上海市第十四届人民代表大会代表，引起媒体有关姜回上海任职的猜测；12月27日，姜斯宪被免去在[[海南省|海南省]]的职务<ref>{{cite web |accessdate=2013-12-30 |url=http://www.zaobao.com.sg/realtime/china/story20131230-294054 |title = 姜斯宪不再任海南常委 料调回上海 |publisher=联合早报网}}</ref>。

2014年1月14日，姜斯宪被[[中共中央组织部|中共中央组织部]]任命为[[上海交通大学|上海交通大学]]党委书记，接替即将退休的[[马德秀|马德秀]]<ref>{{cite web |accessdate=2014-01-14 |url = http://politics.people.com.cn/n/2014/0114/c1001-24111651.html |title = 姜斯宪任上海交通大学党委书记 | publisher = 人民网}}</ref>。

2014年1月23日，当选为[[上海市人大常委会|上海市人大常委会]]副主任<ref>{{cite web |accessdate=2014-01-23 |url = http://news.takungpao.com/mainland/zgzq/2014-01/2237506.html |title = 上海交大新任书记姜斯宪罕见兼职上海市人大 |publisher = 大公报}} </ref>。
