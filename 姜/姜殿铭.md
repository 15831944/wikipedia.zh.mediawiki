'''姜殿铭'''（{{bd|1931年|||}}）[[辽宁|辽宁]][[大连|大连]]人，[[中华人民共和国|中华人民共和国]][[台湾问题|台湾问题]]专家<ref name=ruishi>{{Cite web |url =http://www.rayse.com.cn/other/8-26-10.html  |title =中国人民政治协商会议第九届全国委员会委员名录（3）  |author =  |date =2011-08-26  |publisher =睿施咨询  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
姜殿铭曾任[[新华社|新华社]]记者<ref name=zhinang/>。后来调到《[[光明日报|光明日报]]》工作。1964年4月，[[日本|日本]][[松村谦三|松村谦三]]第三次访华，互换了《[[廖承志|廖承志]]办事处和[[高碕达之助|高碕]]办事处关于互派代表并互设联络事务所会谈纪要》、《廖承志办事处和高碕办事处关于中日双方互换新闻记者的会谈纪要》，此后中日双方互换新闻记者。1970年代初，姜殿铭担任《光明日报》驻日记者，与该报的[[刘德有|刘德有]]以及《[[文汇报|文汇报]]》的[[刘延州|刘延州]]等人同驻日本<ref>{{Cite web |url =  http://www.sohu.com/a/119644071_519778|title =廖公派我做《光明日报》驻日记者  |author =刘德有  |date =2016-11-22  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。后来姜殿铭担任《光明日报》副总编辑<ref name=zhinang/>。1984年，《光明日报》的姜殿铭获得“全国优秀新闻工作者（一级）”荣誉称号<ref>全国优秀新闻工作者(一级)名单，新闻记者1984年第12期</ref>。1990年，进入[[中国社会科学院台湾研究所|中国社会科学院台湾研究所]]任职<ref name=zhinang>{{Cite web |url = http://www.cnhubei.com/200406/ca485457.htm |title =《中国高层新智囊》（10）  |author =  |date = 2004-06-14 |publisher = 荆楚网|language =  |accessdate =  |quote =  }}</ref>。后成为中国社会科学院台湾研究所所长、研究员<ref name=ruishi/>。他还曾任[[全国台湾研究会|全国台湾研究会]]副会长、名誉副会长等职<ref>{{Cite web |url = http://news.qq.com/a/20060809/001818.htm |title =海峡两岸关系学术研讨会开幕 百余学者专家出席  |author =  |date =2006-08-09  |publisher =  腾讯|language =  |accessdate =  |quote =  }}</ref>。

姜殿铭是第八、第九届[[全国政协委员|全国政协委员]]<ref name=ruishi/>。
