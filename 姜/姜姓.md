'''姜姓'''为[[中文姓氏|中文姓氏]]及[[韓國姓氏|韓國姓氏]]之一，在[[中國|中國]]《[[百家姓|百家姓]]》排第32，[[中国大陆|中国大陆]]第53大姓。

== 來源 ==
遠祖始於上古時期。传说[[炎帝|炎帝]][[神農氏|神農氏]]生於姜水（今[[陝西|陝西]]省[[岐山|岐山]]縣西一帶），並以水名為姓，後代子孫也相沿為姓。

== 朝鮮姜姓 ==
'''姜姓'''是[[朝鲜族|朝鲜族]]的一个常见姓氏，在[[大韩民国|大韩民国]]1985年和2000年的人口统计中都排名第6位，现约有104万人口，主要属于15个不同的本贯。<ref>[[大韓民國統計廳|大韓民國統計廳]]，（2000年），[http://user.chol.com/~namiaz/firstname-number.htm 韓國의 姓氏: 家口 및 人口統計表] {{webarchive|url=https://web.archive.org/web/20071211181522/http://user.chol.com/~namiaz/firstname-number.htm |date=2007-12-11 }}，2000年11月1日刊行。</ref>

== 歷史人物 ==

*[[姜子牙|姜子牙]]：{{Fact|[[周|周]]初姜姓部族的[[酋长|酋长]]。|time=2018-05-30T02:02:44+00:00}}佐武王伐[[纣王|纣]]有功，[[周成王|成王]]时被封于[[齐国|齐]]，为齐国始祖。
*[[齐桓公|姜小白]]：即齐桓公，[[春秋|春秋]]五霸之首。以管仲为相，富国强兵，尊王攘夷，有效地保护了中原文化。
*[[姜维|姜维]]：三国时蜀漢大將軍
*[[姜嫄|姜嫄]]：周始祖后稷之母
*[[姜夔|姜夔]]：南宋文学家、音乐家
*[[姜士桢|姜士桢]]：又名李士桢，[[明|明]]末[[山东|山东]][[昌邑|昌邑]]人，后被[[后金|后金]]掠走，成[[金人|金人]]继子，改姓[[李姓|李]]。[[清朝|清朝]]初期任[[广东|广东]]巡抚，对清朝的[[海洋|海洋]][[贸易|贸易]]政策影响深远。其妻为[[康熙|康熙]]奶妈，其子李煦早年为康熙的伴读，后被派往[[江苏|江苏]]任[[苏州|苏州]]织造多年，其女嫁[[曹寅|曹寅]]为妻（现考疑为[[曹雪芹|曹雪芹]]的外祖母）。[[李煦|李煦]]在位期间两次修造《昌邑姜氏族谱》，现在为[[昌邑姜氏祠堂|昌邑姜氏祠堂]]供为始祖之一
*[[姜立夫|姜立夫]]（1890年7月4日－1978年2月3日），譜名培珦，學名蔣佐，字立夫，以字行，浙江平陽人，中國數學家。
