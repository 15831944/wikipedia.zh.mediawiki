'''R150型柴電機車'''，是[[臺灣鐵路管理局|臺灣鐵路管理局]]所擁有的[[柴電機車|柴電機車]]。

== 概要 ==
時屆1973年，[[臺灣鐵路管理局|臺鐵]]為更加提升[[柴電機車|柴電機車]]的運能，於是再度向[[美國|美國]][[易安迪|易安迪]]公司進口20輛柴電機車，即是R150型，編號為R151－170號，原廠形式為{{link-en|EMD G22CU|EMD G22CU}}型。

1982年時，為了配合[[北迴線|北迴線]]完工及[[臺東線#臺東線拓寬計畫|花東線拓寬計畫]]完成後之運輸需求，於是台鐵再度購入了5輛柴電機車，編號為R171－175號。此五輛車之外觀與第一批20輛的R150型稍有不同，但其差別不大，且機電規格亦無太大的變動。所以這5輛車應不致於自成一種形式。

此型車外觀與[[台鐵R100型柴電機車|R100型]]和[[台鐵R180、R190型柴電機車|R180／R190型]]極為相似。而雖然三款車型皆使用相同的引擎（[[易安迪645系列柴油机|12-645E型柴油引擎]]），但牽引力、車重和最高車速等規格稍有差異（如：R100型總牽引力為15180kgw，而R150及R180／R190型總牽引力則達19650kgw）。

== 使用情況 ==
除R164因瑞源對撞事故已經報廢以及R151停用外，R150型之其他車輛則配屬於[[彰化機務段|彰化機務段]]和[[花蓮機務段|花蓮機務段]]以及[[臺北機務段|臺北機務段]]。而因[[宜蘭線|宜蘭線]]、[[北迴線|北迴線]]的[[電氣化鐵路|電氣化]]，因此R150型大多都以牽引貨物列車為主。而有時亦會看見該型車牽引[[莒光號列車|莒光號]]、[[復興號列車|復興號]]行駛。

此外，R150型使用亦達30餘年，車體各部位都皆有些老舊。因此未來東部幹線全線以及[[南迴線|南迴線]]全線的電氣化完工通車後，也將逐漸停止使用於幹線上，而轉變任務屬性為主線[[電力機車|電力機車]]或[[電聯車|電聯車]]組故障時的緊急救援機車以及動力不足時的輔助機關車（[[補機|補機]]）。

2015年初，原配置在七堵機務段的R151－R162，因應彰化機務段R20陸續停用除籍與E303－E313、E315改配七堵機務段之故，將其改配至彰化機務段，後來因台北機務段習慣使用柴電機車調車之緣故，又再度將R160-162改配台北機務段。<ref>{{cite web |url=http://pttsp.idv.tw/ |title=存档副本 |accessdate=2015-12-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20120618095057/http://pttsp.idv.tw/ |archivedate=2012-06-18 }}</ref>
