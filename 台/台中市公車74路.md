'''台中市公車74路'''，目前行駛於[[太平區_(台中市)|太平]]到[[嶺東科技大學|嶺東科技大學]]，主要行駛於[[中山路_(太平區)|中山路]]、[[學田路|學田路]]，由[[中鹿客運|中鹿客運]]營運<ref>[http://www.ibus.com.tw/city-bus/txg/tl/rout-tl-74/ 中鹿客運74路路線圖]</ref>，為'''太平新幹線'''<ref>[http://www.taichung.gov.tw/ct.asp?xItem=1724226&ctNode=24068&mp=1001D 中市府總體檢討公車無效路線及班次 推公車路網改革計畫]</ref><ref>{{Cite web |url=http://sina-news.com/2016-02-15-02-34-42/1044-2016113006.html |title=台中市府鼓勵低碳運輸 電動公車數量台灣第一 |access-date=2017-10-27 |archive-url=https://web.archive.org/web/20171027231653/http://sina-news.com/2016-02-15-02-34-42/1044-2016113006.html |archive-date=2017-10-27 |dead-url=yes }}</ref>，屬臺中市第9條新幹線公車。本路線設置目的在於加強太平、潭子至高鐵站的交通網絡，以往經市區再轉車往往多耗費一小時以上的時間，此路線開通行走[[台74線|台74線]]，減少停站數量、行駛高架增加車速，可大幅縮減至朝馬轉運站、高鐵台中站的時間。本路線皆為純電公車。

== 路線基本資料 ==
單程行車里數約41.5公里，單程行車時間約70分鐘。往仁友停車場49站，往太平國小48站。行經太平、潭子頭家厝地區、西屯朝馬、烏日高鐵地區、南屯春社地區等5個區域。

自太平國小起，沿太平區中興路、中興東路、東平路、[[中山路_(太平區)|中山路一段～三段]]、環中東路、上[[File:TW_PHW74.svg|20px]][[太原交流道|太原交流道]]、下[[File:TW_PHW74.svg|20px]][[潭子交流道|潭子交流道]]後沿[[潭子區|潭子區]]復興路一段、圓通南路、[[中山路_(潭子區)|中山路二段]]、潭興路、興華二路、中山路二段391巷、中山路二段、環中路一段、上[[File:TW_PHW74.svg|20px]]潭子交流道、下[[File:TW_PHW74.svg|20px]][[西屯二交流道|西屯二交流道]]後沿[[西屯區|西屯區]][[朝馬路|朝馬路]]、朝富路、市政路、上[[File:TW_PHW74.svg|20px]][[西屯一交流道|西屯一交流道]]、下[[File:TW_PHW74.svg|20px]][[高鐵台中交流道|高鐵台中交流道]]後沿[[烏日區|烏日區]]高鐵路、高鐵二路、中山路、高鐵東路、新烏日火車站、高鐵六路、中山路、高鐵二路、高鐵路、[[學田路|學田路]]、[[南屯區|南屯區]][[忠勇路|忠勇路]]、嶺東南路、建功路、中台路、永春南路到仁友停車場。<ref>{{Cite web |url=http://www.ibus.com.tw/wp-content/uploads/2017/05/route-map_tl74_170508.png |title=中鹿客運74路路線圖 |access-date=2017-07-26 |archive-url=https://web.archive.org/web/20170810171723/http://www.ibus.com.tw/wp-content/uploads/2017/05/route-map_tl74_170508.png |archive-date=2017-08-10 |dead-url=yes }}</ref>
