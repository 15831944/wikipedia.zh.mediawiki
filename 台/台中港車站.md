== 車站構造 ==
*[[島式月台|島式月台]]、[[岸式月台|岸式月台]]各一座。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#二等站|二等站]]，但僅有[[臺鐵區間車|區間車]]停靠。貨運業務為本站主要服務項目。

*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]、[[icash|icash 2.0]]及[[HappyCash|HappyCash]]付費。

== 歷史 ==
*1922年10月11日：設「甲南驛」。
*1985年4月1日：改為「台中港」，而原台中港車站改名「台中港車站貨運辦公室」。
*2015年6月30日：啟用多卡通刷卡機。
