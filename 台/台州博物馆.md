'''台州博物馆'''位于[[中国|中国]][[浙江省|浙江省]][[台州市|台州市]][[椒江区|椒江区]]爱华路168号、市民广场西北侧，为市级综合性博物馆，2016年7月正式开馆，建筑地上四层，建筑面积约12500平方米，展陈面积约6000平方米，设计者为胡绍学。其中，一层为临时展厅，二层为“山魂海魄”台州历史文化陈列，三层为“大地情怀”台州民俗陈列（以上为该馆两大基本陈列），四层为民间收藏专题陈列<ref>{{cite web |url=http://paper.taizhou.com.cn/tzrb/html/2016-07/13/content_706541.htm|title=《市博物馆正式开馆》|accessdate=2017-06-30|work=|publisher=台州日报}}</ref>。截至开馆时该馆共有藏品2500余件，包括青铜器、陶瓷器、画像、石刻和木雕等。该馆开放时间为周二至周日上午9：00-下午4：30（4：00停止入馆），周一闭馆（法定节假日除外）<ref>{{cite web |url=http://paper.taizhou.com.cn/tzrb/html/2016-07/01/content_702857.htm|title=《去台州博物馆“探宝”，得了解这些事》|accessdate=2017-06-30|work=|publisher=台州日报}}</ref>。
