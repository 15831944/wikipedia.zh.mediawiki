'''台北小巨蛋站'''位於[[台灣|台灣]][[台北市|台北市]][[松山區_(臺灣)|松山區]]，為[[台北捷運|台北捷運]][[松山新店線|松山新店線]]的[[鐵路車站|捷運車站]]。

== 車站概要 ==
車站位於[[南京東路_(台北市)|南京東路四段]]下方，北寧路口西側，因鄰近[[臺北小巨蛋|臺北小巨蛋]]而得名；車站代碼為'''G17'''。本站原定站名為「'''市立體育場站'''」，2011年（民國100年）7月22日捷運局更為現名。<ref>[http://www.dorts.gov.tw/ct.asp?xItem=2137534&ctNode=16447&mp=115001 臺北捷運4車站站名更名更具辨識性 以方便民眾搭乘] {{webarchive|url=https://web.archive.org/web/20111003213239/http://www.dorts.gov.tw/ct.asp?xItem=2137534&ctNode=16447&mp=115001 |date=2011-10-03 }}</ref>

在BMTC版本規劃中，本站（工程代碼G19）比現址更加接近敦化北路，並與[[南京三民站|南京三民站]]（工程代碼G21）間多設置一站，即[[南京光復站|工程代碼G20站]]。且在G20站取消設站後，原先車站位置被東遷至南京健康路口，隨後才更改規劃設於現址。

== 發音爭議 ==
自通車以來至2015年1月31日止，捷運列車抵達本站時，車內廣播會以國語連續念出三次「台北小巨蛋」，與其他各站會分別用國、台、客語分別念站名有極大不同，高雄捷運[[巨蛋站|巨蛋站]]亦有類似情況。此舉造成國內台、客語學者諸多批評。「巨」字在日治時期所編纂之「台日大辭典」裡即有收錄台語發音「kī（漳州腔）／kū（泉州腔）」，「蛋」字亦明記發音為「tàn」，故「巨」、「蛋」兩字以台語文讀音發音並非硬翻，而是該字原本即可以文讀音讀出，惟今人不查，誤以為「巨蛋」無法以台、客語發音，才造成廣播上的爭議。故台北捷運公司請益台、客語學者後達成共識，自2015年2月1日，開始播送台語、客語版本的廣播。<ref>{{Cite news|url = http://www.metro.taipei/ct.asp?xItem=96051943&ctNode=70050&mp=122035|title = 臺北捷運「臺北小巨蛋」站名4語廣播 2月1日更新上線|last = |first = |date = 2015-01-31|work = 臺北大眾捷運股份有限公司|accessdate = 2015-02-04}}</ref>台語以北部使用者較多的泉州腔文讀音發為「Tâi-pak Sió-kū-tàn」，客語則以四縣腔發為「toiˇ bedˋ seuˋ kiˊ tan」。

==利用狀況==
依據{{TRTS ridership-date}}資料，本站每日旅運量約為{{TRTS ridership|station=台北小巨蛋站}}人次，在台北捷運各站中排行第{{TRTS ridership-rank|station=台北小巨蛋站}}名。

本站的客源主要是前往台北小巨蛋參加活動和演唱會的旅客，而其他方面的旅客為附近通勤的通勤族。
