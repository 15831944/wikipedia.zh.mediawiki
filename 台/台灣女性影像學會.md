'''社團法人台灣女性影像學會'''（[[英語|英語]]：Taiwan Women's Film Association）為[[臺灣|臺灣]]一[[社團法人|社團法人]]，成員多為影像工作者、電影學者及藝文工作者，致力於推廣女性影像和性別平權教育，每年定期舉辦影展以及不定期舉辦各項女影專案，鼓勵女性影像工作者發展專業能力，又以建立女性影像發表之管道為首要任務。

其每年舉辦之[[台灣國際女性影展|台灣國際女性影展]]，藉由女性影片之國際觀摩，拓展女性創作視野，並增進台灣與國際女影人之交流，不僅是台灣第一個{{le|女权主义电影|feminist film theory}}影展，僅次於[[金馬國際影展|金馬國際影展]]台灣第二個影展，更創全國首例於各戲院、院校、藝文中心、社區等據點巡迴放映，也是亞洲兼具重要性與代表性的影展之一。<ref name="台灣女性影像學會官網">{{Cite web |url=https://www.wmw.com.tw/about.php?It=11 |title=台灣女性影像學會官網 |access-date=2017-09-07 |archive-url=https://web.archive.org/web/20171012182656/http://www.wmw.com.tw/about.php?It=11 |archive-date=2017-10-12 |dead-url=yes }}</ref><ref name="社團法人台灣女性影像學會- 外交部NGO雙語網">{{Cite web |url=http://www.taiwanngo.tw/files/13-1000-12755-1.php?Lang=zh-tw |title=社團法人台灣女性影像學會- 外交部NGO雙語網 |access-date=2017-09-07 |archive-url=https://web.archive.org/web/20170907213931/http://www.taiwanngo.tw/files/13-1000-12755-1.php?Lang=zh-tw |archive-date=2017-09-07 |dead-url=yes }}</ref><ref>{{cite news|url=http://www.chinatimes.com/realtimenews/20140516004187-260405 |title= 台灣女性影展20年 高師大展出 |publisher= 中國時報 |date= 2014-05-16 |author= 吳江泉 |accessdate= 2017-09-06}}</ref>

== 沿革 ==
台灣女性影像學會成立於2000年，前身為1998年9月19日成立的「台北市女性影像學會」。在正式成立學會之前的1993年－2000年，已舉辦了六屆[[台灣國際女性影展|台灣國際女性影展]]。<ref name="社團法人台灣女性影像學會- 外交部NGO雙語網"/>
