'''[[台灣|台灣]]網路用語'''是指通常使用於台灣網站上的[[網際網路用語|網際網路用語]]，其中不少以[[臺灣閩南語常見流行語|臺灣話]]、[[客家語|客家語]]或[[台灣國語|台灣國語]]的諧音轉寫為中文字的詞彙，也有部份是受到其它地區如[[中國大陸網路語言列表|中國大陸]]、[[香港網路語言列表|香港]]、[[日本|日本]]的網路用語影響而產生。

== 網路語言 ==
*[[火星文|火星文]]：是指大量使用同音字、音近字、特殊符號來表音的文章或語句，例如「偶」表示「我」，例如「粉」表示「很」，例如很表示hen。

*[[注音文|注音文]]：是指以注音符號代替文字的表現方式，例如「ㄋ好ㄇ」表示「你好嗎」，「ㄋ看ㄉ懂這句話ㄇ？」表示「你看得懂這句話嗎？」，「好ㄌ」表示「好了」，「我ㄉ」表示「我的」，也有以「ㄋ」表示「呢」，「ㄚ」表示「啊」，「ㄅ」表示「吧」等語助詞，對於不諳[[注音符號|注音符號]]者會造成判讀困擾。注音文是火星文中的一種。

*[[繪文字|繪文字]]：是使用在網頁和聊天中的形意符號，最初是日本在無線通訊中所使用的視覺情感符號（圖畫文字），繪意指圖形，文字則是圖形的隱喻，可用來代表多種表情，如笑臉表示笑、蛋糕表示食物等。

*[[顏文字|顏文字]]：是指[[表情符號|表情符號]]。是以文字與符號來傳達心情的方式。它原本只是一種網上次文化，但隨著網際網路和行動電話簡訊的普及，已經為社會廣泛接受。

==網站機制==
===網路稱謂===
*巴友:　[[巴哈姆特電玩資訊站|巴哈姆特]]使用者的互稱。
*'''[[鄉民_(網路用語)|鄉民]]'''：[[批踢踢|批踢踢]]使用者的自稱，後泛指大部分的年輕網路使用者。出自香港影星[[周星馳|周星馳]]的電影《[[九品芝麻官|九品芝麻官]]》裡[[方唐鏡|方唐鏡]]的對白，指事不關己卻愛湊熱鬧的人。

*'''島民'''：[[Komica|Komica]]使用者的自稱。由於Komica又稱K島，故自稱島民。

*'''酸民'''：是指說話語帶諷刺的網友。

*'''偶'''：第一人稱自稱詞，即[[台灣國語|台灣國語]]發音的「我」。

*'''大大'''：對地位較高的網友的敬稱，同「高手」。因為不知對方真實性別及年齡，不方便以「大哥」、「大姐」、「大叔」、「大嬸」等稱呼，用「大人」又過於恭敬，就用「大大」來代表不明身份，敬語的統稱來稱呼對方，衍生用法有「巨巨」及「碩碩」。

*'''水水'''：是對女性的稱呼。在台灣話中，「水」（或寫為媠，台羅：súi）是稱呼女性漂亮的意思。

*'''肥宅'''：字面意義是指「[[肥胖|肥胖]]的[[御宅族|御宅族]]」<ref>{{cite news |language = 中文 |author =  |coauthors =  |url = http://m.appledaily.com.tw/realtimenews/article/international/20161213/1011472 |title =這肥宅超勵志！拼命衝浪終於變身型男 |publisher =蘋果日報 |pages =  |date = 2016-12-13 |accessdate = 2016-12-28}}</ref><ref>{{cite news |language = 中文 |author =  |coauthors =  |url = http://www.ettoday.net/news/20161229/838546.htm |title = 神文釣出「興奮哥」！　日本偶像奶貼懷抱打趴一票人 |publisher = 東森新聞雲 |pages =  |date = 2013-12-29 |accessdate = 2016-12-28}}</ref>。

*'''[[魯蛇|魯蛇]]'''：取自英文「Loser」的諧音，是在工作上不順遂或失業、低收入、沒有[[愛人|愛人]]者。相對地，有成就、收入高或小康、有愛人者則被稱為'''[[溫拿|溫拿]]'''（「Winner」）<ref>{{cite web | url=http://tw.news.yahoo.com/%E7%A4%BE%E7%BE%A4%E6%96%B0%E8%81%9E-ptt%E8%87%AA%E5%89%B5%E8%A9%9E-%E9%AD%AF%E8%9B%87%E6%AD%A3%E5%A4%AF-093158868.html | title=社群新聞／Ptt自創詞 「魯蛇」正夯 - Yahoo奇摩新聞 | accessdate=2014-02-28}}</ref>。此類網友可能會以「小魯」或「小蛇」（[[漢語人稱代詞#自謙|第一人稱自謙]]『小弟』的變化）自稱，或使用「本魯蛇」、「本魯」、「本蛇」等。

===論壇機制===
*'''[[頭香_(網路)|頭香]]'''：指第一個回覆文章的內容。此詞源自台灣過年的「搶頭香」活動。

*'''卡'''：是指透過發表沒有實質內容的回覆來「卡位」（佔位置）的意思。在論壇中，較先回覆文章者的回覆內容通常會放在網頁中較前面的位置，較後回覆文章者的內容則在後面，當出現了一篇自己認為可能會有許多人回覆的文章時，透過先行回覆一段沒有實質內容，可以搶得較前的位置。

*'''推文'''：是網路發文的機制。在別人或自己的文章下方留下回覆推薦這篇文章，或者對此文章發表意見。

*'''噓文'''：是網路發文的機制。在別人或自己的文章下方留下回覆反對這篇文章。

*'''五樓'''：是網路的流行語，指推文或噓文的第五則回覆。由於推文流行之初，五樓的評論恰巧很專業或中肯，網友便漸漸地約定成俗、習慣叫五樓出面評論，例如「五樓你說呢？」、「五樓中肯」<ref>{{cite news |language = 中文 |author =  |coauthors =  |url = http://www.ettoday.net/news/20151224/618580.htm |title =徐欣瑩不懂網路用語求助網友　「她」竟搶盡鋒頭成亮點|publisher = 東森新聞雲 |pages =  |date =2015-12-24 |accessdate = 2016-12-29}}</ref>。後來也會將各種事情推給五樓。

*'''關鍵字之亂'''：是指批踢踢中的一種現象，指短時間內某個詞彙高頻率地出現在文章的標題中。例如2016年時，由於圍棋對弈程式[[AlphaGo|AlphaGo]]和韓國職業棋士[[李世乭|李世乭]]的[[AlphaGo李世乭五番棋|對弈]]受到關注，於是在對弈的一星期之間、批踢踢上便出現許多以[[AlphaGo|AlphaGo]]的諧音「阿法狗」為題的文章。

[[File:Wooden_bucket.jpg|thumb]]
*'''水桶'''：是指由於違反了網路論壇的規定，而被管理方封鎖部份使用者權限。

*'''幫高調'''：是一種希望某篇文章讓更多人關注的方式。當一篇文章有了新的回覆，該篇文章就會被系統移動至網頁的前面，因此透過不斷地回覆，可以讓一篇文章較容易被看到。通常會出現在尋人啟事、發生意外等需要眾人協助的文章中。

*'''sage'''：是[[2ch|2ch]]及類似風格的討論版（如[[Komica|Komica]] ）的機制，sage指的是壓下（{{lang|ja|下げ}}）。當一篇文章有了新的回覆，該篇文章就會被系統移動至網頁的前面，而透過在回覆中填寫sage會讓文章不會被移動至前面，此機制的目的在於防止某些被過度討論的舊文章不斷地擠掉新文章。

*'''潛水'''：在論壇中借指「只瀏覽文章、但不發表文章的行為」。

*'''神'''：是指「搜尋」，當用於尋人時稱為「神人」<ref>{{cite news |language = 中文 |author =  |coauthors =  |url = https://www.thenewslens.com/article/14670 |title = 【影片】台灣鄉民太難懂！馬來西亞華人：什麼是「神」一下這正妹？ |publisher =關鍵評論網 |pages =  |date = 2015-04-01 |accessdate = 2016-12-28}}</ref>。

===特定網路使用者===
*'''[[母豬教|母豬教]]'''：源自[[批踢踢|批踢踢]]的虛擬組織，該組織將提出「'''母豬'''」用法的批踢踢[[鄉民_(網路用語)|鄉民]]「obov」尊稱為教主<ref name = "黃上銓">{{Cite web |url = http://www.bnext.com.tw/article/41521/xiaomiao-versus-sumade-with-sowism-dispute-on-ptt |title = 苗蘇論劍：母豬教爭議是否有解？ |author = [[黃上銓|黃上銓]] |date = 2016年10月27日 |publisher = 《數位時代》 |language = zh-hant |accessdate = 2016年11月3日 |quote =  }}</ref><ref name = "柯采岑">{{Cite web |url = http://womany.net/read/article/10857 |title = 性別觀察：誰是母豬教徒？當仇女成為一種流行 |author = [[柯采岑|柯采岑]] |date = 2016年6月6日 |publisher = 女人迷 |language = zh-hant |accessdate = 2016年10月22日 |quote =  }}</ref>。「obov」為批踢踢資深使用者，在「母豬母豬，夜裡哭哭」一句提出後，因經常在推文發表相關話語<ref name = "黃上銓"/>，知名度迅速增加<ref name = "黃厚銘 240-246">{{Cite book | author = 黃厚銘 | title = 《婉君妳好嗎？：給覺醒鄉民的PTT進化史》 | location = 臺灣新北 | publisher = 群學出版社 | date = 2016年5月14日 | pages = 第240頁至第246頁 | ISBN = 978-9869280327 | accessdate = 2016年10月23日 | url =  | language = zh-hant | quote =  }}</ref>。之後許多鄉民跟進仿效與起鬨，開玩笑地表示建立「母豬教」，自稱為信奉「母豬教」的信徒<ref name = "黃上銓"/><ref name = "黃厚銘 240-246"/>。

*'''ㄈㄈ尺'''：跨文化戀愛（'''C'''ross '''C'''ultural '''R'''omance、'''CCR'''）的蔑稱。跨文化戀愛指的是戀愛當事人分屬不同的文化、社會或國家，而「CCRomance版」是批踢踢討論該話題的版面，由於該板常有台灣女性發表與外國男友戀愛經驗，因而使得部份台灣男性反感、進而搗亂該板面，最終造成該版一度禁止發表文章。此一事件後，便出現與字形「CCR」相似的「ㄈㄈ尺」來蔑稱該板面、或蔑稱跨文化戀愛的女性。

*'''三十公分'''：又寫做'''30cm'''。指某些男性網友喜歡吹噓自己的[[陰莖|陰莖]][[人類陰莖尺寸|長度]]至少三十公分<ref>{{cite news |language = 中文 |author = 施旖婕 |coauthors = |url = http://www.appledaily.com.tw/realtimenews/article/new/20161215/1012598/ |title = 30cm都是過譽　調查證明男人愛誇大陰莖 |publisher = 蘋果日報 |pages =  |date = 2016-12-15 |accessdate = 2016-12-27}}</ref>，後也演變成對男性網友的敬稱以及對巨大陰莖的代稱<ref>{{cite news |language = 中文 |author =  |coauthors =  |url = http://www.setn.com/News.aspx?NewsID=205040 |title = 鄉民GG了！「30公分」不受女性歡迎　這長度是她們最愛 |publisher = 三立新聞網 |pages =  |date = 2016-12-07 |accessdate = 2016-12-27}}</ref>。

*'''G罩杯'''：與三十公分用法類似，為女性網友的相對應用法、指[[胸罩|胸罩]][[罩杯|罩杯]]為G罩杯，也可做為對女性網友的敬稱。相較於三十公分，此級數較不固定，也可用其它大於台灣女性平均罩杯的級數取代。

*'''[[網軍|婉君]]'''：指的是[[網軍|網軍]]，是指被認為是受特定政府機構或其他政治組織僱傭，為達成某一目標，通過網際網路進行特定活動的人員。「婉君」原指台灣言情小說家[[瓊瑤|瓊瑤]]的作品《追尋》裡的主角「夏婉君」，因為改編成同名電視劇《[[婉君|婉君]]》而廣為人知。「婉君」一詞在[[2014年中華民國地方公職人員選舉|2014年地方公職人員選舉]]時開始大量出現、並成為「網軍」的代稱<ref>{{cite news |language = 中文 |author =黃厚銘  |coauthors =  |url = https://www.thenewslens.com/article/43236 |title = 《婉君妳好嗎？》：網路與PTT「鄉民研究」的評析 |publisher = 關鍵評論網 |pages =  |date = 2016-07-03 |accessdate = 2016-12-29}}</ref>。

*'''8+9'''：代指小混混、街頭[[流氓|流氓]]與[[黑社會|黑社會]]小弟，源自於[[八家將|八家將]]。因為近年許多什家將、八家將等陣頭遭不良組織操控，成員常在廟會之餘從事非法行為，因此使八家將團體被視為不良團體的一種。特別是常在批踢踢等論壇瀏覽的鄉民們，由於八家將的台語發音「pat-ka-tsiòng」類似「8+9」的念法，因而衍生這個詞<ref>[http://photo.chinatimes.com/20151214002231-260809 「八嘎囧」晚上都在幹嘛？網友專業文被推爆]</ref>。另有「八嘎囧」、「巴嘎囧」等其他稱法。

*'''工具人'''：諷刺時常被人利用，甚至主動被利用的人。這類的人通常較不擅社交，缺乏摯友與伴侶，但有渴望不遭邊緣化，因此寧願選擇貶低自身來奉承他人以換取團體中的存在價值。

*'''邊緣人'''：諷刺不擅社交、存在感低迷的人。在團體中時常遭人遺忘，也缺乏摯友與伴侶，與工具人類似。不少邊緣人也兼具肥宅身份，因此也衍生出合義詞「邊緣肥宅」。

===社交詞令===
*'''安安'''：打招呼的用語<ref>{{cite news |language = 中文 |author = 蔡惠嵐 |coauthors =  |url = http://www.appledaily.com.tw/realtimenews/article/new/20161204/1003869/ |title = 不小心說出這句話　代表你老了 |publisher = 蘋果日報|pages =  |date = 2016-12-04 |accessdate = 2016-12-28}}</ref>，是結合了各時段的問候，即早安、午安和晚安的用法。由於無法得知對方網友是在什麼時間查閱這些電子訊息，因此會使用不帶有時間的問候語。

*'''啾咪'''：是指「[[親吻|親]]我」，通常用於表現可愛時使用。啾指的是親吻，咪是英文「me」（我）的音譯。

=== 文章類型 ===
*'''[[回鍋文|回鍋文]]'''：指該文章是先前已有人發佈過的舊文。所謂「回鍋」是烹飪用語，是「再次烹調」的意思。如果該文章定期出現又稱'''[[月經|月經]]文'''，更頻繁者稱'''日經文'''。如果文章被轉貼至原來的出處，則稱'''回娘家文'''。
**'''OP'''：在[[批踢踢|批踢踢]]中，回鍋文又稱「OP」，源自於一位暱稱為overpass的網友經常重複張貼其他人已經發過的笑話文章。

*'''做夢'''：在討論未經驗證的消息時，由於這些內容可能涉及人身攻擊或有法律疑慮，如不願意斬釘截鐵地表示意見，便會以「做夢」形容所見所聞相當虛幻，如同一場夢<ref>{{cite news |language = 中文 |author =葉國吏 |coauthors =  |url =http://www.appledaily.com.tw/realtimenews/article/new/20160511/858338/ |title =鄭捷伏法他最先知！　爆掛正確鄉民朝聖 |publisher =蘋果日報 |pages =  |date =2016-05-11 |accessdate = 2016-12-28}}</ref>。有時會以'''[[黃梁夢|煮黃粱]]'''一詞代替，或稱文章中提及的相關事證是「撿來的」。

*'''[[釣魚文|釣魚文]]'''：執法警察以引誘、試探的方式實施偵查並陷[[性工作者|性工作者]]入罪，此一教唆犯罪的行為稱之為釣魚<ref>{{cite news |language = 中文 |author =王顥中 |coauthors =  |url = http://www.coolloud.org.tw/node/86656 |title = 行徑囂張！先教唆再逮人 同志團體揭警察釣魚亂象 |publisher =苦勞網 |pages =  |date =2016-10-19 |accessdate = 2016-12-28}}</ref>。網友便將有類似效果的文章，例如「標題聳動」但「內容空洞」的文章、或是「純屬杜撰」的文章稱之為釣魚文。

*'''灌水'''：內容無意義、空洞或陳舊的文章。由於許多論壇都有發表文章後可累積論壇內積分並提升權限的機制，因此發佈灌水文章可以較迅速地提高積分。

*'''地雷文'''：含義有三
**其一，地雷是指「捏他八雷」，亦即日語的「[[劇情透露|劇情透露]]」。表示該文章的內容中會透露涉及劇情的作品的劇情內容。
**其二，地雷是指「令人感到負面的事物」，就如同[[地雷|地雷]]的本義是指放置或埋在地下的爆裂物。該文章的內容通常會有撰文者一段糟糕的體驗。
**其三，地雷同樣是指「令人感到負面的事物」，但感到負面的並非撰文者，而是讀者。用法類似於釣魚文。

*'''[[囧|囧]]'''：表情符號，字的形狀類似一個人的[[五官|五官]]整個皺起來，感到無奈或無話可說的樣子。

*'''[[XD|XD]]'''：表示笑臉。將其順時針旋轉九十度後，X在上面、可以看做瞇起的眼睛，D在下面，是大笑的嘴。

*'''[[Orz|Orz]]'''：是指[[失意體前屈|失意體前屈]]，它看起來像是一個人跪倒在地上，低著頭，一副「天啊，為什麼是這樣」的動作。此符號起源於日本，其後流傳至香港、台灣和中國大陸。台灣於2006年[[大學學科能力測驗|大學學科能力測驗]][[國文|國文]]科考題，非選擇題出現「3Q得Orz」，意為「感謝得五體投地」，之後Orz才被廣泛使用，在拜托他人時也會用Orz，指跪下來求。

*'''[[OGC_(符號)|OGC]]'''：表示男性[[手淫|手淫]]。將其順時針旋轉九十度後，O為人體頭部，G為上下兩側為左右手，中間則一橫為[[男性生殖器官|男性生殖器官]]，C則表示為雙腳。
