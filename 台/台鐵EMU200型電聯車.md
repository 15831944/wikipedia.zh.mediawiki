'''EMU200型'''是一款屬於[[臺灣鐵路管理局|臺灣鐵路管理局]]的[[交流電|交流]]中長途用[[電聯車|電聯車]]<ref name="IH1">《台灣鐵路火車百科》 第92頁</ref>，於2002年至2004年間全數改造成[[台鐵EMU1200型電聯車|台鐵EMU1200型電聯車]]<ref name="TC2">{{cite web|url=http://club.ntu.edu.tw/~club20322/train/Taiwan_trains/page/Tzechiang.htm|title=悠遊台灣鐵道-台灣火車介紹-自強號|accessdate=2011-02-14 |language=zh}}</ref><ref name="IH3">《台灣鐵路火車百科》 第94頁</ref>。

== 概要 ==
台灣從1978年開始在[[西部幹線|西部幹線]][[十大建設#十大建設的內容|鐵路電氣化]]後所行駛的車種「[[自強號列車|自強號]]」，最初由[[台鐵EMU100型電聯車|EMU100型電聯車]]所行駛。到了1980年代中期，由EMU100型所行駛的自強號，經常一票難求。臺鐵為了提升客運運量上的需求，並解決EMU100型運量上的不足，並且輔助輸運旅客，於是以1984年、1985年的年度預算向南非[[聯邦鐵路客貨車公司|聯邦鐵路客貨車公司]]訂購了11組一共33輛的'''EMU200型電聯車'''<ref name="TC5">{{cite web|url=http://blog.yam.com/zeorymer1020/article/2817252 | title=流浪攝影記: 車種介紹-EMU200自強號|date=2006-08-21|accessdate=2011-02-14|language=zh-hant}}</ref><ref>{{cite web|url=http://jizuo.tripod.com/d001/200.htm | title=新阿霖車站：電聯車EMU200型|date=1999-07-21|accessdate=2011-02-14 |language=zh-hant}}</ref><ref name="TC1">{{cite web|archiveurl=https://web.archive.org/web/20070212055932/http://emu300ct.myweb.hinet.net/index/tramus/EMU200.htm |archivedate=2007-02-12 |url=http://emu300ct.myweb.hinet.net/index/tramus/EMU200.htm |title=自強號電聯車噪音王~EMU200 |publisher=Train Collection |accessdate=2011-02-14 |language=zh-hant |deadurl=yes }}</ref><ref name="TC3">{{cite web|url=http://blog.yam.com/zeorymer1020/article/2817252 | title=流浪攝影記: 車種介紹-EMU200自強號|accessdate=2011-02-14 |language=zh-hant}}</ref>。

1986年開始，EMU200型列車由南非聯邦鐵路客貨車公司進行列車的設計與承造，並由原廠生產8組共24輛，其中3組共9輛由[[唐榮鐵工廠|唐榮鐵工廠]]組裝而成，在1986年底一共生產了11組總共33輛<ref name="IH1" />。到了1987年EMU200型列車開始進行驗收及試運轉之後，同年6月開始營運，成為第二款自強號電聯車<ref name="TC1" />。

== 規格與構造 ==
; 編組體系
臺鐵為了能夠方便調度車輛，不同於EMU100型的5輛一組（1M4T），EMU200型改為3輛為一個編組，而[[動拖比|動拖比]]為2M1T，平常使用3個編組共9輛，而最多能夠連結至5組共15輛，EMU200型列車從[[高雄車站|高雄]]到[[臺北車站|臺北]]方向開始連結，分別是駕駛附車長室馬達車'''EMC200型'''、電源動力車'''EP200型'''以及駕駛馬達車'''EM200型'''，<ref name="IH1" /><ref name="TC1" />。
; 車體
EMU200型列車與EMU100型列車的車體外型不相同，與後來臺鐵在1989年所引進的[[台鐵EMU300型電聯車|EMU300型電聯車]]一樣，外型較為方正，並將排水管隱藏，車燈增加兩組，列車[[排障器|排障器]]採用密閉式設計。而避免乘客搭錯車，在車頭以及車門上方設置了捲簾式[[路線牌|車側終點顯示器]]（Destination Indicator，簡稱DI），節省換目的地的壓克力板的時間。為了能夠讓乘客快速上下車，動力車的車門設置在車體的末端，而無動力車的車門則是前後兩端皆設有車門，所有車門皆為手動門<ref name="IH1" /><ref name="TC2" /><ref name="TC1" /><ref name="TC4">{{cite web|url=http://140.117.11.2/txtVersion/treasure/railway/M.941792539.A/M.941793021.F.html | title=EMU200型自強號簡介|accessdate=2011-02-14 |language=zh-hant}}</ref>。

; 客室設備
座椅採用迴轉式斜倚，並且以每行2+2的配置方式設置，在行李架下方座椅上方設置了冷氣個人出風口以及閱讀燈。為了保持車廂內的溫度，列車的通道門採用了紅外線感應式自動門<ref name="TC2" /><ref name="TC1" /><ref name="IH2">《台灣鐵路火車百科》 第93頁</ref>。

; 機電
EMU200型的機電系統採用[[英國通用電氣公司|英國通用電氣公司]]（GEC）製造，主[[電動機|電動機]]採用功率250[[千瓦|kW]]牽引馬達，一輛動力車廂使用四組牽引馬達，驅動裝置採用[[吊掛式傳動|吊掛式傳動]]<ref>{{cite web |url=http://www.c-player.com/ac44458/thread/1100072814413?page=1#1100072842558 |script-title=ja:鉄道ファン大集合　「テツ＆レイル」：【台湾】吊り掛け式特急電車 |accessdate=2011-02-14 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20111106121614/http://www.c-player.com/ac44458/thread/1100072814413?page=1#1100072842558 |archivedate=2011-11-06 }}</ref><ref>{{cite web|url=http://www1.odn.ne.jp/~cas29350/2002taiwan1226-7.htm|script-title=ja:ＦＲＯＮＴ　ＶＩＥＷ:海外旅行記のコーナー:松山～苗栗（電車自強号）|accessdate=2011-02-14|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20071030143524/http://www1.odn.ne.jp/~cas29350/2002taiwan1226-7.htm|archivedate=2007-10-30}}</ref>，而控制裝置使用[[斬波器|斬波器]]，因此列車的加速能力比起EMU100型較佳，而相同的是列車最高速度一樣維持在120km/h不變<ref name="IH1" /><ref name="TC2" />。

[[集電弓|集電弓]]採用單臂式集電弓，設置在EP200型電源動力車上<ref name="TC1" /><ref name="TC4" />，供電採用[[馬達交流發電機組|馬達交流發電機組]]〈MA-set〉，提供冷氣機以及電燈所需的電力<ref name="IH1" /><ref name="TC1" />。

; 轉向架與制軔
[[轉向架|轉向架]]採用南非Dorbyl Transport Products公司製造的空氣彈簧轉向架，動力車廂採用TR53A轉向架，而無動力車廂則是採用TR53B型轉向架<ref name="TC1" />，為了防止車輛產生蛇行而裝設了車廂減擺器。制軔系統不同於EMU100型所使用{{Tsl|en|Westinghouse Air Brake Company|西屋空氣制軔}}的製品{{noteTag|1='''西屋空氣制軔公司'''（英語：Westinghouse Air Brake Company），台鐵將其譯為'''西屋西碼式'''。}}，而改為德國[[克諾爾集團|克諾爾制動公司]]（{{lang|en|Knorr-Bremse AG}}）製造的[[電軔|電軔]]與[[空氣制動|氣軔]]並用系統<ref name="TC4" /><ref name="IH2" />。

; 保安裝置
EMU200型全列車編組的[[鐵路安全裝置|保安裝置]]依照臺鐵的要求全面裝設[[自動列車警報裝置|自動列車警報裝置]]（AWS）以及[[自動列車停止裝置|自動列車停止裝置]]（ATS-SN以及ATS-P）<ref name="IH2" />。

== 形式說明 ==
;55EMC200型
:駕駛馬達車，車頭面向高雄方向開始連結編組。設有車長室，以及方向幕的控制裝置。車廂尾端設置廁所以及車門，車頂上則設有電阻器。
;45EP200型
:電源拖車，車頂上設置集電弓，車底下設有變壓器，車體兩端皆設置車門。
;55EM200型
:駕駛馬達車，車頭面向台北方向連結，車頂上設有電阻器。

== 重大事故 ==
EMU200型的營運時速為120公里，煞車距離也比時速100公里的莒光號來的要長，自從營運以來已有多次與EMU200型相關的平交道事故，也造成了三節車廂報廢。
=== 意外事件列表 ===
*1992年10月31日，[[苗栗縣|苗栗縣]][[三義鄉_(臺灣)|三義鄉]]一輛拖板車經過八股頭平交道時，由於平交道過於陡峭使得拖板車的底盤卡進平交道動彈不得，而被北上的1010次自強號撞上，55EM211與45EP211嚴重損毀而報廢解體<ref name="Accidents1">{{cite web|url=http://www.railway.gov.tw/tw/cp.aspx?sn=3705 | title=平交道安全資訊專區：平交道停看聽|publisher=交通部臺灣鐵路管理局|accessdate=2011-02-14 |language=zh}}</ref><ref name="Accidents2">{{cite web|url=http://140.117.11.2/txtVersion/treasure/railway/M.868779407.A/M.893387817.A/M.893666424.A.html | title=自強號電聯車重大車禍一覽|accessdate=2011-02-14 |language=zh-hant}}</ref>。
*1997年4月3日，台中縣烏日鄉（現[[台中市|台中市]][[烏日區|烏日區]]）一輛砂石車經過學田路平交道時，由於車輛突然熄火，而被北上的1012次自強號迎面撞上，這起事故造成二死十五傷，55EM209嚴重損毀而報廢解體<ref name="Accidents1" /><ref name="Accidents2" /><ref>{{cite web|url=http://140.117.11.2/txtVersion/treasure/railway/M.868779407.A/M.893387832.A/M.893666403.A.html | title=1012次烏日學田路平交道事故|accessdate=2011-02-14 |language=zh-hant}}</ref>。

== 車輛改造 ==
{{main|台鐵EMU1200型電聯車}}
而EMU200型、[[台鐵EMU300型電聯車|EMU300型]]以及[[推拉式自強號|推拉式自強號]]等自強號相繼引進後，台鐵對於EMU200型電聯車設備老舊以及各種缺失，而在2001年開始進行招標，由[[台灣車輛|台灣車輛]]得標後，從2002年開始至2004年將剩下來的30輛電聯車全面進行改造與更新<ref>{{cite web |url=http://www.trsc.com.tw/products2.htm |title=台灣車輛：銷售實績 |accessdate=2011-02-14 |language=zh-hant |deadurl=yes |archiveurl=https://web.archive.org/web/20101210181648/http://www.trsc.com.tw/products2.htm |archivedate=2010-12-10 }}</ref>，改造後改為9輛為一個編組，而避免編號上的不便，更名為[[台鐵EMU1200型電聯車|EMU1200型電聯車]]，並將原有的形式與車號在重新編碼後消滅<ref name="IH1" /><ref name="IH3" /><ref>{{cite web|url=http://emu300ct.myweb.hinet.net/index/tramus/EMU1200.htm | title=Train Collection：九輛一組的"紅斑馬"特急~EMU1200|accessdate=2011-02-14 |language=zh-hant}}</ref><ref>{{cite web |url=http://www.ccmetro.com/newsite/readnews.aspx?id=8780 |title=台灣自強號EMU200型電聯車：中國城市軌道交通網 |accessdate=2011-02-14 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20160128175722/http://www.ccmetro.com/newsite/readnews.aspx?id=8780 |archivedate=2016-01-28 }}</ref>。
