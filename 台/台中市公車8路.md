'''台中市公車8路'''目前行駛自道禾六藝文化館至逢甲大學（逢甲路），由[[台中客運|台中客運]]營運，主要行駛於[[北屯路|北屯路]]、[[興安路|興安路]]、[[松竹路|松竹路]]和[[后庄北路|后庄北路]]，由[[台中客運|台中客運]]營運。

== 歷史 ==
*2013年2月18日：調整部份路段北屯路－文心路－興安路－松竹路，新增文心路之「四維國小」及興安路之「平田-{里}-」、「平興里」、「松安里」等四站位，並裁撤北屯路之「特力屋北屯店」、「平田-{里}-」、「北新國中」及松竹路之「舊社公園（松竹路）」、「真耶穌教會」等五站位。<ref>[http://60.249.10.173/newsfile/shownews.php?newsno=29 台中客運 最新消息 8路調整部份路段]</ref>
*2014年1月20日：原終點「七張犁」廢止，裁撤「松竹敦化路口」、「松竹-{后}--{庄}-路口」改行經四平路、-{后}--{庄}-路、-{后}--{庄}-北路至「-{后}--{庄}-北中清路口」端點（迄點），新增「三民錦中街口」、「松竹修齊街口」、「馬禮遜學校」、「四平衛生所」、「四張犁國小」、「-{后}--{庄}-松竹路口」、「-{后}--{庄}-七街口」等站位，「松竹國小」更名為「松竹昌平路口」；班次由原16班增加至28班。<ref>{{Cite web |url=http://www.tcbus.com.tw/newsfile/shownews.php?newsno=125 |title=8路1/20延駛至后庄北中清路口公告 |accessdate=2016-03-01 |date=2014-01-15 }}</ref>
*2014年4月20日：於「馬禮遜學校」與「四平衛生所」之間新增「四平水湳路口」。
*2014年5月20日：「四平衛生所」更名為「四民衛生所」；「松竹梅川東五路口」更名為「松竹梅川東路口」。
*2014年11月1日：原「-{后}--{庄}-北中清路口」端點延駛至「逢甲大學（逢甲路）」，路線名更改為（綠川東站－逢甲大學），途中新增「-{后}--{庄}-國宅」、「中清同榮路口」、「僑光科技大學」、「漢翔公司（星享道）」、「福星西安街口」、「逢甲大學（福星路）」；「平田-{里}-」更名為「興安旅順路口」、「平興里」更名為「興安崇德二路口」、「崇德六路口」更名為「北新國中（興安路）」、「松安里」更名為「興安松竹路口」。<ref>{{Cite web |url=http://www.tcbus.com.tw/newsfile/shownews.php?newsno=234 |title=8路11/1起延駛至逢甲大學 |accessdate=2016-03-01 |date=2014-10-21 }}</ref>
*2014年12月15日：增停「文心興安路口」。<ref>[http://www.tcbus.com.tw/newsfile/shownews.php?newsno=250 8路增停文心興安路口]</ref>
*2015年3月1日：「第一信用」撤站並併入「彰化銀行（臺灣大道）」。<ref>{{Cite web |url=http://www.tcbus.com.tw/newsfile/shownews.php?newsno=275 |title=3/1站位新增、取消及更名公告 |accessdate=2016-03-01 |date=2015-02-26 }}</ref>

== 路線基本資料 ==
單程行車里數去程往逢甲大學17.7公里、返程往道禾六藝文化館15.8公里，單程行駛時間約1小時10分。<br>
往逢甲大學（逢甲路）共51站，往往道禾六藝文化館共54站。<br>

自道禾六藝文化館起，去程沿貴和街、[[自由路_(台中市區)|自由路]]、[[民權路_(台中市)|民權路]]、[[建國路_(台中市)|建國路]]、[[臺灣大道|臺灣大道]]、興中街、[[公園路|公園路]]、[[三民路|三民路]]三段、[[北屯路|北屯路]]、[[文心路|文心路]]、興安路、[[松竹路|松竹路]]、四平路、-{后}--{庄}-路、-{后}--{庄}-北路、[[中清路|中清路]]、[[黎明路|黎明路]]三段、福星北路、福星路、逢甲路到逢甲大學（逢甲路）；返程行駛至三民路後，則沿[[中山路|中山路]]、[[自由路|自由路]]、[[臺灣大道|臺灣大道]]接原線至道禾六藝文化館。<ref>[http://www.tcbus.com.tw/image/lineimage.php?imagetest=8 台中客運8路路線圖]</ref>
