'''台湾白及'''（[[学名|学名]]：{{lang|la|'''''Bletilla formosana'''''}}），又名'''小白及'''。多年生草本植物，為臺灣原生蘭。與常見中藥材[[白芨|白及]](學名:''B. striata'') 同為[[蘭科|蘭科]](Orchidaceae)[[白芨屬|白及屬]](''Bletilla'')植物。主要分佈於臺灣與中國南部。其假球莖可做為中藥材，花朵具觀賞價值。 <ref name=":0">{{Cite web|url=http://tai2.ntu.edu.tw/ebook/ebookpage.php?book=Fl.%20Taiwan&volume=5&list=13448|title=電子書 台灣植物誌第一版  Flora of Taiwan, 1st edition   5: 887 - Plants of Taiwan 台灣植物資訊整合查詢系統|accessdate=2017-03-30|work=tai2.ntu.edu.tw}}</ref><ref>{{Cite book|title=世界原生蘭圖鑑|last=奇藤龜三|first=|publisher=晨星出版社|year=2007|isbn=|location=|pages=141}}</ref>

== 植株型態 ==
地生性。植株大小差異大，高度介於10-60公分。平地或低海拔的植株較高(可達60公分)，而中高海拔則以小型株(10公分)居多。其假球莖可為球狀、卵狀、陀螺狀與不規則扁壓狀。葉片由假球莖上的角錐狀小芽球發育而來，一個球莖可生長4-7片線狀葉。葉為青綠色、紙質。葉長10-50公分、寬0.8-3.5公分。花莖則由假球莖頂葉間向上抽出。花軸纖細直立，長15-90公分。總狀花序，著花5至30朵，由下往上逐次開放，每次僅1至2朵盛開。花朵半張，花色白，略帶淡粉或紫色。唇盤上有五條波狀龍骨，龍骨前端為鮮黃色，末端則為白色並具有棕紅色斑，龍骨間則佈有紫色或褐色斑點。萼片為窄橢圓形，長1.9-2.1公分，寬0.56-0.65公分。花瓣為歪橢圓形，長1.9-2.1公分。唇瓣成三裂片: 側裂片甚小並彎曲向上，中裂片則近乎圓形且邊緣為波浪狀。蕊柱長約1.3公分，淡紅色，腹面具有紅棕色斑。具8塊花粉塊，花藥帽為圓球形。花期3-6月，於海拔1800公尺上的植株花期可延長至10月。果實為橢圓形[[蒴果|蒴果]]，長約2.5公分。<ref name=":0" /> <ref name=":1">{{Cite book|title=台灣野生蘭賞蘭大圖鑑（上）|last=林維明|first=|publisher=天下遠見出版股份有限公司|year=2006|isbn=|location=|pages=48}}</ref><ref name=":2">{{Cite book|title=台灣蘭科植物|last=林讚標|first=|publisher=魯風印書有限公司|year=1977|isbn=|location=|pages=62-64}}</ref><ref name=":3">{{Cite book|title=台灣野生蘭(上冊)|last=鐘詩文|first=|publisher=行政院農業委員會林務局|year=2008|isbn=|location=|pages=30}}</ref>

== 地理分佈 ==
臺灣白及分佈於臺灣全島、[[蘭嶼|蘭嶼]]、[[龜山島|龜山島]]及中國南部。好生於濕潤向陽岩石與平野潮濕的草叢中。海拔100至3300公尺皆可發現其蹤跡。可見於[[蘇花公路|蘇花公路]]與[[中橫公路|中橫公路]]路旁，花蓮洛韶與[[天祥_(台灣)|天祥]]的山壁邊坡。<ref name=":1" /> <ref name=":2" /><ref name=":3" />

== 命名故事 ==
1906年6月11日[[川上瀧彌|川上瀧彌]]與[[森丑之助|森丑之助]]共同於宜蘭縣九芎湖附近採集，並在1911年由[[早田文藏|早田文藏]]發表命名為''Bletia formosana''。但由於白及屬(''Bletilla'')與布萊特蘭屬(''Bletia'')名稱相似，早田文藏便將台灣白及屬名一開始誤用為''Bletia''。此外，於1906年11月川上瀧彌與森丑之助也在玉山海拔3300公尺處採得一蘭，並於1911年早田文藏將之發表為''Bletia morrisonicola'' ，其模式標本植株較小，葉長約為12公分、葉寬為0.6公分。後經鑑定，判定''Bletia morrisonicola'' 型態差異是受生態環境影響，其與台灣白及(''Bletia formosana'')應為同種植物。 <ref name=":2" />

== 染色體數目 ==
1939年T. Midzuno和1965年R.Tanaka計數了台灣白及的染色體數為2n=36，而白及染色體數為2n=32，若將上述兩種白及雜交則可得染色體數2n=34的後代。 <ref name=":2" />

== 親緣關係 ==
利用分子標誌之逢機增幅多型性DNA (random amplified polymorphic DNA, RAPD)對採集自台灣不同區之台灣白及與白及屬種原進行UPGMA(unweighted pair-group mean arithmetical)親源分析，發現蘭嶼白及被分類於台灣白及群組內，推測蘭嶼白及與台灣白及應為同種。 <ref>{{Cite journal|title=以RAPD技術進行白及種原之遺傳歧異度分析|url=http://www.airitilibrary.com/Publication/alDetailedMesh?DocID=00224847-200912-201004160135-201004160135-254-264|last=曹進義|last2=許絾誌|date=2009-12-01|journal=台灣農業研究|issue=4|doi=10.6156/jtar/2009.05804.02|volume=58|issn=0022-4847|last3=闕甫伈|last4=陳威臣|last5=夏奇鈮}}</ref>

== 栽培技術 ==
臺灣白及種子於自然環境下不易發芽，傳統採用塊莖分割法繁殖，但因生長速度緩慢、產量不足，無法大量生產以供藥用。故近年來農業試驗單位積極研究[[組織培養|組織培養]]的栽培模式，利用無菌播種誘導種子萌芽，並提高萌芽率。根據農試所研究，在2000 [[流明|流明]]光照下，使用3%[[过氧化氢|過氧化氫]]預措處理種子5分鐘，可使種子提早發芽，且發芽率可達98%。 <ref>{{Cite journal|title=臺灣白及 [Bletilla formosana (Hayata) Schlechter] 之組織培養 (二) 培養基組成分對未成熟種子萌芽與小苗發育之影響|author=葉豐次|url=|journal=中華農藝|issue=|doi=|others=林郁進.陳忠川.邱年永.蔡新聲|year=1994|volume=43|page=40-50|pmid=}}</ref>

== 藥用功能 ==
白及於《[[神农本草经|神農本草經]]》記載: 「味苦，平。主治癰腫，惡瘡，敗疽，傷陰，死肌，胃中邪氣，賊風鬼擊，痱緩不收。」。並根據藥理學研究發現白及甘露聚醣具有止血、抗炎與抗腫瘤之作用，可至肺炎、肺結核與十二指腸潰瘍等疾病。<ref>{{Cite web|url=http://www.coa.gov.tw/ws.php?id=22091|title=台灣原生蘭科植物—台灣白及成分分析及種苗繁殖技術(農委會)|accessdate=2017-03-30|last=COA|work=www.coa.gov.tw}}</ref><ref>{{Cite journal|title=白及甘露聚糖抗胃溃疡及抗炎|author=王红英|url=|journal=浙江中醫藥大學學報|issue=1|doi=|others=|year=2009|volume=33|page=119-121|pmid=}}</ref>
