'''中興號'''為[[台灣|台灣]]的[[公路客運|公路客運]]車款之一，1976年由[[臺灣省公路局|臺灣省公路局]]開始使用於[[長途客運|長途客運]]，1980年[[臺灣汽車客運公司|臺灣汽車客運公司]]（臺汽）成立後繼續使用。

== 歷史 ==
隨著1970年代台灣[[中華民國國道|高速公路]]的通車，1976年[[臺灣省公路局|臺灣省公路局]]開始打造高級冷氣車車種，命名為中興號<ref name="中華公路史"/>{{rp|316}}<ref name="吳宗憲"/>{{rp|77}}，並於該年8月1日正式加入公路局車隊<ref name="時光客運"/>{{rp|40}}<ref name="yahoo"/><ref name="吳宗憲"/>{{rp|78}}。中興號車內配置有坐臥兩用絨布椅、四聲道音響、大功率空調冷氣，被視為當時最高級的客運車輛，並逐漸取代了其他無冷氣空調的車種<ref name="yahoo">{{cite web|title=那些年陪著我們南征北討的公車(一) 公路客運龍頭 台汽公路客運介紹-Part I|publisher=Yahoo!奇摩汽車頻道|date=2014-03-05|url=https://autos.yahoo.com.tw/news/那些年陪著我們南征北討的公車-一-公路客運龍頭-台汽公路客運介紹-part-075146911.html|accessdate=2018-09-24}}</ref><ref name="吳宗憲"/>{{rp|77}}。車上有[[車掌小姐|女性隨車服務員]]，負責簡介沿途站別，並遞送書報、茶水與毛巾<ref>{{cite web|title=美麗的驕傲 - 車掌小姐（1930年代-）-台灣女人|publisher=國立臺灣歷史博物館|url=https://women.nmth.gov.tw/information_89_39872.html}}</ref>。

由於中興號的普及，民營客運業者開始引入冷氣車<ref name="yahoo"/>。

中興號早期塗裝為咖啡色線條，1990年代之後改為與[[國光號|國光號]]相同的紅白藍三色塗裝，之後又出現[[雲豹|雲豹]]紋和紫色的新塗裝<ref name="時光客運">{{cite book|title=時光客運: 一趟懷舊與知性的公車之旅|author=楊浩民|publisher=獨立作家－釀出版|date=2015-12-01|isbn= 9789865716684}}</ref>{{rp|42}}。

爾後由於車齡漸高，車輛逐漸被淘汰，自1984年以後營業車輛漸減。2000年僅剩國光號369輛與中興號884輛<ref name="吳宗憲"/>{{rp|144}}。

== 設備 ==
中興號車內配置有坐臥兩用絨布椅、四聲道音響、大功率空調冷氣，每車有43個座位，被視為當時最高級的客運車輛<ref name="yahoo"/><ref name="吳宗憲">{{cite thesis|title=戰後臺灣公路運輸政策與公營客運之變遷|author=吳宗憲|publisher=國立中央大學|degree=碩士|year=2017|url=https://hdl.handle.net/11296/huzpg8}}</ref>{{rp|77}}。

早期中興號使用[[西德|西德]][[朋馳|朋馳]]OF-1417底盤，再由[[國產汽車_(台灣)|國產汽車]]打造車體<ref name="吳宗憲"/>{{rp|78}}。

== 行駛路線 ==
中興號主要行駛於[[中華民國省道|省道]]路線和較短的[[中華民國國道|國道]]路線，較長的國道路線則由[[國光號|國光號]]行駛<ref name="時光客運"/>{{rp|42}}。
*1978年11月啟用：台北至台中、台中至高雄
*1979年啟用：台北至台南、台北至嘉義、台北至彰化、員林、台中至台南、高雄至嘉義<ref name="中華公路史">{{cite book|title=中華公路史 下部|author1=胡美璜|author2=楊延英|author3=石中光|publisher=臺灣商務印書館|isbn=9786660507356|year=1984}}</ref>{{rp|304}}。

此外，也提供包車服務，往來[[台北市|台北市]]與[[中正國際機場|中正國際機場]]之間<ref name="中華公路史"/>{{rp|306}}。
