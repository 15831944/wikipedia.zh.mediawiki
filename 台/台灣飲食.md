'''臺灣飲食'''，從最早期的台灣原住民飲食、荷蘭及西班牙殖民者文化、到清治時期漢族閩南、漢族客家人的飲食，日治時期的日本料理與引進西方飲食、而後[[中華民國國民政府|中華民國國民政府]]播遷來台，帶來中國大陸各省的飲食文化，發展出一套揉合大江南北與西餐的飲食文化。

== 傳統台式飲食==
{{MAIN|臺灣菜}}
以[[中華料理|中華料理]]中的[[閩菜|閩南菜]]與[[客家菜|客家菜]]為主。
[[File:臺灣鄉間的傳統早餐.JPG|thumb]]外，還有[[花生|花生]]、[[筍乾|筍乾]]、[[香腸|香腸]]、[[肉鬆|肉鬆]]、[[麵筋|麵筋]]、[[醬瓜|醬瓜]]等配菜。]]
[[早餐|早餐]]傳統為白粥配醬瓜等小菜、或[[燒餅|燒餅]][[油條|油條]]或[[飯糰|飯糰]]，配上[[豆漿|豆漿]]或[[米漿|米漿]]。也有[[蘿蔔糕|蘿蔔糕]]和[[美而美|美而美]]漢堡。

==素食==
[[素食|素食]]餐廳一般吃素者由所經營，一些比較有規模的星級飯店也會供應素食齋菜，並供應一些仿葷的[[素食|素食]]菜色。很傳統。城市巷弄間的自助餐店亦常有素食菜色可選擇。

== 外省菜 ==
隨著[[國共內戰|國共內戰]]，不少中國大陸移民帶來許多各省特色菜色，如[[上海菜|上海菜]]、[[江浙菜|江浙菜]]、[[淮揚菜|淮揚菜]]、[[廣東菜|廣東菜]]、[[湘菜|湘菜]]、[[川菜|川菜]]、[[魯菜|魯菜]]、[[豫菜|豫菜]]、[[東北菜|東北菜]]、[[晉菜|晉菜]]、[[鄂菜|鄂菜]]。今已與本地口味混合，不全然像原先當地的口味。[[眷村|眷村]]舊址附近常有外省菜小餐館。

== 小吃 ==
{{Main|台灣小吃}}

如[[台灣夜市|台灣夜市]]、[[路邊攤|路邊攤]]、土雞城所見。百元快炒也是台灣飲食文化的一部分，通常以海產為主要食材，有提供啤酒促銷。有別於中國北方熱的鹹的[[豆腐腦|豆腐腦]]，台灣[[豆花|豆花]]通常是甜的，在夏天販售的通常是冰的。另外有一種在燒餅油條店賣的[[鹹豆漿|鹹豆漿]]，是熱的鹹湯裡有豆花。[[熱炒店|熱炒店]]是很本土的餐飲形式。

== 咖啡座 ==
從臺灣日治時期就引進。連鎖咖啡店有丹堤咖啡、怡客咖啡、西雅圖咖啡、日資的羅多倫咖啡、中外合資的統一Starbucks咖啡、85度C、路易莎。咖啡店也是藝術家、音樂家、作家，沉思、聚會、表演的空間。

== 飲料店 ==
臺灣戰後時期，獨自開發出新型態的泡沫紅茶飲料店，從早期的小歇、快可立、葵可利，到後來的[[85度C|85度C]]、[[CoCo|CoCo]]、[[日出茶太|日出茶太]]。
