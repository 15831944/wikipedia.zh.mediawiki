'''台灣浩鼎生技股份有限公司'''（[[英語|英語]]：OBI Pharma, Inc.，簡稱'''浩鼎'''），是一家生物製藥公司，專注於新藥研發。

[[中裕新藥|中裕新藥]]與浩鼎都是[[潤泰企業集團|潤泰企業集團]]總裁[[尹衍樑|尹衍樑]]投資的公司。<ref>{{cite news|author=黃文奇|title=浩鼎市值千億 尹衍樑大贏家|url=http://udn.com/news/story/7251/1338610|accessdate=2015-11-26|newspaper=《[[聯合報|聯合報]]》 |date=2016-04-09}}</ref>

2015年3月23日，浩鼎在[[中華民國證券櫃檯買賣中心|中華民國證券櫃檯買賣中心]][[上櫃|上櫃]]，[[股票代號|股票代號]]為4174<ref>{{otc-gretai|4174}}</ref>。

== 概要 ==
台灣浩鼎於2002年成立，初為美國上市公司{{le|Optimer Pharmaceuticals|Optimer Pharmaceuticals}}在台百分之百投資的子公司。

== 事件 ==
1998年，[[翁啟惠|翁啟惠]]與[[張念慈|張念慈]]等人在美國創辦生技製藥公司"Optimer Pharmaceuticals"。

2002年，美國Optimer Pharmaceuticals來到台灣成立一家百分之百投資的子公司台灣浩鼎，翁啟惠是共同創辦人。

2006年，Optimer Pharmaceuticals申請在美國納斯達克上市，依據當時之公開說明書資料翁啟惠是Optimer Pharmaceuticals的第五大股東，而前四大股東都是法人股東（其它製藥公司與生技創投基金）<ref>[https://www.sec.gov/Archives/edgar/data/1142576/000104746906013830/a2174167zs-1.htm 美國證券交易委員會"Optimer Pharmaceuticals 2006年11月9日S1報表102頁。"]</ref>。翁啟惠辭去Optimer公司的董事職務，接任中研院院長。

2007年，2月Optimer Pharmaceuticals在美國[[納斯達克|納斯達克]]交易所股票上市。

2012年5月15日經中華民國主管機關[[金融監督管理委員會|金融監督管理委員會]]證期局核准成為一家公開發行的公司，於2012年12月12日登錄興櫃，並於2015年3月23日正式掛牌上櫃（{{otc|4174}})。

台灣大富豪之一、潤泰總裁[[尹衍樑|尹衍樑]]在2012年以總價十八億元新台幣，買下浩鼎母公司Optimer所持有的四三％的股權，成為浩鼎的最大股東。浩鼎最受關注的乳癌新藥OBI-822g4則是中研院院長翁啟惠的技轉。<ref>{{Cite web |url=http://ctee.com.tw/Album/Content.aspx?albumid=136d74ef-11e8-4398-bc38-0672210b2a2b&newsid=702098 |title=張念慈聲明 疑雲將真相大白 |access-date=2017-01-09 |archive-url=https://web.archive.org/web/20170109190355/http://ctee.com.tw/Album/Content.aspx?albumid=136d74ef-11e8-4398-bc38-0672210b2a2b&newsid=702098 |archive-date=2017-01-09 |dead-url=yes }}</ref>2012年底張念慈家族與翁家族持股共約10%左右（Optimer在2012年12月總股數約4600萬股，股價約15美元），故所持股價值約7000萬美元。<ref>{{Cite web |url = http://www.setn.com/News.aspx?NewsID=127514 |title = 宇昌案後再陷浩鼎風暴！蔡英文連踩兩次生技地雷 |author = 黃琴雅 |date = Mar 2, 2016 |publisher = 三立新聞網  |language = zh-tw  |accessdate = Apr 5, 2016 }}</ref><ref>{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/1643918 |title = 張念慈再發聲明 說明翁啟惠女兒31元認股由來    |date = Mar 25, 2016 |publisher = 自由時報 |language = zh-tw  |accessdate = Apr 5, 2016 }}</ref>
