'''臺灣個人遊'''計劃，臺灣稱「陸客自由行」，是一個容許[[大陸地區|大陸地區]]人民持[[中華民國臺灣地區入出境許可證|中華民國臺灣地區入出境許可證]]及[[大陸居民往來台灣通行證|大陸居民往來台灣通行證]]以個人旅遊的方式前往[[中華民國|中華民國]]臺灣地區旅遊的計劃，此計畫經過雙方政府核准後實施，自2011年6月28日推行。

一般而言，只要在計劃範圍內的城市，便可經簡單的簽注手續前往臺灣境內旅遊，期間最多可逗留15天。現時，中國大陸共47個城市的居民，可以“個人遊（自由行）”身份到臺灣旅遊。

== 歷程 ==
*2011年6月28日，[[中國大陸|中國大陸]]制订《大陆居民赴台湾地区个人旅游注意事项》、《大陆居民赴台湾地区旅游管理办法》，對三个城市（[[北京|北京]]、[[上海|上海]]、[[厦门|厦门]]）率先开放对台湾的自由行<ref name="cnta-20110629"/>。通常地，来到台湾自由旅游的中国大陆游客，可以在其签证规定的日期内依照自己的意愿访问、参加台湾（除禁止的区域和活动外）任何地方、活动。
*2011年7月29日，開放福建省9個城市居民（包括[[廈門|廈門]]、[[福州|福州]]、[[莆田|莆田]]、[[泉州|泉州]]、[[漳州|漳州]]、[[龍岩|龍岩]]、[[三明|三明]]、[[南平|南平]]、[[寧德|寧德]]）赴金門馬祖澎湖個人遊，但不得中轉至台灣本島（俗稱：小三通自由行）。
*2012年4月28日，開放了第二批城市對台灣的自由行，分成兩階段，第一階段開放[[天津|天津]]、[[重慶|重慶]]、[[南京|南京]]、[[杭州|杭州]]、[[廣州|廣州]]、[[成都|成都]]六座城市，第二階段於2012年8月28日開放[[濟南|濟南]]、[[西安|西安]]、[[福州|福州]]、[[深圳|深圳]]四座城市。<ref>{{cite news|url=http://news.wenweipo.com/2012/04/01/IN1204010037.htm|title=10市納第二批赴台個人遊試點|publisher=文匯網|language=zh-hk}}</ref><ref>{{cite news|url=http://www.nownews.com/2012/08/24/301-2848330.htm|title=陸客自由行　再開放濟南、西安、福州、深圳4個城市|language=zh-tw}}</ref>
*2012年8月28日，大陸官方新增開放海西地區11個城市（包括浙江的[[溫州|溫州]]、[[衢州|衢州]]、[[麗水|麗水]]，江西的[[贛州|贛州]]、[[撫州|撫州]]、[[上饒|上饒]]、[[鷹潭|鷹潭]]以及廣東的[[梅州|梅州]]、[[潮州|潮州]]、[[汕頭|汕頭]]、[[揭陽|揭陽]]）居民赴金門馬祖澎湖自由行政策，但不得中轉至台灣本島（俗稱：小三通自由行）。連同2011年7月29日開放的福建省內的9個城市，合計開放陸客小三通自由行的城市已達20個城市。
* 2013年6月28日，開放第三批城市對台灣自由行，分成兩個階段，第一階段為[[瀋陽|瀋陽]]、[[鄭州|鄭州]]、[[武漢|武漢]]、[[蘇州|蘇州]]、[[寧波|寧波]]、[[青島|青島]]六座台灣自由行城市，第二個階段於8月28日起開放[[石家莊|石家莊]]、[[長春|長春]]、[[合肥|合肥]]、[[長沙|長沙]]、[[南寧|南寧]]、[[昆明|昆明]]和[[泉州|泉州]]七座城市。<ref>{{cite news|url=http://news.ifeng.com/taiwan/3/detail_2013_06/16/26449114_0.shtml|title=沈阳、郑州、武汉等13个城市开放赴台个人游|publisher=鳳凰網}}</ref>
* 2014年7月18日，開放第四批來台自由行，新增開放[[哈爾濱|哈爾濱]]、[[太原|太原]]、[[南昌|南昌]]、[[貴陽|貴陽]]、[[大連|大連]]、[[無錫|無錫]]、[[溫州|溫州]]、[[中山市|中山]]、[[煙台|煙台]]、[[漳州|漳州]]等10個城市，並於8月18日啟動。<ref>{{cite news|url=http://udn.com/NEWS/BREAKINGNEWS/BREAKINGNEWS4/8812428.shtml|title=陸客來台自由行 新增哈爾濱等10城市|publisher=聯合新聞網}}</ref>
* 2015年3月18日，開放第五批來台自由行，新增開放[[海口|海口]]、[[呼和浩特|呼和浩特]]、[[蘭州|蘭州]]、[[銀川|銀川]]、[[常州|常州]]、[[舟山|舟山]]、[[惠州|惠州]]、[[威海|威海]]、[[龍岩|龍岩]]、[[桂林|桂林]]、[[徐州|徐州]] 等11個城市，並於4月15日啟動，合計開放陸客來臺自由行城市已達47個城市。

現時，開放陸客來臺自由行(包含金門馬祖澎湖的小三通自由行)的城市有47個<ref>[https://www.immigration.gov.tw/ct.asp?xItem=1149824&ctNode=32630&mp=1 設籍於哪些城市的大陸地區人民可來臺自由行? 應該如何申請? 可委託親友代辦嗎?]中華民國移民署 2016-02-04</ref>，另外有14個城市([[莆田|莆田]]、[[三明|三明]]、[[南平|南平]]、[[寧德|寧德]]、[[衢州|衢州]]、[[麗水|麗水]]、[[贛州|贛州]]、[[撫州|撫州]]、[[上饒|上饒]]、[[鷹潭|鷹潭]]、[[梅州|梅州]]、[[潮州|潮州]]、[[汕頭|汕頭]]、[[揭陽|揭陽]])的人民可赴金門馬祖澎湖個人遊，但不得中轉至台灣本島（俗稱：小三通自由行）。兩者個人遊最大不同處係是否需要提出財力證明。

== 特殊情形 ==
[[中国大陆|中国大陆]]非自由行城市[[户籍|户籍]]的大学生（含[[研究生|研究生]]）在开放自由行户籍的城市就读的，可以办理自由行手续并享受自由行待遇。此种情形的学生身份申请人需要在学校相关部门开具带有公章的在读证明，之后携带申请人本人学生证、户口本、身份证到所就读学校所在地的公安局关机出入境管理部门办理即可，具体办理程序与规定因各地而异，解释权在当地公安局关机出入境管理部门。

== 陸客到金門、馬祖及澎湖個人遊的型態（含團體旅遊型態介紹） ==
陸客到金馬澎旅遊的型態可分為個人遊及團體遊等二大類，其中個人遊有二種方式，團體遊有三種方式，共五種方式。
*個人：
**第一種：2011年6月28日啟動的廈門、北京、上海等3城市居民來台灣（含金門、馬祖和澎湖）個人遊。截至目前開放了五批共47個城市。
**第二種：2011年7月29日上路的福建省內的9個城市居民赴金門、馬祖和澎湖個人遊，不得中轉至台灣本島（俗稱：小三通自由行）。
***2012年8月28日，再開放海西地區11個城市（包括浙江的溫州、衢州、麗水，江西的贛州、撫州、上饒、鷹潭以及廣東的梅州、潮州、汕頭、揭陽）居民赴金馬澎自由行政策。截至目前開放了20個城市。
***2015年1月1日起，陸客到金門、馬祖和澎湖（不含中轉到台灣）將可辦理落地簽入境。
*團體：
**第一種：2004年12月7日開放的大陸居民赴金門團體遊。
**第二種：2008年9月開放的大陸居民得經金門赴台旅遊或赴台旅遊經金門往返兩岸的團體旅遊。
**第三種：2011年6月13日開跑的非福建省居民來廈暫住赴金門團體遊（屬延伸性旅遊）。
***2013年6月16日第五屆海峽論壇宣佈，非福建省居民在廈門暫住人員得赴金門遊從一日遊開放為二日遊。
