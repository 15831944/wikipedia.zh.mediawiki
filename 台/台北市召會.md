'''臺北市召會'''是[[教會聚會所|教會聚會所]]在[[臺灣|臺灣]][[臺北市|臺北市]]建立的[[召會|召會]]，目前擁有74個聚會所<ref>[http://www.churchintaipei.org/site-administrator/map-meeting-halls.html 《臺北市召會各會所地圖》，2019年2月28日]</ref>。

== 歷史 ==
[[File:Hall_1,_The_Church_in_Taipei_20130919.jpg|thumb]]
1946年[[日本戰敗|日本戰敗]]以後，陸續有來自[[中國大陸|中國大陸]]「[[地方教會|地方教會]]」（現稱「召會」）的信徒遷居[[台灣|台灣]]，開始在一起[[禱告|禱告]]、讀《[[聖經|聖經]]》；1947年5月，開始舉行[[擘餅|擘餅]]聚會，是臺北市召會的開端。 

1948年中國大陸政局變化前夕，原[[煙台教會|煙台教會]]長老[[趙靜懷|趙靜懷]]、[[張晤晨|張晤晨]]、[[孫豐露|孫豐露]]、[[劉效良|劉效良]]都移居台灣，使得臺北市召會的根基大大加強。不久，[[南京教會|南京教會]]長老[[張郁嵐|張郁嵐]]也到台灣。1949年，[[倪柝聲|倪柝聲]]設立上述幾人為臺北市召會的長老。 

1949年初，臺北常聚會的信徒不足100人。這一年5月，[[李常受|李常受]]受倪柝聲指示，全家來到臺北，從8月開始大傳[[福音|福音]]，到年底聚會人數達到900人。幾年內，人數又增加到數萬人。

1990年代，臺北市召會在[[信義區_(臺北市)|信義區]][[信義路|信義路]]與[[基隆路|基隆路]]口建造了[[信基大樓|信基大樓]]，包括樓上二十二層與地下五層。信基大樓三至五樓為會所本身使用，二十二樓有[[臺灣福音書房|臺灣福音書房]]編輯部及部分會所單位，其餘樓層出租。

臺北市召會第五十聚會所成立於2013年1月6日。

== 信仰 ==
[[File:Sanmin_Branch,_SCSB_and_Hall_41,_The_Church_in_Taipei_20120204.jpg|thumb]]）]]
臺北市召會承襲[[宗教改革|宗教改革]]以來基督徒的信仰，強調[[唯獨聖經|唯獨聖經]]。因此他們不強調[[信經|信經]]，也不接受因為[[基督新教|基督新教]]或[[天主教|天主教]]傳統或習俗而與聖經違背的[[神學|神學]]。

臺北市召會的[https://web.archive.org/web/20111005181853/http://www.churchintaipei.org/about-us/statement-of-faith.html 信仰聲明]如下：
# 我們相信：全本聖經是神完整的啟示，字字都是聖靈所默示的。
# 我們相信：神是獨一的三一神，父、子、靈同等，共存，從亙古到永遠。
# 我們相信：神的兒子，就是神自己，成為肉身，成為人，名叫耶穌，由童女馬利亞所生，來作我們的救贖主和救主。
# 我們相信：耶穌是真正的人，曾在地上生活三十三年半，使人認識父神。
# 我們相信：耶穌是神用聖靈所膏的基督，為我們的罪死在十字架上，流血為我們成功了救贖。
# 我們相信：耶穌基督埋葬三日以後，就從死裡復活（肉身上和靈性上）；並在復活裡成為賜生命的靈，將祂自己分賜到我們裡面，作我們的生命和一切。
# 我們相信：基督復活後升到天上，被神立為萬有的主。
# 我們相信：基督升天以後，將神的靈澆灌下來，把祂所揀選的肢體浸入一個身體裡；神的靈，也就是基督的靈，今天在地上運行，叫罪人知罪，重生神所揀選的人，住在基督的眾肢體裡，使他們在生命裡長大，並建造基督的身體，讓祂得著完滿的彰顯。
# 我們相信：在這世代的末了，基督要回來提接祂的眾肢體，審判世界，得著全地，並建立祂永遠的國度。我們相信得勝的聖徒要在千年國度裡與基督一同作王掌權；並且所有在基督裡的信徒，都將有分於新天新地新耶路撒冷裡神聖的福分，直到永遠。
