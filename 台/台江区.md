'''台江区'''（[[闽东语|闽东语]]：''Dài-gĕ̤ng''），旧称南台，是[[中华人民共和国|中国]][[福建省|福建省]][[福州市|福州市]]所辖的一个区。台江区位于福州城区中部，为省会福州中心城区、[[闽江|闽江]]下游北岸，面积18平方千米，是福建省面积最小人口密度最高的的市辖区，人口31万。邮政编码350004。区人民政府驻瀛洲街道台江路88号。

== 名称由来 ==
汉高祖五年(公元前202年)，无诸在南台山(今大庙山)筑台受封为闽越王，该台称越王台。武帝元光元年(公元前134年)至元封六年(公元前110年)，东越王余善在大庙山临江(闽江)垂钓，后筑台称钓龙台。[[五代|五代]][[后梁|后梁]][[开平|开平]]二年(908年)，[[王审知|王审知]]在福州筑“夹城”，“登南城翘望，有台(指越王台、钓龙台)临江”。台江、南台故而得名<ref name="台江区志概述">{{cite web |url=http://www.fzdqw.com/ShowText.asp?ToBook=1110&index=7& |title=概述 |accessdate=2008-08-20 |work=台江区志 |publisher=福州市地方志编纂委员会 |language=zh-hans |archive-url=https://web.archive.org/web/20160304195938/http://www.fzdqw.com/ShowText.asp?ToBook=1110&index=7& |archive-date=2016-03-04 |dead-url=yes }}</ref>。

== 地理概况 ==
台江区的陆域面积为18平方公里，水域面积1.91平方公里。

台江的大部分地区原来均为水域。大约在新石器时代晚期或青铜器时代，北部的闽江流域由于地壳运动和泥沙沉积逐渐形成一片冲积平原<ref name="台江区志概述">{{cite web |url=http://www.fzdqw.com/ShowText.asp?ToBook=1110&index=7& |title=概述 |accessdate=2008-08-20 |work=台江区志 |publisher=福州市地方志编纂委员会 |language=zh-hans |archive-url=https://web.archive.org/web/20160304195938/http://www.fzdqw.com/ShowText.asp?ToBook=1110&index=7& |archive-date=2016-03-04 |dead-url=yes }}</ref>。

台江区内主要河道有晋安河、白马河、打铁港、东西河、茶亭河、达道河、瀛洲河、光明港、大庆河、三捷河、奋斗河、济南河、新透河、浦西河等。

==行政区划==
辖10个街道，分别是茶亭街道、洋中街道、苍霞街道、义洲街道、上海街道、瀛洲街道、新港街道、后洲街道、鳌峰街道、宁化街道<ref>注：2004年，原双杭街道并入后洲街道、原帮州街道并入苍霞街道。</ref>。

==交通==
台江区辖区内主干道有[[八一七路|八一七路]](中路、南路)、江滨西大道、江滨中大道、工业路、五一路、六一路(六一中路)、国货路、台江路、达道路、达江路、广达路、连江路(南段)、白马路(南段)，主要桥梁有[[解放大桥|解放大桥]]、闽江大桥、鳌峰大桥、三县洲大桥、尤溪洲大桥、金山大桥(为鼓楼、台江交界)、瀛洲立交桥（已于2014年拆除）、洋头口立交桥(已于09年拆除)等。
