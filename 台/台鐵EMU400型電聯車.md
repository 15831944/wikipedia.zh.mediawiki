'''台鐵EMU400型電聯車'''，簡稱'''400型電聯車'''，為[[台灣鐵路管理局|台鐵]]於1990年啟用的一款[[電聯車|電聯車]]{{r|大事記}}，亦為台鐵所採用之第一款[[通勤電聯車|通勤電聯車]]{{r|鐵道百科|page=102}}，由於[[台鐵EMU800型電聯車|EMU800型電聯車]]的引進，已於2015年退出正班運用{{r|RN224B|page=54}}{{r|appledaily}}。

== 引進背景 ==
本電聯車引進時的1990年，儘管已[[電氣化|電氣化]]已久，台鐵仍是使用[[普快車|普快車]]作為通勤列車為主流，而[[名片式車票|名片式車票]]亦普遍使用。{{r|RN224B|page=50}}為配合[[台鐵捷運化|台鐵捷運化]]計畫，才引進本型列車{{r|RN224B|page=54}}，以取代原行駛於[[基隆車站|基隆]]至[[新竹車站|新竹車站]]之柴油快車{{r|大事記}}，並訂定車種為[[台鐵通勤電聯車|通勤電車]]{{r|RN224A|page=65}}。2006年11月配合改點，納入新車種[[台鐵區間車|區間車]]{{r|RN224A|page=57}}{{r|鐵道百科|page=103}}，採[[復興號|復興號]]票價計費{{r|票價}}。

=== 概要 ===
本型列車以每4輛為1編組，分別為EM車（駕駛馬達車）、EP車（電力車）、ET車（拖車）和EMC車（駕駛馬達車）{{r|RN224A|page=59}}，最多可4編組共16輛連結運轉{{r|RN224A|page=58}}，其中[[牽引馬達|牽引馬達]]裝於EM車及EMC車，而[[集電弓|集電弓]]位於EP車，ET車則另有[[發電機|發電機]]{{r|RN224A|page=57}}。另本型列車中加入ET車，使以往[[EMU200|EMU200]]、[[EMU300|EMU300]]三輛一組的編組方式改為四輛一組，並延用同設計至[[EMU500|EMU500]]、[[EMU600|EMU600]]等後續車種{{r|鐵道百科|page=86}}{{r|圖鑑}}。

=== 外觀 ===
本型列車初期外觀採南非國家標準SANS 1091之[[群青色|群青色]] (F09, Ultramarine, NCS色碼3355-R80B) 搭配雲白色 (G80, Cloud white, NCS色碼0704-G38Y) 線條做塗裝，台鐵局徽、車號等亦使用群青色{{r|UCW}}。每節車門配置原設計為每側3門，但最終設計採用每側僅設2個車門{{r|鐵道百科|page=103}}，並且配置於靠近車廂中央，而連控自動門、終點顯示器為黑底白字等設計均為台鐵首見。{{r|RN224A|page=60}}在列車進行多次檢修後，逐漸將群青色塗裝改為今日之深藍色，並將終點顯示器改為紅色[[LED|LED]]{{r|RN224A|page=64}}。2010年因配合月台高度提高工程，本型列車亦配合改造車門階梯，以符合一階化之目標，這次改造也是本型列車最後一次之改造{{r|RN224A|page=65}}。

=== 內裝 ===
本型列車自1990年引進至今，內裝並無太大變化，與其餘通勤電聯車相同，均使用深綠色塑膠布長條椅，靠近車門處設置立柱，天花板上亦有拉環，以容納大量乘客{{r|RN224A|page=63}}。

== 運用 ==
EMU400型電聯車投入營運後，隨即營運於[[基隆車站|基隆]]至[[新竹車站|新竹]]間，並以加班車形式支援[[臺中線|山線]]，直到1995年EMU500型電聯車的啟用才逐漸減少。[[EMU700型電聯車|EMU700型電聯車]]啟用之後，EMU400型除了停用待修編組外，多數用於基隆至斗六區間，另外也安排成追線的運用；也因配屬新竹機務段之故，當[[台鐵EMU600型電聯車|EMU600型電聯車]]故障或保養時，EMU400型偶爾會單組行駛六家線。2013年[[富岡基地|富岡基地]]啟用，EMU400型部分閒置編組用於行駛為員工加開的通勤列車。{{r|RN224A|page=65}}直到2014年，[[花東鐵路電氣化|花東鐵路電氣化工程]]需要列車進行試車{{r|RN218A|page=9}}{{r|RN218B|page=26}}，EMU400型因此成為[[台東線|台東線]]通車以來第一輛進入之電聯車種{{r|Itn}}。

== 現況 ==
*EMU403、406、409、411、412因車況不佳停用，目前放置於[[七堵調車場|七堵調車場]]。
*EMU401、402、404、405、407、408、410因車況不佳停用，目前放置於[[臺鐵富岡車輛基地|富岡基地]]，其中405和410尚可動態運轉。

== 軼事 ==
2015年10月，在EMU400型電聯車退出營運之後，於台鐵[[七堵調車場|七堵調車場]]當中停放的部分該型號電聯車被發現遭到塗鴉，由於調車場地形開闊且缺少監控設施，使得塗鴉者的身份無從查起，對此台鐵相關負責人表示該案已經報警處理{{r|eleven}}{{r|ettoday}}。
