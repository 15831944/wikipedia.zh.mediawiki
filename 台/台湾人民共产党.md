'''台湾人民共产党'''(英文:'''T'''aiwan '''P'''eople's '''C'''ommunist '''P'''arty,簡稱'''TPCP''')，[[台灣政黨|台灣政黨]]，于2016年成立，由林德旺担任首任党总理（相當於[[總書記|總書記]]），为[[台灣|台灣]]第6个以“[[共产|共产]]”为名的[[台灣政黨列表|政黨]]。

==概論==
台湾人民共产党發起人林德旺，是一名[[台商|台商]]，在[[中國大陸|中國大陸]]經商。林德旺原是[[中國國民黨|中國國民黨]]中央委員，於[[2016年中华民国立法委员选举|2016年立法委員選舉]]中未獲中國國民黨提名參選[[立法委員|立法委員]]，決定[[退黨|退黨]]參選[[臺南市第一選舉區_(2008年立法委員)|台南市第一選舉區]]立委，但並未當選。因為對於中國國民黨與[[民主進步黨|民主進步黨]]失望，「國民黨不爭氣，民進黨搞均貧」，林德旺決定發起新政黨。林德旺原以「台灣中國共產黨」為名向[[中華民國內政部|中華民國內政部]]申請政黨登記，但因內政部發[[公文|公文]]提醒黨名應依《[[兩岸人民關係條例|兩岸人民關係條例]]》等相關法令審慎衡酌，所以改為「台灣人民共產黨」。2017年2月4日，台灣人民共產黨在[[台南市|台南市]][[新营区|新营区]]小園餐廳舉行成立大會，[[黨魁|黨魁]]稱為「[[總理|總理]]」，林德旺為首任總理<ref>{{cite news|author1=吳政修|title=台灣人民共產黨台南成立 台商林德旺任總理|url=https://udn.com/news/story/1/2264627|accessdate=2017-02-04|agency=聯合報|date=2017-02-04}}</ref>。

台湾人民共产党成立的宗旨为“主张现代式[[社会主义|社会主义]]发展经济，坚持[[九二共识|九二共识]]，推动两岸和平”。<ref>{{Cite web |url = http://news.sina.com.cn/c/2017-02-04/doc-ifyaexzn8894988.shtml |title = “台湾人民共产党”成立 称坚持“九二共识” |author = |date = 2017年2月4日 |publisher = [[环球网|环球网]] |language = zh-hans |accessdate = 2017年2月4日 |quote = }}</ref> 台湾人民共产党成立時，目前有意加入的党员有1000人，林德旺计划一年内把党员发展到3万人；只要认同党的理念的台湾民众，包括中國大陸的台商，都可加入。林德旺说，各地的主要干部正在陆续挑选中，他们也会赴台湾各地举行党务发展说明会，未来也计划赴中國大陸进行政党交流<ref>[http://news.163.com/17/0204/18/CCETQKJK000187VE.html 林德旺：千人欲入“台湾人民共产党”，计划1年发展到3万人]</ref>。

2017年6月9日，台湾人民共产党副总理[[郭建志|郭建志]]说，“国民党[[亲中|亲中]]是卖台，民进党亲中是爱台”这个言论是背道而驰，所以他感觉中國國民黨與民主進步黨都是烂党，他才会投入台湾人民共产党<ref>{{cite news|url=https://www.rfa.org/mandarin/yataibaodao/gangtai/hcm-06092017103830.html|title=党旗宛如日本军旗 台湾人民共产党修正|publisher=[[自由亚洲电台|自由亚洲电台]]|author=黄春梅|date=2017-06-09|accessdate=2018-07-15|quote=}}</ref>。

2017年8月18日，林德旺說，台灣人民共產黨成立半年來，黨員約有三、四千人；以前的[[中國共產黨|中國共產黨]]是政治文化鬥爭多、拚經濟少，現在的中國共產黨已經大翻轉、全力拚經濟，現在的台灣卻在狠拚政治鬥爭、不拚經濟、也不思人民福祉；唯有[[台海兩岸|台海兩岸]]和平，才可以用心發展經濟<ref>{{cite news|url=http://hk.crntt.com/doc/1047/8/3/3/104783366.html|title=台灣人民共產黨總理林德旺：兩岸應和平統一|publisher=中評社|author=趙家麟|date=2017-08-18|accessdate=2018-07-15|quote=}}</ref>。 

2018年7月13日，[[中華民國總統|中華民國總統]]兼[[民主進步黨主席|民主進步黨主席]][[蔡英文|蔡英文]]到[[台南市|台南市]][[麻豆代天府|麻豆代天府]]、[[佳里金唐殿|佳里金唐殿]]參香揭匾，台灣人民共產黨由林德旺率領十餘人揮[[五星紅旗|五星紅旗]]如影隨形抗議，林德旺強調揮五星紅旗是受《[[中華民國憲法|中華民國憲法]]》[[言論自由|言論自由]]之保障。林德旺說，[[台灣價值|台灣價值]]是民主、自由、讓台灣人民過著好日子，而不是假民主；呼籲蔡英文，應該兌現她自己的承諾“兩岸維持現狀”，更應該承認九二共識；一旦台海兩岸發生戰爭，台灣有權有勢的人早就逃往國外，最後受害的是台灣人民，所以[[蔡英文政府|蔡英文政府]]不要想聯合其他國家抵制中國大陸<ref>{{cite news|url=http://hk.crntt.com/doc/1051/3/1/0/105131087.html|title=中評鏡頭：台共林德旺揮五星旗嗆蔡英文|publisher=中評社|author=趙家麟|date=2018-07-13|accessdate=2018-07-14|quote=}}</ref>。
