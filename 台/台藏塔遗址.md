== 地理信息 ==
台藏塔遗址位于[[吐鲁番盆地|吐鲁番盆地]]北缘与[[火焰山|火焰山]]南麓戈壁滩接壤的冲积平原地带, 周围地势平展。行政区划属于新疆维吾尔自治区吐鲁番市三堡乡尤喀买里村。西距[[吐鲁番市|吐鲁番市]]约40公里,东南距[[高昌故城|高昌故城]]遗址1.2公里<ref name="研究">{{cite journal |author1=梁涛 |title=吐鲁番台藏塔遗址保护研究 |journal=西部考古 |date=2008 |issue=00 |pages=297-307 |accessdate=2019-04-13}}</ref>。

== 主要遗迹 ==
=== 台藏塔 ===
台藏塔占地面积939平方米, 佛塔残高19.1米, 黄土夯筑，塔身主体略呈回字形，朝向南偏西。塔基底部外边南北长约36米，东西残长约34米, 塔壁基部厚8至12米。塔内中空，内边约为15米。南北正中有宽约3米的缺口<ref name="病害">{{cite journal |author1=张海锋 |author2=刘俊卿 |title=台藏塔结构病害分析与维修加固措施研究 |journal=施工技术 |date=2009 |volume=38 |issue=S2 |pages=502-504 |accessdate=2019-04-13}}</ref>。

=== 佛龛 ===
[[佛龛|佛龛]]现存32个。多已坍塌, 形制不清。仅东南壁第三级最北一座佛龛保存较好，现存基座、壁画等遗迹 <ref name="简报" />。

== 历史 ==
史籍中未见有关于台藏塔建筑的明确记载。据推测塔的建造与[[高昌|高昌]]统治者有关联。[[麹氏高昌|麹氏高昌]]之后的唐[[西州|西州]]与回鹊高昌时期，[[佛教|佛教]]一直是高昌地区流行的主要宗教。14世纪末，[[察合台汗国|察合台汗国]]统治者强迫高昌居民信奉[[伊斯兰教|伊斯兰教]]之后，台藏塔逐渐毁败<ref name="研究" />。

== 台藏塔结构病害 ==
目前台藏塔遗址的主要结构病害包括人为孔洞及原有孔洞残损 、裂缝 、掏蚀凹陷 、土体悬空倾斜等<ref name="病害" />。病害成因可能由于自然及人为原因。自然原因包括风蚀、雨侵、水融、温差及鼠穴。人为原因包括台藏塔遗址北侧底部壁面有拆除搭建后存留的构件卯孔及掏挖痕迹、当地居 民 台藏塔遗址塔身西侧及塔内空地上圈养羊群，对塔基夯土造成损害与威胁、台藏塔遗址塔顶经常有鸟类停落，造成一定程度的污损与腐蚀以及周围居民生活炊烟、污水和垃圾对台藏塔遗址造成污染<ref name="研究" />。
== 修复 ==
2010年12月14日，新疆吐鲁番台藏塔修缮工程已通过竣工验收，维修加固之前，台藏塔遗址的可靠度只有51.7%。可靠度较低，说明台藏塔存在极大的结构安全隐患，亟待维修加固。经过维修加固遗址可靠度已提升到95.4%<ref>{{cite web |author1=中国新闻网 |title=中国最高土遗址吐鲁番台藏塔修缮完工 |url=http://www.chinanews.com/cul/2010/12-14/2721986.shtml |website=中国新闻网 |accessdate=2019-04-13}}</ref>。
