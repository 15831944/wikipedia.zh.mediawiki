'''台湾颈槽蛇'''（[[学名|学名]]：{{lang|la|''Rhabdophis swinhonis''}}）为[[游蛇科|游蛇科]][[颈槽蛇属|颈槽蛇属]]的[[爬行动物|爬行动物]]，又名'''-{zh-cn:斯文豪氏游蛇;zh-tw:臺灣頸槽蛇;}-'''，是[[台灣|台灣]]的特有物种。分布于[[台灣島|台灣本島]]，多栖息于中低海拔山区以及多见于森林底层、灌木草丛、溪流附近或山区小道旁。该物种的模式产地在台湾。<ref name='id4711'/>
== 特徵 ==
台湾颈槽蛇為小型蛇類，體色呈褐色或棕褐色，頸部的「V」字形黑色斑紋為辨認其最好的方法。
== 習性 ==
無毒，性情溫馴，日行性，卵生。以蛙類為主食。
== 保育 ==
台湾颈槽蛇目前被台灣列入第三級「其他應予保育類」野生動物。
