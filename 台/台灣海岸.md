== 台灣北部岬灣海岸 ==
北部海岸西起[[淡水河|淡水河]]口，東至[[三貂角|三貂角]]，長約80餘公里。

*[[野柳|野柳]]

== 台灣西部砂泥海岸 ==
北起淡水河口，南至[[屏東縣|屏東縣]][[枋山鄉|枋山鄉]]楓港，長約400餘公里。海岸線平直，[[沙灘|沙灘]]綿延。

*[[外傘頂洲|外傘頂洲]]

== 恆春半島珊瑚礁海岸 ==
東起[[屏東縣|屏東縣]][[滿州鄉|滿州鄉]]九棚，西至楓港，長約90公里。

*[[墾丁|墾丁]]

== 台灣東部斷層海岸 ==
北起[[三貂角|三貂角]]，南至九棚，長約380公里。海岸線平直，斷崖緊鄰深海，地形陡峭。

*[[清水斷崖|清水斷崖]]
