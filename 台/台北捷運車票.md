'''台北捷運車票'''指[[台北捷運公司|台北捷運公司]]發行用於[[台北捷運|台北捷運]]上的[[車票|車票]]，包括單程票、電子票證等，由於車費結構複雜，因此設有很多車票種類。捷運公司亦發行[[旅遊|旅遊]]車票。所有發行的車票均受「大眾捷運法」<ref>[http://law.moj.gov.tw/LawClass/LawAll.aspx?PCode=K0120001 大眾捷運法]</ref>及「臺北捷運系統旅客須知」<ref>[http://web.metro.taipei/QRCode/Regulations%20for%20Use%20of%20the%20Taipei%20Metro%20System-Chinese.pdf 臺北捷運系統旅客須知]</ref>所規範；而[[悠遊卡|悠遊卡]]、旅遊票及以悠遊卡儲存的金額亦受「悠遊卡約定條款」<ref>[https://www.easycard.com.tw/terms.pdf 悠遊卡約定條款] {{Wayback|url=https://www.easycard.com.tw/terms.pdf |date=20171005201857 }}</ref>所規範。

==定義==
根據臺北捷運系統旅客須知，「車票」指「供旅客搭乘本公司經營之大眾捷運系統之乘車憑證」。<ref>[http://www.metro.taipei/public/Attachment/7517945496.pdf 臺北捷運系統旅客須知] {{webarchive|url=https://web.archive.org/web/20171005151955/http://www.metro.taipei/public/Attachment/7517945496.pdf |date=2017-10-05 }} - [[臺北捷運公司|臺北捷運公司]]</ref>

==車票==
臺北捷運車票有捷運公司所發行的IC代幣單程票、自行車票、團體票、自行車單程票，並可使用[[一卡通票證公司|一卡通公司]]所發行的[[一卡通_(台灣)|一卡通]]及[[悠遊卡公司|悠遊卡公司]]所發行的[[悠遊卡|悠遊卡]]及[[遠鑫電子票證|遠鑫電子票證]]股份有限公司所發行的[[HappyCash|有錢卡]]及[[愛金卡公司|愛金卡公司]]所發行的[[愛金卡|愛金卡]]。

票卡使用方式是將票卡或代幣輕觸感應區<ref name="車票">{{cite web |language=zh-tw |publisher=臺北大眾捷運股份有限公司 |title=捷運系統使用車票 |url=http://www.metro.taipei/ct.asp?xItem=78475924&CtNode=70093&mp=122035 |author= |date=2016-12-01 |accessdate=2010-12-01}}</ref>。

若站和站之間有多種乘車路徑，不論經由何路徑，其票價皆以最短乘車路徑計費。IC代幣單程票可供旅客搭乘一次，出站回收，僅限發售當日有效。<ref name="車票"/>。

昔日以全票收費，但自2011年4月1日起，以縣市政府補助差額的方式，提供敬老與愛心（身障）票，年滿65歲以上老人憑身分證、身心障礙者憑有效期的身心障礙證明（手冊）至捷運站詢問處購票，即享有4折優惠。另持有各縣市與一卡通或悠遊卡發行的記名式社福優待卡直接刷票乘車，也享4折優惠。

== 票價 ==
票價以乘車里程級距，採最短乘車里程計算，計價標準見下表<ref>[http://www.metro.taipei/ct.asp?xItem=78479325&CtNode=70091&mp=122035 票價及乘車時間]</ref>'''。台北捷運自動收費設備系統'''，為[[台北捷運|台北捷運]]所使用的自動收費設備系統（AFCS）<ref name="AFCS">{{cite web|url=http://www.metro-consultant.com/Symposium.pdf|title=與時俱進的台北捷運自動收費系統|accessdate=|author=|date=|publisher=|language=zh-tw|archiveurl=https://web.archive.org/web/20140115003111/http://www.metro-consultant.com/Symposium.pdf|archivedate=2014-01-15|deadurl=yes}}</ref>。目前平常除團體票和腳踏車搭乘券仍保留紙票以外，其餘均以非接觸式電子票證為主。票證使用非接觸式票卡為主，除了[[IC代幣|IC代幣]]單程票之外，也包括各式[[一卡通_(台灣)|一卡通]]及[[悠遊卡|悠遊卡]]及[[HappyCash|有錢卡]]（HappyCash）以及[[Icash|愛金卡]]（iCash）。目前無障礙寬閘門附掛式多卡通驗票閘門已經申請經費裝設並驗收完成，可使用扣款，在啟用前，曾因為領取交通部補助，完成多卡通建置卻多年不啟用而引發爭議<ref>蔡偉祺, 吳亮儀. [http://www.libertytimes.com.tw/2013/new/jun/5/today-taipei5.htm 多卡通不通 北捷拿補助 虛置9月不開放] {{webarchive|url=https://web.archive.org/web/20130625194911/http://www.libertytimes.com.tw/2013/new/jun/5/today-taipei5.htm|date=2013-06-25}}. 自由時報. 2013-06-05.</ref>。團體票和腳踏車搭乘券則仍為紙票，並使用公務門而非門檔式閘門。人潮眾多時也會販售單程票代用券，此亦為紙票。單程票自動售票機採用觸控式螢幕操作，一次最多可販售10枚IC代幣；可使用紙鈔購票<ref name="AFCS" />。

==目前票務設備==
[[File:July_2016_gate_10_inbound_ticket_barrier_available_iPASS_in_MRT_Nangang_Station.jpg|thumb]]
;[[驗票閘門|驗票閘門]]
三種不同的寬度供一般旅客、持大型行李的旅客及身障旅客使用。主動管制一人一票，以及偵測符合免票證的隨行孩童身高（115公分以下）。
;票證
*使用非接觸式票卡為主，包括各式[[一卡通_(台灣)|一卡通]]及[[悠遊卡|悠遊卡]]及[[愛金卡|愛金卡]]及[[有錢卡|有錢卡]]及[[IC代幣|IC代幣]]單程票。
*2015年9月1日起，無障礙寬閘門除各式[[一卡通_(台灣)|一卡通]]及[[悠遊卡|悠遊卡]]及[[愛金卡|愛金卡]]及[[有錢卡|有錢卡]]及[[IC代幣|IC代幣]]單程票也接受使用。
*團體票及腳踏車搭乘券則仍為紙票，並使用公務門而非門檔式閘門。
*人潮眾多時亦會販售單程票代用券，此亦為紙票。
;單程票自動售票機
採用[[觸控式螢幕|觸控式螢幕]]操作，一次最多可販售十枚IC代幣，可使用[[紙鈔|紙鈔]]及硬幣購票，部份機器亦可加值電子票證。
;悠遊卡加值機
為[[悠遊卡公司|悠遊卡股份有限公司]]提供，可使用現金或金融卡加值，亦可開啟聯名信用卡的自動加值功能。
;悠遊卡售卡加值機
為銀行提供，可使用現金購買悠遊卡或加值。
;悠遊卡單鈔加值機
為[[悠遊卡公司|悠遊卡股份有限公司]]提供，僅部份車站有此設備。其特色是一次僅使用一張紙鈔進行快速加值。
;票卡查詢機
可查詢電子票證或IC代幣卡片資料。
;兌幣機
為銀行提供，僅可以鈔票兌換50元硬幣。
;兌幣兌鈔機
為銀行提供，除可使用面額較大之鈔票兌換面額較小之鈔票及50元硬幣之外，亦可使用50元硬幣換取紙鈔。
;自動提款機
為銀行提供，供旅客提領現金使用。
;數幣機及點鈔機
此為提供站務人員使用之機具。
