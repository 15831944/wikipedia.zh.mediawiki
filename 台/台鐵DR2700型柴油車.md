'''DR2700型柴油車'''是[[臺灣鐵路管理局|臺灣鐵路管理局]]向[[日本|日本]][[東急車輛|東急車輛]]採購之31輛[[不鏽鋼|不鏽鋼]]車體[[柴油車|柴油車]]。其中動力車編號自DR2700開始，共25輛；無動力拖車編號自DR2750開始，共6輛。後因機電老舊，為減少動力車的負擔，全以動力車編組，不再使用拖車，故DR2750型拖車已經全數報廢。在[[花東鐵路電氣化|花東鐵路電氣化]]通車後，本型車已於2014年7月16日起不再開行固定班次[[普快車|普快車]]<ref>{{cite news|url=http://www.cna.com.tw/news/ahel/201405130049-1.aspx|title=由EMU500型電聯車取代行駛區間車}} （資訊已過期）</ref><ref>{{cite news|url=http://163.29.3.98/FullTimeTable/20140716/%E5%8F%B0%E6%9D%B1%E7%B7%9A.xls|title=定期行駛列車時刻表 台東線|publisher=臺灣鐵路管理局|date=2014-07-16|accessdate=2014-06-20|deadurl=yes|archiveurl=https://web.archive.org/web/20140628233515/http://163.29.3.98/FullTimeTable/20140716/%E5%8F%B0%E6%9D%B1%E7%B7%9A.xls|archivedate=2014-06-28}}</ref>，臺鐵挑出其中8輛車況較佳者動態保存，作為包車、郵輪式列車不定期行駛。

== 概述 ==
DR2700型於1966年開始，以[[光華號|光華號]]為名行駛於[[西部幹線|西部幹線]]，創下西部幹線最快列車的紀錄，時速高達110公里，其不鏽鋼車體與當時黑色的[[台鐵蒸汽機車|蒸汽火車]]形成強烈的對比，因而獲得「白鐵仔」的稱號。1979年西部鐵路電氣化完工後，[[自強號列車|自強號]]取代光華號行駛縱貫線，DR2700型改行駛[[對號快車|柴油對號特快車]]於[[北迴線|北迴線]]與花東線，以及西部幹線台中以北之區間。之後由於台灣鐵路管理局引進自強號柴聯車行駛於東部幹線，便轉為中短程[[對號快車|柴對快]]、[[普通車|柴快、柴普]]列車等，活躍於縱貫線北部區間、縱貫線中部區間、[[屏東線|屏東線]]、[[南迴線|南迴線]]、[[台東線|台東線]]、[[內灣線|內灣線]]等等。

早期作為光華號使用時，前端為銀色並掛有列車銘牌（headmark）。於1981年3月8日1002次自強號於[[頭前溪|頭前溪]]發生事故後，台鐵動力客車前端均塗上警示色（DR2100－2400型除外），故DR2700型前端塗上黃色。

車廂中有拱門，鑲以金屬飾板，內藏散熱管線與排氣管，與裝置於車頂突起部之散熱器，為本型車外觀上的主要特色（參見[[巴德柴油鐵路客車|巴德柴油鐵路客車]]）。

唯本型車車內噪音較大，依照1984年的研究，平均高達87至88分貝，為該研究中各車種車廂內噪音最大者。

到1980年代後，原裝出力為335HP的Cummins NHHRTO-6-B1型臥式柴油引擎日漸老化。為此本型車逐漸分批更新動力系統，將引擎換裝為Cummins NT855-R4型臥式柴油引擎，出力為310HP/2100rpm，與當時DR2800型原裝使用之引擎相同。首批車9輛並於1983年6月由台北機廠換裝完成並且於1988、1989年時分別再完成3輛，推測其餘未更新引擎的也在1990年逐步進廠後更新。而在2001－2002年DR2800型柴聯車由隆成發鐵工廠進行更新後所替換下的NT855-R4型引擎，逐步改換至DR2700型使用，由於管線沒有翻新緣故，限制發動機10%的出力以換取良好的運轉壽命。

== 特色 ==
DR2700型配有聯控線（可以多輛行駛而只要1位司機），不過DR2700型的聯控線規格屬於舊型，與DR2000型柴油客車同款，所以不能跟較新的DR2800至DR3100型柴聯車聯控行駛（但是可以有火迴送）。另外DR2700型也跟柴聯車一樣有動力車跟拖車，不過因為無空調設施，拖車並沒有很複雜的機電設備，只有供應電風扇的輪軸發電機（與[[普通車|普通車]]上的相同）。

DR2700型也是第一批在車體內部中間做成一道拱門以隱藏散熱管線的柴油客車，拱門上還鑲以金屬飾板裝飾，因而有「月洞門」之美譽，後續多款柴油客車與柴聯車，包括DR2510型、DR2800型、DR2900型、DR3000型，皆沿用DR2700型拱門的設計。

== 歷史 ==
*1966年10月22日：進行首次北高之間試車。
*1966年10月31日：DR2700型以光華號之名行駛西部幹線。
*1978年8月15日：電化完成前的改點紀錄：海線北高間2003次需時5小時08分，2002次5小時10分。
*1979年7月14日：開行西部幹線光華號最後一班列車，翌日起改開行對號快車行駛於台北蘇澳區間。
*1980年2月1日：開行台北-花蓮間直達柴油對號特快。
*1988年9月1日：臺鐵廢除對號快車費率，以本型車輛行駛之列車改以普快費率計價。
*1991年3月18日：開行后里-斗六間柴油快車。

*北迴鐵路完工以後的2001次直達柴油對號快車，於1980年2月1日北迴鐵路全線通車時開始營運，據[[聯合報|聯合報]]載，07:00自[[台北車站|台北站]]開出的第2001次柴油快車，中途不停任何站，於10:14分便可到達[[花蓮市|花蓮]]，行車時刻僅需3小時14分。後來在DR2800型以[[自強號列車|自強號]]之名開始行駛以後，DR2700型改以柴特快為名繼續行駛此路線，停靠較大車站（但此班車亦停靠[[石城車站|石城站]]）。台鐵取消對號快之後，此班柴特快仍以柴快與平快之名繼續行駛，行車時間與[[莒光號列車|莒光號]]相仿。

*因其為動力分散的柴油客車之故，DR2700型在雙溪站停車後開上牡丹坡，時速仍可達55-60公里，不遜於自強號柴聯車，且較柴電機車牽引列車為優秀。惟作為柴對快期間，曾於三貂嶺站發生列車起火事件。

*DR2700型並用來行駛[[基隆車站|基隆]]-[[中壢車站|中壢]]之間，以柴對快的名義行駛，一般編組為3到4節車廂。於1985年左右曾行駛[[屏東線|屏東線]]，主要是作為[[高雄車站|高雄]][[屏東車站|屏東]]間的柴快，還有一往復延駛至枋寮。而後改由每組3輛的DR2800型自強號柴聯車行駛；在1991年[[春節|春節]]等連續假日期間，原行駛屏東線之[[台鐵DR2800型柴聯車|DR2800型柴聯車]]支援西幹線，或迴送板橋調車場支援東幹線（當時東幹線列車基地；目前已被樹林調車場取代。南迴線當時尚未開放通車），DR2700型則代行其班車，以平快車票價計費。

*[[台鐵EMU400型電聯車|EMU400]]型引進後取代其北部區間的通勤列車，隨後改開行中部通勤區間，主要是在[[后里車站|后里]]-[[斗六車站|斗六]]之間作為柴快車行駛，後來延駛至[[斗南車站|斗南]]，並於[[海線|海線]]做為普通車行駛。並曾於1994年開始行駛於內灣線。

*在[[EMU500|EMU500]]型引進後，DR2700型於1996年開始轉往[[南迴線|南迴線]]及[[臺東線|臺東線]]繼續服役。南迴線柴快改由DR2510型運行時也時常代駛，直到該班次改由柴電機車牽引客車的普通車行駛為止；[[林口線|林口線]]試辦客運初期曾以DR2700型行駛，到DR2510型整修完成後才退出林口線。

*承上述，至[[花東鐵路電氣化|花東鐵路電氣化]]完工通車前，DR2700型以[[臺東線|臺東線]]為其舞台，行駛通勤用柴油普通車（現車種併入[[普快車|普快車]]）（4672次、4681次、4684次除外，為（SPK/TPK編成））<ref>{{cite news|url=http://163.29.3.98/FullTimeTable/20140210/%E5%8F%B0%E6%9D%B1%E7%B7%9A.xls|title=定期行駛列車時刻表 台東線|publisher=臺灣鐵路管理局|date=2014-02-10|accessdate=2014-05-13|deadurl=yes|archiveurl=https://web.archive.org/web/20140514021338/http://163.29.3.98/FullTimeTable/20140210/%E5%8F%B0%E6%9D%B1%E7%B7%9A.xls|archivedate=2014-05-14}}</ref>。偶爾會因包車行駛特別路線，例如2006年10月在台中線-海線-[[內灣線|內灣線]]的特別列車活動。

== 規格 ==
=== DR2700 ===
[[File:DR2705_at_Taichung.JPG|250px]]]]

*1966年日本東急車輛製
*馬力:（原裝）Cummins NHHRTO-6-B1 335HP/2100RPM
*馬力:（改裝後）Cummins NT855-R4 310HP/2100RPM
*變速機：Niigata DBSFG-100
*逆轉減速比：逆轉機 RG-18，減速比3.07
*軔機：SMEE 電磁空氣式以及手煞車
*散熱方式：車頂強制通風式散熱器
*燃料箱：500L
*轉向架：{{tsl|ja|国鉄DT21形台車#海外向け|TS-122}} 5.9t
*自重：34噸
*最高速度110km／hr。現因設備老舊影響引擎散熱之故，行駛最高時速一般為90km／hr左右。
*車廂換氣：通風器6個，迴轉式電扇8座
*座位：60
*車門：兩片式折疊門每側一個，位於無駕駛座之側；駕駛座另有一門。
*廁所：位於上下車門旁，面對前端之右側。
*乘務員室：位於上下車門旁，面對前端之左側。
*生產輛數：25輛

==現況==
===DR2700（動力車）===
* 原有25輛，2016年在籍8輛，已經退役。
* 2002年，40DR2702、40DR2704、40DR2708、40DR2713報廢除籍，2003年1月四輛全數解體<ref name="taipeistation">{{cite web |url= http://blog.xuite.net/vespachu/blog003/15829285|title= 消失的台北後站 沒意義的懷舊廣場|author=  vespachu (Vespa) |date= 2008年2月27日|website= 擺接堡人的遊記|accessdate=2014年8月8日}} </ref>。
* 2004年，40DR2719、40DR2712兩輛除籍，於2004年8月臺灣泰華實業（現為煜翔機械）改造為電力維修車CMB29、CMB30。
* 2007年，40DR2701報廢除籍，招標後於2009年由縱貫線鐵道保存協會收藏<ref>{{Cite web |url= http://www.railfan.org.tw/ |title= 縱貫線鐵道保存協會 |accessdate=2017-03-15 |date= 2017-03-15 }}</ref>。
* 2010年，40DR2710報廢除籍，招標後為私人收藏。
* 2012年，40DR2705、40DR2709報廢除籍，招標後為私人收藏。
* 2016年，40DR2711、40DR2715、40DR2716、40DR2717、40DR2720、40DR2721、40DR2724報廢除籍，除40DR2724放置於花蓮機務段臺東機務分段洗車線止衝擋前外，其他車輛皆放置於臺東站旁貨物線。
* 在籍車中，端面為銀色油漆塗裝車輛：40DR2703、40DR2706、40DR2722、40DR2725。
* 在籍車中，端面已刨除銀色（黃色）油漆，露出不銹鋼原色車輛： 40DR2707、40DR2714、40DR2718、40DR2723。

===DR2750（無動力拖車）===
*原有6輛，2012年在籍0輛。
*1997年，35DR2751－35DR2756全數報廢除籍，2003年1月35DR2751、35DR2753－35DR2756解體<ref name="taipeistation"/>。
*35DR2752報廢後在花蓮機務段台東機務分段留置線放置<ref>{{cite web|url=http://www.east-asia-railroad.com/starthp/subpage81.htm | script-title=ja:「台湾」台東機務分段にイソウロウ|publisher={{lang|ja|東アジア鉄道イソウロウ事務所}}|accessdate=2011-02-06 |language=ja}}</ref>，但局徽與車號牌有佚失。已於2018年1月9日清晨4:20分掛於特開的8709次迴送列，編組：R139+DR2752+R134迴送花蓮機廠整修。
