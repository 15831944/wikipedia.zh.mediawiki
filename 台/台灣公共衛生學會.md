'''台灣公共衛生學會'''（{{lang-en|Taiwan Public Health Association}}）是[[台灣|台灣]]的[[公共衛生學|公共衛生學]]學術組織，原名'''中華民國公共衛生學會'''，在1972年成立。

== 學會沿革 ==
1971年[[中華民國衛生福利部|衛生署]]設立後，次年臺灣首個公共衛生學系成立，當時[[臺灣大學|臺灣大學]]教授[[陳拱北_(醫學學者)|陳拱北]]因此倡議成立公共衛生學會，於1972年11月19日正式成立「中華民國公共衛生學會」，並推舉時任衛生署署長的[[顏春輝|顏春輝]]擔任第一屆學會理事長。1980年代初期開始，常年召開年會，每兩年（2007年起改為三年）重新選舉理監事。1993年時為方便加入國際學術團體和拓展國際空間，曾更名「中華公共衛生學會」，其後2000年10月7號，復正式改為「台灣公共衛生學會」沿用至今。<ref name=about/>

* 中華民國公共衛生學會（1972－1993）
* 中華公共衛生學會（1994－2000）
* 台灣公共衛生學會（2001－）<ref name=about>{{cite web|language=zh-tw|accessdate=2018年12月13日|publisher=臺灣公共衛生學會|title=學會簡介|url=http://www.publichealth.org.tw/about.asp?HidDID=2}}</ref>

== 宗旨 ==
該學會以研究公共衛生，促進其發展，並加強國際間公共衛生團體之交流為宗旨。其承擔的主要任務有：公共衛生相關領域的專題研究及建議、發行學會通訊、學術雜誌、舉辦年會暨學術研討會等、與國際公共衛生相關團體聯繫與合作、加強會員間的聯繫及相關活動和其他有關公共衛生事項。<ref name=about/><ref name=ngotw>{{cite web|language=zh-tw|accessdate=2018年12月13日|publisher=[[中華民國外交部|中華民國外交部]]NGO國際事務會|work=外交部NGO雙語網|title=Medical and Health|url=https://www.taiwanngo.tw/p/404-1000-401.php?Lang=en}}</ref>

== 會員 ==
須符合基本資格和需要兩名會員的介紹信等程序。共四類，分別有個人會員、名譽會員、贊助會員和永久會員。<ref name=about/>其中團體會員普遍是臺灣各地的衛生局、衛生所和醫療機構，個人會員為大專院校相關科系的師生、前述衛生、醫療組織和民間團體的實務工作者。<ref name=ngotw/>

== 學術刊物 ==
《台灣公共衛生雜誌》（Taiwan Journal of Public Health，簡稱Taiwan J Public Health）為學會目前發行之學術刊物，行雙向匿名審查，為「台灣社會科學引文索引」（TSSCI）期刊，也收入Soupus、ProQuest等國內外引文索引資料庫。<ref>{{cite web|language=zh-tw|accessdate=2018年12月13日|publisher=臺灣公共衛生學會|title=投稿需知|url=http://www.publichealth.org.tw/magazine.asp?HidDID=4}}</ref><ref>{{cite web|language=zh-tw|accessdate=2018年12月13日|publisher=臺灣公共衛生學會|title=主編的話|url=http://www.publichealth.org.tw/magazine.asp?HidDID=2}}</ref>

* 中華民國公共衛生學會雜誌（1982－1993）
: 為使公共衛生界間學術交流、聯絡情感之管道更加完備，創辦「中華民國公共衛生學會雜誌」，並於1982年12月15日發行第一刊。<ref name=annal>{{cite web|language=zh-tw|accessdate=2018年12月13日|publisher=臺灣公共衛生學會|title=公衛大事紀|url=http://www.publichealth.org.tw/about.asp?HidDID=14}}</ref>
* 中華公共衛生雜誌（1994－2000）
: 1994年，配合學會更名，學刊雜誌亦改名為《中華公共衛生雜誌》，並由季刊改為「雙月刊」發行。<ref name=annal/>
* 台灣公共衛生雜誌（2001－）
: 2001年，配合學會更名，學刊雜誌亦改名為《台灣公共衛生雜誌》。<ref name=annal/>

== 學術活動 ==
「台灣公共衛生學會年會」為學會最重要之學術活動，年會主題皆為社會重大公共衛生相關議題，自1986年起，至今已舉辦33屆。<ref>{{cite web|language=zh-tw|accessdate=2018年12月13日|publisher=臺灣公共衛生學會|title=歷年年會與大會手冊|url=http://www.publichealth.org.tw/about.asp?HidDID=12}}</ref>
