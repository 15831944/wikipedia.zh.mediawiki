'''臺鐵S300型柴電機車'''，是臺鐵第一款真正完全支線與調車專用的柴電機車。

== 歷史背景 ==
1963年，[[臺灣鐵路管理局|臺灣鐵路管理局]]邀請[[世界銀行|世界銀行]]運輸顧問團來台評估運輸需求，決定實施支線與調車全面柴油化。為實現此一目標，台鐵於1965年以世銀貸款招標採購21輛小型柴油電氣機車。經公開招標，由美國[[易安迪|GM-EMD]]的21輛{{Link-en|EMD GA8|EMD GA8}}型柴電機車得標。1966年9月，21輛GA8型運抵台灣，台鐵命名為S300型，編號S301至S321。

== 車輛概說 ==
S300型不同於S200型，最高速僅75公里，沒有裝置總控重連控制裝置，是一款完全以站場調車和支線用為主的機車。最大的特點是牽引馬達只有兩具，裝置在車架底下，以萬向接頭分別驅動兩組轉向架，如此設計使本型車更加輕量化。S300型採用[[易安迪567系列柴油机|8-567E引擎]]。曲軸箱和同期進口的R50型柴電機車一樣是裝置[[易安迪|GM-EMD]]當時最新款的E型。

== 現況 ==
S300型因原料停產，自新一代調車用[[台鐵DHL100型柴液機車|DHL100型]][[柴液機車|柴液機車]]啟用前，就已經大量廢車：
*1993年S309報廢
*1994年9月21日S303、S312、S314、S320報廢
*1997年3月28日S301、S308、S310、S315、S319報廢
*1999年S305報廢

2002年~2003年間，引進了DHL100型柴液機車後，再度大量廢車：
*2002年S302、S304、S306報廢
*2003年S311、S313、S317報廢
*2004年9月S307、S321報廢
*2011年S316報廢。
目前S300型僅存S318於[[彰化機務段|彰化機務段]]作為[[扇形車庫|扇形車庫]]調度用機車。其他報廢停用車輛除S302放置在[[泰安車站|泰安車站]]、S305修復後移至[[苗栗鐵道文物展示館|苗栗鐵道文物展示館]]與S316在[[車埕車站|車埕車站]]做靜態展示以外，均已經全數解體。
