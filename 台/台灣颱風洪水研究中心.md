'''台灣颱風洪水研究中心'''（英語：Taiwan Typhoon and Flood Research Institute，簡稱TTFRI、颱洪中心），全名'''財團法人國家實驗研究院台灣颱風洪水研究中心'''，簡稱'''颱洪中心'''，是[[國家實驗研究院|國家實驗研究院]]以下，研究[[颱風|颱風]]及[[洪水|洪水]]的研究中心，於2011年8月17日成立<ref>[https://www.ttfri.narl.org.tw/about01.html 國家實驗研究院台灣颱風洪水研究中心 重要記事]</ref>；9月16日，颱洪中心主任由台灣大學大氣科學系李清勝教授接任<ref>{{Cite web|url=http://www.narlabs.org.tw/tw/news/news.php?news_id=1115|title=台灣大學李清勝教授接任國研院颱洪中心主任|accessdate=|author=|date=2011-09-16|work=www.narlabs.org.tw|publisher=|archive-url=https://web.archive.org/web/20180410135406/http://www.narlabs.org.tw/tw/news/news.php?news_id=1115|archive-date=2018-04-10|dead-url=yes}}</ref>。2017年8月1日，由中央大學大氣科學系暨大氣物理研究所黃清勇教授接任<ref>{{Cite news|url=https://money.udn.com/money/story/5723/2619616|title=黃清勇接任國研院颱洪中心主任|author=|last=|date=2017-08-02|work=|newspaper=經濟日報}}</ref>。

== 目的 ==
颱洪中心定位為颱洪防減災關鍵技術研發中心，整合國家實驗研究院之跨領域研發能量，結合學研界與作業單位研發成果，深耕颱洪災害研究及開發前瞻觀測與模擬技術，支援災害防救單位的技術需求。有別於[[國家災害防救科技中心|國家災害防救科技中心]]、及[[交通部中央氣象局|中央氣象局]]、[[經濟部水利署|水利署]]、[[行政院農業委員會水土保持局|水保局]]...等作業單位，颱洪中心著重於發展前瞻防減災科技、從事跨領域與跨國合作、並提供產業界客製化服務(如：颱洪預警實驗應用平台)。所有研發成果均即時提供作業單位於防災應變時參考應用；同時透過參與行政院災害防救應用科技方案<ref>{{Cite web|url=http://astdr.colife.org.tw/|title=應科方案計畫成果溝通分享系統|author=|date=|work=astdr.colife.org.tw|publisher=}}</ref>，針對洪泛、乾旱、劇烈天氣及氣候變遷等課題，強化與國家災害防救科技中心及其他學研或作業單位之合作分工。

== 沿革 ==

* 2007年1月1日，颱洪中心籌備處於新竹科學園區成立。
* 2008年5月1日，籌備處遷至中部科學工業園區。
* 2010年8月1日，設立台北辦公室。
* 2015年8月19日，大氣水文研究資料庫正式啟用。
* 2018年12月31日，裁撤，少數研究人員併入災防中心。

== [[八二三水災|八二三水災]]後續反應 ==
8月24日，蔡英文總統強調：「氣象預報和預警是重中之重。」8月25日，天氣風險公司分析師吳聖宇表示：「明年度(2019)氣象學界的研究經費被砍掉，台灣颱風洪水研究中心也收掉了，讓人不得不懷疑政府真的對氣象預報預警是有心的嗎？」以致後續媒體開始注意颱洪中心、災防中心之整併議題。8月31日，氣象局[[辛在勤|辛在勤]]前局長於臉書說道<ref>{{Cite news|url=https://www.ettoday.net/news/20180831/1248831.htm|title=颱洪中心應改組整併　辛在勤：可惜大人沒人想管|author=|last=|date=|work=|newspaper=|accessdate=|language=}}</ref>：「多年來，我就主張颱風洪水中心（從成立就因定位問題，有些爭議，終究被拿來祭旗）一切兩半，颱風氣象歸會氣象局（目前還在交通部），洪水的到水利署（經濟部）或NCDR。」而颱洪中心黃清勇主任於29日透過自由時報媒體表達以下觀點<ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/2535382|title=鄭明典建議放環資部 颱洪中心：比去災防中心好|author=|date=2018-08-29|work=|accessdate=}}</ref>：

# 中心目前僅有一半的員工願意到災防中心工作。
# 颱洪中心應等環境資源部成立後，再予以整併至環資部，唯科技部長陳良基執意裁撤。
# 整併至災防中心，其業務性質不利於目前颱洪人才所能發揮之專業，會以支援公務需求為主。
# 災防中心無規劃承接颱洪中心大部分觀測設備，包含無人飛機、剖風儀雷達、試驗流域儀器...等大氣水文觀測設備，此儀器未來留於國研院或給予大專院校接收。
