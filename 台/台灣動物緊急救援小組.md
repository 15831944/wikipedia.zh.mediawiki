'''台灣動物緊急救援小組'''是一個位於[[中華民國|中華民國]]的[[動物保護|動物保護]]組織，於1995年由[[倪兆成|倪兆成]]創立，最初的會址設於[[高雄|高雄]]。該組織目前已併入「社團法人台灣動物緊急救援推廣協會」，在全台的志工約有上百人<ref name=":0">{{Cite news|url=http://www.tanews.org.tw/info/5095|title=動物緊急救援 父子偕志工出任務|author=|date=2014-06-04|work=|newspaper=台灣動物新聞網|accessdate=2017-10-29|language=zh-hant}}</ref>。據其官網說法，該組織二十多年來共拯救超過五千隻傷病[[流浪貓|流浪貓]]狗的生命<ref name=":1">{{Cite web|url=http://www.savedogs.org/about|title=台灣動物緊急救援小組 - - 關於我們|accessdate=2017-10-29|author=|date=|publisher=社團法人台灣動物緊急救援推廣協會|language=zh-tw}}</ref>。

== 緣起 ==
1995年，倪兆成住家旁的鄰居將飼養的1隻[[混種狗|混種狗]]「派蒂」趕出家門任其露宿街頭，倪兆成不捨，獨自騎[[摩托車|機車]]在自家附近搜尋，花兩週時間找到「派蒂」並將牠收養，從此開始關注並救援住家附近的[[流浪犬|流浪貓狗]]。<ref name=":0" />有鑑於當時台灣並無類似歐美專為動物成立的救援組織，乃創立「台灣動物緊急救援小組」，開始進行傷病貓狗的救援任務，當時為全台灣第一個仿效歐美專為傷病受虐動物提供緊急救援與醫療的民間組織。組織開始之初，只有少數幾名志工自發性參與流浪貓狗救援，但因設備不足，救援任務常常受阻，因此，該組織便開始向國外進口專業的[[動物救援器材|動物救援器材]]<ref name=":1" /><ref>{{Cite web|url=https://tw.mobi.yahoo.com/news/%E6%98%9F%E6%9C%9F%E4%BA%BA%E7%89%A9-%E5%80%AA%E4%BA%AC%E5%8F%B0-%E8%B2%93%E7%8B%97%E5%AE%88%E8%AD%B7%E7%A5%9E-220106779.html|title=星期人物－倪京台 貓狗守護神|accessdate=2017-10-29|author=林宏聰|date=|publisher=Yahoo奇摩行動版|language=zh-TW}}</ref>。

== 延革 ==
2004年6月，[[高雄|高雄]]兩隻流浪狗掉落[[愛河|愛河]]支流，不會游泳的倪兆成直接跳下水把狗兒救起，並施以口對口[[人工呼吸|人工呼吸]]，全程被媒體拍下，照片並刊上《[[聯合報|聯合報]]》頭版，台灣動物緊急救援小組因此受到台灣民眾關注<ref name=":0" />。

逐漸地，因為理念與行動受到大眾肯定，使越來越多的支持者陸續加入捐助行列，投入動物救援任務；沒想到，台灣動物緊急救援小組隨著組織的擴大，卻遭到高雄當地某動保團體的指控與批評，多次檢舉非法募款，讓志工團隊不堪其擾，乃在2007年向[[內政部|內政部]]申請立案，正式成為全國性公益[[非營利組織|非營利組織]]「社團法人台灣動物緊急救援推廣協會」，也是全台灣第一個經政府同意，准許對外合法勸募的動物救援組織。

近年來，該組織設立「[[台灣貓狗聯合勸募平台|台灣貓狗聯合勸募平台]]」，定期捐贈飼料給全台各地的多個弱勢狗園貓園；每月平均捐出約[[新台幣|新台幣]]160萬元的飼料。截至2017年總共捐出價值一億元的飼料<ref>{{Cite news|url=http://news.ltn.com.tw/news/local/paper/1153343|title=〈南部〉貓狗聯合勸募平台 8年捐億元飼料|author=陳文嬋|date=2017-11-20|work=自由時報電子報|accessdate=2018-10-21}}</ref>，共有80個弱勢私人流浪貓狗收容中心接受其飼料捐助<ref>{{Cite news|url=https://tw.appledaily.com/new/realtime/20171119/1244091/|title=8年捐上億飼料助浪浪　等同37座101|author=周昭平|date=2017-11-19|work=|newspaper=台灣蘋果日報|accessdate=2018-02-05|language=zh-TW}}</ref>。

除了救援、醫療、[[節育|節育]]、認養等面向之外，台灣動物緊急救援小組長期以來持續與各大[[電子媒體|電子]]與平面媒體合作，致力於[[生命教育|生命教育]]；近年來更是走進校園與軍隊中，舉辦各類尊重[[動物權利|動物權利]]等議題的講座；並利用[[網路社群|網路社群]]等平台宣導愛護貓狗的觀念，其官方網站更因此獲得行政院研考會年度優質網站首獎<ref name=":1" />。

2010年，知名[[愛狗人士|愛狗人士]][[賈鴻秋|賈鴻秋]]意外身亡後，台灣動物緊急救援小組主動發起「搶救賈鴻秋斷糧狗大作戰」<ref>{{Cite news|url=https://tw.appledaily.com/headline/daily/20100222/32313740/|title=愛狗女教師摔死 150犬斷糧|author=尹維源、陳韋劭、李俊淇|date=2010-02-22|newspaper=臺灣蘋果日報|accessdate=2018-10-21|language=zh-TW}}</ref>；2011年，另名愛狗人士[[洪秀惠|洪秀惠]]意外身亡後，台灣動物緊急救援小組的義工也主動發起搶救洪秀惠遺下的上百隻狗<ref>{{Cite news|url=http://news.ltn.com.tw/news/local/paper/478050|title=〈南部〉搶救洪媽媽狗場 急徵狗糧與志工|author=蘇福男|date=2011-03-21|work=自由時報電子報|accessdate=2018-10-21}}</ref><ref>{{Cite web|url=http://www.savedogs.org/forum/article_view.html?f_no=25127|title=關注: 內門洪秀惠狗場 救援小組接手25犬|accessdate=2018-10-21|publisher=台灣動物緊急救援小組|archiveurl=https://archive.is/TjrI|archivedate=2012-09-19}}</ref>。

== 相關爭議 ==
因該組織受到媒體關注程度極高，引發其他動物保護團體不滿，曾遭其他動保團體（官網稱是「高雄某動保團體」<ref name=":1" />）質疑台灣動物緊急救援小組未立案卻[[贊助|募款]]「斂財」，更檢舉他們非法持有[[麻醉槍|麻醉槍]]，使[[內政部警政署刑事警察局|刑事局]]介入鑑定，最後判定其救援工具均屬合法。就因種種原因，台灣動物緊急救援小組乃於2007年向內政部立案設立「社團法人台灣動物緊急救援推廣協會」，成為第一個合法申請[[公益勸募|公益勸募]]的動保團體<ref name=":0" />。
