'''臺灣犬'''，又稱'''臺灣土犬'''、'''福爾摩莎犬'''、'''國寶犬'''、'''高砂犬'''，臺灣犬是犬隻品種，屬於馴化動物，並非[[臺灣|臺灣]]原生[[犬科|犬科]]動物，最早起源自臺灣高海拔地區[[臺灣原住民|原住民]]飼養的[[獵犬|獵犬]]，是[[南亞狩獵犬|南亞狩獵犬]]的後代。牠們非常適應台灣崎嶇不平的地形以及茂密的叢林，現在大多作為[[獵犬|狩獵犬]]、護衛犬、特技犬、[[搜索及拯救犬|搜救犬]]、或者是陪伴犬<ref>{{cite web  |url=http://www.taipeitimes.com/News/feat/archives/2005/09/11/2003271280 |title=The Formosan dog: A breed apart |publisher=[[Taipei_Times|Taipei Times]] |author=Meredith Dodge |date=2005-11-05 |accessdate=2007-02-07}}</ref>。

== 特徵與分類 ==
臺灣犬最早於17世紀即有文獻記載。[[台灣清治時期|清代]][[乾隆|乾隆]]年間[[朱仕玠|朱仕玠]]《[[小琉球漫誌|小琉球漫誌]]》描述：「番犬大如黃犢，吠聲殊異。剪其雙耳，以草木蒙密，且多鉤刺，欲其馳騁無掛礙也……。捕獐鹿，發示追蹤，百不失一。」。根據[[世界犬业联盟|世界犬业联盟]]對「台灣犬」的介紹指出，台灣犬是台灣本土品種培育的中型犬種，屬於[[南亞狩獵犬|南亞狩獵犬]]的後裔，主要特徵包含細尖耳、杏仁眼、三角型狀頭部和鐮刀狀尾巴，目前經過官方血統認證的台灣犬有6909隻<ref>{{Cite web  |title = 我不叫「土」狗！國際認證正名「台灣犬」 |publisher = 自由時報 |url = http://news.ltn.com.tw/news/life/breakingnews/1387049 |date = 2015-07-22 |author= |accessdate = 2015-07-25 |language = zh-hant}}</ref>。毛色有黃色，黑色、虎斑、白色、黑白混合花色等多種。大小型態則分為中型和小型二種：中型狗體態標準，腰細、胸寬，成犬身高不過五十公分左右，胸圍四十幾公分；小腿筋骨細長，大腿肌肉緊實。小型犬有二種，一種身高平均四十公分，一在卅公分以下；前者體格沒有中型狗魁梧，但瘦長伶俐，是長跑健將。小型犬臉型較瘦削，顯得機靈，除土黃與黃褐色，還有純黑色；尾巴、耳朵均豎立，好似隨時警戒中；舌頭正面或背面有黑斑<ref>光華雜誌臺灣特稀有動物系列報導(9)臺灣犬 1986年五月出版 </ref>。

== 習性 ==
*智力高、領悟力強，家犬。
*保有相當程度野性，攻擊性較強，不易和陌生人親近，但對主人極為忠心。雖然如此，但飼主可以培育溫馴個性，就像日本[[柴犬|柴犬]]原獵犬培育而來。
*勇敢善獵，面對大型獵物亦不畏懼。
*體力佳。
*地域性強，外人進入自動防衛。
*適應環境能力佳，獨立性良好。
*生活習性佳，服從性高。
*極具靈性。

== 飼育簡史 ==
在[[台灣清治時期|台灣清治時期]]，因由台灣原住民飼養，而被稱為'''番犬'''；{{Fact|後來在[[台灣日治時期|台灣日治時期]]的資料中，由於台灣原住民被稱為「高砂族」，也稱之為'''高砂犬'''。現今[[臺灣戰後時期|臺灣戰後時期]]則改叫為'''台灣土犬'''，以及'''打獵犬'''或簡稱'''土犬'''（Pariah Dog）|time=2015-08-23T07:54:42+00:00}}。然而，因流浪進口犬種氾濫街頭及未嚴格實施寵物[[節育|節育]]等因素，臺灣犬早已與外來犬種大量雜交，目前不易找到純度高的犬隻。1978至1980年間，[[日本|日本]][[名古屋大學|名古屋大學]][[太田克明|太田克明]]教授率領的研究團隊，與[[臺灣大學|臺灣大學]]畜牧系畜產研究室宋永義教授合作調查臺灣犬。該團隊發現西部平地已無法找到純種台灣犬，遂轉往山區調查，以[[中央山脈|中央山脈]]二十處原住民村落為調查點。在找到的160頭臺灣犬隻中，有46頭判定為一級純度。此項估計達到[[聯合國糧食及農業組織|聯合國糧食及農業組織]]（[[FAO|FAO]]）世界生物保種監督警告系統所公布的瀕臨絕種動物標準<ref>林乙華，〈土狗傳奇〉《經典雜誌》第70期，2004年5月出刊，頁78。</ref>。而在1990年代，臺灣社會飼養臺灣犬漸成風潮，臺灣犬身價躍升，加上本地狩獵文化式微，對犬隻不再重視，使得山地部落及鄉間的臺灣犬被平地寵物商家大量購往都會區，供給一般大眾飼養<ref>林乙華，〈土狗傳奇〉《經典雜誌》第70期，2004年5月出刊，頁78。</ref>。

在2015年[[世界犬业联盟|世界犬业联盟]]於[[義大利|義大利]][[米蘭|米蘭]]召開全球會員國大會上，台灣土狗通過70多個會員國代表投票，正式成為代表台灣的全球新興犬種之一，也正名為「台灣犬」（Taiwan Dog）。根據FCI對「台灣犬」的介紹指出，台灣犬是台灣馴化的犬種，屬於南亞狩獵犬的後裔，被過去中部山區居民稱為「賤民狗」（Pariah dog）。另外，也指出，此品種相當忠誠，過去是獵人在森林中的同伴，此品種在現代台灣則被飼養成守衛犬或陪伴犬<ref>{{Cite web  |title = 大膽無畏、忠心台灣土狗 獲國際認證正名「台灣犬」 |publisher = 東森新聞 |url = http://www.ettoday.net/news/20150721/538426.htm |date = 2015-07-21 |author= |accessdate = 2015-07-25 |language = zh-hant}}</ref>。
