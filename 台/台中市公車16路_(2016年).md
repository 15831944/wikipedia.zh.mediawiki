'''台中市公車16路'''停駛前行駛自道禾六藝文化館到三貴城，途中主要行使於[[東山路|東山路]]。

== 歷史 ==
*2013年2月18日：部份班次繞駛[[東山高中|東山高中]]，並延駛至民德僑。<ref>[http://www.traffic.taichung.gov.tw/form/index.asp?Parser=2,7,166,52 交通局市區公車預估到站時刻表]</ref><ref>[http://60.249.10.173/newsfile/shownews.php?newsno=32 台中客運 最新消息 路線班次調整]</ref>
*2013年12月1日：調整2班次行駛至東山樂园。<ref>[http://60.249.10.173/newsfile/shownews.php?newsno=102 12/1起16路及26路調整發車時刻]</ref>
*2015年3月1日：「第一信用」撤站並併入「彰化銀行（臺灣大道）」。<ref>{{Cite web |url=http://www.tcbus.com.tw/newsfile/shownews.php?newsno=275 |title=3/1站位新增、取消及更名公告 |accessdate=2016-03-02 |date=2015-02-26 }}</ref>
*2016年3月1日：「天星大飯店」站更名為「三貴城」，路線名稱變更為（綠川東站－大坑(三貴城)）及調整發車時刻。<ref>{{Cite web |url=http://www.tcbus.com.tw/newsfile/shownews.php?newsno=393 |title=3月份站位新增及更名公告 |accessdate=2016-03-02 |date=2016-02-18 }}</ref><ref>{{Cite web |url=http://www.tcbus.com.tw/newsfile/shownews.php?newsno=387 |title=3/1起16路調整發車時刻 |accessdate=2016-03-02 |date=2016-02-16 }}</ref>
*2016年7月1日：由綠川東站端延駛至道禾六藝文化館，延駛後綠川東站裁撤，途中增停「光明國中（貴和街）」（單邊設站，往道禾六藝文化館方向）、「地方法院」、「臺中女中」、「民權繼光街口」、「臺中火車站」<ref>[http://www.tcbus.com.tw/newsfile/shownews.php?newsno=423 7/1起8路、15路及16路延「道禾六藝文化館」]，台中客運，105年06月17日</ref>
*2016年8月1日：與仁友客運21路整併後停駛。<ref>[http://www.tcbus.com.tw/newsfile/shownews.php?newsno=433 16路8/1起將由仁友客運接駛]</ref>

== 路線基本資料 ==
全程行車里數約11.4(延駛15)公里，全程約50(延駛60)分。往三貴城37(延駛46)站，往綠川東站38(延駛47)站。
綠川東站往大坑，沿途行經[[臺灣大道|臺灣大道]]一段、興中街、[[公園路_(台中市區)|公園路]]、[[三民路_(台中市區)|三民路]]、[[北屯路|北屯路]]、[[東山路|東山路]]。<ref>[http://60.249.10.173/image/16.gif 台中客運16路路線圖]</ref>
大坑往綠川東站，沿途行經[[東山路|東山路]]、[[北屯路|北屯路]]、[[三民路_(台中市區)|三民路]]、[[中山路_(台中市區)|中山路]]、[[自由路_(台中市區)|自由路]]、[[臺灣大道|臺灣大道]]一段
