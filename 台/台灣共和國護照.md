{{otheruse|subject=部分台灣獨立運動支持者自行印製的「護照」|other=中華民國政府官方所發行的正式護照|中華民國護照}}

'''台灣共和國護照'''是一些支持[[台灣獨立運動|台灣獨立運動]]的人士在2001年3－5月間所印製的「[[護照|護照]]」，並於2001年5月17日召開記者會，聲稱已經獲得[[巴西|巴西]]等國簽證<ref>[http://taiwannation.50webs.com/taiwan15.htm 「台灣僑民持台灣共和國護照入境巴西」]，台灣國網站</ref>，但其真實性被人懷疑，認為這是一本「[[模拟护照|模拟护照]]」{{NoteTag|「[[模拟护照|模拟护照]]」可指「camouflage passport」或「fantasy passport」，並沒有確實譯名，詳見相關條目。}}。[[中華民國政府|中華民國政府]]認為這是一本「虛擬護照」或「[[玩具|玩具]]」，而且已經向各國聲明此護照並不被承認，也不具實際效力<ref>[http://www.mofa.gov.tw/webapp/ct.asp?xItem=8015&ctNode=63 「外交部針對所謂「台灣共和國」護照乙事，予以澄清。」，中華民國（台灣）外交部新聞稿] {{webarchive |url = https://web.archive.org/web/20070330184034/http://www.mofa.gov.tw/webapp/ct.asp?xItem=8015&ctNode=63 |date = 2007-03-30 }}</ref>。

類似於「台灣共和國護照」，還有另一本「台灣國護照」的虛擬護照，某些新聞事件的內容並不確定到底是指台灣共和國或台灣國護照。

== 相关事件 ==
2001年3月15日，入境[[台灣|台灣]]的[[中華民國|中華民國]][[國策顧問|國策顧問]][[金美齡|金美齡]]，在[[臺灣桃園國際機場|臺灣桃園國際機場]]接受[[建國黨|建國黨]]秘書長[[黃玉炎|黃玉炎]]贈予之台灣共和國護照，並以「送我這本護照了」之回答暗示接受以[[台灣共和國|台灣共和國]]國民身份返台<ref>[http://www.epochtimes.com/b5/1/3/15/n58205.htm 「金美齡又來台 高舉台灣共和國護照」，大紀元]</ref>。其後，[[新黨|新黨]][[立法委員|立法委員]][[馮滬祥|馮滬祥]]在3月16日到[[臺灣臺北地方法院檢察署|臺灣臺北地方法院檢察署]]按鈴告發金美齡與建國黨秘書長黃玉炎涉及《[[中華民國刑法|中華民國刑法]]》[[預備內亂罪|預備內亂罪]]。馮滬祥認為，金美齡公然接受該護照、且於眾多媒體記者前高舉該護照之行為，已明顯超越[[言論自由|言論自由]]層次，涉嫌違反[[中華民國刑法第一百條|中華民國刑法第一百條]][[內亂罪|內亂罪]]之「預備階段」。另外，馮滬祥並建議[[檢察官|檢察官]]將該兩人限制出境<ref>[http://www.epochtimes.com/b5/1/3/17/n59198.htm 「馮滬祥決定取消反制世台會遊行 改控告金美齡」，大紀元]</ref>。

2001年5月17日，自稱「[[台灣共和國政府|台灣共和國政府]]聯絡人」的[[高金郎|高金郎]]召開[[記者會|記者會]]，拿出[[美籍台灣人|美籍台灣人]][[郭樹人|郭樹人]]的台灣共和國護照，內有[[宏都拉斯|宏都拉斯]]、[[薩爾瓦多|薩爾瓦多]]、[[貝里斯|貝里斯]]及[[厄瓜多|厄瓜多]]簽證<ref>[http://www.libertytimes.com.tw/2001/new/may/18/today-p8.htm 「台灣共和國護照獲多國簽證」，自由時報] {{webarchive|url=https://web.archive.org/web/20131005113000/http://www.libertytimes.com.tw/2001/new/may/18/today-p8.htm |date=2013-10-05 }}</ref>。這些簽證都是於2001年4月在上述中南美各國駐美國[[洛杉磯|洛杉磯]]的領事館獲取。[[中華民國外交部|中華民國外交部]]獲訊後表示，這本護照不具法律效力，並將主動周知各國注意。人民自行發行虛構國家的護照，並非屬仿造，無相關法律可以規範，然使用護照申請外國簽證仍可能觸犯外國法律（例如對公務員詐欺，使其做出不實記載文書的罪）。該發行單位嗣後聲稱該護照獲得[[巴西|巴西]]等簽證，然皆難以查證。

2003年8月11日，《[[台灣蘋果日報|台灣蘋果日報]]》報導，日前在美國成立的台獨團體「正綠社」強調，正綠社比現在的台獨更[[基本教義|基本教義]]，成員中以拿台灣共和國護照通行中南美洲五國的郭樹人的知名度最高，目前在洛杉磯擁有成員數百人，共通之處是都擁有台灣共和國護照，但「台灣獨派人士對正綠社多半不清楚，也不願評論」<ref>[http://www.appledaily.com.tw/appledaily/article/headline/20030811/261643 「新獨派正綠社反扁」，台灣蘋果日報]</ref>。

== 政府見解 ==
事件發生當時的外交部發言人[[張小月|張小月]]則表示，「台灣共和國護照」是「[[虛擬護照|虛擬護照]]」，並不是合法、也不是有效的旅行文件，[[中華民國政府|中華民國政府]]與邦交國也不會承認<ref>[http://www.ettoday.com/2002/06/20/303-1318262.htm 「總統出訪　外交部表示總統護照無需加註台灣」，東森新聞網]</ref><ref>[http://www.epochtimes.com/b5/1/5/17/n89548.htm 「中華民國護照加上「台灣」外交部多數駐外館處表贊成」，大紀元]</ref><ref>[http://www.mofa.gov.tw/webapp/ct.asp?xItem=8015&ctNode=493&mp=1 中華民國（台灣）外交部新聞稿，第O六O號] {{webarchive|url=https://web.archive.org/web/20070330184110/http://www.mofa.gov.tw/webapp/ct.asp?xItem=8015&ctNode=493&mp=1 |date=2007-03-30 }}</ref>。

== 支持者內部爭議 ==
由於自稱「台灣共和國政府外交部長」的郭樹人表示台灣共和國護照發行單位是台灣共和國政府，[[建國黨|建國黨]]提出反駁稱建國黨才是該護照的真正發行單位。為此，皆為台獨運動活躍機構的該兩單位，曾發生言語糾紛<ref>[http://www.ettoday.com/2001/07/13/1108-469332.htm 「台灣國護照誰發行？建國黨與台灣國政府互相指控」，東森新聞報]</ref>。
