'''臺北捷運萬大樹林線先導公車'''為[[臺北捷運萬大-中和-樹林線|臺北捷運萬大-中和-樹林線]]之[[捷運先導公車|捷運先導公車]]路線，目前由[[三重客運|三重客運]]營運，2013年12月31日闢駛。<ref>[http://icypenguin.pixnet.net/blog/post/174058047-%E6%96%B0%E5%8C%97%E5%B8%82%E4%B8%89%E7%92%B0%E4%B8%89%E7%B7%9A%E3%80%80%E8%90%AC%E5%A4%A7%E6%A8%B9%E6%9E%97%E7%B7%9A%E5%85%88%E5%B0%8E%E5%85%AC%E8%BB%8A 新北市三環三線　萬大樹林線先導公車].企鵝的交通手札</ref>本線於2014年12月17日獲配路線編號「985」，自2015年1月1日正式使用。

== 路線基本資料 ==
捷運萬大樹林線路線規劃乃由[[捷運中正紀念堂站|捷運中正紀念堂站]]出發，經[[南海路|南海路]]、[[萬大路_(台北市)|萬大路]]、[[萬華區|萬華]]第一果菜批發市場、[[永平國小|永平國小]]轉至[[中和區|中和]][[連城路|連城路]]，再行經[[土城區|土城]]（規劃經過[[捷運土城站|捷運土城站]]）、[[樹林區|樹林]]至終點[[捷運迴龍站|捷運迴龍站]]，可轉搭[[捷運新莊線|捷運新莊線]]與[[桃園捷運棕線|桃園捷運棕線]]。

先導公車路線上與未來捷運路線稍有出入，由三重客運新莊站發車，起點站為[[捷運輔大站|捷運輔大站]]出發，過捷運迴龍站後轉入樹林區中正路、大安路，再由[[城林橋|城林橋]]進入土城（經[[捷運海山站|捷運海山站]]），後行駛土城金城路與中和連城路，後經[[華中橋|華中橋]]抵達萬華及終點站[[捷運龍山寺站|捷運龍山寺站]]。

== 爭議 ==
捷運萬大樹林線路線規劃有經過[[永和區|永和區]]的，但公車路線卻是從下華中橋後，直接轉往景平連城路口，完全沒有經過永和，反而納入了不在捷運路線規劃的新莊輔大段，而引發了爭議。<ref>[http://news.housefun.com.tw/news/article/11239852397.html 萬大中和樹林線先導公車　卻不經過永和挨批]</ref>
