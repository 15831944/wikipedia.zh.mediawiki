'''社團法人台灣照顧生命協會'''，簡稱'''照生會'''，是[[台灣|台灣]]的[[動物保護|動物保護]]組織，於2005年由[[董冠富|董冠富]]創立<ref name=":0">{{Cite news|url=http://www.tanews.org.tw/info/6045|title=風言風語中 照生會屹立10年|author=江幸芸|date=2014-11-03|newspaper=台灣動物新聞網|accessdate=2018-04-28|language=zh-hant|deadurl=no|archiveurl=https://web.archive.org/web/20160908082505/http://www.tanews.org.tw/info/6045|archivedate=2016-09-08}}</ref><ref name=":1">{{Cite news|url=https://tw.appledaily.com/new/realtime/20140812/450260/|title=【民報】跳脫舒適圈的照生會創辦人董冠富|author=魏翊庭|date=2014-08-12|newspaper=臺灣蘋果日報（民報）|accessdate=2018-04-28|language=zh-TW|deadurl=no|archiveurl=https://web.archive.org/web/20180430093754/https://tw.appledaily.com/new/realtime/20140812/450260/|archivedate=2018-04-30}}</ref><ref>{{Cite web|url=http://www.merit-times.com.tw/newspage.aspx?unid=441142|title=飼主的教育不能等 突破3大困境|accessdate=2018-04-28|author=李碧華|date=2016-07-09|publisher=人間福報|deadurl=no|archiveurl=https://web.archive.org/web/20180430093755/http://www.merit-times.com.tw/newspage.aspx?unid=441142|archivedate=2018-04-30}}</ref>，總部設於[[台北市|台北市]][[文山區|文山區]]<ref>{{Cite news|url=https://n.yam.com/Article/20170605683074/%E7%AD%89%E5%BE%85%E5%96%84%E5%BF%83%E7%9A%84%E4%BD%A0%E4%BC%B8%E5%87%BA%E6%8F%B4%E6%89%8B%E3%80%80%E5%8F%B0%E7%81%A3%E7%85%A7%E9%A1%A7%E7%94%9F%E5%91%BD%E5%8D%94%E6%9C%83%E3%80%8C%E8%B2%93%E7%8B%97119%E3%80%8D%E7%B6%93%E8%B2%BB%E5%9A%B4%E9%87%8D%E4%B8%8D%E8%B6%B3|title=等待善心的你伸出援手　台灣照顧生命協會「貓狗119」經費嚴重不足|author=袁慧心、羅蔚舟|date=2017-06-05|newspaper=蕃新聞（大成報）|accessdate=2018-04-28}}</ref>。前身為「台北縣照顧生命協會」，後才更名為台灣照顧生命協會。該協會尚有新北市照顧生命協會、台北市照顧生命協會與照生企業有限公司等多個相關組織，共同組成「照生聯盟」<ref name=":0" />。

截至2017年，照生會在台灣共有五家狗場，收容七百多隻狗<ref>{{Cite news|url=http://www.nextmag.com.tw/realtimenews/news/259762|title=以暴制暴救貓狗　照生會募款組暗黑部門惹議|author=謝東明|date=2017-03-14|newspaper=臺灣壹週刊|accessdate=2018-04-29|language=zh-TW|deadurl=no|archiveurl=https://web.archive.org/web/20180429090405/http://www.nextmag.com.tw/realtimenews/news/259762|archivedate=2018-04-29}}</ref>，其中還包括了[[愛狗人士|愛狗人士]][[賈鴻秋|賈鴻秋]]老師身故後遺留下來的[[流浪狗|流浪狗]]<ref>{{Cite web|url=http://www.merit-times.com.tw/newspage.aspx?unid=189761|title=【社區部落格】台南護生園 流浪狗新家|accessdate=2018-04-29|date=2010-07-02|publisher=人間福報|deadurl=no|archiveurl=https://web.archive.org/web/20180429160145/http://www.merit-times.com.tw/newspage.aspx?unid=189761|archivedate=2018-04-29}}</ref>。

== 沿革 ==
照生會最初的前身為董冠富於2004年在奇摩家族上成立的「阿虎家族急難救助會」，當時尚兼職保全工作的董冠富親自與朋友前往救援[[網友|網友]]通報的流浪貓狗；後來他辭去工作，於2005年、2007年、2010年分別成立社團法人新北市照顧生命協會、台灣照顧生命協會和台北市照顧生命協會<ref name=":1" /><ref name=":2">{{Cite news|url=http://www.nextmag.com.tw/realtimenews/news/10682361|title=讓貓狗脫離被公立收容所安樂死命運！|date=2014-11-17|work=臺灣壹週刊|accessdate=2018-04-28|language=zh-TW|deadurl=no|archiveurl=https://web.archive.org/web/20180430093750/http://www.nextmag.com.tw/realtimenews/news/10682361|archivedate=2018-04-30}}</ref><ref>{{Cite news|url=https://www.watchinese.com/article/2016/22321|title=「阿虎」董冠富 風雨中成為動物救援急先鋒|author=劉佩絨|date=2016-09-05|newspaper=看雜誌|accessdate=2018-04-28|language=zh-hant|deadurl=no|archiveurl=https://web.archive.org/web/20170708070049/https://www.watchinese.com/article/2016/22321|archivedate=2017-07-08}}</ref>。

後照生會於[[台南|台南]]設立一座示範性護生狗場（月華護生狗場）<ref name=":2" /><ref name=":3">{{Cite web|url=http://www.lco.org.tw/modules/pages/main|title=協會介紹|accessdate=2018-04-28|publisher=社團法人臺灣照顧生命協會|language=zh-tw}}</ref>。

因藝人[[費玉清|費玉清]]於2013年捐助一百萬元予照生會，使其有足夠經費蓋建新的收容場地，在先後舉辦兩次[[募款|募款]]活動後，照生會「觀音[[素食|素食]]狗場」在2015年[[元旦|元旦]]正式啟用，地址設於台南市[[左鎮區|左鎮區]]，其中收容的犬隻皆一律素食<ref>{{Cite news|url=http://www.fingermedia.tw/?p=226492|title=台灣照生會「台南觀音素食狗場」啟用  6犬隻入園後開心玩耍|author=游勝鈞|date=2015-01-02|newspaper=指傳媒 Fingermedia|accessdate=2018-04-28|language=zh-TW|deadurl=no|archiveurl=https://web.archive.org/web/20180430093749/http://www.fingermedia.tw/?p=226492|archivedate=2018-04-30}}</ref><ref>{{Cite news|url=http://www.tanews.org.tw/info/6505|title=照生會觀音素食狗場　元旦啟用了|author=呂幼綸|date=2015-01-03|newspaper=台灣動物新聞網|accessdate=2018-04-28|language=zh-hant}}</ref><ref>{{Cite news|url=https://www.nownews.com/news/20141008/1447594|title=尊重生命！建蓋「素食狗場」　照生會辦募款音樂餐會|author=李念庭|date=2014-10-08|newspaper=NOWnews 今日新聞|accessdate=2018-04-28|language=zh-TW}}</ref><ref>{{Cite news|url=https://www.ettoday.net/news/20141008/410914.htm|title=興建「素食狗場」　台北照生會舉辦募款音樂餐會|date=2014-10-08|newspaper=ETtoday新聞雲|accessdate=2018-04-28|language=zh-TW|deadurl=no|archiveurl=https://web.archive.org/web/20141129092838/http://www.ettoday.net/news/20141008/410914.htm|archivedate=2014-11-29}}</ref><ref>{{Cite news|url=http://www.chinatimes.com/newspapers/20161118000570-260107|title=修繕流浪狗家園 亟待援助|author=曹婷婷|date=2016-11-18|newspaper=中時電子報|accessdate=2018-04-28|language=zh-TW|deadurl=no|archiveurl=https://web.archive.org/web/20161119020425/http://www.chinatimes.com/newspapers/20161118000570-260107|archivedate=2016-11-19}}</ref>。

== 貓狗119 ==

== 相關組織 ==
「照生聯盟」除台灣照顧生命協會外尚有新北市照顧生命協會、台北市照顧生命協會、照生企業有限公司、照生人志工團、廈門董氏貿易有限公司，共六個組織<ref name=":3" />。其中新北市照顧生命協會於2009年推出全台首部「貓狗[[結紮|結紮]]車」<ref>{{Cite news|url=http://news.ltn.com.tw/news/local/paper/291768|title=〈台北都會〉設備全 貓狗巡迴結紮車上路|author=何玉華|date=2009-03-31|work=自由時報電子報|accessdate=2018-04-28|deadurl=no|archiveurl=https://web.archive.org/web/20180430093749/http://news.ltn.com.tw/news/local/paper/291768|archivedate=2018-04-30}}</ref>，並陸續設立貓狗[[捐血|捐血]]中心、親善119等部門。而台北市照顧生命會則著重於關懷弱勢家庭，推動[[陪伴犬|陪伴犬]]計畫<ref name=":3" />。
