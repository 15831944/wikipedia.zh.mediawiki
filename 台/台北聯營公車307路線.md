'''臺北聯營<span style="color: red;">307</span>路線'''，為[[臺北聯營公車|臺北聯營公車]]路線之一，由[[臺北客運|臺北客運]]與[[首都客運|首都客運]]經營，起點為松山莊敬里，終點為台北客運板橋前站
，途經[[臺北市|臺北市]][[松山區_(臺灣)|松山區]]、[[中山區_(臺北市)|中山區]]、[[中正區_(臺北市)|中正區]]、[[萬華區|萬華區]]及[[新北市|新北市]][[中和區|中和區]]、[[板橋區_(台灣)|板橋區]]。總共行經民生國中、萬大國小、中和高中、積穗國中、板橋國中等五所各級學校，以及[[臺北車站|臺北車站]]、[[西門町|西門町]]、[[環球購物中心|中和環球購物中心]]、[[新板特區|新板特區]]等人潮聚集地，是大臺北地區重要、最密集和運量最多的公車路線。全長47.9公里，全臺北目前共配有101台車來回各有124站，平均2至5分鐘來一班，尖峰每2至4分鐘、離峰時段每4至6分鐘一班車，每天發車班次高達780個班次，載客數高達七萬人次，營業額一天超過新台幣110萬元。

== 歷史 ==
*1969年，[[大有巴士|大有巴士]]正式成立，本路線隨即開辦，當時由大有巴士單獨營運，來往松山至忠孝西路，路線編號為'''大有12路'''。
*1970年3月，改經林森北路。
*1975年7月，本路線改為雙向循環線，先駛經走南京東路的稱為'''12右'''，反過來先經信義路的則為'''12左'''。
*1977年4月30日，大有12左線首先加入聯營公車的行列，同時雙向循環線改為雙方向不循環行駛。12左線被併入聯營204(當時來往南松山至東園)而停駛，另外開辦了聯營64（來往南松山至中華路）來完全替代原12左的路線。同年5月15日，大有12右線亦加入聯營公車，更改路線稱為'''307'''，同時延駛板橋，臺北客運亦於當天加入本線營運<ref>〈公車聯營明天開始 第1梯次33條線聯營中心今晚調換站牌〉，《[[聯合報|聯合報]]》1977年4月29日第6版</ref>。當日此路線改為3段票收費。
*1979年9月1日，票價調整為2段票收費。
*1997年9月11日，由於307線營運單位[[大有巴士|大有巴士]]多次未依核定路線行駛307線，[[臺北市政府交通局|臺北市政府交通局]]除依規定處以[[罰鍰|罰鍰]]外，並-{吊}-扣307線10部車輛行車執照及[[車輛號牌|車輛號牌]]一個月，減少班次由其他客運公司車輛補足。
*2005年，本線南松山線與撫遠街線正式合併，改為一律行駛至撫遠街迴轉。
*2012年3月15日，因為大有巴士虧損及財務日遽惡化，[[臺北市政府交通局|臺北市政府交通局]]公共運輸處自2012年3月16日起永久收回大有巴士的307線經營權。<ref name="蘋果日報">{{Cite news|url=http://www.appledaily.com.tw/appledaily/article/headline/20120316/34093564/|title=虧12億 大有307路線遭取消|language=zh-tw|publisher=《蘋果日報》|author=許麗珍|location=臺北|date=2012-03-16|accessdate=2014-08-08}}</ref>
*2012年3月16日，原本由臺北客運與大有巴士經營的307線改由臺北客運獨自經營。<ref name="蘋果日報"></ref>
*2012年6月12日，臺北客運公告，307線申請調整部分班次行駛[[西藏路|西藏路]]並增設「廈安-{里}-」及「西藏路」雙邊站位，核定自同年6月16日起試辦3個月（有行經車輛前擋風玻璃有放置「行經西藏路」看板）。
*2013年2月21日，307線正式由[[臺北客運|臺北客運]]及[[首都客運|首都客運]]接駛。
*2014年起，307線將使用三門低地板公車，以及配上[[圓仔|圓仔]]圖案。
*2014年4月28日至7月31日，307線平常日尖峰時間試辦五班次（板橋國中發車時間：6:30，6:48，7:08，7:25，7:45），板橋往撫遠街單向直行中和中山路（即不經員山路與連城路），且直行區段不提供上下客。即積穗站過後，下一站為華中橋。
*2014年8月5日，因應[[2014年高雄氣爆事故|2014年高雄氣爆事故]]，307線全線義跑一天，當日該線最後營收為新台幣143萬3028元，比平常日約新台幣110萬元多出3成。此外，首都客運董事長李博文對乘客的愛心表示感謝和感動，決定增添心意，將捐助總額提升為新台幣200萬元，全部匯款至「高雄市政府社會局社會救助金專戶」統籌濟助災民，希望災民感受到雙北市民和公車業者的愛心。<ref>{{Cite news|url=http://www.appledaily.com.tw/appledaily/article/headline/20140806/36004788/%E5%85%AC%E8%BB%8A%E4%B9%8B%E7%8E%8B307%E7%BE%A9%E8%B7%91%E5%8A%A9%E9%AB%98%E9%9B%84|title=公車之王307 義跑助高雄|language=zh-tw|publisher=《蘋果日報》|author=郭美瑜|location=臺北|date=2014-08-06|accessdate=2014-08-08}}</ref>
*2014年11月25日，[[2014年中華民國地方公職人員選舉|2014年中華民國地方公職人員選舉]]中[[中國國民黨|中國國民黨]][[臺北市市長|臺北市市長]]侯選人[[連勝文|連勝文]]陣營公佈了一部批評同樣競選臺北市市長的[[無黨籍|無黨籍]]侯選人[[柯文哲|柯文哲]]陣營的影片，名為「台北‧明天過後」，嘲諷若讓柯文哲當選，台北市會不斷退步而讓選民後悔投柯，但有位網友fabg指出，片中有部場景為女主角自西藏路莒光新城的「西藏路」站，搭本路線公車想到[[捷運大橋頭站|捷運大橋頭站]]，但卻發現一張寫著「因本站牌經海量數據計算不符合效率，自2015年12月取消服務」的公告，但問題是本線公車根本無法到捷運大橋頭站，離[[西藏路_(台北市)|西藏路]]不遠的[[萬大路_(台北市)|萬大路]]明明有「萬大路」站與「西藏路口」站停靠著可以直達捷運大橋頭站的[[台北聯營公車601路線|601號公車]]，卻大老遠跑來搭不能到達捷運大橋頭站的本線公車，質疑連陣營根本不認識台北
*2018年2月3日改由板橋前站調度，板橋端增設臺北客運板橋前站（藝文）新月橋（單邊）等站位，裁撤新海抽水站（雙邊）名萊新城（南邊）等站位
*2018年3月2日配到HINO弘鉅低地板新車，因此少數三門低地板公車調離本線。

== 營收 ==
2003至2012年台北公車的年度總營收，307線皆蟬聯營收冠軍，是雙北之間最熱門的公車路線之一，年營收新台幣4億元，為全台灣最賺錢的公車<ref>{{Cite news|url=https://www.youtube.com/watch?v=DSt_4h-kFFo|title=全台最賺公車"307" 年收4億稱霸11年|language=zh-tw|publisher=非凡電視台|location=臺北|date=2014-07-31|accessdate=2019-01-11}}</ref>。

== 紀錄 ==
# 承上所述，年營收新台幣4億元，是全台灣最賺錢的公車路線。
# '''臺北聯營公車307路線'''，曾經被民眾目擊，數度有連續七輛307公車並行於[[公車專用道|公車專用道]]上{{fact}}。
# 2014年8月5日，行駛大臺北地區的307線公車，為了幫助在高雄連環氣爆中受災的災民，特別「愛心義跑」一天，首都客運及臺北客運在隔天上午表示，統計307線公車，當天義跑一天的營收，結果創了'''這條路線單日營收的新高紀錄'''，同時也是創下'''國內公車單日營收的最高紀錄'''（結算為一百四十三萬三千零二十八元，比平常日約一百一十萬元，多出營收三成）。
# 307線配車車輛數共有101部車，為國內公車單一路線之最。
