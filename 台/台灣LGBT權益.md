'''[[中華民國|中華民國]]（[[臺灣|臺灣]]）'''通常被認為是[[東亞|東亞]]、乃至[[亞洲|亞洲]]地區之中，少數[[LGBT權益|LGBT權益]]相對友善之地區<ref>{{cite news|title=臺灣成為亞洲同性戀平權運動的燈塔|url=http://cn.nytimes.com/china/20141031/c31taiwan/zh-hant/|accessdate=2015-12-21|agency=紐約時報|date=2014-10-31}}</ref>。每年十月最後一週星期六在[[臺北市|臺北市]]舉辦的[[臺灣同志遊行|臺灣同志遊行]]吸引數萬人參與，為亞洲僅次於[[以色列|以色列]][[特拉維夫|特拉維夫]]第二大的[[LGBT|LGBT]]活動<ref>{{cite news|title=特稿：當同志大遊行的彩虹旗在臺北飄揚|url=http://www.bbc.com/zhongwen/trad/china/2015/10/151030_sp_taiwan_gay_pride|agency=BBC|date=2015-10-30}}</ref>。

[[同性性行為|同性性行為]]在臺灣是合法的，針對[[LGBT|同志]]的[[暴力|暴力]]事件相當少見，[[警方|警方]]對同志友善空間施壓的情況逐漸減少<ref>{{cite web|title=2009 Human Rights Reports: Taiwan|url=http://www.state.gov/j/drl/rls/hrrpt/2009/eap/135990.htm|website=美國政府|quote=There are no laws prohibiting homosexual activities. 在反歧視法方面，分別在2003年和2007年制定了《[[性別平等教育法|性別平等教育法]]》和《[[性別工作平等法|性別工作平等法]]》，保障教育和工作領域不得有性別、性傾向之差別待遇。在2011年《[[性別平等教育法|性別平等教育法]]》修正條文中，實施「性別平等教育」，以教育方式教導尊重[[多元性別|多元性別]]（指性別除[[生理性別|生理性別]]外，還包括[[性傾向|性傾向]]、[[性別_(文化)|性別特質]]、[[性別認同|性別認同]]）差異，消除[[性別歧視|性別歧視]]，促進性別地位實質平等。2015年通過「長期照顧服務法」，將[[長期照護|長期照護]]納入反性傾向歧視範疇。2016年[[民進黨|民進黨]]為立法院多數後，涉婚姻平權的民法修正案，在[[立法院|立法院]]被提上議程，獲多數立委主持。臺灣[[司法院大法官|司法院大法官]]於2017年5月24日釋憲，[[中華民國民法|中華民國民法]]第4編親屬第2章婚姻規定違憲，要求兩年內修正或制定相關保障同性婚姻法律，使臺灣有機會成為亞洲第一個承認同性婚姻的地区。

根據一份2014年的研究，臺灣社會普遍接納同志，不會因其性別而拒絕他們交友、求職等權利，多數反對對同志的種種[[刻板印象|刻板印象]]。然而問及若自己子女是同志時，其意見則明顯保留<ref>{{cite journal|author1=王素真|author2=陳住銘|author3=洪耀釧|title=民眾對於同性戀的認知與接受程度之探討|journal=工程科技與教育學刊|date=2014|volume=11|issue=1|pages=30-42|url=http://www.engh.kuas.edu.tw/files/ne/vspid05amk.pdf}}</ref>。2012年《臺灣社會變遷基本調查計畫》指出55%受訪者贊成應立法保障同志合法結婚權，37%反對<ref>{{cite web|title=同性戀與學術調查 （三）： 一次公允的代表性研究──2012臺灣社會變遷調查-第六期第三次（下）|url=http://socioequity.blogspot.tw/2013/05/2012_24.html|accessdate=2015-12-21}}</ref>，其中以年輕族群對於[[臺灣同性婚姻|婚姻平權]]特別支持。<ref>{{Cite web|url=https://www.setn.com/News.aspx?NewsID=51829|title=「同志婚姻合法化」最新調查 2X歲民眾高達84%支持 {{!}} 首頁 {{!}} 三立新聞網 SETN.COM|accessdate=2018-08-15|last=三立新聞網|language=zh-Hant-TW}}</ref><ref>{{cite web|title=公民教育大調查出爐！不一樣愛國觀，近5成年輕人不願為國上戰場|url=http://topic.cw.com.tw/2011edu/article/press1-3.aspx|website=天下雜誌|deadurl=yes|archiveurl=https://web.archive.org/web/20151222135114/http://topic.cw.com.tw/2011edu/article/press1-3.aspx|archivedate=2015-12-22}}</ref>

==同性性行為合法性 ==
[[臺灣|臺灣]]沒有任何關於[[同性性行為|同性性行為]]的法律，不像其他西方國家經歷過[[除罪化|除罪化]]的過程。[[成人|成人]]、非[[商業|商業]]、合意一致的同性性行為在臺灣[[合法|合法]]。在[[同意年齡|同意年齡]]上，臺灣，同意年齡均為16歲，在性別上沒有特別規定。

《[[中華民國刑法|中華民國刑法]]》中，對「同性性行為」沒有特別規定。其第10條規定之「非基於正當目的所為之下列[[性侵入行為|性侵入行為]]」，指以[[性器|性器]]或他物，進入他人之性器、[[肛門|肛門]]或[[口腔|口腔]]<ref>{{Cite web|url=http://law.moj.gov.tw/LawClass/LawAll.aspx?PCode=C0000001 |title=《[[中華民國刑法|中華民國刑法]]》|publisher=全國法規資訊網 |language=zh-tw |accessdate=2013-11-29}}</ref>。無論[[異性性行為|異性性行為]]或同性性行為，只要不符合「合意一致」原則，皆為[[違法|違法]]。

== 反歧視法規與措施 ==
1993年2月，[[我們之間|我們之間]]、[[亞洲女同志聯盟|亞洲女同志聯盟]]、[[AIDS中途之家|AIDS中途之家]]等6個LGBT與[[愛滋病|愛滋病]]防治團體在[[立法院|立法院]]舉辦人權公聽會，就尚在審理的《反歧視法》草案未納入保障同志權益一事發表聲明，敦促社會大眾重視同志人權，這是同志議題第一次被納入國會討論<ref name="張娟芬">{{Cite book|title=《揚起彩虹旗》|author=張娟芬、許佑生|publisher=心靈工坊文化|accessdate=2011-06-05|year=2002|isbn=978-9-868-02486-1|location=臺北市|pages=259-262|language=zh-tw}}</ref>。

在同志人權教育方面，臺灣同志諮詢熱線在2001年主辦「認識同志－教師研習營」，共吸引50位高級中學教師參與，並由臺北縣、市政府教育局核發研習時數證明，是臺灣教育主管單位第一次承認由同志團體舉辦的教育研習活動<ref name="張娟芬" />。同年6月，[[教師同盟|教師同盟]]會長[[段健發|段健發]]公開出櫃，受其任教的[[桃園縣立桃園國民中學|桃園國中]]與前教育部長[[曾志朗|曾志朗]]的肯定，並聲明尊重同志教師的人權與工作權<ref name="張娟芬" />。

== 性別認同和表達 ==
{{Main|臺灣跨性別權益}}
=== 身分證與戶籍性別登記 ===
[[中華民國國民身分證|中華民國國民身分證]]與[[戶籍|戶籍]]登記是主要身份證明。其他[[駕照|駕照]]、[[行照|行照]]、學歷證件等都要求這兩項。

身分證字號有性別之分（舊身分證連顏色也有性別之分）。身分證上的照片則要求要符合生理性別。平時各種身分驗證要求的則是外觀與身分證照片，身分證標示的性別一致。另外戶籍登記中有出生登記，出生登記上有標明性別。還有不論性別，一人一生只能更改一次姓名{{ref|戶籍法}}。變性者完成變性手術後可至戶政事務所辦理變更出生登記與身分證。

但是困難之處在於要作變性手術前需要一段長時間的準備，預先體驗另一個性別的生活，稱為「真實生活體驗」。真實生活體驗甚至是醫師同意作變性手術的必要條件。生活體驗多少要求外觀，也就會與身分證標示的性別衝突，而在完成變性手術之前無法改變身分證上之性別。在臺灣正在進行生活體驗之人也就時常遇到身分證與外貌不符之法律、行政問題。

1997年，[[臺北市政府|臺北市政府]][[民政局|民政局]]規定：「男扮女裝之裝扮，與真實性別不符，應屬個人行為，其請領國民身分證所繳之相片人貌應與真實性別相符。」{{ref|1997}}

2002年7月，臺北市民[[蔡雅婷|蔡雅婷]]申請換發身分證，同樣因照片理由遭拒，[[內政部|內政部]][[戶政司|戶政司]]作出相同回應{{ref|2002}}。

=== 性別登記變更 ===
身分證由[[戶籍法|戶籍法]]管理，主管機關為[[內政部|內政部]]。戶籍法中沒有「性別」兩個字。但是第4條規定：「戶籍登記，指下列登記：一、身分登記： (一) 出生登記」{{ref|戶籍法}}。出生登記有註明性別。

根據戶籍法23條：「戶籍登記事項有變更時，應為變更之登記。」以及根據戶籍法施行細則{{ref|戶籍法}} 第23條： 
: 三、換領：
: 申請戶籍登記致國民身分證記載事項變更而換領者，向各該申請登記之戶政事務所申請；有毀損之情形者，得向任一戶政事務所申請。
變性者完成變性手術後可帶舊文件與[[診斷證明|診斷證明]]等至戶政事務所辦理變更出生登記與身分證。{{ref|變更}}

=== 性別重置手術 ===
臺灣法律對性別重置手術沒有規定。不過醫師都會要求[[家屬同意書|家屬同意書]]，作為被控告[[重傷害罪|重傷害罪]]時的保障。在进行性别重置手术后，台湾公民可以合法修改自己的法律性别。

== 企業 ==
同志人權法案遊說聯盟邀請企業簽署「反歧視宣言」與「同志人權公約」，承諾不得對LGBT員工及其家屬有任何形式的歧視、並且提供LGBT伴侶或配偶與異性戀等同的企業福利
<ref>{{cite news|title=LGBT、異性戀福利均等 5台企業「領先國家」簽公約|url=http://www.ettoday.net/news/20140526/361294.htm|agency=東森新聞|date=2014-05-26}}</ref>。
簽署企業包括白金科技股份有限公司、FlyingV昂圖股份有限公司、德臻法律事務所、杰德創意影音管理股份有限公司、Nexdoor 宇軒數位有限公司、好孩子國際娛樂有限公司。

2010年，臺北[[馬偕紀念醫院|馬偕紀念醫院]]資訊工程師周逸人因其跨性別身分遭院方開除，法官認定為歧視並違反《勞基法》。

== 媒體 ==
臺灣的媒體過去對LGBT經常有汙名化與標籤化的報導，臺灣同志壓力處境調查顯示，媒體的不友善為同志的第三大壓力源<ref>{{Cite web|url=http://hotline.org.tw/news/190|title=1010417【調查結果】「臺灣同志壓力處境問卷」結果初步分析|accessdate=2018-08-15|work=臺灣同志諮詢熱線協會|language=zh-hant}}</ref>。

NCC於2012年通過《廣電媒體製播涉及性別相關內容指導原則》，建議廣電媒體應就「避免造成偏見、歧視、物化、刻板印象或偏差性別觀念」、「以正面、積極、多元的方式呈現性別角色」等原則，於相關自律倫理規範中訂定具體落實之執行細節<ref>{{cite web |author1=同志諮詢熱線 |title=台灣同志(LGBTI) 人權政策檢視報告 |url=https://hotline.org.tw/sites/hotline.org.tw/files/20180103_%E4%BA%BA%E6%AC%8A%E5%A0%B1%E5%91%8AA4_%E4%B8%AD%E6%96%87%E7%89%88_pages.pdf}}</ref>。

==影視 ==
台灣同志劇情電影自1985年出現第一部描寫同性情愫的《孤戀花》，在《17歲的天空》、《盛夏光年》、《刺青》的賣座後，文本愈加熱烈的出現<ref>{{cite book |author1=顏培鑫 |title=台灣酷兒／同志劇情電影成長書（1985〜2008） |date=2009 |publisher=國立臺南藝術大學音像管理研究所 |url=https://hdl.handle.net/11296/u75q32}}</ref>。

[[台灣金馬獎|台灣金馬獎]]於 1992 年首次開辦同志影展專題。台灣電影創作上，李安的《囍宴》（1993）獲柏林影展金熊獎、蔡明亮橫掃國際影展的《愛情萬歲》（1995）、《河流》（1997）以及林正盛的《美麗在唱歌》（1997）獲頒坎城影展會外賽金棕櫚獎等，都是國際留名之作<ref>{{cite web |author1=陳韋臻 |title=彩虹光影的折射：小談近30年台灣同志影視 |url=https://www.fountain.org.tw/r/post/taiwan-lgbt-movie}}</ref>。

隨著社會的發展，[[偶像劇|偶像劇]]亦展現其進步性。即使劇中多數同志角色仍然具有傳統刻板印象，但偶像劇在呈現同志的方式更加多元、開放<ref>{{cite book |author1=王皓宇 |title=台灣偶像劇中的同志再現分析（2001—2014） |date=2015 |publisher=輔仁大學 大眾傳播學研究所碩士班 |url=https://hdl.handle.net/11296/x3p5zq}}</ref>。
