'''臺9線南迴公路拓寬改善後續計畫'''，一般被稱為'''南迴公路改善計畫'''，簡稱'''南迴改'''，實施路段為[[臺灣省道|省道]][[台9線|台9線]][[臺東縣|臺東縣]]香蘭－[[屏東縣|屏東縣]]草埔段。[[臺灣東部|臺灣東部]]首要的聯外道路－南迴公路（現編號為省道台9線），南迴公路闢建於[[臺灣日治時期|日治時期]]，1933年（昭和8年）開始興工，1939年（昭和14年）完成楓港－阿塱衛（今[[達仁鄉|達仁]]）－臺東間，可通行[[汽車|汽車]]的「楓港臺東道」。戰後改稱為「[[南迴公路|南迴公路]]」。

惟其道路建設完成至今，受制於地形地勢之限制，道路有設計標準偏低、道路線型不佳、縱坡起伏大及路寬不足等限制，加上高路堤及高邊坡多，易因災害受損而交通中斷，除人命財產損失外，對區域性經濟發展及產業運輸造成重大影響，故[[交通部公路總局|交通部公路總局]]因而提出此改善計畫，並由轄下的[[西部濱海公路|西部濱海公路]]南區臨時工程處第六工務段負責設計監工，著重[[東臺灣|東臺灣]]策略發展軸地區的聯外交通安全，優化在地運輸服務品質，建設經費共204.5億元[[新臺幣|新臺幣]]，建設期程2011年至2020年<ref>{{cite web |url=https://pweb.ceci.com.tw/vision/intro.aspx|title=交通部公路總局 台9線南迴公路拓寬改善後續計畫 }}</ref>。

== 計畫內容 ==
=== [[中華民國行政院|行政院]]南迴公路改善目標 ===
# 提供一條安全、穩定回家的道路：以隧道、橋梁方式，避開地質脆弱段，提高用路人安全與災害替代路線，紓解[[中國春節|過年]]過節返鄉、旅行之車流量。
# 提升區域運輸系統功能服務品質：改善安朔草埔段道路線型不佳之運輸瓶頸，符合[[節能減碳|節能減碳]]效益，整合[[鐵路|鐵路]]及公路系統，提高複合運輸服務績效。
# 帶動沿線觀光[[經濟|經濟]]：整合沿線旅遊景點，活絡東部觀光產業。

=== 計畫路段 ===
* 香蘭－大鳥段：平縱面線形原則上依循現有臺9線公路拓寬，局部半徑過小處予以調整改善。
* 安朔－草埔段：以[[隧道|隧道]]與[[高架橋|高架橋]]形式改善安朔草埔段道路線型不佳之運輸瓶頸。
* 草埔－新路段：以部分高架橋形將既有路線拓寬。

=== 工程內容 ===

* 香蘭－大鳥段<ref>{{cite web |url=https://pweb.ceci.com.tw/vision/DesignPoint.aspx|title=台9線南迴公路拓寬改善後續計畫香蘭至大鳥段新闢工程 }}</ref>

# 橋梁工程：本計畫除既有[[瀧橋|瀧橋]]、[[多良橋|多良橋]]、[[富山橋|富山橋]]及[[加津林橋|加津林橋]]等四座排水橋因大多於1993年完工通車，至今近二十年，並無耐震設計考量，經年累月使用與環境侵蝕，已有局部呈現損壞，需在既有強梁外側新增橋梁，以及[[大溪明隧道|大溪明隧道]]段等需配合道路拓寬外，另430K+250處配合道路截彎取直，需新增橋梁。
# 道路工程：平縱面線形原則上依循現有臺9線道路拓寬，局部半徑過小處予以調整改善。計畫範圍地形陡峭，為減輕工程大挖大填對地貌、[[生態|生態]]衝擊，路線構造以海側單邊拓寬。[[多良村|多良]] - [[加津林|加津林]]段緊鄰[[海岸線|海岸線]]，斷面構造型式著重防止[[海浪|海浪]]淘刷；大鳥段沿山坡而行，路線設計採分離線方式因應。

* 安朔－草埔段<ref>{{cite web |url=https://pweb.ceci.com.tw/vision/DesignPoint.aspx|title=台9線南迴公路拓寬改善後續計畫安朔至草埔段新闢工程 }}</ref>
# [[草埔隧道|隧道工程]]：本計畫長度約4.6公里，行車淨空高為4.6公尺，每車道寬為3.5公尺，兩車道路幅兩側各留路肩寬0.3公尺，隧道之維修步道採15公分高之低步道配置，人行空間寬1.0公尺，高2.0公尺；維修步道下方設置電氣管溝，並視需要設置管線廊道，兩側並設置透水管盲溝排除隧道周圍之地下水。
#道路工程：計畫路線起點鄰近[[達仁社區|達仁社區]]，行經[[安朔村|安朔社區]]，終點鄰近[[草埔村|草埔社區]]，為滿足社區進出需求，本計畫路線提供社區居民於適當路段進出計畫道路。
#橋梁工程：跨越安朔溪及沿[[五福谷溪|五福谷溪]]路段採橋梁設計，以避免對生態棲地阻隔，另為減少路堤邊坡挖填，所造成對崩塌地再度擾動，路線以橋梁跨越崩塌地。

* 草埔－新路段<ref>{{cite web |url=https://n.yam.com/Article/20190322139870|title=台東太平溪左岸路堤共構工程　饒慶鈴陪同立委童惠珍視察 }}</ref>
# 草埔－新路段計畫以全線雙向四車道為主，並分為五標進行施工。2019年3月第一標草埔外環高架路段及丹路外環高架橋施工中，草埔外環高架橋計畫配置單向雙車道作為南下線，保留既有山側舊路作為北上線，丹路外環高架橋計畫配置雙向四車道，用以將南迴公路龐大車流導向新外環路線避開丹路部落，其餘第2、3、5標可行性評估已通過，正進行環境影響評估簽辦中。

== 路權 ==
除了安朔－草埔段隧道因長隧道，機慢車限制僅能行駛舊線（[[台9戊線|台9戊線]]），此規劃造成許多抗議聲浪，質疑省道路線為何不能行駛機慢車；其他南迴公路改善工程不限制車種。

== 道路規格 ==
* 一般省道雙向車道標準，最高設計速率時速50－60公里。
* 草埔隧道、安朔高架橋及[[金崙大橋|金崙高架橋]]與多良高架橋為封閉式道路，草埔隧道禁行機慢車，金崙高架橋為雙向四線混合車道（汽機車共用四線道），與新香蘭高架橋同為舊線截彎取直，大鳥至香蘭部分路段為原線改為南向二車道，原線東側新建北向二車道高架橋。其他路段都為內側禁行機車道及外側混合車道配置，最高設計速率時速50－60公里。

== 文化遺址 ==
2014年7月2日，在金崙大橋施工區的第三號橋墩發現遺址陶片和[[石器|石器]]，公路總局暫停施工<ref>{{cite web |url=http://www.cna.com.tw/news/ahel/201407020343-1.aspx|title=南迴拓寬挖到遺址 工程暫停 }}</ref>。

在經過中研院試開挖十三個洞探勘，最先確認第五橋墩沒有遺址，第二及第四橋墩位置似有遺址，第三橋墩則挖出許多石器、[[陶片|陶片]]、[[木炭|木炭]]及類似石牆的石板排列組合，與舊香蘭、多良遺址為同時期，文物文化層明顯，鑑驗屬[[三和文化|三和文化]]並且暫時命名為[[賓茂三遺址|賓茂三遺址]]<ref>{{cite web |url=https://tw.news.yahoo.com/%E5%8D%97%E8%BF%B4%E6%8B%93%E5%AF%AC%E9%81%87%E9%81%BA%E5%9D%80-%E5%AE%8C%E5%B7%A5%E9%81%99%E9%81%99%E7%84%A1%E6%9C%9F-220110230.html|title=南迴拓寬遇遺址 完工遙遙無期 }}</ref>。
