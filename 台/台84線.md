'''台84線'''為「'''東西向快速公路－北門玉井線'''」，是[[台灣|台灣]]東西向[[省道快速公路|省道快速公路]]之一。全線在[[台南市|台南市]]境內，西起台南市[[北門區|北門區]][[北門交流道|北門交流道]]接[[台61線|台61線]]、[[台17線|台17線]]，東至台南市[[玉井區|玉井區]]玉井端接[[台20線|台20線]]，通車長度41.8公里，全線於2014年9月27日正式通車<ref name="udn"/>。

[[File:半夜的東西向快速道路玉井線與縱貫公路.jpg|thumb]]]]

== 簡介 ==
台84線自[[北門區|北門區]]之台61線分出，沿線經[[學甲區|學甲區]]、[[下營區|下營區]]、[[麻豆區|麻豆區]]、[[官田區|官田區]]、[[大內區|大內區]]及[[玉井區|玉井區]]等七個區，全長41.8公里，為全臺12條東西向快速公路中長度第二長的路線，也是全台快速公路中，第三長的快速公路<ref>台74線與台84線封閉式道路長度皆為37.8公里</ref>，共設8處交流道，是台南市溪北地區聯絡西部濱海及東部丘陵的重要路網，此路是12條東西向快速公路之中三條具有速限100km/hr的快速公路之一<ref>{{cite web | url = http://www.thb.gov.tw/TM/Menus/Menu02/main0202/express_kilo/express_ew_speed.aspx | title = 東西向快速公路通車情形與速限統計表 | accessdate = 2010-10-06}}</ref>。同時也是第二條聯絡台61線、國道1號、國道3號皆是立體交叉的封閉式快速公路，但過走馬瀨隧道後即降為平面道路直到玉井。

渡頭交流道以西路段設有側車道，起點[[北門交流道|北門交流道]]，橋下配置東西雙向各5m寬地方連通道（部份路段為橋樑），銜接[[下營系統交流道|下營系統交流道]]，[https://www.youtube.com/watch?v=mYRO0B66V10&list=UUQ3Nns2cSSTIgmNd0IkV3ug 路線全長約13.729公里]，平面側車道預留穿越中山高涵洞空間（已在下營系統北端設置簡式涵洞一座）<ref>{{cite web | url = http://www.thb.gov.tw/TM/Menus/Discussing/ThbTake/DiscussDetail.aspx?key=10218 | title = 台84線下營系統 橋下側車道 北方涵洞}}</ref>。其中台19甲線至渡頭交流道之路段已納入縣道系統，編號分別為[[市道171號#甲線|市道171甲線]]、[[市道171號#乙線|市道171乙線]]。

台84線全線通車後可與[[台78線|台78線]]完成[http://goo.gl/maps/jFe2Q 雲嘉南地區高快速公路大環線系統（全封閉式道路）]，路線直通[[北門交流道|北門交流道]]、[[官田系統交流道|官田系統交流道]]、[[古坑系統交流道|古坑系統交流道]]、[[台西交流道|台西交流道]]。

== 公路特色 ==
*台84線是台灣第二條以高架橋立體交叉聯絡台61線、國道1號及國道3號的封閉式快速公路。
*台84線起點[[北門交流道|北門交流道]]是台灣第二座2條省道快速公路交叉的交流道。（連結[[台61線|台61線]]、[[台17線|台17線]]，0k匝道與1k匝道合併）
*台84線為東西向快速公路中橋梁長度第二長之路線，高架橋最長者為台74線。
*台84線為台灣快速公路唯一與國道3號交匯後往東之山區仍有路段者，原始路程1小時縮為15分鐘。

== 歷史年表 ==

* 1998年7月，[[渡頭交流道|渡頭交流道]]至[[玉井端|玉井端]]通車。

== 後續計畫 ==
===台84線增設橋下道路連通道新建工程===
*「市道171乙線台鐵橋下道路開闢工程」，道路貫通後為台灣首府大學、台南藝術大學、社區總爺藝文活動中心及西庄、官田工業區等重要聯結，可提升鄰近大學間及工業區間聯結便利性，機慢車能以橋下道路連通，增進農民進出便利。<ref>[https://www.youtube.com/watch?v=CUD9UlOcN8g&t=338s 市道171乙線完工影片]</ref>
*另外，「台84線增設橋下道路連通道新建工程（國1至麻豆大排）」，西起國道1號東迄麻豆大排，於台84線橋下兩側增設地方連通道，總長約3.2公里，可提供下營區大埤里、仁里里及西連里等務農民眾通行便利，提升區域產業運輸效能，麻豆大排另設有封閉式防汛道路可連絡南55線與下營南外環道。
*「台84線增設橋下道路連通道新建工程（麻豆大排至台19甲段）」，規劃中。
*上述工程完工後，北門交流道至渡頭交流道之橋下道路連通道總長為25.5公里。

===曾文溪景觀快速道路===
*升格前台南縣政府規劃從台84線西庄交流道，沿曾文溪北堤岸興建快速道路<ref>[https://www.youtube.com/watch?v=CUD9UlOcN8g&t=128s 曾文溪景觀快速道路]，目前可行路線[[南40線|南40線]]，可連絡台19線，台19甲線，台84線。</ref>，台17線以西路段與[[市道173號|市道173號]][[共線_(道路)|共構]]，目前防汛道路也是重要的「替代」快速道路。
*曾文溪北岸堤防道路：起點台84線西庄交流道，經麻豆堤防堤頂道路（可聯絡台19甲線）→中山高橋下道路截彎取直→西港堤防道路（區道南40線，可聯絡台19線）→七股堤防道路至市道173號（可聯絡台17線），全線20公里無紅綠燈管制。
