'''台北開放原始碼使用者社群'''（{{lang-en|Taipei Open Source Software User Group, TOSSUG}}），簡稱「土虱社群」<ref>{{cite news|title=帶著台灣開源社群向前邁進的土虱－TOSSUG |url=https://www.openfoundry.org/tw/foss-news/8396-tossug |language=zh-TW |author=李婉婷|publisher=OpenFoundry |date=2011-07-22 |accessdate=2018-11-17 }}</ref>，為台北的地方性資訊社群。主旨為要替國內開源技術的愛好者提供自由交流分享的環境，並舉辦定期聚會以便與參與者進行面對面的交流。

自2004年開始舉辦至今，每週二晚上在台北[[MozTW|摩茲工寮]]皆有聚會。聚會內容除了進行心得與經驗分享，也會交流電腦、軟體、程式、通訊等技術，並不定期舉行活動以促進成員之間的關係<ref>{{cite news|title=推動中文化Linux Live CD的第一人 |url=https://www.ithome.com.tw/node/42710 |language=zh-TW |author=楊啟倫|publisher=iThome |date=2007-03-30 |accessdate=2018-11-17 }}</ref><ref>{{cite news|title=臺灣9大知名開源社群聚會介紹 |url=https://www.ithome.com.tw/news/93609 |language=zh-TW |author= |publisher=iThome |date=2015-01-18 |accessdate=2018-11-17 }}</ref><ref>{{cite web | url=https://www.openfoundry.org/tw/resourcecatalog/Groups/Communities-Portal/Community-WebSite/| title=社群網站-OpenFoundry| date=2018-11-17}}</ref>。
