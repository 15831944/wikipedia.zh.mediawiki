'''台灣角川股份有限公司'''（''KADOKAWA TAIWAN CORPORATION''），簡稱為'''台角'''，是[[日本|日本]][[角川集團|角川集團]]在[[台灣|台灣]]設立的[[出版公司|出版公司]]，亦是其首間海外[[子公司|子公司]]。

== 歷史概要 ==
* 1999年4月22日，日本[[角川書店|角川書店]]、[[住友商事|住友商事]]、第一勸業銀行（現[[瑞穗銀行|瑞穗銀行]]）及台灣[[秋雨印刷|秋雨印刷]]（現[[秋雨創新|秋雨創新]]）等共同籌設「'''台灣國際角川書店股份有限公司'''」（KADOKAWA Media (Taiwan) Co., Ltd.）。
* 1999年9月，台灣角川第一本雜誌《Taipei Walker》（台北走路人）創刊。
* 2000年，台灣角川官方網站開站；同年開始出版小說。
* 2001年，台灣角川開始出版[[漫畫|漫畫]]。
* 2002年，台灣角川業務跨足企業刊物代編、產品型錄製作及政府單位觀光宣導刊物。
* 2003年，台灣角川發行第一本[[輕小說|輕小說]]《[[新羅德斯島戰記|新羅德斯島戰記]]》。
* 2009年7月31日至8月16日，台灣角川舉辦「10周年原創角色命名活動<ref>[http://www.kadokawa.com.tw/postlist.asp?f=47&t=5029 台灣角川10周年原創角色！命名活動開跑！]</ref>」，由日本插畫家[[泰|泰]]設計17歲、喜歡看台灣角川漫畫和輕小說的活力[[美少女|美少女]]，經台灣及香港地區讀者參加命名，最後命名為「角角」<ref>[http://www.kadokawa.com.tw/postlist.asp?f=47&t=5490 台灣角川十周年紀念原創角色命名：角角]</ref><ref>[http://www.kadokawa.com.tw/postlist.asp?f=47&t=5548 台灣角川10周年原創角色命名活動結果揭曉！]</ref>。角色設定為角角之妹的原創人物「川川」，則于2011年7月30日至7月31日舉辦的第18屆[[開拓動漫祭|開拓動漫祭]]（FF18）中首次登場<ref>[http://www.chosrepo.com/news/269/ff18%E8%87%BA%E7%81%A3%E8%A7%92%E5%B7%9D%E7%9C%8B%E6%9D%BF%E5%A8%98%E3%80%8C%E8%A7%92%E8%A7%92%EF%BC%86%E5%B7%9D%E5%B7%9D%E3%80%8D%E5%8F%AF%E6%84%9B%E7%99%BB%E5%A0%B4%EF%BC%81 FF18臺灣角川看板娘「角角＆川川」可愛登場！]</ref><ref>[http://www.f-2.com.tw/index.php?q=ff18 FF18活動回顧 | Frontier - 開拓動漫] {{webarchive|url=https://web.archive.org/web/20170622070422/http://www.f-2.com.tw/index.php?q=ff18 |date=2017-06-22 }}</ref>。
* 2013年8月22日，台灣角川公司登記名稱變更為「'''台灣角川股份有限公司'''」（KADOKAWA TAIWAN CORPORATION）<ref name="CmpyInfo"/>。
* 2014年[[台北國際動漫節|台北國際動漫節]]，台灣角川宣布代理多部角川系動畫並在[[中華電信MOD|中華電信MOD]]上架<ref>[https://www.kadokawa.com.tw/p9-news_detail.php?seriesa=8&id=889 台灣角川正式登入中華電信MOD，高畫質動畫《約會大作戰》2/28(五)上線]. 台灣角川. 2014-02-19.</ref>。
* 2015年8月，台灣角川與[[Book_Walker|Book Walker]]合資成立[[台灣漫讀|台灣漫讀]]，之後[[東立出版|東立出版]]、[[聯合發行|聯合發行]]和[[蓋亞文化|蓋亞文化]]亦有投資<ref>[https://www.bookwalker.com.tw/company/about 關於我們]. 台灣漫讀. </ref>。

== 產品 ==
在台灣以發行Walker系列雜誌及角川集團的[[輕小說|輕小說]]、[[漫畫|漫畫]]為主，日本{{Tsl|ja|リブレ出版|Libre出版}}（原{{Tsl|ja|ビブロス (出版社)|BiBLOS}}）也授權台灣角川發行中文版小說和漫畫（也就是BLOOM Series）。此外，台角書籍在[[港澳地區|港澳地區]]的代理與分銷業務現由[[香港角川|香港角川]]負責<ref>之前由[[一代匯集|一代匯集]]負責，現仍是台角書籍在[[港澳|港澳]]經銷商。</ref>。

=== 漫畫與畫集 ===
[[File:Kadokawa_Taiwan_booth_goods_window,_Taipei_International_Comics_&_Animation_Festival_20160211.jpg|缩略图]]
台灣角川將日本角川集團旗下出版社的漫畫與畫集混合代理販售。
<div class="notice metadata"  id="spoiler"><small>※Kadokawa BLOOM Series Comics以外的書系，全名需在前加上「Kadokawa Comics」。</small></div>
; Boy's Series
: 台灣角川的中文版漫畫，內容偏男性取向，主要為[[月刊少年Ace|月刊少年Ace]]作品。
; Girl's Series
: 台灣角川的中文版漫畫，內容偏女性取向，以[[月刊Asuka|月刊Asuka]]作品為中心。
; BLOOM Series Comics
: 台灣角川新發行的漫畫系列，為代理日本Libre出版作品之專門分類，含旗下女性BL向作品與女性一般向作品。
; Illustration
: 台灣角川發行的畫集與設定集。

=== 輕小說 ===
台灣角川出版日本角川集團的輕小說[[文庫|文庫]]—[[角川Sneaker文庫|Sneaker文庫]]和{{Tsl|ja|角川ルビー文庫|角川Ruby文庫|Ruby文庫}}、[[角川Beans文庫|Beans文庫]]（[[角川書店|角川書店]]）、[[富士見Fantasia文庫|Fantasia文庫]]和[[富士見Mystery文庫|Mystery文庫]]（[[富士見書房|富士見書房]]）、[[電擊文庫|電擊文庫]]（[[ASCII_Media_Works|ASCII Media Works]]）、[[Fami通文庫|Fami通文庫]]（[[Enterbrain|Enterbrain]]）、[[MF文庫J|MF文庫J]]（[[Media_Factory|Media Factory]]）混合發行。此外也發行BLOOM Series Novels小說（Libre出版）。
<div class="notice metadata"  id="spoiler"><small>※Kadokawa Novels Ruby Series以外的書系，全名需在前加上「Kadokawa」。</small></div>
; Fantastic Novels
: 內容比較偏向男性讀者喜好類型。
; Ruby Series
: 內容比較偏向女性讀者喜好類型，屬於[[BL|BL]]性質的小說。
; Midori Novels
: 內容比較偏向女性讀者喜好類型。
; BLOOM Series Novels
: 為代理Libre出版作品之專門分類。

== 舉辦賽事 ==
* [[台灣角川輕小說新人王|台灣角川輕小說新人王]] - 台灣角川舉辦，面向全球華人徵稿的中文原創輕小說比賽。前身為[[角川華文輕小說大賞|角川華文輕小說大賞]]。

== 其他 ==
* 台灣角川早期曾設立「鋼彈出版品」書系，作為專門出版[[機動戰士鋼彈系列|機動戰士鋼彈系列]]作品的產品分類。該分類已撤銷，相關作品併入旗下輕小說、漫畫的各子分類。
* [[角川漫畫新人賞|角川漫畫新人大賞]] - [[角川集團|KADOKAWA]]主辦的日本新人漫畫比賽，第9-12屆設有專屬海外獎項「[[角川漫画新人赏#奖金|台灣角川特別賞]]」。台灣角川曾作為除中國大陸外華人投稿和初步審查方，自第13屆起停止徵稿<ref>[https://www.kadokawa.com.tw/p9-news_detail.php?seriesa=8&id=1289 第12屆角川漫畫新人大賞審查結果公開！]. 台灣角川. 2015-05-06.</ref>。
