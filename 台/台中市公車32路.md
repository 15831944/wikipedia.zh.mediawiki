== 歷史 ==
*2015年6月15日：正式通車。<ref>[http://www.ibus.com.tw/news/%E4%BB%81%E5%8F%8B-32/ [仁友] 32路將於2015/06/15正式營運]</ref>
*2015年8月5日：增設「美滿社區站」。<ref>[http://www.ibus.com.tw/news/%E4%BB%81%E5%8F%8B32%E6%96%B0%E5%A2%9E%E7%AB%99%E9%BB%9E%E9%80%9A%E7%9F%A5/ [仁友]32新增站點通知]</ref>
*2016年3月21日：「五權公園路口」站更名為「五權公園路口（光大社區）」站。<ref>[http://www.ibus.com.tw/news/%E4%BB%81%E5%8F%8B32%E8%B7%AF%E3%80%8C%E4%BA%94%E6%AC%8A%E5%85%AC%E5%9C%92%E8%B7%AF%E5%8F%A3%E3%80%8D%E7%AB%99%E6%9B%B4%E5%90%8D%E9%80%9A%E7%9F%A5/[仁友]32路「五權公園路口」站更名通知]</ref>
*2016年4月28日：增設「東英二街長福路口」。<ref>[http://www.ibus.com.tw/news/%E4%BB%81%E5%8F%8B32%E8%B7%AF-%E6%96%B0%E5%A2%9E%E7%AB%99%E9%BB%9E%E5%8F%8A%E4%BF%AE%E6%94%B9%E6%99%82%E5%88%BB%E9%80%9A%E7%9F%A5/ [仁友]32路 新增站點及修改時刻通知]</ref>

== 路線基本資料 ==
本路線途經大鵬國小、[[臺中市私立曉明女子高級中學|曉明女中]]、[[中國醫藥大學|中國醫藥大學]]、五權臺灣大道口、[[國立臺中教育大學|臺中教育大學]]、臺中醫院、[[臺中車站|臺中火車站]]、明功堂醫院、[[臺中市立育英國民中學|育英國中]]等站點。

== 收費 ==
依里程計費；刷電子票證十公里免費
