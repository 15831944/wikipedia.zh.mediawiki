'''臺灣藍鵲'''（[[學名|學名]]：''{{lang|la|Urocissa caerulea}}''），又稱'''臺灣暗藍鵲'''、'''紅嘴山鵲'''、'''長尾山娘'''（[[臺灣閩南語羅馬字拼音方案|閩南語]]：{{Unicode|Tn̂g-bué Suann-niû}}）或'''長尾陣仔'''，為[[臺灣|臺灣]]特有種鳥類。臺灣從[[臺灣清治時期|清領時期]]開始就有文獻紀載臺灣藍鵲的資料。1862年，鳥畫家[[约翰·古尔德|约翰·古尔德]]根據英國博物學家[[郇和|郇和]]寄來的臺灣鳥類標本發表了一篇文章，命名並詳述16種新發現的台灣物種，其中就包含臺灣藍鵲。

臺灣藍鵲主要分布1,500公尺以下的森林裡，除繁殖季外多成群行動，性情兇悍，特徵是兩根長尾羽。牠們的繁殖季在3月至7月，每年生兩窩，每窩大約有3至8顆蛋。[[國際鳥盟|國際鳥盟]]將臺灣藍鵲列為無須採取保育行動的鳥種，而中華民國[[行政院農業委員會|行政院農業委員會]]則將臺灣藍鵲列為保育類動物。

== 發現 ==
根據文獻紀載，在[[臺灣清治時期|清領時期]]，人們將臺灣藍鵲稱為「長尾山娘」<ref name="台灣特有種寫真"/>{{rp|106}}。當時的漢人農夫也稱牠為「長尾陣仔」<ref name="台灣特有種寫真"/>{{rp|106}}。《[[臺灣通史|臺灣通史]]》中記載其「翠翼朱喙，光彩照人」<ref name="經典雜誌"/>。1862年，英國博物學家兼英國在臺領事[[郇和|郇和]]因病回英國倫敦休養，將在臺灣採集的一批標本交給鳥畫家[[约翰·古尔德|约翰·古尔德]]<ref name="台灣鳥類發現史"/>{{rp|71}}。古爾德以此發表了一篇文章，命名並詳述16種新發現的台灣物種，其中就包含臺灣藍鵲（[[學名|學名]]：''{{lang|la|Urocissa caerulea}}''）<ref name="台灣鳥類發現史"/>{{rp|71}}<ref name="Gould"/>。郇和如此描述他發現臺灣藍鵲的經過<ref name="台灣鳥類發現史"/>{{rp|380-381}}：

<blockquote>甫抵淡水不久，我僱用的獵人自內地帶回兩根美麗的尾羽，而身軀則因天熱肉易腐壞而被他們先吃了。他們稱呼這種鳥為「Tung-bay Swanniun」（即長尾山娘，{{lang-nan-tw|tn̂g-bé-suann-niû}}）。我透過這兩根尾羽的亮藍色羽末端的白點判斷這一定是隻[[藍鵲屬|藍鵲屬]]（{{lang|la|學名：''Urocissa''}}）的新種。興奮之餘，我重賞徵求標本。[…]標本完全確定了我的假設：這是美麗的新物種<ref name="台灣鳥類發現史"/>{{rp|380-381}}。</blockquote>

當初郇和採集的臺灣藍鵲[[正模標本|正模標本]]現收藏在[[英國|英國]][[倫敦|倫敦]]的[[自然史博物館_(倫敦)|自然史博物館]]<ref name="台灣鳥類誌" />{{rp|29}}。

== 辭源 ==
臺灣藍鵲的學名「''Urocissa caerulea''」中，''Urocissa''指的是「有長尾的鵲」，而 ''caerulea'' 則是「蔚藍」的意思<ref name="台灣鳥類誌" />{{rp|29}}。臺灣藍鵲與中國大陸的兩種鳥類[[黃嘴藍鵲|黃嘴藍鵲]]（學名：''{{lang|la|Urocissa flavirostris}}''）與[[紅嘴藍鵲|紅嘴藍鵲]]（學名：''{{lang|la|Urocissa erythrorhyncha}}''）是近親<ref name="台灣特有種寫真" />{{rp|108}}<ref name="HBW" />。臺灣藍鵲又稱「臺灣暗藍鵲」或「紅嘴山鵲」<ref name="台灣鳥類誌" />{{rp|29}}。

== 分布 ==
臺灣藍鵲是[[臺灣特有種列表|臺灣特有種]]<ref name="台灣鳥類誌"/>{{rp|29}}。臺灣藍鵲主要分布在海拔1,000公尺以下的地區，而1,800公尺以下的中、低海拔[[照葉林|闊葉林]]或[[次生林|次生林]]也可以看見牠的蹤跡<ref name="台灣特有種寫真"/>{{rp|106}}<ref name="台灣野鳥圖譜"/>{{rp|196}}<ref name="台灣動物新聞網 (2014-05-29)"/>。雖然如此，若山區環境開發過度或林相不夠豐富，往往也不會有臺灣藍鵲棲息<ref name="台灣特有種寫真"/>{{rp|106}}。近年來，臺灣藍鵲已逐漸適應都市，如[[臺北市|臺北市]]靠近山邊的地方就有臺灣藍鵲出沒<ref name="自由時報 (2017-06-14)"/>。

== 形態描述 ==
[[File:Толстоклювая_лазоревая_сорока.jpg|250px]]
臺灣藍鵲成鳥身長63至68公分<ref name="HBW"/>，其中尾羽長34至42公分<ref name="HBW"/>，翅膀長18至21公分<ref name="台灣特有種寫真"/>{{rp|106}}。臺灣藍鵲的體重約254至260公克<ref name="HBW"/>。臺灣藍鵲成鳥的嘴和腳是紅色，[[虹膜|虹膜]]呈金黃色，頭至頸、胸部是黑色，翅膀末端為白色，身體其餘部分皆為藍色<ref name="台灣特有種寫真"/>{{rp|106}}<ref name="台灣鳥類誌"/>{{rp|29}}。尾羽部分，臺灣藍鵲成鳥的兩根中央尾羽較長且末端為白色，其他尾羽則是中段黑末段白<ref name="台灣鳥類誌"/>{{rp|29}}。

== 行為生物學 ==
臺灣藍鵲多成群行動，一群約3至12隻，多由家庭成員組成<ref name="台灣鳥類誌"/>{{rp|30}}。牠們的飛行方式採直線飛行，排列有序<ref name="台灣野鳥圖譜"/>{{rp|196}}。臺灣藍鵲的叫聲是聒噪的「嘎、嘎、嘎」聲，在群體中則也會以「地笛——地笛——」或「嘎嘎——穗」等叫聲互相叫喚<ref name="台灣鳥類誌"/>{{rp|30}}，叫聲變化多端<ref name="台灣特有種寫真"/>{{rp|109}}。臺灣藍鵲性情兇悍，領域性強，會攻擊入侵者，也會集體禦敵<ref name="台灣特有種寫真"/>{{rp|106}}<ref name="台灣鳥類誌"/>{{rp|30}}。牠們可多年維持相同的領域<ref name="台灣鳥類誌"/>{{rp|30}}。

=== 覓食 ===
臺灣藍鵲屬雜食性，食物包含植物的果實（如[[漿果|漿果]]、木瓜、香蕉）或根莖、巢中的幼鳥或蛋、昆蟲、蜥蜴、蛇類、兩棲類和小型哺乳類（如鼠類）等，甚至也有人類的廚餘和動物屍體<ref name="台灣野鳥圖譜"/>{{rp|196}}<ref name="蘋果日報 (2007-07-20)"/>。牠們也會食用自己夭折的幼鳥<ref name="台灣鳥類誌"/>{{rp|30}}。臺灣藍鵲會儲存及分享食物<ref name="台灣特有種寫真"/>{{rp|114-117}}<ref name="台灣鳥類誌"/>{{rp|30}}。

=== 繁殖 ===
臺灣藍鵲的繁殖季在3月至9月，採成對繁殖<ref name="台灣鳥類誌"/>{{rp|30}}<ref name="自由時報 (2017-06-14)"/>。牠們會用叫聲搭配飛舞動作來[[求偶|求偶]]<ref name="台灣特有種寫真"/>{{rp|109}}。在繁殖季，臺灣藍鵲不再成群結隊，變成兩兩一對<ref name="台灣特有種寫真"/>{{rp|109}}。據學者判斷，臺灣藍鵲選定伴侶後就不會隨便更換<ref name="台灣特有種寫真"/>{{rp|109}}。臺灣藍鵲傾向在稀疏的樹林築巢，樹種不拘，夠隱密即可<ref name="台灣特有種寫真"/>{{rp|109}}。牠們多半選擇巢樹約4/5處築巢，高度約10公尺左右<ref name="台灣特有種寫真"/>{{rp|109}}。選好巢位後，雌、雄鳥會一同到地面銜小樹枝當巢的材料<ref name="台灣特有種寫真"/>{{rp|109}}。牠們大約會花上一星期築巢，搭出來的巢像個淺碗<ref name="台灣特有種寫真"/>{{rp|109}}。

臺灣藍鵲的子代存活率不高，因此牠們每年生兩窩，每窩大約有3至8顆蛋，卵殼呈淡青色且帶有褐斑<ref name="台灣特有種寫真"/>{{rp|108-110}}<ref name="台灣鳥類誌"/>{{rp|31}}。臺灣藍鵲的蛋約需17天孵化<ref name="台灣鳥類誌"/>{{rp|31}}。牠們的護巢心很強，會猛烈抵抗入侵者<ref name="台灣特有種寫真"/>{{rp|113}}，因此常有民眾受到影響，對此動物保護單位呼籲民眾切莫傷害臺灣藍鵲，表示牠們多是以翅膀揮擊、驅離，幾乎不會傷害到民眾<ref name="台灣動物新聞網 (2014-05-29)"/>。臺灣藍鵲的幼鳥孵出後約22天會離巢<ref name="台灣鳥類誌"/>{{rp|31}}。離巢後的幼鳥會留在鳥群中與成鳥一同生活<ref name="台灣鳥類誌"/>{{rp|31}}。同巢幼鳥會藉打鬥決定彼此在群體中的地位<ref name="台灣鳥類誌"/>{{rp|31}}。此外，臺灣藍鵲上一季孵出的幼鳥會留在巢邊協助親鳥尋找巢材、餵食雛鳥或共同禦敵，屬於特殊的「合作育雛」行為<ref name="台灣鳥類誌"/>{{rp|30}}<ref name="台灣野鳥圖譜"/>{{rp|196}}。

== 生存狀況 ==
[[國際自然保護聯盟瀕危物種紅色名錄|國際自然保護聯盟瀕危物種紅色名錄]]中，[[國際鳥盟|國際鳥盟]]將臺灣藍鵲列為無須採取保育行動的鳥種<ref name="IUCN"/>。臺灣[[行政院農業委員會|行政院農業委員會]]將臺灣藍鵲分為保育等級Ⅲ（意指其他應予保護之野生動物）的動物<ref name="台灣鳥類誌"/>{{rp|651-652}}，任何捕捉或飼養行為皆違反相關法律<ref name="台灣鳥類誌"/>{{rp|30}}<ref name="台灣野鳥圖譜"/>{{rp|196}}<ref name="台灣動物新聞網 (2016-12-12)"/>。早年，臺灣藍鵲遭人大量獵捕，後來因保護妥善，數量有增加<ref name="台灣鳥類誌"/>{{rp|31}}，但近年來，臺灣藍鵲在被網路票選為臺灣非官方國鳥（見[[#社會文化|社會文化]]一段）後，因知名度提升而遭人大量獵捕，棲地也被外來種[[紅嘴藍鵲|紅嘴藍鵲]]入侵<ref name="中時電子報 (2016-05-02)"/>，甚至還有臺灣藍鵲與紅嘴藍鵲同巢育雛的現象<ref name="蘋果日報 (2007-07-20)"/>。

== 社會文化 ==
臺灣藍鵲是臺灣原住民族[[鄒族|鄒族]]的神鳥<ref name="鄒族-1"/><ref name="鄒族-2"/>。在該族的神話中，很久以前鄒族因大洪水瀕臨滅族，臺灣藍鵲犧牲自己帶回火種挽救鄒族<ref name="鄒族-1"/><ref name="鄒族-2"/>。2007年，臺灣永續生態協會、臺灣國際觀鳥協會與立委推動為期4個月的非官方網路「國鳥選拔」，逾百萬人投票。在[[帝雉|帝雉]]、[[藍腹鷴|藍腹鷴]]、[[黃山雀|黃山雀]]和臺灣藍鵲4種臺灣特有鳥類中，臺灣藍鵲贏得初選<ref name="中時電子報 (2016-05-02)"/><ref name="國鳥選拔"/><ref name="國鳥選拔2"/>。而[[臺北市|臺北市]]、[[雲林縣|雲林縣]]的市鳥、縣鳥也是臺灣藍鵲<ref name="TVBS (2017-02-10)"/>。

2015年11月7日，時任[[中華民國總統|中華民國總統]][[馬英九|馬英九]]在[[兩岸領導人會面|馬習會]]上將一件臺灣藍鵲的手工瓷器贈與[[中國共產黨中央委員會總書記|中國共產黨中央委員會總書記]][[習近平|習近平]]當做見面禮。[[中華民國總統府|臺灣總統府]]表示，會選擇送臺灣藍鵲瓷器是為了凸顯臺灣特色<ref name="馬習會"/>。2016年，中華航空公司在其中一架新進的[[空中巴士A350_XWB|空中巴士A350]]上畫上臺灣藍鵲，並將該機命名為「臺灣藍鵲號」<ref name="A350-1"/><ref name="A350-2"/>。[[中華民國空軍|中華民國空軍]]的新一代高教機「[[T-5教練機|T-5藍鵲高級噴射教練機]]」也以臺灣藍鵲命名，取自臺灣藍鵲捍衛家園且團結的個性<ref name="T-5教練機-1"/><ref name="T-5教練機-2"/>。2018年，[[中華民國電子競技運動協會|中華民國電子競技運動協會]]（CTeSA）公開了以臺灣藍鵲為主體的新標識，意象源於臺灣藍鵲「團結、敏捷、堅毅強悍」的特性<ref name="CTeSA"/>。
