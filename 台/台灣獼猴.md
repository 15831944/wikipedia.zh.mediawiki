'''臺灣獼猴'''，又稱黑肢猴，[[臺灣動物特有種列表|台灣特有種]]，珍稀野生動物，屬[[舊世界猴|舊世界猴]]。也是除了[[人類|人類]]外台灣唯一的[[靈長類動物|靈長類動物]]，和[[日本獼猴|日本獼猴]]、[[邱獼猴|邱獼猴]]、[[普通獼猴|普通獼猴]]有血緣關係。

== 分布 ==
[[臺灣|臺灣]]的原生種<ref>{{cite web | url=http://www.ntsec.gov.tw/FileAtt.ashx?id=3136 | title=科學話題：臺灣生物多樣性中的「猴」| publisher=[[國立臺灣科學教育館|國立臺灣科學教育館]] | work=科學研習No.55-09 | date=2016-09-01 | accessdate=2016-09-18 | pages=76頁 | author1=陳文德}}[http://www.ntsec.gov.tw/USER/Article.aspx?a=3246 No.55-09]{{rp|65-66}}</ref>，分布於由海平面至海拔三千公尺以下的地區。在日本則為外來種，分布於[[伊豆大島|伊豆大島]]（[[伊豆群島|伊豆群島]]），[[大根島|大根島]]（[[静岡縣|静岡縣]][[南伊豆町|南伊豆町]]），[[和歌山縣|和歌山縣]]北部。從1940年起陸陸續續從動物園等觀光設施逃脫，成為日本的外來種。

== 特徵 ==
臺灣獼猴的前肢比後肢短，各肢具有五趾。頭圓，臉扁，額裸出，顏面呈淡紫色，或深或淡，隨個體的不同而有差異，所以[[台灣原住民|台灣原住民]]將其分為白臉與紅臉，臉頰上並長有粗雜的長鬚，尾粗狀多毛，全身被厚而軟的毛，冬季呈棕灰色，夏季呈棕綠色，四肢較黑故有黑肢猿之稱。股間有顯著的紅棕色的大斑。一般而言，雌猴體型較雄猴小，顏色也較淡。

== 分類 ==
種小名cyclopis來自[[希臘神話|希臘神話]]的[[獨眼巨人|獨眼巨人]]。

== 習性 ==
台灣獼猴屬於晝行性活動且群體生活的動物。分布地區遍及3000公尺以下之低至高海拔，[[彰化縣|彰化縣]][[二水鄉|二水鄉]]設有[[二水台灣獼猴生態教育館|二水台灣獼猴生態教育館]](原名二水彌猴自然保護區)。另外，在[[新北市|新北]][[烏來區#行政區|福山]]、[[恆春鎮|恆春]][[墾丁|墾丁]]有台灣獼猴做為過夜地域。

根據記錄，台灣獼猴食用的植物包含85科300種，以果實為主，其他還有[[昆蟲|昆蟲]]、[[甲殼類|甲殼類]]和[[軟體動物|軟體動物]]等。對於食用植物的部位，會因季節月份和乾溼季變化而有所不同，以十至五十隻一群。

台灣獼猴雖然是[[臺灣保育物種列表|保育類動物]]，原有的天敵為台灣雲豹及大型猛禽，但台灣雲豹早已絕跡大型猛禽也漸稀少，加上台灣獼猴適應力強，在立法保育之下族群數量大幅回升，在高雄、屏東等地區的台灣獼猴，因棲息地與人類接壤而不會怕人，經常發生搶奪遊客食物、入侵民宅房舍取食以及破壞農作物等事件，造成居民及遊客困擾。因受歸類為保育類動物，目前尚未有統一且有效的解決方針。

人類於其居住範圍應避免將食物外露，即可避免人猴衝突。唯露食被取事件層出不窮，積怨已久，以至使用毒餌，登山杖，彈弓、捕獸夾等凶器等事件發生，致獼猴死傷。目前於高雄柴山仍可見到彈弓與BB槍等制猴器具以及使用民眾。2015年11月，僅兩周間於高雄地區便有十來隻死傷之獼猴，皆明顯外傷。{{來源請求}}

2014年行政院農委會依據「野生動物保育法」第21條的緊急條款內容，認為特定狀況下獵殺獼猴具法律正當性；然而，此條款之適用，僅限於農民在生命財產受到緊急威脅之情況下，主張「[[正當防衛|正當防衛]]」獵殺野生動物，以保障生命、財產作物安全。但衡量法律保護利益與野生動物保育，其限定以「人道方式」為之，在最短時間內給予最小痛苦。<ref>{{cite news|title=開放「人道方式」獵殺台灣獼猴　動保團體批：國際笑話|url=http://www.ettoday.net/news/20150104/446785.htm|accessdate=2015-09-24|agency=Ettoday東森新聞雲|date=2015-01-04}}</ref>。

== 疾病 ==
[[美国食品药品监督管理局|-{zh-cn:美国食品药品监督管理局;zh-tw:美國食品藥物檢驗局;}-]]（Food and Drug Administration;[[FDA|FDA]]）於2013年8月發表台灣獼猴具有「{{le|猴泡沫病毒|Simian foamy virus}}」（Simian Foamy Virus, SFVmcy-2），是[[逆转录病毒|-{zh-tw:反轉錄病毒;zh-cn:逆转录病毒;zh-mo:逆𨍭錄病毒;}-]]（retrovirus）的一種 。<ref> （Galvin et al., 2013）[http://jvi.asm.org/content/87/15/8792.abstract]</ref>

台灣獼猴亦有第四類法定傳染病「{{le|疱疹B病毒|Herpes B virus}}」（Herpes B virus,正式學名{{sname|Cercopithecine herpesvirus 1(CHV-1)}}，猴疱疹病毒）, [[玉山|玉山]][[塔塔加|塔塔加]]首起猴群主動攻擊, 抓咬傷10遊客。疱疹B病毒感染症是一種會造成中樞神經系統疾病的人畜共通傳染病，主要是經由罹患猴疱疹的猿猴抓咬傷而感染，人若被感染的猿猴抓咬傷，通常在3天至3星期的潛伏期後，會出現急性發燒、頭痛、水泡病灶、淋巴性腦脊髓液內細胞增多和不同型態的神經症狀等。人類罕見疱疹B病毒感染個案，目前全球約僅有40名報告病例。疱疹B病毒一旦感染發病，死亡率非常高，病患如未接受妥適醫療，死亡率超過70%，存活者常會伴隨有嚴重的後遺症。<ref>[http://www.cdc.gov.tw/professional/knowdisease.aspx?treeid=beac9c103df952c4&nowtreeid=3c97216a063e94e8&id=31565F8876A6DAA6&did=683]</ref><ref>{{cite web|url= http://www.baphiq.gov.tw/public/Attachment/841516392271.pdf |title= 第九章 疱疹 B 病毒感染症 | publisher=[[國立台灣大學|國立台灣大學]]獸醫學研究所|pages=5pages|author=何勝裕}}</ref>
