'''台中液化天然氣接收站'''，為[[台灣中油|中油]]於2004年9月開始籌建、2009年7月正式商轉之[[天然氣|天然氣]]接收站<ref name=cpc>{{cite web|language=zh-tw|url=http://new.cpc.com.tw/division/lngb/about.aspx/tc/lpgb/contact/images/information-text.aspx?id=24|archiveurl=https://web.archive.org/web/20171211131428/http://new.cpc.com.tw/division/lngb/about.aspx/tc/lpgb/contact/images/information-text.aspx?id=24|archivedate=2017年12月11日|title=台中液化天然氣接收站簡介|author=|publisher=[[台灣中油|中油]]|date=|accessdate=2017-12-11|deadurl=yes|df=}}</ref>，地點位於[[台灣|台灣]][[台中港|台中港]]。此站主要功能除供應[[台電公司|台電公司]]旗下之發電廠，如[[大潭發電廠|大潭發電廠]]。同時亦供應部分民生用天然氣。

== 簡介 ==
中油於2003年7月取得[[大潭發電廠|大潭發電廠]]的供氣合約，且位於高雄的[[第一天然氣接收站|第一天然氣接收站]]距離台灣中北部較遠，因此開始籌建本站。此站興建時，主要是接收來自[[卡達|卡達]]的天然氣，預計供氣效率為400[[公噸|噸]]／小時。<ref>{{cite news|language=zh-tw|url= http://www.cna.com.tw/postwrite/Detail/28160.aspx#.Wi6FBlWWaUk |archiveurl = https://web.archive.org/web/20171211131518/http://www.cna.com.tw/postwrite/Detail/28160.aspx#.Wi6FBlWWaUk|title=台中液化天然氣接收站成功試俥邁入新里程 |archivedate = 2017-12-11 |author=|publisher=[[中央通訊社|中央通訊社]]|date=2009-04-28|accessdate=2017-12-11|deadurl=no}}</ref>其工程包括3座16萬[[公秉|公秉]]液化天然氣儲槽及卸收、氣化設施，同時也包括台中經[[苗栗|苗栗]][[通霄|通霄]]至[[大潭_(桃園市)|大潭]]約135公里之36吋海底輸氣管線，總投資金額約新台幣315億元。中油表示，海底管線興建時氣候不佳，影響施工進度，且台中港一帶地質複雜，因此施工延宕到2009年才完工。

== 擴建 ==
台電因不滿中油供氣百分之60都用於供應發電，被中油搶走市場，又向經濟部請求興建[[第三天然氣接收站|第三天然氣接收站]]失敗。因此於2017年2月宣布與中油拆夥，也成功爭取到第四天然氣接收站的興建權利。<ref>{{cite news|language=zh-tw|url=http://news.ltn.com.tw:80/news/business/paper/1076889|archiveurl=https://web.archive.org/web/20170513073605/http://news.ltn.com.tw/news/business/paper/1076889|archivedate=2017-05-13|title=與中油拆夥 台電自建LNG接收站|author=|publisher=[[自由時報|自由時報]]|date=2017-02-10|accessdate=2017-12-11|deadurl=no}}</ref>所以中油為了提升競爭力，投資新台幣237億元，欲興建3座16萬公秉地上儲槽及兩條輸氣陸管。不過2017年5月28日中油董事長[[陳金德|陳金德]]認為，現有台中天然氣接收站可直接調節、擴建，加上中油經營經驗久及國外盟友多，台電無經驗，台電優勢相對不高。<ref>{{cite news|language=zh-tw|url=https://money.udn.com/money/story/5641/2490180|archiveurl=https://web.archive.org/web/20170903053032/https://money.udn.com/money/story/5641/2490180|archivedate=2017-09-03|title=第五天然氣接收站 陳金德：中油具優勢|author=|publisher=[[經濟日報_(臺灣)|經濟日報]]|date=|accessdate=2017-12-11|deadurl=yes}}</ref>又為了配合[[蔡英文政府|蔡英文政府]][[非核家園|非核家園]]目標，經濟部近期規劃能源轉型計畫，2025年天然氣發電佔比要達50%、燃煤發電30%以及再生能源20%，以填補核電缺口，且環保署原訂5月上旬初審桃園觀塘工業區環差案卻因臨時停止，恐延宕第三天然氣接收站工程。而此天然氣接收站也會顯得更重要。

== 設備 ==
現有3座容積為16萬公秉的地上型全覆式液化天然氣儲槽，每座儲槽均設置一級輸出泵、加熱系統、起重設施、密度及液位監視系統、地震監測、儀控系統、[[電力系統|電力系統]]、超壓及低壓保護系統、沉陷及傾斜監測、消防系統及其附屬設施。以及為引取海水供製程使用，一座獨立的海水進水渠道，並設置8台海水[[泵浦|泵浦]]，及4台海水消防泵浦的泵站，進水渠道總設計進水量為32,000噸／小時。<ref>{{cite web|language=zh-tw|url=http://www.ezsino.org/index.php?mod=journal&action=show&cid=14&pid=65&id=1386|title=中油第二座進口液化天然氣接收站―台中液化天然氣廠|author=陳進興|publisher=Ezfree華人網|date=|accessdate=2017-12-11|archiveurl=https://archive.today/20171211133557/http://www.ezsino.org/index.php?mod=journal&action=show&cid=14&pid=65&id=1386|archivedate=2017-12-11|deadurl=no}}</ref>

值得一提的是，因為液化天然氣變成氣態時需要引取大量的海水做為熱交換之媒介，而使用完畢之冷排水非常乾淨，且在炎熱的夏天亦可以保持24到26[[攝氏度|攝氏度]]的冷度，因此深受[[養殖漁業|養殖漁業]]戶歡迎。<ref name=cpc/>
