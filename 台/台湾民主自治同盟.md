'''臺灣民主自治同盟'''，简称'''臺盟'''，是[[中华人民共和国|中华人民共和国]]排名第八的[[民主党派|民主党派]]。同时，该党也是现时[[中华人民共和国|中华人民共和国]][[全国人大常委会|全国人大常委会]]的第八大党。

臺盟创立于1947年。其早期盟员多為過去[[台灣日治時期|台灣日治時期]]的[[台湾共产党|台湾共产党]]成員。目前，其盟员主要来自居住在中国大陆的臺灣籍人士，為中国大陆對臺[[統戰|統戰]]的組織之一<ref>[https://nccur.lib.nccu.edu.tw/bitstream/140.119/49841/1/801101.pdf 臺灣民主自治同盟案件研究]，第37頁，曾培強，2010-07</ref><ref>[https://books.google.com.tw/books?id=dpKU-UB6i58C&pg=PA113 兩岸真相密碼: 中共對台宣傳的政策、作為與途徑]，第113頁，杜聖聰，秀威出版，2008-08-01</ref><ref>[https://www.chinatimes.com/newspapers/20150314000881-260301 習對臺4堅定不移納政協決議] {{webarchive|url=https://web.archive.org/web/20151216175118/http://www.chinatimes.com/newspapers/20150314000881-260301 |date=2015-12-16 }}，中時電子報，2015-03-14</ref>。截至2017年，臺盟在19个省、直辖市建立组织，共有盟員3000人。

== 歷史 ==
=== 成立背景 ===
1947年[[二二八事件|二二八事件]]後，一部分先前在[[台灣日治時期|台灣日治時期]]的前[[台灣共產黨|台灣共產黨]]人士逃往[[中國大陸|中國大陸]]、[[香港|香港]]。其中統獨立場皆有。

臺灣民主自治同盟1947年11月12日（孫中山诞辰纪念日）由[[謝雪紅|謝雪紅]]、[[楊克煌|楊克煌]]、[[蘇新|蘇新]]等人在香港成立<ref>[https://books.google.com.tw/books?id=qiJxCgAAQBAJ&pg=PA17 1949．國共內戰與台灣]，第17頁，曾健民，聯經出版事業公司，2009-11-18</ref>，但當時偽稱在[[臺北市|臺北]]，以躲避[[中華民國政府|中華民國政府]]國安人員的耳目。1948年臺盟宣布拥护[[中国共产党|中国共产党]]领导的「[[新民主主义革命|新民主主义革命]]」，成員陸續加入中國共產黨，而成為中共分部。同年5、6月間成立「香港支部」——事實是總部的伪装，其主要是為了轉移中華民國國安人員注意力的障眼法。至於[[中國共產黨台灣省工作委員會|中國共產黨台灣省工作委員會]]在台灣往往以「台盟」為名義對外吸收成員，其實是中國共產黨台灣省工作委員會的外圍組織。

7月20日宣布成立總部領導機構，由謝雪紅擔任台盟主席。

=== 在台發展 ===
1949年7月10日，台盟由香港輾轉得到一份《臺灣恥政三週年告同胞書》並予以複製在台灣發送。但7月14日即遭舉發，台灣省[[警備總司令部|警備總司令部]]隨即逮捕印刷成員[[廖學銳|廖學銳]]、[[鄭慶龍|鄭慶龍]]，這就是[[白色恐怖|白色恐怖]]中的[[廖學銳案|廖學銳案]]。過了2天，又有陳東海、郭錕銘、張如松等人逮捕陸續到案。但僅有廖、鄭兩人承認參加台盟，其餘3人一概否認。

台灣省警備總司令部調查此案後，隨即起訴5人，並以「『台灣民主自治同盟』為共產黨外圍組織，共產黨在北平組織政權時，該同盟也公然參與，其為叛亂組織，殆無疑義」為由，將廖、鄭兩各判處有期徒刑5年，陳東海等3人則因確未參加該同盟，而獲判無罪開釋。

=== 移至中國大陸 ===
[[File:台湾民主自治同盟中央委员会_20100107.jpg|thumb]]
1949年9月，台盟参加[[中國共產黨|中國共產黨]]召開的[[中国人民政治协商会议第一届全体会议|中國人民政治协商会议第一届全体会议]]，参与制定《[[中国人民政治协商会议共同纲领|中国人民政治协商会议共同纲领]]》。1955年，台盟中央委员会迁至[[北京|北京]]。 

當[[臺灣省保安司令部|臺灣省保安司令部]]獲悉台盟已遷至北京後，下令嚴辦[[廖學銳|廖學銳]]案，又再牽扯出30餘人，廖學銳與其他11名涉案者最後被槍決。

1957年[[中國大陸|中國大陸]][[反右运动|反右运动]]中，台盟创始人[[谢雪红|谢雪红]]被打成[[右派|右派]]，于1970年[[文革|文革]]期间被迫害。

== 運作 ==
中华人民共和国成立后，台盟宣稱政治主張為「团结推动其盟员参加[[社会主义|社会主义]]建设，并争取[[中国统一|中国统一]]」。 

1987年11月，台盟第四次全盟大会通过的台盟新章程，表明台盟对台湾的立场：
# 台湾是中华人民共和国神圣领土的一部分，台湾人民同大陆人民是骨肉同胞，完成祖国统一大业是包括台湾人民在内的全国各族人民的神圣职责。
# 反对任何形式的[[台湾独立|台湾独立]]，反对任何外国侵犯台湾。
# 支持台湾人民争取民主权利，反对[[国民党当局|国民党当局]]对台湾人民的压迫。
# 致力于按「[[一国两制|一国两制]]」原则争取用和平方式实现台湾和大陆的统一，使台湾作为[[特别行政区|特别行政区]]享有高度自治权。

== 組織 ==
台盟最高機構為全盟代表大會，五年召開一次，聽取現任中央委員會的報告，討論修改方針與章程，並選舉新任的中央委員會。中央委員會由全盟代表大會組成，任期五年，主席、副主席與常務委員由選舉產生，組成中央常務委員會。秘書長由主席提名，中央常務委員會任命。選舉後，中央常務委員會負責全盟事務，主席與副主席負責中央常務委員會事務。

台盟在北京、天津、上海、福建、廣東、吉林、遼寧、陝西、湖北、浙江、雲南、海南等十二省與直轄市設有地方委員會，之下再設置省轄市與直轄市各區的委員會。地方委員會由盟員大會或盟員代表大會負責事務，五年召開一次，程序與全盟代表大會大致相同。地方委員會之下再分成支部（五人以上）或小組（三人以上）。
