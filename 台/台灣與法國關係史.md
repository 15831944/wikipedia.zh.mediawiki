'''台灣與法國關係史'''是指[[台灣|台灣]]有[[信史|信史]]以來至今與[[法國|法國]]雙方在歷史上不同階段的關係。随着[[臺灣戰後時期|1949年中华民国政府迁往台湾]]后，实际上與其国家外交关系重疊。

==清治時期==

{{主條目|臺灣清治時期|西仔反}}
在1884－1885年[[中法戰爭|法清戰爭]]後期，[[法國海軍|法國海軍]]戰場轉闢中國東南沿海一帶，自1884年8月起（清[[光緒|光緒]]九年七月）法軍正式入侵當時由[[清朝|清帝國]]統至下的台灣北部（[[雞籠|雞籠]]、[[淡水區|滬尾]]），一度封攻佔基隆港佔據煤礦，但卻於[[淡水之役|滬尾之役]]受挫清軍（[[劉銘傳|劉銘傳]]領導的[[淮軍|淮軍]]），爾後，法軍指揮孤拔鑒於台灣北部久攻不下，便改為鎖台灣沿海的戰略，於是將目標轉往佔據台灣西部的澎湖群島，做為與清廷交涉的中繼和補給據點。

1885年3月（光緒十一年二月），[[法國軍事|法軍]]發動[[澎湖之役_(1885年)|澎湖之役]]，攻占[[澎湖群島|澎湖群島]]。6月9日法使與清廷簽訂〈[[中法新約|中法新約]]〉，法軍主帥[[孤拔|孤拔]]因水土不服於6月11日病逝澎湖媽宮澳，6月13日法艦接獲和談通知，陸陸續續開始撤軍，直到8月4日法軍才完全撤離澎湖。

==日治時期==
{{主條目|臺灣日治時期}}
[[大日本帝國|日本]]統治初期，由於當時日本殖民政府在台施政屢受挫折，日本政界曾一度出現放棄[[台灣|台灣]]、並將其賣給[[法國|法國]]的「台灣賣卻論」；但後來因對台灣的統治成效漸入佳境而作罷。

==中華民國臺灣時期==

{{main|中華民國與法國關係}}

1949年，[[臺灣戰後時期|中華民國政府遷台]]後，法國亦在首都[[台北市|台北]]保留'''法蘭西共和國駐中華民國大使館'''，但僅派駐[[代辦|代辦]]而非大使。

1950年11月30日，法國在[[聯合國安全理事會|聯合國安全理事會]]上[[聯合國安全理事會否決權|否決]]中共代表[[伍修權|伍修權]]等提出之「美國侵略台灣案」。<ref>{{cite conference|title=聯合國安全理事會正式紀錄 第七十二號 第五三0次會議|date=1950-11-30|publisher=[[聯合國安全理事會|聯合國安全理事會]]|location=[[紐約|紐約]]發拉星草場|url=http://www.un.org/en/ga/search/view_doc.asp?symbol=S/PV.530&referer=/english/&Lang=C|accessdate=2015-08-24|language=zh-hant}}</ref>

1960年代，法國開始與[[中華人民共和國|中華人民共和國]]進行關係正常化行動，當時[[中華民國政府|中華民國政府]]極力阻止，其間[[中華民國總統|中華民國總統]][[蔣中正|蔣中正]]還致親筆信給時任[[法國總統|法國總統]][[戴高樂|戴高樂]]以穩固邦交。

1964年1月27日，法國和中華人民共和國建交，中華民國外交部抗議法國與中共政權建交，表明反對「[[兩個中國|兩個中國]]」；1月25日，法國總統戴高樂派前駐華大使{{link-en|貝志高|Zinovy Peshkov}}向總統蔣中正解釋；2月7日，外交部重申反對「兩個中國」，指[[戴高樂主義|戴高樂中立政策]]是分期投降；2月10日，宣布與法國斷交。<ref name="記" >{{cite book |authors=[[呂芳上|呂芳上]]總策畫，朱文原、周美華、葉惠芬、高素蘭、陳曼華、歐素瑛編輯撰稿 |title=《中華民國建國百年大事記》 |location=台北 |publisher=國史館 |year=2012 |isbn=978-986-03-3586-6 }}</ref>{{rp|698}}12日，關閉大使館。<ref>{{Cite web |url = http://multilingual.mofa.gov.tw/web/web_UTF-8/almanac/almanac1999/attach/chapter52.htm |title = 《中華民國88年外交年鑑》〈附錄五、我國與世界各國關係一覽表〉 |publisher = 中華民國外交部}}</ref>除了總部在法國巴黎的[[聯合國教育、科學及文化組織|聯合國教育、科學及文化組織]]（UNESCO）之外，兩國關係全面中斷。<ref name="駐館與駐地關係">{{Cite web |url = http://www.taiwanembassy.org/fr/post/2936.html |title = 駐館與駐地關係 |publisher = 中華民國駐外單位聯合網站}}</ref>

斷交後，直到1989年成立[[法國在台協會|法國在台協會]]整併相關業務進行外交。

2001年11月，總統夫人[[吳淑珍|吳淑珍]]前往法國，先在巴黎參訪兩天後，再轉赴東部[[史特拉斯堡|史特拉斯堡]]的[[歐洲議會|歐洲議會]]，代表總統[[陳水扁|陳水扁]]接受[[國際自由聯盟|國際自由聯盟]]頒發的「2001年自由獎」，並會見歐洲議會議長{{link-en|妮可·方亭|Nicole Fontaine|方亭}}。隨同前往的包括外交部次長、總統府第三局長等人。對於此行，[[法國政府|法國政府]]也提供相關的禮遇與接待。<ref>{{Cite web |url = https://www.president.gov.tw/NEWS/2024 |date = 2001年11月11日 |title = 總統晚間為夫人吳淑珍女士送行 |publisher = [[中華民國總統府|中華民國總統府]]}}</ref>

2018年7月25日，立法院長[[蘇嘉全|蘇嘉全]]率團訪問[[法國國民議會|法國國民議會]]，創下立法院長進入國民議會議場的首例。<ref>{{Cite web |url = http://www.storm.mg/article/469371 |author = 曾詩婷 |date = 2018年7月29日 |title = 國會外交！蘇嘉全率團訪歐洲 英國上議院議長親自會晤 |publisher = [[風傳媒|風傳媒]]}}</ref>
