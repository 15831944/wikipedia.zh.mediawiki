'''台灣夜市'''以當中的[[台灣小吃|台灣小吃]]聞名，通常在路邊設有小桌子小凳子就地坐下享用，或是打包帶走。小吃的隨著潮流快速的更新，但有些極富特色受大歡迎的小吃，例如[[蚵仔煎|蚵仔煎]]、[[沙威瑪|沙威瑪]]和[[臭豆腐|臭豆腐]]隨處可見。有些城市，或是某個夜市因為某種小吃而出名，例如台南的[[擔仔麵|擔仔麵]]和[[棺材板|棺材板]]。目前台灣夜市大約有300多個分佈在不同的城市。

根據[[交通部觀光局|觀光局]]於2010年進行的網路票選，臺灣前5名夜市分別為[[高雄市|高雄]][[六合夜市|六合夜市]]、[[臺北市|臺北]][[士林夜市|士林夜市]]、[[宜蘭縣|宜蘭]][[羅東夜市|羅東夜市]]、[[臺南市|臺南]][[花園夜市|花園夜市]]及[[臺中市|臺中]][[逢甲夜市|逢甲夜市]]<ref>[http://www.libertytimes.com.tw/2010/new/aug/16/today-life1.htm 南北大PK…夜市網路票選 六合大勝士林] {{webarchive|url=https://web.archive.org/web/20130820234714/http://www.libertytimes.com.tw/2010/new/aug/16/today-life1.htm |date=2013-08-20 }}，自由電子報</ref>。

== 歷史 ==
夜市在臺灣，有超過100年的歷史。通常是因為當地的交通方便，或是有遼闊場地以及多人群聚集而加以形成。據《[[臺灣日日新報|臺灣日日新報]]》資料，1908年，有人在[[鳳山廳|鳳山廳]][[旗津天后宮|旗津天后宮]]廟前空地設夜市，營業時間為18時至24時<ref>《舊日時光》，[[陳柔縉|陳柔縉]]，[[大塊文化|大塊文化]]，2012年，ISBN 9789862133927</ref>。

== 四大形態 ==
臺灣夜市主要分為四種形態：
* 觀光夜市：由政府相關單位輔導原本街邊夜市為整齊、規劃良好的觀光區，並結合當地特色俾使成為觀光休閒的夜市。
* 街邊型夜市：又稱商圈型夜市，多為既有商業區延長營業至深夜，商家多有自家或承租店面，亦會吸引攤販會在路邊開張營業，都會區的夜市常屬於商圈夜市。
* 商場型夜市：集中於大型建築物內的夜市。
* 流動型夜市：多為在市區空地或市郊營業，平時可能作為[[停車場|停車場]]，只在特定日期（例如每週三、五）營業；商家全為攤販形態，傍晚時到達場地，午夜過後全部撤除。流動夜市的移動特性，使得夜市可以巡迴深入各鄉鎮。在[[南臺灣|南臺灣]]，流動夜市過去稱作'''商展'''，表商業產品的展示<ref>{{cite web | url=http://blog.xuite.net/khoguan/blog/47040550 | title=南部台語：「商展」「夜市仔」 @ 潘科元台語文理想國 :: 隨意窩 Xuite日誌 | accessdate=2013-06-15}}</ref><ref>{{cite web| url=http://www.town-all.org.tw/viewweb/stu323/from.htm| title=夜市的由來| accessdate=2013-06-15}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。現在的人習慣稱夜市，商展這一名詞才漸漸被夜市取代。

== 貨品選擇 ==
夜市不但供應小吃也提供各式各樣的珍奇貨品，例如衣服、書包、鞋子、工藝品、鐵器，還有一些小遊戲，顧客只要投入硬幣就能玩上一段時間。
* 雜貨：販賣五金、雜貨、日常用品等。
* 美食：販賣飲食料理，以及許多地方小吃。
* 遊戲：經營一些簡單遊戲，例如打電動玩具、打牌、[[打彈珠|打彈珠]]、[[射飛鏢|射飛鏢]]、[[撈金魚|撈金魚]]、套圈圈、玩具搶擊等，近代更出現[[夾娃娃機|夾娃娃機]]等。
* 休閒：尤其夏天夜市人潮較多。男女朋友約會吃喝玩樂都有的地方形成一種另類文化場所，常有藝人演出造勢，文化表演等等。
