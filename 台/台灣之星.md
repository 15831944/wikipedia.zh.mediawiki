'''台灣之星電信股份有限公司'''（[[英語|英語]]：Taiwan Star Telecom Corporation Limited，[[縮寫|縮寫]]：'''{{lang|en|T STAR}}'''，簡稱'''台灣之星'''），是中華民國（台灣）五大電信業者的其中一家，前身是2000年4月成立的'''聯邦電信股份有限公司'''，於2013年改為目前的公司名稱。2014年，公司併購[[威寶電信|威寶電信]]。2014年8月25日台灣之星正式開台，2018年3月底總用戶數已達205萬戶<ref>{{Cite web|url=https://news.cnyes.com/news/id/4094449|author=楊伶雯|title=台灣之星Q1總用戶數205萬 年增21% 連15個月攜碼冠軍|publisher=[[鉅亨網|鉅亨網]]|date=2018-04-12|accessdate=2018-11-19}}</ref>。

== 歷史沿革 ==
=== 聯邦電信（1999年－2003年9月） ===
1999年7月，'''聯邦電信股份有限公司籌備處'''成立。2000年4月，'''聯邦電信股份有限公司'''以[[資本額|資本額]][[新台幣|新台幣]]3億元整設立完成；5月，[[中華民國|中華民國]][[財政部證券暨期貨管理委員會|財政部證券暨期貨管理委員會]]核准聯邦電信[[股票|股票]]公開發行；5月30日，聯邦電信在[[臺灣證券交易所|臺灣證券交易所]][[股票上市|股票上市]]，[[股票代號|股票代號]]為3157。

2002年2月，聯邦電信[[第三代流動電話|第三代行動通信]]業務競標得標。10月，聯邦電信董監事改選，選任[[金寶電子|金寶電子]]代表人[[許勝雄_(電機)|許勝雄]]擔任[[董事長|董事長]]。

=== 威寶電信（2003年9月－2014年） ===
2003年9月，聯邦電信改名案獲[[中華民國經濟部|經濟部]]核准通過，並正式更名為'''威寶電信股份有限公司'''。

2005年9月，威寶電信向金融機構辦理金額高達新台幣90.6億元的聯貸案，由[[兆豐國際商業銀行|中國國際商業銀行]]、[[中國信託商業銀行|中國信託商業銀行]]、[[日盛國際商業銀行|日盛國際商業銀行]]、[[大眾商業銀行|大眾商業銀行]]、[[中華開發工業銀行|中華開發工業銀行]]、[[合作金庫銀行|合作金庫銀行]]等六家銀行聯合主辦，共23家銀行參與聯貸。

2005年9月6日，威寶電信與[[東訊|東訊]]合資成立「威寶電通股份有限公司」，負責經營手機通路。12月，威寶電信正式啟用第三代行動通信商用服務。

2006年4月，威寶電信突破10萬用戶數，與[[震旦電信|震旦電信]][[策略聯盟|策略聯盟]]經營[[移动虚拟运营商|虛擬行動電話服務]]（MVNO）業務。12月，「威寶威通卡」上市。

2007年3月，「威寶電信儲值卡」上市。5月，威寶電信再與東訊合資成立[[威邁思電信|威邁思電信]]，力推[[WiMAX|WiMAX]]通訊服務。6月，推出台灣首張3G預付卡「威寶旺卡（VIBO one card）」上市。9月4日，「威通卡及其附屬天線組」手機非接觸感應式（[[NFC|NFC]]）服務啟用；同一天與[[聯邦商業銀行|聯邦商業銀行]]合作的「威通聯邦信用卡」上市<ref>{{Cite web|url = http://www.vibo.com.tw/CWS/About_01_06,2c9681c6146b8a800114fd7306610160,,,.html?pageDirect=index|title = 威寶「威通卡」再創新應用 推出「威通卡及其附屬天線組」領先同業 全球首推雙卡合一 非接觸感應式(NFC)技術,與聯邦銀行合作　搭配一般市售3G手機 方便又好用|accessdate = 2014-10-03|date = 2007-09-04|author = |publisher = 威寶電信|deadurl = yes|archiveurl = https://web.archive.org/web/20141006124541/http://www.vibo.com.tw/CWS/About_01_06,2c9681c6146b8a800114fd7306610160,,,.html?pageDirect=index|archivedate = 2014-10-06}}</ref>。11月28日，威寶電信與[[悠遊卡公司|悠遊卡公司]]合作推出該公司首張手機[[悠遊卡|悠遊卡]]「威通悠遊卡」並上市<ref>{{Cite web|url = http://www.vibo.com.tw/CWS/About_01_06,2c9681c6164002720116a31e276e0126,,,.html?year=2007|title = 全台首張手機悠遊卡 ─「威通悠遊卡」－即日起上市搭捷運、乘貓纜　轉公車、停車場　手機對準感應設備 輕鬆「嗶」一聲就上路 大多數3G手機均適用 現在申辦 立即可用 全套商品外加通話費、3G手機 價值9,900元 現在只要990元 通通一起帶回家|accessdate = 2014-10-03|date = 2007-11-28|author = |publisher = 威寶電信|deadurl = yes|archiveurl = https://web.archive.org/web/20141006153324/http://www.vibo.com.tw/CWS/About_01_06,2c9681c6164002720116a31e276e0126,,,.html?year=2007|archivedate = 2014-10-06}}</ref>。12月，威寶電信完成4,000多座[[WCDMA|WCDMA]][[基地台|基站]]優化。

2009年4月，威寶電信突破100萬用戶數。9月，「MyCard影音名片」上市。11月，威寶電信完成全台首座試驗網整合WCDMA／[[TD-SCDMA|TD-SCDMA]]系統資源。12月，威寶電信突破150萬用戶數。

2010年3月，威寶電信推動品質提升年，用戶數目標二百萬，營收突破新台幣120億元。4月，威寶電信舉辦「行動台北科技走廊」成果發表會，完成首創雙品牌系統[[3.5G|3.5G]]與[[WiMAX|WiMAX]]實驗網路共站認證。12月，威寶電信[[自有品牌|自有品牌]][[平板電腦|平板電腦]]「Vibo Vpad」上市。

2011年6月，威寶電信99年稅後淨損大幅改善，股東常會改選董監事，總經理周鐘麒當選董事；同時威寶電信更新[[企業識別|企業識別]]，[[英文|英文]][[商標|商標]]「VIBO」第一個字母「V」作為公司識別代表字，象徵「Victory（勝利）」，而「V」 的造型如同一隻展翅向上翱翔的雄鷹，象徵威寶將展翅高飛、重新出發。7月，威寶電信獨家在台販售雙卡雙待[[Android|Android]]智慧型手機。8月，威寶電信與中國信託商業銀行等銀行團簽訂五年期新台幣50億元聯貸。11月，威寶電信通過減資新台幣128.04億元。

2012年6月，威寶電信與[[连我|LINE]]攜手合作，另完成全網過半基地台為[[HSPA+|HSPA+]]（21Mbps）3.75G服務建置（六大都會地區全面提供HSDPA高速上網服務）；同月根據[[消基會|消基會]]委託[[台大|台大]]電機研究所蒐集10萬筆行動上網連線測速資料（2011年8月15日至2012年5月28日）調查結果顯示，威寶電信的行動數據服務品質和國內三大電信公司並駕齊驅，有些地方之網路品質甚至優於三大電信公司。9月4日，威寶電通解散。

2013年6月，威寶電信更新店格，另榮獲《[[工商時報|工商時報]]》「2013年臺灣服務業大評鑑」連鎖電信通路類銀牌獎。11月8日，金寶電子與[[仁寶電腦|仁寶電腦]]宣布將出售手中威寶電信股份予台灣之星，並計畫於2014年4月底以股權轉換方式轉換為持有8%台灣之星股權<ref>{{Cite web|url=http://www.appledaily.com.tw/appledaily/article/finance/20131109/35425617/|author=王郁倫、劉煥彥|title=威寶嫁頂新台灣之星 金仁寶先認列98億業外損失 再拿106億現金完成交易|publisher=[[臺灣蘋果日報|臺灣蘋果日報]]|date=2013-11-09|accessdate=2014-05-10}}</ref>；股份轉換後，金寶電子與仁寶電腦各約持有台灣之星4%股權，不再持有威寶電信股權<ref>{{Cite web|url=http://www.cna.com.tw/news/firstnews/201311080029-1.aspx|author=韓婷婷|title=威寶賣了 台灣之星娶回家|publisher=[[中央通訊社|中央通訊社]]|date=2013-11-08|accessdate=2014-06-04}}</ref>。

=== 台灣之星電信 （2013年至今）===

=== 台灣之星電信時期斷網事件 ===
*2018年1月25日：約下午7：00～8：40時，因於低訊務時段，執行網路[[域名系统|DNS]]系統升級作業，廠商設定出現異常，以致於自19:00開始，許多用戶出現壅塞甚至無法連網的現象，但語音服務則不受影響。經緊急重啟DNS後，於當日20:40恢復連網服務，影響時間約1小時40分，影響用戶約3萬名。此次網路異常雖未達法定故障2小時需賠償的標準，但台灣之星宣布針對當時上網受影響的用戶主動提供補償，減免5%月租費。<ref>{{Cite news|author=江明晏|url=http://m.cna.com.tw/news/afe/201801260025.aspx|title=台灣之星估3萬戶連網異常 減免5%月租費|publisher=中央社|date=2018-01-26|accessdate=2018-02-12}}</ref>另有用戶指出，影響時間遠超過2小時。

=== 2018年雙11活動非議 ===
*2018年11月9日，由台灣之星副總朱曉幸於官方臉書直播宣布88元方案20000名，並於直播後發放保證入場序號10000名。官方宣稱為2018雙11活動準備一年之久，但於11日活動當日申辦系統異常，發放10000名晚間六點前保證入場序號，卻有超過6000名以上保證入場客戶未申辦成功。
*2018年11月11日，上午11時正式開賣「雙11 ▪ 終身$88」20000名額，晚間18：57售完、「雙11 ▪ 高速$188」40,000名額也於晚間23：09全數銷售完畢，但活動中由於大量人潮湧入造成系統無法即時處理並當機，產生頁面跳出及空轉的現象；加上短時間多次重覆點擊導致系統乘載更加劣化，以致開賣後前幾個小時的入場驗證碼、保證入場序號輸入等相關系統處於極度不穩定狀態，造成許多用戶無法順利進入賣場申辦或不斷重試以及許多用戶反應在[[信用卡|信用卡]]資料輸入完畢後卻突然被踢出，而又要重新排隊進入引起不滿與爭議。<ref>{{Cite news|author=黃晶琳|url=https://udn.com/news/story/7098/3475830|title=雙11搶爆 台灣之星加開6,000門封館搶購|publisher=經濟日報|date=2018-11-12|accessdate=2018-11-12}}</ref>
*2018年11月14日，加開封館補償搶購6000門，其中加碼88元方案2000名，188元方案4000名，於晚間8時開賣後迅速額滿，許多民眾向新聞媒體投訴，搶出一肚子怨氣。本次活動僅限當初雙11有搶到序號的人可以「封館」搶購，亦迅速售完。部分民眾批評台灣之星再一次欺騙消費者<ref>{{Cite news|author=陳怡文|url=https://tw.appledaily.com/new/realtime/20181115/1466938/|title=台灣之星雙11補償活動又挨批　民怒「資料亂填即可入場」|publisher=蘋果日報|date=2018-11-15|accessdate=2018-11-15}}</ref><ref>{{Cite news|author=爆爆哥|url=https://tw.news.appledaily.com/headline/daily/20181116/38180862/?fbclid=IwAR2PN5creBSTmqb641vznkIttCcfryMX-s0h3QmPjtQC6B361KvQw9lBtPQ|title=推雙11補償活動 台灣之星又惹怨|publisher=蘋果日報|date=2018-11-16|accessdate=2018-11-16}}</ref>，要求台灣之星應更積極立即回應及解決。有些民眾已向主管機關[[國家通訊傳播委員會|國家通訊傳播委員會]]申訴。
