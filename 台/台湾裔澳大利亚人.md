'''臺灣裔澳洲人'''是指來自[[臺灣|臺灣]]並且入籍成為[[澳洲公民|澳洲公民]]的人士。在2016年澳洲人口普查中，澳洲該年共有70,790位[[臺灣裔|臺灣裔]]移民。

== 定義 ==
臺灣裔澳洲人的確切人數難以計算，因為多數人口研究往往將來自[[中國大陸|中國大陸]]，[[香港|香港]]，[[澳門|澳門]]和[[臺灣|臺灣]]的移民分類為廣義的“華裔澳洲人”，因此對於一般的臺灣人來說，自認[[臺灣裔|臺灣裔]]，主要是對自己臺灣血緣和文化土地感情的認同，另外也有和中華人民共和國公民 (中國人)有所區隔的意味在，被視為[[臺灣正名運動|臺灣正名運動]]和[[臺灣獨立運動|臺灣獨立運動]]的一個面向。在2016年人口普查中，有46,882人宣佈出生在臺灣。<ref>http://www.censusdata.abs.gov.au/census_services/getproduct/census/2016/communityprofile/036?opendocument</ref>

== 歷史 ==
在1950年代之前，澳洲的臺灣移民是微不足道的。至1970年代後，在[[雪梨|雪梨]]和[[墨爾本|墨爾本]]等東岸大城才開始形成眾多的臺裔社區。

== 語言 ==
臺灣的第一代移民通常會使用[[現代標準漢語|現代標準漢語]]，儘管許多人也會使用[[臺灣閩南語|臺灣閩南語]]；[[客家話|客家話]]在臺灣移民中較少被使用。第二代出生的臺灣裔澳洲人使用[[臺灣閩南話|臺灣閩南話]]當作母語但可能對[[現代標準漢語|現代標準漢語]]不太精通，如同其他[[臺僑|海外臺灣人]]一樣。來自臺灣北部特別是[[大台北|大台北]]地區的臺灣移民使用[[現代標準漢語|現代標準漢語]]及一些[[臺灣閩南語|臺灣閩南語]]當作母語。第二代[[臺灣客家人|臺灣客家人]]移民傾向使用普通話。第一代的[[臺灣客家人|臺灣客家人]]則精通[[客家話|客家話]]、[[現代標準漢語|現代標準漢語]]及[[英語|英語]]。使用[[臺灣話|閩南語]]和[[客家話|客家話]]的臺灣裔澳洲移民第二代可能只會說[[現代標準漢語|現代標準漢語]]，而[[臺灣外省人|臺灣外省人]]和[[臺灣閩南人|臺灣閩南人]]通婚移民澳大利亞的後代，也只會使用[[現代標準漢語|現代標準漢語]]而不會使用[[臺灣話|臺灣話]]。

== 定居 ==
由於鄰近[[墨爾本|墨爾本]]和[[雪梨|雪梨]]，[[布里斯本|布里斯本]]為[[臺灣人|臺灣人]]移民澳洲較多的城市。
