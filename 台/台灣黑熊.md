'''臺灣黑熊'''（[[学名|学名]]：''{{lang|la|Ursus thibetanus formosanus}}''、[[布農語|布農語]]：tumaz、[[賽夏語|賽夏語]]：somay、[[泰雅語|泰雅語]]：ngarox、[[鄒語|鄒語]]：cmoi、[[卡那卡那富語|卡那卡那富語]]：cumai、[[德路固語|太魯閣語]]：kumay），是[[臺灣|臺灣]][[特有種|特有]]的[[亞洲黑熊|亞洲黑熊]][[亞種|亞種]]，胸前的V字型白色斑紋（太魯閣語稱做 rabang）是[[亞洲黑熊|亞洲黑熊]]共有的特徵。台灣黑熊現存族群數量粗估為200~600隻（2014年估計）<ref>{{cite news|url=http://www.tanews.org.tw/info/2922|title=黃美秀與台灣黑熊 愛的三部曲|date=2014-03-13|author=何宜|publisher=台灣動物新聞網|access-date=2019-03-17}}</ref>，出沒於[[台灣|台灣]][[中央山脈|中央山脈]]海拔1000[[公尺|公尺]]-3500公尺的山區，活動範圍可達50[[平方公里|平方公里]]。早在1932年[[日本|日本]]探險家{{Tsl|ja|堀川安市|堀川安市}}所著的《台灣哺乳動物圖說》中，因土地開發導致棲息地喪失，臺灣黑熊的數量在下降中。在1989年依照中華民國《[[s:野生動物保育法|野生動物保育法]]》，台灣黑熊被列為[[瀕危|瀕危]]。2001年台灣黑熊被選為[[台灣|台灣]]最具代表性的野生動物，不少台灣公司、企業、地方政府，都選用台灣黑熊作為吉祥物，是著名的台灣意象之一。

== 生物特徵 ==
[[File:Ursus_thibetanus_formosanus_V_character.jpg|thumb]]
臺灣黑熊極為強壯，體長120-180[[公分|公分]]，肩寬60-70[[公分|公分]]，最重可達200[[公斤|公斤]]。牠的頭呈圓形、頸部短、眼睛小、吻部很長。牠的頭約26-35公分長，頭圍約40-60公分。耳長8-12公分，牠的吻部形狀像狗，因此許多人習慣稱呼牠為「狗熊」。牠的[[尾巴|尾巴]]短，通常不到10公分。身體覆蓋著粗，濃密而具有光澤的黑毛，在[[頸|頸]]部的毛可以超過10公分長。牠的[[頷|頷]]部末端是白色的，「在胸口有黃色或白色毛呈V字型」、或是弦月的形狀，因此英文中也有「月熊」（Moon Bear）的稱呼。

== 習性 ==
[[File:Ursus_thibetanus_formosanus_cub_on_tree.jpg|thumb]]
除了交配期或撫育小熊期間外，通常獨居。白天在樹洞或岩洞內休息，黃昏或夜晚時外出覓食。活動範圍大，擅爬樹，孔武有力，在受傷、受逼迫或保護幼熊時攻擊性強，速度快。冬季時並不冬眠，可能會移至較低海拔區域覓食。
屬[[雜食性|雜食性]]，以小型動物、昆蟲、植物嫩葉、嫩芽、果實為食，台灣黑熊也喜歡吃[[蜂蜜|蜂蜜]]和蜂蛹，也會在河邊捕魚、捕螃蟹、捕蝦等食用。

== 與人的關係 ==
[[File:Mascot_of_Formosan_Black_Bear_@_Yangmingshan_National_Park_in_Taipei,_TAIWAN_臺灣黑熊吉祥物布偶在臺灣臺北陽明山國家公園.jpg|thumb]]布偶在雪季[[臺灣|臺灣]][[臺北|臺北]][[陽明山國家公園|陽明山國家公園]]]]
台灣黑熊不會主動攻擊人類，若碰上，最好是靜靜、慢慢地退開，不可轉身驚慌大叫和狂奔，這樣可能會引發黑熊加速攻擊。

== 保護狀態 ==
從1998到2000年共有15隻[[玉山國家公園|玉山國家公園]]的臺灣黑熊被裝上無線電項圈，目擊黑熊的事件非常少，沒有證據證明尚有多少隻臺灣黑熊存在。從1989年開始臺灣黑熊就正式被列入保護動物。但盜獵可能沒有因此禁絕，對台灣黑熊的危害依然存在。當年捕捉到的15隻黑熊中，由於盜獵陷阱有8隻喪失了腳趾或是腳掌。
 	
從1989年至2001年間，臺灣黑熊依中華民國《[[文化資產保存法|文化資產保存法]]》列入瀕危動物或珍貴稀有動物，之後在[[農委會|農委會]]公告的《保育類野生動物名錄》中屬於「瀕臨絕種野生動物 」<ref>{{cite report|title=保育類野生動物名錄|publisher=行政院農委會林務局|page=7|year=2019|url=https://conservation.forest.gov.tw/0002021|}}</ref>。在國際上，[[亞洲黑熊|亞洲黑熊]]屬《[[瀕臨絕種野生動植物國際貿易公約|瀕臨絕種野生動植物國際貿易公約]]》（CITES）的附錄I，禁止國際間對這個物種任何產品與任何形式的交易<ref>{{cite report|url=https://cites.org/sites/default/files/notif/E-Notif-2016-068-A.pdf|title=瀕臨絕種野生動植物國際貿易公約附錄|year=2017|publisher=聯合國環境署|language=en}}</ref>、也是[[世界自然保護聯盟|世界自然保護聯盟]]的《[[IUCN紅色名錄|IUCN紅色名錄]]》易危物种<ref>{{cite report|author=Garshelis, D. & Steinmetz, R.|year=2016|title= Ursus thibetanus (errata version published in 2017)|journal= The IUCN Red List of Threatened Species |page=e.T22824A114252336|doi=10.2305/IUCN.UK.2016-3.RLTS.T22824A45034242.en|language=en|access-date=2019-03-17|url=https://www.iucnredlist.org/species/22824/114252336}}</ref>。

== 吉祥物 ==
[[File:YLS_8822.jpg|thumb]][[吉祥物|吉祥物]][[熊讚Bravo|熊讚]]]]
台灣黑熊是台灣特有的黑熊亞種，所以有些機構以台灣黑熊為藍本設計[[吉祥物|吉祥物]]。
* [[熊讚|熊讚]]，2017年台北世運大及台北市吉祥物「Bravo」
* [[高雄熊|高雄熊]]，高雄市吉祥物「Hero」
* 喔熊，[[台灣觀光局|台灣觀光局]]吉祥物「OhBear」
* 威熊，[[威航|威航]]吉祥物「V air」
* 台鐵便當餐旅熊，[[台鐵|台鐵]]吉祥物「鐵魯、漢娜將」
* 寧夏熊，[[寧夏夜市|寧夏夜市]]吉祥物
