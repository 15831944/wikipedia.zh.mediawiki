'''台灣東方出版社股份有限公司'''，簡稱'''東方出版社'''，[[台灣|台灣]]一家以出版[[語言教育|語言教育]]和[[兒童文學|兒童文學]]作品為主的[[出版社|出版社]]，建立於1945年10月。東方出版社創立於1945年（中華民國三十四年）[[臺灣戰後時期|戰後]]不久滿目瘡痍的[[臺北市|臺北市]]，積極推行[[國語文|國語文]]，揭開了台灣兒童讀物出版的序幕<ref>{{Cite web |url=http://children.moc.gov.tw/search/website_detail.php?id=com000011| language=zh-tw |author=|title=臺灣東方出版社|publisher=文化部兒童文化館|date= |accessdate=2015-12-24}}</ref><ref name="辛廣偉">{{Cite book|author=辛廣偉|title=《台灣出版史》|publisher=[[河北教育出版社|河北教育出版社]]|location=[[石家庄市|石家庄市]]|year=2000|isbn=7543439964|language=zh-cn|accessdate=2013-07-19}}</ref>。

==歷史==
[[File:Eastern_Building_on_Chongqing_South_Road,_Taipei_City_20130903.jpg|thumb]]
1945年10月，[[游彌堅|游彌堅]]、[[林呈祿|林呈祿]]、[[范壽康|范壽康]]等人共同創辦東方出版社，總部設於[[臺北市|臺北市]][[重慶南路_(台北市)|重慶南路]]一段，游彌堅任總負責人兼[[總編輯|總編輯]]，林呈祿任社長<ref name="辛廣偉"/>。1950年4月27日，[[臺北市政府|臺北市政府]]核准東方出版社公司登記，公司全名「台灣東方出版社股份有限公司」。當時東方出版社總部位於重慶南路、[[衡陽路|衡陽路]]口的[[紅磚|紅磚]]三層樓大樓，這幢大樓乃是[[台灣日治時期|台灣日治時期]][[新高堂書店|新高堂書店]]舊址<ref name="辛廣偉"/>；1980年新高堂書店舊址就地改建為8層樓高「東方大樓」後，東方出版社總部搬到東方大樓4樓。<ref>翁聿煌 台北報導，[http://www.libertytimes.com.tw/2012/new/may/19/today-taipei10-2.htm 〈東方出版社 台灣兒少文學先驅〉] {{webarchive|url=https://web.archive.org/web/20120519023018/http://libertytimes.com.tw/2012/new/may/19/today-taipei10-2.htm |date=2012-05-19 }}，《[[自由時報|自由時報]]》2012年5月19日。</ref>

1954年，東方出版社創辦《東方少年》月刊，是東方出版社為慶祝成立8周年而創辦的月刊，封面皆為當時臺北市[[小學生|小學生]]照片。<ref>郭美懿 採訪，[http://www.appledaily.com.tw/appledaily/article/finance/20060910/2879821 〈漫畫迷 癡情45年 畢生積蓄換滿室珍藏〉]，《[[台灣蘋果日報|台灣蘋果日報]]》2006年9月10日。</ref>1955年至1964年，東方出版社陸續出版《東方少年文庫》、《世界偉人傳記》、《世界少年文學選集》、《中國少年通俗小說》。1965年至1974年，東方出版社陸續出版《亞森羅蘋全集》與《福爾摩斯探案全集》。1975年至1984年，東方出版社陸續出版 《史記》、《西頓動物故事》、《少年冒險故事》、《小小研究生》、《小科學家》。1985年至1994年，東方出版社出版《漫畫科學小百科》。2003年，東方出版社開闢《繪本精選》系列與《青春悅讀》系列。2005年，東方出版社開闢《故事摩天輪》系列。2006年，東方出版社開闢《金獎小說》系列。2007年，東方出版社開闢《123到台灣》系列與《古典小說全家讀本》系列。2010年，東方出版社開闢《陽光少年遊》系列。

東方出版社為[[第二次世界大戰|第二次世界大戰]]後的台灣兒童文學的啟蒙者。1965年－1974年東方出版社翻譯出版的[[亞森羅蘋|亞森羅蘋]][[推理小說|推理小說]]《亞森羅蘋全集》，並非直接譯自作者[[莫理斯·盧布朗|莫理斯·盧布朗]]的[[法文|法文]]原作，而是譯自[[日本|日本]][[冒險小說|冒險小說]]作家[[南洋一郎|南洋一郎]]改寫（例如刪除部分短篇、改寫不適合兒童閱讀的部分[[獵豔|獵豔]]內容）、[[POPLAR社|POPLAR社]]從1958年開始發行的15集童書版（1971年才將書系修訂且增為30集）<ref>{{Cite web |url=http://thinkread.asdc.tw/eastern| language=zh-tw |author=|title=思想，重慶南路——東方出版社|publisher= [[中央研究院|中央研究院]]數-{}-位文化中心|date= |accessdate=2015-07-11}}</ref>，為1960年代－1970年代台灣受歡迎的兒童讀物之一，亞森羅蘋在台灣幾乎與[[福爾摩斯|福爾摩斯]]齊名。甚至因為莫理斯·盧布朗筆下刻意貶低福爾摩斯，在1990年代之前的台灣，亞森羅蘋的知名度遠高於福爾摩斯。
