'''台灣夏天'''的[[氣候|氣候]][[炎熱|炎熱]]（除[[山地|山地]]外，其餘皆在20℃以上，最高平均[[溫度|溫度]]可達30℃以上），[[雨量|雨量]]豐沛（山地多於[[平地|平地]]、東岸多於西岸、北部多於南部），吹西南風，屬於[[颱風|颱風]]季。

依照[[二十四節氣|二十四節氣]]來看，[[傳統|傳統]]的[[夏天|夏天]]為[[立夏|立夏]]至[[大暑|大暑]]，即從5月初至7月底之間。不過台灣因為有[[北回歸線|北回歸線]]通過，以[[北|北]]為[[副熱帶季風氣候|副熱帶季風氣候]]，以[[南|南]]為[[熱帶季風氣候|熱帶季風氣候]]，整體氣候較為炎熱，因此有些人認為台灣的夏天為最早從5月開始，最晚可以到9月為止（西方人以6月下旬的[[夏至|夏至]]到9月下旬的[[秋分|秋分]]為夏天）。

== 飲食 ==
[[台灣人|台灣人]]在夏天喜好吃[[冰品|冰品]]。由於台灣夏天盛產[[芒果|芒果]]，因此近年來[[芒果冰|芒果冰]][[流行|流行]]一時。另外像[[仙草冰|仙草冰]]、[[愛玉冰|愛玉冰]]等也很受歡迎。

台灣夏天盛行的[[水果|水果]]有芒果、[[西瓜|西瓜]]、[[香蕉|香蕉]]、[[釋迦|釋迦]]、[[葡萄|葡萄]]、[[荔枝|荔枝]]、[[水蜜桃|水蜜桃]]等等。

有些台灣年輕人喜歡故意在夏天的時候去吃[[麻辣鍋|麻辣鍋]]，可以說是一種特別的[[次文化|次文化]]。

== 服裝 ==
台灣人在夏天時常穿著[[T恤|短袖上衣]]、[[短褲|短褲]]、[[短裙|短裙]]、[[拖鞋|拖鞋]]或[[涼鞋|涼鞋]]出門。[[吊帶背心|吊帶背心]]在年輕男女之前
間則頗為流行。

由於北台灣一到夏天就呈現不穩定的[[天氣|天氣]]，因此無論[[天氣預報|天氣預報]]為何，出門時攜帶[[雨傘|雨傘]]或[[折傘|折傘]]已經成為許多人的[[習慣|習慣]]。許多愛[[美|美]]的台灣[[女性|女性]]在夏天即使沒有[[下雨|下雨]]，也會帶[[遮陽傘|遮陽傘]]出門，以避免[[曬黑|曬黑]]。

== 家居 ==
家中有[[冷氣|冷氣]]的台灣人在夏天時喜歡開冷氣消暑，這也是為何台灣夏天的整體[[電費|電費]]比[[冬|冬]]天來得高很多。

台灣夏天雖然雨量豐沛，但是由於[[地形|地形]]的關係不易蓄水，加上台灣人在夏天時喜好[[游泳|游泳]]，因此台灣夏天不時會發生缺水狀態，通常政府以限區停水方式來解決。

== 教育 ==
台灣的[[小學生|小學生]]到[[高中|高中]][[高職|職]]生在夏天通常會有兩個[[月|月]]的[[暑假|暑假]]，一般從7月初開始到8月底結束。而[[大學|大]][[專科|專]][[學生|學生]]則有長達快三個月的暑假，一般從6月中開始到9月中結束。暑假對於許多台灣學生來說是一年當中最[[快樂|快樂]]的時候。

== 娛樂 ==
[[File:Sanjhih-Beach.jpg|thumb]]
台灣人在夏天的時候喜好前往[[游泳池|游泳池]]或[[海邊|海邊]][[游泳|游泳]]。不過由於台灣習俗上[[農曆|農曆]]7月「[[鬼門開|鬼門開]]」，去水邊玩水被認為是危險率比較高的時候，因此一些人農曆7月時便有所[[忌諱|忌諱]]去水邊玩。

台灣的海邊[[水上運動|水上運動]]勝地包括有[[新北市|新北市]][[三芝區|三芝區]]的[[白沙灣_(台灣)|白沙灣]]、[[貢寮區|貢寮區]]的[[福隆海水浴場|福隆海水浴場]]、[[宜蘭縣|宜蘭縣]][[頭城鎮|頭城鎮]]的[[蜜月灣|蜜月灣]]、[[屏東縣|屏東縣]][[恆春鎮|恆春鎮]]的[[芎蕉灣|芎蕉灣]]等等。水上活動包括有[[游泳|游泳]]、[[衝浪|衝浪]]、[[香蕉船|香蕉船]]、[[水上摩托車|水上摩托車]]等。

台灣夏天舉辦的一些全國甚至[[國際|國際]]知名的活動包括有：[[宜蘭國際童玩藝術節|宜蘭國際童玩藝術節]]、[[貢寮國際海洋音樂季|貢寮國際海洋音樂季]]等。
