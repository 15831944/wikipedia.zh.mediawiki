'''台灣國際女性影展'''（[[英語|英語]]譯名：{{lang|en|Women Make Waves Film Festival, Taiwan}}，[[簡稱|簡稱]]：WMWFF），創辦於1993年的[[臺灣|臺灣]][[影展|影展]]，由[[台灣女性影像學會|台灣女性影像學會]]在每年十月舉辦，藉由女性影片之國際觀摩，拓展女性創作視野，並作為增進台灣與國際女影人之交流平台。

== 歷史 ==
該影展是[[亞洲|亞洲]]第一個創辦的{{le|女权主义电影|feminist film theory}}影展，也是臺灣僅次於[[金馬國際影展|金馬國際影展]]所創辦的第二個影展。<ref>{{cite news|url=http://www.epochtimes.com/b5/15/10/27/n4559808.htm |title= 2015女性影展巡迴台中 國際佳片感受女導風華 |publisher= 大紀元 |date= 2015-10-27 |author= 林艾琦 |accessdate= 2016-03-19}}</ref><ref>{{cite news|url=http://star.ettoday.net/news/414733 |title= 情慾導演來台愛愛初體驗　鄧九雲支持女性影展 |publisher= 東森新聞 |date= 2014-10-17 |author= 林淑娟 |accessdate= 2016-03-19|quote = 台灣國際女性影展是亞洲第一個女性影展，在女導演比例極低的電影產業下，始終是女導演重要的創作平台。}}</ref><ref>{{cite news|url=http://www.chinatimes.com/realtimenews/20140516004187-260405 |title= 台灣女性影展20年 高師大展出 |publisher= 中國時報 |date= 2014-05-16 |author= 吳江泉 |accessdate= 2017-09-06}}</ref>另外，在2001年開始至各地的電影院、學校、藝文中心、社區等據點巡迴放映影展作品，此舉是臺灣首創。<ref>{{Cite web |url = https://www.wmw.com.tw/about.php?It=11 |title = 學會宗旨 |author = 台灣女性影像學會 |date =  |publisher =  |language =  |accessdate = 2017-10-25 |quote = 本學會自2001年起辦理女性影展全國巡迴映演，與各縣市轄下戲院、社區、校園及藝文中心共同辦理，此舉為全國首創。 |archive-url = https://web.archive.org/web/20171012182656/http://www.wmw.com.tw/about.php?It=11 |archive-date = 2017-10-12 |dead-url = yes }}</ref>

== 獎項 ==
自2007年於影展創設「觀眾票選最佳台灣獎」，並於2014年將該獎項修改為競賽項目「光榮時刻：台灣競賽獎」<ref>{{cite news|url=http://www.lihpao.com/?action-viewnews-itemid-9305|title=台灣國際女性影展開始徵件|publisher=立報|date=2009-02-03|author=史倩玲|accessdate=2016-03-19|deadurl=yes|archiveurl=https://web.archive.org/web/20160328075635/http://www.lihpao.com/?action-viewnews-itemid-9305|archivedate=2016-03-28}}</ref><ref>{{cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20151009/708290/ |title= 謝佳見願正面全裸　「蠻自豪」自己的第3點 |publisher= 蘋果日報 |date= 2015-10-09 |author= 張哲鳴 |accessdate= 2016-03-19}}</ref><ref>{{cite news|url=http://www.nownews.com/n/2014/10/18/1461965 |title= 「台灣女性影展」挑戰女同志A片　尺度無極限 |publisher= 今日新聞 |date= 2014-10-18 |author= 翁新涵 |accessdate= 2016-03-19}}</ref>。
