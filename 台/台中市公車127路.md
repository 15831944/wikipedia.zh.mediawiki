'''台中市公車127路'''目前行駛自[[豐樂雕塑公園|豐樂雕塑公園]]到[[臺中洲際棒球場|臺中洲際棒球場]]，主要行駛於[[東興路_(臺中市)|東興路]]和[[汉口路_(台中市)|漢口路]]，為東興漢口線。

==歷史==
*2012年9月29日～9月30日：127試營運，免費搭乘。
*2012年10月1日：127通車，正式營運。
*2013年2月1日：調整離峰班次，從40分鐘改為30分鐘一班。<ref>[http://www.gbus.com.tw/system/modules/tadnews/index.php?nsn=6 自102年2月1日起，臺中市市公車127路調整發車時間] {{webarchive|url=https://web.archive.org/web/20130206105101/http://www.gbus.com.tw/system/modules/tadnews/index.php?nsn=6 |date=2013-02-06 }}</ref>
*2013年7月1日：原北屯區行政大樓之端點，延駛至「臺中洲際棒球場」。「漢口興安路口」站變更為雙邊設站。<ref>{{Cite web |url=http://www.gbus.com.tw/system/uploads/tadnews/file/nsn_18_1.pdf |title=127路於7月1日路線延駛至「臺中洲際棒球場」 |access-date=2014-07-05 |archive-url=https://web.archive.org/web/20140714150636/http://www.gbus.com.tw/system/uploads/tadnews/file/nsn_18_1.pdf |archive-date=2014-07-14 |dead-url=yes }}</ref>
*2013年12月2日：平日（禮拜一至禮拜五）增加以下班次：豐樂公園端06:50、洲際棒球場端16:50。<ref>{{Cite web |url=http://www.gbus.com.tw/system/uploads/tadnews/file/nsn_43_1.pdf |title=127路於7月1日路線延駛至「臺中洲際棒球場」 |access-date=2014-07-05 |archive-url=https://web.archive.org/web/20140714205856/http://www.gbus.com.tw/system/uploads/tadnews/file/nsn_43_1.pdf |archive-date=2014-07-14 |dead-url=yes }}</ref>
*2014年6月1日：「下頭張」站併入「頭張」站。<ref>{{Cite web |url=http://www.gbus.com.tw/system/uploads/tadnews/file/nsn_61_1.pdf |title=自103年06月1日起，127路「下頭張」站位整併至「頭張」站 |access-date=2014-07-05 |archive-url=https://web.archive.org/web/20140714174837/http://www.gbus.com.tw/system/uploads/tadnews/file/nsn_61_1.pdf |archive-date=2014-07-14 |dead-url=yes }}</ref>

==路線基本資料==
全程行車里數約10.5公里，全程行車時間約43分。往豐樂雕塑公園36站，往北屯區行政大樓35站。

127路自豐樂雕塑公園起，沿文心南五路、東興路、漢口路、興安路、昌平路、[[松竹路|松竹路]]、[[崇德路_(台中市)|崇德路]]到北屯區行政大樓。<ref>[http://www.gbus.com.tw/system/uploads/img537c4661c2ec2.jpg 豐榮客運127路營運路線圖] {{webarchive|url=https://web.archive.org/web/20140714121746/http://www.gbus.com.tw/system/uploads/img537c4661c2ec2.jpg |date=2014-07-14 }}</ref>
