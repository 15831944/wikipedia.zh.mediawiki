'''孫震'''（{{bd|1892年|2月5日|1985年|9月9日|catIdx=S孫}}），[[譜名|譜名]]'''定懋'''，後改名'''孫楙'''，再改名'''孫震'''，字'''德操'''，別號'''夢僧'''。[[中華民國|中華民國]]軍事人物，原籍[[浙江|浙江]][[紹興縣|紹興]][[楊家壠|楊家壠]]（今[[齊賢鎮|齊賢鎮]]），[[光緒|光緒]]十八年（1892年）出生於[[四川|四川]][[綿竹|綿竹]]<ref>有一說為[[四川|四川]][[成都|成都]]人，見 [http://taipedia.cca.gov.tw/taipedia/dict/DispText.aspx?id=2086&CurrPage=1&menu=0&n=10&key=%20&type=1&flags=2 台灣歷史辭典] {{webarchive|url=https://web.archive.org/web/20080326185933/http://taipedia.cca.gov.tw/taipedia/dict/DispText.aspx?id=2086&CurrPage=1&menu=0&n=10&key=&type=1&flags=2 |date=2008-03-26 }} 與 [http://www.bashu.net/people/s/sunzhen.htm 此] {{webarchive|url=https://web.archive.org/web/20080327202927/http://www.bashu.net/people/s/sunzhen.htm |date=2008-03-27 }}</ref>。他是[[秦漢_(演員)|秦漢]]父親[[孫元良|孫元良]]的叔父。

== 早年 ==
孫震於[[宣統|宣統]]元年（1909年）進入[[西安|西安]][[陸軍第二中學堂|陸軍第二中學堂]]就讀。之後加入[[中國同盟會|中國同盟會]]。1912年，他進入[[保定陸軍軍官學校|保定陸軍軍官學校]]第一期步兵科就讀，1913年[[二次革命|二次革命]]時，他在學業未完成的狀況下離校跟隨[[熊克武|熊克武]]參加戰役。1914年加入四川陸軍第二師擔任[[排長|排長]]一職，後來持續升為[[連長|連長]]、[[營長|營長]]。1917年升為騎兵旅第一團[[團長|團長]]，後來又升任為[[旅長|旅長]]、[[師長|師長]]、[[川西北屯殖軍|川西北屯殖軍]]司令（1933年<ref>{{cite web |title= |url=http://www.generals.dk/general/Sun_Zhen/_/China.html }}</ref>）等職。1926年經姪兒[[孫元良|孫元良]]介紹投效[[國民革命軍|國民革命軍]]，擔任第二十九軍副軍長，參加[[北伐|北伐]]。1935年升任第二十九軍[[軍長|軍長]]。1936年2月25日被[[國民政府|國民政府]]授予[[陸軍|陸軍]][[中將|中將]]軍銜。

== 中國抗日戰爭 ==
1937年7月，[[中國抗日戰爭|中國抗日戰爭]]爆發。7月8日，川康軍事整理委員會在重慶召開第二次大會，主任委員[[何應欽|何應欽]]宣布日軍在[[蘆溝橋|蘆溝橋]]挑釁經過，謂中日大戰已不可避免，與會人員聞之憤慨異常，第四十一軍軍長孫震當場請纓，願率所部出川，參加抗戰<ref name="民" >{{cite book |editors=[[李新_(1918年)|李新]]總主編，中國社會科學院近代史研究所中華民國史研究室，韓信夫、姜克夫主編 |title=《中華民國史大事記》 |location=北京 |publisher=[[中華書局|中華書局]] |year=2011 |isbn=9787101079982}}</ref>{{rp|5467}}。孫震與四川省主席[[劉湘|劉湘]]一起出川，擔任第二十二集團軍副司令，在[[太原會戰|太原會戰]]後，1938年升任為第二十二集團軍總司令，後來奉令轉飭第四十一軍守衛[[滕縣|滕縣]]，造就[[滕縣戰鬥_(抗日戰爭)|滕縣戰鬥]]的勝利，也使得之後的[[台兒庄會戰|台兒庄會戰]]與[[徐州會戰|徐州會戰]]獲得勝利。1939年5月，被授予[[陸軍|陸軍]][[上將|上將]]軍銜。之後亦參與了[[隨棗會戰|隨棗會戰]]、[[棗宜會戰|棗宜會戰]]、[[豫西鄂北會戰|豫西鄂北會戰]]等等戰役。

== 國共內戰 ==
{{main|第二次國共內戰}}
1947年5月26日，第五綏靖區奉令與陸軍總司令部鄭州指揮所合併，孫震繼范漢杰為鄭州陸軍指揮所主任<ref name=民 />{{rp|8361}}。[[國共內戰|國共內戰]]期間，孫震曾先後擔任[[徐州剿匪總司令部|徐州剿匪總司令部]]副總司令兼[[徐州剿匪總司令部|徐州剿匪總部]]鄭州指揮部主任、[[華中剿匪總司令部|華中剿匪總司令部]]副總司令、[[第五綏靖區|第五綏靖區]]司令官兼川鄂邊區綏靖公署主任、西南軍政長官公署副長官<ref>長官是[[張群|張群]]</ref>等職。

== 台灣時期 ==
1949年12月，隨[[國民政府|國民政府]]，從[[四川|四川]]到達[[台灣|台灣]]，1952年退役，後長期擔任總統府[[中華民國總統府戰略顧問|戰略顧問]]、[[光復大陸設計研究委員會|光復大陸設計研究委員會]]委員、[[憲政研討委員會|憲政研討委員會]]委員等職務。1985年9月9日病逝於[[台北|台北]]，享年93歲。

== 其他 ==
孫震於1929年於[[四川|四川]][[成都|成都]]成立[[樹德小學|樹德小學]]和[[樹德中學|樹德中學]]。1937年開辦高中。他曾著有《[[八十年國事川事見聞錄|八十年國事川事見聞錄]]》與《[[楙園隨筆|楙園隨筆]]》等書。
