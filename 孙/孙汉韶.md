'''孙汉韶'''（{{bd|884年||955年|8月30日|catIdx=S}}），[[表字|字]]'''享天'''，[[五代十国|五代十国]]时[[晋_(五代)|晋国]]将领[[李存进|李存进]]（本名[[孙重进|孙重进]]）长子。最初效力晋国后身[[后唐|后唐]]，后效力[[后蜀|后蜀]]。

== 生平 ==
本名'''李汉韶'''。幼年有器局，风仪峻整，<ref name=HFD53>[[:s:旧五代史/卷053|《旧五代史》卷五十三]]</ref>李存进尤其喜爱他，<ref name=h9k>《九国志》</ref>推荐给义父唐末[[河东节度使|河东节度使]]晋王[[李克用|李克用]]，李克用录用他充随使军将。[[天祐_(唐朝)|天祐]]初年，转充定海军副兵马使。<ref name=wy>《孙汉韶墓志》</ref>

=== 效力后唐 ===
==== 庄宗年间 ====
天祐四年（907年）春，李汉韶被署为定安（一作安定）军使，<ref name=HFD53/><ref name=h9k/>[[墨制|墨制]]授[[银青光禄大夫|银青光禄大夫]]、检校国子祭酒兼[[御史中丞|御史中丞]]、[[上柱国|上柱国]]。李克用死后，子[[李存勖|李存勖]]继位。十年（913年），李汉韶转五院第五院军使。十四年（917年）以功升第二院兼都知兵马使，加检校左散骑常侍兼御史大夫。十五年（918年），迁河东牢城指挥使，授[[金紫光禄大夫|金紫光禄大夫]]、检校[[兵部尚书|兵部尚书]]。<ref name=wy/>

天祐十九年（922年），李汉韶父李存进在讨伐叛将[[张处球|张处球]]时阵亡。

[[同光|同光]]元年（923年），李存勖建立后唐，灭[[后梁|后梁]]。时[[孟知祥|孟知祥]]权知太原军府事，正逢[[契丹|契丹]]侵前晋北境。孟知祥上表令李汉韶率师进讨，庄宗知其可用，委以军事，于是大破契丹，以功加检校右仆射。三年（925年）十二月，授检校尚书、右仆射、守[[蔡州|蔡州]]刺史。<ref name=HFD53/><ref name=h9k/><ref name=wy/>

==== 明宗年间 ====
[[天成_(后唐)|天成]]初年，李汉韶复姓孙。<ref name=NHFD36>[[:s:新五代史/卷36|《新五代史》卷三十六]]</ref><ref name=SGCQ>《十国春秋》</ref>二年（927年）八月，加竭忠建策兴复功臣，超授检校[[司空|司空]]，仍守蔡州刺史。三年（928年）三月除授检校司徒，充[[彰国军节度使|彰国军节度观风留后]]，<ref name=h9k/><ref>[[:s:旧五代史/卷039|《旧五代史》卷三十九]]</ref>封'''乐安县开国男'''，食邑三百户。四年（929年）三月，[[后唐明宗|后唐明宗]]因家乡在彰国军，追封祖宗，孙汉韶奉诏修庙，得累加[[光禄大夫|光禄大夫]]、检校[[太保|太保]]，<ref name=HFD53/>仍充彰国军留后。四月，因丧母回太原，哀毁过礼，虽然后来起复，但仍披縗麻。<ref name=wy/>

[[长兴_(后唐)|长兴]]二年（931年）秋，孙汉韶服丧期满入朝，受任充西面行营步军都指挥使。三年（932年）正月，仍除授检校太保，遥授[[昭武军节度使|昭武军节度使]]（在已背叛朝廷的时任[[西川节度使|西川节度使]]孟知祥控制下），充西面行营副部署兼步军都指挥使，受命参与讨伐[[东川节度使|东川节度使]][[董璋|董璋]]，并迁封'''开国子'''，加食邑二百户。八月，兼西面行营招讨使。<ref>[[:s:旧五代史/卷043|《旧五代史》卷四十三]]</ref>四年（933年）六月，移授[[武定军节度使|武定军节度使]]兼西面诸州本城屯驻马步军副都部署，<ref name=HFD53/><ref name=h9k/><ref name=NHFD36/><ref name=SGCQ/><ref>[[:s:旧五代史/卷044|《旧五代史》卷四十四]]</ref>封'''开国伯'''，加食邑二百户，改赐耀忠匡定保节功臣。<ref name=wy/>

==== 闵帝年间 ====
[[后唐闵帝|后唐闵帝]]嗣位，[[应顺|应顺]]元年（934年）正月，转孙汉韶检校[[太傅|太傅]]，迁封'''开国侯'''，加食邑三百。<ref name=wy/>又加孙汉韶[[特进|特进]]，孙汉韶因该官职犯其父名讳，上表辞让，改检校左仆射。诏书称：“改[[会稽|会稽]]之字，抑有前闻；换瑰宝之文，非无故事（举的都是避父讳的例子）<ref>[[东晋|东晋]][[王舒|王舒]]因父名[[王会|王会]]，不就任会稽太守，于是朝廷改会稽为郐稽；[[唐玄宗|唐玄宗]]年间[[中书舍人|中书舍人]][[萧嵩|萧嵩]]奉命起草的任[[苏頲|苏頲]]为[[宰相|宰相]]诏书有“国之瑰宝”语，玄宗指出此语犯了苏頲父[[苏瑰|苏瑰]]名讳，命萧嵩改之，萧嵩改为“国之珍宝”。</ref>。”<ref name=h9k/>同月，孙汉韶上言于雒谷路造仓舍。<ref>《[[册府元龟|册府元龟]]》</ref>

=== 效力后蜀 ===
==== 高祖年间 ====
闵帝养兄[[凤翔节度使|凤翔节度使]]潞王[[李从珂|李从珂]]起兵反抗朝廷，三月，孙汉韶与[[山南西道节度使|山南西道节度使]][[张虔钊|张虔钊]]、[[护国节度使|护国节度使]][[安彦威|安彦威]]、[[彰义节度使|彰义节度使]][[张从宾|张从宾]]、[[静难节度使|静难节度使]][[康福|康福]]等各率本部兵在岐山下会合朝廷军队。孙汉韶到山南西道军部兴元，与张虔钊同议进兵，并上奏此事。<ref>[[:s:旧五代史/卷045|《旧五代史》卷四十五]]</ref>张虔钊留孙汉韶守兴元。后来朝廷西征凤翔的军队除张虔钊、孙汉韶的军队外都叛变了，<ref name=NHFD36/>四月，二人各自逃归本镇。<ref name=HFD53/><ref name=SGCQ/><ref name=ZZTJ279>[[:s:资治通鉴/卷279|《资治通鉴》卷二百七十九]]</ref>

不久，李从珂夺取皇位，张虔钊、孙汉韶闻讯心不自安。张虔钊以治所归附当时已为后蜀皇帝的孟知祥，<ref>[[:s:旧五代史/卷074|《旧五代史》卷七十四]]</ref>孙汉韶因而与后唐道路阻隔，也以本镇归附孟知祥。<ref name=NHFD64>[[:s:新五代史/卷64|《新五代史》卷六十四]]</ref>孟知祥派右匡圣马步都指挥使、[[宁江节度使|宁江节度使]][[张知业|张知业]]率兵一万屯大漫天相迎。五月，孙汉韶、张虔钊举族迁往后蜀都城[[成都|成都]]，孟知祥因他是故人，尤其善待。孙汉韶与孟知祥叙说当初在太原的旧事，及导致庄宗丧生的[[邺都兵变|邺都兵变]]，面对面感泣。孟知祥说：“丰沛故人（指[[汉高祖|汉高祖]]回乡遇故人之典），相遇于此，何乐如之！”于是赐第宅金帛，供帐什物都由官府供应。七月，下诏赐孙汉韶为安时顺国全节功臣，仍光禄大夫、检校太傅，封'''开国公'''，加食邑五百户，署为[[永平军节度使|永平军节度使]]。<ref name=HFD53/><ref name=h9k/><ref name=wy/><ref name=NHFD36/><ref name=SGCQ/><ref name=ZZTJ279/>

因孙汉韶归蜀，其弟东面马步军都指挥使[[孙汉殷|孙汉殷]]（史书作[[孙汉英|孙汉英]]）、孙汉筠受到牵连，长期不得调任。<ref>[[:s:旧五代史/卷129|《旧五代史》卷一百二十九]]</ref>

==== 后主年间 ====
[[孟昶|孟昶]]嗣位后，[[明德_(后蜀)|明德]]二年（935年）正月，加孙汉韶[[开府仪同三司|开府仪同三司]]、[[同平章事|同平章事]]，加食邑五百户，改赐匡国奉圣叶力功臣。三年（936年），入为匡圣马步军都指挥使。<ref name=h9k/>四年（937年）三月，充右匡圣马步军都指挥使。[[广政|广政]]元年（938年）正月，除授昭武军节度使，加食邑五百户，食实封一百户。二年（939年）二月，后主将妹妹[[兰英长公主|兰英长公主]]嫁给孙汉韶长子[[孙晏琮|孙晏琮]]。七月，转授孙汉韶山南西道节度使。<ref name=NHFD36/>四年（941年），任满回朝。<ref name=zyl>[[朱玉龙|朱玉龙]]《[[五代十国方镇年表|五代十国方镇年表]]》</ref>七年（944年），加检校太尉兼[[中书令|中书令]]，增食邑五百户，食实封一百户。六月，再授山南西道节度使。<ref name=HFD53/><ref name=wy/><ref name=SGCQ/><ref>《九国志》误作广政十年事。</ref>

广政十年（947年）三月，时值[[辽朝|辽]]灭[[后晋|后晋]]、治所在[[秦州|秦州]]的晋[[雄武军节度使|雄武军节度使]][[何重建|何重建]]归附后蜀，[[凤州|凤州]]防御使[[石奉頵|石奉頵]]坚守，不接纳后蜀使者。翰林承旨[[李昊|李昊]]对[[枢密使|枢密使]][[王处回|王处回]]说：“敌（后晋余部）复据固镇，则[[兴州|兴州]]道路断绝，不复能救秦州了。请遣山南西道节度使孙汉韶率兵急攻凤州。”孟昶命孙汉韶去凤州行营。孙汉韶率兵二万攻凤州，屯于固镇，分兵扼守[[散关|散关]]以绝援军道路。<ref>[[:s:资治通鉴/卷286|《资治通鉴》卷二百八十六]]</ref>石奉頵投降，后蜀达到[[前蜀|前蜀]]疆域，孙汉韶有功。<ref name=NHFD64/>孟昶派孙汉韶入凤州宣慰，号令严肃，秋毫无犯。孙汉韶驻扎数日后回去了。<ref name=h9k/>七月，转授[[武信军节度使|武信军节度使]]。<ref name=HFD53/><ref name=NHFD36/>次年卸任。<ref name=zyl/>不久获授左匡圣马步军都指挥使。<ref name=wy/><ref name=SGCQ/>

七月，时任右匡圣都指挥使孙汉韶与[[司空|司空]]兼[[中书侍郎|中书侍郎]]、[[同平章事|同平章事]][[张业_(后蜀)|张业]]（即张知业，避讳孟知祥改名）有隙，密告张业及其子检校左仆射[[张继昭|张继昭]]谋反。李昊与奉圣控鹤马步都指挥使[[安思谦|安思谦]]也说张业谋反。孟昶趁张业入朝，命壮士在都堂击杀之，下诏暴其罪恶，抄没其全部家产。<ref>[[:s:资治通鉴/卷288|《资治通鉴》卷二百八十八]]</ref>

广政十三年（950年）正月，孙汉韶奏请为孟昶上徽号。册礼后，孙汉韶获加守[[太保|太保]]，改赐匡时翊圣推忠保大功臣。十四年（951年）正月，转充捧圣控鹤都指挥使。<ref name=wy/><ref name=SGCQ/>同年，因当初夺取凤州之功，加检校[[太师|太师]]、雄武军节度使。<ref name=h9k/>

广政十七年（954年）三月，孙汉韶因病请求解除兵权，加武信军节度使，<ref>《十国春秋》误作十八年事。</ref>赐爵'''乐安郡王'''，罢军职，<ref>[[:s:资治通鉴/卷291|《资治通鉴》卷二百九十一]]</ref>加守太傅，仍兼中书令。后不久，获赐抬肩舆入见。次年八月，卒于成都县武檐坊私宅。<ref name=HFD53/><ref name=h9k/><ref name=NHFD36/><ref name=SGCQ/>孟昶闻讯哭泣，辍朝七日，赐使持节行礼，追赠太尉、梁州牧，谥'''忠简'''。十二月，以本官仪卫葬于华阳县升仙乡贸仙里原。<ref name=wy/>

== 家族 ==
孙汉韶的高祖父为振武节度都押衙、银青光禄大夫、检校右散骑常侍、兼御史大夫孙岩，曾祖父为[[云州|云州]]别驾、金紫光禄大夫、守[[胜州|胜州]]刺史、检校刑部尚书兼御史大夫、柱国孙勤，祖父[[岚州|岚州]][[刺史|刺史]]、振武节度都押衙左校练使、银青光禄大夫、检校左散骑常侍兼御史大夫、上柱国、赠[[司徒|司徒]][[孙昉|孙昉]]（《旧五代史·李存进传》作[[孙佺|孙佺]]）。嫡母彭城刘氏，未详是否孙汉韶生母。<ref>[[:s:后唐招讨使李存进墓碑|吕梦奇《后唐招讨使李存进墓碑》]]</ref>

== 墓葬 ==
孙汉韶墓位于成都市城北驷马桥以北约1.5公里，西北距凤凰山园林3公里，北邻[[川陕公路|川陕公路]]。1984年3月，四川省商业厅基建安装公司在成都市金牛区青龙乡西林村发现孙汉韶墓葬。6月下旬，市文物管理处完成墓葬清理。该墓曾被盗，随葬器物很少保存完整，陶器仅40%左右可修复，但墓志基本保存完好。

墓志盖篆书《大蜀故守太傅乐安郡王赠太尉梁州牧赐谥忠简孙公内志》，墓志题为《大蜀故匡时翊圣推忠保大功臣、武信军节度、遂合渝泸昌等州管内观风营田处置等使、开府仪同三司、守太傅兼中书令、使持节遂州诸军事、守遂州刺史、上柱国、乐安郡王、食邑三千户、食实封二百户、赠太尉、梁州牧、赐谥忠简孙公内志》，由其门吏前遂合渝泸昌等州管内观风支使、[[将仕郞|将仕郞]]兼监按御史、赐绯鱼袋王乂撰写、前摄保胜军团练巡官、将仕郞、试秘书省秘书郎白守谦书写及篆刻。

== 评价 ==
*《[[九国志|九国志]]》：孙汉韶忠勇有父风，其持重敦厚又胜过父亲，在历任藩镇上都有政绩。他为自保而投后蜀，为后蜀尽股肱之力，保全富贵，其敦厚诚实为有识者所称道。<ref name=h9k/>
*《[[十国春秋|十国春秋]]》论曰：孙汉韶、张虔钊皆后唐忠荩之臣，兵溃来归，一则赐爵安乐，一则忿懑丧躯，成与不成，命也。<ref name=SGCQ/>
