'''孙思邈'''（{{bd|541年或581年||682年||S孫}}），[[唐朝|唐朝]]京兆华原（现[[陕西|陕西]][[耀县|耀县]]）人，是著名的[[醫師|醫師]]與[[道士|道士]]。他是[[中国|中国]]乃至世界史上著名的医学家和药物学家，被誉為'''药王'''，[[宋朝|宋朝]]追封'''妙应[[真人|真人]]'''，[[道教|道教]]尊為'''天醫妙應廣援善濟真君'''，許多[[華人|華人]]奉之為[[醫神|醫神]]。

== 生平 ==
据《[[新唐书|新唐书]]》载，他出生于[[西魏|西魏]]，七岁就学，日诵千余言，[[北周|北周]]元老[[独孤信|独孤信]]称他为圣童。他精通百家学说，善于谈[[老子|老子]]、[[庄周|庄周]]之道，擅长[[阴阳|阴阳]]、推步，妙解数术，兼好《[[佛經|释典]]》。

孙思邈从小多病，“汤药之资，罄尽家产”。周围的百姓都很贫苦，很多人因为没钱治病而死去，因此他18岁时立志学医，终身勤奋不辍。他认为“人命至重，有贵千金。一方济之，德逾于此”<ref>《[[千金要方|千金要方]]》自序</ref>。十八岁之后就“志于学医”，最后终有所成。

相传，[[周宣帝|周宣帝]]，[[周静帝|周静帝]]，[[隋文帝|隋文帝]]、[[唐太宗|唐太宗]]、[[唐高宗|唐高宗]]曾以高官厚爵與之，都被他婉拒，孙思邈曾经作过[[唐太宗|唐太宗]]的[[御医|御医]]，[[上元|上元]]元年他称病退隐山中，[[唐高宗|唐高宗]]以良马賜之。[[卢照邻|卢照邻]]奉他为师。

== 思想 ==

孙思邈著有《[[备急千金要方|备急千金要方]]》，简称《千金要方》，共三十卷，二百三十二门，已接近现代[[临床医学|临床医学]]的分类方法。他汲取《[[黄帝内经|黄帝内经]]》关于脏腑的学说，在《千金要方》中第一次完整地提出了以脏腑寒热虚实为中心的杂病分类辨治法，收集药方五千三百首。《[[千金翼方|千金翼方]]》是对《千金要方》的补编，共有三十卷{{r|周|page=52}}。

孙思邈重视医德修养，详论医德规范。他特別强调医家的职业道德。特别是《备急千金要方》中的“[[大医习业|大医习业]]”和“[[大医精诚|大医精诚]]”两篇，系统的论述了医德规范。其内容大致可归纳为两个方面：第一是技术要精湛，第二是品德要高尚。《千金要方》卷首以显著地位论述了《[[大医精诚|大医精诚]]》与《[[大医习业|大医习业]]》，把医德放在了极其重要的位置上，强调为人治病，不分“贵贱贫富，长幼妍蚩，怨亲善友，华夷愚智”，皆一视同仁。还强调以德养性、以德养身。较之汉晋医家，孙思邈对医德的论述，可以说是最全面的了。而这些基本医疗道德，至今仍具有重要的现实意义。

他重视前人的宝贵经验，但尊古而不泥古。他将《伤寒论》的内容较完整地收集在《千金翼方》之中。在他一生行医的过程中，他十分重视妇人与儿童疾病的诊治，他认为这是关系人类繁衍的大事，故曰：“先妇人小儿……则是崇本之意也。”在针灸学方面，他不仅强调针药并用，还创设了新的穴位，创制彩色经络图，还常配合按摩、灸治。同时，他还是食疗治法的积极倡导者。他反对魏晋时期盛行的服石求长生的风气，倡导积极养生，强身长寿。

孙思邈将[[道家|道家]]内修理论和医学、卫生学相结合，把[[养生学|养生学]]也作为医疗内容。认为人若善摄生，当可免于病。只要“良医导之以药石，救之以针剂”，“体形有可愈之疾，天地有可消之灾”。孙思邈还对[[炼丹|炼丹]]、[[服食|服食]]以求长生成仙的道教方术作了探索。

== 信仰 ==
[[File:Statue_of_Sun_Simiao_in_Jintai_Taoist_Temple_2012-09.JPG|thumb]]内的药王像，牌位为“药王广援妙济真君之位”。]]
临终时，孙思邈遗嘱“薄葬，不藏明器，祭去牲牢”，並遗有大量的医书，人尊其為「'''孫藥王'''」。在[[陕西|陕西]][[耀县|耀县]][[药王山|药王山]]（原稱五臺山），有祭祀孫思邈的药王庙{{r|周|page=52}}，并在每年农历[[二月初二|二月初二]]有「二月二药王庙会」。此外在耀縣還建有「真人祠」、聖母殿與祖塋碑{{r|周|page=52}}。[[宋徽宗|宋徽宗]][[崇宁|崇宁]]二年（1103）[[追封|追封]]'''妙应[[真人|真人]]'''。

今[[台灣|台灣]]仍有祭祀孫思邈之廟宇<ref>{{cite web|url=http://www.baoantemple.com.tw/god2.html|title=大潭保安宮|accessdate=2013-08-29|language=zh-tw}}</ref>，並稱之「'''天醫妙應孫真人'''」，每年農曆正月初四為其祭典，稱之為「孫天醫聖誕」。
== 保生三真人信仰 ==
在[[保生大帝|保生大帝]]信仰中，有所謂的「保生三真人」，認為孫思邈與[[保生大帝|吳夲]]、[[許遜|許遜]]在成神之後成了[[結拜|結拜]]兄弟{{r|周|page=55}}。在[[廈門|廈門]]一帶認為三神排行依序是大哥「金面」孫真人、老二「紅面」吳真人、老三「黑面」許真人{{r|周|page=55}}，但在[[臺灣|臺灣]]其排行則因地而異{{r|周|page=63}}，[[閭山派|閭山派]]的廟宇則是依照「玅濟」許真人、「善濟」孫真人、「慈濟」吳真人的順序排列。

== 傳說 ==
*醫虎喉：據說孫思邈一次夜間出診的時候，遇到一隻似乎要向他求救的老虎。他仔細一看發現老虎被人骨骾住，便幫老虎治療取出骨頭。之後老虎為了報恩而成為他的侍者。{{r|周|page=52}}
*開棺救活產婦：傳說孫思邈一次出診偶遇出殯隊伍，發現棺材滲出鮮血，乃請求開棺並救活棺內難產的婦人與腹中的嬰兒。{{r|周|page=52}}

== 成語：爐火純青 ==
一般解說「[[爐火純青|爐火純青]]」這句成語，都以為是道家煉丹完成時，爐火發出純青火焰，由此延伸而用來比喻學問、技術、功夫到達精純完美的境地。比較早的文獻證據是[[唐代|唐代]]孫思邈所寫的一首[[四言詩|四言詩]]，收錄在《[[全唐詩|全唐詩]]》中。[[孫思邈|孫思邈]]通[[百家|百家]]、[[陰陽|陰陽]]、[[醫學|醫學]]之術，他這首詩主要是描述[[煉丹|煉丹]]的過程。詩中大意是說：煉丹時，將鉛汞置入丹鼎中，點燃爐火。紅光烈火，所冒之煙，已無黑色，火焰變成純正的藍光。許多成語典以為[[孫思邈|孫思邈]]這裡說的就是煉丹完成時的「爐火純青」現象。由此可知，「爐火轉青」時，正是煉丹成就時，因此可以用來比喻[[學問|學問]]、[[技術|技術]]、[[功夫|功夫]]等事情到達[[精純完美|精純完美]]的境地。
