'''孙笑侠'''（1963年8月生），[[浙江|浙江]][[温州|温州]]人，法学教授、[[博士生导师|博士生导师]]，[[中国法理学研究会|中国法理学研究会]]副会长。享受国务院特殊津贴（2001年），主要研究领域是[[法理学|法理学]]、宪法学和比较法学<ref name="介绍">{{cite web |url=http://www.chinalegalmethod.com/contents/10/569_v.aspx |language= |title=孙笑侠 |publisher=法律方法和法律思维网 |date=2010-12-22 |accessdate= |archive-url=https://web.archive.org/web/20160306212428/http://www.chinalegalmethod.com/contents/10/569_v.aspx |archive-date=2016-03-06 |dead-url=yes }}</ref>。

== 生平 ==
1984年获得[[杭州大学|杭州大学]]法学学士；1990年取得[[武汉大学|武汉大学]]法学硕士学位；1998年至2000年在[[中国社会科学院|中国社会科学院]]学习，获[[法学博士|法学博士]]<ref name= "介绍"/>。

1984年至2010年担任杭州大学、[[浙江大学|浙江大学]]法律系讲师、副教授、教授、硕士生导师、博士生导师、[[浙江大学光华法学院|光华法学院]]副院长、院长[http://bbs.netbig.com/archiver/?tid-35089]。

2010年至今担任[[复旦大学|复旦大学]]特聘教授、博士生导师、[[复旦大学法学院|法学院]]院长<ref>{{cite web |url=http://wq.zfwlxt.com/newLawyerSite/blogshow.aspx?user=186561&itemid=307621b3-39cf-47f8-ae49-a0c001213711 |language= |title=复旦大学：孙笑侠教授简介 |publisher=法治天下网 |date=2012-09-02 |accessdate= }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>[http://bbs.sinovision.net/forum.php?mod=viewthread&tid=307823]。

== 著作 ==
主要著作：
*《法的现象与观念》（1995群众出版社初版，2001修订版）
*《法律对行政的控制》（1999年山东人民出版社版）
*《'''法理学'''》（主编，中国政法大学出版社，1996年版）。

此外，在《[[法学研究|法学研究]]》、《[[中国法学|中国法学]]》等刊物发表《法治国家及其政治构造》（1998）、《法律家的技能与伦理》（2001）等70余篇学术论文<ref>[http://www.edu1488.com/flxr/show.asp?pkno=87 个人简介]</ref>。
