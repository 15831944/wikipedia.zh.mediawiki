'''孙笑川'''（{{bd|1990年|5月12日}}），[[新浪微博|新浪微博]]用户名'''带带大师兄'''，有'''孙狗'''、'''孙哥'''、'''mata川'''、'''网恋教父'''等称呼，是来自中国[[四川省|四川省]][[成都市|成都市]][[新津县|新津县]]的一名前电子游戏[[网络主播|网络主播]]，原[[斗鱼TV|斗鱼TV]]抽象工作室成员，因一系列争议事件被中国内地多个直播平台封杀，之后改在[[Twitch|Twitch]]直播，中国国内粉丝群转而活跃于[[新浪微博|新浪微博]]上。因其常与观看直播的粉丝以及一起玩游戏的玩家发生言语冲突，且言语较有节目效果，故引发了「嗨粉」这一[[亚文化|亚文化]]和相关[[网络迷因|网络迷因]]。

== 生平 ==
孙笑川自称在中國四川省成都市新津縣的一个[[单亲家庭|单亲家庭]]长大，跟随卖零食、童装的母亲生活。中学时期的孙笑川性格较为内向，在学校里面并不突出，成绩也并不是很优秀，喜欢在课余时间打电子游戏。2008年，时为高三文科生的孙笑川参加中国[[普通高等学校招生全国统一考试|普通高等学校招生全国统一考试]]，英语成绩比较优秀，但其他科目成绩不佳或一般，最后被[[双流区|双流区]]一所[[专科学校|专科学校]]录取。大专毕业后，孙笑川就职为安全监理员，在各个建筑工地工作，负责巡视监察工作。对于此份工作，孙笑川觉得较为平淡，没有激情。2015年9月，在好友[[李赣|李赣]]的推荐下，孙笑川辞职转行到游戏直播行业，因为他觉得游戏直播行业较为有趣且难度不高，收入也不低。<ref name="胡正达">{{cite web |author1=胡正达 |title=新津青年孙笑川 |url=http://www.chuapp.com/article/285712.html |website=触乐 |accessdate=2019-01-29 |date=2018-09-13 |archive-url=https://web.archive.org/web/20190126220854/http://www.chuapp.com/article/285712.html |archive-date=2019-01-26 |dead-url=no }}</ref>

== 成名历程 ==
{{external media|
|video1=[https://www.youtube.com/watch?v=_8nCB4oTWPM 「抽象圣经」]

2015年9月1日，孙笑川的好友李赣聚集召集了另外三名游戏主播图图、于超和啊不多，组建了抽象工作室，孙笑川作为抽象工作室的成员，也开始正式成为斗鱼TV6324直播间的一名游戏主播，主要直播[[多人在线战斗竞技场游戏|多人在线战斗竞技场游戏]]《[[英雄联盟|英雄联盟]]》。在工作室成立之前，孙笑川虽然没有正式进入游戏直播行业，但他时不时会到李赣家里进行直播客串。在直播过程中，孙笑川经常边打游戏边和观众闲聊一些关于自己的生活经历，例如自己[[网恋|网恋]]却被骗8000元[[人民币|人民币]]等。他也会在游戏中和队友发生争执和言语冲突，甚至争执的时间占据了游戏的大部分。这吸引了不少粉丝，这些粉丝也自称为「狗粉丝」或「嗨粉」，后者即[[四川话|四川话]]「黑粉」的[[諧音_(語言學)|谐音]]。<ref name="阿蟹大人">{{cite web |author1=阿蟹大人 |author2=胡正达 |title=抽象的人:斗鱼6324、嗨粉 以及一个时代的终结和开始 |url=http://tech.sina.com.cn/csj/2018-08-01/doc-ihhacrce6780320.shtml |website=新浪科技创事记 |publisher=触乐 |accessdate=2018-12-13 |date=2018-08-01 |archive-url=https://web.archive.org/web/20180801042227/http://tech.sina.com.cn/csj/2018-08-01/doc-ihhacrce6780320.shtml |archive-date=2018-08-01 |dead-url=no }}</ref>

某日孙笑川在直播时，他好友列表的一位用户私聊孙笑川并对孙笑川的粉丝进行辱骂，孙笑川表示，如果直播间的粉丝赠送礼物「火箭」，他便删除那位辱骂粉丝的好友。但在粉丝送出礼物后，孙笑川并没有履行诺言，这引起了粉丝的不满，于是嗨粉们开始把一些和孙笑川无关的罪行嫁祸到孙笑川身上，例如造谣其使用[[双节棍|双节棍]]殴打祖母等事件，在新浪微博的热门事件下，也常有嗨粉留言「某某事件的凶手的微博找到了@带带大师兄」。<ref name="胡正达" /><ref>{{cite web |title=hape孙笑川什么意思 主播孙笑川说的哈皮是骂人的话吗 |url=http://www.mnw.cn/news/ent/xx/2043399.html |website=闽南网  |accessdate=2018-12-13 |date=2018-08-08}}</ref>

2017年7月18日，北京钧丰至律文化科技有限公司旗下的微博帐号乐范音乐秀在新浪微博上发表推文介绍《[[中国有嘻哈|中国有嘻哈]]》的选手的真实姓名，但误将嘻哈饶舌歌手[[孙八一|孙八一]]的真实姓名写成孙笑川。网友便利用这次失误，将孙笑川称为「'''成都地下说唱皇帝'''」。<ref>{{cite web |author1=差评 |title=网络上流传的梗大多是怎样流传起来的？ |url=http://www.sohu.com/a/251304682_355019 |website=搜狐 |accessdate=2018-12-13 |date=2018-09-01}}</ref><ref>{{cite web |author1=科技唆麻 |title=“好嗨哟”背后的西南方言如何占领互联网 |url=http://tech.sina.com.cn/csj/2019-01-08/doc-ihqfskcn5021034.shtml |website=新浪科技创事记 |accessdate=2019-01-29 |date=2019-01-08 |archive-url=https://web.archive.org/web/20190108052920/http://tech.sina.com.cn/csj/2019-01-08/doc-ihqfskcn5021034.shtml |archive-date=2019-01-08 |dead-url=no }}</ref>

2017年8月23日，孙笑川在直播《英雄联盟》时和往常一样表情严肃，有不少观看直播的观众便通过[[视频弹幕|视频弹幕]]发言，称孙笑川这种表情是「死妈脸」，即亲人去世后哀伤忧郁的表情。孙笑川对此很愤怒，回应称自己过于劳累，所以表情严肃，是人的正常生理现象。随后又跟观众发生长达五分钟的言语冲突，围绕「死妈脸」这个主题回击观众，呵斥乱发视频弹幕的观众离开直播间。他表示自己为人儒雅随和，对网友的攻击很不满，且称网友只攻击他一人而不去攻击其他网络主播有失公正。这段夹杂这四川话的言语冲突由于较有特色，所以成为了一个网络迷因，被网友称为「抽象圣经」，并录制上传到[[Bilibili|Bilibili]]、[[YouTube|YouTube]]、[[Pornhub|Pornhub]]等视频分享网站上。但由于视频中孙笑川使用的言辞较为激烈，违反了中国大陆一些视频网站的[[使用条款|使用条款]]，故被网站管理员删除。而网友在使用文字复述孙笑川的这段话语时，也有很多不文明词汇触发[[關鍵詞過濾|關鍵詞過濾]]系统而无法成功发布，故网友使用了[[emoji|emoji]]来代替这些敏感词，如用表示马的emoji「🐎」或「🐴」来代替发音相近的「妈」字，用表示啤酒的emoji「🍺」来代替四川口音的表示[[女性生殖器官|女性生殖器官]]的「[[屄|屄]]」字等。类比于抽象圣经，这种用法也被网友称为「抽象话」。<ref name="阿蟹大人" />

而有一次直播时，孙笑川的邻居指责他直播过于吵闹影响休息，并要求其关窗，但孙笑川认为邻居态度不友好，且自己没有关窗的义务，让邻居去找[[物业管理|物业管理]]人员投诉，随后便于邻居发生言语争吵，且越来越激烈。这一段视频也成为类似于「抽象圣经」的另一个嗨粉亚文化素材，不少网友开始模仿孙笑川在这段视频中的语录，并将视频上传到一些视频分享网站上。<ref>{{cite web |title=那你去找物管啊是什么梗 孙笑川和邻居的互动 |url=http://www.1272.cn/q/256768016_100023676.shtml |website=天天快报 |accessdate=2018-12-14 |date=2018-09-28}}</ref>

2018年1月13日上午，孙笑川在[[虎牙直播|虎牙]]直播间播放了一首歌曲，其中歌词是由粉丝由[[法轮功|法轮功]]宣传词改编而来的一篇[[嵌字文|嵌字文]]。但孙笑川一开始并未发现，并称赞了这首歌曲，在此同时，粉丝对孙笑川的直播间进行举报，导致直播间因违反法律法规被永久关闭，他本人也被禁止在中国大陆的网络直播平台上直播。<ref>{{cite web |title=十万观众面前公开宣传邪教！虎牙孙笑川成当代李洪志？ |url=http://wy.92wy.com/news/8/0010138510161.html |website=万宇在线 |accessdate=2018-12-13 |date=2018-01-17}}</ref>

被封禁之后，孙笑川进入失业状态，对粉丝的这次举动感到非常失望。他尝试放弃直播行业去找一份新工作，但因为工资不符合他的预期要求而没有成功，同时他也没有重操旧业回工地的打算，于是他开始转到[[Twitch|Twitch]]平台进行直播。由于Twitch面向国际，并不是中国大陆的本土网络直播平台，所以中国大陆用户较少，人气较低。孙笑川通过在直播界面张贴[[支付宝|支付宝]]收款[[QR碼|QR碼]]的方式来获取观众的捐赠。不久后，有网友利用了支付宝输错密码次数过多会冻结帐号以保障安全这个特性，故意输错孙笑川的支付宝密码，使得孙笑川无法使用支付宝中的余额和[[花呗|花呗]]，现实生活受到影响。孙笑川对此感到委屈和无奈。<ref name="胡正达" />

2018年2月9日，孙笑川宣布正式退出直播行业，不再直播。<ref>{{cite web |title=带带大师兄宣布退出直播界 从此不见孙笑川！ |url=http://mp.163.com/v2/article/detail/DA9IQTTG0517DMLL.html |website=网易号 |publisher=52PK游戏网 |accessdate=2018-12-13 |date=2018-02-10}}</ref>但他仍保持在新浪微博上活跃，也作为嘉宾出席过几次[[漫展|漫展]]。<ref name="胡正达" />

2018年5月14日，孙笑川在网易cc复出首播，和另一位主播“远洋君”进行直播，但首播后就没在直播里露过面，5月29日，孙笑川表示“种种原因”无法再进行直播。在与触乐网的采访中，孙笑川称“种种原因”是有人对他的直播进行了大量举报。平台曾试图助他解封，却无能为力。<ref name="胡正达" />

孙笑川看不到解封的希望，决定转型幕后。2018年6月，他开始筹划招募直播员，成立新的直播工作室。8月，孙笑川招募的4位主播《带带带公寓》在网易CC平台开播。<ref name="胡正达" />

== 影响 ==
[[File:NM$L_logo_inspired_by_Supreme.png|150px]]。]]
孙笑川的语言风格引发了「嗨粉」、「狗粉丝」这一网络迷因。与[[膜蛤文化|膜蛤文化]]、[[六学_(网络迷因)|六学]]等发源于中国大陆的网络迷因类似，嗨粉文化体现在嗨粉模仿孙笑川的语录，如「你就是个弟弟」<ref>{{cite web |title=你就是个弟弟是什么梗 你就是个dd内涵意思、表情图 |url=http://www.mnw.cn/keji/internet/2090496.html |website=闽南网  |accessdate=2018-12-13 |date=2018-11-27}}</ref>、「你妈死了」、「安排上了」等等。除此之外，嗨粉常常在新浪微博等平台的热门新闻评论区，把和孙笑川无关的一些罪名加在他头上，如谎称其是日本人且参拜[[靖国神社|靖国神社]]、举报[[腾讯|腾讯]][[WeGame|WeGame]]平台上的《[[怪物猎人|怪物猎人]]》致其下架、到[[广州|广州]]观看[[蔡徐坤|蔡徐坤]]的演唱会时用[[激光笔|激光笔]]照射他的眼睛等等。<ref>{{cite web |author1=刺猬公社 |title=狗粉丝：出圈的梗和失控的狂欢 |url=https://36kr.com/p/5149017.html |website=36氪 |accessdate=2018-12-13 |date=2018-08-20}}</ref>面对嗨粉的这些行为，孙笑川认为一些[[哏|哏]]很有趣，但也有一些嗨粉没有把握好尺度，发表了一些涉及到政治和色情的违反法律的言论，并不可取。<ref>{{cite web |author1=洪岳 |title=我们找到了出现在每条微博热评底下的 “带带大师兄”，还把他给安排了 |url=http://www.vice.cn/read/nmsl-is-a-subculture-in-china-now |website=Vice中国 |accessdate=2019-01-14 |date=2018-05-22 |archive-url=https://web.archive.org/web/20190127034944/http://www.vice.cn/read/nmsl-is-a-subculture-in-china-now |archive-date=2019-01-27 |dead-url=no }}</ref>面对一些嗨粉较为出格的行为，孙笑川也曾出面制止。如2018年7月起，由[[日本漫画|漫画]]改编的[[日本动画|日本动画]]《[[工作细胞|工作细胞]]》上映，受到很多日本动漫爱好者的喜欢，特别是其中的[[萝莉|萝莉]]角色[[血小板_(工作细胞)|血小板]]。而有一些嗨粉则伪装成《工作细胞》的爱好者，到[[百度贴吧|百度贴吧]]的「血小板吧」对[[血友病|血友病]]患者进行人身攻击并制造矛盾。对此，孙笑川认为此事不当，便发布微博请求嗨粉停止这一不当举动。<ref>{{cite web |author1=海蒂 |title=既在场又不在场的带带大师兄，拥有社交网络海洋里最棒的泳姿 |url=http://www.vice.cn/read/vbest-03-daidai-dashixiong-sun-xiaochuan |website=VICE中国 |accessdate=2019-01-30 |date=2018-12-16}}</ref>

此外，孙笑川的相关素材也被制作成一些[[音MAD|鬼畜]]视频。也有视频制作者通过使用视频剪辑软件，把《[[致青春·原來你還在這裡|致青春·原來你還在這裡]]》等电影中人物的脸替换成孙笑川的脸，并把人物的声音替换为孙笑川的声音，制作成新的搞笑视频。<ref>{{cite web |title=孙笑川，狗粉丝与这个时代的孤独狂欢 {{!}} 亚文话 |url=https://news.baidu.com/share/detail/8655471222738595499 |website=百度新闻 |publisher=华云头条 |accessdate=2018-12-13 |date=2018-09-10}}</ref>

由于嗨粉文化的传播，孙笑川和一个粉丝合作创建了NMSL[[潮牌|潮牌]]，名称来源于他的[[口头禅|口头禅]]「你妈死了」的[[汉语拼音|汉语拼音]]首字母缩写，有时也常使用[[美元符号|美元符号]]「$」代替英文字母「S」，写成「NM$L」，主要设计生产一些印有孙笑川语录及相关延伸元素的[[文化衫|文化衫]]、[[手机壳|手机壳]]等产品，通过[[淘宝|淘宝]]发售。孙笑川表示NMSL潮牌除了作为停止直播以后的一个收入来源之外，还起到一个推动亚文化的作用。<ref>{{cite web |title=主理人孙笑川讲述创办NM$L潮牌的故事，以及何时回国还有劳，卷，目三人为何可以直播。 |url=https://www.bilibili.com/video/av21190990/ |website=Bilibili |publisher=Twitch |accessdate=2019-01-29 |date=2018-03-24}}</ref><ref>{{cite web |title=国潮崛起的希望找到了，原来是孙笑川的NMSL！ |url=http://fashion.ifeng.com/a/20180928/40430973_0.shtml |website=凤凰网时尚 |publisher=一点资讯 |accessdate=2018-12-13 |date=2018-09-30}}</ref><ref>{{cite web |title=“国潮第一人”专访：人人都说抽象话，无人再识孙笑川 |url=http://h5.china.com.cn/wm/doc_1_80781_5181937.html |website=中国网 |publisher=中青网 |accessdate=2019-01-29 |date=2019-01-16}}</ref>

== 评价 ==
2018年8月28日，《[[光明日报|光明日报]]》针对孙笑川和嗨粉文化发表评论，批评了嗨粉文化，批评孙笑川没有管理好自己的粉丝，把嗨粉文化定义为新型网络暴力。对此孙笑川回应，自己一向遵守法律法规，也反对人身攻击。<ref>{{cite web |author1=颜维琦 |title=【光明时评】“狗粉丝”背后的新型网络暴力亟需遏制 |url=https://xw.qq.com/cmsid/20180828A0C1WK00 |website=腾讯网 |publisher=[[光明日报|光明日报]] |accessdate=2018-12-13 |date=2018-08-28}}</ref><ref>{{cite web |author1=洞察工程车 |title=网络第一背锅侠！为人儒雅随却被点名批评，官媒：管好你的粉丝！ |url=https://news.baidu.com/share/detail/9684929735139366891 |website=百度新闻 |accessdate=2018-12-13 |date=2018-09-06}}</ref>
