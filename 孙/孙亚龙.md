'''孙亚龙'''（1988年3月24日－，绰号'''笑笑'''），原为[[地下城与勇士|地下城与勇士]]、[[英雄联盟|英雄联盟]]职业选手。

==简介==
孙亚龙于1988年3月24日出生于[[重庆市|重庆市]][[渝中区|渝中区]]的一个教师家属院，从小家境不错，但由于他母亲管教较严，使得他从小养成了在困境中不走寻常路和另辟蹊径找捷径的性格。在初中时，孙亚龙痴迷于《[[星际争霸|星际争霸]]》、《[[反恐精英|CS]]》、《[[拳皇'97|拳皇97]]》、《[[拳皇'98|拳皇98]]》、《[[石器时代_(游戏)|石器时代]]》等游戏，而且水平很高，但他因此导致学习成绩下降，使得他学了一年画画，并产生了报考美术学院的想法，但最终由于文化课成绩过差，使得他最终放弃报考美术学院。

在大学时，他为了追一名同校的女生而接触《[[诛仙_Online|诛仙]]》、《[[劲舞团|劲舞团]]》等游戏。在《[[地下城与勇士|地下城与勇士]]》中国大陆地区内测时，他加入了DNF，在公测后没多久就打到了至尊6，并位列西南区第一，因使用剑魂而知名。<ref>{{Cite web |url = http://play.163.com/14/0618/10/9V11MDPL00314QEH.html |title = 中国电竞幕后史：从剑舞红颜笑到IG.XiaoXiao   |author = BBKing |date = 2014年6月18日 |publisher = 网易 |language = zh-cn }}</ref>

2010年，孙亚龙在[[世界电子竞技大赛|WCG]][[地下城与勇士|地下城与勇士]]中国区总决赛获得团队赛冠军。<ref>[http://dnf.131.com/dnfgf/972320_1.html 131专访DNF国服第一人孙亚龙 曾获WCG两连冠] {{webarchive|url=https://web.archive.org/web/20160126235124/http://dnf.131.com/dnfgf/972320_1.html |date=2016-01-26 }}</ref>

2010年6月，孙亚龙与女网友“笑笑笑笑笑”之间发生了性行为，孙亚龙在得其初夜并使其怀孕后又提出了分手，后孙亚龙进行道歉。<ref>{{Cite web |url=http://news.7k7k.com/m-news/144702 |title=DNF鸭脖门孙亚龙道歉 只道当时太傻太年轻 |access-date=2016-01-19 |archive-url=https://web.archive.org/web/20160610131020/http://news.7k7k.com/m-news/144702 |archive-date=2016-06-10 |dead-url=yes }}</ref>

2013年10月24日成为[[英雄联盟|英雄联盟]]IG职业选手并担任辅助，现为[[英雄联盟|英雄联盟]]前职业解说，在S2时期因使用琴瑟仙女娑娜多次大招没有击中敌人从而知名。<ref>[http://www.taihainet.com/news/media/hot/2015-12-22/1619063_9.html 盘点职业选手的那些招牌英雄 Faker老琴女笑笑资料介绍（9）] {{webarchive|url=https://web.archive.org/web/20161028220905/http://www.taihainet.com/news/media/hot/2015-12-22/1619063_9.html |date=2016-10-28 }}</ref><ref>[http://games.sina.com.cn/o/z/lol/2015-12-24/fxmykrf2265434.shtml 盘点因操作失误而“封神”的LOL职业选手] {{webarchive|url=https://web.archive.org/web/20161028220150/http://games.sina.com.cn/o/z/lol/2015-12-24/fxmykrf2265434.shtml |date=2016-10-28 }}</ref>

2018年5月，笑笑与其妻子潘慧离婚。
