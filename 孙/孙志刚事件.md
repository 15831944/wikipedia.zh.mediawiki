'''孙志刚事件'''或'''孙志刚案'''於2003年發生，是指原籍[[湖北|湖北]][[黄冈|黄冈]]的孙志刚被[[广州市人民政府|广州市人民政府]]执法机关以[[三无|三无]]人员的理由收押，拘禁期间被收容所员工殴打身亡的事件。此事件虽经官方声称为收容所员工犯罪的个案，却引发了中国国内对[[收容遣送制度|收容遣送制度]]的大讨论。[[中华人民共和国政府|中华人民共和国政府]]之后颁发新法规，废除了广泛被认为是有弊端漏洞、并有[[违宪|违宪]]指责的[[收容遣送制度|收容遣送制度]]。

== 事件背景 ==
{{参见|收容遣送|暂住证}}

在孙志刚事件之前，为了救助[[流浪漢|流浪汉]]和乞讨者，中国地方[[人民政府|人民政府]]设有收容所，目的据称是为了帮助在城市的无业游民返回家乡。但是，在实行过程中，部分收容所员工不但不是救助弱者，反而对被收容者进行敲诈勒索甚至辱骂殴打的违法行为。

根据许多大中城市（北京、上海、广州）的规定，外来务工人员必须办理[[暂住证|暂住证]]，否则视同流浪者。警察有权抓捕这些没有暂住证的人，并把他们遣返原籍或者关押于收容所。在实行过程中，警察往往只锁定那些没有职业的外来农民工、流浪者、乞讨者等[[弱势群体|弱势群体]]。在相关城市的一些大型活动之前，警察为了保安和城市形象，往往采取集中抓捕和遣返的行动。

== 事件经过 ==
'''孙志刚'''（1976年－2003年3月20日），[[湖北|湖北]][[黄冈|黄冈]]人，2001年毕业于[[武汉科技学院|武汉科技学院]]艺术系艺术设计专业，2003年2月24日被[[广州市|广州市]]达奇服装有限公司雇佣。由于刚来广州，他未办理[[暂住证|暂住证]]。3月17日晚上他出門上网，也没有带身份證。在當晚11點左右，他在路上被查暂住证的警察送往黃村街派出所。在這裡，他打了一個電話给朋友，要求對方把他的身份證明文件送往該派出所。可是當對方携带保释金及孙志刚的身份证到達派出所之後，疑因孙志刚之前顶撞警察，仍被当事民警无理拒绝保释。孙志刚随后被轉送往收容站，其收容表格上莫名其妙说其是三无人员，符合收容条件；而事实是孙本人有正常住所、有合法工作、有合法的身份证件，并不符合收容条件。3月20日，孙志刚被-{}-发现在一家收治收容人员的医院死亡<ref>{{Cite web|url = http://ndnews.oeeee.com/html/201302/28/26725.html|title = 被收容者孙志刚之死|date = 2013-02-28|accessdate = 2015-03-19|deadurl = yes|archiveurl = https://web.archive.org/web/20150316075015/http://ndnews.oeeee.com/html/201302/28/26725.html|archivedate = 2015-03-16}}</ref>。

官方最早坚持孙志刚是正常因病死亡，但《[[南方都市报|南方都市报]]》记者调查发现他是被毒打致死。後官方不得不重新进行调查，最后公布的结果是孙是在医院中被护工和同房病人殴打致死。廣州市當局後來拘捕了喬燕琴等十多名涉案人士，當地人民法院於同年6月9日一審判決主犯喬燕琴[[死刑|死刑]]、[[剝奪政治權利|剝奪政治權利]]終身，第二主犯李海婴被判處死刑、緩期兩年執行，其餘十名罪犯分別被判處[[有期徒刑|有期徒刑]]三年至[[無期徒刑|無期徒刑]]。另有六名有關官員因而被控[[瀆職罪|瀆職罪]]，判監一至三年<ref>[http://www.hrol.org/uploads/soft/121217/1-12121F00422.doc 孙志刚案判决书]</ref>。乔燕琴等十二人不服，提出上诉。6月27日，[[广东省高级人民法院|广东省高级人民法院]]作出终审裁决，驳回乔燕琴等人上诉，维持原判。依据[[中华人民共和国最高人民法院|最高人民法院]]授权，核准乔燕琴死刑。同日，[[广州市中级人民法院|广州市中级人民法院]]对涉案的官员李耀辉等五人的上诉作出终审判决，驳回上诉，维持原判<ref>{{Cite web |url=http://news.sina.com.cn/c/2003-06-27/1050280515s.shtml |title=孙志刚案终审 主犯乔燕琴今天被执行死刑(组图)|accessdate=2019-03-21 |author= |date=2003-06-27 |work= |publisher=新浪网，来源：中国新闻网|language=Zh-hans }}</ref>。

== 事件影响 ==
此前也发生过收容所员工犯法的案件，但是由于此次孙志刚身亡，并且其身份不是流浪汉而是大学生，因而产生极大影响。许多媒体详细报道了此一事件，并曝光了许多同一性质的案件，在社会上掀起了对收容遣送制度的大讨论。先后有8名学者上书[[全国人大常委会|全国人大常委会]]，要求就此对收容遣送制度进行[[违宪审查|违宪审查]]。

*2003年5月14日，三名法学博士[[俞江|俞江]]（[[华中科技大学|华中科技大学]]法学院）、[[滕彪|滕彪]]（[[中国政法大学|中国政法大学]]法学院）、[[许志永|许志永]]（[[北京邮电大学|北京邮电大学]]文法学院）向全国人大常委会递交审查《城市流浪乞讨人员收容遣送办法》的建议书，认为：《收容遣送办法》中限制公民[[人身自由|人身自由]]的规定，与《[[中华人民共和国宪法|中华人民共和国宪法]]》和有关法律相抵触，应予以改变或撤销。
*2003年5月23日，五位著名法学家[[贺卫方|贺卫方]]、盛洪、沈岿、萧瀚、何海波以中国公民的名义，联合上书全国人大常委会，就孙志刚案及收容遣送制度实施状况提请启动特别调查程序。

2003年6月20日，[[中华人民共和国国务院总理|中华人民共和国国务院总理]][[温家宝|温家宝]]签署[[国务院令|国务院令]]，公布《[[:s:城市生活无着的流浪乞讨人员救助管理办法|城市生活无着的流浪乞讨人员救助管理办法]]》，自2003年8月1日起执行，标志着《[[城市流浪乞讨人员收容遣送办法|城市流浪乞讨人员收容遣送办法]]》的废止。但是根据媒体的披露，地方人民政府执法机关针对非本地户籍人口的滥用暴力情况，仍然时有发生。<ref>[http://news.qq.com/a/20130109/001230.htm 记者扮聋哑人暗访救助站，被工作人员缚手围殴]</ref>
