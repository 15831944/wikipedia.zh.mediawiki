'''孙钿'''（{{bd|1917年||2011年|6月1日}}）原名'''郁钟瑞'''，字'''文源'''，笔名'''孙钿'''，后以笔名行，[[上海|上海]]人。[[中华民国|中华民国]]及[[中华人民共和国|中华人民共和国]]作家、诗人<ref name=zgzjw>{{Cite web |url = http://www.chinawriter.com.cn/zxhy/member/3764.shtml |title = 孙钿 (1917～2011.6.1) |author =  |date =  |publisher = 中国作家网 |language =  |accessdate = 2017-05-11 |quote =  }}</ref><ref name=zgnbw>{{Cite web|url=http://www.cnnb.com.cn/gb/node2/channel/node13890/node14006/node14007/userobject7ai647402.html |title=诗路坎坷终不悔 教书育人柳成荫 — 诗人孙钿的故事 |author=桂心仪、张良鸿 |date= |publisher=中国宁波网 |language= |accessdate=2017-05-11 |quote= |deadurl=yes |archiveurl=https://web.archive.org/web/20040411065708/http://www.cnnb.com.cn/gb/node2/channel/node13890/node14006/node14007/userobject7ai647402.html |archivedate=2004-04-11 }}</ref>。

==生平==
孙钿出生于上海小南门<ref name=yaonian/><ref name=zgnbw/>。小学毕业后进入上海[[大同大学|大同大学]]附属中学学习，后升入大同大学。1933年，大学生孙钿开始从事左翼戏剧及文学活动，参加“暴风雨剧团”，演出《叛徒》、《南归》等戏剧，并和其他文学青年自筹资金出版文学刊物《天明》<ref name=zgnbw/>。

1933年<ref name=zgzjw/>（一说1934年<ref name=zgnbw/>），孙钿因受[[国民政府|国民政府]]迫害{{Clarify}}，流亡[[日本|日本]]，先后在[[日本大学|日本大学]]、[[早稻田大学|早稻田大学]]学习，并且参加了[[东京|东京]]的中国剧人协会、社会科学座谈会、诗歌座谈会等左翼社团的活动。1937年[[抗日战争|抗日战争]]全面爆发后，孙钿于1937年7月20日回到上海，参加“留日学生抗日救亡协会”。同时，孙钿写下很多诗歌及散文，寄给[[巴金|巴金]]主持的《烽火》及[[胡风|胡风]]主编的《七月》，并出版第一本诗集《击退敌人去》，很快又发表长诗《给敏子》。随后孙钿参加了[[新四军|新四军]]，1938年8月秘密加入[[中国共产党|中国共产党]]，还继续给《七月》等报刊写诗。他是[[七月派|七月派]]诗人之一<ref name=zgnbw/>。孙钿历任新四军第四支队留守处工作人员，[[大别山|大别山]]第八团团部秘书、参谋、民运干事<ref name=zgzjw/>。加入中国共产党后，孙钿受中共党组织派遣到[[香港|香港]]开展地下工作，受[[廖承志|廖承志]]直接领导，主编《东江》、《侨胞》等刊物，并继续作诗，被誉为“战士诗人”<ref name=zgnbw/>。

[[中华人民共和国|中华人民共和国]]成立后，1950年，孙钿在等待新任务时，到[[宁波|宁波]]探访亲友，遇到宁波市市立中学（今[[宁波市第二中学|宁波市第二中学]]）教导主任[[翁心惠|翁心惠]]（后来担任宁波市主管教育文卫的副市长），应翁心惠邀请到该中学教政治思想课<ref name=zgnbw/>。

1953年春，宁波市政府任命郁文源（孙钿）为广济街的宁波卫生学校副校长。经他向宁波市卫生局建议，又经[[浙江省卫生厅|浙江省卫生厅]]审批，在宁波卫生学校的基础上成立了宁波护士学校，于1953年暑期招录第一届新生100人，由宁波市卫生局局长[[郭绍兴|郭绍兴]]任名誉校长，郁文源（孙钿）任副校长。孙钿聘请教师，主持教学楼建设，1954年在今孝闻街[[宁波市第二医院|第二医院]]新大楼的原址上建起了宁波护士学校的新教学楼。后来，宁波护士学校于1957年与广济街的助产学校合并，恢复原名宁波卫生学校<ref name=zgnbw/>。

1955年5月，孙钿接到去[[杭州|杭州]]开会的通知。到杭州后被宣布逮捕，关进杭州乌龙巷监狱，罪名是“[[胡风反革命集团案|胡风反革命集团]]骨干分子”。关押一年多后被宣布释放，结论为：不是叛徒，不曾投敌当汉奸，不是胡风分子，可恢复原职。回到宁波后，孙钿未被准许回宁波护士学校，只能在街道办事处暂领生活费。数月后，被分到机关干部学校任教<ref name=zgnbw/>。

1957年，孙钿在[[反右运动|反右运动]]中被打成“右派”，他的诗歌被说成是“大毒草”。不久，孙钿被下放到[[梅山盐场旧址|梅山盐场]]劳动改造，后又先后在[[四明山_(浙江)|四明山]]挖土石修筑公路，在浙西[[江山縣|江山]]修筑铁路。作为管制分子，孙钿曾拾垃圾，制煤渣砖，当临时工。孙钿的孩子无法上学，孙钿的妻子多次卖血以维持家庭生计。孙钿还曾借钱买蜜蜂放养，但放蜂到[[内蒙古|内蒙古]]时，又被诬为“空降特务”，遭关押两年，后被押回宁波。50多岁的孙钿不得不从头开始学当油漆工、广告匠<ref name=zgnbw/>。

[[中共十一届三中全会|中共十一届三中全会]]后，胡风反革命集团案的平反，孙钿也在1980年获得平反，恢复名誉，恢复中国共产党党籍。后来又先后当选为宁波市第八届人大常委会委员、宁波市第九届政协常委<ref name=zgnbw/>。他还历任[[浙江省|浙江省]]作协顾问，[[宁波市|宁波市]]作协主席。1983年加入[[中国作家协会|中国作家协会]]<ref name=zgzjw/>。

2011年6月1日，孙钿逝世，享耆壽94岁<ref name=zgzjw/><ref name=yaonian>{{Cite web |url =  http://news.163.com/11/0630/16/77QFU59500014AED.html|title = 遥念孙钿 |author =  |date = 2011-06-30 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。

==著作==
*小说集《在乡村里》、《初生期》、《高野良雄之死》（译为[[法文|法文]]）
*诗集《击退敌人去》、《旗》、《望远镜》、《孙钿短诗选》
*《孙钿诗文选》
*译著《日本当代诗选》<ref name=zgzjw/>
