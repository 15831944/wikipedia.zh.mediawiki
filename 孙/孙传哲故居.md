'''孙传哲故居'''位于[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[海曙区|海曙区]]塔前街23-24号，建于[[清代|清代]]，由位于23号的蜗寄庐藏书楼和位于24号的故居组成。其中蜗寄庐藏书楼是[[孙家桂|孙家桂]]的藏书处，故居为中华人民共和国首位邮票设计家[[孙传哲|孙传哲]]的故居。2001年8月，孙传哲故居被公布为海曙区文物保护单位。现故居为'''孙传哲纪念馆'''。
==历史==
孙传哲故居始建于清代，蜗寄庐藏书楼初为[[孙家桂|孙家桂]]私人藏书楼，藏书包含[[天一阁|天一阁]]、抱经楼、烟屿楼等处所散失的约二万卷书籍，孙家桂过世后其长子[[孙定观|孙定观]]继续其父的藏书事业。1915年，中国邮票设计家孙传哲出生于此。<ref>{{cite web |url=http://www.haishu.gov.cn/zjhs/NewsInfo.aspx?Id=174953 |title=孙传哲 |date=2014-04-14 |publisher=宁波市海曙区人民政府 |quote=孙家桂先生竭力抢救收藏行将灭绝的善本，甚于包括天一阁、卢氏抱经楼、徐氏烟屿楼散失于民间的珍本，其中著名的有元刻《隋书》、《范文正公全集》，明刻本《蔡中郎集》、明抄本《圣宋名贤四六丛珠》等，约有447部，共约二万卷，其中珍贵的善本占一半。此外不能购到的，动手抄录。孙家桂先生过世后，其长子孙定观子承父业 |access-date=2017-07-27 |archive-url=https://web.archive.org/web/20140819025041/http://www.haishu.gov.cn/zjhs/NewsInfo.aspx?Id=174953 |archive-date=2014-08-19 |dead-url=yes }}</ref>1999年，孙传哲故居被宁波市文化局公布为宁波市第二批文物保护点。2001年8月，孙传哲故居成为海曙区文物保护单位。<ref>{{cite web |url= http://www.nbwb.net/info.aspx?Id=2196|title=孙传哲故居：蜗寄庐藏方寸情 |author=东南商报 |date=2006-12-01|publisher=宁波市文物保护管理所 |quote=1999年被宁波市文化局核定公布为宁波市第二批文物保护点、2001年被核定公布为海曙区文保单位。}}</ref>
==建筑结构==
孙传哲故居由位于23号的蜗寄庐藏书楼和位于24号的故居两部分组成，其中藏书楼坐北朝南，故居坐西朝东。<ref>{{cite web |url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=927&type=2 |title= 孙传哲故居|date=2005-11-25 |publisher=宁波市文物保护管理所 |quote=23号藏书楼主体建筑坐北朝南，24号故居清代建筑，坐西朝东。}}</ref>
==保护状况==
2005年，[[宁波日报|宁波日报]]报导了孙传哲故居内户籍人口多，安全隐患大，违章建筑影响故居面貌的情况，引起了海曙区政府的关注，后将故居内部违规搭设的雨棚等设施拆除。<ref>{{cite web |url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=945 |title=政府部门勘查孙传哲故居|author=南华|date=2005-12-29|publisher=宁波市文物保护管理所|quote=孙传哲故居内户籍人口众多、安全隐患严重、原貌古风不再一事，经本报连续两天报道，引起了海曙区政府的高度重视。}}</ref><ref>{{cite web |url=http://news.sina.com.cn/o/2006-01-28/07308100576s.shtml |title=孙传哲故居违法分割物全部拆除 |author=南华|date=2006-01-28|publisher=宁波日报}}</ref>

2013年，孙传哲故居进行了修缮工作，开始为孙传哲纪念馆作前期准备工作。2015年4月18日，孙传哲纪念馆成立，内设有三个展厅，其中一个为孙传哲生平展，其余二者为其所创作的邮票展。<ref>{{cite web |url=http://www.nbwb.net/info.aspx?Id=30990 |title=海曙区孙传哲纪念馆正式成立 遗作《瘦西湖》特种邮票首发|date=2015-04-20|publisher=宁波市文物保护管理所}}</ref>
