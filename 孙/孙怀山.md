'''孙怀山'''（{{bd|1952年|7月|||catIdx=}}），[[江苏|江苏]][[涟水县|涟水县]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员。

==生平==
1970年加入中国共产党。[[哈尔滨工业大学|哈尔滨工业大学]]毕业，硕士学位<ref>{{Cite web|url =http://renwuku.news.ifeng.com/index/detail/833/sunhuaishan|title =孙怀山|author =|date =|publisher =凤凰网|language =|accessdate =2017-06-02|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20150906175958/http://renwuku.news.ifeng.com/index/detail/833/sunhuaishan|archivedate =2015-09-06}}</ref>。1970年到1973年，历任江苏省[[金湖县|金湖县]]农机厂工人，农村工作队队长、副指导员。1973年到1978年，历任共青团江苏省金湖县工业系统团委书记、共青团金湖县委书记。1978年到1987年，历任共青团中央办公厅秘书处干事、副处长、处长。1987年到1994年，历任共青团中央办公厅副主任、主任<ref name=ifeng>{{Cite web |url =  http://news.ifeng.com/a/20170602/51192576_0.shtml|title = 政协原常委、港澳台侨委员会原主任孙怀山被双开 |author =  |date = 2017-06-02 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

1994年到1999年，任[[全国政协办公厅|全国政协办公厅]]正局级干部、行管局局长、机关事务管理局局长兼中协服务开发中心（服务局）总经理（局长）。1999年到2008年，任[[中国人民政治协商会议第十届全国委员会委员名单|第十届全国政协]]副秘书长、[[中国共产党中国人民政治协商会议全国委员会机关党组|机关党组]]成员。2008年到2010年，任[[中国人民政治协商会议第十一届全国委员会委员名单|第十一届全国政协]]常委、副秘书长、机关党组成员<ref name=ifeng/>。2010年到2013年，任第十一届全国政协常委、副秘书长、机关党组书记<ref>{{cite web|title=孙怀山任全国政协机关党组书记|url=http://news.xinhuanet.com/politics/2010-10/27/c_12706127.htm|publisher=新华网|date=2010-10-27}}</ref><ref name=ifeng/>。2013年到2015年，任[[中国人民政治协商会议第十二届全国委员会|第十二届全国政协]]常委、常务副秘书长、机关党组书记（正部级）。2015年到2016年，任第十二届全国政协常委、常务副秘书长、机关党组副书记（正部级）<ref name=ifeng/>。2016年8月31日，政协第十二届全国委员会常务委员会第十七次会议任命孙怀山为全国政协港澳台侨委员会主任，孙怀山不再担任第十二届全国政协副秘书长<ref>{{cite news|title=袁贵仁被增补为全国政协外事委员会副主任|url=http://www.thepaper.cn/newsDetail_forward_1522384|date=2016-08-31|newspaper=澎湃新闻}}</ref>。

他是[[中国共产党第十七次全国代表大会|中共十七大]]、[[中国共产党第十八次全国代表大会|中共十八大]]代表，[[中国共产党第十八届中央委员会委员列表|第十八届中央委员]]，第十届全国政协委员，第十一、十二届全国政协常委<ref name=ifeng/><ref>{{cite news|title=第十一届全国政协委员名单|url=http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|accessdate=2013-04-09|newspaper=新华社北京1月25日电 中国人民政治协商会议第十一届全国委员会委员名单(共2237人，2008年1月25日政协第十届全国委员会常务委员会第二十次会议通过)|date=2008年1月25日|deadurl=yes|archiveurl=https://web.archive.org/web/20130928014403/http://www.indaa.com.cn/zt/2009qglh/zxzs/200903/t20090303_127573.html|archivedate=2013年9月28日}}</ref><ref>{{cite news|title=全国政协信息-孙怀山|url=http://www.cppcc.gov.cn/CMS/wylibary/showJcwyxtInfoWylibary.action?tabJcwyxt.guid=11W0020630&wytype=wjwy|accessdate=2013-04-09|newspaper=全国政协委员会}}</ref>。

早在2015年3月，在孙怀山从全国政协机关党组书记降职为副书记后，便有传言称孙怀山可能因涉嫌贪腐落马。但在2016年调任全国政协港澳台侨委员会主任后，外界认为其问题并不严重，打破了此前的传言。2017年2月7日，孙怀山在中共金湖县委书记[[张志勇|张志勇]]陪同下调研[[大佛寺_(金湖县)|大佛寺]]，此后坊间再度传出孙怀山可能落马的消息<ref >{{cite web|url= http://www.zaobao.com.sg/realtime/china/story20170302-731109| language= | title=中国全国政协常委孙怀山涉嫌严重违纪被查|publisher=联合早报网 |date=2017-03-02  |accessdate= }}</ref>。2017年2月26日《[[明報|明報]]》报道，孙怀山在农历年假期后因涉及贪腐被中央纪委带走“[[双规|双规]]”<ref>{{cite web| url=http://www.chinazhsb.com/news_detail_id_55051.html| language=| title=正部级全国政协港澳台侨委主任孙怀山落马| publisher=中华时报| date=2017-02-26| accessdate=}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2017年2月27日，政协第十二届全国委员会常务委员会第十九次会议在北京开幕，七个专委会都由主任汇报工作，惟有港澳台侨委员会由副主任[[杨衍银|杨衍银]]汇报，主任孙怀山缺席<ref>{{Cite web |url = http://news.gmw.cn/2017-02/28/content_23842107.htm |title = 全国政协十二届常委会第十九次会议开幕 |author =  |date =2017-02-28  |publisher = 光明日报 |language =  |accessdate =  |quote =  }}</ref>。2017年3月2日，据[[中国共产党中央纪律检查委员会|中央纪委]][[中华人民共和国监察部|监察部]]网站消息，第十二届全国政协常委、港澳台侨委员会主任孙怀山涉嫌严重违纪，正在接受组织审查<ref>{{Cite web |url =  http://finance.sina.com.cn/roll/2017-03-02/doc-ifyazwha3578808.shtml|title =十二届全国政协常委孙怀山涉嫌严重违纪接受组织审查  |author =  |date =2017-03-02  |publisher =  |language =  |accessdate =  |quote =  }}</ref><ref>{{cite web |url = http://www.ccdi.gov.cn/xwtt/201703/t20170302_94934.html |title = 十二届全国政协常委、港澳台侨委员会主任孙怀山涉嫌严重违纪接受组织审查 |date = 2017-03-02|publisher = 中央纪委监察部网站}}</ref>。

2017年6月2日，据中央纪委监察部网站消息，“依据《中国共产党纪律处分条例》等有关规定，经中央纪委常委会会议研究并报中央政治局会议审议，决定给予孙怀山开除党籍、开除公职处分；终止其党的十八大代表资格；收缴其违纪所得；将其涉嫌犯罪问题、线索及所涉款物移送司法机关依法处理。给予其开除党籍的处分，待召开中央委员会全体会议时予以追认。”<ref name=ifeng/>
