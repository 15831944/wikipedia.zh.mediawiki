'''孙力'''{{BD|1949年|10月16日|2010年|5月9日|catIdx=S孙}}，[[广西人|广西人]]，籍贯[[河北省|河北省]][[定县|定县]]，中国共产党党员，毕业于[[天津师范大学|天津师范大学]]，中国大陆小说家，曾和妻子余小惠合著《都市风流》荣获1991年第三届茅盾文学奖。<ref name="fish" >{{cite web |url=http://www.chinawriter.com.cn/zxhy/member/3819.shtml |title=孙力 (1949～2010) |accessdate= |publisher=中国作家网 |work= |language=zh-hans}}</ref><ref name="dang" >{{cite web |url=http://news.163.com/10/0511/10/66D714AH00014AED.html |title=追忆孙力：办最好的报纸 做最好的报人 |accessdate=2010 |publisher=网易 |work= |language=zh-hans}}</ref>

==生平==
1949年，孙力出生在[[广西省|广西省]]。<ref name="bee" >{{cite web |url=http://news.xinmin.cn/rollnews/2010/05/10/4819516.html |title= 悼 著名当代作家孙力同志逝世 享年61岁(图)|accessdate=2010 |publisher=新民网 |work= |language=zh-hans}}</ref>

1969年，孙力加入内蒙古生产建设兵团。<ref name="fish" /><ref name="bee" />

1974年，孙力从[[天津师范大学|天津师范大学]]毕业。<ref name="fish" /><ref name="bee" />

1984年，孙力开始发表作品。<ref name="bee" />

1991年，孙力和妻子余小惠合著的《都市风流》荣获第三届茅盾文学奖。<ref name="bee" />

==家庭==
孙力的妻子余小惠也是小说家。
