'''孙紫晴'''，[[黑龙江省|黑龙江省]][[鸡西市|鸡西市]]人，[[中国大陆|中国大陆]][[歌手|歌手]]、[[演员|演员]]。

== 简介 ==

*2007年11月，拍摄情景剧《富爸爸智多星》  
*2008年3月，拍摄电视剧《遗忘过去》在剧中饰演刘雪
*2008年5月参加录制CCTV3公益歌《把爱奉献》
*2008年6月，应邀参加兵团卫视主持嘉宾
*2008年7月，拍摄电视栏目剧，《铁棍的龊事》饰演周小凤
*2008年参加北京奥运会火炬巡展嘉宾

2008年11月，拍摄电影《乌鸦》。<ref>{{Cite web |url=http://music.yule.sohu.com/20150902/n420322947.shtml |title=孙紫晴签约新公司 创作才女争做全能艺人 |work=搜狐音乐 |date=2015-09-02}}</ref>

2009年8月参加“魔乐寻找好声音”活动并与该公司签约。2009年10月发行首支单曲《傻女人》，2010年發行單曲《說了》。<ref name=qq2010>{{Cite web |url=http://ent.qq.com/a/20100305/000462.htm |title=孙紫晴新单曲《说了》 续写《傻女人》网络热潮 |work=騰訊娛樂 |date=2010-03-05}}</ref>
