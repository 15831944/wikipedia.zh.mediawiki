'''孙海英'''（{{bd|1956年|10月7日|catIdx=Sun,孙}}），[[中华人民共和国|中国]][[辽宁省|辽宁省]][[沈阳市|沈阳市]]人，[[中国基督教新教|中国基督教新教]][[基督徒|基督徒]]<ref>[http://news.ifeng.com/a/20141222/42770534_0.shtml 军报微博批孙海英：基督教徒能演好优秀共产党员？]</ref>；在其[[微博|微博]]中称自己非[[中国共产党|中国共产党]]党员<ref>{{cite web |author = 孙海英 |work = 孙海英的微博 |quote = 更正：孙海英，男，59岁，非党员。1975年写过申请书，到今天已退休六年，没人回我是否合格。}}</ref><ref name="hq">{{cite web |url = http://news.ifeng.com/a/20141226/42804636_0.shtml |title = 孙海英成美“扳倒中国”的第三种武器 |publisher = 凤凰网 |author = [[环球时报|环球时报]] |accessdate = 2014-12-26 |date = 2014-12-26}}</ref>。当代[[话剧|话剧]]、[[电影|电影]]、[[电视剧|电视剧]]演员，因在中国大陆热播电视剧《[[激情燃烧的岁月|激情燃烧的岁月]]》中扮演男主角石光荣而一举成名。

== 生平 ==
孙海英出生于辽宁省[[沈阳市|沈阳市]]，17岁去西藏当文艺兵。1979年二次参军入伍，进入福建军区话剧团。1985年转业，89年进入沈阳话剧团，后成为职业演员。

在出演《激情燃烧的岁月》之后，孙海英与该剧女主演[[吕丽萍|吕丽萍]]结婚。2002年，吕丽萍与孙海英合作演出彭小莲导演的影片《假装没感觉》，并拍摄纪录片《心中的节日》。

孙海英、吕丽萍夫妇均为热心的[[基督徒|基督徒]]，2009年1月1日，他们在“第三届CCTV电视剧群英汇”演唱内容出自{{Bible|箴言|31}}的歌曲《才德的妇人》<ref>[http://www.gospeltimes.cn/news/2009_01_02/4007.htm 孙海英、吕丽萍央视合唱《才德的妇人》] {{webarchive|url=https://web.archive.org/web/20110905072832/http://www.gospeltimes.cn/news/2009_01_02/4007.htm |date=2011-09-05 }}</ref>，他们夫妇的信仰也体现在其博客[http://blog.sina.com.cn/lvliping 海英丽萍]中。

曾于2009年春节期间协助[[远志明|远志明]]在美国加州[[湾区|湾区]][[庫比蒂諾_(加利福尼亞州)|庫比蒂諾]]举办华人基督教布道会 《一代人的见证》。

==事件==
2007年接受记者采访，被问起对目前中国的婚恋状况怎么看时，他便痛骂[[同性恋者|同性恋者]]，声称[[同性恋|同性恋]]「決不允許」、「毀滅人類」、「談都不要談」、「非常骯髒」、「這是在[[犯罪|犯罪]]」<ref>{{cite video |url = http://v.youku.com/v_show/id_XNzQwMTE0OA==.html |title = 视频: 孙海英称同性恋为犯罪 引发众多争议}}</ref>。“同性恋犯罪论”一出，立刻遭到多数知名学者反对，中国社会学家[[李銀河_(社會學家)|李银河]]说道：“孙海英的言论让我想到了[[二战|二战]]时期的[[希特勒|希特勒]]，把同性恋关进[[集中营|集中营]]杀掉一样。”崔子恩、程青松、[[吴幼坚|吴幼坚]]等社会活动人士均批评他[[无知|无知]]。

此外，作为基督徒的孙海英常在新浪微博发布一些基于宗教的过激言论，如“唐朝基督教正式传入中国，才有了盛唐”“顺治皇帝基督教及宣教士汤若望等人的意见，才有了后来的康乾盛世”，在其他领域也常用过激的基督徒观点来看待事情，因此经常受到中国网民的批评和抵制<ref>[http://www.nanrenwo.net/yule/63160.html 孙海英疯了? 冯小刚回击炮轰孙海英]</ref>。其部分涉及宗教的言论也遭到官媒的抨击。<ref name="hq"/>

孙海英夫妇2011年2月23日在微博留言称：“我正式提出申请将本人户籍调入中国[[钓鱼岛|钓鱼岛]]，我渴望成为这个岛上的居民，从人类有文字记录开始那就是中国的一部分。本人谨通过互联网请求国家有关部门批准，申请人：孙海英、吕丽萍。”

2015年8月，孫海英在微博發帖稱「[[文革|文革]]是[[反人類罪|反人類罪]]，絕不原諒文革」，该微博发出当日，随即成為微博熱門話題；此前，他還在微博上質疑「[[沒有共產黨就沒有新中國|沒有共產黨就沒有新中國]]」的說法，公開表明「沒有誰個都有中國」，而他在微博上还發表了關於[[宗教自由|宗教自由]]的言論<ref>{{cite web |url=https://hk.news.appledaily.com/china/realtime/article/20180523/58225400| |work = 香港苹果日报|title =【靚女作家惹火】男星講「文革」被封 微博操控言論  }}</ref>。2016年2月26日，[[国家互联网信息办公室|国家互联网信息办公室]]发布消息，孙海英的微博因多次发布反对宪法所确定的基本原则、损害国家荣誉和利益以及造谣传谣、扰乱社会秩序等违法违规信息的行为而被封号<ref>{{cite web |url=http://cnews.chinadaily.com.cn/2016-02/26/content_23662059.htm| |work = 搜狐娱乐|title =网信办持续打击违法信息 孙海英等大V遭封号 }}</ref>。
