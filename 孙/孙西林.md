'''孙西林'''（{{bd|1910年||1946年|1月31日}}）原名'''孙锡麟'''，化名'''孙山'''、'''孙昌克'''，[[辽宁省|辽宁省]][[昌图县|昌图县]]人，[[中国共产党|中国共产党]]官员，中共[[合江省_(解放區)|合江省]]工委委员、[[佳木斯地区|佳木斯地区]]行署专员、[[佳木斯市|佳木斯市]]副市长，遇刺身亡<ref name=jmsi>{{Cite web |url = http://www.jmsmz.gov.cn/index.php?id=471 |title =  孙西林|author =  |date =  2017-08-25|publisher = 佳木斯民政信息网 |language =  |accessdate =  |quote =  }}</ref><ref name=huainian>{{Cite web |url =  http://www.caf.ac.cn/news/xwzx/201301/2013-01-23-18-52.html  |title =  深切怀念李万新老院长|author =  |date = 2013-01-23 |publisher = 中国林业科学研究院 |language =  |accessdate =  |quote =  [}}</ref>。

==生平==
1924年，孙西林考入奉天省立第一中学。1925年[[五卅运动|五卅运动]]在[[上海|上海]]爆发，孙西林在学联的领导下参加了反帝爱国示威，自此开始参加中共领导的革命。1926年9月，孙西林考入[[东北大学|东北大学]]工学院预科乙班，在校期间接受了[[马列主义|马列主义]]学说，并在中共党员佟玉华等同学影响下，秘密加入[[中国共产主义青年团|中国共产主义青年团]]，此后在中共党组织直接领导下参与和组织校内外反对[[日本帝国主义|日本帝国主义]]的活动<ref name=jmsi/>。

1928年底，孙西林等同学赴[[欧洲|欧洲]]，孙西林到[[比利时|比利时]]学习。1931年[[九一八事变|九一八事变]]消息传到欧洲后，孙西林等中国留学生迅速组织了“反帝大同盟”，率华侨在比利时[[布鲁塞尔|布鲁塞尔]]、[[列日|列日]]、[[鲁汶|鲁汶]]和煤城等地示威游行，反对日本侵略者，遭比利时当局禁止。1932年，孙西林等主要成员被比利时驱逐出境<ref name=jmsi/>。

此后孙西林流亡[[法国|法国]]，在[[巴黎|巴黎]]郊区一家人造丝厂当工人。该厂有200余名华工，多为不识字的穷苦工人。孙西林组织工人要求改善生活待遇，获得华工拥护。孙西林还在工人中成立了抗日救国会、红色救济会等群众团体，支援[[中国东北|中国东北]]抗日。1932年秋，孙西林加入[[中国共产党|中国共产党]]<ref name=jmsi/>。

1934年8月，孙西林经[[法国共产党|法国共产党]]中央介绍到[[苏联|苏联]][[莫斯科|莫斯科]]，在外国工人出版社工作。不久，组织派他到[[东方劳动者共产主义大学|东方劳动者共产主义大学]]学习[[马列主义|马列主义]]理论。1936年4月，[[东北抗日联军|东北抗日联军]]第四军军长[[李延禄|李延禄]]抵达莫斯科，孙西林为李延禄当秘书。此后三年间，他协助李延禄开展上层[[统战|统战]]工作<ref name=jmsi/>。

1936年9月，由[[潘汉年|潘汉年]]、李延禄、孙西林三人组成的代表团，以东北抗日联军和东北3000万同胞的名义向[[南京国民政府|南京国民政府]]请愿，敦促[[蒋介石|蒋介石]]抗战。同年11月，李延禄、孙西林绕道[[意大利|意大利]]到达上海。代表团在[[国民政府军事委员会|国民政府军事委员会]]办公室副主任[[刘光|刘光]]召集的小型会议上汇报了东北抗日联军的情况，递交了请愿书，并提出六项要求。同时，李延禄、孙西林扩大统战工作，先后争取到[[宋哲元|宋哲元]]、[[张学良|张学良]]、[[冯玉祥|冯玉祥]]支持<ref name=jmsi/>。

1936年11月22日，上海[[中国人民救国会|救国会七君子]]被国民政府逮捕入狱。代表团以东北3000万同胞的名义提出抗议，要求立即释放七君子。[[西安事变|西安事变]]后，代表团决定由孙西林草拟电文呼吁[[张学良|张学良]]、[[杨虎城|杨虎城]]以民族利益为重，敦促蒋介石停止内战，共同抗日<ref name=jmsi/>。

1937年3月，[[周恩来|周恩来]]两次到上海同东北抗日联军代表团研究工作。根据周恩来的指示，孙西林起草了以李延禄的名义向各省发出的通电，倡议在全国范围内组织抗日救亡团体筹备会议。6月中旬，在北平召开“东北抗日救亡总会”成立大会，孙西林等30多人当选常务委员<ref name=jmsi/>。

[[八一三事变|八一三事变]]爆发后不久，因工作关系，李延禄和秘书孙西林离开上海赴东北但未果，乃转回[[香港|香港]]。在李延禄等待上级决定期间，经中共党组织批准，孙西林和李延禄之女[[李万新|李万新]]在香港结婚。婚后不久，李延禄和孙西林奉调到[[武汉|武汉]]，准备转延安参加[[中共六届六中全会|中共六届六中全会]]<ref name=huainian/>。

1938年底，经周恩来安排，李延禄和孙西林辗转抵达[[陕甘宁边区|陕甘宁边区]]首府[[延安|延安]]<ref name=jmsi/>。在延安期间，孙西林曾下乡当过两年乡长。孙西林过去学的专业是冶金和法律，精通[[英语|英语]]、[[法语|法语]]、[[俄语|俄语]]，曾为[[毛泽东|毛泽东]]、[[周恩来|周恩来]]担任法语翻译<ref name=huainian/>。

1945年[[抗日战争|抗日战争]]胜利后，[[中共中央|中共中央]]组织了第一批赴[[中国东北|东北]]的干部，孙西林和其他200人于1945年9月2日自延安启程赴东北。11月，李延禄、[[李范五|李范五]]、孙西林等人抵达[[佳木斯市|佳木斯市]]。11月21日，[[合江省_(解放區)|合江省]]成立，李范五任中共合江省工委书记，李延禄任合江省政府主席，孙西林任中共合江省工委委员<ref name=jmsi/><ref name=huainian/>。

刚刚“光复”的佳木斯形势异常复杂，城内到处残垣断壁，有日伪残余势力及国民党特务暗地破坏，城外有“中央胡子”（指“光复”后接受国民政府收编的各种地方武装、土匪）横行。大资本家囤积居奇，市民生活困难<ref name=huainian/>。为恢复城市秩序，稳定群众生活，孙西林提出成立佳木斯市人民政府的建议，获中共合江省工委书记[[李范五|李范五]]采纳，李范五代表中共合江省工委和省政府任命[[董仙桥|董仙桥]]为市长、孙西林为副市长兼佳木斯地区专员。市政府刚成立，孙西林便投入繁忙的工作中，组织召开工商业者会议，宣传中共的工商政策，鼓励员工商户恢复生产营业，保障群众生活。他还到发电厂、火磨厂敦促制定恢复生产的方案<ref name=jmsi/>。为尽早发电，给民众解决御寒用煤，孙西林亲率市政府秘书长到100多公里外的[[双鸭山|双鸭山]]，顶风冒雪冲破土匪重重堵截，将煤运回佳木斯<ref name=huainian/>。

1945年12月中旬，妻子李万新带着4岁的儿子抵达佳木斯。这时国民党军队在[[美国|美国]]支持下已攻占[[山海关|山海关]]、[[锦州|锦州]]，并向北推进，佳木斯城内外反共组织密谋暴动、纵火、暗杀，陆续发生了隐藏在中共部队军械所的国民党特务哗变，[[萝北县|萝北]]、[[勃力|勃力]]、[[同江市|同江]]等地杀害中共党政干部事件。孙西林不断叮嘱身边人员要提高警惕<ref name=huainian/>。

1946年1月30日，中共合江省工委书记李范五要到[[中共中央北满分局|中共中央北满分局]]开会汇报工作，孙西林决定派自己唯一的警卫员护送李范五前往。1月31日清晨，孙西林上班时，将已随自己跳上马车的警卫员赶下车，并叮嘱一定要保证李范五的安全。孙西林到办公室后，随即与市长商议各项工作。10点左右，伪装混入民主大同盟的[[中国国民党|国民党]]特务邬捷飞以及宁宪君、沈洪福、任哲贤等人潜入佳木斯市政府大楼，任哲贤、沈洪福分守前、后门，邬捷飞率宁宪君直奔市长办公室，宁宪君首先闯入办公室，邬捷飞在宁宪君身后开枪，孙西林头部中弹当即身亡，年仅36岁<ref name=jmsi/>。

孙西林遇刺牺牲后，佳木斯市将其遗体安葬在市内最大的公园中，将公园命名为“[[西林公园|西林公园]]”，并市内一条主要道路命名为“西林大街”<ref name=jmsi/>。
