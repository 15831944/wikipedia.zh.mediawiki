'''孙伟'''（{{bd|1961年|12月||}}），[[山东省|山东省]][[蓬莱市|蓬莱市]]人。男，汉族。1977年12月参加工作，1988年2月加入中国共产党<ref name=jianli>{{Cite web |url = http://cpc.people.com.cn/GB/64162/123659/123796/14417502.html |title = 孙伟同志简历 |author =  |date =  |publisher = 中国共产党新闻网 |language =  |accessdate =2017-03-05 |quote =  }}</ref><ref name=fusheng>{{Cite web|url =http://www.shandong.gov.cn/col/col208/index.html|title =副省长 孙伟|author =|date =|publisher =山东省人民政府|language =|accessdate =2017-03-05|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170226224113/http://www.shandong.gov.cn/col/col208/index.html|archivedate =2017-02-26}}</ref>。北京大学地理系经济地理-区域与城市规划专业本科毕业。

==生平==
早年为[[浙江省|浙江省]][[金华县|金华县]]多湖公社下乡[[知青|知青]]。1979年考入[[北京大学|北京大学]]地理系经济地理-区域与城市规划专业，毕业获得[[理学学士|理学学士]]学位。1983年8月起，先后在[[中华人民共和国城乡建设环境保护部|中华人民共和国城乡建设环境保护部]]、国务院旅游协调小组办公室、[[国务院办公厅|国务院办公厅]]工作，历任副主任科员、主任科员<ref name=jianli/><ref name=fusheng/>。

1991年4月，历任国务院办公厅秘书（副处级）、秘书二局四组副组长、秘书二局四组一秘（正处级）兼副组长、秘书二局四组组长、秘书二局三处处长。1999年5月，任国务院办公厅秘书二局助理政务专员兼三处处长。2000年6月，任[[吴邦国|吴邦国]]副总理秘书（2002年7月升为正局级）。2003年3月，任吴邦国委员长秘书。2003年7月，任[[全国人大常委会办公厅|全国人大常委会办公厅]]吴邦国委员长办公室主任。2004年10月，任[[第十届全国人大常委会|第十届全国人大常委会]]副秘书长、机关党组成员、委员长办公室主任。2008年1月，当选[[第十一届全国人大代表|第十一届全国人大代表]]（浙江）。2008年3月，任[[第十一届全国人大常委会|第十一届全国人大常委会]]副秘书长、机关党组成员<ref name=jianli/><ref name=fusheng/>。

2011年2月，任[[中共山东省委|中共山东省委]]常委，[[山东省人民政府|山东省人民政府]]党组成员。2011年3月，任中共山东省委常委、副省长、山东省人民政府党组成员。2012年6月，任中共山东省委常委、山东省人民政府党组副书记、常务副省长。2012年11月，兼任[[山东行政学院|山东行政学院]]院长<ref name=jianli/><ref name=fusheng/>。

2017年3月，任[[中共甘肃省委|中共甘肃省委]]副书记<ref>{{cite news|title=孙伟任甘肃省委副书记，黄强、宋亮任省委常委|url=http://www.thepaper.cn/newsDetail_forward_1650026|accessdate=2017-03-29|newspaper=澎湃新闻}}</ref>。
