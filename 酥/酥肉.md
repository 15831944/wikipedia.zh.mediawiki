'''酥肉'''是一种小吃，流行于中国[[四川|四川]]、[[山西|山西]]、[[陕西|陕西]]、[[河南|河南]]等地，通过油炸裹过淀粉的猪肉做成<ref name="李幼谦2015">{{cite book|author=李幼谦|title=李幼谦散文集：东邪西怪话小吃|url=https://books.google.com/books?id=3FVHCgAAQBAJ&pg=PT62|date=015|publisher=华文创意写作中心|pages=59–62|id=GGKEY:SWRX1UAAGXK}}</ref>。各地的酥肉做法有一定差异<ref name="夢芝2018">{{cite book|author=夢芝|title=舌尖上的中國 古鎮篇|url=https://books.google.com/books?id=wRFPDwAAQBAJ&pg=PT303|date=28 February 2018|publisher=崧博出版事業有限公司|isbn=978-957-735-395-5}}</ref>。酥肉是四川的一道特色菜，也是当地的家常菜<ref name="李幼谦2015"/>。在山西，小酥肉是晋式三蒸之一<ref>{{cite web |title=晉菜兵法全攻略（上）：沒打過來福的常威，與他發明的醬梅肉 |url=https://www.thenewslens.com/article/112361 |website=關鍵評論 |accessdate=2019-03-29}}</ref>。河南的炸紫酥肉是当地的名菜，酥肉是当地农村的“[[八大碗|八大碗]]”之一<ref>{{cite web |title=河南向世界发布“中国菜” 34个地域菜系集体亮相 |url=http://hen.chinadaily.com.cn/2018-09/12/content_36901263.htm |website=中国日报 |accessdate=2019-03-29}}</ref><ref>{{cite web |author1=汪莉絹 |title=浚縣八大碗 體現農村辦桌文化 |url=https://udn.com/news/story/7332/3365056 |website=聯合晚報 |accessdate=2019-03-29}}</ref>。酥肉也是陕西的「九鬥碗」之一<ref name="夢芝2018"/>。
