'''酥醪观'''{{noteTag|1=“酥”读音“Sū”，音同“苏”；“醪”读音“Láo”，音同“劳”。 }} ，位于[[广东省|广东省]][[惠州市|惠州市]][[博罗县|博罗县]][[罗浮山|罗浮山]]，为[[东晋|东晋]][[道士|道士]][[葛洪|葛洪]]建立，后历朝历代屡次兴废，现存为[[清朝|清朝]][[雍正|雍正]]时期的道长[[柯善智|柯善智]]所建，为著名的人文景观和道家圣地，被誉为“神仙古洞”和“洵栖灵之奥区、修养之福地”，同时和“[[冲虚观|冲虚观]]、[[黄龙观|黄龙观]]、[[白鹤观|白鹤观]]、[[九天观|九天观]]”一起被誉为罗浮山道家五大观。<ref name="fox ">{{cite web|url=http://www.taoist.org.cn/webfront/webfront_viewContents.cgi?id=3699|title=罗浮山酥醪观|accessdate=|publisher=中国道家网|work=|language=zh-cn}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url=http://www.gdwh.com.cn/whyc/2009/0223/article_780.html |title= 清、民国 博罗 酥醪观|accessdate=2009-02-23 |publisher=广东文化网 |work= |language =zh-cn}}</ref>

== 历史 ==
[[东晋|东晋]]时期，道士[[葛洪|葛洪]]建立了“北庵”，它是今天“酥醪观”的前身。<ref name="fox " />

[[清朝|清朝]][[雍正|雍正]]五年（1727年），道长[[柯善智|柯善智]]重新恢复部分遭到战火焚毁的建筑，一直保存至今。<ref name="fox " />

[[中华民国|中华民国]]初年，道长[[张寓泉|张寓泉]]与人“幕集巨金，重修殿宇”，1930年，建筑全部修建完成。1925年10月至11月，[[黄埔军校|黄埔军校]]校长、东征军总指挥[[蒋中正|蒋中正]]来到酥醪观拜神和修养，他在观中抽的一支签，寓意“胜不离川，败不离台。”。<ref>{{cite web |url= http://taiwan.huanqiu.com/taiwan_history/2012-03/2495236.html|title= 蒋介石早年抽签应验：胜不离川败不离台|accessdate=2012-03-05  |publisher=环球时报 |work= |language =zh-cn}}</ref>

1989年，博罗县政府出资重新翻修部分建筑。<ref name="duck " >{{cite web |url= http://www.hznews.com/xw/zhxw/200703110016.html|title= 酥醪观印象|accessdate=2007-03-11 |publisher= 惠州新闻网|work= |language =zh-cn}}</ref>
