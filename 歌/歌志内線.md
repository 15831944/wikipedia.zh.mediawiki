'''歌志内線'''（{{jpn|j=歌志内線|hg=うたしないせん|rm=Utashinai sen|f=y}}）曾經是一條連結[[北海道|北海道]][[砂川市|砂川市]]（[[空知支廳|空知支廳]]管內）的[[砂川站|砂川站]]至[[歌志內市|歌志內市]]的{{link-ja|歌志內站|歌志内駅}}，屬於[[日本國有鐵道|日本國有鐵道]]（國鐵）與[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）的鐵路線（[[地方交通線|地方交通線]]）。根據國鐵再建法的施行獲指定為第2次特定地方交通線，1988年（[[昭和|昭和]]63年）4月25日全線{{link-ja|廢線|廃線|廢除}}<ref name="hokkaido-np-1988-4-25">“歌志内線 96年の歴史閉じる”. [[北海道新聞|北海道新聞]] (北海道新聞社). (1988年4月25日)</ref>。

== 路線資料（廢除前） ==
* 管轄：北海道旅客鐵道（第一種[[鐵路公司|鐵道事業者]]）、[[日本貨物鐵道|日本貨物鐵道]]（第二種鐵道事業者）
* 路線距離（{{link-ja|營業距離|営業キロ}}）砂川－歌志內 14.5公里
* [[軌距|軌距]]：1067毫米
* [[複線鐵路|複線]]路段：沒有（全線[[單線鐵路|單線]]）
* [[電氣化鐵路|電氣化]]方式：沒有（全線{{link-ja|非電氣化|非電化}}）
* [[閉塞_(鐵路)|閉塞方式]]：Tablet閉塞式
** 可交會車站：沒有（全線1閉塞）

== 歷史 ==
此鐵路線沿途有許多[[煤礦|煤礦]]，生產出[[煤|煤]]，在1891年（[[明治|明治]]24年），[[北海道炭礦鐵道|北海道炭礦鐵道]]啟用。在1906年（明治39年），按[[鐵道國有法|鐵道國有法]]把鐵路線國有化，成為官設鐵道路線。當時鐵路線是主要是運輸煤礦，在1963年（[[昭和|昭和]]38年）中成為其中一條很高盈利的鐵路線，可是後來因為煤礦業的衰退，運輸量下降，成為一條將要停運的鐵路線。

在1980年（昭和55年），因為[[日本國有鐵道經營再建促進特別措置法|國鐵再建法]]的施行，在1984年（昭和59年）被指定為[[特定地方交通線#第2次停運對象|第2次特定地方交通線]]，在1987年（昭和62年）[[國鐵分割民營化|國鐵分割民營化]]後一年，即是在1988年（昭和63年）便停運。改為以巴士替代服務。此外，在砂川站的另一條鐵路線：函館本線[[上砂川支線|上砂川支線]]也是因為煤礦業的衰退而在1994年（平成6年）停運。

* 1891年（[[明治|明治]]24年）7月5日：'''北海道炭礦鐵道'''砂川 - 歌志内之間 (8[[英里|英里]]64[[鏈_(量度單位)|鏈]]) 的部分空知線啟用。新設歌志内站。
* 1896年（明治29年）10月21日：新設神威站。
* 1901年（明治34年）9月：與空知線分離，空知線改名為'''歌志内支線'''。
* 1906年（明治39年）10月1日：砂川 - 歌志内 (8.9英里) 鐵路線國有化。
* 1909年（明治42年）10月12日：按[[國有鐵道線路名稱|國有鐵道線路名稱]]，砂川 - 歌志内之間的鐵路線名稱為'''歌志内線'''。
* 1946年（[[昭和|昭和]]21年）11月1日：新設文珠臨時乘降站。
* 1947年（昭和22年）2月20日：文珠臨時乘降站升級為車站，開始貨物起卸服務。
* 1960年（昭和35年）2月：文珠站民間委託化。
* 1960年（昭和35年）3月：開始使用柴油車運行開始。並把客運與貨運列車分開運行，砂川 - 歌志内之間的旅客運輸所需時間縮短至約20分館
* 1960年（昭和35年）12月26日：新設歌神站（客運站）。
* 1961年（昭和36年）2月10日：新設西歌站。文珠站的貨物起卸服務停止。
* 1961年（昭和36年）12月25日：新設燒山站（客運站）。
* 1972年（昭和47年）3月15日：西歌站、神威站的貨物起卸服務停止。
* 1984年（昭和59年）6月22日：指定為第2次特定地方交通線。列入停用鐵道線表中。
* 1986年（昭和61年）11月：除了砂川站、歌志内站外，其他站的車站售賣服務停止。
* 1987年（昭和62年）4月1日：因[[國鐵分割民營化|國鐵分割民營化]]，本線成為北海道旅客鐵道（第一種鐵道事業者）與日本貨物鐵道的鐵路線（第二種鐵道事業者）。
* 1988年（昭和63年）4月25日：全線 (14.5km) 停運，由[[北海道中央巴士|北海道中央巴士]]代替鐵路服務。

== 運行狀態（客運服務，停運前） ==
在停運前最後一次的時間表修正是在1986年3月3日<ref name="time">[https://archive.is/20121120021705/http://www.nihonkai.com/railroad/utashinai/utashinai.html  歌志內線列車時間表]</ref>，全日只有八班來回普通列車往返砂川與歌志內站<ref name="time" />。早上有三班來回列車、中午與黃昏各一班、晚上有三班<ref name="time" />。在早上，有一班列車會不停燒山站<ref name="time" />。
