《'''歌唱二小放牛郎'''》，著名的革命[[儿童歌曲|儿童歌曲]]，[[方冰|方冰]]词，[[劫夫|劫夫]]曲。创作于1942年，曲子借鉴了[[昆曲|昆曲]]“尼姑思凡”的旋律，采用五声音阶的民族调式，该曲在《[[晋察冀日报|晋察冀日报]]》副刊发表后，很快传开了，甚至[[敌占区|敌占区]]的[[保定|保定]]、[[北平|北平]]的青年中也在传唱。<ref name="Bunny_chinanews.com">{{Cite web|url=http://www.news365.com.cn/wxpd/bhygb/ygb/201104/t20110418_3012535.htm|title=从生活中寻觅创作源泉|author=李定国|date=2011-04-15|publisher=新民晚报|language=zh-hans|accessdate=2011-6-10导入|quote=|deadurl=yes|archiveurl=https://archive.is/20130101153712/http://www.news365.com.cn/wxpd/bhygb/ygb/201104/t20110418_3012535.htm|archivedate=2013-01-01}}</ref>
== 创作历史 ==
1942年，当时在八路军[[西北战地服务团|西北战地服务团]]的方冰、劫夫在河北[[平山县|平山县]]创作了歌曲《歌唱二小放牛郎》，由该团女演员[[顾品详|顾品详]]首唱。虽然关于歌曲主角“王二小”的身份说法众多，但词作者方冰曾说“这是一个复合人物，是综合许多可歌可泣的人物故事后创作出来的。”

== 词曲版权 ==
=== 作者逝世时间 ===
*词作者'''方冰'''于1997年去世。
*曲作者'''李劫夫'''于1976年12月17日在[[沈阳|沈阳]]病逝。
== 歌曲影响 ==
*中华人民共和国建国后，这首歌被编入小学音乐教材，成为全中国少年儿童的必唱歌曲。
*马来西亚华语歌手庄学忠，与台湾华语歌手龙飘飘将此歌曲翻唱为《白云下的牧歌》。
