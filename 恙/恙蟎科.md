'''恙蟎科'''（學名：{{lang|la|'''Trombiculidae'''}}；({{IPAc-en|t|r|ɒ|m|b|ɨ|ˈ|k|juː|l|ɨ|d|iː}}）是[[蛛形綱|蛛形綱]][[蜱蟎亞綱|蜱蟎亞綱]][[蟎形總目|蟎形總目]][[恙蟎目|恙蟎目]]<ref name = "dyn"/>（[[Catalogue_of_Life|Catalogue of Life]]作[[前氣門亞目|前氣門目]]<ref name = "COL"/>）[[恙蟎總科|恙蟎總科]]之下的一個科，常見於農作物<ref name = "Smith1998" />。這些物種的幼蟲階段經常都咬其宿主，造成過敏反應及[[皮膚炎|皮膚炎]]。
