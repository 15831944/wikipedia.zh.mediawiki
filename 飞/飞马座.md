'''飞马座'''（別名'''天馬座'''{{efn|1=源自日本漫畫《[[聖鬥士星矢|聖鬥士星矢]]》角色「[[天馬座_星矢|天馬座 星矢]]」。}}）的大四边形是[[秋季|秋季]]星空中北天区中最耀眼的星象，整个这片天区远离[[银河系|银河系]]的[[银盘|银盘]]。所以布满了明暗各异的[[星系|星系]]。这里有一个[[梅西耶天体|梅西耶天体]]，即[[球状星团|球状星团]][[M15|M15]]。

== 神话和历史 ==
在[[希腊神话|希腊神话]]中，[[珀尔修斯|珀尔修斯]]杀死了[[戈耳工|戈耳工]][[美杜莎|美杜莎]]後，从美杜莎颈腔喷出的血中跳出了一匹飞马[[珀伽索斯|珀伽索斯]]。这匹长著翅膀的神马立刻飞到了天上，后来它降落到[[赫利孔山|赫利孔山]]上，在那里创造了[[灵泉|灵泉]]，这[[泉|泉]]後来成了[[诗人|诗人]]的灵感之源。[[雅典娜|雅典娜]]後来驯服了这匹马，把它赠给[[柏勒洛丰|柏勒洛丰]]，让他骑着这匹马旅行。在一次旅行中，飞马将柏勒洛丰甩下摔死，独自继续旅行，最後到达了[[天界|天界]]，成为了一个[[星座|星座]]。

== 有趣的星 ==
* [[室宿二|室宿二]]（飞马座β）：这是一颗不规则[[变星|变星]]，[[星等|星等]]变化在+2.2到+2.75间

== 深空天体 ==
* [[M15|M15]]：这是[[球状星团|球状星团]]家族中一颗细致，明亮的代表，值得注意的是它那非常紧密的核。即使使用最小的[[望远镜|望远镜]]也可以看见它，用8英寸（20厘米）的望远镜可以看清它的外围区域内的数百颗明亮的[[恒星|恒星]]。它的星等是+6.4，[[直径|直径]]是6.8[[弧分|弧分]]。
* [[NGC_7331|NGC 7331]]：这个[[漩涡星系|漩涡星系]]看起来好像是[[仙女座|仙女座]][[M31|M31]]的一个遥远的兄弟，它的星等是+9.5，使用8厘米的望远镜可以隐约看见，而使用8英寸的望远镜可以看得更为有趣。
* [[NGC_7479|NGC 7479]]：这是[[哈勃棒旋星系|哈勃棒旋星系]]中SBb型的代表，尽管使用小型仪器不太容易看见它。用20厘米的望远镜才能看清，它是一个直棒，中间有一个明亮的核。
* [[NGC_7814|NGC 7814]]：这是一个边缘对着[[地球|地球]]的相当明亮的Sa型或Sb型旋臂[[星系|星系]]，20厘米的望远镜可以看的很清楚，它的星等是+11.4。
* [[斯蒂芬五重奏|斯蒂芬五重奏]]星系：这是[[星群|星群]]以它19世纪的发现者[[法国|法国]][[天文学家|天文学家]][[埃杜瓦·斯蒂芬|埃杜瓦·斯蒂芬]]的名字命名的，斯蒂芬五重奏星系是一个紧密的[[星系群|星系群]]中的五个显著的成员，这个星系群还有六个较暗的成员（[[NGC_7320C|NGC 7320C]]）。这些星系都是+14等或更暗的星系，使用20厘米望远镜可以看见一两戈较明亮的成员。
