'''擊球員'''（'''打者'''）是指[[棒球|棒球]]比賽中，進攻方上場上場擔任打擊任務的球員。擊球員必須手握[[球棒|球棒]]站在[[本壘|本壘]]兩側的[[打擊區|打擊區]]內，設法將防守方[[投手|投手]]所投的球擊出，以便上壘。如能經過[[一壘|一壘]]、[[二壘|二壘]]和[[三壘|三壘]]，最终回到[[本壘|本壘]]，即可[[得分_(棒球)|得分]]。擊球員若無法得分，則有可能形成[[出局_(棒球)|出局]]或[[殘壘|殘壘]]。 

補充說明：擊球員或有人稱之為打擊手，但打擊手這個說法並非正式用語。

== 击球次序 ==
比赛中球队教练会按击球员打击能力安排九名队员按固定次序上场打击，通常球队中打击能力较强的队员会被安排在第三四五棒，亦称为中心打线。
根据比赛规则不同，比赛中[[投手|投手]]可能不需要上场击球，由一名指定[[代打|代打]]代为上场击球。<ref>[http://baseball.sport.org.cn/jsgz/guize/2004-03-10/48410.html]，棒球规则：第6章 击球员。</ref>

== 装备 ==
*[[手套|手套]]：帮助击球员握棒击球。
*[[头盔|头盔]]、护肘、护腿板：保护击球员在被球砸到时减少伤害。
*[[钉鞋|钉鞋]]：帮助击球员加快跑垒速度。

== 特殊的打法 ==
在[[職業棒球選手|職業棒球選手]]間，特別被命名的打法（姿勢、揮棒法的總稱、打擊姿勢）舉出下列幾種。

* 一本足打法（單腳打法、稻草人打法、金雞獨立打法、佛朗明哥打法）
:配合投手的投球姿勢抬起靠投手那一側的腳之打法。很難學會，據說適合打全壘打，實際上是為了讓球靠近手而抬腳。不過使用此姿勢的全壘打選手很多。如王貞治是相當的有名。[[田淵幸一|田淵幸一]]、[[門田博光|門田博光]]、[[大豐泰昭|大豐泰昭]]等人也模仿了此姿勢。類似此打法的有[[木保達彥|木保達彥]]的'''伐木打法'''。

* 高爾夫打法
:由下方揮上來的打法。極端的上划打法。[[小鶴誠|小鶴誠]]、[[田代富雄|田代富雄]]、[[石井浩郎|石井浩郎]]、[[小久保裕紀|小久保裕紀]]等人採行此法。

* 天平打法
:[[近藤和彥|近藤和彥]]用過的打法。右手持球棒握柄，左手持球棒中間，使球棒在左肩上呈水平。

* [[神主打法|神主打法]]
:如神主一般將球棒立在面前的打法。[[岩本義行|岩本義行]]為始祖。[[落合博滿|落合博滿]]的三冠王也主要是由此打法達成的。其他著名選手有[[小笠原道大|小笠原道大]]、[[中村紀洋|中村紀洋]]、[[江藤智|江藤智]]等人的姿勢雖然也被稱為神主打法但各選手仍有相當差異在。

* 提燈打法
:[[中登志雄|中登志雄]]用過。伸縮身體來調整好球帶，瞄準對自己有利的擊球。 

* 蒟蒻打法
:[[梨田昌孝|梨田昌孝]]用過。扭曲身體擺動的打法。據說是為了舒緩緊張而開始的。現在梨田的學生[[大西宏明|大西宏明]]採行與此相近的打法。 

* 單手打法 
:韓國的-{[[朴正泰|朴正泰]]}-用過的打法。左手置於左大腿附近，以右手單獨握棒。另外，阪神[[金本知憲|金本知憲]]手腕受傷時曾用過類似此種的打法。

* [[鐘擺打法|鐘擺打法]]
:讓投手側的腳好像晃到捕手側似的動作。[[鈴木一朗|鈴木一朗]]、[[坪井智哉|坪井智哉]]等人。

* 螃蟹腳打法
:[[種田仁|種田仁]]所用的打法。被分類為開放性姿勢。右腳向三壘方向大幅跨出，呈螃蟹腳型態的特徵而得此名稱。

* [[同步打法|同步打法]]
:[[手塚一志|手塚一志]]所提倡的打法。大多數的一流職業選手使用。

必須注意的是，不管哪種打法都是先學會最基本打法後，個人依照自己的風格所產生出來的。一開始就模仿這些打法反而可能會妨礙選手習得正確的打法，因此少棒與青棒的教練多半會嚴厲要求這些地方。
