'''休达'''（{{lang-es|'''Ceuta'''}}），是[[西班牙|西班牙]]兩個位於海外的[[自治市|自治市]]之一，它位于[[馬格里布|马格里布]]的最北部，在[[直布羅陀海峽|直布罗陀海峡]]附近的[[地中海|地中海]]沿岸，与[[摩洛哥|摩洛哥]]接壤。它的面积大约为18.5km²。

休达有一座叫Monte Hacho的山丘，从它的顶峰可以俯瞰到休达的全景。在山丘上有一个西班牙駐軍的碉堡。Monte Hacho山和在[[摩洛哥|摩洛哥]]最北部的Jebel Musa山都有可能是希腊传说中的[[赫拉克勒斯|赫拉克利斯]]海角。

== 历史 ==
从前5世纪[[迦太基|迦太基]]人（另一說為[[腓尼基|腓尼基人]]）统治时期开始，休达的地理位置就让他成为了商路上的重镇以及军事要地。但在公元42年[[羅馬帝國|罗马]]人占领休达之后，这座贫穷的城市就只有了军事上的作用。约400年后，[[汪达尔人|汪达尔人]]赶走了罗马人，之后[[西班牙|西班牙]]的[[西哥特人|西哥特人]]和[[拜占庭|拜占庭]]帝国又相继占领这块地方。

710年，一支[[穆斯林|穆斯林]]的军队逼近休达，但在休达的[[西哥特|西哥特]]统治者[[朱利安|朱利安]]（同时也是Ghomara的国王）的劝说下，这支军队转而进攻[[西班牙|西班牙]]——据[[阿拉伯人|阿拉伯]]的史书记载，朱利安仇恨西班牙的原因是西班牙的西哥特国王[[罗德里克|罗德里克]]虐待他的女儿。因此，休达成为了这支[[塔里克·伊本·齊亞德|塔里克·伊本·齐亚德]]将军率领的[[柏柏尔人|柏柏尔]]军队进攻[[伊比利亚半岛|伊比利亚半岛]]的根据地。

後來一度曾被[[伊斯兰教|伊斯蘭]]勢力統治差不多五個世紀。自朱利安（約公元7世紀初）死后，[[阿拉伯|阿拉伯]]人开始统治这座城市，这招致了城市周围柏柏尔部落的仇视。740年，在[[马萨拉·阿尔·哈希尔|马萨拉·阿尔·哈希尔]]的领导下，一场以“[[聖戰主義|圣战]]”为名义而进行的叛乱摧毁了休达。直到9世纪，[[柏柏尔人|柏柏尔人]]的[[马哈克萨|马哈克萨]]部落在首领[[马哈克斯|马哈克斯]]的领导下重建了休达，以建立短暂的[[巴努·伊萨姆王朝|巴努·伊萨姆王朝]]。之后在他的孙子的领导下，伊萨姆王朝曾短暂地向[[马格里布|马格里布]]的阿拉伯人[[伊德里斯王朝|伊德里斯王朝]]效忠。最终在931年，伊萨姆王朝的国王让位于[[科爾多瓦_(西班牙)|科尔多瓦]]的[[后倭马亚王朝|后倭马亚王朝]][[哈里发|哈里发]][[阿卜杜勒·拉赫曼三世|阿卜杜勒·拉赫曼三世]]，伊萨姆王朝终结。1031年，后倭马亚王朝的哈里发逐渐失去统治权，王朝陷入混乱，最终在1084年被柏柏尔人建立的[[穆拉比特王朝|穆拉比德王朝]]取代。之后休达再次成为侵略西班牙的基地，1147年，取代穆拉比德王朝的[[穆瓦希德王朝|阿尔摩哈德王朝]]终于占领了西班牙。阿尔摩哈德王朝的统治持续到1242年被[[突尼西亞|突尼斯]]的[[哈夫斯王朝|哈夫斯王朝]]取代，在1232年曾经历了[[伊本·胡德|伊本·胡德]]的叛乱。哈夫斯王朝在休达的影响很快衰弱，1249年他们被驱逐出休达。

此后休达在政治上一直处于不稳定的状态。1309年至1415年间，统治休达的是{{link-en|菲兹王國|Kingdom of Fez}}。1415年，[[葡萄牙君主列表|葡萄牙国王]][[若昂一世|若昂一世]]又占领了休达，成了葡萄牙人的領地。為了消除[[伊斯兰教|伊斯兰教]]在[[伊比利亚半岛|伊比利亚半岛]]的影响，同时传播[[基督教|基督教]]。1668年1月1日，西班牙国王[[卡洛斯二世_(西班牙)|卡洛斯二世]]和[[葡萄牙|葡萄牙]]国王[[阿方索六世_(葡萄牙)|阿方索六世]]在[[里斯本|里斯本]]签订条约，葡萄牙把休达割让给西班牙。西班牙控制休达至今。

在文化上，现代的休达可以看作[[安達魯西亞|安达卢西亚]]的一部分，它距离[[安達魯西亞|安达卢西亚]]的[[加的斯省|加迪斯省]]海岸只有12英里。同时休达是一座国际城市，这里居住着大量的[[柏柏尔人|柏柏尔]][[穆斯林|穆斯林]]、[[犹太人|犹太人]]以及[[印度教|印度教]]徒。

== 自治市 ==
休达的全称是休达自治市（西班牙语：{{lang|es|'''Ciudad Autónoma de Ceuta'''}}），其性质介于普通城市和[[自治区|自治区]]之间。在取得自治地位之前，休达是[[加的斯省|加迪斯省]]的一部分。

休达目前是[[欧洲联盟|欧盟]]的一部分，而在1986年西班牙加入[[欧洲联盟|欧盟]]之前，休达是一个[[自由港|自由港]]。现在，休达则是[[欧盟|欧盟]]货币体系内的低税区。

== 人口 ==
根据1994年的[[人口普查|人口普查]]，休达的人口为71,926人。根據2011年的官方數字，人口增至82376。

== 气候 ==
休达属于受海洋影响的[[地中海气候|地中海气候]]，与周边西班牙和摩洛哥的很多城市类似，比如[[塔里发|塔里发]]、[[阿尔赫西拉斯|阿尔赫西拉斯]]和[[丹吉尔|丹吉尔]]。<ref>{{cite web|url=http://www.weatherbase.com/weather/weather-summary.php3?s=2306&cityname=Ceuta%2C+Ciudad+Aut%F3noma+de+Ceuta%2C+Spain&units=|title=Ceuta, Spain Climate Summary|publisher=Weatherbase|accessdate=8 December 2014}}</ref> 日均温差非常小，年平均气温18.8℃，日均最高气温21.4℃，日均最低气温15.7℃。<ref name="Standard climate values for Ceuta">{{cite web|url=http://www.aemet.es/es/serviciosclimaticos/datosclimatologicos/valoresclimatologicos?l=5000C&k=ceu|title=Standard climate values for Ceuta|publisher=Aemet.es|accessdate=11 August 2015}}</ref> 由于纬度原因，休达的冬季较为温暖。

== 领土争端 ==
[[摩洛哥|摩洛哥]]政府一直认为休达、[[梅利利亚|梅利利亚]]以及地中海上的一些无人岛屿——比如[[佩雷希爾島|佩雷希尔]]——是[[摩洛哥|摩洛哥]]的领土，和西班牙一直宣称[[直布罗陀|直布罗陀]]是西班牙的领土是同一性质。但无论是西班牙政府，还是休达和[[梅利利亚|梅利利亚]]的自治政府和市民，都声称这两座城市属于西班牙，和[[直布罗陀|直布罗陀]]是完全不同的性质，西班牙的南部也曾经被[[伊斯兰教|伊斯兰]]政权所统治，现在却是西班牙领土。但摩洛哥政府认为这种辩解没有意义。
