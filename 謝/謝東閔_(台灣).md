'''謝東閔'''（{{bd|1908年|1月25日|2001年|4月8日|catIdx=X謝}}），原名謝進喜，自號'''求生'''，[[台灣|台灣]]政治人物，[[彰化廳|彰化廳]][[北斗支廳|北斗支廳]][[二八水區|二八水區]]-{[[二八水庄|二八水庄]]}-人（今[[彰化縣|彰化縣]][[二水鄉|二水鄉]]光化村）。[[中華民國|中華民國]]第六任副總統，是[[台灣人|台灣人]]中第一位擔任此要職者。晚年被人們尊稱為'''謝求公'''<ref>[http://archives.lib.ntnu.edu.tw/doc/c1_2.pdf 國立臺灣師範大學歷任校長]，[[國立臺灣師範大學|國立臺灣師範大學]]數位校史館</ref><ref>[http://catalog.digitalarchives.tw/item/00/51/56/c0.html 謝東閔任事用人別具風格--紀念謝求公百年冥誕兼述其二三事]，中央研究院數位文化中心</ref>。二水鄉當地又有「東閔仙」的稱呼{{r|二八仔}}。親近[[中央俱樂部|CC派]]。

== 生平 ==
=== 早年生活 ===
[[File:二水謝東閔故居.jpg|thumb]]
謝東閔出生於小康之家，原名謝進喜。幼時就讀於二八水公學校（後改稱二水公學校，今二水國小），中學就讀臺中州立第一中學（今[[臺中一中|臺中一中]]）{{r|二八仔}}。在中學時代，謝東閔因為對[[日本|日本]]人統治[[殖民地|殖民地]]的不滿，所以萌生到[[祖國|祖國]][[中華民國|中華民國]]本土求學的念頭<ref>{{cite book|author=梅可望|title=古今人物的感情世界：華夏一百六十年指標人物心態管窺|date=2012年9月30日|publisher=威秀資訊|isbn=9789862219904|page=226|quote=下面是節自《歸返》一書中謝先生對冒險赴祖國大陸的回憶：「民國十四年春天，我已十九歲，決心前往祖國升學。……」在這種精神和決心之下，他義無反顧，從基隆搭船去日本門司，轉往上海，身上只有三百日圓可用。}}</ref>。

1925年，中學尚未畢業的謝東閔自台湾抵达[[日本|日本]]後，在[[長崎|長崎]]搭船前往[[上海|上海]]。

謝東閔先赴[[上海|上海]][[東吳法學院|東吳法學院]]，1925年再前往[[廣東|廣東]]讀書。1931年於[[廣州|廣州]][[中山大學|中山大學]]政治系畢業後，留在廣州中山大學擔任日文助教，並且在廣州、[[香港|香港]]等地寫稿維生。

=== 從政時期 ===
1931年，在廣州市自治會兼任幹事一職。

1936年，在軍事訓練委員會，擔任少校秘書。

1938年至1941年，在香港郵政總局郵電檢查處，擔任檢查員。

1942年，在[[桂林|桂林]]《廣西日報》社，先後擔任該報的電訊室主任、總編輯。隨後被邀請參加[[中國國民黨|中國國民黨]][[台灣省黨部|台灣省黨部]]的籌備工作。

1943年5月至1945年，成為中國國民黨臺灣黨部的執行委員，謝東閔在[[漳州|漳州]]、[[永安市|永安]]、[[福州|福州]]等地從事[[抗日|抗日]]活動。

1945年，[[抗日戰爭|抗日戰爭]]結束，以[[半山仔|半山仔]]身分，隨著[[中華民國|中華民國]][[國民政府|國民政府]]接收台灣的船隊回到台灣，同行的行政人員幹部有[[連震東|連震東]]等人。而後出任[[高雄州|高雄州]]接管委員會主任委員至1946年1月。

1946年後，歷任高雄縣政府（官派第一任）首任縣長至1947年、[[台灣省行政長官公署|台灣省行政長官公署]]民政處副處長（1946年10月）、[[台灣省政府|台灣省政府]]教育廳副廳長、[[台灣省立師範學院|台灣省立師範學院]]院長、[[台灣新生報|台灣新生報]]董事長、[[中國青年反共救國團|中國青年反共救國團]]副主任、中國國民黨中央黨部副秘書長、台灣省政府秘書長、台灣省議員、[[台灣省議會|台灣省議會]]副議長、議長。

1949年，[[四六事件|四六事件]]中，謝東閔擔任[[國立臺灣師範大學|國立臺灣師範大學]]（師大）校長，協助警方逮捕師大學生，{{FACT}}部分學生被捕後遭槍決。

[[File:桃園文昌宮台灣省主席謝東閔贈匾.jpg|thumb]][[台灣省主席|台灣省主席]]謝東閔贈匾，[[1972年|1972年]][[7月|7月]]。]]

1958年3月創辦私立實踐家政專科學校（後來發展成[[實踐大學|實踐大學]]），以培育家政專才改善社會為目標{{r|二八仔}}。

1971年在省主席任內，時任[[台中一中|台中一中]]校長段茂廷以紅樓為二層樓建築，而學校教室空間漸出現不足現象為由，倡議拆除紅樓改建四層樓混凝土大樓，消息一出，遭到師生及廣大校友激烈反對，段茂廷於是求援於當時台灣省議會議長謝東閔，謝東閔慨允協助，返母校演講謂「舊的不去、新的不來」，注定了紅樓被拆的歹運，在其影響下，台灣省政府迅速核撥經費，於1971年拆除全部紅樓，並改建為四層「莊敬樓」迄今。<ref>{{cite web|language=zh-tw|url=http://www.peoplenews.tw/news/445bd667-81a9-42f7-99ab-91788d4ee101|title=台灣新創！中一中紅樓拆了47年 校友會預計募2億5千萬原地重建|author=|publisher=民報|date=2018-03-25|quote=}}</ref>

1972年6月，任命謝東閔為台灣省政府主席，[[張豐緒|張豐緒]]為台北市長。<ref name="年表，第128頁">[[陳布雷|陳-{布}-雷]]等編著，《蔣介石先生年表》，台北，[[傳記文學|傳記文學]]出版社，1978年6月1日，第128頁</ref>蔣召見二人，垂詢省市各項建設情形，並慇囑積極貫徹行政為民服務的要求。<ref name="年表，第128頁"/>

1976年10月10日的[[雙十節|雙十節]]，謝東閔於[[台灣省政府主席|台灣省政府主席]]任內，[[王幸男郵包炸彈案|收到]]由[[台灣獨立運動|台灣獨立運動]]參與者[[王幸男|王幸男]]寄的[[炸彈|炸彈]]郵包，在拆封時炸彈爆炸，左手受創。醫院因擔心引發[[敗血症|敗血症]]，動手術予以切除，裝上[[義肢|義肢]]，從此永遠失去一臂。謝東閔也是迄今唯一在任內遭遇暗殺的省主席。

1978年到1984年间出任中華民國第六任副總統，為首位台籍副總統。卸任後，獲聘為[[總統府資政|總統府資政]]。

=== 去世 ===
2001年1月，因[[心肌梗塞|心肌梗塞]]被送往[[臺北榮總|臺北榮總]]就醫<ref name="新聞局"/>。同年4月8日，因心肌梗塞伴隨上呼吸道感染，最後於深夜11時40分病逝於[[外雙溪|外雙溪]]至善路的家中，享壽93歲<ref name="新聞局">{{cite web |url=http://internationalnewsstation.tw/ct.asp?xItem=22109&ctNode=4615 |title=張院長弔唁故總統府資政謝東閔 感念其一生為台灣建設史的縮影 |work=90年度行政院即時新聞 |author= |publisher=行政院新聞局 |accessdate=2014-12-07 |language=zh-tw |date= |deadurl=yes |archiveurl=https://web.archive.org/web/20141213103704/http://internationalnewsstation.tw/ct.asp?xItem=22109&ctNode=4615 |archivedate=2014-12-13 }}</ref>，骨灰安放在[[彰化|彰化]][[二水|二水]]示範公墓內的「謝增福公派下墓園」。

== 教育貢獻 ==
謝東閔鑑於家為國之本，欲國家富強、民生樂利，必先健全[[家庭|家庭]]，曾向時任[[中華民國副總統|中華民國副總統]][[陳辭修|陳辭修]]建議國家應成立三所[[專科學校|專科學校]]－體育專科學校、藝術專科學校及家事專科學校。此建議後經政府採納，因而陸續有省立體育專科學校（今[[國立臺灣體育運動大學|國立臺灣體育運動大學]]）、國立藝術學校（今[[國立臺灣藝術大學|國立臺灣藝術大學]]）的成立，唯家事專科學校的建議未被政府採納與執行，謝東閔便決定親自擘劃、建立一所家事專科學校（後正式創校時改稱家政專科學校）<ref>實踐五十年：實踐大學創校五十年校史特刊。頁13-15。</ref>，於1957年決定創辦[[家政|家政]]學校。1958年3月26日，謝東閔成立中華民國第一所家政學校——私立實踐家政專科學校（今[[實踐大學|實踐大學]]）。1984年任[[台北語文學院|台北語文學院]]名譽董事長。至今彰化縣田中鎮外環仍存有東閔路一段至三段，紀念謝東閔。

== 故居 ==
謝東閔在二水鄉的故居，現位在實踐大學附設二水鄉家政推廣實驗中心北側{{r|彰化縣}}。該故居原為三合院，位於該實驗中心的土地上{{r|彰化縣}}{{r|二八仔}}。謝東閔之母去世，舊居拆除只留下東護龍，並旋轉90度移到現今的位置上{{r|彰化縣}}{{r|二八仔}}。舊居土地之後蓋ㄇ字型大樓，1972年成立「私立實踐家政專科學校附設二水家政推廣中心」{{r|彰化縣}}{{r|二八仔}}。

== 家庭 ==
謝東閔的胞弟[[謝敏初|謝敏初]]，曾任台灣青果運銷合作社總經理及理事主席。長子[[謝孟雄|謝孟雄]]醫師，曾任第二屆[[監察委員|監察委員]]、[[考試院|考試院]][[典試委員|典試委員]]、[[台北醫學院|台北醫學院]]校長，實踐大學董事長。長媳[[林澄枝|林澄枝]]，謝孟雄之妻，曾任[[行政院文化建設委員會|行政院文化建設委員會]]主委，曾任[[國民黨|國民黨]]副主席。參子[[謝大立|謝大立]]設計師，曾任實踐大學媒體傳達設計學系系主任。

==其他==
在1980、90年代的台湾，有國民黨党内「八大老」之说。指[[黃少谷_(政治人物)|黃少谷]]、謝東閔、[[倪文亞|倪文亞]]、[[李國鼎|李國鼎]]、[[蔣彥士|蔣彥士]]、[[袁守謙|袁守謙]]、[[辜振甫|辜振甫]]等「七大老」，後加上[[陳立夫|陳立夫]]，共為「八大老」。 <ref>《[[民眾日報|民眾日報]]》，1990年3月10日，三版</ref>

[[總統府|總統府]]方面曾利用不同方法勸[[林洋港|林洋港]]退[[1990年中華民國總統選舉|選]]，包括請出國民黨內「八大老」勸退。
