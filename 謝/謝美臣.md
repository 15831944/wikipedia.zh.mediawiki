'''謝美臣·德·赫蘇斯·納西門托'''（Jemerson de Jesus Nascimento，{{bd|1992年|8月24日||}}）是[[巴西|巴西]]的職業足球運動員，司職[[中後衛|中後衛]]，現效力於[[法國足球甲級聯賽|法國足球甲級聯賽]][[摩納哥體育會足球會|摩納哥]]。<ref>{{Cite web|title = Jemerson set for AS Monaco {{!}} News {{!}} AS Monaco FC|url = http://www.asmonaco.com/en/article/jemerson-set-for-as-monaco-69924.html|website = www.asmonaco.com|access-date = 2016-01-31}}</ref>

2016年1月31日，謝美臣和[[摩納哥體育會足球會|摩納哥]]簽約5年。<ref>{{Cite web|title = Jemerson set for AS Monaco {{!}} News {{!}} AS Monaco FC|url = http://www.asmonaco.com/en/article/jemerson-set-for-as-monaco-69924.html|website = www.asmonaco.com|access-date = 2016-01-31}}</ref>.
