'''谢尔盖·格奥尔基耶维奇·戈尔什科夫'''（'''Сергей Георгиевич Горшков'''；'''Sergey Georgyevich Gorshkov'''，{{bd|1910年|2月6日|1988年|5月13日|catIdx=Горшков}}），苏联军事家，[[苏联元帅列表|苏联海军元帅]]，[[苏联英雄|苏联英雄]]。

== 生平 ==
*1910年2月6日：生于[[乌克兰|乌克兰]]（現[[赫梅利尼斯基州|赫梅利尼斯基州]]）；
*1927年：加入[[苏联海军|苏联海军]]；
*1931年：毕业于[[伏龙芝海军学校|伏龙芝海军学校]]；
*1931年——1932年：在[[黑海舰队|黑海舰队]]任[[驱逐舰|驱逐舰]]航海长；
*1932年——1939年：在[[太平洋舰队|太平洋舰队]]历任[[护卫舰|护卫舰]]舰长、[[驱逐舰|驱逐舰]]舰长、驱逐舰支队支队长，期间于1937年进[[驱逐舰|驱逐舰]]舰长班受训；
*1940年6月：任[[黑海舰队|黑海舰队]][[巡洋舰|巡洋舰]]支队长；
*1941年9月：于[[敖德萨|敖德萨]]保卫战中，指挥[[黑海|黑海]]的首批登陆兵在格里高里耶夫卡地域登陆，支援[[敖德萨|敖德萨]]防御地域的部队成功进行反突击；
*1941年10月：任[[亚速海|亚速海]]区舰队司令；
*1941年12月：于[[克赤|克赤]]——费奥多西亚登陆战役中，组织区舰队兵力，遣送登陆兵（约6，000人及军事装备）在[[克赤半岛|克赤半岛]]登陆；
*1942年：加入[[苏联共产党|苏联共产党]]（[[布尔什维克|布尔什维克]]）；
*1942年[[夏季|夏季]]：成功指挥区舰队支援外高加索方面军和北高加索方面军，苏军向新罗西斯克撤退后，区舰队冲出[[亚速海|亚速海]]进入[[黑海|黑海]]；
*1942年8月：任新罗西斯克防区副司令，参加新罗西斯克保卫战；
*1942年11月：代理[[高加索|高加索]]第47集团军司令，参加保卫[[高加索|高加索]]的战斗；
*1943年2月：复任[[亚速海|亚速海]]区舰队司令；
*1943年[[夏季|夏季]]：指挥区舰队于敌军和后方多次进行战术登陆，成功支援北高加索方面军消灭德军塔曼集团的作战及南方面军第44集团军在滨海方向上的进攻；
*1943年9月：组织区舰队遣送登陆兵占领奥西片科（别尔江斯克）港；
*1943年11月——12月：参加[[克赤|克赤]]——埃利季根登陆战役，后领导第56集团军渡过[[克赤海峡|克赤海峡]]挺进[[克里米亚|克里米亚]]；
*1944年4月：任[[多瑙河|多瑙河]]区舰队司令，参加雅西——[[基什尼奥夫|基什尼奥夫]]战役，此后率部突入[[多瑙河三角洲|多瑙河三角洲]]，参加[[贝尔格莱德|贝尔格莱德]]和[[布达佩斯|布达佩斯]]战役；
*1945年1月：任[[黑海舰队|黑海舰队]]分舰队司令；
*1948年——1951年：任[[黑海舰队|黑海舰队]]参谋长；
*1951年8月：任[[黑海舰队|黑海舰队]]司令；
*1955年7月：任[[苏联|苏联]]海军第一副司令；
*1956年1月：接替[[库兹涅佐夫|库兹涅佐夫]]任[[苏联|苏联]]国防部副部长兼海军总司令；
*1956年：当选为[[苏共|苏共]]中央候补委员；
*1961年起：担任[[苏共|苏共]]中央委员；
*1965年5月7日：被授予[[苏联|苏联]]最高荣誉称号——“苏联英雄”；
*1967年：被授予[[苏联元帅列表|苏联海军元帅]]军衔；
*1970年代：两次指挥[[苏联海军|苏联海军]]在世界各大洋举行大规模[[军事演习|军事演习]]；
*1985年12月：任[[苏联|苏联]][[国防部|国防部]]总监察员；
*1988年5月13日：去世。

== 著作 ==
戈尔什科夫著有《海军学术的发展》、《战争年代与和平时期的海军》、《国家的海上威力》等著作。

== 紀念 ==
[[蘇聯海軍|蘇聯海軍]]的[[基輔級航空母艦|基輔級航空母艦]]的[[戈爾什科夫海軍上將號航空母艦|4號艦]]，以及[[俄罗斯海军|俄罗斯海军]]的[[22350型护卫舰|22350型护卫舰]]以其命名。

== 评价 ==
戈尔什科夫是20世纪苏联海军史上很有成效的一任海军总司令。他是[[苏联|苏联]]军事领导人中在一个高级军事职务上唯一连续任职长达近30年的高级将领，历经[[赫鲁晓夫|赫鲁晓夫]]、[[勃列日涅夫|勃列日涅夫]]、[[安德罗波夫|安德罗波夫]]、[[契尔年科|契尔年科]]四任[[苏共|苏共]][[总书记|总书记]]时期。[[美国|美国]]前[[美国海军部长|海军部长]][[小约翰·莱曼|小约翰·莱曼]]曾说：“关于“戈尔什科夫舰队”特别令人不安的是在广阔的大洋上，他们的单位正在到处提升能力，他们的潜艇更快了，更安静了并有了更好的感应和自卫能力。水面舰只带上了更好的雷达和导弹，飞机有了更久的寿命和更大载荷，而操纵这支苏联的平衡化海军的人更加训练有素和自信了。”<ref>有毫无来源的说法称莱曼用了“天才般的才能”等措词 {{cite web |url=http://www.navy.mil/navydata/cno/n87/usw/issue_28/yankee.html |title=存档副本 |accessdate=2013-04-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20130723022550/http://www.navy.mil/navydata/cno/n87/usw/issue_28/yankee.html |archivedate=2013-07-23 }}</ref>在其执掌[[海军|海军]]大权29年又11个月零2天的日子里，大力发展[[战略导弹潜艇|战略导弹潜艇]]和[[远程航空兵|远程航空兵]]，同时均衡发展其他[[海军|海军]][[兵种|兵种]]，为[[苏联海军|苏联海军]]的壮大和发展作出了卓越的贡献。1962年[[古巴导弹危机|古巴导弹危机]]大大刺激了苏联海军的发展，经过长达20年之久的不懈努力，在他领导下的[[海军|海军]]终于由一支“要塞舰队”发展为一支“远洋导弹核海军”。

[[戈尔什科夫|戈尔什科夫]]也是一位颇有建树的军事理论家和战略思想家：在他领导下苏联开始发展远洋进攻能力。在他的《战争年代与和平时期的海军》、《国家的海上威力》等作品中，阐述了他“积极进攻”的观点，对[[苏联|苏联]][[海军|海军]]军事学术理论的发展产生了重大影响，因此被誉为“现代苏联海军之父”。一艘[[航空母舰|航空母舰]]亦以他命名。他很喜欢一句格言：“要更好，不要够好（"Лучшее - враг хорошего"）”并把它挂在办公室墙上，这句话被《猎杀红色十月》引用过，也曾经被诺尔玛在1983年第三版《苏联海军指南》中引用。<ref>Polmar, N: Guide to the Soviet Navy, p. xii (upper left corner), 1983. </ref>
