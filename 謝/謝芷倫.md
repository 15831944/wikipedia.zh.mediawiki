'''謝芷倫'''（{{lang|en|'''Jan Tse Chi Lun'''}}，{{bd|1988年|10月14日|catIdx=Tse謝芷倫}}<ref>{{cite web |url=https://www.instagram.com/p/BaO4x2tHFtV/ |title=生日同博思過，放飯！ |work=[[Instagram|Instagram]]（jan.tse） |date=2017年10月14日}}{{zh-hant}}</ref>），[[香港|香港]]女演員、主持和模特兒。前[[香港電視網絡|香港電視網絡]]藝人，現為[[無綫電視藝員列表|無綫電視基本藝人合約]]藝員。

== 簡歷 ==
謝芷倫早年曾就讀[[合一堂學校|合一堂學校]]和[[協恩中學|協恩中學]]，曾擔任田徑隊副隊長；<ref>{{cite web |url=http://orientaldaily.on.cc/cnt/sport/20100421/00286_003.html |title=謝芷倫跨入光影世界 |work=[[東方日報_(香港)|東方日報]] |date=2010年4月21日}}{{zh-hant}}</ref>中學畢業便到[[澳洲|澳洲]][[麥覺理大學|麥覺理大學]]升學，修畢[[工商管理系|工商管理系]]學士課程後就回港涉足演藝圈。2012年，她加入[[香港電視網絡|香港電視網絡]]，成為旗下藝員；2014年參加《[[2014年度香港小姐競選|2014年度香港小姐競選]]》，且以熱門佳麗姿態進入決賽，當晚為自己的跨欄賽事作旁述，了得口才獲主持讚賞；<ref>{{cite web |url=http://hk.epochtimes.com/b5/14/8/21/198025.htm |title=謝芷倫否認有合約糾紛 蔡曉瑩無虛報學歷 |work=[[大紀元時報|大紀元時報]] |date=2014年8月21日}}{{zh-hant}}</ref>翌年正式加入[[無綫電視|無綫電視]]。

2017年，謝芷倫參演該台[[處境劇|處境劇]]《[[愛·回家之開心速遞|愛·回家之開心速遞]]》中拜金[[港女|港女]]「林美香（Mandy）」一角而受到注目。<ref>{{cite web |url=https://www.hk01.com/%E5%A8%9B%E6%A8%82/103446/-%E9%96%8B%E5%BF%83%E9%80%9F%E9%81%9E-%E6%BC%94%E6%8B%9C%E9%87%91%E5%A5%B3Mandy%E5%85%A5%E6%99%92%E8%A1%80-%E8%AC%9D%E8%8A%B7%E5%80%AB%E5%85%B6%E5%AF%A6%E5%A5%BD%E6%A8%B8%E7%B4%A0-%E5%94%94%E5%A5%BD%E5%90%8D%E7%89%8C |title=【開心速遞】演拜金女Mandy入晒血　謝芷倫其實好樸素：唔好名牌 |work=[[香港01|香港01]] |date=2017年7月10日}}{{zh-hant}}</ref>另外，她現時亦以業餘芭蕾舞兼鋼琴教師為副業。
