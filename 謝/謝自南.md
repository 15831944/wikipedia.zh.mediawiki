'''謝自南'''{{BD|1907年|2月7日|3=1990年|4=7月7日}}{{Efn|關於謝自南的誕生年份記錄紛紜、莫衷一是；[[李乾朗|李乾朗]]《台灣古建築圖解事典》中載1906年、許雪姬《續修澎湖縣志．卷14．人物-{R|志}-》和張志成撰[[臺灣大百科全書|臺灣大百科全書]]則作1907年、臺灣民俗文物辭典又另記為1908年。因許雪姬《續修澎湖縣志．人物-{R|志}-》中提供謝自南出生及逝世的年月日，記載最為詳盡，遂採出生年為1907年之說法。}}，字'''啟薰'''，出身[[臺灣|臺灣]][[澎湖縣|澎湖]]後窟潭（重光里），傳統木造廟宇大木匠師與當代水泥建築的設計師。謝自南曾承包機場與發電廠等多項官方大型工程，亦參與廟宇修建工程達七十餘所，作品遍布澎湖、[[高雄市|高雄]]、[[日本|日本]]與[[新加坡|新加坡]]等地。<ref name=":0">{{Cite web|url=http://nrch.culture.tw/twpedia.aspx?id=8905|title=謝自南|accessdate=2018-10-23|author=張志成|date=2009-09-09|publisher=臺灣大百科全書}}</ref><ref name=":3">{{Cite web|url=https://penghu.info/OB45B54B2A8ABA7F1D32|title=謝自南|accessdate=2018-10-23|author=許雪姬|date=2005|publisher=Penghu.Info｜澎湖知識服務平台}}</ref>

== 師承 ==
謝自南受業於父親謝江，謝江出生於[[同治|同治]]九年（1870－卒年不詳），約莫於[[光绪|光緒]]11年（1885年）十五歲拜入同住後窟潭處的鄰居葉媽利門下<ref name=":1">{{Cite book|title=《宮廟巧藝：跨越傳統的葉根壯大木作技術》|last=林|first=世超|last2=張|first2=宇彤|publisher=文化部文化資產局|year=2018|isbn=9789860561944|location=台中市|pages=|language=zh-tw}}</ref>{{Rp|22}}，業界尊稱其為「江司」。<ref name=":0" />

葉媽利（1821－1895）早年曾赴[[金門縣|金門]]學習大木作技藝<ref name=":1" />{{Rp|22}}，金門匠藝含括[[同安縣|同安]]與[[廈門|廈門]]地區，皆屬[[泉州市|泉州]]匠派<ref name=":2">{{Cite journal|title=〈論澎湖傳統廟宇木構之形式淵源〉|author=林世超|url=https://www.penghu.gov.tw/phlm/ch/home.jsp?id=2&act=view&dataserno=201510080011|journal=《澎湖研究第十屆學術研討會論文輯》|publisher=澎湖縣文化局|issue=|doi=|others=|year=2011|volume=|page=|isbn=9789860315295|pmid=}}</ref>{{Rp|117-118}}，葉媽利傳藝於二子葉元與葉一貫，謝江是葉媽利「葉系」唯一的外傳弟子<ref name=":1" />{{Rp|25}}。

謝江收徒甚多，不僅傳藝於兒子謝建{{Efn|謝自南兄長之名在不同資料來源中記載有所出入；張志成於[[臺灣大百科全書|臺灣大百科全書]]作「謝建」，而林世超在《宮廟巧藝：跨越傳統的葉根壯大木作技術》與〈 論澎湖傳統廟宇木構之形式淵源〉論文中則作「謝見」。}}、謝自南、謝自北、謝自東與謝自西，也收姪子謝自本、外甥余金木、小舅胡朝印以及蔡帕等為徒，另成「謝系」一脈。<ref name=":2" />{{Rp|117-118}}

而「葉系」始祖葉媽利逝世於清、日政權之交的1895年，「葉系」第二代葉元未及傳承技藝便先於大正二年（1913）逝世，待葉元三子[[葉得令|葉得令]]（1905－2000）年長至17歲時，便於[[大正|大正]]九年（1920）拜謝江為師，「葉系」遂得以延續下去。<ref name=":1" />{{Rp|22-25}}

== 生平 ==
謝自南大正12年（1923）自馬公公學校畢業後，16歲起正式跟隨父兄謝江與謝建左右學習版築、木工與圖面繪製等技術，未及一年便公推為謝江的「木匠頭手」，17歲即可獨當一面「出師」，堪稱天分卓絕。<ref name=":0" /><ref name=":3" />

大正14年（1925），謝江延攬[[澎湖觀音亭|澎湖觀音亭]]整建工程，當時觀音亭的廟宇重修圖案係由時年僅18歲的謝自南所繪，足見其才華。<ref name=":3" />[[昭和|昭和]]8年（1933），謝江受聘為[[媽宮城隍廟|媽宮城隍廟]]修建的大木匠師，據稱廟身整體設計與現場工程實際上是由謝自南負責。<ref name=":3" /><ref name=":4">{{Cite book|title=《台灣古建築圖解事典》|last=李|first=乾朗|publisher=遠流|year=2003|isbn=957324957X|edition=2009年四版五刷|location=台北市|pages=162|language=zh-tw}}</ref>

昭和10年（1935）謝自南移居高雄經營營造廠，陸續承攬多份官方大型現代化工程，如[[大甲溪|大甲溪]]上游天冷發電所（今[[天輪發電廠|天輪發電所]]）、[[第二次世界大战|二戰]]末期（1943－1945）派往[[海南岛|海南島]]參與其他工程。謝自南在戰後返台，時已易幟，中華民國37年（1948）謝自南又參與[[臺北松山機場|松山機場]]擴建與[[桃園市|桃園]]軍用機場工程，卻由於正值[[国共第二次内战|國共第二次內戰]]，島內通貨膨脹嚴重、政府財力匱乏，謝自南雖完成兩項重大工程，卻落得大失血本。民國47年（1958），謝自南遭陷害入獄九個月餘，雖於翌年獲得平反出獄，卻從此不願再承接任何官方工程，回歸宮廟建築的本業，並以設計為主。<ref name=":0" /><ref name=":3" />
[[File:Singapore_Tempel_Thian_Hock_Keng_1.jpg|缩略图]]
民國48年（1959）重返寺廟工程的謝自南建築風格丕變，因應廟宇建材多以[[钢筋混凝土|鋼筋水泥]]為主，設計也多採北方宮殿的形式；這一時期的代表作有[[高雄市文武聖殿|高雄文武聖殿]]（1953）、[[日月潭文武廟|日月潭文武廟]]（1968）與[[嘉義九華山地藏庵|嘉義九華山地藏庵]]（1971）等等，其中位於[[南投縣|南投縣]]的日月潭文武廟在興建期間，時任總統的[[蔣中正|蔣介石]]（1887－1975）多次視察工程，次數達到六次之多。<ref name=":0" /><ref name=":3" />

民國61年（1972），[[新加坡天福宮|新加坡天福宮]]重啟修建，當地華僑委請謝自南繪製設計藍圖<ref name=":3" />，翌年7月6日，[[新加坡政府|新加坡政府]]指定天福宮為國定古蹟<ref>{{Cite web|url=http://www.beokeng.com/disptemple.php?temple=thian-hock-keng|title=Thian Hock Keng 天福宮|accessdate=2018-10-25|author=|date=|publisher=心想獅城}}</ref>。

== 個人生活 ==

* 除了建築本行之外，謝自南喜好研究數學，曾發表過數學相關的研究論文；另外在民國42年（1953）拜[[南管|南管]]樂器大師黃朝枝為師，後自組「清音南樂社」，曾率樂團赴[[美国|美國]]與[[日本|日本]]等學校交流。<ref name=":3" />
* 謝自南於昭和5年（1930）和歐菊結婚，生有三男七女；歐氏於民國64年（1975）往生，謝自南續絃黃妙丹，另生一男一女。<ref name=":3" />
*謝自南與第一任妻子所生的三女[[謝掌珠|謝掌珠]]，是臺灣首位女性飛行員。<ref name=":3" />
*謝自南外孫女{{Link-en|趙心綺|Cindy Chao}}為國際知名珠寶設計師，作品「牡丹」發表於2018年倫敦大師傑作展（Masterpiece London），並獲選「大師傑作獎」。<ref>{{Cite journal|title=〈國際時尚圈炙手可熱的設計師 趙心綺：她的珠寶藝術，啟蒙自台灣廟宇建築師〉|author=陳建豪|url=https://www.gvm.com.tw/article.html?id=14462|date=November 9, 2010|journal=遠見雜誌|issue=11月號|doi=|others=|year=2010|volume=|page=|language=zh-tw|pmid=}}</ref><ref>{{Cite news|url=https://www.chinatimes.com/realtimenews/20180726001382-260405|title=〈台灣珠寶設計師揚名國際 趙心綺奪獎、溫婉心攜手大品牌〉|author=陶福媛|date=2018-07-26|publisher=中時電子報|accessdate=2018-10-25|language=zh-tw}}</ref>

== 宮廟作品 ==

=== 風格 ===

==== 青壯期 ====
澎湖觀音亭與媽宮城隍廟為[[闔澎公廟|闔澎公廟]]，歷代不乏多次翻修，近代樣貌乃奠定於謝江、謝自南於[[台灣日治時期|日治時期]]的整建工程，其施作風格可歸屬謝自南早年代表作品。

澎湖觀音亭格局為五開間兩進落，俯瞰廟身呈現正方形，穿廊兼具護龍與迴廊的功能，且左右護龍後方各置鐘鼓樓（今不開放）的設計，當時放諸全臺灣係較為罕見。<ref name=":5">{{Cite book|title=《2010澎湖縣文化資產手冊》|last=陳|first=英俊|last2=高|first2=啟進|last3=林|first3=文鎮|last4=郭|first4=金龍|publisher=澎湖縣政府文化局|year=2010|isbn=9789860262797|location=澎湖縣|pages=|language=zh-tw}}</ref>{{Rp|47}}<gallery widths="150">

==== 中晚期 ====
謝自南五十歲後重操舊業，一改早年閩南式木架構簡樸的廟宇風格，此一時期的廟宇舉凡外觀、屋頂與斗栱多採用北方宮殿形式，正殿左右捨棄傳統常見的單層護龍，改置樓閣式的鐘鼓大樓，廟身闕巍壯闊，外觀更顯氣派華麗，此建築型態影響深遠，甚至成為1970年代宮廟興建的主流風格。<ref name=":4" /><gallery widths="150">
