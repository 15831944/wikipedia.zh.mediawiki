'''救世軍林拔中紀念學校'''（{{lang-en|The Salvation Army Lam Butt Chung Memorial School}}，簡稱：SALBCMS），為[[香港救世軍|救世軍]]的屬下[[小學|小學]]，位於[[香港|香港]][[新界|新界北]][[離島區|離島區]][[大嶼山|大嶼山]][[東涌|東涌西]][[逸東邨|逸東邨]][[逸東街|逸東街]]8號[[美逸樓|美逸樓]]小徑之間。對面之[[建築物|建築物]]為[[香港教育工作者聯會黃楚標學校|香港教育工作者聯會黃楚標學校]]。

==歷史==
{{Main|香港救世軍}}
-2001年：此校落成，此校為一所千禧型校舍

-2011-2012年度為此校十周年校慶

-2016-2017年度為此校十五周年校慶

==班別==
一至三年級分為A-D班，四至六年級分為A-E班。

; 精英班
此學校使用精英班制度。逢在56名內之學生將會被平均分配到A、B兩班。


另外,5-6年級的學生會直升,但班主任有可能變動。

==學生職務==
學生職務有很多種，最近更推出了「一人一職」。

; 班長
班長（男班長：{{lang-en|Monitor}}；女班長：{{lang-en|Monitress}}）為班內負責幫助老師，每班也設有，而普通話班及小組將會另外添設多一個。

; 雜務長／科長
雜務長及科長會幫助學科老師及班主任做平常的東西，例如派簿子等。大部分人都是被安排這職務。

服務團隊:

; 風紀
風紀負責幫忙老師監察秩序，其也分為隊長、副隊長、隊員。

; 圖書館管理員
圖書館管理員是幫忙圖書館館長進行職務。

; 資訊小先鋒
資訊小先鋒是負責管理電腦室秩序、協助同學前往電腦室。
