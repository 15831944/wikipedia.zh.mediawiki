'''国际电子竞技联合会'''（{{Lang-en|'''International e-Sports Federation'''}}）是一个位于[[韩国|韩国]][[首尔市|首尔市]]，旨在推动[[电子竞技|电子竞技]]合法化的国际组织。

== 组织使命 ==
国际电子竞技联合会的使命旨在推动[[电子竞技|电子竞技]]成为真正被认可的[[体育竞技|体育竞技]]项目，并成为负责推广、维护和支持电竞运动的全球性组织。<ref>{{cite web|url=http://ie-sf.com/en/about/introduction.php|title=IeSF|author=leSF|publisher=|accessdate=2015-09-02|deadurl=yes|archiveurl=https://web.archive.org/web/20150908092204/http://ie-sf.com/en/about/introduction.php|archivedate=2015-09-08}}</ref> 为了实现这个宗旨，国际电子竞技联合会重点关注在四个核心项目：
# 增加成员数量；
# 为国际电子竞技制定法规和标准；
# 通过人力资源计划培训电子竞技裁判；
# 举办一个国际承认的电子竞技冠军赛事。

为了成为一个真正意义上的国际体育组织，国际电子竞技联合会与国家机关、国际体育组织和电子游戏开发商展开合作，以实现在其管辖范围内整合整个电子竞技世界的伟大目标。
