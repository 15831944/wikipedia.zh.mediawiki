'''国家高山滑雪中心'''，位于[[中国|中国]][[北京市|北京市]][[延庆区|延庆区]][[张山营镇|张山营镇]][[海坨山|小海坨山]]，是中国[[高山滑雪|高山滑雪]]比赛和训练中心。

==简介==
国家高山滑雪中心位于[[北京市|北京市]][[延庆区|延庆区]]西北部的小海坨山主峰西南坡，西大庄科村东北侧<ref name=baogao/>。是[[2022年冬季奥林匹克运动会|2022年冬季奥林匹克运动会]]延庆赛区2个比赛场馆之一，将作为[[高山滑雪|高山滑雪]]比赛场地<ref>{{Cite web |url = http://news.xinhuanet.com/sports/2015-03/22/c_127609030_2.htm |title =  22日至29日 京张申冬奥迎“考官”|author =  |date = 2015-03-22 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://zhuanti.hebnews.cn/2015/2015-08/01/content_4940780.htm |title = 延庆赛区：雪橇雪车和滑雪大项中的高山滑雪项目 |author =  |date =2015-08-01  |publisher = 河北新闻网 |language =  |accessdate =  |quote =  }}</ref>。

2015年，[[北京松山国家级自然保护区|北京松山国家级自然保护区]]保护范围调整，将原保护区东侧的小海坨山主峰的东坡和北坡整体调入，将原保护区北部部分区域调出，调整后保护区总面积比原保护区面积增加31%，国家高山滑雪中心、[[国家雪车雪橇中心|国家雪车雪橇中心]]不在调整后的保护区范围内<ref>{{Cite web |url = http://news.sina.com.cn/c/2015-08-13/doc-ifxfxwks4802276.shtml |title =  北京延庆：冬奥赛场不在小海坨山保护区内|author =  |date = 2015-08-13 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。
