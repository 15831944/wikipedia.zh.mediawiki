《'''国家宝藏'''》是自2017年12月3日起在[[中国中央电视台|中国中央电视台]][[中国中央电视台综艺频道|综艺频道]]播出的文博{{efn|name=wenbo|“文博”指文物与博物馆}}探索类节目，由[[中央电视台|中央电视台]]、央视纪录国际传媒有限公司承制。第一季共十期，前九期为每期由[[故宫博物院|故宫博物院]]等九家博物馆提供馆藏的三件文物，由“001號講解員”[[張國立|張國立]]负责分别介绍守护三件国宝的守护人及历史背景，每件文物的明星守护人与节目助演以舞台剧的形式表演“前世传奇”，而后由该文物相关的人物、普通文物工作者或者与该文物文化内涵相关的从业者来讲述“今生故事”；每期播出后，该期博物馆的三件文物都将由民众进行投票，选出一件在故宮六百年特展中展示；第十期为入選文物揭晓。

== 博物馆 ==
据节目总导演-{于}-蕾称，故宫博物院是最早确定的博物馆，而其他八家是由故宫博物院院長[[单霁翔|单霁翔]]建议下确定的，这八家博物馆为2009年11月中旬由中国[[国家文物局|国家文物局]]公布的第一批[[中央地方共建国家级博物馆|中央地方共建国家级博物馆]]<ref>国家文物局、财政部，《关于开展中央地方共建国家级博物馆的通知》（文物博函〔2009〕1387号），2009年11月18日</ref>，而且片头介绍的次序也是按照2009年名单排列的。

== 文物的选择 ==
节目中九家博物馆各三件文物，一共播出了27件文物。据节目总导演于蕾称，文物选择首先是由各大博物馆提供给节目组一个备选名单，然后节目组再研究，而最终选定的文物不一定要选一级文物，不一定是用材珍贵、工艺考究的文物，而挑选那些背后充满了人文精神和情怀的东西，之所以选定这27件，是因为它们“有着荡气回肠的前世传奇和今生故事，到今天都还在产生影响”<ref>{{cite web|title=《国家宝藏》节目总导演：挑宝物先看有没有故事|url=http://www.chinanews.com/cul/2017/12-15/8400932.shtml|website=中国新闻网-新京报|accessdate=2018-02-27}}</ref>。

== 观众反应与媒体评论 ==
2017年12月10日，《華僑時報》关注了《國家寶藏》第一集播出后网络上的反应，称其“一夜刷屏”，認為《國家寶藏》是繼2017年年初大火的《[[見字如面|見字如面]]》《[[中國詩詞大會|中國詩詞大會]]》《[[朗讀者|朗讀者]]》三檔文化類節目之後的又一部好口碑作品，表明文化類節目的崛起趨勢已更爲明顯。<ref>{{cite web|title=9.4分《國家寶藏》刷爆B站；局座張召忠付費音頻|url=http://www.chinesepress.com/Network/49820.html|website=華僑網}}</ref>

2017年12月25日，《[[光明日报|光明日报]]》发表评论文章称，《国家宝藏》是继2016年纪录片《[[我在故宫修文物|我在故宫修文物]]》之后的又一与文物有关的现象级电视产品。该节目通过传统与现代相结合的方式，让国宝重器“说话”，拉近了文物、历史与公众的距离，以恰当的方式将文物的魅力、历史的细节展示、诠释给了公众。该节目为博物馆文化发展改编创新能力上的不足的现状开了个好头。该节目的成功既实现了一种高效的历史文化科普，也为发展博物馆经济乃至整个文博、文旅产业提供了有说服力的启示。统计显示，该节目开播后，通过“博物馆”一词搜索旅游产品的数据骤然上升了50%。<ref>{{cite web|author1=朱昌俊|title=《国家宝藏》走红，博物馆经济大有可为|url=http://epaper.gmw.cn/gmrb/html/2017-12/25/nw.D110000gmrb_20171225_4-02.htm|website=光明网|accessdate=2017-12-27}}</ref>

2017年12月28日，仅播出四集的《国家宝藏》获得第4届豆瓣电影年度榜单“最受关注的大陆电视综艺”奖<ref>{{cite web |title=第4届豆瓣电影年度榜单 The 4th Douban Film Annual Awards (2017) |url=https://movie.douban.com/awards/doubanfilm_annual/4/ |website=豆瓣}}</ref>。

2018年2月12日，中国新闻网文章称《国家宝藏》掀起“国宝热”，播出后“引爆了电视、网络和微信朋友圈”，属于现象级文化综艺节目。<ref name="zxw0212">{{cite web|title=《国家宝藏》圆满收官 节目将推同名图书|url=http://www.chinanews.com/cul/2018/02-12/8447304.shtml|website=中国新闻网|accessdate=2018-02-25}}</ref>

《国家宝藏》的热播还获得了包括《人民日报》、《光明日报》、《中国日报》等在内的20家权威媒体的关注，超40次的大篇幅评论引起全民对《国家宝藏》的关注。该节目最后一集播出后，已有豆瓣网友近5万人参与打分，评分为9.2分，且播出期间曾创下最高评分9.5分的好成绩。该节目为2017豆瓣最受关注的大陆综艺节目榜单中的第一名。截止到2月11日13:00，在新浪微博上，《国家宝藏》主话题#CCTV国家宝藏#阅读量达17.1亿，粉丝讨论量破100万，居文化类综艺节目第一；节目相关视频仅在微博秒拍播放量近4亿，位列在播文化综艺播放量排名第一。<ref name="zxw0212"/>

在获得好评的同时，《国家宝藏》部分内容的正确性也受到了文博专业人士的质疑，片中对贾湖骨笛来源的解说出现了事实错误。同时，部分演出服装也并不合适。有学者指出，博物馆的作用应当是提升公众的文化素养而非迎合。此外，观众对一些演员的演技也提出了批评<ref>{{Cite web|url=http://www.cankaoxiaoxi.com/culture/20180112/2251684.shtml|title=《国家宝藏》真的做到文博类节目的极致了吗？|accessdate=2018-02-26|author=赵一尘|date=2018-01-12|work=|publisher=参考消息网}}</ref>。

== 其他相关 ==
《国家宝藏》的节目口号“让文物活起来”出自中共中央总书记[[习近平|习近平]]谈文物保护工作的三句箴言“让收藏在博物馆里的文物、陈列在广阔大地上的遗产、书写在古籍里的文字都活起来”{{efn|“三句箴言”曾被习近平多次提及，还有一个版本为“让收藏在禁宫里的文物、陈列在广阔大地上的遗产、书写在古籍里的文字都活起来”<ref>{{cite web|title=让书写在古籍里的文字活起来|url=http://newpaper.dahe.cn/hnrb/html/2015-10/22/content_1323484.htm|author=王守国| website=河南日报|accessdate=2018-02-26}}</ref><ref>{{cite web|author1=潘婧瑶|title=习近平谈文物保护工作的三句箴言|url=http://politics.people.com.cn/n1/2016/0413/c1001-28273470-3.html|website=人民网}}</ref>}}<ref>{{cite web|title=《国家宝藏》总导演详述幕后 如何让国宝"活起来"|url=http://www.xinhuanet.com/zgjx/2017-12/25/c_136850017.htm|website=中国记协网|accessdate=2018-02-26}}</ref>。

2018年2月11日，《国家宝藏》节目组在北京举行收官观影活动，制片人、总导演于蕾在活动现场透露，《国家宝藏》确定将制作[[国家宝藏_(第二季)|第二季]]，并将联合[[中信出版社|中信出版社]]出版相关书籍，更详尽地介绍节目中的27件国宝。据悉，该书籍将于2018年3月预售，4月正式发售<ref name="zxw0212"/>。

2018年2月15日除夕[[2018年中国中央电视台春节联欢晚会|中国中央电视台春节联欢晚会]]上，《国家宝藏》以特设环节“国宝回归”的形式亮相，节目内容为2017年底香港商人[[许荣茂|许荣茂]]斥亿元购回《[[丝路山水地图|丝路山水地图]]》并捐赠给故宫博物院的事情，节目中张国立以“《国家宝藏》001号讲解员”身份向大家介绍了捐赠概况，许荣茂和故宫博物院院长单霁翔登台讲述了具体情况<ref>{{cite web|title=春晚舞台国宝“丝路”回归 王菲那英携手“回忆杀”|url=http://news.hnr.cn/shxw/201802/t20180216_3068971.html|website=映象网|accessdate=2018-02-26}}</ref><ref>{{cite web|title=让文物“回家” 盘点那些寻回国家宝藏的港澳台人士|url=http://hope.huanqiu.com/toutiao/2018-02/11624138.html|website=环球网-新华网|accessdate=2018-02-26}}</ref>。

2018年4月16日，中国中央电视台《[[开门大吉|开门大吉]]》节目播出《国家宝藏》助演团专场，参加节目有朱峰、陈湘岳、董彥麟、李依洋、李岱、冒海飞、蔡忻如、刘洋和趙振藝共九位前世传奇助演演员。<ref>{{cite web|title=【《国家宝藏》官方番外】《开门大吉》之《国家宝藏》助演团专场|url=https://www.bilibili.com/video/av22197717/|website=哔哩哔哩|accessdate=2018-05-26}}</ref><ref>{{cite web|title=《开门大吉》 20180416 《国家宝藏》助演团|url=http://tv.cctv.com/2018/04/16/VIDE1DbVjl43xnBrFZXp3bWs180416.shtml|website=央视网|accessdate=2018-05-26}}</ref>
