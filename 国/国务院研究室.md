'''国务院研究室'''，简称'''国研室'''，是承担综合性政策研究和决策咨询任务的[[国务院办事机构|国务院办事机构]]，主要为[[国务院常务会议#组成人员|国务院主要领导人]]服务。其财务及后勤服务由[[国务院办公厅|国务院办公厅]]代管<ref name="官方简称">{{cite web |author1=国务院办公厅秘书局 |title=关于转发国务院机构简称的通知 |url=http://www.gdllw.org/tzgg/201806/W020180622515099385181.pdf |website=广东老龄网 |accessdate=2018-11-25 |location=北京 |language=zh-cn}}</ref>。

==职责==
根据《{{s|国务院研究室职能配置、内设机构和人员编制规定}}》，国研室承担下列职能<ref name="三定方案">{{cite web |author1=国务院办公厅 |title=国家统计局职能配置、内设机构和人员编制规定 |url=http://www.gov.cn/gjjg/2005-12/26/content_137261.htm |website=中华人民共和国中央人民政府门户网站 |accessdate=2018-11-25 |location=北京 |language=zh-cn}}</ref>：
{{columns-list|2|
# 负责组织或参与对改革开放和经济社会发展中的重大问题进行调查研究和决策咨询，提出政策性建议和咨询意见。
# 负责起草《政府工作报告》，牵头组织国务院重要会议的文件起草，参与党中央重要会议的文件起草工作。
# 根据国务院领导同志指示，单独或组织、协同有关方面起草、修改国务院有关重要文件，起草国务院领导同志部分重要讲话等文稿。
# 对国内国际经济形势、各主要国家经济政策和社会发展政策进行分析和研究，提出政策建议。收集、分析、整理和报送经济社会发展的重要信息、动态，为国务院决策提供参考建议。
# 承办国务院交办的其他事项。
}}
