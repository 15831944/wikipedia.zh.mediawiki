《'''国家命运'''》，2012年上映于[[中国大陆|中国大陆]]的史诗剧，讲述了以[[钱学森|钱学森]]、[[邓稼先|邓稼先]]、[[钱三强|钱三强]]、[[张洁|张洁]]为主的[[科学家|科学家]]们经过千辛万苦研制出“[[两弹一星|两弹一星]]”的过程，主演是[[韩中|韩中]]，[[刘劲|刘劲]]，[[谢钢|谢钢]]，[[史鑫|史鑫]]，[[胡亚捷|胡亚捷]]。

== 剧情 ==
1953年，[[抗美援朝|抗美援朝]]的[[朝鲜战争|朝鲜战争]]爆发，战争不久之后，[[美国|美国]]看到[[中华人民共和国|中华人民共和国]]在朝鲜战争中的强势，想要动用[[核武器|核武器]]打击[[中国人民志愿军|中国人民志愿军]]，毛泽东、周恩来、朱德、刘少奇等人得知这一消息之后，开始向[[苏联|苏联]]提出要求，请求在核武器技术方面的援助，但是遭到[[斯大林|斯大林]]和其继任者[[赫鲁晓夫|赫鲁晓夫]]的拒绝，在万般无奈之下，毛泽东等拍板决定中国依靠自己的力量研制核武器，于是，朝鲜战争一结束，周恩来就用美国战俘换回来科学家钱学森，随后，组织了一批科学家，开始研制核武器。

== 制作 ==
该剧的制作原则是“大事不虚，小事不拘”。<ref>{{cite web |url=http://yule.sohu.com/20121009/n354514490.shtml |title=《国家命运》央一开播 吕一丁演绎名将别样风采 |accessdate=2012年10月9日 |publisher= 搜狐网|work= |language =zh-cn}}</ref>

该剧的制作方是：浙江华策影视股份有限公司、[[中国人民解放军总装备部电视艺术中心|中国人民解放军总装备部电视艺术中心]]、北京天星亿源影视文化传播有限公司、中国电视剧制作中心有限责任公司。<ref name="fish " >{{cite web |url=http://ent.qq.com/a/20121007/000137.htm|title=《国家命运》今开播 看两弹一星精神如何炼成 |accessdate=2012年10月7日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

该剧的拍摄是实地取景，包括[[酒泉卫星发射中心|酒泉卫星发射中心]]、[[中国核试验基地|中国核试验基地]]、[[中国航天科技集团|航天科技集团]]、[[中国航天科工集团|航天科工集团]]，由[[中央军委委员|中央军委委员]]、[[中国人民解放军总装备部|中国人民解放军总装备部]]部长[[常万全|常万全]]等负责编剧，总用了三千多兵员。<ref>{{cite web |url=http://ent.qq.com/a/20121005/000110.htm |title= 《国家命运》将播 两弹一星元老影像当“主演”|accessdate=2012年10月5日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

== 花絮 ==
该剧被中国国家广电总局选为[[中共十八大|中共十八大]]的重点献礼剧。<ref>{{cite web |url= http://yule.sohu.com/20121009/n354502848.shtml|title=《国家命运》热播 王乐君还原女科学家多面人生 |accessdate=2012年10月9日 |publisher=搜狐网 |work= |language =zh-cn}}</ref>

谈到钱学森，饰演者[[胡亚捷|胡亚捷]]说：“既然是史实作品，就要最大程度的还原伟人的一切，我是心怀敬畏的走近钱学森这个角色的，作为一名[[炎黄子孙|炎黄子孙]]，我会永远跪拜[[中华民族|中华民族]]的英灵，这是我的态度。”<ref>{{cite web |url=http://ent.163.com/12/1009/11/8DCCSRK400031GVS.html |title=《国家命运》央视开播 胡亚捷演绎写实版钱学森 |accessdate=2012-10-09  |publisher=网易 |work= |language =zh-cn}}</ref>

== 评价 ==
中国国家广播电影电视总局电视剧管理司司长[[李京盛|李京盛]]也评价：“该剧对以往，无论是电影、电视剧、写同类题材的作品都是一种超越，还有当下的现实意义。”<ref name="fish " />
