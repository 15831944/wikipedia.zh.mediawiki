'''国防专利审查中心'''，位于[[北京市|北京市]][[海淀区|海淀区]][[新街口外大街|新街口外大街]]23号，是[[中央军委装备发展部|中央军委装备发展部]]直属[[事业单位|事业单位]]<ref name=jiuye/>。

==简介==
2008年，国防专利审查中心成立。主要职责是承担国防专利的申请、受理和审查以及相关研究培训工作<ref name=beijing/>。这是[[中华人民共和国|中华人民共和国]]唯一承担国防专利审查工作的军队[[事业单位|事业单位]]<ref name=jiuye/>。国防专利由[[中央军事委员会装备发展部国防知识产权局|中国人民解放军总装备部国防知识产权局]]受理并审查，具体工作由国防专利审查中心承担<ref>{{Cite web |url =http://www.caigou2003.com/shouye/recijiedu/2015-02-10/2994.html  |title = 国防专利 |author =  |date = 2015-02-10 |publisher = 政府采购信息网 |language =  |accessdate =  |quote =  }}</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，国防专利审查中心由中国人民解放军总装备部转隶中央军委装备发展部<ref name=jiuye>{{Cite web |url = http://job.bit.edu.cn/main/news/jobs/detail9/Article/News/detail/27085.html |title =国防专利审查中心  |author =  |date =2017-03-15  |publisher = 北京理工大学就业信息网 |language =  |accessdate =  |quote =  }}</ref>。
