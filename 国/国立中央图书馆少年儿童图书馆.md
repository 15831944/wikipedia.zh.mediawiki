'''国立中央图书馆少年儿童图书馆'''<ref>{{cite web |title=국어청, 테마 도서전시 '뒹굴 뒹굴 만화책방' 공개 |url=https://kizmom.hankyung.com/news/view.html?aid=201812280209o |website=KizMom |accessdate=2019-01-16}}</ref><ref>{{cite web |title=천안성환도서관 겨울방학 독서교실 운영 |url=http://www.ccdailynews.com/news/articleView.html?idxno=993829 |website=충청일보 |accessdate=2019-01-16}}</ref>（{{lang-ko|국립어린이청소년도서관}}）位於[[大韓民國|大韓民國]][[首爾|首爾]][[江南區_(首爾)|江南區]]，於2006年6月28日成立，為韓國的國家圖書館之一，該圖書館負責處理專門為兒童和青少年提供圖書服務的工作。从2018年5月开始，该图书馆会每月举办一次儿童编程活动<ref>{{cite web |author1=육준수 기자 |title=국립어린이청소년도서관, 2018년 창의 코딩 프로그램 '도서관에서 배우다' 종료 |url=http://www.news-paper.co.kr/news/articleView.html?idxno=31433 |website=뉴스페이퍼 |accessdate=2019-01-16}}</ref>。

==設施==
*地下2樓 - 自助餐廳/商店
*地下1樓 - 兒童獨島體驗中心、嬰兒庇護所（護理室）
*1樓 - 兒童室、外國兒童室
*2樓 - 多媒體室、視頻音響室、動手童話室、圖書室、展覽室
*3樓 - 青年資源中心、研究資源中心
*4樓 - 禮堂、研討室、閱讀討論室
