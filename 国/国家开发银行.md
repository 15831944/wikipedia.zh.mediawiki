'''国家开发银行'''（英语：{{lang|en|China Development Bank}}，[[縮寫|縮寫]]：'''CDB'''），简称'''国开行'''、'''开发银行'''，是[[中华人民共和国|中华人民共和国]]的一家[[政策性银行|开发性金融机构]]。

1994年3月17日，中华人民共和国财政部和中央汇金投资有限责任公司共同发起设立国家开发银行，作为国有独资政策性银行。2008年12月11日正式改制为[[股份有限公司|股份有限公司]]，其改制标志着中国政策性银行的改革取得了进展。2015年3月，[[中华人民共和国国务院|中华人民共和国国务院]]明确其定位为开发性金融机构<ref>{{cite web |url=http://www.cdb.com.cn/web/NewsInfo.asp?NewsId=3359 |title=存档副本 |accessdate=2013-05-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20101204101857/http://www.cdb.com.cn/web/NewsInfo.asp?NewsId=3359 |archivedate=2010-12-04 }}</ref>。2017年4月19日起改制为[[有限责任公司|有限责任公司]]。

国家开发银行是中国最大的债券银行、最大的对外投融资合作银行以及全球最大的开发性金融机构。截至2016年底，资产总额超过14万亿元人民币，总行设有31个职能部门和住宅金融事业部、扶贫金融事业部，在沪设有协调性机构上海业务总部；境内设37家一级分行和3家二级分行，境外设香港分行和开罗、莫斯科、里约热内卢、加拉加斯、伦敦、万象等6家代表处。

== 历史 ==
1994年3月17日，国家开发银行成立。1998年12月，中国投资银行并入国开行。

2009年6月25日，[[香港特别行政区|香港特别行政区]][[金融管理专员|金融管理专员]]已根据《银行业条例》向国家开发银行授予银行牌照，并於当日生效。目前，香港分行有贷款餘额近3000亿港元，约佔国开行外币贷款餘额的六分之一。

2009年5月，经中国银监会批准<ref name="cbrc2009">{{cite web|author1=政策银行部|title=中国银监会关于同意国家开发银行股份有限公司设立香港分行的批复|url=http://www.cbrc.gov.cn/govView_1EF36E0DA5914F5FA574AD0B6EE8E667.html|accessdate=2017-09-07|date=2009-05-19|language=zh-hans}}</ref>，香港代表处升格为首家境外分行——香港分行。

2010年，国家开发银行获[[社会保障基金_(中国)|国家社会保障基金]]以200亿元[[人民币|人民币]]入股。<ref>[http://finance.sina.com.hk/cgi-bin/nw/show.cgi/41/2/1/3030264/1.html 社保230亿入股国开行] 星岛日报，2010年8月1日</ref>

2014年9月，推动设立总规模达1200亿元的国家集成电路产业基金。2015年3月20日，国务院批复国开行深化改革方案<ref name="gov">{{cite web|author1=[[国务院|国务院]]|title=国务院关于同意国家开发银行
深化改革方案的批复|url=http://www.gov.cn/zhengce/content/2015-04/12/content_9596.htm|accessdate=2017-09-07|date=2015-04-12|language=zh-hans}}</ref>，明确了其[[开发性金融机构|开发性金融机构]]定位及相关政策支持和制度安排。2015年7月，[[国家外汇管理局|梧桐树投资平台有限公司]]注资国开行480亿美元。

2016年11月，国务院审定批准《国家开发银行章程》，标志着开行债信、集团架构和立法改革“三步走”战略的实现。

== 业务范围 ==
根据《{{s|国家开发银行章程}}》和国家有关规定，国家开发银行是直属国务院领导的政策性金融机构，在金融业务上接受[[中国人民银行|中国人民银行]]的指导和监督。国家开发银行经营下列业务：
{{columns-list|2|
# 管理和运用国家核拨的预算内经营性建设基金和贴息资金；
# 向国内金融机构发行金融债券和向社会发行财政担保建设债券；
# 办理有关的外国政府和国际金融组织贷款的转贷，经国家批准在国外发行债券，根据国家利用外资计划筹借国际商业贷款等；
# 向国家基础设施、基础产业和支柱产业的大中型基本建设和技术改造等政策性项目及其配套工程发放政策性贷款；
# 办理建设项目贷款条件评审、咨询和担保等业务。为重点建设项目物色国内外合资伙伴，提供投资机会和投资信息；
# 经批准的其他业务。
}}

== 业务 ==
国家开发银行主要服务构建开放型经济新体制，促进中国债券市场建设和助力宏观调控政策实施。其贷款对象集中于公共基础设施、公路、铁路、水利、电力等领域政府类客户和大中型企业客户，同时，通过建立社会化合作机制，以批发的方式开展助学贷款、小微企业贷款等普惠金融业务。

国家开发银行自创立以来，已经向[[黄河|黄河]]、[[长江|长江]]、[[西部大开发|西部大开发]]、[[振兴东北|振兴东北]]、[[新农村建设|新农村建设]]、[[三峡水利枢纽|三峡水利枢纽]]等国家重大建设项目和基础产业培育工程等数千个项目进行投资，[[贷款|贷款]]总额达1.6兆[[人民币|人民币]]。其资金来源主要通过在国际金融市场发行[[债券|债券]]筹措。2015年，在美国《财富》杂志世界企业500强中排名第87位<ref name="fortune">{{cite web|work=[[财富_(杂志)|财富]]|title=国家开发银行|url=http://www.fortunechina.com/global500/636/2015|accessdate=2017-09-07|date=2015|language=zh-hans}}</ref>。
