'''国家机关事务管理局北戴河接待服务中心'''（'''国家机关事务管理局北戴河服务局'''），是[[国家机关事务管理局|国家机关事务管理局]]直属[[事业单位|事业单位]]。

==沿革==
原名'''国务院机关事务管理局北戴河接待服务中心'''（'''国务院机关事务管理局北戴河服务局'''）。2013年，[[国务院机关事务管理局|国务院机关事务管理局]]更名[[国家机关事务管理局|国家机关事务管理局]]，该中心随之更名'''国家机关事务管理局北戴河接待服务中心'''（'''国家机关事务管理局北戴河服务局'''）<ref>{{Cite web |url =http://www.gjsy.gov.cn/djgg/bggg/201404/t20140411_2662.html  |title =  国家事业单位登记管理局公告|author =  |date = 2014-04-11 |publisher =事业单位在线  |language =  |accessdate =  |quote =  }}</ref>。

==职责==
#负责指定的党和国家领导人、已退出领导岗位的党和国家领导人、最高人民法院院长、最高人民检察院检察长和已故领导人的配偶以及国务院指定的其他领导同志暑期在北戴河休养、度假的接待服务工作。
#负责国务院在北戴河办公、开会的接待服务工作。
#负责中心财务管理和国有资产管理工作。
#负责中心房屋的基建、修缮和物资设备的购置配备工作。
#负责中心的经营管理工作。
#负责中心消防安全、卫生和绿化美化工作。
#负责与地方政府及有关单位的协调保障工作。
#承办局领导交办的其他事项。
