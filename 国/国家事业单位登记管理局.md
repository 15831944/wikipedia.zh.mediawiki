'''国家事业单位登记管理局'''，是[[中央机构编制委员会办公室|中央机构编制委员会办公室]]直属正司局级中央财政全额拨款[[事业单位|事业单位]]，受权承担行政职能，在履行[[中华人民共和国|中华人民共和国]][[事业单位|事业单位]]登记管理职能时使用“'''国家事业单位登记管理局'''”名义。

== 沿革 ==
1998年，经[[中央机构编制委员会|中央机构编制委员会]]批准，'''中央机构编制委员会办公室事业单位登记管理中心'''成立，为中央机构编制委员会办公室直属[[事业单位|事业单位]]，承担《事业单位登记管理暂行条例》中规定的事业单位登记管理职能，并在登记管理活动中使用“'''国家事业单位登记管理局'''”的牌子以及印章。

2000年，《中央机构编制委员会办公室机构改革方案》（厅字〔2000〕30号）明确中央编办增加事业单位登记管理职能<ref>{{Cite web |url = http://news.sohu.com/20130326/n370476442.shtml |title = 资料：改革开放以来的机构编制管理机构  |author =  |date = 2013-03-26 |publisher = 搜狐 |language = zh-cn |accessdate =  |quote =  }}</ref>。中央编办相应成立事业单位登记管理局，作为中央编办内设机构。

2009年，该局向[[政务和公益机构域名注册管理中心|政务和公益机构域名注册管理中心]]办理开通了“[http://国家事业单位登记管理局.政务 国家事业单位登记管理局.政务]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}”专用[[中文域名|中文域名]]<ref>{{Cite web |url = http://news.qq.com/a/20090225/001777.htm |title = “国家事业单位登记管理局．政务”域名开通使用 |author =  |date = 2009-02-25 |website = 腾讯 |language = zh-cn |accessdate = |quote = }}</ref>。

==职责==
根据有关规定，中央机构编制委员会办公室事业单位登记管理局（国家事业单位登记管理局）承担下列职能<ref>{{Cite web |url = http://www.scopsr.gov.cn/zybbzzjg/zzjg/201412/t20141204_268845.html |title = 中央编办内设机构及下属单位 |author =  |date = 2014-12-04  |publisher = 中国机构编制网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.gjsy.gov.cn/djgljg/zyzz/201204/t20120420_1323.html |title = 主要职责 |author =  |date =2012-04-20  |publisher = 事业单位在线 |language =  |accessdate =  |quote =  }}</ref>
{{columns-list|2|
# 拟订事业单位登记管理的方针、政策和规定，起草有关法律法规草案和部门规章，并组织实施和监督检查。
# 依法保护核准登记的事业单位有关登记事项的合法权益。
# 监督事业单位贯彻落实《事业单位登记管理暂行条例》（以下简称《条例》）和《事业单位登记管理暂行条例实施细则》（以下简称《实施细则），依法处理违反《条例》和《实施细则》的行为。
# 指导和监督检查地方登记管理机关的事业单位登记管理工作。
# 负责《实施细则》规定的管辖范围事业单位的登记管理工作。
# 指导和实施事业单位法人治理结构、绩效评估和年度报告公开工作。
# 组织和实施全国事业单位网上登记管理系统及信息管理工作。
# 承担事业单位改革相关工作。
# 开展有关事业单位登记管理的社会服务。
}}
