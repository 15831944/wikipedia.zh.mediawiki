'''国家统计局'''（官方英语译名：{{lang|en|'''National Bureau of Statistics'''}}），简称'''统计局'''，是[[中华人民共和国国务院|中华人民共和国国务院]]主管全国[[统计|统计]]和[[国民经济核算|国民经济核算]]的副部级[[国务院直属机构|直属机构]]<ref name="官方简称">{{cite web |author1=国务院办公厅秘书局 |title=关于转发国务院机构简称的通知 |url=http://www.gdllw.org/tzgg/201806/W020180622515099385181.pdf |website=广东老龄网 |accessdate=2018-11-25 |location=北京 |language=zh-cn}}</ref>。

== 沿革 ==
1949年10月，[[政务院财政经济委员会|政务院财政经济委员会]]于中央财经计划局内设'''统计处'''，后来改称'''统计总处'''<ref>{{Cite web|url =http://www.stats.gov.cn/ztjc/zthd/50znjn/tjlsnb/200206/t20020606_35569.html|title =1949年|author =|date =2002-06-06|publisher =中华人民共和国国家统计局|language =|accessdate =|quote =|deadurl =no|archiveurl =https://web.archive.org/web/20170216055747/http://www.stats.gov.cn/ztjc/zthd/50znjn/tjlsnb/200206/t20020606_35569.html|archivedate =2017-02-16|df =}}</ref>。

1952年8月7日，[[中华人民共和国中央人民政府_(1949-1954)|中央人民政府]]委员会第十七次全体会议决定成立'''中央人民政府国家统计局'''，为[[中央人民政府政务院|政务院]]直属机构<ref name=licheng>{{Cite web|url =http://www.stats.gov.cn/zjtj/gjtjj/201311/t20131108_457871.html|title =新中国政府统计历程|author =|date =2013-11-08|publisher =中华人民共和国国家统计局|language =|accessdate =|quote =|deadurl =no|archiveurl =https://web.archive.org/web/20161201171133/http://www.stats.gov.cn/zjtj/gjtjj/201311/t20131108_457871.html|archivedate =2016-12-01|df =}}</ref><ref>{{Cite web|url =http://www.stats.gov.cn/ztjc/zthd/50znjn/tjdjs/200206/t20020617_35692.html|title =国家统计局的成立|author =|date =2002-06-17|publisher =中华人民共和国国家统计局|language =|accessdate =|quote =|deadurl =no|archiveurl =https://web.archive.org/web/20170216051550/http://www.stats.gov.cn/ztjc/zthd/50znjn/tjdjs/200206/t20020617_35692.html|archivedate =2017-02-16|df =}}</ref>。1954年11月，[[第一届全国人民代表大会常务委员会|第一届全国人民代表大会常务委员会]]第二次会议批准，将中央人民政府国家统计局更名为'''中华人民共和国国家统计局''', 作为[[国务院直属机构|国务院直属机构]]。

[[文革|文革]]期间，1968年8月10日，[[中共中央|中共中央]]、国务院、[[中共中央军委|中央军委]]、[[中央文革|中央文革]]派军代表进驻国家统计局（军代表直到1972年8月才撤走）。1969年12月，撤销国家统计局，大批干部被下放到[[湖北|湖北]][[襄樊|襄樊]]的国家计委[[五七干校|“五七”干校]]劳动。从1967年到1969年，自中央到地方，绝大多数统计机构被撤销<ref>{{Cite web|url =http://www.stats.gov.cn/ztjc/zthd/50znjn/tjdjs/200208/t20020806_35721.html|title =国家统计局被撤消|author =|date =2002-08-06|publisher =中华人民共和国国家统计局|language =|accessdate =|quote =|deadurl =no|archiveurl =https://web.archive.org/web/20170216055435/http://www.stats.gov.cn/ztjc/zthd/50znjn/tjdjs/200208/t20020806_35721.html|archivedate =2017-02-16|df =}}</ref>。

1970年4月，国务院总理[[周恩来|周恩来]]指示：“统计工作不能取消，统计机构还要有，基本统计还是要搞的，但不能搞繁琐哲学。”1970年6月22日，国家计划委员会革命委员会生产组下设'''统计组'''，随后逐步恢复基本统计制度<ref>{{Cite web|url =http://www.stats.gov.cn/ztjc/zthd/50znjn/tjlsnb/200206/t20020606_35587.html|title =1970年|author =|date =2002-06-06|publisher =中华人民共和国国家统计局|language =|accessdate =|quote =|deadurl =no|archiveurl =https://web.archive.org/web/20170216050522/http://www.stats.gov.cn/ztjc/zthd/50znjn/tjlsnb/200206/t20020606_35587.html|archivedate =2017-02-16|df =}}</ref><ref>{{Cite web|url =http://www.stats.gov.cn/ztjc/zthd/50znjn/tjdjs/200208/t20020806_35733.html|title =1970年4月周恩来指示恢复统计工作|author =|date =2002-08-06|publisher =中华人民共和国国家统计局|language =|accessdate =|quote =|deadurl =no|archiveurl =https://web.archive.org/web/20170216060843/http://www.stats.gov.cn/ztjc/zthd/50znjn/tjdjs/200208/t20020806_35733.html|archivedate =2017-02-16|df =}}</ref>。

1978年3月21日，国务院通知：恢复中华人民共和国国家统计局，直属国务院。随后任命[[陈先|陈先]]为国家统计局局长。各省、自治区、直辖市陆续恢复统计局，作为同级人民政府的工作部门<ref>{{Cite web|url =http://www.stats.gov.cn/ztjc/zthd/50znjn/tjdjs/200208/t20020806_35722.html|title =国家统计局恢复|author =|date =2002-08-06|publisher =中华人民共和国国家统计局|language =|accessdate =|quote =|deadurl =no|archiveurl =https://web.archive.org/web/20170216045923/http://www.stats.gov.cn/ztjc/zthd/50znjn/tjdjs/200208/t20020806_35722.html|archivedate =2017-02-16|df =}}</ref><ref name=licheng/><ref>中华人民共和国国家统计局关于一九八○年国民经济计划执行结果的公报，中华人民共和国国务院公报1981年08期</ref>。

1982年8月23日，《全国人民代表大会常务委员会关于批准国务院直属机构改革实施方案的决议》决定在[[国务院机构改革|国务院机构改革]]中，保留国家统计局，作为国务院直属机构<ref>{{Cite web|url =http://www.npc.gov.cn/wxzl/gongbao/2000-12/26/content_5328225.htm|title =全国人民代表大会常务委员会关于批准国务院直属机构改革实施方案的决议|author =全国人大常委会|date =1982年8月23日通过|publisher =|language =|accessdate =|quote =|deadurl =no|archiveurl =https://web.archive.org/web/20170216050322/http://www.npc.gov.cn/wxzl/gongbao/2000-12/26/content_5328225.htm|archivedate =2017年2月16日|df =}}</ref>。1988年，国务院机构改革，根据1988年8月[[国家机构编制委员会|国家机构编制委员会]]批准的《国家统计局“三定”方案》，国家统计局是国务院主管全国统计和国民经济核算工作的职能部门。1993年，国务院机构改革，国家统计局继续保留，作为主管全国统计和国民经济核算工作的国务院直属机构。1998年, 国务院机构改革，国家统计局继续保留，作为主管全国统计和国民经济核算工作的国务院直属机构。2001年1月4日，国务院办公厅发出《关于国家统计局使用中华人民共和国国家统计局名称和印章的通知》，经国务院批准，国家统计局开始使用“中华人民共和国国家统计局”的名称和印章<ref>{{Cite web|url =http://www.stats.gov.cn/ztjc/zthd/50znjn/tjlsnb/200206/t20020621_35617.html|title =2001年|author =|date =2002-06-21|publisher =中华人民共和国国家统计局|language =|accessdate =|quote =|deadurl =no|archiveurl =https://web.archive.org/web/20170216054953/http://www.stats.gov.cn/ztjc/zthd/50znjn/tjlsnb/200206/t20020621_35617.html|archivedate =2017-02-16|df =}}</ref><ref>{{Cite web|url =http://www.gov.cn/gongbao/content/2001/content_61264.htm|title =国务院办公厅关于国家统计局使用中华人民共和国国家统计局名称和印章的通知（国办函〔2001〕4号）|author =国务院办公厅|date =2001-01-04|publisher =|language =|accessdate =|quote =|deadurl =no|archiveurl =https://web.archive.org/web/20170216050024/http://www.gov.cn/gongbao/content/2001/content_61264.htm|archivedate =2017-02-16|df =}}</ref>。2003年至2013年，国务院机构改革，国家统计局继续保留，作为主管全国统计和国民经济核算工作的国务院直属机构。

== 职责 ==
根据《{{s|国家统计局职能配置、内设机构和人员编制规定}}》，国家统计局承担下列职能<ref name="三定方案">{{cite web |author1=国务院办公厅 |title=国家统计局职能配置、内设机构和人员编制规定 |url=http://www.gov.cn/zhengce/content/2010-11/17/content_7820.htm |website=中华人民共和国中央人民政府门户网站 |accessdate=2018-11-25 |location=北京 |language=zh-cn}}</ref>：
{{columns-list|2|
# 拟定统计工作法规、统计改革和统计现代化建设规划以及国家统计调查计划，组织领导和监督检查各地区、各部门的统计和国民经济核算工作，监督检查统计法律、法规的实施。
# 建立健全国民经济核算体系和统计指标体系，制定全国统一的基本统计制度；拟定国家统计标准；审定部门统计标准;组织管理全国统计调查项目,审批各地区、各部门的统计调查计划、调查方案。
# 研究提出重大的国情国力普查计划，经批准后组织实施；统一组织各地区、各部门的社会经济调查，汇总、整理全国的基本统计资料；对国民经济、科技进步和社会发展等情况进行统计分析、统计预测和统计监督，向党中央、国务院及有关部门提供统计信息和咨询建议。
# 统一核定、管理、公布全国性的基本统计资料，定期向社会公众发布全国国民经济和社会发展情况的统计信息。
# 建立健全和管理国家统计信息自动化系统和国家统计数据库体系；制定各地区、各部门统计数据库网络的基本标准和运行规则。
# 统一管理县及县以上地方人民政府统计部门的统计事业费；协助地方管理省、自治区、直辖市统计局局长和副局长；组织管理全国统计专业资格考试和职务评聘工作。
# 领导设在各地的直属调查机构，指导其他直属事业单位的工作。
# 承办国务院交办的其他事项。
}}
