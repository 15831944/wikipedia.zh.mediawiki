'''国际网联男子巡回赛'''（{{Lang-en|'''ITF Men's Circuit'''}}）是由[[国际网球联合会|国际网球联合会]]在世界各地所举办的一系列职业男子[[网球|网球]]赛事的总称。国际网联男子巡回赛是男子职业网球赛事中的最低级别，但是本项赛事的积分被纳入[[ATP排名|ATP排名]]，为年轻选手提供了迈入更高级别职业赛事——[[ATP挑战赛|ATP挑战赛]]的机会。<ref>{{cite web |title=2017国际网联男子网球巡回赛福州站明日开拍 |url=http://www.hxnews.com/news/fj/fz/201705/19/1220673.shtml |publisher=海峡网 |accessdate=2018-06-11}}</ref><ref>{{cite web |author1=蔡毅 |title=ITF巡回赛武汉站开拍 20多个国家和地区选手参赛 |url=http://www.hb.xinhuanet.com/2018-04/29/c_1122762411.htm |website=[[新华网|新华网]] |publisher=[[湖北日报|湖北日报]] |accessdate=2018-06-11}}</ref>

== 赛制 ==
最初国际网联男子巡回赛由{{Link-en|卫星巡回赛|Satellite tournament|卫星赛}}所组成，每场卫星赛都持续四周以上。但是到了1990年代后期，国际网联为男子巡回赛引入了“未来赛（{{lang|en|Futures Tournaments}}）”。新引入的未来赛为组织赛事的国家或地区网球协会以及参与赛事的选手提供了更大的便利。随着时间的推移，国际网联男子巡回赛中卫星赛的比重在逐步下降；直至2007年，卫星赛被完全取消。<ref>{{cite web |title=Men's History |url=https://www.itftennis.com/procircuit/about-pro-circuit/men's-history.aspx |publisher=国际网球联合会 |accessdate=2018-06-11}}</ref>

未来赛允许参赛选手获得他们职业生涯的赛事头衔和对应的ATP积分。而赛事分为单打和双打两个项目，每场赛事持续时间约为一周的时间。截止2017年，男子巡回赛的赛事奖金分为1.5万[[美元|美元]]和2.5万美元两个级别；同时部分赛事还提供免费住宿。<ref>{{cite web |title=Player Pathway Review |url=https://www.itftennis.com/procircuit/about-pro-circuit/player-pathway.aspx |publisher=[[国际网球联合会|国际网球联合会]] |accessdate=2018-06-11}}</ref>未来赛通常有相当数量的资格抽签，这使得很多没有排名的准职业选手获得进入巡回赛的资格并有机会拿到属于他们的ATP积分。<ref>{{cite web |title=Overview |url=https://www.itftennis.com/procircuit/about-pro-circuit/overview.aspx |publisher=国际网球联合会 |accessdate=2018-06-11}}</ref>
