'''国际通货符号'''（¤）是用于表示未指定的[[通货|通货]]的符号。也称作''scarab''。<ref name="Bemer_1980"/>也用于字符集没有指定通货符号时的代用品。 [[Unicode|Unicode]]编码{{unichar|00A4|CURRENCY SIGN|html=|note=Windows <kbd>Alt+0164</kbd>}}。

==歷史==
1972年[[ISO_646|ISO 646]]引入这个字符。<ref>{{Cite web|url=http://czyborra.com/charsets/iso646.html|title=ISO 646 (Good old ASCII)|website=czyborra.com|access-date=2016-04-13}}</ref> 意大利建议用作美元符的可选替代。<ref>{{cite web|url=http://www.cs.tut.fi/~jkorpela/latin1/ascii-hist.html#24|title=Character histories – notes on some Ascii code positions<!-- Bot generated title -->}}</ref> 

法语键盘支持此符号。<ref>{{Cite web|url=http://www-01.ibm.com/software/globalization/topics/keyboards/registry_index.html|title=IBM Globalization – Keyboard layouts|date=2013-11-11|website=ibm.com|language=en-US|access-date=2016-04-13}}</ref>

==指示意义==
适宜应用，例如¤12.50解释为某种通货的12.5个单位，但此种通货未指明，依赖于上下文可能知道是哪种通货。 

国际贸易中，常用[[ISO_4217|ISO 4217]]的通货三字符代码，而不是通货符号，以免不同字符集对同一字符解释不同。
