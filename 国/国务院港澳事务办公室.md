'''国务院港澳事务办公室'''，简称'''国务院港澳办'''<ref>{{cite web |url = http://www.hmo.gov.cn/gabjj/Channel_326.htm |title = 国务院港澳事务办公室 |accessdate = 2017-10-23 |website = www.hmo.gov.cn }}{{dead link|date=2018年6月 |bot=A Chinese Wikipedian |fix-attempted=yes }}</ref>{{NoteTag|1=由於[[臺灣|台灣]]受到[[国务院台湾事务办公室|國台辦]]名稱影響，因此台灣媒體有時會错误使用“國港辦”或“國澳辦”來稱呼港澳辦<ref>{{cite news|url=http://www.chinatimes.com/realtimenews/20170326002490-260409|title=林鄭月娥當選港特首 陸國港辦：符合中央標準|newspaper=中時電子報|accessdate=2017-10-23|language=zh-TW|deadurl=no|archiveurl=https://web.archive.org/web/20170803170812/http://www.chinatimes.com/realtimenews/20170326002490-260409|archivedate=2017-08-03|df=}}</ref>}}，是[[中华人民共和国国务院|中华人民共和国国务院]]的[[国务院办事机构|办事机构]]，与[[中央港澳工作协调小组|中央港澳工作协调小组]]的办事机构'''中央港澳工作协调小组办公室'''[[一个机构两块牌子|一个机构两块牌子]]，协助[[中华人民共和国国务院总理|国务院总理]]办理[[香港|港]][[澳門|澳]]事務。中央纪委国家监委驻国务院办公厅纪检监察组归口管理国务院港澳办并同時兼管港澳两个[[中央人民政府駐特別行政區聯絡辦公室|中央政府聯絡辦]]的紀檢监察工作<ref>{{cite news|title=女組長李秋芳向中紀委負責 紀檢組派駐港澳辦兼管中聯辦|url=http://www.thepaper.cn/newsDetail_forward_1426226|accessdate=2016-01-28|website=雅虎香港新闻|deadurl=no|archiveurl=https://web.archive.org/web/20160130023854/http://www.thepaper.cn/newsDetail_forward_1426226|archivedate=2016-01-30|df=}}</ref>。

== 职责 ==
根据《{{s|国务院港澳事务办公室职能配置、内设机构和人员编制规定}}》，港澳办承担下列职能<ref name="三定方案A">{{cite web |author1=国务院港澳事务办公室 |title=主要职能 |url=http://www.hmo.gov.cn/gab/zygn/ |website=国务院港澳事务办公室门户网站 |accessdate=2018-10-07 |location=北京 |language=zh-cn}}</ref>：
{{columns-list|2|
# 贯彻执行“一国两制”方针和中央对香港、澳门的政策规定，执行香港特别行政区基本法、澳门特别行政区基本法。
# 了解香港、澳门的有关情况，提出政策建议。
# 负责与香港、澳门特别行政区政府的有关工作联系。
# 承办国务院交办的与香港、澳门有关的法律事宜，就基本法实施涉及的相关法律问题研究提出意见。
# 负责指导和管理内地与香港、澳门因公往来的有关事务，协同有关部门和地方推动与香港、澳门在经济、科技、文化等领域的交流与合作。
# 参与拟订对驻香港、澳门中资机构有关管理的政策，参与内地企业和中资机构在香港、澳门的有关协调工作。
# 对中央驻香港、澳门机构提出的有关事宜提供意见、建议和工作协助。
# 承办国务院交办的其他事项。
}}
