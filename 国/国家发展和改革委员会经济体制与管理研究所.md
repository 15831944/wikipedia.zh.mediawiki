'''国家发展和改革委员会经济体制与管理研究所'''（简称体管所、体改所），位于[[北京市|北京市]][[西城区|西城区]][[南草厂街|南草厂街]]1号，是[[中华人民共和国国家发展和改革委员会|中华人民共和国国家发展和改革委员会]]下属科研[[事业单位|事业单位]]。

==简介==
该所是以研究经济体制改革为主的科研事业单位，为[[中华人民共和国国家发展和改革委员会|中华人民共和国国家发展和改革委员会]]的中心工作服务。该所于1991年5月在人员重组的基础上，启用经济体制与管理研究所的名称。1991年5月前，该所就人员结构而言有两个前身，即原'''中国经济体制改革研究所'''，原'''国家经济体制改革委员会经济管理研究所'''，均隶属[[中华人民共和国国家经济体制改革委员会|中华人民共和国国家经济体制改革委员会]]<ref name=liren/>。中国经济体制改革研究所成立于1984年12月，首任所长是[[高尚全|高尚全]]<ref>{{Cite web |url = http://renwu.hexun.com/figure_1940.shtml |title =王小鲁  |author =  |date =  |publisher =和讯  |language =  |accessdate =2017-10-29  |quote =  }}</ref><ref name=fuchen>{{Cite web |url = http://news.takungpao.com/mainland/focus/2013-11/2043804_4.html |title =国家体改委“沉浮录”  |author =  |date =2013-11-17  |publisher =大公网  |language =  |accessdate =  |quote =  }}</ref>。在1980年代中国社会科学的恢复重建中，最引人注目的有两个相互交叉的圈子，一是以《走向未来》丛书和《20世纪文库》为代表的学者圈，二是以中国经济体制改革研究所和[[中共中央政策研究室|中共中央书记处农村政策研究室]]为中心的“智囊圈”<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1405189  |title =项飙：中国社会科学“知青时代”的终结 |author =  |date = 2015-12-09 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。1990年，中国经济体制改革研究所被合并到国家经济体制改革委员会经济管理研究所，随后重组为国家经济体制改革委员会经济体制改革与管理研究所。原中国经济体制改革研究所的大部分成员后来各奔东西<ref name=fuchen/>。1991年8月21日，国家经济体制改革委员会经济体制与管理研究所在北京成立<ref>国家体改委经济体制与管理研究所成立，中国经济体制改革1991年第10期</ref>。

1991年5月后，该所法定全称的沿革是：1991年至1996年为“'''国家经济体制改革委员会经济体制与管理研究所'''”；1996年至1998年为“'''国家体改委经济体制改革研究院'''”；1998年至2003年为“'''国务院经济体制改革办公室经济体制与管理研究所'''”；2003年后为“'''国家发展和改革委员会经济体制与管理研究所'''”。该所是中央二级预算拨款科研事业单位<ref name=liren>{{Cite web |url =http://www.china-reform.org/?content_34.html  |title = 国家发展改革委经济体制与管理研究所简介 |author =  |date =2011-04-18  |publisher = 国家发展和改革委员会经济体制与管理研究所 |language =  |accessdate =  |quote =  }}</ref>。
