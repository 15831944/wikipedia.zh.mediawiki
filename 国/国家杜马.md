'''国家杜马'''（{{lang-rus|Госуда́рственная ду́ма|Gosudarstvennaya duma}}）是[[俄羅斯|俄羅斯]]的[[下議院|下議院]]，為[[俄羅斯聯邦會議|俄羅斯聯邦會議]]（國會）兩個構成機構之一，有450個議席。國家杜馬是俄羅斯的常設[[立法機構|立法機構]]，主要負責起草和制定國家[[法律|法律]]。參加[[議員|議員]]的資格為最少21歲以上的[[公民|公民]]，任期五年。

== 名称由来 ==
“[[杜马|杜马]]”源于[[俄文|俄文]]的{{lang|ru|Дума}}；而{{lang|ru|Дума}}又来源于俄文[[动词|动词]]{{lang|ru|думать}}，意为“思考”。

== 城市杜馬 ==
從1870年開始[[歐俄|歐俄]]地區開始建立地方自治的城市區劃，所有屋主、納稅的手工業技工和工人依其財富狀況被政府依序登記在[[戶口|戶口]]，再經過估算來劃分三個等級，各自代表三大群體向城市杜馬派出代表，每個群體的代表人數相近，城市的行政權由民選的首席市長和一位經理（uprava）來領導，而這些人是由民選的杜馬議員組成。然而[[沙皇|沙皇]][[亚历山大三世_(俄国)|亞歷山大三世]]在1892年到1894年頒布法令，每個自治地區的城市杜馬必須聽命於政府派任的總督；在1894年城市被限制越來越多權力，但在[[西伯利亞|西伯利亞]]以及[[高加索地區|高加索地區]]設立一些市鎮。

== 沙俄時代國家杜馬 ==
受到[[1905年俄國革命|1905年俄國革命]]的影響，於同年8月6日[[謝爾蓋·維特|謝爾蓋·維特]]發表宣言召開國家杜馬，在最初杜馬不過被認為是中国清末[[立憲運動|立憲運動]]产物[[資政院|資政院]]的「类似」產品，在《[[十月宣言|十月宣言]]》發布之後，[[沙皇|沙皇]][[尼古拉二世|尼古拉二世]]承諾這個議會將用以維護人民基本自由，國家杜馬會在各界廣泛的參予下組成，國家杜馬將被賦予監督以及立法大權。
 
然而尼古拉二世其實還是想保留[[君主專制|君主專制]]的權力，杜馬為[[立法機構|立法機構]]，卻在杜馬成立以前，沙皇就先頒佈了憲法，其中明定[[君主|君主]]未喪失[[大臣|大臣]]（部長）的任命權，也不用向杜馬負起行政上的政治責任，甚至可以隨心所欲的[[解散議會|解散議會]]。

== 現代國家杜馬 ==
國家杜馬是[[俄罗斯聯邦会议|俄罗斯聯邦会议]]（國會）的[[下議院|下議院]]，而上議院[[俄羅斯聯邦委員會|俄羅斯聯邦委員會]]是非民選。在1993年制定新憲法後，其中第95條規定杜馬有450個席次，依據第96條每四年改選一次，在08年年底改為五年一屆，在1993年、1995年、1999年以及2003年一半的席次採[[比例代表制|比例代表制]]，另一半採單一選區選舉，然而在2007年杜馬選舉採用新制度：全部450個席次全部改為政黨比例代表制，依據憲法第97條俄國公民需年滿21歲才能角逐杜馬議員的選舉，政黨須得票7%才可進入杜馬，但2016年再度改回之前的選舉制度，政黨得票門檻由7%降至5%。

==選舉方式==
其選舉方式從1993年[[混合制|混合制]]實施了十多年後，2005年5月，在當時[[普京|普京]]總統的主導下，俄羅斯國會修改法律，將選舉制度由混合制改為政黨[[名單比例代表制|名單比例代表制]]。在新的選舉制度下，全部的450名國家杜馬議員皆由以全國為範圍的政黨名單比例代表制選舉產生，且將政黨比例分配席次門檻由原本的5％提高為7％。2016年修改選舉法，把政黨比例分配席次門檻降為5%，並重新實行混合制。

== 职能 ==


根據[[俄羅斯憲法|俄羅斯憲法]]，下議院的职能包括：
*通過法律，得[[俄羅斯聯邦委員會|聯邦委員會]]（上議院）唱名，由[[俄羅斯總統|總統]]簽署生效；
**以三分之二多數方式通過修改憲法；
*以三分之二多數推翻聯邦委員會對法案的否決；
*會同聯邦委員會四分之三，以三分之二多數推翻總統對法案的否決；
*通過或否決總統對[[俄羅斯總理|總理]]的提名；
*委任副審計長及一半審計官；
*以三分之二多數通過[[彈劾|彈劾]]總統議案。

== 禁止官員擁有國外帳戶 ==
