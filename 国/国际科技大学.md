'''国际科技大学'''（International Technological University），又称加州科技大学，简称ITU，位于[[美国|美国]][[加利福尼亚州|加州]][[旧金山湾区|旧金山湾区]]南部[[聖荷西_(加利福尼亞州)|圣何塞市]]（San Jose）境内 [1]  ，是一所享有世界声望的非盈利私立研究型大学。1994年由美国[[圣克拉拉大学|圣塔克拉拉大学（Santa Clara University）]]陈树柏教授（Shu-Park Chan）创建 [2]  ，是华人在海外创办的第一所新型大学，其前身是上世纪80年代由[[邓小平|邓小平]]于中国改革开放期间在深圳经济特区批准计划筹建的中国实验大学 [3]  。

国际科技大学地处世界著名高科技产业区硅谷的核心位置，与谷歌、Facebook、惠普、英特尔、苹果、思科、英伟达、甲骨文、特斯拉、雅虎等多家知名科技公司、斯坦福大学、加州伯克利大学等顶尖高校为邻，将全球顶尖教育、最新产业技术、企业需求相互融合，为学生提供优质的高科技产业所需的环境和资源（包括人工智能、嵌入式系统、芯片设计、物联网、大数据、生物电子、能源管理、数字艺术等领域）。

国际科技大学拥有人工智能、物联网、生物电子、数字艺术等领域实验室和研究中心，与[[史丹佛大學|斯坦福大学]]、[[北京大学|北京大学]]等院校设有联合实验室，与中国多所院校（北京大学 [4]  、武汉大学、同济大学 [5]  、西安交通大学 [6]  、电子科技大学 [7]  、四川大学、西安邮电大学 [8]  、深圳大学 [9]  等）达成合作。是美国西部学校与学院教育联盟（WASC认证）成员、新丝绸之路大学联盟成员、“一带一路”国际人才培养产学联盟成员、“SQA HND”项目成员院校。

==学术==
国际科技大学在以下六个方向提供硕士、博士方向的教育：
*[[工商管理|工商管理]]
*[[计算机科学|计算机科学]]
*[[数字艺术|数字艺术]]
*[[电子工程|电子工程]]
*[[工程管理|工程管理]]

ITU设有Kryterion测试中心，是集大量各个专业领域认证考试及练习测试为一体的辅助中心。<ref>http://www.kryteriononline.com/test_centers/become_a_test_center/Host_Location_Requirements.pdf{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

ITU现在以下方向设有实验室：人工智能/机器人<ref>{{cite web |url=http://ee.itu.edu/index.php/research/ |title=存档副本 |accessdate=2012-08-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20121015213924/http://ee.itu.edu/index.php/research/ |archivedate=2012-10-15 }}</ref>、绿色能源<ref>{{cite web |url=http://ee.itu.edu/index.php/research/ |title=存档副本 |accessdate=2012-08-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20121015213924/http://ee.itu.edu/index.php/research/ |archivedate=2012-10-15 }}</ref>、生物电子<ref>{{cite web |url=http://ee.itu.edu/index.php/research/ |title=存档副本 |accessdate=2012-08-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20121015213924/http://ee.itu.edu/index.php/research/ |archivedate=2012-10-15 }}</ref>及企业管理解决方案（SAP）。其中人工智能/机器人实验室是与[[北京大学软件与微电子学院|北京大学软件与微电子学院]]联合开设。SAP实验室由工商管理系开设，提供SAP相关培训，ITU也是SAP大学联盟项目成员之一。

2011年6月，ITU获得[[西部学校和学院协会|西部学校和学院协会]]（WASC）授予的候选资格<ref>http://www.wascsenior.org/files/Commission%20actions%20June%202011_0.pdf</ref>。2011年6月4日，美国教育数据库显示ITU获得了西部学校和学院协会的候选资格<ref>http://ope.ed.gov/accreditation/InstAccrDetails.aspx?756e697469643d3137313231362663616d70757349643d30267264743d322f32372f3230313220373a30373a313220504d</ref>。2012年2月，美国高等教育认证委员会（CHEA）将ITU的WASC候选资格加入其数据库<ref>{{cite web |url=http://www.chea.org/search/actionInst.asp?CheaID=189831 |title=存档副本 |accessdate=2012-08-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20140128120859/http://www.chea.org/search/actionInst.asp?CheaID=189831 |archivedate=2014-01-28 }}</ref>。2013年2月22日， ITU获得西部学校和学院协会（WASC）授予的正式认证<ref>http://directory.wascsenior.org/international-technological-university{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==校园生活==
ITU鼓励学生参加学校举办的各项活动，如：舞蹈、篮球、保龄球、乒乓球、桌球、板球等，亦支持在校学生和教职员工利用业余时间组织和参与节日及文化方面庆祝活动，从而增进师生间交流及了解不同地域间文化。

ITU校刊是由学生及教职员工共同编辑的季刊。校刊所刊登的文章，社论及研讨论文均由該校师生撰写。

ITU救助是由学校组织的面向当地乃至全球的慈善救助机构。现有救助服务包括义务[[献血|献血]]、食物和衣服的捐赠及分发。

ITU设有专门为学生服务，帮助学生了解和制定实习计划的学生就业指导中心。学生可通过课程实用培训（CPT）获得与其学科域相关的职业经验及了解职业要求和环境。

学校每年设有[[奖学金|奖学金]]、研究和教学助理等项目。学生可在学生注册办公室递交申请。
