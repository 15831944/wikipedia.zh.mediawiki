'''国际女子网球协会'''{{NoteTag|中文译名见其[http://www.wta.cn/ 官方中文网站]。}}（{{Lang-en|Women's Tennis Association}}），简称“{{lang|en|WTA}}”，是世界女子职业[[网球|网球]]的管理组织，由[[比利·简·金|比利·简·金]]等人于1973年9月创立<ref name="about" /><ref name="wta" />。国际女子网球协会拥有并管理着{{Link-en|WTA巡回赛|WTA Tour}}，这是全球女子职业网球巡回赛事，旨在为女子网球创造更美好的未来。国际女子网球协会总部位于[[美国|美国]][[佛罗里达州|佛罗里达州]]的[[聖彼德斯堡_(佛羅里達州)|圣彼得斯堡]]；欧洲总部位于[[英国|英国]][[伦敦|伦敦]]；亚太区总部位于[[中国|中国]][[北京|北京]]；并在[[新加坡|新加坡]]还拥有一个总部。<ref>{{cite web |title=WTA Contact Information |url = https://www.wtatennis.com/content/wta-contact-information |publisher=WTA |accessdate=2019-01-16 }}</ref>

国际女子网球协会成立于1973年，但是其历史最早可以追溯到“{{Link-en|维珍妮香烟|Virginia Slims}}巡回赛”。该赛事由{{Link-en|格拉迪斯·海德曼|Gladys Heldman}}主办，而由[[菲利普莫里斯国际|菲利普·莫里斯公司]][[首席执行官|首席执行官]]乔·卡尔曼（{{lang|en|Joe Cullman}}）赞助。“维珍妮香烟巡回赛”首次赛事于1970年9月23日在美国[[德克萨斯州|德克萨斯州]]休士顿市的休士顿球拍俱乐部（{{lang|en|Houston Racquet Club}}）内举行<ref name="about" />。{{Link-en|罗斯玛丽·卡塞尔斯|Rosemary Casals}}取得首届赛事的金牌。

当国际女子网球协会成立的时候，协会成员是包括比利·简·金在内的九位女子选手，她们也被称为“创始九人（{{lang|en|Original 9}}）”。创始九人包括：{{Link-en|朱莉·海德曼|Julie Heldman}}、{{Link-en|瓦莱莉·泽根芙丝|Valerie Ziegenfuss}}、{{Link-en|朱迪·特格特-道尔顿|Judy Tegart-Dalton|朱迪·道尔顿}}、{{Link-en|克里斯蒂·佩珍|Kristy Pigeon}}、{{Link-en|简·巴克维茨|Jane Bartkowicz|桃美人··巴克维茨}}、{{Link-en|凯莉·麦尔维尔|Kerry Melville|凯莉·麦尔维尔·里德}}、{{Link-en|南希·里切|Nancy Richey}}和{{Link-en|罗斯玛丽·卡萨斯|Rosemary Casals|罗西·卡萨斯}}<ref name="history">{{cite news |url=http://www.sonyericssonwtatour.com/3/global/includes/TrackIt.asp?file=http%3A%2F%2Fwww.sonyericssonwtatour.com%2F3%2Fglobal%2Fpdfs%2Fevents%2F2008%2Ftournamentfinals.pdf |format=PDF |title=Tournament Finals - 2007 |work=索尼爱立信WTA巡回赛 |publisher=国际女子网球协会 |date=2008-01-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20080529060754/http://www.sonyericssonwtatour.com/3/global/includes/TrackIt.asp?file=http%3A%2F%2Fwww.sonyericssonwtatour.com%2F3%2Fglobal%2Fpdfs%2Fevents%2F2008%2Ftournamentfinals.pdf |archivedate=2008-05-29 |accessdate=2019-01-16 }}</ref><ref>{{cite web |author = WTA国际女子网球协会 |title=我有一个梦想：女子网球的抗争与独立 |url = https://new.qq.com/omn/20171224/20171224B0FBI0.html |website=[[腾讯网|腾讯网]] |publisher=腾讯新闻 |accessdate=2019-01-16 }}</ref>。现在WTA巡回赛有超过2500名选手代表近100个国家或地区参赛，共同竞争高达1.46亿[[美元|美元]]的总奖金<ref name="wta" />。

国际女子网球协会2018年赛季共计有54场比赛，这其中包括：{{Link-en|WTA顶级巡回赛|WTA Premier tournaments}}（含皇冠赛、超五赛和常规定级赛）、{{Link-en|WTA国际巡回赛|WTA International tournaments}}、[[联合会杯网球赛|联合会杯]]（由[[国际网球协会|国际网球协会]]主办）、年终锦标赛（含[[WTA年终总决赛|WTA年终总决赛]]和[[珠海WTA超级精英赛|WTA超级精英赛]]）和“[[大滿貫_(網球)|四大满贯]]”（由国际网球协会监督）。这些赛事将在全球30个国家或地区举行。而2018赛季将在10月21日到28日于新加坡举行的[[WTA年终总决赛|WTA年终总决赛]]和在10月30日至11月4日于中国[[珠海|珠海]]举行的[[珠海WTA超级精英赛|WTA超级精英赛]]两大赛事中结束<ref>{{cite web |title=WTA Calendar |url=https://www.wtatennis.com/calendar |publisher=WTA |accessdate=2019-01-16}}</ref><ref>{{cite web |title=2018年赛程 |url=http://www.wta.cn/schedule/scintro_year.aspx |publisher=WTA中文官网 |accessdate=2019-01-16}}</ref>。2018年赛季中还包括[[霍普曼杯|霍普曼杯]]，这也是由国际网球协会所举办的赛事，但是该赛事不发放任何WTA积分。
