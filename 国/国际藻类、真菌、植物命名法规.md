《'''国际藻类、真菌、植物命名法规'''》（{{lang-en|International Code of Nomenclature for algae, fungi, and plants}}；{{lang|en|ICN}}）<ref name="決議2011">[http://www.ibc2011.com/downloads/Resolutions%20final%2030%20Jul.pdf 《第十八屆國際植物學會議決議文》]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}，2011年7月於墨爾本的會議決議更改原名'''''International Code of Botanical Nomenclature'''''；'''''ICBN'''''為 '''''International Code of Nomenclature for algae, fungi, and plants'''''；'''''ICN'''''。</ref>是一部關於[[植物|植物]]命名的規則與建議，其中確立每一個[[分類單元|分類單元]]（或分類群, taxon）只有一個正確的[[學名_(植物)|學名]]。此規則通行於全世界，更改必須經由[[國際植物學會議|國際植物學會議]]（{{lang-en|International Botanical Congress}}, {{lang|en|IBC}}）的討論。全部的事項皆由[[國際植物分類學學會|國際植物分類學學會]]組織完成。

== 歷史 ==
首次的法規出版是1905年在[[維也納|維也納]]舉行的第二屆[[國際植物學會議|國際植物學會議]]（IBC），並於隔年出版現稱為Vienna Rule的第一版'''命名法規'''<ref name="中法"/>。

== 法規內容 ==
命名法規主要的條文(以維也納法規為準)：
* '''原則''' (principles)：共有六條最基本的原則，「係構成植物命名法之基礎」<ref name="中法">[植物命名指南, 第三版]，黃增泉，2007。植物命名指南, 第三版。臺北市:行政院農業委員會林務局。(ISBN 978-986-01-1771-4)</ref>。
* 詳細的法條區分為許多規則，而以'''條款''' (Article) 及'''擬則''' (Recommand)陳述。'''例証''' (Ex.) 則附加於規則及擬則內供為說明<ref name="中法"/>。條款中的規則乃整理歷來的命名法，作為將來之準繩；凡植物名稱抵觸規則者不能再使用。擬則供補充條款之用，「其目的在使將來之命名方法可逐步統一、清晰。凡植物命名無論在名稱上或形式上與擬則抵觸者，雖不予取消，但此後不得援以為例」。<ref name="中法"/>
* 規則與擬則適用於所有傳統認知之現生或化石植物。例如：[[藍綠菌|藍綠菌]](藍綠藻)、[[真菌類|真菌類]]，包括壺菌類、卵菌類、黏菌類等；及能施行光合作用之原生生物。
* 國際栽培植物命名委員會為使用於農業上、林業上、園藝上若干特定群之名稱，授權制定「國際栽培植物命名法規」。
* 法規一直強調：名稱變更之唯一正確理由，乃分類學者經充份研究後，有確鑿之事實証據，或必須廢棄違反本規則之學名。<ref name="中法"/>

== 最新變革 ==
[第十八屆國際植物學會議] (XVIII IBC) 國際植物命名法規重要變更：<ref name="臺分">[http://www.tsps.org.tw/ 臺灣植物分類學會網頁《相關新聞》]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}，此部文字經維基化與對照後附資料改寫，資料來源參見[http://www.tsps.org.tw/images/download/Key%20decisions%20of%20Nomenclature%20Section.pdf 圖檔英文：Key decisions of the nomenclature section of the XVIII IBC] {{webarchive|url=https://web.archive.org/web/20120412175018/http://www.tsps.org.tw/images/download/Key%20decisions%20of%20Nomenclature%20Section.pdf |date=2012-04-12 }}。</ref>

'''一、自2012年1月1日起接受在電子出版品上發表新名'''

自2012年1月1日起，在印刷出版品上發表新名稱已非必要；於[[網路|網路]]上以[[PDF|PDF]]檔（便攜式文件格式）發表且具有[[國際標準書號|國際標準書號]]（ISBN） 或[[國際標準連續出版物號|國際標準連續出版物號]]（ISSN）之[[出版品|出版品]]亦可視為有效發表。

'''二、自2012年1月1日起命名時可用英文描述'''

自2012年1月1日起，發表新種時可用[[拉丁文|拉丁文]]或[[英文|英文]]描述。

'''三、一種[[化石|化石]][[分類群|分類群]]一個[[學名_(植物)|正確名稱]]；一種[[真菌|真菌]]，一個[[學名_(植物)|正確名稱]]'''

放棄[[形態分類群|形態分類群]] (morphotaxa) 之概念，若具不同名稱之斷片、[[生活史|生活史]]階段或儲藏階段之[[化石|化石]]可被証實屬同一種時，僅可具有一正當名；該名稱之決定應符合優先權之相關規定。真菌的部份則是捨棄有性世代、無性世代具不同學名的概念。

'''四、於MycoBank註冊新名稱是發表真菌新名的必要條件'''

近年來，真菌學家於''MycoBank資料庫''、''Index Fungorum資料庫''發表新種新名，為法規中規定的必要條件。

'''五、國際植物命名法規更名為國際藻類、真菌及植物命名法規'''

《國際植物命名法規》（International Code of Botanical Nomenclature）為有效反映該法規所涵蓋之分類群，更名為《國際藻類、真菌及植物命名法規》（International Code of Nomenclature for algae, fungi, and plants）。
