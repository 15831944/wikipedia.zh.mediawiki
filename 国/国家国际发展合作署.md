'''国家国际发展合作署'''，简称'''国际发展合作署'''，是[[中华人民共和国国务院|中华人民共和国国务院]]主管对外援助事务的副部级[[国务院直属机构|直属机构]]<ref name="官方简称">{{cite web |author1=国务院办公厅秘书局 |title=关于转发国务院机构简称的通知 |url=http://www.gdllw.org/tzgg/201806/W020180622515099385181.pdf |website=广东老龄网 |accessdate=2018-11-25 |location=北京 |language=zh-cn}}</ref>。

国家国际发展合作署主要负责拟订对外援助战略方针、规划、政策，统筹协调援外重大问题并提出建议，推进援外方式改革，编制对外援助方案和计划，确定对外援助项目并监督评估实施情况等。援外的具体执行工作仍由[[相关部门|相关部门]]按分工承担<ref>{{cite web|title =王勇：组建国家国际发展合作署|website =新华网|url =http://www.xinhuanet.com/politics/2018lh/2018-03/13/c_137035595.htm|deadurl =no|archiveurl =https://web.archive.org/web/20180313154406/http://www.xinhuanet.com/politics/2018lh/2018-03/13/c_137035595.htm|archivedate =2018-03-13|df =}}</ref>。

==沿革==
2018年3月17日[[第十三届全国人民代表大会第一次会议|第十三届全国人民代表大会第一次会议]]通过《第十三届全国人民代表大会第一次会议关于国务院机构改革方案的决定》，批准《国务院机构改革方案》。方案规定：“组建国家国际发展合作署。将[[中华人民共和国商务部|商务部]]对外援助工作有关职责、[[中华人民共和国外交部|外交部]]对外援助协调等职责整合，组建国家国际发展合作署，作为国务院直属机构。对外援助的具体执行工作仍由有关部门按分工承担。”<ref>{{cite web |url = http://www.npc.gov.cn/npc/dbdhhy/13_1/2018-03/14/content_2048552.htm |title = 关于国务院机构改革方案的说明——2018年3月13日在第十三届全国人民代表大会第一次会议上 |author = 王勇 |date = 2018-03-13 |website = 中国人大网 |deadurl = no |archiveurl = https://web.archive.org/web/20180609103422/http://www.npc.gov.cn/npc/dbdhhy/13_1/2018-03/14/content_2048552.htm |archivedate = 2018年6月9日 |df =  }}</ref><ref>{{cite web|url =http://www.npc.gov.cn/npc/xinwen/2018-03/17/content_2049996.htm|title =第十三届全国人民代表大会第一次会议关于国务院机构改革方案的决定|author =|date =2018-03-17|website =中国人大网|deadurl =no|archiveurl =https://web.archive.org/web/20180318120500/http://www.npc.gov.cn/npc/xinwen/2018-03/17/content_2049996.htm|archivedate =2018-03-18|df =}}</ref><ref>{{cite web|url =http://www.npc.gov.cn/npc/xinwen/2018-03/18/content_2050371.htm|title =国务院机构改革方案|author =|date =2018-03-17|website =中国人大网|deadurl =no|archiveurl =https://web.archive.org/web/20180318120458/http://www.npc.gov.cn/npc/xinwen/2018-03/18/content_2050371.htm|archivedate =2018-03-18|df =}}</ref>

2018年4月18日，国家国际发展合作署举行揭牌仪式，白底黑字的“国家国际发展合作署”揭牌，[[中共中央政治局|中共中央政治局]]委员、[[中央外事工作委员会办公室|中央外事工作委员会办公室]]主任[[杨洁篪|杨洁篪]]、国务委员兼外交部部长[[王毅|王毅]]出席揭牌仪式并讲话。揭牌仪式后，国家国际发展合作署召开了成立大会<ref>{{cite web |url = https://www.thepaper.cn/newsDetail_forward_2083927 |title = 杨洁篪出席国家国际发展合作署揭牌仪式 |date = 2018-04-18 |website = 澎湃新闻 |deadurl = no |archiveurl = https://web.archive.org/web/20180419053341/https://www.thepaper.cn/newsDetail_forward_2083927 |archivedate = 2018-04-19 |df =  }}</ref>。
