'''中国福利会国际和平妇幼保健院'''与'''上海交通大学医学院附属国际和平妇幼保健院'''[[一个机构两块牌子|一个机构两块牌子]]，简称'''国际和平妇幼保健院'''（{{lang-en|International Peace Maternity And Child Health Hospital, IPMCH}}<ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node69322/node69344/node69542/userobject1ai67722.html 专业志 >> 中国福利会志 >> 附录 >> 专名].</ref>）是一所[[三级甲等医院|三级甲等医院]]，位于[[中国|中国]][[上海市|上海市]][[徐汇区|徐汇区]][[衡山路_(上海)|衡山路]]910号。该院系1952年用[[宋庆龄|宋庆龄]]捐赠的[[国际斯大林和平奖|国际斯大林和平奖]]奖金建成。

==历史==
国际和平妇幼保健院的前身是上海胶州路妇幼保健站和沪西女工保健站。1951年，[[宋庆龄|宋庆龄]]获[[国际斯大林和平奖|国际斯大林和平奖]]（后更名为[[列宁和平奖|列宁和平奖]]），9月18日获得奖金10万[[卢布|卢布]]，宋庆龄在支票上写上“此款捐赠[[中国福利会|中国福利会]]作妇儿福利事业之用”，遂决定在原妇幼保健站的基础上，利用此奖金建设一所妇幼保健院，以“国际和平”冠名。

1952年9月18日，国际和平妇幼保健院在上海市[[普陀区|普陀区]][[长寿路_(上海)|长寿路]]170号成立，占地面积3,000多平方米，有单一的产科，拥有房屋3幢，床位50张，医护人员107名。

[[File:1952-08_上海国际和平妇幼保健院.png|thumb]]
1956年10月，该院迁至现址，占地面积达11,700平方米，另设独立的妇科，并设妇科病房、门诊及肿瘤病床，有房屋2幢，病床150张。1962年设[[计划生育|计划生育]]科，设置计划生育门诊及手术室；1963年设保健科和新生儿科。1966年，医院床位达到200张，并以上海市拨款的120多万元建设6层的病房大楼，面积达5,459平方米，并设置放射性治疗设施。1984年，医院又利用上海市卫生局的100万元拨款建设了门诊实验楼，面积3,600平方米，医院床位达到350张。1992年9月18日医院投资2,400万元，建设9层的产儿科大楼，建筑面积达7,242平方米，1994年10月竣工，为感谢[[郭鹤年|郭鹤年]]捐赠的1,000万元善款，该楼以其母郑格如的名字命名为“格如大楼”。到1995年，医院拥有卫生、技术人员496人。2001年12月15日斥资2亿多元，在1966年建设的病房大楼原址上建设18层（地下2层、地上16层）的妇产科综合大楼，占地面积2,352平方米，建筑面积30,966平方米，2004年12月25日投入使用。

1989年该院成为[[上海第二医科大学|上海第二医科大学]]的教学医院，1999年被评为三级甲等专科医院，2006年成为[[上海交通大学医学院|上海交通大学医学院]]的附属医院。<ref name="jieshao">{{Cite web |url=http://www.ipmch.com.cn/website1/detail.aspx?fid=90DF7DD2&menu=z&nav=z11 |title=中国福利会国际和平妇幼保健院（三级甲等专科医院、上海交通大学医学院附属医院） |accessdate=2012-06-30 |work= |publisher=国际和平妇幼保健院 |date=2012-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20120711174718/http://www.ipmch.com.cn/website1/detail.aspx?fid=90DF7DD2&menu=z&nav=z11 |archivedate=2012年7月11日 |df= }}</ref><ref>[http://www.shtong.gov.cn/newsite/node2/node2247/node4593/node79682/node79702/userobject1ai103324.html 上海通志 >> 第三十六卷卫生 >> 第一章机构、网络、人员 >> 节 >> 第一节 机构].</ref><ref>[http://www.shtong.gov.cn/newsite/node2/node2245/node69322/node69332/node69398/userobject1ai67465.html 专业志 >> 中国福利会志 >> 第五编妇幼保健 >> 概述].</ref>

==现状==
[[File:1952-08_上海国际和平妇幼保健院2.png|thumb]]
该院以[[宋庆龄|宋庆龄]]和[[周恩来|周恩来]]对中国福利会“实验性、示范性、加强科学研究”的倡导为方针，以保健为中心，结合临床，已经发展成为一所含医疗、保健、教学、科研等功能的市级妇幼保健专科医院。该院的孕产妇死亡率以及围产儿死亡率等指标已接近或达到国际先进水平。

目前，该院占地面积为1.23万平方米，建筑面积4.7万平方米，共有核定床位370张，近年床位使用率保持在95%以上；年门诊量90万余人次，年住院病人将近2万人次，年孕妇分娩数1万余人次。截至2011年底，该院共有职工844人，其中85%以上为卫生技术人员。<ref name="jieshao" />

== 科室设置 ==
该院现设临床、医技、保健三个部，每个部下设若干科室和中心。<ref>{{Cite web|url=http://www.ipmch.com.cn/website1/DeptInfo.aspx?pageid=site1_deptinfo&menu=z|title=科室介绍|accessdate=2012-06-30|work=|publisher=国际和平妇幼保健院|date=2012-05|deadurl=yes|archiveurl=https://web.archive.org/web/20120622123338/http://www.ipmch.com.cn/website1/DeptInfo.aspx?pageid=site1_deptinfo&menu=z|archivedate=2012年6月22日|df=}}</ref>
