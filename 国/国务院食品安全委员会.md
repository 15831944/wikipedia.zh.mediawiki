'''国务院食品安全委员会'''，是[[中华人民共和国国务院|中华人民共和国国务院]]成立的[[国务院议事协调机构|国务院议事协调机构]]，负责食品安全工作。

==沿革==
2009年2月28日[[第十一届全国人民代表大会常务委员会|第十一届全国人民代表大会常务委员会]]第七次会议通过《[[s:中华人民共和国食品安全法|中华人民共和国食品安全法]]》，自2009年6月1日起施行。该法第四条规定：“国务院设立食品安全委员会，其工作职责由国务院规定。”<ref>{{Cite web |url = http://www.npc.gov.cn/wxzl/gongbao/2009-05/07/content_1517190.htm |title =中华人民共和国食品安全法  |author =  |date = 2009-05-07 |publisher =中国人大网  |language =  |accessdate =  |quote =  }}</ref> 2015年4月24日[[第十二届全国人民代表大会常务委员会|第十二届全国人民代表大会常务委员会]]第十四次会议修订该法，修订后的该法第五条规定：“国务院设立食品安全委员会，其职责由国务院规定。”<ref>{{Cite web |url =http://www.npc.gov.cn/npc/cwhhy/12jcwh/2015-04/25/content_1934591.htm  |title = 中华人民共和国食品安全法 |author =  |date =2015-04-25  |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>

2010年2月6日，《国务院关于设立国务院食品安全委员会的通知》（国发〔2010〕6号）称，“为贯彻落实食品安全法，切实加强对食品安全工作的领导，设立国务院食品安全委员会，作为国务院食品安全工作的高层次议事协调机构。”<ref name=chengli>{{Cite web |url =http://www.gov.cn/zwgk/2010-02/10/content_1532419.htm  |title =国务院关于设立国务院食品安全委员会的通知  |author =  |date = 2010-02-06 |publisher =中国政府网  |language =  |accessdate =  |quote =  }}</ref>2010年2月9日，国务院食品安全委员会召开第一次全体会议，[[中共中央政治局常委|中共中央政治局常委]]、国务院副总理、国务院食品安全委员会主任[[李克强|李克强]]出席会议并讲话<ref>聚焦食品安全委员会，工商行政管理2010年04期</ref>。

根据2013年3月[[第十二届全国人民代表大会|第十二届全国人大一次会议]]通过的《关于国务院机构改革和转变职能方案的决议》，由新组建的[[国家食品药品监督管理总局|国家食品药品监督管理总局]]承担委员会的日常工作<ref>[http://news.ifeng.com/mainland/special/2013lianghui/content-3/detail_2013_03/10/22931487_0.shtml 国务院组建国家食品药品监管总局 撤销药监局、食安办]</ref>。

2018年3月17日[[第十三届全国人民代表大会第一次会议|第十三届全国人民代表大会第一次会议]]通过《第十三届全国人民代表大会第一次会议
关于国务院机构改革方案的决定》，批准《国务院机构改革方案》。方案规定：“组建[[国家市场监督管理总局|国家市场监督管理总局]]。将[[国家工商行政管理总局|国家工商行政管理总局]]的职责，[[国家质量监督检验检疫总局|国家质量监督检验检疫总局]]的职责，[[国家食品药品监督管理总局|国家食品药品监督管理总局]]的职责，[[中华人民共和国国家发展和改革委员会|国家发展和改革委员会]]的价格监督检查与反垄断执法职责，[[中华人民共和国商务部|商务部]]的经营者集中反垄断执法以及[[国务院反垄断委员会|国务院反垄断委员会办公室]]等职责整合，组建国家市场监督管理总局，作为国务院直属机构。同时，组建[[国家药品监督管理局|国家药品监督管理局]]，由国家市场监督管理总局管理。将国家质量监督检验检疫总局的出入境检验检疫管理职责和队伍划入[[中华人民共和国海关总署|海关总署]]。保留国务院食品安全委员会、[[国务院反垄断委员会|国务院反垄断委员会]]，具体工作由国家市场监督管理总局承担。[[国家认证认可监督管理委员会|国家认证认可监督管理委员会]]、[[国家标准化管理委员会|国家标准化管理委员会]]职责划入国家市场监督管理总局，对外保留牌子。不再保留国家工商行政管理总局、国家质量监督检验检疫总局、国家食品药品监督管理总局。”<ref>{{Cite web |url =  http://www.npc.gov.cn/npc/dbdhhy/13_1/2018-03/14/content_2048552.htm|title = 关于国务院机构改革方案的说明——2018年3月13日在第十三届全国人民代表大会第一次会议上 |author =王勇  |date = 2018-03-13 |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.npc.gov.cn/npc/xinwen/2018-03/17/content_2049996.htm|title =  第十三届全国人民代表大会第一次会议关于国务院机构改革方案的决定|author =  |date =2018-03-17  |publisher =  中国人大网|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.npc.gov.cn/npc/xinwen/2018-03/18/content_2050371.htm|title =  国务院机构改革方案|author =  |date = 2018-03-17  |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>

== 职责 ==
2010年《国务院关于设立国务院食品安全委员会的通知》规定，国务院食品安全委员会的主要职责是：
# 分析食品安全形势，研究部署、统筹指导食品安全工作；
# 提出食品安全监管的重大政策措施；
# 督促落实食品安全监管责任<ref name=chengli/>。

==办事机构==
2010年《国务院关于设立国务院食品安全委员会的通知》规定，“国务院食品安全委员会设立'''国务院食品安全委员会办公室'''，具体承担委员会的日常工作。”<ref name=chengli/>经国务院和中央编委领导同意，2010年12月6日[[中央编办|中央编办]]印发《关于国务院食品安全委员会办公室机构设置的通知》（中央编办发〔2010〕202号），具体规定了国务院食品安全委员会办公室的主要职责、内设机构、人员编制及与其他部门的职责分工等事项。国务院食品安全委员会办公室设综合司、协调指导司、监督检查司、应急管理司4个内设机构，机关党委办事机构设在综合司；机关行政编制30名（其中主任1名、副主任1名，司局级领导职数12名）<ref>{{Cite web |url =http://news.cntv.cn/20101207/107806.shtml  |title =  中编办：国务院食品安全委员会办公室设4个司30个编制|author =  |date = 2010-12-07 |publisher = 中国网络电视台 |language =  |accessdate =  |quote =  }}</ref>。

2013年《国务院办公厅关于调整国务院食品安全委员会组成人员的通知》规定，“国务院食品安全委员会办公室设在食品药品监管总局，承担国务院食品安全委员会日常工作。”2013年3月26日《国务院办公厅关于印发国家食品药品监督管理总局主要职责内设机构和人员编制规定的通知》（国办发〔2013〕24号）印发了《国家食品药品监督管理总局主要职责内设机构和人员编制规定》，其中规定国家食品药品监督管理总局“承担国务院食品安全委员会日常工作”，国家食品药品监督管理总局综合司（政策研究室）“承担国务院食品安全委员会办公室日常工作”，“国家食品药品监督管理总局加挂国务院食品安全委员会办公室牌子。”<ref>{{Cite web |url = http://www.gov.cn/gongbao/content/2013/content_2412268.htm |title = 国务院办公厅关于印发国家食品药品监督管理总局主要职责内设机构和人员编制规定的通知 |author =  |date = 2013-03-26 |publisher = 中国政府网 |language =  |accessdate =  |quote =  }}</ref>国家食品药品监督管理总局综合司（政策研究室）兼国务院食品安全委员会办公室秘书处<ref>{{Cite web |url =http://news.163.com/17/0801/17/CQP5SBUT00018AOQ.html  |title = 孙梅君任国家食品药品监管总局副局长 |author =  |date =2017-08-01  |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。
