'''国家海洋局极地考察办公室'''（简称'''极地办'''）是[[中华人民共和国|中华人民共和国]][[国家海洋局|国家海洋局]]直属，[[参照公务员法管理|参照公务员法管理]]的财政补助<ref name="介绍">{{cite web |url = http://www.chinare.gov.cn/caa/gb_article.php?modid=02001 |title = 极地考察办公室主要职责 |publisher=国家海洋局极地考察办公室 |work = 国家海洋局极地考察办公室 |date=2011-07-21 |accessdate=2014-01-04 |quote = 国家海洋局极地考察办公室......的财政补助事业单位}}</ref>公益性[[事业单位|事业单位]]<ref name="公益性">{{cite journal |url = http://www.cnki.com.cn/Article/CJFDTotal-TPYX201108001.htm |title=国家海洋局极地考察办公室 |author=战海霞 |journal=太平洋学报 |issue=8 |quote = 是国家海洋局直属公益性事业单位...... |ISSN = 1004-8049|date=2011年8月}}</ref>，机构[[行政级别|行政级别]]为正司局级<ref name="级别">{{cite web |url = http://www.chinare.gov.cn/caa/gb_article.php?modid=02002 |title = 极地考察办公室内设机构 |publisher=国家海洋局极地考察办公室 | work=国家海洋局极地考察办公室 |date=2013-05-29 |accessdate=2014-01-04 |quote = 极地办为部委正司级单位......}}</ref>。极地办的任务是代表[[国家海洋局|国家海洋局]]进行[[南极|南]][[北极|北极]]科学考察、管理相关极地事务<ref name="职责" /><ref name="guanli" />。办公室与[[國際北極研究中心|國際北極研究中心]]亦有合作<ref name="beijiyanjiuzhongxin">{{cite web |url = http://www.iarc.uaf.edu/gmap/ |title = IARC International Collaborations, Partners, and Related Organizations |date = 2009-02-13 |work = iarc.uaf.edu |publisher = IARC, University of Alaska, Fairbanks |accessdate = 27 July 2010 |location = Fairbanks |deadurl = yes |archiveurl = https://web.archive.org/web/20080515143417/http://www.iarc.uaf.edu/gmap/ |archivedate = 2008年5月15日 }}</ref> 。

== 主要职责==
国家海洋局极地考察办公室的主要职责为<ref name="主要职责">{{cite web | url = http://www.chinare.gov.cn/caa/gb_article.php?modid=02001 | title=极地考察办公室主要职责 | publisher=国家海洋局极地考察办公室 | work=国家海洋局极地考察办公室 | date=2011-07-21 | accessdate=2014-01-04 | quote=（一）组织拟订我国极地工作的发展战略......交办的其他事项；}}</ref><ref name="zhize">{{cite journal | url=http://www.cnki.com.cn/Article/CJFDTotal-TPYX201108001.htm | title=国家海洋局极地考察办公室 | author=战海霞 | journal=太平洋学报 | issue=8 | quote=其主要职责如下:(一)......| ISSN=1004-8049|date=2011年8月}}</ref><ref name="EncyclopediaAntarctic">{{cite book|last=Riffenburgh|first=Beau |title=Encyclopedia of the Antarctic|url=http://books.google.com/books?id=fRJtB2MNdJMC&pg=PA226&dq=Chinese+Arctic+and+Antarctic+Administration&hl=en&ei=ZTtOTMalFI_CsAOb39EL&sa=X&oi=book_result&ct=result&resnum=1&ved=0CCwQ6AEwAA#v=onepage&q=Chinese%20Arctic%20and%20Antarctic%20Administration&f=false|volume=1|year=2007 |publisher=CRC Press|isbn=0-415-97024-5|page=226}}</ref>：
# 负责中国极地工作发展战略、方针、政策与极地考察工作规划、计划的制定，研究有关极地的问题；
# 拟订中国极地科考、相关极地事务的法律、法规、标准与规范，并依法管理相关事务；
# 协调组织及监督极地考察建设项目、考察队工作、训练基地与驻外机构管理，组织开展极地领域的科研工作；
# 协调组织极地领域的国际事务及相关国际组织活动，组织极地领域对外及对港澳台地区交流合作；
# 承担极地[[科学普及|科学普及]]与公众宣传等工作；
# 承办国家海洋局交办的其他事项。

== 机构设置 ==
极地办内设六个处级机构，还拥有一个冬训基地<ref name="内设机构">{{cite web | url=http://www.chinare.gov.cn/caa/gb_article.php?modid=02002 | title=极地考察办公室内设机构 | publisher=国家海洋局极地考察办公室 | work=国家海洋局极地考察办公室 | date=2013-05-29 | accessdate=2014-01-04}}</ref>。

=== 内设机构 ===
;综合处（党委办公室）
该机构承担以下方面工作：[[秘书|文秘]]、[[档案|档案]]、安全、[[保密|保密]]、[[信访|信访]]、会务等；[[人事管理|人事管理]]、党委纪委、[[退休|退休]]人员、[[工会|工会]]、[[共青团|共青团]]等；极地考察科普、宣传和中文网站运行管理；固定资产实物管理；以及极地考察临时党组织管理。<ref name="内设机构" />

;政策与规划处
该机构组织开展极地事业发展战略、方针、政策研究；拟订监督实施极地事业发展规划，和极地事务相关法律、法规及规章；编制国家极地考察年度报告；并承担[[中国极地考察工作咨询委员会|中国极地考察工作咨询委员会]]日常工作。<ref name="内设机构" />

;考察业务处（财务处）
该机构拟订并组织实施极地考察年度计划；承担极地能力建设项目立报申报并监督实施；组建极地考察队；负责国内选拔训练基地和驻外机构的业务管理；指导监督极地考察安全与应急管理；并负责财务管理、政府采购与极地综合统计。<ref name="内设机构" />

;科技发展处
该机构拟订极地考察的相关标准规范；拟订并监督实施极地科技发展规划、极地科技项目年度计划；参与申报并组织实施国家有关极地领域科研项目；指导协调极地科考。<ref name="内设机构" />

;国际处
[[File:Teadlased_jääl.jpg|thumb]]
该机构组织负责研究极地领域国际公约条约政策制度；承担极地领域之国际交流合作；拟订并监督实施极地活动环境管理制度，开展极地环保的教育培训；管理驻外机构、获取国外极地考察相关信息与管理英文网站。<ref name="内设机构" />

极地办在中华人民共和国驻[[澳大利亚|澳大利亚]]与[[智利|智利]]的大使馆中分别设有代表处。代表处为去南极的中国考察队员提供便利，并向[[雪龙号|雪龙号]]提供油、[[水|水]]与[[食物|食物]]等等。此外，代表处还负责与国外的机构合作，从而促进[[科学家|科学家]]之间的国际合作。<ref name="BriefIntroduction">{{cite web |url = http://www.chinare.gov.cn/en/index.html?pid=about |title=A Brief Introduction to CAA|work=chinare.gov.cn|accessdate=27 July 2010}}</ref>

;人事处
该机构拟定人事管理制度和办法；承办人员调配、工资、考核、培训、奖惩和录用等工作；承办出国人员[[审查制度#审查的种类|政审]]与驻外机构人员选派；负责人事信息系统建设和管理，各类人事统计报表及档案管理；办理[[退休|退休]]手续并负责退休人员日常管理；此外还承办极地办交办的其他事项。<ref name="内设机构" />

=== 直属机构 ===
;冬训基地
极地办拥有的冬季训练基地位于[[黑龙江省|黑龙江省]][[尚志市|尚志市]]南的[[亚布力滑雪旅游度假区|亚布力滑雪旅游度假区]]。该基地[[海拔|海拔]]1374米，冬季平均温度约为-25[[℃|℃]]，最低温度可达-33℃；通常每年十月中旬山的顶部下雪，直到第二年五月初；因此，该基地的自然环境适合南极探险队的训练。1986年3月，第三次中国南极考察队在此进行训练，从此它成为中国南极考察队员的定期培训基地。<ref name="BriefIntroduction"/>

== 极地科学考察站 ==
总部位于上海的[[中国极地研究中心|中国极地研究中心]]管辖：
* [[中国南极长城站|中国南极长城站]]，中国在[[南极|南极]]建立的第1个[[南极科学考察站|科学考察站]]。
* [[中国南极中山站|中国南极中山站]]，中国在南极建立的第2个科学考察站。
* [[中国南极昆仑站|中国南极昆仑站]]，中国在南极建立的第3个科学考察站。
* [[中国南极泰山站|中国南极泰山站]]，中国在南极建立的第4个科学考察站。
* [[中国北极黄河站|中国北极黄河站]]，中国在[[北极|北极]]建立的唯一[[北极科学考察站|科学考察站]]。
