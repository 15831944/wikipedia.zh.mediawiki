'''国民党抗战老兵'''是[[中华人民共和国政府|中华人民共和国政府]]对参加[[抗日战争|抗日战争]]，效忠[[国民政府|国民政府]]的[[国民革命军|国民革命军]]官兵的特定称谓。这是与[[国民党军队|国民党军队]]类似的用法。媒体称之为'''国军抗战老兵'''，或认为应更准确的称为'''国民政府抗战老兵'''<ref name="a1">{{Cite web |url=http://news.ifeng.com/history/zhongguoxiandaishi/detail_2013_07/06/27210659_0.shtml|title=国军抗战老兵：什么都不要 只要国家发枚勋章|accessdate=2019-04-13 |author=刘霞 |date=2013-07-06 |work=责任编辑：高飏 |publisher=凤凰网，来源：云南网|language=Zh-hans }}</ref>。

== 过去 ==

1945年8月[[日本投降|日本投降]]，抗日战争正式结束。其后，[[国共谈判|国共谈判]]失败。[[第二次国共内战|第二次国共内战]]随即爆发。期间，[[国民革命军|国民革命军]]在1947年改称[[中華民國國軍|中华民国国军]]。随着中国共产党军事优势的扩大，大量国军士兵[[起义|起义]]、[[投诚|投诚]]，相当一部分参加解放军，是谓[[解放兵|解放兵]]。1949年，[[中华人民共和国|中华人民共和国]]成立。1950年初，在中国大陆，成建制的国军部队已被解放军消灭殆尽。但有许多国军士兵，包括未参加第二次国共内战的已退役官兵留在了大陆。由于政治原因，这些老兵在中华人民共和国成立后的历次政治运动中受到冲击。改革开放后的数年里，这一群体虽然不再受到政治压力，但是其抗战功绩仍然未得到官方的公开嘉奖。2013年的报道称，民间寻访人数超2000人，基本年过九旬，生活困苦<ref name="a1"/>。

== 现状 ==

在过去的十多年来，中华人民共和国政治氛围宽松，对原效忠于中华民国政府的抗战老兵也逐渐恢复了荣誉。2005年，中华人民共和国政府举办了[[中国人民抗日战争暨世界反法西斯战争胜利60周年重要纪念活动|中国人民抗日战争暨世界反法西斯战争胜利60周年重要纪念活动]]，给每一位参加过抗日战争的老兵颁发纪念章。

2013年[[两会|两会]]期间，[[香港特别行政区全国人民代表大会代表|香港全国人大代表]][[王敏剛|王敏刚]]向[[中华人民共和国民政部|民政部]]递交优抚原抗战老兵的提案<ref>{{Cite web |url=http://www.chinanews.com/sh/2013/07-03/4996376.shtml|title=媒体称民政部将原国民党抗战老兵纳入社会保障范围|accessdate=2019-04-13 |author= |date=2013-07-03 |work=编辑：燕磊 |publisher=中国新闻网，来源：云南网|language=Zh-hans }}</ref>。6月4日，[[中华人民共和国民政部|中华人民共和国民政部]]发布[[红头文件|红头文件]]，将原国军抗战老兵列入优抚对象，享有与退役中国人民解放军同等的社会养老保障待遇。

在2015年8月，民政部、[[中华人民共和国财政部|财政部]]下发通知，向部分健在的抗战老战士、老同志发放一次性生活补助金。具体发放对象包括：1.抗日战争时期的在乡复员军人和残疾军人；2.移交政府安置的抗日战争时期军队离休干部、无军籍职工；3.抗日战争时期在[[国民党军队|国民党军队]]服役，后在解放战争中起义、投诚编入解放军序列的在乡复员军人（解放兵）；4.参加过抗日战争，后回乡务农的原国民党抗战老兵。发放标准为每人5000元<ref>{{Cite web |url=http://www.mof.gov.cn/zhengwuxinxi/caizhengxinwen/201508/t20150811_1413745.html|title=民政部 财政部向部分抗战老战士老同志发放一次性生活补助金|accessdate=2019-04-13 |author= |date=2015-08-11 |work= |publisher=中华人民共和国财政部网站，来源：民政部网站|language=Zh-hans }}</ref>。 

在2015年9月3日的[[纪念中国人民抗日战争暨世界反法西斯战争胜利70周年阅兵式|纪念中国人民抗日战争暨世界反法西斯战争胜利70周年阅兵式]]中，有两个抗战老同志乘车方队参加了阅兵式。这两个抗战老同志乘车方队由从中国大陆遴选的、目前健在的国共两党抗战老兵和抗日英烈后代，以及抗战[[支前|支前]]模范代表组成。
