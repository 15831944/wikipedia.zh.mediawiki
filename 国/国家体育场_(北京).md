'''国家体育场'''位于[[中国|中国]][[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[奥林匹克公园_(北京)|奥林匹克公园]]，是[[2008年夏季奧林匹克運動會|2008年北京奥运会]]的主场馆，由于於独特造型又俗称「'''鸟巢'''」。体育场在奥运会期间设有10万个座位，承办该届奥运会的开、闭幕式，以及[[2008年夏季奧林匹克運動會田徑比賽|田径]]同[[2008年夏季奧林匹克運動會足球比賽|足球]]等比赛项目。奥运会後座位数将减至8万个<ref>[http://2008.sohu.com/20080102/n254422454_2.shtml 鸟巢首次揭开神秘面纱 奥运后将成中超国安主场]</ref>。除2008年夏奧外，此場館將會於[[2022年冬季奧運|2022年冬季奧運]]再次使用。

国家体育场长330米，宽220米，高度为69.2米，占地25万平方米。运用高达121,000噸鋼建成的体育馆，建筑费用高达[[人民币|人民币]]23亿元。此馆于2003年12月动土，2004年3月正式开工。但因庞大的建筑经费而不得不于2004年8月暂时停工。经过一连串的讨论与修改，原计划中体育场的上盖被取消，专家称这不但会使体育场更加安全，同时也可以消减部分开支。最终体育场于2005年初恢复开工，并于2008年6月28日完工。

== 历史 ==
在2002年，[[北京市规划委员会|北京市规划委员会]]举办了2008年[[北京市|北京市]][[2008年夏季奧林匹克運動會|奥运会]]主场馆的全球方案征集活动。由[[普利兹克奖|普利兹克奖]]获得者瑞士[[赫尔佐格和德梅隆|赫尔佐格和德梅隆]]建筑大师[[赫爾佐格|赫爾佐格]]（Herzog）和[[皮埃尔·德麦隆|皮埃尔·德麦隆]]（Pierre Demeuron）和[[中国建筑设计研究院|中国建筑设计研究院]]中方总设计师[[李兴刚|李兴刚]]合作完成。<ref name=":0">{{Cite news|url=http://2008.sina.com.cn/hx/other/2008-08-06/2207157624.shtml|title=资料：国家体育场(鸟巢)介绍|work=新浪奥运|accessdate=2008-08-26}}</ref>「鳥巢」在众多作品中脱颖而出，成为评审委员会选出的3个“优秀方案”之一，最后以压倒多数票获胜，成为[[2008年夏季奧林匹克運動會|2008年北京奥运会]]的主场馆。

[[File:Beijing_National_Stadium_Interior_200709.jpg|thumb]]其後，國家體育場於2003年8月9日簽定合約，确定「以[[中國中信集團公司|中國中信集團公司]]為代表的聯合體，成為2008年奧運會國家體育場項目法人招標的中標人」<ref>[http://www.chinanews.com/n/2003-08-10/26/333660.html 北京奥运会主体育场招标成功 首次引入社会资金]，中国新闻网，2003年08月10日刊载，2014年7月13日引用</ref>。同年12月24日上午10時国家体育场正式開始建造，<ref name=":0" />翌年，[[北京|北京]]市長[[王岐山|王岐山]]提出“节俭办奥运”的口号，希望能将工程造价降到最低。这导致體育場-{zh-hans:于; zh-hant:於}-2004年8月暫停施工，並於同年決定將体育场的可开启上盖去除并擴大開口、取消大部分地下车库，節省3億元和15,000噸的鋼材<ref>[http://unn.people.com.cn/BIG5/14748/2691580.html 北京：國家體育場“鳥巢”去蓋 可望節省3億元] {{webarchive|url=https://web.archive.org/web/20110310155145/http://unn.people.com.cn/BIG5/14748/2691580.html |date=2011-03-10 }}</ref>，以減輕成本和場館的負荷。

在2005年的11月15日，體育場的[[混凝土|混凝土]]主體結構封頂<ref>{{Cite news|url=http://www.china.com.cn/chinese/sy/1031211.htm|title=国家体育场鸟巢混凝土主体结构提前封顶|last=|first=|date=|work=中国网|accessdate=2005-11-16}}</ref>，及後的2006年8月31日中午1時，[[鋼結構|鋼結構]]立面次結構的26個合龙焊口全部完成焊接，鋼結構工程合龙完成<ref>{{Cite web |url=http://www.beijing2008.com/15/99/article212039915.shtml |title=60焊工连续作业12小时 “鸟巢”钢结构工程圆满合龙 |access-date=2006-08-31 |archive-url=https://web.archive.org/web/20070929100042/http://www.beijing2008.com/15/99/article212039915.shtml |archive-date=2007-09-29 |dead-url=yes }}</ref>，同年的9月17日，體育場鋼結構卸載順利完成，共卸載了15,000噸鋼，安裝了78個卸載點。當中，最大的卸載點負荷了320噸的重量。<ref>{{Cite web |url=http://www.beijing2008.com/45/43/article212044345.shtml |title=北京2008年奥运会主会场国家体育场钢结构主桁架梁成功卸载 |access-date=2006-09-17 |archive-url=https://web.archive.org/web/20070927001916/http://www.beijing2008.com/45/43/article212044345.shtml |archive-date=2007-09-27 |dead-url=yes }}</ref>3個月後，钢结构吊装工程也全部完成，共動用了24台吊車、1209人參與。<ref>{{Cite web |url=http://www.beijing2008.com/63/07/article212060763.shtml |title=“鸟巢”钢结构吊装工作全部完成 |access-date=2006-11-30 |archive-url=https://web.archive.org/web/20061207213756/http://www.beijing2008.com/63/07/article212060763.shtml |archive-date=2006-12-07 |dead-url=yes }}</ref>至2008年6月28日，鳥巢正式竣工。<ref>{{cite news|url=http://news.bbc.co.uk/chinese/simp/hi/newsid_7470000/newsid_7478900/7478961.stm|title=北京"鸟巢"举行落成典礼|publisher=BBC中文网|date=2008年6月28日|accessdate=2008年7月25日}}</ref>

「鳥巢」每年的營運開支高達1.5億元人民幣，單靠承辦體育賽事及旅遊門票收入，根本難以維持正常運作。為求生存，「鳥巢」擬出售冠名權，目前已有10多家中外企業角逐，有消息稱，國際知名品牌德國體育用品商-{阿迪达斯|zh-hk:adidas;zh-tw:爱迪斯;zh-cn:阿迪达斯}-有可能奪得「鳥巢」的冠名權，代價是每年7000萬元人民幣。相當於每回$1344。<ref>{{cite news|url=http://hk.news.yahoo.com/article/090225/4/avew.html |title=鳥巢冠名權 |publisher=明報 |date=2009-02-26 |language={{Lan|zh-hans=zh-hans|zh-hant=zh-hant}} |deadurl=yes |archiveurl=https://web.archive.org/web/20090301214758/http://hk.news.yahoo.com/article/090225/4/avew.html |archivedate=2009-03-01 }}</ref>

2009年4月2日，國際巨星[[成龍|成龍]]宣佈於5月1日在國家體育場舉行「龍的傳人」和他的朋友們2009北京演唱會，成為在「鳥巢」舉辦演唱會的第一位歌手。<ref>[http://paper.wenweipo.com/2009/04/02/EN0904020001.htm 「鳥巢」開唱第一人 成龍「五．一」打頭炮] {{webarchive|url=https://web.archive.org/web/20090405084546/http://paper.wenweipo.com/2009/04/02/EN0904020001.htm |date=2009-04-05 }}香港文匯報，2009年4月2日</ref>[[File:Beijing_National_Stadium_1.jpg|thumb]]中國音樂家協會、北京中信聯合體體育場運營公司、中國對外演出公司三家單位聯合宣佈，由2009年開始，每年都將固定在6月30日在國家體育場舉辦「中國北京鳥巢夏季音樂會」。<ref>[http://www.beijinggym.com/c5482/c5942/w10054319.asp 宋祖英鸟巢音乐会多明戈郎朗周杰伦携手进鸟巢-2009《魅力·中国》] {{webarchive|url=https://web.archive.org/web/20090430155951/http://www.beijinggym.com/c5482/c5942/w10054319.asp |date=2009-04-30 }}北京體育場館信息網</ref>

中國兒童少年基金會等多家單位在人民大會堂北京廳聯合舉行新聞發佈會，宣佈於十月國慶六十周年之際，在國家體育場舉行「光榮與夢想，輝煌耀祖國——獻給祖國母親六十華誕、《同一首歌》走進鳥巢」演唱會。<ref>[http://www.beijinggym.com/c5482/c5942/w10055719.asp <同一首歌>十月走進鳥巢 經典體育歌曲獻禮國慶]{{dead link}}北京體育場館信息網</ref>

2009年10月6日晚，張藝謀攜奧運會開閉幕式原班創作團隊執導的世界經典劇目《圖蘭朵》鳥巢版在北京國家體育場上演。<ref>[http://www.n-s.cn/cn/dynamics/venueconstruction/n214610828.shtml 鸟巢版《图兰朵》隆重上演] {{webarchive|url=https://web.archive.org/web/20091126132402/http://www.n-s.cn/cn/dynamics/venueconstruction/n214610828.shtml |date=2009-11-26 }}</ref>

國家體育場的外形是以眾多鋼鐵不規則地「編織」而成，外形如一個[[鸟巢|鸟巢]]般，因此又稱為“鳥巢”体育场。而鋼架的空隙拉上半透明的充氣薄膜，既可防水又可讓陽光滲進。

體育場建築面積25.8萬[[平方米|平方米]]，用地面積20.4萬平方米，<ref>{{Cite news|url=http://news.xinhuanet.com/politics/2008-06/28/content_8455370.htm|title=“鸟巢”举行落成典礼 刘淇、郭金龙等出席|work=新华网|accessdate=2008-06-28}}</ref>总面积约为55万平方米。<ref>{{Cite news|url=http://style.sina.com.cn/des/design/2012-06-27/070799002.shtml|title=中国最昂贵的十大建筑|work=新浪尚品|accessdate=2012-06-27}}</ref>混凝土結構主體分地下一層，地上七層，地上高度69.21米，整個建築造型呈橢圓的馬鞍形。<ref>{{Cite news|url=http://sports.enorth.com.cn/system/2013/01/08/010496463.shtml|title=国内最烧钱建筑排行TOP10 鸟巢水立方在列(图)|work=北方网|accessdate=2013-01-08}}</ref>內設100,000個座席，當中包括永久座席80,000個，臨時座席11,000個。<ref>{{Cite news|url=http://news.sohu.com/20080725/n258378096.shtml|title=艾未未：鸟巢是这个时代野心和理想碰撞的结果|work=搜狐新闻|accessdate=2008-07-25}}</ref>同時，場館使用了[[聚四氟乙烯|聚四氟乙烯]]的气枕式薄膜，這薄膜的好處是耐腐蝕、透光性能好，可以柔和的散射开阳光，避免结构框架投下凌乱的阴影。

== 鸟巢文化中心 ==
2015年3月21日，经过一年多时间筹备的鸟巢文化中心在国家体育场正式成立。同日揭牌的还有作为鸟巢文化中心组成部分的创客孵化平台“亮中国”<ref>{{Cite web |url = http://www.chinanews.com/ty/2015/03-21/7147787.shtml |title =鸟巢文化中心成立 推出体育经济孵化器  |author =  |date = 2015-03-21 |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref>。鸟巢文化中心的定位是“体育文化金融孵化平台”和“首都公共文化空间”。鸟巢文化中心自成立后，主要举办路演、论坛、发布、展览、比赛等活动。2015年底，鸟巢文化中心引入雅昌艺术书店，成立了“雅昌鸟巢体育艺术书店”并对外开放，店内大量使用雅昌艺术数据库产品，并提供数字艺术图书博物馆服务<ref>{{Cite web |url =http://www.sohu.com/a/81193603_413933  |title =鸟巢文化中心：打造体育双创平台  |author =  |date = 2016-06-06 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。
