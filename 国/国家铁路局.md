'''国家铁路局'''{{NoteTag|该机构官方标准全名中不冠以“[[中华人民共和国|中华人民共和国]]”或“[[中国|中国]]”[[国号|国号]]。}}，是[[中华人民共和国国务院|中华人民共和国国务院]]负责[[中华人民共和国铁路运输|全国铁路运输]]监管工作的[[国务院部委管理的国家局|国务院部委管理的国家局]]，现由[[中华人民共和国交通运输部|交通运输部]]管理<ref name="官方简称">{{cite web|url=http://www.gdllw.org/tzgg/201806/W020180622515099385181.pdf|title=关于印发国务院机构简称的通知|accessdate=2018-11-25|author=国务院办公厅秘书局|date=2018-06-22|publisher=广东老龄网}}</ref>。

国家铁路局主要负责拟订铁路技术标准，监督管理铁路安全生产、运输服务质量和铁路工程质量等工作<ref name="报道0312B">{{cite web |author1=中华人民共和国国务院 |title=关于国务院机构改革和职能转变方案的说明 |url=http://www.xinhuanet.com//2013lh/2013-03/10/c_114969788_2.htm |website=新华网新闻 |accessdate=2019-03-12 |location=北京 |language=zh-cn}}</ref>。

== 沿革 ==
1949年10月1日，根据9月27日[[中国人民政治协商会议第一届全体会议|中国人民政治协商会议第一届全体会议]]通过的《[[s:中華人民共和國中央人民政府組織法|中华人民共和国中央人民政府组织法]]》第十八条的规定，撤销中央军委铁道部，设立[[中央人民政府铁道部|中央人民政府铁道部]]<ref name="组织史资料F1"/>。铁道部成立初期设22个部、局、室、处，铁道部机关职工2034人<ref name="组织史资料F1">{{cite book|title=中国共产党组织史资料（附卷一）|last=中国共产党组织史资料编审委员会|accessdate=2018-12-27|year=1998|location=北京}}</ref>。

1954年9月，[[第一届全国人民代表大会第一次会议|第一届全国人民代表大会第一次会议]]在[[北京|北京]]召开，会议通过了《[[中华人民共和国宪法|中华人民共和国宪法]]》和《中华人民共和国国务院组织法》，成立[[中华人民共和国国务院|中华人民共和国国务院]]。根据国务院《关于设立、调整中央和地方国家机关及有关事项的通知》，中央人民政府铁道部即告结束。国务院按照《国务院组织法》的规定，将原[[中央人民政府铁道部|中央人民政府铁道部]]改为'''[[中华人民共和国铁道部|中华人民共和国铁道部]]'''，作为[[国务院组成部门|国务院组成部门]]<ref name="组织史资料F1"/>。

[[文化大革命|文化大革命]]爆发后，铁道部一度被造反派夺权。1967年1月28日，国务院决定成立铁道部临时业务监督小组，处理铁道部日常工作。同年5月31日，铁道部实施军管，铁道部军事管制委员会设政工组、生产组、后勤组、办事组<ref name="组织史资料F1"/>。

1970年6月，铁道部与[[中华人民共和国|交通部]]、[[中华人民共和国邮电部|邮电部]]合并成立新的交通部，交通部内设政工组、运输组、办事组、计划基建组<ref name="组织史资料F1"/>。

1975年，经中央批准，恢复独立建制，重新组建'''中华人民共和国铁道部'''。铁道部设办公室、运输局、机车车辆局、工电局、工业局、基建局、计划统计局、物资局、财务局、人事局、公安局、外事局、安全监察委员会、科技委员会共14个内设机构<ref name="组织史资料F1"/>。1982年底，铁道部内设机构扩编为21个，即办公厅和18个局、3个办公室。

1983年10月1日，[[中国人民解放军铁道兵|中国人民解放军铁道兵]]移交铁道部，转为企业编制<ref name="组织史资料F1"/>。1988年，在国务院机构改革中，保留铁道部，将铁道部内设机构调整为24个，即办公厅、11个司、6个局、6个政工机构和老干部局<ref name="组织史资料F1"/>。同时由[[中华人民共和国监察部|监察部]]、[[中华人民共和国审计署|审计署]]派驻监察、审计机构<ref name="组织史资料F1"/>。

1993年，国务院决定将铁道部内设机构缩减为22个，部机关设置办公厅、10个司、6个局、5个政工机构和离退休干部局<ref name="组织史资料F1"/>。1998年，国务院宣布逐步实施铁路政企分开，明确界定政府管理职能、社会管理职能、企业管理职能<ref name="组织史资料F1"/>。铁道部内设机构精简为16个，即办公厅、9个司、2个局、4个政工机构和离退休干部局<ref name="组织史资料F1"/>。

2013年2月26日至28日召开的[[中国共产党第十八届中央委员会第二次全体会议|中国共产党第十八届中央委员会第二次全体会议]]审议通过《[[国务院机构改革|国务院机构改革]]和职能转变方案》，建议国务院将这个方案提交[[第十二届全国人民代表大会#第一次会议|第十二届全国人民代表大会第一次会议]]审议<ref>{{cite web|url=http://news.xinhuanet.com/politics/2013-02/28/c_114843346.htm|title=中国共产党第十八届中央委员会第二次全体会议公报|date=2013-02-28|publisher=[[新华网|新华网]]}}</ref>。

2013年3月10日，[[国务委员|国务委员]]兼[[中华人民共和国国务院秘书长|国务院秘书长]][[马凯_(官员)|马凯]]受国务院委托，向第十二届全国人民代表大会第一次会议第三次全体会议作关于《国务院机构改革和职能转变方案》的说明<ref>{{Cite web|url=http://www.gov.cn/2013lh/content_2350848.htm|title=关于国务院机构改革和职能转变方案的说明|accessdate=2013-04-21|date=2013-03-10|publisher=[[中国政府网|中国政府网]]}}</ref>。该方案提及：“组建国家铁路局，由[[中华人民共和国交通运输部|交通运输部]]管理，承担铁道部的其他行政职责；组建[[中国国家铁路集团有限公司|中国铁路总公司]]，承担铁道部的企业职责，不再保留[[中华人民共和国铁道部|铁道部]]。”

2013年3月14日，[[第十二届全国人民代表大会#第一次会议|第十二届全国人民代表大会第一次会议]]表决通过《[[s:国务院机构改革和职能转变方案|国务院机构改革和职能转变方案]]》，规定“组建'''国家铁路局'''，由交通运输部管理，承担铁道部的其他行政职责，负责拟订铁路技术标准，监督管理铁路安全生产、运输服务质量和铁路工程质量等。”<ref>{{Cite news|url=http://www.gov.cn/2013lh/content_2354397.htm|title=第十二届全国人民代表大会第一次会议关于国务院机构改革和职能转变方案的决定|date=2013-03-14|work=中国政府网|accessdate=2013-04-21}}</ref>

2013年3月19日，原铁道部副部长[[陆东福|陆东福]]被任命为国家铁路局局长，兼任[[中华人民共和国交通运输部|交通运输部]]副部长<ref name="wzkt">{{cite web|url=http://www.nra.gov.cn/zwzc/xwdt/xwlb/201401/t20140106_4492.htm|title=国家铁路局开通政府网站|accessdate=2014-01-07|date=2014-01-06|publisher=国家铁路局|archiveurl=https://web.archive.org/web/20140106143415/http://www.nra.gov.cn/zwzc/xwdt/xwlb/201401/t20140106_4492.htm|archivedate=2014-01-06|deadurl=yes}}</ref>。2013年3月21日，[[中华人民共和国国务院办公厅|国务院办公厅]]发出《国务院办公厅关于印发国家铁路局主要职责内设机构和人员编制规定的通知》（国办发〔2013〕21号）<ref name="gdtz">{{cite web|url=http://www.gov.cn/zwgk/2013-05/15/content_2403673.htm|title=国务院办公厅关于印发国家铁路局主要职责内设机构和人员编制规定的通知|accessdate=2019-01-14|date=2013-03-21|publisher=[[中华人民共和国国务院办公厅|中华人民共和国国务院办公厅]]}}</ref>。2013年7月11日，原铁道部的[[中华人民共和国公务员|公务员]]完成分流，国家铁路局召开第一次干部大会，开始边组建、边履职。2013年12月28日，国家铁路局迁入北京市[[复兴路_(北京)|复兴路]]9号（原[[中国铁路北京局集团有限公司|北京铁路局]]所在地）办公<ref name="wzkt"/>。2014年1月6日，国家铁路局召开组建后的第一次工作会议，[http://www.nra.gov.cn/ 国家铁路局政府网站]开通。

== 职责 ==
根据《{{s|国家铁路局主要职责内设机构和人员编制规定}}》，国家铁路局承担下列职能<ref name="gdtz" />：
{{columns-list|2|
# 起草铁路监督管理的法律法规、规章草案，参与研究铁路发展规划、政策和体制改革工作，组织拟订铁路技术标准并监督实施。
# 负责铁路安全生产监督管理，制定铁路运输安全、工程质量安全和设备质量安全监督管理办法并组织实施，组织实施依法设定的行政许可。组织或参与铁路生产安全事故调查处理。
# 负责拟订规范铁路运输和工程建设市场秩序政策措施并组织实施，监督铁路运输服务质量和铁路企业承担国家规定的公益性运输任务情况。
# 负责组织监测分析铁路运行情况，开展铁路行业统计工作。
# 负责开展铁路的政府间有关国际交流与合作。
# 承办国务院及交通运输部交办的其他事项。
}}
