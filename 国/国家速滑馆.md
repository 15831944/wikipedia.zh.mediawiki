'''国家速滑馆'''，位于[[中国|中国]][[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[奥林匹克森林公园|奥林匹克森林公园]]。

==简介==
国家速滑馆的原址是[[2008年夏季奧林匹克运动会|2008年夏季奧林匹克运动会]]的[[北京奥林匹克公园曲棍球场|曲棍球场]]和[[北京奥林匹克公园射箭场|射箭场]]，两者均按照临时赛场建设。2017年，为兴建国家速滑馆，将两座场馆拆除<ref name=bsd>{{Cite web |url = http://news.163.com/17/0425/03/CIRB2V70000187VI.html |title = 国家速滑馆飘起“冰丝带” |author =  |date = 2017-04-25 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{cite web|url=http://www.bjzdb.gov.cn/jsxx/t1102645.htm|title=本市加快推进国家速滑馆项目地上物拆除工作|publisher=北京市重大项目建设指挥部办公室|date=2017-04-17|accessdate=|deadurl=yes|archiveurl=https://web.archive.org/web/20170418082005/http://www.bjzdb.gov.cn/jsxx/t1102645.htm|archivedate=2017年4月18日|df=}}</ref>。

国家速滑馆是[[2022年冬季奧林匹克运动会|2022年冬季奧林匹克运动会]]北京赛区唯一新建的竞赛场馆，将在赛时承担[[速滑|速滑]]项目比赛和训练。该馆位于奥林匹克森林公园西部，[[国家网球中心|国家网球中心]]南侧，原址是2008年北京奥运会的临时场馆射箭场、曲棍球场<ref>{{Cite web |url = http://sports.163.com/16/0226/13/BGOMUQ4500051CAQ.html |title = 2022冬奥会场馆巡礼 新建国家速滑馆颜值爆表 |author =  |date = 2016-02-26 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。2016年6月6日，北京冬奥组委协调[[北京市规划和国土资源管理委员会|北京市规划和国土资源管理委员会]]作为国家速滑馆的代业主，展开国家速滑馆建筑概念方案国际竞赛<ref>{{Cite web |url = http://bj.people.com.cn/n2/2016/0726/c339781-28732954.html |title = 冬奥会场馆规划设计全面启动 国家速滑馆明年上半年开工 |author =  |date =2016-07-26  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。2016年7月6日，经资格预审，12家设计机构获得正式参赛资格<ref>{{Cite web |url = http://news.xinhuanet.com/local/2016-07/24/c_129172634.htm |title = 国家速滑馆建筑概念方案国际竞赛揭幕 |author =  |date =  2016-07-24|publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref name=bsd/>。最终以“冰丝带”为理念的A04号设计方案获胜，此竞赛方案由中国建筑师郑方创作完成。2017年4月25日该设计方案开始在[[北京市规划展览馆|北京市规划展览馆]]奥运工程建设馆对外展出。为解决赛后利用问题，该项目还在场地西南角规划预留了副馆。北京冬奥会后，国家速滑馆将成为举办[[滑冰|滑冰]]、[[冰球|冰球]]、[[冰壶|冰壶]]等国际赛事，以及群众冰上活动的多功能场馆<ref name=bsd/><ref>{{Cite web |url = http://news.xinhuanet.com/local/2017-04/26/c_129574217.htm |title = 北京冬奥会国家速滑馆设计方案亮相 |author =  |date = 2017-04-26 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

2017年7月14日，[[北京市人民政府新闻办公室|北京市人民政府新闻办公室]]召开“北京2022年冬奥会国家速滑馆政府与社会资本合作（PPP）项目推介新闻发布会”，国家速滑馆开始招募社会资本合作者对场馆进行建设、运营，国家速滑馆项目PPP模式的实施机构是[[北京市重大项目建设指挥部办公室|北京市重大项目建设指挥部办公室]]，政府出资代表是北京市国有资产经营有限责任公司<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1732943 |title = 北京冬奥会国家速滑馆开始招募社会资本合作者 |author =  |date = 2017-07-14  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

国家速滑馆占地面积约17公顷。除了地下停车场外，建筑面积还有约8万平方米。场馆由22条像“冰丝带”的灯带环绕。场馆座席1.2万个。场馆东西约180米，南北约240米，地上3层，地下2层。建筑高度33.8米。内设标准400米速滑赛道<ref name=bsd/>。
