'''国家海洋委员会'''，是[[中华人民共和国国务院|中华人民共和国国务院]]拟成立的高级别[[国务院议事协调机构|国务院议事协调机构]]，其具体工作由[[中华人民共和国国土资源部|中华人民共和国国土资源部]]管理的[[国家海洋局|国家海洋局]]承担<ref>{{cite web |url = http://xmwb.news365.com.cn/yw_101/201303/t20130310_999007.html |language = zh |title = 职责一分为三 铁道部不再保留 |publisher = [[新民晚报|新民晚报]] |date = 2013-03-10 |accessdate = 2013-03-10 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==沿革==
2013年2月，[[中共十八届二中全会|中共十八届二中全会]]审议通过了《国务院机构改革和职能转变方案》，建议国务院将该方案提交第十二届全国人民代表大会第一次会议审议。2013年3月，国务委员兼国务院秘书长[[马凯_(官员)|马凯]]在[[第十二届全国人民代表大会第一次会议|第十二届全国人民代表大会第一次会议]]上作《关于国务院机构改革和职能转变方案的说明》，其中提出重新组建国家海洋局，由[[中华人民共和国国土资源部|中华人民共和国国土资源部]]管理，还提出“设立高层次议事协调机构国家海洋委员会，负责研究制定国家海洋发展战略，统筹协调海洋重大事项。国家海洋委员会的具体工作由国家海洋局承担。”<ref>{{Cite web |url = http://cpc.people.com.cn/n/2013/0311/c164113-20743165.html  |title = 马凯：拟重新组建国家海洋局 设立国家海洋委员会 |author =  |date =2013-03-11  |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>

2013年6月9日，《国务院办公厅关于印发国家海洋局主要职责内设机构和人员编制规定的通知》（国办发〔2013〕52号）发出，其中规定国家海洋局战略规划与经济司“承担国家海洋委员会办公室日常工作”<ref>{{Cite web |url = http://www.gov.cn/zwgk/2013-07/09/content_2443023.htm   |title = 国务院办公厅关于印发国家海洋局主要职责内设机构和人员编制规定的通知（国办发〔2013〕52号） |author =  |date =2013-07-09  |publisher =  中国政府网|language =  |accessdate =  |quote =  }}</ref>。

但是此后，国家海洋委员会一直未能成立。
