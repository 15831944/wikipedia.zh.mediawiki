'''国家机关事务管理局东坝服务中心'''，是[[国家机关事务管理局|国家机关事务管理局]]直属[[事业单位|事业单位]]。

==沿革==
1991年，[[国务院机关事务管理局|国务院机关事务管理局]]和[[中华人民共和国人事部|人事部]]下发《关于加强中央国家机关后勤干部岗位培训工作的通知》（〔91〕国管培训字第229号）。1993年4月22日，[[国务院机关事务管理局|国务院机关事务管理局]]办公室以〔93〕局办字第11号印发《关于中央国家机关后勤干部岗位培训工作的实施意见》，内称“培训工作的日常管理和组织实施由我局中央国家机关后勤干部培训中心负责”，并附“中央国家机关后勤干部培训中心1993年——1995年培训工作重点”，该工作重点“（二）抓紧中央国家机关后勤干部培训基地建设，争取早日建成。”<ref>{{Cite web |url =http://gkml.ggj.gov.cn/zfxxgk/zhxxml/xxgkzcfg/201103/t20110325_569.htm  |title =关于中央国家机关后勤干部岗位培训工作的实施意见  |author =  |date =1993-04-22  |publisher =国家机关事务管理局  |language =  |accessdate =  |quote =  }}</ref>

'''中央国家机关后勤干部培训基地'''，隶属[[国务院机关事务管理局|国务院机关事务管理局]]，承担着为后勤干部培训提供服务保障任务<ref>中央国家机关后勤干部培训基地 中央国家机关会议定点单位，中国机关后勤2009(4)</ref>。2013年，中央国家机关后勤干部培训基地（中央国家机关农林生产基地）更名为'''国家机关事务管理局东坝服务中心'''<ref>{{Cite web |url = http://www.gjsy.gov.cn/djgg/bggg/201311/t20131112_2206.html |title = 国家事业单位登记管理局公告 |author =  |date = 2013-11-12 |publisher = 事业单位在线 |language =  |accessdate =  |quote =  }}</ref>。
