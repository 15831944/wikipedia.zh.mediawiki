'''国民党青岛市党部大楼'''建于1914年日军占领青岛后不久，最初是一栋商业建筑，1931年[[中国国民党|国民党]]青岛市党部曾设于此。该建筑现已不存在，原址位于太平路37号华能大厦址。

==历史==
===初建===
[[File:青岛占领军司令神尾光臣回国.jpg|thumb]]
[[File:青岛太平路第一次日治时期.jpg|thumb]]
德租时期，威廉皇帝海岸（Kaiser-Wilhelm-Ufer，今[[太平路_(青岛)|太平路]]）、阿尔贝特街（Albertstraße，今[[安徽路_(青岛)|安徽路]]）、路易特波尔德街（Luitpoldstraße，今[[浙江路_(青岛)|浙江路]]）与海因里希亲王街（Prinz-Heinrich-Straße，今[[广西路_(青岛)|广西路]]）所合围成的街区因地价偏高而长期闲置，直至1911年德国商人马克斯·格里尔在东北角建设了他的[[广西路_(青岛)#吉利洋行|百货商店楼]]。

1911年末总督府废除“华洋分区”后，华人开始大规模投资欧人区房地产。原江苏盐道李梅荪买下了该街区的其他三个地块，并于大约1913年在西北角建设了[[广西路_(青岛)#李氏商业楼|两栋商业楼]]，在东南角建设了一栋[[太平路_(青岛)#李氏住宅楼|住宅楼]]。1913年地籍图显示西南角李梅荪的地块有一个形如大乌龟的建筑轮廓，一般来说，这代表该处已经建起了一栋建筑<ref>{{cite web |url=http://www.tsingtau.org/kataster-karten-tsingtau-1914/|title=Katasterkarten Tsingtau}}</ref>。然而1914年日军[[青岛战役|占领]]青岛后，三船写真馆发行的《日独战役青岛名所写真帖》中包含的两张展现1914年12月青岛占领军司令神尾光臣自[[青岛栈桥|栈桥]]乘船回国的情景的照片中，“大乌龟”的位置却空无一物，只能看到北侧两栋商业楼的背面。直至后来的一些照片中，该位置才出现了一栋上下三层加蒙莎屋顶、四角各有一塔楼的庞大建筑。据推测，该楼的建设计划在1913年便已提交总督府相关部门并获批准，只是因为战争原因，直至约1915年才建起来。至于该楼的设计师，青岛本地学者认为，从风格上判断，该楼的设计者极有可能是实业家[[刘子山|刘子山]]的合伙人兼业余建筑师、浙江籍商人[[杨浩然_(商人)|杨浩然]]<ref>{{cite book |author=李明 |title=《塔楼上的青岛》|year=2008 |publisher=青岛出版社 |location=青岛 |isbn=9787543649705}}</ref>。而杨浩然为自己设计的在今甘肃路的[[甘肃路_(青岛)#杨家大宅|住宅]]同样曾于1914年因战争原因停工。

该建筑后曾为1922年成立的中日合资鲁大矿业公司的地产，鲁大矿业公司成立后便于当年因欠债将大楼抵押给[[刘子山|刘子山]]开办的[[东莱银行|东莱银行]]。东莱银行入手该楼后，刘子山全家迁居至此，却因临海建筑过于潮湿而在此仅仅居住一年左右便搬离此处。由于找不到整楼租赁者，大楼曾于此后闲置了一段时间。1926年，因[[美国海军|美国海军]]舰船常至青岛驻泊，该楼内设置了美国海军俱乐部（Navy Club），直至1929年。约1927年，大楼周围加建围墙。约1929年后，有一家名叫“Castle Inn”（城堡旅店）的旅店在这里开设，可能仅租赁了该楼的一部分<ref Name=feng>{{cite book|author=贺伟|title=《风雨半城山：刘子山传奇》|year=2017|publisher=青岛出版社|location=青岛|isbn=978-7-5552-4728-9}}</ref>。

===国民党青岛市党部大楼===
[[File:青岛国民党市党部被焚毁.jpg|thumb]]
1924年，国民党青岛市党部执行委员会成立，1927年被[[张宗昌|张宗昌]]取缔。1928年，青岛的国民党日常党务活动恢复，丁振潢为代表。1929年4月，南京国民政府接收青岛后再度成立市党部，5月2日确认由行政院直接管辖。国民党青岛市党部执行委员会改组为国民党青岛特别市党务指导委员会<ref Name=feng/>。

1929年6月，青岛接收专员督办公署致函青岛市总商会，称国民党青岛市党部临时租用青岛农商银行房屋办公，因其面积狭小，欲搬到[[天津路_(青岛)|天津路]][[河南路_(青岛)|河南路]]路口的东莱银行大楼内办公，请总商会联系洽谈。当时的东莱银行因不堪军阀借款已停业半年，[[东莱银行|东莱银行]]青岛清理处主任朱品生请示总行后，同意将大楼二层、三层、四层借给市党部使用。由于政府财政异常拮据，市党部在使用天津路银行大楼约一年之内没有交付任何房租<ref Name=feng/>。

约1930年至1931年上半年间，市党部迁入同属东莱银行财产的太平路原美国海军俱乐部办公，在此期间政府是否曾交付租金已无法获知<ref Name=feng/>。

{{Main|青岛民国日报事件}}
1932年1月9日，青岛《民国日报》以《韩国不亡，义士李奉昌刺日皇未遂》为题报导[[樱田门事件|樱田门事件]]，遭到日本驻青岛总领事馆的抗议。报社按照日方要求道歉后，日本侨民又对报社所在地和国民党青岛市党部大楼进行多轮破坏，造成两楼被焚毁。市党部大楼为1月12日晚上9点被纵火焚烧，大楼内部结构及阁楼被焚毁。该事件后，《民国日报》停刊，市党部迁往[[栖霞路|栖霞路]]办公<ref name=ri>{{cite web |url=http://news.bandao.cn/news_html/201512/20151222/news_20151222_2595720.shtml|title=1932年日本侨民暴动:曾火烧国民党青岛市党部}}</ref><ref>{{cite web |url=http://pinglun.qingdaonews.com/content/2014-07/30/content_10594521.htm|title=沈鸿烈捍卫言论自由}}</ref><ref>{{cite web |url=http://arc.dailyqd.com/2015-12/23/content_300691.htm|title=日侨暴动焚烧国民党青岛市党部 捣毁民国日报社}}</ref>。

===三十年代以后的命运===
[[File:青岛中央银行大楼_原国民党市党部.jpg|thumb]]
1934年，刘少山（[[刘子山|刘子山]]之子）赴青岛处理大楼修复事宜，由于他认为带有尖顶的楼房会带来凶兆，按照他的意见，整个阁楼和四个塔冠被拆除，大楼改为平顶。修复后，东莱银行以每月700银元的价格将其出租给中央银行青岛分行<ref Name=feng/>。

1938年日军占领青岛后，[[日本国驻青岛总领事馆|日本驻青岛总领事馆]]迁入该楼，同时在此设立[[兴亚院|兴亚院]]华北联络部青岛出张所。1945年[[日本投降|日本投降]]后，山东敌伪产业管理局将此楼作为敌产接收。刘子山家族向市政府交涉，要求将其作为私产发还。[[国民党中央党部|国民党中央党部]]战地党务处处长葛覃出任青岛市党部主任兼青岛市副市长后，接见东莱银行青岛分行原经理、刘子山女婿徐希文，希望刘家能主动将大楼赠送给市党部，被刘家拒绝。敌伪产业管理局则认为，凡是从[[日军|日军]]和日本政府机关接收来的地产均为敌产，国民党青岛市党部因此再次迁入办公<ref Name=feng/>。

1949年[[解放军|解放军]]进驻青岛后，该楼因曾为国民党市党部使用，被青岛市房产局接收代管。此时仍作为私营银行营业的东莱银行要求返还私产，房产局以尚待审查为由未准。之后，东莱银行又申请将其捐献给国家，[[青岛市人民政府|青岛市人民政府]]亦未接受。1951年年底，青岛市开始房地产权登记，该建筑被认定为刘子山私产，由房管局代管。1956年3月，青岛市正式开始对私有房产进行[[社会主义改造|社会主义改造]]，该建筑后长期为[[中国共产主义青年团|共青团]]青岛市委、青岛市[[妇联|妇联]]和市人大的办公楼，该楼后来被加盖三层楼。1992年，该楼连同该街区其他建筑全部被拆除，原址建设华能大厦<ref Name=feng/><ref name=ri/>。
