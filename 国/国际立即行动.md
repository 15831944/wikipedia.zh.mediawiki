'''国际立即行动'''（{{Lang-en|'''OutRight Action International'''}}）是一个[[女同性恋|女同性恋]]、[[男同性恋|男同性恋]]、[[双性恋|双性恋]]、[[跨性别|跨性别]]、[[双性人|双性人]]和[[疑性恋|疑性恋]]（{{lang|en|LGBTIQ}}）[[人权|人权]]的[[非政府组织|非政府组织]]，<ref>{{cite web |title=OutRight Action International |url=https://www.charitynavigator.org/index.cfm?bay=search.summary&orgid=8256 |publisher=Charity Navigator |accessdate=2019-04-14}}</ref> 旨在解决人权暴力及针对女同性恋者、男同性恋者、双性恋者和跨性别者的虐待行为。国际立即行动在地方、地区、国家及国际层面与活动家、倡导者、媒体、非政府组织及其他盟友合作，以记录基于[[性取向|性取向]]或[[性别认同|性别认同]]及[[性别表达|表达]]的人权歧视和虐待行为。国际立即行动拥有联合国经济及社会理事会的咨询商地位。<ref>{{cite web |title=Continuing Regular Session, Non-Governmental Organizations Committee Takes Note of 114 Quadrennial Reports, Considers Requests for Name Changes |url=https://www.un.org/press/en/2019/ecosoc6960.doc.htm |publisher=[[联合国|联合国]] |accessdate=2019-04-14}}</ref>

=== 菲丽帕奖 ===
