'''國際獅子會'''（[[英語|英語]]：{{lang|en|'''Lions Clubs International'''}}）於1917年由{{link-en|梅爾文·瓊斯|Melvin Jones}}成立，是世界最大的服務[[组织_(社会学)|組織]]，總部設於[[美國|美國]]。擁有46,000個分會及140萬個會員分佈於世界。

== 標誌 ==
字母L之外圍圈圈代表獅友們與獅子會結合成一個整體，兩旁之獅頭側向代表獅友們向每一方向尋覓為全世界人群服務及邀請新會員之機會。

== 分會 ==
狮子会于[[美國|美國]][[伊利诺斯州|伊利诺斯州]]的{{link-en|橡木溪 (伊利諾州)|Oak Brook, IL|橡木溪}}成立。在1920年3月12日成为一个真正的国际性组织。其第一个分会在[[加拿大|加拿大]][[安大略省|安大略省]]的[[温莎_(安大略省)|温莎]]成立。

通常，会员们每月或每两周一次参加会议。政治、宗教或其他争议性的主题是被禁止讨论的。狮子会是一个真正的社团组织，其成员按地区、区域、国家等分别组成。狮子会运营中另一个有趣的现象是它所获得的公众捐助均用于慈善事业，自身的运营费用严格与捐助分开，由会员自行承担。

在西方，狮子会会员的平均年龄不断上升，但会员数量在逐渐下降。这和其他[[服务性社团|服务性社团]]如[[扶轮社|扶轮社]]面临的情况相同。

== 拓展 ==
* 1917年：{{USA}}
* 1920年：{{CAN}}
* 1926年：{{ROC-TW}}[[天津|天津]]（後於1949年隨[[國民政府|國民政府]]遷至臺灣）
* 1927年：{{MEX}}[[新拉里多|新拉里多]]
* 1927年：{{CUB}}[[哈瓦那|哈瓦那]]
* 1955年：{{BHKG}}（今{{HKG}}）和{{PMAC}}（今{{MAC}}）
* 2002年5月14日：{{PRC}}[[廣東|廣東]][[深圳|深圳]]，是1949年后中華人民共和國政府首次批准在[[中国大陆|中国大陆]]设立国际性服务团体。

自[[联合国|联合国]]从1945年成立伊始，国际狮子会一直保持着与之的良好合作关系。它也是被邀请参与起草《[[联合国宪章|联合国宪章]]》的非政府组织。

狮子会在为盲人和视力受损人士提供服务方面享誉全球。这一服务是从1925年6月30日为[[海伦凯勒|海伦凯勒]]出席在[[俄亥俄州|俄亥俄州]][[雪松市|雪松市]]召开的国际会议提供服务之后开始。因此狮子会也被称为“盲人的骑士”。

针对这些服务：
*狮子会为盲人提供[[白杆|白杆]]作为盲人的标记。
*狮子会建议并支持世界上绝大部分的眼库和数百家眼科研究中心、眼科医院和诊所。
*狮子会每年为发展中国家募集超过5百万副旧眼镜。

=== 澳大利亚 ===
狮子会在社区听力和癌症检查项目上提供了大力支持。在[[珀斯|珀斯]]，提供了超过30年的听力检查服务，并提供了基金，在2001年9月9日成立了[[狮子会耳科和听力研究中心|狮子会耳科和听力研究中心]]。这家中心在听力受损的诊断、研究方面非常卓越。

在[[珀斯|珀斯]]，[[西澳大利亚|西澳大利亚]]狮子会还捐赠了[[狮子会眼科研究中心|狮子会眼科研究中心]]。

在[[布里斯班|布里斯班]]，[[昆士兰|昆士兰]]狮子会医疗研究基金会为很多研究人员提供资金支持。

== 青少獅會 ==
第一間'''青少獅會'''是1957年由GLENSIDE獅子會在美國宾夕法尼亚州創立。時至今日，全世界青少獅會約5,700間，遍佈139個國家，會員人數多達150,000人。<ref>{{cite web|title=青少年義工機會 – 青少獅會|url=http://www.lionsclubs.org/TC/our-work/youth-programs/leo-clubs/index.php|accessdate=2014-05-07}}</ref>

青少獅LEO英文名其原意是L（Leadership）、E（Experience）、O（Opportunity），意即發揮領導才能，透過社會服務去吸取經驗及爭取機會去達成國際了解。

*讓青年人有機會參與社會服務工作
*發展個人的潛能和責任感
*激發會友接受更高的道德標準
*發揮領導才能
*提供機會去達成國際了解

== 香港青年獅子會 ==
'''簡介'''
*國際獅子總會中國港澳303區青年獅子區會（Leo Distrinct 303, Hong Kong & Macau, China，簡稱青獅會）於1979年6月17日成立，附屬國際獅子會總會中國港澳303區獅子區會之青年義工服務團體，現有40間分會，會員800多名。青獅會本著「我們服務」We Serve的宗旨，服務社會及有需要之人士，讓會員能從服務中認識自我，並對社會作出貢獻。

'''活動類型'''
#策劃社會服務工作
#籌募社會服務基金
#安排會員聯誼及康樂活動
#定期例會和年會
#舉行各項訓練活動和比賽
#參加國際青年交換計劃

'''入會資格'''
*任何人士凡年齡在12至30歲（申請入會時年齡須為25歲或以下）之間的青年朋友，有志參加社會服務工作，學習領導才能和發揮一己所長，均可申請加入。
