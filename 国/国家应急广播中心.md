'''国家应急广播中心'''，是[[中华人民共和国|中华人民共和国]]国家应急广播体系的中央机构，设在[[中央广播电视总台|中央广播电视总台]]。

==简介==
21世纪初，[[中共中央|中共中央]]和[[中华人民共和国国务院|国务院]]在一系列文件中提出，要建立统一联动、安全可靠的国家应急广播体系，同时将应急广播列入国家[[十二五规划|“十二五”规划]]。国务院在推进应急体系重点项目建设实施意见里提出，要由[[中华人民共和国国家新闻出版广电总局|国家新闻出版广电总局]]依托国家应急广播体系，建设预警信息自动适配播发系统，同时在应急预案里也有相关要求<ref name=jianshe>{{Cite web |url = http://news.cnr.cn/special/cebc/zb/wzzy/201409/t20140928_516519609.shtml |title = 王效杰：国家应急广播体系规划与建设 |author =  |date = 2014-09-28 |publisher = 央广网 |language =  |accessdate =  |quote =  }}</ref>。

2008年起，[[国家广播电影电视总局|国家广播电影电视总局]]组织相关单位启动了应急广播关键技术的研究攻关，并开始制定涵盖制作和播发、调度控制、传输覆盖、接收终端等环节的技术标准。2010年，经中央批准，启动制定国家应急广播方案。2011年，组织制定《国家应急广播工程建设方案》，并上报管理部门<ref name=jianshe/>。

2013年，[[国家新闻出版广电总局|国家新闻出版广电总局]]印发《推进国家应急广播体系建设工作方案》，并成立了'''国家应急广播中心'''<ref name=jianshe/><ref>{{Cite web |url = http://www.gapp.gov.cn/govpublic/4307/184749.shtml |title = 聂辰席听取《推进国家应急广播体系建设工作方案》汇报 |author =  |date = 2013-10-22 |publisher = 中华人民共和国国家新闻出版广电总局 |language =  |accessdate =  |quote =  }}</ref>。2013年12月3日，国家应急广播中心在[[北京市|北京市]]揭牌成立，设在[[中央人民广播电台|中央人民广播电台]]；同日，国家应急广播社区网站正式上线，目标是到2015年底前，国家实现各类灾害预警通过国家应急广播体系实时发布<ref>{{Cite web |url = http://china.cnr.cn/ygxw/201312/t20131203_514299587.shtml |title = 国家应急广播中心揭牌 将实现灾害预警实时发布 |author =  |date = 2013-12-03 |publisher = 央广网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.ce.cn/xwzx/gnsz/gdxw/201312/03/t20131203_1845569.shtml |title =  国家应急广播中心今日北京揭牌 灾害预警将实时发布|author =  |date =2013-12-03  |publisher = 中国经济网 |language =  |accessdate =  |quote =  }}</ref>。2014年，国家应急广播中心与相关部门建立日常信息传递机制，并组织在北京市等9个省、自治区、直辖市开展应急广播四级联动及村村响试点，还正在编制修订国家应急广播技术白皮书<ref name=jianshe/>。2014年9月，国家应急广播网独立域名www.cneb.gov.cn正式确立并上线，国家应急广播网由中央人民广播电台主办<ref>{{Cite web |url =http://www.cneb.gov.cn/2016/12/04/ARTI1480842689338465.shtml  |title =《国家应急广播》简介  |author =  |date = 2016-12-04  |publisher = 国家应急广播网 |language =  |accessdate =  |quote =  }}</ref>。

此后，逐步建立中央、省、市四级国家应急广播体制，确定各级广电部门承担应急广播信息发布职责。在中央人民广播电台建立集信息收集、处理、播报为一体的国家应急广播中心系统，在全国各级广电系统建立各级制作播发平台、调度控制平台，升级改造传输覆盖网，发布国家和当地预警信息<ref name=jianshe/>。

2013年4月22日，国家应急广播[[芦山地震|芦山抗震救灾]]应急电台在[[四川|四川]][[芦山|芦山]]开播，这是国家应急广播第一次开播的定向应急频率，是国家发生重大灾害时首次以“国家应急广播”为呼号对灾区民众定向播出的应急广播。芦山抗震救灾应急电台由中央人民广播电台、[[四川广播电视台|四川广播电视台]]、雅安人民广播电台、芦山县广播电视台联合开办，全天24小时滚动播出<ref>{{Cite web |url = http://politics.people.com.cn/n/2013/0423/c70731-21237299.html |title = 国家应急广播芦山抗震救灾应急电台开播 |author =  |date = 2013-04-23 |publisher =  人民网|language =  |accessdate =  |quote =  }}</ref>。

到2015年，中央人民广播电台已分别与西藏自治区新闻出版广电局、河北省新闻出版广电局、江苏省新闻出版广电局就共同开展应急广播体系建设签署合作框架协议。根据协议，中央人民广播电台将和这三个局分别组成工作组，联合当地广播电视播出机构（广播电台、电视台）及广播电视传输机构，开展应急广播系列试点、试验工作。中央人民广播电台将应急信息通过“国家应急广播预警信息适配系统”与省级应急广播试验平台共享，分别在三省区试验地区开展发布试验<ref>{{Cite web |url = http://www.sarft.gov.cn/art/2015/6/15/art_114_26982.html |title = 中央电台推进国家应急广播体系建设 |author =  |date = 2015-06-15 |publisher =中华人民共和国国家新闻出版广电总局  |language =  |accessdate =  |quote =  }}</ref>。

2014年9月16日，由国家新闻出版广电总局主办、中央人民广播电台承办的首届国家应急广播大会“2014中国应急广播大会” 在北京召开<ref>{{Cite web |url = http://expo.people.com.cn/n/2014/0917/c57922-25677163.html |title =  首届中国应急广播大会召开 |author =  |date = 2014-09-17 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。2015年11月10日，由国家新闻出版广电总局主办、中央人民广播电台承办的第二届国家应急广播大会“2015中国应急广播大会” 在北京召开，以“大数据、社交媒体与应急广播”为主题<ref>{{Cite web |url = http://www.cneb.gov.cn/2015/11/10/ARTI1447128235912971.shtml |title =2015中国应急广播大会在京举办 探索媒体融合下的应急广播发展之路  |author =  |date = 2015-11-10 |publisher =国家应急广播网  |language =  |accessdate =  |quote =  }}</ref>。

'''国家应急广播'''是国家应急广播中心与受灾地区的广播电台联合开办的广播频道，通常在灾区进行24小时不间断播放。

*2013年4月22日，“国家应急广播·芦山抗震救灾应急电台”开播，使用FM92.7MHz、SW9800kHz和12000kHz播出<ref>[http://news.xinhuanet.com/politics/2013-04/22/c_115493814.htm 国家应急广播首个定向应急频率开播] [[新华网|新华网]] 2013年04月22日</ref>。
*2014年8月7日，“国家应急广播·鲁甸抗震救灾应急电台”开播，使用FM98.6播出<ref>[http://china.cnr.cn/yaowen/201408/t20140807_516157584.shtml 国家应急广播鲁甸应急电台开播 专为灾区服务]，中国广播网</ref>。
*2014年10月8日，“国家应急广播·景谷抗震救灾应急电台”开播，使用FM100.5MHz播出<ref>[http://news.xinhuanet.com/zgjx/2014-10/16/c_133720481.htm “景谷应急电台” 震后18小时开播纪实 74小时不间断]，新华网</ref>。
