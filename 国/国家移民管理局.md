'''国家移民管理局'''（{{lang-en|National Immigration Administration}}<ref>{{cite web|url=http://www.mps.gov.cn/n2254996/index.html|title=国家移民管理局官网|accessdate=2019-01-14}}</ref>，縮寫{{lang|en|NIA}}），[[一个机构两块牌子|同时加挂]]'''中华人民共和国出入境管理局'''（{{lang-en|Entry and Exit Administration of the People's Republic of China}}）牌子，简称'''移民局（出入境管理局）'''，是[[中华人民共和国|中华人民共和国]][[中华人民共和国公安部|公安部]]管理的副部级[[国务院部委管理的国家局|国家局]]<ref name="官方简称">{{cite web|url=http://www.gdllw.org/tzgg/201806/W020180622515099385181.pdf|title=关于转发国务院机构简称的通知|accessdate=2018-11-25|author=国务院办公厅秘书局|date=2018-06-22|publisher=广东老龄网}}</ref>。

国家移民管理局主要负责协调拟定移民政策并组织实施，负责出入境管理、口岸证件查验和边民往来管理，负责外国人停留居留和永久居留管理、难民管理、国籍管理，牵头协调“三非”外国人治理和非法移民遣返，负责中国公民因私出入国（境）服务管理，承担移民领域国际合作等<ref>{{cite web|url=http://www.xinhuanet.com/politics/2018lh/2018-03/13/c_137035628.htm|title=王勇：组建国家移民管理局|accessdate=2018-03-13|date=2018-03-13|publisher=新华网}}</ref>。

== 历史 ==
2018年之前，[[中华人民共和国公安部|中华人民共和国公安部]]内设机构有'''[[公安部出入境管理局|公安部出入境管理局]]'''（公安部六局），主管公安出入境管理工作。各省、自治区、直辖市公安厅、局下设出入境管理局（处），负责本地区的公安出入境管理工作；绝大多数的地市和工作量大、工作任务比较重的县级公安机关也成立了出入境管理专门机构，具体执行出入境管理的各项工作任务。地方各级公安机关出入境管理部门在业务上接受上级公安机关出入境管理部门的指导，同时接受同级公安机关的领导。国家在全国对外开放的口岸设立边防检查站，执行出入境边防检查任务。北京、天津、上海、广州、深圳、珠海、厦门、海口、汕头9个城市成立了出入境边防检查总站，由公安部垂直领导<ref>{{Cite web|url=http://www.mps.gov.cn/n16/n84147/n84165/1291532.html|title=中国公安出入境管理机构设置与职责分工|accessdate=2018-03-24|date=2008-07-03|publisher=中华人民共和国公安部|archiveurl=https://web.archive.org/web/20150101074522/http://www.mps.gov.cn/n16/n84147/n84165/1291532.html|archivedate=2015-01-01}}</ref>。 

2018年3月17日[[第十三届全国人民代表大会#第一次会议|第十三届全国人民代表大会第一次会议]]通过《第十三届全国人民代表大会第一次会议关于国务院机构改革方案的决定》，批准《国务院机构改革方案》。方案规定：“组建'''国家移民管理局'''。将[[中华人民共和国公安部|公安部]]的出入境管理、边防检查职责整合，建立健全签证管理协调机制，组建国家移民管理局，加挂'''中华人民共和国出入境管理局'''牌子，由公安部管理。”<ref>{{Cite web|url=http://www.npc.gov.cn/npc/dbdhhy/13_1/2018-03/14/content_2048552.htm|title=关于国务院机构改革方案的说明——2018年3月13日在第十三届全国人民代表大会第一次会议上|accessdate=2018-03-13|author=[[王勇_(国务委员)|王勇]]|date=2018-03-13|publisher=中国人大网}}</ref><ref>{{Cite web|url=http://www.npc.gov.cn/npc/xinwen/2018-03/17/content_2049996.htm|title=第十三届全国人民代表大会第一次会议关于国务院机构改革方案的决定|accessdate=2018-03-17|date=2018-03-17|publisher=中国人大网}}</ref><ref>{{Cite web|url=http://www.npc.gov.cn/npc/xinwen/2018-03/18/content_2050371.htm|title=国务院机构改革方案|accessdate=2018-03-18|date=2018-03-18|publisher=中国人大网}}</ref>

2018年4月2日，国家移民管理局正式挂牌成立<ref>{{Cite web|url=http://www.mps.gov.cn/n2254996/n2254999/c6092805/content.html|title=国家移民管理局挂牌仪式举行|accessdate=2018-04-03|date=2018-04-03|publisher=国家移民管理局官网}}</ref>。

在此次机构改革中，原[[公安边防部队|公安边防部队]]各省级总队整体转隶并集体退出现役，调整组建省（区）出入境边防检查总站，加挂所在省（区）公安厅边境管理总队牌子，由国家移民管理局垂直管理<ref name="报道0419C">{{cite web|author1=内蒙古新闻网 |title=中华人民共和国内蒙古出入境边防检查总站挂牌仪式举行 |url=http://www.nmg.gov.cn/art/2019/1/3/art_1081_246419.html |website=内蒙古自治区人民政府门-{}-户网站 |accessdate=2019-04-19 |location=呼和浩特 |language=zh-cn}}</ref>。根据[[中央机构编制委员会办公室|中央编办]]批复，深圳边检总站等7个职业化边防检查总站规格由副厅级调整为正厅级，机构隶属关系和人员编制不变<ref name="报道0419A">{{cite web|author1=郑小红 |author2=尹萧 |title=深圳等七职业化边检总站升格正厅建制 编制不变 |url=http://m.sohu.com/n/336312182/ |website=搜狐网新闻 |accessdate=2019-04-19 |location=北京 |language=zh-cn}}</ref>。撤销汕头边检总站，下属单位划入深圳边检总站。改革后，国家移民管理局领导管理33个出入境边防检查总站<ref name="报道0419B">{{cite web|author1=韩莹雪 |author2=刘涵 |title=国家移民管理局招录计划 |url=https://mp.weixin.qq.com/s?src=11&timestamp=1555675212&ver=1556&signature=LkAP-v8TilUCYyWCnMBoA*72Nkx3p6lW*XjnUxmOoSAcdOG5eNaBb1PjIHUNteOiMeGL4ewolhwAy3DKkyS--5pq8XhkIFk*mLzGjb-GzrTwrSRriEcGdC*NocJy6-aE&new=1 |website=同济大学就业网 |accessdate=2019-04-19 |location=上海 |language=zh-cn}}</ref>。此外，国家移民管理局还组建常备力量第一、第二总队<ref name="报道0419B"/>。常备力量第一总队驻北京市朝阳区金盏乡楼梓庄东高路；常备力量第二总队驻云南省昆明市官渡区云大西路705号。

==职责==
根据《{{s|国家移民管理局职能配置、内设机构和人员编制规定}}》，国家移民管理局承担下列职能<ref name="报道0404A">{{cite web |author1=国家移民管理局 |title=国家移民管理局2019年部门预算 |url=http://www.mps.gov.cn/n2254996/n2254999/c6455870/content.html |website=国家移民管理局门-{}-户网站 |accessdate=2019-04-04 |location=北京 |language=zh-cn}}</ref>：
{{columns-list|2|
# 协调拟订移民政策并组织实施。 
# 负责出入境管理、口岸证件查验和边民往来管理。 
# 负责外国人停留居留和永久居留管理、难民管理、国籍管理。 
# 牵头协调非法入境、非法居留、非法就业外国人治理和非法移民遣返。 
# 负责中国公民因私出入国（境）服务管理。 
# 承担移民领域国际合作等。 
}}
