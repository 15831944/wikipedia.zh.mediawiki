'''国际地震中心'''（{{lang-en|International Seismological Centre}}）是一个于1964年在[[联合国教科文组织|联合国教科文组织]]的协助下成立的收集全球范围[[地震|地震]]观测数据的[[非政府组织|非政府组织]]机构，简称'''ISC'''，现任理事为D·A·斯托尔恰克<ref name=about>{{Cite web |url=http://www.isc.ac.uk/about/ |title=About the International Seismological Centre |publisher=国际地震中心 |language=en |accessdate=2018-04-21 |deadurl=no |archiveurl=https://web.archive.org/web/20171001210631/http://www.isc.ac.uk/about/ |archivedate=2017-10-01 }}</ref><ref>{{Cite web |mode= cs2 |first1= Robin |last1= Adams |date= 2010-03-25 |title= Notes on the history of the International Seismological Centre |url= http://www.isc.ac.uk/about/history/radams/ |language= en |publisher= 国际地震中心 |deadurl= no |archiveurl= https://web.archive.org/web/20180226091924/http://www.isc.ac.uk/about/history/radams/ |archivedate= 2018-02-26 }}</ref>。国际地震中心是全世界第一个收集全世界范围地震观测资料的组织<ref name=history /><ref>{{Cite book|title=不可不知的2000个科学常识|last=王晓梅著|first=|publisher=北京：[[中国长安出版社|中国长安出版社]]|year=2009.09|isbn=978-7-5107-0069-9|location=|pages=359}}</ref>。国际地震中心紧跟约翰·米尔恩教授和[[哈罗德·杰弗里斯|哈罗德·杰弗里斯]]爵士的初始工作，在世界范围内与其他地震数据中心和其他地震机构合作<ref>{{Cite journal|author1=白永福|author2=张伟|title=国际地震中心（ISC）|journal=国际地震动态|year=2017|issue=3|pages=9-14|language=zh-hans|url=http://www.wanfangdata.com.cn/details/detail.do?_type=perio&id=gjdzdt201703004|deadurl=no|archiveurl=https://web.archive.org/web/20180421131546/http://www.wanfangdata.com.cn/details/detail.do?_type=perio&id=gjdzdt201703004|archivedate=2018-04-21}}</ref>。对于[[地震学|地震学]]而言，国际地震中心被认为是世界上拥有最长期、稳定、完成和可靠的资料数据积累的机构<ref name=cz />。

== 历史沿革 ==
国际地震中心的前身是国际地震学总结（{{lang|en|International Seismological Summary}}）。1960年，[[联合国教科文组织|联合国教科文组织]]在[[法国|法国]][[巴黎|巴黎]]召开了关于讨论如何更好地利用ISS资料的研讨会，在本次会议中决定成立国际地震中心。1964年，国际地震中心成立于[[英国|英国]][[爱丁堡|爱丁堡]]<ref>{{Cite book|title=世界地学数据信息资源实用手册|last=[[蔡晋安|蔡晋安]]，[[卢振恒|卢振恒]]，赵旭等编著|first=|publisher=北京：[[中国科学技术出版社|中国科学技术出版社]]|year=2006.12|isbn=7-5046-4306-8|location=|pages=135}}</ref>。在此之后，国际地震中心共迁址2次。第一次于1975年搬迁至[[纽伯里|纽伯里]]，第二次于1986年搬迁至现址[[伯克郡|伯克郡]][[萨彻姆|萨彻姆]]<ref name=history>{{Cite web |url=http://www.isc.ac.uk/about/history/ |title=History of the ISC |language=en |accessdate=2018-04-21 |publisher=国际地震中心 |deadurl=no |archiveurl=https://web.archive.org/web/20170911115009/http://www.isc.ac.uk/about/history/ |archivedate=2017-09-11 }}</ref>。

== 研究范围 ==
全世界有逾千个[[地震台|地震台]]的资料都定期送至国际地震中心进行地震资料的汇集、分析和出版。目前国际地震中心的主要任务，是收集全世界范围内的地震信息，其中包括仪器记录数据和宏观地震资料，从而建立完成的资料档案，并对相关地震问题提供咨询服务<ref name=cz>{{Cite journal |author1=陈颙 |author2=张伯明 |title=国际地震中心（ISC）简介 |journal=地震地磁观测与研究 |year=1985 |issue=5 |pages=37-43 |language=zh-hans |accessdate=2018-04-21 |url=http://kns.cnki.net/KCMS/detail/detail.aspx?filename=dzgj198505003&dbname=CJFD&dbcode=CJFQ |deadurl=no |archiveurl=https://web.archive.org/web/20180421131546/http://kns.cnki.net/KCMS/detail/detail.aspx?filename=dzgj198505003&dbname=CJFD&dbcode=CJFQ |archivedate=2018-04-21 }}</ref><ref>{{Cite book|title=地震数据信息技术|last=[[中国地震局|国家地震局]]科技监测司编|first=|publisher=北京：[[地震出版社|地震出版社]]|year=1995.11|isbn=7-5028-1203-2|location=|pages=252}}</ref>。在收集到各地上传的地震事件的震源位置、震相数据和宏观地震资料后，国际地震中心会在地震发生后两年之内以月刊和半年一次目录的形式出版<ref>{{Cite journal|author1=休斯|author2=赵仲和|title=国际地震中心的发展简史及其在国际地震资料交换中的作用|journal=国际地震动态|year=1985|issue=4|pages=20-22|language=zh-hans|accessdate=2018-04-21|url=http://kns.cnki.net/KCMS/detail/detail.aspx?filename=gjzt198504006&dbname=CJFD&dbcode=CJFQ|deadurl=no|archiveurl=https://web.archive.org/web/20180421135432/http://kns.cnki.net/KCMS/detail/detail.aspx?filename=gjzt198504006&dbname=CJFD&dbcode=CJFQ|archivedate=2018-04-21}}</ref>。

国际地震中心拥有5项数据产品，即国际地震中心期刊<ref>{{Cite journal |url=http://www.isc.ac.uk/iscbulletin/ |title=Bulletin of the International Seismological Centre |accessdate=2018-04-21 |language=en |publisher=国际地震中心 |deadurl=no |archiveurl=https://web.archive.org/web/20180228161608/http://www.isc.ac.uk/iscbulletin/ |archivedate=2018-02-28 }}</ref>、国际地震中心EHB期刊<ref>{{Cite journal |url=http://www.isc.ac.uk/ehbbulletin/ |title=EHB Bulletin |accessdate=2018-04-21 |language=en |publisher=国际地震中心 |deadurl=no |archiveurl=https://web.archive.org/web/20180228161603/http://www.isc.ac.uk/ehbbulletin/ |archivedate=2018-02-28 }}</ref>、国际地震中心全球仪器地震目录<ref>{{Cite web |url=http://www.isc.ac.uk/iscgem/ |accessdate=2018-04-21 |language=en |title=ISC-GEM Catalogue - Introduction |publisher=国际地震中心 |deadurl=no |archiveurl=https://web.archive.org/web/20180228161605/http://www.isc.ac.uk/iscgem/ |archivedate=2018-02-28 }}</ref>、[[国际地震学与地球内部物理学联合会|国际地震学与地球内部物理学联合会]]地面实况参考事件<ref>{{Cite web |url=http://www.isc.ac.uk/gtevents/ |title=IASPEI Ground Truth (GT) reference events |accessdate=2018-04-21 |language=en |publisher=国际地震中心 |deadurl=no |archiveurl=https://web.archive.org/web/20180228161536/http://www.isc.ac.uk/gtevents/ |archivedate=2018-02-28 }}</ref>和国际地震中心事件目录<ref>{{Cite web |url=http://www.isc.ac.uk/event_bibliography/index.php/ |title=Motivation for the ISC Event Bibliography |accessdate=2018-04-21 |language=en |publisher=国际地震中心 |deadurl=no |archiveurl=https://web.archive.org/web/20180302164038/http://www.isc.ac.uk/event_bibliography/index.php/ |archivedate=2018-03-02 }}</ref>。
