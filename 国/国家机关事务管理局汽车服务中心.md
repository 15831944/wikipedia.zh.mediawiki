'''国家机关事务管理局汽车服务中心'''（'''国兴汽车服务中心'''），是[[国家机关事务管理局|国家机关事务管理局]]直属[[事业单位|事业单位]]。

==沿革==
1992年，为适应机关后勤体制改革发展的需要，[[国务院机关事务管理局|国务院机关事务管理局]]党组决定将[[首都宾馆|首都宾馆]]汽车队与财务司汽车服务中心合并，成立“'''国务院机关事务管理局汽车服务中心'''”，为国务院机关事务管理局属自收自支[[事业单位|事业单位]]。1993年在[[国家工商行政管理局|国家工商行政管理局]]登记注册（1994年更名“'''国兴汽车服务中心'''”），注册资金2300万元人民币。

2013年，[[国务院机关事务管理局|国务院机关事务管理局]]更名[[国家机关事务管理局|国家机关事务管理局]]，国务院机关事务管理局汽车服务中心随之更名'''国家机关事务管理局汽车服务中心'''<ref>{{Cite web |url =http://www.gjsy.gov.cn/djgg/bggg/201404/t20140411_2662.html  |title =  国家事业单位登记管理局公告|author =  |date = 2014-04-11 |publisher =事业单位在线  |language =  |accessdate =  |quote =  }}</ref>。该中心负责为中央国家机关提供用车服务<ref name=zaixian/>。
