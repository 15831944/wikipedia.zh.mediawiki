'''国家气候中心'''（{{lang-en|National Climate Center}}），于1995年1月10日创立于[[北京|北京]]，隶属于[[中国气象局|中国气象局]]，为独立的[[司局级|司局级]]法人[[事业单位|事业单位]]，是中国气象基本业务系统的重要组成部分，主要承担着建设国家级气候和气候变化业务系统的任务。它是世界气象组织东亚季风活动中心、全球长期预报产品中心，是[[世界气象组织|世界气象组织]]亚洲区域气候中心、[[區域專責氣象中心|區域專責氣象中心]]，也是世界上少数几个能利用气候预测模式发布预测产品的机构。

== 机构沿革 ==
1994年2月,[[中华人民共和国国务院|中华人民共和国国务院]]批准组建国家气候中心，为[[中国气象局|中国气象局]]直属正司局级[[事业单位|事业单位]]<ref>国办通[1994]10号</ref>。经过近一年的筹备，1995年1月10日由时任[[国务委员|国务委员]][[宋健|宋健]]正式授牌成立。迄今，其发展可大致划为三个阶段：

第一阶段：1995年1月至2001年9月；

第二阶段：2001年9月至2004年4月；

第三阶段：2004年4月至今。<ref>{{cite web |url = http://ncc.cma.gov.cn/Website/index.php?ChannelID=14&WCHID=1 |title = 国家气候中心历史沿革 |accessdate = 2009年1月10日 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 机构编制 ==
国家气候中心职工编制人数200人，现有职工141人，其中[[中国工程院院士|中国工程院院士]]1人，[[研究员|研究员]]（正研级高工）23人、副研究员（高级工程师）55人，博士（后）77人、硕士27人。
