'''國家新聞出版署'''，简称'''新聞出版署'''，是[[中共中央宣傳部|中共中央宣傳部]]對外加掛的牌子，同時加掛'''[[國家版權局|國家版權局]]'''牌子<ref name="官方简称">{{cite web |author1=国务院办公厅秘书局 |title=关于转发国务院机构简称的通知 |url=http://www.gdllw.org/tzgg/201806/W020180622515099385181.pdf |website=广东老龄网 |accessdate=2018-11-25 |location=北京 |language=zh-cn}}</ref>。

國家新聞出版署主要负责貫徹落實黨的宣傳工作方針，擬定新聞出版業管理政策並督促落實，管理新聞出版行政事務，監督管理出版物內容和質量，管理出版物進口等。

== 历史沿革 ==
2018年3月，[[中華人民共和國國務院|中華人民共和國國務院]]進行機構改革，撤銷正部級[[國務院直屬機構|直屬機構]][[国家新闻出版广电总局|国家新闻出版广电总局]]，在其广播电视管理职责的基础上组建[[国家广播电视总局|国家广播电视总局]]<ref>{{Cite web |url =  http://www.npc.gov.cn/npc/dbdhhy/13_1/2018-03/14/content_2048552.htm|title = 关于国务院机构改革方案的说明——2018年3月13日在第十三届全国人民代表大会第一次会议上 |author =王勇  |date = 2018-03-13 |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.npc.gov.cn/npc/xinwen/2018-03/17/content_2049996.htm|title =  第十三届全国人民代表大会第一次会议关于国务院机构改革方案的决定|author =  |date =2018-03-17  |publisher =  中国人大网|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://www.npc.gov.cn/npc/xinwen/2018-03/18/content_2050371.htm|title =  国务院机构改革方案|author =  |date = 2018-03-17  |publisher = 中国人大网 |language =  |accessdate =  |quote =  }}</ref>。原[[国家新闻出版广电总局|国家新闻出版广电总局]]的新聞出版管理職責被劃歸中宣部，後者加掛國家新聞出版署（國家版權局）牌子<ref>{{cite news|title=三台合組「中國之聲」 加強對外宣傳力量|date=2018-03-21|accessdate=2018-03-21|url=http://std.stheadline.com/daily/news-content.php?id=1769654&target=2|work=星島日報|language=zh-hk}}</ref>。

2018年3月22日，国务院下发《[[s:国务院关于机构设置的通知_(2018年)|国务院关于机构设置的通知]]》（国发[2018]6号），在“国务院直属机构”一节中，明确國家新闻出版署（国家版权局）在中共中央宣传部加挂牌子，由[[中共中央宣传部|中共中央宣传部]]承担相关职责。<ref>{{Cite web|url=http://www.gov.cn/zhengce/content/2018-03/24/content_5277121.htm|title=国务院关于机构设置的通知（国发〔2018〕6号）|accessdate=2018-03-25|author=|date=|work=|publisher=中华人民共和国国务院}}</ref>

2018年4月16日，国家新闻出版署正式挂牌<ref>{{Cite web |url = http://www.xinhuanet.com/politics/2018-04/16/c_1122688002.htm|title = 国家新闻出版署（国家版权局）、国家电影局揭牌 |author =  |date = 2018-04-16  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。
