'''国家级临空经济示范区'''（{{lang-en|'''National Airport Economic Demonstration Zone'''}}，简称'''临空经济示范区'''），包括航空港经济综合试验区、航空经济示范区等类型，由[[中华人民共和国国务院|中华人民共和国国务院]]提出、[[国家发展和改革委员会|国家发展和改革委员会]]和[[中国民用航空局|民航局]]组织批准设立，是指通过向指定航空枢纽及邻近特定区域给予特殊政策，探索以航空枢纽为依托的航空运输业、高端制造业和现代服务业集聚发展而设立的临空经济试验区域。相关政策试点涵盖了民航管理模式、海关监管模式、金融支持政策、土地管理政策、服务外包政策等多个方面<ref name="zhengzhou1">{{Cite web |url=http://www.ndrc.gov.cn/zcfb/zcfbghwb/201304/W020140221372408841861.pdf |title=郑州航空港经济综合实验区发展规划（2013―2025年） |accessdate=2018-07-20 |author=国家发展改革委 |date=2013-03-08 |archive-url=https://web.archive.org/web/20171203101142/http://www.ndrc.gov.cn/zcfb/zcfbghwb/201304/W020140221372408841861.pdf |archive-date=2017-12-03 |dead-url=no }}</ref>。

2013年3月8日，《郑州航空港经济综合实验区发展规划正式（2013-2025年）》由国家发改委批复，[[郑州|郑州]]航空港经济综合实验区成为首个国家级临空经济示范区<ref name="zhengzhou2">{{Cite web |url=http://www.ndrc.gov.cn/zcfb/zcfbghwb/201304/t20130422_588370.html |title=国家发展改革委关于印发郑州航空港经济综合实验区发展规划（2013―2025年）的通知(发改地区[2013]481号) |accessdate=2018-07-20 |author=国家发展改革委 |date=2013-03-08 |archive-url=https://web.archive.org/web/20171203101140/http://www.ndrc.gov.cn/zcfb/zcfbghwb/201304/t20130422_588370.html |archive-date=2017-12-03 |dead-url=no }}</ref>。截至2017年5月，国家发改委先后批准设立了12个国家级临空经济示范区，依次包括郑州、[[北京|北京]]、[[青岛|青岛]]、[[重庆|重庆]]、[[广州|广州]]、[[上海|上海]]、[[成都|成都]]、[[长沙|长沙]]、[[贵阳|贵阳]]、[[杭州|杭州]]、[[宁波|宁波]]和[[西安|西安]]<ref>{{Cite web |url=https://www.yicai.com/news/5420483.html |title=临空经济示范区增至12个 内陆省会成对外开放新高地 |accessdate=2018-07-20 |author=林小昭 |date=2018-05-05 |publisher=第一财经 |archive-url=https://web.archive.org/web/20180726001214/https://www.yicai.com/news/5420483.html |archive-date=2018-07-26 |dead-url=no }}</ref>。

==设立背景==
2012年7月12日，[[中华人民共和国国务院|国务院]]印发《国务院关于促进民航业发展的若干意见》，意见中提出要在目前存在域资源配置不合理、基础设施发展较慢等情况的背景下，选择部分地区开展航空经济示范区试点<ref name="state">{{Cite web |url=http://www.gov.cn/zhengce/content/2012-07/12/content_3228.htm |title=国务院关于促进民航业发展的若干意见（国发[2012]24号） |accessdate=2018-07-20 |author=国务院 |date=2012-07-12 |archive-url=https://web.archive.org/web/20150429183710/http://www.gov.cn/zhengce/content/2012-07/12/content_3228.htm |archive-date=2015-04-29 |dead-url=no }}</ref>。随后于2013年3月，首个示范区郑州航空港经济综合试验区获得国务院批复成立<ref name="zhengzhou2"/>，试验区提出要发展航空物流和与航空相关的高端制造业<ref>{{Cite web |url=http://www.ndrc.gov.cn/fzgggz/tzgg/ggkx/201403/t20140312_725587.html |title=郑州获批建设国内首个航空港经济综合实验区 |accessdate=2018-07-20 |author=国家发展改革委经济体制综合改革司 |date=2013-03-25 |publisher=国家发展改革委 |archive-url=https://web.archive.org/web/20180726001213/http://www.ndrc.gov.cn/fzgggz/tzgg/ggkx/201403/t20140312_725587.html |archive-date=2018-07-26 |dead-url=no }}</ref>。2015年，为进一步推开试点，国家发展改革委和民航局联合发布指导意见，意见正式将示范区定名为临空经济示范区，并具体列明了示范区的申报条件和程序等信息<ref name="ndrc1">{{Cite web |url=http://www.ndrc.gov.cn/gzdt/201507/t20150703_710711.html |title=国家发展改革委 民航局关于临空经济示范区建设发展的指导意见（发改地区[2015]1473号） |accessdate=2018-07-20 |author=国家发展改革委 |date=2015-06-24 |archive-url=https://web.archive.org/web/20180726001214/http://www.ndrc.gov.cn/gzdt/201507/t20150703_710711.html |archive-date=2018-07-26 |dead-url=no }}</ref>。此后，国家发改委分多个批次陆续批准了11个新的示范区。
