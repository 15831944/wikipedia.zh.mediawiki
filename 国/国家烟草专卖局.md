'''国家烟草专卖局'''（{{lang-en|State Tobacco Monopoly Administration}}），简称'''国家烟草局'''，成立于1984年，与'''中国烟草总公司'''[[合署办公|合署办公]]，对[[中国内地|中国内地]][[烟草业|烟草业]]实行统一领导、垂直管理、专卖专营的管理体制，对全行业“人、财、物、产、供、销、内、外、贸”进行集中统一管理。

中国烟草总公司轄下子公司的煙草總產量達到世界第一，為全球最大的煙草業營運機構。

== 沿革 ==
1981年，[[中华人民共和国国务院|国务院]]決定實行煙草[[專營|專營]]，成立'''中国烟草总公司'''作爲負責機構。1982年1月，中国烟草总公司正式挂牌成立。1983年9月，国务院发布《[[烟草专卖条例|烟草专卖条例]]》，正式确立了国家[[烟草专卖|烟草专卖]]制度。

1984年1月，国家烟草专卖局成立，與中國煙草總公司實行[[一个机构两块牌子|一個機構兩塊牌子]]，屬[[中华人民共和国轻工业部|輕工業部]]管理，[[李益三|李益三]]任局長、經理。1991年，[[第七届全国人民代表大会常务委员会|第七届全国人民代表大会常务委员会]]第二十次会议通過《中华人民共和国烟草专卖法》。1993年，根據[[第八届全国人民代表大会第一次会议|第八届全国人民代表大会第一次会议]]的決議，国家烟草专卖局劃歸[[中華人民共和國國家經濟和貿易委員會|國家經貿委]]管理。

2003年，[[第十届全国人民代表大会#第一次会议|第十届全国人民代表大会第一次会议]]决定组建[[中华人民共和国商务部|商务部]]，将国家经贸委的内贸管理、对外经济协调和重要工业品/原材料进出口计划组织实施、[[中华人民共和国国家计划委员会|国家计委]]的农产品进出口计划组织实施、[[中华人民共和国对外经济贸易部|外经贸部]]的职能整合划入商務部，国家烟草专卖局改由[[中华人民共和国国家发展和改革委员会|国家发改委]]管理。

2008年3月15日，[[第十一届全国人民代表大会第一次会议|第十一届全国人民代表大会第一次会议]]通过《第十一届全国人民代表大会第一次会议关于国务院机构改革方案的决定》，批准《国务院机构改革方案》。方案规定：“组建[[中华人民共和国工业和信息化部|工业和信息化部]]。将[[中华人民共和国国家发展和改革委员会|国家发展和改革委员会]]的工业行业管理有关职责，[[中华人民共和国国防科学技术工业委员会|国防科学技术工业委员会]]核电管理以外的职责，[[中华人民共和国信息产业部|信息产业部]]和[[国务院信息化工作办公室|国务院信息化工作办公室]]的职责，整合划入工业和信息化部。组建[[国家国防科技工业局|国家国防科技工业局]]，由工业和信息化部管理。国家烟草专卖局改由工业和信息化部管理。不再保留国防科学技术工业委员会、信息产业部、国务院信息化工作办公室。”<ref>{{Cite web |url =http://www.npc.gov.cn/npc/xinwen/syxw/2008-03/12/content_1413911.htm  |title = 关于国务院机构改革方案的说明——2008年3月11日在第十一届全国人民代表大会第一次会议上 |author =华建敏  |date =2008-03-11  |publisher =中国人大网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://www.gov.cn/2008lh/content_921411.htm |title = 两会授权发布：国务院机构改革方案 |author =  |date = 2008-03-15 |publisher = 中国政府网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.gov.cn/2008lh/content_921283.htm  |title = v |author =  |date =  2008-03-15 |publisher = 中国政府网 |language =  |accessdate =  |quote =  }}</ref>

2010年，中國煙草總公司实现[[淨利潤|淨利潤]][[人民幣|人民幣]]1177亿元，平均每天净赚人民幣3.2亿元，赚钱能力超过[[中国石油|中国石油]]<ref>{{cite web |url = http://news.163.com/12/0306/16/7RU7ROR00001124J.html | language= | title=中国烟草总公司首曝巨额净利 赚钱能力超中石油 |publisher=中国网 |date=2012-03-06 |accessdate= }}</ref>。2012年，中国烟草总公司实现利润总额2437.6亿元，实现净利润1786.9亿元，分别增长15.0%和14.5%。2013年，中国烟草总公司实现利润总额2749.2亿元，实现净利润2019亿元，上缴国有资本收益295.68亿元，上缴专项税后利润400亿元，合计约占中央国有资本经营收入的47.7%；全年上缴财政总额8161.22亿元，同比增长13.9%。2014年，烟草行业工商税利达到10517.6亿元，同比增加957.7亿元，增长10.02%；全年上缴财政总额9110.3亿元，同比增加949.1亿元，增长11.63%。2018年，中國煙草總公司實現工商稅利總額11556億元，同比增長3.69%；上繳國家財政總額10000.8億元，同比增長3.37%；實現工業增加值7877億元，同比增長4.88%。

== 职责 ==
根据《{{s|国家烟草专卖局职能配置、内设机构和人员编制规定}}》，国家烟草专卖局承担下列职能<ref name="三定方案"/>：
{{columns-list|2|
# 拟定烟草行业发展战略、规划和政策，调整产业结构，促进行业合理布局。
# 依法实施烟草专卖管理，查处违法违规案件，依法查禁和关停计划外烟厂，保护合法经营。
# 组织贯彻执行国家财务会计制度，依法对所属的国有资产行使出资人权利，经营和管理国有资产，承担保值增值责任，管理监督烟草行业财务资金，组织实施烟草行业内部审计工作。
# 组织烟草行业生产、经营和对外经济技术合作工作，拟定并组织实施烟草专卖品产供销及进出口的年度计划，编制烟草行业固定资产投资规划，审批烟草制品生产企业固定资产投资项目，指导行业安全生产工作，依法处理重大安全事故。
# 提出烟草行业有关经济政策建议，制定烟草专卖品的价格政策，管理烟草专卖品的价格。
# 负责组织实施烟草行业体制改革，指导烟草企业改革，推进卷烟工业企业联合重组，培育具有国际竞争力的大企业，依法审批烟草企业的设立、分立、合并、撤销。
# 负责烟草制品减害降焦工作，制定烟草行业科技发展政策，组织实施技术创新和重大科研项目攻关及科技成果应用推广工作，负责烟草行业技术监督、质量管理工作，组织实施行业标准化工作。
# 对全国烟草行业实行统一领导、垂直管理、专卖专营。
# 管理烟草系统人事、劳动工资工作。
# 承办国务院及工业和信息化部交办的其他事项。
}}
