'''国务院、中央军委空中交通管制委员会'''，简称'''国家空管委'''，是[[中华人民共和国国务院|中华人民共和国国务院]]和[[中央军事委员会|中央军事委员会]]间的[[国务院议事协调机构|议事协调机构]]，是[[中华人民共和国|中华人民共和国]]全国空域管制的最高机构。

==沿革==
1950年11月1日，[[毛泽东|毛泽东]]签发了中华人民共和国第一个飞行基本规则——《中华人民共和国飞行基本规则》，这是规范中华人民共和国空中交通活动的基本法规。1957年10月5日，[[周恩来|周恩来]]总理在民航局关于中缅航线通航情况报告上批示：“保证安全第一，改善服务工作，争取飞行正常。”这成为中华人民共和国空管工作的指导方针。1973年底，周恩来总理批准引进中华人民共和国第一套自动化空管系统<ref name=shui/>。

1986年1月30日，中央军委主席[[邓小平|邓小平]]批准成立国务院中央军委空中交通管制委员会，由一名[[中华人民共和国国务院副总理|国务院副总理]]兼任国家空管委主任，負責空中交通管制工作<ref name=makai>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1280433 |title = 马凯兼任国家空管委主任，已至少身兼两主任和两组长职务|publisher=澎湃新闻|date=2014-11-24}}</ref><ref name=shui>{{Cite web |url = http://news.xinhuanet.com/newscenter/2004-05/29/content_1497464.htm |title = 谁持彩练当空舞━━我国空中交通管制发展历程 |author =  |date = 2004-05-29 |publisher =  新华网|language =  |accessdate =  |quote =  }}</ref>。1986年2月24日，[[民航总局|民航总局]]转发《国务院、中央军委关于改革国内空中交通管制体制逐步实现空中交通管制现代化的通知》<ref>{{Cite web |url = http://www.shtong.gov.cn/Newsite/node2/node2245/node68464/node68480/node68523/userobject1ai66240.html |title = 上海民用航空志 第十二章空中交通管制 第一节　管制体制及机构 |author =  |date =  2003-12-22|publisher = 上海市地方志办公室 |language =  |accessdate =  |quote =  }}</ref>。1986年8月20日，[[国务院办公厅|国务院办公厅]]、[[中央军委办公厅|中央军委办公厅]]联合发出《国务院办公厅、中央军委办公厅关于空中交通管制委员会组成人员的通知》（国办发〔1986〕63号）<ref name=zucheng>{{Cite web |url = http://www.gov.cn/zhengce/content/2013-09/09/content_7388.htm |title =国务院办公厅、中央军委办公厅关于空中交通管制委员会组成人员的通知（国办发〔1986〕63号）  |author =  |date = 2013-09-09 |publisher =中国政府网  |language =  |accessdate =  |quote =  }}</ref>。国家空管委的实际工作由[[中国人民解放军总参谋部|中国人民解放军总参谋部]]负责。国家空中交通管制委员会办公室设在[[中国人民解放军总参谋部作战部|中国人民解放军总参谋部作战部]]<ref>{{Cite web |url = http://news.163.com/air/13/0805/11/95GSBCBL00014P23.html |title = 民航空域超负 流量控制致航班恶性延误|date=2013-08-05|publisher=网易}}</ref>。2016年[[深化国防和军队改革|深化国防和军队改革]]中，中国人民解放军总参谋部撤销，国家空中交通管制委员会办公室改设在[[中央军事委员会联合参谋部作战局|中央军委联合参谋部作战局]]。

1994年10月1日[[江泽民|江泽民]]题词：“保证飞行安全，提高服务质量，为我国改革开放和经济建设服务。”<ref>{{Cite web |url =http://news.xinhuanet.com/politics/2013-11/30/c_125787077_5.htm|title = 新中国民航60周年大事记|publisher=新华网|date=2013-11-30}}</ref><ref name=shui/>

2003年9月，国务院中央军委空中交通管制委员会对空管工作提出了新要求，為了适应经济發展和国防安全要求的空管体制，进一步开发利用空域资源，满足经济建设、国防建设和社会发展的需求<ref name=shui/>。

2005年4月，[[中华人民共和国公安部|公安部]]加入国家空管委，继军航、民航、航空制造业、航空体育之后成为中国第五家加入国家空管委的成员单位<ref>中国警航雏鹰凌空，人民公安2007年第12期</ref>。

==办事机构==
国务院、中央军委空中交通管制委员会的实际工作由[[中央军委联合参谋部|中央军委联合参谋部]]承担。'''国务院、中央军委空中交通管制委员会办公室'''设在[[中央军委联合参谋部作战局|中央军委联合参谋部作战局]]。

== 空管制度 ==
根据《中华人民共和国飞行基本规则》<ref>{{Cite web|url = http://www.caac.gov.cn/b1/b5/200612/t20061218_790.html|title = 中华人民共和国飞行基本规则|deadurl = yes|archiveurl = https://web.archive.org/web/20150915050229/http://www.caac.gov.cn/B1/B5/200612/t20061218_790.html|archivedate = 2015-09-15}}</ref>，国家空管委協調中国大陸飞行管制工作，大陸境内的飞行管制由[[解放军空军|解放军空军]]实施。

[[民航|民航]]班机日常运行过程中改航、绕航均须得到空军管制部门同意。

军民航协调机制是国家空管委組織下由[[国务院组成部门|国务院各部委]]及[[中国人民解放军军兵种列表|解放军各军兵种]]组成。

预战术层面是地区空管协调委领导下由地区军民航各单位组成，办公室设在军区空军航行处；战术层面由军民航相对应的管制运行单位组成。

=== 审批权限 ===
* 航路航线开通或者调整由[[中国民航局|中国民航局]]按照职责报总参或者空军审批，
* 空中禁区、危险区、限制区由空军报总参审批，
* 各类军事训练空域由空军或者军区空军审批。

== 空域概况 ==
* 中国大陆的[[領空|領空]]划分为11个[[飞行情报区|飞行情报区]]，总面积约1081万[[平方公里|平方公里]]<ref>{{Cite web |url = http://news.163.com/special/00012Q9L/flight201008012.html |title = 航班晚点：不能全怪航空公司 }}</ref>。
* 目前中国大陸划设19个高空管制区，28个中低空管制区，25个进近管制区，1个终端管制区<ref>{{Cite web |url = http://hk.crntt.com/doc/1018/0/0/7/101800723.html?coluid=7&kindid=0&docid=101800723 |title = 航班延誤乃世界難題　空中“塞飛機”非奇談 }}</ref>。
* 航路航线总距离约16.4万[[公里|公里]]，其中，国际航路航线约占46.2%，临时航线约占16.2%<ref>{{Cite web |url = http://finance.sina.com.cn/roll/20120803/022112746398.shtml |title = 临时航线主动释放：空域权管理改革挪步 }}</ref>。
* 解放軍划设了2个空中禁区、66个空中危险区、199个空中限制区，以及若干个军事训练空域。部分训练空域在航线上方或下方，通过建立高度差的办法避免军民航飞行冲突<ref>{{Cite web |url =http://zhenhua.163.com/14/0725/08/A201IR9R000465B7.html |title =航班延误:飞机多了路没加宽 |access-date =2015-09-28 |archive-url =https://web.archive.org/web/20150929001812/http://zhenhua.163.com/14/0725/08/A201IR9R000465B7.html |archive-date =2015-09-29 |dead-url =yes }}</ref>。
* 据[[中国民航局|中国民航局]]局长[[李家祥|李家祥]]介绍，留给[[民航|民航]]使用的只有大约20%的空域，其他為[[軍機|軍機]]使用<ref>{{Cite web |url = http://www.bbc.com/zhongwen/trad/china/2015/05/150527_china_flight_delays |title = 中國民航航班去年凖點率不足七成 }}</ref>。而[[美國|美國]]有90%的空域給[[民航機|民航機]]使用。
