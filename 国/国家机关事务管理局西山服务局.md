'''国家机关事务管理局西山服务局'''，是[[国家机关事务管理局|国家机关事务管理局]]直属[[事业单位|事业单位]]。

==沿革==
前身为1960年代组建的万寿山休养所。1998年10月，'''国家机关事务管理局西山服务局'''成立，是[[国家机关事务管理局|国家机关事务管理局]]所属事业单位，主要承担领导同志及中央国家机关各部门召开的各类型会议的接待服务，对外承接各类会议的接待服务<ref>{{Cite web |url = http://career.bnu.edu.cn/front/zpxx.jspa?channelId=764&tid=107509 |title =国家机关事务管理局西山服务局  |author =  |date =  |publisher =  北京师范大学|language =  |accessdate = 2018-08-17 |quote =  }}</ref><ref>营建秀美山水 创建文明单位 构建和谐西山——记国务院机关事务管理局西山服务局，中国机关后勤2006(7)</ref>。

==职责==
#负责规定范围内党和国家领导人以及国务院指定的其他领导同志在[[北京西山|西山]]办公、休息的接待服务工作。
#负责各部门在西山组织召开各种会议的接待服务工作。
#在完成领导同志接待服务任务的基础上，充分利用设施设备，依法组织经营。
#负责物资、设备的购置配备和固定资产管理。
#负责西山办公区房屋的基建、修缮和管理。
#负责西山办公区内消防安全、卫生、绿化美化工作。
#负责职工队伍的管理和日常工作的组织。
#承办局领导交办的其他事项。
