== 識字率 ==
估計全球共有超過7.81億成人沒有基本識字能力，其中三分之二是女性。此外，將近1.03億孩童無法上學，進而無法學習閱讀、書寫和計算。

根據聯合國教科文組織的《2006年全球教育報告》<ref>''2006 Education for All - Global Monitoring Report - Literacy for Life'' - 參見：{{cite web |url=http://www.uis.unesco.org/ev.php?URL_ID=5204&URL_DO=DO_TOPIC&URL_SECTION=201 |title=存档副本 |accessdate=2007-02-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20110610232025/http://www.uis.unesco.org/ev.php?URL_ID=5204&URL_DO=DO_TOPIC&URL_SECTION=201 |archivedate=2011-06-10 }}[http://portal.unesco.org/education/en/ev.php-URL_ID=43009&URL_DO=DO_TOPIC&URL_SECTION=201.html]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[南亞|南亞]]和[[西亞|西亞]]是成人識字率最低的地區（58.6%），倒數二、三名分別是[[撒哈拉以南非洲|撒哈拉非洲地區]]（59.7%）和[[阿拉伯世界|阿拉伯世界]]（62.7%）。全世界識字率最低的國家是[[布吉納法索|布吉納法索]]（12.8%）、[[尼日尔|尼日尔]]（14.4%）和[[马里共和国|马里共和国]]（19%）。

[[聯合國|聯合國]]對「識字」的定義為能閱讀和書寫簡單句子的能力，任何語言皆可。因此以上的識字率僅為基本、而非進階的識字能力。

== 識讀日主題 ==
* 2006年国际扫盲日的40周年慶主题是「识字保持社会发展」（{{lang-en|Literacy sustains development}}）。
* 2016年国际扫盲日邁入50周年慶，主題為「{{lang-en|Reading the Past, Writing the Future}}」（閱讀過去,書寫未來）。

== 響應機構 ==
* 英國[[博愛精神|慈善]]機構「{{tsl|en|National Literacy Trust|國家讀寫素養基金會}}」在2015年配合國際識讀日的「讀寫素養和永續社會」主題，分別和政府、媒體等機構合作，廣泛報導社群的識讀服務與活動。

* 歐洲讀寫政策網路（{{lang-en|The European Literacy Policy Network}}）則是從國際識讀日9月8日起推動「識讀週」（{{lang-en|Literacy Week}}），全歐洲各地不同國家、文化、語言維持一週的閱讀活動，到9月17日閉幕。

* 英國高中圖書館響應國際識讀日將9月定為識讀月，展開為期一個月的閱讀活動。<ref name = "讀不讀, 有關係: 從「國際識讀日」談起_2018" />
