'''国家邮政局'''，是由[[中华人民共和国交通运输部|中华人民共和国交通运输部]]管理的副部级[[国务院部委管理的国家局|国家局]]，负责[[郵件|邮政]][[行政管理|行政管理]]。

国家邮政局负责拟订邮政行业政策和规划，以及承担邮政（含[[速遞|快递]]公司）监管责任。

== 沿革 ==
1948年，中央决定设立[[华北人民政府交通部邮电总局|华北人民政府交通部邮电总局]]，主管邮电事务。1949年，华北人民政府调整组建[[华北人民政府交通部邮政总局|华北人民政府交通部邮政总局]]，为[[华北行政区|华北行政区]]邮政主管部门。

1949年11月1日，[[中央人民政府郵電部|中央人民政府郵電部]]成立，同年12月郵電部召開全國郵政會議，將原先[[中華民國交通部郵政總局|中華民國交通部郵政總局]]的產業予以接收，划归为中国人民邮政所有；12月27日，中央郵政經濟委員會第九次會議決定成立郵政總局。1954年，[[中华人民共和国邮电部|中华人民共和国邮电部]]成立。

1970年，原邮电部并入[[中华人民共和国交通部|中华人民共和国交通部]]，称“交通部邮政总局”。1973年，恢复组建中华人民共和国邮电部。

1994年3月1日，[[中華人民共和國國務院|国务院]]批准郵電部機構改革方案，郵政總局由機關行政序列分離，成為專業核算的企業局。1995年10月4日，郵政總局在[[中华人民共和国国家工商行政管理局|中华人民共和国国家工商行政管理局]]註冊了企業法人營業執照，獲得[[法人|法人]]資格，企業名稱為「中國郵電郵政總局」，簡稱「中國郵政」。

2005年7月20日，[[国务院常务会议|國務院常務會議]]通過中國郵政「政企分開、郵儲分離、完善機制」的體制改革方案：組建新的國家郵政局，由[[中华人民共和国信息产业部|信息产业部]]管理，作為政府機構依法監管郵政市場，並協調郵政普遍服務與機要通信等特殊服務的實施；組建[[中国邮政|中國郵政集團公司]]，作為國有獨資企業經營各類郵政業務；成立由中國郵政集團公司控股的[[中国邮政储蓄银行|中國郵政儲蓄銀行]]，實現金融業務規範化經營；完善郵政普遍服務機制、特殊服務機制、安全保障機制和價格形成機制<ref>{{Cite web|url=http://zqb.cyol.com/content/2007-01/30/content_1659831.htm|title=国家邮政局、中国邮政集团公司挂牌|accessdate=2018-06-12|date=2007-01-30|publisher=中国青年报}}</ref>。2006年8月28日國務院以國函〔2006〕79號批覆，原則同意《中國郵政集團公司組建方案》和《中國郵政集團公司章程》，並指示「中國郵政集團公司」在原郵政總局所屬的[[經營性資產|經營性資產]]和部份企事業單位基礎上，依照《{{s|中华人民共和国全民所有制工业企业法}}》組建，不設[[董事會|董事會]]，出資人職責暫由[[中华人民共和国财政部|財政部]]代表國務院履行，實行總經理負責制，總經理為公司法定代表人。中國郵政集團註冊資金800億元[[人民幣|人民幣]]，但實有[[國有|國有]][[資本|資本]]數額待公司正式成立後，財政部進行資產評估和審計驗資後核定。

2006年11月27日，原國家郵政局在北京市[[西城區|西城區]][[宣武门西大街|宣武門西大街]]131號辦公樓召開最後一屆全體機關幹部大會，宣佈留任國家郵政局的具體人選；11月28日，重組後的國家郵政局舉行全體幹部大會。11月29日，中國郵政集團公司在[[中华人民共和国国家工商行政管理总局|中华人民共和国国家工商行政管理总局]]註冊登記，註冊號為1000001001822<ref>{{Cite web|url=http://news.xinhuanet.com/fortune/2006-12/01/content_5415848.htm|title=中国邮政集团公司成立 注册资本800亿元|accessdate=2019-01-14|date=2006-12-01|publisher=新华网|archiveurl=https://web.archive.org/web/20100202123629/http://news.xinhuanet.com/fortune/2006-12/01/content_5415848.htm|archivedate=2010-02-02}}</ref>。同年12月31日，[[中国银行保险监督管理委员会|中國銀監會]]批准[[中国邮政|中國郵政集團公司]]控股的[[中国邮政储蓄银行|中國郵政儲蓄銀行]]開業。2007年1月29日10時30分，國家郵政局和中國郵政集團公司在[[人民大會堂|人民大會堂]]舉行揭牌儀式。當天稍後，國家郵政局在北京[[西直門|西直門]]辦公樓前，中國郵政集團公司在北京[[西便門|西便門]]辦公樓前，分別舉行掛牌儀式。3月20日（[[農曆|農曆]][[二月初二|二月初二]]）10時，[[中国邮政储蓄银行|中國郵政儲蓄銀行]]在北京總部大樓舉行成立暨掛牌儀式。

2008年3月15日，[[第十一届全国人民代表大会#第一次会议|第十一届全国人民代表大会第一次会议]]通过《第十一届全国人民代表大会第一次会议关于国务院机构改革方案的决定》，批准《国务院机构改革方案》。方案规定：“组建[[中华人民共和国交通运输部|交通运输部]]。将[[中华人民共和国交通部|交通部]]、[[中国民用航空总局|中国民用航空总局]]的职责，[[中华人民共和国建设部|建设部]]的指导城市客运职责，整合划入交通运输部。组建国家民用航空局，由交通运输部管理。国家邮政局改由交通运输部管理。保留[[中华人民共和国铁道部|铁道部]]，继续推进改革。不再保留交通部、中国民用航空总局。”<ref>{{Cite web|url=http://www.npc.gov.cn/npc/xinwen/syxw/2008-03/12/content_1413911.htm|title=关于国务院机构改革方案的说明——2008年3月11日在第十一届全国人民代表大会第一次会议上|author=[[华建敏|华建敏]]|date=2008-03-11|publisher=中国人大网}}</ref><ref>{{Cite web|url=http://www.gov.cn/2008lh/content_921411.htm|title=两会授权发布：国务院机构改革方案|date=2008-03-15|publisher=中国政府网}}</ref><ref>{{Cite web|url=http://www.gov.cn/2008lh/content_921283.htm|title=十一届人大一次会议关于国务院机构改革方案决定|accessdate=2018-06-12|date=2008-03-15|publisher=中国政府网}}</ref>

== 职责 ==
根据《{{s|国家邮政局主要职责内设机构和人员编制规定}}》，国家邮政局承担下列职能：
{{columns-list|2|
# 拟订邮政行业的发展战略、规划、政策和标准，提出深化邮政体制改革和促进邮政与交通运输统筹发展的政策建议，起草邮政行业法律法规和部门规章草案。
# 承担邮政监管责任，推动建立覆盖城乡的邮政普遍服务体系，推进建立和完善普遍服务和特殊服务保障机制，提出邮政行业服务价格政策和基本邮政业务价格建议，并监督执行。
# 负责快递等邮政业务的市场准入，维护信件寄递业务专营权，依法监管邮政市场。
# 负责监督检查机要通信工作，保障机要通信安全。
# 负责邮政行业安全生产监管，负责邮政行业运行安全的监测、预警和应急管理，保障邮政通信与信息安全。
# 负责邮政行业统计、经济运行分析及信息服务，依法监督邮政行业服务质量。
# 负责纪念邮票的选题和图案审查，负责审定纪念邮票和特种邮票年度计划。
# 代表国家参加国际邮政组织，处理政府间邮政事务，拟订邮政对外合作与交流政策并组织实施，处理邮政外事工作，按照规定管理涉及港澳台工作。
# 垂直管理各省（自治区、直辖市）邮政管理局。
# 承办国务院及交通运输部交办的其他事项。
}}
