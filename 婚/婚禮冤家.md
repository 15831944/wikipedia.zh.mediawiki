{{作品名稱|name=''Destination Wedding''|hk=婚禮進行識|tw=婚禮冤家|cn=终点的婚礼}}是一部2018年[[美国|美國]][[浪漫|浪漫]][[喜劇|喜劇]][[电影|電影]]，由維克多·萊文執導和編劇，其主演包括[[薇诺娜·瑞德|薇諾娜·瑞德]]及[[基努·里维斯|基努·李維]]。該片於2018年8月31日由Regatta在[[美國|美國]]發布<ref>{{cite web|url=https://www.vscinemas.com.tw/film/detail.aspx?id=3305 |language=zh-hant |title=婚禮冤家 |publisher=威秀影城 | accessdate=2018-09-04 }}</ref>。

== 劇情 ==
琳賽（[[薇诺娜·瑞德|薇諾娜·瑞德]]飾演）和法蘭克（[[基努·里维斯|基努·李維]]飾演）是兩位在[[加利福尼亞州|加利福尼亞州]][[帕索羅布爾斯|帕索羅布爾斯]]參加共同朋友婚禮的陌生人。在那裡，兩人爆發了對彼此和他們的朋友的不滿<ref>{{cite news|title=度假婚禮遭丟包 電影台詞是基努李維真實心聲 |url=http://www.epochtimes.com/b5/18/8/29/n10675575.htm |language=zh-hant |author=蘇漾 |publisher=大紀元 |date=2018-08-29 |accessdate=2018-09-04 }}</ref>。

== 製作 ==
主要拍攝於2017年8月在[[加利福尼亚州|加利福尼亞州]]中部進行。

== 發行 ==
2017年11月，Rowing Pictures收購了[[美国|美國]][[电影|電影]]的發行權，並以其帆船賽旗幟發行<ref>{{cite web|url=https://variety.com/2017/film/global/destination-wedding-keanu-reeves-1202608337/|title=Aviron Pictures Acquires ‘Destination Wedding’ With Keanu Reeves, Winona Ryder|website=[[Variety_(magazine)|Variety]]|first=Elsa|last=Keslassy|date=2017-11-27|accessdate=2018-09-02}}</ref>。在[[美国|美國]]，該片於2018年8月31日上映。

== 反響 ==
這部電影大多收到了不同的評論。在評論聚合網站[[爛番茄|爛番茄]]，該影片的評級為48％，基於52條評論，平均評分為5.7 / 10。

《洛杉磯時報》的Micheal Rechtshaffen稱這部電影稱其為“頑固不動的86分鐘感覺更長時間。《紐約時報》的Jeanette Catsoulis對電影“酷刑”進行了評論，並稱“這部可能是浪漫的喜劇是光柵看起來很便宜，看起來很強大：它還將兩個經驗豐富，討人喜歡的演員變成了你想要在演唱會之前扼殺的角色。《娛樂周刊》的Leah Greenblatt給了影片一個C +說劇本”來了就像一個rom-com David Mamet貨運列車；它的口頭轉彎被如此瘋狂地覆蓋，以至於所有演員真正做到的就是抓住車輪，通過大量的對話。

更為積極的是，Indiewiregave的Kate Erbland在電影中寫道：“也許這就是墜入愛河的情況，現在是時候rom-coms有像這樣的奇怪的空間了。《衛報》的本傑明·李給了五分之三的影片承認“一些對話確實會被覆蓋......有足夠的機智和精心觀察的迂腐以彌補偶爾的放縱。”在這部影片的明星裡，李說：“這對人分享了一種輕鬆，尖刻的化學反應，特別是里夫斯表現出自己在提供這種充滿膽汁的對話方面有著驚人的技巧。
