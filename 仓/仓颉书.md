'''倉頡書'''是傳說中[[漢字|漢字]]最早的[[文字|文字]]，相傳是漢字鼻祖[[倉頡|倉頡]]所寫，總共只有28個字。

由於年代久遠，此書歷來被[[書法家|書法家]]懷疑為漢儒[[劉歆|劉歆]]所作，而學者[[劉志一|劉志一]]多年研究後認為，此書是用[[古彝文|古彝文]]書寫的一段[[彝族|彝族]]先民[[祭祀|祭祀]]的記錄。由於[[黃帝|黃帝]]和彝族均出於[[西羌|西羌]]，所以這一說法有一定的根據。

學者劉志一將他它直譯為：「一妖來始，界轉臥鴉杈，祭神青腦，禍小馬念，師五除掃，幡齋解果，過鼠還魂。」意譯為：「一群妖魔剛來到，樹上烏鴉滿天飛；割青宰羊祭山神，念經消災騎馬歸；五位經師施法術，做齋完畢魂幡回；消滅鼠精魂歸位。」

== 仓颉字母说==
仓颉书传说于公元前26世纪，由黄帝的史官仓颉所写。经过破译，仓颉书是一套字母，每个字母的含义与腓尼基字母的含义有很大重合。

腓尼基字母约在公元前14世纪出现，因而仓颉书可能是腓尼基字母的前身。也就是说，仓颉字母可能是全世界现代字母系统的始祖。

=== 仓颉字母破译方法===
# 每一个腓尼基字母都有比较准确的含义，先找到每一个腓尼基字母的含义；例如，腓尼基字母表第一个字母的含义是“牛”；
# 再找到这些含义所对应的中文甲骨文；
# 把这些甲骨文的文字字体，进行简单的拆解变形，会发现22个腓尼基字母全都包含在仓颉书的28个符号之中，甚至字母顺序上都有较强的相关性。

===古文字变为仓颉字母的规律===
# 对于古中文是独体字的，仓颉字母采用原文字一部分，不旋转，保留原样或进行左右镜像、上下颠倒的变化；如：1、牛；   3、矛；   10、手；   17、孔；
# 
# 对于古中文是独体字的，进行90度旋转；如：11、水；    12、叉；    13、草；
# 
# 对于古中文是左右结构的合体字，字母采用原文字的或左或右一半，进行90度旋转变化；如：6、欢；    15、园；
# 
# 对于古中文是上下结构、上中下结构、半包围结构的合体字，字母采用原文字一部分，并且保持不变；如：2、房；    4、唇；    5、鱼；    7、轮；    21、雪；
# 
# 对于古中文是合体字，如果左右结构中的一边还可以分为上下结构，或是左中右结构，也就是原文字可以由三部分组成的，则字母采用原文字一部分，并且保持不变；如： 8、剑；    8、夹；    19、标；    20、风；    26、雾；
# 
# 仓颉字母采用部分的文字笔顺方向，可能和原字方向相反；如：3、矛；    16、头；    17、孔；    25、眼；
# 
# 原文字中出现短竖“|”、“田”字形的，简化成点“ · ”；如：11、水；    22、车；    27、雷；
<br>

古文字变为仓颉字母的规律，可以简单总结为：
“三取一，二取一，独字取一半；上不变，下不变，左右要旋转”。<br>

即原字结构由较为独立三部分或两部分字形组成的，采取其中的一部分；<br>

原字是独体字的，采取其字形的一半；<br>

原字是上下结构、半包围结构的，采取的部分字形保持原样不变；<br>

原字是左右结构的，采取的部分大多要顺时针旋转90度或镜像变化。
