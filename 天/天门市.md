'''天门市'''是[[中国|中国]][[湖北省|湖北省]]四个[[省直辖县级行政区|省直辖县级市]]（[[副地级市|副地级市]]）之一，茶圣陆羽的故里。地处鄂中腹地[[江汉平原|江汉平原]]，地理位置优越，上通荆沙，下接武汉，南濒江汉黄金水道，北枕三峡过境铁路，紧衔107、318国道和宜黄高速公路。全市国土面积2622平方公里，人口173万；在长江，汉水经济开发带中，具有强大的接纳和辐射能力。2014年荣膺“中国最具生态竞争力城市”称号。

== 历史 ==

先秦时，此地属[[风国|风国]]。春秋时属[[鄖國|郧国]]，战国时为[[楚国|楚国]]竟陵邑。秦置[[竟陵县|竟陵县]]。竟陵意为“陵之竟也”，即山陵至此终止。从秦置竟陵县算起，有2000多年县制历史。北宋以后，辖区范围固定<ref name="a1"/>。清雍正四年（1726年），为避[[康熙帝|康熙帝]]墓名（[[清景陵|景陵]]）讳，改为[[天门县|天门县]]。

民国时期，县域基本维持在光绪年间范围内。1939年6月，因抗日战争局势，县政府外迁。1945年9月，政府回迁竟陵镇。抗日战争期间，[[中共政权|中共政权]]曾在此地设置[[天汉县|天汉县]]、[[天京潜县|天京潜县]]。1948年9月24日，中共政权政权攻占天门县。其后多有变迁。中华人民共和国建立后，历属[[荆州专区|荆州专区]]、[[荆州地区|荆州地区]]。1987年8月3日，撤县建市，1994年10月24日，批准为[[省直管县|省直管县]]（[[省直辖县级行政区|省直辖县级行政区]]）<ref name="a1">{{Cite web |url=http://www.tianmen.gov.cn/root10/szbm/0036/201004/t20100419_147180.shtml |title=《天门市志》（地理环境）志稿公示……2、区域演变……3.建置沿革|accessdate=2019-02-25 |author= |date=2010-04-19 |work= |publisher=天门市人民政府网站|language=Zh-hans }}</ref>。

=== 侨胞 ===
天门市是中国内地著名侨乡，湖北省的重点侨乡。天门旅居海外的华侨、华人及港澳同胞有280,000多人，分布在世界五大洲40多个国家和地区。全市有归侨500多人，侨、港、澳眷属82,000多人。 

天门华侨出国始于18世纪末，先后经历了北上（欧洲）时期、南下（东南亚）时期和新移民时期等几个阶段。

== 文化 ==
天门市是享有盛誉的文化之乡。世界文化名人“茶圣”[[陆羽|陆羽]]、唐代诗人[[皮日休|皮日休]]、明代[[竟陵派|竟陵派]]文学代表人物[[钟惺|钟惺]]、[[谭元春|谭元春]]和清代状元[[蒋立镛|蒋立镛]]诞生于此。恢复高考制度以来，天门共为国家输送大学生54000多人，被誉为全国的“[[状元|状元]]之乡”。 

名胜古迹遍天门。境内有迄今为止长江流域最大的原始社会村落遗址——[[石家河新石器时代村落遗址|石家河新石器时代村落遗址]]，有保存完好、气势宏伟，始建于[[南朝|南朝]]的著名佛寺——[[白龙寺|白龙寺]]，有纪念陆羽的陆羽亭、文学泉、古雁桥，有凭吊钟惺、谭元春的钟惺墓、谭元春墓，还有景色秀丽的东湖、西湖风景区。

湖北省天门中学是一所享誉全国的省级示范高中，始建于1912年，坐落在茶圣陆羽故里——湖北省天门市竟陵钟惺大道西端，是湖北省园林式学校、省级示范中学、全国教育系统先进集体、全国精神文明建设工作先进单位，至今已走过了一百多年的辉煌历程。学校以一流的环境，一流的设施，一流的师资，一流的管理，一流的业绩，确立了天门中学在全省乃至全国一流名校的地位，于2012年迎来100年华诞。

== 经济 ==
截至2015年底，天门市地区生产总值达440.1亿元，增长8.9%；公共财政预算收入19.58亿元，增长12.9%；工业增加值201.67亿元，增长8.9%；固定资产投资373.41亿元，增长17.9%；社会消费品零售总额258.9亿元，增长12.1%；城镇和农村常住居民人均可支配收入分别为22618元和13178元，分别增长9.68%和9.04%。[https://web.archive.org/web/20161016180820/http://www.tianmen.gov.cn/root10/0000/auto950/201602/t20160226_324579.html 天门市2016年政府工作报告]
