很多[[天體|天體]]都會以[[地震波|地震波]]的形式釋放大量能量，造成該天體的劇烈震動，是為'''天體震動'''。它以震動的主體分類，如地震、月震等。

==地震==
{{Main article|地震}}
發生於[[地球|地球]]的[[地殼|地殼]]震動為'''地震'''，成因是[[板塊|板塊]]的搖動和位移。[[海嘯|海嘯]]亦會伴隨着地震發生。

==月震==
發生於[[月球|月球]]的震動為'''月震'''，由[[阿波羅計劃|阿波羅計劃]]中[[宇航員|宇航員]]設置的[[地震儀|地震儀]]發現，於1972年至1977年間共錄得28次表層月震(見下表)。相對於強地震來說，月震的程度較為輕微。而由於月球上缺乏[[水體|水體]]去吸收地震波的緣故，月震持續的時間較長，有些甚至達一小時<ref>{{cite journal|last1 = Latham|first1 =  Gary|year = 1972|last2 = Ewing|first2 = Maurice|last3 = Dorman|first3 = James|last4 = Lammlein|first4 = David|last5 = Press|first5 = Frank|last6 = Toksőz|first6 = Naft|last7 = Sutton|first7 = George|last8 = Duennebier|first8 = Fred|last9 = Nakamura|first9 = Yosio|title = Moonquakes and lunar tectonism|url = http://www.springerlink.com/content/jt12323560565365/|journal = Earth, Moon, and Planets|volume = 4|issue = 3–4| pages = 373–382|doi = 10.1007/BF00562004|bibcode = 1972Moon....4..373L}}</ref>。

根據[[美国国家航空航天局|NASA]]，月震可分類如下：
#由於[[隕石|隕石]]撞擊造成的月震
#由於日夜溫差造成地殼[[熱脹冷縮|熱脹冷縮]]
#深層月震─地殼以下700公里的震動
#表層月震─地殼以下20至30公里的震動

第1至第3種地震強度較輕微，而表層月震強度可達[[芮氏地震規模|芮氏地震規模]]5.5級。

==金星震==
發生於[[金星|金星]]的震動為'''金星震'''。

透過[[麥哲倫號金星探測器|麥哲倫號金星探測器]]於1990年11月及1991年7月拍得的兩幀照片，清楚顯示一峭谷出現裂縫和滑坡，科學家相信這是金星存在地震的證據。

==日震==
{{Main article|日震學}}
發生於[[太陽|太陽]]的震動為'''日震'''。1995年[[太陽和太陽風層探測器|SOHO]]發射升空觀察太陽，當中就包括探測日震。

根據[[自然_(期刊)|自然期刊]]，1996年7月9日發生一次日震，達芮氏地震規模11.3級，比所有有記錄的地球地震要強得多。這次日震引發了X2.6類的[[閃焰|閃焰]]和強烈的[[日冕物質拋射|日冕物質拋射]]<ref>{{cite web|title=Solar Flare Leaves Sun Quaking|url=http://soi.stanford.edu/press/agu05-98/press-rel.html|accessdate=31 March 2012}}</ref><ref>{{cite journal|coauthors=A. ==星震==
'''星震'''是指天體的震動，特指[[中子星|中子星]]的地殼震動。利用[[多普勒效應|多普勒效應]]來觀察天體的震動，研究天體的震動可以讓[[科學家|科學家]]及人們了解到[[星體|星體]]的內部。其成因有二：
*在強大[[磁場|磁場]]下造成地殼的壓力
*[[參考系拖拽|參考系拖拽]]和磁偶極子的轉動令[[角動量|角動量]]損失，造成地殼的壓力

當地殼積累一定壓力時，它會調整自己的形狀，使自己更相似於球體。每次調整皆不超過一微米，但皆完成於一百萬分之一秒內。
