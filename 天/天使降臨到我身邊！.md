{{日本作品|天使降臨到我身邊！|私に天使が舞い降りた！}}是[[椋木ななつ|椋木ななつ]]創作的日本[[四格漫畫|四格漫畫]]作品。2016年11月18日於《[[Comic百合姬|Comic百合姬]]》的2017年1月號開始連載<ref>{{cite web|archiveurl=https://web.archive.org/web/20161120201249/http://www.ichijinsha.co.jp/yurihime|archivedate=2016年11月20日|url=http://www.ichijinsha.co.jp:80/yurihime|title=コミック百合姫1月号のお知らせ|publisher=[[一迅社|一迅社]]|accessdate=2018年6月24日|language=ja|deadurl=yes|df=}}</ref>。改編的[[電視動畫|電視動畫]]由[[動畫工房|動畫工房]]製作，於2019年1月至3月間播放。

== 故事簡介 ==
女大學生星野宮子對小學生妹妹星野日向的同學白咲花一見鍾情，雖然宮子一心想跟花很友好，但都遭到了花的抗拒。最後靠著製作美味點心的好手藝，成功虜獲芳心，日後展開了一連串的日常喜劇故事。

== 登場角色 ==

=== 主要角色 ===
; {{nihongo|星野宮子|星野 みやこ（ほしの みやこ）|聲：[[上田麗奈|上田麗奈]]<ref name="character">{{cite web|url=http://watatentv.com/staff.html|title=CHARACTER|accessdate=2018-10-17|language=ja}}</ref>}}
: 大學1年級，設計專業科系，宅女。
: 個性內向怕生，不擅與人交際。對白咲花一見鍾情，遇到與花有關的事情會很興奮。擅長製作COSPLAY服與甜點。

; {{nihongo|白咲花|白咲 花（しろさき はな）|聲：[[指出毬亞|指出毬亞]]<ref name="character"/>}}
: 小學5年級。
: 稱宮子為「姐姐」，對宮子的初次印象是個危險的人。個性冷靜，非常喜歡吃甜點，談到甜點就會變得很興奮。審美觀與常人不同，喜歡各種奇怪卡通人物的周邊產品。

; {{nihongo|星野日向|星野 ひなた（ほしの ひなた）|聲：[[長江里加|長江里加]]<ref name="character"/>}}
: 小學5年級，星野宮子的妹妹，花的同學。
: 稱宮子為「宮姐」。個性開朗活潑，姐控，十分喜欢宮子，認為宮子無所不能，喜歡跟人誇獎宮子，導致宮子在別人眼中是個完美形象的人。常要求跟宮子一起睡覺，但睡相非常的差。

; {{nihongo|姬坂乃愛|姫坂 乃愛（ひめさか のあ）|聲：[[鬼頭明里|鬼頭明里]]<ref name="character"/>}}
: 小學5年級，與花同年的轉學生。
: 星野家隔壁的鄰居，稱宮子為「宮小姐」。覺得自己很可愛，也喜歡別人稱讚自己很可愛，會因為宮子對花的可愛稱讚多於自己而感到忌妒。

; {{nihongo|種村小依|種村 小依（たねむら こより）|聲：[[大和田仁美|大和田仁美]]}}
: 小學5年級，日向的同班同學，班上的班長。
: 希望有人能依賴自己，但由於做什麼都失敗的能力，令大家轉向依賴可靠的夏音。與夏音是青梅竹馬，關係親密，都在一起行動。

; {{nihongo|小之森夏音|小之森 夏音（このもり かのん）|聲：[[大空直美|大空直美]]}}
: 小學5年級，日向的同班同學，班上的班長。
: 為人和善禮貌，做事沉穩可靠，因此很多同學會依賴她而不是小依。在自己有困難的時候，小依都會不顧一切的全力幫助，所以最喜歡小依，認為自己對小依非常依賴。

=== 次要角色 ===
; {{nihongo|松本香子|松本 香子（まつもと こうこ）|聲：[[Lynn_(配音員)|Lynn]]}}
: 星野宮子高中時期的社團同伴。
: 高中時，被星野宮子做衣服的姿態所迷恋，從此经常偷偷跟踪宫子，自認為是宮子的好朋友兼對手，但實際上宮子根本不熟識她，也不認為兩人是朋友關係。後來跟踪事情曝光，但也跟宮子成為真正的朋友。

; {{nihongo|松本友奈|松本 友奈（まつもと ゆな）|聲：[[木野日菜|木野日菜]]}}
: 松本香子的妹妹。

; {{nihongo|星野千鶴|星野 千鶴（ほしの ちづる）|聲：[[小清水亞美|小清水亞美]]}}
: 宮子和日向的媽媽。
: 有抽[[電子菸|電子菸]]的習慣，對宮子一些玩鬧過頭的行為會嚴厲地懲罰。

; {{nihongo|白咲春香|白咲 春香（しろさき はるか）|聲：[[藤村步|藤村步]]}}
: 花的媽媽。

; {{nihongo|姬坂愛蜜莉|姫坂 エミリー（ひめさか エミリー）|聲：[[豐崎愛生|豐崎愛生]]}}
: 乃愛的媽媽。

; {{nihongo|宮子（寵物）|みやこ|聲：[[大空直美|大空直美]]}}
: 松本家的寵物狗，與星野宮子同名。

; {{nihongo|山中老師|山中先生|聲：[[衣川里佳|衣川里佳]]}}
: 日向班的女老師兼[[班主任|班主任]]。
