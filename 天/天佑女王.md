《'''-{天佑女皇}-'''》（{{lang-en|God Save the Queen}}），男性[[英国君主|君主]]在位时称为《'''-{天佑吾王}-'''》（{{lang|en|God Save the King}}），是[[英國|英國]]、英國的[[皇家屬地|皇家屬地]]、[[海外領土|海外領土]]和[[英联邦王国|英联邦王国]]及其领地作为[[国歌|国歌]]或皇家[[礼乐|礼乐]]使用的[[颂歌|颂歌]]<ref>{{cite web |title=Isle of Man |publisher=nationalanthems.info |url = http://www.nationalanthems.info/im.htm |accessdate=17 August 2010}}</ref>。歌词和歌名随当朝[[英国君主|君主]]的性别而有所改变：例如在男性[[英国君主|君主]]在位时歌词中的“-{女王}-”改成“-{吾王}-”（“国王”）、“她”(She)改成“他”(He)等。曲调作者不详，可能源自單聲[[聖歌|聖歌]]，但也有传统说法认为是1619年由{{link-en|约翰·布尔 (作曲家)|John Bull (composer)|约翰·布尔}}所作。

《天佑吾王》[[英国|英国]]的国歌，在历史上曾广泛在[[大英帝国|大英帝国]]及[[英联邦|英联邦]]国家作为国歌使用。现在，它在部分[[英國海外領土|英國海外領土]]仍作为国歌使用<ref>{{Cite web |url=http://www.royal.gov.uk/MonarchUK/Symbols/NationalAnthem.aspx |title=存档副本 |access-date=2014-10-22 |archive-url=https://web.archive.org/web/20140902162152/http://www.royal.gov.uk/MonarchUK/Symbols/NationalAnthem.aspx |archive-date=2014-09-02 |dead-url=yes }}</ref>。它也是[[新西兰|新西兰]]的两首国歌之一（1977年之前是唯一的国歌），一些英国海外领土也同样将它与另一首歌并列为国歌。在[[澳大利亚|澳大利亚]]（1984年起）、[[加拿大|加拿大]]（1980年起）<ref>{{Citation |last=MacLeod |first=Kevin S. |authorlink=Kevin S. MacLeod |title=A Crown of Maples |place=Ottawa |publisher=Queen's Printer for Canada |year=2008 |edition=1| page=I |url = http://www.pch.gc.ca/pgm/ceem-cced/fr-rf/crnCdn/crn_mpls-eng.pdf |ISBN = 978-0-662-46012-1 |accessdate=25 June 2010 |ref=CITEREF_MacLeod_2008}}</ref>、[[巴巴多斯|巴巴多斯]]和[[图瓦卢|图瓦卢]]，《天佑吾王》不是国歌，但作为皇家礼乐使用。

此外，在未曾受[[大英帝国|大英帝国]]统治的国家，《天佑吾王》的音乐时常被用来创作当地的爱国歌曲，但多数仍和王室仪式有关<ref>{{cite web |url = http://nationalanthems.me/united-kingdom-god-save-the-queen/ |title=United Kingdom - God Save the Queen |publisher=NationalAnthems.me |accessdate=23 November 2011 }}</ref>。[[美国|美国]]爱国歌曲《[[我的国家属于你|我的国家属于你]]》（即《亞美利加》）也使用了《天佑吾王》的曲调。

《天佑吾王》的第一段歌词只有一种版本，但其它段落在历史上和现在都有各种版本：历史上出现的版本中不断有加上和删除各段落，甚至今天此歌曲仍有选取不同段落或对段落进行不同排列的版本{{efn|参见颂歌集《英格兰颂歌集》（''English Hymnal''）、《古今颂歌集》（''Hymns Ancient and Modern''）、《赞美之歌》（''Songs of Praise''）收集的，以及英国王室网站上（[http://www.royal.gov.uk/output/page5010.asp royalty.gov.uk]）的不同版本。}}。一般情况下，《天佑吾王》作为英国国歌演唱时會唱第一段和第三段，很少情況下會唱第二段<ref name=bgkoxw>{{cite web |url = http://www.royal.gov.uk/MonarchUK/Symbols/NationalAnthem.aspx |title = Monarchy Today pages at royal.gov.uk website |accessdate = 1 April 2007 |archive-url = https://web.archive.org/web/20140902162152/http://www.royal.gov.uk/MonarchUK/Symbols/NationalAnthem.aspx |archive-date = 2014-09-02 |dead-url = yes }}</ref>。第二段的來源據說是教會所指定的一篇用於"火藥陰謀"(1605年11月5日)週年日的古老禱詞。這一古老的禱詞中有這樣幾行：“驅散我們的敵人……消除他們的惡意,挫敗它們的詭計。"

《天佑吾王》作为礼乐使用时，对君主本人致敬时应演奏整首歌曲，对其它受王室礼乐待遇的王室人员（例如[[威尔士亲王|威尔士亲王]]）则适用前六小节。英国以外的部分英联邦王国也使用《天佑吾王》的前六小节作为总督礼乐，或礼乐的部分：例如，加拿大总督和各省省督的礼乐是《天佑吾王》的前六小节加上《[[啊，加拿大|啊，加拿大]]》的最后四小节，英国海外领土总督的礼乐是《天佑吾王》的前六小节。

== 历史 ==
一般认为这首歌现在的曲调由[[亨利·凯里|亨利·凯里]]博士所作，尽管在一些早期版本中出现过的乐段也出现在现在的版本当中，导致了一些混淆。

人们相信，这首歌的第一次公演发生在1740年的一次宴会上，在这次宴会上凯里为了祝贺[[詹金斯的耳朵戰爭|从西班牙人手中夺取]]了港口[[皮特贝罗|皮特贝罗]]的[[爱德华·弗农|爱德华·弗农]][[海军上将|海军上将]]唱了这一支歌。

传统上，此曲的第一次公开演唱被认为发生在1745年。在英王[[乔治二世|乔治二世]]的时代，要求英国王位的[[詹姆斯党|詹姆斯党]]领袖[[查尔斯·爱德华·斯图亚特|查理·斯图亚特]]（亦即“小觎位王子”，为[[詹姆斯二世|詹姆斯二世]]长孙，在苏格兰有很多支持者，因此就有了支持苏格兰的最后一段）重返英伦，谋求复辟[[斯图亚特王朝|斯图亚特王朝]]，在一次叛军获胜的战斗后，人们唱着这支歌曲来支持国王。

== 作为英国国歌 ==
[[File:It_is_far_better_to_face_the_bullets.jpg|thumb]]
《天佑吾王》是英国国歌，但如同[[英国宪法|英国宪法]]的许多方面，国歌的地位来自于习惯和长期使用，从未有一条[[御令|御令]]或[[法律|法律]]宣布其为国歌。一般而言，会演唱第一段和第三段（很少的时候唱第二段）。<ref>[http://www.royal.gov.uk/output/page5010.asp 英国皇室官方网站：联合王国国歌]</ref>在英国使用的习惯版本是现行版本中最古老的，也是英联邦其它国家现在使用的版本的基础，但英国使用的传统版本歌词在历史上也时有变动。

在英国的各部分别出现的体育赛事中（例如足球世界杯赛），《天佑吾王》一般作为[[英格兰|英格兰]]和[[北爱尔兰|北爱尔兰]]的国歌使用，[[苏格兰|苏格兰]]和[[威尔士|威尔士]]则分别有自己的“国家歌曲”和国歌。但在[[板球|板球]]比赛中则使用《[[耶路撒冷_(讚歌)|耶路撒冷]]》代表英格兰， 2010年起在[[英联邦运动会|英联邦运动会]]中也使用《耶路撒冷》代表英格兰。

== 演奏 ==
传统上，这首歌以慢而低沉的节奏演奏，被人认为非常沉闷。在某些情况下为了减轻这种效果，这首歌以一个较快而且较活泼的节奏演奏。喜剧演员[[比利·康诺利|比利·康诺利]]在一场电视节目中比较了英国慢而低沉的国歌和很多其他国家的活泼的国歌，并建议将这首歌的曲调以「[[The_Archers|The Archers]]」的主题曲代替。

在2002年6月4日于[[白金汉宫|白金汉宫]]举行的纪念女王即位50周年的流行音乐会上，[[皇后乐队|皇后乐队]]的[[布赖恩·梅|布赖恩·梅]]在宫顶用电[[吉他|吉他]]演奏了这首歌。

== 在英联邦中使用 ==
這首歌曾經被包括[[澳大利亚|澳大利亚]]、[[加拿大|加拿大]]、[[英屬香港|香港]]、[[马来西亚|马来西亚]]、[[新加坡|新加坡]]、[[文莱|文莱]]和[[牙买加|牙买加]]等在内的大部分[[英联邦|英联邦]]成員及英國屬土作為國歌。在[[英屬香港|英屬香港]]，每當有英國皇室成員到訪，或[[香港總督|香港總督]]就職履新，或出席官式活動和檢閱，[[駐港英軍|駐港英軍]]（或[[香港義勇軍|香港義勇軍]]、[[皇家香港警察|皇家香港警察]]）儀隊行[[皇室敬禮|皇室敬禮]]（Royal Salute），軍樂團演奏國歌，其中港督和其他皇室成員奏半首，若是女皇到訪的就要全首奏畢。

现在，这些国家和地区中国歌的地位已经被《[[前进的澳大利亚|前进的澳大利亚]]》（澳大利亚国歌）、《[[哦！加拿大|哦！加拿大]]》（加拿大国歌）、《[[義勇軍進行曲|義勇軍進行曲]]》（中华人民共和国国歌）、《[[我的祖國_(馬來西亞)|我的祖國]]》（马来西亚国歌）、《[[前进吧，新加坡|前进吧，新加坡]]》（新加坡国歌）、《[[真主保佑蘇丹|真主保佑蘇丹]]》（文莱国歌）和《[[牙買加，我們熱愛的家園|牙買加，我們熱愛的家園]]》（牙买加国歌）等所代替。

这首歌和《[[天佑新西兰|天佑新西兰]]》一起被[[新西兰|新西兰]]承认为国歌。

此外，《天佑女王》也曾经是[[爱尔兰自由邦|爱尔兰]]的国歌，惟在1920年代被《[[战士之歌|战士之歌]]》取代。

在80年代香港已經出現過「[[惡搞|惡搞]]」的廣東話版本，首句為「個個揸住個兜（個個拿着碗）」，但之後如何「譯」就沒有定案。版本包括「嗰啲湯啱啱夠（那些湯剛剛好足夠）」、「刀叉都生晒鏽 （刀叉也生了鏽）」）、「踎响街邊乞米（蹲在街邊乞食）」等。相傳這些惡搞版本源自監獄。

== 在其它地方使用 ==
此曲調曾被幾個其他的國家的國歌所採用，包括[[丹麥|丹麥]]、[[德國|德國]]《[[万岁胜利者的桂冠|万岁胜利者的桂冠]]》（第二帝國時期，1871年—1922年）、[[俄國|俄國]]《[[俄罗斯人的祈祷|俄罗斯人的祈祷]]》（至1833年）、[[瑞典|瑞典]]和[[瑞士|瑞士]]《祖国请你召唤》。

[[列支敦士登|列支敦士登]]的國歌《[[在年輕的萊茵河上|在年輕的萊茵河上]]》的曲調與這首歌相同。[[挪威|挪威]]的王室頌歌《[[國王之歌|國王之歌]]》，[[美國|美國]]的愛國流行歌曲《[[My_Country,_'Tis_of_Thee|My Country, 'Tis of Thee]]》和法屬[[新喀里多尼亞|新喀里多尼亞]]其中一首土著創作的獨立運動歌曲的曲調也與其相同。

英國搖滾樂團「[[皇后樂隊|皇后樂隊]]」在他們的音樂會的結束總是演奏這首曲的搖滾版。

== 联合王国的其它国歌 ==
在需要播放联合王国成员国国歌的场合（如国际体育比赛）下，下列歌曲经常使用：
* [[威尔士|威尔士]]使用《[[父辈的土地|父辈的土地]]》；
* [[苏格兰|苏格兰]]使用《[[苏格兰之花|苏格兰之花]]》或《{{link-en|勇敢的苏格兰人|Scotland the Brave}}》；
* [[英格兰|英格兰]]使用《[[耶路撒冷_(歌曲)|耶路撒冷]]》或《[[希望与光荣的土地|希望与光荣的土地]]》；
* [[北爱尔兰|北爱尔兰]]使用《[[伦敦德里小调|伦敦德里小调]]》。

在国际[[足球|足球]][[比赛|比赛]]的演奏[[國歌|國歌]]儀式中：
* [[英格兰|英格兰]]和[[北愛爾蘭|北愛爾蘭]]使用《天佑吾王》；
* [[苏格兰|苏格兰]]使用《[[蘇格蘭之花|苏格兰之花]]》；
* [[威尔士|威尔士]]使用《[[父辈的土地|父辈的土地]]》。
故此若[[英格兰|英格兰]]隊及[[北愛爾蘭|北愛爾蘭]]隊对赛時便會出現只奏一次《天佑吾王》的有趣情況。
