{{作品地区用词|cn=天才在左，疯子在右|hk=瘋子的世界}}，是由高铭所著于2010年出版发行的[[精神病人|精神病人]]访谈手记<ref name="ifeng">{{cite web  |url = http://phtv.ifeng.com/program/ymxtx/detail_2013_09/14/29614328_0.shtml    |title =  高铭《天才在左，疯子在右》：与精神病人单独聊天三次被打   |publisher =  凤凰网   }}</ref>，并参选第六届国家图书馆文津图书奖<ref>{{cite web     |url =http://wenjin.nlc.gov.cn/2010/jj.jsp?bookid=2000144_51&flagtime=2010-09-14%2012:16:44     |title =文津图书奖     |deadurl =yes     |archiveurl =https://web.archive.org/web/20130321043707/http://wenjin.nlc.gov.cn/2010/jj.jsp?bookid=2000144_51&flagtime=2010-09-14%2012:16:44     |archivedate =2013-03-21     }}</ref>。

作者高铭生于1974年，本职为影视策划，2004年起因对精神病人群体的兴趣而开始此项访谈计划並持續了四年，本來先在天涯發表部份章節,後來完成该书並發佈。<ref name="ifeng"/>。

==内容==
《天才在左，疯子在右》以访谈录的形式记载了精神病患者及心理障碍者等边缘人视角独特的思想，當中有包含宗教、哲學、量子物理和價值觀的看法<ref>{{cite web     |url =http://www.xinqing.sjtu.edu.cn/index.php?option=com_content&view=article&id=439:2013-04-19-01-10-57&catid=64&Itemid=306     |title =上海交通大学心理咨询中心书籍推荐     |deadurl =yes     |archiveurl =https://web.archive.org/web/20141204175443/http://www.xinqing.sjtu.edu.cn/index.php?option=com_content&view=article&id=439:2013-04-19-01-10-57&catid=64&Itemid=306     |archivedate =2014-12-04     }}</ref>
<ref>{{cite web  |url = http://book.people.com.cn/n/2013/1223/c69401-23915454.html|title = 高圆圆推荐《天才在左，疯子在右》：一本精神病人访谈手记|publisher=人民网}}</ref>。
