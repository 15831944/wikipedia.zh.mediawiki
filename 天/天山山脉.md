'''天山山脉'''（[[维吾尔语|维吾尔语]]：'''{{lang|ug|تەڭرىتاغ}}''' ''Tengri Tagh/Тәңри тағ''；{{lang-kk|Тәңіртау/تأڭئرتاۋ}}；{{lang-ky|Ала-Тоо/تەڭىر-توو}}；[[东干文|东干文]]：{{Script|Cyrs|Тянсан}}，[[小儿经|小儿经]]：{{rtl-lang|ar|تِيًا صًا}}）是[[亚洲|亚洲]]中部最大的一条[[山脉|山脉]]，横亘[[中国|中国]][[新疆|新疆]]的中部、[[吉尔吉斯斯坦|吉尔吉斯斯坦]]和[[乌兹别克斯坦|乌兹别克斯坦]]，西端伸入[[哈萨克斯坦|哈萨克斯坦]]等四國。古名白山，又名雪山，冬夏有雪。故名，[[匈奴|匈奴]]谓之天山，[[唐|唐]]时又名折罗漫山，高达二万一千九百尺，长约2500km，宽约250～300km，平均海拔约5km。最高峰是[[托木尔峰|托木尔峰]]，海拔为7435.3m，在中国和吉尔吉斯斯坦边界<ref>{{cite web |url=http://discovery-kyrgyzstan.com/archive/2007/9_10.php |title=Victory Peak|website=Discovery Kyrgyzstan |access-date=30 January 2017}}</ref>；第二高峰是[[汗腾格里峰|汗腾格里峰]]海拔6995m，在中国、哈萨克斯坦和吉尔吉斯斯坦交界处<ref>{{cite web |url=http://www.summitpost.org/khan-tengri/150339 |title=Khan Tengri|website=SummitPost.Org |access-date=30 January 2017}}</ref>；[[博格达峰|博格达峰]]的海拔5445m，在中国境内。前两者是前苏联雪豹登山奖必要五峰之二<ref>{{cite web |url=http://www.summitpost.org/the-snow-leopards/235234 |title=Snow Leopards|website=SummitPost.Org |access-date=30 January 2017}}</ref>。新疆的三条大河——[[锡尔河|锡尔河]]、[[楚河|楚河]]和[[伊犁河|伊犁河]]都发源于此山。天山山脉把新疆分成两部分：南边是[[塔里木盆地|塔里木盆地]]；北边是[[准噶尔盆地|准噶尔盆地]]。天山由东西走向的褶皱断块山组成，山间有陷落盆地，如[[哈密盆地|哈密盆地]]、[[吐鲁番盆地|吐鲁番盆地]]，西部有[[伊犁|伊犁]][[谷地|谷地]]。

位于[[乌鲁木齐市|乌鲁木齐市]]以东的[[博格达峰|博格达峰]]海拔5445米，峰上的积雪终年不化，人们称它“雪海”。位于博格达峰山腰的[[天池_(天山)|天池]]，清澈透明，是新疆著名的旅游胜地。目前，博格达峰自然保护区已纳入联合国“人与生物圈”自然保护区网。[[托木尔峰|托木尔峰]]，海拔7435米，是天山的最高峰，登山界一般承认1956年阿巴拉科夫首次登顶成功<ref>{{cite web |url=http://peakware.com/peaks.php?pk=1013|title=Peak Pobeda Overview|website=Peakware |access-date=30 January 2017}}</ref>，但也有说1938年已有苏联登山队登顶<ref>{{cite web |url=http://www.advantour.com/kyrgyzstan/mountains/peak-pobeda.htm|title=Pobeda Peak|website=Avantour |access-date=30 January 2017}}</ref>；1975年7月25日首个中国登山队登顶成功。

== 地理概况 ==
[[File:天山垂直自然带.jpg|right]]
天山东西横亘2500余公里，南北宽度250-350公里，西起中哈边境，东至[[星星峽|星星峡]]，在中国境内绵延1700千米。占地57万多平方公里，占新疆全区面积约1/3。整个天山山系由3列山脉组成，由北往南分别称为北天山、中天山和南天山。天山山体由山地、山间盆地和山前平原三部分组成。

=== 地质成因推测 ===
天山属于比较年轻的山系，形成距今约二三百万年前。距今约1200-200万年前，天山在其演化中发生较为突出的变化，东西向分布，条状隆起，形成今天的规模。同时[[吐鲁番盆地|吐鲁番盆地]]也在此次演化中形成，并由于断裂后的长期[[沉降|沉降]]，成为中国海拔最低的地底，与[[博格达峰|博格达峰]]相映成趣。

=== 垂直自然带 ===
由于天山的海拔跨度，在形态上呈现出典型的垂直自然带分布。

=== [[天池_(天山)|天池]] ===
天池位于北天山中段的博格达山北坡。

=== 主要山峰 ===
* [[托木尔峰|托木尔峰]]
* [[汗腾格里峰|汗腾格里峰]]
* [[博格达峰|博格达峰]]，海拔5445米，与另外两座山峰构成了当地著名的“雪海三峰”，也成为新疆的象征之一。

== 世界遗产 ==
在2013年召开的第37届世界遗产大会上，'''新疆天山'''被正式列入[[世界遗产|世界遗产]]的自然遗产名录<ref>[http://news.163.com/13/0621/13/91T6F65N00014JB6.html 新疆天山成功申遗]</ref>。列入范围包括[[阿克苏|阿克苏]]的[[托木尔峰|托木尔]]、[[伊犁哈萨克自治州|伊犁哈萨克自治州]]的[[喀拉峻|喀拉峻]]-[[库尔德宁|库尔德宁]]、[[巴音郭楞蒙古自治州|巴音郭楞蒙古自治州]]的[[巴音布鲁克|巴音布鲁克]]和[[昌吉回族自治州|昌吉回族自治州]]的[[博格达峰|博格达]]四个区域<ref>[http://www.ts.cn/special/lyzt/2012-06/11/content_6920032.htm 新疆天山申遗进入冲刺阶段]</ref>。在2016年召开的第40届世界遗产大会上，位于[[吉爾吉斯斯坦|吉爾吉斯斯坦]]、[[烏茲別克斯坦|烏茲別克斯坦]]和[[哈薩克斯坦|哈薩克斯坦]]的'''西部天山'''被正式列入[[世界遗产|世界遗产]]的自然遗产名录<ref>{{cite web |url=http://whc.unesco.org/en/list/1490/ |title=Western Tien-Shan|publisher=United Nations Educational, Scientific and Cultural Organization|website=UNESCO World Heritage Centre|access-date=30 January 2017}}</ref>。

== 文化 ==
* 此区域曾为古代游牧民族的活动场所，今天留下了许多草原石人雕像。还有大量岩画、墓葬群。
* 20世纪50年代，中国作家[[碧野|碧野]]曾多次来天山地区采风，并创作出《天山景物记》等作品。
