'''天石門別神'''（{{lang-ja|アメノイワトワケノカミ}}）為《[[古事記|古事記]]》的記述，又記作'''櫛石窗神'''（{{ja}}クシイワマドノカミ）、'''豐石窗神'''（{{ja}}トヨイワマドノカミ），祂是[[日本神話|日本神話]]裡出現的神明；另有別名'''天岩間別神'''。

== 概要 ==
=== 古事記之記載 ===
依據《古事記》上卷記載的[[天孫降臨|天孫降臨]]故事中，[[天照大神|天照大御神]]賜[[八尺瓊勾玉|八尺瓊勾玉]]、[[八咫鏡|八咫鏡]]、[[天叢雲劍|天叢雲劍]]等[[三神器|三神器]]給[[瓊瓊杵尊|邇邇藝命]]，並遣[[思兼神|思金神]]、[[天手力男神|手力男神]]、'''天石門別神'''轉諭邇邇藝命曰：「八咫鏡就像我（意指天照大神）的魂魄，所以你祭拜這些物品就要像奉祀我一樣尊崇。我將會派遣思金神與你前往[[葦原中國|葦原中國]]，讓他輔佐你。」

於是邇邇藝命、思金神等二神，今祭於[[伊勢神宮|伊勢神宮]]內殿；登由宇氣神坐伊勢神宮外殿。'''天石戶別神'''又名'''櫛石窗神'''或'''豐石窗神'''，乃御門之神；[[天手力男神|手力男神]]則坐鎮在[[伊勢國|伊勢]]的佐那縣。

== 解說 ==
自古以來，天石門別神便是[[天皇|天皇]][[皇居|皇居]]四方之門的[[門神|門神]]。

== 信仰 ==
在日本，供奉天石門別神為主祭神的神社計有下述：
* 櫛岩窗神社（[[兵庫縣|兵庫縣]][[篠山市|篠山市]]）
* 天石立神社（[[奈良縣|奈良縣]][[奈良市|奈良市]]柳生町）
* 天岩門別神社（[[岡山縣|岡山縣]][[美作市|美作市]]）
* 大祭天石門彥神社（[[島根縣|島根縣]][[濱田市|濱田市]]相生町）
