'''天主教中国大陆主教团'''是[[中国天主教地下教会|中国天主教地下教会]]于1989年违反中华人民共和国法律成立的[[主教团|主教团]]，因忠于[[教宗|教宗]]而违反中华人民共和国的相关法律法规和政策，而遭到中华人民共和国政府的打击和惩处。

== 历史 ==
1988年2月，[[陕西|陕西]]的[[张刚毅|张刚毅]]神父受[[意大利|意大利]]政府邀请前往[[罗马|罗马]]领奖，期间得到了时任[[教宗|教宗]][[若望·保禄二世|若望·保禄二世]]的接见。教宗希望张刚毅返回中国后能与[[易县教区|易县教区]]主教[[刘冠东|刘冠东]]商讨在中国大陆成立忠于[[圣座|圣座]]的主教团的可行性。刘冠东在与张刚毅接洽后，开始与易县教区助理主教[[刘书和|刘书和]]一同筹备中国地下教会的主教会议。<ref name=chinacath>{{cite web|url=http://www.chinacath.com/article/guia/renwu/2013-11-21/24675.html|title=深切缅怀刘冠东主教|date=2013-11-21|author=彭鉴道|work=天主教在线|deadurl=yes|archiveurl=https://web.archive.org/web/20150203095923/http://www.chinacath.com/article/guia/renwu/2013-11-21/24675.html|archivedate=2015-02-03}}</ref>

1989年11月21日，多位忠于教宗的主教在陕西[[三原县|三原县]]张二册村举行了会议，决定成立“天主教中国大陆主教团”。会上推选[[保定教区|保定教区]]主教[[范学淹|范学淹]]（当时因被软禁而未到会）为主教团团长，刘冠东为执行主席，刘书和为秘书长。同时，当时在香港的[[广州总教区|广州总教区]]总主教[[邓以明|邓以明]]与在美国的[[上海教区|上海教区]]主教[[龚品梅|龚品梅]]被选为名誉团长。<ref name=chinacath /><ref name=hsstudyc>{{cite web |url = http://www.hsstudyc.org.hk/big5/china/b5_cinfo_hist.html|title=中国天主教会历史|work=天主教香港教區聖神研究中心}}</ref><ref>{{cite book |title=''Sino-Vatican Relations: Problems in Conflicting Authority, 1976-1986''|language=en|author=Beatrice Leung |year=1992 |pages=131-132 |publisher=Cambridge University Press}}</ref>

因“中国大陆主教团”的成立本身违反了中华人民共和国的相关法律法规和政策，从而受到了到了[[中华人民共和国政府|中华人民共和国政府]]的限制和处罚，参加三原会议的主教与神父先后遭到了逮捕或软禁，其中包括：<ref>{{cite book |title=''Freedom of Religion in China''|pages=18-21|author=Human Rights Watch/Asia|year=1992}}</ref><ref>{{cite web |url = http://www.ucanews.com/story-archive/?post_name=/1990/11/29/detention-marks-first-anniversary-of-underground-episcopal-conference&post_id=31614 |title=DETENTION MARKS FIRST ANNIVERSARY OF UNDERGROUND EPISCOPAL CONFERENCE |date=1990-11-29|work=UCANews}}</ref><ref>{{cite web |url = http://www.ucanews.com/story-archive/?post_name=/1991/12/27/last-free-bishop-of-clandestine-episcopal-conference-detained&post_id=40452 |title=LAST ´FREE´ BISHOP OF CLANDESTINE EPISCOPAL CONFERENCE DETAINED|date=1991-12-27|work=UCAnews}}</ref>
*[[易县教区|易县教区]]主教[[刘冠东|刘冠东]]：1989年11月26日被捕，后判处劳教三年。
*[[兰州总教区|兰州总教区]]总主教[[杨立柏|杨立柏]]：1989年12月底被捕，后被判劳教三年。
*[[秦州教区|秦州教区]]辅理主教[[陆振声|陆振声]]：1989年12月底被捕。
*[[献县教区|献县教区]]主教[[李振荣|李振荣]]：1991年底被捕。
*[[汉中教区|汉中教区]]主教[[余成悌|余成悌]]：1989年12月被捕，次年7月释放。
*[[天津教区|天津教区]]主教[[李思德|李思德]]：1989年12月9日被捕，拘押至1991年7月。
*[[齐齐哈尔监牧区|齐齐哈尔监牧区]]主教[[郭文治|郭文治]]：1989年12月14日被捕，次年3月释放。
*[[绥远教区|绥远教区]]助理主教[[姜立人|姜立人]]：1989年底被捕，次年4月释放。
*易县教区辅理主教[[师恩祥|师恩祥]]：1989年底被捕，1993年释放<ref>{{cite web |url = http://china.ucanews.com/2015/01/31/經歷十四年秘密監禁，師恩祥主教在關押中逝世/ |title=经历十四年秘密监禁，师恩祥主教在关押中逝世 |date=2015-01-31 |work=天亚社}}</ref>。

刘冠东于1992年获释。1995年，因病辞去易县教区主教、中国大陆主教团主席职务。此后他虽因病丧失自理能力，但仍旧被当局软禁。1997年，在部分教友的帮助下他逃脱了当局的监控，转移至秘密地点，开始流亡生涯。在16年的隐密生活后，他于2013年10月去世。为避免当局干扰，他被安葬于秘密地点。

1993年7月，齐齐哈尔监牧区的魏景义神父被选为主教团新任秘书长，以接替两个月前去世的首任秘书长刘书和主教<ref>{{cite web |url = http://www.ucanews.com/story-archive/?post_name=/1994/04/29/underground-bishops-conference-secretary-general-among-china-detainees&post_id=45172 |title = UNDERGROUND BISHOPS´ CONFERENCE SECRETARY GENERAL AMONG CHINA DETAINEES|date=1994-04-29|work=UCANews}}</ref>1995年，刘冠东主教因病辞去了主教团主席的职务<ref>{{cite web |url = http://cn.rfi.fr/中国/20131107-中国地下教会主教刘冠东/|title=中国地下教会主教刘冠东|date=2013-11-07|work=法广}}</ref>。其后，[[上海教区|上海教区]][[范忠良|范忠良]]主教出任中国大陆主教团主席。在范忠良主教于2014年3月去世后，主席一职现空缺<ref>{{cite web |url = http://china.ucanews.com/2014/03/17/「地下」主教團團長、上海教區范忠良主教安息主/ |date=2014-03-17|title=「地下」主教团团长、上海教区范忠良主教安息主怀 |work=天亚社}}</ref>。

1995年，易县教区刘冠东主教荣休，师恩祥接任主教一职。2001年4月耶稣受难节时他又在北京被捕，此后一直被秘密关押，直至2015年初在监禁中逝世。
