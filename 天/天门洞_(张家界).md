'''天门洞'''，位于[[湖南省|湖南省]][[张家界|张家界]][[天门山|天门山]]，高131.5米，宽57米，深60米，是世界海拔最高的天然穿山[[溶洞|溶洞]]<ref name="web1 " >{{cite web |url=http://mil.news.sina.com.cn/p/2006-03-02/1513354599.html|title=《张家界世界自然遗产之天门洞开》|accessdate=2006年3月2日|work=|publisher=新浪网|language=zh-hans}}</ref>
[[File:Tian_Menshan_Mountain_6.jpg|thumb]]

== 简史 ==
[[三国|三国]]时期[[吴国|吴国]][[永安_(東吳)|永安]]六年（263年），嵩梁山突然裂开一个洞口，[[吴王|吴王]][[孙休|孙休]]视为吉祥，故而命名“天门山”。<ref name=" web2">{{cite web|url=http://www.tianmenshan.com.cn/webinfo/xswh.asp|title=《天门山仙山文化》|accessdate=2006年|work=|publisher=天门山官方网|language=zh-hans}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

1999年12月，世界特技飞行大师驾驶飞机穿越天门洞，实现人类首次驾飞机穿越自然溶洞的壮举，轰动全球。<ref name=" web2" />

2006年3月，[[俄罗斯|俄罗斯]][[空军|空军]]特技飞行表演在天门山上演。曾计划以重型战斗机苏-27、苏-30穿越天门洞，但没有实现。<ref name=" web2" />

2007年11月18日，[[法国|法国]][[冒险家|冒险家]][[阿兰·罗伯特|阿兰·罗伯特]]徒手攀上天门山洞顶，耗时仅40分钟。<ref>{{cite web|url=http://hunan.news.163.com/07/1118/23/3TKAC7VC0055007K.html|title=《“蜘蛛人”阿兰·罗伯特成功攀上天门洞》|accessdate=2007-11-18|work=|publisher=网易|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20160305002654/http://hunan.news.163.com/07/1118/23/3TKAC7VC0055007K.html|archivedate=2016-03-05}}</ref>

2011年9月24日，[[美国|美国]]著名极限[[跳伞|跳伞]][[运动员|运动员]][[杰布·克里斯|杰布·克里斯]]在第一次试飞失败的情况下穿无动力翼装成功穿越张家界天门洞。<ref>{{cite web |url=http://news.163.com/photoview/00AP0001/17770.html#p=7EPLU8BN00AP0001|title=《美国“鸟人”成功穿越张家界天门洞》|accessdate=2011-09-25|work=|publisher=网易|language= zh-hans}}</ref>

== 地理 ==
天门洞高131.5米，宽57米，深60米，海拔1300米，史载有“天门吐雾”、“天门灵光”、“天门翻水”和“天门转向”的自然奇观。地方志记载天门洞有“天门十景”。<ref name="web1 " />

== 文化 ==
[[元朝|元朝]]末年，[[张兑|张兑]]辞官回乡遇见了[[杨辀|杨辀]]。杨辀绘制了天门山图，张兑给他赋诗配画:“天门洞云气通，江东峨嵋皆下风。”并放言：“他日功成期早归，相与为邻住山下。” <ref>{{cite web|url=http://www.tianmenshan.com.cn/webinfo/info_list.asp?infoid=72|title=《天门山：张兑》|accessdate=2008-01-28|work=|publisher=天门山官方网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20101221062018/http://tianmenshan.com.cn/webinfo/info_list.asp?infoid=72|archivedate=2010-12-21}}</ref><br />

[[清朝|清朝]]诗人[[俞已谟|俞已谟]]赋诗说：“莫畏山高空仰止，此中真有上天梯。”
