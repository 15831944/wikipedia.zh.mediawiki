'''天恩堂'''是[[中国|中国]][[上海|上海]]一座被占用的古老教堂，原由[[美国圣公会|美国圣公会]]差会开辟，原址位于今[[黄浦区|黄浦区]]南部（原[[南市区_(上海市)|南市区]]）的松雪街。

天恩堂的历史源于1844年。美国圣公会传教士[[文惠廉|文惠廉]]初抵上海，在县城中部的虹桥堍设立基督堂。1899年，另一位传教士[[汤蔼礼|汤蔼礼]]将其迁往松雪街。<ref>{{cite web

1958年献堂献庙运动时，天恩堂最初仍被作为邑庙区联合礼拜教堂加以保留<ref>{{cite web| last =| first =| authorlink =| title =上海通志 >> 第十四卷民族、宗教 >> 第六章基督教 >> 第二节 差会、教派、教堂| url =http://www.shtong.gov.cn/newsite/node2/node2247/node79044/node79331/node79340/userobject1ai103718.html| accessdate =2010-05-12| deadurl =yes| archiveurl =https://web.archive.org/web/20110805001404/http://www.shtong.gov.cn/newsite/node2/node2247/node79044/node79331/node79340/userobject1ai103718.html| archivedate =2011-08-05}}</ref>。不过不久也献给[[人民政府|人民政府]]，改为上海联合汽配厂。目前建筑尚存，由松雪街幼儿园占用。
