《'''天空侵犯'''》是由三浦追儺原作、大羽隆廣作畫的[[日本漫畫|日本漫畫]]作品。由[[讲谈社|讲谈社]]出版<ref>{{Cite web |url=http://natalie.mu/comic/news/165610 |title=DeNA「マンガボックス」、連載作品が初単行本化 |publisher=ITmediaニュース |date=2014-05-09 |accessdate=2016-12-28}}</ref>，截至2019年4月已出版20冊漫畫。

== 故事概要 ==
高中生本城遊理意外進入了一個莫名的世界，這裡只有高樓大廈與連結的吊橋，還有戴著面具的人在她面前殺人。遊理為了逃離面具人的追殺，迅速逃跑，然而想往下走卻看到通知：不可回到地面，所以只能再度回到高樓樓頂。

遊理拿出手機打給哥哥，哥哥接起了電話，而且哥哥也在這莫名的世界中。本城遊理為了能見到她最敬愛的哥哥，而在這世界努力克服自己的恐懼，將各式面具人擊倒並尋找隊友。兄妹倆分別在不同地方集結隊友，對抗這世界的各種怪象，兄妹倆究竟能否見面呢……

== 登場人物 ==
=== 主要人物 ===

; {{nihongo|本城遊理|本城 遊理（ほんじょう ゆり）}}
: 女主角。女高中生16歲，對於哥哥的要求逼自己一定要做到，所以哥哥說(妳一定不要出事)，而決心要平平安安活下去。
: 其後在營救二瀨真由子因戴上破損的笑臉面具而進入「休眠模式」時，在服務員面具人的指引下得到無口面具，成為近神者。

; {{nihongo|二瀨真由子|二瀬 真由子（にせ まゆこ）}}
: 本作女配角。與遊理在戰鬥中相識，對於遊理從原本的不信任，到最後有著超越親情般的羈絆，與遊理一同對抗面具人。
: 曾一度為了對抗面具人的攻擊而戴上了破損的笑臉面具而進入「休眠模式」，其後在新崎九遠以近神者能力的強制命令下而解除此狀態。

; {{nihongo|本城理火|本城 理火（ほんじょう りか）}}
: 女主角的哥哥。男高中生18歲，名字在面上意思以理制火，因為名字難聽而自己十分討厭，使用大錘，想早日見到妹妹。

; {{nihongo|城勇火（狙擊手面具人）|スナイパー仮面}}
: 一開始便被迫戴上面具，成為面具狙擊手，在與遊理的戰鬥過程中面具破裂，而逐漸恢復記憶，對於近神者的九遠有著想保護她的情感。

; {{nihongo|新崎九遠|新崎 九遠（しんざき くおん）}}
: 近神者之一，擁有操控「超電磁砲」權限的能力。

== 用語解說 ==
'''面具'''

面具背後有著像電腦程式般的代碼，若人看到面具背面，便會像著了魔般，失去自己身體的控制權（僅笑臉面具），對於面具的命令唯命是從，就算是要殺人抑或是自殺都不可違抗，這世界的面具有四種分別為：

'''''笑臉面具'''''   
: 又被稱為「天使的裝備」，由無表情面具人隨機放置在各處，得到者只要自身臉部靠近至一定的極近距離，就會被內涵的程式慫恿而戴上，最後被程式操控成為面具人而攻擊其他人。

'''''面具人'''''
: 即被笑臉面具控制的人。又被稱為「天使」。面具人沒有特定名稱，幾乎以佩戴的裝備或者打扮而稱呼之。
: 面具人無法對近神者攻擊，甚至可被近神者控制。但對一般人卻有著迫使其自殺或殺害的任務。即使面具破損依然能使戴上者被操控，但會因為程式指令不完整而不能完整執行程式指令，而進入「休眠狀態」。


'''''憤怒面具'''''    
: 管理者方戰鬥人員，又被稱為「守護天使」，除非近神者大幅接近神，或被攻擊，不然並不會主動出擊。分別有等級1~9的面具人。


'''''無表情面具'''''   
: 管理者方補給人員，負責運送物資給所有人。會將食物、水、笑臉面具放置在各處。


'''''無口面具'''''    
: 隨機出現在領域內的特定地方的面具，數量極少，得到者都有一定的運氣，只有管理者、守護天使知道所有無口面具所在處。
: 戴上後便可近神者，當程式執行完後會自動破壞面具，因此可以真面目出現，以成為神為首要目的，也是結束這世界的關鍵角色。

'''''近神者'''''
: 普通人戴上無口面具後的狀態，能力比一般人、面具人高上許多。能力強大的近神者甚至會被管理者方給予「稱號」識別。
: 近神者會因個人素質不同而獲得不同的能力與權限，只要自身有所成長即有可能使新的能力覺醒。多數近神者都擁有操控面具人的能力。
: 近神者可以將自身能力及權限移轉給自己所操控的面具人，但僅限於自身快死亡時或者死亡後才能將能力和權限進行移轉。
