{{日本作品|天津御空！|あまつみそらに！}}是[[Clochette|Clochette]]在2010年5月28日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]<ref>{{cite web |title=あまつみそらに！ 初回限定版 |url=http://www.getchu.com/soft.phtml?id=661172 |website=Getchu.com |language=ja}}</ref>，後來由PROTOTYPE在[[PlayStation_Portable|PlayStation Portable]]和[[PlayStation_Vita|PlayStation Vita]]發售並更名為《天津御空！雲之盡頭》（{{lang|ja|あまつみそらに! 雲のはたてに}}）。<ref>{{cite web |title=《天津御空！雲之盡頭》2012年2月16日發售 來確認一下遊戲的概要吧 |url=https://gnn.gamer.com.tw/4/62164.html |website=巴哈姆特電玩資訊站 |language=zh-tw |date=2012-02-04}}</ref><ref>{{cite web |title=PS Vita版「あまつみそらに！ 雲のはたてに」のOPムービーが公開に。ショップ別オリジナル特典情報も |url=https://www.4gamer.net/games/300/G030030/20150914064/ |website=4gamer.net |language=ja |date=2015/09/14}}</ref>

==故事==

==角色==
聲優順序為PC／PSP、PSV
===主要角色===
;{{nihongo|觀崎高久|観崎 高久}}
:本作的男主角，巡宮學園的2年級學生。父母已過世，從小與妹妹美唯和神奈同住。
;{{nihongo|一橋神奈|一ツ橋 神奈|聲優：[[櫻川未央|櫻川未央]]／[[小林真紀|小林真紀]]}}
:高久的同學，美空島的神，真名是「邇岐志神奈媛命」。
;{{nihongo|觀崎美唯|観崎 美唯|聲優：三菱アイ／[[清水愛|清水愛]]}}
:高久的妹妹和學妹，三年前出道的全國知名偶像。
;{{nihongo|清澄芹夏|清澄 芹夏|聲優：[[松田理沙|松田理沙]]／[[力丸乃梨子|力丸乃梨子]]}}
:高久的學妹，美唯的親友，網球社的成員。
;{{nihongo|帷千紗|帷 千紗|聲優：木村あやか／[[豬口有佳|豬口有佳]]}}
:高久的青梅竹馬和同學，民宿「帷家」女老闆的女兒，[[弓道|弓道]]社的成員。
;{{nihongo|葉月深景|葉月 深景|聲優：[[風音|風音]]／[[櫻井浩美|櫻井浩美]]}}
:從[[鎌倉市|鎌倉市]]搬到美空島的3年級轉學生，上纳神社的巫女。
;{{nihongo|音無夏帆|音無 夏帆|聲優：七原ことみ／鈴本かおり}}
:高久的同學，網球社的社長。PC版是配角，遊戲機版升格為女主角之一。
;{{nihongo|隱神穗乃里|隠神 穂乃里|聲優：[[福圓美里|福圓美里]]}}
:[[隱神刑部|隱神刑部]]的獨生女。遊戲機版的新角色。

===其他角色===
;{{nihongo|音無京佳|音無 京佳|聲優：新堂真弓}}
:夏帆的姊姊，甘味處「おとなし」的老闆。
;{{nihongo|榛名優依|榛名 優依|聲優：ヒマリ／朋永真季}}
:高久的班級老師，剛上任三個月負責教日本歷史。後來被高久說服兼任網球社的顧問。
;{{nihongo|吉野川姬|吉野川 姫|聲優：[[牧泉美|牧泉美]]／[[柳瀨夏美|柳瀨夏美]]}}
:高久的學姊。
;{{nihongo|根津満廣|根津 満広|聲優：八神大輔／[[野宮一範|野宮一範]]}}
:高久的青梅竹馬和同學，游泳社的成員。

==評價==
天津御空！在[[萌系遊戲大賞|萌系遊戲大賞]]2010中獲得情色系作品獎中的金獎。<ref>{{cite web |title=萌えゲーアワード2010 結果発表特設ページ |url=http://moe-gameaward.com/prize/2010/result2.html#bumon9 |website=萌えゲーアワード |language=ja}}</ref>另外在由Getchu.com所舉辦的美少女遊戲大獎2010中獲得系統部門第14名、繪圖部門第13名、情色部門第3名。<ref>{{cite web |title=美少女ゲーム大賞2010 ユーザー投票結果発表!! |url=http://www.getchu.com/pc/2010_g_ranking/ |website=Getchu.com |language=ja}}</ref>
