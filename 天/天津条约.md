《'''天津條約'''》包括《'''中俄天津条约'''》、《'''中美天津条约'''》、《'''中英法天津条约'''》，是指[[1858年|西元1858年]] (清[[咸丰|咸丰]]8年) [[清朝|清朝]]政府在[[第二次鴉片戰爭|第一次英法聯軍]]戰敗後與[[俄罗斯帝国|俄國]]、[[美國|美國]]、[[英國|英國]]、[[法國|法國]]在[[天津|天津]][[海光寺|海光寺]]所簽訂的的一系列[[不平等條約|不平等條約]]。清朝委派大學士[[桂良|桂良]]和各國代表談判並簽約。中方的真確文本原存於[[中華民國外交部|中華民國外交部]]，現存於[[臺北市|臺北]][[外雙溪|外雙溪]][[國立故宮博物院|國立故宮博物院]]恆溫恆濕的庫房保存。

== 对俄国 ==
{{Wikisource}}
清朝代表[[桂良|桂良]]和俄國代表於6月13日簽約，共12款。
# 向俄国增开上海、宁波、福州、厦门、广州、台湾（台南[[安平港|安平舊港]]）、琼州七处通商口岸，俄国兵船可在各口岸停泊。
# 扩大陆路通商，嗣后对俄国陆路通商人数、所带货物与资本，不加限制。
# 俄国得在各通商口岸设领，俄人若与中国人发生纠葛或其他事故，由两国官员“会同办理”。俄人在华犯罪，按俄国法律受审。
# 俄国人在内地传教，中国方面不得禁止。
# 中国今后给予别国的一切政治、贸易及其他特权，“毋庸再议，即与俄国一律办理施行”。
# 由两国派员查勘“从前未经定明边界”，“务将边界清理补入此次和约之内”。

== 對美國 ==
清朝代表[[桂良|桂良]]和美國代表於6月18日簽約，共30款。
# [[清政府|清政府]]若就[[公使|公使]]駐[[顺天府|京]]問題與別國另有應允或立约，美国同時享受同等權利。
# 若美國官船駛至中國近海，清朝應就採買食物、汲取[[淡水|淡水]]、修理船隻等給予協助。若美國船隻被毁、被劫，應准許美国官船追捕盗贼。若美国人受到匪徒侵害，地方官须立即派兵驅逐匪徒，嚴拿治罪，以保护美国人。
# 增开[[潮州|潮州]]（后改[[汕头|汕头]]）、[[台南|台南]]（台南[[安平港|安平舊港]]）為[[通商口岸|通商口岸]]。美国人可在通商口岸居住，或租地自行建楼以及设立医院、教堂及墓地等。美国的官员及人民可以雇佣清朝買辦、厮役、工匠、水手、引水，可以延纳漢人教授语言及[[幫辦|幫辦]]文墨，地方官民均不得稍有阻挠、陷害。
# 對於傳教士，地方官当一体保护，他人毋得骚扰，即“宽容条款”{{NoteTag|[[美北长老会|美北长老会]]传教士、传教史学者丹尼斯在《近代传教事业的号召》一书中说，“著名的‘宽容条款'的订入条约，就是得力于卫三畏博士的，后来也订入了英国条约中。”}}。“宽容条款”的要害是，不仅外国传教士，连中国信徒也受不平等条约的保护。
# 嗣後清朝有何惠政、恩典、利益施及他国及其商民，美国官民一体適用{{NoteTag|即[[最惠國|最惠國]]地位。}}。

== 對英國、法國 ==
清朝代表桂良和英法代表於6月26、27日簽約，各有56、42款，再加上附約。
# 增開牛莊（后改[[营口|营口]]）、[[登州|登州]]（后改[[烟台|烟台]]）、台灣（今台南[[安平港|安平舊港]]）、[[淡水港|淡水]]、[[打狗|打狗]]（今[[高雄港|高雄港]]）、雞籠（今[[基隆港|基隆港]]）、[[潮州市|潮州]]（后改[[汕头|汕头]]）、[[瓊州|瓊州]]、[[南京|南京]]及[[镇江市|鎮江]]、[[漢口|漢口]]、[[九江|九江]]為通商口岸。
# 英法人士可在內地遊歷及傳教。
# 英法商船可以在長江各口往來。
# 英法人士在華犯罪，享有領事裁判權。
# [[關稅|關稅]]由雙方協定，每十年修訂一次。
# 雙方互派公使；外使可行西禮，並進駐北京。
# 清朝賠償英國四百萬兩、法國二百萬兩白銀。

附約亦規定：
# [[鴉片|鴉片]]改稱洋藥，可自由買賣及進口。
# [[海關|海關]]聘用英人幫辦稅務。
# 修改稅則，以「值百抽五」(即5%)為原則。
