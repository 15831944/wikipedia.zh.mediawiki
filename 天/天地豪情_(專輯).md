《'''天地豪情'''》是[[黎明|黎明]]的[[音樂專輯|音樂專輯]]<ref>{{cite web|url= http://jackli51.pixnet.net/blog/post/152932761-%E8%A9%B1%E8%AA%AA%E5%AF%B6%E9%BA%97%E9%87%91%E5%94%B1%E7%89%87%E5%87%BA%E7%89%88%E7%9B%AE%E9%8C%84%28%E5%9B%9B%29 |title=話說寶麗金唱片出版目錄(四)|website= 流行音樂言舊苑-痞客邦 }}</ref>，於1995年5月推出<ref>{{Cite web |url =http://mypaper.pchome.com.tw/ileenho/post/1247516836|title =黎明(Leon Lai)唱片作品年表|date = 2005年5月27日 |publisher =壽司店的肥美亞彌-PChome 個人新聞台|language = 中文}}</ref><ref>{{Cite book|title=當世事再沒完美：香港流行樂壇1974-2013|last=陳怡東|first=|publisher=法律出版社|year=2015|isbn=7511880754|location=|pages=324}}</ref>，專輯內有十首粵語歌曲，當中八首是本土原創作品<ref>{{Cite web |url =http://allaboutleon.com/index.cgi?md=read;id=34284|title =如何評價黎明？|date = 2017年4月26日 |publisher =allaboutleon.com|language = 中文}}</ref>，分別由[[林慕德|林慕德]]、[[江志仁|C.Y.Kong]]、[[唐奕聰|唐奕聰]]及[[譚詠麟|譚詠麟]]作曲，《月亮下求你一吻》及《Just For Fun》是翻唱作品。
