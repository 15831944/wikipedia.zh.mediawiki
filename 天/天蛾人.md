'''天蛾人'''（{{lang-en|Mothman}}）又稱'''飛蛾人'''，是一種[[美国|美国]][[民俗|民间传说]]中的[[神秘生物列表|神秘生物]]，通常被描述为长有翅膀以及红色眼睛的生物。有理论认为天蛾人是外星生命，并且与[[黑衣人|黑衣人]]有关联。

1966年11月12日至1967年12月15日，據報導說在[[西維吉尼亞州|西維珍尼亞州]]的{{link-en|波因特普萊森特 (西維珍尼亞州)|Point Pleasant, West Virginia|波因特普萊森特}}看到天蛾人。第一份報道出版於1966年11月16日的報紙《Point Pleasant Register》，標題為「Couples See Man-Sized Bird ... Creature ... Something」<ref>{{Cite web |url = https://web.archive.org/web/20071011230219/http://www.westva.net/mothman/1966-11-16.htm |title = Couples See Man-Sized Bird...Creature...Something |date = 1966-11-16 |publisher = Point Pleasant Register |language = en}}</ref>。全國媒體很快就獲得報導，並且幫助將這個故事傳播至整個美國。1970年，{{link-en|格雷·百克|Gray Barker}}向廣泛觀眾介紹了天蛾人。

== 目击 ==
*1966年11月12日，5名目击者在美国西弗吉尼亚州克伦德宁（Clendenin）一处墓地目击看见一个像男人的身影从树林掠过。

*1967年12月15日，{{link-en|銀橋|Silver Bridge}}发生垮塌，造成46人死亡。作家{{link-en|約翰·基爾|John Keel}}在1975年出版的书籍《{{link-en|天蛾人的预言|The Mothman Prophecies}}》中将天蛾人与{{link-en|银桥事故|Silver Bridge|银桥}}关联，并形成“天蛾人预言灾难”的都市传说，但1971年的官方调查已确认银桥是由于{{link-en|Eyebar|Eyebar}}结构{{link-en|应力腐蚀开裂|Stress corrosion cracking}}引发的倒塌事故。<ref>{{Cite web|url=https://news.google.ca/newspapers?id=cJ1eAAAAIBAJ&sjid=MVMNAAAAIBAJ&pg=4537,3056163|title=The Portsmouth Times - Google News Archive Search|accessdate=2018-09-16|work=news.google.ca}}</ref>
