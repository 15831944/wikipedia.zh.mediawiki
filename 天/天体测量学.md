'''天体测量学'''或'''測天學'''（Astrometry）是[[天文学|天文学]]中最古老也是最基礎的一個分支，主要以測量[[恆星|恆星]]的位置和其他會運動[[天體|天體]]的距離和動態。他是傳統[[科學|科學]]中的一個子科目，後來發展出以定性研究為主體的[[位置天文學|位置天文學]]。天文測量學的歷史，在西方可以追溯到[[喜帕恰斯|喜帕恰斯]]（Hipparchus），他編輯了第一本的[[星表|星表]]，列出了肉眼可見的恆星並發明了到今天仍沿用的[[視星等|視星等]]的尺標。現代的天體測量學建立在[[弗里德里希·威廉·贝塞尔|白塞耳]]的[[基本星表|基本星表]]上，這是以[[布拉德雷|布拉德雷]]在西元1750至1762年間的測量為基礎，提供了3,222顆恆星的平均位置。

除了提供[[天文學家|天文學家]]基本的[[參考座標系|參考座標系]]作為她們在天文觀測報告之用外，天文測量學也是[[天體力學|天體力學]]、[[恆星動力學|恆星動力學]]和[[星系天文學|星系天文學]]等學門的基礎。在[[觀測天文學|觀測天文學]]中，天文測量的技術協助鑑別出各種天體獨特的運動。他的設備也用於[[守時|守時]]（keeping time），因為[[協調世界時|協調世界時]]（UTC）是在確切觀測[[地球自轉|地球自轉]]的基礎上，以[[閏秒|閏秒]]的調整與[[國際原子時|原子時間]]取得協調與一致。天文測量學也與極端複雜的[[宇宙距離尺度|宇宙距離尺度]]有所關聯，因為他用於建立[[視差|視差]]以估計[[銀河系|銀河系]]內恆星的距離。

== 发展历史 ==
古时候人们为了辨别[[方向|方向]]、确定[[时间|时间]]，创造出[[日晷|日晷]]和[[圭表|圭表]]来。古代天文学家为了测定星星的[[方位|方位]]和[[运动|运动]]，又设计制造了许多[[天体测量|天体测量]]的仪器。通过对星空的观察，将星空划分成许多不同的[[星座|星座]]，并编制了[[星表|星表]]。通过对天体的测量和研究形成了早期的天文学。直到十六世纪中叶，[[哥白尼|哥白尼]]提出了日心体系学说，從只是单纯描述天体[[位置|位置]]、运动的经典天体测量学，发展成寻求造成这种运动力学机制的[[天体力学|天体力学]]。

== 天文測量學的進展 ==
* [[日晷|日晷]]曾有效的測量時間。
** 中國最古老的[[河南登封觀星台|河南登封觀星台]]是大型的圭表，在沒有鐘錶和日曆的時代，以日影的長短來訂四時。
* [[星盤|星盤]]被發明用來測量天體的高度角。
* 天體測量的應用導致[[球面幾何學|球面幾何學]]的發展。
* [[第谷|第谷]]小心的測量[[行星運動|行星運動]]，導致[[開普勒|刻卜勒]]推論出地球繞著[[太陽|太陽]]公轉的[[哥白尼原理|哥白尼原理]]。
* [[六分儀|六分儀]]戲劇化的被用於測量天體間的角度。
* [[布拉德雷|布拉德雷]]以精確的[[中星儀|中星儀]]測量出[[光行差|年周光行差]]，證明了地球繞日公轉。
* [[電荷耦合元件|電子耦合放大器]]（CCD）的發展，並且在1980年代被天文學家所接受，改進了專業天文學家在觀測工作上的精確度。
* 發展出低價位的電子耦合放大器與應用軟體，並且大規模的應用在望遠鏡上，使得[[業餘天文學|業餘天文學家]]也能夠觀察和發現[[小行星|小行星]]。
* 從1983至1993年，歐洲太空總署的[[依巴谷衛星|依巴谷衛星]]（Hipparcos）進行的天體位置測量，編製了精確至20-30微角秒，超過百萬顆恆星的位置表。

== 天文測量 ==
'''天文測量'''是[[測量|量度]][[恆星|恆星]]和[[行星運動|行星運動]]的科學。在1990年代，天文測量被用於檢測[[軌道|軌道]]繞著個別地[[外太陽系|外太陽系]]的[[氣體巨星|氣體巨星]]。經由觀察[[恆星擺動|恆星擺動]]和計算造成這種擺動所需的的[[重力|重力]]，然後可以推算造成這種影響的行星的質量。

== 研究对象 ==
天文學家利用天體測量的技術來追蹤[[近地小行星|近地小行星]]，也利用天體位置微小的週期性變動，這是行星與恆星互繞質量中心產生的位置偏移，用來搜尋[[系外行星|系外行星]]。NASA計畫在[[太空干涉儀任務|太空干涉儀任務]]（SIM行星搜尋）中，應用天體測量的技術來偵測在200光年的距離內，或是最接近的[[類太陽恆星|類太陽恆星]]中，可能存在的[[類地行星|類地行星]]。

天體測量學的測量結果被用來修正[[天文物理學家|天文物理學家]]在[[天體力學|天體力學]]下建立的一些模型。基於測量得到的[[中子星|中子星]]速度，可能會導致[[超新星|超新星]]爆炸是[[非對稱|非對稱]]的結論。同樣的，天體測量的結果也用於確認[[暗物質|暗物質]]在星系內的分布狀態。

== 研究方法 ==
通过研究天体投影在天球上的坐标，在天球上确定一个基本参考坐标系，来测定天体的位置和运动，这种参考坐标系，就是星表。在实际应用中，可用于大地测量、地面定位和导航。地球自转和地壳运动，会使天球上和地球上的坐标系发生变化。为了修正这些变化，建立了时间和极移服务，进而研究天体测量学和地学的相互影响。

古代的天体测量手段比较落后，只能凭肉眼观测，对于天体测量的范围有限。随着时代的发展，发现了[[红外线|红外线]]、[[紫外线|紫外线]]、[[X射线|X射线]]和[[γ射线|γ射线]]等波段，天体测量范围从可见光观测发展到肉眼不可见的领域，可以观测到数量更多的、亮度更暗的[[恒星|恒星]]、[[星系|星系]]、[[射电源|射电源]]和[[红外源|红外源]]。随着各种精密测量仪器的出现，测量的[[精度|精度]]也逐渐提高。

== 历史成就 ==
* 远古时候，并没有现在的时钟和日历，人们通过对太阳的观察，发明了日晷，根据阴影的长短来判断时间。
* 為紀念這一學科的重要性，[[小行星25000|小行星25000]]被稱為“天體測量”（Astrometria）。
