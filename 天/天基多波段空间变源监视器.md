'''天基多波段空间变源监视器'''（英语：{{Lang|en|Space-based multi-band astronomical Variable Objects Monitor}}<ref>{{Cite web|url=http://www.svom.fr/#filter=.missionarticle.portfolio366load|title=Sursauts gamma, des messages venus du passé|accessdate=2019-04-04|author=|date=|publisher=|language=fr}}</ref>，缩写为'''SVOM'''）是一个中法合作的空间观测项目，目标是探测[[伽马射线暴|伽马射线暴]]并确定其特征。SVOM由[[中国国家航天局|中国国家航天局]]和[[法國國家太空研究中心|法国国家空间研究中心]]共同研制，并计划于2021年发射<ref>{{Cite web|url=https://cn.ambafrance.org/article26528|title=中国国家航天局与法国国家空间研究中心将共同执行SVOM天体物理科学任务|accessdate=2019-04-04|author=|date=2014-08-19|publisher=}}</ref>。

== 背景 ==
SVOM的主要仪器借用了被法国国家空间研究中心放弃的项目ECLAIRs。中国航天局将提供[[卫星平台|卫星平台]]和SVOM上的两个仪器。法国国家空间研究中心将提供另外两个主要仪器（包括ECLAIRs）。 

== 任务概述 ==
通过空间仪器与地面仪器的协同工作，该项目主要将：
*探测所有已知类型的伽玛射线暴
*准确快速地探测伽玛射线暴的位置
*快速识别伽玛射线暴在X射线和可见光波段的余辉，包括高度红移的波段（z>6）

== 科学仪器 ==
卫星上有四个主要仪器<ref>{{Cite web|url=https://svom.cnes.fr/fr/SVOM/Fr/GP_satellite.htm|title=SATELLITE|accessdate=2019-04-04|author=|date=|publisher=}}</ref>，能够覆盖从伽马射线、X射线到近红外线的波段。

== 地面工作 ==
为了对伽马射线暴进行详细研究，地面观测工作将同步进行。由于伽玛射线暴的持续时间非常短暂，SVOM探测到伽玛射线暴的位置后会立刻将其坐标信息传输到地面上更为精准的观测站。通过40个[[甚高频|甚高频]]天线组成的、可以完全覆盖低轨道卫星通过区域的网络，这些数据将及时被传输到位于[[萨克雷_(法国)|萨克雷]]的法国科学中心（French Science Center）<ref>{{Cite web|url=http://www.svom.fr/#filter=.instrumentsarticle.portfolio219load|title=Le réseau d’alerte|accessdate=2019-04-04|author=|date=|publisher=|language=fr}}</ref>。数据经处理后将被发给两台自动控制的地面跟踪望远镜（Ground Follow-up Telescopes）。
