'''天水之戰'''是[[中國|中國]][[三国|三国]]时期，於228年春季在[[中国西北地區|中国西北地區]]所爆发的戰役。[[蜀汉|蜀汉]]军队在[[丞相|丞相]][[诸葛亮|诸葛亮]]率领下，计划夺取敌国[[曹魏|曹魏]]的战略城市[[长安|长安]]。漢军一度取得[[南安|南安]]、[[天水|天水]]及[[安定郡|安定]]三郡，但在[[街亭之战|街亭之战]]后失去；正如《[[三国志|三国志]]》中魏将[[张郃|张郃]]的传记所言，南安、天水、安定郡叛亂投靠诸葛亮，张郃将他们全部平定。<ref>《三国志》卷一十七张郃传：南安、天水、安定郡反應亮，郃皆破平之。</ref>

== 背景 ==
在[[汉中|汉中]]的战事会议上，诸葛亮提出於左翼以优势兵力夺取[[渭河|渭河]]河谷上游的天水、安定、南安、[[祁山|祁山]]，進而夺取长安。丞相司马[[魏延|魏延]]反对，建议率一万精兵从[[秦岭|秦岭]]中的[[子午谷|子午谷]]奇袭占领长安，但诸葛亮认为这太冒險而拒绝了。

== 過程 ==
228年春，诸葛亮扬言要经[[斜谷|斜谷]]进军攻取[[郿|郿]]。他派镇东将军[[赵云|赵云]]、扬武将军[[邓芝|邓芝]]为疑兵，作威胁郿、攻取冀城状。[[曹真|曹真]]率本部军阻止他们。诸葛亮亲自率军围[[祁山|祁山]]，行伍齐整，赏罚整肃，号令严明。南安、天水、安定三郡都叛离曹魏，投靠诸葛亮，[[关中|关中]]震动。[[曹叡|曹-{叡}-]]亲自坐镇长安监督防御。曹真守郿对抗赵云，同时张郃率马步军共5万人西进对抗诸葛亮主力大军。

此时中郎将[[姜维|姜维]]随长官天水太守[[马遵|马遵]]在郊区巡逻。马遵害怕姜维和蜀军串通，趁夜遁去[[上邽|上邽]]。姜维发现时为时已晚，回上邽时，被自己人拒之门外。于是姜维和功曹[[梁绪|梁绪]]、主簿[[尹賞_(三國)|尹赏]]、主记[[梁虔|梁虔]]等轉投诸葛亮。

其实在天水没有战事，只有響應，周边地区很快投降蜀汉，使蜀军得以稳步进军，但在[[街亭|街亭]]遭到反擊，张郃击败诸葛亮派去迎敌的蜀参军[[马谡|马谡]]，迫使诸葛亮下令撤回蜀国境内。同时，奉诸葛亮命迎战曹真的赵云、邓芝军力不足而在[[箕谷|箕谷]]战败，但收敛部众而避免了大败，赵云率部撤退。起事响应诸葛亮的三郡后被张郃平定，重归魏掌控。

== 演义 ==
在14世纪的[[罗贯中|罗贯中]]历史小说《[[三国演义|三国演义]]》第九十二和九十三回中，姜维是诸葛亮继续北伐的原因之一，在姜维与赵云快速交战后，让姜维归蜀也成为诸葛亮的一个目的。诸葛亮派赵云当先，在一场小规模战斗和一些筹划后亲临战阵。战斗期间，姜维长官马遵怀疑姜维通敌。姜维在天水城外时，马遵关闭城门不让姜维进入，姜维无奈叛投诸葛亮。
