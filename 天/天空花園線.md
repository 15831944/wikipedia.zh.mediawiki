'''天空花園線'''（{{lang-en|Skypark Link}}）是[[馬來亞鐵道公司|馬來亞鐵道]]<ref>{{cite web |author=priya menon |url=http://www.thestar.com.my/News/Community/2014/08/08/Full-steam-ahead-Work-on-railway-line-from-Subang-airport-to-KL-Sentral-has-begun/ |title=Work on railway line from Subang airport to KL Sentral has begun - Community | The Star Online |publisher=Thestar.com.my |date=2014-08-08 |accessdate=2015-03-17 |language=en |deadurl=no |archiveurl=https://web.archive.org/web/20150402145437/http://www.thestar.com.my/News/Community/2014/08/08/Full-steam-ahead-Work-on-railway-line-from-Subang-airport-to-KL-Sentral-has-begun/ |archivedate=2015-04-02 }}</ref><ref>{{cite web |url=http://www.spad.gov.my/ms/projek/projek-akan-datang/projek-landasan-keretapi-dari-subang-ke-terminal-skypark-subang |title=PROJEK LANDASAN KERETAPI DARI SUBANG KE TERMINAL SKYPARK SUBANG | Laman Web Rasmi Suruhanjaya Pengangkutan Awam Darat |publisher=S.P.A.D. |date= |accessdate=2015-03-17 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20150924104345/http://www.spad.gov.my/ms/projek/projek-akan-datang/projek-landasan-keretapi-dari-subang-ke-terminal-skypark-subang |archivedate=2015-09-24 }}</ref>所屬由[[馬來西亞|馬來西亞]][[吉隆坡|吉隆坡]]市中心往返[[蘇丹阿都阿兹沙機場|蘇丹阿都阿兹沙機場]]（舊稱梳邦機場）之間的特快列車服務。該線是大吉隆坡/巴生谷綜合交通系統的一部分，路線編號為10號，識別色為棕色。本線與[[巴生港線|巴生港線]]共軌，通過[[梳邦再也站|梳邦再也站]]之後轉往終端站{{link-en|天空花園航站樓站|Terminal Skypark Komuter station}}。巴生港線列車靠停在[[吉隆坡中央車站|吉隆坡中央車站]]至[[梳邦再也站|梳邦再也站]]之間的所有車站，而天空花園線只有停靠[[吉隆坡中央車站|吉隆坡中央車站]]、[[梳邦再也站|梳邦再也站]]與新增的3個車站。这条线路将于2018年5月1日投入运作，起初提供一个月免费搭乘<ref>{{cite web|title=KTM Berhad - Facebook|url=https://www.facebook.com/ktmberhad/photos/pcb.1779058108823442/1779058042156782/?type=3&theater|accessdate=27 April 2018}}</ref>，截至2018年8月此项服务依然提供免费搭乘直到2019年1月1日为止<ref>[http://www.sinchew.com.my/node/1781308/ 梳邦空中花园线电动火车．免费乘搭反应佳]</ref>。這條路線也是馬來西亞第2個提供机场快速鐵路服務的機場（第1個機場為[[吉隆坡國際機場|吉隆坡國際機場]]有[[吉隆坡機場快鐵|吉隆坡機場快鐵]]和[[吉隆坡機場支線|吉隆坡機場支線]]提供服務）。

== 背景 ==
[[File:160527_Subang_Airport_Branch_Line_-_Under_construction.jpg|thumb]]

天空花園線是由[[馬來西亞交通部|馬來西亞交通部]]出資興建[[蘇丹阿都阿兹沙機場|蘇丹阿都阿兹沙機場]]3號航站樓（天空花園航站樓）連接外部鐵路路網之電氣化雙軌鐵路，目的是為了減輕公路交通負擔、提供旅客快捷便利到離機場的運輸工具並為[[蘇丹阿都阿兹沙機場|蘇丹阿都阿兹沙機場]]的工作人員提供服務<ref>{{cite web|url=http://www.freemalaysiatoday.com/category/nation/2018/03/16/kl-sentral-subang-skypark-train-service-starts-in-may/ |title=KL Sentral-Subang Skypark train service starts in May |publisher=Free Malaysia Today |date=2018-03-16 |accessdate=2018-04-01 |language=en}}</ref>。總值5.21億[[令吉|令吉]]的鐵路於2011年9月決定興建並於2012年開始興建，原本預計在2016年可以完工，但後來展延至2017年6月<ref>{{cite web |author=EdgeProp.my |url=http://www.theedgemarkets.com/article/kl-sentralsubang-skypark-rail-service-starts-early-may |title=KL Sentral-Subang Skypark rail service starts early May |publisher=The Edge Markets |date=2018-03-27 |language=en |deadurl=no |archiveurl=https://web.archive.org/web/20180401213359/http://www.theedgemarkets.com/article/kl-sentralsubang-skypark-rail-service-starts-early-may |archivedate=2018-04-01 }}</ref>，最後訂於2018年5月1日啟用通車<ref>{{cite web |url=http://www.freemalaysiatoday.com/category/nation/2018/03/16/kl-sentral-subang-skypark-train-service-starts-in-may/ |title=KL Sentral-Subang Skypark train service starts in May|date=2018-03-16 |language=en}}</ref>。

天空花園線興建計畫由2個區段鐵道組成：第1個區段是改建[[梳邦再也站|梳邦再也站]]至{{link-en|斯里梳邦站|Sri Subang Komuter station}}由[[馬來西亞國家石油公司|馬來西亞國家石油公司]]鋪設4.09公里的燃料運輸鐵路，第2個區段為新建由{{link-en|斯里梳邦站|Sri Subang Komuter station}}沿著雙溪白沙羅河保護區終點在[[蘇丹阿都阿兹沙機場|蘇丹阿都阿兹沙機場]]附近的停車場<ref>{{cite web|url=http://www.spad.gov.my/ms/projek/projek-akan-datang/projek-landasan-keretapi-dari-subang-ke-terminal-skypark-subang|title=PROJEK LANDASAN KERETAPI DARI SUBANG KE TERMINAL SKYPARK SUBANG - Laman Web Rasmi Suruhanjaya Pengangkutan Awam Darat (S.P.A.D.)|language=en|deadurl=yes|archiveurl=https://web.archive.org/web/20150924104345/http://www.spad.gov.my/ms/projek/projek-akan-datang/projek-landasan-keretapi-dari-subang-ke-terminal-skypark-subang|archivedate=2015-09-24|access-date=2018-04-13}}</ref>長度為4.067公里高架鐵路，全線皆為電氣化雙軌鐵路。

電氣化雙軌計畫（EDTP）下此线包括2個阶段。其中第1阶段由[[梳邦再也站|梳邦再也站]]至{{link-en|天空花園航站樓站|Terminal Skypark Komuter station}}段，由Skypark Link-Lion Pacific公司承建，第2阶段则将会从{{link-en|天空花園航站樓站|Terminal Skypark Komuter station}}到[[雙溪毛糯站|雙溪毛糯]]与[[双溪毛糯－沙登－布城线|双溪毛糯－沙登－布城线]]交汇。[[東海岸鐵路線|東海岸鐵路線]]（ECRL）項目第2階段已經由Skypark公司開始建設由[[雙文丹|雙文丹]]到[[巴生港|巴生港]]的{{link-en|海關路站|Jalan Kastam Komuter station}}軌道及周邊工程<ref>{{cite web |url=https://www.railway-technology.com/features/featurechina-turns-malaysias-east-coast-rail-link-into-reality-5938409/ |title=China turns Malaysia’s East Coast Rail Link into reality |date=2017-10-01 |language=en |deadurl=no |archiveurl=https://web.archive.org/web/20171119233755/http://www.railway-technology.com/features/featurechina-turns-malaysias-east-coast-rail-link-into-reality-5938409/ |archivedate=2017-11-19 }}</ref>，主要供貨物運輸服務，作為[[雙溪毛糯－加影線|雙加捷运]]的替代線路，现阶段暂时未开始。

== 概覽 ==
天空花園線使用4組翻新的[[马来亚铁道83型电力动车组|马来亚铁道83型电力动车组]]。為了與[[巴生港線|巴生港線]]的列車區別，將原本巴生港線列車藍色和黃色的塗裝變更為灰色和橙色，預計每辆列車最多可以搭乘300名乘客。列车抵达吉隆坡中央车站时将于第三站台停靠。
