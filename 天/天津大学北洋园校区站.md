'''天津大学北洋园校区站'''位于[[天津市|天津市]][[津南区|津南区]][[海河教育园区|海河教育园区]]同砚路与新元南路交口（[[天津大学北洋园校区|天津大学北洋园校区]]南门），是规划中的[[天津地铁8号线|天津地铁8号线]]的地下车站。

== 车站结构 ==
天津大学北洋园校区站设三处出入口、两座风亭。

== 车站历史 ==
*该站最初为[[天津轨道交通|天津轨道交通]][[天津地铁6号线|天津地铁6号线]]的[[车站|车站]]<ref>{{cite news|url=http://www.022net.com/2012/12-17/504142273381727.html|title=天津地铁5、6号线车站规划名称公布|publisher=人民网·天津视窗|date=2012-12-17|deadurl=yes|archiveurl=https://web.archive.org/web/20141021005702/http://www.022net.com/2012/12-17/504142273381727.html|archivedate=2014-10-21}}</ref>。
*在2016年公布的规划中，已经说明该段线路调整为[[天津地铁8号线|天津地铁8号线]]。
*在2019年1月7日天津轨道交通集团公布的《天津地铁6号线工程（梅林路站-咸水沽西站）环境影响报告书（征求意见稿）》中，该站被命名为'''天津大学北洋园校区站'''<ref>{{cite web |title=天津地铁6号线工程（梅林路站-咸水沽西站）环境影响报告书征求意见稿信息公开 |url=http://www.tjgdjt.com/xinwen/content_20202.htm |website=天津轨道交通集团 |date=2019-01-07|accessdate=2019-01-13}}</ref><ref>{{cite web |title=天津地铁6号线工程（梅林路站-咸水沽西站）环境影响报告书（征求意见稿）|url=http://www.tjgdjt.com/touzi/img/site2/20190107/1803734483021d9d754f03.pdf |website=天津轨道交通集团 |date=2019-01-07 |accessdate=2019-01-13}}</ref>，线路远期与天津地铁8号线贯通运营。
