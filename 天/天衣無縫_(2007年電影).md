《'''天衣無縫'''》（{{lang-en|'''''Flawless'''''}}），是一部[[英国|英国]][[虛構|虛構]]電影，由{{link-en|迈克尔·瑞福|Michael Radford}}執導、愛德華·安德森（Edward Anderson）編劇，並且由[[黛米·摩尔|黛米·摩尔]]與[[米高·肯恩|米高·肯恩]]主演。2007年2月11日於[[德國|德國]]首映，並且在2008年3月28日於美國限定地區上映。

== 演員陣容 ==
* [[米高·肯恩|米高·肯恩]] 飾演 Mr. Hobbs
* [[黛米·摩尔|黛米·摩尔]] 飾演 Laura Quinn
* [[兰伯特·威尔森|兰伯特·威尔森]] 飾演 Finch
* {{link-en|喬思·艾克蘭|Joss Ackland}} 飾演 Milton Kendrick Ashtoncroft
* {{link-en|康斯坦丁·格雷戈里|Constantine Gregory}} 飾演 Dmitriev
* [[艾哈邁德·艾曼|艾哈邁德·艾曼]] as Bondok

== 作品評價 ==
該片獲得評論家普遍正面的評論。截至2008年4月24日止，影評網站[[爛番茄|爛番茄]]根據93則評論，給予了56%評論家的正面評價。[[Metacritic|Metacritic]]則根據21則評論，給予了平均分數57分（滿分為100分）。

== 票房表現 ==
本片於2007年10月5日在[[西班牙|西班牙]]上映，截至2008年1月6日為止，全球票房已達220萬美元。
