'''天梭菊石'''（[[學名|學名]]：{{lang|la|'''''Tissotia'''''}}）是生存在晚[[白堊紀|白堊紀]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]。其化石被發現於[[秘魯|秘魯]]、[[奈及利亞|奈及利亞]]、[[喀麥隆|喀麥隆]]和[[法國|法國]]等地。<ref name="Fossilworks" />

== 特徵 ==
天梭菊石的殼口形狀特殊，近似於[[正三角形|正三角形]]。其殼體的直徑可達20[[釐米|釐米]]，外觀和{{tsl|en|Vascoceras|巴斯克菊石}}頗為相似。本屬雖為白堊紀晚期的菊石，但其縫合線卻簡化到和[[三疊紀|三疊紀]]時期的[[齒菊石目|齒菊石類]]相近。<ref name="Tissotia">{{Cite web |url = http://liusir.pixnet.net/blog/post/248632922-tissotia-%E5%A4%A9%E6%A2%AD%E8%8F%8A%E7%9F%B3-%28%E7%A5%95%E9%AD%AF%29 |title = Tissotia 天梭菊石 (祕魯) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-13 }}</ref>
