'''天箭座'''（[[拉丁文|拉丁文]]：''{{lang|la|Sagitta}}''），面积79.93平方度，占全天面积的0.194%，在全天88个星座中，[[星座面积列表|面积排行]]倒数第三（只有[[小馬座|小馬座]]和[[南十字座|南十字座]]比它更小）。天箭座中亮于5.5等的恒星有8颗，最亮星为左旗五（天箭座γ），视星等为3.47。每年7月16日子夜天箭座中心经过[[中天_(天文学)|上中天]]。

== 特点 ==
天箭座被以下的星座包圍，由北面開始順時針為：[[狐狸座|狐狸座]]、[[武仙座|武仙座]]、[[天鷹座|天鷹座]]及[[海豚座|海豚座]]。

天箭座是古代[[克劳狄乌斯·托勒密|托勒密]]48星座之一，在那個時代，它比現在更小，只佔僅僅的4平方度而已。这个星座位於[[天球|天球]]赤道以北不遠處，因此除了[[南極|南極]]地區之外，在地球任何地方都可見到。虽然天箭座的最亮星仅有3.47等，但由于星座四颗主要亮星组成Y字形结构，所以在灯光污染不严重的地区还是可以辨认出来。其中[[左旗一|左旗一]]（天箭座α）和[[左旗二|左旗二]]（天箭座β）一起组成箭羽，而[[左旗五|左旗五]]（天箭座γ）和[[左旗三|左旗三]]（天箭座δ）一起组成箭杆。

== 神话和历史 ==
=== 神話傳說 ===
雖然天箭座並沒有亮星，但是在歷史上很多民族都把這幾顆星聯想成一支[[箭|箭]]，包括[[波斯人|波斯人]]、[[希伯來人|希伯來人]]、[[希臘人|希臘人]]和[[羅馬人|羅馬人]]。所以有很多有關這個星座的神話故事。

最重要的兩個神話故事都與'''天箭座'''相鄰的星座有關，包括西面[[武仙座|武仙座]]所代表的[[赫拉克勒斯|赫拉克勒斯]]和東面的[[天鷹座|天鷹座]]。 

* [[泰坦|泰坦]]神之一的[[普羅米修斯|普羅米修斯]]（Promethus）從天界盜火授人，觸怒主神[[宙斯|宙斯]]（Zeus），遂把他以鎖鏈綁於[[高加索|高加索]]山懸崖上，每天都派一隻[[鷹|鷹]]來啄食他的[[肝臟|肝臟]]，又讓他的肝臟重新癒合，天天如是。直至[[赫拉克勒斯|赫拉克勒斯]]（Heracles）把他從這種無止境的痛苦中拯救出來。[[赫拉克勒斯|赫拉克勒斯]]把那頭鷹用箭射死，然後把[[普羅米修斯|普羅米修斯]]釋放。

* 另一個故事是關於[[赫拉克勒斯|赫拉克勒斯]]十二苦差的第六件，他要軀趕盤據[[希臘|希臘]][[阿卡迪亚|阿卡迪亚]]（Arcadia）的斯廷法羅斯湖的幾隻怪鳥（Stymphalian Birds）。這種怪鳥有銅爪，和能向敵人發射的金屬羽毛。[[赫拉克勒斯|赫拉克勒斯]]用響板把牠們軀趕，並用箭把牠們射殺。[[天琴座|天琴座]]、[[天鷹座|天鷹座]]和[[天鹅座|天鹅座]]原本是代表那些怪鳥的，而'''天箭座'''的箭頭對準[[天鷹座|天鷹座]]。

* '''天箭座'''更被演譯為小愛神[[邱比特|邱比特]]的箭，或是[[人馬座|人馬座]]射向[[天蝎座|天蝎座]]的箭。也有傳這支箭其實是屬於[[半人馬座|半人馬座]]向[[天鷹座|天鷹座]]射出的，支持者所持理據就是在星圖上看[[半人馬座|半人馬座]]的時候，它成一個恰當的角度面向'''天箭座'''，但是[[人馬座|人馬座]]則在'''天箭座'''的下方而面向相反的角度。

* [[半人馬座|半人馬座]]所代表的是半人馬[[喀戎|喀戎]]（Chiron），他是一眾英雄的導師。神話故事中他被徒弟[[赫拉克勒斯|赫拉克勒斯]]用箭誤傷，本來[[喀戎|喀戎]]擁有不死身，但傷他的箭頭浸過[[九頭蛇|九頭蛇]]許德拉（Hydra）的毒血，令[[喀戎|喀戎]]異常痛苦，生不如死。遂以不死身向因盜火授人被綁在山上的[[普羅米修斯|普羅米修斯]]所受的痛苦交換，以求解脫。然後[[赫拉克勒斯|赫拉克勒斯]]（有說是[[喀戎|喀戎]]）把啄食[[普羅米修斯|普羅米修斯]]肝臟的那頭鷹射死。以上星座再加上[[豺狼座|豺狼座]]，就組成了[[赫拉克勒斯|赫拉克勒斯]]十二苦差的第四件--活捉厄律曼托斯山上的野豬（Erymanthian Boar）的故事基礎。

=== 歷史 ===
在以前，這天區是叫作"Sham"的，現在"Sham"這個名字只是指天箭座的一顆亮星左旗一（α Sge）。天箭座是[[拜耳命名法|拜耳命名法]]根據恒星光度命名時，次序錯誤的例子，最亮星被錯誤命名為天箭座γ。另一個同類例子可在[[人馬座|人馬座]]找到。

== 恒星 ==
天箭座的亮星：
* [[左旗一|左旗一]]（天箭座α）：是一顆黃色巨星，光譜型屬 G1 II，視星等4.37，距地球610光年。
* [[左旗五|左旗五]]（天箭座γ）：天箭座最亮星，視星等3.47，是一顆紅色巨星，光譜型屬 M0 III，距地球170光年。
* [[左旗三|左旗三]]（天箭座δ）：怀疑是[[雙星|双星系统]]，光譜型分別屬 M2 II和A0 V，（也有可能是單一影像內的複合光譜），視星等3.82。
* 天箭座ε：光譜型8 III，視星等5.66，共有四顆成員，包括一組物理三合星（A, C, D）加一密近星（B）。 
* 天箭座η：光譜型K2 III，視星等5.1，屬於畢宿星團的其中一員。
* 天箭座θ：三合星，视星等6.51。
* 天箭座U：[[变星|大陵型食变星]]，亮度最大时为6.45等，亮度最小时为9.28等，变光周期3.38日。

== [[深空天体|深空天体]] ==
[[File:Messier71.jpg|200px]]
* 球状星团[[M71|M71]]：位于[[左旗五|左旗五]]（天箭座γ）和[[左旗三|左旗三]]（天箭座δ）之间。分類具爭議性，有些天文學家認為M71是一個很鬆散的[[球状星团|球状星团]]，也有些卻認為是一個緻密的[[疏散星团|疏散星团]]，现在的研究倾向于认为M71是一个球状星团。距地球13,000光年，视星等8.3。1746年[[法國|法國]][[天文學家|天文學家]]錫蘇（Philippe Loys de Chéseaux）發現該星團。
