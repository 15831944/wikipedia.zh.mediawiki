《'''天使演唱會'''》是[[日本|日本]]{{nihongo|[[工畫堂工作室|工畫堂工作室]]|工画堂スタジオ}}所發行的[[音樂遊戲|音樂]][[冒險遊戲|冒險遊戲]]。

故事敘述想成為音樂家的一對青梅竹馬，為了參加一年一度在王都舉行的音樂會，而離開故鄉，旅行前往王都，一路上經歷的冒險遊戲。

遊戲中途的選擇項目會影響女角的好感度及前進的關卡，每一關需要用鍵盤彈奏歌曲(除非設定成自動演奏)，評價達到第4級便可過關，否則會要求重彈或回到主標題畫面。玩家可自行設定彈奏難度，Easy只需彈奏4個鍵，Noraml為8個，Hard則高達30個。

[[工畫堂|工畫堂]]其後推出的[[音樂|音樂]][[冒險遊戲|冒險遊戲]]有『[[AS～天使小夜曲|AS～天使小夜曲]]（{{lang|ja|AS～エンジェリックセレナーデ}}）』及『[[交響樂之雨|交響樂之雨]]（{{lang|ja|2=シンフォニック=レイン}}）』。

== 人物 ==
; {{nihongo|高吉·史多法特|カウジー･ストファート|Kauzy Stufato}}
:遊戲的主人翁，玩家扮演的16歲少年。和瑟菲是從小長大的青梅竹馬，擅長演奏魔法樂器符德魯琴。

; {{nihongo|瑟菲·史威尼|サフィ·スィーニー|Saphy Sweenie|[[聲優|CV]]：[[堀江由衣|堀江由衣]]}}
:夢想成為歌姬的16歲少女，玩家的旅伴。她的歌聲具有不可思議的力量。

; {{nihongo|克麗儂·利巴雷|クリノン･リヴァーレ|Klinon Rivale|CV：[[田村由香里|田村由香里]]}}
:出身名門貴族的10歲少女，為了參加王都的歌唱比賽，與管家蓋雷森一起旅行。

; {{nihongo|蓋雷森·比斯特克|ギャレソン･ビステク}}
:利巴雷家的管家，擔任克麗儂唱歌時的演奏，克麗儂稱他為「老爺子」。

; {{nihongo|可可娜･裘特|ココナ･キュート|Cocona Cute|CV：[[おみむらまゆこ|麻績村真由子]]}}
:經常出現在高吉和瑟菲前的神秘少女，但原來是傳說中像慧星般一下出現成名，一年後又突然引退的歌姬。

; {{nihongo|莉安･愛爾薩斯|リアン･エルサス|Leann Elsas|CV：[[水樹奈奈|水樹奈奈]]}}
:倔強又有點任性的謎樣少女，真正身份是風之國飛魯拉達的第二公主。

; {{nihongo|畢雅拉･賽提斯|ピアラ･セティス|Piarra Thetis|CV：[[澤城美雪|澤城美雪]]}}
:在利巴雷家工作，克麗儂的歌唱老師。外表看來很溫柔賢淑，但其實是格鬥專家。
