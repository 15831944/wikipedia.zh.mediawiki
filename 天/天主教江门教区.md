'''江门教区'''（拉丁文：Dioecesis Chiammenensis）是[[天主教|天主教]]在中国[[广东省|广东省]]设立的一个[[教区|教区]]。

== 历史 ==

1924年1月31日，罗马教廷从'''广州代牧区'''划出原由巴黎外方傳教會管轄的新会、台山、阳江、赤溪、茂名、信宜、电白、罗定、鬱南、花县、云浮等县成立'''江门监牧区'''，交给美国[[玛利诺外方传教会|玛利诺外方传教会]]负责。

1927年2月3日，江门监牧区升格为'''江门代牧区'''。
 
1946年4月11日，罗马教廷在中国建立[[圣统制|圣统制]]，将江门代牧区升格为'''江门教区'''。

== 教徒 ==

至1937年，江门代牧区有外国传教士50多人，中国传教士12人，教徒7000余人。

至1949年，江门教区有教职人员70余人，教徒达12800人。

== 主教座堂 ==

圣母圣心主教座堂，江门市北街海傍街六十七号

== 历任主教 ==
*[[华理柱|华理柱]]（James Edward Walsh），[[玛利诺外方传教会|玛利诺外方传教会]]士，1891年4月30日生于美国，1915年12月7日晋升神父，1927年2月1日任江门代牧，5月22日祝圣主教，1936年7月回国任马利诺外方传教会第二任总会长，1948年再度来华，出任天主教教务协进会秘书长，在1958年10月18日在上海被判处20年徒刑，直到1970年7月10日，因中美关系将要有重大变化，才被提前释放，成为最后一个被中国釋放的西方传教士。1981年7月29日去世（90岁）。瑪利諾神父會於1963年在香港開辦了一所名為「[[華德學校|華德學校]]」的小學以表揚華理柱神父在華傳教的辛勞。

*[[阿道夫·柏增|阿道夫·柏增]]主教（[[柏增|Adolph John Paschang]]），玛利诺外方传教会士，1895年4月16日生于美国，1921年5月21日晋升神父，1937年6月17日任江门代牧，11月30日祝圣主教，1946年4月11日升为江门主教，1968年2月3日去世（72岁）。瑪利諾神父會亦於1969年在香港開辦一所名為「[[柏德學校|柏德學校]]」的小學以紀念這一位來華傳教的瑪利諾會士柏增神父的辛勞。

*李盘石（1911年－2007年），1981年晋牧。

*梁建森，2011年晋牧。2009年11月25日经选举为天主教江门教区正权主教<ref>[http://www.mzzjw.gd.gov.cn/zjtd/ShowArticle.asp?ArticleID=4273 天主教江门教区隆重举行梁建森主教祝圣典礼] {{webarchive|url=https://web.archive.org/web/20110720144113/http://www.mzzjw.gd.gov.cn/zjtd/ShowArticle.asp?ArticleID=4273 |date=2011-07-20 }}</ref>，获得圣座正式批准<ref>[http://www.fides.org/aree/news/newsdet.php?idnews=6226&lan=chi 亚洲/中国 - 江门主教祝圣] {{webarchive|url=https://web.archive.org/web/20120319205241/http://www.fides.org/aree/news/newsdet.php?idnews=6226&lan=chi |date=2012-03-19 }}，信仰通讯社，2011-04-08 </ref>后，于2011年3月30日在圣母圣心主教座堂举行祝圣典礼<ref>[http://www.sara.gov.cn/nsjg/ywes/gzdt4/7778.htm 天主教江门教区隆重举行梁建森主教祝圣典礼] {{webarchive|url=https://web.archive.org/web/20110728094918/http://sara.gov.cn/nsjg/ywes/gzdt4/7778.htm |date=2011-07-28 }}</ref>
