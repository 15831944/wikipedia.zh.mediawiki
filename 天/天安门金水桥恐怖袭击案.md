'''天安门撞桥事件'''（或称'''金水桥事件'''；'''“10·28”暴力恐怖袭击事件'''），是一起于[[2013年10月|2013年10月]]28日发生在[[中华人民共和国|中华人民共和国]][[北京市|北京市]][[天安门城楼|天安门城楼]]前的[[汽车袭击|汽车袭击]]事件<ref name="BBCOct29">{{cite news |url = http://www.bbc.co.uk/news/world-asia-china-24722898 |title = Tiananmen crash: China police 'seek Xinjiang suspects' |date = 29 October 2013 |work = BBC News Online |language = en |accessdate = 2013-10-30}}</ref>。事件中共有5人丧生，其中3人在事发吉普车内随汽车的燃烧当场死亡，其他2人分别是来自[[菲律宾|菲律宾]]和[[广东|广东]]的游客。恐怖组织“[[突厥斯坦伊斯兰党|突厥斯坦伊斯兰党]]”称对吉普车冲撞天安门事件负责<ref>[http://www.rfa.org/mandarin/yataibaodao/shaoshuminzu/sd-11242013133403.html “突厥斯坦伊斯兰党”称对吉普车冲撞天安门事件负责]</ref>。

事件发生后，中国各省市进入北京的高速公路陆续安装了[[进京检测站|进京检查站]]对过往车辆全员逐一进行身份证查验，开车通过检查站的过程与开车过境通过[[海关|海关]]的过程雷同。此后在[[进京证|北京限行外地车辆的相关政策]]中，添加了禁止外地车全天驶入天安门、[[中南海|中南海]]周边的[[长安街|长安街]]等道路这一规定。

2014年6月24日，中国[[国家互联网信息办公室|国家互联网信息办公室]]发布电视专题片《恐怖主义的网上推手》，该电视专题片首次披露了该事件的现场监控视频<ref>[http://v.ifeng.com/news/world/2014006/01275875-a2a5-4825-b550-16fca7458dfb.shtml  《恐怖主义的网上推手——东伊运恐怖音视频》专题片]</ref>。

==事件经过==
2013年10月28日中午12点05分左右（[[北京时间|北京时间]]），一辆吉普车在北京[[天安门广场|天安门广场]]前由东闯入长安街便道。[[路透社|路透社]]援引一名目擊者的話說，這輛車拖着一個寫着黑字的白色標語。在撞倒多名行人之后，又撞上了天安门城楼前[[金水桥|金水桥]]的护栏，随即起火。事發後，部分消息称警察迅速封鎖了現場和天安門廣場，并对周边地区进行交通管制。中國官方网络警察也立即删除了互联网上有关这件事的非官方信息，以限制信息的传播<ref name="BBC Panic" /><ref name="NYT1">[http://cn.nytimes.com/china/20131030/c30crash/zh-hant/ 學者擔心天安門撞車案加劇民族矛盾], [[紐約時報|紐約時報]], 2013/10/30</ref><ref>[http://edition.cnn.com/2013/10/29/world/asia/china-tiananmen-suspects/ China censors pounce as Tiananmen Square jeep deaths investigated], [[CNN|CNN]], 2013/10/30</ref>。

外流照片上可見在天安門前[[毛澤東|毛澤東]]畫像的下方，一台吉普車正起火燃燒，同時冒出大量濃煙<ref name="民視">[http://m.ftv.com.tw/newscontent.aspx?sno=2013A28I21M1 自焚攻擊? 天安門火燒車3死數傷], [[民視|民視]]新聞, 2013/10/28</ref>。

最初的报道指车内3人当场死亡<ref name=BBCOct29/><ref name="Jiji">{{cite news|url=http://www.jiji.com/jc/zc?k=201310/2013102800839&g=pol |script-title=ja:天安門で車炎上、5人死亡＝突入し邦人含め38人負傷－政治的抗議の可能性・北京|publisher={{lang|ja|時事ドットコム}} |date=2013-10-29 |language=ja|accessdate=2013年10月28日}}</ref>。另有38人在事件中受伤，其中有11人被送往医院救治，其中两名当时在广场上的游客伤重死亡<ref name="xinhua">{{cite news |url = http://news.xinhuanet.com/local/2013-10/28/c_117902718.htm |title = 一吉普车撞天安门金水桥护栏后起火 车内3人死亡|author=卢国强、李亚红|date=2013年10月28日|work=新华网 |accessdate=2013-10-31}}</ref>。受伤的38人中包括3名菲律宾游客（2女1男）和1名[[日本|日本]]男游客<ref name=qianlong>{{cite news |url = http://beijing.qianlong.com/3825/2013/10/28/7044@9091428.htm|title=北京迅速处置一起吉普车冲撞天安门金水桥事件|author=岳崎|date=2013年10月28日|publisher=千龙网|accessdate=2013-10-31}}</ref>。至[[北京时间|当地时间]]10月30日下午，伤者中有5人已进行了手术，12人在[[加護病房|ICU]]病房治疗，其余21人则转为一般住院或留院观察<ref name=ICU>{{cite news|author=杨文娟、侯崇智|title=北京全力救治"10·28"事件伤者|url=http://he.people.com.cn/n/2013/1030/c192235-19804254.html|accessdate=2013-10-31|newspaper=新京报|date=2013年10月30日|deadurl=yes|archiveurl=https://web.archive.org/web/20131103183525/http://he.people.com.cn/n/2013/1030/c192235-19804254.html|archivedate=2013年11月3日}}</ref>。至31日14时，送医的42名伤患中（包括已經重伤死亡的两名）累计3人出院，8人在ICU病房，10人已达出院标准<ref>{{cite news|author=|title=“10·28”暴力恐怖袭击案件伤者得到全面救治|url=http://news.xinhuanet.com/legal/2013-11/01/c_125635470.htm|accessdate=2013-11-01|publisher=新华网|work=央视新闻|date=2013年11月1日}}</ref>。

中国大陆警方随后发出通知，要求北京各大酒店搜寻两名来自中国[[新疆|新疆]]的可疑人物<ref name=BBCOct29/>。通知中同时提到了一辆汽车和新疆的车牌<ref name=BBCOct29/><ref name=cn.reuters/>。警方还指示酒店要注意可疑的客人<ref name=toi>{{cite web|title=China names Tiananmen Square car crash 'suspects'|url=http://timesofindia.indiatimes.com/world/china/China-names-Tiananmen-Square-car-crash-suspects/articleshow/24877935.cms|work=[[印度时报|印度时报]]|date=29 October 2013|language=en|accessdate=2013-10-30}}</ref>。警方还要求酒店报告所有自10月1日登记入住的客人，以及他们所驾驶的车辆。並表示此指示“为了防止犯罪嫌疑人及车辆犯下更多的罪行”<ref name=TIMEOct/>。

犯罪嫌疑人被指有[[维吾尔族|维吾尔族]]名，来自常有暴力事件发生的新疆地区<ref name=BBCOct29/><ref =asashi>{{cite news|url=http://www.asahi.com/articles/TKY201310290060.html |script-title=ja:天安門突入、ウイグル出身者の犯行か 当局通知で名指し|publisher={{lang|ja|朝日新聞デジタル}}|author=林望|date=2013年10月29日|language=ja|accessdate=2013-10-31}}</ref>。其中一名嫌犯疑来自[[鄯善县|鄯善县]]。

《[[產經新聞|產經新聞]]》說，香港人權組織指這次涉案死亡者中有人是6月新疆血腥鎮壓時一名死亡維吾爾族人的家屬<ref name="BBC1031" />。[[阿波罗网|阿波罗网]]报道，维吾尔族一家三口中死者妻子死时已身怀六甲。<ref>{{cite web |url = http://www.aboluowang.com/2013/1105/346655.html |language = |title = 至亲被杀复仇天安门维族女身怀六甲 当局秘令严控新疆所有被镇压者家庭 |publisher=阿波罗新闻网 |date=2013-11-05 |accessdate= }}</ref>。

11月24日，路透社消息指出，一个自称是“[[突厥斯坦伊斯兰党|突厥斯坦伊斯兰党]]”的恐怖组织团体承认发动了10月28日天安门袭击事件，他们在網路上发表了一份录音：“60年来，你们(中華人民共和國政府)欺骗了东突厥，如今，我们覺醒了。人民明白了谁才是他们真正的敌人，他们接受了教训，回到自己的宗教中。” 并表示这是他们针对北京当局发起的第一波攻击行动，他们会陆续发起针对中国政权的行动，包括袭击人民大会堂<ref>{{cite web |url = http://www.crntt.com/doc/1028/8/4/2/102884277.html?coluid=45&kindid=0&docid=102884277&mdate=1124164337 |title = “突厥斯坦伊斯兰党”承认袭击天安门 |publisher=  |work=中国评论通讯社 | date = 2013-11-24 | accessdate=2013-11-24}}</ref>。

金水桥事件发生后，[[中共新疆维吾尔自治区党委|中共新疆维吾尔自治区党委]]常委，[[新疆军区|新疆军区]][[司令员|司令员]]、党委副书记[[彭勇|彭勇]]中将被免职，继任者为新疆军区[[政治委员|政治委员]]刘雷<ref>{{cite web |url = http://www.nytimes.com/2013/11/04/world/asia/china-demotes-a-military-commander-after-attack-in-beijing.html |title = China Demotes a Military Commander After Attack in Beijing |publisher = The New York Times | date = 2013-11-03 |accessdate = 2013-11-03 | language = en}}</ref>。

==侦破结果及后续调查==
2013年10月30日，[[中国中央电视台|中国中央电视台]]报道警方已破案。车内发现盛汽油装置、砍刀、铁棍，及印有“[[圣战|圣战]]”等宗教极端主义内容的旗帜，犯罪嫌疑人[[乌斯曼·艾山|乌斯曼·艾山]]、其母[[库完汗·热依木|库完汗·热依木]]及妻[[古力克孜·艾尼|古力克孜·艾尼]]3人在该恐怖袭击案件中当场死亡。[[玉江山·吾许尔|玉江山·吾许尔]]、[[玉苏甫·吾买尔尼亚孜|玉苏甫·吾买尔尼亚孜]]、[[玉苏普·艾合麦提|玉苏普·艾合麦提]]、[[古丽娜尔·托乎提尼亚孜|古丽娜尔·托乎提尼亚孜]]、[[布坚乃提·阿卜杜喀迪尔|布坚乃提·阿卜杜喀迪尔]]等5名同伙也被已被抓获<ref>{{cite web |url = http://news.ifeng.com/mainland/detail_2013_10/30/30808076_0.shtml |title = 乌斯曼·艾山及其妻子母亲驾车撞人 点燃车内汽油 |publisher=央视 |work=凤凰网 |date=2013-10-30 |accessdate = 2013-11-01}}</ref>。据朝日新闻社披露在事件发生后的五个小时，公安机关已经找到了肇事车辆车牌登记者玉江山·吾许尔在新疆伊宁市的亲人<ref>{{cite web|url =http://asahichinese.com/article/china_taiwan/AJ201311010019|title =天安门起火案嫌疑人父母否认儿子参与袭击|publisher =朝日新闻中文网|work =朝日新闻中文网|date =2013-11-01|accessdate =2013-11-01|deadurl =yes|archiveurl =https://web.archive.org/web/20131104101729/http://asahichinese.com/article/china_taiwan/AJ201311010019|archivedate =2013-11-04}}</ref>。
