==劇情簡介==
大成末年，'''寧世徵'''起兵造反，大成國破。負責大成皇室安全的秘密組織血浮屠，帶著剛出生的九皇子逃亡，'''寧世徵'''派長子'''寧川'''和六子'''寧弈'''前去追捕，最終血浮屠首领'''顧衡'''抱着九皇子跳崖不知所終。天盛王朝成立。

天盛十八年，'''寧世徵'''大赦天下，被關八年的'''寧弈'''重新回歸朝堂。楚王'''寧弈'''，看似風流散漫的當朝六皇子，內心卻背負著慘痛往事。他以天下為棋局，洗雪冤屈、懲治奸佞、整肅朝綱，在腥風血雨的朝堂爭鬥中步步為營。'''鳳知微'''，被逐高門之女，不甘屈服於坎坷的命運，女扮男裝進入青溟書院，一躍成為無雙國士'''魏知'''，風雲漸起於朝野。一個是城府深藏的親王，一個是初露鋒芒的官場新秀，兩人在風雲詭變的朝堂裡彼此試探、頻頻過招，相互排斥卻又不自禁相互吸引。而當彼此的心漸漸向對方敞開，邂逅的卻是命運徹骨的森涼<ref>{{Cite web|url=http://www.iqiyi.com/a_19rrh9uc9p.html?vfm=2008_aldbd|title=愛奇藝獨家播出官方劇情簡介|accessdate=|author=|date=|publisher=}}</ref>。

== 幕後製作 ==

==== 劇本改編 ====
電視劇《天盛長歌》改編自天下歸元小說《凰權》。

原著作者兼編劇天下歸元在採訪中談及《天盛長歌》的改編將遵循「四不」原則：不動人設、不動主線、不刪主要人物、不改主要情節。

該劇對原著劇情取捨的原則是：不能通過廣電審核以及不便拍攝的部分，會進行一定的改編處理；在權謀的基礎上，進一步深化細化了對男女主相愛相殺的述說，對原本不夠飽滿細膩的情節會做進一步的充實細化和提升。

==== 演員選擇 ====
早在項目籌劃之初，製片方認為[[陈坤|陳坤]]是男主角寧弈的不二人選。因為寧弈這個角色對於演員的演技、顏值、人氣和新鮮度缺一不可。為邀請[[陈坤|陳坤]]出演，在過去兩年裡，製片方三顧茅廬，重視角色的豐富性和塑造空間，並精益求精不斷優化劇本，決心打造精品劇，陳坤也因此決定接了該劇。

在女主角的選擇上，劇組也經過了反反覆覆的論證比較，製片人鈕繼新表示[[倪妮|倪妮]]是鳳知微的不二人選，她的形象和氣質非常獨特，演技出眾，鳳知微的剛柔並濟在她身上都有所體現，但最終確認倪妮出演的另一個硬性標準還有出品方堅持的選角的新鮮感。而在邀請[[倪妮|倪妮]]首次出演電視劇的過程中，出品方也是三顧茅廬。

==== 導演團隊 ====
該劇由導演沈嚴、劉海波執導，電視劇《[[中国式离婚_(电视剧)|中國式離婚]]》、《[[辣妈正传|辣媽正傳]]》、《[[我们无处安放的青春|我們無處安放的青春]]》、《手機》、《[[中国式关系|中國式關係]]》等現實主義之作，均出自兩人之手。沈嚴表示，題材不重要，有趣的故事最重要，有意思的人物最重要。

==== 幕後班底 ====
製片方為將該劇的中國文化、中國元素淋漓盡致地發揮出來，請來了美術指導、服裝設計師[[張叔平|張叔平]]。張叔平帶領呂鳳珊、方思哲設計唐風造型。還薈萃了電影劇照師包翔宇（《[[归来_(电影)|歸來]]》《[[我不是潘金莲_(电影)|我不是潘金蓮]]》《[[寻龙诀|尋龍訣]]》）、作曲董潁達（《[[夏有乔木雅望天堂|夏有喬木雅望天堂]]》、《[[北平无战事|北平無戰事]]》）、攝影指導李希（《[[失恋33天_(电影)|失戀33天]]》《[[红楼梦_(2010年电视剧)|新版紅樓夢]]》）、美術指導申小湧（《[[秋喜|秋喜]]》《[[疯狂的赛车|瘋狂的賽車]]》）、剪輯指導張佳（《[[杜拉拉升职记|杜拉拉升職記]]》《[[海上传奇|海上傳奇]]》），並請來曾指導過《[[瑯琊榜_(電視劇)|瑯琊榜]]》《[[芈月传|羋月傳]]》《[[九州·海上牧云记|海上牧雲記]]》的禮儀指導李斌。

==== 電影規格 ====
制作方面，劇組還採取了古裝劇罕見的現場收音模式，並選用2.35:1的電影常用畫幅，力圖「讓觀眾在電視熒屏上也能體驗電影畫面的觀影感受」。

== 拍攝過程 ==
《天盛長歌》於2017年5月27日在湖北開機<ref name=":0">{{Cite web|url=http://ent.ifeng.com/a/20170527/42938111_0.shtml|title=《凰权·弈天下》开机 陈坤微博晒戎装照|accessdate=2017-11-12|author=|date=2017-05-27|work=玄燕凤 PK065|publisher=凤凰网娱乐|language=Zh-hans}}</ref>；同年12月9日歷時197天拍攝，全劇組殺青<ref name=":1">{{Cite web|url=http://ent.sina.com.cn/v/m/2017-12-12/doc-ifypnyqi4040361.shtml|title=《凰权·弈天下》杀青 陈坤倪妮夺位之战大幕开启|accessdate=2017-12-12|author=|date=2017-12-12|work=得得|publisher=新浪娱乐|language=Zh-hans|}}</ref>。

該劇主要取景於襄陽唐城，覆蓋皇宮、環殿、東西街市等除凱旋門之外的所有景點。之所以選擇唐城，不僅是因為在朝代上與原著契合，更是因為不常在熒幕上曝光的唐城能帶來更具新鮮感的取景效果，也是唐城首次用于電視劇拍攝。
