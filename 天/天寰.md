'''天寰'''（{{lang-en|Victoria Skye}}）位於[[香港|香港]][[九龍|九龍]][[啟德發展計劃|啟德發展區]]沐寧街1號，[[發展商|發展商]]為[[建灝地產集團|建灝地產集團]]（{{Lang-en|K&K Property Holdings Limited}}），由{{link-en|王董建築師事務|Wong Tung & Partners}}設計，發展項目的認可人士為袁灼堯先生，總承建商為[[顯利工程|顯利工程有限公司]]，室內裝潢工程承建商為[[承達集團有限公司|承達集團有限公司]]，關鍵日期原為2018年12月31日，後由發展項目的認可人士袁灼堯先生批予改為2019年4月30日。<ref name=":1">{{Cite web|url=http://www.victoriaskye.com.hk/admin/uploads/VSSB_20180624Examination_FINAL.pdf|title=售樓說明書|accessdate=2018.09.20|author=建灝地產集團|date=2018.06.20|publisher=建灝地產集團|pages=7}}</ref>

天寰是[[建灝地產集團|建灝地產集團]]首個香港市區大型住宅項目，項目以「擁抱 觸得到的未來」（{{lang-en|Live the Future}}）為主題，取名「天寰」—「天」象徵得天獨厚的條件於一身，「寰」象徵廣闊的地域，寰宇天下之意。<ref name=":0">{{Cite web|url=http://kandkpropertyholdings.com/index_en.php|title=K&K Property Holdings Ltd|accessdate=2018-09-18|last=Limited|first=Website created by Li&RC Design Associates|work=kandkpropertyholdings.com}}</ref>

== 單位 ==
項目共822個住宅單位，由3座高座及4幢低座所組成，高座名為天海匯（{{lang-en|Ocean Waves}}），低座名為星寰匯（{{lang-en|Star Waves}}）。分層單位面積為200多至1,200平方呎，另有少量特色單位，部分能遠眺[[維港|維港]]景色。住宅停車位共145個，訪客停車位共10個，電單車停車位共15個，單車停車位共23個。<ref name=":1" />

== 位置 ==
天寰位於[[啟德|啟德]]豪宅區，毗鄰[[啟德1號|啟德1號]]、啟德1號(II)、Oasis Kai Tak、嘉匯、[[龍譽|龍譽]]等私人樓盤，校網為34小學校網及九龍城中學校網，為九龍著名校網。

== 會所及其他設施 ==
項目會所分為3層，包括室內會所及戶外園林空間，提供70項設施，室內會所佔地面積逾2.1萬方呎，而戶外園林空間佔地5.7萬方呎，投資額共約6,000萬元。

項目主入口有耗資300萬港元的13.5米高的主題牆宇及地標式廊柱，而且特邀著名雕塑家李展輝先生創作大型雕塑。

項目的綠化（如垂直綠場、花庭）面積佔整體達30%，戶外空間全部採用環保[[風力發電|風力發電]]照明燈。<ref name=":0" />
