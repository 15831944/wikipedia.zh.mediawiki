'''天羽槌雄神'''（{{lang|ja|アメノハヅチオノカミ}}）為《[[古語拾遺|古語拾遺]]》的記述，又記作'''倭文神'''（{{lang|ja|シズノカミ}}）、'''建葉槌命'''（{{lang|ja|タケハヅチノミコト}}）、'''天羽雷命'''（{{lang|ja|アメノハヅチノミコト}}）等別名，祂是[[日本神話|日本神話]]裡出現的神明；亦是倭文氏（{{lang|ja|しどりし}}）的祖先。

== 概要 ==
=== 古語拾遺之記載 ===
[[File:Amaterasu_cave.JPG|thumb]]
依據《古語拾遺》的記載，[[天照大神|天照大神]]因[[素戔嗚尊|素戔鳴神]]胡作非為，嚇得躲入[[天岩戶|大石窟]]，[[高皇產靈神|高皇產靈神]]便集合八十萬眾神共同協商。[[思兼神|思兼神]]建議[[布刀玉命|太玉神]]率眾製造和幣、[[伊斯許理度賣命|石凝姥神]]取天香山之銅鑄造日像之鏡、[[長白羽神|長白羽神]]種[[麻|麻]]以織作青和幣、[[天日鷲神|天日鷲神]]和津咋見神種植穀木以作白和幣、'''天羽槌雄神'''織文布、天棚機姬神織神衣、[[玉祖命|櫛明玉神]]作[[八尺瓊勾玉|八坂瓊襲五百箇御統玉]]、手置帆負命和彥狹知命二神以天御量伐大峽小峽之木材而造瑞殿，兼作御笠、矛、盾等、[[天目一箇神|天目一箇神]]作雜刀、斧及鐵鐸。於是掘天香山之五百箇真賢木，上枝懸玉、中枝懸鏡、下枝懸青白和幣，由[[布刀玉命|太玉命]]捧持，而[[天兒屋命|天兒屋命]]口誦祝詞祈禱。

== 解說 ==
神名裡的「羽」即為[[棉花|棉花]]、[[絲綢|絲綢]]等組成的布帛，而《古語拾遺》所謂的「文布」即全名為「倭文（{{lang|ja|シドリ}}或{{lang|ja|シヅリ}}）布」的[[紡織品|紡織品]]。因此該神祇為倭文氏的祖先，且一般日本民間將之視為[[紡織|紡織]]之神。神格和祂同等的，另有[[栲幡千千姬命|栲幡千千姬命]]、[[長白羽神|長白羽神]]、天棚機姬命等神明。

=== 建葉槌命 ===
該神明之別名為'''建葉槌命'''（{{lang|ja|タケハヅチノミコト}}，也寫作「武葉槌命」），典故出自《[[日本書紀|日本書紀]]》卷第二神代下。在[[天孫降臨|天孫降臨]]前，[[經津主神|經津主神]]與[[建御雷|武甕槌神]]受[[高皇產靈神|高皇產靈尊]]之命一同前往平定[[葦原中國|葦原中國]]，唯有[[天津甕星|星神香香背男]]不服。於是另外派遣倭文神'''建葉槌命'''降伏，使前述二神得以回去覆命。

但是，為何紡織之神能降伏頑強抵抗的星神香香背男？出現以下許多說法：
# 在[[日語|日語]]中，「建」的讀音同「武」、「葉」同「刃」，因此建葉槌命實際上是[[戰神|武神]]<ref>參看{{ja}}[http://kamnavi.jp/en/higasi/oomika.htm 大甕倭文神社]，「御祭神の御事」的段落。</ref>。
# 將星星織入紡織品中，象徵星星被紡織品封印起來。
# 建葉槌命將織物贈與星神香香背男，加以懷柔勸服。

另外，《日本書紀》卷第二神代下的第二種說法裡提到天神派遣[[經津主神|經津主神]]與[[建御雷|武甕槌神]]平定葦原中國，但天有惡神[[天津甕星|天津甕星]]，亦名天香香背男。前二神請天神先誅此惡神，才願降臨下凡。此時，有位號曰齋之大人的齋主神，位於于東國檝取之地。由於此段故事與正文相似，於是後世有人懷疑齋之大人便是建葉槌命。

== 信仰 ==
在日本，供奉天羽槌雄神為主祭神的神社除了倭文神社外，另計有下述：
* 靜神社（[[茨城縣|茨城縣]][[那珂市|那珂市]]）
* 服部神社（[[石川縣|石川縣]][[加賀市|加賀市]]）

祭拜建葉槌命的神社則有：
* 石井神社（茨城縣[[笠間市|笠間市]]）
* 大甕倭文神社（茨城縣[[日立市|日立市]]）
