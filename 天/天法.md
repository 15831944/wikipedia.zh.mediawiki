{{藏傳佛教}}
'''天法'''（{{bo|b=གནམ་ཆོས་|w=gnam chos|t=namchö}}），直譯為「天或空間之法」，乃一[[伏藏|伏藏]]法，在[[藏傳佛教|藏傳佛教]][[寧瑪派|寧瑪派]][[白玉寺|白玉傳承]]中廣為流傳。其為大伏藏師南卻明珠多傑（Tertön Mingyur Dorjé）所發現，傳承予昆桑謝拉（Kunzang Sherab），最後由[[噶舉派|噶舉派]]大師噶瑪恰美（Karma Chagme）所編撰成文。

天法含括了完整的教法，從{{le|前行|Ngöndro}}（{{bo|b=སྔོན་འགྲོ་|w=sngon 'gro|t=ngöndro}}）至[[大圓滿|大圓滿]]的「心性直指」。明就多傑之伏藏據說緣起於他與天神相遇，但也包括了由他見地所引發的指導。

== 主題 ==
天法內容包括了如下主題：
# 供養儀式（如[[煙供|煙供]]（bsang）、水供（chab gtor）、寶瓶供養（bum gter））
# 喪禮儀式（byang chog)
# 廣為流行之長壽灌頂（tshe dbang）、健康灌頂（sman lha dbang）以及財富灌頂（nor dbang）
# 護身儀式（mdos srung ba）
# 祈請護法儀式
# 占卜和天象（r'deu dkar mo / spar kha / rtsis）
# 前行
# 密續修習（rmi lam, 'pho ba, gtum mo, phur ba, gcod）與論述（rgyud 'grel）
# 淨土修行（zhing khams sgrub）以及[[寂靜尊|寂靜尊]]、[[忿怒尊|忿怒尊]]觀想修習
