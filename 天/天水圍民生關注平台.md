'''天水圍民生關注平台'''（{{lang-en|'''Tin Shui Wai New Force'''}}，簡稱：天民台），是一個於2014年[[雨傘革命|雨傘革命]]後催生、成立的[[香港|香港]]地區組織，以[[新界西|新界西]]、[[天水圍|天水圍]]和[[元朗|元朗]]為主要服務地區，從地區層面悍衛香港人權益。

== 背景 ==
2014年，雨傘革命金鐘清場前夕，各佔領區中都有市民對雨傘革命的失敗感到不甘，並對現今的議會失望，探討當中原因，認為香港人應嘗試重奪議會。有網民於[[香港高登討論區|香港高登討論區]]發起「2015區議會計劃 (又稱高登區議會計畫)」<ref>{{Cite web|url=http://forum5.hkgolden.com/view.aspx?type=CA&message=5488356|title=【反攻建制】2015區議會計劃(1)|accessdate=2014-11-04|author=茶呢就兩糖匙|date=2014-11-04|last=|publisher=}}</ref>，並於各佔領區中進行線下宣傳，希望號召網民參選2015年區議會選舉，挑戰上一屆中自動當選的建制派區議會席位，以打亂建制派參選部署，防止議席因自動當選而自動落入建制派手上，並希望藉此形成地區勢力，將年青人聲音帶入議會，長遠劍指立法會。

== 成立 ==
同年11月下旬，一群元朗及天水圍年輕人因雨傘革命而啟發，並響應「高登區議會計畫」，經歷數次會議後，創立天水圍民生關注平台，以抗衡一直被建制派佔多數的元朗區議會議席，成立後致力以為天水圍居民締造更好的生活環境為己任，悍衛香港人權益，專注民生動向並提高區內公民意識，堅持獨立無政黨，維持組織獨立性。

天民台成員及義工多為居於屯元天的年青政治素人，來自四面八方、不同背景、不同學歷專長、不同年齡，以自己空餘時間投入組織及地區工作，服務社區，同時吸收經驗，為往後發展作準備。

== 組織宗旨 ==
# 為天水圍居民締造更好的生活環境
# 悍衛香港人權益
# 專注民生動向
# 提高區內公民意識
# 堅持獨立無政黨

==過往活動==

=== 天水地攤：復活市集 ===
2015年4月，天水圍民生關注平台假[[香港管理專業協會羅桂祥中學|香港管理專業協會羅桂祥中學]]舉行「[[天水地攤：復活市集|天水地攤：復活市集]]」，希望透過市集能過釋放社區活力，社區幹事[[王百羽|王百羽]]接受《[[蘋果日報|蘋果日報]]》訪問。<ref>{{zh-hant}}{{cite web|url=http://hk.apple.nextmedia.com/news/art/20150405/19102266|title=八十後搞地攤 天水圍的復活市集 |publisher=[[蘋果日報|蘋果日報]]|date=2015-04-05 |accessdate=2015-11-24}}</ref>

=== 洪水橋新發展區公眾論壇 ===
2015年8月，天水圍民生關注平台社區幹事王百羽及成員參與[[洪水橋新發展區|洪水橋新發展區]]第三階段公眾咨詢的公眾論壇，代表天水圍居民就洪水橋新發展區對天水圍的交通影響﹑未來規劃等問題提出意見。

=== 發現天水圍北有可疑選民登記 ===
2015年8月，天水圍民生關注平台社區幹事王百羽發現天水圍北兩個屋邨共有10個可疑選民登記，並向[[選舉事務處|選舉事務處]]舉報。<ref>{{zh-hant}}{{cite web|url=http://www.mingpaocanada.com/realtimenews/VAN/content_hk_new.cfm?aid=179442&m=0 |title=【不存在的地址】天水圍兩屋邨現10個可疑選民登記 |publisher=[[明報|明報]]|date=2015-08-25|accessdate=2015-11-24}}</ref>

== 選舉 ==

=== 2015年區議會選舉 ===
{{main|2015年香港區議會選舉}}
天水圍民生關注平台自2015年起參加區議會選舉。

2015年10月，天水圍民生關注平台派出社區幹事[[王百羽|王百羽]]參選2015年區議會選舉（M22[[天恒選區|天恒選區]]）<ref>{{zh-hant}}{{cite web|url=http://www.elections.gov.hk/dc2015/chi/intro_to_can/M.html |title=候選人簡介 |publisher=[[香港區議會|香港區議會]] |accessdate=2015-11-24}}</ref> ，挑戰上屆區議會自動當選的[[工聯會|工聯會]][[陸頌雄|陸頌雄]]。王百羽就四大範疇﹑提出九大政綱，尋求居民支持<ref>{{zh-hant}}{{cite web|url=http://wongpakyu.wix.com/tinheng-new-changes#!----/rtiw0|title=王百羽。心繫天恆｜參選政綱 - WIX.com|publisher=[[王百羽|王百羽]]|accessdate=2015-11-24|deadurl=yes|archiveurl=https://web.archive.org/web/20151124184549/http://wongpakyu.wix.com/tinheng-new-changes#!----/rtiw0|archivedate=2015-11-24}}</ref>。

2015年11月22日，王百羽在是次區議會選舉落選，得票1,317票，僅次於陸頌雄的2,749票。

=== 2016年立法會選舉 ===
{{main|2016年香港立法會選舉}}
天水圍民生關注平台自2016年起參加立法會選舉。

2016年立法會選舉，天水圍民生關注平台與[[青年新政|青年新政]]，[[東九龍社區關注組|東九龍社區關注組]]，[[慈雲山建設力量|慈雲山建設力量]]，[[長沙灣社區發展力量|長沙灣社區發展力量]]，[[屯門社區關注組|屯門社區關注組]]組成[[民族自決派|ALLinHK]]選舉聯盟，並派出發言人[[王百羽|王百羽]]與召集人[[黃俊傑|黃俊傑]]合組名單參選新界西選區。
