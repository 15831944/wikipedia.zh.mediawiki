'''天城文'''（{{lang|sa|देवनागरी}} {{IAST|''devanāgarī''}}）又称'''天城体'''。是[[印度|印度]]和[[尼泊尔|尼泊尔]]的一种文字，用来书写[[印地语|印地语]]、[[梵語|梵語]]、[[尼泊尔语|尼泊尔语]]等语言。天城文最早出现在13世纪初，是[[城文|城文]]变体之一，天城文是对城文的改良而成，改良后的城文为突出其神圣加多了个梵文“天”字（{{lang|sa|देव}} ''deva''）成为天城文<ref>{{Cite web |url=http://www.harekrsna.com/sun/features/06-07/features681.htm |title=存档副本 |access-date=2014-02-19 |archive-url=https://web.archive.org/web/20141128093725/http://www.harekrsna.com/sun/features/06-07/features681.htm |archive-date=2014-11-28 |dead-url=no }}</ref>。城文来自[[笈多文|笈多文]]，笈多文犹如印度的其他文字一样，源自於前3世纪的[[婆罗米文|婆罗米文]]。現在亞洲不少民族使用的字母与天城文的关系密切，而从[[帕拉瓦文|帕拉瓦文]]派生的文字则在[[緬甸|緬甸]]、[[泰國|泰國]]、[[柬埔寨|柬埔寨]]、[[老撾|老撾]]等地使用。<ref>Braj B. Kachru (2008), Language in South Asia p.302,  Cambridge University Press, ISBN 978-0-521-78653-9.</ref>

== 原理 ==
作為[[婆羅米系文字|婆羅米系]][[元音附標文字|元音附標文字]]，天城文的基本原理是每個字母表示一個輔音，并承載一個固有的元音{{IAST|a}} {{IPA|[ə]}}。<ref>{{Harvcoltxt|Salomon|2003|p=70}}</ref>例如，字母{{lang|sa|क}}讀作{{IAST|ka}}，兩個字母{{lang|sa|कन}}讀作{{IAST|kana}}，三個字母{{lang|sa|कनय}}讀作{{IAST|kanaya}}等等。与其他元音拼合或去掉元音时，這些輔音或它們的字母会有变化：

* [[輔音音叢|輔音音叢]]要[[合字|合寫]]（{{IAST|saṃyuktākṣara}}“复合”）。例如，三個字母{{lang|sa|कनय}} {{IAST|kanaya}}可以合寫形成{{lang|sa|क्नय}} {{IAST|knaya}}、{{lang|sa|कन्य}} {{IAST|kanya}}或{{lang|sa|क्न्य}} {{IAST|knya}}。
* 除固有元音{{IAST|a}}外的元音在和辅音拼合时寫為[[附加符号|元音符号]]。例如，{{lang|sa|क}} {{IAST|ka}}可能写成{{lang|sa|के}} {{IAST|ke}}、{{lang|sa|कु}} {{IAST|ku}}、{{lang|sa|की}} {{IAST|kī}}、{{lang|sa|का}} {{IAST|kā}}等。 
* 對於沒有輔音的元音，當位於詞首或其他元音后时，写成完整的字母。例如，{{IAST|ū}}在{{lang|sa|कू}} {{IAST|kū}}中寫為[[變音符號|變音符號]]{{lang|sa|ू}}，而在{{lang|sa|ऊक}} {{IAST|ūka}}和{{lang|sa|कऊ}} {{IAST|kaū}}中会写成字母{{lang|sa|ऊ}}。
* 詞尾的輔音標記的[[變音符號|變音符號]]{{lang|sa|्}}是去元音符号，梵語中称做{{IAST|virāma}}，在印地語中又叫halanta。它去除固有元音，所以{{lang|sa|क्नय}} {{IAST|knaya}}可以写成{{lang|sa|क्नय्}} {{IAST|knay}}。当輔音音叢不能通过字母变形做到合寫的時候經常就会使用去元音符号。

這样一个字母或帶有變音符號的合寫，称做一个{{IAST|akṣara}}“音節”。例如{{lang|sa|कनय}} {{IAST|kanaya}}被算作三個音節，而{{lang|sa|क्न्य}} {{IAST|knya}}和{{lang|sa|कु}} {{IAST|ku}}则都被算作一個音節。

== 字母 ==
{{IPA notice}}
天城文的字母排序，類似於幾乎所有婆羅米系文字，是以考虑了元音和輔音[[發音方式|發音方式]]和[[發音位置|發音位置]]二者的[[音韻學|語音]]原理作为依据的。這種安排通常稱為{{IAST|varṇamālā}}“字母的花環”。<ref>{{Harvcoltxt|Salomon|2003|p=71}}</ref>用於[[梵語|梵語]]的天城文樣式被看做是原型，被其他語言使用时会有小的變化或扩充。<ref name="Salomon2003p75">{{Harvcoltxt|Salomon|2003|p=75}}</ref>

=== 合寫 ===
[[File:JanaSanskritSans_ddhrya.svg|thumb]]

如上所述，其間去除元音的連續的輔音可以按照法则合寫在一起為[[合字|合字]]（复合辅音）。這些支配輔音音叢的規則，有的普遍适用，有的仅对少数辅音适用，还有一些例外。儘管绝大部分已经标准化，但是在輔音音叢中仍有特定的變體，本頁所基於的[[Unicode|Unicode]]用到的只是一種方案。下面是一些規則：

* 36個輔音中有24個包含右豎杠（{{lang|sa|2=<span style="font-size:150%;">ख</span>、<span style="font-size:150%;">घ</span>、<span style="font-size:150%;">ण</span>}}等）。作為輔音音叢的第一或中間片段，它們要去掉這個豎杠。比如{{lang|sa|2=<span style="font-size:150%;">त</span> + <span style="font-size:150%;">व</span> = <span style="font-size:150%;">त्व</span>，<span style="font-size:150%;">ण</span> + <span style="font-size:150%;">ढ</span> = <span style="font-size:150%;">ण्ढ</span>，<span style="font-size:150%;">स</span> + <span style="font-size:150%;">थ</span> = <span style="font-size:150%;">स्थ</span>}}。{{lang|sa|2=<span style="font-size:150%;">श</span> {{IAST|ś(a)}}在<span style="font-size:150%;">व</span> {{IAST|va}}、<span style="font-size:150%;">न</span> {{IAST|na}}、<span style="font-size:150%;">च</span> {{IAST|ca}}、<span style="font-size:150%;">ल</span> {{IAST|la}}}}和{{lang|sa|2=<span style="font-size:150%;">र</span> {{IAST|ra}}}}之前時，要寫為{{lang|sa|2=<span style="font-size:150%;">श्व</span> {{IAST|śva}}、<span style="font-size:150%;">श्न</span> {{IAST|śna}}、<span style="font-size:150%;">श्च</span> {{IAST|śca}} <span style="font-size:150%;">श्ल</span> {{IAST|śla}}}}和{{lang|sa|2=<span style="font-size:150%;">श्र</span> {{IAST|śra}}}}。

* {{lang|sa|2=<span style="font-size:150%;">र</span> {{IAST|r(a)}}}}去元音后作為合字的第一个字母時，写作后面字母或后面字母{{lang|sa|2=<span style="font-size:150%;">ा</span> {{IAST|ā-}}}}附標上的彎曲上折線，比如{{lang|sa|2=<span style="font-size:150%;">र्व</span> {{IAST|rva}}、<span style="font-size:150%;">र्वा</span> {{IAST|rvā}}、<span style="font-size:150%;">र्स्प</span> {{IAST|rspa}}、<span style="font-size:150%;">र्स्पा</span> {{IAST|rspā}}}}。当它被置于去元音后的{{lang|sa|2=<span style="font-size:150%;">ट ठ ड ढ ङ छ</span>}}字母之後，两者合写时，则写成這些字母下分叉的兩条斜線，即{{lang|sa|2=<span style="font-size:150%;">ट्र ठ्र ड्र ढ्र ङ्र छ्र</span>}}。在其他地方作為最后一個字母时，它被表示為左下角的斜線，比如{{lang|sa|2=<span style="font-size:150%;">क्र ग्र भ्र</span>}}。{{lang|sa|2=<span style="font-size:150%;">त</span> {{IAST|ta}}}}上移得到{{lang|sa|2=<span style="font-size:150%;">त्र</span> {{IAST|tra}}}}。

* 第一個字母無右豎杠（例如{{lang|sa|2=<span style="font-size:150%;">द</span> {{IAST|d(a)}}}}和{{lang|sa|2=<span style="font-size:150%;">ह</span> {{IAST|h(a)}}}}）的，可以對第二个字母做縮小和去橫杠處理，然后写在第一个字母的下方。{{lang|sa|2=<span style="font-size:150%;">क</span> {{IAST|k(a)}}}}、{{lang|sa|2=<span style="font-size:150%;">छ</span> {{IAST|ch(a)}}}}和{{lang|sa|2=<span style="font-size:150%;">फ</span> {{IAST|ph(a)}}}}则将右半边钩状的部分縮短然后直接連接上隨后的字符。

* 合寫{{lang|sa|2=<span style="font-size:150%;">क्ष</span> {{IAST|kṣ}}}}和{{lang|sa|2=<span style="font-size:150%;">ज्ञ</span> {{IAST|jñ}}}}不能直接從被复合的辅音中变形而来。{{lang|sa|2=<span style="font-size:150%;">क्ष</span>}}是{{lang|sa|2=<span style="font-size:150%;">क्</span> + <span style="font-size:150%;">ष</span>}}的合写，而{{lang|sa|2=<span style="font-size:150%;">ज्ञ</span>}}是{{lang|sa|2=<span style="font-size:150%;">ज्</span> + <span style="font-size:150%;">ञ</span>的}}合寫。

=== 书写形式 ===
天城文，原本的书写形式是用连书形式，自左向右，横向书写，词与词之间不留空格，一句话从头到尾连续不断地拼写，词与词之间用空格隔开是近代的事。

把天城文轉寫成[[拉丁字母|羅馬文字]]的方法有许多。最廣泛使用的轉寫方案是[[IAST|IAST]]。下面列出主要的天城文轉寫方法：

=== IAST ===
[[IAST|國際梵語轉寫字母（IAST）]]是梵語羅馬化的學術標準。IAST成为了用在書籍和期刊等印刷品中事实上的標準，加之其Unicode字體更容易得到，它也日益用於電子文本。它根据的是1912年在[[雅典|雅典]]召開的“東方學學者大會”所確立的標準。

[[加爾各答國家圖書館羅馬化|加爾各答國家圖書館羅馬化]]意圖羅馬化所有[[婆羅米系文字|印度文字]]，它是IAST的擴展。

=== Harvard-Kyoto ===
相對於IAST，[[Harvard-Kyoto|Harvard-Kyoto]]看起來更簡單。它不包含所有IAST的變音符號。這使得錄入Harvard-Kyoto要比IAST容易。Harvard-Kyoto使用[[大寫字母|大寫字母]]。

=== ITRANS ===
[[ITRANS|ITRANS]]是廣泛用於[[Usenet|Usenet]]的天城文到[[ASCII|ASCII]]的無損轉寫方案。它是Harvard-Kyoto方案的擴展。在ITRANS中，詞{{IAST|Devanāgarī}}寫為“Devanaagarii”。ITRANS有同名應用程序確保錄入[[婆羅米系文字|印度文字]]。用戶輸入羅馬字母而ITRANS預處理器把羅馬字母顯示為天城文（或其他印度語言）。最新版本的ITRANS是2001年7月的5.30版。

=== ISO 15919 ===
{{Main|ISO 15919}} 
ISO 15919是2001年確立的轉寫國際標準。特定於天城文的那部分內容幾乎同一於作為梵語學術標準的IAST。

=== ALA-LC羅馬化 ===
[http://www.loc.gov/catdir/cpso/roman.html ALA-LC]是[[美國國會圖書館|美國國會圖書館]]和[[美國圖書館協會|美國圖書館協會]]的轉寫標準，廣泛的用於北美圖書館。轉寫表基於各種語言，有對[http://www.loc.gov/catdir/cpso/romanization/hindi.pdf 印地語]的表格，也有對[http://www.loc.gov/catdir/cpso/romanization/sanskrit.pdf 梵語和俗語]的表格。

== 編碼 ==
=== ISCII ===
[[ISCII|ISCII]]是[[定長編碼|定長]]8-位編碼。低128代碼點是普通[[ASCII|ASCII]]，高128代碼點是ISCII-專屬。

它被設計用來不只表示天城文，所以還有各種其他[[婆羅米系文字|印度文字]]和用於轉寫印度文字的基於拉丁文字的變音符號。

ISCII在很大程度上被Unicode所替代，但已經嘗試在它的印度語區塊中保持了ISCII格局。

=== Unicode中的天城文 ===
Unicode的天城文區間是U+0900 .. U+097F，U+1CD0 .. U+1CFF和U+A8E0 .. U+A8FF。灰色區域指示未指派的代碼點。

== 天城文鍵盤布局 ==
=== INSCRIPT ===
[[InScript|InScript]]是給天城文的標準[[電腦鍵盤|鍵盤]]佈局。它內建於所有主要[[操作系统|操作系统]]中。[[Microsoft_Windows|Microsoft Windows]]支持InScript佈局（使用[[Mangal_(字體)|Mangal]]字體），它可以被用來輸入Unicode天城文字符。InScript在一些觸控螢幕[[手機|手機]]上也可獲得。[[File:KB_Devanagari_InScript_text_1.svg|800px]]

=== 打字機 ===
這種佈局在没有計算機或计算机不常用的時代用在手動[[打字機|打字機]]上。出於向後兼容性一些錄入工具如[[Indic_IME|Indic IME]]仍提供這種佈局。[[File:Hindi_typewriter.jpg|800px]]
