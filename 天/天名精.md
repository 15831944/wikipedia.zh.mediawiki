'''天名精'''（[[学名|学名]]：{{lang|la|''Carpesium abrotanoides''}}）是[[菊科|菊科]][[天名精属|天名精属]]的植物。

== 形态 ==
多年生[[草本|草本]]；茎直立，上部多分枝呈二叉状；茎下部的长椭圆形叶子互生；头状花序多数，腋生半下垂，夏秋开黄色管状花；褐黑色[[瘦果|瘦果]]。

== 分布 ==
分布在[[俄罗斯|俄罗斯]]、[[朝鲜|朝鲜]]、[[越南|越南]]、[[伊朗|伊朗]]、[[缅甸|缅甸]]、[[日本|日本]]以及[[中国大陆|中国大陆]]的[[华南|华南]]、[[西南|西南]]、[[华中|华中]]、[[华东|华东]]、[[陕西|陕西]]等地，生长于海拔2,000米的地区，一般生于溪边、村旁、路边荒地及林缘，目前尚未由人工引种栽培。

== 别名 ==
鹤？（梦溪笔谈，蜀本草）、天蔓青、天菘（名医别录）
