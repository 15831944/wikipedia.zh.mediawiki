'''天水圍第109區文物修復保存中心'''（前稱 '''博物館中央文物貯藏大樓'''），為[[香港|香港]][[康樂及文化事務署|康樂及文化事務署]]計劃於香港[[新界|新界]][[元朗區|元朗區]][[天水圍|天水圍]]109區（即[[香港濕地公園|香港濕地公園]]旁）興建，佔地約5,000平方米，樓高10層，樓面總面積約為18,700[[平方米|平方米]]，作為集中貯藏現時分散於香港各區的館藏，使到香港所收藏的[[文物|文物]]得到更妥善的[[保護|保護]]及[[管理|管理]]等。此外，其中1層、約2,000平方米，計劃作為[[新界西|新界西]]及[[新界北|新界北]]首座專題[[展覽廳|展覽廳]]──'''天水圍大樓展覽廳'''。博物館中央文物貯藏大樓原預計於2015年至2016年啟用。唯到2016年4月，康文署表示正在積極籌劃中，未有確實落成日期。

== 位置 ==
天水圍109區位於[[天業路公園|天業路公園]]旁，毗鄰學校及[[天晴邨|天晴邨]]，屬於「政府、機構或社區」用地。至今，部分用地暫時供予[[土木工程拓展署|土木工程拓展署]]作為臨時工地辦公室。

== 設施 ==
*文物貯藏庫：室內環境具有恆常濕度、防火及防蟲等條件；用戶包括[[香港歷史博物館|香港歷史博物館]]、[[香港藝術館|香港藝術館]]、[[香港電影資料館|香港電影資料館]]、[[香港科學館|香港科學館]]、[[香港太空館|香港太空館]]、[[香港文化博物館|香港文化博物館]]及[[古物古蹟辦事處|古物古蹟辦事處]]。
*[[專科|專科]][[文物復修|文物復修]][[工作室|工作室]]
**文物滅蟲室
**攝影室
**檔案室，等等
*天水圍大樓展覽廳：佔博物館中央文物貯藏大樓一整層，面積約2,000平方米，開放予公眾使用為文化教育設施。當中約600平方米為專題展覽廳，供予定期舉辦以[[藝術|藝術]]、[[歷史|歷史]]及[[科學|科學]]為主題的[[展覽|展覽]]。
*多功能教育活動室：兩座
*文物資源中心：提供嶄新學習平台，方便公眾對文物藏品，包括一些未展示過的文物，進行實物研習。
*[[天文觀測|天文觀測]]及[[研究|研究]]工作室：位於[[天台|天台]]，供予公眾使用先進設施進行天文觀測。

== 歷史 ==
至2012年年底，康樂及文化事務署轄下的14間公共博物館館藏數量達逾20萬件，香港電影資料館館藏數量達80萬件；隨著香港文物的收藏品數量與日俱增，大部分[[香港博物館|香港博物館]]均面臨貯藏空間短缺的問題。2012年，康樂及文化事務署表示計畫於天水圍109區興建香港首座博物館中央文物貯藏大樓，集中貯藏上述14間公共博物館現時分散於香港各區的館藏。康樂文化事務署計畫於博物館中央文物貯藏大樓啟用後，於該處舉辦文物修護展覽，及介紹其[[專業|專業]]，更考慮日後將一些位於[[香港市區|香港市區]]主要博物館所舉辦的重要展覽移師至天水圍大樓展覽廳展覽。有關建議倘若獲得[[區議會|區議會]]及居民支持，康樂及文化事務署將會根據既定程序確定發展範圍，然後交由[[建築署|建築署]]展開工程可行性研究<ref>[http://hk.news.yahoo.com/開放-層作專題展覽廳-天水圍建文物貯藏大樓-220607692.html; 開放一層作專題展覽廳 天水圍建文物貯藏大樓] 《星島日報》 2013年1月1日</ref><ref>[http://www.singtao.com/yesterday/loc/0101ao08.html 天水圍建文物貯藏大樓] 《星島日報》 2013年1月1日</ref>，預計於2015年至2016年啟用<ref>[http://orientaldaily.on.cc/cnt/news/20130101/00176_061.html 天水圍建文物儲藏大樓] 《東方日報》 2013年1月1日</ref>。

2013年1月4日，康樂及文化事務署於[[元朗區議會|元朗區議會]]會議上，就有關建議諮詢[[區議員|區議員]]的意見<ref>[http://paper.takungpao.com/html/2013-01/01/content_13_2.htm 14博物館百萬件藏品將有安身之所 濕地公園旁建文物大樓] 《大公報》 2013年1月1日</ref>。民政事務局於2014年7月批出工程界定書，並邀請[[建築署|建築署]]進行技術可行性研究。

2019年4月4日，建築署就此大樓展開的建築顧問招標開標，確定由[[巴馬丹拿|巴馬丹拿]]建築及工程師有限公司負責設計此大樓，並預計2026年5月完工。
