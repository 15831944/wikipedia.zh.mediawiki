《'''天士夢'''》（{{lang-ko|천사몽}}）是一部2001年上映的韓國電影，由朴熙俊（{{kor|박희준}}）執導，[[黎明|黎明]]、朴銀惠（{{kor|박은혜}}）、[[李娜英|李娜英]]（{{kor|이나영}}）等演出，以科幻、動作、愛情為電影題材，在韓國拍攝及上映<ref>{{Cite web |url =http://people.com.cn/BIG5/channel1/14/20001015/272478.html|title =黎明接拍韓國電影《天士夢》|date = 2000年10月15日|publisher =人民網|language = 中文}}</ref> <ref> {{Cite book | author = 鄭軍 | title =光影兩萬里：世界科幻影視簡史| location =中國| publisher =百花文藝出版社| date =2012年4月1日|ISBN = 9787530660096 | language = 中文}} </ref>。

==情節介紹==
電影內容講述城真是警局的特戰隊的成員，他每天晚上都做了同樣的夢，夢見一個女人被怪物追趕，每當這個女人到了危險關頭，城真就會從睡夢中驚醒。第二天，城真回到警局與上司討論案情，上司播放了關於張博士的研究室遭到宗教暴力集團闖入破壞的錄影片段，希望能夠破獲這個重大案件。在調查行動中，城真來到張博士的研究室，期間看到一張照片，照片中的女人正好就是他每晚夢見的女人，原來她是張博士的女兒南紅。張博士正進行一項關於前生的研究，為了用科學來證明前生的存在，張博士製造了時光隧道，並利用女兒來做實驗，實驗期間卻遭到暴力集團闖入破壞，導致女兒無法從時光隧道回來。張博士認為城真能夠幫助拯救女兒南紅，而城真亦希望能解開夢境的謎團，於是答應張博士的要求。城真在實驗過程中發現自己和南紅在前生是一對戀人，但在門當戶對的制度下不能相愛。城真所身處的時空突然發生了戰爭，他冒著不能返回現在的風險，嘗試去拯救身處險境的南紅，展開了一段科幻的生死愛情故事<ref>{{Cite web |url =http://ent.sina.com.cn/m/c/2004-04-24/0324374170.html|title =黎明《天士夢》通過審查 音像製品獲准內地發行|date = 2004年4月24日|publisher =華西都市報|via=新浪影音娛樂|language = 中文}}</ref><ref>{{Cite web |url =http://www.hjzlg.com/move/YCMS_ShowArt.asp?id=883|title =韓國電影劇情《天士夢》| publisher =韓粉樂園| language = 中文}}</ref>。

==電影歌曲==
《天士夢電影原聲大碟》於2001年在韓國發行，專輯收錄了十六首電影原聲配樂及歌曲，當中由香港知名歌手黎明演唱主題曲《天士夢》（천사몽）<ref>{{Cite web |url =http://www.maniadb.com/album/130288?o=l&s=0|title =천사몽（Dream Of A Warrior）|publisher =www.maniadb.com |language =ko}}</ref>。
