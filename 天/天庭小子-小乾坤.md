== 品德教育推廣 ==

2017年，國際獅子會贈送雲林縣政府所屬各小學，以小乾坤作為品德教育補充教材，與小學合辦品德體驗營、品德達人徵文比賽、小乾坤動動操等活動<ref name="LION1">{{cite news|author1=詹士弘|title=獅子慶百年 品德小達人大跳「小乾坤動動操」|url=http://news.ltn.com.tw/news/life/breakingnews/2324687|publisher=自由時報|date=2018-01-27|archiveurl=https://web.archive.org/web/20180501165059/http://news.ltn.com.tw/news/life/breakingnews/2324687|archivedate=2018-05-02}}</ref><ref name=ltn2>{{cite news|author1=林國賢|title=捐贈百套弟子規動畫教材 獅子會300-D1區推品德教育|url=http://news.ltn.com.tw/news/life/breakingnews/2170607|publisher=自由時報|date=2017-08-22|archiveurl=https://web.archive.org/web/20180501165218/http://news.ltn.com.tw/news/life/breakingnews/2170607|archivedate=2018-05-01|deadurl=yes}}</ref><ref>{{cite news|author1=洪佳伶|title=翻轉雲林教育　縣長期勉各校「進學益教、勇敢創新」|url=https://www.nownews.com/news/20170822/2601046|publisher=Nownews|date=2017/08/22|archiveurl=https://web.archive.org/web/20180501165904/https://www.nownews.com/news/20170822/2601046|archivedate=2018-05-02}}</ref>。苗栗竹南鎮公所，合辦「品德列車」活動，邀集各幼稚園參加。<ref>{{cite news|title=小乾坤品德列車到竹南 學習生活常規|url=https://www.tbc.net.tw/News/NewsDetail?id=d112976b-b4dc-41c8-be6a-dd17c08039c4&newsType=3|publisher=TBC電視新聞|date=2018-01-23}}</ref>

== 劇情簡介 ==

阿雅、豆豆和小乾坤在天上是好朋友，兩人為了拯救小乾坤而犧牲了生命，轉生到人間。小乾坤帶著《弟子規》下凡報答他們的恩情。在人間的豆豆和阿雅總是在生活中製造許多麻煩，小乾坤運用《弟子規》中的智慧，教導他們面對考驗，改正不良的習慣，成為善良懂事的好孩子。
