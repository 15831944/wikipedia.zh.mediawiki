'''天台小學'''是指在[[徙置大廈|徙置大廈]][[天台|天台]]開設的[[小學|小學]]，是[[香港|香港]]在[[二次大戰|二次大戰]]後出現高出生率及[[國共內戰|中國國共內戰]]引致大量移民湧入後，[[香港人口|香港人口]]短時間內急劇增長，但教育資源貧乏，學校缺乏獨立校舍而產生的現象。

== 背景 ==
大部分天台小學都開辦於1950年代及1960年代，這段時期的香港仍處於[[戰後嬰兒潮|戰後嬰兒潮]]時期，加上大量移民湧入，導致香港人口激增，但當時[[香港經濟|香港經濟]]仍處於發展中地區，所以政府缺乏資源投放教育。雖然政府設有官立學校提供基本教育，但學額嚴重不足所以收生競爭激烈，擁有自設校舍的私立學校，其學費又不是一般家庭所能負擔，私人辦學的團體在當時也難以得到政府提供土地設置校舍，所以在天台設立的小學，成為當時一般家庭的兒童接受教育的途徑，而政府以象徵式每年收取1港元的租金，提供徙置大廈的天台給志願及宗教團體開辦學校<ref name="apple"/>。

== 出現 ==
[[微勞士|微勞士]][[牧師|牧師]]（Rev. Verent John Russell Mills）在1950年代，建議[[香港殖民地時期#香港政府|香港政府]]在徙置大廈的[[天台|天台]]上設立[[小學|小學]]，由[[志願機構|志願機構]]（例[[信義會|信義會]]）開辦，為基層家庭的子女提供[[教育|教育]]機會。早期的天台小學設備相當簡陋，但當時能把子女送進天台小學已經難能可貴<ref>陳振威，[http://www.godoor.net/text/history/zhuanji13.htm 孤兒之父--微勞士牧師]。</ref><ref>[http://www.baptist.org.hk/b5_info01_pop.php?id=197 九龍灣浸信會每週講道資料庫]，可見到微勞士牧師，確有其人</ref>。天台小學只開設在第一型至第三型的徙置大廈。

== 淘汰 ==
為解決校舍不足的問題，香港政府除增加官立學校數量外，政府同時為接受政府資助條件的辦學團體提供資金援助及協助興建校舍成為政府資助學校，而新落成的屋邨及新市鎮都會規劃足以應付區內需要的學校校舍，興建學校不但令學童可以在設施較佳的校舍上課，建設足夠的校舍也為1978年起推行的[[香港教育|九年義務教育]]鋪平道路。由於徙置大廈從1970年代中期起被拆卸重建為有獨立起居設施的屋邨，天台小學因此逐漸停辦或遷到新建校舍繼續辦學，天台小學到1980年代初期已完成其歷史角色而全面淘汰<ref name="apple">{{cite news|title=1977年政府大幅加租 天台學校成歷史|url=https://hk.news.appledaily.com/local/realtime/article/20140928/52942727|publisher=蘋果日報|date=2014-09-08}}</ref>。

== 演變 ==
雖然天台小學於1970年代末至80年代初期被淘汰，但其實不少天台小學後來都獲配獨立校舍繼續辦學，如由[[泛民主派|泛民主派]]元老[[司徒華|司徒華]]創辦的[[葛量洪校友會觀塘學校|葛量洪校友會觀塘學校]]，原址在[[觀塘新區|觀塘新區]]某徙廈天台，後遷往[[順安邨|順安邨]]第二校舍辦學至2009年結束，又如[[葵盛邨|葵盛邨]]的[[聖公會主恩小學|聖公會主恩小學]]早年是[[大窩口邨|大窩口徙置區]]的天台學校、[[長沙灣|長沙灣]][[聖公會聖紀文小學|聖公會聖紀文小學]]早年是位於[[李鄭屋邨|李鄭屋徙置區]]，現於牛頭角[[樂華邨|樂華邨]]興辦的樂善堂楊仲明學校，前身是位於[[油塘邨|油塘新區]]第10座天台的九龍樂善堂油塘灣學校等等，例子不勝枚舉。在2010年3月上映的電影《[[歲月神偷|歲月神偷]]》，電影中多幕可見到戲中角色「羅進二」在天台小學讀書的景象。
