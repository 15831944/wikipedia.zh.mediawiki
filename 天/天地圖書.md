'''天地圖書有限公司'''（'''Cosmos Books Ltd.'''），在1976年成立<small><ref>{{cite web |language=zh-hant |url=http://www.cns.hk:89/ga/news/2006/12-09/834695.shtml |title=香江漫話：“天地圖書”卅歲慶 金庸難得二字贈 |publisher=[[中國新聞網|中國新聞網]] |date=2006-12-09 |accessdate=2009-10-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20100114013801/http://www.cns.hk:89/ga/news/2006/12-09/834695.shtml |archivedate=2010-01-14 }}</ref></small>，是[[香港|香港]][[出版社|出版社]]之一，集出版、發行及零售於一身。陳松齡是天地圖書董事長。天地是中資公司[[聯合出版集團|聯合出版集團]]成員之一<ref>{{cite news|url=https://www.hk01.com/特首選舉2017/56145/選委戰-近半出版界選委參選人-曾任中聯辦旗下公司|title=選委戰-近半出版界選委參選人-曾任中聯辦旗下公司}}</ref>。

天地圖書以「以兩條腿走路」的方式營運，一方面出版通俗流行讀物，同時亦出版具有水準文藝作品，兼顧不同讀者群的需要。成立之初，天地圖書出版一些時事評論方面的著作，也出版一些翻譯[[文學作品|文學作品]]、[[政治|政壇]]要人的回憶錄及[[武俠小說|武俠小說]]，著名武俠小說作家[[梁羽生|梁羽生]]的著作都是委託天地圖書出版<small><ref>{{cite web |language=zh-hk |url=http://www.takungpao.com/news/09/02/17/liangyusheng01-1033881.htm |title=追思梁羽生/ 孫立川: 病床前仍吟書中詩詞 |publisher=[[大公報|大公網]] |date=2009-02-17 |accessdate=2009-10-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20091108200847/http://www.takungpao.com/news/09/02/17/liangyusheng01-1033881.htm |archivedate=2009-11-08 }}</ref></small>；1980年代初期，隨著[[中國內地|中國內地]]思想解放，出版了一個系列內地作家作品集《天地文叢》。

==門市部==
位於香港[[灣仔|灣仔]][[莊士敦道|莊士敦道]]30號的總店。

總店佔地面積一萬六千多平方呎，為港島最大規模的書店，除中文部外，尚設有英文部、文具部和雜誌部，此外還有附設的小展覽廳，定期舉辦專題展銷和公開座談會。

=== 歷史 ===
天地圖書初時只有在[[香港島|香港島]]開設[[書籍|書籍]][[商店|門市部]]（總店），自1999年起在[[尖沙咀|尖沙咀]][[彌敦道|彌敦道]]開設分店<small><ref>{{cite web |language=zh-hk |url=http://www.hkbsia.org/index.php?gOo=article_details.dwt&articleid=116 |title=天地圖書新尖沙咀門市現已開張 |publisher=香港圖書文具業商會 |date=2008-08-28 |accessdate=2009-10-21}}</ref></small>，直至2011年九龍店遷至[[旺角|旺角]]。

1999年天地圖書的灣仔總店因電線漏電而引致火警，大批書刊被浸被焚，連裝修估計損失達3,000萬港元。

==出版部==
天地圖書出版[[時尚|流行]]讀物、[[娛樂|消閒]]小品、嚴肅[[文學|文學]]、[[歷史|歷史]]、[[政治|政治]]、[[社會科學|社會科學]]類[[圖書|圖書]]等。天地圖書的三大皇牌作者是[[亦舒|亦舒]]、[[李碧華|李碧華]]及[[蔡瀾|蔡瀾]]<small><ref>{{cite web |language=zh-hk |url=http://hknews.hksyu.edu/index.php/%E5%A4%A9%E5%9C%B0%E5%9C%96%E6%9B%B8%E7%9A%84%E4%B8%89%E5%A4%A7%E7%9A%87%E7%89%8C%E4%BD%9C%E8%80%85%E5%90%84%E6%9C%89%E5%8D%83%E7%A7%8B |title=天地圖書的三大皇牌作者各有千秋 |publisher=[[樹仁大學|樹仁大學]] |date=2007-03-21 |accessdate=2009-10-21}}</ref></small>。

=== 歷史 ===
最初是出一些時事評論方面的著作，部份當代西方思想普及讀物（如[[存在主義|存在主義]]、[[結構主義|結構主義]]、[[佛洛依德|佛洛依德]]等），對介紹新思潮起了一些作用﹔也出版一些翻譯文學作品，如[[松本清張|松本清張]]的[[推理小說|推理小說]]、政壇要人的回憶錄（[[戴高樂|戴高樂]]《[[希望回憶錄|希望回憶錄]]》等）﹔後來有於梨華的留學生文學，有[[亦舒|亦舒]]、[[梁羽生|梁羽生]]的愛情和武俠小說，[[巴金|巴金]]的《[[家_(小说)|家]]》、《[[春_(小说)|春]]》、《[[秋_(小说)|秋]]》（後來又出版了《巴金小說全集》），然後是[[李碧華|李碧華]]和[[蔡瀾|蔡瀾]]加入。
