'''天龍峽站'''（{{jpn|j=天竜峡駅|hg=てんりゅうきょうえき|rm=Tenryūkyō eki|f=y}}）是位於[[日本|日本]][[長野縣|長野縣]][[飯田市|飯田市]]川路天龍峽的[[東海旅客鐵道|東海旅客鐵道]]（JR東海）[[飯田線|飯田線]]鐵路車站。

== 概要 ==
此站座落於[[伊那谷|伊那盆地]]之南、可以看見觀光地「[[天龍峽|天龍峽]]」的位置。1927年飯田線前身的[[伊那電氣鐵道|伊那電氣鐵道]]起點站開業，其後[[三信鐵道|三信鐵道]]進入，是兩公司連繋的重要點。其後國有化而成為中途站，[[國鐵分割民營化|國鐵分割民營化]]後的現在普通列車與本站止的列車相當多。[[特別急行列車|特急]]「[[伊那路_(列車)|伊那路]]」亦全列車停車。

== 車站構造 ==
車站為地上車站，有1面1線的[[單式月台|單式月台]]與1面2線的[[島式月台|島式月台]]以及數條留置線[[側線|側線]]。下行線（單式月台）側為站房。是不配置[[站長|站長]]的站員配置站（[[直營站|直營站]]），[[飯田站|飯田站]]負責本站管理，有設[[綠色窗口|綠色窗口]]。

== 車站周邊 ==
車站週邊為觀光地與有名的溪谷「[[天龍峽|天龍峽]]」（國家指定[[名勝|名勝]]），設有遊歩道和展望台。附近有[[天龍峽溫泉|天龍峽溫泉]]湧出、設有旅館與土產物店等[[溫泉街|溫泉街]]。
* [[天龍川|天龍川]]
* 姑射橋
* 天龍舟下行
* [[三遠南信自動車道|三遠南信自動車道]][[天龍峽IC|天龍峽IC]]

== 歷史 ==
* 1927年（[[昭和|昭和]]2年）12月26日 - [[伊那電氣鐵道|伊那電氣鐵道]]駄科站延伸之際為終着站開業。一般站。
* 1932年（昭和7年）10月30日 - [[三信鐵道|三信鐵道]]門島站開業後進入本站。
* 1943年（昭和18年）8月1日 - 伊那電氣鐵道及三信鐵道[[國有化|國有化]]成為飯田線一部份，成為[[日本國有鐵道|國鐵]]車站。
* 1971年（昭和46年）12月1日 - 停辦貨運。
* 1985年（昭和60年）3月14日 - 停辦行李業務。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]，由東海旅客鐵道繼承。
* 1988年（昭和63年）9月4日 - 車站開業60周年花電車運行
* 1990年（[[平成|平成]]2年）4月 - [[天龍峽溫泉|天龍峽溫泉]]發掘時站房改築。
