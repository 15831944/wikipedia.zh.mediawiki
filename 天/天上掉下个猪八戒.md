《'''天上掉下个猪八戒'''》是2005年上映在[[中国中央电视台综合频道|中国中央电视台综合频道]]的[[中国大陆|中国大陆]]国产动画片，原型是[[明朝|明朝]]小说家[[吴承恩|吴承恩]]《[[西游记|西游记]]》中角色“[[猪八戒|猪八戒]]”。该片由[[江通动画|江通动画]]制作。<ref name="fish ">{{cite web|url=http://www.cccnews.com.cn/2012/1023/13957.shtml|title=《天上掉下个猪八戒》在央视推出|accessdate=2012-10-23|publisher=中国动漫产业网|archiveurl=https://web.archive.org/web/20130413054311/http://www.cccnews.com.cn/2012/1023/13957.shtml|archivedate=2013年4月13日|work=|language=zh-cn|deadurl=yes}}</ref>

== 剧情 ==
在[[王母娘娘|王母娘娘]]的生日宴会上，猪八戒看了月宫仙子[[嫦娥|嫦娥]]的美貌之后，垂涎她的美色，结果被[[玉皇大帝|玉皇大帝]]贬下人间。猪八戒在押送过程中企图逃亡，结果错误地[[投胎|投胎]]，变成一头[[猪|猪]]。[[观音菩萨|观音菩萨]]为帮助猪八戒，给了他灵芝露。[[土地公公|土地公公]]无意间打翻了灵芝露瓶子，猪八戒的容貌从猪变成了猪头人身。猪八戒不堪受辱，飞回天庭向玉皇大帝讨个说法，遭到他昔日副将福迪的奚落。为了报复福迪，猪八戒把天河挖开了，福迪因此也被贬下人间。福迪因此对猪八戒怀恨在心，两人在人间开始斗智斗力，由此衍生了一系列的嬉笑幽默的故事。<ref>{{cite web |url=http://sd.people.com.cn/GB/166203/172859/14323703.html |title=天上掉下个猪八戒 |accessdate= 2011年4月6日|publisher=人民网 |archiveurl= http://sd.people.com.cn/GB/166203/172859/14323703.html|archivedate= 2012年12月18日|work= |language=zh-cn}}</ref>

== 花絮 ==
该片2000年开始策划，2005年首次推出。<ref>{{cite web |url=http://news.163.com/12/0508/03/80V0TC1K00014AED.html |title= 酝酿5年诞生“猪八戒”|accessdate=2012-05-08 |publisher=网易 |archiveurl=http://news.163.com/12/0508/03/80V0TC1K00014AED.html |archivedate=2012年12月18日 |work= |language=zh-cn}}</ref>

该片曾参加[[昂西动画节|昂西动画节]]、[[东京动画节|东京动画节]]、[[渥太华动画节|渥太华动画节]]。

2005年，该片的收视率是4.19%。<ref name="bull " >{{cite web |url=http://politics.gmw.cn/2012-04/08/content_3923074.htm |title= 武汉江通动画创新民族文化制作方式：用动画再现历史经典|accessdate=2012-04-08 |publisher=光明网 |archiveurl= http://politics.gmw.cn/2012-04/08/content_3923074.htm|archivedate=2012年12月18日 |work= |language=zh-cn}}</ref>

该片已经制作到了208集。<ref>{{cite web|url=http://www.cccnews.com.cn/2011/1227/1299.shtml|title=江通动画：从“猪八戒”到“小喇叭”|accessdate=2011-12-27|publisher=中国动漫产业网|archiveurl=https://web.archive.org/web/20130414002838/http://www.cccnews.com.cn/2011/1227/1299.shtml|archivedate=2013年4月14日|work=|language=zh-cn|deadurl=yes}}</ref>
