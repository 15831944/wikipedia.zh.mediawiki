'''天干'''<ref>{{標音|字=干|拼音=gān|注音=ㄍㄢ|粵拼=gon<sup>1</sup>}}</ref>，是中国古代的一种文字计序符号，共10个字: [[甲_(天干)|甲]]、[[乙|乙]]、[[丙|丙]]、[[丁_(天干)|丁]]、[[戊|戊]]、[[己_(天干)|己]]、[[庚|庚]]、[[辛_(天干)|辛]]、[[壬|壬]]、[[癸|癸]]，循环使用。

[[中国|中国]]等[[漢字文化圈|漢字文化圈]]國家古代常以之来命名、排序、纪时。

== 天干的產生 ==
天干最晚在先商已经开始，明显的证据是商族領袖之稱號爲上甲、匸乙、匸丙、匸丁、主壬、主癸。正式将这十个符号命名为天干应该与[[地支|地支]]有关：-{干}-为主干，支为分支，两个相对的命名应当是一起出现的。在[[商朝|商朝]]开始出现天干与地支配合用以纪日，使用[[干支|干支]]纪年要更晚一些。

關於天干何來說法眾多，其中一種是古人根據事物十個狀態排序以定的，視為循環。其中植物的生長就是一例︰按
# 甲就是「鎧甲」，指萬物破殼，植物破土萌芽；
# 乙就是「軋」，指萬物初出地面，呈彎曲狀；
# 丙就是「炳」，指萬物長勢旺盛；
# 丁就是「壯」，指萬物的長勢猶處於[[青春期|青春期]]，成長壯實；
# 戊就是「茂」，指萬物生長最茂盛的時期，是為最頂峰；
# 己就是「起」，指植物開始屈曲其形；
# 庚就是「更」，指植物枯萎衰亡，始入土中，與乙相反；
# 辛就是「新」，指萬物更新；
# 壬就是「姙」，指陽氣正潛地中，萬物被養育；
# 癸就是「揆」，指萬物閉藏於土，萬物萌芽。

现代研究的结论之一是天干源于原始人类以人体部位计数、排序，下面列出《[[说文解字|说文解字]]》中的释义：
# 一曰人头宐为甲，甲象人头。
# 乙承甲，象人颈。
# 丙承乙，象人肩。
# 丁承丙，象人心。
# 戊承丁，象人肋。
# 己承戊，象人腹。
# 庚承己，象人斋。
# 辛承庚，象人股。
# 壬承辛，象人胫。
# 癸承壬，象人足。

== 天干與地支 ==
天干用于纪时超出10即会重复，地支用于纪时超出12也会开始重复，用天-{干}-和地支组合纪时可以减少重复，从而避免混淆。中国古代采用了一种类似[[拉链|拉链]]的[[干支|干支]]组合，一边链齿是循环的天干，一边链齿是循环的地支，最终可以出现两者的[[最小公倍数|最小公倍数]]即60个组合。

天干是[[十进制|十进制]]，容易直观地计算；地支是[[十二进制|十二进制]]，在记录时辰和月份方面具有优势；干支组合是[[六十进制|六十进制]]，适合间隔较密(纪日)和不要混淆(纪年)的情形。中国古代灵活地使用天干、地支之干支组合来表达[[年|年]]、[[月|月]]、[[日|日]]、[[时辰|时辰]]，这样就出现了所谓的[[生辰八字|生辰八字]]。

== 天干與阴阳五行、五方 ==
[[File:Heaven_stem_relation.png|400px]]
天干的十个符号按其自身的含义可以与[[五行|五行]]的五个符号和五方进行匹配。
* 甲、乙为木、东方
* 丙、丁为火、南方
* 戊、己为土、中央
* 庚、辛为金、西方
* 壬、癸为水、北方

天干也有陰陽之分：
* 甲、丙、戊、庚、壬，屬陽。
* 乙、丁、己、辛、癸，屬陰。

== 組合 ==
[[File:Heaven_stem_affect.png|400px]]
天干有五個組合，而這種天干組合之後，會變化成另一種不同的五行，因此會產生不同的較應。除了化合之外天干還有互相沖剋，故此亦會產生不同的效應。

* 天干化合：甲己合化土　乙庚合化金　丙辛合化水　丁壬合化木　戊癸合化火
* 天干相沖：甲庚相沖　乙辛相沖　丙壬相沖　丁癸相沖
* 天干相剋：甲乙木剋戊己土　丙丁火剋庚辛金　戊己土剋壬癸水　庚辛金剋甲乙木　壬癸水剋丙丁火

== 用途 ==
由於天干具有甲、乙、…、癸的固定順序，所以可作為代號使用，類似於第一、第二、…、第十；另一方面，也能作為名目之用，僅視其代稱，不代表其順序。
* 表示順序尺度，代表先後關係： 
**[[农历|农历]]纪年中的干支組合
**用於表示優劣。例如臺灣是[[國民小學|國民小學]]、[[國民中學|國民中學]]、[[高級中學|高級中學]][[學生|學生]]成績的等第，分爲優（不是天干）、甲、乙、丙、丁等。
**[[化学|化学]]中对一些相近[[有机物|有机物]]的中文名称经常使用天干来区别。如[[甲烷|甲烷]]、[[乙烷|乙烷]]、[[乙烯|乙烯]]、[[丙烷|丙烷]]、[[甲基|甲基]]、[[乙醇|乙醇]]等，通常表示的是该[[分子|分子]]或[[基團|基團]]中C（[[碳|碳]]原子）的数量。例如[[癸烷|癸烷]]表示的就是有10个碳原子的烷分子。
** 代表[[流水號|流水號]]，例如臺灣部分學校以天干命名班級，如：甲班、乙班、丙班等。台灣鐵路1950年代以前，座位號碼的車別是以天干作區分，例如：甲車24號。
* 表示名目尺度，僅作為代稱而不具先後關係
** 人的代稱，例如「路人甲」、「某甲」，詳見[[無名氏|無名氏]]
** 用于[[中华人民共和国|中华人民共和国]][[军车牌照|军车牌照]]（2004年廢止）
** 用於行政區劃，例如[[臺灣|臺灣]][[台灣日治時期|日治時期]]的[[臺南市_(堡里)|臺南市]]
** 西方数学刚传入中国时，中国的数学家用天干地支代表字母，十天干分别代表a-j这10个字母，大写字母用代表小写字母的汉字加口字旁表示。
