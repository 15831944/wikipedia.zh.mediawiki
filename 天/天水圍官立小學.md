'''天水圍官立小學'''（{{lang|en|'''Tin Shui Wai Government Primary School'''}}），[[香港|香港]][[元朗區|元朗區]][[天水圍新市鎮|天水圍新市鎮]]的一所男女共合[[官立學校|官立學校]]，位於[[天瑞路|天瑞路]]8號，毗鄰[[天頌苑|天頌苑]]和[[翠湖居|翠湖居]]。

== 班級 ==
目前該校小學一年級至小學六年級，每級開設5班，合共30班。<ref name="概覽">小學概覽2011/12</ref>

== 科目 ==
該校開設9個科目：[[中文|中文]]、[[英文|英文]]、[[數學|數學]]、[[香港小學通識科課程|通識]]、[[視覺藝術|視覺藝術]]、[[音樂|音樂]]、[[體育|體育]]、[[電腦|電腦]]及[[普通話|普通話]]。各科亦會利用[[資訊科技|資訊科技]]來輔助教學。<ref>{{Cite web |url=http://www.tswgps.edu.hk/s/subjectintro/intro.html |title=學校官方網站課程簡介 |access-date=2012-08-15 |archive-url=https://web.archive.org/web/20121201035132/http://www.tswgps.edu.hk/s/subjectintro/intro.html |archive-date=2012-12-01 |dead-url=yes }}</ref>2012年9月開始推行「愛國教育」。<ref>[http://www.epochtimes.com/b5/12/8/29/n3670151.htm%E3%80%8C%E5%BE%9E%E5%A8%83%E5%A8%83%E6%8A%93%E8%B5%B7%E3%80%8D%E4%B8%AD%E5%85%B1%E9%A6%99%E6%B8%AF%E6%B4%97%E8%85%A6-%E5%BC%95%E4%BA%BA%E9%97%9C%E6%B3%A8 「從娃娃抓起」中共香港洗腦 引人關注]《大紀元》2012年8月29日</ref>

== 升中學位分配 ==
該校與三所同屬元朗區的[[官立中學|官立中學]]為聯繫中學，分別為[[趙聿修紀念中學|趙聿修紀念中學]]、[[元朗公立中學|元朗公立中學]]及[[天水圍官立中學|天水圍官立中學]]。<ref>{{cite web|title=中學學位分配|url=http://www.tswgps.edu.hk/schoolpage/secondary/secondary.html|publisher=天水圍官立小學|accessdate=2015-11-02|deadurl=yes|archiveurl=https://web.archive.org/web/20150512032554/http://www.tswgps.edu.hk/schoolpage/secondary/secondary.html|archivedate=2015-05-12}}</ref>
