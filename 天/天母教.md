== 天母地區的開發 ==
教團的中心是位在[[台北|台北]][[士林街|士林街]][[三角埔|三角埔]]的'''天母神社'''（現中山北路七段191巷22號），御神體是湄州媽祖像。

布教的同時，也在本部所在地三角埔（之後的[[天母|天母]]）採掘[[溫泉|溫泉]]、經營[[旅館|旅館]]、[[巴士|巴士]]，並且計畫以[[教會|教會]]為中心開發高級住宅地等開發事業。

天母教在終戰之後，教祖中治稔郎歸國後'''事實上消滅'''。資産全部被[[中國國民黨|中國國民黨]]接收，轉用為黨政高官的住宅等。之後，因為良好的居住環境成為美軍關係者、大使館等外國人的聚居地，最後發展成為天母教當初構想的高級住宅地。

== 略歷 ==
*[[大正|大正]]14年（1925年），在台北市[[永樂町|永樂町]]二丁目五十二番地（現[[迪化街|迪化街]]一段附近）創設教團
*大正15年（1926年），移轉至台北市[[元園町|元園町]]二五三番地（現成都路110號），建設教會、神殿、拜殿、付屬集會所和教主住宅。
*[[昭和|昭和]]2年（1927年），為求台灣人和日本人的融合，設立台灣最初的結婚介紹所「御柱會」<ref>[http://www.books.com.tw/books/series/series9789861371382-2.php　『台灣幸福百事：你想不到的第一次』] {{webarchive|url=https://web.archive.org/web/20110405002945/http://www.books.com.tw/books/series/series9789861371382-2.php |date=2011-04-05 }}陳柔縉、究竟、2011/03/29</ref>。
*昭和5年（1930年），教主中治稔郎為布教和募集資金開始全台灣行脚。
*昭和6年（1931年），獲得[[台北州|台北州]]士林街三角埔湧出的溫泉權利。將本部設在當地，和[[重田榮治|重田榮治]]推進開發當地的共同事業。（開發的全用地約9萬坪）
*昭和8年（1933年），着手引湯工事，引自[[紗帽山|紗帽山]]。
*昭和10年（1935年），引湯工事終了。付屬公衆浴場、神苑、假神殿竣工，移轉。「[[天母溫泉|天母溫泉]]」開始營業，往來溫泉和[[士林車站|士林車站]]間的巴士開始運行。
*昭和20年（1945年），終戰。教主引揚回日本，教團消滅。御神體媽祖像由天母東・西路交差點付近的「[[天母三玉宮|三玉宮]]」保管<ref name="repo1"/>。

== 教祖 ==
[[明治|明治]]11年（1878年）1月13日，教祖中治生於[[兵庫縣|兵庫縣]]。明治28年（1895年），成為兵庫縣[[朝來郡|朝來郡]]和田山（現[[朝來市|朝來市]]）竹田尋常高等小学校尋常科准訓導。明治35年（1902年）左右渡台。明治39年（1906年），開始在[[台灣總督府|台灣總督府]]民政部通信局勤務。大正10年（1921年），成為[[台南郵便局|台南郵便局]]庶務課長。[[台北郵便局|台北郵便局]]異動後，大正14年（1925年），依願退職。對[[扶桑教|扶桑教]]傾倒，和宗教思想家[[中西牛郎|中西牛郎]]相識，為發展扶桑教的教義創設天母教<ref name=repo1>[http://www.koryu.or.jp/ez3_contents.nsf/15aef977a6d6761f49256de4002084ae/1cc81b163edf6053492579eb001d2211/$FILE/03-05.pdf　台北の歴史を歩く　天母の歴史を探る] {{webarchive|url=https://web.archive.org/web/20130609162249/http://www.koryu.or.jp/ez3_contents.nsf/15aef977a6d6761f49256de4002084ae/1cc81b163edf6053492579eb001d2211/$FILE/03-05.pdf |date=2013-06-09 }}片倉佳史、台湾情報誌『交流』2013.3 No.864</ref>。天母教的營運受到[[台灣電力|台灣電力]]前社長[[高木友枝|高木友枝]]、台北第一高額納稅的綿布商重田榮治<ref>[http://kindai.ndl.go.jp/info:ndljp/pid/907877/30　『大日本人物名鑑』〔巻４の２〕] {{webarchive|url=https://web.archive.org/web/20141225095925/http://kindai.ndl.go.jp/info:ndljp/pid/907877/30 |date=2014-12-25 }} ルーブル社出版部 編 (ルーブル社出版部, 1922)</ref>等人的支援。
