《'''天山雪'''》（{{lang-en|'''Snow of Tianshan Mountain'''}}），导演和编剧都是[[张辉|张辉]]，主演杨圣文、胡晓光、杨萌、王劲松。<ref>{{cite web |url=http://ent.sina.com.cn/m/2008-09-06/22192158984.shtml |title=金鸡百花电影节新片展：《天山雪》 |accessdate=2009 |publisher=新浪网 |work= |language=zh-hans}}</ref>

== 剧情 ==
1976年，这年[[毛泽东|毛泽东]]去世了，毛泽东发起的[[文化大革命|文化大革命]]也即将被终止，[[华国锋|华国锋]]、[[叶剑英|叶剑英]]一举粉碎了[[四人帮|四人帮]]，社会终于安定下来，在遥远的边疆[[新疆维吾尔自治区|新疆维吾尔自治区]]，马洪光、孔小雪、马天山、陈毓秀等人的命运却纠缠在了一起。<ref>{{cite web |url=http://ent.sina.com.cn/m/2007-11-10/21261786581.shtml |title= 资料：影片《天山雪》故事梗概|accessdate=2007 |publisher=新浪网 |work= |language=zh-hans}}</ref>
