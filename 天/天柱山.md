'''天柱山'''是位于[[安徽省|安徽省]][[安庆市|安庆市]][[潜山|潜山]]的山脉，因主峰如柱倚天而得名。天柱山又名皖山、皖公山，古称[[衡山_(消歧义)|霍山]]、衡山，有“江淮第一山”、“古南岳”之称。天柱山为[[古皖|古皖]]文化荟萃地。

天柱山座落于[[大别山|大别山]]南麓与[[长江|长江]]中下游平原交汇处，主要山体距[[安徽省|安徽省]][[安庆市|安庆市]][[潜山|潜山]]县城西北9公里。风景区总面积333（主景区82.46）平方公里，是首批国家级重点风景名胜区、[[国家森林公园|国家森林公园]]。全山有45峰、17岭、18崖、14岩、25洞、86怪石、18瀑、17泉、7关寨、2寺观、有植物119科、1000种以上。

== 传说 ==
[[七仙女|七仙女]]紫儿，传说是“玉皇大帝”的第七个女儿，她最心灵手巧，且心地十分善良。一天，七仙女得知“丹阳”境内（今湖北孝感）有一年轻人姓董名永，因家贫无钱安葬死去的父亲，只得卖身为奴。七姐深受感动，私下天庭与董永结百年之好，在大槐树下成就姻缘。可惜作媒人的大槐树一时高兴，把"百年好合"说成"百天好合"，害得七姐与董永只有百日缘分。七姐用巧手金梭织出了10匹锦绢赎出了董永，准备修整家园恩爱白头。不料“玉皇大帝”得知十分震怒，急召七姐回宫，七姐不得不忍痛飞去。她的织布校化成了飞梭石，织机留在了人间，人们在夜深人静时仍能听到"咔喳、咔喳"的织机声。每年正月十五的晚上，身着盛装的姑娘聚集在飞梭石旁，手捧针线盒向七姐乞巧，唱乞巧歌。唱完乞巧歌后，还要询问年成好坏、吉凶如何、能否婚嫁等事宜。

七仙女名水珠，外祖为“盘古”，母为“妈祖”,曾降世寻其夫玄武大帝（董永）为世所传。

如今，安徽天柱山被誉为七仙女在人间的故乡。

== 文化 ==
[[File:Statue_of_Lord_Wan_(20170116133453).jpg|thumb]]
据说，称万岁山是因[[汉武帝|汉武帝]][[元封|元封]]五年([[西元|西元]][[前106年|前106年]])南巡时，亲临皖山设台祭岳，敕封皖山为“南岳”，被誉为“[[江南|江南]]第一山”。又因[[春秋|春秋]]时为[[皖国|皖国]]封地，山名皖山，水名皖水；旧称山南为皖南，山北为皖北；[[安徽省|安徽省]]简称“皖”即源于此，所以又称为安徽的“源头山”。在祭岳时，人群高呼万岁，所以将此山称为万岁山。后[[隋文帝|隋文帝]]诏废，[[隋唐|隋唐]]以后，南岳改为衡山。

唐代大诗人[[李白|李白]]曾写了一首赞美天柱山风景奇秀的诗“江上望皖公山”（皖公山即天柱山），诗云：“奇峰出奇云，秀木含秀气。清晏皖公山，巉绝称人意......”

== 宗教 ==
前106年，[[汉武帝|汉武帝]]南巡，登临天柱，封为“南岳”，由此直至[[隋朝|隋]][[开皇|开皇]]九年（589年），[[隋文帝|隋文帝]]又将[[衡山|衡山]]封为南岳，但天柱山仍历代均有加封，备受世人仰慕，但[[北京|北京]][[地坛|地坛]][[皇祇室|皇祇室]]的「天柱山之神」的神位，指的是瀋陽的天柱山，是為清朝「五陵山神」，而非此安徽的天柱山。

早在唐宋时期，佛、道两教视此为“洞天福地”，争相建观造刹，传道布经。当年佛教禅宗三祖僧璨在此以禅之妙义悟世，安贫乐道，传钵立化的遗迹仍依晰可见。而今[[三祖寺|三祖寺]]殿宇林立，香烟缭绕，已成为全国重点寺庙。

天柱山与[[佛教|佛教]]、[[道教|道教]]都有关联，比如道家把全中国的奇山归纳为36洞天、72福地。天柱山就是第 14洞天、57福地.佛教禅宗也与天柱山颇有渊源，禅宗[[达摩祖师|达摩祖师]]后的二祖、三祖都曾在天柱山修行，[[南北朝|南北朝]]的[[梁朝|梁朝]]时，天柱山山谷寺显赫一时，僧众甚多，俨为四海名刹. 到了唐、宋时期，天柱山依然盛名不减，[[李白|李白]]、[[白居易|白居易]]、[[苏轼|苏轼]]、[[王安石|王安石]]、[[黄庭坚|黄庭坚]]等文人墨客都登临此山，遗留下近400方摩崖碑刻，既可怀古，又是学习、欣赏书法的绝佳范本。
