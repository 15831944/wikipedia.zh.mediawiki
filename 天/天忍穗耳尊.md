'''天忍穗耳尊'''（{{jpn|j = 天忍穂耳尊|hg = あまのおしおみみ の みこと|rm = Amanootsipomimi no mikoto|kyu = 天忍穗耳尊|hgt = あまのおしほみみ の みこと}}）是[[日本|日本]]傳説中的[[治天|治天]]（[[高天原|天]]的統治者），也是[[地神五代|地神五代]]之一，也是[[日本皇室|日本皇室]]的祖。天忍穗耳尊是《[[日本書紀|日本書紀]]》中的稱呼，全名為'''正哉吾勝勝速日天忍穗耳尊'''，《[[先代舊事本紀|先代舊事本紀]]》稱之為'''天押穗耳尊'''、'''天忍穗別尊'''，全名為'''正哉吾勝勝速日天押穗耳尊'''、'''正哉吾勝勝速日天忍穗別尊'''，《[[古事記|古事記]]》即稱之為天之忍穗耳命，全名正勝吾勝勝速日天之忍穗耳命。又名'''正勝尊'''。

== 事績 ==
天照大神曾派遣正哉吾勝勝速日天忍穗耳尊前往統治[[葦原中國|葦原中國]]，但正哉吾勝勝速日天忍穗耳尊在[[高天原|高天原]]遙望葦原中國時，發現該地正處於大動亂時期，因此並未前往赴任，返回天照大神處報告此事。後來天照大神派遣[[武甕槌神|武甕槌神]]前往[[葦原中國平定|平定]]，迫使[[大己貴神|大己貴神]]讓國。於是天津彥彥火瓊瓊杵尊奉命降臨該國，成為新的統治者（即[[天孫降臨|天孫降臨]]）。

=== 父母 ===
* （父：[[素戔嗚尊|素戔嗚尊]]（進雄尊、素盞嗚尊、素戔烏尊） - [[伊奘諾尊|伊奘諾尊]]之子。天照大神之同母弟。[[大己貴神|大己貴神]]之父。{{link-ja|三輪氏}}等祖。）
* 母：[[天照大神|天照大神]] - [[太陽神|日神]]。[[伊奘諾尊|伊奘諾尊]]之女兒。

正哉吾勝勝速日天忍穗耳尊，是天照大神和素戔嗚尊的第一子也。

根據《日本書紀》記載，天照大神與素戔嗚尊達成[[誓約|誓約]]之後，前者將[[勾玉|勾玉]]送給後者做為信物。素戔嗚尊將勾玉洗淨、打碎並吹氣，生下了正哉吾勝勝速日天忍穗耳尊。
