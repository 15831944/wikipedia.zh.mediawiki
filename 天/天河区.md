'''天河区'''是[[中国|中国]][[广东省|广东省]][[广州市|广州市]]的一个[[市辖区|市辖区]]，位于广州老城区东部，1985年5月24日设立，是广州近年来新城区发展中心。广州三大商圈之一的“天河商圈”就在此，是广州的商业中心之一。区政府驻[[天园街道|天园街道]]天府路1号。

== 名称由来 ==
根據《元大德南海志》，北宋時區內已經有大水墟，是當時廣州[[衛星城市|衛星城镇]]。

天河区名来源于1984年兴建的[[天河体育中心|天河体育中心]]，而天河体育中心名称来源于1930年建在这里的[[广州天河机场|广州天河机场]]，天河机场名称来源于天河村。天河村，原名大水圳，在1926年由于军阀[[李福林|李福林]]將所在的大塘村改名為天池村，而大塘村的李氏與天河村的李氏同宗，經李福林的提議，改名为天河村。

== 行政区划及历史沿革 ==
* 1985年5月24日，天河区经国务院批准成立，从广州市郊区分出，成为广州市辖行政区。新建区时，有两镇四街：沙河区（镇）（含同和乡、京溪乡）、东圃区（镇），以及沙河街、五山街、员村街、车陂街。当时面积102.5平方公里，人口20.04万人。此后辖区经过多次变化。1987年9月，同和镇（由同和乡和京溪乡组成）划归白云区管辖。1992年4月，杨箕村从东山区划归天河区。1994年2月，原市农工商联合总公司凤凰农工商公司属下的柯木塱、渔沙坦两行政村划归天河区。1994年8月，沙东村二队（自然村）从东山区划归天河区。2000年9月，原属广州市新塘农工商公司的新塘、沐陂、凌塘、玉树4个行政村移交给天河区。2005年6月7日，登峰街和杨箕村划归越秀，新塘街的玉树村划归萝岗区。随着天河区城市化的发展，乡村逐渐由行政村改设行政街，2002年12月，沙河镇和东圃镇撤销。
* 天河区行政区域总面积约137.38平方公里，辖有21个行政街：沙河街（1950年7月成立，以行文批准为准，下同）、五山街（1950年成立）、员村街（1960年7月成立）、车陂街（1981年5月成立）、石牌街（1987年3月成立）、天河南街（1992年10月9日成立）、林和街（1995年8月11日成立）、沙东街（1995年8月11日成立）、兴华街（1995年8月11日成立）、棠下街（1997年11月18日成立）、天园街（1999年9月30日成立）、冼村街（1999年12月29日成立）、猎德街（1999年12月29日成立）、元岗街（1999年12月29日成立）、黄村街（1999年12月29日成立）、龙洞街（2002年12月成立）、长兴街（2002年12月成立）、凤凰街（2002年12月成立）、前进街（2002年12月成立）、珠吉街（2002年12月成立）、新塘街（2002年12月成立）。

== 经济 ==
[[File:Guangzhou_dusk_11-5-2008.png|thumb]]
2015年天河区GDP已达3432.79亿元，增长8.8%。为[[广州市|广州市]]第一，[[广东省|广东省]]第二（仅次于[[深圳|深圳]][[南山区_(深圳市)|南山区]]）。<ref>{{cite web | url=http://kb.southcn.com/content/2016-03/12/content_143954473.htm | title=2015年粤GDP总量十强区出炉　广州三区上榜：天河黄埔越秀| publisher=南方日报 | date=2016年3月12日 | language = zh-cn}} </ref>
“[[天河路_(广州)|天河路]]黄金商业大道”汇聚了[[天河城|天河城]]、[[正佳广场|正佳广场]]、[[廣州太古滙|太古汇]]、[[万菱汇|万菱汇]]、[[时尚天河广场|时尚天河广场]]、[[天环广场|天环广场]]、[[维多利广场|维多利广场]]、[[天娱广场|天娱广场]]、[[摩登百货|摩登百货]]、[[广州友谊商店|广州友谊商店]]，[[广州购书中心|广州购书中心]]，[[百脑汇|百脑汇]]，[[广州电脑城|广州电脑城]]等一批高、中端卖场，电脑通讯数码产品市场，餐饮娱乐场所等，沿线街道2011年社会消费品零售总额达351.02亿元，占全区比重超过1/3，已达35.4%。<ref>{{cite web | url=http://gzdaily.dayoo.com/html/2012-02/14/content_1610582.htm | title=天河GDP领跑各区县 | publisher=广州日报 | date=2012年2月14日 | language=zh-cn | deadurl=yes | archiveurl=https://web.archive.org/web/20120218050849/http://gzdaily.dayoo.com/html/2012-02/14/content_1610582.htm | archivedate=2012年2月18日 }} </ref>

=== 企业 ===

2014年市认定的328家总部企业中，在天河区注册的有96家，占全市的29.3%，如包括“在天河区办公、但注册地不在天河”的28家，天河区总部企业有124家，占全市的37.8%。在124家总部企业中，世界500强企业2家，占全市的100%;2014中国500强企业7家，占全市的70%;2014中国民营企业500强3家，占全市的33.3%。<ref>{{cite web | url=http://finance.sina.com.cn/leadership/mroll/20150206/221021497340.shtml | title=广州市天河区发展总部经济的启示| publisher=广东经济 | date=2015年2月6日 | language = zh-cn}} </ref>
