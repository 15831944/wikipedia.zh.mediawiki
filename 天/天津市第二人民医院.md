'''天津市第二人民医院'''，曾名为'''天津市传染病医院'''，始建于1920年，坐落于[[中華人民共和國|中華人民共和國]][[天津市|天津市]][[南开区|南开区]]苏堤路75号津河西侧，是一所以“传染病学”方向为特色的[[三级甲等医院|三级甲等专科医院]]，同时也是全国最大的三级甲等传染病专科医院之一。<ref>{{cite web| url=http://www.tjcrbyy.com/yyjs/yyjs.asp| title=医院介绍| publisher=天津市传染病医院官网| accessdate=2013-01-26| deadurl=yes| archiveurl=https://web.archive.org/web/20130514062202/http://www.tjcrbyy.com/yyjs/yyjs.asp| archivedate=2013-05-14}}</ref>2013年1月正式更名“天津市第二人民医院”，医院性质为非盈利性事业单位。在SARS、甲型H1N1流感期间，医院受权设有可接待发热病人和疑似病例的发热门诊。<ref name="更名为天津市第二人民医院">{{cite web | url=http://news.enorth.com.cn/system/2013/01/26/010571268.shtml | title=传染病医院更名第二人民医院 着重解决五大疾病 | publisher=天津北方网 | accessdate=2013-01-26}}</ref>

== 历史沿革 ==
天津市第二人民医院始建于1920年，最初隶属于[[北洋防疫处|北洋防疫处]]。1928年，北洋防疫处被划归合并至当时的天津市卫生局，医院也随之更名为“'''天津平民医院'''”。1945年，抗战结束，日本战败。天津平民医院与'''[[天津日本居留民团|天津日本居留民团]]疗养院'''合并，更名“'''天津市传染病医院'''”，此一名称沿用至2012年4月。<ref>{{cite web| url=http://www.tjwsj.gov.cn/news/481a3715-9f63-4c47-9442-d6491905242b| title=天津市传染病医院| publisher=天津市卫生局综合信息网| accessdate=2013-01-26| deadurl=yes| archiveurl=https://web.archive.org/web/20120415154404/http://www.tjwsj.gov.cn/news/481a3715-9f63-4c47-9442-d6491905242b| archivedate=2012-04-15}}</ref>天津市卫生局于2012年4月批准天津市传染病医院更名为“天津市第二人民医院”。2012年，天津市传染病医院原址扩建计划对外公布，曾引起争议。2013年1月25日，天津市传染病医院举行更名仪式，正式更名“天津市第二人民医院”，同时将由“传染病学”方向为特色的[[三级甲等医院|三级甲等专科医院]]转型为综合性三级甲等医院。<ref>{{cite web | url=http://news.163.com/13/0126/09/8M4T8U4M00014JB6.html | title=天津市传染病医院更名为天津市第二人民医院 | publisher=网易 | accessdate=2013-01-26}}</ref><ref name="更名为天津市第二人民医院" />

== 主治方向 ==
天津市第二人民医院自1945年以来一直名为天津市传染病医院，承担天津市的传染病治疗和研究，是天津市传染病学中心。同时肝脏疾病治疗是该院的特色，天津市肝病研究所亦挂靠在该院。2013年，正式更名后的天津市第二人民医院，以重症肝衰的、肝硬化并发症、肝癌、慢性乙丙肝的抗病毒治疗和以儿童传染病等五个方向为主要治疗和研究方向。<ref name="更名为天津市第二人民医院" />

== 原址扩建争议 ==
由于天津市第二人民医院（天津市传染病医院）历史悠久，最初的原址曾为城区边缘，目前这一地带已经发展成为城市的核心区域。同时因为就诊患者多为传染性疾病，虽无传染周边的案例发生，但仍因而引起周边居民的担忧。市民曾多次建议该医院搬离天津市区，但这一建议未获得政府方面接受。天津市政府鉴于[[天津市卫生局|天津市卫生局]]的4次安全性论证和上海、北京等传染病医院迁址新建失败的经验，坚持原址扩建的计划。<ref>{{cite web | url=http://news.163.com/12/1101/11/8F7KN7TK00014JB6.html | title=人民网网友建言天津市传染病医院搬离市区等19条留言获市委书记张高丽回应 | publisher=网易 | work=人民网 | accessdate=2013-01-26}}</ref>{{fact|但有相关医院负责人传出传染病医院要与第一中心医院合并，传染病医院和其他与一中心医院合并的医院迁至复康路一中心原址，预计时间2017年。}}
