'''天宫二号'''是[[中华人民共和国|中国]]的一个[[太空实验室|太空实验室]]，也是中国[[921工程|921-2]]空间站计划的一部分。<ref>{{cite news|title=官方介绍航天未来计划：2015年前后发天宫二号|url=http://www.chinanews.com/gn/2013/06-26/4971491.shtml|accessdate=2016-09-16|agency=[[中国新闻社|中国新闻社]]|publisher=[[中国新闻网|中国新闻网]]|date=2013-06-26|ref=中新网|location=北京|language=zh-hans}}</ref>空间实验室由[[中国空间技术研究院|中国太空技术研究院]]研制<ref name=mission>{{Cite web |url=http://www.cmse.gov.cn/art/2016/9/15/art_1724_31041.html |title =天宫二号肩负重任 应用项目丰富 |publisher=中国载人航天工程网|date=2008年9月17日|accessdate=2013年6月23日 |language=zh-cn}}</ref>，设计可同时承载三名[[宇航员|航天员]]在轨工作，用于取代2011年发射的[[天宫一号|天宫一号]]。

2016年9月15日晚22时04分09秒，天宫二号由[[长征二号F运载火箭|长征二号F改进型运载火箭]]（T2）从[[酒泉卫星发射中心|酒泉卫星发射中心]]发射升空<ref>{{Cite web |url = http://www.cmse.gov.cn/art/2016/9/15/art_1724_31041.html|title = 中国航天科技集团|publisher=中国航天科技集团 |date=2014年10月17日 |accessdate=2014年10月17日|language=zh-cn}}</ref><ref>{{Cite web |url = http://www.spacechina.com/n25/n144/n206/n214/c1046313/content.html|title = “长二F”完成首轮测试 明年将发射“天宫二号”|publisher=中国航天报 |date=2015年10月14日 |accessdate=2016年1月2日|language=zh-cn}}</ref>，成为中国首个以勘驗中長期（約三十天以上）滯留为目标的空间站。天宫二号有前后两个交会对接口，可以与[[神舟载人飞船|神舟载人飞船]]以及类似天宫一号目标飞行器的[[中国货运飞船|天舟货运飞船]]同时对接。

== 历史 ==

1992年，中國政府決定實施載人航天工程，並確定了中國載人航天工程「三步走」的發展计划。第一步是發射載人飛船，[[神舟五號|神舟五號]]和[[神舟六號|神舟六號]]成功飛行，標誌第一步的完成。第二步是完善航天員出艙活動技術、空間飛行器的交會對接技術，發射空間實驗室，解決有一定規模的、短期有人照料的空間應用問題。[[神舟七號|神舟七號]]任務標誌着中國掌握了出艙活動關鍵技術；[[天宮一號|天宮一號]]與[[神舟八號|神舟八號]]和[[神舟九號|神舟九號]]交會對接，標誌着中國掌握了自動和手動交會對接技術；[[神舟十號|神舟十號]]飛行任務是工程第二步第一階段任務的收官之戰。第二階段為空間實驗室階段，包括發射[[天宮二號|天宮二號]]、[[神舟十一號|神舟十一號]]、[[天舟一號|天舟一號]]。第三步是建造空間站，解決有較大規模的、長期有人照料的空間應用問題。

原有消息称，天宫一号成功地完成了「三步走」規劃的第二步的第一阶段，工程人员决定提高天宫二号的质量，约十三吨，即原计划2017至2018年发射的[[天宫三号|天宫三号]]的质量级别。但最後因為天宮一號飞行寿命延长，完成任務超越預估，決定採用消耗燃料较少的方案，即计划使用已完成研制的[[天宫一号|天宫一号]]目标飞行器的备份器，对其进行所要求的改进后，由[[长征二号F运载火箭|长征二号F改进型运载火箭]]（T2）或长征七号运载火箭从[[酒泉卫星发射中心|酒泉卫星发射中心]]发射升空，即类似天宫一号目标飞行器的发射情况。

== 历程 ==
* 2014年10月，天宫二号空间实验室开始进行总装工作。<ref name="中国航天科技集团公司2015-01-03" />

* 2015年1月3日，天宫二号空间实验室完成空间应用系统载荷设备安装并交付电测。<ref name="中国航天科技集团公司2015-01-03">{{cite web|url=http://www.spacechina.com/n25/n144/n206/n214/c824873/content.html|title=天宫二号空间实验室进入电测阶段|accessdate=2016-10-30|date=2015-01-03|language=zh-hans|agency=[[中国航天报|中国航天报]]|location=北京}}</ref>

* 2016年3月25日，用于发射天宫二号的[[长征二号F运载火箭|长征二号F T2运载火箭]]完成总装工作，并进入测试阶段。<ref>{{cite web|url=http://www.cmse.gov.cn/art/2016/3/25/art_18_27541.html|title=发射神舟十一号的长二F火箭开始测试|accessdate=2016-10-30|date=2016-03-25|publisher=[[中国载人航天|中国载人航天网站]]|language=zh-hans|agency=[[中国航天科技集团公司|中国航天科技集团公司网站]]|location=北京}}</ref>

* 2016年4月15日，天宫二号空间实验室完成大型试验后的总装并交付电测。<ref>{{cite web|url=http://www.spacechina.com/n25/n144/n206/n214/c1241346/content.html|title=天宫二号空间实验室完成总装并交付电测|accessdate=2016-10-30|date=2016-04-15|language=zh-hans|agency=[[中国航天报|中国航天报]]|location=北京}}</ref>

* 2016年6月3日，[[长征二号F运载火箭|长征二号F T2运载火箭]]通过出场测试质量验收，火箭测试工作全部完成。<ref>{{cite web|url=http://www.cmse.gov.cn/art/2016/6/3/art_18_27945.html|title=用于发射神舟十一号飞船的长二F火箭完成出厂测试|accessdate=2016-10-30|date=2016-06-03|language=zh-hans|agency=[[中国航天报|中国航天报]]|publisher=[[中国载人航天|中国载人航天网站]]|location=北京}}</ref>

* 2016年7月7日，天宫二号空间实验室测试完成后开始从北京运往[[酒泉卫星发射中心|酒泉卫星发射中心]]。<ref name="新浪网2016-07-10">{{cite news|url=http://news.sina.com.cn/o/2016-07-10/doc-ifxtwchx8401065.shtml|title=天宫二号空间实验室运抵发射场 计划9月中旬发射|date=2016-07-10|publisher=[[新浪网|新浪网]]|accessdate=2016-10-30|location=北京|agency=[[中国航天科技集团公司|中国航天科技集团公司网站]]|language=zh-hans}}</ref>

* 2016年7月9日，天宫二号空间实验室安全运达[[酒泉卫星发射中心|酒泉卫星发射中心]]，并开展发射场区的总装和测试工作。<ref name="新浪网2016-07-10" />

* 2016年8月6日，[[长征二号F运载火箭|长征二号F T2运载火箭]]安全运达[[酒泉卫星发射中心|酒泉卫星发射中心]]，并开展发射场区的总装和测试工作。<ref>{{cite web|url=http://www.cmse.gov.cn/art/2016/8/8/art_18_30806.html|title=发射天宫二号和神舟十一号的两枚长征二号火箭安全运抵发射场|accessdate=2016-10-30|date=2016-08-08|language=zh-hans|agency=[[中国载人航天工程办公室|中国载人航天工程]]publisher=[[中国载人航天|中国载人航天网站]]|location=北京}}</ref>

* 2016年9月9日，天宫二号空间实验室与[[长征二号F运载火箭|长征二号F T2运载火箭]]组合体垂直转运到发射区调试，计划在9月15日至20日择机发射。<ref>{{cite news|url=http://news.ifeng.com/a/20160909/49943973_0.shtml?_zbs_baidu_bk|title=箭在弦上！天宫二号已矗立在发射架|date=2016-09-09|publisher=[[凤凰网|凤凰网]]|accessdate=2016-10-30|location=北京|agency=[[中国航天科技集团公司|中国航天科技集团公司网站]]|language=zh-hans}}</ref>

* 2016年9月15日22时04分，天宫二号空间实验室使用[[长征二号F运载火箭|长征二号F T2运载火箭]]在酒泉卫星发射中心成功发射升空，大约575后，天宫二号与助推火箭分离并成功进入预定轨道。<ref>{{cite web|url=http://www.spacechina.com/n25/n144/n206/n214/c1440015/content.html|title=天宫二号空间实验室中秋夜发射成功|accessdate=2016-10-30|date=2016-09-15|language=zh-hans|agency=[[中国航天科技集团公司|中国航天科技集团公司]]|location=北京}}</ref>

* 2016年10月19日，03时24分，[[神舟十一号|神舟十一号]]与天宫二号对接环接触；03时31分，[[神舟十一号|神舟十一号]]与天宫二号成功完成自动交会对接；06时32分，两位航天员[[景海鹏|景海鹏]]和[[陈冬_(1978年)|陈冬]]先后进入天宫二号空间实验室。<ref>{{cite web|url=http://www.spacechina.com/n25/n144/n206/n214/c1471939/content.html|title=神舟十一号与天宫二号成功实现自动交会对接|accessdate=2016-10-30|date=2016-10-19|language=zh-hans|agency=[[中国航天报|中国航天报]]|location=北京}}</ref>

* 2016年10月25日，天宫二号伴随卫星飞越组合体上空并传回天宫二号空间实验室与[[神舟十一号|神舟十一号]]飞船组合体的全貌。<ref>{{cite web|url=http://news.xinhuanet.com/tech/2016-10/27/c_1119795368.htm|title=天宫二号伴星传回组合体图像|accessdate=2016-10-30|date=2016-10-27|publisher=[[新华网|新华网]]|language=zh-hans|location=北京}}</ref>

==设计==
天宫二号担负的任务有[[地球观测|地球观测]]、[[空间地球系统科学|太空地球系统科学]]、[[空间应用新技术|太空应用新技术]]、[[航天技术|太空技術]]、[[航天医学|航天医学]]以及其他领域的应用和试验<ref name=mission/>。

设计飛行壽命大于两年。天宮二號總設計師朱樅鵬透露，尽管上級下達的任務是要求飛行兩年，但實際上會飛更長的時間。之前中國天宮一號設計壽命也是兩年，但是卻服役長達四年多時間。
