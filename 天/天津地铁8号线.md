'''天津地铁八号线'''，是[[天津轨道交通|天津轨道交通]][[天津地铁|天津地铁]]规划中的线路之一。在2000年天津市地铁规划中，天津地铁八号线规划北起[[北辰区|北辰区]]小淀至[[西青区|西青区]]中北镇，全长20公里，设站31座。沿线主要途经南仓道、密云路、芥园西道、西横堤外居住区、铁东居住区、北仓工业区、铁东工业区等地区，与当时的地铁七号线（线位大致相当于目前的[[天津地铁10号线|天津地铁10号线]]）组成城市外环线。<ref>[http://news.enorth.com.cn/system/2009/02/19/003900172_08.shtml 天津地铁八号线简介]</ref>但在2016年环评招标公示中显示，天津地铁8号线一期工程已经修改为西起南开区绿水园资阳路站，南至津南区咸水沽站的中心城区南部骨干线路。天津地铁8号线一期工程正线线路全长33千米，共设27座车站，均为地下站，在南开大学津南校区南侧地块设车辆段1座，新建主变电站2座，控制中心接入华苑车辆段线网控制中心。

2018年12月26日，[[天津轨道交通集团|天津轨道交通集团]]在其官网上公示了[[天津地铁6号线|天津地铁6号线]]工程（梅林路站-咸水沽西站）项目招标文件<ref>{{cite web |title=天津地铁6号线工程（梅林路站-咸水沽西站）标段 施工招标 招标文件 |url=http://211.81.63.130/cache/9/03/www.tjgdjt.com/7248584b90ee667b1611ce852ba034ef/b083feb921001d8d8a8c01.pdf |website=天津轨道交通集团 |accessdate=2018-12-30}}</ref>，线路自渌水道站到咸水沽西站区间远期将与8号线贯通运营<ref>{{cite web |title=天津地铁6号线工程（梅林路站-咸水沽西站）项目招标文件公示 |url=http://www.tjgdjt.com/xinwen/content_20008.htm |website=天津轨道交通集团 |accessdate=2018-12-30}}</ref>。2019年1月7日天津轨道交通集团公开了天津地铁6号线工程（梅林路站-咸水沽西站）环境影响报告书征求意见稿<ref>{{cite web |title=天津地铁6号线工程（梅林路站-咸水沽西站）环境影响报告书征求意见稿信息公开 |url=http://www.tjgdjt.com/xinwen/content_20202.htm |website=天津轨道交通集团 |date=2019-01-07|accessdate=2019-01-13}}</ref><ref>{{cite web |title=天津地铁6号线工程（梅林路站-咸水沽西站）环境影响报告书（征求意见稿）|url=http://www.tjgdjt.com/touzi/img/site2/20190107/1803734483021d9d754f03.pdf |website=天津轨道交通集团 |date=2019-01-07 |accessdate=2019-01-13}}</ref>。

2019年2月4日除夕当日，天津地铁6号线工程（梅林路站－咸水沽西站）正式开工。

== 概要 ==
渌水道站到咸水沽西站段线路正线全长13.42公里，共设9座地下车站，线路南端设双港车辆段，车辆段位于[[海河教育园区|海河教育园区]]内，工程于2019年2月4日启动建设。
