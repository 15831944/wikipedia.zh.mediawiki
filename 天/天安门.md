'''天安门'''坐落在[[中國|中國]][[北京|北京]]市中心，[[故宫|故宫]]的南端，与[[天安门广场|天安门广场]]隔[[长安街|长安街]]相望，是[[明|明]][[清|清]]两代[[北京皇城|北京皇城]]的正门。設計者為明代的御用建築匠師[[蒯祥|蒯祥]]。天安門始建于[[明|明朝]]永樂十五年（1417年），時稱為承天門。[[清|清朝]]順治八年（1651年）更名為天安門。1925年10月10日，[[國立故宮博物院|國立故宮博物院]]成立，天安門始對民眾開放。1949年10月1日，在这里举行了[[中華人民共和國|中華人民共和國]]的[[开国大典|开国大典]]，它由此被设计入[[中華人民共和國國徽|国徽]]，並成了中華人民共和國的象徵之一。1961年，天安门被[[中华人民共和国国务院|中华人民共和国国务院]]公佈为第一批[[全国重点文物保护单位|全国重点文物保护单位]]之一<ref>[http://www.china-culture.com.cn/ww/sc/1.htm 第一批全国重点文物保护单位名单（180处）]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}（1961年3月4日国务院公布）</ref>。

== 历史 ==
=== 明清时期 ===
天安门原为[[明朝|明]][[清朝|清]][[皇城|皇城]]的正门，始建于明[[永乐_(明成祖)|永乐]]十五年（1417年）。 最初时，它仅是一座三层五间式的木结构牌楼，名字叫做“承天门”。[[天顺_(明英宗)|天顺]]元年（1457年），牌楼毁于雷火，八年后的[[成化|成化]]元年（1465年）重建为面阔九间、进深五间的重檐歇山式城楼。{{fact|[[崇祯|崇祯]]十七年（1644年），[[李自成|李自成]]攻入北京，仅逗留40日便被迫撤出，撤退前草草登基，当晚“焚毁宫殿及九门城楼”，承天门再次被毁}}。清[[顺治|顺治]]八年（1651年）在废墟上进行了大规模改建，重修为一座城楼，名字也改成“天安门”<ref>{{cite news|url=http://www.people.com.cn/GB/historic/1031/3639.html|title=1651年10月31日 清改承天门为天安门|newspaper=《[[人民日报|人民日报]]》|publisher=[[人民网|人民网]]|date=2003-08-01}}</ref>

明清的皇帝们一般都在天安门颁布重要诏令，称为“[[金凤颁诏|金凤颁诏]]”。此外皇帝大婚、将领出征时祭旗、御驾亲征时祭路、[[刑部|刑部]]在秋天提审要犯（“秋审”）、[[殿试|殿试]]公布“三甲”（“[[金殿传胪|金殿传胪]]”）等重大仪式也都在此举行。

===整修與重建===
1949年后，天安门进行过多次整修，其中1952年和1970年的两次整修规模较庞大。近年有报道披露1969年12月15日至1970年3月7日的所谓「整修」其实是秘密将整个天安门拆除重建。<ref>{{cite news|url=http://news.sina.com.cn/c/sd/2013-09-27/131228317056.shtml|title=天安门城楼秘闻：1969年按原尺寸秘密翻建|newspaper=《[[解放日报|解放日报]]》|publisher=[[新浪网|新浪网]]|date=2013-09-27}}</ref>1968年3月24日天安門城樓安全檢查小組提交《關於天安門城樓木結構構架安全檢查報告》，1968年3月初北京市革委會工交城建組召集市建委、規劃局、建築設計院等其中14人組成檢查小組詳細檢查，1968年5月開始翻修籌備。1969年10月下旬搭好腳手架木，11月下旬舊城樓拆完，1970年1月中旬做完大木安裝，3月中旬扣上瓦頂及關連設施、油漆彩畫，3月底以前落架清理。建設者口風極嚴，沒有絲毫外露消息<ref>[http://www.yhcqw.com/30/8857.html 天安門城樓秘密翻建始末]，陳徒手，《[[炎黃春秋|炎黃春秋]]》2012年第8期</ref>。本次重建的原因在於此前的整修未能完全解决建筑结构变形的问题以及1966年的[[邢台地震|邢台地震]]对建筑的影响。<ref>[http://www.chinanews.com/gj/kong/news/2007/10-23/1056991.shtml 《中华文摘》文章：秘密重建天安门始末]《北京紀事》2005年4期</ref>

1988年11月中旬起，对社会公众收费开放（初期每張門券30元人民幣，外國游客和港澳人士付同值代用券；現為每人15元，學生票5元）。城樓東西兩邊八大觀禮台全部開放的話，共能容納五、六千名遊客。

== 城楼 ==
天安门坐北朝南，原通高33.7米，1970年重修后改为34.7米。建筑的底部为[[汉白玉|汉白玉]]精雕的[[须弥座|须弥座]]，高1.59米。底座上建有朱红色的高大城台，面积约4800平米。城台的两侧有登城马道，台上建城楼。

城楼面阔九间（57.14米），进深五间（20.97米），重檐[[歇山式|歇山式]]顶，覆蓋黄[[琉璃瓦|琉璃瓦]]。在两层正檐之间有用滑轮固定的一枚[[中华人民共和国国徽|中华人民共和国国徽]]，直径1.7米，位置与[[北京中轴线|北京中轴线]]严密重合。屋檐下悬挂着17盏大红宫灯，其中主灯重达450公斤，辅灯也各重350公斤。整座屋顶由60根大红木柱支撑，每根柱的直径约2米。城楼的正背面各排列有36扇菱花门窗，天花、[[斗拱|斗拱]]和梁枋上都绘制了最高等级的“金龙和玺”彩画。楼体的四周为回廊环绕，廊深3米左右，廊外设汉白玉栏杆和矮墙。整座天安门城楼的外观富丽堂皇，雄伟壮观，氣勢磅礡，非常莊嚴肅穆。

在天安门城台的下部开有城门五阙。正中的门洞高大宽敞，专供皇帝出入。城门上原挂“天安门”牌匾，国民政府期间（抗戰勝利到1949年）曾挂上[[蔣中正|蔣中正]]画像，1949年以后改挂巨大的[[毛泽东|毛泽东]]画像，但兩者的擺放位置不同，蔣中正畫像是放置於城樓上。毛澤東的画像每年中华人民共和国国庆节前都要更换，历史上在1950年、1953年和1969年进行过三次改版（1950年版由張振仕所繪）。目前使用的是1969年第四版的作品（繪者為葛小光），绘制在一整块玻璃钢板上，外框高6.4米、宽5米，内框高5.8米、宽4.4米。

画像的两侧镶嵌有两幅巨型标语，是1949年开国大典时，由两个[[日本人|日本人]]小野澤亙和森茂设计的。1949年西、東兩側標語内容原各為“中华人民共和国万岁”和“中央人民政府万岁”。1950年国庆时经[[胡乔木|胡乔木]]建议，把天安门东侧的標語换为“世界人民大团结万岁”，西侧“中华人民共和国万岁”未动。标语各长30米、高2.2米，原以[[繁体字|繁体字]]刻在木板上，1964年换成[[简化字|简化字]]，1970年后改用了金属材料制作。

[[File:天安門史達林肖像.jpg|235px]]
值得一提的是，於1953年3月5日蘇聯領導人[[史達林|史達林]]去世，9日北京市政府為他舉辦萬人追悼會，將天安門毛澤東肖像更換為史達林肖像一天，並且搭配「斯大林同志永垂不朽」標語及肖像兩側各五面旗幟，是天安門首次懸掛中國以外人士肖像。

== 周围 ==
[[File:Tiananmenpic2.jpg|thumb]]天安门城台前有[[金水河|金水河]]，河上架五座[[汉白玉|汉白玉]][[石桥|石桥]]与五个门洞一一对应，称为“金水桥”。中间的桥略宽，又叫御路桥，两侧的栏杆望柱上雕刻着蟠龙图案。另外的四座桥则叫做“王公桥”，栏杆[[望柱|望柱]]上雕刻着[[荷花|荷花]]图案。在桥边还有两对[[石狮子|石狮子]]，是明永乐朝的遗物。东面雄狮的右爪戏弄绣球，西面雌狮的左爪抚弄小狮，分别象征着皇权的威严和永久。

天安门城楼的前后各有一对[[华表|华表]]，通体用汉白玉雕刻而成，每根高10多米，重万余斤。华表上刻有云纹和盘龙图案，在靠近顶部的地方横插着一块云板，称作“日月板”。华表的最顶端有承露盘，盘上蹲立着“[[龙生九子|龙生九子]]”之一的石[[犼|犼]]，也叫做“朝天吼”或“望天吼”。其中[[城楼|城楼]]内侧的石犼面向北方，正对着[[紫禁城|紫禁城]]，称为“望君出”，外侧的石犼面向[[南方|南方]]，称为“望君归”。

== 歷史事件 ==
[[天安門廣場|天安門廣場]]上歷來發生了許多重大的、對[[中國歷史|中國歷史]]發展之沿革有著相當影響力的歷史事件。
* [[宣統|宣統]]三年[[腊月廿五|腊月廿五]]（公元1912年2月12日），[[隆裕太后|隆裕太后]]在這裡頒布末代[[中國皇帝|皇帝]][[愛新覺羅溥儀|愛新覺羅溥儀]]的[[宣統帝退位詔書|退位詔書]]，從此天安門不再歸[[皇室|皇室]]所有。
* [[中華民國|中華民國]]大陸時期的1919年「[[五四運動|五四運動]]」、1925年「[[五卅運動|五卅運動]]」、1926年「[[三一八惨案|三一八惨案]]」中“反对八国最后通牒的国民大会”、1935年「[[一二九運動|一二九運動]]」、1948年5月中下旬之「[[反饑餓、反內戰、反迫害運動|反饑餓、反內戰、反迫害運動]]」等先後在這裡發生。
* 1949年10月1日，[[天安門廣場|天安門廣場]]上舉行[[中華人民共和國|中華人民共和國]][[中華人民共和國建國儀式|開國大典]]，[[毛澤東|毛澤東]]及其他[[中國共產黨|中國共產黨]][[中國共產黨領導人|領導人]]與[[民主黨派|民主黨派]]人士在[[城樓|城樓]]上宣佈[[中華人民共和國|中華人民共和國]][[中華人民共和國中央人民政府|中央人民政府]]成立。從那時起，天安門便成為中華人民共和國的象徵，後來更把它的[[圖案設計|圖案設計]]在[[中華人民共和國國徽|中華人民共和國國徽]]之正中。[[毛澤東|毛澤東]]及後來的歷屆中共領導人亦以該處上舉行[[閱兵|閱兵]]、[[遊行|遊行]][[儀式|儀式]]。[[文化大革命|文化大革命]]中，[[毛澤東|毛澤東]]曾經在[[天安門城樓|天安門城樓]]八次接見過從[[中國|中國]]各地來的[[紅衛兵|紅衛兵]]。
* 1976年「[[四五運動|四五運動]]」、1989年「[[六四事件|六四事件]]」和2001年「[[天安门自焚事件|天安门自焚事件]]」都發生在天安門前的[[廣場|廣場]]上。
* 2013年10月28日12时许，三个人驾驶一辆吉普车车闯入[[长安街|长安街]]便道，撞到多名游客后，撞向天安门城楼毛泽东像对出的金水桥护栏上。车辆撞击后，发生大火，詳見[[金水橋事件|金水橋事件]]。
