《'''天開了'''》是香港歌手[[劉德華|劉德華]]於2001年發行的國語專輯<ref>[http://www.kuwo.cn/album/526/ 天开了]</ref><ref>{{cite web |url=http://ent.china.com/zh_cn/music/popular/newcd/166944/20010619/193572.html |language= |title=刘德华《天开了》 |publisher=中华网 |date=2001-06-19 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20131203034625/http://ent.china.com/zh_cn/music/popular/newcd/166944/20010619/193572.html |archivedate=2013-12-03 }}</ref>。

== 簡介 ==
专辑名称之所以取名《天开了》，按刘德华的解释：他的演艺生涯一路走来他都和“天”字都沾上边，他开的电影公司叫“[[天幕制作有限公司|天幕]]”“天中”，别人叫他“[[天王|天王]]”，2001年赴[[黄山|黄山]]拍[[道地|道地]][[绿茶|绿茶]][[广告|广告]]时，他觉得“不上黄山，不知山之美，上了黄山，才知天地之大、之美”，心胸也变开阔，也因为这样，他才取名《天开了》。<ref>{{cite web |url=http://ent.china.com/zh_cn/music/popular/popnews/166937/20010525/183025.html |language= |title=刘德华将推出最新国语专辑《天开了》 |publisher=听听音乐网 |date=2001-05-25 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20150610205833/http://ent.china.com/zh_cn/music/popular/popnews/166937/20010525/183025.html |archivedate=2015年6月10日 |df= }}</ref>
