'''天猫'''（{{lang-en|'''Tmall'''}}，亦称'''淘寶商城'''、'''天猫商城'''），是[[中國|中國]]最大的[[企業對消費者|企業對消費者]]購物網站，由[[淘宝网|淘宝网]]分离而成，由[[浙江天猫网络有限公司|浙江天猫网络有限公司]]运营<ref>{{cite news|title=天猫备案信息查询|url=http://tool.chinaz.com/beian.aspx|newspaper=网站备案信息查询|date=2015年8月17日}}</ref>。多为知名品牌的直营旗舰店和授权专卖店组成。浙江天猫网络有限公司現為[[阿里巴巴集團|阿里巴巴集團]]的子公司之一。天猫同时支持淘宝的各项服务，例如[[支付宝|支付宝]]及[[集分宝|集分宝]]等等。

== 历史 ==
天猫的前身淘寶商城創立於2008年4月，最初是[[淘寶網|淘寶網]]旗下的企業對消費者頻道。隨著淘寶商城日趨成熟，愈來愈多中外知名品牌在其平台上開設官方旗艦店，2010年11月，淘寶商城採用獨立域名www.tmall.com，<ref>{{Cite web|url=http://tech.sina.com.cn/i/2010-11-08/10234839980.shtml|title=淘宝商城启用独立域名：为分拆上市序曲？_互联网_科技时代_新浪网|accessdate=2018-05-27|work=tech.sina.com.cn}}</ref>淘寶商城於2011年6月从淘宝网分拆独立。以加強品牌認知度並更好地將其企業對消費者商家與淘寶網的小型[[C2C|C2C]]商家區分。目前，淘寶商城展示逾30,000個本地和全球品牌的產品，並且有多個為不同行業而設的垂直商城，包括電器城、名鞋館及家裝館等，日後會進一步加強分行業垂直市場的發展。2012年1月11日，淘寶商城更改名稱為'''天貓'''。

2014年2月19日，阿里巴巴宣布「'''天貓國際'''」正式上線，為國內用戶直供海外原裝進口商品。

===世界纪录===
2015年11月12日，吉尼斯世界纪录认证官Charles Wharton宣布，由天猫主办的“[[光棍节|双11]]全球狂欢节”创造了包括手机销量在内的9项[[吉尼斯世界纪录|吉尼斯世界纪录]]<ref>{{cite news|title=天猫“双11”打破9项吉尼斯世界纪录 |url=http://tech.sina.com.cn/i/2015-11-14/doc-ifxkrwks3902113.shtml |publisher=京华时报 |date=2015年11月14日 |accessdate=2015年11月14日 }}</ref>。

== 争议 ==
===数据造假===
天猫于2014年10月1日开始[[Smartisan_T1|Smartisan T1]]的预约申请，之后有人发现预约数总能给3整除，之后天猫发表声明承认数据造假，因为由于意外导致数据清零，为了尽快恢复和逼近实际预约数，前端程序人员私自将显示的预约数乘3，并表示预约数已恢复。<ref>{{cite web|title=微博 文章 - 关于锤子预约数据意外的说明|url=http://weibo.com/p/1001603764861036317855}}</ref><ref>{{cite web|title=天猫回应锤子手机预订数造假 程序员将预约数乘3-观察者网|url=http://www.guancha.cn/Science/2014_10_13_275541.shtml?ZXW}}</ref><ref>{{cite web |title=天猫承认锤子手机预约数造假 被人工设定乘以"3"_ 联盟中国 _ 中国网 |url=http://union.china.com.cn/cmdt/txt/2014-10/13/content_7289605.htm |deadurl=yes |archiveurl=https://web.archive.org/web/20141017085121/http://union.china.com.cn/cmdt/txt/2014-10/13/content_7289605.htm |archivedate=2014年10月17日 |df= }}</ref>但之后仍有人发现其他预约商品有类似代码，怀疑是长久以来的做法并形成模块化处理。<ref>{{cite web|title=锤子天猫预定，不管什么时候预定数都能被 3 整除，这是什么高科技？ - 知乎|url=http://www.zhihu.com/question/25951778}}</ref>

===貨品真假===
2014年9月[[中國青年報|中國青年報]]的調查發現，天猫、天貓國際的賣家商品是否為仿冒，本質上天貓提不出證明與鑑驗報告，尤其是外國品牌產品，就算有真實直郵紀錄也沒有意義，因為商家可以將貨品真正寄往該國，再寄回來。消費者只能選擇官方直營旗艦攤位比較有保障，其他私營攤位就難說，這是各國網路電商的通病，然而天貓因為一直主打全站都是100%正品所以被格外關注。<ref name=254l>[http://zqb.cyol.com/html/2014-11/06/nw.D110000zgqnb_20141106_2-12.htm 中青-拿什么证明天猫国际产品的真假]</ref>

深入業內調查發現，天貓上的確假冒貨較少但不能說沒有，然而有許多非官方直營的授權代理商攤位走一種擦邊球模式，將次級品大量銷售於天貓，這些商品是原廠出產但是有生產瑕疵，以格外便宜的價錢販售，無法指控為仿冒品且有些經驗豐富的消費者，專門喜歡買這類商品，<ref name=254l />容忍一些缺點但能用便宜價買到正品，然而不懂其中門道的新手消費者可能在買到後產生糾紛。
