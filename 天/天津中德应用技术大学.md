'''天津中德应用技术大学'''，前身是'''天津中德职业技术学院'''，位于[[海河教育园区|天津海河教育园区]]，是天津市属[[应用技术类型高等学校|应用技术类型高等学校]]，由天津市工业和信息化委员会管理，被教育部评为国家示范性高职院校。2015年11月，教育部正式批复在天津中德职业技术学院的基础上整合[[天津图书馆|天津市海河教育园图书馆]]的资源，建立天津中德应用技术大学<ref name="教育部批复">教育部关于同意建立天津中德应用技术大学的函（教发函[2015]135号）</ref>，成为中国第一所本科层次的应用技术教育大学<ref>{{cite news|title=天津中德应用技术大学成立|url=http://news.xinhuanet.com/local/2015-12/04/c_128497756.htm|accessdate=2015-12-04|agency=天津日报|date=2015-12-04}}</ref>。

== 概况 ==
根据《教育部关于同意建立天津中德应用技术大学的函》，天津中德应用技术大学系本科层次的普通高校，由天津市领导和管理，发展所需经费由天津市安排解决。学校办学定位于[[应用技术类型高等学校|应用技术类型高等学校]]，既实施本科层次应用技术教育，又实施专科层次高等职业教育，全日制在校生规模暂定为12000人。<ref name="教育部批复" />

== 历史沿革 ==
1985年，由中德两国政府合作建立'''天津职工工业技术学院'''，是天津市属全日制高等院校<ref name="高等学校图书情报工作指导委员会2008">{{cite book|author=高等学校图书情报工作指导委员会|title=中国高等学校图书馆大全|url=http://books.google.com/books?id=700zAQAAIAAJ|year=2008|publisher=印刷工業出版社|language=zh-cn|page=122|quote=天津中德 职业技术学院图书馆天津中德职业技术学院(天津中德培训中心)是一所天津市属全日制高等院校,是 1985 年经天津市人民政府批准,教育部备案的独立设置...}}</ref>。1986年，中日两国政府合作在天津建立'''天津市职工现代企业管理学院'''。2001年，在天津职工工业技术学院和天津市职工现代企业管理学院合并组建全日制普通高等院校。2006年，天津中德职业技术学院与空客天津总装公司联合建立“空客A320蓝领技能评估中心和培训中心”<ref>{{cite news|title=蓝领人才培养基地为空客A320总装线输送人才|url=http://www.tj.xinhuanet.com/bhxq/2009-01/05/content_15367359.htm|accessdate=2015-12-07|agency=新华网|deadurl=yes|archiveurl=https://web.archive.org/web/20160304105417/http://www.tj.xinhuanet.com/bhxq/2009-01/05/content_15367359.htm|archivedate=2016-03-04}}</ref>。2009年12月，天津市决定将天津中德现代工业技术培训中心和天津企管培训中心整合为'''天津中德职业技术学院'''。2011年春季，学院迁入[[海河教育园区|天津海河教育园区]]<ref>{{cite news|title=海河教育园区今起全面搬迁|url=http://news.163.com/11/0210/14/6SHN7UFU00014AED.html|accessdate=2015-12-07|agency=每日新报|date=2011-02-10}}</ref>。

2013年6月28日，应用技术大学（学院）联盟在天津成立，天津中德职业技术技术学院成为联盟的创始会员<ref>{{cite news|title=应用技术大学（学院）联盟、地方高校转型发展研究中心在津成立|url=http://education.news.cn/2013-06/28/c_116333110.htm|accessdate=2015-12-07|agency=新华网|deadurl=yes|archiveurl=https://web.archive.org/web/20160305154452/http://education.news.cn/2013-06/28/c_116333110.htm|archivedate=2016-03-05}}</ref>。2014年，[[天津市人民政府|天津市人民政府]]向国家教育部申请：在天津中德职业技术学院的基础上，设立应用技术大学，拟定名'''天津海河应用技术大学'''<ref>《天津市人民政府关于申请设立天津海河应用技术大学的函》（津政函〔2014〕109号）</ref>。2015年11月，教育部批复同意建立天津中德应用技术大学<ref name="教育部批复" />。

== 专业设置 ==
天津中的应用技术大学首批设置本科专业3个，即飞行器制造工程、机械电子工程、自动化，并沿袭天津中德职业技术学院的专科层次的专业设置。<ref name="教育部批复" />
