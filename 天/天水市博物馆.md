'''天水市博物馆'''，位于[[甘肃省|甘肃省]][[天水市|天水市]][[秦州区|秦州区]]伏羲路110号[[伏羲庙|伏羲庙]]，是天水市属综合性博物馆。

==简介==
1958年成立'''天水地区博物馆'''，1962年撤销，1979年11月复建。1985年9月，[[天水地区|天水地区]]与天水市合并，天水地区博物馆更名为'''天水市博物馆'''。1986年正式开放，隶属天水市文化局<ref name=lishu>{{Cite web|url =http://dili.gscn.com.cn/system/2012/02/07/010001461.shtml|title =天水市博物馆|author =|date =2012-02-07|publisher =中国甘肃网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20130513010601/http://dili.gscn.com.cn/system/2012/02/07/010001461.shtml|archivedate =2013-05-13}}</ref>。1986年，从[[天水城隍庙|天水城隍庙]]迁至天水[[伏羲庙|伏羲庙]]。 2004年5月，全国重点文物保护单位[[胡氏古民居建筑|胡氏古民居建筑]]南宅子交由天水市博物馆管理。2014年，成立天水市文物考古研究所<ref name=jinji>{{Cite web |url =  http://www.tianshui.gov.cn/News/get/tsxw/2017/124/17124181951JCD98BBFFCKI49H8BKI8.htm|title =天水市博物馆成功晋级国家一级博物馆  |author =  |date = 2017-01-24 |publisher = 天水市人民政府 |language =  |accessdate =  |quote =  }}</ref>。2017年，天水市博物馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref>。

天水市博物馆负责天水市文物征集、收藏、陈列、研究以及[[全国重点文物保护单位|全国重点文物保护单位]][[伏羲庙|伏羲庙]]、[[胡氏古民居建筑|胡氏古民居建筑]]的保护管理工作<ref name=lishu/>。天水市博物馆馆藏文物31909件，分为石器、陶器、瓷器、铜器、铁器、书画古籍、民俗文物、杂项、现代书画、古钱币等类别。藏品主要是天水出土文物，辅以天水周边地区出土文物<ref name=jinji/>。
