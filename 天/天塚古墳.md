'''天塚古墳'''（{{jpn|j=白米山古墳|hg=あまづかこふん/あまつかこふん|rm=Amadukakofun}}）是位於日本[[京都市|京都市]][[右京區|右京區]]{{Link-ja|太秦|太秦}}的一座{{Link-ja|前方後圓墳|前方後円墳}}，日本國史跡。

== 概要 ==
古墳位於[[京都盆地|京都盆地]]西部，[[嵯峨野|嵯峨野]]台地南邊{{Sfn|天塚古墳（平凡社）|1981年}}。雖然尚未對古墳展開過考古調查{{Sfn|京都府埋蔵文化財情報 第65号|1997年|pp=41-42}}，但是在1887年出土了大量[[陪葬品|陪葬品]]{{Sfn|天塚古墳（平凡社）|1981年}}。1900年，出土文物交由京都帝室博物館保管（現[[京都國立博物館|京都國立博物館]]），並且由京都帝國大學（現[[京都大學|京都大學]]）校長[[濱田耕作|濱田耕作]]展開調查{{Sfn|京都府埋蔵文化財情報 第65号|1997年|pp=41-42}}。[[大正|大正]]年間（1912年至1926年），[[梅原末治|梅原末治]]提出了有關古墳的報告{{Sfn|京都府埋蔵文化財情報 第65号|1997年|pp=41-42}}。1966年，京都大學考古學研究會提出了天塚古墳和清水山古墳的概要報告{{Sfn|京都府埋蔵文化財情報 第65号|1997年|pp=41-42}}。1973年，位於古墳東北面的清水山古墳消失{{Sfn|天塚古墳（平凡社）|1981年}}。1978年3月3日，古墳獲指定日本國史跡<ref name="国指定"/>。

古墳是前方後圓墳，前方部分朝南{{Sfn|天塚古墳（平凡社）|1981年}}{{Sfn|京都府埋蔵文化財情報 第65号|1997年|pp=41-42}}。墳丘有兩層{{Sfn|天塚古墳（平凡社）|1981年}}{{Sfn|天塚古墳（古墳）|1989年}}{{Sfn|京都府埋蔵文化財情報 第65号|1997年|pp=41-42}}，表面發現了{{Link-ja|圓筒埴輪|円筒埴輪}}{{Sfn|天塚古墳（古墳）|1989年}}，周圍由周堀所包圍{{Sfn|天塚古墳（平凡社）|1981年}}{{Sfn|天塚古墳（古墳）|1989年}}。主體部分的墓室有兩座{{Link-ja|橫穴式石室|横穴式石室}}，均位於後圓部分{{Sfn|京都府埋蔵文化財情報 第65号|1997年|pp=41-42}}。古墳西北面建有一座被認為是[[陪葬墓|陪葬墓]]的[[圓墳|圓墳]]{{Sfn|天塚古墳（古墳）|1989年}}。

古墳推測建於[[古墳時代|古墳時代]]後期（約6世紀前半）{{Sfn|天塚古墳（平凡社）|1981年}}{{Sfn|天塚古墳（古墳）|1989年}}{{Sfn|京都府埋蔵文化財情報 第65号|1997年|pp=41-42}}。在嵯峨野，天塚古墳以及墳丘長約75米的日本國史跡{{Link-ja|蛇塚古墳|蛇塚古墳}}、墳丘長約63米的[[仲野親王|仲野親王]]墓[[垂箕山古墳|垂箕山古墳]]和墳丘長約60米已消失的清水山古墳等，均屬於後期古墳，因此推測這些古墳與當時在嵯峨野一帶進行開發的[[渡來人|渡來人]][[秦氏_(日本)|秦氏]]一族有關<ref>{{cite book|url=http://www.kyoto-arc.or.jp/news/leaflet/190.pdf  |language=ja |script-title=ja:太秦の古墳を歩く |publisher=京都府埋藏文化財調查研究中心、{{Link-ja|京都市考古資料館|京都市考古資料館}} |date=2004-09-24 |accessdate=2018-07-17}}</ref>。

== 墓室 ==
主體部分的墓室有兩座橫穴式石室，均位於後圓部分，一座在後圓部分內的西面，另一座則位於中間細部。

;後圓部分西面開口石室
: 無袖式石室，朝西北方向露出。石室規模如下{{Sfn|天塚古墳（古墳）|1989年}}。
:* 全長：約10米
:* 寬：兩米（中央）、1.1米（入口）
:* 高：約兩米（中央）

;中間細部露出石室
: 右片袖式石室，朝西南方向露出。石室規模如下{{Sfn|天塚古墳（古墳）|1989年}}。
:* 全長：7.7米
:* {{Link-ja|玄室|玄室}}
:** 長：4.7米
:** 寬：1.7米（中央）
:** 高：約兩米
:* {{Link-ja|羨道|羨道}}
:** 長：約3米
:** 寬：1.2米
:** 高：1.5米
:兩座石室的側壁同樣由亂石堆積而成，盡頭位址方向均祭有[[稻荷神|稻荷神]]{{Sfn|天塚古墳（古墳）|1989年}}。

== 出土文物 ==
《日本古墳大辭典》指出土文物的出土位置不明，猜測是來自其中一座石室{{Sfn|天塚古墳（古墳）|1989年}}，《京都市的地名》則指出是來自中間細部露出石室{{Sfn|天塚古墳（平凡社）|1981年}}。1887年出土的陪葬品如下{{Sfn|天塚古墳（古墳）|1989年}}：
* 玉類
* 金銅製中空丸玉7個
* 仿製鏡
* 鐵刀3把
* {{Link-ja|鐵鏃|鉄鏃}}
* {{Link-ja|馬具|馬具}}類，鐵地金銅製杏葉、鏡板等等。
* {{Link-ja|須惠器|須恵器}}，器台、寬口壺。
以上出土文物均為[[京都大學綜合博物館|京都大學綜合博物館]]的館藏{{Sfn|京都府埋蔵文化財情報 第65号|1997年|pp=41-42}}。

== 清水山古墳 ==
[[File:Shimizuyama_Kofun,_sekihi.jpg|thumb]]
古墳東北面以前建有'''清水山古墳'''{{Sfn|天塚古墳（平凡社）|1981年}}，屬於前方後圓墳，在1973年被破壞，現在已經不存在{{Sfn|天塚古墳（平凡社）|1981年}}。

按當時的調查，墳丘長約60米{{Sfn|天塚古墳（平凡社）|1981年}}{{Sfn|京都府埋蔵文化財情報 第65号|1997年|pp=41-42}}。主體部分的墓室是橫穴式石室{{Sfn|天塚古墳（平凡社）|1981年}}{{Sfn|京都府埋蔵文化財情報 第65号|1997年|pp=41-42}}。建造時期[[古墳時代|古墳時代]]後期（約5世紀末至6世紀初），為早於天塚古墳建成的首長墓{{Sfn|天塚古墳（平凡社）|1981年}}{{Sfn|京都府埋蔵文化財情報 第65号|1997年|pp=41-42}}。
