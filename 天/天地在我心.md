《'''天地在我心'''》是一首中国歌曲，为1999年上映的动画电影《[[宝莲灯_(动画电影)|宝莲灯]]》的主题曲兼三首插曲之一，由[[郑方|郑方]]、[[刘欢|刘欢]]谱曲，刘欢填词、演唱。该歌曲受到较高评价，曾入选中国原创歌曲排行榜“十大金曲”。

== 创作与发行 ==
《天地在我心》是1998年，[[上海美术电影制片厂|上海美术电影制片厂]]（“上美影”）请[[刘欢|刘欢]]为其新制作的动画片《[[宝莲灯_(动画电影)|宝莲灯]]》填词、演唱的主题曲兼插曲，由[[郑方|郑方]]、刘欢共同谱曲，曾名《天人合一》。《天》的最初版本由上海美术电影制片厂提供，刘欢听了之后觉得不错，“有点儿《[[狮子王|狮子王]]》 那种动画大片的意思”，遂承接下来。<ref name="nanfang">{{cite web|author1=彭苏|title=刘欢：人生就在音乐里|url=http://news.ifeng.com/shendu/nfrwzk/detail_2013_08/12/28532191_0.shtml|publisher=[[南方人物周刊|南方人物周刊]]|accessdate=2018-06-09|archiveurl=https://archive.today/20180609045506/http://news.ifeng.com/shendu/nfrwzk/detail_2013_08/12/28532191_0.shtml|archivedate=2018年6月9日|language=zh-cn|date=2013-08-12|deadurl=no|df=}}</ref><ref name="edushi" /><ref name="sina1999" />该歌曲创作时长达八个月，刘欢在原有编曲的基础上重新写了旋律并填词。<ref name="edushi">{{cite web |title=宝莲灯发布主题曲天地在我心 刘欢演绎得令人回肠荡气-娱乐音乐-E都市 |url=http://hz.edushi.com/bang/info/4-112-n3079315-p0.html |website=hz.edushi.com |publisher=超级明星 |editor=小傻瓜 |accessdate=2018-06-09 |language=zh-cn |date=2016-06-09 |archiveurl=https://archive.today/20180609033103/http://hz.edushi.com/bang/info/4-112-n3079315-p0.html |archivedate=2018年6月9日 |deadurl=no |df= }}</ref>《天》创作完成后，动画制作人员再根据歌曲所表达的意象创作出对应的画面。因为刘欢签约的[[索尼音乐|索尼音乐]]与上美影达成了版权合作，上美影并没有为此歌曲的创作支出任何额外的花费。<ref>{{cite web |title=《宝莲灯》请明星配音唱歌没花钱_影视音乐_生活空间_新浪网 |url=http://eladies.sina.com.cn/movie/news/movie/1999-9-8/9742.shtml |website=eladies.sina.com.cn |publisher=[[凤凰卫视|凤凰卫视]] |accessdate=2018-06-09 |archiveurl=https://archive.today/20180609041502/http://eladies.sina.com.cn/movie/news/movie/1999-9-8/9742.shtml |archivedate=2018年6月9日 |language=zh-cn |date=1999-09-08 |deadurl=no |df= }}</ref>

《天》与《宝莲灯》中的其他三首插曲一道，收录在《〈宝莲灯〉电影原声音乐》中，于《宝莲灯》上映前发售。<ref name="sina1999">{{cite web |title=三歌星为国产动画片《宝莲灯》“点灯” |url=http://news.sina.com.cn/richtalk/news/movie/9902/021119.html |website=[[新浪|新浪]] |publisher= [[新民晚报|新民晚报]] |accessdate=2018-06-09|archiveurl=https://web.archive.org/web/20050410193846/http://news.sina.com.cn/richtalk/news/movie/9902/021119.html|archivedate=2005-04-10|deadurl=no|date=1999-02-11}}</ref>这些歌曲在电影的前期宣传中发挥了重要作用，电影还创下了当年中国动画片[[票房|票房]]收入的最高纪录。<ref name="zgwypl">{{cite web |author1=金桥 |title=20世纪中国动画电影音乐评述：记忆童年记忆深处的旋律 - 音乐评论 - 中国文艺评论网 |url=http://www.zgwypl.com/show-260-33133-1.html |website=www.zgwypl.com |publisher=[[文汇报|文汇报]] |accessdate=2018-06-09 |archiveurl=https://archive.today/20180609044411/http://www.zgwypl.com/show-260-33133-1.html |archivedate=2018年6月9日 |language=zh-cn |date=2017-06-21 |deadurl=no |df= }}</ref>

== 反响 ==
《天》在各大音乐排行榜上收获颇丰<ref name="zgwypl" />，也获得了普遍的正面评价。

=== 荣誉 ===
2000年，《天》入选1999年度中国原创歌曲排行榜“十大金曲”，刘欢本人亦凭借此歌曲和《从头再来》获得此排行榜最高奖项“杰出成就奖”。<ref name="edushi" /><ref name="yuledaily">{{cite web |title=中国新歌声《天地在我心》原唱是谁及歌词_每日娱乐 |url=http://www.yuledaily.com/Video/20170714407.html |website=www.yuledaily.com |publisher=每日娱乐网 |accessdate=2018-06-09 |archiveurl=https://archive.today/20180609032346/http://www.yuledaily.com/Video/20170714407.html |archivedate=2018年6月9日 |language=zh-cn |date=2017-07-14 |deadurl=no |df= }}</ref>2004年，《天》获得了中央电视台首届国产动画“我最喜爱的卡通歌曲”第四名。<ref>{{cite web|author1=宋学春|title=全国“我最喜爱的卡通歌曲”在青评选揭晓--图片频道--人民网|url=http://www.people.com.cn/GB/tupian/1099/2713027.html|website=www.people.com.cn|publisher=人民网|accessdate=2018-06-09|language=zh-cn|archiveurl=https://archive.today/20180609043507/http://www.people.com.cn/GB/tupian/1099/2713027.html|archivedate=2018年6月9日|deadurl=no|df=}}</ref>

=== 评价 ===
参加《[[中国好声音|中国好声音]]》并师从刘欢组的[[王乃恩|王乃恩]]说，《天》“整个架构非常动漫，又通俗易懂”，在加入刘欢的队伍之后第一件事就是找他要了《天》的伴奏、唱了一晚上。<ref name="nanfang" />《超级明星》评价《天》气势恢宏地展示了动画电影主角沉香的成长历练，“是将影片推向高潮的扛鼎之作”。整部作品采用了三段体结构，首段迷惘徘徊、魂牵梦萦，二段将旋律提升一个八度，情绪更为炽烈，末段则转入平和，表达了自信与对未来之向往。《超级明星》认为，与其他通俗歌曲比较，《天》“在[[音域|音域]]和情绪上的提升相当显著”，能做到这样的反差很不容易，[[附点|附点]]和[[切分音|切分]]节奏使得音乐充满活力，演唱者刘欢的嗓音也十分到位。<ref name="edushi" />《两岸大众文化比较》一书评价《天》和刘欢的许多其他作品类似，大气磅礴、鼓励上进。<ref>{{cite book |author1=赵淑梅 |author2=王书洋 |author3=唐淑宏 |author4=张雪英 |title=两岸大众文化比较 |date=2017 |publisher=九州社 崧博出版社 |isbn=9787510826528 |pages=186 |url=https://books.google.com/books?id=ePM_DwAAQBAJ&lpg=PT6&ots=XigK9uOCls&dq=%E4%B8%A4%E5%B2%B8%E5%A4%A7%E4%BC%97%E6%96%87%E5%8C%96%E6%AF%94%E8%BE%83%20%E5%88%98%E6%AC%A2%20%E5%A4%A9%E5%9C%B0%E5%9C%A8%E6%88%91%E5%BF%83&hl=zh-CN&pg=PT6#v=onepage&q=%E4%B8%A4%E5%B2%B8%E5%A4%A7%E4%BC%97%E6%96%87%E5%8C%96%E6%AF%94%E8%BE%83%20%E5%88%98%E6%AC%A2%20%E5%A4%A9%E5%9C%B0%E5%9C%A8%E6%88%91%E5%BF%83&f=false |accessdate=2018-06-09 |language=zh-cn}}</ref>《[[新民晚报|新民晚报]]》评价，《天》表现出了主角历尽劫难、[[劈山救母|劈山救母]]的一片赤诚。<ref name="sina1999" />
