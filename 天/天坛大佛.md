'''天壇大佛'''（{{lang|en|The Big Buddha}}）是一座位於[[香港|香港]][[大嶼山|大嶼山]][[寶蓮寺|寶蓮寺]]前[[木魚峰|木魚峰]]上的[[釋迦牟尼|釋迦牟尼]]佛像，座落於海拔520[[米_(單位)|米]]的[[昂坪|昂坪]]。天壇大佛由[[中国航天航空工业科学技术咨询公司|中国航天航空工业科学技术咨询公司]]設計和制作，佛像坐在268級石階上，由207塊銅片組成（佛身165塊、蓮花36塊、雲頭6塊），高26.4米，連蓮花座及基座總高約34米，重250[[公噸|公噸]]，坐於3層祭壇上，耗資超過6000萬[[港元|港元]]，是世界上第二大的戶外[[青铜|青銅]]座佛，僅次於[[佛陀紀念館#佛光大佛|佛光大佛]]。天壇大佛於1990年動工，1993年12月29日（农历十一月十七日）共用了3年多的時間，佛家阿彌陀誕辰日[[開光|開光]]。現在是[[香港|香港]]著名的旅遊景點之一。

== 命名 ==
大佛之基層分三層次第而上，第一層面積10,000[[平方英尺|平方英尺]]（929[[平方米|平方米]]），第二層8,000平方尺（743.20平方米），第三層6,000平方尺（557.40平方米），其設計參考
[[北京|北京]][[天壇|天壇]]祈年殿天壇的[[地基|地基]]形貌，故名「天壇大佛」。下有一個博物館

== 造型 ==

天壇大佛是依據佛經[[如來|如來]]32相而設計，面相參照[[龍門石窟|龍門石窟]]的盧舍那佛，而衣紋和頭飾則參照[[敦煌石窟|敦煌石窟]]第360窟的[[釋迦牟尼|釋迦牟尼]]佛像，因此大佛兼備[[隋朝|隋]][[唐朝|唐]]佛教全盛時期造像的特色。
*坐向：坐南向北，稍微偏东，朝向全香港。
*坐姿：稱為「結跏趺」，又稱「全跏」、「正跏」。相傳是釋迦牟尼在[[菩提樹|菩提樹]]下修悟正道時的坐姿。
*面相：面如[[滿月|滿月]]、額廣平正、雙耳垂肩，慈顔微笑，頂部肉髻螺，青蓮花眼、眉如初月，兩眉之間，有一白如雪，軟若棉的毫毛，向右旋轉。重5噸高4.3米的佛面，在澆注時加入約2千克[[黃金|黃金]]，更添光彩。
*[[手印|手印]]：右手示「施無畏印」，左手下垂腳上，反掌向外，指端微微向下，稱「予願印」，雙掌中心現[[法輪|法輪]]，代表佛法長存，流轉十方。

== 基座 ==
天壇大佛的地基面積為2,239平方米，内分三層：功德堂、法界堂及紀念堂，遊客可付費入場參觀。三層殿堂由圓型樓梯貫通，中掛一口直徑2米的「瑜伽鐘」，外壁有佛像、手印及捐贈善信芳名，内壁刻有經文，由電腦控制大鐘每隔7分鐘便會自動敲打一次，共敲108次，目的讓眾人「解除108種煩惱」。 

*功德堂：供奉一尊以[[楠木|楠木]]雕成的[[地藏王|地藏王]][[菩薩|菩薩]]，同時展覽[[斯里蘭卡|斯里蘭卡]]法師維巴那沙拿4幅描述[[釋迦牟尼|釋迦牟尼]]佛出生、成道、[[初转法轮|初转法轮]]和[[涅槃|入大涅槃]]的繪畫。
*法界堂：内設[[佛教|佛教]]書畫展。
*紀念堂：供奉[[佛陀|佛陀]]真身[[舍利|舍利]]，於1992年由寶蓮寺法師遠赴斯里蘭卡迎來。同層并刻有曾經捐資興建天壇大佛之善信芳名。
*六天母：大佛對開的6尊舞姿銅像，名爲「六天母獻供」，供品分别是[[花|花]]、香、[[燈|燈]]、塗、果、樂，代表[[六波羅密|六波羅密]]（[[布施|布施]]、[[持戒|持戒]]、[[忍辱|忍辱]]、[[精進|精進]]、[[禅定|禅定]]、[[智慧|智慧]]）。
