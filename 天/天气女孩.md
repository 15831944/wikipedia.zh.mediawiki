'''天氣女孩'''<ref>{{cite web|title=天气女孩 简介|url=http://www.femc.com.tw/artist.php?aid=1|accessdate=2016-09-11}}</ref>（{{lang-en|'''Weather Girls'''}}；{{lang-ja|ウェザーガールズ}}）是2012年在日本出道的臺灣女子偶像團體，於2012年10月17日發行首張日文單曲《{{lang|ja|恋の天気予報}}》正式出道。團名源於同名[[壹電視新聞臺|壹電視]]天氣預報節目《Weather Girls》。粉絲名稱為「彩虹」（{{lang-en|Rainbow}}；{{jpn|j=虹|kn=にじ}}）。2017年11月1日，十全娛樂以商業理由單方面與各成員同時解約，同時解散團體。2018年，成員[[Yumi|Yumi]]、[[Ria|Ria]]、[[Dara|Dara]]、[[Hijon|Hijon]]，重組四人組合形式的[[天氣女孩|天氣女孩]]。

== 歷史 ==

=== 出道前 ===
;播報天氣
Weather Girls成軍前，在[[壹電視新聞臺|壹電視新聞台]]製播的[[天氣預報|天氣預報]]節目《Weather Girls》中輪值擔任播報每日天氣。節目的內容是女孩們每月以不同的主題及裝扮報告氣象，由於以日式「卡哇伊」形象為造型主軸，讓此節目意外地在日本受到歡迎，造成一股臺灣氣象姐姐（原文：{{lang|ja|お天気お姉さん}}）的話題，也在一些日本媒體上被介紹<ref>{{Cite news|url = http://getnews.jp/archives/132885|title = 周年慶! 我們獨家專訪到臺灣的「Weather Girls」氣象女孩以及他們是如何受歡迎的秘密|last = |first = |date = |work = 東京産業新聞社|accessdate = 2011-08-01}}</ref>。節目自2010年8月開拍，到2012年3月停止拍攝新單元為止，總計共38名女孩參與此節目<ref>{{Cite news|url = http://www.j-cast.com/2012/08/13142643.html|title = 韓流の次は「華流」ブームが来る？　台湾のお天気姉さんアイドル日本デビュー|last = |first = |date = |work = J-CASTニュース|accessdate = 2012-08-13}}</ref>。由《Weather Girls》播出畫面可見，參與此節目者都佩戴了當時壹電視紅蘋果標誌胸章。

=== 日本出道===
因《Weather Girls》節目在日本受到歡迎，2012年4月，經紀公司Ritz Production從《Weather Girls》中挑選出七名團員，組成偶像團體，準備於日本出道。團員分別是Hi Jon、Ice、Dara、Mia、Mini、Yumi和NueNue，每位團員各代表一星期中的一天並有象徵各自個性的天氣符號。同年8月9日正式宣布日本出道消息。在日本，Weather Girls屬於波麗佳音體系，經紀約由Ritz Production與L Motion Entertainment共同管理，Jeff Miyahara為歌曲製作人。<ref>{{Cite news|url = http://getnews.jp/archives/241375|title = 「台湾のお天気お姉さん」から日本のアイドルへ　ついにデビューが決定したウェザーガールズに完全密着！|last = |first = |date = 2012-08-10|work = ガジェット通信|accessdate = }}</ref>記者會上，波麗佳音的代表說：{{cquote|我想日本人會喜歡Weather Girls，因為Weather Girls是在台灣創造形成，而台灣又深深受到日本文化的動漫或遊戲的影響。除此之外，一邊跳舞一邊報天氣這個表演對觀眾來說是很新鮮有趣的，所以女孩們有自己的特色。（翻譯自<ref>{{Cite news|url = http://blogs.wsj.com/japanrealtime/2012/08/30/will-the-sun-shine-on-weather-girls-japan-debut/|title = Will the Sun Shine on Weather Girls' Japan Debut?|last = |first = |date = 2012-08-30|work = JAPAN REAL TIME|accessdate = }}</ref>）}}

團員Mini說：「我從小就是個哈日族，我一直很希望有機會在日本生活，我也很希望能成為歌手。能夠完成這些夢想讓我很開心。」而隊長Yumi補充表示：「我們的夢想是參加NHK的紅白歌唱大賽，還希望能參加日本的氣象預報節目。」（翻譯自<ref>{{Cite news|url = http://www.excite.co.jp/News/asia_ent/20121016/Searchina_20121016008.html|title = ウェザーガールズが日本“上陸”、「恋の天気予報」でデビュー|last = |first = |date = 2012-10-16|work = Searchina.ne.jp|accessdate = }}</ref><ref>{{Cite news|url = http://www.tokyohive.com/article/2012/08/weather-girls-from-taiwan-to-make-their-cd-debut-in-japan/|title = "Weather Girls" from Taiwan to make their CD debut in Japan|last = |first = |date = 2012-08-09|work = TOKYOHIVE|accessdate = }}</ref>)

;《{{lang|ja|恋の天気予報}}》發行
《{{lang|ja|恋の天気予報}}》單曲於2012年10月17日發行，單曲內包括兩首歌，{{lang|ja|恋の天気予報}}與{{lang|ja|Loving Pass ～恋のパスワード～}}，並附上了伴奏版，共四條曲目。DVD版中附上了自我介紹及主打歌的MV。YouTube上的〈{{lang|ja|恋の天気予報}}〉MV觀看數到目前已突破了141萬<ref>{{Cite web|url = https://www.youtube.com/watch?v=jd2-Nf86jvA|title = ウェザーガールズ『恋の天気予報』Music Video|accessdate = }}</ref>。

從10月1號開始，Music On! TV的SELECT中開始播出以Weather Girls 為主角的電視實境節目《密著ウェザーガールズ》，內容是紀錄宣傳期時Weather Girls在家中發生的事情以及活動花絮。《密著ウェザーガールズ》共播出了三季。第三季改到YouTube上播出，主題定為料理女王決定戰，比賽結果為料理女王這個頭銜由Mini贏得<ref>{{Cite news|url = http://prtimes.jp/main/html/rd/p/000000115.000002331.html|title = 台湾のお天気アイドル「ウェザーガールズ」に2ヶ月にわたり密着取材！|last = |first = |date = 2012-09-24|work = PRTIMES|accessdate = }}</ref><ref>{{Cite web|url = https://www.facebook.com/WRGirls/posts/552361621491405|title = 官方臉書告知|accessdate = 2013-07-04}}</ref>。

=== 日本發展時期 ===

;日本氣象預報員考試
2013年6月17日宣布，她們將會參加在2014年1月舉行的日本氣象預報員考試。這個考試從1994年開始舉辦，錄取率僅有5%<ref>{{Cite news|url = http://star.ettoday.net/news/206539|title = 台灣天氣女孩甜美走紅東瀛　力拚日本氣象預報員|last = |first = |date = |work = |accessdate = 2013-05-13}}</ref>。由於考試，他們請來TBS的氣象預報員國本未華授課，這些課程稱為『絶対合格！気象予報士への道～』，粉絲可以在場<ref>{{Cite news|url = http://en.rocketnews24.com/2013/05/16/taiwanese-weather-reporting-idol-group-to-take-japans-grueling-national-weather-forecasting-license-exam/|title = Taiwanese Weather Reporting Idol Group to Take Japan’s Grueling National Weather Forecasting License Exam|last = |first = |date = |work = rocket news 24|accessdate = 2013-05-16}}</ref><ref>{{Cite news|url = http://www.ettoday.net/news/20130513/206539.htm|title = 台灣天氣女孩甜美走紅東瀛　力拚日本氣象預報員|last = |first = |date = 2013-05-13|work = Etoday|accessdate = }}</ref>。

同年的2月初和6月初，知名製作人Jeff Miyahara為Weather Girls製作了第二張單曲《{{lang|ja|恋はトキメキ注意報}}》和第三張單曲《{{lang|ja|恋のラブ♥サンシャイン}}》。同年的11月又為Weather Girls發行了第四張單曲《HEY BOY～ウェイシェンモ?～》。

;《WEATHER GIRLS》發行
《[[WEATHER_GIRLS|WEATHER GIRLS]]》專輯於2013年7月3號發行，專輯收錄了前三張單曲的主打歌，與受到粉絲好評的B面曲「{{lang|ja|Loving Pass ～恋のパスワード～}}」，還挑戰了不同類型的新歌，總共收錄了十首歌。視覺包裝以雲為主題，專輯封面表達的是Weather Girls穿著雲朵，乘著高氣壓來到日本<ref>{{Cite news|url = http://www.billboard-japan.com/d_news/detail/13048/2|title = Sexyお天気アイドルのウェザーガールズ“ふわふわのマシュマロ”|last = |first = |date = |work = billboard japan|accessdate = 2013-06-12}}</ref>。封面服裝被粉絲們稱為雲朵裝。{{citation needed}}

2013年8月10日，再次參加由日本[[愛貝克思|愛貝克思]]舉辦的「[[a-nation|a-nation]]」演唱會，擔任開場演出嘉賓<ref>{{Cite news|url = http://mdpr.jp/music/detail/1255427|title = AKB48、SKE48ら、「a-nation」に出演決定|last = |first = |date = |work = |accessdate = 2013-06-27}}</ref>。

;《{{lang|ja|アタックしますけど何か?}}》播出

《我要Attack咯 怎麼了嗎？》（原文：{{lang|ja|アタックしますけど何か?}}）是Weather Girls主持的日本綜藝節目，在名古屋電視台播出，共播出十集<ref>{{Cite web|url = http://www.nagoyatv.com/attack/|title = アタックしますけど何か？ 官方網站|accessdate = }}</ref>。在節目中，Weather Girls也首次挑戰日本動畫配音，為[[最強銀河_究極ZERO_Battle_Spirits|最強銀河 究極ZERO Battle Spirits]]代言<ref>{{Cite web|url = https://www.youtube.com/watch?v=jS5ZgX4Gj2U|title = Hijon and Mini animated|accessdate = }}</ref><ref>{{Cite web|url = http://www.mylifenews.net/resort/2013/08/post-42.html|title = バトルスピリッツ新シリーズが始動、宇宙を舞台にカードバトルを展開、新カード「アルティメット」も発表|accessdate =2013-08-19 }}</ref>。

=== 正式解散===

2016年10月，Mia因身體因素宣布將離開天氣女孩<ref>{{Cite web|url=http://news.ebc.net.tw/news.php?nid=44197|title=表演硬撐！天氣女孩Mia驚傳罹癌 暴瘦只剩38公斤|accessdate=|author=|date=|publisher=}}</ref>，10月份的[[WGSx7|WGSx7]]品牌發表為最後一次公開活動，天氣女孩將另覓新成員繼續活動。新加入的韓國籍成員紅雪於2017年5月11日的公開活動正式亮相。但2017年11月1日，十全娛樂以商業考量理由與天氣女孩各成員提前解約，並解散該團體。

=== 重新出發 ===
2018年4月在澳門商演中，經紀人找回成員[[Yumi|Yumi]]、[[Ria|Ria]]、[[Dara|Dara]]、[[Hijon|Hijon]]，重新組合天氣女孩，並開始活動。<br />
2019年3月Dara舉辦個人「女孩的日常・天氣 馬上看攝影展」，與天氣女孩[[Yumi|Yumi]]、[[Ria|Ria]]、[[Hijon|Hijon]]共同演出。

==  團員資料 ==

=== 汪沛㼆（Hi Jon） ===
Hi Jon畢業於國立臺北商業技術學院（現[[國立臺北商業大學|國立臺北商業大學]]），在2009年和2011年參加由國立臺北商業技術學院所舉辦的大專院校歌唱比賽。2010年9月首次在天氣預報節目《Weather Girls》登場，之後便為固定班底之一<ref>{{Cite web|url = https://www.youtube.com/watch?v=SUqtQs-D1kE|title = 影片始於1分40秒到2分39秒。Hi-Jon 於2010年9月首次在天氣預報節目《Weather Girls》登場的影片|accessdate = |author = |date = |publisher = }}</ref>，2012年3月時被選入團體中成為團員之一出道。<ref>{{Cite web|url = https://tw.celebrity.yahoo.com/news/娛樂星正妹-weather-girls鄭宜婷-嗨獎-逐夢飛翔的勇敢女生-071421073.html|title = 【【娛樂星正妹】Weather Girls鄭宜婷 (嗨獎)／逐夢飛翔的勇敢女生|accessdate = |author = |date = |publisher = }}</ref><ref>{{Cite web|url = https://www.facebook.com/thewonderfulwedding2015/photos/a.774586799272988.1073741829.740748302656838/793927410672260/|title = 【大囍臨門】人物介紹|accessdate = |author = |date = |publisher = }}</ref>

=== 林彣潔（Ria） ===
Ria，臺日混血兒，從小學習舞蹈，在肢體方面有很大的天賦。在2010年參加「[[蝴蝶蝴蝶生ㄉ真美麗|蝴蝶蝴蝶生ㄉ真美麗]]」節目舉辦的「水水高中生」比賽，在眾多參賽者之網路人氣投票後，以人氣投票率第二名進入總決賽(於2010年8月15日播出)，Ria曾多次參加舞蹈相關比賽及表演，舞藝過人。在經過長達一年的練習生訓練，於2013年被選為天氣女孩的成員之一（替補Dara）,也是少數會說韓文的藝人之一，各方面都有驚人的表現

加入Weather Girls之前為伊林娛樂旗下，並有多次的演出經驗。

=== 林-{采薇}-（Yumi） ===
Yumi畢業於文化大學，其兄為樂團[[1976|1976]]的貝斯手[[林子喬|林子喬]]<ref>{{Cite web|url = https://zh-tw.facebook.com/yumilinn/photos/a.619980448023891.1073741827.406946395993965/973066302715302/|title = Yumi臉書下方留言|accessdate = |author = |date = |publisher = }}</ref>。Yumi是Weather Girls中唯一於團體出道前即為藝人（以本名身分）之團員，擁有些許作品。2008年底參加[[超級星任務|超級星任務]]選秀節目，並榮獲冠軍。2009年成為[[伊林娛樂|伊林]]模特兒之一，並於2010年11月12日與[[小蠻|小蠻]]、[[詹乃蓁|珍奶]]、[[梁沛妤|梁沛妤]]、[[胡欣雅|胡欣雅]]、[[劉湘怡|劉湘怡]]等六名模特兒出寫真集《萌幻想》<ref>{{Cite web|url = https://www.youtube.com/watch?v=0qv_GNrCTEs#t=20|title = Yumi-《萌幻想》宣傳影片|accessdate = |author = |date = |publisher = }}</ref>，Yumi在裡面擔任女僕角色。Yumi有演出2009年[[愛就宅一起|愛就宅一起]](飾演毛毛)、[[熱血青春|熱血青春]]（飾演小美）、2010年[[犀利人妻|犀利人妻]]（飾演琦琦）。{{citation needed}}

2012年3月首次在天氣預報節目《Weather Girls》登場<ref>{{Cite web|url = https://www.youtube.com/watch?v=cu722gJtODU|title = yumi 於2012年3月在天氣預報節目《Weather Girls》首次登場的影片|accessdate = |author = |date = |publisher = }}</ref>，之後被選入團體中成為團員之一出道。<ref>{{Cite web|url = https://tw.celebrity.yahoo.com/news/娛樂星正妹-weather-girls林采薇-yumi-感性的溫柔少女-113627609.html|title = 【娛樂星正妹】Weather Girls林采薇 (Yumi)／感性的溫柔少女|accessdate = |author = |date = |publisher = }}</ref> 2014年1月和[[哈孝遠|哈孝遠]]合作拍攝微電影《[https://www.youtube.com/watch?v=ljshOoQgv7g 男人的小確幸 # 2]》，獲得43萬的高點擊率。

=== 趙靜儀（NueNue） ===
NueNue畢業於輔仁大學，2011年7月首次在天氣預報節目《Weather Girls》登場<ref>{{Cite web|url = https://www.youtube.com/watch?v=ZtURm5hdfTM|title = NueNue於2011年7月首次在天氣預報節目《Weather Girls》登場的影片|accessdate = |author = |date = |publisher = }}</ref>，2012年3月時被選入團體中成為團員之一出道。2015年與團員Ria一同客串演出東京電視台日劇《[[孤獨的美食家|孤獨的美食家]] Season 5 》臺灣篇。2016年在臺灣電視劇《[[我家是戰國|我家是戰國]]》任不少戲份<ref>{{Cite web|url = http://d.hatena.ne.jp/M14/20151024/p1|title = ウェザーガールズ「孤独のグルメ」レポ|accessdate = |author = |date = |publisher = }}</ref>
