'''天業路公園'''（'''Tin Yip Road Park'''）是[[香港|香港]][[新界|新界]][[天水圍新市鎮|天水圍新市鎮]]一個[[公園|公園]]，位於天水圍東部[[天業路|天業路]]，佔地2.5萬平方米，建於2014年，毗鄰[[天業路多用途沙灘球場|天業路多用途沙灘球場]]，由[[康樂及文化事務署|康樂及文化事務署]]管理，<ref name=oriental01>[http://orientaldaily.on.cc/cnt/news/20141006/00176_049.html 天水圍沙灘球場後日啟用]，東方日報，2014-10-06。</ref>興建費用為2億3200萬港元。

== 設施 ==
天業路公園設有區內首個大型[[人造草地球場|人造草地球場]]、[[緩步徑|緩步徑]]及階梯式園景休憩處。<ref name=oriental01/>公園內廣植樹木，採用多種節能措施，包括[[太陽能|太陽能]]路燈和熱水系統，以及雨水循環系統灌溉園景，營造寫意的綠化空間。

當中的大型人造草地球場更是[[新界|新界]]首個專為[[欖球|欖球]]而設計的場地，可用作11人足球或15人欖球的練習或比賽，還設有可容納220人的有蓋觀眾看台，以及球證室和更衣室等輔助設施。<ref>{{cite web |language=zh-hant |url=http://hk.ulifestyle.com.hk/spot/detail.html?id=ABdGDVowBwNZFwNt&utm_source=uhk&utm_medium=title&utm_campaign=search-all&utm_content=title1-2 |title=天業路公園 區內首個人造草地球場 |publisher=[[港生活|港生活]] }}</ref>人造草地球場每日上午8時至晚上11時開放。

== 問題 ==
天業路公園未對外開放前，園內植物早已枯萎。有市民發現公園有數以百棵的植物枯死，泥土亦全然乾旱，懷疑缺乏澆灌所致。區議員批評康文署監督建築承辦商不力，未能妥善護理植物，要求調查植物枯萎的原因，並加強管理。康樂及文化事務處發言人證實，公園內種植4,000多棵三色竹芋，其中200多棵未能適應新環境而枯萎。<ref name=sun01>[http://the-sun.on.cc/cnt/news/20141002/00681_001.html 新公園植物枯 康署監管疏懶]，太陽報，2014-10-02。</ref>
