'''天雄軍節度使'''，唐朝為'''魏博節度使'''，是[[唐朝|唐朝]]在今[[河北省|河北]]地区设置的[[節度使|節度使]]，唐末到[[五代|五代]]割据河北[[魏博|魏博]]（今[[邯郸市|邯鄲]]）一帶，为[[河朔三镇|河北三镇]]之一。在8世纪末是河北三镇的[[領袖|首领]]，9世纪衰落。

== 田氏割据 ==
首任[[节度使|节度使]][[田承嗣|田承嗣]]原为[[史朝义|史朝义]]旧将。[[廣德_(唐朝)|广德]]元年（763年）投降唐朝，唐朝为了笼络河北旧部，任命田承嗣为'''魏博德沧瀛五州都防御使'''（同年[[农历六月|六月]]改为'''魏博节度使'''），統率[[魏州|魏]]、[[博州|博]]、[[德州_(山东)|德]]、[[沧州_(古代)|沧]]、[[瀛州_(北魏)|瀛]]五州（今[[河北省|河北]]到[[山东省|山東]]一帶），驻[[魏州|魏州]]（今[[河北|河北]][[大名县|大名]]），拥兵十万，形同割据，与[[成德节度使|成德节度使]][[李宝臣|李宝臣]]、[[幽州节度使|幽州节度使]][[李怀仙|李怀仙]]称“[[河北三镇|河北三镇]]”。

[[大历|大历]]十年（775年），攻陷[[相州_(北周)|相州]]、[[卫州|卫州]]，田承嗣暗中勾结[[淄青节度使|淄青节度使]][[李正己|李正己]]，挑拨李宝臣与[[幽州节度使|幽州节度使]][[朱滔|朱滔]]，使[[朝廷|朝廷]]只得承认田氏割据。大历十四年（779年）田承嗣卒。临终时命侄子[[田悦|田悦]]为[[留后|留后]]。开了藩镇[[世袭|世袭]]先例。唐德宗[[建中_(唐朝)|建中]]年间，田悦先与[[成德|成德]][[李惟岳|李惟岳]]、[[淄青|淄青]][[李纳|李纳]]、[[山南东道|山南东道]][[梁崇义|梁崇义]]，后与成德[[王武俊|王武俊]]、淄青李纳、幽州朱滔起兵叛乱，史称四镇之乱。田承嗣的儿子[[田绪|田绪]]杀田悦，归附朝廷，依然享有[[魏博|魏博]]的[[政权|政权]]。之后，田绪的儿子[[田季安|田季安]]接任魏博节度使。

== 田氏归附 ==

[[元和_(唐朝)|元和]]七年（812年），田季安死后，子[[田怀谏|田怀谏]]年幼，田承嗣的堂侄[[田弘正|田弘正]]被军队拥立，田弘正归顺朝廷，放弃割据，为[[唐宪宗|唐宪宗]]平定[[淄青|淄青]]、[[成德|成德]]、[[淮西节度使|淮西]]立下大功，是[[元和中兴|元和中兴]]的关键人物。元和十五年（820年），田弘正调为[[成德节度使|成德节度使]]，[[李愬|李愬]]为魏博节度使。次年七月，田弘正被成德军都知兵马使[[王廷凑|王廷凑]]杀害。唐廷以田弘正之子[[田敦禮|田敦禮]]为魏博节度使，全军三万人讨王庭凑，同时命令横海、昭义、河东、义武诸军协同作战。值大雪缤纷，军不得进，田敦禮被[[牙将|牙将]][[史宪诚|史宪诚]]逼害，剚劍[[自刎|自刎]]，魏博由史憲誠割据。

== 何氏、韩氏、樂氏、罗氏割据 ==
[[唐文宗|唐文宗]][[太和_(唐朝)|太和]]三年（829年），軍中盛傳史宪诚即[[貪污|貪污]]軍餉，改調他處，[[何进滔|何进滔]]起兵杀死史宪诚而自立，开始何氏割据。何氏对[[朝廷|朝廷]]比较恭敬，[[朝貢|朝貢]]不绝，死後，子[[何弘敬|何弘敬]]即位。弘敬死，又傳子[[何全皞|何全皞]]。

870年，魏州兵變，何全皞被[[韩君雄|韩君雄]]所殺，君雄死後，子[[韩简_(唐朝)|韓簡]]即位。883年，韩簡被部下[[樂行達|樂行達]]推翻，行達之子[[樂從訓|樂從訓]]施政無道，888年，魏博局勢震動，樂行達剃髮为僧，部將[[罗弘信|罗弘信]]自立为节度使，樂從訓向[[汴州節度使|汴州節度使]][[朱溫|朱溫]]借兵討伐羅弘信，卻被羅弘信[[陣斬|陣斬]]，羅弘信賠款給朱溫，兩人和解。天祐元年（904年），賜魏博節度使號'''天雄軍節度使'''。

罗弘信的儿子[[罗绍威|罗绍威]]即位之後，归附[[朱温|朱温]]。他認為[[魏博|魏博]]長期以來局勢不稳，是牙军[[士卒|士卒]]随立节度使的结果。[[天祐_(唐朝)|天祐]]三年（906年），罗绍威遭遇兵變，他向朱温軍隊請求支援，进入[[魏博|魏博]]，杀死了大部分牙军，此举使魏博軍元气大伤，沦为小藩镇，罗绍威死後，子[[羅周翰|羅周翰]]即位，為[[後梁|後梁]]公主[[駙馬|駙馬]]，卻被宣武節度使[[楊師厚|楊師厚]]驅逐。[[後梁|後梁]]命羅周翰為宣武節度使，封[[楊師厚|楊師厚]]為魏博节度使，915年三月楊師厚死後，[[梁末帝|梁末帝]]将相、卫、澶三州分割别为一镇，以[[贺德纶|贺德纶]]为魏博节度使，以[[张筠_(五代)|张筠]]为相州节度使，导致魏博大亂，当月二十九日夜，楊師厚的银枪军親兵們发动叛乱，乱军首领[[张彦_(五代)|张彦]]派曹廷隐向河东[[李存勗|李存勗]]求援，李存勗立即领军从晋阳赶赴魏州，杀死张彦，收编银枪军，魏博镇遂归后唐。

[[李存勗|李存勗]]稱帝，是為[[後唐莊宗|後唐莊宗]]，同光四年（926年），魏博軍的一名士兵[[皇甫暉|皇甫暉]]，因其部隊被發配[[貝州|貝州]]（今[[河北省|河北省]][[南宮市|南宮市]]），不能歸[[鄴城|鄴城]]（今[[河北省|河北省]][[大名縣|大名縣]]），在一晚赌博没有取胜后怒而發動[[兵變|兵變]]，煽動士兵殺主帥[[楊仁晸|楊仁晸]]，立偏將[[趙在禮|趙在禮]]為[[留後|留後]]，是為[[鄴城之變|鄴城之變]]，存勗命[[李嗣源|李嗣源]]討伐，嗣源被軍士挾持，被迫與魏博叛軍合兵造反，往[[洛邑|洛邑]]前進。莊宗正打算御駕親征時，卻被懷有私仇的部將[[郭從謙|郭從謙]]起兵殺害，史稱[[興教門之變|興教門之變]]，[[興教門之變|興教門之變]]亦是[[鄴城之變|鄴城之變]]的延續，故莊宗也可說死於[[魏博|魏博]][[兵變|兵變]]。
