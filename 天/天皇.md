'''天皇'''（{{jpn|j=天皇|hg=てんのう|rm=tennō|f=y}}）是[[日本|日本]]的[[君主|君主]]，為當今世界唯一使用[[皇帝|皇帝]]名號的[[國家元首|國家元首]]，以其為首的[[日本皇室|日本皇室]]則是世界上现存最古老的[[皇室|皇室]]。在[[明治維新|明治維新]]至1947年期間施行的《[[大日本帝國憲法|大日本帝國憲法]]》明文規定天皇為[[日本|日本]]的國家元首<ref>新村出『広辞苑 第六版』（岩波書店、2011年）1952頁；松村明『大辞林 第三版』（三省堂、2006年) 1758頁</ref>；之後接續施行至今的《[[日本國憲法|日本國憲法]]》則定義其為「日本國以及[[日本|日本]]國民整體的[[國家象徵|象徵]]」，為[[象徵天皇制|有實無名]]的[[虛位元首|虛位元首]]。主要職責是任命[[日本內閣總理大臣|首相]]、批准[[國務大臣_(日本)|國務大臣]]的任免、簽署[[法律|法律]]和[[命令|命令]]、以及進行禮儀性的外交事務等。

天皇族系號稱「萬世一系」，從首任的[[神武天皇|神武天皇]]以來一脈相傳，並不像[[中国|中国]]、[[朝鮮_(稱謂)|朝鮮]]、[[越南|越南]]等[[汉字文化圈|周邊]]國家经历過改朝换代，這是由於天皇在[[日本歷史|歷史]]上能掌握實權的時間很短，反而能避過戰亂，使皇室更加長壽。由於天皇與整個皇室在日本古代被認為是超乎普通人的存在，因此時至今日都沒有[[姓氏|姓氏]]（历史学研究上称其为[[天皇氏|天皇氏]]或[[天皇家|天皇家]]）。而在[[神道|神道教]]傳統中，天皇被認為是[[天照大神|天照大神]]的後裔，故具有「[[神_(神道)|神]]性」；但[[二戰|二戰]]後，[[昭和天皇|昭和天皇]]發表《[[人間宣言|人間宣言]]》，放棄天皇被賦予的神性。

通常，現任天皇被稱為「[[今上天皇|今上天皇]]」，敬稱「[[陛下|陛下]]」。[[退位|退位]]後被稱為「[[太上天皇|太上天皇]]」（簡稱上皇）；若其退位後[[出家|出家]]則被稱為「[[太上法皇|太上法皇]]」（簡稱法皇）；天皇[[駕崩|駕崩]]時，將先被稱為「[[大行天皇|大行天皇]]」，之後再正式諱稱為「［在位時年號］天皇」（例如[[明治天皇|明治天皇]]）。

現任天皇（第125任）為[[明仁|明仁]]<!--（第126任）為[[德仁|德仁]]-->，是[[昭和天皇|昭和天皇]]<!--[[太上天皇|上皇]][[明仁|明仁]]-->的長子，年号[[平成|平成]]<!--[[令和|令和]]-->，於1989年1月7日<!--2019年5月1日-->即位。

== 名稱由來 ==
「天皇」又可稱[[天帝|天帝]]，語源源自中國[[道教|道教]]的[[天帝|天帝]]即[[天皇大帝|天皇大帝]]。日本方面最早的文字记载是《{{link-ja|飛鳥淨御原令|飛鳥浄御原令}}》，由第40代[[天武天皇|天武天皇]]制定，689年頒佈。

早期[[日本历史|日本历史]]於西元第一至三世纪，小邦林立，各小邦的[[君主|君主]]称“[[王|王]]”或“[[君|君]]”，约到西元第四、五世纪诸邦形成[[联盟|联盟]]，[[首领|首领]]改称为「[[大王|大王]]」或「[[大君|大君]]」，以及稱「[[治天下大王|治天下大王]]」。到了天武天皇時代，受到同時期自稱天皇的[[唐高宗|唐高宗]]所影響，又因崇尚中國道教的北斗信仰，選用天皇成為稱謂。其後还曾有天皇和「[[皇帝|皇帝]]」称号并用的情况，如43代元明、45代圣武（追谥）、46代孝谦、50代桓武。民間過往亦稱天皇為「君上」或「大君」。

外交场合使用首次“天皇”一词据称是推古天皇时期，[[日本書紀|日本書紀]]記載，[[推古天皇|推古天皇]]时第二次出使[[隋朝|隋朝]]的[[国书|国书]]中有“东天皇敬白西皇帝”之句<ref>《日本書紀・推古天皇》：「九月辛未朔乙亥、饗客等於難波大郡。辛巳、唐客裴世淸罷歸。則復以小野妹子臣爲大使、吉士雄成爲小使、福利爲通事、副于唐客而遺之。爰天皇聘唐帝、其辭曰「東天皇敬白西皇帝。使人鴻臚寺掌客裴世淸等至、久憶方解。季秋薄冷、尊何如、想淸悆。此卽如常。今遣大禮蘇因高・大禮乎那利等往。謹白不具。」。」</ref>，但是中國對此並無任何記載、可能是天武天皇欲加強其名銜正當性而杜撰的<ref>古代東亞政治環境中天皇與日本國的產生  呂玉新</ref>。到了[[明治|明治]]元年（1868年）后，日本致外国首脑信件、国际条约批准书、宣战诏书使用的都还是[[皇帝|皇帝]]称号。1936年推行[[大東亞共榮圈|大東亞共榮圈]]的口号後，对外就完全使用天皇称号。中国称日本元首为日本天皇大约是在清末的[[同治|同治]]时期。

== 歷史 ==
[[File:Politics_Under_Constitution_of_Japan_04.svg|缩略图]]
=== 古代 ===
[[日本神話|日本神話]]書籍《[[日本書紀|日本書紀]]》称，天皇乃是[[太陽神|太陽神]][[天照大神|天照大神]]之後裔，此理論成为日本[[君权神授|君权神授]]的依据。旧时日本史书一直宣称天皇是「{{link-ja|萬世一系|万世一系}}」，即所有天皇都来自同一家族，[[日本历史|日本历史]]上，[[皇室|皇室]]从来没有出现过家族更迭。事实上，中古时期的日本小国林立，最后由位于[[本州岛|本州岛]]中部（[[奈良縣|奈良縣]]）的[[倭朝廷|倭朝廷]]实现统一，[[大和國|倭]]（{{lang|ja|やまと}}）也就成了日本的别称。倭朝廷成立前出现过多次[[國王|王位]]的争夺，而當時歷史紀錄靠史官口頭背誦，沒有留下可供佐證的文字史料，是東亞國家中除中國之外，存留有最早文字史料的國家即是日本。韓國的《百濟記》、《百濟新撰》、《百濟本記》據傳比日本書紀早，卻除了《日本書紀》所引用的部分之外完全失傳。今存的《三國史記》整整比《日本書紀》晚了數個[[世紀|世紀]]，僅有[[中國歷史|中國歷史]]學家[[陳壽|陳壽]]《[[三國志|三國志]]》的東夷傳[[倭人|倭人]]條可做參考。日本是在[[律令制|律令制]]开始后才有「天皇」一名。在《日本國史略》中有「自天照皇太神創業垂統，而[[神武天皇|神武天皇]]初都中國、一統天下，歷正天皇正統一系，亘萬世而不革。天下即一人之天下。」一辭，不過萬世一系四字則是到19世纪末[[明治维新|明治维新]]左右才形成並寫入《[[大日本帝国宪法|大日本帝国宪法]]》之中。

自公元6世纪[[倭王權|倭国]]征服[[本州|本州]]及[[九州_(日本)|九州]]大部分地区后，天皇的权力达到顶峰。但天皇作为日本实际统治者的时间并不长。从10世纪开始，日本经历了[[摄关政治|摄关政治]]、[[源氏|源]][[平氏|平]]相争，政權[[攝關政治|被權貴門閥把持]]；有些天皇甚至負氣[[禪讓|禪讓]]，自稱[[上皇|上皇]]，在[[江湖|江湖]]之中實施[[院政|院政]]；或者[[剃髮|剃髮]][[出家|出家]]，自稱[[法皇|法皇]]，在[[伽藍|伽藍]]之中掌控大權。12世紀以後，日本進入數世紀的[[武家政權|武家政權]]（[[軍政|軍事政權]]）時代，政府被[[征夷大將軍|幕府將軍]]或者僭稱[[公卿|公卿]]的[[武家|武家]]階層掌控，包括了[[平氏政權|平氏政權]]、[[镰仓幕府|镰仓幕府]]、[[室町幕府|室町幕府]]、[[安土桃山时代|織豐政權]]與[[江户幕府|江户幕府]]。天皇的[[政治權力|政治權力]]及存在作用被架空長達千年。

至1867年[[明治天皇|明治天皇]]登基後，江戶幕府迫於時局而行[[大政奉還|將政權交還朝廷]]，武家統治時代結束，天皇重新掌握政治权力。在明治天皇與[[伊藤博文|伊藤博文]]等[[元老_(日本)|元老]]的主導下，日本引進歐陸國家的[[二元制君主立憲制|二元制君主立憲制]]、並在1890年施行《[[大日本帝國憲法|大日本帝國憲法]]》，天皇成為「神聖不可侵犯」的統治者，擁有絕對的政治權力。另一方面，當時的日本政府將[[神道|神道]]信仰發展成近乎[[國教|國教]]地位的「[[國家神道|國家神道]]」，由於天皇在傳統上被認為是[[天照大神|天照大神]]後裔，因此天皇被[[神格化|神格化]]尊為「{{link-ja|現人神|現人神}}」，意為以人的型態存在於現世的神。

=== 近代 ===
[[昭和天皇|昭和天皇]]登基後不久，日本在[[東條英機|東條英機]]等軍部帶領下投入[[中國抗日戰爭|侵略中國]]與[[第二次世界大戰|第二次世界大戰]]戰事中。二戰後，日本被以[[美國|美國]]為首的[[同盟國_(第二次世界大戰)|同盟國]]佔領，並在[[同盟國軍事佔領日本|軍政統治]]下建立[[议会民主制|议会民主制]]；同盟國原計畫廢除天皇制度，但為穩定局勢，允许保留天皇作为象征性的国家元首，即「[[象徵天皇制|象徵天皇制]]」。1946年，昭和天皇在[[盟軍最高司令官總司令部|同盟國佔領當局]]的主導下发表了《[[人间宣言|人间宣言]]》，承认天皇是人而非神。

依照《[[日本國憲法|日本國憲法]]》，現今日本天皇的主要职责是任命[[日本內閣總理大臣|内阁总理大臣]]（首相），批准法律、政令及条约，召集国会，批准[[国务大臣|国务大臣]]的任免，出席礼仪性的外交事務活動和国家儀典等。

== 日本以外的称呼 ==

=== 中華民國 ===
[[中華民國|中華民國]]政府與民間，一般直接稱呼'''天皇'''。

=== 中華人民共和國 ===
[[中華人民共和国政府|中華人民共和国政府]]一般直接使用[[日文|日文]]汉字称其“天皇”<ref>{{cite web|url=https://www.fmprc.gov.cn/web/gjhdq_676201/gj_676203/yz_676205/1206_676836/1206x0_676838/|title=日本国家概况|date=2018年6月|publisher=中華人民共和國外交部}}</ref>。

=== 朝鮮半島 ===
朝鲜半岛受传统[[华夷|华夷]]思想影响，认为只有过去中国的君主才可以称为“皇帝”。因此除了[[朝鲜日治时期|日本殖民统治]]时期外，朝鲜半岛一般称日本天皇为“日王”（{{lang|ko|일왕}}，“日本国王”的简称）<ref>{{cite web|url=http://imnews.imbc.com/20dbnews/history/1989/1817731_19354.html|script-title=[히로히토 사망]일본 천황 오늘 새벽 사망[김승한]|publisher=[[文化廣播公司|文化廣播公司]]|date=1989-01-07|quote=히로히토 일왕이 오늘 오전 6시 33분 87살의 일기로 숙환으로 사망했습니다.|language=ko}}</ref>。

=== 英语 ===
[[英語|英語]]的称呼是「Emperor」、「Emperor of Japan」，也有音译为「Tenno」者。

== 三神器 ==
天皇的[[三神器|三神器]]包括：
* [[八咫鏡|八咫鏡]]：一柄镜子，现在陈列于[[三重县|三重县]][[伊势市|伊势市]]的[[伊势神宫|伊势神宫]]<ref>{{Cite web |url= http://www.isejingu.or.jp/chinese/myth/myth4.htm |title= 伊勢神宮 |deadurl= yes |archiveurl= https://archive.is/20120804031526/http://www.isejingu.or.jp/chinese/myth/myth4.htm |archivedate= 2012-08-04 }} </ref>。
* [[天叢雲劍|天叢雲劍]]：又名「草薙劍」，源于創世神话中[[素戔嗚尊|素戔嗚尊]]的寶剑，他斬殺[[八岐大蛇|八岐大蛇]]後從其尾部取得。现在供奉于[[名古屋|名古屋]]市[[热田区|热田区]]的[[热田神宫|热田神宫]]<ref>{{Cite web |url= http://www.atsutajingu.or.jp/jingu/about/history.html |title= 熱田神宮の歴史 }} </ref>。
* [[八尺瓊勾玉|八尺瓊勾玉]]：又稱「八阪瓊曲玉」，一枚勾玉。现在供奉于{{Fact|time=2007-11-01T17:18:08Z|date=June 2007}}[[东京|东京]]的[[皇居|皇居]]。

日本神話中，天叢雲劍是素戔嗚尊斬殺八岐大蛇而得。關於天叢雲劍後來的下落，有一說指出，1185年日本两大武士集团平氏和源氏在[[坛之浦|坛之浦]]海域决战，源氏获胜，而平氏所立的[[安德天皇|安德天皇]]带着此剑葬身海底；另有一說指出，[[日本武尊|日本武尊]]带着此劍在海底長眠。今日熱田神宮所供之天叢雲劍是否為傳說中的寶劍，實已不得而知。至於八咫鏡以及八尺瓊勾玉都在日本「記紀神話」（即《[[古事記|古事記]]》和《[[日本書紀|日本書紀]]》）裡和[[天照大神|天照大神]]有關故事的部分中被提及；名稱中的「八咫」與「八尺」原先只是在形容這兩者的尺寸相當大（有人说八尺是周朝尺寸），後來就變為這兩種神器的專屬名稱<ref>新訂 古事記 {{cite web |url=http://applepig.idv.tw/kuon/furu/text/kojiki/kojiki_top.htm |title=存档副本 |accessdate=2007-08-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20070824171319/http://applepig.idv.tw/kuon/furu/text/kojiki/kojiki_top.htm |archivedate=2007-08-24 }}</ref><ref>新編 日本書紀 {{cite web |url=http://applepig.idv.tw/kuon/furu/text/syoki/syokitop.htm |title=存档副本 |accessdate=2007-08-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20071004221715/http://applepig.idv.tw/kuon/furu/text/syoki/syokitop.htm |archivedate=2007-10-04 }}</ref>。

三神器在[[日本历史|日本历史]]上通常是上任天皇传给下任天皇。偶尔也有通过各種手段争夺的时候，如[[南北朝_(日本)|南北朝]]时期，北朝[[後小松天皇|後小松天皇]]用下任天皇繼承權做交換，誘騙南朝[[後龜山天皇|後龜山天皇]]交出所保有的三神器，南朝天皇在交出三神器後退位，但皇統最後由後小松天皇的兒子繼承，到此日本南北朝的分裂合而為一。
