'''天宫院站'''位于[[北京市|北京市]][[大兴区|大兴区]]，是北京地铁[[北京地铁大兴线|大兴线]]的终点站。

== 命名 ==
天宫院站以天宫院村命名。据《[[宸垣识略|宸垣识略]]》，天宫院位于在[[宛平县|宛平县]]城南六十华里处，本名史家庄，由于[[金章宗|金章宗]]驾幸该村打围举膳，改名天宫院。<ref>{{cite news|title=地铁新线路的站名掌故|url=http://cul.china.com.cn/minsu/2011-01/04/content_3936362_3.htm|accessdate=2016-09-03|agency=文化中国-中国网|date=2011-01-04}}</ref>

== 位置 ==
天宫院站位于[[新源大街|新源大街]]与[[天府路_(北京)|天府路]]交汇处。<ref>{{cite web|url=http://www.bjghw.gov.cn/web/static/articles/catalog_14300/article_ff80808123fa497b0124529f19b30013/ff80808123fa497b0124529f19b30013.html|title=轨道交通大兴线规划方案公告|publisher=[[北京市规划委员会|北京市规划委员会]]|date=2009-10-14|accessdate=2010-11-27|archive-url=https://web.archive.org/web/20101227061642/http://www.bjghw.gov.cn/web/static/articles/catalog_14300/article_ff80808123fa497b0124529f19b30013/ff80808123fa497b0124529f19b30013.html|archive-date=2010-12-27|dead-url=yes}}</ref>

== 结构 ==
该站为地下车站，[[岛式站台|岛式站台]]设计。设4个出入口：A（西北）、B（东北）、C（东南）、D（西南）<ref name="走近轨道交通大兴线">{{cite web|url=http://zw.bjjtw.gov.cn/jtzw/dtxx/201010/t20101029_38915.htm|title=走近轨道交通大兴线|date=2010-10-29|accessdate=2010-11-27|publisher=[[北京市交通委员会|北京市交通委员会]]}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。站外有7000平方米的P+R停车场。<ref name="走近轨道交通大兴线"/>
