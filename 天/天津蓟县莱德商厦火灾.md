'''天津蓟县莱德商厦火灾'''，官方称之为'''蓟县“6·30”火灾事故'''，是指2012年6月30日发生在[[中国|中国]][[天津市|天津市]][[蓟县|蓟县]]县城莱德商厦的一起[[火灾|火灾]]。

== 背景 ==
莱德商厦位于蓟县县城中昌北大街与新华大街交叉路口东北侧<ref>{{cite web| url=http://www.tjsjx.com/html/2012/07/111017444282.htm| title=也谈蓟县莱德商厦的6.30火灾| author=蓟州孟| publisher=蓟县生活网| date=2012-07-11 |language=zh-cn}}</ref>，{{fact|主要经营范围包括服装、化妆品、床上用品等|time=2012-07-09}}。

== 经过 ==
2012年6月30日下午3点起火，造成多人死伤。起火原因可能是空调室外机超负荷工作起火。《新世紀》雜誌称火灾发生时商场經理因怕顾客不付钱就离开，曾下令将西南門和西門兩道出入口卷簾門拉下而關閉出口<ref>{{cite news| url=http://magazine.caixin.com/2012-07-06/100408126.html| title=薊縣大火：悲劇是這樣發生的 |work=財新《新世紀》雜誌2012年第27期 |date=2012-07-06| language=zh-cn| quote=火灾原因可能是空调室外机超负荷工作起火}}</ref>。据众多自称目击者的网民称，很多3楼以上的被困者选择跳楼逃生<ref name="bjnews">{{cite news| url=http://www.bjnews.com.cn/news/2012/07/01/207701.html| title=天津蓟县莱德商厦火灾被扑灭| work=新京报| date=2012-07-01| language = zh-cn| quote=据记者报道，当时3层以上的被困者，有很多人选择跳楼逃生，“场面惨不忍睹”。}}</ref>。当晚火被扑灭<ref name="bjnews"></ref>，整栋商厦均被烧毁。根据官方提供的监控录像，所谓“跳楼逃生”、“封闭大门”、“死伤300余人”的说法并不属实。<ref>{{cite news|url=http://news.cntv.cn/china/20120717/113871.shtml |title=真相调查：天津蓟县莱德商厦火灾| publisher=中国网络电视台 |language=zh-cn}}</ref>。

2013年4月11日市政府公布官方調查，認定事故直接原因：商廈一層東南角中轉庫房內空調電源線未穿金屬管保護，其導線[[电弧|电弧]]持續作用，導致線路異常溫升及相間[[絕緣體|絕緣]]性能下降，發生相間電弧性[[短路|短路]]，引燃周圍可燃物。間接原因是，商廈領導層、薊縣消防支隊（原公安薊縣消防大隊）、薊縣公安局、薊縣商務委、[[渔阳镇|渔阳镇]]政府、薊縣青少年宮（萊德商廈五樓出租方）、天津電力公司薊縣分公司（原天津薊縣供電局）、薊縣政府未能切實有效落實各自消防安全職責<ref name="en20130411">{{cite news|url=http://news.enorth.com.cn/system/2013/04/11/010846087.shtml |title=薊縣萊德商廈「6·30」重大火災處理結果公布| publisher=天津北方網 |language=zh-cn}}</ref>。

== 灾后 ==
===官方===
7月3日下午，公安部召开莱德商厦“6·30”重大火灾事故通报电视电话会议，通报称次事故造成10死16重伤，过火面积约5000平方米，初步认定起火部位位于该商厦东南角中转仓库内，起火原因正在调查中<ref name=nddaily>{{cite news | url=http://epaper.oeeee.com/A/html/2012-07/07/content_1668074.htm | title=蓟县商厦大火10人遇难名单公布 | work=南方都市报 | date=2012-07-07 | language=zh-cn }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。7月6日，天津市人民政府正式公布10名遇难者名单。10名遇难者均为女性，其中9名工作人员，一名顾客<ref name=nddaily></ref><ref>{{cite web | url=http://society.people.com.cn/n/2012/0706/c86800-18461790.html | title=天津蓟县“6·30”火灾10名遇难者名单公布 均为女性| publisher=人民网| date=2012-07-06| language = zh-cn}}</ref>。遇难者遗体已被家属认领<ref>{{cite web | url=http://news.enorth.com.cn/system/2012/07/06/009592075.shtml | title=天津蓟县商场火灾事故善后处理工作有序开展| publisher=天津北方网 | date=2012-07-06| language = zh-cn}}</ref>。

另从公安部门获悉，“有个别网民在互联网上编造、传播谣言，造成了恶劣的社会影响”。“根据已查清的相关事实，公安部门依照有关法律法规，对造谣传谣者进行了处理”。当事人承认夸大死亡人数、冒充武警和目击者编造传播虚假信息等事实，表示悔过。<ref>{{cite web |url=http://epaper.bjnews.com.cn/html/2012-07/09/content_355014.htm | title=蓟县火灾造谣者受处理 当事人承认夸大死者数| publisher=新京報 | date=2012-07-09 | language = zh-cn}}</ref>

===媒体===
有媒体报道转述网上消息称实际死亡人数为378人，认为天津市政府有瞒报死亡人数之嫌<ref>{{cite news| url=http://www.bbc.co.uk/zhongwen/simp/chinese_news/2012/07/120706_china_fire_death.shtml | title=蓟县大火官方死亡数字受网民质疑| publisher=BBC中文网 | date=2012-07-06| language = zh-hans}}</ref>。但是这一数字也被认为过于精确、违背常识，非常符合网络谣言的特征<ref name="tencent" />。

有记者报道，在现场拍照时遭警方阻拦、威胁“再拍就没收相机”。后来欲与不知亲人下落的家属接触时，又被有关人员跟踪。<ref>{{cite news | url=http://news.hexun.com/2012-07-01/143060057.html | title=天津蓟县6.30火灾现场追踪 | publisher=人民网 | accessdate=2012-07-01 | language=zh-cn | deadurl=yes | archiveurl=https://web.archive.org/web/20150606051819/http://news.hexun.com/2012-07-01/143060057.html | archivedate=2015-06-06 }}</ref>

事发之后，天津本土的媒体全部失声，仅以170字官方通稿报道此事，引发外界批评<ref name="tencent">{{cite news| url=http://view.news.qq.com/zt2012/tjdh/bak.htm | title=天津大火：十条人命与170字简讯 | publisher=腾讯网 | accessdate=2012-07-07| language = zh-cn}}</ref>。《[[云南信息报|云南信息报]]》作为异地媒体发表社论《蓟县大火需要彻底的信息公开》，指出公众应当享有对此次火灾事故的知情权，蓟县政府应当彻底的公开全部信息<ref>{{cite web | url=http://henan.sina.com.cn/smx/news/gdtt/2012-07-06/309-13416.html | title=蓟县大火需要彻底的信息公开 | publisher=云南信息报 | accessdate=2012-07-07| language = zh-cn}}</ref>。

===悼念===
7月6日下午，很多蓟县当地民众聚集在当地的[[蓟县鼓楼|鼓楼]]广场“悼念”莱德商厦火灾死难者<ref>{{cite news| url=http://www.bbc.co.uk/zhongwen/simp/chinese_news/2012/07/120706_jixian_tianjin_fire.shtml | title=蓟县火灾“头七”民众广场聚集纪念| publisher=BBC中文网| date=2012-07-06| language = zh-hans}}</ref>。

===追责與判刑===
事發後數天，新浪微博有网民题为“敦促天津市原副市长只升华、蓟县县委书记肖松引咎辞职”的微博被广为转载，但多被删除<ref>{{cite web | url=http://www.molihua.org/2012/07/blog-post_9205.html | title=敦促天津市原副市長只升華、薊縣書記肖松引咎辭職 | publisher=中國茉莉花革命網站 | accessdate=2012-07-07| language = zh-cn}}</ref>。

2013年4月11日，天津市政府對薊縣該事故調查處理意見作出批覆，認定這是一起責任事故。以重大勞動安全事故罪，一審判處萊德商廈實際控制人姚海利有期徒刑六年，商廈法定代表人姚學良有期徒刑四年，商廈副總經理高卓海、劉學良、王芳有期徒刑三年，緩刑四年，商廈電工宗福群有期徒刑三年緩刑三年；分別以涉嫌濫用職權罪、受賄罪和涉嫌玩忽職守罪，對原薊縣消防大隊大隊長楊繼倉、原薊縣消防大隊防火科參謀呂桂河刑事拘留。

依照相關紀律規定，給予薊縣漁陽鎮黨委副書記、漁陽鎮鎮長李武鎖和薊縣商務委主任、黨組書記高建國行政警告處分，給予市公安局薊縣分局副局長李思義、薊縣商務委副主任胡元華、薊縣商務委辦公室主任潘曉東行政記過處分，給予薊縣商務委內貿科副科長張起行政記大過處分，給予薊縣消防支隊長趙春波、薊縣消防支隊防火處處長楊燕傑記過處分，給予薊縣消防支隊防火處參謀陳天雲記大過處分。

對薊縣副縣長馬占亭、劉素俠誡勉談話並作深刻檢查；對薊縣青少年宮、天津電力公司薊縣分公司分別予以通報批評，並約談主要負責人；責成薊縣政府向市政府作出書面深刻檢查；對萊德商廈罰款90萬元人民幣，對商廈實際控制人姚海利處以上一年年收入60%的罰款<ref name="en20130411"></ref>。
