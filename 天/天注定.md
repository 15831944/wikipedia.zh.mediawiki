《'''天注定'''》（{{lang-en|'''''A Touch Of Sin'''''}}）是一部2013年[[贾樟柯|贾樟柯]]编剧、导演的中国电影，由[[姜武|姜武]]、[[王宝强|王宝强]]、[[赵涛|赵涛]]和[[罗蓝山|罗蓝山]]主演<ref name="multiple">{{cite web |url = http://ent.gd.sina.com.cn/news/20130419/775048.html | language= | title=《天注定》是一部“当代人的武侠片”|publisher= |date=2013-04-19 |accessdate= }}</ref>，获得[[第66届戛纳国际电影节|第66届戛纳国际电影节]][[坎城影展最佳劇本獎|最佳剧本奖]]<ref>[http://www.apdnews.com/photo/26870.html 中国导演贾樟柯获戛纳电影节最佳剧本奖] {{webarchive|url=https://web.archive.org/web/20131126035206/http://www.apdnews.com/photo/26870.html |date=2013-11-26 }}，亚太日报，2013年5月27日</ref>。中国原计划于2013年11月上映<ref>{{cite web |url = http://news.mtime.com/2013/05/17/1511894.html | language= | title=《天注定》确定过审今秋上映 姜武盼拿金棕榈
 |website = Mtime[[时光网|时光网]] |date=2013-05-17 |accessdate= }}</ref>，却迟迟没有上映。虽然传言被禁映，但是目前还没有任何官方消息可以证实该片被禁<ref>{{cite news |title = 贾樟柯《天注定》中国上映期未定 |url = http://cn.tmagazine.com/film-tv/20131125/t25film/ |accessdate = 2013-12-04 |website = 纽约时报中文网 |date = 2013年11月25日 |archive-url = https://web.archive.org/web/20131129082032/http://cn.tmagazine.com/film-tv/20131125/t25film |archive-date = 2013-11-29 |dead-url = yes }}</ref>。本片於2014年10月7日在[[齐齐哈尔电视台|齐齐哈尔电视台]]播出<ref>{{cite web |url = http://www.weibo.com/1719014980/Bqtzb5jSA |language = |title=新浪娱乐快报2014.10.7內容 |publisher=新浪娱乐快报 |date=2014-10-07 |accessdate= }}</ref>。其後被导演贾樟柯指是电视台盗播<ref >{{cite web |url = http://www.zaobao.com/wencui/entertainment/story20141008-397948 |language= |title = 贾樟柯《天注定》遭地方台盗播 主创齐声讨 |website = 早报网 |date=2014-10-08 |accessdate= }}</ref>。

贾樟柯介绍，片名“天注定”这三个字的字体，取自山西书法家[[傅山|傅山]]的手迹<ref name="multiple" />。

== 劇情簡介 ==
讲述四个中国人各自面对生活中的忧伤、爱和希望。此片是分段式電影。

== 创作 ==
《天注定》的英名片名是“{{lang|en|A Touch of Sin}}”（一丝[[恶|恶]]念），与[[胡金铨|胡金铨]]的《[[侠女|侠女]]》英文名“{{lang|en|A Touch of Zen}}”（一缕[[禅|禅]]机）十分相似，贾樟柯表示，它是向[[胡金铨|胡金铨]]、[[张彻|张彻]]等[[武侠|武侠]]巨匠的致敬作品，此片灵感在筹备《在清朝》时突然产生，贾称之为“闪电一般的灵感”。2012年10月初开机，历时5个月拍摄完成<ref name="multiple" />。

导演表示对于影片的风格可以称之为一部“现代侠义电影”，着重构建古人与现代人的关系，而这种关系的纽带便是“侠义”。通过人物的[[暴力|暴力]]行动，来刻画如今的中国社会和中国人的生活面貌<ref >{{cite web|url=http://news.mtime.com/2013/05/17/1511847-2.html | language= | title=贾樟柯《天注定》题材大胆 反映中国两极分化|publisher=Mtime时光网 |date=2013-05-17 |accessdate= }}</ref>。

贾樟柯在本片中亲自客串饰演一位到[[东莞|东莞]]“天上人间”寻快活的[[山西|山西]][[土豪|土豪]]。他游走在一群身穿[[红军|红军]]军装的妙龄[[少女|少女]]中间，出场不足一分钟却做了三件事：「买了[[徐悲鸿|徐悲鸿]]的一幅画、和朋友约定有时间到[[澳门|澳门]][[赌场|赌场]]、叫了一个[[妓女|小姐]]」。<ref >{{Cite web|url=http://ent.ifeng.com/movie/news/mainland/detail_2014_02/10/33660021_0.shtml | language=zh-Hans | title=[视频]《天注定》东莞选妃片段曝光 导演亲上阵|publisher=凤凰网娱乐 |date=2014-02-10 |accessdate= }}</ref>

== 评价 ==

2013年5月，本片入围[[第66届戛纳电影节|第66届戛纳电影节]]「主竞赛单元」单元<ref>{{cite web|url=http://ent.163.com/13/0418/19/8SP2SUSK00034RCN.html | language= | title=戛纳电影节公布片单 贾樟柯新片入围竞赛单元
 |publisher= 网易娱乐 |date=2013-04-18 |accessdate= }}</ref>，于2013年5月17日在[[戛纳|戛纳]]进行了全球首映，电影节官方场刊为该片打出3.0分（满分为4分）<ref>{{cite web|url=http://news.sina.com.cn/o/2013-05-19/063927163623.shtml | language= | title=《天注定》戛纳获高分|publisher= 新浪 |date=2013-05-19 |accessdate= }}</ref>。后来又入围第51届[[纽约电影节|纽约电影节]]。

在美国上映后，媒体综评77分，其中《[[纽约时报|纽约时报]]》与《[[偏锋杂志|Slant]]》杂志给满分；[[烂番茄|烂番茄]]新鲜度93%，获得许多好评。

美媒代表性评价：“一个劲酷且小说化的中国黑色故事，渗透着人类社会进步中的一点罪孽和一点不朽”，“贾樟柯用他犀利却沉稳的笔触描绘着人们对战利的追求以及道德的腐败，从这个角度而言，其震撼力远远超过对空洞民族主义的贬低和污化”，“快速且残忍的流血事件，那在机车后座的一抹血红和开放式扭曲描绘，恍惚间似乎塔伦提诺附体，那血迹斑斑的壁花更是有如《[[杀死比尔|杀死比尔]]》的另类风情”，“一个导演大胆且精力充沛的自我改造与自我放逐”。<ref>[http://ent.sina.com.cn/m/f/2013-10-07/ba4019275.shtml 新片《地心引力》《逆转王牌》《天注定》]</ref>

== 上映多舛 ==

2013年10月30日传出《天注定》被禁消息。當天下午1点，出版人王小山在其微博上最先发布了《天注定》被禁的消息，不过他之后不久即删除了这条微博。片方回应称并不知道影片被禁这件事，也没有接到任何书面通知，影片仍然有望于11月份国内上映<ref>{{cite web |url=http://she.21cn.com/emotions/social/a/2013/1030/11/24764593.shtml |language= |title=传《天注定》被禁 北美上映口碑好评 |publisher=21CN女人频道 |date=2013-10-30 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20131202223804/http://she.21cn.com/emotions/social/a/2013/1030/11/24764593.shtml |archivedate=2013-12-02 }}</ref>，但是最终并没有在11月上映。贾樟柯在12月8日接受采访表示目前还处在一个讨论沟通的过程中，并且认为该片一定会上映，但多久之后上映还未确定<ref>{{cite web|url=http://ent.ifeng.com/movie/news/mainland/detail_2013_12/09/31911773_0.shtml | language= | title=[独家]贾樟柯谈新片被禁：仍在评估 明年春季海外发碟|publisher= 凤凰网娱乐 |date=2013-12-09 |accessdate= }}</ref>。

2014年2月28日，《天注定》在中国大陆还没有公映网络上出现了该片的下载版本，导演贾樟柯在他微博中说道自己仍在为《天注定》上映一事不断奔走，而网上盗版的流传让“大陆市场尽失”，並表示自己的公司将承担起相应的经济责任，弥补合作者的损失<ref>{{cite web|url=http://ent.ifeng.com/a/20140301/40006223_0.shtml | language= | title=《天注定》盗版下载一夜蹿红 贾樟柯愿赔偿投资损失|publisher= 凤凰网娱乐 |date=2014-03-01 |accessdate= }}</ref>。

根据业内人士的看法，由于影片深刻地反应了一些中国当代社会的黑暗面，已经被广电总局列为[[禁片|禁片]]。在2013年度[[中国电影导演协会|中国电影导演协会]]颁奖典礼上，最佳影片和最佳导演两个大奖空缺，引起中国网民和影迷的热议。有电影人表示中国导演们的此举是在抗议中国苛刻的[[审查制度|审查制度]]；有网友表示，空缺是留给贾樟柯和《天注定》的<ref >{{cite web|url=http://www.rfa.org/mandarin/yataibaodao/kejiaowen/vt-04102014150644.html | language= | title=中国导协大奖选不出最佳影片和导演 |publisher=自由亚洲电台 |date=2014-04-10 |accessdate= }}</ref>。
