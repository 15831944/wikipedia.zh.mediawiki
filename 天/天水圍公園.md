'''天水圍公園'''（{{lang-en|'''Tin Shui Wai Park'''}}）位於[[香港|香港]][[新界|新界]][[天水圍新市鎮|天水圍新市鎮]]中央，佔地14.8[[公頃|公頃]]，是[[元朗區|元朗區]]的大型公園，公園的四周均由建築物所包圍，包括有[[嘉湖山莊|嘉湖山莊]]、[[置富嘉湖|置富嘉湖]]、[[天瑞邨|天瑞邨]]及[[天耀邨|天耀邨]]等，[[天水圍體育館|天水圍體育館]]、[[天水圍運動場|天水圍運動場]]和[[天水圍游泳池|天水圍游泳池]]座落天水圍公園旁。

==歷史==
天水圍公園前稱'''天水圍市鎮公園'''，於早期規劃稱為「天水圍第22區分區公園」<ref>《217LS ─ 天水圍第22區分區公園第一期工程—提升為第三類工程》(區域市政局委員會備忘錄，RSD 6/HQ 726/86(9) III，1991年8月5日)</ref><ref>《217LS ─ 天水圍第22區分區公園第一期工程—提升為第二類工程》(區域市政局委員會備忘錄，S/F RSD 6/HQ 726/86(9)，1992年12月28日)</ref>，為天水圍新市鎮的主公園。整個公園原本屬於[[嘉湖山莊|嘉湖山莊]]的發展項目之一，並且由[[長江實業|長江實業]]附屬公司「巍城有限公司」斥資1億1000多萬元興建<ref>《257LS — 天水圍第22區地區公園第二期工程政府與巍城有限公司合作發展的工程》(區域市政局委員會備忘錄，RSD 5/HQ 726/89(9)，1993年12月24日)</ref>。1991年時，公園一期的棕櫚園外觀已成形，同時鄰近的[[嘉湖銀座|嘉湖銀座]]（現時的置富嘉湖）也正進行前期工程<ref>{{Cite web |url=https://www.facebook.com/TSWODS/photos/a.118203771711105.1073741828.118162585048557/528561900675288/?type=3 |title=新發展區 天水圍新市鎮 |author=天水圍老照片}}</ref>。天水圍公園分為兩期落成，第一期在1993年3月26日開放，第二期在1997年1月16日全面開放予居民使用。公園落成初期曾預留大量[[草坪|草坪]]供居民在該處休憩。不過到了1998年，原有的草坪種滿了樹木，草坪不對外開放。此外，公園的[[人工湖|人工湖]]在落成初期大受歡迎，准許進行划船的水上單車租賃服務，並設有小食部<ref>{{Cite web |url=https://www.facebook.com/TSWODS/photos/a.118203771711105.1073741828.118162585048557/647849952079815/?type=3 |title=天水圍公園水上單車 |author=天水圍老照片}}</ref>，不過後期因不明原因，人工湖於2003年開始不准划船<ref name=r1>[http://www.lcsd.gov.hk/parks/tswp/b5/index.html 康樂及文化事務署 - 天水圍公園 - 背景] {{webarchive|url=https://web.archive.org/web/20060718162345/http://www.lcsd.gov.hk/parks/tswp/b5/index.html |date=2006-07-18 }}</ref>。

==設施==
園內有水景設施、噴水池及人工湖營造了「天」與「水」的效果，以配合「天水圍」這名稱，還包括以下其他設施：

* 太極園
* 觀景亭
* 模型船區（前身為進行水上單車活動的人工湖）
* 模型車場
* 香味花園
* 門球場及練習場
* 網球場
* 露天劇場
* 滑板場
* 迷宮
* 人造地面籃球暨排球場
* 健身站
* 兒童遊樂場

==禁煙安排==
