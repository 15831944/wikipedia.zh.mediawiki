'''天外天劇場'''（{{jpn|j=天外天劇場|hg=てんがいてんげきじょう|rm=Tengaiten Gekijyo|f=h}}）是一座位於[[臺灣|臺灣]][[臺中市|臺中市]]，啟用於[[臺灣日治時期|臺灣日治時期]]1936年3月的[[電影院|電影院]]和[[劇場_(建築)|劇場]]。該劇院建築之建造是由臺中仕紳[[吳子瑜|吳子瑜]]出資15萬，[[臺灣總督府|臺灣總督府]]技師[[齋藤辰次郎|齋藤辰次郎]]負責設計建造；該建築物曾於2014年被[[臺中市政府|臺中市政府]]列為暫定古蹟，現仍屬於私人財產。

==興建==
{{Cquote|其規棋之宏壯華麗與東京寶塚無二。|張麗俊<ref>{{Cite web |url = http://taichustory.blogspot.tw/2014/03/tengaiten.html |title = 台中櫻町 天外天劇場 即將被拆 |author = 臺中文史復興組合|date = 2014年3月1日 |publisher = Blogspot |language = zh-tw |accessdate = 2016年8月29日 |quote = 與吳子瑜同為櫟社成員的豐原仕紳張麗俊，在在該劇場落成前參觀盛讚「其規棋之宏壯華麗與東京寶塚無二」。 }}</ref>|}}{{Cquote|子瑜導余與幼春觀其建築將半之戲台，又觀歌仔戲2、30分鐘。|林獻堂<ref>{{Cite web |url = http://newtalk.tw/news/view/2015-09-10/64492 |title = 不輸東京寶塚 天外天劇場卻遭拆除殆盡 |author = 黃東軍 |date = 2015年9月10日 |publisher = 新頭殼 |language = zh-tw |accessdate = 2016年8月29日 |quote = 台灣議會之父－林獻堂在1935年3月3日到吳子瑜家中作客，並參加「全台聯吟大會」，會後還特地前去參觀天外天劇場，在他的日記中記載著：「子瑜導余與幼春（林幼春）觀其建築將半之戲台，又觀歌仔戲2、30分鐘。」然而，再不久，這間劇場將成絕響，徹底從台中人的記憶中移除。 }}</ref>|}}

1919年，曾任[[臺灣省城|臺灣省城]]董工總理的仕紳[[吳鸞旂|吳鸞旂]]在[[臺中市|臺中市]]櫻町興建私人戲院，其子[[吳子瑜|吳子瑜]]（[[臺灣日治時期|臺灣日治時期]]三大詩社之一的[[櫟社|櫟社]]成員）再出資15萬，並由[[臺灣總督府|臺灣總督府]]技師[[齋藤辰次郎|齋藤辰次郎]]設計。於1933年9月13日擴建，1936年3月竣工啟用。這座劇場有三層樓的空間，可容納630席觀眾，當時輪流上演[[電影|電影]]、[[歌仔戲|歌仔戲]]、[[京劇|京劇]]，還設置了[[食堂|食堂]]、[[咖啡廳|咖啡廳]]、跳舞場、茶店等空間<ref>{{Cite web |url = https://medium.com/napotsioh/765688a0c2c5 |title = 日本時代的臺中戲院「天外天劇場」面臨拆除危機 |author = 許組長 |date = 2016年8月29日 |publisher = 藍寶石 |language = zh-tw |accessdate = 2016年8月29日 |quote = 原由臺中仕紳吳鸞旂出資，興建於大正8年（1919年）的私人戲院，其子吳子瑜（臺灣日治時期三大詩社之一的「櫟社」成員）再出資15萬，於昭和8年（1933年）9月13日擴建、昭和11年（1936年）3月竣工啟用這座劇場，是臺灣總督府技師齋藤辰次郎在臺中唯一的現存作品，開幕之時，630席的劇院輪流上演著戲劇與電影，裡頭還設置了食堂、咖啡廳、跳舞場、茶店⋯⋯等空間，劇場在二戰時停業，戰後吳子瑜將天外天賣給仕紳王博，所得款項拿來修建臺北料亭「梅屋敷」（現逸仙公園），天外天劇場在戰後更名為「國際戲院」，繼續播放電影，直到1975年轉為經營太源冷凍廠，又經歷國際賽鴿中心、收費停車場，至今呈現荒廢狀態，成為外拍 Model 的廢墟熱門景點。 }}</ref>，為[[臺灣|臺灣]]當時規模最大的民營歐式戲院，也是臺中繼[[樂舞台|樂舞台]]之後第二座由[[臺灣人|臺灣人]]投資興建的現代化劇場。

==大事記==
*1919年，[[台灣日治時期|台灣日治時期]]原[[清朝|清朝]][[臺灣省城|臺灣省城]]建城工程總理[[吳鸞旂|吳鸞旂]]出資在吳宅旁興建私人戲院。<ref>{{Cite web |url = http://napotsioh.com/news/42 |title = 日本時代的臺中戲院「天外天劇場」面臨拆除危機 |author = 許組長 |date = 2016年8月29日 |publisher = 藍寶石 |language = zh-tw |accessdate = 2016年8月29日 |quote = 原由臺中仕紳吳鸞旂出資，興建於大正8年（1919年）的私人戲院，其子吳子瑜（臺灣日治時期三大詩社之一的「櫟社」成員）再出資15萬，於昭和8年（1933年）9月13日擴建、昭和11年（1936年）3月竣工啟用這座劇場，是臺灣總督府技師齋藤辰次郎在臺中唯一的現存作品。 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
*1933年，[[吳鸞旂|吳鸞旂]]之子[[吳子瑜|吳子瑜]]出資擴建並命名為'''天外天劇場'''。
*1936年，在[[臺中市_(州轄市)|臺中市]][[櫻町|櫻町]]正式開幕。
*1938年，《[[臺灣日日新報|臺灣日日新報]]》報導其建物在[[二戰|二戰]]時屋頂塗上大紅色違反防空規定。<ref>{{Cite web |url = http://a.udn.com/focus/2016/03/16/18839/index.html |title = 台中天外天劇場瀕危 掀文資保衛戰 |author = 洪敬浤、陳秋雲 |date = 2016年3月16日 |publisher = 聯合報 |language = zh-tw |accessdate =2016年8月29日 |quote = 文史工作者表示，這座劇院除了娛樂，也是當時仕紳交流的空間；日本統治推動皇民化、說日語，但「天外天」放無聲電影解說時，都講閩南語，戰爭末期日本政府要求屋頂漆成迷彩的防空色，劇院圓頂硬漆成大紅色，還被當時報紙批評「非常（不像話）」。 }}</ref>
*1943年，台灣總督府與吳家協商，將吳公館部分廂房與戲院的部分空間作為臺灣總督府臺中農林專門學校（今[[國立中興大學|國立中興大學]]）的臨時學生宿舍。<ref>[https://udn.com/news/story/6965/3744858 百年中興 在臺中的第一代宿舍群]，聯合報，2019-04-12</ref>
*1945年，因第二次世界大戰停業。
*1950年，[[臺灣戰後時期|民國戰後時期]]吳將劇場售予王博，後更名為'''國際大戲院'''。
*1975年，劇場建物轉作太原冷凍廠（泰源冷凍廠）經營之用。<ref>{{Cite web |url = http://udn.com/news/story/7314/1566089 |title = 傳地主想拆 台中天外天劇場掀保衛戰 |author = 洪敬浤、陳秋雲 |date = 2016年3月16日 |publisher = 聯合報 |language = zh-tw |accessdate = 2016年8月29日 |quote =中日戰爭後，吳子瑜把劇院與土地賣給當時仕紳王博，更名國際大戲院，但逐漸沒落，一度改為冷凍製冰廠、鴿舍等，最後閒置荒廢；如今所有權人多達50多人，產權複雜很難處理。 }}</ref>
*1990年，冷凍廠關閉轉作鴿舍與釣蝦場經營。
*2012年，週遭再轉作停車場經營，建物上層荒廢中。
*2014年，首度依照《[[文化資產保存法|文化資產保存法]]》進入文資審議，獲得為期半年的暫定古蹟保護；當時為[[台中市長|台中市長]]候選人[[林佳龍|林佳龍]]（同年底當選市長）亦呼籲保存該文化資產。<ref>[https://udn.com/news/story/7266/2468996 中市府准拆天外天劇場 綠議員：林佳龍的競選承諾呢?]，聯合報，2017-5-17</ref>
*2016年8月，房屋所有權人開始動工拆除建物內部設施，台中市政府相關局處官員到場關切後停工再次進入文資審議。<ref>{{Cite web |url = http://www.peoplenews.tw/news/a9570a7f-c15a-4437-80b0-481eaeff9ae6 |title =「天外天」未經許可擅拆立面 都發局制止 文化局盼所有權人整建 |author = 王伯仁 |date = 2016年3月24日 |publisher = 民報 |language = zh-tw |accessdate = 2016年8月29日 |quote = 東區老戲院「天外天劇場」今（24）日驚傳施工人員拆除立面，台中市政府都發局、警察局與文化局等皆前往「關心」。施工人員宣稱是收到危樓改善通知，因此施工改善危險部分。都發局表示，改善通知是要求所有權人施設安全防護措施，並非拆除，因此當場要求停止作業；文化局則表示，天外天劇場雖未通過文資審議，但確有歷史意義，期待所有權人以整建代替拆除。 }}</ref>
*2017年4月，文史團體送審文資身分再度遭駁回，[[台中市文化局|台中市文化局]]准許所有權人可自由處分。<ref>[https://udn.com/news/story/7325/2479874 中市府准拆「天外天」 文史團體連署搶救]，聯合報，2017-5-23</ref><ref>[http://m.ltn.com.tw/news/life/breakingnews/2077834 搶救天外天劇場 網路連署籲中市府撤回恐龍決議]，自由時報，2017-5-24</ref>
