'''天{{zy|台|-{zh-hans:tāi; zh-hant:ㄊㄞˊ}-}}宗'''又稱'''法{{zy|華|-{zh-hans:huā; zh-hant:ㄏㄨㄚ}-}}宗'''，[[漢傳佛教十宗|漢傳佛教十宗]]之一，始於[[南北朝|南北朝]]末期，是漢傳佛教中最早一個完全由[[中國佛教|中國佛教]][[法師|法師]]所創立的本土性宗派。因其實際開創者[[智顗|智顗]]（智者）大师常駐[[浙江|浙江]][[天台山_(浙江)|天{{zy]]說法，故稱'''天台宗'''。以《[[妙法蓮華經|妙法蓮華經]]》為其根本經典，又被稱為'''[[法華宗|法華宗]]'''。因为该宗注重於修行[[止观|止观]]，又称'''止观宗'''。因為它重視佛教教理的研究，同時結合了止觀的實際修行，以「教觀雙美」而聞名。中國天台宗祖庭在浙江[[天台山國清寺|天台山國清寺]]，湖北[[玉泉寺_(当阳市)|玉泉寺]]和[[寧波|寧波]][[延庆寺_(宁波)|延慶寺]]、[[觀宗寺_(寧波)|觀宗寺]]亦為傳播天台教法的重要[[道場|道場]]<ref>{{cite web|title=追根溯源 尋宗問祖 —— 佛教宗派祖庭略解|url=http://www.hkbuddhist.org/zh/page.php?p=booklet_attachment&page=8&cid=7&kid=2&attach_id=874#7}}</ref>。[[宋朝|宋朝]]之後，天台宗在中國衰弱，多數[[門徒|門徒]]歸於[[禪門|禪門]]。

天台宗于9世纪初被日本僧人[[最澄|最澄]]传到[[日本|日本]]，盛極一時。與中國不同，在[[日本佛教|日本佛教]]成為一個很重要的宗派，並引入[[密教|密教]]修法，稱為[[台密|{{zy]]。日本天台宗[[本山_(佛教)|總本山]]則為[[比叡山|比叡山]][[延曆寺|延曆寺]]。

== 概論 ==
天台宗与[[禅宗|禅宗]]、[[华严宗|华严宗]]都是由中國本土佛教論師在中國開創，被認为最具中国特色的三大佛教宗派。天台宗的最大特色在于“教观双美”。在[[汉传佛教|汉传佛教]]大乘八宗之中，华严、法相、三论诸宗偏于教义理论的发挥；禅、净、律、密诸宗偏于观行实践的进取。天台宗讲究将“教观”两者发挥到极致并圆融一体。

天台宗学统上承九祖：[[龙树|龙树]]、[[慧文|慧文]]、[[慧思|慧思]]、[[智顗|智顗]]、[[灌顶大师|灌顶]]（章安大师）、[[智威|智威]]、[[慧威|慧威]]、[[玄朗|玄朗]]、[[湛然|湛然]]。實際立宗開創者為[[智顗|智顗]]，他的著作為天台宗宗義的根據。天台宗以《[[妙法莲华经|妙法莲华经]]》為根本經，指南是《[[大智度论|大智度论]]》、《[[法华玄义|法华玄义]]》和《[[法华文句|法华文句]]》，扶疏是《[[大般涅槃經_(大乘)|大般涅槃经]]》，观法是《[[大品般若经|大品般若经]]》。

9世纪初，日本僧人最澄将此宗传到日本，在[[平安时代|平安时代]]（784～1192），与[[真言宗|真言宗]]并列发展，史称“[[平安二宗|平安二宗]]”。13世纪由此宗分出日莲宗。当今，日本天台本宗、[[日莲宗|日莲宗]]都很兴旺，而日莲宗在20世纪又分出几个新兴教派。

==歷史==
天台宗的始祖，據[[章安灌頂|章安灌頂]]《[[摩訶止觀|摩訶止觀]]》序中的說法，可分為金口傳承與今師傳承二系：金口傳承始於[[釋迦牟尼|釋迦牟尼]]，[[大迦葉|大迦葉]]受法後一系傳承，至師子，其弟子[[末田地|末田地]]與[[商那和修|商那和修]]二師同時，依《[[付法藏因緣傳|付法藏因緣傳]]》在印度共傳承24祖<ref>[[章安灌頂|章安灌頂]]《[[摩訶止觀|摩訶止觀]]》序：「大覺世尊，……付法藏人，始迦葉，終師子，二十三人。末田地與商那同時，取之則二十四人。諸師皆金口所說，並是聖人，能多利益。」</ref>。今師傳承，始於《付法藏因緣傳》中13祖[[龍樹|龍樹]]，至[[慧文|慧文]]、[[慧思|慧思]]，加上立宗者智顗，共四師<ref>[[章安灌頂|章安灌頂]]《[[摩訶止觀|摩訶止觀]]》序：「南岳事慧文禪師，當齊高之世，獨步河淮，法門非世可知。文師用心，一依《釋論》，論是龍樹所說，付法藏中，第十三師。智者《觀心論》云：『歸命龍樹師。』驗知龍樹，是高祖師也。」</ref>。但是由龍樹至慧文之間的傳承世系不明。

唐朝[[荊溪湛然|荊溪湛然]]另外提出九師傳承說法，列出明、最、嵩、就、監、慧六師，隨後為慧文、慧思、智顗<ref>[[荊溪湛然|荊溪湛然]]《[[止觀輔行傳弘決|止觀輔行傳弘決]]》卷1：「若准九師，相承所用。第一諱明，多用七方便，恐是小乘七方便耳。自智者已前，未曾有人立於圓家七方便故。第二諱最，多用融心，性融相融，諸法無礙。第三諱嵩，多用本心，三世本無來去，真性不動。第四諱就，多用寂心。第五諱監，多用了心，能觀一如。第六諱慧，多用踏心。內外中間心不可得，泯然清淨，五處止心。第七諱文，多用覺心。重觀三昧，滅盡三昧，無間三昧，於一切法心無分別。第八諱思，多如隨自意安樂行。第九諱顗，用次第觀，如次第禪門。用不定觀，如六妙門。用圓頓觀，如大止觀。」</ref>。但是前六師的事蹟不明。北宋宗鑑《釋門正統》中，在慧文之前，列出明最、神最、嵩多、就多、鑒多、慧多六師，這六師都是行小乘禪觀，但事蹟不詳，是前後相承關係<ref>北宋宗鑑《釋門正統》：「慧多從鑒多，鑒多從就多，就多從嵩多。嵩多從神最，神最從明最。明最至師凡七世，只修小乘七方便耳。」</ref>。但南宋志磬《佛祖統記》認為這六師都是同時代的人，非前後相承<ref>《佛祖統紀》卷6：「又云：『智者受業思師。思師從道就師，就師受法最師。』今詳思師，本承文師，今又言從就師。是知諸師多同時，互相咨稟，而法門改轉，後多勝前，非復可論相承也。」</ref>。

日本[[最澄|最澄]]的天台宗教脈傳承中，在龍樹與慧文之間，列入[[鳩摩羅什|鳩摩羅什]]與[[善慧大士|善慧大士]]。

各項記載中，都推崇[[慧文|慧文]]為天台初祖，但因為他沒有著作傳世，除了他生活在[[北齊|北齊]]時代，其餘事蹟不詳，實際的教學也不詳。

== 教理 ==
=== 性具善惡說 ===
天台宗主張「如來不斷性惡，一闡提不斷性善」。他們認為，[[佛陀|佛陀]]並沒有把性惡斷除，只是不受性惡影響，正如同[[一闡提|一闡提]]惡性眾生，仍然具備性善，只是性善沒被發揮。
=== 判教思想 ===
{{main|天臺宗判教}}
由於[[印度佛教|印度佛教]]長時期的發展，形成大、[[小乘|小乘]]等不同的派別和經論，形成龐雜的關係。在[[魏晉南北朝|魏晉南北朝]]期間，由西來譯僧逐漸翻譯出來，[[中國佛教|中國佛教]]徒面對內容有別、思想分歧的大、小乘經典，因而產生[[判教|判教]]思想。而天台宗的判教，有「五時判教」、「化儀四教」及「化法四教」。
==== 五時判教 ====
五時判教是將[[佛陀|佛陀]]一生的說法內容，判釋為五個時期，相應於每一時，便有每一時所說的經典與所度化的對象。五時中，第一是「華嚴時」，第二為「阿含時」，第三是「方等時」，第四是「般若時」，第五是「法華涅槃時」。

==== 化儀四教 ====
化儀四教是對眾生說法的不同形式，因為眾生的根基不一。教化形式可分為「頓」、「漸」、「秘密」、「不定」四種。
==== 化法四教 ====
化法四教是佛陀對眾生不同的理解能力而講說適當的內容，分別為「藏」、「通」、「別」、「圓」四種。

=== 主要修行方式 ===
{{Main|止观}}
止观者，止是[[禅定|禅定]]之胜因，观是智慧之由藉，谓若能成就二法，自然定慧圆明也。<ref>《三藏法数》明·一如等</ref> 止是[[止息|止息]]一切[[妄念|妄念]]，观是观察一切真理。止属于[[定|定]]，观属于[[慧|慧]]，止观就是[[定|定]][[慧|慧]]双修的意思。是天台宗的主要修行方式。
=== 天台忏法 ===
智者大师所撰的忏仪著述主要有四本：《法华三昧行事运想辅助仪礼〈法华经〉仪式》（法华三昧忏仪）、《方等三昧行法》（方等忏法）、《金光明忏法》、《请观世音忏法》，这是天台宗忏法的依据<ref>{{cite|title=试论两宋间的天台忏法|author=觉慧法师}}</ref>。同时，故智者大师以天台忏法确立了中國佛教懺法門的組織模式<ref>{{cite|title=天台忏法之研究|author=释大睿|publisher=中华佛教研究所论丛25}}</ref><ref>{{cite|title=天台宗以忏法为修行规范的僧制建设|publisher=《法音》2007年05期}}</ref>。

=== 歷代禪師主張 ===
# '''[[慧文|慧文]]禅师'''：其理论主張“三智与三观”
# '''[[慧思|慧思]]禅师'''：其理论基础上发展了“[[一心三观|一心三观]]”的理论：一切事物都由因缘所生，没有固定不变的实体，这是观空；但虽然如此，一切事物都有自己的像貌，这是观假；然而空与假是同一的，此谓观中。在这个基础上智顗认为世上有3000种世间，但这3000种世间都出于一念心中，因此称之为“[[一念三千|一念三千]]”。能将这一切都领会，那么人就达到了顿断三惑，圆证三智的境界。这變成是天台宗的中心思想。十如是与《大乘止观法门》、定慧双修、禅教一致、安乐行。
# '''[[智者大师|智者大师]]'''：其理论主張如下：詳見[[一念三千|一念三千]]、三谛圆融、五时八教、性具善恶（与华严性起善恶区别）六即佛、三种止观的著作、三大部与五小部。

== 天台宗（中國） ==
*[[慧文|慧文]]禪師，俗姓高，渤海人，南北朝魏齐年间的佛教高僧，佛教天台宗初祖。
*[[慧思|慧思]]（515年－577年），南北朝佛教禅师，為天台宗二祖。十五岁出家，曾师当时著名的慧文禅师，从受禅法。
*[[智顗|智顗]]（538年－597年），俗姓陈，字德安，世稱'''智者大师'''。生于梁武帝大同四年，十八岁投湘州果愿寺法绪出家，二十三岁拜[[慧思|慧思]]为师。天台宗（或称“法华宗”）的实际创始人，以《法华经》为中心，龍樹的《大智度論》及《中論》為輔。
*[[章安灌頂|章安灌頂]]（561年－632年），天台宗[[智顗|智顗]]大師之徒，被尊為天台祖四祖。他整理出天台三大部，在天台宗教義確立上有重大貢獻。
*[[荊溪湛然|荊溪湛然]]（711年－782年），著有《止觀輔行傳弘決》，釋天台之摩訶止觀。
*{{tsl|ja|道邃|興道道邃}} (?)，荊溪湛然的弟子，日本僧[[最澄|最澄]]（日本天台宗和台密創立人）向其請學。

會昌法難（845）以降，唐末五代戰亂不休，教典散失。至宋初遣使向韓國請回教典，爾後[[四明知禮|四明知禮]]復興天台教學。知禮圓寂後的山家派，分為神照系（神照本如）、廣智系（廣智尚賢）、南屏系（南屏梵臻），與禪宗、凈土多有交融。

*[[四明知禮|四明知禮]]（960年－1028年）
*[[遵式|慈雲遵式]] (964年-1032年)
*淨覺仁岳 (992-1064)  曾師事四明知禮十餘年，後放棄山家派觀點，力倡山外，投[[遵式|遵式]]門下，自成一派，為天台宗後山外派的代表人物。
*從諫慈辯 (1036-1110)，[[高丽|高丽]]僧[[义天|义天]]（韓國天台宗創立人）向其學習天台教觀，屬於南屏一系。

*[[幽溪傳燈|幽溪傳燈]]（1554年－1627年），一生專修《楞嚴經》耳根圓通，晚年期心淨土

== 日本天台宗 ==
{{seealso|:ja:天台宗#日本の天台宗|日蓮法華宗}}
唐[[貞元|貞元]]二十年（804年），日僧[[最澄|最澄]]來唐，從師[[道邃|道邃]]、[[行滿|行滿]]學習天台宗教義；次年，返國創立'''日本天台宗'''。日後，[[圓仁|圓仁]]（慈覺大師）、[[圓珍|圓珍]]（智証大師）等數多知名僧侶輩出。

日本天台宗的正式名稱是“天台法華圓宗”，或“法華圓宗”、“天台法華宗”。古時常單以“法華宗”稱之。又有[[日蓮宗|日蓮宗]]、[[日蓮法華宗|日蓮法華宗]]、{{lj|日蓮法華宗 (正福寺)|日蓮法華宗|正福寺日蓮法華宗}}，都曾以“法華宗”自稱，分立後，為避免混淆，都不單以“法華宗”相稱。

此宗融入了包含[[法華宗|法華宗]]、[[密宗|密宗]]、[[禪宗|禪宗]]、[[淨宗|淨宗]]的要素，尤其密宗，故稱'''天台密宗'''、'''台密'''。

=== 天台密教（台密） ===
和[[真言宗|真言宗]]的密教「[[東密|東密]]」相對，天台宗的[[密教|密教]]稱為「[[台密|{{zy]]」。日本天台宗和中國天台宗的不同之處，在於中國天台宗並未導入密教、也不包含密教。

=== 四宗兼學 ===
最澄所開的日本天台宗是繼承智顗的學說、以法華經為中心，同時也包含禪、戒、念佛、密教的要素，在性格上已異於中國天台宗。延曆寺亦稱為'''四宗兼學'''的道場。

== 高麗天台宗 ==
宋[[元丰_(北宋)|元豐]]八年（1085年），高麗僧人[[義天|義天]]來朝學習宗門教義；[[元祐_(年號)|元祐]]元年（1086年），攜經書返高麗。[[高麗肅宗|高麗肅宗]]二年（1097年），創立高麗天台宗，而後在此基礎上形成了本土的[[曹溪宗|曹溪宗]]<ref> 徐新啦：《朝鲜半岛天台宗的传承——以高丽大觉国师义天为中心》</ref>。

== 台灣天台宗 ==
[[台灣|台灣]]天台宗有日本與中國大陸兩個傳承。
*日本傳承：[[台灣日治時期|日治時代]][[明治|明治]]44年（1911年），以[[比叡山|比叡山]][[延曆寺|延曆寺]]為總本山的天台宗來台灣布教。不屬於[[八宗十四派|八宗十四派]]的天台宗[[修驗道|修驗道]]也相繼前來布教。天台宗在[[基隆|基隆]]設立[[法王寺_(基隆市)|法王寺]]，並在各地設立布教所。戰後，日籍僧人皆被遣回日本，法脈斷絕。
*中國大陸傳承：
**大明延平郡王世子[[鄭經|鄭經]]為其母[[董太妃|董太妃]]創建[[台南開元寺|台南開元寺]]開始，歷經禪門曹洞、臨濟叢林如月眉山[[靈泉禪寺|靈泉禪寺]]釋善慧、觀音山[[凌雲禪寺|凌雲禪寺]]釋本圓、苗栗[[大湖法雲禪寺|大湖法雲禪寺]]釋覺力等法師之弘揚，始獲偏佈於全省。
**[[斌宗法師|斌宗法師]]師從[[靜權法師|靜權法師]]，在臺灣新竹古奇峰創法源寺。<ref>[http://buddhism.lib.ntu.edu.tw/museum/formosa/people/1-bin-zong.html 佛教人物：斌宗法師]臺灣佛教數位博物館網頁。</ref>
**[[釋曉雲|䁱雲法師]]師從[[倓虛法師|倓虛法師]]。
