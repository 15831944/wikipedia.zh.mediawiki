《'''天下一大五郎系列'''》是[[日本|日本]][[推理|推理]]作家[[東野圭吾|東野圭吾]]原作的，以[[偵探|偵探]]天下一大五郎為主人公的連續[[推理小說|推理小說]]系列的總稱。系列首部作品為《[[名偵探的守則|名偵探的守則]]》。

==概要==
本系列小說以輕鬆[[幽默|幽默]]的[[諷刺|諷刺]]口吻，對[[推理小說|推理小說]]常用的故事構成方式，及對推理作家或讀者進行批判。連對推理小說中的[[潛規則|不成文規則]]甚至是常識也毫不留情地鞭撻。<!--また、著者自身推理作家として、作品内で述べた皮肉について「意識している」と語っている。-->

主要登場人物有2個，他們本身有著自己是「小說人物」的自我意識，所以一離開作品世界，他們會就推理小說中的問題加以討論。帶有[[後設小說|後設小說]]的元素。

==登場人物==
===主要角色===
; <span id="天下一大五郎">{{nihongo|天下一 大五郎|{{ruby|天下一|てんかいち}} {{ruby|大五郎|だいごろう}}|Tenkaichi Daigorou}}</span>
: 自認「頭腦清晰、博學多才、行動力拔群的[[名偵探|名偵探]]」。大河原警部似乎對他有敬而遠之的感覺，但在舞台背後會和從作品世界抽離角色的警部一起討論[[推理小說|推理小說]]的問題點。如果對案件有不明白的地方，會悄悄從大河原警部那裡獲得提示。因為某個案件而對大型密室感到厭惡。

; <span id="大河原番三">{{nihongo|大河原 番三|{{ruby|大河原|おおがわら}} {{ruby|番三|ばんぞう}}|Oogawara Banzou}}</span>
: 縣警[[警視廳#刑事部_Criminal_Investigation_Bureau|搜查一課]][[警部|警部]]，42歲。一直飾演和名偵探天下一相對的、在推理小說中必定出現的蹩腳警部。但實際上他會比天下一更早解決案件，但故意裝出懷疑並非真兇的嫌疑人的樣子。在《[[名偵探的詛咒|名偵探的詛咒]]》中也有出場。
