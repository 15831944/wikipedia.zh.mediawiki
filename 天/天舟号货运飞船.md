'''“天舟号”货运飞船'''是[[中国航天|中国航天]]计划用于提供[[中國空間站|中國空間站]]补给服务的无人[[货运飞船|货运飞船]]。于2017年4月20日从[[西昌卫星发射中心|西昌卫星发射中心]][[海南文昌航天发射场|海南文昌航天发射场]]使用新一代[[长征七号运载火箭|长征七号运载火箭]]进行首次发射。用于补给[[天宫二号|天宫二号空间实验室]]以及计划中的[[中國空間站|“天宫号”空间站]]。

== 名称 ==
2011年4月25日中国载人航天工程办公室向公众开放征名活动，为未来的中国货运飞船命名。截至5月20日，已收到征名5万余{{参|1={{Cite web|url=http://news.cntv.cn/20110520/112339.shtml|title=中国货运飞船征名超5万个|publisher=中国网络电视台|date=2011-05-20|accessdate=2013-06-23|language=zh-cn}}}}。7月8日，中国首位飞天的航天员[[杨利伟|杨利伟]]与中国科学院院士、空间技术专家[[戚发轫|戚发轫]]座谈[[中国中央电视台|中国中央电视台]]时揭晓入选前十名的名称，它们分别为：“天梭”、“鲲鹏”、“天舟”、“神龙”、“龙舟”、“神骥”、“天马”、“云梯”、“神驹”、“行者”{{参|1={{Cite web |url = http://news.163.com/11/0709/01/78G0TK3600014AED.html |title=中国货运飞船征名结束 龙舟等10个名字入选 |publisher=网易新闻 |date=2011-07-09 |accessdate=2013-06-23 |language=zh-cn}}}}。2013年10月31日，定名“天舟”，由[[中國空間站|“'''天'''宫号”空间站]]和[[神舟飞船|“神'''舟'''号”载人飞船]]各取一字组成，代号“TZ”{{参|1={{Cite web|url=http://www.chinanews.com/gn/2013/10-31/5449814.shtml|title=中国载人空间站命名“天宫”货运飞船为“天舟”|publisher=中国新闻网|date=2013-10-31|accessdate=2014-01-16|language=zh-cn}}}}，来源于[[汉语拼音|汉语拼音]]“'''T'''iān '''Z'''hōu”。

== 构造 ==
天舟号货运飞船的构造基于[[天宫一号|天宫一号目标飞行器]]。该货运飞船的[[低地轨道|低地轨道]]上行运载能力约为6.5吨左右，高于[[俄罗斯联邦航天局|俄罗斯联邦航天局]]研制的[[進步號太空船|进步号M型]]（2.60吨）、[[美国|美国]][[軌道科學公司|軌道科學公司]]的[[天鵝座宇宙飛船|天鵝座宇宙飛船]]扩展型（2.70吨）、美国[[太空探索科技公司|太空探索科技公司]]的[[天龙号太空船|天龙号太空船]]扩展型（3.31吨加压/非加压总和）以及[[日本|日本]][[宇宙航空研究開發機構|宇宙航空研究開發機構]]的[[H-II运载飞船|H-II运载飞船]]（6.00吨加压/非加压总和），但低于[[欧洲空间局|欧洲空间局]]已退役的[[自動運載飛船|自動運載飛船]]（7.67吨）。下行运载能力约为6.0吨左右。但天舟只具備加壓艙貨物的運輸能力，而不具備如H-II運載飞船與天龙号太空船可運送非加壓貨物的能力。载荷比即运载货物的质量与货运飞船船体本身的质量之比，天舟号货运飞船的载荷比高达46%，高于日欧的货运飞船。

天舟号货运飞船船长约9米，最大直径约3.35米，发射质量低于13吨。天舟号货运飞船各部位是由大直径的货物舱和小直径的推进舱组成。货物舱用于装载货物，而推进舱为整个飞船提供动力与电力。推进舱两侧各有一翼太阳能帆板三板。货运飞船旨在补给未来[[天宫二号|天宫二号空间实验室]]以及中国空间站的推进剂、空气、航天员的饮料、食物以及用于维修空间站的更换设备，以延长空间实验室和空间站的运行寿命{{参|1={{Cite web|url=http://www.cmse.gov.cn/cmssname/show.php?itemid=9|title=中国货运飞船简介|publisher=中国载人航天工程网|date=2011-04-23|accessdate=2013-06-23|language=zh-cn}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}}}。货运飞船发射后会与空间实验室或空间站进行自动交会对接。为空间实验室或空间站自动补加推进剂以及空气。饮水、食物、以其他器物设备则需驻空间实验室或空间站的航天员从货运飞船上搬运到站内。同时，货运飞船亦可充当空间站的“垃圾桶”，航天员取出货运飞船升空携带的物品后，可将站内的废弃物品搬运到货运飞船上，关闭气闸后，货运飞船脱离空间站，返回地球时废弃物连同货运飞船在大气层中烧毁。这种做法与俄、欧、日、美无人货运飞船的运行做法相同。

== 任务 ==
[[天宫二号|天宫二号空间实验室]]于2016年9月15日由[[长征二号F运载火箭|长征二号F改进型运载火箭]]（天二）从[[甘肅省|甘肅省]][[酒泉卫星发射中心|酒泉卫星发射中心]]发射升空。

于2017年从[[西昌卫星发射中心|西昌卫星发射中心]][[海南文昌航天发射场|海南文昌航天发射场]]使用长征七号运载火箭发射中国新研制的首艘货运飞船。货运飞船的[[低地轨道|低地轨道]]运载能力约为6.5吨左右。与天宫二号自动交会对接成功后，为空间实验室补加推进剂{{参|1={{Cite journal|title=我国空间站总体构想|author=[[周建平|周建平]]|journal=《载人航天》|publisher=中国载人航天工程办公室|volume=第19卷|issue=第二期|page=第6页、7页、9页|language=zh-cn|date=2013年3月}}}}。

2017年9月22日18时左右，天舟一号在完成空间实验室阶段任务及后续拓展试验后受控离轨，再入大气层烧毁。
