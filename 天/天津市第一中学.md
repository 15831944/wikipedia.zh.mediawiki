'''天津市第一中学'''是[[天津市教育委员会|天津市教育委员会]]直属重点高级中学，位于[[天津市|天津市]][[和平区_(天津市)|和平区]]西安道117号。
== 概述 ==

前身是组建于[[1947年9月|1947年9月]]的[[天津市立中学|天津市立中学]]。1949年后，更名为天津市第一中学，简称[[天津一中|天津一中]]。2000年，[[和平中学|和平中学]]整建制并入天津一中。2019年1月，天津市第一中学与[[河北省雄县中学|河北省雄县中学]]签署对口支援协议，成立天津一中雄安校区。

== 校训 ==

坚定正确的政治方向  艰苦奋斗的作风  自理自学的能力

== 办学思想 ==

育人为本，坚持全面发展，培养个性特长，为使学生成为二十一世纪拔尖人才，奠定坚实的基础。 

== 培养目标 ==

具有强烈的历史使命感和社会责任感，不满足优秀，崇尚创新，追求卓越。
