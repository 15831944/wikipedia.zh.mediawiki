'''淘宝网'''是[[中国|中国]][[阿里巴巴集团|阿里巴巴集团]]旗下[[网络购物|网络购物]]网站，由[[马云|马云]]创立于2003年5月10日，是面向[[中国大陆|中国大陆]]、[[香港|香港]]、[[澳门|澳门]]、[[台湾|台湾]]的大中華地區消费者與[[马来西亚|马来西亚]]之[[C2C|C2C]]购物网站（[[B2C|B2C]]模式网站——天猫已拆出），个人或企业均可在淘宝网开设自己的网络[[店铺|店铺]]，此外淘宝网上还拥有[[拍卖|拍卖]]平台。2011年6月16日，网站分拆为淘宝网、[[天猫|天猫]]、[[一淘网|一淘网]]。

2018年1月，淘宝网再度登上美国贸易代表办公室年度“恶名市场”黑名单。对此阿里巴巴集团发表声明表示“基于[[贸易保护主义|贸易保护主义]]抬头，阿里巴巴再次成为美国贸易代表办公室高度政治化环境下的牺牲品。美国贸易代表办公室的恶名市场名单仅仅针对非美国国家，它已不是关乎知识产权保护，而是实现美国政府贸易政策目标的工具，其真实意图终将为人所知<ref>{{Cite web|url = http://www.dw.com/zh/淘宝再上美国黑名单-自称政治牺牲品/a-42136018?&zhongwen=simp|title =淘宝再上美国黑名单 自称政治牺牲品|accessdate =2018-01-13  |author = 德国之声 }}</ref>。”

=== 拍卖平台 ===
2012年6月26日，浙江省[[高级人民法院|高级人民法院]]和淘宝网联合推出的司法拍卖平台正式上线<ref>{{Cite web|url = http://it.sohu.com/20120710/n347751948.shtml|title = 淘宝网免费提供平台 浙江高院试行网上司法拍卖|accessdate = |author = 法制日报|date = 2012年7月10日|publisher = 法制日报}}</ref>。部分[[法院|法院]]将淘宝作为司法拍卖平台<ref>{{Cite web|url =http://www.sc.xinhuanet.com/content/2015-12/11/c_1117425688.htm|title =四川省27家法院试水淘宝网络司法拍卖 法院变卖家|accessdate =|author =四川日报|date =2015-12-11|publisher =新华网|deadurl =yes|archiveurl =https://web.archive.org/web/20160304124306/http://www.sc.xinhuanet.com/content/2015-12/11/c_1117425688.htm|archivedate =2016-03-04}}</ref>，在淘宝上进行司法拍卖不需要收取佣金，透明度高<ref>{{Cite web|url = http://tech.cnr.cn/techgd/20151216/t20151216_520816567.shtml|title = 推行淘宝司法拍卖见成效|accessdate = |author = 光明网|date = 2015-12-16|publisher = 央广网}}</ref>。网络拍卖可以有效降低串标和暴力制止其他买家出价的行为<ref>{{Cite web|url =http://60.216.0.164:99/html/2016-01/04/content_184764.htm|title =一位拍卖师眼中的网络拍卖|accessdate =|author =|date =2016-01-04|publisher =山东商报}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[南通市|南通市]][[中级人民法院|中级人民法院]]公布自2014年年初出台《关于司法公开建设的实施方案》规定拍卖应当通过淘宝到2016年1月1日以来，全市法院共上传拍品1175宗，成交476宗，成交额达11.1余亿元，节约佣金约3900余万<ref>{{Cite web|url = http://csj.xinhuanet.com/2016-01/01/c_134969597.htm|title = 南通法院淘宝网司法拍卖两年成交476宗超11亿元|accessdate = |author = |date = 2016-01-01|publisher = 新华网}}</ref>。另外淘宝网还上线了“不良资产处置平台”拍卖专场<ref>{{Cite web|url = http://www.chinanews.com/fortune/2015/12-15/7672391.shtml|title = 华融资产515亿元不良资产上线淘宝拍卖|accessdate = |author = |date = 2015年12月15日|publisher = 中国新闻网}}</ref>。截至2014年8月13日，淘宝上超过500家法院，10余家产权交易所，200家国内外拍卖机构、画廊、艺术商店入驻淘宝拍卖平台<ref>{{Cite web|url =http://szsb.sznews.com/html/2014-08/13/content_2970435.htm|title =淘宝拍卖会 成国内最大在线拍卖|accessdate =|author =|date =2014年8月13日|publisher =深圳商报|deadurl =yes|archiveurl =https://web.archive.org/web/20170417160538/http://szsb.sznews.com/html/2014-08/13/content_2970435.htm|archivedate =2017年4月17日|df =}}</ref>。

== 影响 ==

=== 经济发展 ===
[[File:淘宝大学_20150122_102134.jpg|thumb]]淘宝网提供了大量创业的机会，为大量中小型企业和个人减少了创业成本<ref>{{Cite web|url = http://news.xinhuanet.com/eworld/2010-06/21/c_12241454.htm|title = 淘宝网：看不见的“远征军”|accessdate = |author = 南方都市报|date = 2010年6月21日|publisher = 新华网}}</ref>，截止2011年年底，淘宝创造了270万就业机会<ref name=":17" />。在[[山东省|山东省]][[滨州市|滨州市]][[博兴县|博兴县]][[湾头村|湾头村]]，有1617多户<ref>{{Cite web|url = http://www.nbd.com.cn/articles/2013-11-22/789802/print|title = “淘宝网”的诱惑|accessdate = |author = |date = 2013-11-22|publisher = 每日经济新闻}}</ref>，500多户居民在淘宝开设网店，根据阿里巴巴集团的调查，在中国有14个类似于湾头村的“淘宝村”，其网店达到1万家，拉动就业人数4万人<ref>{{Cite web|url = http://news.ifeng.com/shendu/zgxwzk/detail_2013_09/03/29268561_0.shtml|title = 淘宝下乡|accessdate = |author = 中国新闻周刊|date = 2013年9月3日|publisher = 凤凰网}}</ref>。

=== 购物节 ===
2005年成交金额破80亿元，超越沃尔玛。
2007年，淘宝的交易额实现了433亿元，比2006年增长156%。
2008年上半年，淘宝成交额就已达到413亿元。
2009年全年交易额达到2083亿人民币。
2009年双11，淘宝+天猫销售额0.5亿元。
2010年双11，淘宝+天猫销售额9.36亿元。
2011年双11，天猫双11销售额33.6亿元，淘宝和天猫共52亿。 [10] 
2012年双11，截至上午8点16分，开始8个小时的天猫双11购物节，支付宝交易额已经达到50亿元，接近2011年双11全天淘宝和天猫的总交易额。双11全天，淘宝天猫平台交易金额已经达到191亿元。
2013年双11，开场仅1分钟成交的订单数量达到33.9万笔，总成交金额达到1.17亿元。第二分钟，成交数字突破3.7亿元。到了零时6分7秒，成交额直接冲上10亿元。截至11日24时，“双11”天猫及淘宝的总成交额破300亿元，达350.19亿元。
2014年双11，淘宝+天猫成交额再次刷新记录，达到571亿元。
2015年双11，淘宝+天猫成交额再次刷新记录，达到912亿元 [11]  。
2016年双11，淘宝+天猫成交额再次刷新记录，达到1207亿元，15个小时天猫销售额达到去年912亿元的销售总额，线上占比为82%。
2017年双11，淘宝+天猫成交额再次刷新记录，达到1682亿元，无线成交额占比90%。全球消费者通过支付宝完成的支付总笔数达14.8亿笔，比2016年增长41%。截至24点，全球有225个国家和地区加入2017天猫双11全球狂欢节。 [10]
