《'''消失'''》，是[[倪匡|倪匡]]筆下[[科幻小說|科幻小說]][[衞斯理系列|衞斯理系列]]之一，系列編號19，連載於1969年9月16日至1969年10月15日的《[[明報|明報]]》<ref>[http://yehleehwa.net/nni17.htm 衛斯理故事年表]</ref>，作品與《[[貝殼_(小說)|貝殼]]》以《貝殼》之名共同出版。
故事描述接二連三的消失案件，最後[[衛斯理|衛斯理]]也消失，卻發現原來是外星人在搜集標本。

== 故事概述 ==

[[衛斯理|衛斯理]]參加余全祥和雲妮的[[婚禮|婚禮]]，在他想回家的前一天，余全祥打[[電話|電話]]要求衛斯理到他的[[蜜月|蜜月]]地點——一幢海邊別墅，因為發生了意外。衛斯理到達後，得知雲妮在浴室放水進[[浴缸|浴缸]]時，突然消失了。但是從四周的環境顯示這是不可能<ref name="新娘突然不見了">《貝殼》 消失 第一部：新娘突然不見了 [[倪匡|倪匡]]著</ref>。

衛斯理發現[[水龍頭|水龍頭]]關掉了，余全祥否認他關了水龍頭，衛斯理認為這是一個重要的關鍵。衛斯理從窗外看見一塊凸出的巖石上有[[綠|綠]]光閃了兩下，並發現雲妮站在那巖石之上。當他倆下去巖石後，發覺一個人也沒有，卻聞到一股很濃的焦味，並發現大片草木被重物壓過<ref name="新娘突然不見了"/><ref name="新郎也失蹤了">《貝殼》 消失 第二部：新郎也失蹤了 [[倪匡|倪匡]]著</ref>。

[[警方|警方]]來到卻毫無發現，在警方離去後衛斯理回到屋子，發現余全祥也消失了。通知警方後，[[異種情報處理局|異種情報處理局]]的賓納來到。藉著[[儀器|儀器]]發現屋內有過量的[[輻射|輻射]]能<ref name="新郎也失蹤了"/><ref name="我也失蹤了">《貝殼》 消失 第三部：我也失蹤了 [[倪匡|倪匡]]著</ref>。

因消失的地點是在浴室，所以衛斯理留在浴室等待消失。突然水龍頭的水流停止了，綠光閃過，衛斯理被轉移到另一個地方再會余全祥和雲妮。衛斯理視察四周，感覺在一隻方形的大盒子中，他們三人就是[[標本|標本]]，被[[外星人|外星人]]搜集來作研究之用，而這種外星人的科技十分發達<ref name="我也失蹤了"/><ref name="在一隻大盒子中">《貝殼》 消失 第四部：在一隻大盒子中 [[倪匡|倪匡]]著</ref>。

一個[[日本|日本]]女子正村薰子忽然出現，她指自己是在[[長崎|長崎]][[原子彈|原子彈]][[爆炸|爆炸]]時被外星人救起並來到這艘[[太空船|太空船]]。衛斯理向正村薰子表示他們想回[[地球|地球]]。突然，正村薰子、余全祥、雲妮都消失了。他發現太空船外有四個圓盤飛來，圓盤頂有多隻[[雞蛋|雞蛋]]似的[[生物|生物]]出來。然後，綠光又再閃過，衛斯理再次被轉移，回到地球。但余全祥和雲妮卻從此消失了<ref name="在一隻大盒子中"/><ref name="雞蛋一樣的生物">《貝殼》 消失 第五部：雞蛋一樣的生物 [[倪匡|倪匡]]著</ref>。

== 故事角色 ==

*'''衛斯理'''
書中主角。故事中參與余全祥和雲妮的婚禮，在余全祥和雲妮被外星人擄走時作出調查，其後也被外星人擄走。在太空船上見識到外星人的面貌，最後被送回地球。

*'''余全祥'''
[[工程學博士|工程學博士]]，其父執全死在戰場，因其監護人和老蔡是同鄉所以和衛斯理認識。在結婚沒多久後被外星人擄走。

*'''雲妮'''
余全祥的新婚妻子，在結婚沒多久後被外星人擄走。

*'''賓納'''
異種情報處理局的助手，曾在余全祥和雲妮被外星人擄走時協助調查。

*'''正村薰子'''
日本人，在長崎原子彈爆炸時被外星人救走，在太空生活十多年，對地球感到厭惡，希望能移民外星。在衛斯理被擄走後出現解釋一切。

*'''「雞蛋」外星人'''
形狀像雞蛋，只有拳頭般大小，乳白色。身上有很多細絲，功能像人的雙手。利用身上的斑點作溝通。在太空某處有一個工作站。因要研究地球人而將四個地球人擄走。
