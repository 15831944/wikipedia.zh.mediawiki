{{日本作品|消滅都市|消滅都市（しょうめつとし）}}是由[[GREE|GREE]]旗下子公司Wright Flyer Studios所開發的[[手機遊戲|手機遊戲]]，2014年5月26日在日本iOS、Android平台同時發佈。遊戲先後在2016年11月17日及2018年11月25日改版為《消滅都市2》及《消滅都市0.》。

中國大陸版於2015年11月12日發佈。

繁體中文版由[[戰谷|戰谷]]取得代理權，並於2016年6月1日發佈<ref>{{Cite news |url=http://www.chinatimes.com/realtimenews/20160628003268-260412 |title=解開《消滅都市》消逝之謎！遊戲體驗3D化 如臨現場的實境感受 |work=中時電子報 |date=2016-06-28}}</ref>，於2017年9月19日改版為《消滅都市2》。其後在2018年3月30日結束營運。<ref>{{Cite news |url=https://newtalk.tw/news/view/2018-01-31/112637 |title=震驚！知名跑酷手遊《消滅都市2》停止營運 |work=新頭殼 |date=2018-01-31}}</ref>。

2018年5月27日，日本官方宣布與動畫公司[[MADHOUSE|MADHOUSE]]合作動畫化以及新開發TV掌機遊戲衍伸作品。

== 劇情簡介 ==
3年前，一座都市被『消滅』了！

3年後，拓也接到委託，和神秘少女小雪前往人稱『失落之地』的消滅都市展開意想不到的旅程……

== 角色介紹 ==
===主要人物===
;{{nihongo|拓也|タクヤ|聲：[[杉田智和|杉田智和]]}}
: 原是普通的送貨員，後因接受委託而從謎之組織手中救出被囚禁的神秘少女小雪，兩人就此展開了一連串的冒險旅程......

;{{nihongo|小雪|ユキ|聲：[[花澤香菜|花澤香菜]]}}
: 被拓也從組織手中救出的神秘少女，擁有召喚'''魂體'''的能力，父母捲入消滅事件而下落不明，為了尋找父親而向拓也提出了委託...

;{{nihongo|調查員|リサーチャー(動畫版名字：ユミコ)|聲：[[中惠光城|中惠光城]]}}

;{{nihongo|仲介商人|ヘッドハンター|聲：[[福田賢二|福田賢二]]}}
: 靠著買賣個人資料賺錢的商人，很有生意頭腦，賣給拓也和小雪消滅都市的人口資料，使得小雪可以'''召喚他們'''。

;{{nihongo|蒼真|ソウマ|聲：少年 - [[朝井彩加|朝井彩加]]、青年 - [[入野自由|入野自由]]}}

: 為謎之組織工作，女科學家，曾經是小雪爸爸的同事，在都市消滅時失蹤，個性也變的和之前有些不同。

;{{nihongo|太陽|タイヨウ|聲：[[高橋英則|高橋英則]]}}
: 男科學家，曾經也是小雪爸爸的同事，在消滅後為謎之組織工作，進行許多魂體實驗，目的是讓魂體的力量能被加以利用。

;{{nihongo|銀河|ギンが|聲：[[拝真之介|拝真之介]]}}
: 擁有操作'''波動物質'''的能力，在謎之組織地位似乎很崇高。

;{{nihongo|麗莎|リサ|聲：[[淵上舞|淵上舞]]}}
: 偵探組織的成員，以前拓也的上司。

;{{nihongo|浩太|コウタ|聲：[[高橋信|高橋信]]}}
: 偵探組織成員，麗莎的手下。

;{{nihongo|宇宙||聲：[[山本彩乃|山本彩乃]]}}
: 目前只在謎之組織的對話中出現過的人物，擁有超越想像力的強大力量。

;{{nihongo|明|アキラ|聲：[[中村悠一|中村悠一]]}}

;{{nihongo|桔梗|キキョウ|聲：[[愛美|愛美]]}}

;{{nihongo|菘|スズナ|聲：[[佐倉綾音|佐倉綾音]]}}
: 於'''第二次消滅'''登場的謎之組織裡的神秘少女，和小雪同齡個性卻冰冷無情，似乎認識桔梗且十分恨她。

== 用語 ==
;{{nihongo|失落之地|ロスト}}
:三年前被消滅的都市，後來被稱「失落之地」，凡是進入過的人都會產生如人格分裂、失憶、精神障礙等精神疾病。事實上失落之地是謎之組織打開連接平行世界的黑洞。
;波動性的物質
:失落之地裡充滿的「物質」，使人精神變不正常的元兇，據研究員所說其實波動性的物質並不存在，是為了方便而這樣稱呼它。事實上為來自各平行世界的記憶，當人靠近時變會『倒灌』在人身上。
;{{nihongo|魂珠|スフィア}}
:雪的力量並不完全，需要靠拓也（玩家）在路上收集魂珠才能攻擊敵人。
;魂體
:被消滅所波及死亡的人，一般來說對人世間有留念、遺憾，事實上是因謎之組織打開平行世界黑洞，人們死亡，所產生的記憶實體化現象。
;謎之組織
:一切都是謎的組織，開啟失落之地的原因似乎和自己的世界所發生的「第三次大戰」有關。
;偵探組織
:拓也和調查員曾待過的組織，據拓也描述待在裡面有些「不得不的事情」。
;觀測者
:謎之組織偶爾會提到的名詞，但是指什麼便不得而知。
