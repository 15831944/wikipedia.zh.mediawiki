'''消防员学校'''，是[[应急管理部消防救援局|应急管理部消防救援局]]直属的一所专门消防救援院校，位于[[江苏省|江苏省]][[南京市|南京市]]<ref name=ngd>{{Cite web |url = http://jiuye.njtech.edu.cn/campus/view/id/588581 |title = 公安消防部队士官学校 |author =  |date =  2017-01-11|publisher =南京工业大学就业创业网  |language =  |accessdate =  |quote =  }}</ref>。

==沿革==
1982年6月，经[[江苏省人民政府|江苏省人民政府]]批准，成立'''江苏省消防教导大队'''，主要承担江苏消防部队预提警官培训、新兵集训和干部轮训任务<ref name=gfb/>。

1991年7月，经[[公安部政治部|公安部政治部]]批准，在江苏省消防教导大队的基础上，成立'''南京消防学校'''（学校和教导大队实行[[两块牌子一套人马|两块牌子一套人马]]，并延续至2001年6月），培养部队生长干部<ref name=gfb/>。

1992年3月，南京消防学校更名为'''公安消防部队南京指挥学校'''<ref name=gfb/>。

2001年7月，经公安部政治部批准，南京消防指挥学校更名为'''公安消防部队南京士官学校'''，并继续保留'''公安消防部队南京指挥学校'''的名称，承担全国消防部队士官骨干培训工作。该校位于[[南京市|南京市]][[江宁区|江宁区]][[汤山街道|汤山街道]]古泉社区，隶属[[公安部消防局|公安部消防局]]，由[[江苏省公安厅|江苏省公安厅]]、江苏省消防总队具体领导管理<ref name=gfb>{{Cite web |url =http://www.mod.gov.cn/service/2009-09/09/content_4086555.htm  |title = 南京消防士官学校 |author =  |date =2009-09-09  |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。

后更名为'''公安消防部队士官学校'''，隶属公安部消防局，仍位于江苏南京，为正师级建制<ref name=ngd/><ref>{{Cite web |url = http://xf119.cpd.com.cn/n2634881/n2635210/c23428725/content.html |title = 江西举行2014年度公安消防部队士官学校招生统考 |author =  |date =2014-06-10  |publisher = 中国警察网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://119.china.com.cn/zbjs/txt/2015-06/09/content_7976985.htm |title =  山东消防总队完成2015年士官学校招生统考工作|author =  |date =  2015-06-09 |publisher =中国消防在线  |language =  |accessdate =  |quote =  }}</ref>。

2018年10月9日，[[公安消防部队|公安消防部队]]移交应急管理部交接仪式举行<ref name="报道7"/>。机构改革后，消防部队现役编制全部转为行政编制，成建制划归应急管理部<ref name="报道7">{{cite web|author1=华商报|title=消防部队退役|url=http://news.sina.com.cn/o/2018-10-10/doc-ihkvrhpt4120472.shtml|website=新浪网新闻|accessdate=2018-10-10|location=北京|language=zh-cn|df=}}</ref>。10月10日，应急管理部决定，自2018年10月10日零时起，至国家综合性消防救援队伍制式服装配发前，原公安消防部队、武警森林部队和警种学院人员停止使用武警部队制式服装和标识服饰，统一穿着无武警标识的作训服，并在作训服左兜盖上方佩带消防救援队伍身份标识牌<ref name="报道8">{{cite web|author1=李强|title=应急管理部发布国家综合性消防救援队伍改革过渡期身份标识牌|url=http://www.chinasafety.gov.cn/xw/bbgz/201810/t20181010_221092.shtml|website=中华人民共和国应急管理部门户网站|accessdate=2018-10-10|location=北京|language=zh-cn|df=}}</ref>。其中，'''消防员学校'''学员佩戴队员标识牌<ref name="报道8"/>。
