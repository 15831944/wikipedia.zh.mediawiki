'''消光'''（{{lang|en|Extinction}}）是[[天文學|天文學]]中[[觀測者|觀測者]]用來描述被觀測的[[天體|天體]]發射的[[电磁辐射|电磁辐射]]被路途中的物質（[[氣體|氣體]]和[[塵埃|塵埃]]）[[吸收|吸收]]和[[散射|散射]]的过程。對地面的觀測者而言，消光來自於[[星際物質|星際物質]]（ISM）和[[地球大氣層|地球大氣層]]，它也可能來自於被觀測天體周圍的[[星周尘埃|星周尘]]。大氣層的消光在一些波段（[[X射線|X射線]]、[[紫外線|紫外線]]和[[紅外線|紅外線]]）上非常強烈，必須進入太空才能觀測。在可見光的波段上，由于较短的波长被吸收散射更加严重，[[藍色|藍色]]遠比[[紅色|紅色]]被稀釋的強烈，結果是天體會比預期的偏紅，星際消光也會使天體'''紅化''' （不要與[[紅移|紅移]]混淆）。

== 一般特徵 ==
就廣義來說，星際消光對波長的影響是越短的波長消光越嚴重。一般用R(V)描述消光的大小，參數R(V)等於A(V)/E(B-V)和A(V)，而E(B-V)为B波段和V波段的[[色余|色余]]，其值为观测[[色指数|色指数]]减去内禀[[色指数|色指数]]。
:<math>E_{B-V} = (B-V)_{\text{Observed}} - (B-V)_{\text{Intrinsic}}</math>

== 監測天體來測量消光 ==

要測量一顆[[恆星|恆星]]的消光曲線，必須將他的光譜與一顆光譜類似且不受消光影響（沒有紅化）的恆星做比較。<ref>{{cite journal

== 大氣消光 ==
大氣消光的點隨高度角而變化。天文觀測場所一般都能非常精準的描繪當地的消光曲線，使觀測能做有效的改正。不過，大氣對許多波長是完全不透明的，這就必須利用[[人造衛星|人造衛星]]來進行觀測。
大氣消光有三種主要的成分：氣體分子的[[瑞利散射|瑞利散射]]、[[粒子|懸浮微粒]]的散射和分子的吸收。分子吸收通常被稱為[[大氣污染|大氣吸收]]，是由[[地球|地球]]自身造成的。最主要的大氣吸收源就是[[氧|氧分子]]和[[臭氧|臭氧]]，在近[[紫外線|紫外線]]造成強烈的吸收；還有[[水|水]]在[[紅外線|紅外線]]的強烈吸收。
相當數量的大氣消光與物體的[[高度|高度]]有關，在[[天頂|天頂]]的量最小，而在接近[[地平|地平]]的量最大。它是以標準的大氣消光曲線與觀測期間平均[[氣團|氣團]]的乘積來計算的。
