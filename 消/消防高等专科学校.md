'''消防高等专科学校'''，是[[应急管理部消防救援局|应急管理部消防救援局]]直属的一所专门消防救援学校，校址位于[[云南省|云南省]][[昆明市|昆明市]]<ref name=ngd>{{Cite web |url = http://jiuye.njtech.edu.cn/campus/view/id/588581 |title = 公安消防部队士官学校 |author =  |date =  2017-01-11|publisher =南京工业大学就业创业网  |language =  |accessdate =  |quote =  }}</ref>。

==沿革==
1991年7月13日，经[[中华人民共和国公安部|公安部]]批准，成立'''公安消防部队昆明指挥学校'''（简称'''昆明消防指挥学校'''）。建校初期，主要面向西部11个省、自治区、直辖市消防部队招收中专学员<ref name=xfzx>{{Cite web |url =  http://119.china.com.cn/xfyx/txt/2007-05/31/content_3681409.htm|title =  公安消防部队昆明指挥学校简介|author =  |date =2007-05-31   |publisher =中国消防在线  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://news.yntv.cn/content/15/20110709/173518_15_333813.shtml  |title = 公安消防部队昆明指挥学校庆祝建校20周年 |author =  |date =2011-07-09  |publisher =  云视网|language =  |accessdate =  |quote =  }}</ref>。

2000年起，根据[[公安部政治部|公安部政治部]]招生计划，学校开始面向全国消防部队招生。2002年开始招收消防指挥专业大专学员。是当时[[中华人民共和国|中华人民共和国]]唯一一所面向全国消防部队招生，专门培养基层消防指挥干部的学校<ref name=xfzx/>。

2015年4月28日，[[中华人民共和国教育部|教育部]]发出《教育部关于同意在公安消防部队昆明指挥学校基础上建立公安消防部队高等专科学校的函》（教发函〔2015〕76号），同意公安部《关于申请组建公安消防部队高等专科学校的函》（公政治〔2014〕408号）中提出的在公安消防部队昆明指挥学校基础上建立公安消防部队高等专科学校。教育部在该函中指出，“公安消防部队高等专科学校系专科层次的普通高校，学校应逐步过渡到以实施专科教育为主”<ref>{{Cite web |url =  http://www.119.gov.cn/xiaofang/xfyw/30821.htm|title =教育部批复同意组建公安消防部队高等专科学校  |author =  |date = 2015-05-12 |publisher =公安部消防局  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://old.moe.gov.cn//publicfiles/business/htmlfiles/moe/s181/201506/188359.html |title = 教育部关于同意在公安消防部队昆明指挥学校基础上建立公安消防部队高等专科学校的函 |author =  |date = 2015-04-28 |publisher = 中华人民共和国教育部 |language =  |accessdate =  |quote =  }}</ref>。

2015年10月15日，'''公安消防部队高等专科学校'''在昆明正式揭牌。新成立的该校设有消防指挥、抢险救援等专业，重点培养优秀消防专业人才和开展消防领域的理论研究、科技创新研究。公安部政治部现役工作办公室主任[[牟玉昌|牟玉昌]]、公安部消防局政委[[王沁林|王沁林]]参加了成立仪式并揭牌<ref name=jiepai>{{Cite web |url =http://119.china.com.cn/jdxw/txt/2015-10/16/content_8300199.htm  |title =公安消防部队高等专科学校在昆明揭牌成立  |author =  |date = 2015-10-16 |publisher =中国消防在线   |language =  |accessdate =  |quote =  }}</ref>。

2018年10月9日，[[公安消防部队|公安消防部队]]移交应急管理部交接仪式举行<ref name="报道7"/>。机构改革后，消防部队现役编制全部转为行政编制，成建制划归应急管理部<ref name="报道7">{{cite web|author1=华商报|title=消防部队退役|url=http://news.sina.com.cn/o/2018-10-10/doc-ihkvrhpt4120472.shtml|website=新浪网新闻|accessdate=2018-10-10|location=北京|language=zh-cn|df=}}</ref>。10月10日，应急管理部决定，自2018年10月10日零时起，至国家综合性消防救援队伍制式服装配发前，原公安消防部队、武警森林部队和警种学院人员停止使用武警部队制式服装和标识服饰，统一穿着无武警标识的作训服，并在作训服左兜盖上方佩带消防救援队伍身份标识牌<ref name="报道8">{{cite web|author1=李强|title=应急管理部发布国家综合性消防救援队伍改革过渡期身份标识牌|url=http://www.chinasafety.gov.cn/xw/bbgz/201810/t20181010_221092.shtml|website=中华人民共和国应急管理部门户网站|accessdate=2018-10-10|location=北京|language=zh-cn|df=}}</ref>。其中，消防高等专科学校学员佩戴干部标识牌<ref name="报道8"/>。

2018年11月16日，该校召开学习宣贯习总书记授旗训词动员大会，[[应急管理部消防救援局|应急管理部消防救援局]]授予'''消防高等专科学校'''[[中国消防救援队队旗|中国消防救援队队旗]]，学校全体指战员面向队旗宣读中国消防救援队誓词<ref name="报道9">{{cite web|author1=消防高等专科学校办公室|title=牢记训词 践行精神--学校召开学习宣贯习总书记授旗训词动员大会|url=http://www.xfgdzkxx.cn/index.php/Show/index/cid/5/id/828.html|website=消防高等专科学校|accessdate=2018-10-10|location=北京|language=zh-cn|df=}}</ref>。
