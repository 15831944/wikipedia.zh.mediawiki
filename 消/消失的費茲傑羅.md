《'''消失的費茲傑羅'''》（{{lang-en|'''''Camino Island'''''}}）是一本由[[約翰·葛里遜|約翰·葛里遜]]所寫的[[犯罪小說|犯罪]][[驚悚|驚悚]]小說，2017年6月6日由{{link-en|Doubleday (publisher)|Doubleday (publisher)|Doubleday}}公司負責出版。這本書背離了葛里遜主要的法律驚悚主題，並集中講述被偷的珍貴書籍。葛里遜在25年內首次進行廣泛的書籍巡迴演講來宣傳該書。<ref>{{Cite web |url = https://www.usatoday.com/story/life/nation-now/2017/04/19/john-grisham-camino-island/100630236/ |title = John Grisham ready for 1st big book tour in 25 years |author = Jerry Mitchell |date = 2017-04-19 |publisher = USA Today |language = en}}</ref>

== 情節 ==
本書開始於[[普林斯頓大學|普林斯頓大學]][[普林斯頓大學圖書館|費爾斯通圖書館]]中被偷走的五部珍貴的[[法蘭西斯·史考特·費茲傑羅|法蘭西斯·史考特·費茲傑羅]]手稿，然後前往佛羅里達島上的一個度假小鎮，尋找有關盜竊案的線索。<ref>{{Cite web |url = https://www.nytimes.com/2017/06/12/books/review/camino-island-john-grisham.html |title = In ‘Camino Island,’ John Grisham Takes a Vacation From Writing John Grisham Novels |author = Ken Tucker |date = 2017-06-12 |publisher = The New York Times |language = en}}</ref>儘管[[聯邦調查局|聯邦調查局]]和「地下機構」的調查人員幫普林斯頓保險公司追捕在[[黑市|黑市]]中的犯罪者，但故事卻集中在一位追查盜竊犯的小說家身上。<ref>{{Cite web |url = http://ew.com/books/2017/02/07/john-grisham-camino-island/ |title = John Grisham announces new summer thriller, Camino Island — exclusive |author = Isabella Biedenharn |date = 2017-02-17 |publisher = Entertainment Weekly |language = en}}</ref><ref>{{Cite web |url = https://www.usatoday.com/story/life/books/2017/05/28/john-grisham-camino-island-book-review/101849544/ |title = 'Camino Island': Grisham's tasty thriller about a literary heist |author = Jocelyn McClurg |date = 2017-05-28 |publisher = USA Today |language = en}}</ref>
