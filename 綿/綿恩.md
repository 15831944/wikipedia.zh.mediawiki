'''綿恩'''（{{lang-mnc|ᠮᡳᠶᠠᠨ ᡝᠨ|v=miyan en}}；{{bd|1747年|9月18日|1822年|7月18日|catIdx=A}}），定恭亲王，[[乾隆帝|乾隆帝]]孙子，[[永璜|定安親王永璜]]次子。

== 生平 ==

[[乾隆|乾隆]]十二年八月十四日辰時生，生母是永璜[[福晋|側福晉]]伊爾根覺羅氏，七品官[[明泰|明泰]]之女。綿恩聰敏恭謹，頗得乾隆帝喜愛。乾隆四十一年，其兄[[綿德|綿德]]犯事被奪爵，定[[郡王|郡王]]一職由綿恩補上。乾隆五十八年，進封[[親王|親王]]。[[道光|道光]]二年，定亲王逝世，[[道光帝|道光帝]]賜喪銀五千兩，謚曰「恭」。

据军机章京赵翼记载“一日至张三营行宫，上坐较射，皇子皇孙以次射。皇次孙绵恩方八岁，亦以小弓箭一发中的，再发再中。上大喜，谕令再中一矢赏黄马褂。果又中一矢，辄收弓跪而不言。上大笑，趣以黄马褂衣之。仓促间不得小褂，则以大者裹之，抱而去。”由此可知绵恩受赐[[黄马褂|黄马褂]]要比诸如绵庆、绵宁等孙辈早得多。绵恩是最早晋封亲王爵位的绵字辈皇孙第一人。据[[禮親王|礼亲王]][[昭槤|昭梿]]《啸亭续录》记载，定恭亲王绵恩“弱冠即领火器营总统，凡50余年。”嘉庆八年（1803年），皇宫神武门内突发失业差役[[陳德_(清朝)|陈德]]谋刺嘉庆帝案，绵恩因护驾擒凶有功，赏穿御用补褂，并加封其子奕绍为[[固山贝子|贝子]]。
== 皇位继承传说 ==
绵恩是“长房长子次孙”，但乾隆帝实际上是把他当长孙看的。乾隆帝对长子有一份歉疚，因此将孙子封为亲王，封亲王的时候乾隆帝的儿子还没有全部封王。绵恩身材魁梧，仪表出众，武功超群，长期担任京师禁卫军的领导，负担保卫北京的重担。当时，[[朝鲜王朝|朝鲜王朝]]使臣来访回国后认为，清朝皇帝继承人将在十五子永琰和皇次孙绵恩之间产生，绵恩的可能性更大。绵恩比他的叔叔[[嘉庆帝|永琰]]还大十四岁。但是乾隆帝曾批评过明太祖[[朱元璋|朱元璋]]将皇位传给孙子，埋下了后来骨肉相残的祸根。<ref>{{Cite web|url=http://history.people.com.cn/GB/205396/14111848.html|title=乾隆为何选择在执政六十年时禅让皇位？|accessdate=2018-06-30|publisher=人民网}}</ref>

== 家庭 ==

[[福晋|嫡福晋]]富察氏，[[副都统|副都统]][[福敬|福敬]]之女；側福晉尤佳氏，[[員外郎|員外郎]][[長齡|長齡]]之女；側福晉李氏，[[佛保|佛保]]之女；側福晉完颜氏，[[筆帖式|筆帖式]][[慶恕|慶恩]]之女。

四子：第一子未命名，第二子和碩定端親王[[奕紹|奕紹]]，（嫡福晉伊爾根覺羅氏，主事[[太倫|太倫]]女；繼福晉楊佳氏，主事[[楊奎|楊奎]]之女），第三子未命名，第四子奕(糸支)。
