'''傑·弗里曼'''（{{lang-en|Jay Freeman}}，{{bd|1981年|11月27日}}）是一名[[美國|美國]]商人及軟體工程師。他因為開發[[Cydia|Cydia]]應用程式及相關[[iOS越獄|iOS越獄]]軟體，使[[蘋果公司|蘋果公司]]的[[iOS|iOS]]能夠安裝並自訂在[[App_Store_(iOS)|App Store]]系統規範之外的軟體而著名。

== 教育及早期工作 ==
費里曼自[[加利福尼亞大學聖塔芭芭拉分校|加利福尼亞大學聖塔芭芭拉分校]]{{tsl|en|College of Creative Studies|創意研究學院}}畢業，取得[[電腦科學|電腦科學]]學位。<ref>{{cite news |last= Crook |first= Evan |date= 2013-05-07 |title= UCSB Alumnus Among the First to Hack Google Glass |url= http://dailynexus.com/2013-05-07/ucsb-alumnus-among-the-first-to-hack-google-glass/comment-page-1/ |newspaper= The Daily Nexus |location=  |accessdate= 2014-10-08 |deadurl= no |archiveurl= https://web.archive.org/web/20141018191314/http://dailynexus.com/2013-05-07/ucsb-alumnus-among-the-first-to-hack-google-glass/comment-page-1/ |archivedate= 2014-10-18 }}</ref>2009年，他在該校取得電腦科學博士學位。<ref>{{cite news |last= Kane |first= Yukari Iwatani |date= 2009-03-06 |title= Breaking Apple's Grip on the iPhone |url= https://online.wsj.com/articles/SB123629876097346481 |newspaper= Wall Street Journal |location=  |accessdate= 2014-10-08 |deadurl= no |archiveurl= https://web.archive.org/web/20141017040018/http://online.wsj.com/articles/SB123629876097346481 |archivedate= 2014-10-17 }}</ref>

費里曼在2002年建構一個[[.NET框架|.NET]][[反編譯器|反編譯器]]Anakrino。<ref>{{cite web |url= http://oreilly.com/pub/a/dotnet/2002/01/21/vbnet.html |title= Top Five Visual Basic .NET Tips |last1= Grundgeiger |first1= Dave |last2=  |first2=  |date= 2002-01-21 |website= O'Reilly WindowsDevCenter |publisher= O'Reilly Media, Inc. |accessdate= 2014-10-08 }}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>他在2005年與他人合著發布一篇關於「[[Java|Java]]全局事件規範與監視資料庫」的論文。<ref>{{cite journal |last= Karaorman |first= Murat |last2= Freeman |first2= Jay |date= 2005 |title= jMonitor: Java Runtime Event Specification and Monitoring Library |url= http://dl.acm.org/citation.cfm?id=1705799 |journal= Electronic Notes in Theoretical Computer Science |publisher= Elsevier |volume= 113 |issue= |pages= 181–200 |doi= 10.1016/j.entcs.2004.01.027 |access-date=2015-05-05}}</ref>他與[[布萊恩·福克斯|布萊恩·福克斯]]共同執行多項自由業工作。<ref>{{Cite news|url = http://www.linuxinsider.com/story/62209.html|title = Android, Schmandroid: Linux on the iPhone|last = Adhikari|first = Richard|date = 2008-03-20|work = Linux Insider|access-date = 2015-07-19|via = |deadurl = no|archiveurl = https://web.archive.org/web/20160205122551/http://www.linuxinsider.com/story/62209.html|archivedate = 2016-02-05}}</ref>
