'''傑克·羅斯福·羅賓森'''（{{lang-en|'''Jack Roosevelt Robinson'''}}，{{bd|1919年|1月31日|1972年|10月24日|catIdx=Robinson}}），暱稱'''傑基·羅賓森'''（{{lang|en|Jackie Robinson}}），是美國職棒大聯盟現代史上第一位[[非裔美國人|非裔美國人]]（[[美國黑人|美國黑人]]）球員，在1947年4月15日羅賓森穿著42號球衣以先發[[一壘手|一壘手]]的身分代表[[洛杉磯道奇|布魯克林道奇]]隊上場比賽，之前黑人球員只獲准在[[黑人聯盟|黑人聯盟]]打球。雖然美國[[種族隔離|種族隔離]][[政策|政策]]廢除已久，但無所不在的[[種族主義|種族偏見]]仍強烈左右著社會各個階層。因此傑基羅賓森踏上大聯盟舞台被公認為近代[[美國民權運動|美國民權運動]]最重要的事件之一。

== 傳奇球星 ==
傑基·羅賓森是一個極佳的[[運動員|運動員]]，在[[大學|大學]]時曾參與[[棒球|棒球]]、[[美式足球|美式足球]]、[[籃球|籃球]]和[[田徑|田徑]]比賽，也積極爭取市民權利並曾在軍中服役。1945年10月23日，羅賓森正式與道奇隊簽約，利用待在[[小聯盟|小聯盟]]的一年，努力加強其棒球技巧，以待登上大聯盟。

當布魯克林道奇隊的總經理布蘭奇·瑞基（{{lang|en|Branch Rickey}}）在大聯盟主席[[哈皮·钱德勒|哈皮·钱德勒]]首肯下力抗其他15支球團，積極尋覓黑人球員。與羅賓森簽署契約並將其帶入大聯盟後，黑白球員分隔的棒球政策將永遠改寫，將此舉動稱為棒球的最佳實驗的瑞奇知道，選擇跨越膚色界線的球員必須是一個經得起大眾嚴密監視，及在遭遇污辱及敵意時，能避免衝突發生的堅強個體。

1947年4月15日，羅賓森穿上了道奇隊制服，背號42號，以先發[[一壘手|一壘手]]的身分登上大聯盟。同年，羅賓森獲得了該年度的[[新人王|新人王]]，並於1949年獲得[[國家聯盟|國家聯盟]]的[[美國職棒大聯盟最有價值球員獎|最有價值球員獎]]，且於1955年，和隊友們一起打敗了死對頭[[紐約洋基|紐約洋基]]隊，贏得[[世界大賽_(棒球)|世界大賽]]。

當時大聯盟仍有球隊不滿黑人球員的加入，[[聖路易紅雀隊|聖路易紅雀隊]]曾揚言若與羅賓森對戰將會罷賽，此時國聯理事長福特佛利克（Ford Frick）與[[大聯盟|大聯盟]]主席錢德勒（Happy Chandler）表示任何罷賽的球員都將受到禁賽的處份，對戰紅雀隊時甚至受到對手針對性的人身攻擊。

1947年時，雖然道奇隊在[[世界大賽|世界大賽]]被洋基隊打敗，但洋基隊史上最偉大的捕手[[尤吉·貝拉|尤吉·貝拉]]曾預言：「道奇隊現在擁有羅賓森，我們以後將與他們有無數次的決戰」，之後道奇隊果然與洋基隊在[[世界大賽|世界大賽]]中對決多次，終於在1955年打敗洋基隊，這不僅是道奇隊在[[世界大賽|世界大賽]]中第一次打敗洋基隊，也是道奇隊成立七十三年來的第一次[[世界大賽|世界大賽]]冠軍，更讓羅賓森完成艱難且完美的[[大聯盟|大聯盟]]之旅。

由於二十八歲才加入大聯盟，傑基·羅賓森待在大聯盟的時間並不長，一共只有九年，於1956年退休。短暫的職業生涯不影響後人對傑基羅賓森的評價。除了贏得[[新人王|新人王]]、MVP、[[世界大賽|世界大賽]]，傑基·羅賓森的生涯平均打擊率是優異的.311，生涯保送數多於[[三振|三振]]數。除了是名優秀的[[打者|打者]]，也是絕佳的防守者，幾乎每個守備位置都能勝任愉快，此外他最膾炙人口的就是[[盜壘#戰術與技巧|盜本壘]]絕技，生涯一共擁有多達十九次的盜本壘成功記錄。

== 不向命運低頭 ==
傑基羅賓森的故事可以當很多人遭遇困境時的一盞明燈，從他小時候父親遺棄家庭，母親做工養育他們兄弟姊妹，到他得到獎學金進入[[加州大學|加州大學]]就學，是該校唯一的棒球、美式足球、籃球、田徑四棲選手。

身為第一個進入美國職棒大聯盟的黑人選手，壓力之大不是一般人可以承受，從隊友對他的杯葛，到不能去白人餐館吃飯，以致隊友同情他，偷偷拿食物給他吃，到南方[[三K黨|三K黨]]揚言要槍殺他，守[[一壘|一壘]]時，對方球員跑上壘時會故意用腳踢傷他，或是投手會故意的對他投頭部觸身球，他都忍下來並用成績表現給世人看，因此逐漸被大部分隊友接受，甚至在對手球隊攻擊他時為他出頭。

從羅賓森加入道奇隊，即將成為大聯盟第一個黑人選手的消息傳出後，道奇隊就有許多人表示要離隊；在他第一次出賽時，警察要將他趕出場；在恐嚇與揚言槍殺他的風聲傳出時，他只能躲到黑人朋友家中；客場比賽時，主場球隊揚言要罷賽，以及面對全場對他叫囂辱罵的球迷，羅賓森還是一再的忍辱負重，並不斷用一支支的[[安打|安打]]和一次次的[[盜壘|盜壘]]來證明一切。

許多南方的白人說這一輩子永遠看不起黑人，但是看了傑基羅賓森在如此壓力下，還能有如此傑出的表現，已徹底的改變了他們對黑人的看法，然而羅賓森的友人說，因為壓力太大，羅賓森才英年早逝，不得不令人遺憾。

== 永遠的42號 ==
{{see also|榮譽背號}}
1962年，傑基羅賓森經引薦進入[[棒球名人堂|棒球名人堂]]；1972年6月4日，道奇球團將他的球衣背號42號退役，代表這件球衣與榮耀，將永遠與傑基羅賓森同在，並紀念當年他在球壇的卓越貢獻。而羅賓森於同年10月，於[[史丹佛|史丹佛]]的自家中逝世。

1997年，適逢傑基羅賓森進入大聯盟五十週年，為了紀念這位偉大球員不凡的意義與卓越的人生，所有大聯盟球隊都將42號球衣退役，以提醒世人珍惜這個[[自由|自由]]平等的年代。因此42號球衣在大聯盟正式全面退役，所有球隊新進選手均不得再使用這個背號，這項職業體壇的空前創舉，正是由羅賓森代表的42號球衣，光榮的首開先例。

基於不溯及既往條款，原有13名選手能繼續擁有42號作為球衣背號直到退休。紐約洋基隊投手[[馬里安諾·李維拉|馬里安諾·李維拉]]（{{lang|en|Mariano Rivera}}）在2013年後成為最後1個42號球衣選手，但隨著馬里安諾·李維拉於2013年的引退，42號成為永久退役的大聯盟球衣背號。<ref>{{cite web|url=http://mag.udn.com/mag/sports/storypage.jsp?f_ART_ID=478011|title=李維拉退休 大聯盟42號成絕響|date=2013-09-24|work=聯合新聞網}}</ref>

2004年，大聯盟將4月15日訂為「羅賓森日（Jackie Robinson Day）」。

從2007年開始，部分選手、教練在羅賓森日改穿42號球衣紀念；2009年起，在羅賓森日，大聯盟每位選手都穿42號球衣。

== 後世影響 ==
1947年，大聯盟的第一個年度[[新人王|新人王]]得主，就是首位黑人選手，道奇隊的傑基羅賓森，1987年是羅賓森進入大聯盟的四十周年，大聯盟為紀念羅賓森非凡的貢獻，把年度[[新人王|新人王]]獎項，命名為「傑基羅賓森獎」。

1997年，羅賓森進入大聯盟的五十周年紀念，所有大聯盟的球隊，在球衣手臂上也都繡上「{{lang|en|Jackie Robinson}} 50」的五十周年的紀念臂章，象徵打破大聯盟黑白藩籬五十周年，並表達對羅賓森的敬意。當時的[[美國總統|美國總統]][[乔治·克林顿|柯林頓]]也親赴球場表彰羅賓森，讚揚其對[[美國|美國]]體育與[[民權運動|民權運動]]的貢獻，並由羅賓森的遺孀代表接受。

從2004年開始，[[大聯盟|大聯盟]]將4月15日定為「傑基羅賓森日」。

2007年4月15日，在[[小葛瑞菲|小葛瑞菲]]的奔走與建議下，為了慶祝羅賓森進入大聯盟六十週年，大聯盟官方特地選在羅賓森大聯盟出賽一甲子的這一天，讓大聯盟每支球隊的數名球員或教練改穿42號球衣出賽，以紀念與表彰當年羅賓森的勇氣與壯舉，其中道奇隊、[[大都會隊|大都會隊]]、太空人隊、[[費城人隊|費城人隊]]、紅雀隊、[[釀酒人隊|釀酒人隊]]則是全隊都換穿42號球衣，追憶這位道奇隊出身的大聯盟偉大球星。

2008年，全隊換穿42號球衣的球隊更包括了大都會隊、紅雀隊、[[國民隊|國民隊]]及魔鬼魟隊（現為光芒隊）。

2009年的「傑基羅賓森日」，大聯盟每一位穿著球衣的人員全部換穿了42號球衣。

在美國，羅賓森並不僅是棒球選手，還被視為黑人的[[民權|民權]]鬥士，所以紀念羅賓森的活動不僅限於棒球界，而是全美國各地、各階層都會參與的全民運動。

==相關作品==
1997年10月Sony曾發行過一張向羅賓森的致敬的[[音樂|音樂]]專輯，專輯名稱叫做《傑基羅賓森：盜本壘 "用音樂致敬"》（''Jackie Robinson: Stealing Home "A Musical Tribute"''）。

1950年上映一部電影：《傑基羅賓森的故事》（''{{lang|en|The Jackie Robinson Story}}''），羅賓森甚至在電影中飾演他本人，這也是第一部描寫黑人成為美國英雄的電影。<ref>{{cite web|url=http://www.imdb.com/title/tt0042609/|title=The Jackie Robinson Story (1950)|language=en-US}}</ref>

2013年4月12日，美國上映一部以傑基羅賓森進入大聯盟經歷為基礎拍攝的電影《[[傳奇42號|傳奇42號]]》（42）。<ref>{{cite web|url=http://www.imdb.com/title/tt0453562/|title= 42 (2013)|language=en-US}}</ref>

== 效力的球隊 ==
傑基羅賓森當時所效力的布魯克林道奇隊，原本是位於[[紐約市|紐約市]][[布魯克林區|布魯克林區]]的職業球隊，後來道奇球團於1958年從[[美國東岸|美國東岸]]遷移至西岸的[[洛杉磯|洛杉磯]]，才成為現今眾所熟知的[[洛杉磯道奇|洛杉磯道奇]]隊。
