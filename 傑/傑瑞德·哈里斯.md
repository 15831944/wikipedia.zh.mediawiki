'''傑瑞德·法蘭西斯·哈里斯'''（{{lang-en|'''Jared Francis Harris'''}}，{{bd|1961年|8月24日}}）是一名[[英国|英国]]男演员，他知名角色有電視劇《[[广告狂人|广告狂人]]》中的{{link-en|Lane Pryce|Lane Pryce}}、科幻電視劇《[[危机边缘|危机边缘]]》中的David Robert Jones、歷史電視劇《[[王冠_(電視劇)|王冠]]》中的英國國王[[喬治六世|喬治六世]]、科幻電視劇《[[太空無垠|太空無垠]]》中的Anderson Dawes<ref>{{Cite web |url = https://variety.com/2014/tv/news/syfys-the-expanse-adds-walking-dead-mad-men-alums-1201361228/ |title = Syfy's ‘The Expanse’ Adds ‘Walking Dead,’ ‘Mad Men’ Alums |author = Kevin Noonan |date = 2014-11-20 |publisher = Variety |language = en}}</ref>，以及AMC電視劇《[[極地惡靈|極地惡靈]]》中的船長[[弗朗西斯·羅頓·莫伊拉·克羅茲|弗朗西斯·克羅茲]]<ref>{{Cite web |url = https://www.amc.com/shows/the-terror/talk/2016/09/mad-mens-jared-harris-boards-amcs-new-anthology-series-the-terror |title = Mad Men's Jared Harris Boards AMC's New Anthology Series The Terror |author = Ashleen Wicklow |publisher = amc |language = en}}</ref>。他也在電影中演出配角，例如《[[奇幻逆緣|奇幻逆緣]]》（2008年）、《[[大侦探福尔摩斯2：诡影游戏|大侦探福尔摩斯2：诡影游戏]]》（2011年）、《[[林肯_(2012年電影)|林肯]]》（2012年）和《[[伴諜同盟|伴諜同盟]]》（2016年）。
