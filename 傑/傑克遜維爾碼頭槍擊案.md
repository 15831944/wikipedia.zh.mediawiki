2018年8月26日的下午1:30（[[EDT|EDT]]），在美國佛羅里達州[[傑克遜威爾|傑克遜威爾]]的{{link-en|傑克遜維爾碼頭|Jacksonville Landing}}內舉行的一場[[電子競技|電競比賽]]發生了[[大規模槍擊|大規模槍擊]]。一名單獨的槍手大衛·卡茨（David Katz）開槍殺死兩人，另外有十人受傷，大衛·卡茨後來自殺身亡。<ref>{{Cite web |url = https://www.bbc.co.uk/news/world-us-canada-45315970 |title = Florida shooting: Video gamer kills two at tournament |date = 2018-08-27 |publisher = BBC |language = en}}</ref><ref>{{Cite web |url = https://edition.cnn.com/2018/08/26/us/jacksonville-madden-shooting/index.html |title = Two killed in shooting at Jacksonville video game tournament |author = Eric Levenson, AnneClaire Stapleton, Darran Simon |date = 2018-08-27 |publisher = CNN |language = en}}</ref>該事件也被稱為傑克遜維爾槍擊案。<ref>{{Cite web |url = https://www.inquisitr.com/5045086/david-katz-identified-as-jacksonville-shooting-suspect/ |title = David Katz Identified As Jacksonville Shooting Suspect, Report Claims He Opened Fire After Losing Game |author = Nathan Francis |date = 2018-08-26 |publisher = Inquisitr |language = en}}</ref><ref>{{Cite web |url = https://www.tallahassee.com/story/news/2018/08/26/democrats-call-more-gun-control-wake-jacksonville-shooting/1106119002/ |title = Florida Democratic gubernatorial candidates blame GOP inaction for Jacksonville shooting |author = James Call |date = 2018-08-26 |publisher = Tallahassee Democrat |language = en}}</ref><ref>{{Cite web |url = https://www.bbc.co.uk/news/av/world-us-canada-45317696/jacksonville-shooting-suspect-took-his-own-life |title = Jacksonville shooting: 'Suspect took his own life' |date = 2018-08-27 |publisher = BBC |language = en}}</ref>

== 行兇者 ==
槍手是大衛·卡茨（David Katz），是一名來自[[馬利蘭州|馬利蘭州]][[巴爾的摩|巴爾的摩]]的24歲[[電子競技|職業遊戲玩家]]，曾作為參賽者參加傑克遜維爾比賽。他的假名包括「Bread」、「mrslicedbread」、「RavensChamp」和「ravens2012champ」。他之前在2017年的《''{{link-en|Madden NFL 19|Madden NFL 19|Madden}}''》比賽中贏了10,000美元。法庭記錄顯示，卡茨有精神病記錄，而且給他開了抗精神病藥物。<ref>{{Cite web |url = https://www.independent.co.uk/news/world/americas/florida-shooting-david-katz-baltimore-twitch-shooter-mental-illness-gun-laws-a8510311.html |title = Florida shooting suspect had history of mental illness, court records show |author = Chris Riotta |publisher = independent.co.uk |language = en}}</ref>

== 受害者 ==
=== 死者 ===
* Elijah Clayton，22歲，是一名職業遊戲玩家，化名叫「True」和「Trueboy」。
* Taylor Robertson，27歲，是一名職業遊戲玩家，化名叫「SpotMePlzzz」。

=== 傷者 ===
11人被送往醫院接受治療。有10人受到槍傷，1人因為試圖逃離現場而受傷。
