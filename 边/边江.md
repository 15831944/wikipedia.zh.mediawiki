'''边江'''（{{bd|1982年|5月24日}}），出生于[[河北|河北]][[秦皇岛|秦皇岛]]<ref>{{Cite web |url=http://culture.people.com.cn/n1/2016/1221/c87423-28965158.html |title=边江：替“男神”“说话”的“御用配音”|accessdate=2018-03-16 |author=陈苑 |date=2016-12-21 |work=陈苑、熊旭 |publisher=人民网|language=Zh-hans }}</ref>。中国男配音演员、配音导演。

==生平==
毕业于[[北京电影学院|北京电影学院]]表演系02级高职配音班，与演员[[黄渤|黄渤]]<ref>{{Cite web |url=http://www.sohu.com/a/221470757_260616 |title=专访｜边江：一名优秀的配音演员，要花工夫在录音棚外|accessdate=2018-03-16 |author=夏奕宁 |date=2018-02-27 |work= |publisher=澎湃新闻|language=Zh-hans }}</ref>，配音演员[[邱秋|邱秋]]、[[王凯_(配音演员)|王凯]]、[[白雪岑|白雪岑]]、[[张磊|张磊]]等是同班同学。代表作有电视剧《[[三生三世十里桃花|三生三世十里桃花]]》的夜華、《[[太子妃升职记|太子妃升职记]]》的齐晟，动漫《[[狐妖小红娘|狐妖小红娘]]》的颜如玉、梵云飞，《[[全职高手|全职高手]]》的张新杰、《魔道祖師》的藍忘機，游戏《[[恋与制作人|恋与制作人]]》的周棋洛等。2018年加盟湖南卫视综艺节目《[[声临其境|声临其境]]》<ref>{{Cite web |url=http://ent.sina.com.cn/tv/zy/2018-02-08/doc-ifyrkuxs1638494.shtml |title=《声临其境》 边江：遇到“乱演”的演员会生气|accessdate=2018-03-16 |author= |date=2018-02-08 |work=冬冬 |publisher=新浪娱乐|language=Zh-hans }}</ref>。曾为配音团队[[光合积木|光合积木]]成员，2016年创立[[边江工作室|边江工作室]]。

由于边江、[[张杰_(配音演员)|张杰]]与另两位女配音演员[[乔诗语|乔诗语]]、[[季冠霖|季冠霖]]分别组合为多部热门电视剧男女主角配音，故有网友戏称[[中国电视剧|中国电视剧]]都是他们四人在谈恋爱<ref>{{Cite web |url=http://www.chinanews.com/yl/2018/08-15/8600287.shtml |title=国产剧都是他们在谈恋爱？揭秘荧幕后的“好声音”|accessdate=2018-08-20 |author=任思雨 |date=2018-08-15|work= |publisher=中国新闻网|language=Zh-hans }}</ref>。
