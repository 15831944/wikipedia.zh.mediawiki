'''边缘设备'''（{{lang-en|'''edge device'''}}）是向[[企业|企业]]或服务提供商核心网络提供入口点的设备。它们例如：[[路由器|路由器]]、路由[[網路交換器|交换机]]、{{tsl|en|Integrated access device|集成接入设备}}（IAD）、多路复用器，以及各种[[城域网|城域网]]（MAN）和[[广域网|广域网]]（WAN）接入设备。边缘设备也提供到运营商和服务提供商网络的连通性。

== 功能 ==
通常来说，边缘设备是提供经[[身份验证|身份验证]]的接入（通常以[[PPPoA|PPPoA]]或[[PPPoE|PPPoE]]）的更快、更有效的骨干和核心网络路由器。目前趋势是边缘设备智能化、核心设备“笨而快速”，因此边缘路由器通常包括[[服务质量|服务质量]]（QoS）和多业务功能来管理不同类型的流量。因而，核心网络通常为设计有如[[开放式最短路径优先|开放式最短路径优先]]（OSPF）、[[多协议标签交换|多协议标签交换]]（MPLS）等路由协议的交换机，以实现可靠性和可扩展性，允许边缘路由器具有到核心网络的冗余链路。核心网络之间的链路则不同，例如[[边界网关协议|边界网关协议]]（BGP）路由器经常用于对等交换。

== 翻译 ==
边缘设备可能支持将一种类型的网络协议转换为另一种类型的网络协议。例如[[以太网|以太网]]或[[令牌环|令牌环]]类型的[[局域网|局域网]]（LAN）或[[DSL|xDSL]]设备可能使用[[异步传输模式|异步传输模式]]（ATM）骨干网连接到其他核心网络。 ATM网络在单元中发送数据，并使用面向连接的虚拟电路。IP网络则是面向[[网络封包|网络封包]]；所以如果使用ATM作为核心，数据包必须封装在单元中，并且目标地址须转换为虚拟电路标识符。部分新型[[光導纖維|光纤]]使用例如[[GPON|GPON]]的[[无源光网络|无源光网络]]用户环路，边缘设备通过[[行動網路回傳|行動網路回傳]]连接到[[以太网|以太网]]。

== 多业务单元 ==
用于WAN的边缘交换机可能是一个多业务单元，这意味着它支持各种通信技术，包括[[综合业务数字网|综合业务数字网]]（ISDN）、T1电路、[[帧中继|帧中继]]和ATM。边缘设备还可能提供增强服务，例如[[虛擬私人網路|虛擬私人網路]]（VPN）支持、[[網際協議通話技術|VOIP]]，以及QoS服务。
