'''盧奇亞諾·費南多'''（{{lang-pt|Luciano Fernando}}，{{bd|1992年|4月13日|catIdx=Fernando, Luciano}}）是一名巴西職業棒球選手，擔任[[外野手|外野手]]，右投右打，現效力於[[日本棒球機構|日本棒球機構]][[太平洋聯盟|太平洋聯盟]]球隊[[東北樂天金鷲|東北樂天金鷲]]，[[南馬托格羅索州|南馬托格羅索州]][[大坎普|大坎普]]出身。

== 經歷 ==
===早期===
費南多在5歲時與父母移居日本[[群馬縣|群馬縣]][[太田市|太田市]]<ref name="megabrasil141026">{{cite web |url=http://megabrasil.jp/20141026_15053/ |language=ja |script-title=ja:東北楽天イーグルス、ドラフトで日系ブラジル人3世のルシアノ・フェルナンド選手（白鳳大）を4位指名 |publisher={{Link-ja|日系新聞|ニッケイ新聞}} |date=2014-10-26 |accessdate=2018-05-17 |deadurl=no |archiveurl=https://web.archive.org/web/20180518085305/http://megabrasil.jp/20141026_15053/ |archivedate=2018-05-18 }}</ref>，並且就讀於太田市立寶泉南小學校，小四時加入當地棒球隊，最初擔任[[投手|投手]]和[[捕手|捕手]]，及後升讀{{Link-ja|太田市立寶泉中學校|太田市立宝泉中学校}}，在學校棒球部則再兼任外野手<ref name="hochi141101">{{cite web |url=http://www.hochi.co.jp/baseball/npb/20141031-OHT1T50235.html |language=ja |script-title=ja:楽天】ドラ4フェルナンド、304キロ背筋力で本塁打量産だ！ |publisher=[[體育報知|體育報知]] |date=2014-11-01 |accessdate=2018-05-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20141108032153/http://www.hochi.co.jp/baseball/npb/20141031-OHT1T50235.html |archivedate=2014-11-08 }}</ref>。他就讀{{Link-ja|桐生大學附屬中學校、桐生第一高等學校|桐生大学附属中学校・桐生第一高等学校|桐生第一高等學校}}時則曾經兩度骨折<ref>{{cite web |url=http://www.sankei.com/region/news/141026/rgn1410260060-n1.html |language=ja |script-title=ja:【栃木この人】楽天4位指名の白鴎大・ルシアノ・フェルナンド選手 |publisher=[[產經新聞|產經新聞]] |date=2014-10-26 |accessdate=2018-05-17 |deadurl=no |archiveurl=https://web.archive.org/web/20180518085300/http://www.sankei.com/region/news/141026/rgn1410260060-n1.html |archivedate=2018-05-18 }}</ref>。雖然，他打出了22支[[全壘打|全壘打]]，但是桐生高校仍然未能打進[[日本高中棒球|甲子園]]<ref name="sponichi150118">{{cite web |url=http://www.sponichi.co.jp/baseball/news/2015/01/18/kiji/K20150118009646530.html |language=ja |script-title=ja:楽天ドラ4フェルナンド バレ目標「指定席」4番狙う右の大砲 |publisher=[[體育日本|體育日本]] |date=2015-01-18 |accessdate=2018-05-17 |deadurl=no |archiveurl=https://web.archive.org/web/20180518085303/https://www.sponichi.co.jp/baseball/news/2015/01/18/kiji/K20150118009646530.html |archivedate=2018-05-18 }}</ref>。

費南多升讀[[白鷗大學|白鷗大學]]，並且在大一春天時開始成為常規外野手<ref name="hochi141101" />，負責第四棒<ref name="megabrasil141026" />，與大三前輩[[岡島豪郎|岡島豪郎]]、{{Link-ja|塚田正義|塚田正義}}一同並肩作戰<ref name="sponichi150118" />。同年春天，他在{{Link-ja|關甲新學生棒球聯盟|関甲新学生野球連盟|關甲新學生棒球聯賽}}取得聯賽最多[[打點|打點]]，獲選為最佳九人之一<ref>{{cite web |url=http://kankoushin.web.fc2.com/23harujushou.html |language=ja |script-title=ja:2011年春季リーグ戦 |publisher={{Link-ja|關甲新學生棒球聯盟|関甲新学生野球連盟}} |accessdate=2018-05-17 |deadurl=no |archiveurl=https://web.archive.org/web/20180518085300/http://kankoushin.web.fc2.com/23harujushou.html |archivedate=2018-05-18 }}</ref>，大二秋季聯賽時成為首位[[打者|打者]]<ref name="sponichi150118" />，大三秋季聯賽則再度獲選為最佳九人之一，成為球隊的主將<ref name="megabrasil141026" />。大四春季聯賽時，他分別成為全壘打王、打點王和最佳九人之一<ref name="sponichi150118" />。他在大學生涯總共打出17支全壘打，距離該聯賽的紀錄只差一支<ref name="megabrasil141026" />。

2014年10月23日，費南多在日本職業棒球選秀會議中獲東北樂天金鷲第四指名<ref>{{cite web |url=http://www.rakuteneagles.jp/news/detail/4673.html |language=ja |script-title=ja:【2014年ドラフト会議】「契約交渉権獲得」選手のお知らせ |publisher=東北樂天金鷲 |date=2014-10-23 |accessdate=2018-05-17 |deadurl=no |archiveurl=https://web.archive.org/web/20150924120644/http://www.rakuteneagles.jp/news/detail/4673.html |archivedate=2015-09-24 }}</ref>，以契約金4000萬日元、年薪800萬日元為條件加入球隊<ref>{{cite web |url=http://www.sponichi.co.jp/baseball/news/2014/11/22/kiji/K20141122009328830.html |language=ja |script-title=ja:4カ国語OK日系4世フェルナンド 通訳と「二刀流」任せろ |publisher=體育日本 |date=2014-11-22 |accessdate=2018-05-17 |deadurl=no |archiveurl=https://web.archive.org/web/20180518085307/https://www.sponichi.co.jp/baseball/news/2014/11/22/kiji/K20141122009328830.html |archivedate=2018-05-18 }}</ref>，背號為51<ref>{{cite web|url=http://www.rakuteneagles.jp/news/detail/4755.html |language=ja |script-title=ja:2015年度：新入団選手発表会 |publisher=東北樂天金鷲 |date=2014-12-11 |accessdate=2018-05-17}}</ref>。同年12月，他接受了左手{{Link-en|鈎骨|Hamate bone}}鈎切除手術<ref>{{cite web |url=http://www.daily.co.jp/baseball/2015/01/13/0007653667.shtml |language=ja |script-title=ja:楽天ドラ4フェルナンドが左手手術 |publisher={{Link-ja|每日體育|デイリースポーツ}} |date=2015-01-13 |accessdate=2018-05-17 |deadurl=no |archiveurl=https://web.archive.org/web/20160304101603/http://www.daily.co.jp/baseball/2015/01/13/0007653667.shtml |archivedate=2016-03-04 }}</ref>。

=== 樂天 ===
2015年4月17日，費南多獲登錄為出場選手<ref>{{cite web |url=http://www.rakuteneagles.jp/news/announce/entry/201504 |language=ja |script-title=ja:公示（出場選手登録・抹消） |publisher=東北樂天金鷲 |date=2015-04-17 |accessdate=2018-05-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20160331004834/http://www.rakuteneagles.jp/news/announce/entry/201504 |archivedate=2016-03-31 |title=存档副本 }}</ref>，並且在4月19日對[[北海道日本火腿鬥士|北海道日本火腿鬥士]]的賽事中，作為大學前輩岡島豪郎的[[代打|代打]]首次登場，對[[大谷翔平|大谷翔平]]時打出投滾而出局<ref>{{cite web |url=http://www.rakuteneagles.jp/gamelive/stats/2015041901/ |language=ja |script-title=ja:2015年4月19日 楽天イーグルス 対 北海道日本ハム 成績詳細 |publisher=東北樂天金鷲 |date=2015-04-19 |accessdate=2018-05-17 |deadurl=no |archiveurl=https://web.archive.org/web/20160817002634/http://www.rakuteneagles.jp/gamelive/stats/2015041901/ |archivedate=2016年8月17日 }}</ref>。6月2日，他在對[[東京養樂多燕子|東京養樂多燕子]]的賽事中，以第七棒[[左外野手|左外野手]]的身份首次先發出賽<ref>{{cite web |url=http://www.sponichi.co.jp/baseball/news/2015/06/03/kiji/K20150603010467700.html |language=ja |script-title=ja:楽天 日系4世フェルナンド初安打お預け 初先発出場 |publisher=體育日本 |date=2015-06-03 |accessdate=2018-05-17 |deadurl=no |archiveurl=https://web.archive.org/web/20160304104542/http://www.sponichi.co.jp/baseball/news/2015/06/03/kiji/K20150603010467700.html |archivedate=2016-03-04 }}</ref>。翌日，他則在第9局以代打身份登場，並且對{{Link-ja|木谷良平|木谷良平}}打出全壘打，亦是他首支[[安打|安打]]<ref>{{cite web |url=http://www.sponichi.co.jp/baseball/news/2015/06/04/kiji/K20150604010474960.html |language=ja |script-title=ja:楽天「日系４世ルーキー」フェルナンド、プロ初安打が代打弾！ |publisher=體育日本 |date=2015-06-04 |accessdate=2018-05-17 |deadurl=no |archiveurl=https://web.archive.org/web/20171113013241/http://www.sponichi.co.jp/baseball/news/2015/06/04/kiji/K20150604010474960.html |archivedate=2017-11-13 }}</ref>。他在二軍則出賽67次，打出9支全壘打，為隊中之冠<ref>{{cite web|url=http://npb.jp/bis/2015/stats/idb2_e.html |language=ja |script-title=ja:2015年度 東北楽天ゴールデンイーグルス 個人打撃成績（イースタン・リーグ） |publisher=[[日本棒球機構|日本棒球機構]] |accessdate=2018-05-17}}</ref>。

費南多在2017年沒有在[[打席|打席]]出現過，季後為了尋求競爭激烈的比賽，參加了{{Link-es|墨西哥太平洋聯賽|Liga Mexicana del Pacífico|墨西哥冬季聯賽}}<ref>{{Cite web |url=https://www.nikkansports.com/baseball/column/bankisha/news/201712030000539.html |language=ja |script-title=ja:楽天フェルナンド、年越しでメキシコ武者修行中 |date=2017-12-06 |publisher=[[日刊體育|日刊體育]] |accessdate=2018-01-26 |deadurl=no |archiveurl=https://web.archive.org/web/20180126012943/https://www.nikkansports.com/baseball/column/bankisha/news/201712030000539.html |archivedate=2018-01-26 }}</ref>。

== 私生活 ==
費南多以[[弗拉迪米爾·巴倫汀|弗拉迪米爾·巴倫汀]]為目標<ref name="sponichi150118"/>。

他在大學時期扔球距離達120米<ref>{{cite web |url=http://www.sponichi.co.jp/baseball/npb/2014/draft/profile/rushiano-ferunando.html |language=ja |script-title=ja:2014年ドラフト ◇楽天4位指名 |publisher=體育日本 |accessdate=2018-05-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20150419212127/http://www.sponichi.co.jp/baseball/npb/2014/draft/profile/rushiano-ferunando.html |archivedate=2015-04-19 }}</ref>，高中時生時背肌力達304公斤<ref name="hochi141101"/>。

費南多的父母雖然都是[[巴西人|巴西人]]，但是曾祖母為日本人，因此他是[[日本裔巴西人|日本裔巴西人]]{{Link-ja|第四代 (日裔人)|四世 (日系人)|第四代}}<ref name="sponichi150118"/>。因此，他不但會說日文，也懂巴西的官方語言葡萄牙語、西班牙語和英語<ref name="sponichi150118" />。而目，雖然他的國籍是巴西，但是由於他在日本的中學和高中在學超過三年以上，因此獲豁免{{Link-ja|外國人限制 (日本棒球)|外国人枠 (日本プロ野球)|外國人限制}}。2016年9月，他獲選肩巴西代表，出戰[[2017年世界棒球經典賽|2017年世界棒球經典賽]]預賽，然而巴西最終未能晉級。
