'''盧慕貞'''（{{bd|1867年|7月30日|1952年|9月7日|盧慕貞}}），[[廣東|廣東]][[香山縣|香山]][[外堂村|外堂村]]人，[[孫中山|孫中山]][[元配|元配]]夫人。

==生平==
盧慕貞出生於累世積善的書香門第之家，父親盧耀顯曾至美國檀香山經商致富。盧慕貞與一般傳統中國女性一樣裹小腳，自小勤快、擅女紅，以孝敬長輩而聞名鄉里，其与孫中山的婚事亦属[[媒妁|媒妁]]之言。1885年5月26日，盧慕貞18歲時，與孫中山在[[翠亨村|翠亨村]][[結婚|結婚]]，並誕下三名子女─[[孫科|孫科]]、[[孫娫|孫娫]]及[[孫婉|孫婉]]。由於孫中山需為革命到處奔波，兩人共處時間很少。

[[辛亥革命|辛亥革命]]成功後，1911年12月29日，孫中山被推選為中華民國大總統，1912年2月15日，盧慕貞攜兩女[[孫娫|孫娫]]、[[孫婉|孫婉]]、侄女孫順霞搭乘英國「亞舍號」郵船自南洋檳榔嶼到上海，受到滬軍都督[[陳其美|陳其美]]等人的熱烈歡迎，被譽為“中華民國國母”，下榻滄州別墅。2月20日，盧慕貞母女在鄧澤如和孫科的護送下抵達南京，與[[孫中山|孫中山]]團聚。

盧慕貞並無受過西方先進教育，在思想上與孫中山有很大的距離。1913年[[二次革命|二次革命]]失敗，孫中山遭[[袁世凱|袁世凱]]之[[北洋政府|北洋政府]]通緝逃亡日本，翌年 ( 1914年 ) 孫在日本組織[[中華革命黨|中華革命黨]]繼續討袁，遭[[黃興|黃興]]等黨內人士反對，一時之間失去了革命意識的團結力量；在孫的低潮期，英文秘書[[宋靄齡|宋靄齡]]也要回國結婚而辭去秘書之職，而宋靄齡的父親[[宋嘉澍|宋嘉澍]][[牧師|牧師]]，是向來支持孫中山革命的友人，又從上海派自己另一位21歲女兒[[宋慶齡|宋慶齡]]來日本擔任孫的英文秘書，與孫朝夕相處，日久生情，兩人相約成婚；宋慶齡回上海與父母談及與孫的婚事，不料宋嘉澍夫婦以兩人年齡差距太大 ( 相差27歲 ) 而極力反對，揚言女兒若與孫成婚，將斷絕父女關係與孫的朋友關係，並且從此不再支持孫的革命事業。宋慶齡迫於無奈以及與孫的無悔愛情，遂決定連夜私奔返回日本，自行與孫結婚。

盧慕貞無心當第一夫人，亦感未能輔助孫中山主理國事。在宋慶齡婚事问题上，尽管她接受丈夫[[納妾|納妾]]，但這樣與革命理念中的一夫一妻制相悖，孫受到不少壓力。加以宋家為基督徒，不可能接受此種做法。孫於是決定跟無感情基礎的盧慕貞[[離婚|離婚]]。1915年4月14日，盧慕貞於[[美國|美國]][[夏威夷|夏威夷]]成為[[基督徒|基督徒]]。9月，孫派人接盧慕貞到[[日本|日本]]商談離婚事宜，雙方離婚協定後，盧慕貞回到澳門。同年10月25日下午，49歲的孫中山與22歲的宋慶齡在[[梅屋莊吉|梅屋莊吉]]家舉行婚禮。<ref>周谷<宋慶齡第一張出國護照及其結婚誓約書>(下)，《[[明報月刊|明報月刊]]》，[[香港|香港]]，1993年2月號，頁122。</ref>

盧慕貞晚年定居於[[澳門|澳門]]文第士街一號(現在是[[澳門國父紀念館|澳門國父紀念館]])，信仰虔誠，熱心幫助貧困。1933年，被[[澳門浸信教會|澳門浸信教會]]按立為該會第一任[[會佐|會佐]]。1952年9月7日逝世，安葬於[[澳門|澳門]][[舊西洋墳場|舊西洋墳場]]，後於1973年遷葬於[[氹仔|氹仔]]孝思永遠墓園。2005年，應[[中國|中國]][[廣東省|廣東省]][[中山市|中山市]]當局的要求，盧氏再遷葬於中山市。
