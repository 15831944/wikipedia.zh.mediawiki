'''盧龍光'''（{{lang-en|Rev. Dr. Lo Lung-kwong}}，{{bd|1951年|8月27日||catIdx=L}}），前[[香港基督教協進會|香港基督教協進會]]總幹事（2018年－2019年3月12日）<ref>{{Cite web|url=https://news.mingpao.com/pns/要聞/article/20190314/s00001/1552502172294/早前涉性騷擾-盧龍光辭基督教協進會總幹事|title=早前涉性騷擾 盧龍光辭基督教協進會總幹事}}</ref>；曾任[[崇基神學組|崇基神學組]]主任（1995年－2004年）、[[香港中文大學崇基學院神學院|香港中文大學崇基學院神學院]]院長（2004年－2014年）及前[[循道衛理聯合教會|循道衛理聯合教會]]會長（2006年－2012年），專研新約聖經及教牧神學，曾是循道轄下多間中小學校監及校董<ref>{{cite news |title=循道前會長盧龍光涉性騷擾，炮台山循道衛理中學：盧已辭任校監 |date=2019-02-18 |work=明報 |url=https://news.mingpao.com/ins/%E6%B8%AF%E8%81%9E/article/20190218/s00001/1550464541625/%E3%80%90%E9%A0%AD%E6%A2%9D%E8%B7%9F%E9%80%B2%E3%80%91%E5%BE%AA%E9%81%93%E5%89%8D%E6%9C%83%E9%95%B7%E7%9B%A7%E9%BE%8D%E5%85%89%E6%B6%89%E6%80%A7%E9%A8%B7%E6%93%BE-%E7%82%AE%E5%8F%B0%E5%B1%B1%E5%BE%AA%E9%81%93%E8%A1%9B%E7%90%86%E4%B8%AD%E5%AD%B8-%E7%9B%A7%E5%B7%B2%E8%BE%AD%E4%BB%BB%E6%A0%A1%E7%9B%A3 }}</ref>。

==簡歷==
盧龍光的母親盧顏志鈞曾於[[九龍城|九龍城]][[獅子石道|獅子石道]]英如小學和[[深水埗|深水埗]][[崇真小學暨幼稚園|崇真小學]]任教<ref name="盧顏志鈞師太生平" />，亦曾是有「傳理系之父」之稱的[[余也魯|余也魯]]的家庭女傭。盧龍光自小在香港[[調景嶺|調景嶺]]難民營第四木屋區長大<ref name="盧顏志鈞師太生平" />，小學就讀[[深水埗|深水埗]][[崇真小學暨幼稚園|崇真小學]]<ref name="盧顏志鈞師太生平" />，中學畢業於[[救恩書院|救恩書院]]<ref>{{cite web |url=https://www.kyc.edu.hk/attachment/publication/1542611261K33A4.pdf |title=救恩校訊 第14期 |last= |first= |date=2016-5 |website= |publisher=救恩書院 |access-date= 2019-3-26 |quote=}}</ref>。他在[[觀塘|觀塘]]雞寮返教會，長大後以「災胞」身份，到台灣[[國立中興大學|國立中興大學]]森林系讀書，希望以農業救國。<ref name="yahoo131120">{{ cite web |url=https://hk.news.yahoo.com/%E7%A4%BE%E9%81%8B%E5%85%83%E8%80%81-%E7%9B%A7%E9%BE%8D%E5%85%89-160000843.html |title=社運元老 盧龍光 |date=2013-11-20 |publisher=雅虎新聞 |author= }}</ref> 1973年畢業後回香港，入讀[[香港中文大學|香港中文大學]][[香港中文大學崇基學院神學院|崇基學院神學組]]。1979年按立為循道衛理聯合教會牧師，曾任柴灣愛華村堂及服務中心主任、 香港堂主任牧師。

在1980年代，盧龍光活躍於社運界，關注民生交通、[[八九民運|八九民運]]、居英權等議題。他曾參與發起全港反對巴士加價行動，亦曾為柴灣居民爭取興建[[東區醫院|東區醫院]]。1984年中英談判結束後，他以「民主公義協進會」組織政黨，成為「[[民協|民協]]」的前身。1985年，他從政黨抽身，遠趁英國深造。1988年於英國[[德倫大學|德倫大學]]完成羅馬書研究之哲學博士論文："Paul's Purpose in Writing Romans: The Upbuilding of a Jewish and Gentile Christian Community in Rome"。1989年六四事件時，他再度活躍於社運，參與[[支聯會|支聯會]]及[[香港基督徒愛國民主運動|香港基督徒愛國民主運動]]。之後他漸漸接受回歸的事實，主動搞國慶慶祝活動，並參選宗教界選委會。<ref name="yahoo131120" /><ref>{{ cite web |url=http://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=35343&Pid=2&Version=980&Cid=391&Charset=big5_hkscs |title=破立之間──專訪盧龍光 |date=2006-06-08 |publisher=時代論壇 |author=羅民威 }}</ref><ref>{{ cite web |url=http://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=60744&Pid=5&Version=0&Cid=220&Charset=big5_hkscs |title=生命不能沒火，也不能只有火，《時代論壇》專訪盧龍光 |date=2010-07-24 |publisher=時代論壇 |author=羅民威 }}</ref>

1995年盧龍光開始淡出政治舞台，出任[[崇基神學組|崇基神學組]]主任（1995年－2004年），2004年神學組易名為「[[香港中文大學崇基學院神學院|崇基學院神學院]]」，盧龍光繼續擔任神學院院長（2004年－2014年）。2006年至2012年，盧龍光同時擔任循道衛理聯合教會會長。2014年，在崇基神學院院長崗位退休後，任客席教授及名譽高級研究員，並任[[汕頭大學|汕頭大學]]文學院基督教研究中心主任。<ref name="yahoo131120" /><ref>{{ cite web |url=https://www.theology.cuhk.edu.hk/tc/academics/adjunct-teachers/lo-lung-kwong |title=盧龍光客座教授 |date= |publisher=崇基神學院 |author= }}</ref><ref>{{ cite web |url=https://www.fhms.edu.hk/213_supervisor.html |title=校監介紹 |date= |publisher=炮台山循道衛理中學 |author= }}</ref>

== 個人生活 ==
盧龍光父母都曾當兵，兩人相識在抗日戰場，都很關心國家和社會，盧父在1920年代曾因參與學運而被開除。

盧龍光妻子馬安妮原籍澳洲，是[[教院|教院]]高級講師，二人在教會認識。2013年時，兩名子女分別只有17和21歲。<ref name="yahoo131120" />

== 爭議 ==
根據2019年2月18日《明報》報導，盧龍光被投訴於2018年12月4日性騷擾循道轄下其中一間學校的女職員。性騷擾行為包括「拖手、多次摸臉、掃背及正面熊抱等」。循道其後成立5人小組調查，成員包括循道前會長李鼎新牧師、總議會書記劉麗詩牧師、陳建基牧師、馮丹媚及鄭大雅，經會面及翻看閉路電視片段，5人一致裁定性騷擾指控成立，並建議停止盧在教會學校教育部所有職務，以作懲治<ref>{{cite news |title=校監涉性騷擾，教會裁投訴成立，沒通知教局、沒報警，學校為保循道聲譽低調處理 |date=2019-02-18 |work=明報 |url=https://news.mingpao.com/pns/%E5%81%B5%E6%9F%A5%E5%A0%B1%E9%81%93/article/20190218/special/1550427238341 }}</ref><ref>{{cite news |title=盧龍光被指性騷擾停任校監，王美鳳重申教內教育及政策制定之重要 |date=2019-02-18 |work=時代論壇 |url=https://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=157113&Pid=102&Version=0&Cid=2141&Charset=big5_hkscs }}</ref><ref>{{cite news |title=邀見會內牧師提質疑 稱調查小組不夠資格|url=https://news.mingpao.com/pns/港聞/article/20190219/s00002/1550513732201/邀見會內牧師提質疑-稱調查小組不夠資格 }}</ref>。
另一方面，盧龍光於2019年1月16日召開臨時會議，向循道其他牧師公開事件，並提供一封聲稱於今年1月底發給林崇智的辭職信，內容提及自己在[[香港基督教協進會|香港基督教協進會]]總幹事及[[優質生命教育中心|優質生命教育中心]]總監的工作責任下，「近日在身、心、靈方面感覺疲乏」，故辭退[[循道學校|循道學校]]、[[炮台山循道衛理中學|炮台山循道衛理中學]]、[[沙田循道衛理中學|沙田循道衛理中學]]共3間學校的校監，以及[[華英中學|華英中學]]校董一職。<ref>{{cite news |title=4校更新資料 教局收通知 涉性騷擾 循道牧師辭校監|url=https://news.mingpao.com/pns/港聞/article/20190219/s00002/1550513730844/4校更新資料-教局收通知-涉性騷擾-循道牧師辭校監 }}</ref>
