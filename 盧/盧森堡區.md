'''盧森堡區'''（{{lang-lb|Distrikt Lëtzebuerg}}；{{lang-de|Distrikt Luxemburg}}；{{lang-fr|District de Luxembourg}}）是[[西歐|西歐]]國家[[盧森堡|盧森堡]]的3個[[区_(卢森堡)|区]]之一，于2015年10月3日废除。盧森堡首都[[盧森堡_(城市)|卢森堡]]的所在地。曾下設有4個縣和47個市鎮。

# [[Esch-sur-Alzette縣|Esch-sur-Alzette縣]] - Bettembourg - Differdange - Dudelange - Esch-sur-Alzette - Frisange - Kayl - Leudelange - Mondercange - Pétange - Reckange-sur-Mess - Roeser - Rumelange - Sanem - Schifflange
# [[盧森堡縣|盧森堡縣]] - Bertrange - Contern - Hesperange - [[盧森堡_(城市)|卢森堡]] - Niederanven - Sandweiler - Schuttrange - Steinsel - Strassen - Walferdange - Weiler-la-Tour
# [[梅爾施縣|梅爾施縣]] - Bissen - Boevange-sur-Attert - Colmar-Berg - Fischbach - Heffingen - Larochette - Lintgen - Lorentzweiler - Mersch - Nommern - Tuntange

東與[[格雷文馬赫區|格雷文馬赫區]]、北與[[迪基希區|迪基希區]]、西與[[比利時|比利時]]的[[盧森堡州|盧森堡州]]、南與[[法國|法國]]的[[摩澤爾省|摩澤爾省]]接壤。
