'''領展房地產投資信託基金'''（{{lang-en|Link Real Estate Investment Trust}}），2015年8月19日以前稱'''領匯房地產投資信託基金'''<ref>{{cite web|title=更改信託基金之名稱及標誌、更改網站地址及更改管理人之名稱|url=http://www.hkexnews.hk/listedco/listconews/SEHK/2015/0819/LTN20150819234_C.pdf|publisher=領展資產管理有限公司|accessdate=2015-08-19}}</ref>，於2005年11月25日在香港聯交所上市，是香港首家上市及最大型的房地產投資信託基金領展房地產投資信託基金，以市值<ref>{{cite web |author1=公司網頁 |title=業務里程碑 |url=http://www.linkreit.com/TC/aboutus/Pages/Milestones.aspx |website=www.linkreit.com}}</ref>計現時亞洲地區最大型[[房地產投資信託基金|房地產投資信託基金]]，亦是全球以零售為主最大的房地產投資信託基金之一。由領展資產管理有限公司擔任管理人，基金單位全數由私人（個人或機構）投資者持有，香港政府並無持有任何權益。領展旗下物業組合遍及香港、北京、上海和廣州，包括於香港有約900萬平方呎零售物業、約61,000個泊車位<ref>{{cite web|title=領展公司概覽及歷史|url=http://www.linkreit.com/TC/aboutus/Pages/Corporate-History-and-Profile.aspx}}</ref>，以及一個發展中項目，並於中國內地有約276萬平方呎零售及辦公室物業。現任[[董事會主席|董事會主席]][[聶雅倫|聶雅倫]]（Nicholas Charles Allen）於2016年4月1日起接替第二任主席[[蘇兆明|蘇兆明]]（首任主席為[[鄭明訓|鄭明訓]]）。現任領展[[行政總裁|行政總裁]][[王國龍|王國龍]]（George Kwok Lung HONGCHOY）於2009年加入。至2017年3月31日投資物業組合之估值達1740.6億港元，其零售設施以提供附近居民日常所需為主。2014年，領展晉身為[[恒生指數|恒生指數]]成份股，並於2015年獲納入[[恒生綜合大型股指數|恒生綜合大型股指數]]成份股。於2016年7月，領展發行了首批綠色債券。該批2026年到期的五億美元定息債券，息率為2.875%，為香港企業取得最低息率之一。此債券分別獲穆迪及標準普爾評為「A2」及「A」級別。<ref>{{cite web|title=領展發行5億美元 "綠色債券"|url=http://www.linkreit.com/TC/corporate/Pages/Green-Bond.aspx|website=www.linkreit.com}}</ref>

==上市==
領展的資產是由[[香港房屋委員會|香港房屋委員會]]（房委會）分拆其零售物業及停車場而來。房委會早年由於特區政府決定停售[[居者有其屋|居屋]]，導致房委會面臨財政入不敷支，陷入困境，又加上2003年[[香港|香港]]爆發[[嚴重急性呼吸系統綜合症|嚴重急性呼吸系統綜合症]]（SARS），會影響公共設施衛生環境的惡劣問題。所以在2004年初房委會決定分拆其部分零售物業及停車場，將之[[證券化|證券化]]並成立領匯基金（現名為領展基金）於[[香港交易所|香港交易所]]上市，藉以籌集資金。就有關領展上市的決定，房屋及規劃地政局局長在傳媒答問環節中表示：「我們一向表示，我們是需要財政上的支持，令我們能夠繼續公屋建設計劃。」<ref>{{cite web |author1=政府資訊中心 |title=新聞公報 |url=http://www.info.gov.hk/gia/general/200507/20/07200147.htm |website=www.info.gov.hk |accessdate=2018-08-02}}</ref>

2004年12月6日，領展進行第一次招股，獲分配上市編號2382（第二次上市獲分配上市編號才是0823），單位的定價在10.5至10.8港元，集資250億元，並向散戶提供百分之三的折扣優惠。受惠於當時的低息環境，領展第一次招股時吸引了51萬名散戶認購，涷結資金2800億元，超額認購130倍，打破11年前由[[駿威汽車|駿威汽車]]創下的凍資紀錄，成為當時凍結資金最多的公司。<ref>{{cite web|url=http://the-sun.on.cc/channels/fina/20041122/20041122023614_0002.html|title=太 陽 ??： SUN 時 事|website=the-sun.on.cc}}</ref><ref>{{cite web|url=http://www.singtao.com/archive/fullstory.asp?andor=or&year1=2004&month1=12&day1=6&year2=2004&month2=12&day2=6&category=all&id=20041206a05&keyword1=&keyword2=|title=星島日報網頁 - www.singtao.com|website=www.singtao.com}}</ref><ref>{{cite web|url=http://paper.wenweipo.com/2004/12/10/YO0412100001.htm|title=無懼訴訟領匯凍資2800億 - 香港文匯報|website=paper.wenweipo.com}}</ref>

公屋商場私有化引起公眾關注。由於[[香港公共房屋|公屋]]商場為房委會主要的財政來源，公屋居民擔心公屋商場[[私有化|私有化]]後，將使房委會唯一的盈利收入來源斷絕，最終導致公屋租金上升。此外，他們亦擔心屋邨商場私有化後以盈利為先，不再提供現有的廉價消費品，並認為房委會這行動違反《[http://www.legislation.gov.hk/blis_ind.nsf/FB2D3FD8A4E2A3264825647C0030A9E1/D3491E1062ECD5DA4825655C002DC455?OpenDocument 房屋條例]》中，香港房屋委員會需確保為居民提供生活設施的規定。以[[盧少蘭|盧少蘭]]為代表的公屋居民，在[[香港立法會議員|香港立法會議員]][[鄭經翰|鄭經翰]]、[[陳偉業|陳偉業]]等的支持下，於公開發售期間向[[高等法院|高等法院]]提請[[司法覆核|司法覆核]]。期間房委會為求令領展早日上市，曾要求法庭將盧少蘭本來28日的上訴期縮短到少於24小時。盧少蘭最終在無代表律師的情況下在上訴庭敗訴。但是由於[[終審法院|終審法院]]沒有權力像高等法院的原訟庭及上訴庭縮短[[上訴|上訴]]期限，形成不明朗因素，加上房委會沒有在招股書中披露法律風險，受法律程序所礙，最終房委會宣佈擱置上市，未能成功套取資金。這導致2005年1月1日，投資領展的小投資者在香港[[皇后像廣場|皇后像廣場]]舉行集會，抗議公屋居民被喻為「阻人發達」的行為。另一方面，領展事件亦引發香港對公共資產私有化的討論，2004年12月25日130名學者及民間人士聯合發表聖誕宣言，呼籲社會討論公產私有化，希望各界正視當下嚴峻的社會不公平及貧富分化問題。2004年12月30日，超過30個民間團體發出聯合聲明，反對公共資產私有化。

當時於立法會會議上房屋及規劃地政局局長孫明揚曾表示：「分拆出售計劃有充足的法理依據。房委會以及參與計劃的投資銀行和律師們，在籌備分拆出售的前期，做了大量小心論證工作，並取得本地和英國資深大律師的明確法律意見，認為房委會有權根據《房屋條例》分拆出售其商場和停車場設施。因此，並無需要修改法例，才推行計劃。事實上，自從二零零三年七月房委會公布分拆出售的決定以來，傳媒屢作廣泛報道，立法會房屋事務委員會亦多次討論而本會也曾於十二月一日作出冗長的休會辯論。一直以來，分拆出售計劃的法理依據，從未受到任何質疑。」<ref>{{cite web|url=http://www.info.gov.hk/gia/general/200501/05/0105202.htm|title=立法會一題︰領匯房地產投資信託基金押後上市|website=www.info.gov.hk}}</ref>

2005年7月，[[香港終審法院|香港終審法院]]認為《房屋條例》只要求房委會「確保提供」商場及停車場等設施，並無必要擁有該等設施，判房委會勝訴。<ref>{{cite web |url=http://news.gov.hk/tc/category/businessandfinance/050720/html/050720tc03001.htm |title=存档副本 |accessdate=2008-11-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20081120220747/http://news.gov.hk/tc/category/businessandfinance/050720/html/050720tc03001.htm |archivedate=2008-11-20 }}</ref>

===再度上市===
2005年11月14日，領展再次公開招股，共發行19.25億基金單位，每手500個單位，連同經紀[[佣金|佣金]]等費用，入場費最多為4940港元。單位的定價在9.7至10.3港元，[[散戶|散戶]]獲5%折扣，即9.22至9.78港元。由於投資環境及[[利息|利息]]與去年有別，領展再次招股對公眾的吸引力，估計不如去年。同年11月25日領展（{{sehk|0823}}）正式上市，當時財政司司長唐英年出席了上市儀式，領展是首隻在香港上市的房地產信託基金及當年最大規模的私營化項目，從上市起，領展管理的運作獨立於所有公營部門。<ref>{{cite web |author1=港交所 |title=領展上市 |url=http://www.hkexnews.hk/listedco/listconews/SEHK/2006/0714/00823/CWF104.pdf |website=www.hkexnews.hk |publisher=港交所}}</ref>首日掛牌股價勁升，[[股價|股價]]大漲14.56%至11.80港元，中午一度觸及11.85高位。

==投資策略==

領展投資和管理可獲取持續回報、位處香港及中國內地一線城市的零售和商業物業 (不包括酒店及服務式住宅)。自2015年1月起，領展採取涵蓋六大增長動力的業務拓展策略，包括資產提升、資產管理、資產收購、資產出售、物業發展及物業重建<ref>{{cite web |author1=領展 |title=經營方式 |url=https://www.linkreit.com/tc/corporateProfile/ |website=https://www.linkreit.com/tc/corporateProfile/}}</ref>；業務模式帶來全面的增長原動力，<ref>{{cite web |author1=領展 |title=領展投資策略 |url=https://www.linkreit.com/tc/faq/ |website=https://www.linkreit.com/tc/faq/ |publisher=領展 |accessdate=2019-01-18}}</ref>包括：
*(a) 為有潛力提升收入的物業進行資產優化；
*(b) 為整個物業組合進行資產管理；
*(c) 收購收入及資本均具增長潛力之優質資產；
*(d) 透過出售非核心資產，使資金得以循環投資；
*(e) 設計、建造及持有物業發展項目；及
*(f) 重建現有物業，以達致長遠可持續增長。

=====樂富廣場=====

=====赤柱廣場=====
[[File:Stanley_Plaza_viewed_from_1st_floor.JPG|thumb]]
{{main|赤柱廣場}}

赤柱廣場位於赤柱海旁，鄰近著名的赤柱大街和歷史地標美利樓。赤柱廣場於1999年落成，翻新前，由於原有物業設計及佈局所限，未能有效利用場內各樓面及天然環境。領展2011年11月完成投資額達2.27億港元的提升工程，把商場發展成地標性休閒好去處，吸引香港居民及外地遊客。

赤柱廣場設置綠化天台，並使用風力發電街燈、節能LED燈及環保建築物料，2012年獲香港環保建築協會頒發最高「白金」評級。<ref>[http://news.sina.com.hk/news/20111114/-33-2491152/1.html?rtext 赤柱廣場翻新 保育節能]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }} 明報，2011年11月14日</ref>

赤柱廣場亦是香港首個寵物友好的購物商場，提供設施包括狗帶牽引位置、寵物及主人專用升降機等，方便竉物主人使用。2011至2013年間，赤柱廣場與愛護動物協會合辦「海濱寵物嘉年華」，挑戰各項與狗隻相關的健力士世界紀錄。

王國龍接受信報時透露，領展一直有留意大灣區內的收購機會，5年前進軍內地之初亦打算以深圳作首站，主因「廣東人消費、文化都較相似」。問到會否有意在大灣區「9+2」各城巿中均設有據點時，王國龍說，大灣區提倡的一小時生活圈，將令城市間往來更便捷，使消費人口更集中，「好多（消費者）都會返去廣州、深圳及香港。」故領展在大灣區內的發展，短期亦會先集中在香港、廣州及深圳。

==愛．匯聚計劃==

2013年1月，領展成立「愛．匯聚計劃」，將慈善款項與物業收益掛鈎，每年將相等於上一個財政年度之物業收入淨額最高0.25%的款項資助慈善項目。自2013/2014年度起，領展「愛•匯聚計劃」合共投放約4,700萬港元予社區及慈善項目。<ref>{{cite web |author1=領展 |title=領展「愛•匯聚計劃」之最新動態 |url=http://www.linkreit.com/TC/investor/Documents/CW00823%20book%201.pdf |website=www.linkreit.com |publisher=領展 |accessdate=2018-08-20}}</ref>

2018/19年度領展「愛．匯聚計劃」經領展遴選委員會和董事局嚴謹甄選和審批後，共選出8個資助項目。

;三代．共融．查篤撐
*項目第四年繼續透過生動方式向兒童推廣粵劇，並以這中國傳統藝術作為共同興趣，促進三代共融。

;綠活地圖探索及社區導賞
*經過第一年的青少年「綠色大使」培訓及綠活地圖製作，今年的導賞員培訓活動進一步推廣至長者，並開發更多在領展商場附近的綠活路線。

;耆青小丑～快樂速遞
*延續項目第一年的成功，鼓勵長者透過學習小丑表演、非洲鼓、拳擊舞等項目並參與表演，與社區保持連繫，藉此提倡活齡社區及傳播正能量

;領展音樂獎學金
*項目第二年繼續資助有志學習音樂至專業水平的基層兒童接受專業音樂培訓，並提供平台讓學生參與演奏表演及音樂快閃活動，增強他們於表演的自信心。

;童心耆創綠家園
*透過培訓長者成為導賞員及舉辦綠色工作坊，推動活齡社區、文化共賞、關注環境及周邊社區。

;導盲犬寄養家庭發展及公眾教育
*作為第五年資助項目，繼支持導盲犬繁殖後，今年進一步推廣寄養家庭發展及公眾教育，協助視障人士走入社區，促進共融

;「領行員」－青少年職志培育計劃
*向基層青少年提供職業生涯諮詢，透過專業證書課程及以社區活動作實習機會，讓他們學習新技能，從而提升自信及自立能力。

;「惜食全方位」食物回收及教育
*項目第三年在領展鮮活街市和商場進行食物回收，製作熱飯盒及食物包予有需要人士和家庭。計劃於本年度期望可減少625噸食物浪費。
<ref>{{cite web |author1=領展 |title=領展「愛．匯聚計劃」 |url=https://www.linkreit.com/tc/community/togetherInitiatives/ |website=https://www.linkreit.com/tc/community/togetherInitiatives/ |publisher=領展 |accessdate=2019-02-15}}</ref>

該計劃專注於兩個核心範疇：社會項目資助及領展第一代大學生獎學金。社會項目資助至今推出第五年，為六個社區項目提供的金額達到610萬港元，其中包括「『惜食全方位』食物回收計劃」及「音樂兒童@領展」。「『惜食全方位』食物回收計劃」是一項於領展鮮活街市及商場推行的全面剩食回收計劃，第二年計劃已於2017年10月展開，預計惠及約130萬人士，減少約345,000公斤食物浪費。另外，第一代大學生獎學金以支援有需要的學生。計劃旨在資助家庭三代中的首代大學生於香港其中一所大學就讀，{{When|至今|time=2018-12-16}}受惠的大學生達370人。2017/2018年度計劃資助了140名學生，資助額合共280萬港元。整個計劃由中學校長直接提名學生申請，其中40名大學二年級及三年級學生則獨立於網上申請，並由香港社會服務聯會旗下的HKCSS WiseGiving Limited獨立監管。獲頒獎學金的學生加入成為「領展同學會」的一員，並於領展的實習生計劃中獲優先考慮。<ref>{{cite web |author1=領展 |title=領展「愛•匯聚計劃」之最新動態 |url=https://www.linkreit.com/linkcorp/api/v1/file/SiteAssets/CorporateWebsite/InvestorRelations/FinancialReports/CW00823%20book%201.pdf?y=1c2cf47d936a607b02e5b9aebee97c9b |website=www.linkreit.com |publisher=領展 |accessdate=2018-09-11}}</ref>

==對香港社會的負面影響==
領展街市及熟食檔位在2012 - 13年度（截至2012年9月）的平均月租為每平方呎67.3港元；相若時期香港房屋委員會街市的平均月租為每呎約37港元，熟食檔位約24港元；而食物環境衛生署街市轄下街市攤檔的平均月租為每呎31港元，熟食市場為30.9港元。<ref>[http://www.info.gov.hk/gia/general/201303/27/P201303270417.htm 立法會六題：領匯鋪租及住戶開支]，2013年3月27日。</ref>

[[張超雄|張超雄]]於2012年11月21日在立法會發言時批評：「領匯與[[壟斷|壟斷]]、[[通脹|通脹]]和[[貧窮|貧窮]]均有莫大關係。大家想起領匯，便會想起它不斷加租……商鋪若要續租，便要付出更高的成本，必然會將成本轉嫁顧客，造成通脹。而且，迫走小商戶，引入大財團。小商戶無生意可做，便會走向貧窮。公共屋邨的居民面對加租、加價，導致他們更貧窮。現今領匯對很多基層市民，尤其是很多居於公屋的老弱和貧窮市民，造成很大困擾。」<ref>[http://www.legco.gov.hk/yr12-13/chinese/counmtg/floor/cm1121-confirm-ec.pdf 立法會─2012年11月21日]，第168－169頁</ref>[[馮檢基|馮檢基]]議員批評：「多年來，我們看到領匯公司用多種高壓手段管理商業設施，不但迫令商戶搬遷和裝修、迫小商戶結業，更藉重組商場的行業組合，引入大型連鎖店，扼殺小商戶的生存空間。」<ref>[http://www.legco.gov.hk/yr12-13/chinese/counmtg/floor/cm1121-confirm-ec.pdf 立法會─2012年11月21日]，第180頁</ref>

特區政府把大量公屋街市出售予領匯，被指造成基層市民[[創業|創業]]更加困難，「領展街市勢將成為社會矛盾的焦點」。<ref>[http://hk.apple.nextmedia.com/nextmag/art/20150925/19307836 領展上市埋禍根　街市失守基層創業更難]，2015-09-25</ref>《東方日報》有文章指出：「領展幾乎壟斷了公共屋邨的商場及街市，居民無從選擇，只能任人宰割，那些靠[[綜援|綜援]]生活的居民更是苦不堪言。」「羊毛出自羊身上，領展唯利是圖，屋邨居民成為最大受害者。」<ref>[http://orientaldaily.on.cc/cnt/news/20160410/00186_001.html 出售商場惹爭議 賺錢亦須顧民意]，東方日報，2016-04-10。</ref>領展的所作所為被指「已經到了民怨沸騰的地步」，特區政府被批評對於領展造成的問題束手無策。<ref>{{cite news |url=http://orientaldaily.on.cc/cnt/news/20160727/00190_001.html |title=寧可養蚊不減租 領展成魔因官助 |work=東方日報 |date=2016-07-27 |quote=已經到咗民怨沸騰嘅地步……港府講就天下無敵，做就無能為力，對領展問題束手無策}}</ref>

自2014年9月領展成功出售[[彩霞邨|彩霞邨]]商場予大富基業後後，邨內出現種種管理問題，如嘗試向社福機構租戶徵收管理費。<ref>{{cite news |url= http://the-sun.on.cc/cnt/news/20160316/00407_023.html |title=彩霞商場新業主 違規徵管理費 |date= 2016-03-16 |work=太陽報 |location=香港}}</ref>2016年10月，彩霞邨部份停車場亦改變用途作安老院之用<ref>{{cite news |url= http://www.swd.gov.hk/doc/LORCHE/lorcheno1343_12515_20oct2016.pdf |title=彩霞居安老院牌照 |publisher=社會福利署 |date=2016-10-20 |access-date=2017-07-25}}</ref>。同時，一些居民亦表示停車場車位除了加租外，亦取消固定車位，逼使住戶與時租車位人士爭用車位。2018年領展轄下的觀塘曉麗苑固定車位由月租3380元加至3800元，加幅達12.4%，高於房委會轄下停車場今年平均6%的加幅。而曉麗苑加價後的車位月租金，比太古城住戶於商場的固定車位月租3580元更高。<ref>{{cite news |url= https://news.mingpao.com/pns/dailynews/web_tc/article/20180415/s00002/1523729067996 |title=梁翊婷：領展10屋苑泊位加租一成 加幅高房委會一倍 曉麗苑月租超太古城 |publisher=明報 |date=2018-04-15 | access-date = 2018-04-15}}</ref>

[[田灣邨|田灣邨]]商場自2015年售予傑熙集團後，商場陸續被圍封，即使2017年4月向[[地政總署|地政總署]]申請改變用途開辦{{tsl|en| Nord Anglia Education  | Nord Anglia International School}}國際學校被拒後，學校表示會繼續磋商<ref>{{cite news | url = http://hk.on.cc/hk/bkn/cnt/news/20170429/bkn-20170429160639734-0429_00822_001.html | title = 田灣商場國際校加建天橋違契　地署否決申請 | date = 2017-04-29 | work = 東方日報}}</ref>。由於商場一直被圍封，區內於2017年只剩下一間惠康超級市場，嚴重影響區內市民的選擇。

===土地契約問題===
====停車場加租與違法出租====
領展出售物業亦帶動停車場加租，以[[安蔭邨|安蔭邨]]停車場為例，有「舖王」之稱的新業主鄧成波把停車場時租加價近四成半（平日11元或假日13元加至劃一16元），月租亦由2,680元加至2,900元。<ref>{{cite news | url = https://news.mingpao.com/ins/instantnews/web_tc/article/20170725/s00001/1500962769802 | title = 團體批林鄭拒處理領展問題　2年半轉售28物業影響居民商戶 | work = 明報 | date = 2017-07-25 }}</ref>。另外，資深投資者林子峰購入的[[東熹苑|東熹苑]]及[[華貴邨|華貴邨]]商場的車位更被拆售，單個車位以高達150萬元出售。部分拆售的車位更被發現違反地契，由非屋苑居民使用。然而，領匯監察亦關注拆售車位引伸出的維修保養等的權責問題。

====「阻礙」公屋重建====
由於領展的前身領匯上市的原因，純綷是為缺錢的房委會解困，不是事情沒有太多詳細研究，包括自計劃上市至後上市後，不少屬於地契的法律條文詮釋沒有多大的關注，直至[[香港房屋委員會|香港房屋委員會]]在2014年2月的一份文件指出：「已拆售屋邨乃領匯及房委會共同擁有業權的，故此我們需要得到領匯的同意，一起檢討相關的法例、地契、公契及業權等問題，擬定一套領匯及房委會皆能接受的重建模式，然後才可推展重建。」<ref>[http://www.housingauthority.gov.hk/tc/common/pdf/about-us/housing-authority/ha-paper-library/HA6-14.pdf 香港房屋委員會議事備忘錄　包括華富邨在內的22個非拆售高樓齡屋邨的重建檢討]</ref>這意味特區政府如果打算重建其範圍內有領展物業的舊[[屋邨|屋邨]]，必須取得領展同意才可以進行<ref>{{Cite web|author=姚松炎 |url=http://news.stheadline.com/dailynews/headline_news_detail_columnist.asp?id=290770&section_name=wtt&kw=222 |title=房產學人——領匯非拆售商場咁簡單 |publisher=頭條網 |date=2014-06-24}}</ref>，而領展或會透過其業權份數影響重建計劃的設計，甚至可能要求房委會需先收購其業權份數。

====逃避居屋的管理和維修費責任====
[[傳真社|傳真社]]揭發領展並無依照2005年與房委會簽訂的《買賣協議》攤分23個居屋公用地方的管理和維修費，令屋苑業主白白多付千萬元維修費。僅有3個屋苑知悉此條款，與領展按比率分攤相關費用。房委會沒有履行執行人責任，採取監察措施，監督領展遵守協議。<ref>{{cite news|title=領展須分擔23居屋公用地方維修費 僅3屋苑知情 房委會無執行契諾累業主損失|url=https://www.factwire.org/single-post/2018/07/08/%E9%A0%98%E5%B1%95%E9%A0%88%E5%88%86%E6%93%9423%E5%B1%85%E5%B1%8B%E5%85%AC%E7%94%A8%E5%9C%B0%E6%96%B9%E7%B6%AD%E4%BF%AE%E8%B2%BB-%E5%83%853%E5%B1%8B%E8%8B%91%E7%9F%A5%E6%83%85-%E6%88%BF%E5%A7%94%E6%9C%83%E7%84%A1%E5%9F%B7%E8%A1%8C%E5%A5%91%E8%AB%BE%E7%B4%AF%E6%A5%AD%E4%B8%BB%E6%90%8D%E5%A4%B1|accessdate=2018-07-11|publisher=傳真社|date=2018-07-08}}</ref><ref>{{cite web|url=http://hd.stheadline.com/news/realtime/hk/1260348/|title=領展須分擔23居屋維修費 僅3屋苑知情穗禾苑白付千萬|publisher=}}</ref>

===食環署食品零售價格調查===
以上調查的結果卻與政府委託顧問調查公司的報告有很大的差異。食環署在 2016 年進行了一項調查，搜集其轄下街市（食環街市）的食品零售價格，並與其他主要零售渠道的食品零售價格作出比較，調查報告指出「領展街市沒有食品的價格為最高」，以及「領展街市有17 種食品的價格較食環街市及超市低」。<ref>{{cite web |author1=弘達顧問有限公司 |title=不同零售渠道出售的新鮮食品之價格調查調查報告 |url=http://www.fehd.gov.hk/tc_chi/pleasant_environment/tidy_market/report_freshfood_survey_c.pdf |website=www.fehd.gov.hk |publisher=弘達顧問有限公司}}</ref>
