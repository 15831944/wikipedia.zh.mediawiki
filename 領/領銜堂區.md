'''領銜堂區'''（{{lang-la|Titulus cardinalitius}}）是被分配給一位[[司鐸級樞機|司鐸級樞機]]，使之在名義上擔任這個堂區的主任[[司鐸|司鐸]]（或稱“本堂[[神父|神父]]”）之從屬於[[天主教羅馬教區|羅馬教區]]的[[堂區|堂區]]，部分會被分配給執事級樞機，因而轉變成領銜執事區<ref name="輔神">{{cite web |title = 樞機（CARDINALS）|url = http://www.catholic.org.tw/theology/public/liyi/topics_sacraments_domination.html |publisher=天主教輔仁神學院禮儀研究中心}}</ref>。

== 沿革 ==

在早期的羅馬地區教會，會把羅馬城以多個被稱為Titulus的聚會場所為中心分為多個區域方便管理，根據《教宗名錄》[[教宗法比盎|教宗法比盎]]有將其轄下的司鐸派遣到各個Titulus負責教務。<ref>[http://www.newadvent.org/cathen/14745b.htm Titulus]，Catholic Encyclopedia</ref><ref>《歷代教宗簡史》，鄒保祿，
碧岳學社</ref>而這些司鐸也組成了初期的羅馬教區聖職團，也就是今日[[樞機團|樞機團]]最早的前身。

這些分駐於羅馬教區各堂區的神父由於職務的關係，需要輔佐羅馬主教即教宗舉行儀式，因此逐漸擁有在各場合代表教宗或是擔任教宗的顧問等重要職責。這些人數隨時代有所變動的司鐸們爾後與駐於羅馬教區內的執事區執事和羅馬郊區的七個教區的主教形成名為樞機團的教宗顧問團。自1059年開始教宗[[尼各老二世|尼各老二世]]把選舉教宗的權利交給主教級樞機，而[[教宗亞歷山大二世|教宗亞歷山大二世]]則司鐸級和執事級樞機也納入選舉團中<ref>{{cite web|title =神學辭典－枢机|url =http://www.chinacath.org/book/html/131/7789.html}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。此後，[[樞機|樞機]]在[[天主教会圣统制|天主教的體制]]中逐漸有了更普面的意義，而不是僅限於在羅馬教區內事務上輔佐教宗。

後來這些樞機頭銜逐漸開始交給了羅馬教區以外的神職人員，由於樞機司鐸和樞機執事早先都出自羅馬教區內的典故，為了保留這傳統被任命為司鐸級和執事級的樞機們都會象徵性的被交託一個羅馬教區內的堂區或執事區，以使他們在名義上隸屬羅馬教區的聖職團<ref name="輔神" />，而原本的Titulus也被用來專指樞機的領銜堂區了。<ref>[https://hereisthechurch.wordpress.com/2011/07/05/what-is-a-titular-church/ What Is A Titular Church?]，Here is the Church…</ref>

== 制度 ==
今日所有的司鐸級樞機和執事級樞機都會和他被交託的領銜堂區或執事區保有一定的聯繫，雖然他們並不得直接管轄該堂區，如干涉其事務或財政等<ref>天主教法典，0357條第1項</ref>，但是該堂區聖堂內會掛上該位樞機的牧徽或照片以表示他與該聖堂的關聯。而該位司鐸級或執事級樞機被任命後也必須到他被交託的堂區聖堂參與任命為領銜司鐸或執事的儀式<ref name="輔神" />。

現存的領銜堂區有143處<ref>{{cite web |title = Cardinal Titles |url = http://www.gcatholic.com/hierarchy/cardinals-title-c2.htm |publisher=Gcatholic.com |accessdate=2012-09-14}}</ref>，領銜執事區則有69處<ref>{{cite web |title = Cardinal Deaconries |url = http://www.gcatholic.com/hierarchy/cardinals-title-c3.htm|publisher=Gcatholic.com}}</ref>。

相較於司鐸級樞機被交託領銜堂區，主教級樞機則給予了羅馬近郊的七個教區的作為領銜教區，其中首席樞機以奧斯底亞教區為銜，並兼任他被選為首席前所領的教區銜，因此實際上的領羅馬近郊教區銜的樞機只有六位。至於以[[東儀天主教|東方禮]]的宗主教的身分而被拔擢為樞機者為主教級樞機甚至是樞機團中的特例，他們直接以他們所管轄的教區作為他們的頭銜<ref name="輔神" /><ref>《[[天主教法典|天主教法典]]》，0350條第3項</ref>，現在有3位<ref>{{cite web |title=Cardinal-Bishops|url=http://www.gcatholic.com/hierarchy/cardinals-title-c1.htm |publisher=GCatholic}}</ref>。
