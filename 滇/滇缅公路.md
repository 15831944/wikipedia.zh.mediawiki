'''滇緬公路'''，前称'''昆畹公路'''，起點在[[中華人民共和國|中國]][[雲南省|雲南省]][[昆明市|昆明市]]，終點是[[緬甸|緬甸]][[臘戍|臘戍]]，全長1,453[[公里|公里]]，公路始建於1938年[[春季|春]]，於1938年[[12月|12月]]初步建成通車，以後陸續加以修改，是[[中國抗日戰爭|中國抗日戰爭]]時期，中國西南後方的一條歷時最久，運量最大的國際通道，有力地支援了中國抗日戰爭的初期。
[[File:US_convoy_on_the_24_turns.jpg|thumb]]的代表照。实际上该处公路被叫做“[[「二十四道拐」抗戰公路|二十四拐]]”，位于贵州省[[晴隆县|晴隆县]]县城南郊1公里，属于[[贵昆公路|贵昆公路]]。而滇缅公路终点是昆明。援华物资经滇缅公路到达昆明后，还需进入贵州省经由“二十四拐”，才能最终运往贵阳市，再经[[川黔公路|川黔公路]]运往陪都重庆和抗日前线。]]

== 简介 ==
滇緬公路在[[中國|中國]]境內的部分属于'''[[320國道|320國道]]'''，從[[昆明|昆明]]起點至[[瑞麗|瑞麗]][[畹町口岸|畹町口岸]]出境，中國境內全長959公里（现长度為850公-{里}-）。

出口岸至終點[[緬甸|緬甸]][[臘戍|臘戍]]為603公-{里}-。然后通过{{tsl|en|Northern Shan State Railway|北禅邦铁路}}由[[臘戍|臘戍]]至[[曼德勒|曼德勒]]；再经缅甸中央铁路由曼德勒至[[仰光|仰光]]海港。

== 修築背景 ==
1937年，中國的抗日戰爭爆發後，日軍很快就占領了中國[[華北|華北]]、[[華東|華東]]、[[華南|華南]]地區，僅剩以[[香港|香港]]和[[越南|越南]][[海防|海防]]等第三國港口轉運之通路，其餘直屬國民政府的[[港口|港口]]大都落入了日軍的手中。為了避免剩餘的兩條轉口線被封閉，因此中國應該要有一條後方的國際通道。居於此考慮，1937年8月，當時的[[雲南省|雲南省]]主席[[龍雲|龍雲]]向[[蔣中正|蔣中正]]提出「修改滇緬公路的計劃」，建議各修一條通往[[印度洋|印度洋]]的[[鐵路|鐵路]]和[[公路|公路]]，蔣介石非常贊同。1937年10月，[[國民政府|國民政府]]官與雲南省政府協商，確定滇緬公路由[[昆明|昆明]]經[[下关镇|下關]]、[[保山|保山]]、[[龍陵|龍陵]]、[[芒市|芒市]]、[[畹町|畹町]]出境，在[[緬甸|緬甸]]的[[臘戍|臘戍]]與该国鐵路网接通。11月，[[雲南省政府|雲南省政府]]主席[[龍雲|龍雲]]下達命令立即修築滇緬公路<ref name="駝">{{cite book |editors=湯漢清、邵貴龍主編 |year=2005 |title=《駝峰：1942-2002》 |location=[[昆明市|昆明市]] |publisher=雲南人民出版社 |isbn=7-222-04443-4 }}</ref>{{rp|16}}。

== 工程開工 ==
1937年末，中國向國際招標工程，約有數家國際性大型公司對此有興趣，表示若中國能提供先進設備，於六、七年內便可開通，但當時中國缺乏先進機械，更無六、七年時間等待，遂決定自行修建。同時開工的還有緬甸境內從仰光至[[臘戍|臘戍]]之臘仰鐵路，按計畫，物資由仰光港口卸下後由鐵路運抵臘戍，再由滇緬公路內運。

1938年1月，滇緬公路總工程處成立。公路的測量工作在極其艱苦的條件下開展。由於當時缺乏受過訓練的專業技術人員，在滇緬公路總工程處中對因戰禍遷往[[昆明|昆明]]的一些無業知識青年進行速成培訓。這批青年後來成了滇緬公路的技術骨幹。

臨近1938年[[春節|春節]]，公路沿線的20萬各族勞工被征集到工地。這些勞工大部分是老人、小孩和婦女。由於當時機械缺乏，他們只能依靠手工作业。很多艱苦、危險的工作幾乎都是由婦女、兒童來完成。8月，滇緬公路全線通車<ref name="駝"/>{{rp|25}}。原定三個月初步通車的工程幾經推遲，終於在1938年11月实现初步通車，1938年10月國民政府成立直屬交通部之滇緬公路管理局負責公路營運，運輸業務則是由「軍事委員會西南進出口物資運輸總經理處」（西南運輸處）承攬，該機構在1941年改組為中緬運輸總局。

== 太平洋战争爆发前 ==
1940年6月下旬，法国战败投降，日本与德國傀儡政權[[维希法國|维希法國]]签署协定，9用派兵進駐[[法屬印度支那|法屬印度支那]]並封闭了[[滇越铁路|滇越铁路]]，滇緬公路成為重慶政府最重要的聯外通道。

滇緬公路在通車初期每月输入中国物资4,000吨，隨著運輸管理改善、採購新車及中途站點逐漸完遂，運能也迅速提升。滇緬公路的主要使用單位是西南運輸處，該機構下配備了3,300輛運輸貨車，貨車分為21個大隊，每個大隊約有180-200輛貨車，車輛駕駛除了自訓外，還自東南亞招募雇用具有相關駕訓保修知識的[[海外華人|海外華人]]成立兩個華僑大隊。<ref>[http://221.237.164.10:8081/wsxt/readbookcontent.do?docid=107778&showtype=content 战时西南运输回顾 - 四川文史</ref>；至1941年底，滇緬公路計有7,580輛車營運；單月運能最大的紀錄是在1941年11月，該月自緬甸運入1.75萬噸物資。<ref>[http://history.sina.com.cn/bk/kzs/2015-11-06/1319127976.shtml 滇缅公路：西南补给大动脉]</ref>整年度運能紀錄中，1939年運貨噸位27,980噸、1940年61,934噸、1941年132,193噸。<ref>[http://mz.ynlib.cn:1309/dmgl/ShowArticle.asp?ArticleID=80 滇缅公路与中国抗战]</ref>

1940年6月20日，[[外務大臣_(日本)|日本外相]]{{tsl|en|Hachirō Arita|有田八郎}}向[[英國駐日大使館|英国驻日大使]]{{tsl|en|Robert Craigie (diplomat)|克莱琪}}正式提出关闭滇缅公路的通牒。克莱琪于6月25日致电英国外交部主张对日妥协。日军在[[深圳|深圳]]集结5000名陆军备战演习，港英防务压力吃紧。7月7日，日本政府对克莱琪下达最后通牒，限期十天内达成协议。1940年7月10日，英国丘吉尔战时内阁正式做出决定关闭滇缅公路。1940年7月17日在美国默许下（当时英国主要靠美国《租借法案》支撑对德战争，美国务卿赫尔表示“有点遗憾，但不表示异议”），英国与日本签订了《英日关于封闭滇缅公路的协定》及秘密备忘录，滇緬公路在換約後封閉3個月。至此，中国对外运输通道仅剩下大西北的到苏联的公路。此事件也称作“[[远东慕尼黑|远东慕尼黑]]”。

由於日本在1940年8月1日發表[[大東亞共榮圈|大東亞共榮圈]]構想，1940年9月初，日本与维希政府达成协议，日军进驻印度支那北部与滇越、桂越国界。1940年9月27日《[[三國同盟條約|三國同盟條約]]》在柏林签订，日本正式參加[[軸心國|軸心國]]。這些政治表態英國認定日本在英國最危急之際背信忘義，還有改變亞洲勢力的意圖，因此在1940年10月14日英國駐國民政府大使告知英國政府政策調整的意向，在1940年10月底滇緬公路重新開放。

為保護滇緬公路運輸少受日軍空襲，1941年4月15日，中美達成一項秘密協議，並由[[美國總統|美國總統]][[富蘭克林·德拉諾·羅斯福|富蘭克林·德拉諾·羅斯福]]簽署行政命令，允許美國預備役軍人和陸軍航空隊，以及海軍和海軍陸戰隊組「退役人員」前往中國參加戰鬥，並同意中國利用美國「[[租借法案|租借法案]]」貨款給中國資金來購買美國戰鬥機；7月10日，第一批美國戰鬥機駕駛員自願從美國[[舊金山|舊金山]]啟程前往中國昆明參戰，8月1日[[陳納德|陳納德]]在昆明設立[[飛虎隊|美國志願航空隊]]總部<ref name="駝"/>{{rp|56-57}}。

== 中国远征军時期 ==
1941年12月23日，中英在[[重庆|重庆]]签署《中英共同防御滇缅路协定》，中英[[军事同盟|军事同盟]]形成，中華民國为支援英军在滇缅（时为英属地）抗击[[日本|日本]][[法西斯|法西斯]]、并为了保卫中国西南大后方，组建了[[中国远征军|中国远征军]]。这是中華民國与盟国直接进行军事合作的典范，也是[[甲午战争|甲午战争]]以来中国军队首次出国作战，立下了赫赫战功。从中華民國军队入缅算起，中缅印大战历时3年零3月，中国投入兵力总计40万人，伤亡接近20万人。

1942年初，日軍第5飛行師團出動54架轟炸機，轟炸滇緬公路沿線重鎮[[保山市|保山]]，不到1個月，造成5萬餘人死亡<ref name="駝"/>{{rp|52}}。

== 現狀 ==
1966年，中國境內的滇緬公路段稱為昆畹公路，並不斷整治和改建，現在是云南境內重要的國道干線。目前，云南境內的滇緬公路將實現全線高速化。[[昆明|昆明]]至[[龍陵|龍陵]]段現已建成[[高速公路|高速公路]]，属于G56[[杭瑞高速|杭瑞高速]]的一部分。
