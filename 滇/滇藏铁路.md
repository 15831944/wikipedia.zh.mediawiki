'''滇藏铁路'''，指从[[云南|云南]][[昆明|昆明]]至[[西藏自治区|西藏自治区]][[拉萨|拉萨]]的一条规划建设中铁路，沿线经[[大理|大理]]、[[丽江|丽江]]、[[香格里拉市|香格里拉]]、[[林芝|林芝]]，全长1573.3千米。正线数目为单线。其中，邦达到拉萨段与川藏铁路共线。昆明至丽江为既有铁路，已经通车。而丽江至香格里拉段（即[[丽香铁路|丽香铁路]]，原计划2018年建成通车），由于途径地域地质条件复杂，施工难度大，现在正在施工，通车尚需时日。香格里拉到邦达段为国家于2016年新规划铁路，尚未开工建设。滇藏铁路行经地形复杂的大山、峡谷，横跨怒江、澜沧江、金沙江，与[[青藏铁路|青藏铁路]]、[[川藏铁路|川藏铁路]]、[[新藏铁路|新藏铁路]]以及[[甘藏铁路|甘藏铁路]]共同入围进藏铁路的规划线路<ref>{{cite web|url=http://www.yn.xinhuanet.com/newscenter/2006-08/09/content_7732655.htm|date=2014-01-26|title=滇藏共谋滇藏铁路|publisher=新华网|deadurl=yes|archiveurl=https://web.archive.org/web/20160304105145/http://www.yn.xinhuanet.com/newscenter/2006-08/09/content_7732655.htm|archivedate=2016-03-04}}</ref>，但由于难度较大延后了建设日程。滇藏铁路一旦建成，将为西藏开通东南出海口，对西藏经济的快速发展有促进作用。<ref>{{cite web|url=http://news.sina.com.cn/c/2006-08-07/07579676971s.shtml|date=2014-01-26|title=中国未来要建滇藏铁路|publisher=新浪网}}</ref>

==沿途地质==
云南省与西藏自治区地界相连，由于地质结构复杂，两地之间尚未开通铁路。地质结构复杂之处在于从云南大理到西藏林芝段之间属于横断山区，该山区为印度洋板块和亚欧板块碰撞挤压地带，地质结构极其不稳定，形成了地质断裂带及强地震带。 进一步说，沿途地质断裂带构造过度发育，与地下水和风化强度大等不利因素相组合，形成大范围分布的松散破碎的不稳定围岩。

==铁路规划==
国务院2004年通过的《中长期铁路网规划》中明确“从云南入藏的滇藏线仍继续做好地质调查和技术经济分析，是否建设视研究论证结果再定”。2016年，[[国家发展改革委|国家发展改革委]]、[[交通运输部|交通运输部]]和[[中国铁路总公司|中国铁路总公司]]共同修订和发布了修订版《中长期铁路网规划》<ref>{{cite web |title=关于印发《中长期铁路网规划》的通知 |url=http://www.ndrc.gov.cn/zcfb/zcfbtz/201607/t20160720_811696.html |publisher=国家发展和改革委员会 |accessdate=2016年7月13日}}</ref>，规划明确了“完善进出西藏、新疆通道。建设川藏铁路雅安～昌都～林芝段、滇藏铁路香格里拉～邦达段”<ref>{{cite web |title=中长期铁路网规划 |url=http://www.ndrc.gov.cn/zcfb/zcfbtz/201607/W020170213333938328309.pdf |publisher=国家发展和改革委员会 |accessdate=2016年7月13日}}</ref>等内容。由此，滇藏铁路形成完整的路线规划；同时，滇藏铁路大规模建设提上日程。

==线路分段==

'''昆广段'''：滇藏铁路昆明到广通段与现有成昆铁路共线，总计153千米，已开通运营多年。<br>

'''广大段'''：从广通到达大理，途经楚雄、南华、祥云、弥渡，长度206千米，1998年建成通车。<br>

'''大丽段'''：即[[大丽铁路|大丽铁路]]，铁路全长164公里，南起大理东站，沿洱海东侧北行，经鹤庆抵丽江，该段2004年开工，2009年建成通车。<br>

'''丽香段'''：即[[丽香铁路|丽香铁路]],铁路全长140公里，自丽江站引出，向北经虎跳峡、小中甸至迪庆州香格里拉县。 2009年5月，国家发改委批复同意开展丽香铁路建设。2014年7月，丽香铁路正式宣布开工。目前，该段铁路正在建设，尚未通车。<br>

'''香邦段'''：从迪庆州香格里拉县到达西藏自治区八宿县邦达镇。香邦段始于香格里拉，向西北经云南德钦县，西藏芒康县、左贡县，于八宿县邦达镇与拟建的川藏线康定至林芝段相连，线路全长461公里，其中云南境内185公里。2017年12月，中铁二院完成该段可行性研究。滇藏铁路香邦段建设规模为I级单线客货共用电气化铁路，桥隧比90%，设计时速160公里，设计建设工期8年<ref>{{cite web |title=滇藏铁路香格里拉至邦达段建设推进正在进行，将连接拉萨丽江 |url=https://www.thepaper.cn/newsDetail_forward_2038872|publisher=澎湃新闻  |accessdate=2018-03-23}}</ref>。该段开工日期尚未明确。<br>

'''邦林段'''：即邦达到林芝铁路。该段为[[康林铁路|康林铁路]]一部分。[[川藏铁路|川藏铁路]]与滇藏铁路在邦林段共线。预计[[康林铁路|康林铁路]]将于2019年完成可行性研究<ref>{{cite web |author1=王眉灵 |title=川藏铁路中间段有望明年确定线路走向 将建世界最大跨度最大高度铁路桥 |url=http://www.sc.gov.cn/10462/12771/2018/4/3/10448240.shtml |publisher=四川省人民政府网站 |accessdate=2018年04月03日}}</ref>，具体开工日期尚未确定。<br>

'''拉林段'''：即[[拉林铁路|拉林铁路]]。该段为[[川藏铁路|川藏铁路]]一部分（因为川藏铁路与滇藏铁路此段共线运营，又是滇藏铁路的一部分），拉林铁路即西藏林芝-西藏拉萨市，已于2014年底开工。<br>

==滇藏铁路意义==
'''政治意义'''：西藏自元代以来就纳入中国版图，成为中国不可分割的一部分。然而，中央政府对西藏的管理控制程度并不一样。一般认为，元朝和清朝鸦片战争之前中央政府对西藏又比较强的控制；明朝对西藏的控制比较弱（但是，西藏也没有脱离中国统治）。从鸦片战争到辛亥革命这段期间，西藏上层在英帝国主义的挑拨下有了独立倾向。中华民国成立之初，西藏地方政府甚至曾经一度宣布独立。当时的北洋政府拒绝承认西藏独立<ref>{{cite web |title=西藏独立运动 |url=https://zh.wikipedia.org/wiki/西藏独立运动}}</ref>。<br>

因此，加强对西藏的管理是历来为中央政府看重。中央政府通过扶持当地经济，开通前往西藏的公路等方式，加强对藏区的管理。需要特别说明的是，青藏铁路开通后，中央政府对西藏的管理能力大大增强。滇藏铁路开通后，中央对西藏的管理能力会再次增强。<br>

'''军事意义'''：交通运输困难导致中国西南边防后勤保障严重不足。后勤问题是1962年中印边境冲突时中方撤军的重要原因<ref>{{cite web |title=中印边境战争 |url=https://zh.wikipedia.org/wiki/中印边境战争}}</ref>。青藏铁路开通后，中国西藏驻军的后勤得到极大的改善。滇藏铁路开通后，中国西南部队后勤会进一步改善，中国西南边防形势会进一步增强。<br>

'''经济意义'''：滇藏公路沿线旅游资源丰富。大理、丽江和香格里拉都是世界著名的旅游胜地。由于交通限制，这些地区旅游开发程度不足。滇藏铁路开通后，沿途各地旅游资源会有更大程度的开发。西藏也可以得到极大实惠，川藏、滇藏铁路开通后，沿途居民出行更方便，同时铁路会带动当地经济的发展。同时，西藏是很多中外电影中的圣地，很多人对西藏充满了向往。滇藏铁路开通后，西藏旅游资源会得到进一步的开发。
