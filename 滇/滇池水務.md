'''昆明滇池水務股份有限公司'''（簡稱'''滇池水務'''）（{{lang-en|'''Kunming Dianchi Water Treatment Co., Ltd'''}}，{{hkex|3768}}）成立於2010年12月23日，是[[雲南省|雲南省]]最大的市政污水處理商。滇池水務的前身為[[昆明市|昆明市]]政府於1989年設立的「昆明市第一水質淨化廠」，並於2004年由昆明滇池投資有限責任公司接管。公司業務覆盖[[污水处理|污水处理]]、[[自来水|自来水]]供应、[[再生水|再生水]]利用、水处理技术研究及应用、污染水体生态保护治理，垃圾处理、污泥资源化利用等固废处置等。現任董事長為郭玉梅<ref name="introduction">{{Cite web |url = http://www.kmdcwt.com/Profile.html |title = 公司簡介 |author = 昆明滇池水務股份有限公司 |language = zh-hans | accessdate = 2017-10-28}}</ref>。

股份在2017年4月6日於[[港交所|港交所]][[主板|主板]]上市。招股價介乎3.91至4.7港元，發售股份數目為339,430,000股H股，而集資額介乎13.27至15.95億港元，用作於雲南省從事污水處理及再生水服務<ref>{{cite web|title=昆明滇池水務股份有限公司招股資料|url=http://www.hkexnews.hk/listedco/listconews/SEHK/2017/0324/LTN20170324024_C.pdf|accessdate=2017-10-28|date=2017-03-24|language=zh-hant}}</ref><ref>{{cite web|author=[[中國泛海金融|中國泛海金融]]|title=滇池水務(03768)明日招股 入場費4747.36元|url=http://www.quamnet.com/newscontent.action?articleId=5125399|accessdate=2017-10-28|date=2017-03-23|language=zh-hant}}</ref><ref>{{cite web|title=昆明滇池水務股份有限公司招股公告|url=http://www.hkexnews.hk/listedco/listconews/SEHK/2017/0324/LTN20170324018_C.pdf|accessdate=2017-10-28|date=2017-03-24|language=zh-hant}}</ref>。

== 歷史 ==
[[昆明市|昆明市]]政府於1989年成立了「昆明市第一污水處理廠」（及後更名為昆明市第一水質淨化廠）並投入營運。這是中國西南最早的污水處理廠之一，當時設計的處理能力為5.5萬立方米/日<ref name="history">{{cite web|title=昆明滇池水務股份有限公司歷史、發展及公司架構|url=http://www.hkexnews.hk/listedco/listconews/SEHK/2017/0406/a10161/CKDWTCL-20170210-16.PDF|accessdate=2017-10-29|date=2017-04-06|language=zh-hant}}</ref>。

其後於1996年，[[昆明市城市排水有限責任公司|昆明市城市排水有限責任公司]]從[[昆明市|昆明市]]政府接管了管理及經營污水處理的業務，並在1996年到2003年期間與市政府共同建成了第二至第六水質淨化廠<ref name="history"></ref>。

2004年，公司現時的控股股東[[昆明滇池投資有限責任公司|昆明滇池投資有限責任公司]]成立並作為國有獨資企業接管了第一至第六水質淨化廠。當時設計的處理能力增至55.5萬立方米/日。第七及第八水質淨化廠亦於2009年建成並投入營運<ref name="history"></ref>。

2010年，滇池水務正式成立，並自控股股東接管了污水處理業務，成為昆明獨家污水處理股務供應商。當時設計的處理能力增至110.5萬立方米/日<ref name="history"></ref>。

2014年，公司收購並獲得了施甸污水處理廠的特許經營權，並投資建設[[施甸縣|施甸縣]]第二自來水廠項目，代表滇池水務正式將業務拓展至[[昆明|昆明]]以外的[[雲南省|雲南省]]地區<ref name="history"></ref>。

2015年，滇池水務完成了[[浙江|浙江]][[諸暨|諸暨]]污水項目，同年訂立了[[老撾|老撾]][[金三角|金三角]]污水及自來水項目合同，令公司業務拓展至[[華東地區|華東地區]]及[[老撾|老撾]]<ref name="history"></ref>。

==業務板塊==
===污水處理===
滇池水務於2011年獲得了[[昆明市|昆明市]]行政區域30年污水處理特許經營權，是雲南省污水處理規模最大的企業<ref name="introduction"></ref>。

截至2017年6月30日，公司共有26間污水處理廠已投入營運，日總污水處理能力為160萬立方米。其污水處理能力的90.8%達到[[水質#.E4.B8.AD.E5.9C.8B.E5.A4.A7.E9.99.B8|國家一級A類排放標準]]<ref name="2017semireport">{{cite web|author1=昆明滇池水務股份有限公司|title=2017中期報告|url=http://www.kmdcwt.com/Public/Upload/article/day_170828/201708282217192728.pdf|accessdate=2017-10-28|date=2017-08-28|language=zh-hant}}</ref>。

===再生水===
滇池水務擁有[[昆明市|昆明市]]主城區內唯一的再生水輸配系統，水質達到《國家再生水水質標準》<ref name="introduction"></ref>。公司現時共有6間再生水廠生產[[再生水|再生水]]，日總設計產能達4.4萬立方米。客戶主要為[[昆明市|昆明市]]的工商業機構及企事業單位<ref name="2017semireport"></ref>。

==外地業務==
===老撾===
公司首先於2015年10月與[[老撾|老撾]][[金三角|金三角]]特區簽訂了有關自來水廠及污水處理廠的[[BOT模式|BOT協議]]，其後又於2017年9月簽訂了一份全面合作框架協議。根據協議，滇池水務計劃在未來五年內投資3至5億元人民幣<ref>{{Cite web |url = http://www.kmdcwt.com/article374.html |title = 老挝北部合作工作组第八次会议并与老挝金三角经济特区签订全面合作框架协议 |author = 昆明滇池水務股份有限公司 |language = zh-hans | accessdate = 2017-10-28|date=2017-09-29}}</ref>。污水處理項目有望預計將於2018年2月[[木棉節|木棉節]]竣工，設計規模為遠期1.2萬立方米/日<ref name="article382">{{Cite web |url = http://www.kmdcwt.com/article382.html |title = 滇池水务老挝项目喜获最美澜湄合作生态奖 |author = 昆明滇池水務股份有限公司 |language = zh-hans | accessdate = 2017-10-28|date=2017-10-26}}</ref>。

2017年10月，公司的污水處理廠工程獲得了「老撾金三角經濟特區最美瀾湄合作生態獎」<ref name="article382"></ref>。
