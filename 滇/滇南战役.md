'''滇南战役'''，是1949年12月27日至1950年2月19日，发生在中国[[云南|云南地区]]，[[中国人民解放军|中国人民解放军]]同[[中华民国国军|中华民国国军]]进行的一场战役。解放军最终占领[[西双版纳|西双版纳]]，控制云南全境<ref>{{Cite book|title=中国人民解放军军史大辞典|last=[[郭化若|郭化若]]主编|first=|publisher=长春：[[吉林人民出版社|吉林人民出版社]]|year=1993.07|isbn=7-206-01814-9|location=|pages=267}}</ref>。

== 背景 ==
1949年12月9日，中华民国[[雲南省主席|云南省主席]][[卢汉|卢汉]]率第74、第93军于[[卢汉云南起义|昆明起义]]，云南宣告和平解放。但当时云南的国民党势力仍然存在，国军第8、第26军试图进犯[[昆明|昆明]]，与[[西昌县_(清朝)|西昌]]国军相呼應。对此，蒋中正十分赞赏，即派陆军副总司令[[汤尧|汤尧]]从[[台湾|台湾]]飞抵[[云南|云南]]，将上述两个军组编为第8兵团，由汤尧兼任司令。12月15日，国军第8兵团两个军分别由[[沾益|沾益]]、[[宜良|宜良]]两地前出，并于18日对昆明发起进攻。中国人民解放军第二野战军即令滇桂黔边纵队和卢汉起义部队，进行[[昆明保卫战|昆明保卫战]]。同时，令解放军第五兵团之第49师，乘汽车由滇西向滇东北驰援。22日，解放军第49师进至[[曲靖|曲靖]]、沾益地区歼灭国军一部。国军遂放弃对昆明的进攻，于12月下旬南撤并集结于与[[越南|越南]]、[[老挝|老挝]]、[[缅甸|缅甸]]接壤的边境地区。国军26军位于[[蒙自|蒙自]]、[[箇舊|箇舊]]地区，第8兵团部及第8军位于[[建水|建水]]、[[石屏|石屏]]地区，企图控制[[蒙自机场_(二战时期)|蒙自机场]]及通往国境的道路，伺机由空中或陆路逃往国外或台湾<ref>{{Cite book|title=邓小平主政大西南|last=吴得民主编|first=|publisher=北京：光明日报出版社|year=2012.12|isbn=7-5112-3885-8|location=|pages=50-53}}</ref>。

中共中央军委提出的以远距离迂回包围消灭国军残余的作战方针。到1949年12月，[[第四野战军|第四野战军]]和第二野战军一部，分别由湖南、江西、广东向广西境内进击，结束了围歼[[白崇禧|白崇禧]]部的[[广西战役|广西战役]]。同时，[[中国人民解放军第二野战军|第二野战军]]主力和[[第一野战军|第一野战军]]、第四野战军及[[华北野战军|华北野战军]]各一部，分别从鄂西、湘南、陕南出击，进行了围歼四川境内[[胡宗南|胡宗南]]部的[[成都战役|成都战役]]。这两次战役使白崇禧、胡宗南部队撤逃云南的企图成为泡影，为进行滇南战役创造了良好条件<ref>{{Cite book|title=开国大将|last=王晓建主编|first=|publisher=北京：中国社会出版社|year=2005.07|isbn=7-80088-611-5|location=|pages=261}}</ref>。

== 准备 ==
为歼灭滇南汤尧集团，中共中央军委决定组织滇南战役。除令入滇驰援昆明的第49师暂停前进外，又令位于广西[[百色|百色]]、[[南宁|南宁]]等地的第四野战军[[第38军|第38军]]主力第114师、第151师和第二野战军第4兵团之[[第13军|第13军]]，由第4兵团首长统一指挥，立即向云南进军。第4兵团决心采取远距离迂回包围。具体部署：以第13军（第37、第38师为第一梯队，第39师为第二梯队）经[[百色|百色]]、[[富宁县|富宁]]、[[砚山|砚山]]、[[文山市|文山]]向[[开远|开远]]、[[蒙自|蒙自]]方向兼程急进，首先夺占蒙自机场，断国军空中逃路；以第114、第151师及滇桂黔边纵队第1支队经富宁、支山直插[[马关县|马关]]、[[河口瑶族自治县|河口]]、[[金平苗族瑶族傣族自治县|金平]]，切断国军陆上逃路；滇桂黔边纵队以积极的行动配合主力作战，聚歼国军于云南境内<ref name=ok1>{{Cite book|title=中国人民解放军著名战役战斗 第4卷 社会主义革命和建设时期|last=[[周晓鹏|周晓鹏]]主编；袁静伟，刘胜如，贾俊明分卷主编|first=|publisher=北京：[[蓝天出版社|蓝天出版社]]|year=2013.07|isbn=7-5094-0877-6|location=|pages=46-50}}</ref>。

1950年1月1日，各参战部队相继按指定方向兼程急进。左路第114、第151师和边纵第1支队出发后，沿中越边境山岳丛林地带连续疾进500余公里，先头于11日进至河口地区，控制了国军向国外逃跑的陆上主要通道；右路第13军一梯队第37、第38师出发后，行程900余公里，先头于13日进至砚山及其以东地区<ref>{{Cite book|title=巍巍长城 纪念中国人民解放军建军七十周年|last=沈阳军区政治部编研室编|first=|publisher=沈阳：[[白山出版社|白山出版社]]|year=1997.06|isbn=7-80566-536-2|location=|pages=266-267}}</ref>。至此，解放军部署己基本形成<ref name=ok9 />。

== 过程 ==
为了防止被歼，[[汤尧|汤尧]]集团开始做逃跑的准备。国军第26军全部退集蒙自地区，利用蒙自机场开始向国外空运眷属和重要物资，并在蒙自东北的[[鸣鹫镇|鸣鹫街]]派出1个团的警戒部队掩护空运。国军第8军也似有东移迹象。为防国军逃跑，解放军第37师第一梯队先在不等主力到达前，即大胆迂回，迅速隐蔽地绕过国军警戒部队，于15日夜，突然向蒙自机场发起攻击。至次日清晨，占领了机场，国军第92师一部被灭，1200余人被俘，缴获飞机2架、战防炮和山炮11门，及大量枪支弹药和待运军用物资，切断了国军空中逃路。这时，解放军第37、第38师主力陆续赶到，首先攻占[[蒙自|蒙自]]县城，尔后由第38师派出2个团，在边纵部队第1支队1个团协同下，向蒙自东北进击，围歼了鸣鹫街国军警戒团，继而转兵向西继续进击，于17日在开远东侧之大庄地区歼灭国军一部<ref>{{Cite book|title=云南省志 卷43 中共云南省委志|last=云南省地方志编纂委员会总纂；中共云南省委员会办公厅编撰|first=|publisher=昆明：[[云南人民出版社|云南人民出版社]]|year=2000.10|isbn=7-222-02580-4|location=|pages=198-200}}</ref>。与此同时，解放军第114师在边纵第1支队1个团协同下，突然攻占了[[红河|红河]]下游的主要渡口[[蔓耗镇|曼耗]]，歼灭国军4个工兵连，控制了浮桥，并乘胜向红河上游地区机动；第151师解放了曼耗以东的屏边，主力进至蒙自南侧，完全切断了国军南逃出境的陆上通道，并与右路解放军形成了对开远、箇舊、建水国军的钳击之势。解放军这一突然行动，取得了战役的主动权，为全歼滇南地区国军创造了良好条件<ref name=ok1 /><ref>{{Cite book|title=中国大百科全书 军事 1|last=中国大百科全书出版社编辑部编|first=|publisher=北京：[[中国大百科全书出版社|中国大百科全书出版社]]|year=1989.05|isbn=7-5000-0242-4|location=|pages=182}}</ref>。

18日，国军第26军开始向西逃窜，第8兵团部及第8军则向南退逃。解放军第4兵团即令各参战部队抓住战机，大胆追击围歼国军。截至19日，解放军各追击部队先后推进至箇舊、[[鸡街镇_(箇舊市)|鸡街]]、金平及金平以北地区与国军主力接触，尔后大胆楔入国军纵深实施迂回包围，歼灭国军第26军军直、第93师、第161师各一部，第193师残部和第8军第237师大部。国军第8军副军长[[田仲达|田仲达]]见逃跑无望，在[[建水|建水]]以东率部2000余人向解放军边纵部队第10支队投诚<ref name=ok1 />。

21日，解放军进占建水、石屏，控制了[[蒙宝铁路|箇舊至石屏铁路]]全线。这时，国军司令汤尧率其兵团部和第8军主力第42、第170师、教导师和第3师1个团近2万人，向[[石屏|石屏]]西南[[元江哈尼族彝族傣族自治县|元江]]方向逃跑。鉴于石屏至元江一带高路窄，不适宜解放军大部队作战，第四兵团以营或团为单位，大胆对解放军进行多路追击。解放军第37师之第109团第2营，追击至营盘山附近地区，与国军后卫部队接触，为防止国军脱逃，该营以1个连伪装国军，利用夜暗抢占了营盘山制高点，将国军拦腰斩断，尔后全营分路向国军出击，歼灭国军1700余人。此时，逃到元江附近地区的国军，在解放军边纵部队西进支队的截击下，除前卫第170师和教导师一部通过元江铁索桥西逃外，国军教导师主力和第3师第9团纷纷向解放军投诚<ref>{{Cite book|title=中国战典 下|last=中国人民革命军事博物馆编著|first=|publisher=北京：[[解放军出版社|解放军出版社]]|year=2008.08|isbn=978-7-5065-5675-0|location=|pages=1053}}</ref>。

22日下午，国军第8兵团司令汤尧率其兵团部和第42师逃至[[元江_(红河)|元江]]东岸地区，计划渡河西逃，解放军第109团第6连，依托有利地形进行了顽强阻击。23日，解放军担任追击和迂回的各路部队相继赶到，并立即对国军形成合围。24日拂晓，各部队对被围国军发起总攻，第8兵团司令汤尧率残部实施多次反扑和突围，但均被解放军击退。在穿插围歼战斗中，解放军第109团第2营1个班歼灭国军800余人；第110团第1、第2连，分别活捉了陆军副总司令汤尧和第8军军长[[曹天戈|曹天戈]]<ref name=ok9>{{Cite book|title=解放战争战略追击 西南地区|last=《中国人民解放军历史资料丛书》编委会编|first=|publisher=北京：解放军出版社|year=2006.03|isbn=7-5065-4999-9|location=|pages=658-670}}</ref>。在搜捕时，刚从昆明赶到的解放军边纵部队和卢汉起义部队也加入战斗，为继续追歼西逃残国军第170师等，解放军第13军由第37师师长和一名副师长各率一部兵力，在边纵部队第9支队配合下，于24日夜分南北两路乘胜向西进击<ref name=ok1 />。

经过8天的强行军，解放军于2月4日在[[镇沅彝族哈尼族拉祜族自治县|镇沅]]西北追上了国军，并连夜急进90公里，迂回到撤退中的国军前面，在[[勐大镇|勐统]]附近地区对撤逃国军形成包围。在解放军攻击之下，7曰，国军第170师师长[[孙进贤|孙进贤]]率残部2400余人向解放军投降。与此同时，解放军边纵第9支队和卢汉起义部队2个团，在[[墨江哈尼族自治县|墨江]]地区也迫使国军教导师残部1000人投降<ref name=ok9 />。2月16日，解放军执行南路追击任务的部队，在[[澜沧江|澜沧江]]以西临近中缅边境的[[勐海县|南桥]]地区，歼灭了国军第8兵团最后一股流散的官兵500余人。19日，解放军攻占[[打洛镇|打洛]]，进而占领整个[[西双版纳|西双版纳]]地区。至此，滇南战役胜利结束<ref name=ok34 /><ref>{{Cite book|title=军事大辞典|last=郑文翰主编|first=|publisher=上海：[[上海辞书出版社|上海辞书出版社]]|year=1992.12|isbn=7-5326-0219-2|location=|pages=1352}}</ref><ref>{{Cite book|title=当代云南简史|last=《当代云南简史》编委会编|first=|publisher=北京：[[当代中国出版社|当代中国出版社]]|year=2004.05|isbn=7-80170-291-3|location=|pages=41-42}}</ref>。

== 后续 ==
{{Main|泰缅孤军}}
在第八兵团被歼灭时，第八軍第二三七師709團長[[李國輝|李國輝]]率領約千人突圍逃出，但最後僅殘存600多人。3月，李國輝團長的600多人自[[雲南|雲南]]、[[廣西|廣西]]、[[貴州|貴州]]南部退至緬甸境內，在孟棒與第二十六軍第九十三師第二七八團副團長[[譚忠|譚忠]]的部隊約800多人會合，共約1500人，成为[[泰缅孤军|泰缅孤军]]最早的起源<ref>{{cite journal |author1=孙纯福 |title=金三角，蒋残军何以能留下6万后代？ |journal=文史天地 |date=2009 |issue=2 |url=http://history.people.com.cn/GB/205396/14484259.html}}</ref><ref>{{cite web |url  = http://www.cares.org.tw/s_2440_detail.asp?sn=502 |title=泰北孤軍簡介(上)|publisher=中華救助總會}}</ref>。
