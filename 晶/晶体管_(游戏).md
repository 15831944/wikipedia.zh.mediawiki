{{Vgname|晶体管|en=Transistor|又译“'''晶体剑'''”}}是一款科幻[[动作角色扮演游戏|动作角色扮演游戏]]。游戏由{{le|Supergiant Games|Supergiant Games}}开发，并于发行2014年首次发行。游戏对应[[Microsoft_Windows|Microsoft Windows]]、[[PlayStation_4|PlayStation 4]]、[[OS_X|OS X]]、[[Linux|Linux]]、[[iOS|iOS]]<ref>{{Cite web |url = http://supergiantgames.com/index.php/2014/04/transistor-arrives-may-20-on-ps4-and-pc/|title = Transistor arrives May 20 on PS4 and PC|first = Greg|last = Kasavin|date = April 10, 2014|accessdate = April 10, 2014|publisher = [[Supergiant_Games|Supergiant Games]]}}</ref><ref>{{Cite web |url = http://www.supergiantgames.com/blog/transistor-out-now-on-mac-and-linux/|title = Transistor Out Now on Mac and Linux!|first = Greg|last = Kasavin|date = October 30, 2014|accessdate = October 31, 2014|publisher = [[Supergiant_Games|Supergiant Games]]}}</ref><ref>{{cite web|last1 = Scalzo|first1 = John|title = Supergiant’s Transistor is now available on iOS App Store|url = http://www.warpzoned.com/2015/06/supergiants-transistor-is-now-available-on-ios-app-store/|website = Warp Zoned|date = June 11, 2015|accessdate = June 24, 2015}}</ref>。《晶体管》所有平台到2015年1月时售出60万套 <ref name="600k">{{cite web|url = http://www.gamespot.com/articles/bastion-sells-3-million-transistor-hits-600-000/1100-6424555/|title = Bastion Sells 3 Million, Transistor Hits 600,000|first = Eddie|last = Makuch|work = [[GameSpot|GameSpot]]|publisher = [[CBS_Interactive|CBS Interactive]]|date = 2015-01-09|accessdate = 2015-01-09|archiveurl = https://web.archive.org/web/20150110043631/http://www.gamespot.com/articles/bastion-sells-3-million-transistor-hits-600-000/1100-6424555/|archivedate = 2015-01-10|deadurl = no}}</ref>。游戏在2016年12月在Steam版本上增加了中文游戏内容。

本作的音乐是一大亮点，获得了2014年[[游戏大奖|游戏大奖]]和IGN 2014大奖中音乐相关奖项的提名。<ref>{{cite web|url=http://www.forbes.com/sites/erikkain/2014/12/06/all-the-winners-of-the-2014-game-awards/|title=All The Winners Of The 2014 Game Awards|last=Kain|first=Erik|date=December 6, 2014|website=Forbes|archiveurl=https://web.archive.org/web/20141214100049/http://www.forbes.com/sites/erikkain/2014/12/06/all-the-winners-of-the-2014-game-awards/|archivedate=December 14, 2014|deadurl=no|accessdate=December 14, 2014}}</ref><ref>{{cite web|url=http://www.ign.com/lists/best-of-games-achievement/music/transistor|title=Best Music|date=January 13, 2015|website=IGN|publisher=Ziff Davis|archiveurl=https://web.archive.org/web/20150212013614/http://www.ign.com/lists/best-of-games-achievement/music/transistor|archivedate=2015年2月12日|deadurl=no|accessdate=February 12, 2015}}</ref>
