'''晶圆'''（{{lang-en|Wafer}}）是指制作矽(硅)[[半导体|半导体]][[集成电路|集成电路]]所用的矽(硅)晶片，由于其形状为圆形，故称为晶圆。晶圆是生产[[集成电路|集成电路]]所用的载体，一般晶圆產量多為[[单晶硅|单晶硅]]圆片<ref name=Whatiswafer/>。

晶圆是最常用的[[半导体|半导体]]材料，按其直径分为4[[英寸|英寸]]、5英寸、6英寸、8英寸等规格，近来发展出12英寸甚至研發更大规格（14英吋、15英吋、16英吋、20英吋以上等）。晶圆越大，同一圆片上可生产的[[集成电路|集成电路]]（integrated circuit, '''IC'''）就越多，可降低成本；但对材料技术和生产技术的要求更高，例如均勻度等等的問題。一般認為矽(硅)晶圆的直径越大，代表著这座晶圆厂有更好的技術，在生产晶圆的过程当中，[[設備效率評價#良率(Quality)|良品率]]是很重要的条件<ref name=Whatiswafer>[http://technews.tw/2015/07/18/what-is-wafer/ 半導體產業的根基：矽晶圓是什麼？]</ref>。

== 製造過程 ==
[[File:Czochralski_Process.svg|thumb]]
{{see also|半导体器件制造}}
很簡單的說，首先由普通矽(硅)砂拉製[[提煉|提煉]]，經過[[溶解|溶解]]、[[提純|提純]]、[[蒸餾|蒸餾]]一系列措施製成單晶硅棒，單晶硅棒經過[[切片|切片]]、[[拋光|拋光]]之後，就得到了單晶矽(硅)圓片，也即晶圓。
#将[[二氧化硅|二氧化硅]]礦石（[[石英砂|石英砂]]）与焦炭混合后，經由電弧爐加热还原，即生成粗矽（纯度98%，冶金级）。SiO<sub>2</sub> + C = Si + CO<sub>2</sub> ↑ <ref>痞客帮 - [http://zhe09.pixnet.net/blog/post/50868359-半導體產業上游---矽晶圓產業 半導體產業上游---矽晶圓產業]</ref>
#[[鹽酸|鹽酸]]氯化並經蒸餾後，製成了高純度的[[多晶硅|多晶硅]](半导体级[[純度|純度]]11个9，[[太阳能|太阳能]]级7个9），因在精密電子元件當中，矽晶圓需要有相當的純度（99.999999999%），不然會產生缺陷。
#晶圓製造廠再以[[柴可拉斯基法|柴可拉斯基法]]將此多晶矽(硅)熔解，再於溶液內摻入一小粒的矽(硅)晶體晶種，然後將其慢慢拉出，以形成圓柱狀的單晶矽(硅)晶棒，由於矽(硅)晶棒是由一顆小晶粒在融熔態的矽(硅)原料中逐漸生成，此過程稱為「長晶」。這根晶棒的直徑，就是晶圓的直径。
#矽(硅)晶棒再經過切片、研磨、拋光後，即成為[[積體電路|積體電路]]工廠的基本原料——矽(硅)晶圓片，這就是「晶圓」（wafer）。
#晶圓經多次光罩處理，其中每一次的步驟包括感光劑塗佈、曝光、顯影、腐蝕、滲透、植入、[[蝕刻|蝕刻]]或蒸著等等，將其光罩上的電路複製到層層晶圓上，製成具有多層線路與元件的[[集成电路|IC]]晶圓，再交由後段的測試、切割、封裝廠，以製成實體的[[積體電路|積體電路]]成品。一般來說，大晶圓(12吋)多是用來製作記憶體等技術層次較低的IC，而小晶圓(6吋)則多用來製作類比IC等技術層次較高之IC，8吋的話則都有。

從晶圓要加工成為產品需要專業精細的分工。
