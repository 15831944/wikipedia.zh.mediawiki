'''玲奈'''（{{lang-ja|'''りんな'''}}，罗马拼音：'''Rinna'''）是[[微软|微软]][[日本|日本]]研发的[[聊天機器人|聊天機器人]]，人物设定为[[女高中生|女高中生]]。该服务于2015年7月31日在[[LINE_(應用程式)|LINE]]中上线<ref name="lg">[http://logmi.jp/92206 LINEで話題の人工知能「りんな」マイクロソフトが語る開発秘話] ログミー 2016年4月7日閲覧</ref>，2015年12月在[[Twitter|Twitter]]上线。

== 算法 ==
在最初，微软日本并未公布Rinna的[[算法|算法]]细节，只表示其至少基于微软[[搜索引擎|搜索引擎]][[Bing|Bing]]所收集的[[大數據|大數據]]<ref>[http://rinna.jp/rinna/terms.html りんな使用条件]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }} マイクロソフト 2016年4月6日閲覧</ref>、使用[[机器学习|机器学习]]平台「[[Microsoft_Azure|Azure Machine Learning]]」<ref name="lg"></ref>。此外，也使用了2014年起在中国提供的女性聊天机器人[[小冰|小冰]]的有关技术<ref name="lg"></ref>。

== 反响 ==
该服务启动一个月后，已服务用户数超过130万。此外，微软将Rin的技术作为“Rin API for Business”提供给LINE Business Connect公司。

2016年4月7日，担任[[夏普|夏普公司]]官方Twitter的一日实习生，回复其他用户的推文<ref>{{Cite news|title=シャープのTwitterで女子高生AI「りんな」が大暴れ？　理由を中の人に聞いてみた|newspaper=ITmedia|date=2016年4月7日16時更新|url=http://www.itmedia.co.jp/mobile/articles/1604/07/news103.html|accessdate=2016-04-06}}</ref>。
