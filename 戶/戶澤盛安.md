'''戶澤盛安'''（戸沢 盛安（とざわ もりやす），1566年(永祿9年)-1590年(天正18年)）[[日本戰國|日本戰國]]武將，是[[出羽國|出羽國]][[角館|角館]]的國人，後成為[[戰國大名|戰國大名]]。血統是[[桓武平氏|桓武平氏]]以[[平衡盛|平衡盛]]作祖先的——[[戶澤氏|戶澤氏]]，在世二十四年間與[[奧羽|奧羽]]地區各大勢力周旋，因智勇雙全被稱為'''「夜叉九郎」'''。戶澤氏在盛安成為當主年間，由一個地方莊領，成為大名。

== 早年 ==
戶澤氏是[[仙北郡|仙北郡]]著名豪族，由鐮倉時期便於當地發蹟。在盛安祖父[[户泽秀盛|秀盛]]年代，與北方的[[安東家|安東家]]開始激戰，被安東聯軍所敗，其父[[户泽道盛|道盛]]更是失去祖地，退守北游。因其祖母奮鬥，方抵禦了[[小野寺義道|小野寺義道]]的攻勢，最終成功收回了[[北浦郡|北浦郡]]和[[仙北郡|仙北半郡]]。在安東的壓力下，戶澤氏爆發內亂，親[[南部氏|南部氏]]的家臣擁立16歲的[[戶澤盛重|戶澤盛重]]為當主。戶澤氏至此被南部氏兼併，失去大名地位。

[[天正|天正]]6年（[[1578年|1578年]]），戶澤盛安代替體質虛弱的哥哥戶澤盛重，以13歲之齡繼承戶澤氏。天正10年（1582年），盛安元服，並重新奪回實權，致力於掌握家臣團及領國經營的安定化，穩定家中，並藉機爭取獨立。戶澤盛安為爭取[[雄物川|雄物川]]流域的支配權，與[[安東愛季|安東愛季]]統領的[[安東氏|安東氏]]1000兵激戰，雖擊敗安東卻也負傷。此外，也跟角館城南方的[[小野寺氏|小野寺氏]]爭奪[[大曲平野地區|大曲平野地區]]。1584年，盛安通過一撃即逃的方式襲擾小野寺氏，令小野寺誤判安東家再度南下攻擊，盛安由[[横澤谷|横澤谷]]取[[大仙口|大仙口]]夜襲[[角館城|角館城]]，一夜奪回父祖的城池。並依靠山城成功抵禦小野寺軍反攻，盛安開始逐步收復戶澤氏的舊領，向周邊小勢力出兵。

戶澤盛安是一員智勇出色的名將，得到『鬼九郎』或『'''夜叉九郎'''』等別名。每逢與他家開戰的時候，盛安雖然是總司令，但是經常站立在前線，經常單騎殺入敵人陣中。盛安對於俘虜的敵方士兵，總是採取釋放的做法，而非處決，盛安由此贏得和善的名聲。

== 唐松野之戰 ==
[[安東愛季|安東愛季]]知道仙北三郡的四家（[[小野寺氏|小野寺氏]]、[[戶澤氏|戶澤氏]]、[[本堂氏|本堂氏]]、[[六鄉氏|六鄉氏]]）素來不和，於是他拉攏盛安聯合進攻小野寺氏。盛安基於戶澤氏地理上將被安東氏包圍而拒絕。結果，安東氏藉此向戶澤氏大舉進攻。1587年4月，安東愛季手下[[嘉成播磨守|嘉成播磨守]]、[[鐮田河內守|鐮田河內守]]攻擊戶澤家[[淀川城|淀川城]]，並將之攻下。四月二日，安東愛季親率3000騎精銳集結于仙北[[唐松野|唐松野]]，對[[荒川城|荒川城]]發動攻擊，四月五日，戶澤盛安引1000騎由[[小松山|小松山]]出發赴援。此時遭到危脅的[[小野寺氏|小野寺氏]]和[[本堂氏|本堂氏]]，[[六鄉氏|六鄉氏]]也派出援軍加入戶澤方，是為[[唐松野之戰|唐松野之戰]]。戶澤盛安首先以突襲戰術主動攻擊安東軍，並討取安東方[[嘉成重盛|嘉成右馬頭重盛]]。連續5天的激戰，安東愛季突然病發，安東軍被逼撤退。戶澤軍死傷百多人，安東軍則是戶澤軍的三倍。戶澤氏重新奪回[[淀川城|淀川城]]，[[唐松野之戰|唐松野之戰]]以安東軍戰敗而完結。愛季不久病亡，享年49歲。

不久，由愛季生前統一的[[檜山安東氏|檜山安東氏]]和[[湊安東氏|湊安東氏]]宣告分裂。戶澤氏和南部氏乘機支持湊安東氏的[[安東通季|安東通季]]與愛季次子[[秋田實季|實季]]爭奪家督，虽然谋反失败，但是安東氏至此走向衰落。

== [[阿氣野合戰|阿氣野合戰]] ==
小野寺氏礙於受到安東氏的威脅，於是出兵緩助戶澤氏，以免唇亡齒寒。但兩家的內在矛盾並沒有因此而解決。唐松野之戰一個月後，戶澤氏突然出兵攻擊小野寺的[[上浦郡|上浦郡]][[沼館城|沼館城]]，小野寺氏雖然正和[[最上氏|最上氏]]對歭，依然派出2000援軍。阿氣野合戰爆發，戶澤氏聯合盟友[[楢岡氏|楢岡氏]]與小野寺氏開戰，小野寺方的[[小清水藏人|小清水藏人]]被戶澤盛安討取，戶澤氏乘勝攻陷[[沼館城|沼館城]]。此後，[[豐臣氏|豐臣氏]]已經平定[[九州|九州]]，並派出使者聯絡奧羽諸位大名，調停他們之間的戰爭，剛剛才壯大的戶澤氏再沒有大規模的軍事行動。

== 晚年 ==
天正7年（1579年），戶澤家致贈鷹和名馬給[[織田信長|織田信長]]，已建立外交邦誼。天正18年（1590年），[[豐臣秀吉|豐臣秀吉]]發起[[小田原征伐|小田原征伐]]，戶澤盛安化裝為商人，與侍衛6人、足輕3人動身隨秀吉“參戰”。盛安西行的路綫，是自仙北去到莊內，再由海路前往[[京都|京都]]。當他們到達京都時秀吉已動身前往[[關東|關東]]，於是盛安一行又開始追趕秀吉。當在[[駿河國|駿河國]]與秀吉會面後，秀吉當場賜予他寶刀“[[太閣兼光|太閣兼光]]”，而且保證盛安[[北浦郡|北浦郡]]4萬4千石所領安堵。然而，盛安在[[小田原之戰|小田原之戰]]中病逝，享年25歲。盛安死後，戶澤氏被周邊勢力乘機圍攻，但因[[奥州仕置|奥州仕置]]之故，得以保存，子孫於江戶時代成為[[出羽|出羽]][[新庄藩|新庄藩]]8萬3千石大名，家格延續至今。

盛安以十三歲繼承家督，當時戶澤氏不過是五百石左右的地方國人<ref>大友『シリーズ藩物語、新庄藩』、P11</ref>。十六歲時聯合安東撃敗南部聯軍，同年又聯合南部撃敗安東軍。十八歲奪回舊領。二十歲大破[[安東家|安東家]]、[[小野寺家|小野寺家]]，二十一歲統一仙遊地區，成為3萬石大名。二十三歲參加[[安東家|安東家]]內亂，對抗[[檜山安東家|檜山安東家]]，共奪得1萬4300石領地，十年間領地增加至4萬4300石<ref>大友『シリーズ藩物語、新庄藩』、P12</ref>。盛安送與民間養育之子[[戶澤政盛|戶澤政盛]]在其弟[[户泽光盛|光盛]]死後繼任家督，其子政盛於[[關原之戰|關原之戰]]慶長出羽的[[東禪寺合戰|東禪寺合戰]]以寡撃眾突襲上杉軍，大敗[[上杉景勝|上杉景勝]]，被東禪寺城主[[志駄義秀|志駄義秀]]贊曰：「不愧是戶澤炊介之子，為父者當以此兒子為榮！」<ref>大友『シリーズ藩物語、新庄藩』、P14</ref>

== 家督繼承者 ==
由於盛安嫡子政盛當時只有4歲的緣故，豐臣秀吉將戶澤氏託付給盛安之弟光盛，由光盛擔任家督繼承人。不過，光盛在[[萬曆朝鮮戰爭|朝鮮出兵]]之時，也以17歲之齡病亡於前去九州大本營的途中，政盛繼任家督。
