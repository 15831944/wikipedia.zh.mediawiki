'''戶塚安行站'''（{{jpn|j=戸塚安行駅|hg=とづかあんぎょうえき|rm=Tozuka-angyō eki|f=y}}）是一個位於[[埼玉縣|埼玉縣]][[川口市|川口市]]大字長藏新田，屬於[[埼玉高速鐵道線|埼玉高速鐵道線]]的[[鐵路車站|鐵路車站]]。車站編號是'''SR 24'''。

== 歷史 ==
* 2001年（[[平成|平成]]13年）3月28日 - 開業。站名是現在的地區名戶塚與安行合併得出。另外，站名決定為止的臨時名稱是「川口戶塚站」。

== 車站結構 ==
島式1面2線的[[地底車站|地底車站]]。裝有保安用的月台閘門系統。車站顏色是黃色。

地下1樓是閘口層，地下2樓是月台層。

== 使用情況 ==
2016年度的1日平均上車人次為'''7,149人'''<ref name="sr">[http://www.s-rail.co.jp/about/corporate/financial.html 経営状況・輸送状況] - 埼玉高速鉄道</ref>。
埼玉高速鐵道的車站當中乘客人數僅多於相鄰的[[新井宿站|新井宿站]]，排行尾二。
