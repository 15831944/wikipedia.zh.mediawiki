《'''舞變'''》是[[香港|香港]][[歌手|歌手]][[黎明|黎明]]約滿離開[[寶麗金唱片|寶麗金唱片]]後，由[[寶麗金唱片|寶麗金唱片]]發行的精選專輯<ref>{{Cite web |url =http://mypaper.pchome.com.tw/ileenho/post/1247516836|title =黎明(Leon Lai)唱片作品年表|date = 2005年5月27日 |publisher =壽司店的肥美亞彌-PChome 個人新聞台|language = 中文}}</ref>，於1998年10月24日推出<ref>{{Cite web |url =http://music.cdhaha.net/?p=679|title =黎明 舞變|publisher =CDHaha Music|language = 中文}}</ref>，專輯收錄了十四首歌曲，首次收錄於專輯的《舞變DANCE MEDLEY》是由《情深說話未錯講》、《那有一天不想你》、《Oh!夜》、《100樣可能》、《DNA出錯》、《今夜你會不會來》、《如果這是情》、《夏日傾情》、《對不起我愛你》、《你令愛了不起》、《送你一瓣的雪花》、《一夜傾情》、《情是我所有》、《危情追蹤》、《火舞艷陽》、《無名份的浪漫》十六首歌曲重新編曲組合而成<ref>{{Cite web |url =http://www.wsyyxz.com/huayu/21785.html|title =寶麗金時代黎明快歌的總結《舞變(DANCE MEDLEY)》|author =juanjuan|date = 2018年9月28日|publisher =www.wsyyxz.com|language = 中文}}</ref>。其餘十三首精選歌曲來自[[黎明|黎明]]於1991-1997年間<ref>{{Cite book|title=當世事再沒完美：香港流行樂壇1974-2013|last=陳怡東|first=|publisher=法律出版社|year=2015|isbn=7511880754|location=|pages=324}}</ref>發行的[[音樂專輯|音樂專輯]]。
