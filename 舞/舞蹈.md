'''舞蹈'''為[[八大艺术|八大艺术]]之一<ref>[http://61.30.235.248/entelechy/pub/LIT_12.asp?ctyp=LITERATURE&catid=3227&pcatid=0 藝術語彙›八大藝術] {{webarchive|url=https://web.archive.org/web/20160304204702/http://61.30.235.248/entelechy/pub/LIT_12.asp?ctyp=LITERATURE&catid=3227&pcatid=0 |date=2016-03-04 }}</ref>，是於三度空間中以身體為語言作「心智交流」現象之人体的运动表达艺术，一般有[[音乐|音乐]]伴奏，以有[[节奏|节奏]]的动作为主要表现手段的艺术形式。它一般借助音乐，也借助其他的道具。舞蹈本身有多元的社會意義及作用，包括[[運動|運動]]、[[社交|社交]]／[[求偶|求偶]]、[[祭祀|祭祀]]、[[禮儀|禮儀]]等。

== 歷史 ==
{{Main|舞蹈歷史}}
===西方===
在人類文明起源前，舞蹈在儀式、禮儀、慶典和娛樂方面都十分重要。[[考古|考古]]提供舞蹈的痕跡，例如9000年前在[[印度|印度]]的[[賓姆貝特卡的岩石壁畫|賓姆貝特卡的岩石壁畫]]和公元前3300年[[埃及|埃及]]墓壁畫描繪的繪畫舞蹈人物。

===東方===
[[中國|中國]]在五千年以前就已經出現了舞蹈，<ref name=cqw>[http://big5.chinaqw.cn:89/node2/node116/node1486/node1495/node1523/index.html 中國舞蹈] {{webarchive|url=https://web.archive.org/web/20101225105631/http://big5.chinaqw.cn:89/node2/node116/node1486/node1495/node1523/index.html |date=2010-12-25 }}</ref>產生於[[奴隸社會|奴隸社會]]<ref name=co>[http://www.skhlkmss.edu.hk/chinesedanceclub/chinese_old.htm 中國古典舞]</ref>，發展到[[秦|秦]][[漢|漢]]之際已形成一定特色：長袖細腰、飄逸輕盈、技藝並重、剛柔相濟。<ref name=co />[[魏晉南北朝|魏晉南北朝]]以來中外樂舞的大交流以及唐代帝王的重視，[[楊貴妃|楊貴妃]]等舞蹈家的出現，使中國表演性舞蹈空前繁榮。<ref name=co />

[[宋|宋]][[元|元]][[明|明]][[清|清]]以後，中國的表演性舞蹈逐漸融入戲曲，成為塑造角色、表達情感、展示劇情的[[戲曲|戲曲]]化舞蹈。

[[中華人民共和國|中華人民共和國]]建立後，作為獨立的劇場藝術的舞蹈日益完善和成熟起來。<ref name=cqw />

== 特色 ==
每一個舞蹈有它的主題，而這主題也要進行發展和變化。這和[[音樂|音樂]]的情形是很相似的。<ref name=cr>[http://big5.china.com.cn/book/txt/2006-10/18/content_7251195.htm 中國舞蹈的特點]</ref>

===中國舞蹈===
{{Main|中國舞蹈}}
中國舞蹈有一個明顯的特點，那就是「構圖」的對稱性，左邊的人做了一種姿勢，右邊的人也做一種相應的姿勢。舞蹈的三要素是指表情（不單是臉部的表情，是全身的表情）、節奏和構圖。中
國舞蹈的表情和節奏各有特點，但特徵最顯著的是構圖。這種構圖在舞蹈中叫做「對稱的平衡」。<ref name=cr />

== 分類 ==
根據舞蹈的功能、用途，可分「實用舞蹈」（自娛）與「表演舞蹈」（娛人）兩大類；每大類下，再根據「時間」分為傳統與流行，共計即四類；
然後可再往下細分但無論那一個分類方法，其界線均已模糊，分類表示如下。
