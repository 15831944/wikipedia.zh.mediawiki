'''舞鶴號誌站'''位於[[臺灣|臺灣]][[花蓮縣|花蓮縣]][[瑞穗鄉|瑞穗鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]已廢止的鐵路[[臺灣鐵路管理局車站等級#號誌站|號誌站]]。2017年9月26日，路線切換至[[自強隧道_(花蓮縣)_#自強隧道（第二代）|新自強隧道]]新線後正式裁撤<ref name="ltn">{{cite web|url=http://news.ltn.com.tw/news/life/breakingnews/2204527|title=花東鐵路歷史的一刻！新自強隧道切換新線完成|publisher=自由時報|date=2017-09-26|accessdate=2017-10-20}}</ref>。

== 車站構造 ==
*站內兩股交會線。
*北端設有[[安全側線|安全側線]]，防止列車對撞。
*曾設有[[島式月台|島式月台]]一座，大理石鋪面，已經拆除。

== 利用狀況 ==
*無人[[臺灣鐵路管理局車站等級#號誌站|號誌站]]，由[[瑞穗車站_(台灣)|瑞穗車站]]管理。
*僅供列車交會待避使用，不辦理客運。

== 歷史 ==
*1956年：設立舞鶴車站，招呼站。
*1957年12月1日：改為三等站。
*1971年6月1日：降為簡易站。
*1972年7月1日：改為招呼站。
*1982年：東線鐵路拓寬，車站重建完成但並未設站營業。
*2005年9月30日：設立號誌站。拆除原本廢棄的月臺，另鋪上一股道。
*2017年9月26日：切換至新自強隧道新線，本站正式裁撤。<ref name="ltn" />
