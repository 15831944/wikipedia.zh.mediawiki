'''猶太人區警察'''（{{lang-de|Jüdische Ghetto-Polizei}} 或 {{lang|de|Jüdischer Ordnungsdienst}}），[[猶太人|猶太人]]則稱其為'''猶太警察'''，是1939年至1944年間在[[納粹德國|納粹德國]]佔領的猶太人區下由[[猶太人委員會|猶太人委員會]]遵循納粹指令選出的[[輔助警察|輔助警察]]。<ref name = "Wiesen1">{{cite web

== 簡介==

猶太人區警察沒有官方制服，不被允許持有火器。他們最初被用作押運猶太人前往[[集中營|集中營]]之用。這些警察與他們監管的社區可以有微弱的溝通、聯繫聽從納粹的號召。<ref name = "Wiesen1" /> 第一個華沙猶太人區警察指揮官是波蘭裔猶太上校Józef Szeryński，他曾從地下[[猶太戰鬥組織|猶太戰鬥組織]]的一名成員Yisrael Kanal的暗殺下倖存。<ref name = "Wiesen1" />

華沙猶太人區警察也是最大的猶太人區警察單位之一，成員數一度達到二千五百人，其次的[[羅茲猶太人區|羅茲猶太人區]]警察約一千二百人，[[利沃夫猶太人區|利沃夫猶太人區]]則有五百人。<ref>Raul Hilberg: ''The Destruction of the European Jews'', Quadrangle Books, Chicago 1961, p. 310.</ref>
