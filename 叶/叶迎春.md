== 简介 ==
叶迎春早年毕业于[[北京广播学院|北京广播学院]]。1991年，担任[[景德镇电视台|景德镇电视台]]的[[主持人|主持人]]。1993年，担任[[江西电视台|江西电视台]]《江西新闻联播》播音员。1994年，考入[[北京广播学院|北京广播学院]]新闻系的首届研究生课程进修班。1996年，开始在[[中国中央电视台|中国中央电视台]]《[[军事报道|军事报道]]》担任主持人。<ref name="zjw " />

另外，她還參與過兒童科普[[動畫片|動畫片]]《[[藍貓淘氣三千問|藍貓淘氣三千問]]》的策劃工作<ref>[http://www.iqiyi.com/lib/s_207596005.html 爱奇艺-叶迎春]</ref>。

== 奖项 ==
1992年11月，叶迎春荣获江西电视台“全省电视主持人大赛”二等奖。<ref name="zjw " />

2008年，叶迎春荣获中国播音主持“[[金话筒奖|金话筒奖]]”电视播音作品奖。<ref>{{cite web |url=http://www.tjtv.com.cn/system/2008/12/24/003838953.shtml |title=中央电视台 叶迎春 |accessdate= 2008年|work= |publisher=天津电视台 |language =zh-cn}}</ref>
