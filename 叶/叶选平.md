'''叶选平'''（{{bd|1924年|12月20日|catIdx=Y叶}} ）<ref>[http://www.szmxsh.org/newsView.aspx?cid=35&id=42 深圳市梅縣商會《恭贺叶选平首长生日快乐》]</ref>，[[广东|广东]][[梅县|梅县]]人，[[中华人民共和国元帅|中华人民共和国元帅]][[叶剑英|叶剑英]]的长子；[[延安自然科学院|延安自然科学院]]机械系毕业。1945年9月，加入[[中国共产党|中国共产党]]；是中共第十二届候补中央委员，第十三届、十四届中央委员；曾任[[广东省省长|广东省省长]]、[[广州市市长|广州市市长]]等职。1991年4月，被增选为第七届[[全国政协副主席|全国政协副主席]]；后连任第八届、第九届全国政协副主席（排名第一）。

==早年经历==
叶选平1924年12月20日出生在[[广州市|广州市]]，后在家乡梅县雁洋下虎村生活；曾就读于[[广东省|广东省]][[中山市|中山市]][[中山纪念中学|中山纪念中学]]。1939年，赴[[重庆|重庆]]。[[皖南事变|皖南事变]]发生后，经中共组织安排转往[[延安|延安]]，和父亲叶剑英团聚，并进入[[延安自然科学院|延安自然科学院]]机械系学习，是该校第一批学生。1945年毕业后，被分配到延安兵工厂工作，成为一名机械师，并于该年9月加入[[中国共产党|中国共产党]]。[[国共第二次内战|国共第二次内战]]时期，曾在[[晋绥边区|晋绥边区]]第一机械厂工作。

==留苏工程师==
[[中华人民共和国|中华人民共和国]]建国后，叶选平先后在[[哈尔滨工业大学|哈尔滨工业大学]]和[[清华大学|清华大学]]进行过短期学习；1950年，调入当时中国最大的机床厂—[[沈阳机床集团有限责任公司|原沈阳第一机床厂]]工作。1952年，被派往[[苏联|苏联]]学习进修机床专业；1954年回国后，先后在[[沈阳|沈阳]]和[[北京|北京]]担任两家机床厂的总工程师。曾历任沈阳第一机床厂副厂长、兼总工程师，沈阳市机械局副总工程师，北京第一机床厂生产技术副厂长、兼总工程师等职。

==仕途==
1973年，49岁的叶选平出任北京市机械局领导小组副组长、党委常委，开始步入政界。[[文化大革命|文化大革命]]结束后，于1977年进入[[中共中央党校|中共中央党校]]学习；1978年，出任[[国家科委|国家科委]]三局局长。1979年，叶选平调往[[广东省|广东省]]工作，出任副省长、兼省科委主任；1983年，任[[中共广州市委|中共广州市委]]副书记、市长；1985年，晋升[[广东省省长|广东省省长]]，成为[[改革开放|改革开放]]初期，广东省的主要领导之一。1988年1月12日，在广东省七届人大第一次会议上，叶选平连任广东省省长；在750张有效票选票中，获得了746张<ref>{{cite web |url=http://www.gd.chinanews.com/2013/2013-02-17/2/231651_5.shtml|title=李瑞环谈叶选平当选省长：“世袭”是无稽之谈|accessdate= 2016-07-11|publisher= |work= |language=zh-hans}}</ref>。

在中共大力提倡干部年轻化的政策下，叶选平在67岁时，才卸任省长职务；并于1991年4月，在第七届[[全国政协|全国政协]]第四次会议上，增选为[[全国政协副主席|全国政协副主席]]。此后，他分别于1993年和1998年连任第八届、第九届全国政协副主席（排名第一位）。

==家庭==
父亲是[[中华人民共和国元帅|中华人民共和国元帅]][[叶剑英|叶剑英]]，生母是冯华。

妻子[[吴小兰|吴小兰]]，延安“五老”之一的[[吴玉章|吴玉章]]的外甥女，是一位机械专家，曾任[[中国机械进出口公司|中国机械进出口公司]]副总经理、[[深圳市|深圳市]]副市长、深圳市人大常委会副主任。<ref>{{cite web |url=http://data.book.163.com/book/section/0000FUWe/0000FUWe19.html|title= 延安“五老”之一的吴玉章|accessdate= |publisher= |work= |language=zh-hans}}</ref>

儿子[[叶新福|叶新福]]，1992年在香港创立[[万信证券|万信证券]]。<ref>[http://www.zaobao.com/zaobao/pages/hkjj0305.html 联合早报：国叶拟收购宝雅科技借壳上市]</ref>现任广东南湖高尔夫球会董事长，另一子姓名职业不详。

孙子[[叶仲豪|叶仲豪]]。广东云浮市高新区主任。<ref>{{cite web |url= http://politics.caijing.com.cn/2013-05-10/112762221.html|title=政治舞台上的红三代四代：叶剑英曾孙正处级 |accessdate= |publisher= |work= |language=zh-hans}}</ref>
另一名为叶丁。[http://politics.caijing.com.cn/20140812/3652749.shtml]
