'''叶莲娜·欧列伽芙娜·谢罗娃'''（{{lang-ru|Еле́на Оле́говна Серо́ва}}，{{bd|1976年|4月22日|}}）是[[俄罗斯|俄罗斯]]女性[[政治人物|政治人物]]，俄罗斯历史上继[[叶莲娜·孔达科娃|叶莲娜·孔达科娃]]后的第四位飞上太空的女性[[宇航员|宇航员]]<ref>{{cite web |title=妇女节特稿：宇宙那么大，我想去看看 |url=http://www.taikongmedia.com/Item/Show.asp?m=1&d=20112 |publisher=中国太空网 |archiveurl=https://web.archive.org/web/20190413120002/http://www.taikongmedia.com/Item/Show.asp?m=1&d=20112 |archivedate=2019-04-13 |date=2016-03-08}}</ref>，也是第一位在[[国际空间站|国际空间站]]工作的俄罗斯女性<ref>{{cite web |author=王凤昌 |title=俄罗斯女航天员首次飞赴国际空间站 |url=https://www.reuters.com/article/space-station-launch-idCNL3S0RR2CQ20140926 |publisher=[[路透社|路透社]] |archiveurl=https://web.archive.org/web/20190414031702/https://www.reuters.com/article/space-station-launch-idCNL3S0RR2CQ20140926 |archivedate=2019-04-14 |date=2014-09-26}}</ref>。
==早年==
2001年3月毕业于[[莫斯科航空学院|莫斯科航空学院]]航空航天系工程学专业，毕业后在科罗廖夫能源火箭航天集团担任工程师。2006年成为宇航员人选之一，开始接受基础训练，2009年6月获得实验宇航员资格<ref>{{cite web |title=Yelena Olegovna Serova |url=https://web.archive.org/web/20160413225556/http://www.jsc.nasa.gov/Bios/htmlbios/serova.html |publisher=[[NASA|NASA]] |accessdate=2019-04-14}}</ref>。

她是2014年2月7日[[索契|索契]][[2014年冬季奥林匹克运动会|冬奥会]][[2014年冬季奧林匹克運動會開幕式|開幕式]]上的五名宇航员旗手之一<ref>{{cite web |url=http://www.sochi2014.com/en/news-the-xxii-olympic-winter-games-in-sochi-in-2014-has-opened-with-a-grand-show-thrilling-spectators |title=The XXII Olympic Winter Games in Sochi in 2014 has opened with a grand show |date=2014-02-08 |publisher=www.Sochi2014.com |accessdate= 2014-02-11}}</ref>。
==宇航员生涯==
2014年8月15日，[[国际空间站|国际空间站]]远征41/42备乘组队员在[[加加林宇航员培训中心|加加林宇航员培训中心]]开始接受临行前的最后考验。其中谢罗娃和[[亚历山大·萨莫库佳耶夫|亚历山大·萨莫库佳耶夫]]、巴里·威玛是主乘组成员；[[根纳季·帕达尔卡|根纳季·帕达尔卡]]、[[米哈伊尔·科尔尼延科|米哈伊尔·科尔尼延科]]和[[斯科特·凯利|斯科特·凯利]]是备份乘组成员。最后考验的主要是模拟操纵[[欧洲航天局|欧洲航天局]]的[[自動運載飛船|自動運載飛船]]（ATV）与国际空间站的靠近、停泊和对接的自动控制过程<ref>{{cite web |author=杨敬荣 |title=国际空间站第4142长期考察团乘组通过系列考试 |url=http://taikongmedia.com/Item/Show.asp?m=1&d=17516 |publisher=中国太空网 |archiveurl=https://web.archive.org/web/20190414034327/http://taikongmedia.com/Item/Show.asp?m=1&d=17516 |archivedate=2019-04-14 |date=2014-08-26}}</ref>。
===远征41/42===
[[File:Serova_near_a_hatch_of_the_Russian_segment.jpg|thumb]]
[[File:Elena_Serova_works_with_the_Kaskad_cell_cultivation_experiment.jpg|thumb]]
她和主乘组成员于2014年9月26日乘坐由[[联盟-FG|联盟-FG]]运载火箭搭载的[[联盟TMA-14M|联盟TMA-14M]]载人飞船进入飞往国际空间站的预定轨道。到达空间站后，她和同事接收了多艘货运飞船，并完成了几十项[[科学实验|科学实验]]<ref>{{cite web |author=张继业 |title=国际空间站３名宇航员结束任务进入返回舱 |url=http://www.xinhuanet.com/world/2015-03/12/c_1114608612.htm |publisher=新华网 |archiveurl=https://web.archive.org/web/20190414040430/http://www.xinhuanet.com/world/2015-03/12/c_1114608612.htm |archivedate=2019-04-14 |location=[[莫斯科|莫斯科]] |date=2015-03-12}}</ref>。

2014年11月27日[[感恩节|感恩节]]，她和国际空间站5名航天员共同享用了[[火鸡|火鸡]]、甜[[山芋|山芋]]、玉米[[面包|面包]]以及水果[[馅饼|馅饼]]等经过冻干以及恒温等处理过的特制感恩节美食<ref>{{cite web |title=国际空间站公布宇航员感恩节菜单 |url=http://www.cnsa.gov.cn/n6758823/n6759010/c6774362/content.html |publisher=国家航天局 |archiveurl=https://web.archive.org/web/20190414035454/http://www.cnsa.gov.cn/n6758823/n6759010/c6774362/content.html |archivedate=2019-04-14 |date=2014-12-01}}</ref>。

2015年3月11日，她和远征42任务成员乘坐[[联盟TMA-14M|联盟TMA-14M]]飞船返回地球，3月12日降落在[[杰兹卡兹甘|杰兹卡兹甘]]附近<ref>{{cite web |title=国际空间站远征42机组宇航员返航 |url=http://tech.qq.com/a/20150318/011309.htm#p=1 |archiveurl=https://web.archive.org/web/20150423121008/http://tech.qq.com/a/20150318/011309.htm#p=1 |archivedate=2015-04-23 |date=2015-03-18}}</ref>。

==政治生涯==
她是[[統一俄羅斯|統一俄羅斯]]党员。[[2016年俄羅斯國家杜馬選舉|2016年俄羅斯國家杜馬選舉]]中，在[[科洛姆纳|科洛姆纳]]选区获得50.84％的选票<ref>[http://www.vybory.izbirkom.ru/region/izbirkom?action=show&global=true&root=1000158&tvd=100100067796012&vrn=100100067795849&prver=0&pronetvd=0&region=0&sub_region=0&type=463&vibid=100100067796012 Результаты выборов по одномандатному избирательному округу № 119]</ref>，成功当选[[国家杜马|国家杜马]]议员，并宣布从宇航员退役。
