'''叶莲娜·弗拉基米诺芙娜·孔达科娃'''（{{lang-ru|Еле́на Влади́мировна Кондако́ва}}，{{bd|1957年|3月30日|}}）是[[俄罗斯|俄罗斯]]女性[[政治人物|政治人物]]，俄罗斯首位同时搭乘过[[联盟号宇宙飞船|联盟号宇宙飞船]]和[[亚特兰蒂斯号航天飞机|亚特兰蒂斯号航天飞机]]的女航天员。<ref>{{cite web |title=妇女节特稿：宇宙那么大，我想去看看 |url=http://www.taikongmedia.com/Item/Show.asp?m=1&d=20112 |publisher=中国太空网 |archiveurl=https://web.archive.org/web/20190413120002/http://www.taikongmedia.com/Item/Show.asp?m=1&d=20112 |archivedate=2019-04-13 |date=2016-03-08}}</ref>

==早年==
1957年3月30日日出生于[[苏联|苏联]][[莫斯科州|莫斯科州]][[梅季希|梅季希]]，在[[加里宁格勒|加里宁格勒]]长大。1980年毕业于[[鮑曼莫斯科國立技術大學|鮑曼莫斯科國立技術大學]][[工程学|工程学]]专业，后在科罗廖夫能源火箭航天集团工作。1989年被选为宇航员候选人。<ref>{{cite book |author=Umberto Cavallaro |title=Women Spacefarers: Sixty Different Paths to Space |date=2017 |publisher=Springer |isbn=9783319340487 |url=https://books.google.com/books?id=VCNBDgAAQBAJ&pg=PA182&}}</ref>
==宇航员生涯==
1994年10月4日，她和两名男宇航员一起乘坐[[联盟-TM20|联盟-TM20]]号宇宙飞船到达[[和平号空间站|和平号空间站]]，在太空飞行169天后于1995年3月22日返回地面，创下了女性连续太空滞留时间纪录。1995年4月12日，时任总统[[叶利钦|叶利钦]]授予她“[[俄罗斯联邦英雄|俄罗斯联邦英雄]]”荣誉称号。<ref>{{cite web |title=曾经“和平号”上生活过的人们 |url=http://www.people.com.cn/GB/tupian/20010320/420941.html |publisher=[[人民网|人民网]] |archiveurl=https://web.archive.org/web/20140710023600/http://www.people.com.cn/GB/tupian/20010320/420941.html |archivedate=2014年7月10日 |date=2001年3月20日}}</ref>一年后，这一纪录被美国女宇航员[[珊农·露茜德|珊农·露茜德]]打破。<ref>
{{cite web | url = http://www.nasa.gov/mission_pages/shuttle/shuttlemissions/sts117/news/STS-117-16.html | title = STS-117 MCC Status Report #16 | publisher = NASA | accessdate = 2009-12-15}}</ref>

1997年5月15日，她乘坐[[亚特兰蒂斯号航天飞机|亚特兰蒂斯号航天飞机]]第二次前往太空执行[[STS-84|STS-84]]任务，在太空待了9天。1999年退役转而进入政界，成为[[祖国-全俄罗斯|祖国-全俄罗斯]]（现[[統一俄羅斯|統一俄羅斯]]党）在[[国家杜马|国家杜马]]的议员。<ref>{{cite book |author1=Nicholas Faulkner |title=Top 101 Women of STEM |date=2016 |publisher=Encyclopaedia Britannica |isbn=9781680484991 |page=108 |url=https://books.google.com/books?id=9e-ODAAAQBAJ&pg=PT108&}}</ref>
