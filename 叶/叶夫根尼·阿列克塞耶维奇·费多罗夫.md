'''叶夫根尼·阿列克塞耶维奇·费多罗夫'''（俄语：{{lang|ru|Евге́ний Алексе́евич Фёдоров}}，{{bd|1963年|5月11日}}) 是[[俄罗斯|俄罗斯]]政治人物，出生於[[列寧格勒|列寧格勒]]（今[[聖彼得堡|聖彼得堡]]）。曾任[[国家杜马|俄联邦国家杜马]]议员（1993—1996、2003—）<ref name=duma_2016>{{Cite web|url = http://www.duma.gov.ru/structure/deputies/1756583/|title = Фёдоров Евгений Алексеевич (седьмой созыв)|publisher = Официальный сайт Государственной Думы Федерального Собрания Российской Федерации|archiveurl = https://www.webcitation.org/6lx512P4T?url=http://www.duma.gov.ru/structure/deputies/1756583/|archivedate = 2016-11-12|accessdate = 12.11.2016|dead-url = no}}</ref>，俄联邦国家杜马国家预算委员, [[统一俄罗斯|统一俄罗斯]]中央政治会议委员, 俄罗斯联邦一等官，经济学副博士<ref name=duma>{{Cite web|url = http://www.duma.gov.ru/structure/deputies/23523/|title = Фёдоров Евгений Алексеевич (пятый созыв)|author = |work = |date = |publisher = Официальный сайт Государственной Думы Федерального Собрания Российской Федерации|archiveurl = https://www.webcitation.org/6fgVvxeG2?url=http://www.duma.gov.ru/structure/deputies/23523/|archivedate = 2016-03-01|access-date = 2018-11-04|dead-url = no}}</ref>，曾参战于[[阿富汗战争|阿富汗战争]]，预备上校（2017），[[俄罗斯联邦民族解放运动|俄罗斯联邦民族解放运动]]调配员。
