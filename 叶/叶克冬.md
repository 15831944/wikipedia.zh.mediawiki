'''叶克冬'''（{{bd|1953年|9月||}}）[[广西|广西]][[贺州|贺州]]人，[[中华人民共和国|中华人民共和国]]官员<ref name=ykd>{{Cite web|url =http://www.gwytb.gov.cn/gtb/yekedong/|title =叶克冬|author =|date =|publisher =中共中央台办、国务院台办|language =|accessdate =2017-10-12|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20150315021439/http://www.gwytb.gov.cn/gtb/yekedong/|archivedate =2015年3月15日|df =}}</ref>。

==生平==
1969年8月参加工作，1982年5月加入[[中国共产党|中国共产党]]。1969年8月至1973年7月，为[[黑龙江生产建设兵团|黑龙江生产建设兵团]][[知青|知青]]。1973年7月至1978年10月，为[[内蒙古生产建设兵团|内蒙古生产建设兵团]]知青。1978年10月至1982年8月，在[[中山大学|中山大学]]哲学系学习，获得哲学学士学位<ref name=ykd/>。

1982年8月至1983年2月，任[[共青团中央|团中央]]宣传部干部。1983年2月至1985年7月，任团中央办公厅秘书处 秘书。1985年7月至12月，借调为[[中共贵州省委|中共贵州省委]]办公厅秘书。1985年12月至1993年10月，任团中央统战部港澳台工作处副处长、处长<ref name=ykd/>。

1993年10月至1994年9月，任[[新华社香港分社|新华社香港分社]]教育科技部普教处处长。1994年9月至2000年10月，任新华社香港分社台湾事务部副部长。2000年10月至2001年5月，任新华社香港分社台务部副部长（正局级）<ref name=ykd/>。

2001年5月至7月，任职[[国务院台湾事务办公室|中央台办、国台办]]港澳涉台事务局。2001年7月至2005年5月，任中央台办、国台办港澳涉台事务局局长。2005年5月至2006年7月，任中央台办、国台办主任助理兼港澳涉台事务局局长<ref name=ykd/>。2006年7月至2015年4月，任中央台办、国台办副主任<ref name=ykd/><ref>{{Cite web |url = http://district.ce.cn/newarea/sddy/201504/10/t20150410_5074688.shtml |title = 龙明彪任国务院台湾事务办公室副主任 叶克冬不再担任 |author =  |date =2015-04-10  |publisher =中国经济网  |language =  |accessdate =  |quote =  }}</ref>。

他是[[第十二届全国政协委员|第十二届全国政协委员]]<ref>{{Cite web |url = http://www.cppcc.gov.cn/zxww/2014/02/27/ARTI1393487184783942.shtml |title =中国人民政治协商会议第十二届全国委员会委员名单  |author =  |date = 2017-06-29 |publisher =中国政协网  |language =  |accessdate =  |quote =  }}</ref>。
