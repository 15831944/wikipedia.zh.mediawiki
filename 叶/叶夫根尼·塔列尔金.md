'''叶夫根尼·伊戈列维奇·塔列尔金'''（{{lang-ru|Евге́ний И́горевич Таре́лкин}}，{{bd|1974年|12月29日|}}）是[[俄罗斯|俄罗斯]]宇航员。
==早年==
出生于[[赤塔州|赤塔州]]（现[[外貝加爾邊疆區|外貝加爾邊疆區]]）。1992至1996年在[[叶伊斯克高等军事航空学校|叶伊斯克高等军事航空学校]]学习，1998年毕业于[[加加林空军学院|加加林空军学院]]，毕业后在[[加加林宇航员培训中心|加加林宇航员培训中心]]担任高级工程师<ref>{{cite web |title=Евгений Игоревич Тарелкин |url=http://www.astronaut.ru/as_rusia/2003/text/tarelkin.htm?reload_coolmenus |publisher=Энциклопедия |accessdate=2019-04-16 |language=ru}}</ref>。他不仅有驾驶[[L-29海豚|L-29海豚]]、[[L-39信天翁|L-39信天翁]]和[[伊尔-76|伊尔-76]]的丰富经验，还拥有[[降落伞|降落伞]]和潜水员教练资格<ref>{{cite web |title=Российский космонавт Евгений Тарелкин получил звание PADI Assistant Instructor! |url=http://www.divelife.ru/cgi-bin/show_news.pl?option=FullNews&div=ind&id=762 |publisher=PADI |accessdate=2019-04-16 |language=ru}}</ref>。
==宇航员生涯==
2003年成为宇航员候选人，之后开始接受基础训练。2005年6月获得测试宇航员资格<ref>{{cite web|last=Wade|first=Mark|title=Tarelkin|url=http://www.astronautix.com/astros/tarelkin.htm|publisher=[[Encyclopedia_Astronautica|Encyclopedia Astronautica]]|accessdate=2011-04-30|deadurl=yes|archiveurl=https://web.archive.org/web/20121005071045/http://astronautix.com/astros/tarelkin.htm|archivedate=2012-10-05}}</ref>。
===远征33/34===
[[协调世界时|协调世界时]]2012年10月23日10时51分，俄制[[联盟TMA-06M|联盟TMA-06M]]号载人飞船连同[[联盟-FG|联盟-FG]]号运载火箭搭载塔列尔金等宇航员，从哈萨克斯坦的[[拜科努尔航天发射场|拜科努尔航天发射场]]发射升空，前往[[国际空间站|国际空间站]]<ref>{{cite web |title=俄“联盟”号飞船搭载3名宇航员发射升空 |url=http://www.chinanews.com/gj/2012/10-23/4270661.shtml |publisher=中国新闻网 |accessdate=2019-04-16 |archiveurl=https://web.archive.org/web/20121229094035/http://www.chinanews.com/gj/2012/10-23/4270661.shtml |archivedate=2012-12-29 |date=2012-10-23}}</ref>。

2013年3月16日，已在国际空间站驻守近5个月的塔列尔金等3位宇航员乘坐联盟TMA-06M飞船顺利返回地球，于协调世界时16日3时6分在哈萨克斯坦[[阿尔卡雷克|阿尔卡雷克]]地区的草原上着陆<ref>{{cite web |author1=贺颖骏 |title=国际空间站3名宇航员顺利返回地球 |url=http://news.sciencenet.cn/htmlnews/2013/3/275711.shtm |publisher=[[新华社|新华社]] |accessdate=2019-04-16 |archiveurl=https://web.archive.org/web/20130920155835/http://news.sciencenet.cn/htmlnews/2013/3/275711.shtm |archivedate=2013-09-20 |date=2013-03-16}}</ref>。返回舱的安全由3架飞机（[[安托諾夫An-12|安-12]]和[[安托諾夫An-26|安-26]]）、14架[[米-8直升机|米-8直升机]]和7架搜救机保障<ref>{{cite web |author=刘洋 |title=俄将出动17架飞机保障联盟号飞船返回舱安全着陆 |url=http://world.huanqiu.com/exclusive/2013-03/3734568.html |publisher=[[环球网|环球网]] |archiveurl=https://web.archive.org/web/20190416045942/http://world.huanqiu.com/exclusive/2013-03/3734568.html |archivedate=2019-04-16 |date=2013-03-14}}</ref>。
===模拟登月===
2019年3月19日，塔列尔金作为指令长，与美国专家莱因霍尔德·波维赖提斯在[[莫斯科|莫斯科]]的[[俄罗斯科学院医学生物问题研究所|俄罗斯科学院医学生物问题研究所]]开始了为期120天的“SIRIUS-19”模拟联合登陆[[月球|月球]]隔离实验<ref>{{cite web |title=俄美宇航员将在SIRIUS实验框架内联合登月 |url=http://world.huanqiu.com/article/2019-03/14571501.html?agt=15422 |publisher=[[环球网|环球网]] |archiveurl=https://web.archive.org/web/20190416044417/http://world.huanqiu.com/article/2019-03/14571501.html?agt=15422 |archivedate=2019-04-16 |date=2019-03-19}}</ref>。
