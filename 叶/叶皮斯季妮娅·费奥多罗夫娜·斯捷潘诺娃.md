'''叶皮斯季妮娅·费奥多罗夫娜·斯捷潘诺娃'''（{{lang-ru|Епистини́я Фёдоровна Степа́нова}}；{{bd|1874年|11月18日|1969年|12月7日}}）是一名[[俄罗斯|俄罗斯]]妇女，她的9个儿子死于[[苏德战争|卫国战争]]，并因此而被授予[[英雄母亲|英雄母亲]]称号以及[[卫国战争勋章|一级卫国战争勋章]]。

== 生平 ==

叶皮斯季妮娅·费奥多罗夫娜·斯捷潘诺娃出生于[[俄罗斯帝国|俄罗斯帝国]]境内相当于今日[[乌克兰|乌克兰]]的地区，小时候在库班长大<ref>[http://www.timashevsk.ru/?q=stepanovi Музей и история семьи Степановых // Тimashevsk.ru — информационный сайт г. Тимашёвска.]</ref>。8岁时她成为一名农工并在库班的农场工作，饲养家禽、清理谷物。在那里，她结识了她的丈夫，当地名为[[格奥尔基·季米特洛夫|“格奥尔基·季米特洛夫”]]的[[苏联集体农庄|集体农庄]]的领班——米哈伊尔·尼古拉耶维奇·斯捷潘诺夫（1878年生人，卒于1934年），并接受了他的求婚。

斯捷潘诺娃住在[[克拉斯诺达尔边疆区|克拉斯诺达尔边疆区]][[季马绍夫斯基区|季马绍夫斯基区]]“五月一日”农场里，并生下了15个孩子，其中<ref>[http://www.timashevsk-zt.ru/ArticleSection/Details/154 Жемчужина Епистиния…]</ref>：

*斯特莎，女，第一个孩子也是最早去世的一个，死于沸水烫伤；
*二孩，男，出生便夭折；
*格里沙，男，5岁时死于[[流行性腮腺炎|流行性腮腺炎]]；
*维拉，女，1939年死于[[一氧化碳中毒|一氧化碳中毒]]。

至卫国战争开始时，斯捷潘诺娃还有9个儿子和1个女儿在世.

她的9个儿子分别是：

*'''亚历山大·米哈伊洛维奇·斯捷潘诺夫'''（{{lang-ru|Александр Михайлович Степанов}}；{{bd|1801年||1918年}}），因其兄弟为[[苏联红军|红军]]而遭[[白军|白军]]杀害。
*'''尼古拉·米哈伊洛维奇·斯捷潘诺夫'''（{{lang-ru|Николай Михайлович Степанов}}；{{bd|1903年||1963年}}<ref>Военная энциклопедия в 8 томах. Том 7:"Прод"-«Таджикистан». — Москва, Военное издательство, 2003. — Стр.658.</ref>），1941年8月成为近卫第5顿河骑兵军骑兵，1944年在解放乌克兰战役中受伤，炮弹破片割断了他的右腿，落下终身残疾，而后重返战场。1963年死于旧伤。育有一子：瓦伦丁·尼古拉耶维奇·斯捷潘诺夫（{{lang-ru|Валентин Николаевич Степанов}}）<ref name=autogenerated2>{{Cite web|accessdate = 2015-12-11|title = Музей и история семьи Степановых {{!}} Тимашевск. Городской сайт. Газеты, видео, афиша, новости, объявления|url = http://www.timashevsk.ru/?q=stepanovi|publisher = www.timashevsk.ru}}</ref>。
*'''瓦西里·米哈伊洛维奇·斯捷潘诺夫'''（{{lang-ru| Василий Михайлович Степанов}}；{{bd|1908年||1943年}}），二战军人，1941年5月21日成为第9集团军第106步兵师第553炮兵团士官，1941年10月30日在[[克里米亚苏维埃社会主义自治共和国|克里米亚]]被德军俘虏，1942年从集中营越狱，参加“为了祖国”游击队，1943年11月2日再次被俘，12月被处决<ref>{{Cite web|accessdate = 2015-12-10|title = Память народа :: Донесение о безвозвратных потерях :: Степанов Василий Михайлович, __.12.1943, убит,|url = https://pamyat-naroda.ru/heroes/memorial-chelovek_dopolnitelnoe_donesenie71870007/|publisher = pamyat-naroda.ru}}</ref>。葬于[[第聂伯罗彼得罗夫斯克州|第聂伯罗彼得罗夫斯克州]][[苏尔西科-米海利夫卡|苏尔西科-米海利夫卡镇]]的万人坑。
*'''菲利普·米哈伊洛维奇·斯捷潘诺夫'''（{{lang-ru|Филипп Михайлович Степанов}}；{{bd|1910年||1945年}}），库班“五月一日”集体农庄的庄员，1939年由于他培养了谷物和甜菜为提马施瓦克地区的最大产量，参加了莫斯科全苏农业展览<ref name=autogenerated2 />。1941年5月22日参军<ref name=autogenerated1>{{Cite web|accessdate = 2015-12-11|title = Память народа :: Донесение о безвозвратных потерях :: Степанов Филипп Михайлович, __.__.1942, пропал без вести,|url = https://pamyat-naroda.ru/heroes/memorial-chelovek_dopolnitelnoe_donesenie58803767/|publisher = pamyat-naroda.ru}}</ref>，成为第1集团军第1营第699团士兵。1942年5月24日在哈尔科夫战役中被俘。1945年2月10日死于[[帕德博恩|帕德博恩]]第326集中营<ref>{{Cite web|accessdate = 2015-12-10|title = Память народа :: Донесение о безвозвратных потерях :: Степанов Филипп Михайлович, , ,|url = https://pamyat-naroda.ru/heroes/memorial-chelovek_plen72229364/|publisher = pamyat-naroda.ru}}</ref>。妻子：史蒂潘诺娃·亚历山德拉·莫伊西瓦娜<ref name=autogenerated1 />。
*'''费奥多尔·米哈伊洛维奇·斯捷潘诺夫'''（{{lang-ru|Фёдор Михайлович Степанов}}；{{bd|1912年||1939年}}），1939年在克拉斯诺达尔军校完成指挥专业课程后被授予少尉军衔，毕业后被送往远东的外贝加尔军区，加入驻扎在菲德的第149红旗摩托化步兵团。1939年8月20日，他在与日军的激烈的战斗中牺牲。后被追授勇敢奖章<ref name=autogenerated2 />。
*'''伊万·米哈伊洛维奇·斯捷潘诺夫'''（{{lang-ru|Иван Михайлович Степанов}}；{{bd|1915年||1942年}}），共产党员，1937年在乌克兰加入苏联红军，随后在奥尔忠尼启则红旗军事学院学习指挥专业课程并毕业。1940年冬天参加了对芬兰的攻势，展示出他是一名勇敢果断的指挥员。1941年<ref>{{Cite web|accessdate = 2015-12-11|title = Память народа :: Донесение о безвозвратных потерях :: Степанов Иван Михайлович, __.__.1941, пропал без вести,|url = https://pamyat-naroda.ru/heroes/memorial-chelovek_prikaz74603990/|publisher = pamyat-naroda.ru}}</ref>在西部战线被俘<ref>{{Cite web|accessdate = 2015-12-11|title = Память народа :: Донесение о безвозвратных потерях :: Степанов Иван Михайлович, между 22.06.1941 и 31.07.1941, пропал без вести,|url = https://pamyat-naroda.ru/heroes/memorial-chelovek_donesenie2988380/|publisher = pamyat-naroda.ru}}</ref>，1942年秋逃出集中营，并躲藏于游击队中，后被德军射杀。葬于明斯克斯莫列维奇区的万人坑中<ref>[http://www.molodguard.ru/heroes57-146.htm Письма братьев Степановых // Молодая гвардия.]</ref>。
*'''伊里亚·米哈伊洛维奇·斯捷潘诺夫'''（{{lang-ru|Илья Михайлович Степанов}}；{{bd|1917年||1943年}}），1937年10月应征入伍，两年后在萨拉托夫第10坦克学院毕业并被授予中尉军衔，随后被派往驻波罗的海地区的第250坦克旅服役。卫国战争第一天他就在战斗中受伤，在后方医院接受治疗，并和母亲一起生活。不久他就归队，到当年12月他第三次受伤入院。1943年7月14日，第5坦克军第70独立坦克旅指挥员伊里亚·米哈伊洛维奇·斯捷潘诺夫大尉死于库尔斯克战役。后被追授共产党员。
*'''帕维尔·米哈伊洛维奇·斯捷潘诺夫'''（{{lang-ru|Павел Михайлович Степанов}}；{{bd|1919年||1941年}}），1939年入伍并进入基辅炮兵军事学院学习，1941年毕业并被授予中尉军衔，在布良斯克方面军第55步兵师第141榴弹炮兵团服役，共青团员。1941年12月在战争中失踪<ref>{{Cite web|accessdate = 2015-12-11|title = Память народа :: Донесение о безвозвратных потерях :: Степанов Павел Михайлович, До 09.01.1942, пропал без вести,|url = https://pamyat-naroda.ru/heroes/memorial-chelovek573801933/|publisher = pamyat-naroda.ru}}</ref>。
*'''亚历山大·米哈伊洛维奇·斯捷潘诺夫（小）'''（{{lang-ru|Александр Михайлович Степанов (младший)}}；{{bd|1923年||1943年}}），1941年10月加入红军，在奥尔忠尼启则军校完成了迫击炮部队指挥专业课程并被授予中尉军衔。随后他被送往斯大林格勒。1943年4月25日因一个人摧毁两个机枪碉堡而被授予红星勋章。1943年10月2日，在基辅附近的战斗中，他指挥部队击退了德军6次攻击，在德军发起第7次进攻时，他使用轻武器顽强抵抗，一个人射杀了15名德军<ref>{{Cite web|accessdate = 2015-12-11|title = Память народа :: Документ о награде :: Степанов Александр Михайлович, Герой Советского Союза (Орден Ленина и медаль «Золотая звезда»)|url = https://pamyat-naroda.ru/heroes/podvig-chelovek_nagrazhdenie150040434/|publisher = pamyat-naroda.ru}}</ref>，最后引爆手榴弹与敌人同归于尽<ref>{{Cite web|accessdate = 2015-12-11|title = Память народа :: Донесение о безвозвратных потерях :: Степанов Александр Михайлович, , ,|url = https://pamyat-naroda.ru/heroes/memorial-chelovek_zahoronenie84020181/|publisher = pamyat-naroda.ru}}</ref>。后被追授“苏联英雄”称号及列宁勋章。

1966年，时任苏联国防部长的[[安德烈·安东诺维奇·格列奇科|安德烈·安东诺维奇·格列奇科]]元帅写信给她：

“九个儿子，由您抚养并教育，九个您最亲爱的人，为了保卫伟大的苏维埃祖国而献出了生命。由于他们的军事成就，他们从敌人手中夺取了伟大的胜利日，并将流芳千古……您，战士的母亲，也是所有战士的母亲，他们将会为您带去最孝顺的温暖，并向您，一名平凡的俄罗斯妇女，下跪。”

1969年12月7日，斯捷潘诺娃在唯一一个女儿的家中去世<ref>{{Cite web |url=http://rostof.ru/article.php?chapter=8&id=20103310 |title=Смысленко О. Земля, где мои сыны? // Ростов официальный. — № 33 (820) от 11.08.2010 |access-date=2017-07-19 |archive-url=https://web.archive.org/web/20160408062629/http://rostof.ru/article.php?chapter=8&id=20103310 |archive-date=2016-04-08 |dead-url=yes }}</ref>，被以军礼安葬于军人墓地当中。1974年4月14日，《[[共青团真理报|共青团真理报]]》报道了斯捷潘诺夫一家的事迹。

斯捷潘诺夫家族并非绝后，至2010年，共有44个孙子和曾孙。

== 荣誉 ==

*斯捷潘诺娃被授予[[英雄母亲|英雄母亲]]荣誉称号；
*1977年，她被追授[[卫国战争勋章|一级卫国战争勋章]]。
