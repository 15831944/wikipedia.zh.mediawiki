《'''叶挺将军'''》，中国大陆电视剧，改编自[[张笑天|张笑天]][[纪实文学|纪实文学]]作品《叶挺将军》，导演[[王冀邢|王冀邢]]，主演[[高发|高发]]、[[刘劲|刘劲]]、[[汪裴|汪裴]]、[[盖美|盖美]]、[[卢奇|卢奇]]。

== 剧情 ==
{{See Also|叶挺}}
该剧时间跨度从1922年开始一直到1946年，讲述了叶挺从军到牺牲的人生路程。

== 花絮 ==
该剧是“纪念[[新四军|新四军]]建军70周年”和“叶挺将军诞辰11O周年”的献礼剧。<ref>{{cite web|url=http://news.newsxc.com/Xcnews/200704/20070409183709_3788.shtml|title=电视连续剧《叶挺将军》在泾县云岭热拍|accessdate=2007-04-09|publisher=宣城网|work=|language=zh-cn}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

该剧制作方是中国电视剧制作中心。<ref>{{cite web |url=http://ent.qq.com/a/20090224/000103.htm |title=《叶挺将军》登陆央视 再现叶挺传奇人生 |accessdate=2009 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

由于高发出演叶挺的形象得到认可，诸多不明身份的人以叶挺相关纪念馆的名义骚扰高发，邀其拍片。<ref>{{cite web |url=http://msn.ent.ynet.com/3.1/1112/14/6601539.html |title=《叶挺将军》掀高潮 高发惨遭不明人士骚扰 |accessdate=2011 |publisher=MSN中文网 |work= |language=zh-cn }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>高发获得“神似形不似”的好评。<ref>{{cite web |url=http://et.21cn.com/gundong/etscroll/2011/12/21/10213054.shtml |title=《叶挺将军》收官 高发“神似形不似”受好评 |accessdate=2011-12-21 |publisher=21CN |work= |language=zh-cn |archive-url=https://web.archive.org/web/20150512185450/http://et.21cn.com/gundong/etscroll/2011/12/21/10213054.shtml |archive-date=2015-05-12 |dead-url=yes }}</ref>

女演员汪裴出演叶挺妻子李秀文。<ref>{{cite web |url=http://ent.qq.com/a/20090302/000337.htm |title= 《叶挺将军》演绎叶挺夫人 汪裴被角色大爱感动|accessdate=2009年3月2日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

在2009年2月22日，[[中国中央电视台综合频道|中国中央电视台综合频道]]，该剧首播。<ref>{{cite web |url=http://ent.qq.com/a/20090218/000215.htm |title= 传记剧《叶挺将军》22日将登央视一套黄金档|accessdate= 2009年2月18日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

== 争议 ==
新四军政治部主任[[袁国平|袁国平]]之子、[[南京海军指挥学院|南京海军指挥学院]]将军[[袁振威|袁振威]]看过该剧之后，控诉该剧歪曲污蔑其父亲袁国平的形象，指出其父亲是自杀而不是在逃跑时被中国国民党军人追杀而死。<ref>{{cite web |url=http://ent.sina.com.cn/x/2009-06-09/05192554897.shtml |title=烈士后代指《叶挺将军》歪曲史实损害英雄形象 |accessdate=  2009年6月9日|publisher= 新浪网|work= |language =zh-cn}}</ref><ref>{{cite web |url=http://ent.163.com/09/0608/18/5BACFOU2000335TL.html |title=烈士后代指《叶挺将军》歪曲史实情节造假 |accessdate= 2009-06-08|publisher=网易 |work= |language =zh-cn}}</ref>
