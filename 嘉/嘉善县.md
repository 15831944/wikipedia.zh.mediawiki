'''嘉善县'''是[[中国|中国]][[浙江省|浙江省]]东北部的一个[[县|县]]，隸屬[[嘉兴市|嘉兴市]]管轄，地处太湖流域[[杭嘉湖平原|杭嘉湖平原]]，位于[[江苏省|苏]]浙[[上海市|沪]]三省市交汇处。

2013年2月28日，《浙江嘉善县域科学发展示范点建设方案》经[[国务院|国务院]]同意、由[[国家发改委|国家发改委]]正式批复实施。<ref>[http://www.gov.cn/zwgk/2013-04/28/content_2392900.htm 国家发展改革委关于浙江嘉善县域科学发展示范点建设方案的批复]，中国政府网，2013年2月28日</ref>

== 历史 ==
嘉善县历史悠久，明[[宣德|宣德]]四年（1429年）三月，巡抚、大理寺卿[[胡概|胡概]]巡视江南后，以地广赋繁奏请划增县，翌年（1430年）三月二十八日敕分[[嘉兴县|嘉兴县]]东北境之迁善、永安、奉贤三个乡和胥山、思贤、麟瑞三乡之部分置嘉善县，定治魏塘，隶嘉兴府。“因旧有迁善六乡俗尚敦庞，少犯宪辟，故曰嘉善”。清循明制，[[咸丰_(年号)|咸丰]]十年（1860年）至[[同治|同治]]二年（1863年），[[太平天国|太平天国]]曾在境内建立地方政权。 

[[民国|民国]]元年（1912年）废府，嘉善属钱塘道。民国16年（1927年），直属省辖。民国21年（1932年）全省设12个县政督察区，嘉善属第二县政督察区，民國22年（1933年）改称第二区。民国26年（1937年）11月被日军侵占。民国28年（1939年）成立浙西行署，次年（1940年）改为第十区，嘉善隶属浙西行署第十行政督察区。民国37年（1948年）4月，属第一行政督察区管辖。民國38年（1949年）5月11日，嘉善被[[中國共產黨|中國共產黨]]控制，隶属浙江省第一专员公署，11月改属嘉兴专署。1958年11月21日嘉善县建制撤销，并入嘉兴县。1961年4月9日恢复县置。1983年8月，实行市辖县制，嘉善县隶属嘉兴市。

== 地理 ==
境内一马平川，属典型的江南水乡。

== 行政区划 ==
嘉善县下辖3个[[街道办事处|街道]]、6个镇：
*街道：[[魏塘街道|魏塘街道]]、[[罗星街道_(嘉善县)|罗星街道]]、[[惠民街道_(嘉善县)|惠民街道]]
*镇：[[姚庄镇|姚庄镇]]、[[西塘镇_(嘉善县)|西塘镇]]、[[天凝镇|天凝镇]]、[[干窑镇|干窑镇]]、[[陶庄镇_(嘉善县)|陶庄镇]]、[[大云镇|大云镇]]

== 经济 ==
嘉善县经济发达，为全国最具实力的[[县级行政区|县市]]之一，2016年[[国内生产总值|GDP总量]]457亿元。

2013年4月，嘉善县成为浙江省第二批金融创新示范县。2014年5月，嘉善高铁产业新城正式开工奠基，主要由智慧产业园、城市商业综合体和五星级度假酒店组成，配套建设城市科技馆、城市大剧院等设施。<ref>[http://fccszt.fccs.com/jxjs/2015/0624kqc/index.html 嘉善高铁新城 创新城市的十年梦想]，嘉善房产超市网，2015年6月25日</ref>

== 教育 ==
嘉善县有三所高中，分别为[[嘉善高级中学|嘉善高级中学]]、[[嘉善第二高级中学|嘉善第二高级中学]]和[[嘉善中学|嘉善中学]]。
