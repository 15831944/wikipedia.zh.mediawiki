'''嘉兴市文物保护单位'''是中国[[浙江省|浙江省]][[嘉兴市|嘉兴市]]的市级[[文物保护单位|文物保护单位]]，截止到2017年共公布五批。

*第一批：1981年10月公布，数量不详。
*第二批：1992年10月公布，数量不详。
*第三批：2000年5月公布，数量不详。
*第四批：2009年9月10日公布，共18处<ref>{{cite web |title=《嘉兴市人民政府关于公布第四批嘉兴市级文物保护单位的通知》 |publisher=法搜网 |url=http://www.fsou.com/html/text/lar/171280/17128005.html}}</ref>。
*第五批：2010年3月29日公布，共9处，同时撤销1处<ref>{{cite web |title=《嘉兴市人民政府关于公布第五批嘉兴市级文物保护单位和撤销一处市级文物保护单位的通知》|publisher=嘉兴市政府 |url=http://www.jiaxing.gov.cn/zwxx/szfxx/zcwj_9983/jzf/201004/t20100406_174861.html}}</ref>。
