'''嘉明湖'''（[[布農語|布農語]]：'''Cidanumas-Buan'''）位於[[台灣|台灣]][[台東縣|台東縣]][[海端鄉|海端鄉]]，座落於[[三叉山|三叉山]]東側，距[[向陽山|向陽山]]直線距離7公里處，海拔3,310/11公尺，是[[台灣湖泊|台灣湖泊]]中第二高的高山湖泊，僅次於3,520公尺的[[雪山_(台湾)|雪山]][[翠池|翠池]]。[[台灣日治時期|日治時期]]稱為「{{lang|ja|シヌべシ池}}」（Shinubeshi池）<ref>台灣高山湖沼の話，台灣水產雜誌，1936年12月20日</ref>。湖面是橢圓形，湖水為湛藍色，湖面長約120公尺，寬約80公尺，面積約1.9公頃，湖水深度6公尺<ref>[http://jhuangwei.e-land.gov.tw/88/20080818025607.pdf 台灣高山湖泊發現小史] {{webarchive|url=https://web.archive.org/web/20131019160004/http://jhuangwei.e-land.gov.tw/88/20080818025607.pdf |date=2013-10-19 }}，2008年7月25日查閱</ref>，沒有任何山澗或溪流流入其中，湖水卻常保終年不枯。嘉明湖位於[[布農族|布農族]]傳統領域，布農族人稱呼其為「月亮的鏡子」<ref>[http://titv.ipcf.org.tw/news-11963 "月亮的鏡子" 布農宣告嘉明湖傳統名]，2015年7月22日查閱</ref>；又因其湖水顏色深藍如寶石，被登山界稱為「上帝遺失在人間之藍寶石」、「天使的眼淚」，是近年來著名的台灣深山景點<ref>[http://www.traveling.tw/taitung102090.php 台灣旅遊景點 - 台東縣 - 嘉明湖] {{webarchive|url=https://web.archive.org/web/20070927020221/http://www.traveling.tw/taitung102090.php |date=2007-09-27 }}，2008年7月10日查閱</ref>。

== 成因 ==
1998年，[[經濟部中央地質調查所|經濟部中央地質調查所]]所長[[陳肇夏|陳肇夏]]對嘉明湖採集樣品分析後，在2000年11月，於[[中國地質學會|中國地質學會會刊]]發表論文，以在當地發現微角[[礫岩|礫岩]]中的[[石英|石英]]沙粒極度碎裂、石英熔融留下的玻璃絲、不屬於當地原生[[岩盤|岩盤]]的[[鎳|鎳]]、[[銅|銅]]等金屬碎片，提出嘉明湖是[[隕石|隕石]]撞擊地球所造成的[[隕石坑|隕石坑]]的理論，並認為撞擊年代約在數萬年到一萬年前。

不過，同為[[中央地質調查所|中央地質調查所]]的學者[[朱傚祖|朱傚祖]]卻以無法在嘉明湖找到隕石坑應有的[[柯石英|柯石英]]與[[斯石英|斯石英]]；2004年，中央地質調查所出版的《[[地質_(期刊)|地質]]》期刊中，[[馮奎智|馮奎智]]等人發表〈[[嘉明湖畔石英脈變質構造之研究|嘉明湖畔石英脈變質構造之研究]]〉也證實在嘉明湖畔的[[石英|石英]]脈並無撞擊變質的現象，因此均不支持[[隕石|隕石]]說。

而2003年，當時的[[高雄師範大學|高雄師範大學]]地理系主任[[齊士崢|齊士崢]]教授數次的野外調查，依照嘉明湖現場的[[岩坎|岩坎]]（冰坎）、[[堆積堤|堆積堤]]（端冰磧）及擦痕，以及堆積堤的沉積相具有冰河沉積相之地形與沉積物的空間組合、分布，認為嘉明湖為[[冰河|冰河]]所遺留的[[冰斗|冰斗]]地形，並在2010年，取得嘉明湖出口[[端冰磧|端冰磧]]的[[光螢光定年|光螢光定年]]年代為距今6.47±0.93ka(6470±930年)，也符合了台灣最後一次在[[全新世|全新世]]早期的冰河作用年代。雖然台灣的冰河地形研究最早始於[[台灣日治時期|日治時期]]的學者，但直到1990年代起才又陸續發表而證實[[台灣|台灣]]在海拔3000公尺以上的[[高山地區|高山地區]]大多曾受到[[冰河作用|冰河作用]]而遺留[[冰河地形|冰河地形]]，因此，從[[地形學|地形學]]與[[沉積學|沉積學]]的角度均可說明嘉明湖為[[冰斗|冰斗湖]]，在[[學術界|學術界]]已無爭議。

== 山難 ==
=== 吳子暉山難 ===
* 2003年4月26日，新竹友達登山社：1
: 黃郁明（34歲，高雄縣人）
: 王科順（29歲，台北市人）
: 陳琬琳（27歲，嘉義縣人）
: 溫嘉怡（27歲，台北市人）
: 陳添賜（28歲，台中縣人）
: 彭建凱（29歲，新竹市人）
: 江明峰（28歲，台北縣人）
: 劉竹育（28歲，台中縣人）
: 吳子暉（29歲，清華大學物理所博士生）等九人。
: 由台東縣海端鄉南橫公路台東段向陽山區入嘉明湖，吳子暉與其他兩名隊員在週日（4月27日）中午率先抵達嘉明湖，由於當時天氣炎熱、景致美麗，第一次到嘉明湖的吳子暉向友人說要下水消暑，其他兩人則往回走要拿相機拍照。
: 吳裕明說，當他們再度返回嘉明湖時，竟然不見吳子暉蹤跡，沿著湖邊搜尋，只看到吳子暉的衣服、眼鏡放在一旁，因此研判吳子暉可能溺水，等其他隊員抵達後，才攀爬到高處以無線電求救。

=== 李奇叡山難 ===
2005年2月26日下午，國立中正大學登山社18名師生在向陽派出所登記入山前往嘉明湖，預計2月28日下午下山，但是清晨五時許，物理系二年級學生李奇叡忽然發生高山症，領隊林柏川以無線電向外求援。
: 2月27日，2100時，發現李奇叡身體不適。
: 2月28日，等到凌晨不見好轉。
: 2月28日，0200時，嚮導下山求救。
: 2月28日，0345時，留守獲嚮導求救電話。
: 2月28日，0900時，求救同學（筆者按：根據聲明稿的上下文看來應該是指先行下山的 18 名隊員在下山途中）與搜救人員會合。
: 2月28日，1200時，救難人員抵達嘉明湖避難小屋與五名照顧奇叡的隊員會合：另外情況良好的十八名學生，也同時於中午十二時抵達向陽工作站。
: 2月28日，2200時，山上天氣驟變，下大雪積雪深厚，搜救人員與學生前進困難。
: 3月1日，0143時，李同學已停止呼吸。
