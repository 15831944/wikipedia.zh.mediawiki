'''嘉莉·法蘭西絲·費雪'''（{{lang-en|'''Carrie Frances Fisher'''}}，{{bd|1956年|10月21日|2016年|12月27日}}）是一名美国女演员、小说家、剧作家、表演艺术家。她是歌手[[艾迪·費雪|艾迪·費雪]]與女演員[[黛比·雷諾|黛比·雷諾]]的女兒。她最出名的演出是在《[[星際大戰|星際大戰]]》三部曲（1977年至1983年）及《[[星球大戰：原力覺醒|星球大戰：原力覺醒]]》（2015年）中饰演[[莉亞公主|莉亞公主]]。其他知名作品包括電影《{{link-en|洗髮精 (電影)|Shampoo (film)|洗髮精}}》（1975年）、《[[福祿雙霸天|福祿雙霸天]]》（1980年）、《[[姊妹情深|姊妹情深]]》（1986年）、《[[地獄來的芳鄰|地獄來的芳鄰]]》（1989年）和《[[90男歡女愛|90男歡女愛]]》（1989年）。

她还写了半自傳小说《{{link-en|來自邊緣的明信片 (小說)|Postcards from the Edge|來自邊緣的明信片}}》，并担任[[來自邊緣的明信片_(電影)|同名電影]]编剧，還有她的自傳一人秀和根據其表演所著的非小說《''{{link-en|Wishful Drinking|Wishful Drinking}}''》。她同時也是一名{{link-en|劇本醫生|Script doctor}}，替其他編劇的電影劇本加工<ref>{{Cite web |url = http://www.independent.co.uk/arts-entertainment/films/news/carrie-fisher-dead-star-wars-script-doctor-a7497951.html |title = Carrie Fisher wasn't just a great actress, she was one of Hollywood's best script doctors |author = Clarisse Loughrey |date = 2016-12-27 |publisher = The Independent |language = en}}</ref>。2015年起，費雪亦於電視劇《[[大禍臨頭|大禍臨頭]]》中參演。在晚年，她在公開地講述有關她的心理健康狀況和毒癮經歷而得到讚揚。<ref>{{Cite web |url = https://www.boxoffice.harvard.edu/Online/default.asp?BOparam::WScontent::loadArticle::permalink=humanism |title = Carrie Fisher: Cultural Humanism Award |publisher = Harvard Box Office |language = en}}</ref>

2016年12月23日，費雪在跨越大西洋的長途機上陷入[[心搏停止|心搏停止]]狀態，2016年12月27日去世，享壽60歲。
