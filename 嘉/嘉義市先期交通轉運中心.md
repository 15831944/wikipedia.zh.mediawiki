'''嘉義市先期交通轉運中心'''，又稱作'''嘉義轉運站'''<ref>{{cite web | language = zh-tw | author=國光客運 | title =營業據點-【嘉義轉運站】| url=http://www.kingbus.com.tw/guideBranchDetail.php?cid=10&pid=31|accessdate = 2018-02-02
|archivedate = 2018-02-02|deadurl = no}}</ref><ref>{{cite web | language = zh-tw | author=統聯客運 | title =據點查詢-雲嘉地區/嘉義市| url=http://www.ubus.com.tw/html/stronghold/stronghold_show.php?ix=1&kind1=5&kind2=20|accessdate = 2018-02-02
|archivedate = 2018-02-02|deadurl = no}}</ref><ref>{{cite web | language = zh-tw | author=台中客運 | title =路線圖| url=http://www.tcbus.com.tw/image/lineimage.php?imagetest=6187|accessdate = 2018-02-02
|archivedate = 2018-02-02|deadurl = no}}</ref>
，是位於[[台灣|台灣]][[嘉義市|嘉義市]][[西區_(嘉義市)|西區]]的一個[[台灣汽車客運|汽車客運]]轉運站，位於中興路1號，為與[[嘉義車站|嘉義車站]]後站(西口)共構之複合式轉運中心<ref>{{cite web | language = zh-tw | author=台灣鐵路管理局嘉義車站 | title = 車站位置及周邊交通轉乘資訊| url= https://www.railway.gov.tw/Chiayi/CP.aspx?SN=13928|accessdate = 2017-12-03
|archivedate = 2017-12-01|deadurl = no}}</ref>，亦為台灣第一座複合式客源轉運中心，由國光客運[[子公司|子公司]][[國光商行|國光商行]]營運<ref>{{cite web | language = zh-tw | author = Peopo公民新聞 | title = 嘉義市全台首座無接縫複合交通轉運中心啟用 | url = https://www.peopo.org/news/69818 | accessdate = 2017-12-03 | archiveurl = https://web.archive.org/web/20171203224623/https://www.peopo.org/news/69818 | archivedate = 2017-12-03 | deadurl = yes }}</ref>。由[[國光客運|國光客運]]以營運後轉移模式（OT）取得為期7年的地上營運權後，由[[嘉義市政府|嘉義市政府]]所興建的一座地上2層、地下一層樓，並分前後棟的建築，做為整合國道客運及一般公路客運，並設有商店街的多功能轉運中心<ref>{{cite web | language = zh-tw | author=交通部觀光局借問站 | title = 國光商行股份有限公司| url= http://www.taiwan-askme.tw/shop/13|accessdate = 2017-12-03
|deadurl = no}}</ref>。嘉義市先期交通轉運中心於公元2011年1月20日啟用，為嘉嘉雲地區規模最大之轉運中心。

==沿革==
轉運中心最初為配合[[嘉義市區鐵路高架化計畫|嘉義市區鐵路高架化計畫]]預留高架化嘉義車站站體與商業空間之興建、[[嘉義公車捷運|嘉義公車捷運]]高鐵聯外系統之月台規劃、嘉義市國道客運、公路客運等整合運用及嘉義車站前站用地重新規劃之用，並擴大嘉義車站特定區的可利用土地與轉乘空間。在國光客運接手營運後並納入商店街與各項公共服務設施之經營管理。營運期間[[嘉義客運|嘉義客運]]之[[嘉義市公車|嘉義市公車]]路線亦曾進駐轉運中心前棟月台，但改由[[嘉義縣公共汽車管理處|嘉義縣公共汽車管理處]]接手之後便撤出轉運站並將市公車站牌移至轉運站外博愛路上，改稱後火車站。
