'''嘉悅'''（{{lang-en|'''Twin Peaks'''}}），位於[[香港|香港]][[新界|新界]][[將軍澳南|將軍澳南]][[至善街|至善街]]9號，是[[嘉華國際集團|嘉華國際集團]]旗下的私人住宅屋苑，建築師為{{link-en|王董建築師事務|Wong Tung & Partners}}。項目於2015年前命名為稱The Big East。物業分為兩座樓高25層的大廈，設計特色為雙拱門設計（是第4個採此類設計的樓宇），共提供372個單位，2015年3月中開售，於2016年第4季入伙<ref>{{cite news|title=嘉悅 將軍澳版凱旋門 高層向南享海景 |url=http://property.mpfinance.com/cfm/pd3.cfm?File=20150321/pdg01/gma1.txt |language=zh-hk |publisher=明報 |date=2015年3月21日 |accessdate=2015年4月26日 }}</ref><ref>{{cite news|title=80後搶購嘉悅兩房戶 積蓄80萬不足付首期 須父資助40萬 |url=http://hk.apple.nextmedia.com/financeestate/first/20150401/19097498 |language=zh-hk |publisher=蘋果日報 |date=2015年4月1日 |accessdate=2015年4月26日 }}</ref>。管理公司為仲量聯行（物業管理）有限公司，每呎管理費逾3.9元。

== 單位 ==
物業分為兩座樓高24層的大廈，每層一梯九伙，提供372伙單位，採1房至4房間隔，實用面積約300餘至約1,700餘平方呎，主打2房及3房，佔約七成。其中105伙單位採用開放式廚房設計，只設電磁爐煮食。而單位樓底高度為3.325米，比同區其他項目樓底高。<ref>{{cite news|title=【即時】嘉悅樓底高逾3.3米 採光好又通爽|url=http://ps.hket.com/content/568142|language=zh-hk |publisher=蘋果日報|date=2015年3月18日 |accessdate=2016年4月22日}}</ref>

物業有21伙屬特色單位，包括8伙複式戶和13伙連平台戶，實用面積由324至1,743平方呎，複式戶以招標形式推售。

== 設施 ==
屋苑地下至1樓為住客會所及商場，2樓為有蓋園林。住客會所總面積達14,565平方呎，設施包括18米長室外游泳池、[[健身室|健身室]]、多用途室內運動場、派對天地、露天茶座、鋼琴房、兒童遊樂場及宴會廳等設施。另外亦設有14個單車位供可住戶租用。

地庫至地下設停車場，提供165個車位。

== 商場 ==
租戶以幼稚園及地產代理為主。

== 宣傳推廣 ==
嘉悅的廣告宣傳「以嘉多一點、悅目新角度」為主題。廣告亦運用航拍及[[GoPro|GoPro]]錄像器材拍攝，展示[[西貢|西貢]]及[[將軍澳|將軍澳]]景色。另外，在2017年9月，嘉華地產向[[香港大學|香港大學]]建築學院捐贈本項目的建築模型（曾於售樓時使用），以供建築學課程學生教學用途。
