'''嘉義仁武宮'''為臺灣嘉義市之古蹟，公告於民國八十七年（1998年）4月30日，是一間主祀[[保生大帝|保生大帝]]的廟宇{{r|文}}。該廟主神據說是在[[鄭經|鄭經]]西征時從今天高雄市[[仁武區|仁武區]]調到諸羅地區鎮守的部隊隨迎來的，諸羅知縣[[毛鳳綸|毛鳳綸]]建廟時即因主神是仁武鎮官兵迎來而將廟名取為「仁武宮」{{r|諸}}。日治時期由於嘉義市尹[[川添修平|川添修平]]廢合寺廟的措施，使得仁武宮除了開基保生大帝神像與道光年間石香爐等以外的文物全部遺失{{r|嘉}}。

== 沿革 ==
仁武宮的主神保生大帝是明鄭仁武鎮官兵自泉州所迎來，他們從今天高雄仁武調到今嘉義市一帶駐守時亦將神像迎去，而後建小祠祭祀，作為營鎮的守護神{{r|諸}}。清康熙四十年（1701年）時正式建廟，諸羅知縣毛鳳綸因為主神是仁武鎮官兵所迎來的，遂將廟宇命名為「仁武宮」{{r|諸}}。

日治末期的[[皇民化運動|皇民化運動]]期間，嘉義市尹川添修平大舉廢除嘉義市內廟宇，將主神神像全集中到[[嘉義城隍廟|嘉義城隍廟]]，仁武宮亦受影響，在昭和十三年（1938年）時將神像寄奉在城隍廟{{r|文}}。之後仁武宮被日人伴八郎拿來開設南和商行，期間造成仁武宮除神像及古香爐等物以外的文物全部遺失{{r|文}}。

二次大戰之後，嘉義市的仕紳才在民國三十四年（1945年）集資將仁武宮產權買回來，於民國三十六年正月初一（1947年1月22日）安座{{r|嘉}}。之後從1961年起，多次陸續整修廟宇各處{{r|嘉}}。

== 建築 ==
仁武宮原為兩進兩廊式建築，後增建而變成三進兩廊式，但大致上仍有廟宇原貌{{r|諸}}。廟門的文武門神為[[潘麗水|潘麗水]]所繪，具有藝術價值{{r|嘉}}。

== 祀神 ==
仁武宮的主神為保生大帝，除了據說是康熙年間建廟時供奉的軟身神像外，還有一尊是太子太保[[王得祿|王得祿]]於嘉慶二十三年（1818年）所贈奉的神像{{r|諸}}。

而除了主神之外，廟裡所從祀的諸神還有左右護法康元帥與趙元帥、三十六神將等，之後又增祀[[中壇元帥|中壇元帥]]、[[地藏王菩薩|地藏王菩薩]]、[[觀世音菩薩|觀世音菩薩]]、[[濟公活佛|濟公活佛]]、[[關聖帝君|關聖帝君]]、[[註生娘娘|註生娘娘]]、[[三官大帝|三官大帝]]等神明{{r|諸}}。
