'''嘉北國民小學 '''，簡稱'''嘉北國小'''，位於[[嘉義市|嘉義市]]東區，於1978年成立<ref name="嘉義市志">{{cite book |author=姜添輝編纂 |title=嘉義市志．卷六．教育志 |url=http://memory.ncl.edu.tw/tm_cgi/hypage.cgi?HYPAGE=publication_local_detail.hpg&subject_name=%E9%A4%A8%E8%97%8F%E5%85%89%E5%BE%A9%E5%BE%8C%E5%8F%B0%E7%81%A3%E5%9C%B0%E5%8D%80%E5%AE%98%E4%BF%AE%E5%9C%B0%E6%96%B9%E5%BF%97%E7%9B%AE%E9%8C%84&subject_url=publication_local_category.hpg&project_id=local&dtd_id=51&xml_id=0000348110|publisher=嘉義市政府 |date=2002 |page=頁315－321}}</ref>，自民國1995年起在學學生人數超過[[嘉義市東區崇文國民小學|崇文國小]]而成為嘉義市班級數與學生數最多的國民小學<ref>{{cite web |author1=教育部 |title=國民中小學校概況統計 |url=https://www.edu.tw/News_Content.aspx?n=829446EED325AD02&sms=26FB481681F7B203&s=C0D2B286EA66BBD5 |website=https://www.edu.tw/ |accessdate=2018-10-16}}</ref><ref>{{cite book |author1=張剛政 |editor=國立教育研究院籌備處 |title=校長傳心三峽情 |date=2006-03 |publisher=國立教育研究院籌備處 |isbn=9789860047288 |pages=89-96 |url=https://www.govbooks.com.tw/books/62382 |accessdate=2018-10-16}}</ref>。

==學校沿革==
1975年2月5日開始籌備設立該校，於1978年11月3日正式成立'''嘉義縣嘉義市嘉北國民小學'''，第一年共招收十個班級。成立時校地1.7632公頃，原本土地為[[臺灣省農業改良所|臺灣省農業改良所]]實驗用地，於1974年變更為學校用地<ref name="嘉義市志"/>。

1982年7月1日，因嘉義市升格為[[省轄市|省轄市]]，學校更名為'''嘉義市嘉北國民小學'''<ref>{{cite web |title=各級學校沿革 |url=http://www.cy.edu.tw/client/NewsDetail.php?REFDOCTYPID=0kwfo3m0nsjagekm&REFDOCID=0pek8ek5ng20u7ij&MBRIFOTYPID=0kxo4qytb6icajy2&Page_Num=1 |website=www.cy.edu.tw |publisher=嘉義市教育處 |accessdate=2018-10-11}}</ref>。
