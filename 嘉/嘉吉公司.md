'''嘉吉公司'''（Cargill, Incorporated）是[[美國|美國]]最大（以2013年的年營收論）的私人[[跨國企業|跨國企業]]，主要事業是[[食品加工|食品加工]]，版圖現已擴大至[[醫藥|醫藥]]、[[金融|金融]]、[[天然資源|天然資源]]等。

== 历史 ==
嘉吉公司由William Wallace Cargill創立於1865年，總部設於[[明尼蘇達州|明尼蘇達州]]的[[明尼亞波里斯|明尼亞波里斯]]郊區[[Minnetonka|Minnetonka]]。2013年的年營收達1,360億美元，可排入2013年度[[上市|上市]]企業營收排行榜[[Fortune_500|Fortune 500]]的第9名。

==中美嘉吉公司==
[[台灣|台灣]]的中美嘉吉公司是美商嘉吉公司透過在台百分之百持股之子公司—嘉冠投資公司於1970年與[[台糖公司|台糖公司]]合資成立；其中美商嘉冠投資公司持股60%，台糖持股40%。主要營業項目為產銷各種家禽及家畜飼料，共有兩座飼料廠分別座落於台中市大肚區與高雄市前鎮區。

其中高雄廠區1971年10月17日落成，佔地3,000坪，投資額為新台幣1億元；並已於2016年2月25停工撤廠，將進行土地重劃開發。<ref>{{Cite web |url = http://www.taisugar.com.tw/Monthly/CPN.aspx?ms=1401&s=13385479&p=13385639 |title = 台糖投資事業專題系列—中美嘉吉（股）公司 陪著台灣經濟起飛 |author = 台糖公司企劃處 |date = 2015年11月 |publisher = 台糖公司 |language = 中文 |accessdate = 2017年5月23日 }}</ref>
