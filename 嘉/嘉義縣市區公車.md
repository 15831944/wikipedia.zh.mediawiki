'''嘉義縣市區公車'''，為[[嘉義縣政府|嘉義縣政府]]主管（承辦單位：建設處）之公車路線，以嘉義縣為營業區域，並有部分路線依《汽車運輸業管理規則》第42條延長至[[嘉義市|嘉義市]]及[[台南市|台南市]]。原由[[嘉義縣公車處|嘉義縣公車處]]獨家經營，其後另有新業者[[路得寶交通|路得寶交通]]於2017年1月成立[[阿里山客運|阿里山客運]]來投標並營運新路線166路(高鐵鹽水線)及168路(西濱快捷線)，並擴大服務至南瀛地區的[[鹽水區|鹽水區]]及[[北門區|北門區]]，共計2家業者。<ref>http://www.chinatimes.com/newspapers/20161208000065-260202</ref><ref>{{cite web |url=https://www.1111.com.tw/job-bank/job-description.asp?eNo=79716055 |title=存档副本 |accessdate=2017-01-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20170116145534/https://www.1111.com.tw/job-bank/job-description.asp?eNo=79716055 |archivedate=2017-01-16 }}</ref><ref>{{cite web |url=https://www.1111.com.tw/job-bank/job-description.asp?eNo=79715996 |title=存档副本 |accessdate=2017-01-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20170116145143/https://www.1111.com.tw/job-bank/job-description.asp?eNo=79715996 |archivedate=2017-01-16 }}</ref><ref>{{cite web |url=https://www.1111.com.tw/job-bank/job-description.asp?eNo=79716040 |title=存档副本 |accessdate=2017-01-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20170116143905/https://www.1111.com.tw/job-bank/job-description.asp?eNo=79716040 |archivedate=2017-01-16 }}</ref>目前嘉義縣市區公車共計有9條路線，其中168路另有A、B兩條支線。

==概要==
嘉義縣市區公車目前由[[嘉義縣公車處|嘉義縣公車處]]及[[阿里山客運|阿里山客運]]營運。過去[[嘉義客運|嘉義客運]]亦曾自行開闢過嘉義縣市區公車路線，但因經營不善而停駛，其中[[故宮南院|故宮南院]]開館後，嘉義客運原停駛的906改號公路客運7235復駛，並增加繞駛故宮南院。

==歷史==
1982年嘉義市恢復為[[省轄市|省轄市]]建制前，當時的嘉義縣嘉義市（縣轄市）即有市區公車，由嘉義縣政府主管；改制後由[[嘉義市政府|嘉義市政府]]主管，請參閱[[嘉義市公車|嘉義市公車]]頁面。現今的嘉義縣市區公車是2004年開辦，初有「嘉義－塘興村」1條路線，由[[嘉義縣公共汽車管理處|嘉義縣公共汽車管理處]]兼營。2007年[[嘉義客運|嘉義客運]]加入，陸續闢駛「高鐵嘉義站－北港」(改編為7235)、「嘉義－中埔」、「北港－梅山」、「朴子－溪口」等路線，至2015年全部停駛、退出營運，後「高鐵嘉義站-北港」路線復駛並改為公路客運7235並行經[[故宮南院|故宮南院]]。後由嘉義縣公共汽車管理處獨家經營，直至2017年1月25日採純[[電動巴士|電動巴士]]行駛的新業者[[阿里山客運|阿里山客運]]加入成為第二家業者為止。
