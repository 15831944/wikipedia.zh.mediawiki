'''財團法人嘉義縣私立嘉惠教養院'''（[[英文|英文]]：Foundation of Private Chia Huey Nursing Institution in ChiaYi County），簡稱'''嘉惠教養院'''，是一間位在[[嘉義縣|嘉義縣]][[水上鄉|水上鄉]]的教養院，建立於[[民國75年|民國75年]]，由已故院長  [[呂賴怡靜|呂賴怡靜]][[女士|女士]]擔任第一任院長，對[[身心障礙者|身心障礙者]]提供陪伴終身到老的[[服務|服務]]，使[[身心障礙者|身心障礙者]]亦能得到妥善的照護，謀得求生、求存之道，以減輕家庭負擔，增進[[社會福利|社會福利]]制度與[[品質|品質]]。

== 創立原因 == 
已故院長 [[呂賴怡靜|呂賴怡靜]][[女士|女士]]在前院長 [[呂嘉來|呂嘉來]][[先生|先生]]就讀[[國小|國小]]三年級的時候罹患[[胰臟炎|胰臟炎]]，[[母親|母親]]在病榻忍受病魔折騰時發願，倘若病癒就願成立教養院以照顧社會上那些殘疾孤苦的人。兩年後呂賴怡靜女士的病情逐漸好轉，便以身邊的積蓄在[[台南縣|台南縣]][[永康區|永康鄉]]租地創辦'''嘉惠教養院'''，後來輾轉遷院址到[[嘉義縣|嘉義縣]][[水上鄉|水上鄉]]，[[民國八十二年|民國八十二年]]由第二任院長[[呂嘉來|呂嘉來]]接手，現任院長為[[陳菲菲|陳菲菲]]，自在[[民國九十三年|民國九十三年]][[十二月|十二月]]六日正式完成立案，收容本國國民年滿[[三歲|三歲]]以上至[[六十五歲|六十五歲]]領有[[身心障礙手冊|身心障礙手冊]]之輕、中、重度、極重度[[身心障礙者|身心障礙者]]及多重障礙者以及經由各[[直轄市|直轄市]]、[[縣(市)政府|縣(市)政府]]轉介者。

== 服務對象 ==
* 凡年滿[[三歲|三歲]]以上至[[六十五歲|六十五歲]]，領有[[身心障礙手冊|身心障礙手冊]]之輕、中、重、極重度[[身心障礙者|身心障礙者]]。
* 經由[[嘉義縣政府|嘉義縣政府]]轉介者，以列冊[[低收入戶|低收入戶]]及[[嘉義縣政府|嘉義縣政府]]轉介個案為優先收容。

== 服務項目 ==
* 入院前[[諮詢|諮詢]]、檢測及轉介
* 個案教學與[[評估|評估]]
* [[復健|復健]]護理、衛教訓練及保健[[預防|預防]]等服務
* [[社會|社會]]適應能力訓練

== 工作組織 ==
* [[教保|教保]]組:掌理院生教養、生活訓練、職業陶冶及職業訓練。
* [[社工|社工]]組:掌理院生個案[[輔導|輔導]]、各界來賓參觀、[[諮詢|諮詢]]服務及[[志願服務|志願服務]]推展事項。
* [[醫務|醫務]]組:掌理院生[[疾病預防|疾病預防]]、[[復健|復健]]訓練、衛生教育及保健服務事項。
* [[行政|行政]]室:掌理[[文書|文書]]、[[出納|出納]]、財產購置管理及總務，並依法辦理[[人事管理|人事管理]]、考核和[[福利|福利]]事項。
* [[會計|會計]]室:依法辦理[[歲計|歲計]]、[[會計|會計]]、[[統計|統計]]等事項。
