'''嘉義營林俱樂部'''為臺灣嘉義市之古蹟，公告於民國八十七年（1998年）4月30日{{r|文}}，是供臺灣總督府營林機關賓客與派駐嘉義地區職員的休閒場所{{r|諸}}。

== 沿革 ==
營林俱樂部約是興建於大正三年（1914年）左右，隸屬於嘉義營林所出張所，為臺灣總督府營林局的休閒娛樂場所{{r|嘉}}。當時周圍是營林局的官舍群，而出張所與[[北門車站_(嘉義)|北門驛]]也在附近{{r|諸}}。

二次大戰後，該建築除作為林務局禮堂、標售杉木的場所外，在民國三十七年（1948年）時曾作為「忠孝幼稚園」使用{{r|文}}{{r|嘉}}。而後在民國七十三年（1984年）時法務部調查局向林務局借用此建築為嘉義縣調查站，進行整修改建之後於隔年正式遷入，七十六年（1987年）時歸還{{r|文}}。在這之後，林務局又將該建築作為單身員工宿舍{{r|嘉}}。

該建築已於2009年由林務局整修完成{{r|文}}，目前作為檜藝森活村店舖營業及展覽用。

== 建築 ==
營林俱樂部為臺灣唯二的[[都鐸式|都鐸式]]建築<ref>{{Cite web |url=https://www.hinokivillage.com.tw/ |title=檜意森活村 |access-date= }}</ref>，而另一個都鐸式建築為[[臺北故事館|臺北故事館]]。建築構造為半木構造，平面略呈長方形，佔地1562平方公尺（建築面積195平方公尺），高4.2公尺，寬20公尺，長10公尺{{r|諸}}。其外牆為雨淋板，入口設有凸窗，上方的山牆為了裝飾除直線木板外還釘有曲線形木板，而屋頂上還設有小尖塔{{r|諸}}。
