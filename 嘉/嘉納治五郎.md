'''嘉納治五郎'''（{{jpn|j=嘉納 治五郎|hg=かのう じごろう|rm=Kanō Jigorō}}，{{bd|1860年|10月28日|1938年|5月4日}}），日本神戶人，[[日本|日本]][[柔道|柔道]]的[[創始人|創始人]]。

== 生平 ==
嘉納治五郎生於[[日本|日本]][[神戶|神戶]]奈特，出身名門生源寺家族，母親是當地釀酒業大戶的女兒。因為從小身體不好，受人欺負，於是開始學習傳統的[[柔術|柔術]]。1871年隨父母移居東京，1877年就讀[[東京大學|東京大學]]，主修[[哲學|哲學]]，畢業後開始從事教育工作。1886年任舊制第五高等學校校長，1894年升任[[東京高等師範學校|東京高等師範學校]]校長。他也曾在日本的[[文部省|文部省]]擔任部門的主管，曾为[[中国|中国]]留学生开设[[东京弘文学院|东京弘文学院]]。他在教育界一共服務了二十六年的時間。

他在東京大學就讀時，跟隨福田八學習天神真揚流[[柔術|柔術]]，福田八過世後，繼續跟隨其同門磯正智深造。磯正智死後，進入飯久保恆門下學習起倒流。

1882年（明治15年），嘉納治五郎在東京成立[[講道館|講道館]]，與數名學生開始教授與研究[[柔術|柔術]]，這是柔道的起源。明治18年（1886年），講道館參加[[警視廳|警視廳]]武術大賽，大獲全勝，擊敗實力堅強的戶塚流，一時聲名大起。

== 逸話 ==
1891年1月由歐美視察歸國途中，曾豪邁的使用投技把對其挑釁的俄國士兵摔飛。經同年4月10日的[[讀賣新聞|讀賣新聞]]報導後才為人所知。
== 影響及貢獻 ==
柔道是日本最先廣傳到世界各地的武術，同時現在也是[[奧運|奧運]]的正式比賽項目之一。嘉納對現今武術最重要的影響，就是發明以色帶的方式來區分段位，許多的武術現在都學習嘉納的方法來區分學員的功力。

他同時也是將柔道和[[劍道|劍道]]列入日本學生必修課程的幕後推手。同時嘉納也是第一位擔任[[國際奧林匹克委員會|國際奧林匹克委員會]][[國際奧林匹克委員會委員列表|委員]]一職的亞洲人。
