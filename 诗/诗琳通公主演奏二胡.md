《'''诗琳通公主演奏二胡'''》是由尼迪功·盖威迁{{NoteTag|在诗琳通公主的摄影图集《光是色，色是光》（{{lang-th|แสงคือสี สีคือแสง}}，{{ISBN|9789747628616}}）中翻译为“尼提功·格莱威仙”。}}（{{lang-th|นิติกร กรัยวิเชียร}}，{{lang-en|Nitikorn Kraivixien}}）拍摄的众多[[摄影作品|摄影作品]]中的其中一张<ref>{{cite web |url=http://www.chaoprayanews.com/2015/12/27/%E0%B8%AB%E0%B8%99%E0%B8%B1%E0%B8%87%E0%B8%AA%E0%B8%B7%E0%B8%AD-%E0%B8%9B%E0%B8%B4%E0%B8%A2%E0%B8%A3%E0%B8%B2%E0%B8%8A%E0%B8%81%E0%B8%B8%E0%B8%A1%E0%B8%B2%E0%B8%A3%E0%B8%B5-%E0%B9%81/ |title=หนังสือ “ปิยราชกุมารี” และ ภาพถ่ายฝีพระหัตถ์ “อยู่มานาน กาลเวลาพาสุข” |last= |first= |date=2015-12-27 |website=昭披耶新聞 |publisher= |access-date= |quote=}}</ref>，照片高34.3厘米，宽23.8厘米，现藏于[[北京|北京]][[国际友谊博物馆|国际友谊博物馆]]。1990年4月{{noteTag|其历程后来写成游记《[[平沙万里行|平沙万里行]]》。}}，泰国[[诗琳通公主|诗琳通公主]]将这张照片赠送给當時的中国时任国家主席[[楊尚昆|楊尚昆]]<ref>{{cite web |url=http://alumni.whu.edu.cn/info/1041/7275.htm |title=诗琳通公主诗选 |last= |first= |date=2010-03-29 |website=武汉大学 |publisher=校友会 |access-date=2019-02-08 |quote=}}</ref>。

== 细节 ==
照片中诗琳通公主身穿[[水蓝色|水蓝色]]{{link-en|立领|Mandarin collar}}长裙，头戴白色[[发夹|发夹]]，脖子上挂着一条带有[[肖像|肖像]]的[[项链|项链]]，赤足跪坐演奏着白色的泰式[[二胡|二胡]]（{{link-en|都旺胡|Saw duang}}），琴杆上挂有一个[[花环_(泰国)|花环]]。[[相框|相框]]为泰式装饰，上部有诗琳通公主的王家标志。在照片中的腰际处上，有公主本人的亲笔[[签名|签名]]“{{lang|th|สิรินธร}}”。
