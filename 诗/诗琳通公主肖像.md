《'''诗琳通公主肖像'''》是[[四川省|四川省]][[南充市|南充市]][[蓬安县|蓬安县]]画家[[卢霖|卢霖]]的[[水墨画|水墨画]]作品，曾在2015年9月18日于“庆祝中泰建交40周年暨[[2015年诗琳通公主60寿辰|诗琳通公主殿下60寿辰]]中国书画慈善展”中展出<ref>{{cite web |url=http://www.xinhuanet.com/overseas/2015-09/22/c_128254081.htm |title=中泰建交40年 中国艺术家献礼泰国公主60寿辰 |last= |first= |date=2015-09-22 |website=新华网 |publisher=华西都市报 |access-date=2018-11-18 |quote=}}</ref><ref>{{cite web |url=http://www.sohu.com/a/78304018_119824 |title=[人物]蓬安七旬翁 画作赠给泰国公主 |last= |first= |date=2016-05-29 |website=搜狐 |publisher=南充新闻网 |access-date=2018-11-18 |quote=}}</ref>，在该展览的作品集中，该画作放在了首页位置。之后被泰国企业家以200万[[泰銖|泰铢]]收购<ref>{{cite web |url=http://news.ncxww.com/hot/2015/0924/4454.html |title=蓬安山水入画引得公主驻足 |last= |first= |date=2015-09-24 |website=南充新闻网 |publisher=南充日报 |access-date= |quote=}}</ref><ref name="发展中国" />。

== 内容 ==
[[File:Portrait_of_Princess_Sirindhorn_by_Nitikorn_Kraivixien.jpg|thumb]]
[[File:태국_방콕_-_올드앤션트_파크_1_(cropped).JPG|thumb]]
画中的[[诗琳通公主|诗琳通公主]]为年轻时的模样。[[黑发|黑色短发]]，头上别着白色[[发夹|发夹]]；上身穿蓝紫色{{link-en|立领|Mandarin collar}}窄长袖上衣，披有一条黄色带[[流苏|流苏]]的{{link-en|纱摆|Sabai|披肩}}，上有花藤、{{link-en|白象 (动物)|White elephant (animal)|白象}}和[[却克里王朝|却克里王朝]]的标志，斜披[[金色|金色]]王家[[饰带|绶带]]，上面挂有两枚[[勋章|勋章]]（分别为{{link-th|拉玛九世拉达那蓬勋章|เหรียญรัตนาภรณ์ รัชกาลที่ 9}}和{{link-en|杜沙迪马拉奖章|Dushdi Mala Medal}}）；下身穿镶金边的紫色泰式传统长裙，脚蹬金色{{link-en|宫廷鞋|Court shoe|浅口鞋}}<ref name="发展中国">{{cite web |url=http://www.cdf.bj.cn/LiterArt/details.aspx?id=8103 |title=名家观察｜卢林艺术作品 |last= |first= |date=2018-09-17 |website=发展中国 |publisher= |access-date=2018-11-18 |quote=}}</ref>。脖子上佩戴的[[项链|项链]]是{{link-en|却克里王室勋章|Order of the Royal House of Chakri}}，左手佩戴有一条金[[手链|手链]]，无名指上有一枚金[[戒指|戒指]]；右手戴有一条[[祈福线|白色的细线]]，似乎是要接触旁边的[[芭蕉|芭蕉]]叶，站在湖畔的一棵树下。背景远处为一座以[[银色|银白色]]为主的泰式建筑，下部为[[土黄色|土黄色]]，湖中映着倒影<ref>{{cite web |url=http://test.pengan.gov.cn/Item/87616.aspx |title=庆祝中泰建交40周年 蓬安画家捐赠5幅作品 |last= |first= |date= |website= |publisher= |access-date= |quote=}}{{Dead link|date=2018年11月 |fix-attempted=yes }}</ref>。这座建筑的原型是泰国{{link-en|北榄古城|Mueang Boran}}的善佩寺巴塞王宫（Sanphet Prasat Palace）。

== 原作 ==
《诗琳通公主肖像》照片由泰国皇家摄影学会<!-- Royal Photographic Society of Thailand (RPST) -->会长尼迪功·蓋威遷{{NoteTag|在诗琳通公主的摄影图集《光是色，色是光》（{{lang-th|แสงคือสี สีคือแสง}}，{{ISBN|9789747628616}}）中翻译为“尼提功·格莱威仙”。}}（{{lang-th|นิติกร กรัยวิเชียร}}，{{lang-en|Nitikorn Kraivixien}}）拍摄<ref>{{cite web |url=http://www.meeboard.com/view.asp?user=absoem&groupid=31&rid=1579&qid=494 |title=ส่งรูปพระราชินี ครั้งที่2 |last= |first= |date= |website= |publisher= |access-date= |quote=}}</ref>，收錄在其1989年至2015年攝影的御照專輯《心爱的公主》（{{lang-th|ปิยราชกุมารี}}）中<ref>{{cite web |url=http://www.chaoprayanews.com/2015/12/27/%E0%B8%AB%E0%B8%99%E0%B8%B1%E0%B8%87%E0%B8%AA%E0%B8%B7%E0%B8%AD-%E0%B8%9B%E0%B8%B4%E0%B8%A2%E0%B8%A3%E0%B8%B2%E0%B8%8A%E0%B8%81%E0%B8%B8%E0%B8%A1%E0%B8%B2%E0%B8%A3%E0%B8%B5-%E0%B9%81/ |title=หนังสือ “ปิยราชกุมารี” และ ภาพถ่ายฝีพระหัตถ์ “อยู่มานาน กาลเวลาพาสุข” |last= |first= |date=2015-12-27 |website=昭披耶新聞 |publisher= |access-date= |quote=}}</ref>。在原照片中，背景是{{link-th|节基殿|หมู่พระที่นั่งจักรีมหาปราสาท}}内的[[拉玛五世|拉玛五世]]全身像（[[:File:King_Chulalongkorn.jpg|图片]]），右手则是放在摆放王家供品的圆桌上，旁边有一张宫廷椅。2015年，为庆祝[[2015年诗琳通公主60寿辰|六十寿辰]]，[[泰国银行|泰国银行]]发行限量100铢纸币，背面是其图像；泰国邮局也根据此照片发行了诗琳通公主全身像纪念[[邮票|邮票]]<ref>{{cite web |url = http://news.china.com.cn/2015-04/07/content_35256596.htm |title=泰国诗琳通公主 60寿宴亲点中国川菜 |last= |first= |date=2015-04-07 |website=中国网 |publisher= |access-date=2018-07-25 |quote= |archive-url = https://web.archive.org/web/20161101044507/http://news.china.com.cn/2015-04/07/content_35256596.htm |archive-date=2016-11-01 |dead-url=no }}</ref>。纪念诗琳通公主的歌曲《[[心爱的公主|心爱的公主]]》的MV也使用此照片<ref>{{YouTube|LqOHESkRtEE|link=no|《心爱的公主》MV|language=th}}</ref>。
