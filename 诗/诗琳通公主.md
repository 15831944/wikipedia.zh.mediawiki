'''玛哈·扎克里·诗琳通公主殿下'''（{{lang-th|สมเด็จพระเทพรัตนราชสุดา เจ้าฟ้ามหาจักรีสิรินธร รัฐสีมาคุณากรปิยชาติ สยามบรมราชกุมารี}}，{{bd|1955年|4月2日|||Catnav=S}}），[[泰国|泰国]][[扎克里王朝|扎克里王朝]]公主<ref>{{Citation|first=Duncan|last=McCargo|title=Thailand|work=Regional Oulook: Southeast Asia 2010-2011|publisher=Institute of Southeast Asian Studies|year=2010|page=55|accessdate=20 September 2014}}</ref>。她是[[普密蓬·阿杜德|普密蓬国王]]的次女。她的称呼为“[[公主|公主]][[殿下|殿下]]”。1977年12月5日，封号變更為“玛哈·扎克里公主”，又被称为'''帕贴'''（{{lang-th|พระเทพ}}，{{lang-en|Phra Thep}}，意为“天佛”、“天使公主”）<ref>{{cite web|url=http://news.fjnet.com/jjdt/jjdtnr/200505/t20050512_10110.htm|title=国际佛光会向泰国公主的慈善基金会捐赠南亚赈灾款-大陆教界内容-佛教在线|website=news.fjnet.com}}</ref>。

在家中排名第三，僅次於姐姐[[烏汶叻公主|烏汶叻公主]]和哥哥[[瑪哈·瓦集拉隆功|瑪哈·瓦集拉隆功]]（現任泰王[[拉瑪十世|拉瑪十世]]）。另有一妹[[朱拉蓬公主|朱拉蓬公主]]殿下。

== 生平 ==
诗琳通公主在1955年4月2日生於泰國曼谷的泰國皇宮中，1978年獲[[朱拉隆功大学|朱拉隆功大学]][[梵語|梵語]]和[[巴利語|巴利語]]碩士，1980年獲[[金石學|金石學]]（[[梵文|梵文]]和[[柬埔寨文|柬埔寨文]]）碩士，1987年[[詩納卡寧威洛大學|詩納卡寧威洛大學]]發展教育學[[哲學博士|哲學博士]]<ref name="EEF">Pattayamail, [http://www.pattayamail.com/localnews/a-most-happy-birthday-to-her-royal-highness-princess-maha-chakri-sirindhorn-36213 more education information]</ref>。现任{{link-en|泰国红十字会|Thai Red Cross Society}}副会长、[[帕尊宗浩陆军军官学校|帕尊宗浩陆军军官学校]]历史专业教授。

诗琳通公主聪颖好学，年幼时便在国王及王后指导下开始学习[[中国历史|中国历史]]和[[中国文学|中国文学]]，能说一口流利的[[汉语|汉语]]，鉴赏[[漢詩|漢詩]]、[[宋词|詞]]，擅长中国[[书画|书画]]，能用[[二胡|二胡]]等[[中国民族乐器|中国民族乐器]]演奏。

詩琳通公主深受泰國人愛戴，在國內除其已故父親，前任國王[[普密蓬·阿杜德|普密蓬·阿杜德]]（拉瑪九世）及詩麗吉王太后外無出其右。和父王一樣，詩琳通公主非常關心貧苦大眾，經常探訪貧民，如[[2004年印度洋大地震|2004年印度洋大地震]]中，探望受灾災民。泰国人民对她的爱戴，似乎遠超其兄长，現任國王[[摩訶·瓦集拉隆功|摩訶·瓦集拉隆功]]。

詩琳通公主曾就王位繼承權表示無意和兄長[[摩訶·瓦集拉隆功|摩訶·瓦集拉隆功]]爭位。

詩琳通公主至今仍然未婚。

== 作品 ==

著有《[[顽皮透顶的盖珥|顽皮透顶的盖珥]]》、《[[踏访龙的国土|踏访龙的国土]]》、《[[平沙万里行|平沙万里行]]》、《[[採桃花|採桃花]]》、《[[長城遊|長城遊]]》、《[[青木瓜沙拉之歌|青木瓜沙拉之歌]]》等小说、散文和诗歌作品。1994年1月，中国的[[生活·读书·新知三联书店|生活·读书·新知三联书店]]出版了《[[诗琳通公主诗文画集|诗琳通公主诗文画集]]》。

=== 关于笔名 ===
名字“诗琳通”源于[[巴利语|巴利语]]{{lang|pi|Sirindhara}}，意思是“光荣的”<ref>{{cite web|url=http://dictionary.sutta.org/browse/s/siri|title=siri - Definition and Meaning - Pali Dictionary - Pāli to English, Chinese, Japanese, Vietnamese, Burmese Dictionary|website=dictionary.sutta.org}}</ref>。除“诗琳通”（{{lang|th|สิรินธร}}）外，她还曾使用过4个笔名：
# 功欣功古（{{lang|th|ก้อนหินก้อนกรวด}}）：原意为[[鹅卵石|鹅卵石]]，诗琳通公主的化名之一。
# 菀盖珥（{{lang|th|แว่นแก้ว}}）<ref>{{cite web|url=http://118.122.126.41:8088/opac/book/76921;jsessionid=738CF768D1238E5E1FE6E38DAB292058|title=顽皮透顶的盖珥, (泰)菀盖珥著|publisher=}}</ref>：诗琳通公主为自己定下的笔名，原意为玻璃镜（在泰语里也是{{link-en|伞形天胡荽|Hydrocotyle umbellata}}的俗名）。
# 小宝贝（{{lang|th|หนูน้อย}}）：公主的别名之一。
# 班丹（{{lang|th|บันดาล}}）：原意为“创造”。<ref>{{cite web|url=http://bclc-tu.com/biograhy_b3_%20a1_homePage.html|title=bclc-tu - พระราชกรณียกิจ|publisher=}}</ref>
