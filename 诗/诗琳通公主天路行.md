《'''诗琳通公主天路行'''》（{{lang-th|การเสด็จพระราชดำเนิน ของสมเด็จพระเทพรัตนราชสุดาฯ สยามบรมราชกุมารี ตามเส้นทางสู่ที่ราบสูง}}）是由[[中国外文局|中国外文局]]主持编辑，[[中国画报出版社|中国画报出版社]]、[[人民画报社|人民画报社]]、[[国图总公司|国图总公司]]等合作推出的中泰双语书籍。讲述的是2007年4月[[泰国|泰国]][[诗琳通公主|诗琳通公主]]第26次访华，乘坐[[青藏铁路|青藏铁路]]访问[[青海省|青海省]]和[[西藏自治区|西藏自治区]]的经历<ref>{{cite web |url=http://www.china.com.cn/fangtan/zhuanti/60flkf/2008-10/05/content_16566885.htm |title=诗琳通公主天路行 |last= |first= |date=2008-10-05 |website=中国网 |publisher= |access-date= |quote=}}</ref>，内含100余张配图<ref>{{cite web |url=http://epaper.gmw.cn/gmrb/html/2015-04/06/nw.D110000gmrb_20150406_7-03.htm |title=意犹未尽逛书店——泰国诗琳通公主访问中国外文局侧记 |last= |first= |date=2015-04-06 |website= 光明网|publisher=刘彬 |access-date=2018-10-12 |quote=}}</ref>。

2007年11月14日，《诗琳通公主天路行》于泰国[[曼谷|曼谷]]首发<ref>{{cite web |url=http://paper.people.com.cn/rmrb/html/2007-11/15/content_31773116.htm |title=《诗琳通公主天路行》画册在泰首发 |last= |first= |date=2007-11-15 |website=人民网 |publisher=人民日报 |access-date=2018-10-12 |quote=}}</ref><ref>{{cite web |url=https://www.mfa.gov.cn/chn//pds/gjhdq/gj/yz/1206_30/1206x2/t434672.htm |title=《诗琳通公主天路行》画册首发式隆重举行 |last= |first= |date=2007-11-15 |website=中华人民共和国外交部 |publisher= |access-date=2018-10-12 |quote=}}</ref>。

== 内容 ==
2007年4月4日，[[诗琳通公主|诗琳通公主]]一行参观[[青海师范大学|青海师范大学]]、[[西宁市|西宁市]]广场中心，4月5日参观[[青海省|青海省]][[同仁县|同仁县]][[吾屯下寺|吾屯下寺]]；4月6日参观[[格尔木|格尔木]]昆玉工艺制品有限公司、青海盐湖集团公司、格尔木炼油厂展厅；4月7日，公主乘坐[[青藏铁路|青藏铁路]]K917次列车，从格尔木前往拉萨；4月8日参观[[大昭寺|大昭寺]]、[[拉鲁湿地|拉鲁湿地]]、[[布达拉宫|布达拉宫]]、[[拉萨中学|拉萨中学]]和拉萨啤酒厂。<ref>{{cite web |url=http://www.china.com.cn/news/txt/2007-11/14/content_9223209.htm |title=《诗琳通公主天路行》画册精选 |last= |first= |date= |website=中国网 |publisher= |access-date= |quote=}}</ref>

=== 背景 ===
此次为诗琳通公主第26次访华。时间为2007年4月2日至4月13日，访问[[北京市|北京市]]，[[青海省|青海省]][[西宁市|西宁市]]、[[格尔木市|格尔木市]]，[[西藏自治区|西藏自治区]][[拉萨市|拉萨市]]，[[湖北省|湖北省]][[武汉市|武汉市]]，[[浙江省|浙江省]][[宁波市|宁波市]]、[[舟山市|舟山市]]（包括[[普陀山|普陀山]]朝圣参访），以及[[上海市|上海市]]，但本书只收录了西宁市到拉萨市的行程。
