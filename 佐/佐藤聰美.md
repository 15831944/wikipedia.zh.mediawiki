'''佐藤聰美'''（{{lang-ja|さとう さとみ}}，{{bd|1986年|5月8日|catIdx=Satou Satomi}}）是[[日本|日本]][[女性|女性]][[聲優|聲優]]、[[歌手|歌手]]，隸屬[[青二Production|青二Production]]。[[宮城縣|宮城縣]][[仙台市|仙台市]]出身<ref>網絡廣播「{{lang|ja|えどっ娘 天下大変!}}」第6回。</ref>。身高153cm<ref name="profile"> {{Cite web |url= http://www.aoni.co.jp/junior/sa/pdf/sato-satomi.pdf |title= 佐藤 聡美 |format= PDF |publisher= [[青二Production|青二Production]] |date= 2009-08-08 |deadurl= yes |archiveurl= https://web.archive.org/web/20081230155339/http://www.aoni.co.jp/junior/sa/pdf/sato-satomi.pdf |archivedate= 2008-12-30 }} </ref>。[[血型|血型]]O型。常盤木學園高等學校、東京廣播學院畢業。有一個妹妹。
[[File:Satomi_Sato's_signature,_K-On!_20121214.jpg|thumb]]

== 經歷 ==
在東京廣播學院就讀時，成為廣播「{{lang|ja|智一・美樹のラジオビッグバン}}」的第七代助手（第5期BGP）。同時結成聲優組合「Kisty」。

以少女、女性為中心出演各種角色，不過其中以老實文靜的少女居多。因而當被選定出演《[[K-ON!|K-ON!]]》的田井中律時，包括本人在內周圍所有人都很驚訝<ref>[http://osatomi.jugem.jp/?eid=368 本人的BLOG] {{webarchive|url=https://web.archive.org/web/20090217102329/http://osatomi.jugem.jp/?eid=368 |date=2009-02-17 }}。</ref><ref name="声優プリンセス">{{Cite journal|title=PICK-UP FRESH VOICE ACTRESS|author=|journal=声優プリンセス|publisher=近代映画社|accessdate=2009-07-20|year=2009|pages=pp.73-75}} ISBN 978-4-7648-7153-3 </ref>。

高中時曾參加[[NHK|NHK]]杯全国高校放送大賽并獲得宮城縣賽區第一名，在全國大賽上也進入了半決賽，在朗讀方面有相當的實力<ref>《[[聲優Animedia|聲優Animedia]]》2009年5月号。</ref><ref>[http://www.asahi-net.or.jp/~me5t-dt/housou/nhk51th_a4.htm 宮城縣高等學校文化聯盟放送專門部第51回（2004年）NHK杯放送大賽朗讀部門決勝結果]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

喜歡的動物是[[馬|馬]]、[[柴犬|柴犬]]及[[羊駝|羊駝]]，喜歡的顏色是水色、白色、淡粉色。

和[[高本惠|高本惠]]很要好經常一起出去玩，佐藤的BLOG上也經常提到高本。和在[[K-ON!|K-ON!]]等作品中共演的[[日笠陽子|日笠陽子]]和[[竹達彩奈|竹達彩奈]]也很要好。另外和隸屬青二Production（特別是Junior）的女性聲優也有親交。

在K-ON! web radio「らじおん!」的企劃中，和自己演出的角色一樣實際進行樂團活動。佐藤本身完全沒有[[爵士鼓|爵士鼓]]的經驗，一切從零開始。

在2010年3月6日發表的第4回[[聲優獎|聲優Awards]]上，以《[[K-ON!|K-ON!]]》劇中結成的團體“放學後Tea Time”的一員，與[[豐崎愛生|豐崎愛生]]、[[日笠陽子|日笠陽子]]、[[壽美菜子|壽美菜子]]、[[竹達彩奈|竹達彩奈]]一起獲得歌唱賞。2011年3月5日發表的第5回聲優獎上又獲得新人女優賞。

2017年7月6日凌晨，佐藤聪美在个人博客上发布了结婚的消息，对象为同为声优的[[寺島拓篤|寺島拓篤]]<ref>{{Cite web|url=http://ameblo.jp/sato-satomi58/|title=結婚のご報告。|accessdate=2017-07-06|date=2017-07-06}}</ref>。
