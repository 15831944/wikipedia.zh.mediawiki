'''佐藤健'''（{{lang-ja|{{jp|j=佐藤 健|hg=さとう たける|rm=Satō Takeru|help=no}}}}，{{bd|1989年|3月21日|catIdx=Satō}}），[[日本|日本]][[演員|男演員]]，隸屬[[Amuse|Amuse]]經紀公司。2007年在[[特摄剧|特摄剧]]《[[假面騎士电王|假面騎士电王]]》中主演，以一人分饰八角的演技獲得注目。2012年起，主演話題漫畫改編的真人版電影《[[浪客劍心_(電影)|神劍闖江湖]]》大获成功，在日本本土电影票房突破125億，並於2014年8月1日推出其第二篇續作「[[神劍闖江湖_京都大火篇|京都大火篇]]」及9月13日的「[[浪客剑心：传说的最后时刻篇|傳說的最後時刻篇]]」。2015年10月3日《[[爆漫王|爆漫王]]》於日本公映，榮獲日本當週票房冠軍。東京Drama Award 2015（東京ドラマアウォード2015）以《[[天皇的御廚|天皇的御廚]]》榮獲最佳男主角。

== 簡歷 ==
佐藤健在高中二年級的[[黃金週|黃金週]]於[[原宿|原宿]]的[[竹下通|竹下通]]被現在事務所的星探挖角<ref>『ムーブ!』2008年10月6日播放。</ref>。
2006年，初次在《[[變身公主|變身公主]]》亮相；初次出演，就被選為主角「河野亨」。翌年，被選中當《[[假面騎士電王|假面騎士電王]]》的主角「野上良太郎」。
2009年，以《[[我的帥管家|我的帥管家]]》獲得[[第60回日劇學院賞|第60回日劇學院賞]]男配角獎。2010年，出演NHK大河劇《[[龍馬傳|龍馬傳]]》，並於同年2010年10月，初次主演連續劇《[[Q10|Q10]]》。
2011年，出演電影《[[BECK|BECK]]》主角田中幸雄，並獲得{{link-ja|黃金飛翔獎|エランドール賞}}及[[日本電影影評人大獎|日本電影影評人大獎]]的新人賞。
2012年4月，出演舞台劇《羅密歐與茱麗葉》中的羅密歐一角，該劇也是佐藤初次挑戰舞台劇。<
2012年8月，飾演話題漫畫改編的真人版電影《[[浪客劍心_(電影)|神劍闖江湖]]》主角緋村劍心，連續兩週日本票房冠軍，吸引237萬人次觀眾，更創下30.1億日幣超高票房。並於2014年公開兩篇續作，3集作品累計票房紀錄高達125億日元。
[http://www.eiren.org/toukei/index.html 2012年度（平成24年）興収１０億円以上番組　（平成25年1月発表）]日本映画製作者連盟 2013年1月
2015年，主演[[TBS電視台|TBS電視台]]開台電視劇《[[天皇的御廚|天皇的御廚]]》，平均收視率達14.9%。

2015東京Drama Award（東京ドラマアウォード2015） 以《[[天皇的御廚|天皇的御廚]]》榮獲最佳男主角。

== 人物 == 
* 不服輸，尤其是在遊戲上。
* 拍攝《[[浪客劍心_(電影)|神劍闖江湖]]》時，所有武打場面全都自己上陣。因為覺得「既然那些特技演員們都做得到，自己和他們也沒什麼不一樣，沒有道理做不到」而努力！
* 是會讓特技演員汗顏感動的演员。
* 國中時非常努力唸書，曾經得過全學年第一名，上了高中就覺得不想再這麼認真了，所以幾乎都在睡覺！
* 高中時，曾經有一年的時間故意裝得很酷，不和同學們打交道。而在某次因前堂課上課睡著，要換教室上課時沒有半個同學來叫醒他，當他醒來時教室只剩自己一人後，覺得還是不要再裝酷好了，而開始和同學們打成一片。
* 家中只有媽媽、他和一個小4歲的妹妹及2隻貓。興趣是和貓玩耍<ref name="koushiki">[http://www.amuse.co.jp/artist/Takeru_Satou/profile.html 佐藤健官方網站]</ref>。2隻貓名字分別是「こちろー」及「ぷちろー」。本人曾在官方BLOG介紹他們，並附上圖片<ref>
[https://archive.is/20130430230142/ameblo.jp/takeru-s/entry-10156822892.html 佐藤健官方BLOG於2008年10月27日刊登。][https://archive.is/20130501172439/ameblo.jp/takeru-s/entry-10157129255.html 佐藤健官方BLOG於2008年10月28日刊登。]</ref>。
* 擅長玩水平思考遊戲、奧賽羅、[[魔術方塊|魔術方塊]]和[[Breaking|Breaking]]<ref name="koushiki"/>。在演出的《[[假面騎士電王|假面騎士電王]]》中，亦有他親自跳舞的場面。
* 2007年11月1日，因為左胸被診斷有[[氣胸|氣胸]]，休息了約10天<ref>[http://www.nikkansports.com/entertainment/p-et-tp0-20071104-278607.html 日刊體育新聞 「最弱ライダー」佐藤健、肺気胸で療養2007年11月4日刊登。]</ref>。現在已經康復。
* 與[[ONE_OK_ROCK|ONE OK ROCK]]主唱[[森內貴寬|TAKA]]為朋友。和[[城田優|城田優]]及[[三浦翔平|三浦翔平]]私底下也經常在一起活動。
* 讀初中時開始是[[深津繪里|深津繪里]]支持者。 
* 喜歡的藝人有[[Mr.Children|Mr.Children]]<ref>『東京フレンドパークII』2008年10月20日播放。</ref>、[[ONE_OK_ROCK|ONE OK ROCK]]與[[椎名林檎|椎名林檎]]<ref>佐藤健舊官方BIOG「WITH」中於2010年1月15日刊登。</ref>。
*圈内的好朋友有同事务所的[[三浦春馬|三浦春馬]]，[[三浦翔平|三浦翔平]]和[[神木隆之介|神木隆之介]]。
* 拍攝電影《BECK》期間，在做拍攝開始前的廣播體操時，被導演發現手碰不到肩膀。<ref>『笑っていいとも』2012年8月23日播放。</ref>
* 2013年底電影《她愛上了我的謊》上『新堂本兄弟』<ref>『新堂本兄弟』2013年12月8日播放</ref>宣傳，提到手碰不到肩膀，於是堂本剛即興表演了〝佐藤健之舞〞，笑翻大眾。
* 2016年5月電影《如果這世界貓消失了》5月29日的舞台問候時提到去給整骨師整骨之後居然手能碰到肩膀了，並且立刻表演了，得到粉絲的熱烈掌聲（於2016年6月13日『たけてれ』<ref>『たけてれ』為網路直播節目，不定期直播（通常一個月一次）與粉絲互動，節目中會聊近況。</ref>中播放畫面）。
