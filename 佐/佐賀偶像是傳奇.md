== 製作 ==
企划酝酿了4年的时间。制作喪屍偶像作品的想法源于Cygames的竹中信广觉得《[[巴哈姆特之怒|巴哈姆特之怒 GENESIS]]》中的莉塔很可爱<ref name=producer>{{cite web |script-title=ja:衝撃の展開を見せたアニメ『ゾンビランドサガ』の仕掛け人、竹中信広プロデューサーにいろいろ聞いてみた |url=https://www.famitsu.com/news/201810/08165516.html |website=[[ファミ通|ファミ通]] |date=2018-10-08 |language=ja |title=存档副本 |access-date=2018-10-13 |archive-url=https://web.archive.org/web/20181013093335/https://www.famitsu.com/news/201810/08165516.html |archive-date=2018-10-13 |dead-url=no }}<br/>（有节选）{{cite web |title=采访制作人！《佐贺偶像》内幕曝光 |url=https://new.qq.com/cmsn/20181015/20181015006300.html |website=腾讯网 |language=zh-CN |access-date=2018-10-16 |archive-url=https://web.archive.org/web/20181016081410/https://new.qq.com/cmsn/20181015/20181015006300.html |archive-date=2018-10-16 |dead-url=no }}</ref>，在与MAPPA的制作人大塚学交谈后，大塚学认为很有趣，于是着手制作企划书。然而在Cygames的稟議中未获得批准，而[[佐贺县|佐贺县]][[伊萬里市|伊萬里市]]<ref name=":0">{{Cite news|url=https://www.saga-s.co.jp/articles/-/349178|title=ゾンビランドサガ「聖地巡礼なび！」きょう14日発行　購入申し込みは「佐賀新聞LiVE」から|author=佐賀新聞社|date=2019-03-14|work=佐賀新聞|accessdate=https://www.saga-s.co.jp/articles/-/349178|language=Ja}}</ref>出身的Cygames社長渡邊耕一说：“希望做一部刻画佐贺县的动画”，因此利用这个机会将二者合在一起。为此社長还一同前往佐贺进行取景并向团队介绍。片名也从暂定的《偶像喪屍》变更为现在的《喪屍樂園佐賀》（直译），因为本作有别于一般的偶像作品，而且以当时的风潮来说，片名使用「偶像」太普通了。<ref name=SAGA02>{{cite web |script-title=ja:『ゾンビランドサガ』スタッフ陣が明かすプロジェクトの全貌【連載】 |url=https://www.animatetimes.com/news/details.php?id=1539407067 |website=animate Times |date=2018-10-13 |language=ja |title=存档副本 |access-date=2018-10-13 |archive-url=https://web.archive.org/web/20181013144003/https://www.animatetimes.com/news/details.php?id=1539407067 |archive-date=2018-10-13 |dead-url=no }}</ref> 竹中信广表示本作是Cygames推出的第1个无关游戏的原创动画企划，至于「为什么而做」，其实只是在做想做的东西<ref name=producer/>。

在竹中信广和大塚学制作企划的过程中，动画制作人员中导演一职是最先决定的。为此在企划尚未固定时便同导演[[境宗久|境宗久]]进行了交谈，负责角色设计的深川可純是导演介绍过来的；系列剧本构成由擅长搞笑和[[喜劇|喜劇]]的村越繁担任，这也是村越繁首次担任系列剧本构成；配乐交给了喜欢[[死亡金屬音樂|死亡金屬音樂]]的[[高梨康治|高梨康治]]。<ref name=SAGA02/>

== 故事大綱 ==
渴望成為偶像的女高中生源櫻在2008年某日被卡車撞死，她在十年後清醒並發現自己成為殭屍，收留源櫻的神秘經紀人巽幸太郎要求源櫻與六位來自不同時期的女性殭屍組成偶像團體，以振興[[佐賀縣|佐賀縣]]。在初次演出後，對生前記憶毫無所悉的源櫻想起了片段記憶。

之後，她們將團體定名為「法蘭秀秀」，二階堂咲自薦擔任隊長。她們在[[佐賀城|佐賀城]]進行了{{link-en|饒舌對決|Battle rap}}、在[[唐津車站|唐津車站]]快閃演出、代言[[嬉野市|嬉野市]]的製藥公司產品和佐賀的雞肉料理專賣店Drive-In Tori、參與{{link-ja|鹿島泥林匹克|鹿島ガタリンピック}}，知名度逐漸提高，同時引起了記者大古場新太的注意。

在一次攝影見面會中，成員紺野純子與水野愛就和粉絲互動一事的觀點不同而產生衝突，並拒絕參與練習。櫻在與純子對談時得知她死於飛機失事，而咲則得知水野愛遭雷擊而死。在巽告訴純子並不需要迎合現代的偶像標準後，她回歸團隊並一同參與Rock Saga。在演出進行到一半時，她們再度遭閃電擊中，不但沒有受到傷害，反而還在全身發光的情況下演出電音舞曲，進而受到大篇幅報導。在法蘭秀秀進行多場演出後，成員星川莉莉的父親認出了自己的孩子，法蘭秀秀特別設計了演出，讓莉莉向父親道別。咲阻止了以前率領的暴走族與另一群暴走族的紛爭，並透過舉辦演唱會來讓兩個幫派和解。

法蘭秀秀即將在唐津市鄉土會館Arpino舉辦觀眾數達500人的大型演出，櫻認為這是找回記憶的線索，因此十分焦躁。她們被送到山上進行修練，但因其它成員都在考慮野外求生，讓重視排練的櫻和其它成員產生不合，她透過觀察其它成員排練意識到自己的自私並重新融入團隊。演唱會開始前一週，櫻再次被貨車撞到，恢復了生前的記憶，也失去殭屍期間的記憶。

在得知自己的經歷後，櫻變得十分消極並認為自己的努力不會產生好結果，不過其它成員仍沒有放棄，她們協力說服櫻登台演出，強調即使失敗也在所不惜。演出當日，大雪導致會場崩塌，但法蘭秀秀仍舊在全毀的舞台上熱歌勁舞，櫻也在演出期間恢復了記憶。之後，大古場對其中幾個成員的身分產生懷疑，作為未解決的伏筆。

== 聲優演唱會 ==
2018年12月15日幕張展覽館的CygamesFes2018中為法蘭秀秀配音的聲優進行約一小時的演出，包括《徒花ネクロマンシー》和《光へ》的演唱。

2019年3月17日在品川Intercity Hall舉辦了獨立演唱會， 佐賀偶像是傳奇LIVE~Fran Chou Chou大家一起喊出来~（{{lang-ja|ゾンビランドサガLIVE~フランシュシュみんなでおらぼう！~}}），共日間及夜間兩場。為法蘭秀秀、2008及2018年Iron Frill配音的聲優均有參演並獻唱2018年動畫的大部分曲目，節目約兩小時。

== 衍生漫畫 ==

== 迴響 ==
2018年9月22日举行了第1话、第2话的先行上映会，来场者签订严禁剧透誓约书，调查投票显示：认为是预想外的展开的有88％，满意度达96％<ref>{{cite web |script-title=ja:ゾンビランドサガは空前絶後一心不乱絶叫伝説？ネタバレ厳禁上映、満足度96％ |url=https://natalie.mu/comic/news/300856 |website=[[Natalie_(網站)|comic natalie]] |date=2018-09-23 |language=ja |title=存档副本 |access-date=2018-10-17 |archive-url=https://web.archive.org/web/20181017203300/https://natalie.mu/comic/news/300856 |archive-date=2018-10-17 |dead-url=no }}</ref>。
播放前，故事詳情不明、公開的宣傳動畫看上去像是恐怖動畫，但實際上是變成了喪屍的少女們目標成爲當地偶像振興佐賀這樣的故事；第一話播出後，引發巨大反響，網絡上紛紛留言稱「和預想的不一樣」「原來是偶像……」，并且登上[[Twitter|Twitter]]趨勢榜第一。<ref>{{cite web |script-title=ja:ゾンビランドサガ：ゾンビ×佐賀×アイドル×コメディー　意外すぎる展開に「まさか！？」の声が続々 |url=https://mantan-web.jp/article/20181005dog00m200000000c.html |website=MANTANWEB |date=2018-10-05 |language=ja |title=存档副本 |access-date=2018-10-07 |archive-url=https://web.archive.org/web/20181005121147/https://mantan-web.jp/article/20181005dog00m200000000c.html |archive-date=2018-10-05 |dead-url=no }}</ref><ref>{{cite web |script-title=ja:『ゾンビランドサガ』はゾンビ×アイドル×ギャグアニメだった!? SNSでも大反響! 第1話を場面カットとともに紹介【ネタバレ注意】 |url=https://cho-animedia.jp/anime/61255/2/ |website=超! アニメディア |date=2018-10-05 |language=ja |title=存档副本 |access-date=2018-10-07 |archive-url=https://web.archive.org/web/20181007073757/https://cho-animedia.jp/anime/61255/2/ |archive-date=2018-10-07 |dead-url=no }}</ref>
动画播出后被各国观众喜爱和讨论，被评价为季度新番最大黑马。<ref>{{cite web |title=“佐贺万圣趴”来了！《 佐贺偶像是传奇》万圣主题观影会在京举办 |url=http://www.donews.com/news/detail/3/3025274.html |website=[[DoNews|DoNews]] |date=2018-11-01 |access-date=2018-11-09 |archive-url=https://web.archive.org/web/20181109112155/http://www.donews.com/news/detail/3/3025274.html |archive-date=2018-11-09 |dead-url=no }}</ref>

其意外的巨大反響亦吸引佐賀縣本地關注，當地報章[[佐賀新聞|佐賀新聞]]從動畫播出後亦開始報導作品動向，並在2019年3月14日刊行登載製作人員訪問及取景地資訊的聖地巡禮特刊<ref name=":0" />。另外在動畫播放中的2018年12月14日，取景地之一的[[唐津市_(日本)|唐津市]]的市議會的一名議員，也在第五次正式會議中向市政府質詢相關内容<ref>{{Cite web|title=唐津市議会－インターネット中継－録画中継（再生）|url=http://www.karatsu-city.stream.jfit.co.jp/?tpl=play_vod&inquiry_id=1180|work=www.karatsu-city.stream.jfit.co.jp|accessdate=2019-04-14}}</ref>。

=== 評價 ===
觀看完第一話後，[[動畫新聞網|動畫新聞網]]的James Beckett認爲女主角突然被車撞死並伴隨著[[死亡金屬音樂|死亡金屬音樂]]這一開場以完美的方式為本作的怪誕恐怖奠定了基礎；Theron Martin稱将偶像与[[重金属音乐|重金属音乐]]结合起来实际上并不夸张（比如[[BABYMETAL|BABYMETAL]]），但在此之前并没有这样的動畫；Paul Jensen表示喪屍恐怖與偶像喜劇的組合非常新鮮，第一話非常令人愉悅。<ref>{{cite web|url=https://www.animenewsnetwork.com/preview-guide/2018/fall/zombie-land-saga/.137710|title=Zombie Land Saga - The Fall 2018 Anime Preview Guide|website=Anime News Network|date=2018-10-04|language=en|access-date=2018-10-08|archive-url=https://web.archive.org/web/20181008061508/https://www.animenewsnetwork.com/preview-guide/2018/fall/zombie-land-saga/.137710|archive-date=2018-10-08|dead-url=no}}</ref>

===銷量===
根據[[Oricon公信榜|Oricon公信榜]]數字，《佐賀》一片的第一卷在發售首周錄得12,640套的銷售量，為2018年秋季動畫中的第一名<ref>{{cite web |title=週間　アニメBlu-rayランキング 2018年12月17日～2018年12月23日 |url=https://www.oricon.co.jp/rank/ba/w/2018-12-31/ |accessdate=2018-12-30}}</ref>。截至2019年3月25日公布的每周榜單為止，第一卷的銷售量為23,798套。
