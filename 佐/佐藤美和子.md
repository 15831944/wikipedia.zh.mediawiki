'''佐藤美和子'''（{{lang|ja|佐藤 美和子}}/{{lang|ja|さとう みわこ}}，Satō Miwako），是[[日本|日本]][[漫画|漫画]]和[[动画|动画]]作品《[[名侦探柯南|名侦探柯南]]》中的[[名偵探柯南角色列表|虚构角色]]。

== 人物 ==
[[警視廳|警視廳]][[警視廳#.E5.88.91.E4.BA.8B.E9.83.A8_Criminal_Investigation_Bureau|搜查一課強行犯捜査三系]]的[[警部補|警部補]]，是[[目暮十三|目暮十三]]、[[白鳥任三郎|白鳥任三郎]]的部下，[[高木涉_(名偵探柯南)|高木涉]]的上司，[[宮本由美|宮本由美]]的好友；亦為警視廳的警花，受許多男性警官的愛慕，甚至成立「'''佐藤美和子防線'''」監視佐藤與高木兩人的行動<ref group="參" name="TV 358">漫畫第40卷，TV 358-359《本廳刑事戀愛物語5》</ref>。

== 形象 ==
行事果敢堅決、細心敏銳、精練能幹、執著冷靜、堅強勇敢；嫉惡如仇，擁有強烈的正義感和犧牲精神。同時也很重感情，天真燦爛，平易近人，不失孩子氣。

愛車為紅色[[馬自達RX-7|馬自達RX-7]]。其駕車技術高超，一般人在她風馳電掣的時候都會暈車，是飆車族傳說中的「冬名山魔女」，但其本人並不自覺<ref group="參" name="TV 546"/>。

與導師'''小林澄子'''（白鳥的初戀情人）有幾分神似，所以被白鳥以為其初戀情人一直追求，並曾經假扮小林老師制伏犯人，但最後自己都沒有發覺<ref group="參">漫畫第68卷，TV 585《超越時空的櫻花之戀》</ref>。

== 情感 ==
佐藤曾喜歡三年前殉職的同僚'''松田陣平'''<ref group="參">漫畫第36卷，TV 301-302《惡意與聖者的遊行》</ref>。松田的死令佐藤覺得自己受到詛咒而不敢跟自己重視的人發展得太親近，還因此跟剛剛開始接近的[[高木涉_(名偵探柯南)|高木]]提出分手。不過隨著炸彈案的解決<ref group="參">漫畫第37卷，TV 304《震撼警視廳 1200萬人質》</ref>，佐藤也重拾信心，現在跟高木發展成戀人關係。放假的時候也經常瞞過同僚的監視跟他約會（其實整個警視廳都知道約會的事，並經常組成監視隊伍跟蹤他們），甚至在辦案的時候（在逮捕犯人之前）也會突然進入兩人世界<ref group="參" name="TV 358"/>。

曾經在母親的慫恿下跟同僚白鳥警部相親。但那只是佐藤一邊看電視一邊從大量的相片中用腳趾夾上來，偶然選中白鳥而已。雖有出席相親，卻完全沒有考慮結婚的意思，卻也因此明白自己對高木的心意<ref group="參">漫畫第32-33卷，TV 253-254《本廳刑事戀愛物語4》</ref>；後來高木為了辦案需要而和[[宮本由美|宮本由美]]警官假結婚，其中由美態度過度親熱，讓佐藤吃了不少醋<ref group="參">漫畫第59卷，TV 449《本廳刑事戀愛物語 虛偽的婚禮》</ref>。

由美曾捉弄佐藤在左手[[無名指|無名指]]上戴[[戒指|戒指]]可以驅蟲，引起高木和警視廳男警官一陣騷動（因為佐藤不知道在左手無名指上戴戒指有「訂婚」的意義）<ref group="參">漫畫第59卷，TV 487《本廳刑事戀愛物語8 左手的無名指》</ref>。後來在高木被餓了幾乎三天的情況下被柯南等人推理出被綁地點，在眾警官的監控中相吻，公開確立兩人關係<ref group="參">漫畫第77卷，TV 683《以性命為賭注的戀愛轉播（衝入現場）》</ref>。
