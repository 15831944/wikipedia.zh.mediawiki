'''佐兰·米拉诺维奇'''（{{lang-hr|Zoran Milanović}}，{{bd|1966年|10月30日|catIdx=Milanović}}），[[克罗地亚|克罗地亚]]政治人物，目前是[[中间偏左|中间偏左]]的[[克羅埃西亞社會民主黨|克羅埃西亞社會民主黨]]（SDP）领导人。他于2011年12月23日成为[[克罗地亚总理|克罗地亚总理]]，至2016年1月由[[蒂霍米爾·奧雷什科維奇|蒂霍米爾·奧雷什科維奇]]接任。

== 早年与事业 ==
1986年考入[[萨格勒布大学|萨格勒布大学]]法律专业。大学毕业后，他进入贸易法院（Trade Court）实习工作，1993年进入克罗地亚外交部工作，在这里他得到了他未来的政治对手[[伊沃·萨纳德|伊沃·萨纳德]]的赏识。一年后他到[[納戈爾諾-卡拉巴赫|納戈爾諾-卡拉巴赫]]做[[联合国|联合国]]主导的和平斡旋工作<ref>[http://www.zivotopis.hr/biografija/zoran-milanovic/ Zoran Milanović - Biografija] {{webarchive|url=https://web.archive.org/web/20120124113754/http://www.zivotopis.hr/biografija/zoran-milanovic/ |date=2012-01-24 }}</ref>。1996年他到[[布鲁塞尔|布鲁塞尔]]任克罗地亚派驻[[欧盟|欧盟]]和[[北约|北约]]的顾问。1999年他又回到外交部任职。

== 政治生涯 ==
2000年社会民主党赢得大选后，他被该党委任为同[[北约|北约]]的联络人。2003年他成为外交部长Tonino Picula的助理，直到民主联盟党（HDZ）赢得2003年的选举为止。

2004年他被选为社会民主党的中央委员。2006年成为该党的发言人。2007年通过竞选成功当选该党主席<ref name="obecanje">http://www.24sata.hr/politika/milanovic-obecao-ostavku-ako-ne-pobijedi-sanadera-61860</ref>。

2007年11月25日在国会选举中，社会民主党获得56席位居第二，比民主联盟党的66席相差10席。

2011年12月4日的国会选举中，该党赢得151席的81席成为第一大党。2011年12月23日，佐兰·米拉诺维奇成为[[克罗地亚总理|克罗地亚新一任总理]]。

== 个人生活 ==
他的父母Stipe和Gina生于[[錫尼|錫尼]]。他有一个兄弟名叫Krešimir。

1994年他与Sanja Musić结婚，之后两人生育了2个儿子Ante和Marko<ref>[http://arhiva.dalje.com/hr-hrvatska/biografija--zoran-milanovic/49361 Biografija: Zoran Milanović]</ref><ref>[http://www.vecernji.hr/izbori/premijerova-supruga-samozatajna-je-lijecnica-majka-clanak-353079 Premijerova supruga samozatajna je liječnica i majka]</ref>。

他能说[[英语|英语]]、[[俄语|俄语]]和[[法语|法语]]。
