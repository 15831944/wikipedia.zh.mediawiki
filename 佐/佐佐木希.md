'''佐佐木希'''{{BD|1988年|2月8日|catIdx=Sasaki Nozomi}} 是[[日本|日本]][[模特兒|模特兒]]，同時從事[[寫真偶像|寫真偶像]]、[[演員|演員]]等演藝工作，[[秋田县|秋田县]][[秋田市|秋田市]]出身。出道時為少女雜誌《[[PINKY|PINKY]]》模特兒，現為時尚雜誌《[[non-no|non-no]]》模特兒。所属事務所為[[TOP_COAT|TOP COAT]]（{{lang|ja|トップコート}}）。身高168公分。

她於17歲時被星探發掘入行做模特兒，曾獲美國電影網站評選為「全球最美的100張面孔」第33位、名列日本藝人之首，被譽為「美國人最喜歡的日本第一純情美少女」；日媒亦評她為日本20名最美女星冠軍，有「日本美少女」封號。

== 简介 ==
1988年2月，佐佐木希出生于秋田县秋田市。<ref name=" 佐佐木希美少女">{{cite web |url=http://fashion.ifeng.com/body/hdphoto/detail_2011_12/03/11080116_9.shtml|title=《佐佐木希清纯可人 被评美国人最喜爱的日本第一美少女》|accessdate=2011-12-03 |work=|publisher=凤凰网|language=zh-hans}}</ref>高中毕业后，在秋田市的一家名叫“流行屋”的杂货店里当店员。<ref name=" 佐佐木希美少女">{{cite web |url=http://fashion.ifeng.com/body/hdphoto/detail_2011_12/03/11080116_9.shtml|title=《佐佐木希清纯可人 被评美国人最喜爱的日本第一美少女》|accessdate=2011-12-03 |work=|publisher=凤凰网|language=zh-hans}}</ref>

2005年，佐佐木希被日本[[漫画|漫画]]杂志举办的《日本全国寻找美女》活动的搜查员发掘，照片登上杂志封面后，一夜走红。此后，佐佐木希一直担任模特，而且也是广告商及影视界的新秀。<ref name=" 佐佐木希美少女">{{cite web |url=http://fashion.ifeng.com/body/hdphoto/detail_2011_12/03/11080116_9.shtml|title=《佐佐木希清纯可人 被评美国人最喜爱的日本第一美少女》|accessdate=2011-12-03 |work=|publisher=凤凰网|language=zh-hans}}</ref>

2010年，佐佐木希作为“最美丽面孔日本美女”入选[[美国|美国]]电影网站“[[TC_Candler|TC Candler]]”“2010年最美丽的脸100人”排行榜（[[MOST_BEAUTIFUL_FACES_2010|MOST BEAUTIFUL FACES 2010]]），排行第三十三位，并且被誉为“美国人最喜欢的日本第一纯情美少女”。<ref name=" 佐佐木希美少女">{{cite web |url=http://fashion.ifeng.com/body/hdphoto/detail_2011_12/03/11080116_9.shtml|title=《佐佐木希清纯可人 被评美国人最喜爱的日本第一美少女》|accessdate=2011-12-03 |work=|publisher=凤凰网|language=zh-hans}}</ref>

2011年，《[[日经|日经]]杂志》评选出日本20名最美[[女优|女优]]，佐佐木希排行第1位。此后，佐佐木希成为“日本美少女”的代表。<ref name=" 佐佐木希美少女">{{cite web |url=http://fashion.ifeng.com/body/hdphoto/detail_2011_12/03/11080116_9.shtml|title=《佐佐木希清纯可人 被评美国人最喜爱的日本第一美少女》|accessdate=2011-12-03 |work=|publisher=凤凰网|language=zh-hans}}</ref> 同年，佐佐木希再次入选[[美国|美国]]电影网站“[[TC_Candler|TC Candler]]”“2011年最美丽的脸100人”排行榜（[[MOST_BEAUTIFUL_FACES_2011|MOST BEAUTIFUL FACES 2011]]），排名71位，仍然名列日本第1位。

2017年4月，佐佐木希与日本搞笑艺人{{link-ja|渡部建}}宣布结婚。<ref>[https://mainichi.jp/articles/20170409/dyo/00m/200/019000c アンジャッシュ渡部 佐々木希と結婚へ　「行列」SPで生発表] {{webarchive|url=https://web.archive.org/web/20170415100237/https://mainichi.jp/articles/20170409/dyo/00m/200/019000c |date=2017-04-15 }}.每日新闻.2017-04-09.[2017-04-09].</ref>

2018年2月，佐佐木希於Instagram 上宣布懷孕。

2018年9月13日，佐佐木希於Instagram 上宣布生下一名男嬰，母子平安。
