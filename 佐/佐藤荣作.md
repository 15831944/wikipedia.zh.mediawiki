'''佐藤榮作'''（{{jpn|j=佐藤 栄作|hg=さとう えいさく|rm=Satō Eisaku|f=h}}，{{bd|1901年|3月27日|1975年|6月3日|catIdx=Satou, Eisaku}}），[[日本|日本]][[政治家|政治家]]，前[[日本內閣總理大臣|日本內閣總理大臣]]，同時也是[[日本內閣總理大臣在任時間排行|日本連續任期中任期最久的首相]]，執政達7年8個月，其親兄[[岸信介|岸信介]]亦曾為總理大臣。他也是现任日本首相[[安倍晋三|安倍晋三]]的外叔公。岸信介、佐藤榮作、安倍晋三被稱為「一家三[[宰相|宰相]]」{{notetag|「宰相」，[[內閣總理大臣|內閣總理大臣]]的雅稱，即[[首相|首相]]。}}<ref>[http://www.chinatimes.com/realtimenews/20170302005584-260408 吾三連？ 日安倍首相望能獲黨內修法支持續任]</ref><ref>{{Cite web |url=http://www.worldjournal.com/5144949/article-%E6%8A%97%E6%88%B0%E7%A0%94%E7%A9%B6%E6%9C%83%EF%BC%9A%E9%98%B2%E6%97%A5%E6%9C%AC%E8%BB%8D%E5%9C%8B%E4%B8%BB%E7%BE%A9%E5%BE%A9%E6%B4%BB/ |title=抗戰研究會：防日本軍國主義復活 |access-date=2017-10-23 |archive-url=https://web.archive.org/web/20171023174829/http://www.worldjournal.com/5144949/article-%E6%8A%97%E6%88%B0%E7%A0%94%E7%A9%B6%E6%9C%83%EF%BC%9A%E9%98%B2%E6%97%A5%E6%9C%AC%E8%BB%8D%E5%9C%8B%E4%B8%BB%E7%BE%A9%E5%BE%A9%E6%B4%BB/ |archive-date=2017-10-23 |dead-url=yes }}</ref><ref>[http://ca.ntdtv.com/xtr/b5/2016/09/24/a1288092.html 在美慶生夫人當眾示愛 62歲安倍滿臉羞]</ref>。佐藤榮作在1974年獲得[[諾貝爾和平獎|諾貝爾和平獎]]。

1924年[[東京帝國大學|東京帝國大學]]法學部畢業，開始在[[鐵道省|鐵道省]]任職。1946年為鐵道總局長官，1947年為[[運輸省|運輸省]]次官。1948年加入自由黨（[[自由民主黨_(日本)|自民黨]]的前身之一）。在[[吉田茂|吉田茂]]任內閣總理大臣時，吉田大量開除[[黨人派|黨人派]]，[[官僚派|官僚派]]得勢，特別是佐藤和[[池田勇人|池田勇人]]多番獲得提拔。1948年獲吉田任命為[[內閣官房長官|內閣官房長官]]，1949年1月成為眾議院議員（其後連任了11次），1949年2月任自由黨政調會長。1954年[[造船丑闻事件|造船丑闻事件]]中，幸得時任總理大臣[[吉田茂|吉田茂]]要求[[犬養健|犬養健]]法務大臣發動[[檢察指揮權|檢察指揮權]]，免於被捕，但吉田内阁因此倒台。

佐藤時期的日本為高速成長期的黃金時代，任內政績斐然，除了連接[[東京|東京]]和[[神戶|神戶]]的[[東名高速公路|東名高速公路]]和[[名神高速公路|名神高速公路]]全線通車外，還核定了[[山陽新幹線|山陽新幹線]]的興建。佐藤除了維持著平均10%以上的[[經濟成長率|經濟成長率]]外，透過斡旋成功從[[美國|美國]]手上拿回[[沖繩|沖繩]]的統治權，納入日本領土。比起前任[[池田勇人|池田勇人]]著手於經濟外，佐藤處理了其他更錯綜複雜的政治性問題，也獲得成功。1969年日本[[GDP|GDP]]超越[[西德|西德]]成為世界第二大經濟體。

佐藤在外交上極力偏向[[中華民國|中華民國]]，反對[[毛澤東|毛澤東]]的[[中華人民共和國|中華人民共和國]]政權，也是最後一個於任內訪問[[台灣|台灣]]的[[日本首相|日本首相]]，然而佐藤後期因為[[田中角榮|田中角榮]]迅速崛起，無法指定其屬意的接班人[[福田赳夫|福田赳夫]]繼任[[自民黨總裁|自民黨總裁]]，繼而造成70年代日本政局不穩甚至[[角福戰爭|角福戰爭]]的開端。

== 個人生涯 ==

=== 家庭背景 ===
佐藤出生於現在[[日本|日本]][[山口縣|山口縣]][[熊毛郡|熊毛郡]][[田布施町|田布施町]]，為製酒業者佐藤秀助、佐藤茂世夫婦的第三子（其二哥[[岸信介|岸信介]]，亦曾為日本首相），父親秀助曾於山口縣廳任職，1898年轉而從事製酒業。

佐藤秀助本姓“岸”，由于是被[[招贅|招贅]]的[[婿養子|婿養子]]，所以改用妻姓「佐藤」。而「岸信介」因为被[[過繼|過繼]]為其父本家的[[养子|养子]]，而改稱岸信介。

=== 學生時代 ===
1907年佐藤進入小學就讀，就讀高中（舊制第五高等學校，現[[熊本大學|熊本大學]]）時，偶然認識往後成為日本首相的[[池田勇人|池田勇人]]，不過佐藤選丙類朝法學就讀；池田則是選了乙類的文科就讀。1921年4月，佐藤進入[[東京帝國大學|東京帝國大學]][[法學部|法學部]]就讀。1923年通過了[[高等文官試驗|高等文官試驗]]，1924年順利從[[東京帝國大學|東京帝國大學]]畢業。

== 首相生涯 ==
[[池田勇人|池田勇人]]首相因喉部疑患癌症，決定辭職，並選佐藤為繼任者。

=== 經濟政策 ===
1965年上半年，日本經濟蕭條，政府實行以下政策以助經濟：
* 發行國債
* 增加財政融資
* 擴大公共事業投資
* 減稅
* 增強出口

經濟迅速轉好。由1965年10月起，日本更出現了「[[伊奘諾景氣|伊奘諾景氣]]」。

=== 外交事務 ===
[[File:Sato_and_Nixon_1972.png|200px]][[尼克森|尼克森]]，摄于1972年]]
1965年和[[韓國|韓國]]簽署《[[日韓基本條約|日韓基本條約]]》，[[日韓關係|日韓關係]]正常化。

==== 日中關係 ====
佐藤多次拒絕[[中華人民共和國|中華人民共和國]]代表訪日，並阻礙雙方貿易關係發展。

1965年，貸款1億5千万[[美元|美元]]予[[中華民國|中華民國]]政府。1967年9月7日，訪問[[臺灣|臺灣]]。9月9日，[[蔣經國|蔣經國]]函電[[宋美齡|宋美齡]]：
{{quote|
「佐藤訪華三日已於今午離台謹聞遙祝一路平安福體康泰兒」<ref>[[周美華|周美華]]、[[蕭李居|蕭李居]]編，《蔣經國書信集——與宋美齡往來函電》（上），[[台北|台北]]「[[國史館|國史館]]」出版，2009年，第322頁。</ref>
}}

1969年11月21日，與[[美國總統|美國總統]][[尼克遜|尼克遜]]簽署聯合公報，提出了臺灣對[[日本|日本]]安全是重要因素。1971年7月15日，尼克遜宣布有訪問[[中國大陸|中國大陸]]的計劃。[[美國|美國]]轉變對中華人民共和國的態度，使一向支持[[中華民國|中華民國]]的日本失利。另一方面，同年8月，尼克遜宣布不再以[[美元|美元]]兌[[黃金|黃金]]，對進口品徵收一成附加稅，令[[日元|日元]]升值17%，固定匯率制瓦解。

在佐藤執政期間，曾多次參拜[[靖國神社|靖國神社]]。

==== 沖繩問題 ====
[[沖繩|沖繩]]在戰後一直是[[駐日美軍|駐日美軍]]基地。1965年1月佐藤訪問美國，要求歸還冲繩；同年8月成為首位訪問當地琉球的戰後首相；1969年尼克遜為了令日本限制向美國出口[[紡織品|紡織品]]，和日本發表三年後歸還該地的聲明；1972年（[[昭和|昭和]]47年）5月15日，美國將沖繩[[行政權|行政權]]連同[[釣魚台列嶼|釣魚台列嶼]]（尖閣諸島）交還給日本，引起[[中華民國|中華民國]]不滿。

=== 長期執政的終結 ===
佐藤掌政7年8個月，共在職2,798天，能夠持續這麼久，原因包括：
* 內外政策的成功
* 黨人派勢力減弱：[[大野伴睦|大野伴睦]]、[[河野一郎|河野一郎]]等著名黨人派政治家逝世，令他有機會平衡黨內派系
* 在野黨數目增加，反對力量分散

後來因外交政策的失利，自民黨內部要求佐藤辭職。

== 总理退任後的生活 ==
[[File:1974_Nobel_Peace_Prize_awarded_to_Eisaku_Satō.jpg|thumb]]的金牌（国立公文書館所藏）]]
1967年12月，佐藤首次提出「[[非核三原則|非核三原則]]」：日本不會擁有、製造、運進[[核武|核武]]。但佐藤荣作的遗嘱中留有日美两国首脑签署的“美国保留在危机时刻将核武器再度运入日本”的秘密文件。

1974年晚秋，因令日本加入《[[不扩散核武器条约|不扩散核武器条约]]》而與[[肖恩·麦克布赖德|肖恩·麦克布赖德]]獲[[諾貝爾和平獎|諾貝爾和平獎]]，这也是日本人第一次获得此奖。

1975年5月19日，佐藤在位于[[築地|築地]]的料亭「{{link-ja|新喜乐|新喜楽}}」与財政界名人应酬中，起身欲上厕所却突然倒下，接着发出呻吟。当時赶来的[[慶應義塾大学|慶應義塾大学]]与[[東京大学|東京大学]]的医師团体与家人都没有准备将佐藤送去医院。佐藤晕倒的原因为[[脑溢血|脑溢血]]。在寛子夫人的強烈要求下先在「新喜乐」观察了4天情况後，佐藤才被送去{{link-ja|東京慈惠会医科大学附属医院|東京慈恵会医科大学附属病院}}，但他本人一直处于昏迷状态，最终于6月3日死去。享年74岁。

6月16日，在[[日本武道館|日本武道館]]举行自[[大隈重信|大隈重信]]以来首次的「国民葬」。葬儀委員長为[[田中角荣|田中角荣]]。遺族代表为其兄・[[岸信介|岸信介]]。

== 家庭 ==
1926年佐藤榮作和[[佐藤寬子_(佐藤榮作之妻)|佐藤寬子]]結婚，生有兩子，次子[[佐藤信二|佐藤信二]]曾為參議員。
