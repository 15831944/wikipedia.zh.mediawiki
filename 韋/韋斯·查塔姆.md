'''韋斯·查塔姆'''（{{lang-en|'''Wes Chatham'''}}，本名'''約翰·衛斯理·查塔姆'''（'''John Wesley Chatham'''），{{bd|1978年|10月11日}}）是一名美國演員。

他曾演出多部電影，例如《{{link-en|震撼效應|In the Valley of Elah}}》、《[[殊不簡單|殊不簡單]]》、《[[寫出友共鳴|寫出友共鳴]]》和《{{link-en|搏命擂台|The Philly Kid}}》，以及在《[[飢餓遊戲終極篇：自由幻夢1|飢餓遊戲終極篇：自由幻夢1]]》和《[[飢餓遊戲終極篇：自由幻夢2|自由幻夢2]]》中飾演Castor。<ref>{{Cite web |url = http://www.hollywoodreporter.com/news/hunger-games-mockingjay-adds-wes-634728 |title = 'Hunger Games: Mockingjay' Adds Wes Chatham |author = Rebecca Ford |date = 2013-09-23 |publisher = The Hollywood Reporter |language = en}}</ref>2015年，他在[[Syfy|Syfy]]電視劇《[[太空無垠|太空無垠]]》中飾演Amos Burton。

== 早年 ==
韋斯·查塔姆於1978年10月11日出生，在北佐治亞州長大。查塔姆的父母在他二歲時離婚，而他的童年大部分時間都與他的母親、妹妹和兄弟一起度過。有一天心血來潮，查塔姆的母親帶著自己的妹妹到佐治亞州薩凡納（Savannah）的Tide廣告試鏡，並帶了韋斯一起去。在大廳等候妹妹的時候，韋斯被選角導演發現，並在五歲時，他獲得了一場Tide的全國性運動。

在十三歲的時候，查塔姆和父親同住，沒有太多的監管下，而成為一個不安分和叛逆的青少年，他從高中被踢出來，並送到佐治亞州勞倫斯維爾（Lawrenceville）的谷內縣排解教育中心（Give Center）完成學業。谷內縣排解教育中心是為麻煩青少年而設的一間給予第二次機會的學校，提供非常小的班級。當他在中心上課的時候，一間在亞特蘭大郊外的專業劇院公司開始了與學校的輔導計劃，而查塔姆被選中來編寫一個後來由他同學演出的戲劇。他從這個經驗中發現了他對藝術的熱愛。

== 生涯 ==
高中畢業後，查塔姆加入了軍隊。他在{{link-en|USS Essex (LHD-2)|USS Essex (LHD-2)|USS Essex}}的飛行甲板擔任航空消防員，在墜毀和打撈方面工作了四年。查塔姆闖進演藝界是在當[[丹素·華盛頓|丹素·華盛頓]]選擇他的船拍攝電影《''{{link-en|Antwone Fisher (film)|Antwone Fisher (film)|Antwone Fisher}}''》時，他的巡視工作完成前的三個月。查塔姆是由選角導演{{link-en|羅比·里德|Robi Reed}}挑選的，而里德正在為電影尋找看起來真實的軍事人員。這是查塔姆的第一部電影製作經驗，這引導他進一步追求他畢生的演藝夢想。在《''Antwone Fisher''》之後，里德說服查塔姆搬到荷里活，而不久之後就選他在Showtime的電視劇《''{{link-en|Barbershop (TV series)|Barbershop (TV series)|Barbershop}}''》中飾演他的第一個常設角色。

查塔姆在[[保羅·哈吉斯|保羅·哈吉斯]]選他在《{{link-en|震撼效應|In the Valley of Elah}}》中飾演下士Steve Penning時獲得進一步的注意。2009年，查塔姆在[[奧利華·史東|奧利華·史東]]的《[[殊不簡單|殊不簡單]]》中飾演Frank Benedict，喬治·W·布殊的學會兄弟。第二年，查塔姆在另一部熱門的CBS電視劇《{{link-en|秘密部队 (电視剧)|The Unit|秘密部队}}》中演出常設角色。查塔姆飾演新小隊成員Sam McBride中士（又名Whiplash）。查塔姆還在Brett Simmons的《''{{link-en|Husk (film)|Husk (film)|Husk}}''》（2011年）中飾演Brian Danielson。

查塔姆在2011年的電影《[[寫出友共鳴|寫出友共鳴]]》中扮演[[愛瑪·史東|愛瑪·史東]]角色的哥哥Carlton Phelan。這部電影贏得了2012年[[美國演員工會獎|美國演員工會獎]]最佳集體演出獎。2012年，查塔姆在《{{link-en|搏命擂台|The Philly Kid}}》中飾演他的第一個片名角色。作為一個混合武術的粉絲，查塔姆一心投入Dillion McGwire的人物，表演了他所有的絕技。該片於2012年5月在電影院首映。在《搏命擂台》後，查塔姆出演《''This Thing With Sarah''》，電影於2013年10月在聖地牙哥電影節亮相。2013年，查塔姆專注於兩部電影，《''Broken Horses''》和《''{{link-en|The Town That Dreaded Sundown (2014 film)|The Town That Dreaded Sundown (2014 film)|The Town That Dreaded Sundown}}''》。

2015年11月，查塔姆開始在[[Syfy|Syfy]]電視劇《[[太空無垠|太空無垠]]》中飾演Amos Burton。

== 個人生活 ==
查塔姆的配偶是電視圈名人{{link-en|詹·布朗|Jenn Brown}}。<ref>{{Cite web |url = http://thebiglead.com/2011/09/28/espns-jenn-brown-is-engaged-to-actor-wes-chatham/ |title = ESPN's Jenn Brown is Engaged to Actor Wes Chatham |author = Jason McIntyre |date = 2011-09-28 |publisher = The Big Lead |language = en}}</ref>他們有兩名兒子，2015年出生的John Nash和2016年出生的Rhett Jameson。
