'''韋卡殊·多拉素'''（'''{{lang-te|వికాష్ దొరసూ}}'''，{{bd|1973年|10月10日|catIdx=దొరసూ, వికాష్}}），出生在{{Link-en|哈弗勒爾|Harfleur}}，是一名前[[法國|法國]]足球[[中場|中場]]。多拉素現在是法國球會[[聖格拉西永|聖格拉西永]]的主席。

== 球會生涯 ==
多拉素在[[勒哈費爾足球會|勒哈費爾]]展開其球員生涯並在1993年8月對[[聖伊天足球會|聖伊天]]的賽事中首次亮相，雙方最終打悶和0-0。他在效力了勒哈費爾5年後於1998年轉投[[里昂足球會|里昂]]，他被看作是一個有潛質的中場以及里昂史上最擅長盤球的球員。

2001年至2002年間，多拉素曾被短暫外借至[[波爾多足球會|波爾多]]。他曾協助里昂贏得2003年和2004年的[[法國足球甲級聯賽|法甲]]冠軍。2004年，多拉素轉投[[意大利足球甲級聯賽|意甲]]球會[[AC米蘭|AC米蘭]]。2005年，他雖然名列在[[2005年歐洲冠軍聯賽決賽|歐聯決賽]]的後備名單中，不過並沒有上陣，球會最終在[[互射十二碼|互射十二碼]]階段不敵[[利物浦足球會|利物浦]]。

2005年，多拉素重返法國加盟[[巴黎聖日耳門足球會|巴黎聖日耳門]]。2006年，他是球會奪得[[法國盃|法國盃]]的骨幹球員，他在決賽中憑籍一記25碼外的入球擊敗[[馬賽足球會|馬賽]]而奪冠。

2006年9月，多拉素批評領隊{{Link-en|居伊·拉孔貝|Guy Lacombe}}在與[[隊報|隊報]]訪問中的言論。一個月後，多拉素的合約被會方所終止<ref>{{en}}{{cite news|url=http://edition.cnn.com/2006/SPORT/football/10/11/france.dhorasoo/index.html|title= Midfielder Dhorasoo sacked by PSG|publisher=CNN.com|date=2006-10-11|accessdate=2007-07-03}}</ref>，成為自1973年以來第一位被法國球會解僱的球員。

2007年7月3日，多拉素加盟[[利禾奴足球會|利禾奴]]<ref>{{en}}{{cite news|url=http://uk.reuters.com/article/UK_WORLDFOOTBALL/idUKL0382547820070703|publisher=Reuters|title=Soccer-French midfielder Dhorasoo signs for Livorno|date=2007-07-03|accessdate=2007-07-03}}</ref>。然而，由於他與會方的分歧，在同年10月，利禾奴取消了多拉素的合約。據球會主席說，多拉素拒絕與球會的青年隊作賽，所以球會主席認為離開球會是他唯一的選擇，這正是為何終止他合約的原因。即使，[[法國足球乙級聯賽|法乙]]球會[[格洛堡足球會|格洛堡]]提出希望簽入多拉素，不過他最終選擇在2008年1月11日對外宣佈退役。

== 國際賽生涯 ==
1999年3月27日，多拉素在對[[烏克蘭國家足球隊|烏克蘭]]的賽事中首次代表[[法國國家足球隊|法國]]上陣。他合共代表法國隊上陣了18場並攻入1球。多拉素亦有參與過在[[德國|德國]]舉行的[[2006年世界盃|2006年世界盃]]。

2004年9月，多拉素被法國隊徵召出戰[[2006年世界盃外圍賽|2006年世界盃外圍賽]]。世界盃之後，多拉素以紀錄片的形式公開披露他在法國隊一個月期間的點滴。時任法國隊領隊[[雷蒙·杜明尼治|雷蒙·杜明尼治]]和[[法國足球總會|法足總]]對於多拉素出版紀錄片表示憤怒並警告他不要出版。其後，多拉素宣佈退出法國隊，並說道：「我對於為法國隊比賽已經沒有興趣了，就這樣吧<ref>{{en}}{{cite web|url=http://www.uefa.com/footballeurope/news/Kind=2/newsId=443314.html|title=Munitis moves, Toni stays|publisher=UEFA|date=2006-08-08}}</ref>。」

== 個人 ==
多拉素出身自[[毛里裘斯|毛里裘斯]]的[[印度人|印度人]]家庭。他是一名[[泰盧固人|泰盧固人]]，多拉素的祖先從[[印度|印度]]南部的[[安得拉邦|安得拉邦]]遷遷移到毛里裘斯的。多拉素本身亦是一名[[印度教|印度教]]信徒。他成為職業足球員之前是在大學修讀[[經濟|經濟]]的。

多拉素與拍拖很長時間的埃米莉（Émilie）結婚並育有兩名女兒羅絲（Rose）和莎拉（Sara）。

2003年，多拉素開始支持一隊反對[[同性戀恐懼症|同性戀恐懼症]]和其他[[歧視|歧視]]的足球會<ref>{{en}}{{cite web|url=http://en.uefa.com/magazine/news/Kind=16384/newsId=394550.html|title=Dhorasoo tackles great taboo|publisher=UEFA|date=2006-02-17|deadurl=yes|archiveurl=https://web.archive.org/web/20090215002412/http://en.uefa.com/magazine/news/Kind=16384/newsId=394550.html|archivedate=2009-02-15}}</ref>。

2009年7月，他成為法國球會[[聖格拉西永|聖格拉西永]]的主席。

==電影==
2006年世界盃期間，多拉素製作了一部個人電影{{Link-en|Substitute (film)|Substitute (film)|後備}}，這套電影在隨後的一年在法國發佈。

他亦曾參演一部名為{{Link-en|La très très grande entreprise|La très très grande entreprise}}的電影，該套電影在2008年11月發行。
