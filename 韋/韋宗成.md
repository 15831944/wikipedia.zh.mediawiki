'''韋宗成'''{{BD|1982年|1月23日||catIdx=W}}是[[臺灣|臺灣]]的[[漫畫家|漫畫家]]。是網路漫畫發表平台「創意漫畫大亂鬥」的站長。在2009年以國內第一部長篇諷刺政治漫畫書名《[[馬皇降臨|馬皇降臨]]》成為注目的焦點，並以此展開商業漫畫的創作。他的漫畫作品《[[冥戰錄|冥戰錄]]》分別獲得第三、九屆[[金漫獎|金漫獎]]最佳[[少年漫畫|少年漫畫]]類獎，和第五屆[[金漫獎|金漫獎]]最佳原型設計獎，女主角林默娘也成為西門町街頭的看板娘。
他的網路聲量相當活躍，根據網站“DailyView網路溫度計”統計的台灣漫畫家“熱門漫畫大師聲量排名”（從2017年4月起至2018年4月的統計期間）贏得第一名<ref>[https://tw.news.yahoo.com/%E5%8F%B0%E7%81%A3%E7%9A%84%E6%BC%AB%E7%95%AB%E5%AE%B6-%E9%87%91%E6%95%96-%E7%9B%A4%E9%BB%9E%E7%B6%B2%E8%B7%AF%E7%86%B1%E9%96%80%E6%BC%AB%E7%95%AB%E5%A4%A7%E5%B8%AB-160000456.html 台灣的漫畫家「金敖」！盤點網路熱門漫畫大師]</ref>。

== 簡歷 ==
[[File:Wei_Tsong-cheng_signature_in_Comic_Exhibition_20120814.jpg|thumb]]的韋宗成簽名]]
*在2000年成立[[網路漫畫|網路漫畫]]發表網站《[[創意漫畫大亂鬥|創意漫畫大亂鬥]]》，創作很多種長篇短篇的連環漫畫、和[[四格漫畫|四格漫畫]]以及真人漫畫，題材沒有限制，甚至有短片的拍攝等等。
*當兵的時候在《創意漫畫大亂鬥》刊載四格漫畫《[[中華民國國軍四格|中華民國國軍四格]]》後，引起網路民眾轉載。
*2007年《八卦山》是以[[馬關條約|清朝割讓臺灣]]後、[[台灣日治時期|日治時期]]之前[[台灣客家人|台灣客家人]]抗日英雄[[吳湯興|吳湯興]]為故事的主角，敘訴台灣人民不願土地被外國人所強奪的故事，可以說是電影《[[一八九五|一八九五]]》的漫畫版，總共花了前後三年才完成。並贏得2007年[[國立編譯館優良漫畫獎|國立編譯館優良漫畫獎]]乙類第三名<ref>[http://comic.nict.gov.tw/award_list2.php?prizecomic_id=7&prizecomic_id2=19 民國96年優良漫畫獎乙類作品]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2011年7月8日，韋宗成在[[噗浪|噗浪]]發文說明本作不能登上《[[台灣漫畫月刊|台灣漫畫月刊]]》的原因<ref>[http://www.plurk.com/p/d0i325 天之將明，其黑尤烈]</ref>。

*《[[馬皇降臨|馬皇降臨]]》是以中華民國前後兩任總統[[馬英九|馬英九]]、[[陳水扁|陳水扁]]在政治上發生的事情為故事的主要內容，故事當中被影射模仿的政客超過三十人，在第14屆[[開拓動漫祭|開拓動漫祭]]上開始發售，接受網上預購。由於在發行之前媒體的廣泛報導和積極宣傳，使漫畫備受關注，成為動漫祭中的焦點，銷量十分理想。<ref>[http://www.nownews.com/2009/07/26/91-2483639.htm 熱賣1千5百本 作者韋宗成：純粹想惡搞]</ref>進而進入商業通路。

*《[[AV端指|AV端指]]》是以諷刺臺灣色情產業為主的短篇綜合，內容構成為四部短篇，雖然各自獨立卻互相牽連，維持了一貫的嘲諷風格，諷刺了一切不合理的社會現象。由於前部作品《馬皇降臨》的原因，本作也在第15屆開拓動漫祭上火熱的銷售。

*《[[冥戰錄|冥戰錄]]》以[[921大地震|921大地震]]後為故事背景，敘述臺灣警察因為妖魔肆虐而成立「黑日專案」，主角陳柏戎在一次的行動中找到了與傳統著名神祇[[林默娘|林默娘]]（[[媽祖|媽祖]]）同名同姓的小女孩。為作者從2000年開始構思的作品《冥戰錄》的最後決定作品，作品人物以[[新北市|新北市]][[三峽區|三峽區]]為活動主要範圍，大量融入地方特色。本作以單行本連載方式發行銷售。

*《[[五都爭霸|五都爭霸]]》為《馬皇降臨》續作，以臺灣2010年底五直轄市（五都）選舉為故事內容，將五都[[擬人化|擬人化]]由[[泛藍|泛藍]]、[[泛綠|泛綠]]派出人馬相爭，承繼《馬皇降臨》之影射風格而再度引起風潮，本作於2010年10月30日於台北市汀州路[[金石堂書店|金石堂書店]]汀州店首賣。

*《[[跳躍吧！大同萌會|跳躍吧！大同萌會]]》以清末民初歷史為背景，描述[[蘿莉控|蘿莉控]]孫玟（影射[[孫文|孫文]]）為把對[[蘿莉|蘿莉]]的愛推廣到全中國進行的革命，以時下流行的[[真人秀|真人秀]]歌唱比賽重新詮釋滿清末年至民國肇建的歷史，並將各種勢力用歌唱隊伍詮釋。本作在第19屆[[開拓動漫祭|開拓動漫祭]]上熱賣。<ref>[http://tw.nextmedia.com/applenews/article/art_id/34002151/IssueID/20120205 動漫祭吸3萬人 日名家今掀高潮]</ref>

*《[[新世紀國軍戰士|新世紀國軍戰士]]》以四格漫畫為主，使用架空的機器人部隊「漢神旅」為故事背景，嘲諷[[中華民國國軍|中華民國國軍]]的各種不合理規定與事物，以及作者軍旅生活的回憶。本作上市前夕遇上了[[洪仲丘事件|洪仲丘事件]]，作品與時事的聯想，使本作在第22屆開拓動漫祭上熱賣。<ref>[http://www.nexttv.com.tw/news/realtime/lifestyle/10783082/%E6%B4%AA%E4%BB%B2%E4%B8%98%E4%BA%8B%E4%BB%B6%E5%BB%B6%E7%87%92%E3%80%80%E3%80%8C%E5%9C%8B%E8%BB%8D%E9%A1%8C%E6%9D%90%E3%80%8D%E6%BC%AB%E7%95%AB%E5%A4%AF 洪仲丘事件延燒　「國軍題材」漫畫夯]</ref>

*2012年，作品《冥戰錄》透過[[未來數位|未來數位]]與西門徒步區街區發展促進會合作，將林默娘打造成西門街區的形象角色。<ref>[http://gnn.gamer.com.tw/3/64463.html 韋宗成《冥戰錄》林默娘化身西門町形象角色 歡慶媽祖誕辰小動畫街頭開播]</ref>2013年，林默娘正式上任成西門町[[看板娘|看板娘]]，未來數位並和[[台北天后宮|台北天后宮]]合作推出林默娘[[護身符|護身符]]。<ref>[http://gnn.gamer.com.tw/0/83560.html5 「西門動漫月」台北天后宮媽祖娘娘將上任西門町街區動漫看板娘]</ref>

*2014年1月，作品《冥戰錄》在日本電子漫畫書店「電子貸本Renta！」發行日文數位版，這也是以臺灣為主體創作的臺灣漫畫第一次登上日本數位漫畫書店。<ref>[http://www.chinatimes.com/newspapers/20140120000382-260115 韋宗成冥戰錄 進攻日電子書店-中國時報]</ref>

*《[[六都爭霸|六都爭霸]]》為《五都爭霸》續作，以臺灣[[2014年中華民國直轄市長及縣市長選舉|2014年六都選舉]]為故事內容，除了前作的五都外，升格後的[[桃園市|桃園市]]與其他縣市的擬人化也一起登場；本作於2014年10月25日於[[開拓動漫祭|開拓動漫祭]]首賣。

*《[[我的委員長哪有這麼萌|我的委員長哪有這麼萌]]》以民國初期[[國民政府軍事委員會|國民政府軍事委員會]]委員長[[蔣中正|蔣中正]]為主角，描述蔣中正領導[[國民革命軍北伐|國民革命軍北伐]]的事蹟。首次發表於[[Line_Webtoon|Line Webtoon]]平台，2015年10月24日集結單行本出版。

*2015年8月，作品《冥戰錄》與日本原型師[[宮川武|宮川武]]合作，推出女主角林默娘的模型，在漫博首次登場。<ref>{{cite news |language = zh-tw |url =http://gnn.gamer.com.tw/2/119072.html |title = 【漫博 15】 韋宗成×宮川武《冥戰錄》林默娘模型可愛亮相 |publisher = 巴哈姆特電玩資訊站 |date = 2015-08-07}}</ref>

*2015年10月，在REALITY FANTASY2模型展中，由開拓動漫與未來數位聯合舉辦，以《冥戰錄》女主角林默娘作為立體創作雛形的天妃大賽，為該活動首次以臺灣漫畫角色為主募集的模型比賽，最終有30項入圍作品<ref>{{cite news |language = zh-tw |url = http://www.f-2.com.tw/index.php?q=ff/121746 |title = RF2模型大賽入圍名單公告!! |publisher = 開拓動漫祭 |date = 2015-10-19 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，於10月25日活動當天頒獎<ref>{{cite news |language = zh-tw |url =https://www.facebook.com/media/set/?set=a.970996412958551.1073741837.221511137907086&type=3 |title = 天妃原型大賽參賽作品 |publisher = 韋宗成FaceBook粉絲團 |date = 2015-10-25}}</ref>

*《[[霸海皇英|霸海皇英]]》繼承了《馬皇降臨》的風格，此次以中華民國總統[[蔡英文|蔡英文]]為主角，描述前總統[[馬英九|馬英九]]執政下的八年重大事件為主，嘲諷當紅政治人物。本作發行的同時也販售了週邊產品抱枕跟洋蔥餅，<ref>{{cite news |language = zh-tw |url =http://news.tvbs.com.tw/life/671125 |title = 動漫祭總統人氣高！　漫畫《霸海皇英》軋一角 |publisher = TVBS新聞 |date = 2016-08-27}}</ref>在第28屆開拓動漫祭上熱賣並造成話題。<ref>{{cite news |language = zh-tw |url =http://www.setn.com/News.aspx?NewsID=177040-%E5%B0%8F%E8%8B%B1KUSO%E6%BC%AB%E7%95%AB%E3%80%8A%E9%9C%B8%E6%B5%B7%E7%9A%87%E8%8B%B1%E3%80%8B%E4%B8%8A%E5%B8%82%E3%80%80%E4%B8%AD%E5%A4%96%E6%94%BF%E6%B2%BB%E4%BA%BA%E7%89%A9%E5%A4%A7%E4%BA%82%E9%AC%A5 |title = 小英KUSO漫畫《霸海皇英》上市　中外政治人物大亂鬥 |publisher = 三立新聞 |date = 2016-08-27}}</ref>

*2016年11月23日，由韋宗成、[[哈亞西|哈亞西]]、[[三色坊|三色坊]]、紅豆子等多位漫畫家，成立[[未來數位|未來數位]]漫畫部門的「創漫工作室」，隨後有[[清水_(繪師)|清水]]、[[Coin_(漫畫家)|Coin]]等多位創作者加入，致力於漫畫出版與同人場方面的持續創作、對外接案及海外推廣。

*2017年11月24日，韋宗成、[[漢寶包|漢寶包]]、[[左萱|左萱]]、[[AKRU|AKRU]]等多位漫畫家前往日本，參加由[[Comic_Catapult|Comic Catapult]]舉辦的海外漫畫家推廣活動，於[[臺北駐日經濟文化代表處|臺北駐日經濟文化代表處]]舉行座談。
