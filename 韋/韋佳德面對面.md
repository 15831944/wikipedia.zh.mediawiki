《'''韋佳德面對面'''》，2017年由[[韋佳德|韋佳德]] (Riccardo Moratto) 建立的一檔在個人YouTube頻道上的網路訪談節目，韋佳德 面對面 Face2Face採用採訪談話的方式，邀請全球政界、商界、文藝界、體育界知名人士分享他們在各領域的經驗。不僅知名人士，只要是曾在社會上發揮過貢獻，或具話題性但其故事能夠鼓舞人心的平凡人物，也是該節目的邀請對象。 <ref>{{cite web |url= https://star.ettoday.net/news/1045416|title= “專業藏在生命歷程裡！ 韋佳德「面對面」各界名人|accessdate= 2018年|language =zh-cn}}</ref> 
。

== 概述 ==
節目中通過主持人和受訪者的交流，展現受訪者人物性格和獨到見解，經過主持人韋佳德的訪談，觀眾可以更好地了解世界及自我，也能夠一窺世界的多變樣貌，成就出與各種領域專家的相遇紀事。 <ref>{{cite web |url= https://star.ettoday.net/news/1045416|title= “專業藏在生命歷程裡！ 韋佳德「面對面」各界名人|accessdate= 2018年|language =zh-cn}}</ref> <ref>{{cite web |url= https://news.tvbs.com.tw/entertainment/976455|title= “義大利籍卻擁有神級中文他被譽為「蔡康永接班人」|accessdate= 2018年|language =zh-cn}}</ref>
