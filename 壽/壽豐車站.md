'''壽豐車站'''位於[[台灣|台灣]][[花蓮縣|花蓮縣]][[壽豐鄉|壽豐鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]，因配合[[花東鐵路電氣化|花東鐵路電氣化]]及花東車站效能提升計畫，已於2015年8月26日切換為高架車站<ref>[http://news.ltn.com.tw/news/local/paper/910225 壽豐新站啟用 鄉民讚像高鐵]2015-08-27 自由時報
記者：王錦義</ref>。

== 車站構造 ==
*[[高架車站|高架車站]]
*[[島式月台|島式月台]]兩座

== 新站完工後的效益 ==
*2015年中完工後的車站改為高架式車站，站房在月台下方，一方面可改善此路段經常面臨的淹水之苦，另一方面也可平衡前、後站的發展<ref>[http://www.chinatimes.com/realtimenews/20150625004435-260405 壽豐車站高架化 交通部今完成履勘]2015-6-25 中時電子報 記者：楊竣傑</ref>。
*新完工的月台跟以往相同為二座[[島式月臺|島式月臺]]，但卻有四股軌道，全面電氣化，雙軌化，改善過往列車須在本站[[交會|交會]]，大大地提升壽豐站至[[南平車站_(臺灣)|南平站]]間的運輸容量<ref>[http://www.ettoday.net/news/20150519/508822.htm 台鐵壽豐高架車站 9月將雙軌啟用可少5分待車時間] ETtoday東森新聞雲 2015-5-19 記者：林潔玲</ref><ref>[http://www.chinatimes.com/realtimenews/20150625004435-260405 壽豐車站高架化 交通部今完成履勘]2015-6-25 中時電子報 記者：楊竣傑</ref>。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]。
*停靠狀態：
**[[普悠瑪列車|普悠瑪號]]：
***南下：406、412、422/6422、222/6222、232/6232(終點)、448/6448、252次（終點）停靠
***北上：207次（起點）、225/6225、425/6425、431/6431、283/6283次（起點）停靠。
**[[太魯閣列車|太魯閣號]]：不停靠。
**[[自強號列車|自強號]]：
***南下：420、306、308、316、（328、332次逢週五、日行駛）、（340、350次逢週五、日停駛）停靠
***北上：407、303、307、439(週日停駛)、317、321（逢週五、日停駛）、323（逢週五、日行駛）次停靠。
**[[莒光號列車|莒光號]]：除了[[觀光列車|觀光列車]]以及[[團體列車|團體列車]]和609次(週五行駛)，604次(週一行駛)未停靠外，其餘皆停靠。
**[[臺鐵區間車|區間車]]：皆停靠。
*對號列車停靠率：53.8%；所有車種停靠率：65.5%。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1910年12月16日：設'''鯉魚尾驛'''。
*1917年9月26日：驛名隨地名改為'''壽'''（{{lang|ja|ことぶき}}；Kotobuki），為臺灣鐵路有史以來唯一一個採用一字站名的車站。
*1946年1月1日：地名和站名均改為'''壽豐'''。
*2015年8月26日：因配合花東鐵路電氣化及花東車站效能提升計畫切換為高架新站。
*2016年6月28日：啟用多卡通刷卡機。
*2016年12月：全部完工。
