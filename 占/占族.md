== 分佈 ==
歷史上，占婆國是以占族人為主體的國家（位於今天的越南中南部），後來在18至19世紀中葉，占婆國被越南滅國，大部分占族人要麼被越南人屠殺，要麼四處逃散。如今，占族比較集中的地區包括柬埔寨的[[磅湛省|磅湛省]]，越南的[[寧順省|寧順省]]和[[平順省|平順省]]等。在[[泰國|泰國]]、[[老撾|老撾]]、[[馬來西亞|馬來西亞]]以及中國的海南島也有占族人及其後裔分佈，其中在海南岛的占族人被中华人民共和国政府认定为回族。

== 歷史 ==
[[File:Cham_People_in_Vietnam_and_Cambodia.jpg|thumb]]
占族历史悠久（祖先来自[[汶莱|汶莱]]），曾經創造灿烂的文明。其地在[[汉代|汉代]]屬於[[日南郡|日南郡]]的[[象林县|象林县]]，隶属于[[交趾|交趾]]刺史部，是汉代疆域的最南端。当地土著又称“[[林邑|林邑]]”。[[东汉|东汉]]末年（192年），占人[[区连|区连]]杀死象林县令，从[[中国|中国]]独立，占据了日南郡的南部地区，以[[婆罗门教|婆罗门教]]为国教，建立占城国，与东汉以[[顺化|顺化]]为界。至[[唐代|唐代]]又改称“占婆”、“占城”，國力一度非常兴盛。到15世纪，逐渐衰落，被越南逐步蠶食，1832年越南[[阮朝|阮朝]][[明命帝|明命帝]]在顺城鎭[[改土归流|改土归流]]，占城完全灭亡。部分遺民仍在[[越南|越南]]南部和[[柬埔寨|柬埔寨]]生活，亦有部份族人迁至今天的[[马来西亚|马来西亚]]地区。

== 語言 ==
占族的語言[[占語|占語]]分幾種方言，均屬於[[南岛语系|南岛语系]]的[[马来-波利尼西亚语族|马来-波利尼西亚语族]]。

== 宗教 ==
占人早期多信仰[[印度教|印度教]]和[[佛教|佛教]]，占婆國後期，受[[阿拉伯|阿拉伯]]商人的影響，部分皈依[[伊斯蘭教|伊斯蘭教]]。有人说他们是受[[马六甲|马六甲]]影响而信教。他們不吃豬肉但飲酒，[[齋月|齋月]]也只齋戒三天。

據法國學者喬治·馬司培羅的說法，伊斯蘭教的傳入占婆，大約是[[中國|中國]]的[[宋代|宋代]]（960年至1279年）。而在占婆傳說當中，亦有[[阿拉|阿拉]]（Allah）於1000年至1036年「君臨都城」之事。

馬司培羅估計，伊斯蘭教之傳入占婆，有可能源於「高棉之回教徒為馬來人所化，因以傳布回教於安南」。

馬司培羅又指出，時至20世紀初，[[柬埔寨|柬埔寨]]的六萬占族人口，皆信奉伊斯蘭教；而[[越南|越南]]九萬占族人口，信奉伊斯蘭教的則佔三分之一，他們自稱為「占白尼」（Chams Beni）。「占」即占人，「白尼」即宗教子弟之意。<ref>馬司培羅《占婆史》第一章《土地及人民》，臺灣商務印書館中譯本，6-7頁。</ref>
