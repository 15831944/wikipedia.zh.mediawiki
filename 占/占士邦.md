'''詹姆士·龐德'''（{{lang-en|James Bond}}；代號'''007'''，俗稱鐵金-{}-剛）是一名由[[伊恩·佛萊明|伊恩·佛萊明]]創作的概念[[間諜|間諜]]。

小説原作者是英國作家[[伊恩·佛萊明|伊恩·佛萊明]]。在故事裡，龐德是[[英國|英國]]情報機構[[軍情六處|軍情六處]]的特務代號'''007'''，擁有殺人執-{}-照（可以除去任何妨礙行動的人的權力）。曾任[[皇家海軍後備隊|英國皇家海軍後備隊]][[中校_(英國皇家海軍)|中校]]。

詹姆士·龐德總是有美女相伴，那些女士被稱為[[邦女郎|邦女郎]]。

== 人物 ==

[[File:007_evolution.svg|thumb]]
{{Main|占士·邦}}

伊恩·佛萊明於1962年10月於《[[泰晤士報|泰晤士報]]》說詹姆士·龐德是一位真正特務的高度浪漫化版本，那人是[[第二次世界大戰|第二次世界大戰]]時[[加拿大|加拿大]]商人、資助[[盟軍|盟軍]]的{{le|威廉·史蒂芬生|William Stephenson}}。<ref>{{Cite web |url=https://www.theguardian.com/uk/2006/nov/11/film.jamesbond |title=Fact and fiction: how real spies see Bond |date=2006-11-11 |publisher=The Guardian}}</ref>

《賭城諜血》（''Casino Royale''，1954年黑白電視電影，主角[[巴里·尼爾遜|巴里·尼爾遜]]（Barry Nelson））、《-{zh-hans:皇家赌场;zh-hk:鐵金剛橫掃皇家賭場;zh-tw:皇家夜總會;}-》（''Casino Royale''，1967年喜劇電影，主角為[[大衛·尼文|大衛·尼文]]（David Niven））和《-{zh-cn:巡航导弹; zh-tw:巡弋飛彈; zh-hk:鐵金剛勇奪巡航導彈;}-》（''Never Say Never Again''，1983年，主角[[史恩·康納萊|史恩·康納萊]]（Sean Connery））等三齣詹姆士·龐德電影，由於並非由[[米高梅公司|米高梅公司]]及EON Production製作，因此並沒有列入詹姆士·龐德官方電影列表中。

== 其他資料 ==
在進入1990年代後，007系列電影逐漸受到許多世界知名企業的注目，開始與片商合作在影片中製造自家產品的曝光機會，而使得該系列電影變成[[置入性行銷|置入性行銷]]的著名範例。曾經投資贊助過影片拍攝的主要廠商包括[[Aston_Martin|Aston Martin]]、[[BMW|BMW]]、[[勞斯萊斯|勞斯萊斯]]、[[福特汽車|福特]]、[[Audi|Audi]]等汽車集團，電腦/通訊大廠[[新力易利信|-{zh-hans:索尼爱立信;zh-hk:新力愛力信;zh-tw:新力易利信;}-]]（Sony Ericsson，[[索尼|索尼]]与[[爱立信|爱立信]]合资品牌），另外如[[卡尔·瓦尔特运动枪有限公司|瓦尔特]]（Walther，[[德國|德國]]槍械製造商）與[[歐米茄|歐米茄]]（OMEGA，[[瑞士|瑞士]]鐘錶廠，[[斯沃琪集團|斯沃琪集團]]旗下）在該系列電影中都有非常高的曝光度。

不少其他國家的作品都把故事的特務角色以他作參考，不論是現代還是古裝。[[周星馳|周星馳]]的《[[國產凌凌漆|國產凌凌漆]]》（''From China With Love''或''From Beijing With Love''，英文戲名仿原版007电影''From Russia With Love''）和《[[大內密探零零發|大內密探零零發]]》的主角都或多或少影射詹姆士·龐德。
