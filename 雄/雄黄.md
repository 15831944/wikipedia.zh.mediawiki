'''雄黄'''，又称作'''石黄'''、'''黄金石'''、'''鸡冠石'''，是一种含[[硫|硫]]和[[砷|砷]]的矿石，质软、性脆，通常为粒状、紧密状块，或者粉末，[[条痕|条痕]]呈浅桔红色。主要成分是[[四硫化四砷|四硫化四砷]]（As<sub>4</sub>S<sub>4</sub>，佔90%以上）。雄黄主要产于低温热液矿床中，常与[[雌黄|雌黄]]（As<sub>2</sub>S<sub>3</sub>）、[[辉锑矿|辉锑矿]]、[[辰砂|辰砂]]共生；产于温泉沉积物和硫质[[火山喷气孔|火山喷气孔]]内沉积物的雄黄，则常与雌黄共生。不溶于水和[[盐酸|盐酸]]，可溶于[[硝酸|硝酸]]，溶液呈黄色。置于阳光下曝晒，会变为黄色的雌黄和[[砒霜|砷华]]，所以保存应避光以免受风化。雄黄要被[[煅烧|煅烧]]才会被氧化为剧毒成分[[三氧化二砷|三氧化二砷]]，即[[砒霜|砒霜]]。

== 用途 ==

砷礦物主要用於提煉元素砷、製造砷酸和砷的化合物，如[[砷酸鈣|砷酸鈣]]、[[砷酸鈉|砷酸鈉]]、[[砷酸鉛|砷酸鉛]]等。

在冶金工業中用於煉製[[砷合金|砷合金]]；[[砷鉛合金|砷鉛合金]]在軍事工業中用以製造子彈頭、軍用毒藥和煙火；[[砷銅合金|砷銅合金]]等用於製造雷達零件和汽車。

在輕工業中用以製造乳白色玻璃、玻璃脫色、浸洗羊毛、製革藥劑以及用於木材防腐。

在農業上用作殺蟲劑、除草劑、滅鼠藥等含砷農藥。

人工煉製的砷華藥物名稱為砒霜。

雄黃精礦粉可用於製造鞭炮、煙花。

雄黃加雙氧水生成砷酸，再加熱脫水，便脫氧生成三氧化二砷，即精製砒霜。

== 雄黄晶体结构 ==

雄黄属单斜晶系，[[晶体群|晶体群]]为P2(1)/n。结晶为桔红色，透明到半透明的雄黄晶体，其柱状晶体长短参差，粗细相伴，多方向生长。晶体呈金刚光泽，断口为树脂光泽。单晶呈细小的柱状、针状，但天然单晶较为少见。

== 文化 ==

[[漢字文化圈|漢字文化圈]]在每年農曆五月初五[[端午節|端午節]]有喝[[雄黃酒|雄黃酒]]以闢邪的傳統習慣。

== 中藥 ==

[[中醫|中醫]]認為雄黃性味辛、苦、平，歸心、肝、脾、胃、[[大腸經|大腸經]]。

含雄黃的中成藥：七珍丸、小兒化毒散、小兒至寶丸、小兒驚風散、小兒清熱片、牙痛一粒丸、牛黃至寶丸、牛黃抱龍丸、牛黃消炎片、牛黃清心丸、牛黃解毒丸（片）、牛黃鎮驚丸、六應丸、安宮牛黃丸（散）、紅靈散、醫癇丸、局方至寶散、阿魏化痞膏、純陽正氣丸、珠黃吹喉散、梅花點舌丸、紫金錠、暑症片、痧藥、升麻鱉甲湯。

== 毒性 ==

雄黄有毒，虽然毒性不像砒霜那么大。如果从一天口服大约10毫克雄黄，即可导致慢性砷中毒。中医常用药[[牛黄解毒片|牛黄解毒片]]每片（大片）含有50毫克雄黄，按药典规定一天服4～6片。<ref>[http://www.xys.org/xys/netters/Fang-Zhouzi/blog/niuhuangjiedupian.txt 常用中成药的真相——牛黄解毒片][[方舟子|方舟子]]，[[新语丝|新语丝]]，2008-12-17</ref>
