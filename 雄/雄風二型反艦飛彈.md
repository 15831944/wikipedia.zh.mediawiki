'''雄風二型'''（英語：Hsiung Feng II，縮寫：HF-2）飛彈是[[中華民國|中華民國]][[中山科學研究院|中山科學研究院]]以[[美國|美國]][[AGM-84魚叉反艦飛彈|魚叉反艦飛彈]]的功能為目標發展的[[反艦飛彈|反艦飛彈]]，於1988年2月29日正式公佈<ref>全球防衛雜誌第44期，1988年4月1日，ISSN 10103228</ref>。具有較好的電子反反制（ECCM）能力。除艦射型外，另發展出陸射型和空射型，是中華民國武器發展史上第一種可被戰機攜掛並發射的反艦飛彈。2000年時，[[中華民國海軍|中華民國海軍]]曾計畫以[[AGM-84魚叉反艦飛彈|RGM-84魚叉（Harpoon）反艦飛彈]]替換原裝設於[[成功級巡防艦|成功級巡防艦]]的雄風二型飛彈，但因預算被刪除而取消<ref>{{Cite web |url=http://www.newtaiwan.com.tw/bulletinview.jsp?period=247&bulletinid=7638 |title=替換雄二飛彈 違反國防自主 |access-date=2007-11-03 |archive-url=https://web.archive.org/web/20070317020906/http://www.newtaiwan.com.tw/bulletinview.jsp?period=247&bulletinid=7638 |archive-date=2007-03-17 |dead-url=yes }}</ref><ref name = "雄風二型反艦飛彈_2018"> {{ cite web | title = 雄風二型反艦飛彈 | url = http://www.ncsist.org.tw/csistdup/products/product.aspx?product_id=273&catalog=41 | publisher = 國家中山科學研究院 | language = zh-tw | accessdate = 2018-09-01 }} </ref>。

2001年，以雄風二型反艦飛彈為基礎並參考自美國[[戰斧巡弋飛彈|戰斧巡弋飛彈]]發展的攻陸[[巡弋飛彈|巡弋飛彈]]版本的[[雄風二E巡弋飛彈|雄風二E巡弋飛彈]]發展計畫被公開。

==發展歷史==
於1982年至1994年，[[韓光渭|韓光渭]]擔任中科院雄風飛彈計畫主持人長達十三年之久，被稱為「雄風飛彈之父」。雄風二型反艦飛彈，從艦射到空射到岸置等，各種衍生型飛彈，都是在韓光渭當計畫主持人時打造建立出來的<ref name = "學習的人生：韓光渭回憶錄_2010"> {{ cite web | title = 學習的人生：韓光渭回憶錄 | url = http://m.sanmin.com.tw/product/index/001313360 | publisher = 中研院 | author = 韓光渭 | language = zh-tw | date = 2010-06-01 }} </ref>。在雄風二型正式建案之前，中科院因中流計畫需求，在民國七十年以「介山計畫」之代號購入[[以色列|以色列]]貝特謝梅什發動機公司研製的{{link-he|梭烈4渦輪噴射發動機|שורק 4}}（Sorek 4）200具於介壽二廠組裝，中科院稱之為「S引擎」。中流計畫最後沒有完成研製，但S引擎卻成為雄風二型飛彈的動力裝置。

雄風二型的原型彈測試稱為「旭鋒演習」，原型測試到民國七十八年三月三十日之「旭鋒八號」才完成。作戰驗證測考稱為「順鋒演習」，在民國七十八年九月二十二日之「順鋒七號」演習完成驗證，<ref>張力訪問／紀錄，伍世文先生訪問記錄（南港：中央研究院近代史研究所，2018年7月），頁279。</ref>自民國八十年起開始量產。

==飛彈規格==
*氣動力結構
*固推加力器
*渦輪引擎
*慣性導航
*數位式彈內計算機
*自鍛射丸高爆彈頭
*無線電高度計
*主動雷達尋標器
*影像式紅外線尋標器

==飛彈結構==
*導控段
**中途導引：
***整合慣性導航、衛星定位及無線電高度計等定位及高度量測機制，引導飛彈循預設戰術轉折軌跡並掠海低飛近迫敵艦
**終端導引：
***複式尋標器總成包含主動雷達(RF Seeker)及影像紅外線(IIR Seeker)，標靶偵檢和鎖定資訊融合運用於終端導引期，充分地具備抵禦敵電子反制作為的功用
***整合慣性導航、衛星定位及尋標器等敵艦相對定位定向量測機制，引導彈體歸向及至撞擊敵艦
*彈頭段
*油箱段
*引擎段
*制動器段
*加力器段

==作戰性能==
掠海巡航，速度高次音速，接戰模式可採RBL／BOL射擊模式並以多彈STOT／DTOT執行飽和攻擊

==衍生型==
雄風二型飛彈除作為艦對艦飛彈攻擊武器之外，另有數種不同的衍生型式，作為岸基與空載之對面攻擊武器之用。雄風二型反艦飛彈正式型號為MGB-2B，分為陸射型與艦射型兩種，後期的空射型正式型號為MGB-2C。<ref name = "我是军事科技知识大王_2014"> {{ cite web | title = 我是军事科技知识大王 | url = https://books.google.com.tw/books?id=KBLWBQAAQBAJ&printsec=frontcover&hl=zh-TW#v=onepage&q&f=false | publisher = 青苹果数据中心, 2014 | author = 作者：李晓鸽编著 | language = zh-cn | date = 2014-12-15 }} </ref>

===艦射型===
第一種進入服役階段的是艦射型，首艘配備2座雙聯裝箱型發射器擔任測試任務的是[[陽字號|DDG-907富陽號驅逐艦]]。

在1990年代後，中華民國海軍水面艦艇除了[[濟陽級巡防艦|濟陽級巡防艦]]、[[基隆級驅逐艦|基隆級驅逐艦]]與[[海鷗級飛彈快艇|海鷗級飛彈快艇]]之外，皆陸續裝備二聯裝或四聯裝發射器。

===空射型===
中科院於1990年代初期發展空射型雄風二型飛彈，並計畫以AT-3／A-3與F-CK-1經國號戰機作為發射平臺。空射型飛彈於測試期間則以[[AT-3自強號高級教練／輕攻擊機|AT-3B自強號高級教練／輕攻擊機]]和[[AT-3自強號高級教練／輕攻擊機|A-3雷鳴式攻擊機]]各1架（前者編號0825，後者編號0902）進行攜掛和發射測試，後續0901號A-3原型機亦整合了發射能力。1996年台海飛彈危機時，這3架測試機與原型空射雄二飛彈曾肩負應擊戰備之任務。中科院曾聲稱升級後的[[經國號戰鬥機|經國號戰鬥機]]亦將具備攜掛和發射雄風二型飛彈的能力<ref>[http://www.libertytimes.com.tw/2003/new/sep/16/today-life1.htm ＩＤＦ掛載雄二飛彈 首度試射] {{webarchive|url=https://web.archive.org/web/20080303083315/http://www.libertytimes.com.tw/2003/new/sep/16/today-life1.htm |date=2008-03-03 }}</ref>，但最後中華民國空軍並沒有為經國號戰機採購國造反艦飛彈，制海主力仍是F-16A／B所配備的魚叉反艦飛彈。

===陸射型===
陸地發射型於1990年代早期完成發展階段，1990年代中期開始大量部署於海鋒大隊各固定陣地汰換原配備的[[雄風一型反艦飛彈|雄風一型反艦飛彈]]，標準接戰射程為80[[海浬|海浬]]（148.16公里）。

部署於海鋒大隊的雄風二型反艦飛彈除安置於強化[[掩體|掩體]]作為固定反艦火力之外，機動發射型於2002年[[花蓮縣|花蓮縣]][[賀田山|賀田山]]固海營區首次試射成功，並於2005年進入[[中華民國海軍海鋒大隊|中華民國海軍海鋒大隊]]機動中隊服役。

媒體報導聲稱，海軍正進行雄風二型陸射型性能提升計畫，升級後的陸射型雄二將具備250公里射程；飛行可設計8個轉向點迴避假想敵的偵測與攔截。

===潛射型===
潛射型於1990年代中期開始計畫發展，但除了名詞存在外並無其它間接資訊揭露細節，而[[中華民國海軍|中華民國海軍]]向美國採購潛射魚叉反艦飛彈（UGM-84）配備供[[劍龍級潛艇|劍龍級潛艇]]運用，並在2013年完成實彈射擊驗收<ref>[http://news.ltn.com.tw/news/politics/paper/822755 海軍潛射魚叉飛彈試射 正中標靶]</ref>；短時間不太可能進行同屬性武器的研製方案，故目前無實際進行的研發證據。
