'''雄風三型反艦飛彈'''（Hsiung Feng III），簡稱「雄三」，是由[[中華民國|中華民國]][[中山科學研究院|中山科學研究院]]自行研發的[[超音速|超音速]][[反艦飛彈|反艦飛彈]]，於2007年10月10日[[中華民國國慶|中華民國國慶]]國防表演時正式展示。

==簡介==
雄三的導引及尋標系統，目前推測應該是由[[雄風二型反艦飛彈|雄風二型反艦飛彈]]的X波段單脈衝平面陣列搜索天線改良而來，以改良的積體電路設計與射控程式提升反應速度，因應超音速導引需求。由於彈體超音速飛行的摩擦熱問題目前未有合理的解決方案，雄二的紅外線導引系統並未安裝；雄三的有效射程目前軍方尚未公佈資料，[[臺海安全研析中心|臺海安全研析中心]]主任[[梅復興|梅復興]]的文章推測为130公里<ref name=tdr1>梅復興， "Supersonic ASCM Equips Frigate," Taiwan Defense Review, August 10, 2006.</ref>，台湾媒体在雄三公開初期一般报道認為它有和雄二相等射程的性能，或許有150公里；但在雄三量產後且因誤射事件聲名大噪之後，報章媒體多度聲稱雄三射程可達300公里、甚至400公里<ref>{{Cite web |url=https://www.twreporter.org/a/opinion-accidentally-fires-missile |title=陳彥升／雄三飛彈誤射背後的漏洞：一個火箭科學家的觀點 |access-date=2016-07-31 |archive-url=https://web.archive.org/web/20171201035522/https://www.twreporter.org/a/opinion-accidentally-fires-missile |archive-date=2017-12-01 |dead-url=no }}</ref>，但所有數據當前都沒有更明確的權威性說明實證。<ref name="雄三飛彈-航艦致命武器"/>有媒體報導增程型雄風三型已於九鵬基地進行測試，射程達400公里，符合軍方預期。<ref>{{cite news|url=http://news.ltn.com.tw/news/politics/breakingnews/2265692|title=雄三飛彈射程增至400公里 中科院上週完成測試|publisher=自由時報|date=2017-11-27|accessdate=2017-11-27|archive-url=https://web.archive.org/web/20171127082255/http://news.ltn.com.tw/news/politics/breakingnews/2265692|archive-date=2017-11-27|dead-url=no}}</ref>同时梅復興的文章中提到，由於衝壓引擎啟動後需要足夠速度才可運作，初始加速以及姿勢控制等過程，雄風三型飛彈可能至少需要4海浬的距離才可完成作戰開機手續。

目標對於高速的次音速反艦飛彈如[[魚叉飛彈|魚叉飛彈]](0.85 Mach)大約有兩分鐘左右的反應時間。但對於雄三這樣的飛彈，反應時間大幅降低到約只有35秒。

== 發展歷史 ==
[[中山科學研究院|中山科學研究院]]對[[衝壓發動機|衝壓發動機]]的研究要追溯回1970年代末開發[[天弓飛彈|天弓飛彈]]時在飛彈動力上的重複投資爭議，根據《[[全球防衛雜誌|全球防衛雜誌]]》的報導，雄風三型飛彈的技術來源可追溯至美國{{tsl|en|Vought|沃特公司}}，由沃特公司研發的[[冲压发动机|冲压发动机]]（Air Launched Low Volume Ramjet，ALVRJ）和其對應的STM[[导弹|导弹]]（Supersonic Tactical Missile），ASALM在1976年完成了開發<ref>[http://ammtiac.alionscience.com/ammt/iacdocs.do?AM035534 ALVRJ Flight Test Program Summary] {{webarchive|url=https://web.archive.org/web/20140203132739/http://ammtiac.alionscience.com/ammt/iacdocs.do?AM035534 |date=2014-02-03 }}</ref>，不過之后ASALM和衍生產品數次参與的美國軍方競標均失敗，最終於1980年代由中山科學研究院購得此技術並研發天弓二型冲壓飛彈<ref>{{Cite web| url =http://www.diic.com.tw/comment/9611/961101Hsiungfeng3.pdf| title =雄三研發秘辛| work =全球防衛雜誌| publisher =| date =| accessdate =2013-02-18| deadurl =yes| archiveurl =https://web.archive.org/web/20090902085335/http://www.diic.com.tw/comment/9611/961101Hsiungfeng3.pdf| archivedate =2009-09-02}}</ref>。

不過天弓計畫負責人[[陳傳鎬|陳傳鎬]]於2014年出版的自傳中否定了這項爆料，天弓計畫在初期階段確實是有兩項推進設計項目同時進行獨立研發，分別為沖壓動力與固體火箭，但是當時並沒有向沃特收購ALVRJ之案件，所以天弓系列與此案牽連性相當低；而最後衝壓引擎防空飛彈開發案也在技術門檻難以突破的問題下於1984年起擱置技術研發，天弓一型飛彈則採用技術難度較低之固態火箭推進。<ref>{{cite book|title =天弓：我們的向世界之最挑戰 |author1= 陳傳鎬 |publisher=科技圖書|origyear=2014|date=2014-03-20 | ISBN=978-957-655-522-0 |ref={{sfnref|陳傳鎬|2014}} |pages = 416頁|location=臺灣}}</ref>

雖然1980年代開發衝壓引擎防空飛彈的努力並無成果，但中科院並沒放棄在這技術的投資，而是引入了更多新設備轉向其它領域開發。中科院在引進技術與持續研發並行下，經過數十年的技術累積，開發出小型化的沖壓反艦飛彈。

雄風三型的開發計畫則因衝壓引擎的長期實驗從1990年代末期資訊逐漸明朗，最早外界得知的資訊是1990年「[[雄風二型反艦飛彈|雄二計畫]]」告一段落後，中科院將原先開發團隊與研發衝壓引擎的「擎天計畫室」整合為雄三開發小組；當時外界預測新一代的反艦飛彈將會以衝壓發動機作為推進動力，1999年中科院30週年院慶時展示的擎天衝壓載台與測試照片也證明此點，隨後好幾年的時間公開資訊沉寂，內部消息指稱雄三原型於2001年7月開始測試，約在2004年底到2005年初於成功號巡防艦定保時安裝上艦<ref name=tdr1 />。直到2006年，由民間軍事愛好者拍攝到成功級巡防艦上的大型飛彈發射箱後，雄三的開發再度引發關注。

2007年10月10日，中華民國96年[[雙十國慶|雙十國慶]]大會兼國軍「同慶操演」地面部隊展示中，正式公開雄風三型（HF-3）超音速反艦飛彈實體。雄風三型的計畫代名為「追風」，{{fact|原始設計採用整合式衝壓發動機，即助推器（SRB）與續航衝壓發動機整合一體裝置於彈體後段，節省飛彈的長度與體積。然而到量產時為了安裝在成功級巡防艦的上層甲板，由於長度限制，最後仍放棄整合式燃燒室設計，採取可拋棄助推火箭設計}}。發射時，由2具側掛的火箭助推引擎和飛彈主體的整合式四具衝壓引擎進氣道將飛彈送至高空，並加速到一定速度後點燃衝壓發動機進行後續航程，大幅提升飛行速度。

雄風三型的量產與部署計畫稱為「追風」，總預算達118億9300多萬新台幣，從2007到2013年編列執行，到2013年度生產120枚，第一階段完成的雄三飛彈安裝在8艘[[成功級巡防艦|成功級巡防艦]]和7艘[[錦江級巡邏艦|錦江級巡邏艦]]上<ref>[http://www.mdc.idv.tw/mdc/information/hf3.htm 雄風三型超音速反艦飛彈] {{Wayback|url=http://www.mdc.idv.tw/mdc/information/hf3.htm |date=20130217202225 }}{{vc|time=2016-07-05}}</ref>，船載板本以二連裝發射器為一組，並以對稱方式配載，每艘船上裝載4發雄三飛彈<ref name="雄三飛彈-航艦致命武器">{{Cite web |url=http://tw.news.yahoo.com/%E9%9B%84%E4%B8%89%E9%A3%9B%E5%BD%88-%E8%88%AA%E8%89%A6%E8%87%B4%E5%91%BD%E6%AD%A6%E5%99%A8-022410863.html |title=雄三飛彈-航艦致命武器 |access-date=2019-02-08 |archive-url=https://web.archive.org/web/20160317022239/https://tw.news.yahoo.com/%E9%9B%84%E4%B8%89%E9%A3%9B%E5%BD%88-%E8%88%AA%E8%89%A6%E8%87%B4%E5%91%BD%E6%AD%A6%E5%99%A8-022410863.html |archive-date=2016-03-17 |dead-url=no }}</ref>，[[沱江級巡邏艦|沱江級巡邏艦]]設計中曾嘗試在單一發射架上裝載4枚飛彈，使該艦可搭載16枚反艦飛彈，但因配重平衡問題未解，因此目前仍維持2發配載方式；車載的機動岸射版也在2013年台北國際航太暨國防工業展公開，現場說明官表示機動型已配發至海鋒大隊服役，發射拖車組重32噸，在拖板車頭安置了30kw發電機供給發射架操作動力。

=== 完全戰備 ===
2014年10月18日，中華民國海軍進行該年度唯一一次雄三超音速反艦飛彈的射擊驗證。2艘驗證的成功級軍艦，在台東外海發現目標後，依作戰程序各發射1枚、共2枚雄三飛彈，均成功的擊中百公里外的靶船，也代表雄三飛彈完成海軍所有的作戰測評，成為海軍作戰艦艇的作戰裝備，目前部署在海軍8艘成功級飛彈巡防艦，7艘錦江級巡邏艦上，每艘戰艦部署4枚雄三超音速飛彈，正式進入完全戰備的時代。<ref name="航母殺手現身 雄三反艦飛彈試射成功_2018">{{ cite web | title = 航母殺手現身 雄三反艦飛彈試射成功 | url = http://www.ncsist.org.tw/csistdup/products/productRelated.aspx?id=147 | publisher = 國家中山科學研究院 | language = zh-tw | accessdate = 2018-09-04 | archive-url = https://web.archive.org/web/20180904084838/http://www.ncsist.org.tw/csistdup/products/productRelated.aspx?id=147 | archive-date = 2018-09-04 | dead-url = no }}</ref>

2017年11月中旬，台灣媒體報導中科院在九鵬基地進行增程型雄三的作戰測試，射程已達400公里，總統同意增程雄三的量產計畫，數量至少為60枚，未來不但可以採陸上固定陣地與車輛機動發射方式部署，至於未來是否可在海軍新一代主力艦及新一代巡防艦上部署，將要視海軍需求以及新式軍艦載台能否承載條件再做考量。中科院發聲明，表示有關報載「增程型」雄三反艦飛彈於九鵬基地進行飛彈測試一事，「為媒體自行臆測報導，內容非屬事實。」<ref>{{Cite web |url=http://news.ltn.com.tw/news/focus/paper/1155662 |title=中科院執行「磐龍計畫」 雄三射程增至400公里 |access-date=2018-05-22 |archive-url=https://web.archive.org/web/20180523172619/http://news.ltn.com.tw/news/focus/paper/1155662 |archive-date=2018-05-23 |dead-url=no }}</ref><ref>{{Cite web |url=http://www.chinatimes.com/realtimenews/20171128002688-260417 |title=射程400公里？傳增程型雄三性能強原型兩倍！ |access-date=2018-05-22 |archive-url=https://web.archive.org/web/20180523100155/http://www.chinatimes.com/realtimenews/20171128002688-260417 |archive-date=2018-05-23 |dead-url=no }}</ref>

== 誤射事件 ==
{{main|2016年雄風三型反艦飛彈誤射事件}}
2016年7月1日，中華民國海軍[[錦江級巡邏艦|錦江級巡邏艦]]金江艦（PGG-610）進行雄風三型飛彈系統檢查測試時，發生一枚雄三飛彈誤射事件。國防部在當天舉行的記者會上證實，該枚飛彈貫穿高雄籍翔利昇漁船船身，造成船長黃文忠不幸死亡、三名船員受傷<ref>{{cite news|author=尹俞歡|title=林全證實 海軍飛彈誤射擊中漁船造成一死三傷|url=http://www.storm.mg/article/136788|accessdate=2016-07-01|newspaper=[[風傳媒|風傳媒]]|date=2016-07-01|archive-url=https://web.archive.org/web/20160702140921/http://www.storm.mg/article/136788|archive-date=2016-07-02|dead-url=no}}</ref>。

該枚飛彈從[[左營港|左營港]]內射出，飛至40海浬外（約74公里）、[[澎湖縣|澎湖群島]]東南方海域的預設目標點搜敵，該海域作業中民用船隻因此遭到鎖定。飛彈在高速貫穿漁船後，因為引信未能作動，繼續飛行約1.9浬後墜海，過程約兩分多鐘。事發後，國軍隨即派出S-70C直升機、錦江級巡邏艦、諾克斯級巡防艦、飛彈快艇等六艘艦艇在附近海域搜索。海軍司令部參謀長梅家樹說，初步判斷意外發生原因應是「人員操作未依據正常程序」<ref>{{cite news|author=|title=海軍測試雄三飛彈系統 官兵不慎誤射|url=http://news.pts.org.tw/article/327481|accessdate=2016-07-01|newspaper=[[公視新聞|公視新聞]]|date=2016-07-01|archive-url=https://web.archive.org/web/20160702152146/http://news.pts.org.tw/article/327481|archive-date=2016-07-02|dead-url=no}}</ref><ref>{{cite news|author=涂鉅旻、黃旭磊、羅添斌|title=疑點重重》海軍誤射雄三 軍官不在場|url=http://news.ltn.com.tw/news/focus/paper/1006916|accessdate=2016-07-01|newspaper=[[自由時報|自由時報]]|date=2016-07-01|archive-url=https://web.archive.org/web/20160702160038/http://news.ltn.com.tw/news/focus/paper/1006916|archive-date=2016-07-02|dead-url=no}}</ref>。
