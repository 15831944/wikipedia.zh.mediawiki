《'''逃避雖可恥-{}-但有用'''》（{{lang-ja|逃げるは恥だが役に立つ}}、{{lang-hu|[[:wikt:hu:szégyen_a_futás,_de_hasznos|Szégyen a futás, de hasznos]]}}，发音 {{IPA|ˈse̝ːɟɛnɒ ˌfutɑ̈ːʃ dɛ ˈhɒsnoʃ}}）為[[海野綱彌|海野綱彌]]創作的[[日本漫畫|日本漫畫]]作品。首篇漫畫於[[講談社|講談社]]《[[Kiss_(雜誌)|Kiss]]》2012年的第22期連載<ref>{{Cite web |url = http://blogs.dion.ne.jp/0809_umino_273/archives/10979765.html |script-title = ja:『逃げるは恥だが役に立つ』第1話 |publisher=海野つなみ*Information |date=2012-11-10 |accessdate=2013-08-10 |language=ja}}</ref><ref>{{Cite web |url=http://natalie.mu/comic/news/79697 |script-title=ja:海野つなみ新連載「逃げるは恥だが役に立つ」Kissで開幕 |publisher=[[ナタリー_(ニュースサイト)|コミックナタリー]] |date=2012-11-09 |accessdate=2013-08-10|language=ja}}</ref>，後於2013年5月號月刊連載，自6月號起固定連載，於2017年2月號結束連載。單行本全9冊，最終集已於2017年3月出版。本作的日文略稱為「{{lang|ja|逃げ恥}}」。

本作為海野綱彌繼《[[回轉銀河|回轉銀河]]》之後以契約結婚為主題的漫畫作品。原作標題以[[日文|日文]]及[[匈牙利文|匈牙利文]]對照表示，其中匈牙利文是一句諺語，日文是其大意的翻译。<ref>{{lang|ja|後書き 海野つなみ『逃げるは恥だが役に立つ』 第1巻}} p.164 - p.165 {{ja}}</ref><ref>{{Cite web |url = http://blogs.dion.ne.jp/0809_umino_273/archives/11014777.html |script-title=ja:『逃げるは恥だが役に立つ』第2話 |publisher=海野つなみ*Information |date=2012-12-10 |accessdate=2013-08-10|language=ja}}</ref>

2015年獲得第39回[[講談社漫畫賞|講談社漫畫賞]]少女漫畫類大獎，2016年10月改編為電視劇。

== 劇情概要 ==
研究所[[臨床心理學|臨床心理學]]系畢業的女主角森山實栗求職接連碰壁，無計可施之下，經由父親的介紹，到父親因公認識的IT工程師津崎平匡家擔任派遣幫傭。津崎平匡是個認真的IT公司上班族，平時絕不流露出自己的真感情，但在她坦率真誠的態度之下，心中拒人之外的那道高牆逐漸崩塌，對她逐漸產生好感。而實栗也對平匡提出「契約結婚」，兩人的[[同居|同居]]生活就此開始…
