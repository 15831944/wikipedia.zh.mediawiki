'''逃出香港'''（{{lang-en|Freeing Hong Kong}}）是[[香港|香港]]第一間[[真人|真人]][[密室逃脱|逃脫密室]]遊戲公司，玩家必須在45分鐘內，解開密室內的謎題及機關，才能逃出密室。玩家人數由2至6人不等，根據過往經驗，只有約20至25%的玩家成功逃出。逃出香港為本地掀起一股「逃出熱」，現在香港的分店共有3間，分別位於旺角以及銅鑼灣，在[[澳門|澳門]]、[[新加坡|新加坡]]和[[意大利|意大利]]都有分店。

== 意念 ==
逃出香港以創意及真人體驗遊戲為開業理念，致力讓玩家逃離一成不變的職場生活，用謎題及機關激活玩家的生活。

== 遊戲模式 ==
逃出香港除了基本的密室逃脫以外，還有一些故事性較強的密室，在進入密室之前也會有相關的道具分發，這些道具將有助完成密室裡的特定機關。

'''RPG[[角色扮演遊戲|角色扮演遊戲]]：'''這是真人互動遊戲，故事性最強，更有演員扮演故事內的角色，他們會協助帶出故事及提供線索給玩家。玩家在這裡扮演著故事裡一個特定的角色，每位玩家都有各自不同的任務需要解開，更有不同結局。例子是觀塘店的聖杯爭霸戰。

'''闖關式鬼屋'''：逃出香港曾在2013年9月至11月，推出期間限定「逃出喪屍城」(Z-CITY)。它與一般鬼屋不同，除了一般的恐怖元素之外，更需要玩家運用解迷智慧，闖過不同關卡，途中更要使用在入口所獲得的武器，擋出喪屍的進攻，才能成功逃出。

== 創新科技 ==
逃出香港使用了3D立體影像技術及[[全息摄影|全息摄影]]（HOLOGRAM憑空出現畫面的視覺科技）等元素，增強玩家在密室內的視覺衝擊。

== 訂位方式 ==
逃出香港採取網上訂位的方式，訂位之後系統會發出確認電郵到玩家的信箱。如果訂位時未達到房間容納人數的上限，玩家有機會會與其他人一起組隊參與密室遊戲。

== 營業方向 ==
逃出香港正積極擴展海外業務，最近他們以特許經營方式在加拿大、新加坡及意大利。未來計劃在[[英國|英國]]、[[澳洲|澳洲]]及台灣發展其業務。

== 協同合作 ==
逃出香港舉辦密室設計比賽，讓玩家自行設計密室，勝出的作品將會被落實在店舖內。元朗分店的所有密室都是這個比賽的得獎作品。
