《'''奈奈與薰的SM日記'''》（{{lang-ja|ナナとカオル}}）是日本漫畫家{{link-ja|甘詰留太|甘詰留太}}的青年漫畫作品，簡稱「{{lang|ja|ナナカオ}}」。{{link-ja|ヤングアニマル嵐|ヤングアニマル嵐}}（白泉社）2008年No.2開始連載。之後移籍至同出版社的[[ヤングアニマル|ヤングアニマル]]，2011年時改編為真人電影。描寫奈奈與薰的高中3年級的暑假「奈奈與薰的SM日記 夏日篇（Black Label）」於2010年No.8開始連載。

==故事==
全能的女主角千草奈奈與無用的男主角杉村薫，是同一棟大廈的鄰居兼[[青梅竹馬|青梅竹馬]]。薫整天沉溺在[[SM|SM]]妄想。在某一天，奈奈不小心穿上了[[綁縛|綁縛]]的SM服裝。自此奈奈與薫之間「放鬆」的秘密就此開始。

===主角===
;千草 奈奈（({{lang|ja|ちぐさ なな}}) 簡稱「奈奈」，[[聲優|聲優]]：[[水原薫|水原薫]]）
:17歲，櫻溝高中二年級學生。本作的女主角。
:班級為成績優異的A班，是一位美少女，學生會的副會長並為田徑部的王牌，目標是考上[[東京大學|東京大學]]並成為律師。受到同學及後輩的景仰，有著強烈的正義感，缺點是不懂得拒絕別人的要求，並無形中累積壓力。
:名字的來源是{{link-ja|千草忠夫|千草忠夫}}<ref name="multiple">{{lang|ja|ヤングアニマル嵐}}2012年No.2、P44 - 46。</ref>。

;杉村 薰（({{lang|ja|すぎむら かおる}}) 簡稱「薰」，聲優：[[間島淳司|間島淳司]]）
:17歲，櫻溝高校二年級學生。本作的男主角。
:班級為成績最差的F班，興趣是SM的妄想，綽號「噁心村」，很容易緊張，擅長裁縫與料理。
: 名字的來源是[[杉村春也|杉村春也]]<ref name="multiple"/>。

;館 涼子（({{lang|ja|たち りょうこ}})，聲優：[[高森奈津美|高森奈津美]]）
:17歲，白鳥女子學院高等部二年級，是薰的夥伴，皮膚淺黑色，第一人稱為「僕({{lang|ja|ボク}})」為田徑部成員，主攻[[七項全能|七項全能]]，在以前曾跑輸奈奈。
:一開始對於SM採取批判性的思考，加入「放鬆」的活動之後其想法改觀。
: 名字的來源是{{link-ja|館淳一|館淳一}}<ref name="multiple"/>。

: 櫻溝高中2年生。學生會的會計。是奈奈的好友並同屬於A班。喜歡矢神浩史，稱男主角「噁心村」並完全看不起他。
: 名字的來源是{{link-ja|睦月影郎|睦月影郎}}<ref name="multiple"/>。

: 櫻溝高中2年生。學生會的會長。喜歡著女主角奈奈，有一個姊姊。
: 對於奈奈與薰的青梅竹馬關係表示厭惡，同時輕視著薰。
: 名字的來源是[[矢神朧|矢神朧]]<ref name="multiple"/>。

: 發現了男女主角間的關係，而對弟弟以「完全沒有希望」({{lang|ja|脈は殆ど無い}})勸他不必苦苦追求。
; 堂本(（{{lang|ja|どうもと}}），聲優： [[内海靖明|内海靖明]]）
: 櫻溝高中2年生。薰的損友。臉上有著[[雀斑|雀斑]]。
: 名字的來源是[[堂本烈|堂本烈]]<ref name="multiple"/>。
; 星野(（{{lang|ja|ほしの}}），聲優： [[松本忍|松本忍]]）
: 櫻溝高中2年生。薰的損友。體型肥胖。
: 名字的來源是{{lang|ja|[[星野ぴあす|星野ぴあす]]}}<ref name="multiple"/>。
; 川上老師，聲優：  [[山川琴美|山川琴美]]
: 櫻溝高中教師。奈奈的班級導師。進路相談時、勸奈奈要多「放鬆」自己，對奈奈之後的影響頗大。

;橘 満子(（{{lang|ja|たちばな みつこ}}），聲優：[[原田瞳|原田瞳]]）
:薰常去的性玩具商店「Fun Love」的店長。年齡約30歲。是一位美人、對於SM Play十分熟悉。 東京大學文學部畢業、並専攻日本史（專門於明治以後的文化風俗）。實際上是更科的SM夥伴，穿有乳環。
:名字的來源是{{link-ja|橘真兒|橘真児}}<ref name="multiple"/>。

:SM小說作家，同時也是薰所尊敬的作家，他的著作也是薰對「放鬆」的參考。實際上是橘的的SM夥伴，年齡約50歲。
