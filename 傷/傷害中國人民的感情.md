'''傷害中國人民的感情'''，是[[中华人民共和国政府|中华人民共和国政府]]使用的一種用語，用以表達对外国[[政府|政府]]在[[台湾|台湾]]、[[西藏|西藏]]、[[新疆|新疆]]等问题上举动的不满。另外，中國政府或一些中國留學生在反對海外人士或[[持不同政見者|持不同政見者]]針對中國的批評時亦會使用該用語<ref>{{cite news |language = zh-cn |author = 周贺 |url = http://politics.people.com.cn/GB/1026/4174516.html |title = 李肇星:日本领导人不应再做伤害中国人民感情的事 |publisher = [[人民網|人民網]] |date =2006-03-07}}</ref><ref name ="Bandurski2016"/><ref name="GW01">{{cite news|author1=安静|title=中国留学生“玻璃心”缘何而来？|url=http://www.dw.com/zh/%E4%B8%AD%E5%9B%BD%E7%95%99%E5%AD%A6%E7%94%9F%E7%8E%BB%E7%92%83%E5%BF%83%E7%BC%98%E4%BD%95%E8%80%8C%E6%9D%A5/a-40428293?&zhongwen=simp|agency=[[德國之聲|德國之聲]]|date=2017-09-09}}</ref>。

== 研究及數據 ==
[[File:人民日報「傷害中國人民的感情」次數.png|thumb]]
[[香港大學|香港大學]]中國傳媒研究計劃的[[班志远|班志远]]（David Bandurski）曾進行一項研究，從中國官媒《[[人民日報|人民日報]]》（由1959年至2015年）中選取了143篇出現「傷害中國人民的感情」句子的文章。以國家來看，[[日本|日本]]是被指責傷害感情次數最多的國家，達51次。[[美國|美國]]排名第二，達35次。以議題來看，與[[台灣|台灣]]有關的事件共有28次，其次則是[[西藏問題|西藏問題]]，共有12次<ref name ="Bandurski2016">{{cite web|url=http://cmp.hku.hk/2016/01/29/39625/|title=Hurting the feelings of the “Zhao family”|publisher=China Media Project|author = David Bandurski|date=2016-01-29}}</ref>。

《[[時代雜誌|時代雜誌]]》有文章指出，以非正式統計，單純依靠1946年至2006年；以[[中国大陆|中国大陆]]官方立場發言的傳媒「《[[人民日報|人民日報]]》電子版」的數據庫進行搜索，結果得出中国大陆官方發表提出「傷害了中國人民的感情」的文章有一百餘篇<ref>{{cite news |url = http://world.time.com/2008/12/11/hurt-feelings-blame-deng-xiaoping/ |title = Hurt Feelings? Blame Deng Xiaoping |accessdate = 2013-05-30 |date = 2008-11-11 |language = en}}</ref>。《[[環球網|環球網]]》有文章指出，「中國人民的感情」、「傷害」為關鍵詞檢索《人民日報》1946年5月15日至2015年5月1日發佈內容，出現相關文章237篇，相關詞組240次，共涉及29個國家和地區。其中印度9次，法國16次，美國62次，日本96次<ref>{{cite news |language = zh-cn |author = 6536号档案 |url = http://history.huanqiu.com/relation/2015-06/6606997.html |title = 揭秘：是谁“伤害了中国人民的感情”？ |publisher = [[环球网|环球网]] |deadurl = yes |archiveurl = https://web.archive.org/web/20160529140849/http://history.huanqiu.com/relation/2015-06/6606997.html |archivedate = 2016年5月29日 |df =  }}</ref>。

[[臺灣|臺灣]][[中央研究院|中央研究院]]社會學研究所副研究員[[汪宏倫|汪宏倫]]根據《人民日報》資料庫進行整理，發現從1949年至2013年《人民日報》共出現了319次「傷害中國人民的感情」。<ref name="whl">{{cite journal|author=汪宏倫|year=2014|issue=19|journal=文化研究|title=理解當代中國民族主義：制度、情感結構與認識框架|pages=189-250}}</ref>。

2016年8月，[[澳大利亞|澳大利亞]][[雪梨|雪梨]]羅伊國際政策研究所亞洲項目主任夏美林（Merriden Varrall）在《[[紐約時報|紐約時報]]》上發表標題為《中國留學生威脅澳大利亞大學開放性》的評論文章，文中描述了一個「令人不安的趨勢」，「15萬中國留學生中有很多正把支持北京的立場帶入教室，扼殺辯論和開放性。」她認為：「中國學生在自我審查或監視、報告同學時，不一定是因為中國政府的壓力。而是很多中國學生認為，公開反對官方認同的觀點是不合適的……因為很多中國學生已經把認同官方觀點的需要內化了，所以保持澳大利亞自由開放辯論的標准仍將是一個頗為棘手的挑戰。澳大利亞的大學可以先從直面這個問題著手。」<ref name="GW01"/>

2016年9月8日，夏美林在《[[衛報|衛報]]》發表評論文章。文章分析中國學生為何會如此「玻璃心」、經常使用「傷害中國人民感情」這種說法。她又認為如果澳大利亞政府的政策在國際上受到批評，澳大利亞人民不會有類似中國人民這種「受傷」的感覺<ref name="GW01"/>。

== 批评 ==
《麦克拉奇报》北京办公室主任Tim Johnson評論說，「這句陈腔滥调被中共当局惯用以获同情並批评他人」。<ref>{{cite news |language = zh-cn |author = 陈迈克 |url = http://mag.epochtimes.com/gb/107/5854.htm |title = 伤害中国人民的感情 |publisher = [[新紀元周刊|新紀元周刊]] 52期 |date =  }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

政治立场为[[自由主义|自由主义]]的《[[經濟學人|經濟學人]]》對此評價道，公眾憤怒往往是非常好用的工具，令中國拋開「不干涉其他國家內政」的外交原則，例如中方常常抱怨日本政客參拜[[靖國神社|靖國神社]]。但另一方面共產黨甚少願意讓民眾表達自己的感情，其實是擔心民意爆發會挑戰到黨中央的控制能力。<ref> {{cite news|url = http://www.economist.com/news/china/21690097-poor-china-so-vast-and-so-sensitive-world-hurt| title = A world of hurt Poor China: so vast and so sensitive |accessdate = 2016-02-06 |date = 2016-02-06 |language = en}}</ref>

=== 欧洲 ===
