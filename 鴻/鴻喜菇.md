'''鴻喜菇'''（學名：{{lang|la|''Hypsizygus marmoreus''}}）又稱[[真姬菇|真姬菇]]、[[白玉菇|白玉菇]]、[[班玉蕈|班玉蕈]]及'''蟹味菇'''（學名：''{{lang |la |Hypsizygus tessellatus}}''），是一種原生於[[东亚|东亚]]溫帶以北、現時主要從日本引進的高經濟價值[[食用蕈|食用蕈]]。本物種目前已於[[温带|温带]]的歐洲、北美洲及澳大利亞種植作食用。在自然界，這種[[口蘑科|口蘑科]][[玉蕈屬|玉蕈屬]]的[[伞菌目|伞菌目]]物種生長於樹林裡。由於常發現於[[山毛榉科|山毛榉科]]物種之上，所以在英語俗稱作「山毛榉蘑菇」（Beech Mushroom）<ref name="mushroom">{{cite web |url=https://www.mushroommatter.com/zh-CN/index.php/details/item/145-shanghai-city-of-china-up-to-date-mushroom-quotation |title=中國上海市：最新的蘑菇報價 |access-date=2019-01-14 |language=zh-hans }}</ref>。蟹味菇在食用菇當中比較小型和瘦，但在全世界很多國家都很及歡迎<ref>"Brown Beech Mushroom - Hypsizygus Tessellatus." Brown Beech Mushroom - Hypsizygus Tessellatus. N.p., n.d. Web. 06 Dec. 2014.</ref>。

==變種==
本物種已知有兩個變種：
* {{link-ja|ブナシメジ|棕毛榉蘑菇}}（''[[Hypsizigus_tessellatus|Hypsizigus tessellatus]]''）
* {{link-ja|ブナピー|白毛榉蘑菇}}：從棕毛榉蘑菇揀選，註冊商標，由日本{{link-ja|ホクト|Hokto}}企業擁有。

==特徵==
在秋季，生長在如[[山毛榉科|山毛櫸科]]等闊葉植物的倒腐枯木上，其生長的習性像[[金針菇|金針菇]]一樣，常有好幾個[[菌柄|菌柄]]叢聚一起生長，呈叢生（clustered）,每一叢約15-50個[[子實體_(真菌)|子實體]]。 有时也會單獨生長且會有較大的[[菌傘|菌傘]]。剛長出的[[子實體_(真菌)|子實體]][[菌盖|菌傘]]呈半圓之後遂時間慢慢變成扁平狀表面呈白色或紅褐色直徑4-15cm，上面有深色大理石條紋。[[子实层|子實層]]近白色，[[菌柄|菌柄]]的長度為3-10cm，從菌傘中心開始生長，直徑為0.3-0.6cm。 [[孢子|孢子]]呈寬卵形或近圓形，在顯微鏡下半透明，堆疊時呈白色。

當初和''[[H._ulmarius|H. ulmarius]]'' 被視為同種，但其實是不同種可以由菌傘上的大理石紋路判別。 

== 名稱誤用 ==
「鴻喜」這個名稱是引進栽培的業者從日語中的「ホンシメジ」（Honshimeji）的前兩個音節音譯而來。不過，市場上看到的鴻喜菇，和日語中的ホンシメジ未必是同一個物種。ホンシメジ的學名是''[[玉蕈离褶伞|Lyophyllum shimeji]]'' (Kawam.) Hongo，中文稱為「[[玉蕈离褶伞|玉蕈褶離傘]]」，是一種長在土壤中的樹木共生菌，雖然可食美味但不容易人工培養，但是它的俗名長期借給鴻喜菇這種市面上常見且受歡迎的栽培菇類，日語稱為ブナシメジ（Bunashimeji）。日本市面上簡稱為シメジ（Shimeji），通常也是指這種可栽培的鴻喜菇，而不是指[[離褶傘屬|離褶傘屬]](''Lyophyllum'')的其他野生食用菇類。

== 食用 ==
工程產品廣泛分佈在日本和其他地方作為食用菌。 鴻喜菇的菌傘在溼潤時會稍微黏滑，煮食後嚐起來柔滑適口。沒有味道的特性，因此適合任何菜餚。 此外，天然生長比人工栽培更大肉質更紮實，口感更佳。
