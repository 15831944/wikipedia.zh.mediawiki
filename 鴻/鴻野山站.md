'''鴻野山站'''（{{jpn|j=鴻野山駅|hg=こうのやまえき|rm=Kōnoyama eki|f=y}}）是位於[[日本|日本]][[栃木縣|栃木縣]][[那須烏山市|那須烏山市]]鴻野山的[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[烏山線|烏山線]]鐵道車站。

== 車站構造 ==
[[側式月台|側式月台]]1面1線地上車站。開業起即為無人車站，沒有站舎。

== 車站周邊 ==
* 東宇都宮 Garden Club
* 鴻蘭（拉麵店）
* 鴻野山動物病院
* 高橋齒科醫院
* 南那須老人保健施設
* [[高根澤溫泉|高根澤溫泉]]
* [[栃木縣道10號宇都宮那須烏山線|栃木縣道10號宇都宮那須烏山線]]

== 历史 ==
* 1934年（[[昭和|昭和]]9年）8月15日 - 開業。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]成為JR東日本車站。
