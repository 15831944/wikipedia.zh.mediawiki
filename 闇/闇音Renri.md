{{nihongo|'''闇音レンリ'''|やみね れんり|Yamine Renri}}是yuzuri{{notetag|yuzuri是个二十多岁的女生，从学生时代就一直在制作歌曲，但因没有足够数量的歌曲投稿无法称之为歌手。<ref>{{cite web|url = http://blog.hangame.co.jp/hadoron1203/article/42933965/ |title = UTAU界隈に現れた新星・闇音レンリ|author = ハドロン|date = 2016-05-16|language = ja}}</ref>}}录制开发的歌声数据库，kgr担当[[人物设计|人物设计]]，EyesOnly设计[[标志|标志]]。[[Utau|Utau]]引擎的数据库Ver1.0于2014年9月28日发布，时隔一年，在之前基础上添加三种表情音源的数据库Ver1.5于2015年9月28日发布。Synthesizer V引擎的数据库于2018年11月11日发布。

==角色==

她有一头蓬松的粉红色长头发，还有蓝绿色的眼睛。
服装分有冬装和夏装。冬装随Ver1.0发布，样式为礼服加红色高跟鞋，戴有月亮和星星形状的装饰品。夏装随Ver1.5发布，样式为短袖加迷你裙和高跟凉鞋。

== 反响 ==
{{lang|ja|闇音レンリ}}以丰富的资源数据而广受好评，尤其是当中拥有的呼吸音和气泡音数据，是她最受称赞的地方。像拥有“{{lang|ja|伝説のレンリマスター}}”等标签的Police Piccadilly创作的《[http://www.nicovideo.jp/watch/sm28038974 {{nihongo|消歧义|曖昧さ回避}}]》和《[http://www.nicovideo.jp/watch/sm28668522 Cynic]》，以其拥有以假乱真的换气声和自然的力度转换带来的极似真人、强烈感情的歌声而闻名。<ref>{{cite news|url = http://hinanogimaya.com/1663/ |title = 新歌姫の「闇音レンリ」！美しすぎる歌声に魅了される人続出♪その正体にまた驚愕！！|author = 雛乃木まや|date = 2016-01-26|language = ja}}</ref><ref>{{cite news|url = http://do-ra.org/2017/05/18/119459/|title = 闇音レンリといえばこの人！ポリスピカデリーさんの神調教ボカロ曲をご紹介！|author = dora|date = 2017-05-18|language = ja}}</ref>
