{{Vgname|闇影詩章|ja=シャドウバース|en=Shadowverse}}，是一款由[[Cygames|Cygames]]开发和运营的[[集换式卡牌游戏|集换式卡牌游戏]]<ref name=IGN>{{cite web| author=Cam Shea| title=The Japanese Collectible Card Game That May Just Surprise You| url=http://www.ign.com/articles/2016/07/20/the-japanese-collectible-card-game-that-may-just-surprise-you| website=[[IGN|IGN]]| date=2016-07-20| accessdate=2016-11-15}}</ref>。首先于2016年6月17日在[[iOS|iOS]]、[[Android|Android]]平台开始运营，及后在[[DMM.com|DMM GAMES]]、[[Steam|Steam]]平台上运营<ref>{{cite web|title=Shadowverse|url=https://shadowverse.com/news/important/post-125|website=Shadowverse|accessdate=2017-09-11}}</ref>。游戏基本内容[[免费游戏|免费]]。

== 遊戲 ==
[[File:Shadowverse_ad_on_San-Chung_Bus_027-FU_20170725.jpg|缩略图]]
该游戏为Cygames旗下以幻想世界观为主要内容的線上对战型集換式卡牌遊戲，与同公司的社交游戏《[[巴哈姆特之怒|巴哈姆特之怒]]》共用世界观；此外，《闇影詩章》還將《巴哈姆特之怒》以及《[[碧藍幻想|碧藍幻想]]》中的部分插圖重新繪製後引用至當中<ref name=Fam>{{cite web| url=http://www.famitsu.com/news/201608/01112342.html| title=シャドウバース』の今後のアップデート情報について聞く！　優木かなさんと彩瀬てかさんとともに宮下尚之氏に直撃インタビュー| trans_title=The Future of Shadowverse, Interview with Naoyuki Miyashita, Yana Kuro, and Akase Taka| work=[[Famitsu|Famitsu]]| language=ja| date=2016-08-01| accessdate=2017-01-07}}</ref>。

本遊戲共包含三種大版本與營運區：日文版、國際版（包含英文、韓文、正體中文等版本）<ref>{{Cite web |url = https://gnn.gamer.com.tw/6/146416.html |title = 卡牌對戰遊戲《闇影詩章》今日舉辦記者會 宣布中文版 5 月 11 日登場 |author = Jerry |date = 2017-04-27 |publisher = [[巴哈姆特電玩資訊站|巴哈姆特電玩資訊站]]|language = zh-tw |accessdate = 2018-2-22}}</ref>、簡體中文版；前兩者由Cygames直營且為同一伺服器，後者則由[[網易|網易]]代理。2018年8月21日於國際版客戶端追加簡體中文版，惟翻譯文本與網易版有極大差異點。

本遊戲的卡包於2017年12月起加入「退環境」的制度，稱「指定系列（Rotation）」，允許玩家使用最新五季卡包，並隨著每季新卡包推出，最舊的卡包將退出環境；「無限制」模式則可以使用過季卡包<ref>{{Cite web |url = https://www.youtube.com/watch?v=h6bT5Opf74Y |title = 【Shadowverse 闇影詩章】官方節目 Shadowverse NEXT（中文字幕） |author = 闇影詩章 Shadowverse 台灣官方頻道
 |date = 2017-11-1 |publisher = Cygames |language = zh-tw |accessdate = 2018-2-26}}</ref>。另外，自第八季《起源之光·終焉之闇》開始，每季第二個月底將新增「擴充卡包」。

=== 系统 ===
; {{nihongo|戰鬥系統|Matches}}
: 玩家可以在8名{{nihongo|'''主戰者'''|リーダー}}中选择一位，在1398张卡牌（截至OoT）<ref>{{Cite web|url=http://sv.bagoum.com/cardSort|title=Card Sorting|accessdate=2018-10-16|work=sv.bagoum.com|language=en-US}}</ref>中构筑自己的卡组。根据角色的不同，可用于构筑卡组的卡牌会有所不同。
; {{nihongo|进化系統|進化|Evolving}}
: 在「神击的巴哈姆特」发布后，本作新增的进化系统。在经过一定的回合数后，玩家可以消耗進化點來进化自己的從者。<ref name="GameWatch">{{cite web|url=http://game.watch.impress.co.jp/docs/interview/1005836.html|title=ついにサービス開始！ デジタルTCG「Shadowverse」インタビュー|accessdate=2016-11-30|author=泊裕一郎|date=2016年6月20日|coauthors=安田俊亮|publisher=GameWatch}}</ref>
: 進化後的從者不止能增加攻擊力和生命值(部分例外)，還可以和擁有突進能力的從者一樣，即便是剛下場的回合也能立即攻擊敵人的從者。而有部分從者具有進化時發動，或進化後獲得的能力。
: 玩家每場對戰所擁有的進化點數量都有限，而且只有極少數的卡片能力可以回復消耗掉的進化點。妥適運用有限的進化點，並判斷進化的時機，就能夠創造決定性的勝利關鍵，或扭轉頹敗的劣勢。
; {{nihongo|卡牌系統|カード|Cards}}
:卡牌有從者、法術及護符三种类型，使用方法有所不同。
:; {{nihongo|從者|フォロワー|Followers}}
:: 從者是场上可以战斗的卡牌。從者可以对己方角色进行辅助，并可以对敌方角色或從者造成伤害。战斗中，可以进化自己的從者。
:; {{nihongo|法術|スペル|Spells}}
:: 打出时立刻发动效果的卡牌，效果结束后立刻移至墓场。
:; {{nihongo|護符|アミュレット|Amulets}}
:: 打出时即发动、随后停留在场内发挥效果的卡牌。饰品本身无法进行攻击或被攻击。

=== 剧情 ===
故事敘述被称作「{{nihongo|'''天啟之樹'''|災いの樹}}」的闪耀图纹突然出现在世间。精灵少女'''亚里莎'''为了成为「森林守护者」，与同是精灵少女的{{nihongo|'''羅莎莉亞'''|ロザリア}}每天进行着修行。但某一天，他们遭到了谜之怪物「{{nihongo|'''虚之影'''|虚の影}}」的袭击，萝沙丽亚被掳走。

为了拯救萝沙丽亚，亚里莎下定决心，踏上了旅程。同时，阿雷斯达王国公主的护卫兼女仆「'''艾莉卡'''」、宫廷魔术师「'''伊莎贝尔'''」、原骑士团长「'''罗文'''」、居住在洋房的少女「'''露娜'''」、被封印的吸血鬼「'''尤里亚斯'''」、教会的祭司「'''伊莉丝'''」、举起反抗之旗的复仇者「'''伊昂'''」，也因各种机缘巧合，开始了他们的行动。

== 發行 ==
本作的PC版于2016年8月22日在DMM GAMES平台、2016年10月28日在Steam平台发行。可选的操作系统包括Windows、Mac OS。可选的语言包括英语、繁体中文及日语。PC版拥有手机版没有的分辨率变更、60fps画面、战斗中自定义鼠标操作、战斗中快捷键操作等功能。

PC版与手机版间可以进行数据共享，在多个设备上可以共享一份游戏数据，以下为注意事项：
* 可以与手机版进行数据共享的的PC版帐户仅限一个。
* 已与其他PC游戏平台进行数据共享的账号，不能与Steam平台进行数据共享。
* 一旦与PC版进行了数据共享，则无法解除。
* 在手机版或DMM GAMES版中购买的水晶不能在Steam版中使用，反之亦然。{{notetag|进行数据共享并不会将已购买的水晶删除，但只能在所购买的平台进行使用。}}

*因為遊戲中若干主戰者和卡牌的正體中文譯本，被中國政府審查部門指出有流血、暴力、恐怖等負面意涵，遂要求網易進行[[河蟹_(網路用語)|文句修飾]]為較少煽動性的同義詞<ref>{{Cite web |url = https://sv.163.com/revisions/index.html |title = 國服本地化內容搶先看 |author = 網易 |date = 2018-1-10 |language = zh-cn |accessdate = 2018-2-22}}</ref><ref>{{Cite web |url = https://sv.163.com/news/strategy/20180322/26644_745796.html |title = 关于影之诗你需要知道的二三事 |author = 網易 |date = 2018-3-22 |language = zh-cn |accessdate = 2018-5-15}}</ref>。舉例如下：
**死靈法師→喚靈師
**吸血鬼→暗夜伯爵
**復仇者→超越者
**復仇→反擊（吸血鬼在生命值10以下的被動效果）
*在2018年4月27日起正式上線的中國服，初始版本為第七季卡包《時空轉生》，而非Cygames直營服的當前版本─第八季卡包《起源之光·終焉之闇》<ref>{{Cite web |url = https://www.iyingdi.cn/web/article/verse/54870 |title = 影之诗国服5月4日公测，4月27日IOS先行 |author = 星野餅美 |publisher = 旅法師營地 |date = 2018-4-10 |language = zh-cn |accessdate = 2018-5-15}}</ref>。
*根據網易公告表示，本遊戲的中國伺服器，因為程式架構差異、相容性測試、審查上線流程等因素，平均時間相比國際服慢45~60天；因此第八季卡包《起源之光·終焉之闇》在中國服延到2018年6月14日實裝，並且尚無國際服於2018年5月30日實裝的「擴充卡包」，往後的更新延後也依此類推<ref>{{Cite web |url = http://sv.163.com/news/update/20180523/26643_755819.html |title = 國服版本差別說明公告 |author = 網易 |date = 2018-5-23 |language = zh-cn |accessdate = 2018-5-23}}</ref><ref>{{Cite web |url = http://sv.163.com/news/update/20180523/26643_755816.html |title = 國服版本差詳細時間說明公告 |author = 網易 |date = 2018-5-23 |language = zh-cn |accessdate = 2018-5-23}}</ref>。

== 爭議事件 ==
=== 港澳玩家數據擬遷移事件 ===
根據Cygames於2018年6月14日所發布之公告，'''由於中國代理商網易取得香港、澳門兩地的遊戲經營權，因此Cygames擬將港澳玩家數據遷移至網易營運之中國服'''，並訂出如下時程<ref>{{Cite web |url = https://shadowverse.com/cht/news/detail.php?announce_id=662 |title = 【2018年6月28日實施】中國香港澳門地區遷移數據公告 |author = 闇影詩章中文官網 |date = 2018-6-14 |publisher = Cygames |language = zh-tw |accessdate = 2018-6-16}}</ref>：
#2018/6/28~2018/8/10 0900：開放受理港澳用戶申請將遊戲數據遷移至網易（''非強制性，若不願轉移資料者，其原始資料仍會持續留在Cygames國際服''）
#2018/8/14：'''終止提供於港澳Google Play、App Store等手機商城下載遊戲'''（''Steam電腦版暫未受影響''）
#各項在中國服未授權的合作內容，如《[[快打旋風|快打旋風]]》人物造型、[[杉田智和|杉田智和]]廣播劇《{{Link-ja|Anigera|杉田智和のアニゲラ!ディドゥーーン}}》合作內容，均以等值金幣或水晶補償<ref>{{cite web|url=https://ma68na.bbs.easebar.com/chore/hant/faq/transfer.html|title=網易港澳版《影之詩》數據遷移問題回答合集|publisher=網易|date=2018-06-14|accessdate=2018-06-16|language=zh-hk}}</ref>。
此一舉措除了將本遊戲的國際繁中版客戶端，自手機商城下架外，甚至'''封鎖港澳玩家用本地IP位置遊玩'''（''除非用中港澳以外IP的VPN''），故引起港澳玩家一片譁然<ref>{{cite web|url=https://forum.gamer.com.tw/C.php?page=1&bsn=29330&snA=14028&tnum=102|title=中國香港澳門地區遷移數據公告- 香港澳門的玩家要被丟棄了?|author=LLS|publisher=[[巴哈姆特電玩資訊站|巴哈姆特討論區]]|date=2018-6-14|accessdate=2018-06-16|language=zh-hant}}</ref>。

Cygames於8月1日更新前述公告內容，表示'''港澳玩家的伺服器轉移期限，將無限期順延'''，直到網易另行發布通知，故港澳玩家至今能繼續在Cygames直營服進行遊戲。
