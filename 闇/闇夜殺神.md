《'''闇夜殺神'''》（{{lang-en|''Rendel''}}）是一部於2017年上映，由[[耶塞·哈亚|耶塞·哈亚]]（Jesse Haaja）編寫、製作和執導的芬蘭[[超級英雄電影|超級英雄電影]]。本片是根據17年前由哈亚創作的漫畫角色所改編<ref>[https://tw.news.yahoo.com/%E8%8A%AC%E8%98%AD%E5%BD%B1%E5%8F%B2%E9%A6%96%E5%89%B5%E6%9A%97%E9%BB%91%E7%B3%BB%E8%B6%85%E7%B4%9A%E8%8B%B1%E9%9B%84%E3%80%8A%E9%97%87%E5%A4%9C%E6%AE%BA%E7%A5%9E%E3%80%8B%E9%9C%87%E6%92%BC%E7%99%BB-105144121.html 芬蘭影史首創暗黑系超級英雄《闇夜殺神》震撼登場] 	Yahoo奇摩電影，2017年11月26日</ref><ref>[http://ipop.sina.com.tw/posts/278318 芬蘭版黑暗英雄《闇夜殺神》用復仇打造超級正義] 	POP!Weibo，2017年11月29日</ref>。由{{tsl|fi|Kris Gummerus|克里斯·貢梅魯斯}}、{{tsl|fi|Matti Onnismaa|馬蒂·翁尼斯馬}}、{{tsl|fi|Rami Rusinen|拉米·魯西寧}}和{{tsl|fi|Renne Korppila|伦内·科尔皮拉}}領銜主演。其他演員包括{{tsl|fi|Alina Tomnikov|阿莉娜·通尼科夫}}、{{tsl|fi|Tero Salenius|泰罗·萨勒纽斯}}、{{tsl|fi|Aake Kalliala|阿克·卡利阿拉}}、{{link-en|阿努·帕勒瓦拉|Anu Palevaara}}、{{link-en|克里斯蒂娜·卡尔亚莱宁|Kristina Karjalainen}}。

== 劇情 ==
主人公变成闇夜殺神，带着超级英雄的面具去向谋杀了他的家人的犯罪組織VALA复仇。

==製作==
電影的預算製作為145萬歐元，於2015年秋季和2016年春季拍攝了50天。本片拍攝於[[卡亞尼|卡亞尼]]、[[米凱利|米凱利]]、[[瓦爾考斯|瓦爾考斯]]、[[赫爾辛基|赫爾辛基]]、[[索特卡莫|索特卡莫]]、[[庫奧皮奧|庫奧皮奧]]和[[普奧蘭卡|普奧蘭卡]]。

== 發行 ==
本片於2017年9月22日在芬蘭上映，2018年1月26日在台灣上映<ref>[https://zeekmagazine.com/archives/55771 北歐式的復仇正義！芬蘭首部暗黑系超級英雄電影《闇夜殺神》明年1/26震撼上映] ZEEK玩家誌，2017年11月29日</ref>。

== 獲獎 ==
本片於2017年10月在墨西哥的Feratum電影節上獲得最佳動作電影，並參加2017德國[[奇幻電影節|奇幻電影節]]、2017​赫爾辛基國際影展、2017比利時布魯日Razor Reel Flanders影展、2017芝加哥Cinepocalypse恐怖影展。
