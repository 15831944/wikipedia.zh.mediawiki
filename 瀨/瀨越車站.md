'''瀨越站'''（{{jpn|j=瀬越駅|hg=せごしえき|rm=Segoshi eki|f=y}}）是位於[[北海道|北海道]][[留萌市|留萌市]]沖見町1丁目，隸屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[留萌本線|留萌本線]]，已廢除的[[鐵路車站|鐵路車站]]。電報略號為'''コセ'''。

車站名稱估計源於[[阿伊努語|阿伊努語]]「{{lang|ja|腕のように曲がった所}}」（像手臂一樣彎曲的地方），但詳細不詳。

最初為了方便泳客而設立的季節性限定臨時車站（{{lang|ja|仮乗降場}}），但由於周邊居民的使用量上升而升格為全年營運的臨時車站（{{lang|ja|臨時駅}}）。隨著[[國鐵分割民營化|國鐵分割民營化]]而再升格為一般車站，並最終設定營業距離。長時間營運而沒有設定營業距離的臨時車站，除了本車站外就只有[[常磐線|常磐線]][[偕樂園車站|偕樂園車站]]。

== 歷史 ==
* 1926年（大正15年）7月1日 - [[鐵道省|鐵道省]]'''留萌線'''[[留萌車站|留萌車站]]至[[禮受車站|禮受車站]]之間設立'''瀨越臨時車站'''（{{lang|ja|仮乗降場}}）。
* 1930年（昭和6年）10月10日 - 留萌線改稱為'''留萌本線'''。
* 1949年（昭和24年）6月1日 - 由[[日本國有鐵道|日本國有鐵道]]接管。
* 1969年（昭和44年）10月1日 - 車站升格為臨時站（{{lang|ja|臨時駅}}）。
* 1980年後半 - 候車室改建，轉為使用貨物列車改建的候車室。
* 1987年（昭和62年）4月1日 - 由於[[國鐵分割民營化|國鐵分割民營化]]，車站由[[北海道旅客鐵道|JR北海道]]管理，同時升格為一般車站'''瀨越站'''。
* 1990年（平成2年）3月10日 - 設定營業距離。
* 日期不詳 - 候車室由貨物列車改為現在的建築。
* 2016年（平成28年）12月5日 - 由於[[留萌車站|留萌車站]]至增毛站路段被廢除，本站正式廢站<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2016/160428-3.pdf |language=ja |title=留萌線（留萌・増毛間）の鉄道事業廃止届の提出について |publisher=[[北海道旅客鐵道|JR北海道]] |date=2016-04-28 |accessdate=2017-06-13 |archive-url=https://web.archive.org/web/20160429052355/http://www.jrhokkaido.co.jp/press/2016/160428-3.pdf |archive-date=2016-04-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20160429052355/http://www.jrhokkaido.co.jp/press/2016/160428-3.pdf |archivedate=2016-04-29 }}</ref><ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2016/160628-1.pdf |language=ja |title=留萌線（留萌・増毛間）の廃止日繰上げの届出について |publisher=[[北海道旅客鐵道|JR北海道]] |date=2016-06-28 |accessdate=2017-06-13 |archive-url=https://web.archive.org/web/20160701083826/http://www.jrhokkaido.co.jp/press/2016/160628-1.pdf |archive-date=2016-07-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20160701083826/http://www.jrhokkaido.co.jp/press/2016/160628-1.pdf |archivedate=2016-07-01 }}</ref>。

== 車站構造 ==
廢站時為1面1線的地面車站，[[側式月台|側式月台]]位於路軌的東方（增毛方向的左方），沒有[[轉轍器|轉轍器]]的棒線車站。

由臨時車站時代開始已經是無人車站。月台北方設有候車室。以前曾設置與禮受車站同款，由列車改建而成的車站建築。由於鄰近海邊令列車受到嚴重損害，因此改為使用由[[混凝土|混凝土]]的[[預製組件|預製組件]]建成的候車室。沒有設置[[廁所|廁所]]。

== 載客量 ==
* 1992年的每天載客量為32人，每天上下車乘客量為64人。
* 2011年至2015年度平均每天載客量少於10人<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |language=ja |title=平成26年度 線区別の収支状況等について |publisher=JR北海道 |date=2016-02-10 |accessdate=2017-06-21 |archive-url=https://web.archive.org/web/20170407133949/http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |archive-date=2017-04-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20170407133949/http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |archivedate=2017-04-07 }}</ref>。
