'''瀨戶線'''（{{jpn|j=瀬戸線|hg=せとせん|rm=Seto sen|f=y}}）是一條連結日本[[愛知縣|愛知縣]][[名古屋市|名古屋市]]至鄰市[[瀨戶市|瀨戶市]]的[[鐵路線|鐵路線]]。由中部地方[[大手私鐵|大手私鐵]]之一的[[名古屋鐵道|名古屋鐵道]]（名鐵）所經營的這條路線，起訖點分別是位於名古屋市[[東区_(名古屋市)|東区]]的[[榮町車站_(愛知縣)|榮町車站]]與瀨戶市的主車站——[[尾張瀨戶車站|尾張瀨戶車站]]。

== 路線資料 ==
; 營業中的路段（榮町－尾張瀨戶間）
:* 路線距離（{{link-ja|營業距離|営業キロ}}）：20.6公里
:* [[軌距|軌距]]：1,067毫米
:* 站數：20個（包括起終點站）
:* 複線路段：全線
:* 電氣化路段：全線（直流1,500 V）
:* [[閉塞_(鐵路)|閉塞方式]]：自動閉塞式
:* 認可最高速度：95公里/小時<ref group="注釋">比其他的名鐵路線運行速度低，信號現示對限制速度也是注意信號45公里/小時、減速信號65公里/小時與20公里/小時為低。</ref>
:
; 1976年（昭和51年）廢除路段（堀川－（舊）東大手間，廢除時）
:* 路線距離（營業距離）：1.9公里
:* 軌距：1,067毫米
:* 站數：4個（包括{{link-ja|休止站|休止駅|休止}}中的東大手站）
:* 複線路段：全線（但是存在{{link-ja|單複線|単複線}}）
:* 電氣化路段：全線（[[直流電|直流電]]600 V）
:* 閉塞方式：[[閉塞_(鐵路)#自動閉塞式|自動閉塞式]]
:* 認可最高速度：70公里/小時（但是站間距離較短，也存在半徑60公尺的急曲線）
