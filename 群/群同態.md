在[[數學|數學]]中，給定兩個[[群|群]]（''G'', *）和（''H'',·），從 (''G'', *)到 (''H'',·)的'''群同態'''是[[函數|函數]]''h'' : ''G'' → ''H''使得對於所有''G''中的''u''和''v''下述等式成立
: ''h''（''u'' * ''v'') = ''h''(''u'')·''h''(''v''）
在這裡，等號左側的群運算*，是''G''中的運算；而右側的運算·是''H''中的運算。

從這個性質，可推導出''h''將''G''的單位元''e<sub>G</sub>''映射到''H''的單位元''e<sub>H</sub>''，并且它還在''h''(''u''<sup>-1</sup>) = ''h''(''u'')<sup>-1</sup>的意義上映射逆元到逆元。因此我們可以說''h''“兼容於群結構”。

更老的給同態''h''(''x'')的符號是''x''<sub>''h''</sub>，它容易混淆於索引或一般下標。更新近的傾向是把群同態寫在它們的自變量的右側，省略括號，如此''h''(''x'')簡化成了''x h''。這種方法特別流行於[[自動機|自動機]]扮演角色的群論領域，因為它更適應自動機從左至右讀字詞的習慣。

在考慮配備了加法結構的群的數學領域中，同態有時關照的不只是（如上）群結構而且還有額外的結構。比如[[拓撲群|拓撲群]]的同態經常要求是連續性的。

== 像和核 ==

我們定義''h''的[[核_(代數)|核]]為被映射到''H''中單位元上的''G''中的那些元素的集合
:ker(''h'') = { ''u'' ∈ ''G'' : ''h''(''u'') = ''e<sub>H</sub>'' } 
定義''h''的[[像|像]]為
:im(''h'') = { ''h''(''u'') : ''u'' ∈ ''G'' }。
核是''G''的[[正規子群|正規子群]]（事實上，''h''(''g''<sup>-1</sup> ''u'' ''g''）= ''h''(''g'')<sup>-1</sup> ''h''(''u'') ''h''(''g'') = ''h''(''g'')<sup>-1</sup> ''e<sub>H</sub>'' ''h''(''g'') =
''h''(''g'')<sup>-1</sup> ''h''(''g'') = ''e<sub>H</sub>'')而像是''H''的[[子群|子群]]。同態''h''是[[單射|單射]]（并叫做单同態）當且僅當ker(''h'') = {''e''<sub>''G''</sub>}。

同態的核和[[像|像]]可以被解釋為對它接近於同構程度的程度。[[同構基本定理|第一同構定理]]聲稱群同態的[[像|像]]im(''h'')同構於商群''G''/ker(''h'')。

== 例子 ==

* 考慮帶有加法的[[循環群|循環群]]'''Z'''/3'''Z''' = {0, 1, 2}和整數集'''Z'''的群。映射''h'' : '''Z''' → '''Z'''/3'''Z'''，有著''h''(''u'') = ''u'' [[模運算|模]]以3，是群同態。它是[[滿射|滿射]]并且它的核由被三整除的所有整數構成。

* [[指數函數|指數映射]]產生從帶有加法的[[實數|實數]]集'''R'''的群到帶有乘法的非零實數集'''R'''<sup>*</sup>的群的群同態。核是{0}而像由正實數組成。

* 指數映射還產生從帶有加法的[[複數|複數]]集'''C'''的群到帶有乘法的非零複數集'''C'''<sup>*</sup>的群的同態。這個映射是滿射并且有核{ 2π''ki'' : ''k'' ∈ '''Z''' }，這可以從[[歐拉公式|歐拉公式]]得出。

* 給定任何兩個群''G''和''H''，映射''h'' : ''G'' → ''H''，把所有''G''的元素對應到''H''的單位元，是同態；它的核是集合''G''。

* 給定任何群''G''，恒等映射id : ''G'' → ''G''，有著id(''u'') = ''u''對於所有''G''中的''u''，是群同態。

== 群範疇 ==

如果''h'' : ''G'' → ''H''和''k'' : ''H'' → ''K''是群同態，則''h'' o ''k'' : ''G'' → ''K''也是群同態。這證明所有群的[[類_(數學)|類]]，和群同態作為態射一起形成一個[[範疇論|範疇]]。

== 同態映射的類型 ==

如果[[同態|同態]]''h''是[[雙射|雙射]]，則你還可以證明它的逆映射仍是同態，這種''h''叫做[[群同構|群同構]]；在這種情況下，群''G''和''H''被稱為是“同構的”:它們只在元素的符號上有差異而對於所有實踐用途都是同一的。

如果''h'': ''G'' → ''G''是群同態，我們稱之為''G''的'''[[自同態|自同態]]'''。如果它進一步的是雙射并且因此是同構，則稱為[[同構|同構]]。群''G''的所有自同構的集合，帶有函數復合作為運算，自身形成一個群，叫做''G''的自同構群。它指示為Aut(''G'')。作為例子，('''Z''', +)的自同構群只有兩個元素，恒等變換和乘以 -1；它同構於'''Z'''/2'''Z'''。

'''滿同態'''是[[滿射|滿射]]同態，'''單同態'''是[[單射|單射]]同態。

== 阿貝爾群的同態 ==

如果''G''和''H''是[[阿貝爾群|阿貝爾群]]（就是交換群），則所有從''G''到''H''的群同態的集合Hom(''G'', ''H'')自身是阿貝爾群：兩個同態的和''h'' + ''k''定義為
:(''h'' + ''k'')(''u'') = ''h''(''u'') + ''k''(''u'')，對於所有''G''中''u''。
''H''的交換律對于證明''h'' + ''k''也是群同態是必需的。同態的加法在如下意義上兼容於同態的復合：如果''f''在Hom(''K'', ''G'')中，''h'', ''k''是Hom(''G'', ''H'')的元素，并且''g''在Hom(''H'',''L'')中，則 
:(''h'' + ''k'') o ''f'' = (''h'' o ''f'') + (''k'' o ''f'')，并且''g'' o (''h'' + ''k'') = (''g'' o ''h'') + (''g'' o ''k'')。
這證明了一個阿貝爾群的所有自同態的集合End(''G'')形成了一個[[環|環]]，即''G''的[[自同態環|自同態環]]。例如，由'''Z'''/2'''Z'''的兩個復本的[[直積|直積]]構成的阿貝爾群（[[克萊因四元群|克萊因四元群]]）的自同態群同構於帶有'''Z'''/2'''Z'''內元素的2×2 [[矩陣|矩陣]]的環。上述兼容性還證明所有阿貝爾群帶有群同態的範疇形成了[[預加法範疇|預加法範疇]]；直積和良好定義的核的存在性使這個範疇成為[[阿貝爾範疇|阿貝爾範疇]]的原型例子。
