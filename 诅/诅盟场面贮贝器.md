'''诅盟场面贮贝器'''，或称'''诅盟场面铜贮贝器'''、'''诅盟场面青铜贮贝器'''，中国[[西汉|西汉]]时期文物，1955年至1960年间于[[云南|云南]][[晋宁|晋宁]][[石寨山古墓群|石寨山]]出土。该器为[[青铜|青铜]]铸造，通高53厘米，盖径32厘米，底径29.7厘米。出土时器内贮[[贝币|贝]]300余枚，器盖上铸圆雕立体人物127人，以干栏式建筑上的人物活动为中心，表现了滇王杀祭诅盟的典礼场面<ref name="cnm1">{{cite web |title=诅盟场面青铜贮贝器 |url=http://www.chnmuseum.cn/(S(rrtjck45ia3r32452svtgr55))/Default.aspx?TabId=450&antiquelanguageid=701 |website=中国国家博物馆 |accessdate=2018-07-14 |archive-url=https://web.archive.org/web/20180714164345/http://www.chnmuseum.cn/(S(rrtjck45ia3r32452svtgr55))/Default.aspx?TabId=450&antiquelanguageid=701# |archive-date=2018-07-14 |dead-url=yes }}</ref>。2013年8月19日，诅盟场面贮贝器被[[国家文物局|国家文物局]]收录到公布的《第三批禁止出境展览文物目录》中，属于[[禁止出境展览文物|禁止出境展览文物]]<ref name=cnnew>{{cite web |title=94件一级文物列入"第三批禁止出境展览文物目录" |url=http://www.chinanews.com/cul/2013/08-19/5180173.shtml |website=中国新闻网 |accessdate=2018-07-14}}</ref>。现藏于[[中国国家博物馆|中国国家博物馆]]<ref name=cnnew/>。

== 出土 ==
[[云南省博物馆|云南省博物馆]]从1950年代开始在该省全省范围内进行的科学考古发掘、试掘、调查<ref>{{cite web |title=云南省博物馆 |url=http://www.china.com.cn/book/zhuanti/1bowuguan/2008-11/11/content_16746569.htm |website=中国网 |accessdate=2018-07-14}}</ref>。1955年，在[[云南省|云南省]][[昆明市|昆明市]][[晋宁县|晋宁县]]的石寨山西面坡上发掘了新石器时代遗址一处，获得大量陶器和石器。1955年至1960年间，先后四次在石寨山山麓发掘了[[春秋战国|春秋战国]]至[[东汉|东汉]]时期的墓葬48座，出土文物四千多件<ref>{{cite web |author1=白寿彝主编 |title=第八节 汉代少数族地区的考古资料 |website=中国通史}}</ref>。[[石寨山古墓群|石寨山古墓群]]出土的诅盟场面贮贝器、[[滇王之印|“滇王之印”金印]]、[[杀人祭柱场面贮贝器|杀人祭柱场面贮贝器]]等均为禁止出境展览文物。

== 形制 ==
[[贮贝器|贮贝器]]是[[滇国|滇国]]特有的贮放[[贝币|贝币]]的青铜器，器上铸造出姿态各异的群像，具体而生动地表现了祭祀、战争、狩猎、祈年、播种等情形，是研究滇国社会历史的重要资料。出土时器中贮满源自印度洋和太平洋的暖水区域的海贝。<ref>{{cite web |author1=陈成军 |title=古滇国的外汇 |url=http://www.chnmuseum.cn/tabid/438/InfoID/78565/frtid/285/Default.aspx |website=中国国家博物馆 |accessdate=2018-07-14 }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

诅盟场面贮贝器器身呈筒形，腰微束，两侧有对称的虎形耳，底部有三只兽爪足。器盖上铸有表现滇王杀祭诅盟场面的雕塑，包括干栏式房屋一间，巨型[[铜鼓|铜鼓]]两台，以及各类人物和牲畜，据统计除残缺者以外，器盖上人物仍达127个之多，是此次发掘的众多贮贝器中，再现场面最大、最丰富的一件。<ref name="cnm1" /><ref>{{cite web |title=《文化大百科》 20131228 诅盟场面铜贮贝器 |url=http://tv.cntv.cn/video/C34057/3f53a370dd314187a7ee3eea83fc3123 |website=央视网 |accessdate=2018-07-15}}</ref>

器盖上的房屋建筑主要由屋顶和平台构成。屋顶分上、下两层，上层呈人字形，下层四面出檐。平台由小柱支撑，前后均通过梯子与地面（器盖面）连接。平台上高凳上垂足坐着一位妇女，似为主祭人，其周围放置有十六面铜鼓。妇女的左前方及右侧均为参与祭祀者，面前则摆放着各类祭品。房屋外有椎牛刑马、屠豕宰羊等场面，有持刀的屠夫，有喂猪、喂孔雀的妇女，还有虎、犬等动物，有击打青铜鼓和[[錞于|錞于]]的，有被捆绑在石碑上的裸体男子和持器盛物的妇女等。<ref name="cnm1" /><ref name="cnm2">{{cite web |title=诅盟场面铜贮贝器 |url=http://www.chnmuseum.cn/Default.aspx?TabId=450&AntiqueLanguageID=3740 |website=中国国家博物馆 |accessdate=2018-07-14 |archive-url=https://web.archive.org/web/20180714193406/http://www.chnmuseum.cn/(S(0hsdwdqk13mx1d55ewf3gnff))/Default.aspx?TabId=450&AntiqueLanguageID=3740# |archive-date=2018-07-14 |dead-url=yes }}</ref>
