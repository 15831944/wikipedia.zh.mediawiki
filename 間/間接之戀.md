{{日本作品|間接之戀|ひとつ飛ばし恋愛}}是[[HOOKSOFT|ASa Project]]在2013年4月26日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]<ref>{{cite web|url=http://www.game-style.jp/special/201303/18/00tob_ap.php?permission=true|title=「Game-Style」特集|website=Game-Style|date=2013年3月18日|accessdate=2017年4月23日|language=ja}}</ref>，2015年1月29日由Entergram發售[[PlayStation_Vita|PlayStation Vita]]版《間接之戀V》<ref>{{cite web|url=https://www.famitsu.com/news/201410/10063200.html|script-title=ja:『ひとつ飛ばし恋愛V』PS Vitaで2015年1月29日に発売決定 新規ヒロインが2名追加、シナリオが大幅加筆！|publisher=Fami通|date=2014年10月10日|accessdate=2017年4月23日|language=ja}}</ref>。

== 故事簡介 ==
跟隨父親單身赴任的三枝和樹，經過三年後回到了自己的家，並進入春野學園就讀。在故鄉展開新生活的和樹認識了四名女孩，而這四名女孩都是和樹親人的好友<ref>{{Cite web|url=http://www.game-style.jp/soft/201303/11/01tob_ap.php|script-title=ja:【Game-Style】「身内」を挟んでその「友達」と恋仲になる、もどかしくて刺激的なドキドキ間接恋愛劇！ 『ひとつ飛ばし恋愛』4月26日発売！|publisher=Game-Style|date=2013年3月11日|accessdate=2017年4月23日|language=ja}}</ref>。

== 登場角色 ==
* 聲優為PC版／PSV版<ref>{{Cite web|url=http://www.entergram.co.jp/hitotsutobashiV/character01.html|script-title=ja:ひとつ飛ばし恋愛Ｖ オフィシャルサイト - 登場人物|publisher=Entergram|date=2014年10月20日|accessdate=2017年4月23日|language=ja}}</ref>
=== 主要角色 ===
; {{nihongo|三枝和樹|三枝 和樹（さえぐさ かずき）}}
: 本作男主角，跟隨父親單身赴任三年後回到了家鄉，並進入春野學園二年級就讀。
; {{nihongo|園原碧里|園原 碧里（そのはら あおり）|聲：美美永跳<ref name="character">{{cite web|url=http://www.asa-pro.com/hitotsutobashi/chara/index.html|title=character|publisher=ASa Project|accessdate=2018-04-22|language=ja}}</ref>／[[水橋香織|水橋香織]]}}
: 春野學園一年級生，和樹的妹妹紅的朋友，游泳社成員。和紅雖然經常吵架但還是交情很好。
; {{nihongo|蒼木夏芽|蒼木 夏芽（あおき なつめ）|聲：[[遠野Soyogi|遠野Soyogi]]<ref name="character"/>／[[岡嶋妙|岡嶋妙]]}}
: 春野學園三年級生，和樹的姐姐惠的朋友，游泳社社長。由於態度冷淡，讓周遭的人覺得恐怖。
; {{nihongo|玉森櫻|玉森 桜（たまもり さくら）|聲：あじ秋刀魚<ref name="character"/>／前田惠}}
: 春野學園二年級生，和樹的青梅竹馬千乃的朋友。個性溫柔且氣質出眾，因而有「學園的偶像」之稱。
; {{nihongo|壽理沙|寿 りさ（ことぶき りさ）|聲：[[五行薺|五行薺]]<ref name="character"/>／[[小野涼子|小野涼子]]}}
: 春野學園二年級生，和樹的表姐阿知華的跟班。個性活潑開朗，喜歡開黃腔。
; {{nihongo|神木明日葉|神木 明日葉（かみき あすは）|聲：無／[[佐倉綾音|佐倉綾音]]}}
: PSV版追加女主角。春野學園三年級生，和樹班級的副導師的女兒，不太會把感情表現在臉上。
; {{nihongo|神木郁琉|神木 いくる（かみき いくる）|聲：無／[[丹下櫻|丹下櫻]]}}
: PSV版追加女主角。春野學園一年級生，和樹班級的副導師的女兒，明日葉的妹妹，有著像小動物般的行為舉止，但也有腹黑的一面。
