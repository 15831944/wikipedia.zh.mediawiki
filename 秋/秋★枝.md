'''秋★枝'''（あきえだ、4月29日<ref>[http://www.h7.dion.ne.jp/~fakefur/about.html about this site] {{webarchive|url=https://web.archive.org/web/20120128231411/http://www.h7.dion.ne.jp/~fakefur/about.html |date=2012-01-28 }} 2010年1月更新、2011年12月5日瀏覽。</ref>－），是[[日本|日本]]女性[[漫画家|漫画家]]、[[插畫家|插畫家]]、同人作家、同人社团[[wikt:主催|主催]]。[[山口縣|山口縣]]出身。[[愛媛大學|愛媛大學]]畢業<ref>[https://twitter.com/aidaigi/status/614269033681043456]</ref>。「{{lang|ja|秋☆枝}}」單純是標示錯誤或誤植。

== 人物 ==
最喜歡[[咖啡|咖啡]]、[[紅茶|紅茶]]、日本的犬貓、[[睡覺|睡覺]]。第二的故鄉為[[愛媛県|愛媛県]]的[[松山市|松山市]]。曾公開說是[[SPITZ|SPITZ]]、[[the_pillows|the pillows]]、[[SUMMERS|SUMMERS]]的大粉絲。也常在自己作品內讓戴眼鏡的角色出現很多次。

『[[月刊COMIC_FLAPPER|月刊COMIC FLAPPER]]』雜誌上曾經公開說出「想嫁給[[網球王子角色列表#青春學園國中部（東京都）|乾貞治]]！」這句話。

曾是[[MTG|MTG]]professional gamer的中村修平信者。創建同人社團『{{lang|ja|ロケット燃料★21}}』與開設個人社團、很擅長[[東方Project|東方Project]]的二次創作、後來擔當『[[東方儚月抄|東方儚月抄]]』漫畫版的主要作畫。
