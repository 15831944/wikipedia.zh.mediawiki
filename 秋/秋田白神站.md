'''秋田白神站'''（{{jpn|j=あきた白神駅|hg=あきたしらかみえき|rm=Akitashirakami eki|f=y}}）位于[[秋田县|秋田县]][[山本郡|山本郡]][[八峰町|八峰町]]八森字御所之台，是[[东日本旅客铁道|东日本旅客铁道]]（JR東日本）管辖的[[五能线|五能线]]上的[[鐵路車站|鐵路車站]]<ref name="station"/><ref>{{cite web|url=https://www.sakigake.jp/news/article/20171002AK0018/|title=五能線あきた白神駅、２０周年祝う　吹奏楽にぎやかに|publisher=秋田魁新報|language=ja}}</ref>。

临时快速“[[Resort白神号|Resort白神号]]”停车站。

== 历史 ==
* 1997年10月1日 - 车站投入运营<ref name="station"/>。

== 车站结构 ==
[[侧式站台|侧式站台]]1台1线的[[地上车站|地上车站]]<ref name="station"/>。

[[能代站|能代站]]管理的业务委托站<ref name="station"/>。站房为左侧白色和浅蓝色建筑物，包含候车室，无人值守，但旁边相邻的深蓝色屋顶建筑物为秋田白神中央管理中心，内设有POS级，可以办理车票发售业务（窗口营业時間8時30分 - 17時00分）<ref name="station"/>。

秋田白神中央管理中心内设女性观光站长一名，“[[Resort白神号|Resort白神号]]”停车时会在站台上迎送乘客，并介绍周边观光信息。

== 使用情况 ==
2017年度日平均乘车人数24人。自2004年来，本站为JR东日本管辖范围内乘车人数最少的车站。
