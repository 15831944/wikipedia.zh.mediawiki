'''秋田自動車道'''（{{jpn|j=秋田自動車道|hg=あきたじどうしゃどう|rm=Akita jidōshadō|f=y}}）是起於[[日本|日本]][[岩手縣|岩手縣]][[北上市|北上市]][[北上系統交流道|北上系統交流道]]，經過[[秋田縣|秋田縣]][[秋田市|秋田市]]，終於[[秋田縣|秋田縣]][[鹿角郡|鹿角郡]][[小坂町|小坂町]][[小坂系統交流道|小坂系統交流道]]的[[日本高速公路|高速公路]]。簡稱為'''秋田道'''（{{lang|ja|秋田道／あきたどう}}、AKITA EXPWY）。

高速公路路線編號分開編號，在北上系統交流道 - 河邊系統交流道間編為{{Ja Exp Route Sign|E46}}，河邊系統交流道 - 小坂系統交流道與[[日本海東北自動車道|日本海東北自動車道]]共同編為{{Ja Exp Route Sign|E7}}。

== 概要 ==
於岩手縣北上市起與東北自動車道分岐，經秋田縣[[橫手市|橫手市]][[湯澤橫手道路|湯澤橫手道路]]、秋田縣[[秋田市|秋田市]][[河邊町|河邊]][[日本海東北自動車道|日本海東北自動車道]]連絡至秋田縣能代市的二井白神IC約180[[km|km]]的路線。收費區間由[[東日本高速道路|東日本高速道路株式會社]]（NEXCO東日本）管理、營業。
該自動車道為[[東北橫斷自動車道|東北橫斷自動車道]]釜石秋田線的北上JCT以西區間、[[日本海沿岸東北自動車道|日本海沿岸東北自動車道]]的[[河邊JCT|河邊JCT]]-[[秋田北IC|秋田北IC]]和[[昭和男鹿半島IC|昭和男鹿半島IC]]-[[琴丘森岳IC|琴丘森岳IC]]、[[國道7號_(日本)|一般國道7號]]秋田外環狀道路琴丘能代道路等總稱的道路名。

秋田自動車道於1997年時一度全線通車，2001年終點為昭和男鹿半島IC，[[琴丘能代道路|琴丘能代道路]]的琴丘森岳-[[八龍IC|八龍]]間於2002年3月30日通車時，合併[[日本海沿岸東北自動車道|日本海沿岸東北自動車道]]的通車區間以秋田自動車道的名稱來營運。未來[[日本海沿岸東北自動車道|日本海沿岸東北自動車道]]在全線通車時，河邊JCT以北的區間保留為秋田自動車道，而在[[日本海東北自動車道|日本海東北自動車道]]路線方面的變更依然部份不明。

秋田北IC-昭和男鹿半島IC間、琴丘森岳IC-二井白神IC間雖名義上為秋田自動車道，但正式名稱為一般國道7號。

=== 一般國道7號秋田外環狀道路 ===
'''秋田外環狀道路'''（{{lang|ja|秋田外環状道路}}、{{lang|ja|あきたそとかんじょうどうろ}}）是秋田北交流道至昭和男鹿半島交流道間長度9.5公里的高速公路（[[高規格幹線道路|高規格幹線道路]]）。

在日本海沿岸東北自動車道法定化之前，作為以秋田市為終點的東北橫斷自動車道釜石秋田線實質的秋田市外延長事業化與國道7號[[繞道|繞道]]（為東北橫斷自動車道[[高速自動車國道並行一般國道快速道路|並行一般國道快速道路]]）整備區間。1997年11月13日啟用。

=== 一般國道7號琴丘能代道路 ===
'''琴丘能代道路'''（{{lang|ja|琴丘能代道路}}、{{lang|ja|ことおかのしろどうろ}}）是從秋田縣山本郡三種町至能代市全長 33.8公里的快速道路。
同時本道路被升級為日本海沿岸東北自動車道並行國道7號之快速道路，八龍交流道~二井白神交流道間目前為免費道路。

==== 路線資訊 ====
* 起點 : 秋田縣[[山本郡|山本郡]][[三種町|三種町]]鹿渡
* 終點 : 秋田縣能代市二井町駒形
* 全長 : 33.8公里
* [[道路構造令|規格]] : 第1種第2級
* 設計速度 : 100[[km/h|km/h]]
* 車道數 : [[2車道高速公路|暫定2車道]]（完成4車道）
* 建設費用預定償還完畢時間 : 2025年3月27日

=== 一般國道7號大館西道路 ===
'''大館西道路'''（{{lang|ja|大館西道路}}、{{lang|ja|おおだてにしどうろ}}）是從秋田縣大館市櫃崎至大館市商人留全長 8.8公里的快速道路。

==== 路線資訊 ====
* 起點 : 秋田縣大館市櫃崎
* 終點 ： 秋田縣大館市商人留
* 全長 ： 8.8公里
* [[道路構造令|規格]] ： 第1種第2級、第1種第3級
* 設計速度 ： 100km/h、80km/h
* 道路寬度 ： 12.0m（暫定）
<!--* 車道寬度 ： 3.5m-->
* 車道數 ： [[2車道高速公路|暫定2車道]]（完成4車道），禁止超車（設有回復式導桿）

== 歷史 ==
* 1983年 : 一般國道7號琴丘能代道路事業化。
* 1991年7月25日 : 橫手IC-秋田南IC間通車，通過秋田市的首條高速道路開業。
* 1993年3月17日 : 一般國道7號琴丘能代道路的八龍IC-能代南IC間（4.1km）暫定2車道通車。
* 1994年8月4日 : 北上JCT-北上西IC間通車，與東北自動車道連接。
* 1994年11月12日 : 於橫手IC與湯澤橫手道路連接。
* 1995年11月10日 : 湯田IC-橫手IC間通車。
* 1997年7月23日 : 北上西IC-湯田IC間通車（秋田自動車道第1次全線通車）。
* 1997年11月13日 : 秋田南IC-秋田北IC間通車，同時一般國道7號秋田外環狀道路的秋田北IC-昭和男鹿半島IC間通車。
* 2001年7月7日 : 協和IC-秋田南IC間4車道化完成，同時河邊JCT通車與日本海東北自動車道連接。
* 2002年3月30日 : 一般國道7號琴丘能代道路的琴丘森岳IC-八龍IC間（13.0km）暫定2車道通車。
* 2002年4月10日 : 西仙北IC通車。
* 2002年9月28日 : 昭和男鹿半島IC-琴丘森岳IC間通車，同時設置琴丘森岳主線收費站。
* 2003年10月29日 : 大-{曲}-IC-西仙北IC/SA間4車道化。
* 2004年3月26日 : 西仙北IC/SA-協和IC間4車道化。
* 2005年10月1日 : [[日本道路公團|日本道路公團]]民營化，為東日本高速道路株式會社繼承。
* 2006年7月29日 : 一般國道7號琴丘能代道路的能代南IC-能代東IC間（6.7km）暫定2車道通車。
* 2007年8月12日：一般國道7號琴丘能代道路的能代東IC-二井白神IC間（10.0km）暫定2車道通車。
