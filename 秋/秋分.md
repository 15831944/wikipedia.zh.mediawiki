'''秋分'''，是[[节气|二十四节气]]之一 ，每年9月23日前后<small>（公曆9月[[晝夜平分點|晝夜平分點]]）</small>，太阳到达[[黄经|黄经]]180°时开始。《月令七十二候集解》：“八月中，解见秋分”、“分者平也，此當九十日之半，故謂之分。”分就是半，这是[[秋季|秋季]]九十天的中分点。

== 地球上的现象 ==
[[阳光|阳光]]几乎直射[[赤道|赤道]]，此后太阳直射点继续南移，故秋分也称'''降分'''。

秋分时，全球昼夜等长。秋分之后，北半球各地昼渐短夜渐长，[[南半球|南半球]]各地昼渐长夜渐短。秋分时，全球无[[极昼|极昼]][[极夜|极夜]]现象。秋分之后，[[北极|北极]]附近极夜范围渐大，[[南极|南极]]附近极昼范围渐大<ref>[http://astro.sina.com.cn/z/qiufen/ 24节气之秋分]，新浪网</ref>。

中國古籍《[[春秋繁露|春秋繁露]]、陰陽出入上下篇》中說：「秋分者，陰陽相半也，故晝夜均而寒暑平。」「秋分」的意思有二：一是太陽在這時到達黃徑180度，一天24小時晝夜均分，各12小時；二是按中國古代以立春、立夏、立秋、立冬為四季開始的季節劃分法，秋分日居秋季90天之中，平分了秋季。 秋分以後，氣溫逐漸降低，所以有「白露秋分夜，一夜冷一夜」、「一場秋雨一場寒，十場秋雨穿上棉」和「八月雁門開，雁兒腳下帶霜來」的說法。

秋分也是美好宜人的時節，風和日麗，秋高氣爽，丹桂飄香。春種秋收，[[唐|唐]]詩人李紳的《憫農》說春種一粒粟，秋收萬顆子，春天開花，秋天結果。秋分時節結果累累，螃蟹魚蝦肥美，金[[菊|菊]]正黃。秋分也是種麥正當時，農業生產上重要的節氣。南方此時才剛變冷，則可以繼續農作，延續到秋天結束[[霜降|霜降]]為止。

== 物候 ==
* 一候雷始收聲：鮑氏曰：[[雷|雷]]，二月陽中發聲，八月陰中收聲，入地則萬物隨入也。
* 二候蟄蟲培戶：淘瓦之泥曰壞，細泥也，按《[[禮記|禮記]]》注曰：壞，益其蟄穴之戶，使通明處稍小，至寒甚，乃墐塞之也。
* 三候水始涸：《禮記》注曰：「水本氣之所為」，春夏氣至，故長，秋冬氣返，故涸也。

§ 即不再會有夏季雷電交加的情況，春天出現的昆蟲亦會減少出現，而因水氣不再充沛，天氣亦會開始變得乾燥。

== 节气养生 ==
素問•四氣調神大論曰：「秋三月，早臥早起，與雞俱興。」起居作息應相應調整。

秋分時令養生食材有秋[[蟹|蟹]]，[[芝麻|芝麻]]，[[核桃|核桃]]，[[木耳|木耳]]，[[菊花茶|菊花茶]]等。

秋分之后，气温逐渐下降，加以秋季天气干燥，常出现凉燥，皮膚搔癢，应锻炼身体，多喝水，吃清润、温润的食物。因日照减少，人們的情緒未免有些垂暮之感，故有「秋風秋雨愁煞人」之說，故宜培养乐观精神，避免憂鬱或躁鬱纏身。有胃病的人应特别注意胃部的保暖。

== 节假日 ==“
秋分在[[日本|日本]]是[[日本節日|國定假日]]，稱為「秋分之日」。

自2018年起，[[中华人民共和国|中华人民共和国]]将每年农历秋分设立为[[中国农民丰收节|中国农民丰收节]]<ref>{{Cite web|url=http://www.gov.cn/zhengce/content/2018-06/21/content_5300129.htm|title=国务院关于同意设立“中国农民丰收节”的批复（国函〔2018〕80号）_政府信息公开专栏|accessdate=2018-06-21|work=www.gov.cn}}</ref>。
