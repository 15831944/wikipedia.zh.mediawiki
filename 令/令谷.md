== 生平 ==
令谷2011年毕业于[[北京大学国际关系学院|北京大学国际关系学院]]，后来成为[[北京大学教育学院|北京大学教育学院]]的学生<ref name="wsj" />。令谷起初保持了低调，仅对几个人透露了自己的背景<ref name="wsj" />。同学透露，他穿名牌服装，居住在私人的居所而不是宿舍，上课常常迟到、早退<ref name="wsj" />。

令谷曾仿照美国[[耶鲁大学|耶鲁大学]]的“[[骷髅会|骷髅会]]”，在[[北京大学|北京大学]]建立了一个俱乐部<ref name="wsj" />。为了掩人耳目，起名“战略及国际研究委员会”<ref name="wsj" />。

== 法拉利事件 ==
2012年3月18日凌晨4点，北京市[[海淀区|海淀区]][[保福寺桥|保福寺桥]]附近，一条因下雪而变得湿滑的环路上，发生一起重大车祸<ref name=":0">{{Cite web|url=http://epaper.bjnews.com.cn/html/2012-03/19/node_13.htm|title=2012年03月19日 星期一-A10版-北京新闻·社会|accessdate=2017-09-15|author=|date=2012-03-19|work=|publisher=新京报电子报}}</ref>，很多家国际媒体报道，令计划儿子令谷驾驶的一辆[[法拉利|法拉利]][[跑车|跑车]]严重损坏，车上一男两女，男子当场死亡，两女重伤送医<ref name=":0" /><ref>{{cite news|1=|author=|title=北京一辆法拉利跑车撞桥造成1人遇难2人重伤|url=http://v.26vv.cn/s_boosj/2581222.html|work=|publisher=[[辽宁卫视|辽宁卫视]]|date=|accessdate=2012-09-07|deadurl=yes|archiveurl=https://web.archive.org/web/20120914001211/http://v.26vv.cn/s_boosj/2581222.html|archivedate=2012-09-14}}</ref>。重傷的兩名女子皆為25歲、藏族，其中扎西卓瑪為[[青海省公安厅|青海省公安厅]]副厅长的女兒，畢業於[[中央民族大学|中央民族大学]]；另一位楊吉，為享有名聲的活佛女兒，畢業於[[中国政法大学|中国政法大学]]。後者燒傷重，曾被救活，但同年7-8月之間去世<ref name="wsj" /><ref>[http://yzzk.com/cfm/content_archive.cfm?id=1363610921872&docissue=2012-43 法拉利車禍藏族美女死亡內情]，紀碩鳴，[[亞洲週刊|亞洲週刊]]2012年43期</ref>。《[[南華早報|南華早報]]》爆料称，与时任中央政治局常委[[周永康|周永康]]关系密切的中石油高官[[蒋洁敏|蒋洁敏]]将数以千万元计的金钱从[[中国石油天然气集团|中石油集团]]汇进两名受伤女子的家属银行户口<ref>{{cite web |url = http://www.chinese.rfi.fr/中国/20130902-令计划儿子去年北京死亡车祸蒋洁敏曾被调查|title=令计划儿子去年北京死亡车祸蒋洁敏曾被调查|publisher=RFI|date=2013年9月2日 |language =zh-hans }}</ref>，让其封口<ref>{{cite web |url = http://www.dw.de/令计划哥哥令政策被中纪委调查/a-17722762 |title=令计划哥哥令政策被中纪委调查|publisher=德国之声 |date=2014-06-19 |language =zh-hans }}</ref>。

=== 信息审查 ===
{{See also|中华人民共和国网络审查|中華人民共和國新聞自由}}

中國政府当时封锁了所有关于此次车祸的信息。救护人员、媒体记者被噤声，令谷（化名王子云）在[[北京大学|北京大学]]的同班同学被告知，去了国外<ref name="wsj" />。互联网上一度删除了相关信息，也禁止相关讨论。

2012年3月19日，中共北京市委宣傳部主辦的《[[新京报|新京报]]》刊登了新闻《法拉利撞墙解体 三人被甩车外》，之前不能访问的新京报官网页面已经可以访问<ref name=":0" />。
