'''南昌市青山湖區豫章書院修身專修學校'''是[[中华人民共和国|中國]][[南昌|南昌]]一間於2013年成立，用以接收管教存在叛逆行為的青少年的學校，其前身是2007年建立的一家名為「龍悔學校」的戒[[網癮|網癮]]學校和2011年年底成立的'''豫章书院德育学校'''。书院的公开资料顯示书院重点开设青少年修身科课程，但学生们認為讲课较少，礼仪训练较多。許多學生表示學校設有一系列體罰政策來管理學生，高压甚至将学生逼迫至自杀<ref name="huanhuanshuo" />。此书院在經網路曝光後引發了輿論广泛关注。2017年底，該校已主动申请停止辦學。

== 成立背景 ==
豫章书院成立的前身是2007年建立的一家名為“龙悔学校”的戒[[网瘾|网瘾]]學校。<ref name="法制"/>2010年6月，吴军豹申请注册商标“豫章书院”。2011年年底，商標註冊成功，豫章书院德育学校在南昌市青山湖区罗家镇儒溪吴村正式成立，其师资主要来自龙悔学校。吴军豹对外宣称书院的成立，是[[豫章書院_(南昌)|歷史上豫章书院]]的“复学”，他希望“通过‘传统’的回归，唤醒社会风气的改良”，用国学启发“90后”“00后”<ref>[http://www.chinakongmiao.cn/templates/T_CatalogList/index.aspx?nodeid=67&page=ContentPage&contentid=5464 豫章书院：停办百年后复学]{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}.中国孔庙.[2011-12-23].</ref>。原南昌市市长李豆罗则为学校名誉校长，为书院背书<ref name="huanhuanshuo">{{cite web|author=缓缓说|title=从杨永信到豫章书院：你是有多恨孩子，才把他们送去戒网瘾？
|url=http://www.sohu.com/a/201044557_358485|website=搜狐新闻|date=2017-10-29|accessdate=2017-11-18}}</ref><ref name="rfa">{{cite web|title=“戒网瘾”学校再被曝光 江西“豫章书院”虐待学生被查
|url=http://www.rfa.org/mandarin/yataibaodao/meiti/yf2-11012017102143.html|website=自由亚洲电台-普通话|date=2017-11-11|accessdate=2017-11-20}}</ref>。2013年5月16日，豫章书院修身教育学校在距离吴村不到10分钟路程的万村正式成立<ref name="澎湃">{{cite web|author1=何利权|author2=胡芮默|title=豫章书院被体罚学生报案家长不支持，警方称调查困难但在查|url=http://www.thepaper.cn/newsDetail_forward_1868920|website=澎湃新闻|accessdate=2017-11-18}}</ref>，其資歷為民办非学历教育机构，学生们則搬至新校区。2014年1月，書院獲得了增加一般不良行为青少年转化的工作职能。同年2月23日，書院获得了江西省南昌市西湖区[[检察院|检察院]][[未成年人|未成年人]]观护帮教基地的挂牌，并成为江西省首家未成年人观护帮教基地。<ref name="法制"/>

== 課程設置 ==
根據书院的公开资料，书院重点开设青少年修身科课程，涉及书院历代学约、学训乃至养正、教女、训俗等经典学习。学生提供的教科书中有“教女遗规”、“修身科”、“心理禅”等书籍。但学生们認為“真正讲课比较少，一些坐姿、站姿等礼仪训练比较多。”<ref name="鳳凰"/>此外，《[[中華人民共和國预防未成年人犯罪法|中華人民共和國预防未成年人犯罪法]]》规定，工读学校应当加强法制教育的内容；不得体罚、虐待和歧视学生。但該校一名學生表示在里面“没有法制课。”<ref name="新京報"/>

== 學校管理 ==
曾在豫章书院担任老师和教官职务的周先生向《[[法制晚报|法制晚报]]》证实，新生在入学后，会被迫进入“烦闷室”（學生稱“小黑屋”）关足7天，在此期間，学生無法外出，一日三餐由工作人员送进去。<ref name="法制"/>書院的一名學生稱，煩悶室是一間“大概10多平的小黑屋，有些发霉的绿色军被，褥子旁边就是一个蹲坑”。豫章书院官方解释称其為“[[森田疗法|森田疗法]]”（森田疗法主张对精神病人隔离静养）。<ref name="法制"/>从烦闷室出来，新生就開始正常上课、生活。早晨五点半，学生们开始新的一天，他們先拜孔子、进行“晨仪”，再進行学习。晚上時，書院會進行考德，對大家的表现進行点评，犯错误者會受到體罰。<ref name="新京報">{{cite web|title=关禁闭、打戒尺、打龙鞭，豫章书院管理层竟多数来自网瘾学校|url=https://mp.weixin.qq.com/s/3GEBepe7ACmSiYNKDoS84Q|publisher=新京報|accessdate=2017-11-08}}</ref>學生依問題嚴重程度會遭到程度不一的體罰，有时是两戒尺，有时候则是“龙鞭”。多数学生确认“龙鞭”的材质是[[钢筋|钢筋]]，亦有說法是[[玻璃钢|玻璃钢]]，并有人晒出被打至紫黑色的臀部照片<ref name="jndsb">{{cite web|title=豫章书院体罚学生事件再升级 家长详细爆料全过程
|url=http://jx.sina.com.cn/zimeiti/2017-11-04/detail-ifynmzrs6973044.shtml|website=新浪江西|publisher=江南都市报|date=2017-11-04|accessdate=2017-11-19}}</ref>。然而书院的最高管理者吴军豹表示，“龙鞭”只是空心的[[塑料|塑料]]管。<ref name="法制"/>學校有一套类似於[[中国古代行政监察制度|中国古代行政监察制度]]的管理办法。不同管理级别的人有着不同的权限，还有一系列的处罚上报程序。<ref name="法制"/>學生伙食极为恶劣，除不能保证足够的蛋白质及其它营养外，甚至餐食鍋中混有杂物抹布，乃至食物中毒等事件也时有发生<ref name="huanhuanshuo" />。除此之外，学生会有超负荷劳动，生存条件堪比集中营<ref name="huanhuanshuo" />。

為了防止学生说出不利于学校的话，学生跟家人打电话時，都会有老师在旁边看着。学生为了能尽早出去，会竭尽全力地说书院的好话<ref name="觀察者網">{{cite web|author=黎娜|title=山长、龙鞭、密告：南昌百年豫章书院里的“权力游戏”|url=http://www.guancha.cn/society/2017_11_02_433316_3.shtml|website=觀察者網|publisher=北京青年报|date=2017-11-02|accessdate=2017-11-08}}</ref>。同時，書院的一名學生表示，為了應付家长参观，學生必須“演”，來製造和谐的景象，學生如果不配合，就會挨打。而家长来问學校情況時，有學生也表示不敢说实情，怕是老师让家长来问的，從而遭受虐打。<ref name ="houhui">{{cite web|author1=时鑫|title=送孩子进豫章书院的家长:这是我最后悔的决定|url=http://news.sina.com.cn/s/pa/2017-11-02/doc-ifynnnsc4163539.shtml|website=新浪|publisher=成都商報|accessdate=2017-11-08}}</ref>

== 学生自杀 ==
因为在高压下，学生无力反抗，部分人绝望至[[自残|自残]]、[[自杀|自杀]]。自残者最初希望博得教官同情，然而往往招致一顿“龙鞭”的处罚<ref name="diaocha">{{cite web|author1=何利权|author2=胡芮默|title=南昌豫章书院学生陆续被接走，警方正调查书院是否涉嫌违法|url=http://www.thepaper.cn/newsDetail_forward_1855676|website=澎湃新闻|accessdate=2017-11-18}}</ref>。而由于金属器具的管制，大多数企图自杀者通过吞服[[牙膏|牙膏]]或者[[洗衣液|洗衣液]]自杀<ref name ="houhui" /><ref>{{cite web|title=独家专访引爆豫章书院事件当事人：曾被铁棍打 吞牙膏自杀未遂|url=https://v.qq.com/x/cover/u14ize2tlofphve/k056764cn0m.html|publisher=腾讯视频|accessdate=2017-11-19}}</ref>。然而校方在发现自杀并送至医院后，却不敢在病危通知书上签字承担责任，于是将学生带回书院，通过灌水催吐的粗糙方式[[洗胃|洗胃]]，甚至洗至学生吐血<ref>{{cite web|author1=杨宝璐 曹慧茹 郭蒙|title=山长、龙鞭、密告：南昌百年豫章书院里的“权力游戏”
|url=http://www.thepaper.cn/newsDetail_forward_1847601|website=澎湃新闻|publisher=北京青年报|accessdate=2017-11-19}}</ref>。

== 停止辦學 ==
2017年，[[知乎|知乎]]用户“温柔”发表帖子“中国到底有多少个[[杨永信|杨永信]]”，爆料南昌豫章书院修身教育专修学校存在问题后，豫章書院引发了輿論广泛关注。<ref name="鳳凰">{{cite web|author1=焦俊杰|author2=黄思农|title=南昌豫章书院深陷“暴力门” 学生称“绝望”|url=http://jx.ifeng.com/a/20171031/6110398_0.shtml|website=鳳凰網|publisher=中國江西網|accessdate=2017-11-08}}</ref>南昌市青山湖区有關部门调查表示，帖子反映的问题部分确实存在，並责成区教科体局对该校教育机构和相关责任人进行处罚。同年10月30日，豫章书院执行山长吴军豹在朋友圈回应称：“全体师生于今日正式宣告彻底停用戒尺管教。”<ref name="鳳凰"/>11月2日，吴军豹建立媒体[[微信|微信]]群，发布豫章书院申请停办的消息，称因“戒尺”等古代教育方法不能容于现行教育制度，另外，学生对象特殊，停止“戒尺”后会置老师于危险之地，豫章书院修身学校已于当天主动申请停办，待政府部门批准后，进行在校生逐步分流。<ref>{{cite web|title=江西南昌豫章书院被查实严重体罚学生 已申请停办|url=http://www.zaobao.com.sg/realtime/china/story20171103-808137|website=联合早报网|accessdate=2017-11-08|date=2017-11-03}}</ref>11月7日，南昌青山湖区委宣传部相关负责人表示，豫章书院办学资格已经注销。並根据有关规定，责令書院在一个月内妥善安置在校学生和老师，同時，負責人表示公安部门已介入此事<ref name="diaocha" />，然而并无后续回应<ref name="rfa" />。雖然一些孩子的父母對送孩子來書院學習后悔不已<ref name="鳳凰"/>，然而，豫章书院举行媒体开放日時，一些自称学生家长的人在校门口拉起横幅，呼吁复学。<ref name="法制">{{cite web|author1=朱健勇|title=豫章书院办学资格将被注销 部分家长：坚决支持办学|url=http://www.kanfanews.com/pc/index/article/1005035|website=看法新聞|publisher=法制晚報|accessdate=2017-11-08}}</ref>

2017年12月7日，南昌市公安局青山湖分局对学校的相關報案進行非法拘禁罪立案侦查。<ref>{{cite news|url=http://www.cqcb.com/headline/2017-12-08/589884_pc.html|title=豫章书院涉嫌非法拘禁学生，江西警方立案侦查|publisher=红星新闻|date=2017-12-08|accessdate=2017-12-09}}</ref>2018年1月[[央視|央視]]專題調查報導豫章書院專修學校問題，並實地拍攝書院傳聞中的黑屋和餐廳、戒尺龍鞭等，司法專家於節目中表示其獲得的是國學[[補習班|補習班]]的辦學證書和行為不當青少年的矯正服務證書，其關押行為違反[[中华人民共和国刑法|刑法]]，戒尺龍鞭等毆打違反《[[中华人民共和国未成年人保护法|未成年人保護法]]》，而其所謂日本[[森田疗法|森田疗法]]更是荒誕不經，張冠李戴，且該療法百年前被提出是針對精神病患，後來也證明無效，且書院沒有醫療資格證書。報導表示當地縣政府有疏於督導之責，核發辦學證書後就放任不管，未來社會上將成立第三方民間評價機構，對私人辦學進行民間檢查增強官方之外的第二股力量。<ref>[https://www.youtube.com/watch?v=Dgsv9vuWyaM 央視-问题学校系列之书院漩涡 ]</ref>
