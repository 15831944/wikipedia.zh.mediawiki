{{Vgname|1=汪達與巨像|en=Shadow of the Colossus|ja=ワンダと巨像|tw=汪達與巨像|official-cn=巨像之咆哮}}是由[[蓝点游戏工作室|蓝点]]与[[索尼互动娱乐|索尼互动娱乐]]共同开发的[[动作冒险游戏|动作冒险游戏]]，属于[[ICO小组|ICO小组]]系列作品，运行于[[PlayStation_4|PlayStation 4]]平台。该作是以[[Playstation_2|Playstation 2]]上[[汪达与巨像|原版游戏]]为基础，对场景与角色进行艺术重制的新作。重制由Bluepoint领导，其也参与了之前，与SIE Japan工作室合作的Playstation 3上的高清版的重制工作。虽然游戏模型皆进行了重新制作，但尽量保持了原作的游玩风貌，仅引入了新的控制器映射方案，且玩家可以自行选择。

该作受到评论家的好评，主要在于保留了原版的艺术感觉，同时在新一代主机上加强了模型表现和控制方案；不过在游戏玩法与摄像机视角上，也有批评声音，认为其仍存在改善空间<ref>{{Cite web |url = https://game.udn.com/game/story/10451/2974005 |title = 萊斯垃圾話：《汪達與巨像》十二年前的神作經典重現 |author =Lice萊斯  |date = 2018-02-07  |website= [[聯合新聞網|聯合新聞網]] |language = zh-cn |accessdate = 2018-09-03 }}</ref>。

== 开发与发行 ==
索尼在[[E3_2017|2017年电子娱乐博览会]]新闻发布会上，宣布了PS4版重制汪达与巨像的消息<ref>{{cite web | url = https://www.destructoid.com/shadow-of-the-colossus-is-getting-a-remake-441922.phtml | title = Shadow of the Colossus is getting a remake | first = Nick | last = Valdez | date = June 12, 2017 | accessdate = June 12, 2017 | work = [[Destructoid|Destructoid]] | deadurl = no | archiveurl = https://web.archive.org/web/20170617093936/https://www.destructoid.com/shadow-of-the-colossus-is-getting-a-remake-441922.phtml | archivedate = June 17, 2017 | df =  }}</ref>。 重制工作由Bluepoint领导，其之前也与索尼第一方工作室保持合作关系，一同推出过多款重制游戏，包括了PS3版的汪达与巨像 Remaster。重制该作的想法实在蓝点内部的一次开发人员对话中产生，他们希望在新的主机平台上创造一个决定性的版本，在与SIE Japan交换意见后，该想法落实为具体项目。 蓝点的制作人Randall Lowe与技术总监Peter Dalton在接受''[[Game_Informer|Game Informer]]''采访时表示，最初版本的《汪达与巨像》 在许多员工心中，是列于『有史以来最好的五款游戏』的存在<ref name="GI Interview">{{cite web | url = https://www.youtube.com/watch?v=ec6t1xyuLzU | title = How Bluepoint Rebuilt Shadow Of The Colossus For PS4 | date = December 15, 2017 | accessdate = January 2, 2018 | work = [[Game_Informer|Game Informer]] | deadurl = no | archiveurl = https://web.archive.org/web/20180131002352/https://www.youtube.com/watch?v=ec6t1xyuLzU | archivedate = January 31, 2018 | df =  }}</ref>。

开发者从头开始，重新制作了游戏模型，但游戏保留了与原始版本相同的玩法，并且引入了新的针对DS4手柄的控制方案<ref name = "GI Interview" />。游戏使用了Playstation 2游戏的原始代码库，艺术部分工作人员借Playstation 2版本来创建与PlayStation 4版本相同的游戏体验<ref name=":1">{{cite web | url = http://www.eurogamer.net/articles/2017-06-29-fumito-ueda-submitted-a-proposal-of-changes-for-the-shadow-of-the-colossus-ps4-remake | title = Fumito Ueda submitted a proposal of changes for the Shadow of the Colossus PS4 remake | first = Robert | last = Purchase | date = June 29, 2017 | accessdate = July 3, 2017 | work = [[Eurogamer|Eurogamer]] | deadurl = no | archiveurl = https://web.archive.org/web/20170701115039/http://www.eurogamer.net/articles/2017-06-29-fumito-ueda-submitted-a-proposal-of-changes-for-the-shadow-of-the-colossus-ps4-remake | archivedate = July 1, 2017 | df =  }}</ref>。

==特典与彩蛋==
===特典道具===
游戏为先期预订者提供了游戏内道具『生命之剑』，可以随时间回复生命值。预订用户还会获得一款PS4静态主题。

豪华版则提供了若干道具，包括『遠古之弓』『命運斗蓬』与『斑紋馬』；除了特典道具，其余特殊道具均可在游戏内，随流程而解锁<ref name="remind" />。

===彩蛋===
重制版在其中设置了多个[[彩蛋_(媒体)|彩蛋]]，与ICO小组其他游戏相关，或为向游戏社区致敬：
* 游戏内的一个洞窟有木桶道具，与[[最后的守护者|最后的守护者]]联动
* 游戏内B5区域海滩，有西瓜，与[[ICO_(游戏)|ICO (游戏)]]联动
* 游戏内存在不少于4个山羊壁画，意义不明
* 游戏内存在发光的收集物，一共有79枚，可以用于解锁道具『多爾暝之劍』

发光收集物在玩家群体中可能称作硬币(Coin)、遗物(Relic)或者启示(Enlightenment)。游戏媒体一般事先得到由发行商递交的游戏，用于撰写评测，[[IGN|IGN]]的Marty Sliva在2018年1月24日撰文，表示游戏中存在一些发光的硬币，他猜测与一些隐藏内容相关<ref>{{cite web | url = http://www.ign.com/articles/2018/01/24/the-last-guardian-easter-egg-found-in-shadow-of-the-colossus-remastered | title = The Last Guardian Easter Egg Found in Shadow of the Colossus Remastered |  accessdate = 2018-09-03 | website = [[IGN|IGN]] | author=Marty Sliva | language=en }}</ref>。由于艺术设定集中对开发背景的介绍，社区一般认为PS2的初始版本做好了24个巨像，但最终隐藏了8个；硬币是否对应、或是PS4重制版是否有重构巨像，成为社区的关注点<ref>{{cite web | url = https://www.gcores.com/articles/96598 | title = 深埋在《汪达与巨像》重制版中的一些神秘线索 |  accessdate = 2018-09-03 | website = 机核 |author=之乎 | language=zh }}</ref>。

由于实体版游戏光碟铺货早于数字版，部分玩家先行得到了游戏光碟，于是大面积的搜索先于游戏的宣称发行日即展开。2月7日，[[福布斯|福布斯]]网站发表文章，当时玩家协力已经找到了70枚硬币，文章认为『硬币解锁新巨像』的期待不切实际<ref>{{cite web | url = https://www.forbes.com/sites/davidthier/2018/02/07/enlightenments-theres-a-new-secret-hiding-in-shadow-of-the-colossus-but-what-is-it/#52fc8c3a21d1 | title = Don't Expect The Secret Gold Coins Hiding In 'Shadow Of The Colossus' To Unlock A New Colossus |  date=2018-02-07|accessdate = 2018-09-03 | website = [[福布斯|福布斯]] | author=Dave Thier
 | language=en }}</ref>，并且指出，游戏结尾字幕提示'''Nomad Colossus and the 79 steps to enlightenment'''，可能对应硬币总量为79枚，且是向Nomad Colossus博客致敬。

2月9日，Youtube频道PS4Trophies发布视频，宣告完成搜索，找到了全部79枚硬币位置，并解锁了新道具<ref>{{cite web | url = https://www.youtube.com/watch?v=awFlJLIUCgw | title = Shadow of the Colossus - All 79 Relic (Coin) Locations, Secret Underground Area and Sword of Dormin |  date=2018-02-09|accessdate = 2018-09-03 | website = [[Youtube|Youtube]] | author=PS4Trophies | language=en }}</ref>。许多游戏社区转载该视频，[[IGN|IGN]]发布文章，宣告全民狂欢划上完美句号<ref>{{cite web | url = http://www.ign.com/articles/2018/02/09/mystery-of-shadow-of-the-colossus-secret-coins-revealed | title = Mystery of Shadow of the Colossus' Secret Coins Revealed |  date=2018-02-09|accessdate = 2018-09-03 | website = [[IGN|IGN]] | author=Brian Barnett | language=en }}</ref>。
