'''汪锦元'''（{{bd|1909年||1992年|3月26日}}）原名'''汪国梁'''，[[江苏|江苏]][[苏州|苏州]]人，[[中国共产党|中国共产党]]情报人员<ref name=wodi>{{Cite web |url = http://cpc.people.com.cn/GB/64093/67507/5191020.html |title =  共产党人中的著名卧底英雄 (2)|author =王树人  |date =2006-12-20  |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
汪锦元少年时，因父亲早逝，被母亲（[[日本|日本]]裔）送到日本。1929年随母亲到[[上海|上海]]谋生，并且更名为汪锦元。后来受日本籍[[中共|中共]]党员[[西里龙夫|西里龙夫]]等人影响，于1936年加入[[中国共产党|中国共产党]]。1938年初，汪锦元奉命打入伪上海市政府（先后为[[上海市大道政府|上海市大道政府]]、[[督办上海市政公署|督办上海市政公署]]、[[上海特别市政府_(日占时期)|上海特别市政府]]）秘书处。因其有一半日本血统，且日语熟练，办事负责，所以深受[[汪精卫|汪精卫]]私人秘书[[周隆庠|周隆庠]]的信任。1940年9月，汪锦元因周隆庠举荐去[[南京|南京]]，打入“汪公馆”，担任汪精卫的随从秘书兼日语翻译。1940年到1942年的两年多时间，汪锦元随汪精卫参加了同日本人的一些会谈。汪锦元搜集[[汪精卫政权|汪精卫政权]]和日本“交易”的各类情报，例如汪精卫同日本签订的密约《日支新关系调整纲要》，汪精卫从日本政府获得的武器，汪精卫处往来人员的情况等。这些情报被汪锦元及时送交中共在南京的情报小组，由上海情报部门经秘密电波传给[[延安|延安]]，获得[[周恩来|周恩来]]称赞<ref name=wodi/>。

1942年8月，汪锦元等人因受西里龙夫牵连而被日本警方逮捕。1945年5月，汪锦元等人获释后，随[[新四军|新四军]]联络部部长[[杨帆_(中国政治人物)|扬帆]]抵达新四军淮南根据地。经审查及上级组织批准，汪锦元恢复了中共的组织关系。1945年9月，汪锦元奉命到[[国统区|国统区]]长期潜伏，相机打入[[中国国民党|国民党]]机关，开展隐蔽斗争。从此汪锦元与中共党组织失去联系。1949年[[中国人民解放军|中国人民解放军]]取得[[上海战役|上海战役]]胜利后，汪锦元曾在东方经济研究所和保卫部门工作。1955年夏，他因[[潘汉年|潘汉年]]、扬帆案（“潘扬案”）而被捕，关押在[[北京|北京]]。1982年8月，潘汉年、扬帆案彻底平反，汪锦元也获得平反。有关部门对汪锦元的评价是：“在从事党的情报工作期间表现积极，认真负责，对革命事业有一定贡献。”<ref name=wodi/>

1992年3月26日，汪锦元病逝<ref name=wodi/>。
