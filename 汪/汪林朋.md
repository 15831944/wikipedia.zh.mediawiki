==生平==
出生于[[湖北省|湖北省]][[黄冈市|黄冈市]][[罗田县|罗田县]]<ref name=“湖北籍富豪名录看看黄冈的你认识几个”>{{cite web | url=http://www.sohu.com/a/136348382_251143 | title=湖北籍富豪名录看看黄冈的你认识几个 |publisher=搜狐网|}}</ref>，1990年毕业于[[北京工商大学|北京工商大学]]会计系，中共党员<ref name=“汪林朋：打响黄冈楚商品牌”>{{cite web | url=http://rb.hgdaily.com.cn/html/2016-05/27/content_230579.htm?div=-1 | title=汪林朋：打响黄冈楚商品牌 |publisher=黄冈日报|}}</ref>，曾任职于中华人民共和国商业部<ref name=“汪林朋：打响黄冈楚商品牌”>{{cite web | url=http://rb.hgdaily.com.cn/html/2016-05/27/content_230579.htm?div=-1 | title=汪林朋：打响黄冈楚商品牌 |publisher=黄冈日报|}}</ref>，1999年，年仅30岁的汪林朋临危受命，接管居然之家，从机关走进企业，开启掌管居然之家的家居生涯<ref name=“十大领袖人物”>{{cite web | url=http://home.163.com/15/1211/11/BAI4GTP900104MEE.html | title=十大领袖人物|publisher=网易家居|}}</ref><ref name=“居然之家汪林朋:家居流通业的食物链”>{{cite web | url=http://home.ifeng.com/special/ifengmingjia-wlp/ | title=居然之家汪林朋:家居流通业的食物链|publisher=凤凰网|}}</ref>。
