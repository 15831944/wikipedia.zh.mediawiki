'''汪志敏'''（{{bd|1922年|||}}）原名'''汪文源'''，化名'''汪楚'''，[[山东|山东]][[临清|临清]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[国务院机关事务管理局|国务院机关事务管理局]]副局长，[[中国宋庆龄基金会|中国宋庆龄基金会]]秘书长。

==生平==
1938年秋加入中国共产党。历任卫东县第一、二区委书记兼游击队政委、县委委员、补充营副教导员兼分支书记、补充团组织股长、旅政治部民运队长兼政委、[[中国人民解放军|中国人民解放军]]五兵团十六军四十六师政治部科长。1949年调[[贵州省|贵州省]]，历任中共[[织金县|织金县]]委书记、中共[[赫章县|赫章县]]委书记兼县长<ref name=linnan/>（1950年1月任县长）。根据[[中共中央|中共中央]]、[[中共中央西南局|中共中央西南局]]、[[中共贵州省委|中共贵州省委]]、[[贵州省军区|贵州省军区]]的指示，中共赫章县委、县人民政府成立了汪志敏、李超尘为主的剿匪指挥部<ref>{{Cite web |url =http://www.chinamzw.cn/qxbpd/ShowArticle.asp?ArticleID=11159  |title =赫章苗族的清匪反霸斗争  |author =罗国锦  |date = 2013-08-03 |publisher = 黔西北频道 |language =  |accessdate =  |quote =  }}</ref>。1952年调任[[中央人民政府第二机械工业部|中央人民政府第二机械工业部]]工程处副处长兼总支书记。1965年任[[中华人民共和国水利电力部|中华人民共和国水利电力部]]第二工程局党委书记，后任[[中华人民共和国水利部|水利部]]办公厅副主任、主任。“[[文革|文革]]”时受错误审查，停职两年。1978年调任[[中国人民解放军基本建设工程兵|中国人民解放军基本建设工程兵]]办公室副主任。1980年任[[国务院机关事务管理局|国务院机关事务管理局]]副局长，1982年任[[中国宋庆龄基金会|中国宋庆龄基金会]]秘书长。1986年离休<ref name=linnan>{{Cite web |url = http://www.linqing.gov.cn/2015/0702/wNMDAwMDAyNTUwNw.html |title = 临南革命斗争回忆 |author =  |date = 2015-07-02 |publisher = 中国临清|language =  |accessdate =  |quote =  }}</ref>。
