'''王瑞振'''（{{bd|1953年|12月6日|2017年|2月16日}}），[[筆名|筆名]]'''汪笨湖'''，生於台灣臺南縣安定鄉（現[[台南市|台南市]][[安定區_(台灣)|安定區]]），[[作家|作家]]、媒體工作者、政論[[名嘴|名嘴]]。

== 早年及寫作生涯 ==
出身地方望族，國中時看了[[索忍尼辛|索忍尼辛]]的《[[古拉格群島|古拉格群島]]》後在[[週記|週記]]寫下「[[蔣介石|蔣介石]]是獨裁者」引起校方注意，顯露叛逆性格。家族原希望他習醫，但因個人無意願，執意選讀[[哲學|哲學]]系。退伍後接掌家族事業，卻因經營失敗而於1984年至1987年間因違反[[票據法|票據法]]入獄，在獄中開始寫作，作品以「汪笨湖」[[筆名|筆名]]陸續投稿《[[中國時報|中國時報]]》「人間」[[副刊|副刊]]受到好評，以《落山風》、《[[嬲|嬲]]》、《那根所有權》、《三字驚》、《長江有愛》、《口令：來談戀愛的》等多部小說成名，作品以描寫鄉土氣息及人性情慾見長。其《廈門新娘》、《[[草地狀元|草地狀元]]》與《台灣豪門爭霸記》已被改編為[[電視劇|電視劇]]。《廈門新娘》曾經是[[中視|中視]]八點檔連續劇，電視劇《[[草地狀元_(電視劇)|草地狀元]]》曾經是[[華視|華視]]八點檔連續劇。《台灣豪門爭霸記》被改編為華視八點檔連續劇《舊情綿綿》。電影《陰間響馬／吹鼓吹》<ref>{{cite web|url=http://www.books.com.tw/exep/prod/dvd/dvdfile.php?item=D020010926|title=陰間響馬/吹鼓吹 DVD|work=博客來|access-date=2008-02-20|archive-url=https://web.archive.org/web/20070218214923/http://www.books.com.tw/exep/prod/dvd/dvdfile.php?item=D020010926|archive-date=2007-02-18|dead-url=yes}}</ref>是部1988年分別由[[何平_(台灣導演)|何平]]、[[李道明|李道明]]導演的兩段式電影，皆改編自汪笨湖《落山風》裡的二短篇小說〈陰間響馬〉、〈吹鼓吹〉。《[[嬲|嬲]]》被改編成[[電影|電影]]《[[阿爸的情人|阿爸的情人]]》，1995年上映。

== 進入電視圈 ==
汪笨湖出獄後曾一度浪跡[[日本|日本]]及[[中國大陸|中國大陸]]求發展，但在均無發展機會的情況下返台，後在[[小野|小野]]引薦下與[[中央電影公司|中央電影公司]]簽約開始其演藝圈生涯，長期於電視、電影界擔任編劇、企劃、製作、顧問等幕後職位，曾擔任[[三立電視|三立電視]]戲劇監製，也曾擔任[[年代MUCH台|年代MUCH台]]執行董事。

入主[[年代MUCH台|年代MUCH台]]後，2002年底，汪笨湖開始主持《台灣心聲》。《台灣心聲》標榜「100％本土原味政論節目，抓妖、嗆聲、說真話，檢驗所有政治人物，走透台灣基層，跟人民借膽，向[[中華民國總統府|總統府]]發聲」。汪笨湖以所謂的「草地人」模式主持節目：大量使用鄉土[[俚語|俚語]]，以故事性表達立場，構成其個人魅力。汪笨湖在節目中塑造「本土意識」，在台灣中、南部起家，在北部也得到一定觀眾支持。該節目於2005年12月3日停播，理由並未公開。但是仅仅幾天之內，同為親綠色彩的周玉蔻之《新台灣高峰會》也遭停播。

前《[[Taiwan_News|Taiwan News]]》社長[[楊憲宏|楊憲宏]]認為，《台灣心聲》「在年代頻道消失，不等於《台灣心聲》走向末路，也可能是《台灣心聲》另一個生命旅程的開始。主持人汪笨湖是《台灣心聲》的創意原始，這個節目『用[[台灣話|台灣話]]說國家大事』，是許多視台灣話為國語的人心中的驕傲。……《台灣心聲》在汪笨湖的戲劇化手法下，讓許多台灣人感受到台灣話的真善美，而且發現用台灣話論政更有力量<ref>{{cite news|url=http://www.ettoday.com/2005/11/10/301-1867277.htm|archiveurl=https://web.archive.org/web/20070929211538/http://www.ettoday.com/2005/11/10/301-1867277.htm|title=看台灣心聲停播|author=楊憲宏|date=2005-11-10|archivedate=2007-09-29|agency=ETToday|deadurl=yes}}</ref>。」

2004年夏季，汪笨湖曾經應當時的[[中華電視公司|中華電視公司]]總經理[[江霞|江霞]]的邀請，主持[[華視|華視]]每週一至週五晚間八點至九點播出的政論節目《台灣起動》，由'''年代網際事業股份有限公司'''製作，風格類似《台灣心聲》。2005年起，《[[全民大悶鍋|全民大悶鍋]]》有一單元以《台灣起動》為模仿對象。2005年12月4日，汪笨湖宣布，他主持的《台灣心聲》與《雞婆新聞》於該日「正式全面停播」。

[[2006年5月|2006年5月]]，汪笨湖推出《笨湖網路開講》<ref>{{cite web|url=http://www.wangbenhu.com.tw/forum.asp|archiveurl=https://web.archive.org/web/20130105040838/http://www.wangbenhu.com.tw/forum.asp|title=汪笨湖─福爾摩沙的私家偵探|archivedate=2013-01-05|deadurl=yes}}</ref>。

2007年11月12日，汪笨湖主持的[[台灣藝術電視台|台灣藝術電視台]]政論節目《笨湖開講》開播，標榜「絕不相罵，也絕不作假」，暗批[[三立新聞台|三立新聞台]]政論節目《[[大話新聞|大話新聞]]》攻訐[[李登輝|李登輝]]與[[台灣團結聯盟|台灣團結聯盟]]。2007年11月6日，汪笨湖說，民進黨既然是執政黨，本來就應該要接受全民監督；「因為你執政，你做不好，被人家批評，是會怎樣？批評民進黨，批評[[陳水扁|陳水扁]]，並不代表你反對台灣，反而更愛台灣。」汪笨湖又說：「我看不起陳水扁動用那麼多力量，包括用他的媒體、他的打手，聯合欺負一個台灣民主老人。你（陳水扁）當初需要民主老人（的時候），陳水扁，你（是）怎麼捧李登輝（的）？<ref>{{cite news|url=http://www.ettoday.com/2007/11/06/11453-2183347.htm|archiveurl=https://web.archive.org/web/20080111053414/http://www.ettoday.com/2007/11/06/11453-2183347.htm|title=汪笨湖打扁幫李討公道　自爆新節目王文洋資金相挺|author=楊琇惇、劉世森|date=2007-11-06|archivedate=2008-01-11|agency=ETToday|deadurl=yes}}</ref>」

2011年9月起，在[[王文洋|王文洋]]的資金援助下，汪笨湖擔任總部位於[[台南市|台南市]][[安平區|安平區]]的台灣番薯電視台（台灣番薯傳播事業股份有限公司）[[董事長|董事長]]，並於該台主持多個政論性節目。但因開播一年多以來虧損連連，以及在評論中批評王文洋胞妹[[王雪紅|王雪紅]]等因素，2012年12月17日遭無預警解除董事長職務，王文洋表示任何媒體以傷害別人來賺錢，他是作不下去。

2017年2月16日，成大醫院代發汪笨湖家屬聲明，汪笨湖因病住進成大醫院加護病房，於當日9時35分病逝，家屬隨侍在側，享年63歲；由於汪笨湖生前曾簽署「安寧療護意願書」，在無力回天時拒絕[[心肺復甦術|心肺復甦術]]、[[去顫|心臟電擊術]]、[[氣管切開術|氣管切開術]]等無效的急救，以珍惜國家醫療資源，並享有尊嚴，辭別人世<ref>{{cite news|url=http://www.appledaily.com.tw/appledaily/article/headline/20170217/37554406|title=本土政論先驅 汪笨湖63歲辭世|agency=蘋果日報|date=2017-02-17}}</ref><ref>{{cite news|url=http://www.appledaily.com.tw/realtimenews/article/politics/20170216/1057088/|title=【63歲病逝】汪笨湖自簽放棄急救書 不願浪費社會資源|date=2017-02-16|agency=蘋果日報}}</ref>。

==政治活動==
汪笨湖早期的作品有著強烈的[[大中華主義|大中華主義]]情結，但是後來轉為[[臺灣獨立|臺灣獨立]]的支持者。

[[2006年中華民國直轄市市長暨市議員選舉|2006年中華民國直轄市市長暨市議員選舉]]選前2天，汪笨湖與黃光芹站台時，跟台下的民眾說：只要[[陳菊|陳菊]]當選[[高雄市|高雄市]]市長，他們就跳[[愛河|愛河]]。2006年12月25日，陳菊宣誓就任高雄市長；同日，汪笨湖與黃光芹身穿[[泳衣|泳衣]]，跳入愛河水中，以實踐其諾言。兩人被拉上岸之後，汪笨湖說：也希望[[陳水扁|陳水扁]][[總統|總統]]實踐自己的諾言，來高雄買房子，搬來高雄定居<ref>{{cite news|url=http://news.tvbs.com.tw/politics/340049|title=陳菊當選我跳河！汪笨湖跳河履約|author=李葆仁|date=2006-12-25|agency=TVBS}}</ref>。

[[民進黨|民進黨]][[2008年中華民國總統大選|2008年中華民國總統大選]]黨內初選前兩天，2007年5月3日，汪笨湖召開[[記者會|記者會]]力挺[[游錫堃|游錫堃]]、批評[[謝長廷|謝長廷]]。汪笨湖說：「民進黨不要再騙台灣人！民進黨是不是吃定了台灣人一定要投給你們？沒有那麼一回事！（謝長廷）『憲法一中』跟[[馬英九|馬英九]]『馬統一』，有什麼差別？我在這裡呼籲2008年不用選，民進黨、[[國民黨|國民黨]]兩黨合一，看要是『謝馬配』還是『馬謝配』，他們兩個乾脆搭檔。」又說：「台灣人假使沒有半個人，我汪笨湖願意代表台灣獨立出來參選總統。我會去跪求[[林義雄|林義雄]]，我甚至會跪求游錫堃脫黨出來選。不是選贏選輸的問題，是選一個台灣精神的問題<ref>{{cite news|url=http://www.ettoday.com/2007/05/03/10844-2091278.htm|archiveurl=https://web.archive.org/web/20070929210030/http://www.ettoday.com/2007/05/03/10844-2091278.htm|title=08卡位／選邊站　汪笨湖挺游錫堃、打謝長廷|author=楊琇惇、吳嘉堡|date=2007-05-03|archivedate=2007-09-29|agency=ETToday|deadurl=yes}}</ref>。」

汪笨湖被外界視為[[李登輝|李登輝]]子弟兵，其立場偏於[[台灣團結聯盟|台灣團結聯盟]]。之後曾於[[2012年|2012年]]12月自行籌組政黨，名為「[[台灣番薯黨|台灣番薯黨]]」，汪笨湖稱創黨宗旨為「第一條一邊一國；第二條救出陳水扁；第三條清理...我們不是用[[清算|清算]]，[[清理|清理]]國民黨不義的[[黨產|黨產]]；第四條[[老人年金|老人年金]]發兩萬」<ref>[https://news.pts.org.tw/article/227815?NEENO=227815 號召救扁 汪笨湖成立"台灣番薯黨" - 公共電視台 新聞網]</ref>，並完成內政部政黨備案登記程序，政黨編號231號，負責人為汪笨湖本人，中央黨部及通訊處均為[[臺南市中西區|臺南市中西區]]某處<ref>[https://party.moi.gov.tw/pgms/politics/party!detail.action?id=231 政黨資訊-政黨-政黨及全國性政治團體資訊網-內政部]</ref>。惟成立之後並未提名參選台灣各項公職選舉，汪笨湖逝世之後，台灣番薯黨運作狀況不明。

曾於2008年[[扁案|扁案]]爆發時公開表示陳水扁應該[[洗門風|洗門風]]，並指陳水扁不應怪別人，後來轉為大力支持陳水扁。
