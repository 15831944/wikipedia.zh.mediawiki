'''凌 濤'''（{{bd|1989年|10月22日||catIdx=L凌}}），台灣政治人物，[[雲林|雲林]][[台西|台西]]人。第七屆[[中國國民黨青年團|中國國民黨青年團]]總團長兼黨指定中央常務委員。

== 簡介 ==
出身於台灣[[雲林縣|雲林縣]][[台西鄉|台西鄉]]，高中就讀於[[國立台中第一高級中學|國立台中第一高級中學]]。大學則入[[國立政治大學|國立政治大學]]就讀政治學系。期間曾指出大學教師評鑑制度問題，並號召政大師生連署修改評鑑制度，盼留下優良教師<ref>[http://news.cts.com.tw/cna/life/201010/201010070580390.html]</ref>。2011年8月進入[[台灣加油讚|台灣加油讚]]，負責青年活動規劃。2012年9月由時任[[中華民國總統|中華民國總統]]兼[[中國國民黨主席|中國國民黨主席]][[馬英九|馬英九]]與六位評審委員面試，當選[[中國國民黨|中國國民黨]]第七屆青年團總團長兼任中央常務委員<ref>[http://news.cts.com.tw/cna/life/201010/201010070580390.html]</ref>。


2014年退伍後，隨即加入新北市長[[朱立倫|朱立倫]]競選團隊，擔任活動組長。<ref>[http://www.storm.mg/article/36580]</ref>。後任職於[[新北市政府|新北市政府]]民政局機要秘書。<ref>[http://udn.com/news/story/7323/866361-%E5%85%A8%E5%9C%8B%E9%A6%96%E5%89%B5%EF%BC%81-%E6%96%B0%E5%8C%97%E5%B8%82%E6%8E%A8%E5%BE%B5%E5%85%B5%E4%B8%80%E6%A2%9D%E9%BE%8D%E6%95%B4%E5%90%88%E7%B3%BB%E7%B5%B1]</ref>他於近年考取[[中山獎學金|中山獎學金]]政治組，目前赴美攻讀碩士。

== 任期內事蹟 ==
'''凌濤'''擔任[[中國國民黨青年團|中國國民黨青年團]]總團長期間，推出「[[民主|民主]]深化論壇」，探討台灣多元歷史記憶，包含台籍[[慰安婦|慰安婦]]、[[蔣渭水|蔣渭水]]。[[國民黨主席|國民黨主席]][[馬英九|馬英九]]對此表示認同：「對台灣的歷史，一定要有中道的聲音出來，就事論事」<ref>[http://www.worldjournal.com/view/full_news/20093880/article-%E8%97%8D%E9%9D%92%E6%B0%91%E4%B8%BB%E8%AB%96%E5%A3%87-%E8%A6%81%E8%AE%93%E6%AD%B7%E5%8F%B2%E8%A8%98%E6%86%B6%E5%B0%8D%E8%A9%B1?instance=tw_bull]</ref><ref>[http://tw.news.yahoo.com/%E8%AA%8D%E8%AD%98%E8%94%A3%E6%B8%AD%E6%B0%B4-%E8%97%8D%E9%9D%92%E5%B9%B4%E5%9C%98%E8%BE%A6%E8%AB%96%E5%A3%87-070429070.html]</ref>。[[紅十字會|紅十字會]]總會長[[王清峰|王清峰]]表示：「年輕人願意關心[[慰安婦|慰安婦]]阿嬤議題，讓她看見一股屬於台灣、願意「省思」的年輕力量」<ref>[http://www.ettoday.net/news/20121121/129929.htm]</ref>。


他也推動「岸青對話」系列，邀在台就讀的[[中國大陸|中國大陸]]、[[澳門|澳門]]學生，及具兩岸交流經驗的台灣學生，討論兩岸議題，包括當時熱門的陸生納入[[全民健康保險|全民健康保險]]議題<ref>[http://www.taiwannews.com.tw/etn/news_content.php?id=2069662]</ref><ref>[http://tw.news.yahoo.com/%E5%85%A9%E5%B2%B8%E6%B7%B1%E5%8C%96%E4%BA%A4%E6%B5%81-%E8%97%8D%E9%9D%92%E5%B9%B4%E5%9C%98%E4%B8%8D%E7%BC%BA%E5%B8%AD-122212913.html]</ref>。同時辦理「日知會」系列，於[[台南市|台南市]][[安平區|安平區]]札哈木原住民公園，以「新世代[[原住民|原住民]]青年論壇─文化與政策」為題舉辦論壇。<ref>[http://tw.news.yahoo.com/%E9%97%9C%E6%B3%A8%E5%8E%9F%E6%B0%91%E6%96%87%E5%8C%96-%E8%97%8D%E9%9D%92%E5%B9%B4%E5%9C%98%E9%82%80%E5%B0%8D%E8%AB%87-122620846.html]</ref>並於[[高雄市|高雄市]]與[[行政院原住民族委員會|行政院原住民族委員會]]主委[[孫大川|孫大川]]及[[原住民|原住民]]作家奧威尼‧卡露斯對談原住民文學創作，持續關注原住民文化發展。<ref>{{Cite web |url=http://www.cna.com.tw/News/aIPL/201304290155-1.aspx |title=存档副本 |access-date=2013-11-23 |archive-url=https://web.archive.org/web/20131202222041/http://www.cna.com.tw/News/aIPL/201304290155-1.aspx |archive-date=2013-12-02 |dead-url=yes }}</ref>
