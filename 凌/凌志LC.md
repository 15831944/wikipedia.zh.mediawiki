'''凌志LC'''（日文：レクサス·LC），是[[豐田汽車|豐田汽車]]豪華車部門[[凌志|凌志]]的[[豪華旅行車|豪華旅行車]]／{{link-en|個人豪華車|Personal luxury car}}。它源於2012年由[[加利福尼亞州|加利福尼亞州]][[新港灘|新港灘]]的{{link-en|Calty設計研究|Calty Design Research|豐田Calty設計研究公司}}設計的{{link-en|凌志LF|Lexus LF|LF-LC概念車}}。從2011年到2016年，它以計畫代號“950A”進行開發，設計工作在2013年1月從Calty轉移到位於日本愛知縣的豐田技術中心，並在2014年上半年確認了最終產品設計。<ref>{{cite web |last=Pettendy |first=Marton |date=2014-07-19 |url=http://www.motoring.com.au/lexus-lf-lc-set-for-production-44805/ |title=LF-LC Set for Production |website=Motoring |accessdate=2016-04-07}}</ref><ref>{{cite web |last=Pincott |first=Karla |date=2014-06-11 |url=http://www.carsguide.com.au/car-news/2017-lexus-sc-rendered-28642 |title=2017 Lexus SC Rendered |website=CarsGuide |accessdate=2016-04-07}}</ref><ref name="designdates" />
