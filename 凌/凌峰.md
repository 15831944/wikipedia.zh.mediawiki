'''凌峰'''（{{bd|1945年|||}}），本名'''王正琛'''，出生於[[中華民國|中華民國]][[青島市_(中華民國)|青島市]]<ref name=xinhuanet/>，台灣[[歌手|歌手]]、[[主持人|主持人]]。1977年在[[鳳飛飛|鳳飛飛]]主持的[[中視|中視]]綜藝節目《[[你愛周末|你愛周末]]》[[出道|出道]]，是[[台視|台視]]《[[八千里路雲和月_(電視節目)|八千里路雲和月]]》的製片人及主持人。<ref>{{cite web |author=楊慈郁 |title=凌峰憶鄧麗君 痛批成龍不負責 |url= |work=[[旺報|旺報]] |date=2013-04-03 |accessdate=2013-04-04}}</ref><ref>{{cite web |author=黃雯犀 |title=凌峰憶鄧麗君 轟成龍不負責任 |url= |work=[[中國時報|中國時報]] |date=2013-04-04 |accessdate=2013-04-04}}</ref>

==經歷==
凌峰曾經在台灣[[秀場|秀場]]時期與[[張魁_(藝人)|張魁]]、[[張菲|張菲]]、[[倪敏然|倪敏然]]及[[夏雲飛|夏雲飛]]合組「溫拿五鼠」<ref name=libertytimes/>，也曾經演過電影《[[八二三炮戰_(電影)|823砲戰]]》、《[[辛亥雙十|辛亥雙十]]》。1980年，凌峰與張魁、[[張帝|張帝]]主持[[華視|華視]]綜藝節目《[[週末2100|週末2100]]》，紅極一時。

凌峰曾是歌手，曾加盟[[麗歌唱片|麗歌唱片]]與[[東尼機構|東尼機構]]。2015年7月，凌峰上[[陳凱倫|陳凱倫]]主持的廣播節目《好好品歌詞》，他說「當年離開歌壇就是因為無法超越自己」<ref>{{Cite web|url=http://www.chinatimes.com/newspapers/20150724001907-260112|title=凌峰相隔15年 重啟《八千里路》|work=中國時報|author=洪秀瑛|date=2015-07-24|accessdate=2018-09-02}}</ref>。

凌峰曾在[[台視|台視]]主持綜藝節目《電視街》，創下27%的[[收視率|收視率]]。凌峰在中視時獲得1985年[[第20屆金鐘獎|第20屆金鐘獎]]男歌唱演員獎。凌峰以《八千里路雲和月》獲得1990年[[第25屆金鐘獎|第25屆金鐘獎]]最佳教育文化節目主持人。

1990年，為打破阻隔[[台海兩岸|台海兩岸]][[文化交流|文化交流]]30多年的冰封，凌峰捐獻廣告代言之酬勞[[新台幣|新台幣]]1000萬元，成立[[民族文化基金會|民族文化基金會]]並擔任董事長，贊助[[希望工程|希望工程]]及兩岸交流。1991年，凌峰接受[[中國青少年發展基金會|中國青少年發展基金會]]邀請，擔任希望工程海外愛心委員會主任委員。2001年6月，中國青少年發展基金會聘凌峰為扶貧助學愛心大使。2001年8月，凌峰獲中國青少年發展基金會頒贈希望工程特殊貢獻獎<ref>{{Cite web|url=http://www.observer-taipei.com/article.php?id=1792|author=紀欣、陳淑英|title=凌峰用鏡頭見證兩岸交流30年|work=《觀察》雜誌第52期|publisher=觀察雜誌社|date=2017-12|accessdate=2018-09-18}}</ref>。

1994年7月22日中午，凌峰聽說台視將停掉《八千里路雲和月》，去和台視節目部經理[[顧安生|顧安生]]理論，並在一怒之下當場掀桌子，玻璃碎了一地；隨後，台視委由律師提出聲明，要求凌峰登報公開道歉，否則將針對其「毀損公物」及「恐嚇」等兩項行為尋求法律途徑解決；凌峰表示，他是在推動「電視新文化運動」，反對電視台永遠以收視率來衡量一切。

2000年，凌峰找來一群年輕人共同成立「中華八千網股份有限公司」，凌峰自任[[董事長|董事長]]，員工幾乎都是三十來歲的年輕人，運用[[網際網路|網際網路]]無國界、即時的特性，將凌峰腦中的創意化為可行的計畫，按部就班推向國際。中華八千網員工對凌峰的印象是「愛讀書、精力充沛、尊重專業」的董事長，凌峰常說「網路的東西我不懂，交給專業的人去執行；我只要專心管內容和創意的部分就好了」<ref>{{cite web|author=江煒琦|url=http://www.businesstoday.com.tw/article-content-80417-153109|title=凌峰的「影像銀行」|accessdate=2018-09-02|work=《[[今周刊|今周刊]]》|publisher=|language=zh-tw|date=2001-02-22|pages=46}}</ref>。

2013年6月14日，凌峰獲頒第5屆[[海峽影視季|海峽影視季]]兩岸影視交流貢獻獎。

凌峰與[[鄧麗君|鄧麗君]]交情深厚，他一直想做鄧麗君全紀錄，籌備多年卻找不到適合的切入點。2015年12月，台灣知名廣播人[[王一明|王一明]]跟凌峰聊起[[中視新聞|中視新聞]]當家主播[[哈遠儀|哈遠儀]]的故事，哈遠儀是鄧麗君的忠實粉絲，凌峰馬上決定向哈遠儀敲定採訪，凌峰說：「我跟中視淵源很深。像之前當初鳳飛飛的《一道彩虹》等等，我也是在中視做節目時得到獎。」<ref>{{cite news|url=http://www.chinatimes.com/realtimenews/20151225005651-260404|title=歌聲似鄧麗君 凌峰來台專訪哈遠儀|publisher=[[中時電子報|中時電子報]]|author=林俐瑄|language=zh-tw|date=2015-12-25|accessdate=2018-09-02}}</ref>
