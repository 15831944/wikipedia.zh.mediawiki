'''荔城区'''（[[莆仙语|莆仙语]]：{{lang|cpx|Lā̤-síng kṳ}}）是[[中华人民共和国|中国]][[福建省|福建省]][[莆田市|莆田市]]下辖的一个[[市辖区|市辖区]]，是莆田市的中心城区。荔城区北面为[[涵江区|涵江区]]，南面为[[秀屿区|秀屿区]]，西面为[[城厢区|城厢区]]，东面为[[兴化湾|兴化湾]]。辖区总面积269.66平方公里，总人口46.38万人，居民通行[[莆仙语|莆仙语]]<ref> {{cite web |url=http://www.lcxww.com/show.aspx?id=371&cid=33 |title=告诉你一个真实的荔城 |accessdate=2008-10-10 |date=2006-12-31 |publisher=荔城新闻网 |language=zh-hans }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 行政区划 ==
荔城区人民政府驻县巷，现辖镇海、拱辰2个街道和西天尾、黄石、新度、北高4个镇，共有118个村委会，12个居委会。

== 历史沿革 ==
2002年2月1日，莆田县撤销并设立莆田市荔城区和秀屿区<ref> {{cite web|url=http://www.gov.cn/test/2007-03/23/content_559272.htm |title=中华人民共和国行政区划（2002年） |accessdate=2008-10-10 |publisher=中央政府门户网站 |language=zh-hans }}</ref>。

== 地理 ==
荔城区属[[亚热带海洋性季风气候|亚热带海洋性季风气候]]，常年多为东南风，气候温和湿润。

== 文化 ==
2007年11月，荔城区被[[中国文化部|中国文化部]]授予“全国民间绘画之乡”、“全国民间戏剧之乡”两个称号<ref>{{cite news| first=| last=| coauthors=| title=荔城:中国民间绘画之乡| date=2008-07-21| publisher=福建之窗| url=http://fjnews.66163.com/Fujian_w/dskx/20080721/xs256185.html| work=| pages=| accessdate=2008-10-10| language=zh-hans}}{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 经济 ==
荔城区农业以蔬菜、畜禽、水果、水产、食用菌为主，而[[新度镇|新度镇]]则是中国最大的禽苗生产基地之一。

== 旅游 ==
荔城区的主要旅游景点有[[壶公山|壶公山]]、[[三清殿|三清殿]]、[[南少林寺|南少林寺]]及[[延寿溪|延寿溪]]。
