'''荔枝角公園體育館'''（{{lang-en|'''Lai Chi Kok Park Sports Centre'''}}），為一位於[[香港|香港]][[荔枝角|荔枝角]][[荔枝角公園|荔枝角公園]]內之[[室內體育場館|室內體育場館]]，由[[康樂及文化事務署|康樂及文化事務署]]管理。體育館毗鄰[[美孚新邨|美孚新邨]]，乃荔枝角惟一一個公共室內體育場館。體育館於1983年4月6日啟用。<ref name="lcsd">[http://www.lcsd.gov.hk/lsb/b5/facilities.php?ftid=0&did=9 康樂及文化事務署：荔枝角公園體育館簡介] {{webarchive|url=https://web.archive.org/web/20090615021736/http://www.lcsd.gov.hk/lsb/b5/facilities.php?ftid=0&did=9 |date=2009-06-15 }}</ref>

== 概述 ==
荔枝角公園體育館是第2個於[[深水埗區|深水埗區]]內建成的室內運動場，僅晚於[[長沙灣體育館|長沙灣體育館]]<ref name="lcsd"/>，對面為[[荔枝角政府合署|荔枝角政府合署]]。

== 場內設施及設備 ==
*1個可提供2個[[籃球場|籃球場]]或2個[[排球場|排球場]]或8個[[羽毛球場|羽毛球場]]的多用途主場
*1個面積約89[[平方米|平方米]]的舞蹈室
*1個多用途活動室，可作壁球或乒乓球活動
*1個壁球或乒乓球室
*2個乒乓球室

== 開放時間 ==
體育館於每日07:00-23:00開放（定期保養日除外；定期保養日為每月第1及第3個星期一的07:00-15:00）。<ref name="lcsd"/>
