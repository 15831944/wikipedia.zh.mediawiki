'''荔蝽科'''（[[学名|学名]]：'''Tessaratomidae'''<ref>{{Cite web |url = http://digimuse.nmns.edu.tw/Default.aspx?tabid=265&ObjectId=0b00000181d796b1&Domin=z&Field=i0&ContentType=Specimen&Language=CHI |title = 自然與人文數位博物館 » 一般大眾 » 數位典藏 » 動物學 » 昆蟲 |author =  |date =  |publisher = 國立自然科學博物館 |language = zh-tw |accessdate = 2018-05-01 |quote = 碩椿科(荔椿科)Tessaratomidae 介紹：荔椿科又稱為碩椿科，本科台灣已知有3屬5種。成蟲體型扁平，較一般椿象大，體色多異，若蟲體態多變。頭部小呈三角形，觸角4 節，成蟲外形酷似椿科，但是腹部第2節氣孔外露，不被後胸腹板所覆蓋，各足跗節2或3節。本科昆蟲棲息於喬木植物，多以果實為食。 }}</ref><ref>{{Cite web |url = http://www.nature.edu.hk/species_database/species/tessaratomidae |title = 主頁 > 物種資料庫 > 動物界 > 節肢動物門 > 昆蟲綱 > 半翅目 > 荔蝽科 |author =  |date =  |publisher = 香港生物多樣性訊息系統 |language = zh-hk |accessdate = 2018-05-01 |quote =  }}</ref>），又稱'''硕蝽科'''， 是[[半翅目|半翅目]]的一[[科|科]]，是一类体形较大的昆虫。头小，很多种类颜色艳丽，还可能有金属光泽<ref>[http://nhjy.hzau.edu.cn/kech/ptkc/study-online/insect/chapter30/banchi/lichun.htm 华中农业大学南湖教苑] {{webarchive|url=https://web.archive.org/web/20070505081537/http://nhjy.hzau.edu.cn/kech/ptkc/study-online/insect/chapter30/banchi/lichun.htm |date=2007-05-05 }}</ref>。以植物为食，有些种类被认为是农业[[害虫|害虫]]，如[[荔枝椿象|荔枝椿象]](''Tessaratoma papillosa'')。有些种类但在一些国家被当做食物。已知约240多种，主要分布于[[旧大陆|旧大陆]]地区。
