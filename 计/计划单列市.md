'''计划单列市'''，全称'''国家社会与经济发展计划单列市'''，是[[中国大陆|中国大陆]]的省级单位对所辖大城市下放部分经济管理权限的城市。计划单列市仍保留[[省辖市|省辖市]]的行政地位，继续接受省级行政机构的领导。

1950年代中、1960年代初，中国大陆曾实行过两次对省辖大城市的计划单列，但实行不久就取消了。

1980年代，中国大陆开始第三次实行计划单列市体制：计划单列市的收支直接与中央挂钩，由中央财政与地方财政两分，而无须上缴省级财政。四川省[[重庆市|重庆市]]当时财政已经接近四川省的1/3，为避免造成[[省会|省会]]权限集中，而开始尝试设立单列经济计划。1983年重庆市成为计划单列市后，中央不断给予一些经济发展超过省会城市的非省会城市和一些经济发展较快的省会城市单独的经济政策。

== 历史 ==
1958年2月11日，[[直辖市|直辖市]][[天津市|天津市]]并入[[河北省|河北省]]，河北省省会迁往天津时间达八年。期间，在[[中华人民共和国国务院|中华人民共和国国务院]]的协调下，天津市实行计划单列政策。1967年1月2日，河北省将省会迁回[[保定|保定]]，[[中共中央|中共中央]]决定天津市恢复为[[中央直辖市|中央直辖市]]。

至1993年时，有14个计划单列市：[[重庆市|重庆市]]（1983年2月8日）、[[武汉市|武汉市]]（1984年5月21日）、[[沈阳市|沈阳市]]（1984年7月11日）、[[大连市|大连市]]（1984年7月13日）、[[广州市|广州市]]（1984年10月5日）、[[西安市|西安市]]（1984年10月5日）、[[哈尔滨市|哈尔滨市]]（1984年10月5日）、[[青岛市|青岛市]]（1986年10月15日）、[[宁波市|宁波市]]（1987年2月24日）、[[厦门市|厦门市]]（1988年4月18日）、[[深圳市|深圳市]]（1988年10月3日）、[[长春市|长春市]]（1989年2月11日）、[[成都市|成都市]]（1989年2月11日）、[[南京市|南京市]]（1989年2月11日）。

根据1994年2月25日[[中央机构编制委员会|中央机构编制委员会]]发布并施行的“中编[1994]1号”文件，省会城市取消计划单列，同时设立[[副省级城市|副省级城市]]，原14个计划单列市和[[杭州市|杭州市]]、[[济南市|济南市]]共16个市被正式确定为副省级城市<ref>{{cite web |url=http://www.docin.com/p-51864614.html |title=中央机构编制委员会印发《关于副省级市若干问题的意见》的通知. 中编发[1995]5号 |publisher=豆丁网 |date=1995-02-19 |accessdate=2014-05-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20140529084536/http://www.docin.com/p-51864614.html |archivedate=2014-05-29 }}</ref>；计划单列市减少为6个，即重庆市等6个非省会城市。1997年重庆市升格为[[直辖市|直辖市]]后，计划单列市为以下5个：
*[[大连市|大连市]]（1984年7月13日批准）
*[[青岛市|青岛市]]（1986年10月15日批准）
*[[宁波市|宁波市]]（1987年2月24日批准）
*[[厦门市|厦门市]]（1988年4月18日批准）
*[[深圳市|深圳市]]（1988年10月3日批准）

另外，1986年8月，[[海南行政区|海南行政区]]被国务院批准赋予省一级的经济管理权限，实行计划单列，和各计划单列市具有同等地位；直到1988年[[海南省|海南省]]正式建省。这段时间，国务院及各部委发给各省的文件，均附计划单列市和海南行政区的名字<ref>{{cite web|url=http://www.law-lib.com/law/law_view.asp?id=47879|title=民政部、财政部关于妥善解决“红军失散人员”生活困难问题的通知_全文|publisher=}}</ref>。

1993年1月1日成立的、作为[[上海市人民政府|上海市人民政府]]派出机构的[[浦东新区|上海市浦东新区管理委员会]]，在计划管理、项目审批、建设管理、财政金融、外经外事、劳动人事等方面，拥有计划单列市的各类权限，包括中央批准扩权的部分<ref>{{cite web|url=http://shtong.gov.cn/newsite/node2/node2247/node4566/node79157/node79168/userobject1ai103380.html|title=第四节 浦东新区管理委员会|publisher=上海地方志办公室}}</ref><ref>{{cite web|url=http://shtong.gov.cn/newsite/node2/node2245/node72907/node72912/node72927/node72947/userobject1ai85642.html|title=第三节 浦东新区管委会|publisher=上海地方志办公室}}</ref>。

== 地位 ==
計劃單列市在政治方面全面行使副省級的地位，如[[中共深圳市委|中共深圳市委]]和[[深圳市人民政府|深圳市人民政府]]即為副省級的行政編制；在經濟方面則擁有省部級的地位，財政越過省一級別的管制，直接與中央掛鉤。國務院的一些相應部門在計劃單列市直接設立特派員辦事處或派出机构，如：[[中華人民共和國商務部|中華人民共和國商務部]]駐青島特派員辦事處<ref>[http://qdtb.mofcom.gov.cn/ 中華人民共和國商務部駐青島特派員辦事處]</ref>、[[中国证券监督管理委员会|中国证券监督管理委员会]]大连监管局<ref>[http://www.csrc.gov.cn/pub/dalian/ 中国证券监督管理委员会大连监管局]</ref>。
