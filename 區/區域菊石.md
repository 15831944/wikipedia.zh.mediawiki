'''區域菊石'''（[[學名|學名]]：{{lang|la|'''''Endemoceras'''''}}），又名'''恩德莫菊石'''，是生存在[[白堊紀|白堊紀]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]。牠們是[[區域菊石超科|區域菊石超科]]的代表屬，但其化石卻相當地稀有。<ref name="Endemoceras">{{Cite web |url = http://liusir.pixnet.net/blog/post/248636045-endemoceras-%E5%8D%80%E5%9F%9F%E8%8F%8A%E7%9F%B3-%28%E5%BE%B7%E5%9C%8B%29 |title = Endemoceras 區域菊石 (德國) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-07-20 }}</ref>其[[化石|化石]]被發現於[[德國|德國]]及[[波蘭|波蘭]]等地。<ref name="Fossilworks">{{Cite web |url = http://fossilworks.org/bridge.pl?a=taxonInfo&taxon_no=81827 |title = Fossilworks: Endemoceras |language = en |accessdate = 2018-07-20 }}</ref>區域菊石和與其同科的其他屬菊石都是演化自較早期的{{tsl|en|Neocomitidae|新康菊石科}}。<ref name="Endemoceras" />

== 特徵 ==
區域菊石具有典型的旋菊石式[[Y|Y]]型分叉肋條，肋條的末端在腹肩處形成突起的疣點狀結節，並在腹側向殼口方向折曲，從腹側看來貌似[[魚|魚]][[骨|骨]]。<ref name="Endemoceras" />
