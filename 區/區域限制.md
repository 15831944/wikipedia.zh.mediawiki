'''區域限制'''或称'''锁区'''（{{lang-en|regional lockout}}）是一個在[[儲存媒體|儲存媒體]]中「限定在某[[區域|區域]]內才能正常使用」的[[程式|程式]]、碼、條或[[物理|物理]]障礙限制，用以保障每個[[地區|地區]]的影音產品[[經銷商|經銷商]]與[[代理商|代理商]]權益。[[DVD-Video|DVD-Video]]、[[藍光光碟|藍光光碟]]及[[通用媒體光碟|通用媒體光碟]]等都有區域碼之劃分，但劃分方法各不相同。而電子遊戲會有「日版」、「美版」等發行地區的區別，有些廠牌的遊戲軟體與[[遊戲機|遊戲機]]若分屬不同地區版本則無法使用。部份廠牌的[[印表機|印表機]]會辨識墨水或碳粉等耗材內的晶片，若不是同一區域時會以燈號或訊息提醒而無法列印。

{{fact|另外，美國出版的大學[[教科書|教科書]]若有發行國際版時，為防止較便宜的國際版被回銷至美國國內販售而擠壓到美國版的銷路，會禁止國際版在美國國內銷售。|time=20151228}}

==游戏机==
===雅达利===
[[雅达利7800|雅达利7800]]拥有区码限制。
===任天堂===
家用机：[[红白机|红白机]]（物理锁区：日版卡带和美版卡带引脚数不同）、[[超级任天堂|超级任天堂]]（物理锁区：方法同红白机）、[[任天堂64|任天堂64]]（物理锁区：卡槽处安装限位器）、[[GameCube|GameCube]]、[[Wii|Wii]]、[[Wii_U|Wii U]]拥有区码限制。
掌机：[[任天堂DSi|任天堂DSi]]、[[任天堂3DS|任天堂3DS]]拥有区码限制。
===索尼互动娱乐===
PlayStation和PlayStation 2拥有区码限制。在中国內地发售的PlayStation 4、PlayStation Vita锁服。

===微软===
Xbox和Xbox 360拥有区码限制。而Xbox One只有在中国內地发售的型号拥有区码限制（锁区锁服）。但PlayStation 4在华发售后，Xbox One变为不锁区锁服。<ref>{{Cite web |url = https://www.ithome.com/html/next/138607.htm |title = 终于来了，国行Xbox One确认不再锁区！ |author = 死骑 |date = 2015-04-02 |publisher = IT之家 |language = zh-cn |accessdate = 2017-10-08 }}</ref>
