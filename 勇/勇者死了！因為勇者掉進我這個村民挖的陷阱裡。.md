《'''勇者死了！因為勇者掉進我這個村民挖的陷阱裡。'''》（{{lang-ja|勇者が死んだ! 村人の俺が掘った落とし穴に勇者が落ちた結果。}}）是日本漫畫家{{link-ja|昇一|スバルイチ}}所畫的漫畫。2014年12月16日開始在{{Tsl|ja|裏サンデー|裏Sunday}}和{{Tsl|ja|MangaONE}}連載中。

== 故事大綱 ==
三年前，勇者'''席昂·布雷登'''利用聖劍之力將地獄之門封印，人與惡魔的戰鬥得以結束，但由於封印的不完整，經過一段時間地獄之門再次開啟，世界各地的惡魔直到現在也不斷的湧現出來襲撃人類。

然後在地獄之門開啟後的三個月，一座村的一名平凡好色的農夫少年'''托卡·斯科特'''，某日自製的陷阱居然意外殺害勇者席昂，為了湮滅證據，便埋葬了勇者席昂屍體，但隔天早上托卡成了勇者席昂的外貌，原來是勇者的原夥伴屍術師的'''安妮·赫伊斯沃斯'''將托卡他的精神用屍術轉移到勇者的體內。而後，被威脅為了代替勇者席昂為了拯救世界，並與青梅竹馬'''尤娜·尤尼斯'''為了將地獄之門封印而共同踏上旅途冒險的故事。

== 登場角色 ==
; {{nihongo|托卡·斯科特|トウカ・スコット}} 
: 本作的主角。原本是崔薩村一名種植白蘿蔔的農夫。因意外殺害勇者席昂，而被安妮將精神被移到勇者的體內，代替勇者拯救世界而踏上了旅途。相當好色，但主要只對大腿有興趣，會情不自禁地將腿狀物體穿上絲襪。
; {{nihongo|尤娜·尤尼斯|ユナ・ユニス}}
: 托卡的青梅竹馬。金髮雙馬尾，有著豐滿胸部和理想身材的美少女。總是對帥哥沒有免疫力，愛慕著兩次從惡魔手中救下自己的勇者席昂，對好色的的青梅竹馬托卡感到失望。與托卡踏上拯救世界的旅途。
; {{nihongo|安妮·赫伊斯沃斯|アンリ・ヘイズワース}}
: 屍術師。原是與勇者席昂共同冒險的少女。有著白色的頭髮和紅色的瞳孔的特徵，時常流鼻水。在被人類對屍術師的贈恨被執行死刑時被席昂拯救。遵循與勇者的約定，將托卡的靈魂用屍術轉移至席昂的身體中，托卡的肉身則封印在「生者之棺」，並要求托卡代替勇者拯救世界。
; {{nihongo|席昂·布雷登|シオン・ブレイダン}}
: 原[[勇者_(虛構職業)|勇者]]。被聖劍選中而去拯救世界的少年。三年前用聖劍之力封印了惡魔，卻被盜賊行竊時解除封印，在鎮壓惡魔後，因意外掉落到托卡自製的陷阱而死，後來安妮使用屍術將屍體成為托卡靈魂的容器。
; {{nihongo|瑪格麗特法洛姆|マルグリット・ファロム}}
: 法洛姆王國的皇女(第三公主)。武藝幾乎樣樣精通。勇者的未婚妻。從10年前開始得了魔漏症，身體變成不能儲存魔力。為了補充魔力，需要用食物轉為能量。
; {{nihongo|埃塞爾·博格寧|エセル・ボーグナイン}}
: 被懸賞的女盗賊。托卡的師傅，因完美製造陷阱的技術而被托卡拜師。
; {{nihongo|凱爾·奧斯蒙特|カイル・オズメント}}
: 貴族的後代，是一名劍士，擁有優秀的力量魔法和英俊的外貌。原勇者的夥伴，對於力量相當執著。認為自己才是真正的勇者，但在三年前敗給了席昂而沒能成為勇者，從此痛恨席昂，便埋伏在席昂身邊裝作勇者的夥伴。
; {{nihongo|貝拉哥|ベラーコ}}
: 冒險者，稱號是「{{nihongo|雙槍的貝拉哥|双槍のベラーコ}}」。孤特高潔的男人，很看不起偽裝勇者的托卡，總是暗中觀察托卡，想揭發他的黑歷史。
