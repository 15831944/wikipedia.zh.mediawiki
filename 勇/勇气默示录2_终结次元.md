{{vgname|勇气默示录2 终结次元|ja=ブレイブリーセカンド エンドレイヤー|en=Bravely Second: End Layer}}是[[Silicon_Studio|Silicon Studio]]和[[史克威尔艾尼克斯|史克威尔艾尼克斯]]为[[任天堂3DS|任天堂3DS]]开发的[[电子角色扮演游戏|电子角色扮演游戏]]。是任天堂3DS游戏《[[勇气默示录|勇气默示录]]》的续作。

== 玩法 ==

和前作一样，本作也是电子角色扮演游戏<ref>{{cite web |url=http://www.polygon.com/2013/12/4/5174778/bravely-second-3ds-bravely-default-sequel |title=Bravely Second is a 3DS sequel to Bravely Default |last=Tach |first=David |publisher=Polygon |date=2013-10-24 |accessdate=2013-12-04}}</ref>。

== 故事 ==

游戏和《勇气默示录》世界观相同，发生于前作情节数年之后<ref name="kotaku1">{{cite web |url=http://kotaku.com/first-look-at-bravely-second-the-totally-new-bravely-d-1476290194 |title=First Look at Bravely Second, the Totally New Bravely Default Sequel |last=Ashcraft |first=Kevin |publisher=Kotaku |date=2012-12-04 |accessdate=2013-12-04}}</ref>。游戏沿用冒险、剑与魔法的世界<ref>{{cite web|url=http://gematsu.com/2013/12/bravely-default-sequel-announced-for-3ds |title=Bravely Default sequel announced for 3DS |last=Romano |first=Sal |publisher=Gematsu |date=2013-09-12 |accessdate=2013-12-04}}</ref>。新登场角色马格诺利亚·阿奇将称为主角之一<ref name="sildec10" /><ref>{{cite web|url=http://www.siliconera.com/2013/12/04/bravely-default-sequel-bravely-second-announced-3ds/ |title=Bravely Default Sequel, Bravely Second, Announced For 3DS |author=Ishaan |publisher=Siliconera |date=2012-12-04 |accessdate=2013-12-04}}</ref>。她似乎来自世界的月亮，他的家乡被恶魔攻击，她是一族中最后的幸存者<ref name="CharacterInfo">{{cite web |url=http://www.siliconera.com/2013/12/08/bravely-seconds-new-character-devil-buster-moon/ |title=Bravely Second’s New Character Is A Devil Buster From The Moon |author=Spencer |publisher=Siliconera |date=2013-12-08 |accessdate=2013-12-09}}</ref>。浅野称游戏将进一步探索卢森达尔克大陆<ref name="BraveSecInterview" />。

== 开发 ==

最早关于《勇气默示录》续作的言论是在2012年12月，制作人浅野智也表示会应爱好者要求制作一部续作<ref>{{cite web|url=http://www.siliconera.com/2012/12/17/bravely-default-sequel-in-the-works-producer-asking-for-fan-feedback/ |title=Bravely Default Sequel In The Works |author=Sato |publisher=Siliconera |date=2012-12-17 |accessdate=2013-12-04}}</ref>。2013年6月又出现了关于续作的言论，游戏开发者[[松野泰己|松野泰己]]在[[推特|推特]]上宣称，角色设计师兼主美工[[吉田明彦|吉田明彦]]正在创作《勇气默示录》续作<ref>{{cite web|url=http://www.siliconera.com/2013/06/05/akihiko-yoshida-working-on-next-bravely-default-says-yasumi-matsuno/ |title=Akihiko Yoshida Working On Next Bravely Default Says Yasumi Matsuno |author=Ishaan |publisher=Siliconera |date=2013-06-05 |accessdate=2013-12-04}}</ref>。2013年8月史克威尔艾尼克斯公布了原版游戏的完全版《勇气默示录 For the Sequel》，它使用开发续作的新游戏性理念<ref>{{cite web|url=http://www.computerandvideogames.com/441296/bravely-default-sequel-announced-for-3ds/ |title=News: Bravely Default sequel announced for 3DS |last=Hussain |first=Tamoor |publisher=ComputerAndVideoGames |date=2012-12-04 |accessdate=2013-12-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20141210224726/http://www.computerandvideogames.com/441296/bravely-default-sequel-announced-for-3ds/ |archivedate=2014-12-10 }}</ref><ref>{{cite web|url=http://www.destructoid.com/bravely-default-for-the-sequel-expansion-announced-260894.phtml |title=Bravely Default For the Sequel expansion announced |last=North |first=Dale |publisher=Destructoid |date=2013-08-28 |accessdate=2013-12-04}}</ref>。史克威尔艾尼克斯在2013年9月注册“Bravely Second”商标<ref>{{cite web|url=http://www.siliconera.com/2013/09/18/hey-these-trademarks-sound-like-future-bravely-default-games/ |title=Hey, These Trademarks Sound Like Future Bravely Default Games |author=Ishaan |publisher=Siliconera |date=2013-09-18 |accessdate=2013-12-04}}</ref>，并在2013年12月期的《[[Jump|Jump]]》杂志上正式宣布游戏名称<ref name="Eurogamer1">{{cite web|last=Phillips |first=Tom |url=http://www.eurogamer.net/articles/2013-12-04-bravely-default-sequel-announced-for-3ds-in-japan |title=Bravely Default sequel announced for 3DS in Japan • News • 3DS • |publisher=Eurogamer |date= |accessdate=2013-12-04}}</ref>——根据“勇气默示录 For the Sequel”游戏机制命名的“勇气默示录2”<ref>{{cite web|url=http://www.vg247.com/2013/12/04/bravely-second-announced-as-sequel-to-bravely-default/ |title=Bravely Second announced as sequel to Bravely Default |last=Nunneley |first=Stephany |publisher=VG247 |date=2013-12-04 |accessdate=2013-12-04}}</ref>。随之公布的只有游戏平台为[[任天堂3DS|任天堂3DS]]<ref>{{cite web|url=http://www.nintendolife.com/news/2013/12/bravely_default_sequel_confirmed_for_3ds_in_japan |title=Bravely Default Sequel Confirmed for 3DS in Japan |last=Whitehead |first=Thomas |publisher=Nintendolife |date=2012-12-04 |accessdate=2013-12-04}}</ref>。尚未有非日本（或日语）版本发行的公告<ref name="kotaku1" />。游戏在2013年12月大约完工30%<ref name="sildec10" />。

2013年12月有消息透漏吉田已经离开史克威尔<ref name="polygon2">{{cite web |url=http://www.polygon.com/2013/12/5/5177724/bravely-default-character-designer-leaves-square-enix |title=Bravely Default character designer leaves Square Enix |author=Gera, Emily |publisher=[[Polygon_(website)|Polygon]] |date=2013-12-05 |accessdate=2013-12-05}}</ref>，但他称仍计划创作勇气默示录游戏<ref name="Eurogamer1" />。虽然他称《勇气默示录2》仍然延续前作的[[日本动画|动画]]画风<ref name="Eurogamer1" />，但角色略微靠近现实[[头身|比例]]，低身长设计略有弱化<ref name="sildec10">{{cite web |author=Ishaan |url=http://www.siliconera.com/2013/12/10/bravely-second-will-larger-fields-things/ |title=Bravely Second Will Have Larger Fields, More Things To Do |publisher=Siliconera |date=2013-12-10 |accessdate=2013-12-14}}</ref>。最初的头脑风暴诞生了马格诺利娅身着宇航服或兔耳的理念，但终因“成人外观”而废弃，开发人员认为认为一些内容在第一作中有失<ref>{{cite web |author=Sato |url=http://www.siliconera.com/2013/12/12/bravely-second-producer-aims-create-new-pillar-support-square-enix/ |title=Bravely Second Producer Aims To Create A New Pillar Of Support For Square Enix |publisher=Siliconera |date= |accessdate=2013-12-14}}</ref>。浅野还确认游戏会延续前作的故事导向，而非改为探险导向<ref name="BraveSecInterview">{{cite web|url=http://www.siliconera.com/2014/02/21/bravely-second-will-focus-story-rather-exploration/|title=Bravely Second Will Focus On Story Rather Than Exploration|author=Ishaan|publisher=Siliconera|date=2014-02-21|accessdate=2014-02-22}}</ref>。2014年4月，制作人浅野在《[[Fami通|Fami通]]》的传言栏中宣称游戏剧本已经完成<ref>{{cite web |author=Mauro Piccillo |url=http://www.eirgames.com/13580/bravely-second-scenario-completato/ |title=Bravely Second Scenario Compleato! |publisher=EIR Games |date= |accessdate=2014-04-12 |archive-url=https://web.archive.org/web/20140413125906/http://www.eirgames.com/13580/bravely-second-scenario-completato/ |archive-date=2014-04-13 |dead-url=yes }}</ref>。《Fami通》杂志在2014年7月下旬报道[[Revo|Revo]]因时间表冲突，将不再为游戏谱写原声，[[Supercell|Supercell]]乐团的Ryo将接替此职务<ref>{{cite web|last1=Sato|title=Supercell’s ryo In Charge Of Music For Bravely Second|url=http://www.siliconera.com/2014/07/29/supercells-ryo-charge-music-bravely-second/|website=Siliconera|accessdate=2014-07-29}}</ref>。

== 评价 ==

中国大陆电子游戏杂志《[[游戏机实用技术|游戏机实用技术]]》打出24/30的高分，称赞游戏比前作更加庞大的职业系统<ref>{{cite journal|title=黄金眼|journal=游戏机实用技术|year=2015|issue=370|page=14}}</ref>。
