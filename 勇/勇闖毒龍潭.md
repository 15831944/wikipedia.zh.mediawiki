{{engname|《勇闖毒龍潭》|'' One Way Ticket to Bangkok ''}}是1988年香港[[電視廣播有限公司|電視廣播有限公司]]出品的一部[[電視電影|電視電影]] <ref>{{Cite web |url= https://www.hkfact.com/movie/14758|title =電影－勇闖毒龍潭| publisher =香港電影反刪減小組|language = 中文}}</ref><ref>{{Cite web |url= http://www.ihkmusic.com/detail.asp?Product_id=VCD020011&txtParam=2&txtCat=TVB%20Movies|title =勇闖毒龍潭 One Way Ticket To Bangkok| publisher =現代音像﹝國際﹞有限公司|language = 中文}}</ref>，由[[李鼎倫|李鼎倫]]執導，[[黎明|黎明]]、[[萬梓良|萬梓良]]、[[吳鎮宇|吳鎮宇]]、[[楊寶玲|楊寶玲]]、[[杜德偉|杜德偉]]、[[陳雅倫|陳雅倫]]等演出<ref>{{Cite web |url= http://hkmdb.com/db/movies/view.mhtml?id=14758&display_set=big5|title =勇闖毒龍潭（1988）| publisher =香港影庫|language = 中文}}</ref><ref>{{Cite web |url= https://moviecool.asia/film/m1l_9_eCwOEV|title =勇闖毒龍潭－演員及工作人員| publisher =影史庫|language = 中文}}</ref>，於1988年12月10日在[[無綫電視|無綫電視]][[翡翠台|翡翠台]]播出。

==劇情介紹==
劇情講述林子俊與May是一對夫妻，過著安定的生活，有一天他們偶然目擊香港和泰國毒販尋仇殺人的事件，May更被人殺害滅口，林子俊既傷心又憤怒，決定追查事件真相，找兇手報仇。楚留華是林子俊的好友，他查出殺人事與泰國的毒販金老大有關，在好友雷森的協助下，林子俊和楚留華一起前往泰國追事件真相，由此展開了一段驚險的生死故事<ref>{{Cite web |url = http://ent.163.com/05/1109/17/224R9N4H00031MF2.html |title =黎明早期演繹的角色形象的幾大類別|author =王曉易|date =2005年11月9日|publisher =網易娛樂|language =中文}}</ref><ref>{{cite news |language =中文| url = https://hk.entertainment.appledaily.com/enews/realtime/article/20170605/56784129|title =【那些年的黎明】《原振俠》卡士最強 《阿Sir早晨》演教師暫別公仔箱|publisher =蘋果日報|date =2017年6月5日}}</ref><ref>{{Cite web |url= https://movie.douban.com/subject/4925429/|title =勇闖毒龍潭| publisher =豆瓣電影|language = 中文}}</ref>。
