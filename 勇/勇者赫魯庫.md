{{日本作品|勇者赫魯庫|ヘルク|Helck}}是日本漫畫家{{link-ja|七尾七希|七尾ナナキ}}所畫的漫畫。2014年5月5日開始在{{Tsl|ja|裏サンデー|裏Sunday}}和{{Tsl|ja|MangaONE}}連載，2017年12月4日連載結束。

==概要==
本作為七尾七希於[[niconico動畫|niconico插畫]]、[[TIADIUM|TIADIUM]]及[[ACARIA|ACARIA]]上連載的漫畫作品。於2014年5月5日開始，每周星期一更新。原為{{Tsl|ja|裏サンデー|裏Sunday}}2013舉辦的「第2回連載投稿トーナメント」上的第二名<ref>{{cite web|title=裏サンデー|2=|url=http://urasunday.com/tournament02/index.html/|deadurl=yes|archiveurl=https://web.archive.org/web/20160305161039/http://urasunday.com/tournament02/index.html|archivedate=2016-03-05}}</ref>。後於2015年[[niconico動畫|niconico]]舉辦「次にくるマンガ大賞」中獲得第八位<ref>{{cite web|title=【次にくるマンガ大賞】『これから売れてほしいマンガ部門』結果発表！|url=http://ch.nicovideo.jp/tsugiman/blomaga/ar718601/}}</ref><ref>{{cite web|title=「次にくるマンガ大賞」発表！　第1位は『僕のヒーローアカデミア』に決定！|url=https://ddnavi.com/news/225599/}}</ref>。

==故事==
魔王被勇者一個人打倒了，人類方面正在大肆慶祝。三個月後，魔族為選出新魔王而舉辦新魔王決定戰，魔族列強雲集，而當中的大熱門卻是人類勇者！以謎之勇者赫魯庫而展開的冒險故事。

==人物==
; {{nihongo|赫魯庫|ヘルク}}
:戰鬥等級99的人類勇者，宣言「人類太可惡」參加新魔王決定戰。
:各種非戰鬥能力均屬頂級。
:在人頖國家為通緝犯，罪名為「殺害弟弟」。
:弟弟為一個人打倒了魔王的勇者。
:自幼因魔獸襲擊而父母雙亡，與弟弟雙依為命，在階級差距極大的人類國家過著被搾取的底層生活。
:懂事前已覺醒勇者之力卻無自覺。

; {{nihongo|瓦米里奧|ヴァミリオ}}
:戰鬥等級78的魔族帝國四天王之一，外號「赤之瓦米里奧」，擅長火炎魔法。
:新魔王決定戰的大會負責人，為妨礙赫魯庫勝出而弄出各種非戰鬥項目，但全數失敗。
:性格火爆易怒，不擅說謊，笑容只有奸笑。
:為了近距離觀察赫魯庫而使用假名「安妮」隨行，後來一直被暱稱「小安妮」。

; {{nihongo|嗶嗚咿}}
:說話很大聲的鳥，孤島的住民。
:在赫魯庫和瓦米里奧被傳送到孤島就一直一起行動，在兩人離開時偷偷跟上船。

; {{nihongo|祖拉多|アズドラ}}
:戰鬥等級73的魔族帝國四天王之一，外號「青之祖拉多」，擅長咒術，非常溺愛瓦米里奧。
:等級為四天王中最低，但如果場地合適戰力就是最強。
:跟被有翼士兵打倒的魔王烏魯姆為好友。

; {{nihongo|阿司塔|アスタ}}
:間諜等級55的諜報員，瓦米里奧的手下，奉命潛入人類國家調查勇者。
:外型打扮較中性，以前曾被誤會是男生。

; {{nihongo|伊司塔|イスタ}}
:阿司塔的姐姐，負責伺奉祖拉多。
:跟阿司塔有心靈感應，藉此快速通信諜報。
:對阿司塔被誤會成男生非常火大，曾為此無視阿司塔本人意見將其打扮一番，導致訓練中穿禮服而被赫尤拉教訓。

; {{nihongo|赫尤拉|ヒュラ}}
:戰鬥等級50，外號「不死身的赫尤拉」。
:新魔王決定戰參賽者，不信任赫魯庫。
:跟伊司塔為好友，曾訓練阿司塔。

; {{nihongo|克恩羅斯|ケンロス}}
:新魔王決定戰參賽者，對腳程有自信。

; {{nihongo|多魯士|ドルーシ}}
:戰鬥等級45，外號「鐵壁的多魯士」。
:新魔王決定戰參賽者，吃下豆子就能一直張開強力結界。

; {{nihongo|蘿珂珂|ロココ}}
:新魔王決定戰的主持。

; {{nihongo|克萊斯|クレス}}
:赫魯庫的弟弟，一個人打倒了魔王的勇者。
:打倒魔王後被人類國家抓住用作覺醒勇者的實驗。

==用語==
'''契約書'''
:祖拉多為新魔王決定戰而製作，為防止參戰選手胡亂鬧事或被威脅，令參加者「除比賽場合，對他人和物品的攻擊無效」。

'''有翼士兵'''
:強制讓人類覺醒成勇者而誕生的「人造勇者」。具有和「遊戲中死了就從儲存點再來」一樣的無限復活能力。

'''弒殺勇者'''
:赫魯庫的佩劍，據說有殺死勇者的能力。

'''雪'''
:人類國家高層為了強行讓全國人民覺醒成有翼士兵而散佈。
