'''勇气号火星探测器'''，-{zh-tw:中國譯作'''勇氣號火星探測器''';zh-cn:台湾译作'''精神号火星探测车''';zh-hk:又有譯'''勇氣號火星探測車''';}-{{mn|name|1}}（Spirit, MER-A），是[[美国国家航空航天局|美国国家航空航天局]][[火星探测漫游者|火星探测漫游者]]计划的第一部[[火星车|火星漫游车]]。火星探测漫游者计划旨在将勇气号和[[机遇号|机遇号]]两部双胞胎火星车送往[[火星|火星]]，对火星这颗红色行星进行详细考察。

== 概述 ==

勇气号的主要任务是探测火星上是否存在[[水|水]]和[[生命|生命]]，并分析其物质成份，評估火星上的環境是否有益於生命。

勇气号已于2004年1月3日04时25分在火星表面成功着陆。

== 命名 ==
美国国家航空航天局2003年[[火星探测|火星探测]]计划，官方正式名称火星探测漫游者，代号MER。勇气号是两部双胞胎火星车中的第一部，代号'''MER-A'''。

除了官方正式名称和代号，2002年11月4日，两部火星车发射之前，美国国家航空航天局宣布与丹麦著名玩具制造商[[乐高|乐高公司]]和[[行星學會|行星學會]]合作，由乐高赞助举办火星漫游车命名竞赛，面向5至18岁美国青少年为两部双胞胎火星车征集名字。{{mn|rover_naming_contest_1|2}}凡是在美国学校学习，并在2002年秋季学习注册过的学生均可报名参加。{{mn|rover_naming_contest_2|3}}竞赛活动于2003年1月31日截止，最后结果于2003年6月8日，第一部火星车勇气号发射前夕对外公布。{{mn|rover_naming_contest_3|4}}

这次活动总共为两部双胞胎火星车征集到了将近10000个名字，其中来自美国[[亚利桑那州|亚利桑那州]]的9岁女孩[[索菲·柯林斯|索菲·柯林斯]]提出的方案从众多方案中脱颖而出，被美国国家航空航天局采用。

== 背景 ==
1997年7月4日，[[火星探路者|火星探路者]]登陆火星，它携带的漫游车[[旅居者号|旅居者号]]成为首部在火星表面成功漫游的火星车。火星探路者和旅居者号漫游车在火星表面工作了将近三个月，向地球传回了超过10000张照片和大量科学数据。火星探路者的成功以及此前传出的在地球南极附近发现的[[火星陨石|据信可能来自火星的陨石]]中可能存在生命迹象的消息{{mn|background_1|6}}，使得火星这颗红色行星再次成为人们的关注焦点。但是旅居者号体积较小，重量很轻，能够携带的科学仪器十分有限。而且旅居者号必须通过火星探路者登陆器中继才能与[[地球|地球]]进行通信，这使它的活动范围非常受限，于是人们开始计划发射一部更加强大的火星漫游车。

火星探测漫游者就是这一计划的产物。美国国家航空航天局原计划于2001年发射一枚[[火星勘测者2001|轨道器]]环绕火星轨道运行，同时发射一个登陆器携带漫游车[[居里号|居里号]]在火星表面登陆。但是由于1999年[[火星氣候探測者號|火星氣候探測者號]]和[[火星极地登陆者|火星极地登陆者]]两枚探测器在接近火星时相继失败，居里号漫游车被取消，轨道器则被更名为[[火星奥德赛号|火星奥德赛号]]于2001年成功发射升空。1999年的失败是对美国国家航空航天局火星计划的一次重大打击，多项火星计划被取消或推迟。火星探测漫游者则在此前已经正在进行研究的FIDO（野外综合设计与操作）原型车的基础上被提上日程。

== 生涯 ==
2003年6月10日发射，2004年1月3日着陆火星表面。最初预计的工作寿命只有三个月。

由于太阳能电池板的蒙尘，勇气号的电力供应一直在持续下降；幸运的是，2005年3月12日和2009年2月6日的两次大风吹散了尘埃，使得其电力得到恢复。

2006年，六只车轮中的右前轮失灵。

2009年5月，在通过特洛伊沙地时，车轮陷入软土，又使其中一個故障，勇气号无法动弹，之后的观测一直被限制在原地，此后有过几次解救行动但都失败。

2010年1月26日，NASA宣布放弃拯救，勇气号从此转为静止观测平台。

2011年3月22日，NASA最後一次聯絡上勇气号；

2011年5月24日，NASA在最後一次嘗試聯絡後結束勇气号的任務。{{mn|endmission_1|7}}
