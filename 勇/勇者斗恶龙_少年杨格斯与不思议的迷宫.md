{{vgname|勇者斗恶龙 少年杨格斯与不思议的迷宫|ja=ドラゴンクェスト 少年ヤンガスと不思議のダンジョン}}是《[[勇者斗恶龙VIII|勇者斗恶龙VIII]]》的前传游戏，是[[roguelike|roguelike]]游戏系列[[不可思议迷宫|不可思议迷宫]]的作品之一，并如同系列使用了随机生成的迷宫<ref name="source" />。本游戏由[[Cavia_(公司)|Cavia]]开发，[[史克威尔艾尼克斯|史克威尔艾尼克斯]]于2006年在日本[[PlayStation_2|PlayStation 2]]平台发行。

== 游戏玩法 ==

游戏使用了随机生成的迷宫以及回合制的战斗系统<ref name="quest" />。玩家必须和不同楼层的敌人作战直到遇到头目怪兽，为了推进剧情，玩家必须将头目击败<ref name="IGNS" />。当与敌人接触时战斗就会发生，而不会出现单独的战斗屏幕会菜单系统<ref name="IGNS" />。游戏加入了一个新功能“蓄力系统”，它可以让玩家积累能量并给予敌人强力打击，然而在这种状态下角色将不能移动<ref name="IGNS" />。在游戏的后期，玩家可以使用特殊的壶捕捉怪物，并使用它们攻击对手<ref name="IGNS" />。

玩家可以在任何时刻带领三个怪物，它们可通过使用道具与积累战斗经验值来学会特殊技能<ref name="IGNS" />。当怪物达到4级，得到它们喜欢的食物并拥有武器后，才会协助杨格斯，随着战斗次数的增多，它们就能将自己的力量与其它捕捉怪物的力量进行联合<ref name="IGNS" /><ref name="spy">{{cite web | title=Dragon Quest: Young Yangus' Mysterious Dungeon | url=http://ps2.gamespy.com/playstation-2/dragon-quest-yangus-mysterious-dungeon/704045p2.html | author=Andrew Alfonso | work=[[Gamespy|Gamespy]] | date=May 1, 2006 | accessdate=2012-08-04 | deadurl=yes | archiveurl=https://web.archive.org/web/20120717015116/http://ps2.gamespy.com/playstation-2/dragon-quest-yangus-mysterious-dungeon/704045p2.html | archivedate=2012年7月17日 }}</ref>。游戏亦使用了过场动画以及电脑生成的漫画风格场景<ref name="IGNS" />。

== 情节 ==

游戏围绕着孩子时期的角色杨格斯——《[[勇者斗恶龙VIII|勇者斗恶龙VIII]]》中的主角之一<ref name="source">{{cite web | title=Four million Dragon Quest VIIIs shipped | url= http://www.gamespot.com/news/four-million-dragon-quest-viiis-shipped-6143340 |author=Hirohiko Niizumi| work=[[Gamespot|Gamespot]]|date=January 30, 2006 | accessdate=2012-08-04}}</ref>。作为一个“突然加入的盗贼”，在一个神秘的壶被带回家时，他参与了父亲杨帕盗贼团<ref name="IGNS">{{cite web | title=Dragon Quest Yangus: Import Playtest | url= http://ps2.ign.com/articles/702/702444p1.html |author= Anoop Gantayat |work=[[IGN|IGN]]|date=April 24, 2006 |accessdate=2012-08-04}}</ref><ref name="jump">{{cite web | title=Jump Fest 2005: Square Enix and Mistwalker show upcoming titles | url= http://www.gamespot.com/news/jump-fest-2005-square-enix-and-mistwalker-show-upcoming-titles-6141529 |author=Staff|work=[[Gamespot|Gamespot]]|date=December 19, 2005 | accessdate=2012-08-04}}</ref>。尽管杨格斯被告诫不要触摸壶，但他还是做了，然后他被吸入壶中，来到了另一个被叫做“壶之土地”的世界<ref name="IGNS" />。

《勇者斗恶龙VIII》中的女盗贼歌尔达也在这片新世界出现，同样出现的还有莫利、特鲁尼克以及新角色波比，每个人都开始探索这片新大陆<ref name="quest" /><ref name="IGNS" /><ref name="jump" />。

== 开发 ==

游戏的一个试玩版在2005年12月的东京Jump Festival上进行了展出<ref>{{cite web | title= Eyes On: Dragon Quest Yangus | url= http://ps2.ign.com/articles/677/677302p1.html |author= Anoop Gantayat |work=[[IGN|IGN]]|date=December 17, 2005 | accessdate=2012-08-04}}</ref>。

== 评价 ==

游戏的“卡通般3D图像”受到了注意，此外他的全动态视频也受到了好评<ref name="spy" /><ref>{{cite web | title=Dragon Quest VIII spin-off dated for Japan | url= http://www.gamespot.com/news/dragon-quest-viii-spin-off-dated-for-japan-6143244 |author= Tim Surette |work=[[Gamespot|Gamespot]]|date=January 27, 2006 | accessdate=2012-08-04}}</ref>。精彩的原画风格和单元着色图形受到了好评<ref name="spy" />。[[Destructoid|Destructoid]]将游戏列为了它们的十个最喜欢[[勇者斗恶龙|勇者斗恶龙]]游戏的第十名<ref name="quest">{{cite web | title= Top 10 Dragon Quest Games | url= http://www.destructoid.com/blogs/Stealth/top-10-dragon-quest-games-230922.phtml|author= Stealth |work=[[Destructoid|Destructoid]]|date=July 8, 2012 | accessdate=2012-08-04}}</ref>。[[IGN|IGN]]给了游戏迷宫移动系统“笨拙”的描述<ref name="IGNS" />。
