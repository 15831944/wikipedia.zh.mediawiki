==劇情簡介==
魔王巴特拉被[[勇者_(虛構職業)|勇者]]阿邦（艾班）擊敗後，世界回復了和平。一班跟隨魔王的怪物回復善良，並到伶仃島（台版漫畫：迪魯木靈島，台版動畫：戴爾馬林島）生活。及後一名嬰孩飄流到島上，並被一魔怪收養；該魔怪為嬰孩取名為「小呆」，而小呆也一直稱收養他的魔怪為爺爺。

多年後，嬰孩長大成人，並結識了來到島上的人類公主-美奧娜（TVB譯名，台版譯名為蕾娜），並初次顯露其神祕的力量。美奧娜離開後後又過了一段日子，怪物忽然回復了魔性，開始亙相攻擊和破壞。小呆的爺爺為免自己失去理性後傷害達爾，正打算把他送走之際，勇者阿邦出現並施展出結界使怪物回復平靜。

眾怪物的失常預示了[[魔王|魔王]]軍即將復活，阿邦認為事態刻不容緩，便收了小呆為徒，讓小呆和其師兄何布一起接受特訓。在特訓了一段日子後，之前被打倒的魔王巴特拉突然出現在伶仃島，並表示有一名比他更強的魔王「巴恩」已經復活了。阿邦再度與巴特拉交戰，這次卻無法將他打敗，只好施展[[自爆|自爆]]咒文。然而，卻只能讓巴特拉負傷；看見阿邦死亡而憤怒的小呆頭頂出現龍形紋章。小呆的龍紋的力量及其師的必殺劍招一下子就擊敗了大驚失色的魔王巴特拉。

小呆為了打敗魔王軍，便離開伶仃島，與何布踏上了征途。

== 主要登場人物 ==

; 小呆/小泰（ダイ，港譯：達爾，聲：日本：[[藤田淑子|藤田淑子]]；香港：[[方煥蘭|方煥蘭]]；台灣：[[賀世芳|賀世芳]]）
:初登場為12歲。 原本是一名孤兒，漂流到迪魯木靈島受到魔物的撫養，從小夢想當打敗魔王的勇者，後來勇者阿邦來到迪魯木靈島，收了小呆做最後一個徒弟，從此踏上與魔王軍戰鬥的征途，事實上小呆是龍騎士的後代，本名鐵諾，擁有龍紋的力量。（龍騎士是人類之神，魔族和龍族的神為了平衡世界而創作出來的勇者.龍紋代表著人類之神的心、魔神的魔力、龍神的力量），最後繼承父親巴藍的龍紋，成為史上第一個具有雙龍紋的龍騎士。
* 武器：帕布尼加（公主、王家）之刃、小呆之劍（達伊之劍），曾經也使用過鎧甲魔劍及真魔鋼龍劍，但劍體不是粉碎就是斷裂。
* 絕技：阿邦流刀殺法—大地斬、海波斬、火焰魔法劍、空裂斬、阿邦式快速劍、阿邦式快速劍X、基加丁快速劍、萊丁快速劍、石破天驚（億萬轟天雷）、龍鬥氣砲咒文—杜爾奧拿，紋章閃。咒文方面，當處於龍騎士狀態下，除了回復系列咒文、極大毀滅咒文、破邪系列咒文、時間凍結密法外，都會使用，另外能使用只有勇者才會的電擊咒文。能同時使用鬥氣跟魔法。
* 每回合行動次數：到後期能一回合行動兩次，譬如阿邦式快速劍X是一回合行動兩次的招式。
* 小呆的日文名字ダイ是「dinosaur（恐竜）」日文發音的前兩個音、他的本名ディーノ是故事裡王國語言中「強之龍」的意思。

:漫畫版揭示最後勇者會回來，並以劍插在土地上，以示勇者靈魂不滅，後記有提及過意念來自RPG GAME。

; 何布（ポップ，港譯：阿布，聲：日本：[[難波圭一|難波圭一]]；香港：[[郭志權|郭志權]]）
: 初登場為15歲。原本是名武器店老闆的兒子，某天決心離家出走跟隨阿邦修行。在故事初期是膽小又怕死的男生，後來從大魔導士麻德利身邊學習後，漸漸成為大家信賴的魔法師（後來自稱大魔導士），後期成為小呆一行人的重心之一。死神奇魯巴恩視他比小呆更具威脅性，曾經引誘他出來想殺死他。在劇裡何布的單戀也是令書迷津津樂道，從第一天見面後一直愛慕著瑪姆，但因為害怕關係改變而一直不敢說出口。在中期被一位占卜師少女愛慕。（結局好像預示著三人生活在一起？）
* 武器：黑魔杖
* 絕技：本身較擅長火焰系魔法，直到後期才會回復咒文，極大毀滅咒文—米杜洛亞、五指爆炎彈，重壓咒文、超級蘇生咒文。
* 每回合行動次數：到最終戰能一回合行動兩次，可以一手用黑魔杖打擊，一手用閃熱咒文魔法攻擊。
* 何布日文名字由來為「pop（通俗、大眾）」。

; 瑪姆（マァム，港譯：瑪文，聲：日本：[[富永美衣奈|富永美衣奈]]；香港：[[蔡惠萍|蔡惠萍]]）
: 初登場為16歲。其父母曾經與阿邦一起打倒魔王巴特拉，她亦曾受過阿邦的指導，小呆何布離開迪魯木靈島後不久就在大陸上的森林與瑪姆結識，外表堅強和粗暴，但內心善良。原本是位僧侶戰士，因為不會使用攻擊咒文，所以初期是以一支魔法槍為武器，後來明白到自己的實力遠不及魔王的部下後，曾離開小呆等人獨自前去修練拳法。戰鬥戲份不及小呆和何布，代表作是打敗了西洋棋戰隊中的女王(皇后)亞比娜斯。剛開始時有點看不起好色又膽小的何布，但後來看見何布勇敢堅強的一面慢慢對他改觀，對於愛情遲頓的瑪姆，直到第26、27集何布2次的告白後，才漸漸正視自己的愛，三圍：B88、W59、H90。
* 武器：魔彈槍、魔甲拳。
* 特技：武神流-閃華裂光拳、武神流奧義-猛虎破碎拳
* 瑪姆日文名字由來為「man（母）」。

; 獸王克洛克達因（獣王クロコダイン、古洛單伊、庫羅克泰因）
: 初登場為30歲(換算人類年齡)。原為魔王軍百獸魔團長，敗給小呆後開始對人類改觀，最後並變成了小呆的伙伴。其重大功績是曾拯救垂死的休葛爾及參與對冰炎將軍之戰等。有關克洛克達因的手指在原作漫畫為四根，動畫為五根。
* 武器：真空斧，真空斧改，雷光斧。
* 特技：獸王會心擊，獸王激烈掌。

; 休葛爾（ヒュンケル，港譯：休葛爾）（休凱爾、邱葛爾）
:初登場為21歲。原為魔王軍不死軍團長，是阿邦的大弟子，卻誤以為其父之死與阿邦有關而加入魔王軍，並攻擊帕布尼加(蕾娜公主的國家)。精湛的劍法加上一件能抵擋任何魔法的盔甲，使其擁有強大的戰鬥力。最後在得知其父之死的真相後敗在小呆的雷電魔法劍下。之後轉為同伴一方參與了許多對抗魔王軍的戰鬥。原本就是個用劍高手，在打倒龍騎眾後接收鎧之魔槍，轉為精進使槍技術。瑪姆甚為傾慕他，何布視其為情敵。
* 武器：鎧之魔槍，鎧甲魔劍。
* 特技：大十字勳章、地雷閃、海嗚閃、虛空閃（空裂斬）、大地斬、海波斬、血腥必殺劍法。其實會虛空閃就應該也可使用阿邦式快速劍、鬥魔傀儡掌(本人擁有黑暗鬥氣才能使用)。
* 休葛爾日文名字由來為「ヒュンと剣（けん）を振るう」。

; 蕾娜公主 (レオナ，港譯：美奧娜公主，聲：日本：[[久川綾|久川綾]]；香港：[[黃鳳英|黃鳳英]]）
:初登場為14歲。帕布尼加的公主，因舉行祭典儀式而與小呆熟識，對於熟識的人喜歡別人叫她本名而討厭叫她公主。當得知休葛爾是破壞她國家，但最後卻改邪歸正時，非但沒有給與任何懲罰，還心胸寬大的要求休葛爾繼續為正義而戰，是個外表看似高傲其實內心堅強且極富有正義感的女孩。缺點是愛購物和打開不知名的寶箱，三圍：B83、W55、H85。
* 武器：帕布尼加（王族）之刃（原本的公主之刃因承受不了小呆的龍紋之力而粉碎）、輝石及聖石羽毛。
* 特技：基本上公主有賢者之質，所以攻擊系及回覆系魔法都會，但在何布面前都算是小魔法。大破邪咒文。
* 蕾娜日文名字由來為「leo（獅子）」。

; 阿邦 (アバン，港譯：艾斑，聲：日本：[[田中秀幸|田中秀幸]])
:初登場為31歲。15年前打倒魔王巴特拉的勇者，是小呆、何布、瑪姆、休葛爾的師父，武器、魔法樣樣精通，而且智慧超群。在他的影響下小呆才最终成為勇者，算是故事的重要人物。故事開始是大家都以為阿邦在與巴特拉的戰鬥中犧牲，但阿邦没有死(芙蘿拉女王的護身符)，他以他的方式訓練自己(去破邪洞窟)，在最終決戰重新回歸，此外性格上阿邦十分有趣，喜愛開玩笑，並不像一般的勇者那樣嚴謹，他的人格魅力、戰鬥的實力、對人的影響力，就連大魔王巴恩都十分顧忌，並稱阿邦為地上最聰明的人。不過他做料理手藝好，阿邦以前是廚師。
: 結局成為了國王。
* 阿邦日文名字由來為法語的「avant（前）」。

; 小吱/小米（ゴメちゃん，港譯：哥鈴）
: 被稱作是夢幻奇珍的黃金金屬史萊姆，一開始就陪伴在小呆身邊，在小呆一行人出現困境時會顯現出神力。然而，它的真面目是「神的眼淚」，可說是一種活的道具，運用得宜甚至可以動搖神、魔界的平衡，連大魔王巴恩都害怕的一種「諸神遺產」。

==魔王軍==

===六大軍團===

在大魔王巴恩底下是魔軍司令巴特拉，巴特拉底下率領大魔王的六大軍團。

; 魔軍總司令—巴特拉（ハドラー、巴鐸拉）
: 初登場為357歲。在大魔王巴恩及其親信密斯脫巴恩的黑魔力之下復活，為了擁有超越小呆的力量接受了妖魔軍團長尚波拉私下研究的超魔生物的改造，進化為超魔生物。最後帶領西洋棋戰隊與小呆等人對戰。使小呆及眾伙伴陷入苦戰。
: 個性冷酷，甚至會殺害老弱婦孺，卻從不假手於人，也不會以此為樂。
* 巴特拉的日文名前由來為希臘神話的傳說生物「[[勒拿九頭蛇|勒拿九頭蛇]]（hydra）」。

; 百獸魔團長—獸王克洛克達因
: 詳情參考[[勇者鬥惡龍_達伊的大冒險#主要登場人物|主要登場人物]]。

; 不死軍團長—休葛爾
: 詳情參考[[勇者鬥惡龍_達伊的大冒險#主要登場人物|主要登場人物]]。

; 妖魔士團長—尚波拉（妖魔司教ザボエラ、薩波耶拉）

; 冰炎魔團長—弗雷札特（氷炎将軍フレイザード、菲扎度）
: 由巴特拉自「禁咒法」產生的魔物，擁有半冰半火的身體，魔王軍中最殘暴不仁的團長。最後小呆領悟了阿邦最後的劍招「空裂斬」後攻擊他的核心才總算把他打敗。
* 絕技：五指爆炎彈，冰炎結界咒文。

; 超龍軍團長—巴藍（竜騎将バラン、伯倫）
: 率領魔王軍戰力最強的惡龍軍團攻陷了許多國家，是小呆的生父，同樣擁有龍紋之力，連大魔王巴恩也甚為顧忌他。本來亦是一名正義的使者，但因其力量強大而被人類妒忌，終導致其妻被殺及失去孩子。自此憎恨人類並加入魔軍。小呆為了阻止他以龍紋共鳴的方法控制自己，只好把龍紋轉移到手背上。而電視版的小呆大冒險亦是結束於小呆和巴藍的第一次接觸戰後；巴藍最後死於魔王埋於巴特拉身上的強力魔界炸彈「黑核晶」之下。
* 武器：真魔剛龍劍（不屬於人間的武器，是神使用碳鎢鈦合金打造的最強兵器）
* 絕技：龍鬥氣砲咒文

; 「死神」奇魯巴恩（キルバーン、基魯邦、基爾霸恩、死神喬拜恩）
: 事實上不是大魔王巴恩的直屬部下，而是大魔王巴恩在魔界中的競爭對手—'''「冥龍王」維爾沙'''（已被巴藍擊敗且被精靈封印）另有目的地借用到大魔王巴恩旗下，初次出現在大魔王面前時自稱為奇魯巴恩、意指Kill Vearn【殺了巴恩】。其真身是身邊的小魔怪，外表像死神的身體實際只是一個強力的黑核晶木偶而矣。小呆打敗大魔王巴恩後，為了阻止此黑核晶摧毀世界，拉著他飛上天際，最終不知所終（故事結束）。
: 個性卑劣殘忍，喜歡以各種陷阱暗算他人，並且以此為樂。
* 奇魯巴恩的日文名前由來為「Kill（殺）+バーン」。但是他的名字帶有暗號為「殺了巴恩（Kill Vearn）」。

; 魔影軍團長—密斯脫巴恩（魔影参謀ミストバーン、密斯特霸恩、密度拜、米斯托邦）
: 大魔王巴恩的親信。大多以披著白袍及沒有臉孔的外形示人。實際上並無肉身，真面孔是一團黑暗鬥氣。一直幫助魔王保管年輕的肉身。最後把肉身交還魔王後，入侵並打算奪取休葛爾的身體時卻反被休葛爾預先儲備的光鬥氣消滅。
* 的日文名前由來為「myst（霧）+バーン」。意思是「影（ミスト）的巴恩」。

; 「大魔王」巴恩（バーン、霸恩、棒）
: 魔界最強的魔王與'''「冥龍王」維爾沙'''為競爭對手，與'''「冥龍王」維爾沙'''一樣希望擁有太陽的照耀，但不同的是，巴恩是想把地上世界摧毀，使地下的魔界世界取代地上世界。其邪眼的能力可以使任何怪物進化及增強威力，號稱力量魔力都遠超過善神的魔王。以年老的肉身出戰就輕易擊敗了小呆並打斷了他的寶劍。巴藍死後，小呆繼承了他的力量，擁有雙龍紋後才足以與巴恩一戰。其後巴恩回復真大魔王巴恩之態與小呆一戰，並說明自身與小呆都是戰神的後裔，擁有好戰的本能，本來處於優勢，卻中何布的激將法，導致失去一隻手、一顆心臟，最後巴恩挖出邪眼，並把邪眼之力使用在自己身上與雙龍紋加龍魔人化的小呆進行終極之戰。
* 武器：光魔杖（老巴恩）
* 絕技：各種魔法跟暗黑鬥氣，災難之牆，凱薩之鷹（帝王不死鳥），災難終結，鷹之翼（不死鳥之翼），以及三者合一的天地魔鬥的姿勢。
* 每回合行動次數：最終戰的天地魔鬥姿勢可以一回合行動三次，可以在一回合中，使出攻擊、防守、咒文三個動作。
* 巴恩的日文名前由來為「burning（燃燒）」。

==其他勢力==

; 「冥龍王」維爾沙 (冥竜王ヴェルザー、沃爾哲):與大魔王巴恩二分魔界的強者。過去與雷龍波利庫斯(雷竜ボリクス)展開「真龍之鬥」並獲得最後勝利。曾與巴藍戰鬥中使用黑核晶，但因為認為付出的代價太大，而沒有繼續使用，最後被巴藍擊殺，雖擁有不滅的靈魂，而能重生，但卻被天界精靈封印成為一石雕，雖與巴恩為競爭對手，但卻又有共同的目標-太陽，不同的是，冥龍王是希望擁有地上世界。

; 天界精靈 : 只出現在冥龍王與巴恩的對話之中，從對話得知巴恩在摧毀地上之後的下一步就是進攻天界，間接證明還有天界的存在，精靈沒有太大的力量，但卻有奇妙神祕能力，其中之一便是封印，而冥龍王被封印在魔界深處之中。

===龍騎眾===
陸戰騎、海戰騎、空戰騎等三人組成巴藍的直屬龍使者。陸戰騎的拉哈路特在漫畫後期時被密斯脫巴恩說背叛魔王軍之類的話，拉哈路特對密斯脫巴恩反駁說龍騎眾不是魔王軍的一部分，龍騎眾只為巴藍而戰。
