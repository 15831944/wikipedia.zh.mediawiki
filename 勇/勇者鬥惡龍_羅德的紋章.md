《'''勇者鬥惡龍 羅德的紋章'''》（日文原名：{{lang|ja|ドラゴンクエスト列伝 ロトの紋章}}），是日本漫畫家藤原カムイ以頭三集[[勇者鬥惡龍|勇者鬥惡龍]]遊戲世界觀為本的冒險漫畫作品。

== 概要 ==

== 故事簡介 ==

[[魔王|大魔王]]被[[勇者_(虛構職業)|勇者]]羅德消滅後一百年的世界，新魔王異魔神的出現讓世界重新充滿危機。

繼承勇者羅德血統的人- 羅藍的王子和卡門的王子被魔王手下攻擊。卡門的王子亞魯被救出，藏身修練，決心打倒龍王。羅藍的王子則被邪惡名號詛咒而成為魔人王。

故事講述擁有勇者羅德血統的亞魯知道自己出生的秘密的後，為了恢復世界和平，決定踏上旅程。

== 主要登場人物 ==

; 亞魯
: 繼承羅德血統的卡門王子。父母皆被龍王捉走，決心打倒龍王。
; 基拉
: 砂船渡船船主葛浪的兒子，為卡打爾所賞識一起與亞魯在蜃塔樓中修行。
; 賈剛
: 因龍王的攻擊而被征服的羅藍國王之子，他被邪惡名號詛咒住，而以「魔人王」登基。
