'''裴宇'''（{{bd|1989年}}），[[瑞典|瑞典]]籍[[华裔|华裔]]<ref name="nationality">{{cite web |url=http://finance.sina.com.cn/zl/2016-04-02/zl-ifxqxqmf3877824.shtml |title=在印度，他们为中国品牌而奋斗 |accessdate=2018-05-20 |date=2016-04-02 |work=[[新浪|新浪]]财经 |deadurl=no |archiveurl=https://web.archive.org/web/20180520122852/http://finance.sina.com.cn/zl/2016-04-02/zl-ifxqxqmf3877824.shtml |archivedate=2018-05-20 }}</ref>，中国智能手机制造商[[一加|一加]]的联合创始人，一加全球的负责人<ref name="I'm Carl Pei, Co-Founder of OnePlus, and This Is How I Work"/>。

==早年生活==
裴宇于1989年出生在中国北京<ref name="birth">{{cite web |url=http://cn.ceibs.edu/emba/views/10056 |title=刘作虎（EMBA2013）：一加手机，意外的故事 |accessdate=2018-05-20 |work=CEIBS |deadurl=no |archiveurl=https://web.archive.org/web/20180520122852/http://cn.ceibs.edu/emba/views/10056 |archivedate=2018-05-20 }}</ref>，不久后他和父母移居美国[[纽约|纽约]]。两年后，一家人搬到了[[瑞典|瑞典]]，裴宇的童年时代大部分都在那里度过<ref name="I'm Carl Pei, Co-Founder of OnePlus, and This Is How I Work">{{cite web |last=Orin |first=Andy |title=I'm Carl Pei, Co-Founder of OnePlus, and This Is How I Work |url=http://lifehacker.com/im-carl-pei-co-founder-of-oneplus-and-this-is-how-i-w-1688930538 |website=Lifehacker |accessdate=2017-07-01 |archiveurl=https://archive.is/20170701174000/http://lifehacker.com/im-carl-pei-co-founder-of-oneplus-and-this-is-how-i-w-1688930538 |archivedate=2017-07-01 |deadurl=no |language= |date=2015-03-04 }}</ref>。2008年，裴宇进入[[斯德哥尔摩经济学院|斯德哥尔摩经济学院]]攻读[[理學士|理學士]]学位，但于2011年退学，开始从事[[智能手机|智能手机]]行业<ref name="LinkedIn">{{cite web |url=https://www.linkedin.com/in/peicarl/ |title=LinkedIn profile Carl Pei |accessdate=2017-10-06 |date=2017-10-15 |work=LinkedIn}}</ref><ref name="droppingout">{{cite web |url=http://www.slush.org/news/never-settle-said-carl-pei-founded-oneplus/ |title="Never settle," said Carl Pei and founded OnePlus |accessdate=2017-10-06 |date=2017-10-02 |work=Slush |deadurl=no |archiveurl=https://web.archive.org/web/20171007021915/http://www.slush.org/news/never-settle-said-carl-pei-founded-oneplus/ |archivedate=2017-10-07 }}</ref>。

==生涯==
裴宇于2010年入职[[诺基亚|诺基亚]]，工作了三个月。离开诺基亚后，他创建的有关[[魅族|魅族]]的粉丝网站引起了魅族香港分公司的注意。2011年，裴宇加入魅族的营销团队开始在魅族的营销团队工作。一年后，裴宇入职[[OPPO|OPPO]]，担任国际市场部经理，直接在[[刘作虎|刘作虎]]手下工作<ref name="LinkedIn"/><ref name="Forbes">{{cite web |url=https://www.forbes.com/sites/ckgsb/2015/06/24/after-making-it-big-globally-oneplus-turns-its-gaze-to-china/#1e2f8463311e |title=After Making It Big Globally, OnePlus Turns Its Gaze To China |accessdate=2017-10-15 |date=2015-06-24 |work=Forbes |deadurl=no |archiveurl=https://web.archive.org/web/20171015152154/https://www.forbes.com/sites/ckgsb/2015/06/24/after-making-it-big-globally-oneplus-turns-its-gaze-to-china/#1e2f8463311e |archivedate=2017-10-15 }}</ref><ref name="SamMobile">{{cite web |url=https://www.sammobile.com/2015/10/01/oneplus-co-founder-carl-pei-wants-to-be-an-intern-at-samsung/ |title=OnePlus co-founder Carl Pei wants to be an intern at Samsung |accessdate=2017-10-15 |date=2015-10-01 |work=SamMobile |deadurl=no |archiveurl=https://web.archive.org/web/20171015150541/https://www.sammobile.com/2015/10/01/oneplus-co-founder-carl-pei-wants-to-be-an-intern-at-samsung/ |archivedate=2017-10-15 }}</ref>。
