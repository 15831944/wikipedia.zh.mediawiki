'''雲-{}-吞麵'''，或叫做'''餛-{}-飩麵'''（[[福建|福建]]和[[臺灣|臺灣]]等地又稱扁食麵），是一種在[[香港|香港]]的代表麵食，[[廣東|廣東]][[廣州|廣州]]亦十分普遍。雲吞麵是以煮熟的[[餛飩|餛飩]]和[[蛋|蛋]][[麵條|麵]]，加入煮滾的湯底即成<ref name="hket-20160418">[http://topick.hket.com/article/1410123/%E9%9B%B2%E5%90%9E%E9%BA%B5%E5%A4%A7%E6%9C%89%E5%AD%B8%E5%95%8F%20%20%20%20%E6%9C%89%E9%8C%A2%E4%BA%BA%E5%85%88%E9%A3%9F%E7%B4%B0%E8%93%89%EF%BC%9F 雲吞麵大有學問 有錢人先食細蓉？]，香港經濟日報，2016-04-18</ref>。

== 起源 ==
雲吞麵最早在[[清朝|清]]末[[中華民國|民]]初，[[廣州|廣州]][[西關|西關]]一帶出現。據考究，雲吞麵起初於宋前是用於祭祀的。直到[[宋代|宋代]]，每逢[[冬至|冬至]]，各家包餛飩祭祖，祭畢全家長幼分食祭品。富家的一盤祭祀餛飩，有十多種餡，為“百味餛飩”。南宋以後，餛飩傳入市肆。初期多數是由小販肩挑著賣。因為小販通常在晚間到住宅區出售以應宵夜的習慣，小販會以小木棒敲打竹板叫賣。後來一些小販改設店舖經營，甚或加售生滾粥類。經過數十年後，成為今日的雲吞麵老店。雲吞麵在香港要到二次大戰後才普及，由於中國發生[[國共內戰|國共內戰]]，大量移民從中國大陸逃亡到香港，由於香港人口短時間內激增，有居民為了謀生而開設麵檔，並把廣東雲吞進一步演變，在雲吞餡料加入大量蝦肉，又以海鮮作為湯底，遂成為具有香港特色的雲吞麵<ref name="apple-20130129">[http://hk.apple.nextmedia.com/supplement/food/art/20130129/18149250 消失中的細蓉]，蘋果日報，2013-01-29</ref>。

== 別稱 ==
廣東地區的雲吞麵在行內稱為「蓉」（粵音jung2「湧」），據說是「芙蓉麵」的簡稱(請註明出處)。而「芙蓉麵」則源自[[唐代|唐代]]詩人[[白居易|白居易]]《[[長恨歌|長恨歌]]》中的「芙蓉如面柳如眉」，當時文人便將「芙蓉」比喻為麵。另一說：廣東人慣稱炒雞蛋為炒芙蓉蛋，故廣東人把獨有的梘水蛋麵叫做芙蓉麵，習慣性美化及抬高其身價的做法。雲吞麵再按份量分為「細蓉」、「中蓉」和「大蓉」：「細蓉」為1個1[[両|両]]重的乾麵餅配以4顆餛飩；「中蓉」為1.5個麵餅配以6顆餛飩；「大蓉」則為2個麵餅配以8顆餛飩<ref>[http://hk.news.yahoo.com/article/110322/4/ne2g.html 細蓉來自長恨歌]，《明報》，2011年3月23日</ref>。亦有指「蓉」應為「擁」，據說是早期的雲吞麵比較小（細），冬天時低下階層把小碗「擁」在懷裡，既飽肚又取暖，故有「細擁」之稱。另外，也有指分量最小的「細蓉」源自廣州富裕人家的西關大少，由於家境富有，對食物是吃巧而不是吃飽，於是要求食店把麵的分量減半，而對於一般老百姓，雲吞麵只是點心而不能作為正餐<ref name="hket-20160418" />。

習慣上，稱廣東的鮮蝦豬肉雲吞梘水蛋[[竹昇麵|竹昇麵]]為雲吞麵，而外省的為菜肉餛飩無梘水無蛋麵為餛飩麵。

== 製法 ==
雲吞麵一般以雲吞拌麵，分為湯麵與撈麵：
* 湯麵：以灼熟麵條再加入已灼熟的雲吞，再配以大豆芽、豬骨、蝦頭及蝦殼、烤大地魚乾的湯底，最後加上少許[[韭黃|韭黃]]段。
* [[撈麵|撈麵]]：通常以麵條灼熟後稍剩乾水後上碟再放雲吞，加上青菜就成為雲吞撈麵，也有[[蠔油|蠔油]]雲吞撈麵、薑葱雲吞撈麵、[[蝦子|蝦子]]雲吞撈麵。

== 食法 ==
有人吃雲吞麵時加入少許浙[[醋|醋]]，可中和麵中剩餘鹼水味，增添風味。傳統麵家提供廣東泡菜，以作雲吞麵佐料，不過現今大部分麵家為省開支，不再提供廣東泡菜。

== 海外發展 ==
[[File:Char_siew_wan_tan_noodle.JPG|thumb]]
雲吞麵隨著19世紀末至20世紀的華人移民潮，傳入了[[馬來西亞|馬來西亞]]與[[新加坡|新加坡]]一帶，由於當地炎熱的氣候及偏好重口味，因此較為盛行吃以[[醬油|醬油]]、[[黑醬油|黑醬油]]和[[香油|香油]]混合的乾撈口味，再搭配青菜及[[叉燒|叉燒]]，雲吞湯則獨立盛在另一個小碗內，當地也習慣將雲吞麵稱之“乾撈麵”。除了一般的青菜及[[叉燒|叉燒]]，也可搭配炆冬菇雞腳、燒肉或淋上咖哩醬等，吃的時候，搭配醃製的青辣椒一起享用。由於有搭配各種的配菜，因此當地的雲吞會做得較小顆，以符合商業成本。

[[馬來西亞|馬來西亞]]各地乾撈雲吞麵的醬料會略有不同，在[[吉隆坡|吉隆坡]]一帶，因添加較多的老抽、生抽、[[蠔油|蠔油]]、[[豬油|豬油]]等，顏色會較為暗沉；在北馬的[[檳城|檳城]]一帶，較少添加生抽，因此顏色較為淺白；南馬的[[柔佛|柔佛]]一帶多加上[[蕃茄醬|蕃茄醬]]，因此顏色帶點紅，麵條的味道略帶酸甜<ref>[http://news.sinchew.com.my/node/180504 笨珍雲吞麵續篇]</ref>。
