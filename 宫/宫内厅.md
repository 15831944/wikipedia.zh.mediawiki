{{日本政治}}
'''宮內廳'''（{{jpn|j=宮内庁|hg=くないちょう|rm=Kunaichō|f=h}}、{{lang-en|Imperial Household Agency}}）是[[日本|日本]]政府中掌管[[日本天皇|天皇]]、[[日本皇室|皇室]]及皇宮事務的機構，其前身為「'''[[宮內省|宮內省]]'''」與「'''宮內府'''」。

宮內廳除了負責與皇室有關的國家事務外，還有協助天皇接見[[駐日本外交機構列表|外國駐日使節]]與舉行儀式相關事務；保管御璽、國璽也是宮內廳的職責。 

宮內廳過去是舊總理府的[[外局|外局]]，但現在非內閣府的外局（內閣府設置法第49條），而是內閣府下的獨立機關（內閣府設置法48條）{{Sfn|山本淳|小幡純子|橋本博之|2003|p=23-24}}。[[官報|官報]]上會列在內閣府的「外局」第一位。

== 沿革 ==
日本[[文武天皇|文武天皇]]在[[大寶|大寶]]元年（701年）頒布'''[[大寶令|大寶令]]官制'''，首次出現類似後來「[[宮內省|宮內省]]」的組織。

明治2年（1869年）7月8日，明治天皇依古代[[太政官制|太政官制]]設立「二官八省」作為政府組織。此時仍遵循過去的大寶令設置'''宮內省'''，長官為{{link-ja|宮內卿|宮内卿}}。1885年（明治18年）創立[[內閣_(日本)|內閣]]制度，宮內卿由{{link-ja|宮內大臣|宮内大臣}}代替，但依循「宮中府中區別」原則，宮內大臣不屬於內閣成員。此時設有[[內大臣府|內大臣]]、{{link-ja|宮中顧問官|宮中顧問官}}等官職。1886年（明治19年），宮內省下分2課5職6寮4局。1889年（明治22年）裁定{{link-ja|皇室典範 (1889年)|皇室典範 (1889年)|舊皇室典範}}，確立皇室自律原則。1908年（明治41年），依{{link-ja|皇室令|皇室令}}施行宮內省官制，宮內大臣為輔弼天皇、處理皇室一切事務的機關。

1945年（昭和20年）[[第二次世界大戰|第二次世界大戰]]結束時，宮內省除了1官房2職8寮2局，還有[[內大臣府|內大臣府]]、{{link-ja|掌典職|掌典職}}、{{link-ja|御歌所|御歌所}}、{{link-ja|帝室博物館|帝室博物館}}、{{link-ja|帝室林野局|帝室林野局}}、[[學習院大學|學習院]]等13[[外局|外局]]與京都事務所，職員高達6,200人。之後開始將宮內省事務轉移給其他機關或分離獨立以縮小規模。1947年（昭和22年）5月3日[[日本國憲法|日本國憲法]]施行，宮內省更名為'''宮內府'''，為[[日本內閣總理大臣|內閣總理大臣]]所轄機關。宮內府在宮內府長官下設1官房3職4寮（<small>侍從職、皇太后宮職、東宮職、式部寮、圖書寮、內藏寮、主殿寮</small>）與京都事務所，職員數僅1,500人左右<ref name="agency-hist"> {{cite web|author=宮内庁|title=沿革|work=宮内庁(公式サイト)|url=http://www.kunaicho.go.jp/kunaicho/kunaicho/enkaku.html|accessdate=2013年3月}} </ref><ref>{{cite book|editor=印刷局|title=各庁職員抄録|publisher=印刷局|year=1947|format=preview|url={{NDLDC|1078939/8}}|pages=12-13}}</ref>。

1949年（昭和24年）6月1日施行總理府設置法，宮內府改稱'''宮內廳'''，為總理府的[[外局|外局]]。宮內廳長官下設宮內廳次長、1官房3職2部與京都事務所。2001年（平成13年）1月6日，內閣府設置法施行，宮內廳成為[[內閣府|內閣府]]所屬機關。

== 廳舍 ==
廳舍建於1935年（昭和10年），外觀沒有「宮內廳」的銘版。在明治宮殿燒毀至現在宮殿完成為止，本廳舍曾作為'''臨時宮殿'''使用。現在廳舍與宮殿之間設有走廊連接。

* 所在地：[[東京都|東京都]][[千代田區|千代田區]][[千代田_(千代田區)|千代田]]1番1（皇居內、坂下門北側）
* {{link-ja|宮內廳內郵便局|宮内庁内郵便局}}：1924年（大正13年）9月開設。現在是{{link-ja|日本郵便|日本郵便}}的[[郵局|郵局]]。僅宮內廳職員等相關人士可使用。
* 食堂：宮內廳職員與相關人士、記者俱樂部相關人士可使用，另設有販售{{link-ja|御料牧場|御料牧場}}牛奶的自動販售機。一瓶牛乳60日圓。
* {{link-ja|警視廳職員信用組合|警視庁職員信用組合}}宮內廳出張所

== 組織 ==
宮內廳下設[[內部部局|內部部局]]（長官官房、3職、2部）、2[[設施等機關|設施等機關]]、1[[地方支分部局|地方支分部局]]。宮內廳長官與侍從長（侍從職長官）為{{link-ja|認證官|認証官}}。宮內廳的組織是沿襲自戰前制度，因此與其他省廳的組織架構有所不同。

[[侍從|侍從]]職與[[儲君|東宮]]職分別侍奉天皇一家、[[皇太子|皇太子]]一家，因此在皇位繼承時，東宮職職員在皇太子一家就任天皇皇后後轉成侍從職，原來的侍從職職員則轉成侍奉[[皇太后|皇太后]]的{{link-ja|皇太后宮職|皇太后宮職}}，或侍奉新皇太子的東宮職。

負責{{link-ja|宮中祭祀|宮中祭祀}}的{{link-ja|掌典職|掌典職}}屬於「內廷」的職員，非宮內廳或國家機關職員（[[國家公務員|國家公務員]]）。

== 長官 ==
宮內廳的首長為宮內廳長官（{{link-ja|宮內廳法|宮内庁法}}8條1項）。宮內廳長官任免由天皇認證（同條2項）。1947年（昭和22年）5月[[日本國憲法|日本國憲法]]施行時設立宮內府及宮內府長官，1949年（昭和24年）改稱宮內廳與宮內廳長官。長官職為{{link-ja|認證官|認証官}}。另外，宮內廳長官是{{link-ja|特別職|特別職}}的[[國家公務員|國家公務員]]（{{link-ja|國家公務員法|国家公務員法}}2條3項10號）。

宮內廳長官統籌宮內廳事務及監督職員的服務（{{link-ja|宮內廳法|宮内庁法}}8條3項），並可請求[[日本內閣總理大臣|內閣總理大臣]]發布內閣府令（同條4項）、發布告示（同條5項）、對所轄各機關及職員發布訓令或通達（同條6項）、[[皇宮警察|皇宮警察]]相關事務、向[[警察廳長官|警察廳長官]]要求採取必要措施（同條7項）等。

宮內廳長官多為舊內務省系官廳的[[事務次官|事務次官]]或同等職位（[[警視總監|警視總監]]）出身，一般由宮內廳次長升任。

== 次長 ==
宮內廳設有一位宮內廳次長（宮內廳法9条1項）。宮內廳次長負責協助長官、代理廳務、監督各部局事務等（同條2項）。

宮內廳次長不同於特別職的宮內廳長官，是一般職的國家公務員。
