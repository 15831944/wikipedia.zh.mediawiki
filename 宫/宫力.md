'''宫力'''（{{lang|en|Li Gong}}，{{bd|1963年}}），[[美籍华人|美籍华人]]，现任[[ACADINE_Technologies|ACADINE Technologies]]<ref>{{cite web|title=ACADINE Technologies官网|url=http://www.acadine.com/|access-date=2015-07-23|archive-url=https://web.archive.org/web/20160304235646/https://www.acadine.com/|archive-date=2016-03-04|dead-url=yes}}</ref>董事长兼CEO。ACADINE Technologies是一家专注于为移动设备、可穿戴设备及物联网设备打造创新移动操作系统的系统软件公司，其核心产品H5OS是一款以Web为核心、基于当代互联网技术开放标准的操作系统。

== 教育经历 ==

宫力在北京出生长大，在[[清华大学|清华大学]]获得计算机科学的学士学位和硕士学位，并在英国[[剑桥大学|剑桥大学]]获得计算机科学博士学位。

== 学术成就 ==

宫力博士拥有18项美国专利，合著过三本著作并由Addison Wesley和O'Reilly出版，发表过多篇技术文章，并在《自然》（Nature）杂志发表过5篇评论文章。1989年，他在1989 IEEE计算机安全年会中获最佳论文奖；1994年，宫力博士因杰出贡献获得IEEE Communications Society每年一度的Leonard G. Abraham大奖并被授予“IEEE技术文化领域最重要贡献”<ref>{{cite web|title=comsoc|url=http://www.comsoc.org/~awards/abraham.html}}</ref>的评价。

== 职业经历 ==

宫力博士的职业生涯从研究员开始，主要专注于计算机系统、网络和信息安全领域。他曾担任[[計算機協會|ACM]] CSS，IEEE S&P以及IEEE CSFW的大会程-{序}-主席及大会主席。宫力博士曾是ACM TISSEC的编委和IEEE互联网计算的副主编。在进入公司之前，他最初在纽约Ithaca的Odyssey研究所工作，后前往加州Menlo Park的斯坦福国际研究院SRI的计算机科学研究室工作。宫力博士曾以访问学者身份在康奈尔大学和斯坦福大学工作，并曾在北京的清华大学担任过客座教授。

1996年，宫力博士加入[[Sun公司|Sun公司]]新成立的JavaSoft分部，担任Java首席安全架构师并设计了今天为数百万用户所广泛使用的Java平台安全架构。宫力博士被评为Sun杰出工程师，随后成为Java嵌入式服务器和JXTA的技术团队负责人。此外，他曾任国际标准组织[[OSGi|OSGi]]的首任专家组主席并领导制定了OSGi 1.0规范。

2001年他在北京组建中国Sun工程研究院，担任院长，领导团队从事Solaris系统以及浏览器和OpenOffice等相关桌面应用软件的研发。

2005年，宫力博士加入[[微软|微软]]公司任[[MSN|MSN]]中国总经理及微软中国研究院副总裁，期间领导北京和上海的团队，参与MSN提供的多项业务，包括Messenger、[[Hotmail|Hotmail]]、MSN Spaces、安全、移动、搜索以及广告平台和地图。

2007年，宫力博士加入[[Mozilla公司|Mozilla公司]]并在中国成立子公司[[北京谋智网络技术有限公司|北京谋智网络技术有限公司]]，担任董事长兼CEO。2011年，他在台北成立[[美商谋智台湾分公司|美商谋智台湾分公司]]并担任CEO。此后，他在Mozilla总部先后担任一系列高管职位，包括移动设备高级副总裁、亚洲运营总裁<ref>{{cite web|title=Gearing up for the Next Chapter|url=https://blog.mozilla.org/blog/2013/04/10/gearing-up-for-the-next-chapter/}}</ref>、首席运营官<ref>{{cite web|title=Mozilla Leadership Changes|url=https://blog.mozilla.org/blog/2014/03/24/mozilla-leadership-changes/}}</ref>及全球总裁。

宫力博士于2015年4月底离开Mozilla公司，创办ACADINE Technologies。宫力博士还以创办人、投资人、以及顾问等身份参与了多个在美国硅谷和国内的创业项目。2007年至2009年期间，他曾担任美国风险投资公司Bessemer Venture Partners的投资合伙人和中国负责人。

== 所获殊荣 ==

1989年还是博士生的宫力获得IEEE Symposium on Security and Privacy的最佳论文奖。1994年宫力博士荣获IEEE通讯学会颁发的年度大奖Leonard G. Abraham Prize。宫力博士著有三部英文专著，获颁发18项美国专利。2003年，宫力博士被《互联网周刊》和[[新浪网|新浪网]]联合评为“中国新经济人物”，同期被中国软件联合会授予“中国开源运动领袖奖”。2013年获得中国开源软件推进联盟颁发的“中国杰出开源人物”证书，并被《蓝狮子经理人》杂志评为2013年度杰出经理人。他长期担任清华大学校友总会理事会常务理事，并于2008年参与创办清华企业家协会并担任北京分会的创始主席。
