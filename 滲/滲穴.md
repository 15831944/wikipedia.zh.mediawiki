'''沉洞'''（{{lang-en|'''Sinkhole'''}}）<ref>{{Cite web |url = http://www.termonline.cn/list.htm?k=sinkhole# |title = sinkhole 落水洞 |author = 全国科学技术名词审定委员会 |date = 2006 |publisher = 术语在线 |language = zh |accessdate = 2019-01-15 }}</ref><ref>{{Cite web |url = http://terms.naer.edu.tw/detail/3157253/ |title = 滲穴 sinkhole |author = 國家教育研究院 |publisher = 雙語詞彙、學術名詞辭暨書資訊網 |language = zh |accessdate = 2019-01-15 }}</ref>，亦作'''天坑'''、'''岩溶塌陷'''、'''動搖洞'''、'''吞口'''，是一種自然現象，是指具有巨大的容积，陡峭而圈闭的岩壁，深陷的井状或者桶状轮廓等非凡的空间与形态特质，发育在厚度特别巨大、地下水位特别深的可溶性岩层中，从地下通往地面，平均宽度与深度均大于100米，底部与地下河相连接（或者有证据证明地下河道已迁移）的一种特大型[[喀斯特地形|喀斯特地形]]<ref>朱学稳 Tony WALTHAM 朱学稳（著译） 张远海（校）.天坑释义[J].中国岩溶，2006，25(B08)：35-42.</ref>。一般出現於有水流經的[[石灰岩|石灰岩]]岩層。當空氣中的二氧化碳溶解在雨水裡，就會令雨水變成弱酸性；但當這些弱酸性的雨水從土表滲進地下，植被及地表中的植物物質會令這些弱酸變成強酸<ref name=opennatgovtw>{{cite web | url=http://webcache.googleusercontent.com/search?q=cache:sFFo_6S2VHcJ:open.nat.gov.tw/OpenFront/report/show_file.jsp%3FsysId%3DC09702329%26fileNo%3D005+%E6%B2%89%E6%B4%9E&cd=10&hl=en&ct=clnk&gl=hk&client=firefox-a | title=Devil's Millhopper州立地理公園 | accessdate=2010-06-02 }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，使雨水流過[[石灰岩|石灰岩]]岩層時，會慢慢的把[[石灰岩|石灰岩]]層侵蝕成小洞。當小洞日漸擴大時，可能會令地表突然塌陷，形成沉洞。

== 形成機制 ==
[[File:Dead_Sea_sinkhole_by_David_Shankbone.jpg|thumb]]，流進死海的淡水把鄰近地層的地下鹽溶解，令地面出現一個又一個的沙質沉洞。]]

沉洞的形成機制可能是因為滲濾水把水溶度較高的基岩（如[[石灰石|石灰石]]）慢慢的溶解侵蝕，又或是因為[[洞穴|洞穴]]屋頂的塌陷，又或是因為[[水表|水表]]降低。有時，透過沉洞可以看見底下的洞穴。對於特別大型的沉洞，如：美國[[馬默斯洞穴國家公園|馬默斯洞穴國家公園]]的雪松庫，可能會在沉洞的底部看到[[地下水|地下水]]從洞的一邊流動到另一邊，形成了[[伏流|伏流]]現象。

沉洞一般發生於一些位於地下，而且能被大自然的水慢慢溶解的岩石種類，例如：[[石灰石|石灰石]]、[[碳酸鹽岩|碳酸鹽岩]]或鹽床。當岩石被溶解，就會在地下形成空間或溶洞。若地面環境不變，這些地下空間就能保持完整；但若空間的岩面不足以承托其上的重量，就會突然塌陷，形成沉洞。舉例說：在死海，由於海平面不斷下降，使海水不再流入，流進死海的淡水把鄰近地層的地下鹽溶解，令地面出現一個又一個的沙質沉洞<ref>{{cite news |url=http://www.niwolife.com/2010/0310/9976.html |title=死海面积缩小 海岸出现沙洞 |date=2010-03-10 |accessdate=2010-06-02 |publisher=你我生命自然網站 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

沉洞亦可能是因為人類間接做成的，而這些沉洞的形成往往與土地的不適當利用有關。由於發展及建造，令地下水管在市區變得普遍。當地下水管滲漏，其作用就好像從地表滲進地下的雨水。一旦地面環境改變，令地表不能再承受原來設計的重量，就會令沉降發生，形成沉洞。
[[File:Sinkhole.jpg|thumb]]可算是一種比較小規模的沉洞，但不是在自然環境裡發生；路陷一般是因為雨水透過行人路滲進地下，對地底喉管侵蝕，一旦一些比較大型的喉管崩坍，令破損部位附近的沙泥突然被沖走，就會令到路面坍塌。]]

== 等級 ==
按天坑分级原则，深度和宽度均超过500米的为超级天坑，全世界仅有三例，全在中国（重庆[[小寨天坑|小寨天坑]]、广西[[乐业大石围天坑|乐业大石围天坑]]、广西[[巴马号龙天坑|巴马号龙天坑]]，另外贵州平塘[[打岱河天坑|打岱河天坑]]尚待确认）；深度和宽度在300米到500米之间的为大型天坑，全世界有16例，中国有9例；深度和宽度在100米到300米之间的为标准天坑。

== 分布 ==
目前在世界上发现确认的天坑约80个，其中有超过50个在中国。中国的天坑分布在南方喀斯特地区，绝大多数位于黔南、桂西、渝东的峰丛地貌区域。{{fact}}
