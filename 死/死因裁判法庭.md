'''死因裁判法庭'''（{{lang-en|'''Coroner's Court'''}}，簡稱'''死因庭'''）是[[香港|香港]]的一個特別法庭，源於[[英國|英國]][[倫敦|倫敦]]的同等機構，由[[死因裁判官|死因裁判官]]主持，在一般情況下，與5人所組成的[[陪審團|陪審團]]商議對案件的裁決。

==概况==
死因裁判法庭位於[[九龍|九龍]][[深水埗區|深水埗區]][[長沙灣|長沙灣]][[通州街|通州街]]501號[[西九龍法院大樓|西九龍法院大樓]]A座9樓，遇重大事件時，例如[[馬尼拉人質事件|馬尼拉人質事件]]的死因研訊時，由於預計聽審人數眾多，所以死因研訊會移師至[[香港高等法院|香港高等法院]]的[[科技法庭|科技法庭]]舉行。

每逢有死者的親人或[[香港律政司|律政司]]認為死者的死因有可疑，就可以向死因裁判法庭要求裁判，以決定死者的死因。此外，當死者的致死原因不清晰之時，亦會由死因裁判法庭裁決死者的死因。另外，一些受到政府看管的人士在[[香港政府|香港政府]]看管的期間死亡，香港政府需要對大眾交代，所以死因裁判法庭也要為其死因作出裁決。

裁判官會引導陪審員各種可選擇的裁決。死因裁判法庭的權限只在於決定死者的死因，如死者是何時於何處以何種方式死亡，而與決定死因無關的內容，例如：嫌兇的[[行兇|行兇]]動機、[[法律責任|法律責任]]與[[醫療失誤|醫療失誤]]等，都不包括在死因裁判法庭的結論當中。

==死因裁判结果==
死因裁判法庭的裁判結果，可以是：
# 死於自然
# 工業/職業疾病
# [[濫用藥物|濫用藥物]]
# [[自殺|自殺]]
# [[人工流產|人工流產]]
# 意外
# 自我疏忽
# [[合法被殺|合法被殺]]（執法部門執法）
# 自衛被殺
# 死於不幸
# 非法被殺
# [[胎兒|胎兒]]死亡
# 死因存疑
