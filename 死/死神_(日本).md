'''死神'''（{{lang-ja|しにがみ}}）乃是將死者之[[靈魂|靈魂]]帶至[[陰間|陰間]]或另一世界進行審判，或者在陰間司掌審判死者一生功過之神明。本條目乃就[[日本|日本]]之[[宗教|宗教]]、[[流行文化|大眾文化]]、[[民间宗教|民間信仰]]等面向詳述如後，其餘則請參照「[[死神|死神]]」之條目。

== 日本宗教上之死神 ==
在[[佛教|佛教]]裡，「[[魔#魔之分类|死魔]]」乃直接為[[生命|生命]]帶來障礙的[[魔|魔]]。遭到此魔迷惑，可能使人衝動地想[[自殺|自殺]]<ref>參看《図説「理趣経」入門 密教の核心》，大栗道栄著，鈴木出版，1997年，ISBN 978-4-7902-1074-0，頁101。</ref>。不過，[[瑜伽行唯識學派|瑜伽行唯識學派]]文獻《[[瑜伽師地論|瑜伽師地論]]》則認為死魔乃是決定[[有情|眾生]][[死亡|死亡]]之日的神明。其他諸如掌管陰間的[[閻羅王|閻羅王]]與其轄下的[[牛頭馬面|牛頭馬面]]等，亦屬死神之列。

至於[[神道|神道]]的死神，則出現在[[日本神話|日本神話]]中妹神[[伊奘冉尊|伊邪那美命]]產下火神[[迦具土|迦具土]]時遭灼身亡，兄神[[伊奘諾尊|伊邪那岐命]]追往[[黃泉|黃泉國]]探望的橋段。因伊邪那美命之別名為黃泉津大神、道敷大神，含有黃泉國主宰神之意，但是伊邪那美命和閻羅王卻跟[[西方世界|西方世界]]對死神的定義不同。在日本佛教的觀念裡，鬼神怨靈不一定奪走人的性命，反而引導亡者走向死後的世界<ref>參考《生死の仏教学》，木村文輝著，法藏館，2007年，ISBN 978-4-8318-2418-9，頁141。</ref>。

== 人形淨瑠璃之死神 ==
在日本古籍中「死神」的名字並不多見，相關記述甚少。[[江戶時代|江戶時代]]後，[[劇作家|劇作家]][[近松門左衛門|近松門左衛門]]才在[[文樂|人形淨瑠璃]]《[[曾根崎情死|曾根崎心中]]》提到「死神」一詞。近松在[[寶永|寶永]]3年上演的《內心刃為冰之朔日》（{{lang|ja|心中刃は氷の朔日〔しんじゅうやいばはこおりのついたち〕}}）人形淨琉璃劇中，男性和內心裡打算一起尋短的女性說：「『死神』邀約的生命真是虛幻哪....」等語。此段橋段比喻雖然死神存在於打算[[自殺|自殺]]的男女心中，但其樣貌卻仍不明<ref>參看《よくわかる「世界の死神」事典》，七会静著，廣済堂，2009年，ISBN 978-4-331-65459-0，頁168-193。</ref>；但也有人認為近松採用「死神」一詞來表達純粹的虛幻<ref>請見{{lang|ja|[http://media.excite.co.jp/book/daily/monday/010/ 第10回キュ、キュ、キュートな死神たん――大場つぐみ・小畑健『DEATH NOTE』]，スズキトモユ}}，第五段末，2012年10月22日閲覧。</ref>。另外，他在[[享保|享保]]5年（1720年）公開演出的作品《心中天網島》（{{lang|ja|心中天の網島（しんじゅうてんのあみじま）}}）裡，描寫劇中人物紙屋治兵衛的心境：「在版摺紙裡，仍有未知的死神（{{lang|ja|版摺る紙のその中に、あるとも知らぬ死神に}}）。」這段話藉由治兵衛經營的「紙」引出「死神」，說明他面臨[[死亡|死亡]]的決心<ref>請見[http://www2.ntj.jac.go.jp/dglib/contents/learn/edc13/jidai/sewajoruri/p1.html 近松門左衛門｜文化デジタルライブラリー]。</ref>。不過，有人按字面解釋「{{lang|ja|あるとも知らぬ死神に}}」的句子，提出近松本人不認為世上存在著死神的看法<ref>參考《よくわかる「世界の死神」事典》，七会静著，廣済堂，2009年，ISBN 978-4-331-65459-0，頁168-193。</ref>。

== 古典文學之死神 ==
[[File:ShunsenShinigami.jpg|thumb]]》]]
[[江戶時代|江戶時代]]的古典文學作品中，也有提到取走[[靈魂|靈魂]]附體的死神，像發表於[[天保|天保]]12年（1841年）的《[[繪本百物語|繪本百物語]]》就出現「死神」（[[縊鬼|縊鬼]]）的[[怪談|怪談]]故事。縊鬼即是[[绞刑|上吊]][[自殺|自殺]]者的鬼魂，因心有積怨惡念，引誘同樣有惡念的生者至同地自殺或予以[[附身|附身]]。類似的怪談題材，還有[[幕末|幕末]]時期鈴木桃野在《反古裡書》卷之一所寫的〈縊鬼〉<ref>請看[http://home.att.ne.jp/red/sronin/_koten/0834iki.htm 鈴木桃野『反古のうらがき』巻之一「縊鬼」]。</ref>。

江戸時代後期三好想山在其[[隨筆|隨筆]]文集《想山著聞奇集》卷四裡，描寫死神帶走遭[[妓女|妓女]]附身之男子的內心話（原文为{{lang|ja|死に神の付たると云は嘘とも云難き事}}）。{{link-ja|河竹默阿彌|河竹黙阿弥}}在[[明治|明治]]19年（1886年）上演的[[歌舞伎|歌舞伎]]《{{link-ja|盲長屋梅加賀鳶|盲長屋梅加賀鳶}}》裡也提到，據說死神會進入人的腦袋中，使此人想起自己曾做過的壞事而身亡；這種觀念近似[[鬼|幽靈]]或[[邪靈|邪靈]]。[[幕末|幕末]]時期[[落語|落語]]家{{link-ja|三遊亭圓朝|三遊亭圓朝}}的演目《{{link-ja|死神 (落語)|死神 (落語)|死神}}》<ref>中文翻譯可見「[http://moro-miya.jp/koji/koji-5.html 死神]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}」。</ref>則並非原創，故事原型乃出自[[義大利|義大利]][[歌劇|歌劇]]《{{link-en|鞋匠遇仙記|Crispino e la comare}}》（{{lang-it|Crispino e la comare}}）或者《[[格林童話|格林童話]]》故事〈{{link-de|死神教父|Der Gevatter Tod}}〉（{{lang-de|Der Gevatter Tod}}）。

== 民間信仰之死神 ==
[[熊本縣|熊本縣]]宮地町（2005年2月11日已併入[[阿蘇市|阿蘇市]]）有個[[風俗|習俗]]：凡是在外過夜者，返家後必先飲一杯[[茶|茶]]、吃一碗飯才就寢，否則會被死神纏上<ref>請見《旅と伝説第6年7月号》〈熊本県宮地町地方〉，八木三二著，三元社，1933年7月，頁178。</ref>。[[靜岡縣|靜岡縣]]濱松地方的居民則認為，不論靠山或靠海，以[[鐵路運輸|鐵路]]前往人死之處必定會被死神取走靈魂。在這種地方，亡者如果找不到下一個替死鬼，是永遠無法[[超渡|超渡]]的。一般來說，[[彼岸_(日本)|彼岸]][[掃墓|掃墓]]通常在正午時分進行，不過[[岡山縣|岡山縣]]居民傳說如果不是在[[彼岸_(日本)|彼岸]]時間內掃墓，會被死神帶走。

== 近代大眾文化之死神 ==
[[第二次世界大戰|第二次世界大戰]]後西方人的死神觀念傳入日本，大家開始認為死神具有人格，尤其多半出現在[[科幻|科幻]]作品中。[[昭和|昭和]]時期[[漫畫家|漫畫家]][[水木茂|水木茂]]即在《[[鬼太郎|鬼太郎]]》作品裡創作了死神的角色；1979年[[電視劇|電視劇]]《日本名作怪談劇場》中死神的角色即由[[歌舞伎|歌舞伎]][[演員|演員]]中村鴈治郎擔綱演出。[[平成|平成]]時期以降，《[[死亡筆記|死亡筆記]]》、《[[BLEACH|BLEACH]]》、《[[死神的精確度|死神的精確度]]》等膾炙人口的作品往往從[[漫畫|漫畫]]、[[小說|小說]]改編成[[電影|電影]]、電視劇、[[動畫|動畫]]等；《[[女神轉生系列|真·女神轉生]]》、《[[最终幻想系列|Final Fantasy]]》、《[[勇者鬥惡龍系列|勇者鬥惡龍]]》等[[电子游戏|電玩遊戲]]也有死神的角色，詳情請參照[[:Category:死神題材作品|死神題材作品]]。
