'''死亡沼澤'''（{{lang-en|Dead Marshes}}）是[[托爾金|托爾金]]奇幻小說裡[[中土大陸|中土大陸]]的一處[[沼澤|沼澤]]，位於[[魔多|魔多]]和[[艾明莫爾|艾明莫爾]]之間，[[佛羅多|佛羅多]]、[[山姆·詹吉|山姆]]和[[咕嚕|咕嚕]]在前往魔多途中曾路經此處。

== 歷史概述 ==
死亡沼澤位於魔多[[黑門_(中土大陸)|黑門]]和[[艾明莫爾|艾明莫爾]]山脈（{{lang-en|Emyn Muil}}）兩處的中間地帶。在[[第二紀元|第二紀元]]末期，[[人類_(中土大陸)|人類]]與[[精靈_(中土大陸)|精靈]]組成的「[[最後聯盟戰役|最後聯盟]]」跟黑魔王[[索倫_(魔戒)|索倫]]的[[半獸人|半獸人]]大軍在[[達哥拉平原|達哥拉平原]]上展開了一場大戰，當戰役結束後，不斷蔓延的沼澤吞沒了戰死者的[[墳墓|墳墓]]，使當地變成了恐怖的死亡沼澤。<ref name="雙城奇謀">{{Cite book|title=魔戒二部曲：雙城奇謀|last=托爾金|publisher=聯經|year=2012年12月|isbn=978-957-08-4101-5|location=臺北}}</ref>

[[第三紀元|第三紀元]]，[[亞拉岡_(魔戒)|亞拉岡]]在死亡沼澤處抓住了他多年尋找未果、自魔多逃出的生物[[咕嚕|咕嚕]]（{{lang-en|Gollum}}），並將其帶至[[幽暗密林|幽暗密林]]<ref name=":1">{{Cite book|title=魔戒首部曲：魔戒現身|last=托爾金|publisher=聯經|year=2012年12月|isbn=978-957-08-4100-8|location=臺北}}</ref>。

至[[魔戒聖戰|魔戒聖戰]]時期，持有[[至尊魔戒|至尊魔戒]]的佛羅多與山姆經咕嚕引導，穿越死亡沼澤，在那裡他們見到了四處飄浮的[[鬼火|鬼火]]及在水中浮現的死者面孔<ref name="雙城奇謀" />。

== 靈感 ==
作者托爾金曾於一封信件中表示，對於死亡沼澤的描述有部分靈感是來自於他在[[第一次世界大戰|第一次世界大戰]]中的親身經歷，尤其是在「[[索姆河戰役|索姆河戰役]]之後的[[法國|法國]]北部」<ref>{{Cite web|url=http://tolkiengateway.net/wiki/Dead_Marshes|title=Dead Marshes|accessdate=2017-12-04|publisher=Tolkien Gateway|language=en}}</ref><ref>{{Cite news|url=https://cn.nytimes.com/opinion/20160704/how-jrr-tolkien-found-mordor-on-the-western-front/zh-hant/|title=「一戰」大屠殺刺激托爾金寫下《指環王》|author=約瑟夫·洛孔特|date=2016-07-04|newspaper=紐約時報中文網|accessdate=2017-12-04|language=zh}}</ref><ref>{{Cite book|title=聖經、魔戒與奇幻宗師|last=柯林·杜瑞茲|publisher=啟示出版|year=2004年6月|isbn=986-7470-03-6|location=臺北}}</ref><ref>{{Cite book|title=魔戒的鍛造者：托爾金傳|last=柯林·杜瑞茲|publisher=黑龍江教育出版社|year=2015年2月|isbn=978-7-5316-7894-6|location=哈爾濱|pages=}}</ref>。

== 改編 ==
在[[彼得·傑克森|彼得·傑克森]]的電影《[[魔戒二部曲：雙城奇謀|魔戒二部曲：雙城奇謀]]》中，死亡沼澤主要是在[[紐西蘭|紐西蘭]][[哈特村|哈特村]]的一處[[停車場|停車場]]內拍攝的，布景部門動用了1萬1千個沙包來堆砌出死亡沼澤的外觀；之後又在[[石頭街片廠|石頭街片廠]]作了另一處死亡沼澤布景，以拍攝死者面孔的特寫<ref>魔戒二部曲：雙城奇謀DVD - 特別收錄</ref>。而片裡死亡沼澤的空中鏡頭則是取景自紐西蘭的{{Le|克普勒泥地|Kepler Mire}}<ref>{{Cite book|title=魔戒電影誕生密史|last=布萊恩·希伯理|publisher=奇幻基地|year=2002年12月|isbn=957-28136-6-8|location=臺北}}</ref>。
