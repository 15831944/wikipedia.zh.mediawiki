{{nihongo|'''死神的精確度'''|しにがみのせいど|''Accuracy of Death''}}為[[伊坂幸太郎|伊坂幸太郎]]的[[日本|日本]][[推理|推理]][[小說|小說]]，另外本作也被改編成[[廣播劇|廣播劇]]和[[電影|電影]]及[[舞台劇|舞台劇]]。

== 概要 ==
本作為一個有點冷酷的死神，要在七天的觀察期間內來決定所調查的對象是否要死亡或者存活，並以這些人構成六部短篇內容的故事。<ref>[http://okapi.books.com.tw/article/3065 【伊坂幸太郎小說教我的事】陳夏民：每個人都在與死神相處中，重新領悟死亡的意義] - [[博客來|博客來]]OKAPI</ref>

* 2004年、第57回[[日本推理作家協會賞|日本推理作家協會賞]]短篇部門受賞   
* 2006年、第134回[[直木三十五賞|直木三十五賞]]候補、2006年本屋大賞第3位名  
  
以下為本作的六篇短篇
* 死神的精確度（『オール讀物』2003年12月号）
* 死神和藤田（『オール讀物』2004年4月号）
* 暴風雪中的死神（『オール讀物』2004年8月号）
* 在戀愛中的死神（『オール讀物』2004年11月号）
* 在旅程中的死神（『別冊[[文藝春秋_(雑誌)|文藝春秋]]』第255号・2005年1月）
* 死神對老婆婆（『オール讀物』2005年4月号）

== 登場人物 ==
* 千葉
: 本作的[[主角|主角]]和[[死神|死神]]。
: 以死神「調査部」的一員被派到人世上、對於所要調查的人以一個禮拜來觀察、決定是否該死。而在「可以」的情形下、對象將會在第八天死亡、而在「放行」的情形下，則能不用死並且保持全壽。在調査期間，所要調察的對象是不會死的、另外「許可」和「放行」沒有明確的定義、全交由死神自己來定奪。大部份的調査對象都是「許可」的情形、少數人會是「放行」。
: 自己對於執行死亡的工作會把它加以切割、對於人類的[[死|死]]完全沒有興趣、認為「人死的沒意義的話，就一點價值也沒有」。
: 當他被派往人世工作時，一定會下[[雨|雨]]、而無法看到藍天。
: 當死神被派到人世時、外表和年齢會在事前由「情報部」加以改變、以便容易執行工作。容貌和名字會隨每次工作而改變、每個死神必定會有名字、「千葉」和「秋田」則是與[[市町村|市町村]]（類似街、庄）同樣的名字。
: 不論那種[[音樂|音樂]]都很喜歡、工作的空檔時會去唱片行一直試聽著音樂。這種偏愛，讓他說「對於人類的死沒有興趣、因為人類滅絕的話，我就會因為沒有音樂而難過」。這也是每個死神的心聲、因此在唱片行常會遇到其他的死神。
: 對於音樂有著偏執的喜愛、把[[交通堵塞|交通堵塞]]認為人類世界作出來最醜的東西。
: 無法理解人類世界的價値観和語言、還把「雪男」和「[[雨女|雨女]]（日語也稱作雨男）」認為是同一事物、會作出和人類不一樣的發言、因此常常在和人講話時有微妙的不合。
: 碰到死神的手（不戴手套的情況下）的人、會瞬間昏倒、壽命也會減短一年。

=== 死神的精確度 ===
*藤木 一恵
: 在大手電機總公司工作的女性。千葉的調査對象。
: 屬於公司的產品客服部、每天過被人抱怨反應的生活。
: 對於一通客訴電話老是指定自己處理而感到苦惱並且因為在目前的生活中看不到活下去的理由、而感到想死。

* 投訴顧客
: 打投訴電話給一惠所屬客服部的神秘人士。一直都以產品的瑕疵來電反應並且指名要由一惠處理。

=== 死神和藤田 ===
* 藤田
: 中年的流氓。千葉的調査對象。
: 由於自己的老大遭栗木所殺、因此一直想著要向他報仇。
: 阿久津說他是一個「鋤強扶弱」的好流氓、和千葉初見面時就感到他有一種像是「沉重且尖銳的箭頭」的感覺。一直抱著不能報仇比死還可怕的心情。

* 阿久津
: 和藤田同組的小弟。對於重視義氣而和別人不一樣的藤田感到尊敬而跟著他。

* 栗木
: 和藤田同組的流氓。企圖領導整個組織而殺了自己的老大、過去因為[[殺人|殺人]]而在[[監獄|監獄]]坐牢過。

=== 暴風雪中的死神 ===
* 田村 幹夫
: 洋館的住客。在[[東京|東京]]開診所的男性。因為中了旅行社的抽獎而來到洋館。

* 田村 總江
: 洋館的住客。田村幹夫的妻子。千葉的調査對象。過去自己的兒子因為失戀而[[自殺|自殺]]。

* 權藤
: 洋館的住客。中年男性。曾擔任[[刑警|刑警]]，因為年資到了而退休。

* 英一
: 洋館的住客。權藤的兒子。有一身肥胖的體型和帶著一雙銀邊的眼鏡。

* 真由子
: 洋館的住客。在東京是個準備要加入演藝圈的新人。

* 娃娃臉的廚師
: 洋館所雇用的廚師。一個月前在東京的[[旅館|旅館]]擔任料理長。

* 秋田
: 洋館的住客。真由子的戀人。比其他客人還要晚到洋館。

=== 戀愛中的死神 ===
* 荻原
: 在服裝店工作的二十三歳男性。千葉的調査對象。
: 有著光頭和外表帥氣的人、但總戴著沒有度數且鏡片混濁的厚重[[眼鏡|眼鏡]]。對於因為特價而來到店內購買東西的古川朝美一見鍾情而喜歡上她。

* 古川 朝美
: 在電影發行公司上班的二十一歳女性。
: 住在荻原家對面的高級公寓。對於跟蹤自己的跟蹤狂感到苦惱並且懷疑他的真正身份是荻原。

=== 旅程中的死神 ===
* 森岡 耕介
: 犯下殺人並且逃亡的年輕人。千葉的調査對象。
: 在[[澁谷|澁谷]]殺人而被警察追捕、為了避免被[[逮捕|逮捕]]而強硬地搭上千葉的車子並且和千葉一起要到目的地[[十和田湖|十和田湖]]的[[奧入瀬渓流|奧入瀬]]。
: 過去曾遭到[[綁架|綁架]]、並在那時，長時間地被關在後車廂中。在被關的時候，因為一直躺在後車廂和[[床|床]]上，結果造成他對床和後車廂有[[心理創傷|心理創傷]]。後面由於監視他的人給予他撐下去的勇氣而獲救，所以一直很感激他。

* 深津
: 在森岡被綁架時，監視森岡的男人。因為不良於行而使用拐杖、是森岡的救命恩人。

===死神對老婆婆===
* 新田太太
: 在對著[[太平洋|太平洋]]的海岸上開[[美容院|美容院]]的七十歲老婆婆。千葉的調査對象。
: 有著敏銳的直覺、才剛遇到千葉就立刻知道並且說他不屬於這個人世。
: 根據千葉的調查發現她的周圍有不少遭遇死亡的人、因此讓千葉感到很不自然。但她卻對千葉以「一生一次的請求」拜託他幫忙完成一件事情。

《'''死神的精準度'''》（英語：''Accuracy of Death''），日本於[[2008年|2008年]][[3月22日|3月22日]]上映，台灣於[[2008年|2008年]][[4月30日|4月30日]]上映。由原著小說中的「死神的精確度」、「死神和藤田」、「死神對老婆婆」三個章節來構成電影版的內容。這也是[[金城武|金城武]]自從《[[回歸者|回歸者]]》以来，闊別六年而再度演出日本電影。而作者伊坂從電影公司的工作人員得知死神千葉打算邀請金城武來主演時，也覺得此人選非常適合。<ref>[http://okapi.books.com.tw/article/3093 【米果｜日本小說教我的事】死神千葉與金城武的幾種必然] - [[博客來|博客來]]OKAPI</ref>

=== 劇情 ===
千葉（[[金城武|金城武]]飾演）是個不大理解人情世故、酷酷的卻又有點少根筋的死神。每當死神出現在人間時，必定會下雨。死神的工作是在一個星期內接近並觀察目標對象，他會依任務內容變化外型與年紀以便於接近目標對象。另外出任務也常戴著手套，因為如果人類直接碰到死神的手，不但會昏倒還會折壽。在為期一週的觀察期間結束後，死神會向高層提出報告，判斷目標對象是要「認可」（死亡）或「放行」（存活）。這次千葉又來到人間執行任務，目標對象是預定在七天後死亡的藤木一惠（[[小西真奈美|小西真奈美]]飾演），而這位愛聽音樂的死神千葉在觀察中卻意外發現一惠自己並沒察覺的過人之處……<ref>[https://blog-imgs-23-origin.fc2.com/a/p/e/apexpress/paper08042.html#g1 《死神的精確度》 生或死，由他決定] - [[獨步文化|獨步文化]]電子報</ref>
