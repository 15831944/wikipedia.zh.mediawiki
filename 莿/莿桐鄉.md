'''莿桐鄉'''（{{lang-nan-tw|Tshì-tông-hiong}}）位於[[臺灣|台灣]][[雲林縣|雲林縣]]東北部，[[台1線|台1線]]貫穿全鄉，便捷的路網連接[[西螺交流道|西螺交流道]]、[[虎尾交流道|虎尾交流道]]僅數分鐘車程，其地理位置優越。鄉內面積最大的村落[[五華村|五華村]]因位於[[濁水溪|濁水溪]]南岸，地廣人稀，屬於偏遠地區。本鄉居民多以務農為業，是典型的「農業鄉」。

== 歷史 ==
莿桐古稱「莿桐巷」，清朝以前[[刺桐|刺桐]]叢生，人煙絕跡，先民到此墾荒，擇一小巷建屋成村，刺桐樹綠佈全巷，故取名為莿桐巷。

《諸羅縣志》記載「打馬辰陂」，即源由[[虎尾溪|虎尾溪]]分支流出灌溉孩沙里，莿桐、饒平厝，直至西螺一帶。由此可知在康熙年間，已經有相當的開墾地。又乾隆年間，閩人真、張、王等姓開拓樹子腳、頂麻園、湖子內等地。清光緒13年（1887年）成立雲林縣，本鄉莿桐、孩沙里、新莊、番子、甘厝劃屬西螺堡，新-{庄}-、湖子
內、 頂麻園、後埔、樹子腳、大埔尾、油車等地則隸屬溪州堡。

1898年，實施縣制，隸屬溪州廳、西螺堡、樹子腳區，日治時期開闢[[台1線|縱貫公路]]，地方漸漸繁榮，1920年改為[[臺南州|台南州]][[斗六郡|斗六郡]][[莿桐庄|莿桐-{庄}-]]，至今成為正式地名。1946年改為台南縣莿桐鄉；1950年行政區域重新調整，改隸雲林縣莿桐鄉。

== 行政區域 ==
*莿桐村：莿桐鄉人口最多之村，亦是行政中心所在地，與大美村、甘厝村、甘西村、埔子村、興桐村相鄰
*麻園村
*義和村
*興桐村
*四合村:由后埔、油車口、鐵岐路、苦苓腳四個部落組成。
*六合村:由新庄村、土地公埒、九支煙焙、三塊厝、下竹圍、道場等六個部落命名之。
*興貴村:
*[[五華村|五華村]]:由前半治、後半治、湖仔內、溪頭、牛埔厝等五個部落命名之，原名湖仔內。為砲兵部隊擇定射擊訓練陣地。
*埔子：埔子村。
*油車：包話溪埔仔、頂-{庄}-、大-{庄}-、西邊竹圍仔、十八戶仔、奇力新仔（榮橋頭）等六個部落，合為[[埔尾村|埔尾村]]。
*大美村:包含三個部落,依序為大美(舊稱大埔尾)、溪美(舊稱溪底仔)、溪底(舊稱溪埔寮)。
**大美村人口數僅次於莿桐村之第二大村
*樹仔腳：饒平村。
*甘厝-{庄}-：因為人口眾多分甘厝、甘西兩村，「甘」是指「椪柑」，不是姓氏。[[莿桐林家|莿桐林家]]為南台灣望族之一，日治時期開發莿桐全-{庄}-、東勢厝名譟一時。

根據2017年5月統計資料，全鄉人口29,000人。大體而言，莿桐鄉分為莿桐及饒平(樹仔腳)兩大聚落，兩大聚落皆有市場、郵局、便利商店，生活機能方便，其中莿桐村、埔子村、大美村三村的總人口共10,812人，佔全鄉人口超過三分之一，人口多集中在莿桐鄉西南方

面對大環境影響，全鄉總人口持續減少，但因與鄰近之西螺鎮、斗六市成為一共同生活圈，鄉內的大美村，緊鄰斗六市，與雲林科技工業區僅一橋之隔，並有高鐵聯絡道貫穿全村，人口是少數正成長的村。因交流路網便捷，連結鄰近鄉鎮多為十來分鐘車程，工作機會多，較於縣內其他鄉鎮人口外流數少，鄉內產業結構仍為農業。

== 生活機能 ==
鄉內多數商店集中於莿桐村及饒平兩大聚落，多為販售日用品、五金、小吃、冷飲之商店，以及提供基礎服務業之鐘錶、相館、傳統理髮廳與美容院等。
近年連鎖商店大舉在莿桐鄉拓展，目前鄉內有著名的連鎖店[[鮮茶道|鮮茶道]]、[[全聯福利中心|全聯福利中心]]、[[85度C|85度C]]、[[7-11|7-11]]、[[萊爾富|萊爾富]]、[[遠傳電信|遠傳電信]]、[[三媽臭臭鍋|三媽臭臭鍋]]、[[台灣大哥大|台灣大哥大]]等，其中連鎖便利商店在莿桐鄉內計有七家。

莿桐夜市：每周五營業，為莿桐鄉規模最大的夜市，位於莿桐國小後方，緊鄰[[台1線|台1線]]省道

饒平夜市：每周一營業，位於后埔社區
