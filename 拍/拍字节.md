{{字节单位}}
-{zh-cn:'''拍[[字节|字节]]'''或'''拍它字节'''; zh-tw:'''拍[[位元組|位元組]]''';}-（{{lang-en|Petabyte}}，縮寫為'''PB'''）是一種資訊計量[[计量单位|單位]]，現今通常在標示[[网络硬盘|網路硬碟]]、[[服务器农场|伺服器農場]]總容量，或具有大容量的儲存媒介之儲存容量時使用。

== 使用示例 ==
* 在2009年，[[阿凡達|阿凡達]]3D版在[[維塔數碼|維塔數碼]]使用超過1PB來儲存。
* [[Google|Google]]的服务器集群的估计容量大約是5.625PB(截止2004年)。
* [[國會圖書館_(美國)|国会图书馆 (美国)]]藏品数量的估计近似值大約是10PB（包括非书籍资料，截止2005年）
* 在2012年的8月[[Facebook|Facebook]]共使用約100PB。
* 全世界[[印刷|印刷]]材料的内容總量大約是300PB。
* [[Google相簿|Google相簿]]在发布1年之后总共为用户节省13.7 PB的存储空间（截至2016年5月）<ref>18.84[https://googleblog.blogspot.com/2016/05/google-photos-one-year-200-million.html|Google Photos: One year, 200 million users, and a whole lot of selfies]</ref>。
* [[匯豐銀行|匯豐銀行]]儲存全球70個國家和地區，共3,700萬客戶的資料和交易紀錄，資料總量由2014年的56 petabytes (PB)，激增至2017年的93PB<ref>［https://www.silicon.co.uk/cloud/hsbc-google-cloud-211037?inf_by=5c0f2ca9671db8aa248b4d03］</ref>。
* 人類大腦能所儲存的容量約為2.5PB。

== 與其他儲存單位的換算 ==
* 1PB = 1,000 (10<sup>3</sup>) [[太字节|TB]] 
* 1PB = 1,000,000 (10<sup>6</sup>) [[吉字节|GB]]
* 1PB = 1,000,000,000 (10<sup>9</sup>) [[百万字节|MB]]
* 1PB = 1,000,000,000,000 (10<sup>12</sup>) [[千字节|KB]]
* 1PB = 1,000,000,000,000,000 (10<sup>15</sup>) [[字节|B]]

另外，Petabyte往往亦可以指[[Pebibyte|Pebibyte]]，其换算公式是：

* 1PiB = 1,024 [[Tebibyte|TiB]] 
* 1PiB = 1,048,576 (1024<sup>2</sup>) [[Gibibyte|GiB]]
* 1PiB = 1,073,741,824 (1024<sup>3</sup>) [[Mebibyte|MiB]]
* 1PiB = 1,099,511,627,776 (1024<sup>4</sup>) [[Kibibyte|KiB]]
* 1PiB = 1,125,899,906,842,624 (1024<sup>5</sup>) [[字节|B]]
