'''醚'''（[[漢語拼音|漢語拼音]]：mí，{{lang-en|Ether}}）是具有醚[[官能团|官能团]]的一类[[有机化合物|有机化合物]]。醚官能团是由一个[[氧|氧]]原子连接两个[[烷基|烷基]]或[[芳基|芳基]]所形成，醚的通式为：R–O–R。它还可看作是[[醇|醇]]或[[酚|酚]][[羟基|羟基]]上的[[氢|氢]]被烃基所取代的化合物。<ref>{{GoldBookRef|title=ethers|file=E02221}}</ref>

醚类中最典型的化合物属：[[乙醚|乙醚]]，它常用于有机[[溶剂|溶剂]]与医用[[麻醉剂|麻醉剂]]。由于其在化学中的常用性（乙醚是最常用的醚类提取溶剂），我们还有时将乙醚直接简称为“醚”。醚类化合物的应用常见于有机化学和生物化学，它们还可作为[[糖类|糖类]]和[[木质素|木质素]]的连接片段。

== 结构和化学键 ==
醚的结构通式为：R-O-R(R')、Ar-O-R或Ar-O-Ar(Ar')（R＝烃基，Ar＝芳烃基）。醚的键角约为110°，C-O键长为140pm，C-O键的旋转能垒的能量很小，而水、醇与醚分子中氧的键合能力也与此相似。根据[[价键理论|价键理论]]，氧原子的杂化状态是sp<sup>3</sup>。
 
氧原子的电负性比碳更强，因此与氧连接的α氢原子酸性强于碳连接的α氢原子，然而其酸性比不上羰基α氢原子。

== 分类 ==
醚可以根据醚键是否成环分为：'''直链醚'''和'''环醚'''两大类。
分子中含有多个乙烯氧基（-CH2CH2O- ）结构单元的大环醚称为[[冠醚|冠醚]]。[[冠醚|冠醚]]是一种大环多醚，因分子形状类似王冠而得名。
在直链醚中，氧原子所连接的两个烃基相同则称为：'''单醚'''； 两个烃基不相同则称为：'''混醚'''；两个烃基中有一个或两个是芳基的称为：'''芳香醚'''。

== 命名 ==
醚的普通[[命名法|命名法]]是于烃基后加上“醚”字，习惯上对称醚的“二”字可省略。醚的两个不同基团排列顺序通常是：先小基团后大则为：苯基或芳烃基在前。英文命名则按字母顺序。冠醚的命名为 x-冠-y，x代表环总原子数，y代表环中氧原子数。

== 物理性质 ==
醚[[分子|分子]]不能互相形成[[氢键|氢键]]，因此它们具有和[[醇|醇]]类相比较低的[[沸点|沸点]]。醚具有微弱的极性，这是由于醚官能团中碳氧碳的键角约110度，而碳氧之间的极性差异没有抵消（不同于二硫化碳之类的线型分子）。醚类极性不如[[醇|醇]]、[[酯|酯]]与[[酰胺|酰胺]]类化合物，但是强于烯烃的极性。醚氧原子的孤电子对使它有可能与水分子形成氢键。

环状醚类比如[[四氢呋喃|四氢呋喃]]和[[1,4-二噁烷|1,4-二噁烷]]能与水混溶，这是因为这类醚分子的氧原子比起烷基醚（链状醚）来说更暴露于分子之外，所以极性比起后者更大。

== 反应 == 
[[File:Diethylether_peroxide_chemical_structure.png|frame]]
醚类总体来说化学[[活性|活性]]都较低，但强于烷烃（[[环氧化物|环氧化物]]、[[酮|酮]]、[[缩醛|缩醛]]参见各自条目）。

与醚相关的重要反应讲解如下：<ref name=Ullmann>Wilhelm Heitmann, Günther Strehlke, Dieter Mayer "Ethers, Aliphatic" in Ullmann's Encyclopedia of Industrial Chemistry" Wiley-VCH, Weinheim, 2002. {{DOI|10.1002/14356007.a10_023}}</ref>

=== 醚的裂解 ===
虽然醚不能轻易的发生[[水解反应|水解反应]]，但于酸性条件下（如：[[氢溴酸|氢溴酸]]或[[氢碘酸|氢碘酸]]），可断裂为醇。[[盐酸|盐酸]]条件下，断裂醚的速度非常慢。
如：甲醚制备溴代甲烷：
:ROCH<sub>3</sub>  +  HBr   →   CH<sub>3</sub>Br  +  ROH
该类反应都是通过[[鎓|鎓]]中间体进行，如：
[RO(H)CH<sub>3</sub>]<sup>+</sup>Br<sup>-</sup>.

有些醚类能于[[三溴化硼|三溴化硼]]条件中迅速分解，有时还可用更强的路易斯酸如：[[三氯化铝|三氯化铝]]反应，制备溴代烷烃。<ref>{{OrgSynth | author = J. F. W. McOmie and D. E. West| title = 3,3'-Dihydroxylbiphenyl| collvol = 5 | collvolpages = 412| year = 1973 | prep = CV5P0412}}</ref> 这些都取决于取代基的不同，还有些醚类可在特定的试剂中发生裂解，如非常强的碱。

=== 过氧化物 ===
临近氧原子，含有CH基团的一级醚或二级醚易形成有机[[过氧化物|过氧化物]]，如：[[乙醚过氧化物|乙醚过氧化物]]。该反应需要氧气（或者空气）参与，且在光、[[金属|金属]]和醛的催化下加速反应。得到的过氧化物在高温或高浓度下具[[爆炸|爆炸]]性，因此操作[[二异丙基醚|二异丙基醚]]、[[四氢呋喃|四氢呋喃]]等醚类溶剂大多都不可蒸干，而厂商在分装醚类试剂也会放入一些稳定剂，防止大量产生过氧化物。

=== 作为路易斯碱 ===
醚类可作为[[路易斯碱|路易斯碱]]及[[Bronsted碱|Bronsted碱]]。强酸与醚发生反应，可提供给醚氧原子一个质子，从而形成：“鎓离子”，如：乙醚可以与[[三氟化硼|三氟化硼]]形成络合物(BF<sub>3</sub><sup>.</sup>OEt<sub>2</sub>) 。醚还可与二价[[镁|镁]]络合成[[格氏试剂|格氏试剂]]。多聚醚都可与[[碱金属|碱金属]]离子牢固的结合。

=== α-卤代反应 ===
该反应类似于醚的α氢原子发生过氧化的反应，如：单质[[氯|氯]]制备α-氯代醚的反应。

== 合成 ==<!-- 该章节链接至[[有机反应|有机反应]]-->
醚在实验室条件下可通过许多方法合成：

=== 醇的脱水 ===
[[醇|醇]]可通过[[脱水反应|脱水反应]]制备醚：
: 2 R-OH → R-O-R +[[水|H<sub>2</sub>O]]（高温下）

该反应过程需要高温（通常在125°C）。该反应还需要酸的催化（通常为硫酸）。上述方法对于制备对称醚来说有效，但对于不对称醚却无能为力，如：乙醚易于通过此法制备，环醚也同样可用此方法制备（分子内脱水）。另外此方法还会引入一定的副产物，如分子内脱水产物：
: R-CH<sub>2</sub>-CH<sub>2</sub>(OH) → R-CH=CH<sub>2</sub> + H<sub>2</sub>O

另外此法只能合成一些简单的醚，对于复杂的分子醚类分子不太试用。对于复杂分子则需要更温和的条件来合成。

=== 威廉姆逊醚合成 ===
[[卤代烃|卤代烃]]和[[醇盐|醇盐]]发生[[亲核取代反应|亲核取代反应]]：
: R-ONa + R'-X   →   R-O-R' + Na[[卤素|X]]

该反应称作：[[威廉姆逊合成|威廉姆逊合成]]。该反应通过用强[[碱|碱]]处理醇，形成[[醇盐|醇盐]]，而后与带有合适[[离去基团|离去基团]]的烃类分子反应。这里的离去基团包括：[[碘|碘]]、[[溴|溴]]等卤素，或[[磺酸酯|磺酸酯]]。该方法对于芳香卤代烃一般不适用（如：溴苯，参见[[Ullmann缩合|Ullmann缩合]]）。该方法还只局限于一级卤代烃才可得到较好的收率，对于二级卤代烃与三级卤代烃则由于太易生成E2消除产物而不适用。

在相似的反应中，烷基卤代烃还可与[[酚负离子|酚负离子]]发生亲核取代反应。R-X虽不能与醇反应，但[[酚|酚]]却能够进行该反应（酚酸性远高于醇），它可通过一个强[[碱|碱]]，如：[[氢化钠|氢化钠]]先形成酚负离子再进行反应。酚可取代卤代烃中的X离去基团，形成[[酚醚|酚醚]]的结构，该过程为[[SN2|SN2]]机理。

: C<sub>6</sub>H<sub>5</sub>OH + OH<sup>-</sup> → C<sub>6</sub>H<sub>5</sub>-O<sup>-</sup>  +  H<sub>2</sub>O

: C<sub>6</sub>H<sub>5</sub>-O<sup>-</sup> + R-X   →   C<sub>6</sub>H<sub>5</sub>OR

=== Ullmann二芳醚合成 ===
[[Ullmann二芳醚合成|Ullmann二芳醚合成]]的反应很类似于威廉姆逊反应，不同之处在于底物是芳香卤代烃。该反应需要催化剂才能进行，如：[[铜|铜]]。

=== 醇对于烯烃的亲电加成反应 ===
醇可与活化后的[[烯烃|烯烃]]进行亲电加成：
:[[烯烃|R<sub>2</sub>C=CR<sub>2</sub>]]+ R-OH → R<sub>2</sub>CH-C(-O-R)-R<sub>2</sub>
该反应需要[[酸|酸]][[催化|催化]]，三氟醋酸汞(Hg(OCOCF<sub>3</sub>)<sub>2</sub>)常可作为这种反应的催化剂，反应生成具有[[弗拉基米尔·瓦西里耶维奇·马尔科夫尼科夫|弗拉基米尔·瓦西里耶维奇·马尔科夫尼科夫]]（Markovnikov）立体化学的醚类。使用相似的反应条件，[[四氢吡喃醚|四氢吡喃醚]]（THP）可作为一种醇的[[保护基|保护基]]。

=== 制备环氧化合物===
{{main|环氧化合物}}
[[环氧化合物|环氧化合物]]通常由烯烃氧化制备。在工业生产中，最重要的环氧化合物是：[[环氧乙烷|环氧乙烷]]，它通过乙烯和氧气制备。其他的过氧化合物还可通过以下方法制备：
* 通过[[过氧酸|过氧酸]]和烯烃来制备，如：[[间氯过氧苯甲酸|间氯过氧苯甲酸]]（mCPBA）。
* 通过[[卤代醇|卤代醇]]分子内的亲核取代反应来制备。
