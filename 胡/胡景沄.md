'''胡景沄'''（{{bd|1909年|10月13日|1995年|12月23日}}），[[山西省|山西省]][[文水县|文水县]]人，1938年加入[[中国共产党|中国共产党]]。[[民国时期|民国时期]]曾担任[[冀南银行|冀南银行]]副总经理、总经理，[[太行工商管理总局|太行工商管理总局]]局长，[[华北银行|华北银行]]副总经理。1949年后，历任[[中国人民银行|中国人民银行]]副行长、[[中国农业银行|中国农业银行]]行长、[[交通銀行|交通银行]]董事长、[[中国人民保险集团|中国人民保险公司]]总经理等职<ref name="diyi">{{cite web|url=http://shuo.news.esnai.com/article/201701/149142.shtml|title=第一家登上《人民日报》的会计师事务所|accessdate=2018-03-07|author=张辉|date=2017-01-05|publisher=中国会计视野|archiveurl=https://web.archive.org/web/20180307151131/http://shuo.news.esnai.com/article/201701/149142.shtml|archivedate=2018-03-07|deadurl=no}}</ref>。

== 早年生平 ==
1909年10月13日，胡景沄出生于山西省文水县[[汾河|汾河]]谷地的南胡家堡村<ref name="dianji">{{cite journal|author1=高星|title=记新中国金融奠基人胡景澐|journal=金融博览|date=2016|issue=13|pages=68-70}}</ref>，父亲胡理治是[[阎锡山|阎锡山]]手下的一名军人。胡景沄3岁时，其父亲把守[[娘子关|娘子关]]时失守被处斩。胡景沄在7岁时入小学读书。1923年，14岁时辍学到[[太谷县|太谷县]][[太谷农工银行|太谷农工银行]]做学徒。太谷农工银行位于太谷县县城西大街，成立于1922年，次年正式开业，首任大掌柜胡世泽，行中主要管理人员皆是[[票庄|票庄]]出身。在行内，一般店员称胡景沄为二师兄，深受掌柜伙计们的喜爱<ref name="tuohuang">{{cite web|url=http://shh.sinoins.com/2015-07/17/content_162813.htm|title=胡景澐：拓荒者|accessdate=2018-03-23|author=陈国庆|date=2015-07-17|publisher=中保网|archiveurl=https://web.archive.org/web/20180319174541/http://shh.sinoins.com/2015-07/17/content_162813.htm|archivedate=2018-03-19|deadurl=no}}</ref>。1928年，胡景沄考上[[山西银行专科学校|山西银行专科学校]]，进一步学习银行业务知识，1931年毕业后在太原晋恒造纸厂等处从事会计、审计工作<ref name="dianji"/>。

== 投身革命 ==
1937年8月，胡景沄参加[[八路军|八路军]]，由于在游击干部培训班表现突出，先后任太谷县八路军游击队三大队副队长、支队参谋、太行纵队司令部合作社社长<ref name="tuohuang"/>。1938年1月，胡景沄加入[[中国共产党|中国共产党]]，同年转任太行纵队司令部太行合作总社四科副科长、社长<ref name="dianji"/>。

1939年秋，[[中共中央北方局|中共中央北方局]]和[[晋冀边区|晋冀边区]]政府决定建立自己的金融机构，10月，[[冀南银行|冀南银行]]在山西省[[黎城县|黎城县]]成立，胡景沄被调往冀南银行工作，任副行长。1942年，在日本侵略者对太行山根据地发动的“大[[掃蕩|扫荡]]”，胡景沄被日军俘虏，后关押在山东德州监狱。由于胡景沄坚称自己为棉花商人，后经营救获释，出狱后辗转回到八路军总部[[左权县|左权县]]。其后，胡景沄先后任冀南银行行长、太行工商总局局长、晋冀鲁豫中央财经办事处金融处处长等职。1945年6月10日，晋冀鲁豫边区会计学会成立，胡景沄任学会理事长。1946年，胡景沄受[[中共晋冀鲁豫中央局|中共晋冀鲁豫中央局]]指派化名胡竹轩到河北[[邯郸|邯郸]]筹建民营性质的[[瑞华银行|瑞华银行]]。6月11日，瑞华银行成立。<ref name="tuohuang"/>

1948年7月22日，[[晋察冀边区银行|晋察冀边区银行]]与冀南银行在河北省[[平山县|平山县]]合并成立[[华北银行|华北银行]]，[[南汉宸|南汉宸]]任总经理，胡景沄和晋察冀边区银行经理[[关学文|关学文]]任副经理。随后华北银行又与[[北海银行|北海银行]]、[[晋西北农民银行|晋西北农民银行]]、[[陕甘宁边区银行|陕甘宁边区银行]]等合并于1948年12月1日在河北省[[石家庄市|石家庄市]]成立[[中国人民银行|中国人民银行]]，南汉宸、胡景沄和关学文三人职务未变。<ref name="tuohuang"/>

1949年1月15日，中国人民解放军占领[[天津市|天津]]，当时的天津是中国的金融重镇，在[[解放北路_(天津市)|解放北路]]分布有49家中外银行，其中12家为中国各银行的总部所在。为了做好对天津的金融接管，中国人民银行特别成立的金融接管处，由胡景澐亲自率队赴天津开展工作。1949年2月底，胡景澐回到已在当月早些时候迁到[[北平市|北平]]的中国人民银行总行。

== 中共建政后 ==
1949年10月19日，胡景澐被[[中央人民政府委员会|中央人民政府委员会]]正式任命为中国人民银行副行长。1949年10月20日，[[中国人民保险公司|中国人民保险公司]]在北京成立，胡景澐兼任总经理。1951年，胡景沄以中国人民银行副行长和中国人民保险公司总经理名义率团访问[[苏联|苏联]]的银行和苏联国家保险局<ref>{{cite web |url = http://finance.ifeng.com/money/insurance/hydt/20110812/4385554.shtml |title=胡景澐：忠贞保险魂 |accessdate=2018-03-23|date=2011-08-12|publisher=凤凰网|archiveurl=https://web.archive.org/web/20180312150527/http://finance.ifeng.com/money/insurance/hydt/20110812/4385554.shtml|archivedate=2018-03-12|deadurl=no}}</ref>。1952年，中国人民保险公司划归[[中央人民政府财政部|中央人民政府财政部]]领导，总经理由财政部部长[[吳波_(政治人物)|吴波]]兼任<ref name="tuohuang"/>。1952年12月，[[公私合营银行|公私合营银行]]在上海成立，胡景沄以中国人民银行副行长身份兼任公私合营银行联合董事会董事长<ref>{{cite web|url=http://www.shtong.gov.cn/newsite/node2/node2245/node75491/node75498/node75569/node75591/userobject1ai92408.html|title=第十三节 公私合营银行|accessdate=2018-03-23|publisher=上海市地方志办公室}}</ref><ref>{{cite web|url=http://shh.sinoins.com/2015-02/13/content_146164.htm|title=周作民与银行、保险百年渊源|accessdate=2018-03-23|date=2015-02-13|publisher=中保网}}</ref>。

1963年11月16日，胡景澐被任命为[[中国农业银行|中国农业银行]]行长<ref>{{cite web|url=http://finance.people.com.cn/bank/GB/202484/14862594.html|title=中国农业银行历届行长简介 (4)|accessdate=2018-03-23|date=2011-06-09|publisher=人民网|archiveurl=https://web.archive.org/web/20180319174518/http://finance.people.com.cn/bank/GB/202484/14862594.html|archivedate=2018-03-19|deadurl=no}}</ref>、党组书记。中国农业银行曾与中国人民银行多次合并分拆，1963年11月是农行的第三次建立，工作人员多为原人民银行的农村金融相关工作人员，由[[中华人民共和国国务院|国务院]]直属独立运行。1965年11月，中国农业银行与中国人民银行再次合并<ref>{{cite web|url=http://www.abchina.com/cn/AboutABC/nhfm/nhlch/201511/t20151127_808534.htm|title=农行历程|accessdate=2018-03-23|date=2015-11-27|publisher=中国农业银行|archiveurl=https://web.archive.org/web/20180319174518/http://www.abchina.com/cn/AboutABC/nhfm/nhlch/201511/t20151127_808534.htm|archivedate=2018-03-19|deadurl=no}}</ref>。文化大革命期间，胡景澐遭受迫害，曾被强迫在银行大楼的厕所打扫卫生，还曾到银行五七干校劳动。1979年胡景澐被平反后，回中国人民银行工作。1979年11月29日，中国保险学会成立，胡景澐担任会长。1982年12月，胡景澐从中国人民银行离职休养。1995年12月23日，胡景澐在北京逝世，享年86岁。<ref name="tuohuang"/>
