'''胡光宝'''（{{bd|1934年|12月|2019年|2月20日}}），[[河北|河北]][[丰南|丰南]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]政府官员，曾任[[中共中央办公厅|中共中央办公厅]]原副主任负责中央办公厅的常务工作，正部长级）<ref name=jl>{{Cite web |url =http://news.sohu.com/20060215/n241846089.shtml  |title =简历：全国人大常务委员会委员胡光宝  |author =  |date =  2006-02-15|publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref>。

== 生平 ==
1956年加入中国共产党。1952年自东北工业会计统计专科学校毕业。1952年3月参加工作。历任[[国家建委|国家建委]]核心组办公室副处长，[[中华人民共和国进出口管理委员会|中华人民共和国进出口管理委员会]]办公厅副主任，[[国务院办公厅|国务院办公厅]]特区组副组长，[[国务院特区办公室|国务院特区办公室]]副主任，[[中央宣传思想工作领导小组|中央宣传思想工作领导小组]]成员，[[中共中央办公厅|中共中央办公厅]]副主任（正部长级）。1992年至1993年5月任国务院特区办公室副主任。1993年任[[全国拥军优属拥政爱民工作领导小组|全国双拥工作领导小组]]副组长。2000年4月29日[[第九届全国人大常委会|第九届全国人大常委会]]第十五次会议通过，补充任命胡光宝为第九届[[全国人大法律委员会|全国人大法律委员会]]副主任委员。他是[[中共十五大|中共十五大]]代表，[[第九届全国人大常委会|第九届全国人大常委会]]委员。2003年4月26日[[第十届全国人大常委会|第十届全国人大常委会]]第二次会议通过，任命胡光宝为第十届[[全国人大常委会代表资格审查委员会|全国人大常委会代表资格审查委员会]]副主任委员。2003年3月17日[[第十届全国人民代表大会第一次会议|第十届全国人民代表大会第一次会议]]通过，任命胡光宝为第十届全国人大法律委员会副主任委员<ref name=jl/><ref>{{Cite web |url = http://www.people.com.cn/item/zgjgk/jgmd/rdzwhmd.html |title = 第九届全国人民代表大会专门委员会组成人员名单 |author =  |date =2000-05-01 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =https://news.qq.com/a/20060302/001006.htm  |title =资料：第十届全国人大常委会  |author =  |date =2006-03-02 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。

2019年2月20日胡光宝因病医治无效在北京逝世，享年85岁。
