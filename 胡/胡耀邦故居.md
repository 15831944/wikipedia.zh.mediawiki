'''胡耀邦故居'''指的是曾经在八十年代担任[[中国共产党中央委员会总书记|中国共产党中央委员会总书记]][[胡耀邦|胡耀邦]]的故居。位于[[中华人民共和国|中华人民共和国]][[湖南省|湖南省]][[浏阳市|浏阳市]][[中和镇|中和镇]]苍坊村敏溪河畔，始建于[[清朝|清朝]][[咸丰|咸丰]]年间，占地面积450平方米，共计19间。<ref name="fhsi" >{{cite web |url=http://news.qq.com/a/20130507/000156.htm?pgv_ref=aio2012&ptlang=2052 |title= 胡耀邦故居成全国重点文保单位 去年获批4A景区|accessdate=2013-5 |publisher=腾讯网 |work= |language=zh-hans}}</ref><ref name="time" >{{cite web |url=http://travel.sohu.com/20140414/n398197691.shtml?pvid=6aa0c692c0b968f8 |title=走进湖南胡耀邦故居 |accessdate=2014-4 |publisher=搜狐网 |work= |language=zh-hans}}</ref>

==历史==
1915年11月20日，胡耀邦出生在此。<ref name="fhsi" />

1930年，15岁的胡耀邦离开此地。<ref name="fhsi" />

1963年1月，胡耀邦回浏阳农村调查期间，在故居客房住了一晚。<ref name="fhsi" />

== 建筑 ==
胡耀邦故居是坐北朝南的砖木江南民居，青瓦尖顶。<ref name="fhsi" />

===陈列馆===
2005年2月，胡耀邦故居陈列馆开始修建，2006年11月投入使用，占地13000多平方米，建筑面积3800平方米，展厅面积2100多平方米，融合现代科技用影像、照片、雕塑、文物等展现胡耀邦的一生经历。

==保护==
1995年，故居修复并且对外开放。

1996年，故居被[[湖南省人民政府|湖南省人民政府]]列为湖南省重点文物保护单位。

2002年，故居被湖南省人民政府列为湖南省[[爱国主义教育基地|爱国主义教育基地]]。

2007年，故居申报批准为长沙市党员教育基地。

2009年，故居申报批准为国家三级博物馆、湖南省党风廉政教育基地等。

2012年1月，胡耀邦故里旅游景区获批国家AAAA级景区。<ref name="bee" >{{cite web |url=http://politics.people.com.cn/n/2013/0506/c1001-21375524.html |title= 胡耀邦故居列入第七批全国重点文物保护单位|accessdate=2013-5 |publisher=人民网 |work= |language=zh-hans}}</ref>

2013年5月，胡耀邦故居列入第七批全国重点文物保护单位。<ref name="bee" />

2014年4月，已经退休的前中共中央总书记、国家主席、中央军委主席[[胡锦涛|胡锦涛]]到胡耀邦故居参观。<ref name="tiger" >{{cite web |url=http://money.163.com/14/0414/11/9PPO0T5S00253B0H.html |title=港媒：胡锦涛到访胡耀邦故居 向其铜像鞠躬致敬 |accessdate=2014-4 |publisher= 163.COM|work= |language=zh-hans}}</ref>
