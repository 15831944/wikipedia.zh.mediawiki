'''胡絜青'''（{{bd|1905年|12月23日|2001年|5月21日}}）原名'''玉贞'''，笔名'''燕岸'''、'''胡春'''，女，[[满洲正红旗|满洲正红旗]]人，[[中华人民共和国|中华人民共和国]]画家。

==生平==
1931年自[[北京师范大学|北京师范大学]]国文系毕业。自幼喜爱文艺、绘画、书法。曾受画家[[汪采白|汪采白]]（汪礼祁）、[[孙诵昭|孙诵昭]]、[[杨仲子|杨仲子]]的影响。[[中华人民共和国|中华人民共和国]]成立后，专业从事绘画，在历届全国画展、书展都有作品参展，并且多次获奖。1958年受聘于[[北京画院|北京中国画院]]，是一级美术师。历任中国画研究会常务理事、[[中国美术家协会|中国美术家协会]]会员、[[中国书法家协会|中国书法家协会]]会员、[[满族|满族]]书画研究会会长、北京市文联顾问、中国画研究会顾问、北京中国花鸟画研究会顾问等。她是第二至五届北京市政协委员，第五、六、七届[[全国政协委员|全国政协委员]]，第二至四届全国文代会代表<ref>{{Cite web |url =http://www.xinhuanet.com/shuhua/2015-12/30/c_128581490.htm  |title =老舍、胡絜青简介  |author =  |date =2015-12-30  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

2001年5月21日，胡絜青在[[北京|北京]]和平里医院病逝，享年96岁<ref>{{Cite web |url = http://news.sohu.com/91/10/news145281091.shtml |title = 心平气和、随遇而安 老舍夫人胡絜青平静辞世 |author =  |date = 2001-05-22 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

2005年8月24日是胡絜青的丈夫[[老舍|老舍]]自杀身亡39周年忌日。2005年8月23日，其子[[舒乙|舒乙]]在27年后再次开启父亲的[[骨灰盒|骨灰盒]]，将老舍1966年[[文革|文革]]受难时穿的一件血衣的残片放入盒中；同日即8月23日，老舍、胡絜青夫妇合葬在[[北京八宝山革命公墓|北京八宝山革命公墓]]一区的苍松翠柏下，该陵墓由舒乙设计<ref>{{Cite web |url = http://news.163.com/05/0824/14/1RU61FMC00011211.html |title =老舍夫妇合葬八宝山 血衣残片代替肉身  |author =  |date =  2005-08-24|publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。
