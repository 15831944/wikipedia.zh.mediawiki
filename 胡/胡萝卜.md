'''胡蘿蔔'''（[[學名|學名]]：{{sname|''Daucus carota'' subsp. ''sativus''}}），又称'''紅蘿蔔'''、'''番蘿蔔'''、'''丁香蘿蔔'''<ref>{{cite web |title=丁香萝卜 |url=http://shop.bytravel.cn/produce3/dingxiangluobo.html}}</ref>、'''胡芦菔金'''、'''赤珊瑚'''、'''黄根'''、'''甘筍'''<ref>香港區分不同種胡蘿蔔</ref>、'''金筍'''<ref>[http://gjk.scib.ac.cn/hnzwz/hnzwzfile1-4/3-182.pdf 《海南植物志》]</ref>、'''紅菜頭'''（[[臺語|臺語]]、[[閩南語|閩南語]]、[[潮汕话|潮汕话]]、[[客家語|客家語]]）等，是[[繖形科|繖形科]][[胡蘿蔔属|胡蘿蔔属]][[二年生植物|二年生植物]]，以呈肉質的[[根|根]]作[[蔬菜|蔬菜]]食用。

== 名字由来 ==
[[File:Daucus_carota_flowers,_peen_'Napoli'_(1).jpg|thumb]]
胡萝蔔原产于[[亚洲|亚洲]]的西南部，[[阿富汗|阿富汗]]为最早[[演化|演化]]中心，栽培历史在两千年以上。公元10世纪从[[伊朗|伊朗]]引入[[欧洲|欧洲]]大陆，15世纪见于[[英国|英国]]，发展成欧洲生态型，尤以[[地中海|地中海]]沿岸种植最多。16世纪传入[[美洲|美洲]]。

约在13世纪，胡萝蔔从[[伊朗|伊朗]]引入[[中国|中国]]，发展成中国生态型<ref>{{cite web |author1=Pclady |title=胡萝卜的营养价值有哪些 |url=http://hlj.ifeng.com/a/20180528/6610094_0.shtml |website=凤凰网黑龙江}}</ref>，根据[[李时珍|李时珍]]《[[本草纲目|本草纲目]]》的记载，因为从胡地传来，味道像[[萝卜|萝卜]]，故名「胡萝蔔」。中国的胡萝蔔以[[山东|山东]]、[[河南|河南]]、[[浙江|浙江]]、[[云南|云南]]等省份种植最多。胡萝蔔于16世纪从中国传入[[日本|日本]]，當時漢字名為「[[人蔘|人參]]」。

== 形态 ==
[[File:Carrots_of_many_colors.jpg|thumb]]
胡萝蔔为两年生草本植物，有9对[[染色体|染色体]]，属半耐寒性长日照植物，喜冷凉气候。三回羽状全裂叶，丛生于单生的短缩茎上。顶端各着生一复伞形[[花序|花序]]，有糙硬毛，花朵为白色或淡红色，花期5至7月。异花传粉。双悬果，果实圆卵形。肉质根有长筒、短筒、长圆锥及短圆锥等不同形状，白、黄、橙、橙红、青、紫等不同颜色。

== 栽培 ==
胡萝蔔一般在[[秋|秋]]季播種冬季收获，不过也可以进行反季節栽培，也就是春天播種夏天收获。主要是看所在地域，因地制宜。胡萝蔔肉质根在[[摄氏温标|摄氏温标]]{{convert|18|to|23|°C}}时发育良好，发芽温度为摄氏温标{{convert|20|to|25|°C}}。由于胡萝蔔[[根|根]]系发达，保持土质疏松对于胡萝蔔的生长有利，适宜种植在沙质土壤之中。栽培的前期不能過量施水，以防地面部分生长过度。后期则要保证供水充足以使肉质根能够充分发育<ref>[http://shucai.ag365.com/shucai/huluobo/2006/2006051947236.html 胡萝蔔栽培技术]</ref>。优秀的胡萝蔔栽培种在合适条件下，最高[[亩|亩]]产可达6000[[公斤|公斤]]（每[[公顷|公顷]]90[[吨|吨]]）其常见病虫害包括黑斑病、{{tsl|en|Black rot|黑腐病}}<ref>{{cite web|url=https://market.cloud.edu.tw/content/vocation/garden/tc_ag/dis_ind/pn3.htm |title=十字花科黑腐病 ( Black rot ) |publisher=教育大市集 |date= |accessdate=2016-08-09}}</ref><ref>{{cite web|url=http://web.tari.gov.tw/techcd/%E6%9E%9C%E6%A8%B9/%E5%B8%B8%E7%B6%A0%E6%9E%9C%E6%A8%B9/%E6%9C%A8%E7%93%9C/%E7%97%85%E5%AE%B3/%E6%9C%A8%E7%93%9C-%E9%BB%91%E8%85%90%E7%97%85.htm |title=果樹-木瓜 |publisher=Web.tari.gov.tw |date= |accessdate=2016-08-09}}</ref><ref>{{cite web|url=http://kmweb.coa.gov.tw/techcd/%E4%BD%9C%E7%89%A9%E7%97%85%E8%9F%B2%E5%AE%B3%E8%88%87%E8%82%A5%E5%9F%B9%E7%AE%A1%E7%90%86%E6%8A%80%E8%A1%93%E8%B3%87%E6%96%99/%E8%94%AC%E8%8F%9C/%E8%91%89%E8%8F%9C%E9%A1%9E/%E7%94%98%E8%97%8D/%E7%97%85%E5%AE%B3/%E7%94%98%E8%97%8D-%E9%BB%91%E8%85%90%E7%97%85.htm |title=蔬菜－甘藍 |publisher=Kmweb.coa.gov.tw |date= |accessdate=2016-08-09}}</ref>、根结[[线虫|线虫]]病、{{tsl|en|Bacterial soft rot|软腐病}}<ref>{{cite web|url=https://www.baphiq.gov.tw/userfiles/09%281%29.pdf |title=細菌性軟腐病之診斷與鑑定 |format=PDF |date= |accessdate=2016-08-09}}</ref><ref>{{cite web|url=http://kmweb.coa.gov.tw/techcd/%E4%BD%9C%E7%89%A9%E7%97%85%E8%9F%B2%E5%AE%B3%E8%88%87%E8%82%A5%E5%9F%B9%E7%AE%A1%E7%90%86%E6%8A%80%E8%A1%93%E8%B3%87%E6%96%99/%E8%94%AC%E8%8F%9C/%E6%A0%B9%E8%8E%96%E8%8F%9C%E9%A1%9E/%E5%A4%A7%E8%92%9C/%E7%97%85%E5%AE%B3/%E5%A4%A7%E8%92%9C-%E8%BB%9F%E8%85%90%E7%97%85.htm |title=軟腐病 (Bacterial soft rot) |publisher=Kmweb.coa.gov.tw |date= |accessdate=2016-08-09}}</ref><ref>[https://www.baphiq.gov.tw/Publish/plant_protect_pic_6/orchidPDF/02-6.pdf 軟腐病 - 胡萝蔔[[营养|营养]]丰富。人们一般食用其肉质根，有时也食用胡萝蔔叶。胡萝蔔种子内含有挥发性油。胡萝蔔根可以直接生食，也可加工成块、丁、丝同其他食材一同烹饪。胡萝蔔汁也是一种常见的胡萝蔔加工制成品，胡萝蔔汁或胡萝蔔泥也可和面粉或米粉等淀粉类物质混合做成糕饼类食物<ref>[http://www.xiaochu.cn/html/caipu/shijie/hanguoliaoli/2009/0824/5561.html 胡萝蔔饼——韩式风味] {{webarchive|url=https://web.archive.org/web/20100809084025/http://www.xiaochu.cn/html/caipu/shijie/hanguoliaoli/2009/0824/5561.html |date=2010-08-09 }}</ref>。还可[[醃|醃]]制、[[醬|醬]]渍、制干或作[[饲料|饲料]]。其风味主要来自于[[萜烯|萜烯]]类物质，该物质味道较为独特，并非所有人都能接受。

胡萝蔔的营养成分中，最重要的就是因其得名的[[胡蘿蔔素|胡蘿蔔素]]，胡萝蔔根内含有[[Α-胡萝卜素|α]]、[[Β-胡萝卜素|β]]（大多藏在外皮）、γ、ε-胡萝蔔素和番茄烃、六氢番茄烃等[[類胡蘿蔔素|類胡蘿蔔素]]，胡萝蔔素有治疗[[夜盲症|夜盲症]]、保护[[呼吸道|呼吸道]]和促进儿童生长等功能。此外，胡萝蔔还含较多的维生素，[[钙|钙]]、[[磷|磷]]、[[铁|铁]]等[[矿物质|矿物质]]，淀粉、[[纤维素|纤维素]]等糖类物质。

== 文化 ==
有些时候，胡萝蔔被视作是赏赐的代名词。<ref>{{cite web |title=红萝卜名字由来 为什么叫胡萝卜 |url=http://news.daxues.cn/name/2017/375823.html |website=大学生知识网}}</ref>如政治用语[[胡蘿蔔加大棒|胡蘿蔔加大棒]]，意思为赏赐与威逼共同使用，以使对手同意自己的要求。在流行文化中（原来在西方最普遍），胡萝蔔被视为[[兔|兔子]]最喜欢的食物（但其實是錯誤的），这一组合如同[[鼠|老鼠]]与[[乾酪|奶酪]]一样，经常用在[[寓言|寓言]]、[[笑話|笑話]]甚至[[管理学|管理学]]中。

== 不同名稱的使用 ==
在香港不同種的紅蘿蔔會分別命名為紅蘿蔔或甘筍。紅蘿蔔個子比較大，纖維較粗，生食時口感差，適合用來煲湯；反之甘筍個子較小，纖維比較幼細，可供生食，如做沙律，但卻不宜煲湯。
