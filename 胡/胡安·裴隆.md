'''胡安·多明戈·裴隆'''（{{lang-es|Juan Domingo Perón}}；{{bd|1895年|10月8日|1974年|7月1日|catIdx=Peron, Juan}}），[[阿根廷總統|阿根廷總統]]、[[阿根廷副總統|阿根廷副總統]]、[[阿根廷|阿根廷]][[政治家|政治家]]，[[阿根廷|阿根廷]]迄今為止任職時間第二長的[[阿根廷總統|阿根廷總統]]，統治阿根廷長達11年。

== 早年生涯 ==
1895年10月8日，裴隆出生於[[阿根廷|阿根廷]][[布宜诺斯艾利斯省|布宜诺斯艾利斯省]]的一个[[农民|农民]]家庭。他在16岁时进入军事学校学习。毕业后仕途顺利。1930年代，他作为军事观察员在[[意大利|意大利]]服役。

1929年1月5日，裴隆与[[奥莱利亚·提宗|奥莱利亚·提宗]]结婚，13年后奥莱利亚死于子宫癌。

== 發動政变奪權 (1943年-1944年) ==
1943年5月，作为[[阿根廷陆军|阿根廷陆军]]上校的裴隆在军官联合小组发动的[[军事政变|军事政变]]中扮演了重要角色，声名鹊起，取得政權。同年11月，他被任命为劳动和福利部长。

== 副总統 (1944年-1945年) ==
1944年2月，裴隆被任命为副总统及[[陆军|陆军]]部长，掌握軍政大權。

1945年10月9日，裴隆在军队内部的反对声中被迫[[辞职|辞职]]，并一度遭到[[逮捕|逮捕]]。随即[[工会|工会]]组织了大规模示威活动，10月17日，裴隆被释放，並成立工黨（[[正義黨|正義黨]]前身）準備參加新的總統選舉。

== 第一任总統 (1946年-1952年) ==
1946年2月24日举行的大选中，他以56%的得票当选成為[[阿根廷总统|阿根廷总统]]。1946年6月4日，裴隆成为了总统。

裴隆上任后采取有利于[[工人阶级|工人阶级]]的政策。他大规模扩大了加入[[工会|工会]]的[[工人|工人]]数量，帮助建立了[[劳工总同盟|劳工总同盟]]，他提拔了工人阶级出生的官员进入他的内阁。他将此称为介于资本主义和社会主义之间的“[[第三位置|第三位置]]”，后來被人称为“[[庇隆主义|庇隆主义]]”，而劳工总同盟作為全國最大的工會，一直是[[正義黨|正義黨]]的重要支持者。他致力于经济独立，反对[[美国|美国]]和[[英国|英国]]，没收了这两个[[国家|国家]]在[[阿根廷|阿根廷]]的大量资产。他致力于推进国家的[[工业化|工业化]]，1947年，他颁布了旨在发展国有工业的第一个五年计划。他的思想成为了阿根廷裴隆主义的主流。1940年代后期，在他的允许下，阿根廷成为了[[纳粹|纳粹]]战犯的避难地。

他在第一次任期内大大提高了劳动者的工资，并且建造了将近500,000栋新房，其中大部分是给低收入者的公寓。仅在1954年，就有近100,000栋楼房被兴建。裴隆政府的建造工程还包括一大批学校、医院、诊所等设施。

裴隆在任期間，拒绝了使用军队镇压[[同性恋|同性恋]]者。他还下令警察和司法系统对同性恋者采取宽容态度。他改良了[[探戈|探戈]]，还使[[卖淫|卖淫]]合法化，在當時仍較為保守的阿根廷社會，算是比較開明。

1945年裴隆与[[伊娃·裴隆|伊娃·裴隆]]（即著名的裴隆夫人）结婚。在她的帮助下，裴隆在劳工和妇女团体中获得了更多的支持。

== 第二任總統 (1952年-1955年) ==
1951年，裴隆順利获得连任。隨後伊娃于1952年死于子宫癌（一说白血病），年仅33岁。但是经济问题，政府高度的貪污腐败，以及与[[罗马天主教会|罗马天主教会]]之间的矛盾最终导致他在1955年9月的军事政变中被推翻，未能完成六年的總統任期。

== 流亡 (1955年-1973年) ==
被迫下台后，他先流亡到[[巴拉圭|巴拉圭]]，后来在[[西班牙|西班牙]][[马德里|马德里]]定居，受當地的[[佛朗哥|佛朗哥]][[極右翼|極右翼]][[法西斯|法西斯]][[軍事獨裁|軍事獨裁]][[政權|政權]]保護。1961年，他与夜总会歌手[[伊莎貝爾·裴隆|伊莎貝爾·裴隆]]结婚。

1950年代到1960年代，阿根廷軍人和文人政府更迭频繁，裴隆在海外仍試圖操縱國內政治。[[阿根廷经济|阿根廷经济]]增长缓慢，社会动荡剧烈。1960年代后期至1970年代前期，政府未能使经济复苏，对亲裴隆的[[蒙特内罗|蒙特内罗]]（Montonero）的游击队打击不力，这些都为裴隆回国创造条件。1971年3月，[[亚历杭德罗·阿古斯丁·拉努塞|拉努塞]]将军（Alejandro Lanusse）掌权并宣布将于1973年恢复民主宪政。

== 第三任总統 (1973年-1974年) ==
1973年3月11日，阿根廷再次举行了总统大选。不過，裴隆被排除在候选人之外，但是裴隆領導的[[正义党|正义党]]卻取得了胜利，正义党候選人[[埃克托尔·何塞·坎波拉|坎波拉]]（Héctor Cámpora）当选总统。不久裴隆回國，結束18年流亡生涯。7月13日，坎波拉宣布辞职，为举行新一届大选铺平了道路。9月23日，裴隆再度当选总统。10月，他宣誓就职，其妻伊莎贝尔同时出任副总统。裴隆政权在极左与极右势力的冲突中生存，恐怖事件大量发生。政府不得不颁行一系列紧急法令以维持公共秩序。而裴隆健康也開始出現問題。

== 去世 ==
[[File:Altar_de_la_Patria_(Argentina).jpg|thumb]]
1974年7月1日，重新上台僅九個月的裴隆逝世，享壽78歲。在任期間遺下的上述问题仍未解决。他的第三任妻子、阿根廷副总统[[伊莎贝尔·裴隆|伊莎贝尔·裴隆]]接任总统。1976年3月24日，她同樣被军事政变推翻。

== 墓地 ==
[[File:Mausoleo_de_Juan_Domingo_Perón_03.jpg|thumb]]
[[File:Mausoleo_de_Juan_Domingo_Perón_02.jpg|thumb]]
裴隆被安葬在[[布宜诺斯艾利斯|布宜诺斯艾利斯]]。1987年，他的[[墓地|墓地]]遭盗掘，遗骨的双手不翼而飞。至今此案仍未破获。2006年10月17日，裴隆迁往[[布宜诺斯艾利斯|布宜诺斯艾利斯]]市西南约64公里处的新墓，该墓地也是裴隆生前与[[伊莎贝尔·裴隆|伊莎贝尔·裴隆]]的一处住所。迁坟当日即有市民欢送，又有一些裴隆主義反对派表示抗议。但裴隆的遗体最终还是迁到了造价高达100万[[美元|美元]]的新墓地，并且新墓地专门为[[伊莎贝尔·裴隆|伊莎贝尔·裴隆]]留出了空间，也就是说，两人合葬已经准备妥当，只等伊莎贝尔本人百年歸老。
