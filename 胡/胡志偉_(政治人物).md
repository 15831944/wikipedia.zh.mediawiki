'''胡志偉'''{{MH}}（{{lang-en|Wu Chi-wai}}，{{BD|1962年|10月18日||}}），[[香港|香港]]出生，現任[[香港|香港]][[黃大仙區議會|黃大仙區議會]][[瓊富|瓊富]]選區議員、[[香港|香港]][[九龍東選區|九龍東選區]][[立法會議員|立法會議員]]及[[香港民主黨|香港民主黨]]主席。[[皇仁書院|皇仁書院]]畢業，[[美國|美國]][[威斯康辛大學密爾沃基分挍|威斯康辛大學密爾沃基分挍]]經濟學碩士。

==簡歷==
在香港土生土長的胡志偉，出身於草根階層，曾在皇仁書院就讀，同窗包括前任[[律政司司長|律政司司長]][[黃仁龍|黃仁龍]]。1981年[[中學會考|中學會考]]後，胡志偉便投身於社會工作。1985年，他決定重拾書本，前往[[澳門|澳門]]修讀[[副學士|副學士]]學位，並於1987年負笈美國威斯康辛大學密爾沃基分校繼續學業，主修經濟，副修數學。1991年，完成碩士課程後，他放棄原本計劃修讀的[[博士|博士]]課程，返回香港，擔任[[立法局|立法局]]議員[[林鉅成|林鉅成]]的議員助理。

[[1994年香港區議會選舉|1994年]]首次投身於選舉，在[[黃大仙上邨|黃大仙上邨]]參選[[區議會|區議會]]但落敗，後來於1995年當選[[市政局|市政局]]議員。[[1999年香港區議會選舉|1999年]]在黃大仙瓊富選區參選區議員並當選晉身區議會，[[2003年香港區議會選舉|2003年香港區議會選舉]]以4,480票連任，所得票數為全港第二，僅次於[[葵芳_(選區)|葵芳]]選區的[[梁耀忠|梁耀忠]]<ref>{{cite web |url=http://www.elections.gov.hk/elections/dc2003/tc_chi/result/result_view_all.html |title=存档副本 |accessdate=2003-12-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20031204051528/http://www.elections.gov.hk/elections/dc2003/tc_chi/result/result_view_all.html |archivedate=2003-12-04 }}</ref>，[[2007年香港區議會選舉|2007年香港區議會選舉]]以4,370票連任，所得票數為全港最高，傳媒稱之為當屆「票王」<ref>http://specials.mingpao.com/cfm/News.cfm?SpecialsID=131&Page=2&Ne{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/results_hk.html |title=存档副本 |accessdate=2007-12-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20071217203619/http://www.elections.gov.hk/dc2007/chi/results_hk.html |archivedate=2007-12-17 }}</ref>。[[2011年香港區議會選舉|2011年香港區議會選舉]]以4,300票連任<ref>http://www.elections.gov.hk/dc2011/chi/results_wong_tai_sin.html</ref>。[[2015年香港區議會選舉|2015年香港區議會選舉]]，胡志偉再以3,907票第四度連任黃大仙區區議員<ref>http://www.elections.gov.hk/dc2015/chi/results_wong_tai_sin.html?1492574084667</ref>。

[[1998年香港立法會選舉|1998年香港立法會選舉]]，循[[體育、演藝、文化及出版界功能界別|體育、演藝、文化及出版界功能界別]]出選落敗。[[2000年香港立法會選舉|2000年立法會選舉]]參與[[九龍東|九龍東]]地區直選，排在以[[司徒華|司徒華]]、[[李華明|李華明]]為首的[[民主黨|民主黨]]參選名單第三位；[[2004年香港立法會選舉|2004年]]司徒華宣佈不再參選，胡志偉排在民主黨參選名單第二位，再次夥拍李華明出選九龍東，但未能當選。[[2008年香港立法會選舉|2008年]]以民主黨名義，在九龍東以個人名單參選，同樣未能當選。

2011年，胡志偉於[[香港城市大學|香港城市大學]]完成仲裁及爭議解決學課程，取得深造文憑<ref>{{cite web |url=http://aro.cityu.edu.hk/apps/spotlight/?p=736 |title=存档副本 |accessdate=2012-10-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20141213224817/http://aro.cityu.edu.hk/apps/spotlight/?p=736 |archivedate=2014-12-13 }} 香港城市大學</ref>。

[[2012年香港立法會選舉|2012年香港立法會選舉]]，代表民主黨於九龍東選區勝出。

[[2016年香港立法會選舉|2016年香港立法會選舉]]，再度代表[[民主黨_(香港)|民主黨]]出選[[九龍東選區|九龍東選區]]，最後取得該區票數排名第二的50,309票，成功連任。<ref>[http://www.elections.gov.hk/legco2016/chi/rs_gc_LC3.html?1473089116779 2016年立法會選舉- 九龍東選舉結果]</ref>

2016年12月民主黨領導層改選，胡志偉以92%信任票接替劉慧卿，當選為民主黨新主席。<ref>[http://hk.apple.nextmedia.com/realtime/news/20161204/560038581 胡志偉以92%信任票當選民主黨新主席]，蘋果日報，2016-12-04</ref>

2017年8月11日，民主黨召開緊急記者會，講述黨員林子健於8月10日被懷疑強力部門擄走禁錮，[[林卓廷|林卓廷]]、[[何俊仁|何俊仁]]、[[李柱銘|李柱銘]]及[[李永達|李永達]]等人高調參與記者會，協助及發佈黨友[[林子健|林子健]]了聲稱被擄事件，胡志偉並無出席。次日，胡志偉在香港電台促請警方盡快查明林子健事件<ref>[http://news.rthk.hk/rthk/ch/component/k2/1347787-20170812.htm?nsukey=QWiMGLdIpYGk7jMKleqUOnaH7oOE0DUiOyNlscjdZcoNJ2iPF%2B6Uxn%2BX9fIpcWD%2FJzJauIBI2jJrGuebzkqvXKqLVYUqFbzB8%2B3507SFA%2Bi8F13%2FMUmcucHfcSm1BJptJljdg5AMOBZA%2FfLXVNyBXEKCtFTxO0p2UwJkWHfLPZwHjrcbgPEEiYAn10XFDj%2B6 胡志偉促請警方盡快查明林子健事件，香港電台新聞網站，2017年8月12日。]</ref>。8月14 日，《傳真社》發放及公開旺角砵蘭街商戶的閉路電視片段，發現林子健的講法矛盾，及後，林子健於深夜因涉嫌誤導警務人員被拘捕。<ref>[https://www.factwire.org/single-post/2017/08/15/%E9%87%8D%E7%B5%84%E9%96%89%E8%B7%AF%E9%9B%BB%E8%A6%96%E7%9A%84%E8%AA%BF%E6%9F%A5-%E6%98%AF%E4%B8%80%E6%AD%A5%E4%B8%80%E6%AD%A5%E8%B5%B0%E5%87%BA%E4%BE%86 《重組閉路電視的調查 是一步一步走出來》，香港《傳真社》，2017年8月15日]</ref>在8月15日下午12:30分，民主黨再次召開記者招待會，唯只有胡志偉一人會見傳媒，[[林卓廷|林卓廷]]、[[何俊仁|何俊仁]]、[[李柱銘|李柱銘]]及[[李永達|李永達]]等人並無出現，胡指由於事件正在調查，故民主黨暫不宜作進一步評論，亦拒絕就事件道歉<ref>[http://hk.on.cc/hk/bkn/cnt/news/20170815/bkn-20170815111514077-0815_00822_001.html?nsukey=uO0UA1V2CRKwMR4FY3c8hjlUyugMEczD4CzlJMHjYIzoYO0B0WRZTGrpHcjGTMmwOXAxG%2BlsJARpq0rfuVdpWRwKW%2FBHECaQm55Tl6ICBnqPnX2KokJyScDoYPHJw%2FBPcGQ3Bd7SwJwGfJpOTnXxSB7no3DnbYoiyWnVK7vFx33VijPTidRgSUSBcqMDsHFV 林子健案：卸責李柱銘開記招 胡志偉拒道歉]，東方報業集團網站，2017年8月15日。</ref>。

==争议==
===大和解​論===
2017年4月17日胡志偉對記者表示，希望候任行政長官[[林鄭月娥|林鄭月娥]]運用《[[中華人民共和國香港特別行政區基本法|香港特別行政區基本法]]》第48條的權力，[[特赦|特赦]][[雨傘運動|雨傘運動]]中所有被檢控的人士，包括所有參與者以及[[七警|七警]]、[[朱經緯|朱經緯]]等；建議香港政府成立獨立調查委員會，就"雨傘運動"事件的本質作詳細分析及背景研究並深切檢討，避免往後施政時重複犯錯，通過“政治手段解決政治問題”，以修補社會撕裂。該言論在香港社會引發巨大反響，[[建制派|建制派]]和[[泛民主派|泛民主派]]人士均反駁了胡志偉的言論。 [[民主建港協進聯盟|民建聯]]主席[[李慧瓊|李慧瓊]]認為認為特赦會損害[[法治|法治]]精神；[[經民聯|經民聯]]主席[[盧偉國|盧偉國]]認為特赦的做法雖然可以舒緩社會撕裂氣氛，令社會和諧，但會違反香港秉持的法治精神；而[[田北辰|田北辰]]認為胡志偉的建議是“把玩政治”，會“教壞下一代”。泛民方面，部分[[民主黨_(香港)|民主黨]]黨員告訴記者胡在向傳媒提出建議前並未知會民主黨立法會黨團，批評其說法把立場當成貨幣交易，破壞民主運動和法治原則；民主黨中央委員[[許智峯|許智峯]]認為社會撕裂的源頭並不是佔中，而是民間對政府的不信任，大和解與法治理念背道而馳；[[香港眾志|香港眾志]]主席[[羅冠聰|羅冠聰]]認為該建議不實際，在政改之間便做出和解是不負責任，而特赦違法的警察可能會造成更大的撕裂<ref>{{cite news|title=港民主党主席倡议特赦论 修补社会撕裂 备受抨击|url=http://www.dw.com/zh/%E6%B8%AF%E6%B0%91%E4%B8%BB%E5%85%9A%E4%B8%BB%E5%B8%AD%E5%80%A1%E8%AE%AE%E7%89%B9%E8%B5%A6%E8%AE%BA-%E4%BF%AE%E8%A1%A5%E7%A4%BE%E4%BC%9A%E6%92%95%E8%A3%82-%E5%A4%87%E5%8F%97%E6%8A%A8%E5%87%BB/a-38478405?&zhongwen=simp|publisher=德国之声|date=2017-04-18}}</ref>。4月18日胡志偉表示正式收回有關言論，並對特赦論表示深切道歉，承認發表有關言論前未經黨內深思熟慮討論，亦未有與其他民主派政黨交換意見<ref>{{cite news|title=胡志伟撤回占中特赦论并深切致歉|url=http://www.zaobao.com.sg/realtime/china/story20170418-750361|publisher=联合早报|date=2017-04-18}}</ref><ref>[http://hk.on.cc/hk/bkn/cnt/news/20170419/bkn-20170419033014729-0419_00822_001.html?eventid=4028828d47d3fe7e01482b1c75912433&eventsection=hk_news 東方日報A1：全城怒轟胡志偉踐踏法治]，東方報業集團網站，2017-04-19</ref>。
