'''胡海峰'''（{{bd|1972年|11月||}}），中華人民共和國政治人物，祖籍[[安徽|安徽]][[绩溪|绩溪]]，出生于[[甘肃|甘肃]]。1995年5月加入中国共产党，1998年5月参加工作，研究生学历，北京交通大学本科毕业，清华大学EMBA硕士毕业。前[[中国共产党中央委员会总书记|中国共产党中央委员会总书记]][[胡锦涛|胡锦涛]]之子，[[胡海清|胡海清]]之兄。曾任[[嘉兴市|嘉兴市]]市长，现任中国浙江省[[丽水市|丽水市]]市委书记。

== 生平 ==
胡海峰本科就读于[[北方交通大学|北方交通大学]]（现[[北京交通大学|北京交通大学]]）[[计算机|计算机]]与科学技术系。[[清华大学|清华大学]]经济管理学院第一届[[EMBA|EMBA]]毕业生，曾经在[[中粮集团|中粮集团]]任职，也曾担任威视股份有限公司（Nuctech Co.）总经理。

== 争议 ==
{{Main|威视腐败案}}
2009年7月19日，英國和非洲媒體分別報導，胡海峰曾任[[清華同方|清華同方]]集團威視公司董事長，在非洲國家[[納米比亞|納米比亞]]為了取得機場港口掃瞄設備的合同涉嫌行賄，金額高達1320萬美元，有三人已被拘禁，目前正接受該國反貪機構調查。事件發生後，胡海峰的名字成為大陸的各網絡搜索引擎的敏感詞，不予显示相关结果<ref>[http://news.bbc.co.uk/chinese/simp/hi/newsid_8150000/newsid_8158700/8158759.stm BBC中文网：威视公司涉嫌行贿在纳米比亚遭调查]</ref>。

== 家庭 ==
*妻子：[[王珺|王珺]]（1973年－），胡海峰的大学同学，1995年毕业于北方交通大学。1998年获得北京大学经济学硕士。2008年获得清华大学经济学博士。2009年6月，出任由前总理朱镕基担任创院院长的清华大学经济管理学院院长助理。
*父亲：[[胡锦涛|胡锦涛]]（1942年12月21日－），中國共產黨第十六屆和十七屆中央委員會總書記。
*母亲：[[刘永清|刘永清]]（1940年10月3日－）。
*妹妹：[[胡海清|胡海清]]（1973年12月－）。
**妹夫：[[茅道临|茅道临]]（1963年1月10日－），曾任[[新浪网|新浪网]]CEO。
