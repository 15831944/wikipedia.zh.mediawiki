'''胡幼偉'''（{{bd|1961年||||}}），生於[[臺北市|臺北市]]，台灣的新聞傳播學者。[[國立政治大學|國立政治大學]]新聞學系學士、碩士，美國{{Link-en|南伊利諾州立大學|Southern Illinois University}}新聞學博士。曾任《[[民生報|民生報]]》[[記者|記者]]、[[國立臺灣師範大學|國立臺灣師範大學]]大眾傳播所所長、馬英九政府[[行政院發言人|行政院發言人]]、中國文化大學專任教授兼系主任。

是中華民國第一位因為於[[Facebook|Facebook]]發言不當及[[師生戀|師生戀]]而請辭下台的政治人物。

== 公職生涯 ==
2012年5月20日，擔任行政院發言人。10月，因師生戀醜聞，主動請辭。共擔任140天發言人。

在[[台灣進口美國牛肉問題|台灣進口美國牛肉問題]]中，立法院在野黨團（包括台聯黨、親民黨及民進黨）要求等待[[國際食品法典委員會|國際食品法典委員會]]決議，國民黨反對，雙方形成僵局<ref>{{cite news|url=http://news.chinatimes.com/politics/130502/132012062801061.html|title=美牛案 橘籲與Codex接軌|3=|publisher=中央社|date=2012-06-28}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2012年7月2日，胡幼偉表示，含萊克多巴胺肉品在安全容許量內不危害健康，之所以連年討論是國際貿易政治角力的延伸，並非健康問題，[[聯合國|聯合國]][[國際食品法典委員會|國際食品法典委員會]]制定的標準並不科學，希望在野黨不要以此阻擋美國牛肉進口<ref>{{cite news|url=http://www.cna.com.tw/News/FirstNews/201207020057.aspx|title=政院：Codex論戰政治角力|publisher=中央社|date=2012-07-02}}</ref>。7月5日，[[國際食品法典委員會|國際食品法典委員會]]通過萊克多巴胺允許殘留量，行政院多方考量下決定接受此一標準。民進黨也同意採取國際標準<ref>{{cite news|url=http://www.nownews.com/2012/07/05/91-2831507.htm|title=Codex決議訂瘦肉精標準　民進黨同意採取國際標準|publisher=NOWnews|date=2012-07-05}}</ref>，台聯立委以行政院發言人曾認為此標準不科學為由，繼續反對美國牛肉進口。

9月29日，在[[臉書|臉書]]貼上自己購買新西裝以及手持台灣尚未核准進口的[[iPhone_5|iPhone 5]]照片，因為行政院先前曾鼓吹台灣民眾應購買[[宏達電|HTC手機]]，此留言被民進黨中常委[[洪坤智|洪坤智]]在臉書轉載後披露<ref>{{Cite web |url=http://udn.com/NEWS/BREAKINGNEWS/BREAKINGNEWS1/7398730.shtml|language=zh-tw |title= 臉書貼手機掀波 胡幼偉謝關心|publisher=聯合報 ||date=2012-09-30 |accessdate=2012-09-30}}</ref>。胡幼偉辯解，iphone5是朋友所有，他只是借來拍照，只是一種實驗，用來測試[[白色恐怖|白色恐怖]]氣氛<ref>{{Cite web |url=http://www.libertytimes.com.tw/2012/new/oct/1/today-p2.htm | author=邱燕玲、劉榮|title= 胡幼偉臉書秀iPhone5挨轟 還扯白色恐怖|publisher=自由時報 ||date=2012-09-30 |accessdate=2012-09-30}}</ref>，引起民進黨立委不滿，認為胡幼偉的發言對於台灣歷史缺少了解，成為新聞話題<ref>{{Cite web |url=http://udn.com/NEWS/NATIONAL/NAT1/7398569.shtml|language=zh-tw | author=蘇芳禾|title= 胡幼偉「實驗」白色恐怖？網友嗆挑釁|publisher=聯合報 ||date=2012-09-30 |accessdate=2012-09-30}}</ref>。10月2日，胡幼偉在臉書再度發言，認為自己的目的在於吸引大眾注意力，以推銷行政院經濟改革方案，是一種[[草船借箭|草船借箭]]。陳沖面對媒體詢問，只表示，「憑良心講應該是我講話，他幫我解釋，不是他講話我來幫他解釋」<ref>{{Cite web |url=http://udn.com/NEWS/NATIONAL/NAT3/7402201.shtml|language=zh-tw | author=楊湘鈞|title= 胡幼偉：我沒瘋…是草船借箭|publisher=聯合報 ||date=2012-10-02 |accessdate=2012-10-02}}</ref>。立法院朝野立委皆表示不滿，要求胡幼偉公開道歉<ref>{{Cite web|url=http://news.chinatimes.com/politics/110101/112012100200119.htmll|language=zh-tw|author=李明賢、楊毅|title=胡幼偉「草船借箭」 藍綠都開罵|publisher=中國時報|6=|date=2012-10-02|accessdate=2012-10-02}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，10月3日，胡幼偉在立法院公開道歉，以平息風波<ref>{{Cite web |url=http://www.libertytimes.com.tw/2012/new/oct/3/today-p9.htm|language=zh-tw | author=施曉光、陳璟民|title= 臉書扯白色恐怖、草船借箭 胡幼偉道歉|publisher=自由時報 ||date=2012-10-03 |accessdate=2012-10-03}}</ref>。

2012年10月6日，在[[臉書|臉書]]公布自己與曾經指導過的師範大學研究生張蓉君正在戀愛，引起新聞報導。胡幼偉自2002年離婚以後，一直保持單身，2004年底公開與[[民視|民視]][[陳淑貞|陳淑貞]]戀愛，兩人沒有結婚，於2008年分手，但仍保持友好關係<ref>{{Cite web |url=http://www.libertytimes.com.tw/2012/new/oct/7/today-fo2-3.htm|author=黃湞尹|language=zh-tw|title= 胡離婚後 曾和民視主播陳淑貞交往|publisher=自由時報 |date=2012-10-07 |accessdate=2012-10-07}}</ref>。在2011年開始，曾任張蓉君論文指導教授，直到2012年5月，因兩人開始戀愛，同時胡幼偉擔任行政院發言人，論文指導教授轉由大傳所所長胡綺珍擔任<ref>{{Cite web |url=http://www.appledaily.com.tw/appledaily/article/headline/20121007/34558325/|language=zh-tw | |title= 愛上24歲嫩女 自掀師生戀 胡幼偉辭丟官|publisher=蘋果日報 ||date=2012-10-07 |accessdate=2012-10-07}}</ref>。在請辭被接受後，胡幼偉向記者聲稱自己接獲黑函，裏面提及個人隱私，並揚言向外揭露，胡因懷疑自己可能遭監聽或是裝設針孔，擔心可能影響行政院團隊而主動請辭<ref>{{Cite web |url=http://www.ettoday.net/news/20121007/111585.htm|language=zh-tw|title= 為「愛的助理」張蓉君下台　胡幼偉曾與主播陳淑貞交往|publisher=ETtoday|date=2012-10-07 |accessdate=2012-10-07}}</ref>。

10月16日，行政院在youtube網站推出委託民視文化公司花費450萬<ref>{{cite news|url=http://www.ettoday.net/news/20121018/115926.htm|title=政院腦殘廣告　網友推出簡潔加強版更有力|publisher=ETtoday|date=2012-10-18}}</ref>製作的「經濟動能推升方案」整體篇宣傳廣告，影片內容長達41秒，重點在於宣傳標語「拼經濟，做就對了」。<ref>{{Cite web|url=http://www.cepd.gov.tw/m1.aspx?sNo=0017594|language=zh-tw|title=經濟動能推升方案|publisher=行政院經濟建設委員會|deadurl=yes|archiveurl=https://web.archive.org/web/20121019164223/http://www.cepd.gov.tw/m1.aspx?sNo=0017594|archivedate=2012-10-19}}</ref>但因廣告內容空洞，沒有實際講出任何內容，招致惡評，網友向youtube網站檢舉為詐騙影片，行政院帳號暫時遭到停權。經濟部長[[施顏祥|施顏祥]]表示廣告做得不好，行政院發言人[[黃敏恭|黃敏恭]]表示，這個影片是胡幼偉策劃<ref>{{Cite web |url=http://udn.com/NEWS/NATIONAL/NAT1/7435832.shtml|author=蔡佩芳、甘芝萁|language=zh-tw|title= 宣傳廣告挨轟 政院：胡幼偉想的|publisher=聯合報|date=2012-10-17 |accessdate=2012-10-17}}</ref>。胡幼偉表示，這個廣告是委託廣告人[[黃文博_(廣告人)|黃文博]]設計規劃，並不是他想的。

==政治評論==
{{see also|九月政爭}}
2013年9月15日，胡在[[臉書|臉書]]（facebook）發文指出，「在法治社會中，法治精神和法官才是真正的大老！」他認為，「只有法官說那確實是關說，我們才能跟著說。」 <ref>[http://www.nownews.com/2013/09/15/301-2986360.htm  看問題／是不是關說？　胡幼偉解讀馬英九王金平認知框架 ]，NOWnews 今日新聞網，王鼎鈞，2013年9月15日</ref>
==爭議==
2014年4月7日，胡在臉書上稱挺學運學生將會找不到工作（有數個老闆向他說不願任用反體制者，但並未說明所謂「數個老闆」是誰），引起網友譏笑胡因為[[師生戀|師生戀]]事件成為史上最短命的行政院發言人。然依據今日新聞報導，包括TOYOTA台灣代理和泰汽車、中華電信及家樂福等企業，都表示不會受此影響；而北市商業會理事長王應傑更說，沒聽過圈內有如此傳言，尤其林飛帆和陳為廷若真來求職，他們的統籌領導能力傑出，我還會考慮優先錄用。
