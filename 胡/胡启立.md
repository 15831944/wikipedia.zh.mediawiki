'''胡启立'''（{{bd|1929年|10月||catIdx=H胡}}），[[陕西|陕西]][[榆林|榆林]]人。他是[[中國共產黨|中國共產黨]]前任主要[[党和国家领导人|領導人]]之一，曾经于1987年至1989年出任[[中共中央政治局常委|中共中央政治局常委]]，在当时五位政治局常委中，排名第四。同时担任[[中共中央书记处|中共中央书记处]]排名第一的[[书记|书记]]，负责常务工作。在[[六四事件|六四事件]]前后，因为同情[[学生运动|学生运动]]、支持[[中共中央总书记|中共中央总书记]][[趙紫陽|趙紫陽]]，被免去党内领导职务，仅保留中央委员会委员资格。1991年复出工作，1998年3月，当选第九届[[全国政协副主席|全国政协副主席]]，名义上恢复“[[党和国家领导人|党和国家领导人]]”地位。

== 早年經歷==
1946年至1951年，胡启立在[[北京大学|北京大学]][[物理|物理]]系、[[机械|机械]]系學習。大学毕业后，留校从事学生、共青团委工作，1951年至1956年任[[共青團|共青團]][[北京大学|北京大学]]委员会书记，受到时任[[共青团中央|共青团中央]]负责人[[胡耀邦|胡耀邦]]的赏识。1955年8月，胡启立在当选[[中华全国学生联合会|中华全国学生联合会]]主席后，受到[[毛泽东|毛泽东]]的接见。1956年被调入[[共青团中央|共青团中央]]联络部，曾经担任驻[[捷克|捷克]][[布拉格|布拉格]]国际学生联合会书记处书记、副主席。1961年，在[[胡耀邦|胡耀邦]]的推荐下，共青团九届三次会议破格增选胡启立为团中央书记处候补书记<ref>[http://news.xinhuanet.com/book/2005-12/07/content_3888040.htm 胡启立回忆胡耀邦：他对我影响最大最深] {{webarchive|url=https://web.archive.org/web/20090221191426/http://news.xinhuanet.com/book/2005-12/07/content_3888040.htm |date=2009-02-21 }} 原载《中国青年报》2005年12月7日</ref> ，为其日后的政治生涯奠定了坚实的基础。

[[文化大革命|文化大革命]]开始后，[[毛泽东|毛泽东]]在1966年8月警告[[共青团中央|共青团中央]]，称[[胡耀邦|胡耀邦]]、[[胡克实|胡克实]]、胡启立等负责人犯了“错误”。胡启立随即被下放到“[[五七干校|五七干校]]”劳动，直到1972年才被恢复工作<ref>[http://www.chinanews.com.cn/gj/kong/news/2008/08-13/1346687.shtml 聚焦共青团中央第一书记] 原载《凤凰周刊》 作者，倪方六</ref>。 此后，他历任中共[[宁夏回族自治区|宁夏回族自治区]][[西吉|西吉]]县委副书记，[[固原|固原]]地委副书记，[[中共宁夏回族自治区委|中共宁夏回族自治区委]]办公厅主任等职。1977年，返回[[北京|北京]]，出任[[清华大学|清华大学]][[党委|党委]]副书记、副校长。1978年，出任团中央书记处书记、[[中华全国青年联合会|中华全国青年联合会]]主席。

== 進入中央 ==
1980年，随着[[胡耀邦|胡耀邦]]接任[[中共中央总书记|中共中央总书记]]，胡启立迅速成为当时中国政坛上的一颗新星。於1980年6月出任[[中共天津市委|中共天津市委]]书记、[[天津市|天津市]][[市长|市长]]。1982年4月，升任[[中央办公厅|中央办公厅]]主任；同时，在当年9月召开的[[中共十二大|中共十二大]]上，当选中央委员；后又当选[[中央书记处|中央书记处]]书记，正式进入中共领导层。在1985年9月召开的中共十二届五中全会上，中共高层人事大调整，一批元老退休，胡启立与[[田纪云|田纪云]]、[[乔石|乔石]]、[[李鹏|李鹏]]等“新秀”被增补为[[中央政治局|中央政治局]]委员。<ref>[http://news.xinhuanet.com/ziliao/2004-10/25/content_2136727_4.htm 第十二届中央委员会（1982年9月～1987年11月）] {{webarchive|url=https://web.archive.org/web/20100808111433/http://news.xinhuanet.com/ziliao/2004-10/25/content_2136727_4.htm |date=2010-08-08 }}</ref>

1987年，胡耀邦因反“[[資產階級自由化|資產階級自由化]]”不力下台，在赵紫阳的保护下，胡啟立並沒有受到影響。在1987年11月2日举行的[[中国共产党十三届一中全会|中国共产党十三届一中全会]]上，晋升为[[中共中央政治局常委|中央政治局常委]]；并在五位常委中，排名第四；同时还兼任[[中央书记处|中央书记处]]排名第一的书记，负责书记处的常务工作。媒体评论认为，当时年富力强的胡启立有力成为中共的第三代接班人。<ref>[http://www.zaobao.com/special/china/congress16/pages/congress16151102g.html 中共新的权力三角] 原载《联合早报》</ref> 

1989年[[六四事件|六四事件]]前后，因同情學生運動，胡启立與[[閻明復|閻明復]]、[[芮杏文|芮杏文]]等中共高级领导，随总书记[[趙紫陽|趙紫陽]]下台。胡启立被免去政治局委员、常委，中央書記處書記的職務，但仍保留中央委員會委員资格。

== 复出 ==
1991年，胡启立复出工作，被安排到[[中华人民共和国机械电子工业部|原机械电子工业部]]，担任排名最後的副部长。1993年3月，出任恢复设立的[[电子工业部|电子工业部]]部长。在任部长期间，胡启立倡导对[[中国电信业|中国电信业]]进行改革，积极推动[[中国联通|中国联通]]的成立，引入竞争，最终促使[[国家邮政局|邮政]]和[[中华人民共和国信息产业部|电信]]于1998年分离。<ref>[http://www.ccidcom.com/html/xinwenku/200909/28-77512.html 通信产业网]</ref>

1998年3月举行的[[中国人民政治协商会议|中国人民政治协商会议]]第九届全国委员会第一次会议上，胡启立当选为[[全国政协副主席|全国政协副主席]]，再次进入[[国家级副职|国家级副职]][[党和國家領導人|党和國家領導人]]行列，2003年3月届满离任。自2001年起，胡启立还一直担任[[中国福利会|中国福利会]]主席和[[中国宋庆龄基金会|中国宋庆龄基金会]]主席等职务。2016年11月，不再担任中国宋庆龄基金会主席<ref>{{cite web|url=http://www.thepaper.cn/newsDetail_forward_1555383|title=王家瑞当选中国宋庆龄基金会主席，马晓伟等当选副主席|publisher=澎湃新闻网|accessdate=2016-11-04}}</ref>。2017年9月，不再担任中国福利会主席<ref>{{cite web|url=http://www.cwi.org.cn/zh/content.aspx?id=9973|title=中国福利会第七届执行委员会第一次会议召开|publisher=中国福利会|accessdate=2017-09-30}}</ref>。
