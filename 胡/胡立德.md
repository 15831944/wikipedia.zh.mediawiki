'''胡立德'''（{{lang-en|'''David L. Hu'''}}；{{bd|1979年||||catIdx=H胡,立德David}}）是[[美國|美國]][[喬治亞理工學院|喬治亞理工學院]]的一位[[流體力學|流體力學]]及[[生物物理學|生物物理學]]專家，他現時是机械工程与生物学系的副教授，利用[[高速照相機|高速照相機]]，透過研究「从动物身上寻找有助于制造机器人的潜质」<ref name="nfpeople">{{cite journal |url=http://www.nfpeople.com/story_view.php?id=6520 |title=科学顽童胡立德 |publisher=[[南方人物周刊|南方人物周刊]] |author=李珊珊 |date=2015-06-05 |accessdate=2015-12-09 |language=zh-hans |journal=— |archiveurl=https://web.archive.org/web/20150914210520/http://www.nfpeople.com/story_view.php?id=6520 |archivedate=2015-09-14 |deadurl=yes }}</ref>。當中部份研究是利用其流體力學的專業去解釋有關流體（例如：雨點及排尿）對動物的影響<ref name="pnas_search">{{cite web |url=http://www.pnas.org/search?author1=David+L.+Hu&sortspec=date&submit=Submit |title=胡立德在PNAS的出版 |language=en |accessdate=2015-12-09 }}</ref>，令他獲得[[菠萝科学奖|菠萝科学奖]]及[[搞笑諾貝爾獎|搞笑諾貝爾獎]]。在2003年8月和2005年9月分別以「昆蟲在水上行走」及「昆蟲如何爬上岸」為題<ref>{{cite web |url=http://www.epochtimes.com/b5/5/10/9/n1080166.htm |title=26歲華裔胡立德 學術界明日之星 |publisher=[[大紀元時報|大紀元時報]] |author=[[韋樞|韋樞]] |date=2005-10-09 |accessdate=2015-12-09 |language=zh }}</ref>，兩度登上《[[自然_(期刊)|自然雜誌]]》的封面<ref name="nfpeople" />。

他最為人所知的研究，是他與其他三位同事透過比對動物園內所有體重在3公斤以上的哺乳動物的排尿時間的[[對數|對數]]值及動物的體重，發現兩者有[[線性關係|線性關係]]。研究結果表明：幾乎都可以在21±13秒之內完成排尿，關鍵在於牠們的尿道的構造，令排尿的效率提高了3,600倍<ref>{{cite web |url=http://www.guokr.com/article/438685/ |title=严肃点，这是尿尿的流体力学 |author=[http://www.guokr.com/i/0412344783/ Paradoxian] |date=2014-06-27 |accessdate=2015-12-09 |language=zh-hans }}</ref><ref>{{cite journal |first1=Patricia J. |last1=Yang |first2=Jonathan |last2=Pham |first3=Jerome |last3=Choo |first4=David L. |last4=Hu |authorlink4=胡立德|volume=111 |issue=33 |title=Duration of urination does not change with body size |year=2014 |journal=[[PNAS|PNAS]] |doi=10.1073/pnas.1402289111 |language=en }}</ref>。這項研究令他們四人獲得[[搞笑諾貝爾獎得獎者#2015年|2015年搞笑諾貝爾獎物理學獎得獎者]]。

由於胡立德熱愛運動，加上本科生時曾任大學男子體操隊成員，他在出席菠萝科学奖時還在台上打側手翻<ref name="nfpeople" />。
