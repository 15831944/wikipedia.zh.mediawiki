'''胡冰卿'''（{{bd|1992年|1月25日|catIdx=H胡}}），中国女演员，出生于[[安徽|安徽]][[合肥|合肥]]，毕业于[[复旦大学上海视觉艺术学院|复旦大学上海视觉艺术学院]]表演系。2012年以微电影《美》进入演艺圈<ref>{{Cite web |url=http://ent.gd.sina.com.cn/news/20120813/734235.html |title= 李静跨界拍微电影 女主酷似小董洁(图)|date=2012-08-13 |publisher= 新浪娛樂 |language= zh-hans }} </ref>；2014年在古裝劇《[[秦時明月_(電視劇)|秦時明月]]》中饰演“高月”一角逐漸引起關注；2015年在《[[旋风少女|旋风少女]]》中饰演女主角“戚百草”而走紅；2018年主演的《[[獨孤天下|獨孤天下]]》及《[[櫃中美人|櫃中美人]]》皆引起熱烈討論。

== 个人背景 ==
小时候喜欢艺术类，如唱歌、跳舞，但家里让她以文化课为重。到了高中，偶然一次机会听说艺考。胡冰卿的一个老师让她可以试一试，后在2010年被复旦大学上海视觉艺术学院表演系录取。上了大学后，胡冰卿才开始懂得表演是什么，在大二时发现可以去塑造不同性格、不一样的人物，从而喜欢上了表演 。大学期间，胡冰卿在2011年9月、2012年9月分别获得上海视觉艺术学院“校内综合奖学金二等奖”和“校内综合奖学金三等奖”

== 经历 ==
*2012年，胡冰卿主演李静投拍的以倡导国民新女性为主题的微电影《美》，扮演了一个从小没有父亲、由哑巴母亲含辛茹苦养大的女孩 。同年，在国内首部消防题材青春励志剧《火线英雄》饰演了消防新兵姜二宝的恋人姜小妹。
*2013年，胡冰卿参与拍摄由[[杨幂|杨幂]]首次担任制片人的[[微时代|微时代]]，饰演系主任女儿郭晶。同年，腾讯视频自制剧《腾空的日子》寻找演员，同校同学鼓励胡冰卿试镜，被公司选中出演女主角张珊，这也是她第一次尝试喜剧 。
*2014年，在由新丽与唐人联合出品的《[[女医·明妃传|女医·明妃传]]》电视剧中，胡冰卿饰演钱皇后身边贴身女官如香，虽然戏份不多，但这是胡冰卿第一次出演古装剧 。之后，[[唐人影视有限公司|唐人影视]]、杭州玄机科技等公司将国内第一部3D武侠动画[[秦时明月|秦时明月]]搬上荧屏，并由胡冰卿与[[蒋劲夫|蒋劲夫]]合作，在剧中饰演墨家巨子之女高月 ；经过169天的拍摄，该剧于2015年1月16日杀青。
*2015年，与[[杨洋|杨洋]]、[[白敬亭|白敬亭]]、[[陈翔|陈翔]]合作主演根据[[明晓溪|明晓溪]]同名小说改编的青春偶像励志剧《[[旋风少女|旋风少女]]》，饰演女主角百草，讲述元武道女孩戚百草的成长故事。<ref>{{Cite web |url=http://ent.sina.com.cn/v/m/2015-03-24/doc-iawzuney1032261.shtml |title= 《旋风少女》曝阵容 胡冰卿搭杨洋白敬亭)）|date=2015-03-24 |publisher= 新浪娛樂 |language= zh-hans }} </ref>之后，与[[归亚蕾|归亚蕾]]、[[秦汉|秦汉]]、[[韩东君|韩东君]]等主演由[[唐人影视有限公司|唐人影视]]出品根据电影版改编的《[[重返20岁_(电视剧)|重返20岁]]》，饰演重返20岁的“奶奶”孟丽君。<ref>{{Cite web |url=http://ent.sina.com.cn/v/m/2015-11-17/doc-ifxkszhk0336099.shtml|title= 剧版《重返20岁》揭秘：奶奶爷爷一起穿越)）|date=2015-11-17 |publisher= 新浪娛樂 |language= zh-hans }} </ref>
*2016年，与[[萧敬腾|萧敬腾]]出演由日本同名漫画改编的《[[深夜食堂_(亚洲华语版)|深夜食堂 (亚洲华语版)]]》，饰演西西。<ref>{{Cite web |url=http://ent.sina.com.cn/v/m/2016-04-14/doc-ifxriqqx2446575.shtml|title= 《深夜食堂》曝定妆照 萧敬腾胡冰卿组cp)）|date=2016-04-14 |publisher= 新浪娛樂 |language= zh-hans }} </ref>之后，与[[周渝民|周渝民]]、[[陈瑶|陈瑶]]等主演由[[唐人影视有限公司|唐人影视]]出品根据小说《胭脂醉》改编的《[[柜中美人|柜中美人]]》，饰演黄鼬精黄轻风。<ref>{{Cite web |url=http://ent.sina.com.cn/v/m/2016-05-24/doc-ifxsktkp9246015.shtml|title= 《柜中美人》初探班 胡冰卿陈瑶床咚互撩)）|date=2016-05-24 |publisher= 新浪娛樂 |language= zh-hans }} </ref>
