'''胡建淼'''（1957年-），[[浙江|浙江]][[慈溪|慈溪]]人，中国当代[[行政法|行政法]]学家、教授，[[国家行政学院|国家行政学院]]法学部主任<ref>{{cite web |url=http://www.zhuzhou.gov.cn/zhuantizhuanji/fcld/zjfc/102500.htm |language= |title=胡建淼 |publisher=株洲政府门户网站 |date=2012-07-10 |accessdate= }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。他终生享受国务院政府特殊津贴，任[[中国法学会|中国法学会]]行政法研究会副会长、[[教育部|教育部]]法学专业教学指导委员会委员、[[浙江省|浙江省]]法学会副会长、[[杭州市|杭州市]]人大常委会立法咨询委员会主任、杭州市仲裁委员会主任。<ref name= "个人简介"/>

== 生平 ==
1982年2月毕业于[[杭州大学|杭州大学]]（今[[浙江大学|浙江大学]]）[[哲学|哲学]]系，1989年毕业于[[中国政法大学|中国政法大学]]研究生院。

1990年起，任杭州大学法律系副主任；1996年起任杭州大学法学院院长；1998年9月-2007年10月任浙江大学副校长；2001年起担任浙江大学[[宪法|宪法]]与[[行政法|行政法]]专业[[博士生导师|博士生导师]]；2007年10月任[[浙江工商大学|浙江工商大学]]校长，2010年12月离任<ref>{{cite web |url=http://www.zjgsu.edu.cn/News/20111/News_3880.Html |language= |title=胡建淼校长调离浙江工商大学 |publisher=浙江工商大学 |date=2010-12-31 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20110715213228/http://www.zjgsu.edu.cn/News/20111/News_3880.Html |archivedate=2011-07-15 }}</ref>。

[[1995|1995]]至1999年曾多次赴[[英国|英国]]、[[澳大利亚|澳大利亚]]、[[日本|日本]]、[[韩国|韩国]]、[[香港|香港]]、[[台湾|台湾]]等地进行合作研究与学术访问<ref name="个人简介">{{cite web|url=http://www.law-lib.com/fxj/fxj.asp?id=450 | language= | title=胡建淼个人简介 |publisher=法律图书馆 |date= |accessdate=2012年11月 }}</ref>。

== 著作 ==
个人著作：《行政法学》，法律出版社1998年1月；《中国行政法学论纲》，杭州大学出版社1998年1月；《比较行政法》，法律出版社1998年7月；

主编著作：《民政行政法》，中国人事出版社1993年10月；《国家赔偿法教程》，浙江人民出版社1994年10月；《行政法教程》，法律出版社1996年10月；《外国行政法规与案例评述》，中国法制出版社1997年6月；《行政法与行政诉讼法》，高等教育出版社1999年6月；《宪法学十论》，法律出版社1999年9月；

合著：《行政法学原理》，劳动人事出版社1989年9月。<ref name= "个人简介"/>
