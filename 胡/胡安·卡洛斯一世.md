'''胡安·卡洛斯一世'''（{{lang-es|'''Juan Carlos I'''}}，全名：'''胡安·卡洛斯·阿方索·维克托·玛利亚·德波旁—波旁-两西西里'''，{{lang|es|'''Juan Carlos Alfonso Víctor María de Borbón y Borbón-Dos Sicilias'''}}，{{bd|1938年|1月5日|catIdx=C}}），是[[西班牙|西班牙]]前任[[西班牙國王|國王]]，也是[[佛朗哥時期|佛朗哥時代]]結束後的第一任國王，於1975年11月27日登基。

他在位期間，導引西班牙民主化，推動君主立憲限縮自身權力，並粉碎1981年的未遂軍事政變，深受西班牙人愛戴。由於受到王室醜聞及長期的健康問題困擾，他於2014年6月2日宣布退位<ref>[http://www.cna.com.tw/news/firstnews/201406020199-1.aspx 西班牙國王宣布退位]中央社，2014年6月2日</ref>，19日正式退位，結束39年的統治。<ref>[http://www.cna.com.tw/news/aopl/201406190366-1.aspx 西班牙新王登基 儀式從簡【影片】]中央社，2014年6月19日</ref>其子[[费利佩六世|费利佩六世]]继位。

== 家世背景 ==
其祖父為西班牙国王[[阿方索十三世|阿方索十三世]]。其父為[[胡安·德·波旁|胡安亲王]]，原為西班牙王室繼承人，因[[弗朗西斯科·佛朗哥|弗朗西斯科·佛朗哥]]將軍1947年恢復王室後，擔憂其父過于偏向自由主義，因而自任攝政，繼任進行軍事獨裁統治，因此無法繼位。

其母為[[两西西里王国|两西西里王国]]的[[玛利亚·德·拉·玛塞迪斯·德·波旁-两西西里|玛利亚]]公主。她是西班牙国王[[卡洛斯三世_(西班牙)|卡洛斯三世]]的第六代子孙，也就是[[胡安·德·波旁|胡安亲王]]的远房堂姐。

胡安·卡洛斯一世為家中的嫡長子。

== 早年生活 ==
胡安·卡洛斯1938年生于意大利[[罗马|罗马]]，当时西班牙王室因躲避1931年成立的[[西班牙第二共和国|西班牙第二共和国]]而迁居到意大利。1948年他回到西班牙接受教育，1954年毕业于马德里圣伊西德罗学校。然后-{zh-cn:参军; zh-tw:入伍}-，并于1955年至1957年在萨拉戈萨军事学院深造。1957年他又在马林海军学校，圣哈维耶尔空军学校学习。1961年他毕业于[[马德里康普顿斯大学|马德里康普顿斯大学]]。此后就生活在萨拉戈萨宫。

== 西班牙王子（1969-1975） ==
[[File:Juan_Carlos_de_Borbón,_Prince_of_Spain.jpg|thumb]]
[[第二次世界大戰|第二次世界大戰]]後，西班牙國內反對[[長槍黨|長槍黨]]及[[佛朗哥|佛朗哥]]將軍長期維持的獨裁統治的聲音一直未有間斷，部分支持君主制的人把希望寄予[[自由主義|自由主義]]者、原王位繼承人胡安亲王推上王位，但一直為佛朗哥拒絕。后来1947年佛朗哥宣佈恢復[[王室|王室]]，由於擔心胡安亲王會把他過往所推行的改革推翻，所以他一直反對胡安繼位及擔任國王（因此君主一直懸空），並於1969年親自指定了胡安亲王的長子胡安·卡洛斯为自己的继承人，獲「西班牙王子」的稱號。

== 恢復君主立憲制 ==
1975年佛朗哥逝世，胡安亲王在左派反对下，为了国家的稳定，宣布放弃法定繼承人的地位，胡安·卡洛斯顺利登基成为[[国王|国王]]，胡安·卡洛斯没有选择胡安三世或者卡洛斯五世的尊号，而选择胡安·卡洛斯一世为[[尊号|尊号]]。

由于[[巴塞罗那伯爵|巴塞罗那伯爵]]是西班牙國王的其中一个附属头衔，胡安亲王在之前的流亡生涯中为了表示享有西班牙王位的继承权便自称巴塞罗那伯爵。胡安·卡洛斯一世登基后正式封父亲为巴塞罗那伯爵，并在1993年父亲逝世后追尊他为'''胡安三世'''。

胡安·卡洛斯一世在位期间实行了[[西班牙民主转型|民主改革]]，1976年，他任命[[阿道弗·苏亚雷斯|阿道弗·苏亚雷斯]]为[[西班牙首相|首相]]，並於1977年舉行了西班牙戰後全國四十一年第一次真正普選。他注重外交，曾于1978年访问[[中華人民共和国|中華人民共和国]]，成為西班牙1973年與中华人民共和国建交後，第一位訪問中國的西班牙元首。虽然他一直关心国内局势，但依然放手由首相全權管理。

== 未遂政變 ==
1981年，部分右翼军方人士发动{{le|23-F|23-F|政变}}，试图恢复专制统治，国王在政变領導人来访时借故避开，通过电视台宣布自己不能容忍民主的倒退，政变想成功除非把他杀了。由于国王在军队中有很多好朋友，在他们的支持下，政变被迅速平息，参与政变的军方人员也大多得到特赦。自[[西班牙内战|西班牙内战]]后最有可能导致国家分裂和流血的此次事件，就这样成功瓦解。结束后，西班牙各政党的领袖手挽手走在[[马德里|马德里]]游行队伍的首列，当时[[西班牙共产党|西班牙共产党]]领袖卡利约还喊出了这样的口号：“上帝拯救了西班牙！”

由於國王成功應對，使他的支持率一直保持著非常高的水準。次年，左派的[[西班牙工人社會黨|西班牙工人社會黨]]終於取得執政權，40歲的[[费利佩·冈萨雷斯|费利佩·冈萨雷斯]]成為新首相，建立46年首個左翼政府。西班牙其後加入[[北大西洋公約組織|北大西洋公約組織]]。

== 國內政治影響力 ==
[[File:Juan_Carlos_2004.jpg|thumb]]
1990年代末期，[[巴斯克自治區|巴斯克]]恐怖分子策劃一次死傷很嚴重的爆炸案，胡安·卡洛斯一世国王罕有地在電視上發表宣言，指出此类流血事件“應該夠了”。他這一次亮相再次得到了国民的尊重和爱戴，也逼使國內各個分離組織重回談判桌上。

2007年第十七屆[[智利|智利]][[聖地亞哥|聖地亞哥]]伊比利亚－美洲領袖高峰會議中，[[委內瑞拉|委內瑞拉]][[委內瑞拉總統列表|總統]][[烏戈·查維茲|烏戈·查維茲]]在爭論中，多番打斷[[西班牙首相|西班牙首相]][[何塞·路易斯·羅德里格斯·薩帕特羅|薩帕特羅]]發言。胡安·卡洛斯一世為制止[[烏戈·查維茲|烏戈·查維茲]]，當場斥責「[[為什麼你不閉嘴？|為什麼你不閉嘴？]]」（{{lang|es|¿Por qué no te callas?}}），形成新聞焦點。

2011年，由於王室成員[[伊納基·烏丹加林|伊納基·烏丹加林]]被指控挪用公款作為私人用途，西班牙國民對於西班牙王室的質疑聲浪高漲，西班牙國內出現廢除君主制的呼聲。胡安·卡洛斯一世首次公開西班牙王室每年8.3億歐元預算的支出明細，以平息民怨。

== 退位 ==
2014年6月2日，西班牙首相[[马里亚诺·拉霍伊|拉霍伊]]宣布胡安·卡洛斯一世即将退位<ref>[http://www.cna.com.tw/news/firstnews/201406020199-1.aspx 西班牙國王宣布退位]中央社，2014年6月2日</ref>，并传位给[[費利佩_(阿斯圖里亞斯親王)|費利佩王子]]，卡洛斯在電視講話中宣布退位，結束39年的統治<ref>{{cite news | url=http://rt.com/news/163036-spain-king-abdication-pm/ | title=Spain's King Juan Carlos to abdicate, Crown Prince Felipe to succeed | publisher=RT | accessdate=2014-06-02}}</ref>。

== 家庭狀況和私人生活 ==
* 1962年，與[[希腊君主列表|希臘国王]][[保罗一世_(希腊)|保罗一世]]的长女[[索菲婭_(西班牙王后)|索菲婭公主]]結婚，二人是遠親（维多利亚女王的后裔）。
* 1963年12月10日，長女[[盧戈女公爵|盧戈女公爵]][[艾蓮娜_(西班牙公主)|艾蓮娜公主]]出生。
* 1965年6月13日，次女[[帕爾馬女公爵|帕爾馬女公爵]][[克里斯蒂娜_(西班牙公主)|克里斯蒂娜公主]]出生。
* 1968年1月30日，長子[[費利佩_(阿斯圖里亞斯親王)|費利佩王子]]出生，2014年6月繼位，是為費利佩六世。

== 健康狀況 ==
2010年5月8日，国王在巴塞罗那医院接受开刀手術，从右肺中取出一个17-19mm良性肿瘤。
