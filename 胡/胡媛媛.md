'''胡媛媛'''（ '''Vivienne Hu'''，'''薇薇安胡'''）， 出生于中国，求学于德国并在纽约开创了个同名品牌的华裔时装设计师。

==生平==
胡媛媛出生于并成长于[[中国|中国]][[安徽|安徽]]的一个从商的家庭。她曾在德国生活六年， 并于2000年从[[德国霍恩海姆大学|德国霍恩海姆大学]]和[[歌德学院|歌德学院]]获得[[金融|金融]]学位<ref>{{Cite web|title = FLASHforwardFASHION|url = http://issuu.com/thegreenwichgirl/docs/flashforward_fashion|website = Issuu|accessdate = 2015-12-23}}</ref>。毕业后，她移居美国攻读商业和经济硕士学位<ref name=":5">{{Cite web|title = Sitting down with Vivienne Hu|url = http://thatgirldenise.com/post/74400723645/sitting-down-with-vivienne-hu|website = that girl denise|accessdate = 2015-12-23|deadurl = yes|archiveurl = https://web.archive.org/web/20160305075528/http://thatgirldenise.com/post/74400723645/sitting-down-with-vivienne-hu|archivedate = 2016-03-05}}</ref>。她于2010年考入纽约[[帕森设计学院|帕森设计学院]]并学习时装设计。

==职业生涯==
在投身于时尚圈之前，胡媛媛曾在纽约[[投资银行|投资银行]]领域工作了三年。胡媛媛创立了同名品牌时装薇薇安胡并于2012年纽约春夏时装周进行了首秀<ref name=mb>{{Cite web|title = Mercedes-Benz Fashion Week|url = http://mbfashionweek.com/designers/vivienne-hu|website = mbfashionweek.com|accessdate = 2015-12-23}}</ref>。2013年，胡媛媛在[[纽约苏豪区|纽约苏豪区]]建立了第一家旗舰店<ref name=":5" />。2016年，胡媛媛在[[纽约苏豪区|纽约苏豪区]]中心建立了在美国的第二家店<ref name=":5" />

从2014年9月开始，胡媛媛开始在[[奔驰|奔驰]][[纽约时装周|纽约时装周]]举办个人时装秀<ref name=mb/>。2015年7月，胡媛媛被[[施华洛世奇|施华洛世奇]]水晶选中成为2016年度系列设计师之一<ref>{{Cite web|title = Swarovski Collective 2016 Designers Announced|url = http://wwd.com/fashion-news/fashion-scoops/swarovski-collective-2016-designers-10199295/|website = WWD|accessdate = 2015-12-23|language = en-US}}</ref>。2015年3月，胡媛媛在深圳时装周举办了她的秋冬系列设计，这是她品牌的第一次海外亮相<ref>{{Cite web|title = International designs grace local runway_Welcome to Dongguan Today|url = http://www.dongguantoday.com/news/guangdong/201503/t20150323_5179451.shtml|website = www.dongguantoday.com|accessdate = 2015-12-23}}</ref><ref>{{Cite web|title = A model walks the runway during Vivienne Hu show as part of the...|url = http://www.gettyimages.com/detail/news-photo/model-walks-the-runway-during-vivienne-hu-show-as-part-of-news-photo/467109942|website = Getty Images|accessdate = 2015-12-23|language = en-US}}</ref>。

在运营个人品牌之外，胡媛媛也曾担任[[帕森设计学院|帕森设计学院]]的时装设计评审<ref name=":0">{{Cite web|title = About Us|url = https://www.viviennehu.com/shop/about_us|website = www.viviennehu.com|accessdate = 2015-12-23}}</ref>。{{Fact|2015年6月，薇薇安胡获得富国银行集团“亚太华裔终身成就奖”|time=2015-12-25}}。
