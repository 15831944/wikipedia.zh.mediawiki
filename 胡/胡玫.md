'''胡玫'''（{{bd|1958年|9月2日|catIdx=Hu,胡}}），中国大陆女导演。现任全国[[人大代表|人大代表]]、全国文联委员、中国电视艺术家协会副主席、[[中国电影集团|中国电影集团]][[一级导演|一级导演]]。

== 生平 ==

1982年毕业于中国[[北京电影学院|北京电影学院]]导演系。前夫是导演[[苏舟|苏舟]]，现夫是[[何新|何新]]。 父亲[[胡德风|胡德风]]是[[八路军|八路军]]文艺兵，后任总政治部文工团歌舞团团长。哥哥[[胡琤|胡琤]]亦常参加电视剧制作，并担任《[[曹操_(电视剧)|曹操]]》一剧的导演（胡玫为该剧总导演）。

== 奖项 ==
*导演的电影《[[女儿楼|女儿楼]]》，1985年被[[中国电影家协会|中国电影家协会]]、[[中国电影报|中国电影报]]评为全国十佳影片。
*导演的电影《[[远离战争年代|远离战争年代]]》，1988年获[[苏联|苏联]]第10届[[亚非拉国际电影节|亚非拉国际电影节]]银奖。1989年获[[意大利|意大利]]第32届[[萨尔索国际电影节|萨尔索国际电影节]][[评委会特别奖|评委会特别奖]]。
*1987年获[[中国电影家协会|中国电影家协会]]、[[当代电影|当代电影]]杂志、[[中国电影报|中国电影报]]三家联合评出的“中国十大青年导演”称号。
*1990年获[[法国政府|法国政府]]颁发的国家奖学金。
*1998年被评为中国十大优秀女导演，[[双十佳|双十佳]]电视剧导演。
*导演的电视剧《[[雍正王朝|雍正王朝]]》，1999年获[[飞天奖|飞天奖]]、[[金鹰奖|金鹰奖]]，[[五个一工程奖|五个一工程奖]]，[[优秀电视剧奖|优秀电视剧奖]]。
*导演的电视剧《[[忠诚_(电视剧)|忠诚]]》，获2001年度飞天奖，优秀电视剧二等奖，五个一工程奖。
*导演的电视剧《[[香樟树_(电视剧)|香樟树]]》，获2005年[[飞天奖|飞天奖]]优秀电视剧奖。
*导演的电视剧《[[汉武大帝|汉武大帝]]》，2005年8月获2005年飞天奖优秀电视剧奖，同时胡玫获飞天奖最佳导演一等奖。
*2005年9月26日，获得[[大众电视|大众电视]]2005“双十佳”导演奖。
*2008年10月，被[[中国广播电视协会|中国广播电视协会]]评为中国电视剧辉煌30年最具影响力导演之一。
*2011年，电影《孔子》获得中国电影华表奖“最优合拍片奖。”

== 娱乐圈潜规则事件 ==
 
2007年末，男演员[[丁军|丁军]]在博客上发表过多篇有关何新和胡玫的文章，《胡玫的封建核心与何新的专制狐媚》讽刺了这两口子以艺术为幌子合伙谄媚权贵，而《我曾经被导演胡玫潜规则过!》一文更是引起了娱乐圈的轩然大波。胡玫和何新随之发表声明要和丁军打名誉官司<ref>[http://bjyouth.ynet.com/article.jsp?oid=26410196  北京青年报：胡玫辟谣“潜规则”发布会开得蹊跷]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://ent.newssc.org/system/2008/01/22/010655762.shtml 谁制造了胡玫潜规则？] {{Wayback|url=http://ent.newssc.org/system/2008/01/22/010655762.shtml |date=20160305103021 }}</ref>。此事后来无疾而终，但却成为娱乐圈影视圈以及曲艺界调侃的素材<ref>[http://news.ifensi.com/tv/2007-12-26/article-127580.html 潜规则春晚成热门话题 胡玫恼怒郭德纲开涮 - 娱乐新闻] {{webarchive|url=https://web.archive.org/web/20080724045920/http://news.ifensi.com/tv/2007-12-26/article-127580.html |date=2008-07-24 }}</ref>。有内幕知情者透露，这个事件可能是丁军妻子[[武丹丹|武丹丹]]与演员[[王子文|王子文]]以及著名作家[[王朔|王朔]]的官司纠纷的附属产物。
