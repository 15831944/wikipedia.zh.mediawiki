'''胡乔木'''（{{bd|1912年|6月1日<ref>[http://news.xinhuanet.com/politics/2013-06/18/c_124869623.htm 胡乔木如何进入和淡出毛泽东视野] {{webarchive|url=https://web.archive.org/web/20130827131105/http://news.xinhuanet.com/politics/2013-06/18/c_124869623.htm |date=2013-08-27 }}</ref>|1992年|9月28日|胡乔木}}），原名'''胡鼎新'''，1912年出生于[[江苏省|江苏省]][[盐城市|盐城市]][[龙冈镇|龙冈镇]]。1930年毕业于[[扬州中学|江苏省立扬州中学]]，1930年至1932年就读于[[清华大学|清华大学]]物理系和历史系，1933年经上海抵达杭州，插班就读于浙江大学外语系二年级，1935年被浙江大学开除。[[马克思主义|马克思主义]]理论家。从1941年2月起至1969年（66年-69年养病，但未撤职），任[[毛泽东|毛泽东]]的[[秘书|秘书]]。曾担任[[中国社会科学院|中国社会科学院]]院长，[[新华社|新华社]]社长，[[中共中央书记处|中共中央书记处]]候补书记，[[中共中央政治局|中共中央政治局]]委员，[[中共中央顾问委员会|中共中央顾问委员会]]常委等职。

== 生平 ==
*1930年加入[[中国共产主义青年团|中国共产主义青年团]]。1932年转入[[中国共产党|中国共产党]]。曾任[[共青团|共青团]][[北平|北平]]西郊区委书记，[[共青团|共青团]][[北平市|北平市]]委宣传部部长。参与领导北平学生和工人的抗日爱国运动。1935年后，任[[中国社会科学家联盟|中国社会科学家联盟]]书记，[[中国左翼文化界总同盟|中国左翼文化界总同盟]]书记，[[中共|中共]][[江苏省|江苏省]]临时工委委员。
*1937年后，任[[安吴青训班|安吴青训班]]副主任，[[中共中央青委|中共中央青委]]委员，[[中国青年联合会|中国青年联合会]]办事处宣传部部长。
*1941年任[[毛泽东|毛泽东]]秘书，[[中共中央政治局|中共中央政治局]]秘书。
*1945年参与起草了《[[中国共产党中央委员会关于若干历史问题的决议|中国共产党中央委员会关于若干历史问题的决议]]》。奉毛澤東指示，胡喬木代[[延安|延安]]《[[解放日報|解放日報]]》撰寫社論《評國民黨大會各文件》，着重批判「看似漂亮實質反動」的文件。<ref>[[錢鋼|錢鋼]]《國共分手備忘錄》二之一，[[香港|香港]]，《[[明報|明報]]》，2005年5月6日，D8版。</ref>
*1948年后任[[新华通讯社|新华通讯社]]社长。
*1949年后，历任[[新华社|新华社]]社长（1949年10月1日至10月19日），[[中央人民政府新闻总署|中央人民政府新闻总署]]署长，[[中共中央宣传部|中共中央宣传部]]副部长，[[政务院文化教育委员会|政务院文化教育委员会]]秘书长，中共中央副秘书长。参与起草了第一部《[[中华人民共和国宪法|中华人民共和国宪法]]》（1954年）。
*1956年当选為中共第八届中央委员、[[中央书记处|中央书记处]]候补书记。1975年后任[[中華人民共和國國務院|国务院]]政治研究室负责人。1977年后任[[中国社会科学院|中国社会科学院]]院长、顾问、名誉院长，[[中共中央|中共中央]]副秘书长、[[毛泽东|毛泽东]]著作编辑出版委员会办公室主任，[[中共中央党史研究室|中共中央党史研究室]]主任。 1978年补选为[[中國共產黨第十一次全國代表大會|中共第十一屆]][[中共中央委員會|中央委员]]。
*1964年12月在杭州對西湖風景區整頓，以封建迷信及破舊立新名義，將佛教遺存、辛亥諸烈士等墓冢拆毀，並炸毀革命烈士[[秋瑾|秋瑾]]女士墓所。
*1975年，[[鄧小平|鄧小平]]復出，任國務院副總理職務。隨即成立“政治研究室”，避開[[四人幫|四人幫]]，設在國務院，由自己直接領導。其中共有七個負責人：[[胡喬木|胡喬木]]、[[吴冷西|吳冷西]]、[[胡繩|胡繩]]、[[熊复|熊復]]、[[于光遠|于光遠]]、[[李鑫_(中办)|李鑫]]、[[鄧力群|鄧力群]]。[[龔育之|龔育之]]稱之為鄧小平的“秀才班子”。<ref>{{cite web|title=《数风流人物》往期人物之于光远《老马嘶风》|url=http://www.iqilu.com/html/weishi/sflrw/wangqi/2010/0429/229803.html|website=山东电视台齐鲁网}}</ref>
*1980年当选为[[中共中央书记处|中共中央书记处]]书记。
*1982年9月当选为中共第十二届[[中共中央政治局|中央政治局]]委员。主持起草了《[[中国共产党中央委员会关于建国以来党的若干历史问题的决议|中国共产党中央委员会关于建国以来党的若干历史问题的决议]]》等重要文件。
*1987年当选为[[中共中央顾问委员会|中共中央顾问委员会]]常委。曾任中共中央党史工作领导小组副组长，中共中央文献研究室主任。是第一、二、三、五届[[全国人大|全国人大]]常委。
*1992年9月28日在[[北京|北京]]逝世，终年81岁。遗体于10月4日火化。逝世后，中共中央，中央顾问委员会的讣告称其“久经考验的忠诚的共产主义战士、无产阶级革命家、杰出的马克思主义理论家、政论家和社会科学家、我党思想理论文化宣传战线的卓越领导人。”是毛历任秘书中评价最高的。按照生前遗愿，10月26日将骨灰撒在[[延安|延安]]地区。
*1994年，《回忆胡乔木》、《胡乔木文集》出版发行。
*1995年12月，胡乔木生前1.4万件档案资料和4万余册图书，由其子女分别捐赠给[[中央档案馆|中央档案馆]]、当代中国研究所和[[江苏|江苏]][[盐城市|盐城市]]。

=== 长子涉贪 ===
[[胡耀邦|胡耀邦]]担任中共中央总书记期间开展反腐败行动。1985年，胡耀邦批准对胡乔木「抄家」，带走他的长子[[胡石英|胡石英]]。当时，中央办公厅趁胡乔木到中央书记处开会时，安排公安人员进驻他家，从当事人房间的床底下搜出15万人民币现金，在当时是天文数字。事隔一天，中共政治局开会，胡乔木提前来到会场，大发脾气质问：「為什麼抄我的家？為什麼逮捕我的儿子？」胡耀邦解释：「三个部门要求立案审查，我们不能干涉司法程序，等案子结论出来了再说吧」。<ref>{{cite web |url = http://www.chinese.rfi.fr/%E4%B8%AD%E5%9B%BD/20121103-%E8%83%A1%E8%80%80%E9%82%A6%E5%BD%93%E5%B9%B4%E5%8F%8D%E8%B4%AA%E5%B0%B1%E9%9A%BE%E5%A6%82%E9%80%86%E6%B0%B4%E8%A1%8C%E8%88%9F |title = 胡耀邦当年反贪就难如逆水行舟 |publisher = [[法国国际广播电台|法国国际广播电台]] |website = 法广中文网 |date = 2012年11月3日 }}</ref>

== 評價 ==
曾任中宣部理論局副局長的[[李洪林|李洪林]]認為胡喬木成功地阻止了中國的改革進程。第一次是1979年[[胡耀邦|胡耀邦]]提出一個“生產目的”的討論，胡喬木要求停止討論，以免引起黨內不和。第二次是1980年，[[李維漢|李維漢]]找[[鄧小平|鄧小平]]談封建專制主義對黨的影響，鄧小平同意李維漢的觀點，政治局也決定要反封建主義，胡喬木主張先指定幾個人研究出個方案，結果束之高閣。第三次是政治制度的改革，鄧小平的講話《改革黨和國家的領導制度》，非常明確地提到政治改革。正在這時，國際上發生了波蘭工人大罷工事件，胡喬木趕緊又寫了一封信給鄧小平，說要防止波蘭事件在中國重演，要引為殷鑒，政治改革就此停止<ref>{{cite book | title=中國思想運動史: 1949-1989| author=李洪林 | publisher=天地圖書有限公司 | year=1999 |isbn=9629506807 |page=277-289}}</ref>。
