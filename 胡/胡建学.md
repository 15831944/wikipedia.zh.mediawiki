'''胡建学'''{{BD|1950年||||catIdx= H胡建学}}出生于[[中国大陆|中国大陆]][[山东省|山东省]]，毕业于[[南京大学|南京大学]]化学系，最高职务为[[中共|中共]][[山东省|山东省]][[泰安市|泰安市]][[市委书记|市委书记]]，因为贪污受贿而被判处死缓。<ref>{{cite web |url= http://news.163.com/05/1101/14/21FU61VL0001126S.html|title= 山东泰安市委原书记胡建学贪污被判处死缓|accessdate= 2005年|work= |publisher= 网易|language =zh-cn}}</ref>

== 生平 ==
1950年，胡建学出生于[[中国大陆|中国大陆]][[山东省|山东省]]。1965年，加入[[中国人民解放军|中国人民解放军]]。1971年，加入了[[中国共产党|中国共产党]]。1990年，出任[[山东省|山东省]][[泰安市|泰安市]]委副书记、[[市长|市长]]，6月升职为[[中共|中共]][[泰安市|泰安市]][[市委书记|市委书记]]。胡建学在任职中共泰安市委书记期间，贪污巨额物资财产，1995年11月被[[中国共产党|中国共产党]]开除党籍、开除公职。

1996年6月，被判处死刑，缓期二年执行。

== 个人生活 ==
胡建学热爱读书学习，他经常阅读中国古代相术书籍，长期携带《[[麻衣相法|麻衣相法]]》、《[[柳庄相法|柳庄相法]]》、《[[相术大全|相术大全]]》，以便研究<ref>{{cite web |url=http://news.qq.com/a/20120319/000520.htm?pgv_ref=aio2012&ptlang=2052 |title= 人民日报评贪官读书：胡长清爱看《肉蒲团》|accessdate= 2012年|work= |publisher=腾讯网 |language =zh-cn}}</ref><ref>{{cite web |url= http://china.rednet.cn/c/2012/03/19/2553224.htm|title= 人民日报评贪官读书 胡长清爱看肉蒲团金瓶梅|accessdate= 2012年|work= |publisher=中国网 |language =zh-cn}}</ref>。

有“大师”预测其可当副总理，但命里缺桥，因此他下令将已按计划施工的国道改道，使其穿越一座水库，并顺理成章地在水库上修起一座大桥，帮助其“飞黄腾达”<ref>{{cite web |url= http://news.ifeng.com/mainland/detail_2013_07/31/28091627_0.shtml|title= 山东一官员因“大师”称其命里缺桥下令将国道改道|accessdate= 2013年|work= |publisher=成都晚报 |language =zh-cn}}</ref>。
