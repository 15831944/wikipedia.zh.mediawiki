'''胡风'''（{{bd|1902年|11月2日|1985年|6月8日|catIdx=H胡}}），[[湖北|湖北]][[蕲春|蕲春]]人，原名'''张光人'''，笔名谷非、高荒、张果等。文艺理论家，文学评论家，翻译家，[[七月派|七月派]]诗人，中国[[左翼|左翼]]文化代表人之一。曾任[[中國左翼作家聯盟|中國左翼作家聯盟]]宣传部部长，与[[鲁迅|鲁迅]]甚有交往<ref name="胡风生平">
{{cite web 
|url = http://www.people.com.cn/GB/wenhua/22226/29632/29633/2069742.html
|title = 胡风生平
|publisher = 人民网
|language = zh-hans
}}</ref>。[[中华人民共和国|中华人民共和国]]成立后，因其文艺思想与主政者不和而遭到整肃，并掀起一场涉及面巨大的政治批判运动。[[改革开放|改革开放]]后被[[平反冤假错案|平反]]。

==生平==
===早年===
因家中贫寒，11岁才上学。1922年入[[國立東南大學|國立東南大學]]附屬中學（後改為[[國立中央大學|中央大學]]實驗中學與附屬中學、[[南京大學|南京大學]]附中、[[南師附中|南師附中]]），曾與[[巴金|巴金]]等人同學，加入[[中国共青团|中国共青团]]。1925年下半年考入[[北京大学中文系|北京大学中文系]][[预科|预科]]，1926年又考入清华大学西洋文学系，未转团组织关系。

在[[国民革命军北伐|国民革命军北伐]]的风潮中，胡风于1926年回到湖北家乡参加打倒[[民国军阀|军阀]]的斗争。大革命失败后他从鄂东家乡逃到武汉，正逢汪精卫主持[[武汉国民政府|武汉国民政府]]“[[清党|清党]]”，胡风对革命前途悲观，于1927年至1928年期间，先后在中国国民党湖北省党部宣传部任干事、在[[江西剿共军|江西剿共军]]第三十一军政治部宣传科担任了28天科长，负责编辑湖北省党部机关刊物《武汉评论》、江西省党部机关报《国民日报》副刊，曾撰写《反共宣传大纲》等有反共内容的文章。1980年，[[中共中央组织部|中共中央组织部]]、公安部给胡风案平反时，认定胡风的这一段历史，属于政治历史问题，不应以历史反革命论罪。1985年4月，公安部对胡风历史上的问题（关于胡自动退团问题；在国民党省党部任职；在剿共军任反动职务；写“反共宣传大纲”等）又进一步复查，经研究并请示中央书记处同意，认为胡申诉的五个历史问题，有的仍然证据不足，如“撰写反共宣传大纲”仅一人检举，胡一直否认。<ref>[http://www.mps.gov.cn/n16/n1327/n4834/n1452547/1467152.html 公安部网站：公安史话——《“胡风案件”的前前后后》] {{webarchive|url=https://web.archive.org/web/20150725002703/http://www.mps.gov.cn/n16/n1327/n4834/n1452547/1467152.html |date=2015-07-25 }}</ref>

胡风于1929年赴日[[庆应大学|庆应大学]]英文科留学，加入[[日本普罗科学研究所|日本普罗科学研究所]]艺术研究会<ref name="胡风生平"/>，并在日本刊物《[[艺术学研究|艺术学研究]]》和《[[普罗文学讲座|普罗文学讲座]]》上介绍过中国左翼革命文学。加入了[[日本共产党|日本共产党]]、反帝同盟和日本左翼作家组织，担任[[中国左翼作家联盟|中国左翼作家联盟]]东京分盟负责人之一。由于胡风在留日中国学生中宣传马克思主义和参加反战（918事变）活动，1933年3月被日本政治警察机关特高科密捕，6月被驱逐回国。

===进入左联===
[[File:Hu_feng_and_his_wife.jpg|thumb]]]]

1933年回上海参加左翼文艺运动，曾任左联宣传部长、行政书记。

1935年，编辑秘密丛刊《[[木屑文丛|木屑文丛]]》发表小说，反映[[中國共產黨根據地|苏区]]斗争，并且介绍[[苏联|苏联]][[社会主义现实主义|社会主义现实主义]]的理论。

1936年初，合作编辑《[[海燕|海燕]]》，翻译反映[[朝鲜|朝鲜]]、[[台湾|台湾]]人民反殖民抗争的短篇小说集《[[山灵|山灵]]》，收錄[[台灣|台灣]]左翼作家用[[日本語|日本語]]創作的小說，如[[楊逵|楊逵]]《[[送報伕|送報伕]]》、[[呂赫若|呂赫若]]《[[牛車|牛車]]》等，是世界漢譯日本語[[台灣文學|台灣文學]]的第1人。

1937年[[中國抗日戰爭|抗战]]爆发后，胡风在上海编辑《[[七月_(雜誌)|七月]]》文学周刊，出版了《七月诗丛》和《七月文丛》丛书。

1938年，当选为[[中华全国文艺界抗敌协会|中华全国文艺界抗敌协会]]常委，兼任[[复旦大学|复旦大学]]教授，著有诗集《为祖国而歌》、杂文集《棘原草》、文艺批评论文集《剑·文艺·人民》、《论民族形式问题》，译文集《人与文学》。

1941年，[[皖南事变|皖南事变]]后他编辑出版了文学杂志《[[希望_(雜誌)|希望]]》，其间著有文艺批评论文集《为了明天》《论现实主义的路》等。

1949年，[[中华人民共和国|中华人民共和国]]建国后，1949年起任[[中国文学艺术界联合会|中国文学艺术界联合会]]委员、[[中国作家协会|中国作家协会]]理事、[[第一届全國人民代表大會代表|第一届全國人民代表大會代表]]。著有抒情长诗《时间开始了！》和特写集《和新人物在一起》，杂文短记《从源头到洪流》，长诗《为了朝鲜·为了人类》<ref name="胡风生平" />。

===遭到批判===
{{Main|胡风反革命集团案}}

胡风虽然在政治上拥护中共，但在文艺理论的主张上多次与中共在文艺界领导人（如[[周扬_(学者)|周扬]]）的观点相冲突，从1945年起，他的文艺理论多次受到批评，性情直率的胡风则也毫不客气的做反批评<ref name="洪子诚">

1951年起，一些人写信给《文艺报》编辑部，要求再次批评胡风文艺思想。1952年初，《〈文艺报〉通讯员内部通报》陆续刊登了这些来信，从此拉开运动序幕。1955年1月中共中央批转中央宣传部《关于开展批判胡风思想的报告》。2月，中国作协主席团扩大会议决定对胡风文艺思想进行全面批判<ref name="洪子诚"/>。4月13日，胡风旧友[[舒芜|舒芜]]在《[[人民日报|人民日报]]》上发表抨击文章《胡风文艺思想反党反人民的实质》，同时还交出了在1940年代与胡风的私人通信。5月13日至6月10日[[毛泽东|毛泽东]]发起批判[[胡适|胡适]][[资产阶级|资产阶级]][[唯心主义|唯心主义]]运动，认为胡风反对《[[在延安文艺座谈会上的讲话|在延安文艺座谈会上的讲话]]》，因此令《人民日报》在5月13日至6月10日发表《关于[[胡风反革命集团|胡风反革命集团]]的材料》共三批，并亲自写编者按<ref>
{{cite book 
|author = 毛泽东
|editor = 
|title = 《建国以来毛泽东文稿》
|volume = 第5卷
|pages = 第112–15页
|language = zh}}</ref>。在此期间，《人民日报》还在每天第三版用五分之二以上甚至全幅整版，刊登声讨文章、读者来信和漫画鼓动声势。胡风随后被判为[[反革命|反革命]]，并于同年5月18日被捕。同时还涉及2100多人，逮捕其中92人，正式定为“胡风反革命集团分子”的78人，骨干23人。胡风入狱十年后被正式判刑14年，后[[文革|文革]]时期重判为无期徒刑。

===平反及去世===
1979年获释，1980年、1986年、1988年，经历三次平反过程，才彻底平反。其间曾出任[[第六届全国政协|第六届全国政协]]常务委员、[[中国文联|中国文联]]委员、[[中国作家协会|中国作家协会]]和[[中華人民共和國文化部|文化部]][[文学艺术研究院|文学艺术研究院]]顾问等职。在彻底平反前的1985年因患癌症，在北京逝世，享年83歲<ref>
{{cite web|title =胡风的三次平反|author =周正章|date =2005年12月|publisher =[[党史信息报|党史信息报]]|url =http://www.dqdaj.gov.cn/show.aspx?id=452&cid=36|accessdate =2009年3月|language =zh-hans}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==家人==
妻子[[梅志|梅志]]，原名屠玘华，作家。儿子[[张晓山|张晓山]]，中国社会科学院农村发展研究所原所长，研究员，农村问题研究专家。

== 作品及著作 ==
在中国文坛是一位创作颇丰、活跃的[[文艺理论|文艺理论]][[批评家|批评家]]，他的理论涉及诗歌、报告文学、小说、戏剧、电影、儿童文学、小品文等，其文学理论著述辑成三卷本《胡风评论集》，由人民文学出版社出版。胡风一生主要致力于文学理论研究，著有《为祖国而歌》，杂文集《棘原草》，文艺批评论文集《剑·文艺·人民》、《论民族形式问题》、《在混乱里面》、《逆流的日子》、《为了明天》、《论现实主义的路》，散文集《人环二记》，译文集《人与文学》，特写集《和新人物在一起》，杂文短记《从源头到洪流》，《关于几年来文艺实践情况的报告》（即《[[三十万言书|三十万言书]]》）等。
