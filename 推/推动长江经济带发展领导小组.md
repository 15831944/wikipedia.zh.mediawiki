'''推动长江经济带发展领导小组'''，是[[中华人民共和国国务院|中华人民共和国国务院]]成立的[[国务院议事协调机构|国务院议事协调机构]]，负责推动[[长江经济带|长江经济带]]发展。

==沿革==
2013年7月，[[中共中央总书记|中共中央总书记]][[习近平|习近平]]在[[武汉|武汉]]调研时指出，长江流域要加强合作，发挥内河航运作用，把全流域打造成黄金水道。2014年12月，习近平作出批示<ref name=wen>{{Cite web |url = http://www.xinhuanet.com/politics/2016-09/11/c_1119546883.htm |title = 推动长江经济带发展领导小组办公室负责人就长江经济带发展有关问题答记者问 |author =  |date =2016-09-11  |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。2016年1月初，习近平在[[重庆|重庆]]主持召开推动长江经济带发展座谈会<ref>{{Cite web |url = http://www.chinanews.com/cj/2016/05-09/7863100.shtml |title = 发改委：强力推进长江经济带生态保护和治理 |author =  |date =2016-05-09  |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref>。

2014年9月12日，《国务院关于依托黄金水道推动长江经济带发展的指导意见》（国发〔2014〕39号）发出，该意见的附件是《长江经济带综合立体交通走廊规划（2014—2020年）》<ref>{{Cite web |url = http://www.gov.cn/zhengce/content/2014-09/25/content_9092.htm |title = 国务院关于依托黄金水道推动长江经济带发展的指导意见（国发〔2014〕39号） |author =  |date = 2014-09-12 |publisher =中国政府网  |language =  |accessdate =  |quote =  }}</ref>。2014年12月，中央经济工作会议提出，要重点实施长江经济带等三大战略，争取2015年有良好开局。[[长江|长江]]是中国第一大河流，是货运量居全世界内河第一的黄金水道。长江经济带覆盖九省两市（[[上海市|上海市]]、[[江苏省|江苏省]]、[[浙江省|浙江省]]、[[安徽省|安徽省]]、[[江西省|江西省]]、[[湖北省|湖北省]]、[[湖南省|湖南省]]、[[重庆市|重庆市]]、[[四川省|四川省]]、[[云南省|云南省]]、[[贵州省|贵州省]]），面积约205万平方公里，人口和生产总值均超全国的40%<ref name=zhashi>{{Cite web |url =  http://www.xinhuanet.com/politics/2015-02/07/c_1114291720.htm|title = 张高丽：扎实有序做好推动长江经济带发展工作 |author =  |date = 2015-02-07 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。

2014年，成立'''推动长江经济带发展领导小组'''，并成立领导小组办公室。为贯彻落实中共中央、国务院决策部署，2014年12月底，领导小组办公室会同有关部门和沿江11省市启动《长江经济带发展规划纲要》研究编制工作。2016年，《长江经济带发展规划纲要》印发<ref name=wen/>。

2015年2月6日，推动长江经济带发展工作会议在[[北京|北京]]召开，[[中共中央政治局常委|中共中央政治局常委]]、[[中华人民共和国国务院副总理|国务院副总理]][[张高丽|张高丽]]主持会议并讲话<ref name=zhashi/>。

2015年6月9日，[[中华人民共和国国家发展和改革委员会|国家发展改革委]]发出《关于建设长江经济带国家级转型升级示范开发区的实施意见》（发改外资〔2015〕1294号）<ref name=yijian>{{Cite web |url = http://www.ndrc.gov.cn/zcfb/zcfbtz/201506/t20150630_709628.html |title =关于建设长江经济带国家级转型升级示范开发区的实施意见（发改外资〔2015〕1294号）  |author =  |date = 2015-06-09 |publisher = 中华人民共和国国家发展和改革委员会 |language =  |accessdate =  |quote =  }}</ref>。

2016年，推动长江经济带发展领导小组办公室围绕建立和完善长江经济带11省市协商合作机制，印发了《长江经济带省际协商合作机制总体方案》，并到沿江各省市调研，召开专题座谈会，研究起草长江经济带省际协商合作机制实施意见<ref>{{Cite web |url = http://finance.sina.com.cn/roll/2016-05-16/doc-ifxsenvn7224390.shtml?/www.sina.com.cn/favicon.svg |title =省部级官员谈长江经济带建设  |author =  |date = 2016-05-16 |publisher =新浪 |language =  |accessdate =  |quote =  }}</ref>。
