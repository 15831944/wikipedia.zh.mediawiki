'''推进海南全面深化改革开放领导小组'''，是[[中华人民共和国国务院|中华人民共和国国务院]]成立的[[国务院议事协调机构|国务院议事协调机构]]，负责推进[[海南|海南]]全面深化改革开放。

==沿革==
2018年4月11日，《中共中央 国务院关于支持海南全面深化改革开放的指导意见》（中发〔2018〕12号）发出<ref>{{Cite web |url = http://news.hainan.net/hainan/yaowen/yaowenliebiao/2018/04/15/3641730.shtml |title = 重磅发布！中共中央 国务院关于支持海南全面深化改革开放的指导意见 |author =  |date = 2018-04-15 |publisher = 海南在线 |language =  |accessdate =  |quote =  }}</ref>。

2018年4月13日，[[习近平|习近平]]在庆祝海南建省办经济特区30周年大会上讲话强调，[[中共中央|中共中央]]支持海南全面深化改革开放，争创[[新时代中国特色社会主义|新时代中国特色社会主义]]生动范例，指出[[海南经济特区|海南经济特区]]的四个战略定位是“改革开放的重要窗口、改革开放的试验平台、改革开放的开拓者、改革开放的实干家”，并表示中央最近研究制定了《关于支持海南全面深化改革开放的指导意见》<ref>{{Cite web |url = http://www.xinhuanet.com/politics/2018-04/13/c_1122680106.htm |title =习近平在庆祝海南建省办经济特区30周年大会上发表重要讲话  |author =  |date = 2018-04-13 |publisher =  新华网|language =  |accessdate =  |quote =  }}</ref>。

2018年7月1日至3日，[[中共中央政治局常委|中共中央政治局常委]]、国务院副总理、推进海南全面深化改革开放领导小组组长[[韩正|韩正]]在海南省调研。7月2日，韩正在[[海口|海口]]主持召开'''推进海南全面深化改革开放领导小组'''会议<ref name=diyi>{{Cite web |url = http://www.gov.cn/guowuyuan/2018-07/04/content_5303519.htm |title =韩正在海南调研并主持召开推进海南全面深化改革开放领导小组会议  |author =  |date = 2018-07-04 |publisher = 中国政府网 |language =  |accessdate =  |quote =  }}</ref>。
