《'''最近，妹妹的樣子有點怪？'''》（{{lang-ja|最近、妹のようすがちょっとおかしいんだが。}}）是日本漫畫家{{tsl|ja|松沢まり}}的漫畫作品。於《[[月刊Dragon_Age|月刊Dragon Age]]》2010年12號開始連載。

2014年1月開始播放電視動畫<ref>{{cite web|url=http://www.animenewsnetwork.com/news/2013-10-09/saikin-imoto-no-yosu-ga-chotto-okaishiin-da-ga-to-air-in-january|title=''Saikin, Imōto no Yōsu ga Chotto Okaishiin Da Ga'' to Air in January|publisher=[[動畫新聞網|Anime News Network]]|date=2013年10月9日|accessdate=2013年11月4日|language=en}}</ref>。電視版中，TST以心型大锁作遮挡，DVD（BD）版为无锁。

2014年5月上映真人电影版。

== 故事簡介 ==
母親再婚，與新「哥哥」夕哉一起生活——光是生活上的劇變就已經煩惱不已的神前美月，某日突然出現一名自稱是幽靈的神祕少女日和，並且附身在美月身上。能讓日和成佛的條件，竟然是要和她愛慕的「哥哥」相親相愛。最過分的是被迫穿上一件奇怪的內褲，这是为什么？日和在美月身上一丝丝不安开始展开……

== 登場人物 ==
=== 主要人物 ===
; {{nihongo|神前美月|神前 美月（かんざき みつき）|Kanzaki Mizuki|聲：[[橋本千波|橋本千波]]}}
: 就讀美櫻高中1年A班，生日12月21日。身高156cm，三圍B76，W55，H78。
: 親生父親是位有暴力傾向，愛賭博，拈花惹草的男性，後拋棄其母。
: 母親再婚後成為夕哉的[[繼親|繼妹]]，舊姓「明坂」。
: 第二學期中時，在天橋下遇上幽靈日和，被其附身。
: 美術部部員，不擅長運動，學習方面不擅長數學，原本喜歡吃[[鹹味|鹹味]][[煎蛋卷|煎蛋卷]]（{{tsl|ja|卵焼き|玉子燒}}）。
: 但因為日和的關係而變得喜歡吃[[甜味|甜味]][[煎蛋卷|煎蛋卷]]，喜歡[[南瓜|南瓜]]。
: 對夕哉有好感，有時看到跟夕哉要好的雪那會吃醋，
: 因為這樣日和常常跟美月講要積極一點，不然會被雪那搶走呢。
: 漫畫中，夕哉的記憶中浮出小時候美月及日和在水族館玩的記憶。
: 可能認識在生時的日和。於最終話跟子瑞希做朋友。

; {{nihongo|神前夕哉|神前 夕哉（かんざき ゆうや）|Kanzaki Yuuya|聲：[[間島淳司|間島淳司]]、[[Lynn|Lynn]]}}
: 美櫻高中2年級生。親生母親在懂事前過世，父親再婚後成為美月的[[繼親|繼兄]]。
: 努力與美月建立良好的關係。非常單純。
: 擅長[[家務|家務]][[烹饪|烹饪]]。動畫中提到不太擅長唱歌一事<ref>動畫13集</ref>。
: 有時能夠看見日和，但把她認為是美月的朋友，而不知道為幽靈。
: 對於美月性格的不同轉換，認為美月是[[人格分裂|人格分裂]]。
: 多次目擊到美月的TST（[[貞操帶|貞操帶]]），現在認為美月有[[Cosplay|Cosplay]]的興趣。
: 對繼妹、義姊沒有特別的情感，只把她們當成普通的姐姐與妹妹而已，
: 而同樣的也只認為日和是美月的朋友，沒有特別的情感。
: 漫畫中，記憶中浮出小時候美月及日和在水族館玩的記憶，
: 小時候可能認識日和及美月。

; {{nihongo|壽日和|寿 日和（ことぶき ひより）|Kotobuki Hiyori}}/{{nihongo|子瑞希|子 瑞希（こ みずき）|Ko Mizuki，聲：[[小倉唯|小倉唯]]}}
: 附在美月身上的幽靈。生前在天橋下路口中發生事故死亡，
: 變成幽靈後沒有任何記憶，只記得自己喜歡夕哉，跟美月一樣夕哉跟雪那要好時會吃醋，
: 有時還會因為這樣借用美月身體對雪那展開反擊。
: 壽日和不是本名，這名字是從故事書中取得靈感所改的。
: 而要[[成佛|成佛]]的話，就需要完成生前的心願—體驗和夕哉卿卿我我的感覺。
: 本身想取代美月的存在，可是受到美月抗拒而無法如願，
: 只好要求美月協助她完成心願。跑得很快，擅長運動，也擅長料理，喜歡吃甜味煎蛋卷。
: 漫畫中，夕哉的記憶中浮出小時候美月及日和在水族館玩的記憶，可能認識美月。
: 於最終話復活，真名叫做子瑞希(第54話的醫院病房有標示出來)，並且跟美月做朋友。
: 復活後的她，髮型改成長的跟美月很像的波浪型長髮，並且把粉紅色蝴蝶結改成粉紅色髮夾，頭髮顏色從橘色改成土黃色。

; {{nihongo|桐谷雪那|桐谷 雪那（きりたに ゆきな）|Kiritani Yukina|聲：[[金元壽子|金元壽子]]}}
: 美櫻高中3年級生。身高163厘米，三圍B95、W63、H85。
: 夕哉青梅竹馬的義姐。小時候曾搬走，現搬到神前家的旁邊。能看見日和及交換身體後變成幽靈的美月。
: 幼年時曾和祖父學習[[劍道|劍道]]，外貌性格和男生一模一樣，長大後變得非常有女人味。被日和稱為「大奶女」。
: 對夕哉有好感，但不明顯，不過有時候卻又會以極其大膽的方式展現好身材給夕哉看，是故意還是只是天然呆就不得而知了，可這種行為常惹得美月、日和吃醋，雪那也曾說過夕哉很遲鈍，對自己以及美月都是。
: 漫畫中，把美月的母親稱為「水族館的姐姐」，認識美月的母親。

; {{nihongo|根子|根子（ねこ）|Neko|聲：[[佐佐木未來|佐佐木未來]]}}

: 自稱父親是游泳池和水族館的老闆。
: 實際上根子是日和生前最愛的一隻白貓，目睹日和出車禍，之後就一直停留在車禍現場直到精力用盡死去。被天神化身為人形以協助日和及神前美月。
: 漫畫最終話恢复成白貓與神前美月相見。

; TST
: 正式名稱是[[貞操帶|貞操帶]]。上面有著心型的計量器，刻度是否上升則視日和與夕哉親熱的滿足度而定，若是滿足度到頂點，通往天之門的階梯會有變化，若滿足度低落，則穿上TST的人類會有死亡的可能。有時會散發出具有[[性激素|性激素]]般功用之光芒，似乎為根子所造成，不過其說法為這是兩人互相想念所發出的心靈之光。
; {{nihongo|時之狹縫|時の狭間}}
: 通往天之門的空間，當把階梯累積到一定程度後再通過天之門，就有成佛的可能。
