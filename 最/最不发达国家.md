'''最不发达国家'''（也稱作-{zh-cn:'''最低度开发国家''';zh-tw:'''最不發達國家''';zh-hk:'''最低度開發國家''';zh-sg:'''最不发达国家''';}-、-{zh-hans:'''未开发国家''';zh-hant:'''未開發國家'''}-，{{lang-en|'''Least developed country'''}}，[[縮寫|縮寫]]：{{lang|en|'''LDC'''}}、{{lang|en|'''LDCs'''}}）是指那些经[[联合国|联合国]]认定的[[社會|社會]]、[[經濟發展|經濟發展]]水平以及[[人類發展指數|人類發展指數]]最低的一系列国家。“最不发达国家”一词最早出现在1967年10月[[77国集团|77国集团]]通过的《阿尔及尔宪章》（{{lang|en|Charter of Algiers}}）中<ref>{{Cite web |url = http://www.g77.org/doc/algier~1.htm | title = CHARTER OF ALGIERS（《阿尔及尔宪章》全文） |publisher = 七十七国集团官网 |language = en}}</ref><ref name="最不发达">{{cite news | author = 杨立杰 | url = http://news.xinhuanet.com/ziliao/2003-09/11/content_1076595.htm | title = 世界最不发达国家 | publisher = 新华网 | date = 2003年9月11日}}</ref>。最不發達國家的概念最初出現於1971年11月18日的聯合國第2768號決議案<ref>[http://www.unitar.org/resource/sites/unitar.org.resource/files/document-pdf/GA-2767-XXVI.pdf 第2768號決議案原稿]{{en}}</ref><ref>[http://www.un.org/chinese/aboutun/prinorgs/ga/26/26all1.htm 聯合國大會第二十六屆會議決議列表]，有鏈結至第二七六八號決議案中文原文－查明發展中國家發展最差各國</ref>。目前，[[联合国经济及社会理事会|联合国经济及社会理事会]]每三年根据[[发展政策委员会|发展政策委员会]]的建议对“最不发达国家名单”作一次审查<ref name="联大批准">{{cite news |language = zh-cn | author = 程浩 | url = http://www.unmultimedia.org/radio/chinese/archives/179533/ | title = 联大批准赤道几内亚和瓦努阿图从“最不发达国家”行列“毕业” | publisher = [[联合国电台|联合国电台]]中文官网 | date = 2013年2月21日}}</ref>。[[聯合國貿易和發展會議|聯合國貿易和發展會議]]每年都會发表《最低度開發國家報告》。

=== 1971年[[联合国大会2768号决议|联合国大会2768号决议]] ===
# [[人均国民生产总值|人均国民生产总值]]在100[[美元|美元]]以下；
# 在国内生产总值中[[制造业|制造业]]所占比重低于10%；
# 人口[[识字率|识字率]]在20%以下。
根据这个标准，当时联合国把24个[[联合国会员国|联合国会员国]]列为最不发达国家。

1981年联合国[[发展计划委员会|发展计划委员会]]{{efn|1998年7月，发展计划委员会（又译为发展规划委员会，the Committee for Development Planning）改名为发展政策委员会（the Committee for Development Policy）。<ref>[http://www.un.org/esa/documents/aac54.htm Committee for Development Planning (A/AC.54)]</ref><ref>[http://www.un.org/en/development/desa/policy/cdp/cdp_res_dec/e_1998_46.pdf ECOSOC Decision on the Committee for Development Policy (1998/46)]</ref>}}把第一条标准改为人均国民生产总值在250美元以下，当时全世界有39个最不发达国家。<ref name="最不发达" />

=== 1991年3月联合国发展计划委员会制定的标准 ===
1991年3月，联合国发展计划委员会对最不发达国家的划定作出新的规定。这一规定把最不发达国家定义为那些长期遭受发展障碍的低收入国家，特别是人力资源开发水平低和有严重结构性缺陷的国家，并将其衡量标准调整为：
# [[人均国内生产总值|人均国内生产总值]]在600美元以下；
# 扩大的实际[[生活质量指数|生活质量指数]]（包括[[预期寿命|预期寿命]]、人均摄取热量、入学率、[[识字率|识字率]]等）不超过47点；
# 经济多种经营指数（包括[[制造业|制造业]]、[[工业|工业]]就业比重等）不超过22点。（人口超过7500万的国家被排除在外）<ref name="最不发达"/>

=== 2015年3月联合国发展政策委员会制定的标准 ===
# 人均收入标准。这项标准依据的是人均[[国民总收入|国民总收入]]（GNI）（3年平均数），1035美元以下的国家可列入最不发达国家名单，1242美元以上的国家从最不发达国家名单中剔除；
# 人力资产标准。这项标准依据的是一项综合指数（人力资产指数），其中包括[[营养|营养]]、健康、入学率、识字率等多项指标；
# 经济脆弱性标准。这项标准依据的是一项综合指数（经济脆弱性指数），其中包括自然冲击程度、经济易受冲击程度、经济规模狭小程度、地理位置偏远程度等多项指标。<ref name="联大批准"/><ref name="标准">{{Cite web |url = http://unohrlls.org/about-ldcs/criteria-for-ldcs/ |title = Criteria for Identification and Graduation of LDCs |publisher = UN－OHRLLS website |language = en}}</ref> 
符合所有三项标准列出的列入名单要求，而且人口不超过7500万的国家，将列入最不发达国家名单；在对名单进行的至少连续两次的三年期审查中达到三项标准中至少两项标准规定的“毕业”门槛的国家，通常可从最不发达国家名单中剔除。不过，如果一最不发达国家的人均GNI增至至少两倍于“毕业”门槛的水平，该国家即被认为有资格“毕业”，不论其在另外两项标准之下的表现如何。<ref name="标准"/><ref>{{Cite web |url = http://unctad.org/en/PressReleaseLibrary/PR12040_ch_LDC.pdf | title = UNCTAD/PRESS/PR/2012/40（新闻稿）|publisher = 联合国贸易和发展会议官网 |language = zh-cn}}</ref>

== 现行最不发达国家名单 ==
根据最新制定的标准，截至目前，全世界经[[联合国|联合国]]批准的最不发达国家总共有47國。其中[[非洲|非洲]]就有33國，[[亞洲|亞洲]]有9国，[[大洋洲|大洋洲]]有4个[[岛国|岛国]]，[[美洲|美洲]]仅有[[海地|海地]]一國，而在[[欧洲|欧洲]]则没有最不发达国家。具体名单如下：<ref>{{Cite web |url = http://unohrlls.org/about-ldcs/ |title = About LDCs |publisher = UN－OHRLLS website |language = en}}</ref><ref>{{Cite web |url = https://www.un.org/development/desa/dpad/wp-content/uploads/sites/45/publication/ldc_list.pdf | title = LIST OF LEAST DEVELOPED COUNTRIES |publisher = 联合国官网 |language = en}}</ref>

== 獲除名的最不发达国家 ==
原最不发达国家从最不发达国家名单中剔除，摆脱最不发达国家身份，联合国称之为“毕业”。最不发达国家经[[联合国大会|联合国大会]]批准“毕业”后，还需要经过一个审查过渡期（一般为期三年，可延期；在此期间仍保留最不发达国家身份），才能够真正“毕业”。迄今为止，只有五个国家从最不发达国家名单“毕业”。此外，[[锡金|锡金]]獲除名不是因为“毕业”，而是因为其被并入[[印度|印度]]。名单如下：
* {{SKIN}}（1975年成为印度的一个[[印度行政区划|邦]]）<ref>{{cite web|url=http://www.un.org/en/development/desa/policy/cdp/cdp_publications/2008cdphandbook.pdf |title=UN Handbook on the LDC Category |format=PDF |date= |accessdate=2014-07-28}}</ref><ref>{{cite web|url=http://sikkim.gov.in/ASP/Miscc/aboutsikkim.htm |title="About Sikkim" from the Government of Sikkim's website |publisher=Sikkim.gov.in |date= |accessdate=2014-07-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20090525171423/http://www.sikkim.gov.in/asp/Miscc/aboutsikkim.htm |archivedate=2009-05-25 }}</ref>
* {{BWA}}（1994年12月除名）
* {{CPV}}（2007年12月除名）
* {{MDV}}（2011年1月除名）
* {{SAM}}（2014年1月除名）
* {{EQG}}（2017年6月除名）<ref name="联大批准"/><ref name="标准"/>
