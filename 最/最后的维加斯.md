《'''賭城大丈夫'''》（{{lang-en|'''''Last Vegas'''''}}）是一部2013年[[強·托特陶|強·托特陶]]导演的美国喜剧片，[[米高·德格拉斯|米高·德格拉斯]]、[[羅拔·迪尼路|羅拔·迪尼路]]、[[摩根·費曼|摩根·費曼]]和[[奇雲·格連|奇雲·格連]]主演<ref>{{cite web |url=http://www.deadline.com/2012/05/morgan-freeman-casting-last-vegas-cbs-films-official/ |title=It’s Official: Morgan Freeman Joins ‘Last Vegas’ |work=Deadline Hollywood |publisher=PMC |date=May 17, 2012 |accessdate=September 18, 2012}}</ref>。美国于2013年11月1日上映<ref>{{cite web |url=http://www.deadline.com/2012/09/cbs-films-last-vegas-gets-december-20-2013-release-date-robert-deniro-michael-douglas-morgan-freeman-kevin-kline-cbs-films-good-universe/ |title=‘Last Vegas’ Gets December 20, 2013 Release Date |work=Deadline Hillywood |publisher=PMC |date=September 5, 2012 |accessdate=September 18, 2012}}</ref><ref>{{cite web |url=http://www.cinemablend.com/new/Bachelor-Party-Comedy-Last-Vegas-Set-Late-2013-Release-32815.html |title=Bachelor Party Comedy Last Vegas Set for a Late-2013 Release |work=cinemablend |publisher=Cinemablend, LLC |date=September 6, 2012 |accessdate=September 18, 2012}}</ref>。

== 剧情 ==
四个居住在全国各地的60多岁的老友为其中最后一位结束单身的好友比利举办单身派对，而来到[[拉斯维加斯|拉斯维加斯]]准备狂欢一次。然而比利和派迪之间却存在着矛盾，原来在派迪的妻子去世时比利没有出席葬礼，而她年轻时是两人共同追求的对象。由于比利的结婚对象只有30多岁，在派迪发现比利对这里的驻唱女歌手戴安娜有好感后，建议比利要做好人生选择，最终比利与戴安娜走到了一起<ref>{{cite web |url=http://www.imdb.com/title/tt1204975/ |title=Last Vegas |work=IMDB |accessdate=September 18, 2012}}</ref>。

== 评价 ==
影片获得混合评价，[[烂番茄|烂番茄]]新鲜度44%，平均得分5.2，一般认为本片是部老年版的《[[醉后大丈夫|醉后大丈夫]]》；[[Metacritic|Metacritic]]上媒体综评48分。

代表性评价：“影片还是展现出了绝对的诚意，为片中每一位大腕都留够了发挥空间”，“显然这些老家伙们玩的非常开心，四人的表演堪称游刃有余，可惜对于银幕前的观众来说不见得如此”，“公式化设计的剧情中分配了足额的笑料，所以从本质上而言它并不会让你感到沉闷，但也不会有多少惊喜”。<ref>[http://ent.sina.com.cn/m/f/2013-11-04/ba4036227.shtml 新片《最后的维加斯》《火鸡》《达拉斯》]</ref>

== 票房 ==
