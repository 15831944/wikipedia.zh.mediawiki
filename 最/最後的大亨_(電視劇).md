《'''最後的大亨'''》（{{lang-en|'''''The Last Tycoon'''''}}）是一部美國電視劇。本劇主演是[[馬修·波莫|馬修·波莫]]和[[凱爾塞·格拉瑪|凱爾塞·格拉瑪]]，鬆散地改編自[[法蘭西斯·史考特·費茲傑羅|法蘭西斯·史考特·費茲傑羅]]的最後一本小說《[[最後的大亨|最後的大亨]]》。{{link-en|亞馬遜工作室|Amazon Studios}}於2016年7月27日將試播集買下成連續劇<ref>{{Cite web |url = http://deadline.com/2016/07/the-last-tycoon-picked-up-series-amazon-matt-bomer-1201793630/ |title = 'The Last Tycoon' Drama Picked Up To Series By Amazon |author = Nellie Andreeva |date = 2016-07-27 |publisher = Deadline.com |language = en}}</ref>。第一季在2017年7月28日首播，<ref>{{Cite web |url = http://deadline.com/2017/06/amazon-last-tycoon-premiere-date-trailer-1202106449/ |title = Amazon Sets 'The Last Tycoon' Premiere Date & Unveils Trailer |author = Greg Evans |date = 2017-06-02 |publisher = Deadline |language = en}}</ref>但本劇於2017年9月9日被取消。<ref>{{Cite web |url = http://www.hollywoodreporter.com/live-feed/last-tycoon-canceled-by-amazon-one-season-1037171 |title = Amazon Abruptly Cancels Pricey Drama 'The Last Tycoon' (Exclusive) |author = Lesley Goldberg & Erik Hayden |date = 2017-09-09 |publisher = The Hollywood Reporter |language = en}}</ref>

== 前提 ==
故事改編自法蘭西斯·史考特·費茲傑羅的最後一本小說《最後的大亨》，小說在他去世後出版，本劇發生於1936年的荷里活電影製作。講述鬆散地根據著名的製片人[[歐文·托爾伯格|歐文·托爾伯格]]而成的Monroe Stahr，與他的老闆Pat Brady鬥爭。
