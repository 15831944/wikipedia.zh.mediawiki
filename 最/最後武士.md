《'''最後武士'''》（{{lang-en|'''''The Last Samurai'''''}}）是一部於2003年上映的[[電影|電影]]，主要描述[[日本|日本]]傳統的[[武士道|武士道]]精神，以1876年－1877年中的[[西南戰爭|西南戰爭]]和[[明治維新|明治維新]]作為背景。故事描述一個前[[美國|美國]]軍人，到日本助明治新政而組建的日本新軍做教官，在[[戰爭|戰爭]]途中被俘，慢慢被[[日本文化|日本傳統文化]]所吸引。片中起用了很多日本[[演員|演員]]，而工作人員大多是[[美國人|美國人]]（在主要製作群中，只有一個是[[日本人|日本人]]），而外景大多數在[[新西蘭|新西蘭]]所拍攝，所以這電影算是由美國、日本和新西蘭合作的電影。

== 劇情 ==
曾經參加過[[南北戰爭|南北戰爭]]及[[北美印第安戰爭|北美印第安戰爭]]的退伍軍人，納森·阿爾格倫[[上尉|上尉]]（CPT. Nathan Algren），由於在戰爭中經歷過太多的鮮血與暴力，特別是看到自己的長官開槍屠殺手無寸鐵的[[印第安|印第安]][[兒童|兒童]]時，他更感到作為軍人的榮譽感消失殆盡。在義勇與犧牲精神被新時代的功利與自私主義代替時，沉浸在往日光榮夢想中的阿爾格倫只能整日借[[酒|酒]]澆愁。

一個偶然的機會，他被推薦給一位[[日本政府|日本政府]]的大臣——大村松江（Omura），並被大村聘請為[[軍事|軍事]]顧問，去協助日本政府訓練[[明治維新|明治維新]]後第一支現代化軍隊。阿爾格倫便帶上自己的軍裝和行囊來到了[[日本|日本]]。

當時的日本正值[[明治維新|明治維新]]時期，整個國家正如飢似渴地吸收著[[西方文化|西方文化]]，日本的[[明治天皇|明治天皇]]致力要整個國家全盤西化，求取富強，徹底拋棄日本古老的傳統文化，其中包括廢除古老的[[武士道|武士道]]以及訓練現代化的軍隊。

然而，維新政府的政策遭到了大多數保守派[[日本武士|日本武士]]的反對，他們力爭保護武士精神，而不願成為現代化的殺人機器。並且他們認為使用西洋[[火槍|火槍]]對一個武士來說是一種恥辱。這些頑固的保守派武士們在武士集團的[[首領|首領]]——-勝元盛次（Katsumoto）的帶領下，隱居在深山裡的一個小村落。他們無時無刻不在尋找機會破壞新政府的西化運動。

終於有一天，在勝元帶領[[武士|武士]]們破壞了一段[[鐵路|鐵路]]後，阿爾格倫接到大村的命令，要求他帶領官軍一舉將武士集團徹底剿滅。無奈之下阿爾格倫只好帶領訓練不全的官軍軍隊，來到了武士村莊附近的一片[[森林|森林]]中。經過一場激戰後，缺乏經驗的官軍最終被擊敗，[[士兵|士兵]]們死的死、逃的逃，傷亡十分慘重。阿爾格倫也被武士團首領勝元[[戰俘|俘虜]]。出乎意料的是，勝元不但沒有[[處死|處死]]他，反而把他帶回了村子，並派自己的妹妹多香（Taka）照顧他，為他療傷。隨後阿爾格倫在與勝元的對話中得知，多香的丈夫在叢林戰鬥中戰死，而殺死多香丈夫的人就是自己。

冬天來臨，大雪封住村落向外的通路，身處在大雪封山的武士村落裡，與武士和村民們朝夕相處的阿爾格倫發現自己已不由自主地愛上了[[武士|武士]]文化，從[[武士道|武士道]]的精神與肉體修煉中，他似乎又找到了一個[[軍人|軍人]]應有的榮譽感。在與勝元一起生活中，阿爾格倫感到彼此之間已經產生了一種情同手足的默契。而勝元也對這個昔日的敵人惺惺相惜。同時，在多香的悉心照料下，似乎有某種微妙的感情在他們之間產生。然而，作為一個受命幫助維新政府訓練官軍的西洋[[教官|教官]]，阿爾格倫深深的明白自己和這個自己深愛的村落始終是敵對關係。

冬天結束，積雪化為雪水。維新政府自然不會放棄現代化進程，武士與官軍之間終有一戰。已經為武士道深深著迷的阿爾格倫，在軍人的職責與博愛和友情之間做出了自己最後的選擇。

== 分級 ==
由於劇中含有不少斬殺鏡頭，故而在美國列為 R 級 ( 建議 17 歲以上對象觀賞 )，日本不分級 ( 全年齡對象皆可觀賞 )，臺灣上映時為輔導級（未滿十二歲不得觀賞，十二歲以上未滿十八歲須由家長陪同觀賞）．發行影碟則降為保護級（未滿六歲不得觀賞，六歲以上未滿十二歲須由家長陪同觀賞）。

== 幕後花絮 ==
* 電影的佈景位在[[姬路市|姬路市]]的古剎和[[書寫山|書寫山]][[圓教寺|圓教寺]]，其他的戰爭和在村莊場景都在[[紐西蘭|紐西蘭]]拍攝。
* 在日本引起了不少話題，而當時的渡邊謙和真田廣之的演技也在國外被廣泛認同。
* 渡邊謙在2004年的[[奧斯卡金像獎|奧斯卡金像獎]]被提名[[奧斯卡最佳男配角獎|最佳男配角]]。
* 影片闡述勝元的軍隊為了保持日本的傳統，所以不會使用任何[[火器|火器]]。其實火器早在1543年便流傳至日本，在16世紀的[[戰國_(日本)|戰國時代]]中期已開始普遍使用[[火繩槍|火繩槍]]，雖然[[德川幕府|德川幕府]]之後又逐漸被限制或禁止使用，但在幕末[[戊辰戰爭|戊辰戰爭]]、[[明治|明治]]時代已再度變得普遍；並且史實的西南戰爭也沒有如電影般地以義和團式的冷兵器戰熱兵器，而也是與佐幕軍、政府軍等一樣採用近現代的火器對抗。
