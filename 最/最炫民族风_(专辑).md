《'''最炫民族风'''》是中国大陆音乐组合[[凤凰传奇|凤凰传奇]]的第三张音乐专辑，2009年5月27日由[[孔雀唱片|孔雀唱片]]发行，制作人是汤旭东、[[何沐阳|何沐阳]]，一共包含11首歌曲。<ref>{{cite web |url=http://news.163.com/09/0605/14/5B25LTNA000120GR.html |title=凤凰传奇新专辑《最炫民族风》发行 |accessdate=2009 |publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://music.yule.sohu.com/20090527/n264203706.shtml |title=凤凰传奇第三张专辑发行 引领民族流行最强音 |accessdate=2009 |publisher=搜狐网 |work= |language=zh-hans}}</ref>2014年7月，[[交响乐|交响乐]]版本的《最炫民族风》发行。<ref>{{cite web |url=http://yue.ifeng.com/neidiliuxing/detail_2014_07/03/37148985_0.shtml |title= 凤凰传奇《最炫民族风》交响乐版发布 首次尝试跨界|accessdate=2014 |publisher=凤凰网 |work= |language=zh-hans}}</ref>
