《'''最終兵器少女'''》（{{lang-ja|最終兵器彼女}}），是一部[[日本|日本]]的[[漫画|漫画]]和[[动画|动画]][[悲劇|悲劇]]作品，原作者為[[高橋真|高橋真]]。由於推出之後大受歡迎，所以於2005年8月至9月推出[[OVA|OVA]]版本《{{lang|en|Another love song}}》（另一個愛情故事）<ref>{{cite web|url=http://www.amazon.co.jp/dp/B0008JF67A/|title=最終兵器彼女 Another love song mission 1 スペシャル限定版（DVD）|publisher=[[Amazon.co.jp|Amazon.co.jp]]|accessdate=2014年10月21日|language=ja}}</ref><ref>{{cite web|url=http://www.amazon.co.jp/dp/B0009PIVS4/|title=最終兵器彼女 Another love song mission 2 (完) スペシャル限定版（DVD）|publisher=[[Amazon.co.jp|Amazon.co.jp]]|accessdate=2014年10月21日|language=ja}}</ref>，由[[前田亞季|前田亞季]]主演的真人[[电影|电影]]版於2006年1月28日在日本上映<ref>{{cite web|url=http://movie.walkerplus.com/mv35714/|title=最終兵器彼女|publisher=Walkerplus|accessdate=2014年10月21日|language=ja}}</ref>。《最終兵器少女 Another love song》獲選為2005年第九回[[日本|日本]][[文部省|文部省]][[文化廳媒體藝術祭|文化廳媒體藝術祭]]動畫部門推薦的作品。

== 劇情簡介 ==
故事的主角是一個[[北海道|北海道]]的[[學生|學生]]修次。他的同班同學千瀨在故事開始的時候就已向修次表白了。但因為千瀨的害羞性格和修次的不善辭令，所以他們就以寫[[交換日記|交換日記]]來開始他們的情侶關係。

一日，當修次和朋友到[[札幌|札幌]]購物的時候，不明來歷的[[轰炸机|轰炸机]]對札幌進行了猛烈的轟炸。修次和他的朋友找地方作掩護時，他見到了一個快速的小型飛行物體向敵軍的轟炸機進行攻擊。修次和他的朋友失散了之後，他在廢墟堆中徘徊。當他在廢墟堆當中見到增生了翅膀、武器、不再心跳且衣著破爛的千瀨之後，千瀨告訴他，她已經成為[[日本自卫队|日本自卫队]]的「最終兵器」，故事於是開始……

== 人物簡介 ==
; {{nihongo|修次|シュウジ||[[配音員|CV]]：[[石母田史朗|石母田史朗]]、演：[[窪塚俊介|窪塚俊介]]}}
: 男主角，高中生，千瀨的男朋友，希望還給千瀨正常的生活。千瀨暱稱他為「阿修」。

; {{nihongo|千瀨|ちせ||CV：[[折笠富美子|折笠富美子]]、演：[[前田亞季|前田亞季]]}}
: 女主角，高中生，修次的同班同學，修次的女朋友，做事有些莽撞糊塗。
: 軍隊在偶然的情況下找到身體最適合成為兵器的她改造成兵器，能力強大，發動千瀨之火時城市會被毀滅。

; {{nihongo|阿鐵|テツ||CV：[[三木真一郎|三木真一郎]]、演：[[涉川清彦|涉川清彦]]}}
: [[日本自衛隊|日本自衛隊]]隊長，冬美的[[丈夫|丈夫]]，將千瀨當[[人|人]]而非無感情的殺人兵器看待。

; {{nihongo|冬美|ふゆみ||CV：[[伊藤美紀|伊藤美紀]]、演：[[酒井美紀|酒井美紀]]}}
: 阿鐵的妻子，曾是修次的任課老師和所屬田徑社的指導老師，常被田徑社的學生暱稱「冬美學姐」。
: 由於阿鐵長期在部隊未歸，使冬美的寂寞感有增無減。

; {{nihongo|明美|アケミ||CV：[[杉本ゆう|杉本ゆう]]、演：[[貫地谷栞|貫地谷栞]]}}
: 高中生，修次的同班同學，千瀨的好友，很愛護千瀨，但亦暗戀著修次，淳志的女友。

; {{nihongo|淳志|アツシ||CV：[[白鳥哲|白鳥哲]]、演：[[木村了|木村了]]}}
: 高中生，修次的同班同學，喜歡明美。後來為了保護明美，與她告白後退學加入[[日本自卫队|自衛隊]]。因陰錯陽差的關係加入部隊後未與同梯次的士兵一樣見到千瀨而始終不知她與最終兵器的關係。在千瀨一次毀滅性攻擊中為撿回明美的相片而捲入爆炸中，生死未卜。

; {{nihongo|瑞希|ミヅキ||CV：[[折笠愛|折笠愛]]}}
: OVA登場角色，[[日本自衛隊|日本自衛隊]]中校。在一場戰役中失去了所有隊員和手腳各一，為繼續參戰同意做最終兵器開發的試作品。由於是國防大學畢業的正規軍人，戰鬥技巧比千瀨優秀，但因只是試作品所以性能極限不如千瀨。
