'''最高檢察署檢察總長'''，通常簡稱'''檢察總長'''，為[[最高檢察署|最高檢察署]]的首長，也是全國檢察體系的首長。

原稱'''檢察長'''，下級檢察署設首席檢察官，《[[s:法院組織法|法院組織法]]》於[[石明江|石明江]]任內修訂，改為最高檢設檢察總長，下級檢察署設[[檢察長|檢察長]]。

== 資格 ==
依《司法人員人事條例》第16條之規定，[[最高檢察署|最高檢察署]]檢察總長，應就具下列資格之一，並有領導才能者遴任之：
*曾任[[司法院大法官|司法院大法官]]、[[最高法院|最高法院]]院長、[[中華民國最高行政法院|最高行政法院]]院長或[[公務員懲戒委員會|公務員懲戒委員會]]委員長者。
*曾任[[最高法院|最高法院]]法官、[[最高檢察署|最高檢察署]]檢察官、[[高等法院|高等法院]]院長或[[高等檢察署|高等檢察署]]檢察長合計五年以上者。
*曾任[[簡任官|簡任]]法官、[[檢察官|檢察官]]十年以上，或任簡任法官、檢察官並任司法行政人員合計十年以上者。

== 任命與任期 ==
原本檢察總長依《法院組織法》第66條之規定，為[[特任官|特任官]]，實際上均由[[中華民國總統|總統]]任命，需[[立法院|立法院]]同意，有任期限制。

由於檢察總長為檢察體系的最高首長，原直接由總統任命，不需經[[立法院|立法院]]同意，又無任期限制。為了防止政治干預司法，有[[立法委員|立法委員]]遂提案要求修改檢察總長的提名任命方式以及建立任期制（法院組織法第66條修正）。後來在2006年1月13日立法院通過修正，明訂檢察總長由總統提名，經立法院同意後任命，任期4 年，且不得連任。而在檢察總長任滿前1 個月，總統應向立法院提出新任檢察總長人選。

但修正條文施行後，[[陳水扁|陳水扁]][[總統|總統]]第一次提名之檢察總長被提名人[[謝文定|謝文定]][[檢察官|檢察官]]即被立法院否決（2006年4月11日），也引發了修正案並未防止「政治干預司法」，只是從「行政干預」改成「立法干預」而已的爭議。2007年1月18日，[[臺灣高等檢察署高雄分署|高雄高分檢]]檢察長[[陳聰明|陳聰明]]在[[民進黨|民進黨]]與[[親民黨|親民黨]]力挺、[[國民黨|國民黨]]開放投票的情況下，獲得立法院過半數同意，成為首位由總統提名，經立法院同意後任命的檢察總長，開始檢察總長需民意機關同意任命的時代。

== 職掌 ==
檢察總長實際上即為最高檢察署首長，除原首長職務內容外，檢察總長最特別的即是其有提起[[非常上訴|非常上訴]]的權限。
