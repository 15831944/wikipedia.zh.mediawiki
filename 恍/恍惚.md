{{作品名稱|name=''Trance''|hk=催眠潛凶|tw=索命記憶|cn=迷幻}}是一部於[[2013年電影|2013年]]上映的[[英国|英国]]{{link-en|心理驚悚|Psychological thriller}}電影，由[[丹尼·博伊尔|丹尼·博伊尔]]执导，{{link-en|喬·阿亨|Joe Ahearne}}与{{link-en|約翰·霍奇|John Hodge (screenwriter)}}担任编剧。[[占士·麥艾禾|占士·麥艾禾]]、[[羅莎里奧·道森|羅莎里奧·道森]]和[[文森·卡索|文森·卡索]]主演<ref name="McAvoy-Dawson-Cassel" />。2013年3月27日在英国上映<ref>{{cite web |title=Danny Boyle's ''Trance'' Heads to UK Cinemas |url=http://www.blu-ray.com/news/?id=10242 |publisher=Blu-ray.com |accessdate=10 January 2013 |date=10 January 2013}}</ref>，美国于2013年4月5日上映<ref>{{cite web|last=Chitwood|first=Adam|title=Danny Boyle’s TRANCE Set for April 5th U.S. Release; New Poster Unveiled|url=http://collider.com/trance-release-date-poster-april-5/|publisher=Collider.com|accessdate=29 January 2013|date=29 January 2013}}</ref>。

== 剧情 ==
賽門（[[詹姆斯·麥艾維|詹姆斯·麥艾維]] 飾演）是一個藝術拍賣師，為了償還鉅額賭債，他和黑幫集團結盟，想要裡應外合竊取價值高達數千萬美元的藝術品。但在搶劫中他頭部受到重創，醒來後發現自己喪失記憶，完全不記得他將畫藏到哪裡了。

犯罪集團用盡心機想逼他說出藏畫地點，卻不得其門而入，集團首腦（[[文森·卡索|文森·卡索]] 飾演）請來催眠治療師（[[蘿莎瑞·道森|蘿莎瑞·道森]] 飾演），挖掘賽門黑暗的內心深處。她在賽門破碎的潛意識中越挖越深，風險也越來越大，遊走在慾望、現實、催眠暗示的虛實交界之處。
