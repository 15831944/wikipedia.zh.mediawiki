'''廷布'''（{{dz|d=ཐིམ་ཕུག།|t=Thimphu}}），又譯為'''辛布'''、'''亭孚'''、'''亭布'''等，是[[不丹王国|不丹王国]]的[[首都|首都]]。

== 概况 ==
廷布位于[[不丹|不丹]]西部，[[喜马拉雅山脉|喜马拉雅山脉]]中高处于2,400米的[[盆地|盆地]]之中，瀕臨旺河[[河谷|河谷]]（Wong Chu）<ref>{{cite web|url=http://tw.britannica.com/MiniSite/Article/id00062071.html|title=廷布}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；人口69,000人（2005年末）。1962年，正式成為不丹首都。<ref>Thimphu. (2008). In Encyclopædia Britannica. Retrieved March 31, 2008, from Encyclopædia Britannica Online: http://search.eb.com/eb/article-9072131</ref>

== 历史 ==
廷布最早建设于13世纪，不丹宗教之父[[帕久·竹贡西博|帕久·竹贡西博]]（Phajo Drugom Zhigpo）[[喇嘛|喇嘛]]在此处修建了小寺庙。1641年，[[沙布东|沙布东一世]][[阿旺·纳姆加尔|阿旺·纳姆加尔]]将寺庙改建为一座较大的城堡。

廷布原有的大部分建筑均毁于[[火灾|火灾]]和[[地震|地震]]，后经过多次重建，特别是1961年重新修缮[[札西秋宗|扎西却城堡]]之后，廷布才有了迅速的发展。

自1955年以来，廷布一直是[[不丹王国|不丹王国]]的[[政治|政治]]、[[经济|经济]]、[[军事|军事]]、[[文化|文化]]和[[宗教|宗教]]的中心。不丹的王宫、议会大厦、高等法院、陆军总部、警察总部、中央寺院、政府首脑机构和其他重要部门都坐落在廷布。

== 经济 ==
[[旅游业|旅游业]]和[[农业|农产业]]是[[廷布经济|廷布经济]]的重要部份；而近年来，[[食物加工|食物加工]]亦逐渐兴起。

== 交通 ==
在廷布西面有全国唯一的一个[[国际机场|国际机场]]：[[帕羅機場|帕羅機場]]，一条[[跑道|跑道]]服务[[不丹|不丹]]的国家航空公司：不丹航空公司 （Bhutan Airlines）和不丹皇家航空公司 (Druk Air).

== 城市趣闻 ==
廷布是世界上唯一没有[[交通灯|交通信号灯]]的国家首都。几年前，当地政府部门安装了一套[[交通灯|交通灯]]，但廷布居民抱怨说这些灯是“无人情味的机器”，没多久不得不拆除。
