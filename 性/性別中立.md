'''性別中立'''（{{lang-en|Gender neutrality}}）運動。一般是指推行去[[性別|性別]]化的[[語言|語言]]、社會[[政策|政策]]和其他社會[[機構|機構]]應該避免根據人們的性別來區分[[性別角色|性別角色]]的想法，以推動及促進[[性別平等|性別平等]]，避免因為認為一種性別比另一種性別的社會角色更適合而產生的歧視。但在推行方面，因社會普遍存在[[性別定型|性別定型]]，加上被指有關措施是[[政治正確|政治正確]]的舉措，在文化及社會意識上充滿爭議。

== 性別中的灰色區域 ==
{{main|生物性別和社會性別的區別}}
與性別中立有關的問題是性別中存在的灰色地帶。試圖在法律上界定性別界線已經被證明是一個困難的問題，因為存在[[性別認同|性別認同]]或被他人識別為[[雙性人|雙性人]]，[[X性別|X性別]]，[[變性人|變性人]]以及更普遍的[[非常規性別|非常規性別]]及性別非二元的人。

== 性別盲 ==
{{Main|性別盲}}
有些人自認为[[性别盲|性别盲]]的人在日常生活中采用性别中立的观点。

== 政策 ==
[[英國|英國]]80間公立中小學祭出「性别中立」的校服政策，未來將允許男學生穿著裙子，到學校裡上課，希望藉此保護「跨性別兒童」的權益。不過，仍有基督教團體持反對立場。根據新的校服規定，無論男孩或女孩，都能穿著感覺最舒適的服裝上課，這意味小至5歲的兒童，就能依據自己性別意識，像大人一樣「出櫃」，而且不會違反校規。此前，許多倡議已發出警告，當前的校服政策（只容許男生穿褲，女生穿裙）歧視[[LGBT|LGBT]]族群（[[女同性戀|女]]、[[男同性戀|男同性戀]]者、[[雙性戀|雙性戀]]者、與[[跨性別|跨性別]]者的統稱）。另外，英國私立學校委員會（Independent Schools Council）的秘書長羅賓森（Julie Robinson）也指出，其他公、私立學校很快也會跟進，而且部份學校也已經調整政策，希望能照顧「[[跨性別|跨性別]]學生」。但仍有保守的基督教團體表示，新規定會影響孩子的性別認同，兩性的界限已被逾越，相當令人憂心。報導指，伯明翰的艾倫斯學校（Allens Croft School）將是第一間採行「性別中立」校服政策的學校，服裝將不分男女性别，男生可以穿著灰色或黑色長裙，女生也可以身穿灰或黑色的長褲。<ref>[http://news.ltn.com.tw/news/world/breakingnews/1728415 英國80校力推「性别中立」 男生可穿裙上學] 2016-06-13 [[自由時報|自由時報]]</ref>

英國近來陸續有中學推行性別中立校服政策，即女生也一律穿褲代替穿裙，估計全國有大約四十間。由於部分中學有學生是變性人，為了推廣性別中立的平等原則，加上避免出現歧視，令不少學校推行這政策。然而，有學生家長對禁止穿校裙的規定表示不滿，認為沒有諮詢。他們表示，學校應容許女生選擇穿裙或穿褲，而非完全禁止穿校裙。<ref>[http://www.orientaldaily.on.cc/cnt/china_world/20180704/mobile/odn-20180704-0704_00180_025.html 推廣性別中立原則 英40中學禁穿裙] 2018-07-04 東方日報</ref>
