'''兴山县'''是[[中国|中国]][[湖北省|湖北省]][[宜昌市|宜昌市]]所辖的一个[[县|县]]。

== 历史 ==
始建于[[三国|三国]][[吴景帝|吴景帝]]永安三年（公元260年），距今已有1700多年历史。因“环邑皆山，县治兴建于群山之中”，故名兴山。兴山历史悠久，'''古代四大美女之一的[[王昭君|王昭君]]'''在此出生，它还是诗人[[屈原|屈原]]的家乡。

因[[三峡工程|三峡工程]]建设需要，兴山县城整体搬迁，向大山深处挺进了16公里，从[[香溪|香溪]]河东岸的有1700多年历史的县治[[高阳镇|高阳镇]]迁到了上游西岸[[古夫镇|古夫镇]]。

兴山新县城获得2003年度“中国人居环境范例奖”。

== 地理 ==
兴山县位于大巴山余脉，长江[[西陵峡|西陵峡]]北侧，东邻[[宜昌市|宜昌市]]、保康县，南连[[秭归县|秭归县]]，北靠[[神农架林区|神农架林区]]，西接[[巴东县|巴东县]]。

兴山县有丰富的自然资源。山上有茂密的森林和众多的土特产品，山下有丰富的矿产，山间河流蕴藏有充足的水能、山水林结合有奇特的自然风光。
总面积为2328平方公里版图内有大小山头3580座。

== 人口 ==
2004年人口为18万人。

== 行政区划 ==
兴山县辖古夫、昭君、峡口、[[南阳镇_(兴山县)|南阳]]、黄粮，水月寺6个镇，[[高桥乡_(兴山县)|高桥]]、榛子2个乡。
