'''兴凯湖'''或'''汗卡湖'''（[[俄语|俄语]]：'''{{lang|ru|О́зеро Ха́нка}}'''），是位于[[中国|中国]]和[[俄罗斯|俄罗斯]]边境的界湖，距中国[[黑龙江省|黑龙江省]][[密山市|密山市]]35公里。其面积4380<ref>{{Cite book|url=https://books.google.co.jp/books?id=wxUqDAAAQBAJ&pg=PT92&lpg=PT92&dq=%E5%85%B4%E5%87%AF%E6%B9%96+4380&source=bl&ots=WHhVQtN4LM&sig=T-Zynu2kC8k1AlpNTfJaYYizy60&hl=zh-CN&sa=X&ved=0ahUKEwj0zcuCjorUAhWLGZQKHRp5DJcQ6AEITDAN#v=onepage&q=%E5%85%B4%E5%87%AF%E6%B9%96%204380&f=false|last=董恒年|date=2014-09-01|publisher=XinXueTang|isbn=9787509412701|language=zh|chapter=美丽黑龙江（读酷高清插图版）}}</ref>平方公里，是全中國境內的第二大湖泊<ref>如只算整個均在中國境內的湖泊，第二大湖應為[[鄱陽湖|鄱陽湖]]。</ref>，仅次于[[青海省|青海省]]的[[青海湖|青海湖]]。該湖在俄罗斯境内水面约为3030平方公里（72%），而在中国境内水面1160平方公里（28%）。

== 水文和气候 ==
兴凯湖是[[乌苏里江|乌苏里江]]的主要源头之一，湖水通过乌苏里江注入[[黑龙江|黑龙江]]，流入[[鞑靼海峡|鞑靼海峡]]。整个湖泊被北部的一道宽约10米的沙丘分为大小两个部分。湖水平均深度为4.5米，最深处10.6米。平均容量为18.3立方公里，最大容量为22.6立方公里。湖岸以[[沼泽|沼泽]]为主，尤其是西北湖岸。

兴凯湖流域总面积16890平方公里，97%以上属俄罗斯领土，共有23条河流（中国8条，俄罗斯15条）流入兴凯湖。

全年降水量约为750毫米，主要集中在[[夏季|夏季]]，夏季平均气温[[摄氏|摄氏]]21度，冬季平均气温摄氏零下19度，有冰冻。

== 经济 ==
盛产中国四大淡水湖鱼之一的[[翘嘴鲌|翘嘴鲌]]（大白鱼）。由于主要产区位于中俄两国的界河，而对界河的生产未加限制，过度捕捞，尤其是中俄两国渔民大量捕捞体重100-200克的幼鱼，直接影响了资源的正常补充。

== 历史 ==
[[File:1903-Syberia-Amur.jpg|thumb]]
自[[汉代|汉代]]起就有東北的游牧民族的居住的紀錄，歷史上曾經為[[渤海國|渤海國]]（当时称为湄沱湖）、[[遼國|遼]]、[[金朝|金]]、[[元朝|元]]和[[清朝|清]]的領土，清代時當地屬[[吉林將軍|吉林將軍]]管轄地。兴凯湖在唐代称为湄沱湖，因湖形如“月琴”，故金代有“北琴海”之  称，清代改为兴凯湖。＂兴凯＂是[[满语|满语]]，意为＂水从高处往低处流＂之意。<ref name="兴凯湖名称来源">[http://www.lvyou114.com/youji/13/13658.html 山水旅游黄页]，2011-9-2查阅。</ref>

1858年[[清朝|中]][[俄罗斯帝国|俄]]雙方簽訂《[[瑷珲条约|瑷珲条约]]》，[[俄罗斯帝国|俄方]]取得該湖南面管理权，1860年《[[北京条约|北京条约]]》[[俄罗斯帝国|俄羅斯]]正式获得大部分湖的主权，之後該湖為[[中华人民共和国|中]][[俄罗斯联邦|俄]][[界湖|界湖]]至今。

== 自然保护 ==
中方所辖湖面为[[兴凯湖自然保护区|兴凯湖自然保护区]]的主要组成部分，是[[国际重要湿地|国际重要湿地]]之一。俄方所辖湖面也属于同名的[[自然保护区|自然保护区]]。[[兴凯湖松|兴凯湖松]]是兴凯湖特有的树种，主要分布于黑龙江省东南部兴凯湖区域，属国家二级保护渐危树种。<ref name="兴凯湖松">[http://heilongjiang.dbw.cn/system/2011/08/31/053369140.shtml 兴凯湖管委会强化措施保护湖松]，东北网。</ref>
