'''鄺廣傑[[退休|榮休]][[大主教|大主教]]'''{{GBSHK}}（The Most Revd Dr Peter K.K. Kwong, Archbishop Emeritus，{{bd|1936年}}），[[廣東|廣東]][[台山市|台山]]人，[[香港|香港]]宗教界人士，[[香港聖公會|香港聖公會]][[教省|教省]]榮休大主教。

== 簡歷 ==
鄺廣傑1936年生於香港，早年就讀[[聖保羅書院|聖保羅書院]]，1962年畢業於香港[[崇基學院|崇基學院]]中文系，獲文學[[文憑|文憑]]；1965年畢業於[[美國|美國]][[凱尼恩學院|堅仁大學]]([[Kenyon_College|Kenyon College]])神學系。1971年獲美國[[羅切斯特伯思理堂神學院|羅切斯特伯思理堂神學院]] (Bexley Hall,  Colgate Rochester)神學碩士學位。<ref name="cu071206">{{ cite web |url=https://www.cpr.cuhk.edu.hk/resources/press/pdf/071206citations_c.pdf |title=榮譽社會科學博士鄺廣傑大主教讚辭 |date= |publisher= |author=張洪年 }}</ref>

1965五年在聖公會封職，1966年冊封[[牧師|牧師]]。先後服務於聖公會聖雅各堂、聖保羅堂。1972年，鄺氏由聖公會委派加入[[崇基神學院|崇基神學院]]擔任副講師一職，任教基督教倫理學、宗教社會學、禮儀學及實習督導，同時兼任文林堂舍監， 亦曾任崇基學院[[校董|校董]]、崇基學院神學院校董（1981-1990）；並經常於[[胡仲揚|胡仲揚]]牧師離港期間署理校牧職務。1976年至1977年，鄺氏兼任[[崇基學院|崇基學院]]校牧。2007年6月獲委任為崇基學院榮譽院務委員。<ref name="cc65">{{cite book |url=https://www.ccc.cuhk.edu.hk/images/ChungChiPublication/ChungChi_65Book.pdf |title=善彌雅集-崇基學院六十五週年校慶特刊 |page=51 |publisher=崇基學院}}</ref>

1981年，鄺氏獲祝聖成為[[聖公會港澳教區|聖公會港澳教區]]第九任主教，成為首位華人主教。1998年，因[[聖公會港澳教區|聖公會港澳教區]]改組，香港教區升格為獨立的「香港聖公會[[教省|教省]]」，鄺氏獲選為香港聖公會教省首任[[大主教|大主教]]及教省[[主教長|主教長]]（Archbishop and Primate）、[[香港島教區|香港島教區]]首任主教。
<ref name="hku2000">{{ cite web |url=https://www4.hku.hk/hongrads/index.php/chi/archive/graduate_detail/223 |title=香港大學第159屆頒授典禮讚詞：鄺廣傑|date= |publisher= |author= }}</ref> 2007年榮休。

鄺廣傑是[[全國政協委員|全國政協委員]]，亦曾任[[香港基本法起草委員會|香港基本法起草委員會]]委員（1985-1989年，[[六四事件|六四事件]]後退出）、[[香港基本法諮詢委員會|香港基本法諮詢委員會]]委員、[[港事顧問|港事顧問]]（1992-1997年）、[[香港特別行政區籌備委員會|香港特別行政區籌備委員會]]委員、[[香港特別行政區第一屆政府推選委員會|香港特別行政區第一屆政府推選委員會]]委員。1998年起加入[[中國人民政治協商會議|中國人民政治協商會議]]。

== 家庭 ==
鄺廣傑夫人夏慧中女士，曾擔任[[聖公會林護紀念中學|聖公會林護紀念中學]][[校長|校長]]。膝下二女一子。

== 榮譽 ==
* 獲[[堅仁大學|堅仁大學]] ([[Kenyon_College|Kenyon College]]) 授予榮譽神學博士。<ref name="hku2000" />
* 獲[[羅切斯特伯思理堂|羅切斯特伯思理堂]][[神學院|神學院]] ([[Bexley_Hall|Bexley Hall]],  [[Colgate_Rochester|Colgate Rochester]]) 授予榮譽[[神學|神學]][[博士|博士]]。<ref name="hku2000" />
* 2000年，獲[[香港大學|香港大學]]授予[[名譽|名譽]]神學博士。<ref name="hku2000" />
* 2007年，獲[[香港特區政府|香港特區政府]]授予[[金紫荊星章|金紫荊星章]]。<ref>{{ cite web |url=http://www.info.gov.hk/gia/general/200707/01/P200706300169.htm |title=二○○七年授勳名單 |date=2007-07-01 |publisher= |author= }}</ref>
* 2007年，獲[[香港中文大學|香港中文大學]]授予榮譽[[社會科學|社會科學]]博士。<ref name="cu071206" />
