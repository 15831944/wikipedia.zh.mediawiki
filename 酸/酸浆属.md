'''酸浆属'''（''Physalis''），是[[茄科|茄科]]下的一個屬，分布在在世界各地温带和亚热带地区。此属的特点是果实大小与小橙子相似的，形状和结构类似小番茄,但它部分或完全地被封闭在一个大的由花萼形成的纸质外皮内。

它是[[草本植物|草本植物]]，成体高度为0.4-3米，类似于普通[[番茄|番茄]]，但通常有更硬更直的茎；一年生或多年生。耐高温，需要充足阳光、温暖的气候，有些[[种|种]]对霜冻敏感，但其他種如[[酸浆|酸浆]]（''P. alkekengi''），会在冬天休眠以忍受严寒。

== 养殖和用途 ==
[[File:Physalis.jpg|left]]
这种植物可以生长在大多数土壤中，也很适应生长在贫瘠的土壤和盆中。在除了果实成熟期以外的全年生长中都需要大量的水。并且它会感染大多数番茄所感染的疾病和害虫。它依靠种子繁殖。

它的用途与普通的番茄或其他水果相似。将其从外壳中取出后，就可以直接食用或者制成沙拉、甜点、果酱、果冻、调味品。它们也可以像无花果、杏、葡萄一样被风干吃。 它也常被作为治疗嗓子痛的草药。''[[Physalis_peruviana|Physalis peruviana]]''（也称作“[[Cape_Gooseberry|Cape Gooseberry]]”）的花是唯一的一个与番茄或菠萝相似的混合物。它生长在南美北美以及美洲中部，但是它主要生长在包括南非的亚热带地区。另一个最常见的类型是[[黏果酸漿|黏果酸漿]]（tomatillo，''P. philadelphica''）。

一些[[种|种]]也被种植作为[[观赏植物|观赏植物]]。比如，[[红姑娘|红姑娘]]（''Physalis alkekengi''）是因它的大的、明亮的橘黄色至红色的外壳而著名。

== 种类 ==
[[File:Illustration_Physalis_alkekengi0.jpg|right]]
* [[酸浆|酸浆]]（''Physalis alkekengi'' L.）红姑娘、挂金灯、锦灯笼、金灯、泡泡草等 
* [[苦蘵|苦蘵]]（''Physalis angulata'' L.）炮仔燈、天泡子、天泡草、黃姑娘、小酸漿、樸樸草、打額泡。
* [[棱萼酸浆|棱萼酸浆]]（''Physalis cordata'' P. Mill.）
* [[毛苦蘵|毛苦蘵]]（''Physalis minima'' L.） 
* [[灯笼果|灯笼果]]（''Physalis peruviana'' L.） 秘鲁苦蘵
* [[毛酸漿|毛酸漿]]（''Physalis pubescens'' L.）洋姑娘、甜姑娘、戈力 
* [[黏果酸漿|黏果酸漿]]（''Physalis philadelphica'' L.），又稱墨西哥酸漿，為綠色[[莎莎醬|莎莎醬]]等原料
