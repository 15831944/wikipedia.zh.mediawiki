{{engname|《贤妻》|Good Wife}}，2013年上映于[[中国|中国]][[湖南卫视|湖南卫视]][[金鹰独播剧场|金鹰独播剧场]]的爱情剧，导演林添一，主演刘涛、娄淇、保剑锋、洪小鈴、谢紫彬。[[加拿大|加拿大]][[城市电视|城市电视]]2高清台2014年5月29日起逢星期一至五西岸时间5:00 AM,12:00 PM及7:30 PM播放。

== 剧情 ==
该剧讲述中国大陆现代[[爱情|爱情]]和[[家庭|家庭]]生活，诠释女性守卫[[婚姻|婚姻]]和家庭。<ref>{{cite web |url=http://ent.163.com/13/0313/11/8PRFRIVK00031GVS.html |title=《贤妻》热播近尾声 高仁情感"狙击"保剑锋 |accessdate=2013 |publisher=网易 |work= |language=zh-cn}}</ref><ref>{{cite web |url=http://fashion.ifeng.com/health/detail_2013_03/27/23568819_0.shtml |title=电视剧《贤妻》收官 刘涛诠释完美女性 |accessdate=2013 |publisher=凤凰网 |work= |language=zh-cn}}</ref>
