在[[数学|数学]]之中，'''有限域算术'''是一种在[[有限域|有限域]]之内的[[算术|算术]]，因为[[域_(數學)|域]]仅包括有限数量的[[元素_(數學)|元素]]，而有限域算术则相对于无限域算术，后者是包括无限数量的元素的算术（如在[[有理数|有理数]]之下的算术）。

由于并没有任何有限域是无限的，因此存在着无限多个不同的有限域。它们的[[势_(数学)|势]]需要是能够在''p<sup>n</sup>''的形式下，这其中的''p''是一则[[素数|素数]]，而''n''则是一则[[自然数|正整数]]，同时两个持有等量的有限域可以构成[[同构|同构]]。素数''p''被称之为有限域的[[特征_(代数)|特征]]，而正整数''n''则被称之为有限域的[[向量空间的维数|向量空间的维数]]，凌驾于它的[[prime_field|最初域]]之上，最初域为最小的包括1F的[[域_(數學)|子域]]。

有限域应用于各种领域，这其中包括在[[分組碼|线性分组码]]之内的[[编码理论|编码理论]]，譬如[[BCH码|BCH码]]和[[里德-所罗门码|里德-所罗门码]]，还有在[[密码学|密码学]]之中的[[算法|演算法]]，比如[[高级加密标准|Rijndael加密法]]之下的加密算法。

==实效多项式表示==
持有''p<sup>n</sup>''元素的有限域可由GF(''p<sup>n</sup>'')（同时可被<math>\mathbb{F}_{p^n}</math>, {{math|'''F'''<sub>''p<sup>n</sup>''</sub>}}}）所表示，被命名为'''[[有限域|伽罗瓦域]]''', 以荣誉有限域学说的创始者[[埃瓦里斯特·伽罗瓦|埃瓦里斯特·伽罗瓦]]. GF(''p'')（同时可表示为{{math|'''Z'''/''p'''''Z'''}}, <math>\mathbb{F}_{p}</math>, 或{{math|'''F'''<sub>''p''</sub>}}), 其中''p''为一则素数, GF(''p'')简明称之为就是，一个[[环_(代数)|环]]整数群[[模除|模除]] ''p''之后，而得到的结果. 这表明是,可以在整数范围中，先行执行常规运算(加法,减法,乘法), 然后模除''p''而以简化，举例说明, 在GF(5)内, {{nowrap|1=4 + 3 = 7}}被简化为2，因为是7模除5的结果。 除法则是在模除''p''的倒数上实施乘法, 并且可以选择使用[[扩展欧几里得算法|扩展欧几里得算法]]来计算。

一则特别的范例就是GF(2), 其加法为[[异或门|XOR]],而乘法是[[与门|AND]]. 由于唯一具有[[倒数|倒数]]的元素是数字1，除法则是[[恆等函數|恆等函數]]。

GF(''p<sup>n</sup>'')的元素可表示为，在GF(''p'')之上严格小于''n''[[冪|次数]]的[[多項式|多项式]]。运算则实行在先模除''R''，而''R''是一则在GF(''p'')之上，拥有''n''次数的[[不可约多项式|不可约多项式]], 例如运用[[多项式长除法|多项式长除法]]。两则多项式''P''和''Q''则按常规运算；乘法则按如下进行: 先按常规计算{{nowrap|1=''W'' = ''P''⋅''Q''}}, 然后计算模除''R''之后的余项(存在有更方便方法）。

当素数是2时，一般按常规可以把GF(''p<sup>n</sup>'')的元素表示为[[二进制|二进制数]], 按对应元素的二进制表示，多项式的每一项表示为一[[比特|比特]]的，相对应元素的二进制数位，并且括号( "{"和"}" )或类似的分隔符也普遍附加于这些二进制数，或对应它们的[[十六进制|十六进制]]的同等数，以表明数值确确是域内的一则元素。举例说明, 下列数都在具有2的特征下持有相同的数值。

;多项式<nowiki>:  </nowiki> ''x''<sup>6</sup> + ''x''<sup>4</sup> + ''x'' + 1
;二进制<nowiki>:      </nowiki>{01010011}
;十六进制<nowiki>: </nowiki>{53}

==加法和减法==
[[加法|加法]]和[[减法|减法]]可实施在加与减这两则[[多项式|多项式]]，再而使用[[模除|模除]]其[[特征_(代数)|特征值]]以简化。

在一则特征值为2的有限域之中，加法模2，减法模2，如同使用[[异或门|XOR]]. 因此，

;多项式<nowiki>:  </nowiki> (''x''<sup>6</sup> + ''x''<sup>4</sup> + ''x'' + 1) + (''x''<sup>7</sup> + ''x''<sup>6</sup> + ''x''<sup>3</sup> + ''x'') = ''x''<sup>7</sup> + ''x''<sup>4</sup> + ''x''<sup>3</sup> + 1
;二进制<nowiki>:      </nowiki>	{01010011} + {11001010} = {10011001}
;十六进制<nowiki>: </nowiki> {53} + {CA} = {99}

在常规的无限域的多项式的加法下，计算之和需要包含单项 2''x''<sup>6</sup>. 但在有限域的加法下， 0''x''<sup>6</sup>则被去掉，因为其计算结果被模2所消除。

在计算机科学的诸多应用程序之中，特征值为2的有限域运算被简单化，称之为GF(2<sup>''n''</sup>) [[伽罗瓦域|伽罗瓦域]], 使的这些领域在应用程序中，体现出一种特别大众化的选择。

==乘法==

乘法是在有限域之内，把乘积模除于，一则用来表示有限域的，简约过的[[不可约多项式|不可约多项式]]。 (换句话说, 乘法再跟上使用，简约了的多项式充当除数的除法，然后余数则是它们的乘积。) 符号 "•" 可以用作于在有限域之内的乘法。

=== Rijndael有限域 ===
[[高级加密标准|Rijndael]]（Rijndael的發音近於"Rhine doll")使用包含256个元素的持有特征值是2的有限域， 同时可被称之为伽罗瓦域'''GF'''(2<sup>8</sup>). 在乘法上它依赖下列简约多项式。

:''x''<sup>8</sup> + ''x''<sup>4</sup> + ''x''<sup>3</sup> + ''x'' + 1.

例如, {53} • {CA} = {01} 因为在Rijndael域中

:    (''x''<sup>6</sup> + ''x''<sup>4</sup> + ''x'' + 1)(''x''<sup>7</sup> + ''x''<sup>6</sup> + ''x''<sup>3</sup> + ''x'')

: = (''x''<sup>13</sup> + ''x''<sup>12</sup> + ''x''<sup>9</sup> + '''x<sup>7</sup>''') + (''x''<sup>11</sup> + ''x''<sup>10</sup> + '''x<sup>7</sup>''' + ''x''<sup>5</sup>) + (''x''<sup>8</sup> + '''x<sup>7</sup>''' + ''x''<sup>4</sup> + ''x''<sup>2</sup>) + ('''x<sup>7</sup>''' + ''x''<sup>6</sup> + ''x''<sup>3</sup> + ''x'')

: = ''x''<sup>13</sup> + ''x''<sup>12</sup> + ''x''<sup>9</sup> + ''x''<sup>11</sup> + ''x''<sup>10</sup> + ''x''<sup>5</sup> + ''x''<sup>8</sup> + ''x''<sup>4</sup> + ''x''<sup>2</sup> + ''x''<sup>6</sup> + ''x''<sup>3</sup> + ''x''

: = ''x''<sup>13</sup> + ''x''<sup>12</sup> + ''x''<sup>11</sup> + ''x''<sup>10</sup> + ''x''<sup>9</sup> + ''x''<sup>8</sup> + ''x''<sup>6</sup> + ''x''<sup>5</sup> + ''x''<sup>4</sup> + ''x''<sup>3</sup> + ''x''<sup>2</sup> + ''x''
