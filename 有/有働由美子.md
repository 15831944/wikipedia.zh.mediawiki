'''有働由美子'''（{{bd|1969年|3月22|||Udō Yumiko}}）是[[日本|日本]]女性[[新聞|新聞]][[播報員|播報員]]，所屬經紀公司為{{link-ja|NATURAL EIGHT|ナチュラルエイト}}。出生於[[鹿兒島縣|鹿兒島縣]]，但在[[大阪府|大阪府]]成長；畢業於{{link-ja|神戶女學院大學|神戸女学院大学}}，1991年起加入[[NHK|NHK]]，當中最為人熟識是多次出任NHK的除夕節目《[[NHK紅白歌合戰|NHK紅白歌合戰]]》的主持人，曾負責[[NHK綜合頻道|NHK綜合頻道]]的生活雜誌式節目《{{link-ja|朝一|あさイチ}}》主持、及為[[NHK_BS_Premium|NHK BS Premium]]的《{{link-ja|世界瞬間最美|世界で一番美しい瞬間}}》擔任旁述。2018年3月，从NHK辞职，成為{{link-ja|自由播報員|フリーアナウンサー}}，並自該年10月起成為[[日本電視台|日本電視台]]深夜新聞節目《[[news_zero|news zero]]》主持人。

==簡歷==
===成長時期===
出身於鹿兒島縣，父母均來自熊本縣<ref>本人曾於NHK的節目《{{link-ja|Studio Park LIVE talk show|スタジオパークからこんにちは}}》中談及過。</ref>，年少時搬到大阪府生活，並在大阪接受教育，在大阪府立北野高等學校畢業後，原希望進入[[大阪大學|大阪大學]]但失敗，最後改進入{{link-ja|神戶女學院大學|神戸女学院大学}}，1991年加入NHK當播報員。

興趣有閱讀、旅行、烹飪，並擁有[[劍道|劍道]]2段資格。中學及高校時代是劍道部成員。

===工作經歷===
入局後最初在大阪放送局負責報導體育新聞。入局4年後，被委任擔任《NHK新聞—日本早晨》，開始於東京工作。1995年[[關西地區|關西地區]]發生了[[阪神大地震|阪神大地震]]，作為在關西地區成長的有働由美子被派往受災地進行採訪，而憑著她的一口流利的[[關西腔|關西腔]]，她的採訪片段在全國新聞中播出，也是讓她漸漸受到了重視的重要因素。及後更獲委派負責[[奧運會|奧運會]]採訪、《NHK十點新聞》等的重要新聞節目。

2001年，首次擔任《NHK紅白歌合戰》紅組主持，隨後2年也是繼續由她擔任；2006年4月起，伙拍{{link-ja|小川浩司|小川浩司}}擔任《{{link-ja|Studio Park LIVE talk show|スタジオパークからこんにちは}}》主播，但不久小川浩司因健康理由而缺席數週的節目，有働只能一個人獨自支撐，但亦再一次印證她的主播能力並確立其主持風格。2007年6月，完成了一年的《Studio Park LIVE talk show》主持身份後（此節目的主持人通常以一年左右為任期），被調往[[美國|美國]]分局（[[紐約|紐約]]）任職，2008年6月獲昇任進入管理層<ref>NHK有働アナ、管理職昇進，[[體育報知|體育報知]]，2008年6月7日。</ref>，2010年3月29日回日，隨即出任新節目《朝一》的主持至今。2012年起連續四年再次參與《NHK紅白歌合戰》，但改為綜合主持。

2013年2月1日獲邀於[[日本電視台|日本電視台]]的節目《PON!》中出演，成為了首位女性在民營電視台中出演的NHK主播<ref>{{Cite web |date=2013-02-01|url=http://www.daily.co.jp/newsflash/gossip/2013/02/01/0005710737.shtml|title=ＮＨＫ有働アナ、日テレで民放初出演|publisher=デイリースポーツ|accessdate=2013-02-02}}</ref>。

2018年3月31日，离开供职长达28年的NHK。<ref>{{cite web|url=https://mainichi.jp/articles/20180404/spn/00m/200/004000c|title=有働由美子さん電撃退局　屈指の実力人気アナ、今後はジャーナリストの道へ|publisher=每日新闻}}</ref>
