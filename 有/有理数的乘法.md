对于任意的[[有理数|有理数]]a和b，必有唯一的[[有理数|有理数]]等于a及b的'''乘积'''，记为<math>a \cdot b</math>或<math>ab</math>。

== 性质 ==
# [[交换律|交换律]]：<math>ab=ba</math>
# [[结合律|结合律]]：<math>(ab)c=a(bc)</math>
# <math>a \cdot 1 = a</math>
# 对于任意不等于0的[[有理数|有理数]]a，必存在a的[[倒数|倒数]]（记为<math>\frac{1}{a}</math>），使<math>a \cdot \frac{1}{a} = 1</math>
# [[分配律|分配律]]：<math>\left ( a + b \right ) \cdot c = a \cdot c + b \cdot c</math>
# 若<math>a>b</math>，<math>c>0</math>，则<math>a \cdot c > b \cdot c</math>

注：上述性质可作为[[有理数的基本性质|有理数的基本性质]]而不加证明。

== 推论 ==
根据上述性质，可以得到一些有用的推论：
# 若[[有理数|有理数]]a的倒数存在，则其[[倒数|倒数]]必唯一
# 若a为[[有理数|有理数]]且<math>a \neq 0</math>，则a与<math>\frac{1}{a}</math>互为[[倒数|倒数]]
# [[分配律|分配律]]：<math>\left ( a - b \right ) \cdot c = a \cdot c - b \cdot c</math>
# <math>a \cdot 0 = 0 \cdot a = 0</math>
# 若<math>a \cdot b = 0</math>且<math>b \neq 0</math>，则<math>a=0</math>
# <math>\left ( -a \right ) \cdot b = - \left ( a \cdot b \right )</math>
# 若<math>a>0</math>，<math>b>0</math>，则<math>a \cdot b > 0</math>
# 若<math>a>0</math>，<math>b<0</math>，则<math>a \cdot b < 0</math>
# 若<math>a<0</math>，<math>b<0</math>，则<math>a \cdot b > 0</math>
# 若<math>a>b</math>，则<math>a > \frac{a+b}{2} > b</math>
