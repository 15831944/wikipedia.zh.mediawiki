'''有甲目'''（學名：{{lang|la |Cingulata}}），又名'''[[貧齒目_(消歧義)|貧齒目]]'''，是[[異關節總目|異關節總目]]下兩個目的其中之一。有甲目代表了[[美洲大陸|美洲大陸]]上{{link-en|有甲的 |armour (zoology)}}[[胎盤|胎盤類動物]]。[[犰狳科|犰狳科]]（[[Dasypodidae|Dasypodidae]]）和[[倭犰狳科|倭犰狳科]]（[[Chlamyphoridae|Chlamyphoridae]]）是本目現時仍有現生種的兩個[[科_(生物)|科]]<ref name = MSW3>{{MSW3| id = 11700001}}</ref>。
另外還有兩個已滅絕的科的物種，與現存最大的犰狳——[[大犰狳|大犰狳]]的{{Convert|45|kg|abbr=on |sigfig=1}}相比，仍要巨大得多<ref>[http://www.faunaparaguay.com/mamm6Priodontesmaximus.pdf Giant Armadillo ''Priodontes maximus'' (Kerr, 1792)].) FaunaParaguay.com</ref>。

==描述==
有甲目物種的背部有細小的、呈鱗片狀、[[角質|角質化]]的重疊表皮，稱為“鱗甲”（scute），是由[[皮膚骨化|皮膚骨化]]（osteoderm）形成，並成為其獨有的背甲。大多數物種在肩部及臀部亦有堅實的護盾，並有三到九條以具彈性皮膚分隔開的鱗甲帶覆蓋其側面和底部。<ref name="EoM">{{cite book |editor=Macdonald, D. |author=Dickman, Christopher R. |year=1984 |title=The Encyclopedia of Mammals |publisher=Facts on File |location=New York |pages=781–783 |isbn=0-87196-871-1}}</ref>。

潘帕獸的甲殼仍有一定的活動空間，這是由於甲殼之間仍有三條可動的橫向皮膚骨化帶。<ref name = "Vizcaíno"/>但與犰狳不同，這些帶是由單一的鱗甲所覆蓋，而不是像犰狳般是由多於一條鱗甲帶所連接。<ref name = "Vizcaíno"/>雕齒獸的背甲像龜的甲殼般，是由已融合的骨板所組成，完全沒有可動的鱗甲帶。

潘帕獸和犰狳均有鱗甲覆蓋於其頭部之上。雕齒獸則擁有被鱗甲厚實包覆的尾部，甚至如[[星尾獸|星尾獸]]在尾巴的末端擁有尾槌，就如同[[恐龍|恐龍]]中的[[甲龍亞目|甲龍亞目]]。證據顯示這些尾槌可以用來防禦或是雄性間競爭。<ref name="Blanco2009" />

許多種犰狳以[[昆蟲|昆蟲]]或其他無脊椎動物為食；部分種類為[[雜食性|雜食性]]，以小型脊椎動物與植物為食。潘帕獸則認為是以啃食地面植披為主<ref name = "Vizcaíno"/>而同位素分析顯示雕齒獸是以[[C4類二氧化碳固定|C4]]類[[禾本科|禾本科]]為主食。<ref name="Pérez-CrespoArroyo-Cabrales2011">{{cite journal|last1=Pérez-Crespo|first1=V. A.|last2=Arroyo-Cabrales|first2=J.|last3=Alva-Valdivia|first3=L. M.|last4=Morales-Puente|first4=P.|last5=Cienfuegos-Alvarado|first5=E.|title=Diet and habitat definitions for Mexican glyptodonts from Cedral (San Luis Potosí, México) based on stable isotope analysis|journal=Geological Magazine|volume=149|issue=01|date=2011-10-18|pages=153–157|issn=0016-7568|doi=10.1017/S0016756811000951}}</ref>另外[[六帶犰狳亞科|六帶犰狳亞科]]中已滅絕的[[巨六帶犰狳屬|巨六帶犰狳屬]]較為特別，為特化的[[肉食動物|肉食動物]]。
