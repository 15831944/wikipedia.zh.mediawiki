'''有爪动物'''（[[學名|學名]]：{{lang|la|Onychophora}}）是[[蜕皮动物|蜕皮动物]]（Ecdysozoa）中的一[[门_(生物)|门]]，经常被简称为''有腿的虫''，种类极少，包括少数[[化石|化石]]种类在内，共约90种。

有爪动物门的[[动物|动物]]极有可能和[[节肢动物|节肢动物]]（Arthropoda）和[[缓步动物门|缓步动物门]][[动物|动物]]（Tardigrada）是[[近亲|近亲]]，牠们共同构成[[泛節肢動物|泛節肢動物]]（Panarthropoda）。牠们的[[祖先|祖先]]可能是[[寒武纪|寒武纪]][[叶足动物|叶足动物]]。

== 歷史 ==
在现代[[动物学|动物学]]中，牠们奇特的[[繁殖|繁殖]]方式引起了很多[[科学家|科学家]]的兴趣。1825年，[[Landsdown_Guilding|Landsdown Guilding]]以[[科学|科学]]的方式描述了该门的第一个[[属|属]]的动物，他当时认为这种动物是[[腹足纲|腹足纲]]（Gastropoda）动物，如[[蜗牛|蜗牛]]的[[变种|变种]]。这种[[理论|理论]]的根据是两者皆蠕虫样，有简单[[脑部|脑部]]、[[眼部|眼部]]结构，有[[体节|体节]]，成对的[[排泄|排泄]]器官。但这些并不为最新的分类法所接受。後來「有爪蟲綱」被獨立成為「有爪動物門」。「有爪动物」（Onychophora）一词出现于1853年。

== 分布 ==
全部产于[[热带|热带]]，[[非洲|非洲]]中部、[[马来半岛|马来半岛]]、[[美洲|美洲]]中部均有分布，主要在[[南半球|南半球]]分布。

== 习性 ==
有爪动物门的[[动物|动物]]陆栖、夜行、肉食性，靠捕食小动物例如[[昆虫|昆虫]]生活。牠们身体内有[[腺体|腺体]]，会喷出一些粘稠的液体去粘住猎物然后[[进食|进食]]。

== 特征 ==
体呈[[蠕虫|蠕虫]]形，不分节。雌雄异体，多为[[胎生|胎生]]，直接[[发育|发育]]。形态介于[[环节动物|环节动物]]和[[节肢动物|节肢动物]]之间，兼具这两门动物的特征。
