'''有马赖徸'''（{{jpn|j=有馬 頼徸{{efn|因为“徸”字生僻以及技术限制，日文中经常以平假名书写，使得人名变成“{{lang|ja|有馬頼ゆき}}”}}|hg=ありま よりゆき|rm=Arima Yoriyuki}}，{{bd|1714年|12月31日|1783年|12月16日}}）是日本[[江户时代|江户时代]]的一位[[大名_(称谓)|大名]]，也是一位[[数学家|数学家]]<ref name="yamamoto"/>{{rp|20}}。他是[[筑后国|筑后国]][[久留米藩|久留米藩]]第7代藩主、久留米有马家第8代当主。

有马赖徸作为数学家，修习[[关孝和|关流算术]]，以笔名'''丰田文景'''著有代表同时代日本最高算学水准的《拾玑算法》，以“算学大名”著称于世。

== 生平 ==
有马赖徸生于[[正德_(日本)|正德]]4年（1714年）12月31日，是第6代藩主{{link-ja|有马则维|有馬則維|则维}}的四男<ref name="nakae"/>（一说五男<ref name="koudan"/>）。

[[享保|享保]]4年（1729年），其父因为领内发生一揆，且与[[家老|家老]]对立而引咎[[隱居_(家制度)|隐居]]<ref name="koga"/>，由于其3位兄长都已夭折<ref name="nakae"/>，有马赖徸在16岁时继任藩主。不过他在[[元文_(日本)|元文]]2年（1737年）才亲政。他亲政这一年，适逢天灾，百姓[[饥荒|饥馑]]<ref name="kurume"/>。赖徸为了救济领民，发放了救济金和救济粮；为了广泛采纳意见，仿效将军[[德川吉宗|德川吉宗]]设置了{{link-ja|目安箱}}（意见箱）；为了向民众提供娱乐，他组织了[[猿乐|猿乐]]的演出。

尽管有马赖徸努力施行善政，由于饥荒发生于九州各藩，各地[[一揆|百姓一揆]]频发，久留米有马藩也未能幸免。赖徸不仅处死一揆首谋者，也令家老稻次因幡承担责任，令其[[蛰居|蛰居]]<ref name="inatsugu"/>。

[[宝历_(日本)|宝历]]4年（1754年）2月21日，由于征收“人别银”（按人头摊派的税赋），久留米藩内发生[[一揆|一揆]]，其时有马赖徸赴江户[[参勤交代|参勤交代]]，最终在4月由家臣平息了骚乱。<ref name="koga"/>

有马赖徸致力于学术事业，聘任了不少文化人，但另一方面对他也有课税过重花费过多的批评。<ref name="kurumenmon"/>

有马赖徸死于1783年（[[天明_(日本)|天明]]3年）12月16日，享年70。家主由其长子{{link-ja|有马赖贵|有馬頼貴}}继任。

== 数学事业 ==
[[File:First_page_of_拾璣算法.jpg|thumb]]
有马赖徸博学多才，不仅长于法令和[[有职故实|故实]]（礼法），更是一位[[和算|和算家]]，师从[[关孝和|关流]]和算家{{link-ja|山路主住}}。<ref name="nakae"/>

他在32岁时编著了第一本和算书《初学天元门》，最后一本著作写于53岁时，一生著有算学书40余册。其中最著名的是他以笔名丰田文景发表的《拾玑算法》5卷，其中收录算法问题150道<ref name="endo"/>{{rp|341}}，这也是他唯一刊行的著作<ref name="heibon"/>。该书不仅代表了当时日本数学的最高水准，也第一次将原本私下传授的关流算学介绍给广大民众。<ref name="fujiwara"/>{{rp|91,92}}例如在此之前[[圆周率|圆周率]]的值经[[关孝和|关孝和]]计算，精确到12位数，而有马赖徸计算到了50位，其中30位数值（小数点后29位）正确：<ref name="endo"/>{{rp|346}}

:<math>\pi\approx\frac{428224593349304}{136308121570117}=3.14159265358979323846264338327(569...)</math>{{r|approx|Hardy}}

在他之前{{link-ja|松永良弼}}已经算出此值，但写于秘传书中，不像《拾玑算法》是公开刊行的书籍。在循环小数的计算上有马赖徸也有划时代的成就。<ref name="endo"/>{{rp|346}}

有马赖徸不仅自己从事数学研究，作为大名他也积极支持数学研究事业，例如他招纳关流数学家{{link-ja|藤田贞资|藤田貞資}}、入江修敬<ref name="nakayama-irie"/>为家臣，并资助了村井中渐。<ref name="fujiwara"/>{{rp|91,92}}<ref name="heibon"/>在和算诸流派中，关流的一枝独秀受益于他的资助。<ref name="shougaku"/>

== 评价 ==
有马赖徸的才能获得幕府赏识，4次被任命为[[江户|江户]][[增上寺|增上寺]]的御火消役（一年一度任命的荣誉职务）。有马家历代官位为侍从，但他破格被任命为从四位左近卫少将<ref name="nakae"/>。他曾经3次获得“国鹤下赐”，即将军所赐予的亲自猎获的鹤肉，这是通常是大大名[[德川御三家|德川御三家]]、[[伊达家|伊达家]]、[[岛津家|岛津家]]、[[前田家|前田家]]才能享受的殊荣。

有马赖徸治政长达54年，加之他优秀的政治才能，久留米藩一直较为安定。他因此被称为“久留米藩的（德川）[[德川吉宗|吉宗]]”。他由于教养出众，与同时代的[[新发田藩|新发田藩]]主{{link-ja|沟口直温|溝口直温}}和[[松江藩|松江藩]]主{{link-ja|松平宗衍}}并称“风流三大名”<ref name="nakae"/>。

为了表彰他在数学上的功绩，1911年明治政府追赠有马赖徸从三位的官位。<ref name="kurumenmon"/>
