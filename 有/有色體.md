'''有色體'''（{{lang-en|'''Chromoplast'''}}，又稱'''雜色體'''），是[[植物|植物]]、[[藻類|藻類]]等能行光合作用的真核[[細胞|細胞]]中的一種[[質粒體|質粒體]]<ref>{{cite web|url=http://www.iams.sinica.edu.tw/lab/wbtzeng/terms_biochem.htm|title=生物化學和生物物理相關名詞與意義|publisher=中央研究院 生化分子質譜與光譜學實驗室|language=zh-tw}}</ref>，通稱含有色素的質粒體，常和沒有色素的[[白色體|白色體]]比較。有色體因含有[[胡蘿蔔素|胡蘿蔔素]]（Carotene）、[[類胡蘿蔔素|類胡蘿蔔素]]（Carotenoid）、[[番茄紅素|番茄紅素]]（Lycopene）、[[辣椒紅素|辣椒紅素]]（Capsanthin）等色素而呈紅色、橘色或黃色，其中類胡蘿蔔素占一半以上<ref name="ency">{{cite web|url=http://ap6.pccu.edu.tw/Encyclopedia/data.asp?id=6137&htm=09-387-5451 |title=雜色體|publisher=中華百科全書|author=吳純寬|language=zh-tw}}</ref>。有色體是進行生產與儲存色素的場所<ref name="Whatley_1987">{{cite journal |vauthors=Whatley JM, Whatley FR |title=When is a Chromoplast |journal=New Phytologist |volume=106 |issue=4 |pages=667–678 |year=1987 |doi=10.1111/j.1469-8137.1987.tb00167.x |url=http://onlinelibrary.wiley.com/doi/10.1111/j.1469-8137.1987.tb00167.x/abstract}}</ref>，為多數植物[[花瓣|花瓣]]及[[果實|果實]]的顏色來源。

有色體常被用作與白色體相對，[[葉綠體|葉綠體]]因含有色素，也能被視為廣義的有色體之一。但有色體仍多專指含有葉黃素、類胡蘿蔔素等非葉綠素色素的質粒體。<ref>{{cite web|url=http://www.answers.com/topic/chromoplast|title=chromoplast|language=en}}</ref><ref>{{cite web|url=http://www.mondofacto.com/facts/dictionary?chromoplast|title=chromoplast definition|date=2008-03-11|publisher=mondofacto|language=en}}</ref>

== 形態 ==
{{File|zh-hant=Plastids_types_zh.svg|zh-hans=Plastids_types_zhs.svg|thumb|220px|desc=各種[[色素體|質粒體]]的關係圖}}
有色體大多在植物有色的器官細胞中被發現，如[[水果|水果]]和[[花瓣|花瓣]]，其中的[[色素|色素]]為這些部位的顏色來源，主要的目的為吸引昆蟲等動物前來協助[[授粉|授粉]]或傳播[[種子|種子]]。
有色體的形態隨其中貯存的色素而異，如[[胡蘿蔔|胡蘿蔔]]中的有色體在成熟後呈帶狀、針狀或板狀，[[辣椒|辣椒]]果皮中的有色體呈細絲束狀，番茄果皮中呈柱狀，[[向日葵|向日葵]]、[[蒲公英|蒲公英]]等的花瓣中的有色體則呈球狀<ref name="ency"/>。

== 轉換 ==
有色體和[[白色體|白色體]]、[[葉綠體|葉綠體]]等[[質粒體|質粒體]]關係均很密切，皆是來自[[藍綠菌|藍綠菌]]的[[內共生|內共生]]<ref name="Egea_2010">{{cite journal |vauthors=Egea I, Barsan C, Bian W |title=Chromoplast differentiation: current status and perspectives |journal=Plant Cell Physiol. |volume=51 |issue=10 |pages=1601–11 |date=October 2010 |pmid=20801922 |doi=10.1093/pcp/pcq136 |url=http://pcp.oxfordjournals.org/cgi/pmidlookup?view=long&pmid=20801922|display-authors=etal}}</ref>，在某些情況下可互相轉換，如植物的[[果實|果實]]成熟時，葉綠體和白色體可以轉換成有色體，造成果實的顏色。目前仍不清楚其作用機制，只知道葉綠體可能將其[[類囊體|類囊體]]破壞後，在[[基質|基質]]中堆積葉黃素等色素的顆粒。<ref name="ency"/>

== 混淆 ==
能在秋冬變紅的[[樹葉|樹葉]]，其顏色來源是[[葉綠素|葉綠素]]的分解，導致原本就存在於葉片中的[[類胡蘿蔔素|類胡蘿蔔素]]表現顏色，使葉子的顏色從綠色變為橘紅色，因此其顏色來源和花果中的有色體無關。
