'''下沼站'''（{{jpn|j=下沼駅|hg=しもぬまえき|rm=Shimonuma eki|f=y}}）是位於[[北海道|北海道]][[天鹽郡|天鹽郡]][[幌延町|幌延町]]下沼的[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[宗谷本線|宗谷本線]]的[[無人站|無人站]]，車站編碼為'''W73'''。電報略號為'''モマ'''。

在中午前，只會有往來各1班普通列車停靠本車站。

== 車站構造 ==
*設有一個側式月台及一條乘車線的地面車站
*由[[車長車廂|車長車廂]]改建的候車室，沒有設有洗手間。
*自2008年（平成18年）起，候車室的外牆情況不斷惡化。而候車室內部有不少由當地居民佈置的裝飾。
*[[無人站|無人站]]，亦是[[秘境車站|秘境車站]]。附近主要是防止環境惡化的森林。

== 站名由來 ==
車站名稱來自[[阿伊努語|阿伊努語]]「pan-ke-to/{{lang|ja|パンケ・トー}}」，意思為「下方的湖」。

== 車站周邊 ==
*{{Link-ja|國道40號|国道40号}}
*{{Link-ja|佐呂別原野|サロベツ原野}} - 位於車站以西約1.5公里。
*下沼泉 - 由車站步行約2分鐘，由當地居民管理。

*龐克池塘 - 位於車站以西約2公里。

== 歷史 ==
*1926年（[[大正|大正]]15年）9月25日 - [[日本國有鐵道|日本國有鐵道]]'''天鹽線'''下沼站啟用。一般車站。
*1977年（[[昭和|昭和]]52年）5月25日 - 廢止貨物裝卸。
*1984年（昭和59年）
**2月1日 - 廢止行李處理。
**11月10日 - 導入[[調度集中系統|CTC]]的同時成為[[無人站|無人站]]。
*1985年（昭和60年）～1986年（昭和61年） - 車站建築改建。
*1987年（昭和62年）4月1日 - 正式民營化並進行分割，下沼站劃歸由JR北海道繼承。
