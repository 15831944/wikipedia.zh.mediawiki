'''下马碑'''，别称'''下马牌'''，是[[中国传统建筑|中国]]乃至整个[[东亚|东亚地区]]的传统建筑中常见的一种碑牌。多用[[汉白玉|汉白玉]]、[[花崗岩|花崗岩]]、[[大理石|大理石]]或[[木材|木材]]等石木材料制成。竖立在寺庙、墓陵、宫殿等庄肃建筑群的正门（即南门）左右两侧，提醒路过人员从马骡上或[[轿|轿子]]上下来步行通过，以表示对建筑物中的人物或神灵的恭敬。下马碑是一种谕令碑，即碑文等同于帝王口谕的命令，门前有看守兵卒监督，违令者甚至会被[[斬首|斬首]]惩罚。它也是一种[[封建社会|封建社会]]等级礼仪的象征，来源于[[儒家|儒家思想]]中的五伦观念。

== 分布 ==

下马碑主要立于以下三种建筑群前：各朝各代的皇家墓陵（如[[明十三陵|明十三陵]]、[[清西陵|清西陵]]）、祭祀或供奉先君圣祖的庙宇（如[[孔庙|孔庙]]、[[历代帝王庙|历代帝王庙]]）、帝王宫殿（如[[故宫|沈阳故宫]]、[[故宫|北京故宫]]）。可知最早的下马碑在[[山东省|山东省]][[曲阜市|曲阜市]][[曲阜孔庙|孔庙]]的正门前，初立于[[金朝|金]]·[[明昌|明昌]]二年（1191年），而后又于[[明朝|明]]·[[永乐_(明朝)|永乐]]十五年（1417年）重立，保留至今。中国历史上的各朝各代均在其疆域上立过下马碑，除[[中国大陆|中国大陆地区]]以外，在[[汉字文化圈|汉字文化圈]]影响到的周边国家与地区亦有竖立下马碑的习惯。各地下马碑举例有：[[中国大陆|中国大陆]][[北京市|北京]][[历代帝王庙|历代帝王庙]]{{参|1={{Cite == 碑文 ==

[[后金|后金]][[皇太极|皇太极]]在位时期，[[建州女真|建州女真人]]在[[中国东北地区|中国东北地区]]使用过[[满语|满文]]或满文汉文合璧的下马碑。而后[[清朝|清朝]][[康熙|康熙]]年间首先立满文汉文蒙古文三文合璧的下马碑。[[乾隆|乾隆]]年间增加了[[藏语|藏文]]、[[察合台語|察合台文]]和[[托忒蒙古文|托忒文]]，使得现今所见文字最多的下马碑为六文合璧的下马碑，北京历代帝王庙正门前左右侧的两幢下马碑便是一例。北京历代帝王庙正门（南门）前左右两侧各立一幢下马碑，各刻三文。右侧东下马碑碑阳面由左向右分别刻着汉文、满文、托忒文。左侧西下马碑碑阳面由左向右分别刻着藏文、蒙古文、察合台文。两碑碑阴均有文字，各碑碑阳文字均是另碑碑阴文字。满文别称“清文”，即[[满语|满语]]书面语在清朝的书写形式；托忒文即漠西蒙古文，今称[[卫拉特语|蒙古语西部方言]]，使用接近蒙古文字的托忒字母书写；藏文别称“番文”、“西番文”、“两番文”，即藏语书面语在清朝的书写形式；蒙古文简称“蒙文”，即蒙古语书面语在清朝的书写形式；察合台文别称“回文”、“回子文”、“回回文”，即今[[维吾尔语|维吾尔语]]在清朝的书面语——察合台语的书写形式{{注|1=[[察合台語|察合台語]]是现今[[维吾尔语|维吾尔语]]和[[乌兹别克语|乌兹别克语]]的原语，虽然口语上早在[[察合台汗國|察合台汗國]]灭亡后就开始异别，但作为二语共同的书面语一直使用到1920年代。此外，中国历史上所谓的“回回语”另可泛指[[回鹘语|回鹘语]]、[[高昌语|高昌语]]、[[波斯语|波斯语]]、[[阿拉伯语|阿拉伯语]]等来自西域的言语。}}。历代帝王庙前的[[景德街|景德街]]早在明·嘉靖年间因兴建帝王庙改为[[步行街|步行街]]{{参|1={{Cite book|title=《宸垣识略》|author=[[吴长元|吴长元]]|year=[[清朝|清朝]]}}}}，是北京最早的步行街之一。

下马碑碑文多为提醒路过人员下马的言辞。其汉文碑文一律采用[[繁体中文|繁体字]][[楷书|楷书]][[東亞文字排列方向|纵排]]人工刻写。满文、蒙古文和托忒文同样按照各自书写习惯的楷体纵排人工刻写。而藏文和察合台文则按照二语的书写习惯横排刻写，各行一至三字，刻不下则按音节断音改行刻写。其中藏文由左向右刻写，察合台文由右向左。满、蒙、藏、托忒文句末有各自文字的传统句号标点（{{ManchuSibeUnicode|᠈|valign=middle}}与 {{Bo-textonly|།}}），汉、察二文句末无标点。以下不完整的列出各地各文的下马碑碑文（鼠标悬浮词上可获取文字转写注音信息）：

== 用途 ==
[[文件:Shangmashi.JPG|缩略图]]外的上马石]]
{{另见|拴马桩|上马石}}

人员在逾过下马碑前，下马出轿，或步行进入庙宇、陵墓、宫殿等建筑物内，或步行走过门前到达另一侧的下马碑后方可上马回轿。为停留马匹的方便，客人若在殿府内过夜则有仆人迁走马匹喂饮水和饲料，若暂留，下马碑旁有[[拴马桩|拴马桩]]供马匹主人拴迁马匹使用。一些较大的王府官府有专门的马厩，不在门前立拴马桩。北京[[宣武门|宣武门]]原有一条[[拴马桩胡同|拴马桩胡同]]{{注|1=今分为北小拴胡同、马桩胡同、双马桩胡同等等。}}，功能类似于现代的[[停車場|停車場]]，是进宫人士暂养马匹的场所。离走时又有“{{Lang|en|L}}”形的两台阶式[[上马石|上马石]]，方便客人登马。明代翰林院大学士[[金幼孜|金幼孜]]作了一首《晚出西長安街詩》，谈到出宫步行[[西长安街|西长安街]]走过[[西长安门|西长安门]]后便可“登鞍”上马：

{{Quote|白髮蒼蒼老從官，籃輿自便懶登鞍。<br />京華久住郊遊少，惟有西山總耐看。{{参|1={{Cite book|title=《晚出西長安街詩》|author=[[金幼孜|金幼孜]]|year=[[明朝|明朝]]}}}}}}

== 保护状况 ==
[[File:서울_동관왕묘_01.jpg|thumb]]
虽然下马碑并不属于石刻艺术，但由于其多为朝廷或地方衙门官方所制，石料质地以及雕刻的工匠水平都高于凡常。在古代，下马碑的数量相对较多，因其平凡所以一直没有引起文物保护单位的重视。经过数百年的周折，石碑数量大减，而且现存石碑多有[[風化作用|风化]]、破损、断裂等痕迹。北京历代帝王庙的两幢石碑在[[文化大革命|文革期间]]被[[红卫兵|红卫兵]]以[[破四旧|破四旧]]的名义从基石上搬下、砸断并埋入地里，直至1999年10月文化部门在修缮历代帝王庙时才被挖掘出来，移回正门两侧，恢复了原貌。现在中国各地的下马碑都被列入保护文物名单，成为不可交易的国宝。[[沈阳故宫|沈阳故宫]]东门外南北两侧各立一幢下马碑，立于清·乾隆四十八年（1783年），刻有汉、满、蒙、回、藏五文。1999年6月21日凌晨2时许，福满楼餐饮有限公司司机，卢志会，驾驶一辆奔驰轿车撞上北侧的下马碑，石碑倒裂，卢当场死亡。因为下马碑不是售卖物品，所以属于“无价之宝”，沈阳故宫博物院方面参考鉴定小组的建议，确定被撞裂的北下马碑为中国国家一级文物，索取赔款2700万元人民币。2000年此案在沈阳市中级人民法院公开开庭审理，2005年送到辽宁省高级人民法院审理，下马碑最终以2000万元人民币为基数，车主赔偿100万元人民币。2008年10月22日沈阳故宫博物院将现代复制的一幢下马石碑立于北碑原位置，将撞裂的原石碑放在馆内保存。{{参|1={{Cite web|title=奔驰撞断故宫“下马碑”一案在沈阳开庭|url=http://news.creaders.net/breaking/newsViewer.php?nid=12041&id=326695&aid=13|publisher=万维读者网引自辽沈晚报|date=2000-07-12|accessdate=2013-01-18|language=zh-cn}}}}{{参|1={{Cite web|title=沈阳故宫下马碑的前世今生(图)|url=http://www.china.com.cn/aboutchina/txt/2008-10/27/content_16669901.htm|publisher=中国网|date=2008-10-27|accessdate=2013-01-18|language=zh-cn}}}}{{参|1={{Cite web|title=专家详解沈阳故宫新“下马碑”|url=http://www.ln.xinhua.org/xwzx/2008-11/05/content_14830338.htm|publisher=新华网|date=2008-11-05|accessdate=2013-01-18|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20160304095648/http://www.ln.xinhua.org/xwzx/2008-11/05/content_14830338.htm|archivedate=2016-03-04}}}}
