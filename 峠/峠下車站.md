'''峠下站'''（{{jpn|j=峠下駅|hg=とうげしたえき|rm=Tōgeshita eki|f=y}}）是位於[[北海道|北海道]][[留萌市|留萌市]]大字留萌村字峠下，隸屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[留萌本線|留萌本線]]的[[鐵路車站|鐵路車站]]。電報略號為'''トタ'''。

車站名稱源自[[阿伊努語|阿伊努語]]「rucis-pok」，即峠之下。主要指位於古時[[石狩國|石狩國]]及[[天鹽國|天鹽國]]之間的山道。

== 歷史 ==
* 1910年（明治43年）11月23日 - [[鐵道省|鐵道省]]'''留萌線'''深川車站至留萌車站之間設置本車站。
* 1931年（昭和6年）10月10日 - 留萌線改稱為'''留萌本線'''。
* 1949年（昭和24年）6月1日 - 由[[日本國有鐵道|日本國有鐵道]]接管。
* 1977年（昭和52年）5月25日 - 取消貨物裝卸業務。
* 1984年（昭和59年）2月1日 - 取消行李裝卸業務。同時因為停止賣票及改票而不再派駐職員，只保留負責閉塞系統的人員。
* 1987年（昭和62年）4月1日 - 由於[[國鐵分割民營化|國鐵分割民營化]]，車站由[[北海道旅客鐵道|JR北海道]]管理。
* 1998年（平成10年）3月 - 由於閉塞系統由[[閉塞_(鐵路)#電氣路籤（牌）閉塞|電氣路籤閉塞]]改為特殊[[自動閉塞|自動閉塞]]，車站完全無人化。

== 車站構造 ==
現時為2面2線的地面車站，設有2個[[側式月台|側式月台]]並配置成千鳥式（非對稱）。亦是深川站至留萌站之間唯一能夠進行[[列車交會|列車交會]]的中途車站。車站建築旁的月台西側能夠使用[[平交道|平交道]]前往對面月台的東方。車站建築的南方為下行路軌，而對面月台則為上次路軌，沒有編號。此外，上行路軌往深川方向設有[[側線|側線]]並連接形往留萌方向，兩條路軌亦設有[[安全側線|安全側線]]。1983年4月時，下行路軌往留萌方向設有貨物側線連接位於車站建築西面的缺角式貨物月台，但於1993年3月移走。設有[[蒸汽機車|蒸汽機車]]的供水塔。

現時是無人車站。車站建築位於車站的南方（留萌方向的左方），與下行路軌的月台中央部份連接。現時車站建築為有人車站時代興建，外觀為茶色的木造建築，內部仍殘留售票及處理小型行李的窗口。候車室約有3張[[疊蓆|疊蓆]]的空間，亦放置由當地居民手造的[[人造花|人造花]]裝飾。無人化後仍設有保養路軌的休息室供冬季時負責除雪的職員之用。廁所設置於列車停靠的位置約10米距離。

== 載客量 ==
* 1981年度的每天載客量為18人。
* 1992年度的每天載客量為4人。
* 2011年至2015年度平均每天載客量少於1人<ref>{{cite web|url=http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |language=ja |title=平成26年度 線区別の収支状況等について |publisher=JR北海道 |date=2016-02-10 |accessdate=2017-06-21 |archive-url=https://web.archive.org/web/20170407133949/http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |archive-date=2017-04-07 |deadurl=yes }}</ref>。

== 車站周邊 ==
北面為天鹽山地，南面為增毛山地，車站前有群山聳立，後方為一大片叢林。亦有小河流在車站前流動。除了有1戶農家外人煙稀少。本車站與惠比島站之間有2座隧道，亦是深川站至留萌站之間的唯一有隧道的區間。
* {{Link-ja|國道233號|国道233号}}（留萌國道）
* [[北海道道549號峠下沼田線|北海道道549號峠下沼田線]]
* 峠下森林公園
* [[留萌川|留萌川]]
