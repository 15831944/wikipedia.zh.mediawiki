《'''難波愛～現在、想到的事情～'''》（{{lang-ja|難波愛〜今、思うこと〜}}）是日本女子組合[[NMB48|NMB48]]的第三張[[录音室专辑|原创专辑]]，於2017年8月2日由[[YOSHIMOTO_R_and_C|laugh out loud! records]]發售。

== 概要 ==
與前作《世界的中心是大阪 ～難波自治區～》將近約3年的距離的專輯。專輯分為5種版本，初回限定盤 Type N・M・B，通常盤以及劇場盤<ref>{{Cite news|url=http://news.nmb48.com/release/20170627/210056.html|script-title=ja:【発売決定】NMB48 約3年振りとなる待望の3rdアルバム、2017年8月2日（水）リリース決定！|publisher=NMB48 官方網站|date=2017-06-27|language=ja}}</ref>。

全部版本都收錄了新曲《竟然是新加坡》（{{lang|ja|まさかシンガポール}}），《難波愛》以及第10張單曲《[[真不像我|真不像我]]》至第16張單曲《[[除我之外的人|除我之外的人]]》的A面曲。通常盤收錄了白間美瑠演唱的單人曲《我沒有被愛過》（{{lang|ja|僕は愛されてはいない}}），劇場盤收錄了新曲《最最最 最高！》（{{lang|ja|サササ　サイコー！}}）。

第10張單曲至第16張單曲中，初回限定盤 Type N收錄了Team N的歌曲，初回限定盤 Type M收錄了Team M的歌曲以及初回限定盤 Type B收錄了Team BII的歌曲。此外，通常盤跟劇場盤收錄了其他的B面曲。
