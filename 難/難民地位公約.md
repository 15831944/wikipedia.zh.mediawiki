'''《難民地位公約》'''（{{lang-fr|''Convention relative au statut des réfugiés''}}; {{lang-en|''Convention Relating to the Status of Refugees''}}）於1951年7月28日通過，並於1954年4月22日正式生效。難民地位公約和[[有關難民地位的議定書|有關難民地位的議定書]]是國際難民保護的兩項核心規範。該公約定義了[[難民|難民]]、難民的資格及權利、以及提供難民庇護的國家所應負責任的一項國際公約。公約亦載明難民的消極資格（如戰爭罪犯不屬難民）。此外，為協助受母國壓迫者、難民、無國籍者尋求庇護的必要，只要持有依據該公約簽發之旅行文件，即可免簽證（visa-free）旅行遷徙。自2000年12月起，聯合國決議以每年6月20日為[[世界難民日|世界難民日]]<ref name="難民地位公約">{{cite web |url = http://www.un.org/chinese/hr/issue/docs/82.PDF | title = 聯合國《關於難民地位的公約》_中文全文 |publisher=聯合國 |date=1951年}}</ref>。

== 歷史 ==
《難民地位公約》於1951年7月28日經聯合國會議批准，並於1954年4月22日正式生效。本公約在[[日內瓦|日內瓦]]批准，因此常被稱「日內瓦公約」，但並非規範戰爭時期行為的《[[日內瓦公約|日內瓦公約]]》。此公約原始目的僅在處理[[第二次世界大戰|第二次世界大戰]]後歐洲地區的難民保護，而不適用於1951年1月1日以後的難民，亦不適用歐洲地區外的難民。

;1966年議定書擴及全球範圍
不過，鑑於難民已擴大為全球共同面臨的議題，因此[[聯合國|聯合國]]在1966年11月18日通過[[有關難民地位的議定書|有關難民地位的議定書]]<ref name="難民地位議定書">{{cite web |url = http://www.ohchr.org/CH/Issues/Documents/other_instruments/83.PDF |title = 《關於難民地位的議定書》 |publisher = 聯合國 | date=1967年}}</ref>，取消了地域和時間的限制，將原公約保護擴大至全球範圍，議定書於1967年10月4日正式生效，適用至今。<ref name="國際人權法文獻選集">{{cite book |title = 國際人權法文獻選集與解說 |publisher = 前衛出版社 | author=陳隆志主編 | year=2006 | location=台北 | pages=202-223| isbn = 957-801-497-X}}</ref>

== 難民的定義 ==
依據此公約第一條第2項第1款，無論請求庇護者具有國籍、或[[無國籍|無國籍]]都適用，難民的定義是：「具有正當理由而畏懼會因為種族、宗教、國籍、特定社會團體的成員身分或政治見解的原因，受到迫害，因而居留在其本國之外，並且不能或，由於其畏懼，不願接受其本國保護的任何人。」<ref name="難民地位公約"/>

依據公約定義，主要限於遭到「人為政治迫害」的「政治難民」（種族、宗教、國籍、特定社會團體的成員身分、政治見解）。因此，此公約庇護對象尚未包括因為天災、戰爭或其他經濟社會因素的「[[戰爭難民|戰爭難民]]」或「經濟災民/[[流民|流民]]」，例如為了尋求更好經濟生活的[[偷渡犯|偷渡犯]]。由於其他類型的難民，並未受《難民地位公約》的保護，但持續發生的現實，業已成為[[國際人權法|國際人權法]]、[[國際人道法|國際人道法]]的新課題<ref name="國際人權法文獻選集"/>。

== 難民的權利 ==
公約[[締約國|締約國]]有義務保障[[難民|難民]]的法律上地位、司法地位、提供身分證件、接受正當法律程序審判的權利、財產權、有償工作權、結社權利、遷徙自由，福利救助例如：居住權、（與締約國本國人民同等之）公共救濟與援助、房屋優遇<ref name="難民地位公約"/>。

== 締約國 ==
[[丹麥|丹麥]]是第一個批准此公約的國家（1952年12月4日）。截至2011年6月已有148個國家簽署了1951年公約或1967年​​議定書<ref name="批准國">{{cite web |url = http://www.unhcr.org/protect/PROTECTION/3b73b0d63.pdf | title=難民地位公約&議定書的締約國 |publisher=聯合國難民署 | date=2011年4月 | pages=頁1}}</ref>，最新批准的第148國是南太平洋島國[[諾魯|諾魯]]<ref>{{cite web |url = http://www.theaustralian.com.au/national-affairs/naurus-un-move-on-refugee-convention-adds-to-pressure-on-labor/story-fn59niix-1226077250521 | title=Nauru's UN move on refugee convention adds to pressure on Labor | work = The Australian(澳洲人報) | date=2012-06-17 |author = BEN PACKHAM}}</ref>。

=== 責任 ===
依據公約，已批准《難民地位公約》的國家，有責任保護在其領土上的[[難民|難民]]。批准1951年公約及1967年議定書的締約方，必須遵守的部分條款如下<ref name="國際難民法指南">{{cite web |url = http://www.ipu.org/pdf/publications/refugee_en.pdf |title =《REFUGEE PROTECTION: A Guide to International Refugee Law》（國際難民法指南） | publisher=聯合國難民署 | accessdate=2012-11-22 |author = ISBN 92-9142-101-4}}</ref>

;締約國保證與[[聯合國|聯合國]]專責機構合作：
根據難民公約第35條、1967年議定書第2條之規定，締約國家同意配合[[聯合國難民署|聯合國難民事務高級專員公署]]以利公署行使其職務，並應特別使公署在監督適用本公約規定行使職務時獲得便利，尤其應便利公署監督本議定書各項條款之實施。此外，締約國需提供相關資訊，例如：難民情況、公約履行情況、締約國國內現行有效或日後可能生效之涉及難民的法令<ref name="國際難民法指南"/>。

;締約國之國內立法通報義務：
依據難民公約第36條，締約國應向[[聯合國秘書長|聯合國秘書長]]送交它們(締約國)可能採用為保證執行本公約的法律和規章；亦即締約國為了實踐本公約義務，所因此制定之相關法令（例如國內法），需將法令內容通知聯合國。<ref name="國際難民法指南"/><br/>

;對等互惠原則之豁免（例外不適用）：
不同國家之間，給予他國人民之權利，通常依照「對等互惠原則」的概念——亦即「根據A國法律，賦予（外國人）B國人在A國的權利，通常會依據B國法律是否給予A國人在B國享有類似的待遇。」然而，這項對等互惠之概念，並不適用於[[難民|難民]]的情況，因為難民並未會喜歡保護其母國。

=== 難民非法進入締約國 ===
;非法入境之難民，不得施以刑罰：
根據公約第31條第1項，若[[難民|難民]]因為生命、自由受到（公約第一條所指的）威脅，因而未經許可而進入、逗留締約國的[[領土|領土]]，締約國不得因其非法入境或逗留，而對難民加以刑罰。但前提是，難民必須毫不遲疑地自行投向締約國當局，說明其非法入境、逗留之正當理由。

依據第2項，締約國對非法入境、逗留的難民之行動，不得施加「除了必要以外」的任何限制，這種必要的限制，也只能適用於難民在締約國境內地位正常化以前、或者難民獲得第三國入境許可之前。締約國還必須給予難民合理期間、一切必要便利，以便難民得以獲得第三國入境的許可<ref name="難民地位公約" />。

== 禁止驅回原則 ==
任何國家，不得將在領土內的[[難民|難民]]強行遣返回母國，或驅逐出境。亦稱'''不推回原則'''或'''不遣返原則'''（non- refoulement）。

根據1951年《難民地位公約》第33條：「任何遞約國，不得以任何方式，將難民驅逐或送回至其（母國或其他國家，若這可能導致難民的）生命或自由因為他的種族、宗教、國籍、參加某一社會團體或具有某種政治見解而受威脅的領土邊界。」<ref name="難民地位公約" />

這項原則被國際社會廣泛接受，已屬於[[國際習慣法|國際習慣法]]。因此，即使每一國家，即便非《難民地位公約》的締約國，皆有義務尊重[[禁止驅回原則|禁止驅回原則]]；締約國是依據《難民地位公約》，非締約國則是依據[[國際習慣法|國際習慣法]]。

如果此原則受到威脅、可能被違反時，[[聯合國難民署|聯合國難民署]]可以與有關當局干預，必要時亦得向國際社會公告相關情事<ref name="國際難民法指南" />。
==中国加入《关于难民地位的公约》和《关于难民地位的议定书》==
1982年08月23日，[[第五届全国人民代表大会常务委员会|第五届全国人民代表大会常务委员会]]第二十四次会议决定：中华人民共和国加入《关于难民地位的公约》和《关于难民地位的议定书》，并同时声明对下列条款予以保留：
* 一、《公约》第十四条后半部分，即“他在任何其他缔约国领土内，应给以他经常居住国家的国民所享有的同样保护。”；
* 二、《公约》第十六条第（三）款；
* 三、《议定书》第四条。

==聯合國專責機構==


'''難民高級專員公署'''：聯合國的前身[[國際聯盟|國際聯盟]]於1921年所創立，專責處理國際難民。

'''聯合國難民事務高級專員公署'''（UN High Commissioner for Refugees，UN HCR）：簡稱[[聯合國難民署|聯合國難民署]]，第二次世界大戰後，聯合國於1950年12月14日決議成立，迄今一直是最重要的難民保護國際組織<ref>{{cite web | url = http://www.unhcr.org/pages/49da0e466.html |title = 聯合國難民署：1951年至今日 |work = 聯合國難民署}}</ref>。

==2016年史上首度難民高峰會==
由於敘利亞與伊斯蘭國的戰亂等問題，截至2015年底全世界有6,530萬難民，是世界二次大戰後最大難民危機。2016年9月19日，聯合國在紐約就難民和移民問題召開史上第一次最高級別峰會，193國領袖和代表通過《關於難民與移民的紐約宣言》。根據宣言：(1)國際社會將與聯合國推動建立一項多方參與的難民問題全面響應框架，以因應突發及長期存在的大規模難民流動問題。(2)將啟動政府間的談判進程，期望於2018年制定「安全、有序和正常移民的全球契約」。 (3)保護難民和移民權利，改善兒童教育、難民就業，強化支持難民接受國，改善搜救運作，增加經費及安置努力，推動反仇外心理。<ref name="EPO2016">{{cite news|author1=夏雨|title=史上首個聯合國難民峰會 193國通過宣言|url=http://www.epochtimes.com.tw/n182316/%E5%8F%B2%E4%B8%8A%E9%A6%96%E5%80%8B%E8%81%AF%E5%90%88%E5%9C%8B%E9%9B%A3%E6%B0%91%E5%B3%B0%E6%9C%83-193%E5%9C%8B%E9%80%9A%E9%81%8E%E5%AE%A3%E8%A8%80.html|publisher=大紀元時報|date=2016-09-20}}</ref>。
