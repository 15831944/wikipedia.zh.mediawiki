== 御影町 ==
=== 主要人物（P1） ===

'''{{nihongo|主角|主人公}}'''（配音：[[石塚堅|石塚堅]]/ [[石田彰|石田彰]]/ [[谷山紀章|谷山紀章]]）
: 遊戲：沒有預設名稱 / 漫畫版：'''{{nihongo|藤堂尚也|藤堂 尚也/ とうどう なおや}}''' / 小説版：'''{{nihongo|朱雀院慈平|朱雀院 慈平/ すざくいん じへい}}''' / 廣播劇CD：'''{{nihongo|鳴海優也|鳴海 優也/ なるみ ゆうや}}'''
: [[冷兵器|冷兵器]]：[[單手劍|單手劍]]
: [[銃|銃]]：[[機關槍|機關槍]]
:* Persona：{{nihongo|[[青面金剛|青面金剛]]|セイメンコンゴウ}}→{{nihongo|[[阿蒙|阿蒙]]．[[拉|拉]]|アメン・ラー}} / 秘儀：皇帝
: 綽號「'''{{nihongo|藤堂君|なおりん}}'''（漫畫版）」、「{{nihongo||ジェイ}}（小説版）」，《女神異聞錄Persona》的主角，代表玩家的角色，聖白貂學園2年4班學生。麻希的青梅竹馬。

'''{{nihongo|園村麻希|園村 麻希/ そのむら まき}}'''（配音：[[水沢潤|水沢潤]]）
:* Persona：{{nihongo|[[媽祖|媽祖]]|マソ}}→{{nihongo|[[薇兒丹蒂|薇兒丹蒂]]|ヴェルザンディ}} / 秘儀：女教皇
: [[冷兵器|冷兵器]]：[[弓箭|弓箭]]
: [[銃|銃]]：[[手槍|手槍]]
: 綽號「'''{{nihongo|麻希|マキ}}'''」，主角的同班同學。因體弱多病學期大部分時間在御影町綜合病院療養。
: 在校時的課余愛好是繪畫，和同屬繪畫部的稻葉正男（マーク）關系甚好，但真正喜歡的人是足球部的健將內藤陽介。
: 入院後班級內常去探望她的除了マーク就只有同樣愛好幻想的女生姬野由子。因為大部分時間在病床上發呆，因而幻想也成為了生活的一部分。由子給她帶去的空想小說《樂園之門》成為了麻希的現實與空想開始混淆的導火線。

'''{{nihongo|南條圭|南条 圭/ なんじょう けい}}'''（配音：[[山野井仁|山野井仁]]）
:* Persona：{{nihongo|[[愛染明王|愛染明王]]|アイゼンミョウオウ}}→{{nihongo|山岡|ヤマオカ}} / 秘儀：教皇→審判（P1）
: [[冷兵器|冷兵器]]：[[雙手劍|雙手劍]]
: [[銃|銃]]：[[狙擊槍|狙擊槍]]
: 主角的同學，綽號「'''{{nihongo|南條君|なんじょうくん}}'''」，大型聯合企業南條會社的公子，典型的精英人士，頭腦聰明才能出眾，但也因為自負高傲外加愛說教人而被同學敬而遠之。
: 脖子上醒目的「1」字圍巾，是南條家的老管家山岡從幼兒園開始就給他佩戴的個性標志，代表了希望他將來成為日本第一男兒的願望，南條本人也以此為目標努力。

'''{{nihongo|稻葉正男|稲葉 正男/ いなば まさお}}'''（配音：[[石塚堅|石塚堅]]）
:* Persona：{{nihongo||オグン}}→{{nihongo|[[須佐之男|須佐之男]]|スサノオ}} / 秘儀：戰車
: [[冷兵器|冷兵器]]：[[斧|斧]]
: [[銃|銃]]：[[霰彈槍|霰彈槍]]
: 2年4班學生，綽號「{{nihongo||マーク}}」，性格熱血直爽的男孩，雖然長相平凡但因為仗義的性格和直爽的言行在班級裡有不小人氣。
: 興趣愛好是街頭塗鴉（背包裡裝著的是顏料噴管），也是學園裡美術部和舞蹈部的成員。

'''{{nihongo|黛雪野|黛 ゆきの/ まゆずみ ゆきの}}'''（配音：[[半場友恵|半場友恵]]）
:* Persona：{{nihongo|[[維斯塔|維斯塔]]|ヴェスタ}}→{{nihongo|[[難近母|難近母]]|ドゥルガー}} / 秘儀：女皇
: [[冷兵器|冷兵器]]：[[飛刀|飛刀]]
: [[銃|銃]]：[[霰彈槍|霰彈槍]]
: 學校有名的大姐頭，綽號「{{nihongo||ゆきのさん}}」。
: 家庭貧寒，母親在酒吧做陪酒女郎。
: 曾經有一段時間是不良少女頭目，在冴子老師幫助下回到了正途，因而一直懷有感恩之情，在雪之女王篇裡為了救回被冰封的老師而奔走，是唯一沒有於SEBEC篇出現的主要角色。

'''{{nihongo|桐島英理子|桐島 英理子/ きりしま えりこ}}'''（配音：[[半場友恵|半場友恵]]）
:* Persona：{{nihongo|[[尼克|尼克]]|ニケー}}→{{nihongo|[[米迦勒|米迦勒]]|ミカエル}}（P1） / {{nihongo|[[加百列|加百列]]|ガブリエル}}（P2EP） / 秘儀：審判
: [[冷兵器|冷兵器]]：[[單手劍|單手劍]]
: [[銃|銃]]：[[狙擊槍|狙擊槍]]
: 主角的同學，綽號「{{nihongo||エリー}}」，有1/4白人血統的混血女生，小時候在海外長大因而說話習慣夾帶英語。
: 出身上流家庭的千金小姐，父親是國際航班飛行員，在班級裡對任何人都報以微笑的處世方式博得了廣泛的人氣但也私下遭到非議。不為人知的私下愛好是超常現像研究，因而全部人裡只有她為超能力的覺醒而狂喜。

'''{{nihongo|上杉秀彥|上杉 秀彦/ うえすぎ ひでひこ}}'''（配音：[[吉川虎範|吉川虎範]]）
:* Persona：{{nihongo|[[尼曼|尼曼]]|ネヴァン}}→{{nihongo|[[提爾|提爾]]|ティール}} / 秘儀：正義
: [[冷兵器|冷兵器]]：[[槍|槍]]
: [[銃|銃]]：[[機關槍|機關槍]]
: 主角的同學，綽號「{{nihongo||ブラウン}}」，有很強的表演欲，喜歡嘩眾取寵和說冷笑話，因而在學園內相當為人注目。但其實這種希望被他人注視的表演欲源自自卑感。也因此從內心裡羨慕能夠自然而然得到同學人氣的稻葉，因而時常以哥們身份與稻葉混在一起，但稻葉並沒有將他看做哥們，加上在一旁說教的南條，被女生稱為傻瓜三人組。

'''{{nihongo|綾瀨優香|綾瀬 優香/ あやせ ゆか}}'''（配音：[[倉田雅世|倉田雅世]]）
:* Persona：{{nihongo||フーリー}}→{{nihongo|[[弗雷|弗雷]]|フレイ}} / 秘儀：魔術師
: [[冷兵器|冷兵器]]：[[鞭|鞭]]
: [[銃|銃]]：[[手槍|手槍]]
: 有「照相帖紙機女王」的稱號，綽號「{{nihongo||アヤセ}}」，死語（已經過氣的流行語）連發的元氣明快的女生，典型的1996年流行的コギャル，是相當反映作品時代風貌的設計。
: 追求新潮與流行，無拘無束地生活。雖然這種天真有時候表現為任性但也暗地裡為人羨慕。

'''{{nihongo|城戸玲司|城戸 玲司/ きど れいじ}}'''（配音：[[山野井仁|山野井仁]]）
:* Persona：{{nihongo||ブレス}}→{{nihongo|莫特|モト}} / 秘儀：惡魔→死神
: [[冷兵器|冷兵器]]：[[拳|拳]]
: [[銃|銃]]：[[狙擊槍|狙擊槍]]
: 主角的同學，綽號「{{nihongo||レイジ}}」，半年前轉入的學生，性情孤僻待人冷淡因此在班上沒有什麼朋友，在校外據說是不良番長但都是獨來獨往。
: 實際是神取鷹久的同父異母的弟弟。作為神取家的私生子，從出生起就不被允許公開亮相，和母親二人被父親送到御影町過著隱姓埋名的生活。生活在神取的陰影裡，並把向神取一族復仇作為自己的動力。
