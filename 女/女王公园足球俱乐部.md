'''女王公园足球俱乐部'''（'''Queen's Park Football Club'''），是位於[[蘇格蘭|蘇格蘭]][[格拉斯哥|格拉斯哥]]的足球會，於2008/09年球季由[[蘇格蘭足球乙級聯賽|蘇格蘭乙組聯賽]]降級到[[蘇格蘭足球丙級聯賽|蘇格蘭丙組聯賽]]作賽，現時是[[蘇格蘭足球聯賽|蘇格蘭足球聯賽]]中唯一的業餘球隊。

女王公园的[[格言|格言]]是「''{{lang|la|Ludere Causa Ludendi}}''」，意为“为踢球而踢球”，从1867年成立至今的历史上所有的球员和教练都不从足球获取薪酬<ref>{{cite website|url=http://www.queensparkfc.co.uk/history/intro.html|title=It all began so long ago ... |publisher=女王公园俱乐部官方网站|accessdate=2009-01-28}}</ref>。在职业足球占据绝对优势的当今世界，女王公园因此也只能位居低级别联赛，但是它对于足球发展所做出的重大贡献则始终被广泛称道，2008年它曾被苏格兰足协向[[国际足联|国际足联]]提名申请最高荣誉奖以表彰其对足球的贡献，不过虽然这一提名得到了英国首相[[戈登·布朗|戈登·布朗]]、[[曼联|曼联]]傳奇主帅[[亚历克斯·弗格森|弗格森]]的支持，最终国际足联并未批准<ref>{{cite news|url=http://news.bbc.co.uk/sport1/hi/football/teams/q/queens_park/7282076.stm|title=Queen's Park vie for Fifa honour|publisher=BBC|date=2008-03-06|accessdate=2009-01-28}}</ref><ref>{{cite news|url=http://news.bbc.co.uk/sport1/hi/football/scot_div_1/7382086.stm|title=Queen's Park lose out in Fifa bid|publisher=BBC|date=2008-05-03|accessdate=2009-01-28}}</ref>。

== 简介 ==
女王公园成立於1867年，是[[苏格兰|苏格兰]]歷史最悠久的足球會，也是全世界除[[英格兰|英格兰]]及[[威尔士|威尔士]]地區以外最古老的足球俱乐部<small><ref>{{Cite web
 |url         = http://www.scottishfa.co.uk/scottish_football.cfm?curpageid=420
 |title       = For the sake of the Game
 |publisher   = [[蘇格蘭足球總會|蘇格蘭足球總會]]
 |accessdate  = 2009-01-28
 |deadurl     = yes
 |archiveurl  = https://web.archive.org/web/20160304171033/http://scottishfa.co.uk/scottish_football.cfm?curpageid=420
 |archivedate = 2016-03-04
}} {{en}}</ref></small>。女王公园的主場是[[欧洲足联|欧洲足联]]29座[[歐洲足協五星級足球場|五星級足球場]]之一的[[汉普顿公园球场|汉普顿公园球场]]（Hampden Park），位於[[格拉斯哥|格拉斯哥]]東南部，為全坐席可容52,500名觀眾，对于世界其它业余球队来说几乎不可思议。这同時也是[[苏格兰足球代表队|苏格兰国家队]]的主場。

女王公园曾经作为最早的苏格兰足球的代表与英格兰球队比赛，因为其先进的战术理念而引发了苏格兰足球运动员和教练大举进军英格兰的潮流。

雖然最後一次於1893年贏取[[蘇格蘭足總盃|蘇格蘭足總盃]]，但仍是繼[[凯尔特人足球俱乐部|-{zh-hans:凯尔特人; zh-hk:些路迪;}-]]及[[流浪者足球俱乐部|-{zh-hans:流浪者; zh-hk:格拉斯哥流浪;}-]]後第三奪標次數最多的球隊。昆士柏亦曾跨境奪得兩屆[[英格蘭足總盃|英格蘭足總盃]]亞軍。

== 著名球员 ==
* [[:en:Robert_W._Gardner|罗伯特·加德纳]]（Robert W. Gardner），苏格兰第一位国家队队长、守门员，世界足球已知的最早担任队长的守门员，[[苏格兰足球协会|苏格兰足球协会]]的建立者和主席之一。
* [[:en:Andrew_Watson_(footballer)|安德鲁·沃特森]]（Andrew Watson），世界上最早参加国际足球比赛的黑人球员。
* [[:en:Andy_Roxburgh|安迪·罗克斯伯]]（Andy Roxburgh），前苏格兰国家队領隊。
* [[亚历克斯·弗格森|亚历克斯·弗格森]]爵士（Sir Alex Ferguson），前[[鴨巴甸足球會|鴨巴甸]]、蘇格蘭國家隊以及[[曼联|曼联]]主帅。
* [[:en:Ronnie_Simpson|罗尼·辛普森]]（Ronnie Simpson），[[里斯本雄狮|里斯本雄狮]]門將。
* [[埃登·麦克格迪|-{zh-hans:埃登·麦克格迪; zh-hk:艾登·麥基迪;}-]]（Aiden McGeady），蘇格蘭裔愛爾蘭中場，愛爾蘭國家隊成員，曾效力[[些路迪|些路迪]]及[[愛華頓|愛華頓]]。
* [[安祖·羅拔臣|安祖·羅拔臣]]（Andrew Robertson），[[利物浦|利物浦]]後衛，蘇格蘭國家隊成員。
