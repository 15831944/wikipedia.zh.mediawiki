'''女性避孕套'''（{{lang-en|Female Condom}}，因在[[英國|英國]]等市場上市初期使用'''Femidom'''的品牌名稱行銷，而成為此類產品的代稱{{R|Guardian}}）是一种供[[女性|女性]]使用的[[避孕套|避孕套]]，同[[男性|男性]]避孕套一样，是一種一次性使用的可拋式用品。作為男性保險套的替代，女性保險套給予女性自行決定是否要在進行[[性行為|性行為]]時加強自我保護的權利，這點尤其在其男性伴侶因個人好惡或其他原因而不願意使用保險套時，更顯助益{{R|Guardian}}。

==歷史==
女性保險套是[[丹麥|丹麥]]醫生兼發明家{{link-en|拉瑟·赫賽爾|Lasse Hessel}}（Lasse Hessel）在1980年代後期發明，並在1992年時於英國等歐洲國家上市{{R|Guardian}}。

在推出初期女性保險套曾短暫地掀起一陣熱潮，但卻在短短的幾年之內回歸平淡，在2005年時全英國的銷售佔比甚至低到在[[英國國家統計局|國家統計局]]的相關報告中掛零{{R|Guardian}}。針對女性避孕套的使用，常見的負面評價包括配戴上沒有男性保險套直覺需額外的練習、在性交過程中會發出材料摩擦的異音、或單純只是在視覺上不太美觀等{{R|NCBI}}{{R|Guardian}}。

雖然在歐洲等主要市場失利，但女性保險套並沒有完全消失，卻反而在[[非洲|非洲]]等[[開發中國家|開發中地區]]意外地受歡迎。在1990年代末期女性保險套的專利與製造是由[[美國|美國]]業者[[女性健康公司|女性健康公司]]（Female Health Company, FHC）擁有，透過與[[世界衛生組織|世界衛生組織]]（WHO）及[[聯合國人口基金|聯合國人口基金]]（UNFPA）等國際組織的合作，FHC以折扣價格大量將女性保險套銷往包括[[辛巴威|辛巴威]]在內的非洲國家，以便抑制當地嚴重的[[愛滋病|愛滋病]]擴散問題{{R|Zimbabwe}}。在2004年至2008年間女性保險套的銷售成長了接近三倍，於全球90個國家賣出3300萬個，但相對於男性保險套，其普及率仍然非常低{{R|Zimbabwe}}。

除了握有產品專利的FHC外，全球尚有多家不同的企業有在生產與製造用途類似的商品。

== 外觀與材料 ==
女性避孕套的外觀像是放大版的男性保險套，外面涂有[[润滑剂|润滑剂]]，但不似男性保險套只有開口端處有環狀的肋圈，女性保險套在開口端與頂點端都各自有一個肋圈支撐。在剛推出時，一代的女性保險套（FC1）是採[[聚氨酯|聚氨酯]]（PU）製作而成，但在2005年時FHC宣佈將開始在美國以外的市場推出改採[[腈|合成腈]]材質製造的二代產品「FC2」，以便大幅降低產品的製造成本以符合開發中國家的市場需求{{R|FC2}}，並自2007年起實際開始銷售FC2。

相對於男性保險套大部分都是使用[[乳膠|乳膠]]為原料，女性保險套因為使用聚氨酯或是腈作為材料，比起一般乳膠套導熱性和韌度更好。除此之外這些非乳膠材料也能提供一些對乳膠[[過敏|過敏]]的人適當的替代方案。

== 使用方法 ==
