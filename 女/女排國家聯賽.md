'''女排國家聯賽'''（{{lang|en|'''FIVB Volleyball Women's Nations League'''}}，英文簡稱：'''VNL'''），或稱'''世界女排聯賽'''，是[[國際排球總會|國際排聯]]（[[FIVB|FIVB]]）舉辦的一年一度大型世界級女子排球比賽，首屆賽事於2018年舉行，以取代已舉辦25年的[[世界女排大獎賽|世界女排大獎賽]]。賽事有16支隊伍參賽，其中12支為核心隊伍、4支為挑戰隊伍。

與世界女排大獎賽相比，國家聯賽不再劃分三個檔次組別，統一以一個組別進行比賽，但參與隊伍的總數會由去年大奬賽的32隊降低至16隊，以提高競爭程度，其中12隊核心隊伍大部分來自以往大獎賽第一組別的隊伍，擁有至2024年的參賽資格；餘下4隊為挑戰隊伍，其中一個名額每年均會作出調整，需參加國家挑戰杯競爭。<ref>{{Cite web|url=https://www.fivb.org/viewPressRelease.asp?No=73085&Language=en#.WpUIIB3miEc|title=FIVB announces the Volleyball Nations League}}</ref>另外，國家聯賽的分站賽數目由大獎賽時的三站增加到五站、各隊圴會在賽季中交手一次，避免球隊之間重複交手的現象。

== 起源 ==
在2017年10月13日，國際排聯推出了一項名為排球國家聯賽 (VNL) 的新比賽，並於2018年首次亮相。通过先进技术，创新和数字广播的界限，观众将目睹多元化的角度，展现每个球员的世界级运动能力。

FIVB, IMG和21個國家聯合會合作男子排球國家聯賽和女子排球國家聯賽的比賽。每隊至少打15場比賽，整個賽季總共打130場比賽。每個核心队伍都將舉辦至少一次分站賽，以向世界各地展示新的競賽。<ref>{{Citeweb|url=http://www.sportspromedia.com/news/fivb-to-launch-volleyball-nations-league|title=FIVB to launch Volleyball Nations League}}</ref>这是国际排联首次与IMG合作，将全面控制内容制作，最大限度地提高质量，并推出场内讲述，对球迷来说更容易接触和理解。

2018年2月22日，國際排聯宣佈首屆女排國家聯賽的決賽舉辦城市，最終由[[中國|中國]][[南京|南京]]承辦，並宣佈南京將連續三年承辦決賽輪。

== 賽制 ==
賽事分為兩輪，分站賽及總決賽，決賽主辦國的球隊會自動獲得六強名額。國家聯賽比賽歷時六週，前五週屬分站賽賽事，最後一週屬總決賽。分站赛主要在周二、周三、周四进行，分站賽每個分站均有四隊參加單循環賽事。對每一隊而言，每個分站的对手都不一樣，每支球隊都會與其他15支球隊交手，而同一週均有四個分站賽事進行。每隊都要先經5個分站共15場比賽，<ref>{{Citeweb|url=http://sports.sina.com.cn/others/volleyball/2018-02-25/doc-ifyrwsqi1148957.shtml|title=国家联赛中国女排将辗转五地 15个对手由弱到强}}</ref>
最後成績最佳的首5名（不包括總決賽主辦國）連同總決賽主辦國隊伍進入總決賽輪。 <ref>{{Citeweb|url=http://www.worldofvolley.com/News/Latest_news/FIVB/88188/this-is-the-official-format-of-new-volleyball-league-2018-both-mens-and-womens.html|title=official format of New Volleyball League 2018, both men's and women's}}</ref>总决赛採用分组加交叉的赛制。

===預賽輪（分站賽）===
16支球队经过抽签后，比赛将会分20組進行，每支球队都与其他15支球队交手。而每支球队都是分站賽东道主之一，均会承繼至少一個分站賽事，其中部分核心球队将不止有一次成为分站賽东道主的机会。所有比赛完成之后，排名前5名的球队，与决赛的东道主球队参加最终的决赛。在所有比賽結束後排名最后的挑戰球队將被淘汰，其名額由國家挑战杯的冠軍球隊取代。<ref>{{cite web |url=http://www.fivb.com/en/about/news/fivb-executive-committee-embraces-digital-transformation?id=73296 |title=FIVB Executive Committee embraces digital transformation |date=2017年12月7日 |publisher=[[國際排球總會|國際排聯]] |accessdate=8 February 2018}}</ref>

===决赛轮===
6支晋级决赛的球队将分为两个小组，每个小组3支球队。小组前两名直接晋级半决赛。每个小组第一将对阵小组第二，并最终决出冠军。

== 參賽資格 ==
賽事有16支球队参加。 其中十二队的核心队伍無須经历降级，大部分均是世界排名前十五名的隊伍。 其他四支球队被选为挑战队伍，可能会被淘汰。12支核心队无论排名如何都将是2018年至2024年的参赛球队，4支挑战队排名前3名将保留参赛资格，最后一名的名額將由國家挑戰杯的冠军取代。

== 國家挑战杯 (Challenger Cup) ==
國際排聯的排球國家挑戰杯與排球國家聯賽同時舉行，旨在讓更多隊伍能以挑戰球隊身份參與VNL。 挑戰杯由不參加本屆排球國家聯賽的球隊組成，包括來自五大洲地區聯合會的五支球隊和一支主辦國主隊，地區及洲份聯合會可通過組織洲份資格賽，或使用現有的比賽來確定參加排球國家挑戰杯的球隊，其中歐洲獲得兩個席位。賽事會採用小組賽形式進行，之後經過半決賽，最後在決賽的勝者將獲得參加下屆排球國家聯賽的參賽資格。
