{{日本作品|女友＊步驟|カノジョ＊ステップ}}是[[HOOKSOFT|SMEE]]在2016年9月30日<ref>{{Cite web|url=https://www.game-style.jp/soft/backnumber/201607/28/01kst_sm.php|script-title=ja:愛しい彼女との関係をステップアップさせながら、理想の恋人生活を手に入れよう！ 『カノジョ＊ステップ』9月30日発売！ |publisher=Game-Style|date=2016-07-28|accessdate=2018-04-19|language=ja}}</ref>發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]。2018年3月21日由[[GN_Software|GN Software]]發售[[PlayStation_Vita|PlayStation Vita]]版本<ref>{{Cite web|url=https://www.jp.playstation.com/games/kanojo-step-psvita|script-title=ja:カノジョ＊ステップ |publisher=PlayStation官網|accessdate=2018-04-19|language=ja}}</ref>。

== 故事簡介 ==
小早川秋一直都跟著姐姐小早川槙過著到處搬家的生活。在升上高二的春天，秋跟槙到了奶奶在鄉下「白鷺町」經營的民宿「牧田莊」居住。生活穩定下來後，槙向秋說這次是最後一次的搬家，並要秋積極去追尋自己的戀愛<ref>{{Cite web|url=https://www.game-style.jp/special/backnumber/201608/04/88kst_sm.php|script-title=ja:『カノジョ＊ステップ』ストーリー|publisher=Game-Style|date=2016-08-04|accessdate=2018-04-19|language=ja}}</ref>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|小早川秋|小早川 秋（こばやかわ あき）}}
: 作品男主角，在高二的春天時和姐姐槙一起搬到奶奶家的民宿生活。非常害怕蟲。
; {{nihongo|華野椎名|華野 椎名（かの しいな）|聲：柚木サチ}}
: 白鷺學園三年級女學生，學生會長，是秋來到白鷺町後第一個認識的人，家裡為神社。
: 深得學生信賴，但會偷偷騎校規不允許的機車上學。
; {{nihongo|如月乃繪|如月 のえ（きさらぎ のえ）|聲：歩サラ}}
: 白鷺學園二年級，秋的同學，個性穩重的少女，被秋稱為乃繪老師。
: 圖書委員，假日時會在學校的圖書館讀書。
; {{nihongo|柳明日香|柳 明日香（やなぎ あすか）|聲：木住葵}}
: 白鷺學園二年級，秋的同學，個性活潑的少女，家裡經營烏龍麵店。
: 運動神經優秀，是新體操社的王牌。
; {{nihongo|芹澤久遠|芹沢 久遠（せりざわ くおん）|聲：桜乃ひよ}}
: 白鷺學園一年級女學生，比秋早幾天搬到白鷺町的轉學生。

=== 其他角色 ===
; {{nihongo|岡田富子|岡田 とみ子（おかだ とみこ）|聲：三井京子}}
: 可攻略角色，白鷺學園二年級女學生。
; {{nihongo|小早川槙|小早川 槙（こばやかわ まき）|聲：[[川島梨乃|川島梨乃]]}}
: 秋的姐姐，在搬到白鷺町後擔任白鷺學園的特別講師。
; {{nihongo|安達健勝|安達 健勝（あだち けんしょう）|聲：夏村伊介}}
: 白鷺學園二年級男學生，秋在前一個學校的同學，跟秋同一天轉學到白鷺學園。
; {{nihongo|八木正吾|八木 正吾（やぎ しょうご）|聲：どっこいしょーいち}}
: 白鷺學園二年級男學生。染金髮並穿著不合校規的紅色襯衫。雖然看似不良少年，但其實並非壞人。
; {{nihongo|環奈奈|環 奈々（たまき なな）|聲：春山色葉}}
: 白鷺學園二年級女學生，秋的同學，班上的氣氛製造者。
; {{nihongo|春日井桃美|春日井 モモ美（かすがい ももみ）|聲：夢野ぼたん}}
