《女主角失格》（{{lang-ja|ヒロイン失格}}）是日本漫画家[[幸田百子|幸田百子]]的[[少女漫画|少女漫画]]作品。自2010年4月號開始于[[集英社|集英社]]杂志《瑪格莉特別冊》上连载至2013年4月。

2012年獲[[講談社漫畫賞|講談社漫畫賞]]提名（但最終落選）。根据漫画改编，由[[桐谷美玲|桐谷美玲]]、[[山崎贤人|山崎贤人]]、[[坂口健太郎|坂口健太郎]]共同主演的真人电影<ref>{{cite web|script-title=ja:幸田もも子「ヒロイン失格」桐谷美玲主演で映画化、劇中には変顔シーンも|url=http://natalie.mu/comic/news/136952|website=[[Natalie_(網站)|Natalie]]|accessdate=2016-04-03|date=2015年1月26日}}</ref>於2015年在日本上映。

== 故事簡介 ==
每个女孩都有一个梦想，就是想要成为爱情故事女主角的。 深信自己才是利太心中真正的女主角的羽鸟。但实际情况并非如此甜蜜……

== 主要登場人物 ==
; {{nihongo|松崎羽鸟|松崎 はとり / まつざき はとり}}
:女主角。寺坂利太的青梅竹马。暗恋从小一起长大的寺坂利太。第一人稱是あたし。毫无疑问地相信自己绝对是利太的女主角，但却被毫不起眼的同班同学 - 安達未帆抢走了她的位置。对于这事实产生深深嫉妒。血型是A型{{NoteTag|1=2015年7月4日，原作者在推特里注明}}。

; {{nihongo|寺坂利太|寺坂 利太 / てらさか りた}}
:男主角。松崎羽鸟的青梅竹马。曾帮助过被欺负的安達未帆，之后和自己告白的安達交往。其实喜欢羽鸟但没自觉。血型是O型{{NoteTag|1=2015年7月4日，原作者在推特里注明}}。

; {{nihongo|安達未帆|安達 未帆 / あだち みほ}}
:利太的女朋友。在班上是一名毫不起眼的眼镜女。曾被寺坂利太帮助过，之后向他告白并且交往。

; {{nihongo|弘光廣祐|弘光 廣祐 / ひろみつ こうすけ}}
:学校受欢迎的男生，长得帅却性格略显轻浮，但是认真起来却十分专一。对任何事情都是抱以看热闹的态度，觉得很有趣开始接近羽鸟，却渐渐开始喜欢上她了。

== 真人電影版 ==
由女演员[[桐谷美玲|桐谷美玲]]主演，是2015年少女漫画真人版中的最卖座电影。该片截至2015年11月3日已有199万人进入影院观看，票房收入突破23.3亿日元。<ref>{{Cite news|url=http://japan.people.com.cn/n/2015/1106/c35468-27786983.html |title=桐谷美玲主演《女主失格》大受欢迎 少女漫画真人版中No.1 |publisher=人民網 |date= }}</ref>
