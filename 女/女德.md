'''女德'''是指中国社会在2000年后兴起的[[思想|思想]]，内容多以要求女性[[守贞|守贞]]、忍受[[家暴|家暴]]、放弃职业追求为主。宣讲者以开设女德班或讲座的形式推广，并从中获利<ref name="a1"/>。主流舆论对此类思想多有批评，有观点认为其是束缚女性、奴役女性、压迫女性的歪理邪说<ref>{{Cite web |url=http://www.guancha.cn/Education/2017_05_22_409561_1.shtml |title=中国妇女报：某些人宣扬的所谓“女德”，已经触碰性别平等的文明底线 |accessdate=2017-11-16 |author= |date=2017-05-22 |work= |publisher=|language=Zh-hans }}</ref>。但同时有评论指出此类思想的兴起，符合社会实情，预示中国社会将放弃[[男女平等|男女平等]]原则，恢复[[男尊女卑|男尊女卑]]的社会形态<ref name="a1">{{Cite web |url=http://news.newseed.cn/p/1333170 |title=女德背后的捞金产业链：一个丁璇倒下了，千千万万个丁璇站起来 |accessdate=2017-11-16 |author=浪潮工作室：杨红 |date=2017-06-06 |work= |publisher=新芽网|language=Zh-hans }}</ref>。

== 背景 ==

[[中国传统社会|中国传统社会]]推崇[[男尊女卑|男尊女卑]]，提倡[[三从四德|三从四德]]，更片面要求女性[[守贞|守贞]]。[[寡妇|寡妇]]丧偶再婚，长期被[[污名化|污名化]]，基本人权丧失。而在家庭生活中，奉行儒家的[[三纲五常|三纲五常]]，推行[[封建家长制|封建家长制]]，男性及长辈握有主导权。

[[辛亥革命|辛亥革命]]后，随着社会巨变，追求[[男女平等|男女平等]]的[[中國女性主義|女性主义在中国萌芽]]。女性开始走出家庭，参与社会工作成为职业女性。中华人民共和国建立后，[[中华人民共和国政府|政府]]推行男女平等，女性权益得到提高，但[[男尊女卑|男尊女卑]]、[[重男轻女|重男轻女]]思想仍普遍存在。仅就财产[[继承权|继承权]]一项而言，2015年的调查显示<ref>{{Cite web |url=http://yancheng.house.qq.com/a/20150610/034725.htm |title=都是亲生的，为啥房子只传男不传女|accessdate=2018-12-18 |author= |date=2015-06-10 |work= |publisher=腾讯网|language=Zh-hans }}</ref>，民众普遍将女儿排除出继承人范围{{efn|通过遗嘱来指定继承人，可以避免被继承人逝世后，无遗嘱继承，按《[[:s:中华人民共和国继承法|中华人民共和国继承法]]》进行法定继承时，配偶、女儿和儿子平等的继承财产。}}。社会中，片面要求女性[[守贞|守贞]]的言论至今不绝。2000年后，在低生育率和[[老龄化|老龄化]]背景下，社会号召女性回归家庭。但实际上，中国社会并不承认[[家庭主妇|家庭主妇]]的劳动价值。2010年，[[全国政协委员|全国政协委员]][[张晓梅_(1963年)|张晓梅]]提出“建议家务劳动工资化，切实保障女性权益<ref>{{Cite web |url=http://www.chinanews.com/cul/news/2010/03-03/2148053.shtml |title=政协委员张晓梅回应“女性家务劳动工资化”质疑|accessdate=2018-12-18 |author=记者：姜妍、天蓝，实习生：徐娴 |date=2010-03-03 |work=编辑：张中江 |publisher=中国新闻网，来源：新京报|language=Zh-hans }}</ref>”提案时，除了质疑之外，更有无数的嘲讽和挖苦<ref>{{Cite web |url=http://news.163.com/10/0315/09/61QC1PRK00012Q9L.html |title=长平：家庭主妇劳动的价值真需要重估|accessdate=2018-12-18 |author=长平 |date=2010-03-15 |work=责任编辑：王晓易_NE0011 |publisher=网易，来源：东方早报(上海)|language=Zh-hans }}</ref>。
