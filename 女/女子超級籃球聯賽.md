'''女子超級籃球聯賽'''（'''Women's Super Basketball League'''），簡稱'''WSBL'''，是[[台灣|台灣]]的半職業女子籃球聯賽，成立於2004年，目前有[[國泰人壽女子籃球隊|國泰人壽]]、[[台元女子籃球隊|台元纺织]]、[[中華電信女子籃球隊|中華電信]]、[[台電女子籃球隊|台湾電力]]等四支球隊組成，為現今台灣女子籃球最高級別的聯賽。

== 簡介 ==
* 成立於2004年，由[[國泰人壽女子籃球隊|國泰人壽]]、[[台元女子籃球隊|台元]]、[[中華電信女子籃球隊|中華電信]]、[[台電女子籃球隊|台電]]4隊組成，[[佛光大學|佛光大學]]、[[艾柏斯女子籃球隊|艾柏斯]]先後於2009年、2010年加入聯賽，之後又陸續離開。

* 為配合國際賽事，以至於每一季的開打時間皆不固定。

* 由於台灣對女籃關注力較男籃弱，導致沒有固定的轉播單位，也不是每一季都有轉播。

* 由於對岸[[WCBA|WCBA]]放寬球員資格，目前已經有12位台將確定將西進挑戰，籃協替WSBL找專屬行銷團隊。

== 重大事件年表 ==
*2004年，WSBL成立。
*2009年，佛光大學加入聯賽。
*2010年，國泰人壽女子籃球隊退出聯盟，賽季只有4隊比賽。
*2010年，[[艾柏斯女子籃球隊|艾柏斯]]正式投入WSBL，資深裁判[[馮同璋|馮同璋]]擔任經理，[[李鴻棋|李鴻棋]]掛技術顧問一職，北教大教練[[蔡博隆|蔡博隆]]接掌兵符，球隊陣容除了已確定的[[姜憶蓮|姜憶蓮]]與[[何珊珊|何珊珊]]外，[[文祺|文祺]]也繳交了意願書。艾柏斯女籃在當年11月初對外公布完整名單。
*2010年，[[中華民國籃球協會|中華民國籃球協會]]秘書長黃朝和今天表示，WSBL將於明年1月10日在[[臺北體育館|台北市立體育館]]展開。除了[[台元紡織女子籃球隊|台元纺织]]、[[國泰人壽女子籃球隊|國泰人寿]]、[[台灣電力女子籃球隊|台湾電力]]、[[佛光大學|佛光大學]]、[[中華電信女子籃球隊|中華電信]]外，今年成立的[[艾柏斯女子籃球隊|艾柏斯]]首度報名參賽，共有6支球隊參加。本季所有賽程都在台北市跟新北市。每隊規劃20場例行賽，例行賽的1、2名直接爭總冠軍，3、4名則爭季軍，總冠軍戰跟季軍戰都是3戰2勝制。
*2011年，WSBL原訂3月20日起將在[[臺北體育館|臺北體育館]]展開後續賽事，因場館檔期安排因素，本季後續賽事將自3月21日起全部移往[[新莊體育館|新莊體育館]]進行；三戰二勝制決賽則更改至4月11日至14日進行。原訂3月30日將舉辦[[國泰人壽女子籃球隊|國泰]]女籃老將[[錢薇娟|錢薇娟]]的引退活動亦順延至3月31日。
*2011年，[[艾柏斯女子籃球隊|艾柏斯]]結束和北教大的合作，僅成軍一年即走入歷史。至於不是北教大學生的球員則取得自由球員身分，可以到其他球隊。
*2011年，[[佛光大學|佛光大學]]因為傷兵導致球員不足，暫時退出女子超級籃球聯賽，而[[顏嘉萱|顏嘉萱]]、[[陽詩慧|陽詩慧]]等幾位陣中好手下季將借將給[[中華電信女子籃球隊|中华電信]]隊。
*2011年，效力於[[台元女子籃球隊|台元纺织]]女籃的國手[[劉君儀|劉君儀]]宣布退休；而[[姜鳳君|姜鳳君]]新球季將重新披上[[國泰人壽女子籃球隊|國泰人寿]]球衣。
*2018年，「籃球精靈」[[文祺|文祺]]和「小南」[[徐千惠_(籃球運動員)|徐千惠]]確定新球季將重新披上[[中華電信女子籃球隊|中华電信]]，中華電信女籃與台灣師大女籃延續合作，雙方簽署合作發展契約，電信女籃與台師大女籃、南山高中女籃3合1模式宣告正式成立。9月台灣女籃掀起西進潮，繼黃利英加盟河南後，今年在中華電信打得火熱的劉希曄和林仙芳將一起前往WCBA陝西天澤女籃。陝西天澤女籃在WCBA是相當新的隊伍，2017-2018才由資格賽出線，陝西女籃和中國CUBA清華大學合作，球員平均年齡僅21歲。劉希曄和林仙芳的加入可提升陝西女籃的經驗值。

==女子超級籃球聯賽季後賽==
賽制：
第一季(2005年)至第十一季採例行賽的1、2名直接爭總冠軍，3、4名則爭季軍，總冠軍戰跟季軍戰都是3戰2勝制。

第十二季新增三戰二勝制的交叉準決賽，由例行賽第一名對決第四名、第二名出戰第三名，決賽依舊採取三戰二勝制。

== 台灣女籃特殊文化 ==

==選秀==
2018年12月18日[[中華籃協|中華籃協]]召開WSBL球團會議研議選秀辦法內容，經會議中達成共識108年度WSBL開始辦理新人選秀，凡具有[[高級中等學校籃球聯賽|HBL]]甲級、[[大專校院籃球運動聯賽|UBA]]公開一級經歷之球員均可報名，前二輪獲選球員可獲得薪資保障，各球隊亦可在建教合作學校中優先保留球員。 選秀方式與[[SBL|SBL]]相同依據前一季例行賽名次，每輪依序由上季最後一名先選，次由最後第二名接續遴選球員，直至所有球隊放棄為止。薪資待遇第一輪獲選球員保障月薪45000元、第二輪獲選球員保障月薪40000元，如入選國際賽中華隊正選名單(瓊斯盃、世大運、青年隊除外)薪資保障月薪55000元起。 另針對國內女籃生態，明年度各球隊可保留五位建教合作學校之球員，109年度每隊保留三位，110年度起不保留，將在實施前視當時現況再行研議。
