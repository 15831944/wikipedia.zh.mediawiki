{{Vgname|女神異聞錄3|ja=ペルソナ3|en=Shin Megami Tensei: Persona 3|diff=yes}}，簡稱「'''P3'''」，是日本[[電子遊戲|電子遊戲]]公司[[Atlus|Atlus]]於2006年7月13日發售的[[PlayStation_2|PlayStation 2]][[電子角色扮演遊戲|角色扮演遊戲]]，《[[女神異聞錄系列|女神異聞錄系列]]》的第三部主要作品，與同系列的前作相隔七年。主題顏色為「藍色」。'''《Persona 3 FES》''' (日版名：ペルソナ3フェス Perusona Surī Fesu)在日本于2007年4月19日发售。[[PlayStation_Portable|PlayStation Portable]]版的移植作{{Vgname|女神異聞錄3 携带版|ja=ペルソナ3  ポータブル|en=Shin Megami Tensei: Persona 3 Portable |diff=yes}}于2009年推出，其針對《女神異聞錄3》作了修改，大幅利用了在《[[女神異聞錄4|女神異聞錄4]]》大受好評的設計，并在遊戲內追加了女性主角。

== 概要 ==
:距離前作發售相隔6年，《女神異聞錄3》採用全新的設定，加入時間經過與人際關係系統，加重學園的日常生活的部分，故事時間為2009年4月。由於《[[真・女神轉生III-NOCTURNE|真・女神轉生III-NOCTURNE]]》的製作團隊製作的「Press Turns」及全書等系統，在《真III》時獲得好評，《Persona 3》亦採用同樣系統。在收到玩家的要求，於2007年4月發售收錄了追加劇情《[[女神異聞錄3#女神異聞錄3_FES|女神異聞錄3 FES]]》。
:其後推出電話遊戲《[[Aegis_THE_FIRST_MISSION|Aegis THE FIRST MISSION]]》、《[[女神異聞錄3em|女神異聞錄3em]]》。
:於2008年1月開始播放以女神異聞錄3為原案的電視動畫《[[女神異聞錄_～三位一體之魂～|PERSONA -trinity soul-]]》，時間是《Persona 3》10年後的故事。

:《女神異聞錄3》是以學園城市為舞台，敘述擁有稱為「Persona」特殊能力的少年少女的角色扮演遊戲。隨著《[[女神異聞錄2|女神異聞錄2]]》的完結，《女神異聞錄3》採用全新的世界設定、登場人物、遊戲系統及故事背景。遊戲分為白天與朋友一起的學園生活及夜晚與在「影時間」出現的神秘怪物「shadow」的戰鬥。

:除了與前作一樣可以培養Persona 外，本作還可以將Persona 合體成為新的Persona。本作由《[[真·女神转生III－Nocturne|真·女神轉生III]]》的團隊製作，同樣加入技能繼承和技能轉換系統，使Persona的培養更加深入。戰鬥系統是由《真·女神轉生III》的「Press Turns」戰鬥系統改良，亦而加入女神轉生系統中受歡迎的要素，例如天鵝絨房間（Velvet Room）加入Persona全書等。

== 故事劇情 ==
'''Episode Yourself'''{{noteTag|美版：'''The Journey'''}}

故事時間為2009年4月。10年前因事故而失去雙親而由親戚養育的[[主角_(女神异闻录3)|主角]]，因為家庭的事情而轉校到小中一條龍學校「私立月光館學園」的高中部中。在入住學生宿舍後不久，他被奇形怪狀的怪物「陰影（Shadow）」襲擊，令隱藏在他體內Persona的能力甦醒。因為這個契機，主角被迫知道這個被掩蓋的世界真實的一端。

那就是「影時間」，一天與另一天之間存在著的隱藏時間，「陰影」就是這個影時間的居民。在影時間中，普通人會變成棺柩形的形態，失去這時的記憶（這稱為「象徵化」）。所以大部分的人都不知道影時間的存在，但有一少部份人，在影時間還保持著人的形態，那些人會被陰影吃掉他們的精神，而成為無力狀態（陰影人）。

在月光館學園理事長幾月修司下，組成使用Persona的陰影討伐部隊「特別課外活動部（S.E.E.S.）」。得到特殊Persona能力的主角被部長桐条美鶴看中，成為隊員與特別課外活動部的隊長與危害人類的影作戰。

== 設定及用語 ==
; {{Nihongo|Persona|ペルソナ|Persona}}
: 自己的另一面人格。主角可擁有（身上裝備）最多十二個Persona力量，其餘Persona使用者都各只有一個Persona。
: 在《[[女神異聞錄4|Persona4 黃金版]]》中，正式中文翻譯為「'''人格面具'''」。

; {{Nihongo|召喚器|召喚器|Evoker}}
: 本作召喚Persona的道具，由桐條集團旗下的研究所開發的，S.E.E.S.的基本裝備。
: 眾人都是使用槍型召喚器，只是表示槍的外型，沒有裝置實彈。
: 虎狼丸由於是一隻小狗，所以使用頸圈型召喚器。
: 埃癸斯由於是由桐條研究所製作的對陰影機器「反陰影特別鎮壓武裝七式」，所以本身已經內置召喚Persona的功能。

; {{Nihongo|影時間|影時間|The Dark Hour}}
: 日與日之間的隔縫，沒有適應能力的人會象徵化成為棺木。

; {{Nihongo|塔耳塔羅斯|タルタロス|Tartarus}}

; {{Nihongo|黃昏的羽毛|黄昏の羽根}}
: 主角專用道具，使用時機是主角瀕死狀態，效果是小隊中、主角以內全員復活，HP／SP全回復，狀態異常解除，「easy」、「BEGINNER」模式使用限定。

; {{Nihongo|Wild|ワイルド|Wild}}
: 是包含了「宇宙」在內，能使用所有塔羅牌Persona的究極的秘儀（Arcana）和將Persona合體的能力。
: 相較於其他Persona，成長的速度異常的緩慢，但只要和同伴結下強烈的羈絆就能提升力量。
: 塔羅牌的編號是「0」，代表著無限的可能性。
: 在《[[女神異聞錄4|Persona4 黃金版]]》中正式中文翻譯為「'''不羈'''」。

; {{Nihongo|Death|デス}}
: 十年前在桐條集團旗下研究所的研究中誕生的陰影，死之宣告者。
: 在不完全的狀態下和埃癸斯決戰，卻被埃癸斯封印。它的力量分裂成十二個大型陰影，只會在滿月時出現。

== 开发与发行 ==
製作人是「[[真・女神轉生III|真・女神轉生III]]」的[[橋野桂|橋野桂]]。角色設計及新的專用Persona設計由{{link-jp|副島成記}}負責，而其他的Persona則是使用惡魔畫師{{link-jp|金子一馬}}的插畫。主題曲是{{link-jp|川村ゆみ}}的《你的記憶（キミの記憶）》。遊戲畫面的設計很有流行的感覺，而背景音樂使用英語主唱的歌曲，圖像與音樂都是過去的作品所沒有的。

本作獲得《[[Fami通|Fami通]]》遊戲大獎2006的優秀獎（角色扮演遊戲獎），而在日本以外亦得到一致好評。

== 女神異聞錄3 FES ==
《'''女神異聞錄3 FES'''》是女神異聞錄3追加碟，「FES」是節慶（festival）的簡稱。收錄描述女神異聞錄3結束後的新故事「Episode Aegis（美版：The Answer）」，而正編「Episode Yourself（美版：The Journey）」追加各種各樣的新要素，亦可以繼承女神異聞錄3的一部份遊戲記錄。本作分為原有女神異聞錄3的玩家的「追加版」及沒有女神異聞錄3的新玩家的「完整版（單體啟動版）」兩種，兩款的封面有異，但追加的遊戲內容一樣。追加版需以《Persona 3》光碟啟動而得到認證，所以價錢相對便宜。

2006年10月20日發售的電玩通中刊登了Aegis的廣告，上面的口號寫著「只要沒有放下我，我們一定再能遇見」。而Persona 3的官方網站中出現「如果沒有賣掉Persona 3的光碟，一定會有好事發生。」意義的預告。最後於2006年12月8日正式發表。

除了本編故事以調整後的型式再推出之外，追加和本編劇情相關的後日談：因為迷宮一下子崩潰，世界反向穿了一個大洞，而不管這一個大洞，世界會不停重複3月31日。而在過程之中，更會看到主人公，風花、Aegis以外的人為何會和影時間扯上關係‥‥‥不過，這個後日談大部份只是追加劇情，對本編沒有影響，既不能使用本編裡的Persona全書，Commu也沒有繼承，而且是使用Aegis進行操作，通常攻擊手段限是貫通系‥‥‥

《'''女神異聞錄3 前夜'''》 (PERSONA3 THE NIGHT BEFORE）是《女神異聞錄3》預先發表的迷你網絡遊戲，供人免費遊玩。現在已停止服務，同時以女神異聞錄系列為基礎製成新的網頁遊戲《[[女神異聞錄_AIN_SOPH|女神異聞錄 AIN SOPH]]》。如其名一樣，是女神異聞錄3的「前夜」，遊戲系統和故事等與P3正編不同，只是體驗女神異聞錄3的世界，每晚可看到如夢般世界的原創故事。玩家需挑戰在港灣人工島中毎夜出現的迷宮之塔「塔爾塔羅斯」，打倒「影」並往頂峰進發為目標。遊戲中行動每次都要消耗「時間」，每日只能在遊戲內活動1小時。玩家可加入抵抗一方或影一方，按照「戰爭」而取得名聲和金錢。
