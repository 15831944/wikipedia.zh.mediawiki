'''女生节'''，是中國大陸的一個節日，日期在定在[[国际妇女节|国际妇女节]]前一天的[[3月7日|3月7日]]，一般认为从中国大学中发展而成，逐渐在全国形成浓厚的节日氛围，又稱為'''三七“校园女生节”'''<ref>{{cite web| url=http://www.chinadaily.com.cn/dfpd/shehui/2012-03/08/content_14790010.htm|title = 三七“校园女生节” 看高校男生怎样献殷勤(图)|publisher = 中国在线|date = 2012-03-08}}</ref>。在中国社会普遍对女性进行[[性别歧视|性别]]、[[年龄歧视|年龄歧视]]的背景下<ref>{{Cite web |url=https://m.huanqiu.com/r/MV8wXzEwMTgyNTkwXzEyNThfMTQ4Nzc2Mjk0MA== |title=中国妇女报:对女性的年龄歧视何时休 |accessdate=2019-03-11 |author=评论咒：莫兰 |date=2017-02-22 |work= |publisher=环球网，来源：中国妇女报|language=Zh-hans }}</ref>，国际妇女节和“[[妇女|妇女]]”一词亦被污名化<ref>{{Cite web |url=http://cul.qq.com/a/20160308/015936.htm |title=除了显老，女生为何不愿过“妇女节”？|accessdate=2018-12-18 |author=李大白 |date=2016-03-08 |work= |publisher=腾讯网|language=Zh-hans }}</ref>、被排斥<ref>{{Cite web |url=http://news.163.com/10/0308/10/618EPA6R000146BD.html|title=80后女性排斥"妇女节" 专家:改个名字就好|accessdate=2018-12-18 |author=|date=2010-03-08 |work=责任编辑：王晓易_NE0011 |publisher=网易，来源：新华网|language=Zh-hans }}</ref>，高校女生逐以女生节替代妇女节，彰显与大龄、已婚<ref name = rmw>{{cite web| url = http://edu.people.com.cn/h/2012/0307/c227696-2566280295.html| title = 厦大传统女生节"驾到" 男生们全都"豁出去"了|accessdate=2019-03-11|author=记者：梁静，通讯员：丁月婷、李珊珊| publisher = 人民网，来源：东南网-海峡导报 | date = 2012-03-07|language=Zh-hans}}</ref>及年老女性的区别，使之成为一个受欢迎的节日<ref name="a20190312">{{Cite web |url=http://news.163.com/10/0307/04/61581IVI00011229.html|title=90后女生不屑过"三八妇女节" 创意"三七女生节"|accessdate=2018-12-18 |author=张梦园、蔡蕴琦|date=2010-03-07 |work=责任编辑：王晓易_NE0011 |publisher=网易，来源: 新华报业网-扬子晚报(南京)|language=Zh-hans }}</ref>。同时，由于女生节对“妇女”一词的回避和起源于“一日之差”的关系，女生节被认为是对女性歧视侮辱的节日<ref name="a20190307">{{Cite web |url=http://www.uschinapress.com/2019/0307/1158050.shtml |title=妇女节快乐还是女生节快乐？ 哪种祝福是别有用心？ |accessdate=2019-03-11 |author= |date=2019-03-07 |work=编辑：邹姆斯 |publisher=侨报网|language=Zh-hans }}</ref>。

== 历史 ==

這個节日的起源极富争议，但大体归于两类，一是来自于高校的学生活动。二是选定3月7日为女生节，盖因其与3月8日妇女节的“一日之差”，即“女生和妇女只差一日<ref>{{Cite web |url=https://www.worldjournal.com/6167821/article-%E7%A5%9D%E3%80%8C%E5%A5%B3%E7%94%9F%E7%AF%80%E5%BF%AB%E6%A8%82%E3%80%8D-%E6%BC%AB%E5%A8%81%E9%9B%BB%E5%BD%B1%E7%82%BA%E4%BD%95%E6%83%B9%E6%80%92%E4%B8%AD%E5%9C%8B%E7%B6%B2%E5%8F%8B-2/|title=祝「女生節快樂」 漫威電影為何惹怒中國網友? |accessdate=2019-03-11 |author=中国新闻组 |date=2019-03-08 |work= |publisher=世界新闻网|language=Zh-hant }}</ref><ref name="b20190307"/>”的[[双关语|双关语]]。汉语口语中，“日”亦指“[[做爱|做爱]]”。“一‘日’之差”除指时间上的“一‘日’之差”外，又被民众普遍的理解<ref name="b20190307">{{Cite web |url=http://www.jiaodong.net/news/system/2011/03/07/011157362.shtml |title=大学女生拒绝被"三八" 提前一日庆祝女生节(图) |accessdate=2019-03-11 |author=记者：焦冬 |date=2011-03-07 |work=责任编辑：沈广安 |publisher=胶东在线|language=Zh-hans }}</ref>为女生（[[处女|处女]]）经历一次[[性行为|性行为]]后成为妇女。

多种起源说法外，进入21世纪，女生节经过全中國大陸的各級高校学生会或民间的引进，女生节在高校中逐渐风靡。另外，在十月下旬各大高校也会举办“女生节”帮助学生在[[光棍节|光棍节]]前脱[[单身|单]]。

2010年代，[[中国微博|微博開始流行以後]]，其中的精彩创意，往往热闹非凡，很多活动的照片被上传到网络上并通过微博等媒介广为流传，使得节日的人气愈加旺盛。<ref name = gzrb>{{cite web|url = http://cq.qq.com/a/20120308/000252.htm|title = 女生节高校女生微博晒福利(组图)|publisher = 广州日报| date = 2012-03-08}}</ref>。

2011年，[[清华大学|清华大学]]和[[北京大学|北京大学]]的学生在校园内悬挂的女生节条幅由于妙语连珠在网络上走红<ref>{{cite news |language = zh | author = 网络 | url = http://www.mitbbs.com/news_wenzhang/Headline/31500931.html | title = 清华PK北大 "女生节"谁的横幅最有才?}}</ref>，标志着女生节已经开始走出校园，进而成为全社会的一个节日{{cn}}。

此后数年，女生节被商家，特别是[[电商|电商]]所青睐，成为一个针对女性进行营销的节日，有电商的[[大数据|大数据]]报告指出女生节比妇女节更具消费潜力<ref>{{Cite web |url=http://tech.huanqiu.com/net/2016-03/8652138.html?agt=62 |title=百度糯米大数据：3.7女生节比妇女节更具消费潜力 |accessdate=2019-03-12 |author= |date=2016-03-04 |work=责编：李文瑶 |publisher=环球网，来源：中国经济新闻网|language=Zh-hans }}</ref>。

== 宗旨 ==
這個節日的宗旨是关爱女生，展现女生的风采。期间通常会由学生会或者学生自发组织进行包括聚会，讲座，向女生赠送礼物，唱情歌，灯光组字，悬挂横幅，提供“呵护”女生服务等种种活动<ref name = rmw/><ref name = wy>{{cite web|url=http://news.163.com/12/0308/09/7S2JAEOO00014AEE.html|title = 那些为我们过女生节的男生 谢谢(组图)|publisher = 网易|date = 2012-03-08}}</ref><ref>{{cite web| url=http://www.chinadaily.com.cn/dfpd/shehui/2012-03/08/content_14790010.htm|title = 三七“校园女生节” 看高校男生怎样献殷勤(图)|publisher = 中国在线|date = 2012-03-08}}</ref><ref>{{cite web| url=http://www.chinanews.com/cul/news/2010/03-07/2155763.shtml|title = 中国90后乐过“三七女生节”|publisher = 中国新闻网|date = 2010-03-07}}</ref>。

== 反應 ==
大多数女生对此一节日表示欢迎，称十分感动于男生的殷勤和热心<ref name = wy/><ref name = gzrb/>。2010年，[[南京师范大学|南京师范大学]]金陵女子学院教授金一虹接受采访时，表示女生节“是值得支持和提倡的一件事情。”“是一个符合大学生朋友心理，体现大学生向善、向美、向上的有创意的好节日！值得受到推广。<ref name="a20190312"/>”

除了正面内容和反应外，部分高校拉的女生节横幅内容低俗，如“自从大一认识你，从此电脑无AV！”“女生节，晚归要放开，不归要张开”，充斥着性意淫的意味，但校方一般不会出面干涉<ref name="李思磐" /><ref>{{cite web|title=反三七过三八:防男生集体骚扰|url=http://top.sina.cn/edu/2016-03-07/tnews-ifxqafha0441591.d.html?oid=139&vt=4&wm=ty10&cid=6652|website=新浪教育|accessdate=2017-03-10|date=2016-03-07}}</ref>。同时，商家在女生节前后的营销，更期推动“女生节取代妇女节”。2016年，某家电商举办第三届3.7女生节时，于3月3日播放范湉湉呼吁女生节取代妇女节的视频<ref>{{Cite web |url=http://tech.huanqiu.com/internet/2016-03/8646603.html?agt=1 |title=打造3.7女生节 百度糯米发布女性消费大数据报告 |accessdate=2019-03-12 |author= |date=2016-03-03 |work=责编：周涛 |publisher=环球网|language=Zh-hans }}</ref>。前一日，此视频曾网络引起争议，亦称获得了“一边倒的支持”<ref>{{Cite web |url=http://eladies.sina.com.cn/2016-03-04/1226/doc-ifxqafha0362483.shtml |title=范湉湉喊话拒过妇女节引爆微博 百度糯米3.7女生节火了 |accessdate=2019-03-12 |author= |date=2016-03-04 |work= |publisher=新浪网|language=Zh-hans }}</ref>。

女权组织[[新媒体女性|新媒体女性]]于2016年发起“反三七过三八”的话题，认为“三七女生节”长期政治化，不断在年轻男女当中强化性别不平等的文化规范，使人们渐渐遗忘妇女节的女权意义，呼吁人们回归妇女节，关注男女平等权利<ref name="李思磐">{{cite web|title=李思磐：妇女节变成女生节？当中国女权遇上营销和去政治化|url=https://theinitium.com/article/20170308-opinion-lisipan-international-womens-day/|website=端传媒|accessdate=2017-03-10|date=2017-03-08}}</ref><ref>{{cite web|title=我們跟「反三七過三八」發起人聊了聊女權主義|url=https://read01.com/gRMoQa.html|website=壹读|accessdate=2017-03-10|date=2016-03-08}}</ref>。

2016年时，[[复旦大学|复旦大学]][[复旦大学社会发展与公共政策学院|社会发展与公共政策学院]]副教授沈奕斐撰文指出，从妇女节到女生节，背后是通过年龄、容貌、性、生育等条件把女性分为三六九等，在分等的过程中形成了隐性歧视，即年轻漂亮的“女生”被放在了等级最高的位置<ref>{{Cite web |url=https://www.shobserver.com/news/detail?id=10192 |title=妇女节VS女生节，争的是什么 |accessdate=2019-03-11 |author=沈奕斐 |date=2016-03-08 |work= |publisher=上观网站|language=Zh-hans }}</ref><ref name="a20190307"/>。
