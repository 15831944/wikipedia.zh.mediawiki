'''资政'''，是[[中華民國|中華民國]]、[[新加坡|新加坡]]、[[緬甸|緬甸]]等國的一項高級官銜，乃使退職之高级官員或者是德高望重者继续參與、輔助國政的崇高職銜。

== 起源 ==
[[古代中国|古代中国]]曾經設有名稱相近的官职，例如[[資政大夫|資政大夫]]、[[資政殿大學士|資政殿大學士]]。

== 中華民國 ==
在中華民國，[[中華民國總統府資政|總統府資政]]簡稱資政，与[[中華民國總統府國策顧問|国策顾问]]、[[中華民國總統府戰略顧問|戰略顧問]]於民間雅稱為「[[三公|三公]]」，來自各專業領域、為國建言。三者皆是[[总统|总统]]必要时请益、[[谘询|谘询]][[公共政策|公共政策]]的人士。一般任期为一年，為無給薪[[榮譽|榮譽]]職。<ref name = "中華民國總統府資政_2018"> {{ cite web | title = 中華民國總統府 | url = https://www.president.gov.tw/Page/109 | language = zh-tw | accessdate = 2018-07-17 }} </ref>

依據《中華民國總統府組織法》第十五條之規定，總統府資政為無給職，聘期不得逾越總統任期，人數不得逾三十人。<ref name = "中華民國總統府_2018"> {{ cite web | title = 中華民國總統府組織法 | url = https://www.president.gov.tw/Page/284/1459/%E4%B8%AD%E8%8F%AF%E6%B0%91%E5%9C%8B%E7%B8%BD%E7%B5%B1%E5%BA%9C%E7%B5%84%E7%B9%94%E6%B3%95 | language = zh-tw | accessdate = 2018-07-17 }} </ref>

== 新加坡 ==
新加坡的资政分为[[内阁资政|内阁资政]]（Minister {{link-en|Mentorship|Mentorship}}，原译高级部长）和[[国务资政|国务资政]]（Senior Minister）。其中内阁资政祇设一位，由[[李光耀|李光耀]]任职；李光耀因此受[[新加坡|新加坡]]人尊称为李资政。资政是[[总理|总理]]或国家領導人退職後方能擔任，是仅次于国家[[总统|总统]]、[[总理|总理]]之下的国家[[领袖|领袖]]。

[[新加坡|新加坡]]的资政是参照[[中華民國|中華民國]]的职级，赋予了“资政”新内涵，於是成為[[新加坡|新加坡]]和[[中華民國|中華民國]]的通用名词。

== 中華人民共和國 ==
[[改革开放|改革开放]]之初，曾經有[[中央顾问委员会|中央顾问委员会]]由老[[干部|干部]]所組成，中顾委的委员亦是当时现任的黨和國家領導們諮詢國政的重要對象。目前部分超龄退休的地方政府官员会转任资政发挥余热。

== 缅甸 ==
{{main|国务资政}}
[[国务资政|国务资政]]是[[缅甸|缅甸]]自2016年4月起设立的职务，任期与[[缅甸总统|缅甸总统]]一样为五年，被外界视为相当于实行[[总统制|总统制]]前的[[缅甸总理|缅甸总理]]职务，首任资政为[[昂山素季|昂山素季]]。
