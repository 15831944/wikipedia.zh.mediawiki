'''資產階級'''（{{lang-fr|'''bourgeoisie'''}}，音譯作'''布爾喬亞'''）是根據一些[[西方世界|西方]][[经济学|经济学]]思想學派，尤其是[[马克思主义|马克思主义]]，為[[资本主义|资本主义]][[社会|社会]]所做的[[社会阶级|階級]]劃分當中的階級之一。

== 辭源 ==
資產階級的[[英语|英语]]“{{lang|en|bourgeoisie}}”來自[[法语|法语]]，源於[[意大利语|意大利语]]的“{{lang|it|borghesia}}”，而後者又是源於從[[希腊语|希腊语]]“{{lang|el|pyrgos}}”演化而來的“{{lang|it|borgo}}”，意思是[[村庄|村庄]]。因此“{{lang|it|borghese}}”就是指在村莊中心擁有房子的自由人。

資產階級出現在[[中世纪|中古時期]]的[[意大利|意大利]]，那時住在村莊中的居民開始變得比住在附近鄉間的人還要富有。因此他們可以獲得相對上較多的權力和影響力，越來越接近統治階級和[[圣职者|神職人員]]，同時逐漸遠離平民階級。這種中古時期的資產階級原型就是[[磨坊|磨坊]]擁有者，他們能夠快速地對當地[[经济|经济]]產生重大影響力，以至於可以對統治者表達否決權。

在接下來的世紀裡，資產階級這個名詞則較適合用來指稱最初的[[銀行|銀行家]]，以及那些從事新興活動如[[貿易|貿易]]和[[金融|金融]]的人。19世紀之前，這個詞大部分情況下指的就是低於貴族、高於農奴和無産階級的廣大人群。

在十九世紀時，資產階級可以指因為賦稅達到相當的地步，在未有[[普選權|普選權]]的歐洲[[代議制|代議制]]國家中，獨佔了政治權利的富人。而且在自己的工廠等場所，資本家更有把工人當作奴隸喚使的權力，以及把賣不出的貨物（包括在飢荒時的食物）消毀的權力，擁有像以前的貴族般的權力，所以成為了[[社會主義|社會主義]]者矛頭指向的對象了。

== 馬克思主義中的資產階級 ==
在[[馬克思主義|馬克思主義]]裡，資產[[階級|階級]]被定義為在生產[[商品|商品]]的資本主義社會中擁有[[生产资料|生产资料]]的階級。馬克思主義認為[[无产阶级|无产阶级]]與資產階級在本質上是互相敵對的。无产阶级付出了全部劳动，理应获得全部的劳动果实。然而資本家卻希望工资能夠越低越好，以提高利润。也就是说，资本家的利益（利润）与无产阶级的利益（工资）之间存在矛盾。资本家在生产中并未付出劳动。劳动价值被资本所剥夺。換句話說，資本家會[[剝削|剝削]]勞工。

马克思主义者认为，「資產階級」是压迫者和剥削者；那些被認為與資產階級合作的人通常被稱作“資產階級的走狗”。

在當代的馬克思主義用語中，資產階級主要是指那些控制了[[公司|公司機構]]的人，控制的方法有透過對公司大多數[[股份|股份]]的掌握、選擇權、[[信托|信托]]、[[基金|基金]]、仲介或關於市場業務的公開發言。因此「資本家」是指財富主要透過“[[投资|投资]]”得來的人，而他們不須要工作以求生，没有劳动就得到了劳动果实，本质上也是通过资本剥削无产阶级。

在現代非馬克思主義的中文用法裡，資產階級（或布爾喬亞）和[[无产阶级|普羅大眾]]（普羅是從無產階級''{{lang|en|Proletarian}}''的頭音而來）時常被用來指稱富人與窮人，而不一定是指擁有或不擁有生產工具的人。

[[階級|階級]]並非一個同質的完整實體，許多分析家會對資產階級進行更細膩的劃分。這些劃分有：
* [[大資產階級|大資產階級]]（high bourgeoisie）：由最富有的人所組成，包括了[[企業家|企業家]]、貿易家、大地主等等。
* [[中產階級|中產階級]]（middle bourgeoisie）：那些擁有固定繼承財產或收入的人，他們比起上層資產階級較不富有。
* [[小資產階級|小資產階級]]（petite bourgeoisie）：占有少量生产资料和财产，主要依靠自己劳动为生，一般不剥削别人的阶级，包括手工业者、小商人、自由职业者等。

== 資產階級的價值觀 ==
[[File:20110420-RD-LSC-0376_-_Flickr_-_USDAgov.jpg|缩略图]]]]
资本家一般被马克思主义者描绘为：
* 奉從主義者（conformism）
* [[保守主义|保守主義者]]
* [[庸俗主義者|庸俗主義者]]（philistinism）
* [[个人主义|個人主義者]]
* 對於個人[[財產|財產]]過度關注
* 對[[宗教|宗教]]的虛偽信仰
* 資產階級民族主義者（Bourgeois nationalism）

在[[资本主义|资本主义]]社会下，资本家主要信奉“[[自由主义|自由主义]]”，但马克思主义者认为，在资本主义社会中，所谓的“自由主义”是资产阶级统治的幌子。

== 中国当代的資產階級情况 ==
{{Refimprove|章节|time=2017-05-26T08:26:06+00:00}} 
因为[[中国共产党|中国共产党]]理论的说法是“在[[中华民国|中华民国]]时的[[中国|中国]]的矛盾是资产阶级和无产阶级的矛盾”<ref>{{cite web|url=http://ks.cn.yahoo.com/question/1406112704242.html|title=人民民主专政是提出来的?|publisher=中国雅虎知识堂|deadurl=yes|archiveurl=https://web.archive.org/web/20070108024244/http://ks.cn.yahoo.com/question/1406112704242.html|archivedate=2007-01-08}}</ref>，而且这两者之间是“不可调和的矛盾”，并称只要[[无产阶级|无产阶级]]夺权后建立起的[[人民民主专政|人民民主专政]]政权经过“[[社会主义|社会主义]]改造”后，资产阶级就会“消亡”。

所以在1956年[[三大改造|三大改造]]后，[[中华人民共和国|中华人民共和国]]官方就认为“资产阶级在中国已不存在”。

[[改革开放|改革开放]]之后中国官方仍沿用此说法即“资产阶级在[[中华人民共和国|中华人民共和国]]已经在[[三大改造|三大改造]]中消亡”，而更多的使用“企业家”<ref>{{cite web|url=http://www.google.com/search?hl=zh-CN&oq=qiyejia&sa=X&oi=pinyin&resnum=0&ct=result&cd=1&q=%E4%BC%81%E4%B8%9A%E5%AE%B6&spell=1|title=google给出关键词“企业家”的搜索结果页面}}</ref>或“实业家”<ref>{{cite web|url=http://www.google.com/search?hl=zh-CN&q=%E5%AE%9E%E4%B8%9A%E5%AE%B6&btnG=Google+%E6%90%9C%E7%B4%A2&lr=|title=google给出关键词“实业家”的搜索结果页面}}</ref><ref>{{cite news|url=http://news.xinhuanet.com/tai_gang_ao/2006-11/15/content_5333205.htm|title=爱国实业家刘鸿生纪念铜像在沪落成|date=2006年11月15日|publisher=新华网}}</ref>来称呼资本家。
