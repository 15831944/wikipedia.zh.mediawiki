'''埃瓦爾德-海因里希·馮·克萊斯特-舒曼森'''（{{lang-de|Ewald-Heinrich von Kleist-Schmenzin}}；{{bd|1922年|7月10日|2013年|3月8日}}<ref>[http://www.finanznachrichten.de/nachrichten-2013-03/26225292-letzter-verbliebene-teilnehmer-des-hitler-attentats-vom-20-juli-gestorben-003.htm 7月20日密谋案最後一名生還者逝世（Letzter verbliebene Teilnehmer des Hitler-Attentats vom 20. Juli gestorben - finanznachrichten.de）]</ref>）是一名出版商，在[[第二次世界大战|二戰]]期間曾經是[[德意志國防軍|德意志國防軍]][[納粹德國陸軍|陸軍]][[中尉|中尉軍官]]，在1940年18歳時加入步兵團<ref>[http://www.guardian.co.uk/world/2013/mar/13/ewald-heinrich-von-kleist 衛報：埃瓦爾德-海因里希·馮·克萊斯特逝世（Ewald-Heinrich von Kleist obituary - guardian.co.uk）]</ref>。他是{{translink|en|Von Kleist|馮·克萊斯特家族}}的成員之一，他的父親{{translink|en|Ewald von Kleist-Schmenzin|Ewald von Kleist-Schmenzin|埃瓦爾德}}曾經為了[[德國人反抗納粹|反抗納粹]]而對抗[[阿道夫·希特勒|希特勒]]。他們父子兩人也一同參與[[7月20日密谋案|7月20日密谋案]]，在生前他是最後一名生還者。

== 生平事蹟 ==

1940年，他在18歳時加入[[德意志國防軍|德意志國防軍]]，成為[[步兵|步兵]]軍官。在軍隊裡，他被[[克勞斯·馮·施陶芬貝格|施陶芬貝格]][[招聘|吸收]]並成為反抗[[納粹德國|納粹]]的成員。1944年1月，原先是由{{translink|en|Axel Freiherr von dem Bussche-Streithorst|Axel Freiherr von dem Bussche-Streithorst|艾克塞爾·男爵·馮·迪·布斯-史特萊特何斯特}}準備執行刺殺任務，帶著他父親給予的祝福，自願與{{translink|en|Axel Freiherr von dem Bussche-Streithorst|Axel Freiherr von dem Bussche-Streithorst|布斯-史特萊特何斯特}}交換，去行刺[[阿道夫·希特勒|希特勒]]。

== 流行文化 ==

在2004年由德國製作的電視影集{{translink|en|Stauffenberg (film)|史陶芬堡 (電影)|《史陶芬堡》}}<ref>{{imdb title|0388437|Stauffenberg}}</ref><ref>[http://www.new-video.de/film-stauffenberg/ New-video.de]</ref>由賽巴斯蒂安·魯格{{notetag|{{Lang-de|Sebastian Rüger}}}}飾演他本人。
