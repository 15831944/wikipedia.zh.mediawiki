'''埃默森·丹布德佐·姆南加古瓦'''（{{lang|en|Emmerson Dambudzo Mnangagwa}}，{{IPA-all|m̩.na.ˈᵑɡa.ɡwa|IPA|En-us-Emmerson Mnangagwa from Zimbabwe pronunciation (Voice of America).ogg}}；{{bd|1942年|9月15日||}}）生于[[南罗德西亚|南罗德西亚]]（今[[津巴布韦|津巴布韦]]）{{link-en|兹维沙瓦内|Zvishavane}}，[[津巴布韦|津巴布韦]]政治人物、現任[[津巴布韦总统|总统]]和和执政党[[辛巴威非洲民族联盟－爱国阵线|辛巴威非洲民族联盟－爱国阵线]][[第一书记|第一书记]]<ref name=gcz>{{Cite web |url =http://www.guancha.cn/Third-World/2017_11_22_436033_1.shtml |title =  姆南加古瓦被推选为津巴布韦总统|author =  |date =2017-11-22 |publisher =  观察者网|language =   }}</ref>。

== 生平 ==
===早年经历===
1960年代，姆南加古瓦加入[[辛巴威非洲民族聯盟－愛國陣線|津巴布韦非洲民族联盟]]，投身于推翻罗得西亚白人政权。姆南加古瓦出色的战斗能力为他赢得了“[[鳄鱼|鳄鱼]]”的绰号，他领导的团队也被称作“鳄鱼”游击队。在此期间，他曾遭逮捕入狱十年，和[[罗伯特·穆加贝|穆加贝]]成为狱友。获释后的姆南加古瓦被驱逐到[[赞比亚|赞比亚]]，他在赞比亚学习了法律。同时他一直与穆加贝保持着密切联系。1977年，姆南加古瓦成为穆加贝的特别助理，并任津巴布韦非洲民族联盟军事部门负责人。年轻时，姆南加古瓦还曾到[[中华人民共和国|中华人民共和国]][[北京大学|北京大学]]进修[[马克思主义|马克思主义]]课程，还在[[南京|南京]]等地学习<ref name=jiuzhi>{{Cite web |url =  http://news.ifeng.com/a/20171124/53559465_0.shtml|title = 穆加贝继任者宣誓就职 与中国颇有渊源 |author =  |date =2017-11-24  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

1980年[[津巴布韦|津巴布韦]]正式独立后，作为[[津巴布韦总统|津巴布韦总统]]穆加贝的重要幕僚，姆南加古瓦被任命为国家安全部长，后来还曾出任财政部长、国防部长等要职<ref name=wy/><ref name=jiuzhi/>。2013年9月起任司法部长<ref name=wy>{{Cite web |url = http://news.163.com/14/1213/05/ADAPQVOL00014JB5.html |title =  津巴布韦两名副总统宣誓就职|author =  |date = 2014-12-13 |publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。2013年9月10日，津巴布韦总统穆加贝组建新内阁，任命执政党核心成员担任国防部、司法部、财政部、新闻部等政府关键部门部长职务。9月11日穆加贝在[[哈拉雷|哈拉雷]]总统府内为新内阁成员、司法部长埃默森·姆南加古瓦签署任命书<ref>{{Cite web |url =http://news.xinhuanet.com/photo/2013-09/12/c_125372399_2.htm |title =  津巴布韦新内阁宣誓就职|author =  |date = 2013-09-12 |publisher =  新华网|language =  |accessdate =  |quote =  }}</ref>。

===第一副总统===
姆南加古瓦和副总统穆朱鲁曾被认为是接替穆加贝的最佳人选。2014年12月初，穆加贝公开指责副总统穆朱鲁及其“党羽”密谋暗杀他，但穆朱鲁一再否认。2014年12月9日，穆加贝宣布解除副总统穆朱鲁以及7名内阁部长的职务。2014年12月10日，姆南加古瓦被任命为执政党第一副主席兼津巴布韦第一副总统。2014年12月12日，由津巴布韦总统穆加贝任命的两名副总统与9名内阁部长宣誓就职，其中第一副总统为姆南加古瓦，第二副总统为姆波科。根据津巴布韦宪法，总统在任期内因故不能履行职务期间，将由第一副总统代行总统职权<ref name=wy/>。穆加贝任命姆南加古瓦为第一副总统，削减了姆南加古瓦主管内务和国防事务的权力<ref name=siling/>。

此后姆南加古瓦仍是接替穆加贝的热门人选。自2014年起，津巴布韦执政党内部围绕“后穆加贝时期”的领导权之争有所加剧，浮现出两大阵营，一方以姆南加古瓦为首，另一方是以穆加贝夫人[[格蕾絲·穆加貝|格雷丝·穆加贝]]为首的“40一代”（因为该派系成员多为四十至五十岁）<ref name=tao/><ref name=siling>{{Cite web |url =  http://news.163.com/17/1115/17/D3A34HG500018AOQ.html|title =津巴布韦局势突变军事“政变”还是“介入”？　-新华网  |author =  |date = 2017-11-15 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。2017年7月28日，格雷丝·穆加贝公开催促年过九旬的丈夫罗伯特·穆加贝尽早确定接班人<ref>{{Cite web |url = http://news.xinhuanet.com/world/2017-07/30/c_129667037.htm |title =津巴布韦“第一夫人”催穆加贝早定接班人  |author =  |date = 2017-07-30 |publisher =新华网 |language =  |accessdate =  |quote =  }}</ref>。2017年11月5日，格雷丝·穆加贝表态说，她已准备好接替丈夫罗伯特·穆加贝成为总统<ref>{{Cite web |url = http://news.xinhuanet.com/world/2017-11/07/c_129734177.htm |title = 穆加贝妻子表态：我已经做好准备接任总统 |author =  |date =  2017-11-07|publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。当天她还公开表示，“这个名叫姆南加古瓦的人曾在1980年企图发动政变”，“他和白人共谋”，“想从总统手中夺权”<ref name=gczw/>。

===解职与复出===
2017年11月6日，穆加贝宣布解除姆南加古瓦的第一副总统职务，指姆南加古瓦觊觎总统宝座，在执政党[[辛巴威非洲民族聯盟－愛國陣線|津巴布韦非洲民族联盟—爱国阵线]]定于2017年12月举行的特别会议前煽动派系斗争以谋取支持，甚至找巫医帮忙。2017年11月8日，姆南加古瓦发表长5页的声明称，“我突然离开是因为我的人身安全和家人遭到不断的威胁”，包括“下毒等多种形式”，他没有透露正身处哪国。姆南加古瓦同时指责穆加贝领导的执政党[[辛巴威非洲民族聯盟－愛國陣線|津巴布韦非洲民族联盟—爱国阵线]]，并发誓会东山再起。该声明发表的同日，穆加贝在[[哈拉雷|哈拉雷省]]向其数以千计的支持者宣布，姆南加古瓦“不具备我们高层应展现的最高纪律性”，“我们对他进行处分，并希望处分他的同伙”。当天执政党解除了姆南加古瓦在党内担任的领导职务<ref name=tao>{{Cite web |url = http://news.xinhuanet.com/world/2017-11/10/c_129737072.htm |title = 津巴布韦被解职副总统出逃 向穆加贝下战书 |author =  |date =2017-11-10   |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref name=siling/><ref name=gczw>{{Cite web |url =http://www.guancha.cn/global-news/2017_11_15_434973_1.shtml |title =  津巴布韦执政党：只是不流血的权力过渡，总统被妻子利用了|author =  |date =  2017-11-15|publisher =观察者网  |language =  |accessdate =  |quote =  }}</ref>。

2017年11月13日，津巴布韦国防军司令[[康斯坦蒂诺·奇温加|奇文加]]发表声明，敦促执政党停止排挤和清洗参加过独立战争的党内干部，并表示军队对此不会无动于衷，会采取措施阻止“反革命分子”劫持执政党的图谋。该声明被认为是对穆加贝11月6日解除第一副总统姆南加古瓦职务的回应。2017年11月14日晚，执政党发表声明，强烈谴责奇文加前一日发表的“叛逆”言论及军方将干政的威胁<ref name=gczw/>。

2017年11月15日凌晨1时左右，津巴布韦首都哈拉雷连续发生数起爆炸。随后军方封锁了该市外围地区的交通要道，还控制了津巴布韦国家广播公司。外界怀疑[[2017年津巴布韦政变|津巴布韦发生政变]]。但2017年11月15日凌晨，津巴布韦军方最高级别将领之一西布西索·莫悠（Sibusiso Moyo）少将在国家电视台宣读电视声明，否认发动军事政变，表示总统穆加贝目前“安全”。军方发言人还表示，穆加贝及其家人的安全有保证，军方的行动目的是让执政党内的“罪犯”被绳之以法，并尽快恢复津巴布韦正常的政治局面。自2017年11月14日晚起到11月15日，执政党津巴布韦非洲民族联盟—爱国阵线（ZANU PF）官方[[推特|推特]]账号频繁更新，该账号称此次军方的行动不是政变，而是“不流血的权力过渡”，总统穆加贝被其妻子“利用”，目前“第一家庭”安全，执政党还强调“津巴布韦和该国执政党都不是属于穆加贝和他的妻子。”该账号还宣布，第一副总统姆南加古瓦已成为执政党领袖<ref name=gczw/><ref name=siling/>。

===就任总统===
2017年11月21日，穆加贝总统被迫辞职。根据津巴布韦宪法规定，如果总统去世、辞职或被罢免，其空缺将由总统所在政党推选的人来填补。11月22日，执政党津巴布韦非洲民族联盟—爱国阵线向议会表示，推选姆南加古瓦为总统和党的[[第一书记|第一书记]]，姆南加古瓦将完成穆加贝的剩余总统任期，直到2018年总统大选。同日，姆南加古瓦从[[南非|南非]]回到津巴布韦。当地时间11月24日上午，津巴布韦候任总统姆南加古瓦在首都哈拉雷宣誓就任津巴布韦总统，就职仪式在国家体育场举行，姆南加古瓦在就职仪式上发表演讲时首先向穆加贝致敬，表示“对我而言，穆加贝依然是父亲般的人物、导师、战友和领袖”，并称[[2018年津巴布韋大選|2018年大选]]将如期以民主的方式举行<ref name =gcz/><ref name=jiuzhi/><ref>{{Cite web |url =http://news.xinhuanet.com/world/2017-11/24/c_129748025.htm  |title =  姆南加古瓦准备就任津总统 民众夹道欢迎其回国|author =  |date = 2017-11-24 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.xinhuanet.com/world/2017-11/25/c_129749266.htm |title = 姆南加古瓦宣誓就任津巴布韦新总统 向穆加贝致敬：依然如父 |author =  |date = 2017-11-25 |publisher = 新华网  |language =  |accessdate =  |quote =  }}</ref>。

2018年7月30日大選獲得連任，8月26日姆南加古瓦宣誓就任新一屆津巴布韋總統<ref>{{Cite news |url=http://www.xinhuanet.com/world/2018-08/26/c_129940456.htm |title=姆南加古瓦宣誓连任津巴布韦总统 |publisher=新華網 |date=2018-08-26}}</ref>。
