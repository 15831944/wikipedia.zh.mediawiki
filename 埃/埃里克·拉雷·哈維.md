'''埃里克·拉雷·哈維'''（{{lang-en|'''Erik LaRay Harvey'''}}）是一名美國演員，他的知名演出是在《[[酒私風雲|酒私風雲]]》中飾演Dunn Purnsley<ref>{{Cite web |url = https://www.rollingstone.com/tv/news/dunns-violent-boardwalk-empire-exit-20131027 |title = Dunn's Violent 'Boardwalk Empire' Exit |author = Sarene Leeds |date = 2013-10-27 |publisher = Roling Stone |language = en}}</ref>和《[[盧克·凱奇_(電視劇集)|盧克·凱奇]]》中飾演{{link-en|Diamondback (Willis Stryker)|Diamondback (Willis Stryker)|Willis Stryker / Diamondback}}<ref>{{Cite web |url = https://www.cbr.com/look-new-luke-cage-photos-reveal-classic-marvel-comics-villain/ |title = LOOK: New “Luke Cage” Photos Reveal Classic Marvel Comics Villain |author = Meagan Damore |date = 2016-05-25 |publisher = Comic Book Resources |language = en}}</ref>。

== 傳記 ==
埃里克·拉雷·哈維在參加教堂服務時看過《''{{link-en|The Wiz|The Wiz}}''》的舞台演出後被吸引到演藝業中。他長大後想成為一名牙醫，直到他在[[紐約大學|紐約大學]]上學時，決定全職演戲。<ref>{{Cite web |url = http://hydrogen-mag.com/archives/5824 |title = THE BOARDWALKER: A SIT DOWN WITH ERIK LARAY HARVEY |date = 2013-09-04 |publisher = Hydrogen |language = en}}</ref>
