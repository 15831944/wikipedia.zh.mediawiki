'''埃及航空'''（{{lang-ar|مصر للطيران}}）為一間以[[開羅|開羅]]為基地的[[埃及|埃及]]國營航空公司。由埃及政府全資擁有，經營定期航班服務至超過50個目的地，包括[[歐洲|歐洲]]、[[非洲|非洲]]、[[中東|中東]]、[[東南亞|東南亞]]，[[東亞|東亞]]、[[美國|美國]]及國內航線，過往曾開辦來往[[澳洲|澳洲]]之航線。埃及航空的主要基地為[[開羅國際機場|開羅國際機場]]。

== 歷史 ==
埃及航空成立於1932年6月7日，並於1933年7月開始營運。當時聯同Airwork Company共同成立Misr Airwork。[[第二次世界大戰|第二次世界大戰]]期間，埃及政府接管航空公司並於1949年改名為 Misr Air。1961年1月，因埃及及敘利亞建立新的政治聯繫，Misr Air與敘利亞航空公司（Syrian Airways）共同成立阿拉伯聯合航空公司（United Arab Airlines (UAA)），但這協議只維持了短暫時間。此名稱維持了一段時間，直至1971年10月才更改為埃及航空（EgyptAir）。

埃及航空為[[中東地區|中東地區]]首間經營噴射客機的航空公司。航空公司共聘請約22,000員工，於2000財政年度，埃及航空的全球總收入達10億[[美元|美元]]，利潤達3,500萬[[美元|美元]]。航空公司曾於2002年進行重組，進一步提升公司效率。(參考：[https://web.archive.org/web/20070212140628/http://weekly.ahram.org.eg/2003/640/eg9.htm A new take-off?])

埃及航空全資擁有[[西奈航空|西奈航空]]（Air Sinai）及擁有[[開羅航空|開羅航空]]60%股份。
 
埃及航空於2007年4月起開辦區域航線支部。新成立的子公司名為EgyptAir Express，於2007年首季擁有6架[[巴西航空工業|巴西航空工業（Embraer）]] 生產的[[Embraer_170|Embraer 170]]，航空公司亦有另外6架的選擇購買權。

== 意外及事故 ==
{{未完成列表}}
主要有死傷事故包括：
* 1963年7月28日，[[阿拉伯聯合航空869號班機空難_(1963年)|阿拉伯聯合航空869號班機]]為一架編號為SU-ALD的[[哈維蘭彗星型|哈維蘭彗星型]]，在降落孟買機場時遭遇湍流墜海，機上63人全部遇難。
* 1972年3月19日，{{link-en|埃及航空763號班機空難|EgyptAir Flight 763|埃及航空763號班機}}（租借[[亚德里亚航空|亚德里亚航空]]編號YU-AHR的[[DC-9|DC-9]]）於[[南也門|南也門]][[亞丁|亞丁]]墜毀，機上30人全部遇難。
* 1976年12月25日，當{{link-en|埃及航空864號班機空難|EgyptAir Flight 864|埃及航空864次航班}}（[[波音707|波音707]]，編號SU-AXA）的駕駛員嘗試調整下降速度失敗後，墜落於泰國曼谷一工業綜合設施。機上52名乘客中有20名喪生，地面亦有另外72人喪生。
* 1985年11月24日，[[埃及航空648號班機|埃及航空648號班機]]，一架波音737-266客機（編號：SU-AYH）被三名來至[[尼達爾組織|尼達爾組織]]（Abu Nidal Organization）的男子劫機至[[馬爾他|馬爾他]][[盧加|盧加]]。Omar Rezaq為當中唯一生還的劫機者。經過數小時的談判，埃及軍隊強行攻入機艙，與劫機者駁火，期間劫機者引爆幾枚手榴彈並槍殺了5名[[以色列人|以色列人]]及[[美國人|美國人]]。飛機於爆炸及火中受到嚴重破壞。最後，6名機員中有2名死亡，91名乘客中亦有58名被殺。
* 1999年10月31日，[[埃及航空990號班機空難|埃及航空990號航班]]，一架來往[[紐約|紐約]][[甘迺迪國際機場|甘迺迪國際機場]]及[[開羅|開羅]]的波音767-366ER客機（編號：SU-GAP）墜落於[[楠塔基特島|楠塔基特島]]對出的[[大西洋|大西洋]]水域。該航班的副駕駛員[[埃爾巴圖迪|埃爾巴圖迪]]（Gameel Al-Batouti）被美國有關當局指他進行自殺，蓄意將飛機撞毀。但埃及官員已對有關說法提出強烈質疑。
* 2002年5月7日，[[埃及航空843號班機空難|埃及航空843次航班]]，一架由[[開羅|開羅]]飛往[[突尼斯|突尼斯]]的波音737-566客機（編號：SU-GBI），在降落[[迦太基國際機場|迦太基國際機場]]途中，墜毀於約6公里外的山區，3名機組人員及11名乘客死亡。
* 2011年7月29日，埃及航空一架波音777-200ER（註冊編號SU-GBP）在[[開羅國際機場|開羅國際機場]]準備執行飛往吉達的MS667次班機，停泊期間駕駛艙失火，機上人員全數逃生，而起火的客機則因無法修復而報廢。<ref name=SU-GBP>{{cite web |url=http://aviation-safety.net/database/record.php?id=20110729-0 |title=EgyptAir Flight 667 ground fire |publisher=Aviation Safety Network |accessdate=2013-07-08}}</ref>
*2016年3月29日，[[埃及航空181號班機劫機事件|埃及航空181號班機]]由[[空中巴士A320|空中巴士A320]]客機執飛，由[[亞歷山大港|亞歷山大港]]飛往開羅時被劫持，其後被逼降落於[[拉納卡國際機場|拉納卡國際機場]]，機上有55名乘客及7名機組人員被劫持。最終，機組人員及所有乘客獲悉，劫機者舉手投降。<ref>{{cite news |language = 中文  |url = http://hk.apple.nextmedia.com/enews/realtime/20160329/54922291?top=4h |title = 【武裝劫機】埃及航空客機遭劫持 劫機者身上有炸彈 |publisher = 蘋果日報  |date = 2016年3月29日 |accessdate = 2015年3月29日}}</ref><ref>{{cite news |language = 中文  |url = http://hk.apple.nextmedia.com/realtime/international/20160329/54924628 |title = 【武裝劫機】人質獲釋 劫機者舉手投降被捕 |publisher = 蘋果日報  |date = 2016年3月29日 |accessdate = 2015年3月29日}}</ref>
*2016年5月19日，[[埃及航空804号班机空难|埃及航空804号班机]]由[[空中巴士A320|空中巴士A320]]客機執飛，由[[巴黎|巴黎]]飛往開羅時失联，后被证实坠海。
