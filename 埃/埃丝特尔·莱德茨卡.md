'''埃丝特尔·莱德茨卡'''（{{lang-cs|Ester Ledecká}}，{{IPA-cs|ˈɛstɛr  ˈlɛdɛtskaː}}，{{bd|1995年|3月23日||}}）生于布拉格，是一名捷克女子单板滑雪和高山滑雪运动员。在2018年平昌冬奥会期间，她先后获得高山滑雪女子超大曲道项目和单板滑雪女子平行大曲道两个项目的金牌，既成为史上第一位在单板和双板滑雪项目上都夺得金牌的运动员，又成为史上第一位在同一届冬奥会上夺得两个不同项目金牌的女子运动员<ref>{{cite web|url=http://sports.qq.com/a/20180224/016208.htm|title=天才！单板双板均夺冠 莱德茨卡成跨项摘金冬奥第一人|publisher=腾讯体育|date=2018-02-24|accessdate=2018-06-15|deadurl=no|archiveurl=https://web.archive.org/web/20180615190545/http://sports.qq.com/a/20180224/016208.htm|archivedate=2018-06-15|df=}}</ref>。除此之外她也是第一位在单板滑雪世界杯平行曲道项目当中夺冠的捷克运动员。

==运动生涯==
===2014年索契冬奥周期===
莱德茨卡在2012–13赛季单板滑雪世界杯上完成了自己的世界杯首秀，当时她在平行大曲道项目上获得第13名<ref name=cupdebut />。2013年3月，她在世界青年单板滑雪锦标赛上斩获平行曲道和平行大曲道两个项目的冠军。<ref>{{cite web |url=http://www.fis-ski.com/snowboard/parallel/news-multimedia/news/article=ledecka-and-baumeister-win-parallel-slalom-junior-world-champs.html |title=Ledecka and Baumeister win parallel slalom at Junior World Champs |work=FIS |date=2013-03-07 |accessdate=2014-02-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20140211041756/http://www.fis-ski.com/snowboard/parallel/news-multimedia/news/article=ledecka-and-baumeister-win-parallel-slalom-junior-world-champs.html |archivedate=2014-02-11 |df= }}</ref>同年年底，她获得了年度最佳捷克青少年运动员奖<ref>{{cite web |first1=Jaroslav |last1=Beránek |first2=Martina |last2=Kučerová |url=http://sport.idnes.cz/zuzana-hejnova-sportovec-roku-2013-dq1-/sporty.aspx?c=A131218_145033_sporty_ber |script-title=cs:Neporazitelná běžkyně Hejnová vyhrála i souboj o titul Sportovec roku |work=Mladá fronta DNES |date=2013-12-18 |accessdate=2014-02-07 |deadurl=no |archiveurl=https://web.archive.org/web/20131223102414/http://sport.idnes.cz/zuzana-hejnova-sportovec-roku-2013-dq1-/sporty.aspx?c=A131218_145033_sporty_ber |archivedate=2013-12-23 |df= }}</ref>。

在2013–14赛季单板滑雪世界杯期间，她在[[奥地利|奥地利]][[巴特加斯泰因|巴特加斯泰因]]站的两次平行曲道比赛中分别获得第二名和第三名。之后，她在[[斯洛文尼亚|斯洛文尼亚]][[罗格拉|罗格拉]]站的比赛中首次夺冠，既成为首位在单板滑雪世界杯平行曲道项目中夺冠的捷克运动员<ref name=onetowatch>{{cite web |first=Andrew |last=Lawton |url=https://www.telegraph.co.uk/sport/othersports/winter-olympics/10619915/Winter-Olympics-2014-Blaggers-guide-to-snowboarding.html |title=Winter Olympics 2014: Blagger's guide to snowboarding |work=The Telegraph |date=2014-02-05 |accessdate=2014-02-07 |deadurl=no |archiveurl=https://web.archive.org/web/20140206225621/http://www.telegraph.co.uk/sport/othersports/winter-olympics/10619915/Winter-Olympics-2014-Blaggers-guide-to-snowboarding.html |archivedate=2014-02-06 |df= }}</ref>，又成为历史上第三位在单板滑雪世界杯上夺冠的捷克运动员<ref>{{cite web |url=http://www.rogla.eu/en/winter/news/n/careers-first-for-ledecka-and-mathies-in-rogla-pgs-first-czech-win-on-alpine-snowboard-world-cup/ |title=Career's first for Ledecka and Mathies in Rogla PGS | First Czech win on Alpine Snowboard World Cup tour | Exciting race for Crystal Globes |work=rogla.eu |date= |accessdate=2014-02-07 |archiveurl=https://web.archive.org/web/20140207114754/http://www.rogla.eu/en/winter/news/n/careers-first-for-ledecka-and-mathies-in-rogla-pgs-first-czech-win-on-alpine-snowboard-world-cup/ |archivedate=2014-02-07}}</ref>。

在[[2014年冬季奥林匹克运动会|2014年索契冬奥会]]前，她被《[[每日电讯报|每日电讯报]]》编辑安德鲁·劳顿（{{lang|en|Andrew Lawton}}）视为女子单板滑雪项目的其中一个看点<ref name=onetowatch />，而《[[体育画报|体育画报]]》也认为她和[[玛尔蒂娜·萨布利科娃|玛尔蒂娜·萨布利科娃]]（{{lang|cs|Martina Sáblíková}}）、[[加布丽埃拉·绍卡洛娃|加布丽埃拉·绍卡洛娃]]（{{lang|cs|Gabriela Soukalová}}）以及[[埃娃·萨姆科娃|埃娃·萨姆科娃]]（{{lang|cs|Eva Samková}}）这三位队友一样都具备为捷克队拿下奖牌的实力<ref>{{cite web |url=http://sportsillustrated.cnn.com/-olympics/news/20140116/czech-republic-sochi-olympics-team.ap/ |title=Czech Republic names 85 athletes to Sochi Olympic team |work=AP |publisher=Sports Illustrated |date=2014-01-16 |accessdate=2014-02-07 |deadurl=no |archiveurl=https://web.archive.org/web/20140222062418/http://sportsillustrated.cnn.com/-olympics/news/20140116/czech-republic-sochi-olympics-team.ap/ |archivedate=2014-02-22 |df= }}</ref>。

莱德茨卡在索契冬奥会期间参加平行曲道和平行大曲道两个项目，在率先进行的平行大曲道项目上，她被后来获得该项目金牌的瑞士选手[[帕特丽齐亚·库梅尔|帕特丽齐亚·库梅尔]]（{{lang|de|Patrizia Kummer}}）淘汰，止步八强，获得第七名<ref name=olympicdebut>{{cite web |first=Filip |last=Grim |url=http://oh.idnes.cz/rozhovor-ledecka-0ea-/zoh-ostatni-sporty.aspx?c=A140219_124710_zoh-ostatni-sporty_fil |script-title=cs:Desítka, jak jsem plánovala, chválila se Ledecká po olympijském debutu |work=Mladá fronta DNES |date=2014-02-19 |accessdate=2014-02-19 |deadurl=no |archiveurl=https://web.archive.org/web/20140226142816/http://oh.idnes.cz/rozhovor-ledecka-0ea-/zoh-ostatni-sporty.aspx?c=A140219_124710_zoh-ostatni-sporty_fil |archivedate=2014-02-26 |df= }}</ref>。随后她又参加了平行曲道项目，最终在四分之一决赛中被德国选手[[安克·卡斯滕斯|安克·卡斯滕斯]]（{{lang|de|Anke Karstens}}）淘汰，位居第六<ref>{{cite web| last = Grim| first = Filip| script-title = cs: Snowboardistka Ledecká opět končí ve čtvrtfinále, ve slalomu je šestá| publisher = Mladá fronta DNES| date = 2014-02-22| accessdate = 2014-02-22| url = http://oh.idnes.cz/paralelni-slalom-ledecka-0xc-/zoh-ostatni-sporty.aspx?c=A140222_060653_zoh-ostatni-sporty_ber| deadurl = no| archiveurl = https://web.archive.org/web/20140227095225/http://oh.idnes.cz/paralelni-slalom-ledecka-0xc-/zoh-ostatni-sporty.aspx?c=A140222_060653_zoh-ostatni-sporty_ber| archivedate = 2014-02-27| df = }}</ref>。

===2018年平昌冬奥周期===
2015年1月，莱德茨卡参加了[[奥地利|奥地利]][[申贝格-拉赫塔尔|克里斯赫伯格]]举行的世界自由式滑雪和单板滑雪锦标赛，在率先进行的单板滑雪平行曲道比赛上，她成功夺得个人生涯首个世锦赛冠军<ref>{{cite web|url=http://sport.idnes.cz/snowboardistka-ledecka-na-ms-paralelni-slalom-fos-/lyzovani.aspx?c=A150122_110428_lyzovani_par|script-title=cs:V devatenácti světovou šampionkou. Ledecká ovládla paralelní slalom|publisher=Mladá fronta DNES|date=2015-01-22|accessdate=2018-06-12|deadurl=no|archiveurl=https://web.archive.org/web/20180612210720/https://sport.idnes.cz/snowboardistka-ledecka-na-ms-paralelni-slalom-fos-/lyzovani.aspx?c=A150122_110428_lyzovani_par|archivedate=2018-06-12|df=}}</ref>。而在一天后进行的平行大曲道比赛当中，她在四分之一决赛负于瑞士选手[[尤莉·佐格|尤莉·佐格]]（{{lang|de|Julie Zogg}}），获得第五<ref>{{cite web|url=http://medias4.fis-ski.com/pdf/2015/SB/7037/7037RLRF.pdf|title=FIS Snowboard World Championships 2015 Ladies' Parallel Giant Slalom - Finals|work=FIS|date=2015-01-22|accessdate=2018-06-12|deadurl=no|archiveurl=https://web.archive.org/web/20180612184633/http://medias4.fis-ski.com/pdf/2015/SB/7037/7037RLRF.pdf|archivedate=2018-06-12|df=}}</ref>。同年年底，她与自行车选手[[伊日·亚诺舍克|伊日·亚诺舍克]]（{{lang|cs|Jiří Janošek}}）共同获得年度最佳捷克青少年运动员奖<ref>{{cite web|url=http://sportovecroku.cz/wp-content/uploads/2015/12/15_TZ_21.12.15.pdf|script-title=cs:Sportovec Roku 2015 slavnostní vyhlášení výsledků ankety|website=sportovecroku.cz|accessdate=2018-06-12|date=2015-12-21|deadurl=no|archiveurl=https://web.archive.org/web/20160304224142/http://sportovecroku.cz/wp-content/uploads/2015/12/15_TZ_21.12.15.pdf|archivedate=2016-03-04|df=}}</ref>，这是她第二次获得该奖项。

莱德茨卡于2016年2月完成了自己的高山滑雪世界杯首秀，她在自己参加的第一场比赛当中获得第24名<ref name=fis>{{cite web |url=http://www.fis-ski.com/snowboard/alpine-snowboard/news-multimedia/news/article=ester-ledecka-surprises-ski-world.html |title=Ester Ledecka surprises ski world |date=2016-02-09 |work=FIS |access-date=2016-12-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20161220190612/http://www.fis-ski.com/snowboard/alpine-snowboard/news-multimedia/news/article=ester-ledecka-surprises-ski-world.html |archivedate=2016-12-20 |df= }}</ref>。之后她又参加了四场比赛，并在其中的三场比赛中获得积分<ref name=ctv/>。2017年2月，她首次出战[[2017年世界高山滑雪锦标赛|世界高山滑雪锦标赛]]，成为史上首位参加过单板滑雪和高山滑雪两个项目世锦赛的选手，比赛期间她在滑降、超大曲道和全能项目中均闯入前30名。而在之后进行的世界自由式滑雪和单板滑雪锦标赛上，她也有出色表现，获得了平行大曲道项目的冠军和平行曲道项目的亚军。<ref name=nyt/>
