'''埃米略·门德斯·佩雷斯'''（{{lang-es|Emilio Méndez Pérez}}，{{bd|1949年|5月22日}}）是西班牙[[物理学|物理学]]家。
==经历==
门德斯于1934年6月15日出生在西班牙[[加泰罗尼亚|加泰罗尼亚]]莱里达。毕业于马德里康普顿斯大学，后取得麻省理工学院物理学博士学位。1979年至1994年在[[IBM研究院|IBM研究院]][[湯瑪士·Ｊ·華生研究中心|湯瑪士·Ｊ·華生研究中心]]先后担任[[博士后|博士后]]研究员、一般工作人员和高级干事，期间还与[[马德里自治大学|马德里自治大学]]和[[西班牙高等科学研究理事会|西班牙高等科学研究理事会]]的研究所开展过密切合作<ref>{{cite web |author=Javier Cuartas |title=Pedro Miguel Etxenike y Emilio Méndez, premio Príncipe de Asturias de Investigación |url=https://elpais.com/diario/1998/05/30/sociedad/896479201_850215.html |publisher=[[国家报|El País]] |accessdate=2019-01-30 |location=[[奥维耶多|Oviedo]] |language=es |date=1998-05-30}}</ref>。

1995年开始担任[[美国|美国]][[纽约州立大学石溪分校|纽约州立大学石溪分校]]物理学教授，1995年至2002年担任该校[[相_(物质)#界面現象|界面现象]]研究所主任<ref>{{cite web |title=Mr. Emilio Méndez Pérez |url=http://www.spainusa.org/en/patronages/mr-emilio-mendez-perez |publisher=Fundación Consejo España – EE. UU. |archiveurl=https://web.archive.org/web/20190130070803/http://www.spainusa.org/en/patronages/mr-emilio-mendez-perez |archivedate=2019-01-30 |language=en}}</ref>。1998年获得[[阿斯图里亚斯亲王奖|阿斯图里亚斯亲王奖]]科学技术奖<ref>{{cite web |title=EMILIO MÉNDEZ PÉREZ Y PEDRO MIGUEL ECHENIQUE LANDIRÍBAR |url=http://www.fpa.es/es/premios-princesa-de-asturias/premiados/1998-emilio-mendez-perez-y-pedro-miguel-echenique-landiribar.html?texto=acta&especifica=0 |publisher=FUNDACIÓN PRINCESA DE ASTURIAS |archiveurl=https://web.archive.org/web/20190130071444/http://www.fpa.es/es/premios-princesa-de-asturias/premiados/1998-emilio-mendez-perez-y-pedro-miguel-echenique-landiribar.html?texto=acta&especifica=0 |archivedate=2019-01-30 |date=1998|language=es}}</ref>。2006年至2016年担任纽约州[[布鲁克黑文国家实验室|布鲁克黑文国家实验室]]纳米功能材料中心主任，2016年成为该实验室能源科技部门主任<ref>{{cite web |title=Emilio Mendez Perez |url=http://gadeaciencia.org/teams/mendez-perez-emilio-2/ |publisher=Fundación GADEA |archiveurl=https://web.archive.org/web/20190130073944/http://gadeaciencia.org/teams/mendez-perez-emilio-2/ |archivedate=2019-01-30 |language=en |date=2018}}</ref>。
