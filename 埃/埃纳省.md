'''埃纳省'''（{{lang-fr|Aisne}}）是[[法国|法国]]的第02[[省_(法国)|省]]，位于[[上法兰西大区|上法兰西大区]]，它因流经的同名河流[[埃纳河|埃纳河]]而得名。

== 历史 ==
埃纳省是1790年创建的83个省之一，它由[[法兰西岛省|法兰西岛省]]的一部分（拉昂地区 Laonnois、苏瓦松地区 Soissonais、努瓦荣地区 Noyonais、瓦卢瓦 Valois），[[皮卡迪省|皮卡迪省]]的一部分（Thiérache, Vermandois）以及[[香槟省|香槟省]]的一部分组成的。

在[[第一次世界大战|第一次世界大战]]中，数场战斗在该地展开。[[德国|德国]]人在1918年对[[贵妇小径|贵妇小径]]的争夺是其中最著名的，它也被称为[[第二次埃纳战役|第二次埃纳战役]]。

== 地理 ==
=== 地形 ===
[[File:Aisne.png|thumb]]
埃纳省被[[诺尔省|诺尔省]]、[[索姆省|索姆省]]、[[瓦兹省|瓦兹省]]、[[塞纳-马恩省|塞纳-马恩省]]、[[马恩省|马恩省]]以及[[阿登省|阿登省]]围绕，同时它也与[[比利时|比利时]]接壤。

该省的地形较为平坦，只有中心隆起一个外露层，中世纪古城拉昂（Laon）坐落其上。虽然较为偏远的圣康坦（Saint-Quentin）人口是其三倍，这个位置还是使得拉昂成为了省府。

[[降水量|年降水量]]平均500至750毫米。

== 经济 ==
[[农业|农业]]主导当地经济。拉昂市是一个“分水岭”：省的南部多为树木繁茂的地区，而从该市以北开始种植[[甜菜|甜菜]]（七十年代的甜菜生产危机标志了当地农业的衰落）。Guise则是埃纳省北部的农业中心。此外，[[牛|牛]][[羊|羊]]的饲养业也比较发达，四处遍布大大小小的牧场。

[[工业|工业]]方面，[[丝|丝]]、[[棉|棉]]、[[羊毛|毛]]织品繁荣于圣康坦和其它城镇。[[圣戈班|圣戈班]]是起源于当地的著名[[玻璃|玻璃]]制造商，它的业务开始于十七世纪，为[[凡尔赛宫|凡尔赛宫]][[镜厅|镜厅]]提供装饰。

== 人口 ==
埃纳省包括几个中等城市，每个城市环绕着许多很小的村庄。该省在十九世纪下半叶出现了[[乡村人口外流|乡村人口外流]]，造成了总人口的下降，而这种情况部分被本地的工业发展所平衡。而两次世界大战对当地的人口造成很大的影响，而战后逐渐回升到1900年的水平。三十多年来，该地的工业衰退引发了人口的停滞。而只有该省西南部靠近[[大巴黎|大巴黎]]的地区仍有[[人口增长|人口增长]]。

== 文化 ==
在第一次世界战争中相当数量的建筑名胜被破坏，但是在保存下来的建筑中，拉昂等地的[[中世纪|中世纪]][[教堂|教堂]]仍具举足轻重的价值。已荒废的La Ferté-Milon[[城堡|城堡]]则在战争中躲过了进一步的破坏。
