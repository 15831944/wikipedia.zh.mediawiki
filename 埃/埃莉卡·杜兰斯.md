'''埃莉卡·杜兰斯'''（{{lang-en|''Erica Durance''}}，又译：'''爱丽卡·杜伦斯'''），出生于1978年6月21日，加拿大女演员，她因在《[[超人前传|超人前传]]》中饰演[[露易絲·蓮恩|露易絲·蓮恩]]而为大家所熟知。

== 生平 ==
杜兰斯生于加拿大[[艾伯塔省|艾伯塔省]][[卡尔加里|卡尔加里]]，但有人说是在艾伯塔省三树镇。高中毕业后杜兰斯搬到[[不列颠哥伦比亚省|不列颠哥伦比亚省]]，并开始追求她感兴趣的演艺事业。

== 生涯 ==

== 个人生活 ==
　　2005年1月8日杜兰斯和加拿大演员David Palffy结婚。
