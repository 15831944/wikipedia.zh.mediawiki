'''傑佛瑞·羅根·休洛'''（{{lang-en|Jeffrey Logan Sciullo}}，{{bd|1988年|2月14日|||Elias}}[http://www.espn.com/wwe/story/_/id/19152814/wwe-profile-page-elias]）是出生於[[美國|美國]][[賓州|賓州]][[匹茲堡|匹茲堡]]的職業摔角手。他在[[WWE|WWE]]的摔角節目裡是以'''埃利亞斯'''（Elias）做為擂台名稱。

== 摔角經歷 ==
=== 獨立圈時期 ===
傑佛瑞在出道時期是於美國東北部一帶的地方摔角圈上活動，當時他除使用個人本名之外，也有採用過重金屬耶穌（Heavy Metal Jesus）之類的擂台名稱。

他主要活躍的獨立圈是國際摔角同業聯盟（International Wrestling Cartel、IWC），並曾奪下該聯盟的超級印地冠軍與重量級冠軍頭銜。<ref>{{cite web |url=http://www.wrestleview.com/roh-indy-wrestling-news/39317-gcw-on-2-2-alliance-podcast-cpw-on-3-9-iwc-on-1-26/ |title=GCW on 2/2, Alliance podcast, CPW on 3/9, IWC on 1/26 |work=Wrestleview |author=Adam Martin |language=en |date=2013年1月25日 |accessdate=2018年2月14日}}</ref>

=== WWE時期 ===
=== NXT ===
2014年傑佛瑞被[[WWE|WWE]]給簽下，一開始被安排為《[[WWE_NXT|NXT]]》節目裡的選手，並開始改用埃利亞斯·參孫（Elias Samson）的擂台名稱。<ref>{{cite web |url=https://www.pwtorch.com/artman2/publish/The_Specialists_34/article_75101.shtml#.WoQv5XUjFD8 |title=FALLS COUNT ANYWHERE: Donofrio's column returns with review of WWE Raw, Mania build, Network, TNA Impact, new Developmental signing, more  |work=PWtorch |author= |language=en |date=年月日 |accessdate=2018年2月14日}}</ref>他在《NXT》的首次登場，是在2014年4月24日與{{tsl|en|Buddy Murphy|巴迪·莫菲}}組成雙打來對上[[The_Ascension|The Ascension]]。<ref>{{cite web |url=http://www.profightdb.com/wrestlers/elias-samson-9588.html?year=2014 |title=Elias Samson |work=Pro Fight DB |language=en |accessdate=2018年2月14日}}</ref>該期間他在節目裡多擔任用來襯托出對手強度，負責輸掉比賽的一方。

2015年8月WWE給了埃利亞斯·參孫一個新的造型，外觀上是個會拿著吉他演唱的流浪樂手風格。之後曾與{{tsl|en|Tucker Knight|塔克·奈特}}合組來角逐NXT的雙打錦標賽，不過在第一回合便遭{{tsl|en|The Revival (professional wrestling)|The Revival}}給淘汰。<ref>{{cite web |url=http://www.wrestleview.com/wwe-news/55985-spoiler-8-29-dusty-rhodes-tag-team-classic-taping/ |title=SPOILER: 8/29 Dusty Rhodes Tag Team Classic taping |work=Wrestleview |author=Jason Namako |language=en |date=2015年8月31日 |accessdate=2018年2月14日}}</ref>在2015年年底至2016年年初期間，埃利亞斯·參孫有得到較多在節目上取得勝利的安排，但6月3日埃利亞斯·參孫在與{{tsl|en|Apollo Crews|阿波羅·克魯茲}}的一對一單打戰後，因傷勢問題而離開節目休養一陣子，之後在9月29日回歸。

2017年3月15日WWE安排了一個戲碼給了埃利亞斯·參孫，內容是讓埃利亞斯·參孫嘲弄失去冠軍頭銜的{{tsl|en|Chris Hero|克里斯·希洛|卡修斯·歐諾}}，讓他們兩人結下樑子而舉行一場誰輸就得離開NXT的賭注比賽。該比賽於3月29日舉行，最後結果是埃利亞斯·參孫輸給了卡修斯·歐諾。輸掉比賽後的埃利亞斯·參孫，在4月1日改以蒙面造型的「漂流者」（El Vagabundo）身份於節目上登場，不過他在當天的單打賽輸掉後即被扯下面具而遭揭穿身份。<ref>{{cite web |url=https://www.f4wonline.com/wwe-results/nxt-pre-takeover-tv-taping-results-el-vagabundo-appears-232926 |title=NXT PRE-TAKEOVER TV TAPING RESULTS: EL VAGABUNDO APPEARS |work=F4wonline |author=F4W STAFF |language=en |date=2017年4月1日 |accessdate=2018年2月14日}}</ref>

=== RAW ===
2017年4月10日WWE把埃利亞斯·參孫到主秀《[[WWE_Raw|Raw]]》節目上。在《Raw》上的埃利亞斯·參孫開始被塑造成不時會在節目裡冒出，拿著吉他像是音樂浪人般獨自伴唱的模樣。埃利亞斯·參孫在《Raw》的第一次出賽是5月22日對上[[迪恩·安布羅斯|迪恩·安布羅斯]]，之後比賽中途因[[米茲|米茲]]闖入並刻意攻擊他，讓他以比賽取消方式獲勝。<ref>{{cite web |url=http://www.wwe.com/shows/raw/2017-05-22#full-detail-40024734 |title=Elias Samson def. Intercontinental Champion Dean Ambrose via Disqualification |work=[[WWE|WWE]] |language=en |date=2017年5月22日 |accessdate=2018年2月14日}}</ref>

到了6月19日WWE加強了埃利亞斯·參孫的個人風格，讓他開始會在節目的擂台上拿著吉他，演唱著嘲諷對手或是數落現場觀眾的歌曲，也開始以「誰願意跟隨埃利亞斯？」（Who wants to walk with Elias?）來做為他的個人代表台詞。在7月底，WWE則將他的擂台名稱改成較簡明易記的埃利亞斯（Elias）。<ref>{{cite web |url=http://www.wrestleview.com/wwe-news/87486-name-change-wwe-elias-samson/ |title=Name change for Elias Samson, John Cena signs 25,000 autographs |work=Wrestleview |author=Adam Martin |language=en |date=2017年7月28日 |accessdate=2018年2月14日}}</ref>

加入《Raw》後的埃利亞斯，他首次登場的大賽是9月21日《[[完全終結|完全終結]]》上與{{tsl|en|Apollo Crews|阿波羅·克魯茲}}的一對一單打戰，並且有取得勝利。<ref>{{cite web |url=http://www.wwe.com/shows/nomercy/2017/apollo-crews-elias-results |title=Elias def. Apollo Crews |work=WWE |author=Anthony Benigno |language=en |date=2017年9月24日 |accessdate=2018年2月14日}}</ref>而10月22日的《[[桌子、梯子、椅子|TLC]]》裡，在埃利亞斯輸給{{tsl|en|Jason Jordan|傑森·喬登}}後，<ref>{{cite web |url=http://www.wwe.com/shows/wwetlc/2017/jason-jordan-elias-results |title=Jason Jordan def. Elias |work=WWE |author=Anthony Benigno |language=en |date=2017年10月22日 |accessdate=2018年2月14日}}</ref>節目安排他與對方結下樑子的劇情，之後在11月6日《Raw》裡舉行了一場埃利亞斯與喬登要搶下吉他來痛砸對手才能取得勝利的特殊規則賽事，最後結果仍由喬登獲勝。<ref>{{cite web |url=http://www.wwe.com/shows/raw/2017-11-06/article/jason-jordan-elias-guitar-on-a-pole-match |title=Jason Jordan and Elias to battle in a Guitar-on-a-Pole Match |work=WWE |author=Michael Burdick |language=en |date=2017年11月3日 |accessdate=2018年2月14日}}</ref>

到了2017年底推出了埃利亞斯與[[約翰·希南|約翰·希南]]發生衝突的戲碼，其中2018年1月22日《Raw》第25週年慶的特別節目裡，安排出讓埃利亞斯以吉他痛砸希南的橋段，為數天後舉行的《[[皇家大戰|皇家大戰]]》劇情鋪路。<ref>{{cite web |url=https://www.thesportster.com/news/elias-cena-attack-explained/ |title=Why Elias Attacked John Cena On Raw |work=The Sportster |author=Alex Hoegler |language=en |date=2018年1月24日 |accessdate=2018年2月14日}}</ref>之後埃利亞斯在28日《皇家大戰》30人大混戰裡以第6順位進場，結果是遭第20位進場的希南給淘汰。

結束《皇家大戰》後的埃利亞斯，開始角逐進入在2月25日舉行的《[[行刑室鐵籠戰|行刑室鐵籠戰]]》裡，向未來[[WWE環球冠軍|WWE環球冠軍]]挑戰的機會。埃利亞斯首先在1月29日《Raw》裡淘汰了[[麥特·哈迪|麥特·哈迪]]，獲得進入《行刑室鐵籠戰》的資格，<ref>{{cite web |url=http://www.wwe.com/shows/raw/2018-01-29/article/elias-def-woken-matt-hardy-qualify-mens-elimination-chamber |title=Elias def. "Woken" Matt Hardy to qualify for the Men's Elimination Chamber Match |work=WWE |author=Anthony Benigno |language=en |date=2018年1月29日 |accessdate=2018年2月14日}}</ref>之後也在隔週2月5日節目裡和約翰·希南以及[[布勞恩·史卓曼|布勞恩·史卓曼]]的三人攻防戰上勝出，得到能在《行刑室鐵籠戰》以最後順位入場的優先權。<ref>{{cite web |url=https://www.cagesideseats.com/wwe/2018/2/5/16977130/elias-wins-right-to-enter-elimination-chamber-match-last |title=Elias wins the right to enter the Elimination Chamber match last |work=Cage Sideseats |author=Tommy Messano |language=en |date=2018年2月5日 |accessdate=2018年2月14日}}</ref>
