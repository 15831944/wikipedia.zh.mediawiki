'''埃洛伊·古铁雷斯·梅诺约'''（Eloy Gutiérrez Menoyo，{{bd|1934年|12月8日|2012年|10月26日}}）生于[[西班牙|西班牙]][[马德里|马德里]]，[[古巴|古巴]]政治人物<ref name=rmw>{{Cite web |url =  http://www.people.com.cn/GB/paper68/9926/911535.html|title = 卡斯特罗政敌回国了 |author =  |date =2003-08-13  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref><ref name=laomu>{{Cite web |url = http://opinion.huanqiu.com/opinion_world/2017-09/11245538.html |title = 劳木：为报纸一句话，12国大使找上门 |author =  |date = 2017-09-14 |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1934年，梅诺约生于西班牙马德里。1948年随家人迁居古巴。1957年，梅诺约参加了反对[[富尔亨西奥·巴蒂斯塔|巴蒂斯塔]]独裁统治的斗争。后逃进埃斯坎布雷山，在当地组织了游击队。1959年1月，当[[菲德尔·卡斯特罗|卡斯特罗]]进入[[哈瓦那|哈瓦那]]时，梅诺约在哈瓦那和他会师<ref name=rmw/><ref name=laomu/>。

1961年，因政治分歧，梅诺约与卡斯特罗分道扬镳，独自流亡[[美国|美国]][[佛罗里达州|佛罗里达州]]。1964年，梅诺约率一支武装别动队登陆古巴，但迅速失败。梅诺约被捕后被判处死刑，后改为30年有期徒刑。梅诺约在监狱中关押22年后，于1986年获释。获释后梅诺约在西班牙生活了一段时间，后来定居美国[[迈阿密|迈阿密]]。1993年，梅诺约成立了名为“古巴的变革”的古巴流亡者组织，以反对美国对古巴的封锁。这一立场促成1995年梅诺约同卡斯特罗的数次会晤。会晤后，梅诺约认为已经看到古巴政治变革的萌芽，尽管他回古巴定居的要求被古巴政府拒绝<ref name=rmw/><ref name=laomu/>。

2003年，美国助理国务卿{{link-en|罗杰·诺列加|Roger Noriega|罗杰·诺列加}}对记者表示，美国对古巴的政策集中在对古巴的经济制裁，而未考虑将持不同政见者作为变革的代理人，这是“一场悲剧”。该谈话被视为[[乔治·沃克·布什|小布什]]政府改变对古巴策略的信号。诺列加还表示，小布什政府近期派3名高官来到迈阿密，探索促进古巴政权民主化的新思路。在这一背景下，2003年8月7日，在哈瓦那[[何塞·马蒂国际机场|何塞·马蒂国际机场]]，68岁的梅诺约宣布决定留在古巴定居，不再回美国继续流亡。但是，鉴于他的政治立场未变，古巴政府这次依旧不允许他回古巴定居<ref name=rmw/><ref name=laomu/>。然而此后梅诺约终于定居古巴。

2012年10月26日，梅诺约在古巴哈瓦那病逝<ref>{{cite news|last=Martin|first=Douglas|title=Eloy Gutiérrez-Menoyo, Cuban Dissident, Dies at 77|url=https://www.nytimes.com/2012/10/27/world/americas/eloy-gutierrez-menoyo-cuban-dissident-dies-at-77.html?_r=1|newspaper=The New York Times|date=October 26, 2012}}</ref>。
