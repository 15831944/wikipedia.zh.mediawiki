'''埃克森美孚'''<ref>该公司官方中文名“埃克森美孚”。见{{Cite web |url = http://www.exxonmobilchemical.com.cn/Chem-Chinese/default.aspx |title = 埃克森美孚公司 |author =  |date =  |publisher =埃克森美孚公司  |language =  |accessdate = 2017-01-11 |quote =  }}，以及{{Cite web |url = http://www.mobil1.tw |title = Taiwan |author =  |date =  |publisher = Exxon Mobil（埃克森美孚公司全球網站）|language =  |accessdate = 2017-01-11 |quote =  }}</ref>（{{lang|en|Exxon Mobil}}，{{nyse|XOM}}）是世界上[[市值|总市值]]最大的股票公开上市的石油公司，前身分別為'''埃克森'''和'''[[美孚石油|美孚]]'''，於1999年11月30日合併重組。該公司也是埃克森、美孚及[[埃索|Esso]]全球分公司的母公司。此外，埃克森美孚與[[殼牌公司|殼牌公司]]（Shell）、[[英國石油公司|英國石油公司]]（BP）及[[道達爾|道達爾]]（Total）同為全球最大的石油公司之一。

2007年底公司石油储量为720亿当量桶原油，按现在的生产速度可以持续14年以上。<ref>[https://archive.is/20120718084532/findarticles.com/p/articles/mi_m0EIN/is_2008_Feb_15/ai_n24264728 Exxon Mobil Corporation Announces 2007 Reserves Replacement]</ref>公司在21个国家共有38个[[炼油厂|炼油厂]]，可日处理原油630万桶<ref>{{cite web |url=http://www.exxonmobil.com/Corporate/about_what_refining.aspx |title=Exxon Mobil - Refining and supply |accessdate=2009-01-30 |publisher=Exxon Mobil Corporation |deadurl=yes |archiveurl=https://web.archive.org/web/20081114193730/http://www.exxonmobil.com/Corporate/about_what_refining.aspx |archivedate=2008-11-14 }}</ref><ref>{{cite web |url=http://www.exxonmobil.com/Corporate/about_who_profile.aspx |title=Exxon Mobil - Company profile |accessdate=2009-01-30 |publisher=Exxon Mobil Corporation |deadurl=yes |archiveurl=https://web.archive.org/web/20081114041736/http://www.exxonmobil.com/Corporate/about_who_profile.aspx |archivedate=2008-11-14 }}</ref><ref>{{cite web |url=http://www.exxonmobil.com/Corporate/history/about_who_history.aspx |title=Exxon Mobil - Our history |accessdate=2009-01-30 |publisher=Exxon Mobil Corporation}}</ref>。世界石油企业中最大的一个<ref>{{cite web |url=http://www.ft.com/cms/s/0/0b2f7b36-cda0-11db-839d-000b5df10621,dwp_uuid=0bda728c-ccd0-11db-a938-000b5df10621.html |title=FT's profile of ExxonMobil |accessdate=2008-04-21 |publisher=Financial Times}}</ref>，2008年日产量为392万1千[[桶_(单位)|原油当量桶]]<ref>{{cite web

埃克森美孚被主要科学组织批评，因为它为了让全球变暖问题变得不确定而发起宣传错误信息的活动。<ref>{{cite web |url=http://www.ucsusa.org/news/press_release/ExxonMobil-GlobalWarming-tobacco.html |title=Scientists' Report Documents ExxonMobil’s Tobacco-like Disinformation Campaign on Global Warming Science |accessdate=2009-04-24 |publisher=Union of Concerned Scientists |deadurl=yes |archiveurl=https://web.archive.org/web/20090429210005/http://www.ucsusa.org/news/press_release/ExxonMobil-GlobalWarming-tobacco.html |archivedate=2009-04-29 }}</ref><ref>{{cite web |url=http://royalsociety.org/document.asp?tip=1&id=5851 |title= Royal Society and ExxonMobil |accessdate=2009-04-24 |publisher= The Royal Society}}</ref>

1998年12月1日，美国石油界的老大埃克森出资766亿美元购买了美孚。2009年12月14日。埃克森美孚以總額410億美元，收購美國最大的天然氣生產商XTO能源公司，XTO股東每1股可換得0.7098股的埃克森美孚股票，換算之下為XTO能源每股51.69美元，溢價25％。還將承擔后者100億美元的債務。這項交易預定在2010年第2季完成。

== 组织 ==
埃克森美孚总部位于[[美国|美国]][[德克萨斯州|德克萨斯州]]的[[歐文_(德克薩斯州)|欧文]]。该公司在全世界销售以[[埃克森|埃克森]]、[[美孚|美孚]]、[[埃索|埃索]]为品牌的产品。它旗下还有诸如加拿大的[[帝国石油|帝国石油有限公司]]（69.6%所有权）以及名为{{link-en|海河海事公司|SeaRiver Maritime}}的石油运输公司等数百家子公司。上游部门主导了公司的现金流，占公司收入大约70%。据公司2006年集团员工报告，公司全球共有82000名员工，其中大约有4000人在[[费尔法克斯|费尔法克斯]]下游总部和27000人在其[[休斯顿|休斯顿]]上游总部。

現任公司董事長兼CEO為{{le|达伦·W·伍兹|Darren W. Woods}}。

== 香港 ==
[[File:HK_EssoOilStation_MOS.JPG|thumb]][[馬鞍山_(香港市鎮)|馬鞍山]]的埃克森美孚[[加油站|加油站]]]]
埃克森美孚擁有全香港最龐大的[[加油站|加油站]]網絡（包括集團旗下埃索 Esso 、美孚及[[東方油站|東方油站]]），並以標準石油氣以及美孚石油氣為名義供應罐裝石油氣和中央石油氣，辦事處位於[[中環廣場|中環廣場]]22樓，美孚之油庫舊址亦重建為現在之[[美孚新邨|美孚新邨]]。

2007年1月，[[香港海關|香港海關]]發現位於[[屯門市中心|屯門市中心]]一個埃克森美孚的油站的入油槍不準確，使顧客每次多付5%的油費，涉嫌違反[[香港度量衡制度|度量衡條例]]。當局已在4月票控埃克森美孚。
