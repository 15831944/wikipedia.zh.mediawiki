'''埃因納博區'''（{{lang-so|''Degmada Caynabo''}}）是[[索馬利亞|索馬利亞]]的一個[[索馬利亞行政區劃|區]]，位於該國北部的[[蘇爾州|蘇爾州]]<ref>[http://www.statoids.com/yso.html Districts of Somalia], Statoids</ref>，而蘇爾州實際上由[[索馬利蘭|索馬利蘭]]政府控制。首府{{tsl|en|Aynabo|埃因納博}}。

== 人口 ==
最近一次的人口普查，顯示該地的人口共有97,188人，居民主要為{{tsl|en|Isaaq|伊薩克族}}的Habar Jeclo人。<ref>{{cite web|title=EASO Country of Origin Information Report Somalia Security Situation|url=https://www.ecoi.net/file_upload/1226_1457606427_easo-somalia-security-feb-2016.pdf}}</ref>

== 村落 ==
埃因納博區下轄以下的村落：
* Wadaamagoo
* Habrai-Heshai
* Ceel-Dhaab
* Oog
* BadWayn
* Gowsaweyn
* Qori-Dheere
