'''埃丁森·罗伯托·卡瓦尼·戈麦斯'''（{{lang-es|'''Edinson Roberto Cavani Gómez'''}}<ref>{{cite web |url=http://www.ilpalermocalcio.it/en/0809/scheda.jsp?id=4962 |publisher=US Città di Palermo |title=Edinson Roberto Cavani Gómez |accessdate=2008-09-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20080923175254/http://www.ilpalermocalcio.it/en/0809/scheda.jsp?id=4962 |archivedate=2008-09-23 }}</ref>，{{bd|1987年|2月14日|catIdx=Cavani}}）是一名[[乌拉圭|乌拉圭]][[足球员|足球员]]，司职[[前锋_(足球)|前鋒]]，目前效力于[[法甲|法甲]]俱樂部[[巴黎聖日耳門|巴黎聖日耳門]]。

== 俱乐部生涯 ==
卡瓦尼绰号 “El Matador”(意为[[斗牛士|斗牛士]]),<ref name="cbcsports">{{cite web|url=http://www.cbc.ca/sports/fifau20/newsmakers/cavani.html|publisher={{tsl|en|CBC Sports|}}|title=Edinson Cavani: South America's best-kept secret|accessdate=2007-09-05}}</ref>他于12岁的时候移居到[[蒙得维的亚|蒙得维的亚]]，然后加入了当地球会[[多瑙河足球俱乐部|达努比奥]]的青训营，2006年进入了俱乐部的一线队，他在第一个赛季就攻入9球並入选了最佳阵容。<ref>{{cite web|url=http://www.gazzetta.it/gazzetta/scalcio/ricerca_index.jhtml?PageToLoad=SchedaGiocatore&idAnagrafica=92412|title=Scheda – Edinson Cavani|accessdate=2007-09-05|language=it|publisher=[[米兰体育报|米兰体育报]]|deadurl=yes|archiveurl=https://web.archive.org/web/20070926220433/http://www.gazzetta.it/gazzetta/scalcio/ricerca_index.jhtml?PageToLoad=SchedaGiocatore&idAnagrafica=92412|archivedate=2007-09-26}}</ref>

2007年的南美洲青年锦标赛裡，卡瓦尼的优异表现更引来了诸如[[尤文图斯|尤文图斯]]和[[AC米兰|AC米兰]]这样的豪门俱乐部的目光。<ref>{{cite web|url=http://www.calciomercato.biz/news/news425276.html|language=it|author=Calciomercato.biz|title=Cavani, il Palermo sorpassa tutti|accessdate=2007-01-30|work=|deadurl=yes|archiveurl=https://web.archive.org/web/20070315213722/http://www.calciomercato.biz/news/news425276.html|archivedate=2007-03-15}}</ref>不过在这年的1月29日[[巴勒莫足球俱乐部|巴勒莫]]老板赞帕里尼宣布抢先签下了这位乌拉圭天才射手。<ref>{{cite web|url=http://www.repubblica.it/news/sport_gen/sport_n_2014424.html|language=it|author={{tsl|en|La Repubblica|}}|title=Calcio, Palermo; Zamparini: abbiamo preso Cavani|accessdate=2007-01-30|work=}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>31日，俱乐部官方声明完成了这项交易。<ref>{{cite web |url=http://www.gazzetta.it/Calcio/Squadre/Palermo/Primo_Piano/2007/01_Gennaio/31/CAVANI.shtml |language=it|author=[[米蘭體育報|米蘭體育報]] |title=Palermo, arriva il gioiello Cavani|accessdate=2007-01-31|format=|work=}}</ref>

===巴勒莫===
卡瓦尼的欧洲首演是2007年3月11日巴勒莫主场对阵[[佛罗伦萨足球俱乐部|佛罗伦萨]]的联赛。<ref>{{cite web|title=Tactical Formation|work=Football-Lineups.com
|url=http://www.football-lineups.com/games/?game=5061 | accessdate=March 14, 2007}}</ref>在第55分钟本队0-1落后时卡瓦尼打进了一粒令人印象深刻的进球，这球甚至让人联想到了[[范巴斯滕|范巴斯滕]]在1988年[[欧洲冠军杯|欧洲冠军杯]]决赛里打进的那个经典入球。<ref name="cbcsports" />而在意甲的第二个赛季，卡瓦尼就同[[阿毛里|阿毛里]]和{{tsl|en|Fabrizio Miccoli|米科利}}对主力位置展开了争夺。

2008年夏天，随着巴勒莫原来的主力中锋[[阿毛里|阿毛里]]转投[[尤文图斯|尤文图斯]]，卡瓦尼开始与前尤文图斯前锋{{tsl|en|Fabrizio Miccoli|米科利}}搭档锋线，经过两个赛季虽然俱乐部频繁更换主教练，但是这对前锋组合一直是他们的首选。2010年4月，俱乐部与卡瓦尼续约到2014年夏天。<ref>{{cite news |url=http://www.mediagol.it/articolo.asp?idNotizia=176625 |publisher=Mediagol.it |accessdate=2010-05-05 |date=2010-04-29 |title=Ag.Cavani: “Contratto? C´è stato adeguamento" |language=it |deadurl=yes |archiveurl=https://web.archive.org/web/20100503091703/http://www.mediagol.it/articolo.asp?idNotizia=176625 |archivedate=2010-05-03 }}</ref>

2010年6月28日巴勒莫队老板赞帕里尼称俱乐部接到了来自于[[英超联赛|英超联赛]]强队[[托特纳姆热刺|托特纳姆热刺]]的报价，他们正是想要卡瓦尼以及丹麦中卫[[西蒙·克亚尔|西蒙·克亚尔]]，赞帕里尼表示如果托特纳姆能开价3500万欧元把两人打包购买的话他将会答应这笔生意。 <ref>{{cite web |url=http://www.skysports.com/story/0,19528,11661_6236539,00.html |language=en|author=[[天空新聞台|天空新聞台]] |title=Spurs 'Bid' For Palermo Duo|accessdate=2010-06-28|format=|work=}}</ref>不过热刺方面见对方狮子大开口于是很快就叫停了这桩转会。7月17日卡瓦尼被出售给了意甲另外一家球会[[那不勒斯足球俱乐部|那不勒斯]]，签约五年。

===那不勒斯===
2010年卡瓦尼加盟[[那不勒斯足球俱乐部|那不勒斯]]後，穿上7號球衣；連續三個賽季，出場數和總進球分別都達到40場和30球以上。

2012-13賽季以29球領銜意甲射手榜，以場均0.85球的效率排全歐第四，僅次於[[基斯坦奴·朗拿度|基斯坦奴·朗拿度]]、[[拉达梅尔·法尔考|拉达梅尔·法尔考]]和[[梅西|梅西]]，成為球队获得意甲亚军的头号功臣，而那不勒斯对卡瓦尼标价高达6300万欧元<ref>http://www.chinanews.com/ty/2013/05-21/4841942.shtml</ref>，卡瓦尼因此成為歐洲市場上最炙手可熱的射手。

===巴黎圣日耳曼===

2013年6月13日，卡瓦尼加盟法国球队[[巴黎圣日耳曼足球俱乐部|-{zh-hans:巴黎圣日耳曼;zh-hk:巴黎聖日門;zh-tw:巴黎聖日耳曼;}-]]，合约为期五年。转会费大约6千4百万欧元。这是历史上第五高的身价，打破了这个夏季之前由[[拉达梅尔·法尔考|法尔考]]创造的6千万欧元法国足球转会记录。在巴黎期間傳出與伊布不和，原因為兩人爭奪中鋒位置互不相讓，卡瓦尼受訪時直言"伊布是好隊友，但不是好朋友"，話雖如此卡瓦尼仍競爭不過伊布，長期擔任大巴黎邊鋒。

2016-17賽季由於巨神伊布離隊，卡瓦尼取得夢寐以求的中鋒位置。在歐聯十六強的次回合對陣巴塞時，憑著一球'Real Thread Cavani'勁射取得作客入球，但最終仍被奇蹟絕殺。
2017-18球季，巴黎聖日耳曼購入內馬爾，二人亦傳出因爭奪點球的主射資格而傳出不和

2018年1月27日，卡瓦尼在11分鐘打進1球，助大巴黎於聯賽終場4：0勝蒙彼利埃，也讓他賽後打破由伊巴保持的156球隊史紀錄，以157球躍居隊史進球王。

== 技術特點 ==
卡瓦尼是全能型前鋒，除了能扛在最前端擔任中鋒以外亦能客串邊鋒，技術、身體素質、頭球與射術都有上乘水準；最難得的是卡瓦尼的防守跑動相當積極，以前鋒來說他的防守態度是數一數二的勤奮，是世界上前場的巨星之一。

== 国家队生涯 ==
卡瓦尼在2007年1月入选了乌拉圭U20国家队，参加了当年举行的南美洲青年足球锦标赛，卡瓦尼在9场比赛里打进7球令其大放异彩，乌拉圭队则赢得了本次杯赛的第三名。

2008年2月6日卡瓦尼首次入选成年国家队，而成年国家队首演2-2打平哥伦比亚的比赛里卡瓦尼就打进了一个进球。

==个人==
卡瓦尼的同母异父哥哥[[瓦尔特·古格里埃尔姆奥内|瓦尔特·古格里埃尔姆奥内]]也是一名足球运动员，曾效力于[[中国足球甲级联赛|中国足球甲级联赛]]球队[[北京理工大学足球俱乐部|北京理工大学]]。<ref>[http://es.fifa.com/tournaments/archive/u20worldcup/canada2007/news/newsid=541366.html Cavani, el nuevo matador de la Celeste]</ref>
