《'''机动战士GUNDAM NT'''》（{{lang-ja|機動戦士ガンダムNT}}）是[[日本|日本]][[日本动画|动画]]系列GUNDAM系列中的一部[[動畫電影|动画电影]]<ref name="info180420">{{Cite web|url=https://www.gundam.info/news/video-music/news_video-music_20180420_23.html|title=『UC』の先を描く宇宙世紀サーガ最新作『機動戦士ガンダムNT（ナラティブ）』2018年11月劇場公開！|website=GUNDAM.INFO|date=2018-04-20|accessdate=2018-04-20}}</ref>。2018年11月30日在日本全国90所[[电影院|电影院]]上映<ref name="info180816">{{Cite web|url=https://www.gundam.info/news/video-music/news_video-music_20180816_16.html|title=『機動戦士ガンダムNT』11月30日より全国ロードショー！特報映像・キャストなど最新情報公開！|website=GUNDAM.INFO|date=2018-08-16|accessdate=2018-08-19}}</ref>，2019年1月11日於[[台灣|台灣]]上映，1月24日於[[新加坡|新加坡]]上映，2月14日於[[香港|香港]]上映。

== 概要 ==
作为在[[宇宙世紀|宇宙世纪]][[GUNDAM系列作品|GUNDAM系列作品]]中继《[[机动战士GUNDAM_F91|机动战士GUNDAM F91]]》（1991年）之后27年来的第一部新作电影。本作也是作为在各种媒体上展开，讲述《[[機動戰士GUNDAM_UC|机动战士GUNDAMUC]]》（以下简称《UC》）之后UC世纪发生的故事的《UC NexT 0100》计划中发布的第一部作品<ref name="gigazine180420">{{Cite news|url=https://gigazine.net/news/20180420-gundam-new-project/|title=「ガンダムシリーズ 新作発表会」でシリーズ最新作『機動戦士ガンダムＮＴ』と最新プロジェクト情報を発表|work=[[GIGAZINE|GIGAZINE]]|date=2018-04-20|accessdate=2018-04-20}}</ref>。

本作是动画版《UC》的续集故事，以担任动画版剧本的[[福井晴敏|福井晴敏]]先生此前执笔的小说《机动战士GUNDAMUC》的外传短篇小说《[[機動戰士GUNDAM_UC|机动战士GUNDAMUC 狩猎不死鸟]]》为[[原案|原案]]，描述动画版《UC》一年后的故事<ref>{{Twitter status2|g_narrative|987635005064015873|4=機動戦士ガンダムＮＴの2018年4月21日19時10分のツイート|5=2018-04-21}}</ref>。

动画制作由[[SUNRISE|SUNRISE]]第一工作室进行，动画版《UC》原班的工作人员也有大量参与。另一方面，起用大量30多岁的动画师作为主创作群，推動新一代對作品创作的参与<ref name="GMG2018S">{{Harvnb|GMG2018S|2018|pp=5-7|loc=【特集】ガンダムシリーズの未来(1) 機動戦士ガンダムNT UC Next 0100 PROJECTが描く宇宙世紀の目的 【インタビュー】サンライズ 小形尚弘}}</ref>。

== 故事 ==
在机动战士高达 UC 事件之后，这场被称为“拉普拉斯事件”（Laplace Incident）的冲突随着[[新吉恩|新吉恩]]残余团体“带袖的”解散而结束。事件中活跃的独角兽高达和报丧女妖都被封存了，再也没有人见过他们。根据《宇宙世纪宪章》的原始草案，[[新類型人|新類型人]]的存在及其被承诺的权利也已为公众所知。

一年后的U.C.0097，世界仍然基本没有变化。但在此时两年前失踪的「独角兽高达3号机 凤凰」突然重新出现。为了捕捉逃亡的机动战士，[[地球聯邦|联邦]]和新吉恩决定合作进行捕捉行动，称为“狩猎不死鸟”行动。

== 推广 ==
本作在2018年4月20日举行的“GUNDAM系列作品新作发表会”上发表制作，发表了主视觉图和主创人员等<ref name="info180420" />。之后，在同年8月16日公布了主要角色以及各种相关设定<ref name="info180816" />。

预售票是在制作发表后不久就推出的，第一款是使用Key Visual的clear file<ref name="info180420" />，第二款则是设定由剧中的匿名人士所整理的数量限定特制小册子《报告书-U.C.0097-》<ref name="info180816" />。另外，将于2018年10月19日推出与同一时期剧场上映的《[[哥斯拉_(動畫電影)|哥斯拉 噬星者]]》合作的插图（带有clear file的预售票）<ref>{{Cite news|url=https://natalie.mu/comic/news/302819|title=「GODZILLA」×「ガンダムNT」“金色”コラボ！クリアファイルが数量限定販売|newspaper=[[ナタリー_(ニュースサイト)|コミックナタリー]]|publisher=ナターシャ|date=2018-10-08|accessdate=2018-10-08}}</ref>。

; ARX-014 银弹·镇压者

: 在本作中，巴纳吉·林克斯驾驶的本机涂装变更为深灰色且头部进行了部分修改，配备了原属于RX-0 独角兽高达的光束马格南步枪，由于光束马格南步枪在非RX-0系列机体上使用时会损坏机体手臂的缘故而在背包处携带了4条额外的手臂以供替换。
