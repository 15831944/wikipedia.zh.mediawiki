'''貴族院'''（{{jpn|j=貴族院|hg=きぞくいん|rm=Kizoku In}}）是昔日[[日本|日本]][[帝國議會_(日本)|帝國議會]]的兩個[[議院|議院]]之一，從1890年（[[明治|明治]]23年）11月29日至1947年（[[昭和|昭和]]22年）5月2日間設置，略稱「貴院」。和[[眾議院_(日本)|眾議院]]同格，但是眾議院擁有[[預算|預算先議權]]<ref name="shouwa">『事典 昭和戦前期の日本』 37頁。</ref>。由非[[民選|民選]]的[[日本皇族|皇族]]議員、[[華族_(日本)|華族]]議員、勅任議員所構成，不實施[[解散国会|解散]]制度<ref name="shouwa"/>，議員多數是終身任期。貴族院廢止後，其地位由[[參議院_(日本)|參議院]]取代。

== 概要 ==
[[Image:Fumimaro_Konoe_President_of_the_House_of_Peers.jpg|thumb]]11年）、[[貴族院議長|貴族院議長]][[近衛文麿|近衛文麿]]在貴族院開院式朗讀勅語奉答文]]
議院或議員之權限依[[議院法|議院法]]、貴族院令（明治22年勅令第11號）<ref name="shouwa"/>、其他法令規定。

議員的任期原則是7年，皇族議員、華族議員之中的公爵・侯爵議員、勅任議員之中的勅選議員是終身議員。華族議員之中的伯爵・子爵・男爵議員由同爵位者互選產生<ref name="shouwa2">『事典 昭和戦前期の日本』 38頁。</ref>。議員的[[歲費|歲費]]（議員報酬）由議院法規定。

1890年（[[明治|明治]]23年）開會的第1回通常會到1946年（昭和21年）開會的第92回通常會為止，議員總數約250名到400名。第92回議會停會當時的議員總數共373名。

==資格==
=== 皇族議員 ===
滿18歲的[[皇太子|皇太子]]・皇太孫和滿20歲的其他皇族男子自動成為議員（貴族院令第2條）。無固定人數、也無歲費<ref name="shouwa"/>。

=== 華族議員 ===
華族議員從[[華族_(日本)|華族]]中選任。依爵位不同，選任方法、任期等規定也相異。

====公爵議員・侯爵議員====
滿25歲的[[公爵|公爵]]・[[侯爵|侯爵]]自動成為議員（貴族院令第3條）。無固定人數、也無歲費。

1925年（大正14年），貴族院令改正（大正14年勅令第174號），年齡改成滿30歲<ref name="shouwa"/>。

====伯爵議員・子爵議員・男爵議員====
滿25歲的伯爵・子爵・男爵之中同爵位者互選產生（貴族院令第4條第1項）。任期7年。互選方法等依貴族院伯子男爵議員選舉規則（明治22年勅令第78號）規定<ref name="shouwa2"/>。選舉採[[全票制|完全連記制]]。

=== 勅任議員 ===
====勅選議員====
對國家有勳勞、或有學識的30歲以上的男子，由[[天皇|天皇]]任命（貴族院令第5條）<ref name="shouwa2"/>。勅選議員是終身議員（貴族院令第5條）。

1905年（明治38年）以後，固定人數在125人以內。

====帝國學士院會員議員====
1925年（大正14年），新設。[[帝國學士院|帝國學士院]]會員之中30歲以上的男子互選。任期7年。固定人數4名。互選方法等依貴族院帝國學士院會員議員互選規則（大正14年勅令第233號）規定<ref name="shouwa2"/>。

====多額納稅者議員====
土地或工業・商業的多額[[國稅|直接國稅]]納付者之中30歲以上的男子互選（貴族院令第6條）。任期7年。互選方法等依貴族院多額納稅者議員互選規則（大正14年勅令第234號）規定<ref>『事典 昭和戦前期の日本』 38-39頁。</ref>。

1925年（大正14年），固定人數在66人以內。1944年（昭和19年），改成固定人數在定員67人以內。

====朝鮮勅選議員・台灣勅選議員====
[[朝鮮日治時期|朝鮮]]或[[台灣日治時期|台灣]]在住滿30歲以上的男子之中有名望者由勅任產生。固定人數在10人以內<ref name="shouwa3">『事典 昭和戦前期の日本』 39頁。</ref>。

1946年（昭和21年），放棄朝鮮・台灣的統治權後廢止<ref name="shouwa3"/>。

== 歷史 ==
貴族院關係法令之起草由[[金子堅太郎|金子堅太郎]]担當。金子在當初使用「[[元老院|元老院]]」的名稱，但是被[[伊藤博文|伊藤博文]]否定，結果定為「貴族院」。

[[第二次世界大戰|第二次世界大戰]]後，也参加[[日本國憲法|日本國憲法]]的審議。1947年（昭和22年），由於大日本帝國憲法廢止和日本國憲法施行，貴族院和華族制度廢止。貴族院的議場由新設的[[參議院_(日本)|参議院]]繼承。

== 院內會派 ==
以下是主要的院內會派。
; [[火曜會|火曜會]]
: [[公爵|公爵]]和[[侯爵|侯爵]]議員組成的會派。少數派，但是全由終身議員構成，所以擁有強大的影響力。[[德川家達|德川家達]]（第4代議長）、[[近衛文麿|近衛文麿]]（第5代議長）、[[德川圀順|德川圀順]]（第7代議長）、[[德川家正|德川家正]]（第8代議長）等所屬。
; [[研究會_(貴族院)|研究會]]
: 1891年（明治24年）11月，由反對勤儉尚武建議案的同志會、有志者會、[[千家尊福|千家尊福]]等人的團體結成。結成時是以[[子爵|子爵]]議員為中心，和藩閥政府合作，之後標榜「不偏不黨」。1892年，組織互選團體尚友會。擁有許多[[伯爵|伯爵]]・子爵議員，貴族院院內會派中的最大勢力<ref>『事典 昭和戦前期の日本』 224頁。</ref>。之後官僚出身的勅選議員也多數所屬。[[松平賴壽|松平賴壽]]（第6代議長）等所屬。
; [[公正會|公正會]]
: 1919年（大正8年），以[[男爵|男爵]]議員為中心結成。
; [[茶話會|茶話會]]
: [[平田東助|平田東助]]等人為中心結成的[[官僚|官僚]]系勅撰議員的會派。由[[山縣有朋|山縣有朋]]系統相關的議員結成，貴族院中的官僚派・反政黨主義的牙城。
; [[交友俱樂部|交友俱樂部]]
: [[原敬|原敬]]等人的策畫而結成的官僚系勅撰議員的會派。由[[伊藤博文|伊藤博文]]・[[西園寺公望|西園寺公望]]系統相關的議員結成，支持政黨政治，實質上是貴族院中的[[政友會|政友會]]的別働隊。
; [[同成會|同成會]]
: 土曜會的後繼會派，以官僚系勅撰議員為中心。多為親[[立憲民政黨|民政黨]]的議員，實質上是貴族院中的民政黨的別働隊。
; [[三曜會|三曜會]]
: 貴族院議長[[近衛篤麿|近衛篤麿]]所屬。
; [[同和會|同和會]]
: 茶話會的後繼會派，由舊茶話會和無所屬議員為中心結成。有反研究會・反政友會的色彩，和同成會同樣，實質上是貴族院中的民政黨的別働隊。
; [[無所屬俱樂部|無所屬俱樂部]]
: 1941年4月30日發足。由[[廣田弘毅|廣田弘毅]]、[[後藤文夫|後藤文夫]]勅選議員為中心結成。之後，[[東鄉茂德|東鄉茂德]]、[[小林一三|小林一三]]等人也加入。

1920年（大正9年）7月，各會派的所屬者數：研究會143、公正會65、茶話會48、交友俱樂部44、同成會30、無所屬67、計397

1947年（昭和22年）3月，帝國議會終了時各會派的所屬者數：研究會142、公正會64、交友俱樂部41、同成會33、火曜會32、同和會30、無所屬俱樂部、22、無所屬8、計373名

戰後初期，取代貴族院的参議院中，有許多舊貴族院議員出身的當選者，構成標榜不偏不黨的院內會派・[[綠風會|綠風會]]，一時之間作為参議院最大會派對國政擁有相當大的影響力。但是，最後所屬議員被政黨（大多數是[[自由民主黨_(日本)|自由民主黨]]等保守政黨）吸收。

== 內閣總理大臣輩出 ==
大日本帝國憲法下，[[內閣總理大臣|內閣總理大臣]]必須是國會議員，但未必是衆議院議員。現任的眾議院議員擔任首相者，在大日本帝國憲法下的33位首相之中，最早的是[[原敬|原敬]]，還有[[濱口雄幸|濱口雄幸]]、[[犬養毅|犬養毅]]，共3人。

另一方面，現任的貴族院議員擔任首相者以伊藤博文為開端，有[[松方正義|松方正義]]、[[大隈重信|大隈重信]]、[[桂太郎|桂太郎]]、[[西園寺公望|西園寺公望]]、[[高橋是清|高橋是清]]、[[清浦奎吾|清浦奎吾]]、[[加藤高明|加藤高明]]、[[若槻禮次郎|若槻禮次郎]]、[[近衛文麿|近衛文麿]]、[[東久邇宮稔彦王|東久邇宮稔彦王]]、[[幣原喜重郎|幣原喜重郎]]、[[吉田茂|吉田茂]]等多人。
