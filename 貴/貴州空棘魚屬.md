'''貴州空棘魚屬'''（[[學名|學名]]：{{lang|la|''Guizhoucoelacanthus''}}，或譯作'''貴州腔棘魚'''）為[[三疊紀|三疊紀]]中期[[腔棘魚目|腔棘魚目]][[懷特魚科|懷特魚科]]{{r|Geng}}的一[[屬|屬]]，全長45<ref name = Liu>{{ cite journal | author = 刘冠邦; 尹恭正; 罗永明; 王雪华; 王尚彦 | title = "贵州关岭晚三叠世法郎组瓦窑段鱼类化石初步观察" | journal = 古生物学报 | volume = 45 | issue = 1 | pages = 1–20 | year = 2006 | issn = 1000-3118 | url = http://159.226.74.248:8000/pagelinks/139726.pdf }}</ref>至55公分<ref name = Geng/>，屬於中等體型的腔棘魚。

目前已被發表的貴州空棘魚的化石皆出土於[[中國|中國]][[貴州|貴州]]的[[關嶺|關嶺]]，屬於[[法郎組|法郎組]]的[[瓦窯段|瓦窯段]]及[[竹竿坡段|竹竿坡段]]，相當於[[三疊紀|三疊紀]]中期的{{link-en|拉丁尼階|Ladinian}}（約2億4200萬年至2億3500萬年前）。其下已知只有一個物種，即[[模式種|模式種]]'''關嶺貴州空棘魚'''（''G. guanlingensis''），屬名及種名皆取自發現地的名稱，由[[南京大學|南京大學]]的[[古生物學家|古生物學家]][[劉冠邦_(古生物學家)|劉冠邦]]等人於2006年發表與命名；[[正模標本|正模標本]]收藏於[[貴州省地質調查院|貴州省地質調查院]]（編號F001），為一具近乎完整的單一個體化石<ref name = Liu/>。

== 特徵 ==
身體大致呈短紡錘狀。頭部側視略呈三角形，和身體比起來相對的大。{{link-en|顱頂甲|Skull roof}}狹長。脊柱分節約有45節，前六至七節較短。椎體和[[肋骨|肋骨]]未骨化（Unossified）。前背鰭非常偏前，基板（Basal plate）大而略呈半圓形；後背鰭細長呈短圓柱狀，分為水平而較細長的前突和較粗壯的前腹突，其中前腹突深入兩個髓突（Neural spines）間。[[臀鰭|臀鰭]]基板小而細長，遠端略向兩側開展。具有三片較大的[[尾鰭|尾鰭]]，上面的背葉和下面的腹葉互相對稱，後端呈截切狀；中葉明顯分離並向後突出。[[鱗片|鱗片]]薄，上面有扁平的縱向溝，具明顯細刻紋<ref name = Geng/><ref name = Liu/>。

== 分類學 ==
最初在2006年發表的報告中，指出貴州空棘魚具有諸多和其他腔棘魚（大部分物種的化石都不完整）明顯不同的特徵，包括較粗短的身形、明顯呈葉狀的鰭、偏前的前背鰭、未骨化的肋骨等，而在文中本屬最後被歸類至[[腔棘魚科|腔棘魚科]]下。三年後，[[中國科學院|中國科學院]][[古脊椎動物與古人類研究所|古脊椎動物與古人類研究所]]的[[耿丙河|耿丙河]]等人透過新發現的化石標本（編號IVPP V15153），重新修訂2006年的報告中提及的特徵，並藉由[[種系發生學|種系發生學]]指出貴州空棘魚和[[懷特魚科|懷特魚科]]的[[懷特魚|懷特魚]]與[[皮爾逖魚|皮爾逖魚]]形成[[姊妹群|姊妹群]]的關係，認為貴州空棘魚應屬於[[懷特魚科|懷特魚科]]，文中列出[[肉鰭魚|肉鰭魚]]家族的50%多數決共同樹（50% majority-rule consensus tree），其中懷特魚科部分如下：
