'''扬雄'''（{{bd|前53年||18年||CatIdx=Yang,扬}}），一作'''杨雄'''，字'''子雲'''，[[西汉|西汉]]哲学家、文学家、语言学家，蜀郡成都（今[[四川|四川]][[成都|成都]][[郫都区|郫都区]]）人。

== 生平 ==
为人[[口吃|口吃]]，相貌平平，身高偏矮，不能剧谈，专心于潜心思考，早年倾慕[[司马相如|司马相如]]，模仿司马相如的《子虚》、《上林》等赋，常作[[辞赋|辞赋]]，名声远播，如《蜀都賦》，此賦開啟了京都一派題材，[[班固|班固]]《兩都賦》、[[張衡|張衡]]《二京賦》以及[[晉代|晉代]][[左思|左思]]《三都賦》皆受其影響。

大司馬車騎將軍[[王音|王音]]召為門下史。后经蜀人[[楊莊|楊莊]]推荐，[[汉成帝|汉成帝]]命他随侍左右。前11年正月与成帝前往[[甘泉宫|甘泉宫]]，作《甘泉赋》讽刺成帝铺张。十二月又作《羽猎赋》仍然以劝谏为主题。被封[[黄门侍郎|黄门侍郎]]，与[[王莽|王莽]]、[[刘歆|刘歆]]等为同僚。前10年扬雄作《长杨赋》，继续对成帝铺张奢侈提出批评。

扬雄后来认为辞赋为“雕虫篆刻”，“壮夫不为”，转而研究[[哲学|哲学]]。仿《[[论语|论语]]》作《[[法言|法言]]》，模仿《[[易经|易经]]》作《[[太玄|太玄]]》。提出以“玄”作为宇宙万物根源之学说。有人笑他<ref>[[陳善|陳善]]《捫蝨新話》卷一說：“楊子雲作法言，以擬論語。……可發千載一笑。”</ref>，於是他寫了一篇《解嘲》。為了寬慰自己，又寫了一篇《[[s:逐貧賦|逐貧賦]]》。[[王莽|王莽]]当政，拉拢扬雄，任他为中散大夫。他写过《剧秦美新》美化[[王莽|王莽]]，但此文是主动投靠还是避祸之作，还有争论<ref>[[洪邁|洪邁]]在《容齋隨筆》中為其辯護說：“揚雄仕漢，親蹈王莽之變，退托其身於列大夫中，抱道沒齒。世儒或以《劇秦美新》貶之；是不然，此雄不得已而作也。夫誦述[[新莽|新莽]]之德，止能美于暴秦，其深意固可知矣。序所言配五帝冠三王，開闢以來未之聞，直以戲莽爾。”</ref>。

后在[[天祿閣|天祿閣]]校书，写作，进行[[语言学|语言学]]研究。曾著《[[輶轩使者绝代语释别国方言|方言]]》，叙述[[西汉|西汉]]时代各地方言，为研究古代语言的重要资料。但是因弟子[[劉棻|劉棻]]牵连，有人前来[[逮捕|逮捕]]扬雄，扬雄恐惧而[[跳楼|跳楼]]，未死，后得免，[[京師|京師]]諺曰：“惟寂寞，自投閣。”後召為[[大夫|大夫]]，默默无闻而终。

== 思想 ==
[[File:Yang_xiong.jpg|thumb]]

他提出以“玄”作为[[宇宙|宇宙]]万物根源的学说，强调如实的认识自然现象，并认为“有生者必有死，有始者必有终”，驳斥了[[方士|方士]]的学说。<!--在社会[[伦理|伦理]]上，认定“人之性也善恶混，修其善则为善人，修其恶则为恶人”。-->

在[[人性|人性論]]上，揚雄於《法言·修身》說：「人之性也，善惡混，修其善，則為善人；修其惡，則為惡人。」他以氣稟言人的實然之性，採[[告子|告子]]、[[荀子|荀子]]「化性起偽」的思路，謂善惡相混的人性有待後天外鑠性的教化。他標準「[[君子|君子]]」為教育的目標，教學的內容是禮義，「視、聽、言、貌、思」的導正是陶塑品德的必要重點。其中「思」是心的發用所在，揚雄稱心的難測之用為「神」，為教化的關鍵所在。

== 評價 ==
* [[張衡|張衡]]稱他「妙極道術」。
* [[朱熹|朱熹]]認為他「拙底工夫」。

== 文学形象 ==
扬雄的形象常作为潜心于学术，不慕名利，不谙世故的学者的典型出现在后世的文学作品中。
* [[左思|左思]]《咏史·济济京城内》（八首其四）:“寂寂扬子宅，门无卿相舆。寥寥空宇中，所讲在玄虚。言论准仲尼，辞赋拟相如。悠悠百世后，英名擅八区。”
* [[卢照邻|卢照邻]]《长安古意》:“寂寂寥寥扬子居，年年岁岁一床书。独有南山桂花发，飞来飞去袭人裾。”
* [[李白|李白]]《古风·咸阳二三月》（五十九首其八）:“子云不晓事，晚献《长杨》辞。赋达身已老，草《玄》鬓若丝。投阁良可叹，但为此辈嗤。”
* [[李白|李白]]《古风·一百四十年》（五十九首其四十六）:“独有扬执戟，闭关草《太玄》。”
* [[李白|李白]]《侠客行》：“谁能书阁下，白首《太玄》经？”
* [[李贺|李贺]]《绿章封事》：“金家香衖千轮鸣，扬雄秋室无俗声。”
* [[戴叔伦|戴叔伦]]《行路难》：“扬雄闭门空读书，门前碧草春离离。”
