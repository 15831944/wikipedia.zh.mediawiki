'''扬·奥韦·瓦尔德内尔'''（{{bd|1965年|10月3日}}，'''Jan-Ove Waldner'''，[[瑞典|瑞典]][[乒乓球|乒乓球]]运动员。因其打法變化多端，並不拘泥于任何一種打法，打破人们对乒乓球攻防的认知，故被球迷昵稱為「遊擊隊長」。他在[[瑞典|瑞典]]和[[中国|中国]]都很受欢迎，被中国的观众亲切的称为'''老瓦'''。

瓦尔德内尔和[[白俄罗斯|白俄罗斯]]的[[萨姆索诺夫|萨姆索诺夫]]、[[克罗地亚|克罗地亚]]的[[普里莫拉茨|普里莫拉茨]]都是20世纪末至21世纪初的赛场上，来自欧洲的[[乒乓球|乒乓球]]顶尖选手，他们被乒乓球迷并称为“'''欧洲三虎'''”。

== 生涯回顧 ==
瓦尔德内尔15岁的时候就在欧洲锦标赛取得了很好的成绩。他和其他年轻人在1980年代初到[[中国|中国]]参加了一个乒乓球训练营，见识了众多中国乒乓球好手的绝技。

1990年代初期，以瓦尔德内尔、[[约尔根·佩尔森|佩尔森]]、[[米克尔·阿佩伊伦|阿佩伊伦]]、[[彼得·卡尔松|卡尔松]]等人为核心的瑞典乒乓球男队成为了乒坛的霸主，连夺三届世乒赛团体冠军（1989年、1991年、1993年），缔造了“瑞典王朝”。

1992年，瓦尔德内尔于[[1992年夏季奥林匹克运动会|巴塞罗那奥运会]]乒乓球男子单打决赛中，击败法国选手[[盖亭|盖亭]]，成为首位[[世乒赛|世乒赛]]、[[乒乓球世界杯|世界杯]]和[[奥运会|奥运会]]男單“[[大满贯_(中国体坛)|大满贯]]”得主。

1995年，瓦尔德内尔在天津世乒赛团体决赛中连胜[[王涛　(乒乓球运动员)|王涛]]、[[马文革|马文革]]两人，但瑞典队最终以2：3负于[[中国国家乒乓球队|中国队]]。此战过后，瑞典队的辉煌时期结束，战绩每况愈下。瓦尔德内尔更在一年后的[[1996年夏季奥林匹克运动会|亚特兰大奥运会]]中，被[[加拿大|加拿大]]选手[[黄文冠|黄文冠]]爆冷击败。正当所有人都认为，他将会淡出人们视线的时候，他却在一年后的[[1997年世界乒乓球锦标赛|曼彻斯特世乒赛]]中时隔八年再获男单冠军，同时创造了单打7轮比赛未失一局的记录。其后又在2000年的[[吉隆坡|吉隆坡]]世乒赛中发挥神勇，击败了势头正盛的[[刘国梁|刘国梁]]，帮助瑞典队再次夺得团体冠军。同年的[[2000年夏季奥林匹克运动会|悉尼奥运会]]，瓦尔德内尔在单打半决赛上再次击败[[刘国梁|刘国梁]]杀入决赛，但最终2:3负于另一位中国选手[[孔令辉|孔令辉]]，遗憾地与第二次奥运单打冠军擦肩而过。

[[2004年夏季奥林匹克运动会|2004年雅典奥运会]]，瓦尔德内尔先在双打比赛中，与[[约尔根·佩尔森|佩尔森]]爆冷击败中国组合[[王皓_(乒乓球运动员)|王皓]]/[[孔令辉|孔令辉]]，其后又在单打比赛中杀入半决赛，负于当届冠军[[柳承敏|柳承敏]]而止步，铜牌争夺战中则负于[[王励勤|王励勤]]，获得第四名。

2006年[[2006年世界乒乓球团体锦标赛|不莱梅世乒赛]]后，41岁的瓦尔德内尔宣布退出瑞典国家队，从此淡出乒坛。

== 评价 ==
在中国，他是最受欢迎的瑞典人和最受欢迎的运动员之一，被称为'''常青树'''，人们经常来看他打球。他的球员生涯延续了20余年，但年龄没有阻挡他的体育生涯。直到职业生涯的后期，他仍然可以与小他10余岁的新生代选手抗衡。
他为横拍两面弧圈打法的发展做出极大贡献，他首先开始使用并得到推广的技术有：直拍板型发球，反手侧切，正手下切，站球台中间发球等等。

== 個人 ==
*中国是瓦尔德内尔的半个故乡。他曾在[[北京|北京]]三里屯南街开了一家餐厅，叫做'''维京锐点'''。
*他是足球队[[哈马比|哈马比]]的支持者。
