'''扬中市'''是[[中華人民共和國|中華人民共和國]][[江蘇省|江蘇省]][[鎮江|鎮江市]]轄下的[[縣級市|縣級市]]。

== 历史 ==
扬中的前身是[[太平洲|太平洲]]，隶属于[[镇江府|镇江府]]的[[丹徒县|丹徒]]、[[丹阳县|丹阳]]，[[常州府|常州府]]的[[武进县|武进]]，[[扬州府|扬州府]]的[[江都县|江都]]、[[泰兴县|泰兴]]等四府六县。

清末，在太平洲地方设立[[太平厅|太平厅]]，隶属于镇江府。[[中华民国|中华民国]]建立后，废[[厅_(行政区划)|厅]]，改为[[太平县_(中华民国)|太平县]]。1914年因与[[安徽省|安徽省]][[太平县_(天宝)|太平县]]同名，而更名为[[扬中县|扬中县]]。

1994年撤县设市。

== 地理 ==
扬中市的主體為[[長江|長江]]上的一個島嶼，面积332平方公里，人口33.78万（2011年），位于东经119度、北纬32度，是仅次于[[崇明|崇明]]的长江第二大岛。扬中的地名是因其位于扬子江中而得名。

扬中市北、东隔长江主航道與[[泰州市|泰州市]][[高港区|高港区]]、[[泰兴市|泰兴市]]、[[揚州市|揚州市]]、[[镇江|镇江]]江北属地[[高桥镇_(镇江市)|高桥镇]]相望，西南隔长江南汊与[[鎮江市|鎮江市]][[镇江新区|镇江新区]]、[[丹阳市|丹阳市]]及[[常州市|常州市]][[新北区|新北区]]相望。

1994年，建成由民间集资建造（扬中市地方政府筹资建造）的[[揚中長江大橋|揚中長江大橋]]和2004年建成的[[扬中长江二桥|扬中长江二桥]]分别将扬中本岛及其第二大附岛与江南镇江市区和常州新北区连接。在建的[[泰州长江大桥|泰州长江大桥]]S35镇泰高速跨越长江主航道、长江南汊穿越其市区，至镇江新区姚桥枢纽连接S39江宜高速，使扬中成为沟通扬州、泰州、镇江、常州的重要城际交通枢纽。

新建的[[镇江港扬中港区|镇江港扬中港区]]将使扬中的沿江产业优势得到体现。

即将立项的新{{中国省道名|238|P=江苏}}和{{中国省道名|231|P=江苏}}扩容工程：“镇江—扬中快速通道”线形沿[[镇江新区|镇江新区]]的金港大道东延跨越长江夹江新建[[扬中长江三桥|扬中长江三桥]]直插扬中南城区。该通道为全立交、互通式出入口、全封闭、双向6车道、每小时100公里的准高速设计，将大幅度缩短扬中至镇江老城区的车程时间，预计从扬中新客运中心到镇江左湖互通车时为15分钟。

扬中周边的重要交通设施：镇江境内：镇江老城区—扬中市区快速通道、[[file:China_Railways.svg|15px]][[京沪高铁|京沪高铁]]、{{China Expwy Name|G42|沪宁高速公路}}、[[file:China_Railways.svg|15px]][[沪宁铁路|沪宁铁路]]、[[file:China_Railways.svg|15px]][[沪宁城际高铁|沪宁城际高铁]]、[[镇江港|镇江港]][[大港港|大港港]]港区等；常州境内：[[奔牛机场|奔牛机场]]、常州西绕城高速、江都~宜兴 S39江宜高速（京杭高速）常州港等；泰州境内：泰州~丹阳S35镇泰高速、宁通高速、宁启高速、泰州港等；扬州境内：{{China Expwy Name|G2|[[京沪高速|京沪高速]]}}、[[扬州泰州机场|扬州泰州机场]]、[[file:China_Railways.svg|15px]][[淮扬镇铁路|淮扬镇铁路]]、[[file:China_Railways.svg|15px]][[宁启铁路|宁启铁路]]、[[扬州港|扬州港]]等。

规划建设的[[file:China_Railways.svg|15px]][[淮扬镇铁路|淮扬镇铁路]]及[[file:China_Railways.svg|15px]]五峰山长江公铁路大桥、[[file:China_Railways.svg|15px]][[镇南铁路|镇南铁路]]、[[京沪高速|京沪高速]]南延线、[[扬州泰州机场|扬州泰州机场]]紧邻而建，将进一步完善其周边区域内的立体交通格局。

== 行政区划 ==
扬中市下辖2街道办事处，4个镇：[[三茅街道办事处|三茅街道办事处]]、[[兴隆街道办事处|兴隆街道办事处]]、[[新坝镇|新坝镇]]、[[油坊镇|油坊镇]]、[[八桥镇|八桥镇]]、[[西来桥镇|西来桥镇]]。其中三茅街道办事处面积133.19平方千米，人口13万，占扬中全市40%, 是扬中市委、市政府所在地。

== 经济 、文化==
扬中是一个以电力电器产业为主的“工程电器岛”，电力电器产业占全市经济总量近70%，占国内市场份额的20%以上。

扬中是著名的“[[鱼米之乡|鱼米之乡]]”、“江鲜之乡”。该市江域附近是长江众多[[洄游鱼类|洄游鱼类]]的繁殖地。

扬中水产资源丰富，所盛产的[[刀鱼|刀鱼]]、[[鲥鱼|鲥鱼]]、[[河豚|河豚]]，即所谓的“扬中三鲜”，乃是扬中饮食文化精华之所在。扬中3月左右是吃河豚的季节，每年举办河豚节。

== 方言 ==
扬中市使用的方言“[[扬中话|扬中话]]”是[[江淮方言|江淮方言]][[通泰片|通泰片]]（[[扬中小片|扬中小片]]）的一种。
