==導言==
<big>[[游東村|游東村]]是[[中華民國|中華民國]][[臺灣|臺灣]][[嘉義縣|嘉義縣]][[溪口鄉|溪口鄉]]轄下的行政區。位於[[嘉義縣|嘉義縣]]東北方,北鄰[[雲林縣|雲林縣]][[大埤鄉|大埤鄉]],南鄰[[溪口鄉|溪口鄉]][[溪東|溪東]]社區。面積約3.1391平方公里,行政區包含14個鄰、409戶、1134人口數人<ref>{{cite web |title=嘉義縣溪口鄉戶籍人口登記數 |url=https://minsyong-hro.cyhg.gov.tw/News.aspx?n=29E21B3B6985F761&sms=DF03EAE01C95F9AE |website=嘉義縣民雄鄉戶政事務所 |accessdate=2018-12-15}}</ref>，村里以姓游和邱者居多。[[游厝|游厝]]是[[游東村|游東村]]的主要街區,而157縣道則貫穿整座村落。[[慈濟宮|慈濟宮]]和[[天宋宮|天宋宮]]皆是村內歷史久遠的祭祀場所。[[酸菜|酸菜]]為本地最為知名的特殊產業，產量僅次於鄰近的[[雲林縣|雲林縣]][[大埤鄉|大埤鄉]]，亦是本村主要的經濟來源。</big>

==地理==
<big>[[游東村|游東村]]座落於[[溪口鄉|溪口鄉]]北部，由[[游厝庄|游厝庄]]、[[米粉宮|米粉宮]]、[[牛埔寮|牛埔寮]]等三大聚落組成，華興溪(也稱石龜溪，上游為倒孔山溪）與三疊溪於村的東南處交會，環繞於社區的東南方，位置最北的[[牛埔寮|牛埔寮]]與[[雲林縣|雲林縣]][[大埤鄉|大埤鄉]]接壤，而[[米粉宮|米粉宮]]則位於最南，鄰近三疊溪，[[米粉宮|米粉宮]]部落中最大姓為邱姓，過去便是依靠三疊溪維生，多從事擺渡，使居民得以跨越三疊溪水，與對外聯絡，今日則由157縣道貫穿本村，成為連接南北、對外聯絡的主要道路。</big><ref>{{cite web |title=游東村地理 |url=https://sikou.cyhg.gov.tw/cp.aspx?n=5DA34B341EB31075&s=B56213A33D91A597 |website=嘉義縣溪口鄉公所 |accessdate=2018-12-15}}</ref>

==聚落歷史==
<big>[[游厝庄|游厝庄]]為清朝年間[[溪口鄉|溪口鄉]]四大庄之一，[[游厝庄|游厝庄]]大多數居民皆姓游，但現已有很多人移居外出，以前初來開墾者財富甚多，有一人叫[[游天|游天]]，曾經擔任[[保正|保正]]，其子也擔任過協議員(今縣議員)，也有一子在[[溪口庄|溪口庄]]庄役場(今鄉公所)待過。
[[溪口鄉|溪口鄉]]境內有龍、龜兩穴。龜穴則位於[[游厝庄|游厝庄]]。兩穴使得地方經濟不富裕，但人才輩出，子弟身懷博士資格者高達七、八十人。<ref>{{cite web |title=游厝庄聚落歷史 |url=http://gn.moi.gov.tw/geonames/index.aspx |website=地名資訊服務網 |accessdate=2018-12-15}}</ref>

[[米粉宮|米粉宮]]大多為邱姓，邱姓祖先應是從河南省過來，起初在[[雲林縣|雲林縣]][[元長鄉|元長鄉]]落腳，有四位兄弟遷徙至此，利用三疊溪水製作[[米粉|米粉]]為生，因有兩溪在此會流，於是稱之為雙溪口，直到日本來臺才改為[[溪口庄|溪口庄]]，隸屬台南州嘉義邵，一開始[[米粉宮|米粉宮]]皆是邱姓人家，只有一位姓周者入贅至此為婿，目前社區大部分是後來(約二十幾年前)才興建的，邱姓祖先搬移至此已有上百年了，[[米粉宮|米粉宮]]地名的由來是因製作[[米粉|米粉]]而來，居住於[[米粉宮|米粉宮]]的邱姓族人，過去以製作[[米粉|米粉]]為生，到了民國二、三十年時，也經營擺渡，載運河流兩岸的村民來往，從當地居民的記憶來看，此一聚落的形成晚於[[游厝庄|游厝庄]]。<ref>{{cite web |title=米粉宮聚落歷史 |url=https://sikou.cyhg.gov.tw/cp.aspx?n=5DA34B341EB31075&s=B56213A33D91A597 |website=嘉義縣溪口鄉公所 |accessdate=2018-12-15}}</ref>

[[牛埔寮|牛埔寮]]為座落於[[游東村|游東村]]最北側之聚落，157縣道亦從此經過，第十公墓也設置於此，另有「[[天宋宮|天宋宮]]」建於公墓旁，雖然是陰廟卻香火鼎盛。</big><ref>{{cite web |title=牛埔寮聚落歷史 |url=http://gn.moi.gov.tw/geonames/index.aspx |website=地名資訊服務網 |accessdate=2018-12-15}}</ref>

==教育==
<big>原先舊[[游東慈濟宮|游東慈濟宮]]旁設有托兒所，作為村民年幼子女就讀的場所，如今隨著交通發達，以及[[游東慈濟宮|游東慈濟宮]]的重建，子女皆送至鄰近[[溪口鄉|溪口鄉]]立托兒所就讀。村子內並無設立國、中小學，國小生必須到鄰近的[[溪口國小|溪口國小]]就讀，而國中生則需到鄰近的[[溪口國中|溪口國中]]就學。</big><ref>{{cite web |title=游東村國中小學區 |url=https://edu.cyhg.gov.tw/News4.aspx?n=8B04CBD4471A6D45&sms=745127AC0E5C0036 |website=嘉義縣教育處─ 國小學區劃分表、國中學區劃分表 |accessdate=2018-12-15}}</ref>

==文化資源/文化遺產==
[[File:游東慈濟宮.jpg|thumb]][[File:游東天宋宮.jpg|thumb]]
[[File:游東天宋宮碑文.jpg|thumb]]<big>[[游東慈濟宮|游東慈濟宮]]，關於慈濟宮創立的年份，有兩種說法，一個是1933年相良吉哉《台南州祠廟名鑑》中的1815年（嘉慶乙亥年） ，另外一個是2002年翻譯再版的《台南州祠廟名鑑》中的1833年（道光癸巳年）；然而後者的版本在版本上較新，又有經過重新增修，所以應是前者紀載年份較正確。祭祀[[保生大帝|保生大帝]]的[[慈濟宮|慈濟宮]]是[[溪口鄉|溪口鄉]]具有歷史意義的寺廟之一，位於[[游東村|游東村]][[游家庄|游家庄]]。舊建築因年代久遠而毀損，從2002年開始在原址進行重建工作，重建過程中並未將舊廟留存下來。[[慈濟宮|慈濟宮]]旁邊原有一間小屋，早期曾是目前四、五十歲以上的居民接受國小以下教育的地方，直到分校設立，村民才轉至分校接受教育。農曆三月十五會舉辦祭神大典，現今是[[游東村|游東村]]的信仰中心。<ref>{{cite web |title=游東慈濟宮 |url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/shikou/1006004-CJG |website=文化資源地理資訊系統 |accessdate=2018-12-15}}</ref>
</big>
<big>[[游東天宋宮|游東天宋宮]]，為「陰廟」，本廟主神天宋媽發跡於[[乾隆|乾隆]]年間，舊廟建於1815年，1982年重建並遷移至現今所在地，位於[[游東村|游東村]][[牛埔寮|牛埔寮]]，座落於溪口鄉第十公墓旁，另有兩位副神[[大眾爺|大眾爺]]和[[萬善爺|萬善爺]]，有求必應，早期村民生病、不舒服都會去求，也有外地考生拿准考證來[[天宋宮|天宋宮]]祈求考運。「天宋」是以前顧廟的管理人取的名字。農曆四月十五為[[天宋媽|天宋媽]]的生日，村民多會擺設流水席，早年常有電子花車、電子琴、和鋼管舞表演，後來因經常遭到檢舉，近年來盛況不如以往。</big><ref>{{cite web |title=游東天宋宮 |url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/shikou/1006004-TSG |website=文化資源地理資訊系統 |accessdate=2018-12-15}}</ref>

==特殊產業==
<big>[[酸菜|酸菜]]是本村的主要產物之一，雖然鄰近的[[雲林縣|雲林縣]][[大埤鄉|大埤鄉]]有「酸菜的故鄉」的美名，然而本村所產的[[酸菜|酸菜]]也十分優良，外縣市可以常常見到來自[[游東村|游東村]]外銷的[[酸菜|酸菜]]製品。目前村內仍可見到醃漬[[酸菜|酸菜]]的大井，許多村民也以製作[[酸菜|酸菜]]作為經濟的來源。[[酸菜|酸菜]]的主要原料--「[[芥菜|芥菜]]」在入冬前後十天，大概[[農曆|農曆]]九月至十月時開始栽種，七十天後就可以開始採收，接著將芥菜倒入6～7公尺深的大型水泥井內再倒入一定比例的粗鹽，大約三十天以後，就可以將淹漬好的[[酸菜|酸菜]]撈起，經過加工分級處理，再裝到特製的空桶內，就可運到全國各地販售。</big><ref>{{cite web |title=游東村酸菜 |url=https://sikou.cyhg.gov.tw/cp.aspx?n=5DA34B341EB31075&s=B56213A33D91A597|website=溪口鄉公所 |accessdate=2018-12-15}}</ref>[[File:游東酸菜井_02.jpg|thumb]]
