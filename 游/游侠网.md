'''游侠网'''创立于1999年，為中國大陸最大的[[單機遊戲|單機遊戲]]介紹網站，也有網路遊戲類介紹，在電玩類網站中也動態位居前三，<ref>[http://www.ali213.net/about/ 遊俠歷史]</ref>隨時在線訪問人數達數十萬人。

游侠网最大特點為提供大量的遊戲修正檔與[[遊戲模組|MOD]]檔、漢化檔、修改器下載，站上的論壇也充斥眾多程式設計高手，子板塊有活躍的各種電玩攻略情報和專業文章。多名创作室成员为各大杂志社的特约作者，中国十佳撰稿人的评选中游侠创作室拥有五个席位。但是在国外由于中国未加入版权协会，所以Steam投诉无果。

== 工作組 ==

; 一般組
* 原创组
* 攻略组
* 技术组
* 修改组
* MOD组
* 美工组
; 特化工作組
* 剑阁工作组：游侠会员群众性创作组织，交流联谊平台，製作眾創遊戲、動漫。
* 光荣工作组：[[光荣|光荣]]游戏大区的会员组成，漢化、修改器與攻略等。
* 实况工作组：制作[[实况足球|实况足球]]系列补丁、漢化、修改器等。
* NBA工作组：篮球系列游戏的补丁和MOD面补，修改器等。
* 足球经理工作组：[[足球经理|足球经理]]遊戲区版主和会员建立补丁、漢化、修改器等。
* 金书游戏工作组：主要由金庸群侠传系列专区版主和会员建立，更新自製遊戲[[金書紅顏錄|金書紅顏錄]]和文創作品。
* SIMS梦工厂工作组：专门制作[[模拟人生|模拟人生]]系列游戏补丁、修改器等。
* U.N.I宣传部：为游侠网所发布的补丁，汉化组，论坛活动等进行向其他网站的推广，海外採訪遊戲公司等新聞。
