'''木村繪理子'''（{{lang-ja|木村 絵理子}}，{{bd|1961年|7月5日<ref>《日本聲音製作者名鑑2007》，第290頁，[[小學館|小學館]]，2007年 ISBN 978-4095263021</ref>||}}），[[日本|日本]]女性[[音效指導|音響監督]]<ref name ="akiba" />。{{link-ja|東北新社|東北新社}}外畫製作事業部演出部總監<ref name ="tfc">{{Cite web|url=http://www.vta.tfc.co.jp/instructor/index.php?page=4|script-title=ja:講師ラインナップ|publisher=影像技術學院|accessdate=2018-02-27|language=ja}}</ref>。現職影像技術學院聲優科講師<ref name ="tfc" />。出身於[[神奈川縣|神奈川縣]][[橫濱市|橫濱市]]<ref name ="mainichi" />。

== 人物簡介 ==
早年在影像製作公司上班，至1989年進入東北新社<ref name ="mainichi">{{Cite web|url=https://mainichi.jp/articles/20160314/ddm/018/070/066000c|script-title=ja:東北新社外画制作事業部演出部長·木村絵理子さん|publisher=[[每日新聞|每日新聞]]|accessdate=2018-03-02|language=ja}}</ref>開始。當過國外電影製作事業部演出部長<ref name ="akiba"/>，2016年4月1日起升任同部門的總監<ref>{{Cite web|url=https://www.nikkei.com/article/DGXLMSJJ00101_Z20C16A3000000/|title=人事、東北新社|publisher=[[日本經濟新聞|日本經濟新聞]]|accessdate=2018-03-02|language=ja}}</ref>。

作為音響監督以來，以前專門負責海外[[動畫|動畫]]、[[電影|電影]]、[[電視影集|電視影集]]的日語配音之演出。2004年播出的電視動畫《[[日式麵包王|烘焙王]]》之後，轉向參加日本國內動畫公司包含[[吉卜力工作室|吉卜力工作室]]作品<ref name ="akiba">{{Cite web|url=https://akiba-souken.com/article/26429/|script-title=ja:アニメ業界ウォッチング第19回：ガンダムからジブリ作品まで、声優の演技を裏で支える音響監督、木村絵理子インタビュー！|publisher=Akiba總研|pages=1－2|accessdate=2018-03-02|language=ja}}</ref>。
