'''木浦文化广播'''（[[朝鲜文|朝鲜文]]：목포문화방송，[[朝鲜汉字|朝鲜汉字]]：{{lang|ko|木浦文化放送}}；[[英语|英语]]：Mokpo Munhwa Broadcasting Corporation）是位于[[韩国|韩国]][[全罗南道|全罗南道]]的[[文化广播_(韩国)|文化广播]]的放送局。

== 所在地 ==
503-728[[全罗南道|全罗南道]][[木浦市|木浦市]]荣山路334

== 沿革 ==
*1968年8月17日 - 开局，开始播音（呼号'''HLAM'''）。
*1980年5月21日 - [[光州事件|光州事件]]发生，导致节目播出暂停。
*1984年5月29日 - 开始播放电视节目（CH7，图像功率500W，语音功率100W）。
*1986年7月1日 - 开始播放MBC FM4U（呼号'''HLAM-FM'''，频率为102.3MHz，1kW）。
*1987年11月6日 - 电视发射功率增加至1kW。
*1989年3月3日 - 办公楼迁至现址。
*1995年11月15日 - 开始使用实况转播车。
*1997年5月15日 - 与[[中国|中国]][[连云港电视台|连云港电视台]]结为友谊关系。
*1999年10月22日 - 开始播放MBC标准FM（89.1MHz，2kW）。
*2003年11月3日 - 开始在大芚山播放数字电视，功率2kW。
*2008年1月28日 - 开始播放[[DMB|DMB]]。
*2012年10月30日 - 模拟电视停播。
