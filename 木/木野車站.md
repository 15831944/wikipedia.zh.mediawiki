'''木野車站'''（{{jpn|j=木野駅|hg=きのえき|rm=Kino eki|f=y}}）是位於[[日本|日本]][[京都市|京都市]][[左京區|左京區]]的[[叡山電鐵|叡山電鐵]][[叡山電鐵鞍馬線|鞍馬線]]的車站。車站編號為'''E11'''。

== 車站結構 ==
擁有[[相對式月台|相對式月台]]2面2線的[[無人車站|無人車站]]。設有幾乎隆罩住整個月台的遮蓬。出入口位於[[出町柳車站|出町柳車站]]側。兩個月台之間以站內[[平交道|平交道]]（第1種甲）連結，但有許多行人和自行車將此處當作一般的道路通過。另外，此平交道對輕型、小型特殊車輛之外[[道路交通標誌|禁止通行]]（參考圖片）。平地到月台之間設有斜坡，但是坡度大且幅度窄，使用輪椅需要有人從中協助。由於本站是最靠近[[京都府立北稜高等學校|京都府立北稜高等學校]]的車站，除了停課的日子以外，早上的時間帶常配置有站務員，也設有站務員的辦公室。不過，在一般的平日，只有下行月台會配置約20分鐘的站務員。

== 車站周邊 ==
本站位於離[[京都府道106號神山岩倉停車場線|京都府道106號神山岩倉停車場線]]數十公尺處。周邊以獨立住宅為主。除了府道沿路之外也有許多農田，住宅區的開發正在進行之中。

開通當時位於西邊數百公尺處[[平交道|平交道]]的東（現在位置）側，地處街道交錯的小型聚落中心（也是[[京都巴士|京都巴士]]的終點站）的旁邊，且一併設置了車庫。1944年隨著[[單線鐵路|單線化]]而轉移到了平交道的西側。1968年[[京都精華大學|京都精華短期大學]]（當時）創校後，在1989年[[京都精華大前車站|京都精華大前車站]]設立之前，本站成了離該校最近的車站，從車站到學校的街道（現府道106號）沿路建設了許多學生公寓。另外本車站也位於1980年創校的[[京都府立北稜高等學校|京都府立北稜高等學校]]的正前方。但是由於1990年[[京都精華大前車站|京都精華大前車站]]的開通和[[複線鐵路|複線化]]，本站轉移到了離南側的新開發住宅區較近的現在地點。

* 你橋（-{あんた橋}-）

== 歷史 ==
* 1928年（昭和3年）12月1日 開業。當時為[[鞍馬電氣鐵道|鞍馬電氣鐵道]]的車站。
* 1942年（昭和17年）8月1日 由於公司合併而成了京福電氣鐵道的車站。
* 1944年（昭和19年）11月10日 包含本站、山端（現[[寶池車站|寶池]]）～[[二軒茶屋車站_(京都府)|二軒茶屋車站]]之間，為了向戰爭提供資源而[[單線鐵路|單線]]化。
* 1986年（昭和61年）4月1日 由於公司的讓渡而成了叡山電鐵的車站。
* 1990年（平成2年）9月28日 [[岩倉車站_(京都府)|岩倉車站]]～二軒茶屋車站之間再度複線化。
