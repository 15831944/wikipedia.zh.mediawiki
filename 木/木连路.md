'''木连路军民府'''是[[元朝|元朝]][[云南等处行中书省|云南行省]]所辖的一个土司府，即[[孟连土司|孟连土司]]，隶属于[[大理金齿等处宣慰司|大理金齿等处宣慰司]]<ref>{{cite book |last=谭 |first=其骧 |date= |title=《中国历史地图集 第七册》 |url= |location=[[北京|北京]] |publisher=[[中国地图出版社|中国地图出版社]] |page= |isbn=978-7-5031-1844-9 |author-link=谭其骧 }}</ref>（另有说法由[[乌撒乌蒙宣慰司|乌撒乌蒙宣慰司]]领有<ref>{{cite book |author=李治安, 薛磊  |date=2009-04-01 |title=《中国行政区划通史·元代卷》 |url= |location=[[上海|上海]] |publisher=[[复旦大学|复旦大学]]出版社 |page=210 |isbn=978-7-3090-5601-3 |author-link= }}
</ref>），治所在今[[孟连傣族拉祜族佤族自治县|孟连]][[娜允古镇|娜允古镇]]<ref>{{cite web |url=http://cpc.people.com.cn/GB/165240/165562/167249/10105422.html |title=云南娜允 中国最后一个傣族古城 |last= |first= |date=2009-09-23 |website=cpc.people.com.cn |publisher=新华网 |access-date=2017-10-30 |quote=}}</ref>。木连即“孟连”的同名异写<ref name="傣族网">{{cite web |url=http://www.daizuwang.com/Article_Show.asp?ArticleID=1527 |title=孟连土司历史简述 |last=莫 |first=汝德 |date=2009-02-17 |website=傣族网 |publisher=孟连旅游网 |access-date=2017-10-30 |quote=}}</ref>。

[[至元_(忽必烈)|至元]]二十六年（1289年）析[[孟定路|孟定路]]东南置木连路，[[傣族|傣族]]首领罕罢法为总管，[[天历_(元朝)|天历]]二年（1329年）以木连路、[[银沙罗甸|银沙罗甸]]置[[银沙罗甸宣慰司|银沙罗甸等处宣慰司]]，[[至顺|至顺]]二年（1331年）改为银沙罗甸军民府。[[至正|至正]]八年（1348年），木连路被[[麓川路|麓川路]]并吞，至正十五年（1355年）设[[平缅宣慰司|平缅宣慰司]]，[[明朝|明朝]]改为[[麓川平缅宣慰司|麓川平缅宣慰司]]。<ref name="傣族网"></ref>
