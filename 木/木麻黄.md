'''木麻黄'''（[[学名|学名]]：{{lang|la|''Casuarina equisetifolia'' L.}}）为[[木麻黄科|木麻黄科]][[木麻黄属|木麻黄属]]的植物。

== 形态 ==
常绿[[乔木|乔木]]，高可达20米；灰绿色小枝细软，颇似针叶，多节，每节有极退化的苞片状叶6－8枚；初夏开花，雌雄同株或异株；球形果实，直径10－16毫米，小[[坚果|坚果]]，上部有翅。

== 分布 ==
原产于[[澳大利亚|澳大利亚]]，分布于[[亚洲|亚洲]]东南部沿海地区、[[美洲|美洲]]热带地区、[[台湾|台湾]]以及[[中国大陆|中国大陆]]的[[广东|广东]]、[[广西|广西]]、[[福建|福建]]等地，目前已由人工引种栽培。

== 用途 ==
最佳的防風林，可植於海邊，樹高且堅硬。因生长迅速，抗风力强，是中国南方滨海[[防風林|防风固林]]的常见树种之一。可作为建筑、家具、造纸用材。树皮可提制[[栲胶|栲胶]]，也有用来制备[[染料|染料]]用来染[[渔网|渔网]]的。因为木麻黄生长迅速，而且能够耐受土壤中相对较高的盐环境，在被重金属污染的土壤中木麻黄也能生长，因此木麻黄在土壤修复方面也有重要的应用价值。蘭嶼由於引進木麻黃導致入侵原生植物棲地所造成的威脅。<ref>http://jgi.doe.gov/our-projects</ref>。

== 别名 ==
[[马毛树|马毛树]]（中国种子植物分类学）、[[短枝木麻黄|短枝木麻黄]]、[[驳骨树|驳骨树]]（广州）
