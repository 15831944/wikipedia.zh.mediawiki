'''木履寮車站'''位於[[台灣|台灣]][[嘉義縣|嘉義縣]][[竹崎鄉|竹崎鄉]]復金村，為[[行政院農業委員會林務局|林務局]][[阿里山森林鐵路|阿里山森林鐵路]][[阿里山線|阿里山線]]之[[鐵路車站|鐵路車站]]，昔日此地曾有製造木履之草寮，因此大家都叫它為木履寮，位於海拔324公尺，目前僅作為上山列車及下山列車臨時停靠站和交會站，其他加開列車暫不停靠，此地早期盛產柑橘與熱帶水果，為主要農作物。

== 車站構造 ==
*木履寮站原有一個木造站房，但已遭拆除，站內只剩下一股軌道。

== 利用狀況 ==
*阿里山線（主線）的臨時停靠站，正常不會停靠。

== 車站週邊 ==
== 歷史 ==
*1912年10月1日：設立「木履寮驛」。
