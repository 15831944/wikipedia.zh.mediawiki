'''木佬人'''（木佬语自称“嘎沃”，国际音标：：/qɑ<sup>24</sup> ɣo<sup>53</sup>/；“木佬”为当地汉语他称）是中國西南的一支[[少數民族|少數民族]]。中华人民共和国成立后，一直定为“未识别民族”，期间有些人虚填为[[苗族|苗族]]或[[布依族|布依族]]。直到1993年，贵州省人民政府才批复认定贵州的木佬人为[[仫佬族|仫佬族]]。然而实际上木佬人的本族语言[[木佬语|木佬语]]更接近贵州[[仡佬族|仡佬族]]的[[仡佬语|仡佬语]]，与其同属[[仡央语群|仡央语群]]，与广西仫佬族属于[[侗水语支|侗水语支]]的[[仫佬语|仫佬语]]差别较大。贵州的木佬人接受“仫佬族”的称呼是改字不改音，只是接受历史上与仫佬族有一个近音的民族他称的事实。历史上有过“'''木老'''”、“'''木佬'''”、“'''木僚'''”、“'''狇狫'''”、“'''狇獠'''”、“'''狇狫苗'''”等称呼。<ref>《木佬语研究》，木仕华 著，民族出版社，中国北京，2003年2月，第7—8页，ISBN 9787105052912</ref>

據歷史文獻，過去貴州省的西部如水城、大方等地也有木佬人的分佈。如今木佬人居住在貴州東南的麻江、黃平、福泉、都勻、凱裡等地，人口有3萬多。

== 語言 ==

有本民族語言[[木佬語|木佬語]]，是[[侗台語系|侗台語系]][[仡央语族|仡央语族]]的一種語言，最近瀕危語言調查顯示，一说使用人數只有百人<ref>{{cite web|title=UNESCO Interactive Atlas of the World’s Languages in Danger|url=http://www.unesco.org/culture/languages-atlas/en/atlasmap/language-id-2422.html|publisher=[[聯合國教科文組織|聯合國教科文組織]]|accessdate=2014-01-03|language=en}}</ref>，一說只有麻江境內的1名年過九十的老人會說<ref>{{cite journal|title=值得关注的中国濒危语言|journal=科学时报|year=2011年8月23日|url=http://www.wenming.cn/wmzh_pd/fw/whyc/zxdt/201108/t20110823_294183.shtml|accessdate=2014-01-03|language=zh-cn}}</ref> 。
== 生活 ==

以從事農耕與畜牧為主。解放前經濟文化落後，目前已獲得較大改善。

== 歷史 ==

木佬人分佈範圍唐朝史書記載的仡佬族範圍多有重合，是這些地方最早的開拓者。如[[黔東南苗族侗族自治州|黔東南苗族侗族自治州]]首府[[凱里市|凱里市]]，是苗語的音譯，意思就是“木佬人的田地”。
