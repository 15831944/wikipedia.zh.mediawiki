'''瑜伽'''<ref>[[中國|中國]]早在[[唐代|唐代]]已譯為「瑜伽」，近年[[臺灣|臺灣]]坊間有人生造了'''瑜-{珈}-'''這新詞，這用法，臺灣教育部《國語辭典》並未予以收錄。</ref>（{{lang-hi|'''योग'''}}，{{lang-en|'''yoga'''}}），源于[[古印度|古印度]]文化，义爲探寻「[[梵|梵]]我一如」的道理与方法，[[古印度六大哲学派别|古印度六大哲学派别]]中有{{le|瑜伽 (印度哲学)|Yoga (philosophy)|瑜伽派}}。而现代人所称的瑜伽则是主要是一系列的修身养心方法，包括调身的体位法（参考[[瑜伽体位集|瑜伽体位集]]）、调息的[[呼吸|呼吸]]法、调心的[[冥想|冥想]]法等，以達至身心的合一。

2016年12月1日，在[[埃塞俄比亚|埃塞俄比亚]]首都[[亚的斯亚贝巴|亚的斯亚贝巴]]举行的[[联合国教科文组织|联合国教科文组织]]政府间保护非物质文化遗产委员会会议通过决议，将印度瑜伽列入该组织《[[人类非物质文化遗产代表作名录|人类非物质文化遗产代表作名录]]》<ref>{{cite web |url=http://www.un.org/chinese/News/story.asp?NewsID=27189 |title=印度瑜伽被列入教科文组织《人类非物质文化遗产代表作名录》 |publisher=联合国新闻中心|accessdate=2016-12-01}}</ref>。

== 字源 ==
瑜伽（Yoga）這個詞在[[梵文|梵文]]的意思是合一、相應，源自梵文的''YUJ''，意思就是一种称为“[[轭|轭]]”的工具，用于驾驭牛马。這個詞的最初原型是[[原始印歐語|原始印歐語]]的字根 *''yeug''- ，意思就是「连结、結合、统一」<ref>《美国传统英语词典》[http://www.bartleby.com/61/roots/IE600.html yeug词条。]</ref>。具體來說，「瑜伽」這個詞在梵語可以譯為「內在真我的統一」。

“瑜伽”、“瑜珈”兩種譯法現今皆有人使用，由於是音譯，因此並無對錯之分，但中國早期已普遍採用「瑜伽」為譯名（如唐代[[玄奘|玄奘]]所譯《[[瑜伽師地論|瑜伽師地論]]》）。瑜伽在巴利文亦解作實踐的修行者。[[陳那|陳那]]大師親身體驗後再記錄成《[[瑜伽師地論|瑜伽師地論]]》，當[[玄奘|玄奘]]聽到胡僧解說，便立即前往印度學習。

== 起源 ==
{{See also|印度哲學}}

瑜伽（Yoga）是一个通过提升意识，帮助人类充分发挥潜能的体系。瑜伽姿势运用古老而易于掌握的技巧，改善人们生理、心理、情感和精神方面的能力，是一种达到身体、心灵与精神和谐统一的运动方式。古印度人更相信人可以与天合一，他们以不同的瑜伽修炼方法融入日常生活而奉行不渝：道德、忘我的动作、稳定的头脑、宗教性的责任、冥想和宇宙的自然和创造。

瑜伽已有數千年的歷史，公元前300年的著名瑜伽大圣哲[[波颠阇利|波颠阇利]]（Patañjali）將所有口耳相傳的經典，集結而成了《[[瑜伽經|瑜伽經]]》（瑜伽的重要理论经典著作），印度瑜伽因此而有所依據。瑜伽行法正式订为八支体系。波颠阇利是一个对瑜伽有巨大貢獻的圣人。《瑜伽经》，彙整了瑜伽所有的理论和知识，形成完整的理论体系和实践系统。在这部著作里，他阐述了瑜伽的定义、 瑜伽的内容 、瑜伽给身体内部带来的变化等等。在波颠阇利之前，瑜伽已经有了很长的实践期，但是没有任何人给瑜伽一个系统的解释，波颠阇利开创了一个完整的瑜伽体系。所以波颠阇利被尊为瑜伽之祖。严格来说，瑜伽是一種身心鍛鍊的統稱，好比中國講返本歸源、導引等等，瑜伽在印度也是一個身心修練的通泛名詞。有一段時期進行各種身心修練的人不管任何派別，都被尊稱為瑜伽士（Yogi，女性為Yoginī）。

古印度的宗教[[哲學|哲學]]派別林立，不過有三本著作被大多數印度人尊為經典，一是「[[奧義書|奧義書]]」，二為「[[薄伽梵歌|薄伽梵歌]]」，三「[[阿育吠陀|阿育吠陀]]」，而《[[瑜伽經|瑜伽經]]》集結各經典統合瑜伽八支。古印度[[婆羅門|婆羅門]]教提倡「梵我合一」理論，由於[[印度教|印度教]]的普及，加上另一位有名的瑜伽祖師同時也是印度教祖師[[商羯罗|商羯罗]]的影響，這三本書也被後來大多數的瑜伽士奉為經典。瑜伽術本是一種身心修持術，表面上看似與宗教無關，也可以說古印度任何宗教都採用。它的最高目的是實現人的一切可能，從[[精神|精神]]（小我）與[[自然|自然]]（[[梵|梵]]，大我，最高意識）的合一（即「梵我合一」），一直到成[[佛|佛]]成[[仙|仙]]，或者其他教派所說的最高目的，瑜伽術都是被認可的途徑之一。

印度古語有-{云}-：世上有兩種超越[[太陽|太陽]]軌道（獲得永恆）的方式，一是在瑜伽中離棄世間；二是在戰場上委棄身體。這其實與中國傳統[[價值觀|價值觀]]有所契合，例如[[道教|道教]]的“功德成神”說，與儒家的“忠烈祠”信仰。

==== 诃陀瑜伽 ====
{{main|诃陀瑜伽}}
[[File:Yoga_person.JPG|thumb]]
[[File:Vrksasana.jpg|thumb]]
'''诃陀瑜伽'''（{{lang-sa|हठयोग}} Hathayoga）台灣譯為'''哈達瑜伽'''。「诃 (哈)（{{lang-sa|ह}}）」的字意為[[太陽|太陽]]，「陀 (達)（{{lang-sa|ठ}}」為[[月亮|月亮]]<ref>{{cite web |url=http://www.sanatansociety.org/yoga_and_meditation/hatha_yoga.htm |title=Hatha Yoga |publisher=sanatansociety.org |language=en |accessdate=2011-03-15}}</ref>。以學習[[呼吸|呼吸]]與身體鍛鍊為主。[[现代瑜伽|現代的诃陀瑜伽(哈沓瑜伽)]]被統括為體位法的練習，所以所有強調體位法身體鍛鍊的派別都可以歸納到诃陀瑜伽(哈沓瑜伽)裡。

==== 勝王瑜伽 ====
{{le|王瑜伽|Rāja Yoga}}，又稱“羅阇瑜伽”、“胜王瑜伽”。勝王瑜伽以波颠阇利的《[[瑜伽經|瑜伽經]]》為主要經典，強調八步功法身心並行的鍛鍊，尤其著重於靜坐與[[冥想|冥想]]的鍛鍊，以達到[[天人合一|天人合一]]的[[三摩地|三摩地]]為目標。目前印度有些寺院團體仍以勝王瑜伽作為標榜。

==== 知识瑜伽 ====
{{le|智瑜伽|Jñāna Yoga}}，又稱“知識瑜伽”。以[[理性|理性]]的思維、[[邏輯|邏輯]]思辯以探求真理的瑜伽。用印度人的觀點來看，中國的[[老子|老子]]、[[莊子|莊子]]等可以視為智慧瑜伽修行者。

==== 至善瑜伽 ====
{{le|信瑜伽|Bhakti Yoga}}，又稱“虔信瑜伽”、“信爱瑜伽”、“奉愛瑜伽”、“至善瑜伽”。追求超越情緒、理性純善的道之愛，以冥想、唱誦、[[禮拜|禮拜]]等嚴謹的[[宗教|宗教]]生活態度以及積極行善來達成。爲笈多王朝时期印度教{{le|信爱运动|Bhakti Movement|巴克提运动}}的中心内容。

==== 行動瑜伽 ====
{{le|行瑜伽|Karma Yoga}}，又称“業瑜伽”、“羯磨瑜伽”、“行动瑜伽”，通過積極入世的一系列實際行動來達致功德圓滿。[[聖雄甘地|聖雄甘地]]即行動瑜伽奉行者。

==== 梵頌瑜伽 ====
{{le|咒语瑜伽|Mantra Yoga}}，“[[曼特罗|曼怛罗]]”（mantra）义爲咒语、梵頌、真言、神咒、秘密语，“曼怛罗瑜伽”也可译作“梵頌瑜伽”、“咒语瑜伽”。瑜伽的眾多修練方法之一，許多瑜伽派別都採用，各自的差別為宗教歌唱、[[咒語|咒語]]唸誦、秘密咒語唸誦等等。唱誦時講求心氣合一、專注、信心，長時間的唱誦時常容易引發唸者的宗教經驗，效果強烈又簡單，所以各大派別都很重視此練法。

唸誦一般採用7個「[[種子字|種子字]]」，各自代表人的[[七輪|七輪]]。

==== 密傳瑜伽 ====
{{le|密瑜伽|Tantra Yoga}}，“[[怛特羅密教|怛特羅]]”义爲密传、密教，“怛特羅瑜伽”也可译作“密瑜伽”、“密传瑜伽”。千年師徒的秘密傳承，著重於開發生命能量，超越凡人境界的修練。密宗瑜伽的特色為複雜[[曼陀羅|曼陀羅]]圖案(Yantra)、詳密的宗教儀軌、不對外公開的內容、利用男女性能量引出生命能量的修練法、變換物質的練法、利用[[屍體|屍體]]的修練法、[[太陽能|太陽能]]修練法等等。

=== 瑜伽的“八支分法” ===
瑜伽之祖[[波颠阇利|波颠阇利]]撰写的古典经典文献《瑜伽经》，详细说明瑜伽的“八支”（Eight Limbs），即八种方法：
# {{le|禁戒 (印度哲學)|Yamas|禁戒}}（Yama）:是指外在控制，宇宙的道德戒律。
# {{le|正道 (印度哲學)|Niyamas|遵行}}（Niyama）：是指内在控制，通过自律进行自我净化。
# 体位（Āsanas）：是指瑜伽姿势，也称调身。
# 调息（Prāṇāyāma）：系统的呼吸方法，呼吸的控制和能量的处理。
# 制感（Pratyāhāra）：精神从感觉和外部事物的奴役中解脱出来，是指感觉消失，控制内心，也称调心。 
# 专-{注}-（Dhāraṇa）：集中注意，一心一意。集中专-{注}-力以提升生命之气。
# [[禅那|禅那]]（Dhyāna）：即[[冥想|冥想]]。
# [[三摩地|三摩地]]（Samādhi）：由冥想而来的超意识全部集中到灵魂中，和宇宙合二为一。超越意识的境界。身体和感官静止，看似在睡眠，但头脑仍保持警惕。

== 瑜伽與宗教 ==
===印度教瑜伽派===
{{main|{{le|瑜伽 (印度哲学)|Yoga (philosophy)}}}}}
在印度哲学和印度教中，有一派以研究瑜伽爲重点的瑜伽派，爲[[古印度六大哲学派别|古印度六大哲学派别]]之一。

=== 佛教中瑜伽的意義 ===
* 佛教中所說的瑜伽，是指'''相應'''的意思，與一般人認爲的瑜伽是一種身、心、靈修練的觀念迥然不同，因為佛法認為有情的身、心、靈屬於無常、終將壞滅的[[五蘊|五蘊]]，竭盡再大的努力與時間，要想讓這些法能夠變成永恆，是不可能的，這是一種錯誤的觀念。
* 佛教中所說的具有相應意義的'''瑜伽'''，唐朝弘揚[[唯識|唯識]]的著名大師[[窺基|窺基]]論師，在其《[[成唯識論述記|成唯識論述記]]》中有說明，它包含有五種意義：與境相應、與行相應、與理相應、與果相應、與機相應。<ref>《成唯識論述記》卷2：「述曰。言瑜伽者名為相應。此有五義故不別翻。一與境相應。不違一切法自性故。二與行相應。謂定慧等行相應也。三與理相應。安非安立二諦理也。四與果相應。能得無上菩提果也。五得果既圓利生救物。赴機應感藥病相應。此言瑜伽。法相應稱。取與理相應。多說唯以禪定為相應。瑜伽之師即依士釋。師有瑜伽名瑜伽師。即有財釋。若言瑜祇即觀行者。是師之稱。以假想惠。非謂實以刀等折之。於麁色相即是所折之色相也。半半破之漸次而折。除麁至細至不可折假說極微。不同小乘體無方分而不可折。」(CBETA, T43, no. 1830, p. 272, c6-18)</ref>
* 跟'''瑜伽'''關係密切的《[[瑜伽師地論|瑜伽師地論]]》是未來成佛的[[彌勒菩薩|彌勒菩薩]]所說的法，由[[無著|無著]]菩薩紀載下來，這是一部涵蓋[[三乘|三乘]]佛法所有內涵的佛教百科全書。

=== 基督教對瑜伽的態度及看法 ===
雖然瑜伽修煉者不斷強調其運動與宗教無關，但現時有部分[[基督教|基督教]]的[[牧師|牧師]]及傳道都建議會眾不可以參與瑜伽活動<ref>{{cite web|url=http://www.chinesechristiandiscernment.net/NAM/yoga.htm |title=基督徒需要知道瑜伽術的危險 |publisher=Chinesechristiandiscernment.net |date=2005-09-05 |accessdate=2014-01-11}}</ref>，“一個人不能事奉兩個主”。這關係到三個理由：
# 在瑜伽有不少宗派都與古代東方的秘術相關。例如：瑜伽就有一個名為[[密宗瑜伽|密宗瑜伽]]的分支，會在修煉時誦唸[[真言|真言]]；
# 即使沒有誦念咒語的活動，其儀式及姿勢有不少亦與宗教信仰相關。例如：有一種把腿盤屈在頭後的招式，名為向「太陽神敬禮」，就是一種對印度神祇敬拜的儀式。此外，有不少姿勢與佛教的[[手印|手印]]類似。
# 冥想實則為通邪靈，涉及到靈界的活動。
