[[臺灣鐵路管理局|台鐵]]'''水拔溪橋'''是位於[[台灣|台灣]][[臺東縣|臺東縣]][[加鹿溪|加鹿溪]]上的一座鐵路[[橋樑|橋樑]]，連接[[臺東縣|臺東縣]][[關山鎮|關山鎮]][[關山車站|關山車站]]與[[鹿野鄉|鹿野鄉]][[瑞和車站|瑞和車站]]，1986年於舊橋原址重建新橋。

== 沿革 ==
[[臺東線|臺東線]]鐵路臺東（今稱[[台東舊站|台東舊站]]）至里壠（今[[關山車站|關山]]）間軌距762mm的輕便鐵路，由[[台東開拓會社|台東開拓會社]]興建，1919年12月16日通車，推測跨越加鹿溪的「水拔溪橋」於當時即已存在，其長度與型式不詳。1920年代初期，[[臺灣總督府|臺灣總督府]]在交通系統一元化政策下，收購[[台東開拓會社|台東開拓會社]]興建的這段鐵路，改為官營，並自1921年起續築里壠（今[[關山車站|關山]]）至璞石閣（今[[玉里車站|玉里]]）路段，銜接1919年5月17日即已完工通車的璞石閣（今[[玉里車站|玉里]]）至[[花蓮車站|花蓮]]段762mm輕便鐵路，並在1926年完成[[臺東線|臺東線]]鐵路全線通車。

1978年7月1日起，台鐵局進行東線鐵路拓寬工程，將[[臺東線|臺東線]]鐵路由原來的762mm軌距拓寬為1067mm，於1982年6月27日竣工通車<ref>〈東線鐵路拓寬完成 明天上午通車〉，《[[聯合報|聯合報]]》，1982年6月26日，第2版。</ref>（但舞鶴－三民間[[自強隧道_(花蓮縣)|自強隧道]]前後路段除外），其中水拔溪橋採就地加固橋台及[[橋墩|橋墩]]方式辦理，並將鋼梁抽換為2孔20公尺預力梁，完工後橋梁全長40公尺{{r|老橋技術}}，橋面鋪設1067mm軌距鐵軌。

[[日本時代|日本時代]]後期及[[臺灣戰後時期|戰後]]，[[臺灣鐵路管理局|台鐵局]]陸續改建或補強老舊鐵路橋梁，並自1980年代起，實施大規模舊橋重建，除了已重建完成者{{NoteTag|1=未納入重建計畫的老舊橋梁，舉例如下：
*已於日本時代後期重建者，例如配合[[台南車站|台南車站]]至[[高雄車站|高雄車站]]間雙線工程，於1931年完工通車的第二代[[二層行溪橋|二層行溪橋]]。
*已配合戰後[[美援|美援]]時期重建者，例如1953年竣工通車的第二代[[臺鐵曾文溪橋|臺鐵曾文溪橋]]。
*已列入其他鐵路擴建雙線計畫重建者，例如配合高屏鐵路擴建雙軌工程，於1987年6月17日完工通車的新[[高屏溪|高屏溪]]橋<ref>交通部：[[中華民國|中華民國]]77年《交通年鑑》，[[臺北市|臺北市]]，1989年10月10日，頁329。</ref>。}}之外，計羅列出29座橋梁，分別納入「[[五大橋梁重建工程|五大橋梁重建工程]]」及「[[鐵路沿線老舊橋梁重建工程|鐵路沿線老舊橋梁重建工程]]」兩項計畫{{r|百週年}}。水拔溪橋由於受到1982年起實施「東部治山防洪計畫」整治河川上游、興建堤防並加寬堤距的影響，導致鐵路落於行水區內，影響排洪與行車安全，於是，[[臺灣鐵路管理局|台鐵局]]配合堤防法線及最高洪水位，規劃改建水拔溪橋，並納入「鐵路沿線老舊橋梁重建工程」項下。

== 設計施工 ==
水拔溪橋採取就地重建方式施工，為了維持列車行駛，施工單位先在原舊橋下游側另建臨時軌(即施工便線)與臨時橋。此外，因水拔溪橋南側四百餘公尺尚有一座也需要重建的[[奸仔典溪橋|奸仔典溪橋]]（跨越加典溪），因此，兩橋的重建工程同步施作，並共同鋪設一條臨時軌便線。臨時軌在1985年12月16日切換通車啟用，列車改駛臨時軌之後，工程單位隨即動工拆除舊橋，改建新橋。

重建之水拔溪橋為4孔跨距22.9公尺單線箱型預力梁橋，橋長增延至91.6公尺，橋梁高程提高1.23公尺，橋墩為單圓柱懸臂式，基礎為圓形開口沉箱，於1986年11月21日完工，待鋪軌完成後，在1987年2月23日自臨時軌切換至新橋通車{{r|老橋技術}}，並使用至今。

== 近況 ==
*2000年8月受碧利斯颱風影響，水拔溪橋南堤岸路基淘空坍塌（路線里程130k+573～650）<ref>國家災害防救科技中心[https://den.ncdr.nat.gov.tw/Search/Detail?id=809&type=HillSideSummary 災害事件簿查詢展示系統] </ref>{{r|環島提昇}}。
*[[臺灣鐵路管理局|台鐵局]]初步檢查評估，水拔溪橋因梁底高程不足及線形不佳，有必要優先辦理改建{{r|環島提昇}}。
*2014年6月28日，包含水拔溪橋在內的[[臺東線|臺東線]]完成電氣化通車營運<ref>[[交通部鐵路改建工程局|交通部鐵路改建工程局]][https://www.rrb.gov.tw/04100.aspx?id=49&lan=ch 花東線鐵路瓶頸路段雙軌化暨全線電氣化計畫] </ref>。
