'''水晶宮'''（[[英语|英语]]：'''The Crystal Palace'''）是一個以[[鋼鐵|鋼鐵]]為骨架、[[玻璃|玻璃]]為主要建材的建築，是[[十九世紀|十九世紀]]的[[英國|英國]]建築奇觀之一，也是[[工業革命|工業革命]]時代的重要象徵物。它原先是[[世界博覽會|世界博覽會]]首次於1851年在[[倫敦|倫敦]]舉行時的展示館，這場世界博覽會的正式名稱為[[萬國工業博覽會|萬國工業博覽會]]，一直到1936年以前曾經吸引過無數的社會各種階級的遊客前來參觀，「水晶宮」是來自一家以諷刺文章著名的《笨拙》[[雜誌|雜誌]]因其建築通體透明寬敞明亮而給予的名稱（這個雜誌過去也曾經對一些[[歷史|歷史]]事件評論，例如[[霍亂|霍亂]][[傳染病|傳染病]]和骯髒的倫敦街道）。後來這個名稱又被一個足球會引用，及後成為[[水晶宫足球俱乐部|水晶宫足球會]]，「水晶宮」也常用來指示倫敦南區的一塊範圍。

== 建築本身 ==
[[File:Crystal_Palace_Centre_transept_&_north_tower_from_south_wing.jpg|thumb]]

對當時而言，以一個璀璨而華麗的水晶宮在[[倫敦|倫敦]][[海德公園|海德公園]]展示世界各國的產品，象徵[[英國|英國]]國力之大，不僅作為一個展覽場地，更作為博覽會的主要科技成果之一，引起世界各國不小的騷動（當年參展者共計25國）。在[[華特豪斯自然歷史博物館|華特豪斯自然歷史博物館]]的天花板，也是採用類似水晶宮的[[玻璃|玻璃]]與[[鋼鐵|鋼鐵]]架構建成。水晶宮因展現了[[園藝|園藝]]師[[約瑟夫·帕克斯頓|約瑟夫·帕克斯頓]]的天賦，這位設計者便受封為[[騎士|騎士]]。約瑟夫．帕克斯頓曾經率領花園園丁試驗以[[玻璃|玻璃]]與[[鋼鐵|鋼鐵]]建造巨大[[溫室|溫室]]的可能性，也因此見識到這些建材的強度與[[耐久力|耐久力]]，他以這項技術知識申請世界博覽會的建築計畫並且創造出驚人的結果。建築物計畫者們，曾試圖找尋強韌、耐久、形式又簡單的建築物和並希望伴隨著快速的建造進度，約瑟夫．帕克斯頓在這一點使博覽會的籌辦人如願以償，從1850年8月至隔年5月，這棟建築使用了面積九十萬平方英尺、相當於八萬四千平方[[公尺|公尺]]的玻璃（或是八個半的標準[[足球|足球]]場地），由[[伯明罕|伯明罕]]一玻璃供應商Chance Brothers提供，這家公司是當時唯一擁有可提供如此龐大需求的工廠，卻仍然必須從[[法國|法國]]聘請勞工協助，以趕上展期的建築進度。而另一個主要建材是[[鐵|鐵]]，包括鐵柱三千三百根，鐵樑兩千三百條，佔地面積七萬四千平方[[公尺|公尺]]，長 {{convert|1851|ft|m|0}}，室内净高{{convert|128|ft|m|0}}<ref name="oregon">{{cite web

《[[海德公園|海德公園]]》這首詩，曾描述過這棟壯觀的建築物：

<blockquote>
At Rotten Row around a tree,<br />
With Albert's help did Mr P.<br />
His stately pleasure dome design:<br /> 
The greatest greenhouse ever seen;<br />
A glass cathedral on the green,<br />
Beside the crystal Serpentine.
</blockquote>

而有關這個展覽場地的風貌，可參考[[日本|日本]]於2005年發行上映的[[動畫|動畫]][[電影|電影]]《蒸氣男孩》，其故事背景以[[英國|英國]]1851年[[萬國博覽會|萬國博覽會]]為主，有不少水晶宮以及週遭風景，以动画的形式呈現出來，不過其建築外貌與實際的水晶宮並不相同。

== 重建與遷址 ==
當年世界博覽會僅舉辦六個月，這棟建築本來是要被處理掉的，但是最後決議卻由議會顛覆，水晶宮因此被重新矗立在[[肯特郡|肯特郡]]的[[塞登哈姆|塞登哈姆]]，其中包括許多的修飾與場地擴大，其中最重大的改變是，將中央走廊原來的階梯改成弧形，與原本的弧形設計呈現一個交叉的外觀。兩年內，由[[維多利亞女王|維多利亞女王]]再次舉行開幕儀式，作為長期性的展出，為此倫敦特別增設兩座火車站，其中之一是至今仍然運行的[[水晶宮站|水晶宮站]]，另一座車站則是通往遊行區，今日仍可見到其[[義大利|義大利]]風格的[[鑲嵌畫|馬賽克]]屋頂。

水晶宮的建造運用了大約五千名工人，在緊迫的少許時間中勤奮工作以完成這個對當代而言巨大的工程，Catherine Marsh這位女性則非常關切他們的薪資，當她發現這些工人的窮苦狀況以及工作待遇，她對於協助爭取公平待遇以及正常薪資的事情不遺餘力，更進一步地確保建設當局能夠提供這些工人餐點，因此許多工人對這位女士景仰有加。

== 水晶宮公園 ==
[[Image:Crystal_Palace_-_Queen_Victoria_opens_the_Great_Exhibition.jpg|right]]
[[約瑟夫·帕克斯頓|約瑟夫·帕克斯頓]]身為首要的園丁，他對[[庭園|庭園]]、[[噴泉|噴泉]]、小[[瀑布|瀑布]]的設計就像他的能力一樣無庸置疑，在設計的過程他遇到最大的問題應該是水源的供應，由於他對這項設計的熱衷，使得十二萬[[加崙|加崙]]的水必須充滿位於水晶宮公園的無數小[[噴泉|噴泉]]和[[瀑布|瀑布]]，而噴泉口總計高達一萬兩千枚，其中兩個主要的噴泉口高達250英尺，相當於76[[公尺|公尺]]的高度。原先的計畫是建造[[水塔|水塔]]，卻因為所含的量過重導致裝置毀壞，諮詢[[伊桑巴德·金德姆·布魯內爾|伊桑巴德·金德姆·布魯內爾]]之後，他想出一個計畫，希望在公園的南北各設一座強力的[[水塔|水塔]]，每一個各提供龐大的水量，而這些水來自三個別位於公園北中南的蓄水池。兩年後這個大噴泉與瀑布的建設開放遊覽，當然[[女王|女王]]照例出席訪視，當日不慎因為一陣風將水吹灑，而使得女王在[[皇家馬車|皇家馬車]]中被水濺濕。

== 景點 ==
[[File:Iguanodon_Crystal_Palace.jpg|thumb]]
在眾多吸引人的景觀中，班傑明·華特豪斯·霍金斯設計的實體大小[[恐龍|恐龍]]模型，被安排放置在低湖，接近Anerley入口處。儘管依照[[解剖學|解剖學]]的觀點這些模型並不正確，這些模型仍舊被保留下來。霍金斯甚至曾舉辦一場二十二人的晚宴，就在其中一只[[恐龍|恐龍]]雕像內部。而[[維多利亞時代|維多利亞時代]]的雕像群則在一個四百萬[[英鎊|英鎊]]的計畫中維修，在2002年這個消息才正式由[[愛丁堡公爵|愛丁堡公爵]]宣布。

展示品之中，幾乎每一個[[維多利亞女王|維多利亞女王]]時代留下的都令人感到驚奇，有[[陶器|陶器]]場、[[瓷器|瓷器]]場、[[鐵|鐵]]工廠、蒸氣錘、液壓、[[香水|香水]]、[[鋼琴|鋼琴]]、房舍、潛水裝、[[火槍|火槍]]、[[氣壓計|氣壓計]]、[[紡織品|紡織品]]以及[[煙火|煙火]]等等。

== 衰落 ==
水晶宮的命運自從財務告罄難以延續而逐漸低靡，主要是由於不能從入場費得到充足的資金，一部份也是因為該處越來越難以迎合大多數的人們，當然社會上不乏具有閒情逸致的人，儘管這些人不少，卻因為他們唯一有時間參觀的日子是週日，這一天不巧也是水晶宮閉館的一天，而他們的抗議聲幾乎沒有達到預期的改變，因為主日儀式團體堅持人民不應該被鼓勵於週日在水晶宮工作，也不支持人民在主日徜徉自己的心靈，而如果人們想參觀，那麼雇主應該給予員工在工作日的休假時間，如此一來機關團體主管也不同意，因此水晶宮的財務更是每況愈下。

隨著[[第一次世界大戰|第一次世界大戰]]的爆發，水晶宮被使用成海軍營區，並且暫時更名為海軍艦隊勝利六號，隨著戰事平息，才又以[[大英帝國戰爭博物館|大英帝國戰爭博物館]]之名重新開放，後來有一段時間，這個展覽館有著每週四的煙火節目。

== 焚毀 ==
[[File:Crystal_Palace_fire_1936.jpg|thumb]]

1936年11月30日一場災難襲擊了這個美麗的宮殿，幾個小時內，[[火|火]]焰吞噬全部為了偉大的大英帝國豎立的種種成就，以及無數人對水晶宮的夢想，在毀壞的過程中，英國居民可在幾里外遙望這個烈火，比較近的地區甚至夜空也變得明亮，就像1866年火焰焚毀北支的走廊，加上沒有適當的保險可以給付重建的經費，一些輿論認為不應該再重建，因為這個帝國偉大的象徵已經老舊，失去原有的價值了。[[溫斯頓·丘吉爾|溫斯頓·丘吉爾]]從[[英國下議院|英國下議院]]打道回府途中表示：「這是一個時代的終點。」

遺留僅存的兩個水塔在[[第二次世界大戰|第二次世界大戰]]的期間被取下，理由是[[德國|德國]]人可能用來作為通往倫敦的引導標誌，北塔最後被炸毀，南邊的一個則是一磚一瓦地拆除，因為它實在太靠近周圍的建築物，英國人不希望週遭房舍受到波及。

為了保存民族的共同記憶，並緬懷英國史上壯闊的年代，水晶宮基金會於1979年成立。雖然市民對這個基金會的未來走向有很多想法與計畫，但至今卻無一實現，不過圍繞水晶宮遺址的水晶宮公園目前設立了[[水晶宮國家體育中心|水晶宮國家體育中心]]。
