'''水猴子'''，又称'''水猴'''、'''水鬼'''、'''水狮鬼'''等，是一种[[中国民间传说|中国民间传说]]中的[[神秘生物|神秘生物]]。据说它生存于[[淡水|淡水]]区域，具有[[两栖动物|两栖]]的特征；主要以水中的[[鱼|鱼]][[虾|虾]]为食，偶尔害人淹死后也会吸[[尸体|尸体]]的[[血|血]]。<ref>http://www.hepan.com/thread-853630-2-1.html 关于“水猴”</ref> 对于水猴子的真面目，有[[水獭|水獭]]一说。
传说还常将水猴子和[[河童|河童]]混为一谈。

== 定义 ==
有说法认为水猴子是一种类似于猴子的未知动物。也有说法认为水猴子是溺水死亡的人的鬼魂<ref>{{Cite web|url=http://entertainment.dbw.cn/system/2015/10/26/056898986.shtml|title=神秘事件少女被水鬼侵犯美女洗澡遭水鬼摸大腿私处|accessdate=2017年1月24日|author=|date=|publisher=|quote=在中国的民间传说中，投水自杀或者意外而死的人会变成“水鬼”，徘徊在淹死的地方等待引诱，或者是强迫他人落水而死，以此充当自己的替死鬼，然后自己就可以投胎转世，摆脱苦难。}}</ref>。两种说法的共同之处都是水猴子都会攻击杀害在水中的人。

== 传说与记录 ==
清代官员[[薛福成|薛福成]]所著的《庸盒笔记》里，有《水鬼白昼拉人》和《水鬼假冒舢板船》的故事<ref>{{Cite web|url=http://ctext.org/wiki.pl?if=gb&chapter=479771|title=《庸盒筆記卷之六》|accessdate=2017年1月24日|author=薛福成|date=|publisher=中國哲學書電子化計劃}}</ref>。

中国多地都有水猴子拖人下水的传说<ref>{{Cite journal|title=揭开水猴子的神秘面纱|author=韩长民|url=http://kns.cnki.net/KCMS/detail/detail.aspx?dbcode=CFJD&dbname=CJFDN0508&filename=XGBB200605002&v=MTkzNzVtVWIvQlBTckpiTEc0SHRmTXFvOUZab1I4ZVgxTHV4WVM3RGgxVDNxVHJXTTFGckNVUkwyZlp1UnJGQ3I=|journal=小哥白尼.野生动物画报|issue=|doi=|others=|year=2006年|volume=|page=|pmid=}}</ref>。2016年，安徽池州一名洗衣妇人在水深仅及腰的河里溺水，部分网友看过视频猜测有水猴子拖扯该妇人<ref>{{Cite news|url=http://hk.on.cc/cn/bkn/cnt/news/20161003/bkncn-20161003204536313-1003_05011_001.html|title=每日一片：水鬼抓腳？　婦人水深及腰竟遇溺|last=|first=|date=2016年10月3日|work=東網電視|accessdate=2016年10月3日}}</ref>。

== 被辟谣的目击事件 ==
* 1953年后，中国大陸江苏、安徽等省曾流传着“毛人水怪”的谣言，影响了人民生活和生产，也有发生人在夜间活动被当成毛人水怪枪击的意外事故。各级政府努力将谣言平息<ref>{{Cite news|url=http://sc.sina.com.cn/news/z/2012-08-12/111517644.html|title=揭秘60年前专挖人心扒人脸的毛人水怪(图)|last=谷|first=岳飞|date=2012年8月12日|work=扬子晚报|accessdate=}}</ref>。
* 2015年，网络上流传一段“深圳水库捕获水猴子”的视频，但专家看过视频表示动物为生病的[[马来熊|马来熊]]<ref>{{Cite news|url=http://nextplus.nextmedia.com/news/spot/20150418/185220|title=網傳深圳水庫驚現「水鬼」 實為馬來熊|last=|first=|date=2015年4月18日|work=一周plus|accessdate=2015年4月18日}}</ref>。也有标题版本是“湛江发现水鬼”<ref>{{Cite news|url=http://js.ifeng.com/city/detail_2015_04/09/3761606_0.shtml|title=南京警方辟“水鬼”谣言：是“脱发”的马来熊（图）|last=王|first=煜|date=2015年4月9日|work=现代快报网|last2=陶|first2=维洲|accessdate=2015年4月9日}}</ref>，也有版本是在吴川江心岛<ref>{{Cite news|url=http://www.ycwb.com/ePaper/ycwb/html/2015-04/09/content_681960.htm?div=-1|title=抓到水鬼?没这回事!|last=袁|first=增伟|date=2015年4月9日|work=羊城晚报|accessdate=2015年4月9日|deadurl=yes|archiveurl=https://web.archive.org/web/20150410042427/http://www.ycwb.com/ePaper/ycwb/html/2015-04/09/content_681960.htm?div=-1|archivedate=2015年4月10日}}</ref>。
* 2016年，在微信朋友圈流传一段“朋友抓到水猴子”的14秒视频，但专家看过视频后表示动物是[[树懒|树懒]]<ref>{{Cite news|url=http://news.southcn.com/py/content/2016-06/13/content_149308597.htm|title=辟谣！网传一男子抓到“水鬼” 其实只是树獭|last=|first=|date=2016年6月13日|work=中国新闻网|accessdate=}}</ref>。

== 衍生文化 ==

* '''水鬼部队'''：台湾军方的特种部队[[中華民國陸軍101兩棲偵察營|“海龙”部队]]，擅长于海上长泳，有“水鬼”称号<ref>{{Cite news|url=http://news.ltn.com.tw/news/society/paper/1040997|title=小檔案》海龍部隊剽悍 稱號「水鬼」|last=吳|first=正庭|date=2016年10月12日|work=自由时报|accessdate=2016年10月12日}}</ref>。

* '''水鬼手表'''：[[劳力士|劳力士]]手表的潜航者系列，在中国被俗称为“水鬼”<ref>{{Cite web|url=http://fashion.sohu.com/20161021/n470929123.shtml|title=正装运动皆相宜，三款最热门劳力士水鬼！|accessdate=2017年1月25日|author=OFashion迷橙|date=2016年10月21日|publisher=搜狐时尚}}</ref>。

* '''水猴子病'''：传染性软疣俗称“水瘊子”<ref>{{Cite news|url=http://szb.dlxww.com/dlwb/html/2016-06/02/content_1272042.htm|title=医院传来孩子哭喊声 原来在刮水瘊子|last=常|first=婷|date=2016年6月2日|work=大连晚报|accessdate=2016年6月2日}}</ref>，“瘊”与“猴”相似，所以“水瘊子”有时会被误用成“水猴子”<ref>{{Cite web|url=http://www.120ask.com/question/62171469.htm|title=水猴子皮肤病不能吃什么|accessdate=2017年1月24日|author=|date=|publisher=}}</ref><ref>{{Cite web|url=http://ask.39.net/question/14445850.html|title=水猴子是什么原因导致|accessdate=2017年1月24日|author=|date=|publisher=}}</ref>。

* '''水鬼植物'''：中国植物分类中，石蒜科下有[[水鬼蕉属|水鬼蕉属]]（学名：Hymenocallis）。

* '''黄河水鬼'''：黄河边有一些以打捞溺水者尸体为业的人，被称为黄河“水鬼”<ref>{{Cite web|url=http://bingodu.oeeee.com/2015/12/09/nh/141320.html|title=黄河边的职业捞尸人 常年盯着河面等浮尸|accessdate=2017年1月25日|author=耿旭娜|date=|publisher=网易新闻图片中心|deadurl=yes|archiveurl=https://web.archive.org/web/20170202083356/http://bingodu.oeeee.com/2015/12/09/nh/141320.html|archivedate=2017年2月2日|df=}}</ref>。
