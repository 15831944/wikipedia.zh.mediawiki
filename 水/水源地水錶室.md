'''水源地水錶室'''為臺灣嘉義市之歷史建築，公告於民國九十一年（2002年）8月6日{{r|文}}，是日治時期興建的嘉義水道一部分{{r|嘉}}，後來因為不符時代需求，於民國七十年（1981年）停止使用{{r|諸}}。

== 沿革 ==
在[[臺灣日治時期|臺灣日治時期]]的明治四十四年（1911年）7月，開始興建嘉義水道，於大正三年（1914年）3月落成，水錶室即是該工程的一部分，當時稱為「量水器室」{{r|諸}}。嘉義水道的取水口設在竹崎、番路、觸口三處，之後經由管線送到水源地淨（濾）水池（井），再送到配水池去，而「量水器室」即是用來測量管制給水量之處{{r|諸}}。

二次大戰後，該建築在民國四十三年（1954年）設置嘉義縣自來水廠時更名為水錶室，之後因為不敷使用而於民國七十年（1981年）停用{{r|諸}}。

== 建築 ==
水源地水錶室為鋼筋混擬土建築，佔地29平方公尺{{r|諸}}。其外觀為新古典式建築，使用渾厚的柱式配合簡化的山牆{{r|文}}。正立面有四根凸柱，大門上方有弧形漸層，與柱頭搭配收邊{{r|諸}}。
