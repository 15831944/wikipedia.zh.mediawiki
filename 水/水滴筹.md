'''水滴筹'''是中国大陆一家[[众筹|众筹]]平台，前身为'''水滴爱心筹'''，于2016年7月上线，隶属于北京纵情向前科技有限公司。有众筹需求的用户通过向水滴筹递交相关资料，经审核通过后便可开启众筹，有意愿捐赠的用户则通过[[微信支付|微信支付]]等[[移动支付|移动支付]]方式向其捐赠。<ref>{{cite web |title=公司介绍 |url=https://www.shuidichou.com/about |website=水滴筹}}</ref>

== 争议 ==
水滴筹平台曾被指出资料审核不合格，有个别筹款发起人提供了虚假的信息开启众筹，也存在着在审核期中就可以进行筹款的情况。对此水滴筹表示，审核用户的资料需要耗费的时间较长，但不少筹款发起人是因为患重病缺少医疗费而请求众筹的，如果等审核通过后再开启众筹环节，可能会错失最佳治疗时间。不过水滴筹也表示，会尝试通过与医院的直接沟通来进行求证，以加快审核速度。另一家众筹平台[[轻松筹|轻松筹]]也被指出存在相同的问题。<ref>{{cite web |title=轻松筹与水滴筹上虚构故事也能筹款 回应:将加强审核 |url=http://finance.sina.com.cn/consume/puguangtai/2018-08-01/doc-ihhacrce6983292.shtml |website=新浪财经 |publisher=澎湃新闻 |date=2018-08-01}}</ref>

== 相关新闻 ==
[[微信|微信]]朋友圈是水滴筹的筹款项目传播的主要途径之一，不过微信朋友圈曾多次出现传言称因为水滴筹筹款项目的转发次数不够，导致筹款无法被筹款发起人提现。对此水滴筹澄清是谣言，表示筹款发起人可以提现所有筹款，并不受转发次数所限制。水滴筹也呼吁用户不要造谣传谣。<ref>{{cite web |title=水滴筹官方声明:筹多少取多少 转发次数不影响提款 |url=https://gongyi.sina.com.cn/gyzx/2019-02-12/doc-ihrfqzka5117052.shtml?wm=ty10 |website=新浪公益 |date=2019-02-12}}</ref>
