'''水晶报表'''（英文：'''Crystal Reports'''），是一款[[商务智能|商务智能]]（BI）软件，主要用于设计及生成报表。

==源由==
;产品的源由

Crystal Services Inc.一開始开发了水晶报表，后来由 Seagate Software 买下，并将Crystal Services Inc.更名为 Crystal Decisions。

*2003-12 Business Objects 买下 Crystal Decisions 公司，当时的产品版本为 10；
*2007-10 SAP 买下 Business Objects 公司；
*2011-05 发布Crystal Reports 2011 (version 14)。

;被Microsoft推广使用
*1997年，由于被[[Microsoft|Microsoft]]附加在 [[Visual_Basic|Visual Basic]] 5 里而被广泛使用；
*在[[Visual_Studio|Visual Studio]] 6 更改为需另外安裝；
*Visual Studio .NET 2002 再次搭配 Crystal Reports；
*Visual Studio .NET 2003 仍搭配 Crystal Reports；
*Visual Studio 2005 Pro板以上，提供「CRYSTAL REPORTS FOR MICROSOFT VISUAL STUDIO 2005」；
*Visual Studio 2008 Pro板以上，提供「CRYSTAL REPORTS BASIC FOR MICROSOFT VISUAL STUDIO 2008」；
*Visual Studio 2010 / 2012 / 2013 改为需从 Crystal Reports 官网下载安装。

== 支持的数据源 ==
Crystal Reports可以存取的数据源包括有：
* 储存在[[数据库|資料庫]]系统內的数据，例如：[[PostgreSQL|PostgreSQL]]、[[Adaptive_Server_Enterprise|Sybase]]、[[IBM_DB2|IBM DB2]]、[[Ingres|Ingres]]、[[Microsoft_Access|Microsoft Access]]、[[Microsoft_SQL_Server|Microsoft SQL Server]]、[[MySQL|MySQL]]、[[Interbase|Interbase]]及[[Oracle_database|Oracle]]等。<ref>{{Cite web|url=https://www.evget.com/article/2008/1/22/7709.html|title=水晶报表使用经验谈－－使用sql语句直接生成dataset做为报表的数据源-控件新闻-慧都控件网|accessdate=2018-04-20|work=www.evget.com|language=zh-CN}}</ref>
* [[dBase|dBase]] dbf
* [[Btrieve|Btrieve]]
* 储存在[[試算表|試算表]]內的数据，例如：[[Microsoft_Excel|Microsoft Excel]]
* 文本文档
* [[XML|XML]]文件
