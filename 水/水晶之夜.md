'''水晶之夜'''（{{lang-de|kristallnacht}}，或譯为“'''碎玻璃之夜'''”，是指1938年11月9日至10日凌晨，[[納粹|納粹]]黨員與[[黨衛隊|黨衛隊]]襲擊[[德國|德國]]全境的[[猶太人|猶太人]]的事件。這被認為是對猶太人有組織的[[猶太人大屠殺|屠殺]]的開始。

== 起因 ==
[[File:Herschel_Grynszpan_nov_7_1938.jpg|left]]
1930年代，許多猶太裔波蘭人居住在德國。1938年10月28日，1.7萬名猶太人在無預警的情況下，在午夜被德國政府[[驅逐出境|驅逐出境]]到[[波蘭|波蘭]]。一開始[[波蘭政府|波蘭政府]]拒絕收容這批猶太人，他們只好日夜待在德國和波蘭邊境，直到德國政府說服波蘭政府同意他們入境。

1938年11月3日，遠在[[法國|法國]]的猶太裔德國青年{{link-en|赫舍·格林斯潘|Herschel Grynszpan}}接到一封來自他父母的[[明信片|明信片]]，敘述他們在被驅逐境况下的可怕經歷；他決心去政府機構要求帮助。

1938年11月7日（星期一），格林斯潘買了一支[[左輪手槍|左輪手槍]]和一盒子彈，前往德國使館求見負責官員。接待員把格林斯潘領到秘書{{link-en|恩斯特·馮·拉特|Ernst vom Rath}}的辦公室。格林斯潘拔槍射出五發子彈，其中兩發擊中馮拉特胃部。格林斯潘犯案後，在現場等候[[法國警察|法國警察]]到來，交給他們一張事先寫好的明信片：「我必須讓世界知道我的抗議。」諷刺的是，格林斯潘刺殺的馮拉特是一位[[外交官|外交官]]，當時因為同情猶太人的遭遇被認為政治上不可靠，正在接受[[蓋世太保|蓋世太保]]的調查。

馮拉特于1938年11月9日死去，死讯传到[[柏林|柏林]]，這名德國官員之死反而成為納粹德國對猶太人採取暴力行動的藉口。

==經過 ==
[[File:Prayerbooks_lie_scattered_on_the_floor_of_the_choir_loft_in_the_Zerrennerstrasse_synagogue,_destroyed_on_Kristallnacht-97574.jpg|left]]
許多猶太商店的窗戶在當晚被打破，破碎的玻璃在月光的照射下有如水晶般的發光，這就是該事件名稱的由來。

這次攻擊看起來像是民間自發的，不過事實上卻是由德國政府策劃。在這場事件中，有約1574間[[猶太教堂|猶太教堂]]（大約是全德國所有的猶太教堂）、超過7000間猶太商店、29間百貨公司等遭到縱火或損毀。

在[[奧地利|奧地利]]也有類似的事件，94間位於[[維也納|維也納]]的猶太教堂遭到破壞。

超過3萬名猶太男性遭到逮捕並被關入[[集中營|集中營]]，主要是在[[達豪|達豪]]、[[布痕瓦爾德|布痕瓦爾德]]和[[薩克森豪森|薩克森豪森]]；集中營裏的囚犯得不到人道的對待，然而，多數人在三個月內就獲釋放，但他們必須離開德國；而一些則在兩日暴亂中被毆打致死，一般估計有91人死亡。估計死於集中營的，大約有2000-2500人。

另外，有幾個非猶太的德國人錯被當作猶太人而遇难。

在這次事件以後，大約有10000個孩子被迫離開父母和家庭，多數去了[[英國|英國]]；後來只有1000人能再與父母重聚。

== 外國反應 ==
這次事件很快傳到世界各地。經過這次事件，歐洲和北美的許多偏納粹團體的名聲遭到敗壞。[[美國|美國]]召回了大使（不過並沒有斷交），不過許多其他國家的政府選擇與德國斷交以示抗議。

這次事件並沒有停止德國對猶太人的迫害。這個事件創造了一個新名詞「[[反猶太主義|反猶太主義]]」。雖然在當時只有極少數人知道，不過水晶之夜是後來對全歐洲猶太人有組織迫害和集體屠殺的第一步。

== 現代回應 ==

許多年後，一些團體抗議將[[柏林圍牆倒塌|柏林圍牆倒塌]]的11月9日定為德國[[國定假日|國定假日]]，因為這一天同時是水晶之夜和[[啤酒館政變|啤酒館政變]]的日子；後來德國統一紀念日定在東德加入西德的10月3日。
