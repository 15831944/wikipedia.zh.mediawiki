'''水上車站'''位於[[台灣|台灣]][[嘉義縣|嘉義縣]][[水上鄉|水上鄉]]，鄰近[[台1線|台1線]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線南段]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[岸式月台|岸式月台]]兩座。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#簡易站|簡易站]]，僅有[[臺鐵區間車|區間車]]停靠。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
* 本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]、[[Icash|icash2.0]]以及[[有錢卡|HappyCash]]進出站。

== 歷史 ==
*原名'''水堀頭乘降場'''。
*1902年4月20日：改為「停車場」。
*1911年3月20日：配合南靖糖廠興建，搬移至現今[[南靖車站|南靖車站]]位置。
*1949年8月20日：原址重建，稱'''水頭簡易站'''；同年10月10日改稱'''水上'''。
*1986年5月1日：由三等站降為簡易站，並指定由[[嘉義車站|嘉義站]]管理。
*2013年9月30日：啟用多卡通刷卡機。
*2018年6月：站內缺乏無障礙設施及電梯，台鐵為改善此一情況編列預算設置相關設施，但已多次[[流標|流標]]，屢被投訴為最不友善的鐵路車站<ref>鄭榮文，[https://news.ftv.com.tw/news/detail/2018605U09M1 最不友善車站！嘉義水上車站挨批階梯高又陡]，民視新聞，2018-06-05</ref>。
