'''水母'''（''Jellyfish''，又名'''白鮓'''、'''{{Ruby|䖳|-{zh-cn:zhà;zh-tw:ㄓㄚˋ;zh-hk:caa3}-}}'''<ref>{{cite web |url = http://ctext.org/dictionary.pl?if=en&char=%E4%96%B3 | title = 中國哲學書電子化計劃·䖳 |  work = [[廣韻|廣韻]]  | accessdate = 2018-08-20 }}</ref>）是[[無脊椎動物|無脊椎動物]]，屬於[[刺胞動物門|刺胞動物門]]中的一員，其中包括水母、[[海葵|海葵]]、[[珊瑚|珊瑚]]和[[水螅|水螅]]。全世界的海洋中有超過兩百種的水母，牠們分布於全球各地的水域裡，無論是熱帶的水域﹑溫帶的水域﹑淺水區﹑約百米深的[[海洋|海洋]]，甚至是淡水區都有牠們的蹤影。水母早於六億五千萬年前就已經存在。水母的形狀大小各不相同，最大的水母其[[觸手|觸手]]可以延伸約十米遠。 在分類上有些屬於[[水螅綱|水螅綱]]，有些屬於[[缽水母綱|缽水母綱]]，其生活史中，幾乎所有種類都有兩型，即'''水螅型'''和'''水母型'''，並有兩型在[[有性生殖|有性生殖]]與[[無性生殖|無性生殖]]之間的世代交現象，而人們常見的水母則是有性的水母型。

== 壽命 ==
水母壽命很短，平均只有數個月的生命。但有一種[[灯塔水母|灯塔水母]]（學名：{{lang|la|''Turritopsis nutricula''}}）能通過反覆通常[[生殖|生殖]]和轉分化達到「不老不死」。

== 構造 ==

大部分水母都有三個主要部位：圓傘狀或是鐘狀的身體，觸器和口腕。 水母鐘狀身體下面有一些特殊的肌肉能擴張然後迅速收縮，把身體內的水排出體外，通過噴水推進的方法，水母便能向相反的方向游動。一些水母有一層能夠收縮鐘狀體的皮層，使水母能夠快速移動。水母並不擅長游泳，牠們常常要借助風、浪和水流來移動。

水母體內百分之九十五是[[水|水]]，百分之三是[[鹽|鹽]]，百分之二是[[蛋白質|蛋白質]]。水母的[[傘狀體|傘狀體]]內有一種特別的腺，可以發出[[一氧化碳|一氧化碳]]，使傘狀體膨脹。牠們沒有[[心臟|心臟]]﹑[[血液|血液]]﹑[[鰓|鰓]]和[[腦|腦]]。牠們簡單的感應器官，使牠們能分辨氣味、味道並幫助牠們在水裏保持平衡。鐘狀體的邊緣有一排圓形的小囊，當水母向一方過度傾斜的時候，這些囊就會刺激神經末梢來收縮[[肌肉|肌肉]]，並把水母轉到正確的方向上去。位於鐘狀體邊緣的光感器官能使牠們分辨光亮與黑暗。 通過化學感受器水母可感覺到氣味和味道。通過觸手和口腕上的感受器，水母還可以感到物體的運動，幫助牠們尋找食物。有一種水母，牠們有一個或更多的口溝。裏面有口腕，功能就像嘴巴。牠們的觸手裏面有一個小小的器官，也就是水母們的「耳朵」，能夠預報[[海洋風暴|海洋風暴]]的來臨，當[[海浪|海浪]]和[[空氣|空氣]]磨擦而產生的[[次聲波|次聲波]]衝擊它，便會刺激周圍的[[神經感受器|神經感受器]]。所以，在風暴來臨之前的十幾個小時，牠們就會接到消息從海面一下子全部消失了。水母沒有[[肺|肺]]或鰓，他們通過外層的組織來呼吸，整個外層的組織都可以和海水交換氧氣和二氧化碳。鐘狀體上有閃光的白色線條，稱為[[徑向水管|徑向水管]]，用來輸送養分至水母的身體各部份。水母的[[觸手|觸手]]和身體上都佈滿[[刺絲囊|刺絲囊]]。這些絲囊可以在幾毫秒內迅速螫傷、捕捉或征服獵物。有些水母是無毒。

== 形態 ==
[[File:Jellyfish_in_Sumida_Aquarium2_20180215.jpg|thumb]]
不同的水母有不同的形態，地球上最大的水母是「[[獅鬃水母|獅鬃水母]]」，直徑有2[[米|公尺]]，[[觸手|觸手]]長35米，最小的[[伊魯康吉水母|伊魯康吉水母]]全長則只有12[[毫米|毫米]]。[[帶水母|帶水母]]的周圍和中間部分，分佈著幾條平行的光帶，當牠游動的時候，光帶隨波搖曳，非常優美。水母大多因為獨特的外表被人冠以特別的名字。

有些水母不單顏色多變，而且還會在水中發光，有些閃耀著微弱的淡綠色或藍紫色光芒，有些還帶有彩虹般的光暈，當牠們在海中遊動時，就變成了一個光彩奪目的彩球，光影隨波搖曳，非常優美。水母發光靠的是一種叫[[埃奎明|埃奎明]]的奇妙[[蛋白質|蛋白質]]，這種蛋白質和[[鈣|鈣]][[離子|離子]]相混合的時候，就會發出強藍光來。[[埃奎明|埃奎明]]的量在水母體內越多，發的光就越強，每隻水母平均只含有五十[[微克|微克]]的這種物質。

== 繁殖 ==

[[雌雄異體|雌雄異體]]，有[[生殖腺|生殖腺]]在近[[胃囊|胃囊]]處。成熟的[[精子|精子]]流入雌彥萁體內受精。[[受精卵|受精卵]]發育成幼蟲離開母體，在水裏游動一會兒後，沉下海底形成[[幼體|幼體]]，後變成[[橫裂體|橫裂體]]，橫裂體分裂成多個碟狀幼體，再發育成成體。

近年，世界各地的海域常常有突如其來的水母群的出現。早前，[[日本|日本]]海域便出現兩米長的巨型「[[越前水母|越前水母]]」群，估計[[中國|中國]][[長江|長江]]流域是越前水母的來源之一。隨著中國沿海業的過度捕撈，減少了與水母爭食的魚類，增加了[[浮游生物|浮游生物]]，造成水母繁衍過剩，加上[[長江|長江]]流域連場反常的大雨，將巨型水母沖到日本海域。此外，海洋的水温變暖亦加速水母的繁殖和生長。

== 獵物與獵者 ==
[[File:Jellyfish_in_Sumida_Aquarium4_20180215.jpg|thumb]]
所有水母都是肉食性動物，牠們以[[魚類|魚類]]和[[浮游生物|浮游生物]]為食。當獵食的時候，水母很被動，只捕食游到牠們身邊的動物。牠們用觸手上的刺絲囊來螫傷或是殺死獵物，然後把食物送到嘴和消化腔裏。大部份水母幾乎是透明的，令敵人難於發現。有些水母能夠發光，單憑身上發出的幽光，水母不費吹灰之力就能吸引獵物。

儘管水母有刺絲囊的保護，但[[海龜|海龜]]和[[翻車魚|翻車魚]]還是以牠們為食，而水母亦是許多人的美食。若海龜數目減少，水母的[[繁殖率|繁殖率]]便會激增，水母過度繁殖，自然又會影響到其他魚類及浮游生物的繁衍，令生態系統失衡。

在[[馬來西亞|馬來西亞]]至[[澳大利亞|澳大利亞]]的海面上，有兩種分別叫做[[澳洲箱形水母|澳洲箱形水母]]和[[曳手水母|曳手水母]]，其分泌的毒性很強，如果被它們刺到，在幾分鐘之內就會呼吸困難而死亡，因此它們又被稱為殺手水母，被這樣的水母螫傷的人可以在短時間之內喪命。但大多數情況下，螫傷只會引起極端'''[[疼痛|疼痛]]'''、'''[[惡心|惡心]]'''、'''[[紅疹|紅疹]]'''和'''[[鞭痕|鞭痕]]'''，有時會持續數周。被水母螫後，發生[[呼吸困難|呼吸困難]]症狀時，應立即實施[[人工呼吸|人工呼吸]]，或注射[[強心劑|強心劑]]，千萬不可大意，並應迅速尋求醫護救助。

== 天敵 ==

水母的天敵除了[[人類|人類]]之外，目前已知大自然中會捕食水母的生物有：

* [[鯊魚|鯊魚]]：[[鯊魚|鯊魚]]偶爾會[[捕食|捕食]]水母，但大部分的時候不會。
* [[玳瑁|玳瑁]]：目前已知[[玳瑁|玳瑁]]除了[[捕食|捕食]]小[[魚|魚]]為生，另一[[食物|食物]]來源為水母。
* [[海龜|海龜]]：捕食水母為生，海龜除了[[眼睛|眼睛]]外，身體其他部分都可以抵抗水母的毒性，牠們在捕食水母時會閉上眼睛。不过因為目前[[海|海]]中[[垃圾|垃圾]]很多，經常發生[[海龜|海龜]]將漂浮的[[塑膠袋|塑膠袋]]當作水母吸食而[[窒息|窒息]][[死亡|死亡]]。
* [[曼波魚|曼波魚]]：[[捕食|捕食]]水母為生。
* [[紫螺|紫螺]]：分布在熱帶太平洋溫暖的水域，愛吃漂浮在水面的水母。
