《'''水手战士V'''》是[[日本漫画家|日本漫画家]][[武内直子|武内直子]]所繪的[[日本漫画|日本漫画]]作品，为《[[美少女战士|美少女战士]]》的[[前传|前传]]。讲述最初觉醒的水手战士·[[爱野美奈子|爱野美奈子]]与敌人作战的故事。

该漫画于[[讲谈社|讲谈社]]发行的1993年5月号的[[漫画杂志|漫画杂志]]《るんるん》连载。[[单行本|单行本]]3卷完结，新装版则合编为2卷。

== 概要 ==
本作以水手金星（爱野美奈子）为主角，既是《美少女战士》的前传也是它的母体作品。当时武内直子是受到了当时所热播的[[电视剧|电视剧]]《[[美少女假面|美少女假面]]》而深受影响才得到了创作该作的灵感。

在一开始，本作就受到动画制作公司的青睐，预定把忠实于漫画原作的内容制作为全4卷[[OVA|OVA]]来发售，后来因为日本的一家造笔公司“[[水手万年笔|水手万年笔]]”早已注册了与该作同名的商标，所以原订OVA化的制作被紧急终止了。之后2003年随着《美少女战士》电视剧化之际，把本作也单独制作成了[[DVD|DVD]]音像作品《[[美少女战士真人版|美少女战士真人版]]》的前篇Act.Zero。

漫画中均有《美少女战士》中所出现的人物[[月野兔|月野兔]]、[[火野丽|火野丽]]、古幡元基的客串。

在《美少女战士》第21集“守护孩子们的梦想！因为动画被连结起来的友情”中提及到《水手战士V》的TV动画放送的预定，后因为动画制作人的不和而终止制作的事。第42集“水手金星的过去，美奈子的凄美恋爱”中的主要剧情讲述原作完结之后的事，美奈子在击败黑暗代理人之后，以水手战士V的名义去到[[伦敦|伦敦]]打击黑暗王国的妖魔，之后认识了国际女刑警与男友阿兰，后因恋爱失败而回到了[[东京|东京]]。

台湾中文版由于2003年[[大然文化|大然文化]]投资失利的影响，导致讲谈社向大然文化发出停止授权出版漫画的紧急通告，所以台湾地区只出版了前两卷的单行本而终止了第三卷的发行。

== 劇情簡介 ==
活泼开朗的中学一年级美少女·爱野美奈子成绩不是相当的理想，但体育运动确实非常拿手的。有一天，她与一只会说话的白猫·阿尔特米斯相遇了，便得到了不可思议的粉盒与一支神奇的变身笔，化身成为了正义的水手服美少女战士·外号是水手战士V，背负着沉重的使命与邪恶的敌人进行着非同寻常的较量，并努力寻得整个事件的真相，最终她前世的记忆被唤醒了……

== 登场角色 ==
=== 正义一方 ===
; 爱野美奈子／水手战士V、水手金星
: 本作的主人公。实际上在水手战士当中是第一个觉醒的战士。至始至今都担任着众水手战士的领导者。
; 阿尔特米斯
: 一只白色的小公猫，是美奈子的搭档，拥有会说人话的能力。
; 首领（变身笔）
: 其实是变身笔在危机时刻所发出的神秘声音，向美奈子与阿尔特米斯通信、发起指示或提供情报的声音。真实身份不明。

=== 美奈子的朋友 ===
; 空野晶晶
: 美奈子的同班同学，也是好朋友。相貌与[[水野亚美|水野亚美]]相似。
; 天野久和
: 美奈子的同班同学，是个电子游戏狂。相貌与海野相似。
; 古幡元基
: 在游戏厅中打工的大哥哥。

=== 刑警 ===
; 樱田夏菜
: 警察厅的总监。若木的上司。是水手战士V的铁杆粉丝。是《美少女战士》中月野兔的班主任—樱田春菜的妹妹。
; 若木敏男
: 樱田夏菜手下的特殊警官。因水手战士V每次都抢走了他的功劳而怨恨水手战士V。

=== 黑暗代理人 ===
黑暗代理人是《美少女战士》里出现的邪恶组织—黑暗王国在日本东京的一家演艺公司，主要目的是派出偶像对粉丝们进行洗脑，收集他们的能量，使他们成为黑暗代理人的奴隶，好统治整个东京。
; 最上A／达恩怀特
: 在美奈子眼前以“怪盗A”的角色名义所出现的偶像。其实真实身份是黑暗王国四天王的直属部下，是黑暗代理人的幕后黑手。
