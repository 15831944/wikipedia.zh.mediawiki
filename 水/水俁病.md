'''水俁病'''（{{lang-ja|水俣病}}），為[[公害病|公害病]]的一種，成因為[[汞中毒|汞中毒]]。1956年左右於[[熊本縣|熊本縣]][[水俁市|水俁市]]附近發生，經確認後依地得名。不久，於[[新潟縣|新潟縣]]發現的新公害病亦稱為水俁病。其區別為：前者稱'''熊本水俁病'''；後者則為'''[[第二水俁病|第二水俁病]]'''或'''新潟水俁病'''。

以上兩種水俁病，與[[痛痛病|痛痛病]]、[[四日市哮喘|四日市哮喘]]並列為[[日本四大公害病|日本四大公害病]]。

== 病徵 ==

'''水俣病'''實際為有機水銀中毒，分為急性、亞急性、慢性、潛在性和胎兒性等多种类型。患者手足麻痺，甚至步行困難、運動障礙、失智、聽力及言語障礙；重者例如痙攣、神經錯亂，最後死亡，至今仍無有效的治療法。

發病起三個月內約有半數重症者死亡，孕妇和胎儿亦会受到影响。

== 原因 ==
1932年，{{link-en|窒素株式會社|Chisso}}（窒素即[[氮|氮]]，該公司於2012年改名為「新日本窒素肥料」）於水俁工場生產[[氯乙烯|氯乙烯]]與[[醋酸乙烯|醋酸乙烯]]，其製程中需要使用含[[汞|汞]]的催化劑。由於該工廠任意排放廢水，這些含汞的劇毒物質流入成海，被水中生物所食用，並轉成[[甲基氯汞|甲基氯汞]]（[[化學式|化學式]]CH<sub>3</sub>HgCl）与[[二甲汞|二甲汞]]（[[化學式|化學式]](CH<sub>3</sub>)<sub>2</sub>Hg）等[[有機汞化合物|有機汞化合物]]。

當人類捕食海中生物後，甲基汞等有機汞化合物通過魚蝦進入人體，被腸胃吸收，侵害腦部和身體其他部分，造成[[生物累積|生物累積]]。

== 發現經過 ==
[[File:Minamata_map_illustrating_Chisso_factory_effluent_routes_jp.png|thumb]]

1950年，有大量的海魚成群在水俣湾海面游泳，任人網捕，海面上常見死魚、[[海鳥|海鳥]]屍體，水俁市的漁獲量開始銳減。1952年，水俁當地許多[[貓|貓]]隻出現不尋常現象，走路顛顛跌跌，甚至發足狂奔，當地居民稱「貓舞蹈病」，或直接稱為「怪病」。1953年1月有貓發瘋跳海自殺，但當時尚未引起注意，一年內，投海自殺的貓總數達五萬多隻。接著，狗、豬也發生了類似的發瘋情形。1956年4月21日，人類亦被確認發生同樣的症例，來自入江村的小女孩田中靜子成為第一位患病者，被送至窒素公司（{{lang|ja|チッソ株式会社}}，「窒素」是指[[氮|氮]]）附屬醫院，病況急速惡化，一個月後雙眼失明，全身性痙攣，不久死亡。死者二歲的妹妹也罹患相同的病症，不久又發現許多村民都有問題。這些人開始只是口齒不清，走路不穩，最後高聲大叫而死。醫師細川一認為事態嚴重，向官方提出正式報告。當中的患者多為漁民家庭出身，1956年8月日本學者發現水俁灣海水中有污染物質，研究人員偵察的矛頭指向窒素公司。這種怪病被稱為「水俁奇病」。此事造成水俁近海魚貝類市場價值一落千丈，水俁居民由於陷入貧困，反而大量食用有毒的魚貝，使得災情擴大。鎮有4萬居民，先後有1萬人不同程度的患有此種病狀。1957年8月水俁病患者家庭互助會成立。

1959年，[[熊本大學|熊本大學]]醫學部水俁病研究班發表研究報告，指出水俁病原因為當地窒素工場所排出的有機水銀，1932至1968年間有數百噸的汞被排入水俁灣。1959年底，漁民開始向窒素公司進行示威抗議。1960年正式將「甲基汞中毒」所引起的工業公害病，定名為「水俁病」。然而，新日本窒素肥料立即否認此說，窒素公司認為：它只用金屬汞，不用甲基汞，因此，不可能是水俁病的來源。工廠對事件處理相當消極，不但沒有停止排放污水，還企圖掩蓋真相，阻撓相關的調查研究，甚至買通打手，以暴力嚇阻。美國攝影師[[尤金·史密斯|尤金·史密斯]]即被日本窒素公司所雇的暴民殴打，以至一目失明。日本政府於1968年左右乃確認兩者之間的關係，但這樣的遲誤已造成災害擴大。

1966年新潟又爆發水俁病，史稱「[[第二水俁病|第二水俁病]]」，這次的禍首[[昭和電工|昭和電工]]仍企圖逃避責任，但新潟市民對昭和公司不負責任的態度極為不滿，展開激烈的示威抗爭。1967年新潟市民正式向法院提出控訴。纏訟數年之後，1971年法院終於做成判決，昭和公司敗訴，要負賠償責任，新潟的受害者又主動與水俁市的苦主聯手向法院控告窒素公司。1973年，法院判決窒素公司必須立即付出相當於3200萬[[美元|美元]]的賠償金，被認確為水俁病的患者，可從政府及新日本窒素肥料取得相關醫療費用。兩年內窒素公司一共賠出了八仟萬美元。由於病情認定是由政府處理，也產生是否涉入政治的疑問及批判。

该事件被认为是一起重大的[[工业灾难|工业灾难]]。1997年10月，由官方所認定的受害者高達12,615人，當中有1,246人已死亡。

== 水俁病發生區域 ==
*1956年日本熊本縣水俁市周邊（[[八代海|八代海]]沿岸）。
*1964年日本新潟縣阿賀野川流域以外 - 河流上流的[[乙醛|乙醛]]工場排出未經處理的含有有機汞化物的廢水引起。{{Fact|time=2007-07-16T11:04:11Z}}
*1980年代[[中華人民共和國|中國]][[黑龍江省|黑龍江省]][[松花江|松花江]] - [[中石油|中石油]]吉林石化公司污染了河流，導致大範圍的漁民水銀中毒。<ref>{{lang|ja|原田正純。『水俣病と世界の水銀污染』。実教出版. 1995}}</ref>
*1990年代[[南美洲|南美洲]][[亞馬遜河|亞馬遜河]] - 當地金礦山開採時所流出的有機水銀所致。{{Fact|time=2007-07-16T11:04:11Z}}
