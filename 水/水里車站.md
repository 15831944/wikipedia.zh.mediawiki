'''水里車站'''位於[[台灣|台灣]][[南投縣|南投縣]][[水里鄉|水里鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[集集線|集集線]]的[[鐵路車站|鐵路車站]]，不僅是水里鄉民的對外門戶，也是全線最大的車站。

== 車站構造 ==
*[[島式月台|島式月台]]一座。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#簡易站|簡易站]]。
*行駛車輛為DRC1000型[[柴油|柴油]]客車，等級為[[臺鐵區間車|區間車]]。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
[[File:Shuili_station.jpg|thumb]]
*1922年1月14日：設立'''水-{裡}-坑驛'''。
*1952年12月1日：改為'''水-{裡}-'''。
*1959年8月8日：改為'''水-{裏}-'''。
*1967年10月10日：與[[成功車站|成功]]、[[豐富車站|豐富]]、[[雙溪車站|雙溪]]等車站同時改稱今名。
*1989年11月1日：由三等站降為簡易站，並指定由[[濁水車站|濁水站]]管理。
*1989年12月10日：管理站由濁水站改為[[二水車站|二水站]]，同時本站在濁水站於2008年5月15日升等以前也是集集線唯一一個由[[臺灣鐵路管理局|台鐵]]派有站員的簡易站。
*2015年6月30日：啟用多卡通刷卡機。
