'''水密隔艙'''（Watertight compartments），亦作'''水密艙室'''或'''防水艙'''，是{{tsl|en|Cabin (ship)|船艙}}的安全結構設計<ref>{{cite web |url=http://www.globalsecurity.org/military/library/policy/navy/nrtc/14057_ppr_ch3.pdf |title=SHIP COMPARTMENTATION AND WATERTIGHT INTEGRITY |last= |first= |website= |publisher=United States Navy}}</ref>，其位於[[船體|船體]]內，是船身內部經[[水密艙壁|水密艙壁]]所區隔劃出的多間獨立{{tsl|en|Compartment (ship)|艙室}}。最早見於[[梁_(南朝)|梁 (南朝)]]《宋書》關於「八艚艦」的記載<ref>{{Cite web |url = https://zh.wikisource.org/wiki/%E5%AE%8B%E6%9B%B8/%E5%8D%B71 |title = 《宋書》 |author = 梁（南朝）沈約 |date = |publisher = |language = |accessdate = |quote = 卷一、本紀第一。武帝上：……別有八艚艦九枚，起四層，高十二丈……}}</ref>。

== 用途 ==
當船舶遭遇意外使船艙少部份破損進水時，其他尚未受波及的水密隔艙則還能提供船舶[[浮力|浮力]]，減緩立即下沉的風險。另外，水密隔艙也可規劃成多用途的空間，例如[[郵輪|郵輪]]中的：住客艙、貨物艙、機房⋯⋯等。
