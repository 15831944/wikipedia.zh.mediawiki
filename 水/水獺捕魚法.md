'''水獺捕魚法'''是一種馴化[[水獺|水獺]]來捕魚的傳統技能，有上千年的歷史，在[[亞洲|亞洲]]、[[美洲|美洲]]、[[歐洲|歐洲]]和[[非洲|非洲]]等地皆有紀錄。漁民會训练水獺將魚蝦趕入網中，或直接讓水獺帮助捕魚。[[圭亞那|圭亞那]]的漁民则採用另一種方法：以水獺作为标志来判斷魚的位置。早期，水獺捕魚法十分普遍，但如今水獺捕魚法幾乎失傳，只剩[[中國大陸|中國大陸]]和[[孟加拉|孟加拉]]等少數地區尚存此一傳統。此法的失传或将加快水獺的滅絕速度。

== 歷史 ==

[[File:OlausMagnusFishingOtterCartaMarina1539.png|thumb]]

水獺捕魚法在千年以前的[[唐朝|唐朝]]就有记载，當時中國漁民便會馴化[[水獺|水獺]]來捕魚，據說是從[[東南亞|東南亞]]的漁民那邊学得<ref name="Simoons1990"/><ref name="GabrielEtAl2008"/>。此一捕魚法也在[[马可·波罗|馬可波羅]]的遊記中提及<ref name="經典雜誌 (2018-08)"/>。亞洲其他地區，如[[南亞|南亞]]（[[孟加拉|孟加拉]]和[[印度|印度]]）及[[東南亞|東南亞]]地區漁民亦在百年前就懂得這招<ref name="Simoons1990"/><ref name="GabrielEtAl2008"/><ref name="大紀元"/><ref name="昔日東方"/><ref name="ETtoday新聞雲"/>。除了亞洲，[[中美洲|中美洲]]和[[南美洲|南美洲]]、[[中歐|中歐]]（[[德國|德國]]）和[[東歐|東歐]]（[[波蘭|波蘭]]）、[[北非|北非]]、[[不列顛群島|不列顛群島]]（[[英格蘭|英格蘭]]和[[蘇格蘭|蘇格蘭]]）及[[斯堪地那維亞|斯堪地那維亞]]也有水獺捕魚的紀錄<ref name="Simoons1990"/><ref name="GabrielEtAl2008"/>。

== 作法 ==

此一傳統技能是由父親傳給兒子，一代傳一代<ref name="IUCN"/><ref name="Geographical"/>。早期漁民馴養來捕魚的水獺主要是[[歐亞水獺|歐亞水獺]]（如歐洲和北非地區）和[[江獺|江獺]]（如亞洲地區）<ref name="GabrielEtAl2008"/>，但目前只剩江獺<ref name="IUCN"/>。採用水獺捕魚法的漁民會捕捉小水獺，從小訓練<ref name="湖南省志"/>。也有漁民選擇訓練成年水獺，因為成年水獺較好訓練<ref name="經典雜誌 (2018-08)"/>。捕魚時間通常是在晚上，漁民會把漁網從船上撒到靠近河岸的水下，並將一條繩子拴在水獺脖子上，然後讓牠下水<ref name="大紀元"/><ref name="ETtoday新聞雲"/>。水獺會從植物叢、石頭底或縫隙處將魚、蝦趕進漁網中，漁夫再用漁網撈捕<ref name="經典雜誌 (2018-08)"/><ref name="大紀元"/><ref name="ETtoday新聞雲"/><ref name="湖南省志"/>。另一種方法則是漁民先讓水獺餓一陣子，接著放牠下水。因水獺不會在水下進食，浮上水面後漁夫便能從牠口中取走魚<ref name="湖南省志"/>。[[圭亞那|圭亞那]]的漁民採用的是另一種方法：觀察水獺的下潛去向，确认捕魚的位置，然後展开捕魚行动<ref name="GabrielEtAl2008"/>。有文獻紀載，在[[不列顛群島|不列顛群島]]，漁民會給水獺戴上口套，避免水獺把鱼吃掉<ref name="GabrielEtAl2008"/>。表現良好時，漁夫通常會給予水獺食物作為獎勵<ref name="Simoons1990"/><ref name="經典雜誌 (2018-08)"/>。

== 現狀 ==

早期，水獺捕魚法十分普遍，有的漁民甚至馴養了十多隻的水獺<ref name="經典雜誌 (2018-08)"/><ref name="大紀元"/><ref name="IUCN"/><ref name="Geographical"/><ref name="湖南省志"/>。但如今，水獺捕魚法幾乎失傳，只剩[[中國大陸|中國大陸]]（如[[四川省|四川省]][[瀘州市|瀘州市]]的[[合江县|合江縣]]）和[[孟加拉|孟加拉]]（如[[諾拉爾縣|諾拉爾縣]]）等少數地區尚存此一傳統<ref name="經典雜誌 (2018-08)"/><ref name="大紀元"/><ref name="昔日東方"/><ref name="ETtoday新聞雲"/>。水獺捕魚法難以延續的原因包含漁獲量因環境污染而大幅減少，以及水獺的瀕危<ref name="經典雜誌 (2018-08)"/><ref name="大紀元"/><ref name="ETtoday新聞雲"/>。野生動物專家也表示，漁民飼養水獺捕魚有利維持水獺的數量，且馴養過的水獺仍可在野外存活；若此一傳統消失，水獺會滅絕得更快<ref name="大紀元"/><ref name="ETtoday新聞雲"/><ref name="IUCN"/>。但據[[中華人民共和國|中華人民共和國]][[中華人民共和國農業部|農業部]]訂定的{{单双书号转换|[[长江|长江]]渔业资源管理规定}}第六條，在長江用水獺捕魚是違法行為<ref name="法律"/>。
