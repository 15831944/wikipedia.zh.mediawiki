'''水曆'''是[[水族|水族]]特有的[[历法|历法]]。现行水曆与[[夏历|夏历]]大体一致，主要区别在于水曆建[[戌|戌]]，以夏历九月为岁首。

== 物候曆 ==
水曆起初是一种物候曆（自然曆）。水族先民通过观察气象变化、候鸟迁栖等自然现象来预测寒暑交替、指导农业生产。最早的水历将一年分为“盛”与“挪”两个季节，其中“盛”泛指热天，大约为[[仲春|仲春]]至[[孟秋|孟秋]]，“挪”泛指冷天，大约为[[仲秋|仲秋]]至[[孟春|孟春]]。<ref>{{cite book|title=《中国水书探析》|author=韦章炳|pages=372-373|publisher=中国文史出版社|year=2007}}</ref>

== 古水曆==
古水曆盛行于元代，至明清时仍有使用。古水曆是一种[[阳历|阳历]]，每年360天，分为12个月、每月30天，以[[地支|地支]]纪年、月、日。据说水曆最初以[[秋分|秋分]]为岁首，但可能是由于[[岁差|岁差]]的原因，岁首后来推迟至夏历九月（建戌）。古水曆不设闰年、闰月或闰日，以星象调整360天与[[回归年|回归年]]长度（约365.2422日）之间五天多的差异，并以[[亥|亥]]日为元旦，称为“[[端节|端节]]”。<ref name=huang>{{cite book|title=《中国民族古文字与文献研究论文集》|editor=黄健民|publisher=中央民族大学出版社|year=2010|chapter=古今水曆及其与农历的关系}}</ref><ref name=chen>{{cite book|title=《中国少数民族天文学史》|author=陈久金|publisher=中国科学技术出版社|year=2013|pages=120-123}}</ref>

== 新水历 ==
明清时期水族借鉴夏历创立了新水曆，属于[[阴阳历|阴阳历]]。其曆月为[[朔望月|朔望月]]，大月为30天、小月为29天。现行水曆与夏曆一样采用[[闰月|闰月]]调整朔望月与回归年之间的关系。不过水历将闰月置于九月与十月之间，与夏曆[[无中气月|无中气置闰]]的原则并不相同。此外，古水曆以夏曆九月为岁首、以亥日为元旦的习惯则延续至今。<ref name=huang /><ref name=chen />

=== 纪年法 ===
水曆纪年同样采用七元甲子法，以420年为一元。汉族曆法亦曾使用过此种纪年法，后为日本[[貞享曆|貞享曆]]所沿用。<ref name=huang />

== 节日 ==
水族的主要节日包括：<ref>{{cite book|title=《水族》|author=徐非|publisher=吉林出版集团有限责任公司|year=2010|pages=96-101}}</ref><ref>{{cite book|title=《中国的少数民族节日》|editor=范玉梅|publisher=社会科学文献出版社|year=2013|pages=397-402}}</ref>
* [[端节|端节]]：[[水语|水语]]称“借端”，是水族的[[年节|年节]]，也是水族最隆重的节日，为水历十二月至二月间的[[亥|亥]]日，各地按传统分期分批过节。
* 卯节：水语称“借卯”，是部分水族地区的年节，主要盛行于贵州[[黔南州|黔南州]][[三都县|三都县]]与[[荔波县|荔波县]]。卯节为水历九、十月间的[[卯|卯]]日，与端节同样为各地分批过节。水族中有“过端不过卯、过卯不过端”的传统。
* 苏宁喜节：水历四月[[丑_(地支)|丑]]日，相传为生母娘娘送子嗣的日子。“苏宁喜”即为水语“四月丑日”之意。
* 荐节：水语“正月”之意，即汉族[[春节|春节]]。
* 敬霞节：“霞”为水语“水神”之意，是水族祭祀水神的节日，为水历丑未年或子年九、十月间的[[酉|酉]]日。
