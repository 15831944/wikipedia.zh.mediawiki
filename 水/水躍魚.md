'''水躍魚'''（{{lang-ja|'''ミズゴロウ'''}}，{{lang-en|'''Mudkip'''}}）是[[精靈寶可夢|精靈寶可夢]]系列登場的{{NUMBEROFPOKEMON}}種虛構[[角色|角色]]（[[怪物|怪獸]]）中的一種。

== 特徵 ==
為水系的沼魚寶可夢。主要顏色為藍色。頭上的鰭有雷達的作用，可以判別周圍的狀況。

== 遊戲中的水躍魚 ==
* 和草屬性的[[木守宮|木守宮]]，火屬性的[[火稚雞|火稚雞]]為[[神奇寶貝_紅寶石·藍寶石|紅寶石、藍寶石、綠寶石版]]最初拿到的三隻寶可夢之一，特性為激流，沒有野生種，全國圖鑑編號為258，在16級時進化成[[沼躍魚|沼躍魚]]。
* 可在42級學會水屬性強力招式「水炮」，但進化便學不到該招。
* [[神奇寶貝_心金、魂銀|心金、魂銀版]]中，在白銀山上擊敗小智後，前往金黃市的西爾佛公司，在右上角的櫃檯會遇見大吾，與其對話後並選擇「藍色的石頭」可獲得水躍魚。

== 動畫中的水躍魚 ==
* 小剛的水躍魚，一開始並不信任小剛，但在經過與小剛的互動，開始對小剛的態度變得友善，後來被小剛收服。
* 在《[[神奇寶貝超世代|神奇寶貝超世代]]》第147集中，在與小智的[[森林蜥蜴|森林蜥蜴]]對戰時進化成[[沼躍魚|沼躍魚]]。

== 漫畫中的水躍魚 ==
* 在[[神奇寶貝特別篇|神奇寶貝特別篇]]第四章紅寶石·藍寶石篇中，男主角路比所持有的寶可夢，並稱呼牠為「ZUZU」，也是路比隊伍中唯一的男生，在華麗大賽中參加「強壯」部門，後進化成巨沼怪。

== 其他關於水躍魚 ==
* 寶可夢樂園中的設施「水躍魚水花四濺的滑水道冒險之旅」即是以水躍魚的名字命名。
