'''水中毒'''（{{lang|en|Water intoxication}}）是一種因為人體於短時間內攝取了過量[[水|水分]]而產生稀釋性低鈉症的[[中毒|中毒]]徵狀。

人體[[腎臟|腎臟]]的持續最大利[[尿|尿]]速度是每分鐘16毫升，一旦攝取了超過這個速度的水分，過剩的水分會使[[細胞|細胞]]膨胀，從而引起稀釋性低鈉血症。當飲用過量水分時，血液內的[[電解質|電解質]]因為被水分排出體外而降至低於安全水分的濃度，影響到[[腦部|腦部]]的運作，可能會致命<ref>{{cite web|url=http://www.hhp.ufl.edu/faculty/pbird/keepingfit/ARTICLE/toomuchwater.htm|title=You Can Drink Too Much Water|accessdate=2007-01-21|last=Bird|first=Patrick J.|year=2000|publisher=University of Florida|deadurl=yes|archiveurl=https://web.archive.org/web/20070207053046/http://www.hhp.ufl.edu/faculty/pbird/keepingfit/ARTICLE/toomuchwater.htm|archivedate=2007-02-07}}</ref>，雖然致死的機會非常低。

== 徵狀 ==
血液中的[[鈉|鈉]][[離子|離子]]濃度過低程度不同，會出現以下不同的徵狀：
*低於130mEq/L：開始出現輕度的疲勞感
*低於120mEq/L：開始出現[[頭痛|頭痛]]、[[嘔吐|嘔吐]]或其他精神症状
*低於110mEq/L：除了性格變化，還伴隨[[痙攣|痙攣]]、[[昏睡|昏睡]]的感覺
*低於100mEq/L：[[神經|神經]]訊號的傳送受到影響，導致[[呼吸困難|呼吸困難]]，可能會引致[[死亡|死亡]]

== 原因 ==
水中毒的成因一般都是因為各種原因而攝取了過剩的水分，但這個分量在日常的生活裡很少機會超過。以下是部份常見的可能原因：

*在大量出[[汗液|汗]]後馬上大量補充水分。人在大量出汗後，不但會流失水分，也會流失不少[[鹽|鹽]]分，此時若一次大量喝水而不補充鹽分的話，血液中的鹽分就會減少，吸水能力也隨之降低，一些水分就會很快被吸收到組織細胞內，使[[細胞|細胞]]水腫，造成「慢性水中毒」<ref>《[[am730|am730]]》2007年11月22日號，p.39 HEALTH版am健康。</ref>。這時人就會覺得有頭暈、口渴的現象，嚴重的會突然昏倒，極端情況下有機會致死。
*另一個常見的可能一般在西方國家舉行的「飲水競賽」發生。在這些競賽裡，參賽者需要在數分鐘內飲用超過3[[加侖|加侖]]（約10[[公升|公升]]）的水。
*還有一種可能是因為受到某種不明的[[精神分裂症|精神分裂症]]藥物的影響而飲用過量的水分。這種藥物的副作用會影響到[[抗利尿激素|抗利尿激素]]的分泌，使病人覺得口渴而不斷飲水<ref>{{lang-ja|現代臨床精神医学}}</ref><ref>[http://news.sina.com.hk/cgi-bin/nw/show.cgi/2/1/1/1387015/1.html 精神病漢狂飲水 懷疑水中毒暈倒]</ref>。

== 治療法 ==
*減少飲水
*尋求專業醫療協助

== 人類以外的其他動物 ==
現時發現除了人類以外，其他動物亦有可能在[[點滴|點滴]]時因為低張性電解質過剩而引起水中毒。在幼牛發生的水中毒，會引起[[血色素尿|血色素尿]]<ref>{{lang-ja|獣医学大辞典編集委員会編集『明解獣医学辞典』チクサン出版}}、1991年ISBN 4885006104</ref>。
