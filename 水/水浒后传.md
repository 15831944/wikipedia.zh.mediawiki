'''水浒后传'''，[[清代|清代]]长篇[[白话|白话]][[章回小说|章回小说]]，作者为[[陈忱|陈忱]]，为《[[水浒传|水浒传]]》诸续书中文学价值较高的一种。

全书共四十回，主角為[[李俊_(水浒传)|李俊]]。故事内容为[[梁山泊|梁山泊]]幸存的梁山好汉及部分英雄后代的故事，他们因为官吏压迫，再度起义，在面临金军入侵时，他们又奋勇抗金，几经坎坷之后，他们不得不远渡海外。

== 创作背景 ==
陈忱生活于[[明朝|明]][[清朝|清]]之交，以明朝遗民自居，对明亡抱有伤感之意。在晚年，他借《水浒后传》一书来表达“古宋遗民之心”（语见《水浒后传》序）。<ref>章培恒，骆玉明.《中国文学史》，复旦大学出版社，2004年。（第八编-清代文学）ISBN 730901854</ref>

== 故事情节 ==
阮小七被王禀、赵谭诬告谋反，结果被罢官返回故里。一日，阮小七到梁山泊祭奠[[宋江|宋江]]等，被原[[蔡京|蔡京]]爪牙[[张干办|张干办]]抓捕。阮小七杀死张干办，携母出逃。在路上，遇到开酒店的[[孙新|孙新]]、[[顾大嫂|顾大嫂]]等人。原梁山余众因不堪官吏欺压，再次起义，聚集登云山、饮马川等地。[[李俊_(水浒传)|李俊]]隐居[[太湖|太湖]]以捕鱼为生，受到湖霸、官府欺压，与[[花逢春|花逢春]]、[[乐和|乐和]]等出海共建王业。不久之后，金兵南侵，[[宋徽宗|宋徽宗]]、[[宋钦宗|宋钦宗]]二帝被掳走。[[李应|李应]]、[[呼延灼|呼延灼]]等人决定报国勤王，加入以[[李纲|李纲]]、[[宗泽|宗泽]]等人领导的抗金阵营，并处死了[[蔡京|蔡京]]、[[高俅|高俅]]等奸臣。最后由于形势所迫，梁山众人弃寨出海，在[[暹罗|暹罗]]与李俊会合，李俊成为暹罗国主。

== 与《[[水浒传|水浒传]]》的比较 ==
除了幸存的三十二人以外，还增加了[[花荣|花荣]]之子[[花逢春|花逢春]]，[[徐宁|徐宁]]之子[[徐晟|徐晟]]，[[呼延灼|呼延灼]]之子[[呼延钰|呼延钰]]、[[宋清|宋清]]之子[[宋安平|宋安平]]，等人物。

徐晟生得唇紅齒白，面目俊秀，慣使父親傳下的一根金鎗，武藝高強，而且心機靈巧，具正義感。在父母亡過後，一直流落[[東京|開封]]，守護著祖傳的護甲「賽唐猊」，跟乳母相依為命。一次在小巷裡奪回被流氓搶去的「[[賽唐猊|賽唐猊]]」，因而遇上了亡父昔日的同伴「[[雙鞭|雙鞭]]」[[呼延灼|呼延灼]]，相認之後[[呼延灼|呼延灼]]憐其孤苦，便把徐晟接回自己府中撫養，更讓他與自己的兒子[[呼延鈺|呼延鈺]]共同受業於[[聞煥章|聞煥章]]，學習文史經傳。徐晟跟[[呼延鈺|呼延鈺]]情投意契，二人一直以兄弟相稱；後來二人遇上[[宋江|宋江]]之姪、[[宋清|宋清]]之子[[宋安平|宋安平]]及[[花榮|花榮]]遺孤[[花逢春|花逢春]]，四人並結同心，成為梁山第二代傑出人物。隨著梁山餘眾東征西討，鋤強扶弱，最後一眾人等統治了[[暹羅|暹羅]]國，佔地為王。而且在[[呼延灼|呼延灼]]的促成下，徐晟更娶了[[呼延灼|呼延灼]]的女兒[[呼延玉英|呼延玉英]]為妻，兩家結為姻親。

== 评价 ==
[[鲁迅|鲁迅]]在《[[中国小说史略|中国小说史略]]》中评价道：“故虽游戏之作，亦见避地之意矣”。<ref>鲁迅《中国小说史略》第十五篇：“清初，有《后水浒传》四十回，云是“古宋遗民著，雁宕山樵评”，盖以续百回本。...(陈忱)生平著作并佚，惟此书存，为明末遗民（《两浙輶轩录》补遗一《光绪嘉兴府志》五十三），故虽游戏之作，亦见避地之意矣。”</ref>[[胡适|胡适]]则评价道：“《后水浒》绝不是‘游戏之作’，乃是很沉痛地寄托他亡国之思，种族之感的书。”。
