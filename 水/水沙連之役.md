== 背景 ==
{{Main|骨宗事件}}
雍正三年（1725年）[[農曆|農曆]]八月至雍正四年（1726年）十二月間，水沙連地區各社原住民以[[水社|水社]][[土目|土目]][[骨宗|骨宗]]（[[邵族|邵族]]）為首，於今臺中[[南屯區|南屯]]、彰化[[社頭鄉|社頭]]、雲林[[斗六市|斗六]]一帶傷人或出草<ref name=":0">{{Cite web|url=http://nrch.culture.tw/twpedia.aspx?id=5593|title=骨宗事件|accessdate=2018-12-23|author=簡史朗|date=2009-09-09|publisher=臺灣大百科全書|language=zh-tw}}</ref>，造成死亡人數為62人、耕牛140頭、焚燒房屋近百間。<ref name=":1">{{Cite journal|title=〈水沙連之役及其相關問題〉|author=陳哲三|url=http://www.cohss.fcu.edu.tw/wSite/public/Attachment/f1378095906031.pdf|journal=《逢甲人文社會學報》|issue=18|doi=|others=|year=2009|page=|pmid=}}</ref>

== 戰略 ==
雍正三年（1725年）十一月，官方原本採取嚴禁漢民入番界、不積極反抗之政策，但因內山原住民出山殺人案件層出不窮，官方遂轉變立場。雍正四年（1726年）正月，[[福州市|福州]]將軍署理[[閩浙總督|閩浙總督]][[宜兆熊|宜兆熊]]上奏「以番攻番」之計，提出逮捕首犯一、二名以示威鎮、其餘者撫綏的戰略（順者撫之，抗者剿之）<ref>{{Cite book|title=〈宜兆熊：奏報請撫臺灣生番摺（雍正四年正月三日）〉．《宮中檔雍正朝．第五輯》|last=|first=|publisher=國立故宮博物院|year=1978|isbn=|location=台北市|pages=501|language=zh-tw}}</ref>，獲得[[雍正皇帝|雍正皇帝]]許可，地方當局開始規劃水沙連用兵大略。<ref name=":1" />{{Rp|89}}

討伐水沙連原住民的部署至雍正四年八月已然完成，浙閩總督[[高其倬|高其倬]]與[[福建巡抚|福建巡撫]][[毛文銓|毛文銓]]八月十日特召[[臺廈道|台廈道]][[吳昌祚|吳昌祚]]回福建省，當面詳問用兵之計，並委吳昌祚為總統，令其於十一月返台後開始發動。<ref name=":1" />{{Rp|89-90}}

== 部署 ==
水沙連之役採兩路進發，一為[[南港溪_(南投縣)|南港]]一路（[[烏溪|烏溪]]上游支游）一路、二為[[北港溪_(南投縣)|北港]]（亦為烏溪上游支流），而以[[吳昌祚|吳昌祚]]率領的南路軍為主力，後勤支援、辦運軍糧由[[彰化縣_(清朝)|彰化縣]][[知縣|知縣]][[張縞|張縞]]負責，[[巡視臺灣監察御史|巡台御史]][[索琳|索琳]]隨軍進剿：<ref name=":1" />{{Rp|100-103}}

== 落幕 ==

* 12月17日，監軍索琳先行離開水裡社，東至西循竹腳寮渡虎尾溪，經南投社、北投社（南投[[草屯鎮|草屯鎮]]）、[[洪雅族|猫羅社]]（彰化[[芬園鄉|芬園鄉]]）與半線社，安然返回彰化縣縣治。<ref name=":1" />{{Rp|96}}
* 總統吳昌祚持續駐留山區，下令調查南港、北港、中港各處番口，要求諸社輸糧納餉，其中貓里眉社（[[泰雅族|泰雅族]]）押解涉案兇番阿密氏貓著至軍前，後亦押交臺灣府發禁。<ref name=":1" />{{Rp|92、96}}
*水沙連周遭諸社（含邵族、布農、泰雅、[[賽德克族|賽德克族]]）沿路被招撫，表達願意納番餉者各社人口共4445名。<ref name=":1" />{{Rp|104}}
* 12月27日，吳昌祚率同何勉、王汧循蛤裏難社（今[[埔里鎮|埔里]]）下山，並在半線社與索琳會合，戰事乃平。
* 首犯骨宗與貓著被判「[[斬立決|斬立決]]」，二犯被押解至北路番子山口，梟首示眾。拔思弄等十三人則判「[[斬監候|斬監候]]」，又水-{里}-萬等五人照為從例枷責，發落於[[台灣省城|省城]]之中永久監禁。<ref>{{Cite book|title=〈高其倬：奏報審辦臺灣兇番摺（雍正五年四月四日）〉．《宮中檔雍正朝．第七輯》|last=|first=|authorlink=高其倬|publisher=國立故宮博物館|year=1978|isbn=|location=台北市|pages=892-894|language=zh-tw}}</ref>
