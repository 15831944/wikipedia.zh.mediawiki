'''水北阁'''位于[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[海曙区|海曙区]]，为[[徐时栋|徐时栋]]的私人藏书楼。水北阁最初位于西门外花池巷18号，始建于[[同治|同治]]三年（1864年）六月，1984年10月26日被列入海曙区文物保护单位。1993年，出于保护目的，水北阁迁建至[[天一阁博物馆|天一阁博物馆]]内。1999年，水北阁成为中国地方志珍藏馆。

==历史==
[[File:Ningbo_Xu_Shidong_Guju_2013.07.27_07-13-25.jpg|thumb]]
徐时栋最初的藏书楼位于月湖恋湖楼（又称烟屿楼），但因[[咸丰_(年号)|咸丰]]十一年（1861年）[[太平天国|太平天国]]军队攻宁波，恋湖楼内藏书大部分被盗窃或焚毁。<ref>{{cite web|url=http://zw.nbwh.gov.cn/art/2015/3/10/art_608_4998.html|title=水北阁|date=2015-03-10|publisher=宁波市文化广电新闻出版局|quote=他的读书处原在月湖旁，初名恋湖楼，又称烟屿楼，积书六万卷，毁于咸丰十一年（1861年）}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>[[同治|同治]]元年（1862年）徐时栋迁居宁波西门外，将藏书楼命名为城西草堂，但于[[同治|同治]]二年（1863年）十一月二十九日被大火将书与楼一并焚毁。[[同治|同治]]三年（1864年）六月，在城西草堂旧址建立水北阁，有河将水北阁与住宅隔开，意在防火，由陈劢书匾。<ref>{{cite web |url=http://yzsz.nbyz.gov.cn/art/2011/6/22/art_9115_259454.html |title=徐时栋与水北阁 |author=龚烈沸 |date=2011-06-22 |publisher=中共宁波市鄞州区委党史办公室　宁波市鄞州区地方志办公室 |quote=同治三年(1864)六月，在城西草堂旧址重建新宅，新书楼名“水北阁”，陈劢书匾（已毁）}}</ref>同治八年（1869年），徐时栋将鄞县志局移至水北阁。[[宣统|宣统]]三年（1911年），水北阁30大柜藏书被售给上海书商，至此水北阁藏书全部散失，后水北阁成为徐氏后人居所。1984年10月26日，水北阁被公布为海曙区文物保护单位。1993年，宁波市政府借宁波市旧城改造的机会决定将水北阁从海曙区西门外花池巷18号迁至天一阁建设控制范围内，并将全屋落架，于1996年在[[天一阁|天一阁]]南园原状复原。<ref>{{cite web |url=http://yzsz.nbyz.gov.cn/art/2011/6/22/art_9115_259454.html |title=徐时栋与水北阁 |author=龚烈沸 |date=2011-06-22 |publisher=中共宁波市鄞州区委党史办公室　宁波市鄞州区地方志办公室 |quote=藏书散出后，在宁波西门外花池巷8号的水北阁为徐氏后裔居住。新中国成立后，于1984年10月26日被公布为海曙区重点文物保护单位，但由于居民众多，存在火险等隐患。1993年，在宁波旧城改造中，为更好地保护水北阁，宁波市政府决定将水北阁迁移至天一阁建设控制范围内进行保护，在对原屋进行丈量、拍摄等技术处理后，全屋落架，于1996年在天一阁南园按原貌恢复。}}</ref>1999年，水北阁成为中国地方志珍藏馆。

==建筑结构==
[[File:水北阁.jpg|thumb]]
水北阁坐南朝北，现占地242.2平方米，建筑面积523.3平方米，为两层三开间间一弄木结构建筑。<ref>{{cite web|url=http://zw.nbwh.gov.cn/art/2015/3/10/art_608_4998.html|title=水北阁|date=2015-03-10|publisher=宁波市文化广电新闻出版局|quote=阁坐南朝北，三开间木结构}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url=http://yzsz.nbyz.gov.cn/art/2011/6/22/art_9115_259454.html |title=徐时栋与水北阁 |author=龚烈沸 |date=2011-06-22 |publisher=中共宁波市鄞州区委党史办公室　宁波市鄞州区地方志办公室 |quote=占地面积242.2平方米，建筑面积523.3平方米}}</ref>
