《'''水行俠'''》（{{lang-en|'''''Aquaman'''''}}）是一套2018年[[美國|美國]][[水行俠_(電影)|同名电影]]的[[電影原聲帶|原聲帶]]，由{{link-en|魯伯特·葛雷森-威廉姆斯|Rupert Gregson-Williams}}創作，於2018年12月14日由{{link-en|水塔音樂|WaterTower Music}}發行<ref>{{cite web |last1=Fabi |first1=Tia |title=Aquaman's Soundtrack Has Release Date |url=https://gvnation.com/aquamans-soundtrack-has-release-date/ |accessdate=2018-11-17 |publisher=Gvnation.com |date=2018-11-17|archive-url=https://web.archive.org/web/20181118081638/https://gvnation.com/aquamans-soundtrack-has-release-date/|archive-date=2018-11-18|dead-url=no }}</ref><ref>{{cite web |title=Aquaman |url=http://www.hans-zimmer.com/index.php?rub=disco&id=1669 |publisher=Hans-Zimmer.com |accessdate=2018-11-17|archive-url=https://web.archive.org/web/20181118081502/http://www.hans-zimmer.com/index.php?rub=disco&id=1669|archive-date=2018-11-18|dead-url=no }}</ref> 。

2018年3月7日，葛雷森-威廉姆斯被選來為電影《[[水行俠_(電影)|水行俠]]》的配樂作曲。他先前也曾為[[DC擴展宇宙|DC擴展宇宙]]電影《[[神力女超人_(2017年電影)|神力女超人]]》（2017年）作曲<ref name="Gregson-Willimas">{{cite web |title=Rupert Gregson-Williams to Score James Wan's 'Aquaman' |url=http://filmmusicreporter.com/2018/03/07/rupert-gregson-williams-to-score-james-wans-aquaman/ |publisher=''Film Music Reporter'' |accessdate=2018-03-07|archive-url=https://web.archive.org/web/20180308041906/http://filmmusicreporter.com/2018/03/07/rupert-gregson-williams-to-score-james-wans-aquaman/|archive-date=2018-03-08|dead-url=no }}</ref>。該原聲帶收錄了美國音樂家[[斯盖拉·格蕾|斯盖拉·格蕾]]与艾略特·泰勒创作的原創歌曲〈Everything I Need〉<ref>{{cite web |title=James Wan's 'Aquaman' to Feature Original Song by Skylar Grey |url=http://filmmusicreporter.com/2018/11/28/james-wans-aquaman-to-feature-original-song-by-skylar-grey/ |3= |publisher=''Film Music Reporter'' |accessdate=2018-11-29|archive-url=https://web.archive.org/web/20181130030448/http://filmmusicreporter.com/2018/11/28/james-wans-aquaman-to-feature-original-song-by-skylar-grey/|archive-date=2018-11-30|dead-url=no }}</ref>。而第二支預告片中的〈Sidewinder〉則由Ghostwriter Music的作曲家菲爾·洛普所創作<ref>{{cite web |last1=Lin |first1=Craig |title=Listen To The Epic Music From The Final 'Aquaman' Trailer |url=https://dcfilmshub.com/listen-to-the-epic-music-from-the-final-aquaman-trailer/ |website=DC Films Hub |accessdate=2018-12-22}}</ref>。
