'''包头市'''（{{mong|m= ᠪᠤᠭᠤᠲᠤ<br /> ᠬᠣᠲᠠ |k=Бугат хот|Ppp=Buɣutu qota|PS=Bûgt hôt|ZM=Buɡutu xota}}）是[[中华人民共和国|中华人民共和国]][[内蒙古自治区|内蒙古自治区]]下辖的[[地级市|地级市]]和最大城市<ref>[http://www.baotounews.com.cn/content/2015-02/11/content_431507.htm 包头市城镇人口总量居全区之首 城镇化居第二]{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，全国重要的冶金、材料和制造业基地，[[中华人民共和国国务院|国务院]]批准设立具有地方立法权的[[较大的市|较大的市]]，[[华北地区|华北地区]]重要的工业城市。

包头市区由各具经济特色的3个中心街区组成。东南部是东河区，又称旧城（老包头），建筑在[[黄河|黄河]]一级阶地上，以商业和轻工业及有色冶金工业为主；中部为青山区，以机械制造业为主；西部是昆都仑区，即包钢所在地，以钢铁及重型机械等重工业为主；旧城与新城之间的九原区有大片绿地间隔，<ref>[http://ecph.cnki.net/Schresult.aspx 中国大百科全书 包头市]</ref>高新技术产业开发区的滨河新区和石拐区的喜桂图新区也是构成市区的重要组团。城市的标志是第一工人文化宫门前的“人”字形雕塑，顶部有三只奔跑中的[[鹿|鹿]]，象征“鹿城”。同时获得中国优秀旅游城市、国家园林城市、全国文明城市等称号。

== 释名 ==
关于包头地名的由来通常认为是[[蒙古语|蒙古语]]“包克图”（{{Mong|image=Bugutu.svg|16px}}）谐音而来，意为“有鹿的”，官方采用此说法；第二种说法是蒙古语“博托”（{{mong|m=ᠪᠠᠭᠠᠲᠣᠷ}}，即“[[巴特尔_(称谓)|巴特尔]]”，蒙语“英雄”意<ref>{{Cite web |url=http://www.btradio.com.cn/view.asp?%2F=7384 |title=东河改造唤醒260年沉睡记忆 |access-date=2017-06-22 |archive-url=https://web.archive.org/web/20150923194932/http://www.btradio.com.cn/view.asp?%2F=7384 |archive-date=2015-09-23 |dead-url=yes }}</ref>）的谐音，“博托”河就是指今天的东河槽；第三种说法是“泊头”的变音，意为停泊的码头，因包头原是水旱码头。<ref>{{Cite web |url=http://www.btradio.com.cn/fm1059/view.asp?%2F=33 |title=包头地名的由来与传说 |access-date=2017-06-22 |archive-url=https://web.archive.org/web/20150402124212/http://www.btradio.com.cn/fm1059/view.asp?%2F=33 |archive-date=2015-04-02 |dead-url=yes }}</ref>

== 历史 ==
黄河流经包头的地段是原始人类较早活动的地方，在这里蕴藏着大量的古人类文化遗迹，已发掘的就有10多处。在位于东河区以东15公里的阿善沟门的格膝盖沟，发掘出一处保存完好的新石器时代村落遗址和大量的文物。

[[战国|战国]]时期赵武灵王于公元前306年（武灵王20年）筑九原城。

前221年，[[秦朝|秦]]统一中国，设[[九原郡|九原郡]]。公元433年，鲜卑族建立[[北魏|北魏]]，设[[怀朔镇|怀朔镇]]。

[[辽代|辽代]]，属云内州管辖。[[元代|元代]]后，蒙古族各部落陆续进驻河套，包头地区成为[[土默特部|土默特部落]]游牧之地。

[[清|清]]乾隆五年（1741年），建筑萨拉齐，设协理通判，这是包头地区最早出现的行政建制。1809年设包头镇。1870年（同治九年）前后，包头修筑城墙，辟东、南、西、东北、西北5座城门，形成了近代包头的城市规模。

[[中华民国|中华民国]]初年，包头已发展成为中国西北著名的皮毛集散地和水旱码头。1923年平绥铁路通车包头，1926年（民国十四年），设包头县。1931年，包头电灯面粉公司和永茂源甘草公司创办，包头开始有了近代工业。1934年，中德双方组织的“欧亚航空邮运股份有限公司”在包头修筑飞机场，开辟包头--宁夏--兰州航线，定期航班每周往返一次。饮食、服务业日益兴旺，市面日趋繁荣。1937年，[[抗日战争|抗日战争]]爆发，日军进占包头。日占期间归属“[[蒙疆联合自治政府|蒙疆联合自治政府]]”统治。1938年，包头设市，1945年实行市县并存。

1949年9月19日，绥远发动“9·19”起义，包头获得和平解放。1950年2月13日，包头市人民政府成立。
1954年，[[綏遠省_(中華人民共和國)|绥]][[内蒙古自治区|蒙]]合并，包头市成为内蒙古自治区辖市。

== 自然地理 ==
=== 位置 ===
包头位于北纬41°20′－42°40′，东经109°50′－111°25′，处河套平原前缘，南濒黄河，北界中蒙边界，东西南分别与[[呼和浩特|呼和浩特]]、[[巴彦淖尔|巴彦淖尔]]、[[鄂尔多斯|鄂尔多斯]]接壤，北靠[[蒙古国|蒙古国]]，是内蒙古自治区的中心位置。全市面积27691平方千米，南北最长处约235公里，东西最宽处约126公里。其中市区面积为361平方千米。

=== 地形 ===
包头境内有阴山山脉的大青山、乌拉山（以昆都仑河为界），山峰平均海拔2000米，最高峰海拔2324米。全市由中部山岳地带、山北高原草地和山南平原三部分组成，呈中间高，南北低，西高东低的地势。

=== 气候 ===
包头属半干旱大陆性[[温带季风气候|温带季风气候]]，四季分明，季风影响严重。冬季严寒，夏季温暖。2002年包头市年均气温8.5℃，年最低气温-27.6℃，年最高气温40.4℃，年降水总量262.9毫米，年最大风速11.0米/秒，平均风速1.8米/秒，年日照时数2806小时，年平均相对湿度52%，全年沙尘天气12次。

=== 民族 ===
包头市是个多民族聚居的地方。全市共有43个民族，其中少数民族人口数为13.08万人，占全市总人口的6.25%。千人以上少数民族共有3个，按人数多少排列分别为[[蒙古族|蒙古族]]、[[回族|回族]]和[[滿族|滿族]]。<ref>[http://61.138.111.246/xxzx/pages/pagelist.jsp?id=9526 包头市人口现状与对策分析 ]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 包頭蒙古族主體分佈在[[百靈廟|百靈廟]]一帶。回族主體分佈在東河區北梁一帶。全市常住人口中，汉族人口为2499508人，占94.31%；蒙古族人口为85121人，占3.21%；其他少数民族人口为65735人，占2.48%。

包头市是个移民城市，居民主體来自[[华北|华北]]和[[中国东北地区|东北]]。[[清朝|清朝]]末年至民国初年的[[走西口|走西口]]移民过程中，大批[[山西|山西]]、[[陕西|陕西]]、[[河北|河北]]的移民涌入包头，包头人口开始逐渐增加。[[中华人民共和国|中华人民共和国]]建国后，发起“支援边疆建设运动”、流放右派分子等行動，大量知识分子、工人从东北、河北等地迁入包头，这些人及其后裔在包头的城市人口中的比例很高。

包头市人口主要是由外来移民构成，其中[[昆都仑区|昆都仑区]]和[[青山区_(包头市)|青山区]]在[[中华人民共和国|中华人民共和国]]建国后外来移民最多，原因是该地区是包头的新区和大型国有企业聚集区。<ref>[http://www.china.com.cn/aboutchina/zhuanti/pbfznmg/2009-04/13/content_17595490.htm 包头概况]</ref>

=== 城市轨道交通 ===
[[包头轨道交通|包头轨道交通]]，通称[[包头地铁|包头地铁]]。它的作用是为了缓解包头市内日益拥堵的交通状况，与包头公交高低搭配形成立体交通网络，同时还兼有改善城市形象、加强新老城区联系、缩小新老城区差距的作用。包头轨道交通共规划有6条线路，这6条线路东西方向贯通城区西部的包钢工业区和城区东部的东兴工业区，南至包头黄河大桥，北至京藏高速南侧，基本覆盖了城区各主要交通干道和人流密集区。该项目已于2016年8月21日通过国务院审批，2017年5月开始建设，预计2022年年底1号线和2号线（一期）建成通车。2017年8月，包头地铁被叫停，目前恢复时间未知。

== 经济 ==
===工业===
包头市是一个新兴的重工业城市，是[[第一个五年计划_(中国)|第一个五年计划]]重点建设的工业中心之一，苏联援建的156个重点项目，包头就有6个。一些与军工相关的重工业全国领先，拥有机械电子、石油化工、冶金、建材、轻纺、食品等6大支柱产业和[[包頭鋼鐵|包钢]]、北方奔驰、稀土高科、明天科技、小尾羊、鹿王集团、小肥羊等一大批国际国内名牌产品。

但是随着改革开放后暴露的一些体制问题，20世纪90年代国有的大型重工业企业十分艰难。近年来由于政府的成功运作，包头的发展恢复了较快的速度，以机械、冶金工业为主，还有石化、纺织、食品等门类的工业开始复苏，是一个恢复生机的工业城市。

=== 发展定位 ===
“五个基地”，即努力建成国家和自治区重要的稀土新材料基地、清洁能源输出基地、新型冶金基地、现代装备制造业基地、新型煤化工基地。
“四个中心”，即努力建成区域生产性物流中心、区域性金融中心、区域性消费中心、区域性文化旅游中心。
“两个屏障”，即努力建成北方重要的生态安全屏障和祖国北疆安全稳定屏障。
“一个支点”，即发挥满都拉口岸利用俄蒙资源和对外贸易优势，推动达茂白云合并建设边境口岸型城市，努力建设自治区向北开放的重要战略支点。<ref>[http://www.baotou.gov.cn/info/1709/49841.htm 中共包头市十一届七次全委会议决议]</ref>

===特色美食===
涮羊肉：又称羊肉[[火锅|火锅]]，将羊肉切成薄片，放入沸腾的水或汤中涮，片刻就可以出锅品尝，深受包头人民喜爱。著名的火锅企业有[[小肥羊|小肥羊]],以及小尾羊等。

[[莜面|莜面]]：莜面是内蒙古地区“三件宝”之一。

[[羊肉|手扒肉]]：手扒肉是红食中的一种。蒙古族人民称肉食为红食，蒙语叫“乌兰伊德”。顾名思义，“手扒肉”就是手抓羊肉，是蒙古族千百年来的传统食品，是牧民们的家常便饭。

[[烤全羊|烤全羊]]：蒙古族传统名菜，为招待贵宾或举行重大庆典时的盛宴特制的佳肴。一般选用[[草原|草原]]上膘肥、体重40斤左右的绵羊宰杀后，去毛带皮腹内加[[葱|葱]]、[[薑|姜]]、[[胡椒|椒]]、[[食盐|盐]]等佐料整体烤制而成。烤完以后，羊形完整，羊跪在方木盘内，色泽金红，羊皮酥脆，羊肉嫩香。

拔丝[[奶豆腐|奶豆腐]]：内蒙古风味名菜。以奶豆腐为主料烹制而成。

[[烧卖|烧卖]]（在包头一般写为'''稍美'''）：是形容顶端蓬松束折如花的形状，是一种以烫面为皮裹馅上笼蒸熟的面食小吃。烧麦最晚起源于元代初期、今内蒙古呼和浩特一带的商途茶馆。虽起源于呼和浩特，但现如今已成为包头市民早餐不可或缺的美味佳肴。

[[羊杂碎|羊杂碎]] 集羊内脏于一锅，形色繁多，肉质各异，味道酸辣兼宜，无腥不腻，汤鲜味美，入口生津，营养丰富，在气候偏冷的北方食用可御冷逐寒，是很多包头市民喜爱的食品。

此外还有铁锅排骨焖面、羊架子、羊棒骨、沙葱包子、炸糕、锅盔等具有包头地方特色的美食。

== 荣誉 ==
2000年，中国优秀旅游城市。

2002年，[[联合国|联合国]]人居奖。当年全球共有9个城市获奖，包头市政府获此殊荣。这是中国西部地区首个获此殊荣的城市。

2003年，全国园林绿化先进城市。继1994年、1999年两次被[[中华人民共和国|国家]][[建设部|建设部]]授予“全国园林绿化先进城市”荣誉称号之后，当年包头市第三次获此殊荣。

2005年，中国首批十个[[全国文明城市|文明城市]]。包头市是中西部惟一荣获该荣誉的城市。

2006年，第三届中华环境奖。

2007年，国家森林城市。 2007年5月9日被全国绿化委员会、国家林业局授予“[[国家森林城市|国家森林城市]]”称号。至此，中国已有7个城市获得这一称号。

2012年，稀土之都。8月8日在中国包头·稀土产业(国际)论坛上，包头市被正式命名为“稀土之都”。
