'''包喜樂'''，英文全名為'''喬伊·瑪利·波克'''（{{lang-en|Joy Marie Burke}}<ref name="dameligaen" />，{{bd|1990年|11月7日|catIdx=C}}），[[臺灣人|台]][[美國人|美]][[混血|混血]]、雙籍女子籃球員，司職[[中鋒_(籃球)|中鋒]]，代表[[中華台北女子籃球代表隊|中華台北女子籃球代表隊]]出賽，是中華女籃隊史上的第一長人<ref>{{cite news |url = http://www.fiba.basketball/news/burke-primed-to-lead-chinese-taipeis-podium-quest |title = Burke primed to lead Chinese Taipei's podium quest |website = [[FIBA|FIBA]] |location = Taipei City |date = 2015-08-05 |language = en |accessdate = 2018-07-28 }}</ref>。現效力於[[澳大利亞|澳大利亞]]的{{link-en|東南澳洲籃球聯賽|South East Australian Basketball League}}（SEABL）{{link-en|巴拉瑞特淘金熱|Ballarat Rush}}<ref>{{cite web |url = http://www.ballaratbasketball.com/new-name-same-joy/ |title = New Name, Same Joy |website = Ballarat Rush |language = en |accessdate = 2018-07-27 }}</ref><ref name="rush">{{cite web |url = http://www.ballaratbasketball.com/international-superstar-joy-burke-will-return-to-the-rush-in-2018/ |title = International Superstar Joy Burke will return to the Rush in 2018 |website = Ballarat Rush |language = en |accessdate = 2018-07-28 }}</ref>。

== 生平 ==
包喜樂於1990年出生於臺灣[[嘉義|嘉義]]，父親史蒂夫·波克當年以牧師身分來臺傳教，媽媽是台灣高雄人<ref name="sundevils">{{cite web |url = https://thesundevils.com/news/2013/4/17/208252096.aspx |title = Joy Burke Profile |website = Arizona State University Official Athletic Site |date = 2013-05-08 |accessdate = 2018-07-30 |language = en }}</ref><ref name="career">{{cite news |author = 蔡裕隆 |url = https://tw.appledaily.com/sports/daily/20150715/36666197 |title = 女籃第1長人 196cm包喜樂報到 |date = 2015-07-15 |publisher = 蘋果日報 |accessdate = 2018-07-27 }}</ref>。她有一個相差三歲的弟弟[[包提傑|包提傑]]。包喜樂在四歲時曾經拍過洗髮精廣告<ref name="sundevils" />。直到十二歲之前，包喜樂一直在台中的[[馬禮遜美國學校|馬禮遜美國學校]]就讀。高中畢業後，她到了美國[[坦佩_(亞利桑那州)|坦佩]]的[[亞歷桑納州立大學|亞歷桑納州立大學]]就讀，主修市場行銷，僅花費五年時間便拿到了學士和碩士學位<ref name="career" />。期間，她效力於[[NCAA第一級別|NCAA第一級別]]的{{link-en|亞歷桑納州立大學太陽惡魔隊|Arizona State Sun Devils}}。

=== 職業生涯 ===
2014年，大學畢業後，包喜樂在{{link-da|丹麥女子籃球聯賽|Dameligaen}}開啟了她的職業生涯。她為{{link-da|赫城79人|Hørsholm 79ers}}效力了一個賽季，出賽26場次，平均每場比賽可貢獻14.0分、11.2籃板，其場均籃板數在聯盟排名第三<ref name="dameligaen">{{cite web |url = http://www.dameligaen.dk/da/top/player/?player_id=4953785&season_id=89589 |title = Player - Joy Marie Burke |website = Dameligaen |language = da |accessdate = 2018-07-27 }}</ref>。2015年，她與{{link-en|澳大利亞國家女子籃球聯賽|Women's National Basketball League}}（WNBL）{{link-en|本迪哥精靈|Bendigo Spirit|本迪哥精靈隊}}簽下一年合約，轉往[[澳大利亞|澳大利亞]]發展<ref>{{cite news |url = https://www.bendigoadvertiser.com.au/story/3177611/joy-burke-signs-with-bendigo-spirit/ |title = Joy Burke signs with Bendigo Spirit |website = Bendigo Advertiser |date = June 29 2015 |language = en |accessdate = 2018-07-28 }}</ref>。2015-16賽季在本迪哥精靈，她拿下場均2.7分、2.7籃板<ref name="bendigo_profile">{{cite web |url = http://websites.sportstg.com/team_info.cgi?action=PSTATS&pID=200769252&client=1-4478-56416-375689-24591062 |title = Bendigo Spirit Player Profile - Joy Burke |website = SportsTG |language = en |accessdate = 2018-07-28 }}</ref>。2016年，包喜樂轉往{{link-en|東南澳洲籃球聯賽|South East Australian Basketball League}}（SEABL）發展，效力於{{link-en|巴拉瑞特淘金熱|Ballarat Rush}}<ref name="rush" />。迄今，她在巴拉瑞特淘金熱效力了三個球季，皆繳出場均[[籃球術語#double-double|雙十]]的成績<ref name="ballarat_stats" />，2017–18年兩個賽季以場均抓下近12個籃板分別在聯盟排名第二、第三名<ref>{{cite web |url = http://websites.sportstg.com/comp_info.cgi?c=1-3363-0-436966-0&a=STATS |title = Competition Statistics for SEABL WOMEN 2017 |website = SportsTG |language = en |accessdate = 2018-07-31 }}</ref><ref>{{cite web |url = http://websites.sportstg.com/comp_info.cgi?c=1-3363-0-478981-0&a=STATS |title = Competition Statistics for SEABL WOMEN 2018 |website = SportsTG |language = en |accessdate = 2018-07-31 }}</ref>。

=== 國家隊 ===
包喜樂擁有臺灣、美國[[雙重國籍|雙重國籍]]<ref>{{cite news |author = 蔡裕隆 |url = https://tw.appledaily.com/sports/daily/20150716/36668362 |title = 體壇新女神駕到 包喜樂台客魂 |publisher = 蘋果日報 |date = 2015-07-16 |accessdate = 2018-07-30 }}</ref>，2015年透過中華女籃的美籍教練[[歐提斯·休利|歐提斯·休利]]（Otis Hughley）牽線，加入中華女籃隊<ref>{{cite news |author =  陳筱琳 |url = https://news.tvbs.com.tw/sports/607900 |title = 籃球／中華女籃史上最高 196公分華裔包喜樂添戰力 |publisher = TVBS News |date = 2016-07-14 |accessdate = 2018-07-30 }}</ref>。她在[[2015年第三十七屆威廉瓊斯盃國際籃球邀請賽|第三十七屆威廉瓊斯盃]]首度與中華女籃藍隊隊員身分登場，同年九月隨隊參加[[2015年亞洲女子籃球錦標賽|2015年亞洲女子籃球錦標賽]]、首度披上國家隊球衣，最後以第四名作收<ref name="fiba2015">{{cite web |url=http://www.fiba.com/asiawomen/2015/Hsi-Le-BAO |title=Hsi-Le BAO (TPE)'s profile FIBA Women's Asia Championship 2015 |publisher=fiba.com |language = en |accessdate = 2018-07-28 }}</ref><ref>{{cite web |url = http://www.fiba.basketball/asiawomen/2015/0509/Chinese-Taipei-Korea |title = Chinese Taipei vs Korea - FIBA Asia Women's Championship 2015 |publisher = fiba.com |language=en |accessdate = 2018-07-30 }}</ref>。

==個人生活==
包喜樂的弟弟包提傑也是籃球運動員，曾於2016年到2018年以華裔球員的身分效力於[[SBL|SBL]]聯賽的[[富邦勇士|富邦勇士]]隊<ref>{{cite web|author=林宗偉|title=SBL》嚮往姐姐替台灣打球 包提傑也有國手夢|url=http://sports.ltn.com.tw/news/breakingnews/1850068|publisher=自由時報|date=2016-10-08|accessdate=2018-08-30}}</ref>。
