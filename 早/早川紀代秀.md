'''早川紀代秀'''（{{jp|j=早川 紀代秀|hg=はやかわ きよひで|rm=Hayakawa Kiyohide|lead=yes}}，{{bd|1949年|7月14日|2018年|7月6日}}）是一名[[奥姆真理教|奥姆真理教]]教徒、前教团高级干部（正悟师、教团建设省大臣）<ref name=Hayakawa2005>{{Cite book|author=早川紀代秀|title=私にとってオウムとは何だったのか8|publisher=ポプラ社|date=2005|pages=14-15;107-108;194;331}}</ref>，1949年出生于日本[[兵库县|兵库县]][[川邊郡|川邊郡]]東谷村（今属兵库县[[川西市|川西市]]）。	早川在20世纪80年代后期加入奥姆真理教，之后陆续参与{{link-ja|奥姆真理教男性信徒杀害事件|オウム真理教男性信者殺害事件}}、[[坂本堤律師滅門慘案|坂本堤律師滅門慘案]]等犯罪，同时是奥姆真理教秘密武装化的中心人物。1995年，早川被捕。2009年，早川死刑定谳<ref name=Hayakawa2005/><ref name=JapanTimesAum1>{{cite web |title=Profiles of top Aum Shinrikyo members, including six still on death row |url=https://www.japantimes.co.jp/news/2018/07/06/national/crime-legal/profiles-top-aum-shinrikyo-members-including-six-still-death-row/#.XAkmU3QzYac |website=Japan Times |accessdate=2018-12-06 |archive-url=https://web.archive.org/web/20181112013234/https://www.japantimes.co.jp/news/2018/07/06/national/crime-legal/profiles-top-aum-shinrikyo-members-including-six-still-death-row/#.XAkmU3QzYac#.XAkmU3QzYac |archive-date=2018-11-12 |dead-url=no }}</ref><ref name=Fnnhayakawa>{{cite web |title=“陰の司令塔”早川紀代秀死刑囚　教団武装化の中心的人物の死刑執行 |url=https://www.fnn.jp/posts/00333430HDK |website=FNN |accessdate=2019-01-10 |archive-url=https://web.archive.org/web/20180707131219/https://www.fnn.jp/posts/00333430HDK |archive-date=2018-07-07 |dead-url=no }}</ref>。2018年7月6日，早川与包括奥姆真理教教主[[麻原彰晃|麻原彰晃]]在内的7名奥姆真理教相关死刑犯被执行[[死刑|死刑]]，其中早川是在{{link-ja|福岡拘置所|福岡拘置所}}被执行死刑的<ref name="mainichi20180706">{{cite web|url=https://mainichi.jp/articles/20180706/k00/00e/040/198000c|title=「松本死刑囚ら７人死刑執行」法務省が発表|publisher=每日新闻|date=2018-07-06|access-date=2018-11-08|archive-url=https://web.archive.org/web/20180707005810/https://mainichi.jp/articles/20180706/k00/00e/040/198000c#|archive-date=2018-07-07|dead-url=no}}</ref>。

==早年==
早川紀代秀1949年在日本[[兵库县|兵库县]][[川邊郡|川邊郡]]東谷村出生，父亲是[[日本国铁|日本国铁]]职员<ref name=Furihata2002>{{cite book |author1=降幡 賢一 |title=オウム法廷〈８〉無差別テロの源流 |date=2002 |publisher=朝日文庫 |location=東京 |isbn=978-4022613691 |pages=102-109}}</ref>。在他两岁多的时候，家里搬到了[[大阪府|大阪府]][[堺市|堺市]]。早川小时候就读于与[[天主教会|天主教会]]有关系的[[幼儿园|幼儿园]]，曾在幼儿园唱[[赞美诗|赞美诗]]、参加天主教话剧表演<ref name=Hayakawa2005/>。早川高中就读于{{link-ja|大阪府立三国丘高等学校|大阪府立三国丘高等学校}}，后考入[[神户大学|神户大学]]，在[[农学|农学]]院[[园艺学|园艺学]]专业学习。在神户大学获得[[学士|学士]]学位后，早川又前往[[大阪府立大学|大阪府立大学]][[研究生院|研究生院]]学习，主攻绿地规划工程，毕业后进入一家大型[[承建商|承建商]]的土木技术部开发设计科工作。在工作了5年后，早川从这家公司辞职，又辗转在两家公司{{里}}工作<ref name=AsahiNiimi>{{cite web |title=「グル（麻原彰晃）の指示なら、人を殺すことも喜び」＜教団エリートの「罪と罰」（2）＞ |url=https://dot.asahi.com/wa/2018070600058.html?page=2 |website=週刊朝日 |accessdate=2018-12-18}}</ref><ref>{{cite book|title=現代・|url=https://books.google.com/books?id=8KNAAQAAIAAJ|year=1995|publisher=講談社|page=135}}</ref>。

==入教与罪行==
1986年，因为对[[麻原彰晃|麻原彰晃]]的浮空照片感兴趣，早川紀代秀给奥姆真理教的前身奥姆神仙会打了电话。最开始接线的职员态度很差，但之后比早川小6岁的麻原本人给早川做了诚恳的说明，早川受到感动，在当年加入了奥姆真理教。1987年，早川捐出所有财产，在奥姆真理教出家<ref name=Furihata2002/><ref name=Fnnhayakawa/>。加入奥姆真理教后，早川参与了{{link-ja|奥姆真理教男性信徒杀害事件|オウム真理教男性信者殺害事件|对一名企图脱离奥姆真理教的男性教徒的绑架和杀害}}，[[坂本堤律師滅門慘案|坂本堤律師滅門慘案]]也与他有关<ref name=JapanTimesAum1/>。此外，因为早川曾有承建商的工作经验，在购买土地等行动中，早川是负责人<ref name=Fnnhayakawa/>。1990年，随着奥姆真理教[[第39屆日本眾議院議員總選舉|在众议院选举中大败]]，麻原决定让教团执行武装夺权、颠覆日本政府的方针。早川在这一过程中扮演了核心角色，曾帮助教团设计多种武器。他频繁造访[[俄罗斯|俄罗斯]]，为笼络俄方，在他主导下，奥姆真理教向[[莫斯科物理技術學院|莫斯科物理技術學院]]赠送了高性能的[[计算机|计算机]]。在俄罗斯，早川取得了[[自动武器|自动武器]]的设计图、迷幻药[[麦角酸二乙酰胺|麦角酸二乙酰胺]]（LSD）的制作方法，甚至买到了一架[[Mi-17直昇機|Mi-17直昇機]]<ref name=AsahiNiimi/>。

==逮捕及受审==
1995年地铁沙林事件后，根据早川自己的说法，他感觉到奥姆真理教的末日很快就会到来。4月19日，早川在一次奥姆真理教的直播活动后被捕。在直播中，早川曾矢口否认奥姆真理教与坂本堤律师一家的遇害有关<ref name=Hayakawa2005/><ref name=AsahiNiimi/>。在随后的审判中，早川一共被控7项罪名。早川在审判过程中曾表达过对被害人的歉意，但也表示自己当时无法抗拒教主麻原的命令。2000年，[[东京地方法院|东京地方法院]]一审判处早川死刑，早川不服上诉。2004年，[[东京高等法院|东京高等法院]]维持原判<ref name=JapanTimesAum1/><ref name=AsahiNiimi/>。2009年，早川死刑定谳<ref name=AsahiNiimi/>。

==死刑执行==
截至2018年3月，早川与其他十二名奧姆真理教相关死刑犯一直都关押在{{link-ja|東京拘置所|東京拘置所}}。2018年3月21日，这些死刑犯被分散到到能执行死刑的{{link-ja|東京拘置所|東京拘置所}}等5處設施，其中早川被送到{{link-ja|福岡拘置所|福岡拘置所}}<ref name="オウム死刑囚7人移送 p.1">{{Cite news|url=http://www.sankei.com/affairs/news/180314/afr1803140030-n1.html|title=【オウム死刑囚】中川智正らオウム死刑囚７人を移送　執行施設ある５拘置所に分散か（1／2ページ）|newspaper=産経新聞|publisher=産業経済新聞社|date=2018-03-14|accessdate=2018-03-15|archivedate=2018-03-15|archiveurl=https://web.archive.org/web/20180315132452/http://www.sankei.com/affairs/news/180314/afr1803140030-n1.html}}</ref><ref name="移送7人判明 p.1">{{Cite news|url=http://www.sankei.com/affairs/news/180314/afr1803140037-n1.html|title=【オウム死刑囚】移送７人は新実智光、林泰男、早川紀代秀、井上嘉浩、岡崎一明、横山真人、中川智正の各死刑囚（1／2ページ）|newspaper=産経新聞|publisher=産業経済新聞社|date=2018-03-14|accessdate=2018-03-15|archivedate=2018-03-15|archiveurl=https://web.archive.org/web/20180315132717/http://www.sankei.com/affairs/news/180314/afr1803140037-n1.html}}</ref>。2018年7月6日，經時任日本[[法務大臣|法務大臣]][[上川阳子|上川阳子]]批准，早川紀代秀与奥姆真理教教主[[麻原彰晃|麻原彰晃]]，以及[[井上嘉浩|井上嘉浩]]、[[新实智光|新实智光]]、{{link-jp|土谷正实|土谷正实}}、{{link-ja|中川智正}}和{{link-ja|远藤誠一}}共7名与奥姆真理教相关的死刑犯被执行绞刑，其中早川在福岡拘置所被执行绞刑，年68岁<ref name="mainichi20180706">{{cite web|url=https://mainichi.jp/articles/20180706/k00/00e/040/198000c|title=「松本死刑囚ら７人死刑執行」法務省が発表|publisher=每日新闻|date=2018-07-06|access-date=2018-11-08|archive-url=https://web.archive.org/web/20180707005810/https://mainichi.jp/articles/20180706/k00/00e/040/198000c#|archive-date=2018-07-07|dead-url=no}}</ref>。
