'''早稻田大學'''（{{jpn|j=早稲田大学|hg=わせだだいがく|rm= Waseda Daigaku}}；[[英語|英語]]譯名：{{lang|en|Waseda University}}），简称'''早大'''、'''早稻田'''，是一所本部位於[[日本|日本]][[东京都|东京都]][[新宿区|新宿区]]的[[私立大学|私立大学]]。早稻田大學於1882年，由[[明治維新|明治維新]]時期開國元老之一，前[[內閣總理大臣|內閣總理大臣]][[大隈重信|大隈重信]]創立，建校之精神為「學問之獨立」、「學問之活用」及「成就模範之國民」<ref>[http://www.waseda.jp/top/about/work/mission　早稲田大学教旨]</ref>。

早稻田大學作為日本國內首屈一指的[[私立大學|私立大學]]，十分重視國際合作與交流事業，除校內有6個學部與11個研究科設立[[英語|英語]]課程學位外，從2009年起一直為日本國內[[留學生|留學生]]人數最多的大學<ref>[https://www.jasso.go.jp/about/statistics/intl_student_e/2018/1307071_10358.html 外国人留学生受入数の多い大学]</ref>，在[[中國|中國]]更為日本最具知名度的大學之一。<ref>{{cite web|url=http://zh.cn.nikkei.com/career/abroadstudy/21550-20160927.html|title=在日本一流大學「復活」的中國留學生|date=2016-09-27|language=zh|publisher=日本經濟新聞社|quote=尤其是早稻田大學，據説該校在中國的知名度高的驚人，其名氣遠超慶應大學和一橋大學。|author=古山和弘|location=Tokyo}}</ref>

根据2018年发表的[[QS世界大学排名|QS世界大學排行榜]]显示，早稻田大學位列全球第208名<ref>{{cite web|url=https://www.topuniversities.com/university-rankings/world-university-rankings/2019|title=QS World University Rankings 2019|accessdate=2018.11.12|author=|publisher=}}</ref>，而在[[QS亚洲大学排行榜|QS亞洲大学排行榜]]中則位列亞洲第36名<ref>{{cite web|url=https://www.topuniversities.com/university-rankings/asian-university-rankings/2019|title=QS University Rankings: Asia 2019|accessdate=2018-11-12|author=|date=|publisher=}}</ref>。同时，根据同公司发表的大学毕业生就业排名显示，早稻田大學位列全球第27名，在亚洲区排行第7，在日本排行第1。<ref>{{cite web|url=https://www.topuniversities.com/university-rankings/employability-rankings/2019|title=Graduate Employability Rankings 2016|date=2016-11-17|publisher=}}</ref>

=== 創立·東京專門學校 ===
因為「明治十四年政變」而下台的[[大隈重信|大隈重信]]、[[小野梓|小野梓]]、[[矢野文雄|矢野文雄]]共同組成[[立憲改進黨|立憲改進黨]]，並構想創立一個能夠養成立憲體制領導人才的學校，最終於1882年9月公佈，並於同年10月21日於東京府下早稲田成立東京專門學校，即今日早稻田大學的前身。
*1882年（明治15年） - 因政變而下野的原[[日本|日本]][[日本內閣總理大臣|內閣總理大臣]]（首相）[[大隈重信|大隈重信]]創立'''東京專門學校'''。設政治經濟學科、法學科、理學科、英學科。
*1886年（明治19年） - 發行「早稻田講義錄」，開始接受海外留學生。
*1899年（明治32年） - [[清朝|清朝]]留學生入學，大學預科「早稻田大學高等學院」開設。
*1900年（明治33年） - 首次向海外派遣留學生（2名）。

===早稻田大學 ===
明治三十年以後，該校藉由體制的擴建與組織的改編，於1902年經政府認可改名為早稻田大學。
*1902年（明治35年） - 改名'''早稻田大學'''。
*1902年（明治35年） - 首屆「早慶戰」在[[慶應義塾大學|慶應義塾大學]]舉辦。
*1904年（明治37年） - 由《專門學校令》成為高等教授機關的舊制專門學校。
*1913年（大正2年） - 制定「早稲田大學教旨」。
*1917年（大正6年） - 發生「早稻田騷動」。
*1920年（大正9年） - 2月5日，根據《大學令》成為舊制[[大學|大學]]。
*1921年（大正10年） - 接受12名[[女性|女性]]聽講生。
*1927年（昭和2年） -[[大隈講堂|大隈講堂]]完工
*1928年（昭和3年） - 坪内博士記念演劇博物館開館。
*1949年（昭和24年） - 學制改革成為新制大學，新設11所大學學院。
*1953年（昭和28年） - 開設[[博士|博士]]課程。
*1962年（昭和37年） - 設立戶山校區、第一・第二文學部遷移至此。
*1973年（昭和48年） - 廢除第二政治經濟学部・第二法学部・第二商学部。
*1982年（昭和57年） - 創立100週年。
*1987年（昭和62年） - 設立所澤校區。
*2007年 - 創立125週年。
*2011年 - 早大擁有11個學術院，13個學部與21個研究科（大學院）。

==建校精神 ==
「學問之獨立」、「學問之活用」、「成就模範之國民」為其建學理念，制定於1913年東京專門學校創校30周年。

== 校歌 ==
*作詞為同校文學部畢業的日本詩人[[相馬御風|相馬御風]]、作曲為同校退學的作曲家[[東儀鉄笛|東儀鉄笛]]。於創立25周年（1907年）時制定。
*此歌的別名為「都之西北」（都の西北）。是由於第一段落的開頭為「在都之西北、早稻田之森…」（都の西北早稲田の杜に）而來。

===圖書館系統===
早稻田大學圖書館系統從1882年創立至今共有藏書約536萬冊與4萬6千本定期刊物，為日本大學圖書館中藏書量第三高。早稻田大學圖書館有別於日本其他圖書館，在[[太平洋戰爭|太平洋戰爭]]中未遭遇空襲，也因此早稻田圖書館保存像是《[[塵劫記|塵劫記]]》等許多他處無法尋得的珍貴文獻史料。

=== 學術交流 ===
早稻田大學海外姊妹校分為「學校簽訂姊妹校」與「系所簽訂姊妹校」兩種，學校簽訂姊妹校共478所，系所簽訂姊妹校共370所，總共包含84個國家，遍及世界五大洲。與姊妹校之間除了定期互相派遣交換留學生及教職員之外，在學術研究發展上亦是多有合作。<ref>[http://www.waseda.jp/intl-ac/documents/partnerinstitutions.pdf 早稻田大學協定校一覽（日文/英文）]</ref>

== 知名校友 ==
{{main|早稻田大學知名校友|Category:早稻田大學校友}}

早稻田大學培養了眾多人才活躍於文學、藝術、政治、經濟等諸多領域，校友组织[[稻門会|稻門会]]与[[慶應義塾大學|慶應義塾大學]]的[[三田会|三田会]]同为日本国内最具影响力的校友会。包括[[福田康夫|福田康夫]]及[[野田佳彦|野田佳彦]]等在內共七名在[[日本首相|日本首相]]皆為早稻田大學的畢業生，是[[第二次世界大戰|第二次世界大戰]]後日本最多[[日本內閣總理大臣|內閣總理大臣]]畢業的大學。

[[早稻田大學校友列表|早大校友]]也包括多家世界知名企业领导，包括[[索尼|索尼]]公司的創辦人[[井深大|井深大]]，[[三星集團|三星集團]]的創辦人[[李秉喆|李秉喆]]，[[卡西歐|卡西歐]]公司的創辦人[[樫尾忠雄|樫尾忠雄]]，[[UNIQLO|UNIQLO]]社長[[柳井正|柳井正]]，[[本田|本田]]前社长[[福井威夫|福井威夫]]，[[任天堂|任天堂]]前社长[[山内溥|山内溥]]等。其他领域的著名校友还包括作家[[村上春樹|村上春樹]]、知名演员[[堺雅人|堺雅人]]和花样滑冰世界冠军[[羽生结弦|羽生结弦]]等。

== 與中華圈的關係 ==
早稻田大學與中國有著很深的淵源，該校最早從1896年開始招攬中國留學生，並在1900年率先開設中文課，後來更於1905年特別設立「[[清國|清國]]留學生部」。晚清時期，不少中國青年不忿國家積弱，而早大自由民權風氣濃郁，善於政治經濟學、教育學和法學等學科，故尤其受到對救國知識求知若渴的留學生所矚目，校友包括不少後來參與推翻清政府的革命派人士。<ref>{{cite web|url=https://hk.on.cc/hk/bkn/cnt/cnnews/20190220/bkn-20190220100349524-0220_00952_001.html|title=歷史新說：日本早稻田大學與中國留學生的前世今生|work=https://hk.on.cc/}}</ref>早大至今培養出眾多著名華裔校友<ref>{{cite web |url=https://dspace.wul.waseda.ac.jp/dspace/bitstream/2065/39485/1/SocioScience_19_Sun.pdf |title=存档副本 |accessdate=2014-11-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20141102060105/https://dspace.wul.waseda.ac.jp/dspace/bitstream/2065/39485/1/SocioScience_19_Sun.pdf |archivedate=2014-11-02 }}</ref>；包含[[中華民國|中華民國]]初年的政治家[[宋教仁|宋教仁]]、[[張善與|張善與]]、[[廖仲愷|廖仲愷]]，以及[[中國共產黨|中國共產黨]]的主要創建人[[陳獨秀|陳獨秀]]、[[李大釗|李大釗]]等人皆是畢業於早稻田大學。此外[[中華人民共和國|中華人民共和國]]前領導人[[江澤民|江澤民]]、[[胡錦濤|胡錦濤]]訪日時皆選擇早稻田大學作為唯一大學演講地點。<ref>{{cite web|url=http://cpc.people.com.cn/GB/64093/64103/7227893.html|title=胡主席风采为啥倾倒早稻田？--中国共产党新闻--中国共产党新闻-人民网|work=cpc.people.com.cn}}</ref>
