《'''創始吸血鬼'''》（{{lang-en|'''''The Originals'''''}}），是一部2013年10月3日於[[CW電視網|CW電視網]]開播的[[美國|美國]][[超自然|超自然科幻]][[電視|電視]]影集。作為《[[噬血Y世代|噬血Y世代]]》的[[外傳|衍生劇]]，本劇講述吸血鬼狼人混合種克勞斯·米卡森和他的家族捲入[[紐奧良|紐奧良]]的超自然政治當中<ref name="date change">{{cite web |url=http://tvbythenumbers.zap2it.com/2013/07/29/the-cw-moves-up-the-originals-premiere-to-follow-the-vampire-diaries-supernatural-gets-earlier-start/194519 |title=The CW Moves Up 'The Originals' Premiere to Follow 'The Vampire Diaries' + 'Supernatural' Gets Earlier Start |last=Kondolojy |first=Amanda |work=[[TV_by_the_Numbers|TV by the Numbers]] |date=2013-07-29 |accessdate=2013-07-29 |archive-url=https://web.archive.org/web/20130801193809/http://tvbythenumbers.zap2it.com/2013/07/29/the-cw-moves-up-the-originals-premiere-to-follow-the-vampire-diaries-supernatural-gets-earlier-start/194519/ |archive-date=2013-08-01 |dead-url=yes }}</ref>。

2017年5月10日，CW電視網續訂了系列的第五季<ref>{{cite web |url=http://deadline.com/2017/05/the-originals-izombie-renewed-the-cw-1202089237/ |title='The Originals' & 'iZombie' Renewed By The CW |last=Petski |first=Denise |website=Deadline Hollywood |date=2017-05-10 |accessdate=2017-05-10}}</ref>。在2017年7月20日，系列創作者{{link-en|茱莉‧普雷克|Julie Plec}}在[[聖地牙哥國際漫畫展|漫畫展]]之前宣布第五季將為該系列的最後一季。最終季於2018年4月18日首播<ref>{{cite web |title='The Originals' to End After Season 5 on The CW |url=https://variety.com/2017/tv/news/the-originals-season-5-final-season-1202501407/ |website=Variety |accessdate=2017-07-20 |date=2017-07-20 |first=Joe |last=Otterson}}</ref><ref>{{cite tweet |user=julieplec |number=888088396362649600 |date=2017-07-20 |title=An announcement..... #TheOriginals}}</ref><ref>{{cite web |url=http://www.thefutoncritic.com/news/2018/03/30/the-cw-midseason-schedule-update-390400/20180330cw01/ |title=The CW Midseason Schedule Update |publisher=[[The_Futon_Critic|The Futon Critic]] |date=2018-04-30 |accessdate=2018-04-30}}</ref>。

== 簡介 ==
創始吸血鬼是《[[噬血Y世代|噬血Y世代]]》的[[外傳|衍生劇]]，圍繞著三個米卡森兄妹：克勞斯（[[約瑟夫·摩根|約瑟夫·摩根]]飾）、以利亞（[[丹尼爾·吉爾斯|丹尼爾·吉利斯]]飾）和蕾貝卡（[[克莱尔·霍尔特|克萊兒·荷特]]飾）。米卡森家族被稱為「始祖」，因為他們是線存在世界上最早的吸血鬼。 尼可拉斯·「克勞斯」·米卡森是一個[[巫術|女巫]]和[[狼人|狼人]]的後裔，在《噬血Y世代》中詳細介紹了他如何追求並釋放自己的[[雜交種|混種]]天性。2013年4月25日播出的[[電視試播|後門試播集]]也透露，狼人海莉（[[菲比·托金|菲比·托金]]飾）懷有克勞斯的孩子。本劇講述自1919年以來，始祖兄弟姊妹首次返回[[新奥尔良|紐奧良]]的[[法国区|法語區]]，懷孕的海莉也定居下來。最初建造這座城市後，他們在逃離尋求報復的父親時被迫放棄了這個城市。在他們缺席的情況下，克勞斯的門徒馬塞爾（{{link-en|查爾斯・麥可・戴維斯|Charles Michael Davis}}飾）掌管著這座城市。克勞斯決定，他們必須拿下馬塞爾並回到曾經屬於他們的城市。在這樣做的同時，他們還必須保護自己的城市免於吸血鬼，狼人和女巫之間的戰爭衝擊。隨著時間的推移，新舊友誼形成，愛情無所不在，死亡圍繞著他們，而始祖們更多地了解自己家庭的過去。第一季描述了克勞斯神秘的孩子將如何出生，並談到始祖們必須一起保護她免於受到威脅。第二季講述了克勞斯和他兄弟姐妹的父母的回歸，以及一位威脅嬰兒生命的阿姨的到來。第三季表明米卡森家族面臨著一個預示著他們衰敗的預言，還有那些第一代被轉化並等待了一千年想要復仇的吸血鬼。第四季跳回前五年，當米卡森家族回到紐奧良市，不得不面對重新喚醒的古老威脅，摧毀所有眼前的一切。該系列的第五季也是最後一季是在第四季結束的七年之後，霍普現在是薩爾瓦多寄宿學校的學生，一所給年輕和有天賦的青少年就讀的超自然兒童學校。

== 開發和製作 ==
2013年1月11日CW電視網宣佈將通過[[後門試映集|後門試映集]]<ref>[http://tvfantasy.net/2013/02/08/2013-pilot-full-list-2013-01-22/10/ 小常識：什麼是「試映集」（Pilot）]</ref>的方式開發[[吸血鬼日記|吸血鬼日記]]的衍生劇《The Originals》並表示計畫在4月25日作為吸血鬼日記第四季的第二十集播出該劇的試播集<ref>[http://tvbythenumbers.zap2it.com/2013/01/11/the-cw-orders-backdoor-pilot-for-vampire-diaries-spinoff-the-originals/165059/comment-page-2/#comments The CW Orders Backdoor Pilot for 'Vampire Diaries' Spinoff, 'The Originals']</ref><ref>[http://www.thefutoncritic.com/news/2013/01/11/development-update-friday-january-11-103212/10157/ Development Update: Friday, January 11]</ref><ref>[http://tvfantasy.net/2013/01/12/shows-in-development-2013-01-12/ 1月12日新劇開發消息]</ref>，在隨後的13日的[[電視評論家協會|電視評論家協會]]冬季媒體見面會上CW電視網再次確認本片的開發計畫<ref>[http://www.thefutoncritic.com/news/2013/01/13/the-cw-at-tca-pedowitz-gives-updates-on-new-returning-projects-6310/10160/ The CW at TCA: Pedowitz Gives Updates on New, Returning Projects]</ref><ref>[http://tvfantasy.net/2013/01/14/newsletter-2013-01-14-tca-winter-cw/ 1月14日簡訊：電視評論協會冬季媒體見面會專題報導（CW）]</ref>，Julie Plec隨後表示如果本劇獲得正式預訂自己會交出吸血鬼日記的總監的大權讓他人來主管<ref>[http://tvfantasy.net/2013/01/17/newsletter-glee-big-bang-theory-originals-hannibal-carrie-diaries-and-more/ 1月17日簡訊：歡樂合唱團、生活大爆炸、初代吸血鬼、漢尼拔、卡麗日記及其他]</ref>。
