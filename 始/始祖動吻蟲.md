'''始祖動吻蟲'''（[[學名|學名]]：''{{lang|la|Eokinorhynchus}}''）是一種生存在[[寒武紀|寒武紀]]早期的海生動物。由[[中國科學院|中國科學院]][[南京地質古生物研究所|南京地質古生物研究所]]副研究員[[張華僑|張華僑]]、[[美國|美國]][[維吉尼亞理工大學|維吉尼亞理工大學]]教授[[肖書海|肖書海]]和[[長安大學|長安大學]]教授[[劉云煥|劉云煥]]等組成的聯合研究團隊於[[西元|西元]]2015年在[[四川省|川]]北和[[陝西省|陝]]南早寒武世地層中發現並命名。這些[[化石|化石]]是[[動吻動物門|動吻動物門]]類目前已知的唯一化石記錄，這項發現也為[[寒武紀生命大爆發|寒武紀生命大爆發]]新添了一個動物[[門_(生物)|門]]類。<ref>{{Cite web |url = http://www.ebiotrade.com/newsf/2015-12/2015121174927355.htm |title = 最古老的動吻動物化石——始祖動吻蟲 |date = 2015年12月2日 |language = zh-hans }}</ref>

== 描述 ==
珍稀始祖動吻蟲約2[[毫米|毫米]]長，有20個[[體節|體節]]，每個體節上布滿一圈(約20–40個)長方形的小骨板，有五對較大的具刺骨板對稱分布，[[腹部|腹部]]有一個單獨的具刺大骨板，每個大骨板橫跨2–3個體節，有較多空心的刺隨機分布在體節上。和現生種的動吻動物相比，珍稀始祖動吻蟲的骨板和刺都較大。
