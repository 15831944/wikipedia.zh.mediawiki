{{Vgname|刺客教條IV：黑旗|en=Assassin's Creed IV: Black Flag}}）是[[育碧公司|育碧公司]][[育碧蒙特利尔|蒙特利爾工作室]]開發的一款[[動作冒險遊戲|動作冒險]]歷史奇幻類遊戲。<ref name ="Kotaku preview"/>本作是《[[刺客教条系列|刺客信条]]》主系列的第六部作品兼第四款正统續作，亦是《[[刺客教條III|刺客教條III]]》的前傳。2018年3月8日提供简体中文支持。<ref>{{Cite web |url = https://www.ithome.com/html/game/350220.htm |title = 育碧《刺客信条4：黑旗》简体中文今日上线：玩家追加好评 |author = 白猫 |date = 2018-03-08 |publisher = IT之家 |language = zh-cn |accessdate = 2018-04-26 }}</ref>

== 遊戲系統 ==
界面與操作大致承繼自《[[刺客教條III|刺客教條III]]》。本作強化海戰與動態天氣系統，首次引進以[[加勒比海|加勒比海]]為版圖的開放式海上世界。主角可在海底及海島間尋找寶物、掠奪船隻，進行海上狩獵以及攻佔沿海碉堡。見習刺客招攬與管理系統被刪除，改為招募各處被英軍攻擊的海盜至旗艦「寒鴉」（The Jackdaw）以壯大船員隊伍。

本作繼續保留連殺、100%記憶同步要求和網上多人模式。

== 背景设定  ==
《刺客信條4：黑旗》時空背景將回到1715年的加勒比海，海盜在此建立起無法無天的國度，腐敗、貪婪以及兇殘手段早已司空見慣。玩家將扮演年少輕狂的叛逆刺客“愛德華·肯威”，他是海爾森的父親、也就是前作主角“康納”的祖父。這位海盜船長能駕輕就熟地使用刺客組織的袖劍以及全新的武器，包括全身上下四把燧發槍以及雙手各持西洋彎刀等戰鬥姿態。

==劇情 ==
承接前作《[[刺客教條III|刺客教條III]]》，現代刺客戴斯蒙·邁爾斯（Desmond Miles）為保護地球免受[[閃焰|閃焰]]摧毀而犧牲，其屍體被跨國製藥公局Abstergo（現代[[聖殿騎士團|聖殿騎士團]]）抽取基因樣本上載至[[雲端|雲端]]，使與之沒有血緣關係的人也能透過基因記憶追溯機器「Animus」追溯其祖先之記憶。作為一名Abstergo 旗下部門Abstergo Entertainment 之新晉員工，部門主管希望玩家能進入活躍於海盜黃金時代之刺客[[愛德華·詹姆士·肯威|愛德華·肯威]]（Edward Kenway）的記憶，為新互動電影作資料搜集（真正目的是尋找人類以前「第一文明」（First Civilization）之機器「觀測台」（The Observatory）的所在地）。

1715年，愛德華殺死了一名準備投誠聖殿騎士團之刺客，其身上信件指示前往[[哈瓦那|哈瓦那]]與總督勞雷亞諾·托勒斯（Laureano de Torres）及其秘密組織（聖殿騎士團）會面，愛德華為信上所言的報酬心動而冒充該刺客前往與眾人會面，得悉聖殿騎士欲聯同[[英國|英國]]、[[西班牙|西班牙]]和[[法國|法國]]以打擊加勒比海海盜之名尋找一名能協助它們掌握「觀測台」位置之聖人[[巴沙洛繆·羅伯茨|巴沙洛繆·羅伯茨]]（Bartholomew Roberts）。愛德華不但使當地刺客組織陷入劣勢，其後亦因身分敗露而被帶走囚於船上。愛德華逃離監禁後釋放多名海盜及奪得一艘西班牙[[雙桅橫帆船|雙桅橫帆船]]，隨後將之命名為「寒鴉號」（Jackdaw）。

1718年愛德華與一群海盜同伴（包括「黑鬍子」[[愛德華·蒂奇|愛德華·蒂奇]]（Edward Thatch）、[[瑪莉·瑞德|瑪莉·瑞德]]（Mary Read）等人）為了能活在一個不受英國控制的自由地區而聚集於[[拿骚_(巴哈马)|拿骚]]，但缺乏管理和經濟活動、疫病出現再加上其後英軍進駐拿騷向海盜進行勸降使各人之間出現分歧，愛德華雖然成功偷取疫苗但未能阻止英軍於通牒限期前發動圍剿，黑鬍子在與愛德華搶奪一艘海軍主力艦時被殺，愛德華乘亂返回寒鴉號逃離拿索水域並以[[大伊納瓜島|大伊納瓜島]]作為基地。

由於認為奪得「觀測台」可以獲得數之不盡的財富，愛德華致力追查羅伯茨的下落，亦發現瑪莉乃一名刺客並被邀前往[[圖盧姆|圖盧姆]]與刺客組織會面。由於愛德華以往對聖殿騎士的協助，組織不久受英軍（聖殿騎士）襲擊，愛德華受瑪莉所託為組織完成刺殺合約以換取報酬，其後也逐一刺殺叛變至聖殿騎士的海盜和曾會面的騎士成員。愛德華其後終在[[非洲|非洲]][[普林西比島|普林西比島]]尋獲羅伯茨，並經由羅伯茨的帶領下，在之後於[[長灣|長灣]]找到進入封存「觀測台」之第一文明密室「觀測所」的入口 －「觀測台」之第一文明密室：只要有監視目標的血液樣本，「觀測台」就能把其目視之一切以立體影像呈現。羅伯茨此時背叛愛德華並乘他傷重時把他交予英軍以獲取懸賞。

1720年4月，愛德華與各海盜（包括瑪莉、[[安妮·伯妮|安妮]]）被審判下獄等待處決，二人在刺客組織協助下逃獄但瑪莉因病重而在途中不支去世。其死亡使愛德華決心加入刺客組織，隨後擊殺羅伯茨並在「觀測台」密室內刺殺托勒斯。為免聖殿騎士獲悉「觀測所」位置，刺客組織表示密室將在下任聖人出現前被封印，同時愛德華接獲來自英國的信件：其分居妻子已去世，其女兒珍妮佛·史考特（Jennifer Scott）被安排前往與他會面。

在玩家錄取記憶資料時，Abstergo Entertainment 資訊科技經理約翰（John）暗中表示Abstergo 對自己有所隱瞞，三度給予玩家保安許可以入侵公司電腦和監視系統（所得資料被安排交予現代刺客肖恩‧海斯汀（Shuan Hastings）和麗貝卡‧克琳（Rebecca Crane）），最後指示玩家入侵Animus核心部分，完成後第一文明成員[[朱諾|朱諾]]（Juno）以立體影像現身表示由於中央密室被提早解封以拯救地球，她不能影響世界亦未能佔據玩家身心：約翰乃羅伯茨之轉世並欲使朱諾復活，計劃失敗後打算殺死玩家但及時被保安兵阻止和槍殺。

在把大伊納瓜島基地交予刺客組織後，愛德華與女兒乘坐寒鴉號返回[[倫敦|倫敦]]定居並脫離海盜生涯。多年後，愛德華已再婚並育有一子海爾森·肯威（Haytham Kenway）。

=== 《自由的吶喊》 ===
《自由的吶喊》（Freedom Cry）背景設定在主線劇情十五年後，主角為「寒鴉號」第一任大副阿德瓦勒（Adéwalé）。
   
1735年，已自海盗成為刺客之阿德瓦勒於[[西印度群島|西印度群島]]海域攔截聖殿騎士艦艇，為擺脫追擊而駛進暴風圈，被強風吹離旗艦墜海失去意識，醒來後發現自己漂流至[[法属圣多明戈|海地]][[太子港|太子港]]。當地一所知名妓院的老闆巴斯蒂娜‧約瑟夫（Bastienne Jospeh）乃當地逃亡黑奴種族馬龍人（Maroon）的接頭人，為換取奴隸自由及從當地有權勢的人獲取政治協助而為聖殿騎士團效力。阿德瓦勒與巴斯蒂娜會面後為她效力，於當地解救了一眾馬龍人後漸漸被後者信念吸引，繼以加入馬龍人反抗法國殖民政府暴政的行動，干擾持續不斷的奴隸貿易並解放了大量黑奴。   
 
1737年，阿德瓦勒和馬龍人因反對[[奴隸貿易|奴隸貿易]]而進一步襲擊殖民者，使殖民政府和奴隸間關係愈趨緊張。巴斯蒂娜指出殖民政府之首德法葉只會進一步使被困在奴隸制的人受到進一步的處罰但不獲阿德瓦勒理會，其後一首護衛艦向奴隸船開火，導致大量黑奴被淹死。儘管巴斯蒂娜認為德法葉死後仍會有人繼續欺壓黑奴，阿德瓦勒仍潛進太子港官邸刺殺了德法葉，並在離開太子港前向巴斯蒂娜他表示他將使用自己的新的信念以幫助被壓迫的人。

育碧軟件行政總裁依域斯·桂里莫特（Yves Guillemot）在2013年2月的季度財政報告中確認下一部刺客教條系列遊戲將於2014年4月前推出，遊戲將有新的主角和歷史時期，以及由新的團隊進行開發<ref>{{cite web|url=http://www.ign.com/articles/2013/02/07/next-assassins-creed-has-new-hero-time-period-developer|title=Next Assassin's Creed Has New Hero, Time Period, Developer|last=Dyer|first=Mitch|publisher=IGN|date=2013-02-07|accessdate=2013-02-07}}</ref>。2013年2月28日，育碧軟件公開遊戲首張宣傳圖片和封面設計<ref name="cover">{{cite web|url=http://kotaku.com/5987035/assassins-creed-iv-is-all-about-pirates-according-to-this-poster|title=Assassin’s Creed IV Is All About Pirates, According To This Poster|last=Schreier|first=Jason|publisher=Kotaku.com|date=2013-02-26|accessdate=2013-02-26}}</ref>，顯示遊戲名稱為《刺客教條IV：黑旗》，封面上角色名手持[[燧發槍|燧發槍]]與劍，背景則有一面帶有刺客組織徽號的黑色骷髏海盜旗幟<ref name="cover"/>。遊戲官方網頁上的錯誤顯示遊戲將在10月29日於次世代主機平台發行<ref>{{cite web|last=Philips|first=Tom|title=Assassin's Creed 4: Black Flag release date, next-gen launch leaked|url=http://www.eurogamer.net/articles/2013-03-01-assassins-creed-4-black-flag-release-date-leaked|publisher=Eurogamer}}</ref>，消息由2013年4月正式公開的首段遊戲宣傳片段證實<ref name="IGN reveal"/><ref name = "IGN Reveal Trailer">{{cite web|url=http://www.ign.com/videos/2013/03/04/debut-trailer-assassins-creed-iv|title=Assassin's Creed IV: Black Flag — Debut Trailer|publisher=IGN.com|date=2013-03-04|accessdate=2013-03-04}}</ref><ref>{{cite web|last=Mallory |first=Jordan |url=http://www.joystiq.com/2013/03/02/assassins-creed-4-black-flag-debut-trailer/ |title=Assassin's Creed 4: Black Flag debut trailer leaks [Update: It's gone! |publisher=Joystiq |date=2013-03-02 |accessdate=2013-03-13}}</ref>。遊戲開發在2011年中於育碧軟件蒙特利爾展開，位於[[阿訥西|阿訥西]]、[[布加勒斯特|布加勒斯特]]、[[基輔|基輔]]、[[蒙彼利埃|蒙彼利埃]], [[新加坡|新加坡]]和[[索菲亞|索菲亞]]的工作室也有參與開發.<ref name = "Kotaku preview"/>

遊戲評價普遍正面，大多對其開放式世界、支線任務、[[圖像|圖像]]和海戰予以讚揚。[[GameRankings|GameRankings]]和[[Metacritic|Metacritic]]對遊戲的PlayStation 3版本分別給予87.62分和88分<ref name="GRPS3"/><ref name="MCPS3"/>、Xbox 360版本分別給予85.74分和86分<ref name="GRX360"/><ref name="MCX360"/>、Wii U版本分別給予87分和86<ref name="GRWIIU"/><ref name="MCWIIU"/>、PlayStation 4版本分別給予85.31分和83分<ref name="GRPS4"/><ref name="MCPS4"/>、Xbox One版本各給予81分<ref name="GRXONE"/><!--<ref name="MCXONE"/>-->、個人電腦版本則分別給予86.67分和86<ref name="GRPC"/><ref name="MCPC"/>。

遊戲普遍被認為比前作《刺客教條III》出色<ref name="joystiq — review"/><ref name="VG"/>。[[Game_Informer|Game Informer]]成員祖·朱巴（Joe Juba）表示育碧軟件對前作的批評作出了回應和改善<ref name="GI"/>；[[電腦與電子遊戲|電腦與電子遊戲]]成員馬特·基爾曼（Matt Gilman）稱遊戲是刺客教條系列中回復應有模樣的一作<ref name="CVG"/>；[[PlayStation官方雜誌_(英國)|PlayStation官方雜誌 (英國)]]成員米高·拉柏拉斯（Mikel Reparaz）表示遊戲正是系列自《刺客教條：啟示錄》和《刺客教條III》後需要的一劑良藥<ref name="OXM"/>；[[IGN|IGN]]在遊戲發售兩天後把它列為系列中排行第二的一作，僅次於《[[刺客教條II|刺客教條II]]》<ref>{{cite web |last=Davis |first=Justin |date=2013-10-31 |url=http://au.ign.com/articles/2013/10/31/the-best-assassins-creed-games |title=The Best Assassin's Creed Games |accessdate=2013-12-10 | publisher = [[IGN|IGN]]}}</ref>。

部分評論對遊戲的開放式世界予以高度讚揚，[[Edge|Edge]]認為遊戲為刺客教條系列和開放世界遊戲定下了新的標準<ref name="Edge"/>；[[Eurogamer|Eurogamer]]成員湯姆·布拉姆威爾（Tom Bramwell）稱讚遊戲內的開放式世界為一口新鮮的海洋空氣<ref name="EuroG"/>；基爾曼表示它活化了刺客教條系列<ref name="CVG"/>；IGN也讚揚遊戲內的大規模開放式世界<ref name="IGN"/>；The Escapist成員葛里格·替圖（Greg Tito）表示遊戲開放了整個加勒比海<ref name="RevEscapist"/>。評論員也讚揚遊戲的支線任務和收集品<ref name="CVG"/>，認為它們鼓勵玩家探索世界<ref name="EuroG"/><ref name="IGN"/>；[[GameSpot|GameSpot]]成員肖恩·麥伊尼斯（Shaun McInnis）表示遊戲呈現了一個充滿冒險和機遇的世界，絕對有理由前往探索<ref name="GSpot"/>。支線任務被普遍認為比主線任務更優秀<ref name="EGM"/><ref name="GI"/>，IGN成員馬提·史拉發（Marty Sliva）表示遊戲的最大樂趣是玩家自行尋找其樂趣的時候<ref name="IGN"/>，此外刺殺任務<ref name="GI"/><ref name="PCGUK"/>、流暢的海陸探索也受到讚揚<ref name="GSpot"/><ref name="PCGUK"/>。

遊戲圖像普遍獲得讚揚。Edge表示從圖像角度而言，不論玩家使用哪一個世代的主機進行遊戲，其世界也令人驚嘆<ref name="Edge"/>，其他評論也認為遊戲世界非常美麗<ref name="CVG"/><ref name="GSpot"/>，史拉發稱《黑旗》是2013年內最美觀的其中一款遊戲<ref name="IGN"/>。遊戲中海洋、雨和水等細節也獲表揚<ref name="CVG"/><ref name="EGM"/><ref name="GI"/>。

海戰系統同獲讚揚。評論表示海戰是《刺客教條III》的最佳元素而《黑旗》對它作出了改善<ref name="Edge"/>，替圖表示《刺客教條III》的海軍任務限制較大缺乏變化，《黑旗》賦予玩家極大的自由，可進行探索、戰鬥或航行至任何想前往的地方<ref name="RevEscapist"/>；麥伊尼斯表示遊戲非常強調海戰，基於《刺客教條III》的支線海軍任務並創造了一個與在陸地上來往同樣重要的體驗<ref name="GSpot"/>。評論也對新增的海上掠奪元素和碉堡攻佔任務表示歡迎<ref name="GSpot"/><ref name="PCGUK"/><ref name="RevEscapist"/>

遊戲的潛行系統被指比前作更自由，賦予玩家更多選擇以完成目標<ref name="CVG"/><ref name="GSpot"/>。Edge表示潛行遊戲的成敗取決於面對敵人時可作出的選擇數量，《黑旗》在這方面是刺客教條系列中最慷慨的一作<ref name="Edge"/>，但有評論批評遊戲的潛行因操作問題而令人沮喪<ref name="EuroG"/><ref name="GI"/><ref name="VG"/>。評論也指出主角的戰鬥與前作相近<ref name="OPMUK"/>、過於單調<ref name="EuroG"/>和容易<ref name="CVG"/>而且缺乏差別<ref name="RevEscapist"/>，但也是一次有效和暴力的旅程<ref name="PCGUK"/>而且流暢生動<ref name="GSpot"/>。敵[[人工智能|人工智能]]被批評過分愚蠢<ref name="OPMUK"/><ref name="PCGUK"/><ref name="VG"/>，此外自《刺客教條III》引進的物品打造系統因作出了簡化而獲讚揚<ref name="GSpot"/><ref name="OXM"/>。

遊戲劇情的評論褒貶不一。拉柏拉斯讚揚故事引人入勝，是刺客教條系列中最好的一作，他與麥伊尼斯也讚揚了一眾遊戲角色和劇情探討了海盜的人性一面<ref name="GSpot"/><ref name="OXM"/>；布拉姆威爾認為劇情發展成功並對配角和主角的魅力予以讚揚<ref name="EuroG"/>；PlayStation官方雜誌成員祖兒·葛里高利（Joel Gregory）認為劇情沒有獨特之處，但讚揚角色比前作更有趣、更討人喜愛和更多變<ref name="OPMUK"/>；基爾曼讚揚前半部分劇情但對後半部分加以批評，認為其節奏和結構使玩家對角色失去興趣<ref name="CVG"/>；史拉發和朱巴也批評劇情<ref name="IGN"/>，其中朱巴認為劇情缺乏目的和一個令人信服的敵人，因此大部分配角沒有發展空間<ref name="GI"/>。

由於遊戲各部分以海盜為中心，部分評論認為《黑旗》相對刺客教條系列以往作品更像一款海盜遊戲<ref name="Edge"/><ref name="joystiq — review"/><ref name="OPMUK"/><ref name="PCGUK"/>，調子較輕鬆<ref name="Edge"/><ref name="IGN"/><ref name="VG"/>也明智地把以往曲折的劇情變得較簡單直接<ref name="IGN"/><ref name="PCGUK"/>。基爾曼表示《黑旗》身為一款海盜遊戲比其本身作為刺客教條遊戲更出色<ref name="CVG"/>；PC Gamer成員湯姆·先尼亞（Tom Senior）表示《黑旗》不大希望成為一款刺客教條遊戲，而這是明智之舉<ref name="PCGUK"/>；拉柏拉斯稱遊戲是他所玩過的海盜遊戲中最出色的一款<ref name="OXM"/>；[[Electronic_Gaming_Monthly|Electronic Gaming Monthly]]成員雷·卡斯盧（Ray Carsillo）稱《黑旗》可能是史上最出色的海盜模擬遊戲<ref name="EGM"/>；替圖表示《黑旗》的遊玩方式使他憶起3D版本的海盜遊戲Sid Meier's Pirates!多於任何一款刺客教條系列作品<ref name="RevEscapist"/>。評論對主角作為一個自我中心的海盜而非刺客也表示讚揚<ref name="EuroG"/><ref name="GSpot"/><ref name="IGN"/><ref name="VG"/>。史拉發相信主角的個性是一種清新的改變<ref name="IGN"/>，他與基爾曼也認為主角比《刺客教條III》主角康納（Connor）更生動、更合乎玩家口味和更討人喜愛<ref name="CVG"/><ref name="IGN"/>。
