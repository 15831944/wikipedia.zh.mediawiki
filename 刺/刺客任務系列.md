Dietz}}所寫的《''Hitman: Enemy Within''》和{{link-en|雷蒙·本森|Raymond Benson}}所寫的《''Hitman: Damnation''》。<ref>{{Cite web |url = https://www.goodreads.com/book/show/1302181.Hitman |title = Hitman: Enemy Within |author = William C. 2007年改編成電影《[[殺手47_(2007年電影)|殺手47]]》，雖然得到負面評價，但票房成功。另一部改編電影《[[刺客特攻47|刺客特攻47]]》在2015年上映，獲得負面評價。

== Agent 47 ==
[[殺手47|殺手47]]（Agent 47）為《刺客任務》系列之主角，一名於1964年一所[[羅馬尼亞|羅馬尼亞]]研究設施內誕生之基因改造複製人，其基因來自世上四大犯罪組織之首Lee Hong、Pablo Ochoa、Frantz Fuchs、Arkadij Jegorov以及其創造者科學家Otto Wolfgang Ort-Meyer；名字來自頭後之條碼640509-040147（首六個數字為出生日期；04為第四系列；01為級別；47為複製次序）。

47兒時與約50名第四系列複製人一同由研究設施人員撫養，接受槍械、搏擊及刺殺訓練，於30歲時發現設施內一個由Otto故意種下之保安漏洞後逃脫，不久受僱於提供刺殺與傭兵服務之秘密組織國際契約情報局（International Contract Agency）並迅速成為情報局內效能最高之殺手。

47性格內向，沈默寡言，幾乎從不在言語間流露情感，只有[[西西里|西西里]]神父域多利奧（Father Vittorio）（刺客任務2：無聲殺手角色）及其情報局聯絡人戴安娜·伯伍德（Diana Burnwood）與他有較多交流。於接受合約後47將徹底完成任務刺殺目標（即使客戶後來提出取消合約），亦會排除任何危及任務及得悉其身份之人士。

47之典型服飾為黑色西裝、黑色皮革手套、白色禮服襯衫及紅色領帶；代表武器為纖維線及兩門改裝[[AMT_Hardballer手槍|AMT Hardballer手槍]]（別名Silverballers）。

== 國際契約情報局 ==
國際契約情報局（International Contract Agency，簡稱ICA）為47之僱主，一個提供暗殺服務之跨國地下組織，活動分布於[[美洲|美洲]]、[[環太平洋|環太平洋]]地區以及[[歐亞大陸|歐亞大陸]]，有時也與[[FBI|FBI]]、[[CIA|CIA]]等機構合作。儘管無分道德與政治立場，情報局一般拒絕暗殺前客戶。

47透過其筆記型電腦與聯絡人戴安娜·伯伍德（Diana Burnwood）通訊及接受任務簡報，基於保安理由二人大多不直接會面。
