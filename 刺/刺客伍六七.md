《'''刺客伍六七'''》（{{lang-en|''Killer Seven''}}）是一部由小疯映画制作，何小疯导演的电视[[动画|动画]]。于2018年4月25日在中国大陆各大网络平台播放，总计14集。

==故事简介==
在一座小岛上，有一名叫做伍六七的刺客。他为了招揽刺杀任务而伪装成理发店理发师。
由于知名度不高，且刺杀出价低廉，经常接到一些奇葩的任务。这些任务往往会涉及到亲情、友情以及人性，而伍六七总是会用特殊的方式解决。

== 人物介绍 ==
; 伍六七
: 配音：[[何小疯|何小疯]]
: 本作男主角，刺客，拥有控制剪刀的能力，同时还能变成各种事物的能力。目前在大保健理发店当首席理发师。因不明因素丧失记忆被鸡大保捡到，“伍六七”这个名字就是鸡大保起的。在昏迷或受过度刺激时会短暂恢复记忆，口音由广普变粤语，战斗力翻倍。真实姓名不详，身份背景不详，真实身份为玄武国首席暗影刺客。目前在刺客排行榜第17369名。

; 鸡大保
: 配音：[[姜广涛|姜广涛]]
: 大保健发廊老板。原本为养鸡场的一只普通蓝羽鸡，因好友大飞斗鸡场战死，看清世界的真理后，将山鸡打败得以获得自由。在海边捡到了昏迷的伍六七，为了医好他欠下巨额债务。故为赚钱还债教唆伍六七当刺客。

; 鸡大飞
: 配音：[[陈浩|陈浩]]
: 鸡小飞父亲，是养鸡场的一只普通蓝羽鸡，因想要给小飞幸福而前往斗鸡场，战死在角斗场。临死前将小飞托付给鸡大保。
; 鸡小飞
: 配音：[[赵寒|赵寒]]
: 鸡大保养子，母亲是鸽子，因而能飞。通常是作为伍六七的逃跑工具，将爪子伸长勾在伍六七的裆部飞走。在突发事件或受刺激时会变身成浑身肌肉的“肌肉鸡”，力大无穷。
; 梅花十三
: 配音：[[段艺璇|段艺璇]]
: 本作女主角，为玄武国女刺客，目前在刺客排行榜第37名。曾刺杀斯坦国陈嘉杰博士。对伍六七有好感。
