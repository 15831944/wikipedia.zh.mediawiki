{{Vgname|刺客信条II|en=Assassin's Creed II|cn=刺客信条II|tw=刺客教條II|hk=tw}}是[[育碧公司|育碧公司]]的[[蒙特利尔|蒙特利尔]]工作室开发的一款第三人称歷史奇幻類遊戲。它支援[[Microsoft_Windows|Microsoft Windows]]、[[PlayStation_3|PlayStation 3]]、[[Xbox_360|Xbox 360]]和[[Mac_OS_X|Mac OS X]]四個平臺。它是[[育碧公司|育碧公司]]2007年遊戲《[[刺客教條_(遊戲)|刺客信条]]》的續集。2009年11月PlayStation 3和Xbox 360版本的游戏发布。Windows平台的游戏发布于2010年。另於2016年11月17日在[[PlayStation_4|PlayStation 4]]和[[Xbox_One|Xbox One]]發售與[[刺客教條：兄弟會|刺客教條：兄弟會]]和[[刺客教條：啟示錄|刺客教條：啟示錄]]三部曲合併的[[:en:Assassin's_Creed#The_Ezio_Collection_(2016)|刺客教條：埃奇歐合輯]]。<ref name="amazon.com"/><ref>http://www.amazon.co.uk/UBI-Soft-Assassins-Creed-II/dp/B001TK3DO0/ref=sr_1_3?ie=UTF8&s=videogames&qid=1266501122&sr=8-3</ref>2017年2月14日提供简体中文支持。<ref>{{Cite web |url = http://www.ubisoft.com.cn/news2/show/1301 |title = 《刺客信条2》首发简体中文 《刺客信条》电影今日全国上映 |author =  |date = 2017-02-24 |publisher = 育碧中国 |language = zh-cn |accessdate = 2017-11-04 }}</ref><ref>{{Cite web |url = http://www.sohu.com/a/127242919_282865 |title = 育碧官方中文回归计划正式启动！ |author =  |date = 2017-02-25 |publisher = 搜狐 |language = zh-cn |accessdate = 2017-08-11 }}</ref>

== 系統 ==
界面與指令大致沿用前作（詳見《[[刺客教條_(遊戲)|刺客教條]]》），任務數量與種類、可使用武器、戰鬥動作及敵兵種類均有所增加。遊戲增設金錢及買賣系統，並引入城鎮重建及商店投資設定：在修繕刺客基地城鎮中的商店後，主角能以較低價錢購買各物資，同時定期獲得商店的一定利潤。<ref name="gi-ac2"/>由於記憶追溯機器「Animus」經過改良，因此自本作開始刺客祖先能游泳和潛水。本作擁有更開放的世界觀，城市版圖更大，所有關键地標及人物均附有歷史參考資料（當中部分重要人物外貌輪廓來自真實演員）。

== 劇情 ==
緊接前作《[[刺客教條_(遊戲)|刺客教條]]》，跨國製藥公司Abstergo（現代[[聖殿騎士團|聖殿騎士團]]）已取得酒保戴斯蒙·邁爾斯（Desmond Miles）之刺客祖先－[[阿泰爾·伊本·拉阿哈德|阿泰爾·伊本·拉阿哈德]]（Altaïr ibn La-Ahad）的一段記憶並計劃將之滅口，但在僱員露西·斯蒂爾曼（Lucy Stillman，本為刺客但因長期與組織隔離已暗地轉投聖殿騎士團）協助下二人逃到現代刺客組織之一個藏身之所，與歷史學家肖恩·海斯汀（Shuan Hastings）及電腦專家麗貝卡·克琳（Rebecca Crane）會合後，三人隨即著戴斯蒙使用功能更強之「Animus 2.0」（基因記憶追溯機器「Animus」的強化版）以進入其另一刺客祖先的記憶，希望籍由長期進入Animus 而帶來之「血緣效果」（Bleeding Effect），於短時間內學會其刺客祖先的所有技能以對抗勢力強大之現代聖殿騎士團。

1476年[[文藝復興|文藝復興]]時期，17歲的[[埃奇歐·奧狄托雷·迪·佛羅倫斯|埃奇歐·奧狄托雷]]（Ezio Auditore da Firenze）是意大利[[佛羅倫斯|佛羅倫斯]]之[[貴族|貴族]]，其家族因一場政治陰謀而受牽連，父親（銀行家，真實身份實為刺客）、兄長以及弟弟被判以叛國罪名而被處以[[絞刑|絞刑]]。埃奇歐帶著母親及妹妹逃離佛羅倫斯至[[蒙特里久尼|蒙特里久尼]]投靠伯父馬力歐·奧狄托雷（Mario Auditore），為報家族之仇及保護家人而決定跟隨伯父學習戰鬥技能並繼承父親遺志－為佛羅倫斯統治者[[洛倫佐·德·美第奇|洛倫佐·德·美第奇]]（Lorenzo de Medici）效命及刺殺策劃是次陰謀之[[帕齊家族|帕齊家族]]成員。

在年輕發明家[[李奧納多·達·文西|李奧納多·達·文西]]（Leonardo da Vinci）及其他市民協助下，埃奇歐在往後20年間把相關成員逐一刺殺，並了解眾人之首為聖殿騎士團團長羅德里哥·波吉亞（Rodrigo Borgia），目的為推翻美第奇家族以佔領佛羅倫斯，從而遂漸統領意大利。埃奇歐前往[[威尼斯|威尼斯]]進行調查後發現波吉亞已獲得另外一顆古代遺物「[[伊甸|伊甸]]碎片」，並欲成為[[教宗|教宗]]以獲得[[教宗十字|教宗十字]]（伊甸碎片「聖杖」（The Staff））以開啟位於[[羅馬|羅馬]][[西斯廷小堂|西斯廷小堂]][[聖座|聖座]]下之「地窖」（The Vault）。埃奇歐在伯父及刺客組織成員協助下奪得碎片，並於1499年前往羅馬刺殺已成為教宗[[亞歷山大六世|亞歷山大六世]]之波吉亞。

二人各以其「伊甸碎片」戰鬥，波吉亞奪回埃奇歐手上之碎片並逃至「地窖」但無法將之開啟，埃奇歐追至後二人以徒手搏鬥方式再次進行戰鬥，波吉亞於被擊敗倒地後發現自己從來沒有資格進入「地窖」而求一死，但埃奇歐認為即使將之刺殺也無法使家人復生而拒絕。以「聖杖」及「伊甸碎片」之力量埃奇歐成功開啟「地窖」，此時自稱為[[彌涅耳瓦|彌涅耳瓦]]（Minerva）之立體影像啟動並道出真相及預言：彌涅耳瓦與其他神明皆屬於一個人類以前的先進文明（Those Who Came Before，又稱「第一文明」First Civilization），地球當年因一場災難導致大部分生命被摧毀，彌涅耳瓦與同伴為使地球復蘇而以自己的形象創造了人類並籍由「伊甸碎片」控制思想。雙方其後發生了戰爭，因人類數目遠多於其創造者而勝出，彌涅耳瓦與其倖存同類為避免相同之災難再次發生而把自己封存於世界各地之神殿內。彌涅耳瓦表示往後一切將取決於戴斯蒙的行動，影像至此消失。

此時Abstergo 已追至刺客組織的藏身之所。利用「血緣效果」之功效戴斯蒙成功與露西合力擊敗Abstergo 的衛兵。四人驅車移至另一藏身之所的同時嘗試推敲彌涅耳瓦預言的內容。

戴斯蒙、埃奇歐及波吉亞之事跡於後繼作品《[[刺客教條：兄弟會|刺客教條：兄弟會]]》續有敍述。
