《'''徬徨之刃'''》是[[日本|日本]]作家[[東野圭吾|東野圭吾]]的[[長篇小說|長篇]][[推理小說|推理小說]]。於『[[週刊朝日|週刊朝日]]』連載，2004年「[[朝日新聞社|朝日新聞社]]」集結成[[單行本|單行本]]，[[文庫本|文庫本]]由「角川書店」出版。<br />
由[[東映|東映]]發行、{{link-ja|益子昌一|益子昌一}}導演、[[寺尾聰|寺尾聰]]主演的電影在2009年上映。

本作探討了青少年犯罪的社會問題，而身為執法者的利刃究竟該指向犯罪的青少年，亦或偏激的受害者家屬？在兩者間陷入苦惱至於徬徨的執法者，該如何維持社會正義？

== 故事簡介 ==
公司職員長峰重樹的獨生女繪摩遭人姦殺，長峰悲痛欲絕。幾天後，長峰接到一個告密電話，得悉了兇手的名字和住所。猶豫之下長峰決定依照電話所言前往兇手的住處。在潛入無人的屋裡搜查時，長峰發現了幾支[[錄像帶|錄像帶]]，裡面竟然記錄了女兒被兩個兇手恣意凌辱的場面。悲憤難抑的長峰，將偶然回到家的其中一名兇手伴崎敦也虐殺，並且從臨死的伴崎口中得知另一名兇手菅野快兒的匿藏地方。

[[少年犯罪|少年犯罪]]者們悲痛地慘叫，正義到底是什麼，誰會將兇手制裁？令人意想不到的結局在等待著被揭開。

: 得知女兒被害的真相後，發誓為女兒報仇。在殺害兇手之一的伴崎敦也，並從他口中得知另一個兇手菅野快兒逃到了[[長野縣|長野縣]]的[[民宿|民宿]]旅館後，帶上了以前射擊時用的[[雷明登武器公司|雷明登]]步槍，冒著被通緝的危險追踪菅野，並化名「吉川武雄」入住民宿「Crescent」伺機行動。

: 長峰的獨生女，15歲，高中一年級學生。10歲時母親過世，之後一直和父親相依為命。從煙花大會回家的時候被快兒和敦也襲擊，[[輪姦|輪姦]]後被殺。

; {{nihongo|中井 誠|{{ruby|中井|なかい}} {{ruby|誠|まこと}}|Nakai Makoto}}
: 敦也和快兒的同學，跟他們一起高中輟學後依然來往頻密。因害怕背叛快兒他們遭到報復而把車子借給他們犯案，案發後又被快兒強迫製造不在場證明。因怯於警察的搜查而給長峰打了告密電話，供出了快兒和敦也。

; {{nihongo|伴崎 敦也|{{ruby|伴崎|ともざき}} {{ruby|敦也|あつや}}|Tomozaki Atsuya}}

; {{nihongo|菅野 快兒|{{ruby|菅野|すがの}} {{ruby|快児|かいじ}}|Sugano Kaiji}}

: 警視廳捜査一課的刑警。久塚班班長。

; {{nihongo|丹澤 和佳子|{{ruby|丹沢|たんざわ}} {{ruby|和佳子|わかこ}}|Tanzawa Wakako}}
