'''开罗'''（{{lang-ar|'''القـــاهــرة'''|al-Qāhira}}）是[[埃及|埃及]]首都。

開羅在[[古埃及|古埃及]]時期稱[[優努|優努]]（古埃及語：'''ỉwnw'''，拉丁化：lunu，意為「通道」）或[[安努|安努]]（Anu），聖經中稱作[[安|安]]（On）、[[赫利奧波利斯|赫利奧波利斯]]（{{lang-gr|'''Ἡλιούπολις'''}}，意為「太陽神之城」）、[[米斯爾|米斯爾]]（{{lang-ar|'''مصر'''}}，另作{{transl|ar|DIN|Miṣr}}，意為「軍營」，後來成為整個埃及的自稱），後改稱[[福斯塔特|福斯塔特]]（{{lang-ar|'''الفسطاط'''}}, ，意為「帳棚」），868年[[突倫王朝|突倫王朝]][[埃米爾|埃米爾]][[艾哈邁德·伊本·突倫|艾哈邁德·伊本·突倫]]於福斯塔特北部另建新都稱作[[加塔伊|加塔伊]]（{{lang-ar|'''القطائـع'''}}），到969年[[法蒂瑪王朝|法蒂瑪王朝]][[哈里發|哈里發]][[穆伊茲|穆伊茲]]派部將[[喬海爾|喬海爾]]征服[[阿拔斯王朝|阿拔斯王朝]]統治下的埃及，973年遷都於此，把新首都稱作開羅（al-Qāhira，意為「勝利之都」）為開羅名稱之始。

开罗横跨[[尼罗河|尼罗河]]，气魄雄伟，风貌壮观，是整个[[中东地区|中东地区]]的政治、经济和商业中心。它由[[开罗省|开罗省]]、[[吉萨省|吉萨省]]和[[盖勒尤卜省|盖勒尤卜省]]组成，通称{{link-en|大开罗|Greater Cairo}}。大开罗是埃及和[[阿拉伯世界|阿拉伯世界]]以及[[非洲|非洲]]最大的城市，人口一千多萬，也是世界上最古老的城市之一，是世界[[大都會區列表_(依人口排列)|第十六大都會區]]。<ref name=xinhua>{{cite web|title=千塔之城——埃及首都开罗|url=http://news.xinhuanet.com/ziliao/2002-04/02/content_350464.htm|publisher=新华网|accessdate=2013-10-12}}</ref>现在的开罗地区，阿拉伯[[穆斯林|穆斯林]]占多数，[[科普特人|科普特]][[基督徒|基督徒]]也占有较大比例。

在它的西南约30公里，開羅以南著名渡假城市[[赫勒萬省|赫勒萬省]]赫勒萬市之西岸吉薩省[[拜德爾舍因|拜德爾舍因]]（Al Badrashin）[[拉希納村|拉希納村]]（Mit Rahina），是古都[[孟菲斯|孟菲斯]]遗址（所以孟菲斯實際上並不位於開羅）。

由孟菲斯遗址北行约20公里，即[[世界七大奇观|世界七大奇观]]之一的[[金字塔|金字塔]]。

== 歷史 ==
{{hiero|優努 (Iunu)|<hiero>iwn-nw:O49 ! 或 O28</hiero>|align=left|era=egypt}}開羅地區的歷史，可追溯到公元前约3100年以前的[[前王朝時期|前王朝時期]]，當時位於今日開羅所在地的[[優努|優努]]（Iunu）（希臘化後作[[赫利奧波利斯|赫利奧波利斯]]（Heliopolis））已為[[下埃及|下埃及]]第十三[[諾姆|諾姆]]（nome）[[赫克瓦特|赫克瓦特]]（Heq-At）的首都，及後[[優努|優努]]（Iunu）雖不再為首都，但仍作為太陽神祭祀中心直至希臘化時代，到[[托勒密王朝|托勒密王朝]]中期除了[[祭司|祭司]]以外，大多數市民已離開了，不久赫利奧波利斯也告荒廢，只留下[[方尖碑|方尖碑]]見證她的歷史。[[File:Obelisk-SesostrisI-Heliopolis.JPG|thumb]]另外到了[[古王国时期|古王国时期]]，位於今開羅西南方三十公里，當時[[上下埃及|上下埃及]]交界，[[尼羅河|尼羅河]]西岸原為下埃及第一諾姆{{link-en|阿納赫奇|Aneb Hetch}}）首都的孟菲斯成为了統一的上下埃及[[王國|王國]]首都，直至[[中王國|中王國]]遷都[[底比斯|底比斯]]前，這裡一直作為王國的政治及宗教中心，爾後孟菲斯雖不是首都，仍為重要城市直至西元七世紀阿拉伯人征服[[埃及|埃及]]為止。
到西元642年，[[伊斯蘭|伊斯蘭]][[軍隊|軍隊]]拆毀了孟菲斯城（孟斐斯因而遭到毀滅性的破壞變成了廢墟）把建築材料運到今開羅南部將其建設為一座軍營城市─[[米斯爾|米斯爾]]（Misr），後改稱[[福斯塔特|福斯塔特]]（帳棚）（阿拉伯語：الفسطاط, al-Fusţāţ）。868年突倫王朝埃米爾艾哈邁德·伊本·突倫於福斯塔特北部另建新都稱作加塔伊（阿拉伯語：القطائـع, Al-Qatta'i），905年，阿巴斯王朝的軍隊攻陷加塔伊，伊本·圖倫的子孫被解往巴格達，圖倫王朝滅亡。到969年[[法蒂瑪王朝|法蒂瑪王朝]][[哈里發|哈里發]][[穆伊茲|穆伊茲]]派部將[[喬海爾|喬海爾]]征服[[阿拔斯王朝|阿拔斯王朝]]統治下的埃及，973年遷都於此，把新首都稱作開羅（al-Qāhira 勝利之都）為開羅名稱之始。往後這裡經[[法蒂瑪王朝|法蒂瑪]]、[[阿尤布王朝|阿尤布]]、[[馬木留克王朝|馬木留克]]等幾個以此為首都的王朝全力發展（同時也使開羅四週被他們視作[[異教徒|異教徒]]的建築受到進一步破壞），才把開羅發展為埃及甚至[[中東|中東]]地區的一個中心城市，到了1517年馬木留克王朝被[[奧圖曼帝國|奧圖曼帝國]][[蘇丹|蘇丹]][[塞利姆一世|塞利姆一世]]打敗，開羅淪為奧圖曼帝國[[埃及省|埃及省]]之省會，開始衰落，直到近代才得到恢復。

== 經濟  ==
除了特有的農作物種植如[[洋蔥|洋蔥]]、[[芝麻|芝麻]]之外，這裡的[[棉布|棉布]]等[[紡織品|紡織品]]貿易業也很繁榮。

== 交通 ==
{{see|開羅－達卡高速公路}}
開羅市的道路網絡極大極廣泛，道路除貫穿整個開羅外，亦連接附近的[[衛星城市|衛星城市]]和鄉村。市內路面公共交通有[[出租車|出租車]]和由私人公司營運的[[公共汽車|公共汽車]]。

[[交通擠塞|交通擠塞]]是開羅市的重大交通難題，市內繁忙街道的車輛移動速率極低，但開羅的司機普遍很忍讓。

開羅市內設有[[開羅地鐵|開羅地鐵]]，2條路線以廿字型橫過市中心。通往機場的3號線則是建造中。在繁忙時間，地鐵會變得非常擠擁。

== 氣候 ==
{{wide image|Panocairo1.jpg|900px|尼罗河全景图，[[开罗塔|开罗塔]]在图片中间，两座主要桥梁在左右两边的远端。}}
开罗以及沿尼罗河峡谷地区的气候是热带[[沙漠气候|沙漠气候]]（即[[柯本气候分类法|柯本气候分类法]]中的''BWh''<ref>{{cite web|url=http://koeppen-geiger.vu-wien.ac.at/pics/kottek_et_al_2006.gif |title=World Map of Köppen-Geiger Climate Classification |publisher=Köppen-Geiger |date=|accessdate=22 January 2010}}</ref>）。但是峡谷效应使得该地区的湿度经常很高。该地风暴也比较频繁，每年三、四月风将大量[[撒哈拉沙漠|撒哈拉沙漠]]的沙尘吹进城市（参见[[坎辛風|坎辛風]]。冬季普遍温和，最冷月平均气温范围在9°C至19°C之间，最低温偶尔降至0°C以下。夏季长，较为炎热，但气温较许多沙漠气候地区为低（最热月平均气温不超过28°C），40°C以上的炎热天气也并不常见。开罗降水极少，全年平均仅25毫米，绝大部分集中在11月至次年3月之间；但有时也会有突发的强降雨，并因此引发严重的洪水。 
2013年12月13日开罗降下大雪，这是该地100多年来首次降雪。
