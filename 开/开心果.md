'''阿月浑子'''（[[学名|学名]]：{{lang|la|''Pistacia vera''}}，{{lang-en|Pistachios}}），是一种常见[[乾果|乾果]]，是[[漆树科|漆树科]][[黄连木属|黄连木属]]的植物。开心果营养丰富，味道可口，是很受欢迎的零食。最早的商品名是'''開口笑'''，形容其硬殼裂開如笑容之狀，但迅速被更口語化的'''开心果'''取代。富含[[維生素A|維生素A]]、[[維生素E|維生素E]]。
[[File:ARS_pistachio.jpg|right]]

== 形态 ==
落叶小[[乔木|乔木]]。奇数羽状[[复叶|复叶]]互生，全缘革质卵形小叶。春季开褐绿色花，雌雄异株，腋生总状或圆锥花序。夏秋果熟，卵形或长椭圆形[[核果|核果]]，淡黄色或淡红色，有皱纹。淡绿色或乳黄色种子。

== 历史 ==
现今的阿月浑子最早在[[西亚|西亚]]被培育。在其栽培技术从[[伊朗|伊朗]]中部流传至[[地中海|地中海]]地区之前已长期作为一种重要作物。''Anthimus''在6世纪早期的一份手稿《''De observatione ciborum''》显示阿月浑子在[[罗马|罗马]]时代的晚期就已经广为欧洲人所知。

近年来有更多的阿月浑子在世界被商业化栽培，比如[[澳大利亚|澳大利亚]]，[[新墨西哥|新墨西哥]]和[[加利福尼亚|加利福尼亚]]。[[美国农业部|美国农业部]]的大卫·费尔乔德在1904年将阿月浑子引入加利福尼亚，但直到1929年才作为经济作物推广。

== 分布 ==
原产於[[伊朗|伊朗]]，分布在[[伊朗|伊朗]]、[[伊拉克|伊拉克]]、[[俄罗斯|俄罗斯]]、[[叙利亚|叙利亚]]以及[[中国|中国]]的[[新疆|新疆]]等地，目前已广泛栽培。
