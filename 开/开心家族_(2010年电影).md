《'''開心鬼上身'''》（{{lang-ko|헬로우 고스트}}，{{lang-en|''Hello Ghost''}}）是一部於2010年12月22日在[[韩国|韩国]]上映的喜剧电影。

== 剧情介紹 ==
尚万是一个无亲无靠的青年，被公司裁员后，自杀不成，还被「好色爷爷」、「烟鬼大叔」、「吃货小鬼」、「爱哭大婶」四个亡灵附体，化身四位「开心鬼」，尚万辗转于五个人之间，出尽洋相笑料百出。<ref>{{cite web |url= http://roll.sohu.com/20111208/n328269276.shtml|title=车太贤携“开心家族”现身 |accessdate=2011年12月8日 |work=|publisher= 搜狐网|language= zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/f/2011-11-28/03303492951.shtml |title= 车太贤《开心家族》一人分饰五角搞笑又催泪|accessdate= 2011年11月28日 |work=|publisher=新浪网 |language= zh-hans}}</ref>

== 主题 ==
该片导演[[金英卓|金英卓]]说，该片诠释一个生活真谛：“当我们心生绝望，孤单无助的时候，其实关心我们的人，一直都在。现实中的失意或打击会让人灰心沮丧意志消沉，人们要相信，在自己孤独的背后，那些爱着我们的人，他们一直都站在我们看不到的地方，希望《开心家族》能够给观众们带来轻松的笑声和温暖的希望，还有坚强生活信念的力量。”<ref>{{cite web |url=http://ent.sina.com.cn/r/i/2011-11-08/11243475339.shtml |title=《开心家族》：你不会独行 爱一直都在 |accessdate= 2011年11月8日 |work=|publisher=新浪网 |language= zh-hans}}</ref>
