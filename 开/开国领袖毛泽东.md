《'''开国领袖毛泽东'''》，1999年拍摄并且同年上映的电视剧，讲述中华人民共和国建国初期的史实。导演[[杨光远|杨光远]]、[[王冀邢|王冀邢]]，主演唐国强、刘劲、孙飞虎、徐光明、郭连文、王伍福。该剧改编自王朝柱作品《开国领袖毛泽东》。

==剧情==
1949年，[[中国共产党|中国共产党]]召开了“[[中国共产党第七届中央委员会第二次全体会议|中国共产党第七届中央委员会第二次全体会议]]”，不久之后，[[中国国民党|中国国民党]]领导的[[国民政府|国民政府]]在[[中国大陆|中国大陆]]政权被中国共产党取得，同年10月1日，[[中华人民共和国|中华人民共和国]]建立，中华人民共和国建国初期，经过了一系列的挫折和弯路，直到[[邓小平|邓小平]][[改革开放|改革开放]]。

==花絮==
该剧是中华人民共和国建国50周年的献礼剧。<ref name="cat " >{{cite web |url= http://www.bj.xinhuanet.com/jnmed/dy-14.htm|title=《开国领袖毛泽东》 |accessdate= 1999|publisher= 新华网|work= |language =zh-cn}}</ref>

该剧1999年2月底在北京市开始拍摄，在[[北京市|北京市]]、[[哈尔滨市|哈尔滨市]]、[[天津市|天津市]]、[[上海市|上海市]]、[[宁波市|宁波市]]、[[莫斯科|莫斯科]]、[[圣彼得堡|圣彼得堡]]等地取景。<ref name="cat " />

出演毛泽东的唐国强因为不会讲[[湖南话|湖南话]]而引发争议。<ref>{{cite web |url=http://ent.sina.com.cn/c_film/1999-12-24/2312.html |title=《开国领袖毛泽东》唐国强演主席有争议 |accessdate=1999 |publisher=新浪网 |work= |language =zh-cn}}</ref>
