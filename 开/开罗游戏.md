'''開羅遊戲''' （{{lang-ja|株式会社カイロソフト}}；{{lang-en|Kairosoft Co.,Ltd}}）是一家日本手機遊戲和電腦遊戲的開發商，總部設在東京。曾獲英國手機遊戲網站Pocket Gamer評選為「top 50 developers of 2011」第四十九名<ref>{{Cite web |url = http://www.pocketgamer.biz/list/5208/top-50-developer-2011/entry/49/ |title = Top 50 developers of 2011: #49 Kairosoft |author = Jon Jordan |date = 2011年2月21日 |publisher = The PocketGamer.biz |language = en |accessdate = 2015年4月16日}}</ref>和「top 50 developers of 2012」第三十名<ref>
{{Cite web |url = http://www.pocketgamer.biz/list/5209/top-50-developer-2012/entry/30/ |title = Top 50 developers of 2012: #30 Kairosoft |author = Jon Jordan |date = 2012年2月23日 |publisher = The PocketGamer.biz |language = en |accessdate = 2015年4月16日}}</ref>。專長是設計[[點陣圖|點陣圖]]畫風的模擬經營類型遊戲，缺點是玩法與系統完全相同，只是換了不同主題與建物而已。

'''開羅'''一名並非官方的中文名稱，而是華文圈愛好者對該公司的音譯稱呼。2017年6月，北京识君互娱网络有限公司与开罗游戏达成合作，获得中国区授权<ref>[http://kairogame.hitoky.com/index.php?s=/Article/news/type/about.html 关于我们]，北京识君网络有限公司，2017-10-20</ref><ref>[http://youxiputao.com/articles/12415 “像素游戏大师”开罗游戏终入华，发行商为黑翼集群]，游戏葡萄，2017-07-18 </ref><ref>[http://www.sohu.com/a/159153740_648730 【专访】开罗君大揭秘！你见过真实世界的《游戏发展国》吗？]，手游龙虎豹，2017-07-22</ref>。

== 簡介 ==
公司於2007年9月正式成立，不過早在1996年，公司的創辦人已經開始製作並在雜誌上發佈遊戲。公司最先推出以電腦為平台的遊戲，後來才將遊戲發佈平台由電腦轉變為手機。

現時公司主要是開發一些針對日本市場的手機遊戲，起初遊戲開發的對象包括電訊供應商SoftBank、[[Au_(行動電話)|au]]、[[NTT_DOCOMO|NTT DOCOMO]]等旗下手機及可運作[[JAVA|JAVA]]的手機，而隨著[[iOS|iOS]]和[[Android操作系统|Android操作系统]]興起，公司開始將部份遊戲移植到其他手機作業系統。另外，針對全球各地玩家，公司將部份遊戲，由日本翻譯成英文，推出遊戲的英文版本，供玩家下載。

公司曾經推出實物商品<ref>{{Cite web |url = http://app.famitsu.com/20131111_264632/ |title = みんな待ってた!?　カイロソフトのマスコットキャラ“カイロくん”が初のグッズ化 |date = 2013年11月11日 |publisher = ファミ通.com |language = ja |accessdate = 2015年4月16日}}</ref>。

== 遊戲特點 ==
遊戲類型大多是模擬經營或模擬類，視點採用俯視視角，並採用像素風格。

有不少角色經常以普通角色、隱藏角色或特別角色，跨作品登場。

另外，大部份遊戲都是沒有結局的，而且不會因為完成收集道具、達到某些條件或任務失敗而導致遊戲結束，就算金錢值出現負數，遊戲仍然會繼續，所以理論上可以永遠玩下去。不過，遊戲會在指定時間後計算積分，因此會有最高分。

==主要角色==
下列角色經常在各遊戲中登場，其中中文名稱只是中文圈的同好稱呼。
*開羅君（カイロくん）：官方的吉祥物。根據官方宣稱，開羅君是程式設計師兼設計師，經常故障，總數有20000隻。正式名稱是「開羅君」而非「開羅」，因此遊戲中被稱為「開羅君先生」。在各遊戲中多數充當隱藏角色，能力比其他的角色要強得多，但是在一些遊戲中是量產型的消耗品（再見開羅君、開羅之星）。
*開羅姬（ソフ子姫）：據稱是開羅君的妹妹，又有一說是開羅君的女朋友。興趣是購物和裁縫，通常在遊戲中以隱藏角色登場，是一個比開羅君還要強力的角色，但是出現條件比開羅君還要苛刻。
*歪羅君（ワイロくん）︰一般以反角登場，官方宣稱犯下的罪行包括有︰任意的吃光冰箱裡的布丁、橫穿馬路也不舉手還旁若無人的雙臂亂舞啊什麼的。有一天被偶然來到這個星球的開羅君撞進了醫院（再見開羅君），因此長期沒有在遊戲中登場。後來終於出院了，但卻在為不能輕鬆的量產自己而苦惱，最終多虧了在地球上吃到的美食終於量產成功（吃貨大食堂），從而再次站在了開羅君的面前。在部份遊戲中以普通角色登場（開羅之星），不全然是反角。其具體年齡不明，但是好像是個「中年人」。說話非常客氣，「請～」是其口癖。
*秘山秘書子（秘山ひしょ子）︰管理網站和廣告活動的代言人，興趣是海外旅行，遊戲中就如其名是從事秘書工作的。
*美奈子︰和秘書子一樣，在遊戲中從事秘書工作。姓氏隨遊戲而變。　
*北上川羊︰和秘書子一樣在遊戲中從事秘書或管家工作，姓名有時隨遊戲而更改。
*南瓜商人：在多數遊戲中以道具商人登場，在不同的作品會有不同的名字，在部份作品會以可操作角色（冠軍足球物語）或其他職業（時尚洋品店）登場。
*猩猩G（チンパンＧ）︰官方宣稱，牠是統領圖像部門和音樂部門的高人，偶爾流露出溫柔的一面，喜歡香蕉和橘子。遊戲中因為工作很累所以想靠版權生活（漫畫道場），不能很好的使用筆，雖然想要成為歌手（百萬進行曲），但是在營業之前就被溫泉街的魅力征服，還下落不明（暖暖溫泉鄉），在溫泉街販賣的雜誌中看到他想要過百貨商店的正常生活（百貨商店）。和開羅君一樣，大多是作為隱藏角色登場，能力時強時弱。有女性版的「猩猩子」和幼年版的「小猩猩」（チンパン少年）。
*森野熊（森野クマックス）︰多數以「森野工程公司」或「森野建築公司」的社長登場，有時與開羅君和猩猩G一樣，以隱藏角色登場。
*阿普杜拉國王（アブファラ国王）︰一位喜歡遊戲的中東石油王。在部份遊戲中作為隱藏角色登場。到日本的時候就被日本的遊戲深深的吸引住，然後成為了遊戲公司的職員（遊戲發展國），石油價格暴跌資金鏈惡化時還兼職過老師（名門口袋學院1/2）。也有沒有名字就作為一個叫做「石油王」的普通角色登場（吃貨大食堂）。　
*X先生（七色レスラー、Mr. X）︰以摔跤手形象示人。名字和身份經常不同，時而是公司職員，時而是教師，時而是國王，有時是強力角色，有時又只是路邊的小角色，也有當過隱藏角色。
*桃山桃子（桃山ももこ）︰幾乎在所有遊戲中都是以「偶像桃子醬」登場的。也作為一般角色登場過，名字基本都相同。　
*聖誕子（サンタ子）︰外表像是聖誕老人的女孩子。姓名因作品而不同。
*宇佐美耳子（宇佐美みみこ）︰戴著兔耳頭飾的女子，也曾戴著短短的「貓耳」登場。
*莫奇朴恩博士（モチポン博士）︰通常在遊戲中扮演反面角色。為了推行減肥食品，將某個星球中的食物中混入了肥胖藥（開羅之星），開羅之星中登場的怪物好像全部都是他製造的，每次輸給開羅君就會改造自己（理由是走起路來很麻煩），最終改邪歸正了（純潔的莫奇朴恩）並在地球開始了環保活動。名字基本相同。
*鴨田龍子（アヒル田カモ子）︰一隻鴨子，職業是游泳選手。大部份遊戲中還會有其他種類動物登場，例如貓、雞、豬、熊貓、狗、兔子、企鵝等，有時會有名字，有時則無。
*營養波庫（メタボック）︰雜魚角色，怪物里幾乎都有他的身影。口癖是「～咩啵」。名字會改變。
*章魚星人（マダコ星人）︰宇宙人，曾以章魚皇帝登場過，但是和其他的看起來沒什麼區別。
*火腿公主（ハム姫）︰2011年6月才登場的新角色。在火腿公主旁邊，通常都有一些火腿製作的食物，經常斜視著食物流口水。
