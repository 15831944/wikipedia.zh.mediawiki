'''开放Web基金会'''（'''Open Web Foundation'''，缩写OWF）是一个致力于开发和保护新兴网络技术规范的非营利组织。该基金会遵循类似[[Apache软件基金会|Apache软件基金会]]的开源模式。参与者包括Apache副总裁兼董事会成员{{tsl|en|Geir Magnusson|Geir Magnusson}}、[[歐萊禮|歐萊禮]]CEO[[提姆·奥莱理|提姆·奥莱理]]等。

== 历史 ==
开放Web基金会于2008年7月24日在{{tsl|en|O'Reilly Open Source Convention||O'Reilly开源公约（OSCON）}}上公布。Facebook、Google、MySpace、Six Apart、Plaxo等已宣布支持OWF。通过开放Web基金会，Google和Facebook现在有了一个适当的场所来解决{{tsl|en|Facebook Connect|Facebook联系人}}与[[OpenSocial|OpenSocial]]平台以及用户交互标准方式之间的差异。开放Web基金会也提供了有关这些协议和新兴技术如何相互作用的技术细节以及政策细节。<ref>{{Cite web|url=http://openwebfoundation.org/2008/07/announcing-the-open-web-foundation.html|title=Announcing the Open Web Foundation|publisher=Open Web Foundation|deadurl=yes|archiveurl=https://web.archive.org/web/20080725074423/http://openwebfoundation.org/2008/07/announcing-the-open-web-foundation.html|archivedate=2008-07-25}}</ref><ref>{{Cite web|url=http://www.eweek.com/c/a/Linux-and-Open-Source/Open-Web-Foundation-to-Keep-Data-Open/|title=Open Web Foundation to Keep Data 'Open'|publisher=eWeek}}</ref><ref>{{Cite web|url=http://news.cnet.com/8301-17939_109-9998776-2.html|title=Open Web Foundation a new club for standards makers|publisher=CNET}}</ref><ref>{{Cite news|url=http://www.washingtonpost.com/wp-dyn/content/article/2008/07/24/AR2008072401922.html|title=Open Web Foundation Officially Launches|last=Cubrilovic|first=Nik|date=July 24, 2008|publisher=Washington Post|accessdate=May 20, 2010}}</ref><ref>{{Cite web|url=http://www.thestandard.com/news/2008/07/25/open-web-foundation-formed|title=Open Web Foundation formed|publisher=The Industry Standard}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
