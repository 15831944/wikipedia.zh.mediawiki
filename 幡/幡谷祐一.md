'''幡谷祐一'''（{{nihongo|幡谷 祐一|はたや　ゆういち}}，{{bd|1923年||2018年|1月11日}}<ref name="ibarakixi">{{Cite news|url=http://ibarakinews.jp/news/newsdetail.php?f_jun=15156790507228|title=幡谷祐一さん死去　県信組会長、元県中央会長 94歳|newspaper=茨城新聞|date=2018-01-12|accessdate=2018-01-12|archive-url=https://web.archive.org/web/20180112214800/http://ibarakinews.jp/news/newsdetail.php?f_jun=15156790507228|archive-date=2018-01-12|dead-url=no}}</ref>）是一名已故日本[[企业家|企业家]]，出生于日本[[茨城县|茨城县]][[東茨城郡|東茨城郡]]小川町（今属[[小美玉市|小美玉市]]）。幡谷曾从83岁开始于日本[[筑波大学|筑波大学]]攻读博士学位，并在三年后毕业，成为当时日本以最高年龄获得博士学位者<ref name=sankei-a>{{cite web |title=幡谷祐一・茨城県信会長死去　９４歳　県内経済振興に尽力 |url=https://www.sankei.com/region/news/180112/rgn1801120065-n1.html |publisher=産経ニュース |accessdate=2018-07-29 |language=ja |archive-url=https://web.archive.org/web/20180729084240/https://www.sankei.com/region/news/180112/rgn1801120065-n1.html |archive-date=2018-07-29 |dead-url=no }}</ref>。

==生平==
幡谷祐一1923年出生于日本[[茨城县|茨城县]][[東茨城郡|東茨城郡]]，是家中的长男，生父係茨城县商工信用組合创始人、前茨城县議会議員幡谷仙三郎。1946年，幡谷祐一毕业于[[日本大学|日本大学]]法文学部。其后，幡谷继承家业开始从商，为茨城县战后经济发展做出了一定贡献<ref name=sankei-a/>。2007年，以83岁高龄入学筑波大学博士后期课程，并在3年后毕业，成为日本获得博士学位时年龄最高者<ref>{{cite web |title=幡谷祐一様へ感謝状贈呈 |url=http://www.tsukuba.ac.jp/news/n201504241135.html |publisher=筑波大学 |accessdate=2018-07-29 |archive-url=https://web.archive.org/web/20150630001843/http://www.tsukuba.ac.jp/news/n201504241135.html |archive-date=2015-06-30 |dead-url=no }}</ref>。2008年，幡谷获颁[[旭日章|旭日中綬章]]。2010年，幡谷获颁紺綬褒章。2018年1月11日，幡谷因病去世，享年94岁<ref name = "ibarakixi"/>。

==平成自由诗==

| align = right

幡谷祐一有作[[汉诗|汉诗]]的爱好，1998年，他出版了诗集《水车—平成自由诗》（{{lang|ja|《水車-平成自由詩》}}）。在2010年于筑波大学毕业后亦赠给筑波大学一首汉诗。筑波大学校方曾将该诗制作成石碑，陈列在筑波大学校内。此外，[[茨城机场|茨城机场]]开港时，他亦为茨城机场作过汉诗《先见》（{{lang|ja|《先見》}}）。但他的汉诗因在平仄、音韵，以及文法方面存在严重问题而受到批评。同时，在互联网上，网民亦开始模仿幡谷祐一的诗句进行恶搞创作，在[[推特|推特]]上曾出现过名为“#平成自由诗”的[[主題標籤|主題標籤]]<ref>{{cite web |author1=徐萧 |title=日本老翁写的汉诗行不行？日本网友首先不服 |url=https://www.thepaper.cn/newsDetail_forward_1620400 |accessdate=2018-07-29 |publisher=澎湃新闻 |archive-url=https://web.archive.org/web/20180303004319/http://www.thepaper.cn/newsDetail_forward_1620400 |archive-date=2018-03-03 |dead-url=no }}</ref><ref>{{cite web |author1=蓝建中 |title=一言不合就“斗诗”？这些日本人如果参加诗词大会，会不会令人刮目相看？ |url=http://ihl.cankaoxiaoxi.com/2017/0210/1678052.shtml |publisher=[[参考消息|参考消息]] |access-date=2018-07-29 |archive-url=https://web.archive.org/web/20170315124622/http://ihl.cankaoxiaoxi.com/2017/0210/1678052.shtml |archive-date=2017-03-15 |dead-url=no }}</ref>。
