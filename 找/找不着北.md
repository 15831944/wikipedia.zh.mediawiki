'''找不着北'''（{{lang-es|'''Perdiendo el norte'''}}，{{lang-en|'''Off Course'''}}）是一部西班牙语[[喜剧电影|喜剧电影]]。由{{tsl|es|Nacho G. Velilla||Nacho G. Velilla}}执导。 

它涉及青年失业和高失业率，经济移民，家庭隔离，语言障碍，阿尔茨海默氏症，不孕症等一系列有趣的主题。

这部电影于2015年3月6日在西班牙首映。<ref>{{Cite web |url=http://www.imdb.com/title/tt3500544/ |title=Off Course（找不着北） - imdb |accessdate=2018-11-29 |archive-url=https://web.archive.org/web/20150327013456/http://www.imdb.com/title/tt3500544/ |archive-date=2015-03-27 |dead-url=no }}</ref>

== 简介 ==
雨果和布劳略，两名受过大学教育的年轻人，厌倦了在西班牙找不到工作和没有未来的生活，看到一个“西班牙在世界”的电视节目后，决定移民到德国。但很快他们对德国的希冀就化成了泡影，并且他们所追求的德国梦可能是一个噩梦。
