'''跟蹤'''全稱'''跟隨蹤跡'''，即是在目標的背後暗中緊貼和監視。各国、各地区法律对跟踪行为有不同处理。通常仅允许司法人员对相关人员进行合法的跟踪。譬如[[警察|警察]]跟蹤[[疑犯|疑犯]]，等待時機成熟時將其[[拘捕|拘捕]]。[[香港警務處|香港警務處]]的[[刑事情報科|刑事情報科]]设有[[跟蹤組|跟蹤組]]。

== 非法的跟踪 ==

現代，部份[[媒體|媒體]]則派出[[狗仔隊|狗仔隊]]跟蹤名人，[[偷拍|偷拍]]及報道名人的[[私生活|私生活]]及揭露其[[私隱|私隱]]等。名人亦需要面对[[私生饭|私生饭]]的跟踪、[[骚扰|骚扰]]。个人面对跟踪、骚扰的境遇因各国司法和社会文化的差异而不同。有认为，法律对跟踪犯处罚过轻，是造成女性被跟踪者最终被杀害的主要原因<ref>{{Cite web |url=http://news.sina.com.cn/w/2018-04-10/doc-ifyteqtq6942760.shtml |title=现行法律惩处力度过轻频酿悲剧 韩国推动“跟踪狂”入刑|accessdate=2019-02-18 |author=金惠真 |date=2018-04-10 |work= |publisher=新浪网，来源：环球网|language=Zh-hans }}</ref>。有台湾作者指出对女性而言被跟踪、被骚扰具有高致命性，而社会文化对跟踪、骚扰又缺乏敏感度，同时司法对此类行为无法有效制止<ref>{{Cite web |url=https://tw.appledaily.com/headline/daily/20171214/37873653/ |title=焦點評論：請讓跟蹤騷擾有法可管（王秋嵐、林嘉萍）|accessdate=2019-02-18 |author=王秋嵐、林嘉萍 |date=2017-12-14|work= |publisher=台湾苹果日报网站|language=Zh-hant }}</ref>。

在中国大陆，由于司法缺失，包括名人<ref>{{Cite web |url=http://ent.ifeng.com/a/20170830/42971448_0.shtml |title=被私生饭跟踪堵家门三个月，杨坤实在是忍不住了！|accessdate=2019-02-18 |author= |date=2017-08-30 |work= |publisher=凤凰网|language=Zh-hans }}</ref>在内，个人面对跟踪、骚扰时，几乎无力反击，长期陷于困境。而此类行为在异性间甚至被社会文化所纵容，美化成[[求愛|求爱]]<ref>{{Cite web |url=https://view.news.qq.com/original/intouchtoday/n3900.html |title=欢乐颂2：“死缠烂打”式求爱是性骚扰吗？|accessdate=2019-02-18 |author=奚应红 |date=2017-05-23 |work= |publisher=腾讯网|language=Zh-hans }}</ref>。2018年7月发生的[[河北涞源反杀案|河北涞源反杀案]]中，长期受到跟踪、骚扰的女性及其家庭多次救助警方无果。直到对方持械闯入家中，被其父母“反杀”，方才结束<ref>{{Cite web |url=http://news.163.com/19/0125/08/E6BSE1SC0001875P.html |title=河北涞源反杀案 一场无法摆脱的骚扰|accessdate=2019-02-18 |author=记者：韩茹雪，实习生：王瑞琪 |date=2019-01-25 |work=责任编辑：肖琦_NN6799 |publisher=网易，来源：新京报|language=Zh-hans }}</ref>。
