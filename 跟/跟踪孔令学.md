《'''跟踪孔令学'''》（{{lang-en|'''Tracks Kong Lingxue'''}}）导演[[张骁|张骁]]，主演范伟、孙宁。<ref>{{cite web |url=http://ent.qq.com/a/20110505/000198.htm |title=《跟踪孔令学》：反映市民小人物的卑微与倔强 |accessdate=2011 |publisher=腾讯网 |work= |language=zh-hans}}</ref>

== 剧情 ==
“孔令学”是一名语文老师，一次在课堂上，他没收了一个同学“刘萌”的[[手机|手机]]，结果女生和男生为了拿回手机，开始跟踪他，从而引发一系列事件。<ref>{{cite web |url=http://ent.163.com/11/0506/08/73BUM2SE000300B1.html |title=《跟踪孔令学》：或许明天你也会这样 |accessdate=2011 |publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.163.com/11/0503/13/734R56HA00032DGD.html |title=《跟踪孔令学》：每个人多少都有点被害妄想症 |accessdate=2011 |publisher=网易 |work= |language=zh-hans}}</ref>

== 花絮 ==
该片的主题是“老师和学生如何‘沟通’”。<ref>{{cite web |url=http://ent.sina.com.cn/r/m/2011-05-04/13383298771.shtml |title=《跟踪孔令学》：拳头和手掌的辩证关系 |accessdate=2011 |publisher=新浪网 |work= |language=zh-hans}}</ref>，拍摄于[[辽宁省|辽宁省]][[沈阳市|沈阳市]]、[[本溪市|本溪市]]。

该片在内部放映时，据称赢得了[[郎永淳|郎永淳]]、[[路一鸣|路一鸣]]、[[王良|王良]]等[[中国中央电视台|央视]]知名主持人的赞誉。<ref>{{cite web |url=http://ent.qq.com/a/20110415/000050.htm |title=《跟踪孔令学》内部放映 央视名嘴赞不绝口 |accessdate=2011 |publisher=腾讯网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.163.com/11/0418/09/71TMCLVI000300B1.html |title=《跟踪孔令学》获名嘴力捧 4月30日正式公映 |accessdate=2011 |publisher= 网易 |work= |language=zh-hans}}</ref>

《[[沈阳晚报|沈阳晚报]]》评价该片是“范伟最差的电影”。<ref>{{cite web |url=http://ent.sina.com.cn/r/m/2011-05-09/15373303071.shtml |title=沈阳晚报：《跟踪孔令学》除了表演别的不及格 |accessdate=2011 |publisher=新浪网 |work= |language=zh-hans}}</ref>
