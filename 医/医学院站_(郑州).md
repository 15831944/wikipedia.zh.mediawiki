'''医学院站'''是[[郑州地铁1号线|郑州地铁1号线]]中段的一个车站，位于中原路与大学路交叉口，毗邻[[金水河_(郑州)|金水河]]畔。<ref>{{cite web |url=http://www.zzmetro.com/station.aspx |language= |title=运营线路 |publisher=郑州地铁 |date= |accessdate=2013年12月28日 |deadurl=yes |archiveurl=https://web.archive.org/web/20131224111815/http://www.zzmetro.com/station.aspx |archivedate=2013年12月24日 }}</ref>它为1号线一期工程的设备集中站。

== 出入口 ==
医学院站是郑州地铁目前设置出入口最多的站点，围绕中原东路大学北路路口，一共有9个出入口：
*路口东北侧：A
*路口东南侧：B
*路口西南侧（自东向西）：D、E、F、G
*路口西北侧（自西向东）：J、I、H

== 周边单位 ==
车站周围分布有[[郑州大学|郑州大学]]南校区（郑州大学医学院）、[[郑州大学第一附属医院|郑州大学第一附属医院]]、[[河南省口腔医院|河南省口腔医院]]、河南省交通厅等单位。
