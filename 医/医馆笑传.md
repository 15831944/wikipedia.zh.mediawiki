'''《医馆笑传》'''，中国大陆明朝背景的古装喜剧，共兩季，导演[[丁仰国|丁仰国]]，第一季由[[陈赫_(中国演员)|陈赫]]、[[王传君|王传君]]、[[姜妍|姜妍]]、[[张子萱|张子萱]]、[[赵圆瑗|赵圆瑗]]主演<ref>{{cite web |url=http://ent.qq.com/a/20141013/036276.htm |title= 《医馆笑传》曝海报 陈赫领衔洗剪吹探案四人组|accessdate=2014 |publisher=腾讯网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://news.xinhuanet.com/ent/2014-10/13/c_127090716.htm |title=《医馆笑传》海报曝光 陈赫领衔洗剪吹探案四人组  |accessdate=2014 |publisher= 新华网|work= |language=zh-hans}}</ref>。由于剧中演员部分出自《[[爱情公寓_(电视剧)|爱情公寓]]》，被誉为古代版本的《爱情公寓》。<ref>{{cite web |url=http://news.xinhuanet.com/ent/2014-09/29/c_127048942.htm |title=《医馆笑传》曝搞笑剧照 爱情公寓式喜剧全面升级  |accessdate=2014 |publisher=新华网 |work= |language=zh-hans}}</ref>于2014年7月12日至10月26日累积107天的时间拍摄完成，并于2015年1月28日登陆安徽卫视和陕西卫视播出，这也是丁仰国继《爱情睡醒了》之后再度导演辛迪加影视公司剧集。

第一季讲述大明京城里一群看似逗B的医馆伙计、阴差阳错和来自东厂西厂的杀手一起，组成了一支神奇的探案小分队。他们齐心协力破获了一件件迷案的故事的搞笑故事<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2015-01-06/doc-icesifvy2925562.shtml |title=《医馆笑传》陈赫如花喜剧咖齐亮相跑男  |date=2014-01-06 |publisher=  新浪娱乐|work= |language=zh-hans}}</ref>。

而'''《医馆笑传2》'''，由[[上海克顿文化传媒有限公司|上海辛迪加影视有限公司]]出品的《'''医馆笑传'''》系列第二部，于2015年8月30日在横店低调开机，11月杀青。该剧延续了《医馆笑传》第一部轻松幽默的风格，无论故事还是制作更在原有基础上升级，揭晓谜底的同时，案件设置更为扑朔迷离，情节也将更为紧凑。除了第一部的主演[[姜妍|姜妍]]不变，第二部由全新的制作团队和主创团队；金牌导演[[梁国冠|梁国冠]]执导，[[李佳航|李佳航]]、[[李金铭|李金铭]]和[[王彦霖|王彦霖]]等加盟。<ref>[http://ent.sina.com.cn/v/m/2015-09-08/doc-ifxhqhuf8209456.shtml 《医馆笑传2》开机 李佳航李金铭主演]</ref>，并于2016年8月3日登陆东南卫视播出。

==剧情==
五百年前的[[明朝|明朝]]京师[[北京|北京城]]，有一家小医馆里，住着一群贫嘴乐呵的年轻人:专治疑难杂症的小大夫朱一品([[陈赫|陈赫]] 飾)、花痴又娇蛮的老板娘陈安安([[张子萱|张子萱]] 飾)、好色又小气的会计赵布祝([[佟磊|佟磊]] 飾)、“金刚芭比”的庄田田……虽然生意萧条，穷得叮当响，但他们仍然欢乐而努力的坚持着梦想。 

一天，医馆馆主离奇失踪，徒弟朱一品随后收到一个卷轴，无意中打开后，天生的神速记忆让他快速阅读和记下了卷轴的内容，之后朱一品看完后晕厥过去，卷轴也随之烧成灰烬。东厂西厂得知后，派麾下高手杨宇轩([[王传君|王传君]] 飾)与柳若馨([[姜妍|姜妍]] 飾)前来医馆卧底查探卷轴里的秘密，阴差阳错，他们和朱一品卷入一个个离奇案件。
