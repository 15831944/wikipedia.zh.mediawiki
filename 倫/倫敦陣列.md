'''倫敦陣列'''（{{Lang-en|'''London Array'''}}）為一座位於[[英國|英國]][[泰晤士河|泰晤士河]][[泰晤士河河口|河口]]的[[離岸式風力發電|海上]][[風力發電廠|風力發電廠]]，2013~2018年間為世界上最大的海上風力發電廠<ref>{{Cite web |url= http://www.londonarray.com/ |title= Harnessing the power of offshore wind |author=  |date=  |publisher= 倫敦陣列有限公司（London Array Limited） |language= en |accessdate= 2013年7月6日 |quote=  }}</ref><ref>{{Cite web |url= http://www.energias-renovables.com/articulo/london-array-el-mayor-parque-eolico-marino-20130705 |title= London Array, el mayor parque eólico marino del mundo |author=  |date= 2013年7月5日 |publisher= Energías Renovables |language=  |accessdate= 2013年7月6日 |quote=  }}{{es}}</ref>，現已被[[英國|英國]] Walney Extension 離岸風力發電廠超越，'''倫敦陣列'''退居世界第二大。

'''倫敦陣列'''整個風力發電廠長達超過20[[公里|公里]]（12英哩），橫跨的幅度從[[肯特郡|肯特郡]][[馬蓋特|馬蓋特]]出發、沿著肯特郡{{Link-en|北方沿海|North Foreland}}以及[[雅息士郡|雅息士郡]]的交界海岸最後到達[[濱海克拉克頓|濱海克拉克頓]]<ref>{{Cite web |url= http://www.londonarray.com/the-project/key-facts/location/ |title= Key facts: London Array location |author=  |date=  |publisher= 倫敦陣列有限公司（London Array Limited） |language= en |accessdate= 2013年7月6日 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20130810204037/http://www.londonarray.com/the-project/key-facts/location/ |archivedate= 2013年8月10日 }}</ref><ref>{{Cite web |url= http://www.ofgem.gov.uk/Networks/offtrans/rttt/Documents1/London%20Array%20PIM.pdf |title= Preliminary Information Memorandum London Array (Phase 1)Offshore Transmission Assets |author= Edward Crosthwaite Eyre |date= 2010年11月16日 |publisher= {{Link-en|天然氣暨電力市場管制局|Office of Gas and Electricity Markets}} |language= en |accessdate= 2013年7月6日 |quote=  }}</ref>。整個風力發電廠建造計畫預計在2016年時全數完工，並且估計自2015年開始便能夠將英國能源使用比例中可再生能源的部分再往上提15.4％左右<ref>{{Cite web |url= http://www.londonarray.com/the-project/ |title= How it all began |author=  |date=  |publisher= 倫敦陣列有限公司（London Array Limited） |language= en |accessdate= 2013年7月6日 |quote=  }}</ref><ref>{{Cite web |url= http://www.spiegel.de/wirtschaft/unternehmen/grossbritannien-eroeffnet-den-groessten-offshore-windpark-der-welt-a-909463.html#js-article-comments-box-pager |title= Offshore-Energie:Großbritannien eröffnet größten Windpark der Welt |author=  |date=  |publisher= 《[[明鏡|明鏡]]》 |language=  |accessdate= 2013年7月6日 |quote=  }}{{de}}</ref>。
倫敦陣列的建造計畫最早於2001年時開始進行籌劃，其中在規劃時設計分成二階段施工的方式裝設共341臺[[風力發動機|風力發動機]]，並且在2007年11月時英國政府批准了這項企劃案<ref>{{Cite web |url= http://www.windenergie-agentur.de/deutsch/newsletter/WABnews12_2007.pdf |title= newsletter 12/2007 |author=  |date= 2007年12月 |publisher= {{Link-de|WAB e. V.|WAB e. V.}} |language=  |accessdate= 2013年7月6日 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20140221210540/http://www.windenergie-agentur.de/deutsch/newsletter/WABnews12_2007.pdf |archivedate= 2014年2月21日 }}{{de}}</ref>。自2011年3月的第一期工程中開始安裝175臺、共能夠輸出630百萬瓦(630 MW)電力的渦輪機<ref>{{Cite web |url= http://www.londonarray.com/2011/03/08/first-foundation-installed-at-london-array/ |title= First foundation installed at London Array |author=  |date=  |publisher= 倫敦陣列有限公司（London Array Limited） |language= en |accessdate= 2013年7月6日 |quote=  }}</ref>，2012年12月第一階段在估計耗費22億[[歐元|歐元]]施工完成後於隔年4月8日首次全面進行運轉測試<ref>{{Cite web |url= http://www.londonarray.com/2012/12/13/final-turbine-installed-at-london-array/ |title= Final turbine installed at London Array |author=  |date=  |publisher= 倫敦陣列有限公司（London Array Limited） |language= en |accessdate= 2013年7月6日 |quote=  }}</ref><ref>{{Cite web |url= http://www.renewableuk.com/en/news/press-releases.cfm/hip-hip-array-world-s-largest-offshore-wind-farm-goes-fully-operational |title= Hip Hip Array-World's largest offshore wind farm goes fully operational |author=  |date= 2013年4月8日 |publisher= {{Link-en|英國再生能源組織|RenewableUK}} |language= en |accessdate= 2013年7月6日 |quote=  |deadurl= yes |archiveurl= https://web.archive.org/web/20130712031607/http://www.renewableuk.com/en/news/press-releases.cfm/hip-hip-array-world-s-largest-offshore-wind-farm-goes-fully-operational |archivedate= 2013年7月12日 }}</ref>。2013年7月4日，在英國首相[[大衛·卡麥隆|大衛·卡麥隆]]的主持下正宣布展開營運<ref>{{Cite web |url= http://gulfnews.com/business/economy/london-array-world-s-largest-offshore-wind-farm-inaugurated-1.1205443 |title= London Array, world’s largest offshore wind farm, inaugurated |author= Shehab Al Makahleh |date= 2013年7月4日 |publisher= 《{{Link-en|海灣新聞|Gulf News}}》 |language= en |accessdate= 2013年7月6日 |quote=  }}</ref>。

因為關注於對海鳥的影響，該項目的第二階段規劃在2014年被驳回。
