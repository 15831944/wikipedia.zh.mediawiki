'''鹹鴨蛋'''又稱'''醃鴨蛋'''、'''鹹蛋'''，古稱'''鹹杬子'''<ref>陶宗仪《辍耕录》：咸杬子，今人以米汤和入盐、草灰以团鸭卵，谓曰咸杬子。按《齐民要术》用杬木皮淹渍，故名之。</ref><ref>沈如筠《异物志》：杬子音元，盐鸭子也，以其用杬木皮汁和盐渍之。今吾乡处处有此，乃如苍耳、益母，茎干不纯是木。小人争斗者，取其叶挼擦皮肤，辄作赤肿如被伤，以诬赖其敌。至藏鸭卵，则又以染其外，使若赫色。</ref><ref>《猗觉寮杂记》：南人以盐收鸭子曰咸杬子。</ref>，是一種[[中國|中國]]傳統[[食品|食品]]，以[[江蘇|江蘇]][[高郵|高郵]]所產的鹹鴨蛋最為有名。古人認為鴨蛋有食療效果<ref>《本草綱目》：俗傳小兒多痢，臭鹹鴨蛋食之，亦間有愈者。蓋鴨肉能治痢，而炒鹽亦治血痢耳。</ref>[[粽|粽]]、[[月餅|月餅]]也會加入鹹鴨蛋黃，{{fact|廣東月餅越多鴨蛋黃，價錢亦越貴，蛋黃以出油「顏色紅而油多」為上品。}}

== 歷史 ==

[[南北朝|南北朝]]時期，[[蘇州|蘇州]]、[[揚州|揚州]]一帶已大量醃製，而且可以久藏<ref>《齊民要術》：浸鴨子一月任食，煮而食之，酒食俱用，鹵鹹則卵浮。</ref>且可以下酒佐食<ref>《老学庵笔记》：《齐民要术》有咸杬子法，用杬木皮渍鸭卵。今吴人用虎杖根渍之，亦古遗法。</ref>。

[[元|元]]代《[[農桑衣食摘要|農桑衣食摘要]]》中記載：「水鄉居者宜養之，雌鴨無雄，若足其豆麥，肥飽則生卵，可以供廚，甚濟食用，又可以醃藏。」。說明當時南方各省養鴨業的情況，盛產鹹蛋。1909年的[[南洋勸業會|南洋勸業會]]上，時人以其味美、[[蛋白質|蛋白質]]、[[脂肪|脂肪]]質、[[碳水化合物|碳水化合物]]、[[鈣|鈣]]、[[磷|磷]]、[[鐵|鐵]]的含量豐富，向外推銷，從而遠銷[[日本|日本]]、[[美國|美國]]及[[新加坡|新加坡]]等許多[[國家|國家]]。

== 製法 ==

鹹鴨蛋大多指醃製過的鴨蛋，各地的製法各有差異，當中以[[清朝|清朝]]才子[[袁枚|袁枚]]一句「醃蛋以高郵為佳」，[[江蘇|江蘇]]、[[高郵|高郵]]的鹹鴨蛋因而聞名<ref>袁枚《随园食单》：腌蛋以高邮为佳，颜色红而油多，高文端公最喜食之。席间，先夹取以敬客，放盘中，总宜切开带壳，黄白兼用；不可存黄去白，使味不全，油亦走散。</ref>。各家製手包括：<ref>{{cite web |url=http://www.ahnw.gov.cn/2006nykj/show.asp?ContentID=%7BD3E8F1E7-9817-4492-945C-8B638F3DA401%7D |title=存档副本 |accessdate=2006-06-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20060522134145/http://www.ahnw.gov.cn/2006nykj/show.asp?ContentID=%7BD3E8F1E7-9817-4492-945C-8B638F3DA401%7D |archivedate=2006-05-22 }}</ref>

* 黃沙醃法
*: 醃製時先將黃沙倒入盆中，加入鹽、油和水，拌成糊狀，再將晾乾的鮮鴨蛋逐個放入粘泥封存妥當，3周後可洗去泥沙煮食。一些地方亦會以泥沙代替黃沙，或在黃沙中加入粘土；亦會有人以[[麵粉|麵粉]]加熱水成糊狀，加入食鹽、五香粉和白酒，以同樣方法製作。
* 鹽水醃製法
*: 將食鹽溶於沸水，冷卻後倒入壇中，並將洗淨晾乾的鴨蛋，逐個放進鹽水中，加以密封，置通風處，25天左右即可開壇取蛋煮食。此法醃製的鹹鴨蛋，據說可令[[蛋黃|蛋黃]]多出油。另外亦可在鹽水中加入[[花椒|花椒]]、[[桂皮|桂皮]]、[[茴香|茴香]]、生[[薑|薑]]、[[鹽|鹽]]，將洗淨的鴨蛋泡入，封存後待40天即可煮食。
* 白酒浸製法
*: 浸醃時先將晾乾的鴨蛋放在[[白酒|白酒]]中浸蘸，再滾上精鹽，放入容器內，密封放置在乾燥、陰涼、通風處，約30天即可取出煮食。
* 辣醬醃法
*: 備[[辣椒|辣椒]]醬、精[[鹽|鹽]]各一碗，把鴨蛋逐個在辣椒醬中均勻蘸一下，以精鹽中滾一遍，放入瓷罐，頂層撒鹽少許，加蓋並以牛皮紙密封，放置在陰涼通風處，30至40天即可食用。另有人會將辣椒醬、[[白酒|白酒]]，按8:2的比例拌勻，以同樣方法把鴨蛋醃製70至90天，據說這種醃鴨蛋呈辣紅色，酒香四溢，鹹中微辛。

== 俚語 ==

=== 賣鹹鴨蛋 ===

除了是描述一種職業，在[[粵語|粵語]]中也是一句[[慣用語|慣用語]]，是「[[死|死]]」的意思，與[[閩南語|閩南語]]「'''蘇州（或塗州）賣鴨蛋'''」意義相同。「人人遲早都會賣鹹鴨蛋」，意思就是「人人早晚都會死」。賣鹹鴨蛋是[[委婉語|委婉語]]，暗示死。一說是因為以前死者入棺時棺下放大量[[牙灰|牙灰]]（以稻草燒製的，新的香爐灰），而鹹鴨蛋製作過程中會加入大量牙灰，故有此[[譬喻|譬喻]]。

==== 典故 ====

源自[[閩南語|閩南語]]諺語：「去土丘（周）剝鴨蛋」「土丘（周）」指墳墓，典故來源有數個。

* [[閩南|閩南]]入葬[[掃墓|掃墓]]習俗中，有些人在先人的[[墳|墳]]前祭拜後，會把[[墓紙|墓紙]]用石塊壓在墓上，親屬再立於墳土周圍，把鴨蛋殼剝撒在墳墓隆起的土丘上，以示修繕之意。故曰'''土周剝鴨蛋'''。
* 早年有些[[閩南人|閩南人]]，會把石頭、熟鴨蛋放入死者[[棺材|棺材]]，在[[封棺|封棺]]的那一刻向死者說：「石頭爛了，鴨蛋孵出來，你再回家罷。」熟鴨蛋是孵不出鴨子的。這種話，有勸告死者不要執著於人間的意味。而鴨蛋被死者帶到墳場（土丘），所以「死亡」被戲稱為'''土丘賣鴨蛋'''。
* [[閩南|閩南]]供奉死者的[[腳尾飯|腳尾飯]]多半放有熟鴨蛋。「土丘」指墳墓，死者[[靈魂|靈魂]]拿了[[腳尾飯|腳尾飯]]上的熟鴨蛋上土丘，故曰'''土丘賣鴨蛋'''。

腳尾飯、棺材裏的熟鴨蛋或掃墓時剝的鴨蛋殼，都是死者才拿得到之物。由於閩南語裡「土丘」、「土周」與「蘇州」音近，「剝」與「賣」音近，演變成「[[蘇州|蘇州]]賣鹹鴨蛋」或「蘇州賣鴨蛋」，不過此純粹為閩南人的戲謔之俗，真正的[[蘇州|蘇州]]居民，並不知道此[[閩南|閩南]]俚語。

當「蘇州賣鹹鴨蛋」或「蘇州賣鴨蛋」，傳播到[[廣東|廣東]]一帶時，再度簡化為「賣鹹鴨蛋」，成為粵語俚語，代表死亡。

==== 附會 ====

有人附會「'''蘇州賣鴨蛋'''」，改編為一個故事。

[[清代|清代]]，[[臺灣人|臺灣人]]某甲於[[蘇州|蘇州]]作[[鴨蛋|鴨蛋]]生意，卻因緣際會與[[蘇州府|蘇州府]]的一位千金相戀，[[結婚|結為連理]]。因是[[入贅|入贅]]，遂在己姓之上冠以妻姓。某甲婚後欲返臺省親，卻因[[臺海|臺海]][[天氣|天候]]不佳，數月未行。某甲母在[[臺灣|臺灣]]，苦無音訊，遂詣廟[[扶乩|扶乩]]、[[求籤|求籤]]，問「'''某甲下落'''」。神示曰「'''世間已無某甲此人'''」。母以為某甲死於蘇州，延[[僧|僧]][[超度|超度]]，為立[[衣冠冢|衣冠冢]]。

數月後某甲卻攜妻返家，母大驚，語之以[[神示|神示]]。某甲笑曰：「神不誤，母誤也。吾[[冠姓|冠姓]]，名改矣，'''世間已無某甲此人'''，即是矣。」於是奉母回蘇州。臺人以某甲之母[[迷信|迷信]]之事，傳為笑柄，故曰'''死'''為「'''蘇州賣鴨蛋'''」。

此故事傳抄不同，或曰某甲本為[[蕭姓|蕭姓]]，名叫蕭建泰。亦有說某甲為[[王姓|王姓]]或[[李姓|李姓]]者。或言妻本[[官家|官家]]女，或說為[[閥閱|閥閱]]女，亦有說妻為[[商賈|商賈]]之女。惟市井傳言，不須在意。
