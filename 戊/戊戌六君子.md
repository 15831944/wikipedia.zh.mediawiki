'''戊戌六君子'''是指[[清代|清代]][[光绪|光绪]]二十四年（1898年，[[农历|农历]]戊戌年）[[慈禧太后|慈禧太后]]發動[[戊戌政变|戊戌政变]]時所逮捕並[[斬首|斬首]]的六名變法派人士，分別為[[谭嗣同|谭嗣同]]、[[林旭|林旭]]、[[楊銳_(清朝)|楊銳]]、[[杨深秀|杨深秀]]、[[刘光第|刘光第]]與[[康广仁|康广仁]]。

1898年9月21日，慈禧太后與[[直隸總督|直隸總督]][[榮祿|榮祿]]发动政变，囚禁了[[光绪帝|光绪帝]]，同时逮捕了譚嗣同六人等，[[刑部尚书|刑部尚书]][[赵舒翘|赵舒翘]]奏曰：“此等乱臣贼子，杀无赦，何必问供。”9月28日，[[六君子|六君子]]在未經[[刑部|刑部]]審訊的情況下，被慈禧太后下令押到[[北京|北京]][[菜市口法场|菜市口法场]]問斬。被处斩的顺序是：康广仁、杨深秀、杨锐、林旭、谭嗣同、刘光第。

== 六君子之被捕及遇害<ref name="楊銳年譜簡編" /> ==

* 八月初九日（9月24日），上諭令逮捕楊銳、林旭、譚嗣同、劉光第等人。同日[[繆潤紱|繆潤紱]]上折攻擊楊銳等軍機四卿阻礙言路，蒙蔽光緒帝<ref>《戊戌變法檔案史料》，第464—465頁</ref>。
* 八月初十日（9月25日），楊銳、劉光第、林旭、譚嗣同等七人被由[[步軍統領|步軍統領]]衙門解交[[刑部|刑部]]，分別監禁<ref>《直報》（1898年9月28日）</ref>。是日，[[盛宣懷|盛宣懷]]致電[[張之洞|張之洞]]<ref>《張文襄公全集》第三冊，第493頁</ref>，報告楊銳等人被捕的消息<ref>同時向張之洞報告楊銳等人被捕消息的可能還有喬樹枏。黃尚毅在《楊叔嬌先生事略》中說：「同鄉喬枏電知張文襄公請救」</ref>。張之洞立即致電當時在北京的湖北[[臬台|臬台]][[瞿廷韶|瞿廷韶]]，請其懇求[[王文韶|王文韶]]、[[裕祿|裕祿]]等設法解救楊銳<ref>《張文襄公全集》第三冊，第759頁</ref>，同時，又致電盛宣懷，讓其轉電王文韶等人<ref>《愚齋存稿》卷三十三，沈雲龍主編：《近代中國史料叢刊續編》第123冊，第786頁</ref>。同時四川京官亦打算救楊、劉，均未奏效<ref>高楷：《劉楊合傳》，《碑傳集補》卷十二</ref>。
* 八月十三日（9月28日），[[楊銳|楊銳]]等六人被殺，史稱「戊戌六君子」。本日刑部擬對六人進行審訊，主持此事的[[慶親王|慶親王]][[奕劻|奕劻]]與[[鐵良|鐵良]]、[[陳夔龍|陳夔龍]]商議，主張對六人分別審訊，認為楊銳、劉光第品行、學問均好，羅織一廷，殊非公道<ref>陳夔龍：《夢蕉亭雜記》，上海古籍書店1985年版，第334頁</ref>。在刑部各官員準備審訊楊銳等人時，忽然奉命將六人直接從[[監獄|監獄]]提出，押赴刑場，時任軍機大臣的[[廖壽恆|廖壽恆]]想通過王文韶，與剛毅、裕祿商議挽回，但未成功，他認為楊銳、林旭、劉光第均系被冤殺<ref>《廖壽恆日記稿本》，上海圖書館藏，轉引自湯志鈞：《戊戌變法人物傳稿》（增訂本），中華書局1982年版，第570頁</ref>。
** 臨刑前，楊銳勸劉光第跪下聽旨<ref>高楷：《劉楊合傳》，《碑傳集補》卷十二</ref>，並多次質問自己的罪名，被監斬官[[剛毅|剛毅]]拒絕。就刑時，血噴涌丈余<ref>《楊參政公事略》，《戊戌變法》第四冊，第66—67頁</ref>。楊銳就刑後，縫合屍體費銀七百五十兩<ref>《新聞報》（1898年10月22日）</ref>，由四川同鄉[[李徵庸|李徵庸]]出資。靈柩寄居清字庵（又稱清慈寺），前來祭奠者多人<ref>黃尚毅：《楊叔嶠先生事略》，《碑傳集補》，卷十二</ref>。

== 六君子之昭雪<ref name="楊銳年譜簡編">「楊銳年譜簡編」，葛志毅著。《中國古代社會與思想文化研究論集》.黑龍江人民出版社, 2006年8月</ref> ==
* 宣統元年（1909年）八月十二日（9月25日），楊銳之子楊慶昶詣[[都察院|都察院]]，將密詔上繳，請昭雪楊銳。八月十七日，都察院上奏，請宣布德宗手詔編入實錄，奉旨：'''留中'''。<ref>罗惇曧：《賓退隨筆》，第298頁，沈雲龍主編：《近代中國史料叢刊》三編第二十六輯，第256冊</ref>。
** 時[[趙熙|趙熙]]為[[江西道監察御史|江西道監察御史]]，告知摯友[[京畿道監察御史|京畿道監察御史]][[趙炳麟|趙炳麟]]，速台長代奏，其則別折專奏，請宣付史館，皆不報。監國攝政王[[醇親王|醇親王]]載灃告訴尚書[[戴鴻慈|戴鴻慈]]：『先帝詔不敢秘，但明發上諭，則為悖慈禧，乖國體』。<ref>《四川文史資料選輯》，第29輯。</ref>
* 宣統二年（1910年），[[資政院|資政院]]又提案請昭雪楊銳，全院贊成，但被政府擱置。<ref>趙炳麟：《諫院奏事錄》卷六，收有《請宣布德宗手詔編入實錄疏》、《再請宣布德宗手詔編人實錄疏》。《趙柏岩集》，近代中國史料叢刊，第303冊，第1205-1215頁</ref>

* 宣統三年（1911年）九月十三日（11月3日），陸軍第二十鎮統制[[張紹曾|張紹曾]]奏請赦免黨人（為該奏摺第七條關於國事犯之黨人一體特赦擢用），奉旨：赦免戊戌以來的黨人。
* 民國三年（1914年），四川省民政長[[陳廷傑|陳廷傑]]呈請政府昭雪楊銳、劉光第等六君子。[[北洋政府|北洋政府]]同意交[[內務部|內務部]]從優獎恤，以昭激勸<ref>姜泣群：《朝野新談》，光華編輯社1914年印行，丁編，第60—62頁。</ref>。令在北京建立祠堂祭奠，並將事蹟宣付[[清史館|清史館]]立傳。

== 時人之議論<ref>《字林西報》周刊（1898年11月7日），「不勝惋惜」，《戊戌變法》，第3冊，第520頁。</ref> ==

== 六君子被捕原因 ==
根據[[梁啟超|梁啟超]]的查證，六君子被捕，各有其原因：
* [[楊銳|楊銳]]及[[劉光第|劉光第]]乃[[康有為|康有為]]的得力助手，慈禧太后的助手[[榮祿|榮祿]]亦恨他們最深，自然要求慈禧太后最先拿下他們。
* [[譚嗣同|譚嗣同]]不想像[[漢朝|漢朝]][[張儉|張儉]]一樣，四處望門投止而連累志士與好友。他亦希望以血喚醒國人變革的希望，故一心求死，拒絕逃亡。他步往刑場時，作《絕命辭》一首：「有心殺賊，無力回天。死得其所，快哉快哉。」
* [[林旭|林旭]]於慈禧太后軟禁光绪後，為報光绪帝知遇之恩，不顧安危，向慈禧太后力諫，保存光绪。最後惹至慈禧太后大怒，被擲入黑獄。
* [[康廣仁|康廣仁]]係[[康有為|康有為]]之弟。[[康有為|康有為]]及[[梁啟超|梁啟超]]逃亡時，來不及通知康廣仁，最後廣仁在南海會館被捕。康廣仁雖在戊戌變法中只負責辦報，但因為是康有為親弟，所以慈禧太后要以弟代兄罪，把康廣仁押到菜市口問斬。
* [[楊深秀|楊深秀]]於五人下獄後，感五人年輕有為，不應就此死去，於是為五人向慈禧太后求情。楊深秀正直不阿，竟演變成要求慈禧太后將权力交還光绪帝的爭論，最後亦為此賠上性命。
