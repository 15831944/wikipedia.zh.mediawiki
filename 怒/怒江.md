'''怒江'''（{{bo|z=རྒྱལ་མོ་རྔུལ་ཆུ་|w=rgyal mo rngul chu}}），是[[薩爾溫江|薩爾溫江]]上游中國境內河段的名稱，发源于[[西藏自治區|西藏自治區]][[那曲市|那曲市]][[安多县|安多县]]境内[[唐古拉山|唐古拉山]]南麓的[[将美尔岗尕楼冰川|将美尔岗尕楼冰川]]，流经[[西藏自治区|西藏自治区]]、[[云南省|云南省]]，流入[[缅甸|缅甸]]后改称[[萨尔温江|萨尔温江]]。幹流長度2,013公里，流域面積12.48萬平方公里。<ref>[http://amuseum.cdstm.cn/AMuseum/chuanyueqingzang/m141.html 怒江，中國數字科技館]</ref><ref name="科学网">{{cite news|author=喻菲|title=中国科学家确定雅鲁藏布江等四条国际河流源头|url=http://news.sciencenet.cn/htmlnews/2011/8/251272.shtm|accessdate=2011-08-23|newspaper=科学网|date=2011-08-22}}</ref>。

怒江上游因江水深黑，中国最早的地理著作《[[禹贡|禹贡]]》把它称为'''黑水河'''；藏語名為'''[[那曲|那曲]]'''（{{bo|z=ནག་ཆུ་|w=nag chu}}），義同。怒江在当地的[[怒族|怒族]]中被称为'''阿怒日美'''，“阿怒”是怒族人的自称，“日美”在[[汉语|汉语]]中译为江，“阿怒日美”就是怒族人居住区域的江，故名怒江。流入[[缅甸|缅甸]]后改称[[萨尔温江|萨尔温江]]（{{lang-my|သံလွင်မြစ်}}，{{IPA|[θàɴlwɪ̀ɴ mjɪʔ]}}）。

== 江源 ==
怒江上游为'''[[那曲|那曲]]'''，发源于[[西藏自治區|西藏自治區]][[那曲市|那曲市]][[安多县|安多县]]境内[[唐古拉山|唐古拉山]]南麓的[[将美尔岗尕楼冰川|将美尔岗尕楼冰川]]，源流區河段名为'''将美尔曲'''<ref name="科学网" />。它先向南轉西再轉西南流，匯集左側支流後改稱'''朶爾丁曲'''。它向南以「S」形流，匯集右側支流[[仲朶曲|仲朶曲]]後改稱'''朶果爾曲'''<ref>[http://map.ps123.net/china/UploadFile/201409/2014092706471182.jpg 聶榮縣地圖]</ref>。該曲流至幫美附近納左側支流後，改稱'''桑曲'''，隨後河道轉向西南流，穿越[[青藏鐵路|青藏鐵路]]後注入[[錯那湖|錯那湖]]<ref>[http://www.sc666.com/upimage/map/200511916552883613.jpg 安多縣地圖，寰宇天涯旅遊網]</ref>。幹流由錯那湖南端流出後，始稱'''那曲'''<ref>[http://map.ps123.net/china/UploadFile/201409/2014092206504743.jpg 安多縣地圖，地圖拍掃網]</ref>。

== 流域 ==
[[Image:Chine_208.jpg|thumb]]旁的怒江]]
怒江上游在高原地区，山势较平坦，水量很大，水面较宽，[[流速|流速]]不大；中下游坡降大，水流湍急，形成高山深谷。怒江支流短小，是典型的羽状水系。

怒江与[[澜沧江|澜沧江]]、[[金沙江|金沙江]]穿越[[横断山脉|横断山脉]]并行南流，与澜沧江的最短直线距离不到19公里。[[三江并流|三江并流]]是世界上地质奇观，已列入[[世界自然遗产|世界自然遗产]]。

==支流==
薩爾溫江－怒江水系在中國境內流域面積大於100km²的支流有59條，其中大於1,000km²有37條，大於5,000km²的支流有6條（[[下秋曲|下秋曲]]、[[索曲|索曲]]、[[姐曲|姐曲]]、[[玉曲|玉曲]]（偉曲）、[[枯柯河|枯柯河]]（猛波羅河）、[[南定河|南定河]]），大於10,000km²的支流有1條（索曲）；其中南定河是在緬甸匯入怒江。<ref>[http://www.cws.net.cn/riverdata/Search.asp?CWSNewsID=17981 《薩爾溫江——怒江》，世界江河資料庫，中國水利國際合作與科技網] {{webarchive|url=https://web.archive.org/web/20141225212957/http://www.cws.net.cn/riverdata/Search.asp?CWSNewsID=17981 |date=2014-12-25 }}</ref>

== 水利工程 == 
怒江河床落差大，有丰富的水能资源，天然蕴蓄出力可达到4700万[[千瓦|千瓦]]。中国2003年曾经有过在境内建设13座梯级水电站的规划，其中丙中洛电站位于三江并流保护区内，由于受到众多专家及环境保护组织人士的质疑，以及媒体的大量调查报道，该规划被搁置下来。
