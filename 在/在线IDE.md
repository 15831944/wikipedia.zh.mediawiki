'''在线IDE'''，也被称为Cloud IDE，主要基于[[HTML5|HTML5]]相关技术构建，通常在浏览器里即可完成传统[[集成开发环境|IDE]]的大部分开发工作。在线IDE的重要组成部分为：基于[[Javascript|Javascript]]的源代码编辑器，相关比较请参见：[[:en:Comparison_of_JavaScript-based_source_code_editors|Comparison of JavaScript-based source code editors]]

==历史==
2008年10月6日<ref>{{cite web |url=http://remysharp.com/2008/10/06/js-bin-for-collaborative-javascript-debugging/|title=JS Bin for Collaborative JavaScript Debugging|accessdate=20 Nov 2012}}</ref> - [[Remy_Sharp|Remy Sharp]]开发了jsbin，是最早的支持实时更新JavaScript，CSS和HTML的环境。
