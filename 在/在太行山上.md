《'''在太行山上'''》是中國著名抗日歌曲，[[桂涛声|桂涛声]]作词，[[冼星海|冼星海]]作曲。<ref name="Bunny_xinhuanet.com">{{Cite web |url= http://news.xinhuanet.com/politics/2009-05/25/content_11434058.htm |title= 100首爱国歌曲名单公布 |author=  |date=2009-05-25 |publisher= 新华网 |language= zh-hans |accessdate= 2011-05-29 导入 |quote= }}</ref>创作于1938年。

== 创作背景 ==
1937年，“七七事变”爆发后，日本发动了全面侵华战争。桂涛声在随游击队转战陵川的过程中，创作了歌词《在太行山上》。

1938年5月，桂涛声离开[[太行山|太行山]]，6月返回[[武汉|武汉]]，即带着歌词去见当时在国民政府军事委员会政治部第三厅主持抗战音乐工作的[[冼星海|冼星海]]。

冼星海将歌词《在太行山上》谱写成一首兼有抒情性和进行曲风格的二部合唱曲。<ref name="Bunny_ifeng.com">{{Cite web |url= http://news.ifeng.com/mil/history/detail_2011_02/14/4656318_0.shtml |title= 《在太行山上》往事新提 朱德曾抄录其歌词带身上 |author= 作者 |date=2011-02-14 |publisher= 出版者 |language= zh-hans |accessdate= 2011-05-29 导入 |quote= }}</ref>

== 词曲版权 ==
词作者[[桂涛声|桂涛声]]于1982年在[[上海|上海]]病逝。

曲作者[[冼星海|冼星海]]于1945年10月30日病逝于[[克里姆林宫医院|克里姆林宫医院]]。

== 使用 ==
*1938年7月，该曲在[[周恩来|周恩来]]和[[郭沫若|郭沫若]]组织的武汉纪念抗战一周年群众歌咏大会上首次公开演唱，由[[张曙|张曙]]、[[林路|林路]]、[[赵启海|赵启海]]等在大会上唱出。<ref name="Bunny_ifeng.com"/>
*2005年电影《太行山上》插曲
