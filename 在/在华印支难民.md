'''在华印支难民'''，是指在[[印度支那難民危機|印度支那難民危機]]中由[[中华人民共和国政府|中国政府]]收留的难民，绝大多数为来自[[越南|越南]]，仅有少量来自[[老挝|老挝]]和[[柬埔寨|柬埔寨]]，而且绝大多数是[[华人|华人]]或[[华裔|华裔]]，仅少量为[[京族|越南人]]。

== 历史 ==
1970年代之前，[[中越關係|中越两国关系]]亲密，常以“同志加兄弟”来形容，但两国之间也有一些分歧和矛盾，如中国反对越南建立[[印度支那联邦|印度支那联邦]]，越南批评中国顾问在[[越南土地改革|越南土地改革]]中的作为，两国对[[苏联|苏联]]和[[美国|美国]]的政策分歧，中国对[[越南統一|越南统一]]的支持不足等等<ref>{{cite journal |author1=董应龙 |title=二十世纪七十年代中越关系研究 |journal=云南师范大学 |date=2006}}</ref>。1975年，越南统一后，越南加强了对反对[[越南北屬時期|北方侵略]]历史的宣传，企图侵占[[西沙群岛|西沙群岛]]、[[南沙群岛|南沙群岛]]和[[北部湾|北部湾]]三分之二的海域，并在[[中越边境|中越陆地边境]]不断蚕食中国领土，攻击[[中美建交|中国对美外交政策]]等。1975年开始，越南政府采取了一系列措施强迫在越华侨[[归化|归化]]。1977年初，越南开始在与中国毗连的省份实行“净化边境”政策，将居住在中越边界的华人华侨迁往内地，后改为驱赶他们回中国，且“净化”行动逐步扩展到越南其他省份<ref name="张杰">{{cite web |author1=张杰 |title=越南使用化学武器许世友震怒：谅山一间房不留 |url=http://news.ifeng.com/history/zhongguoxiandaishi/detail_2013_05/22/25594412_1.shtml |website=凤凰网-财经文摘 |date=2013-05-22}}</ref>。到1977年底，有4万多华侨被驱逐回中国<ref>{{cite web |author1=罗雪珍 |title=越南为何开始排华：早已怀恨在心 |url=http://history.m4.cn/2014-05/1231499.shtml |website=四月网 |date=2014-05-16}}</ref>。

1977年5月，中国政府以安置[[歸僑|归国难侨]]为名先后取消多个对越援助项目。1977年7月，中国政府全面停止对越经济技术援助，撤回在越中国工程技术人员。越南则抛弃其此前在[[国际共运|国际共运]]中对[[中苏交恶|交恶的中苏两国]]的中立立场，彻底执行亲苏路线<ref name="张杰" />。1978年初，越南当局开始清理[[户籍|户口]]，强迫国籍识别为中国的在越华人离开越南<ref name="实录">{{cite journal |author1=孙纯福 |title=越南驱赶华侨实录 |url=http://news.ifeng.com/history/zhongguoxiandaishi/detail_2014_02/02/33528241_0.shtml |journal=文史精华 |date=2013 |issue=10 |pages=23-28}}</ref>。1978年4月至8月，被驱逐回中国的华侨数量达16万<ref name="张杰" />。1978年5月8日，中国国务院根据中央指示召开接待安置被[[越南|越南]]驱逐回国难侨工作会议，由国务院副总理[[纪登奎|纪登奎]]、[[国务院侨办|国务院侨办]]主任[[廖承志|廖承志]]、外交部副部长[[韩念龙|韩念龙]]牵头。决定在[[广东|广东]]、[[广西|广西]]、[[云南|云南]]、[[福建|福建]]四省区设立接待安置归国华侨委员会或领导小组<ref name=gxtzqwz/>。至1979年6月，共有26.5万越南难民通过[[云南省|云南省]]的[[河口口岸|河口口岸]]和[[广西壮族自治区|广西壮族自治区]]的[[东兴口岸|东兴]]、[[凭祥口岸|凭祥口岸]]进入中国<ref name="温情的庇护">{{cite journal |author1=张宝砚 |title=温情的庇护——印支难民在中国 |journal=中国民政 |date=2015 |issue=24 |page=59}}</ref>。1979年6月16日，[[中华人民共和国外交部发言人|中国外交部发言人]]发表声明，呼吁国际社会坚决制止越南驱逐难民。1979年7月20日召开印支难民问题[[日内瓦|日内瓦]]国际会议，中国代表[[章文晋|章文晋]]称已经收容25万印支难民，并承诺从[[东南亚|东南亚]]和[[香港|香港]]接收自愿来华的1万印支难民<ref name=gxtzqwz/>。

== 管理机构 ==
1979年8月1日至8月4日，国务院副总理[[李先念|李先念]]、[[陈慕华|陈慕华]]主持召开了接待安置印支难民工作会议<ref name=gxtzqwz/>。1979年8月4日，国务院决定成立“[[国务院接待安置印支难民领导小组|国务院接待安置印支难民领导小组]]”，[[中华人民共和国民政部|民政部]]部长[[程子华|程子华]]任组长，副组长有国务院侨办副主任[[林修德|林修德]]、[[中华人民共和国农垦部|农垦部]]副部长[[张修竹|张修竹]]、外交部部长助理[[张灿明|张灿明]]、[[中国红十字会|中国红十字会]]副会长[[杨纯|杨纯]]。领导小组办公室设在民政部<ref name=mzbjj>{{Cite web |url =http://www.gov.cn/fuwu/2014-02/26/content_2622345.htm |title =民政部简介 |author = |date = 2014-02-26 |publisher =中国政府网}}</ref><ref name=gxtzqwz/>。在有接待安置任务的广东、广西、云南、福建、江西六省区设立接待安置印支难民领导小组<ref name=gxtzqwz>广西通志·侨务志，广西人民出版社，1994年，第123、253页</ref>。1988年11月7日，根据国务院机构改革方案，国务院接待安置印支难民领导小组被裁撤<ref>{{Cite web |url =http://www.gov.cn/zhengce/content/2011-03/25/content_8047.htm |title = 国务院办公厅关于国务院已撤三十八个非常设机构的通知 |author = |date = 2011-03-25 |publisher =  中国政府网}}</ref>，但国务院接待安置印支难民领导小组办公室保留至今，仍由民政部运作。

== 国际合作 ==
1979年6月，中华人民共和国恢复了在[[联合国难民事务高级专员公署|联合国难民署]]执委会的活动，每年派团出席执委会届会，并派部长级代表团多次出席有关难民问题的国际会议。自1978年以来，中国政府先后接收28.3万印支难民。1979年11月，中国和联合国难民署签订该署援助在华印支难民的项目协议。联合国难民署也自1979年下半年起向在华印支难民提供援助，并自1979年10月起在华设立任务代表处<ref name=lhgnmsh>{{Cite web |url = http://www.chinaqw.com/zgqj/qkjc_hnyhw/201304/01/179337.shtml|title =中国与联合国难民署的合作 |author =陶满成  |date = 2013-04-01 |publisher = 中国侨网 |language =  |accessdate =  |quote =  }}</ref>。1980年2月，联合国难民署任务代表处在北京正式成立，职责限于协助中国政府执行联合国难民署的无偿援助项目，帮助有条件的难民到第三国安置等。1995年12月1日任务代表处升格为代表处。1997年该代表处升格为地区代表处， 负责[[中国大陆|中国大陆]]、[[香港特别行政区|香港特别行政区]]、[[澳门特别行政区|澳门特别行政区]]、[[蒙古国|蒙古国]]、[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]的难民事务<ref name=zhouzheng/><ref>{{Cite web |url = http://news.sina.com.cn/c/2006-09-28/130010131797s.shtml |title = 联合国难民署驻华代表处招聘职员 |author =  |date = 2006-09-28 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref><ref name=lhgnmsh/>。

1982年9月，中华人民共和国加入《1951年[[难民地位公约|难民地位公约]]》和《1967年[[难民地位议定书|难民地位议定书]]》<ref name=lhgnmsh/>。1994年以前，中国政府从联合国难民署争取到共9000万美元无偿援助，在民政部指导下，在六省（自治区）接待安置印支难民办公室的努力及当地政府支持下，共执行约600个项目。从1994年开始，无偿援助改为项目贷款形式的“周转金计划”。1994年至2007年，在华印支难民周转金计划的资金所有权和管理权归联合国难民署，周转金项目的最终审批权由联合国难民署驻华代表处执行。2008年，民政部与联合国难民署代表签署协议，联合国难民署将总额5300多万的在华印支难民周转金资金所有权和管理权正式移交中国政府，由民政部执行<ref name=zhouzheng>周聿峨、郑建成，在华印支难民与国际合作：一种历史的分析与思考，南洋问题研究2014(3):41-47</ref>。2008年7月，国务院接待安置印支难民领导小组办公室制定并下发《在华印支难民周转金项目管理办法》<ref>{{Cite web |url =  http://news.hexun.com/2011-07-08/131276775.html|title = 广西伏虎华侨农场获得在华印支难民周转金项目 |author =  |date = 2011-07-08 |publisher =和讯  |language =  |accessdate =  |quote =  }}</ref>。

== 现状 ==
截至2005年底，居留在中国的印支难民及其子女数量已达29.5万人，其中绝大多数是越南籍难民，占99%，且多为[[华裔越南人|华裔越南人]]或者[[越南籍华人|越南籍华人]]，少数是越南人、老挝人、柬埔寨人，分布在中国南方六省（自治区）的196个安置单位<ref name="实录" />。截至2005年底，广西112268人、广东83538人、云南38009人、福建27409人、海南31475人、江西1715人<ref name=fang>方卫军，略论中国境内印支难民的法律地位，国际论坛2015(5):46-51</ref>。六省（自治区）在省、市两级设有“接待安置印支难民办公室”，如海南省接待安置印支难民办公室、钦州市接待安置印支难民办公室。华侨农场按照属地原则管理。

2012年，[[中国法学会|中国法学会]]发布的《中国法治建设年度报告(2011)》称，30多年来，中国政府严格履行《[[关于难民地位的公约|关于难民地位的公约]]》义务，克服各种困难，向近30万印支难民提供了有效保护<ref name=cntv.cn>{{cite web |title=中国法治建设年度报告(2011)全文 |url=http://news.cntv.cn/china/20120830/106569_3.shtml |website=央视网-法制日报 |date=2012-08-30}}</ref><ref>{{cite web |author1=张蔚然 |title=中国30多年来向近30万印支难民提供了有效保护 |url=http://news.163.com/12/0717/18/86KS8O1300014JB6.html |website=网易-环球网 |date=2012-07-17}}</ref>。

2013年7月1日起施行的《[[中华人民共和国出境入境法|中华人民共和国出境入境法]]》第46条进一步完善了难民和申请难民地位的外国人在中国境内停留居留的规定。在华印支难民开始按照永久性安置转变<ref name=zhouzheng/>。
