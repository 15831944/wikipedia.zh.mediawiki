《'''在那遥远的地方'''》（{{lang-en|'''In That Distant Place'''}}）中国大陆电视剧，导演[[俞钟|俞钟]]，主演[[宋汶霏|宋汶霏]]、[[殷桃|殷桃]]、[[李幼斌|李幼斌]]、[[沈晓海|沈晓海]]。<ref>{{cite web |url=http://ent.sina.com.cn/v/2009-07-12/18352607356.shtml |title=资料：电视剧《在那遥远的地方》主创介绍 |accessdate=2009 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/v/2009-07-12/18092607306.shtml |title= 资料：电视剧《在那遥远的地方》演员表 |accessdate=2009 |publisher=新浪网 |work= |language=zh-hans}}</ref>该剧是庆祝中华人民共和国建国六十年[[献礼剧|献礼剧]]。<ref name="fish" >{{cite web |url=http://news.xinhuanet.com/mil/2009-08/20/content_11918283.htm |title=电视剧《在那遥远的地方》作品研讨会在京举行 |accessdate=2009 |publisher=新华网 |work= |language=zh-hans}}</ref>，2009年7月26日[[央视一套|央视一套]]首播。

==剧情==
1970年代，[[八大军区|昆仑军区]]招收士兵，袁鹰、丁浩天、吕强、方扬四个哥儿们报名参军，该剧讲述了青年踊跃参军、服务边疆的故事。
