'''在不瘋狂就等死'''（{{lang-en|Crazy Right Now}}），是[[台灣|台灣]]素人組成的[[網路|網路]][[搞笑|搞笑]]短片團體，於[[Facebook|Facebook]]、[[Youtube|Youtube]]、[[微博|微博]]、[[抖音|抖音]]發表以搞笑、戲劇演出的影像作品，由團長[[游否希|游否希]]在2014年3月13日創立。其著名作品中《瘋狂駕訓班》的點閱次數在Facebook超過1,200萬次<ref>{{Cite web |url = https://www.bnext.com.tw/article/48216/crazy-right-now-not-only-respect-you-because-you-are-a-star |title = 不是你紅了就把你當神拜！繼續瘋狂的在不瘋狂就等死 |author = May Zhuang |date = 2018/2/14 |publisher = 數位時代 |language = 中文 }}</ref>，是台灣第一個Facebook素人破百萬點閱的搞笑團體<ref>{{Cite web |url = https://www.youtube.com/watch?v=hKxw1e2v_1A |title = 網紅真好賺？進入門檻低 人氣就是買氣 新興職業竄升夢幻工作第一名 |author =【大數據i檔案】 |date = 2018/6/28 |publisher = 三立iNEWS |language = 中文 }}</ref>，第一個將團體名字變公司、甚至企業化經營的網紅<ref>{{Cite web |url =https://stars.udn.com/star/story/10092/3330890 |title = 網紅「在不瘋狂就等死」企業化經營 賺人氣前先拚財力
 |author =王郁惠 |date = 2018/8/26 |publisher = 噓！新聞|language = 中文 }}</ref>。少數有專業老師上課培訓的網路團體。<ref>{{Cite news|url=https://www.youtube.com/watch?v=TNnQz29oacc&feature=youtu.be|title =在不瘋狂就等死►新聞特別介紹 [東森網紅大經濟]| date=2016-10-29|language=中文}}</ref>。
2017年10月因為Facebook觸及率換算更改，在不瘋狂就等死決定在10月20號開始經營Youtube，這個時候Youtube頻道訂閱人數為28萬，在2019年3月28日Youtube突破100萬人訂閱。

團長為游否希
現任團員為唐婷婷、亞亞、貝可儀、巴巴烙、本燙、古拉的七人團體

== 團體歷程 ==
=== 2014年 ===
「在不瘋狂就等死」的團長[[游否希|游否希]]以手機錄影紀錄生活短片，由於電腦空間不足，所以開創了粉絲團把影片上傳當作資料庫，也因為隨手的創建在編輯粉絲團名稱時隨手打了「在不瘋狂就等死」。影片上傳後，因為一支《GPS導航系列》引起網友大量轉發而爆紅，隔天粉絲團的讚數到達5,000多人。有網友提醒粉絲團名稱有錯別字，所以立刻進入編輯，但因臉書必須提供證明文件才能更改而作罷。經過一年多粉絲團紅了，這個錯誤也就變成特色。

初期都以游否希自拍的方式曝光，其中以《GPS導航系列》、《KUSO詐騙電話》、《惡搞飲料店外送》最為經典，當時也因此上了電視新聞報導。

團長游否希原是廢水處理廠的技術員<ref>{{Cite web |url = https://www.youtube.com/watch?v=wjyXpCQk_Dc&feature=youtu.be |title = 中天新聞台灣大搜索專訪《在不瘋狂就等死》 |author =  |date =  |publisher = 中天新聞 |language = 中文 }}</ref>，因影片爆紅後無心工作，加上對拍短片走火入魔而辭掉原本工作。當時因沒有收入，所以負債將近三十萬，跟朋友周轉過生活。直到2015年年初加入了[[狂人娛樂|狂人娛樂]]，一起打拼拍片並還清債務。

=== 2015年 ===
*2015年2月 因緣際會的朋友介紹下，[[瑄瑄|瑄瑄]]參與了游否希的拍攝，意外以《抱怨改裝機車》一戰成名，連續上了好幾次電視新聞版面<ref>{{Cite web |url = https://www.bnext.com.tw/article/38205/BN-2015-12-11-190532-178|title = 顛覆影音生態，素人也可以拿到百萬訂單|author =蘇宇庭 |date = 2015/12/11 |publisher = 數位時代 |language = 中文 }}</ref>，也因此締造了許多高瀏覽量的《瑄瑄抱怨系列》。這是台灣首次以吐嘲的方式爆紅的短片，這支影片在Facebook有389萬點閱。

*2015年5月 接到了統一麥香網路宣傳影片的商案，在臨演裡面挖掘了「蛋蛋」，並成功打造了《蛋蛋說故事系列》，其中《別惹刺青師》Facebook有417萬點閱，這支影片也是後來另一位團員婷婷的處女作品。因為瑄瑄、蛋蛋和後來*2015年6月的婷婷加入，「在不瘋狂就等死」由[[狂人娛樂|狂人娛樂]]經紀公司和游否希操盤後慢慢變成了一個團體。

*2015年8月[[唐婷婷|唐婷婷]]正式加入，當時年紀只有16歲，一直在團隊裡扮演著配角，因外表的關係大多都是飾演霸氣女的角色。因為演技出色，近期開始接到電影或者戲劇上的邀約，其中令人印象深刻的是與阿得共同演出的《七分鐘了解愛》，在Facebook將近250萬瀏覽次數。

*2015年9月，藝人[[許振得|許振得]]加入[[狂人娛樂|狂人娛樂]]，並且參與了「在不瘋狂就等死」的拍攝，本來在電影《[[總舖師_(電影)|總鋪師]]》、《[[健忘村|健忘村]]》、《[[報告老師！怪怪怪怪物！|報告老師！怪怪怪怪物！]]》還有《[[我的鬼基友|我的鬼基友]]》就已經演出的他，因為「在不瘋狂就等死」而不單純只是演員，在網路上也造成極大的迴響。尤其阿得拍攝的《抱怨系列》，在Facebook上也有458萬點閱，其中最火紅的《男人崩潰臨界點》在Facebook有713萬點閱，並且上了國外的媒體平台報導。

*2015年11月，因為朋友的推薦亞亞加入了[[狂人娛樂|狂人娛樂]]，起初由於亞亞跟瑄瑄的長相相似，讓許多網友一直分辨不出來，所以亞亞本身讓網友的第一印象就不深刻，中途公司還讓她參加了電影《[[最完美的女孩|最完美的女孩]]》擔任演員管理的工作。經過將近兩年的努力，終於走出自己的特色，而《撩妹系列》讓亞亞爆紅，在Facebook上有352萬的點閱。目前開始在網路戲劇中《[[星不了情|星不了情]]》和郭子乾郭哥合作，並且也開始主持另一個網路新節目《[[校園不思議|校園不思議]]》。

=== 2016年 ===
*2016年1月，否希找貝可儀合作改編歌曲系列，而改編的歌曲《[[小苹果_(歌曲)|小蘋果]]》爆紅，本來粉絲頁只有2萬多，一下爆漲到8萬多，Facebook也有237萬的點閱，之後也加入公司一起打拼。目前貝可儀和後來加入的古拉已經開始錄製EP，預計在2019年推出。

*2016年7月，[[巴巴烙|巴巴烙]]從美國回來，因為機緣下剛好公司有一篇徵選固定班底演員的文章前來公司面試，就加入了「在不瘋狂就等死」。他本名毛祁生，否希覺得這名字沒有特色，所以就隨便想了個「巴巴烙」，諧音為886，讓它成為藝名。他後來考取了台藝大，因為課業繁忙，所以有一段時間沒什麼曝光的機會，之後《瘋狂駕訓班》在Facebook衝到1,200萬點閱，而最近的《撩妹系列》達到巔峰在微博破億點閱。目前正學習導演之路，其作品《攝攝放題》在網路上大受好評。

*2016年11月，因為拍攝一部商業宣傳片擔任臨時演員，由於表現突出，被公司相中，所以被找來公司談簽約與固定班底的意願而讓[[本燙|本燙]]加入「在不瘋狂就等死」。原本早上還有在豬肉攤上班的他，本名陳信彰，但就在煩惱他的藝名時，否希也就順口說了：「你感覺很愛吃胖嘟嘟的很可愛，你就叫飯桶(台語)吧！」這就是本燙的由來。但除了藝名有特色外，在外型這塊否希也要求本燙把原先留的頭髮剃光，在後腦勺刻了一個本字，這樣一來就更能凸顯特色，兩個月就以一支《胖子的好處》創下Facebook有339萬瀏覽。

=== 2017年 ===
*2017年2月，否希想走更多元的風格<ref>{{Cite web |url = https://www.youtube.com/watch?v=KBuhschaQLw |title = 【BIG SHOT網路紅人誌】專訪 x 在不瘋狂就等死：每件事都是一個挑戰，只要有熱情就不怕！  |date = 2017/3/3 |publisher = Web TV Asia |language = 中文 }}</ref>，所以在臉書徵選了原住民演員。古拉就是在這時候私訊應徵，當時留著長髮的古拉來到了公司面試，否希與公司都覺得除了長相沒有什麼特色。之後否希就出了一個《原住民抱怨》的劇本，原本一頭長髮的古拉，被否希改造成雙馬尾的特色，不出意料又再一次在Facebook上創下201萬點閱率。因為古拉會唱歌，他和貝可儀合唱的《不該》，在東森新聞Facebook網路新聞上有643萬的點閱，目前他和貝可儀剛進錄音室錄製完新的單曲EP，預計在2018推出。

*2017年3月，[[狂人娛樂|狂人娛樂]]正式和M17集團合作[[巔瘋娛樂|巔瘋娛樂]]<ref>{{Cite web |url = https://www.facebook.com/17yourvideo/videos/531771707159474/ |title = 巔瘋娛樂加入M17集團|date = 2018/6/22 |publisher = ETtoday新聞雲 |language = 中文 }}</ref>。

*2017年10月16日[[巔瘋娛樂|巔瘋娛樂]]公司開幕營運，公司地址光復北路137號成立公司，在不瘋狂就等死正式進駐。
