{{jp|c='''在日韩国青年同盟'''|j=在日韓国青年同盟|hg=ざいにちかんこくせいねんどうめい|rm=Zainichi Kankoku Seinen Dōmei|extra1={{lang-ko|재일한국청년동맹 / 在日韓國青年同盟}}|lead=yes}}，简称{{jp|c=“'''韩青'''”|j=韓青|hg=はんちょん|rm=Hanchōn|extra1={{lang|ko|한청 / 韓青}}}}，是一个面向16岁至35岁[[在日韩国人|在日韩国人]]的组织，总部位于[[日本|日本]][[东京都|东京都]][[千代田区|千代田区]]。该组织为[[在日韩国民主统一联合|在日韩国民主统一联合]]（韩统联）的下属团体。

==历史==
韩青的前身为成立于1945年11月的朝鲜建国促进青年同盟（建青），该组织分裂自[[在日本朝鲜人联盟|在日本朝鲜人联盟]]<ref>{{cite web |script-title=ja:冷戦体制の煽りで祖国分断化 |url=http://www.marino.ne.jp/~rendaico/seito_nittyoseizishi_reisentaisei.htm |website=marino.ne.jp |accessdate=2018-11-12 |language=ja |archive-url=https://web.archive.org/web/20050319233344/http://www.marino.ne.jp/~rendaico/seito_nittyoseizishi_reisentaisei.htm# |archive-date=2005-03-19 |dead-url=no }}</ref>。1950年8月，建青更名为“大韩青年团”，并成为[[在日本大韩民国民团|在日本大韩民国民团]]（民团）下属组织。在20世纪50年代，大韩青年团一度被视为[[李承晚|李承晚]]政权的“御用团体”<ref name=history>{{cite web |script-title=ja:韓青の歩み |url=http://hanchung.org/history |website=韩青 |accessdate=2018-11-12 |language=ja |archive-url=https://web.archive.org/web/20180831230111/http://hanchung.org/history# |archive-date=2018-08-31 |dead-url=no }}</ref>。

受“[[四·一九运动|四·一九运动]]”影响，1960年10月19日，大韩青年团改组为“在日韩国青年同盟”。1961年[[五一六军事政变|五一六军事政变]]后，韩青与民团“民主派”共同对抗支持[[朴正熙|朴正熙]]政权的“从属派”。1971年，在在[[韩国中央情报部|韩国中央情报部]]（今[[国家情报院|国家情报院]]）的介入下，从属派彻底控制了民团领导层，与民主派的对立日趋激烈。1972年，韩青与民主派一同被开除出民团。此后韩青一度成为独立组织，反对朴正熙政权“维新体制”，推进[[韩国民主化运动|韩国民主化运动]]，并与[[在日本朝鮮人總聯合會|在日本朝鮮人總聯合會]]（总联）下属的{{link-ja|在日本朝鮮青年同盟|在日本朝鮮青年同盟}}展开合作。<ref name=history/><ref>{{cite news |script-title=ja:<青年会中央35周年>従北勢力（総連・韓統連）との闘いから生まれて |url=http://www.mindan.org/front/newsDetail.php?category=1&newsid=15528 |accessdate=2018-11-12 |publisher=在日本大韩民国民团 |language=ja |archive-url=https://web.archive.org/web/20160305153108/http://www.mindan.org/front/newsDetail.php?category=1&newsid=15528# |archive-date=2016-03-05 |dead-url=yes }}</ref>

1973年8月15日，韩统联的前身“-{zh-hans:韩国;zh-hant:南韓;}-民主回复统一促进国民会议”（韩民统）成立，韩青成为韩民统的下属团体。韩民统改组为韩统联后，韩青继续保留在韩统联内<ref name=history/>。

==活动及主张==
与母团体韩统联相似，韩青主要投身于[[朝鮮半島統一|朝鮮半島統一]]运动、民主运动及和平运动、推进[[韓國語|韓國語]]及朝鮮歷史文化教育、促进同胞交流等<ref>{{cite web |script-title=ja:綱領 |url=http://hanchung.org/kouryou |website=韩青 |accessdate=2018-11-12 |language=ja |archive-url=https://web.archive.org/web/20180831214933/http://hanchung.org/kouryou# |archive-date=2018-08-31 |dead-url=no }}</ref>。

==下属组织==
韩青总部与韩统联总部共用一个办公地点，在日本的[[神奈川县|神奈川县]]、[[爱知县|爱知县]]、[[三重县|三重县]]、[[京都府|京都府]]、[[大阪府|大阪府]]、[[兵库县|兵库县]]及[[广岛县|广岛县]]的地方分部也与当地的韩统联地方分部合并办公<ref>{{cite web |script-title=ja:地方本部紹介 |url=http://hanchung.org/address |website=韩青 |accessdate=2018-11-12 |language=ja |archive-url=https://web.archive.org/web/20180901002915/http://hanchung.org/address# |archive-date=2018-09-01 |dead-url=no }}</ref>。
