'''岑麗香'''（英文：{{lang|en|'''Eliza Sam'''}}，{{bd|1984年|11月17日|}}），[[香港|香港]]女演員，前[[無綫電視|無綫電視]]（TVB）經理人合約女藝員。她是2010年[[國際中華小姐|國際中華小姐]]及2009年[[溫哥華華裔小姐|溫哥華華裔小姐]]的冠軍。因為其主演劇集《[[巴不得媽媽...|巴不得媽媽...]]》的「香香公主」角色大受歡迎而被傳媒稱為「香香」。

== 簡歷 ==
岑麗香來自一個加拿大的[[越南華僑|越南華僑]]家庭，其祖父母來自廣州及福建，父母是溫哥華的越南華僑，而她則在溫哥華土生土長。她畢業於加拿大[[昆特侖理工大學|昆特侖理工大學]]，擁有時裝設計、市場管理學及人力資源學三個學士學位<ref>[http://hk.on.cc/hk/bkn/cnt/entertainment/20161218/bkn-20161218121330222-1218_00862_001.html   深得人心　岑麗香全靠性格真食腦]</ref>。

2009年12月，她參選了2009年度[[溫哥華華裔小姐|溫哥華華裔小姐]]成為7號的候選佳麗，其甜美燦爛的笑容給眾人印象深刻。在總決賽上，她對自己的表現甚具信心，最後贏得了2009年[[溫哥華華裔小姐|溫哥華華裔小姐]]冠軍之名銜。

2010年10月，岑麗香代表溫哥華前往香港及[[中國大陸|中國大陸]]參加由無綫電視主辦的2010年度[[國際中華小姐|國際中華小姐]]，成為19號的候選佳麗，而之前她還是位[[英語|英語]]會話老師。在競選過程之中成為了大熱門<ref>明報，2010年10月20日；中華小姐中(3)丁文冰、(18)張慧雯及(19)岑麗香都是較省鏡的佳麗。</ref>，最後亦大熱撃敗亞軍的[[陳美妤|陳美妤]]，再次贏得選美冠軍<ref>大公報, 2010年11月6日；溫哥華佳麗奪中華小姐桂冠</ref>。

此外，岑麗香與[[鍾嘉欣|鍾嘉欣]]、[[李亞男|李亞男]]、[[王君馨|王君馨]]、[[苟芸慧|苟芸慧]]因同為[[基督|基督]][[新教徒|新教徒]]，數人自發組成祈禱小組「Amazing Esthers」，亦常有聚會，因此常為傳媒牽連在一起並稱為「TVB五大處女」。眾人先後在[[李亞男|李亞男]]與[[鍾嘉欣|鍾嘉欣]]的婚宴中擔任姊妹團。直到2016年，岑麗香與[[李亞男|李亞男]]、[[鍾嘉欣|鍾嘉欣]]、[[王君馨|王君馨]]紛紛出嫁，即被改稱為「索爆人妻團」<ref>{{cite web|url=http://www.sina.com.hk/news/article/20170722/0/3/1/好寫意-相繼嫁人-處女黨改名-索爆人妻團-7710359.html|title=相繼嫁人 處女黨改名「索爆人妻團|date=2017-07-22|work=新浪網娛樂}}</ref>。之前為無綫電視監製[[潘嘉德|潘嘉德]]及[[梁材遠|梁材遠]]的御用演員，結婚後開始減產，產量逐漸減少。

2018年6月15日宣布已經與[[無綫電視|無綫電視]]結束合約<ref name="oncc0616">[http://hk.on.cc/hk/bkn/cnt/entertainment/20180616/bkn-20180616160838778-0616_00862_001.html 香香約滿無綫　樂易玲：佢要調理生小朋友]</ref>，她另在社交網站貼出千字文感言，逐一感謝TVB高層人員，上至高層，下至清潔嬸嬸和保安員<ref>{{cite web |title=約滿離台 岑麗香發千字感言：俾心機放假，之後再努力工作 |url=http://nextplus.nextmedia.com/news/latest/20180617/595032 |publisher=壹週刊 |date=2018-06-17}}</ref><ref>{{Cite web |url=http://s.nextmedia.com/realtime/a.php?i=20180617&s=7018882&a=58329795 |title=加拿大度假中 岑麗香千字文正式告別TVB 離巢養身造人  |date=2018-06-18 |publisher=蘋果日報}}</ref>，《[[果栏中的江湖大嫂|果栏中的江湖大嫂]]》成为她告别作。同年7月轉投[[王祖藍|王祖藍]]旗下的手工藝創作<ref>{{Cite news |url=https://m.mingpao.com/ldy/showbiz3.php?nodeid=1532164485127&subcate=latest&issue=20180721 |title=岑麗香轉投王祖藍公司︰需要多點私人時間 |date=2018-07-21 |work=明報}}</ref>。

== 感情生活 ==
2015年10月，岑麗香被媒體揭露與6呎高且健碩男士「阿強」拍拖，其經理人公司承認和祝福他們，於2016年12月18日在温哥华举行婚礼<ref>{{cite news|title=岑麗香認戀6呎大隻男|url=http://news.mingpao.com/pns/男/web_tc/article/20151002/s00016/1443722525631|accessdate=2015-10-04|agency=明報|date=2015-10-02}}</ref>。

2018年12月6日，她於社交媒體宣佈已懷孕5個月，預產期為2019年4月。

2019年3月25日，她於instagram貼出兒子Jacob的照片，宣佈一索得男，乳名為「BooBoo」，但未有透露出生日期。<ref name="son"/>
