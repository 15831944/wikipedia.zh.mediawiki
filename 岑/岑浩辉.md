'''岑浩辉'''（Sam Hou Fai；{{bd|1962年|5月|catIdx=M岑}}），出生于中国广东中山<ref>{{cite web|title=中山市人物志——岑浩辉|url=http://www.zsda.gov.cn/plus/view.php?aid=352669|accessdate=2017-06-20|website=中山市档案信息网|deadurl=yes|archiveurl=https://web.archive.org/web/20170914125212/http://www.zsda.gov.cn/plus/view.php?aid=352669|archivedate=2017年9月14日|df=}}</ref>，现任[[澳门特别行政区终审法院|澳门特别行政区终审法院]]院长。他亦同时担任法官委员会主席、推荐法官的独立委员会委员、区际司法协助及国际司法互助工作小组成员、澳门基本法推广协会名誉会长等职务。<ref>{{cite web |title = 澳门终审法院院长岑浩辉专访 |url = http://www.zsdongfeng.gov.cn/news/2009/12/09/1301422.shtml |accessdate = 2017-06-20 |website = 中山市东凤镇人民政府官网 |archive-url = https://web.archive.org/web/20170914125135/http://www.zsdongfeng.gov.cn/news/2009/12/09/1301422.shtml |archive-date = 2017-09-14 |dead-url = yes }}</ref>

== 个人经历 ==
1962年5月，岑浩辉出生于[[广东省|广东省]]中山东凤公社和平大道11队的一户农村家庭，9岁时进入小学就读。1978年初中毕业后，考入[[中山纪念中学|中山纪念中学]]。1981年，岑浩辉以中山市文科状元身份考取了[[北京大学|北京大学]]法律系。大学毕业后，进入位于广州的对外经济律师事务所，担任[[执业律师|执业律师]]<ref>{{cite web |title = 岑浩辉：从中山小伙子到澳门大法官 |url = http://www.zsnews.cn/Culture/2009/12/07/1298006.shtml |accessdate = 2017-06-20 |website = 中山网 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1986年11月，岑浩辉移居澳门生活。

1990年，岑浩辉通过澳门第一次公开招考，作为法律人才赴[[葡萄牙|葡萄牙]][[科英布拉大学|科英布拉大学]]修读语言及法律课程。1993年由葡萄牙返回澳门，1995年成为首批法院司法参事之一，同时在法院及检察院工作。

1999年12月20日，获行政长官任命为澳门特别行政区终审法院院长，并担任法官委员会主席、推荐法官的独立委员会委员、区际司法协助及国际司法互助工作小组成员、澳门基本法推广协会名誉会长<ref>{{cite web|title =终审法院院长岑浩辉|url =http://portal.gov.mo/web/guest/info_detail?infoid=13599|accessdate =2017-06-20|website =澳门特区政府入口网站}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。
