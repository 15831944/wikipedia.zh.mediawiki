'''韋甘比流浪足球會'''（{{lang-en|'''Wycombe Wanderers Football Club'''}}）是位於[[英格蘭|英格蘭]][[英格兰东南|東南部]][[白金汉郡|白金汉郡]]（Buckinghamshire）[[海威科姆|海威科姆]]（High Wycombe）的足球會，近季成績有如升降機，於[[2008年至2009年英格蘭足球乙級聯賽|2009年]]升班到[[英格蘭足球甲級聯賽|英甲]]，一季後即降回[[英格蘭足球乙級聯賽|英乙]]，[[2010年至2011年英格蘭足球乙級聯賽|翌季]]又回升[[英格蘭足球甲級聯賽|英格蘭甲級聯賽]]參賽，但[[2011年至2012年英格蘭足球甲級聯賽|一季後]]又降回[[英格蘭足球乙級聯賽|英格蘭足球乙級聯賽]]。

韦康比流浪者綽號「''Chairboys''」，主場球衣為四分淺藍及深藍格，以[[亚当斯公园球场|亚当斯公园球场]]（Adams Park）作為主場球場。前領隊包括[[凯尔特人足球俱乐部|凯尔特人]]隊長[[保罗·兰伯特|兰伯特]]<ref>[http://news.bbc.co.uk/sport2/hi/football/teams/w/wycombe_wanderers/5132064.stm Lambert named new Wycombe manager]</ref>及曾執教[[英格蘭21歲以下足球代表隊|英格蘭 U-21]]的[[彼得·泰勒|泰勒]]<ref>[http://news.bbc.co.uk/sport2/hi/football/teams/w/wycombe_wanderers/7424474.stm Wycombe name Taylor as new boss]</ref>。

== 主場球場 ==
[[File:Adams_Park_from_a_northerly_direction.JPG|thumb]]
'''亚当斯公园球场'''（'''Adams Park'''）是韋甘比的主場球場，於1990年啟用，初時只可容納6,000名觀眾（其中1,267個為坐席），球場以球隊前隊長法兰·亚当斯（Frank Adams）命名。亚当斯公园球场亦分租給[[聯合式橄欖球|聯合式橄欖球]]隊「倫敦黃蜂」（London Wasps）作為主場。

於2003-2006年球場曾將命名權以10萬[[英鎊|英鎊]]給予當地科技公司「高士威科技」（Causeway Technologies）而更名「''高士威運動場''」（''Causeway Stadium''）。

;看台
「畢治迪恩（北）看台」（Beechdean Stand (North) ）建於1990年，全坐席可容1,267名觀眾。看台設有球會辦公室、更衣室、球會商店和餐食設施。｢帕纳奇（東）看台｣（Panache Stand (East) ）於1990年建成時是一座階梯看台，於1996年改為全坐席，再擴建加大一倍至現時可容2,053名觀眾，並設有10個輪椅座席。看台於比賽日專供作客球迷入座。於2014年7月東看台獲冠名贊助才改稱為｢帕纳奇看台｣。而｢法蘭‧阿當斯（南）看台｣（Frank Adams Stand (South)） 原本是一座單層階梯看台，於1996年擴建成兩層，現時可以容納4,990 名觀眾加50個輪椅座席。南看台設有供企業款待的廂房，而下層則為家庭座席。｢白金漢郡新大學（西）階梯看台｣ （Bucks New Uni Terrace (West) ）可容1,974名觀眾，於比賽日專供主場球迷入座，是場內僅存的階梯看台。

== 主要對手 ==
韋甘比最重要的對手相信是[[科尔切斯特联足球俱乐部|科尔切斯特]]，兩隊並非近鄰的敵對，其惡劣的關係可追溯回1985/86年球季，當時的韋甘比仍然是非聯賽球隊，在一場[[英格蘭足總盃|足總盃]]比賽在舊主場「魯基斯公園球場」（Loakes Park）以2-0擊敗高車士打，對賽中雙方球迷發生衝突。

其後兩隊在1991-92年球季在[[英格蘭足球協會全國聯賽|足球協會聯賽]]全程領放競逐冠軍，高車士打作客「亚当斯公园球场」，在補時階段由門將斯科特·巴雷特（Scott Barrett）一個獲風勢之助的幸運入球以2-1擊敗韋甘比，而高車士打最終憑[[得失球差|得失球差]]壓倒韋甘比獲得冠軍及加入[[英格兰足球联赛|英格兰足球联赛]]。翌年韋甘比才能取回失落的冠軍及升級丙組《第四級》聯賽。

另一次具爭議的比賽發生在1998-99年球季，備受降級威脅的韋甘比在主場「亚当斯公园球场」領先2-1，直到完場前9分鐘，高車士打獲判及射入一個十二碼追成平手，幸好到季尾韋甘比獲得第19位而成功保級。

韋甘比與鄰近地區的球隊如[[雷丁足球俱乐部|雷丁]]、[[沃特福德足球俱乐部|-{zh-hans:沃特福德;zh-hant:屈福特}-]]及[[牛津联足球俱乐部|牛津联]]的競爭相對沒有那麼激烈，原因是大家缺少定期對賽的歷史。而韋甘比在非聯賽時期的主要對手為[[Slough_Town_F.C.|史洛夫]]（Slough Town），在韋甘比獲得1992-93年足球協會全國聯賽的球季，雙方的碰頭分別吸引7,230（韋甘比主場）及4,500（史洛夫主場）名觀眾入座。
