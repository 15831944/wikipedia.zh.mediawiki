'''韦昌进'''（{{bd|1965年|11月||}}），[[江苏|江苏]][[溧水|溧水]]人，[[中国人民解放军|中国人民解放军]]大校，战斗英雄。

==生平==
1983年10月入伍。1985年3月到1986年6月，韦昌进随部队参加[[两山轮战|两山轮战]]。1985年7月，韦昌进所在排奉命坚守某无名高地。1985年7月19日凌晨，[[越南人民军|越南人民军]]以2个营加强1个连的兵力进攻该无名高地，以撕破中国人民解放军防线。韦昌进在战斗中被弹片穿透右胸、击中左眼，全身负伤22处，仍坚持战斗。身旁4位战友先后牺牲后，韦昌进用报话机向上级呼叫：“为了祖国，为了胜利，向我开炮！向我开炮！”引导炮兵先后击退越南人民军8次连排规模反扑，11个小时独自坚守战位，成功守住了阵地，为中国人民解放军获得此次战斗的胜利作出了重大贡献。战斗结束后，韦昌进因伤势过重，昏迷7昼夜，左眼失明。1986年，韦昌进被[[中央军事委员会|中央军委]]授予“战斗英雄”称号。先后6次受[[党和国家领导人|党和国家领导人]]接见，多次应邀参加党和国家重大纪念活动。1991年被国家及军队表彰为“全国自强模范”，2009年被评选为“100位新中国成立后为国防和军队建设作出重大贡献、具有重大影响的先进模范人物”。后来他历任[[中国人民解放军山东省济南警备区|中国人民解放军山东省济南警备区]]政治部副主任<ref>{{Cite web |url = http://www.chinalawedu.com/falvfagui/fg21752/234797.shtml |title = 济南市人民政府办公厅关于济南市人民政府残疾人工作协调委员会更名及调整充实组成人员的通知 |author =  |date =2006-08-29  |publisher = 正保法律教育网 |language =  |accessdate =  |quote =  }}</ref>、[[中国人民解放军山东省泰安军分区|中国人民解放军山东省泰安军分区]]副政治委员<ref>{{Cite web|url =http://news.xinhuanet.com/mil/2011-05/26/c_121458988.htm|title =震撼：战士咬牙把眼珠塞回眼眶呼叫向我开炮|author =|date =2011-05-26|publisher =新华网|language =|accessdate =|quote =}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、中国人民解放军山东省济南警备区副政治委员<ref>{{Cite web |url =http://military.cnr.cn/gz/20170613/t20170613_523797996_4.html  |title = “八一勋章”颁授对象初步候选人简要事迹 |author =  |date =2017-06-13  |publisher = 央广军事 |language =  |accessdate =  |quote =  }}</ref>。2017年6月升任[[中国人民解放军山东省枣庄军分区|中国人民解放军山东省枣庄军分区]]政治委员<ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1718459  |title =  卢泓任枣庄军分区司令员，韦昌进任枣庄军分区政治委员|author =  |date =2017-06-26   |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2017年7月，中央军委主席[[习近平|习近平]]签署命令，授予韦昌进等10位同志“[[八一勋章_(2017年)|八一勋章]]”<ref>{{Cite web |url = http://news.ifeng.com/a/20170728/51522244_0.shtml |title = 景海鹏、程开甲、王刚等10人被授予“八一勋章” |author =  |date = 2017-07-28 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。2017年7月28日，中央军委颁授“八一勋章”和授予荣誉称号仪式在北京[[八一大楼|八一大楼]]举行，中共中央总书记、国家主席、中央军委主席习近平向韦昌进等10位“八一勋章”获得者颁授勋章和证书。这是“八一勋章”首次颁授。韦昌进同时获授“视死如归、血战到底的战斗英雄”荣誉称号<ref>{{Cite web |url = http://news.ifeng.com/a/20170728/51524452_0.shtml |title = 习近平向“八一勋章”获得者颁授勋章和证书 |author =  |date =2017-07-28   |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。2018年12月18日，在[[中共中央|中共中央]]、[[国务院|国务院]]主持的改革开放四十周年大会上表彰为[[改革先锋|改革先锋]]。
