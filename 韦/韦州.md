'''韦州'''，[[西夏|西夏]]时设置的[[州|州]]。

[[唐宣宗|唐宣宗]][[大中|大中]]三年（849年）以[[安乐州|安乐州]]改置[[威州|威州]]，治今[[宁夏回族自治区|宁夏回族自治区]][[同心县|同心县]][[下马关镇|下马关镇]]红城水古城。领鸣沙、温池二县，辖境约当今中宁县、同心县北部，[[盐池县|盐池县]]西部。[[五代|五代]]时，移至灵州方渠镇（今[[甘肃省|甘肃省]][[环县|环县]]）。[[北宋|北宋]][[咸平|咸平]]年间被西夏攻占。西夏在今宁夏同心县东北[[韦州镇|韦州镇]]复置威州，或称韦州、南威州，置韦州静塞监军司，后改祥祐军。韦州东西屏障大山，当南北交通要冲，为军事重镇，宋、金多次攻打。[[蒙古帝国|蒙古帝国]]灭西夏，废韦州。<ref>《[[宋史|宋史]]·夏国传》：“韦州在横山之北，西人恃此为险扼，故立监军司，屯聚兵马，防拓兴、灵等州。”</ref>

[[明代|明代]]庆王[[朱栴|朱栴]]在此建庆王府，设宁夏群牧千户所，亦称韦州群牧千户所。[[清代|清代]]称韦州堡，属平远县。1913年后曾有镇戎县、豫旺县之称。1937年始称同心县。境内有康济寺塔、韦州喇嘛教式墓塔、明庆王墓、匈奴墓、韦州古城等古迹。韦州城位于同心县东北85公里大罗山东麓。城垣犹存，地处青龙山与大螺山之间。此处有两座古城。东城称新城，为[[明孝宗|明孝宗]][[弘治|弘治]]十三年（1500年）[[巡抚|巡抚]]王珣奏筑；西城称老城，西夏所建，为[[西夏|西夏]]十二监军司之一静塞军司驻所。[[韦州古城|韦州古城]]方形，边长约600米。南面设门，筑有瓮城。夯土城墙残高5—8米，上窄下宽，基宽10米。

韦州在抗日战争时期成为陕甘宁边区的一部分，是[[红军西征|红军西征]]夺取重要城镇，由红十五军[[徐海东|徐海东]]部在1936年解放<ref>{{cite web |author1=宁夏自治区党委党史研究室 |title=豫海县回民自治政府——党的民族区域自治的光辉实践 |website=http://cpc.people.com.cn/GB/64093/64387/7765462.html |accessdate=2012-01-13}}</ref>。

红军西征同样开辟了广大的苏区，建立了[[豫海回族自治政府|豫海回族自治政府]]等苏维埃政权，换起了民众，尤其是广大回族群众和宗教界人士的支持，凝聚了抗日力量。<ref>{{cite web |author1=中国革命战争纪实 |title=中国革命战争纪实--西征战役 |website=http://www.rongwp.com/8life/book/changzhen/book36.htm |accessdate=2018-08-11}}</ref>

由于地理环境和交通限制，韦州经济相对贫困。2012年3月，国务院批复同意《陕甘宁革命老区振兴规划》，强调要加快推进包括韦州在内的老区发展，实施优势资源转化战略，将陕甘宁地区建设成为黄土高原生态文明示范区、国家重要能源化工基地、重点红色旅游区、现代旱作农业示范区和基本公共服务均等化试点区，到2020年实现全面建设小康社会目标。<ref>{{cite web |author1=安江林 |title=陕甘宁地区一体化发展的重要战略思路 |website=http://mcrp.macrochina.com.cn/u/64/archives/2012/2302.html |publisher=甘肃省社会科学院 |accessdate=2018-08-01}}</ref>
