'''韦廷王朝'''（'''House of Wettin'''）是源自[[德國|德國]]的一個王朝，此家族成員曾出任[[薩克森|薩克森]]選帝侯、國王以及[[波蘭|波蘭]]國王、[[英国|英国]]国王等。

== 起源 ==
韦廷王朝起源於現在德國[[萨克森-安哈尔特|萨克森-安哈尔特]]州的[[韦廷|韦廷鎮]]，由當時的麦森藩侯（Markgraf von Meißen）創立於1089年。其後韦廷家族成為[[萨克森|萨克森]]地區最有勢力的家族。1263年韦廷家族獲得今日[[图林根|图林根]]地區。在1423年韦廷家族獲得了薩克森[[選帝侯|選帝侯]]之位。1485年韦廷王朝分裂為兩系，即'''恩斯特系'''和'''阿尔布雷希特系'''。

== 恩斯特系 ==
恩斯特系在1485年韦廷王朝分家後，繼承了薩克森選帝侯之位和家族在[[图林根|图林根]]的領地。在宗教戰爭期間，該分支逐漸傾向支持基督[[新教|新教]]。最終，在1547年，薩克森選帝侯的職位和領地，被當時的神聖羅馬皇帝[[查理五世_(神圣罗马帝国)|卡爾五世]]改封與阿尔布雷希特系。

恩斯特系在图林根继续分裂为[[萨克森-魏玛-艾森纳赫|萨克森-魏玛-艾森纳赫]]、[[萨克森-哥达-阿尔滕堡|萨克森-哥达-阿尔滕堡]]、[[萨克森-迈宁根|萨克森-迈宁根]]、[[萨克森-科堡-萨尔费尔德|萨克森-科堡-萨尔费尔德]]等小[[维丁家族恩斯廷系诸邦国|公国]]，1826年，[[萨克森-哥达-阿尔滕堡|萨克森-哥达-阿尔滕堡]]和[[萨克森-科堡-萨尔费尔德|萨克森-科堡-萨尔费尔德]]通过婚姻合并成為萨克森-科堡-哥达公國。此後，該国的王子们陸續在[[比利時|比利時]]，[[葡萄牙|葡萄牙]]，[[保加利亞|保加利亞]]和[[英國|英國]]建立了[[萨克森-科堡-哥达王朝|萨克森-科堡-哥达王朝]]。

== 阿尔布雷希特系 ==
在1485年韦廷王朝分家後，阿尔布雷希特系陸續獲得了[[德累斯顿|德累斯顿]]和[[莱比锡|莱比锡]]附近的薩克森地區。由於該分支恪守天主教信仰，並幫助神聖羅馬皇帝打擊新教。最終於1547年獲封薩克森選帝侯的爵位，自此成為韦廷王朝的正統。而薩克森選侯国更於1806年被法皇[[拿破仑一世|拿破仑一世]]升格為薩克森王國，直至1918年才被推翻，成為魏瑪共和國的一個州。

該王朝的成員亦曾於1697年到1763年獲選為[[波兰君主列表|波蘭國王]]和在1807年至1814年間被法皇拿破崙委任為華沙大公。
