'''韦恩州立大学'''（'''Wayne State University'''，[[简称|简称]]: '''WSU'''）是一所美国公立大学，位于[[密歇根州|密歇根州]][[底特律|底特律]]。2019年《[[美國新聞與世界報道|美國新聞與世界報道]]》將其列在全國第205位，全国公立大学第112位。<ref>{{cite web|title=Wayne State University|url=http://colleges.usnews.rankingsandreviews.com/best-colleges/wayne-state-university-172644/overall-rankings|publisher=U.S. News and World Report|accessdate=2019-01-01}}</ref>

==校園==
[[File:McGregor_Lobby.JPG|thumb]]
維恩州立大學主校區位於底特律，共有100多棟建築，佔地面積{{convert|203|acre|km2}}<ref>{{cite web |author=United States |url=http://www.wayne.edu/about_wayne2.html |title=Wayne State University - About Wayne State University |publisher=Wayne.edu |date=June 23, 2011 |accessdate=December 31, 2012 |archive-url=https://web.archive.org/web/20071012151948/http://wayne.edu/about_wayne2.html |archive-date=2007-10-12 |dead-url=yes }}</ref>。

維恩州立大學圖書館系統下有五座圖書館，總藏書量達到400萬<ref>{{cite web|url=http://www.mla.lib.mi.us/node/1697|publisher=Michigan Library Association|year=2012|accessdate=February 21, 2013|title=Wayne State MLA Spotlight}}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，是美國第75大圖書館系統<ref>{{cite web|url=http://www.ala.org/tools/libfactsheets/alalibraryfactsheet22|publisher=American Library Association|year=2012|accessdate=February 20, 2013|title=The Nation's Largest Libraries: A Listing By Volumes Held}}</ref>。
