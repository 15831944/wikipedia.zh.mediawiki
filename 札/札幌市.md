'''札幌市'''（{{jpn|j=札幌市|hg=さっぽろし|rm=Sapporo shi|f=y}}）是位於[[日本|日本]][[北海道|北海道]][[道央|道央]]地區的[[都市|都市]]，為北海道[[日本都道府縣廳所在地|首府]]、以及[[石狩振興局|石狩振興局]]本部所在地，也是[[政令指定都市|政令指定都市]]之一<ref name="brief">{{cite web |url=http://www.city.sapporo.jp/city/aramashi/ |script-title=ja:札幌市のあらまし |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170702035912/http://www.city.sapporo.jp/city/aramashi/ |archivedate=2017-07-02 }}</ref>。其擁有超過190萬人口，是北海道最大都市、以及日本人口第五大都市，集中了北海道35%的人口，而{{link-ja|札幌都市圈|札幌都市圏}}更集中了北海道約半數的人口。雖然地理位置位於北海道西部，但卻是北海道鐵路、高速公路、航空路線的放射點，因此在交通上也是北海道的中心。

札幌市的歷史開始於1869年日本設立[[開拓使|開拓使]]時在此設置札幌本府<ref name="brief"></ref>。札幌的建設計劃由時任開拓判官[[島義勇|島義勇]]提出，他參考[[京都|京都]]規劃了棋盤狀城市，以{{link-ja|創成橋|創成橋}}東側為基點，將{{link-ja|創成川|創成川}}作為南北軸，渡島通（現南1條通）作為東西軸<ref>{{cite web |url=http://www.city.sapporo.jp/kensetsu/stn/doroshiryo/rekishi.html |script-title=ja:道路建設の歴史 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170630092147/http://www.city.sapporo.jp/kensetsu/stn/doroshiryo/rekishi.html |archivedate=2017-06-30 }}</ref><ref>{{cite web |url=http://www.city.sapporo.jp/chuo/gaiyo/history/documents/4-01.pdf |script-title=ja:さっぽろの街づくりの基点　創成橋 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20150923204140/http://www.city.sapporo.jp/chuo/gaiyo/history/documents/4-01.pdf |archivedate=2015-09-23 }}</ref>。而[[大通公園|大通公園]]則是現在札幌市的東西主幹線<ref>{{cite web |url=http://www.sapporo-park.or.jp/odori/history/ |script-title=ja:大通公園の歴史と植物 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市公園緑化協会}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170614033525/http://www.sapporo-park.or.jp/odori/history/ |archivedate=2017-06-14 }}</ref>。1972年，札幌市舉辦了[[1972年冬季奥林匹克运动会|亞洲首屆冬季奧運會]]<ref name="olympic">{{cite web |url=https://www.olympic.org/sapporo-1972 |script-title=en:SAPPORO 1972 |accessdate=2017-07-04 |publisher=IOC |language=en |deadurl=no |url=http://tiiki.jp/news/wp-content/uploads/2016/10/2016_city_ranking1.pdf |script-title=ja:地域ブランド調査2016＿魅力度上位100市区町村ランキング |accessdate=2017-07-04 |publisher={{lang|ja|ブランド総合研究所}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170926065641/http://tiiki.jp/news/wp-content/uploads/2016/10/2016_city_ranking1.pdf |archivedate=2017-09-26 }}</ref>。2013年，札幌市加入了[[聯合國教科文組織|聯合國教科文組織]][[全球創意城市網絡|全球創意城市網絡]]中的媒體藝術城市，是世界第二個和亞洲首個加盟城市<ref>{{cite web |url=http://www.city.sapporo.jp/kikaku/creativecity/creativecitynetwork/ |script-title=ja:ユネスコ創造都市ネットワーク加盟について |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=yes 札幌市的市徽外觀是一個六角形，代表[[冰晶|冰晶]]（雪花）的形狀，是北海道的象徵。在六角型內側的圓形其實是[[漢字|漢字]]「札」字的變體，而整個圓形又可視為是[[日文|日文]][[片假名|片假名]]「<span lang="ja">ロ</span>」字的變形，是札幌的日語片假名寫法「<span lang="ja">サッポロ</span>」中的一部分。市徽中央的五芒星代表[[北斗星|北斗星]]，是北方的象徵。並且五芒星的形狀又與日文片假名字母「<span lang="ja">ホ</span>」近似，是日語中「北方」或「北海道」的拼音字首<ref name="brief"></ref>。

== 市名由來 ==
如同大部分北海道地區的地名由來，「札幌」這一地名也是起源於北海道原住民[[阿伊努族|阿伊努族]]的語言[[阿伊努語|阿伊努語]]。關於札幌（[[振假名|注音假名]]：{{lang|ja|さっぽろ}}）的名稱起源有二種說法，一說認為起源於阿伊努語詞句「sat-poro-pet」（{{lang|ja|サッ・ポロ・ペッ}}），意指「乾渴的大河」；另一說則認為起源於阿伊努語詞句「sar-poro-pet」（{{lang|ja|サリ・ポロ・ペッ}}），意思是完全與前者顛倒的「有大片濕地的河流」<ref name="brief"></ref>。

== 歷史 ==
[[File:明治24年札幌中心部.jpg|thumb]]
札幌市自約八千年前的[[繩文時代|繩文時代]]就已有人類居住。在17世紀後期的[[津輕藩|津輕藩]]的文獻中就已有關於札幌的記載。18世紀初[[松前藩|松前藩]]製作的地圖中已記載有札幌<ref name="history">{{cite web |url=http://www.city.sapporo.jp/toukei/kanko/documents/4-2.pdf |script-title=ja:札幌市略年表 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20170823205938/http://www.city.sapporo.jp/toukei/kanko/documents/4-2.pdf |archivedate=2017-08-23 }}</ref>。18世紀後期，松前藩設立的石狩十三場所（[[和人|和人]]與[[愛努人|愛努人]]進行貿易的據點）中有部分位於現札幌市內<ref name="history"></ref>。

札幌得到大規模開發則是在19世紀中期之後。1857年時，江戶幕府開始派民眾開拓札幌地區，是和人大規模移民札幌之始<ref name="history"></ref>。1869年日本政府創建開拓使之後在札幌設立札幌本府。由於札幌地處石狩平原，可供開拓的土地廣闊，且地理位置勝過位於北海道南端的[[函館市|函館]]，因此在北海道開拓中的地位迅速提高<ref name="Hyakunen">{{cite book |author={{lang|ja|平岡昭利}}|script-title=ja:《北海道 (地図で読む百年)》 |year=2004 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=4-7722-1419-4|language=ja}}</ref>{{rp|1}}。政府於1871年在札幌興建開拓使本廳舍，象徵札幌成為北海道開拓的中心<ref name="history"></ref>。1870年代是北海道開拓的黎明期，政府在這一時期開通了札幌至函館之間的公路，創建[[札幌農學校|札幌農學校]]（現[[北海道大學|北海道大學]]的前身），並通過{{link-ja|屯田兵|屯田兵}}制度擴大移民規模<ref name="history"></ref>。1880年，{{link-ja|手宮站|手宮駅|手宮}}至[[札幌站|札幌]]的鐵路通車，這也是北海道的首條鐵路<ref name="history"></ref>。1882年，日本政府廢除開拓使，改設函館縣、札幌縣、根室縣。這一體制只持續四年後就被北海道廳取代，而札幌成為北海道廳的所在地<ref name="history"></ref>。1899年，札幌、函館、[[小樽市|小樽]]設區，相當於現在的市<ref name="history"></ref>。

[[File:SapporoNanboku3000and5000.jpeg|thumb]]南北線]]
1918年，札幌舉辦了開道50年紀念北海道博覽會並開通了[[札幌市電|路面電車]]。同年北海道大學亦正式設立<ref name="history"></ref>。1922年，札幌正式設市，但人口在當時少於函館市和小樽市，是北海道的第三大城市<ref name="Chishi">{{cite book |author={{lang|ja|山下克彦・平川一臣}}|script-title=ja:《日本の地誌3　北海道》 |year=2011 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=978-4-254-16763-4 |language=ja}}</ref>{{rp|235}}。然而札幌在1920年代後迅速發展，在1940年時的人口已經幾乎比1920年翻倍，成為北海道人口最多的城市<ref name="Chishi"></ref>{{rp|235}}。在[[第二次世界大戰|第二次世界大戰]]期間，札幌的丘珠、白石、苗穗等地亦遭到了美軍空襲<ref>{{cite web |url=http://www.city.sapporo.jp/ncms/shimin/heiwa/rekishi_senseki/senseki/senseki_21/ |script-title=ja:札幌空襲（手稲軽川） |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170808194401/http://www.city.sapporo.jp/ncms/shimin/heiwa/rekishi_senseki/senseki/senseki_21/ |archivedate=2017-08-08 }}</ref>。

札幌市於戰後在1947年進行了首次市長普選，並在1950年舉辦了首次[[札幌雪祭|札幌雪祭]]<ref name="history"></ref>。戰後的札幌市在1950年代經歷了多次市町村合併，轄區範圍明顯擴大<ref name="history"></ref>。1964年，包括札幌市在內的道央地區被指定為新興產業都市，也促進了札幌的工業發展<ref name="history"></ref>。進入1970年代後，札幌市的人口突破了百萬大關，並在1972年舉辦了[[1972年冬季奧林匹克運動會|冬季奧運會]]<ref name="history"></ref>。冬奧會對於札幌的都市發展有很正面的影響。札幌市為此不僅興建了眾多體育設施，還在市中心興建了地下街並且成為日本第四個開通地下鐵的城市<ref name="Hyakunen"></ref>{{rp|8}}。冬奧會之後，札幌市在1976年制定了新札幌市長期綜合計劃，繼續興建地下鐵並實施鐵路高架化等工程<ref name="history"></ref>。1980年代初期，札幌市人口突破150萬大關，並超過[[京都市|京都市]]成為日本人口第五多城市<ref name="history"></ref>。1990年代，札幌市舉辦了[[亞洲冬季運動會|亞洲冬季運動會]]和冬季[[世界大學生運動會|世界大學生運動會]]等大型體育賽事，且地下鐵建設亦取得進展，都市規模繼續擴大<ref name="history"></ref>。然而都市的發展亦帶來交通堵塞等問題的發生。1990年代後，北海道總人口中札幌市的比例已經超過三成，顯示北海道內札幌市的中心地位日益提高<ref name="Hyakunen"></ref>{{rp|8}}。

札幌市位於[[石狩平原|石狩平原]]西南部，市轄區面積1,121.26平方公里<ref name="brief"></ref>。札幌市按照地形可以分為西南部山區、東南部台地、中部沖積扇、北部低地四個地區<ref name="geography">{{cite web|url=https://www.city.sapporo.jp/kikaku/kyoujinka/documents/sankou01_gaikyou.pdf|script-title=ja:札幌市の概況（地域特性）|accessdate=2017-07-04|publisher={{lang|ja|札幌市}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20180327031338/https://www.city.sapporo.jp/kikaku/kyoujinka/documents/sankou01_gaikyou.pdf|archivedate=2018-03-27}}</ref>。札幌市西南部地區以山地地形為主，其中靠近市區的部分標高約300米至500米，代表性山峰有[[藻岩山|藻岩山]]。市轄區內最高峰是{{link-ja|余市岳|余市岳}}，標高1,488米<ref name="brief"></ref>。西南部的山地中還有[[手稻山|手稻山]]等火山性山地<ref name="geography"></ref>，部分地區是[[支笏洞爺國立公園|支笏洞爺國立公園]]的一部分<ref>{{cite web |url=http://www.env.go.jp/park/shikotsu/point/index.html |script-title=ja:公園の特長 |accessdate=2017-07-04 |publisher={{lang|ja|環境省}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171105054753/http://www.env.go.jp/park/shikotsu/point/index.html |archivedate=2017-11-05 }}</ref>。市區東南部的丘陵地區地形起伏則較為平緩，但南部的標高較高<ref name="geography"></ref>。札幌市轄區中央的沖積扇是由豐平川、發寒川沖積形成的低地，也是札幌市人口最集中的地區<ref name="geography"></ref>。市轄區北部的低地是石狩平原的一部分，由豐平川、發寒川、石狩川沖積形成<ref name="geography"></ref>。

[[File:From_Mt_Moiwa_Sapporo01s3200.jpg|thumb]]
札幌市轄區的大部分範圍屬於[[石狩川|石狩川]]水系，市中心的[[豐平川|豐平川]]等河流也是石狩川的支流。此外札幌市還有{{link-ja|新川 (北海道)|新川 (北海道)|新川}}、[[星置川|星置川]]兩條二級水系<ref>{{cite web |url=http://www.city.sapporo.jp/kensetsu/kasen/menu02-01.html |script-title=ja:札幌市の河川分類 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20161114111017/http://www.city.sapporo.jp/kensetsu/kasen/menu02-01.html |archivedate=2016-11-14 }}</ref>。

=== 气候 ===
札幌市属于[[柯本气候分类法|柯本气候分类法]]下的[[温带大陆性湿润气候|温带大陆性湿润气候]]，夏季和冬季的温差很大，在日本的氣候分區中則屬於[[日本海側氣候|日本海側氣候]]<ref name="brief"></ref>。在4月至6月期間，札幌市天氣多晴天，並無日本其他地區的[[梅雨|梅雨]]時期。札幌市夏季受鄂霍茨克海高氣壓的影響，雖然白天氣溫較高但早晚則氣候涼爽。冬季的札幌则较为寒冷且多雪，最深積雪量可達1米，全年降雪量累計可達6米<ref name="geography"></ref>。札幌市年平均氣溫有8.9℃，夏季平均最高氣溫有26.4℃，冬季最低平均氣溫有-7℃，是世界少有的位於豪雪地帶卻有近200萬人口的都市<ref>{{cite web |url=http://www.sapporo.travel/learn/basicinfo/ |script-title=ja:さっぽろ早わかり |accessdate=2017-07-04 |publisher={{lang|ja|札幌市経済観光局}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170703191944/http://www.sapporo.travel/learn/basicinfo/ |archivedate=2017-07-03 }}</ref>。

== 人口 ==
1920年日本首次人口普查時，札幌市有人口102,580人，是當時北海道僅次於函館、小樽的第三大城市<ref name="Chishi"></ref>{{rp|235}}。然而此後札幌人口快速增加，1940年時已達206,103人，成為北海道人口最多城市<ref name="Chishi"></ref>{{rp|235}}。二戰結束時，札幌市有人口220,139人<ref name="Chishi"></ref>{{rp|235}}。此後由於市轄區範圍的擴大和北海道各地人口的流入，札幌市人口急速增長。1970年時，札幌市人口突破百萬大關，成為日本第八個百萬人口都市<ref name="Chishi"></ref>{{rp|235}}。然而自1970年代後期開始，由於城市的郊外化，札幌的人口增長速度有所減緩，但仍在1980年代前期突破150萬大關並成為日本人口第五多城市。1990年代之後，隨著人口高齡化、少子化的進展，札幌市人口增長速度進一步減緩。2017年6月1日時，札幌市有人口1,962,064人<ref>{{cite web |url=http://www.city.sapporo.jp/toukei/jinko/jinko.html |script-title=ja:人口統計 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170719142330/http://www.city.sapporo.jp/toukei/jinko/jinko.html |archivedate=2017-07-19 }}</ref>。北海道人口中札幌市的比例則從1920年的4.3%大幅增加到2005年的33.4%，且這一比例仍在提高<ref name="Chishi"></ref>{{rp|235}}。由於高齡化、少子化問題，札幌市人口的自然增長率自2009年開始已轉為負數，現在札幌市的人口增長依賴外地人口的流入<ref name="population">{{cite web |url=http://www.city.sapporo.jp/kikaku/miraisousei/documents/plan-3.pdf |script-title=ja:人口ビジョン編 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20160804202436/http://www.city.sapporo.jp/kikaku/miraisousei/documents/plan-3.pdf |archivedate=2016-08-04 }}</ref>。札幌市在1990年代之前每年有超過1萬人流入，1990年代之後雖大幅減少，但在2008年之後有所上升<ref name="population"></ref>。2014年，札幌市的[[總和生育率|總和生育率]]只有1.16，和[[東京都|東京都]]（1.15）相近，大幅低於北海道（1.27）和日本（1.42）平均值<ref name="population"></ref>。

== 政治 ==
在日本[[眾議院_(日本)|眾議院]]選舉中，札幌市中央區、南區、西區的大部分、北區的一部分屬於[[北海道第1區|北海道第1區]]；北區的大部分與東區屬於[[北海道第2區|北海道第2區]]；白石區、豐平區、清田區屬於[[北海道第3區|北海道第3區]]；西區的一部分、手稻區和[[後志綜合振興局|後志綜合振興局]]屬於[[北海道第4區|北海道第4區]]；厚別區和[[石狩振興局|石狩振興局]]其他地區屬於[[北海道第5區|北海道第5區]]<ref>{{cite web|url=http://www.soumu.go.jp/main_content/000240039.pdf|title=|accessdate=2017-10-22|author=|date=|publisher={{lang|ja|総務省}}|language=ja|script-title=ja:北海道}}</ref>。北海道1、2、3區在過去是[[民主黨_(1998年)|民主黨]]較強的選區，然而由於民主黨的支持率長期低迷，在2014年[[第47屆日本眾議院議員總選舉|第47屆日本眾議院議員總選舉]]中，民主黨候選人在北海道1區當選，而[[自由民主黨_(日本)|自民黨]]候選人在北海道2、3區當選<ref>{{cite web |url=http://www.yomiuri.co.jp/election/shugiin/2014/kaihyou/ya01.html　|script-title=ja:【小選挙区】北海道 |accessdate=2017-07-04|publisher={{lang|ja|読売新聞}} |language=ja}}</ref>。2017年[[第48屆日本眾議院議員總選舉|第48屆日本眾議院議員總選舉]]時，民主黨的後繼政黨之一[[立憲民主黨|立憲民主黨]]候選人在北海道1、3區當選，其他選區則是自民黨候選人當選<ref>{{cite web|url=http://www.yomiuri.co.jp/election/shugiin/2017/kaihyou/ya01.html　|script-title=ja:【小選挙区】北海道|accessdate=2017-10-24|publisher={{lang|ja|読売新聞}}|language=ja}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 經濟 ==
[[File:Sapporo_Beer_Museum.JPG|thumb]]]]
[[File:白い恋人パーク_-_panoramio_(4).jpg|thumb]]博物館]]
2014年度，札幌市市內生產總值有65,478億日元，人均市民所得有261.2萬日元。札幌市經濟的特點之一是第二產業比重很低，只佔經濟總量的10.7%。而第三產業則有極重要的地位，佔經濟總量的88%<ref>{{cite web |url=http://www.city.sapporo.jp/toukei/sna/documents/h26-snagaiyou.pdf |script-title=ja:平成 26 年度札幌市民経済計算 結果の概要 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20170918013207/http://www.city.sapporo.jp/toukei/sna/documents/h26-snagaiyou.pdf |archivedate=2017-09-18 |title=存档副本 }}</ref>。
=== 第一產業 ===
札幌市耕地面積2,750公頃，農業就業人口有932人。農業生產金額有34.5億日元，其中蔬菜佔約七成的比例，呈現典型的都市近郊型農業的特徵<ref>{{cite web |url=http://www.machimura.maff.go.jp/machi/contents/01/100/index.html |script-title=ja:北海道札幌市 |accessdate=2017-07-04 |publisher={{lang|ja|農林水産省}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170808194208/http://www.machimura.maff.go.jp/machi/contents/01/100/index.html |archivedate=2017-08-08 }}</ref>。札幌市生產的主要蔬菜種類有[[洋蔥|洋蔥]]、[[菠菜|菠菜]]、[[馬鈴薯|馬鈴薯]]等<ref>{{cite web |url=http://www.city.sapporo.jp/keizai/nogyo/gaiyou/gaiyou.html |script-title=ja:大地が育てた札幌農業 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20170808194449/http://www.city.sapporo.jp/keizai/nogyo/gaiyou/gaiyou.html |archivedate=2017年8月8日 |df= }}</ref>，札幌黄是札幌生產的著名洋蔥品種<ref>{{cite web |url=http://www.city.sapporo.jp/higashi/about/sapporoki/ |script-title=ja:東区特産のたまねぎ札幌黄 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170808194145/http://www.city.sapporo.jp/higashi/about/sapporoki/ |archivedate=2017-08-08 }}</ref>。近年由於農業人口高齡化，札幌市耕地中有三成是耕作放棄地。札幌市已通過市民農園等方式試圖解決這一問題<ref>{{cite web |url=http://www.city.sapporo.jp/keizai/nogyo/vision/documents/part3.pdf |script-title=ja:札幌市の農業の現状と課題 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170808193314/http://www.city.sapporo.jp/keizai/nogyo/vision/documents/part3.pdf |archivedate=2017-08-08 }}</ref>。

== 交通 ==
[[File:Sapporo_station_2012.jpg|thumb]]
札幌是北海道的政經中心，交通亦較為便利。札幌市內的普通鐵路由[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）運營，市內有[[函館本線|函館本線]]、[[札沼線|札沼線]]（學園都市線）、[[千歲線|千歲線]]三條JR路線<ref>{{cite web |url=https://www.jrhokkaido.co.jp/network/map/pdf/sa_area.pdf |script-title=ja:路線図 |accessdate=2017-07-04 |publisher={{lang|ja|JR北海道}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170622194608/http://www.jrhokkaido.co.jp/network/map/pdf/sa_area.pdf |archivedate=2017-06-22 }}</ref>。[[札幌站|札幌站]]是札幌的鐵路中心，2015年日均客流量達9.5萬人次<ref>{{cite web |url=http://www.city.sapporo.jp/sogokotsu/kotsutaikei/documents/databook_3.pdf |script-title=ja:３.ＪＲ |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170818094412/http://www.city.sapporo.jp/sogokotsu/kotsutaikei/documents/databook_3.pdf |archivedate=2017-08-18 }}</ref>。[[札幌市交通局|札幌市交通局]]是[[札幌市營地下鐵|札幌市營地下鐵]]和[[札幌市電|札幌市電]]的經營者。札幌市營地下鐵包括[[南北線_(札幌市營地下鐵)|南北線]]、[[東西線_(札幌市營地下鐵)|東西線]]、[[東豐線|東豐線]]三條路線，全長48公里<ref>{{cite web |url=http://www.city.sapporo.jp/st/subway/gaiyo/gaiyo.html |script-title=ja:地下鉄（高速電車）の概要 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170315041010/http://www.city.sapporo.jp/st/subway/gaiyo/gaiyo.html |archivedate=2017-03-15 }}</ref>。札幌市電是札幌市的路面電車系統，曾在札幌市公共交通中有重要地位，但隨著汽車的普及而廢除了不少路線，現僅有一條路線<ref>{{cite web |url=http://www.city.sapporo.jp/st/shiden/gaiyo/gaiyo.html |script-title=ja:市電（路面電車）の概要 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170315041218/http://www.city.sapporo.jp/st/shiden/gaiyo/gaiyo.html |archivedate=2017-03-15 }}</ref>。2015年度，札幌市營地下鐵客流量達2.2億人次，札幌市電客流量有835萬人次<ref>{{cite web |url=http://www.city.sapporo.jp/st/study/jigyo-gaiyo/documents/h2801.pdf |script-title=ja:第１編 総 括 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市}} |language=ja }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

札幌市內有[[道央自動車道|道央自動車道]]和[[札樽自動車道|札樽自動車道]]兩條高速公路<ref>{{cite web |url=http://www.jaf.or.jp/member/dguide/exway/pdf/hokkaido/Hokkaido_ex_or.pdf |script-title=ja:高速道路案内図 |accessdate=2017-07-04 |publisher={{lang|ja|日本自動車連盟}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20150616022044/http://www.jaf.or.jp/member/dguide/exway/pdf/hokkaido/Hokkaido_ex_or.pdf |archivedate=2015-06-16 }}</ref>。高速巴士是連接札幌市和北海道內及本州北部主要都市之間重要的交通方式，市內主要巴士樞紐有[[札幌車站客運總站|札幌車站客運總站]]、{{link-ja|北海道中央巴士札幌客運總站|北海道中央バス札幌ターミナル}}等地。[[JR北海道巴士|JR北海道巴士]]<ref>{{cite web |url=http://www.jrhokkaidobus.com/timetable/ |script-title=ja:時刻表・路線図 |accessdate=2017-07-04 |publisher={{lang|ja|ジェイ・アール北海道バス}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170706130602/http://www.jrhokkaidobus.com/timetable/ |archivedate=2017-07-06 }}</ref>、{{link-ja|北海道中央巴士|北海道中央バス}}<ref>{{cite web |url=http://www.chuo-bus.co.jp/highway/ |script-title=ja:札幌から北海道内各地へネットワーク |accessdate=2017-07-04 |publisher={{lang|ja|北海道中央バス}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170625190815/http://www.chuo-bus.co.jp/highway/ |archivedate=2017-06-25 }}</ref>是札幌市主要的巴士業者。

札幌市內有[[札幌飛行場|丘珠機場]]一座機場。丘珠機場設立於1942年，是一座軍民共用機場。丘珠機場現在只有前往[[利尻機場|利尻機場]]、[[釧路機場|釧路機場]]、[[函館機場|函館機場]]、[[三澤飛行場|三澤機場]]、[[靜岡機場|靜岡機場]]的定期航線，2016年度的乘客數有約21.4萬人次<ref>{{cite web |url=http://www.okadama-airport.co.jp/airport/ |script-title=ja:空港概要 |accessdate=2017-07-04 |publisher={{lang|ja|札幌丘珠空港}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170902002209/http://www.okadama-airport.co.jp/airport/ |archivedate=2017-09-02 }}</ref>。位於[[千歲市|千歲市]]、[[苫小牧市|苫小牧市]]交界處的[[新千歲機場|新千歲機場]]則是札幌市現在實質上的主要機場。新千歲機場開通有前往日本各主要機場和部分東亞主要機場的航線，是日本最繁忙的機場之一。

== 觀光 ==
[[File:Odori_Park_Sapporo.JPG|thumb]]
[[File:US_Navy_060209-N-7526R-205_Illuminated_trees_and_ice_sculptures_line_the_streets_leading_up_to_Sapporo's_TV_Tower_during_the_Sapporo_Ice_Festival.jpg|thumb]]]]
札幌市觀光資源十分豐富。[[大通公園|大通公園]]是日本最具代表性的都市公園之一，東西長約1.5公里，面積達7.8公頃<ref>{{cite web |url=http://www.sapporo-park.or.jp/odori/guide/ |script-title=ja:公園案内 |accessdate=2017-07-04 |publisher={{lang|ja|大通公園}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170614012556/http://www.sapporo-park.or.jp/odori/guide/ |archivedate=2017-06-14 }}</ref>。竣工於1957年的[[札幌電視塔|札幌電視塔]]亦位於大通公園內，在其高90.38米的瞭望台可一覽札幌全市風景<ref>{{cite web |url=http://www.tv-tower.co.jp/guidance/ |script-title=ja:展望台のご案内 |accessdate=2017-07-04 |publisher={{lang|ja|さっぽろテレビ塔}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170709133130/http://www.tv-tower.co.jp/guidance/ |archivedate=2017-07-09 }}</ref>。札幌市中心的主要公園還有{{link-ja|中島公園|中島公園}}<ref>{{cite web |url=https://www.sapporo-park.or.jp/nakajima/?page_id=9 |script-title=ja:公園案内 |accessdate=2017-07-04 |publisher={{lang|ja|中島公園}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170729041439/http://www.sapporo-park.or.jp/nakajima/?page_id=9 |archivedate=2017-07-29 }}</ref>、[[圓山公園_(札幌市)|圓山公園]]<ref>{{cite web |url=http://maruyamapark.jp/?page_id=1566 |script-title=ja:円山公園の歴史 |accessdate=2017-07-04 |publisher={{lang|ja|円山公園}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170718153109/http://maruyamapark.jp/?page_id=1566 |archivedate=2017-07-18 }}</ref>等地。位於札幌市近郊的藻岩山不僅是滑雪勝地，自山頂看到的夜景和[[長崎市|長崎]]、神戶的夜景並列為新日本三大夜景<ref>{{cite web |url=http://moiwa.sapporo-dc.co.jp/nightview/ |script-title=ja:藻岩山の夜景 |accessdate=2017-07-04 |publisher={{lang|ja|札幌もいわ山ロープウェイ}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170905185252/http://moiwa.sapporo-dc.co.jp/nightview/ |archivedate=2017-09-05 }}</ref>。

札幌市保留有眾多北海道開拓時代的西洋式建築，代表建築有[[札幌市鐘樓|札幌市鐘樓]]<ref>{{cite web |url=http://www.sapporo.travel/find/culture/clock_tower/ |script-title=ja:札幌市時計台 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市経済観光局}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170718230514/http://www.sapporo.travel/find/culture/clock_tower/ |archivedate=2017-07-18 }}</ref>、[[北海道廳舊本廳舍|北海道廳舊本廳舍]]<ref>{{cite web|url=http://www.sapporo.travel/find/culture/former_hokkaido_government_office/|script-title=ja:北海道庁旧本庁舎（赤れんが庁舎）|accessdate=2017-07-04|publisher={{lang|ja|札幌市経済観光局}}|language=ja|deadurl=no|archiveurl=https://web.archive.org/web/20170713115143/http://www.sapporo.travel/find/culture/former_hokkaido_government_office/|archivedate=2017-07-13}}</ref>、[[豐平館|豐平館]]<ref>{{cite web |url=http://www.s-hoheikan.jp/about |script-title=ja:豊平館について |accessdate=2017-07-04 |publisher={{lang|ja|豊平館}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20160923095715/http://www.s-hoheikan.jp/about |archivedate=2016-09-23 }}</ref>等地。位於厚別區的[[北海道開拓村|北海道開拓村]]保存有眾多開拓時期的珍貴建築，是一座露天博物館<ref>{{cite web |url=http://www.kaitaku.or.jp/go/goannai.htm |script-title=ja:開拓の村のあらまし |accessdate=2017-07-04 |publisher={{lang|ja|北海道開拓の村}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170706184513/http://www.kaitaku.or.jp/go/goannai.htm |archivedate=2017-07-06 }}</ref>。白色戀人公園展示有札幌著名甜點[[白色戀人|白色戀人]]的生產過程<ref>{{cite web |url=http://www.sapporo.travel/find/recreational/shiroi_koibito_park/ |script-title=ja:白い恋人パーク |accessdate=2017-07-04 |publisher={{lang|ja|札幌市経済観光局}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170615173300/http://www.sapporo.travel/find/recreational/shiroi_koibito_park/ |archivedate=2017-06-15 }}</ref>。

[[札幌雪祭|札幌雪祭]]於每年2月在大通公園舉行，於1950年開始舉辦，是北海道冬季規模最大的觀光活動之一<ref>{{cite web |url=http://www.snowfes.com/about/history.html |script-title=ja:さっぽろ雪まつりの歴史 |accessdate=2017-07-04 |publisher={{lang|ja|さっぽろ雪まつり}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170713093653/http://www.snowfes.com/about/history.html |archivedate=2017-07-13 }}</ref>。自1992年開始舉行的[[YOSAKOI索朗祭|YOSAKOI索朗祭]]則是札幌市夏季最大的觀光活動<ref>{{cite web |url=http://www.yosakoi-soran.jp/about.html |script-title=ja:YOSAKOIソーラン祭りって？ |accessdate=2017-07-04 |publisher={{lang|ja|YOSAKOIソーラン祭り}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170630112041/http://www.yosakoi-soran.jp/about.html |archivedate=2017-06-30 }}</ref>。[[定山溪溫泉|定山溪溫泉]]位於札幌市郊區，是札幌市最大的溫泉度假地，已有超過150年的歷史，是北海道歷史較悠久的溫泉度假地之一<ref>{{cite web |url=http://www.sapporo.travel/find/recreational/jozankei_onsen/ |script-title=ja:定山渓温泉街 |accessdate=2017-07-04 |publisher={{lang|ja|札幌市経済観光局}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170705215139/http://www.sapporo.travel/find/recreational/jozankei_onsen/ |archivedate=2017-07-05 }}</ref>。
