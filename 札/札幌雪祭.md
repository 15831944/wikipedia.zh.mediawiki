'''札幌雪祭'''（{{lang-ja|さっぽろ雪まつり}}）是每年2月上旬在[[北海道|北海道]][[札幌市|札幌市]][[大通公園|大通公園]]等地舉辦的[[雪|雪]]和冰的[[祭|祭典]]。札幌雪祭主要展示各種用雪製作的雕刻，在[[薄野|薄野]]會場等地也展示冰雕。札幌雪祭由[[札幌市政府|札幌市]]和札幌商工會議所、市內的企業、團體等構成的「札幌雪祭實行委員会」企劃運營。札幌雪祭每年都吸引眾多來自日本各地和外國的遊客參觀，是北海道規模最大的活動之一。2010年起，「[[雪初音|雪初音]]（{{lang-ja|雪ミク}}）」成為雪祭的應援角色和代言人<ref name="2017_Sapporo_Asian_winter_games">{{Cite web |url = https://kknews.cc/comic/52yj2l.html |title = 雪初音出任2017年度亞洲冬季運動會宣傳大使 - 每日頭條 |author =  |date = 2016-02-05 |publisher =  |language = zh-tw |accessdate = 2016-10-18 |quote =  }}</ref>。
