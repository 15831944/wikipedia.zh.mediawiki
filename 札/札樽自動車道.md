'''札樽自動車道'''（{{jpn|j=札樽自動車道|hg=さっそんじどうしゃどう|rm=Sasson jidōshadō|f=y}}）是從[[北海道|北海道]][[小樽市|小樽市]]至[[札幌市|札幌市]]，總長約38.3[[公里|公里]]的[[日本高速公路|高速公路]]（[[高速自動車國道|高速自動車國道]]），略稱為'''札樽道'''（{{lang|ja|札樽道／さっそんどう}}，SASSON EXPWY），屬[[國土開發幹線自動車道|國土開發幹線自動車道]]中{{link-ja|北海道橫斷自動車道|北海道横断自動車道}}的一部份。高速公路路線編號被編為{{Ja Exp Route Sign|E5A}}。

== 概要 ==
本自動車道在[[札幌西IC|札幌西IC]]-札幌JCT間為位於{{link-ja|札幌新道|札幌新道}}（[[國道274號|國道274號]]、[[國道5號_(日本)|國道5號]]）中央分離帶上的高架橋。

由於急彎道比較多，全線最高速度為[[時速|時速]]80km。

== 歷史 ==
* 1971年12月4日 : 國道5號'''札幌小樽道路（札樽支道）'''小樽IC-札幌西IC間通車，為2車道。
* 1973年4月1日 : 札幌小樽道路昇格為[[高速自動車國道|高速自動車國道]]的'''札樽自動車道'''。
* 1974年8月22日 : 小樽IC-札幌西IC間4車道化。
* 1992年9月30日 : 札幌西IC-札幌JCT間通車，連接道央自動車道。
* 2005年10月1日 : 日本道路公團民營化，成為[[東日本高速道路|東日本高速道路株式會社]]管轄路線。
