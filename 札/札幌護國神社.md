'''札幌護國神社位於'''[[北海道|北海道]][[札幌市|札幌市]]中央區，為[[護國神社|護國神社]]之一。

為了祭祀[[1879年|在1879]] 年(明治的12年)，[[西南戰爭|西南戰爭]]中戰死的屯田兵而建立 <ref name="introduce">[http://www3.ocn.ne.jp/~gokoku30/goannnai1.htm 神社紹介]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }} – 札幌護国神社公式ホームページ、2012年10月2日閲覧。</ref>，之後各個戰爭中的亡靈亦一同祭祀，但是[[1949年|在1949年]] (昭和24年)9月時，祈請了[[多賀大社|多賀大社]]祭神，接續[[1971年|在1971年]] (昭和46年)9月時與鎮守山鼻地區([[札幌郡|札幌郡]]山鼻村) 的山鼻神社之祭神合祭，一直到了現今。

因此，該神社由祭祀創建時祭神之本殿、祭祀祈請之多賀大社諸祭神與山鼻神社祭神的'''多賀殿 '''(多賀神社)組成。

== 沿革 ==
* 1879年(明治12年)–於現在的札幌市北區北6條西7丁目建立了屯田兵招魂纪念碑，被稱為札幌忠魂社，為札幌護國神社的開始 。
* 1911年(明治44年)2月– 移至中島公園。
* 1922年(大正11年)7月1日–重新命名為札幌招魂社。
* 1933年(昭和8年)–於現在地建立社殿。
* 1939年(昭和14年)4月1日–改名為札幌護國神社，且由内政部部長指定為護國神社<ref name="jinjacho">[http://www.hokkaidojinjacho.jp/data/01/01009.html 札幌護国神社] – 北海道神社庁、2012年10月2日閲覧。</ref>。
* 1946年(昭和21年)
** 4月30日– 所屬於神社本廳。
** 12月12日–改名為札幌彰德神社(日本獨立經承認後，重新被命名為札幌護國神社)。
* 1949年(昭和24年)9月–祈請多賀大社之祭神，建立多賀殿 。
* 1968年(昭和43年)–社殿因火災燒毀滅失 。
* 1970年(昭和45年)–重建社殿。
* 1971年(昭和46年)9月–於多賀殿合祭山鼻神社祭神 。
* 1972年(昭和47年)2月3日– [[昭和天皇|昭和天皇]]及[[香淳皇后|香淳皇后]] 親自參拜 。

== 彰德苑 ==
於境內角落，存有以戰歿者慰靈碑為中心，集結了許多紀念碑的「彰德苑」。
; 屯田兵招魂碑
: [[1878年|1878年]]（明治11年）7月，為於[[西南戰爭|西南戰爭]]中戰死的屯田兵所建，應為札幌市内現存最古老的石碑{{Sfn|吉岡|loc=No.1}}。
: 最初位於偕楽園。[[1907年|1907年]]（明治40年）12月，與[[日俄战争|日露戰爭]]中來自札幌連隊區管的戰死者合祭，移至中島遊園地。[[1933年|1933年]]（昭和8年），再移轉至招魂社境内{{Sfn|吉岡|loc=No.1}}。
; 忠魂碑
: 為日露戰爭的戰死者所建。書法題字者為[[乃木希典|乃木希典]]{{Sfn|吉岡|loc=No.1}}。
; 頌德碑
: 表彰財團法人帝國在鄉軍人會創立者松本謄四郎之碑。起初位於旭丘，但[[1960年|1960年]]（昭和35年）3月移至本神社{{Sfn|吉岡|loc=No.11}}。
: 稱頌松本的正面碑文由佐藤昌介所撰。[[1967年|1967年]]（昭和42年）時，則在右側銘刻弔慰在鄉軍人會遺族的文字{{Sfn|吉岡|loc=No.11}}。
; 尼港殉難碑
: 為[[尼港事件|尼港事件]]犠牲者所建之碑。[[1928年|1928年]]（昭和3年）10月建於旭丘，但1960年（昭和35年）3月5日移至本神社{{Sfn|吉岡|loc=No.15}}。
; 諾門罕英魂之碑
: 為[[諾門罕戰役|諾門罕戰役]]犠牲者所建之碑，於1967年（昭和42年）9月16日建立。[[1987年|1987年]]（昭和62年）時增加非洲產的黒御影石所製由来碑{{Sfn|吉岡|loc=No.36}}。
; 阿圖島玉碎雄魂之碑
: 為[[阿圖島戰役|阿圖島戰役]]犠牲者所建之碑，於[[1968年|1968年]]（昭和43年）7月29日建立。寬7公尺的座台上，承載有32噸的神居古壇石碑{{Sfn|吉岡|loc=No.42}}。
; 梅萊詠島戰歿者慰靈碑
: 為在[[沃萊艾環礁|沃萊艾環礁]]犠牲者所建之碑，[[1971年|1971年]]（昭和46年）10月建立。以黒御影石所製，題字者為堂垣内尚弘。左後方立有戰歿者的姓名石碑{{Sfn|吉岡|loc=No.48}}。
; 北千島慰靈碑
: 為[[千島群島|北千島]]犠牲者所建之碑，[[1975年|1975年]]（昭和50年）8月23日建立。以日高産碑原石所製，題字者為堂垣内尚弘{{Sfn|吉岡|loc=No.54}}。
; 北海道全海軍英魂之碑
: 為與北海道・千島・樺太有關的所有海軍關係者所建之碑，於[[1978年|1978年]]（昭和53年）8月20日建立。置有重巡洋艦曾使用的2個船錨2{{Sfn|吉岡|loc=No.56}}。
; 「北鎮砲兵發祥地」之碑
: 為紀念[[1896年|1896年]]（明治29年）12月組成之獨立野戰砲大隊所建之碑，於[[1985年|1985年]]（昭和60年）6月23日建立。非洲産的黒御影石所製，題字者為板垣武四{{Sfn|吉岡|loc=No.63}}。
; 旭川第七連隊砲兵哨舍
: 於「北鎮砲兵發祥地」碑建立的數年前，由自衛隊所搬運{{Sfn|吉岡|loc=No.63}}。
; 步兵第26連隊軍旗奉燒之碑
: 為紀念隨著[[步兵第26聯隊|步兵第26連隊]]解散，在月寒神社奉燒軍旗之碑，於[[1986年|1986年]]（昭和61年）9月14日建立。由白御影石所製，並有沿革碑{{Sfn|吉岡|loc=No.66}}。
