{{vgname|星之卡比 新星同盟|ja=星のカービィ スターアライズ|en=Kirby Star Allies|}}是由[[HAL研究所|HAL研究所]]开发，[[任天堂|任天堂]]发行的[[平台游戏|平台动作游戏]]，在2018年3月16日于[[任天堂Switch|任天堂Switch]]平台发行。本作也是[[星之卡比系列|星之卡比系列]]有史以来首次官方中文化的作品。

== 玩法 ==

在本作中，玩家控制的卡比可与敌人成为伙伴，通过“分享”[[Joy-Con|Joy-Con]]可以操控伙伴，最大支持4人同时游戏。除了[[星之卡比系列|星之卡比系列]]一贯的多样的能力复制之外，玩家还可以连续使出由卡比和伙伴的能力结合而成的强力合体招式，和伙伴一起跨越阻挡去路的陷阱。

== 开发 ==

本作最初公布于[[E3_2017|2017年E3]]，<ref>{{Cite news|url=https://www.engadget.com/2017/06/13/kirby-nintendo-switch-2018/|title=A new 'Kirby' game is coming to Nintendo Switch in 2018|newspaper=Engadget|accessdate=2017-12-06|language=en-US|deadurl=no|archiveurl=https://web.archive.org/web/20171017145906/https://www.engadget.com/2017/06/13/kirby-nintendo-switch-2018/|archivedate=2017-10-17}}</ref>当时预定发行时间为2018年。之后[[任天堂（香港）|任天堂（香港）]]公布本作中文版（支持繁体/简体中文）将于2018年春季发行，并正式确定官方中文译名为《星之卡比 新星同盟》。<ref>{{Cite web|url=http://www.nintendo.com.hk/pressrelease/switch_zelda_bow_20171206_sc.htm|title=任天堂(香港)有限公司|accessdate=2017-12-06|work=www.nintendo.com.hk|deadurl=no|archiveurl=https://web.archive.org/web/20171206021012/http://www.nintendo.com.hk/pressrelease/switch_zelda_bow_20171206_sc.htm|archivedate=2017-12-06}}</ref>2018年1月11日，美国任天堂于迷你任天堂直面会上宣布本作将于2018年3月16日发售。<ref>{{Cite web|url=https://www.nintendo.com/nintendo-direct/01-11-2018/|title=Nintendo Direct Mini 1.11.2018|accessdate=2018-01-11|work=www.nintendo.com|deadurl=no|archiveurl=https://web.archive.org/web/20180111142437/https://www.nintendo.com/nintendo-direct/01-11-2018/|archivedate=2018-01-11}}</ref>

== 反响 ==
2018年4月底公开的第四季（截止至2018年3月底）任天堂财报宣布本作全球销量已达126万份。<ref>{{Cite news|url=https://www.nintendo.co.jp/ir/finance/software/index.html|title=任天堂株式会社：業績・財務情報 - 主要タイトル販売実績 Nintendo Switch専用ソフト|newspaper=任天堂ホームページ|accessdate=2018-04-28|language=ja}}</ref>
