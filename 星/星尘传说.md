《'''星尘传说'''》（[[国语|国语]]：'''星空恋曲Online'''；[[英文|英文]]：'''Luvinia Online'''），是由[[成都星漫科技有限公司|成都星漫科技有限公司]]开发、[[上海盛大网络发展有限公司|上海盛大网络发展有限公司]]代理的[[三维计算机图形|3D]][[大型多人在线角色扮演游戏|大型多人在线角色扮演游戏]]。

== 游戏历史 ==
2008年10月25日游戏进行了首次封测<ref>{{cite web|url=http://www.star07.com/web2012/news_detail.asp?id=10717&bigid=0&classid=21&show=|title=10月25日封测公告|publisher=[[盛大游戏|盛大游戏]]|language=|date=2008年10月25日}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，2008年12月29日又进行了第二次封测<ref>{{cite web|url=http://www.star07.com/web2012/news_detail.asp?id=10872&bigid=0&classid=21&show=|title=《星尘传说》二测开启时间公布|publisher=[[盛大游戏|盛大游戏]]|language=|date=2008年12月29日}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，2009年2月14日又进行了第三次封测<ref>{{cite web|url=http://www.star07.com/web2012/news_detail.asp?id=10964&bigid=0&classid=21&show=|title=《星尘传说》情人节封测客户端下载|publisher=[[盛大游戏|盛大游戏]]|language=|date=2009年2月12日}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，2009年3月6日进行了预告测试也是最后一次测试<ref>{{cite web|url=http://www.star07.com/web2012/news_detail.asp?id=11103&bigid=0&classid=21&show=|title=预告测试今日18：00点正式开启|publisher=[[盛大游戏|盛大游戏]]|language=|date=2009年3月6日}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，最后在2009年3月31日正式[[公测|公测]]<ref>{{cite web|url=http://www.star07.com/web2012/news_detail.asp?id=11133&bigid=0&classid=21&show=|title=《星尘传说》宣布3月31日正式公测|publisher=[[盛大游戏|盛大游戏]]|language=|date=2009年3月28日}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2009年8月17日星漫科技表示《星尘传说》将接入盛大平台<ref>{{cite web|url=http://xc.17173.com/content/2009-08-17/20090817164219890.shtml|title=《星尘传说》将接入盛大平台|publisher=[[17173|17173]]|language=|date=2009年8月17日}}</ref>，随后关闭了星漫通行证的注册以配合对接<ref>{{cite web|url=http://xc.17173.com/content/2009-08-19/20090819174009027.shtml|title=星漫通行证关闭注册公告|publisher=[[17173|17173]]|language=|date=2009年8月19日}}</ref>，月底传出消息星漫科技被盛大收购<ref>{{cite web|url=http://news.17173.com/content/2009-08-29/20090829103152236.shtml|title=盛大再次收购彭海涛公司 或斥资1.4亿元|publisher=[[17173|17173]]|language=|date=2009年8月29日}}</ref>。

2010年3月[[鈊象電子|鈊象電子]]获得《星尘传说》在台湾的代理权<ref>{{cite web|url=http://news.17173.com/content/2010-04-23/20100423120319187.shtml|title=星尘传说新资料片《暗黑之光》即时测评|publisher=[[17173|17173]]|language=|date=2010年4月23日}}</ref><ref>{{cite web|url=http://news.17173.com/content/2010-05-10/20100510143010028,1.shtml|title=盛大3D卡通网游《星尘传说》签约台湾IGS|publisher=[[17173|17173]]|language=|date=2010年5月10日}}</ref>，以《星空戀曲Online》為名營運。2014年3月13日結束營運。

2010年4月GameOn正式签约《星尘传说》在日本的代理权<ref>{{cite web|url=http://xin.52pk.com/list/201004/872757.shtml|title=《星尘传说》进军日本 授权费达百万美元|publisher=52pk|language=|date=2010年4月15日}}</ref>。2011年11月16日游戏在日本开测<ref>{{cite web|url=http://news.17173.com/content/2011-11-18/20111118100018734.shtml|title=星尘传说日本开测 藤原拓海加盟献声|publisher=[[17173|17173]]|language=|date=2011年11月18日}}</ref>。

2010年5月[[娱美德|wemade]]获得《星尘传说》在韩国的代理权<ref>{{cite web|url=http://news.17173.com/content/2010-06-08/20100608153654088.shtml|title=星尘传说逆袭韩国 盛大与wemade再度携手|publisher=[[17173|17173]]|language=|date=2010年6月8日}}</ref>。2011年12月15日《星尘传说》在韩国首测<ref>{{cite web|url=http://news.17173.com/content/2011-12-07/20111207151420038,1.shtml|title=娱美德代理盛大《星尘传说》 12月韩国首测|publisher=[[17173|17173]]|language=|date=2011年12月7日}}</ref>，2012年3月15日进行了第二次测试<ref>{{cite web|url=http://game.sohu.com/content/2012-03-12/20120312172942298,1.shtml|title=韩服《星尘传说》15日二测 娱美德代理|publisher=[[搜狐|搜狐]]|language=|date=2012年3月12日|deadurl=yes|archiveurl=https://web.archive.org/web/20120713190054/http://game.sohu.com/content/2012-03-12/20120312172942298,1.shtml|archivedate=2012年7月13日}}</ref>，最终于4月5日进行公测<ref>{{cite web|url=http://news.17173.com/content/2012-04-06/20120406115843661,1.shtml|title=韩服《星尘传说》开始公测 宣传视频公开|publisher=[[17173|17173]]|language=|date=2010年4月6日}}</ref>。游戏在韩国的官方名称为ASTER<ref>{{cite web|url=http://news.17173.com/content/2012-11-13/20121113000137092_8.shtml|title=雷得外焦内嫩!盘点韩国囧囧有神的游戏译名|publisher=[[17173|17173]]|language=|date=2012年11月13日}}</ref>。

2010年5月31日盛大正式宣布[[智傲互动|智傲互动]]获得《星尘传说》在港澳的代理权<ref>{{cite web|url=http://news.17173.com/content/2010-06-17/20100617113826414.shtml|title=盛大星座网游《星尘传说》签约香港智傲互动|publisher=[[17173|17173]]|language=|date=2010年6月8日}}</ref>，随后智傲集團也宣布获得该游戏的代理权<ref>{{cite web |url=http://lu.gameone.com/news_detail.php?id=3073&type=1 |title=<nowiki>[公告]</nowiki>Gameone宣佈取得《星塵傳說》運營權 |publisher=[[智傲集團|智傲集團]] |language= |archiveurl=https://www.webcitation.org/6CoAQjBDJ?url=http://lu.gameone.com/news_detail.php?id=3073 |date=2010年6月15日 |archivedate=2012年12月10日 |deadurl=yes }}</ref>。2010年10月13日<ref>{{cite web |url=http://lu.gameone.com/news_detail.php?id=3086&type=1 |title=<nowiki>[公告]</nowiki>香港版《星塵傳說》10月13日封測(CB) |publisher=[[智傲集團|智傲集團]] |language= |archiveurl=https://www.webcitation.org/6CoAmyiD0?url=http://lu.gameone.com/news_detail.php?id=3086 |date=2010年10月3日 |archivedate=2012年12月10日 |deadurl=yes }}</ref>到10月26日<ref>{{cite web |url=http://lu.gameone.com/news_detail.php?id=3147&type=1 |title=《星塵傳說》封測10月13日開啟 (限時開放) |publisher=[[智傲集團|智傲集團]] |language= |archiveurl=https://www.webcitation.org/6CoB91vtP?url=http://lu.gameone.com/news_detail.php?id=3147 |date=2010年10月12日 |archivedate=2012年12月10日 |deadurl=yes }}</ref>进行首测，随后在2010年11月16日公测<ref>{{cite web |url=http://lu.gameone.com/news_detail.php?id=3345&type=1 |title=《星塵傳說Online》公測正式開放 |publisher=[[智傲集團|智傲集團]] |language= |archiveurl=https://www.webcitation.org/6CoBPMPZo?url=http://lu.gameone.com/news_detail.php?id=3345 |date=2010年11月16日 |archivedate=2012年12月10日 |deadurl=yes }}</ref>。最终在2012年10月25日智傲在游戏官网宣布将于2013年1月25日关闭游戏服务器<ref>{{cite web |url=http://lu.gameone.com/news_detail.php?id=7079&type=1 |title=香港版《星塵傳說Online》營運中止公告 |publisher=[[智傲集團|智傲集團]] |language= |archiveurl=https://www.webcitation.org/6CoBfzWvK?url=http://lu.gameone.com/news_detail.php?id=7079 |date=2012年10月25日 |archivedate=2012年12月10日 |deadurl=yes }}</ref>。

2010年6月初OutSpark获得《星尘传说》在北美的代理权<ref>{{cite web|url=http://news.17173.com/content/2010-06-11/20100611163044828,1.shtml|title=签约OutSpark 星尘传说进军北美免费市场|publisher=[[17173|17173]]|language=|date=2010年6月11日}}</ref>。

== 遊戲職業 ==
*001-040級：初心者、戰士、行者、法師
*040-080級：騎士、鬥士、斥候、術士、牧師、藝人
*080-120級：聖騎士、狂戰士、刺客、巫師、劍士、主教、暗騎士、死靈法師、巡守、賢者、劍舞者、吟游詩人
*120-160級：預定十二星座職業轉職廿四節氣職業
*160-200級：預定廿四節氣職業轉職卌八星區職業（200級平轉）

== 遊戲任務 ==
*001-040級：劇情：001級-迦那維學院、010級-曙光鎮、025級-國都、040級-禁語；副本：030級-下水道、040級-利齒谷
*040-080級：劇情：050級-瓦克郡、060級-雪語郡、070級-炎斧郡；副本：050級-阿卡隆、060級-圖拉丁、070級-布塔克
*080-120級：劇情：080級-音符郡、090級-林哥郡、100級-藍潮郡；副本：080級-阿爾菲、090級-象牙塔、100級-霍監獄
