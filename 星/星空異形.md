《'''星空異形'''》（{{lang-en|'''''Nightflyers'''''}}）是一部1987年的美國科幻恐怖電影，改編自[[喬治·R·R·馬丁|喬治·R·R·馬丁]]的1980年中篇小說《{{link-en|暗夜飛行者 (中篇小說)|Nightflyers|暗夜飛行者}}》。<ref>{{Cite web |url = https://www.tor.com/2018/03/27/everything-you-need-to-know-about-george-r-r-martins-nightflyers/ |title = Everything You Need to Know About George R.R. Martin's Nightflyers |author = Natalie Zutter |date = 2018-03-27 |publisher = Tor.com |language = en}}</ref><ref>{{Cite web |url = https://www.nytimes.com/1987/10/24/movies/film-nightflyers-science-fiction-thriller.html |title = Film: 'Nightflyers' Science Fiction Thriller |author = Caryn James |date = 1987-10-24 |publisher = The New York Times |language = en}}</ref>

== 劇情 ==
本片是關於一群開始太空旅行去尋找神秘外星生物的科學家，而在過程中受到了該太空船的惡毒電腦所侵害。
