'''星塵'''（{{lang-en|Stardust}}）是音乐社团平行四界推出的[[角色|虚拟歌手]]，采用[[Vocaloid|Vocaloid]][[语音合成|语音合成]]技术。歌声样本取自中国大陆网络歌手茶理理，形象由November透明星设计。软件产品于2016年4月13日发售。该产品为第一款以Vocaloid4引擎为基础开发的中文产品，也是上海禾念制作的首款“聲庫定制企划”产品<ref name="announcement">{{Cite web |url = http://weibo.com/5582156776/CjUFfxBwx |title = 大家好， 今天给大家带来两个好消息。 首先..來自Vsinger的微博 |author = [[Vsinger|Vsinger]] |date = 2015年5月28日 |publisher = 微博 |language = 中文 }}</ref>。

== 角色 ==
*性別：女
*种族：外星人
*外貌：留有钴蓝色的长发且绑着四个马尾，拥有黄色的瞳孔，穿着暗蓝色[[抹胸裙|抹胸裙]]和灰色的[[膝上襪|膝上襪]]，手上托着[[星形八面體|星形八面體]]。

===历史===
星尘原为同人专辑《平行四界》系列的看板娘“平行四界子”。2015年2月16日平行四界官方媒体公开了由November透明星绘制的初（平行四界子）设定图<ref>{{Cite web |url = http://weibo.com/5056093502/C4zrJ38rp |title = #求起名# 由@November透明星 为社团创作的看板娘（就是咱[害羞]）设定图公布。 |author = 平行四界Quadimension |date = 2015年2月16日 |publisher = 微博 |language = 中文 }}</ref>。5月确定了沿用之前的人设，但由于粉丝认为其造型与[[初音未来|初音未来]]雷同，在几天后官方宣布将会绘制新的人设<ref>{{Cite web |url = http://weibo.com/5056093502/Ck81rsYcV |title = 周末了回娘家哭一哭，还不如继续当看板娘卖专辑，做偶像好可怕... |author = 平行四界Quadimension |date = 2015年5月29日 |publisher = 微博 |language = 中文 }}</ref>，人设概念继续由November透明星担当，立绘则由ideolo完成。<ref>{{Cite web |url = http://www.vocaloidnews.net/zh-hans/%E4%B8%AD%E6%96%87vocaloid-%E6%98%9F%E5%B0%98%E7%9A%84%E5%AE%98%E6%96%B9%E7%94%BB%E9%9B%86%E7%A8%BF%E4%BB%B6%E6%8B%9B%E5%8B%9F%E4%BB%A5%E5%8F%8A%E5%8F%91%E5%94%AE%E6%97%A5%E6%9C%9F/ |title = 中文VOCALOID – “星尘”官方画集企划稿件募集&发售日期揭晓！ |author = Louis |date = 2016年9月26日 |publisher = VNN |language = 中文 }}</ref>2016年2月官方公布了其具体设定<ref>{{Cite web |url = http://weibo.com/5599743152/DiT2QadMB |title = #星尘降临日# 星尘人设公开(｡･ω･｡)ﾉ✡ |author =星尘_Official  |date = 2016年2月21日 |publisher = 微博 |language = 中文 }}</ref>。在新的人设中，对称的四马尾被改为下四马尾。

2015年5月28日，上海禾念公开声库开发计划<ref name="announcement" />（在此之前，运营方平行四界曾在愚人节宣布将会推出相关声库<ref>{{Cite web |url = http://weibo.com/5056093502/Cbe9ul8QF |title = 哦另外我们今年还要自己做个Vocaloid声库，恩就是四界子（严肃脸[doge]（我实在编不下去了…  |author = 平行四界Quadimension |date = 2015年4月1日 |publisher = 微博 |language = 中文 }}</ref>）。2016年4月13日，产品正式发售。
