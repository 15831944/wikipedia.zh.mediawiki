'''星海湾跨海大桥'''，又称'''大连南部滨海大道跨海大桥'''，简称'''星海湾大桥'''，位于[[中华人民共和国|中国]][[辽宁省|辽宁省]][[大连市|大连市]]，是中国首座海上修筑锚碇的地锚式[[悬索桥|悬索桥]]，也是中国第一座公路双层钢桁架悬索桥。大桥于2011年开始建设，2015年10月30日开通。星海湾大桥全长6公里；其中，主桥长820米，净空高度30米，中央正对[[星海广场|星海广场]]中轴线<ref name = "design">{{Cite web|url=http://paper.sciencenet.cn/htmlnews/2015/11/330742.shtm|title=大连星海湾跨海大桥正式通车—新闻—科学网|accessdate=2018-03-11|work=paper.sciencenet.cn}}</ref>。星海湾大桥直接连接大连西部的[[大连高新技术产业园区|大连高新区]]以及东部的[[金沙滩_(大连)|金沙滩]]、[[付家庄|付家庄]]地区，使居住在城市西南部的通勤者得以直接跨海通行至[[青泥洼桥|青泥洼桥]]等城市东部区域而不需要经过[[黑石礁|黑石礁]]、[[星海广场_(大连市)|星海广场]]等拥挤地段，为缓解东西向交通拥堵起到了重要的作用<ref>{{Cite web|url=http://liaoning.nen.com.cn/system/2014/12/03/014096998.shtml|title=大连：南部跨海大桥明年6月有望通车（图）-跨海大桥-辽宁频道|accessdate=2018-03-11|work=liaoning.nen.com.cn}}</ref>。

== 简介 ==
[[File:Xinghai_Bay_Bridge_Dalian_China.jpg|thumb]]
随着大连经济发展，市区不断向外延伸，交通问题也愈发严重。大连境内以丘陵、山地为主，可开发土地有限，道路互通性先天不足。而大连的道路网络长期缺乏规划，使得大量车流集中在少数几条道路上，缺乏分流路线，造成拥堵<ref>{{Cite web|url=http://www.xinhuanet.com/2015-09/10/c_1116514162.htm|title=77.1%受访者认为单纯限行只会越限越堵-新华网|accessdate=2018-03-11|work=www.xinhuanet.com}}</ref><ref>{{Cite web|url=http://liaoning.nen.com.cn/system/2017/01/09/019586823.shtml|title=大连市区交通拥堵咋治？ 从供给侧下手-辽宁频道-东北新闻网|accessdate=2018-03-11|work=liaoning.nen.com.cn}}</ref>。市区东西两侧的交通流量都集中在[[中山路_(大连)|中山路]]一条路上<ref name = "zhongshan">{{Cite web|url=http://ln.sina.com.cn/news/b/2012-12-28/071923190.html|title=大连投资40亿星海湾跨海大桥初定2014年通车_新浪辽宁新闻_新浪辽宁|accessdate=2018-03-11|work=ln.sina.com.cn}}</ref>。为了解决交通分流问题，大连市一方面于2009年起兴建[[大连地铁|大连地铁]]，另一方面试图通过建造跨海大桥和山间隧道结合的方式，解决东西向交通问题<ref name = "zhongshan" />。

星海湾跨海大桥是大连南部滨海大道项目的一部分，于2011年10月10日正式开工，2015年1月5日合龙<ref>{{Cite web|url=http://www.xinhuanet.com/tech/2015-01/06/c_127362279_2.htm|title=大连星海湾跨海大桥合龙(组图) -科技频道-新华网|accessdate=2018-03-11|work=www.xinhuanet.com}}</ref>。桥梁西部连接[[大连高新技术产业园区|大连高新技术产业园区]]，向东穿越大连南部的星海湾海域后在金沙滩登陆后分流，一条通过[[莲花山隧道|莲花山隧道]]穿越[[大连森林动物园|大连森林动物园]]的山区，向北连接市中心多条主干道；另一条则和[[滨海路风景区|滨海路]]交汇，沿滨海路风景区继续前行<ref name = "zhongshan" />。桥梁由[[大连理工大学|大连理工大学]]设计<ref name = "design" />，中交集团第一、第二航务工程局有限公司和中铁十九局集团共同承建<ref>{{Cite web|url=http://ln.sina.com.cn/news/b/2012-12-28/071923190_3.html|title=星海湾大桥主设计师是大工教授_新浪辽宁新闻_新浪辽宁|accessdate=2018-03-11|work=ln.sina.com.cn}}</ref>。

大桥为双层设计，上下两层均为单行四车道。车辆在上层桥行车方向为自东向西，在下层桥行车方向为自西向东，限速为60公里/小时<ref>{{Cite web|url=http://www.chinadaily.com.cn/micro-reading/interface_yidian/2015-10-31/14299486.html|title=全长6.8公里大连星海湾跨海大桥通车 似蛟龙横卧(组图)|accessdate=2018-03-11|work=www.chinadaily.com.cn}}</ref>。同时，桥上设计有行人步道以方便游客观景。<ref>{{Cite web|url=http://society.people.com.cn/n/2015/1031/c1008-27760122.html|title=大连星海湾跨海大桥通车--社会--人民网|accessdate=2018-03-11|work=society.people.com.cn}}</ref>星海湾跨海大桥使用了多项新技术，例如道路使用了环氧沥青铺装，比一般沥青更加坚固耐用；大桥设有健康检测系统，以确保桥梁的及时维护<ref name = "record" >{{Cite web|url=http://ln.sina.com.cn/news/b/2012-12-28/071923190_2.html|title=星海湾大桥创国内桥梁史多项纪录_新浪辽宁新闻_新浪辽宁|accessdate=2018-03-11|work=ln.sina.com.cn}}</ref>。星海湾大桥也是世界上首座采用抛石升降[[沉箱|沉箱]]基础施工的桥梁；其中，施工使用的沉箱每个重达26000吨，是中国目前质量最大的沉箱<ref>{{Cite news|url=http://dl.sina.com.cn/news/m/2015-10-30/detail-ifxkhqea2868089.shtml|title=星海湾大桥创7项纪录莲花山隧道有3个第一|publisher = 半岛晨报| accessdate=2018-03-11}}</ref>。与此同时，由于大桥穿越多个风景区，设计方也对桥梁的形状和走向进行了专门设计，使得大桥与整个区域的景观尽量协调一致<ref name = "record" />。

== 影响 ==
根据大连当地媒体的分析，星海湾跨海大桥是一项重要的民生工程，每天承载车流超过3万辆，有效纾解了各个主干道的拥堵，缩短了大连东部和西部城区的通勤时间<ref>{{Cite web|url=http://ln.qq.com/a/20160712/038940.htm|title=彩虹飞渡星海湾连起东西大连城_大辽网_腾讯网|accessdate=2018-03-11|work=ln.qq.com|language=zh-CN}}</ref>。星海湾跨海大桥也成为了一个热门的旅游景点，吸引许多游客上桥赏景，但也产生了违规停车等问题，带来交通安全隐患<ref>{{Cite web|url=http://dalian.runsky.com/2016-02/10/content_5445831.htm|title=违规停车滞留观光致星海湾跨海大桥交通拥堵|3=大连新闻_天健网|accessdate=2018-03-11|work=dalian.runsky.com|archive-url=https://web.archive.org/web/20180311141433/http://dalian.runsky.com/2016-02/10/content_5445831.htm|archive-date=2018-03-11|dead-url=yes}}</ref>。大桥的竣工也带动了周边的房地产市场，使得一些楼盘房价明显上涨<ref>{{Cite web|url=http://jiangsu.china.com.cn/html/house/lsxw/4167591_1.html|title=大连星海湾大桥“桥景房”正流行 新房单价均已过万 - 房产新闻 - 中国网•东海资讯|accessdate=2018-03-11|work=jiangsu.china.com.cn}}</ref>。

一些人质疑星海湾大桥会污染周边海域，阻挡邻近的水流，影响海洋生态。对此，建设部门表示会对桥梁的建设和维护进行管控，减少对海洋环境的危害<ref>{{Cite web|url=http://ln.sina.com.cn/news/b/2012-12-28/071923190_4.html|title=尽量采用场外加工避免废渣污染大海_新浪辽宁新闻_新浪辽宁|accessdate=2018-03-11|work=ln.sina.com.cn}}</ref>。
