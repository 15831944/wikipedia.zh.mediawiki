《'''JoJo的奇妙冒險 第三部 星塵鬥士'''》（{{lang|ja|ジョジョの奇妙な冒険 Part3 スターダストクルセイダース}}）為[[荒木飛呂彥|荒木飛呂彥]]所著的[[日本漫畫|日本漫畫]]《[[JoJo的奇妙冒險|JoJo的奇妙冒險]]》系列的第三部，自1989年起連載至1992年止，收錄於單行本第13至28集。

本作是系列作中最受歡迎的一部，出現了名為[[替身_(JoJo的奇妙冒險)|替身]]的超能力，這個設定也成為之後JoJo系列的核心。

== 故事大綱 ==
故事揭幕於1988年的日本，喬納森·喬斯達與宿敵[[迪奧·布蘭度|DIO]]決戰的一百年後，DIO於大西洋下意外被打撈上岸而重生。同時，喬瑟夫的孫子，'''空條承太郎'''突然發現自己有一種名幽波紋（[[替身_(JoJo的奇妙冒險)|替身]]）的能力，起因於DIO的替身能力的覺醒，而同時影響了對替身沒有抵抗能力的空條荷莉，令她陷入病危；為了拯救命在旦夕的母親，承太郎、'''喬瑟夫'''、'''花京院典明'''與'''阿布德爾'''一行出發前往DIO的藏匿地：[[埃及|埃及]]。

他們一行人經[[英属香港|香港]]、[[新加坡|新加坡]]、[[印度|印度]]、[[巴基斯坦|巴基斯坦]]、[[紅海|紅海]]等地，以[[飛機|飛機]]、[[火車|火車]]、[[馬車|馬車]]、[[輪船|輪船]]、[[駱駝|駱駝]]、[[潛水艇|潛水艇]]等各種各樣的交通工具旅行，沿途不斷受到DIO派出來的塔羅牌替身使者的攻擊，並在戰鬥中結識了夥伴'''J‧P‧波魯納雷夫'''。在[[埃及|埃及]]上岸後、一行人與夥伴'''伊奇'''會合，擊敗了DIO所派出來的手下「九榮神」，最後終於找到了DIO宅邸，了結這段百年來的宿命恩怨。

進入DIO之館後，阿布德爾和伊奇為了從敵人的手中保護波魯納雷夫而死，剩餘的四人與DIO展開最後對決，雙方在開羅鬧市街頭追逐死鬥。花京院為揭開DIO的替身之謎，以法皇結界引出並和其決戰，但隨即被重創，在臨死前推測DIO的替身「世界」可以「暫停時間」，用最後的力量破壞鐘樓將訊息引導給喬瑟夫，而喬瑟夫與DIO對決時意識到自己也非其敵手，只能將訊息傳給承太郎。最終唯承太郎一人與DIO對峙，兩人初時勢均力敵，DIO於是啟動「暫停時間」令承太郎陷入劣勢，波魯納雷夫亦於途中對DIO展開暗殺，始終未能擊敗他。不過由於承太郎的「白金之星」和「世界」是同類型的替身，在戰鬥中承太郎開始逐漸掌握暫停時間的特殊能力，總算戰勝DIO的替身「世界」，擊斃了這與喬斯達家族糾纏百年的死敵。

== 其他細節 ==
* 本作開始引入了[[替身|替身]]能力，並一直沿用至今。作者表示在第二部結束時，因為責任編輯認為「波紋已經太老舊了」，因此創作出新的超能力。
* 這是系列作中第一次以日本為背景來展開故事。本作第一部及第二部中、舞台在現實世界但卻不是由日本人擔任主角、故事舞台也不在日本，通篇故事與日本毫無關聯的漫畫，在當時的日本漫畫界中是很少見的。
* 本部開始追加了喬斯達一族左肩後方有星型[[胎記|胎記]]的設定，動畫版針對此一設定將一、二部的JOJO做了外型上的修正。
* 在雜誌連載時期，本作中的塔羅牌設計涉及了[[著作權|著作權]]的侵害，作者在事後道歉並在單行本重新繪製所有的塔羅牌圖案。
* 在托托神的那一段劇情中，托托神曾經預知了一段未來、內容與11年後的[[九一一襲擊事件|九一一襲擊事件]]有許多部分相似，包括男子身上的衣服、事發的時間、男子與電線杆的圖像、背景的飛機圖案等等。
* 在迪奧命令議員開車的其中一幕，出現了[[威利在哪裡？|威利在哪裡？]]的威利。
