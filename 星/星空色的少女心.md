《'''星空色的少女心'''》（{{lang-ja|乙女のはらわた星の色}}）為[[日本漫畫家|日本漫畫家]][[いしとゆうら|いしとゆうら]]於《[[Jump_Square|Jump Square]]》連載漫畫作品。[[2019年|2019年]]1月号开始連載中。<ref>{{cite web |title=“侵略的婚活”のため地球を訪れた宇宙人とのラブコメ「乙女のはらわた星の色」 |url=https://natalie.mu/comic/news/310762 |website=コミックナタリー |language=ja |date=2018年12月4日}}</ref>

== 故事内容 ==
20年前降落到地球上的外星人，韦维人的目的，是和地球人的侵略性的婚活……韦维人是通过与异种族的交杂，获得了能力的多样性的星人，在地球上也以那个为目标。<br>
为登上国政最高层，以保护“纯血地球人”为目的，考入了由韦维人设立的精英学校“森陵学园”。但是森陵学园鼓励外星人异性恋，世间称之为“婚活学园”。<br>
源能继续守护操守吗……！？<ref>简介来自官网jump.sq介绍</ref>
== 登场人物 ==
; ゲン
: 为了保护地球人而进入森陵学园。对韦维星人的恋爱持怀疑态度。
; ミツヨシ
: 源的同班同学以及朋友。
; ミリカ
: 韦维星人。喜欢上别人的话有冒火的能力。
; タレラ
: 韦维星人。长着兽耳和尾巴。
; キゥリプ
: 韦维星人。拥有操作接触到的对方的快乐中枢的能力。
; ミロミィ
: 韦维星人。小手球会所属的二年级学生。语调和张弛是其特征。
