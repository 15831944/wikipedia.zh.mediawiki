'''[[星期|星期]]中的日子'''（'''日历日'''<ref name="gb">中华人民共和国国家标准GB/T  7408-2005《数据元和交换格式 信息交换 日期和时间表示法》4.3.2.2日历星期</ref>）的名稱在全世界[[語言|語言]]中可分為「[[數字|數字]]制」和「[[宇宙|星象]]制」兩種。數字制中，[[阿拉伯語|阿拉伯語]]、[[希伯來語|希伯來語]]、[[越南語|越南語]]和[[葡萄牙語|葡萄牙語]]以[[週日|週日]]作為一周中的第一天，而[[俄語|俄語]]和其他[[斯拉夫語族|斯拉夫語系]]則以[[週一|週一]]作為一周中的首天。星象制以日（[[太陽|太陽]]）、月（[[月亮|月亮]]）、火（[[火星|火星]]）、水（[[水星|水星]]）、木（[[木星|木星]]）、金（[[金星|金星]]）、土（[[土星|土星]]）等與[[羅馬神話|羅馬神祇]]有關的星體名稱來命名日子；[[英文|英文]]在內的[[日耳曼语族|日耳曼语族]]則使用[[北歐神話|北歐神話]]中神祇的名字。

英文的日子名稱取自[[盎格魯-撒克遜人|盎格魯-撒克遜人]]的神話，其中[[週四|週四]]（Thursday）和[[週五|週五]]（Friday）的名稱來自[[斯堪地納維亞|斯堪地納維亞]]的神祇[[索爾|索爾]]（Thor）和[[弗蕾亞|弗蕾亞]]（Freyja）。[[週六|週六]]（Saturday）是一個例外，其名字取自[[羅馬|羅馬]]神祇[[薩圖爾努斯|薩圖爾努斯]]（Saturn）。與英語相似，大部分主要使用[[日耳曼語|日耳曼語]]的地區也以本身神話中的神祇名稱命名日子。

[[中文|中文]]的名稱使用[[數字|數字]]制，以一到六的[[數字|數字]]以及「日」來作為命名的基礎，在數字前加上「週」、「星期」或「禮拜」。[[中华人民共和国|中华人民共和国]]采用的法定名称为星期一、星期二、星期三、星期四、星期五、星期六、星期日。<ref name="gb"/>除了星期日有別名──「星期天」，其他日子的名稱相當單純，沒有變化（詳細名稱請見下面段落）。「禮拜」這個名稱源於[[中國|中國]][[回教徒|回教徒]]和[[基督教徒|基督教徒]]，這些[[宗教|宗教]]都在[[週日|週日]]做崇拜，因此將週日稱做「禮拜日」或「禮拜天」，其後漸漸延伸到整個星期的名稱。{{Fact|time=2008-09-30T10:57:47+00:00}}<!-- 本模板調用由 {{subst:Fact/auto}} 自動產生 -->

以七日為一個工作周的地區，基本上由五個工作日和兩個非工作日（[[週末|週末]]）組成。不過，哪一天是一周中的「第一天」，各地有各地的習慣，就算有相同週末的國家也會有這種差異。在許多[[英語世界|英語世界]]的國家、[[以色列|以色列]]、[[日本|日本]]、[[香港|香港]]和[[澳門|澳門]]，一般认为一个星期开始于[[星期日|星期日]]。[[埃及人|埃及人]]的一星期是从[[星期六|星期六]]开始的。多數歐洲國家都以星期日為一星期的第一天。[[国际标准化组织|国际标准化组织]]（[[ISO_8601|ISO 8601]]）规定星期一为每星期的第一天。[[中华人民共和国国家标准|中华人民共和国国家标准]]等效采用ISO 8601。<ref name="gb"/>

== 每個日子的名稱 ==
以[[華語|華語]]系的[[語言|語言]]為[[母語|母語]]的[[地區|地區]]來看，每個日子的名稱如下：
*'''[[星期日|星期日]]'''（週日、禮拜日、禮拜天、星期天）
*'''[[星期一|星期一]]'''（週一、禮拜一）
*'''[[星期二|星期二]]'''（週二、禮拜二）
*'''[[星期三|星期三]]'''（週三、禮拜三）
*'''[[星期四|星期四]]'''（週四、禮拜四）
*'''[[星期五|星期五]]'''（週五、禮拜五）
*'''[[星期六|星期六]]'''（週六、禮拜六）

而就[[英語|英語]]以及其它使用和英語相同命名方式的語言每個日子的名稱和由來如下：
*'''Sunday'''（週日）：這個名字是源自於[[古英語|古英語]]的《Sunnandæg》（發音為[sun.nan.dæg]或[sun.nan.dæj]），意思是「[[太陽|太陽]]之日」。意思是[[拉丁語|拉丁語]]的「Dies Solis」直譯過來的。
*'''Monday'''（週一）：這個名字是源自於[[古英語|古英語]]的《Mōnandæg》（發音為[mon.nan.dæg]或[mon.nan.dæj']），意思是「[[月亮|月亮]]之日」。意思是[[拉丁語|拉丁語]]的「Dies Lunae」。
*'''Tuesday'''（週二）：這個名字是源自於[[古英語|古英語]]的《Tiwesdæg》（發音為[ti.wes.dæg]或[ti.wes.dæj]），意思是「[[提爾|提爾]]之日」。 [[提爾|提爾]]（Tyr）是[[北歐神話|北歐神話]]和[[德國|德國]]異教的[[戰鬥|戰鬥]]之神（[[古英語|古英語]]：Tiw、Tew、Tiu）。意思是[[拉丁語|拉丁語]]的「Dies Martis」（[[火星|火星]]之日）。
*'''Wednesday'''（週三）：這個名字是源自於[[古英語|古英語]]的《Wōdnesdæg》（發音為[woːd.nes.dæg]或[woːd.nes.dæj]），意思是「[[奧丁|沃登]]之日」。[[奧丁|沃登]]（[[:en:Wōden|Wōden]]）是[[北歐神話|北歐神話]]職權最高的神祇，同時也是在7世紀前，[[英國|英國]]（以及其它地方）的[[盎格魯-撒克遜人|盎格魯-撒克遜人]]重要的神祇。意思是[[拉丁語|拉丁語]]的「Dies Mercurii」（[[水星|水星]]之日）。
*'''Thursday'''（週四）：這個名字是源自於[[古英語|古英語]]的《Þūnresdæg》（發音為[θuːn.res.dæg]或[θuːn.res.dæj]），意思是「[[索爾|索爾]]之日」。[[索爾|索爾]]（Thor）的原文是「Þunor」，祂是[[北歐神話|北歐神話]]和[[德國|德國]][[異教|異教]]的[[雷神|雷神]]。意思是[[拉丁語|拉丁語]]的「Dies Iovis」（[[木星|木星]]之日）。
*'''Friday'''（週五）：這個名字是源自於[[古英語|古英語]]的《Frigedæg》（發音為[fri.je.dæg]或[fri.je.dæj]），意思是「富里基之日」。富里基（[[:en:Frige_(Anglo-Saxon_goddess)|Frīge]]）是德系神祇象徵[[美麗|美麗]]的神，也是名叫[[弗麗嘉|弗麗嘉]]的北歐神祇的化身，而另外一種說法則是祂和神祇[[弗蕾亞|弗蕾亞]]有關連。意思是[[拉丁語|拉丁語]]的「Dies Veneris」（[[維納斯|維納斯]]之日、[[金星|金星]]之日）。[[維納斯|維納斯]]是[[羅馬|羅馬]]神祇的象徵[[美麗|美麗]]和性感的神。
*'''Saturday'''（週六）：週六是唯一一個在[[英文|英文]]保留其原有羅馬文字的日子名稱，是以羅馬神祇[[薩圖爾努斯|薩圖爾努斯]]來命名的。這個字是源自於[[古英語|古英語]]的《Sæturnesdæg》（發音為[sæ.tur.nes.dæg]或[sæ.tur.nes.dæj]），意思是「[[薩圖爾努斯|薩圖爾努斯]]之日」。意思是[[拉丁語|拉丁語]]的「Dies Saturni」（[[土星|土星]]之日）。

== 每個星期的第一天 ==

在不同地區，一星期的開始時間並不完全一致。多数[[美洲|美洲]]国家如[[美國|美國]]、[[加拿大|加拿大]]、[[墨西哥|墨西哥]]，部分[[亚洲|亚洲]]国家或地区如[[印度|印度]]、[[日本|日本]]、[[香港|香港]]，以及[[以色列|以色列]]等地一般認為一個星期開始於星期日（Sunday）。多数[[歐洲|歐洲]]國家、[[澳洲|澳洲]]、[[南非|南非]]、少数[[南美洲|南美洲]]國家以及部分[[亞洲|亞洲]]國家或地区如[[中国大陆|中国大陆]]、[[韩国|韩国]]都以星期一（Monday）為一星期的第一。週日也是所有星象制和[[希伯來曆|希伯來曆]]星期中的第一天。

在[[猶太人|猶太人]]和[[基督徒|基督徒]]的傳統中，週日是一個星期中的第一天。根據[[聖經|聖經]]，[[上帝|上帝]]花了六天的時間創造了[[世界|世界]]，而在第七天，也就是[[安息日|安息日]]（星期六）休息了，這一天就被視為慶祝和休息之日，這就使得[[星期日|星期日]]成了一個星期中的第一天。在[[歐洲|歐洲]]早期的[[基督徒|基督徒]]開始採用[[星期|星期]]的制度時，一開始是以星期日為一個星期的第一天，{{來源請求|但是星期六也漸漸的被視為較重大的[[安息日|安息日]]，也成了[[主日|主日]]，因此星期日到最後反而被視為了一個星期的最後一天。}}

某些國家的日子名稱的不同之處非常的明顯：在[[希伯來語|希伯來語]]、[[阿拉伯語|阿拉伯語]]和[[希臘語|希臘語]]中，有些日子的名稱是直接以它在這個星期中的順序來稱呼（以週日為第一天），例如：週一稱為「第二天」。而還有一些使用[[斯拉夫語族|斯拉夫語系]]語言的國家也會以其在星期中的順序來稱呼，但是是以週一來做為第一天，例如：週二稱為「第二天」。還有另外一種解釋，那就是有些使用[[斯拉夫語族|斯拉夫語系]]語言的國家並不是將日子以其在星期中的順序來稱呼，而是以其和週日的距離來稱呼，例如：週三稱為「中間日」，而這就證明了仍然有一些使用[[斯拉夫語族|斯拉夫語系]]語言的國家以週日來當作第一天。

但無論如何，[[國際標準化組織|國際標準化組織]]（ISO）在[[ISO_8601|ISO 8601]]中指定週一為一星期中的第一天，用來代表[[軟體|軟體]]的時間格式。

==计算机软件的国际化==
计算机软件设计时为了适应各国各地区的各种不同细节，使用[[区域设置|区域设置]]技术来使得这些细节对程序设计透明化。即同样的源程序产生的输出能自动遵循各国各地区的风格。下述[[C++|C++]]语言程序能在Windows系统上获取当前区域的星期开始的日子：
<source lang="cpp">
#include <windows.h>  
#include <iostream> 

int __cdecl wmain(int argc, wchar_t* argv[])
{
 wchar_t buf[80];
 unsigned ret;
 ret = GetLocaleInfoEx(L"",
		LOCALE_IFIRSTDAYOFWEEK,
		(LPWSTR)buf,
		80);

 std::wcout << std::hex << buf << std::endl;
/* 0表示星期一，1表示星期二，......，5表示星期六，6表示星期日 */ 
}
</source>
