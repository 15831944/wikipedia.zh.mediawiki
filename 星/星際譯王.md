'''星際譯王'''（StarDict）是一套[[自由軟體|自由]]的桌面[[字典|字典]]軟體。它並不包含字典檔，使用者須自行下載配合使用。它可以運行於多種不同的平台，如[[Linux|Linux]]，[[Microsoft_Windows|Microsoft Windows]]，[[FreeBSD|FreeBSD]]及[[Solaris|Solaris]]，並使用[[GPL|GPL]]授權。

== 特色 ==
它透過[[字典檔|字典檔]]的配合，可以翻查數量極其龐大的單詞（現在已經有二百多部內置辭典），並帶有常用詞帶[[音標|音標]]，兼且具有[[模糊匹配|模糊匹配]]、[[屏幕取詞|屏幕取詞]]功能，而且自帶[[中文字體|中文字體]]，獨立於系統之外。目前支持的語言，除了簡體、繁體中文與英文互譯，還支持日文、俄文等。

當星際譯王運行於[[掃描模式|掃描模式]]時，所選取的詞語將會自動地在字典裡找尋，並會將所有的結果展示於[[彈出選單|彈出選單]]。它亦能透過與{{Link-en|Freedict|Freedict}}的整合來翻譯外文網站，但功能还不完善。

== 技術 ==
星际译王使用[[Motif|Motif]]技术开发，包含了星際譯王源程序、[[圖標|圖標]]和背景圖形文件，中文字體、音標字體，詞庫、音標庫，使用幫助、開發計劃等文件。

== 命令行版 ==
{{Main|sdcv}}

星際譯王命令行版（sdcv）是星際譯王的一個補充版本由Evgeniy A. Dushistov和胡正開發。sdcv能使用所有星際譯王的字典文件。sdcv可以運行於多種不同的平台，如[[Linux|Linux]]，[[FreeBSD|FreeBSD]]及[[Solaris_Operating_Environment|Solaris]]，並使用[[GPL|GPL]]授權。

=== 使用方法 ===
基本格式：
:sdcv [選項] [待查單詞序列]
可以使用「-h」選項來查看所有的選項及其使用方法。

sdcv使用了一个環境變數'''STARDICT_DATA_DIR'''。若此變數被設定，則會在目錄$STARDICT_DATA_DIR/dic下查找字典。

== 法律責任嚴正聲明 ==
stardict官方網站不少字典檔并没有得到官方授权，其中包括Oxford Advanced Learner's Dictionary、Longman Dictionary of Contemporary English在内的大量字典均存在版权问题。但使用以下字典檔并不存在相应的版权问题：

* 1913 Webster's<ref>[http://www.gutenberg.org/ebooks/673 The Gutenberg Webster's Unabridged Dictionary by Project Gutenberg]</ref>
* Princeton's Wordnet<ref>[http://wordnet.princeton.edu/ About WordNet]. WordNet</ref>
* Freelang.com dictionaries<ref>[http://www.freelang.net/ FREELANG - Free Dictionaries to download and Free Human Translation]</ref>
* TU Chemnitz dictionary (e.g. Ding) A German-English dictionary<ref>[http://dict.tu-chemnitz.de/ Dictionary / Wörterbuch (BEOLINGUS, TU Chemnitz)]</ref>
* The XML GPL'ed version of Littré.A French-English dictionary<ref>{{cite web |url=http://francois.gannaz.free.fr/Littre/horsligne.php |title=存档副本 |accessdate=2008-09-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20081113183440/http://francois.gannaz.free.fr/Littre/horsligne.php |archivedate=2008-11-13 }}</ref><ref>[http://packages.debian.org/unstable/text/stardict-xmlittre]</ref>
* cedict<ref>[https://www.mdbg.net/chinese/dictionary?page=cedict MDBG Chinese-English dict]</ref>

== 项目终结 ==
星际译王项目终因版权问题走到了尽头，已在sourceforge中被删除。

官方在[http://stardict.sourceforge.net/ 主页]给出了一个暂存的google code链接：[http://code.google.com/p/stardict-3/ Stardict]

另外还推荐了其他几个词典，例如[[GoldenDict|GoldenDict]]、Babiloo等

== 詞典收費風波 ==
2006年底，軟體開發者[[胡正|胡正]]以個人經濟問題爲由，向在其网站下载字典文件的用户进行收费，<ref>2006年12月08日 知名IT資訊網站[http://www.cnbeta.com/ cnBeta.com]報導[http://www.cnbeta.com/articles/19146.htm 《Linux下的著名翻譯軟件StarDict字典開始收費》]</ref>一時激起了Linux社區的强烈質疑<ref>[http://ilihai.com/stardict_pay.html 《Stardict字典什麽時候開始收費了！？》] {{webarchive|url=https://web.archive.org/web/20071101023443/http://ilihai.com/stardict_pay.html |date=2007-11-01 }}</ref>和不滿<ref>[http://www.linuxforum.net/forum/showflat.php?Cat=%20&Board=linuxnews&Number=634223&page=0&view=collapsed&sb=5&o=%20&fpart=1 《胡教主大筆一揮》]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。最終在輿論的壓力下，收費計劃以被迫取消而草草收場。<ref>[http://linux.solidot.org/article.pl?sid=06/12/20/0034238 《StarDict取消字典軟件下載收費》]</ref>
