'''星槎大學'''（{{lang-ja|'''星槎大学'''}}）是位於[[日本|日本]][[神奈川縣|神奈川縣]][[足柄下郡|足柄下郡]][[箱根町|箱根町]]的一所私立[[大學|大學]]，2004年正式設立，目前擁有一個學部一個學科。

== 概要 ==
2004年，{{link-ja|學校法人國際學園|学校法人国際学園}}創辦一所主要以社會人士為學生對象的[[函授|通信制]][[大學|大學]]，學生超過半數是[[教師|教師]]或[[公務員|公務員]]。該校教室遍布日本全國，其中[[蘆別市|蘆別]]校區是於1954年興建的磚造建築，原為蘆別市立賴城小學校所使用，2008年3月7日時校舍及體育館成為國家指定的[[登錄有形文化財|登錄有形文化財]]<ref>{{cite web|url=http://kunishitei.bunka.go.jp/bsys/maindetails.asp?register_id=101&item_id=00006962 |title=星槎大学（旧頼城小学校）校舎 |work=[[文化廳|文化廳]] |date=2008-03-19 |accessdate=2018-09-30 }}</ref><ref>{{cite web|url=http://kunishitei.bunka.go.jp/bsys/maindetails.asp?register_id=101&item_id=00006770 |title=星槎大学（旧頼城小学校）体育館 |work=文化廳 |date=2008-03-19 |accessdate=2018-09-30 }}</ref><ref name="hokkaido-np-2007-12-8">“文化審議会 星槎大校舎などを国の有形文化財に 西洋美術館本館は重文に”. [[北海道新聞|北海道新聞]] (北海道新聞社). (2007年12月8日)</ref>。

== 沿革 ==
* 1980年 - 於靜岡縣設立學校法人長谷川學園
* 1984年 - 法人名變更為學校法人國際學園
* 2003年 - 獲得[[文部科學省|文部科學省]]同意設立星槎大學共生科學部共生科學科（通信課程）
* 2004年 - 星槎大學共生科學部共生科學科（通信課程）開學
* 2009年 - 共生科學部共生科學科設置共生科學、初等教育、福祉專攻三門專攻
* 2012年 - 獲得文部科學省同意設立星槎大學大學院教育學研究科（通信課程）
* 2013年 - 校本部從蘆別校區移至箱根校區、設立星槎大學大學院教育學研究科（通信課程）、共生科學部共生科學科新設置體育身體表現專攻
* 2016年 - 獲得文部科學省同意設立星槎大學大學院教育實踐研究科（通學課程）
* 2017年 - 設立大學院教育實踐研究科（通學課程），與停止招募的{{link-ja|日本教育大學院大學|日本教育大学院大学}}合流。
