{{日本作品|星期一的豐滿|月曜日のたわわ}}是日本漫畫家[[比村奇石|比村奇石]]於[[Twitter|Twitter]]上每[[星期一|星期一]]發表的漫畫。

== 作品概要 == 
最早该作以「{{lang|ja|月曜朝の社畜諸兄にたわわをお届けします}}」（給星期一早上的各位[[社畜|社畜]]哥哥送上豐滿）为标题在作者Twitter账户上发布了第一部作品<ref name="netorabo0801">{{Cite web |date= 2016-08-01|url= http://nlab.itmedia.co.jp/nl/articles/1608/01/news020.html|script-title=ja: 「月曜朝の社畜諸兄にたわわをお届けします」　Twitterの人気コンテンツ“月曜日のたわわ”はどうやって生まれたか、漫画家・比村奇石さんに話を聞いた|publisher= {{lang|ja|ねとらぼ}}|accessdate=2016-10-10|language=ja}}</ref>，之后每周连载，2016年连载第46话时改为现名。主题是為了星期一早上成為[[社畜|社畜]]的大家打氣而画的[[巨乳|丰满胸部]]女生插画<ref name="netorabo0801"/>。相关连载也通过同人志在[[Comic_Market|Comic Market]]发售<ref>{{Cite web |date= 2010-10-10|url= http://ure.pia.co.jp/articles/-/64438|script-title=ja: まさかのアニメ化！比村奇石『月曜日のたわわ』YouTubeで配信スタート、全12話を予定|publisher= {{lang|ja|ウレぴあ総研}}|accessdate=2016-10-10|language=ja}}</ref>，至2018年8月已发售了6本单行本。

有些時候作品也與發佈當時的時事有所關聯，例如在[[第24屆日本參議院議員通常選舉|第24屆日本參議院議員通常選舉]]結束後的次日，也就是2016年7月11日星期一，就發布了與選舉有關的插畫作品。

== 網路動畫 == 
2016年10月10日發佈動畫化消息並由[[NBC环球娱乐|NBC环球娱乐]]官方账号在YouTube上发布第1话<ref>{{cite web |url=https://www.youtube.com/watch?v=cfcqewZdBiU |script-title=ja:月曜日のたわわ　その１  |date=2016-10-10 |accessdate=2016-10-11 |archiveurl=https://web.archive.org/web/20161010083005/https://www.youtube.com/watch?v=cfcqewZdBiU |archivedate=2016-10-10 |deadurl=yes}}</ref>，之后由于违反社区规范而被YouTube删除<ref>{{cite web |script-title=ja:アニメ「月曜日のたわわ」サプライズ公開からサプライズ削除へ　全12回での公開のはずが……どうなるの？ |url=http://nlab.itmedia.co.jp/nl/articles/1610/10/news032.html |website={{lang|ja|ねとらぼ}} |date=2016-10-10 |accessdate=2016-10-11|language=ja}}</ref>。之后，有用户名为“比村奇石”的用户重新设定年龄限制后重新上传发布（后自行删除）<ref>{{cite web|title=YouTube太敏感？比村奇石動畫《月曜日のたわわ》疑因猥褻遭刪除|url=http://www.hk01.com/%E7%A7%91%E6%8A%80%E7%8E%A9%E7%89%A9/47718/YouTube%E5%A4%AA%E6%95%8F%E6%84%9F-%E6%AF%94%E6%9D%91%E5%A5%87%E7%9F%B3%E5%8B%95%E7%95%AB-%E6%9C%88%E6%9B%9C%E6%97%A5%E3%81%AE%E3%81%9F%E3%82%8F%E3%82%8F-%E7%96%91%E5%9B%A0%E7%8C%A5%E8%A4%BB%E9%81%AD%E5%88%AA%E9%99%A4|website=[[香港01|香港01]]|date=2016-10-10|accessdate=2016-10-11}}</ref><ref>{{cite web|script-title=ja:月曜日のたわわ　その１|url=https://www.youtube.com/watch?v=mtEEB_PPt0M|website=[[YouTube|YouTube]]|accessdate=2016-10-11}}</ref>。但作者在Twitter上则只对此表示抱歉，并说明由于这是团队作品，所以不会通过自己的个人账号重新上传，并计划使用新途径发布。<ref>{{cite web|title=Tawawa on Monday's Creator Comments on Anime's Removal From YouTube - News - Anime News Network|url=http://www.animenewsnetwork.com/news/2016-10-11/tawawa-on-monday-creator-comments-on-anime-removal-from-youtube/.107493|accessdate=2016-10-25}}</ref><ref>{{cite web|script-title=ja:「月曜日のたわわ」公開当日に削除　YouTubeと原作者のコメントは...【UPDATE】|url=http://www.huffingtonpost.jp/2016/10/10/tawawa_n_12425064.html|accessdate=2016-10-19|language=ja}}</ref>幾天後，NBC环球娱乐的頻道发布的第1话恢复，<ref>{{cite web|script-title=ja:『月曜日のたわわ』ニコ動で再配信！ 第2話はすでに7万再生突破 - KAI-YOU.net|url=http://kai-you.net/article/34324|accessdate=2016-10-19|language=ja}}</ref>但第2话起只在[[NicoNico|NicoNico]]網站發佈。<ref>{{cite web|script-title=ja:YouTubeで削除されたアニメ『月曜日のたわわ』、公開先をニコニコに変更 {{!}} おたくま経済新聞|url=http://otakei.otakuma.net/archives/2016101701.html|accessdate=2016-10-19|language=ja}}</ref><ref>{{cite web|script-title=ja:アニメ「月曜日のたわわ」　配信サイトをYouTubeからニコニコ動画へ変更 -
