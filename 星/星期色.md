'''星期色'''（{{lang-th|สีประจำวัน}}）或'''七天七色'''（{{lang-th|7 วัน 7 สี}}）是[[泰国文化|泰国文化]]中规定每周七天七种颜色的传统，相传来源自[[印度神话|印度神话]]，自[[阿瑜陀耶王朝|阿瑜陀耶王朝]]时期开始有此习惯<ref>{{cite web |url=https://new.qq.com/cmsn/20150402/20150402048360 |title= 泰国诗琳通公主60大寿 曼谷变身“紫色之都”|last= |first= |date=2015-04-02 |website=腾讯网 |publisher=中国新闻网 |access-date=2018-07-24 |quote=}}</ref>。[[却克里王朝|却克里王朝]]开始，王室成员们和上流社会也按照星期色穿着对应颜色的衣服，直到[[泰国|泰国]]改为[[君主立宪制|君主立宪制]]后，由于[[西服|西服]]的流行，这一习俗才逐渐淡化<ref>{{cite web |url=http://go.huanqiu.com/asia/2013-12/4641833.html |title=五彩斑斓的泰国 人们用颜色表示日期 |last= |first= |date=2013-12-09 |website=环球网 |publisher=环球时报 |access-date=2018-07-25 |quote=}}</ref>。

星期色可根据生日的星期来属于代表色，如[[拉玛九世|拉玛九世]]国王的代表色是[[黄色|黄色]]（星期一出生）、[[诗丽吉王后|诗丽吉王后]]的代表色是[[蓝色|蓝色]]（星期五出生）、[[诗琳通公主|诗琳通公主]]的代表色是[[紫色|紫色]]（星期六出生）等<ref>{{cite web |url=http://www.bala.cc/jiudian/6134.html |title=泰国星期颜色：周一至周天不同颜色各有什么含义？ |last= |first= |date=2016-10-09 |website=芭拉旅游网 |publisher= |access-date=2018-07-25 |quote=}}</ref>。
