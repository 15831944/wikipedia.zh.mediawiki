'''星火 (2017高中原創畢業歌合輯)'''<ref>{{Cite web|url=https://zh-tw.facebook.com/singoursong2017/|title=2017高中原創畢業歌合輯|accessdate=2017-10-13|work=zh-tw.facebook.com|language=zh-Hant}}</ref><ref>{{Cite news|url=https://www.ettoday.net/news/20170819/992659.htm?feature=graduationsong2017&tab_id=1101|title=畢業歌直播觀看衝破150萬　全場高校生唱《星火》High翻 {{!}} 2017 ETNEWS高校畢業歌人氣票選 {{!}} 2017 ETNEWS高校畢業歌人氣票選|newspaper=ETNEWS新聞雲|accessdate=2017-10-28|language=zh-TW}}</ref>是收錄全國高中校畢業生音樂的第六張音樂的專輯，由[[擎天娛樂|擎天娛樂事業股份有限公司]]製作，2017年8月19日於[[台灣|台灣]][[擎天娛樂|擎天娛樂事業股份有限公司]]代理發行正式發行。<ref>{{Citation|title=星火2017高中原創畢業歌合輯|date=2017-08-18|url=https://www.kkbox.com/tw/tc/album/27mLJWTOpVTJG0F1fZXB009H-index.html|language=zh-TW|accessdate=2017-11-12}}</ref><ref>{{Cite web|url=http://www.books.com.tw/products/0020199503|title=星火2017高中原創畢業歌合輯 (3CD)|accessdate=2017-10-13|work=博客來}}</ref><ref>{{Cite web|url=http://www.eslite.com/product.aspx?pgid=1004134842618699&kw=%E9%AB%98%E4%B8%AD%E5%8E%9F%E5%89%B5%E7%95%A2%E6%A5%AD%E6%AD%8C%E5%90%88%E8%BC%AF&pi=0|title=星火: 2017高中原創畢業歌合輯 (3CD) - 流行 - CD|accessdate=2017-10-13|work=誠品網路書店}}</ref><ref>{{Cite web|url=http://www.setn.com/News.aspx?NewsID=284238|title=一聽就上癮！用《星火》點亮青春 高中生自創畢業歌感動人 {{!}} 生活 {{!}} 三立新聞網 SETN.COM|accessdate=2017-10-13|last=三立新聞網}}</ref>

從2012年由全台15所高中畢業生聯合創作《風箏》開始的『高中原創畢業歌合輯』，今年已邁入第六個年頭。之後陸續發表的聯合創作《拼圖》、《起飛》、《破浪》、《[[啟程《2016高中原創畢業歌合輯》|啟程]]》，每年均被數百所全國國中小學選為畢業歌，在YOUTUBE也紛紛創造超過百萬次以上的點擊數，原創畢業歌已然在各級校園中蔚為主流風潮。

而入選的30所學校，延續風箏傳統，攜手以作詞組、作曲組、演唱組、MV組、企宣組各小組接棒，聯合創作發表了「'''2017全國高中生大合唱《星火》'''」<ref>{{Citation|last=王風箏|title=：：首播 ：： 星火  2017全國高中生大合唱(官方正式HD版)|date=2017-08-15|url=https://www.youtube.com/watch?v=4hOB0OIo8a0|accessdate=2017-10-29}}</ref>

== 報名 ==
2017年4月7日，「2017高中原創畢業歌合輯」在粉絲專頁開始報名，報名資格為全國各高中職有製作原創畢業歌且經學校認可對外發表者皆可參與報名。

截至報名結束，共計有126所學校完成報名，涵蓋全國以及海外地區，最後由報名之126所學校代表，交互投票選出30首作品，成為「2017高中原創畢業歌合輯」<ref>{{Cite news|url=https://blow.streetvoice.com/35482-%E5%BE%9E%E9%A2%A8%E7%AE%8F%E9%96%8B%E5%A7%8B%EF%BC%9A2017%E5%8E%9F%E5%89%B5%E7%95%A2%E6%A5%AD%E6%AD%8C%E5%90%88%E8%BC%AF%E3%80%8A%E6%98%9F%E7%81%AB%E3%80%8B%E8%AA%95%E7%94%9F/|title=從風箏開始：2017原創畢業歌合輯《星火》誕生 - Blow 吹音樂|date=2017-08-17|newspaper=Blow 吹音樂|accessdate=2017-10-29|language=zh-TW}}</ref>
