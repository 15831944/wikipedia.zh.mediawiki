== 故事簡介 ==
渚沙町有著崇拜星星「凪」的習俗，並會在每年十二月舉辦「星夜祭」，但對於剛搬到這個城市的群雲景與群雲空羽兩兄妹來說並非特別的活動。某一天，景被學校的學姐鳴瀨川凪邀請跟她一起成立讓更多人知道「凪」這顆星星的社團「星夜社」，起初景感到猶豫，但在了解渚沙町的風俗民情後，讓景決定要跟凪一起進行社團活動<ref>{{Cite web|url=http://www.tgsmart.jp/article.aspx?a=14265|title=【特集！】原画・梱枝りこ氏×シナリオ・なかひろ氏が贈るきゃべつそふとデビュー作『星恋＊ティンクル』をスクープ！|publisher=TG Smart|date=2016-06-23|accessdate=2018-05-11|language=ja}}</ref>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|群雲景|群雲 景（むらくも けい）}}
: 作品男主角，渚沙學園二年級生，在阿姨的邀請下和妹妹空羽一起到渚沙町定居。
; {{nihongo|鳴瀨川凪|鳴瀬川 凪（なるせがわ なぎ）|聲：[[遙空|遙空]]<ref name="CHARACTER">{{cite web|url=http://cabbage-soft.com/products/hoshikoi/|title=CHARACTER|publisher=Cabbage Soft|accessdate=2018年6月16日|language=ja}}</ref>}}
: 景的學姐，渚沙學園三年級生。體型嬌小，好奇心旺盛。積極邀請景加入星夜社。
; {{nihongo|群雲空羽|群雲 そらは（むらくも そらは）|聲：秋野花<ref name="CHARACTER"/>}}
: 景的妹妹，渚沙學園一年級生。體弱多病，生活起居都由景照顧，因此養成了依賴景的習慣。
; {{nihongo|織紙珠希|織紙 珠希（おりがみ たまき）|聲：{{lang|ja|ありあ}}<ref name="CHARACTER"/>}}
: 景的同班同學，渚沙學園二年級生。神社的獨生女。性格友善，與剛轉學進來的景馬上就成為了好朋友。喜歡吃烤地瓜。
; {{nihongo|花花見咲良|花々見 咲良（かがみ さくら）|聲：{{lang|ja|藤咲ウサ}}<ref name="CHARACTER"/>}}
: 空羽的同學，渚沙學園一年級生，與珠希是親戚關係。家中經營診療所。[[冒失娘|冒失娘]]，在為學校花園的花澆水時經常把水潑到自己身上。

=== 其他角色 ===
; {{nihongo|莉貝爾|リーベル|聲：藤吉菜子<ref name="CHARACTER"/>}}
: 穿著哥德蘿莉服的年幼少女，總是面無表情。
; {{nihongo|鳴瀨川風子|鳴瀬川 風子（なるせがわ ふうこ）|聲：早瀨彌生<ref name="CHARACTER"/>}}
: 凪的母親，海洋研究所的所長。
; {{nihongo|花花見楓|花々見 楓（かがみ かえで）|聲：[[小倉結衣|小倉結衣]]<ref name="CHARACTER"/>}}
: 咲良的姐姐，渚沙學園的保健老師，個性隨便。喜歡可愛的女孩子。
; {{nihongo|群雲真千|群雲 真千（むらくも まち）|聲：[[澤澤砂羽|澤澤砂羽]]<ref name="CHARACTER"/>}}
: 景和空羽的阿姨，也是兩人母親的妹妹，請兩人到渚沙町的家裡居住後便獨自外出。外表相當年輕，總是帶著笑容。
