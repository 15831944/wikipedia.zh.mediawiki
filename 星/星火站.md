'''星火站'''，位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[东风地区|东风地区]]酒仙桥南，现状为铁路[[二等站_(中国大陆)|二等站]]，同时是[[北京地铁3号线|北京地铁3号线]]和R4线的车站。

==简介==
星火站始建于1966年。车站计划中名为'''辛庄站'''，但因天津已有一个同名车站，且修建时正值[[文化大革命|文化大革命]]，车站乃以当时的“星火[[人民公社|人民公社]]”（现[[六里屯街道|六里屯街道]]）命名为星火站，文革后未再修改<ref>{{cite news|url=http://epaper.bjnews.com.cn/html/2013-08/03/content_454910.htm?div=1|title=47岁的星火站|date=2013-08-03|accessdate=2013-08-04|author=刘春瑞|work=新京报}}</ref>。1968年建成。是东北环铁路线和[[京包铁路|京包线]]中间的作业货运站<ref name=cyqz>{{Cite web |url =  http://www.bjchy.gov.cn/chaoyang/qznj/8a24f09a24d90f1f0124d94c73fe0019.html |title =朝阳区志第七编交通邮电  |author =  |date = 2009-11-09 |publisher =北京朝阳  |language =  |accessdate =  |quote =  }}</ref>。该站隶属[[北京铁路局|北京铁路局]][[通州车务段|通州车务段]]。曾办理客运业务，现已取消，但仍办理货运业务。为[[二等站_(中国大陆)|二等站]]。

截至1990年代中期，星火站占地面积14.9万平方米，建筑面积0.42万平方米，站台呈南北走向，设有基本站台、货物线、仓库、货场及装卸机械等站区设施。当时星火站有通往酒仙桥电子工业区各工厂的数条专用铁路线。从星火站北行与[[中华人民共和国铁道部|铁道部]][[环行铁路|环行铁路]]试验场接轨，经[[望京站_(铁路)|望京站]]与[[望和支线|望和线]]、[[京包铁路|京包线]]沟通。从星火站南行经[[百子湾站_(铁路)|百子湾站]]与东南环铁路线、[[丰沙铁路|丰沙线]]沟通。从星火站南行西折经[[北京东站|北京东站]]抵[[北京站|北京站]]；南行东折沟通[[京承铁路|京承线]]、[[京秦铁路|京秦线]]、[[通坨铁路|通坨线]]<ref name=cyqz/>。
== 新星火站 ==
{{BJ Future infrastructure}}
新星火站为京沈客专首发站，车站规模为7台15线（不含2条原有到发线），站房面积为18.3万m。计划竣工时间为2020年底，届时京沈客专将全线正式通车。本站设计单位为法国AREP及中国铁设联合体，于2017年9月设计中标。站前施工单位为中国中铁六局，站房施工招标于2018年6月开始<ref>{{cite web|url=http://www.bcactc.com/home/gcxx/tdzbgg_new_show.aspx?bulletinid=ff80808163e8cdcf016401e7c9873dec|title=新建北京至沈阳铁路客运专线星火站站房及相关工程  招标资格预审公告（代招标公告）|publisher=北京市建设工程信息网|date=2018-06-15|accessdate=2018-06-18}}</ref>。

本站在[[北京铁路枢纽|北京铁路枢纽]]中规划为七座主要客运站之一，辅助北京站承担[[京沈客运专线|京沈客运专线]]的到发作业<ref>{{cite journal|title=北京铁路枢纽主要客运站分工研究|author=王浩|journal=铁道标准设计|issue=3|year=2012|pages=20-23}}</ref><ref>{{cite web|url=http://hps.mep.gov.cn/jsxm/xmsl/201211/W020121119382996883800.pdf|title=新建北京至沈阳铁路客运专线环境影响报告书简本|accessdate=2013-08-04|author=[[铁道第三勘察设计院集团有限公司|铁道第三勘察设计院集团有限公司]]|publisher=[[中华人民共和国环境保护部|中华人民共和国环境保护部]]}}</ref>。由于车站东部存在居民区，新星火站将旧星火站向西向南改扩建，并设7座站台<ref>{{cite news|url=http://www.fawan.com.cn/html/2013-08/06/content_448204.htm|title=京沈客专将从星火站开跑|date=2013-08-06|publisher=[[法制晚报|法制晚报]]|accessdate=2015-01-24}}{{dead link|date=2018年3月|bot=InternetArchiveBot|fix-attempted=yes}}</ref>。

== 地铁星火站 ==
{{BJ Future infrastructure}}

[[北京地铁3号线|北京地铁3号线]]星火站是3号线的重点车站。2016年，北京地铁3号线一期工程开建，起自[[北京地铁2号线|北京地铁2号线]][[东四十条站|东四十条站]]，终于[[东坝地区|东坝地区]]。2017年，被称为“北京地铁最大标段”的北京地铁3号线[[朝阳公园站|朝阳公园站]]至[[体育中心站_(北京)|体育中心站]]区间段开工建设。根据工期安排，2017年底，北京地铁3号线一期星火站正式开建（规划中的R4线星火站同步提前代建）。除北京地铁3号线外，规划中的R4线将在此设星火站，正在建设的[[京沈高铁|京沈高铁]]也将从星火站始发。星火站建成后，将实现北京地铁3号线与R4线、京沈高铁的换乘，换乘不用出站。根据计划，北京地铁3号线一期将于2021年开通<ref>{{Cite web|url=http://bj.jjj.qq.com/a/20170429/006186.htm|title=北京地铁最大标段开建 星火站换乘高铁不用出站|accessdate=|author=|date=2017-04-29|publisher=腾讯|language=|quote=}}</ref>。
