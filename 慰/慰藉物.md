'''慰藉物'''或'''过渡物'''或'''安全毯'''，是一种物件，用来提供[[心理|心理]]上的慰藉，尤其是在不同寻常的環境裡。[[儿童|儿童]]在就寝时也常需要它们。对孩童来说，慰藉物可能是[[毯子|毯子]]，一个[[填充玩具|填充玩具]]，或者最喜欢的[[玩具|玩具]]，等等。<ref>{{cite web|url=http://pediatrics.about.com/od/infantparentingtips/a/04_loveys.htm|title=Security and Comfort Objects|last=Iannelli, M.D.|first=Dr. Vincent|date=Dec 12, 2004|publisher=About.com|accessdate=2009-07-17}}</ref>

== 儿童心理学上的作用 ==

[[Image:Objet_transitionnel.svg|thumb]]

在儿童身心发展中，术语“'''过渡物'''”经常被使用。它们通常是一种具有心理作用的物件，可以一定程度上代替母亲与孩子的关系。通常的例子有：[[洋娃娃|洋娃娃]]，[[泰迪熊|泰迪熊]]或者毛毯。

== 成人使用的慰藉物 ==
成人也可能使用慰藉物。在一项2008年的研究中，研究者们发现索尼公司的[[AIBO|AIBO]]机器宠物可以减少养老院里老人们的孤独感。<ref>{{cite web|url=http://www.foxnews.com/story/0,2933,334529,00.html|title=Study: Dogs, Robots Cheer Elderly|date=March 3, 2008|publisher=Fox News|accessdate=2009-07-17}}</ref>

日本的成人有时用慰藉物来应对现代生活中的压力。<ref>[http://www.toddlertime.com/dx/regression/security.htm Security Blankets for Adults] Kathi's Mental Health Review, March 25 2002</ref>

==在心理治疗上的用途== 
在西方，有时[[填充玩具|填充玩具]]会被装备在[[消防车|消防车]]或[[警车|警车]]上，这样可以为经历事故乃至[[创伤性休克|创伤性休克]]的受害者提供慰藉。

[[慈善机构|慈善机构]]常常会为灾难的幸存者提供毛毯，被子等慰藉物。<ref>{{cite web |url=http://crafthope.com/2011/05/security-blankets-for-tornado-survivors/ |title=存档副本 |accessdate=2011-08-08 |0=2011-08-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20110813114500/http://crafthope.com/2011/05/security-blankets-for-tornado-survivors/ |archivedate=2011-08-13 }}</ref>

心理学家正在进行用厚毛毯代替[[紧身衣|紧身衣]]的实验。他们发现，有些罹患[[自闭症|自闭症]]的儿童，在接触到慰藉物后，会变得更加平静，不那么激动。<ref>{{cite web |url=http://www.ot-innovations.com/content/view/51/38/ |title=存档副本 |accessdate=2014-10-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20141006165621/http://www.ot-innovations.com/content/view/51/38/ |archivedate=2014-10-06 }}</ref>

== 流行文化 ==
在[[查尔斯·舒兹|查尔斯·舒兹]]所著的《[[花生漫画|花生漫画]]》系列漫画中，'''安全毯'''一词被使用并流行开来。[[奈勒斯·潘贝鲁特|奈勒斯·潘贝鲁特]]把一张毯子叫做“安全与幸福之毯”<ref>http://www.phrases.org.uk/meanings/security-blanket.html</ref><ref>http://www.phrases.org.uk/meanings/security-blanket.html</ref>

慰藉物的概念在电视连续剧《[[老友记|老友记]]》中也被提到。"The One with the Pediatrician"一集中，罗斯提醒钱德勒，他带了一张安全毯到大学里。
