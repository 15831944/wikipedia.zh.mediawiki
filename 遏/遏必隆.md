'''遏必隆'''（{{lang-mnc|ᡝᠪᡳᠯᡠᠨ|z=Ebilun}}；{{bd|？||1673年||catIdx=E}}），或作'''鄂必隆'''，[[钮祜禄氏|钮祜禄氏]]，[[满洲|满洲]][[镶黄旗|镶黄旗]]人，[[后金开国五大臣|后金开国五大臣]]之[[额亦都|额亦都]]第十六子，母为[[努尔哈赤|清太祖]]之女，和硕四公主[[穆库什|穆库什]]。

== 生平 ==
[[天聪|天聪]]八年（1634年），以父荫袭一等昂邦[[章京|章京]]，授[[侍卫|侍卫]]之職。后因罪，被夺世职。

[[崇德|崇德]]六年（1641年），随[[清太宗|清太宗]]进攻明朝，以战功得优赉。崇德七年（1642年），从奉命大将军[[阿巴泰|阿巴泰]]入长城，攻克蓟州；再进山东，攻夏津，以头功授牛录章京世职。

[[顺治|顺治]]二年（1645年），随顺承郡王[[勒克德浑|勒克德浑]]于[[武昌区|武昌]]之役斩杀[[李自成|李自成]]之侄[[李锦|李锦]]，晋封二等甲喇章京。顺治五年（1648年），以所袭其兄图尔格二等公爵令并袭一等公，授议政大臣、[[領侍衛內大臣|領侍衛內大臣]]，加[[三少|少傅]]兼[[东宫三师|太子太傅]]。顺治十八年（1661年），与[[索尼_(清朝)|索尼]]、[[鳌拜|鳌拜]]、[[蘇克薩哈|蘇克薩哈]]三人并受[[顺治帝|顺治帝]]遗诏为辅政大臣。

[[康熙|康熙]]六年（1667年），[[康熙帝|皇帝]]亲政，特封一等公，加太师。适逢鳌拜独断专行，屡次矫诏诛杀大臣。遏必隆身为顾命大臣，却选择明哲保身，不加阻止也未曾弹劾。康熙八年（1669年），康熙帝惩治鳌拜，並命令下遏必隆獄。亦被康亲王[[傑書|杰书]]以十二项罪名弹劾遏必隆，削遏必隆太师之职夺世爵，论死罪。康熙九年（1670年），康熙帝念其为顾命大臣，而且是勋臣之子，命仍以公爵宿卫内廷。康熙十二年（1673年），遏必隆病重，康熙帝亲临府邸慰问。是年病逝，谥號「恪僖」。

==子女==
*[[孝昭仁皇后|孝昭仁皇后]]，[[康熙帝|康熙帝]]第二位皇后，康熙十七年二月二十六日薨，年二十五歲。
*[[温僖贵妃|温僖贵妃]]，[[康熙帝|康熙帝]]妃嬪，皇十子[[胤䄉|胤䄉]]生母，康熙三十三年十一月初三日卒。
*[[阿靈阿_(鈕祜祿氏)|阿靈阿]]，一等公，女為果郡王[[允禮|允禮]]嫡福晉，康熙五十五年卒。
*[[尹德|尹德]]，一等公，孙女为[[乾隆帝|乾隆帝]][[顺贵人|顺贵人]]。

== 遏必隆刀 ==
此腰刀以遏必隆其人而得名。鞘长94cm，刀长60cm，相傳本為遏必隆所用之指揮刀，死后，刀入皇宫。经略[[大学士|大学士]][[傅恒|傅恒]]征[[大金川之戰|大金川]]、大学士[[赛尚阿|赛尚阿]]讨[[太平天国|太平天国]]，曾分别被[[乾隆帝|乾隆帝]]和[[咸丰帝|咸丰帝]]钦赐遏必隆刀，所以刀的一面刻有“遏必隆玲珑刀一，乾隆十三年赐经略大学士公傅恒平定金川用过”，另一面则有“[[咸丰_(年号)|咸丰]]”印一方和“神锋握胜”四字。
