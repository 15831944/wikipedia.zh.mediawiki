'''解析度'''（[[英語|英語]]：Image resolution）泛指量測或顯示系統對細節的分辨能力。此概念可以用[[時間|時間]]、[[空間|空間]]等領域的量測。日常用語中之分辨率多用於影像的清晰度。解析度越高代表影像品質越好，越能表現出更多的細節；但相對的，因為紀錄的資訊越多，檔案也就會越大。[[個人電腦|個人電腦]]裡的影像，可以使用视频處理軟體（例如[[Adobe_Photoshop|Adobe Photoshop]]、[[PhotoImpact|PhotoImpact]]）調整大小、編輯相片等。

== 像素密度與像素總數  ==
;像素密度
:單位面積內的像素數量除以單位面積，單位為PPI（Pixels Per Inch）。像素密度越高，說明像素越密集，5PPI表示每平方英吋有5*5個像素，500PPI表示每平方英吋有500*500個像素，PPI的數值高，圖片和視頻的清晰度就更高。

;像素總數
:圖片、影像的單獨一幀圖所含像素的數量，單位為像素，計算方式為長邊的像素個數乘以短邊的像素個數。

== 定義 ==
#影像處理或信号处理中的空间或时间分辨率：能夠分辨影像中兩个点或线的能力。最好的[[數學|數學]]定义是使用点扩展函数的分布大小。
#信号处理中的频谱分辨率，由观察的时间窗长所决定的所能够分辨的最小的频谱分量的差别。

==单位==
描述分辨率的单位有：[[每英寸点数|dpi]]（点每英寸）、[[lpi_(分辨率)|lpi]]（线每英寸）和[[每英寸像素|ppi]]（每英寸像素）。但只有lpi是描述光学分辨率的尺度的。虽然dpi和ppi也属于分辨率范畴内的单位，但是他们的含义与lpi不同。而且lpi与dpi无法换算，只能凭经验估算。

另外，ppi和dpi经常都会出现混用现象。但是他们所用的领域也存在区别。从技术角度说，“像素”只存在于电脑显示领域，而“点”只出现于打印或印刷领域。

===相关换算===
ppi和lpi可以换算，lpi等于ppi的一半，但是取决于[[取樣定理|取樣定理]]。

== 種類 ==
===频谱分辨率===

=== 影像分辨率 ===
用以描述影像細節分辨能力，同樣適用於[[數位|數位]]影像、[[膠卷|膠卷]]影像、及其他類型影像。常用'線每毫米'、'線每[[英寸|英寸]]'等來衡量。

通常，「解析度」被表示成每一個方向上的像素數量，比如640x480等。而在某些情況下，它也可以同時表示成「每英寸像素」（pixels per inch，ppi）以及圖形的寬度和高度。比如72ppi，和8x6英寸。

===光學分辨率===
{{see also|光学分辨率}}
與影像分辨率似，惟著重光學透鏡重效果。常用Optical Transfer Function（OTF），Modulation Transfer Function（MTF）來衡量。

=== 顯示分辨率 ===
{{See also|显示分辨率列表}}
對[[電腦|電腦]][[顯示器|顯示器]]等，分辨率是用[[像素|像素]]數目衡量；對數位文件[[印刷|印刷]]，分辨率是通常用每英寸所含點或像素〔dpi〕來衡量。

===数码相机分辨率===
[[像素|像素]]數目大的[[數位相機|數位相機]]，較能够输出高分辨率的图像。相機制造商多数使用像素數目代表分辨率。
