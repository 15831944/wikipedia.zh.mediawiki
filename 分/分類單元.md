'''分類單元'''（'''分類群'''，{{lang-de|Taxon}}）是指[[分類學|分類學]]上的一個群體<ref name="VC1">[http://ibot.sav.sk/icbn/main.htm Vienna Code] {{webarchive|url=https://web.archive.org/web/20121006231936/http://ibot.sav.sk/icbn/main.htm |date=2012-10-06 }} ，「[[國際植物命名法規|國際植物命名法規]]中對此字的釋義較簡單，不管何種[[分類階層|階層]]，是分類學上的一個群體」。</ref><ref name="ICZNg">[http://www.nhm.ac.uk/hosted-sites/iczn/code/ ICZN / Glossary: '''taxon''']，[[國際動物命名規約|國際動物命名規約]]對於分類群的解釋較多，定義分類為「一個分類學上的單元，不管是否具有學名；一個被認為具親緣關係的物種之一個[[族群|族群]]或多個族群，具有能分別出相似的其他一群之共同的[[特徵|特徵]]之單元。一個分類群包含了較低階層的所有分類群及其中的[[生物|個體]]。」。</ref>，不管處哪一個'''[[分類階層|分類階層]]'''（taxonomic rank），稱此群體為'''分類群'''。

==範圍與命名==
各個'''分類階層'''皆可能具多個分類群，而其整體亦為一個分類群。分類群可以給定一個名字，但若指整個分類階層，則其命名便要受國際命名法規規範，例如種應有[[種|種]]的[[學名|學名]]、[[綱|綱]]需有綱的[[學名_(植物)|學名]]。

以[[槐树|槐]]為例，以下列出十個分類階層，主要的分階層（principal ranks）<ref name="VC3">[http://ibot.sav.sk/icbn/frameset/0007Ch1Art003.htm Vienna Code 3.1] </ref> 指界、門、綱、目、科、屬、種等七個，若有需要可以在七個主要的階層之上或之下，再加合適的次級階層（secondary ranks，例如科與屬之間的族、組、系，種下的亞，或是亞科、亞屬等等）<ref name="VC4">[http://ibot.sav.sk/icbn/frameset/0008Ch1Art004.htm Vienna Code 4.1, 4.2] </ref>。每個階層皆有其特定的學名，學名則受到[[國際植物命名法規|命名法規]]的規範。
