'''分眾傳媒'''（{{lang-en|'''Focus Media Network Limited'''}}，簡稱'''Focus Media Network'''，以及'''Focus Media'''，{{hkex|8112}}），於2000年由[[黃雄基|黃雄基]]（主席及總裁）創立名為「i MH」（前身為Wanna Click.com Holdings Limited）。<ref>[http://www.cnyes.com/usastock/news_cn/FMCN.html 鉅亨網]</ref>

現時業務是在全球經營眾多大型辦公室及商業大廈的電梯大堂內[[平面顯示器|平面顯示器]][[廣告|廣告]]播放<ref>{{Cite web |url=http://www.focusmedia.tv/c_index.html# |title=Focus Media Network Limited簡介 |access-date=2012-07-23 |archive-url=https://web.archive.org/web/20120618142328/http://www.focusmedia.tv/c_index.html# |archive-date=2012-06-18 |dead-url=yes }}</ref>，股份則以招股價為0.72港元，發行新股為8200萬股，集資額為5900萬港元，股份則在2011年7月28日於[[港交所|港交所]][[香港創業板|創業板]]上市，其中主要策略股東包括[[其士國際|其士國際]]、[[王氏國際|王氏國際]]、[[城市發展|城市發展]]董事[[郭令裕|郭令裕]]（2015年11月16日去世）等<ref>{{Cite web |url=http://www.focusmedia.tv/chi/shareholder.html# |title=Focus Media Network Limited主要股東 |access-date=2012-07-23 |archive-url=https://web.archive.org/web/20120615234217/http://www.focusmedia.tv/chi/shareholder.html# |archive-date=2012-06-15 |dead-url=yes }}</ref><ref>[http://www.focusmedia.tv/PDF/FocusMedia-prospectus-CHI.pdf Focus Media Network Limited提供招股資料]{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
