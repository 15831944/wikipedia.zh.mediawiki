'''衝吧烈子'''（アグレッシブれつこ，[[英語|英語]]: Aggretsuko）是[[三麗鷗|三麗鷗]]創作的角色。也叫做衝烈子<ref name="cnn170924">{{Cite news |title=衝吧烈子 三麗鷗的新角色會喝啤酒也最喜歡重金屬搖滾 |newspaper=CNN.co.jp |date=2017-09-24 |author= |url=https://www.cnn.co.jp/showbiz/35100096.html |accessdate=2017-12-30 |publisher=CNN |page= |archiveurl= |archivedate = }}</ref>。

== 概要 ==
角色造型為[[小熊貓|小熊貓]]，由イエティ設計，平常個性溫和而不擅長向別人表達自己的感受，但碰上不滿或不開心的事情時，會自己在廁所或卡啦OK中，改變自己的性格並且大唱死亡金屬搖滾，透過吼唱的方式來一掃心中的不滿，2015年，因為在三麗鷗舉辦的「キャラリーマン總選舉」獲得第8名而因此出道<ref name="tvanime" />。

描述擔任[[OL|OL]]的烈子對於上司或同事的行動不滿而無法反擊時，會前往[[卡啦OK|卡啦OK]]用[[死亡金屬|死亡金屬]]的方式唱歌來一掃心中的不滿。

海外以「Aggretsuko」的標題播放，並受到同樣充滿職場壓力的歐美女性支持<ref>{{Cite web |date= 2017-02-13|url= http://www.bbc.com/news/world-asia-38606355|title= The angry red panda that is Japan's new working woman|publisher= bbc news|accessdate=2017-12-29}}</ref><ref>{{Cite web |date= 2017-12-08|url= http://jp.ign.com/aggressive-retsuko/20117/news/olnetflix|title= OL唱死亡重金屬來一掃心中不滿，三麗鷗角色「衝吧烈子」竟然在Netflix動畫化 |publisher= IGN Japan|accessdate=2017-12-29}}</ref>。

== 登場角色 ==
*以下角色資訊及配音以Netflix為主

;烈子(れつこ)
: 声： [[Kaolip|Kaolip]](正常時)；[[ラレコ|ラレコ]](死亡金屬烈子)
: 角色造型為[[小熊貓|小熊貓]]，本作女主角，25歲，是貿易公司的會計[[OL|OL]]。
:個性溫和而不擅長表達自己內心的想法，經常被上司或前輩無理地刁難或要求加班，因此下班後會前往卡啦OK並化身為死亡金屬烈子，用死亡金屬的方式大聲吼唱來消除內心一天的不滿及壓力。
:雖然喜歡做著不切實際的白日夢，但隨著同事以及鷺美、五里的幫助下，而逐漸成長，並終於能夠勇敢地對上司及前輩表達心聲，也主動向零佐助提出分手的要求。

;鷺美(わしみ)
: 声：[[小岩崎小惠|小岩崎小惠]]
: 角色造型為[[蛇鷲|蛇鷲]]，是董事長秘書。
:身材姣好而優雅，自信又意志堅強，頭腦也非常好，與五里在瑜珈教室中認識烈子並因此成為好友，經常給予烈子很多好建議。
:面對懦弱的董事長，會適當給予建議，必要時也會強勢地威脅董事長接受她的建議，某程度展現了蛇鷲的狩獵方式。

;敦(とん)
: 声：[[荒井聰太|荒井聰太]]
: 角色造型為[[豬|豬]]，會計部門的部長，也是烈子等人的上司。
:相當[[沙文主義|沙文主義]]，對烈子經常冷嘲熱諷，帶給她許多壓力和工作。但他本人其實工作經驗豐富，也相當有能力，雖然不擅長使用電腦，但使用算盤計算的能力，卻遠遠勝過使用電腦的下屬。
:一度因為被鷺美向董事長揭發他對烈子職場騷擾以及性別歧視的事情而有所收斂，但其實暗中對烈子懷恨在心。在烈子對他用死亡金屬的方式吼出自己的心聲以後，才進而發現她的內心想法並跟她和解。
:愛好打[[高爾夫球|高爾夫球]]，喜歡在公司內一邊練習揮桿，一邊裝忙。

;小宮(こみや)
: 声：荒井聰太
: 角色造型為[[狐獴|狐獴]]，會計部門的一員，常常在巴結敦，當他的小跟班，喜歡仗著他的權勢對其他人狐假虎威。

;笛音子(ふぇねこ)
: 声：[[井上里奈|井上里奈]]
: 角色造型為[[耳廓狐|耳廓狐]]，會計部門的一員，也是烈子的同事和好友。
:擅長觀察和收集資訊，推理能力也非常強，甚至讓灰田嚇到說不敢與她為敵，喜歡捉弄暗戀烈子的灰田，也努力暗中湊合兩人在一起。

;角田 (つのた)
: 声：井上里奈
: 角色造型為[[鹿|鹿]]，會計部門的一員，也是烈子的同事和後輩。
:個性八面玲瓏並善於討好別人，也喜歡裝無辜和裝可愛，帶著小惡魔的性格，但本性其實不壞。

;灰田(ハイだ)
: 声：[[加藤慎吾|加藤慎吾]]
: 角色造型為[[斑鬣狗|斑鬣狗]]，會計部門的一員，也是烈子的同事。
:個性善良，常常會找機會與烈子攀談，認識烈子五年，並暗戀著她，卻一直苦無機會訴說心意，最後在烈子探病時，親自向她告白，並似乎得到烈子的同意。
:學生時代似乎愛好搖滾，還留有一身皮衣的裝扮和吉他。

;零佐助(れさすけ)
: 声：加藤慎吾
: 角色造型為[[小熊貓|小熊貓]]，在業務部門工作，個性溫和而且遲鈍，經常小錯不斷，所以常常要自掏腰包請客賠罪，因而得到自掏腰包王子的稱號。
: 因某次前往會計部門送件的契機而見到烈子，之後在跟其他同事與烈子等人進行的聯誼中，認識了烈子而開始與她交往。但後來烈子發現兩人個性其實不合，於是由烈子主動帶他前往卡啦OK，並且當面向他展現真正的自己並唱出內心的心聲後，跟他分手。
:似乎喜歡栽種植物，住處內也有許多植栽。

;五里(ごり)
: 声：[[鶴田真希|鶴田真希]]
: 角色造型為[[大猩猩|大猩猩]]，公司的營銷總監，經常和鷺美走在一起，年紀似乎比鷺美和烈子還大。
:個性纖細，身體卻非常強壯，他與鷺美在瑜珈教室中認識烈子，之後透過一起唱卡啦OK的機會而成為好友，之後與鷺美一起暗中支持著烈子。
:似乎有男朋友，但因為忙於工作的關係而分手。

;坪根(つぼね)
: 声：鶴田真希
: 角色造型為[[蟒蛇|蟒蛇]]，會計部分的資深員工
:個性陰險，喜歡對烈子找碴，經常會要求烈子幫她打開難以打開的罐子，曾有一度在自己家裡要打開難以開啟的罐子，卻因此弄斷手腕。
