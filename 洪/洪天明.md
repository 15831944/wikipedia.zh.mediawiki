'''洪天明'''（'''{{lang|en|Timmy Hung}}'''，{{bd|1974年|7月1日|catIdx=H洪}}）[[香港|香港]]出生，中韓[[混血兒|混血兒]]。香港演員及主持人，著名演員及導演[[洪金寶|洪金寶]]的長子。現已婚，妻子是2006年度[[香港小姐|香港小姐]]亞軍[[周家蔚|周家蔚]]。

== 婚姻狀況 ==
洪天明妻子是2006年度[[香港小姐競選|香港小姐競選]]亞軍[[周家蔚|周家蔚]]，兩人於2012年3月21日結婚，當晚，洪天明和周家蔚在[[W酒店_(香港)|W Hotel]]舉行婚禮，婚礼仪式按照中华传统习俗“接新娘、玩新郎、踢轿门、背新娘、敬茶”一步步完成，出席婚礼的人士包括了香港演艺界的名人，包括[[成龍|成龍]]，[[張學友|張學友]]以及[[劉德華|劉德華]]。<ref>{{cite web |url=http://ent.ifeng.com/photo/hot/detail_2012_03/22/13369523_0.shtml |title=洪金宝儿子与港姐大婚 香港群星道贺 |accessdate= 2012年|work= |publisher=凤凰网 |language =zh-cn}}</ref><ref>{{cite web |url=http://yule.sohu.com/20120322/n338511711.shtml |title=洪金宝娶媳妇群星道贺 洪天明与周家蔚玩神秘 |accessdate= 2012年|work= |publisher= 搜狐网|language =zh-cn}}</ref><ref>{{cite web |url=http://ent.ifeng.com/idolnews/hk/detail_2012_03/22/13367230_0.shtml |title=洪天明周家蔚昨日结婚 获好友赠送专属唯一App |accessdate= 2012年|work= |publisher=凤凰网 |language =zh-cn}}</ref>現育有兩子，分別在2013年及2015年出生。

== 軼聞 ==
洪天明現居於[[將軍澳|將軍澳]][[馬游塘村|馬游塘村]]。2012年4月29日，其寓所被一名32歲[[哥倫比亞|哥倫比亞]]籍的賊人潛入搜掠，損失包括結婚時獲贈的金器及現金總值10萬元。<ref>{{cite web |url=http://hk.news.yahoo.com/爆竊洪天明鄰居-即赴蘭桂坊消遣-哥國笨賊碌-贓卡-被捕-220703330.html |title=爆竊洪天明鄰居 即赴蘭桂坊消遣 哥國笨賊碌「贓卡」被捕 |accessdate= 2013年1月2日|work= |publisher= 《明報》|language =zh-cn}}</ref>
