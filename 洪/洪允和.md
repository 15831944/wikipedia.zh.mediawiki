'''洪允和'''<ref>[http://tv.naver.com/v/1436769/list/112569 TV朝鮮電視台《人生紀錄片My Way》E33 170209 播出 '金大熙篇'，洪允和列在金大熙主演的中韓合拍電影《歐巴》特別出演名單中，中文字幕對應她的漢字正名為洪允和。]</ref>（{{kor|k='''홍윤화'''|rm='''Hong Yun Hwa'''|hanja='''洪允和'''}}，{{bd|1988年|7月16日}}），[[韓國|韓國]][[喜劇演員|喜劇演員]]和[[主持人|主持人]]，她最為人所知的作品是[[SBS_(韓國)|SBS]]喜劇節目《[[尋笑人|尋笑人]]》，並憑此獲得2016年第43屆[[韓國放送大獎|韓國放送大獎]]喜劇演員獎<ref>[http://kpopn.niusnews.com/=K0h3tk32 【新聞】第43屆《韓國放送大賞》得獎名單]</ref>。她同時也是戲劇女演員，參演許多韓劇、電影和音樂劇。

== 演藝經歷 ==
* 2006年通過特別招募成為[[SBS_(韓國)|SBS]]第八期喜劇演員，出道當時為17歲高中生。
* 她堅守[[SBS_(韓國)|SBS]]喜劇舞台11年，被稱為「尋笑人之花」、「尋笑人吉祥物」，創造了許多經典單元和流行語，例如時而溫柔時而神經質的〈擁抱火腿的月亮〉<ref>[https://www.youtube.com/watch?v=hSbK-zilVfc 【影片】擁抱火腿的月亮]</ref>、講話童真可愛又恐怖腹黑的電視兒童〈允和七歲〉<ref>[https://www.youtube.com/watch?v=Aw6L9vwe1ys 【影片】允和七歲]</ref>、犯花癡迷上老師卻嘴硬否認的〈撲通撲通民基老師〉<ref>[https://www.youtube.com/watch?v=2QOfmLhIvJE 【影片】撲通撲通民基老師]</ref>、模仿料理研究家Big Mama{{link-ko|李惠貞|이혜정}}維妙維肖的〈白主婦TV〉<ref>[https://www.youtube.com/watch?v=FJgiBOxGx_Q 【影片】白主婦TV]</ref>，以及狂虐兩位保鑣的怪力富千金〈守護小姐〉<ref>[https://www.youtube.com/watch?v=TRDGU9sDlrg 【影片】守護小姐]</ref>等，其展現各種面貌的出色演技受到觀眾歡迎，多次獲得[[SBS演藝大賞|SBS演藝大賞]]肯定，更在2016年、2017年兩度入圍[[百想藝術大賞|百想藝術大賞]]女子綜藝賞，並獲得2016年第43屆[[韓國放送大獎|韓國放送大獎]]最佳喜劇演員獎。
* 她有著胖嘟嘟的身材、可愛的笑臉和爽朗的笑聲，與[[文化廣播_(韓國)|MBC]][[李菊主|李菊主]]、[[韓國放送公社|KBS]]{{link-ko|金珉京|김민경 (희극인)}}合稱三大社小豬。經紀公司曾企劃三人組團「247」（意指合計247公斤，實際體重高於247），但三人一練舞便汗如雨下，一首歌也練不成，出道計劃只能作罷。<ref>[http://www.helloidol.com/12839.html 《Healing Camp》E153 141006 金俊昊 李國主 洪允和]</ref><ref>{{Cite web |url=http://lemonidolshow.com/2737.html |title=《Radio Star》E454 洪允和 |access-date=2017-09-27 |archive-url=https://web.archive.org/web/20170928005639/http://lemonidolshow.com/2737.html |archive-date=2017-09-28 |dead-url=yes }}</ref>
* [[SBS_(韓國)|SBS]]《[[尋笑人|尋笑人]]》於2017年5月31日停播，2017年7月30日她加入[[tvN|tvN]]《[[喜劇大聯盟|喜劇大聯盟]]》。<ref>[http://movie.daum.net/tv/main?tvProgramId=80731 《喜劇大聯盟》〈Best Friend〉]</ref>

== 個人生活 ==
* 酷愛粉紅色，服裝造型、居家擺設常使用粉色系。<ref name="《他人的取向》">{{Cite web |url=http://lemonidolshow.com/9462.html |title=【影片】《今生人類報告書-他人的取向》E06 |access-date=2017-09-12 |archive-url=https://web.archive.org/web/20170912233639/http://lemonidolshow.com/9462.html |archive-date=2017-09-12 |dead-url=yes }}</ref>興趣為縫紉，會親手縫製女裝、布包、潛水衣等。
* 姊姊'''洪善和'''（홍선화），母親'''金渼玹'''（김미현）曾從事美髮、保險、廣告模特。
* 與同為喜劇演員的室友{{link-ko|尹孝童|윤효동}}住在簡陋的半地下室月租房，兩人同睡一張床，直到尹孝童2016年在《[[尋笑人|尋笑人]]》出道之前，由洪允和負擔兩人大部份生活費長達八年。<ref>[http://www.dailymotion.com/video/k15dGH1VZPtPUqgfhb2 【影片】《舊屋變新居》E14 160317]</ref>
* 與喜劇演員{{link-ko|金民基|김민기 (희극인)}}相識兩年後，從同事發展為戀人，自2010年4月19日開始愛情長跑，交往不到一年，金民基便公開向她求婚<ref>[https://www.youtube.com/watch?v=G_k5op6OEgg 【影片】金民基第一次向洪允和求婚]</ref>；2016年9月14日《明天是開球王》節目中，金民基再次公開向她求婚，並當場得到洪母的首肯。<ref name="《開球王》">[https://www.youtube.com/watch?v=vjtOyT5sRdY&t=11 【影片】《明天是開球王》洪允和＋金民基]</ref><ref name="求婚">[http://kpopn.niusnews.com/=K19ycrb6 【新聞】洪允和在《開球王》收到男友的求婚]</ref> 2017年8月17日《[[現場脫口秀Taxi|現場脫口秀Taxi]]》節目中，決定2018年秋天結婚。<ref>[http://allthatstar.com/tw/view.php?action=5205&postID=355126 【新聞】《TAXI》洪允和金民基交往8年曝喜訊：明年秋天結婚]{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
* 2018年1月3日洪允和在電台節目《{{link-ko|金信英的正午希望曲|정오의 희망곡}}》透露兩人已於2017年11月11日完成雙方家長見面禮，計劃2018年11月舉行婚禮。<ref>[http://www.allthatstar.com/view.php?action=1791&postID=382957 【新聞】洪允和曝婚訊 預計11月與金民基結婚]{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
* 2018年11月17日洪允和與金民基在首爾瑞草區蠶院洞的河畔酒店舉行非公開婚禮，順利完婚。<ref>[http://www.mbcsportsplus.com/news/?mode=view&cate=%20&b_idx=99872933.000#07D0 【新聞】洪允和♥金民基 結束8年愛情長跑結婚 司儀梁世炯、祝歌高佑鎮]</ref>
* 兩人在《[[尋笑人|尋笑人]]》《{{link-ko|Gag Tonight|개그투나잇}}》合作過〈洪女士〉、〈擁抱火腿的月亮〉、〈民基的煩惱〉、〈成浩啊〉、〈允和七歲〉、〈洪培達〉、〈撲通撲通民基老師〉等單元，不時在舞台上放閃曬恩愛。
