'''洪珉绮'''（{{Kor|諺=홍민기|漢=洪珉綺|羅=Hong Min-gi|f=y}}，{{bd|1992年|10月5日}}，绰号：'''MadLife'''），韩国《[[英雄联盟|英雄联盟]]》职业选手，原为韩国{{link-en|CJ Entus|CJ Entus}}战队辅助选手，曾效力北美[[Gold_Coin_United|Gold Coin United]]，已現在宣佈退役。<ref>{{Cite web |url = https://www.invenglobal.com/articles/5367/lck-legend-madlife-officially-announces-his-retirement |title = LCK Legend, "Madlife" Officially Announces His Retirement|date = 2018年6月19日 |publisher =  Inven Global |language = en}}</ref>

==早年经历==

洪珉绮于1992年10月5日出生于[[韩国|韩国]][[首尔|首尔]]，7岁时接触《[[星际争霸_(游戏)|星际争霸]]》，2010年9月，他在朋友的推荐下加入《[[英雄联盟|英雄联盟]]》，一开始玩阿卡丽，到后来转玩ADC位置。<ref>{{Cite web |url = http://games.qq.com/a/20151201/041749.htm |title = 从人到神的蜕变，永不放弃的Madlife |author = 糖果 |date = 2015年12月1日 |publisher = 腾讯游戏 |language = zh-cn }}</ref>

==职业生涯==

2012年，洪珉绮加入MIG FROST战队，并担任辅助位置，随后参加了2012年OGN，并一路打进决赛，然而他所在的MIG FROST战队在总决赛中负于MIG BLAZE，仅获得亚军。<ref>{{Cite web |url = http://wangyou.pcgames.com.cn/zhuanti/lol/news/event/1203/2479292.html |title = OGN2012英雄联盟LOL春季联赛视频直播专题报道 |date = 2012年3月21日 |publisher = 太平洋游戏网 |language = zh-cn }}</ref>

在随后的2012OGN夏季赛中，洪珉绮所在的MIG FROST战队再次打进总决赛，在总决赛中，双方比分曾一度扳平，但在最后的盲选局中，MadLife使用布里茨并取得胜利，最终促使MIG FROST让2追3击败CLG.EU取得OGN冠军，<ref>{{Cite web |url = http://www.52pk.com/gl/6521542.shtml |title = 2012年OGN夏季赛决赛：AZUBU.Frost VS CLG.eu |author = FT |date = 2015年8月29日 |publisher = 52PK游戏网 |language = zh-cn }}</ref>MadLife自此封神。同时，他也成为第一个封神的英雄联盟选手。

在此之后，由于Azubu Frost在2012OGN春季赛和夏季赛的表现，使得该战队取得了[[英雄联盟2012赛季全球总决赛|英雄联盟2012赛季全球总决赛]]的参赛资格。在英雄联盟2012赛季全球总决赛的小组赛中，该战队3战全胜，并以小组第一的成绩从A组出线。随后在8强赛Azubu Frost对阵[[Team_SoloMid|Team SoloMid]]的比赛中，Madlife带领他所在的队伍Azubu Frost以2：0的成绩成功晋级半决赛。在半决赛对阵CLG Europe的比赛中，Azubu Frost虽然在第一局负于CLG Europe，但在Madlife的努力下，该队还是以2：1的成绩击败CLG Europe，进入最后的冠军赛。<ref>{{Cite web |url = http://wangyou.pcgames.com.cn/zhuanti/lol/video/events/1210/2655488.html |title = S2全球冠军赛半决赛 CLG.EU VS Frost|date = 2012年10月11日 |publisher = 太平洋游戏网 |language = zh-cn }}</ref>在冠军赛上，虽然Azubu Frost在第一局中获胜，但最终还是以1：3负于[[台北暗杀星|台北暗杀星]]，未能取得冠军。<ref>{{cite news | title =英雄聯盟 台北暗殺星奪冠 | url = http://www.cna.com.tw/News/FirstNews/201210140037.aspx | publisher = 中央通訊社 |author = 吳協昌 | date=2012-10-14 | language = zh-tw}}</ref>

在2016LCK夏季赛中，CJ Entus以3胜15败的成绩排名第10，并进入2017LCK春季季前赛升降赛。在升降赛的第一场中，CJ Entus以0：2负于{{link-ko|KONGDOO Monster|콩두 몬스터}}
，并进入败者组，<ref>{{Cite web |url = http://lol.178.com/201608/265920897065.html |title = 2017LCK春季升降赛第一轮：CJ惊险进入败者组 |author = saralolita |date = 2016年8月23日 |publisher = 178lol英雄联盟 |language = zh-cn }}</ref>在随后的败者组中，CJ Entus击败了[[SBENU_SONICBOOM|SBENU SONICBOOM]]，但在8月27日的最终战上，CJ Entus以0：3负于[[ESC_Ever|ESC Ever]]，并降级至[[Challengers_Korea|Challengers Korea]]。<ref>{{Cite web |url = http://lol.17173.com/news/08282016/122524923.shtml |title = 名门CJ降级 辅助之神MadLife告别LCK |author = 可喜@可乐 |date = 2016年8月23日 |publisher = 17173 |language = zh-cn |access-date = 2016-12-23 |archive-url = https://web.archive.org/web/20160830083247/http://lol.17173.com/news/08282016/122524923.shtml |archive-date = 2016-08-30 |dead-url = yes }}</ref>

在2016年11月9日所举办的韩国2016KeSPA杯的十二强赛资格赛中，CJ Entus2：0负于ESC EVER，最终被淘汰。<ref>{{Cite web |url = http://www.newyx.net/gl/437655_1.htm |title = 《LOL》2016韩国KeSPA杯 ROX夺得冠军 |author = 追风少年 |date = 2016年11月19日 |publisher = 牛游戏网攻略 |language = zh-cn }}</ref>

在此之后，洪珉绮以47.6%的得票率成为[[英雄联盟2016全明星赛|英雄联盟2016全明星赛]]韩国明星队的辅助选手。<ref>{{Cite web |url = http://m.zol.com.cn/article/6131237.html |title = 韩国LCK全明星出炉 M神Madlife再次入选 完 |author = 赵苏凯 |date = 2016年11月9日 |publisher = 中关村在线 |language = zh-cn }}</ref>但在英雄联盟2016全明星赛的射手模式上，MadLife所在的无限烈焰队以0：1负于绝对零度队，<ref>{{Cite web |url = http://games.sina.com.cn/yxbk/wy/lol/2016-12-09/doc-ifxypipu7472335.shtml |title = LOL全明星赛射手模式规则介绍 我狗爆炸输出赢积分 |date = 2016年12月9日 |publisher = 新浪游戏 |language = zh-cn }}</ref>随后，他又在1V1淘汰赛第一天对阵Uzi的比赛中以0：1负于Uzi，并被淘汰。<ref>{{Cite web |url = http://www.lesports.com/news/945198019.html |title = 2016英雄联盟全明星赛 Madlife vs UZI |date = 2016年12月9日 |publisher = 乐视体育 |language = zh-cn }}</ref>

2016年11月30日晚，CJ Entus通过官方[[SNS|SNS]]宣布该战队所有选手续约失败，并结束该战队所有选手与该战队的合约，Madlife亦因此与其他CJ Entus战队的英雄联盟分部队员一起离队。<ref>{{Cite web |url = http://lol.mingrent.com/news/393609.html |title = Madlife终和CJ说再见 CJ战队全员离队 |author = 塞上云 |date = 2016年12月2日 |publisher = 游戏名人堂 |language = zh-cn |deadurl = yes |archiveurl = https://web.archive.org/web/20161220123931/http://lol.mingrent.com/news/393609.html |archivedate = 2016年12月20日 |df =  }}</ref>

2016年12月29日，洪珉绮加入北美[[Gold_Coin_United|Gold Coin United]]战队。<ref>{{Cite web |url = http://wangyou.pcgames.com.cn/656/6560713.html |title = Madlife加入NA GCU战队 征战挑战者联赛 |author = 小布 |date = 2016年12月30日 |publisher = 太平洋游戏网 |language = zh-cn }}</ref>
