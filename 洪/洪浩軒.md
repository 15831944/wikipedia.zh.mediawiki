'''洪浩軒'''（遊戲代號：'''Karsa'''，{{bd|1997年|2月14日}}）是一名《[[英雄聯盟|英雄聯盟]]》職業電競選手，以其積極進攻的打法著名。<ref>{{Cite web|url = http://www.ironforum.com.tw/subjdetail/127/2|title = 《IEM特稿》MMD x Karsa：yoe閃電狼新秀專訪|date = 2015年2月2日|publisher = 鉄人論壇|access-date = 2017-12-14|archive-url = https://web.archive.org/web/20161113064719/http://www.ironforum.com.tw/subjdetail/127/2|archive-date = 2016-11-13|dead-url = yes}}</ref><ref>{{Cite web |url = http://www.flashwolves.com/TeamsMemberIntro.aspx?player=22|title = 閃電狼官方網站選手介紹 }}</ref><ref>{{Cite web |url = https://newtalk.tw/news/view/2017-03-13/82756|title = 獨家專訪！ 閃電狼的核心：隊長Karsa|date = 2017年3月13日 |publisher = 新頭殼newtalk }}</ref>
目前效力於LPL的[[Royal_Never_Give_Up|Royal Never Give Up]]戰隊。
