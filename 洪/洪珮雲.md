'''洪珮雲'''（{{bd|2001年|1月7日|catIdx=Hong, Peiyun}}），中國偶像團體[[SNH48|SNH48]]的成員，所屬隊伍為TEAM HII。她於2015年7月25日的[[SNH48第二屆總選舉|SNH48第二屆總選舉]]中披露，成為SNH48的5期生。在2015年12月4日，於《[[Team_B_5th_Stage「劇場的女神」#SNH48_Team_XII「劇場女神」公演|劇場女神]]》公演正式出道<ref name="sina">{{Cite web |url = http://ent.sina.com.cn/y/yneidi/2015-12-07/doc-ifxmhqac0105704.shtml |title = SNH48五期生宣布出道 4日上海献公演首秀 |date = 2015-12-07 |accessdate = 2016-01-15 |publisher = 新浪娛樂 }}</ref>。她同時也是[[Color_Girls|Color Girls]]的成員。

{{fact|名字由台灣算命師取名，因此使用繁體字寫法。}}

== 經歷 ==
2015年7月25日，在[[SNH48第二屆總選舉|SNH48第二屆總選舉]]中披露，成為SNH48的5期生。12月9日，於《[[Team_B_5th_Stage「劇場的女神」#SNH48_Team_XII「劇場女神」公演|劇場女神]]》公演出道<ref name="sina" /><ref name="wmxa">{{Cite web|url=http://www.wmxa.cn/a/201512/291903_2.html |title=snh48五期生上海公演正式出道 完整成员名单流出颜值爆表 |date=2015-12-08 |accessdate=2016-01-15 |publisher=西安文明網 |deadurl=yes |archiveurl=https://web.archive.org/web/20160512193236/http://www.wmxa.cn/a/201512/291903_2.html |archivedate=2016-05-12 }}</ref>。12月28日，參與了「[[SNH48第二屆年度金曲大賞BEST_30|SNH48第二屆年度金曲大賞BEST 30]]」<ref>{{cite web|title=SNH48第二届年度金曲大赏 《夜蝶》摘取桂冠|url=http://ent.163.com/photoview/00B60003/583039.html#p=BBQ3UM7C00B60003|website=网易|accessdate=2017-02-26|date=2015-12-26}}</ref>。

2016年4月20日，加入團體Color Girls<ref>[http://www.snh48.com/html/allnews/zixun/2016/0420/1646.html  SNH48"2016丝芭传媒战略发布会" 力推多个新型态女子偶像组合]</ref>。7月30日，於SNH48第三屆總選舉，獲得第31名，最終獲得16288.2票，入選第三屆總選舉Under Girls<ref>[http://www.snh48.com/html/allnews/zixun/2016/0730/1873.html “比翼齐飞”SNH48第三届偶像年度人气总决选最终结果公布]</ref>。12月31日，參加湖南衛視跨年晚會。

2017年1月27日，作为表演者之一，与SNH48的其他数十名成员参与了[[2017年中国中央电视台春节联欢晚会|2017年央视春晚]]上海分会场的节目。7月29日，於SNH48第四屆總選舉，最終獲得18878.7票，排名第49名，入選第四屆總選舉Future Girls<ref>[http://www.snh48.com/html/allnews/zixun/2017/0730/2456.html SNH48 GROUP第四届总决选落幕 鞠婧祎问鼎年度人气王]</ref><ref>[http://www.snh48.com/html/allnews/zixun/2017/0729/2453.html “我心翱翔”SNH48GROUP第四届总决选最终结果公布 鞠婧祎连霸]</ref>。

2018年2月3日，於「[[SNH48_Group第四屆年度金曲大賞BEST_50|SNH48 Group第四屆年度金曲大賞BEST 50]]」上宣布所屬隊伍改為Team HII<ref>[http://www.snh48.com/html/allnews/zixun/2018/0203/2883.html SNH48 首次大重组各队成员名单公布 分层级打造彰显新格局]</ref>。
