'''洪祖杭'''{{SBS}}{{JP}}（{{bd|1948年||catIdx=}}），原籍[[福建|福建]][[晉江市|晉江市]]龙湖镇溪前村，[[香港|香港]][[商人|商人]]及[[香港體育|體育界]]名人，人稱「福建杭」。他亦是熱心公益的[[慈善家|慈善家]]，個人用於各項公益事業的捐款高達1億多元[[人民幣|人民幣]]<small><ref>{{cite web |language=zh-hk |url=http://big5.chinaqw.cn:89/zgqj/qxsz/200803/20/110691.shtml |title=全國政協常委港胞洪祖杭：回報家鄉是我一片鄉情 |publisher=中國僑網 |date=2008-03-20 |accessdate=2009-10-13 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref></small>。

洪祖杭是[[金保利亞洲|金保利亞洲]]董事和總經理，曾任[[南華體育會|南華體育會]]主席、[[福建體育會|福建體育會]]會長、[[香港中華業餘體育協會|香港中華業餘體育協會]]會長、[[香港足球總會|香港足球總會]]董事會董事、[[香港康體發展局|香港康體發展局]]董事局董事。第九屆、十屆[[中國人民政治協商會議全國委員會香港地區委員|全國政協委員]]、第十一屆[[中國人民政治協商會議全國委員會香港地區委員#政協常委會常委名單|全國政協常委]]<small><ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2008/03/14/CH0803140021.htm |title=金保利亞洲公司董事長洪祖杭：低調厚道 為港為國棄商機 |publisher=[[文匯報|''wenweipo.com'']] |date=2008-03-14 |accessdate=2009-10-13}}</ref></small>。洪祖杭因長期服務社會，表現卓著，尤其致力發展香港體育，並積極推動香港與內地間的體育交流活動，貢獻良多，2009年獲[[香港特別行政區政府|香港特別行政區政府]]頒授[[銀紫荊星章|銀紫荊星章]]<small><ref>{{cite web |language=zh-hk |url=http://www.takungpao.com/news/09/07/28/SD-1118244.htm |title=體舞盟賀洪祖杭受勳 |publisher=[[大公報|''大公網'']] |date=2009-07-28 |accessdate=2009-10-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20090802212432/http://www.takungpao.com/news/09/07/28/SD-1118244.htm |archivedate=2009-08-02 }}</ref></small>。

洪祖杭和他的一對子女洪定騰、洪羽緁是退役馬「世外桃源」、「騰雲駕霧」、「駿方」、「[[火龍駒|火龍駒]]」、「辣龍」、「龍騰虎震」、「[[佳龍駒|佳龍駒]]」、「馬飛龍」、「無敵飛龍」、「飛龍駒」、現役馬「火箭駒」（來港前曾用名「直上雲霄」）、「[[佳騰駒|佳騰駒]]」（來港前曾用名「發奮」）、「[[達龍駒|達龍駒]]」、「旺旺龍」的馬主，亦是[[2008年夏季奧林匹克運動會香港區火炬接力|2008年夏季奧林匹克運動會香港區火炬接力]]的120名火炬手之一<small><ref>{{cite web |language=zh-hk |url=http://www.lcsd.gov.hk/TorchRelay/b5/torchbearer_info.php?id=34 |title=火炬手：洪祖杭 |publisher=[[康文署|''lcsd.gov.hk'']] |date= |accessdate=2009-10-13}}</ref></small>。

==生平==
洪祖杭在1948年出生於[[晉江市|晉江市]]龍湖鎮溪前村，其父親在[[菲律賓|菲律賓]]經營[[木材|木材]]生意，於1958年隨母親和妹妹移居[[香港|香港]]，1960年以試讀生身份入讀[[福建中學|福建中學]]，畢業後原擬轉到[[新法書院|新法書院]]繼續學業，還未入學便到菲律賓協助年老的父親經商<small><ref>{{cite web |language=zh-hans |url=http://www.jinjiang.gov.cn/lhzzfw/hwxy/qlfc/20060603809608.shtml |title=只觉得应该这样做——记香港金丰盛投资有限公司董事长洪祖杭 |publisher=龙湖镇政府网 |date=2006-06-03 |accessdate=2009-10-13}}</ref></small>。回港後，靠在菲律賓認識的朋友介紹轉營建材生意，首創「建材與裝修」結合的服務模式，獲得客戶的讚賞，賺取事業上的第一桶金。其後得朋友引荐經營[[台灣|台灣]]與香港的[[黃金|黃金]]生意，又取得[[萬寶路|萬寶路]][[香煙|香煙]][[亞洲|亞洲地區]]的經銷代理權。洪祖杭然後在菲建賓興建[[酒店|酒店]]，到台灣投資[[電視台|電視台]]，分別進軍[[金融|金融]]、[[旅遊|旅遊]]、[[房地产|房地产]]、[[香港電影|電影]]、[[高科技|高科技]]及[[可替代能源|新能源]]等<small><ref name="商路遍各業">{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2008/05/14/NS0805140001.htm |title=洪祖杭 商路遍及各業 低調貢獻祖國 |publisher=[[文匯報|''wenweipo.com'']] |date=2008-05-14 |accessdate=2009-10-13}}</ref></small>。 

洪祖杭自1980年代末期開始加入[[南華體育會|南華體育會]]，曾擔任會長及主席。從南華會開始，逐漸擴展到其他不同的體育層面，曾任康體局、運動員支援小組委員會、體育發展委員會、香港教訓培訓委員會委員、香港足球總會、香港特區大型體育活動事務委員會、香港學界體育聯合港島及九龍地區中學分會、[[香港影視明星體育協會|香港影視明星體育協會]]等公職。他於1994年向[[国家体育总局|国家体育总局]]捐資5,000萬港元，發起組織「中華全國體育基金會」。2003年投資在[[泉州|泉州]]興建他與國家體委合作打造的[[内地|内地]]房地產[[品牌|品牌]]「奧林匹克花園住宅」，並聘請前國家或地方教練員和運動員前來指導社區居民進行體育健身活動<small><ref name="商路遍各業"/></small>，他亦有入股世界性的體育事業如美國職業籃球（NBA）等（克利夫蘭騎士隊）一成半股份<ref>http://the-sun.on.cc/cnt/china_world/20091216/00423_001.html</ref>。

==家庭狀況==
洪祖杭已婚，兒子洪定騰為[[中國香港單輪車協會|中國香港單輪車協會]]的創會會長，女兒洪羽緁為[[大律師|大律師]]。
