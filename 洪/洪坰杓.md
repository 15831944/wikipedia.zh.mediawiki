'''洪坰杓 '''（{{lang-ko|'''홍경표'''}}，{{bd|1962年|8月11日|||catIdx=}}）是一位[[韓國|韓國]][[攝影師|電影攝影師]]。名字常被譯作洪慶彪。[[韓國|韓國]][[大鐘獎|大鐘獎]]及[[青龍獎|青龍獎]]最佳攝影得主<ref>{{Cite web |title = 홍경표 |url=https://people.search.naver.com/search.naver?where=nexearch&query=%ED%99%8D%EA%B2%BD%ED%91%9C&sm=tab_etc&ie=utf8&key=PeopleService&os=143596 |publisher = 네이버 인물검색|date = |language = ko |}}</ref>。2018年憑[[李滄東|李滄東]]執導的《[[燒失樂園|燒失樂園]]》入圍[[第13屆亞洲電影大獎|第13屆亞洲電影大獎]]最佳攝影<ref>{{Cite web |title = 第十三屆亞洲電影大獎入圍名單 |url=http://www.afa-academy.com/page?mid=143 |publisher = 亞洲電影大獎學院|date =2019-01-11 |language = zh-hk |}}</ref>。他的攝影作品曾三度入圍[[康城影展|康城影展]]<ref>{{Cite web |title = BURNING ENGLISH PRESS KIT |url=https://cdn-media.festival-cannes.com/pdf/0001/62/8392a5c7513577bb54de8298663f293a960e07c2.pdf |publisher = 第71屆康城影展|date = |language = en |}}</ref>。
