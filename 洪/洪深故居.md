'''洪深故居'''位于[[中华人民共和国|中国]][[山东省|山东省]][[青岛市|青岛市]][[市南区|市南区]][[福山路|福山路]]1号，是中国现代著名剧作家[[洪深|洪深]]在1934年到1936年间任职于[[山东大学|山东大学]]时的居所。2005年2月5日被列为[[青岛市文物保护单位|市级重点文物保护单位]]，毗邻福山路3号[[沈从文故居_(青岛)|沈从文故居]]。<ref name="香港文匯報">{{cite web |title=琴台客聚：洪深故居的垃圾桶 |author=黃仲鳴 |publisher=香港文匯報 |url=http://paper.wenweipo.com/2007/09/09/OT0709090024.htm}}</ref>

洪深故居在[[八关山|八关山]]东北麓的路口，是一座依山而建的欧式二层小楼<ref>{{cite web |title=青岛访文化名人故居 |publisher=中国文明网 |url=http://archive.wenming.cn/hsly/2010-06/17/content_20083465.htm}}</ref>。2009年曾对其整修<ref>{{cite web |title=小鱼山周边楼院整治开工 98栋老楼穿新衣|publisher=青岛房地产信息网 |url=http://www.51fdc.com/html/2009-12-23/00030663.htm}}</ref>
，现由私人居住，未向公众开放。<ref>{{cite web |title=青岛名人故居状况堪忧 沈从文故居成菜园子 |publisher=青岛新闻网 |url=http://www.qingdaonews.com/gb/content/2010-10/23/content_8535902.htm}}</ref>
