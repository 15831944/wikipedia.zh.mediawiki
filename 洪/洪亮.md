'''洪亮'''（{{bd|1985年|3月5日|||HL}}），中国当代青年[[导演|导演]]，同时兼任编剧、摄影<ref>{{cite news|title=洪亮：中国当代青年导演|url=http://hunan.sina.com.cn/hengyang/ssbh/2014-10-27/173015901.html|accessdate=2014-10-27|newspaper=新浪湖南|date=2015-02-11}}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}。</ref>。

== 人物介绍 ==

洪亮，高中毕业之后便赴澳洲深造，就读于[[澳大利亚邦德大学|澳大利亚邦德大学]]影视专业。回国之后开始发展电影事业，跟随中国海派导演王国平，担任副导演，协助拍摄2010[[世博会|世博会]]主题片《2010等你来》。<ref>[http://news.163.com/10/0406/08/63IRFOUT000146BB.html 《2010等你来》用音符书写的真诚请柬]。</ref>

== 生平经历 ==

2010年，在[[澳大利亚|澳大利亚]]留学时成立微谷影视工作室；

2011年3月回国，跟随中国海派导演[[王国平|王国平]]，协助拍摄微电影；

2012年3月成立上海星微文化传播有限公司；

== 拍摄作品 ==


2007年，拍摄《Tears》澳洲MV；2008年，拍摄《空号》澳洲悬疑短剧；2010年，拍摄微电影《Rain Dance》<ref>{{cite news|title= 爱奇艺《Rain Dance》|url=http://www.iqiyi.com/v_19rrhs1wlo.html|date=2015-02-11}}</ref>获得澳大利亚邦德大学学院奖；2010年，为澳洲神仙湾拍摄形象片《Sanctuary Cove》[http://v.youku.com/v_show/id_XNzYyOTY3MjAw.html]； 2010年，拍摄微电影《杰特的吉他梦》、[[《血玉手镯》|《血玉手镯》]]；2012年，与腾讯视频网合作拍摄都市网络喜剧《小雷变奏曲》<ref>[http://movie.douban.com/subject/24743752/ 小雷变奏曲]。</ref>；2012年，拍摄微电影《极速逃亡》；2012年12月，星微传媒与天猫商城百芙润品牌联袂出品了“双十一黄金周”广告片[[天猫style|天猫style]]<ref>{{cite news|title= 爱奇艺《天猫style》|url=http://www.iqiyi.com/v_19rrhgpv0o.html|date=2015-02-11}}</ref>； 2013年，拍摄《奔驰 乌尼莫克》；2013年，根据自己的真实故事改编拍摄微电影《隐藏的讯息》<ref>[http://www.360kan.com/m/fqjpZUH8QXb5Sh.html 《隐藏的讯息》]。</ref>；2015年,执导《星梦少女》SNH48网络剧。

== 澳洲经历 ==

洪亮在澳洲留学相遇马辰（微电影导演）和张耘海（微电影导演）<ref>[http://v.17173.com/v_102_610/OTAwNTk0OQ.html 指导微电影《心之旅》] {{webarchive|url=https://web.archive.org/web/20150211095157/http://v.17173.com/v_102_610/OTAwNTk0OQ.html |date=2015-02-11 }}。</ref>，共同成立微谷影视工作室，合作拍摄[[《血玉手镯》|《血玉手镯》]]<ref>[http://www.iqiyi.com/v_19rrmmhxmg.html 《血玉手镯》爱奇艺]。</ref>。《Jett&Dash》和《Rain Dance》获得2010年亚太电影节短片单元最佳。
