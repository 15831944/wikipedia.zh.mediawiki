'''洪江市'''，原为'''黔阳县'''，位于中華人民共和國[[湖南省|湖南省]]西部，[[沅水|沅水]]的上游，是湘西南与[[贵州|贵州]]交界地区重要的物资集散中心和工业重镇。总面积2279平方公里，人口50万人，市政府驻[[黔城镇|黔城镇]]。

== 名称 ==
洪江市在1997年前称为[[黔阳县|黔阳县]]，后与[[洪江管理区|原洪江市]]合并为新洪江市。由于1999年的“洪江事件”，导致新洪江市分裂为洪江市和洪江管理区，所以目前[[怀化市|怀化市]]存在着两个都叫做“洪江”的[[县级行政区|县级行政区]]。为区分两地，当地人一般习惯依政府所在地黔城镇将洪江市称为“黔城”或依其原名称为“黔阳”。<ref>{{Cite web|url = http://www.bjnews.com.cn/inside/2015/08/12/374009.html|title = 一个“县级行政区”的“黑户”16年|accessdate = 2015-08-12|author = |date = |publisher = }}</ref>

==历史==
[[秦代|秦代]]这里属黔中郡地。[[汉|汉]]置镡城县，[[南朝梁|南朝梁]]置龙剽县，[[唐|唐]]改名龙标县。[[北宋|北宋]]易名为黔阳县，以地处[[贵州|贵州]]（黔）之东（阳）而得名。1949年黔阳县城由[[黔城镇|黔城镇]]迁至[[安江镇|安江镇]]，隶属于[[会同专区|会同专区]]。1950年，析[[会同县|会同县]]洪江镇置县级洪江市，亦属会同地区管辖。1952年撤销会同专区，成立[[芷江专区|芷江专区]]，同年底芷江专区更名为[[黔阳专区|黔阳专区]]，行署迁至黔阳县城安江镇大畲坪。1978年黔阳地区行署迁到了[[怀化县|怀化县]]，黔阳地区也于1981年更名为[[怀化地区|怀化地区]]。

1997年，[[中华人民共和国国务院|中华人民共和国国务院]]批准将黔阳县与原洪江市合并，成立新的洪江市，并由地级怀化市代管。当时的老洪江市，下辖四个街道和三个乡，面积216平方千米，人口9万人。黔阳县下辖七个镇十八个乡，面积2161平方千米，人口40万人。新洪江市政府设在黔城镇，该镇由原属黔阳县的老黔城镇和原属老洪江市的黔城乡合并而来。

=== 洪黔分治 ===
{{POV-section}}

在县市合并后，老洪江市的民众认为在人事安排、财政调拨以及奖金分配等一系列攸关自己切身利益的问题上受到了新洪江市也就是老黔阳县的歧视性对待，从而逐渐蕴积了反感。尤其是在合并的次年即1998年，新洪江市政府就决定将老洪江市的四街三乡合并为一个[[雄溪镇|雄溪镇]]，此举引发老洪江民众的强烈不满，在1999年初雄溪镇挂牌的当日，终于爆发了暴力抗争，震惊全国，被称为“[[洪江事件|洪江事件]]”。在此后的几个月内，由于民众的抗争不断，使雄溪镇政府始终无法成立。该事件最终由湖南省政府出面，以大幅度撤换新洪江市的官员，并成立直属于怀化市的[[洪江管理区|洪江管理区]]而得以了结。作为[[市辖区|市辖区]]建制的“洪江管理区”至今未获[[中华人民共和国国务院|国务院]]批准。

[[洪江管理区|洪江管理区]]即老洪江市自[[明|明]]代以来就是湘西南地区的交通枢纽和重要商埠。当地的贸易和手工业十分发达，盛产[[陶瓷|陶瓷]]，有“小[[南京|南京]]”之称。整座城市目前依然保留有相当的古代商业城市风貌，各类明清时期的建筑遍布全城，历史价值颇高，城西嵩云山上还有明代寺庙大兴禅寺等多处名胜。新洪江市区黔城镇位于[[焦柳铁路|焦柳铁路]]线上，[[沅水|沅水]]和[[㵲水|㵲水]]的交汇之处，1949年以前一直是黔阳县治。黔城历史悠久，有[[芙蓉楼|芙蓉楼]]、南正街、文峰塔等许多古迹和明清建筑群，是湖南省的省级历史文化名城。原黔阳县城[[安江镇|安江镇]]也是一处重要商埠，盛产柑橘，曾长期为地区行署驻地，1961年至1962年间还曾设市，目前以塑料和织染工业而闻名。

==地理==
洪江地处湘西的山地丘陵地带，中部为[[沅水|沅水]]河谷盆地，东西两侧为[[雪峰山|雪峰山]]的两支山脉所夹，东部为主脉，山势高峻，西部为支脉，地势略平缓，以丘陵为主。平均海拔1150米，最高峰[[苏宝顶|苏宝顶]]1934米。雪峰山主峰之一帽子山山顶拥有湖南省最大的电视发射塔；山顶四周因海拔较高，常年刮风，树木生长较少，形成了一个宽阔的天然草场，即平山堂高原草场。

全境年平均气温17℃，降水量1494毫米。

境内溪河纵横，主要河流有[[沅江|沅江]]。沅江纵贯全境，渠水、[[㵲水|㵲水]]分别于托口、黔城注入沅江。

==行政区划==
洪江市辖4个街道、8个镇、18个乡、2个民族乡（包括洪江区所辖4个街道、3个乡）<ref>[http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2013/43/12/431281.html 中华人民共和国国家统计局]</ref>，实际辖8个镇、15个乡、2个民族乡<ref>[http://www.hjs.gov.cn/info/iList.jsp?cat_id=10249 洪江市人民政府-行政区划]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 交通 ==
距怀化城区35公里，距[[芷江机场|芷江机场]]40余公里，[[枝柳铁路|枝柳铁路]]、[[320国道|320国道]]、[[209国道|209国道]]、[[沪昆高速|沪昆高速]]、[[包茂高速|包茂高速]]公路穿境而过。[[沅江|沅江]]可通航。

== 经济 ==
2007年，全市GDP完成31.28亿元，增长12.9%。其中第一、第二、第三产业增加值分别完成9.60亿元、7.35亿元、14.33亿元。2008年全市生产总值（GDP）为38.85亿元，增长13.1%。农业主产[[水稻|水稻]]、[[小麦|小麦]]、[[玉米|玉米]]、[[油菜|油菜]]、[[花生|花生]]等。当地产的[[柑橘|柑橘]]品种较多，质量上佳，极負盛名。林业发达，盛产[[松|松]]、[[杉|杉]]、[[楠木|楠木]]等各种木材和[[油茶|油茶]]、油桐。特产还有香[[梨|梨]]、乌骨[[鸡|鸡]]、[[天麻|天麻]]、[[竹|竹]]编等。工业则有[[塑料|塑料]]、[[化工|化工]]、[[陶瓷|陶瓷]]、[[纺织|纺织]]、[[印染|印染]]、[[医药|医药]]等门类。

== 文化教育 ==
洪江市现有普通中小学219所，其中完全中学3所，初级中学12所，九年一贯制学校14所，城镇完全小学和乡镇中心小学15所，村小175所；有教师进修学校1所，公办职业中专1所，民办学校24所。

2003年4月中国[[杂交水稻|杂交水稻]]发源地安江农校正式成为高职院校——[[怀化职业技术学院|怀化职业技术学院]]安江校区，是教育部备案成立的公办全日制高等院校。

== 风景名胜 ==
[[雪峰山|雪峰山]]：雪峰山生态避暑旅游区被联合国誉为“神奇绿洲”之称的，是一块没有受到任何工业污染的生态宝地，森林覆盖率达83.4%。[[雪峰山森林公园|雪峰山森林公园]]是湘西有名的避暑胜地。雪峰山自然保护区处于雪峰主峰苏宝顶，植物呈垂直分布，山间有李自成墓、闯王庙、闯王屯兵城堡遗址。[[岩鹰洞|岩鹰洞]]风景名胜区地处雪峰山大峡谷和公溪河河谷，景区内悬崖峭壁、流泉飞瀑随处可见，万亩竹海，青翠欲滴，龙口江竹海漂流，惊险刺激，穿越隧洞，妙趣横生。

雪峰山脚下有一美丽古镇——[[雪峰镇|雪峰镇]]，这里远离都市，民风淳朴，传承着华夏最古老的文明。这里的村民住宅是典型的古代汉民族中正式住宅，中间是中堂，两边是厢房，后边是厨房以及配房，有些起在平地，有些傍山而建，也就自然成就了汉式的[[吊脚楼|吊脚楼]]，虽然没有苗式吊脚楼那种小桥流水的韵味，也不乏古道西风瘦马那种意境。这里的村民讲的是一种古老的汉语言，有些词语只能能在古籍方能查阅到，他们仍然使用原始的曲辕犁进行耕作，使用风车进行灌溉。

黔城镇历为古代军事城堡，明清古建筑风格统一，公建、民居、商业、县衙、宗教等功能齐全，古韵犹存。古镇现有省、县、镇三级重点文物保护单位28 处，其中[[芙蓉楼|芙蓉楼]]相传为诗人[[王昌龄|王昌龄]]送[[辛渐|辛渐]]处，旧有“楚南上游第一胜迹”之称。

== 历代名人 ==
世界著名的“杂交水稻之父”[[袁隆平|袁隆平]]教授曾在安江农校任教，并在这里培育出了[[杂交水稻|杂交水稻]]。
