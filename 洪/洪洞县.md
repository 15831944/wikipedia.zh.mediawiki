'''洪{{ruby-py|洞|-{zh-cn:tóng;zh-tw:ㄊㄨㄥˊ;zh-hk:tung<sup>4</sup>}-}}县'''地处[[中华人民共和国|中华人民共和国]][[山西省|山西省]]南部，为[[臨汾市|臨汾市]]辖县，[[汾河|汾河]]自北向南贯穿全境，为晋南最具人口規模的縣份，人口约有80万，縣治位於[[大槐樹鎮|大槐樹鎮]]。有[[大槐树寻根祭祖园|大槐树寻根祭祖园]]、[[苏三|苏三监狱]]、[[广胜寺|广胜寺]]等名胜古迹。

== 歷史 ==
{{Main|洪洞歷史}}

洪洞縣原稱洪趙縣，為1954年合併洪洞縣與趙城縣而來，1958年將新合併的縣又改名為洪洞縣。

洪洞古稱'''楊'''，春秋末期，[[晉國|晉國]]羊舌氏被滅，其采邑被分為楊氏、平陽、銅蚒三縣，是洪洞縣境內最早的置縣。西漢在此設置[[楊縣|楊縣]]。洪洞這個名稱起於[[北朝|北朝]][[東魏|東魏]]，因戰爭需要，於楊縣興建屯兵的城堡──洪洞戎。617年，李淵將楊縣改為洪洞縣。

原[[趙城縣|趙城縣]]，趙城為西周[[造父|造父]]的封地。漢至隋，趙城縣境屬[[彘縣|彘縣]]、[[永安縣|永安縣]]、[[霍邑縣|霍邑縣]]，隋朝始分出趙城縣，直到1954年與洪洞縣合併。<ref>[http://www.yiyou.com:1980/b5/linfen.yiyou.com/html/5/143.html 洪洞縣——洪濤洞駭宣軍威]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 地理 ==
洪洞縣地處[[臨汾盆地|臨汾盆地]]北端，西邊為[[霍岳|霍岳]]，[[汾河|汾河]]自北向南由縣境中央貫穿，面積1494平方公里。<ref name="洪洞黨建">[http://www.htdj.gov.cn/gaikuang.asp 洪洞黨建網站，洪洞縣基本情況簡介] {{webarchive|url=https://web.archive.org/web/20071020145456/http://www.htdj.gov.cn/gaikuang.asp |date=2007-10-20 }}</ref>為山西省南北往來的交通要地<ref>[http://www.tynews.com.cn/big5/site1/whgc/2007-03/30/content_3187735.htm 問我祖先在何處 山西洪洞大槐樹，太原日報] {{webarchive|url=https://web.archive.org/web/20070928023234/http://www.tynews.com.cn/big5/site1/whgc/2007-03/30/content_3187735.htm |date=2007-09-28 }}</ref>。

== 交通 ==
南北向的[[南同蒲鐵路|南同蒲鐵路]]平行於汾水東側貫穿洪洞縣，設有趙城站、洪洞站、甘亭站。其中[[洪洞站|洪洞站]]為三等站<ref>太原鐵路局，車站簡介，{{cite web |url=http://www.tytlj.com.cn/TTCS/YSC/hmcwd-htz.htm |title=存档副本 |accessdate=2007-06-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20070926233918/http://www.tytlj.com.cn/TTCS/YSC/hmcwd-htz.htm |archivedate=2007-09-26 }}</ref>，每日有十班中長程列車停靠，另有焦化廠鐵路專線由趙城鎮分出經明姜鎮至廣勝寺鎮焦化廠。南北向公路運輸有[[大運高速公路|大運高速公路]]、[[108國道|108國道]]，東西向公路則有[[309國道|309國道]]、洪古路、趙克公路<ref name="洪洞黨建"/>。新建的大同至西安客运专线从北到南横穿洪洞县，并设有[[洪洞西站|洪洞西站]]。

== 黑砖窑事件 ==
非法砖窑矿主从人贩中购买非法運来的民工，包括许多童工，民工的主要来自[[河南省|河南省]]，将他们关押在各个窑场，强迫他们从事极高强度的体力劳动，动辄暴力殴打，使得為數不少的工人受到[[職業傷害|職業傷害]]、身残、死亡，这些民工的遭遇跟[[奴隶|奴隶]]类似。案件被媒体曝光后，引起舆论的极大公愤，也得到了中国中央政府的重视，中共中央总书记[[胡锦涛|胡锦涛]]、国务院总理[[温家宝|温家宝]]等作出批示，地方政府进而采取检查、营救等措施。

==“12·5”矿难== 
2007年12月5日23时15分许，洪洞县左木乡红光村新窑煤矿发生矿难。初步判断起因是煤尘爆炸，起火原因不明。事故发生时井下有111人，至12月7日中午，15人获救，105人遇难。为2006年以来中国发生一次死亡人数最多的特大事故<ref>觸目驚心 令人髮指——山西洪洞"12·5"礦難追蹤，新華網，http://news.xinhuanet.com/newscenter/2007-12/10。/content_7225670.htm{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
