'''洪旭'''{{BD|1605年||1670年|}}，原名忠珠；後改名諱旭、[[表字|字]]'''弘藎'''、[[號|號]]'''念衷(另號九峰)'''，[[明朝|明朝]]末年的[[武將|武將]]，[[明鄭時期|明鄭時期]]官至特進光祿大夫左柱國[[少傅|少傅]]兼[[太子太師忠振伯|太子太師忠振伯]][[兵部尚書|兵官]]，于永曆九年四月誥授(晉)封'''忠振侯'''。<ref>http://www.guoxue123.cn/tw/02/069/003.htm 国学导航－鄭氏關係文書 </ref>

== 生平 ==
《臺灣省通志》（1972年出版）記載：

'''洪旭'''，字念藎，號九峰，[[福建|福建]][[同安|同安]]人。<ref>金門在1915年設[[縣|縣]]以前大多數時期皆隷屬於[[福建省|福建省]][[同安縣|同安縣]]，因此金門出生之歷史人物多半被記載為同安人。</ref>初為[[鄭芝龍|鄭芝龍]]部將，[[隆武帝|隆武帝]]封忠振伯。芝龍降清，轉依[[鄭成功|鄭成功]]，治軍嚴整，成功甚重之。1655年2月鄭成功承制設六官，以洪旭為戶官，旋改任水師右軍、水師總督，會中提督[[甘輝|甘輝]]取[[舟山|舟山]]。洪旭應募兵旅時，常與[[陳霸|陳霸]]為伍，之後[[陳霸|陳霸]]先隨[[鄭成功|鄭成功]]和[[陳永華|陳永華]]等人渡台，洪旭後期而至匯合於大員。陳霸是渡台後於安平之[[陳姓|陳姓]]宗祖之一。1658年成功北伐，以旭兼理兵官，同[[黃廷|黃廷]]、[[鄭泰|鄭泰]]，留守[[金門島|金]]、[[廈門島|廈]]二島。1660年[[清軍|清軍]]犯[[廈門|廈門]]，成功以洪旭駐[[鎮海|鎮海]]為犄角，大破之。1661年成功取[[臺灣|臺灣]]，命洪旭、黃廷，佐[[世子|世子]][[鄭經|經]][[留守|留守]][[思明|思明]]。1662年鄭經與庶弟之奶媽不倫戀，鄭成功命洪旭往思明斬經。旭以鄭泰為成功兄，兄可以拒弟，遂示意鄭泰拒之，鄭經因得[[保全|保全]]。5月成功逝，弟[[鄭世襲|世襲]]於臺灣自立。訃至，洪旭請鄭經先嗣位，後發喪；復請速率兵渡臺靖內難。時清軍陰謀進犯，時局岌岌可危。洪旭請以『陽和陰違』緩清軍攻臺，卒使臺局定，內難安。1664年鄭經棄金、廈，經[[澎湖|澎湖]]。洪旭於澎湖立營壘、煙墩、砲臺，戍兵守之。1666年旭復請鄭經令各鎮營於農隙教習武藝[[弓矢|弓矢]]，[[春|春]][[秋|秋]]操演陣法；檄南北路屯兵入深山窮谷採辦桅舵含檀，修造船隻；又遣商船，往各地購買船料，興造洋艘[[鳥船|鳥船]]，以[[白糖|白糖]]、[[鹿皮|鹿皮]]等物，上通[[日本|日本]]，製造銅炮、倭刀、盔甲，並鑄[[永曆|永曆]]錢，下販[[暹羅|暹羅]]、[[交趾|交趾]]各處以富國，致田疇市肆不讓內地。8月，旭以憂勞染寒疾卒。

== 參考資料 ==
* 洪福為洪旭之祖父，字君錫，號連江，因孫洪旭而貴，誥贈[[光祿大夫|光祿大夫]]左[[柱國|柱國]][[太子太師|太子太師]]忠振伯，墓在[[南澳縣|南澳]]東門外，洪旭之祖母曾氏，[[誥命|誥贈]]一品夫人。

* 洪福之長子[[洪渭文|洪渭文]]，為洪旭之伯父，洪旭年幼喪父，幸賴伯父撫育成人，洪旭視之如父。

* 洪旭之父為[[洪公掄|洪公掄]]（字廷揚，號鼎銘），聚妻黃氏，側室劉氏，計生三子，公掄以功授守備，[[萬曆|萬曆]]四十二年（西元1614年）征[[海南島|海南島]]黎族之役壯烈犧牲，劉氏年方二十八，與嫡黃氏撫育三子（孟旭字弘藎、仲曦字弘扆、季暄字弘韻）成立，商販巨富，《金門志》為劉氏立傳，列為節孝人物。

* 洪旭原名忠珠，後來改名旭，字弘藎，號念衷，[[後豐港|後豐港]]人，明抗清時，洪旭以軍功得官，為鄭成功所重用，累官至中軍提督，授戶官、兵官等職，並晉封忠振侯加太子太師。清兵攻佔金門、廈門二島時，諸將若降清者，皆獲顯秩高官，惟洪旭率兵艦，擁鄭經東歸，經營台澎、屢獻良策，洪旭事成功父子，盡效忠悃，始終不二，為[[明鄭|明鄭]]股肱之臣。

* [[永曆|永曆]]二十四年（西元1670年），洪旭憂勞病故，鄭經痛悼不已，親自為其治喪，並授其子[[洪磊|洪磊]]為吏官，兼管戶部事務。

《臺灣省通志》記載者與上述五點相異之處，除洪旭之字、號略有不同外，最大的差異在洪旭的卒年。

另外，有一說洪旭是[[同安|同安]]馬巷蔡浦人，村中有一座古屋，相傳是洪旭的故居，村中和村東還有兩座沒有任何標誌的古墓，村民年年清明掃墓，據傳是洪公掄和洪旭父子的墓葬，此有待考證。

據鷺江出版社出版的《廈門人物辭典》和金門《後豐洪氏族譜》記載，洪旭的父親洪公掄（？—1614年），字廷揚，號鼎銘，洪君錫次子，[[明代|明代]]同安縣後豐港人，後豐洪氏十四世。故洪旭為後豐港人應正確無誤。

== 故居 ==
後豐港洪氏宗祠左側的古厝，為洪旭的故居，現已由[[金門縣政府|金門縣政府]]指定為古蹟建築，名稱定為「[http://www.hach.gov.tw/hach/frontsite/cultureassets/caseBasicInfoAction.do?method=doViewCaseBasicInfo&caseId=WA09602000544&version=1&assetsClassifyId=1.2&siteId=101 洪旭古厝]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}」。

== 來台之後的後世居住地 ==

洪旭對[[鄭氏王朝|鄭氏王朝]]有很大的貢獻，雖然[[鄭成功|鄭成功]]已在封賞時有賞賜洪旭一塊土地，但那時候該土地尚未被發墾。直到旭子洪磊於[[鄭經|鄭經]]時期率兵丁外出巡視方知有塊封地，於是號召族人前往開墾。該土地據考證，大約在現在[[臺南市|臺南市]]的[[青鯤鯓|青鯤鯓]]、[[北門區|北門區]]一帶。入[[清|清]]以後，居住在該地的洪姓家族聚落越來越多，逐漸擴展到今[[學甲區|學甲區]]、[[營頂|營頂]]、[[七股區|七股區]]，甚至有部分的族人遷入[[府城|府城]]（今臺南市市區）居住，但仍以[[北門區|北門區]]、[[七股區|七股區]]等處為最多。所以在台洪姓後代於討論祖源時，都自稱來自[[北門區|北門區]]或[[七股區|七股區]]。

== 軼聞 ==
*相傳今日金門、廈門流傳的[[中秋博餅|中秋博餅]]習俗，為洪旭所創。
*現今[[浯江書院|浯江書院]]內的[[金門朱子祠|金門朱子祠]]，當中有塊匾額提字為'''伯爵軍門'''，講的就是洪旭。
