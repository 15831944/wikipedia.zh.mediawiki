'''洪都保衛戰'''，又稱'''洪都之戰'''，是元朝末年[[陳友諒|陳友諒]]和[[朱元璋|朱元璋]]在洪都（今[[江西省|江西省]][[南昌市|南昌市]]）進行的一次重要戰役。是中國古代軍事史上著名的守城戰役，亦以以少勝多而聞名。
== 過程 ==
元[[至正|至正]]二十三年（1363年）4月，[[陳友諒|陳友諒]]親率六十萬大軍圍攻[[洪都|洪都]]城，<ref>明史列傳第十四鄧愈</ref>朱元璋則任命他的侄子[[朱文正|朱文正]]為洪都大都督，負責洪都軍務，兵力約有四萬（一說兩萬）。戰爭打響前，守城方對洪都城池進行了大規模加固，亦對城門格局進行了調整，並將臨江側城牆撤至距江三十步，這一奇招致使陳友諒的大戰船無法直接攻擊到城池。攻城開始後，陳友諒親自督戰，不分晝夜地對洪都城發動猛烈攻擊，城牆多處被破壞，守軍苦戰，邊戰邊修補城牆。 <ref>明史列傳第二十一趙德勝</ref>最終，憑藉著都督[[朱文正|朱文正]]卓越的軍事指揮，靈活的策略運用，同時也藉由[[鄧愈|鄧愈]]、[[趙德勝|趙德勝]]等名將的輔佐，洪都守軍頑強地抵禦住了陳友諒的進攻。後朱元璋率領援軍趕來，陳友諒撤退，共持續了八十五天的洪都保衛戰宣告結束。

== 影響 ==
*成就了朱文正的不世功勳
*沉痛打擊陳友諒，奠定了元末的政治格局
*在中國軍事史上，該場戰役以守城特色和以少勝多聞名
== 後續 ==
撤兵的陳友諒，與開赴洪都救援的朱元璋在[[鄱陽湖|鄱陽湖]]遭遇，發生了[[鄱陽湖之戰|鄱陽湖之戰]]。
== 逸聞 ==
[[陳友諒|陳友諒]]在[[贛江|贛江]]枯水期到來之際，未能聽從屬下建議及時撤軍。致使枯水期到來，[[贛江|贛江]]和[[鄱陽湖|鄱陽湖]]水位低落，陳友諒大批戰船困在贛江河道，無法駛入鄱陽湖迎戰朱元璋。
