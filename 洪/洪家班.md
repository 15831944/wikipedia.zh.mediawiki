'''洪家班'''，是以[[香港|香港]]著名[[导演|导演]]、[[演员|演员]][[洪金宝|洪金宝]]為核心的电影演员组织的名称，成立於1970年代，同时期组织还有[[成家班|成家班]]。<ref name="lion " >{{cite web |url= http://yule.sohu.com/20080408/n256163641.shtml|title=八大功夫之洪家班——霸王猛将 |accessdate=2008年4月8日 |publisher= 搜狐网|work= |language =zh-cn}}</ref>

== 歷史 ==
洪家班成立於1970年代，后和成家班為香港电影两大开列国际。<ref>{{cite web |url=http://ent.163.com/10/0523/04/67BFUJ3C00032DGD.html |title=成家班vs洪家班 |accessdate=2010-05-23 |publisher= 网易|work= |language =zh-cn}}</ref>
