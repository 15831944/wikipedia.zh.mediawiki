'''洪治纲'''（{{bd|1965年|10月||}}），[[安徽省|安徽]][[东至县|东至]]人，现任[[杭州师范大学|杭州师范大学]]人文学院院长，中国小说学会副会长。2007年，凭借《困顿中的挣扎——贾平凹论》荣获[[第四届鲁迅文学奖|第四届鲁迅文学奖]]文学理论评论奖。<ref>{{cite web |url = http://jd.jnu.edu.cn/yxsz/yxsz2014/detail.asp?id=24009 |title = 洪治纲 |publisher = 暨南大学}}</ref><ref>{{cite web |url = http://rwxy.hznu.edu.cn/c/2014-10-20/585487.shtml |title = 中国现当代文学教研室：洪治纲 教授 |publisher = 杭师大人文学院}}</ref>

== 生平 ==
1988年，毕业于[[安徽师范大学|安徽师范大学]]中文系获文学学士学位。1991年，获[[浙江师范大学|浙江师范大学]]文学硕士学位。1991年至1995年，在[[杭州学军中学|杭州学军中学]]任教。1996年至2005年，供职于浙江省作家协会文学院。2005年，获[[浙江大学|浙江大学]]文学博士学位。2006年至2012年，在[[暨南大学|暨南大学]]中文系任教。2012年至今，任杭州师范大学人文学院教授、院长。

* 《零度疼痛》（2003年）
* 《清平乐》（2003年）
* 《无边的迁徙》（2004年）
* 《余华评传》（2005年）
* 《主体性的弥散》（2009年）
* 《心灵的见证》（2009年）
* 《邀约与重构》（2012年）

== 所获奖誉 ==
* 2000年 首届“冯牧文学奖·青年批评家奖”
* 2007年 第四届鲁迅文学奖
* 2008年 入选教育部“新世纪优秀人才支持计划”
* 2008年 首届“当代中国文学批评家奖”
* 2015年 高等学校科学研究优秀成果奖（人文社会科学）二等奖
