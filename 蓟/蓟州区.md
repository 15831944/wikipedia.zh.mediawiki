'''蓟州区'''原称'''无终县'''、'''渔阳县'''、'''蓟州'''和'''蓟县'''，是[[中华人民共和国|中国]][[天津市|天津市]]下辖的一个[[市辖区|市辖区]]，位于[[天津市|天津市]]北部。蓟州区面积1593平方千米，人口81万。<ref>[http://www.tjjx.gov.cn/ 蓟州政务网] {{webarchive|url=https://web.archive.org/web/20070928034849/http://www.tjjx.gov.cn/ |date=2007-09-28 }}</ref>

== 历史 ==
蓟州区最早称'''无终县'''，[[隋朝|隋朝]]时成为[[渔阳郡|渔阳郡]]的郡治，[[唐朝|唐朝]]因郡改名为“渔阳县”，改渔阳郡为蓟州，这里仍为治所。[[元朝|元朝]]以后，[[渔阳|渔阳]]之名棄用，这里成为[[蓟州|蓟州]]。1913年，从蓟州改名为“蓟县”，为[[河北省|河北省]]辖县，1973年9月划归[[天津市|天津市]]。

2008年，蓟县县城城关镇恢复[[渔阳镇|渔阳镇]]历史名称。

2016年6月，国务院批复《关于撤销蓟县成立蓟州区的请示》（津政报〔2016〕9号），同意撤销蓟县，成立蓟州区<ref>国务院关于同意天津市调整部分行政区划的批复（国函〔2016〕98号），2016-06-21</ref>。

== 物产 ==
蓟州区作为[[天津市|天津市]]唯一的山区，以盛产果木著称，蓟州区的特产有：盘山柿子、下营的[[板栗|板栗]]和邦均的[[核桃|核桃]]。

== 行政区划 ==
蓟州区现下辖1个街道、20个镇、5个乡、1民族乡：
*街道办事处：文昌街道。
*镇：渔阳镇（原为城关镇，2008年改为现名<ref>[http://61.50.229.219/ConInfoParticular.jsp?id=142 关于蓟县城关镇人民政府更名为蓟县渔阳镇人民政府的通知]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}，蓟政发〔2008〕13号</ref>）、下营镇、罗庄子镇、马伸桥镇、出头岭镇、西龙虎峪镇、[[五百户镇|五百户镇]]、官庄镇、白涧镇、邦均镇、洇溜镇、别山镇、桑梓镇、尤古庄镇、侯家营镇、施古镇、上仓镇、下窝头镇、杨津庄镇、下仓镇。
*乡：穿芳峪乡、许家台乡、礼明庄乡、东赵各庄乡、东二营乡。
*民族乡：孙各庄满族乡。
