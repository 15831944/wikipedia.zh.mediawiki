'''唐德剛'''（{{lang-en|Te-Kong Tong}}，{{bd|1920年|8月23日|2009年|10月26日|catIdx=Tang唐德剛}}），生於[[中華民國|中華民國]][[安徽省|安徽省]][[合肥|合肥]]，[[美籍華人|美籍華人]][[學者|學者]]，[[歷史學家|歷史學家]]、[[傳記|傳記]][[文學家|文學家]]、[[紅學|紅學]]家。唐德剛在口述歷史方面有重大貢獻，著有《[[李宗仁回憶錄|李宗仁回憶錄]]》、《[[胡適|胡適]]口述自傳》、《[[顧維鈞|顧維鈞]]回憶錄》等。另外還有一大貢獻是關於中國近代演變等“歷史三峽”説。

== 生平 ==
1920年，'''唐德剛'''生於[[安徽省|安徽省]]合肥縣西鄉山南館唐老圩，德剛幼時在私塾唸書，舊學邃密，十多歲即已圈點過一遍《[[資治通鑑|資治通鑑]]》<ref name = "netease">{{cite web| url = http://book.163.com/special/00923SJV/tangdegangqushi.html| title = 華人歷史學家唐德剛去世| publisher = [[網易|網易讀書]]| accessdate = 2010-05-27}}</ref>。

1936年，于安徽省[[舒城中学|舒城中学]]初中毕业<ref>{{Cite web|url = http://www.ahsczx.com/main/model/childcatalog/ChildCatalog.do?subfcode=0010804|title = 百年舒中-校友之窗|accessdate = 2014-10-07|deadurl = yes|archiveurl = https://web.archive.org/web/20141009205100/http://www.ahsczx.com/main/model/childcatalog/ChildCatalog.do?subfcode=0010804|archivedate = 2014-10-09}}</ref>。

1939年秋考入[[国立中央大学|国立中央大学]]歷史學系，和[[黃彰健|黃彰健]]等人同學；此一時期的中大歷史系有「[[沙坪壩|沙坪壩]]的黃金時代」之說，[[柳詒徵|柳詒徵]]、[[朱希祖|朱希祖]]、[[繆鳳林|繆鳳林]]、[[郭廷以|郭廷以]]、[[向達|向達]]、[[沈剛伯|沈剛伯]]、[[賀昌群|賀昌群]]、[[白壽彝|白壽彝]]、[[韓儒林|韓儒林]]等史學家皆執教於此；1943年畢業，獲學士學位。1944年在[[安徽學院|安徽學院]]史地系講授《西洋通史》。

1948年赴美留學，1952年獲[[哥倫比亞大學|哥倫比亞大學]]碩士，1959年獲史學博士，博士論文為一八四四年至一八六零年間的中美關係。后留校任教，曾講授《漢學概論》、《中國史》、《亞洲史》、《西洋文化史》等課程<ref name = "netease" />，並兼任哥倫比亞大學中文圖書館館長7年並參與負責口述歷史計劃的中國部分。

1972年，唐德剛受到排擠，而離開哥倫比亞大學<ref name = "ifeng">{{cite web| url = http://finance.ifeng.com/opinion/xzsuibi/20091106/1438004.shtml| title =书祭唐德刚| author = 燕舞| publisher = [[鳳凰網|鳳凰網]]| date = 2009-11-06| accessdate = 2010-05-27}}</ref>，后受聘為[[紐約市立大學|紐約市立大學]]教授，執教於紐約市立大學系統下的市立學院近二十年，直至1991年榮休，其間曾任亞洲研究系系主任<ref name = "yzzk">{{cite web| url = http://www.yzzk.com/cfm/Content_Archive.cfm?Channel=aj&Path=2208184152/45aj1a.cfm| title = 口述史大師唐德剛青史留名| author = 林博文| publisher = [[亞洲周刊|亞洲周刊]]| date = 2009-11-15| accessdate = 2010-05-27}}</ref>。曾任[[紐約|紐約]]文藝協會會長。

2009年10月26日，因[[腎功能衰竭|肾衰竭]]卒于美国[[旧金山|旧金山]]佛利蒙家中，享年89岁<ref>{{cite web
| url = http://www.chinareviewnews.com/doc/1011/1/9/1/101119190.html?coluid=0&kindid=0&docid=101119190
| title = 唐德剛去世前夕全部藏書捐給安徽大學
| publisher = [[中國評論新聞網|中國評論新聞網]]
| date = 2009-10-30
| accessdate = 2010-05-27}}</ref>。

唐德剛與師[[郭廷以|郭廷以]]，都是中國近代史的大家，也是華裔史學家中口述史的主要推動人物。

他受[[柳詒徵|柳詒徵]]及其他學衡前輩影響極深，认同中國文化，對美國華人史有所研究。他曾表示中國的現代化必須與傳統相結合，絕對不能完全否定中國傳統而全盤西化。

他在中國時曾在[[國軍|國軍]]內當小兵，做過中學教員。在[[美國|美國]]長期從事[[歷史|歷史]]研究及[[口述歷史|口述歷史]]工作，與當時政要如[[顧維鈞|顧維鈞]]、[[李宗仁|李宗仁]]、[[陳立夫|陳立夫]]等人多有接觸，甚至成為好友。他和當時人在紐約的[[胡適|胡適]]則為忘年之交，有師生之誼，曾著《胡適口述自傳》（1981）、《胡適雜憶》（1979）。

他曾參與發起在全球徵集一億人簽名要求日本償付戰爭賠款的運動。2002年因[[釣魚臺|釣魚臺]]事件與[[楊振寧|楊振寧]]一同發表聲明，批評[[李登輝|李登輝]]。

== 思想 ==
'''歷史三峽觀'''：唐德剛有著名的歷史三峽論。他將四千多年以來的中國政治社會制度變遷分為封建、帝制和民治三大階段。其中要經歷兩大歷史三峽，也就是要經歷兩大歷史階段的轉變。“從封建轉帝制，發生於商鞅與秦皇漢武之間，歷時約三百年」 。「從帝制轉民治則發生於鴉片戰爭之後」 。並說經歷約200年，到大約西元2040年，中國歷史將會走完第二個「長江三峽」，迎來「潮平兩岸闊，風正一帆懸」的新時代。

==評論==
*旅美學人兼[[中研院院士|中研院院士]][[夏志清|夏志清]]教授譽之為「唐派散文」，「應公認是當代中國別樹一幟的散文家」，有詼諧直接、氣勢極盛、妙趣橫生的特点<ref>{{cite web| url = http://www.douban.com/group/topic/1055537/| title = 唐德剛-夏志清論戰錄| date = 2006-03-28| publisher = [[豆瓣|豆瓣]]| accessdate = 2010-05-27}}</ref>。
*[[張學良|張學良]]說唐德剛「把錢看得太重了」<ref>{{cite book|author = 林博文|title = 《張學良、宋子文檔案大揭秘》|publisher = 時報文化|year = 2007|isbn = 9789571347721|pages = 98}}</ref>。
*[[李敖|李敖]]稱唐德剛是「現代中國最活躍最優秀的歷史學家」。<ref>[http://news.fudan.edu.cn/2010/0119/23487.html 趙建民：〈唐德剛與《李宗仁回憶錄》〉，《復旦》第824期，2010年1月19日]</ref>
*[[吳章銓|吳章銓]]：「唐先生的特點是直言，他熱愛中國，他經常說的是「中國很有希望」，這個很有希望不是針對某一個政權，而是對一個國家、民族的反思。他一直在說，愛國不是愛某一個人。在政治上，他不傾向於任何黨派。」

===诗作===
'''金陵怀古'''

孺子沿街赤足，青山為雪白頭，<br>
金風如剪月如鉤，記取秦淮別後。<br>
臨去且行且止，回頭難拾難收。<br>
錯從苦海覓溫柔，曾把鮫綃濕透。

（1956年于美国）

'''海外驚聞毛澤東主席逝世'''（調寄〈沁園春〉，一九七六年九月）<ref>{{Cite book| author =唐德剛| title =毛澤東專政始末1949-1976| location =臺北| publisher =遠流| date =2018-04-12| pages =390| ISBN =9573253992| language =中文| url =https://books.google.com.tw/books?id=WbS7AAAAQBAJ&pg=PA390&lpg=PA390&dq=%E7%B5%82%E4%B8%8D%E8%A7%A3%EF%BC%8C%E5%95%8F%E6%B0%B4%E6%99%B6%E6%A3%BA%E5%85%A7%EF%BC%8C%E5%90%9B%E6%98%AF%E9%BE%8D%E8%9B%87%EF%BC%9F&source=bl&ots=hHhUf04QFj&sig=WfwjV_N7sHIlK77omiMWHoE_gP8&hl=zh-TW&sa=X&ved=0ahUKEwjN7ceIgbTaAhVLEbwKHW1TBJEQ6AEIJjAA#v=onepage&q&f=false| archiveurl =https://archive.today/20180412063656/https://www.google.com.tw/search?tbm=bks&hl=zh-TW&q=%E7%B5%82%E4%B8%8D%E8%A7%A3%EF%BC%8C%E5%95%8F%E6%B0%B4%E6%99%B6%E6%A3%BA%E5%85%A7%EF%BC%8C%E5%90%9B%E6%98%AF%E9%BE%8D%E8%9B%87%EF%BC%9F#v=onepage&q&f=false| archivedate =2018-04-12| deadurl =yes}}</ref>

牧豕韶山，潑水湘河，者個村娃。<br>
隨電雷激盪，手攬日月，身翻奴主，血染中華。<br>
政倚工農，權出槍桿，豎子真能不信邪。<br>
風流甚，看天安門上，吐氣成霞。

無加，霸業堪誇。詎難護重陽霜後花。<br>
聽長樂宮畔，孤彘暗泣；凌煙閣裡，百鬼交嗟。<br>
江上曲終，青峰邃渺，老樹枯藤宿暮鴉。<br>
終不解，問水晶棺內，君是龍蛇？

（作於1976年）
