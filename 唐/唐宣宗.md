'''唐宣宗李忱'''（{{bd|810年||859年|catIdx=Li,李}}），[[唐朝|唐朝]]第19代[[皇帝|皇帝]]（846年—859年在位，未算[[武周|武周]]政权），初名'''怡'''，登基之前封為'''光王'''，在位13年。[[唐宪宗|唐宪宗]]李纯十三子，母[[孝明鄭皇后|鄭宮人]]，[[元和_(唐朝)|元和]]五年（810年）[[六月廿二|六月廿二]]日生於[[大明宫|大明宫]]，是[[唐穆宗|唐穆宗]]李恒的弟弟，[[唐敬宗|唐敬宗]]李湛、[[唐文宗|唐文宗]]李昂、[[唐武宗|唐武宗]]李炎的叔叔。

== 生平 ==
唐宣宗李忱原名李怡，母親郑氏原为镇海节度使[[李錡|李錡]]侍妾，李锜謀反失敗後，郑氏被送入宫后当[[懿安皇后|郭貴妃]]的侍儿，后来被[[唐憲宗|唐憲宗]]临幸，生下李忱，封為光王，故唐宣宗為[[唐穆宗|唐穆宗]]之弟，[[唐敬宗|唐敬宗]]、[[唐文宗|唐文宗]]、[[唐武宗|唐武宗]]之叔父。唐宣宗為光王時，居於十六宅，故作愚鈍，曾被唐文宗及其他宗室作弄<ref>资治通鉴- 卷二百四十八：“初，宪宗纳李锜妾郑氏，生光王怡。怡幼时，宫中皆以为不慧，太和以后，益自韬匿，群居游处，未尝发言。文宗幸十六宅宴集，好诱其言以为戏笑，号曰光叔。”</ref>。

傳說唐宣宗登基之前，为了逃避姪唐武宗的迫害而出家為僧<ref>《贞陵遗事》：“上在藩时，尝从驾回，而上误坠马，人不之觉，比二更方能兴。时天大雪，四顾悄无人声。上寒甚，会巡警者至，大惊。上曰：我光王也，不悟至此，方困且渴；若为我求水。警者即于旁边得水以进，遂委而去……已而体微暖有力，遂步归藩也”。[[韦昭度|韦昭度]]的《续皇王宝应录》说，“武皇虑有他谏，乃密令中常侍四人擒宣宗于永巷，幽之数日，沉于宫厕。宦者仇公武愍之，乃奉武宗曰：前者王子不宜久于宫厕，诛之。武宗曰：唯唯。仇公武取出于车中，以粪土杂物复之，将别路归家，密养之三年，后武皇宫车晏驾，百官奉迎于玉宸殿立之，寻擢仇公武为军容使。”</ref>，传说他在[[河南省|河南]][[淅川县|淅川]][[香严寺_(淅川县)|香严寺]]避难<ref>{{cite web|url=http://xichuan.01ny.cn/wenxue/swsb/200706/4106.html|title=香严寺　唐宣宗的七年避难之地}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>1986年香严寺出土的《重修唐宣宗殿碑记》的碑文记载了唐宣宗李忱在香严寺避难之事</ref><ref>[[尉遲偓|尉遲偓]]《中朝故事》：“宣宗，即宪皇少子也；皇昆，即穆宗也；穆宗、敬宗之后，文宗、武宗相次即位，宣皇皆叔父也。武宗初登极，深忌焉。一日，会鞠于禁苑间，武宗召上，遥睹瞬目于中官，仇士良跃马向前曰：‘适有旨，王可下马。’士良命中官舆出军中，奏云：‘落马已不救矣。’寻请为僧，游行江表间。会昌末，中人请还京，遂即位。”熙宁八年（1075）苏轼游海宁时撰《大悲阁记》并七绝《悟空塔》：“已将世界等微尘，空里浮花梦里身；岂为龙颜更分别，只应天眼识无人。”</ref>，法名琼俊，[[齐安|齐安]]见其举止不凡<ref>《宋高僧传·齐安传》</ref>。不過不少史家，如北宋的[[司马光|司馬光]]，認為唐宣宗曾被唐武宗迫害及出家為僧是沒有根據，因正史中沒有相關記載，而且當時為光王的宣宗只是眾多的庶出皇叔其中之一，對唐武宗的帝位根本沒有威脅，更有史家認為唐宣宗被唐武宗迫害而出家為僧的講法，是後世佛教徒杜撰出來的故事，為的是醜化曾推行[[會昌毀佛|會昌毀佛]]的唐武宗，同時聖化在登基後推崇佛教的唐宣宗。

[[会昌_(唐朝)|会昌]]六年（846年），唐武宗被道士上供的长寿丹毒死。光王李怡被宦官擁立為帝，改名為李忱，是为唐宣宗，年号[[大中_(唐朝)|大中]]。擁立唐宣宗的宦官本以為他愚鈍容易控制，豈料他登基為帝後立即勵精圖治，並贬谪[[李德裕|李德裕]]，结束[[牛李党爭|牛李党爭]]。眾宦官、朝臣及宗室才驚覺唐宣宗以往是故作愚鈍，實際是非常賢明，有如其父唐憲宗
。

唐宣宗登基后，唐朝国势已暮氣沉沉，[[藩鎮割據|藩鎮割據]]，[[牛李党争|牛李党争]]，[[農民起事|农民起义]]，朝政腐败，官吏贪污，[[第二次宦官時代|宦官专权]]，四夷不朝。唐宣宗致力于改变这种状况，宣宗勤俭治国<ref>資治通鑑記載：越守嘗進女樂，有絶色者。上初悅之，數月，錫賚盈積。一旦晨興，忽不樂曰：“玄宗只一楊妃，天下至今未平，我豈敢忘？”乃召美人曰：“應留汝不得。”左右或奏“可以放還”。上曰：“放還我必思之，可命賜酖一杯。”（《資治通鑑》卷二百四十九大中十三年八月《考異》引《續貞陵遺事》）</ref>，体贴百姓，减少赋税，注重人才选拔<ref>[[司馬光|司馬光]]在《進修心治國之要劄子狀》中提到：“唐宣宗重惜服章，故當時得緋紫者以為榮。”</ref>，唐朝国势有所起色，社会矛盾有所缓和，百姓日渐富裕，使十分腐败的唐朝呈现出“[[中兴|中兴]]”的小康局面，史稱[[大中之治|大中之治]]。但只靠政府支支節節的改革，未能完全解決問題。宣宗是唐朝中期以後少數比较有作为的皇帝，另外，唐宣宗还趁[[吐蕃|吐蕃]]、[[回纥|回纥]]衰微，出兵收复河湟，安定塞北，國內百姓亦有申冤之地，又宽减刑法，对百姓加以安抚。又抑制宦官，革除弊制，唐朝出现中兴的局面。

大中十三年（859年）八月，宣宗因丹藥中毒驾崩，时年50岁。

==逸事==
*唐宣宗在位时，有一[[越州|越州]]美女天姿國色。唐宣宗初見之，宠爱异常。不久，唐宣宗怕自己耽誤國事，居然把她[[賜死|賜死]]。<ref>宣宗时，越守进女乐，有绝色。上初悦之，数日，锡予盈积。忽晨兴不乐，曰：“明皇只一杨妃，天下至今未平，我岂敢忘？”召诣前曰：“应留汝不得。”左右奏，可以放还。上曰：“放还我必思之，可赐酖一杯。”（《唐语林》引《续贞陵遗事》）</ref>

*唐末的[[黃巢之亂|黃巢之亂]]和藩镇战争使宣宗朝的實錄散失，使後人難以追查當年發生過的事<ref>《舊唐書·宣宗本紀》：惜乎简藉遗落，旧事十无三四，吮墨挥翰，有所慊然。</ref>。

*传说宣宗继位之前曾在[[淅川县|淅川]][[香严寺_(淅川县)|香严寺]]当过和尚<ref>[[尉迟握|尉迟握]]《中朝故事》记述李忱曾外出做和尚，“寻请为僧，游行江表间。会昌末，中人请还京，遂即位。”</ref>，所以对[[佛教|佛教]]极力推崇，据说曾在大中七年（853年）大拜[[释迦牟尼|释迦牟尼]]的[[舍利|舍利]]，關於這些資料見諸[[韋昭度|韋昭度]]《[[讀皇室運尋|讀皇室運尋]]》、[[令狐绹|令狐绹]]《[[偵陵遺事|偵陵遺事]]》、[[讚寧|讚寧]]《[[宋高僧傳|宋高僧傳]]》及[[僧圓悟禪師|僧圓悟禪師]]《[[碧岩集|碧岩集]]》。

*唐末[[益州|西川]]變民[[韓秀升|韓秀升]]在被[[高仁厚|高仁厚]]征服後，就曾坦言唐宣宗在位時天下尚有公道，唐宣宗故去之後就是勝者才有公道；高仁厚聞言後為之側目。可惜唐朝當時已是病入膏肓之軀，再沒有人能有力回天。

== 評價 ==
* 歷史上評價宣宗在位曾經燒過三把火：“權豪斂跡”、“奸臣畏法”、“閽寺詟氣”，稱之為“明君”，有“'''小太宗'''”的外號<ref>《[[資治通鑒|資治通鑒]]》載：“宣宗性明察沉斷，用法無私，從諫如流，重惜官賞，恭謹節儉，惠愛民物，故大中之政，訖於唐亡，人思詠之，謂之小太宗。”</ref>。據說唐宣宗退朝后還會讀書到半夜，燭灺委積，近侍呼之為“老儒生”<ref>見《[[紺珠集|紺珠集]]·大中遺事·老儒生》、《[[唐语林|唐语林]]》卷二</ref>。
* 明末清初的大儒[[王夫之|王夫之]]在《[[讀通鑑論|讀通鑑論]]》論“唐之亡，宣宗亡之”，“小說載宣宗之政，琅琅乎其言之，皆治像也，溫公亟取之登之於策，若有余美焉。自知治者觀之，則皆亡國之符也。”<ref>《[[读通鉴论|读通鉴论]]》卷二十六：夫宣宗之於吏治，亦勤用其心矣，徒厚疑其臣，而教貪自己。令狐綯父子黷貨於上，省寺相師而流及郡縣，塗飾耳目者愈密，破法以殃民也愈無所忌。唐之亡，宣宗亡之，豈待狡童繼起，始沈溺而莫挽哉？於是藩鎮之禍，且將息矣，河北諸帥皆庸豎爾，是弗難羈靮馭者，彼昏不知，惴惴然防之，而視東南為噬膚不知痛、瀝血不知號之圈豚池鶩也。「人莫躓於山，而躓於垤」，豈不信夫？民者，兵之命也；安者，危之府也；察者，昏之積也；弱者，疆之徒也。可不慎哉！可不慎哉！</ref>評價與《舊唐書》和《資治通鑑》的大力稱頌，實有天壤之別。

==== 子 ====
[[新唐书|新]][[旧唐书|旧唐书]]皆云<ref>《[[新唐书|新唐书]]·列传第七·十一宗诸子》</ref><ref name="jts1">《[[旧唐书|旧唐书]]·列传第一百二十五》</ref>，唐宣宗有十一子，但记为十二人，疑李渼非唐宣宗子。排行出于《旧唐书<ref name="jts2">《[[旧唐书|旧唐书]]·本纪第十八下·宣宗》宣宗即位，因诏有司迎太微宫寓主，祔废寺之新庙，而知礼者非之。制皇长男温可封郓王，二男泾可封雅王，第三男滋可封蕲王，第四男沂可封庆王......[[大中_(唐朝)|大中]]元年春......制宪宗第十七子惕封彭王，第十八子惴为棣王；皇第五子泽为濮王，第六子润为鄂王......五年春正月甲戌，制皇第七子洽封怀王，第八子汭为昭王，第九子汶为康王......（大中十一年六月）......制皇第三男灌封卫王，第十一男澭封广王......</ref><ref name="jts1"/>》和个人记载。正史仅记载过元昭太后生懿宗，出土墓志铭记载仇氏生康王李汶、陈氏生广王李澭，柳氏生昭王李汭，史氏生庆王李沂，吴氏生夔王李滋，其他王子的母親資料已失。

# 长子，懿宗[[唐懿宗|李漼]]，初名溫，原封鄆王。母美人晁氏（追尊元昭皇后）。
# 某子，靖怀太子[[李渼|李渼]]，[[会昌_(唐朝)|會昌]]六年（846年）封雍王，大中六年（852年）[[死亡|薨]]，追封。
# 次子，雅王[[李泾|李泾]]，會昌六年封王。
# 三子，夔王[[李滋|李滋]]，會昌六年封王。母吴氏，是否为吴昭仪无考。
# 四子，庆王[[李沂_(慶王)|李沂]]，會昌六年封王，墓志称第五子，會昌六年封王，大中十四年（846年）薨。母史氏，是否为史才人无考。 
# 五子，濮王[[李泽|李泽]]，[[大中_(唐朝)|大中]]元年（847年）封王
# 六子，鄂王[[李潤|李潤]]，大中元年（847年）封王，乾符三年（876年）薨。
# 七子，怀王[[李洽|李洽]]，大中五年（851年）與昭、康二王一起封王。
# 八子，昭王[[李汭|李汭]]，墓志称第九子。母柳氏，是否为柳婕妤无考。
# 九子，康王[[李汶_(康王)|李汶]]，墓志称第十子，乾符四年（877年）薨。母仇氏。
# 某子，卫王[[李灌|李灌]]，大中十一年（857年）与广王一起封王，大中十四年薨。称第三子<ref name="jts2"/>，疑为第十子误写为第三子。
# 十一子，广王[[李澭|李澭]]，乾符四年（877年）薨。母陈氏。

==== 女 ====

《新唐书》记唐宣宗有11位女儿<ref>[http://guoxue.baidu.com/page/d0c2ccc6cae9/95.html 《新唐书·诸帝公主列传》]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，另[[仇才人|仇才人]]生有一女，是否为其中一位公主，无考：
# 长女<ref name="a1">《全唐文 卷0079》：<封長女等為公主制>：長女可封萬壽公主，第二女可封永福公主，第三女可封西華公主，第四女可封廣德公主，第五女可封和義公主，第六女可封饒安公主。門下：諸女之封，始資漢制，疏邑之典，今為國章。長女等坤道稟柔，條風育德。莊敬柔順，受粹氣於靈源；言容法度，穆昭徽於《內則》。祥降北渚，教襲南薰，克茂天和，更承姆訓。雖年方齠齔，而體備肅雍，用洽寵私，俾開井賦。可依前件。</ref>，[[万寿公主|万寿公主]]（母元昭皇后晁氏，下嫁[[鄭顥|鄭顥]]）
# 次女<ref name="a1"/>，[[永福公主_(唐宣宗)|永福公主]]
# 三女<ref name="a1"/>，[[齊國恭懷公主|齊國恭懷公主]]（先封为西華公主，下嫁[[严祁|严祁]]）
# 四女<ref name="a1"/>，[[廣德公主_(唐朝)|廣德公主]]母元昭皇后晁氏（下嫁[[于琮|-{于}-琮]]）
# 五女<ref name="a1"/>，[[和義公主_(唐宣宗)|和義公主]]，《新唐书 列传第八》记为'''義和公主'''。
# 六女<ref name="a1"/>，[[饒安公主|饒安公主]]
# [[盛唐公主|盛唐公主]]
# 十一女，[[平原长公主|平原长公主]]（薨于[[咸通|咸通]]年间）
# [[唐陽公主|唐陽公主]]
# [[許昌莊肅公主|許昌莊肅公主]]（下嫁[[柳陟|柳陟]]，薨于[[中和_(唐僖宗)|中和]]年间）
# [[豐陽公主|豐陽公主]]
