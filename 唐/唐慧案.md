== 乐乐被迫卖淫案 ==
=== 乐乐被强奸和卖淫 ===
乐乐{{efn|name="note3"|“乐乐”为被媒体广泛使用的化名；相关报道、案件资料有时也会用张某某、张雯某、小青等称呼指代乐乐；其中“小青”为乐乐被迫卖淫时的称呼}}，1995年10月5日出生<ref name="qq72">{{cite web|url=http://shipei.qq.com/a/news/20130702018500|title=湖南永州劳教委：愿意放下姿态对唐慧进行赔偿|date=2013年7月2日|accessdate=2013年7月17日|publisher=腾讯|work=法制晚报}}</ref>，案发时为小学五年级学生<ref name="sohu72">{{cite web |url=http://news.sohu.com/20130413/n372547008.shtml|title=唐慧案庭审：劳教决定是否合法成攻防重点|date=2013年4月13日|accessdate=2013年7月17日|publisher=搜狐|work=中国青年报}}</ref>，系唐慧{{efn|name="note4"|也有的报导用化名“唐娟”}}的[[独生女|独生女]]<ref name="nfzm713">{{cite web|url=http://www.infzm.com/content/92041|title=永州劳教委：不劳教唐慧就是失职|date=2013年7月13日|accessdate=2013年7月17日|publisher=南方周末|work=}}</ref>。2006年10月1日下午，10岁的乐乐一人到自家附近的“快乐溜吧”溜冰场{{efn|位于永州市零陵区永州市职业技术学院医学院对面东方红超市负一层<ref name=ckxx>{{Citeweb|title=唐慧女儿案两主犯二审重审获无期徒刑|url=http://china.cankaoxiaoxi.com/roll10/2014/0905/487557.shtml|publisher=参考消息|date=2014-09-05}}</ref>}}，与正在溜旱冰的19岁<ref name="cds">湖南省高级人民法院刑事附带民事裁定书（2012）湘高法刑三终字第31号</ref>男子周军辉结识，周为医学院附近的“漂亮宝贝”理发店的员工。当晚，乐乐与周在“蓝色吧”出租房{{efn|位于永州市职业技术学院医学院对面<ref name=ckxx/>}}看碟，与周发生性关系后留宿于出租房。次日两人吃完早饭后，乐乐在回理发店途中被舅妈发现而带回家。10月3日，乐乐留下字条出走，找到周军辉后，被周介绍到“柳情缘休闲中心”（以下简称“柳情缘”）卖淫，一直到12月30日被家人发现。<ref name="sina811">{{cite web|url=http://news.sina.com.cn/c/2012-08-11/000024948525.shtml|title=湖南永州被逼卖淫幼女母亲讲述被劳教九天|date=2012年8月11日|accessdate=2013年7月17日|publisher=新浪|work=北京青年报}}</ref><ref name="cb87">{{cite web|url=http://www.cb.com.cn/1634427/20120807/403766_3.html|title=永州幼女被逼卖淫案细节曝光 省政法委介入|date=2012年8月7日|accessdate=2013年7月17日|publisher=中国经营网|work=}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name=nz029>{{Citeweb|url=http://www.infzm.com/content/93029 |title=“永州幼女被迫卖淫案”再调查 唐慧赢了，法治赢了没？|date=2013-08-01}}</ref>

乐乐出走后，唐慧家人到其居住地所属的[[零陵区|零陵区]]七里店派出所求助。该派出所指派副所长郭继仪负责。唐慧一家最初认定乐乐是被邻居王某拐骗而失踪的，并谎称乐乐在10月2日曾对家人说其前一晚与王某在一起{{efn|name="note5"|唐慧的弟弟唐世科引用了一段写于2006年10月2日的“乐乐自述”：“2006年10月1日夜，与王某在一起。王某要带我去广东读书”。}}<ref name=nz030>{{Citeweb|url=http://www.infzm.com/content/93030 |title=什么造就了唐慧 |date=2013-08-01}}</ref>。王某被排除作案可能后，唐慧及家人到拘留所大门围堵不准警方放人，警方只能安排王某翻墙离开。<ref name=nz030></ref>随后唐慧又控告当时警方负责人郭继仪收受王某贿赂致使郭继仪被停职审查，直至二个月后乐乐被找到。<ref name=nz030></ref>

此后此案负责人为[[零陵区|零陵区]]公安分局刑侦队中队长杨军祥。<ref name="sina811" /><ref name="chinawoman814" />

其实，乐乐10月3日出走的当天下午，再次到“漂亮宝贝”理发店找了周军辉，周军辉通过朋友“魏勇”联系，将乐乐带到了秦星和她男友陈刚在开设的“柳情缘休闲中心”{{efn|位于湖南省永州市零陵区潇湘中路柳子大酒店旁<ref name=ckxx/>}}卖淫。期间乐乐在秦星和陈刚的安排下被送到当地各宾馆卖淫。直至12月30日被解救为止，乐乐已经被迫卖淫百余次，并曾于12月下旬，遭刘润、蒋军军、兰小强和秦斌4人[[轮奸|轮奸]]。<ref name="ifeng0811">{{cite web |url=http://news.ifeng.com/society/special/hnyzyian/content-3/detail_2012_08/11/16721868_0.shtml?_from_ralated |title=唐慧：没保护好孩子，我对不起她 |accessdate=2012年8月11日 |publisher=凤凰网 |work= }}</ref>卖淫所得赃款被秦星、陈刚和周军辉分得。<ref name="chinawoman814" /><ref name="sina811" />

12月20日，唐慧从不明人士得知女儿身在柳情缘休闲中心，遂在附近查看监视数天，于30日下午5点多，看见了一个像女儿的人在休闲中心内。两名亲戚伪装为“顾客”进去确认后，唐慧叫来了刑侦队中队长杨军祥。唐慧称杨军祥未采取任何行动即离开，只得拨打110，由民警解救出乐乐。<ref name="cb87" />但杨军祥坚称当时未有人阻拦，而是乐乐不愿回家，而后杨军祥劝唐慧将乐乐先带回家中再去做笔录，被随后赶来的110警车送回家中。<ref name=nz030></ref>

乐乐被救后，经医生确诊为严重的[[創傷後心理壓力緊張綜合症|创伤后应激精神障碍]]<ref name="chinawoman814">{{cite web |url=http://www.china-woman.com/rp/main?fid=open&fun=show_news&from=view&nid=85599&ctype=4 |title=“永州上访妈妈被劳教事件”始末 |date=2012年8月14日 |accessdate=2013年7月17日 |publisher=中华女性网 |work=中国妇女报  }}</ref>和[[性病|性病]]，并极有可能丧失生育能力。<ref name="qq89" >{{cite web |url= http://news.qq.com/a/20120809/000437.htm|title= 湖南11岁幼女被逼卖淫续:染性病基本无生育能力|accessdate= 2012年8月9日|publisher= 腾讯网|work= }}</ref>

2006年12月31日，据唐慧反映，当天她找到负责乐乐失踪案的刑侦中队长杨军祥要求立案，但被其以“这只是个一般的治安案子”为由拒绝。之后，唐慧到“柳情缘休闲中心”所在的当地[[派出所|派出所]]报案，派出所以“我们管不了”为由拒绝，随后，唐慧遂以跳楼相逼。<ref name="chinawoman814" />杨军祥的解释是当时只有乐乐的单方面陈述，并没有“柳情缘”容留卖淫的证据，所以未对秦星等人采取强制措施，1月24日取得证据后才将其刑拘。<ref name=nz030></ref>

2007年4月4日，周军辉因涉嫌犯强奸罪被刑事拘留，同年5月12日被逮捕。<ref name="cds"/>

=== 乐乐案审判 ===
乐乐案共经过永州市中院的一审判决、两次重审判决和湖南省高院的终审判决四次审判。

在省公安厅批示下，永州市公安局成立了专案组。唐慧要求检方对警方掌握的嫖客同案起诉，但未获支持，百名嫖客中只有少数几个受到了拘留15日或罚款的行政处罚。<ref name="qq815">{{cite web|url=http://news.qq.com/a/20120815/003026.htm|title=湖南永州幼女卖淫案百余嫖客仅数人受行政处罚|date=2012年8月15日|accessdate=2013年7月17日|publisher=腾讯|work=法制网}}</ref>

2008年4月8日，永州市人民检察院向永州市中级人民法院提起公诉，指控被告人秦星、陈刚犯组织卖淫罪、强迫卖淫罪；被告人周军辉犯强迫卖淫罪、强奸罪；被告人刘润、兰小强、秦斌犯强奸罪；被告人蒋军军犯强奸罪、故意伤害罪。唐慧要求判决全部7名被告人死刑，附加184万元的民事赔偿。<ref name="cb87" />

2008年6月6日，永州市中级法院作出一审判决，判处秦星、周军辉[[死刑|死刑]]，刘润、陈刚[[无期徒刑|无期徒刑]]，蒋军军[[有期徒刑|有期徒刑]]16年，兰小强有期徒刑15年，7名被告人共赔偿9万元。秦斌因在逃而未受惩。<ref name="cb87" />唐慧申请抗诉，永州市检察院随后抗诉，认为蒋军军、兰小强量刑过轻。<ref name="msn">{{cite web|url=http://roll.msn.com.cn/windows8/20130715/0735173459.shtml|title=湖南永州幼女卖淫案百余嫖客仅数人受行政处罚|date=2013年7月15日|accessdate=2013年7月17日|publisher=MSN}}</ref>同时6名被告也提出上诉。<ref name="ifeng222">{{cite web|url=http://news.ifeng.com/society/1/detail_2012_02/22/12681303_0.shtml|title=湖南幼女被逼卖淫案：民警涉嫌为被告伪造立功材料|date=2012年2月22日|accessdate=2013年7月17日|publisher=凤凰网|work=东方早报}}</ref>

2008年8月8日，湖南省高级人民法院以原判没有为判处死刑的周军辉指定辩护人，限制了其法定诉讼权，影响公正审判为由，作出了撤销原判、发回重审的裁定。<ref name="cb87" /><ref name="ifeng222" />

2009年2月21日，永州市中院作出一审重审判决，秦星、周军辉被判死刑，陈刚、刘润被判无期徒刑，蒋军军被判有期徒刑16年，兰小强被判有期徒刑15年。原、被告双方不服，再次提出上诉。湖南省高院认定该案“部分事实不清，证据不足”，于2009年10月25日再度发回永州市中院重审。<ref name="ifeng222" /><ref name="cb87" />

2010年12月23日，该案第三次开庭，除周军辉以外的6名被告人当庭翻供，庭审场面一度失控。<ref name="sina73" >{{cite web|url=http://news.sina.com.cn/c/2013-07-03/130727565378.shtml|title=唐慧上访6年经历：曾因扰乱社会秩序罪被拘|date=2013年7月3日|accessdate=2013年7月17日|publisher=新浪|work=文汇报}}</ref>

2010年6月1日，零陵区法院发生枪击案，导致3名法官当场死亡，3名受伤，事后唐慧向部分媒体谎称罪犯嫌疑人朱军为乐乐干爹。<ref>{{Citeweb|url=http://news.ifeng.com/society/1/detail_2010_12/29/3757780_0.shtml |title=11岁女孩被逼卖淫案始末：被告人亲属在公安工作}}</ref>不久后唐慧因“扰乱社会秩序”被警方拘留，当时零陵区街道上张贴出大量《告零陵同胞书》，将此事称为是“一位孤胆英雄的剿匪行动”。警方锁定的四名嫌疑人除唐慧外均承认所为并表示唐慧也参与其中的目的是引起社会关注（但其本人否认）。<ref name=nz030></ref>警方申请批捕唐慧时，检察院出现分歧，上报至湖南省检察院最终未被批准。<ref name=nz030></ref>

2011年{{efn|name="note1"|乐乐案第三次判决下达时间，许多媒体笔误作“2010年3月28日”并大量互相转载。实为2011年3月28日。}}3月28日，永州中院再次对7名被告人进行了重新判决，秦星、周军辉死刑，陈刚、刘润、蒋军军、兰小强四人无期徒刑，判处秦斌有期徒刑15年，同时7名被告人赔偿20万元。随后双方再度上诉。<ref name="ifeng222" />

2012年2月21日，湖南省高级人民法院在永州市中级人民法院开庭对此案开庭进行终审。随后在2012年6月5日作出“驳回上诉，维持原判”的裁定。<ref name="qq810" >{{cite web |url=http://news.qq.com/a/20120810/002164.htm |title=湖南被逼卖淫幼女母亲：警察不立案是不是渎职|date=2012年8月10日|accessdate=2013年7月17日|publisher=腾讯网|work=人民网}}</ref>湖南省高院维持了对秦星、周军辉的死刑判决并报[[最高人民法院|最高人民法院]]复核。

最高人民法院经复核认为，第一审判决、第二审裁定认定的[[强迫卖淫罪|强迫卖淫]]、[[强奸罪|强奸]]、[[组织卖淫罪|组织卖淫]]事实清楚，证据确实、充分，定罪准确，审判程序合法。鉴于周军辉、秦星强迫卖淫的暴力、胁迫程度，犯罪情节的恶劣程度尚未达到[[情节特别严重|情节特别严重]]，对二被告人以强迫卖淫罪判处死刑立即执行[[量刑不当|量刑不当]]。本案复核期间出现新的证据，可能影响对秦星是否构成[[立功|立功]]的认定，依法应予查明。依法裁定不核准周军辉、秦星死刑，将案件发回湖南省高级人民法院重新审判。

====秦星救人立功争议====
一审期间，永州市公安局冷水滩分局冷水滩区看守所出具了该案第一被告人秦星羁押期间的多份立功证明材料，证明秦星于2007年6月12日下午解救了一名正在进行上吊自杀的在押人员周兰兰，材料包括：一份盖有看守所公章的《关于报请对在押人员秦星依法从宽处理的报告》、一份看守所管教干部的证词、以及一份13名同押人员签字的书面证明。但在同押人员的签字证明中，并无“自杀被救人”周兰兰的签名。据《南方周末》记者柴会群称，其于永州当地调查时调阅的监狱录像确实能佐证救人之说。但周兰兰否认有此事。<ref name="chinawoman814" />

多次审判中该立功证明材料均被提出，但都未获法院认同。2009年2月21日永州中院的判决中，认定“秦星的立功材料已收集将近一年，但从未报主管部门核查确认，不予采纳”。<ref name="dfdaily87">{{cite web|url=http://epaper.dfdaily.com/dfzb/html/2012-08/07/content_658378.htm|title=有民警协助被告“假立功”，早经法院多次裁定，公安局却未予处理，直到劳教唐慧后才开始自查，是否公平？|date=2012年8月7日|accessdate=2013年7月17日|publisher=东方早报}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
