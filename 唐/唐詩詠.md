'''唐詩詠'''（'''{{lang|en|Natalie Tong}}'''，{{bd|1981年|5月3日|catIdx=T唐}}），原名'''唐思盈'''，[[香港|香港]]女演員、電視女藝人，於2003年加入香港[[無綫電視|無綫電視]]（TVB），現為該公司旗下[[經理人|經理人]]合約藝人。2018年，於《[[萬千星輝頒獎典禮2017|萬千星輝頒獎典禮2017]]》憑《[[不懂撒嬌的女人|不懂撒嬌的女人]]》奪得「最佳女主角」<ref>{{cite web |url=https://www.sundaymore.com/589373/celebrity/2017萬千星輝頒獎典禮-視后/|title=2017萬千星輝頒獎典禮結果大爆冷！唐詩詠奪視后惹網民劣評|publisher=Sunday More|date=2018-01-22}}</ref><ref>{{cite web |url=http://hk.on.cc/hk/bkn/cnt/entertainment/20180124/bkn-20180124140058376-0124_00862_001.html?eventsection=hk_entertainment&eventid=402883474d9a0d4e014da2c02bab5d1e|title=唐詩詠傳續約換獎，宣萱失落視后|publisher=東網|date=2018-01-24}}</ref><ref>{{cite web |url=http://news.modia.com.hk/2018/01/22/50497/揮低宣萱畀網民鬧爆傳唐詩詠續約無綫奪視后/|title=揮低宣萱畀網民鬧爆　傳唐詩詠續約無綫奪視后|publisher=流動新聞|date=2018-01-22}}</ref><ref>{{cite web |url=https://www.thekono.com/articles/12773c67-e762-4cfe-be65-ef3b5d1b46c1|title=成后之路 唐詩詠未夠班|publisher=Kono Magazine|date=2018-07-12}}</ref>。

==背景==
唐詩詠畢業於[[聖公會聖紀文小學|聖公會聖紀文小學]]及[[葵涌循道中學|葵涌循道中學]]，中學時期開始當兼職[[模特兒|模特兒]]。2000年，唐詩詠拍攝[[香港電台電視部|香港電台電視部]]製作的青春劇《青春@Y2K》嶄露頭角，開始為年輕人認識，並多次參與廣告及雜誌拍攝。

2017年，唐詩詠與圈外朋友合資7位數字，於百貨公司櫃位開設日式芝士蛋糕店 <ref>[http://bka.mpweekly.com/channel/%E5%94%90%E8%A9%A9%E8%A9%A0%E5%8F%AB%E6%B4%AA%E6%B0%B8%E5%9F%8E%E5%B9%AB%E8%A5%AF-%E6%83%B3%E7%8E%A9rap%E5%AE%A3%E5%82%B3%E8%87%AA%E5%AE%B6%E5%BA%97 唐詩詠叫洪永城幫襯 想玩rap宣傳自家店]</ref>。

==電視事業==
===上位期（2003年－2015年）===
唐詩詠於2003年加入[[無綫電視|無綫電視]]，拍攝的首部劇集是青春劇《[[當四葉草碰上劍尖時|當四葉草碰上劍尖時]]》中擔任第二女主角，同劇演員有[[黎諾懿|黎諾懿]]、[[姚子羚|姚子羚]]、[[李雨陽|李雨陽]]等。期後亦參演多部電視劇中擔任第二女主角，包括《[[天幕下的戀人|天幕下的戀人]]》、《[[緣來自有機|緣來自有機]]》、《[[最美麗的第七天|最美麗的第七天]]》等電視劇，在《[[心花放|心花放]]》中飾演林小湖一角開始為人認識，她在2007年首次有份拍攝[[無綫月曆|無綫月曆]]。

2010年，入行七年的唐詩詠在[[李添勝|李添勝]]監製的劇集《[[鐵馬尋橋|鐵馬尋橋]]》中飾演悲慘的應雁鳴（阿雁）一角，奪得該年度的《[[萬千星輝頒獎典禮|萬千星輝頒獎典禮]]》飛躍進步女藝員奬，以及入圍角逐最佳女配角。她領獎時哭成涙人，感謝已離世的母親，還表示要帶同獎座去拜祭她。<ref>{{cite news|url=http://paper.wenweipo.com/2010/12/06/EN1012060001.htm|title=黎耀祥 鄧萃雯蟬聯帝后|newspaper=[[文匯報|文匯報]]|date=2010-12-06}}</ref>。唐詩詠也奪得2011年《[[壹電視大獎|壹電視大獎]]》「十大電視藝人的第10位」。

2013年，唐詩詠加入無綫電視已經十年。她在[[李添勝|李添勝]]監製的劇集《[[神探高倫布|神探高倫布]]》中擔演女主角並一人分飾兩角，與三屆視帝[[黎耀祥|黎耀祥]]再次合作演出。劇集播映以來亦劣評如潮，但唐詩詠於劇中飾演精神病患者梅妺，其中梅妺被處死前，與黎耀祥飾演的高大喜臨别一幕，觀眾及網民大讚唐詩詠飾演精神病人極為神似。

2015年7月，唐詩詠因不肯續約而遭TVB雪藏，故拍畢《[[公公出宮|公公出宮]]》後沒有接拍任何新劇。

===晉身當家花旦時期（2016年至今）===
2016年11月，唐詩詠拍攝《[[鐵馬戰車|鐵馬戰車]]》，擔任第一女主角，飾演記憶女王姚瑤一角，。

2017年，唐詩詠在劇集《[[不懂撒嬌的女人|不懂撒嬌的女人]]》中飾演勤力、能幹、能吃苦，擁有「一顆恨嫁的心」，但不懂撒嬌的女主角「凌禹勤」。她在劇中發現男友何志超（[[黎諾懿|黎諾懿]]飾演）北上出軌後，更要忍住眼淚跟男友母親講電話；又在另一幕聲嘶力竭地怒罵上司兼堂姐凌敏（[[宣萱|宣萱]]飾演）。唐詩詠憑此劇成為首位奪得《英國大本鐘金獨角獸大獎》「最受推崇女主角」的[[香港|香港]]女藝人，亦是入行至今首次奪得視-{后}-寶座。另外唐詩詠在100%全民投票的《[[2017年度YAHOO!搜尋人氣大獎得獎名單|YAHOO!搜尋人氣大獎2017]]》中亦奪得「電視女藝人」大獎。其後於2018年1月21日，唐詩詠於《[[萬千星輝頒獎典禮2017|萬千星輝頒獎典禮2017]]》中再下一城，奪得「最佳女主角」獎。<ref>[https://www.orientalsunday.hk/%E6%9C%80%E6%96%B0%E5%A8%9B%E8%81%9E/36%E6%AD%B2%E5%94%90%E8%A9%A9%E8%A9%A0-%E9%A6%96%E5%A5%AA%E8%A6%96%E5%90%8E/ 36歲唐詩詠 首奪視-{后}-]</ref><ref>[http://www.weekendhk.com/entertainment/%E8%90%AC%E5%8D%83%E6%98%9F%E8%BC%9D%E9%A0%92%E7%8D%8E%E5%85%B8%E7%A6%AE2017%E5%BE%97%E7%8D%8E%E5%90%8D%E5%96%AE/4/ 萬千星輝頒獎典禮2017得獎名單 王浩信、唐詩詠奪視帝視-{后}-]</ref>。

== 電影事業 ==
2002年，她開始參與了第一部電影《嫁個有錢人》，並在此電影中飾演[[雷頌德|雷頌德]]老婆，期後主要電影裏擔任配角的演出，包括《[[2002_(電影)|2002]]》、《[[墨斗先生|墨斗先生]]》、《龍咁威2之皇母娘娘呢？》。

2005年，她參演了首部單元女主角電影《[http://www.u333.com/films/firework 擁抱每一刻花火]》，亦是她首部主演的愛情電影，嶄露頭角，開始為大眾認識。

2008年，她參演了兩部電影，包括《[[我的最愛|我的最愛]]》、《[[出水芙蓉_(2010年電影)|出水芙蓉]]》都是擔任配角的演出。

2013年，[[古天樂|古天樂]]欽點唐詩詠在《[[逃出生天|逃出生天]]》飾演其未婚妻，擔任第二女主角，戲份佔重亦成功開拓電影市場。古天樂在訪問說：「她有一場戲給我的感覺很適合該角色。因為我在看（無綫電視）電視劇是別人跟她求婚，我覺得那場戲較重要，我們後來的生離死別就因為那場求婚戲，所以要找對角色才行。」

2016年3月，她接拍[[葉念琛|葉念琛]]監製電影《[[我要發達|我要發達]]》，擔正此電影的女主角。此電影為唐詩詠擔正女主角的處女電影，並於2017年[[農曆新年|農曆新年]]上映。

2017年，她再次接拍葉念琛監製電影《[[我的情敵女婿|我的情敵女婿]]》，並於2018年農曆新年上映。

==感情生活==
2000年，唐詩詠與[[余文樂|余文樂]]拍攝[[香港電台|香港電台]]劇集《青春@Y2K》時擦出愛火，之後余棄愛，戀上[[盧巧音|盧巧音]]。

2005年，唐詩詠與[[崔建邦|崔建邦]]傳出緋聞，多年後兩人公開戀情。唐詩詠與崔建邦在一起後多次離合，均成為傳媒報導及網民討論的話題。而2009年底，發生崔建邦涉嫌虐打中巴混血兒[[鄺敏慧|鄺敏慧]]事件後，唐詩詠對崔建邦不離不棄以及公開發聲護航，令網民對其倍感欣賞，因而得到「100分好女友」之稱。直至數年後崔建邦再次出軌，唐詩詠終於無法忍受男方花心、不忠的性格，最後兩人感情日漸變淡而分手。

在2013年，唐詩詠與[[洪永城|洪永城]]因拍攝劇集《[[載得有情人|載得有情人]]》擦出爱火花。惟在2017年兩人感情日漸生變<ref>[http://www.orientalsunday.hk/%e6%9c%80%e6%96%b0%e5%a8%9b%e8%81%9e/%e6%8b%96%e6%8b%89%e5%85%a9%e5%b9%b4-%e6%ad%bb%e6%80%a7%e4%b8%8d%e6%94%b9-%e5%94%90%e8%a9%a9%e8%a9%a0%e6%b4%aa%e6%b0%b8%e5%9f%8e%e9%99%b7%e5%86%b7%e6%88%b0/ 拖拉兩年 死性不改 唐詩詠洪永城陷冷戰]，[[東方新地|東方新地]]，2017年1月19日</ref><ref>{{Cite news |url=http://hk.apple.nextmedia.com/entertainment/art/20170120/19902587 |title=唐詩詠同洪永城冇見面  |date=2017-01-20 |newspaper=[[蘋果日報_(香港)|蘋果日報]]| language=zh-yue}}</ref>，不過傳言未能得到證實。

2017年5月，兩人宣布正式分手<ref>{{citation|url=https://www.instagram.com/p/BUJrSMblD77/?taken-by=natalietong_tsw|title=唐詩詠instagram帖文|date=2017-05-16}}</ref><ref>{{cite web|url=http://eastweek.my-magazine.me/main/66370 | title=唐詩詠、洪永城宣布分手 強調冇第三者或結婚問題 |publisher=東周網|date=2017年5月17日}}</ref><ref>{{cite web|url=http://hk.apple.nextmedia.com/entertainment/art/20170517/20023413 | title=眼腫講情變 唐詩詠洪永城斬纜「分手不容易」 |publisher=[[蘋果日報_(香港)|蘋果日報]]|date=2017年5月17日}}</ref>。

2017年12月，唐詩詠接受訪問時表示她跟洪永城分開有好多原因，例如壓力，一直以為拍拖只是兩個人的事，她只會活在自己的世界，不理外面的事，原來另一半不是這樣的，原來他是有壓力的，並表示將來再拍拖，要學懂怎樣保護對方，為他設想<ref>[https://bka.mpweekly.com/focus/20171230-96444 【封面故事1】唐詩詠首談分手 一百分女友令洪永城感壓力]</ref>。
