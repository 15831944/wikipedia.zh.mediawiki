'''唐志中'''（{{lang|en|Jason}}，{{bd|1978年|11月17日}}），[[台灣|台灣]]男[[藝人|藝人]]，父親為藝人[[唐威_(演員)|唐威]]及母親為藝人[[韓湘琴|韓湘琴]]，姐姐[[唐可珊|唐可珊]]前[[民視|民視]]新聞主播<ref>任美珍.[http://www.businessweekly.com.tw/KArticle.aspx?id=9018 元大馬志玲看電視選媳婦 老爸推薦、老妹出面，馬維建娶到唐可珊].商業周刊第626期.1999-11-18</ref>。

== 演藝經歷 ==
唐志中14歲赴[[美國|美國]]念中學，就讀於[[美國西岸|美國西岸]]的[[寄宿學校|寄宿學校]]，曾入選[[籃球|籃球]]校隊。放暑假回台灣時，得知[[Channel_V|Channel V]]正在徵求VJ，便報名參加試鏡。Channel V覺得唐志中具有美國西岸青少年的談吐舉止，便錄用他當《燙手區》節目的主持人。2015年，带领女儿“小蜜蜜”、“小香香”，参加由浙江卫视推出的大型明星亲子真人秀节目《[[爸爸回来了|爸爸回来了]]》 第2季。

== 個人生活 ==
唐志中熱愛[[Hip_Hop|Hip Hop]]音樂，在節目中專為觀眾推介最新西洋歌曲。2009年起，唐志中經營一家經紀公司。2011年1月17日，唐志中與女友張庭緁（小咪）在[[台中市|台中市]]舉行[[訂婚|訂婚]]喜宴。2011年3月4日，唐志中與張庭緁(小咪）在台北市舉行[[結婚|結婚]]喜宴。

== 爭議事件 ==
*喜好[[夜店|夜店]]生活，友人稱其綽號為「東區唐不挑」。目前仍以華僑身分在台灣居留，未曾在台灣服兵役<ref>[http://www.nownews.com/n/2010/02/11/593638 「夜店咖」唐志中變乖乖牌！ 賣衣兼做公益].NOWnews 今日新聞.2010-02-11 {{Wayback|url=http://www.nownews.com/n/2010/02/11/593638 |date=20140322104002 }}</ref>。

*2006年[[金曲獎|金曲獎]]主持幕後花絮，介紹藝人[[蕭瀟|蕭瀟]]之項鍊，疑似藉機伸鹹豬手，影片並在網路流傳。

*2013年10月26日在天母運動公園，先請爸爸[[唐威_(演員)|唐威]]下車佔車位，再開車強行切入，惹火原本在後面排隊等車位的駕駛人，雙方吵架驚動警察。

*2013年12月18日在知名美式甜甜圈品牌「Krispy Kreme」，唐志中不顧後面大排長龍的隊伍，企圖強行進入店家，經店員勸阻後，唐志中改要求站在隊伍最前面的高中生一起排隊，成功買到甜甜圈，據稱當天人潮眾多，排隊至少需要四小時。

*2015年1月31日，在台北電玩展的神魔之塔攤位擔任主持人，引發不少爭議<ref>希飛斯 xxnfj.[http://forum.gamer.com.tw/Co.php?bsn=23805&sn=2448852 【問題】真心覺得唐志中的主持很爛!!!就算被刪我也要發文！！].@神魔之塔 哈啦板 - 巴哈姆特.2015-02-04</ref>。
