'''唐槐秋'''（{{bd|1898年10月14日||1954年4月25日||X}}），原名'''唐震球'''，著名[[中國|中國]]演員、導演，[[湖南|湖南]][[湘鄉|湘鄉]]人。其夫人為蘇之卉{{noteTag|出自蔡登山《曾經輝煌: 被遺忘的文人往事》，第183頁，2008年秀威資訊出版}}，女兒為[[香港|香港]]已故著名甘草演員[[唐若菁|唐若菁]]。

== 生平 ==
1911年至1916年，唐槐秋在[[日本|日本]]留學，其後完成[[東京|東京]]成城中學中學課程。1919年，他遠赴[[法國|法國]]華爾曼航空專科學校學習航空機械{{noteTag|出自邵迎建《抗日戰爭時期上海話劇人訪談錄》，第116頁，2011年秀威資訊出版}}。1925年，唐槐秋回到[[上海|上海]]。1926年，他在[[上海|上海]]與[[田漢|田漢]]組織[[南國社|南國社]]及[[廣東|廣東]]戲劇研究所。1929年初，唐氏應[[廣東|廣東]]戲劇研究所所長[[歐陽予倩|歐陽予倩]]的邀請到[[廣州|廣州]]擔任研究所的劇務主任和研究所轄下的戲劇學校教務主任<ref name="唐槐秋（1898—1954）">{{Cite web |url=http://www.mcprc.gov.cn/ggfw/whzyzg/yishu/huaju/201111/t20111121_377983.html |title=中華人民共和國文化部〈唐槐秋（1898—1954）〉 |access-date=2017-06-27 |archive-url=https://web.archive.org/web/20171201034930/http://www.mcprc.gov.cn/ggfw/whzyzg/yishu/huaju/201111/t20111121_377983.html |archive-date=2017-12-01 |dead-url=yes }}</ref>。1933年2月，他成立中國電影文化協會，並當選執行委員。同年11月，他在[[上海|上海]]創辦了只有6名成員的[[中國旅行劇團|中國旅行劇團]]{{noteTag|見焦雄屏《風雲際會 與當代中國電影對話》〈談中國戲劇傳統〉，第195頁，1998年由遠流出版事業股份有限公司出版。}}。在《都會的早晨》、《失戀》、《桃李劫機》、《大地之花》、《[[孔夫子_(電影)|孔夫子]]》、《洪宣嬌》、《大雷雨》等電影中，他都演出主要角色。1934年3月，唐槐秋在[[南京|南京]]首演《梅夢香》。同年7月，他與[[中國旅行劇團|中國旅行劇團]]成員往[[北平|北平]]演出《[[茶花女|茶花女]]》、《牛大王》、《女店主》、《英雄與美人》、《天羅地網》、《復活》、《委曲求全》、《[[雷雨|雷雨]]》等馳中外的戲劇，且培育了[[唐若菁|唐若菁]]、[[陶金|陶金]]、[[趙慧深|趙慧深]]、[[白楊|白楊]]、[[戴涯|戴涯]]、[[藍馬|藍馬]]、[[舒繡文|舒繡文]]、[[孫道臨|孫道臨]]等很多著名演員<ref name="話劇百年 中國旅行劇團">[http://www.library.sh.cn/zthd/hjbn/js/js_13.htm 上海圖書館上海科學技術情報研究所〈話劇百年 中國旅行劇團〉]</ref>。這個劇團在大江南北皆獲好評{{noteTag|出自春去也《華影壇: 中港台影壇趣聞》，第410頁，2014年新銳文創出版}}。1949年，唐氏擔任[[中國|中國]]戲曲研究院導演。至1954年病逝。1984年11月，[[中國戲劇家協會|中國戲劇家協會]]為紀念[[中國旅行劇團|中國旅行劇團]]成立50周年暨唐槐秋逝世30周年舉行了紀念會<ref>{{web reference|URL=http://www.ccb.com/cn/ccbtoday/jhbkhb/20150601_1433152009.html|title=沈寂〈感恩豪爽的孫景璐〉|work=中國建設銀行報客戶版|year=2015年|date=6月3日}}</ref>。

== 評價 ==
前[[黃埔軍校|黃埔軍校]]政治部秘書[[陽翰笙|陽翰笙]]和著名劇作家[[曹禺|曹禺]]曾稱唐槐秋是「中國話劇事業的開拓者」。
