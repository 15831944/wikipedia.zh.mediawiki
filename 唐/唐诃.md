'''唐诃'''（{{bd|1922年|11月|2013年|7月25日}}）原名'''张化愚'''，[[河北|河北]][[易县|易县]]人，[[中华人民共和国|中华人民共和国]]作曲家<ref name=tangtang>{{Cite web |url =http://www.qdlgb.gov.cn/whyd/details.aspx?id=1420160214155243  |title = 堂堂正正一大家——忆唐诃 |author =刘真骅  |date =  |publisher = 中共青岛市委老干部局  |language =  |accessdate = 2017-08-27 |quote =  }}</ref><ref name=zhuchuang/>。

==生平==
1922年11月，唐诃生于河北易县。自幼热爱音乐、文学和书法。1938年[[抗日战争|抗日战争]]期间，唐诃随父兄一起参加了[[晋察冀军区|晋察冀军区]][[八路军|八路军]]。唐诃生长在农村，从未受过正规音乐教育，但接受了农村民间音乐、[[戏曲|戏曲]]、[[学堂乐歌|学堂乐歌]]、地方驻军军歌的影响。歌曲《凌霄花》影响了他的一生，其中最后两句“寄言立身者，勿学柔弱苗”对他影响最大<ref name=tangtang/>。

1940年，在纪念抗日战争全面爆发三周年时，组织干事房晓星写出一首《七·七之歌》，17岁的唐诃首次谱曲，并在万人纪念大会上指挥了这支歌的演唱，受到团领导赞扬。后来，七月剧社的音乐教师韦虹将唐诃调到七月剧社任分队长，韦虹成了唐诃在作曲上的启蒙教师。抗日战争时期的《我们是新中国的先锋》、[[第二次国共内战|解放战争]]时期的《军队向前进》、[[抗美援朝战争|抗美援朝战争]]时期的《志愿军战士之歌》等歌曲都是唐诃的代表作<ref name=tangtang/>。

1954年[[春节|春节]]，唐诃到[[山西|山西]][[祁县|祁县]]采风，学到很多民间乐曲，以及晋中秧歌等等。此后，他用秧歌《洗衣计》中的一个基本曲调揉和了[[山西梆子|山西梆子]]以及[[河北|河北]][[定县|定县]]秧歌谱写出女声合唱《在村外小河旁》，成为部队抒情歌曲，后来被推荐到1956年全国音乐周并获得好评，全国20余家报刊均刊登了这支歌，并且在[[朝鲜民主主义人民共和国|朝鲜]]、[[日本|日本]]、[[苏联|苏联]]及[[东欧|东欧]]各国传唱，日本有人将这支歌改编为轻音乐。这支歌是唐诃第一首成名作<ref name=tangtang/>。

1965年，为纪念红军[[长征|长征]]胜利30周年，[[肖华|肖华]]写出组诗《[[长征组歌|长征组歌]]——红军不怕远征难》并点名要求[[中国人民解放军陆军政治工作部文工团|北京军区战友文工团]]组织作曲家谱曲，[[晨耕|晨耕]]、[[生茂|生茂]]、唐诃、[[遇秋|遇秋]]4人遂参加谱曲。在创作排练中，他们获得了[[周恩来|周恩来]]总理及老红军们的关心及指导。周恩来还专门接见了唐诃等4位曲作者，并对每段提出具体意见<ref name=tangtang/>。

1987年3月，[[中国音乐家协会|中国音乐家协会]]、河北省文联音乐家协会联合在[[河北省|河北省]][[石家庄市|石家庄市]]举办“唐诃作品音乐会”，预定演出3场，后应观众要求增演2场，场场爆满。[[于淑珍|于淑珍]]、[[马国光|马国光]]、[[贾世骏|贾世骏]]、[[张振富|张振富]]、[[耿莲凤|耿莲凤]]等歌唱家参加演唱。《[[解放军报|解放军报]]》、《[[文艺报|文艺报]]》、《[[河北日报|河北日报]]》等媒体进行了报道。[[中央军委副主席|中央军委原副主席]][[迟浩田|迟浩田]]为唐诃音乐会题词：“军歌振军威，民歌扬正气。”老将军[[华楠|华楠]]题词：“奏时代强音，振民族精神。”音乐家[[时乐濛|时乐濛]]题词：“扎根民族土壤，飘洒泥土芳香，展示军人魂魄，闪耀时代之光。”<ref name=tangtang/>

1989年，唐诃定居[[山东省|山东省]][[青岛市|青岛市]]后，又创作了许多歌曲。其中《最美是中华》1994年由于淑珍演唱后，获[[中央人民广播电台|中央人民广播电台]]金奖，并获第一届政府奖等奖项<ref name=tangtang/>。唐诃还义务为不少歌唱团体当指导，并创作了《青岛美》、《海之魂》等歌曲，晚年还创作了很多村歌、校歌<ref name=zhuchuang>{{Cite web |url = http://ent.sina.com.cn/y/2013-07-26/14593973506.shtml |title = 作曲家唐诃去世 曾参与主创《长征组歌》 |author =  |date = 2013-07-26 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

2013年7月23日，唐诃因肺部感染导致呼吸衰竭在青岛逝世，享年91岁<ref name=tangtang/>。

==作品==
唐诃一生创作歌曲千余首，歌剧、电影音乐20多部<ref name=tangtang/>。《牡丹之歌》、《众手浇开幸福花》、《我们的生活充满阳光》、《甜蜜的事业》等歌曲脍炙人口，《长征组歌》是他重要的代表作<ref>{{Cite web |url =http://e.chengdu.cn/html/2013-07/28/content_415483.htm  |title = 91岁作曲家唐诃去世曾创作《牡丹之歌》 |author =  |date = 2013-07-28 |publisher = 成都商报 |language =  |accessdate =  |quote =  }}</ref>。

唐诃创作了大量电影音乐。1961年电影《花儿朵朵》、1962年电影《锦上添花》、1979年电影《甜蜜的事业》、1980年电影《红牡丹》的音乐均受到观众欢迎。其中《我们的生活充满阳光》（故事片《甜蜜的事业》主题歌，秦赤钰作词，[[吕远|吕远]]、唐诃作曲，[[于淑珍|于淑珍]]首唱）获[[中华人民共和国文化部|文化部]]优秀歌曲奖，后来被[[联合国教科文组织|联合国教科文组织]]选为[[亚洲|亚洲]]音乐教材，《牡丹之歌》（故事片《红牡丹》主题歌，[[乔羽|乔羽]]作词，吕远、唐诃作曲，[[蒋大为|蒋大为]]首唱）获[[中央电视台|中央电视台]]“五洲杯”金奖，并获[[中国唱片社|中国唱片社]]“群众喜爱歌曲奖”、文化部“优秀歌曲奖”等奖项<ref name=tangtang/>。

==家庭==
*第一任妻子：张素萍，1987年逝世，与唐诃相伴41年。育有四个儿子<ref name=zhuchuang/>。
*第二任妻子：李爱英，曾是青岛一家市级医院的医生。1987年张素萍逝世后，组织安排悲痛的唐诃到[[烟台|烟台]]疗养，在烟台唐诃结识了李爱英。当时李爱英已离婚，有两个女儿。后来唐诃和李爱英结婚<ref name=zhuchuang/>。
