'''唐纳德·莫法特'''（{{Lang-en|Donald Moffat}}，{{bd|1930年|12月26日|2018年|12月20日|catIdx=Moffat}}）是一名长期活跃在[[美国|美国]]的[[英国|英国]]籍[[演员|演员]]。唐纳德·莫法特从[[外百老汇|外百老汇]]和[[百老汇音乐剧|百老汇音乐剧]]开始了他的演员生涯，他凭借作品《{{Link-en|野鸭 (戏剧)|The Wild Duck|野鸭}}》和《{{Link-en|你正确（所以你正确）|Right You Are (if you think so)}}》两度获得[[托尼奖|托尼奖]]提名，并依靠《{{Link-en|绘画教堂|Painting Churches}}》而获得{{Link-en|奥比奖|Obie Award}}。唐纳德·莫法特也出演过一些真人电影，例如：《[[怪形|怪形]]》和《[[太空先鋒_(電影)|太空先锋]]》；另外他还以客串形式出演了包括《{{Link-en|女医生|Dr. Quinn, Medicine Woman}}》和《[[白宫风云|白宫风云]]》在内的部分电视剧。

== 早年生活 ==
唐纳德·莫法特出生于[[英国|英国]][[德文郡|德文郡]]的[[普利茅斯|普利茅斯]]，他是他们家庭里的独生子。他的母亲名叫凯瑟琳·玛丽·莫法特（{{lang|en|Kathleen Mary Moffat}}），其婚前姓氏为史密斯（{{lang|en|Smith}}）；而他的父亲名叫沃尔特·乔治·莫法特（{{lang|en|Walter George Moffat}}），是一名保险代理人。他的父母后来在[[托特尼斯|托特尼斯]]经营一所寄宿公寓。1949年至1951年期间，唐纳德·莫法特在当地的{{Link-en|爱德华六世国王社区学院|King Edward VI Community College}}念书；后加入[[英国陆军|英国陆军]]服役。随后他在[[伦敦|伦敦]]的[[皇家戏剧艺术学院|皇家戏剧艺术学院]]接受培训。<ref name="nytimes" /><ref name="tdr">{{cite web |author1=William Glover |url=https://www.newspapers.com/clip/13031762/donald_moffat/|title=He's Still Broke But Has Grown As Actor|website=Newspapers.com |publisher=The Danville Register |accessdate=2018-12-24}}</ref>

== 私人境况 ==
1954年，唐纳德·莫法特与女演员安妮·穆雷·艾利斯波曼（{{lang|en|Anne Murray Ellsperman}}）。<ref name="nytimes" /> 他们生有一个名叫温蒂（{{lang|en|Wendy}}）的女儿和一个名叫加布里埃尔（{{lang|en|Gabriel}}）的儿子。他们的婚姻在1968年宣告结束。<ref name="tdr" /> 1970年，唐纳德与另一个女演员{{Link-en|格温·阿纳|Gwen Arner}}结婚。<ref name="SAE">{{cite web |title=Waiting for Rem |url=https://www.newspapers.com/clip/13032667/donald_moffat/ |website=Newspapers.com |publisher=San Antonio Express |accessdate=2018-12-24}}</ref>

唐纳德·莫法特在2018年12月20日逝世于[[美国|美国]][[纽约州|纽约州]]的[[沉睡谷_(紐約州)|沉睡谷]]，此时距离他88岁生日还差6天。据悉，唐纳德的死因是[[中风|中风]]并发症。在他去世的时候，他的妻子、四个孩子、十个孙辈和三个曾孙辈均陪伴在他身边。<ref name="nytimes" />
