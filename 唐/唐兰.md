'''唐兰'''（{{bd|1901年|1月9日|1979年|1月11日|catIdx=Tang唐}}），原名张佩，曾用名佩兰、景兰，号立厂、立庵，笔名曾鸣。[[浙江省|浙江省]][[嘉兴县|嘉兴县]]人。[[中国|中国]]著名历史学家、文字学家、青铜器专家。

== 生平 ==
幼时家境贫寒，早年师从名医[[陈仲楠|陈仲楠]]学习医学，1920年就学于[[无锡国学专科学校|无锡国学专科学校]]，与[[王蘧常|王蘧常]]和[[吴其昌|吴其昌]]被并称为无锡国专三杰。后在[[东北大学|东北大学]]、[[辅仁大学|辅仁大学]]、[[燕京大学|燕京大学]]等多所高校任教。1936年任[[故宫博物院|故宫博物院]]专门委员。1939年起，[[西南联合大学|西南联合大学]]副教授，1946年起任[[北京大学|北京大学]]教授、文科研究所导师、代理系主任。[[中华人民共和国|中华人民共和国]]成立后，继续担任[[北京大学|北京大学]]教授、中文系代理主任，并历任故宫博物院研究员、学术委员会主任、副院长，[[中国科学院|中国科学院]]历史研究所研究员、学术委员。1979年1月11日病逝于[[北京|北京]]。<ref>{{cite web |url=http://www.dpm.org.cn/shtml/402/@/26592.html |title=存档副本 |accessdate=2015-02-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20150221214750/http://www.dpm.org.cn/shtml/402/%40/26592.html |archivedate=2015-02-21 }}，简介。</ref><ref>[http://blog.sina.com.cn/s/blog_728e58880102ux5v.html] {{Wayback|url=http://blog.sina.com.cn/s/blog_728e58880102ux5v.html |date=20160810025303 }}，幼子访谈。</ref>


唐兰在[[古文字学|古文字学]]、[[音韵学|音韵学]]等领域建树颇多。一生著作甚丰，出版和发表《殷墟文字记》《中国文字学》《古文字学导论》等专著以及学术论文180多篇。1953年至1972年，唐兰先生数次将收藏铜器等文物捐献故宫博物院。 [[胡适|胡适]]在日记中提及唐兰曾在1947向其自荐入选[[中央研究院|中央研究院]]院士。


== 唐兰故居 ==
[[唐兰故居|唐兰故居]]，位于嘉兴市秀城区秀水兜106号，是一幢砖木结构的两层楼房，唐兰在这里度过了少年时代。由于年轻时放弃继承权，现在由嘉兴文保部门维护。
