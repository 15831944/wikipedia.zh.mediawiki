'''唐睿宗李旦'''（{{bd|662年|6月22日|716年|7月13日|catIdx=L}}），[[唐高宗|唐高宗]][[李治|李治]]和[[武則天|武則天]]之子，[[唐朝|唐朝]]的第五和第八任[[皇帝|皇帝]]，曾用名'''李旭轮'''、'''李轮'''，他一生兩繼[[大統|大統]]，兩度[[禪位|禪位]]。两次登基，第一次為[[天后|天后]]武氏（登基前的武則天）廢[[唐中宗|唐中宗]][[李顯|李顯]]而繼位，在位时间是文明元年至載初二年（684年2月27日－690年10月16日），後上表自行退位，禪讓予母親武則天；第二次是在[[唐隆之變|唐隆之變]]誅除[[韋皇后|韋皇后]]及其黨羽後復辟，在位时间是景雲元年至延和元年（710年7月25日-712年9月8日），後退位，[[內禪|內禪]]予其子[[李隆基|李隆基]]（[[唐玄宗|唐玄宗]]）。李旦為[[唐高宗|唐高宗]][[李治|李治]]諸子之中排行第八，母为[[武则天|武则天]]，[[李弘_(唐朝)|李弘]]、[[章懷太子|李贤]]、[[唐中宗|唐中宗]][[李顯|李顯]]都是其同母兄长，[[太平公主|太平公主]]則為其同母妹妹。

==生平==
[[龍朔|龍朔]]二年（662年）六月一日生於[[長安|長安]]蓬萊宮含涼殿<ref>《酉陽雜俎》：「睿宗初生含涼殿，則天乃於殿內造佛氏。」</ref>。当年，封殷王，[[遥领|遥领]]冀州[[大都督|大都督]]、单于[[大都护|大都护]]、右金吾卫[[大將軍_(中國)|大将军]]。史載“謙恭孝友，好學，工草隸，尤愛文字[[訓詁|訓詁]]之書”。[[乾封|乾封]]元年（666年），徙封豫王。[[總章|总章]]二年（669年），徙封冀王。[[上元_(唐高宗)|上元]]二年（675年），徙封相王，拜右卫大将军。根据史料显示，仪凤元年（676年），十四岁的李轮即已纳[[豆卢贵妃|豆卢氏]]为[[孺人|孺人]]（[[妾室|妾室]]）。儀鳳三年（678年），迁洛[[牧|牧]]；改名李旦，徙封豫王。在七月的一次宴会上，父亲[[唐高宗|唐高宗]]对叔祖父霍王[[李元軌|李元轨]]说：因为他是最小的儿子，所以最为喜爱<ref>《[[旧唐书|旧唐书]]·本纪第五·高宗下》（仪凤）三年......秋七月丁巳，宴近臣诸亲于咸亨殿。上谓霍王元轨曰：“......又男轮最小，特所留爱，......”</ref>。679年，王妃[[肅明皇后|刘氏]]生下他的长子[[李憲_(寧王)|李成器]]。

[[嗣聖|嗣聖]]元年（684年）二月七日，[[武则天|武则天]]废其兄[[唐中宗|中宗]]帝位，立他为帝，改元[[文明_(唐睿宗)|文明]]。不过，由於是武则天[[執政|執政]]，“政事決于太后”<ref>《資治通鑒》卷二○三后光宅元年</ref>，睿宗毫无实权，甚至連干预国家大政的权力都沒有，淪為[[傀儡|傀儡]]。[[载初_(唐朝)|載初]]元年（690年），武则天废除睿宗後自登[[帝位|帝位]]，改國號[[武周|周]]，睿宗被貶为「皇嗣」（候補性質的皇位繼承人。具儀一比[[皇太子|皇太子]]，卻不給皇太子的名分），改名'''武轮'''，迁居东宫。

武则天[[聖曆|聖曆]]元年（698年），武則天又改立[[唐中宗|中宗]]為[[儲君|儲君]]。睿宗則從「皇嗣」再被貶為[[親王|親王]]，封號'''相王'''，他的五個兒子（[[李成器|李成器]]、[[李成義|李成義]]、[[李隆基|李隆基]]、[[李隆範|李隆範]]、[[李隆業|李隆業]]）被封為[[郡王|郡王]]，唐睿宗從此重獲自由，擁有干预国家大政的权力。

[[神龍|神龍]]元年（705年），[[宰相|宰相]][[張柬之|張柬之]]等五人發動[[神龍革命|神龍革命]]，殺[[张易之|张易之]]、[[张昌宗|张昌宗]]兄弟，逼武則天[[退位|退位]]，迎中宗[[復辟|復辟]]，不久武则天去世。此後中宗封其為'''安國相王'''，隨即辭去。[[景雲|景雲]]元年（710年），中宗駕崩（傳說是被[[韋皇后_(唐中宗)|韋皇后]]毒杀），（後在韋皇后矯詔下）由中宗幼子[[李重茂|李重茂]]登位，[[改元|改元]][[唐隆|唐隆]]，是為少帝。

睿宗的三子[[唐玄宗|李隆基]]與[[太平公主|太平公主]]等聯絡[[禁軍|禁軍]][[將領|將領]]，擁兵入宮，將韦-{后}-誅殺，迫少帝[[李重茂|李重茂]]遜位，史曰[[唐隆之變|唐隆之變]]。六月二十四日睿宗[[復辟|復辟]]於承天門樓，[[大赦|大赦]][[天下|天下]]，与其子李隆基一起铲除了[[韋皇后_(唐中宗)|韋皇后]]一黨的势力。

[[延和|延和]]元年（712年）七月二十五日，唐睿宗無法面對[[唐玄宗|李隆基]]與[[太平公主|太平公主]]的爭端，於是[[禪讓|禪讓]][[皇帝|帝位]]於李隆基，是為唐玄宗，自称「[[太上皇|太上皇]]」，每五天在太極殿接受群臣的朝賀，仍自称“朕”，三品已上除授及大刑狱仍然自决，命令称诰、令，而玄宗每日受朝于武德殿，自称“予”，决定三品已下除授及徒罪，命令称制、敕。后来玄宗发动[[先天政变|先天政变]]消灭太平公主一党，睿宗才不得不彻底放权。

[[開元|開元]]四年六月二十日（716年7月13日）李旦病逝，享年五十五歲。

=== 女 ===
《新唐书·诸帝公主列传》，记载唐睿宗共有11位女儿<ref>[http://guoxue.baidu.com/page/d0c2ccc6cae9/95.html 《新唐书·诸帝公主列传》]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。具体的排行，出自对个人记载：

# 长女，[[寿昌公主_(唐睿宗)|壽昌公主]]（母[[肅明皇后|刘皇后]]，下嫁[[崔真|崔真]]）
# 次女，[[安興昭懷公主|安興昭懷公主]]（早薨）
# 某女，[[荊山公主|荊山公主]]（下嫁[[薛伯阳|薛伯阳]]，存疑，可能是凉国公主、鄎国公主两个公主的混记）
# 三女，[[李花山|淮陽公主]]（名花山，母王德妃，下嫁[[王承庆|王承庆]]）
# 四女，[[李华_(公主)|代國公主]]（名华，字華婉（墓誌銘記為花婉），母刘皇后，下嫁[[郑万钧|郑万钧]]）
# 五女，[[涼國公主|涼國公主]]（字華莊，母王德妃，先封为仙源公主，下嫁[[薛伯阳|薛伯阳]]）
# 某女，[[薛國公主|薛國公主]]（先封为清阳公主，下嫁[[王守一|王守一]]，又嫁[[裴巽|裴巽]]）
# 七女，[[鄎國公主|鄎國公主]]（母[[崔贵妃_(唐睿宗)|崔贵妃]]，先封为荆山公主，下嫁[[薛儆|薛儆]]，又嫁[[郑孝义|郑孝义]]）
# 八女，[[金仙公主|金仙公主]]（母窦德妃，先封为西城县主）
# 九女，[[玉真公主|玉真公主]]（字玄玄，号持盈，母窦德妃，始封崇昌县主）

# {{note|1}}从684年－690年唐睿宗仍然在位期间，还有[[光宅|光宅]]、[[垂拱|垂拱]]、[[永昌_(唐朝)|永昌]]、[[载初_(唐朝)|载初]]几个年号。虽然有文献<ref>陈光，中国历代帝王年号手册，北京燕山出版社 第225-226页 ISBN 7540210311</ref>把这几个年号算作唐睿宗的，但是这段期间，武则天实际操纵朝政，睿宗毫无实权。因此大多文献把这几个年号算在武则天名下<ref>徐红岚，中日朝三国历史纪年表，辽宁教育出版社，1998年 第150页 ISBN 7538276193</ref><ref>李崇智，中国历代年号考，中华书局，2004年 第100页 ISBN 7101025129</ref>。
