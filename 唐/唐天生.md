'''唐天生'''{{BD|1962年|5月|2016年|4月3日|catIdx=T唐天生}}[[广西人|广西人]]，毕业于广西壮族自治区党校，[[中国共产党|中国共产党]]党员，曾先后担任过[[桂林市|桂林市]]机关事务管理局局长，[[桂林市|桂林市]][[秀峰区|秀峰区]]人民政府区长，[[阳朔县|阳朔县]]人民政府县长，中共龙胜各族自治县县委书记。2013年8月13日，唐天生因“公款吃喝”受到“党内严重警告”处分，并且免去中共[[龙胜县|龙胜县]]委书记职务。不过，时隔一年之后，唐天生复出，担任广西桂林市食品药品监督管理局局长，因“公款吃喝”被免职的官员复出之后“管吃喝”，在网络上遭到网友的揶揄和讽刺。<ref>{{cite web |url=http://henan.china.com.cn/renwu/2014/1204/128718.shtml |title=广西县委书记唐天生公款吃喝被免 1年后复出当食药监局局长 |accessdate=2014 |work=中国网 |publisher= |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160415181337/http://henan.china.com.cn/renwu/2014/1204/128718.shtml |archivedate=2016-04-15 }}</ref><ref>{{cite web |url=http://junshi.xilu.com/news/tangtiansheng.html |title= 唐天生 原广西县委书记因吃喝被免 1年复出管吃喝|accessdate=2014 |work= 西陆军事|publisher=  |language=zh-hans}}</ref>

==生平==
1962年，唐天生出生在[[广西壮族自治区|广西壮族自治区]][[玉林市|玉林市]]。1979年12月参加工作，曾陆续担任昆明部队战士、班长、管理员。1983年6月，唐天生加入[[中国共产党|中国共产党]]，之后担任广西[[桂林市|桂林市]]汽车运输公司司机、党委组织干事。1986年，唐天生进入广西桂林市政府办公室，1994年调入 广西桂林市接待办。1997年，唐天生担任广西桂林市机关事务管理局党组副书记、副局长，2002年升任中共广西[[桂林市|桂林市]][[秀峰区|秀峰区]]委副书记、桂林市秀峰区人民政府区长。2005年到2009年，唐天生担任中共广西[[阳朔县|阳朔县]]委副书记、阳朔县人民政府县长。2009年到2013年，唐天生担任中共广西龙胜各族自治县委员会委员、常委、书记，期间，他因“公款吃喝”受到“党内严重警告”处分，并且被免职。2014年3月。唐天生复出，担任桂林市食品与药品监督管理党组书记，2014年10月，又兼任该局局长。<ref>{{cite web |url=http://gs.people.com.cn/n/2014/1205/c188868-23123452.html |title=公款吃喝被免复出，名叫唐天生就“天生”当官？ |accessdate=2014 |work=人民网 |publisher=  |language=zh-hans}}</ref>

2016年4月3日，唐天生在醫院墮樓身亡。<ref>[http://news.mingpao.com/pns/dailynews/web_tc/article/20160404/s00013/1459706533865 桂林食藥監局長墮斃 曾因公款吃喝免職]</ref>
