'''唐士維公園站'''（{{lang-en|Downsview Park Station}}）是一座[[多倫多地鐵|多倫多地鐵]][[央街－大學－士巴丹拿線|央街－大學－士巴丹拿線]]車站，坐落[[加拿大|加拿大]][[安大略省|安大略省]][[多倫多|多倫多]][[北約克|北約克]]雪柏大道西（{{lang|en|Sheppard Avenue West}}）和[[GO運輸|GO通勤鐵路]]巴里線（{{lang|en|Barrie line}}）的交界處，以鄰近的唐士維公園命名，於2017年12月17日聯同該地鐵線的士巴丹拿延線項目其他車站一起開幕<ref name="open">{{cite news | last = Patton| first = Jessica| title = New Toronto-York Spadina Subway Extension officially launches| publisher = Global News| date = 2017-12-17| url = https://globalnews.ca/news/3921542/new-toronto-york-spadina-subway-extension-officially-launches/| accessdate =  2017-12-18}}</ref>。

GO運輸公司亦將於此處為巴里線設立一個新車站，讓乘客在巴里通勤鐵路線和士巴丹拿地鐵線之間轉車<ref>{{cite web |url=http://www3.ttc.ca/PDF/About_the_TTC/TYSSE/_Sheppard_West_Display_Boards_Open_House_REV.pdf |title=Sheppard West Subway Station Open House |author= |date=2010-07-08 |work=Toronto-York Spadina Subway Extension |publisher=Toronto Transit Commission |accessdate=2013-02-02}}</ref>。巴里線的唐士維公園站啓用後，該線現有的約克大學車站將停止運作<ref>{{cite web |url=http://www.metrolinx.com/en/docs/pdf/board_agenda/20100519/GO_BCA_interim_report_May2010.pdf |title=GO Rail Options Benefits Case Assessment |date=2010-05 |work=Metrolinx |accessdate=2013-02-02}}</ref>。

此站於規劃和工程階段名為雪柏西站（{{lang|en|Sheppard West Station}}），但[[多倫多公車局|多倫多公車局]]（TTC）於2010年通過將此站命名為唐士維公園站<ref>{{cite web |url=http://www3.ttc.ca/About_the_TTC/Commission_reports_and_information/Commission_meetings/2010/Sept_30_2010/Supplementary_Reports/Toronto_York_Spadina.pdf |title=Toronto-York Spadina Subway Extension, Station Names |date=2010-09-30 |publisher=Toronto Transit Commission |accessdate=2013-02-02}}</ref>。另外，此站以南的下行車站原名為唐士維站（{{lang|en|Downsview Station}}）；為了避免乘客混淆車站名稱，唐士維站於2017年5月7日改稱[[雪柏西站|雪柏西站]]<ref>{{cite news|url=http://www.thestar.com/news/gta/transportation/article/1263726--ttc-names-new-subway-station-for-black-creek-pioneer-village |title=TTC names new subway station for Black Creek Pioneer Village|last=Kalinowski|first=Tess|publisher=Toronto Star |date=2012-09-28|accessdate=2013-02-01}}</ref>。

下列由TTC營運的巴士線駛經唐士維公園站<ref>{{cite web|url=http://ttc.ca/Spadina/About_the_Project/Bus_Route_Changes.jsp|title=Toronto-York Spadina Subway Extension: Bus Route Changes|publisher=TTC|accessdate=2017-12-18}}</ref>：
*84號雪柏西巴士線（{{lang|en|Sheppard West}}）
*101號唐士維公園巴士線（{{lang|en|Downsview Park}}）
*106號約克大學巴士線（{{lang|en|York University}}）
*107號北基爾街巴士線（{{lang|en|Keele North}}）
*108號唐士維巴士線（{{lang|en|Downsview}}）
