'''唐鳳'''（{{lang-en|Audrey Tang}}，{{bd|1981年|4月18日|||T唐}}），原名'''唐宗漢'''，[[臺灣外省人|臺灣外省人]]第三代<ref name="蔡雅寶">蔡雅寶：鹿港到眷村的歲月 https://www.slideshare.net/autang/ss-56488466</ref>{{dubious}}，父親[[唐光華|唐光華]]曾任[[中國時報|中國時報]]副[[總編輯|總編輯]]及[[陳履安|陳履安]]競選[[總統|總統]][[發言人|發言人]]，母親[[李雅卿|李雅卿]]為[[商業周刊_(臺灣)|商業周刊]]前副總編。[[2016年|2016年]]唐鳳出任[[中華民國行政院|中華民國行政院]][[無黨籍|無黨籍]][[政務委員_(中華民國)|政務委員]]，外媒以「全球首位[[跨性別|跨性別]][[部長|部長]]」介紹她，也是中華民國第一位[[數位|數位]]政委。

唐於政委任內數次以[[機器人|機器人]][[視訊|視訊]][[直播|直播]]、影音屏幕等科技，成功代表[[中華民國|中華民國]]參與[[聯合國|聯合國]]歐洲總部的[[網路|網路]][[治理|治理]]國際會議<ref>中國時報：外交突圍! 數位機器人直播發功 唐鳳「登上」聯合國會議發言（中國時報） http://www.chinatimes.com/realtimenews/20171223001006-260408</ref>、[[聯合國人居署|聯合國人居署]]於[[吉隆坡|吉隆坡]]舉行的[[世界城市論壇|世界城市論壇]]<ref>TVBS：UN世界城市論壇　唐鳳影片分享台灣社創 https://news.tvbs.com.tw/life/869748</ref>、[[聯合國亞洲及太平洋經濟社會委員會|聯合國亞太經社會]]於[[首爾|首爾]]舉行的亞太影響力投資工作坊<ref>上報：沒意願參選北市長　唐鳳要推突破中國阻力的「數位外交」 http://www.upmedia.mg/news_info.php?SerialNo=37817?=fb</ref>，並親自參與聯合國[[永續發展|永續發展]]網絡在[[教廷|教廷]]舉辦的[[人工智慧|人工智慧]]研討會<ref>聯合新聞網：數位外交 唐鳳赴教廷談台灣民主 https://udn.com/news/story/6656/3025627</ref>等。唐鳳認為，中華民國要走一條「暖實力」[[外交|外交]]（Warm Power）的路。<ref>PDIS：2018-06-20 第四次記者茶敘 https://sayit.pdis.nat.gov.tw/2018-06-20-%E7%AC%AC%E5%9B%9B%E6%AC%A1%E8%A8%98%E8%80%85%E8%8C%B6%E6%95%98</ref>

唐鳳原為[[自由軟體|自由軟體]][[程式設計師|程式設計師]]，在[[臺灣|中華民國]]曾被稱為「台灣電腦十大高手」之一<ref name="陳俊名">{{Cite web|url=http://www.ettoday.com/2006/02/08/327-1902964.htm |title=別叫我「先生」！　電腦怪傑唐宗漢變性　改名唐鳳 |author=陳俊名和江志男 |date=2006年2月8日 |publisher=[[ETtoday_東森新聞雲|ETtoday 東森新聞雲]] |language=zh-hant |accessdate=2016年8月28日 |quote= |deadurl=bot: unknown |archiveurl=https://web.archive.org/web/20061019113333/http://www.ettoday.com/2006/02/08/327-1902964.htm |archivedate=2006年10月19日 }}</ref>，曾致力於多項專案的開發維護，是[[Haskell|Haskell]]和[[Perl|Perl]]社群領導者，也是[[g0v零時政府|g0v零時政府]]核心成員。唐鳳早年不習慣傳統正規教育，14歲起便以在家自學方式學習。過去曾擔任[[明基電通|明基電通]]、[[蘋果公司|蘋果公司]]等公司顧問，並長期參與[[開放原始碼|開放原始碼]]社群協作、主導vTaiwan等平臺走向。

在2014年至2015年間擔任行政院虛擬世界法規調適計劃顧問，也是[[中華民國國家發展委員會|中華民國國家發展委員會]]開放資料諮詢委員會和[[十二年國民基本教育|十二年國民基本教育]]課程發展委員會委員，同時與[[法國外交部|法國外交部]]、[[法國經濟財政部|法國經濟財政部]]、[[巴黎|巴黎市政府]]、西班牙[[馬德里|馬德里市政府]]在數位治理領域合作。2016年8月，[[行政院院長|行政院院長]][[林全|林全]]任命曾有網路創業經驗的唐鳳擔任[[行政院|行政院]][[政務委員_(中華民國)|政務委員]]，負責督導數位經濟與開放政府發展，亦是首位[[跨性別女性|跨性別女性]]擔任行政院政務委員。她則表示在進入行政院後，將透過數位技術和系統輔助公務體系解決問題，並強化政府部門與公民科技、公共社群的對話與合作。

=== 遭到霸凌 ===
到了小學時期，唐鳳因為對瑣事的記性差，常因忘記攜帶手帕和衛生紙而遭老師體罰<ref name = "陳柏年"/><ref name = "成為公僕的公僕"/>。不過由於先天聰穎，在學習或思想上均明顯超越同年齡學生，且因強烈求知慾望而讓她在課堂上總會提出許多問題<ref name = "古美蓮"/>，因而曾在小學一年級直接於數學課被帶離<ref name = "王韻齡"/>。在必須顧及大多數學生利益的前提下，唐鳳被特別允許在上課期間，到學校圖書館看書<ref name = "古美蓮"/>。但很快看完圖書館書籍後，她開始感到無聊並不想上課，校長在學校沒有多餘人力照顧下，建議轉往資優班就讀<ref name = "古美蓮"/>。不過在競爭激烈的資優班裡，唐鳳雖然學業成績表現傑出，但也讓她成為同學排擠對象<ref name = "古美蓮"/>。小學二年級時，她便遭到忌妒的資優班同學圍毆<ref name = "成為公僕的公僕"/><ref name = "陳柏年"/><ref name = "上學被圍毆想自殺"/>，亦有同學直接以椅子攻擊腹部<ref name = "古美蓮"/>。

隨著問題越趨嚴重，這些經驗讓唐鳳罹患厭惡上學的[[學校恐懼症|學校恐懼症]]，終日神經緊張、夜晚哭泣與惡夢，且數次透露自殺念頭<ref name = "成為公僕的公僕"/><ref name = "陳柏年"/><ref name = "上學被圍毆想自殺"/>。逐漸崩潰的她每天回家便脫掉所有衣服，整天把自己關在房間讀書、哭泣和發呆<ref name = "李盈穎"/>，全家因而大為緊繃<ref name = "李盈穎"/>。因其無法獲得歸屬，李雅卿必須每晚親自抱著她睡覺，讓她能夠在醒來後立刻獲得安全感<ref name = "古美蓮"/>，弟弟唐宗浩也讓她願意繼續與人相處並信任陌生人<ref name = "王韻齡"/>。從小在體制內適應良好的李雅卿，最早仍鼓勵孩子克服上學問題<ref name = "古美蓮"/>。但唐鳳因不願作弊而被班上同學踢昏後，見到孩子腹部瘀青的李雅卿決定把孩子留在家裡自己教育<ref name = "成為公僕的公僕"/>，在學校放棄處理的情況下從資優班辦理休學<ref name = "陳柏年"/>。

但這舉動最初也被家人質疑和不解，唐光華便不贊成孩子在家自學，孩子教育問題導致夫妻失和且父子關係決裂，唐光華後來決定留學德國<ref name = "陳柏年"/>。向來疼愛唐鳳的的祖母也擔心休學後，將導致她日後無法與社會銜接<ref name = "陳柏年"/>。在小學三年級時，唐鳳曾一度休學、之後繼續復學就讀，並在小學6年期間更換6所小學<ref name = "古美蓮"/>。面對種種責難，李雅卿認為自己不是寵愛孩子，而是不滿教育系統讓孩子受苦<ref name = "陳柏年"/>。在承受種種壓力下，李雅卿繼續尋找不同教育方法，並獲得[[臺北市文山區指南國民小學|臺北市文山區指南國民小學]]校長簡志雄、[[國立臺北師範學院|國立臺北師範學院]]講師楊文貴<ref name = "古美蓮"/>、[[國立台灣大學|國立台灣大學]]數學系教授朱建正、毛毛蟲兒童哲學基金會楊茂秀教授、陳鴻銘老師等人幫助<ref name = "陳柏年"/>。

其中在暑假期間，楊文貴每天帶著唐鳳外出遊玩，2個月後設計出新的教學方法<ref name = "古美蓮"/>。在楊文貴建議與安排下，唐鳳在1周當中有3天時間在[[新北市新店區直潭國民小學|新北市新店區直潭國民小學]]上課，學習與同學間相處的人際關係<ref name = "古美蓮"/>。另外3天則前往國立臺北師範學院滿足求知慾望，課程除了安排知識性內容外，也有打[[合約橋牌|合約橋牌]]、逛[[中華商場|中華商場]]、追求女朋友等<ref name = "古美蓮"/>。另外朱建正在指導期間則提供大量科幻小說，包括以數學或科學理論發展出來的[[硬科幻|硬科幻]]作品，讓當時休學在家的唐鳳會因好奇心而研究更高深數學理論<ref name = "王韻齡"/>。在唐鳳上課期間，李雅卿便帶著小兒子唐宗浩到處遊玩，並在下課後共同回家<ref name = "古美蓮"/>。

=== 短暫出國 ===

隨著唐鳳學習情況趨於安穩，她與父親的關係也逐漸轉好<ref name = "陳柏年"/>；但因弟弟唐宗浩同樣缺乏適當的教育人選<ref name = "古美蓮"/>，李雅卿在其他師長建議下，於1991年帶著兩個孩子前往德國<ref name = "陳柏年"/>。除了與唐光華共同生活外，也讓孩子見識到不同資優教育環境<ref name = "古美蓮"/><ref name = "陳柏年"/>。李雅卿短暫讓孩子在德國唸書，唐光華也帶着唐鳳共同研讀政治學，為後者往後發展帶來一定影響<ref name = "端傳媒"/>。唐光華在德國攻讀博士學位時，因曾在1989年前往中華人民共和國[[北京市|北京市]][[天安門廣場|天安門廣場]]採訪[[六四事件|學生運動]]，六四事件也給他帶來許多影響，因而挑選為研究主題<ref name = "Claire Richard"/>。而唐鳳家中客廳也因而常有流亡學生運動知識分子聚會，討論民主進程等議題<ref name = "Claire Richard"/>，唐鳳也因此在自我介紹中總會提到 「在天安門流亡者之間成長」<ref name = "端傳媒"/>。不過當唐鳳取得當地最好一所中學的入學許可、並獲得跟隨華裔資優學者前往美國學習的機會，年僅11歲的她受德國教育環境影響，堅持返回臺灣就讀<ref name = "陳柏年"/>，並立志改革臺灣教育體系<ref name = "管婺媛">{{Cite web |url = http://udn.com/news/story/1/1917750 |title = 最年輕政委！網路神童、國際駭客唐鳳將入閣 |author = 管婺媛 |date = 2016年8月25日 |publisher = [[聯合新聞網|聯合新聞網]] |language = zh-hant |accessdate = 2016年8月28日 |quote =  }}</ref>。

受到前往德國1年影響，李雅卿則反思德國教育和臺灣教育不同處<ref name = "成為公僕的公僕"/>。一家人在1994年返回臺灣後，已經成長的唐鳳知道調適自己以配合制式教育環境，除了不會在學校引起爭端和造成老師困擾外，並能協助使用電腦與計算成績，與同學間的相處也更為融洽，同時還盡力協助弟弟解決相同教育困擾<ref name = "古美蓮"/>。但由於唐宗浩陸續發生與教育體制衝突、適應不良的問題<ref name = "陳柏年"/><ref name = "成為公僕的公僕"/>，李雅卿為此繼續深入研究另類教育方法、教育改革和實驗教育<ref name = "端傳媒"/>；除了主持毛毛蟲兒童哲學基金會討論教育課程，她還創辦國內第一個以學生為主導的[[臺北縣|臺北縣]][[種籽學苑|種籽親子實驗國小]]<ref name = "管婺媛"/>，並成為[[臺北市自主學習實驗計畫|臺北市自主學習實驗計畫]]主持人<ref name = "陳柏年"/><ref name = "成為公僕的公僕"/>，促成往後推動[[臺灣教育改革|臺灣教育改革]]的訴求<ref name = "王韻齡"/>。其中李雅卿主張應給予學生完全的學習自主權，培養孩子完全獨立思考的能力<ref name = "成為公僕的公僕"/>。

唐鳳在國中一年級時，仍希望能夠進入大學就讀<ref name = "古美蓮"/>。憑藉著自己設計的電腦程式，她贏得全國中小學科學展覽第一名、和保送第一志願[[台北市立建國高級中學|台北市立建國高級中學]]資格<ref name = "王韻齡"/>。不過在進入國中就讀後不久，[[提姆·柏內茲-李|提姆·柏內茲-李]]團隊也於瑞士發明[[全球資訊網|全球資訊網]]<ref name = "Claire Richard"/>，在認識到網際網路世界與全新交流方法後，唐鳳發現人生或學習方式存在許多可能性<ref name = "端傳媒"/>，她也能直接向研究人員學習想學習的東西<ref name = "Claire Richard"/>。由於許多研究人員樂意與唐鳳聯繫並提供解答，網際網路上的最新知識讓她發現課本內容過時<ref name = "Claire Richard"/>，也對系統教育的想法有所轉變<ref name = "古美蓮"/>。在諮詢校長和家人後，雖然態度保守的唐光華認為應繼續升學，獲得李雅卿支持的唐鳳最後決定輟學<ref name = "Claire Richard"/>。之後她離開傳統學校教育而在家自學<ref name = "端傳媒"/>，並由李雅卿簽署放棄保送資格的家長同意書<ref name = "古美蓮"/>。不過也因選擇在家自學，之後也未就讀高級中學、大專院校<ref name = "王韻齡"/>，讓她最高學歷只有國中肄業<ref name = "成為公僕的公僕"/><ref name = "張童恩">{{Cite web |url = http://n.yam.com/yam/politics/20160825/20160825855420.html |title = 台灣首例！ 蔡政府延攬跨性別人士入閣 |author = 張童恩 |date = 2016年8月25日 |publisher = [[天空傳媒|天空傳媒]] |language = zh-hant |accessdate = 2016年8月28日 |quote =  }}</ref>。

== 軟體工作 ==

在家自學期間，唐鳳除了繼續研究電腦軟體，也閱讀《[[易經|易經]]》、[[伯特蘭·羅素|伯特蘭·羅素]]、[[弗里德里希·尼采|弗里德里希·尼采]]、[[柏拉圖|柏拉圖]]、[[威廉·莎士比亞|威廉·莎士比亞]]、[[T·S·艾略特|T·S·艾略特]]等著作<ref name = "古美蓮"/><ref name = "楊沛文"/>。而唐鳳先是與就讀[[國立臺灣大學|國立臺灣大學]]商學研究所的[[賀元|賀元]]等人共同出版《我的電腦探索》，爾後在賀元號召下，結合[[國立政治大學|國立政治大學]]企業管理研究所學生[[薛曉嵐|薛曉嵐]]和唐鳳3人，集資新台幣1,000萬元，在1995年1月成立資訊人文化事業公司<ref name = "Claire Richard"/>，初期以出版電腦書籍為主<ref name = "古美蓮"/>，唐鳳也成為資訊人最年輕的合夥股東<ref name = "楊沛文"/>。隔年隨著網際網路興起，這群年輕人開始朝向網際網路發展，唐鳳成為主要執行者<ref name = "古美蓮"/>。大約花費1周時間，唐鳳自行創作出能在網際網路搜尋資訊的應用軟體「搜尋快手」<ref name = "楊沛文"/><ref>{{Cite web |url = http://www.businessweekly.com.tw/KArticle.aspx?id=4071 |title = 網際網路賺錢秘笈大公開 |author = 古美蓮 |date = 2004年1月8日 |publisher = 《[[商業周刊_(臺灣)|商業周刊]]》 |language = zh-hant |accessdate = 2016年8月28日 |quote =  }}</ref>，使用者只需輸入簡單的中英文字母便可找到所需資訊<ref name = "古美蓮"/>。該套搜尋軟體在1996年12月資訊月展覽會場推出後，立刻受到電腦族群關注，不到1個月便銷售1萬多套<ref name = "古美蓮"/><ref name = "楊沛文"/>。

也因為這套軟體的成功，16歲的唐鳳基於個人興趣很早投入實務工作<ref name = "戴雅真">{{Cite web |url = http://www.cna.com.tw/news/firstnews/201608250110-1.aspx |title = 唐鳳任政務委員 督導數位經濟與開放政府 |author = 戴雅真 |date = 2016年8月25日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2016年8月28日 |quote =  }}</ref>，成為網際網路創業家<ref name = "成為公僕的公僕"/>；資訊人也得以對外招聘12名員工，員工平均年齡在25歲至26歲間<ref name = "古美蓮"/>。在臺灣尚未高度發展軟體產業時，發展迅速的資訊人持續研發中文搜尋引擎；此舉除了讓[[旺宏電子|旺宏電子]]董事長[[胡定華|胡定華]]、[[宏碁|宏碁]]創業元老[[黃少華|黃少華]]、[[英業達集團|英業達集團]]主管等資訊圈著名人士願意擔任顧問協助外<ref name = "古美蓮"/>，甚至讓[[英特爾|英特爾]]投資這家備受關注的電腦軟體公司而成為股東<ref name = "Claire Richard"/>。當時整家公司都以開放原始碼的程式語言[[Perl|Perl]]編寫而成，程式設計師間經常協作並相互支持，公司成員也已經設想將協作方式轉移到社會互動上<ref name = "Claire Richard"/>。這時唐鳳也成為世界知名的網際網路駭客，並協助翻譯程式設計書籍為中文<ref name = "Claire Richard"/>，也曾被商業雜誌列為封面人物<ref name = "成為公僕的公僕"/>，另外她還到[[宏碁|宏碁]]總部向董事長[[施振榮|施振榮]]做非正式簡報<ref name = "楊沛文"/>。

唐鳳之後成為網際網路公司資訊人的技術總監<ref name = "李欣茹"/><ref name = "楊沛文"/>，也與過去不斷在觀念上發生衝突的唐光華溝通，而能互相理解<ref name = "古美蓮"/>，之後她前往美國[[加利福尼亞州|加利福尼亞州]][[矽谷|矽谷]]創立公司<ref name = "李欣茹"/>，曾與[[日月光半導體製造股份有限公司|日月光半導體製造股份有限公司]]總經理[[劉英武|劉英武]]在早餐會報時談及腦神經化學的發展，也曾擔任過明碁電腦<ref name = "楊沛文"/>、[[明基電通|明基電通]]<ref name = "上報快訊"/>、[[逐鹿網|逐鹿網]]等公司顧問<ref name = "陳思翰">{{Cite web |url = http://www.ithome.com.tw/node/72594 |title = 將程式設計視為藝術創作的開源人 |author = 陳思翰 |date = 2002年3月9日 |publisher = iThome |language = zh-hant |accessdate = 2016年8月28日 |quote =  }}</ref>。唐鳳離開資訊人並轉任顧問後，開始研讀哲學、社會學和文學，希望理解網際網路虛擬空間出現的各種複雜行為<ref name = "Claire Richard"/>，之後她多次創辦公司與販售公司<ref name = "端傳媒"/>。另外她也邀集高嘉良、[[何鎧伶|何鎧伶]]、[[鄭智中|鄭智中]]、[[翁千婷|翁千婷]]等軟體界同好共同創立傲爾網公司，主要提供開放原始碼的自由軟體服務<ref name = "李欣茹"/>，並由她擔任總經理<ref name = "楊沛文"/>。唐鳳說服明碁電腦總經理[[李焜耀|李焜耀]]、傲爾網董事長[[李志華|李志華]]、逐鹿網總經理[[陳榮宏|陳榮宏]]等人募集到新台幣2,500萬元資本額，新公司創辦還特別獲得李焜耀支持<ref>{{Cite web |url = http://www.businessweekly.com.tw/KArticle.aspx?id=11132 |title = 科技神童，創業神話？ |author = 劉志明 |date = 2003年6月25日 |publisher = 《[[商業周刊_(臺灣)|商業周刊]]》 |language = zh-hant |accessdate = 2016年8月28日 |quote =  }}</ref>，視為公司秘密武器<ref name = "楊沛文"/>。

其中傲爾網透過自己在網際網路上成立的Elixir社群，由傲爾網負責技術整合並提供企業解決方案，從分析、建議、諮詢、維修、研發等層面，以次數或時數等方式收取費用<ref name = "楊沛文"/>。透過推廣開放原始碼運動，傲爾網一開始取得碩網資訊等3家客戶<ref name = "楊沛文"/>。唐鳳也投入開發全球已有大量人使用的追蹤管理系統Request Tracker，其當有25%程式碼是由其寫出<ref name = "李欣茹"/>。而在[[CPAN|CPAN]]年度會議中，唐鳳還是亞洲唯一一名受邀討論的開發者<ref name = "李欣茹"/>。2014年，33歲的唐鳳完成Socialtext公司和[[蘋果公司|蘋果公司]]面試新人、交接工作等任務後，決定退休<ref name = "塗鵬揚"/>。原本在公司擔任員工的她返回臺灣後，轉而繼續擔任蘋果公司等顧問<ref name = "塗鵬揚"/>，大部分採取遠端工作的模式<ref name = "Claire Richard"/>。不過她開始將更多心力投入在臺灣公共領域和公民科技社群上<ref name = "端傳媒"/>，參與的專案也與政治有著密切關係<ref name = "Claire Richard"/>。而在擔任蘋果公司數位顧問期間<ref name = "王韻齡"/>，她與蘋果公司約定為公司工作1個小時，蘋果公司就必須支付1塊[[比特幣|比特幣]]（相當於新台幣18,000元）<ref name = "塗鵬揚"/><ref name = "張童恩"/>，因而被視為新創界的傳說<ref name = "管婺媛"/>。

== 開放社群 ==
=== 社群發展 ===
[[File:20151013_成大開源月-唐鳳_PA130714_(22007734868).jpg|thumb]]開源月中演講。]]

唐鳳長期推動開放原始碼社群的協作<ref name = "上報快訊"/>，在開放原始碼領域具有一定知名度<ref name = "陳思翰"/>。由於在家自學階段，她在網際網路上總能獲得其他研究者熱情回應，同時不隱藏、無償的開放原始碼精神也幫助她許多，這讓她覺得自己必須對網際網路有所回應<ref name = "Claire Richard"/>，因而投入開放原始碼社群的活動<ref name = "端傳媒"/>。唐鳳認為軟體業界過往全數包攬服務的經濟模式，實際上屬於資本主義的產物，但因為公司會限制受雇者開發軟體的能力，實際上該模式並不經濟<ref name = "楊沛文"/>。相對地，如果轉而採取開放原始碼的概念執行專案的話，就能夠以類似「外包」的方式而將特定軟體交由有能力者處理，得到加倍的效果<ref name = "楊沛文"/>。

她也曾參與由工程師和程式設計師組成的非正式組織[[網際網路工程任務組|網際網路工程任務組]]，共同推動網際網路的建設和發展<ref name = "Claire Richard"/>。其中網際網路工程任務組沒有正式的會員資格，每個自願參加者都能夠藉由參與活動成為會員<ref name = "Claire Richard"/>。線上提案經過廣泛討論後，以[[RFC|RFC]]文件形式發佈協定；網際網路工程任務組甚至提出「我們相信粗略的共識，和進擊的程式。」為座右銘，對此唐鳳表示：「這是我第一次瞭解到的政治系統，一切都相互關聯、相互配合。<ref name = "Claire Richard"/>」在投入專案過程中，唐鳳常常連續48小時乃至於72小時工作，之後再睡眠15個小時之久<ref name = "王韻齡"/>。由於這些公益性質的設計專案不斷有需要改進的問題，她認為這是能不斷獲得快樂的無止盡遊戲，並表示：「我想要做的事，通常都是無法完成的，反過來說，只要是能夠完成的事，我大概一開始就不會想去做了。<ref name = "王韻齡"/>」

而在開放原始碼社群開始發展之際，唐鳳和[[簡信昌|簡信昌]]、[[林克寰|林克寰]]、[[高嘉良|高嘉良]]等信念相同的友人共同創辦使用者社群[[藝術家獨立協會|藝術家獨立協會]]，作為以交流意見為主的技術開發平臺，主要透過{{Link-en|郵寄名單|Mailing list}}、[[IRC|IRC]]、[[部落格|部落格]]互相聯絡<ref name = "陳思翰"/><ref name = "李欣茹"/>。藝術家獨立協會將程式設計師的工作視為像藝術工作般<ref name = "陳思翰"/>，並推行資訊社群規劃、人工智慧改良、軟體技術開發、開放原始碼等計畫<ref name = "郭宣靆">{{Cite web |url = http://mail.nhu.edu.tw/~society/e-j/18/18-08.htm |title = 口語傳統與書寫傳統：編輯與思維模式的變遷 |author = 郭宣靆、林薇瑄和朱珊玉 |date = 2001年1月15日 |publisher = [[南華大學|南華大學]] |language = zh-hant |accessdate = 2016年8月28日 |quote =  }}</ref>。另外她們也與技術同好成立台北Perl推廣組，在國內籌辦各種技術推廣活動，以在本土社群與國際間建立聯絡管道<ref name = "李欣茹"/>。而在投入自由軟體運動的過程中，唐鳳注意到來自各地不同人的創作價值，她曾經提到自己在自由軟體運動中發現到：

{{Cquote 3|一個安全空間，在這裏我們可以互相學習，而不是強加給其他人我們的欲望和希望。結果雖不完美，但總能往更好方向前進，因為這是個反復的過程，我們在這過程中學到了東西，然後又回到這裡。<ref name = "Claire Richard"/>}}

唐鳳還參與自由軟體的國際化工作與全球資訊網的建設，例如設計出Kwiki、RT、Slash等系統的國際化架構，也協助[[維基百科|維基百科]]、[[自由網|自由網]]的中文化<ref name = "端傳媒"/><ref name = "Claire Richard"/>，她還曾把困難的程式編成簡單的饒舌歌曲<ref name = "陳俊名"/>。為了達成其理想，她致力於使用網際網路和其他數位工具，希望藉由各種工具和平臺幫助社會變革以建立更健全的民主體制，並以開放原始碼的方式公開，邀請更多人共同協作<ref name = "端傳媒"/>。

=== 零時政府 ===
{{See Also|g0v零時政府}}
[[File:鳳_(唐鳳)：萌典：moedict.tw_(9722515283).jpg|thumb]]中介紹[[g0v零時政府|g0v零時政府]]推出的萌典專案。]]

2012年，行政院推出的[[經濟動能提升方案|經濟動能提升方案]]廣告造成許多網際網路鄉民的不滿，當中顯示出許多民眾對於政策並不清楚<ref name = "Claire Richard"/>。隨後「Hacker 15」團隊為了回應政府影片，決定公開中華民國政府預算資料，並以更加清晰、易懂、整齊的方式彙整在網際網路上<ref name = "Claire Richard"/>。在2個月後，唐鳳加入這群駭客，並組成網際網路社群[[g0v零時政府|g0v零時政府]]<ref name = "塗鵬揚"/>，她也成為積極參與的社群核心成員<ref name = "上報快訊"/><ref name = "黃驛淵">{{Cite web |url = http://www.upmedia.mg/news_info.php?SerialNo=3038 |title = 黃驛淵觀點 ：別急著推唐鳳上神壇 |author = 黃驛淵 |date = 2016年8月28日 |publisher = [[上報|上報]] |language = zh-hant |accessdate = 2016年8月28日 |quote =  }}</ref>，甚至是社群開放政府的指標人物之一<ref name = "何佩珊">{{Cite web |url = http://www.bnext.com.tw/article/view/id/40717 |title = 有人挺、有人憂，唐鳳能扛下台灣數位發展重任嗎？ |author = 何佩珊和翁書婷 |date = 2016年8月25日 |publisher = [[數位時代|數位時代]] |language = zh-hant |accessdate = 2016年8月28日 |quote =  }}</ref>。「g0v零時政府」的名字源自於政府網站「.gov.tw」，計畫從政府部門預算的視覺化互動網站開始，設計出開放資料、公民協作的資訊系統<ref name = "塗鵬揚"/>，藉由促進公共部門資訊透明化以便公民監督和修正<ref name = "王韻齡"/>。同時社群也塑造出去中心化的風氣，鼓勵共同參與和協作工作<ref name = "黃驛淵"/>，所有程式碼都採開放原始碼授權的方式<ref name = "Claire Richard"/>。自g0v零時政府發起後，唐鳳便為這個公民科技社群貢獻許多心力<ref name = "端傳媒"/>；其中她將自己定位在g0v零時政府中的「補位者」，主要是投入沒有其他用戶參與的工作<ref name = "端傳媒"/>。

在g0v零時政府中，除了推廣由成員主動負起責任投入專案外，並以協作形式來號召更多人同時參與專案<ref name = "蔣珮伊">{{Cite web |url = https://theinitium.com/article/20160601-taiwan-g0v/ |title = 走進台灣「零時政府」，他們改變了什麼？ |author = 蔣珮伊 |date = 2016年6月1日 |publisher = [[端傳媒|端傳媒]] |language = zh-hant |accessdate = 2016年8月28日 |quote =  }}</ref>。為了讓更多參與者協助掃描視覺化呈現所需要的大量資料，g0v零時政府的駭客把掃描檔案分割成非常小、能在幾秒鐘內轉換成「驗證碼」小任務，並增加計數器和進度條鼓勵參與<ref name = "Claire Richard"/>。其中政治獻金視覺化專案在24小時內就吸引超過9,000人參與，g0v零時政府也導入教育、勞動和衛生等部門資料；[[臺北市政府|臺北市政府]]後來也採納預算視覺化專案，並以相容格式公布預算資料<ref name = "Claire Richard"/>。除了負責許多g0v零時政府的基礎建設和專案外，唐鳳也持續連結公民科技社群與政府、外國社群間的關係，不過在公民科技社群與政府的跨界合作間也存在着許多因社群文化差異而產生的衝突<ref name = "端傳媒"/>。

另外在她提供技術支援下，也促成民間版重編的線上國語辭典「萌典」上線<ref name = "王韻齡"/>，讓g0v零時政府專案的能見度和技術水準得以提高<ref name = "端傳媒"/>，對此她形容：「我們做的事情既是軟體的介入、也是文化、甚至是藝術的介入，並沒有哪個角色扮演主角。<ref name = "蔣珮伊"/>」不過g0v零時政府極為重視線下實體世界的社會關係，唐鳳每個月會固定舉辦「黑客松」<ref name = "蔣珮伊"/>，讓知識份子、運動者和專業人士見面交流並互相學習，而不需理會彼此的意識形態<ref name = "Claire Richard"/>。她透過「黑客松」聚會的面對面溝通，解決線上無法釐清的問題，也聯絡原本彼此不認識的政府機關與不同社群，並用簡單的語言媒合各種人才與需求<ref name = "管婺媛"/><ref name = "端傳媒"/>。同時相較於維基百科社群以遠端文字溝通的非同步方式，g0v零時政府則用協作文件Hackpad營造出彼此在同一時空的同步狀態，深化且發展開放、積極的社群文化<ref name = "蔣珮伊"/>。

=== 太陽花學運 ===
[[File:太陽花學運_2014-04-09_退守前夜_(13739518834).jpg|thumb]]

2014年3月18日，[[臺北市|臺北市]]學生佔領立法院與附近道路，反對立法院通過與中國大陸簽署《[[海峽兩岸服務貿易協議|海峽兩岸服務貿易協議]]》，爆發[[太陽花學運|太陽花學運]]<ref name = "Claire Richard"/>。學生認為《海峽兩岸服務貿易協議》允許中國大陸在多個領域進行大量投資，然而在事前並沒有經過適當-{程序}-討論，因此批評政府破壞民主制度並忽略中國大陸干預的風險<ref name = "Claire Richard"/>。很快地，社交網路服務上便傳播各式各樣的學生運動組織動員圖片，唐鳳也立刻攜帶網際網路器材支援安裝<ref name = "Claire Richard"/>。然而她只在立法院議場內部待了1個小時，並認為議場內部在5個不同角度攝影機錄影和直接播出的情況下，所有活動已經成為純粹的展示演出和儀式<ref name = "Claire Richard"/>。相對地，唐鳳認為真正的辯論與行動存在立法院周圍的街道上<ref name = "Claire Richard"/>。

因此在太陽花學運期間，不認為應該扮演反對政府角色的唐鳳，對於佔領立法院或表達自己立場不感興趣，而更加重視針對運動現場開發工具技術，以鼓勵溝通並促進正在發展的言論自由<ref name = "Claire Richard"/>。她也加入架設「g0v.today」網站的駭客團隊，將佔領運動期間的所有言論直接播出、記錄和存檔，並且由眾人製作字幕<ref name = "Claire Richard"/>。其中唐鳳希望藉由網站可以觀看到不同觀點人們在立法院周圍街道上共同生活，台灣獨立運動、左派和環境運動者在運動中交流觀點，並希望民眾藉由這些工具促進共識逐漸形成<ref name = "Claire Richard"/>。同時唐鳳也關注新聞媒體宣傳的工作<ref name = "上報快訊"/>，她建議團隊在立法院外牆、每隔500公尺處安裝投影螢幕，直接轉播議場進行的討論，這讓場面不至於失控<ref name = "Claire Richard"/>。

唐鳳在退休之後，繼續全心投入各種公共事務和網際網路公益事業，逐漸成為台灣最具知名度的公民駭客<ref name = "張童恩"/><ref name = "王韻齡"/>。其中她將重心放在協助社群架構實體和虛擬世界的溝通平臺<ref name = "上報快訊"/><ref name = "張童恩"/>，試圖以不同工具、平臺藉重建民主制度<ref name = "塗鵬揚"/>。與其他開放原始碼社群相比，g0v零時政府除了議題與政治具有緊密關聯外，而在專案裡每個角色都擁有相同的重要性<ref name = "蔣珮伊"/>。不過唐鳳認為g0v零時政府的重點為指出問題並引發更多人社會關懷，但進一步的結構性改變仍需要時間慢慢進行文化感染，而非多數g0v零時政府專案的重心<ref name = "蔣珮伊"/>。也因此g0v零時政府無法真正產生公民能動性，而只是協助將參與門檻降低<ref name = "蔣珮伊"/>。

=== 開放政府 ===
[[File:民間開放資料社群拜會張善政_11.jpg|thumb]][[張善政|張善政]]、[[政務委員_(中華民國)|政務委員]][[蔡玉玲|蔡玉玲]]與民間開放資料社群開會。]]

不過長期推動開放政府的工作的唐鳳<ref name = "黃驛淵"/>，也發現政治形勢在太陽花學運結束後有所改變，民眾開始要求政治納入民主審議的結果，而不只是投票和官員的決定<ref name = "Claire Richard"/>。而隨著[[柯文哲|柯文哲]]和[[張善政|張善政]]先後就任[[臺北市市長|臺北市市長]]和[[行政院院長|行政院院長]]後，唐鳳開始投入開發能夠傳遞公民意見的可能管道<ref name = "Claire Richard"/>，其中她曾擔任[[中華民國國家發展委員會|中華民國國家發展委員會]]開放資料諮詢委員會委員<ref name = "上報快訊"/><ref name = "端傳媒"/>。在2014年至2015年間，唐鳳則擔任行政院「虛擬世界法規調適計劃」顧問<ref name = "上報快訊"/><ref name = "端傳媒"/>。主要原因是2014年12月時，[[政務委員_(中華民國)|政務委員]][[蔡玉玲|蔡玉玲]]主動向g0v零時政府黑客松提出線上法案討論平臺，希望藉此提升行政部門的效率、並讓公務員理解新世代思維與工作模式，之後唐鳳與其合作推動虛擬世界法規調適交流平台vTaiwan的開發工作<ref name = "端傳媒"/><ref name = "黃驛淵"/><ref name = "唐筱恬">{{Cite web |url = http://www.upmedia.mg/news_info.php?SerialNo=3051 |title = 唐鳳「駭」入內閣　啟動數位革命3.0 |author = 唐筱恬 |date = 2016年8月27日 |publisher = [[上報|上報]] |language = zh-hant |accessdate = 2016年8月28日 |quote =  }}</ref>。

受到[[康乃爾大學|康乃爾大學]]公民立法專案RegulationRoom啟發，g0v零時政府的專案貢獻者們創建出讓相關人士參與立法項目和法規調適案討論的平臺<ref name = "Claire Richard"/>。其中由沒有利益關係的唐鳳擔任主持人，除了提供直接播出、錄影、文字紀錄等建設性討論，也設計相應的審議系統以共同探討相關複雜議題<ref name = "Claire Richard"/>。不過唐鳳考量自己對於公共事務並不熟悉，因此僅以見習身分協助蔡玉玲，並沒有正式加入執政團隊<ref name = "何佩珊"/>。這時唐鳳希望能運用民間身份，建立「群眾外包」、與不特定人協作的公共參與機制<ref name = "何佩珊"/>。

爾後各個政府部會、各個縣市也陸續加入平臺，除了藉此協助政府推動[[開放資料|開放資料]]的目標外，並以此為基礎邀請民眾決定預算或建立國家政策<ref name = "塗鵬揚"/><ref name = "林孟汝">{{Cite web |url = http://www.cna.com.tw/news/firstnews/201608250364-1.aspx |title = 唐鳳入閣 張善政：新政府夠大膽 |author = 林孟汝 |date = 2016年8月28日 |publisher = [[中央通訊社|中央通訊社]] |language = zh-hant |accessdate = 2016年8月28日 |quote =  }}</ref>。其中平臺設計決策都是為了促進參與討論，例如為了方便參與討論而必須讓脈絡得以理解，唐鳳便希望任何感興趣者都能在2個小時內理解整個討論的脈絡<ref name = "Claire Richard"/>。當時蔡玉玲便以政府代表角色而在vTaiwan上與民眾討論政策，議題包括開放網際網路販賣酒飲、[[優步|優步]]適用法規、[[比特幣|比特幣]]和[[Airbnb|Airbnb]]存在等<ref name = "Claire Richard"/>，而引發網友討論<ref name = "唐筱恬"/>。其中在2015年於[[立法院|立法院]]三讀通過的《公司法》修正案，就是vTaiwan平台經過4個多月討論後提出的草案<ref name = "戴雅真"/>。也由於vTaiwan的推出，使得唐鳳也與[[法國外交部|法國外交部]]、[[法國經濟財政部|法國經濟財政部]]、[[巴黎|巴黎市政府]]、西班牙[[馬德里|馬德里市政府]]在數位治理領域合作<ref name = "上報快訊"/><ref name = "端傳媒"/>。

=== 教育領域 ===
唐鳳認為過去的教育環境為所有學生規劃一致的路線與方向，此舉無形中阻擋了不同發展的可能性，不過她覺得因為社會環境改變使得教師話語權大幅降低<ref name = "王韻齡"/>。因此過去不就學即可被視為激進行動，而今日隨著實驗教育法律的通過，相同信念者都能夠成立學校，亦能夠結合鄰近資源或大學而提出在家自學計畫<ref name = "王韻齡"/>。唐鳳認為自己因為不斷變換軌道而有不同見解，並認為在家自學者可以使用自己周邊的資源與選項，與體制對話和合作而漸進式地在家自學<ref name = "王韻齡"/>。2014年時，[[國家教育研究院|國家教育研究院]]因為[[教育部國語辭典|教育部國語辭典]]的條目有許多缺漏而舉辦「啄木鳥競賽」，唐鳳的團隊在2週內就提出4,000多筆錯誤，佔整個比賽糾錯8成，並使前國家教育研究院副院長、[[國立臺東大學|國立臺東大學]]特殊教育學系教授[[曾世杰|曾世杰]]對其糾錯能力印象深刻<ref name = "上學被圍毆想自殺"/>。

唐鳳認為面對與社會大眾不同的特殊孩童時，父母應該訓練同理心並嘗試理解孩童所感受到的世界<ref name = "王韻齡"/>。而從2014年開始，由於認同新課程綱要強調自發的價值觀、並承認體制內較無法代替孩童決定，因她接受[[中華民國教育部|中華民國教育部]]的邀請而參與107年課程綱要改革會議，擔任十二年國民基本教育課程發展委員會委員<ref name = "上報快訊"/><ref name = "端傳媒"/><ref name = "管婺媛"/>，並將課綱審議的會議紀錄公開於網際網路<ref name = "王韻齡"/>。為了能夠使審理課程綱要的過程公開透明，她把現場聽到的發言內容具名且逐字記錄，並連續2天打了16個小時的完整逐字稿<ref name = "王韻齡"/>。而她本人也認為由於過去較少接觸國中和高中課本，因此在審理課程綱要時更能夠持中立角度<ref name = "王韻齡"/>。

到了2015年，因為[[臺灣高中課程綱要微調案|臺灣高中課程綱要微調案]]所造成的爭議中，會議透明度遭到質疑的國家教育研究院邀請唐鳳參與課程發展委員會，希望借重她的資訊公開經驗，讓政府與民間的溝通更為順暢<ref name = "上學被圍毆想自殺"/>。不過唐鳳在第一次出席會議便同時發言和打字，除了與所有委員辯論並說服所有委員外，最後還將80多頁發言紀錄逐字紀錄稿對外公開<ref name = "上學被圍毆想自殺"/>。而之後透過將會議紀錄全部公開至網際網路上，化解社會各界理解政府溝通的誠意而化解此次危機<ref name = "王韻齡"/>。

== 政務委員 ==
[[File:林全院長視察亞洲矽谷預定地_04.jpg|thumb]]在桃園市長[[鄭文燦|鄭文燦]]的陪同下視察[[亞洲矽谷計畫|亞洲矽谷計畫]]預定地。]]

由於[[行政院|行政院]]推行的重要經濟政策[[亞洲矽谷計畫|亞洲矽谷計畫]]，反而遭到新創圈人士批評，使得行政院長[[林全|林全]]必須與新創界人士私下溝通政策<ref name = "亞州矽谷">{{Cite web |url = http://www.upmedia.mg/news_info.php?SerialNo=3052 |title = 「亞州矽谷」計畫與新創圈卡卡　林全網羅專業人才修補 |author = 唐筱恬 |date = 2016年8月27日 |publisher = [[上報|上報]] |language = zh-hant |accessdate = 2016年8月28日 |quote =  }}</ref>。而自2016年6月底不定期舉辦的「亞洲矽谷」小組會議，林全也多次與中華民國國家發展委員會主任委員[[陳添枝|陳添枝]]、副主任委員[[龔明鑫|龔明鑫]]、政務委員[[吳政忠|吳政忠]]；新創界代表唐鳳、[[和沛科技|和沛科技]]創辦人[[翟本喬|翟本喬]]、[[AppWorks|AppWorks]]創辦人[[林之晨|林之晨]]、[[funP|funP]]創辦人[[邱繼弘|邱繼弘]]；[[民主進步黨|民主進步黨]]籍立法委員[[余宛如|余宛如]]、[[鄭運鵬|鄭運鵬]]等人會談<ref name = "亞州矽谷"/>。會議中林全則委託唐鳳從新創界或科技界中，尋找具有科技法律專長、或熟悉新創圈者擔任[[政務委員_(中華民國)|政務委員]]<ref name = "亞州矽谷"/>，負責政府與新創圈之間的溝通橋樑<ref name = "張童恩"/>。在因大部份候選人都因其事業無法出任後，同樣是各界推薦的唐鳳則成為重要人選<ref name = "張童恩"/>，而她也經數天思考後同意這項邀請<ref name = "亞州矽谷"/>。

其中林全希望藉由唐鳳在數位技術和系統的經驗，協助公務體系解決問題，並讓數位經濟政策和開放政府推動更加順利<ref name = "林孟汝"/>。2016年8月25日，[[行政院新聞傳播處|行政院新聞傳播處]]發言人[[童振源|童振源]]在行政院院會後記者會上表示，林全已經決定任命網際網路創業家唐鳳擔任政務委員，負責督導數位經濟與開放政府發展<ref name = "塗鵬揚"/><ref name = "上報快訊"/><ref name = "林孟汝"/>。在行政院發布入閣人事消息後，唐鳳隨即在Facebook上發文，並在Wiselike上開設線上訪問系統，讓各界新聞媒體與一般民眾得以透過該系統而在第一時間留言訪問<ref name = "上報快訊"/><ref name = "黃驛淵"/>。她的父親唐光華也在Facebook上表態支持<ref name = "成為公僕的公僕"/>，許多網際網路、新創圈則對於數位人才進入政府表示支持，但亦有人懷疑是否在政府體系的傳統框架中能夠擁有發揮的機會<ref name = "何佩珊"/>。

35歲的唐鳳也將是行政院有史以來最年輕的政務委員<ref name = "何佩珊"/>，也是中華民國歷史上第一位跨性別閣員<ref name = "張童恩"/>。唐鳳表示之後將透過數位技術和系統輔助公務體系解決問題，並強化政府部門與公民科技、公共社群的對話和合作管道<ref name = "上報快訊"/><ref name = "端傳媒"/>，而非只在網際網路上宣導政策<ref name = "唐筱恬"/>。其中行政院計畫由唐鳳負責督導軟體和開放政府工作，亞洲矽谷計畫則由陳添枝主要負責、唐鳳協助相關計劃<ref name = "亞州矽谷"/>。當中唐鳳除了成立溝通界面執行亞洲矽谷計畫，並將概念構想定調為「IoT、桃園試點、鏈結亞洲、連接矽谷」<ref name = "亞州矽谷"/>。當中仍然將[[桃園市|桃園市]]列為示範場域，但是臺灣各地具有潛力的地區都可以形成聚落，連結為概念性產業、新型態經濟，進而從臺灣出發而與亞洲、乃至於矽谷連結<ref name = "亞州矽谷"/>。

== 思想理念 ==
[[File:唐鳳.jpeg|thumb]]

雖然無法融入學校社群一度讓唐鳳感到孤單，但她也發現網際網路上有許多人同樣如此，她並注意到「與眾不同」才是常態<ref name = "王韻齡"/><ref name = "端傳媒"/>。而程式設計師之間互相協作的生態，也使她思考如何這類合作模式引入社會互動中<ref name = "端傳媒"/>。但她也重視實體世界與人際溝通，認為在地人擁有的特殊知識是電腦無法取代的內容<ref name = "王韻齡"/>。由於感受到網際網路中自治、合作和信任集體智慧的特色，使得她將其視為掌握科技的方法，並將網際網路作為思考政策、創造對話、建立共識平台的重要工具<ref name = "端傳媒"/>，也因此不斷投入開發更具包容性、更開放、更具代表性和更公正的民主機制平臺網站<ref name = "Claire Richard"/>。對此她提到：

{{Cquote 3|讓我們說出真心話，讓所有人能無顧忌地表達自己的情感。一切言論審查，包括自我審查，都是系統的缺失。『拒絕審查』本身，就是激進的政治主張。<ref name = "Claire Richard"/>}}

也因為受到網際網路價值的影響，唐鳳稱呼自己是保守[[無政府主義|無政府主義]]者，期望能夠維持網際網路上的無政府主義<ref name = "塗鵬揚"/><ref name = "端傳媒"/>。唐鳳同時還閱讀過[[自由主義|自由主義]]、無政府主義、[[馬克思主義|馬克思主義]]等理論<ref name = "楊沛文"/>，相信利人利己的精神，並希望能夠促成透明且公開的公民社會<ref name = "王韻齡"/>。而她將自己視為聯繫者角色，協助將兩個互不相識的組織得以連結起來<ref name = "管婺媛"/>。而雖然過去未曾正式進入政府體制內，但因陸續擔任行政院顧問等經歷，使得她與各部會間已經有互動經驗，而對於政府部門內部運作或政策思考並不陌生<ref name = "端傳媒"/><ref name = "何佩珊"/>。唐鳳計畫在進入行政院後，將透過vTaiwan界面等新科技引入「群眾外包」概念，讓群眾智慧集體參與政府政策，並讓政府官員、事務官與民眾溝通，商討包括「一例一休」、優步、亞洲矽谷計畫等需要討論的議題<ref name = "唐筱恬"/>。

而對於和公務人員體系的結合，唐鳳則提及維基百科對於無政府主義的解釋「由自由的個體自願結合，以建立互助、自治、反獨裁主義的和諧社會」<ref name = "端傳媒"/>。她認為具體上將不斷提出更加合適的程序，直到公務人員自願採用為止，並認為過往參與的網際網路社群、開放文化社群、維基百科等都體現這一價值，而過往也運用在參與政府職務、及與政府交涉vTaiwan專案的工作過程中<ref name = "端傳媒"/>。而面對不熟悉數位工具的公務人員，她則表示在部門溝通過程中，仍然是以語音、表情、手勢、紙筆等傳統溝通媒介，協調將人工作業交付給預設程式處理、輔助或備份；但重點在於將協調過程充分數碼化，以方便檢索、參與、追蹤、問責等工作<ref name = "端傳媒"/>。

== 個人生活 ==
[[File:Audrey_Tang_2.jpg|thumb]]

唐鳳在經過多次檢驗後，其智商均高於180以上<ref name = "古美蓮"/><ref name = "上報快訊"/><ref name = "張童恩"/>，並在教育界和資訊界有一定知名度<ref name = "陳柏年"/>，甚至被稱作「天才型的早熟少年」或「台灣版電腦兒童」<ref name = "楊沛文"/>。不過自小是天才兒童的她也因為智商高，因為教師和同學無法應對，因而不適應正規教育體系<ref name = "古美蓮"/><ref name = "成為公僕的公僕"/>，在9年內更換了3所幼稚園與6所小學<ref name = "端傳媒"/><ref name = "王韻齡"/>，甚至一度前往德國<ref name = "古美蓮"/>。同時因為先天性心臟病與多種疾病，導致她不能奔跑與跳動，動作緩慢也成為其在學校遭到攻擊的理由<ref name = "陳柏年"/>，他的母親李雅卿則將成長經歷紀錄在1998年的著作《成長戰爭》中<ref name = "陳俊名"/>。

而除了編寫程式外，唐鳳也會創作詩歌、寫散文、彈琴、譜曲等<ref name = "楊沛文"/>，她也曾獲得紙牌遊戲《[[魔法風雲會|魔法風雲會]]》臺灣地區大賽第一名<ref name = "郭宣靆"/>。唐鳳認為每天有16個小時來決定安排工作，而從事思想工作者必須忍受長時間孤獨和茫然，且必須脫離慣性以開展思想<ref name = "王韻齡"/>。她持續每周至少4日、每天45分鐘進行精神分析，並與法國線上心理分析師對話，促進人際互動與對話<ref name = "王韻齡"/>，她在18歲時還曾經前往尼泊爾、西藏體驗奧修<ref name = "楊沛文"/>。由於關注「注意力對稱」的問題，因此在出席任何公眾場合、會議或訪談時，她身上都會攜帶360度全景攝影機，並將影片完整上傳至Facebook和YouTube上<ref name = "王韻齡"/>，目的就是讓網際網路上的公民也能見到現實世界中的場景<ref name = "塗鵬揚"/>。

唐鳳的[[性別認同|性別認同]]為[[跨性別女性|跨性別女性]]，過去便自認為自己屬於女性，並多維持長髮的中性裝扮<ref name = "陳俊名"/>，但因為與社會期待相反而感到困擾<ref name = "塗鵬揚"/>，她便提到：「我是『後類別』。在性別爭論中我不選邊站。並不是說我認為這個議題不重要，而是我認為爭論不能解決任何問題。<ref name = "Claire Richard"/>」2005年年底，24歲的唐鳳在部落格上自行宣告為跨性別<ref name = "李欣芳">{{Cite web |url = http://news.ltn.com.tw/news/politics/breakingnews/1808130 |title = 跨性別宣告的唐鳳 政黨、性別欄都填「無」 |author = 李欣芳 |date = 2016年8月27日 |publisher = 《[[自由時報|自由時報]]》 |language = zh-hant |accessdate = 2016年8月28日 |quote =  }}</ref>，除了更改[[中華民國國民身分證|中華民國國民身分證]]和[[中華民國護照|中華民國護照]]中文名字與英文名字外，也在家人與親友支持下開始接受變性手術<ref name = "塗鵬揚"/><ref name = "成為公僕的公僕"/><ref>{{Cite web |url = http://pugs.blogs.com/audrey/2005/12/runtime_typecas.html |title = Runtime Typecasting. |author = 唐鳳 |date = 2005年12月25日 |publisher = 鳳 |language = zh-hant |accessdate = 2016年8月28日 |quote =  }}</ref>。之後她的姓名由原來的男性化名字「唐宗漢」，改名為女性化名字「唐鳳」<ref name = "成為公僕的公僕"/><ref name = "端傳媒"/>；而她表示自己在行政院人事資料表的性別欄填寫「無」<ref name = "張童恩"/>，且會以個人身分持續關注跨性別議題<ref name = "李欣芳"/>。其父親唐光華則表示：「如果她覺得性別的轉變可以讓她更快樂、更能發揮創造力，又不會傷害任何人，沒有理由不接受。<ref name = "張童恩"/>」而在2007年時，唐鳳則和丁凡共同翻譯美國小說家{{Link-en|茱莉·安·彼得絲|Julie Anne Peters}}的著作《[[她是我哥哥|她是我哥哥]]》。
