{{vgname|唐老鸭梦冒险|ja=わんぱくダック夢冒険|en=DuckTales|diff=yes}}是一款由[[卡普空|卡普空]]制作和发行的[[平台游戏|动作平台类游戏]]。<ref name="mobygames">[http://www.mobygames.com/game/disneys-ducktales ''DuckTales''] game overview at [[MobyGames|MobyGames]]</ref> 本游戏根据迪斯尼《[[唐老鴨俱樂部|唐老鴨俱樂部]]》改编。本游戏于1989年和1990年[[FC游戏机|FC游戏机]]上发行。<ref name="gamefaqs">{{cite web | url = http://www.gamefaqs.com/nes/587253-disneys-ducktales/data| title = Release information | publisher = [[GameFAQs|GameFAQs]]| date = | accessdate = 2015-09-28}}</ref>之后于1990年移植至[[Game_Boy|Game Boy]]掌机。

玩家控制的是[[史高治·麦克老鸭|史高治叔叔]]，他去前往世界探索五大财宝来增加他的财富。游戏中史高治叔叔用它的拐杖来攻击敌人。

《唐老鸭梦冒险》获得商业成功，FC游戏机版本和Game Boy版在全球分别卖出167万份和143万份。每一项都是卡普空在该平台售出的最高纪录。<ref name="Sales">{{cite web |url=http://www.eurogamer.net/articles/sfii-snes-is-capcoms-best-selling-game |date=2008-05-27 |title=SFII SNES is Capcom's best-selling game |publisher=[[Eurogamer|Eurogamer]] |author=Bramwell, Tom |accessdate=2013-03-23}}</ref>

《[[任天堂力量|任天堂力量]]》在2008年将本游戏列为FC游戏第十三位最佳游戏。
