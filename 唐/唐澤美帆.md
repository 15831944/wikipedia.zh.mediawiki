'''唐澤美帆'''（{{lang-ja|唐沢 美帆}}、{{bd|1983年|7月15日|catIdx=Karasawa Miho}}<ref>{{Cite web|url = http://profile.ameba.jp/true-sound/|title = 官方部落格的個人資料|date = |accessdate = 2016年1月27日|website = |publisher = Ameba|last = Karasawa|first = Miho|language = ja}}</ref>）是日本東京都出生的歌手，過往所屬公司為[[崛製作|崛製作]]，目前隸屬於[[Lantis|Lantis]]唱片公司。

唐澤於2000年出道並推出她的首張單曲“anytime, anywhere”。其後她演唱過《[[機甲巴打|機甲巴打]]》、《[[翠星上的加爾岡緹亞|翠星上的加爾岡緹亞]]》、<ref>{{Cite web|url = http://www.animenewsnetwork.com/news/2014-07-13/true-choucho-sing-new-gargantia-video-anime-themes/.76632|title = True, ChouCho Sing New Gargantia Video Anime's Themes|date = 2014年7月13日|accessdate = 2016年1月27日|website = |publisher = [[動畫新聞網|動畫新聞網]]|last = |first = |language = en}}</ref>《[[純潔的瑪利亞|純潔的瑪利亞]]》、<ref>{{Cite web|url = http://www.animenewsnetwork.com/news/2014-11-30/maria-the-virgin-witch-anime-unveils-video-more-cast-theme-songs/.81590|title = Maria the Virgin Witch Anime Unveils Video, More Cast, Theme Songs|date = 2014年11月30日|accessdate = 2016年1月27日|website = |publisher = [[動畫新聞網|動畫新聞網]]|last = |first = |language = en}}</ref>及《[[吹響吧！上低音號～歡迎加入北宇治高中管樂團～|吹響吧！上低音號～歡迎加入北宇治高中管樂團～]]》動畫主題曲等等。<ref>{{Cite web|url = http://www.animenewsnetwork.com/news/2015-02-06/kyoto-animation-unveils-sound-euphonium-additional-cast-songs/.84158|title = Kyoto Animation Unveils Sound! Euphonium's Additional Cast, Songs|date = 2015年2月6日|accessdate = 2016年1月27日|website = |publisher = |last = |first = |language = en}}</ref>

== 人物概要 ==
* 15歲時通過少女組合出道，後於2000年8月發行首張單曲「anytime, anywhere」並受到矚目。同年10月發行了個人的第二張單曲「0°C ～stay with me～」，同樣由島野聰（因為以MISIA製作而出名）擔當3rd單曲「Way to Love」的製作人，因為17歲少女清新的歌詞中對於戀愛態度的直觀表現加上動聽的旋律，配合電視劇「ラブ·レボレーション」（插曲）而受到了相當大的歡迎。 在這樣的前提下，於2001年9月和2002年2月發售了她的第四以及第五張單曲：“ライヴ”、“Endless Harmony”。 個人認為此時的她處於事業的最頂峰，特別是“Endless Harmony”，無疑是她所有歌曲當中最棒的一首。 同樣的，還是由島野聰提供歌曲並製作。 然後，順理成章的發售了她的第一張大碟“sparkle”，收錄了多張熱門單曲。 

* 2002年4月24日，她和今井大介合作了一張單曲「I DO」。值得注意的是這次是跨公司的合作，今井是BMG jp的。從這張開始感覺她有了變化，似乎試圖改變之前的少女印象。 

* 2002年9月4日，發售她的第七張單曲「affection」。 

* 2003年1月16日，發售她的第八張單曲“Jazztronik Presents Mihonova.”。之後，她的事業進入低潮期。 

* 大概是歌迷不適應她的轉變，所以相隔一年才發售了第九張單曲“Cloudy”。 

* 而她的個人第二張大碟“ID.”也於2004年2月18日發售。 但是情況並沒有好轉。 所以她的第十張單曲“君のかけら”於2004年末發售之後直到現在都沒有再發布新碟，偶有與其他歌手的合作，如與SoulJa合作的“Way to Love ~最後の戀~”。 

* 2014年起，她以「'''TRUE'''」名稱在活動時出現<ref>{{Cite web|title = TRUE {{!}} Artist {{!}} Lantis web site|url = http://lantis.jp/artist.php?id=6e5e6335679d6187d66ed28bf5f452de|website = |accessdate = 2016年1月27日|publisher = [[Lantis|Lantis]]|language = ja}}</ref>。現在活躍於主持等演藝活動。
