'''唐军'''（{{bd|1962年|5月||catIdx=T唐}}），[[江苏|江苏]][[建湖|建湖]]人。1983年8月参加工作，1986年1月加入中国共产党。[[武汉大学|武汉大学]]中文系中国文学专业毕业，文学学士。曾任[[中共辽宁省委|中共辽宁省委]]常委、[[大连市|大连市]]委书记。

== 简历 ==
曾任[[中华人民共和国劳动人事部|中华人民共和国劳动人事部]]干部局干部，[[中华人民共和国人事部|中华人民共和国人事部]]流动调配司人员流动处副处长，人事部办公厅部长办公室正处级秘书、副司级秘书，人才流动开发司副司长，规划财务司司长，人事部人事教育司司长。2007年8月，任人事部副部长、党组成员。
 
2008年3月，任[[中共辽宁省委|中共辽宁省委]]常委、组织部部长。2011年6月，任中共辽宁省委常委、大连市委书记<ref>{{cite web |url=http://news.sina.com.cn/c/2011-06-27/124522712855.shtml |title=中央决定唐军任大连市委书记 |publisher=新浪 |accessdate=2012-01-09}}</ref><ref>{{cite web|url=http://www.thepaper.cn/newsDetail_forward_1622978|title=谭作钧任辽宁大连市委书记，唐军不再担任|publisher=澎湃新闻|accessdate=2017-02-20}}</ref>。

2017年2月，任[[国家工商行政管理总局|国家工商行政管理总局]]副局长、党组成员<ref>{{cite web|url=http://www.thepaper.cn/newsDetail_forward_1626854|title=大连市委原书记唐军调任国家工商总局副局长，刘玉亭不再担任|publisher=澎湃新闻|accessdate=2017-02-25}}</ref>。2018年3月，任新组建的[[国家市场监督管理总局|国家市场监督管理总局]]副局长、党组成员<ref>{{cite web |accessdate=2018-04-05 |url = http://www.gov.cn/guowuyuan/2018-04/04/content_5279818.htm |title=国务院任免国家工作人员（4月4日） |publisher=中国政府网}}</ref>。
