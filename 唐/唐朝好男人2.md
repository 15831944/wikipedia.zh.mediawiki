《'''唐朝好男人2'''》，2014年上映于中国大陆的古装喜劇類型的網絡劇，承接第一部《[[唐朝好男人|唐朝好男人]]》，导演孙恺凯，主演张钧涵、李昕岳、夏一瑶、刘唱<ref>{{cite web |url=http://news.xinhuanet.com/ent/2014-04/28/c_126443109.htm |title=《唐朝好男人2》网络热播 不惧与热剧抢夺收视  |accessdate= 2014|publisher=新华网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.ifeng.com/a/20140425/40032834_0.shtml |title=《唐朝好男人2》开播 续写爆笑自制传奇 |accessdate= 2014|publisher=凤凰网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.qq.com/a/20140424/000169.htm |title=《唐朝好男人2》回归 主创聊“唐朝那些事儿” |accessdate= 2014|publisher= 腾讯网|work= |language=zh-hans}}</ref>，2014年4月25日首播。

==剧情==
该剧承接第一部，继续讲述小职员王子豪穿越到[[唐朝|唐朝]]摇身一变没落[[贵族|贵族]]，用21世纪知识在唐朝如鱼得水，过着逍遥快活的生活。
