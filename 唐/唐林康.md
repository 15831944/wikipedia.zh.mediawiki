'''唐林康'''{{BD|1992年|1月12日}}，[[中国|中国]][[安徽|安徽]][[安庆|安庆]]人，从事[[电子商务|电子商务]]职业。<ref>{{cite web |title=中国梦之声波澜哥真名唐宁康奇怪演唱遭韩红骂 波澜哥个人资料 |url=http://m.sohu.com/a/75685264_412028/?pvid=000115_3w_a |website=搜狐网 |accessdate=2018-10-06}}</ref>在参加《[[中国梦之声|中国梦之声]]第二季》[[成都|成都]]试音会时清唱《离不开你》，结果严重跑调，遭到评委[[韩红|韩红]]嫌弃。其节目录像后在[[哔哩哔哩|哔哩哔哩]]等视频网站传播，并被再创作为大量“[[音MAD|鬼畜]]”视频，唐林康亦获得绰号“波澜哥”。<ref>{{cite web |title=青年亚文化视角下的网络视频文化研究 ——以“鬼畜视频”为例 |url=http://media.people.com.cn/n1/2018/0918/c421458-30300451.html |website=人民网 |accessdate=2018-10-06}}</ref>

==试音经过==
唐林康在试音时面对[[任贤齐|任贤齐]]、[[韩红|韩红]]、[[徐若瑄|徐若瑄]]三位评委。在任贤齐引导下做完自我介绍后，他开始清唱[[刘欢|刘欢]]的《离不开你》{{notetag|唐林康自称演唱“黄妈的《离不开你》”是因为[[黄绮珊|黄绮珊]]曾与韩红合唱过此歌。}}，在唱到“太不公平”时被韩红打断，后者发问“你要去哪啊”。唐在第二次唱到“抛弃了我”时轻声问道“可以吗”，韩红回答“我快疯了”；徐若瑄提醒他眼神不要闪躲，韩红则认为他的问题只是调不准。之后韩红亲自示范，唐再次唱起“掀起波澜”，韩红立刻喊道“哎呦我去”。在第二次示范后唐的音调依然没有变化，韩红回应“滚”，之后起身。第三次示范后唐的音调依然如旧，韩红再次起身。徐若瑄请求韩红再给他一个机会，并让乐队播放《你把我灌醉》伴奏，韩红同时拍桌说“我去年买了个表”。在徐若瑄的请求下韩红让唐演唱五句《你把我灌醉》。在唱到“我梦到哪里”时韩红大喊“停停停停”，调侃了几句后请唐林康离场。<ref>{{cite web |title=波澜哥 完整版 |url=https://v.qq.com/x/page/r0147e42nfu.html?ptag=www_sogou_com |website=优酷网}}</ref>
