'''唐代三夷教'''是指於[[唐朝|唐朝]]時期在中國傳佈的三種發源於[[西亞|西亞]]的宗教，受到唐王朝的認可和保護，興盛一時。

== 背景 ==
「唐代三夷教」包括：
* [[瑣羅亞斯德教|瑣羅亞斯德教]]（'''[[祆教|祆教]]'''）
* [[基督教|基督教]][[聶斯脫里派教會|東方教會]]（'''[[景教|景教]]'''）
* [[摩尼教|摩尼教]]（'''[[明教|明教]]'''）

祆教最初傳入中國應是在[[南北朝|南北朝]]初期，景教和摩尼教則都是唐朝始入華，三支宗教無疑皆是通過[[絲綢之路|絲綢之路]]經[[中亞|中亞]]地區傳入中原。唐朝第二代皇帝[[唐太宗|太宗李世民]]於[[貞觀_(唐朝)|貞觀]]十三年（639）率兵出擊[[高昌|高昌]]王[[麴文泰|麴文泰]]，其子[[麴智盛|麴智盛]]即位後歸屬唐王朝，唐朝於高昌設置[[安西大都護府|安西大都護府]]以便管理。此後直到[[唐玄宗|唐玄宗]][[開元之治|開元之治]]，唐朝盛世達到頂峰，開明的制度和對宗教的包容為不同文化的傳播匯集打開門戶。當時的國際都城[[長安|長安]]人口超過一百萬，胡風大盛，充滿異國情調。街市可見[[印度|印度]][[魔術|幻術師]]、[[波斯|波斯]]舞者、歌手、樂師、酒肆胡姬、異域美食，[[馬球|馬球]]一類活動在宮廷非常流行，貴族女子亦鍾意馭馬出行。<ref>{{cite book |author=塚本善隆 |date=1974 |title=《世界の歴史4 唐とインド》 |series=中公文庫 |language=ja |location=東京 |publisher=中央公論社 |pages={{pp.|371|386}} |isbn= |author-link=}}</ref>在這樣的背景下，唐朝自然成為三夷教傳播的最佳時期。

然而，三夷教在9世紀[[唐武宗|唐武宗]][[會昌毀佛|會昌毀佛]]時期同受打擊，從此漸漸退出中國歷史舞臺。不過，它們的影響於不同時期仍然若隱若現。特別是摩尼教在後期轉入民間祕密流傳，同佛教、道教、民間信仰融合混雜。2008年發現的[[福建省|福建省]]《[[霞浦文書|霞浦文書]]》，大概成書於明淸之際。如今在[[霞浦縣|霞浦縣]]有專門的法師保存這些明教科儀書（即經咒和宗教儀式書籍），<ref>{{cite web |url=http://www.chinanews.com/cul/news/2010/04-26/2247185.shtml |title=《倚天屠龍記》明教遺跡藏閩霞浦小山村 (圖) |editor=蒲波 |date=2010-04-26 |website=chinanews.com |access-date=2018-12-22}}</ref>表明霞浦明教改頭換面，以一種中國民間信仰的形式流傳至今。<ref>{{cite news |author=Kameyou |date=2014-12-20 |title=福建明敎遺裔 |url=http://news.memehk.com/posts/%E7%A6%8F%E5%BB%BA%E6%98%8E%E6%95%99%E9%81%BA%E8%A3%94 |website=news.memehk.com |publisher=[[謎米新聞|謎米新聞]] |location=香港 |access-date=2018-12-22}}</ref>

== 概述 ==
=== 祆教 ===
{{Main|瑣羅亞斯德教}}
[[File:Sogdian-Zoroastrian_Deities,_Tunhwang.jpg|thumb]]》，出自[[敦煌|敦煌]]，是用於[[賽祆|賽祆]]的紙本神像。]]
發源於[[波斯|波斯]]的瑣羅亞斯德教在古代中國稱作祆教、火祆教，俗稱拜火教，這支宗教曾是[[阿契美尼德王朝|阿契美尼德帝國]]、[[安息帝國|安息帝國]]同[[薩珊帝國|薩珊帝國]]的國教，盛極一時。[[陳垣|陳垣]]在其著作《火祆教入中國考》中認為祆教最早名聞中國是始自[[北魏|北魏]][[梁_(南朝)|南梁]]時期，即公元6世紀初，北魏[[宣武靈皇后|靈太后]]所祀之胡天神就是祆神；[[林悟殊|林悟殊]]則認為祆教於5世紀中葉已傳入中國；[[榮新江|榮新江]]考證各種文獻資料後，認為其入華時間應在4世紀初。祆教對自身廟宇的稱呼與其他宗教不同，其他宗教一般稱「寺」，祆教稱「祠」，最早的祆祠是[[長安|西京]]布政坊西南的祆祠。祆教還有別於其他宗教的一點是教徒在中國並不傳教，也不翻譯經書，因而信徒只有[[胡人|胡人]]沒有漢人。<ref>{{cite web |url=https://www.chiculture.net/20205/html/xzs/xzs025.html |title=瑣羅亞斯德教（祆教） |author=<!--Staff writer(s); no by-line.--> |website=chiculture.net |access-date=2018-12-22}}</ref><ref>{{cite web |url=http://www.cssn.cn/zjx/zjx_zjyj/mjzjxyyj/201810/t20181010_4667445.shtml |date=2018-10-10 |title=從晚唐五代敦煌「賽祆」探祆教習俗與中國節慶風俗的融合 |author=趙洪娟 |website=cssn.cn |access-date=2018-12-23}}</ref>由出土[[粟特|粟特]]人的墓誌銘可知，隋唐時期設有「[[薩保|薩保]]」（亦作「薩寶」）官職，音譯自[[粟特語|粟特語]]「{{transl|sog|s′rtp′w}}」，本意是「商隊首領」，多為粟特祆教徒擔任，專門管理在中原地區居住的西域胡人以及其宗教活動。該教於在華粟特人中尤其盛行，唐朝五代時的[[敦煌|敦煌]]就有粟特人擧辦的[[賽祆|賽祆]]大會。[[唐武宗|唐武宗]][[會昌毀佛|會昌毀佛]]時期，祆教同受打擊，於[[宋朝|宋朝]]之後在中國消失。<ref>{{cite web |url=https://religion.moi.gov.tw/Knowledge/Content?ci=2&cid=297 |title=瑣羅亞斯德（查拉圖斯特拉） |author=蔡怡佳 |website=religion.moi.gov.tw |access-date=2018-12-22}}</ref>現今中國祆教藝術主要可以從出土的[[中國粟特人墓葬列表|粟特人墓葬]]中看到。

景教是基督教[[聶斯脫里派教會|東方教會]]傳入中國以後的名稱，這派教會歷來被誤認為是[[聶斯脫里主義|聶斯脫里派]]。因其教座設於波斯[[薩珊帝國|薩珊帝國]]，所以初入唐帝國時稱大秦教、波斯經教，最終才定名景教。據[[大秦景教流行中國碑|大秦景教流行中國碑]]所載，[[唐太宗|唐太宗]][[貞觀_(唐朝)|貞觀]]九年（635），[[大秦|大秦]]國有大德[[阿羅本|阿羅本]]攜帶經像來到[[長安|長安]]，太宗派宰相[[房玄齡|房玄齡]]至西郊迎賓入內。考查教義後，太宗下令在義寧坊造[[大秦寺|大秦寺]]一所，也就是景教寺院，並在房玄齡、[[魏徵|魏徵]]的協助下開始譯經工作。此後，景教在中國順利發展了150年，一度有「法流十道，寺滿百城」的盛況。<ref>{{cite book |author=吳昶興 |date=2015年5月29日 |title=《眞常之道：唐代基督教歷史與文獻研究》 |series=基督教學術叢書/論著系列 |volume=11 |url=https://books.google.com/books?id=x70uDwAAQBAJ&pg=PA268&lpg=PA268&dq=%E6%B3%95%E6%B5%81%E5%8D%81%E9%81%93+%E5%AF%BA%E6%BB%BF%E7%99%BE%E5%9F%8E&source=bl&ots=L69t0owxNT&sig=0-QTxE13qqJ0Ze0WEStDu4iIayc&hl=en&sa=X&ved=0ahUKEwjzxO-CkfXbAhVH1xQKHaaxBBgQ6AEIRzAF#v=onepage&q=%E6%B3%95%E6%B5%81%E5%8D%81%E9%81%93%20%E5%AF%BA%E6%BB%BF%E7%99%BE%E5%9F%8E&f=false |location=新北市 |publisher=台灣基督教文藝出版社 |page={{p.|268}} |isbn=978-986-61-3129-5}}</ref>阿羅本還於[[唐高宗|唐高宗]]時獲封「鎭國大法主」；據《[[尊經|尊經]]》所載，波斯僧[[景淨|景淨]]於公元8世紀漢譯了35部景教經文。直到[[唐武宗|唐武宗]]發起[[會昌毀佛|會昌毀佛]]，景教亦受連纍而迅速衰落。該教第二次大擧入傳中國則要等到[[元朝|元朝]]建立之後，但於[[明朝|明朝]]時再度消亡。景教遺留下的藝術品不多，現有[[藏經洞基督像|藏經洞基督像]]絹畫和[[高昌景教寺院壁畫|高昌景教寺院壁畫]]傳世。此外，還有景教徒帶有雕刻裝飾的墓碑、景教銅牌（銅十字架和其他造形的衣袍裝飾件）等。<ref>{{cite web |url=https://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=91021&Pid=5&Version=0&Cid=220&Charset=big5_hkscs |title=元代景教銅牌的文化交流故事 |author=陳天惠 |date=2015-09-05 |website=christiantimes.org.hk |accessdate=2018-12-22}}</ref><ref>{{cite web |url=https://www.pentoy.hk/%E9%8A%85%E5%8D%81%E5%AD%97-%E6%8F%AD%E9%96%8B%E6%99%AF%E6%95%99%E5%8D%83%E5%B9%B4%E6%96%87%E6%98%8E/ |title=銅十字 揭開景教千年文明 |author=<!--Staff writer(s); no by-line.--> |date=2015-06-15 |website=pentoy.hk |accessdate=2018-12-22}}</ref>

=== 摩尼教 ===
{{Main|摩尼教}}
[[File:Scene_from_the_manichean_book_Work_of_Religion.jpg|thumb]]的摩尼教古抄本「[[摩尼教抄本殘頁_MIK_III_4974|MIK III 4974]]」插圖，應是源於《[[大二宗圖|大二宗圖]]》，描繪兩名穿戴白冠白袍的選民和兩名平信徒，他們面前擺放一盤瓜果，是摩尼教徒最鍾意的食物。]]
[[File:Manichaean_Banners.jpg|thumb]]摩尼教繪幡「[[摩尼教繪幡_MIK_III_6286|MIK III 6286]]」和「MIK III 6283」，頂部繪有光明童女、耶穌坐像，下方是摩尼教選民像。]]
摩尼教是發源於[[薩珊帝國|薩珊帝國]]境內的一種[[諾斯底主義|諾斯底]]宗教，於公元3世紀由波斯人[[摩尼|摩尼]]所創，歷來被視為基督教的一門[[異端#基督教的異端|異端教派]]。<ref>{{cite web |url=http://www-bcf.usc.edu/~sbriggs/Britannica/manichaeism.htm |title=Manichaeism |author=<!--Staff writer(s); no by-line.--> |website=[[大英百科全書|www-bcf.usc.edu]] |language=en |access-date=2018-12-22}}</ref>據《[[佛祖統紀|佛祖統紀]]》所載，[[武則天|武則天]][[延載|延載]]元年（694），波斯人拂多誕持摩尼教經書來朝。摩尼教中的女性地位甚高，武則天可能因此對其產生好感而優待摩尼教。<ref>{{cite book |author=林悟殊 |date=2005 |chapter=歷史篇 五 唐代三夷教的社會走向 |title=《中古三夷教辨證》 |url= |location=北京 |publisher=中華書局 |page= |isbn= |author-link=}}</ref>後因所譯經書《[[摩尼光佛教法儀略|摩尼光佛教法儀略]]》借用大量佛教術語被[[唐玄宗|唐玄宗]]判定為「本是邪見，妄稱佛教，誑惑黎元」而遭到禁斷。<ref>{{cite book |author=[[杜佑|杜佑]] |date=801 |chapter=卷四十 |title=《[[通典|通典]]》 |url= |location= |publisher= |page= |isbn= |author-link=}}</ref>[[安史之亂|安史之亂]]後，回鶻[[牟羽可汗|牟羽可汗]]立摩尼教為國教。憑藉回鶻的政治實力，該教開始在中原地區廣泛傳播，並建立摩尼教寺院「大雲光明寺」。<ref>{{cite book |author=[[志磬|志磬]] |date=1269 |chapter=卷41 |title=《[[佛祖統紀|佛祖統紀]]》 |url= |location= |publisher= |page= |isbn= |author-link=}}</ref><ref>{{cite book |author=[[王溥_(宋朝)|王溥]] |date=961 |chapter=卷19 |title=《[[唐會要|唐書會要]]》 |url= |location= |publisher= |page= |isbn= |author-link=}}</ref>直到唐武宗會昌毀佛，摩尼教亦受連纍而被禁。該教此後轉向中國南方沿海地區，如[[福建省|閩]][[浙江省|浙]]一帶，在民間祕密流傳，並漸與其他宗教結合，歷[[五代十國|五代]][[宋朝|宋]][[元朝|元]]仍不衰。宋朝以後改稱明教，被當時的統治者斥為「食菜事魔」。「食菜」是因摩尼教拒絕葷腥，但其教徒並非只食蔬菜，於他們而言，瓜果才是最理想的食物。「事魔」則是敵視摩尼教的人對其之貶稱，言其事奉妖魔。但其實該教的修持非常嚴格，除了堅持素食外，[[陳垣|陳垣]]亦指出：「摩尼教治己極嚴，待人極恕，自奉極約，用財極公，不失為一道德宗教。」因而該教曾頗受民衆歡迎。<ref>{{cite web |url=http://zsdh.library.sh.cn:8080/FCKeditor/filemanager/upload/jsp/UserImages/1266647150796.pdf |title=明教：扎根中國的摩尼教 |author=林悟殊 |website=zsdh.library.sh.cn |access-date=2018-12-29}}</ref>摩尼教還很重視經圖並列的宣教方式，教主摩尼不僅親自撰寫[[摩尼七經|正典七經]]，還親手繪製《[[大二宗圖|大二宗圖]]》。現今傳世的摩尼教藝術品，如手稿插圖、繪幡、壁畫、絹畫掛軸等，用色鮮豔，華麗耀目。

== 補說 ==
{{Main|唐朝伊斯蘭教}}
除了三夷教之外，[[伊斯蘭教|伊斯蘭教]]亦於唐朝時傳播到中國。根據中國穆斯林的傳統說法，[[穆罕默德|穆罕默德]]的舅父[[賽義德·本·阿比·瓦卡斯|賽義德·本·阿比·瓦卡斯]]出使中國謁見唐高宗，位於[[廣州|廣州]]的唐朝淸眞寺[[懷聖寺|懷聖寺]]的修建就與他有關。
