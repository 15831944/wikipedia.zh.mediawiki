'''唐浩明'''（{{bd|1946年|10月||}}），生于[[湖南|湖南]][[衡阳|衡阳]]。毕业于[[华中师范大学|华中师范大学]]。曾任[[湖南省作协|湖南省作协]]主席。长期致力于写作历史小说，代表作有《[[曾国藩_(小说)|曾国藩]]》、《[[张之洞|张之洞]]》、《[[杨度|杨度]]》等<ref>{{cite web|url=http://www.cq.xinhuanet.com/2010/2010-03/30/content_19386361.htm|title=《唐浩明》|accessdate=2010年3月30日|work=|publisher=新华网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20131204001136/http://www.cq.xinhuanet.com/2010/2010-03/30/content_19386361.htm|archivedate=2013年12月4日}}</ref>。

== 简介 ==
唐浩明在1946年時出生于湖南衡阳，这也是明末大哲学家[[王夫之|王夫之]]的故乡。之後在1970年時，他毕业于华中水利学院，学习农田水利，所以毕业后在农场和水电站做技术员<ref name="multiple1">{{cite web|url=http://www.sznews.com/epaper/szwb/content/2006-11/04/content_509039.htm|title=《唐浩明：小说笔法谈古论今》|accessdate=时间|work=刘雨虹|publisher=深圳晚报|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20131202234708/http://www.sznews.com/epaper/szwb/content/2006-11/04/content_509039.htm|archivedate=2013-12-02}}</ref>。

1966年，[[文革|文革]]开始之后，唐浩明潜心阅读了《[[二十四史|二十四史]]》<ref name="multiple1">{{cite web|url=http://www.sznews.com/epaper/szwb/content/2006-11/04/content_509039.htm|title=《唐浩明：小说笔法谈古论今》|accessdate=时间|work=刘雨虹|publisher=深圳晚报|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20131202234708/http://www.sznews.com/epaper/szwb/content/2006-11/04/content_509039.htm|archivedate=2013-12-02}}</ref>。而在1978年恢复高考後，唐浩明考入了华中师范大学中文系，毕业之后在[[岳麓书社|岳麓书社]]从事编辑工作<ref>{{cite web|url=http://www.zg0734.cn/main/showMR.asp?id=336&type=15|title=《唐浩明：敬畏历史 感悟智慧》|accessdate=|work=|publisher=中国衡阳网|language=zh-hans|deadurl=yes|archiveurl=https://archive.is/20130424173344/http://www.zg0734.cn/main/showMR.asp?id=336&type=15|archivedate=2013-04-24}}</ref>。

到了唐浩明40岁那年（1986年），他已研究曾国藩二十几年，在厚重的资料下写成了《曾国藩》一書而一举成名，并且奠定了他在文学界和史学界的地位<ref name="multiple1">{{cite web|url=http://www.sznews.com/epaper/szwb/content/2006-11/04/content_509039.htm|title=《唐浩明：小说笔法谈古论今》|accessdate=时间|work=刘雨虹|publisher=深圳晚报|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20131202234708/http://www.sznews.com/epaper/szwb/content/2006-11/04/content_509039.htm|archivedate=2013-12-02}}</ref>。後來在1993年，唐浩明被批准享受国务院颁发的政府特殊津贴<ref name="multiple2">{{cite web |url=http://news.sina.com.cn/s/2003-07-04/08111276022.html|title=《唐浩明：从来没用过手机》|accessdate=2003年7月4日|work=袁欣|publisher=新浪网|language=zh-hans}}</ref>。

他於1999年4月当选为湖南省海外联谊会第三届理事会理事。

而在完成《张之洞》一書后，唐浩明宣布封笔<ref name="multiple3">{{cite web |url=http://news.163.com/07/1128/05/3UC4T612000120GU.html|title=《唐浩明不再创作历史长篇》|accessdate=2007-11-28 |work=|publisher=网易|language=zh-hans}}</ref>。

2016年4月，不再担任[[湖南省作家协会|湖南省作家协会]]主席职务，由[[王跃文|王跃文]]接任<ref>{{cite news|title=王跃文任湖南作协党组副书记，提名为作协主席人选|url=http://www.thepaper.cn/newsDetail_forward_1454745|accessdate=2016-04-10|newspaper=澎湃新闻}}</ref>。

== 奖项荣誉 ==
*1994年被评为首届全国中青年优秀编辑<ref name="multiple2">{{cite web |url=http://news.sina.com.cn/s/2003-07-04/08111276022.html|title=《唐浩明：从来没用过手机》|accessdate=2003年7月4日|work=袁欣|publisher=新浪网|language=zh-hans}}</ref>。

*1997年获“第三届国家图书奖”<ref name="multiple2">{{cite web |url=http://news.sina.com.cn/s/2003-07-04/08111276022.html|title=《唐浩明：从来没用过手机》|accessdate=2003年7月4日|work=袁欣|publisher=新浪网|language=zh-hans}}</ref>。

*1998年获“八五”期间优秀长篇小说奖<ref name="multiple2">{{cite web |url=http://news.sina.com.cn/s/2003-07-04/08111276022.html|title=《唐浩明：从来没用过手机》|accessdate=2003年7月4日|work=袁欣|publisher=新浪网|language=zh-hans}}</ref>。

*《曾国藩》2003年获得首届[[姚雪垠|姚雪垠]]长篇历史小说奖<ref name="multiple3">{{cite web |url=http://news.163.com/07/1128/05/3UC4T612000120GU.html|title=《唐浩明不再创作历史长篇》|accessdate=2007-11-28 |work=|publisher=网易|language=zh-hans}}</ref>。
