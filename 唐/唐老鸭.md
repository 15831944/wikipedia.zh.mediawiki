'''唐老鴨'''（英語：Donald Duck）是[[迪士尼|迪士尼]]所創的經典[[動畫|動畫]]人物之一。牠首次出現於1934年、官方生日是1934年6月9日。牠有橙黃色的嘴、腳和蹼，品種為北京鴨<ref>{{Cite web|url=https://kmweb.coa.gov.tw/subject/ct.asp?xItem=90704&ctNode=2219&mp=141&kpi=0&hashid=|title=唐老鴨到底是什麼鴨呢？|accessdate=|author=劉秀洲|date=2007/ 12/ 27|publisher=行政院農業委員會}}</ref>，身上穿的是Fauntleroy水手裝，但卻沒有穿褲子，愛吃胡萝卜。女友為[[黛絲鴨|黛絲鴨]]（英語：Daisy Duck）。

唐老鴨的全名是'''Donald Fauntleroy Duck'''（唐納德·方特洛伊·達克）。此名字出現於1942年的動畫''Donald Gets Drafted''，印在牠的水手帽上。在迪士尼的網頁裡，亦有提及牠的全名。<ref>http://disney.go.com/vault/archives/characterstandard/donald/donald.html</ref>

唐老鴨經常性格暴躁、沮喪時容易放棄，但大部分時間都很隨和、容易滿足，講話口齒不清。

== 配音員 ==

自1934年唐老鴨於[[迪士尼|迪士尼]]動畫亮相開始，{{link-en|Clarence Nash|Clarence Nash}}一直擔任唐老鴨專屬配音員，直至逝世。1985年Nash因[[白血病|白血病]]辭世後，改由{{link-en|Tony Anselmo|Tony Anselmo}}擔任配音。

中國大陸的華語版本早期曾由[[張涵予|張涵予]]及[[李揚|李揚]]擔任配音。

台灣版由[[謝文德|謝文德]]、[[蔡閨|蔡閨]]、[[陳志邦|陳志邦]]擔任配音。

日本版由[[山寺宏一|山寺宏一]]擔任配音。

== 動畫中的唐老鴨 ==
=== 早期 ===
1934年6月9日，唐老鴨首次亮相於《糊塗交響樂》系列卡通的中的《聰明的小母雞》。

1936年，唐老鴨於被重新設計，變得稍加豐滿、圓渾。牠亦開始有單獨演出，第一部片是1937年1月9日由Ben Sharpsteen繪製的卡通''{{link-en|Don Donald|Don Donald}}''。牠的女友[[黛絲鴨|黛絲鴨]](Daisy Duck)亦在本片以Donna Duck之名首次出現。

唐老鴨的三個姪兒{{link-en|休依、杜依、路依|Huey, Dewey and Louie}}則於1年後，即1938年4月15日，在''Donald's Nephews''首次在大銀幕上亮相。此影片由傑克·金（Jack King）執導。

除了有自己擔當主演的單元外，在1935年播出的米奇的修車站中與[[米老鼠|米老鼠]]及[[高飛狗|高飛狗]]第一次做三人搭配演出，後續到1937年的兩年間三人經常一起演出，到1938年後開始跟[[高飛狗|高飛狗]]做撘檔演出，或是和[[奇奇與蒂蒂|奇奇與蒂蒂]]做對手戲，直到[[1999|1999]]年的米老鼠新傳中三人才繼續合作演出。

== 漫畫中的唐老鴨 ==

唐老鴨的漫畫週刊及月刊在許多[[歐洲|歐洲]]國家大受歡迎，首先風行於[[挪威|挪威]]和[[芬蘭|芬蘭]]，其後有[[丹麥|丹麥]]、 [[德國|德國]]、[[義大利|義大利]]、[[荷蘭|荷蘭]]及[[瑞典|瑞典]]。它們大多數由[[華特迪士尼公司|華特迪士尼公司]]位於義大利的分公司及位於丹麥、挪威、芬蘭和瑞典的[[Egmont_(media_group)|Egmont]]生產和出版。

下列地方亦有唐老鴨漫畫連載—— [[澳大利亞|澳大利亞]]、[[奧地利|奧地利]]、[[比利時|比利時]]、[[巴西|巴西]]、[[保加利亞|保加利亞]]、[[中國|中國]]（内地和[[香港|香港]]）、[[哥倫比亞|哥倫比亞]]、[[捷克共和國|捷克共和國]]、[[丹麥|丹麥]]（包括[[法羅群島|法羅群島]]）、[[埃及|埃及]]、[[愛沙尼亞|愛沙尼亞]]、[[芬蘭|芬蘭]]、[[法國|法國]]、[[德國|德國]]、[[希臘|希臘]]、[[蓋亞那|蓋亞那]]、[[匈牙利|匈牙利]]、[[冰島|冰島]]、[[印度|印度]]、[[印尼|印尼]]、[[以色列|以色列]]、[[義大利|義大利]]、[[拉脫維亞|拉脫維亞]]、[[立陶宛|立陶宛]]、[[荷蘭|荷蘭]]、[[挪威|挪威]]、[[波蘭|波蘭]]、[[葡萄牙|葡萄牙]]、[[羅馬尼亞|羅馬尼亞]]、[[俄羅斯|俄羅斯]]、[[沙烏地阿拉伯|沙烏地阿拉伯]]、[[斯洛維尼亞|斯洛維尼亞共和國]]、[[西班牙|西班牙]]、[[瑞典|瑞典]]、[[泰國|泰國]]、[[土耳其|土耳其]]、[[英國|英國]]、[[美利堅合眾國|美國]]和前[[南斯拉夫|南斯拉夫]]。

=== 早期發展 ===
1937年，一位名為Mondadori的義大利出版商首度為唐老鴨打造個人漫畫。該漫畫有18頁、由{{link-en|Federico Pedrocchi|Federico Pedrocchi}}撰寫。唐老鴨成為主角，並第一次以冒險者之姿態登場，有別於以往單純的喜劇角色。同時，[[英格蘭|英格蘭]]的[[Fleetway|Fleetway]]也開始出版以唐老鴨為主角的漫畫故事。

=== 在Taliaferro筆下的發展 ===
從1938年2月2日開始，美國的漫畫家[[查爾斯·阿弗雷德·塔里亞菲羅|查爾斯·阿弗雷德·塔里亞菲羅]]和{{link-en|Bob Karp|Bob Karp}}每日發表一篇的唐老鴨短篇漫畫。塔里亞菲羅負責作畫，Karp則負責故事。

翌年，他們更發表星期日短篇漫畫，為唐老鴨的世界創作更多角色。他養了一隻[[聖班納_(狗)|聖班納]]，名叫[[Bolivar_(dog)|Bolivar]]。鴨家族亦開始成長，多了堂叔[[史高治|史高治]]和祖母[[Elvira_Coot|Elvira Coot]]。唐老鴨早期的墨西哥女朋友名叫[[Donna_Duck|Donna]]，於1940年由[[黛絲鴨|黛絲鴨]]所取代。

=== 在Barks筆下的發展 ===
1942年，{{link-en|Western Publishing|Western Publishing}}開始發展關於唐老鴨和迪士尼其他角色的原創漫畫。新發行商除了[[卡尔·巴克斯|卡尔·巴克斯]]和{{link-en|Jack Hannah|Jack Hannah}}外，{{fact|就不信任其他新的畫師|來源請求}}。Barks在很多故事中都重覆用了[[尋寶|尋寶]]這個主題。

Barks很快就負擔起唐老鴨漫畫的主要發展工作，包括寫故事和繪畫。在他筆下，漫畫和動畫中的唐老鴨的性格分別愈來愈大，變得愈來愈愛冒險、更雄辯，亦沒有那麼情緒化。

Barks給了唐老鴨一架電單車，出現於1985年出版的[[A_1934_Belchfire_Runabout|A 1934 Belchfire Runabout]]其中一個故事。<ref>http://coa.inducks.org/story.php/0/CB+OIL+127</ref>

== 褲子問題 ==
[[芬蘭|芬蘭]]曾一度禁止唐老鴨，有說是因為他沒穿褲子<ref>[http://www.snopes.com/disney/films/finland.htm]</ref>。

== 野心 ==
雖然早期較不明顯，但唐老鴨有著想搶下[[米老鼠|米老鼠]]在迪士尼中作為吉祥物的野心，尤其在[[米老鼠新傳|米老鼠新傳]]的片頭曲結束前最為明顯，甚至有些單元中會見[[米老鼠|米老鼠]]消失後自己會多受歡迎的妄想，這點與[[樂一通|樂一通]]的[[達菲鴨|達菲鴨]]極為相像。

== 迪士尼以外 ==
1947年[[俄勒岡大學|俄勒岡大學]]'''（英文：University of Oregon）'''運動總監Leo Harris取得[[華特·迪士尼|華特·迪士尼]]本人的同意，正式讓唐老鴨成為學校的吉祥物。1974年，迪士尼公司正式授權俄勒岡大學，可於與運動相關的地點或事件使用唐老鴨圖像。1991年，此授權更延至學校相關商品。

1984年，唐老鴨50歲生日紀念，俄勒岡大學頒發榮譽學位給唐老鴨，近4000位影迷來到當地機場接機與參與儀式，數千名當地居民連署贈送賀卡歡迎，這些文件目前都存放於迪士尼公司。

唐老鴨的名字和圖像也出現於其他商業產品，例如1940年由Citrus World出品的唐老鴨的橙汁。
