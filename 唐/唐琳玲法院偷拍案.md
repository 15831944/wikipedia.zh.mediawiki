=== 法例 ===
在香港法庭中，嚴禁拍照及拍片屬旁聽人士「首要守則」。執業[[大律師|大律師]][[陸偉雄|陸偉雄]]解釋：「嚴禁[[拍攝|拍攝]]、[[繪畫|繪畫]]或{{link-en|法庭素描|Courtroom sketch|素描}}，為了尊重法庭，和防止[[陪審團|陪審團]]身份外洩，受到外來影響或壓力，令審訊不公平。」<ref>{{Cite news| url =http://orientaldaily.on.cc/cnt/news/20180224/mobile/odn-20180224-0224_00176_003.html| title =法庭「守則」多 嚴禁拍照繪畫| work =《[[東方日報|東方日報]]》| date =2018年02月24日| deadurl =no| archiveurl =https://web.archive.org/web/20180527201539/http://orientaldaily.on.cc/cnt/news/20180224/mobile/odn-20180224-0224_00176_003.html| archivedate =2018年5月27日| df =}}</ref>

根據[[香港法例|香港法例]]相關規定和[[普通法|普通法]]判例，未經准許在法院內偷拍屬藐視法庭的行為，視乎嚴重性可被罰款或監禁。一經發現，有以下兩種方式處理：
* 如不抗辯，毋須審訊，可由主審法官或裁判官直接引用《簡易程序治罪條例》處理，判處罰款二千元<ref>{{cite news|title=與官爭論裁藐視法庭　上訴男得直|url=https://hk.news.appledaily.com/local/daily/article/20170923/20161413|work=蘋果日報|date=2017-09-23|deadurl=no|archiveurl=https://web.archive.org/web/20180612140547/https://hk.news.appledaily.com/local/daily/article/20170923/20161413|archivedate=2018-06-12|df=}}</ref><ref>{{cite news|title=法庭「守則」多 嚴禁拍照繪畫|url=http://orientaldaily.on.cc/cnt/news/20180224/mobile/odn-20180224-0224_00176_003.html|work=東方日報|date=2018-02-24|deadurl=no|archiveurl=https://web.archive.org/web/20180527201539/http://orientaldaily.on.cc/cnt/news/20180224/mobile/odn-20180224-0224_00176_003.html|archivedate=2018-05-27|df=}}</ref>。
*情節嚴重者可被控以[[藐視法庭|藐視法庭]]。

== 藏假名片身份成疑 ==

唐琳玲在法庭上經常提到自己的澳洲背景，向香港法官指自己曾居於澳洲，認為自己的英語帶有澳洲口音，但因為喜歡香港而放棄移居澳洲。《[[香港01|香港01]]》在查閱澳洲[[新南威爾士州|新南威爾士州]]地區法院案件資料，並經過澳洲法院人員核實後，發現有一名生於1983年11月，英文名為「TANG Lin Ling」的女子，曾於2012年11月15日被控以兩項襲擊正執行任務警員罪及一項拒絕及防礙警察執行任務罪，新南威爾士州地區法院於2013年2月13日裁定被告三項控罪全部罪名成立，每項控罪各被判250[[澳元|澳元]]罰款及83元訴訟費，三項控罪合共罰款約750澳元及訴訟費249澳元，全部約為999澳元，折合約為6,000港元，被告曾經提出上訴但不獲接納。該名澳洲法院的被告，姓名英文譯音「TANG Lin Ling」與唐琳玲相同，二人今年均為35歲，而且與唐琳玲自稱的澳洲生活時間相符，但未能確定該名被告就是唐琳玲本人<ref>{{cite news |title=澳洲法庭紀錄 「唐琳玲」5年前曾襲警被罰999澳元 |url=https://www.hk01.com/01%E5%81%B5%E6%9F%A5/194262/%E5%BA%AD%E4%B8%8A%E6%8B%8D%E7%85%A7-%E6%BE%B3%E6%B4%B2%E6%B3%95%E5%BA%AD%E7%B4%80%E9%8C%84-%E5%94%90%E7%90%B3%E7%8E%B2-5%E5%B9%B4%E5%89%8D%E6%9B%BE%E8%A5%B2%E8%AD%A6%E8%A2%AB%E7%BD%B0999%E6%BE%B3%E5%85%83 |work=香港01 |date=2018-06-04 |deadurl=no |archiveurl=https://web.archive.org/web/20180612163306/https://www.hk01.com/01%E5%81%B5%E6%9F%A5/194262/%E5%BA%AD%E4%B8%8A%E6%8B%8D%E7%85%A7-%E6%BE%B3%E6%B4%B2%E6%B3%95%E5%BA%AD%E7%B4%80%E9%8C%84-%E5%94%90%E7%90%B3%E7%8E%B2-5%E5%B9%B4%E5%89%8D%E6%9B%BE%E8%A5%B2%E8%AD%A6%E8%A2%AB%E7%BD%B0999%E6%BE%B3%E5%85%83 |archivedate=2018-06-12 |df= }}</ref>。

== 影響 ==
連同這案件，香港在2018年4月至5月間發生多宗法庭內偷拍事件，影響到香港法庭內使用攝錄器材的規定，變得更為嚴格。[[香港終審法院|終審法院]]首席法官在同年6月21日發出《在涉及陪審團的法庭程序中於法庭內使用流動電話及其他器材》新規定，訂明從2018年7月19日起，無論庭內是否正在聆訊，除獲法官或司法人士准許，公眾人士不可在有[[陪審團|陪審團]]參與研訊的法庭內使用手機或任何有攝錄功能的器材，並須將之放入袋內<ref name="stheadline0622"/>。[[司法機構|司法機構]]稱陪審員乃妥善執行司法工作重要的一環，必須確保他們不受任何干預、壓力或分散他們注意力的事情所影響，因此「禁止攝影或錄影」的措施便更為重要<ref name="stheadline0622">{{Cite web |url=http://hd.stheadline.com/news/daily/hk/678284/ |title=陪審團參與案件 庭上禁用手機 |date=2018-06-22 |publisher=頭條日報}}</ref>。司法機構並加強支援措施，提醒法庭使用者禁止攝影等規定<ref>{{Cite web |url=http://www.hkcd.com.hk/content/2018-06/22/content_3740659.htm |title=旁聽陪審團禁用手機 |date=2018-06-22 |publisher=香港商報}}</ref><ref>{{Cite web |url=https://news.mingpao.com/ins/instantnews/web_tc/article/20180621/s00001/1529551737410 |title=涉陪審員庭訊禁用手機　下月19日生效 |date=2018-06-22 |publisher=明報}}</ref>。
