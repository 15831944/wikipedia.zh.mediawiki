'''唐冠华'''（{{bd|1989年||||}}），生于中国[[青岛|青岛]]，概念艺术家，全球生态社区网络亚太联盟（GEN）中国区首席代表。因在青岛崂山森林中进行四年自给自足的生活实验而闻名。<ref>{{Cite video |language = 中文 |author = 中国中央电视台新闻频道 |title = 唐冠华：当代国人需要换一种生活方式 |publisher = 央视国际  |pages = 新闻1+1栏目 |date = 2014-05-01 | url=http://v.ifeng.com/news/opinion/2014005/01c63392-927c-4b8b-b88c-a91b87427af0.shtml/ }}</ref>
创办中国自给自足实验室和家园计划。获得由《城市画报》颁发的2014年中国85后人物奖。<ref>{{cite news |language = 中文 |author = 杨凡 |title = 《城市画报》15周年特刊：85后精英榜 |publisher = 城市画报 |pages = 99 |date = 2014-10 |accessdate = 2014-12 | url=http://www.aiweibang.com/yuedu/news/2593766.html/}}</ref>
2018年获联合国地球卫士青年奖提名。<ref>{{Cite web|url=http://sn.people.com.cn/n2/2018/0622/c378305-31731248.html|title=“地球卫士青年奖”决赛名单出炉--陕西频道--人民网|accessdate=2018-08-07|last=L_104110|work=sn.people.com.cn}}</ref>他通过立足自然界身体力行近乎边缘化的实际行动和独立思考为世界上少数群体争取应有的尊严和价值。被媒体称作中国[[共识社区|共识社区]]的奠基人。<ref>{{cite news|language =中文|author =刘素楠|coauthors =邓穗琪|title =青年艺术家的“下乡实验室”|publisher =中国财富杂志社|pages =60|date =2014-01|accessdate =2014-12|url =http://hope.huanqiu.com/talk/2014-01/4794736.html/|deadurl =yes|archiveurl =https://web.archive.org/web/20150522172704/http://hope.huanqiu.com/talk/2014-01/4794736.html|archivedate =2015-05-22}}</ref>

==家園計劃==
家园计划（AnotherLand）是一个非营利性[[公益|公益]]计划。<ref>{{cite news |language = 中文 |author = 张安奇  |title = 一个自给自足的梦想
 |publisher = 河北青年报 |pages = 河北青年报A06版 |date = 2014年4月20日}}</ref>

===自给自足實驗室===
中国自给自足实验室（CSS LAB）是一个民营可持续技术实验室。<ref>{{cite news |language = 中文 |author = 张安奇  |title = 一个自给自足的梦想
 |publisher = 河北青年报 |pages = 河北青年报A06版 |date = 2014年4月20日}}</ref>
由唐冠华与邢振夫妇创办。隶属于家园计划（AnotherLand）。通过5年的在地生活实验，制作和发行《独立之道》系列短片和出版物--家庭生活必需品制作指南。<ref>{{Cite video |language = 中文 |author = 中国中央电视台新闻频道 | |title = 唐冠华：当代国人需要换一种生活方式 |publisher = 央视国际  |pages = 新闻1+1栏目 |date = 2014-05-01}}</ref> 为中国[[共识社区|共识社区]]运动做出营造技术储备。<ref>{{cite news |language = 中文 |author = 刘素楠 |coauthors = 邓穗琪 |title = 青年艺术家的“下乡实验室” |publisher = 中国财富杂志社 |pages = 60 |date = 2014-01 |accessdate = 2014-12}}</ref>
