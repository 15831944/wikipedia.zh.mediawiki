'''關野貞'''（{{bd|1868年|1月9日|1935年|7月29日}}），[[日本|日本]][[建築史|建築史]]家、[[東京大學|東京大學]]教授。致力於文化遺產的保存而為人所知。

== 生平 ==
生於高田藩士家。經舊制第一高等學校，在1895年畢業於[[東京帝國大學|東京帝國大學]]工部大學造家學科（建築學科）。畢業論文是[[平等院|平等院]]之研究。在[[辰野金吾|辰野金吾]]的指揮下參加[[日本銀行|日本銀行]]的工事。在古社寺保存法被立案的1896年，因為[[伊東忠太|伊東忠太]]的勸說而成為內務省技師、[[奈良縣|奈良縣]]技師，調查奈良的古建築、判定建築年代。另外，在1899年發現[[平城宮|平城宮]]址。

1901年為東大助教授，1910年為教授。其間，進行平城宮址的研究，之後被授予工學博士。平城宮址在1922年，被指定為國家史蹟。
[[File:140927_Research_Center_for_Buddhist_Art_Materials_of_Nara_National_Museum_Nara_Japan04n.jpg|thumb]]佛教美術資料研究中心]]

1910年因為受到[[朝鮮總督府|朝鮮總督府]]的委託，屢次進行[[朝鮮半島|朝鮮半島]]和[[中國|中國]]的古建築調查，並致力於保護。以《朝鮮古蹟圖譜》，從[[法蘭西學會|法蘭西學會]]得到[[儒蓮獎|儒蓮獎]]。

1906年被派遣到[[清國|清國]]，著手於中國建築的研究。1929年在東方文化學院東京研究所，研究中國歴代[[帝王|帝王]][[陵墓|陵墓]]的研究、[[遼|遼]][[金|金]]時代的建築及其[[佛像|佛像]]。

== 著作 ==
* 《[[支那文化史迹|支那文化史蹟]]》
* 〈支那山東省に於ける漢代墳墓の表飾〉（1916年）
* 《朝鮮古蹟図譜》15冊（1916年-1935年）
* 《韓国建築調査報告》（這些資料對現在[[韓國|韓國]]文化遺產的保存、修復和復元上非常重要。）
* 《熱河》4冊（1934年）

== 逸聞 ==
* 出於重視建築樣式史的立場，而主張[[法隆寺|法隆寺]]非再建論、進行論爭。
* 建築的作品雖然很少，但在[[奈良公園|奈良公園]]內有舊奈良縣物產陳列所（1902年竣工，現[[奈良國立博物館|奈良國立博物館]]佛教美術資料研究中心），被指定為重要文化遺產。
* 兒子[[關野克|關野克]]也是建築史家。
