'''[[關閘|關閘]]總站'''（{{lang-pt|'''PORTAS DO CERCO/TERMINAL'''}}、{{lang-en|'''Border Gate Terminal'''}}），位於[[澳門|澳門]][[花地瑪堂區|花地瑪堂區]]關閘廣場地底的一個終點站，貼近[[中國大陸|中國大陸]]第一大陸路口岸──[[關閘邊檢大樓|關閘]]—[[拱北口岸|拱北]]口岸<ref>[http://gongbei.customs.gov.cn/publish/portal223/tab36960/module86401/info206629.htm 中國海關網站，拱北口岸：拱北海關“元旦”節期驗放進出境旅客71萬人次]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，故人流非常高，是澳門最繁忙的終點站，亦是澳門首個設於地底及室內的終點站；在[[颱風天鴿|颱風天鴿]]襲澳前共有17條路線以本站為終點站和7條路線為中途站，是澳門所有終點站中提供最多路線選擇之一。受[[颱風天鴿|颱風天鴿]]影響，該站至2018年12月15日前一直停用，原停靠該站之路線臨時遷至鄰近街道上落客。2018年12月15日起恢復運作<ref name="macao1215"/>。

== 歷史沿革 ==
*本站確實創立日期沒有資料，但在現時兩間巴士公司（[[澳門新福利公共汽車有限公司|新福利]]及[[澳巴|澳巴]]）成立前（即1988年前），[[澳門巴士5路線|5路線]]已經是以此站為目的地。
*關閘巴士總站在搬到臨時總站及'''關閘地下客運總站'''啟用前，一直位於關閘口岸前的空地（即現時關閘總站隧道的位置）。
*2001年4月23日，[[土地工務運輸局|土地工務運輸局]]批准[[澳門新福利公共汽車有限公司|澳門新福利公共汽車有限公司]][[澳門巴士3A路線|3A]]路線巴士由[[外港碼頭總站|外港碼頭總站]]延長服務至[[黑沙環|黑沙環]]及'''關閘總站'''<ref>新華澳報，01版，三線巴士永久性改線，2001年4月24日</ref>。

*2002年12月31日，為配合[[關閘廣場|關閘廣場]]重整及興建地下巴士總站工程，兩巴停用位於關閘廣場地面巴士總站，搬遷至位於台山口的關閘臨時巴士總站（即現時[[發財車|發財車]]停泊區的位置）。<ref>華僑報，24版，本澳兩巴將使用設於鄰近之臨時總站 關閘巴士總站明搬遷，2002年12月30日</ref>

*2004年11月23日，關閘地下巴士總站正式投入服務，並服務至今。新巴士總站由時任[[澳門特別行政區政府|澳門特別行政區政府]][[運輸工務司司長|運輸工務司司長]][[歐文龍|歐文龍]]揭幕<ref>澳門日報，A01版，新關閘巴士站運作暢順，2004年11月24日</ref>。

*2008年9月24日，[[颱風黑格比_(2008年)|颱風黑格比]]襲澳，受到連場暴雨及海水倒灌影響，關閘總站內出現嚴重水浸，要全面封閉。經連日清理積水、清潔消毒及接駁照明系統等工作後，發現站內通風系統損壞，[[交通事務局|交通事務局]]及[[土地工務運輸局|土地工務運輸局]]評估後認為通風系統在未完全修復前，為免影響公眾健康，不對外開放，只供巴士作停泊之用<ref name="華僑報，11版，澳聞，關閘總站暫不接客，2008年9月27日">華僑報，11版，澳聞，關閘總站暫不接客，2008年9月27日</ref>。

*2008年12月7日，交通事務局實施第一階段巴士路線調整，[[澳門巴士1路線|1路線]]由[[筷子基總站|筷子基總站]]延長服務至'''關閘總站'''，另外新增加[[澳門巴士25X路線|25X路線]]，由'''關閘總站'''往來[[望德聖母灣大馬路|望德聖母灣大馬路]]；配合上述改動，同日起關閘總站內各路線停靠車道亦作出了局部調整<ref>[http://www.gcs.gov.mo/showNews.php?PageLang=C&DataUcn=34099&Member=0 交通事務局，巴士7路線新增重整周日實施，2008年12月5日18:32:00]</ref>。

*2011年1月，為加強站內空氣流通及採光，因此在總站頂部開鑿八個通風口。

*2014年12月13日，交通事務局調整關閘總站部份巴士路線包括[[澳門巴士1路線|1]]、[[澳門巴士3A路線|3A]]、[[澳門巴士10路線|10]]及[[澳門巴士10X路線|10X]]路線的停靠位置，具體安排如下。<ref>[http://www.dsat.gov.mo/bus/tc/bus_news_detail.aspx?a_id=2685 兩口岸巴士站調整路線靠站位置]</ref>

*2017年2月25日，為減輕此總站的負荷，[[澳門巴士51路線|51]]及[[澳門巴士59路線|59]]兩線均不停靠此站，改為停靠新設的「關閘廣場」站，同時[[澳門巴士MT4路線|MT4]]延長至此站。
*2017年4月18日，[[澳門巴士51A路線|51A]]路線往蝴蝶谷大馬路方向增停此站。
*2017年4月29日，交通事務局為優化關閘總站的停靠安排，由4月29日(周六)首班車起，3A、10、10B、10X、18及27(往黑沙環)路線調整停靠之車道，同步往青洲方向的27路線亦改為停靠關閘廣場東側的「關閘廣場」站。<ref>[http://www.dsat.gov.mo/bus/site/bus_news_detail.aspx?a_id=3870 關閘總站6條巴士路線周六起調整停靠車道]</ref>
*2017年8月23日，受到[[颱風天鴿|颱風天鴿]]影響，本站出現水浸，至2018年12月15日前一直無法使用。
*2018年11月29日，交通事務局表示，本站暫定於12月15日局部開放。1、3、10、25、25B、AP1、MT4路線作為該線總站；17、27、30、34、51A路線途經該站；3X路線作為該線落客站，在「關閘馬路」站上客。
*2018年12月15日，耗資1.7億元整改後，本站恢復運作，首階段共8條路線設為终點站，5條路線途經此站。<ref name="macao1215">{{Cite news |url=http://www.macaodaily.com/html/2018-12/16/content_1317524.htm |title=關閘巴站重開運作暢 |date=2018-12-16 |work=澳門日報}}</ref>

== 設施 ==
* 備有[[空氣調節|空氣調節]]的[[洗手間|洗手間]]。
* [[澳門電訊|澳門電訊]][[電話亭|電話亭]]。
* [[澳門新福利公共汽車有限公司|新福利]]及[[澳門公共汽車股份有限公司|澳巴]]員工休息室。
* [[澳門新福利公共汽車有限公司|新福利]]及[[澳門公共汽車股份有限公司|澳巴]]路線詢問處。
* [[關閘廣場|關閘廣場]]往來地下終點站的[[扶手電梯|扶手電梯]]3組、[[升降機|升降機]]2部、公用[[樓梯|樓梯]]5條。
* 保安控制中心。

== 嚴重事故 ==
關閘總站內曾發生多宗事故，包括撞車，更有巴士在總站內起火爆炸。另外，2008年受[[颱風黑格比|颱風黑格比]]影響，關閘總站更發生嚴重水浸，要封閉多日。
*2007年2月9日：[[澳巴|澳巴]][[澳門巴士25路線|25路線]]一部產自[[深圳|深圳]]的[[五洲龍|五洲龍]]客車，在站內起火並發生爆炸，由於事發時正直尖峰時間，'''近300人需要緊急逃生'''。巴士總站亦首次因事故而需要關閉，極多乘客受到影響<ref>[http://news.sina.com/hk/chinapress/103-000-101-106/2007-02-09/09551774239.html 新浪新聞，澳巴大火300乘客倉惶逃命，2007年2月9日] {{webarchive|url=https://web.archive.org/web/20071204194138/http://news.sina.com/hk/chinapress/103-000-101-106/2007-02-09/09551774239.html |date=2007年12月4日 }}</ref>。
*{{Fact|2007年2月25日：一輛[[澳門新福利公共汽車有限公司|新福利]][[新福利3A路線|3A]]路線巴士在站內起火，無人受傷，近30乘客疏散。|time=2010-08-14T13:49:26+00:00}}
*2007年3月31日晚上8時左右，[[澳巴|澳巴]][[澳門巴士18路線|18路線]]一部[[日野|日野]]RX4JFKA型號小巴，在'''關閘總站'''內失控，車頭猛烈撞向一條柱墩，導致車上11名乘客受傷。因事故傷者人數眾多，[[澳門消防局|澳門消防局]]要啟動緊急應變機制，派出三輛救護車到場將傷者送院治理；而其中1名女子因受驚過度，在等待送院期間更一度暈倒。肇事巴士車頭損毀嚴重，登車門被撞飛。<ref>澳門日報，2007年4月1日，A01版，澳巴撞柱十一乘客輕傷 等候送院一女子疑受驚過度一度暈厥</ref>

*2008年9月24日，[[颱風黑格比_(2008年)|颱風黑格比]]襲澳，由於受到連場暴雨及海水倒灌影響，關閘總站內出現嚴重水浸，要全面封閉。經連日清理積水、清潔消毒及接駁照明系統等工作後，發現站內通風系統損壞，[[交通事務局|交通事務局]]及[[土地工務運輸局|土地工務運輸局]]評估後認為通風系統在未完全修復前，為免影響公眾健康，不對外開放，只供巴士作停泊之用<ref name="華僑報，11版，澳聞，關閘總站暫不接客，2008年9月27日"/>。

*2010年5月22日下午3時左右，一輛停泊關閘總站內的旅遊巴士因機件故障冒煙，事件中無人受傷。<ref>澳廣視即時新聞，旅遊巴冒煙無人傷，2010年5月22日</ref>

*2017年8月23日，[[強颱風天鴿_(2017年)|颱風天鴿]]襲澳，由於受到海水倒灌影響，關閘總站發生嚴重水浸，由於電力及各種設備故障，導致關閘總站全面關閉，原行經該站的24條巴士路線臨時停靠在關閘周邊站點。整改歷時一年四個月，至翌年12月15日重開。

== 軼事 ==
*關閘總站巴士事故時有發生，且相關司機均具多年駕車經驗，並非“新手”，有司機認為事件過於巧合，認為與鬼怪作祟有關，於是有巴士司機特意在關閘總站拜神祈福，大灑溪錢，並於休息室外設置香爐，以求安心。 <ref>[http://space.qoos.com/?viewnews-157155  2007年11月12日 - 澳門日報《關閘巴士站員工拜神求安》]</ref>

== 爭議（天鴿風災前） ==
===關閘總站通風差及酷熱===
澳門坊間一直批評'''關閘總站'''內通風差非常酷熱，通風欠佳，[[澳門立法會|澳門立法會]]議員[[吳國昌|吳國昌]]及[[陳明金|陳明金]]亦因此提出書面質詢，澳門日報更作頭條報導<ref>[http://space.qoos.com/?action-viewnews-itemid-95452 澳門日報，關閘地下巴士站酷熱難頂，2007年5月27日]</ref>。而[[交通事務局|交通事務局]]局長[[汪雲|汪雲]]於2010年8月7日指，將在關閘總站開四個天窗加強通風，並興建冷氣候車室。<ref>澳廣視，交局將進一步改善關閘巴士站通風 2010年8月7日 18:30:08</ref>

===站內指示不足===
2007年6月28日，[[澳門日報|澳門日報]]以“指示欠奉地點隱．機場快線站難尋”為題，批評站內沒有機場線（[[澳門巴士AP1路線|AP1路線]]的指示，不便旅客）。<ref>[http://space.qoos.com/?action-viewnews-itemid-110074 澳門日報，指示欠奉地點隱．機場快線站難尋，2007年6月8日]</ref>

===鼠患===
2010年10月27日，澳門日報頭版刊登關閘總站鬧鼠患，指站內冷氣通風管道與地下污水渠已成老鼠藏身之所，鼠輩橫行，排泄物處處，每日數以千計候車乘客在站內與老鼠“同呼吸”，衛生情況令人憂慮。
