'''關山車站'''位於[[台灣|台灣]][[臺東縣|臺東縣]][[關山鎮_(台灣)|關山鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[岸式月台|岸式月台]]、[[島式月台|島式月台]]各一座。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]。
*對號列車停靠率：70.0%；所有車種停靠率：76.2%。
*停靠狀態：
**[[普悠瑪列車|普悠瑪號]]：皆停靠。
**[[太魯閣列車|太魯閣號]]：不停靠南下408次、426次；北上421次（週日停駛） 、441次 （週日停駛）、6421次（週日行駛） 、6441次（週日行駛）。
**[[自強號列車|自強號]]：不停靠北上301、6443次（週日行駛）。
**[[莒光號列車|莒光號]]：皆停靠。
**[[臺鐵區間車|區間車]]：皆停靠。
**[[觀光列車|觀光團體列車]]：停靠南下72（逢週六行駛）、82（逢週一、五至日行駛）次；北上6073（逢週日行駛）、81（逢週一、五、六行駛）、6081（逢週日行駛）次。
*站內設有「鐵路之旅－小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1922年4月20日：「里壠驛」開業。
*1937年11月1日：改名為「關山」。<ref>{{cite book |author = 戴震宇 |authorlink =  |title = 《台灣的老火車站》 |publisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>
*1980年：[[臺東線|臺東線]]東拓重建，新站建於舊站南方100公尺以保存舊站。
*2006年6月8日－10日：於台東關山舊站展出「薪火相傳·鐵道文化～鐵道文物展」活動，由展出者張建忠及林文淵導覽解說；10日當天將有「薪火相傳·鐵道文化」活動，以鐵道悠遊至里壟、文物展、座談會及舊車廂參訪等。
*2013年2月28日：新站改建完成啟用，為鐵改局推動「花東新車站運動」第一個完成的車站{{R|Liberty20130301}}。
*2016年6月28日：啟用多卡通刷卡機。
