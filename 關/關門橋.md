'''關門橋'''（{{jpn|j=関門橋|hg=かんもんきょう|rm=Kanmon kyō|f=y}}）是[[日本|日本]][[山口縣|山口縣]][[下關市|下關市]]與[[福岡縣|福岡縣]][[北九州市|北九州市]][[門司區|門司區]]間橫跨[[關門海峽|關門海峽]]的[[橋|道路橋]]名稱，同時亦是包含此橋梁區間的[[日本高速公路|高速公路]]（[[高速自動車國道|高速自動車國道]]）的通稱。

== 概說 ==
=== 橋梁 ===
於1973年完工通車，超過當時[[若戶大橋|若戶大橋]]成為日本最長的橋梁（直至1983年[[因島大橋|因島大橋]]開通）。型式為標準的[[吊橋|吊橋]]，獲得1973年度[[土木學會田中賞|土木學會田中賞]]（作品部門）。

包含橋梁前後區間的[[下關IC|下關IC]] - [[門司IC|門司IC]]間高速道路法定路線名為'''關門自動車道'''，正式上不屬於[[中國自動車道|中國自動車道]]和[[九州自動車道|九州自動車道]]，是「[[國土開發幹線自動車道|國土開發幹線自動車道]]以外的[[高速自動車國道路線指定政令|指定高速道路]]」<ref>同樣指定形態的高速公路有[[新機場自動車道|新機場自動車道]]（成田國際機場線）、[[關西機場自動車道|關西機場自動車道]]（關西國際機場線）、[[沖繩自動車道|沖繩自動車道]]。</ref>。區間內的距離標識IC編號為[[吹田JCT|吹田JCT]]（中國自動車道起點）計算，在一般道路上各IC資訊標識亦表示為「中國自動車道、九州自動車道」而非「關門橋、關門自動車道」。

是[[亞洲公路1號線|亞洲公路1號線]]（AH1）的構成部份，全線速度限制為80km/h。

== 交流道 ==
* 交流道（IC）編號欄背景色<span style="color:#BFB">■</span>區間是已通車區間，設施欄背景色<span style="color:#CCC">■</span>區間是未通車區間與未啟用設施，未通車區間名稱全部是暫稱。
* (數字)為其他路線的編號，<數字>為預定編號。
* 路線名未特別註記者為[[市町村道|市道]]
* BS（巴士站）：○為運用中，◆為停用中，未註記者即無設置。
* 下關交流道- 門司港交流道間之交流道編號為自中國自動車道連號，門司交流道之交流道編號則屬於九州自動車道。

== 歷史 ==
* 1973年（昭和48年）11月14日 - 完工通車。與中國自動車道、[[北九州高速4號線|北九州道路]]連接
* 1984年（昭和59年）3月27日 - 與九州自動車道連接

== 路線狀況 ==
=== 道路管理者 ===
* [[西日本高速公路|西日本高速公路]]（全區間由[[西日本高速公路九州支社|九州支社]]下關管理事務所管理）

* 門司港IC～門司IC　26,446輛
