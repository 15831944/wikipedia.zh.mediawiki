'''衢州机场'''位于[[中华人民共和国|中华人民共和国]][[浙江省|浙江省]][[衢州市|衢州市]]，是一個位於[[中國|中國]][[浙江|浙江]][[衢州|衢州]]的軍民合用機場，屬於4C級機場。亦是[[中国人民解放军空军|中国人民解放军空军]]使用的重要的空军基地。机场距离市区极近，紧贴[[衢州城墙|衢州城墙]]东段，交通便利，具铁路专用线直通[[衢州站|衢州站]]。由于衢州机场地处东海热点地区，军用地位重要，民航服务发展缓慢并时常受到军事活动干扰。衢州市老城区因在飞机主要起降航线上，建筑物高度受到限制。城市向东扩张亦被机场阻断。

== 历史 ==
衢州机场始建于1933年，由于当时国家经济状况进展缓慢。1937年[[淞沪会战|淞沪会战]]和[[八一四空战|八一四空战]]后，衢州的战略地位陡然上升，在极短时间内抢修完成并用于[[中华民国空军|中华民国空军]]军机起降。1938年日军占领杭州后沿浙赣铁路南下，为阻止日军夺取衢州机场进犯内地，国军破坏了机场并纳入陆军防线的一部分。随着日军在浙江中部一带停止前进，衢州机场于1939年1月被重修以容纳国军飞机。同年6月，空军第十三航空总站在衢州机场成立，统辖附近前线机场。[[太平洋战争|太平洋战争]]爆发后，国军于41年底扩建机场以供未来中美联合对日作战所需。1942年4月[[美国陆军航空队|美国陆军航空队]][[詹姆斯·哈羅德·杜立德|詹姆斯·哈羅德·杜立德]]率领[[B-25米切尔型轰炸机|B-25米切尔型轰炸机]]从[[大黃蜂號航空母艦_(CV-8)|大黃蜂号]]航空母舰上起飞[[空袭东京|空袭东京]]时，亦计划以衢州机场作为备降点，但因天气原因降落失败。机组弃机逃生后得到当地民众救助。

因为衢州机场为太平洋战争后距日本本土最近的盟国控制下的大型机场（当时苏日保持中立），故倍受交战双方重视，中日两军围绕衢州机场展开一系列的战役。1942年日军发动[[浙赣战役|浙赣战役]]，6月3日，国军在撤退前破坏衢州机场。日军占领后立即修复机场供军机起降。8月日军放弃衢州东缩至东阳金华一线，机场再度被破坏，并由国军重修。

== 軍用機場 ==
衢州機場為[[中國人民解放軍空軍|中國人民解放軍空軍]]重要的空軍基地。目前為南京軍區空軍第29師85團，86團駐地，裝備有俄制[[Su-27戰鬥機|Su-27戰鬥機]]，[[Su-30戰鬥機|Su-30戰鬥機]]和[[殲-7戰鬥機|殲-7戰鬥機]]。主要目的是和台灣[[中華民國空軍|中華民國空軍]]，[[日本航空自衛隊|日本航空自衛隊]]和亞太地區[[美國空軍|美國空軍]]及[[美國海軍|美國海軍]]相抗衡。2013年底，為增強東海軍事實力，80架[[殲-10戰鬥機|殲-10戰鬥機]]被部署在衢州機場。

== 歷史事件 ==
[[File:Baggage_Claim_at_Quzhou_Airport.JPG|thumb]]
1939年6月，升格为中华民国空军第十三总站，管辖衢州及玉山，丽水，建瓯三处前线机场

1941年冬珍珠港事件后，机场开始大规模扩建，以备未来[[美国陆军航空队|美国陆军航空队]]使用。

1942年6月3日，衢州机场被日军占领。

1954年4月22日，驻衢机场高炮51营击落击伤侵入螺旋桨战机各一架。

1956年6月22日午夜至23日凌晨，時任[[中國人民解放軍空軍|中國人民解放軍空軍]]第12師34團團長[[魯珉|魯珉]]駕駛[[米格-17戰鬥機|米格-17戰鬥機]]從衢州機場起飛，在[[江西省|江西省]]廣豐，上饒地區擊落了台灣的[[中華民國空軍|中華民國空軍]]侵入大陸的[[B-17轟炸機|B-17轟炸機]]一架，首創中華人民共和國國土防空夜間擊落敵機的戰例。<ref>[http://dangshi.people.com.cn/GB/151935/172706/172708/10303260.html 南京军区空军，中国共产党新闻网，2009-11-02]</ref>

1998年4月27日夜，駐衢中國人民解放軍空軍29師兩架[[殲-7#殲7Ⅲ/ⅢA型|殲-7D型]]戰鬥機在進行夜航訓練時相撞墜毀，長機（25106）飛行員彈射逃生，僚機（25007）飛行員殉職。

1993年，衢州机场开通民航服务。
