'''抢救丘财康'''，时称抢救钢铁工人丘财康，指1958年[[上海市|上海]][[廣慈醫院|广慈医院]]（今[[上海交通大学医学院附属瑞金医院|上海交通大学医学院附属瑞金医院]]）成功抢救烧伤的钢铁工人邱财康{{notetag|根据邱财康本人签名和病案，真名为“邱财康”，当时大部分报道称为“丘财康”，原因不明。<ref name=胡俊>{{Cite journal|title=“大跃进”期间抢救钢铁工人邱财康——张涤生访谈|author=胡俊, 甄橙|url=|journal=中国科技史杂志|issue=2|doi=|others=|year=2011|volume=|page=222-230|pmid=}}</ref>}}的事迹。

== 始末 ==
1958年5月26日深夜，[[上海第三钢铁厂|上海第三钢铁厂]]工人邱财康（1928年-2014年03月17日）被掉落的钢水严重烧伤，全身烧伤面积89.3%，其中[[三度烧伤|三度烧伤]]23%。当时尚无灼伤面积超过80%的生还病例，专家们对治愈普遍表示悲观。中共广慈医院党委召开紧急会议，要求破除迷信，努力抢救。上海第二医学院和广慈医院组织抢救小组，[[董方中|董方中]]任组长，[[史济湘|史济湘]]任副组长，小组成员包括[[傅培彬|傅培彬]]、[[邝安堃|邝安堃]]、[[杨之骏|杨之骏]]、[[戴自英|戴自英]]、[[张涤生|张涤生]]等。

医护人员增加补液尤其是血浆，使邱财康度过“休克关”。6月2日，邱财康被[[绿脓杆菌|绿脓杆菌]]感染，出现[[败血症|败血症]]，医护人员用[[多粘菌素|多粘菌素]]治疗，为控制[[败血症|败血症]]，截去邱财康的左手。后细菌产生对多粘菌素的耐药，医护人员采用输入带有抗体的健康者的血液、噬菌体清洗伤口{{notetag|当时的报道称，噬菌体清洗伤口的效果显著，“不到24小时，化脓现象开始消退了”。张涤生则说：“可惜使用后未见什么特效”。{{r|胡俊}}}}等方法，到8月度过了“感染关”。为防止皮肤感染和褥疮，创造、使用了[[翻身床|翻身床]]。历经5月26日、7月26日、8月26日移植补皮，邱财康痊愈，度过“植皮关”，11月21日出院。

== 影响 ==
“当时业界公认的美国烧伤学科权威——伊文思的理论，仅适用于全身烧伤面积低于50%。”（袁克俭）医护人员后来将休克阶段的补液总结为“烧伤早期液体复苏的瑞金公式”。

1958年7月1日《[[人民日报|人民日报]]》发表通讯《[[党救活了他——记抢救上钢三厂工人丘财康的经过|党救活了他——记抢救上钢三厂工人丘财康的经过]]》，1958年6月28日《[[中国青年报|中国青年报]]》发表通讯《[[共产主义精神的凯歌|共产主义精神的凯歌]]》。[[巴金|巴金]]专程到医院看望邱财康，创作了报告文学《一场挽救生命的战斗》。[[上海科学教育电影制片厂|上海科学教育电影制片厂]]摄制成纪录电影《生命的凯歌——抢救邱财康的胜利》，[[上海电影制片厂|上海电影制片厂]]改编为电影《[[春满人间|春满人间]]》。[[孙道临|孙道临]]等排演了话剧《共产主义凯歌》。另有纪实性电视剧《党救活了他》。

1964年1月22日，[[中华人民共和国卫生部|中华人民共和国卫生部]]召开大会，表彰广慈医院抢救邱财康的突出成就。

== 评价 ==
* 抢救小组成员、中国工程院院士[[张涤生|张涤生]]：“这是知识分子集体智慧的结晶，体现了知难而上的精神。100个日夜里，医患并肩作战，最终病人获得了生命，医生获得了胜利。现在看来，这种精神是不过时的。”
