'''梨園寮車站'''位於[[台灣|台灣]][[嘉義縣|嘉義縣]][[梅山鄉_(台灣)|梅山鄉]]，為[[行政院農業委員會林務局|林務局]][[阿里山森林鐵路|阿里山森林鐵路]][[阿里山線|阿里山線]]之[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*左邊是辦公室，右邊是候車室，站房形式和獨立山站完全相同。

== 利用狀況 ==
*阿里山線（主線）的停靠站。

== 歷史 ==
*1912年10月1日：設站。
