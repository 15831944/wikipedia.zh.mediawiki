'''枋頭之戰'''發生於東晉[[太和_(東晉)|太和]]四年（369年），是[[桓溫|桓溫]]第三次[[桓溫北伐|北伐]]的一場決定性戰役。

== 背景 ==
太和四年四月庚戌日（369年5月22日），桓溫自[[姑孰|姑孰]]（今[[安徽|安徽]][[當塗縣|當塗縣]]）率軍五萬北伐，率水軍自[[清水河_(淮河)|清水]]進入[[黃河|黃河]]。[[前燕|前燕]]皇帝[[慕容暐|慕容暐]]以下邳王[[慕容厲|慕容厲]]為征討大都督抵抗桓溫，但於[[黃墟|黃墟]]大敗。慕容暐再以樂安王[[慕容臧|慕容臧]]率軍抵禦，仍然無法取勝。燕將[[傅顏|傅顏]]亦於[[林渚|林渚]]敗給晉軍前鋒[[鄧遐|鄧遐]]及[[朱序|朱序]]，晉軍士氣大振<ref>《晉書·慕容暐傳》：「溫前鋒朱序又破暐將傅顏於林渚，溫軍大振。</ref>，而前燕只好遣使向[[前秦|前秦]]求援。

== 經過 ==
七月，桓溫駐軍枋頭，當時前燕前[[兗州|兗州]]刺史[[孫元|孫元]]亦起兵響應桓溫，慕容暐及[[太傅|太傅]][[慕容評|慕容評]]見晉軍逼近而十分畏懼，打算逃奔[[和龍|和龍]]。而吳王[[慕容垂|慕容垂]]卻自請出戰，慕容暐於是以[[慕容垂|慕容垂]]為南征大都督，率慕容德等人領兵五萬出拒晉軍。桓溫當時就任用投降的燕人[[段思|段思]]作為嚮導，燕將[[悉羅騰|悉羅騰]]與晉軍交戰，成功生擒了段思；不久晉將[[李述|李述]]的進攻亦為悉羅騰及[[染干津|染干津]]擊敗，李述更戰死。此兩敗後，晉軍士氣變得低落。

另一方面，慕容德及[[劉當|劉當]]分領一萬及五千騎兵於九月出屯石門，妨礙晉將[[袁真|袁真]]開通石門水道運糧，前鋒[[慕容宙|慕容宙]]更命二百兵挑戰，另以八百兵設伏，誘晉軍追擊而大敗對方；燕豫州刺史[[李邽|李邽]]亦率兵五千斷桓溫糧道。而前燕又以虎牢以西的土地為條件向前秦求援，[[王猛|王猛]]勸苻堅暫時保存前燕不被東晉所併，並乘戰後前燕國力減弱才出兵吞併。前秦天王[[苻堅|苻堅]]聽從，命[[苟池|苟池]]及[[洛州|洛州]]刺史[[鄧羌|鄧羌]]出兵救援。

桓溫戰事失利，而當時糧食亦見缺乏，更知前秦派兵援燕，於是在九月丙申日燒毀船艦，拋棄輜重兵器，循陸路撤軍。桓溫自[[東燕郡|東燕郡]]撤至[[倉垣|倉垣]]（今河南[[開封|開封]]東北），為提防前燕軍在[[汴河|汴河]]等河水下毒而下令鑿井取水飲用，走了七百多里路。當時燕軍諸將都要追擊，但慕容垂認定晉軍初退必定會嚴密戒備，留精兵斷後，決意待晉軍以為燕軍不至，加緊趕路時才追擊，以求克捷；慕容垂因而領八千騎在後跟隨晉軍。晉軍隨後果然如慕容垂所估計而加速，數日後慕容垂就下令追擊，並於[[襄邑|襄邑]]（今河南[[睢縣|睢縣]]西）追及晉軍。當時慕容德已率先領四千騎兵設伏於[[襄邑|襄邑]]東澗，終與慕容垂夾擊晉軍，桓溫大敗，死三萬多人。前秦援軍亦於[[譙縣|譙縣]]（今安徽[[亳縣|亳縣]]）邀擊桓溫敗軍，晉軍戰死者亦數以萬計。至十月己巳日（12月7日），桓溫收拾散卒，屯於[[山陽|山陽]]（今江蘇[[淮安市|淮安市]]）。原響應桓溫的孫元亦被前燕平定。

== 影響 ==
*這次戰役中晉軍大敗，亦代表著桓溫第三次北伐的失敗。桓溫於敗後名聲和實力都大降，已經無法依原計劃接受[[九锡|九錫]]並篡位自立，從而導致兩年後桓溫廢黜[[晉廢帝|晉廢帝]]，改立[[晉簡文帝|晉簡文帝]]<ref>《晉書·桓溫傳》：「溫既負其才力，久懷異志，欲先立功河朔，還受九錫。既逢覆敗，名實頓減，於是參軍郁超進廢立之計，溫乃廢帝而立簡文帝。」</ref>。
*慕容垂於枋頭之戰中領導燕軍擊退晉軍，聲名大振，卻令向來猜忌他太傅的[[慕容評|慕容評]]感到厭惡，更與[[可足渾皇后_(慕容儁)|可足渾太后]]謀誅慕容垂，逼得慕容垂出奔前秦。<ref>《晉書·慕容垂傳》：「及敗桓溫於枋頭，威名大振。慕容評深忌惡之，乃謀誅垂。懼禍及己，與世子[[慕容令|全]]奔於苻堅。」</ref><ref>《晉書·慕容暐傳》：「垂既有大功，威德彌振，慕容評素不平之……可足渾氏素惡垂，毀其戰功，遂與評謀殺垂。垂懼，奔於苻堅。」</ref>
*前燕向前秦求救時許以[[虎牢关|虎牢]]以西土地作條件，但戰後前燕卻反悔不給，這令苻堅大怒，最終令前秦以此對前燕發動戰爭，終滅前燕。
