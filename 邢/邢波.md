'''邢波''' （{{lang-en|Eric Xing}}）是[[卡内基梅隆大学|卡内基梅隆大学]]计算机科学院教授，专供于[[机器学习|机器学习]]、[[计算生物学|计算生物学]]和统计方法等方向。<ref>{{cite web |url=http://www.cs.cmu.edu/~epxing/ |title=Eric P. ==生平==
邢波喜爱冒险，1988年考入[[清华大学|清华大学]]，曾经担任[[清华大学登山队|清华大学登山队]]第一任队长。1993年毕业于[[清华大学物理系|清华大学物理系]]，1994年考入[[罗格斯大学|罗格斯大学]]学习分子生物学与生物化学，1996年同校学习计算机科学，1998年拿到[[罗格斯大学|罗格斯大学]]计算机科学硕士，1999年拿到[[罗格斯大学|罗格斯大学]]分子生物学博士学位，2004年拿到[[加州大学伯克利分校|加州大学伯克利分校]]计算机科学博士学位。他评价其三位博士生导师[[理查德·卡普|理查德·卡普]]（计算机学家,偏重理论），[[迈克尔·乔丹_(学者)|迈克尔·乔丹]](计算机学家,偏重概率统计)，[[Stuart_J._Russell|Stuart J. Russell]]（很好的计算机学家、哲学家）。<ref>[http://www.tsinghua.edu.cn/publish/phy/5580/2010/20101221130525613189962/20101221130525613189962_.html 学海无涯，不懈追求——物理系杰出系友邢波访谈]，清华大学物理系官网。</ref>

此外，他与合作者开发了具有大量数据，大型模型和广泛算法的分布式机器学习的“Petuum”框架。<ref>{{cite web |url=http://petuum.github.io/ |title=Petuum is a distributed machine learning framework |last1=Xing |first1=Eric P. |last2= |first2= |date=2013 |website=github.io |publisher= |accessdate=16 December 2014 |deadurl=yes |archiveurl=https://web.archive.org/web/20141216214330/http://petuum.github.io/ |archivedate=2014年12月16日 }}</ref>
