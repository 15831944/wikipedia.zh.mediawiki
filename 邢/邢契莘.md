'''邢契莘'''（1887 - 1957）字'''学耕'''，号'''寿农'''，[[浙江省|浙江]][[绍兴市|绍兴]][[嵊州市|嵊县]][[长乐镇_(嵊州市)|太平乡]]人，是[[第一次庚子赔款留美学生列表|第一次庚子赔款的留美学生]]，中华民国船舶及航空工程师。

邢契莘早年在[[上海市南洋模范中学|上海南洋中学]]就读，1909年考入保定直隶高等学校，次年獲得[[美国|美国]][[庚子拳亂賠款|庚子拳亂賠款]][[獎學金|獎學金]]，成为[[第一次庚子赔款留美学生列表|清华第一期官费生]]，留学美国，在[[麻省理工学院|麻省理工学院]]造船造舰系入学。1914年毕业后，继续选修造舰系，兼习航空机械。1916年，获硕士学位。回国后，先后任[[大沽船坞|大沽造船所]]工程师、[[福州船政局|福州船政局]]制船主任、北平航空署机械厅厅长、东北航空处技师和处长等职，曾在天津工业专门学校、[[马尾海军学校|马尾海军学校]]、[[北京农业大学|国立北京农业专门学校]]及[[中国大学|中国大学]]执教。1927年至1932年，任东北航务局及东北联合航务局总经理，东北造船所所长。1934年1月，任[[青岛市|青岛市]]工务局局长。1937年4月，任国民政府航空委员会机械处处长。次年，转任[[雷允飞机制造厂|中央雷允飞机制造厂]]监理，修建[[南山机场|南山机场]]，后来成为[[盟军|盟军]][[抗日|抗日]]的基地机场。后因眼疾，返[[重庆市|重庆]]休养。1943年3月，任[[行政院农业委员会|农林部]]总务司司长。1945年[[日本二戰投降|日本二戰投降]]之后，先后任[[大连港|大连港]]务委员会主任委员（未就任）、[[中华民国交通部|交通部]]塘沽新港工程局局长、[[经济部水利署|水利部]]珠江水利工程总局局长、[[广州港|广州港]]工程局局长。1949年至1950年，寓居[[香港|香港]]，后隨[[國民黨政府|國民黨政府]]抵達[[台湾|台湾]]，先后任交通部设计委员会委员、台湾省渔业增产委员会委员。著有《葫蘆島築港記》和《青島工程之研究》。<ref>{{cite web |url=http://mren.bytravel.cn/history/7/xingqi.html |title=邢契莘 |last= |first= |date= |website=mren.bytravel.cn |publisher=博雅人物网 |access-date=2017-11-30 |quote=}}</ref><ref>{{cite web |url=http://www.hgzz.net/zhuanti/66166.html |title=邢契莘介绍 |last= |first= |date=2016-04 |website=www.hgzz.net |publisher= |access-date=2017-11-30 |quote=}}</ref><ref>{{cite book |author=林吕建 |date=2013年1月 |title=《浙江民国人物大辞典》 |url=http://xuewen.cnki.net/read-R2013091280000470.html |location=[[杭州市|杭州]] |publisher=[[浙江大学|浙江大学]]出版社 |page= |isbn=978-7-3081-07907 |author-link= }}</ref><ref>{{cite web |url=http://sznews.zjol.com.cn/sznews/system/2015/12/16/020023573.shtml |title=邢契莘传略 |author=邢樟涛 |date=2015-12-16 |website=sznews.zjol.com.cn |publisher=嵊州新闻网 |access-date=2017-11-30 |quote=}}</ref>
