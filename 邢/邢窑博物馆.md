'''邢窑博物馆'''，位于[[河北省|河北省]][[邢台市|邢台市]][[内丘县|内丘县]]中兴北大街，是[[邢窑|邢窑]]专业型博物馆。

2013年4月，内丘[[邢窑遗址|邢窑遗址]]当选2012年全国十大考古新发现。为科学推进邢窑遗址保护利用，内丘县规划了邢白瓷文化产业集聚区，拟投资21.6亿元人民币建设文化产业园区。2013年6月28日，“2012年度全国十大考古新发现——中国内丘邢窑遗址保护利用”启动仪式在内丘县举行，邢白瓷文化产业园同日动工<ref>{{Cite web |url = http://www.xtrb.cn/epaper/xtrb/html/2013-07/01/content_417698.htm |title =邢白瓷文化产业园在内丘奠基  |author =  |date = 2013-07-01 |publisher =邢台日报  |language =  |accessdate =  |quote =  }}</ref>。

2017年8月16日，内丘县举行中国内丘邢白瓷文化产业园“邢窑博物馆、邢瓷文化体验馆、[[邢窑遗址博物馆|邢窑遗址博物馆]]”揭牌仪式，中共内丘县委书记张辉、内丘县人民政府县长杨辉为“三馆”揭牌<ref name=jiepai>{{Cite web |url =  http://www.hbnq.gov.cn/article-view-id-1286.html|title = 我县举行中国内丘邢白瓷文化产业园“邢窑博 物馆、邢瓷文化体验馆、邢窑遗址博物馆”揭牌仪式 |author =  |date = 2017-08-17 |publisher =内丘县人民政府  |language =  |accessdate =  |quote =  }}</ref>。三馆均在内丘县县城中兴北大街西侧，一字排开，北为邢窑博物馆，中间为邢瓷文化体验馆，南为邢窑遗址博物馆<ref name=jiepai/><ref name=jbw>{{Cite web |url =http://www.bjd.com.cn/jjj/yaowenzhanshi/201803/05/t20180305_11081510.html  |title =津冀博物馆再添新面孔  |author =  |date =2018-03-05  |publisher = 京报网 |language =  |accessdate =  |quote =  }}</ref>。邢窑博物馆总投资1.5亿元人民币，占地面积约53亩，建筑面积11000平方米，是收藏、展示、研究邢白瓷的专业型博物馆<ref>{{Cite web |url =http://hebei.ifeng.com/a/20170821/5928736_0.shtml  |title = 河北内丘：邢窑博物馆、邢窑遗址博物馆开馆迎客 |author =  |date = 2017-08-21 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref name=jiepai/>。2015年，该项目被评为河北省“双百工程”特色风貌建筑<ref name=jiepai/>。馆内通过声、光、电等手段展示700多件邢瓷藏品<ref name=jbw/>。

邢窑博物馆南侧的邢瓷文化体验馆占地面积15亩，地上建筑面积3000平方米，作为邢窑博物馆功能扩充并独立运营的体验馆，为游客提供烧制成品瓷器的体验，并展示和出售瓷器<ref name=jiepai/>。
