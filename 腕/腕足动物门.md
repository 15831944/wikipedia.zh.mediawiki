'''腕足动物门'''（[[學名|學名]]：{{lang|la|'''Brachiopoda'''}}）是[[动物界|动物界]]的一个[[門_(生物)|门]]，屬於[[底棲生物|底栖]]、有一對硬壳的[[觸手冠動物|触手冠]][[海|海]]产动物。但與[[雙殼類|雙殼類]]動物不同的是：其殼是上、下開合，而不是左、右開合。[[鉸齒|鉸位]]在後背部，而前方可開合作捕食或防御。牠們自[[寒武紀|寒武紀]]開始演化，现存的种类多分布在高[[纬度|纬度]]的冷水区。

== 語源 ==
腕足動物的學名''brachiopod''源於[[古希臘語|古希臘語]]，由「手臂」（{{lang|grc|βραχίων}}）與「足」（{{lang|grc|πούς}}）兩部份組成{{sfn|Shorter Oxford English Dictionary|(2002)|loc=entry "Brachiopod"}}。

== 特征 ==
介壳两枚，大小相等或不等，掩盖背腹两面；介壳的形状和饰纹以及内部器官的构造，是鉴定腕足纲属、种的依据；身体柔软、左右对称；头顶有突出部；上生许多触手，称为“腕足”；消化道呈U字形弯曲，常缺少肛门；具有体腔和后肾。

== 分類 ==
{{seealso|腕足動物的分類}}
[[File:Pygites_diphyoides_(d'Orbigny).jpg|thumb]][[穆爾西亞|穆爾西亞]][[塞埃欣|塞埃欣]]的[[早白堊世|早白堊世]][[豪特里維期|豪特里維期]]地層出土的''Pygites diphyoides'' ([[d'Orbigny|d'Orbigny]], 1849)化石。這件[[穿孔贝目|穿孔贝目]]化石的特色是其殼上位於中央的孔洞。]]
腕足动物的分類有其獨特的不同之處：現時腕足動物的現存物種與化石物種有着不同的特色。其化石物種的外形變化很大，但其外殼只有少量特徵；而現存物種的外殼變化不大，反而其軟體及外殼均各有特徵。若我們只專注於其中一類特徵，均不能把另一類好好的分類。本門動物也經歷了顯著的[[趨同進化|趨同進化]]和[[逆轉演化|逆轉演化]]（這使較新的組別損失了原來在中間組別出現的特色，又或回復了原來只在較舊的組別出現的特點）。因此，有些分類學的學者認為現時我們仍然未成熟到可以具體把[[目_(生物)|目]]以上的分類確定，建議只適宜透過自底向上的方法，先確定物種的種屬，然後再把他們分成組{{sfn|Carlson: Ghosts|(2001)|}}。不過，亦有學者不同意這個觀點，認為現時的
然而，其他分類學家相信，物種中某些模式的特點已足夠穩定，值得作更高級別的分類，儘管大家可能對更高級別的分類有不同的看法{{sfn|Carlson: Ghosts|(2001)|}}。以下為現時較多人同意的分類法。
