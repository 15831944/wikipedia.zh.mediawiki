'''[[腕足动物门|腕足动物]]的分類'''有其獨特的困難之處：現時腕足動物的現存物種與化石物種有着不同的特色。其化石物種的外形變化很大，但其外殼只有少量特徵；而現存物種的外殼變化不大，反而其軟體及外殼均各有特徵。若我們只專注於其中一類特徵，均不能把另一類好好的分類。本門動物也經歷了顯著的[[趨同進化|趨同進化]]和[[逆轉演化|逆轉演化]]（這使較新的組別損失了原來在中間組別出現的特色，又或回復了原來只在較舊的組別出現的特點）。因此，有些分類學的學者認為現時我們仍然未成熟到可以具體把[[目_(生物)|目]]以上的分類確定，建議只適宜透過自底向上的方法，先確定物種的種屬，然後再把他們分成組{{sfn|Carlson: Ghosts|(2001)|}}。然而，其他分類學家相信，物種中某些模式的特點已足夠穩定，值得作更高級別的分類，儘管大家可能對更高級別的分類有不同的看法{{sfn|Carlson: Ghosts|(2001)|}}。以下我們回歸這一門動物的分類演化。

== 有鉸綱和無鉸綱 ==
傳統上，腕足动物门主要依從Huxley在1869年提出的分類{{sfn|ITIS: Brachiopoda||}}，分為兩綱：[[有鉸綱|有鉸綱]]（Articulate）和[[無鉸綱|無鉸綱]]（Inarticulate）。
有鉸綱的兩殼間有齒和槽，套膜叶在背方连接，依靠簡單的開合肌肉來令兩殼絞合；而無鉸綱的兩殼間沒有有齒和槽，套膜的两叶分开，兩殼依靠一套較為複雜的肌肉系統來令兩邊保持對齊{{sfn|Ruppert etc: Invert Zoo|(2004)|loc=ch. "Lophophorata" sect. "Brachiopoda"|pp=821–829}}{{sfn|Ax: Multicellular Animals|(2003)|loc=ch."Brachiopoda"|pp=87–93}}<ref name=aps>{{cite journal |author1=戎嘉余 |author2=李荣玉 |title=评述腕足动物门高级别分类的新方案 |language=zh-hans |journal=[[古生物学报_(中国)|古生物学报]]（ACTA PALAEONTOLOGICA SINICA） |volume=第36卷 |issue=第3期 |pages=378-386 |url=http://ir.nigpas.ac.cn/bitstream/332004/2896/2/44.pdf |date=1997年7月 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

在1990年代，波波夫等(1993)用[[分支系统学|分支系统学]] (cladistics)的方法，提出對腕足动物门物種作新的高级别分类。他們建議提出加入“[[钙质壳纲|钙质壳纲]]”去優化這個二分分類{{sfn|Ax: Multicellular Animals|(2003)|loc=ch."Brachiopoda"|pp=87–93}}。根據這個建議，加入按外殼的主要成分為分類的原則。無鉸綱生物殼主要成分是[[幾丁質|幾丁質]]或[[幾丁磷灰質|幾丁磷灰質]]<ref name=bioon_11458>{{cite web |url=http://www.bioon.com/biology/Class422/11458.shtml |title=腕足动物门——无铰纲 |language=zh-hans |date=2004-04-16 |accessdate=2013-02-27 |author=南京大学地球科学数字博物馆 |archive-url=https://web.archive.org/web/20180102191521/http://www.bioon.com/biology/Class422/11458.shtml |archive-date=2018-01-02 |dead-url=yes }}</ref>與[[磷酸鈣|磷酸鈣]]組成，有肛門<ref name=bioon_11458 />；有鉸綱生物的殼主要成分是[[碳酸鈣|碳酸鈣]]，無肛門<ref name=bioon_11465>{{cite web |url=http://www.bioon.com/biology/Class422/11465.shtml |title=腕足动物门——有铰纲 |language=zh-hans |date=2004-04-16 |accessdate=2013-03-07 |author=南京大学地球科学数字博物馆 }}{{Dead link|date=2018年8月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，排泄器官為[[原腎|原腎]]。按這原則，[[髑髅贝类|髑髅贝类]]從無鉸綱被歸入有鉸綱，而[[平圆贝目|平圆贝目]]則被歸入無鉸綱{{sfn|Ax: Multicellular Animals|(2003)|loc=ch."Brachiopoda"|pp=87–93}}。但亦有批評指髑髅贝类跟原來的有鉸綱的差別太大，令這個建議變得不合理<ref name=aps />。

== 三綱分類 ==
對於波波夫等方案的問題，Williams等 (1996)提出一個更大膽的方案：取消無鉸綱和有鉸綱的分別，把當中與其他物種有顯着分別的物種獨立出來{{sfn|Milsom etc: 3-part taxonomy|(2009)||}}{{sfn|Williams etc: Suprafamilial Classif|(2000)|loc=Preface|pp=xxxix-xlv}}：
* 原來的有鉸綱的絕大部份物種被歸為[[小嘴贝型亚门|小嘴贝型亚门]]；
* 無鉸綱一分為二，
** 原來的[[髑髅贝类|髑髅贝类]] (craniids)和[[原货贝类|原货贝类]](obolellids)獨立出來成為新的[[髑髅贝型亚门|髑髅贝型亚门]]，
** 而餘下的[[舌形贝类|舌形贝类]]則成立[[舌形贝型亚门|舌形贝型亚门]]。
