'''馮志芬'''（{{lang-en|FUNG Chi-Fen}}；約{{bd|1907年||1962年}}<ref name="hkrep">{{ cite web |url=http://www.hkrep.com/article/footprintsinthesnow/ |title=開戲師爺的浮沉─《一頁飛鴻》戲劇研究文章 |date=2016-03-08 |publisher= |author=陳守仁 }}</ref>），廣東新會人，香港粵劇編劇家，[[唐滌生|唐滌生]]的師傅，曾寫下《胡不歸》和《情僧偷到瀟湘館》等家傳戶曉的粵劇名作。

==生平==
馮志芬，廣東新會人，出身書香世家，自幼喜歡中國古典文學，精通詩詞。曾在廣州市思恩中學當教師。<ref name="ku1999">{{ cite web |url= |title=《廣東戲劇史（紅伶篇之一）》 |date=1999 |publisher=粵劇研究計劃 |author=古岡 |location=香港 |pages=頁99-100 |isbn=9789621600929   }}</ref>

1931年，他曾當編劇家[[梁夢|梁夢]]的助手，協助編撰名劇《火燒阿房宮》，之後加入[[薛覺先|薛覺先]]的「[[覺先聲劇團|覺先聲男女劇團]]」任「開戲師爺」。當時「覺先聲」已有[[麥嘯霞|麥嘯霞]]當開戲師爺。至1930年代後期，因為[[麥嘯霞|麥嘯霞]]電影、粵劇、寫作三棲，馮志芬成為了「覺先聲」的主力師爺，也當上了[[唐滌生|唐滌生]]的師傅，其後更被譽為1930年代最重要的開戲師爺。<ref name="hkrep" />

1939年，馮志芬為覺先聲劇團寫下由薛覺先、上海妹及半日安開山主演的名劇《[[胡不歸|胡不歸]]》，成為家傳戶曉的粵劇經典，歷演不衰，風行超過70年，直至2017年仍傳演不輟<ref>{{ cite web |url=http://www.yuejuopera.org.cn/dtxx/20170811/3496.html |title=8月18-20日晚將在香港上演粵劇名劇《胡不歸》和折子戲專場 |date=2017-08-11 |publisher=中國粵劇網 |author= }}</ref><ref>{{ cite web |url=https://ol.mingpao.com/php/showbiz3.php?nodeid=1510053679440&subcate=latest&issue=20171107|title=孖汪明荃演出《胡不歸》 羅家英：今日有床戲 |date=2017-11-07 |publisher=明報 |author= }}</ref>。

1947年，馮志芬為[[何非凡|何非凡]]的「非凡響劇團」編撰取材自《紅樓夢》的粵劇《情僧偷到瀟湘館》，轟動一時，在香港創下連續演出44天的紀錄。<ref>{{ cite web |url=http://www.hkrep.com/article/footprintsinthesnow/ |title=《香港粵劇劇目概說（1900-2002）》 |date=2007 |publisher=香港中文大學出版社|author=陳守仁編 |location=香港 |pages= |isbn=9789628104130  }}</ref>

1949年後，馮志芬由香港返回廣州定居，1957年[[反右運動|反右運動]]中被誤劃為「[[右派|右派]]分子」，1959年被撤去中國戲劇家協會廣東分會理事職務並開除出會。最後被送到勞教場進行「勞改」，約在1962年死於廣東的勞教場。<ref name="hkhistory">{{ cite web |url= |title=《香港史新編（增訂版）》 |date=2007 |publisher=三聯書店 |author=王賡武 |location=香港 |pages= |isbn=9789620438851   }}</ref>
