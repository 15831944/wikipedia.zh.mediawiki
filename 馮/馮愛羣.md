'''馮愛羣'''（{{bd|1924年||1989年|1月18日}}）常被寫作'''馮愛群'''，是[[中華民國|中華民國]]新聞界、文化界人士，著有《聯合國與國際間合作觀念的進化》、《中國新聞史》和《華僑報業史》等書<ref name = "崔之清 350-351">{{Cite book | author = 崔之清 | title = 《当代台湾人物辞典》 | location = 中國郑州 | publisher = 河南人民出版社 | date = 1994年7月 | pages = 第350頁至第351頁 | ISBN = 978-7215023260 | accessdate = 2018年9月9日 | url =  | language = Zh-hans | quote =  }}</ref><ref name = "尹韵公 94">{{Cite book | author = 尹韵公 | title = 《中国新闻界人物》 | location = 中國北京 | publisher = 中国人事出版社 | date = 2002年10月 | pages = 第94頁 | ISBN = 978-7801399359 | accessdate = 2018年9月9日 | url =  | language = Zh-hans | quote =  }}</ref>。

== 經歷 ==
[[File:台灣學生書局.jpg|thumb]]，並出任董事長一職。]]

1924年，馮愛羣在[[廣東省|廣東省]][[連平縣|連平縣]]出生<ref name = "中國中外名人文化研究會 192">{{Cite book | author = 中國中外名人文化研究會 | title = 《中华文化名人录》 | location = 中國北京 | publisher = 中國青年出版社 | date = 1993年12月 | pages = 第192頁 | ISBN = 978-7500612049 | accessdate = 2018年9月9日 | url =  | language = Zh-hans | quote =  }}</ref>。在[[南京市|南京市]][[中央政治學校|中央政治學校]]（[[國立政治大學|國立政治大學]]的前身）畢業後<ref name = "中國中外名人文化研究會 192"/>，他在[[中國陸軍總司令部|中國陸軍總司令部]]及[[革命實踐研究院|革命實踐研究院]]任職<ref name = "聯合報 01">{{cite news |language = Zh-hant |author =  |coauthors =  |url =  |title = 馮愛群之喪明公祭 |publisher = 《[[聯合報|聯合報]]》 |pages = 14版 |date = 1989年2月23日 |accessdate = 2018年9月9日 }}</ref>。而在[[中國抗日戰爭|中國抗日戰爭]]結束後不久，他便在[[重慶市|重慶市]]創辦《[[新聞雜誌_(中國)|新聞雜誌]]》半月刊，提供具精闢新聞背景的報導和分析<ref name = "中華民國當代名人錄">{{Cite book | author =  | title = 《中華民國當代名人錄》 | location = 臺灣臺北 | publisher = [[中華書局|中華書局]] | date =  | pages =  | ISBN =  | accessdate = 2018年9月9日 | url =  | language = Zh-hant | quote =  }}</ref>。在當時雜誌出版低潮的期間，一度發行多達10萬份，進而創下當時的新紀錄<ref name="中國中外名人文化研究會 192" /><ref name = "中華民國當代名人錄"/>。

1949年，馮愛羣前往[[臺灣|臺灣]]<ref name="中國中外名人文化研究會 192" />。在抵達臺灣後，參加[[聯合國|聯合國]]第一屆論文比賽，獲得中國區第一名<ref name = "中華民國當代名人錄"/>。隨後他在[[中華民國政府|中華民國政府]]轄下機關任職，投身官方的文化事業，包括曾經出任[[中華民國僑務委員會|中華民國僑務委員會]]文化科科長、[[中華民國教育部|中華民國教育部]]電影事業輔導委員會委員、[[中華民國內政部|中華民國內政部]]著作權出版品審議委員會委員等職務<ref name="崔之清 350-351" /><ref name="中國中外名人文化研究會 192" /><ref name="陈国少 384">{{Cite book | author = 陈国少、肖星和常工 | title = 《台港澳手册》 | location = 中國北京 | publisher = [[華藝出版社|華藝出版社]] | date = 1990年9月 | pages = 第384頁 | ISBN = 978-7800391590 | accessdate = 2018年9月9日 | url =  | language = Zh-hans | quote =  }}</ref>。在僑務委員會任職多年期間，他則主持海外宣傳工作，包括[[僑生|僑生]]回國升學、[[海外華人|海外華人]]回國觀光、海外華人商人回國投資等<ref name = "中華民國當代名人錄"/>。

馮愛羣在早年經使用「艾君」的筆名進行論著，在各家報刊創作和發表詩歌、散文，並且撰寫文學理論和國際政治評論等，但是這些文章並未集結成冊<ref name = "中華民國當代名人錄"/><ref name = "陈国少 384"/>。他還先後擔任海內外報刊記者、編輯和主筆，除了曾經出任《[[實踐_(週刊)|實踐]]》雙週刊社社長外，後來他又轉往[[臺北市|臺北市]]的《[[中華日報|中華日報]]》服務，先後擔任總編輯、總主筆<ref name = "中國中外名人文化研究會 192"/><ref name = "聯合報 01"/>。

與此同時，他還創辦[[臺灣學生書局|臺灣學生書局]]，並且出任董事長一職<ref name = "中國中外名人文化研究會 192"/><ref name = "陈国少 384"/>。他主持臺灣學生書局多年，出版近千本著作，而受到學術界的重視<ref name = "中華民國當代名人錄"/>。同時，他還擔任過[[政治作戰學校|政治作戰學校]]副教授<ref name = "陈国少 384"/>、[[中國文化學院|中國文化學院]]教授等<ref name = "中國中外名人文化研究會 192"/>。1989年1月18日，馮愛羣病逝，享年73歲<ref name = "聯合報 01"/>。
