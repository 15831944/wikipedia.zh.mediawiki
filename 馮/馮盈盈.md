'''馮盈盈'''（{{lang-en| Fung Ying Ying Crystal}}，{{bd|1994年|6月10日}}），香港女主持及演員。現為[[無綫電視藝員列表|無綫電視經理人合約]]藝員，<ref>[http://artiste.tvb.com/main/artistedetail/fungyingying 馮盈盈 Fung Ying Ying - TVB藝人資料]</ref>亦是《[[2016年度香港小姐競選|2016年度香港小姐競選]]》[[冠軍|冠軍]]、《2017年度[[國際中華小姐競選|國際中華小姐競選]]》[[季軍|季軍]]、《[[萬千星輝頒獎典禮2018|萬千星輝頒獎典禮2018]]》「[[萬千星輝頒獎典禮飛躍進步女藝員|飛躍進步女藝員]]」獎得主。

== 個人經歷 ==
=== 早年生活 ===
馮盈盈畢業於[[寶血會嘉靈學校|寶血會嘉靈學校]]{{noteTag|見馮盈盈（crystal.fung.378）在2010年1月19日於Facebook公開群組「寶血會嘉靈學校2006畢業生」的留言：「好耐無見la!!! 我係馮盈盈ar!」。}}，為該校第35屆六信班（6E）畢業生。她於2006年至2012年曾就讀於[[石硤尾|石硤尾]][[寶血會上智英文書院|寶血會上智英文書院]]。她曾在2010年參加第62屆香港校際朗誦節的中五級女子組英語散文獨誦比賽，結果她獲評判給予83分的優良級別，<ref>《[http://www.htc.edu.hk/files/20111031-2010-11%20external%20prizes.pdf 2010-2011 External Prizes]》(htc.edu.hk 2011年10月31日)</ref>以1分之差屈居兩位參與該組別比賽而獲得季軍的同校同學之後，得到第五名。<ref>《[http://www.htc.edu.hk/files/speech63_eng.html 62nd HONG KONG SCHOOLS SPEECH FESTIVAL (2010-2011)]》</ref>而她亦於2011年考獲[[香港青年獎勵計劃|香港青年獎勵計劃]]銀章，以及獲香港青年協會頒發200小時義工服務獎狀。<ref>《[http://www.htc.edu.hk/files/20130909-2011-12_External_Prizes.pdf 2011-2012 External Prizes]》(htc.edu.hk，2013年9月9日)</ref>

馮盈盈在2012年度的[[香港中學文憑|香港中學文憑]]考試中，考獲最佳五科28分的成績{{noteTag|資料來自無綫電視於2017年7月13日播出的節目《[[♯後生仔傾吓偈|#後生仔傾-{吓}-偈]]》第4集「[年經] 校園、自修室瘋狂事件簿」}}，同年獲[[香港大學|香港大學]]取錄，入讀食物及營養學系，並於2016年以二級榮譽乙等成績畢業。<ref>[http://hk.apple.nextmedia.com/entertainment/art/20161204/19854146 馮盈盈港大畢業禮搶鏡](蘋果日報，2016年12月4日)</ref>她曾在2015年8月接受雜誌訪問，表示入讀營養系乃因為其弟弟因腎炎病情嚴重時只能吃清菜及白烚肉，長期不能外出用膳，故此其弟自小便有特別的飲食需要，不能吃含高鹽、高糖、高鈉的食物，並須嚴格控制飲食。<ref name='a'>[http://hk.apple.nextmedia.com/ketchup/art/20150831/19273908 港大女神Crystal 為細佬讀食營](FACE周刊，2015年8月31日)</ref>因此她自小覺得健康的飲食習慣對自己及家人也十分重要，加上自己喜愛吃東西，而令自己對食物及營養學有興趣，故馮盈盈早年已立志入讀該學系，並成為一位註冊[[營養師|營養師]]，協助弟弟及家人料理飲食習慣。<ref name='a' />

馮盈盈於中學及大學期間不時參與角色扮演的活動，吸引一些支持者將她的圖片上載到社交網站及網上討論區，馮盈盈亦因此被封為「港大女神」<ref>《[https://www.facebook.com/hkugoddesses/posts/1443224309320833 女神點少得食營女神]》(臉書專頁：HKU Goddesses - 港大女神，2015年7月10日10時45分)</ref>，同時獲得一些平面及[[宣傳模特兒|宣傳模特兒]]的演出機會，亦曾擔任一些產品推廣宣傳活動的主持工作。馮盈盈曾獲邀參加《百變美人季》的大中華總決賽及亞太區總決賽同時勇奪冠軍，成為化妝品牌的亞太區的廣告代言人，並曾於2016年2月及6月到美國參與拍攝廣告硬照的工作。

=== 參加選美活動 ===
馮盈盈在2016年參加[[香港小姐競選|香港小姐競選]]，並獲香港財經界人士張士佳提名支持。<ref>《[https://www.facebook.com/sky788/photos/a.404550066379175.1073741827.247333838767466/665961066904739/?type=3&theater 最後兩天，圓一個二十年的夢]》(Facebook：《張士佳 - Sky Sir》專頁)，2016年9月8日)</ref>馮盈盈於參選後，曾多次闡述童年時的港姐心願。她曾表示自己由三歲開始便出現了成為香港小姐的夢想<ref>《Think Big 天地》之《Little Miss Hong Kong》環節，(TVB，2018年8月10日)</ref>，並從電視中看到香港小姐擁有美麗、雍容華貴和自信的氣質，令自己十分憧憬。<ref>今日VIP《2016香港小姐候選佳麗(下)》(TVB，2016年9月7日)</ref>另一方面，由於自己一直期待做一件出人頭地的事，而令父親感到驕傲，加上其男友鼓勵，最終報名參選。<ref>《[http://www.orientalsunday.hk/%e6%9c%80%e6%96%b0%e5%a8%9b%e8%81%9e/%e4%b8%8a%e7%af%80%e7%9b%ae%e7%99%bc%e6%8f%ae%e6%b8%af%e5%a5%b3%e6%9c%ac%e8%89%b2-%e9%a6%ae%e7%9b%88%e7%9b%88%e5%a4%a7%e7%88%86%e7%82%ba%e7%88%b8%e7%88%b8%e9%81%b8%e6%b8%af%e5%a7%90/2/ 上節目發揮港女本色 馮盈盈大爆為爸爸選港姐]》(東方新地網頁，2017年3月28日)</ref>其後，馮盈盈於經過2016年6月及7月的三輪淘汰之後，成功躋身20強佳麗之列。<ref>《港姐進入最後淘汰階段　最後十強由評審及觀眾選出》(TVB娛樂新聞，2016年8月4日)</ref>而馮盈盈在2016年8月的觀眾投票中，獲得較多觀眾支持晉身十強。自電視台於2016年7月公佈香港小姐參賽名單以來，馮盈盈便一直被傳媒及公眾視為「大熱門」候選佳麗。 <ref>《港姐十強完成福岡拍攝回港　關可欣陳雅思難忘學劍道》(TVB娛樂新聞，2016年8月22日)</ref>而馮盈盈亦於2016年9月2日獲大會頒發「現場最受歡迎佳麗獎」。她最終在《2016年度香港小姐競選》決賽中以36,679人的支持票（共有233,394名投票觀眾）贏得冠軍。<ref>{{youtube|MWBAR89pDF0|《2016香港小姐競選決賽》冠軍得主 - 6號馮盈盈}} (TVB，2016年9月11日)</ref>

馮盈盈在當選香港小姐冠軍之後，無綫電視亦按照慣例，於同年10月15日宣佈由她代表香港出選《2017年度[[國際中華小姐競選|國際中華小姐競選]]》。在參賽期間，她跟馬來西亞參選佳麗[[林宣妤|林宣妤]]一同被香港的傳媒視為「大熱門」。在2017年1月15日《2017年度國際中華小姐競選》於馬來西亞雲頂舉行期間，馮盈盈入圍最後五強，並以魔術作為才藝表演的選項。結果，她於[[馬來西亞|馬來西亞]]贏得2017年度國際中華小姐競選季軍。

=== 演藝事業 === 
馮盈盈於參選香港小姐後，於2017年起成為[[無綫電視|無綫電視]]女藝員，由於其破格言論及「貼地」形象深入民心，而廣受注意。

2017年，她開始擔任資訊節目《[[學是學非|學是學非]]》的主持人和年輕人清談節目《[[♯後生仔傾吓偈|#後生仔傾-{吓}-偈]]》的主持人，與[[陸浩明|陸浩明]]和2015年度香港小姐冠軍[[麥明詩|麥明詩]]搭檔。馮盈盈在節目中所提及的「原來係[[處女|處]]」及「1同0」的言論引起廣泛討論而被視為初期代表作，同時也因為經常在節目中豪放地大談性話題、發表出位言論而受人注目，常成為娛樂新聞的焦點，更被網民封以「性博士」的外號<ref name="獲網友封為「性博士」">{{Cite web |url=https://www.cosmopolitan.com.hk/love/15f6cb07fd06f35 |title=獲網友封為「性博士」！貼地港姐馮盈盈《後生仔傾-{吓}-偈》夠放 |date=2017-10-30 |publisher=[[Cosmopolitan|Cosmopolitan]]}}</ref><ref>{{Cite web |url=https://www.hk01.com/即時娛樂/190905/ |title=被網友封性博士？　馮盈盈：唔會覺得太負面 |date=2018-05-21 |publisher=香港01}}</ref><ref>{{Cite web |url=https://hk.ulifestyle.com.hk/topic/detail/205174/%E5%BE%8C%E7%94%9F%E4%BB%94%E5%82%BE%E4%B8%8B%E5%81%88-%E9%A6%AE%E7%9B%88%E7%9B%88%E7%9B%A1%E9%A1%AF%E7%9C%9F%E6%80%A7%E6%83%85-%E8%A8%80%E8%AB%96%E7%84%A1%E5%BA%95%E7%B7%9A%E4%BB%A4%E4%BA%BA%E6%83%B3%E5%85%A5%E9%9D%9E%E9%9D%9E/1 |title=《後生仔傾-{下}-偈》馮盈盈盡顯真性情 言論無底線令人想入非非 |author=石榴姐 |date=2018-05-11 |publisher=ulifestyle |language=zh-yue}}</ref>，她曾自認其樣貌和[[AV女優|AV女優]][[桃谷繪里香|桃谷繪里香]]相似<ref>{{cite web |url=https://hk.on.cc/hk/bkn/cnt/entertainment/20180704/mobile/bkn-20180704223123293-0704_00862_001.html |title=性博士論盡AV　馮盈盈挑戰桃谷繪里香 |date=2018-07-04 |publisher=東方日報 |language=zh-yue}}</ref>。

=== 爭議事件 ===
馮盈盈於2016年9月11日晚上當選香港小姐冠軍後，隨即被揭發於2014年[[雨傘革命|雨傘革命]]期間，在[[面書|面書]]以大量[[粵語粗口|粵語粗口]]批評[[鍾樹根|鍾樹根]]、[[香港政府|香港政府]]及[[香港警察|警察]]<ref name="爆粗撐佔領">{{Cite web |url = https://www.thestandnews.com/politics/%E6%9B%BE%E7%88%86%E7%B2%97%E6%92%90%E4%BD%94%E9%A0%98-%E9%A6%AE%E7%9B%88%E7%9B%88-%E7%95%B6%E6%99%82%E5%B0%91%E4%B8%8D%E6%9B%B4%E4%BA%8B-%E5%8F%97%E5%A4%A7%E5%AD%B8%E7%8B%82%E7%86%B1%E6%B0%A3%E6%B0%9B%E5%BD%B1%E9%9F%BF/ |title=曾爆粗撐佔領　馮盈盈：當時少不更事　受大學狂熱氣氛影響 |date=2016-09-12 |publisher=立場新聞 |accessdate=2016-09-13}}</ref>，反對[[梁振英政府|梁振英政府]]及表態支持示威者，令她被冠以「粗口港姐」的稱號。其言論被廣泛報導後，馮盈盈於當選翌日接受傳媒訪問時，表示自己當年「少不更事」，但關心香港才「愛之深責之切」，強調自己私下並不會說粗話。加上當時剛入讀大學，受到大學內的「狂熱氣氛」影響，才會發表較激烈的言論<ref name="爆粗撐佔領"/>。然而由於其參選香港小姐時的表現，以及相關的破格言論，使她成為2016年度[[谷歌|谷歌]]搜尋排行榜內「香港熱爆本地時事人物」第八位。<ref>《[https://www.google.com.hk/trends/yis/2016/HK 2016年度谷歌搜尋排行榜]》(谷歌香港，2016年12月14日公佈)</ref>

== 感情及家庭生活 ==
馮盈盈曾透露，她的弟弟比自己小四年。由於感情要好，所以被雜誌形容為「零代溝」。另外，其弟的病患令她養成習慣，在購買食物前必定會參考食品包裝上的營養標籤。<ref>《[http://hk.apple.nextmedia.com/ketchup/art/20150831/19273908 港大女神Crystal 為細佬讀食營]》(FACE周刊，2015年8月31日)</ref>她曾表示自己跟弟弟平常也會吵吵鬧鬧，卻又互相疼惜對方，亦不時會跟弟弟用膳。<ref>《[http://hk.apple.nextmedia.com/ketchup/art/20150831/19273908 港大女神Crystal 為細佬讀食營]》(FACE周刊，2015年8月31日)</ref>

當馮盈盈於2016年參選港姐的時候，她便透露了自己已經有男朋友。其後傳媒報導她的男友為於2005年[[香港中學會考|香港中學會考]]中考獲十優成績、在[[瑪麗醫院|瑪麗醫院]]任職內科駐院醫生的鍾文浩 （Martin）<ref>《[http://www.bastillepost.com/hongkong/4-%E5%A8%9B%E5%9C%88%E4%BA%8B/1399005-%E9%A6%AE%E7%9B%88%E7%9B%88%E7%8C%9B%E6%96%99%E9%86%AB%E7%94%9F%E7%94%B7%E5%8F%8B%E7%82%BA%E5%8D%81%E5%84%AA%E7%8B%80%E5%85%83?r=w 馮盈盈猛料醫生男友為十優狀元]》(巴士的報，2016年9月12日)</ref>。對方當時表示支持她參選香港小姐。<ref>《[http://news.mingpao.com/pns/dailynews/web_tc/article/20160610/s00016/1465494835461 港姐大起底 Crystal Nicole Maggie 網上曬泳照鬥性感]》(明報，2016年6月10日，C02版) </ref>

除此之外，馮盈盈於2017年3月26日深夜播出的電台節目《公子會》訪問中提到，自己於大學一年級時認識了另一位男同學，並於隨後的情人節相約到戲院觀看《孤星淚》而開始拍拖。但後因發覺與當時男友的思想分歧日增，故在拍拖兩周年的情人節後提出分手。<ref>《[https://www.my903.tk/公子會2017-03-25.html 公子會訪問] {{webarchive|url=https://web.archive.org/web/20170809145859/http://www.my903.tk/%E5%85%AC%E5%AD%90%E6%9C%832017-03-25.html |date=2017-08-09 }}》(香港商業電台，2017年3月26日)</ref>
