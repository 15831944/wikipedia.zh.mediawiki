'''馮婉儀'''（{{lang|en|'''Connie Fung Yuen Yee'''}}），香港女性社會工作者及治療師，是首個在港主張以「[[園藝治療|園藝治療]]」回復患者心理與精神健康的人士。

== 簡歷 ==
馮婉儀是位社會工作者，2003年加入美國園藝治療協會，2005年創立了香港園藝治療中心，提供專業園藝治療培訓項目、治療園景設計等服務或課程<ref>{{cite web |author1= |title=我係植物人 |url=https://hk.lifestyle.appledaily.com/nextplus/magazine/article/20060719/2_6134639/%E6%88%91-%E4%BF%82-%E6%A4%8D-%E7%89%A9-%E4%BA%BA |work=[[蘋果日報_(香港)|蘋果日報]] |date=2006-07-19 |accessdate=2018-08-25}}</ref>；2008年亦設置[[香港園藝治療協會|香港園藝治療協會]]（HKATH）和擔任會長至今。她又曾在2014年、2016年推出有關著作推廣園藝治療，包括《園藝治療–種出身心好健康》及《園藝治療理念與應用論文選集》。

馮婉儀現時擔任中國社會工作聯合會名譽顧問<ref>{{cite web |author1=Hong Kong Association of Therapeutic Horticulture |title=香港園藝治療協會會長 |url=http://www.hkath.org/%E5%A7%94%E5%93%A1%E6%9C%83/#president |work=香港園藝治療協會 |date=2018-08-25 |accessdate=2018-08-25}}</ref>，主力在香港、內地、[[澳門|澳門]]、[[台灣|台灣]]與[[韓國|韓國]]等地繼續從事推廣工作<ref>{{cite web |author1=manyee |title=花草樹木無言詩 療癒花園治情困 |url=http://www.3weekly.com/common_emag/view_content.php?issue_no=762&pub_id=7&top_id=18&art_id=190451 |work=[[3週刊|3週刊]] 第762期 |date=2014-05-15 |accessdate=2018-08-25}}</ref><ref>{{cite web |author1=章凱閎 |title=香港首位園藝治療師馮婉儀 用植栽讓失能長輩回春 |url=https://vision.udn.com/vision/story/9946/2812834 |work=[[聯合報|聯合報]] |date=2017-11-11 |accessdate=2018-08-25}}</ref>，直至2015年便獲美國園藝治療協會頌發「2015年度蕾亞·麥肯迪尼斯專業服務獎」以表揚該方面的長期貢獻。
