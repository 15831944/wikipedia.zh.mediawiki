'''冯立环'''（1968年－），画家，[[江苏省|江苏省]][[南京|南京]]人，就读于[[北京大学|北京大学]]艺术学院，现任[[中国书画院|中国书画院]]副秘书长、[[亚洲中文美术学院|亚洲中文美术学院]]客座教授、[[世界教科文卫组织|世界教科文卫组织]]专员、[[联合国和平周|联合国和平周]]中国组委会荣誉委员等职务。成就录入《[[世界名人录|世界名人录]]》、《中国当代书画界名人录》、《世界华人文学艺术界名人录》、《世界优秀人才大典》、《中国历代画梅百家》等权威辞书。  

== 作品收藏 ==
冯立环创作的国画作品被[[天安门|天安门]]城楼、联合国和平周、军事博物馆、[[中国文化部|中国文化部]]、三军仪仗队、[[周恩来纪念馆|周恩来纪念馆]]、[[北京奥组委|北京奥组委]]、[[上海世博会|上海世博会]]等单位收藏。多次应邀出席国内外画展及文化交流、学术报告、讲学等活动。

== 媒体报道 ==
其事迹曾被《世界日报》、《亚洲日报》、《人民日报》、《光明日报》、《中国日报》、《中国社会报》、《西藏日报》、《南京日报》、《神州诗书画报》、《中国企业家报》等报道。

== 社会荣誉 ==
2010年获上海世博会“赢聚世博城市之星”荣誉奖，2012年获全国央视才艺大赛“突出贡献奖”，2013年被中国文化部文化艺术中心授予“艺苑英杰”称号。
