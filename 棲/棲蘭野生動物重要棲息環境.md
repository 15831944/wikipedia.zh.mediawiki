'''棲蘭野生動物重要棲息環境'''是[[台灣|台灣]]一處[[臺灣自然生態保護區#野生動物重要棲息環境|野生動物保護區]]，範圍大致以[[宜蘭縣|宜蘭]]、[[新北市|新北]]、[[桃園市|桃園]]、[[新竹縣|新竹]]四縣市交界的山區為主，涵蓋[[林務局|林務局]]所轄之烏來、大溪、宜蘭、太平山等事業區，全部面積總計55,991.41公頃<ref>{{Cite web |url=http://conservation.forest.gov.tw/0000163 |title=棲蘭野生動物重要棲息環境|work=自然保育網 |publisher=林務局|language=zh-tw |date=2016-07-13}}</ref>。為實行「中央山脈生態廊道」，以保存珍貴動植物資源與多樣性的生態環境，行政院農業委員會於2000年2月15日依《野生動物保護法》公告設立。

==歷史==
2000年2月，[[行政院農業委員會|農委會]]依據《野生動物保育法》，公告位於[[新北市|臺北縣]]、[[宜蘭縣|宜蘭縣]]、[[桃園市|桃園縣]]以及[[新竹縣|新竹縣]]等四縣市交界區域為棲蘭野生動物重要棲息環境<ref>{{Cite web |url=http://conservation.forest.gov.tw/ct.asp?xItem=3264&ctNode=176&mp=10 |title=棲蘭野生動物重要棲息環境 |accessdate=2013-02-17 |publisher=農委會林務局 |deadurl=yes |archiveurl=https://web.archive.org/web/20121031123507/http://conservation.forest.gov.tw/ct.asp?xItem=3264&ctNode=176&mp=10 |archivedate=2012-10-31 }}</ref>。當時公告的保護區域另有[[丹大野生動物重要棲息環境|丹大野生動物重要棲息環境]]以及[[關山野生動物重要棲息環境|關山野生動物重要棲息環境]]，皆是為了配合[[中央山脈|中央山脈]]保育廊道的劃設構想。棲蘭野生動物重要棲息環境劃設範圍達55,991.41公頃，北面接攘[[插天山自然保留區|插天山自然保留區]]，東南方毗鄰[[太魯閣國家公園|太魯閣國家公園]]，西南處則和[[雪霸國家公園|雪霸國家公園]]為界。

2002年，[[中華民國內政部|內政部]]公告成立馬告國家公園，面積約53,602公頃，預定範圍的土地權分屬[[行政院國軍退除役官兵輔導委員會|退輔會]]及[[行政院農業委員會|農委會]]<ref name="說明書">{{Cite web |url=http://npda.cpami.gov.tw/media_src/attach/51/Cpami-np-bk-inp0009-0001.pdf |title=馬告檜木國家公園範圍劃設說明書 |publisher=內政部營建署 |date=2002-05}}</ref><ref>{{Cite web |url=http://npda.cpami.gov.tw/tab2/web02_index.php?k1=F&k2=FC&k3=&kw=&page=2&id=C14325 |title=馬告槍木國家公園預定地範圍圖 |publisher=內政部營建署 |date=2008 |accessdate=2014-07-09}}</ref><ref>{{Cite web |url=http://npda.cpami.gov.tw/tab2/web02_index.php?k1=F&k2=FC&k3=&kw=&page=2&id=C14324 |title=馬告國家公園計畫圖 |publisher=內政部營建署 |accessdate=2014-07-09}}</ref>。公告範圍內目前有由林務局公告之鴛鴦湖自然保留區及棲蘭野生動物重要棲息環境兩處自然保護區域。但由於當地[[臺灣原住民|原住民]][[泰雅族|泰雅族]]反對，至今未成立管理處<ref>{{Cite web |url=http://npda.cpami.gov.tw/tab1/web1_main.php?mod=E2 |title=未完成的國家公園-馬告 |publisher=內政部 |accessdate=2014-07-09}}</ref><ref>{{Cite news|url=http://www.libertytimes.com.tw/2008/new/dec/31/today-north21.htm|title=催生馬告國家公園 發動連署|author=江志雄|publisher=自由時報|date=2008-12-31|deadurl=yes|archiveurl=https://web.archive.org/web/20100507184945/http://www.libertytimes.com.tw/2008/new/dec/31/today-north21.htm|archivedate=2010-05-07}}</ref>。

==台灣世界遺產潛力點==
當地的[[棲蘭山林場|棲蘭山林區]]保有一萬多公頃的[[扁柏屬|檜木]]林，此處檜木林於2002年由[[中華民國文化部|行政院文建會]]徵詢國內專家及函請縣市政府與地方文史工作室提報、推薦評選為臺灣[[世界遺產|世界遺產]]潛力點之一，以[[台灣扁柏|扁柏]]為大部分，[[紅檜|紅檜]]為小面積或零星分布<ref>{{Cite web|url=http://conservation.forest.gov.tw/public/Attachment/61241143771.pdf|title=棲蘭野生動物重要棲息環境植群生態調查之研究|format=PDF|author=呂金誠等人|publisher=林務局新竹林區管理處|date=2002-12|deadurl=yes|archiveurl=https://web.archive.org/web/20160304100933/http://conservation.forest.gov.tw/public/Attachment/61241143771.pdf|archivedate=2016-03-04}}</ref><ref>{{Cite web |url=http://conservation.forest.gov.tw/public/Attachment/612417173971.pdf |title=棲蘭野生動物重要棲息環境植物相調查之研究II |format=PDF |author=呂金誠等人 |publisher=林務局新竹林區管理處 |date=2003-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304105719/http://conservation.forest.gov.tw/public/Attachment/612417173971.pdf |archivedate=2016-03-04 }}</ref><ref>{{Cite web |url = http://twh.boch.gov.tw/taiwan/index.aspx?lang=zh_tw|title = 臺灣世界遺產潛力點|publisher = 文化部文化資產局|accessdate = 2016-01-03}}</ref>。

棲蘭山檜木林處於中高海拔的山區，此山域因峰高、谷深、雨霧足，形成孤島式的封閉性生態環境，使得伴生於檜木林帶之珍稀裸子植物，如[[紅豆杉|紅豆杉]]、[[臺灣杉|臺灣杉]]、[[巒大杉|巒大杉]]、[[臺灣粗榧|臺灣粗榧]]等北極第三紀[[孑遺植物|孑遺植物]]，因長期隔離演化，形成臺灣僅有的特有種，而這些特有種針葉類珍稀裸子植物群，歷經數千萬年至上億年的演替，堪稱「活化石樹」，在生態演化上具有指標地位，符合世界遺產登錄標準第八項<ref>{{Cite web|url=http://twh.hach.gov.tw/TaiwanContent.action?id=9#|title=棲蘭山檜木林|publisher=文化部文化資產局|accessdate=2013-03-04}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

從雨量記錄來看，棲蘭山山區的雨量年平均有高達5,000公釐的紀錄，此山區一年之中，幾乎有250天雨霧濛濛，這種飽含水分終年雲霧繚繞的林地即是植物學界所稱之的「霧林帶」。本區之自然環境生態系，學術界稱為「暖溫帶山地針葉樹林群系」。在[[植物學|植物學]]上又將此群系分為兩大植物生態社會，一為針葉混生社會，另一位檜木林型社會。其中檜木林社會分布海拔（1,600—2,600公尺）因氣候較暖又濕潤多雨，以致植物社會組成非常多樣，就物種歧異度而言，植物種類龐雜，其中有許多臺灣已瀕臨絕種及珍貴物種生長其間，在這些珍稀針葉樹的林況下，並發現有[[臺灣黑熊|臺灣黑熊]]、[[臺灣野山羊|臺灣野山羊]]、[[山羌|山羌]]等大型蹄科動物。此處成為臺灣珍稀野生動物的棲息天堂，極具研究與保育價值，符合世界遺產登錄標準第十項。
