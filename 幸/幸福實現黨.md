'''幸福實現黨'''（{{lang-ja|幸福実現党}}；{{lang-en|Happiness Realization Party}}）是[[日本|日本]]的[[政黨|政黨]]，乃[[宗教法人|宗教法人]][[幸福科學|幸福科學]]的分支，2009年5月成立。

== 历史 ==
=== 宣佈組黨 ===
[[File:Ryuho_Okawa_2009.jpg|thumb]]

在宣佈組黨的[[記者會|記者會]]上，幸福實現黨同時宣佈參選2009年[[第45屆日本眾議院議員總選舉|第45屆日本眾議院議員總選舉]]，由[[饗庭直道|饗庭直道]]出任黨首，[[漫畫家|漫畫家]][[佐藤文也|佐藤文也]]在[[福岡縣|福岡縣]]第8區參選，前[[The_Blue_Hearts|The Blue Hearts]]成員[[河口純之助|河口純之助]]在[[東京都|東京都]]比例區參選<ref>[http://www.jiji.com/jc/c?g=pol_30&k=2009052500641 衆院選に候補擁立=「幸福の科学」]（[[時事通信|時事通信]]・2009年5月25日）</ref><ref>{{cite web |url = http://hochi.yomiuri.co.jp/topics/news/20090528-OHT1T00064.htm |title=元ブルーハーツの河口さん「幸福実現党」から出馬 |date=2009年5月28日 |accessdate=2009年5月29日 |author=[[スポーツ報知|スポーツ報知]] |archiveurl=https://web.archive.org/web/20090531055425/http://hochi.yomiuri.co.jp/topics/news/20090528-OHT1T00064.htm |archivedate=2009-05-31 |deadurl=yes }}</ref><ref>{{Cite web |url = http://sankei.jp.msn.com/politics/situation/090527/stt0905272132007-n1.htm |title=幸福実現党、1次公認候補に漫画家さとうふみや氏ら17人 |date=2009年5月27日 |accessdate=2009年5月29日|author=MSN産経ニュース |deadurl=yes |archiveurl=https://web.archive.org/web/20090530073537/http://sankei.jp.msn.com/politics/situation/090527/stt0905272132007-n1.htm |archivedate=2009年5月30日 }}</ref>。

2009年5月31日，幸福實現黨創辦人[[大川隆法|大川隆法]]相隔十八年在[[大阪市|大阪市]]公開演說，表示組黨原因是「雖然支持[[安倍晋三|安倍晋三]]與[[麻生太郎|麻生太郎]]，但還是沒用」。在黨成立大會後隨即表明要在全國全部選區與比例區派代表參選，目標成為眾議院第一大黨。關於對[[北朝鮮|北朝鮮]]的政策，幸福實現黨宣稱曾經與「[[金正日|金正日]]的守護靈」對話，守護靈宣稱「要以核武之力三年內統一韓國，瞄準日本的飛彈是為了嚇怕日本、削弱美國氣力的掩護作戰」，因而要提防北朝鮮。至於被指責[[政教合一|政教合一]]，黨反駁「政治不能夠脫離宗教獨立存在」<ref>{{cite web |url = http://www.zakzak.co.jp/top/200906/t2009060133_all.html |title=大川隆法18年ぶり公開説法…衛星中継で海外配信も |date=2009年6月1日 |accessdate=2009年6月2日|author=ZAKZAK|deadurl=yes|archiveurl=https://web.archive.org/web/20151222153832/http://www.zakzak.co.jp/top/200906/t2009060133_all.html|archivedate=2015年12月22日}}</ref>。

[[2009年東京都議會議員選舉|2009年東京都議會議員選舉]]，幸福實現黨派出十人參選，全部落選。

2009年6月4日，大川隆法的妻子[[大川恭子|大川恭子]]繼任幸福實現黨黨首。而大川隆法在7月22日出任幸福實現黨總裁。大川恭子在7月29日卸任黨首，大川隆法成為幸福實現黨的單一領導人。

===第45屆日本眾議院議員總選舉===
雖然幸福實現黨在2009年[[第45屆日本眾議院議員總選舉|第45屆日本眾議院議員總選舉]]派出337名候選人，但因為法律上不符合政黨的規定（已得國會議席五席或得票率百分之二以上），故只能稱為政治團體；傳媒歸類為「其他派」，不講其黨名。因此「其他派」的候選人都比[[自由民主党_(日本)|自由民主黨]]、[[民主党_(日本)|民主黨]]等黨的候選人受到不公平待遇，[[NHK|NHK]]一類舉辦的候選論壇都不邀請出席。

大川隆法本人參與東京比例區的第一名，大川今日子出戰東北比例區。2009年8月13日，傳媒報導，幸福實現黨為了不分裂保守系統的選票，希望全面退出選舉(8月18日為選舉公告及報名日)。2009年8月16日，幸福實現黨宣佈不會退出選舉，大川隆法則轉戰近畿比例區第一名。

選舉結果，337名候選人皆落選；而且因為得票率不足，保證金均不能返還，合共11億5800萬圓。<ref>{{cite web|url=http://www.hokkaido-np.co.jp/news/2009syuinsen/186215.html|title=衆院選供託金の没収額　道内１億4100万円　前回の２倍以上に|date=2009年8月31日|accessdate=2009年8月31日|author=Doshin Web(北海道新聞)|deadurl=yes|archiveurl=https://web.archive.org/web/20090905133203/http://www.hokkaido-np.co.jp/news/2009syuinsen/186215.html|archivedate=2009年9月5日}}</ref>

由於眾議院選舉全敗，幸福實現黨於2009年9月2日任命[[本地川瑞祥|本地川瑞祥]]出任黨首。2009年9月12日，黨總裁大川隆法與黨首本地川瑞祥離任，由[[木村智重|木村智重]]出任黨首。

== 政策 ==
全面废除[[消费税|消费税]]、[[遗产税|遗产税]]、[[赠与税|赠与税]]；对[[北朝鲜|北朝鲜]]采取先发制人的军事行动；到2030年为止使日本的人口到达3亿，[[GDP|GDP]]达世界第一。

=== 近来的政策 ===
2010年1月时该党的政策微微缓和，包括：<ref>{{cite web |url = http://www.hr-party.jp/inauguration/dingpolicy.html |title=主要政策（2010年1月）|date=2010年2月15日|accessdate=2010年2月12日|author=幸福実現党|deadurl=yes|archiveurl=https://web.archive.org/web/20100507162910/http://www.hr-party.jp/inauguration/dingpolicy.html|archivedate=2010年5月7日}}</ref>
*废除[[遗产税|遗产税]]、[[赠与税|赠与税]]以及[[证券税|证券税]]。增加对新兴产业以及公共事业的投资。
*对[[中华人民共和国|中国]]以及[[北朝鲜|北朝鲜]]采取强硬态度，以及修改[[日本宪法第九条|日本宪法第九条]]。
*改正教育中的[[自虐史观|自虐史观]]，尝试使公立学校民营化。

== 评价==
幸福实现党本身依靠[[幸福科學|幸福科學]]组织建立，第一任党总裁大川隆法自称能与灵界相通，在党的宣言里宣称党的部分政策是大川隆法从[[诸葛亮|诸葛亮]]的[[灵魂|灵魂]]那里得到的<ref>{{cite web |title=幸福実現党宣言 |url = http://www.hr-party.jp/inauguration/declaration.html |deadurl=yes |archiveurl = https://web.archive.org/web/20090826152328/http://www.hr-party.jp/inauguration/declaration.html |archivedate=2009-08-26}}</ref>。政治思想让很多[[左翼|左翼]]人士不满。由於2012年[[釣魚台國有化|釣魚台國有化]]事件的因素，和電影《[[最後的審判_(電影)|最後的審判]]》<ref>[http://fj2012.com/cnh/ 電影《最後的審判》官方網站]</ref>的上映，部份人士開始認為大川隆法說的有道理，日本必須加強國防，不能一味依靠美國的幫助<ref>[http://www.china.com.cn/international/txt/2009-08/26/content_18405244.htm 近距离看日本大选：“愤青”政党——幸福实现党]</ref>。
