《'''幸福路上'''》（{{Lang-en|'''On Happiness Road'''}}）是臺灣歌手[[蔡依林|蔡依林]]的一支單曲，該歌曲是臺灣2018年電影《[[幸福路上_(電影)|幸福路上]]》主題曲，由[[環球音樂_(台灣)|環球音樂]]於2017年11月20日發行<ref>{{Cite web|url=http://www.sohu.com/a/204462378_267106|title=蔡依林全新单曲《幸福路上》 11月20日上线|accessdate=2017-11-15|date=2017-11-15|work=www.sohu.com}}</ref><ref>{{Cite web|url=http://isrc.ncl.edu.tw/music_view.aspx?item=22424|title=ISRC-國際標準錄音錄影資料代碼查詢系統 International Standard Recording Code|accessdate=2018-01-08|work=isrc.ncl.edu.tw}}</ref>。

== 背景與發行 ==
2017年9月，臺灣歌手[[蔡依林|蔡依林]]表示於同年年底將發行一首歌曲，並表示該歌曲風格特別<ref>{{Cite web|url=http://ent.sina.com.cn/y/ygangtai/2017-09-11/doc-ifykuffc5168922.shtml|title=蔡依林预告新歌不是舞曲 新专辑进度严重落后|accessdate=2017-11-13|work=ent.sina.com.cn}}</ref>。同年11月13日，臺灣電影《[[幸福路上_(電影)|幸福路上]]》Facebook粉絲頁發布電影同名主題曲45秒片段，但並未明確表示演唱者姓名，而網友根據歌聲猜測演唱者為歌手蔡依林<ref name=":2">{{Cite news|url=https://news.tvbs.com.tw/entertainment/814259|title=「隱藏歌姬」獻聲電影主題曲　網友瘋猜：是蔡依林？│TVBS新聞網|last=TVBS|newspaper=TVBS|accessdate=2017-11-14}}</ref>。歌手蔡依林表示願為電影獻唱主題曲，一方面因深受電影精神感動，另一方面亦將該歌曲獻給家鄉[[新莊區|新莊]]。電影《幸福路上》導演宋欣穎同為新莊出身，表示蔡依林為其心中電影主題曲演唱者的不二人選，並表示「不只因為我是Jolin的歌迷，更因為蔡依林本身的故事就是《幸福路上》的故事。她是故鄉土地培養出來的野生玫瑰，美麗而堅毅<ref>{{Cite web|url=https://www.msn.com/zh-tw/entertainment/news/%E8%94%A1%E4%BE%9D%E6%9E%97%E7%8D%BB%E5%94%B1%E3%80%8A%E5%B9%B8%E7%A6%8F%E8%B7%AF%E4%B8%8A%E3%80%8B%E9%9B%BB%E5%BD%B1%E4%B8%BB%E9%A1%8C%E6%9B%B2-%E7%8D%BB%E5%87%BA%E4%B8%89%E6%AD%B2%E5%B0%8Fjolin%E7%85%A7/ar-BBF05b5|title=蔡依林獻唱《幸福路上》電影主題曲 獻出三歲小JOLIN照|accessdate=2017-11-15|work=www.msn.com}}</ref>」蔡依林亦表示：「這次演唱方式跟以往詮釋快歌或者抒情歌非常不一樣，希望能用『述說』的方式，講出《幸福路上》這個女孩的動人故事。<ref>{{Cite news|url=https://www.nownews.com/news/20171115/2645092|title=影／蔡依林全新單曲曝光　三歲可愛照當封面|newspaper=NOWnews 今日新聞|accessdate=2017-11-15|language=zh-hant}}</ref>」該歌曲由音樂人[[李焯雄|李焯雄]]作詞、米奇林作曲、[[陳建騏|陳建騏]]製作、董運昌與劉涵編曲，而該歌曲結尾部分歌詞與旋律取自歌手[[鳳飛飛|鳳飛飛]]的歌曲《[[祝你幸福|祝你幸福]]》，歌手蔡依林亦用溫暖有力的聲音演繹該歌曲。此外，歌曲《幸福路上》的封面選用歌手蔡依林兒時的照片，蔡依林表示此為呼應電影女主角「林淑琪」的自我提問：「成為你理想中的大人了嗎？<ref>{{Cite news|url=https://hk.celebrity.yahoo.com/%E5%BD%B1-%E8%94%A1%E4%BE%9D%E6%9E%97%E5%85%A8%E6%96%B0%E5%96%AE%E6%9B%B2%E6%9B%9D%E5%85%89-%E4%B8%89%E6%AD%B2%E5%8F%AF%E6%84%9B%E7%85%A7%E7%95%B6%E5%B0%81%E9%9D%A2-151639614.html|title=影／蔡依林全新單曲曝光 三歲可愛照當封面|accessdate=2017-11-15|language=zh-Hant-HK}}</ref>」蔡依林表示希望通過該歌曲祝福每一個努力生活的人，並表示：「就算挫折受傷，只要真心好好擁抱自己，你就在自己的幸福路上。<ref>{{Cite web|url=http://ent.sina.com.cn/y/ygangtai/2017-11-15/doc-ifynstfh9197058.shtml|title=蔡依林单曲《幸福路上》将上线 童年照成新歌封面{{!}}蔡依林{{!}}幸福路上{{!}}祝你幸福_新浪娱乐_新浪网|accessdate=2017-11-15|work=ent.sina.com.cn}}</ref>」該歌曲MV亦由電影《幸福路上》導演宋欣穎執導，並於歌曲發行同日發布。

== 現場表演 ==
2017年11月25日，臺灣歌手[[蔡依林|蔡依林]]於[[第54屆金馬獎|第54屆臺灣金馬獎頒獎典禮]]登台表演節目「時代歌后·幸福記憶」，除首度演唱歌曲《幸福路上》外，亦演唱歌手[[鳳飛飛|鳳飛飛]]、[[鄧麗君|鄧麗君]]、[[甄妮|甄妮]]、[[蔡琴|蔡琴]]曾演唱過的電影主題曲<ref>{{Cite news|url=https://www.nownews.com/news/20171118/2646687|title=蔡依林首登金馬獎獻唱　致敬天后前輩鳳飛飛、鄧麗君|newspaper=NOWnews 今日新聞|accessdate=2017-11-19|language=zh-hant}}</ref>。之後，亦於第11屆[[音乐盛典咪咕汇|音樂盛典咪咕匯]]表演該歌曲<ref>{{Cite news|url=http://www.chinatimes.com/newspapers/20171217000618-260112|title=蔡依林抱3獎送性感|last=中時電子報|newspaper=中時電子報|accessdate=2018-04-15|language=zh-TW}}</ref>。

== 商業成績 ==
該歌曲於中國大陸[[腾讯|騰訊]]公司旗下網路音樂平台[[QQ音乐|QQ音樂]]、[[酷狗音乐|酷狗音樂]]與[[酷狗音乐|酷我音樂]]獨家發行付費數位單曲，是蔡依林首次於中國大陸發行的付費數位音樂作品；該歌曲於中國大陸的數位單曲銷量超過16萬<ref>{{Cite web|url=http://y.qq.com/msa/Jolin/index_new.html?g_f=share_music|title=QQ音乐,音乐你的生活!|accessdate=2018-09-19|last=Tencent-ISUX-Music|work=y.qq.com|language=zh-cn}}</ref><ref>{{Cite web|url=http://zhuanjistatic.kugou.com/html/pc_commonchargeV3/index_101968.html?is_go=0&hreffrom=13|title=“幸福天后”邀你一起找回幸福|accessdate=2018-09-19|work=zhuanjistatic.kugou.com}}</ref><ref>{{Cite web|url=http://vip1.kuwo.cn/fans/fans/template/index.html?key=cyl20171120&fromSrc=11&fromkw=share|title=http://vip1.kuwo.cn/fans/fans/template/index.html?key=cyl20171120&fromSrc=11&fromkw=share|accessdate=2018-09-19|work=vip1.kuwo.cn}}</ref>。
