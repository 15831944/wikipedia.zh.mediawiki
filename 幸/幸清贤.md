'''幸清贤'''（{{bd|1966年|6月4日||}}），[[贵阳市|贵阳市]]人，[[维权人士|维权人士]]，[[公民记者|公民记者]]，社会活动家，现居[[成都市|成都市]]。

== 生平 ==

*汉族，初中学历，祖籍四川成都，户口所在地：南明区富源南路370号，1982年参加工作，在中铁二局新运工程有限公司工作25年，在职期间，一直热心公益事业，连任过三届团支部工作，从一个普通工人，自学成为机械维修技师，取得由四川省劳动厅颁发的工程机械2级维修资质证书。<ref name=":0">{{Cite web|url = http://beijingspring.com/c7/xw/rqmy/20111031185920.htm|title = 幸清贤逐户发竞选宣言，陈西、李任科被带走|date = 2011-10-31|author = 张明|publisher = 维权网}}</ref>
*幸清贤原是中国铁路第二工程局的职工，因劳动争议一直在与该局[[打官司|打官司]]维权。<ref name=":1">{{Cite web|url = http://www.msguancha.com/a/lanmu4/2013/0615/2683.html|title = 成都公民记者幸清贤出狱被拷至贵州|date = 2011-03-06|publisher = 民生观察}}</ref>
*2006年12月辞职经商，在与单位劳动纠纷诉讼过程中自学法律，运用自己学到的法律知识，帮助弱势群体维权，并建有qq群专门免费为劳动者提供劳动法律服务，通过网络帮助过成百上千的弱势群体，使他们通过法律手段，为自己讨回了公道。<ref name=":0">{{Cite web|url = http://beijingspring.com/c7/xw/rqmy/20111031185920.htm|title = 幸清贤逐户发竞选宣言，陈西、李任科被带走|date = 2011-10-31|author = 张明|publisher = 维权网}}</ref>
*2009年2月23日成都冤民以铁链锁手在成都中院维权抗议后，鲍俊生、刘继伟、曾理、曾荣康、杨久荣、徐崇丽、黄晓敏、严文汉、幸清贤、陆大春等10余人被抓捕。<ref name=":1" />
*2010年9月28日，幸清贤被乐山中级法院判处有期徒刑2年。<ref name=":0">{{Cite web|url = http://beijingspring.com/c7/xw/rqmy/20111031185920.htm|title = 幸清贤逐户发竞选宣言，陈西、李任科被带走|date = 2011-10-31|author = 张明|publisher = 维权网}}</ref>
*2011年2月28日，幸清贤刑满释放。<ref name=":1" />
*2011年，幸清贤作为独立参选人参加了贵阳市区县人大代表的选举。<ref name=":0">{{Cite web|url = http://beijingspring.com/c7/xw/rqmy/20111031185920.htm|title = 幸清贤逐户发竞选宣言，陈西、李任科被带走|date = 2011-10-31|author = 张明|publisher = 维权网}}</ref><ref>{{Cite web|url = http://wqw2010.blogspot.com/2011/11/blog-post_8590.html|title = 贵阳独立参选人幸清贤投票日被跟踪（图）|author = 张明|publisher = 维权网}}</ref>
*2015年10月6日，幸清贤、[[唐志顺|唐志顺]]、[[包卓轩|包卓轩]]（[[王宇_(律師)|王宇]]、[[包龙军|包龙军]]之子）在[[缅甸|缅甸]]境内被中国政府跨境抓捕。<ref>{{Cite web|url = http://www.boxun.com/news/gb/china/2015/10/201510091411.shtml#.Vr55k5MrLsE|title = 急扩散：中国公民包卓轩、唐志顺、幸清贤在缅甸失联|publisher = 博讯|date = 10/9/2015}}</ref>
*2015年10月8日，成都市公安局北巷子派出所受内蒙古乌兰浩特公安局委托到幸清贤家抄家。<ref>{{Cite web|url = http://www.rfa.org/mandarin/yataibaodao/renquanfazhi/ql2-10102015114247.html|title = 维权人士幸清贤确认被捕 公民联署促当局确保三人安全|author = 乔龙|date = 2015-10-10|publisher = 自由亚洲}}</ref>

*2015年10月12日，幸清贤的妻子[[何娟|何娟]]逃亡到[[美国|美国]][[旧金山|旧金山]]湾区。<ref>{{Cite web|url = http://www.rfa.org/mandarin/yataibaodao/renquanfazhi/ck-10272015113104.html|title = 义救包蒙蒙幸清贤被捕 其妻何娟问中共要人|author = CK|date = 2015-10-27|publisher = 自由亚洲}}</ref>
