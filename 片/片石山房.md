'''片石山房'''位于[[江苏省|江苏省]][[扬州市|扬州市]]城南花园巷、[[何园|何园]]（寄啸山庄）内的东南部，别称'''双槐园'''，是一座规模较小的[[中式古典園林|中式古典园林]]。由于与何园毗邻，后人将片石山房小园与何园打通。

== 辞源 ==

== 历史沿革 ==
片石山房为清朝吴氏的宅第{{r|学林漫录}}{{Notetag|1=《学林漫录》第14卷认为为乾隆年间吴之黼{{r|学林漫录}}，但丁曲等的观点与之相左{{r|扬州大学学报}}。}}。关于片石山房最早最原始的记载，出自笔记《花间笑语》的第五卷{{r|学林漫录}}。

光绪九年（1883年），[[何园|何园]]的宅主埋下了吴氏的片石山房旧址，并将它扩入了何园之中{{r|张理晖}}{{r|心海}}。中华民国时期，何园败落，后裔卖出何园，只保留了片石山房以东的地区，并开辟了直接通往花园巷的门{{r|省志}}。据[[钱泳|钱泳]]《[[履园丛话|履园丛话]]》记载，山房“后为一媒婆所得，以开面馆，兼为卖戏之所，改造大厅房，仿佛京师前门外戏园式样，俗不可耐矣”{{r|钱泳}}。

== 保护与修复 ==
1989年扬州市人民政府决定重修片石山房{{r|省志}}。

== 结构与手法评析 ==
片石山房的中心是“山水”，“山水”的结合又以山为主<ref name=":0">{{Cite journal|title=石涛与扬州片石山房|author=陶济|url=http://www.ixueshu.com/document/6fb46a7333f238c7318947a18e7f9386.html|journal=《风景名胜》|issue=1|year=1994|page=28-29|issn=1003-5516}}</ref>。山水、草木和建筑相互交错，联成一体，体现了中国古典园林的审美与造景，在扬州园林中开“以叠石取胜”之先河<ref name=":0" />。

== 相关分析 ==
片石山房的假山相传为清初画家[[石涛|石涛]]所叠{{r|张理晖}}{{r|省志}}，采用了[[下屋上峰|下屋上峰]]的处理手法{{r|中国网}}。扬州石涛所叠的假山，共万石园和片石山房两处{{r|扬州大学学报}}。前者史料记载很多，但已于乾隆年间废毁{{r|扬州大学学报}}；而现在的片石山房是否为石涛的“人间孤本”，尚待定论，不过同意“人间孤本”之说的文献相对较多，属于主流意见{{r|中国园林}}。

[[陈从周|陈从周]]《扬州片石山房——石涛叠石作品》于1962年发表，认为片石山房就是石涛所作{{r|文物}}。这一说法取得了许多学者的认同{{r|中国园林}}。丁曲认为，从假山的堆叠手法和钱泳《履园丛话》来看，片石山房确实是“人间孤本”{{r|扬州大学学报}}。陶济论文《石涛与扬州片石山房》<ref name=":0" />、孙传余书籍《园亭掠影：扬州名园》等同样赞同“人间孤本”之说{{r|扬州名园}}。然而，怀疑的声音也陆续提出{{r|中国园林}}。2001年，扬州文化学者韦明铧撰文《“片石山房”是石涛手笔么》{{r|中国园林}}{{r|掌故}}；2007年曹汛发表万字长篇《石涛叠山“人间孤品”，一个媕浅而粗疏的园林童话》{{r|建筑师}}并产生了一定的影响{{r|中国园林}}。张家骥在《中国园林艺术大辞典》种却认为山房已经有上百年的历史，“易主改造、修葺、重建不知几许”，因此“何以能妄断为石涛叠山的‘人间孤本’”{{r|中国园林艺术大辞典}}。《[[学林漫录|学林漫录]]》同样不认同“人间孤本”之说{{r|学林漫录}}。徐良和李金宇的论文则驳回了不认同的说法{{r|中国园林}}。
