'''撒冷'''（{{lang-en|Salem}}）是位於[[美國|美國]][[阿肯色州|阿肯色州]][[薩林縣_(阿肯色州)|薩林縣]]的一個[[人口普查指定地區|人口普查指定地區]]。

== 地理 ==
撒冷的座標為{{coord|34|37|46|N|92|33|42|W|type:city_region:US-AL_source:GNS-enwiki|display=inline}}<ref name=GNIS />，而該地最高點為海拔高度143米（即469英尺）。

== 人口 ==
根據[[2010年美國人口普查|2010年美國人口普查]]的數據，撒冷的面積為8.38平方千米，當中陸地面積為8.25平方千米，而水域面積為0.13平方千米。當地共有人口1635人，而人口密度為每平方千米{{formatnum: {{rounddown|{{#expr: 1635 / 8.38}}}} }}人。<ref name="GR8">{{cite web|url=http://factfinder2.census.gov|publisher=[[美国人口调查局|美国人口调查局]]|accessdate=2011-05-14|title=American FactFinder}}</ref>
