'''撒拉语'''（'''{{Lang|slr|Salarça}}'''）归属[[阿尔泰语系|阿尔泰语系]][[突厥语族|突厥语族]][[乌古斯语支|乌古斯语支]]。撒拉語是[[撒拉族|撒拉族]]的本民族语言，他們主要聚居于中华人民共和国青海省[[循化撒拉族自治县|循化撒拉族自治县]]、[[化隆回族自治县|化隆回族自治县]][[甘都镇|甘都镇]]、[[海西蒙古族藏族自治州|海西蒙古族藏族自治州]]、[[海北藏族自治州|海北藏族自治州]]、[[海南藏族自治州|海南藏族自治州]]、[[西宁市|西宁市]]、甘肃省[[临夏回族自治州|临夏回族自治州]][[临夏市|临夏市]]、[[积石山保安族东乡族撒拉族自治县|积石山保安族东乡族撒拉族自治县]]、新疆维吾尔自治区[[伊犁哈萨克自治州|伊犁哈萨克自治州]][[伊宁县|伊宁县]]。

== 土语 ==

撒拉語內部差异并不大，其下可分為街子土語和孟達土語两种土语，可互通。其中以街子土语为多，分布于循化县的大部分地区以及化隆县。孟达土语语音与别处差异颇大，局限于循化县内的[[孟达乡|孟达乡]][[大庄村|大庄村]]、[[清水乡_(循化撒拉族自治县)|清水乡]][[木厂村|木厂村]]和它撒坡。

== 分布 ==

据[[中华人民共和国第六次全国人口普查|中华人民共和国第六次全国人口普查]]撒拉族人口约130,607，其中大多数居住在青海省东部的循化县和化隆县甘都镇，另有少数居住在甘肃和新疆。大多数的撒拉族人以撒拉语为母语并普遍使用[[汉语|汉语]]（[[普通话|普通话]]、[[中原官话|中原官话]]、[[兰银官话|兰银官话]]），青海、甘肃的撒拉族又多会[[藏语|藏语]]（[[安多方言|安多方言]]），新疆的撒拉族多会[[维吾尔语|维吾尔语]]（伊犁方言）。

== 语音 ==

语音方面有辅音音位{{IPA|b}}、{{IPA|p}}、{{IPA|m}}、{{IPA|f}}、{{IPA|v}}、{{IPA|d}}、{{IPA|t}}、{{IPA|n}}、{{IPA|l}}、{{IPA|r}}、{{IPA|s}}、{{IPA|z}}、{{IPA|d͡ʐ}}、{{IPA|t͡ʂ}}、{{IPA|ʂ}}、{{IPA|d͡ʒ}}、{{IPA|t͡ʃ}}、{{IPA|ʃ}}、{{IPA|j}}、{{IPA|ɡ}}、{{IPA|k}}、{{IPA|ŋ}}、{{IPA|x}}、{{IPA|ɣ}}、{{IPA|ɢ}}、{{IPA|q}}、{{IPA|χ}}、{{IPA|ʁ}}、{{IPA|h}}29个、单元音音位{{IPA|i}}、{{IPA|y}}、{{IPA|e}}、{{IPA|ø}}、{{IPA|ə}}、{{IPA|ɑ}}、{{IPA|o}}、{{IPA|u}}8个，另有复合元音{{IPA|iɑ}}、{{IPA|ie}}、{{IPA|io}}、{{IPA|iu}}、{{IPA|iə}}、{{IPA|uɑ}}、{{IPA|ue}}、{{IPA|uo}}、{{IPA|ye}}9个、其中{{IPA|iə}}、{{IPA|uɑ}}、{{IPA|ue}}、{{IPA|uo}}、{{IPA|ye}}5个复合元音仅用于汉语借词。有语音同化、弱化、减音、脱落现象。在固有词词汇当中，音节结构有V、VC、CV、CVC、CVV、CVVC六种，汉藏语借词分别按照各自原语言的音节规律。

== 词汇 ==

词汇方面由于撒拉族人长期与[[汉族|汉族]]、[[藏族|藏族]]、[[回族|回族]]杂居，撒拉语词汇中掺有很多外来语借词。汉语借词约占20%以上，比周边其他的[[突厥语族|突厥语族]]、[[蒙古语族|蒙古语族]]语言的汉语借词比例要高，主要为现代政治、经济、科技、文化等相关的词汇（例词：vunxuɑ，“文化”、dienjin，“电影”）。藏语借词约占10%以上（例词：ʃo，“酸奶”、lɑŋsɑ，“大背斗”）。另有少部分[[波斯语|波斯语]]和[[阿拉伯语|阿拉伯语]]的宗教用词（例词：kirid，（波）“钥匙”、dunjɑ，（阿）“世界”）。还有极少数的借词来自[[蒙古语|蒙古语]]、[[土族语|土族语]]和[[保安语|保安语]]。

固有词与周边突厥语族的亲属语言（如维吾尔语、[[哈萨克语|哈萨克语]]、[[西部裕固语|西部裕固语]]等）拥有共同的词源。还包含[[古突厥语|古突厥语]]遗留的词汇底层（例词：*ajaq→ajaχ，“碗”），以及一个语言来源尚不明确的词汇底层（例词：bɑt，“鸭”）。

== 语法 ==

在语法方面发生了混合语言常见的简化现象，单复数的分别也基本消失。与其他突厥语言同样为主—宾—谓句序，遵循[[元音和谐|元音和谐]]规律，词的附加成分置于词后后缀。

== 文字 ==

撒拉语以前使用以阿拉伯字母为基础的“土尔克撒拉文”（又作“图尔克撒拉文”、“土耳克撒拉文”），而后改为拉丁字母。土尔克文主要用于《[[古兰经|古兰经]]》的释义以及其他宗教书刊的翻译（如手抄的土尔克文《古兰经》《土耳克杂学》别称《杂学本本》）。至今民间仍保存着一些用土尔克文书写的文献。主要使用民间[[中国撒拉尔青年联盟|中国撒拉尔青年联盟]]（Çin Salır Yixdiler Birleşix）推行的“TB31拉丁文”，共31个字母，8个元音字母，23个辅音字母。目前撒拉语没有正式的官方语文管理机构，也没有正式的撒拉文字。[[语言学|语言学]]研究出版物直接使用[[国际音标|国际音标]]书写。
