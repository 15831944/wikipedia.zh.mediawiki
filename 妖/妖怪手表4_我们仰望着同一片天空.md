== 开发 ==
游戏原定于2018年冬季发售，但由于某些原因，Level-5公司更改了游戏的发售时间，并将其定於2019年春季。<ref>{{Cite web|url= https://m.tgbus.com/news/32155/ |title = 传闻：Level-5大量开发人员流失导致其游戏跳票 |publisher = 电玩巴士 |author = 章鱼上校 |date = 2019-01-02}}</ref><ref>{{Cite web |url= https://www.gamerevolution.com/news/476805-rumor-yo-kai-watch-4-developers-leave-level-5-in-mass-exodus |title = Rumor: Yo-Kai Watch 4 Developers Leave Level-5 in Mass Exodus |date = 2018-12-30 |publisher =  GameRevolution |language = en}}</ref>

2019年1月，Level-5公司在[[日本|日本]]“[[东京|东京]]次时代趣味博览会19  Winter”上公布了一段时长只有12分钟的游戏实机运行视频。<ref>{{Cite web |url = http://m.tgbus.com/news/33449 |title = 《妖怪手表4》“次时代趣味博览会”实机演示 |publisher = 电玩巴士 |date = 2019-01-27 }}</ref>

2019年3月，Level-5公司透过日本杂志《コロコロコミック》（{{lang-en|CoroCoro}}）宣布本作将于今年6月6日在日本发售，并附赠限定特典水晶妖怪“弧钥”。<ref>{{cite web |url = https://m.tgbus.com/news/37088/ |title = 《妖怪手表4》发售日确定 6月6号樱花镇再相会 |publisher = 电玩巴士 | date = 2019-03-12 |language = zh}}</ref><ref>{{Cite web |url = https://zhuanlan.zhihu.com/p/59085044 |title = 《妖怪手表4》发售时间泄露！异度神剑或将出新作？|publisher = 知乎 |date = 2019-03-13 |language = zh}}</ref><ref>{{Cite web |url = https://gamingbolt.com/yo-kai-watch-4-launches-in-japan-this-june |title = Yo-kai Watch 4 Launches In Japan This June |publisher = gameingblot |date = 2019-03-12 |language = en}}</ref><ref>{{Cite web |url = https://www.siliconera.com/2019/03/12/yo-kai-watch-4-readies-for-its-next-adventure-with-a-june-6-release-date-in-japan/ |title = Yo-kai Watch 4 Readies For Its Next Adventure With A June 6 Release Date In Japan |date = 2019-03-12 |publisher = Siliconera |language = en}}</ref><ref>{{cite web |url = https://topics.nintendo.co.jp/c/article/ae8dec25-3ff3-11e9-b104-0a6d14145cb1.html?utm_source=twitter&utm_medium=social&utm_content=20190315&utm_campaign=topics |title = シリーズ最新作『妖怪ウォッチ4 ぼくらは同じ空を見上げている』が、6月6日に発売決定！ Nintendo Switchで、全く新しい「妖怪ウォッチ」が楽しめる。 | トピックス |publisher = [[Nintendo|Nintendo]] |date = 2019-03-15 |language = ja}}</ref>
