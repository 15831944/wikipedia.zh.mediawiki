{{日本作品|妖精森林的小不點|ハクメイとミコチ}}是由[[樫木祐人|樫木祐人]]創作的[[日本|日本]][[漫畫|漫畫]]作品，連載於[[KADOKAWA|KADOKAWA]]的漫畫雜誌《{{tsl|ja|ハルタ (漫画誌)|Harta}}》，內容講述兩位女性[[小人_(傳說生物)|小矮人]]哈库梅伊与蜜珂析的日常生活。2017年8月宣布[[電視動畫|電視動畫]]化<ref>{{Cite web|url=https://www.animatetimes.com/news/details.php?id=1502065013|script-title=ja:ハルタ連載『ハクメイとミコチ』がTVアニメ化決定！アニメ制作はLerche、美術は草薙が担当|date=2017-08-07|website=animate Times|accessdate=2017-11-27|language=ja}}</ref>，於2018年1月播出<ref>{{Cite web|url=http://natalie.mu/comic/news/252593|script-title=ja:「ハクメイとミコチ」テレビアニメの放送開始は2018年1月に決定|date=2017-10-13|website=漫畫Natalie|accessdate=2017-11-27|language=ja}}</ref>。

本作最初是名為「{{nihongo|昨日的茜色|きのうの茜}}」的短篇作品，於2011年4月在[[enterbrain|enterbrain]]的漫畫雜誌《Fellows!》（後更名Harta）上刊載，後於同年10月在其兄弟雜誌《Fellows!(Q)》刊載新短篇「{{nihongo|腳下的行走方式|足下の歩き方}}」。2012年4月，本作在《Fellows!》以「{{nihongo|小矮人日和|こびと日和}}」為題開始連載<ref>{{Cite web|url=http://natalie.mu/comic/news/67822|script-title=ja:フェローズで新鋭3人の連載一挙始動、合作マンガの小冊子も|date=2012-04-14|website=漫畫Natalie|accessdate=2017-11-27|language=ja}}</ref>，並於同年12月更改為現名。

《[[這本漫畫真厲害！|這本漫畫真厲害！]]》的加山龍司評論本作「雖然角色以可愛的小矮人形象呈現，但對日常生活的詳細描寫令她們像是擁有呼吸一般」，認為本作是「以不可思議世界為舞台的日常生活故事」<ref>{{Cite web|url=http://konomanga.jp/guide/23446-2|script-title=ja:『ハクメイとミコチ』第3巻 樫木祐人 【日刊マンガガイド】|date=2015-01-31|website=這本漫畫真厲害！|accessdate=2017-11-27|language=ja}}</ref>。

== 劇情簡介 ==
:哈庫梅伊與蜜珂析。
:住在綠意盎然森林裡的，兩位小巧的女孩子。
:在樹洞裡建造住家、把樹葉當做雨傘、坐在昆蟲和小鳥的背上……
:如果身高只有9厘米的話，這種事情是可以做得到的。
:不去稍微地偷看一下嗎？
:她們那安詳而又愉快的，十分可愛的日常生活。

:在日垣與旋毛丸的回憶中，宇海釀造的朱利酒非常好喝。然在喝過蜜珂析的改良版後，才發現過去的利口酒其實相當差勁。
:名字拼音(Ukai)與日文「愉快(Yukai)」諧音。

; {{nihongo|日垣|飞垣|ヒガキ|聲：{{link-ja|竹内良太}}{{r|character}}}}
:首次登場於原作第15之1話（動畫第8話）。
:男性，72歲，黑捲髮，留著絡腮鬍，故鄉在卡南卡。
:「蜂蜜館」的二代目館主，也是元老級住民，宇海的好友。宇海過世後繼任館主，因走不出對宇海的思念而迷失自我，得不到以旋毛丸為首的「老頑固」認同。
:為了再現風蜜館的繁榮而拜託蜜珂析重現宇海的朱利酒。在「漫長的一天」中逐漸重拾初衷，最終與旋毛丸和解，成為實至名歸的館主。
:原本的第一人稱是「僕」，重獲自信後第一人稱變為「私」。

; {{nihongo|旋毛丸|旋毛丸|ツムジマル|聲：[[諏訪部順一|諏訪部順一]]{{r|character}}}}
:首次登場於原作第15之1話（動畫第8話）。
:[[彩虹飛蜥|彩虹飛蜥]]，男性，35歲，故鄉在西諾奇。
:蜂蜜館的元老級住民，「老頑固」實質上的首領。長相兇惡，舉止與談吐都像[[流氓|流氓]]，但性格並不壞。
:與日垣一樣一直惦記著宇海在世的繁華時光，對迷失自我的日垣感到失望，因此架空日垣的館主地位，對哈庫梅伊自稱「現在我們之中沒有老大」。為了給新人下馬威而破壞康居的演唱會，並派人綁架康居。
:在騷動中與日垣一同出手保護了差點被油燈擊中的蜜珂析；在喝過蜜珂析改良的朱利酒後，擺脫對宇海的思念，與日垣重歸於好。
:原作第39話，為協助日垣搬運書籍而來到圖書館，在圖書館裡打瞌睡而造成圖書館員基昂的困擾；並透露他喜歡毛茸茸[[哺乳類|哺乳類]]的[[性偏好|性偏好]]，對同種類[[爬蟲類|爬蟲類]]女性沒有興趣。

; {{nihongo|米奇|ミキ|聲：[[手塚ヒロミチ|手塚ヒロミチ]]{{r|character}}}}
:首次登場於原作第15之1話（動畫第8話）。
:男性，23歲，上吊的三白眼，後梳緊紮黑髮，故鄉在西諾奇。
:「老頑固」的一員，初登場時作為綁架康居的一員。
:蜂蜜館「蔬菜店米奇」的店主。擁有高超的選貨眼光，對蔬果抱有真摯的熱情。店內蔬果的鮮度與口感都有相當高的水平，偶爾還會出現一般店裡找不到的罕見品種。但因為店面位置偏僻、宣傳不足、價位設定過高等因素，只有少數幾間餐廳會進貨，導致許多惡性庫存，因此在同業中評價極差。
:原作第22話中壟斷所有木梅。在蜜珂析的威脅利誘下妥協，自願作為蜜珂析的果醬原物料供應商。

; {{nihongo|修卡|シュンカ|{{r|character}}}}
:首次登場於原作第15之2話（動畫第8話）。
:女性，年齡與出生地皆不詳，黑色直長髮。
:位於蜂蜜館最下層「金層」中央的酒吧「狡黠哈露茲」的老闆兼調酒師。雖然年輕，卻是蜂蜜館的元老級住民。
:過去一段時間曾遠赴他鄉學習酒的知識，歸來後在宇海的贊助下開設酒館，「狡黠哈露茲」之名也是宇海取的。
:由於老頑固和新人都無法在她面前擺架子，使「狡黠哈露茲」成為蜂蜜館中少有的安全地帶。
:左手無名指戴有戒指，但就算向她問起也會被巧妙的轉移話題。

== 故事場景 ==
;瑪提那塔（マキナタ）
:山間都市瑪提那塔是個被遺跡群和大山包圍，充滿著大量勞動者的城市。這城鎮的時間流動得特別悠閒平靜。
:瑪提那塔的大門有體格的限制，所以使得身材特別龐大的旅人無法在此地停留。

;阿拉比（アラビ）
:傳聞港鎮阿拉比的積木市場是由三家店鋪的名士發展起來的，魚師的氣球丸、布店的維莉、咖啡店的丘瓦卡。
:現今變為廢墟的原中央廣場舊址依然保留著積木市場一層一層堆疊建築外型的原型。
:阿拉比獨特的積木構造，據說來自於三人酒席間的玩笑話——「只要不斷往上疊加，店鋪想建多少就能建多少」。

;石貫會（石貫會，いわぬきかい）
:位於瑪提那塔的木匠聯合工會。曾經荒廢一時，因為「眼神銳利的男人」與「慢性子的男人」（即年輕時代的西風與火天）的到來而成功復興。
:公會宗旨是「安全、迅速、認真、同心協力！」。

;蜂蜜館（蜂蜜館）
:由釀酒致富的大富豪享樂家宇海所成立，聚集了藝術家和混混的無法地帶，從下到上有金、石、系、竹、匏、土、革、木共八層，在金層的底下還有秘密通道和酒窖。蜂蜜館的信條是「萬事皆可」。

;夢品商店（むじな商店）
:是一家旗艦店位於馬鈴薯山谷腳下的商店，平時出售日用品、加工食品、麵包及零食等，不過在瑪提那塔舉行祭典時，將會在露天店面上販售限定商品。
