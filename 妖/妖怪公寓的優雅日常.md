《'''妖怪公寓的優雅日常'''》（{{lang|ja|妖怪アパートの幽雅な日常}}）是[[香月日輪|香月日輪]]創作的一部小說，2003年10月至2013年8月由[[講談社|講談社]]發行，2004年，獲得第51屆產經兒童出版文化獎。本作世界觀與香月日輪的另一部作品《[[地獄堂靈界通信|地獄堂靈界通信]]》相通。 

2011年小說改編為漫畫版，由[[深山和香|深山和香]]繪畫，於《[[月刊少年天狼星|月刊少年天狼星]]》2011年7月號開始連載。電視動畫於2017年7月2日起在日本地區[[東京都會電視台|東京都會電視台]]等電視台播出。

== 故事簡介 ==
三年前稻葉夕士因為父母雙亡，搬到伯父家，接受伯父一家的照顧，原本希望在考上高中後就搬進宿舍開始獨立生活……沒想到宿舍竟慘遭火災肆虐而付之一炬！費盡千辛萬苦的夕士在因緣際會之下搬進了房租便宜、附伙食還附溫泉的超好康公寓——壽莊。但是，那裏居然是有妖怪出沒的「妖怪公寓」，夕士「普通」的高中生活就此展開。

== 登場角色 ==
=== 壽莊 ===
; {{nihongo|稻葉夕士|稲葉 夕士（いなば ゆうし）|聲：[[阿部敦|阿部敦]]、[[泊明日菜|泊明日菜]]（幼时）}}
: 條東商業高中一年級生。夢想本來是成為擁有即時戰力的普通上班族。
: 雙親因車禍過世，寄住在伯父家，為了擺脫寄人籬下的生活，便立志考進有宿舍居住的商業高中。
: 但沒想到學校宿舍突發火災，需要半年的重建時間，便費盡千幸萬苦的住進了租金便宜名為“壽莊”的妖怪公寓，在那裡遇到了充滿人情味的“怪誕”房客……
: 本身是一個擁有靈能力的人，但自己表示一直想做個普通人，對靈能力並無自覺，曾無意中救了社團的田代同學。
: 後來從舊書商手裡得到一本魔法書《小希洛佐異魂》，並成為書的主人，能召喚出奇妙的惡魔。
: 也因此正視自己的靈能力，在秋音的教導下開始修行。
: 中途曾經因為宿舍重修好，而搬離妖怪公寓，一直覺得很寂寞，最終下定決心，回到了妖怪公寓。

: 可以驅使式鬼。一直幫助夕士，表示自己很看好他的能力。
: 在夕士中途離開的時候特地回來送行，夕士修行的時候，因為擔心特意留在妖怪公寓。
: 是個溫柔也很嚴厲的人。

: 《小希洛佐異魂》原本是他帶回來的書，但書選擇夕士作為主人，舊書商便將書送給了夕士。

==== 幽靈 ====
; {{nihongo|阿栗（小元）|クリ|聲：[[釘宮理惠|釘宮理惠]]}}
: 小男孩幽靈。與小白總是形影不離。對他人都很親近，特別是對夕士或長谷。生前被母親虐待，在孤苦無依時邂逅小白與之為伴，但後來遭母親發現和殘忍殺害，死後的阿栗和小白因為被母親的怨靈追殺，遂接受犬神的庇護搬進壽莊生活，但礙於母親怨靈的執念無法成佛。後來在眾人的幫助下逐漸開啟心房。最終話因為母親的執念消逝，正式超渡成佛。

; {{nihongo|小白|シロ|聲：[[田村睦心|田村睦心]]}}
: 白狗幽靈。與阿栗總是形影不離。生前原本是隻流浪狗，偶遇被母親虐待的阿栗成為朋友，但後來阿栗被母親殘殺，小白憤而咬破阿栗的母親[[喉嚨|喉嚨]]致其死亡，隨後便被一群人士亂棍打死。死後的小白放心不下阿栗，志願陪伴其身邊，兩者獲得犬神的庇護搬進壽莊生活。

; {{nihongo|麻里子|まり子|聲：[[三石琴乃|三石琴乃]]}}
: 身材姣好的女幽靈，性格奔放不拘小節，總是穿的很少在公寓晃來晃去。生前原本是暴發戶人家的千金，過著放蕩的生活且數度[[墮胎|墮胎]]，後來邂逅心儀的男子與之結婚，卻因為先前多度墮胎導致體質虛弱，最終在21歲時產下嬰兒後辭世。因為生前未盡的心願，對於為人母親有著強烈的憧憬。
