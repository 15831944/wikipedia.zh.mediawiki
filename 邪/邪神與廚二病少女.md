《'''邪神與廚二病少女'''》（{{lang-ja|邪神ちゃんドロップキック}}）是由{{tsl|ja|ユキヲ|Yukiwo}}創作的[[日本|日本]][[漫畫|漫畫]]作品，自2012年4月25日開始在{{tsl|ja|フレックスコミックス|Flex Comix}}的[[網路漫畫|網路漫畫]]網站《{{tsl|ja|COMIC メテオ|COMIC Meteor}}》連載<ref>{{Cite web|url=https://prtimes.jp/main/html/rd/p/000000172.000001474.html|script-title=ja:Gモード発！完全無料・登録不要のWEBコミック誌「COMICメテオ」創刊!!|date=2012-04-25|website=PR TIMES|accessdate=2017-12-17|language=ja}}</ref>。[[電視動畫|電視動畫]]於2018年夏季播出<ref>{{Cite web|url=https://animeanime.jp/article/2017/12/07/36192.html|script-title=ja:邪神×厨二病少女の共同生活描く「邪神ちゃんドロップキック」18年夏TVアニメ化|date=2017-12-07|website=Anime!Anime!|accessdate=2017-12-17|language=ja}}</ref>。

==故事簡介==
故事講述一位中二病女大學生花園百合鈴偶然從魔界召喚了惡魔小邪神，卻不知道讓小邪神回去的方法。小邪神說讓召喚者死亡就行了，於是小邪神為了回到魔界開始各種暗殺行動，然而花園百合鈴擁有徹底碾壓小邪神的強大武力，因此兩人被迫展開日常喜劇的同居生活。

== 登場人物 ==
; {{nihongo|小邪神|邪神ちゃん（じゃしんちゃん）|聲：[[鈴木愛奈|鈴木愛奈]]}}
: 本作主角。被百合鈴召喚至人間界的惡魔，魔界贵族出身，形态为半人半蛇。品行恶劣，總是企圖殺死百合鈴以返回魔界，但卻每次都遭反擊而落到淒慘的下場。由于有再生能力，因此即使遭受残酷的惩罚也不会死去。
: 虽然是上级恶魔，但被米诺斯评价为“魔界最弱”，唯一的能力是控制骰子点数。必杀技为不完整的“小邪神飞踢”，第二必杀技是“皇家哥本哈根”。在本人并不知情的情况下展现出了空想实体化的能力。
: 因为打弹珠机等缘故导致缺钱，经常讹诈美杜莎的钱财进行各种娱乐以及购买各种道具。曾借几千年一次的蜕皮诈死，戏弄美杜莎，结果激怒了对方。
: 热爱食物，曾宣称“即使糟蹋人命也不能糟蹋食物”。擅长料理，意料之外的顧家，能做出相當美味的咖哩與馬鈴薯燉肉。為了報復在推特上噴自己的人，做了派到日本各地到處砸人，最後研發出連百合鈴都讚不絕口的美味派。
: 虽然是蛇妖，但意外地害怕青蛙。
: 有神经性胃炎，精神压力过大会呕吐，严重时会胃穿孔。一段时间内不食用布丁或者做坏事就会有戒断反应。
: 本名很长，据称完整念下来30分钟也念不完，以致好友都无法记住。

; {{nihongo|花園百合鈴|花園 ゆりね（はなぞの ゆりね）|聲：[[大森日雅|大森日雅]]}}
: 有點陰暗、喜欢黑魔法与恐怖电影的女大學生，住在神保町的破舊公寓，家中经营[[神社|神社]]。有[[中二病|中二病]]，總是[[哥德蘿莉|哥德蘿莉]]風格的打扮並一眼帶著眼罩。
: 经常在网上购买各种武器和刑具。每次識破小邪神的陰謀或发现邪神犯下错误後，都会以血腥残忍的方式惩罚邪神（如腰斩、剁成肉酱、做成蛇肉刺身等），但对其他人却很和蔼。
: 非常討厭有人用手指指她，常因為這點把小邪神的手指折斷。
: 儿时的梦想是建立“百合铃王国”，成为独裁者。
: 因听说小邪神将美杜莎找来，在秋叶原购买了防石化手环，故不会被美杜莎石化。但手环被珀耳塞福涅2世指出毫无作用。

; {{nihongo|美杜莎|メデューサ|聲：[[久保田未夢|久保田未夢]]}}
: 小邪神的惡魔朋友，[[美杜莎|蛇髮女妖]]（實際上只是帶著[[眼鏡蛇|眼鏡蛇]]造型的髮箍），在人間界活動時戴著紙袋避免人類被石化。聲音與外表都十分甜美，個性溫和，常常被邪神欺負、指使，被邪神當成寶貴的[[ATM|ATM]]，但是自己沒有太大的知覺，被百合铃吐槽容易招来渣男。愤怒时会讲敬语。

; {{nihongo|米諾斯|ミノス|聲：[[小見川千明|小見川千明]]}}
: 小邪神的惡魔朋友，長著牛角的少女，[[巨乳|巨乳]]，搬到百合鈴的隔壁居住。在人间靠送牛奶和在工地搬运材料维持生计，非常受人类欢迎。年幼時與小邪神相處的狀況和美杜莎相反，時常把小邪神弄很慘，但都是無意造成的。

; {{nihongo|佩珂拉|ぺこら|聲：[[小坂井祐莉繪|小坂井祐莉繪]]}}
: 失去天使之環的天使。本來是以驅逐惡魔（小邪神與她朋友）和魔女（百合鈴）為目的而來。失去天使之環後，幾乎與一般人類無異，窮困潦倒，靠打廉价黑工维持生计，常常餓肚子。

; {{nihongo|波波隆|ぽぽろん|聲：{{tsl|ja|佐々木李子|佐佐木李子}}}}
: 佩可拉過去的部下，但为了一已私欲、权利和地位欲杀害佩可拉。在打翻小邪神为佩可拉烹煮的[[马铃薯炖肉|马铃薯炖肉]]（也因此称呼小邪神为“土豆”、“土豆炖肉”或者“炖肉”，动画改为马铃薯咖喱）後被小邪神殴打，捆綁全身扔进大海；之後尋仇時又因為打翻小邪神買給百合鈴的[[杏仁豆腐|杏仁豆腐]]，天使之環遭小邪神吃下。最终只得留在人间，化名“天使诺艾尔”（{{lang|ja|天使のえる}}）在一家名为“创世纪”的拉面馆打工，同时以偶像的身份活动。

; {{nihongo|珀耳塞福涅2世|ペルセポネ2世|声：[[饭田里穗|饭田里穗]]}}
: 哈迪斯和珀耳塞福涅的独生女，其母为邪神酱、美杜莎和米诺斯的老师。把邪神酱、百合铃、美杜莎、米诺斯当成姐姐看待。
: 体质虚弱，经常佩戴防毒面具。动画中为了寻找小邪神来到人间，但基本每次都意外擦肩而过。后来与米诺斯同居。虽然体质很虚弱，但她的飞踢比小邪神更高一等。初次登场时曾打算将完整的“小邪神飞踢”传授给小邪神，但因为小邪神还需要80年才能练好而作罢。

; {{nihongo|遊佐|遊佐|声：[[荒浪和沙|荒浪和沙]]}}
: 冰族[[雪女|雪女]]，已成年屬於上級惡魔（冰族成年約人類15歲），实力强大，是连小邪神都忌惮的恶魔，经常在小邪神欺负妹妹浩二的时候教训小邪神。
: 有一位名叫查理、经营巧克力工厂的叔父。

; {{nihongo|小冰/浩二|氷ちゃん/浩二|声：{{link-ja|寺田御子}}}}
: 冰族雪女，遊佐的妹妹，因年纪小还未完全发育成人的形态，曾被小邪神招喚至人間降溫並製造剉冰以度過夏天。虽然是下级恶魔，但却能轻易打败小邪神。
: 冰族未成年前通稱叫「小冰」，成年後才會有自己的名字。百合铃預先給她取名“浩二”，讓姐妹名稱捏他自聲優[[遊佐浩二|遊佐浩二]]。

; {{nihongo|貝特|べ－ト|声：[[市道真央|M·A·O]]}}
:百合鈴和小邪神飼養的寵物。

; {{nihongo|橘芽依|橘 芽依|声：[[原奈津子|原奈津子]]}}
:秋葉原万岁桥警察署的女警，經常會做出不像是警察的異常舉動。
:心理扭曲地喜欢小邪神，一开始以偷走了她的心为由将小邪神抓捕，把小邪神稱为“大蛇丸”。也曾因觉得过于可爱而绑架了浩二。

; {{nihongo|恶魔A|悪魔A|声：[[遊佐浩二|遊佐浩二]]}}
:长相与[[巴风特|巴风特]]相似的羊头恶魔，原作中仅在每个故事间出现，动画中经常登场进行解说。

; {{nihongo|棚橋|棚橋|声：[[稻葉和彦|稻葉和彦]]}}
:佩可拉打工时经常遇到的人。称呼佩可拉为“打工的”。

; {{nihongo|皮诺|ぴの|声：[[山田麻莉奈|山田麻莉奈]]}}
:佩可拉過去的部下，受命来到人间追杀佩可拉和波波隆，被假装逃跑的波波隆摔晕，天使之环也被捏碎。之后萌生推翻[[基督教的神|主]]支配天界、再毁灭魔界和人界的想法，为此留在百合铃等人居住的公寓担任管理员。之后被百合铃和小邪神主动拜访，身份被小邪神识破，甚至连在人间生活的原因都被准确猜中。之后小邪神酱视为敌人打算消灭，不过被百合铃制止。被百合铃无意中显露的实力所惊吓，一度怀疑她不是人类。
