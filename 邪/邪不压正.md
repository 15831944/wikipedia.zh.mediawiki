《'''邪不压正'''》是[[2018年中國上映電影列表|2018年]]中国大陆[[武俠|武俠]][[动作片|动作]][[喜剧电影|喜剧片]]，由[[姜文|姜文]]导演，[[彭于晏|彭于晏]]、[[廖凡|廖凡]]、[[周韵|周韵]]、[[许晴|许晴]]、[[泽田谦也|泽田谦也]]、[[安地|安地]]、姜文主演。影片改编自作家[[张北海|张北海]]武侠小说《[[侠隐|侠隐]]》，讲述1936年发生在[[北平城|北平城]]的间谍斗争。影片是姜文“北洋三部曲”的终章，于2018年7月13日在中国大陸（內地）上映，9月6日在香港上映。

== 剧情 ==
1936年的北平，青年侠士李天然为寻找十五年前师门血案的元凶，深入古都的胡同巷陌。随着他调查的深入，京城各路人马的斗智斗狠浮出水面，日本特务、亲日分子、豪门旧户、黑帮老大、交际花、外国记者等轮番上阵。而老百姓的日子依旧悠悠然地过着。庙会、堂会依循旧例；[[东城区|东城]]、[[西城区|西城]]一如往日。人情冷暖、旧京风华扑面而来。然而[[卢沟桥事变|卢沟桥]]一声枪响，北平淹没在战争烟尘中。

== 制作 ==
影片改编自旅美华人作家[[张北海|张北海]]武侠小说《[[侠隐|侠隐]]》<ref>{{cite news|title=侠隐|url=https://book.douban.com/subject/30224187/|work=豆瓣|accessdate=2018-07-12}}</ref>。

影片于2015年4日在杭州开机<ref>{{cite web |title=姜文《侠隐》或上档2018年春节？ 编剧何冀平受访透露后期已做70% |url=http://news.mtime.com/2017/10/24/1574801.html |website=Mtime时光网 |accessdate=2018-07-13 |date=2017-10-24}}</ref>，2017年6月14日于[[古北口|古北口]]杀青<ref name="首映礼">{{cite web |title=姜文归来！《邪不压正》长城脚下首映 宁浩：特别的姜文 崔永元：彭于晏演的是我 |url=http://news.mtime.com/2018/07/11/1582274.html |website=Mtime时光网 |accessdate=2018-07-13 |date=2018-07-11}}</ref>。
