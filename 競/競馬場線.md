'''競馬場線'''（{{jpn|j=競馬場線|hg=けいばじょうせん|rm=Keibajō sen|f=y}}）是一條連結[[東府中站|東府中站]]和[[府中競馬正門前站|府中競馬正門前站]]，屬於[[京王電鐵|京王電鐵]]的鐵路線。全線在[[東京都|東京都]][[府中市_(東京都)|府中市]]內行走。車站編號使用的路線記號為'''KO'''。

通常運行的複線鐵路中，路線距離只有0.9公里是非常的短。一如路線名稱，以往[[東京競馬場|東京競馬場]]為目的而敷設的路線，除東府中站附近外路線都位於府中市{{link-ja|八幡町 (東京都府中市)|八幡町 (東京都府中市)|八幡町}}內，中途橫切{{link-ja|武藏國府八幡宮|武蔵国府八幡宮}}的參道。

== 路線資料 ==
* 路線距離 : 0.9公里
* [[軌距|軌距]] : 1372毫米
* 複線路段 : 全線
* 電氣化路段 : 全線（[[直流電|直流電]]1500[[伏特|V]]）
* 保安裝置 : 京王[[列車自動控制系統|ATC]]、速度制御式
* 最高速度 : 70公里/小時

== 歷史 ==
* 1955年（[[昭和|昭和]]30年）4月29日 - 東府中 - 府中競馬正門前間 (0.9km) 開業。
* 1963年（昭和38年）8月4日 - [[架線|架線]][[電壓|電壓]]1500V昇壓。
* 1999年（[[平成|平成]]11年）7月28日 - 平日的線内折返列車單人運轉（ワンマン運転）開始。
* 2011年（平成23年）10月2日 - ATC使用開始。

== 運行形態 ==
非賽馬日採用線內區間運行，只來往東府中及府中競馬正門前兩站，每20分鐘1班，早上繁忙時間增至11-13分鐘1班。賽馬日則設有來往[[新宿站|新宿站]]的急行列車。列車採用[[單人駕駛|單人駕駛]]模式，但由於兩站均為有人站，所以並沒有採用車內收取車費的方式。

=== 车辆 ===
* 7000系 2节
* 8000系
* 9000系 均为8节

== 其他 ==
終点府中競馬正門前站因为平日的利用客比其他車站相对较少、又因为賽馬時为防止混雜有较宽的[[站台|站台]]，一些[[電視廣告|电视广告]]和[[MV|PV]]等会选择在这里拍摄。

同样，[[电影|电影]]和[[电视剧|电视剧]]也会在这条路线的電車内进行场景撮影。
