'''佛山地铁'''（{{lang-en|'''Foshan Metro'''}}，简称{{lang|en|'''FMetro'''}}）是[[中华人民共和国|中华人民共和国]][[广东省|广东省]][[佛山市|佛山市]]的[[城市轨道交通系统|城市轨道交通系统]]，由[[佛山市铁路投资建设集团有限公司|佛山市铁路投资建设集团有限公司]]建造。[[佛山|佛山]]是继[[广州|广州]]及[[深圳|深圳]]后第三个拥有[[地铁|地铁]]系统的广东省[[城市|城市]]，同时也是全国第一个拥有地铁的[[地级市|地级市]]。广佛线目前已委托交由[[广州地铁|广州地铁]]进行营运。

== 歷史 ==
[[File:Kuiqilustation.jpg|thumb]]]]

2002年10月11日，[[广佛地铁|佛山地铁一号线（广佛地铁）]]试验段建设启动，

2010年11月3日，[[广佛地铁|佛山地铁一号线（广佛地铁）]][[魁奇路站|魁奇路站]]至广州[[西塱站|西塱站]]通車。

[[西塱站|西塱站]]至[[沥滘站|沥滘站]]的後通段，广佛地铁公司总经理钟铨明确表示因沥滘站拆迁问题未决，因此先把[[西塱站|西塱站]]至[[燕岗站|燕岗站]]一段于2015年12月28日正式通车，先行试运营。

2016年12月28日，[[广佛地铁|佛山地铁一号线（广佛地铁）]]二期[[新城東站|新城東站]]至[[魁奇路站|魁奇路站]]通車試運營。

2018年12月28日，[[广佛地铁|佛山地铁一号线（广佛地铁）]]首期剩余段[[沥滘站|沥滘站]]至[[燕崗站_(廣州)|燕崗站]]通車試運營，至此，广佛地铁首期及二期全线通车。

== 标志 ==
佛山市铁路投资建设集团公布佛山地铁标志，该标志造型像一辆飞速的红色列车。广佛地铁落成后，将广泛运用于佛山境内的地铁站、地铁出入口、部分地铁车辆及佛山轨道交通未来的路线。

集團聲稱，该标识创意源于列车和英文字母“F”两项视觉元素，这辆红色的飞速列车，既直观展现了佛山铁路投资建设集团的产业特性，也喻示着轨道交通建设承载佛山人的梦想。而标识图形是“Foshan（佛山）”、“Fortune（财富）”的首个字母“F”的艺术变形，且与“Victory（胜利）”的首个字母“V”形暗合，既表明佛山地铁的地域特征，也象征佛山铁投集团不断实现国有资产的保值、增值及相关利益各方的合作共赢。

此外，该标识图形与英文简称分别采用了红、蓝两种色彩，“佛铁红”象征创造的激情和前景的辉煌，“佛铁蓝”象征投资的理性从容与建设营运的安全稳健。

== 广佛地区轨道交通銜接规划 ==
