'''佛羅倫斯·費佛瑞'''（'''Florence Faivre'''，全名'''Florence Vanida Faivre'''，{{bd|1983年|6月8日||}}）是一名法國裔泰國女演員和模特。<ref>{{Cite web |url = http://www.villagevoice.com/film/the-elephant-king-a-pale-excuse-for-a-cinematographers-holiday-6390483 |title = The Elephant King a Pale Excuse for a Cinematographer's Holiday |author = Nick Pinkerton |date = 2008-10-15 |publisher = Village Voice |language = en}}</ref>

== 職業生涯 ==
費佛瑞曾經出現在《[[暹羅復興錄|暹羅復興錄]]》（2004年）、《''Chok-Dee''》（2005年）、《{{link-en|大象與我|The Elephant King}}》（2006年）和《{{link-en|棺材 (電影)|The Coffin|棺材}}》（2008年）
等电影中。

電視劇方面，費佛瑞在美國劇集《{{link-en|阿尔法屋|Alpha House}}》和《[[門屠|門屠]]》（兩部都是2013年播放）中客串。她被揀選在電視劇《[[太空無垠|太空無垠]]》（2015年）中飾演Julie Mao。<ref>{{Cite web |url = http://deadline.com/2014/10/moniqua-plante-cast-nashville-florence-faivre-the-expanse-862791/ |title = Moniqua Plante Joins 'Nashville'; Florence Faivre In 'The Expanse' |date = 2014-10-27 |publisher = Deadline.com |language = en}}</ref> 2017年，她在电视剧《[[神盾局特工_(第五季)|神盾局特工]]》第五季中饰演Sinara。
