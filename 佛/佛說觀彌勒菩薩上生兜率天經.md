《'''觀彌勒菩薩上生兜率天經'''》，略稱《彌勒上生經》，由劉宋居士[[沮渠京聲|沮渠京聲]]漢譯，收於[[大正藏|大正藏]]經集部。《[[藏文大藏經|藏文大藏經]]》中也有收入藏譯本，編號No.199<ref>{{cite web|title=漢籍全文．佛典經錄資料庫 - 佛教藏經目錄數位資料庫|url=http://jinglu.cbeta.org/cgi-bin/tibet_detail.pl?lang=&id=0199}}</ref>。

本經與《[[彌勒下生經|彌勒下生經]]》同為[[彌勒|彌勒]]信仰的重要經典，敘述彌勒菩薩命終往生[[兜率天|兜率天]]、在[[兜率淨土|兜率淨土]]說法的情景、[[兜率天宮|兜率天宮]]的殊勝、以及講述[[十善|十善]]念佛等往生兜率天的修行方法，並以此[[念佛|念佛]]功德可超越九十六億劫生死之罪，其敘述方式及內容與《[[觀無量壽經|觀無量壽經]]》相似<ref name="my2">{{cite web|title=彌勒上生經、彌勒下生經解題|url=http://www.gaya.org.tw/library/readers/guide-mlj.htm}}</ref>。

== 題解 ==
彌勒（Maitreya），常見的婆羅門姓氏，字根源自梵文Maitrī，慈愛之意。

上生，佛教謂行[[十善|十善]]者，命終後轉生[[天界|天界]]。<ref>《舍利弗阿毘曇論》：「-{云}-何趣天業？若業善增，能令生天上，是名趣天業。」</ref><ref>《法苑珠林》卷三一引《雜寶藏經》：「我聞石室比丘尼，若能信心出家，一日必得生天。」</ref>

== 傳譯 ==
本經譯經時間大致在433年前後，地點在[[高昌|高昌]]；出經時間在439年[[北涼|北涼]]滅亡之後，地點在[[江東|江東]]。譯經者[[沮渠京聲|沮渠京聲]]為[[北涼|北涼]][[沮渠氏|沮渠氏]][[士族_(中国)|士族]]，世代信奉佛教<ref>{{cite web|author1=王素|title=高昌至西州時期的彌勒信仰|url=http://big.hi138.com/wenhua/lishixue/200604/53704.asp}}</ref>。

== 內容 ==
此經描述[[釋迦佛|釋迦佛]]時，有一菩薩，即彌勒菩薩，又名「阿逸多」（ Ajita ），釋迦佛預言他「次當作佛」。阿逸多「具凡夫身，未斷諸漏」；「雖復出家，不修禪定，不斷煩惱」，在十二年後，會上至「[[兜率天|兜率天]]」。 人間的人，如果「不厭生死，樂生天者，愛敬無上菩薩心者」，只要「持五戒、八齋、具足戒，身心清淨，不求斷結，修十善法」，死後也可以往生兜率天，親近彌勒菩薩。然後，五十六億年後，彌勒將從兜率天下生到人間「閻浮提」，「如[[彌勒下生經|彌勒下生經]]說」<ref name="my2"/>。

== 地位 ==
中國[[彌勒信仰|彌勒信仰]]於晉時[[釋道安|釋道安]]法師（314～385）首倡以來，逐漸盛行。初唐時在[[天台宗|天台宗]]和[[法相宗|法相宗]]高僧的推動和[[武則天|武則天]]的提倡下，民間「彌勒佛出世」的觀念廣為流傳，不少僧尼信士或期生兜率，或願與「龍華三會」，或誦念彌勒佛名。十九、二十世紀的[[中國秘密宗教|中國秘密宗教]]猶深受彌勒信仰的影響<ref name="my2"/>。

提倡[[人間佛教|人間佛教]]的[[印順法師|印順法師]]認為，[[兜率淨土|彌勒淨土]]思想，是著重於實現人間淨土。往生兜率淨土的本意是為下生人間，為人間淨土的實現而努力，而不單純是為往生兜率淨土<ref>{{cite web|author1=張阿蓮|title=印順法師的彌勒淨土思想|url=http://www.pss.org.tw/mbr/world_detail.asp?id=2103}}</ref>。
