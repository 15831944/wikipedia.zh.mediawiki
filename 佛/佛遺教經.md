{{佛教}}
《'''佛垂般涅槃略說教誡經'''》，又稱為《'''佛臨涅槃略說教誡經'''》、《'''佛遺教經'''》，簡稱《'''遺教經'''》，共一卷，由[[後秦|後秦]][[龜茲|龜茲]]國[[鳩摩羅什|鳩摩羅什]]三藏翻譯。是[[佛陀|佛陀]]在臨[[涅槃|涅槃]]前對弟子的最後教誡，闡述作為佛教根本的[[三無漏學|戒定慧]]。

== 內容 ==
《佛遺教經》是[[釋迦牟尼佛|釋迦牟尼佛]]於公元前543年二月十五日中夜臨入[[涅槃|涅槃]]前，為弟子們所做的最後一場說法。此經有兩千三百餘字，全經文可歸納為二十一個小節，即：
一、經序，二、持戒，三、制心，四、節食，五、戒睡眠，六、戒瞋恚，七、戒憍慢，八、戒謟曲，九、少欲，十、知足，十一、遠離，十二、精進，十三、不忘念，十四、禪定，十五、智慧，十六、不戲論，十七、自勉，十八、決疑，十九、眾生得度，二十、法身常住，二十一、結論<ref>{{cite web |author1=曉雲法師 |title=遺教經 |url=http://ap6.pccu.edu.tw/Encyclopedia/data.asp?id=3431&forepage=2 |publisher=中華百科全書 |accessdate=2018-08-05}}</ref>。


== 歷代註疏==
歷代對本經的註疏有：
*《佛遺教經論疏節要》一卷，[[天親菩薩|天親菩薩]]論，[[真諦|真諦]]譯，宋[[智圓|智圓]]疏，宋[[淨源|淨源]]節要，明[[袾宏|袾宏]]補註。
*《遺教經論住法記》一卷，宋[[元照|元照]]述。
*《遺教經論記》三卷，宋[[觀復|觀復]]述。
*《佛遺教經補註》一卷，明[[守遂|守遂]]註，[[了童|了童]]補註。
*《佛遺教經解》一卷，明[[智旭|智旭]]述。
*《遺教經指南》一卷，明[[道霈|道霈]]述。
*《佛遺教經箋註》，[[丁福保|丁福保]]註。
*《佛遺教經講義》，[[民國|民國]]寶靜法師述<ref>{{cite web |author1=寶靜大師 |title=佛遺教經講義 |url=https://drive.google.com/file/d/0B-CWIPaIaeaLb05SR05WU3hyU2ROUnA5NDBlaFFsY1dIaUc0/view |publisher=香港佛經流通處}}</ref>。
