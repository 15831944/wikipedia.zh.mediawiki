'''佛曆'''（{{lang-pi|{{IAST|Sāsanā Sakaraj}}}}），一種傳統[[曆法|曆法]]，為[[陰陽曆|陰陽曆]]，這種曆法盛行於[[南亞|南亞]]和[[东南亚|东南亚]]的佛教國家，如[[柬埔寨|柬埔寨]]、[[泰國|泰國]]。其[[紀年|紀年]]法，稱'''佛滅紀元'''，以[[釋迦牟尼佛|釋迦牟尼佛]][[滅度|滅度]]（[[涅槃|涅槃]]）當年度為計算基準。
在中國[[漢傳佛教|漢傳佛教]]有以佛陀出生年為紀元，於《[[魏書|魏書]]》沙門曇謨最說：佛陀以[[周昭王|周昭王]]二十四年四月八日示現降生，[[穆王|穆王]]五十二年二月十五日示現入滅的說法，因此西曆2018年為中國佛曆3045年（西元前1027年是甲寅年）。

==簡介 ==
[[佛教|佛教]]創始人[[釋迦牟尼|釋迦牟尼]]原名乔达摩·悉达多，古[[印度|印度]][[释迦族|释迦族]]人。释迦牟尼为尊称，意为释迦族的圣人。又被称为「[[佛陀|佛陀]]」（觉者）、「世尊」等。其生卒年代，北传佛教根据汉译《[[善见律毗婆娑|善见律毗婆娑]]》中「[[出律记|出律记]]」推断为公元[[前565年|前565年]]至[[前486年|前486年]]，南传佛教则作公元[[前624年|前624年]]至[[前544年|前544年]]或公元[[前623年|前623年]]至[[前543年|前543年]]。

公元[[1950年|1950年]]，首屆「世界佛教徒友誼會」在[[錫蘭|錫蘭]]首都[[可倫坡|可倫坡]]舉辦。會中議決：佛陀誕生於公元前623年，成道於公元[[前588年|前588年]]，去世於公元前543年。[[1954年|1954年]]，該年會於[[緬甸|緬甸]][[仰光|仰光]]舉行，會中再決議佛教國家以「佛曆」紀元，並以釋迦牟尼[[涅槃|涅槃]]年推算，公元[[1954年|1954年]]為佛曆2497年。

== 紀元起始 ==
: 佛曆年 = 公曆年 + 543，若過[[佛誕日|佛誕日]]（[[印度曆|印度曆]]二月初八，[[農曆|農曆]]四月初八）再加1年。
**泰國每年的水燈節是在佛曆12月15日 （農曆10月15日）慶祝.
: 泰國官方之[[泰國曆|佛曆年]] = 公曆年 + 543，採公曆之[[1月1日|1月1日]]為[[歲首|歲首]]。但民間多採公曆之4月1日（[[泼水节|泼水节]]所在的月）為歲首。
: 北傳佛教以公元前544年為元年，佛曆年=公曆年+544，以農曆歲首為歲首，常搭配干支紀年使用。

==月、日的設置==
佛曆有專門的規定來設置每月的日數，佛曆的月份設置是按照月球運動設置的。詳見本條目英文版。

目前中國佛教徒所使用的佛曆，只用佛曆紀年，其餘月、日安排多使用農曆（亦有用格里曆的）。
