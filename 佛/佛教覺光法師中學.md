'''佛教覺光法師中學'''，是一間位於[[香港|香港]][[沙田區|沙田區]][[城門河東|城門河東]][[沙角邨|沙角邨]]的[[中學|中學]]，創校於1981年。該校是[[香港佛教聯合會|香港佛教聯合會]]主辦之第八間中學，也是沙田區第一間佛教團體主辦之政府資助中學。

==歷史==
1979年（佛曆2523年）適逢會長覺光法師六十大壽，香港佛教聯合會倡議籌辦一間中學，以覺光會長法號命名。經會方向教育署申請，獲准撥給沙田沙角邨，新建標準中學校舍，命名為「佛教覺光法師中學」。

學校同時舉辦多項活動，例如：教視障人仕做運動、探訪老人院、晚大掃除、探訪獨居長者。

==辦學宗旨及學校目標==

# 培養學生良好的語文能力。
# 培養學生良好的數學能力。　
# 培養學生勤奮好學的精神，並提昇其自學能力及求知慾。
===體育===
# 鍛鍊學生有強健的體魄。
# 培養學生的紀律及堅強意志。
# 培養學生的體育精神及合作精神。
# 協助學生掌握常見運動的活動技巧。
===群育===
# 透過各種整體活動以培養學生合群的個性及適應團體生活的能力。
# 協助學生認識個人在社會中的權利及義務，並培養其公民意識。
# 協助學生認識中國及培養正確的民族意識。
# 提供自行策劃、組織活動的機會，培養學生的領導才能。
# 透過各種小組活動以發展學生之潛能及互助互勵精神，使其於學業、校園生活及其他方面得到健全發展。
===美育===
# 培養學生對藝術的興趣及對美的欣賞能力。

# 融會佛教慈、悲、喜、捨的精神於生活中，以建立和諧的人生。

# 透過佛教的修證與活動以淨化學生的身心，使能分別善惡、明辨是非。

==校訓==
* 明智（明平等智）：我們要徹底研究明白「平等等智」的佛德，明白有自己亦同時有他人的道理。
* 顯悲（顯同體悲）：一切眾生的身體，與己身同為一體，同樣會發生拔除痛苦，興起快樂的心，這便是同體的慈悲心。

==校內組織==
===一般組織===
*社：設有四社，包括：慧社（紅）、信社（黃）、慈社（藍）、勤社（綠）。於各社裏設有社長及社幹事等職位，以作負責社際活動的安排。
*學生會：每年都會進行選舉，由全校學生投票。主要負責學生福利及舉辦學生活動。
*領袖生：主要負責學生紀律工作。
*義工領袖：主要負責帶領新生投入校園生活，及協助學生進行每年的義工服務。

==校園生活==
===學生會活動===
學生會每年都會舉辦不同的活動予學生，較為傳統的活動有：
*陸運會暨畢業班20x100接力賽
*敬師周
*[[歌唱|歌唱]]比賽

===敬師周===
每年農歷年假前,學生會均會主辦敬師周，以作學生感謝老師的一項大型活動。學生可以邀請老師參加各項活動，藉此拉近師生關係。活動包括綜合表演、班際跳大繩、20人21足等。

===社際比賽===
社際活動為每年的固定學生活動，活動項目有動亦有靜。動態活動包括社際[[排球|排球]]、[[羽毛球|羽毛球]]、[[乒乓球|乒乓球]]等比賽。而靜態活動有社際[[辯論|辯論]]、[[戲劇|戲劇]]等比賽。

===義工活動===
每年中一至中五級學生均會參與「一人一義工」計劃，亦即是每位學生每年均會參與最少一項義工活動，希望學生能愛己愛人。

===課外活動===
學校為培養學生有多方面興趣，學習各種社交技巧，幫助學生個人發展及成長。學校共設有超過30種課外活動小組，大致分為[[學科|學科]]類、[[體育|體育]]類、[[藝術|藝術]]類及[[興趣|興趣]]類。

==學校成就==
大學及大專入學率達九成，近三年DSE最佳五科增值為8級（最高為9級），顯示學校的教學及課研效能甚佳。杜偉傑同學於中學會考中榮獲7A佳績。伍燿煇同學於香港高級程度會考中考獲4A佳績。
