'''佛山一环高速公路'''，简称'''佛山一环'''。是环绕[[广东省|广东省]][[佛山市|佛山市]]的一条城市快速路，全长99.21公里，总投资约120亿元人民币，经过[[禅城区|禅城]]、[[南海区|南海]]和[[顺德区|顺德]]3个区的11个镇街，围绕519平方公里的土地，为广东省内最长的环城公路。2006年11月18日通车，2008年12月被广东省交通厅定为省级[[高速公路|高速公路]]，编号为'''S82'''，目前全线为免费通行<ref>[http://fs.house.sina.com.cn/2010-04-19/090711503.html 佛山一环高速至少两年内不收费 下月更换新国标] {{webarchive|url=https://web.archive.org/web/20160304102436/http://fs.house.sina.com.cn/2010-04-19/090711503.html |date=2016-03-04 }} 新浪网，于2010年10月21日查阅</ref>，2017年根据广东省交通厅对编号作出的调整，原粤高速公路编号取消<ref>[http://bbs.rednet.cn/forum.php?mod=viewthread&tid=46697857&page=1  广东省交通运输厅关于印发广东省高速公路网路线命名和编号规则（2017年修订版）的通知] {{Wayback|url=http://bbs.rednet.cn/forum.php?mod=viewthread&tid=46697857&page=1# |date=20180122001046 }} 红网论坛. 于2017年3月20日查阅.</ref>。佛山一环将高速化，也意味着佛山一环即将变成收费高速公路<ref>[http://gd.qq.com/a/20161229/002560.htm 佛山一环高速化改造启动 预计2020年陆续改为封闭收费高速] 大粤网，于2017年3月30日查阅。</ref>。

== 分段情况 ==
佛山一环由东、西、南、北四段连接成环，另有从东环分出的北延线及南延线。

;东线
[[和顺|和顺]]至[[北滘|北滘]]公路主干线为一环东线，长36.4公里，道路等级主路为一级公路兼城际快速路，辅路为城市主干路，主路设计时速为100公里，设双向八车道，辅路的设计时速为50公里，设双向六车道。总造价约为57亿元人民币，为粤高速'''S47'''的重要组成部分。

;南线
北滘至[[乐从|乐从]]公路主干线为一环南线，长15.3公里，道路等级主路为一级公路兼城际快速路，辅路为城市主干路，主路设计时速为100公里，设双向八车道，辅路的设计时速为50公里，设双向六车道。总造价约为20亿元人民币，为粤高速'''S5'''的重要组成部分。

;西线
乐从至[[狮山|狮山]]公路主干线为一环西线，长31.9公里，道路等级为主路一级公路结合城市道路，设计时速为100公里，设双向八车道。总造价约为26.5亿元人民币，为粤高速'''S16'''的重要组成部分。

;北线（G321新线）
狮山至和顺公路主干线为一环北线，长15.6公里，道路等级为主路一级公路结合城市道路，同时作为[[321国道|321国道]]新线路，设计时速为100公里，设双向八车道。总造价约为16.5亿元人民币，为粤高速'''S8'''的重要组成部分。<ref>[http://epaper.citygf.com/szb/html/2011-09/13/content_452951838.htm G321、G325国道迁移] {{Wayback|url=http://epaper.citygf.com/szb/html/2011-09/13/content_452951838.htm |date=20150110151546 }} 佛山日报，2011年9月13日</ref>

;北延线
又称“佛山市和顺棠溪至料美公路主干线”，全长9.1公里，主要控制点有和顺南立交、西南涌大桥、和顺北立交。起于里水镇和顺棠溪村东北，经小文教西、和顺规划区东侧边缘，与[[广州西二环高速公路|广州西二环高速公路]]相接，再往北终点与花都规划的红棉大道相接。工程共设互通式立交3处，建设标准为一级公路，主路为双向8车道，设计行车速度100Km/h，工程估算总投资6.7亿元，属于粤高速'''S47'''的重要组成部分。北延线是佛山市区前往[[广州白云国际机场|白云机场]]最快捷的通道。
