佛山科学技术学院（Foshan University），原佛山大学，国家硕士学位授予单位，广东省和佛山市重点建设高水平理工科大学。

佛山科学技术学院于1995年合并组建，起源于佛山大学和华南农学院佛山分院，最早可以追溯到1924年。1995年本科招生，2013年获硕士学位授予权，2015年博士联招，2016年部分专业调整至一本招生。

佛山科学技术学院现有江湾、仙溪、河滨三个校区，总面积3789.7亩，建筑面积70.81万平方米。现有在校全日制学生14813人，其中硕士研究生223人，教职工1424人，有42个本科专业，涵盖十大学科门类，有1个博士后创新实践基地，3个一级学科硕士学位授权点，3个专业硕士学位授权点。<ref>{{cite web |url=http://web.fosu.edu.cn/school-intro/school-overview |title=佛山科技学院简介 |publisher=佛山科技学院 |date=2016-01-01 |accessdate=2016-04-08}}</ref><ref>{{cite web |url=http://web.fosu.edu.cn/school-intro/school-evolution |title=佛山科技学院历史沿革 |publisher=佛山科技学院 |date=2015-08-01 |accessdate=2015-08-01}}</ref><ref>{{cite web |url=http://web.fosu.edu.cn/school-intro/present-leaders |title=佛山科技学院现任领导 |publisher=佛山科技学院 |date=2015-12-01 |accessdate=2015-12-01}}</ref>。

== 发展历程 ==
1958年，佛山师范专科学校成立，华南农学院成立华南农学院佛山分院。

1986年2月，在佛山师范专科学校基础上，佛山大学成立。

1986年，华南农学院佛山分院更名为佛山兽医专科学校，后更名为佛山农牧高等专科学校。

1995年3月，佛山大学和佛山农牧高等专科学校合并，更名佛山科学技术学院，对外交流继续沿用 Foshan University 的校名。

2005年2月，佛山职工医学院和佛山教育学院并入佛山科学技术学院。

2007年，佛山科学技术学院以优良成绩通过教育部组织的本科教学工作水平评估。

2013年7月，佛山科学技术学院经国务院学位委员会批准为硕士学位授予单位。

2015年9月，佛山科学技术学院被确定为广东省重点建设高水平理工科大学。

2016年4月，部分专业纳入第一批本科招生。

2016年，成立佛山大学科技园。<ref>{{cite web |url=http://web.fosu.edu.cn/school-intro/school-overview |title=佛山科技学院简介 |publisher=佛山科技学院 |date=2016-01-01 |accessdate=2016-01-01}}</ref>

== 办学条件 ==
佛山科技学院教学科研仪器设备值 17642.20万元。图书馆三座，拥有纸质图书1727133册，电子图书384.8万册，中外文期刊报纸2217种（其中外文期刊81种），电子期刊数据库4个。学校共有多媒体教室和语音室151间，座位数17265个，普通教室52间，座位数3358个，校外实习基地 211 个。拥有运动场、体育馆、游泳馆、网球场、俱乐部等现代化的体育教学设施，建设有高速安全、运行稳定的校园网，为广大师生提供优质的网络服务。<ref>{{cite web |url=http://web2.fosu.edu.cn/jiaowu/jwcjwc/jxdt/ArticleShow.asp?ArticleID=2574 |title=佛山科学技术学院2015年本科教学质量报告 |publisher=佛山科技学院 |date=2015-10-28 |accessdate=2016-01-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20160323084546/http://web2.fosu.edu.cn/jiaowu/jwcjwc/jxdt/ArticleShow.asp?ArticleID=2574 |archivedate=2016-03-23 }}</ref>

== 未來发展 ==
到2017年，在理工科领域学科专业实力明显增强，在全国大学综合排名提升100位左右，在全国理工类院校排名明显提升。

到2018年，高质量完成教育部本科教学工作审核评估建设任务。

到2020年，与珠江西岸先进装备制造业相关的学科专业水平达到国内前列、省内一流，在全国理工类院校排名进入前100位，服务广东省和佛山市经济社会发展成效显著增强，建成与产业深度融合、地方特色鲜明的广东省高水平理工科大学。<ref>{{cite news |url=http://gd.sina.com.cn/fs/yaowen/2015-10-12/city-fs-ifxirmpy1518102.shtml |title=佛科院将建成高水平理工科大学 |publisher=新浪网引珠江商报 |date=2015-10-12 |accessdate=2016-01-06}}</ref>。
