'''佛教茂峰法師紀念中學'''（{{lang-en|Buddhist Mau Fung Memorial College}}，簡稱{{lang|en|BMFMC}}或'''茂峰'''）是座落於[[香港|香港]][[新界|新界]][[元朗區|元朗區]][[天水圍新市鎮|天水圍新市鎮]]的一所[[津貼學校|津貼]]全日制男女中文中學，亦是香港佛教聯合會的其中一間學校，校址位於天水圍[[天柏路|天柏路]]18號，鄰近[[天慈邨|天慈邨]]和[[天水圍游泳池|天水圍游泳池]]。

學校創立於1998年，創校校長為陳燕輝。現時學生逾千人，注重[[資訊科技|資訊科技]]方面的知識及教育，並有出色的成就。該校亦在2003年開辦校園電視台，現逢星期二、四就會在課室進行早會，亦間中會不定時直播典禮，亦有影片、音樂分享。

== 辦學宗旨 ==
學校於1998年由[[香港佛教聯合會|香港佛教聯合會]]創立的。秉承佛教聯合會的辦學方針，以「明智顯悲」為校訓，致力啟發同學的智慧，培養他們的慈悲心。

== 特色 ==
===資訊科技===
佛教茂峰法師紀念中學從2000年代開始，與[[香港中文大學|香港中文大學]][[資訊科技教育促進中心|資訊科技教育促進中心]]的[[李芳樂|李芳樂]]教授合作，成為了推廣[[探索網站|探索網站]]（[[WebQuest|WebQuest]]）的先導學校；後來基於同樣原因，獲[[教育局_(香港)|教育局]]邀請成為第21所[[資訊科技教育卓越中心|資訊科技教育卓越中心]]，由該校負責[[IT|IT]]方面的[[文可為|文可為]]助理校長負責（已離校）。

===中一團隊軍訓===
每一年，為了培養中一學生的訓練團隊合作性和自立能力,中一生在學期初時會到[[黃埔軍校|黃埔軍校]]深圳分校進行三日兩夜的訓練，其間訓練包括步操、射擊、過障礙物及試坐行走中的[[坦克車|坦克車]]等。

從二零一三年起取消中一團隊軍訓。

===一生一體藝活動===
學校由2005年開始設立一生一體藝，全中一級的學生必須挑選最少一項體或一項藝或最多兩項體藝（一體及一藝）的活動參加，學校會聘請校外合格及富經驗導師教授，於課後及小組形式後進行教授，更可有資助，但必須符合以下條件1. 出席率達90%，可獲半費資助，共於課程完結後退回2. 如同學家庭經濟有困難，可向學校申請全額資助

* 藝術課程內容：銀樂隊、合唱團、戲劇、中國敲擊鼓樂、舞蹈，體育課程內容：跆拳道、游泳、籃球、足球、排球、田徑

===學生會===
每年學校也會舉辦學生會選舉，並讓同學一人一票選出心儀的學生會，學生會的職務有很多，包括舉辦各類型運動比賽、社際問答比賽（中國文化/科學）、社際足球/籃球比賽。更有提供外借服務(例如球類,雨傘等)、代購文具、班際足球/籃球比賽等。

===才藝表演===
學校亦會一年一度舉行才藝表演，全校學生師生都必須出席。表演中表演節目有很多，包括歌唱表演（7月上旬），中國鼓，舞蹈，朗誦，中一集誦等，令師生關係更加密切。

===其他活動===
學校間中會舉辦[[盆菜|盆菜]]宴，燒烤晚會，開放日。而不定時地在每一星期舉行不同的主題周，按不同主題有不同的活動。例如數理周就會有[[數獨|數獨]]練習、數字遊戲牌Rummikub，而[[資訊科技|資訊科技]]周則有輸入法比賽、裝嵌[[電腦|電腦]]、圖片設計及問答比賽，寓學習於娛樂。

==社制==
該校共分四社，分別為羅漢社（黃社）、菩提社（綠社）、無憂社（藍社）、合掌社（紅社）。每位新生入學時，均會被隨機分配至以上四社其中一社。

學生會社會舉辦不同的社制活動，例如烹飪、足球、籃球比賽等。而每年都設有社制全年總冠軍，由四社爭奪，分數在陸運會等大小比賽及活動計分，至2011-2012年度為止，羅漢社奪得最多次數全年總冠軍。而2012-13年度則由菩提社以1000分重奪全年總冠軍，至今已連奪四屆全年總冠軍。

== 獎項 ==
*2012年，佛教茂峰法師紀念中學三位老師獲得2011-2012年度[[行政長官卓越教學獎|行政長官卓越教學獎]]嘉許狀（資訊科技教育）。<ref>[http://qef.org.hk/ate/2011_12ate/tchinese/assesresult2.htm#it 行政長官卓越教學獎(2011/2012)]</ref>
*2012年，佛教茂峰法師紀念中學張詠桐同學參加由元朗區議會及元朗區體育會主辦，元朗區議會贊助的獲得「元朗區體育節2012——新界區跆拳道公開賽」，獲得亞軍。
*2012年，10位中三級學生參與「長者友善社區短片製作比賽」，勇奪「突出主題獎」。4月13日頒獎典禮中，行政會議成員梁智鴻醫生更親自頒發獎項予校園電視台的學生代表。該校參賽短片以「理想居所」為主題，介紹理想的長者友善社區中各項設施配套。參賽短片主題鮮明，手法生動，同學以社區的實地拍攝，配以有趣的樂高積木玩偶，並精心設計故事背景、對話及配音，與是次比賽「促進長幼共融」的主題互相呼應。
