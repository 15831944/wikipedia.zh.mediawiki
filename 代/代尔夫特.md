'''代尔夫特'''（{{lang-nl|Delft}}，{{IPA-nl|dɛlft||Nl-Delft.ogg}}），又譯'''台夫特'''，是[[荷兰|荷兰]][[南荷兰省|南荷兰省]]的一个城市，地处[[海牙|海牙]]和[[鹿特丹|鹿特丹]]之间。由于拥有荷兰高等学府[[代尔夫特理工大学|代尔夫特理工大学]]和研究机构荷兰应用科学研究所，代尔夫特也被称为知识之城。

== 历史 ==
代尔夫特起源于此地一条名叫奧德台夫特（{{lang-nl|Oude Delf}}）的人工水道，直譯是「古代排水閘」。Delf 的动词{{lang-nl|delven}}，意思是挖掘。台夫特古代以堤防擋住北海，荷蘭人挖掘出閘水道，用來排乾沿海沼地，奧德台夫特（{{lang-nl|Oude Delf}}）因此得名。

后来在这里逐渐形成了一个重要的市集中心，大约相当于今天市中心市集广场的位置。代尔夫特是荷兰历史上获得城市权力比较早的城市之一。1246年，伯爵威廉二世授予其城市权利。在历史上代尔夫特经历了两次较大的事故，1536年，市中心一场大火摧毁了大量的房屋。1654年，一个军火库爆炸将城市的一部分夷为平地。<br />
代尔夫特是[[荷兰东印度公司|荷兰东印度公司]]在荷兰的6个据点之一。也就是在那时中国的瓷器引入荷兰，并发展成了荷兰著名的代尔夫特[[代尔夫特蓝陶|蓝陶]]。威廉亲王曾在此地短暂居住，并被刺死在如今位于市中心的[[亲王庭院_(代尔夫特)|亲王庭院]]的地方。当时王室所在地[[布雷达|布雷达]]还在[[西班牙|西班牙]]控制下，又因为他住在代尔夫特，于是市中心的新教堂被选作安息之地，此后王室成员死后遗体都照例放入新教堂的地窖，代尔夫特因此和荷兰王室结下了不解之缘。

== 重要机构 ==
[[代尔夫特理工大学|代尔夫特理工大学]]是荷兰三所高等理工类大学之一（另两所是埃因霍温理工大学和屯特大学）。

== 旅游 ==
[[File:Nieuwe_kerk_20040311.jpg|thumb]]
代尔夫特是荷兰著名观光城市之一。代尔夫特拥有一个历史性内城，内城以前有环城水道护城，但现在只能看到部分河段。内城遍布水道和小桥，老建筑林立，和阿姆斯特丹的市中心颇有相似之处，因此也有人称代尔夫特是[[阿姆斯特丹|阿姆斯特丹]]的缩影。代尔夫特最著名的纪念品是源自中国的[[代尔夫特蓝陶|蓝陶]]，荷兰文称Delfts Blauw。

== 交通 ==
代尔夫特的公共交通非常发达，主要有[[列车|列车]]、[[有轨电车|有轨电车]]和[[公共汽车|公共汽车]]。铁路方面，代尔夫特位于[[阿姆斯特丹|阿姆斯特丹]]-[[海牙|海牙]]-[[鹿特丹|鹿特丹]]主线上，出入和换乘都很方便。目前有一条有轨电车线，由海牙电车公司经营，从市区南部北上贯穿海牙一直延伸到海牙著名海滩[[斯赫弗宁恩|斯赫弗宁恩]]。
