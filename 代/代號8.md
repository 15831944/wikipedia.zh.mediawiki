== 製作 ==
=== 開發 ===
2016年，羅比和史蒂芬·艾梅爾發行了一部短片《{{link-en|代號8 (2016年電影)|Code 8 (2016 film)|代號8}}》，該片作為潛在的長篇電影的預告片。他們請求籌款20萬美元，籌款活動迅速籌集了100萬美元，目前已超過200萬美元<ref>{{Cite web |url = https://www.indiegogo.com/projects/code-8-a-film-from-robbie-stephen-amell#/ |title = Code 8 - a film from Robbie & Stephen Amell |publisher = Indiegogo |language = en}}</ref>。2017年6月12日，Laysla De Oliveira被揀選在電影中演出<ref>{{Cite web |url = https://deadline.com/2017/06/laysla-de-oliveira-code-8-james-jagger-the-last-full-measure-1202112111/ |title = Laysla De Oliveira To Co-Star In ‘Code 8’; James Jagger Joins ‘The Last Full Measure’ |author = Amanda N'Duka |date = 2017-06-12 |publisher = Deadline Hollywood |language = en}}</ref>。

=== 拍攝 ===
主要拍攝於2017年6月1日在[[安大略省|安大略省]]的[[多倫多|多倫多]]開始。<ref>{{Cite web |url = https://www.screendaily.com/news/production-begins-on-sci-fi-thriller-code-8/5118888.article |title = Production begins on sci-fi thriller 'Code 8' |author = Elbert Wyche |date = 2017-06-08 |publisher = screendaily |language = en}}</ref>
