'''宜都市'''是地处[[中華人民共和國|中華人民共和國]][[湖北省|湖北省]]西南部的一个[[县级市|县级市]]，隶属地级[[宜昌|宜昌]]市管辖，位于[[长江|长江]]南岸，[[长江|长江]]与其支流[[清江|清江]]交汇处。素有“楚蜀咽喉，鄂西门户”之称，面积1357平方公里，人口39.3万人。隶属于三峡宜昌“半小时经济圈”，鄂、渝、湘三省市交界处，境内建有两座长江大桥、两座清江大桥，从市区到三峡机场最短距離37.6公里，焦柳铁路贯穿南北，两江可通航。宜都古称'''夷道'''，原名'''枝城市'''，1998年6月更名為宜都市。

'''宜都市'''2006，2007年全省县域经济综合排名第一名，2008，2009，2010年全省县域经济综合排名第二名。宜都市位于湖北省西南部，长江南岸，清江与长江两江交汇处，依山傍水，人杰地灵。地理坐标为东经110°05′-111°36′，北纬30°05′-30°36′。东北隔长江与枝江市相望，东南邻松滋市（有一块飞地在松滋市王家桥镇内），西南、正西与五峰、长阳土家族自治县交界，北与宜昌市、夷陵区接壤。

== 基本情况 ==
宜都市位于[[长江|长江]]与[[清江|清江]]交汇处，是[[湖北省|湖北省]]38个山区县市之一，面积1357[[平方公里|平方公里]]，是一座具有2100多年历史的古城。1992年至今属[[宜昌市|宜昌市]]管辖。政府驻[[陆城街道|陆城街道]]。

== 地理 ==
宜都市地处长江中游近三峡出口、鄂西南部，处于[[江汉平原|江汉平原]]向鄂西山区的过渡地带。版图面积1357平方公里，东北隔长江与[[枝江市|枝江市]]接界，东南与[[松滋市|松滋市]]相邻，西南与[[五峰土家族自治县|五峰县]]接壤，西北与[[长阳土家族自治县|长阳]]、[[宜昌|宜昌]]两县市相连。

宜都市处于鄂西山地和江汉平原过渡地带，地势西南高、东北低，由西南向东北倾斜，是一个丘陵起伏的半山区。最高点为[[五峰土家族自治县|五峰]]接壤的帽子尖，海拔1064.6米，最低点为[[枝城镇|枝城镇]]的官洲，海拔仅38米。西南地势高竣，群山连绵，高程在250-800米之间，约占总面积的40%。东部[[丘陵|丘陵]]，海拔在50-250米，沿长江及清江出口地势平坦，土地肥沃，中部丘陵、冲沟与岗地交错，但坡度较缓，形成平畈，是宜昌市粮油和农特产品的主要产地。

宜都市属[[亚热带季风气候|亚热带季风气候]]，四季分明，气候温和，雨量充沛，光照充足，严冬酷暑时间短。全市的农业气候条件比较优越。

*1962年10月22日，復置[[枝江|枝江]]縣。 次年11月，宜都、[[枝江|枝江]]分署辦公。宜都縣劃為6個區（姚店、紅花套、聶家河、潘灣、王畈及[[枝城|枝城]]和2個鎮（城關和枝城）。
* 1975年11月5日，撤銷原6個區，建11個[[人民公社|人民公社]]：姚店公社、[[紅花套|紅花套]]公社、[[古老背|古老背]]公社、[[聶家河|聶家河]]公社、潘灣公社、[[毛湖埫|毛湖埫]]公社、王畈公社、[[松木坪|松木坪]]公社、[[五眼泉|五眼泉]]公社、[[枝城|枝城]]公社、黎坪公社。城關鎮和[[枝城鎮|枝城鎮]]仍為縣轄鎮。
*1981年6月，城關鎮易名為[[陸城|陸城]]鎮。
*1982年12月7日，[[古老背|古老背]]鎮劃歸[[枝江|枝江]]縣。
*1984年，撤[[人民公社|人民公社]]建8個區（姚店、[[紅花套|紅花套]]、[[枝城|枝城]]、 [[松木坪|松木坪]]、[[聶家河|聶家河]]、王畈、[[毛湖埫|毛湖埫]]及潘灣區）和2個鎮（[[陸城|陸城]]鎮和[[枝城|枝城]]鎮）。
*1985年11月16日，調整為5個[[鎮|鎮]]（[[陸城|陸城]]、[[枝城|枝城]]、[[紅花套|紅花套]]、[[聶家河|聶家河]]及[[松木坪|松木坪]]和4個區（姚店、[[毛湖埫|毛湖埫]]、潘灣、王畈）。
*1987年9月，撤4個區，設7個鄉：[[姚家店|姚家店]]、[[曾家崗|曾家崗]]、[[黎家坪|黎家坪]]、[[王家畈|王家畈]]、[[毛湖埫|毛湖埫]]、潘家灣和[[五眼泉|五眼泉]]鄉。
* 1988年2月，撤[[陸城|陸城]]鎮、[[枝城|枝城]]鎮設[[陸城|陸城]]街道辦事處、[[枝城|枝城]]街道辦事處和[[洋溪|洋溪]]街道辦事處，全縣轄3個街道辦事處、3個鎮和7個鄉。
*1998年1月，撤曾家崗鄉建為[[高壩洲|高壩洲]]鎮。
*2001年2月，[[毛湖埫|毛湖埫]]鄉并入王家畈鄉，洋溪鎮、黎家坪鄉并入[[枝城|枝城]]鎮。

== 历史 ==
宜都市是一座有着2100多年历史的古城，中华早期文明的发祥地之一，红花套城背溪新石器时代遗址的发现，证明7000多年前，中华民族的祖先曾在这里农牧渔猎。春秋战国时期，境属楚地，秦朝时县域属南郡，西汉武帝建元六年(公元前135年)置县，名夷道县，至今已有2138年的建制史。
境内有7000多年前的红花套城背溪[[新石器时代|新石器时代]]遗址。[[春秋|春秋]][[战国|战国]]时期，境属楚地，[[秦朝|秦]]时县域属南郡，[[西汉|西汉]][[汉武帝|武帝]][[建元|建元]]六年 (公元前135年)置县，名夷道县。[[东汉|东汉]][[建安|建安]]十五年(210年)[[刘备|刘备]]改临江郡为宜都郡，宜都之名即始于此，取“'''宜于建都'''”之意，并派大将[[张飞|张飞]]为宜都太守。[[建安|建安]]二十四年(219年)，[[东吴|东吴]]大将[[陆逊|陆逊]]占领宜都郡，获取夷道、夷陵、秭归等县，并任宜都太守在此筑城抗蜀，历史上著名的“[[夷陵之战|夷陵之战]]”就发生在境内。
　　　东汉建安十五年(210年)刘备改临江郡为宜都郡，宜都之名即始于此，取"宜于建都"之意，并派大将张飞为宜都太守。建安二十四年(219年)，吴大将陆逊占领宜都郡，获取夷道、夷陵、秭归等县，并任宜都太守在此筑城抗蜀，历史上著名的"夷陵之战"就发生在境内。市府所在地故称"陆城"。自古以来，宜都就有“楚蜀咽喉”、“鄂西门户”、“三峡门城”的美誉，历来为兵家必争之地。 宜都是晚清著名学者杨守敬的故乡，杨守敬在历史地理、书法、金石、版本目录、藏书等多个领域著述宏丰、学高品富，被日本书法界尊称为“[[开山鼻祖|开山鼻祖]]”，今天[[长江|长江]][[三峡|三峡]]的名称，就是由他所定，[[杨守敬纪念馆|杨守敬纪念馆]]座落[[清江|清江]]之滨。宜都还是[[中国人民解放军|中国人民解放军]]独臂将军[[贺炳炎|贺炳炎]]、[[宜昌市|宜昌市]]第一个共产党员[[胡敌|胡敌]]、湖北医科大学创始人[[朱裕壁|朱裕壁]]的故乡。
