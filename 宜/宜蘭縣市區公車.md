'''宜蘭縣市區公車'''，為[[宜蘭縣政府|宜蘭縣政府]]所監理的市區[[汽車客運|汽車客運]]服務，為宜蘭縣政府所規劃的「宜蘭勁好行幸福交通網」的一環。2007年11月，[[國道五號|國道五號]]汽車客運開通，宜蘭縣政府欲藉由帶動宜蘭縣市區汽車客運事業的發展，故就溪北地區、溪南地區分別規劃市區公車路線。2008年5月8日，溪南地區的四條市區客運路線開通，溪北地區路線因故延宕。2011年10年1日，新增九條路線，溪北地區的市區客運路線方告開通。

目前到（2018年10月1日），總計開通了50
條宜蘭縣轄市區公車路線，由[[大都會客運|大都會客運]]、[[首都客運|首都客運]]、[[葛瑪蘭客運|葛瑪蘭客運]]、[[國光客運|國光客運]]分別經營。

==歷史==
* 1960年代：宜蘭汽車客運公司與「蘭陽汽車客運公司」籌備處分別申請經營市區汽車客運業務，但皆未獲宜蘭縣政府核准。
* 2005年：[[大都會客運|大都會客運]]曾得標宜蘭縣市區公車經營權，惟在成本考量下，決定放棄經營權。
*2007年
**11月15日：配合「臺北－宜蘭」國道客運通車，自本日起評選宜蘭縣轄市區公車經營業者，期限至12月30日<ref>{{Cite web|url = http://news.ltn.com.tw/news/local/paper/166018/print|title = 市區公車接駁，溪北線業者沒興趣|accessdate = 2014-12-16|author = 游明金、江志雄|date = 2007-11-06|publisher = 自由時報}}</ref>。
**12月10日：[[首都客運|首都客運]]自本日起提供「羅東－傳藝中心－南方澳」免費接駁車服務，以接駁國道客運<ref>{{Cite web|url = http://www.ly.gov.tw/03_leg/0301_main/leg_news/newsView.action?id=29921&lgno=00063&stage=6&atcid=29921|title = 林建榮促成國道客運免費接駁羅東至蘇澳南方澳五結|accessdate = 2014-12-17|date = 2007-12-06}}</ref>。
*2008年
**3月7日：完成溪北地區路線評選，由蘭陽汽車客運股份有限公司籌備處得標。
**4月：[[葛瑪蘭客運|葛瑪蘭客運]]自本日起提供「羅東－蘇澳」免費接駁車接駁乘客服務。
**5月8日：宜蘭縣轄市區公車溪南地區路線正式通車，原「羅東－傳藝中心－南方澳」免費接駁路線亦變更為宜蘭縣轄公車'''<font color="red">241</font>'''線。
**6月：原定於月中通車的蘭陽客運溪北地區路線，申請展延至7月28日。
**6月25日：葛瑪蘭客運自本日起提供「礁溪－頭城」免費接駁車服務。
**7月28日：原定於本日通車的蘭陽客運，再度申請展延半年。
*2009年
**2月28日：蘭陽客運仍未正式行駛通車，主管機關依法撤銷其經營權。
**7月11日：「礁溪－頭城」免費接駁車於本日起停駛，並更改行駛路線。('''<font color="red">112</font>'''線)
*2010年
**4月5日：葛瑪蘭客運自本日起提供「葛瑪蘭礁溪站－林美石盤步道」免費接駁車服務。
**8月24日：葛瑪蘭客運自本日起提供「羅東後火車站－冬山河親水公園－國立傳統藝術中心」免費接駁車服務。
*2011年
**10月1日：新增九條路線，溪北地區路線上路<ref>{{Cite web|url = https://tw.news.yahoo.com/%E9%81%8A%E5%AE%9C%E8%98%AD%E6%9B%B4%E6%96%B9%E4%BE%BF-%E4%BB%8A%E8%B5%B7%E6%96%B0%E5%A2%9E9%E6%A2%9D%E5%B8%82%E5%8D%80%E5%85%AC%E8%BB%8A%E8%B7%AF%E7%B7%9A-060457075.html|title = 遊宜蘭更方便 今起新增9條市區公車路線|accessdate = 2014-12-17|author = 黃麗鳳|publisher = 中廣新聞網}}</ref>。
*2015年
**6月15日：新增紅綠路線，路線上路。
*2016年
**6月1日：宜蘭勁好行6月試營運，持電子票證免費<ref>{{Cite web|url = https://tw.news.yahoo.com/%E5%88%B7%E9%9B%BB%E5%AD%90%E7%A5%A8%E8%A8%BC-%E9%81%8A%E5%AE%9C%E8%98%AD%E5%85%8D%E8%B2%BB%E6%90%AD%E5%85%AC%E8%BB%8A-160000215.html|title = 刷電子票証 遊宜蘭免費搭公車|accessdate = 2016-05-15|author = 董秀雲|publisher = 台灣新生報 }}</ref>。
**10月1日：宜蘭勁好行正式收費。
*2017年
**5月18日：'''<font color="red">1766</font>'''號、'''<font color="red">1767</font>'''號、'''<font color="red">1792</font>'''號正式移撥為宜蘭縣市區客運路線。
**9月9日：國光客運汰換20輛低地板大客車
*2018年
**2月10日：國光客運先行汰換10輛低地板大客車
**5月7日：蘇花改開放大客車通行，公車暌違36年上路
**10月1日：國光客運20條一般公路客運路線即日起移撥為宜蘭縣市區公車路線
**12月1日：原「羅東轉運站－蘇澳車站」接駁路線亦變更為宜蘭縣轄公車'''<font color="red">243</font>'''線。
**12月22日：[[蘇澳轉運站|蘇澳轉運站]]完工啟用
*2019年
**1月前：國光客運汰換16輛低地板大客車
**2月初前：宜蘭縣公車擬統一改段次收費，2月初前上路
