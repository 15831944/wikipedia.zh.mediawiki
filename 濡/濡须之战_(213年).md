'''濡须之战'''，又作'''濡须口之战'''，是發生於[[东汉|东汉]]建安十八年（213年）在[[濡须口|濡须口]]（今[[安徽|安徽]][[巢縣|巢縣]][[西巢湖|西巢湖]]）的一場戰役。此戰由[[曹操|曹操]]主動進擊，[[孫權|孫權]]領兵成功抵禦，逼使曹操退兵。

== 背景 ==

[[赤壁之戰|赤壁之戰]]後，曹操任命廬江人[[謝奇|謝奇]]為[[蘄春|蘄春]][[典農|典農]]，在[[皖縣|皖縣]]屯田，並屢次出兵侵範孫權轄境。時孫權所任的偏將軍、尋陽令[[呂蒙|呂蒙]]派人誘降謝奇，然遭謝奇拒絕，呂蒙隨後就找機會進攻謝奇。謝奇被逼退走後其部將孫子才及宋豪等帶著家眷老弱歸降呂蒙<REF>《三國志·呂蒙傳》：「魏使廬江謝奇為蘄春典農，屯皖田鄉，數為邊寇。蒙使人誘之，不從，則伺隙襲擊，奇遂縮退，其部伍孫子才、宋豪等，皆攜負老弱，詣蒙降。」</REF>。

作為回應，曹操遂於建安十七年（212年）十月聲稱領四十萬步兵出兵濡須口，孫權早已有預聞，遂聽取呂蒙所議在濡須夾水口建濡須塢，以此作好防禦準備<ref>《建康實錄·卷一》：「是歲，初作濡須塢於江西，以拒曹操。時操以步兵號四十萬，列營出濡須口，權以七萬當之。」</ref><REF>《三國志·呂蒙傳》：「後從權拒曹公於濡須，數進奇計，又勸權夾水口立塢，所以備御甚精。」又裴注引《吳錄》：權欲作塢，諸將皆曰：「上岸擊賊，洗足入船，何用塢為？」呂蒙曰：「兵有利鈍，戰無百勝，如有邂逅，敵步騎蹙人，不暇及水，其得入船乎？」權曰：「善。」遂作之。</ref><ref>《三國志·吳主傳》：「十六年，權徙治秣陵。明年，城石頭，改秣陵為建業。聞曹公將來侵，作濡須塢。」</ref><ref>《三國志·武帝紀》：「（建安十七年）冬十月，公征孫權。」</ref>。

== 战役 ==
建安十八年正月，曹軍兵進濡須口，並攻破孫權的江西營，俘獲都督公孫陽<ref>《三國志·武帝紀》：「十八年春正月，進軍濡須口，攻破權江西營，獲權都督公孫陽，乃引軍還。」</ref>，孫權遂領七萬兵往濡須抵禦曹軍。

孫權又派遣[[甘寧|甘寧]]領三千兵為前部督，命令他夜襲曹營，甘寧遂挑選出百多人隨已進攻，進營壘內殺了數十人，曹軍受驚鼓譟亮火照明，而甘寧已退還<ref>《三國志·甘寧傳》註引《江表傳》：「曹公出濡須，號步騎四十萬，臨江飲馬。權率衆七萬應之，使寧領三千人為前部督。權密勑寧，使夜入魏軍。寧乃選手下健兒百餘人，徑詣曹公營下，使拔鹿角，踰壘入營，斬得數十級。北軍驚駭鼓譟，舉火如星，寧已還入營，作鼓吹，稱萬歲。因夜見權，權喜曰：「足以驚駭老子否？聊以觀卿膽耳。」即賜絹千匹，刀百口。權曰：「孟德有張遼，孤有興霸，足相敵也。」停住月餘，北軍便退。</ref>。

兩軍及後相持，孫權屢屢挑戰之下曹操仍嚴軍不動。孫權於是乘輕船自濡須口向曹軍陣地前進，曹操知孫權要來看曹軍陣壘，於是勒令大軍嚴整待敵，箭矢不得亂發。孫權的船走了五六里後回頭響起鼓吹音樂，曹操見孫軍無論船隊還是上面的兵器及士兵都整齊嚴肅，感歎道：「生兒子就應當像孫仲謀那樣，[[劉表|劉景升]]的兒子就像豬狗那樣！」接著孫權就寫信給曹操稱「春天融雪的大水要來了，你應該快點離去。」另外一張紙就寫：「足下不死，我不得安寧。」曹操於是退軍<REF>《三國志·吳主傳》：「權乃自來，乘輕船，從濡須口入公軍。諸將皆以為是挑戰者，欲擊之。公曰：「此必孫權欲身見吾軍部伍也。」勑軍中皆精嚴，弓弩不得妄發。權行五六里，迴還作鼓吹。公見舟船器仗軍伍整肅，喟然歎曰：『生子當如孫仲謀，劉景升兒子若豚犬耳！』權為牋與曹公，說：『春水方生，公宜速去。』別紙言：『足下不死，孤不得安。』曹公語諸將曰：『孫權不欺孤。』乃徹軍還。」</REF>。

== 結果 ==
*此戰孫權力排眾議，聽從呂蒙建議修建濡須塢，增強了對曹操南攻的防禦能力，此塢遂成為軍事要地，並設立濡須督一職，[[蒋钦_(三国)|蔣欽]]、[[周泰|周泰]]及[[朱桓|朱桓]]等孫吳名將皆先後出任，此塢在隨後兩場濡須口之戰亦是重要據點。

== 相关典故 ==
=== 草船借箭 ===
據《[[魏略|魏略]]》，孫權乘船視察曹軍時曹操曾命令軍隊向孫權的船大發箭矢，孫權的船一舷受箭過多快要翻側，孫權於是下令船隻轉向回頭，讓船的另一舷受箭，待兩邊的箭量平衡後就回到駐點<REF>《三國志·吳主傳》裴注引《魏略》曰：權乘大船來觀軍，公使弓弩亂發，箭著其船，船偏重將覆，權因迴船，復以一面受箭，箭均船平，乃還。</REF>。但這與《三國志》所載不同。
