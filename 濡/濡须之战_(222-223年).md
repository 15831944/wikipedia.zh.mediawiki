'''濡须之战'''或'''濡须口之战'''于222-223年间发生于曹魏和东吴之间。这是曹孙第三次在[[濡须|濡须]]交战，但只有这次是发生在三国时期，前两场分别在213年和217年发生。

== 背景 ==
[[蜀汉|蜀汉]]于[[猇亭之战|猇亭之战]]败绩后，和孙氏重建关系，[[孙权|孙权]]与[[曹丕|曹丕]]联盟寻求保护变得失去意义。曹丕索要孙权长子世子[[孙登_(太子)|孙登]]为质，使双方关系急剧降温。曹丕试图破坏刘、孙外交。222年十月，孙权再次宣称独立于魏，曹丕因此用兵伐吴。

魏计划沿[[长江|长江]]三路攻吴。魏征南大将军[[夏侯尚|夏侯尚]]、上军大将军[[曹真|曹真]]攻[[江陵|江陵]]、[[南郡|南郡]]，征东大将军[[曹休|曹休]]、镇东将军[[臧霸|臧霸]]攻[[洞口县|洞口]]，大司马[[曹仁|曹仁]]攻濡须口。

== 战役 ==
曹仁扬言攻打羡溪，诱吴濡须督[[朱桓|朱桓]]分兵相救，再派大军径奔濡须，真正攻打羡溪的其实是散骑常侍[[蒋济|蒋济]]。朱桓还没追回派往羡溪的军队，曹仁已经杀到。朱桓手下只有五千人，诸将害怕，但朱桓却自认为用兵胜过曹仁，即使魏文帝曹丕亲来，他也不惧怕。朱桓偃旗息鼓，示弱，诱曹仁来攻。曹仁以为濡须城几乎无兵，派儿子[[曹泰|曹泰]]攻城，将军[[常雕|常雕]]、[[诸葛虔|诸葛虔]]、[[王双_(曹魏)|王双]]等乘油船攻打朱桓部曲妻子所在的中洲。蒋济认为不可，曹仁不听，亲率万人留在橐皋为曹泰等人后援。

朱桓派别将[[严圭|严圭]]等击常雕，亲自拒曹泰，曹泰中伏烧营而退。朱桓斩杀常雕，生擒王双，魏军溺死及被杀者千余人。<ref>《三国志》卷二文帝纪</ref><ref>《资治通鉴》卷七十</ref><ref>《三国志》诸夏侯曹列传</ref><ref>《三国志》朱桓传</ref><ref>《三国志》卷一十四蒋济传</ref>朱桓因而得以喘息，为追回派去救羡溪的军队赢得时间。曹仁亲自率军围城，朱桓重整军队守御。激战数月后，因魏军疾病和伤亡严重，曹丕令曹仁撤军。

== 后果 ==
战事于223年结束后，曹仁于三月去世。谥'''忠侯'''。
