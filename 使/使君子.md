{{medical}}
'''使君子'''（[[學名|學名]]：{{lang|la|''Combretum indicum''}}）<ref name = "浸會大學">[http://libproject.hkbu.edu.hk/was40/detail?record=1&channelid=1288&searchword=%28Quisqualis+indica%29&extension=all&sortfield=%2Bname_formal_latin_sort 藥用植物圖像數據庫]香港浸會大學中醫藥學院</ref>為[[使君子科|使君子科]][[使君子屬|使君子屬]]植物<ref name = "浸會大學"/>。本種為古今中外著名的驅蟲藥，於治療小兒病患上至少已有1600多年歷史<ref name = "當代藥用植物典4">{{cite book|title=《當代藥用植物典 4》|author= |series=蕭培根|publisher=香港賽馬會中藥研究院|page=第403-406頁|isbn=978-988-99226-3-4|date=2007年8月}}</ref>。

==名称==
本种別稱“留球子、留求子、史君子、史君根、吏君子、水君子、水君葉、四君子、四蜀使君子、病疳子、舀求子、君子仁、索子果、郭砸滿、杜蒺藜子、郎姆活、山羊屎、仰光藤”等<ref name = "自然標本館">[http://www.cfh.ac.cn/25646.sp Quisqualis indica]中國自然標本館</ref><ref name="史君根">[http://www.tcmlib.com/zy/html9/showdetail-3832342ce4bdbfe5909be5ad90e6a0b92c7a79.html 史君根]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}意古中醫</ref><ref name="水君葉">[http://www.tcmlib.com/zy/html9/showdetail-3832332ce4bdbfe5909be5ad90e58fb62c7a79.html 水君葉]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}意古中醫</ref><ref>[http://plant.tesri.gov.tw/plant/WebShowBPlant.aspx?tno=461003010&spkind=1 物種基本資料]台灣野生植物資料庫</ref>

== 分佈 ==
分佈於[[臺灣|臺灣]]、[[中國|中國]]、[[印度|印度]]、[[緬甸|緬甸]]至[[菲律賓|菲律賓]]等地，[[中國|中國]]國內分佈於[[重庆|重庆]]、[[四川|四川]]、[[廣東|廣東]]、[[廣西|廣西]]及[[福建|福建]]等省份。重庆市铜梁区水口镇是全国最大的使君子生产基地，早在清乾隆时期，该镇居民就开始种植使君子，距今已有200多年历史。在2014年，获得国家地理标识的称号。多生長於[[海拔|海拔]]1000－2500[[米|米]]的山坡、路旁、平地等向陽的[[灌木叢|灌木叢]]中<ref>[http://www.bhl-china.org/data/index.php?name=Quisqualis%20indica 橫斷山植物志上冊]第1240頁</ref><ref name="海南植物志 ">[http://www.cvh.org.cn/difangzhi/hainan/page/2/044.pdf 海南植物志]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}第44頁</ref>。

==分类==
本种最早的学名“{{lang|la|''Quisqualis indica''}}”由[[林奈|林奈]]所取，所在的使君子属（{{lang|la|''Quisqualis''}}）曾被视为与[[风车藤属|风车藤属]]（{{lang|la|''Combretum''}}）不同的属，直到1998年才将两属合并，改用现名。

过去也有学者将本种多毛的个体当作本种的[[变种|变种]]，称为“毛使君子”（{{lang|la|''Quisqualis indica'' var. ''villosa''}}），但现在已被鉴定为无效，只是本种的[[异名|异名]]。

== 形態特徵 ==
[[File:Quisqualis_indica_0003.jpg|thumb]]
使君子是一種[[攀緣|攀緣]]狀[[藤本|藤本]]植物，幼時呈[[灌木|灌木]]狀。幼嫩部份披[[棕黃色|棕黃色]]短柔毛。[[葉|葉]]橢圓形、長圓形至卵狀披針形，對生或近對生，薄紙質，先端漸尖或有時尾尖，基部近圓形，表面近無毛，背面披棕黃色柔毛，全緣或稍呈淺波狀，側脈每邊7－8條，長約5－13厘米，寬約2.5－6厘米；[[葉柄|葉柄]]披毛，無關節，幼時密披棕黃色柔毛，長約1厘米，落葉後宿存的葉柄基部逐漸硬化成剌狀體。
花為近傘房狀的[[穗狀花序|穗狀花序]]，頂生，有花10餘朵，兩性；苞片卵形至線狀披針形，披毛，早落，長約5－8毫米；[[花萼|花萼]]綠色，萼裂片三角形，頂端有5枚小齒，披毛；[[萼管|萼管]]細長，披疏黃色短柔毛，長約5－9厘米；[[花瓣|花瓣]]5片，長圓形或倒卵狀長圓形，頂端鈍圓，初開時為白色後轉變為紅色，長約1.2－2.4厘米，寬約2－10毫米；[[雄蕊|雄蕊]]10枚，[[花絲|花絲]]呈2輪高低排列，上輪著生於柱頭上方，伸出冠外，下輪生於柱頭下方，不伸出冠外；[[子房|子房]]下位，呈紡錘形，披毛，5棱，1室，胚珠3顆；[[花柱|花柱]]柔弱，基部與萼管貼合，上部分離，略伸出冠外；[[柱頭|柱頭]]微擴大，兩側稍扁平。果為[[核果|核果]]，青黑或褐色，卵形至紡錘形，頂端狹尖，基部鈍圓，有明顯的圓形果梗痕，具5條縱棱，偶有4－9棱，無毛，革質，成熟後頂部開裂，開裂為3－5瓣，長約2.5－4厘米，寬約1.2－3.3厘米。[[種子|種子]]1顆，圓柱狀紡錘形，白色，有多數縱皺紋，長約2.5厘米，寬約10毫米。

== 醫藥用途 ==
使君子乾燥成熟果實入藥，味甘，性温，無毒，歸脾、胃經，藥材主產於中國福建、江西、廣東、廣西、湖南、雲南、重庆、貴州等省地，[[中醫|中醫]]歸類為[[驅蟲藥|驅蟲藥]]、[[健脾藥|健脾藥]]。[[中藥|中藥]]名為使君子，藥用之名始載《[[開寶本草|開寶本草]]》，於能殺蟲消積、健脾，具驅蟲、抗皮膚真菌等功效，主治蟲積腹痛、腹脹、瀉痢、[[小兒疳積|小兒疳積]]、乳食停滯等，現代臨床應用於驅[[蛔蟲|蛔蟲]]、[[蟯蟲|蟯蟲]]、[[絛蟲|絛蟲]]及外用於各類皮膚真菌感染、[[陰道毛滴蟲|陰道毛滴蟲]]等治療上<ref name = "浸會大學"/>。

使君子的葉入藥，味辛，性平，歸脾、胃經，中醫歸類為驅蟲藥。中藥名為使君子葉，具殺蟲解毒、理氣健脾等功效，主治小兒疽積、兒積、脘腹脹滿、瘡癤潰瘍等治療上<ref name = "水君葉"/>。

使君子的根入藥，味辛、苦，性平，歸脾、肺經，中醫歸類為健脾藥、宣肺藥。中藥名為使君子根，具降逆止咳、殺蟲健脾等功效，主治咳嗽、[[痢疾|痢疾]]、蟲積等治療上<ref name = "史君根"/>。

== 藥材鑑定 ==
本品乾燥成熟果實，於秋季果皮變為紫黑色時採收，除去雜質後曬乾或烘乾；表面黑褐色至紫黑色，平滑具光澤，橢圓形或卵圓形，橫切面多為五角星形，棱角處果殼較厚，中間呈類圓形空腔，質地堅硬，[[種皮|種皮]]薄易脫落；[[子葉|子葉]]黃白色，2枚，斷面有裂紋，有油性；氣微香，味微甜。本品種載錄於《[[中國藥典|中國藥典]]》2005年版，定為中藥使君子的法定原植物來源種，主要透過性狀為指標，以控制藥材的質量。傳統經驗則認為以個大、色紫黑、具光澤、仁飽滿、色黃白者為佳<ref>{{cite book|title=《常用中藥材鑑別圖典》|author= |series=趙中振、陳虎彪|publisher=萬里機構|page=第336頁|isbn= 978-962-14-4150-8|date=2010年1月}}</ref>。

== 化學成份 ==
種子含[[氨基酸|氨基酸]]及其鹽類、[[脂肪酸|脂肪酸]]類、[[三萜類|三萜類]]及[[固醇類|固醇類]]成份；果肉含脂肪類及[[可水解鞣質類|可水解鞣質類]]成份；花和葉中含[[黃酮類|黃酮類]]成份；種仁含脂肪酸，當中以[[不飽和脂肪酸|不飽和脂肪酸]]為主，亦含[[飽和脂肪酸|飽和脂肪酸]]。當中的[[使君子氨酸|使君子氨酸]]、[[使君子氨酸鉀|使君子氨酸鉀]]為驅蟲活性成份<ref name = "浸會大學"/>。
