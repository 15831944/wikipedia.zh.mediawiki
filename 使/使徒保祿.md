{{作品名稱|name=''Paul, Apostle of Christ''|hk=使徒保祿|tw=使徒保羅|equiv-cn=tw}}是一部2018年的美國聖經電影，由Andrew Hyatt執導和編劇。主演[[詹姆斯·福克纳|詹姆斯·福克纳]]飾演[[保羅_(使徒)|保羅]]，[[占·卡維素|占·卡維素]]飾演[[路加|路加]]（占·卡維素曾經在2004年電影《[[受難曲_(電影)|受難曲]]》中飾演[[耶穌|耶穌]]）。

主要拍攝於2017年9月在[[馬爾他|馬爾他]]開始。本片於2018年3月23日由[[新力影業|新力影業]]負責上映。<ref>{{Cite web |url = https://www.comingsoon.net/movies/news/922643-sony-release-dates-for-the-equalizer-2-paul-apostle-of-christ |title = Sony Release Dates for The Equalizer 2, Paul, Apostle of Christ |author = Jenna Busch |date = 2018-02-13 |publisher = ComingSoon.net |language = en}}</ref>
