{{vgname|使命召唤Online|en=Call of Duty Online}}是一款由[[美国动视|美国动视]]旗下的[[Raven|Raven]]工作室制作、[[腾讯公司|腾讯公司]]独家代理的一款[[第一人称射击|第一人称射击]]的[[网络游戏|网络游戏]]<ref name="Background Information">{{Cite web |url=http://codol.qq.com/act/a20130427newhand/index.htm |title=游戏背景 |accessdate=2013-05-09 |work=使命召唤Online官方网站 |publisher=腾讯公司 |date= }}</ref>。之前的测试由上海动视制作，由于动视对中国这一庞大市场保持期望，改由动视王牌工作室raven接手，腾讯公司代理。
本游戏目前只在中国大陆开放。目前没有相关消息说明使命召唤ol将会走向全球。

== 游戏背景 ==
《使命召唤Online》英雄行动以《[[使命召唤|使命召唤]]》的《现代战争》系列（Modern Warfare）为故事背景，采用《现代战争》系列的世界观，讲述的是《现代战争》1—2代之间发生的故事<ref name="Background Information" />。但随着游戏版本的更新以及开发组的更换导致剧本更改，除部分人物外已与《现代战争》系列无多大联系，已作为一个平行世界。

== 阵营 ==
《使命召唤Online》中的两大PVP阵营分别是141特遣队以及暗影军团。此外还有剧情的第三方阵营裂变委员会。
