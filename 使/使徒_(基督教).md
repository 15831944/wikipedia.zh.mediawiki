'''使徒'''（{{lang-grc|'''ἀπόστολος'''}}；{{lang-en|'''apostle'''}}；[[天主教會|羅馬天主教]]譯為'''宗徒'''），基督教名詞，是為某個特殊[[大使命|使命]]奉召被差派出去的[[門徒|門徒]]。原意是受差遣者、信差、使者。在基督教教義中，它可以指一種職份，也可以指一種屬靈恩賜。

在[[對觀福音書|對觀福音書]]中，耶穌選定的十二[[门徒_(基督教)|門徒]]，也稱十二使徒，是最早被稱為使徒的人。在十二使徒之外，則是自稱為外邦人的使徒的[[保罗_(使徒)|保羅]]。使徒在早期基督教會中的地位不明，在保羅書信中，經常以使徒來稱呼教會中的人，如[[巴拿巴|巴拿巴]]、[[以巴弗提|以巴弗提]]（Epaphroditus）。在《[[約翰福音|約翰福音]]》中，使徒與門徒沒有什麼不同，沒有特殊地位與描寫。因此這個稱呼可能可以被使用在教會中的任何成員身上，不限於十二使徒。但在《[[啟示錄|啟示錄]]》中，使徒特指耶穌的十二門徒，是教會的根基。

在西元一世紀，由耶穌直接門徒與再傳門徒所形成的早期基督教會，成為基督教歷史的開端。這段時期稱為[[使徒時代|使徒時代]]（Apostolic Age），這些使徒的門徒被稱為[[使徒教父|使徒教父]]，他們形成的教會被稱為[[使徒教會|使徒教會]]（Apostolic see）。

== 字源 ==
使徒的字根，源自[[通用希臘語|通用希臘語]]的{{lang|grc|ἀπόστολος}}（apóstolos），原意是被派遣出去的人。它可以被拆解成，από（apo，從）與στολος（stólos，意思是船艦），意思是出外航行的人。也可以被拆解為 από（apo，遠離、從）與στέλλω（stello，送出），它可以被翻譯為拉丁文ex（從）與 mitto（送出），由此組成拉丁文單字emissary，意思是由此送出。也可以翻譯為拉丁文單字missio，它的意思為某個肩負任務的人。英文的任務（mission）、傳教士（missionary）與信差（messenger），就是由此衍生出來的。

在猶太人中，有個官職，被稱為{{lang-he|שליח}}（apostle），也可能是使徒的由來之一。

==歷史==
在[[新約|新約]][[對觀福音書|對觀福音書]]中，使徒最早指[[耶穌|耶穌]]由[[门徒_(基督教)|門徒]]中，直接挑選出的十二個向外傳教的人，也被稱為十二使徒。這十二位使徒，要負責向外傳教，並擁有醫治病人與趕鬼的能力<ref>{{bible|马可福音|3:14-15}}：「他就設立十二個人，要他們常和自己同在，也要差他們去傳道，並給他們權柄趕鬼」</ref><ref>{{bible|马太福音|10:7-8}}：「隨走隨傳，說，天國近了。醫治病人，叫死人復活，叫長大痲瘋的潔淨，把鬼趕出去。」</ref>。《[[約翰福音|約翰福音]]》中也記載了十二使徒，但是不認為他們跟其他門徒有什麼區別。

在耶穌過世後，十二門徒中的[[加略人猶大|加略人猶大]]，被視為背叛者，而自殺。其餘11位門徒繼續領導信眾<ref>{{bible|使徒行传|1:13-14}}：「進了城，就上了所住的一間樓房。在那裡有彼得，約翰，雅各，安得烈，腓力，多馬，巴多羅買，馬太，亞勒腓的兒子雅各，奮銳黨的西門，和雅各的兒子﹝或作兄弟﹞猶大。這些人，同著幾個婦人，和耶穌的母親馬利亞，並耶穌的弟兄，都同心合意的恆切禱告。」</ref>，信徒經選舉與抽籤，選出[[馬提亞|馬提亞]]來取代[[加略人猶大|加略人猶大]]的席位，重新補足了十二門徒的位置<ref>{{bible|使徒行传|1:23-26}}：「於是選舉兩個人，就是那叫作巴撒巴，又稱呼猶士都的約瑟，和馬提亞。眾人就禱告說，主阿，你知道萬人的心，求你從這兩個人中，指明你所揀選的是誰，叫他得這使徒的位分‧這位分猶大已經丟棄，往自己的地方去了。於是眾人為他們搖籤，搖出馬提亞來。他就和十一個使徒同列。」</ref>。

從一些古代記載看來，在耶穌死後，使徒之中，最重要的人物，就是被稱為教會柱石的雅各、彼得、約翰，其中又以[[主的兄弟雅各|主的兄弟雅各]]地位最高<ref>[[革利免|革利免]]''Institutions''第七冊：「主於復活後，將知識的恩賜分授義者雅各、約翰和彼得，藉由他們傳授於其他使徒，使徒們傳於七十門徒，巴拿巴是七十人之一。」</ref>。

在[[保罗_(使徒)|保羅]]加入教會之後，[[耶路撒冷|耶路撒冷]]教會的領導者[[彼得_(使徒)|彼得]]與[[公義者雅各|雅各]]同意他向外邦人傳教<ref>{{bible|加拉太書|2:9}}：「又知道所賜給我的恩典、那稱為教會柱石的雅各、磯法、約翰、就向我和巴拿巴用右手行相交之禮、叫我們往外邦人那裏去、他們往受割禮的人那裏去。」</ref>，保羅自稱為耶穌基督的使徒<ref>{{bible|提多書|1:1}}：「神的僕人，耶穌基督的使徒，保羅」</ref><ref>《提摩太前書》2:7：「我為此奉派，作傳道的，作使徒，作外邦人的師傅，教導他們相信，學習真道‧我說的是真話，並不是謊言。」</ref>，也自稱為外邦人的使徒<ref>{{bible|罗马书|11:13}}：「我對你們外邦人說這話‧因我是外邦人的使徒，所以敬重我的職分。」</ref>。保羅認為使徒的地位，是[[耶穌|耶穌]][[基督|基督]]直接經由[[聖靈|聖靈]]授與，不是經由教會選舉，或是原有使徒的認定，這也是他使徒職位的來源<ref name="羅1:5">{{bible|罗马书|1:5}}：「我們從他受了[[恩典|恩惠]]並使徒的職份，在萬國之中叫人為他的名信服真道。」</ref>。根據保羅的記載，使徒被認為擁有授與[[聖靈|聖靈]]的能力<ref>《使徒行傳》8:14-17：「使徒在耶路撒冷，聽見撒瑪利亞人領受了 神的道，就打發彼得、約翰往他們那裡去。兩個人到了，就為他們禱告，要叫他們受聖靈因為聖靈還沒有降在他們一個人身上‧他們只奉主耶穌的名受了洗。於是使徒按手在他們頭上，他們就受了聖靈。」</ref>，是教會的管理者，位階高於一般信眾<ref>{{bible|哥林多前书|12:28}}：「上帝在教會所設立的：第一是使徒，第二是先知，第三是教師，其次是行異能的，再次是得恩賜醫病的，幫助人的，治理事的，說方言的。」</ref><ref>{{bible|以弗所书|4:11}}：「他所賜的，有使徒，有先知，有傳福音的，有牧師和教師。」</ref>

=== 啟示錄 ===
在《[[啟示錄|啟示錄]]》中，使徒特指耶穌的十二門徒，可能代表了以色列十二支派，象徵耶路撒冷聖殿的十二道門，在教會中有特殊地位<ref>《啟示錄》21:12-14：「有高大的牆‧有十二個門，門上有十二位天使。門上又寫著以色列十二個支派的名字。東邊有三門，北邊有三門，南邊有三門，西邊有三門。城牆有十二根基，根基上有羔羊十二使徒的名字」</ref>。

== 使徒保羅 ==
{{Main|保罗 (使徒)}}

在其致各地教會的書信中，[[保罗_(使徒)|保羅]]稱呼自己是一位使徒，或“[[外邦人|外邦人]]的使徒”<ref>{{Bible|罗马书|1|1}}：「耶穌基督的僕人保羅，奉召為使徒，特派傳神的福音。」</ref><ref>{{Bible|罗马书|11|13}}：「我對你們外邦人說這話，因我是外邦人的使徒，所以敬重我的職分。」</ref>。但是[[保罗_(使徒)|保羅]]並不是耶穌在世時的弟子，也非由當時教會選出。他多次強調，他是由上帝與耶穌基督直接選定的<ref>：「作使徒的保羅，（不是由於人，也不是藉著人，乃是藉著耶穌基督，與叫他從死裡復活的父神）。」</ref><ref>{{Bible|加拉太书|1|11、12}}：「弟兄們，我告訴你們，我素來所傳的福音，不是出於人的意思。因為我不是從人領受的，也不是人教導我的，乃是從耶穌基督啟示來的。」</ref><ref>{{Bible|加拉太书|2|7、8}}：「反倒看見了主託我傳福音給那未受割禮的人，正如託彼得傳福音給那受割禮的人。（那感動彼得，叫他為受割禮之人作使徒的，也感動我，叫我為外邦人作使徒。）」</ref>，位階不在十二使徒之下<ref>{{Bible|哥林多後書|11|5}}：「但我想，我一點不在那些最大的使徒以下。」</ref>。

据《[[使徒行傳|使徒行傳]]》的記載，保羅與巴拿巴可能是在安提阿教會得到聖靈的啟示，由先知將他們按立為使徒<ref>{{Bible|使徒行传|13|1-3}}：「在安提阿的教會中，有幾位先知和教師，就是巴拿巴，和稱呼尼結的西面、古利奈人路求，與分封之王希律同養的馬念，並掃羅。他們事奉主，禁食的時候，聖靈說，要為我分派巴拿巴和掃羅，去作我召他們所作的工。於是禁食禱告，按手在他們頭上，就打發他們去了。」</ref><ref name="使14">{{Bible|使徒行传|14|14}}：「巴拿巴、保羅，二使徒聽見，就撕開衣裳，跳進眾人中間。」</ref>。

== 其他被稱為使徒的聖經人物 ==
在《使徒行傳》中，稱[[巴拿巴|巴拿巴]]為使徒<ref name="使14" />。後世的東方正教會傳統中，保羅的門徒[[提摩太|提摩太]]和[[西拉|西拉]]也被稱為使徒。

保羅在《羅馬人書》中，提到他的親戚[[安多尼古|安多尼古]]和[[猶尼亞安|猶尼亞安]]，稱呼他們「在使徒中是有名望的」<ref>{{Bible|罗马人书|16|7}}：「又問我親屬與我一同坐監的安多尼古和猶尼亞安。他們在使徒中是有名望的，也是比我先在基督裡。」</ref>。這段記載在解釋上有點疑義，因為可以把這段解釋為「使徒們都知道安多尼古和猶尼亞安的名聲」，或是解釋為「安多尼古和猶尼亞安，是有名望的使徒」。

保羅的門徒以巴弗提，也被稱為使徒<ref>{{Bible|腓立比書|2|25}}：「然而我想必須打發以巴弗提到你們那裡去，他是我的兄弟，與我一同作工，一同當兵，是你們所差遣的，也是供給我需用的。」</ref><ref>{{Bible|腓立比書|2|25}}：「再者，我以為必須把我的弟兄、同事和戰友，即你們派來為供給我的急需的使者，厄帕洛狄托給你們打發回去。」</ref>。雖然一般都被譯為你們的使者，或信差，但在希臘原文中，用你們的使徒（{{lang-el|πόστολος}}）來稱呼他。

因為保羅認為使徒地位，是直接由耶穌經由聖靈授與，在教會中擁有特殊地位<ref name="羅1:5" /><ref>{{Bible|哥林多前書|12|28}}：「上帝在教會所設立的：第一是使徒，第二是先知，第三是教師，其次是行異能的，再次是得恩賜醫病的，幫助人的，治理事的，說方言的。」</ref><ref>《{{Bible|以弗所書|4|11}}：「他所賜的，有使徒，有先知，有傳福音的，有牧師和教師。」</ref>。因此，巴拿巴、[[提摩太|提摩太]]、[[西拉|西拉]]、安多尼古與猶尼亞安，可能也是經由聖靈啟示而在教會得到使徒地位，而不是遵循原有十二門徒傳統按立的。

== 紀念物 ==
位於澳大利亞維多利亞州的[[十二門徒石|十二門徒石]]的名稱取材自耶穌的十二門徒。
