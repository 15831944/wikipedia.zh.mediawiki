'''筑北停车区'''（平假名：ちくほくパーキングエリア）是位於[[長野縣|長野縣]][[東筑摩郡|東筑摩郡]][[麻績村|麻績村]]和[[東筑摩郡|東筑摩郡]][[筑北村|筑北村]]的[[長野自動車道|長野自動車道]]之[[休息區|休息區]]。由[[東日本高速公路|東日本高速公路]]管理。

== 連接道路 ==
* [[長野自動車道|長野自動車道]]

== 历史 ==
