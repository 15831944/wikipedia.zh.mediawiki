==历史==
抱朴道院位于葛岭半山腰，又称葛仙庵、抱朴庐<ref name=xhw/>。葛岭地处[[杭州西湖|杭州西湖]]北岸宝石山中，因相传[[东晋|东晋]]道士[[葛洪|葛洪]]（自号“抱朴子”）曾在此炼丹而得名。传说葛洪在这里修道期间，经常为当地民众采药治病、开通山路。[[唐朝|唐朝]]时为纪念葛洪而建“葛仙祠”。南宋时此地建为御花园之一的集芳园，后为丞相[[贾似道|贾似道]]所有，并在院中新建红梅阁、半间草堂作为私家别墅。[[元朝|元朝]]毁于战火，[[明朝|明朝]]重建并改称“玛瑙山居”，[[清朝|清朝]]修葺并改称“抱朴道院”，此名沿用至今<ref name=zhwqn>{{Cite web |url = http://www.zh5000.com/ZHJD/zgdj/2006/djsd/zgdj-0017.htm |title =  杭州市抱朴道院|author =  |date =  |publisher = 中华五千年网 |language =  |accessdate = 2017-04-27 |quote =  }}</ref>。自[[明朝|明]][[清朝|清]]时期至今历经多次重建和重修。过去，葛岭抱朴院与黄龙洞、玉皇山合称西湖三大道院<ref name=xhw/>。[[文革|文革]]期间，抱朴道院被园林部门占用。1982年由道教界收回管理<ref name=zhwqn/>。1983年，抱朴道院被定为[[道教全国重点宫观|道教全国重点宫观]]<ref name=xhw/><ref name=rmw/>。

2001年6月起，杭州市道教协会主持修复抱朴道院的殿堂及神像。2003年1月11日，抱朴道院修复竣工，举办了神像开光仪式，正式重新对外开放。抱朴道院现为杭州道教协会所在<ref name=xhw/><ref name=rmw>{{Cite web |url = http://www.people.com.cn/GB/shenghuo/80/105/20030111/905937.html |title =杭州葛岭抱朴道院重修开放  |author =  |date = 2003-01-11 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。2011年西湖成功申遗，抱朴道院与道教文化列入西湖文化景观[[世界遗产名录|世界遗产名录]]<ref name=wy/>。

1992年6月，杭州抱朴道院音乐团（后改为法务团）成立，以青年乾、坤道为主。当时抱朴道院的道士们将[[工尺谱|工尺谱]]译为[[简谱|简谱]]、韵谱后学习。1995年后又请徐宏图、曹本治教授记谱，整理出一套乐谱，收录在图书《杭州抱朴道院道教音》中。1992年11月，以高信一为团长的杭州抱朴道院法务团以杭州道教音乐团的名义赴[[深圳|深圳]]参加香密湖中华大庙会，这次活动有21家单位参加，杭州道教音乐团是唯一代表道教的乐团，受到[[全国政协副主席|全国政协副主席]][[程思远|程思远]]接见，程思远还现场亲题“弘扬华夏民族文化”。1993年6月，杭州抱朴道院法务团应[[中国道教协会|中国道教协会]]邀请，到[[北京|北京]]参加中国道教协会举办的两岸四地“罗天大醮”。2004年，杭州抱朴道院法务团应邀参加[[新加坡|新加坡]]第四届道教音乐节，获[[新加坡总统|新加坡总统]][[塞拉潘·纳丹|纳丹]]接见。2014年8月，杭州市第五批非物质文化遗产申遗成功，其中包括杭州道教音乐<ref name=wy/>。

==建筑==
抱朴道院面朝杭州西湖，背依葛岭。抱朴道院外的主要建筑有<ref name=xhw/><ref name=rmw/>：
*穹门：位于葛岭山麓，赭黄色，砖石结构。中央开一拱形门洞，门洞正上方嵌有“葛岭”石匾。门洞两侧的墙上分别有“抱朴”、“道院”四字，并有两副对联<ref name=xhw/><ref name=rmw/>。
*阶梯：进入穹门拾阶而上，经过流丹阁，可到葛岭山腰的四角方亭，这段路上有古柏、清泉，石壁上题刻有“人间福地”、“不亚蓬瀛”等。再自涤心池拾阶而上，便可抵达抱朴道院<ref name=xhw/><ref name=rmw/>。

如今的抱朴道院，四周围墙为黄色，墙顶覆青瓦。抱朴道院的主要建筑有<ref name=xhw/><ref name=rmw/>：
*葛仙殿：为正殿，坐西北朝东南，重檐[[歇山顶|歇山顶]]，木结构楼阁式。殿内正中供葛洪像，两侧是纯阳祖师像、朱大天君像<ref name=xhw/><ref name=rmw/><ref name=zhwqn/>。
*半闲草堂：位于葛仙殿东北侧，重檐歇山顶，木结构楼阁式<ref name=xhw/><ref name=rmw/><ref name=zhwqn/>。
*红梅阁：位于葛仙殿东南侧，重檐歇山顶，木结构楼阁式。红梅阁内有木刻画廊，其中包括戏曲《[[李慧娘|李慧娘]]》的故事<ref name=xhw/><ref name=rmw/><ref name=zhwqn/>。
*抱朴庐：位于葛仙殿东南侧，重檐歇山顶，木结构楼阁式<ref name=xhw/><ref name=rmw/><ref name=zhwqn/>。
*抱朴道院遗址：2010年4月至6月，杭州市文物考古所开展了对抱朴道院遗址的考古调查，对葛仙殿西北侧葛岭西南坡上、下层台地进行清理，发现了晚清至[[中华民国|中华民国]]初期的道教建筑遗迹。
*炼丹台<ref name=xhw/><ref name=rmw/><ref name=zhwqn/>
*炼丹井<ref name=xhw/><ref name=rmw/><ref name=zhwqn/>
*葛仙庵碑：立于明朝，记述葛洪生平以及来这里结庐炼丹的过程<ref name=xhw/><ref name=rmw/><ref name=zhwqn/>。
*初阳台：位于葛岭顶端，可观日出，是钱塘十景之一<ref name=zhwqn/>。
*双钱泉：位于院中，水质甜美<ref name=zhwqn/>。
