'''婆猪江战斗'''是1433年[[朝鲜王朝|朝鲜王朝]]攻击[[明朝|明]][[建州女真|建州女真]]的一次战斗。
== 背景 ==
1388年，[[朱元璋|朱元璋]]建立的[[明朝|明朝]]军队攻入[[中国东北地区|中国东北地区]]，准备模仿[[元朝|元朝]]在[[朝鲜半岛|朝鲜半岛]]设立[[双城总管府|双城总管府]]的模式，设立[[铁岭卫|铁岭卫]]，[[高丽|高丽]]抗拒。朝鲜王朝取代高丽后，称公嶮鎭迤南全部属于朝鲜。<ref>朝鲜王朝实录 太宗 7卷, 4年(1404 甲申 /永樂 2年) 5月 19日(己未)</ref>1403年，[[朱棣|朱棣]]将[[咸兴|咸兴]]以北、公崄镇以南划归朝鲜。但是当地多是女真人，拒绝接受朝鲜统治。[[建州卫|建州卫]]女真迁到婆猪江畔（[[鸭绿江|鸭绿江]]支流）的初期，因物资缺乏常常到朝鲜边境乞粮；同时要求“今后但有建州卫人民来往买卖，印信文书，许令施行，毋得阻挡”，<ref>《朝鲜实录》，世宗七年七月己已</ref>；有的女真人趁机抢劫。<ref>《朝鲜实录》，世宗六年十一月甲申</ref>朝鲜极为不安，[[朝鲜世宗|朝鲜世宗]]对明朝使臣讲：“未满彼心，倘若生变，其祸不测，应之之方，予甚虑焉。”<ref>《朝鲜实录》，世宗七年三月甲戌</ref>朝鲜一方面给予少量食物，另一方面“边将固守，不许人境”；<ref>《朝鲜实录》， 世宗六年七月乙亥</ref> “江边居民，督令人堡，清野以备”；<ref>《朝鲜实录》世宗六年十一月甲申</ref>并派[[间谍|间谍]]“潜谍婆猪江等处”，<ref>《朝鲜实录》世宗六年四月壬申</ref>[[建州女真|建州女真]]常掳朝鲜人和[[辽东|辽东]]汉人为[[奴隶|奴隶]]，这些人逃到朝鲜后，朝鲜便将他们留在国内或送归辽东。

世宗七年(明[[洪熙|洪熙]]元年，1425年)七月起，女真多次要求朝鲜“送回”被掳到女真的逃回朝鲜人，<ref>《朝鲜实录》．世宗七年七月丙申</ref><ref>《朝鲜实录》，世宗七年闫七月己未</ref>并发出战争威胁，<ref>《朝鲜实录》，世宗七年九月己酉</ref>朝鲜则将原来五、六人看守的各边界口子改为一名千户率领百名军人看守。<ref>《朝鲜实录》，世宗八年二月乙酉</ref>
    
朝鲜世宗八年（明[[宣德|宣德]]元年，1426年）又有类似事件。<ref>《朝鲜实录》，世宗八年七月壬辰</ref>世宗十年，[[李满住|李满住]]奴婢十口逃往朝鲜，并已属朝鲜札宾寺使唤，“意颇愤怨”<ref>《朝鲜实录》， 世宗十年二月丁丑</ref>世宗十四年建州指挥林加罗等因奴婢逃回朝鲜，到朝鲜抢掠人口。<ref>《朝鲜实录》，世宗十四年八月甲寅</ref>

世宗十四年(明宣德七年，1432年)[[农历|农历]][[十二月|十二月]]甲午，忽刺温人剽掠闻延，杀害军民53人，掳掠男妇77人。<ref>《朝鲜实录》，世宗十五年四月乙酉</ref>建州卫中途从忽刺温人手中夺回64人送归朝鲜，但是朝鲜政府大怒，认为建州卫女真与忽刺温人合谋。<ref>《朝鲜实录》，世宗十五年正月癸亥</ref>决定出兵打击建州卫女真。
== 战争准备 ==
朝鲜以酬谢建州卫送回被掳人为由，遣前少尹朴好问、护军朴原茂，访问女真，实为探察。<ref>《朝鲜实录》，世宗十五年二月甲午</ref>二月甲辰，世宗确定了“其举兵，则不可孤弱，当大举而讨之” 的作战方针。次日决定出兵3000。根据朴好问的报告，朝鲜增加了军用物资，出兵人数增加为万人以上。<ref>《朝鲜实录》，世宗十五年二月壬子</ref>三月庚申，根据朴好问的报告和[[平安道|平安道]]都节制使崔润德的要求，世宗决定出兵1.3万。<ref>《朝鲜实录》，世宗十二年三月庚申</ref>七天后，最终决定马兵1万，步兵5千。
    
朝鲜在内部作战前动员并对女真写“征婆猪江声罪榜”，对明朝通报上奏本作到出师有名。并对可能作为建州卫外援的建州左卫作好应变的各种部署。<ref>《朝鲜实录》，世宗十五年三月戊寅</ref><ref>《朝鲜实录》，世宗十五年四月戊子</ref>
== 战争 ==
建州卫无防备，遭沉重打击，被杀260人，被俘248人，被夺马62匹、牛18头。<ref>《朝鲜实录》，世宗十五年五月己未、壬申</ref>李满住的妻子“死于锋刃”，李满住本人则“身被九创”。<ref>《燕山君日记》二年十一月甲辰</ref>

俘获的248人中有“唐人”24名，遣返辽东。<ref>《朝鲜实录》，世宗十五年五月乙丑</ref>朝鲜人6名回国。<ref>《朝鲜实录》，世宗十五年八月乙亥</ref>世宗下令对女真“除老幼外， 丁壮并令斩之”。<ref>《朝鲜实录》，世宗十五年五月庚申</ref>余下的174名中，“七八岁以下无母无亲族小童等，付京中各司奴婢有产业慈惠者”收养，其他人则“分置京畿及忠清道各官， 限其安业”。<ref>《朝鲜实录》，世宗十五年六月癸未、乙酉</ref>

朝鲜军4人[[死亡|死亡]]，5人受伤。
== 事后 ==
事后仅一个多月，就发生了女真人报复事件。<ref>《朝鲜实录》，世宗十五年六月庚寅</ref>此后，朝鲜令边境13岁以上农民男丁习射，并武装种田。<ref>《朝鲜实录》，世宗十五年六月壬辰、己亥</ref>命平安道各官守令，轮番赴防，又从京城调来军士守边<ref>《朝鲜实录》，世宗十五年六月癸卯</ref>，朝鲜还在东北边境增加兵力，大筑城堡，人心惶惶，庆源筑城之军大规模逃亡。<ref>《朝鲜实录》，世宗十五年九月壬辰</ref>
   
建州女真怕朝鲜再次进攻，向北迁移到兀弥府，明[[正统|正统]]元年(朝鲜世宗十八年，1436年)，向明朝请求西迁辽阳、草河一带。次年九月，朝鲜军分三路进攻兀弥府，女真人事先逃跑，朝鲜军无功而返。正统三年（1438年）春，明[[礼部|礼部]]、[[兵部|兵部]]令李满住率建州卫移居浑河流域。<ref>《明英宗实录》卷43</ref>

[[建州左卫|建州左卫]]300余户女真人在正统五年(朝鲜世宗二十二年，1440年)六月，冲破朝鲜阻挠，迁到苏子河与李满住部汇合。[[毛怜卫|毛怜卫]]部分女真人此前也到此会合。以此为根据地和起点，完成了女真各部的统一。为以后的崛起创造条件。

朝鲜在鸭绿江东岸设置“[[西北四郡|西北四郡]]”，在[[图们江|图们江]]南岸设置“[[东北六镇|东北六镇]]”，控制了鸭绿江以东图们江以南的全部土地。

== 评论 ==
朝鲜世宗：“守成之君，大抵不好游畋声色，则必好大喜功 自古及今，[[继体|继体]]之主当所戒也，予承祖宗之业，抚盈成之运，常以此为念。往者婆猪江之役，大臣将相皆日不可、此乃万世不易之正论。予乃命征成功，此特其幸耳，不足尚也。”<ref>《朝鲜实录》，世宗十五年十一月戊戌</ref>但4年后又命令军队讨女真。

明辽东总兵官[[巫凯|巫凯]]等将帅，反对朝鲜越界击女真，[[明宣宗|明宣宗]]则认为朝鲜和女真是“远夷争竞”。
