'''婆罗洲'''（{{lang-ms|Borneo}}），[[印尼|印尼]]稱之為'''[[加里曼丹|加里曼丹]]岛'''（{{lang-id|Kalimantan}}）是[[世界|世界]]第三大[[岛|岛]]，亞洲第一大島，[[面积|面积]]743,330[[平方公里|平方公里]]，僅次於[[格陵蘭|格陵蘭]]及[[新幾內亞|新幾內亞]]。現全境由[[印尼|印尼]]、[[馬來西亞|馬來西亞]]及[[汶萊|汶萊]]三國管轄，而[[菲律賓|菲律賓]]曾宣稱有[[沙巴|沙巴]]部分的主權。

== 地理 ==
属[[大陆岛|大陆岛]]，位于[[东南亚|东南亚]]，处于岛屿东南亚的[[中心|中心]]部分，[[赤道|赤道]]將婆羅洲大致均分。

=== 四面環海 ===
* 东北部是[[苏禄海|苏禄海]]
* [[东|东]]部[[西里伯斯海|西里伯斯海]]及[[望加锡海峡|望加锡海峡]]
* [[南|南]]部是[[爪哇海|爪哇海]]及[[卡里马塔海峡|卡里马塔海峡]]
* 西、北部則為[[南中国海|南中国海]]

=== 临近岛屿 ===
* 东北部为[[苏禄群岛|苏禄群岛]]、[[巴拉望岛|巴拉望岛]]
* 东部是[[苏拉威西岛|苏拉威西岛]]
* 南部是[[爪哇岛|爪哇岛]]
* 西部是[[苏门答腊岛|苏门答腊岛]]、[[马来半岛|马来半岛]]
* 西北部是[[纳土纳群岛|纳土纳群岛]]
* 北部为[[南沙群岛|南沙群岛]]

=== 河流 ===
自古以來，婆罗洲島內陸地區的[[交通|交通]]主要靠河流運輸，有些內部山區河運更是重要的對外交通方式。婆罗洲島北部的河流可通航里程很少超過160公里，許多河運不通的內部山區仍與現代社會隔絕。

=== 山與山脈 ===
馬來西亞第一高峰在婆羅洲島的東北端，[[沙巴州|沙巴州]][[昆达山|昆达山]]境內的[[京那峇鲁山|京那峇鲁山]]，[[海拔|海拔]]4095.2公尺。

婆羅洲島上有幾條[[山脈|山脈]]。

從東北端的京那峇魯山（神山）開始，往西南延伸出一組連接的各山脈，另一端在島的西南端，西加里曼丹境內。整組山脈東北，西南走向，穿過島的中部。
# [[京那巴鲁山|京那峇魯山]]（神山）開始
# 先是[[依兰山脉|依兰山脉]]（Iran）。
# 接著是[[馬勒山脈|馬勒山脈]]（Muller）：為[[西加里曼丹|西加里曼丹]]與[[中加里曼丹|中加里曼丹]]二省的天然分界線。
# 然後是[[斯赫瓦納山脈|斯赫瓦納山脈]]（Schwaner）：也是西、中加里曼丹二省的天然分界線，此山脈上有加里曼丹地區第一高峰Gunung Bukit Raya，高2278公尺，在中加里曼丹境內。

在馬勒山脈與斯赫瓦納山脈交接處附近，向[[西|西]]延伸出一條[[卡普阿斯胡盧山脈|卡普阿斯胡盧山脈]]（Kapuas Hulu），是西加里曼丹與馬來西亞砂拉越州的天然交界。

在婆羅洲東南部，南加里曼丹境內有一條[[梅拉图斯山脉|梅拉图斯山脉]]。

=== 氣候 ===
盛行[[赤道气团|赤道气团]]，为[[赤道多雨气候|赤道多雨气候]]。終年高溫多雨，年均溫24-25℃，岛上[[降雨量|降雨量]]最少的地區Sangkulirang，年降雨量也有1625毫米。11月到次年3月是降雨多的月份，7-8月少雨。越往島的[[東|東]]部，降雨量越少；島內山區的降雨量比沿海地區多。

== 歷史 ==

{{Globalize/Chinese}}
=== 早期華人接觸 ===
約414年（[[晉安帝|晉安帝]][[隆安_(东晋)|隆安]]十四年），[[中國|中國]]僧侣[[法顯|法顯]]由[[印度|印度]]求得佛法，回歸[[中國|中國]]途中經過[[南洋|南洋]]，曾有一提及[[耶婆提|耶婆提]]（Yavadvipa），根據史家的意見，認為此地是現今的'''婆羅洲'''。

和[[中國|中國]]最早的通航紀錄是出現在《[[梁書|梁書]]》裡，520年（[[梁武帝|梁武帝]]普通元年），在[[中國|中國]]古籍中，當時被稱為渤泥、婆利、或婆羅，後來演變成婆羅乃（Brunei），也就是現在通用的[[汶萊|汶萊]]一名。在[[梁朝|梁]]、[[隋朝|隋]]、[[唐朝|唐]]三朝裡，[[婆羅洲|婆利]]都有遣送信使向[[中國|中國]][[朝貢|朝貢]]方物，直至[[宋朝|宋代]]這種接觸繼續保持，到了[[明朝|明朝]]一系列史無前例的官方航海便在此時開始，最著名的要算是[[鄭和下西洋|鄭和下西洋]]，據記載曾兩次經過渤泥。

約1375年（[[明朝|明]][[洪武|洪武]]8年），14、15世紀時[[華人|華人]]曾在[[沙巴|沙巴]]的[[京那巴当岸河|京那巴当岸河]]（{{lang|en|Sungai Kinabatangan}}）居住，有一[[明朝|明朝]][[使者|使者]][[黄森屏|黄森屏]]傳說還做過沿岸地區的統治者-[[拉惹|拉惹]]（Rajah）。

約1772年（[[清|清朝]][[乾隆|乾隆]]37年），[[華人|華人]]活動逐漸轉移到島西部，主要集中在一些金礦開採地區，如[[坤甸|坤甸]]（Pontinak）和三發（Sambas），近百名[[客家人|客家人]]抵達[[坤甸|坤甸]]，可說是開發此地的先驅者。

1777年[[華人|華人]]羅芳伯在婆羅洲島上（今[[加里曼丹|加里曼丹]]西部）成立了世界第一個由華人所創立的「[[共和國|共和國]]」－「[[蘭芳共和國|蘭芳共和國]]」。國家元首稱「大總制」。

1820年（[[清朝|清]][[道光|道光]]元年），估計約有三萬六千名[[華人|華人]]居住在此礦區內。1823年（[[清朝|清]][[道光|道光]]四年）已達十五萬人之多。

在[[荷蘭|荷蘭]]殖民勢力進入自[[印尼|印尼]][[蘇門達臘|蘇門達臘]]北移至南婆羅洲之後，他們对[[華人|華人]]在金礦區的利益不满，並使用[[權力|權力]]限制華人[[移民|移民]]和[[貿易|貿易]]，爾後逐漸減少，不少礦工也被迫遷徙至[[砂拉越|砂拉越]]。

== 政治 ==
[[File:Borneo_blank_map.svg|thumb]]
分为三国领土：分别属于[[马来西亚|马来西亚]]，[[汶莱|汶莱]]及[[印尼|印尼]]。目前是世界上唯一一个属于三個受普遍承認的国家的岛屿。<ref>{{cite news|url=http://www.epochtimes.com/b5/10/7/18/n2969447.htm|title=組圖：地球肺葉萎縮中 婆羅洲熱帶雨林|date=2010年7月20日|publisher=大紀元}}</ref>

* 北部为[[东马来西亚|东马来西亚]]全境，行政区为[[沙巴|沙巴]]与[[砂拉越|砂拉越]]二州，及[[纳闽|纳闽]]联邦直辖区。（所佔面积第二，地圖橘色部份。）沙巴原稱[[北婆羅洲|北婆羅洲]]，是前[[英国|英国]]的皇家屬地。砂拉越原稱[[砂拉越王国|砂拉越王国]]，由布魯克家族统治，之后也成为了前英國殖民地。[[沙巴|沙巴]]和[[砂拉越|砂拉越]]于1963年9月16日和[[马来亚联合邦|马来亚联合邦]]及[[新加坡|新加坡]]共同组成[[马来西亚|马来西亚]]。[[新加坡|新加坡]]后来退出[[马来西亚|马来西亚]]。
**馬來西亞聯邦成立之前，[[菲律賓|菲律賓]]曾根據[[蘇祿蘇丹國|蘇祿蘇丹國]]與北婆羅州公司之前的合約，宣稱擁有北婆羅洲（沙巴）主權。2013年發生的[[2013年拿篤衝突|拿篤衝突]]，便是與此有關。
* 在沙巴、砂拉越中间为[[汶萊|汶萊]]，全境均在婆罗洲岛内，包括临近岛屿。（所佔面积最小，地圖绿色部份。〕
* 馬來西亞的[[砂拉越|砂拉越]]、[[沙巴|沙巴]]及[[汶萊|汶萊]]合稱“北婆三邦”，簡稱“砂汶沙”。面積共有196,500平方公里。
* 南部为屬于[[印尼|印尼]]的[[加里曼丹|加里曼丹]]地區，面積有539,500平方公里，分[[東加里曼丹|東加里曼丹]]、[[北加里曼丹|北加里曼丹]]、[[南加里曼丹|南加里曼丹]]、[[中加里曼丹|中加里曼丹]]、[[西加里曼丹|西加里曼丹]]五省。加里曼丹1949年以前是[[荷蘭|荷蘭]]殖民地。（所佔面积最大，地圖黃色部份。）

== 經濟 ==
島上[[經濟|經濟]]以[[林業|林業]]為主，[[礦產資源|礦產資源]]有[[黃金|黃金]]、[[鑽石|鑽石]]、[[煤|煤]]、[[石油|石油]]、[[鈾|鈾]]等。

== 人口 ==
婆羅洲島人口分佈不均勻，差別極大。因沿海地區交通較便利，受海洋調節氣溫較適宜，人口主要集中在各大[[城市|城市]]及沿[[海|海]]地區，內部山區人口稀少，居住在內部山區的多是當地土著[[達雅族|達雅族]]，其[[聚落|聚落]]多沿[[河|河]]分佈。

* [[大花草|大花草]]或称莱佛士花（Rafflesia），世界最大的[[花|花]]，目前已知有16種，本島即有3種，包括[[凱依西大花草|凱依西大花草]]（''Rafflesia Keithii''）直徑達80公分，為三種中最巨大的。另有[[普利契大花草|普利契大花草]]（''Rafflesia Pricei''）及[[唐古禮尼大花草|唐古禮尼大花草]]（Rafflesia Tengku Adlini），花型較小。
