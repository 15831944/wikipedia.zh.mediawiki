'''黎明一夜傾情演唱會'''（{{lang-en|Leon Lai in Concert 1992}}）是[[香港|香港]]歌手[[黎明|黎明]]在[[香港體育館|香港體育館]]（俗稱紅磡體育館）舉行的個人演唱會，於1992年10月7日至10月14日期間舉行，合共演出十場，外界將此次演唱會視為黎明自1990年成為歌手以來首個正式演唱會<ref >{{cite news |language = 中文 |url = https://drive.google.com/open?id=1FNrBivfLGiFWQiioQL7Q8JjGoH3vKDD4|title = 黎明演唱會的第一個晚上 |publisher =星島日報|date = 1992-10-08|accessdate = 2019-04-19}}</ref>。

== 演出情況 ==
1992年10月7日至14日期間，黎明在香港體育館舉行十場個人演唱會，由[[電視廣播有限公司|電視廣播有限公司]]參與攝錄工作，除了香港本地的觀眾外，還有來自[[美加|美加]]、[[新加坡|新加坡]]及[[韓國|韓國]]等地的觀眾<ref name="news1">{{cite news |language = 中文 |url = https://drive.google.com/open?id=15hfgDBaLDMUYBlBbuNa8Tp8Om5wu2OjQ|title = 我的親愛結束第一次 黎明台前大灑男兒淚 |publisher =香港商報|date = 1992-10-16|accessdate = 2019-04-19}}</ref><ref name="news3">{{cite news |language = 中文 |url = https://drive.google.com/open?id=1ypZzRmFqKz8MqQ68P7L9T5MdGbp4uwpc|title = 黎明強調狀態仍勇 明年八月紅館再見|publisher =華僑日報|date = 1992-10-16|accessdate = 2019-04-19}}</ref>，同時有不少的藝人及歌手前來觀看黎明的演出，包括：[[周海媚|周海媚]]、[[王祖賢|王祖賢]]、[[李嘉欣|李嘉欣]]、[[郭富城|郭富城]]等<ref name="news2">{{cite news |language = 中文 |url = https://drive.google.com/open?id=1RThtabArQ_dK9cQ8uSW2U4sxdiariTeN|title = 黎明個唱代歌迷售門票 遭歪曲發警告 |publisher =快報|date = 1992-10-16|accessdate = 2019-04-19}}</ref><ref name="news1">{{cite news |language = 中文 |url = https://drive.google.com/open?id=15hfgDBaLDMUYBlBbuNa8Tp8Om5wu2OjQ|title = 我的親愛結束第一次 黎明台前大灑男兒淚 |publisher =香港商報|date = 1992-10-16|accessdate = 2019-04-19}}</ref>，也有不少的演藝界人士送上花籃祝賀<ref>{{Cite book| title =黎明傾情演唱會特刊|location =香港| date =1992|pages =第9頁| url = https://drive.google.com/open?id=1Nw_0F-xMy795UBSKIT9ZzLmwi1XAuj_r |language =中文|access-date =2019-04-19 }}</ref>。這次的演唱會不設嘉賓表演環節，只邀請了演唱會的音樂總監[[倫永亮|倫永亮]]一同合唱，其餘大部份時間都是黎明的個人演唱及分享，期間播放了黎明與[[卡通|卡通]]合演的[[動畫|動畫]]影片，在當時的香港演唱會是未曾出現過的<ref>{{Cite book| title =黎明傾情演唱會特刊|location =香港| date =1992|pages =第21至22頁| url = https://drive.google.com/open?id=13q2e6j_b-Hdr1UZMkbQYuiNQJCe19WDJ|language =中文|access-date =2019-04-20 }}</ref>。黎明和主辦機構負責人[[張耀榮|張耀榮]]均認為這次的演唱會效果及反應理想，隨即計劃於1993年8月在香港體育館再次舉行大型的個人演唱會<ref name="news3">{{cite news |language = 中文 |url = https://drive.google.com/open?id=1ypZzRmFqKz8MqQ68P7L9T5MdGbp4uwpc|title = 黎明強調狀態仍勇 明年八月紅館再見|publisher =華僑日報|date = 1992-10-16|accessdate = 2019-04-19}}</ref>。

== 相關事件 ==
