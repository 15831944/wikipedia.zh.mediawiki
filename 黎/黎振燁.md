'''黎振燁'''（{{lang|en|'''Jackson Lai Chun Ye'''，{{bd|1986年|3月8日|catIdx=L黎振燁}}}}），曾用藝名「'''黎柏麟'''」和「'''黎栢倫'''」，香港男演員、模特兒及《[[2016年度香港先生選舉|2016年度香港先生選舉]]》冠軍，現為[[無綫電視藝員列表|無綫電視經理人合約]]藝員<ref>[http://artiste.tvb.com/main/artistedetail/laichunye 黎振燁 Lai Chun Ye - TVB藝人資料]</ref>。

== 簡歷 ==
黎振燁畢業於[[東華三院陳兆民中學|東華三院陳兆民中學]]，曾是時裝設計師<ref name=JLCY>{{cite web |title=【再創世紀】黎振燁曾放棄演員夢　太太安慰：唔好再諗養我唔起 |url=https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/240722/%E5%86%8D%E5%89%B5%E4%B8%96%E7%B4%80-%E9%BB%8E%E6%8C%AF%E7%87%81%E6%9B%BE%E6%94%BE%E6%A3%84%E6%BC%94%E5%93%A1%E5%A4%A2-%E5%A4%AA%E5%A4%AA%E5%AE%89%E6%85%B0-%E5%94%94%E5%A5%BD%E5%86%8D%E8%AB%97%E9%A4%8A%E6%88%91%E5%94%94%E8%B5%B7 |publisher=[[香港01|香港01]] |date=2018-09-27 |accessdate=2018-09-28}}</ref>。2009年，他以「黎柏麟」及「黎栢倫」之名加入[[無綫電視藝員訓練班|無綫電視第23期藝員訓練班]]<ref name=LCY>{{cite web |title=路人甲兜轉8年後奪冠　黎振燁感謝黎耀祥鼓勵 |url=http://www.hk01.com/%E5%A8%9B%E6%A8%82/42759/-%E6%B8%AF%E7%94%B72016-%E8%B7%AF%E4%BA%BA%E7%94%B2%E5%85%9C%E8%BD%898%E5%B9%B4%E5%BE%8C%E5%A5%AA%E5%86%A0-%E9%BB%8E%E6%8C%AF%E7%87%81%E6%84%9F%E8%AC%9D%E9%BB%8E%E8%80%80%E7%A5%A5%E9%BC%93%E5%8B%B5 |publisher=香港01 |date=2016-09-12 |accessdate=2016-11-18}}</ref>，但當了三年跑龍套便離開無綫電視。2012年，他轉投香港電視網絡，曾參演過《[[警界線|警界線]]》、《[[大眾情性|大眾情性]]》和《[[我阿媽係黑玫瑰|我阿媽係黑玫瑰]]》等劇集，惟兩年合約完結後則轉做[[太陽娛樂文化|太陽英模國際]]旗下模特兒<ref name=LCY/><ref>{{cite web |title=《路人甲做到灰》港男冠軍跟過洗米華搵食 |url=https://www.youtube.com/watch?v=3MKy-Gv_4zA |publisher=[[壹週刊_(香港)|壹週Plus]] |date=2016-09-14 |accessdate=2016-11-18}}</ref>。

2016年，黎振燁約滿又隨即參選《[[香港先生選舉|香港先生選舉]]》<ref name=LCY/>，並以大熱姿態奪取冠軍。而其參選理由是喜歡演戲，故藉此再圓演員夢<ref>{{cite web |title=黎振燁搞喊老婆玩感性：我鍾意做戲 |url=http://nextplus.nextmedia.com/news/ent/20150320/170785 |publisher=[[ON.CC|東網]] |date=2016-09-10 |accessdate=2016-11-18}}</ref>；同年底還以旁聽生身份加入[[無綫電視藝員訓練班|無綫電視演藝進修班]]。2018年，他分別在《[[愛·回家之開心速遞|愛·回家之開心速遞]]》、《[[再創世紀|再創世紀]]》和《[[是咁的，法官閣下|是咁的，法官閣下]]》裡，飾演上市公司副主席「Jacky」<ref>{{cite web |title=《愛·回家之開心速遞》情傾惡死大小姐 ，黎振燁最愛的是與他一起捱足10年的太太起 |url=https://www.cosmopolitan.com.hk/love/Jackson-Lai-real-love |publisher=[[COSMOPOLITAN|COSMOPOLITAN]] |date=2018-04-19 |accessdate=2018-04-20}}</ref>、運輸工頭「傅建業」<ref name=JLCY/>及大律師「鄧大志」，表現受到不少觀眾注目且讚賞，當中「鄧大志」為其出道以來較多戲份的要角<ref>{{cite web |title=【是咁的，法官閣下】被封NG王　港男黎振燁即律師上身：我認罪！ |url=https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/254400/%E6%98%AF%E5%92%81%E7%9A%84-%E6%B3%95%E5%AE%98%E9%96%A3%E4%B8%8B-%E8%A2%AB%E5%B0%81ng%E7%8E%8B-%E6%B8%AF%E7%94%B7%E9%BB%8E%E6%8C%AF%E7%87%81%E5%8D%B3%E5%BE%8B%E5%B8%AB%E4%B8%8A%E8%BA%AB-%E6%88%91%E8%AA%8D%E7%BD%AA |publisher=香港01 |date=2018-11-02 |accessdate=2018-11-03}}</ref>。
