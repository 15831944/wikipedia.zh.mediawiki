'''黎萬宏'''（Lai Man Wang，{{bd||8月30日}}），綽號'''泥鯭'''，香港音樂人，著名樂隊[[RubberBand|RubberBand]]成員，擔任[[鼓手|鼓手]]。在出道時已因樣貌酷似知名藝人及名嘴[[林海峰|林海峰]]令其知名度大增，亦因而獲得「翻版林海峰」之稱<ref name="bka._翻版林海">{{Cite web |title=翻版林海峰 泥鯭求婚成功 - 明周娛樂 |trans-title= |author= |work=明周娛樂 |date= |access-date=28 September 2018 |url= https://bka.mpweekly.com/focus/local/20170116-33267 |language=zh |quote=}}</ref>
。2009年便與樂隊和林海峰合作歌曲《夾硬泥》而首次主唱Rubberband歌曲<ref name="hk.a_Rubb">{{Cite web |title=RubberBand爆笑開騷Jan扮泥鯭孖住唱 |author= |work=Apple Daily 蘋果日報 |date= |access-date=28 September 2018 |url= https://hk.appledaily.com/entertainment/art/20091125/13455797 |language= |quote=}}</ref>
。2010年更獲得廣告商看中，與林海峰合作拍[[EPS易辦事|EPS易辦事]]廣告<ref name="orie_林海峰泥">{{Cite web |title=林海峰泥鯭兄弟重逢合作搵銀 - 東方日報 |trans-title= |author= |work=orientaldaily.on.cc |date= |access-date=28 September 2018 |url= http://orientaldaily.on.cc/cnt/entertainment/20100112/00282_016.html |language=zh |quote=}}</ref>
。他在[[香港城市大學|香港城市大學]]攻讀物理及材料系碩士畢業，入行前曾長時間任職程式設計員。2017年1月，他向拍拖多年的圈外女友「阿欣」求婚成功<ref name="hk.o_泥鯭@R">{{Cite web |title=泥鯭@RubberBand怕醜避談求婚一事 |trans-title= |author= |work=on.cc東網 |date= |access-date=28 September 2018 |url= http://hk.on.cc/hk/bkn/cnt/entertainment/20170118/bkn-20170118162533950-0118_00862_001.html |language=zh |quote=}}</ref><ref name="hd.s_雞年做人">{{Cite web |title=雞年做人夫 泥鯭拖未婚妻睇婚戒 |trans-title= |author= |work=頭條日報 Headline Daily |date= |access-date=28 September 2018 |url= http://hd.stheadline.com/news/daily/ent/541022/ |language=zh |quote=}}</ref>。同年，他亦展開演員生涯，接拍[[viuTV|viuTV]]劇集《[[賤民20|賤民20]]》。
