'''黎里镇'''是中国[[江苏省|江苏省]][[苏州市|苏州市]][[吴江区|吴江区]]所辖的9个镇之一，东接[[上海市|上海市]]，南邻[[嘉兴市|嘉兴市]]。面积258平方千米，人口约20万人。

2001年，莘塔镇并入芦墟镇；2003年，金家坝镇并入芦墟镇，北厍镇并入黎里镇。
<ref>{{cite web |title=吴江1986年至2005年行政区划  |url=http://www.wujiangtong.com/webPages/DetailNews.aspx?id=11962 |publisher=吴江档案局}}</ref>
2006年，芦墟与黎里两镇合并，成立汾湖镇，与[[汾湖高新技术产业开发区|汾湖高新技术产业开发区]]区镇合一。2013年，汾湖镇更名为黎里镇。
<ref>{{cite web|url=http://jiangsu.china.com.cn/html/qwfb/148142_1.html|title=苏州吴江区汾湖镇更名黎里镇|date=2013-05-29|accessdate=2013-07-12|author=人民网|deadurl=yes|archiveurl=https://web.archive.org/web/20140606223323/http://jiangsu.china.com.cn/html/qwfb/148142_1.html|archivedate=2014-06-06}}</ref>

== 地理 ==

黎里镇地处[[太湖平原|太湖平原]]，河网密布，雨量充沛。

黎里镇位于[[江苏省|江苏省]][[苏州市|苏州市]][[吴江区|吴江区]]东南部，东侧是上海市[[青浦区|青浦区]][[金泽镇|金泽镇]]，南侧是[[浙江省|浙江省]][[嘉善县|嘉善县]]，西面是[[平望镇|平望镇]]，北边是[[昆山市|昆山市]][[周庄镇|周庄镇]]。

== 行政区划 ==
黎里镇辖9个社区和48个村。

*社区：镇东、镇西、新区、莘塔、金家坝、黎新、兴黎、振黎、北厍。
*村：秋田、城司、伟明、芦东、汾湖湾、莘南、莘西、元荡、龙泾、港南、三好、三和、高新、东联、雪巷、蚬南、群众、长胜、杨文头、红旗、梅石、大潮、跃进、新钢、星谊、银杏、黎阳、建南、史北、华莺、大联、方联、汤角、青石、乌桥、黎花、雄锋、元鹤、东方、梅墩、大胜、黎星、汾湖、浮楼、沈家港、大长港、川心港、永新。

== 经济 ==

黎里镇历史上是著名的鱼米之乡。现在汾湖高新技术产业开发区在吴江的经济中占有很大份额。

在乡镇合并之前，芦墟镇和黎里镇都在吴江位居七大镇之列。

== 交通 ==
