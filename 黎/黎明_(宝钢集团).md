'''黎明'''（{{bd|1927年|11月21日||}}）生于今[[天津市|天津市]][[宁河县|宁河县]][[芦台镇|芦台镇]]，[[中华人民共和国|中华人民共和国]]钢铁工业部门官员<ref name=zhuanjia>{{Cite web|url =http://www.csm.org.cn/news/show_news.aspx?newsId=11726|title =冶金专家：黎明|author =|date =|publisher =中国金属学会|language =|accessdate =2017-07-19|quote =}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref name=md>{{Cite web |url =  http://www.sohu.com/a/149306357_649931|title = 宝钢黎明：耄耋之年话钢铁人生 |author =  |date =2017-06-16   |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1946年8月，黎明考入北京市高级工业学校机械科。1948年12月参加[[中国共产党|中国共产党]]领导的革命。1949年6月加入中国共产党。1949年8月，进入[[鞍钢|鞍钢]]动力部发电厂任见习技术员。1951年5月起，先后在鞍钢机械处、大型厂、生产处任职。1960年至1966年，历任鞍钢生产处副处长、生产办公室副主任，负责鞍钢全厂生产的管理<ref name=zhuanjia/><ref name=md/>。

1968年初，调任[[攀钢集团有限公司|攀钢]]炼钢厂副厂长。1969年12月4日，中央确定攀钢1号高炉在1970年7月1日投产，临危受命的黎明调整了施工顺序并负责攀钢开工的准备工作，实现高炉如期投产。1973年起，黎明任攀钢革命委员会副主任、总工程师。[[文革|文革]]结束后，1980年4月出任攀钢经理。攀钢是中国首个全部使用国产设备对钒钛磁铁矿进行冶炼的大型钢铁联合企业，在黎明的组织下实现了正常生产。黎明在任内还对攀钢管理体制进行了改革<ref name=zhuanjia/><ref name=md/>。

1982年3月，黎明任[[中华人民共和国冶金工业部|中华人民共和国冶金工业部]]副部长、党组副书记。1982年9月，在[[中共十二大|中共十二大]]上当选为[[中国共产党第十二届中央委员会候补委员列表|中国共产党第十二届中央委员会候补委员]]。1985年9月，在[[1985年中国共产党全国代表会议|中国共产党全国代表会议]]增选为中央委员会委员。1987年，在[[中共十三大|中共十三大]]上继续当选为[[中国共产党第十三届中央委员会候补委员列表|中国共产党第十三届中央委员会候补委员]]。1992年，在[[中共十四大|中共十四大]]上继续当选为[[中国共产党第十四届中央委员会候补委员列表|中国共产党第十四届中央委员会候补委员]]<ref name=zhuanjia/><ref name=md/>。

1983年9月14日起，黎明兼任[[上海|上海]][[宝山钢铁总厂|宝山钢铁总厂]]工程指挥部总指挥、党委书记、总厂厂长。1994年，任[[宝钢集团|宝钢集团]]董事长，一直任至1998年离休。在黎明的领导下，宝钢先后完成一至三期工程建设，按照[[邓小平|邓小平]]“掌握新技术，要善于学习，更要善于创新”题词的要求，开创了[[中华人民共和国|中华人民共和国]]钢铁企业现代化建设的新路<ref name=zhuanjia/><ref name=md/>。

在职期间，黎明还先后兼任[[中国金属学会|中国金属学会]]理事长、[[中国工业协会|中国工业协会]]副会长、[[中国企业管理协会|中国企业管理协会]]副会长、中国企业文化研究会副会长、中国集团公司促进会会长等职务<ref name=zhuanjia/>。
