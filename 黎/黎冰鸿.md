'''黎冰鸿'''（{{bd|1913年|9月|1986年|8月15日}}，原名'''黎炳康'''），原籍[[广东|广东]][[东莞|东莞]]，生于[[越南|越南]][[下龙市|鸿基]]，[[中华人民共和国|中华人民共和国]]画家、美术教育家<ref name=zgmsg>{{Cite web|url=http://www.namoc.org/cbjy/msbk/msrwk/201307/t20130712_255912.htm|title=黎冰鸿|author=|date=2013-07-12|publisher=中国美术馆|language=|accessdate=|quote=}}</ref>。

==生平==
早年深受曾留学[[法国|法国]]的越南画家[[阮有悦|阮有悦]]的影响<ref name=yisi>{{Cite web |url = http://mt.sohu.com/20161117/n473472717.shtml |title = 关于黎冰鸿先生，这几点很有意思！ |author =  |date = 2016-11-17 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。1931年，来到[[香港|香港]]师从[[李铁夫|李铁夫]]<ref name=dyw>{{Cite web |url = http://www.1905.com/mdb/star/3018618/life/ |title = 黎冰鸿的星路历程 |author =  |date =  |publisher = 电影网 |language =  |accessdate =  |quote =  }}</ref>。[[抗日战争|抗日战争]]时期，曾从事抗日宣传。1946年，来到[[苏北解放区|苏北解放区]]的[[华中建设大学|华中建设大学]]担任教授<ref name=jianjie/><ref name=dyw/>。

[[中华人民共和国|中华人民共和国]]时期，黎冰鸿历任[[中国人民解放军上海市军事管制委员会|中国人民解放军上海市军事管制委员会]]文艺处美术干事，《华东画报》记者及制片室负责人。1953年10月调到[[杭州|杭州]]，任中央美术学院华东分院[[中国美术学院|中央美术学院华东分院]]（1958年更名浙江美术学院，1993年更名[[中国美术学院|中国美术学院]]）教授，是研究生班指导教师。1954年，任中央美术学院华东分院油画科主任。1955年3月，参与创建中央美术学院华东分院油画系，任油画系主任。1956年，任中央美术学院华东分院副院长、院党委委员兼油画系主任。1962年10月，到浙江美术学院油画系第三工作室工作。1963年，任浙江美术学院学术委员会副主任委员<ref name=jianjie>{{Cite web |url = http://news.163.com/16/0323/04/BIQL8R2L00014AEF.html |title = 黎冰鸿简介 |author =  |date =2016-03-23   |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref name=dyw/><ref name=sohu>{{Cite web |url =  http://mt.sohu.com/20161125/n474156212.shtml|title =  除了画家，黎冰鸿还有另外一个身份！|author =  |date = 2016-11-25 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

1966年[[文化大革命|文化大革命]]爆发后，黎冰鸿被打成“[[走资派|走资派]]”，撤销一切职务，关进[[牛棚_(文革用语)|牛棚]]，被迫停止创作，许多作品被毁。1971年，恢复教学工作。1978年8月，指导了该学院建院后首批油画系研究生。1979年2月，出席全国艺术教育会议。1979年3月，恢复浙江美术学院副院长、党委委员的职务。1979年6月，任浙江美术学院学术委员会副主任。1981年，浙江美术学院油画系开展工作室教学，黎冰鸿任油画系第三工作室主持人。1984年，任浙江美术学院艺术咨询委员会主任<ref name=sohu>{{Cite web |url =  http://mt.sohu.com/20161125/n474156212.shtml|title =  除了画家，黎冰鸿还有另外一个身份！|author =  |date = 2016-11-25 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。他是[[中国美术家协会|中国美术家协会]]理事、中国美术家协会浙江分会副主席<ref name=jianjie>{{Cite web |url = http://news.163.com/16/0323/04/BIQL8R2L00014AEF.html |title = 黎冰鸿简介 |author =  |date =2016-03-23   |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref name=dyw/>。

黎冰鸿主要创作油画、写生、连环画；题材主要有：历史题材、人物、风景。早期油画多为主体性创作。中期以革命历史及[[阶级斗争|阶级斗争]]题材为主，并且有部分人物画。1954年曾访问[[保加利亚|保加利亚]]，画有一批油画写生。晚年开始创作反映江南水乡、北国风光的风景画，画风恬淡<ref name=zgmsg/><ref name=dyw/><ref name=dyw/>。

1986年8月，黎冰鸿逝世<ref name=zgmsg/>。
