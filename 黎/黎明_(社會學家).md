'''黎明'''{{BD|1985年}}是一名社會學學者，現於[[香港教育大學|香港教育大學]]任職社會學系講師，曾任[[香港中文大學|香港中文大學]]社會學系講師，負責教授醫學社會學、中國社會和性別等科目<ref>[http://www.cuhk.edu.hk/soc/LI_Ming.html CUHK Sociology Lecturer]</ref>。

出生於[[上海市|上海市]]，本科就讀[[復旦大學|復旦大學]]社會學系，期間曾經擔任學生會幹事。其後受[[蔡元雲|蔡元雲]]醫生鼓勵到[[香港|香港]]求學，先後在[[香港中文大學|香港中文大學]]取得社會學碩士和博士<ref>[https://lj.hkej.com/lj2017/artculture/article/id/1681775/%E3%80%90%E4%BA%BA%E7%89%A9%E3%80%91%E4%B8%AD%E5%A4%A7%E7%A4%BE%E6%9C%83%E7%B3%BB%E8%AC%9B%E5%B8%AB%E9%BB%8E%E6%98%8E+%E4%BB%BB%E4%BA%BA%E8%A9%AE%E9%87%8B%E7%9B%BC%E6%8F%9B%E5%90%8C%E7%90%86%E5%BF%83 【人物】中大社會系講師黎明 任人詮釋盼換同理心]</ref>。因為其來自[[中國大陸|中國大陸]]到香港求學並定居的身份，在2017年中大學生會民主墻事件期間積極為學術及言論自由發聲而引起關注。<ref>{{Cite web |url = https://lj.hkej.com/lj2017/artculture/article/id/1681775/【人物】中大社會系講師黎明+任人詮釋盼換同理心 |title = 中大社會系講師黎明 任人詮釋盼換同理心 |author = 吳雄 |date = 2017-10-27 |publisher = [[信報|信報]] |language =  zh-hk}}</ref><ref>{{Cite web |url = https://thestandnews.com/society/顛覆劇本-黎明-這不是野人被civilized的故事/ |title = 顛覆劇本　黎明：這不是野人被civilized的故事 |author = 楊子琪 |date = 2017/9/30 |publisher = [[立場新聞|立場新聞]] |language = zh-hk}}</ref><ref>{{Cite web |url = http://ubeat.com.cuhk.edu.hk/132_people/ |title = 我的行動定義我的身分—專訪黎明 |author = 陳雅琪 |date = 2017-11-28 |publisher = 大學線 |language = zh-hk}}</ref><ref>{{Cite web |url = https://hk.lifestyle.appledaily.com/nextplus/magazine/article/20170923/2_549763_0/-小粉紅懺悔錄-黎明-洗腦教育令你內置思想警察-全文- |title = 黎明：洗腦教育令你內置思想警察 |author = 蔡慧敏 |date = 2017-09-23 |publisher = [[蘋果日報|蘋果日報]] |language = zh-hk}}</ref> 分別在《明報》、《立場新聞》、《時代論壇》均有專欄。

丈夫為著名公共衛生學者、歌手/音樂人[[鍾一諾|鍾一諾]]。
