'''黎火辉'''（{{bd|1963年|||}}），[[江西省|江西]][[丰城市|丰城]]人。[[中国人民解放军中将|中国人民解放军中将]]。中共第十九届中央委员。

==生平==
[[中国人民解放军国防大学|国防大学]]基本系毕业。曾任[[中国人民解放军南京军区|南京军区]]军训和兵种部副部长、[[中国人民解放军福建省军区|福建省军区]]海防某师师长。2011年1月，任[[中国人民解放军陆军第七十一集团军|陆军第十二集团军]]参谋长。2015年，任[[中国人民解放军陆军第七十三集团军|陆军第三十一集团军]]军长<ref>{{Cite web |url =  http://news.qq.com/a/20150611/009172.htm?tu_biz=1.114.1.1|title = 黎火辉少将升任31集团军军长 军部位于厦门 |author =  |date = 2015-06-11 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。

2017年1月，任[[中央军委训练管理部|中央军委训练管理部]]部长<ref>{{cite news|title=黎火辉少将晋升副战区级，曾任陆军第31集团军军长|url=http://www.thepaper.cn/newsDetail_forward_1608918|accessdate=2017-02-06|work=澎湃新闻}}</ref><ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1611081 |title =第31集团军原军长黎火辉少将升任军委训练管理部部长  |author =  |date =  2017-02-03|publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref>。2018年7月，晋升中将军衔<ref>{{cite web |title=中央委员、军委训练管理部部长黎火辉晋升中将 |url=https://www.thepaper.cn/newsDetail_forward_2312411 |website=澎湃新闻 |accessdate=2018-08-01 |date=2018-08-01}}</ref>。
