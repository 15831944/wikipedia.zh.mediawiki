'''黎學勤'''（{{lang|en|'''Jeffery Lai Hok Kan'''}}，{{bd|1988年|12月21日}}），[[香港|香港]]男性[[模特兒|模特兒]]及[[演員|演員]]。現為 [[JamCast_Management_(HK)|JamCast Management]] 旗下藝人。

== 簡歷 ==
黎學勤自小在單親家庭長大<ref name=JLF>{{web reference|URL=https://www.recruit.com.hk/Resources/ArticleDetail.aspx?articleId=37048|title=黎學勤　踏實暖男|work=[[Recruit.com.hk|Recruit.com.hk]]|year=2018年|date=7月3日}}</ref>，母子同住[[香港島|香港島]][[薄扶林|薄扶林]][[華富邨|華富邨]]{{noteTag|黎學勤是[[Facebook|Facebook]][[華富邨|華富邨]]居民群組「Wah Fu Estate」及[[香港南區|香港南區]]街坊群組「南區之友^o^」成員。}}，為[[華富邨寶血小學|華富邨寶血小學]]2000年第30屆上午校畢業生<ref>{{web reference|URL=https://www.facebook.com/events/201128863330810/permalink/463621893748171/|title=華富邨寶血小學上午校 2000年屆6A / 六智班 聚會|work=Lai Ching Lun Janet|year=2013年|date=11月20日}}</ref>。他唸中學期間曾加入香港水球代表隊，可為照顧後來因病入院的母親，選擇放棄集訓<ref name=JLF/>，至2008年則畢業於[[香港專業教育學院|香港專業教育學院]]（遊戲及動畫高級文憑課程），同時亦曾修讀過[[香港公開大學|香港公開大學]] Network Security 課程，但未完成學業便投身電腦保安兼模特兒工作<ref name=JL/>。同年，他也取得了[[香港游泳教練會|香港游泳教練會]] Level I 教練証書<ref>[http://www.hksca.org.hk/new_page_6.htm 香港游泳教練會有限公司 - 証書教練名冊]</ref>。

2017年，黎學勤因拍攝全球酒店預訂服務搜尋網站 [[trivago|trivago]] 的電視廣告而備受注目<ref name=JL/><ref>[https://www.elle.com.hk/celebrity/feature/new-male-god-from-advertisments#sss4xTkPCIGdWvcU.97 廣告新男神崛起！麒麟啤 Louis、Trivago 黎學勤、翠華林耀聲]，[[ELLE|ELLE]]，2017年4月4日</ref><ref>[http://jet.my-magazine.me/article/detail/travel/9720 【JET TRAVELLER】Jeffery Lai 「搵酒店？」新男神]，[[JET_(香港雜誌)|JET]]，2017年9月28日</ref><ref>[http://hk.mensuno.asia/node/%E5%94%94%E6%AD%A2%E9%85%92%E5%BA%97%E6%90%B5%E9%BB%8E%E5%AD%B8%E5%8B%A4%E5%B9%AB%E6%89%8B-8758 唔止酒店搵黎學勤幫手]，[[men's_uno|men's uno]]，2017年12月8日</ref><ref>{{Cite news |url=https://www.hk01.com/article/159845/ |title=Trivago 男神 黎學勤靠一個廣告彈起　又做 ViuTV 主持又北上拍戲 |date=2018年2月14日 |work=[[香港01|香港01]]}}</ref>，更獲邀到中國內地拍攝首部電影《[[王者攻略|王者攻略]]》，飾演將軍「汪鏡」一角<ref name=JL/><ref>{{Cite news |url=http://hk.on.cc/hk/bkn/cnt/entertainment/20170820/bkn-20170820163510499-0820_00862_001.html |title=想搵酒店？靚仔黎學勤想進軍影壇 |work=[[ON.CC|東網]] |date=2017年8月20日}}</ref>。
