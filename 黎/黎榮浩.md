'''黎榮浩'''（{{lang-en|'''Lai Wing Ho Joe'''}}<ref name=info>{{Cite web|url=http://www.elections.gov.hk/legco2016/pdf/intro_to_can/LC3_6.pdf|title=九龍東地方選區候選人簡介|accessdate=2016-09-14|work=2016立法會選舉}}</ref>；{{bd|1973年||||L黎}}<ref name=info/>），[[香港|香港]][[政治人物|政治人物]]，現任[[黃大仙|黃大仙]][[區議會|區議會]]（H11[[橫頭磡邨|橫頭磡邨]]選區）[[區議員|區議員]]，[[黃大仙區議會|黃大仙區議會]]<ref>{{cite web |url=http://www.districtcouncils.gov.hk/wts/chinese/welcome.htm |title=存档副本 |accessdate=2011-08-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20110806020058/http://www.districtcouncils.gov.hk/wts/chinese/welcome.htm |archivedate=2011-08-06 }}</ref>副主席，[[民建聯|民建聯]]成員。

==簡歷==
黎榮浩小學畢業於聖文德天主教小學（下午校）（現為[[慈雲山聖文德天主教小學|慈雲山聖文德天主教小學]]），現為該校校友會主席。

學生時代的黎榮浩，在大學界中已是[[體育|體育]]健將。1994-95年，黎榮浩擔任[[香港浸會大學|香港浸會大學]][[划艇|划艇]]會主席及大學划艇隊隊長，當時香港浸會大學划艇隊連續三屆摘下全港大學划艇錦標賽的團體總[[冠軍|冠軍]]。與及奪取多項個人和隊制項目賽事錦標。<ref>[http://paper.wenweipo.com/2004/12/27/NS0412270001.htm 黎榮浩以體育精神打拚政壇]</ref>

1995年，黎榮浩被香港業餘划艇協會（現稱：[[中國香港賽艇協會|中國香港賽艇協會]]）評選為該年度全港最傑出男子新秀[[運動員|運動員]]。

1999年，黎榮浩首次參與[[1999年香港區議會選舉|1999年香港區議會選舉]]，擊敗當時的選舉對手，即現任[[民主民生協進會|民主民生協進會]]（民協）主席[[廖成利|廖成利]]。黎榮浩得票：2122票，廖成利得票：1829票。<ref>{{cite web |url=http://www.elections.gov.hk/elections/dcelect99/chi/c21_cg.htm |title=存档副本 |accessdate=2008-10-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20080501212539/http://www.elections.gov.hk/elections/dcelect99/chi/c21_cg.htm |archivedate=2008-05-01 }}</ref>

[[2003年香港區議會選舉|2003年香港區議會選舉]]，民建聯雖陷入前所未有的低潮，但黎榮浩仍以高票選舉連任成功，黎榮浩得票3163票，比對手高出1400多票。<ref>{{cite web |url=http://www.elections.gov.hk/elections/dc2003/tc_chi/result/result_wts.html |title=存档副本 |accessdate=2008-04-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20080501202852/http://www.elections.gov.hk/elections/dc2003/tc_chi/result/result_wts.html |archivedate=2008-05-01 }}</ref>

[[2007年香港區議會選舉|2007年香港區議會選舉]]及[[2015年香港區議會選舉|2015年香港區議會選舉]]，黎榮浩以無對手之下，[[自動當選|自動當選]]為新一屆的黃大仙區議會議員。<ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/index.html |title=存档副本 |accessdate=2008-04-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20080503042747/http://www.elections.gov.hk/dc2007/chi/index.html |archivedate=2008-05-03 }}</ref>

社會公職方面，黎榮浩除了擔綱區議員的職務外，現時也是特區政府[[行政上訴委員會|行政上訴委員會]]委員。<ref>{{cite web |url=http://www.admwing.gov.hk/chi/links/aab_mem.htm |title=存档副本 |accessdate=2008-04-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20080401235126/http://www.admwing.gov.hk/chi/links/aab_mem.htm |archivedate=2008-04-01 }}</ref>2007年，黎榮浩被新委任為[[消費者委員會|消費者委員會]]委員<ref>[http://www.info.gov.hk/gia/general/200710/26/P200710260094.htm]</ref>。

2008年，他和[[陳鑑林|陳鑑林]]、[[陳曼琪|陳曼琪]]、[[洪錦鉉|洪錦鉉]]組成四人名單參加[[2008年香港立法會選舉|2008年香港立法會選舉]]的九龍東直選，最後陳鑑林當選，而黎榮浩等人則落選。

2012年，他再次和[[陳鑑林|陳鑑林]]、[[洪錦鉉|洪錦鉉]]、[[柯創盛|柯創盛]]組成四人名單參加[[2012年香港立法會選舉|2012年香港立法會選舉]]的九龍東直選，最後陳鑑林當選，而黎榮浩則未能取得議席。

2016年，他再次和[[柯創盛|柯創盛]]、[[張琪騰|張琪騰]]組成三人名單參加[[2016年香港立法會選舉|2016年香港立法會選舉]]的九龍東直選，最後柯創盛當選，而黎榮浩則未能取得議席。
