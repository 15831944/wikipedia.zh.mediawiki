'''黎拿嘉博士''' 於2017年底，被聖約翰馬爾他聯盟委任為高級外交官：一級領事，亦為該組織大長官。另外身兼聖傑文瑞士大學副校長<ref>{{cite web|url=http://www.stclementsu.org|title=聖傑文瑞士大學}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> [[華南理工大學|華南理工大學]]<ref>{{cite web |url=http://www.scut.edu.cn|title=華南理工大學}}</ref>數園海外教育總監，是多所大學的客座教授. 他成立了供應鏈學會 <ref>{{cite web|url=http://www.i-scm.org/about_eng.htm|title=供應鏈學會}}</ref>,中國商業經理協會<ref>{{cite web|url=http://www.hkcbma.org|title=中國商業經理協會}}</ref>以及香港公益電視臺<ref>{{cite web|url=http://www.carehk.net|title=香港公益電視臺}}</ref>. 公益電視台的成立，旨在支持慈善團體、弱勢社群及專業學會等，提供多一個宣傳公益活動及企業社會責任的平臺；亦可表揚社會賢達對上述團體及社會的貢獻。 [[公益電視台|公益電視台]]的節目將更容易與普羅大眾一同分享，從而達到廣泛宣揚團體公益資訊及企業社會責任的目的。

他同時是[[盛世中華協會|盛世中華協會]]教育顧問<ref>{{cite web|url=http://www.gca.org.hk|title=盛世中華協會}}</ref>，這是一個為中國貧困人士以及名人成就服務的慈善機構. [[中華名人錄|中華名人錄]] <ref>{{cite web|url=http://www.chinesewhoiswho.com|title=中華名人錄}}</ref>，[[亞非發展委員會|亞非發展委員會]]，[[柬埔寨工程院|柬埔寨工程院]]，柬埔寨國王基金, 在2010年成為柬埔寨副首相榮譽顧問 <ref>{{cite web|url=http://www.nhekbunchhay.info|title=2010年成為柬埔寨副首相榮譽顧問|deadurl=yes|archiveurl=https://web.archive.org/web/20120210033419/http://www.nhekbunchhay.info/|archivedate=2012-02-10}}</ref>.亦為國際美洲大學教員會<ref>{{cite web|url=http://www.iau.la|title=國際美洲大學教員會}}</ref>，香港大學基金會<ref>{{cite web|url=http://www.hku.hk/hkuf|title=香港大學基金會}}</ref>，浸會大學發展基金會<ref>{{cite web|url=http://foundation.hkbu.edu.hk/eng/list.php|title=浸會大學發展基金會}}</ref>等組織成員。

==主要經歷==
===早期生活===
黎拿嘉博士在1959年出生於中國大陸，之後遷居[[香港|香港]]，他在一個天主教家庭中長大，從小學到中學就讀于喇沙書院<ref>{{cite web|url= http://www.lasalle.edu.hk|title=La Salle College }}</ref>，這是一所屬於天主教組織的著名書院，但是黎拿嘉不是並沒有成為一名天主教教徒，直到2011年，仍是一名無宗教信仰人士。

==教育背景==
黎拿嘉博士在香港理工大學電子工程系接受教育，並獲得研究生學歷，他現在是香港大學管理學院以及劍橋大學管理學院的校友，持有美國大學國際工商管理學博士學位。

==職業==
黎教授歷任香港上市公司[[順昌集團|順昌集團]]總經理、新加坡滿寶益集團大中華區董事總經理、積維國際集團CEO等。現任供應鏈管理學院主席，為客戶提供專業的供應鏈解決方案培訓。黎教授擁有超過20年的教育、國際商務、計算機以及專案管理經驗，善於處理不同的企業交易。 黎教授曾擔任多家公司的商業顧問和海內外多所大學、教育機構的代表或客席講師。他曾任英國劍橋大學首席考試官，港澳CEPA城香港代表，世界經理學會主席等。此外，香港大學、香港中文大學、香港理工大學、復旦大學、清華大學、浙江大學等也曾聘請他講授國際商業課程。 黎教授活躍於教育界和各個行業協會的社交圈，擔當職位很多，他現時是美國加州國際美洲大學客座教授、聖傑文瑞士大學副校長、浸會大學發展基金會員、盛世中華協會首席教育顧問、中華公益金顧問、中華國學會會長、廣東華文國學院副會長、中國商業技師協會副會長、中國商業經理協會秘書長、生命教育學院副會長、勞資關係協進會副會長，香港多間報紙如[[香港新報|香港新報]]、[[信報|信報]]，香港商報 [[教授廣場|教授廣場]]專欄作家等。他曾任專業持續教育聯盟創會副主席及香港市務學會秘書長。 至今。他的卓越貢獻包括香港浸會大學基金會發起人，可持續發展聯盟成員以及副主席，眾多大型商業相關組織都是該聯盟成員。黎拿嘉博士同時也是盛世中華協會榮譽教育顧問，這個組織在媒體以及國家政府組織中具有很大的影響力。 其本人亦擔任供應鏈協會主席，這是一個專業的物流組織，以及商業經理協會秘書長，商業經理協會是中國商業管理最大的認證機構。他同時受邀為許多報刊寫專欄，包括在香港3大報刊中具有專家專欄，這些報刊被允許在中國大陸發行，從2008年開始，黎拿嘉博士為香港商報專欄寫稿，他對貿易教育體系以及專業發展提出許多犀利有見地見解與評論。同時為哈佛大學商業評論顧問委員會成員<ref>{{cite web|url=http://zh.wikipedia.org/wiki/%E5%93%88%E4%BD%9B%E5%95%86%E6%A5%AD%E8%A9%95%E8%AB%96|title=哈佛大學商業評論顧問委員會成員}}</ref>.

==社會服務 ==
黎拿嘉博士致力於服務貧困及弱勢團體，並擔任[[勞動關係|勞動關係]]委員會副主席，這是一個由Rev Father McGovern S.J.成立的關注勞動關係的團體，立法會成員，是一個為社會弱勢群體以及貧困團體服務的服務者<ref>{{cite web|url=http://archives.catholic.org.hk/memory/P-McGovern.htm|title=Rev. McGOVERN, Patrick Terence SJ|date=February 16, 2006|deadurl=yes|archiveurl=https://web.archive.org/web/20090212233158/http://archives.catholic.org.hk/memory/P-McGovern.htm|archivedate=二月 12, 2009}}</ref>黎拿嘉博士在20-30歲時，成為[[香港民主黨|香港民主黨]]創辦成員之一.
黎拿嘉教授是[[生命教育學會|生命教育學會]]副主席，並花費數十年時間在社會中尋找真正的生活價值。他說生命真正的價值不在於我們得到了什麼，而在於我們能給出什麼，他已經獲得了兩項蒂獎，並與所有的人一起享受與分享生活。<ref>{{cite web |url=http://www.lifeeducationacademy.org/teacher.htm |title=生命教育學會副主席 |deadurl=yes |archiveurl=https://web.archive.org/web/20090106120118/http://www.lifeeducationacademy.org/teacher.htm |archivedate=2009-01-06 }}</ref>.黎拿嘉博士成立了在IPTV，網路以及手機上運行的企業社會責任頻道，該頻道促進了慈善服務以及褒獎在社會責任上具有傑出表現的企業團體以及個人。

透過成立公益電視台www.carehk.tv, 支持弱勢團體, 協助成立企業慈善基金. 支持山區教學多年. 教育山區老師, 校長等. 得清華大學扶貧辦公室在全國聯播。為一些中小型慈善機構募經費的同事，亦提供平臺讓企業一盡社會責任，互惠互利。<ref>{{cite web |url=http://www.hkcbma.org/31-8-2012/HA16820CEPA.Pdf|title=慈善推手黎拿嘉}}</ref>.
