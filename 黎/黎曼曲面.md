[[数学|数学]]上，特别是在[[复分析|复分析]]中，一个'''黎曼曲面'''是一个一维[[复流形|复流形]]。黎曼曲面可以被視为是一个[[复平面|复平面]]的变形版本：在每一点局部看来，他们就像一片复平面，但整体的[[拓扑|拓扑]]可能极为不同。例如，他们可以看起来像[[球|球]]或是环，或者两个页面粘在一起。

黎曼曲面的精髓在于在曲面之间可以定义[[全纯函数|全纯函数]]。黎曼曲面现在被认为是研究这些函数的整体行为的自然选择，特别是像[[平方根|平方根]]和[[自然对数|自然对数]]这样的[[多值函數|多值函數]]。

每个黎曼曲面都是二维实解析[[流形|流形]]（也就是[[曲面|曲面]]），但它有更多的结构（特别是一个[[複結構|複結構]]），因为全純函数的无歧义的定义需要用到这些结构。一个实二维流形可以变成为一个黎曼曲面（通常有几种不同的方式）当且仅当它是[[可定向|可定向]]的。所以球和环有複結構，但是[[莫比乌斯带|莫比乌斯带]]，[[克莱因瓶|克莱因瓶]]和[[射影平面|射影平面]]没有。

黎曼曲面的几何性质是最妙的，它们也给與其它曲线，流形或簇上的推广提供了直观的理解和动力。[[黎曼-罗赫定理|黎曼-罗赫定理]]就是这种影响的最佳例子。

== 形式化定义 ==
令''X''为一个[[豪斯多夫空间|豪斯多夫空间]]。一个从开子集''U''⊂''C''到'''X'''的子集的[[同胚|同胚]]称为[[坐标卡|坐标卡]]。两个有重叠区域的坐标卡''f''和''g''称为相容的，如果映射''f'' o ''g''<sup>-1</sup>和''g'' o ''f''<sup>-1</sup>是在定义域上[[全纯|全纯]]的。若''A''一组相容的图，并且每个''X''中的''x''都在某个''f''的定义域中，则称''A''为一个[[图册_(拓扑学)|图册]]'。当我们赋予''X''一个图册''A'',我们称(''X'',''A'')为一个黎曼曲面。如果知道有图册，我们简称''X''为黎曼曲面。

不同的图册可以在''X''上给出本质上相同的黎曼曲面结构；为避免这种模糊性，我们有时候要求''X''为''极大''的，也就是它不是任何一个更大的图集的子集。根据[[佐恩引理|佐恩引理]]每个图集''A''包含于一个唯一的最大图集中。

== 例子 ==

*[[复平面|复平面]]'''C'''可能是最平凡的黎曼曲面了。映射''f''(''z'') = ''z''（恒等映射）定义了'''C'''的一个图，而{''f''}是'''C'''的一个图集。映射''g''(''z'') = ''z<sup>*</sup>''（[[复数共轭|共轭]]）映射也定义了'''C'''的一个图而{''g''}也是'''C'''的一个图集。图''f''和''g''不相容，所以他们各自给了'''C'''一个黎曼曲面结构。事实上，给定黎曼曲面''X''及其图集''A''，共轭图集''B'' = {''f<sup>*</sup>'' : ''f'' ∈ A}总是不和''A''相容，因此赋予''X''一个不同的黎曼曲面结构。

*类似的，每个复平面的[[开子集|开子集]]可以自然的视为黎曼曲面。更一般的，每个黎曼曲面的开子集是一个黎曼曲面。

*令''S'' = '''C''' ∪ {∞}并令''f''(''z'') = ''z''其中''z''属于''S'' \ {∞}并且令''g''(''z'') = 1 / ''z''其中''z''属于''S'' \ {0}以及定义1/∞为0.则''f''和''g''为图，它们相容，而{ ''f'', ''g'' }是''S''图集，使''S''成为黎曼曲面。这个特殊的曲面称为'''[[黎曼球|黎曼球]]'''因为它可以解释为把复平面裹在一个球上。不像复平面，它是一个[[紧空间|紧空间]]。

*紧黎曼曲面可以视为和定义在复数上的非奇异[[代数曲线|代数曲线]]等效。非紧黎曼曲面的重要例子由解析连续给出（见下面）

== 属性和更多的定义 ==

两个黎曼曲面''M''和''N''之间的[[函数|函数]]''f'' : ''M'' → ''N''称为全纯，如果对于''M''的图集中的每个图''g''和''N''的图集中的每个图''h''，映射''h'' o ''f'' o ''g''<sup>-1</sup>在所有有定义的地方是全纯的（作为从'''C'''到'''C'''的函数）。两个全纯函数的複合是全纯的。两个黎曼曲面''M''和''N''称为'''保角等价'''（或'''共形等价'''），如果存在一个[[双射|双射]]的从''M''到''N''的全纯函数并且其逆也是全纯的（最后一个条件是自动满足的所以可以略去）。两个保角等价的黎曼曲面对于所有的实际应用来讲是完全相同的。

每个[[单连通|单连通]]的黎曼曲面和'''C'''或黎曼球'''C''' ∪ {∞}或开圆盘{''z'' ∈ '''C''' : |''z''| < 1}保角等价。这个命题称为[[单值化定理|单值化定理]]。

每个连通黎曼曲面可以转成有常数[[曲率|曲率]]-1，0或1的[[完备|完备]]实[[黎曼流形|黎曼流形]]。这个黎曼结构除了度量的缩放外是唯一。有曲率-1的黎曼曲面称为'''双曲'''的;开圆盘是个经典的例子。有曲率0的黎曼曲面称为'''抛物'''的；'''C'''是典型的抛物黎曼曲面。最后，有曲率+1的黎曼曲面称为'''椭圆'''的；[[黎曼球|黎曼球]]'''C''' ∪ {∞}是这样的一个例子。

对于每个闭抛物黎曼曲面，[[基本群|基本群]]同构于2阶[[格群|格群]]，因而曲面可以构造为'''C'''/Γ,其中'''C'''是複平面而Γ是格群。陪集的代表的集合叫做[[基本域|基本域]]。

类似的，对每个双曲黎曼曲面，基本群同构于[[富克斯群|富克斯群]]，因而曲面可以由[[富克斯模型|富克斯模型]]'''H'''/Γ构造，其中'''H'''是[[上半平面|上半平面]]而Γ是富克斯群。'''H'''/Γ陪集的代表是[[自由正则集|自由正则集]]，可以作为度量[[基本多边形|基本多边形]]。

当一个双曲曲面是紧的，则曲面的总面积是<math>4\pi(g-1)</math>,其中''g''是曲面的[[亏格|亏格]]；面积可由把[[高斯-博内定理|高斯-博内定理]]应用到基本多边形的面积上来算出。

前面我们提到黎曼曲面，象所有複流形，象实流形一样[[可定向|可定向]]。因为複图''f''和''g''有变换函数''h'' = ''f''(''g''<sup>-1</sup>(''z''))，我们可以认为''h''是从'''R'''<sup>2</sup>开集到'''R'''<sup>2</sup>的映射，在点''z''的[[雅可比矩阵|雅可比矩阵]]也就是由乘以複數''h'(z)''的运算给出的实线性变换。但是，乘以複數α的[[行列式|行列式]]等于|α|^2,所以''h''的雅可比阵有正的行列式值。所以，複图集是可定向图集。
