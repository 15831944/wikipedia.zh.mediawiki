'''黎族'''是[[华南地区|中国南方]][[少数民族|少数民族]]，[[中华人民共和国第五次全国人口普查#民族构成|列中国大陆第18大民族]]。[[海南省|海南]]是黎族世居地，除海南以外，现代黎族聚居区分布于中国大陆[[贵州省|贵州]]、[[广东省|广东]]、[[广西壮族自治区|广西]]和[[江西省|江西]]等省区。根据[[中华人民共和国第五次全国人口普查#民族构成|2000年人口普查]]，中华地区黎族人口有1,248,022人，其中海南有1,172,181人，占黎族人口的93.9%列第一位，其次是中国大陆贵州省，有56,082人占黎族人口的4.5%列第二位，但實際上貴州的“黎族”並非民族學意義上的黎族，而是一支來源和黎族無關的少數族群“里民子”，只是因為他稱接近被劃爲黎族；中国大陆广东、广西、[[浙江省|浙江]]和[[云南省|云南]]四省黎族人口超过千人，依次列第三至第六位，其余[[中国一级行政区|各个省份]]及现役军人均有黎族人口分布。

== 历史 ==
由古代[[民族|民族]][[俚人|俚人]]的其中以[[海南|海南]]为中心活动的那一支，漢代的[[儋耳|儋耳]]、[[珠崖|珠崖]]是他们祖先，演变成今天的'''黎族'''。

== 黎族人口分布 ==
中国有黎族1,248,022人（2000年人口普查），黎族聚居区集中分布于[[海南|海南]]，中国大陆其他[[中国一级行政区|省份]]均有黎族人口分布，其中[[广东省民族构成列表|广东省]]的[[雷州半岛|雷州半岛]]以及[[贵州省民族构成列表|贵州省]]有黎族杂居，其余地区，黎族以散居为主。海南有黎族1,172,181人（2000年人口普查），占海南人口的15.5%，占中华地区黎族人口的93.9%，[[海南省民族构成列表|列海南第一大少数民族]]。

== 民族服饰 ==
[[File:Art_du_tissage_des_Li.jpg|thumb]]
黎族服装以棉麻为原料，多穿对襟无领上衣。黎族妇女喜梳发髻于脑后，用的发簪是箭猪毛、牛骨或金属制成。女子下穿织绣花纹的[[筒裙|筒裙]]，裙上绣有各种图案。遇重要节日或婚丧等要着盛装，妇女要戴项圈、手镯、脚镯、二环等银质饰品。有些地方的[[妇女|妇女]]戴很重的耳环，[[使耳双垂肩|使耳双垂肩]]，称为“儋耳”。

黎族男子也结髻缠头，下穿两幅布做成的“吊幨”。

部份地区还有纹面、纹身的风俗，古代称为“雕题”。
