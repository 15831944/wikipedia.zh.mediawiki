'''杨安泽'''{{efn|1=中文译名参考自[[美国之音|美国之音]]报道《[https://www.voachinese.com/a/andrew-yang-20180326/4316746.html 华裔总统参选人杨安泽接受VOA专访]》。}}（{{Lang-en|Andrew Yang}}，{{bd|1975年|1月13日|||catIdx=杨}}）是[[美国|美国]]一名[[企业家|企业家]]、{{Link-en|为美国创业|Venture for America}}{{efn|1=中文译名参考自杨安泽竞选网站[https://cn.yang2020.com/about_andrew/vfa/ 官方文宣资料]。}}组织创始人、[[2020年美國總統選舉|2020年美国总统选举民主党初选候选人]]。<ref name="voa">{{cite web |author1=方冰 |author2=章真 |title=华裔总统参选人杨安泽接受VOA专访 |url=https://www.voachinese.com/a/andrew-yang-20180326/4316746.html |publisher=[[美国之音|美国之音]] |accessdate=2019-03-09}}</ref><ref name="nytimes20180210">{{cite web |author1=Kevin Roose |title=His 2020 Campaign Message: The Robots Are Coming |url=https://www.nytimes.com/2018/02/10/technology/his-2020-campaign-message-the-robots-are-coming.html |publisher=[[纽约时报|纽约时报]] |accessdate=2019-03-09}}</ref> 杨安泽是美国近半个世纪以来首个宣布参与[[美国总统选举|总统大选]]的[[华裔|华裔]]，<ref name="voa" />在他{{Link-en|杨安泽2020年总统竞选|Andrew Yang 2020 presidential campaign|争取2020年美国总统提名}}时，杨安泽提出的主要竞选目标之一就是提高每个美国成年人的[[無條件基本收入|基本收入]]以应对自动化快速发展而对劳动力市场的挑战。<ref name="nytimes20180210" /><ref>{{cite web |author1=Catherine Clifford |title=This 43-year-old running for president in 2020 wants to give everyone $1,000 a month in free cash |url=https://www.cnbc.com/2018/04/11/andrew-yang-wants-to-run-for-president-promising-free-cash-handouts.html |publisher=[[CNBC|CNBC]] |accessdate=2019-03-09}}</ref>

== 早年生活 ==
杨安泽出生于美国的[[斯克内克塔迪_(纽约州)|斯克内克塔迪]]，<ref>{{cite web |author1=Steve Hughes |title=Schenectady native stumps for president in Latham |url=https://www.timesunion.com/news/article/Schenectady-native-stumps-for-president-in-Latham-13512632.php |publisher={{Link-en|时代联盟报 (奥尔巴尼)|Times Union (Albany)|时代联盟报}} |accessdate=2019-03-09}}</ref> 他的家庭是一个来自[[台湾|台湾]]的移民家庭。<ref name="voa" /> 他的父母是在[[加州大学伯克利分校|加州大学伯克利分校]]研究生院就读的时候相遇的，他的父亲获得了[[物理学|物理学]]领域的[[博士学位|博士学位]]，并分别在[[IBM|IBM]]和[[通用电气|通用电气]]的研究实验室工作且在其职业生涯获得了89个专利；而她的母亲则获得了[[统计学|统计学]]的[[硕士学位|硕士学位]]，但她随后却成为了一名艺术家。<ref>{{cite web |author1=Joe Rogan |title=The Joe Rogan Experience #1245 - Andrew Yang |url=https://vimeo.com/316922117 |website=[[Vimeo|Vimeo]] |accessdate=2019-03-09}}</ref>

杨安泽曾就读于[[新罕布什尔州|新罕布什尔州]]的一所[[寄宿学校|寄宿制学校]]——[[菲利普斯埃克塞特学院|菲利普斯埃克塞特学院]]，并于1992年从该校毕业。<ref name="nytimes2013">{{cite web |author1=Hannah Seligson |title=No Six-Figure Pay, but Making a Difference |url=https://www.nytimes.com/2013/07/14/business/no-six-figure-pay-but-making-a-difference.html |publisher=[[纽约时报|纽约时报]] |accessdate=2019-03-09}}</ref> 他随后进入[[布朗大学|布朗大学]]就读经济系，并获得[[文学士|文学士]]学位。<ref name="cleveland">{{cite web |author1=Robert L. Smith |title=Andrew Yang, Venture for America founder, will help showcase Cleveland's startup scene |url=https://www.cleveland.com/business/2013/09/andrew_yang_venture_for_americ.html |publisher=[[诚恳家日报|诚恳家日报]] |accessdate=2019-03-09}}</ref> 从布朗大学毕业之后，杨安泽又进入[[哥伦比亚法学院|哥伦比亚法学院]]就读，并在这里获得了[[法律博士|法律博士]]的学位。<ref name="cleveland" />

== 职业生涯 ==
1999年，杨安泽从哥伦比亚法学院毕业之后在[[纽约市|纽约市]]的{{Link-en|达维律师事务所|Davis Polk & Wardwell}}开始他的律师职业生涯。<ref>{{cite web |author1=陳玲銖 |title=人物｜少年楊安澤 聰穎、搞怪、好打不平 |url=https://www.worldjournal.com/5948302/article-%E4%BA%BA%E7%89%A9%EF%BD%9C%E5%B0%91%E5%B9%B4%E6%A5%8A%E5%AE%89%E6%BE%A4-%E8%81%B0%E7%A9%8E%E3%80%81%E6%90%9E%E6%80%AA%E3%80%81%E5%A5%BD%E6%89%93%E4%B8%8D%E5%B9%B3/ |publisher=[[北美世界日報|世界日报]] |accessdate=2019-03-09}}</ref> 不过他在2000年就离开事务所并创办了一家名为{{lang|en|Stargiving.com}}的网站，这家网站是一个与名人相关联的慈善筹款网站。<ref>{{cite web |author1=Eilene Zimmerman |title=The 'Teach for America' for Entrepreneurs? |url=https://www.inc.com/articles/201107/andrew-yang-venture-for-america.html |publisher={{Link-en|公司 (杂志)|Inc. (magazine)|公司}} |accessdate=2019-03-09}}</ref> {{lang|en|Stargiving.com}}起初从投资者手中获得了一些投资，但还是在2001年关闭了。<ref>{{cite web |author1=杨安泽 |title=The uncomfortable relationship between entrepreneurship and privilege |url=https://www.linkedin.com/pulse/entrepreneurship-privilege-andrew-yang |publisher=[[领英|领英]] |accessdate=2019-03-09}}</ref><ref name="wnd">{{cite web |title=Dem presidential candidate to run on 'Universal Basic Income' |url=https://www.wnd.com/2018/02/dem-presidential-candidate-to-run-on-universal-basic-income/ |publisher={{Link-en|世界网络日报|WorldNetDaily}}}}</ref> 之后，杨安泽加入了一家从事医疗保健软件的创业公司——{{lang|en|MMF System}}。他在公司担任副总裁，这是他的第三次就业。<ref name="wnd" />

=== 曼哈顿备考 ===
在医疗保健软件行业工作四年之后，杨安泽离开了{{lang|en|MMF System}}公司。随后，他加入了由其朋友泽科·范德霍克（{{lang|en|Zeke Vanderhoek}}）所创办的一家小型备考公司——{{Link-en|曼哈顿备考|Manhattan Prep}}。2006年，在范德霍克的要求下，杨安泽出任这家公司的[[首席执行官|首席执行官]]。在担任曼哈顿备考的首席执行官期间，杨安泽给公司制定的经营策略就是主要针对[[GMAT|研究生管理科入学考试]]提供备考服务。在他的经营之下，曼哈顿备考从最初的5个门店扩展到69个门店，并于2009年12月被{{Link-en|卡普兰公司|Kaplan, Inc.}}所收购。2012年，杨安泽辞去该公司的[[总裁|总裁]]职务。<ref>{{cite news |last=Cheney |first=Steve |url=http://stevecheney.com/the-evolution-of-education-kaplan-acquires-ma/ |title=The Evolution of Education – Kaplan acquires Manhattan GMAT |work=steve cheney – technology, business & strategy |date=2010-01-27 |deadurl=no |archiveurl=https://web.archive.org/web/20130813040118/http://stevecheney.com/the-evolution-of-education-kaplan-acquires-ma/ |archivedate=2013-08-13 |accessdate=2019-03-09 }}</ref><ref>{{cite web |author1=Emily Glazer |title=For Grads Seeking to Work and Do Good |url=https://www.wsj.com/articles/SB10001424052970204257504577152902147107614 |publisher=[[华尔街日报|华尔街日报]] |accessdate=2019-03-09}}</ref><ref name="nytimesblog">{{cite web |author1=Jessica Bruder |title=Starting a Teach for America for Entrepreneurs |url=https://boss.blogs.nytimes.com/2011/10/12/starting-a-teach-for-america-for-entrepreneurs/ |website=[[纽约时报|纽约时报网博客频道]] |publisher=You're the Boss |accessdate=2019-03-09}}</ref>

=== 为美国创业 ===
在曼哈顿备考于2009年年底被收购之后，杨安泽开始致力于建立一个名为“{{Link-en|为美国创业|Venture for America}}”的[[非营利性组织|非营利性]][[奖学金|奖学金]]计划。该计划最终于2011年成立，其目的旨在“通过动员美国下一代企业家在美国城市创业来为美国制造就业机会，并为这些创业者提供创造就业所需的技能与资源。”<ref name="nytimes2013" /><ref name="sbn">{{cite web |author1=Julia Ballard |title=ANDREW YANG STEPS DOWN AS VENTURE FOR AMERICA CEO |url=http://siliconbayounews.com/2017/03/29/andrew-yang-steps-venture-america-ceo/ |publisher=硅湾新闻 |accessdate=2019-03-09}}</ref><ref name="whitehouse">{{cite web |author1=Jon Carson |title=Celebrating a Year of Champions of Change – President Obama Meets with 12 Champions Who Are Making a Difference in Their Communities |url=https://obamawhitehouse.archives.gov/blog/2012/04/27/celebrating-year-champions-change-president-obama-meets-12-champions-who-are-making- |publisher=[[美国白宫|美国白宫]] |accessdate=2019-03-09}}</ref><ref name="vfa">{{cite web |title=Our Mission & Approach |url=https://ventureforamerica.org/about-us/our-mission-approach/ |publisher=为美国创业 |accessdate=2019-03-09}}</ref>

“为美国创业”开始的时候只有20万美元的资金，但他们还是在2012年和2013年分别培训了40名和69名毕业生，并将他们送到[[巴尔的摩|巴尔的摩]]、[[辛辛那提|辛辛那提]]、[[克利夫兰|克利夫兰]]、[[底特律|底特律]]、[[拉斯维加斯|拉斯维加斯]]、[[新奥尔良|新奥尔良]]、[[费城|费城]]和[[普罗维登斯_(罗德岛州)|普罗维登斯]]等地创业。而到了2014年，毕业生的人数上升到106人，而创业的城市名单也扩大至[[哥伦布_(俄亥俄州)|哥伦布]]、[[迈阿密|迈阿密]]、[[圣安东尼奥|圣安东尼奥]]和[[圣路易斯_(密苏里州)|圣路易斯]]等城市。<ref name="nytimesblog" /><ref>{{cite web |author1=Tom Walsh |title=Venture for America start-up program takes a shine to Detroit |url=https://www.freep.com/story/money/business/columnists/tom-walsh/2014/08/17/tom-walsh-venture-for-america-start-up-program-takes-a-shine-to-detroit/14134321/ |publisher={{Link-en|底特律自由报|Detroit Free Press}} |accessdate=2019-03-09}}</ref>

“为美国创业”的培训计划是将全美顶尖大学的优秀毕业生招募到一个为期两年的奖学金计划中，他们将在美国那些发展中城市里有潜力的创业公司里工作或担任学徒。杨安泽在他的《聪明人应创业（{{lang|en|Smart People Should Build Things}}）》的书里认为，这个国家（美国）的顶尖大学是从小城镇里挑选最聪明的孩子，并将他们汇集到同一个大城市的同一个企业去工作。<ref>{{cite web |author1=Ashley May |title=A Book in 5 Minutes: Smart People Should Build Things |url=https://tech.co/news/book-5-minutes-smart-people-build-things-2014-09 |publisher=TechCo |accessdate=2019-03-09}}</ref> 而“为美国创业”的目的则是在全美范围内重新分配这些人才，并激励他们创业以促进经济的增长。

2011年之后，“为美国创业”进入发展期；而到了2017年，这个计划已经拥有高达600万美元的年度预算。<ref>{{cite web |title=Venture For America Financials |url=https://ventureforamerica.org/about-us/financials/ |publisher=为美国创业 |accessdate=2019-03-09}}</ref> 而与此同时，其扶持创业的城市也增加到了14个，这其中包括[[巴尔的摩|巴尔的摩]]、[[辛辛那提|辛辛那提]]、[[克利夫兰|克利夫兰]]、[[底特律|底特律]]、[[新奥尔良|新奥尔良]]、[[费城|费城]]、[[哥伦布_(俄亥俄州)|哥伦布]]、[[迈阿密|迈阿密]]、[[圣安东尼奥|圣安东尼奥]]、[[圣路易斯_(密苏里州)|圣路易斯]]、[[堪萨斯城_(密苏里州)|堪萨斯城]]、[[伯明翰_(阿拉巴马州)|伯明翰]]、[[夏洛特_(北卡羅來納州)|夏洛特]]和[[匹兹堡|匹兹堡]]。<ref>{{cite web |title=Where We Work |url=https://ventureforamerica.org/where-fellows-work/ |publisher=为美国创业 |accessdate=2019-03-09}}</ref> 目前，“为美国创业”已经开始在底特律运行“[[种子加速器|种子加速器]]”，这将为计划参与者提供[[种子基金|种子基金]]或[[投资基金|投资基金]]。

《创新时代（{{lang|en|Generation Startup}}）》是一部讲述位于底特律而通过“为美国创业”计划孵育的6家创业公司故事的[[纪录片|纪录片]]。该影片于2016年公映，由{{Link-en|辛西娅·韦德|Cynthia Wade}}与彻丽尔·米勒·豪斯尓（{{lang|en|Cheryl Miller Houser}}）联合执导。<ref>{{cite web |title=Generation Startup |url=http://www.generationstartupthefilm.com/ |publisher=Generation Startup Official Website |accessdate=2019-03-09}}</ref>

2017年3月，杨安泽辞去了“为美国创业”的首席运营官一职。<ref name="sbn" />

== 竞选总统 ==
[[File:Andrew-Yang-Champion-of-Change.jpg|300px]]于2012年在[[白宫|白宫]]的会面。]]

2017年11月6日，杨安泽向[[联邦选举委员会|联邦选举委员会]]提出申请，要求参与2020年[[美国总统选举|美国总统选举]]。<ref>{{cite web |title=Statement of Candidacy |url=http://docquery.fec.gov/pdf/624/201711069086611624/201711069086611624.pdf |publisher=[[联邦选举委员会|联邦选举委员会]] |accessdate=2019-03-09}}</ref> 他在竞选承诺里提出向每个年满18周岁的美国公民提供每月1000美元的“自由津贴（{{lang|en|Freedom Dividend}}）”；并对其他因技术现代化而预测将出现大规模失业的民众亦有其他回应。他在自己的竞选的网站上写道，“每个18周岁以上的美国公民都将每月获得1000美元，而不管其收入或就业状况如何，这是自由而明确的。”<ref>{{cite web |author1=Chelsea Gohd |title=Meet the Long-Shot 2020 Presidential Candidate Who Might Make UBI a Reality |url=https://futurism.com/2020-presidential-candidate-ubi/ |publisher=Futurism.com |accessdate=2019-03-09}}</ref><ref>{{cite web |author1=Luke Christou |title=Andrew Yang 2020?
