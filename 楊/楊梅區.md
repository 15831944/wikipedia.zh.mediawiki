'''楊梅區'''（[[臺灣客家語|臺灣客家語]][[四縣腔|四縣腔]]：iongˇ moiˇ kiˊ；[[海陸腔|海陸腔]]：rhong moi kiˋ；[[饒平腔|饒平腔]]：rhiong moi kiˇ）位於[[中華民國|中華民國]][[桃園市|桃園市]]南部，現今最大住民族群為[[臺灣客家人|客家]]族群。可分為楊梅、[[埔心_(桃園市)|埔心]]、[[伯公岡台地|富岡]]、[[楊梅高山頂|高山頂]]四區域，境內有幼獅工業區、楊梅工業區、秀才科技園區，民富工業區。人口約17萬餘人<ref>[http://www.tycg.gov.tw/cab/home.jsp?id=175&parentpath=0,24 人口統計-桃園市政府民政局]</ref>。境內以客家人為主要族群，約占全區人口數七成左右<ref>{{cite web | url=http://ebooks.lib.ntu.edu.tw/1_file/hakka/78/51272219271.pdf | title=各鄉鎮客家人口統計 |}}</ref>。

== 地名由來 ==
楊梅區舊名「楊梅壢」，「壢」在[[臺灣客家語|客家語]]裡是「[[台地|台地]]間的河谷低地」的意思，[[臺灣清治時期|清代]][[漢人|漢人]]先後移民到這裡，看到滿山遍野都是野生的[[楊梅|楊梅]]樹，風景美麗宜人，所以把這裡稱為楊梅壢，而楊梅樹成為楊梅區的「區樹」。

== 地理、歷史 ==
[[File:楊梅市次分區.PNG|thumb]]
[[File:楊梅市區夜景.jpg|thumb]]

楊梅區輪廓略成長方形，東西距大於南北距。境內南北地貌不同：北部是兩塊向北緩降的單面山台地，[[楊梅高山頂|高山頂]]在東，[[楊梅長岡嶺|長岡嶺]]在西，其間有個缺口，桃園市境內重要河川[[社子溪|社子溪]]由此向北流向[[新屋區|新屋區]]，注入[[臺灣海峽|臺灣海峽]]。台地靠南的邊緣較陡，頂上頗為平坦，平均[[海拔|海拔]]約為200公尺左右。

中部是一塊略呈東南－西北向的長形[[盆地|盆地]]，楊梅市街位在中央偏東；南部是[[店子湖台地|店子湖台地]]，與龍潭區和新竹縣新埔鎮的台地地形相接，平均海拔在300公尺以上。

楊梅全區面積89.12平方公里，主要居住聚落集中在[[埔心_(桃園市)|埔心]]、楊梅、富岡、[[高山頂|高山頂]]四大區域。

楊梅最早成立的[[中心商業區|市街]]位於伯公山周圍，又被當地人稱為'''老街'''，楊梅歷史最悠久的錫福宮便位於老街上。自古以來，楊梅一直都是[[北臺灣|北臺灣]]的交通要衝，鐵公路交通均十分便利，楊梅區目前便有[[楊梅車站|楊梅]]、[[埔心車站|埔心]]、[[富岡車站|富岡]]、[[新富車站|新富]]共4個火車站。其中，楊梅火車站當年最早設立的位置是在大華街的衛生所舊址，衛生所前面的階梯其實就是當時的[[月台|月台]]。1929年，鐵路改道並舖設複線，於是便將[[楊梅車站|楊梅車站]]往北遷移到現在的'''新街'''。

楊梅市街的信仰中心為[[錫福宮|錫福宮]]，供奉[[客家地區|客家-{庄}-]]的保護神「[[三官大帝|三官大帝]]」。位於[[楊梅高山頂|高山頂]]的[[楊梅高山頂啟明宮|啟明宮]]亦供奉[[三官大帝|三官大帝]]。

楊梅區青山二街附近的[[土牛溝|土牛溝]]經桃園市文化資產審議委員會到場會勘，確定這段土牛溝為臺灣從南到北僅存的遺跡，2011年登錄為桃園市文化景觀<ref>[http://www.libertytimes.com.tw/2012/new/may/13/today-north3.htm?Slots=TPhoto 《清代漢番界線》楊梅發現土牛溝 全台僅存] {{webarchive|url=https://archive.is/20130418210223/http://www.libertytimes.com.tw/2012/new/may/13/today-north3.htm?Slots=TPhoto |date=2013-04-18 }}</ref>。

楊梅戰後設為[[鎮_(中華民國)|鎮]]，2010年5月因人口超過15萬，於同年8月1日改制為[[縣轄市|縣轄市]]<ref>{{cite news|url=http://news.chinatimes.com/domestic/0,5248,11050609x112010060200115,00.html |title=人口破15萬 楊梅鎮升格為市 |author=黃文-{杰}- |date=2010年6月2日 |archiveurl=https://archive.is/20100605212949/http://news.chinatimes.com/domestic/0,5248,11050609x112010060200115,00.html |archivedate=2010年6月5日 |deadurl=yes }}</ref>。

2014年12月25日因桃園縣升格直轄市，原縣轄楊梅市改制為楊梅區。
