'''楊幽幽'''（{{lang-en|Young Yau Yau Cecilia}}），筆名'''張永民'''<ref name="CeciliaYoung2">{{cite web|author=陳奕南|url=https://www.slideshare.net/CeciliaYoung2/ss-115181021|title=牙齒保健與美容 作者：張永民 (楊幽幽)|access-date=2018-11-22 |language=zh-hant }}</ref>，[[香港|香港]][[牙醫|牙醫]]、[[專欄作家|專欄作家]]。

==簡歷==
楊幽幽於[[香港大學|香港大學]][[香港大學牙醫學院|牙醫學院及醫學院公共衛生碩士]]畢業，曾為香港大學牙醫學生會會長，香港牙醫學會會員，現為牙科醫生，亦為40本國際醫學，牙科，生物及各類學術期刊編輯及評論閲卷員。

于1995年楊幽幽被十位業内人士提名參選[[1995年香港立法局選舉|1995年香港立法局選舉]]醫學界[[功能組別|功能組別]]，挑戰在前一屆選舉[[自動當選|自動當選]]的原任[[梁智鴻|梁智鴻]]醫生。其政綱主要希望政府供款於強積金，後於選舉中共得135票落敗<ref>{{Cite book |title=港式法團主義－－功能界別25年 |authorlink=馬嶽 |isbn=9789629372194}}</ref>。

楊曾為《[[信報|信報]]》<ref>{{cite web|url=http://www.h2osteamwatericeforum.com/%E6%A5%8A%E5%B9%BD%E5%B9%BD-%E9%86%AB%E7%94%9F-%E4%BF%A1%E5%A0%B1%E8%B2%A1%E7%B6%93%E6%96%B0%E8%81%9E-%E5%B0%88%E6%AC%84%E3%80%8A%E5%81%A5%E5%BA%B7%E7%89%99%E9%BD%92%E4%BA%BA%E8%A6%8B%E4%BA%BA/|title=楊幽幽 醫生 Dr. Cecilia Young 信報財經新聞 專欄《健康牙齒人見人愛》|date=2018-06-12|publisher=}}</ref>，《健康創富》<ref>{{cite web|url=http://www.h2osteamwatericeforum.com/%E7%89%99%E5%A5%97-%E3%80%8A%E5%81%A5%E5%BA%B7%E5%89%B5%E5%AF%8C%E9%9B%9C%E8%AA%8C%E3%80%8B%E7%AC%AC66%E6%9C%9F-%E5%8E%9F%E4%BE%86%E5%A6%82%E9%BD%92%EF%BC%8D%E4%BB%A5%E7%89%99%E5%A5%97%E4%BF%AE/|title=牙套 《健康創富雜誌》第66期 原來如齒－以 牙套 修補牙齒發育不整|date=2018-06-12|publisher=}}</ref>等撰寫專欄。自2003年起至今為《醫‧藥‧人》<ref>{{cite web|url=http://www.3phk.com/|title=醫藥人 醫藥健康雜誌|website=www.3phk.com}}</ref>撰寫[[醫學|醫學]]文章，內容主要是口腔衛生。其著作包括<ref>{{cite web|url=https://www.slideshare.net/CeciliaYoung2/ss-115181650|title=兒童護齒每事問 作者：楊幽幽|website=www.slideshare.net}}</ref>《兒童護齒每事問》及以筆名'''張永民'''所寫的《牙齒保健與美容—牙醫問答錄》<ref name="CeciliaYoung2"></ref>。

==爭議==
2004年4月至2005年6月期間，楊幽幽曾被指沒有為病人麥寶玲提供合適的治療計劃，並在診斷不成功時，沒有將病人轉介診治。楊幽幽就此案被牙醫管理委員會裁定專業失德，停牌一個月，緩刑一年。2010年3月楊幽幽不服，透過司法覆核推翻裁決。事後牙管會上訴，上訴庭12月裁定牙管會上訴得直，下令恢復楊的裁決。<ref>{{Cite web|url=http://hk.apple.nextmedia.com/news/art/20080527/11157822|title=涉箍壞空姐牙 楊幽幽自辯|accessdate=2018-12-30|work=Apple Daily 蘋果日報}}</ref><ref name="2010apple">{{Cite web|url=http://hk.appledaily.com/news/art/20100914/14448133|title=矯正哨牙失誤 女牙醫楊幽幽敗訴失去笑容 箍牙空姐獲償41萬|accessdate=2018-12-30|work=Apple Daily 蘋果日報}}</ref><ref>{{Cite web |url=http://the-sun.on.cc/cnt/news/20101221/00412_003.html |title=牙醫楊幽幽再停牌 |date=2010-12-21 |publisher=[[太陽報_(香港)|太陽報]] |agency=[[東方報業集團|東方報業集團]]}}</ref>2007年，港龍另一名空姐亦投訴楊專業失德。楊幽幽初犯後曾收牙醫管理委員會警告信，並要完成15小時「自願持續醫學教育課程」。<ref name="2010apple"></ref>
