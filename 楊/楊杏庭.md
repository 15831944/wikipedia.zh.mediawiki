'''楊杏庭'''（外文拼音 Hsing-ting Yang ，{{Bd|1909年|8月9日|1987年|6月4日}}），筆名逸舟（日文拼音 Isshū Yō，外文拼音 Yizhou Yang）、行東。[[臺中|臺中]][[梧棲|梧棲]]人。[[東京|東京]]文理科大學哲學科肄業，東洋文化研究所博士課程修畢。曾任教於[[南京|南京]][[國立中央大學|國立中央大學]]、[[浙江大學|浙江大學]]，並且擔任[[國民政府|國民政府]]內政部參事及專門委員，張厲生部長任內時為常務次長、教育部編審委員等要職，[[二二八事變|二二八事變]]後奉命回臺調查真相。後赴日[[東京文理科大學|東京文理科大學]]，拜{{Internal link helper/ja|小竹文夫|小竹文夫}}門下，專攻[[中國哲學|中國哲學]]及史學，並結識[[廖文毅|廖文毅]]，以易名「楊逸民」參與臺獨在日活動，被中國民國政府列入海外學者黑名單。代表著作為《歷史週期法則論》。

== 生平 ==
楊杏庭出生於[[臺中|臺中]][[清水|清水]][[梧棲|梧棲]]<ref>{{Cite web|url=http://homepage.ntu.edu.tw/~philo/Taiwan/japan/japan.htm|title=臺灣哲學研究資料庫─楊杏庭（徐子雯收集整理）|accessdate=|author=臺灣大學哲學系|date=|publisher=}}</ref>。1923年畢業自梧棲公學校，隔月考入首屆[[臺中師範學校|臺中師範學校]]。1928年秋，為學潮主謀之一，起因為中師體育教員兼舍監小岩辱罵臺灣學生而起。隔年3月，畢業後分發至龍泉公學校任教。1930年陸續赴[[東京|東京]]、[[臺北|臺北]]等參加教員檢定考試及格。1930年臺北參加修身科教員檢定考試落榜，1932年再赴東京參加高等教員檢定及格，為全日本最年輕之考取者。同年秋赴[[臺中州|臺中州]]公民科教學觀摩會授課，極為成功，但論文發表遭禁。

1934年3月辭小學教員，為隔月赴[[東京高等師範學校|東京高等師範學校]]教育研究科進修，苦學[[英文|英文]]、[[德文|德文]]、[[法文|法文]]，隔年4月由臺中[[大甲|大甲]]的上野齊醫師供給學費，入[[東京文理科大學|東京文理科大學]]。1938年學科畢業考試順利通過，於高師附中擔任公民科、修身科的實習教學，獲學生良好反應。期間亦結識文學家翁鬧，參與臺人在日文藝活動。1939年畢業，隔年受中國留學生王維哲資助赴[[南京|南京]]謀職，3月獲南京[[國民政府|國民政府]]教育部長[[顧澄|顧澄]]聘為編審委員、隨任教育部委員，[[國立編譯館|國立編譯館]]之編譯官等職，並於[[中央大學|中央大學]]開授[[日文|日文]]、[[德文|德文]]等課程。1942年3月，任[[浙江省|浙江省]]政府主席[[傅式說|傅式說]]秘書，後於[[浙江大學|浙江大學]]復校時轉任校秘書兼心理學教授，兼任浙江省財政廳與糧食局秘書。1945年終戰時遭[[國民黨|國民黨]]整肅，失業在家專心撰述《歷史週期法則論》。隔年赴任南京教育局督學。1947年[[二二八事變|二二八事變]]爆發，時任內政部委員。8月時奉內政部長[[張厲生|張厲生]]之命，返臺調查民情及省主席[[魏道明|魏道明]]貪污情勢，行政院長[[張群|張群]]據此飭令全國檢肅貪污。1948年舉家以難民返臺，隔年受財政廳長[[嚴家淦|嚴家淦]]推薦任臺灣銀行特約研究員，而教育部部長[[杭立武|杭立武]]則聘其為特約編審，為部內講授臺語課程。1950至1952年間準備留美未果，1953年赴日著手其著《歷史週期法則論》日譯，卻遭法院判決「非法居留」，後自費出版其譯著於1954年東京文理科大學，拜{{link-ja|小竹文夫|小竹文夫}}門下，專攻[[中國哲學|中國哲學]]及史學。同時參加「亞細亞懇談會」結識[[廖文毅|廖文毅]]，涉入在日臺獨活動。1955年任「臺灣臨時議會」中央委員兼宣傳部長，曾代筆[[廖文毅|廖文毅]]《臺灣民本主義》。1958年任[[名古屋|名古屋]][[南山大學|南山大學]]法國紀念建築研究所（フランスのモニュメント）之教授。期間與指導教授{{Internal link helper/ja|小竹文夫|小竹文夫}}日趨齬齟。1961年3月雖《歷史週期法則論》終出版，然博士學位因舊學制廢止等緣故落空，也正式與{{Internal link helper/ja|小竹文夫|小竹文夫}}交惡。

1968年，其自傳《受難者》完稿，1970年則陸續出版《太平洋戰爭前夜》、《臺灣與蔣介石─以二二八民變為中心》，1979年出版譯書《印緬隨軍記》。1981年夏，隨世界臺灣同鄉會遊歐。1983年出版《蔣介石評傳》。1984年美《臺灣公論報》「臺灣文化專刊」始載其自述〈不堪回首話生平〉，同年結識時任[[筑波大學|筑波大學]][[張良澤|張良澤]]教授。1987年病逝於東京都立松澤病院，由[[張良澤|張良澤]]教授整理遺稿，於1990年出版《受難者》由張良澤整理，於[[真理大學|真理大學]]臺灣文學資料館成立「楊逸舟文庫」。

== 理論與主張 ==
楊杏庭一生顛簸，輾轉多地，仍留下汗棟著述。早年論著文化教育學理論，曾於《臺灣文藝》發表〈無限否定與創造性〉（〈無限否定と創造性〉），探討[[柏格森|柏格森]]、[[海德格|海德格]]、{{Internal link helper/ja|謝斯托夫|レフ・シェストフ}}如何就論生之正反兩面的意義，從生的盲目性與暗黑性開出創造性，以[[亞里斯多德|亞里斯多德]]對生的實在基礎（{{Internal link helper/en|Hypokeimenon|Hypokeimenon}}）為起點，從生之[[實在|實在]]與科學實在的差異，討論到謝斯托夫寫作中呈現的以[[虛無主義|虛無主義]]，與「無限[[否定|否定]]」的意義互相對照，進而論證虛無主義與生命的否定面向之不同，再以[[黑格爾|黑格爾]]的辯證法，從生之否定面向的積極意義進一步去討論生之積極意義，並從在時間架構下的「生」，帶出[[海德格|海德格]]對時間的論述及其見解，將之與[[佛學|佛學]]中的[[諦觀|諦觀]]與生命觀做比較。其思想頗受[[歐陸哲學|歐陸哲學]]及佛學之影響。其著述《文化教育哲學概論》概介德國思想之教養論，主張教育文化改革對國家民族建設之重要性 。晚年除譯述政論外，代表著作《歷史週期法則論》則為其思想體系定於一尊<ref>{{Cite book|title=存在交涉：日治時期的臺灣哲學|last=洪子偉|first=|publisher=聯經|year=2016|isbn=|location=臺北|pages=28|chapter=日治時期臺灣哲學系譜與分期}}</ref><ref>{{Cite journal|title=中国文化教育学的起步与发展特征——基于文化教育学早期著述的文本分析|author=蒋红霞|url=http://epc.swu.edu.cn/viscms/epcidex/2017niandi4qi3472/20170428/346328.html|journal=现代大学教育|issue=1|doi=|others=|year=|volume=|page=|pmid=|access-date=2018-05-01|archive-url=https://web.archive.org/web/20180501225858/http://epc.swu.edu.cn/viscms/epcidex/2017niandi4qi3472/20170428/346328.html|archive-date=2018-05-01|dead-url=yes}}</ref>。
