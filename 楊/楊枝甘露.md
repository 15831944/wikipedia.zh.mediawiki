'''楊枝甘露'''是一種[[香港|香港]][[甜品|甜品]]，起源於1980年代香港的粵菜酒家<ref name="wenweipo">[http://info.wenweipo.com/index.php?action-viewnews-itemid-68338-page-5 香港十大特色小吃推薦] {{Wayback|url=http://info.wenweipo.com/index.php?action-viewnews-itemid-68338-page-5 |date=20140903044031 }}，文匯報，2014-7-11</ref>。楊枝甘露除了是一種香港消暑甜品外，還被作為其他食品的味道，在香港便有商家生產楊枝甘露味道的[[雪糕|雪糕]]和[[雪條|雪條]]，在中秋節也有餅店推出楊枝甘露味道的[[冰皮月餅|冰皮月餅]]。

== 起源 ==
「楊枝甘露」公認是由香港利苑酒家發明，而詳細起源則有以下三種說法：
#據報最早於1984年在香港的[[利苑酒家|利苑酒家]]首先出現<ref name="takungpao">[http://www.meishij.net/chufang/naxienianwomenyiqizhuidegangjuzhongdemeishi.html 那些年我們一起追的港劇美食]，大公報，2012-10-23</ref>。
#另一個說法是香港利苑酒家於1987年，準備在[[新加坡|新加坡]]開設第一間海外分店「利苑餐廳」的時候，為了因應當地全年炎熱潮濕的[[東南亞|東南亞]]氣候，於是便在開業前，由香港利苑酒家的名廚[[黃永幟|黃永幟]]<ref name="apple">[http://hk.apple.nextmedia.com/supplement/food/art/20080815/11476287 參見露祖宗]，蘋果日報，2008-8-15</ref>，把原本該酒家的燉製甜品，改良成為一種冷吃的消暑甜品，他以冰凍的糖水、忌廉奶、西米露，結合芒果粒和西柚肉等食材，製成了這種口感滑潤和清甜爽口的新甜品<ref name="businessweekly">[http://www.businessweekly.com.tw/KBlogArticle.aspx?ID=1636&pnumber=3 香港「光頭廚神」的人生四菜]，商業周刊，2012-6-27</ref>。
#另外，還有一個說法是新加坡利苑餐廳開業前，香港利苑酒家根據當地習俗吃[[撈起魚生|撈起魚生]]後，剩下很多[[西柚|西柚]]肉，最後採納點心部的解決方案，將西柚肉加進平常的芒果西米露中<ref>[http://www.p1.cn/not_login_article/e_lulu?sequence=389 楊枝甘露的起源地：利苑餐廳] {{Wayback|url=http://www.p1.cn/not_login_article/e_lulu?sequence=389 |date=20140726112242 }}</ref>。新甜品由總監司徒安取其名為「楊枝甘露」，其含義是「觀音用楊柳枝撒下甘露般令人神清氣爽」。香港利苑酒家同時也在香港推出這種新甜品，而其他酒家和甜品店也使用類似食材，調製同樣以「楊枝甘露」命名的消暑甜品。

== 材料 ==
製作楊枝甘露材料一般包括[[西柚|西柚]]、[[沙田柚|沙田柚]]、[[芒果|芒果]]、[[西米|西米]]、[[鮮忌廉|鮮忌廉]]和糖水。柚子拆成肉，芒果則切粒，與西米及糖水拌勻<ref>[http://hk.apple.nextmedia.com/supplement/food/art/20040421/3995010 熱賣甜品︰足料楊枝甘露]，蘋果日報，2004年04月21日</ref>。有些甜品店會在楊枝甘露中加入[[椰汁|椰汁]]、[[雜果|雜果]]或[[燕窩|燕窩]]等食材。

== 商品 ==
楊枝甘露除了在香港的酒家作為餐後甜點外，在甜品店也是常年售賣的甜品，在炎熱的夏日特別受歡迎。此外，香港有飲料生產商製作預製的樽裝楊枝甘露飲品在[[超級市場|超級市場]]、[[便利店|便利店]]及[[涼茶舖|涼茶舖]]出售<ref>[http://www.sina.com.hk/news/article/20160609/0/7/47/%E6%9E%9C%E6%B1%81%E5%85%88%E7%94%9F%E9%83%BD%E5%87%BA-%E6%A5%8A%E6%9E%9D%E7%94%98%E9%9C%B2-3%E5%A4%A7%E6%A8%BD%E8%A3%9D%E5%93%81%E7%89%8C%E6%AF%94%E6%8B%BCFood-Battle-5911580.html 果汁先生都出 楊枝甘露 3大樽裝品牌比拼]，新浪香港，2016-06-09</ref>。楊枝甘露不單可作為一道甜品，還可用作調製[[雪糕|雪糕]]及[[雪條|雪條]]的味道，香港也有[[西餅店|西餅店]]售賣楊枝甘露味道的[[蛋糕|蛋糕]]和[[冰皮月餅|冰皮月餅]]。現在楊枝甘露及相關產品已流傳到香港以外的地區，包括澳門和台灣<ref>[https://read01.com/aEDRnK.html 統一推新品，既是甜品又是飲料的楊枝甘露]</ref>。

== 健康 ==
楊枝甘露在香港是受歡迎的甜品，而且楊枝甘露使用芒果、西柚肉作為材料，所以口感清新，可是由於楊枝甘露含有椰汁及會加入糖水調味，所以[[脂肪|脂肪]]及[[糖分|糖分]]都較高。[[香港消費者委員會|香港消費者委員會]]於2016年10月就香港常見的[[糖水|糖水]]及甜品進行測試，發現在香港所售賣的楊枝甘露含糖量普遍偏高，平均每100克含有11克糖，吃一碗便會攝取40克糖，接近[[世界衞生組織|世界衞生組織]]建議的每日糖分攝取量50克的上限<ref>[http://hk.on.cc/hk/bkn/cnt/news/20161017/bkn-20161017100003101-1017_00822_001.html 消委會：楊枝甘露最甜「B仔涼粉」1碗超標]，東方日報，2016-10-17</ref>。

== 名稱典故 ==
楊枝甘露是[[觀音|觀音]]菩薩的一種法寶。觀音手執一個插著[[楊柳|楊柳枝]]的寶瓶，瓶內的[[露水|露水]]便叫做「楊枝甘露」，相傳觀音菩薩曾經使用楊枝甘露救活了一棵被[[孫悟空|孫悟空]]推倒的[[人參果_(神話)|人參果樹]]<ref>[http://www.npm.gov.tw/exh94/form9407/page02_02.html 西遊記的世界]，國立故宮博物院</ref>。
