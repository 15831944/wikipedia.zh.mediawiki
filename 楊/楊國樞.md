'''楊國樞'''（{{bd|1932年|12月22日|2018年|7月17日|catIdx=Yang楊}}），中華民國[[心理學家|心理學家]]，山東農村子弟出身，臺大心理系畢業後曾留學美國伊利諾大學並取得博士學位。曾任[[台灣大學|台灣大學]]心理學系及心理科學研究中心講座教授。

==生平==
楊國樞出生於[[山東省_(中華民國)|山東省]]諸葛村的一個農村家庭，中學未畢業即因國共內戰，於1947年隨父母移居台灣。<ref name = "向楊國樞致敬　蔡英文：他啟發了一整個世代的台灣人_2018"> {{ cite web | title = 向楊國樞致敬　蔡英文：他啟發了一整個世代的台灣人 | url = https://www.upmedia.mg/news_info.php?SerialNo=47336 | publisher = 上報 UP Media | author = 上報快訊 | language = zh-tw | date = 2018-09-01 }} </ref>

1958年，楊國樞自[[國立臺灣大學|國立臺灣大學]][[心理學|心理學]]系畢業，赴[[美國|美國]]留學，於1967年、1969年分別獲得[[伊利諾大學|伊利諾大學]][[文學碩士|文學碩士]]學位與[[哲學博士|哲學博士]]學位，論文為「人格及社會心理學」，並返回台灣，成為台灣首位心理學博士。<ref name = "向楊國樞致敬　蔡英文：他啟發了一整個世代的台灣人_2018" />

1970年代，曾擔任《大學》雜誌總編輯。

1980年，發表專文「心理學研究的中國化：層次與方向」成為推動心理學與社會科學中國化的動力之一。<ref name = "向楊國樞致敬　蔡英文：他啟發了一整個世代的台灣人_2018" />

1989年擔任[[澄社|澄社]]創社社長<ref>《影響200》，天下雜誌出版，2006年，第157頁</ref>，時常批評時政。

返回臺灣後，曾任國立臺灣大學心理學系（所）助教（1959年-1963年）、講師（1963年-1969年）、副教授（1969年-1973年）、教授（1973年-1997年），並曾兼任系主任（1980年-1986年)，他曾經擔任[[臺灣心理學會|中國心理學會]]理事長(1977年-1979年）及（1981年-1983年）、中央研究院副院長（1996年-2000年）、國際生命線協會中華民國總會理事長、香港中文大學心理學組高級講師兼主任（1978年-1979年）、中國心理協會理事長、亞洲社會心理學會理事長(1999年-2000年)等職位。

1989年，與[[自由派|自由派]]學者胡佛、張忠棟、[[李鴻禧|李鴻禧]]等人共同創辦[[澄社|澄社]]，要求政府廢止《動員戡亂時期臨時條款》聲明 。<ref name=":0">{{Cite news|url=https://tw.news.appledaily.com/headline/daily/20180718/38072789/|title=華人心理學宗師 創澄社捍衛民主 楊國樞 86歲辭世|author=蘋果日報|date=2018-7-18|work=|accessdate=}}</ref><ref name = "向楊國樞致敬　蔡英文：他啟發了一整個世代的台灣人_2018" />

1990年，擔任學術界反對軍人組閣的召集人。<ref name=":0" />

1993年，創辦心理學期刊《本土心理學研究》。<ref>{{Cite web|url=http://www.indigpsych.org/posts/17|title=【徵稿】「本土心理學研究25週年回顧與展望」專刊徵稿(2018/1/10截止)|accessdate=2018-7-18|author=華人本土心理研究基金會|date=2017-5-4|publisher=}}</ref>

1997年，從台大心理系榮譽退休，與友人創立華人本土心理研究基金會。<ref name = "向楊國樞致敬　蔡英文：他啟發了一整個世代的台灣人_2018" />

1998年，獲選為[[中央研究院院士|中央研究院院士]]，為民族研究所院士。<ref name=":1">{{Cite web|url=https://www.ioe.sinica.edu.tw/Content/Researcher/content.aspx?&SiteID=530167135246736660&MenuID=530167136416504370&Fid=0&MSID=530210547762537705|title=通信/兼任研究人員 - 楊國樞(Yang, Kuo-Shu)|accessdate=2018-7-18|author=中央研究院民族學研究所|date=|publisher=}}</ref><ref name = "向楊國樞致敬　蔡英文：他啟發了一整個世代的台灣人_2018" />

2002年8月，[[中央研究院|中央研究院]]民族學研究所通信研究員。佛光大學心理學系講座教授。<ref name=":1" />

2006年，[[中原大學|中原大學]]心理學系及心理科學研究中心講座教授。<ref name=":1" />

2017年6月，因急性肺炎到萬芳醫院住院。<ref name = "向楊國樞致敬　蔡英文：他啟發了一整個世代的台灣人_2018" />

2018年7月17日，逝世於台大醫院北護分院護理之家。<ref name = "向楊國樞致敬　蔡英文：他啟發了一整個世代的台灣人_2018" />

== 學術專長 ==
楊國樞的學術專長為[[人格心理學|人格心理學]]與[[社会心理学|社會心理學]]<ref name=":1" />，尤其專注於[[本土心理學|本土心理學]]的研究<ref>[http://140.112.62.7/circulations/201208/interview.html 徐詩淳、黃詩婕，《人物專訪－楊國樞老師》，臺大心理系季報，101年8月刊] {{webarchive|url=https://web.archive.org/web/20131207135716/http://140.112.62.7/circulations/201208/interview.html |date=2013-12-07 }}</ref>。
