'''楊豐智'''（{{bd|1989年|1月6日}}，綽號'''小智'''），[[中国大陆|中国大陆]]网路播客，游戏《[[英雄联盟|英雄联盟]]》娱乐解说人。

== 人物经历 ==
* 2008年开始解说《[[澄海3C|澄海3C]]》。
* 2012年9月开始解说《[[英雄联盟|英雄联盟]]》<ref>[http://news.7k7k.com/doc/20150317/502244_3.html lol小智本人照片小智与澄海英雄联盟不得不说的故事]</ref>，其视频在[[优酷|优酷]]的播放数超过6亿。<ref>[http://i.youku.com/u/UMzU3MjkzOTI0 从优酷界面可知]</ref>
* 2014年，成为英雄联盟全明星解说投票第一名。<ref>[http://lol.yzz.cn/news/201404-791777.shtml 2014全明星解说投票 小智小苍暂领衔]</ref><ref>[http://lol.52pk.com/xinwen/5987240.shtml LOL全明星解说票选第一名：小智专访]</ref>
* 小智原在YY直播平台进行直播，2014年10月，小智与YY直播谈判破裂，其后跳槽到[[斗鱼TV|斗鱼TV]]。<ref>{{Cite web |url=http://www.wj001.com/news/game/2014-10-26/339960.html |title=LOL小智与YY谈崩跳槽斗鱼TV 小智斗鱼直播时间 |access-date=2015-08-06 |archive-url=https://web.archive.org/web/20150722055736/http://www.wj001.com/news/game/2014-10-26/339960.html |archive-date=2015-07-22 |dead-url=yes }}</ref>但由于与斗鱼产生矛盾，2015年9月，其直播间被斗鱼强制关闭。<ref>[http://tech.hnr.cn/sm/2015/1116/37832.html LOL小智熊猫TV直播人气高涨 称斗鱼计谋未得逞] {{webarchive|url=https://web.archive.org/web/20151119004755/http://tech.hnr.cn/sm/2015/1116/37832.html |date=2015-11-19 }}</ref>
* 2015年9月，有传闻称小智成为网络直播平台熊猫TV的股东。<ref>[http://gz.ifeng.com/mingqi/detail_2015_09/12/4339979_0.shtml LOL小智逆袭成为熊猫TV股东 五五开小智MISS若风年薪过千万] {{webarchive|url=https://web.archive.org/web/20151107070751/http://gz.ifeng.com/mingqi/detail_2015_09/12/4339979_0.shtml |date=2015-11-07 }}</ref>
* 2015年11月，小智跳槽到[[熊猫TV|熊猫TV]]进行直播。<ref>[http://www.techweb.com.cn/onlinegamenews/2015-11-12/2225536.shtml lol小智“感谢”斗鱼让他休息：钱一分没少拿 人气也没损失]</ref><ref>[http://www.ptbus.com/view/575740 第一娱乐主播归来！150万观众助小智加冕熊猫TV人气王] {{webarchive|url=https://web.archive.org/web/20151117013927/http://www.ptbus.com/view/575740 |date=2015-11-17 }}</ref>
* 2015年12月25日，小智宣布自己即将在熊猫TV和[[全民tv|全民tv]]同时进行直播。<ref name=sohu></ref>

== 争议 ==
=== 直播平台 ===
* 2015年6月，小智在微博上指责[[虎牙直播|虎牙直播]]在未得到他的许可下，私自将他的直播放在首页的显眼位置，侵害了自己的著作权，并称如果虎牙直播给他2000万的授权费他会考虑授权。其后虎牙直播发表道歉。<ref>[http://www.taihainet.com/news/finance/hot/2015-06-07/1435347.html LOL小智撕逼虎牙:致歉小智不买账]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
* 2015年7月，小智指责斗鱼TV纵容其他主播刷人气，并称斗鱼拖欠他鱼丸和两个月工资。<ref>[http://news.candou.com/599001.shtml LOL小智与斗鱼撕逼视频 解约事件是否走文森特之路]</ref>
* 2015年8月，小智承认其直播间上的观看人数为虚假，但称该行为是[[斗鱼TV|斗鱼TV]]所为。<ref>[http://www.265g.com/hotgame/gl/693577.html?_t=t LOL小智爆直播人数水分200W虚假]</ref>
* 2015年9月，小智直播间被斗鱼强制关闭，后使用[[若风|若风]]和小漠的直播间但两者的直播间也被关闭，最终在[[战旗TV|战旗TV]][[碧哥|碧哥]]的直播间进行直播<ref>{{Cite web |url=http://news.chinabyte.com/481/13546981.shtml |title=斗鱼彻底怒了!LOL小智借尸还魂直播遭斗鱼封杀 |access-date=2015-11-18 |archive-url=https://web.archive.org/web/20151118225849/http://news.chinabyte.com/481/13546981.shtml |archive-date=2015-11-18 |dead-url=yes }}</ref>。事后小智称自己没违约没遭起诉，为何要关闭他的斗鱼直播间。<ref>[http://www.techweb.com.cn/onlinegamenews/2015-09-14/2201448.shtml LOL小智欲“借尸还魂”继续直播 若风被连累封房间]</ref><ref>[http://mt.sohu.com/20150915/n421186004.shtml 斗鱼TV起诉蛋糕等多名主播 lol小智：不起诉为何关我直播间]</ref>
* 2015年12月25日，小智宣布自己即将在熊猫TV和全民TV同时进行直播，[[王思聪|王思聪]]指责小智不懂契约精神，但小智称是受熊猫TV指派进行双平台直播。<ref>[http://www.pcpop.com/doc/1/1548/1548227.shtml LOL小智直播脚踩两只船 王思聪称其契约精神都不知]</ref>12月29日，小智称是熊猫TV和全民TV谈好的结果，但由于沟通不畅所致，并进行道歉。<ref name=sohu>{{cite news|url=http://sports.sohu.com/20151231/n433142415.shtml|title=LOL无双小智澄清直播事件 认错称有愧于王思聪}}</ref>

=== 与他人对骂 ===
*2014年4月12日，由于观众里辱骂小智，后小智进行回骂。后来，在游戏里有队友把团队表现不好的情况认为是他的错，于是他们互相对骂<ref>[http://games.qq.com/a/20150413/001413.htm#p=1 LOL小智：再这样下去 我可能会崩溃]</ref>。
*2015年6月，小智在直播中因与队友发生冲突并发生对骂，引来400多万人围观。<ref>[http://news.52pk.com/shtml/20150603/6435372.shtml LOL小智和网友对骂 直播人数飙升500万]</ref>
*2015年7月，小智因在直播过程中遭到观众辱骂导致情绪失控，便随后发布长微博宣布解约斗鱼，还称需要休息一段时间。<ref>{{Cite web |url=http://news.chinabyte.com/245/13479745_3.shtml |title=斗鱼tv小智和女友帅爆了 小智直播受辱解约斗鱼 |access-date=2015-08-24 |archive-url=https://web.archive.org/web/20150918010226/http://news.chinabyte.com/245/13479745_3.shtml |archive-date=2015-09-18 |dead-url=yes }}</ref>7月28日，他在微博上表示会克制自己不去骂人。<ref>{{Cite web |url=http://www.taihainet.com/news/finance/hot/2015-08-02/1492236.html |title=斗鱼tv主播LOL小智自揭喜欢骂人 揭小智完整个人资料 |access-date=2015-08-24 |archive-url=https://web.archive.org/web/20150924112850/http://www.taihainet.com/news/finance/hot/2015-08-02/1492236.html |archive-date=2015-09-24 |dead-url=yes }}</ref>
*2016年2月10日，小智在直播时遇到一名对手的人身攻击与语言嘲讽，后寻求粉丝帮助希望对其进行[[人肉搜索|人肉搜索]]。<ref>[http://www.techweb.com.cn/shoujiyouxi/2016-02-11/2276072.shtml lol 英雄联盟 小智直播怒砸键盘 微博人肉玩家为哪般？]</ref>
*2016年7月12日，小智在直播时，女主播卡卡掉线，此时网友不停跑到小智房间称卡卡被封了，要小智站出来帮忙。小智称帮不了忙，却被观众指责，被某平台巨量水军进行人身辱骂。最后小智忍无可忍，怒摔键盘关闭直播。<ref>[http://games.qq.com/a/20160713/001200.htm?t=1468378306073#p=1 小智刚黑粉摔键盘 到底是谁在膨胀？]</ref>
*2017年4月21日，小智在直播中透露自己是DAN战队的老板（DAN战队晋级LPL），并调侃了[[刘谋|刘谋]]（刘谋的YM战队没有晋级LPL），但刘谋找到DAN的工作人员核实小智没有DAN的股份，小智随后称自己是找人代持的股份，而刘谋咬定小智是在说谎，并曝光小智个人电话且出现了重度言语伤人。4月29日直播中双方相互语音攻击。4月30日，DAN官博在微博表示曾和小智关于股份有过口头协定，正在落实当中，并主动承担所有责任。小智和刘谋在起小点的作和下达成共识，让事情翻篇。<ref>[http://www.hxnews.com/news/dmyx/djyx/yxxw/201706/14/1237094.shtml lol小智与PDD撕逼事件怎么回事始末回顾 lol小智与PDD关系怎么样]</ref><ref>[http://finance.ifeng.com/a/20170502/15334054_0.shtml 英雄联盟：该来的总是要来的：PDD小智撕逼事件始末]</ref>
*2017年5月20日，[[地下城与勇士|地下城与勇士]]主播旭旭宝宝在直播时表示赚的最多的主播是MC天佑，比小智赚的还多。结果小智反骂旭旭宝宝，并称“DNF没见过世面，穿你的西装去吧”。5月21日，小智在直播中和旭旭宝宝连线通话，表示都是误会，并向旭旭宝宝道歉。旭旭宝宝表示谅解。<ref>[http://fj.china.com.cn/p/342448.html lol主播小智再撕名dnf主播旭旭宝宝 互相撕逼事情是这样的！]</ref><ref>[http://sports.huanqiu.com/gdsports/2017-05/10717834.html LOL小智直播向DNF旭旭宝宝道歉：玩笑开过头了]</ref>

===相关官司===
*2015年中旬，女主播西法称其被经纪公司众义娱乐拖欠一万薪资，在直播间哭诉讨薪。小智得知此事后，明白直播行业的内幕隐情，不忍见女主播想自己当初一样收到侮辱，并进行声援，导致众义娱乐多个直播室被封，之后众义娱乐将小智告上法院，法院一审为：小智赔偿众义娱乐经济损失五万元人民币，并在斗鱼首页发布十五天道歉声明。小智赔款五万元人民币，但他认为斗鱼不是媒体，没义务刊登道歉声明，所以未进行道歉。因为没有执行道歉，2015年12月7日，小智被法院列入失信人名单。小智在此事并没有一点受利并大力帮助被压榨的主播最后自己损失很大，但他从不为此事后悔。<ref>[http://games.sina.com.cn/g/g/2016-01-04/fxneept3642915.shtml LOL解说小智回应进入失信名单：我上失信我骄傲]</ref>

===生日聚会===
2016年1月6日18点，小智在葫芦岛振武贵宾楼宴请粉丝过生日，但由于现场人员太多，使得葫芦岛警方出面，警察以聚众人数太多为由对现场进行驱散。最终主持人碧哥帝师在小智授权下，将贵宾楼安排完毕，又包场了隔壁的两个饭店，但由于其到场粉丝量巨大，最后配合警察将剩余粉丝安排疏散。小智和碧哥在微博上对粉丝进行道歉。<ref>[http://mt.sohu.com/20160107/n433791540.shtml LOL小智庆生 三千粉丝空降葫芦岛]</ref><ref>[http://www.gamefy.cn/nview.php?id=6026 三千粉丝空降葫芦岛 LOL解说小智生日聚会仍不露面]</ref>
