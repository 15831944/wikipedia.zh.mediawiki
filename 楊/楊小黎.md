'''楊小黎'''（{{bd|1988年|11月6日|catIdx=H}}），[[童星|童星]]<ref>{{Cite web |url=http://iamrita5.myweb.hinet.net/g16.htm |title=童心歲月：昔日童星 |access-date=2007-10-17 |archive-url=https://web.archive.org/web/20071015175608/http://iamrita5.myweb.hinet.net/g16.htm |archive-date=2007-10-15 |dead-url=yes }}</ref>出身的[[臺灣|臺灣]][[女藝人|女藝人]]，擔任[[配音員|配音員]]、[[女演員|女演員]]、[[歌手|歌手]]和女[[主持人|主持人]]。以《[[台北歌手|台北歌手]]》榮獲第53屆電視金鐘奬戲劇節目最佳女配角獎。

== 出道過程 ==
*楊小黎四歲時和母親一起逛街時被星探發掘，開始出演[[廣告|廣告]]、[[電視劇|電視劇]]及[[電影|電影]]，並擔任多個兒童節目主持人。
*曾參與多套動畫影集的配音工作，其中擔任外國電影《[[哈利波特|哈利波特]]》系列的[[妙麗·格蘭傑|妙麗·格蘭傑]]的中文配音最為人所熟知。
*曾入圍[[金鐘獎|金鐘獎]]最佳兒童節目主持人獎，2011年榮獲[[中央通訊社|中央通訊社]]主辦之〈全國主播爭霸戰〉冠軍，另也從事中英文活動及典禮主持工作。
*目前為《[[聯合報|聯合報]]》週日[[副刊|副刊]]撰寫專欄，並常受邀大專院校演講及教授聲音表情課程。
*2012年，演出[[公視|公視]]《公視人生劇展－[[愛情替聲|愛情替聲]]》，並入圍[[第48屆金鐘獎|第48屆金鐘獎]]迷你劇集／電視電影女主角獎 。
*2016年12月29日，首度以清新的嗓音，與製作人宋小圭合組創作團體[https://www.facebook.com/Miss.Beau.Band/ Miss.Beau!]，推出了首張台語專輯《楊美麗的青春娜卡西》。
*2018年拿下金鐘獎最佳女配角
