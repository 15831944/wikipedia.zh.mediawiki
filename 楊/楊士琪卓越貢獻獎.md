'''楊士琪卓越貢獻獎'''（[[英語|英語]]譯名：'''{{lang|en|Yang Shih-chi Outstanding Contribution Award}}'''）是[[台北電影獎|台北電影獎]]的獎項之一，獎勵具有終身為電影事業貢獻的人士<ref name = A20170712>{{cite news|title=楊士琪卓越貢獻獎 首座電影獎以記者命名|publisher=醒報|url=https://anntw.com/articles/20170712-n0z8|date=2017-07-12|author=胡幼鳳|accessdate=2017-07-17|language=zh-hant}}</ref>。該獎項前身是[[楊士琪紀念獎|楊士琪紀念獎]]與[[台北電影獎卓越貢獻獎|卓越貢獻獎]]<ref>{{cite news|title=楊士琪卓越貢獻獎 首座電影獎以記者命名|publisher=醒報|url=https://anntw.com/articles/20170712-n0z8|date=2017-07-12|author=胡幼鳳|accessdate=2018-04-24|language=zh-hant}}</ref>。

== 歷史 ==
1983年，[[中國影評人協會|中國影評人協會]]看完[[中央電影公司|中央電影公司]]電影《[[兒子的大玩偶_(電影)|兒子的大玩偶]]》後，寄信至[[中國國民黨中央委員會文化傳播委員會|中國國民黨中央文化工作會]]檢舉該部電影，稱戲裡[[萬仁|萬仁]]執導的第三段《蘋果的滋味》有「醜化台灣人民」，因此要求[[中影公司|中影公司]]刪剪四段相關內容，否則禁演。但萬仁在《[[聯合報|聯合報]]》報社門口打電話找到支持[[台灣新浪潮電影|台灣新浪潮電影]]的記者[[楊士琪|楊士琪]]，將該起事件報導出來。由於臺灣多家媒體也跟進報導該起事件，電影審查的主管單位在輿論壓力下收回修剪的成命，《兒子的大玩偶》得以上映未刪減的版本<ref name = A20170712/>。

1984年，楊士琪因[[氣喘病|氣喘病]]發作辭世。1985年，導演[[楊德昌|楊德昌]]在他的電影《[[青梅竹馬_(電影)|青梅竹馬]]》片頭，特地以中文黑底白字並加註英文字幕寫著「獻給楊士琪，感激她生前給我們的鼓勵。——製作全體同仁致敬」，全長六秒鐘。隨後，楊士琪前友人及電影界與新聞界的25位好友聯合倡議組成「楊士琪紀念獎工作委員會」，決定每年選出一位富有道德勇氣並努力開創新局、提升電影地位的傑出人士，頒給「楊士琪電影紀念獎」以資鼓勵，並彰顯楊士琪生前熱愛電影、勇於創新的精神<ref>[http://funscreen.com.tw/TaiwanMade.asp?period=%20 【電影新訊】 楊士琪紀念協會將成立] {{webarchive|url=https://web.archive.org/web/20070624053353/http://www.funscreen.com.tw/TaiwanMade.asp?period= |date=2007-06-24 }}，[[國立中央大學|國立中央大學]]</ref>。
