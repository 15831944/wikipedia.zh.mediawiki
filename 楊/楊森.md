'''杨森'''（{{bd|1884年|2月20日|1977年|5月15日|catIdx=Y杨}}）<ref>徐友春主編『民国人物大辞典 増訂版』2114頁、吴嘉陵「楊森」作生于1884年2月20日。徐必鴻「楊森」作生于1882年3月13日；Who's Who in China 3rd ed.,p.906作生于1887年。徐友春主編『民国人物大辞典 増訂版』、吴嘉陵「楊森」作卒于1977年5月15日。徐必鴻「楊森」作卒于1977年5月17日。</ref>旧名'''淑泽'''、'''伯坚'''，字'''子惠'''，[[四川省|四川省]][[順慶府|順慶府]][[广安州_(明朝)|广安州]]龙台寺（在今[[广安市|广安市]][[广安区|广安区]]）人。[[国民革命军|国民革命军]]將領，[[川军|川军]]高级将领。

== 生平 ==
=== 北京政府時代 ===
杨森早年有志于学問，入[[順慶府聯合中学堂|順慶府聯合中学堂]]学习。后来转而从军，1904年（光緒30年）入[[四川陸軍速成學堂|四川陸軍速成學堂]]。在校期间，1906年（光緒32年）加入[[中国同盟会|中国同盟会]]。

1908年（光緒34年）自速成学堂毕业，加入四川的[[新軍|新軍]][[陆军第十七镇|第十七鎮]]（[[朱慶瀾|朱慶瀾]]任该镇統制）。<ref name=xbh>徐必鴻「楊森」{{Cite book<!--|和書-->|author = |title = 民国高級将領列传 1|year = 1998|publisher = 解放軍出版社|isbn = 7-5065-0261-5}}</ref><ref name=wu>吴嘉陵「楊森」{{Cite book<!--|和書-->|author =謝本書主編|title = 西南十軍閥|year = 1993|publisher = 上海人民出版社|isbn = 7-208-01642-9}}</ref><ref name=xu>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref>

1911[[中华民国|中华民国]]成立後，杨森成为[[川軍|川軍]]。[[二次革命|二次革命]]之際，初属革命派，后被[[滇軍|滇軍]][[黄毓成|黄毓成]]部俘虏，乃加入滇軍。

1915年（[[民国紀元|民国]]4年）12月，[[護国战争|護国战争]]爆发，杨森作为[[蔡鍔|蔡鍔]]手下第2梯团[[趙又新|趙又新]]的随从回到了四川。<ref name=xbh/><ref name=wu/><ref name=xu/>此後，楊森仍属滇軍。

1920年（民国9年）3月，为响应「川人治川」的口号，楊森发表脱离滇軍宣言，从而复归川軍。此後，他积极参与扫荡四川境内的滇軍。

1921年（民国10年）2月，杨森升任川軍第2軍軍長，驻扎[[重慶市|重慶]]。<ref name=xbh/><ref name=wu/><ref name=xu/>

後来，楊森和川軍的[[劉湘|劉湘]]合力击败南方政府派的[[熊克武|熊克武]]。1923年（民国12年）8月，[[吴佩孚|吴佩孚]]任命杨森为中央陸軍第16師師長。9月晉升陸軍中将。

1924年（民国13年）3月，升为陸軍上将，5月任命为四川督理。杨森率部驻扎[[成都|成都]]，成为川軍内最大的勢力。然而，劉湘和其他川軍指揮官对杨森的反感与日俱增。<ref name=liu>{{Cite book<!--|和書-->|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref><ref name=xbh/><ref name=wu/><ref name=xu/>

1925年（民国14年）7月，楊森在吴佩孚的支援下开始了四川統一之战。劉湘和其他川軍指揮官则结成反楊联盟，同[[黔軍|黔軍]][[袁祖銘|袁祖銘]]共同攻撃杨森。8月，楊森惨敗，退出四川，一度依赖吴佩孚。但随后，劉湘同袁祖銘发生对立，楊森和劉湘重新修好，楊森重归四川，和劉湘共同将袁祖銘逐出四川。

1926年（民国15年）5月，吴佩孚任命杨森为四川省長。<ref name=xbh/><ref name=wu/><ref name=xu/>9月[[萬縣慘案|萬縣慘案]]中，杨森同[[英国軍队|英国軍队]]交战<ref>[http://www.mofa.go.jp/mofaj/annai/honsho/shiryo/archives/DT0004/0003/0010/0022/0010/0002/index.djvu 日本外交文書デジタルアーカイブ 大正15年（1926年）第2冊下巻]</ref>。

=== 国民政府時代 ===
1926年10月，随着[[中国国民党|中国国民党]]勢力扩張，楊森宣布加入[[国民革命軍|国民革命軍]]，被任命为[[国民革命军第二十军|国民革命军第二十军]][[軍長|軍長]]。但是，1928年1月，杨森因庇护吴佩孚而被问罪免職。<ref name=xbh/><ref name=wu/><ref name=xu/>

即便如此，楊森仍忠于[[国民政府|国民政府]]。1936年2月，杨森升为陸軍中将。[[抗日战争|抗日战争]]爆发後，杨森任[[國民革命軍第六軍团|國民革命軍第六軍团]]軍团長，在上海方面展开軍事活動。

1938年1月，任[[國民革命軍第二十七集团軍|國民革命軍第二十七集团軍]]总司令。1939年10月，兼任[[第九战区|第九战区]]副司令長官。<ref name=xbh/><ref name=wu/><ref name=xu/>

抗日战争结束前的1945年1月，杨森被任命为[[貴州省_(中華民國)|貴州省]][[贵州省政府|政府]]主席。1947年7月19日，國民政府派楊森為貴州選舉事務所主任委員；國民政府同時派定其為[[國民大會代表|國民大會代表]]，[[立法院|立法院]][[立法委員|立法委員]]<ref name="民國史大事記">{{cite book |editors = 李新 總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室 編，韓信夫、姜克夫 主編 |title=《中華民國史大事記》 |location=[[北京|北京]] |publisher=[[中華書局|中華書局]] |date=2011年7月 }}</ref>{{rp|8385-8386}}。1948年（民国37年）7月，转任[[重慶市政府|重慶市政府]]市長。1949年6月，兼任[[西南軍政長官公署|西南軍政長官公署]]副長官。12月7日，重慶衛戍總司令部着即撤銷；派楊森為西南軍政副長官並兼代川陝甘邊區[[綏靖公署|綏靖公署]]主任<ref>{{cite news |title=《總統府公報》第貳肆伍號 |editor = 總統府第五局 |date=1949-12-31 }}</ref>。同年12月，中国西南部的[[中华民国国军|國軍]]溃败，杨森逃往[[台湾|台湾]]。此後，杨森历任[[總統府國策顧問|總統府國策顧問]]、[[中華全國體育協進會|中華全國體育協進會]]理事長、[[中國奧林匹克委員會_(中華民國)|中國奥林匹克委員會]]主席。<ref name=xbh/><ref name=wu/><ref name=xu/>

1966年8月15日，杨森登上[[玉山|玉山]]。1971年[[重陽節|重陽節]]，楊森再度登上玉山。<ref>中外杂志1999年66卷（389-394期）</ref>{{rp|140}}

1977年5月15日，杨森在[[台北市|台北]]病逝。享耆壽94岁（满93歲）。<ref name=xbh/><ref name=wu/><ref name=xu/>

== 家庭 ==
[[File:Yang_Sen3.jpg|thumb]]

楊森兄長是[[辛亥革命|辛亥革命]]先烈，臨終託孤女於楊森。是為[[楊漢秀|楊漢秀]]。

楊森一生有原配妻子1人、续弦妻1人、妾11人，儿子21个，女儿22个。
*第一任妻子（元配）：张氏，无子女。
*第二任妻子（续弦）：谭正德，又名谭继贞，四川广安人，有长子杨汉忻、次子[[杨汉烈|杨汉烈]]。
*三姨太：刘谷芳，云南禄丰人，有子女5人。
*四姨太：田蘅秋，四川阆中人，有子女7人。
*五姨太：萧邦琼，四川泸县人，有子女5人。
*六姨太：陈顺容，又名陈凤英，有子女5人。
*七姨太：曾桂枝，贵州毕节人，有两个女儿。1929年被杨森杀害。
*八姨太：汪德芬，四川成都人，有子女4人。
*九姨太：蔡文娜，四川泸县人，有两个儿子。被杨森杀害。
*十姨太：郑文如，四川重庆人，有子女2人。
*十一姨太：[[胡洁玉|胡洁玉]]，四川广安人，17岁时成为62岁的杨森的夫人，后成为美国核物理学教授，有一个女儿。
*十二姨太：张灵凤，台湾新竹人，17岁时成为90岁的杨森的如夫人，有一个女儿。<ref>[http://www.shuku.net/novels/mingjwx/wuyuewx/wuyue09.html 吴越，杨森的姨太太们，载 吴越文选，于2012-11-28查阅]</ref>
