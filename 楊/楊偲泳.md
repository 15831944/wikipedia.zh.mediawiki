'''楊偲泳'''（'''Renci Yeung'''，{{bd|1993年|11月6日}}），香港[[now_TV|now TV]]及[[ViuTV|ViuTV]]經理人合約藝人，亦是[[香港排球乙組聯賽|香港排球乙組聯賽]]球隊[[匯青|匯青]]的球員<ref name= ViuTV女神>{{cite web |url=http://hk.apple.nextmedia.com/enews/realtime/20160509/55080449  | title=【ViuTV女神(4)】多謝巴打厚愛32D楊偲泳：好多香港女仔身材比我好|publisher= [[香港蘋果日報|香港蘋果日報]]|date=2016年5月9日 }}</ref><ref name= HK01>{{cite web |url=https://www.hk01.com/%E5%A5%BD%E7%94%9F%E6%B4%BB/47701/-%E5%8B%95%E6%84%9F%E5%A5%B3%E7%A5%9E-%E6%A5%8A%E5%81%B2%E6%B3%B3%E7%9A%84%E7%9F%9B%E7%9B%BE%E4%B8%80%E8%BA%AB-%E7%94%9C%E5%A7%90%E6%8E%92%E7%90%83%E5%B0%87%E8%B5%B0%E5%87%BA%E5%B9%95%E5%89%8D | title=【動感女神】楊偲泳的矛盾一身　甜姐排球將走出幕前 |publisher= HK01 |date=2016年10月11日 }}</ref><ref name=體路>{{cite web |url=http://sportsroad.hk/archives/180467 | title=楊偲泳：藝人與排球員的兩個身分 |publisher= [[體路|體路]]|date=2017年12月5日 }}</ref>。

==簡歷==
楊偲泳在2005年畢業於[[筲箕灣官立小學|筲箕灣官立小學]]上午校。在就讀小五時曾獲得「東區學校模範生及進步生」的獎項<ref name="筲箕灣官立上午小學 2003–2004年度學校報告">[http://webcache.googleusercontent.com/search?q=cache:Ciy_9gzU2yoJ:www.skwgps.edu.hk/am/doc/booklet_F.doc+&cd=8&hl=zh-TW&ct=clnk&gl=hk 筲箕灣官立小學《筲箕灣官立上午小學 2003–2004年度學校報告》第22頁，2003年/2004年]</ref>，後在2010年畢業於[[嶺南中學_(香港)|嶺南中學]]。
楊偲泳在初中時因一套關於沙灘排球的電視劇而加入排球校隊<ref name=體路/>，就讀中一時（2006年至2007年）與校隊成員奪得「香港學界體育聯會學界排球賽」女子甲組亞軍 <ref name="嶺南中學 學校報告（2007/08）">[http://lingnan.edu.hk/affair/dl/0708/overall/2007-08schoolreport(public).pdf 香港嶺南中學《嶺南中學 學校報告（2007/08）》，2007年/2008年]</ref><ref name=體路/>。她於2015年演出歌手[[王灝兒|王灝兒]]《矛盾一生》[[MV|MV]]中的女主角，漸為大眾留意<ref>{{cite web |url=http://the-sun.on.cc/cnt/entertainment/20160115/00470_093.html| title=楊偲泳「演」定終身將來我係張曼玉！|publisher= 太陽日報|date=2016年1月15日 }}</ref>。於2016年加入[[ViuTV|ViuTV]]拍攝節目，比較知名的演出包括有《[[SE7EN_(電視節目)|SE7EN]]》系列、《[[藝術進行中|藝術進行中]]》、《[[小城光影|小城光影]]》等等。2016年11月27號，因拍攝《[[SE7EN_(電視節目)|SE7EN]]》第二輯節目而參加[[Red_Bull|Red Bull]] Flugtag飛行比賽，比賽當日懷疑飛行器落水時受驚或腦震盪導致短暫失憶，一度需送院檢查<ref>{{cite web|url=https://www.hk01.com/%E7%BD%AA%E6%A1%88/56626/ViuTV%E5%B0%8F%E8%8A%B1%E9%A3%9B%E8%A1%8C%E6%97%A5%E6%92%9E%E5%85%A5%E6%B0%B4%E5%9A%87%E5%88%B0%E5%A4%B1%E6%86%B6-%E8%85%A6%E7%A7%91%E5%|title=ViuTV小花飛行日撞入水嚇到失憶　腦科專家：可能腦震盪|publisher=HK01}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref>{{cite web |url=http://news.mingpao.com/pns/dailynews/web_tc/article/20161128/s00002/1480269006583 |title=人力飛行器賽14傷兩嚴重 「龍蝦女」昏迷1分鐘 ViuTV主持曾失憶 |publisher=[[明報|明報]]|date=2016年11月28日}}</ref>。

2017年，楊偲泳接受訪問時公開徵求男友。而於訪問過後，她被譽為最佳女友<ref>[https://bka.mpweekly.com/focus/local/20171229-96000 【明周福利片】超GFable！楊偲泳五個好女友特徵]</ref>。亦曾掦言將來的另一半必須同樣是喜歡周星馳的電影迷<ref>[https://www.mings-fashion.com/%E6%96%87%E9%9D%92-%E5%BB%B6%E7%A6%A7%E6%94%BB%E7%95%A5-%E6%98%8E%E7%8E%89-207406/ 不是女神 | 一舉手一投足也帶點文藝，我們叫她文青]</ref>。
