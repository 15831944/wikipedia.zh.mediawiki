{{作品名稱|name=''The Post''|hk=戰雲密報|tw=郵報：密戰|cn=华盛顿邮报|equiv-sg=tw}}是一部於[[2017年電影|2017年]]上映的美國[[劇情|劇情]]電影，由[[史蒂芬·史匹柏|史蒂芬·史匹柏]]執導，莉茲·漢娜（{{lang|en|Liz Hannah}}）與[[喬許·辛格|喬許·辛格]]編劇，並由[[梅莉·史翠普|梅莉·史翠普]]與[[湯姆·漢克斯|湯姆·漢克斯]]等人主演。

該片的發展始於2016年10月，當時製片人{{link-en|艾美·帕斯卡|Amy Pascal}}獲得了莉茲·漢娜撰寫的電影劇本。該片的製片作業於2017年5月30日在紐約開始。《郵報：密戰》於2017年12月22日在美國發行。而巧合地，美國總統特朗普上任後不時打擊新聞自由，批評傳媒針對他，而當中亦曾提及到電影中所講述《華盛頓郵報》，一如電影情節。而導演史匹堡表示目前的政府作風與尼克遜時代極為相似，他希望藉此片喚起大眾關注新聞自由，站起來支持傳媒報道真相<ref>[https://bka.mpweekly.com/film-music/film/%E5%A4%A7%E8%A9%B1%E8%A5%BF%E6%B4%8B/20180131-99613 【西片推介】《戰雲密報》真相無價　史匹堡梅姨湯漢斯撐新聞界]</ref>。

== 劇情概要 ==

該片是改編自「[[五角大樓文件|五角大樓文件]]」一案，
此機密文件為美國國防部對1945年到1967年，
美國在越南政治和軍事操作及捲入的評估報告。
而《[[華盛頓郵報|華盛頓郵報]]》的[[凱瑟琳·葛蘭姆|凱瑟琳·葛蘭姆]]（[[梅莉史翠普|梅莉史翠普]]飾演），也是美國第一位女性報紙發行人，以及報社編輯班·布萊德利（[[湯姆漢克斯|湯姆漢克斯]]飾演）攜手合作，他們必須使出全力和時間賽跑，趕在《[[紐約時報|紐約時報]]》之前揭露一件牽連四屆美國總統任期、長達三十餘年的龐大掩蓋事件，進而參與一場記者與政府之間的空前鬥爭的故事。

== 製作 ==

2016年10月，{{link-en|艾美·帕斯卡|Amy Pascal}}獲得了莉茲·漢娜（{{lang|en|Liz Hannah}}）撰寫的電影劇本<ref name="screenplay"/>。2017年3月，據報導，[[史蒂芬·史匹柏|史蒂芬·史匹柏]]正在為執導及監製《The Post》一事進行談判，而[[湯姆·漢克斯|湯姆·漢克斯]]與[[梅莉·史翠普|梅莉·史翠普]]則分別為飾演[[凱瑟琳·葛蘭姆|凱瑟琳·葛蘭姆]]及[[本·布萊德利|班·布萊德利]]的事宜與片商會談<ref name="Deadline.com (2017-03-06)"/>。除此之外，該片的演員還包含[[愛麗森·布里|愛莉森·布里]]、{{link-en|凱莉·庫恩|Carrie Coon}}、{{link-en|大衛·科茨|David Cross}}、[[布魯斯·格林伍德|布魯斯·格林伍德]]、[[特雷西·萊特|特雷西·萊特]]、[[鮑勃·奧登科克|鮑勃·奧登科克]]、[[莎拉·保羅森|莎拉·保羅森]]、[[傑西·普萊蒙|傑西·普萊蒙]]、[[馬修·瑞斯|馬修·瑞斯]]、[[麥可·斯圖巴|麥可·斯圖巴]]、[[布萊德利·惠特福德|布萊德利·惠特福德]]與[[扎克·伍茲|扎克·伍茲]]等人<ref name="Cast"/>。《郵報：密戰》的製片作業於5月30日在紐約開始<ref name="Production"/>。6月，該片的片名改為《The Papers》<ref name="Cast"/>，但隨後又於8月25日改回《The Post》<ref name="Variety (2017-08-25)"/>。

== 音樂 ==

2017年7月，據報導，經常與史匹柏合作的[[約翰·威廉斯|約翰·威廉斯]]將為該片譜曲。威廉斯最初計劃為史匹柏的另一部作品《[[一級玩家_(電影)|一級玩家]]》製作配樂，但由於兩部電影的後期製作的時程表相近，於是威廉斯轉而選擇擔任該片的作曲家。《郵報：密戰》是威廉斯與史匹柏的第28部合作電影<ref name="Score"/>。

== 發行 ==

《郵報：密戰》計劃於2017年12月22日在美國[[有限上映|有限上映]]，並於2018年1月12日開始廣泛發行<ref name="release"/>。該片將與同期上映的電影《[[縮小人生|縮小人生]]》、《[[電流大戰_(電影)|電流大戰]]》及《[[大娛樂家_(電影)|大娛樂家]]》競爭。《[[綜藝_(雜誌)|綜藝]]》雜誌指出，《郵報：密戰》的發行日期鎖定了[[第90屆奧斯卡金像獎|第90屆奧斯卡金像獎]]<ref name="release"/>。

=== 評價 ===
[[爛番茄|爛番茄]]上根據265條評論，持有88%的新鮮度，平均得分為8／10。该网站上的批评共识写道：“《邮报》的时代背景掩饰了它犀利的针砭时事的主题，被导演史蒂文·斯皮尔伯格和一群卓越的团体演员栩栩如生地描绘出来。”<ref>{{cite web|url=https://www.rottentomatoes.com/m/the_post|title=The Post (2017) |work=[[爛番茄|Rotten Tomatoes]] |accessdate=2018-01-21}}</ref>在[[Metacritic|Metacritic]]上，電影獲得了83分<ref>{{cite web|url=http://www.metacritic.com/movie/the-post|title=The Post reviews |publisher=[[Metacritic|Metacritic]]|accessdate=2017-12-22}}</ref>。
