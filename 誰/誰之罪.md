《'''誰之罪'''》（{{lang-en|'''''Sinners' Holiday'''''}}）是一部1930年[[有聲電影|全有聲]]的荷里活{{link-en|前編碼荷里活|Pre-Code Hollywood|前編碼}}犯罪電影，由{{link-en|約翰·G·阿道菲|John G. Adolfi}}執導，主演是{{link-en|格蘭特·惠瑟斯|Grant Withers}}、{{link-en|伊沃琳·納普|Evalyn Knapp}}，加上[[詹姆斯·卡格尼|詹姆斯·卡格尼]]（他的首次演出電影）、{{link-en|露西兒·拉·凡爾納|Lucille La Verne}}和{{link-en|瓊·布朗德爾|Joan Blondell}}。本片改編Marie Baumer的1930年話劇《''Penny Arcade''》。卡格尼和布朗德爾都重演在原本[[百老匯劇院|百老匯]]上演的角色。<ref>{{Cite web |url = https://www.ibdb.com/broadway-production/penny-arcade-11072 |title = Penny Arcade |publisher = Internet Broadway Database |language = en}}</ref>
