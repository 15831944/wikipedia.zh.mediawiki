== 故事簡介 ==
[[辦公室女職員|OL]]和泉園子某日在路上和賣畫的佃潤一相愛了。但是在把潤一介紹給好友弓場佳世子的幾個月後的某晚，潤一提出了分手。得知分手原因是潤一變心喜歡上了佳世子，園子墜入絕望深淵。幾日後，園子的哥哥康正發現了妹妹的遺體。憑著至親的直覺，康正看穿這是巧妙偽裝成自殺的他殺，因此決定親手制裁兇手。終於查到潤一和佳代子的康正確信，潤一和佳世子，究竟'''誰殺了她'''？而察覺到康正意圖的加賀，又是否能夠阻止即將發生的慘劇?

與東野圭吾之後一部作品《[[我殺了他|我殺了他]]》一樣，本書最後沒有明說兇手的人名，需由讀者自行推理！講談社小說在初版發行不久後，有關兇手究竟是誰的詢問就寄到了編輯部。因此在發行文庫本的時候附上了解說。不過在解說中依然沒有直接把兇手的名字寫出。而且在發行文庫本的時候，因刪減了書中某些重要的記述，使讀者在解謎的時候難度又加大了。

== 登場人物 ==
關於'''[[加賀恭一郎系列#加賀恭一郎|加賀恭一郎]]'''請參看「[[加賀恭一郎系列#登場人物|加賀恭一郎系列－登場人物]]」。

; {{nihongo|和泉 園子|{{ruby|和泉|いずみ}} {{ruby|園子|そのこ}}|Izumi Sonoko}}
: 被害者。在電子元件製造公司的東京分公司任職OL。身材高挑，背後有星形的疤。喜歡和食和紅酒。第一章是以她的視點來描寫的。
: 死因是電源線漏電引起的[[休克|休克]]死亡。因康正把證據拿走令警察判斷為自殺，但刑警加賀恭一郎懷疑是他殺。

; {{nihongo|和泉 康正|{{ruby|和泉|いずみ}} {{ruby|康正|やすまさ}}|Izumi Yasumasa}}
: 園子的哥哥。隸屬豐橋警察署交通課的警官。從第二章至第六章為止是以他的視點來描寫的。
: 從現場狀況判斷園子死於他殺。為了親手制裁殺害妹妹的兇手，將證物拿走開始獨自調查。

; {{nihongo|弓場 佳世子|{{ruby|弓場|ゆば}} {{ruby|佳世子|かよこ}}|Yuba Kayoko}}
: 園子從高中時代開始的朋友，在保險公司工作。是個有著端正五官的美人。愛車是綠色的[[Mini_Cooper|Mini Cooper]]。

; {{nihongo|佃 潤一|{{ruby|佃|つくだ}} {{ruby|潤一|じゅんいち}}|Tsukuda Junichi}}
: 園子的前戀人，如今和佳代子交往。和園子相遇時他剛好放棄成為畫家的夢想，進入了父親任職社長的大手出版社工作。
