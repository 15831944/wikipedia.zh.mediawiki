'''澳門街坊會聯合總會'''（{{lang-pt|União Geral das Associações dos Moradores de Macau}}），簡稱'''街坊總會'''或'''街總'''，是領導[[澳門|澳門]]各區街坊會和附屬機構的[[社團|社團]]，于1983年12月30日正式成立。该组织在政治上属于[[建制派_(澳门)|建制派]]。

澳門街坊會聯合總會屬下設秘書處和11個工作委員會，分別是：基層委員會、社會事務委員會、社會服務委員會、宣傳教育委員會、康文委員會、體育委員會、青年事務委員會、婦女事務委員會、大廈工作委員會、社區經濟事務委員會、財務委員會。澳門街坊會聯合總會亦與[[澳門社會工作局|前澳門社會工作司]]開辦社區中心，主要有：[[藝駿中心|藝駿中心]]、[[街總望廈社區中心|街總望廈社區中心]]、[[街總祐漢社區中心|街總祐漢社區中心]]和[[青洲社區中心|青洲社區中心]]等。

== 宗旨 ==
澳門街坊會聯合總會領導各區街坊會和附屬機構繼續發揚愛國愛澳的精神，把“團結坊眾、參與社會，關注民生、服務社群、愛國愛澳”的作為工作宗旨。澳門街坊會聯合總會積極參與社會事務，關注房屋、教育、治安、交通、環保、衛生等社會問題，自稱為居民辦實事、謀福祉。

== 社會質疑 ==
澳門街坊會聯合總會一直受政府巨額資助，但款項使用途徑透明度嚴重不足，受到坊間不少質疑。而每屆立法會選舉，街總常以「服務社會」等作為政綱吸納選票，但「服務社會」之經費大部份源自公帑，變相成公帑幫社團買票。

== 歷史 ==
自1991年開始，该组织與其它親北京社團（包括[[澳門婦女聯合總會|澳門婦女聯合總會]]、[[澳門歸僑總會|澳門歸僑總會]]）合作，共同以“'''群力促進會'''”的名义参加[[澳門立法會|立法會]]和[[澳門市政廳|市政議會]]直接選舉，不過在[[2017年澳門立法會選舉|2017年選舉]]中，婦聯則另立「美好家園聯盟」出選，不再與街總合組名單。以下為群力歷年當選直選立法議員名單：

*1996年：梁慶庭、高開賢

*2005年：梁慶庭、容永恩
