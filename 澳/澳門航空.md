'''澳門航空'''（{{lang|pt|'''Air Macau'''}}）是[[澳門|澳門]]唯一的[[航空公司|航空公司]]，以[[澳門國際機場|澳門國際機場]]為[[樞紐機場|樞紐]]，提供到[[中國内地|中國大陸]]、[[台灣|台灣]]、[[東南亞|東南亞]]與[[東亞|東亞]]的跨境航線，航點遍及中國大陸、台灣、東亞及東南亞主要都市。

[[澳門政府|澳門政府]]與澳門航空間存在專營合同，專營權旨在以正規和連續的方式設立、保持和經營前往和來自澳門的旅客、行李、貨物、郵包和郵件的空運服務。專營權為期二十五年，自澳門國際機場投入營運之，日算起。<ref>{{cite web|title=澳門政府公報1995年第19期第二組，第1737-1738頁|url=http://images.io.gov.mo/bo/ii/95/19/bo-ii-19-95.pdf|accessdate=October 2016}}</ref>
2019年1月，民航局證實已於2018年10月已通知澳門航空專營權在2020年11月底約滿後將不會再有專管權，隨後會與中央政府溝通後航權是局部開放還是全面開放未有定案
2016年，運輸旅客280.46萬人次，同比增長21.54%；平均客座率為73.25%，同比增長6.17個百分點。<ref>{{cite web|title=中國國際航空股份有限公司2016年年度報告|url=https://www.airchina.com.cn/cn/images/investor_relations/2017/04/25/AF6D9C2ACCB03F37235CA996EC167275.pdf|accessdate=April 2017}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

2016年，澳門航空實現營業收入人民幣26.62億元，同比增長11.28%。其中，航空運輸收入為人民幣26.35億元，同比增長11.17%。實現稅後利潤人民幣0.20億元，同比下降34.80%。

2017年，澳門航空實現營業收入28.32億元人民幣，同比增長6.39%，稅後利潤0.70億元，同比增長243.21%

== 歷史 ==
===成立===
澳門航空成立於1994年9月13日。配合[[澳門國際機場|澳門國際機場]]於1995年11月9日啟用，同日開始商業飛行，首次派遣飛機從澳門飛往[[北京|北京]]和[[上海|上海]]。2002年10月7日起一条[[深圳|深圳]]经由澳门前往[[台北|台北]]的[[货运|货运]]航线开始运作，这是澳门航空的第一条完全货运航线。

成立之初，澳航作為“兩岸中轉的橋樑”，定位於兩岸客貨運輸中轉市場，主要經營大陸和台灣地區之間的中轉聯程旅客和貨物運輸業務。

===2002-2008年：迅速發展===
2002年10月8日，澳門航空公司一架[[波音727|波音727]]貨機開通第一條以接駁形式往返深圳—澳門—台北之間的全貨機航線。<ref>{{cite web|title=中國飛機百科|url=http://www.chineseaircraft.cn/showss.php?id=1791|accessdate=|archive-url=https://web.archive.org/web/20170114120307/http://www.chineseaircraft.cn/showss.php?id=1791%7Caccessdate%3D%7D|archive-date=2017-01-14|dead-url=yes}}</ref>2004年，澳門航空正式成立獨立的貨運部門<ref>{{cite web|title=澳門航空成立貨運部門|url=http://www.airmacau.com.mo/about/news_articles.php?id=4|accessdate=2004-05-11}}</ref>， 並引進兩架[[空中巴士A300|空中巴士A300B4F]]全貨機，進一步拓展至純貨運航班，提供服務至[[台北|台北]]、[[南京|南京]]，其後更開辦[[上海|上海]]、[[廈門|廈門]]、[[新加坡|新加坡]]、[[曼谷|曼谷]]等的全貨運航點。

2008年以前，往來兩岸的臺灣人，大多知曉或搭乘過所謂「一機到底」澳門模式。當時[[海峽兩岸|海峽兩岸]]尚未直航，往來兩岸必須在[[香港|香港]]或澳門中轉，而所謂「一機到底」模式，乘客只需在澳門機場停留30分鐘左右，更加省時和便捷。{{citation needed}}

===2008-2009：陷入危機===
2008年，海峽兩岸開放直航，對澳航賴以生存的中轉市場產生重大衝擊。兩岸中轉客貨運市場大幅萎縮，澳航失去了佔客運業務60%以上的兩岸中轉旅客，公司收入大幅下滑，之後又遭遇全球金融危機，澳門經濟低迷，澳門航空陷入嚴重虧損，瀕臨破產邊緣。

与此同时，[[澳門特區政府|澳門特區政府]]、[[中央人民政府駐澳門特別行政區聯絡辦公室|澳門中聯辦]]、[[中国国际航空|中國航空集團]]对澳门航空进行资助，由國航牽頭對澳門航空進行了股權重組和注資，向澳航注資1.59億元，同時派出了專業團隊，開啟了澳航艱難的轉身。轉型過程中，澳航首先將兩岸中轉聯程市場轉變為以澳門為旅遊、休閒、娛樂目的地的點到點市場，依託中国經濟和澳門經濟的高速發展，在中国市場加大網絡佈局建設和市場營銷。同時向國航“[[濕租|濕租]]”飛機削減過剩運力，提高飛機資產使用效率，逐漸退出貨機市場，削減虧損航線。{{citation needed}}

===2010至現在：穩步向前===
從2010年至2014年，公司實現了5年連續盈利，總資產比2008年翻了4.5倍。從當年的[[北京|北京]]、[[上海|上海]]、[[台北|台北]]三個航點起步，建成如今覆蓋海峽兩岸及東南亞、東北亞等地區的航線網。<ref>{{cite web|title=【紮根本土 翱翔藍天】寫在澳門航空首航20週年之際|url=http://mp.weixin.qq.com/s?__biz=MzA3MzQ1MDU4Mg==&mid=400344826&idx=1&sn=54b0b8b468258c9aa6ef49effe7c1fc7#rd|accessdate=2015-11-12}}</ref>

澳門航空於2011年11月通過向澳門特別行政區政府配售新股而獲得約7億元現金，是次股權重組，發行420,420股新股，全部均為記名的普通股，每股票面值一百澳門元，每股帳面溢價1,565元，全部由政府認購<ref>http://www.airmacau.com.mo/news_detail.php?id=156&lang=cht{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，使政府持股量從5%增至21.5%，成為澳門航空的第二大股東。但持有66.9%股權的中國國際航空有限公司依然是其最大控股股東。<ref>http://www.bizintelligenceonline.com/content/view/1166/lang</ref>

2012年澳門航空有近2.8億澳門元的淨利潤，較2011年增11.4%。2012年總收入達30.3億澳門元，較2011年增19%。另外，2012年客運量及飛機起降班次分別有161萬人次及16,309架次，同比分別增15.8%及12%。貨運增12.9%至12,438噸。

澳門航空2013的業績顯示，營業收益達35.08億（澳門元，下同），較一二年的32.91億增約6.6%。由於營業成本及費用增加，去年全年盈利較一二年的2.8億微增不到1%至2.815億。澳航已連續四年實現逾兩億盈利。根據澳門航空與特區政府於○九年訂立的優先股發行協議及《商法典》第四百二十條，向優先股股東宣佈分派金額750萬，相當於優先股總票面值5%的優先股息，<ref>{{cite web |url=http://www.macaodaily.com/html/2014-06/25/content_913638.htm |title=存档副本 |accessdate=2014-08-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20140801212135/http://macaodaily.com/html/2014-06/25/content_913638.htm |archivedate=2014-08-01 }}</ref>並要求將已宣佈分派的股息投放於設立澳門航空的公司章程第十四條所指的不可分派的公積金，而253,933,428元的盈餘將帶入2014年。<ref>http://bo.io.gov.mo/bo/ii/2014/26/anotariais_cn.asp#614</ref>

== 服務 ==
2005年12月23日，澳門航空成ISO9001:2000品質管理體系認證。

截至2009年9月15日，澳門航空共有職員1002名。其中澳門本地員工佔 31.8%，員工來自世界28個國家和地區。員工平均年齡28歲。

===餐飲===
澳門航空的航班通常提供正常餐及特別配餐，特別餐包括兒童餐、糖尿病餐、水果餐 、印度餐、低卡路里餐、低脂/低膽固醇、中式素食、西式素食，可於訂位時提出需求，最遲於起飛前24小時必須提出。

=== 機場貴賓室 ===
澳門航空於澳門國際機場離境大廳2樓設有其專屬貴賓室，給予商務艙旅客及鳳凰知音終身白金卡、白金卡及金卡會員使用。澳航貴賓室可容納122人，為旅客提供舒適優雅的環境及輕膳美食，亦有免費無線上網服務，供旅客待機休息。<ref>{{cite web|title=澳門航空澳門國際機場貴賓室啟用|url=http://www.airmacau.com.mo/news_detail.php?id=264&lang=cht{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 合作關係 ==
雖然澳門航空並未加入[[航空聯盟|航空聯盟]]，卻與[[星空聯盟|星空聯盟]]部分成員航空公司的關係非常密切

2010年中，澳門航空正式宣佈與同樣是星空聯盟成員的全日本空輸成為合作伙伴，除了相互共享航班代號外，兩公司的常客獎勵計劃 - 澳尊會及ANA Mileage Club之會員均能在澳航或全日空營運之航班中賺取飛行里數。

2012年3月26日起，中國國際航空屬下之飛行常客計劃 - 國航知音正式開始與澳門航空展開合作，兩個常客獎勵計劃的會員搭乘澳航或國航之航班均能累積里數 。

==  海峽兩岸  ==
自從起步之時，澳門航空一直在[[海峽兩岸|海峽兩岸]]運送旅客的過程中扮演了至關重要的角色。目前，每周有28班澳门至[[台北|台北]]的往返班次及14班澳门至高雄的往返班次。

澳門航空在提供海峽兩岸空中交通服務的重要性方面，也可以從它自1995年營運以來就開始提供的獨有服務中體現出來。在2003年兩岸開辦[[兩岸包機|直航包機]]乃至2009年兩岸定期航班開辦之前，澳門航空是海峽兩岸唯一一家提供“一機到底”服務的航空公司。自台北來往北京、上海、廈門等中國大陸城市的旅客，只需要在[[澳門國際機場|澳門國際機場]]经停期间攜同自己的手提行李，辦理一個快速的換機手續，然後在離境大堂逗留大概30分鐘，就可以乘坐同一班飛機到達其目的地。同時，澳門航空也可以在航班上為台灣旅客辦理進入[[中國大陸|中國大陸]]所需的[[台湾居民来往大陆通行证|台湾居民来往大陆通行证]]。

== 事故 ==
2006年3月4日，一架原定从[[澳門國際機場|澳門國際機場]]飛往[[桃園國際機場|台北桃園國際機場]]的[[空中客车A320系列|空中巴士321-231]]客機（航班編號：NX628，飛機註冊編號：B-MAJ），航機啟航前由1部牽引工程車以拖曳設備拖離機場登機大樓，前往跑道準備起飛；於後推其間，疑似航機煞停，造成拖桿超過設計承受應力斷裂；意外發生原因因急減速時衝力，導致航機上旅客及機組人員來不及防範，向前趴倒或往前衝去，事件造成3傷。<ref>{{cite web|title=台女客澳航廁內摔傷 牽引機急煞肇禍 2空姐滾地葫蘆|url=http://www.appledaily.com.tw/appledaily/article/property/20060305/2446947/|accessdate=2006年3月5日 }}</ref><ref>{{cite web|title=ACCIDENT INVESTIGATION REPORT|url=https://www.hitpages.com/doc/4926559416745984/41#pageTop|accessdate=12 August 2008|deadurl=yes|archiveurl=https://web.archive.org/web/20160408115617/https://www.hitpages.com/doc/4926559416745984/41#pageTop|archivedate=2016年4月8日}}</ref>

===彩繪機===
[[File:Air_Macau_Airbus_A321-231;_B-MAJ@HKG;31.07.2011614ky_(6053203558).jpg|thumb]](已恢復標準塗裝)]]
[[File:B-MBM@PEK_(20141108123213).JPG|thumb]]]]

澳門航空作為[[2005年東亞運動會|2005年東亞運動會]]指定航空公司，故在2003年12月將機隊轄下的一架A321-231客機（航機編號B-MAJ ）的外觀粉飾一新，除噴上“2005年東亞運動會徽”之外，更有運動會吉祥物柏柏的圖案，並將該機命名為「松山燈塔號」，該機成為澳門航空公司首架全彩繪客機。

2014年11月，澳門航空引進的第三架全新空中巴士A321客機（航機編號B-MBM 吉祥號），機身噴繪了[[澳門|澳門]]旅遊形象，噴繪圖案由兩部分組成, 第一部分機身前方, 線條靈感源自龍的鬚鬢，輕重由下而上躍起，比喻飛機翱翔天際時猶如矯健的飛龍；紅綠顔色源自[[澳門旅遊局|旅遊局]]標誌。第二部分線條大面積由上而下（當中包含澳門代表性建築物[[大三巴牌坊|大三巴牌坊]]、[[媽閣廟|媽閣廟]]、[[澳門旅遊塔|旅遊塔]]等）慢慢收窄延伸至其它建築群的輪廓；深綠色源自澳門特區區旗，淺綠色作襯托。兩部分組合起來便構成一種連接狀態，起承轉合。整個意念、構思、設計，就是希望讓更多人認識澳門。這也是也是澳門航空首架配備「鯊鰭小翼」設計的機型。<ref>{{cite web|title=澳門航空再次迎來全新航機|url=http://www.airmacau.com.mo/news_detail.php?id=239&lang=cht|accessdate=November 2015|deadurl=yes|archiveurl=https://web.archive.org/web/20141223025018/http://www.airmacau.com.mo/news_detail.php?id=239&lang=cht|archivedate=2014-12-23}}</ref>

2019年1月，澳門航空與旅遊局再度合作推出彩繪機（航機編號B-MCI 澳門號），該機此前為[[中國國際航空|中國國際航空]]所使用（航機編號B-6676），為 A320-232 型空中巴士，繪有澳門旅遊吉祥物「麥麥」圖案，以推廣澳門形象；暫時主要航線為[[澳門國際機場|澳門]]到[[成田國際機場|東京]]和[[澳門國際機場|澳門]]到[[內排國際機場|河內]]。
