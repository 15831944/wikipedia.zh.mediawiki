'''澳大利亚历史'''是从距今40,000至70,000年前[[人类|人类]]第一次在[[澳洲|澳洲]]大陆北岸定居开始的。这段时间被定义为[[澳大利亚史前时期|澳大利亚史前时期]]。有文字记载的澳大利亚历史开始于[[欧洲|欧洲]]探险家第一次达到这片土地。这段时期又分为两个阶段：这两个阶段以1901年成为[[大英帝国|大英帝国]]领地前后为分界点。

== 史前时期 ==
从人類到达[[澳洲|澳洲]]后，至欧洲人於1606年第一次踏足澳洲期间的大约四万年，被称为[[澳大利亚史前时期|澳大利亚史前时期]]。把这段时期归为史前而非历史是因为1606年以前，澳大利亚没有[[人類|人類]]生活的[[文字|文字]]记载。

== 航海大發現時期(1606至1788年) ==
尽管有葡萄牙人发现澳洲的相关理论，但是这个仍缺少相关证据。<ref>McIntyre, K.G. 17世纪早期，其他欧洲国家探险家（大部分来自荷兰、[[法国|法国]]、[[英国|英国]]）相继来到澳洲。

18世纪初，“新荷兰”的西岸和北岸被绘于[[地图|地图]]。可是，当时并没有人尝试在这里定居。18世纪中叶，库克船长绕行澳大利亚。1768年，[[英国|英国]][[庫克船長|庫克船長]]乘坐「奮進號」從英國出發前往[[大溪地|大溪地]]。1770年，庫克船長發現澳洲東部，並佔領及命名為大英。並绕行澳大利亚。

==英屬新南威爾斯時期==
1788年1月26日（[[澳大利亞日|澳大利亞日]]），英国殖民者在澳大利亚东南建立第一个殖民地「[[新南威爾斯|新南威爾斯]]」，为英國本土犯人的流放地。1797年，麥卡瑟(J. Macarthur)由[[好望角|好望角]]將美麗諾羊（Merino sheep）引進澳洲。

1804年，[[霍巴特|霍巴特]]城建立。1813年，首次引進貨幣 - 有孔錢幣及銀幣（dump）。

== 西澳淪為囚犯流放地時期(1828年至1901年) ==
1829年，[[西澳大利亞|西澳]]被吞併，成為囚犯流放地。1851年，[[新南威爾斯|新南威爾斯]]的貝瑟斯特及[[維多利亞州|維多利亞州]]的[[柏拉瑞特_(澳大利亞)|柏拉瑞特]]發現金礦。1862年，史都特（John Stuart）成為第一位穿越澳洲南至北的探險家。1868年，最後一批囚犯被運送至西澳。

1873年，歐洲人首次見到[[烏魯汝|烏魯汝]]，Uluru（Ayres Rock，愛麗斯岩）。

在经过长达10年的計畫，讨论，投票后，1901年1月1日组成了第一个隶属于[[英国|英国]]的澳洲聯邦（Commonwealth of Australia）。

== 英屬澳大利亞聯邦[[墨爾本|墨爾本]]首府時期 ==
1901年1月1日，几个[[英国|英国]]的[[殖民地|殖民地]]组成了第一个隶属于[[英国|英国]]的澳洲聯邦（Commonwealth of Australia），白澳政策及移民限制法被通過。1914年至1918年，澳洲參與[[第一次世界大戰|第一次世界大戰]]。1920年，澳洲航空公司（Qantas）國內航線成立。1921年，伊迪斯科文（Edith Dircksey Cowan）成為澳洲首位女性[[國會|國會]][[議員|議員]]。1927年前，澳大利亚的[[首都|首都]]是[[墨尔本|墨尔本]]。

== 英屬澳大利亞聯邦[[坎培拉|坎培拉]]首府時期 ==
1927年，墨爾本與悉尼之爭首都，經協議後遷往折衷點[[堪培拉|堪培拉]]。同年，第一次澳洲聯邦會議於[[坎培拉|坎培拉]]臨時國會大廈舉行。

澳大利亚在兩次[[世界大战|世界大战]]中跟[[英国|英国]]並肩作战。[[二次大战|二次大战]]以后，大規模的[[移民|移民]]计划让澳大利亚产生了很大的改变。在[[澳大利亚政府|澳大利亚政府]]的“[[白澳政策|白澳政策]]”被废除后（1972年），很多[[亚洲|亚洲]]及世界各地的人移民到至澳大利亚，改变了澳大利亚的[[人口|人口]]组成、[[文化|文化]]和[[形象|形象]]。

1931年，英国颁布的《[[1931年西敏寺法令|西敏寺法令]]》基本让澳大利亚联邦拥有独立宪政（终止了与英国的宪政联系）。1932年，[[雪梨海港大橋|雪梨海港大橋]]啟用。1933年，西澳公投表決脫離英國獨立，但遭國會否決。

1939年至1945年，澳洲參加[[第二次世界大戰|第二次世界大戰]]。1940年，第一輛澳洲製造的汽車面世。

== 逐漸脫離英国 ==
1942年，澳大利亚联邦正式采纳1931年公布的西敏寺法令。但由于澳大利亚受到[[大日本帝國|大日本帝國]]的潜在威胁和英国在亚洲节节败退，让澳大利亚的外交政策开始转向與[[美国|美国]]友好。

1948年，霍頓（Holden）為第一輛完全澳洲自製汽車。

1951年，澳大利亚成为美国的正式盟友，并签订[[美澳新条约|美澳新条约]] ({{lang|en|ANZUS}})。

1956年，舉辦[[1956年夏季奧林匹克運動會|墨爾本奧運]]。

1967年經過公投後，通過廢除原住民法律歧視。1973年[[雪梨歌劇院|雪梨歌劇院]]啟用。

1972年大选，工党執政后，越来越多澳大利亚国民认为，未来他们的国家會成为亚洲的一部分。1984年，「Advance Australia Fair」 成為澳洲[[國歌|國歌]]。

1986年，澳大利亚宣佈[[澳洲法案|澳洲法案]]（Australia Act），正式脫離英國獨立，英国的当时[[最高法院|最高法院]]－[[枢密院司法委员会|枢密院司法委员会]]从此不再是澳大利亚的[[终审法院|终审法院]]。但直到现在，澳大利亚仍然是[[君主立宪制|君主立宪制]]国家，[[伊丽莎白二世|伊丽莎白二世]]女王仍然名義上的澳大利亚女王。只要英女王仍在位，澳大利亞不会成立共和国。

1988年，澳洲开埠200週年，当年[[國慶日|國慶日]]－坎培拉新國會大廈啟用。


1996年，[[約翰·霍華德|約翰·霍華德]]當選第25任澳大利亚總理，結束了13年的工党主政。

1999年，澳大利亚人否决了转变为[[议会制|议会制]][[共和国|共和国]]的[[全民公投|全民公投]]。

2000年，悉尼主辦[[2000年夏季奧林匹克運動會|奧林匹克運動會]]。

2007年12月，澳大利亚工党领袖[[陆克文|陆克文]]当选第26任澳大利亚總理，结束保守派联盟12年主政。

2008年，陆克文代表澳大利亚政府向[[澳大利亚原住民|澳大利亚原住民]]作正式道歉。

2010年，澳大利亚工党领袖[[朱莉亞·吉拉德|朱莉亞·吉拉德]]取代陆克文成为总理。
