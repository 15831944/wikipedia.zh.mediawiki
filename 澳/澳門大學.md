 [[File:Symbol_UM_logo.png|200px]]'''澳門大學'''（{{lang-pt|'''Universidade de Macau'''}}；{{lang-en|'''University of Macau'''}}；簡稱'''澳大'''、'''UM'''）是澳門本地區最優秀的國際化、唯一一所綜合性公立大學<ref>{{cite web |title=澳门大学簡介 |url= https://www.um.edu.mo/zh-hant/about-um/introduction/about-the-university-of-macau.html}}</ref>，位處全球大學最佳2%<ref>{{cite web |title=澳門日報 趙偉：澳大世界排名仍居前 |url = https://www.cyberctm.com/zh_TW/news/detail/1123823#.WjkpblT1VE4}}</ref>，世界前400強大學<ref>{{cite web |title=TIMES RANKING 2019 |url = https://www.timeshighereducation.com/world-university-rankings/university-macau }}</ref>，國際化發展第五，亞太地區大學排名60<ref>{{Cite web|title=電台新聞>澳門大學亞太區排名升至60位(2019.02.21 10:55)|url=https://www.tdm.com.mo/c_news/radio_news.php?id=410094|work=www.tdm.com.mo|accessdate=2019-02-21}}</ref>，全球最佳年輕大學排名60。[[葡萄牙語國家和地區列表|葡語地區]]大學中位居第二<ref>{{cite web |title=澳門平台 澳門大學在葡語地區排名第二 |url=http://www.plataformamacau.com/zh-hant/%E6%BE%B3%E9%96%80/%E6%BE%B3%E9%96%80%E5%A4%A7%E5%AD%B8%E5%9C%A8%E8%91%A1%E8%AA%9E%E5%9C%B0%E5%8D%80%E6%8E%92%E5%90%8D%E7%AC%AC%E4%BA%8C/ }}</ref>。

澳門大學前身為1981年成立的'''私立東亞大學'''，是[[澳門|澳門]]首所[[研究型大學|研究型大學]]，同時亦是唯一一所全面引入住宿式書院系統<ref>{{Cite web|url=https://rc.um.edu.mo/|title=澳門大學住宿式書院|accessdate=2019-01-15|author=|date=|work=rc.um.edu.mo|publisher=}}</ref>的大學。澳門大學於2014年搬遷到[[橫琴|橫琴]]新校區後，明確以轉型至[[研究型大學|研究型大學]]為目標，同時推行以「四為一體」教育模式為主的「[[博雅教育|博雅教育]]」。根據國際論文資料庫[https://webofknowledge.com/ ISI Web of Science（WOS）]，澳門大學的高端科研論文發表數量由2008年的127篇增至2017年的1,342篇, 2017年澳大論文獲引用近17,246次。在芯片研究方面入選國際固態電路會議論文數量最多前10名的大學。澳門大學獲[[中華人民共和國科學技術部|中華人民共和國科學技術部]]批准設立「模擬與混合信號超大規模集成電路國家重點實驗室」、「智慧城市物聯網國家重點實驗室」及「中藥質量研究國家重點實驗室」三個[[中华人民共和国国家重点实验室列表|國家重點實驗室]]<ref>{{cite web |title = 澳門大學 簡介 |url = http://www.um.edu.mo/zh-hant/about-um/overview/about-the-university-of-macau.html }}</ref>。

澳門大學設有人文學院、工商管理學院、教育學院、健康科學學院、法學院、社會科學學院、科技學院、中華醫藥研究院、微電子研究院、[[應用物理學|應用物理]]及[[材料科學|材料工程]]研究所，[[協同創新|協同創新]]研究所和榮譽學院 <ref>{{cite web |title=澳大簡介 學院、住宿式書院及部門 學院 / 研究院 |url = http://www.um.edu.mo/zh-hant/about-um/faculties-residential-colleges-and-units/faculties.html }}</ref> ，開辦[[學士|學士]]、[[碩士|碩士]]、[[博士|博士]]等約130多個學位課程，授課語言以[[英語|英語]]為主，部分課程以葡萄牙語、中文或日語授課。

澳門大學設有10所書院，包括曹光彪書院、鄭裕彤書院、張崑崙書院、蔡繼有書院、霍英東珍禧書院、呂志和書院、馬萬祺羅柏心書院、滿珍紀念書院、紹邦書院及何鴻燊東亞書院。

澳門大學在2019年[[泰晤士高等教育世界大學排名|泰晤士高等教育]]世界大學排名351-400，國際化排名第五，在2019年亞太地區大學排名60<ref>{{Cite web|title=電台新聞>澳門大學亞太區排名升至60位(2019.02.21 10:55)|url=https://www.tdm.com.mo/c_news/radio_news.php?id=410094|work=www.tdm.com.mo|accessdate=2019-02-21}}</ref>，全球最佳年輕大學排名60<ref>{{Cite web|title=University of Macau|url=https://www.timeshighereducation.com/world-university-rankings/university-macau|work=Times Higher Education (THE)|date=2018-02-05|accessdate=2019-01-25|language=en}}</ref>。在Quacquarelli Symonds（QS）世界大學排名也節節上升，在2019年上升至443位。“工程與科技”與“計算機科學”排名前200、“生命科學”排名251-300、“款待及休閒管理學” 排名46、“語言學” 排名151-200、“法學” 排名251-300位。五大學科領域：臨床醫學學科、工程學、藥理學與毒理學、計算機科學及社會科學總論進入基本科學指標資料庫（ESI）前1%。

澳門大學近年排名及學術地位都有大幅提升，於2016年成為首間獲[[澳門特區政府|澳門特區政府]]頒發[[教育功績勳章|教育功績勳章]]的澳門[[高等院校|高等院校]]<ref>{{cite web |title=澳大獲澳門特區政府頒教育功績勳章
|url=https://ado.umac.mo/um-awarded-medal-of-merit-education-by-macao-sar-government/?lang=zh-hant }}</ref>。澳門大學校長宋永華表示，澳大致力發展成為一所立足澳門、融入灣區、面向全國、 走向世界的卓越大學。<ref>{{Cite web|title=澳門日報電子版|url=http://macaodaily.com/html/2018-04/19/content_1259416.htm|work=macaodaily.com|accessdate=2019-01-28}}</ref>

== 歷史 ==
=== 東亞大學時期 ===
[[File:澳大夜景.JPG|230px]]
[[File:九龙壁.JPG|230px]]
澳門大學的創校歷史最早可以追溯至1978年。當時來自[[香港|香港]]的[[黃景強|黃景強]]、胡百熙與吳毓璘向[[澳葡政府|澳葡政府]]申請撥地開辦大學，最後政府批出位於[[氹仔島|氹仔島]]北岸一處山丘作為校址。三名創辦人隨即成立了大學籌建委員會，並以開辦國際性大學為目標籌辦東亞大學。東亞大學為[[澳門|澳門]]首間現代高等教育機構，其創立標誌著澳門現代高等教育的開端<ref>{{cite web|title=澳門大學歷史|url=http://www.um.edu.mo/zh-hant/about-um/history-milestones/history.html}}</ref><ref>{{cite web|title=澳門大學里程碑|url=http://www.um.edu.mo/zh-hant/about-um/history-milestones/milestones.html}}</ref>。 

大學校舍由香港建築教育學者[[劉秀成|劉秀成]]設計，在1979年底動工。東亞大學籌建委員會於1980年委任[[中文大學|香港中文大學]][[聯合書院|聯合書院]]院長[[薛壽生|薛壽生]]為首任校長。1981年3月28日，大學籌建委員會在當時的[[回力球场_(澳门)|澳門回力球場]]舉行了東亞大學成立典禮，委員會名譽主席[[何賢|何賢]]先生在儀式上向來自世界各地一百三十多所大學的校長和代表、以及二千多位澳門各界嘉賓致歡迎詞，宣佈東亞大學正式成立。1981年9月，大學正式開課。當時大學根據英式教育制度，設有預科學院和大學學院，前者提供一年制的大學[[預科|預科]]和兩年制的[[副學士|副學士]]課程，後者提供三年制的文學、社會科學和工商管理本科專業課程，學生多數來自[[香港|香港]]，也有來自[[澳門|澳門]]本地。1984年，東亞大學培養的首批學士學生共五十多人畢業。

東亞大學創立時以「仁義禮知信」為校訓。在創校的首五年，大學在[[澳葡政府|澳葡政府]]、[[何賢|何賢]]、[[王寬誠|王寬誠]]和[[邵逸夫|邵逸夫]]等的支持和捐獻下，建成了以大豐樓、何賢中心、文化中心、王寬誠樓和學生宿舍為主的校園，同時建立了澳門最具規模的圖書館。1985年，[[中華人民共和國國家教育委員會|中華人民共和國國家教育委員會]]專門打造了仿[[故宮|故宮]][[九龍壁|九龍壁]]的雕塑贈送給東亞大學。

隨著[[中葡聯合聲明|中葡聯合聲明]]在1987年簽署，為了配合[[澳門回歸|澳門回歸]]前過渡期的人力需求，[[澳葡政府|澳葡政府]]屬下的[[澳門基金會|澳門基金會]]在1988年以1.3億[[澳門元|澳門元]]購入東亞大學<ref>《圖片澳門今昔（修訂版）》，劉羨冰、陳樹榮、王國強、冼為鏗著，三聯書店（香港）有限公司出版，2009年</ref>，將次重組為公立大學和增設立科技學院、教育學院和法學院，並將本科學院分拆為文學院、社會科學院及工商管理學院。1989年，[[葡萄牙總統|葡萄牙總統]][[马里奥·苏亚雷斯|蘇亞雷斯]]到訪東亞大學。1990年起，大學將英式學制的三年制學士課程改為葡式學制的四年制學士課程，教學語言仍以[[英語|英語]]為主。

=== 澳門大學時期 ===
[[File:50_patacas_（BOC）_1995_series_-_front.jpg|230px]]1995版50[[澳門元|澳門元]]，正面圖案為澳門大學]]
[[文件:UMacau_25th_Anniversary_Souvenir_Sheet.jpg|230px]]
1991年9月，根据第50/91/M號法令，[[葡澳政府|澳葡政府]]將東亞大學更名澳門大學，並在次年二月頒佈《澳門大學章程》<ref>[http://bo.io.gov.mo/bo/i/91/37/declei50_cn.asp 印務局 - 第50/91/M號法令]</ref>，並以培養澳門人才為目標。大學重組後，學生人數急劇上升。截至澳門回歸前夕，學生人數增至約三千人，而澳門本地學生的比重亦由創校時的39%增至80%。原屬東亞大學的理工學院則獨立為公立的[[澳門理工學院|澳門理工學院]]，而東亞大學的研究院和公開學院則合併為[[澳門城市大學|東亞公開學院]]，以私營模式運作。1995年，首屆碩士生畢業，同年[[中國銀行澳門分行|中國銀行澳門分行]]開始發行1995年版[[澳門幣|澳門幣]]，其中50元紙幣的正面圖案為澳門大學遠景。1997年，大學首位博士生畢業，[[葡萄牙共和國|葡萄牙共和國]][[葡萄牙总统列表|總統]][[沈拜奧|沈拜奧]]於同年到訪澳門大學。

[[澳門回歸|澳門回歸]]後，因應大學完成了配合過渡期培養人才的使命，大學增加學科、優化校園設施、提升師資質量和強化研究實力，並推動設置一系列新課程、調低學費及提供獎學金和財政支援。2002年，大學成立修章工作小組，啟動對大學章程的修訂工作，實現大學步向現代化和國際化，並於2006年通過新大學章程。修章帶來的學術自由和行政彈性加速了大學日後發展步伐。2006年也是澳門大學建校25週年校慶，[[澳門郵政|澳門郵政]]特于9月28日發行了[[紀念郵票|紀念郵票]]，[[小型張|小型張]]的圖案為行政樓和[[澳門總督|澳門總督]][[韦奇立|韦奇立]]贈予澳大的畢業生銅像。2007年2月，[[葡萄牙總理|葡萄牙總理]][[若瑟·蘇格拉底|若瑟·蘇格拉底]]到訪澳門大學。

2009年，[[中華人民共和國國務院|中國國務院]]將[[珠海市|珠海市]][[橫琴島|橫琴島]]東部，[[橫琴口岸|橫琴口岸]]南面，與路氹城一河之隔的一幅約一平方公里土地租借給澳門特別行政區作為興建[[澳門大學新校區|澳門大學新校區]]之用，並批准依[[澳門法律|澳門法律]]實施管轄。

新校園在2009年12月20日奠基，時任中共中央總書記、國家主席[[胡錦濤|胡錦濤]]在[[澳門特別行政區行政長官|澳門特首]][[崔世安|崔世安]]的陪同下出席了新校園的奠基儀式。2013年7月20日，澳門大學新校區正式使用[[澳門法律|澳門法律]]實施管轄，11月5日新校園正式啟用。新校區由中國建築師[[何鏡堂|何鏡堂]]設計，比原校園大20倍<ref>[http://www.um.edu.mo/new_campus_project/chi/background/]</ref>。新校園在行政區劃上被納入[[氹仔|氹仔]]，法定地址為澳門氹仔大學大馬路。2014年5月18日，[[葡萄牙總統|葡萄牙總統]][[阿尼巴尔·卡瓦科·席尔瓦|席爾瓦]]率團訪問澳門大學。2014年12月20日, 中共中央總書記、國家主席[[習近平|習近平]]考察澳門大學，參觀大學展館和住宿式書院，讚揚澳大辦學制度理念創新。2018年4月16日，[[中华人民共和国教育部|中國教育部]]向澳門大學贈送的“博雅之璧”雕塑，喻意弘揚中華優秀傳統文化和美德。

== 形象標識 ==

澳門大學的校徽由一個五座城堡冠頂的盾牌和分別用中葡文書寫的“澳門大學”的校名字樣組成。書上的鑰匙是知識之匙，用以打開知識寶庫；四周的海浪代表著大學身處中國海域的一個小島上。大橋寓意文化的溝通，是連接東西方文化的象徵。“仁、義、禮、知、信”為中國聖賢提倡的五種美德典範。校徽採用紅、藍、金三色，代表著希望、幸福及人類之進步。<ref>{{Cite web|title=澳門大學 {{!}} 澳門唯一的綜合性公立大學|url=https://www.um.edu.mo/zh-hant/about-um/university-identity/elements-of-the-university-identity.html|work=www.um.edu.mo|accessdate=2019-02-11}}</ref>

澳門大學創校時將中國聖賢提倡的五種美德“仁、義、禮、知、信”奉為校訓，為儒家思想的核心，也代表澳門大學致力培養具備這五種崇高品格學生的使命。

== 校園環境 ==

=== 舊校園 ===
{{main|澳門大學舊校園}}
澳大舊校園是指1981年隨東亞大學開學而正式啟用的校園。1991年起因政府收購東亞大學而成為當時澳門大學的校園，直到2014年8月22日遷出該校園為止。

=== 新校園 ===
{{main|澳門大學新校區}}
澳大新校園是澳門大學自2014年8月23日正式啟用的校園。該校園位於橫琴島，始建於2009年12月，歷時三年多於2013年7月正式落成並交由澳門政府管理，澳門大學搬遷工作同期展開，直到2014年8月23日正式啟用。2014年9月5日，[[澳門郵政|澳門郵政]]專門發行了澳門大學新校園郵票。
