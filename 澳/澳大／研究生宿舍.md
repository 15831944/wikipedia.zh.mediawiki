'''澳大橫琴總站'''([[葡萄牙文|葡萄牙文]]'''{{lang|pt|UM DE HENGQIN/TERMINAL}}''')位於[[澳門大學新校區|澳門大學新校區]]內的一個終點站。[[新福利|新福利]]之'''[[澳門巴士37U路線|37U]]'''[[澳巴|澳巴]]之'''[[澳門巴士MT3U路線|MT3U]]路線以該站為終點站。

== 沿革 ==
隨著澳大新校區逐步投入使用，學生數量增多，交通事務局亦已計劃在澳大新校區全面投入使用後，逐步增加路線接駁至校區，以方便學生和教師乘坐巴士出行。

== 歷史 ==
* 2014年6月25日起，本站正式啟用。

== 設施 ==
*終點站路線行車指示牌。
*新福利站長室。
*本站設有站長。

== 各車道安排 ==
*只有一條車道。
