'''澳門土生葡語'''（{{lang-pt|Patuá macaense、patois、Crioulo macaense}}），又稱'''土生葡文'''或'''帕圖亞'''，亦有「澳門甜蜜之語」（{{lang-pt|Dóci Língu di Macau}}）和「甜美的語言」（{{lang-pt|Doci Papiaçam}}）之稱，是一種以15至16世紀的[[葡萄牙語|葡萄牙語]]為基礎的[[克里奧爾語|克里奧爾語]]，是[[澳門土生葡人|澳門土生葡人]]使用的一種獨特[[方言|方言]]，在[[澳門|澳門]]已有三百多年的歷史<ref name = "L01">{{cite journal |author=[[黎熙元|黎熙元]] |date=2001  |title=澳門土生葡人族群及其文化特點 |url= http://210.39.5.17/bitstream/510500.239/5044/1/%E6%BE%B3%E9%97%A8%E5%9C%9F%E7%94%9F%E8%91%A1%E4%BA%BA%E6%97%8F%E7%BE%A4%E5%8F%8A%E5%85%B6%E6%96%87%E5%8C%96%E7%89%B9%E7%82%B9.pdf|journal= 學術研究|volume=12 |issue= |pages= |doi= |access-date= 2017-07-07}}</ref><ref name = "F01">{{Cite book | author = 霍志劍 | title = 《澳門土生葡人的宗教信仰從“「單一」到「多元混融」》 | location = | publisher = 社會科學文獻出版社 | date = 2009-07 | pages = 174-175 | ISBN = 9787509711507| accessdate = 2017-07-07 | url =  | language = 中文 | quote = }}</ref>。截至2017年，能夠流行地操土生葡語的人剩下不到五十人，土生葡語亦被[[聯合國教科文組織|聯合國教科文組織]]列為[[瀕危語言|瀕危語言]]<ref>{{cite news|author1=Matthew Keegan|title=Macau’s Native Language Is on the Verge of Disappearing|url=https://theculturetrip.com/asia/china/articles/macaus-native-language-is-on-the-verge-of-disappearing/|access-date=2018-01-30|agency=The Culture Trip|date=2017-10-18}}</ref>。

==演進==
土生葡語的基本詞彙以葡萄牙語為基礎，且廣泛加入包括[[馬來語|馬來語]]、[[粵語|粵語]]、[[英語|英語]]、[[爪哇語|爪哇語]]、[[日語|日語]]、[[梵文|梵文]]及少量[[西班牙語|西班牙語]]、[[意大利語|意大利語]]等語言的詞語混合而形成<ref>{{cite book  |title=《澳門語言研究》|author=黃翊  |publisher=[[商務印書館|商務印書館]] |isbn=9787100054171}}</ref>。在16世紀，葡萄牙人先後在[[亞洲|亞洲]][[馬六甲|馬六甲]]、[[斯里蘭卡|斯里蘭卡]]、[[印尼|印尼]]等地定居，隨著他們與當地人不斷通婚，帕圖亞語逐漸流行起來，在這個階段，土生葡語深受[[馬來語|馬來語]]和[[僧伽羅語|僧伽羅語]]的影響。到了17世紀，葡萄牙人攜帶在上述地區不同種族的土著在澳門定居，這批新移民對日後土生葡語的影響非常深遠。數個世紀以來，澳門的人種比例一直有所改變，不同國籍的人之間一直互相往來，導致土生葡語的運用、[[語法|語法]]、[[辭彙|辭彙]]都有相應的變化<ref name = "F01"/><ref name = "tdm01"/>。

據[[葡萄牙|葡萄牙]][[語言學家|語言學家]]白妲麗（Graciete Batalha）的研究，葡萄牙人在亞洲數地留下的語言，在葡萄牙人攜帶不同種族的土著人在澳門定居時，已超過了共同交際混合語的範疇。作為交流手段，他們使用「一種以某種程度上來講，已成熟的語言，有豐富的詞彙量，語音、詞法及句法已經穩定。這種語言在此持續了三百年。」<ref name = "F01"/><ref name="tdm01">{{Cite web |url = http://www.tdm.com.mo/c_radio/radionpgm10/?p=1711 |title = 土生葡人－澳門土生土語 |author =  |date = 2012年5月14日 |publisher = [[澳門廣播電視股份有限公司|澳門廣播電視股份有限公司]] |language = zh |access-date = 2017-07-07 |quote =  |archive-url = https://web.archive.org/web/20170911011809/http://www.tdm.com.mo/c_radio/radionpgm10/?p=1711 |archive-date = 2017年9月11日 |dead-url = yes }}</ref>

19世紀中期，土生葡語在澳門已經成為了土生葡人的共同使用的語言。到了19世紀後期，由於官方葡語教育的普及以及[[廣東話|廣東話]]、[[英語|英語]]地位的提高，土生葡語開始衰落。到了20世紀，随着教育和广播电视的普及，這種語言已開始式微，而其地位亦被正统葡語所取代。1951年，白妲麗對土生葡語的形成和前景作出以下描述：「可能正是在19世紀，由於先前同印度和馬六甲之間密切的關係銳減，跟[[宗主國|宗主國]]（葡萄牙）又保持隔離狀態，才使澳門方言達到其發展的最高峰，並形成了獨特的個性。近50年來，澳門方言逐漸失去其活力，如果說它還未完全消失，那是因為它對過去的記憶像花須一樣緊緊盤繞在現代語言上，但是在幾十年之後，它將被宣判為『[[絕跡語言|死的語言]]』。」<ref name = "F01"/>

自[[澳門主權移交|澳門主權移交]]後，再沒有正式統計土生葡語的語言人口。根據《[[民族語|民族語]]》的資料，1977年的土生葡語使用者人口大約在4000人<ref name=ethnologue>{{cite book |editor=Gordon, Raymond G., Jr. |year=2005|title=Ethnologue: Languages of the World|edition=15th edition|location=Dallas, Tex.|publisher=[[SIL國際|SIL國際]]|url=http://www.ethnologue.com/show_language.asp?code=mzs|access-date=2008-07-06}}</ref>。根據聯合國教科文組織於2011年11月的估計，現時能操土生葡語的人口僅餘約五十人<ref>{{cite web|url=http://hk.news.yahoo.com/%E8%81%AF%E5%90%88%E5%9C%8B%E5%88%86%E7%B4%9A-%E6%BE%B3%E9%96%80%E5%9C%9F%E8%AA%9E%E7%80%95%E5%8D%B1-213402648.html|title=聯合國分級 澳門土語瀕危|access-date=2011-11-09}}</ref>。即使如此，在澳門仍然有少量老人家仍然以土生葡語為第一語言<ref name=ethnologue />。澳门也有一些艺术团体坚持以土生葡語进行演出。<ref>{{cite news |author1=羅保熙 |title=混血兒的「甜言蜜語」　土生葡人自己語言自己救 |url=https://www.hk01.com/%E4%B8%96%E7%95%8C%E8%AA%AA/195588/%E6%BE%B3%E9%96%80%E4%BA%BA%E8%AC%9B%E4%B9%9C%E8%A9%B1-%E6%B7%B7%E8%A1%80%E5%85%92%E7%9A%84-%E7%94%9C%E8%A8%80%E8%9C%9C%E8%AA%9E-%E5%9C%9F%E7%94%9F%E8%91%A1%E4%BA%BA%E8%87%AA%E5%B7%B1%E8%AA%9E%E8%A8%80%E8%87%AA%E5%B7%B1%E6%95%91 |accessdate=2018-06-07 |work=香港01 |date=2018-06-06}}</ref>
