'''澳門巴士N6路線'''是一條由[[澳門公共汽車有限公司|澳門公共汽車有限公司]]經營，往返'''[[澳門大學|澳門大學]]'''和'''[[氹仔|氹仔]]'''的循環巴士路線。

== 簡介及歷史 ==
* 2016年4月20日，本線開設<ref>{{cite news|title=交通事務局優化夜間巴士路線 周三起增N6路線調整N5路線 |url=http://www.dsat.gov.mo/bus/tc/bus_news_detail.aspx?a_id=3293 |publisher=[[交通事務局|交通事務局]] |date=2016年4月18日}}</ref>，為澳大新校區第一條夜間巴士線，由[[澳門公共汽車有限公司|澳門公共汽車有限公司]]營運，服務時間為00:00-06:00，班距20分鐘，較原來澳大自行開設的夜間穿梭巴士頻密一倍。
*2018年7月21日，本線新增停靠「運動場/賽馬會」站（原名：運動場道）。
*2019年3月9日，為明確巴士站位置「亞利雅架前地/泉悅」站改名為「信虹花園」站。
*2019年3月16日起配合土地工務運輸局建造行人天橋工程的推進，期間暫不停靠「運動場/賽馬會」站。

== 使用車輛 ==
*五洲龍FDG6751G

== 路線資料 ==
===收費===
*全程收費：澳門幣6圓。
*使用澳門通全民卡，全程一律扣減車費3.0圓，餘額由政府支助。
*使用澳門通學生卡，全程一律扣減車費1.5圓，餘額由政府支助。
*使用澳門通長者卡及殘疾卡，全程一律免費，餘額由政府支助。
