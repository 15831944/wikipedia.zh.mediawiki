'''澳門居民'''（{{lang-pt|'''Residente de Macau'''}}），在[[澳門回歸|澳門回歸]]後稱為'''澳門特別行政區居民'''。根據《[[澳門特別行政區基本法|澳門特別行政區基本法]]》第三章第24條，[[澳門特別行政區|澳門特別行政區]]居民分為「[[永久性居民|永久性居民]]」和「[[非永久性居民|非永久性居民]]」兩類；前者具有在澳門的[[居留權|居留權]]，後者則無。

==歷史==
1950年代之前，[[澳門|澳門]]是受[[葡萄牙|葡萄牙]]管治的，但[[澳葡政府|澳葡政府]]沒有良好身份證明系統制度；當時中國籍的澳門居民到[[澳門民政廳|澳門民政廳]]（及後改由[[身份證明司|身份證明司]]處理）申領居住證明，也要到[[治安警察廳|治安警察廳]]申領居住證明，因此中國籍的澳門居民要同時持有兩種[[身份證|身份證]]才可合法居住。

1980年代，民政廳（及後改由身份證明司負責）正式發出居民身份證，是第一代澳門身份證<ref>[http://bo.io.gov.mo/bo/i/84/30/declei79.asp 澳門政府第79/84/M號法令]，2009年2月22日查閱。。{{pt}}</ref>。同一時期，大量中國[[非法移民|非法移民]][[偷渡|偷渡]]進入澳門，引致在澳門無證人士日漸增多；1989年[[龍的行動|龍的行動]]及1990年[[三二九事件|三二九事件]]導致澳葡政府對有關中國籍人士進行身份識別登記，及後被獲發居民身份證<ref>{{cite book

身份證明司及治安警察廳的「雙證制度」沒有一個良好的記載資料方式及造成混亂等情況，因此澳葡政府於1992年1月廢止了雙證制度，並向澳門居民统一換發身份證——居民身份證<ref>身份證明局，[http://www.dsi.gov.mo/documents/law_6-92-M_index_c.html 澳門政府第6/92/M號法令] {{webarchive|url=https://web.archive.org/web/20080917114756/http://www.dsi.gov.mo/documents/law_6-92-M_index_c.html |date=2008-09-17 }}，2009年2月22日查閱。{{zh}}</ref>。及至1996年，由於進入[[澳門回歸|澳門回歸]]前的後過渡期，經過[[中葡聯絡小組|中葡聯絡小組]]商議後，改發不設效期的新的澳門居民身份證。

澳門回歸後，澳門居民始分為'''澳門永久性居民'''（{{lang|pt|'''Residente Permanente de Macau'''}}）和'''澳門非永久性居民'''（{{lang|pt|'''Residente não Permanente de Macau'''}}）兩類；而在2002年，身份證明局為澳門居民更換智能身份證時，把身份證正式分為「澳門永久性居民身份證」和「澳門非永久性居民身份證」兩類，標誌著永久及非永久性居民「正式分家」<ref>澳門立法會，[http://www.al.gov.mo/lei/col_lei-01/cn/07/3-1-123.htm 第8/2002號法律] {{webarchive|url=https://web.archive.org/web/20041223114425/http://www.al.gov.mo/lei/col_lei-01/cn/07/3-1-123.htm |date=2004-12-23 }}，2009年2月22日查閱。。{{zh}}</ref>。

==《中葡聯合聲明》定義==
[[葡萄牙共和國|葡]][[中華人民共和國|中]]雙方在1987年1月13日，就澳門前途問題簽訂《[[中葡聯合聲明|中葡聯合聲明]]》。《中葡聯合聲明》附件一對在澳門有居留權以力有資格領取澳門永久性居民身份證的定義指在[[澳門特別行政區|澳門特別行政區]]成立前後在澳門出生或在澳門通常居住連續七年以上並以澳門為永久居住地的葡萄牙人、[[中國公民|中國公民]]、其他國籍的人士，以及其在澳門以外出生未滿十八周歲的子女<ref>移交大典統籌辦公室，[http://www.macau99.org.mo/c_doc_declaracao.html 中華人民共和國政府和葡萄牙共和國政府關於澳門問題的聯合聲明] {{webarchive|url=https://web.archive.org/web/20060904103659/http://www.macau99.org.mo/c_doc_declaracao.html |date=2006-09-04 }}，1999年12月20日更新。{{zh}}</ref>。

==《澳門基本法》定義==
{{Wikisource|中華人民共和國澳門特別行政區基本法}}
根據《[[中華人民共和國澳門特別行政區基本法|中華人民共和國澳門特別行政區基本法]]》第三章第24條，澳門特別行政區居民分為永久性居民及非永久性居民兩類，簡稱為澳門居民。

{{Anchor|永久性居民}}<!--[[澳門永久性居民|澳門永久性居民]]鏈接至此，故請勿刪除該錨鏈-->澳門基本法對'''澳門特別行政區永久性居民'''的定義指：
# 在澳門特別行政區成立以前或以後在澳門出生的中國公民及其在澳門以外所生的中國籍子女；
# 在澳門特別行政區成立以前或以後在澳門通常居住連續七年以上的中國公民及在其成為永久性居民後在澳門以外所生的中國籍子女；
# 在澳門特別行政區成立以前或以後在澳門出生並以澳門為永久居住地的葡萄牙人；
# 在澳門特別行政區成立以前或以後在澳門通常居住連續七年以上並以澳門為永久居住地的葡萄牙人；
# 在澳門特別行政區成立以前或以後在澳門通常居住連續七年以上並以澳門為永久居住地的其他人；
# 第5項所列永久性居民在澳門特別行政區成立以前或以後在澳門出生的未滿十八周歲的子女。
澳門特別行政區永久性居民在澳門特別行政區享有居留權和有資格取得載明其[[居留權|居留權]]資格的永久性居民[[澳門身份證|身份證]]。
{{Anchor|非永久性居民}}澳門基本法對'''澳門特別行政區非永久性居民'''的定義是指有資格依照澳門特別行政區法律領取澳門居民身份證但沒有居留權的人<ref>澳門基本法推廣協會，《中華人民共和國澳門特別行政區基本法》，[http://www.basiclaw.org.mo/content.php?artical_id=30 第三章居民的基本權利和義務]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}，2009年2月21日查閱。{{zh}}</ref>。

==澳門法例定義==
1999年12月20日，澳門特別行政區成立；[[澳門立法會|澳門立法會]]在當日凌晨召開會議就表決通過第6/I/1999-6號法案。法案中訂明凡在澳門特別行政區成立前後在澳門出生、在其出生時其父親或母親在澳門合法居住、已取得澳門居留權或在澳門通常居住連續七年以上的中國公民、[[土生葡人|土生葡人]]、葡萄牙人、其他國籍人士及其在澳門所生的未滿十八周歲的子女均為對'''澳門特別行政區永久性居民'''。另一方面，該法案第三條對'''澳門特別行政區非永久性居民'''的定義是指除該法案第一條所列的人士以外的依法獲准在澳門居留的人士<ref>澳門立法會，澳門特別行政區第6/I/1999-6號法案，[http://www.al.gov.mo/lei/col_lei-01/cn/07/1-1-2.htm 澳門特別行政區永久性居民及居留權法律] {{webarchive|url=https://web.archive.org/web/20051223070539/http://www.al.gov.mo/lei/col_lei-01/cn/07/1-1-2.htm |date=2005-12-23 }}，2009年2月21日查閱。{{zh}}
* 法制日報，[http://www.legaldaily.com.cn/gb/content/2002-12/01/content_47416.htm 澳門居民身份證別邁入歷史新紀元]，2009年2月21日查閱。{{zh}}</ref>。
