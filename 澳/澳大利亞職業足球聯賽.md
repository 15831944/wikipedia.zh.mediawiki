'''澳洲職業足球聯賽'''（{{lang-en|'''A-League'''}}）是[[澳大利亞|澳大利亞]]全國性的職業[[足球|足球]]聯賽，為[[澳洲足球協會|澳洲足球協會]]於2005年創建的新聯賽，取代了舊的全國足球聯賽（[[:en:National_Soccer_League_(Australia)|National Soccer League]]）。澳大利亞足協希望借職業足球聯賽使澳大利亞足球水平得到提升。

2005年8月26日聯賽正式開幕。[[現代汽車股份有限公司|現代汽車股份有限公司]]為聯賽贊助商。聯賽現時共有十支球隊，當中九隊來自澳大利亞的城市，而一隊則來自[[紐西蘭|紐西蘭]]（參見下表）。第一屆平均每場吸引 11,281 位觀眾。2006年3月4日季後賽決賽結束。[[悉尼足球俱樂部|FC悉尼]] 1–0 戰勝了[[中岸水手足球會|中岸水手]]。

== 聯賽條例 ==
常規賽為循環積分賽，一共每個隊有 27 場 （三循環比賽），排名最好 4 支球隊晉級常規賽之後的季後賽，季後賽冠軍為'''聯賽總冠軍'''（Champion），常規賽冠軍為'''優勝隊'''（Premier），總冠軍的榮譽比常規賽冠軍更高，常規賽冠軍與聯賽總冠軍都有資格參加下一年度的[[亞洲冠軍聯賽|亞洲冠軍聯賽]]。

每支球隊可以招募 20 到 23 位球員，每隊必招募兩位年輕球員（20 歲以下），此外澳大利亞足球協會禁止球隊薪金總額高於：2006年為澳幣 160 萬元；2016年為澳幣 260 萬元但是每位球員薪水也不能少於 55,000 澳幣；但可有一名傑出球員（marquee player）的薪金可以不編入薪金總額內，通常傑出球員薪水為最高，因此球隊沒有人可以像歐洲球星擁有高薪，但是相對的球隊財政也不會像在歐洲足球聯賽的球隊問題嚴重，同時封閉會員制度，不必擔心自己的實力問題。歷年情形如下表。

== 季前挑戰盃 ==
季前挑戰盃是次要赛事， 每年联赛之前给各俱樂部一个锻炼队伍的机会，目前杯赛有常规赛与季后赛。通常联赛挑战杯都使用城市以外的体育场，[[澳大利亚足球协会|澳大利亚足球协会]]希望借此吸引更多的民众关注足球赛事。

== 參賽球隊 ==
聯賽現時共有 10 支球隊，來自澳洲和紐西蘭的不同地區。當中 3 隊[[阿德萊德聯足球俱樂部|阿德萊德聯]]、[[紐卡斯爾噴氣機足球俱樂部|紐卡素噴射機]]和[[珀斯光榮足球俱樂部|珀斯光輝]]比澳職聯還要早成立。

有別於其他國家的足球聯賽，澳職聯並沒有升降班制度，其運作模式與澳洲和[[北美|北美]]其他職業體育聯賽（如[[美國職業足球大聯盟|美國職業足球大聯盟]]）比較相似。

代表[[奧克蘭|奧克蘭]]的[[新西蘭騎士隊|新西蘭騎士隊]]於2007年解散，而同樣來自紐西蘭的[[惠靈頓鳳凰足球俱樂部|惠靈頓鳳凰隊]]則於同年加盟澳職聯。至於[[北風暴足球會|北昆士蘭風暴]]因財困問題，被主辦當局於2011年3月1日勒令退出。

== 過往發展 ==
2009–10年的球季內，澳超聯有 2 支新球隊加入，它們分別為[[黃金海岸聯足球會|黃金海岸聯]]及[[北風暴足球會|北昆士蘭風暴]]，令參賽隊伍數目會由當時的8隊增加至10隊。<ref>{{cite web| title = 黃金海岸聯北昆士蘭加入2009至2010年澳超聯賽| publisher = [[AFC|AFC]] | date = 2008-08-29| url = http://www.the-afc.com/chi/articles/viewArticle.jsp_167969165.html| accessdate = 2008-08-29}}</ref>

== 未來展望 ==
除了這些還有擴增隊伍的必要，因為[[亞洲冠軍聯賽|亞洲冠軍聯賽]]每個國家最多的參賽席位數，為該個頂級聯賽數目中的三分一，而因為澳甲聯中只有九隊是真正位於澳洲境內，所以澳洲聯賽排名不管在第幾位，最多只可派出三隊參賽，如[[2017年亞足聯冠軍聯賽|2017年亞冠聯賽]]無法以2+2的方式派隊參賽。也因此澳甲聯打算在澳洲各地的地方聯賽招攬優秀球隊達到「滿額參賽」的理想。<ref name="hyundai-extends">{{cite press release | url=http://www.hyundai.com.au/about-hyundai/news/articles/hyundai-extends-a-league-partnership-until-2016 | title=Hyundai Extends A-League Partnership Until 2016 | publisher=Hyundai Motor Company, Australia | accessdate=10 May 2014 | deadurl=yes | archiveurl=https://web.archive.org/web/20140512222908/http://www.hyundai.com.au/about-hyundai/news/articles/hyundai-extends-a-league-partnership-until-2016 | archivedate=2014年5月12日 }}</ref>

== 電視直播 ==
2006年，[[FOX體育台|FOX Sports]]电视台花費 1.2 億[[澳元|澳元]]購買了[[澳大利亞國家足球隊|澳大利亞國家足球隊]]與甲級足球聯賽的轉播權，因此澳大利亞足球協會將獲得建設澳洲足球和幫助各等級聯賽球隊的援助資金。

2008年8月，[[香港|香港]]的[[香港有線電視|有線電視]]購得澳超聯在[[港澳地區|港澳地區]]的獨家播映權，並安排在[[香港有線體育有限公司|有線球彩台]]於每週提供兩至五場的足球[[直播|直播]]。2011年8月，新增的hd203體育頻道也會同步以[[高清|高清]]技術直播賽事。自2013年起，[[有線寬頻|有線寬頻]]官方網站更提供相關賽事的[[免費|免費]]網上直播。<ref>{{cite web |title = 有線寬頻 i-CABLE - 足球賽事直播 |date = 2013-01-04 |url = http://www.i-cable.com/livesoccer/live.php |accessdate = 2012-12-21}}</ref> 從2014年球季開始更提供直播相關聯賽足總盃的賽事。
