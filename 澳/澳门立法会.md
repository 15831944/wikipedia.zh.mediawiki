'''澳門特别行政區立法會'''（[[葡萄牙語|葡萄牙語]]：Assembleia Legislativa），是[[澳門特別行政區|澳門特別行政區]]的[[立法機構|立法機構]]。立法會每4年換屆，[[議員|議員]]經[[直接選舉|直接選舉]]、[[間接選舉議席|間接選舉]]和[[澳門特別行政區行政長官|行政長官]]委任進入議會。現時立法會為第六屆，由14名直選議員、12名間選議員及7名[[澳門官委議員|官委議員]]共33名議員組成。

[[澳門政府|澳門政府]]奉行「[[行政主導|行政主導]]」體制，在實行以行政為主導和核心的前提下行政與立法互相制衡及配合。[[澳門行政長官|行政長官]]在澳門具有超乎行政、立法和司法三個機關之上的地位，是特區的權力和領導核心，在政治體制中處於中心地位，故澳門的立法權與行政權之間並非對等<ref name ="lan2012">{{cite journal |author1=冷鐵勛 |title=論繼續保持澳門立法會現行組成結構的必要性 |journal=《“一國兩制”研究》 |date=2012 |issue=11 |url=http://www.ipm.edu.mo/cntfiles/upload/docs/common/1country_2systems/2012_1/p96.pdf}}</ref>。

澳門立法會的直選議席不過半數，其組成傾向親政府，[[民主派_(澳門)|民主派]]議員每當就政府施政出現問題而動議展開聽證程序時都會被[[建制派_(澳門)|建制]]及官委議員否決。與奉行「三權分立」， 官員受到[[香港立法會|議會]]監督的[[香港|香港]]相比（官員由中央人民政府委任，不用向立法會負責，只是列席立法會會議），奉行「行政主導」體制的澳門，當地官員並不需要向立法會負責，造成「有問無責」的情況，故澳門立法會素有「垃圾會」或「橡皮圖章」之稱<ref>{{cite book |author1=余振 |title=《澳門回歸前後的問題與對策》 |date=1999 |publisher=名流政策 (澳門) 硏究所 |isbn=9789629280864 |pages=76}}</ref><ref>{{cite news |title=官員問責失效　蔡梓瑜：政府思維不改 難改問責制 |url=https://aamacau.com/2018/02/13/%E5%AE%98%E5%93%A1%E5%95%8F%E8%B2%AC%E5%A4%B1%E6%95%88%E3%80%80%E8%94%A1%E6%A2%93%E7%91%9C%EF%BC%9A%E6%94%BF%E5%BA%9C%E6%80%9D%E7%B6%AD%E4%B8%8D%E6%94%B9-%E9%9B%A3%E6%94%B9%E5%95%8F%E8%B2%AC%E5%88%B6/ |agency=論盡媒體 |date=2018-02-13}}</ref><ref>{{cite news |title=立法會選舉後的一些迷思 |url=http://zamag.net/%E7%AB%8B%E6%B3%95%E6%9C%83%E9%81%B8%E8%88%89%E5%BE%8C%E7%9A%84%E4%B8%80%E4%BA%9B%E8%BF%B7%E6%80%9D/ |agency=ZA誌 |date=2017-09-30}}</ref><ref>{{cite news |title=立法會只有問　沒有責　 余永逸：最重要是組成問題 |url=https://aamacau.com/2017/09/02/%E7%AB%8B%E6%B3%95%E6%9C%83%E5%8F%AA%E6%9C%89%E5%95%8F%E3%80%80%E6%B2%92%E6%9C%89%E8%B2%AC%E3%80%80-%E4%BD%99%E6%B0%B8%E9%80%B8%EF%BC%9A%E6%9C%80%E9%87%8D%E8%A6%81%E6%98%AF%E7%B5%84%E6%88%90%E5%95%8F/ |agency=論盡媒體 |date=2017-09-02}}</ref>。而支持行政主導體制的聲音則認為，若立法權與行政權相比下處於一種更加主動和優勢地位並不符合《[[澳門基本法|澳門基本法]]》，認為「立法主導」是不符合實際環境且沒有用處的純議會制度<ref name ="lan2012" />。

== 現屆成員 ==
=== 第六屆立法會 ===
由2017年至2021年，由33名議員組成，其中
* 由直接選舉產生14名，
* 由間接選舉產生12名，
* 由行政長官委任7名。

目前立法會有29名[[建制派_(澳門)|建制派]]議員，4名[[民主派_(澳門)|民主派]]議員。14名議員是直選產生，佔總比例42%（七分之三）。

== 回歸後成員組成變遷 ==
=== 第1屆立法會 ===	
由1999年至2001年，由23名議員組成，其中：	
* 由直接選舉產生8名，
* 由間接選舉產生8名，
* 由行政長官委任7名。	
所有直選及間選議員（除[[容永恩|容永恩]]外）皆由[[澳門立法會_(澳葡時期)|澳葡時期的立法會]]過渡而來。

=== 第2屆立法會 ===	
由2001年至2005年，由27名議員組成，其中	
* 由直接選舉產生10名，
* 由間接選舉產生10名，
* 由行政長官委任7名。

=== 第3屆立法會 ===
由2005年至2009年，由29名議員組成，其中
* 由直接選舉產生12名，
* 由間接選舉產生10名，
* 由行政長官委任7名。	
			
根據《[[澳門基本法|澳門基本法]]》附件二規定，2009年及以後立法會的產生方式如需修改，須有3分之2議員贊成，行政長官同意，並報[[全國人民代表大會常務委員會|全國人民代表大會常務委員會]]備案。

=== 第4屆立法會 ===
由2009年至2013年，由29名議員組成，其中
* 由直接選舉產生12名，
* 由間接選舉產生10名，
* 由行政長官委任7名。
=== 第5屆立法會 ===
由2013年至2017年，由33名議員組成，其中
* 由直接選舉產生14名，
* 由間接選舉產生12名，
* 由行政長官委任7名。

== 運作 ==	
設主席（Presidente）、副主席（Vice-Presidente）各1名，秘書兩名，由全體議員互選產生。四人組成執行委員會，與行政委員會為行政機構。第一至第三屆立法會主席為[[曹其真|曹其真]] ，第四屆是[[劉焯華|劉焯華]]，現任立法會主席為[[賀一誠|賀一誠]]。	
			
另有章程及任期委員會、三個常設委員會（第一、第二、第三）、土地及公共批給事務跟進委員會、公共財政事務跟進委員會及公共行政事務跟進委員會。根據《議事規則》第九十三條，委員會會議閉門進行，但有相反議決者除外。

根據《基本法》第71條，立法會職權如下：	 
# 依照基本法規定和法定程序制定、修改、暫停實施和廢除[[澳門法律|法律]]；	 
# 審核、通過政府提出的[[財政預算案|財政預算案]]；審議政府提出的預算執行情況報告；	 
# 根據政府提案決定[[稅收|稅收]]，批准由政府承擔的債務；	 
# 聽取[[澳門行政長官|行政長官]]的施政報告並進行辯論；	 
# 就公共利益問題進行辯論；	 
# 接受澳門居民申訴並作出處理；	 
# 如立法會全體議員3分之1聯合動議，指控行政長官有嚴重違法或瀆職行為而不辭職，經立法會通過決議，可委托[[澳門終審法院|終審法院]]院長負責組成獨立的調查委員會進行調查。調查委員會如認為有足夠證據構成上述指控，立法會以全體議員三分之二多數通過，可提出[[彈劾|彈劾]]案，報請中央人民政府決定；	 
# 在行使上述各項職權時，如有需要，可傳召和要求有關人士作證和提供證據。	 
		 
而根據第75條，議員依照本法規定和法定程序提出議案。凡不涉及公共收支、政治體制或政府運作的議案，可由立法會議員個別或聯名提出。凡涉及政府政策的議案，在提出前必須得到行政長官的書面同意。而且有關財政監察的法律尚未制訂，相關權力只是一紙空文。

另外，根據第76條，議員亦有權依照法定程序對政府的工作提出質詢，惟在2004年經修訂的《議事規則》中在放寬口頭質詢的同時，限制了書面質詢的數目。這兩條，外加上支援工作不足，都是立法會工作的重大障礙。
