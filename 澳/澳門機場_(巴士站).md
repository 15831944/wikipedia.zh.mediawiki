'''澳門機場'''（[[葡萄牙文|葡萄牙文]]：'''AEROPORTO DE MACAU'''、{{lang-en|'''Macau Airport '''}}）位於[[澳門國際機場|澳門國際機場]]入境層的一個公車站。[[澳巴|澳巴]]'''[[澳門巴士36路線|36]]'''、'''[[澳門巴士MT1路線|MT1]]'''、'''[[澳門巴士N2路線|N2]]'''路線和[[澳門新福利公共汽車有限公司|新福利]]'''[[澳門巴士26路線|26]]'''、'''[[澳門巴士51A路線|51A]]'''、'''[[澳門巴士AP1路線|AP1]]'''、'''[[澳門巴士AP1X路線|AP1X]]'''、'''[[澳門巴士MT4路線|MT4]]'''路線途經該站。

== 沿革 ==
*因應[[澳門國際機場|澳門國際機場]]啟用而同時啟用。
*自[[氹仔臨時客運碼頭|氹仔臨時客運碼頭]]開通後，原以本站為終點站的'''[[澳門巴士AP1線|AP1]]'''路線改為循環線，自此沒有路線再以此站為終點站。

== 歷史 ==
*1995年11月，配合[[澳門國際機場|澳門國際機場]]啟用，本站同時啟用。

== 設施 ==
*候車排隊欄杆。

== 各車道安排 ==
*T356/1(路環方向)：[[澳門巴士26路線|26]]、[[澳門巴士51A路線|51A]]路線。
*T356/2(關閘方向)：[[澳門巴士AP1路線|AP1]]、[[澳門巴士AP1X路線|AP1X]]路線。
*T356/3(氹仔、澳門方向)：[[澳門巴士36路線|36]]、[[澳門巴士MT1路線|MT1]]、[[澳門巴士MT4路線|MT4]]、[[澳門巴士N2路線|N2]]路線。

*沒有路線以此站為終點站。
