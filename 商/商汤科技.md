'''商湯科技'''是由[[香港中文大學工程學院|香港中文大學工程學院]]團隊所創立，以湯曉鷗教授為領軍人物的人工智能初創企業<ref name=mp /><ref name=reuters>{{cite web |last=Jiang |first=Sijia  |url=https://www.reuters.com/article/us-sensetime-ipo-exclusive/exclusive-chinas-sensetime-plans-ipo-u-s-rd-center-as-early-as-2018-idUSKBN1DN0FY |title=Exclusive: China's SenseTime plans IPO, U.S. R&D center as early as 2018 |work=路透社|date=2017-11-23 |accessdate=2018-03-02}}</ref>。其於[[圖像處理|圖像處理]]、[[人臉辨識|人臉辨識]]、[[自動駕駛|自動駕駛]]、[[擴增實境|擴增實境]]、[[深度神經網絡|深度神經網絡]]等方面位處世界前沿位置，在算法、數據和運算加速方面具備深厚的技術積累<ref name=mp>{{cite web |last=祉愉 |first= 鄭 |url=https://bkb.mpweekly.com/cu0001/20170930-53823 |title=中國刷臉「天眼」人臉辨認技術來自香港 商湯科技CEO徐立：一秒監視幾萬人 |work=明報周刊 |date=2017-09-30 |accessdate=2018-03-02}}</ref><ref name=hkej>{{cite web |url=http://startupbeat.hkej.com/?p=51608 |title=團隊資歷深 120博士坐鎮 商湯AI誓挑戰美巨擘 |work=信報 StartupBeat |date=2017-10-24 |accessdate=2018-06-19}}</ref><ref name=hkej2>{{cite web |url=http://startupbeat.hkej.com/?p=53765 |title=內地7初創上榜 港產商湯入AI百強 |work=信報 StartupBeat |date=2017-12-19 |accessdate=2018-03-02}}</ref>。其戰略合作夥伴包括[[高通|高通]]、[[華為|華為]]、[[小米科技|小米科技]]、[[本田汽車|本田汽車]]和[[麻省理工學院|麻省理工學院]]等。商湯科技被指正在籌劃[[首次公開招股|首次公開招股]]<ref name=reuters />。

== 技術 ==
===人臉識別===
商湯科技具備辨識閉路電視中曝光情況不佳及遠距離人臉的能力。2014年香港發生[[讓愛與和平佔領中環|佔領中環]]事件，[[香港警方|香港警方]]曾借用商湯科技的技術作人流管制，該技術在中國大陸已成功協助抓捕數百名犯人<ref name=mp /><ref name=bnext>{{cite web | last=書婷 | first=翁 |url=https://www.bnext.com.tw/article/45350 |title=破120億台幣！商湯科技創全球AI單輪融資最高紀錄 |work=數位時代 |date=2017-07-13 |accessdate=2018-03-02}}</ref>。

===超分辨率===
該公司的技術之一是透過神經網絡來提升圖像和影片的分辨率，例如從4K提升到8K<ref name=bnext />。

===圖像理解===
商湯科技具稱已擁有商業化技術，能自動把球賽視頻等轉換成文字旁述，以便檢索<ref name=bnext />。2015年商湯科技聯同中文大學參加{{le|ImageNet|ImageNet}}視頻辨識比賽（ILSVRC2015），在動態視頻物體檢測中獲得全球冠軍的殊榮<ref name=mp />。

===自動駕駛===
商湯科技正開發自動駕駛技術，能單憑攝像頭影像作出判斷，從而免去安裝[[光學雷達|光學雷達]]等昂貴傳感器的成本<ref name='hkej2' />。

== 融資 ==
2017年7月，商湯科技宣布完成4.1億美元B輪融資，創下當時全球人工智慧領域單輪融資最高紀錄。

2018年4月9日，商湯科技宣布完成6億美元C輪戰略融資，再創全球人工智慧領域融資記錄。本輪融資由[[阿里巴巴集團|阿里巴巴集團]]領投，新加坡主權基金[[淡馬錫|淡馬錫]]、[[苏宁易购|蘇寧]]等投資機構和戰略夥伴跟投。

== 爭議 ==
由於與中國政府主導的[[大规模监控|大規模監控計劃]]（中國政府官方稱為「[[社會信用系統|社會信用系統]]」）持續的合作，商汤科技也被指「充當政府耳目」，不過參與到中國類似的大規模監控計劃的科技企業也遠不止商湯科技，還有[[浙江大華技術股份有限公司|大華技術]]、[[海康威視|海康威視]]、[[寒武紀科技|寒武紀科技]]等等開發製造監控裝置或主營人工智慧處理技術的公司。<ref>{{cite web |title=人臉識別 + 社會信用系統，一場重塑人類行為的社會實驗？ |url=https://theinitium.com/article/20180525-wsj-introduction/ |website=theinitium.com |language=zh-Hant}}</ref>
