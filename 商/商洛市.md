'''商洛市'''是[[中华人民共和国|中华人民共和国]][[陕西省|陕西省]]下辖的[[地级市|地级市]]，因境内商山、[[洛水|洛水]]而得名。地处陕西省东南部。

== 历史 ==
据《[[史记|史记]]》所载，司徒[[契|契]]辅佐[[禹|禹]]治水有功，受封于商，是为商国。[[秦始皇|秦始皇]]二十六年（前221年）设商县，因商山而得名。今丹凤县古城岭出土的文物带有“商”字的瓦当，证明此地为秦国[[商鞅|商鞅]]之封邑。[[西汉|西汉]]属[[司隸_(地名)|司隶]][[弘农郡|弘农郡]]，范围大约是上洛、商县两县。上洛县因居洛河上游而得名。因“水”不利[[汉朝|汉朝]]，故将上洛改为上雒。[[东汉|东汉]]属[[京兆尹|京兆尹]]，[[三国|三国]]属[[曹魏|曹魏]][[京兆郡|京兆郡]]，[[魏文帝|魏文帝]]又将上洛的“雒”改为“洛”。[[西晋|西晋]]属[[司州|司州]]，[[泰始|泰始]]二年（266年），[[晋武帝|晋武帝]]分京兆郡南部置上洛郡。

[[南北朝|南北朝]][[北魏|北魏]][[太和|太和]]十三年（489年），改置洛州于上洛城。[[北周|北周]][[宣政|宣政]]元年（578年）改洛州为商州。[[隋|隋]][[开皇|开皇]]三年（583年）撤上洛、上庸两郡，保留商州；[[大业|大业]]三年（607年）撤商州复设上洛郡，治所上洛，领上洛、商洛（[[隋文帝|隋文帝]]改商县为商洛）、洛南、丰阳、上津五县。[[唐|唐]][[武德|武德]]元年（618年）改上洛郡为商州，[[天宝|天宝]]元年（742年）撤商州改上洛郡，[[乾元|乾元]]元年（758年）再撤上洛郡改为商州。

[[北宋|北宋]]时，商州初属陕西路，后属永兴军路。[[金|金]][[贞元|贞元]]二年（1154年）将商洛、丰阳两县降为镇，并入上洛、洛南两县；[[至元|至元]]元年（1264年）废上洛县，以州代县，为商州。[[元代|元代]]商州先属安西路，后属奉元路。[[明代|明代]]属西安府，称商州，[[洪武|洪武]]七年（1374年），降商州为县。[[成化|成化]]十三年（1477年）复升为州。[[天启|天启]]元年（1621年）为避[[明光宗|明光宗]][[朱常洛|朱常洛]]名讳，又改“洛”为“雒”。[[清|清]][[雍正|雍正]]三年（1725年）升为直隶商州，属陕西省。[[乾隆|乾隆]]四十七年（1782年），析咸宁、蓝田、镇安3县置孝义厅，属西安府，为柞水县前身。 

[[民国|民国]]二年（1913年）废商州，改设商县、洛南、柞水3县，属关中道；山阳、商南、镇安3县属汉中道。1949年7月13日，商县人民政府成立，在商县设置陕甘宁边区陕南行政主任分署，辖商县、雒南、商南、丹凤、山阳、镇安、柞水7县。1950年，改名陕西省[[商洛专区|商洛分区专员公署]]，仍辖以上7县。1958年12月，撤销丹凤（分属商南、商县）、柞水两县（与镇安县合并）。1961年10月恢复为7县。1964年9月，因“雒”字生辟，改“商'''雒'''”为“商'''洛'''”。1988年6月，撤销[[商县|商县]]，改设县级商州市。2001年12月，撤销[[商洛地区|商洛地区]]，设立地级商洛市<ref>[http://www.gov.cn/gongbao/content/2001/content_61028.htm 国务院关于同意陕西省撤销商洛地区设立地级商洛市的批复]</ref>。

== 经济 ==
相对贫困地区，农业，林果，中药材，金矿為主業，但2016年下半陝西省官方宣布在商洛市勘查發現特大[[玉石|玉石]]礦<ref>[http://www.chinatimes.com/realtimenews/20161021002238-260409 商洛市勘查發現特大玉石礦]</ref>，資源量達數百萬噸，潛在經濟價值人民幣逾千億元，此玉石礦由優質石英質玉石礦和白雲質玉石礦組成，紫綠搭配品質稀有，陜西省據此將全省查明礦種由91種增加到92種。

== 方言与民系 ==
{{Unreferenced section|time=2016-12-05T00:12:37+00:00}}
目前，在商洛市使用的方言主要包括兩類，分别是'''主户話'''与'''客户話'''。主户話即本地話，使用者主要是當地土著居民和於較早前的[[明代|明代]]移民至此的“大槐樹人”。客户話是[[下湖話|下湖話]]与[[廣東話_(陕西)|广东话]]的合稱，由[[清朝|清朝]]中前期[[湖廣填四川|湖廣填四川]]的移民帶來。

下湖话的使用者称為下湖人，來自[[湖南省|湖南省]]、[[江西省|江西省]]北部、[[安徽省|安徽省]]南部等地；其中商南有不少人从皖南移民而來，他们说的方言被稱为蛮子話。陝西師範大學教授孟萬軍考証商洛蛮子話的語音特點，认为商洛蛮子話的祖語是[[贛語|贛語]]懐岳片。

“廣東人”來自福建省[[汀州府|汀州府]]、江西省[[贛州府|贛州府]]、廣東省[[惠州府|惠州府]]、[[韶州府|韶州府]]、[[嘉應州|嘉應州]]等地。雖來源地並非祇有廣東省，本地人則統稱他們及其後裔為廣東人，稱他們説的方言為“廣東話”，迄今在商洛市没有發現福建汀州府移民自稱[[福建話|福建話]]。他們居住的所在常常以“廣東”命名，以示不忘祖先。對照現代客语的語音特點，可以看出陝西商洛境内的“廣東話”應該是融合了本地話的[[客家話|客家話]]方言。{{fact|在陝西省，人們常直接稱呼這種方言為“廣東话”；而流行於香港、澳門、廣州等地的[[粤語|粤語]]方言，現當代已為此地人所知悉，目前此地多稱其為“[[廣州话|廣州话]]”（正式）、“港澳話”（非正式）。}}
