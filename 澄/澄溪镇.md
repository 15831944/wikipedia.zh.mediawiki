'''澄溪镇'''位于[[重庆市|重庆市]][[垫江县|垫江县]]南部，距垫江县县城17公里，面积59平方公里，主镇区与重庆市[[长寿区|长寿区]][[海棠镇|海棠镇]]主镇区相连<ref>高德地图两镇的主镇区情况</ref>。澄溪镇是重庆市经济百强镇之一<ref>重庆市经济百强镇名单</ref>，也是重庆市市级中心镇<!-- 该体系未必与区县人民政府划设的中心镇重叠，例如垫江县新民镇 --><ref>{{cite web |url= http://czjs.ccc.gov.cn/zjs/sjzxz/ |title= 重庆市市级中心镇名单 |publisher= 重庆市村镇建设信息网 |language= zh |accessdate= 2016-05-26  }} </ref>，同时还是全国重点镇。<ref>{{cite web |url=http://ln.qq.com/a/20141120/056498_all.htm |title=2014年全国重点镇公示名单 |date=2014-11-20 |accessdate=2016-05-26 |publisher=腾讯网  }}</ref>

== 经济情况 ==
2014年，澄溪镇实现[[公共财政|公共财政]]收入7300万元，全年财政总收入为13724万元，年度[[财政结转资金|财政结转资金]]6500万元；[[招商引资|招商引资]]到位资金13.5亿元,全社会[[固定资产投资|固定资产投资]]20.13亿元；[[农民人均纯收入|农民人均纯收入]]11827元。<ref>{{cite web |url=http://cxz.cqsdj.gov.cn/html/56/zjcx/ |title=澄溪镇简介 |accessdate=2016-05-26 |publisher=澄溪镇人民政府  }}</ref>
