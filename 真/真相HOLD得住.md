《'''真相HOLD得住'''》是[[台灣|台灣]]的談話性[[綜藝節目|綜藝節目]]，由[[衛視中文台|衛視中文台]]開播，目前[[星期一|星期一]]到[[星期五|星期五]]於晚上8時至9時首播。節目有幾名時常來的甚至每集都來的班底：[[知本家文化|知本家文化]][[社長|社長]][[劉燦榮|劉燦榮]]、曾任[[自由時報|自由時報]]社會[[記者|記者]]的[[王瑞德|王瑞德]]、資深[[媒體|媒體]]人[[黃敬平|黃敬平]]，本節目介紹[[外星|外星]]怪談、[[歷史|歷史]]事件、[[生活|生活]]常識等。片尾標示出「以上言論不代表本台立場」

== 主持人 ==
* [[林青蓉|林青蓉]]（2012年2月6日－2013年2月19日）
* [[高伊玲|高伊玲]]（2013年2月21日－2013年9月12日）

== 標題口號 ==
* 第一期：宇宙有多大 真相HOLD得住（2012年2月6日－2013年5月17日）
* 第二期：好奇勘宇宙 真相HOLD得住（2013年5月20日－2013年9月12日）

== 播放狀況 ==
* 首播：每周一至周四 20:00－21:00

== 概要 ==
《真相HOLD得住》，不僅和國家地理頻道合作，內容更是包羅萬象，舉凡歷史、社會、科普、時事、生活、天文...等等主題，邀請相關專家一同為觀眾解密探索真相。是以輕鬆、活潑、知性的角度為大家解密，一個大人、小孩都可以一起收看的好節目。
