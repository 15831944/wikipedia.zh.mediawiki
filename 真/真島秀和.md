'''眞島秀和'''（{{Jpn|j=眞島 秀和|kn=ましま ひでかず|rm=Mashima Hidekazu}}，{{bd|1976年|11月13日|}}<ref>{{cite web |title=真島秀和のプロフィール |url=https://www.oricon.co.jp/prof/330556/profile/ |publisher=オリコン芸能人事典}}</ref>），[[日本|日本]][[男演員|男演員]]，出生於[[山形縣|山形縣]][[米澤市|米澤市]]<ref>{{cite web |title=おしょうしな観光大使　～全国へ米沢をPR～ |url=http://www.city.yonezawa.yamagata.jp/3279.html |publisher=米沢市Official Web Site}}</ref>，隸屬於ザズウ。

== 早期生活 ==
獨生子<ref name="fabulous10">{{cite journal |title=眞島秀和 Works Interview |journal=fabulous act（Vol.10） |page=P. 110 - P. 123}}</ref>，從米澤市立第一中學校<ref>{{cite web |author1=岸たけし　公式ブログ |title=眞島秀和（俳優）/米沢興譲館卒 |url=https://ameblo.jp/take4648kish/entry-12389874131.html}}</ref>畢業後，進入山形縣立米澤興讓館高等學校就讀<ref>{{cite web |title=ぐるっと東北・母校をたずねる：県立米沢興譲館高校／４　部活仲間と励まし合い　俳優・眞島秀和さん　／山形 - 毎日新聞 |url=https://mainichi.jp/articles/20170623/ddl/k06/100/117000c |website=毎日新聞 |language=ja}}</ref>，中學時代將時間都花在打[[籃球|籃球]]上<ref>{{cite journal |title=Interview 001: Hidekazu Mashima |journal=ZERO☆23（Vol.220 2018年8月號）}}</ref>。

高中畢業後進入[[東京|東京]]的[[國士館大學|國士館大學]]法學部法律學科。由於大學的朋友很多在跳舞與玩音樂，想加入那樣的世界，加上對電影也越來越有興趣<ref name="jmovie23">{{cite journal |title=眞島秀和『心に吹く風』 |journal=J Movie Magazine Vol. 23 |page=P.96 - P.97}}</ref>，喜歡上[[北野武|北野武]]的電影，以及受到當時名片《[[猜火車|猜火車]]》、《[[刺激1995|刺激1995]]》帶來的震撼，想成為能演出這類作品的人<ref name="fabulous10" />，因此開始以[[電影演員|電影演員]]為目標，21歲左右參加演員培訓所<ref name="jmovie23" />，並與朋友自行拍攝獨立電影<ref>{{cite web |title=俳優・眞島秀和、夢のきっかけは友人のダンス? 『冬ソナ』監督とタッグで撮影方法に驚き |url=https://news.mynavi.jp/article/20170613-mashimahidekazu/ |publisher=マイナビ ニュース}}</ref>。

== 演藝生涯 ==
1999年被當時還就讀日本映畫大學的[[在日朝鮮人和韓國人|在日朝鮮]]導演[[李相日|李相日]]發掘，主演描述[[在日朝鮮人和韓國人|在日韓人]]青春期生活的畢業作品《青〜chong〜》<ref>{{cite web |title=眞島秀和が主婦を虜に！塩顔イケメン俳優がブレイク間近 |url=https://ciatr.jp/topics/119947 |publisher=ciatr}}</ref>，後來也因拍攝電影而長期缺課，最後不顧父母反對輟學。大學輟學後，母親因此病倒在床上躺了三個月，讓他至今仍有罪惡感<ref>{{cite web |title=「どうしても止められない」星野源と同居人たちが告白！謝りたい『過去の罪』とは？ |url=https://grapee.jp/369056 |website=grapee}}</ref>。

24歲時在導演篠原哲雄介紹下，進到目前的經紀公司「ザズウ」<ref name="バイプレーヤー読本">{{cite book |title=バイプレーヤー読本 |publisher=洋泉社 |page=P.68 - P.81}}</ref>。2005年在獨立電影《心中エレジー》再次擔任主演，2007年則在[[NHK|NHK]]講述日本女子與韓國男子愛情故事的三集連續劇《海峽》擔任準主演。

2012年6月23日起，受邀擔任山形「美姬米」（つや姫）大使<ref>{{cite web |title=俳優・眞島秀和さんが、やまがた特命観光・つや姫大使に就任 |url=https://www.yamagata.nmai.org/perorin_ga_iku/120623/index.html}}</ref>；隔年5月3日起，則擔任山形縣米澤市「おしょうしな観光大使」<ref>{{cite web |author1=米沢市役所 |title=「眞島秀和さん」おしょうしな観光大使就任のお知らせ |url=http://www.city.yonezawa.yamagata.jp/3874.htm}}</ref>。

2018年，連續兩季在《總覺得鄰家更幸福》與《[[大叔之愛|大叔之愛]]》中演出廣受好評的[[同志|同志]]角色後，知名度大為上升，更在《週刊TVガイド》由讀者以明信片回函投票的STAR RANKING（10/27 - 11/2）中，榮登第二名寶座<ref>{{cite web |title=TVガイド スターランキング 2018年10月27日(土)～11月2日(金) |url=https://www.tvguide.or.jp/ranking/starranking/20181130/}}</ref>。

== 個人生活 ==
在朋友間的綽號為'''まっしー'''，母親與阿嬤則稱他為'''秀ちゃん'''<ref name="鄰家4">{{cite conference |title=《總覺得鄰家更幸福》第四話副聲道}}</ref>。自認是愛乾淨、一板一眼的人，但不會干涉他人<ref>{{cite journal |title=談話室くらっしい：眞島秀和 |journal=Classy（2018年9月號） |page=P. 212}}</ref>。演員[[吉田鋼太郎|吉田鋼太郎]]對他的評價則是「那傢伙明明平常很紳士，一喝酒似乎就變得很強勢的樣子。」<ref>{{cite web |title=「おっさんずラブ」最終回直前！田中圭＆林遣都＆眞島秀和＆吉田鋼太郎が明かす、今だから話せる裏話 |url=https://thetv.jp/news/detail/149110/ |website=ザテレビジョン}}</ref>
===寵物===
由於沒有手足，小時候的玩伴是家裡養的狗，而且也非常喜歡動物<ref name="fabulous10" />。

從不明年份到2012年，曾養過一隻年老的公貓。自述當時一整天若沒事的話，腦中都會想著貓的事情，也會讀寵物雜誌了解貓，喜歡貓到希望變成貓的程度<ref>{{cite journal |title=話題沸騰すごイケメンに直撃～眞島秀和 |journal=週刊女性（2012.6.19） |page=P.185}}</ref>。貓咪過世後，由於覺得很寂寞，因此到寵物店買了一隻[[吉娃娃|吉娃娃]]來養，取名叫「小花」（ハナちゃん）<ref>{{cite web |title=GUILD TV （第１回ゲスト　眞島秀和さん＆山中崇さん） |url=https://www.youtube.com/watch?v=UHB2JcTRQDY |date=2013-06-07}}</ref>

如果要變成動物的話，想變成「受主人溺愛的貓與狗」。由於常被人評論長得像[[藏狐|藏狐]]，所以很想跟藏狐見面<ref>{{cite journal |title=もっと知りたい！眞島秀和のコト |journal=週刊ザテレビジョン（2018年11月3日號）}}</ref>。
===嗜好===
喜歡[[麻將|麻將]]<ref name="ドラマ「アカギ」">{{cite news |title=ドラマ「アカギ」南郷役は駿河太郎、竜崎役は六角慎司、矢木役は眞島秀和に決定 |url=https://natalie.mu/comic/news/248444 |publisher=ナタリー}}</ref>，曾在2002年的訪問中，提到平常靠打[[麻將|麻將]]、騎[[機車|機車]]紓壓<ref>{{cite journal |title=若手俳優奮闘記：眞島秀和 |journal=男優倶楽部 |volume=Vol.7 |page=P.98-99}}</ref>，但2018時的訪問則說除了開車，幾乎沒有其他興趣<ref>{{cite web |title=眞島秀和「田中圭くんは距離感の詰め方がすごく素敵」【シブメンFILE#01 眞島秀和】 （2/2） {{!}} テレビ・芸能ニュースならザテレビジョン |url=https://thetv.jp/news/detail/149133/p2/ |website=ザテレビジョン |language=ja}}</ref>。

喜歡的連續劇是《[[來自北國|來自北國]]》<ref name="バイプレーヤー読本" />。聽了[[賽門與葛芬柯|賽門與葛芬柯]]、[[披頭四樂團|披頭四樂團]]、[[老鷹合唱團|老鷹合唱團]]、[[芝加哥樂隊|芝加哥合唱團]]而喜歡上音樂，現在各種類型都聽。從中學聽到現在的療癒愛歌是老鷹合唱團的〈[[加州旅館|加州旅館]]〉<ref name="fabulous10" />。從以前到現在一直有在聽的專輯則是[[東京斯卡樂園管絃樂團|東京斯卡樂園管絃樂團]]的《The Last》，也常聽澳洲樂團Blue King Brwon的音樂<ref name="fabulous04">{{cite journal |title=眞島秀和✖平岡祐太 Works Interview |journal=fabulous act（Vol. 4） |page=P. 130 - P. 149}}</ref>。最近（2018年11月）在意的音樂人是，常常一起喝酒，唱過大叔之愛主題曲的樂團[[無限開關|無限開關]]的主唱大橋卓彌<ref name="fabulous10" />。

有固定看漫畫的習慣<ref name="fabulous04" />，自述是[[鬥牌傳說|鬥牌傳說]]的大粉絲，意外被選上演出電視版第二季的角色「矢木圭次」<ref name="ドラマ「アカギ」" />。
===人際===
與同經紀公司，差一學年的[[山中崇|山中崇]]是好友<ref name="バイプレーヤー読本" />。演出[[大叔之愛|大叔之愛]]後，與男演員[[田中圭|田中圭]]成為好友。由於住的地方很近，兩人一周見面三次左右，會一起吃飯、喝酒、慢跑<ref>{{cite journal |title=優しさと、繊細さと、愛と。眞島秀和 |journal=TVガイドPerson Vol. 75 |page=P.56 - P.59}}</ref>、玩過[[鬼抓人|鬼抓人]]<ref>{{cite web |title=今いちばん知りたい男、田中圭の遊びのある日常 |url=https://www.elle.com/jp/culture/movie-tv/a23776053/kei-tanaka-18-1026/ |publisher=ELLE |accessdate=2018.10.29}}</ref>。藉由田中圭創立的大叔之愛[[LINE|LINE]]群組，以及舞台劇、新劇等機會，頻繁與其他同戲演員，如[[林遣都|林遣都]]、[[金子大地|金子大地]]一起吃喝、玩電動<ref>{{cite web |title=「おっさんずラブ」は終わらない！ |url=https://natalie.mu/eiga/pp/2018hits}}</ref>。
===其他===
完全不會游泳<ref name="鄰家4" />。

會看自己演的連續劇，有時間就看電視播，沒時間就會預錄，看戲也會看到哭。由於讀劇本會讀到其他演員的部分，所以特別好奇其他人演出後的結果，對自己的部分則很普通地看待<ref>{{cite journal |journal=+act（2018年5月號）}}</ref>。
