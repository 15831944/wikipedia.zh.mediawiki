'''真普選聯盟'''（[[英文|英文]]：'''Alliance for True Democracy'''）於2013年3月21日由12個[[香港泛民主派|香港泛民主派]][[香港泛民主派列表|政黨及團體]]聯合成立，目標為推動[[香港行政長官|香港行政長官]]和[[香港立法會|香港立法會]]盡快實現[[雙普選|雙普選]]（即[[香港行政長官選舉|香港行政長官選舉]]及[[香港立法會選舉|香港立法會選舉]]）與「[[真普選|真普選]]」，召集人為[[香港城市大學|香港城市大學]][[政治學系|政治學系]][[講座教授|講座教授]][[鄭宇碩|鄭宇碩]]。<ref name="主場">[http://thehousenews.com/politics/真普選聯盟今成立-人力加入/ 真普選聯盟今成立 人力加入]{{webarchive|url=https://web.archive.org/web/20150522092832/http://thehousenews.com/politics/%E7%9C%9F%E6%99%AE%E9%81%B8%E8%81%AF%E7%9B%9F%E4%BB%8A%E6%88%90%E7%AB%8B-%E4%BA%BA%E5%8A%9B%E5%8A%A0%E5%85%A5/ |date=2015-05-22 }}  主場新聞 2013年3月21日</ref>逢週三舉行[[會議|會議]]。

== 歷史 ==
2013年3月21日，12個香港泛民主派政黨及團體聯合成立真普選聯盟，27名泛民主派[[香港立法會議員|香港立法會議員]]全體加入。人民力量主席[[劉嘉鴻|劉嘉鴻]]批評[[梁振英|梁振英]]在推動普選上全無誠意，只是[[中央人民政府駐香港特別行政區聯絡辦公室|中央人民政府駐香港特別行政區聯絡辦公室]]及[[中華人民共和國中央政府|中華人民共和國中央政府]]的一個[[傀儡|傀儡]]。他表示真普選聯盟是一個全新的平台，讓各個擁有不同想法的黨派將分歧減到最低，努力爭取普選，帶動香港市民爭取普選的聲音。首階段工作是大規模地宣傳普選，及於同年年底前制定一個「可以接受」的[[香港政治制度改革|香港政治制度改革]]方案。<ref>[http://www.bbc.co.uk/zhongwen/trad/china/2013/03/130321_hk_alliance_for_true_democracy.shtml 香港政黨成立真普選聯盟促雙普選], [[BBC|BBC]]中文網, 2013年3月21日</ref>[[公民黨|公民黨]][[黨魁|黨魁]][[梁家傑|梁家傑]]表示，[[香港泛民主派|香港泛民主派]]只有合作的必要，無分裂的條件，好的開始就是成功的一半，必須團結爭取「真[[普選|普選]]」。工黨主席[[李卓人|李卓人]]批評，[[中華人民共和國中央政府|中華人民共和國中央政府]]無誠意推動「[[真普選|真普選]]」<ref>[http://hk.news.yahoo.com/泛民主派成立真普選聯盟-045603111.html 泛民主派成立真普選聯盟]</ref><ref>[http://hk.news.yahoo.com/真普選聯盟成立-公民黨-泛民沒分裂條件-043100987.html 真普選聯盟成立　公民黨：泛民沒分裂條件]</ref><ref>[http://hk.news.yahoo.com/真普選聯盟成立-公民黨-泛民沒分裂條件-043100987.html 真普選聯盟成立　公民黨：泛民沒分裂條件]</ref>，形容目前是與中共及既得利益集團對決的關鍵時刻。民主黨副主席[[蔡耀昌|蔡耀昌]]則強烈譴責，[[中華人民共和國政府|中華人民共和國政府]]在[[全國人民代表大會|全國人民代表大會]]、[[地方各級人民代表大會|地方各級人民代表大會]]及[[中國人民政治協商會議|中國人民政治協商會議]]期間放出風聲，指出香港行政長官選舉或者會有預選機制，表明不排除以[[公民抗命|公民抗命]]的方式爭取普選。真普聯的第一階段工作為啟動[[香港市民|香港市民]]參與討論普選方案，清楚界定香港市民對普選的訴求、方案及底線，之後尋求各界共識<ref>[http://hk.news.yahoo.com/真普選聯盟發動社會討論真普選方案-053610918.html 真普選聯盟發動社會討論真普選方案]</ref>，再向[[香港特別行政區政府|香港特別行政區政府]]提交建議<ref>[http://hk.news.yahoo.com/真普選聯盟-成立-045200131.html 「真普選聯盟」成立] 《星島日報》 2013年3月21日</ref><ref>[http://hk.news.yahoo.com/真普選聯盟今宣布正式成立-042500216.html 真普選聯盟今宣布正式成立]</ref><ref>[http://hk.news.yahoo.com/真普選聯盟今宣布正式成立-042500216.html 真普選聯盟今宣布正式成立]</ref>。第二階工作為就向香港政府、中共[[黨和國家領導人|黨和國家領導人]]以至國際社會宣示訴求<ref>[http://hk.news.yahoo.com/真普聯成立-抗篩選預選-220225105.html 真普聯成立 抗篩選預選] 《星島日報》 2013年3月22日</ref>。真普選聯盟呼籲[[香港政府|香港政府]]盡快展開香港政治制度改革諮詢，而普選方案應該以國際標準制訂，需要符合[[聯合國|聯合國]]《[[公民權利和政治權利國際公約|公民權利和政治權利國際公約]]》的規定。真普選聯盟又表示，如果香港行政長官選舉任何篩選或者預選機制令到廣泛受到香港市民支持的人士無辦法參與選舉，必然是「假普選」<ref>[http://hk.news.yahoo.com/真普選聯盟今日成立-041210189.html 真普選聯盟今日成立] 《明報》 2013年3月21日</ref><ref>[http://hk.news.yahoo.com/真普選聯盟今日成立-041210189.html 真普選聯盟今日成立] 《明報》 2013年3月21日</ref>。

2013年3月29日下午，真普選聯盟10多名代表[[遊行|遊行]]至中聯辦，沿途高呼[[口號|口號]]，要求「[[真普選|真普選]]」，又向[[全國人大法律委員會|全國人大法律委員會]]主任[[喬曉陽|喬曉陽]]「下戰書」，邀請他出席真普選聯盟所舉辦的論壇，與廣大香港市民討論何謂「真普選」<ref>[http://hk.news.yahoo.com/真普選聯盟邀喬曉陽談真普選-072000074.html 真普選聯盟邀喬曉陽談真普選] 《星島日報》 2013年3月29日</ref>。

2013年4月4日，真普選聯盟表示計劃邀請[[建制派|建制派]]人士出席於同年5月舉行的研討會，對象包括[[香港基本法委員會|香港基本法委員會]]委員會副主任[[梁愛詩|梁愛詩]]、[[港區人大代表|港區人大代表]][[譚惠珠|譚惠珠]]、[[行政會議|行政會議]]成員[[張志剛|張志剛]]和[[民建聯|民建聯]]代表等等<ref>[http://hk.news.yahoo.com/真普聯邀建制派對話-220442888.html 真普聯邀建制派對話] 《星島日報》 2013年4月5日</ref>。

2013年4月7日，真普選聯盟舉行第一場[[研討會|研討會]]<ref>[http://hk.news.yahoo.com/真普選聯盟成立-推動特首和立會盡快雙普選-050000696.html 真普選聯盟成立　推動特首和立會盡快雙普選]</ref>，在[[香港城市大學|香港城市大學]]舉行。研討會開始前，[[愛護香港力量|愛護香港力量]]成員在場外叫喊，要求縮短講者的演講時間。研討會一開始時由[[莫乃光|莫乃光]]發言，發言幾分鐘後，台下開始有人叫喊，大會先行暫停十分鐘後再開始，惟同樣情況再發生；台下約30名自稱支持政府的人士與其他在場人士互相對罵，有人拍掌，又有人舉起橫額，場面失控，召集人要求保安員維持秩序，惟仍然未能夠控制場面。最後，真普選聯盟報告[[警察|警察]]，並且宣布[[腰斬|腰斬]]論壇<ref>[http://hk.news.yahoo.com/鄭宇碩對研討會受滋擾表示遺憾-114608514.html 鄭宇碩對研討會受滋擾表示遺憾]</ref><ref>[http://hk.news.yahoo.com/真普選聯盟被迫腰斬論壇-115814993.html 真普選聯盟被迫腰斬論壇] 《明報》 2013年4月7日</ref><ref>[http://hk.news.yahoo.com/真普選聯盟研討會秩序混亂腰斬-104300955.html 真普選聯盟研討會秩序混亂腰斬]</ref><ref>[http://hk.news.yahoo.com/撐政府市民叫喊-真普聯研討會兩度中斷-065800788.html 撐政府市民叫喊　真普聯研討會兩度中斷]</ref><ref>[http://hk.news.yahoo.com/撐政府市民叫喊-真普聯腰斬研討會-070500297.html 撐政府市民叫喊　真普聯腰斬研討會]</ref><ref>[http://hk.news.yahoo.com/真普選聯盟腰斬研討會-073100780.html 真普選聯盟腰斬研討會] 《星島日報》 2013年4月7日</ref><ref>[http://hk.news.yahoo.com/研討會腰斬-撐政府市民-不滿研討會安排-081500598.html 研討會腰斬　撐政府市民：不滿研討會安排]</ref><ref>[http://hk.news.yahoo.com/真普選論壇被滋擾需暫停-084614621.html 真普選論壇被滋擾需暫停]</ref><ref>[http://hk.news.yahoo.com/真普聯研討會謾罵中腰斬-220418455.html 真普聯研討會謾罵中腰斬] 《星島日報》 2013年4月8日</ref><ref>[http://hk.news.yahoo.com/真普聯研討會遭踩場腰斬-守護中環-插言叫囂惹爭執-211616458.html 真普聯研討會遭踩場腰斬 「守護中環」插言叫囂惹爭執] 《明報》 2013年4月8日</ref><ref>[http://orientaldaily.on.cc/cnt/news/20130408/00176_019.html 示威者柴台 真普聯論壇腰斬] 《東方日報》 2013年4月8日</ref>。

2013年5月5日下午，真普選聯盟舉行研討會，出席的香港民主民生協進會主席[[廖成利|廖成利]]表至希望可以透過真普選聯盟、[[香港2020|香港2020]]及[[讓愛與和平佔領中環|佔領中環]]等醞釀「真普選」方案，又希望「真普選」符合國際人權普及而平等的標準和基本法。人民力量主席劉嘉鴻表示，香港民主的迫切性和香港市民爭取普選的決心非常重要。他表示，未必同意佔領中環的做法，但是認同佔領中環可以令到香港市民知道實行普選的迫切性<ref>[http://hk.news.yahoo.com/真普聯舉行研討會-民協冀醞釀真普選方案-074100369.html 真普聯舉行研討會　民協冀醞釀真普選方案]</ref>。前[[全國政協委員|全國政協委員]][[劉夢熊|劉夢熊]]引述兼任[[中央港澳工作協調小組|中央港澳工作協調小組]]組長的[[全國人大常委會委員長|全國人大常委會委員長]][[張德江|張德江]]早前言論，表示香港首要發展經濟以及改善民生，相對鄰近地區，香港的優勢已經變得薄弱為不爭的事實，何時展開政治制度改革諮詢需要視乎香港社會的主流意見，惟前提是不應該令到香港社會內耗、影響民生及經濟發展<ref>[http://hk.news.yahoo.com/劉夢熊指港首要發展經濟-092000691.html 劉夢熊指港首要發展經濟] 《星島日報》 2013年5月5日</ref>。他又表達擔心佔領中環開啟壞的先例，導致香港永無寧日，擔心以後香港人凡是有訴求，都會佔領中環要脅[[中央政府|中央政府]]，希望佔領中環的發起人懸崖勒馬。發起佔領中環的[[戴耀廷|戴耀廷]]就此再次強調，佔領中環只是最後一步，並且表示將會和平進行。民間人權陣線召集人[[孔令瑜|孔令瑜]]表示，過去十幾年香港民主的發展原地踏步，他表示支持佔領中環，希望公開違反不公義的法律。此外，台下公眾發言時，支持及反對者多次互相喝倒采，及後爆發罵戰<ref>[http://hk.news.yahoo.com/真普聯論壇演變成罵戰-104547876.html 真普聯論壇演變成罵戰]</ref>。研討會結束時，雙方發生衝突，嘉賓需要由保安員護送離開現場<ref>[http://hk.news.yahoo.com/劉夢熊-佔領中環先例-開-香港永無寧日-105900742.html 劉夢熊：佔領中環先例一開　香港永無寧日]</ref>。

2013年5月8日，真普選聯盟將會公布就普選方案達成的七大共識<ref>[http://hk.news.yahoo.com/真普聯達七共識-爭取普選提委會-210618275.html 真普聯達七共識 爭取普選提委會] 《明報》 2013年5月6日</ref>。5月9日，真普選聯盟公布了普選行政長官選舉辦法的初步建議，建議以一人一票方式選舉提名委員會會員；而參選人只需要獲得提名1/8委員的提名，或者一定選民聯署提名，及不要求必須取得任何指定界別的提名就可以參與選舉，惟此建議未獲得內部一致贊成。建議普選行政長官選舉若然在首輪投票中，任何候選人獲得逾半有效投票，即宣告當選。然而，如果無候選人在首輪投票中取得逾半有效投票，最高票的兩人就會進入第二輪投票，在第二輪投票得票比較多的一位就會當選行政長官。其他建議重點包括，不接受參選人被預選或者篩選<ref>[http://hk.news.yahoo.com/真普選聯提2017年特首選舉方案-140605078.html 真普選聯提2017年特首選舉方案]</ref>，提名委員會必須由香港選民以一人一票方式選舉出來、提名委員不可以重複提名、不設立候選人的數目限制、參選人不可以提交多於1/6委員提名人數的提名投票等<ref>[http://orientaldaily.on.cc/cnt/news/20130506/00176_011.html 真普聯倡全民普選提名委員] 《東方日報》 2013年5月6日</ref>，長遠廢除提名委員會。初步建議將會交由8名學者組成的學者團體歸納成為方案，預計於同年6月推出草擬方案<ref>[http://hk.news.yahoo.com/真普聯諮詢稿-未獲人力確認-211412886.html 真普聯諮詢稿 未獲人力確認] 《明報》 2013年5月9日</ref><ref>[http://hk.news.yahoo.com/真普聯-爭取全港選民選出提名委員會-131900203.html 真普聯：爭取全港選民選出提名委員會]</ref><ref>[http://hk.news.yahoo.com/真普聯按初步意見與學者專家擬普選特首方案-144000571.html 真普聯按初步意見與學者專家擬普選特首方案]</ref>。前[[律政司司長|律政司司長]]暨[[香港特別行政區基本法委員會|香港特別行政區基本法委員會]]副主任[[梁愛詩|梁愛詩]]表示，上述的委員會產生辦法建議變成將提名權力投放在選民手中，而非提名委員會手中，此舉不合乎《基本法》的規定，亦與2007年在[[人大常委會|人大常委會]]上的決定不相符；但是認為1/8委員提名及兩輪投票的建議皆可以考慮。她表示，現在最重要的是為各方提出的不同方案去蕪存菁，作出討論。初步建議初步獲得泛民主派24名立法會議員接納，人民力量主席劉嘉鴻表示，執事委員會將會進一步討論，若果人民力量不接受此初步建議，不排除將會退出聯盟。人民力量立法會議員陳偉業和黃毓民則表示，與真普選聯盟可以各自表述立場<ref>[http://hk.news.yahoo.com/真普聯倡1-8提名可選特首-220204816.html 真普聯倡1/8提名可選特首] 《星島日報》 2013年5月9日</ref>。

2013年12月8日，
民主黨及工黨分別提出普選方案，同樣建議於2017年香港行政長官選舉「提名三軌制」，即市民只需要循公民、政黨或者提名委員會任何一個中，取得足夠有效提名即可以報請參與選舉<ref>[http://orientaldaily.on.cc/cnt/news/20131209/00176_025.html 真普聯料下月公布「三軌制」方案] 《東方日報》 2013年12月9日</ref>，同時容許行政長官擁有政黨背景，惟工黨建議的公民提名的門檻只訂於30,000人，比較民主黨的方案少一半；真普選聯盟主流對此提案支持<ref>[http://thehousenews.com/politics/%E6%B0%91%E4%B8%BB%E9%BB%A8%E5%B7%A5%E9%BB%A8%E5%80%A1-%E6%8F%90%E5%90%8D%E4%B8%89%E8%BB%8C%E5%88%B6/ 民主黨工黨倡 提名三軌制] {{webarchive|url=https://web.archive.org/web/20140607081647/http://thehousenews.com/politics/%E6%B0%91%E4%B8%BB%E9%BB%A8%E5%B7%A5%E9%BB%A8%E5%80%A1-%E6%8F%90%E5%90%8D%E4%B8%89%E8%BB%8C%E5%88%B6/ |date=2014-06-07 }} 主場新聞 2014年1月13日</ref>。

# '''提名委員會提名'''：由官方提名委員會成員直接提名參選人
}}

提名需要經由提名委員會確認，真普選聯盟認為提名委員會難以不確認得到公民支持的人報請參選，又反駁建制派認為此舉將會削弱提名委員會的質疑。方案並無為候選人數目設置上限，真普選聯盟建議採用兩輪決選法，若然首輪在投票未有人取得過半數選票，即進行次輪投票，由高票者當選。

對於「三軌提名制」提名方法是否缺一不可，民主黨及人民力量意見存有嚴重分歧。真普選聯盟召集人鄭宇碩承認在討論過程中有成員對「缺一不可」的講法感到不舒服，所以真普選聯盟決定不採用有關字眼，以便有足夠的論述空間。鄭宇碩表示，目前所有成員在政府公布政治改革方案前，都會全力推動真普選聯盟的唯一方案。2014年1月11日，行政會議召集人[[林煥光|林煥光]]表示，期望[[政制諮詢專責小組|政制諮詢專責小組]]可以對此仔細研究。行政會議成員[[張志剛|張志剛]]在出席電台節目時，則表示上述方案仍然存有妥協空間。
