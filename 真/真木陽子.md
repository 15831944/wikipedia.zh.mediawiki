'''真木陽子'''（{{Lang-ja|'''真木 よう子'''}}，{{bd|1982年|10月15日|||Maki Yoko}}），[[日本|日本]]女[[演員|演員]]。[[千葉縣|千葉縣]][[印西市|印西市]]出生。原隸屬[[Flying_Box|Flying Box]]事務所，2018年1月，真木發表公開聲明稿，表示已與事務所約滿且不續約；2018年9月25日，加入[[LesPros娛樂|LesPros娛樂]]。<ref>[https://www.oricon.co.jp/news/2120229/ 真木よう子、レプロエンタテインメントと契約]</ref>

==生平==
小学2年級時因為[[安達祐實|安達祐實]]主演的《[[REX_恐龍物語|REX 恐龍物語]]》，開始想要成為藝人，加上當時對唸書完全沒有興趣，因此沒有考慮過升學的事<ref name="minasan">2010年10月28日放送『[[とんねるずのみなさんのおかげでした|とんねるずのみなさんのおかげでした]]』のコーナー「新・食わず嫌い王決定戦」出演時に発言</ref>。但當時父親對於真木選擇進入藝能界強烈反對，真木在父親反對之下，以「不答應，我就去[[援交|援交]]！」的說詞最終取得同意<ref>[http://japan.techinsight.jp/2010/10/makiyouko-enkousuru.html 「ダメなら援助交際する」。真木よう子が女優の道を目指して決意したこと。]</ref>。

1998年中學畢業後，真木陽子從1000人之中選出5人的徵選脫穎而出，得以加入[[仲代達矢|仲代達矢]]管理的演員養成所「無名塾」（同期成員包含[[滝藤賢一|滝藤賢一]]、[[内浦純一|内浦純一]]<ref>{{Cite web |author= 木俣冬 |date= 2012-06-14 |url= http://www.excite.co.jp/News/reviewmov/20120614/E1339605706621.html?_p=2 |title= 「リーガル・ハイ」でも注目の脚本家・古沢良太に「外事警察」堀切園健太郎監督、続編執筆承諾を詰め寄る！|publisher= exciteニュース|accessdate=2012-08-29}}
</ref><ref>{{Cite web |author= [http://www.mumeijuku.net/prof/prof_kamakura.html 鎌倉太郎] |date= 2012-03-23 |url= http://navy.ap.teacup.com/takogasuki/1405.html |title= タキ、久しぶりやね。|work= 鎌倉太郎のプロジェクトB |publisher= |accessdate=2012-08-29}}</ref>）。1999年至2000年間，入社的第二年即獲得拔擢參與公演《{{tsl|ja|どん底|谷底}}》的重要角色，並展開半年巡演。

2006年11月，和[[小田切讓|小田切讓]]共演《{{link-ja|吊橋上的秘密|ゆれる}}》，獲得第30回{{link-ja|山路文子電影賞|山路ふみ子映画賞}}新人女優賞，開始受到注目，期間並演出多部電視連續劇。

2013年6月，主演電影《[[再見溪谷|再見溪谷]]》。翌年，真木陽子以《再見溪谷》和《[[我的意外爸爸|我的意外爸爸]]》獲得第37回[[日本電影金像獎|日本電影金像獎]]最優秀女主角獎、最優秀女配角獎<ref name="37jacademy">{{Cite web|url=http://www.japan-academy-prize.jp/prizes/37.html|title=第37回日本アカデミー賞優秀作品発表!|publisher=[[日本アカデミー賞|日本アカデミー賞]]公式サイト|accessdate=2014-03-07}}</ref>，為日本電影金像獎史上繼1978年[[大竹忍|大竹忍]]之後，第二位「同屆獲得女主角與女配角獎」的演員。

== 個人生活 ==
*真木家共四個小孩，一個哥哥和兩個弟弟，真木排行第二。
*2008年11月10日，真木陽子與前男演員{{tsl|ja|片山怜雄}}登記結婚；2009年5月10日，第一個女兒出生。
*2015年9月與丈夫片山怜雄離婚，結束6年多的婚姻，並獲得親子撫養權。<ref>{{cite news| url= http://www.appledaily.com.tw/realtimenews/article/new/20150926/699448/ |title= 真木陽子宣布離婚　獲6歲女兒撫養權 |date= 2015年9月26日 | accessdate=2016-04-29| author=蘇威全 | publisher=蘋果日報}}</ref>

== 交友喜好 ==
*國中時是[[田徑|田徑]]選手。個性男孩子氣。
*共演者表示長相看似有距離感，但其實性格很迷人。
*從小是[[漫畫|漫畫]]迷，雜誌曾邀請真木和漫畫家[[松本大洋|松本大洋]]、[[東清彥|東清彥]]等人對談。
*因2005年和[[大塚愛|大塚愛]]共演電影《{{link-ja|東京朋友|東京フレンズ}}》而相熟，亦和歌手[[木村KAELA|木村KAELA]]交情好。電影《[[再見溪谷|再見溪谷]]》製片方向真木提出親自演唱主題曲的邀請時，真木向朋友[[椎名林檎|椎名林檎]]提出創作主題曲的邀約<ref>[http://natalie.mu/music/news/86586 椎名林檎×真木よう子、映画「さよなら渓谷」主題歌コラボ]</ref>。
