'''真布站'''（{{jpn|j=真布駅|hg=まっぷえき|rm=Mappu eki|f=y}}）是位於[[北海道|北海道]][[雨龍郡|雨龍郡]][[沼田町|沼田町]]字真布，隸屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[留萌本線|留萌本線]]的[[鐵路車站|鐵路車站]]。電報略號為'''マフ'''。

車站名稱源自[[阿伊努語|阿伊努語]]「pan-ke-sir-utur-oma-p」，即下側的山之間的地域<ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/121130P.pdf|title=アイヌ語地名リスト|accessdate=2017-08-17|author=北海道庁|date=|publisher=北海道庁}}</ref>。

== 歷史 ==
* 1956年（昭和31年）7月1日 - [[日本國有鐵道|日本國有鐵道]]留萌本線石狩沼田站至惠比島站之間開設'''真布臨時車站'''，開始接待旅客業務。
* 1987年（昭和62年）4月1日 - 由於[[國鐵分割民營化|國鐵分割民營化]]，車站由[[北海道旅客鐵道|JR北海道]]管理，同時升格為'''真布站'''。
* 1990年（平成2年）3月10日 - 設定營業距離。

== 車站構造 ==
現時為1面1線的地面車站，側式月台位於路軌的西南方（增毛方向的左方）。沒有設置轉轍器的單線路軌車站。

自臨時車站開設以來已是沒有建築物的無人車站，車站北方設有候車室。候車室是用木板建成，由於身處積雪較多的區域，正門的單披式屋頂令積雪能夠向後卸下。內部亦使用斜支柱加強其支撐力量。牆壁放置一幅細小手製的站名標誌牌。沒有設置[[廁所|廁所]]。月台為木製甲板式，深川方向設有斜路連接車站外的道路，增毛方向的月台邊緣則有「月台終點」的標識。

本車站是[[秘境車站|秘境車站]]之一。

== 載客量 ==
* 1992年度的每天載客量為8人。
* 2011年至2015年度平均每天載客量少於1人<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |language=ja |title=平成26年度 線区別の収支状況等について |publisher=JR北海道 |date=2016-02-10 |accessdate=2017-06-21 |archive-url=https://web.archive.org/web/20170407133949/http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |archive-date=2017-04-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20170407133949/http://www.jrhokkaido.co.jp/press/2016/160210-1.pdf |archivedate=2017-04-07 }}</ref>。

== 車站周邊 ==
位於農地的中央位置。周圍只有約7至8戶人家。以前主要是沼田第5地區及真布地區的學生使用本車站。村落較過去減少，不過周圍都是能夠種植良好稻米的土地。
* [[北海道道1007號惠比島旭町線|北海道道1007號惠比島旭町線]]
* [[北海道道549號峠下沼田線|北海道道549號峠下沼田線]]
* 發堀化石體驗場
* 藤澤儲水池
* 藤澤水庫
* Shirutorumappu（{{lang|ja|シルトルマップ}}）川 - 雨龍川的支流。
