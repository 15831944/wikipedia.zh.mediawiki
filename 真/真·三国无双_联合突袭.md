{{Vgname|真·三国无双 联合突袭|ja=真・三國無双 MULTI RAID|en=Dynasty Warriors: Strikeforce|diff=yes|cn=真·三国无双 联合突袭|hk=真·三國無雙 連袂出擊|tw=真·三國無雙 連袂出擊|以下简称“'''联合突袭'''”}}是由[[光荣|光荣]]公司发售的基于《[[真·三國無雙5|真·三国无双5]]》的动作游戏。其标语是“觉醒吧，尚未谋面的伙伴们”{{NoteTag|原文为「{{Lang|ja|-{覚醒せよ。まだ見ぬ仲間たちよ。}-}}」}}。本作于2009年春季在北美和欧洲地区发布PSP版<ref>{{Cite web|url=http://koeiwarriors.co.uk/dw6-multiraid.php|title=Dynasty Warriors 6/Multi Raid|accessdate=2008-09-28|author=Koei UK|date=|publisher=|archive-url=https://web.archive.org/web/20081007164407/http://koeiwarriors.co.uk/dw6-multiraid.php|archive-date=2008-10-07|dead-url=yes}}</ref>，于同年10月在日本发布PS3版和Xbox 360版《真·三国无双：联合突袭Special》。其续作《[[真·三國無雙_連袂出擊2|联合突袭2]]》于2010年在日本发布，没有海外版。

== 系统 ==
本作是继《[[真·三國無雙4|真·三国无双4]]》之后真·三国无双系列第二次采用“觉醒”系统的作品（称“真·无双觉醒”）。觉醒状态下，人物形象发生显著的变化，能力得到大幅提升<ref name="コーエー、PSP「真・三國無双 MULTI RAID」 マルチプレイに注力した新たなる「無双」">{{cite web|url=http://game.watch.impress.co.jp/docs/20080922/musomr.htm|title=コーエー、PSP「真・三國無双 MULTI RAID」 マルチプレイに注力した新たなる「無双」|accessdate=2010-02-08|language=ja|date=2009-09-22|publisher=GameWatch|author=Shinji Nakano|archive-url=https://web.archive.org/web/20090830065935/http://game.watch.impress.co.jp/docs/20080922/musomr.htm|archive-date=2009-08-30|dead-url=no}}</ref>。角色可以进行更多自定义，装备两件武器及更多增加体力、攻击力等属性的物品<ref name="Review: Dynasty Warriors Strikeforce">{{cite web|url=http://www.destructoid.com/review-dynasty-warriors-strikeforce-130063.phtml|title=Review: Dynasty Warriors Strikeforce|date=2009-04-28|language=en|publisher=Destructoid|author=Jim Sterling|access-date=2018-10-01|archive-url=https://web.archive.org/web/20160305010657/http://www.destructoid.com/review-dynasty-warriors-strikeforce-130063.phtml|archive-date=2016-03-05|dead-url=no}}</ref>。PSP版更支持在线多人游戏，一名玩家最多可与3名其他玩家进入同一战场（可以合作，也可以敌对）<ref name="This entertaining hack-and-slash game is best played with a few friends.">{{cite web|url=http://www.gamespot.com/psp/action/shinsangokumusoumultiraid/review.html?om_act=convert&om_clk=gssummary&tag=summary;read-review|title=This entertaining hack-and-slash game is best played with a few friends.|accessdate=2010-02-08|date=2009-08-25|language=en|publisher=GameSpot|author=Brian Fishman|archive-url=https://web.archive.org/web/20091026193605/http://www.gamespot.com/psp/action/shinsangokumusoumultiraid/review.html?om_act=convert&om_clk=gssummary&tag=summary;read-review|archive-date=2009-10-26|dead-url=no}}</ref>。在准备画面中，玩家可以造访锻炼武器的锻冶铺、提供道具的学者和商店等设施。而这些设施也被加入真·三国无双系列6代以后的作品中。

== 角色 ==
《联合突袭》的角色平时形象取自《真·三国无双5》，角色觉醒后形象大变。注：孟获也在《真·三国无双5》特别版和《联合突袭》PS3和X360版登场。一部分在真·三国无双4代登场而未在5代登场的武将于《联合突袭2》和《[[真·三國無雙6|真·三国无双6]]》中登場。

* 魏：夏侯惇、典韦、曹操、张辽、许褚、张郃、夏侯渊、曹丕、甄姬、徐晃
* 蜀：赵云、关羽、刘备、诸葛亮、马超、庞统、魏延、黄忠、月英
* 吴：周瑜、陆逊、孙尚香、甘宁、孙坚、太史慈、孙策、孙权、吕蒙、周泰、小乔
* 其他：貂蝉、吕布、董卓、孟获、张角

== 开发 ==
《[[週刊少年Jump|周刊少年JUMP]]》在2008年9月13日的杂志上最先刊登了有关《联合突袭》的消息。《[[Fami通|Fami通]]》在一周后更刊登了夏侯惇、赵云和孙尚香“真·无双觉醒”状态的形象和一些截图<ref>{{Cite web|url=http://koeiwarriors.co.uk/images/magazinescans/dwmr/dwmr-2.jpg|title=Dynasty Warriors Multi Raid|accessdate=|author=|date=|publisher=|archive-url=https://web.archive.org/web/20160314081801/http://koeiwarriors.co.uk/images/magazinescans/dwmr/dwmr-2.jpg|archive-date=2016-03-14|dead-url=yes}}</ref>。本作原定于2009年3月发行，但和《[[真·三國無雙5|真·三国无双5：帝国]]》一同被光荣推迟发售。游戏的日本官网提供了一个测试版游戏的下载通道，可以在PSP上游玩。此一版本开放了魏、蜀、吴三个势力所有可操作武将，但在发行几小时后，有人发现了一个可开放“其他”势力武将的代码。在光荣诉诸法律之后，这个代码不再可用，此测试版游戏加入了角色等级的限制，只开放三个关卡，但所有类型的武器依然可用。

后来发行的家用机版本的测试版游戏于日版游戏商店上柜，有6个可用角色和4个关卡；英文测试版则在2010年于[[PlayStation_Network|PSN]]上柜，但只有4个可用角色和1个关卡<ref>{{Cite web|url=http://www.ign.com/articles/2010/02/05/dynasty-warriors-strikeforce-online-demo-available-now|title=Dynasty Warriors: Strikeforce Online Demo Available Now|accessdate=2018-10-01|date=2010-02-04|last=Staff|first=I. G. N.|work=IGN|language=en-US|archive-url=https://web.archive.org/web/20121121210638/http://www.ign.com/articles/2010/02/05/dynasty-warriors-strikeforce-online-demo-available-now|archive-date=2012-11-21|dead-url=no}}</ref>。

== 《联合突袭Special》 ==
移植到PS3版和X360版的《联合突袭Special》在PSP原版游戏的基础上追加了更多要素，例如受PSP机能所限而被删除的动画声音。这些要素也加入到后来的《联合突袭2》当中。这一作的标语是“不一样的速度，不一样的刺激”{{NoteTag|原文为「{{Lang|ja|スピードが違う。刺激が違う。}}」}}。
