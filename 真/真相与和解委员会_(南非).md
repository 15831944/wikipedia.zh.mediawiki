'''真相与和解委员会'''（Truth and Reconciliation Commission ），是[[南非|南非]]为实现“在弄清过去事实真相的基础上促进全国团结与民族和解”的目标，于1995年11月29日，宣布成立的社会调解组织。委员会主席为南非享有厚望的[[诺贝尔和平奖|诺贝尔和平奖]]得主南非圣公会大主教[[德斯蒙德·图图|德斯蒙德·图图]]。

==背景==
参见：[[南非种族隔离|南非种族隔离]]
*南非曾经是世界上运用立法与行政手段把种族隔离作为一种“国策”来推行的唯一国家。自1652年[[荷兰|荷兰]]殖民者入侵南非始，白人殖民者对土著黑人的奴役就开始。1910年，[[南非联邦|南非联邦]]成立，种族隔离制度正式建立。从1911年到1948年，白人统治者制定了近五十项[[种族隔离|种族隔离]]的法律。1948年，右翼的[[南非国民党|南非国民党]]上台，打着“维护白种人的纯洁性”的旗号，以极端的[[种族主义|种族主义]]和狭隘的[[阿非利卡人|阿非利卡人]]（荷裔白人）[[民族主义|民族主义]]为根本指导思想,从1950年起，陆续颁布了《人口登记法》、《集团住区法》、《公共场所隔离保留法》、《土地法》等法律，将种族歧视与隔离推向极端，制造南非历史上最反动、最黑暗的时期。种族隔离制度将[[黑人|黑人]]赶出城市，圈定在贫瘠偏远的穷乡僻壤，占全国人口73%的黑人被驱赶到12.7%的土地上居住。<ref>{{cite web |url=http://news.ifeng.com/history/gundong/detail_2012_09/05/17355682_0.shtml?_from_ralated |title=南非“真相与和解”之路 |accessdate= 2012年9月5日|publisher=凤凰网 |work= |language =zh-cn}}</ref>

==成立==
1995年1月24日南非公布《促进民族团结与和解法案》，该法案提议由11—17名独立人士组成真相与和解委员会（简称真相委员会）。7月19日，[[曼德拉|曼德拉]]总统签署《促进民族团结与和解法》，1995年11月29日，南非政府宣布组成真相与和解委员会。<ref>{{cite web |url=http://worldcup.qq.com/a/20100515/000056.htm?_from_ralated |title=学会宽恕——真相与和解委员会 |accessdate= 2010年5月15日|publisher=腾讯体育|work= |language =zh-cn}} </ref>

==任务==
{{quote|1．全面地调查自1960年3月1日至1994年5月10日这段历史时期内各种严重侵犯人权事件的真相；
*2．让受害者讲出真相以恢复他们的公民尊严，并提出如何对这些受害者给予救助；
*3．考虑对那些服从政治指令严重侵犯人权但已向真相委员会讲出所有事实真相的犯罪者实施大赦。}}

==评价==
普遍认为，真相与和解委员会改变了南非的历史，让南非人看清了自己和自己的国家，并抚平了南非人心中的怨恨，为南非今天的稳定与和解奠定了基础。<ref>{{cite web |url=http://worldcup.qq.com/a/20100515/000056.htm?_from_ralated |title=学会宽恕——真相与和解委员会|accessdate= 2010年5月15日|publisher=腾讯体育 |work= |language =zh-cn}}</ref>
