'''费奥多尔·尼古拉耶维奇·尤尔奇欣'''（{{lang-ru|Фёдор Никола́евич Юрчи́хин}}，{{bd|1959年|1月3日|}}）是[[希腊|希腊]]裔[[俄罗斯|俄罗斯]][[宇航员|宇航员]]，俄罗斯第98位宇航员，世界第423位宇航员。他曾5次前往[[国际空间站|国际空间站]]执行长期任务，累计进行过9次太空行走。

==早年==
1959年1月3日出生于苏联[[格鲁吉亚苏维埃社会主义共和国|格鲁吉亚]][[阿查拉蘇維埃社會主義自治共和國|阿查拉蘇維埃社會主義自治共和國]][[巴统|巴统]]（现[[格鲁吉亚|格鲁吉亚]][[阿扎尔|阿扎尔]][[自治共和国|自治共和国]]），父母是希腊的[[本都人|本都人]]，现居[[希腊|希腊]][[塞萨洛尼基|塞萨洛尼基]]郊区<ref>{{cite web |url=http://www.federalspace.ru/main.php?id=2&nid=9596 |title=Soyuz TMA-19: Conquering the Olympus |author=Russian Federal Space Agency |date=2010-06-05 |accessdate=2010-06-06}}</ref>。

1976年从巴统高中毕业后，进入以[[格里戈里·康斯坦丁诺维奇·奥尔忠尼启则|格里戈里·奥尔忠尼启则]]命名的[[莫斯科航空学院|莫斯科航空学院]]。1983年毕业，获得[[机械工程|机械工程]]学位。毕业后的9月开始在[[科罗廖夫能源火箭航天集团|能源科研生产联合公司]]工作，历任工程师，高级工程师，首席工程师。2001年获得俄罗斯公共服务学院硕士学位<ref name=nasabio>{{cite web |title=Fyodor Nikolayevich Yurchikhin |url=https://web.archive.org/web/20170414153553/https://www.jsc.nasa.gov/Bios/htmlbios/yurchikhin.html |publisher=NASA |accessdate=2019-04-20}}</ref>。
==宇航员生涯==
1997年，成为宇航员候选人，开始在[[加加林宇航员培训中心|加加林宇航员培训中心]]接受基础太空训练，1999年12月1日获得测试宇航员资格<ref name=rsc>{{cite web |title=Fyodor Nikolaevich YURCHIKHIN |url=https://www.energia.ru/en/iss/iss51/yurchikhin.html |publisher=S.P. Korolev RSC "Energia" |accessdate=2019-04-20}}</ref>。
===STS-112===
2002年10月7日至10月18日，他乘坐[[亚特兰蒂斯号航天飞机|亚特兰蒂斯号航天飞机]]在国际空间站执行[[STS-112|STS-112]]任务<ref>{{cite web|url=http://www.space.com/missionlaunches/sts112_preview_021001.html |title=Mission Atlantis: Meet the Crewmembers of STS-112 |author=Jim Banke|accessdate=2010-06-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20100702215411/http://www.space.com/missionlaunches/sts112_preview_021001.html |archivedate=2010-07-02 }}</ref>。飞行时长为10日19时57分<ref name=rsc/>。
===远征15===
北京时间2007年4月8日凌晨1时31分，他和科托夫以及[[微软公司|微软公司]]创始人之一[[查尔斯·西蒙尼|查尔斯·西蒙尼]]乘坐联盟TMA-10飞船前往国际空间站<ref>{{cite web |author1=文刀 |title=Word之父升空前往空间站 成第五位太空游客 |url=http://tech.sina.com.cn/d/2007-04-08/01401454069.shtml |publisher=[[新浪|新浪]]科技 |archiveurl=https://web.archive.org/web/20190419160052/http://tech.sina.com.cn/d/2007-04-08/01401454069.shtml |archivedate=2019-04-19 |date=2007-04-08}}</ref>。

2007年7月23日，他和美国宇航员[[克莱顿·安德森|克莱顿·安德森]]进行了[[舱外活动|舱外太空行走]]任务，主要将空间站的废旧[[照相机|照相机]]支架和废旧氨水罐等废弃设备“扔入”[[太空|太空]]。其中，安德森负责“倒垃圾”，尤尔奇欣则充当辅助。完成所有任务后，两名宇航员于北京时间24日凌晨2时05分安全返回空间站内。本次太空行走共持续7小时41分钟<ref>{{cite web |title=宇航员行走太空“倒垃圾” |url=http://www.most.gov.cn/gnwkjdt/200707/t20070731_51891.htm |publisher=[[中华人民共和国科学技术部|中华人民共和国科学技术部]] |archiveurl=https://web.archive.org/web/20190419144412/http://www.most.gov.cn/gnwkjdt/200707/t20070731_51891.htm |archivedate=2019-04-19 |date=2007-08-01}}</ref>。

联盟TMA-10飞船于协调世界时10月21日7时14分从国际空间站脱离，后因在[[大气层|大气层]]受阻而未到达预定着陆点，而是降落在[[阿尔卡雷克|阿尔卡雷克]]西北约340公里处<ref>{{Cite web |url=http://www.spaceflightnow.com/station/exp15/071021landing.html |title=Soyuz capsule makes 'ballistic' descent to Earth |accessdate= 2007-10-21 |publisher=CBS News – Spaceflight Now |year=2007 |author=William Harwood}}</ref>。

===远征24/25===
[[File:Expedition_25_crew_portrait.jpg|thumb]]
北京时间2010年6月16日5时35分，[[联盟-FG|联盟-FG]]运载火箭携带[[联盟TMA-19|联盟TMA-19]]从哈萨克斯坦拜科努尔航天发射场发射升空，前往国际空间站<ref>{{cite web |title=俄“联盟TMA-19”载人飞船发射升空 |url=http://www.chinanews.com/tp/news/2010/06-16/2345823.shtml |publisher=中国新闻网 |archiveurl=https://web.archive.org/web/20190419153650/http://www.chinanews.com/tp/news/2010/06-16/2345823.shtml |archivedate=2019-04-19 |date=2010-06-16}}</ref>。其中尤尔奇欣担任指令长，美国飞行员[[道格拉斯·惠洛克|道格拉斯·惠洛克]]与女性太空物理学家[[香农·沃克|香农·沃克]]任随航工程师。

北京时间2010年7月27日12时11分，他和[[米哈伊尔·科尔尼延科|科尔尼延科]]穿着海鹰-MK宇航服，打开国际空间站[[码头号对接舱|码头号对接舱]]舱门，开始了太空行走任务。他们主要任务是从[[星辰号服务舱|星辰号服务舱]]外部更换了一个[[摄像头|摄像头]]，用于监测欧洲自动货运飞船与国际空间站俄罗斯舱段的对接过程；然后分别在[[黎明号实验舱|黎明号实验舱]]、星辰号服务舱和[[曙光號功能貨艙|曙光號功能貨艙]]之间铺设并接通了[[电缆|电缆]]，这标志着黎明号小型实验舱将正式投入使用。整个[[舱外活动|舱外活动]]共持续6小时43分<ref>{{cite web |author=聂云鹏 |title=国际空间站俄宇航员成功完成太空行走 |url=http://news.sciencenet.cn/htmlnews/2010/7/235248.shtm |publisher=[[新华社|新华社]] |archiveurl=https://web.archive.org/web/20190413062353/http://news.sciencenet.cn/htmlnews/2010/7/235248.shtm |archivedate=2019-04-13 |date=2010-07-29}}</ref>。

北京时间2010年11月15日22时55分，他和[[奥列格·斯克里波奇卡|奥列格·斯克里波奇卡]]打开国际空间站对接舱舱门，开始了约6个小时的[[舱外活动|舱外活动]]。他的主要任务是在[[星辰号服务舱|星辰号服务舱]]外侧安装一个多功能工作站<ref>{{cite web |author=刘恺 |title=国际空间站俄罗斯宇航员出舱太空行走|url=http://news.sciencenet.cn/htmlnews/2010/11/240235.shtm|publisher=科学时报 |accessdate=2019-04-10 |language=zh |date=2010-11-16}}</ref>。26日12时46分，他和队员乘坐相同飞船，平安降落在哈萨克斯坦[[阿尔卡雷克|阿尔卡雷克]]市以北的草原上<ref>{{cite web |title=国际空间站3名宇航员平安返回地面 |url=https://finance.qq.com/a/20101126/004445.htm |publisher=新华网 |archiveurl=https://web.archive.org/web/20190419154507/https://finance.qq.com/a/20101126/004445.htm |archivedate=2019-04-19 |date=2010-11-26}}</ref>。

===远征36/37===
[[File:Expedition_36_crew_portrait.jpg|thumb]]
北京时间2013年5月29日4时31分，载有尤尔奇欣、[[卡伦·尼贝里|卡伦·尼贝里]]和[[卢卡·帕尔米塔诺|卢卡·帕尔米塔诺]]的[[联盟TMA-09M|联盟TMA-09M]]载人飞船在拜科努尔发射场升空，29日2时30分成功与国际空间站自动对接。对接约两小时后，过渡舱门打开，宇航员顺利进入空间站<ref>{{cite web |title=俄飞船将三名宇航员送进空间站 |url=http://news.sina.com.cn/w/2013-05-30/000027261786.shtml |publisher=新浪新闻 |archiveurl=https://web.archive.org/web/20190419144917/http://news.sina.com.cn/w/2013-05-30/000027261786.shtml |archivedate=2019-04-19}}</ref>。

北京时间2013年6月24日21时32分，他和[[亚历山大·米苏尔金|米苏尔金]]开始太空行走任务，这是第169次对空间站外部的装配和维护任务，共持续了6时34分<ref>{{Cite web |url=http://www.nasa.gov/mission_pages/station/expeditions/expedition36/e36_062413_eva.html#.U4bD7yg3i_w |title=Cosmonauts Complete Spacewalk to Prepare Station for New Russian Lab |author=[[NASA|NASA]] |date=2013-06-24 |accessdate=2014-05-29}}</ref>。

北京时间2013年8月16日22时40分，他和米苏尔金穿着俄制[[海鹰航天服|“海鹰-MK”航天服]]，打开[[码头号对接舱|码头号对接舱]]舱门进入太空，开始了2013年的第三次舱外任务。这次[[舱外活动|舱外活动]]的主要是为12月与[[国际空间站|国际空间站]]对接的俄罗斯科学号多功能实验舱铺设电缆<ref>{{cite web |author=贺颖骏 |title=国际空间站俄宇航员实施今年第三次太空行走 |url=http://news.sciencenet.cn/htmlnews/2013/8/281497.shtm |publisher=[[新华社|新华社]] |archiveurl=https://web.archive.org/web/20160403081733/http://news.sciencenet.cn/htmlnews/2013/8/281497.shtm |archivedate=2016-04-03 |date=2013-08-21}}</ref>。

北京时间22日19时40分，他和米苏尔金打开舱门，进行了8月的第二次舱外任务。他们拆卸了[[星辰号服务舱|星辰号服务舱]]表面为太空实验“激光通信系统”而设置的机载[[激光|激光]]通信终端。他们还在星辰号服务舱表面安装了外设工位，并取回安装在[[探索号实验舱|探索号实验舱]]外的实验涂片<ref>{{cite web |author=贺颖骏 |title=俄两名宇航员一周内完成两次太空行走 |url=http://www.chinanews.com/mil/2013/08-23/5201196.shtml |publisher=[[新华网|新华网]] |archiveurl=https://web.archive.org/web/20190414111348/http://www.chinanews.com/mil/2013/08-23/5201196.shtml |archivedate=2019-04-14 |date=2013-08-23}}</ref>。

北京时间2013年11月11日10时50分左右，他和联盟TMA-09M队员乘坐飞船返回舱成功降落在哈萨克斯坦。之后，尤尔奇欣将曾在太空进行传递（由[[谢尔盖·梁赞斯基|梁赞斯基]]和[[奥列格·科托夫|科托夫]]完成）的[[索契冬奥会|索契冬奥会]]火炬递交给了组委会代表<ref>{{cite web |title=冬奧會火炬太空歸來 |url=http://hk.crntt.com/doc/1028/6/1/8/102861848.html?coluid=20&kindid=252&docid=102861848 |publisher=[[中國評論|中國評論]] |archiveurl=https://web.archive.org/web/20190419152728/http://hk.crntt.com/doc/1028/6/1/8/102861848.html?coluid=20&kindid=252&docid=102861848 |archivedate=2019-04-19 |date=2013-11-13}}</ref>。 

===远征51/52===
2016年12月1日，尤尔奇欣在[[新闻发布会|新闻发布会]]上表示，由于国际空间站俄方宇航员人数将减至2名，俄罗斯宇航员的工作量会加大，在休息日也执行任务<ref>{{cite web |author=杨牧 |title=因人数缩减 俄国际空间站宇航员休息日也将工作 |url=http://world.people.com.cn/m/n1/2016/1201/c1002-28917980.html |publisher=人民网 |archiveurl=https://web.archive.org/web/20190419141341/http://world.people.com.cn/m/n1/2016/1201/c1002-28917980.html |archivedate=2019-04-19 |date=2016-12-01}}</ref>。

北京时间2017年4月20日15时13分，[[联盟MS-04|联盟MS-04]]飞船搭载尤尔奇欣和美国宇航员[[杰克·费希尔|杰克·费希尔]]升空<ref>{{cite web |author=孔庆玲 |title=“联盟MS-04”飞船搭载俄美两国宇航员升空 |url=http://www.chinanews.com/gj/2017/04-20/8204619.shtml |publisher=[[中国新闻网|中国新闻网]] |accessdate=2019-04-19 |archiveurl=https://web.archive.org/web/20190301233001/http://www.chinanews.com/gj/2017/04-20/8204619.shtml |archivedate=2019-03-01}}</ref>。

北京时间2017年8月17日22时，他和[[谢尔盖·梁赞斯基|梁赞斯基]]穿着[[海鹰宇航服|海鹰-MKS宇航服]]进行了太空出舱行走，手动释放了5颗纳米卫星，其中包括一颗用于预测国际空间站运行情况的“TS530-镜子”卫星，一颗[[3D打印|3D打印]]卫星“Tomsk-[[托木斯克理工大学|TPU]]-120”<ref>{{cite web |title=俄首颗3D打印卫星已开始传输包括中文在内的11种语言信号 |url=http://sputniknews.cn/russia/201708181023399021/ |publisher=[[俄罗斯卫星通讯社|俄罗斯卫星通讯社]] |accessdate=2019-04-14 |date=2017-08-18}}</ref>，以及两颗用来庆祝首颗[[人造卫星|人造卫星]]发射60周年的“塔纽莎”卫星。整次太空行走共持续了7小时34分钟<ref>{{cite web |author=张艾京 |title=俄宇航员进行太空行走 手动释放五颗纳米卫星 |url=http://www.chinanews.com/gj/2017/08-18/8308065.shtml |publisher=[[中国新闻网|中国新闻网]] |accessdate=2019-04-14 |archiveurl=https://web.archive.org/web/20181111195920/http://www.chinanews.com/gj/2017/08-18/8308065.shtml |archivedate=2018-11-11 |date=2017-08-18}}</ref>。这是俄宇航员2017年的唯一一次太空行走<ref>{{cite web |title=俄宇航员太空出舱行走 手动释放5颗纳米卫星 |url=http://www.taikongmedia.com/Item/Show.asp?m=1&d=24530 |publisher=资格太空网 |archiveurl=https://web.archive.org/web/20190414133716/http://www.taikongmedia.com/Item/Show.asp?m=1&d=24530 |archivedate=2019-04-14 |date=2017-08-18}}</ref>。

北京时间2017年9月3日9时22分，尤尔奇欣、[[佩吉·惠特森|惠特森]]和费希尔乘坐联盟MS-04返回舱在哈萨克斯坦[[杰兹卡兹甘|杰兹卡兹甘]]市东南方148公里处成功着陆<ref>{{cite web |author1=杨磊 |title=“联盟MS-04”号飞船载俄美宇航员在哈萨克斯坦着陆 |url=http://news.cri.cn/20170903/a2bfb5fa-3a69-ff13-a605-8cac496e83ee.html |publisher=[[国际在线|国际在线]] |archiveurl=https://web.archive.org/web/20190419141108/http://news.cri.cn/inc/6f909b7f-3010-4f70-8165-a7710081f3e9.inc |archivedate=2019-04-19 |date=2017-09-03}}</ref>。
