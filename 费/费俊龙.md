'''费俊龙'''（{{bd|1965年|5月5日|catIdx=F费}}），[[中华人民共和国|中国]][[江苏省|江苏省]][[昆山市|昆山市]][[巴城镇|巴城镇]]东阳澄湖行政村费家浜自然村人。男，汉族。1982年6月通過[[中国人民解放军空军|空軍]]招飛入伍。[[中国航天员大队|中国航天员大队]][[特级航天员|特级航天员]]（曾任大队长），[[中国人民解放军少将|中国人民解放军少将]]。

费俊龙毕业后在[[空军第五航空学校|空军第五航空学校]]任飞行教员<ref name=xhw/>。1992年7月在一次試飛中險些發生意外，但憑其技術成功避過一難，因而成為特級飛行員。1998年加入宇航員集訓隊，隶属[[中国人民解放军总装备部|中国人民解放军总装备部]]，改为陆军。他亦是[[神舟五號|神舟五號]]進入強化訓練的五名宇航員之一。

2005年6月，入选神舟六号载人航天飞行乘组梯队成员。2005年10月12日与[[聂海胜|聂海胜]]搭乘[[神舟六号|神舟六号]]飞船进入太空并担任[[指令长|指令长]]，成为[[中国|中国]]第二批进入太空的[[宇航员|宇航员]]。

2011年6月，晋升为少将军衔<ref>{{cite web |url = http://news.163.com/11/0909/10/7DGMBIM700014JB6.html |title = 聂海胜费俊龙晋升少将 中国少将航天员增至3人 |publisher = 网易新闻|date=2011-09-09}}</ref>。2014年2月，任[[中国航天员科研训练中心|中国航天员中心]]副主任<ref name=xhw>{{cite web |url = http://www.xinhuanet.com/mil/2018-01/22/c_129796393.htm |title =费俊龙：我在太空连做4个“前滚翻”  |publisher =  新华网 |date=2018-01-22}}</ref>。

== 家庭 ==
費俊龍於1991年結婚，其妻亦在航校畢業。截止2005年10月，其妻和兒子费迪在北京居住。
