'''蘆洲線'''為[[臺北捷運|臺北捷運]]營運中的路線，屬於高運量系統，全線皆為地下化路線。蘆洲線屬於'''[[中和新蘆線|中和新蘆線]]'''，與[[新莊線|新莊線]]臺北市區段（「[[大橋頭站|大橋頭]]－[[古亭站|古亭]]」）及[[中和線|中和線]]直通營運。

== 簡介 ==
本線於2001年開始興建，全線皆完工通車。2010年11月3日本線與新莊線台北市區段（「[[忠孝新生站|忠孝新生]]－大橋頭」）同時完工，直通營運（「忠孝新生－[[蘆洲站|蘆洲]]」）。2012年1月5日新莊線「大橋頭－[[輔大站|輔大]]」段通車，營運模式增加「忠孝新生－輔大」，與與原本模式交替運行。2012年9月30日東門站啟用，中和線與新莊線銜接，營運模式改為「[[南勢角站|南勢角]]－輔大」及「南勢角－蘆洲」。全線車站均設有[[月台幕門|月台幕門]]，為台北捷運第一條全線裝設全覆式月台門的高運量路線。

路線自[[台北大橋|台北大橋]]下方與新莊線分叉，東起台北大橋西北側三重區三和路一段與環河北路交叉口，沿三重區三和路、蘆洲區中山一路轉三民路，至環河路前沿水湳溝迴轉，於蘆洲抽水站附近跨越水湳溝抵達[[蘆洲機廠|蘆洲機廠]]。沿線設[[三重國小站|三重國小]]、[[三和國中站|三和國中]]、[[徐匯中學站|徐匯中學]]、[[三民高中站|三民高中]]、[[蘆洲站|蘆洲]]等五站及16公頃的蘆洲機廠，全長6.4公里。潛盾隧道長9243公尺，另置線型公園一處長約800公尺。

本線於2012年初新莊線通車至輔大站後採用列車入站廣播提示，使用的範圍僅於「大橋頭－忠孝新生」重疊區間段，同年9月30日起延伸至中和線。2014年，除增加往南勢角的廣播外，本線也開始採用廣播提示。{{NoteTag|除蘆洲站外，各站與重疊區間的廣播提示為「1月台列車往蘆洲」、「2月台列車往南勢角」。}}

== 沿革 ==
*1994年9月17日：蘆洲線規劃路線奉行政院核定<ref>{{Cite web |url=http://www2.sdpo.dorts.gov.tw/paper560/final5/1.htm |title=第一章 規劃與核定過程 第三節 中和線規劃過程及核定經過 |access-date=2011-11-14 |archive-url=https://web.archive.org/web/20090708105410/http://www2.sdpo.dorts.gov.tw/paper560/final5/1.htm |archive-date=2009-07-08 |dead-url=yes }}</ref>。
*2001年12月3日：蘆洲線隨第二階段捷運網路開始興建<ref>{{cite news |author=謝佳雯 |title=捷運蘆洲線 今動工 |work=民生報 |date=2001-12-03 |page=A3版}}</ref>。
*2007年9月：蘆洲線開始進行鋪設軌道工程。
*2007年11月6日：新莊線與蘆洲線穿越淡水河段的隧道貫通<ref>{{cite news |author=陳燕模 |title=新莊蘆洲線 河底隧道貫通 |work=聯合報 |date=2007-11-07 |page=C1版}}</ref>。
*2008年9月25日：蘆洲線鋪設軌道作業完成，並且與新莊線台北市區段完成接軌作業。
*2009年4月1日：蘆洲線與新莊線台北市區段開始進行車輛動態測試。
*2009年8月22日：報載原先台北市捷運局官員曾口頭答應蘆洲線於2010年春節前通車，但因文湖線系統檢測出問題頻頻出錯，台北市政府捷運工程局回台北縣交通局公函「以2010年9月通車為目標」<ref>{{cite  web |title=捷運蘆洲支線 延明年9月通車－聯合新聞網|url=http://udn.com/NEWS/DOMESTIC/DOM2/5092572.shtml|accessdate=2009年8月22日}}</ref>。
* 2009年8月24日：捷運局針對2010年春節前通車跳票的訊息澄清：未答應於2010年春節前通車。行政院核定通車時程為2010年12月31日。目前捷運局正積極趕工，以期能在2010年9月通車<ref>{{cite  web|title=台北市捷運工程局：捷運局積極趲趕行政院列管捷運蘆洲支線工程成99年底前完工通車之目標(2009.08.24)|url=http://www2.dorts.gov.tw/news/dortssubnews.asp?id=2176|accessdate=2009年8月25日}}{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
* 2010年3月2日：捷運局表示蘆洲線可望在9月1日通車，並配合台北市舉辦[[2010年臺北國際花卉博覽會|花博會]]，同時會以機場捷運的電源作後備電源，並在11月協同新莊線試車至[[頭前庄站|頭前庄站]]<ref>{{cite  web |title=北捷蘆洲支線9月通車 新莊線試駛頭前庄站(2010.03.02)|url=http://udn.com/NEWS/LIFE/BREAKINGNEWS9/5449824.shtml|accessdate=2010年3月2日}}</ref>。
* 2010年7月：沿線所有車站完工。
* 2010年9月4日、9月5日：蘆洲線與新莊線市區段（「忠孝新生－大橋頭」）初勘。<ref>{{cite  web|title=捷運蘆洲支線9月3日和9月4日初勘 台北市政府捷運工程局|url=http://www.dorts.gov.tw/ct.asp?xItem=1235471&ctNode=16447&mp=115001|accessdate=2010年9月3日|deadurl=yes|archiveurl=https://web.archive.org/web/20120304132256/http://www.dorts.gov.tw/ct.asp?xItem=1235471&ctNode=16447&mp=115001|archivedate=2012年3月4日|df=}}</ref>。
* 2010年9月28日：「履勘前須改善事項」改善完成，北市府報請交通部進行履勘。<ref>{{cite  web|title=捷運蘆洲線蘆洲站至新莊線忠孝新生站順利完成初勘 台北市政府捷運工程局|url=http://www.dorts.gov.tw/ct.asp?xItem=1280810&ctNode=16442&mp=115001|accessdate=2010年9月28日|archiveurl=https://web.archive.org/web/20130404012416/http://www.dorts.gov.tw/ct.asp?xItem=1280810&ctNode=16442&mp=115001|archivedate=2013年4月4日|deadurl=yes}}</ref>。
* 2010年10月16日、10月17日：蘆洲線與新莊線市區段（「忠孝新生－大橋頭」）履勘。
* 2010年11月3日：「蘆洲－忠孝新生」段開放載客營運，試乘期間持悠遊卡搭乘享有「蘆洲線區段免費」優惠。捷運營運路網突破 100 公里。
* 2010年12月3日：「蘆洲－忠孝新生」段開始收費，並開放自行車票證通行。
* 2012年9月30日：東門站啟用，「古亭－忠孝新生」段通車，與中和線直通營運。

==列車運行方式==
目前營運模式：
*「蘆洲－南勢角」列車：2012年9月30日[[新莊線|新莊線]][[東門站_(台北市)|東門站]]通車後，原新蘆線與中和線直通營運至今。

廢止的營運模式：
*「蘆洲－忠孝新生」列車：2010年11月蘆洲線與新莊線台北市區段的「大橋頭－忠孝新生」段通車，當時是以蘆洲線名義通車，直到2012年9月30日新莊線東門站通車後為止。

==紀念章==
旅客可以在服務台或諮詢服務台詢問站務員，站務員會拿出紀念印章供旅客使用蓋印。本線於2015年2月11日開始採用「車站專屬紀念章戳」，各站的紀念章如下：
*蘆洲站：以該站的公共藝術舞之羽、蘆葦與李氏古宅為主題。
*三民高中站：以當地的湧蓮寺為主題。
*徐匯中學站：以該站1號出口、徐匯中學與碧華布街為主題。
*三和國中站：以義天宮、貓咪與永盛公園的花為主題。
*三重國小站：以三重國小、三和夜市為主題。
