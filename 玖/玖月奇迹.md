'''玖月奇迹'''，中国器乐演唱组合，以[[双排键|双排键]]和流行、美声、民族、古典风格相糅合，<ref name="multipleJD11 ">{{cite web |url=http://epaper.jinghua.cn/html/2011-07/30/content_685172.htm |title=玖月奇迹8月首次开唱 |accessdate=2011-07-30 |work= |publisher=京东网 |language=zh-hans }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>出道于[[央视|央视]]《[[星光大道|星光大道]]》并且获得冠军<ref name="multiplexlw65 " />，成員為[[王小玮|王小玮]]、[[王小海_(歌手)|王小海]]。

== 名称由来 ==
玖月奇迹的男主唱[[王小海_(歌手)|王小海]]说：“我和小玮从第一次相遇，就是奇迹。在[[沈阳|沈阳]]的一个[[KTV|KTV]]里唱歌，她唱[[民歌|民歌]]特别好听，所以我印象很深刻。我是个‘[[麦霸|麦霸]]’，啥歌都唱，她对我印象也很深。我们共同的一个朋友带我去看她的[[双排键|双排键]]音乐会，一下子给我震住了。”王小海和[[王小玮|王小玮]]两人第一次见面是九月，第二次见面是九月，第三次见面也是九月，三个九月，让他们相信有奇迹发生，于是随即组成了音乐组合玖月奇迹。<ref name="multipleab3 " >{{cite web |url=http://www.cnr.cn/gx/gz/201102/t20110223_507712318.html |title=玖月奇迹成名路：音乐硕士携天赋奇才成就春晚神器 |accessdate=2011-02-23 |work=|publisher=中国广播网 |language=zh-hans}}</ref>
