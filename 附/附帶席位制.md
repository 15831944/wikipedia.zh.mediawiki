{{選舉制度}}
'''附帶席位制'''（''Additional Member System''，AMS），又稱「額外席位制」，是[[單一選區兩票制|單一選區兩票制]]的一種，與[[聯立制|聯立制]]、[[並立制|並立制]]相比，也是接近[[比例代表制|比例代表制]]的選舉制度。

==概要==
*原指德國聯立制、後來日本並立制的性質，即除單一選區外，另有比例代表制席次。換言之，就是單一選區兩票制。
*英國用來專指在[[威爾斯|威爾斯]]、[[蘇格蘭|蘇格蘭]]、[[倫敦|倫敦]]實行的議會選舉制度；日本也有同樣選舉改革方案，稱為{{nihongo|「小選區比例代表連用制」|小選挙区比例代表連用制}}。<ref name="Additional-member system: Politics">{{cite web |title=Additional-member system: Politics|url=http://www.britannica.com/topic/additional-member-system|publisher=[[Encyclopædia_Britannica|Encyclopædia Britannica]]|accessdate= 24 Mar 2016}}</ref><ref name="Elections in Wales">{{cite web |title=Elections in Wales |url=http://blogs.cardiff.ac.uk/electionsinwales/2013/11/05/electoral-systems-in-wales-3-ams/|publisher=[[Cardiff_University|Cardiff University]]}}</ref><ref name="Electoral Reform and Voting Systems">{{cite web |title=Electoral Reform and Voting Systems |url=http://www.politics.co.uk/reference/electoral-reform-and-voting-systems|publisher=Politics.co.uk}}</ref><ref name="Electoral Reform and Voting Systems: MMP">{{cite web |title=Mixed Member Proportional (MMP) System |url=http://www.mec.org.mw/files/column%2040-%20Mixed%20member%20proprtional.pdf}}</ref>

==席次分配==
參考[[威爾斯國民議會|威爾斯國民議會]]官網範例，各區域有四個附加席位名額，假設該區域分成八個選區、四個政黨參選，依[[漢狄法|頓特法]]除以「單一選區當選席次＋1」，從當選席次最少開始分配：

如單一選區當選席次相同，政黨票較高者優先。由於從單一選區當選席次最少者開始分配，同德國聯立制有利於小黨的優點，卻不會產生[[超額席次|超額席次]]，兼具日本並立制優點。

==採用國家==
[[英國|英國]]目前在地方議會選舉使用，國會下議院選舉仍維持[[單一選區制|單一選區制]]。

[[日本|日本]]曾於1993年由民間提出此種選舉改革方案，但執政[[自由民主党_(日本)|自民黨]]想維持[[一黨獨大|一黨獨大]]的優勢，最終採用[[並立制|並立制]]；[[民主党_(日本)|民主黨]]2009年執政後，2012年6月18日提出選舉改革方案，在並立制中導入「連用制」（即附帶席位制），隨著同年12月16日大選[[自由民主党_(日本)|自民黨]]重新執政而不了了之。
