'''附例特檢隊'''（俗稱'''紅帽子'''；[[英文|英文]]：'''By-laws Inspection Unit'''）於2009年4月8日成立，隸屬於[[港鐵|港鐵]]，主要責任為在[[巡邏|巡邏]]沿線[[鐵路站|鐵路站]]、執行《[[香港法例|香港法例]]》第556B章《[[香港鐵路條例|香港鐵路條例]]》（當中主要打擊[[逃票|逃票]]及防止乘客攜帶超重行李及[[危險品|危險品]]進入港鐵範圍）、[[人群管制|人群管制]]及協助身體不適的乘客等等。與此同時，聯同由退役[[啹喀兵|啹喀兵]]組成的單位和輔助保安人員合作，共同協助打擊罪案。成立至今，附例特檢隊主要駐守於[[東鐵線|東鐵線]][[上水站|上水站]]、[[粉嶺站|粉嶺站]]、[[羅湖站_(香港)|羅湖站]]及[[落馬洲站|落馬洲站]]，[[管理|管理]]鐵路站的[[人流|人流]]<ref>[http://hk.apple.nextmedia.com/news/art/20090409/12619847 港鐵特檢隊打擊罪案逃票] 《蘋果日報》 2009年4月9日</ref>。

== 組織 ==
附例特檢隊由一名隊長所領導，下由數名隊目領導每組約10名人員；人員全部均為[[退休|退休]][[香港紀律部隊|香港紀律部隊]]（包括[[香港警務處|香港警務處]]及[[懲教署|懲教署]]等）人員<ref>[http://www.singpao.com/xw/gat/201502/t20150207_547814.html 特檢隊員曾任職紀律部隊] {{webarchive|url=https://web.archive.org/web/20150208155817/http://www.singpao.com/xw/gat/201502/t20150207_547814.html |date=2015-02-08 }} 《成報》 2015年2月7日</ref>，部份為現任[[香港輔助警察隊|香港輔助警察隊]]人員。人員均穿著灰色制服和佩戴紅色[[貝雷帽|貝雷帽]]。

== 歷史 ==
鑑於2009年首3個月在港鐵範圍內發生了近13,000宗逃票案件，當中逾6成發生於[[東鐵線|東鐵線]]；港鐵遂於同年4月8日以臨時的形式成立附例特檢隊，主力打擊逃票<ref>[http://202.55.1.84/news/09/04/08/_IN-1062534.htm 港鐵路成立特檢隊打擊逃票]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }} 《大公報》 2009年4月8日</ref><ref>[http://www.epochtimes.com/b5/9/4/9/n2489120.htm 港鐵設特檢隊全力打擊逃票] 《大紀元》 2009年4月9日</ref><ref>[http://www.info.gov.hk/gia/general/201202/15/P201202150167.htm 立法會八題︰港鐵車廂內滋擾行為] 《新聞公報》2012年2月15日</ref><ref>[http://www.epochtimes.com.hk/b5/9/4/9/98410.htm 港鐵設特檢隊全力打擊逃票] 《大紀元時報》 2009年4月9日</ref>；由[[退休|退休]][[警署警長|警署警長]]黃成根擔任隊長（曾經駐守於[[刑事情報科|刑事情報科]]、[[衝鋒隊_(香港)|衝鋒隊]]及[[防止罪案科|防止罪案科]]等等多個[[香港警務處架構|香港警察單位]]），試行單位運作半年<ref>[http://www.hkcd.com.hk/content/2009-04/09/content_2286711.htm 港鐵設特檢隊嚴打逃票] {{webarchive|url=https://web.archive.org/web/20150209024144/http://www.hkcd.com.hk/content/2009-04/09/content_2286711.htm |date=2015-02-09 }} 《香港商報》 2009年4月9日</ref><ref>[http://202.55.1.84/news/09/04/09/images_0708-1062811.htm 港鐵紅帽子「特檢隊」遏逃票] {{webarchive|url=https://web.archive.org/web/20150209024751/http://202.55.1.84/news/09/04/09/images_0708-1062811.htm |date=2015-02-09 }} 《大公報》 2009年4月9日</ref>。於成立一年多後，港鐵檢討成效，顯示逃票問題有顯著的改善，附例特檢隊遂恆常運作。2010年年初的逃票案件比較2009年年初同期大跌近8成，減至了逾1,900宗<ref>[http://orientaldaily.on.cc/cnt/news/20100704/00176_033.html 特檢隊收效 港鐵逃票減八成] 《東方日報》 2010年7月4日</ref>，至2013年減少至6,780宗。而乘客暴力對待港鐵職員之情況，則從2009年的68宗減少至2013年的22宗。

2009年9月16日，一名休班附例特檢隊隊員在[[羅湖口岸|羅湖口岸]]過關後被水貨攜帶者認出而遭受言語暴力及糾黨圍毆，在[[深圳海關|深圳海關]]人員保護下才告安全，不過仍然被數十人在羅湖關口上遭到圍堵，並且不停叫囂。事件引發[[香港鐵路員工總會|香港鐵路員工總會]]促請[[香港警察|警察]]積極介入及協助港鐵維持鐵路站內之公共秩序，否則擔心職員被毆打之情況繼續發生<ref>[http://orientaldaily.on.cc/cnt/news/20090919/00176_045.html 水貨黨圍毆港鐵特檢隊] 《東方日報》 2009年9月19日</ref>。

2012年起，附例特檢隊在[[羅湖站_(香港)|羅湖站]]及[[上水站|上水站]]站內外設置[[鐵馬|鐵馬]]以疏導輪候入閘人流，並且協助[[保安員|保安員]]截查涉嫌從事[[水貨|水貨]]活動的[[乘客|乘客]]及執行行李重量限制事務<ref>[http://paper.wenweipo.com/2012/09/09/HK1209090033.htm 港鐵特檢隊防逼爆站] 《文匯報》 2012年9月9日</ref> 。鑑於[[新界北|新界北]]受到大量[[中國大陸|中國大陸]][[水貨|水貨]]攜帶者的影響，及配合東鐵線推行行李重量限制試驗，港鐵在[[粉嶺站|粉嶺站]]以北的沿線鐵路站設立電子地磅限制行李在32公斤或者以下，及增加附例特檢隊編制，由29名增加至39名，輔助保安人員編制則由74名增加至92名<ref>[http://hk.news.yahoo.com/港鐵-特檢隊-增人手打擊水客-220255170.html 港鐵「特檢隊」增人手打擊水客] 《明報》 2012年10月18日</ref>。2013年2月，電子地磅設立擴展至[[大埔墟站|大埔墟站]]及[[火炭站|火炭站]]，並且再限制行李至23公斤或者以下。

臨近[[農曆新年_(華人)|農曆新年]]，不少水貨攜帶者均辦理[[年貨|年貨]]，上水站又再成為重災區，其內外每日聚集著大量水貨攜帶者。而每逢人流高峰時段，均容易發生乘客糾紛、肢體衝突，以至襲擊港鐵職員（亦包括多宗涉嫌針對附例特檢隊隊員）的情況<ref>[http://hk.apple.nextmedia.com/news/art/20090503/12711200 港鐵特檢隊被20多人包圍] 《蘋果日報》 2009年5月3日</ref>。有鑑於此，附例特檢隊引入3部隨身攝錄器，於2015年2月9日起在[[上水站|上水站]]試用半年<ref>[https://www.mtr.com.hk/archive/corporate/en/press_release/PR-15-011-C.pdf 農曆新年臨近，港鐵上水站加強人流管理措施]</ref>，每逢遇有衝突情況即時攝錄，期望可以起震攝用途，並且為隊員、其他職員及乘客提供額外保障，計劃於半年後再作檢討，決定是否永久採用<ref>[http://news.tvb.com/local/54d47c8a6db28c3423000005/ 港鐵於上水站加強管理 遇衝突用攝錄機拍攝] 無線新聞 2015年2月6日</ref>。與此同時，附例特檢隊亦將會增加[[人力資源|人力資源]]巡查車廂，若果發現乘客攜帶大型行李，或會要求有關人物離開車廂，以將其所攜帶的行李接受電子地磅測量<ref>[http://orientaldaily.on.cc/cnt/news/20150207/00176_022.html 港鐵特檢隊隨身攝錄衝突情況] 《東方日報》 2015年2月7日</ref>。

== 入職要求 ==
=== 隊目 ===
應聘成為附例特檢隊隊目者需要持有[[香港中學會考|香港中學會考]]或者[[香港中學文憑|香港中學文憑]]學歷，並且擁有兩年執行《香港鐵路附例》之工作經驗或者在[[香港紀律部隊|香港紀律部隊]]服務之資歷<ref>[http://hk.jobsdb.com/HK/EN/Search/JobAdSingleDetail?jobsIdList=100003003426648 附例特檢隊隊目]</ref>。

== 訓練 ==
附例特檢隊隊員於執勤前均需要接受一連串的訓練，包括熟讀《[[香港鐵路附例|香港鐵路附例]]》、了解[[港鐵|港鐵]]架構及運作<ref>[http://paper.wenweipo.com/2009/04/09/HK0904090016.htm 港鐵聘退休警長 組特檢隊捉逃票] 《文匯報》 2009年4月9日</ref>，並且熟識[[鐵路站|鐵路站]]及[[列車|列車]][[車廂|車廂]]環境等。
