[[硫|硫]]有著大量的同素異形體，其數量只是僅次於[[碳的同素异形体|碳]]。<ref name = "Greenwood">{{Greenwood&Earnshaw|pages = 645–662}}</ref>
硫在自然界中最常見的形式是黃色的[[正交晶系|正交晶系]]α-硫，其中包含S<sub>8</sub>的皺褶環。

==氣態同素異形體==
===二硫S<sub>2</sub>===
二硫为硫的双原子分子。在720[[°C|°C]]，硫主要以二硫存在。在530°C、低壓(1[[毫米水銀|毫米水銀]])的環境中，硫蒸汽佔99%是二硫。火焰中產生的S<sub>2</sub>分子使硫燃燒時呈藍色。<ref name = "Greenwood"/>

===三硫S<sub>3</sub>===
三硫的彎曲结构类似[[臭氧|臭氧]]，為一櫻桃紅色的氣體。<ref name = "Greenwood"/>在440[[°C|°C]]、10[[毫米水銀|毫米水銀]]的環境，三硫組成了硫蒸汽的10%。<ref name = "Greenwood"/>
===四硫S<sub>4</sub>===
在硫蒸汽存在。根據理論計算的最新觀點認為四硫有著環狀結構。<ref name = "wong">{{cite journal|title=Structure and spectra of tetrasulfur S<sub>4</sub> – an ab initio MO study|journal=Chemical Physics Letters|volume=379|issue=1-2|year=2003|pages=162–169|doi=10.1016/j.cplett.2003.08.026|author=Ming Wah Wong, Ralf Steudel|bibcode = 2003CPL...379..162W }}</ref>

===環五硫S<sub>5</sub>===
尚未分離，只在硫蒸汽偵測到。<ref name ="Steudel">{{cite book|title=Elemental sulfur and sulfur-rich compounds I (Topics in current chemistry)|editor=R. Steudel|year=2004|publisher=Springer|isbn=3540401911|url=http://books.google.com/?id=9PaU9qnETfEC&printsec=frontcover}}</ref>


==固體環狀結構==
===环六硫S<sub>6</sub>===
S<sub>6</sub>的晶形為菱面體，為橙红色固體，首次透過下列反應发现于1891年。

H<sub>2</sub>S<sub>4</sub> + S<sub>2</sub>Cl<sub>2</sub> → cyclo-S<sub>6</sub> + 2 HCl (在稀釋的[[乙醚|乙醚]]溶液中)<ref name = "Greenwood"/>
===环-S<sub>6</sub>.环-S<sub>10</sub>加合物===
這是由含S<sub>6</sub>和S<sub>10</sub>的[[二硫化碳|二硫化碳]]中之溶液製備所得。
===环七硫S<sub>7</sub>===
為亮黄色固体，有α、β、γ、δ四种结构。環七硫的環有著不尋常幅度的鍵長(199.3–218.1 pm)，因此被認為是最不穩定的同素異形體。
===S<sub>8</sub>===
[[File:Molecular_Sulfur_(S8)_V.1.svg|thumb]]]]
====斜方硫（菱形硫）(α-硫)====
α-硫是硫自然界中最常見的形式，<ref name = "Greenwood"/>其純淨時的顏色是黃綠色(市面上出售的硫因為有著微量的环七硫而呈現更黃的顏色)。α-硫實際上不溶於水，導熱性能差，為一良好的電絕緣體。此是硫由二硫化碳結晶而得之緊密的黃色晶體，融點112.8度。
====單斜硫(β-硫)====
融化硫於部分凝固後，倒出多餘液體，剩下無數之針形晶體即為單斜硫，融點119.2度。

====彈性硫====
{{main|彈性硫}}
為沸騰之硫注入冷水所得之軟黏體，有彈性。

====γ硫====
发现于1890年，为融化硫由二硫化碳缓慢結晶而成。
[[File:Dodecasulfur_3D.png|120px]]]]
===S<sub>n</sub>,(n = 9–15, 18, 20)===
這些同素異形體是由多種方法合成，例如
可以透过[[二氯化硫|二氯化硫]](S<sub>n-m</sub>Cl<sub>2</sub>)和H<sub>2</sub>S<sub>m</sub>反應，即

S<sub>n−m</sub>Cl<sub>2</sub> + H<sub>2</sub>S<sub>m</sub> → cyclo-S<sub>n</sub><ref name = "Greenwood"/>

制取。S<sub>12</sub>、S<sub>18</sub>、S<sub>20</sub>可以由S<sub>8</sub>製得，其中S<sub>12</sub>是繼S<sub>8</sub>後最穩定的環狀同素異形體。S<sub>9</sub>有4種形式，S<sub>18</sub>則有2種。

==固體連鎖硫結構==
生產純淨的連鎖硫已被證明是極其困難的。
===ψ硫===
亦稱「纖維硫」或ω1-硫<ref name ="Steudel"/>，其密度為2.01 g·cm<sup>−3</sup>，會在其融點104°C附近分解。
===χ硫===
亦稱ω2-硫。

==連續硫形式==
===無定形硫===
這是硫在160°C以上時融解的[[淬火|淬火]]產物。
===ω硫===
为ψ硫和χ硫的混合物。ω硫可被可用作橡膠的[[硫化|硫化]]。<ref name ="Steudel"/>
===λ硫===
此為融化後熔融的硫的名字，冷卻此主要給出β-硫。<ref name ="Steudel"/>
===μ硫===
===π硫===
深色液體，由λ硫經[[淬火|淬火]]冷却后形成。含S<sub>n</sub>環的混合物。<ref name = "Wiberg&Holleman">{{cite book|title=Inorganic Chemistry|publisher=Elsevier|year=2001|isbn=0-12-352651-5|author=Egon Wiberg, Arnold Frederick Holleman|url=http://books.google.com/?id=vEwj1WZKThEC&printsec=frontcover}}</ref>
===S<sub>∞</sub>===
把硫加热至170°C可形成S<sub>∞</sub>。
