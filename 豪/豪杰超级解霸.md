'''豪杰超级解霸'''为北京世纪豪杰计算机技术有限公司推出的一款支持[[Windows|Windows]]平台的影音[[媒體播放器|媒體播放器]]，2000年左右在中国大陆的视频播放领域具有领导性地位<ref>[https://www.ixueshu.com/document/01506cc88a752fb2318947a18e7f9386.html 范岳. 豪杰超级解霸2001. 电子测试, 2001(8):69-69.]</ref>。2005~2007年左右，豪杰公司将该产品出售给[[暴风影音|暴风影音]]所属的暴风网际公司{{R|finance}}<ref>[http://tech.sina.com.cn/s/s/2008-03-27/2219616876.shtml 暴风影音完全消化豪杰超级解霸技术]  新浪网转IT168.com 2008年03月27日 </ref>。暴风影音公司2010年曾推出新款“超级解霸2010”<ref>[https://www.ixueshu.com/document/b7829e07badea040318947a18e7f9386.html 王者能否归来? 新超级解霸抢先尝_爱学术]</ref>，但由于市场饱和、缺乏特色等原因而未获得足够关注<ref>[http://tech.cnr.cn/jdxw/201304/t20130403_512283523.shtml 前浪死在沙滩上 祭十大已经逝去的软件]</ref>。

2006年的报道称，超级解霸累计客户使用量超过1亿，拥有3000多万忠实用户，十年来在中国国产视频软件市场中鹤立鸡群<ref>[https://www.ixueshu.com/document/9b399c3d78d8416b318947a18e7f9386.html 王子居. 超级解霸3000万用户的价值. 投资与合作, 2006(11):48-49.]</ref>。

== 主要功能 ==
* 支持近150种主流媒体文件的播放，包括[[DAT|DAT]]、[[VOB|VOB]]、[[AVI|AVI]]、[[WMV|WMV]]、[[MOV|MOV]]、[[RMVB|RMVB]]等。
* 支持AC-3硬解码系统及SPDIF输出技术，可自由控制声道。
* 支持显示[[字幕|字幕]]，并可调整颜色、大小等。
* 支持书签功能，可断点续播。

== 独创技术 ==

* DirectCDROM、DirectDVDROM技术
* 三线滤波
* 四分滤波
* DVB/DAB（digital video/audio broadcast）
* 实时视频流解压技术
* 网络“猝发传送技术”<ref name="独创技术_官方网站">{{cite web |url=http://www.haojie.cn/index3500-1.htm |title=存档副本 |accessdate=2013-02-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20130129120916/http://www.haojie.cn/index3500-1.htm |archivedate=2013-01-29 }}</ref>
