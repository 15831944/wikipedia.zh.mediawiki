'''顧祝同'''（{{bd|1893年|1月9日|1987年|1月17日|catIdx=Gu顧}}），[[表字|字]]'''墨三'''，[[中華民國陸軍|中華民國陸軍]][[一級上將|一級上將]]，[[江蘇省|江蘇省]][[涟水县|涟水縣]]人<ref>{{cite web|url=http://museum.mnd.gov.tw/Publish.aspx?cnid=1484&p=12202|title=顧祝同將軍生平|publisher=國軍歷史文物館|language=zh-tw}}</ref>，曾任[[江蘇省政府|江蘇省政府]]主席、[[中華民國陸軍司令|陸軍總司令]]、[[中華民國參謀總長|參謀總長]]、[[中華民國國防部部長|國防部長]]。

== 早年 ==
顧祝同十九歲入讀江蘇陸軍小學，[[辛亥革命|辛亥革命]]時陸軍小學停課，顧祝同參加推翻滿清的革命。

1912年加入[[中國國民黨|國民黨]]，後畢業於武昌預備軍官學校，再升讀[[保定陸軍軍官學校|保定陸軍軍官學校]]第六期步科畢業。1922年赴[[粵|粵]]，加入粵軍二軍任[[參謀|參謀]]。

== 四處征戰 ==
[[中華民國陸軍軍官學校|黃埔軍校]]成立後，調任軍校戰術教官，管理部主任。1925年東征[[陈炯明|陳炯明]]時任教導團營長。[[國民革命軍北伐|國民革命軍北伐]]時升任[[師長|師長]]、[[軍長|軍長]]。之後參加[[中原大戰|中原大戰]]，任十六路軍總指揮。

1931年，任[[國民政府|國民政府]][[南京市|南京]]警衛軍軍長。12月15日，[[蔣中正|蔣中正]]召開第四十九次國務會議，任命顧祝同為[[江蘇省政府|江蘇省政府]]主席。<ref>上海，《[[中央日報|中央日報]]》，1931年12月16日，第一張第四版</ref>

1933年，指揮第四、第五次圍剿[[中国工农红军|中國工農紅軍]]。

1935年晉升[[上將|上將]]、主管[[四川省_(中華民國)|四川]]、[[西康省|西康]]、[[貴州省_(中華民國)|貴州]]軍政。

== 西安事變 ==
1937年1月1日，蔣介石在南京召見軍事委員會辦公廳主任[[朱培德|朱培德]]、重慶行營主任顧祝同、江西省政府主席[[熊式輝|熊式輝]]、甘肅綏靖公署主任[[朱紹良|朱紹良]]、參謀本部廳長[[林蔚|林蔚]]等會商，決定「以政治為主、軍事為從方略，以解決西北問題」，並令顧祝同駐節潼關，指揮陝西、甘肅軍事<ref name="民">{{cite book |editors=李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編 |title=《中華民國史大事記》 |location=[[北京|北京]] |publisher=[[中華書局|中華書局]] |year=2011 }}</ref>{{rp|5327}}。1月6日，[[軍事委員會委員長|軍事委員會委員長]]西北行營主任顧祝同飛洛陽轉陝西，辦理陝西、甘肅善後事宜；[[何應欽|何應欽]]據蔣電令，密電[[劉峙|劉峙]]、顧祝同，補充《陝甘軍事善後方案》中駐地區分辦法，「[[東北軍|東北軍]]全部應先調集甘境，歸[[王樹常|王樹常]]節制指揮，並負責整理」；第十七路軍除准在西安酌留一部外，餘照方案辦理<ref name=民/>{{rp|5331}}。1月10日，蔣派顧祝同為西安國共兩黨談判中國國民黨方面代表，後又增派[[张冲_(1900年)|張沖]]與[[賀衷寒|賀衷寒]]二人；顧祝同至[[潼關|潼關]]，與中共代表[[周恩來|周恩來]]、[[劉伯承|劉伯承]]會晤<ref name=民/>{{rp|5335}}。1月22日，蔣電[[楊虎城|楊虎城]]，要其直接與顧祝同商量善後問題；顧祝同派[[劉震東|劉震東]]自潼關至西安，與東北軍將領洽談<ref name=民 />{{rp|5345-5346}}。1月24日，楊虎城電覆蔣，同意派代表直接與顧祝同商談<ref name=民/>{{rp|5347}}。1月28日，東北軍代表與顧祝同達成協議：東北軍在七天內將渭河南岸部隊撤到北岸；蔣指示顧祝同，對西安方面上述要求「酌允辦理」，同意部隊撤到渭北後，辦理張學良復權手續<ref name=民/>{{rp|5350-5351}}。1月30日，周恩來經由何柱向顧祝同提出，中共要求派代表參加潼關談判<ref name=民/>{{rp|5352}}。1月31日，蔣電顧祝同稱，紅軍駐地陝北，南京每月給紅軍20萬元至30萬元經費；紅軍代表[[李克農|李克農]]與顧祝同談判，顧祝同同意紅軍在西市設立聯絡處，以第十七路軍為掩護；東北軍代表米春霖、謝珂在潼關與顧祝同、陳誠等商談<ref name=民/>{{rp|5354}}。

1937年2月6日，中央軍進抵西安接防，軍事委員會委員長西安行營主任顧祝同暫住[[臨潼|臨潼]]<ref name=民/>{{rp|5359}}。2月8日，西安行營主任兼第一集團軍總司令顧祝同進入西安，蔣電顧祝同：「安置東北軍辦法，總以調駐豫、鄂、皖省區為唯一方針」<ref name=民/>{{rp|5359}}。2月9日，國共兩黨代表在西安舉行第一次正式會談，中國國民黨代表為顧祝同、張沖、賀衷寒，中國共產黨代表為周恩來、葉劍英、秦邦憲，談判歷時一個月；周恩來會見顧祝同，顧祝同再次表示同意紅軍在西安設辦事處，保證不迫害民眾團體，並稱蔣原定約周恩來2月10日赴杭州談判，因故推遲，命顧祝同先與周恩來會談<ref name=民/>{{rp|5360-5361}}。

1937年3月1日，周恩來與顧祝同、張沖談判，顧祝同同意先給紅軍接濟30萬元，並允為[[河西走廊|河西走廊]]及陝南紅軍送款<ref name=民/>{{rp|5375}}。3月5日，南京給顧祝同、張沖覆電稱，仍堅持紅軍改編為3個師9個團；顧、張再聯名電請<ref name=民/>{{rp|5379}}。3月6日，南京電覆顧祝同、張沖，同意紅軍改編為12個團，但仍堅持只編為3個師；張沖向周恩來提議，按國防師編制，紅軍編為3個師6個旅12個團，每師可編炮兵、交通、特務各1營，每師1.2萬人，加上3個師之上總指揮部4,000人，照此3個師已達4萬人，周恩來建議中共中央接受此項改編條件<ref name=民/>{{rp|5379}}。3月10日，蔣令[[賀國光|賀國光]]為[[軍事委員會委員長|軍事委員會委員長]]重慶行營副主任，在行營主任顧祝同未返四川前，暫代主任<ref name=民/>{{rp|5382}}。3月30日，周恩來攜帶與蔣聯繫之電報密碼由南京飛抵西安，與顧祝同談判接濟紅軍給養事項<ref name=民/>{{rp|5393}}。

1937年5月12日，國民政府令：貴州省政府主席顧祝同因公赴陝，未回任以前，派薜岳為該省政府委員兼代省政府主席<ref name=民/>{{rp|5423}}。

== 抗戰及內戰 ==
{{main|中國抗日戰爭|重點進攻}}
1937年7月21日，中國國民黨中央政治委員會追認特派何應欽為川康軍事整理委員會主任委員，顧祝同、劉湘為副主任委員<ref name=民 />{{rp|5509}}。[[中国抗日战争|中國抗日戰爭]]時期，任[[抗日戰爭第三戰區|第三戰區]]司令，主持[[江蘇省_(中華民國)|江蘇]]、[[浙江省_(中華民國)|浙江]]、[[福建省_(中華民國)|福建]]、[[江西省_(中華民國)|江西]]、[[安徽省_(中華民國)|安徽]]東南五省抗戰長達七年。

1941年1月22日，顧祝同命令各縣政府、縣黨部在最短時間徹底肅清皖南新四軍零散人員<ref name=民 />{{rp|6466}}。[[皖南事变|新四軍事件]]時，顧祝同主要負責包圍[[新四军|新四军]]及扣押[[葉挺|葉挺]]。

1945年12月20日，國民政府任命程潛為軍事委員會委員長武漢行營主任，[[余漢謀|余漢謀]]為[[衢州|衢州]][[綏靖公署|綏靖公署]]主任，顧祝同為徐州綏靖公署主任，[[劉峙|劉峙]]為鄭州綏靖公署主任<ref name=民 />{{rp|7931-7932}}。

1946年，顧任[[中華民國陸軍司令|陸軍總司令]]，[[郑州市|鄭州]][[綏靖公署|綏靖公署]]主任，負責指揮進攻[[山東省_(中華民國)|山東]]中共[[軍隊|軍隊]]，遭遇[[孟良崮戰役|孟良崮戰役]]大敗（顧祝同、[[湯恩伯|湯恩伯]]、[[張靈甫|張靈甫]]、[[黃百韜|黃百韜]]、[[胡璉|胡璉]]指揮），精銳之國軍整編七十四師被殲。7月9日，顧祝同由瀋陽飛往長春，視察附近駐軍<ref name=民 />{{rp|8101}}。

1947年1月9日，蔣電令正在鄭州之陸軍總司令顧祝同，在[[隴海鐵路|隴海鐵路]]東段與山東南部，先集中主力對付陳毅一部，再肅清劉伯承部；稱此「實為堵共軍成敗之唯一關鍵」，「既定不易之方針」<ref name=民 />{{rp|8261}}。1月28日，蔣電示顧祝同，稱陳毅、劉伯承各部正分由東、西兩面對徐州發動鉗形攻勢<ref name=民 />{{rp|8273-8274}}。5月3日，蔣飛徐州、濟南視察，與顧祝同、[[王耀武|王耀武]]等研商作戰部署，次日返南京<ref name=民 />{{rp|8347}}。7月19日，蔣偕顧祝同飛開封視察軍事，部署魯西南戰事，決定抽調6個半旅組成第四兵團，由[[王仲廉|王仲廉]]指揮，另從山東抽調4個師，分別增援魯西南與[[津浦鐵路|津浦鐵路]]中段<ref name=民 />{{rp|8385}}。8月6日，蔣指示陸軍總司令顧祝同，要求加緊隴海鐵路防務部署<ref name=民 />{{rp|8394}}。

1948年5月19日，新任國防部參謀總長顧祝同正式就職視事<ref name=民 />{{rp|8605}}。

== 撤離中國大陸 ==
1949年1月25日，蔣中正希望顧祝同「電令[[李文|李文]]，指揮[[北平|北平]]中央各軍」<ref name="靜" >{{cite book |author=蔣經國 |chapter=〈危急存亡之秋〉 |title=《風雨中的寧靜》 |location=台北 |publisher=[[正中書局|正中書局]] }}</ref>{{rp|142}}，積極準備戰鬥。1月29日，國防部參謀總長顧祝同，奉蔣中正之命，下達《應付時局六要領》密令：一、認清中共企圖；二、中共八條「第一條所謂戰犯，意在侮辱我領袖，動搖我信心」，「須予嚴厲駁斥」；三、「軍事最近作戰重點，封鎖鞏固長江，並保衛西北、西南；第一線部隊宜加強戒備，切實演練殲滅敵人[[战术|戰術]]」；四、「後方整訓部隊及無作戰任務部隊，務須加緊補充訓練，以提高作戰精神及能力，期短期變成勁旅」；五、「對後方治安秩序應加維持，地方散匪及造謠生事、罷工、怠工等行為，應嚴加取締肅清」；六、「對部隊應特別注意激勵士氣」；2月4日，該通令轉發中為解放軍所繳獲<ref name=民 />{{rp|8796}}。2月12日，[[蔣經國|蔣經國]]奉蔣中正命電告顧祝同：建議其通知[[劉安祺|劉安祺]]將軍，「在未奉命令之前，暫勿撤離[[青島|青島]]」<ref name=靜 />{{rp|154}}。4月26日，蔣到達上海[[復興島|復興島]]，即召見[[徐永昌|徐永昌]]、顧祝同、[[湯恩伯|湯恩伯]]、[[周至柔|周至柔]]、[[桂永清|桂永清]]、[[石覺|石覺]]、[[陳大慶|陳大慶]]等陸、海、空高級將領及保密局長[[毛人鳳|毛人鳳]]、上海市長[[陳良|陳良]]等，聽取上海防務報告，並指示方略<ref name=民 />{{rp|8893}}。

8月23日下午2時，蔣自台北飛抵廣州，先行往訪[[李宗仁|李宗仁]]、[[閻錫山|閻錫山]]，繼召見[[劉安祺|劉安祺]]、顧祝同等，面詢軍事部署實情，指責顧祝同對廣州保衛戰之處置太不負責，命其立即改正，切勿再命劉安祺部北調為要<ref>{{cite book |title=《事略稿本》 |date=1949-08-23 |location=台北 |publisher=「國史館」藏 }}</ref>。10月8日，顧祝同向蔣中正電稱：「[[廣東|粵省]]西北與[[湖南|湘]]、[[貴州|黔]]軍事，已趨劣勢；請毅然復任總統，長駐西南」<ref name=靜 />{{rp|253}}。12月7日，特派顧祝同兼西南軍政長官<ref name="總245" >{{cite news |title=《總統府公報》第貳肆伍號 |editors=總統府第五局 |date=1949-12-31 }}</ref>。12月16日，[[西康省|西康省政府]]委員兼主席[[劉文輝|劉文輝]]違抗命令，應予撤職；所有違抗命令情節，着由西南軍政長官顧祝同查明呈候核辦<ref name=總245 />。

== 晚年 ==
1950年3月，赴台兼任代[[中華民國國防部|國防部長]]。1954年，晉任[[中華民國陸軍|陸軍]][[一級上將|一級上將]]。1959年6月，蔣介石以顧祝同為[[國家安全會議_(中華民國)|國防會議]]秘書長，[[彭孟緝|彭孟緝]]為參謀總長，[[羅列_(軍人)|羅列]]為陸軍總司令<ref name="表" >{{cite book |author=[[陳布雷|陳-{布}-雷]]等編著 |title=《蔣介石先生年表》 |location=台北 |publisher=[[傳記文學|傳記文學]]出版社 |date=1978-06-01 }}</ref>{{rp|94}}。1967年，調戰略顧問委員會副主任。1972年，任總統府戰略顧問。1987年，病逝[[台北|台北]]，享壽94歲。

== 親屬==
顧祝同共有子女十人：
* 长子：顧潮生
* 次子：[[顧福生|顧福生]]（知名畫家）
* 长女：顧開生
* 次女：顧惠生
* 三女：顧省生
* 四女：顧瑞生
* 五女：顧錦生
* 六女：顧永生
* 七女：顧珏生
* 八女：顧璉生

其侄兒[[顧安生|顧安生]]，為台灣知名新聞及電視工作者，曾長期擔任[[台視晚間新聞|台視晚間新聞]]主播，及[[臺灣電視事業股份有限公司|臺灣電視事業股份有限公司]]副總經理一職。
