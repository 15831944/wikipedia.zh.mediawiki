[[数学|数学]]上，一个'''射影空间'''可以被看作是通过[[向量空间|向量空间]]''V''的[[原點|原点]]的直线的集合。{{nowrap|1=''V'' = '''R'''<sup>2</sup>}}以及{{nowrap|1=''V'' = '''R'''<sup>3</sup>}}的射影空间分别为实[[射影直线|射影直线]]和实[[射影平面|射影平面]]，其中 '''R'''表示[[实数域|实数域]]，'''R'''<sup>2</sup>表示有序实数对，'''R'''<sup>3</sup>表示实有序三元组。
[[Category:数学|Category:数学]]
[[Category:空间|Category:空间]]

射影空间的概念与[[透视投影|透视投影]]有关。更确切地说，它与眼睛或照相机把3D场景投影到2D图像的方法有关。所有位于同一条投影直线（即与[[小孔成像|相机的入射瞳孔]]相交的"视线"）上的点被投影到同一个图像上的点。在这种情况下，向量空间为'''R'''<sup>3</sup>，相机的入射瞳孔位于原点，而射影空间与图像上的点对应。

==介绍==

[[File:Projektivna_rovina.png|thumb]]
如前文提到的，射影空间是一个把"平面直线相交于无穷远处"的描述进行形式化定义的几何对象。我们以下给出建构[[实射影平面|实射影平面]] '''P'''<sup>2</sup>('''R''')的细节。如下三种定义等价：

#'''R'''<sup>3</sup>中通过原点(0, 0, 0)的所有直线的集合。每条这样的直线与球心在原点、半径为1的[[球面|球面]]恰好在两个点相交，即{{nowrap|1=''P'' = (''x'', ''y'', ''z'')}}与其[[对跖点|对跖点]] {{nowrap|(−''x'', −''y'', −''z'')}}。
#'''P'''<sup>2</sup>('''R''')也可以被描述为球面''S''<sup>2</sup>上的点，其中每个点''P''与其对跖点不进行区分。例如，点{{nowrap|(1, 0, 0)}} (图中红色点)与点{{nowrap|(−1, 0, 0)}} (浅红色点，因渲染关系颜色偏黄)等同。
#最后，另一种等价定义是{{nowrap|'''R'''<sup>3</sup> ∖ {(0, 0, 0)<nowiki>}</nowiki>}}的[[等价类|等价类]]，即不包含原点的三维空间，其中两个点{{nowrap|1=''P'' = (''x'', ''y'', ''z'')}}与{{nowrap|1=''P''<sup>∗</sup> = (''x''<sup>∗</sup>, ''y''<sup>∗</sup>, ''z''<sup>∗</sup>)}}等价[[当且仅当|当且仅当]]存在非零实数''λ''使得{{nowrap|1=''P'' = ''λ''⋅''P''<sup>∗</sup>}}，即{{nowrap|1=''x'' = ''λx''<sup>∗</sup>}}，{{nowrap|1=''y'' = ''λy''<sup>∗</sup>}}，{{nowrap|1=''z'' = ''λz''<sup>∗</sup>}}。射影平面的元素的通常写法，即'''R'''<sup>3</sup>中非零点{{nowrap|(''x'', ''y'', ''z'')}}所对应的等价类，可以写成{{nowrap|1=[''x'' : ''y'' : ''z'']}}。

最后一个公式即为[[齐次坐标|齐次坐标]]。

在齐次坐标下，任意点{{nowrap|[''x'' : ''y'' : ''z'']}} ({{nowrap|''z'' ≠ 0}})等价于点{{nowrap|[''x/z'' : ''y/z'' : 1]}}。因此，射影平面可以分为两个不相交的集合：包含满足{{nowrap|''z'' ≠ 0}}的形如{{nowrap|1=[''x'' : ''y'' : ''z''] = [''x''/''z'' : ''y''/''z'' : 1]}}的点，以及形如{{nowrap|[''x'' : ''y'' : 0]}}的点。 后者可以再被类似地划分为两个不相交子集，即等价于{{nowrap|[''x''/''y'' : 1 : 0]}}的点集和形如{{nowrap|[''x'' : 0 : 0]}}的点集。在最后一种情况下，''x''必不为零，因为原点不属于'''P'''<sup>2</sup>('''R''')。这最后的一个集合即等价于点{{nowrap|[1 : 0 : 0]}}。
从几何上看，第一个子集与'''R'''<sup>2</sup>同构（我们将在后面看到，这不仅仅是集合意义上的同构，也是流形意义上的同构），位于图中黄色的上半球（不包含赤道；等价地也可以说是下半球）。第二个子集则与'''R'''<sup>1</sup>同构，对应于绿色半圆弧（不包含两个标出来的端点；等价地也可以说浅绿色半圆弧）。最后，剩下的即为红色点或与其等价的浅红色点。

==射影空间的公理定义==

'''射影空间'''''S''可以被定义为满足如下公理的一个集合''P''（点集合）与一个''P''的子集的集合''L''（直线集合）：<ref>{{harvnb|Beutelspacher|Rosenbaum|1998|loc= pgs. 6–7}}</ref>

* 每两个不同的点''p''和''q''恰好位于一条直线上。
* [[奥斯瓦尔德·维布伦|维布伦]]公理：<ref>亦被称为''维布伦-杨公理''或错误地称为[[帕施公理|帕施公理]] {{harv|Beutelspacher|Rosenbaum|1998|loc=pgs. 6–7}}. 帕施公理是关于试图在实射影空间引入序关系，而这并非维布伦-杨公理所关注的。</ref>如果''a'', ''b'', ''c'', ''d''为不同的点，并且通过''ab''和''cd''的直线相交，那么通过''ac''和''bd''的直线也相交。
* 任意直线上至少有3个点。
