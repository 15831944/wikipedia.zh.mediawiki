《'''帥到分手'''》是[[臺灣|臺灣]]歌手[[周湯豪|周湯豪]]的一首歌曲，收錄於他的第3張錄音室專輯《[[REAL_(周湯豪專輯)|REAL]]》中。這首歌曲于2016年9月30日由[[太陽娛樂文化|太陽娛樂文化]]正式發行為專輯的首發單曲。歌曲由周湯豪和[[崔惟楷|崔惟楷]]創作，并由周湯豪製作，是一首[[華語流行|華語流行]]歌曲，歌詞表達了現代男子在愛情上遇到的無言窘態。歌曲獲得了主流樂評的普遍好評，大部分評論讚賞了曲風的獨特性。歌曲在臺灣[[KKBOX|KKBOX]]、[[Omusic|Omusic]]和[[myMusic|myMusic]]的榜單均獲得冠軍。《帥到分手》曾作為[[三立電視|三立]][[三立電視戲劇節目列表#華人電視劇|華人電視劇]]《[[飛魚高校生|飛魚高校生]]》的片頭曲。

周湯豪在2016年[[華劇大賞|華劇大賞]]、Hito流行音樂獎演唱了《帥到分手》。歌曲的音樂錄影帶由鄺盛執導，周湯豪化身樂團主唱。影片展示了就算「拜金女」如何對其展現性感，周湯豪依舊不爲所動，暗示了他對拜金的厭惡。

在發行前張專輯《[[S.N.G|S.N.G]]》后，[[周湯豪|周湯豪]]創作了《[[REAL|REAL]]》共4年，他希望嘗試一些「新的、酷的、屌的、真的東西」<ref name="myMusic">{{cite web |url=https://www.mymusic.net.tw/ux/w/editOffice/news/show/2002 |title=NICK周湯豪－挑戰華語音樂窠臼 |publisher=myMusic |date=2016-12-30 |accessdate=2017-09-07 |archive-url=https://web.archive.org/web/20170910175048/https://www.mymusic.net.tw/ux/w/editOffice/news/show/2002 |archive-date=2017-09-10 |dead-url=no }}</ref>。他認爲之前以往的模式「十之八九都是在講難過、傷心、後悔，但感情有很多東西可以講，我想要講一些周遭真正發生的事情」<ref name="myMusic"/>。《帥到分手》是以周湯豪依照身邊朋友的故事為題材<ref name="nextmag">{{cite web|url=http://www.nextmag.com.tw/realtimenews/news/44049498 |title=周湯豪帥到分手還流血|publisher=[[壹周刊|壹周刊]] |date=2016-09-22 |accessdate=2017-09-07}}</ref>，而他説道歌曲標題并非在稱呼自己，是一個大家都會遇到的情況，「我去買東西，人家叫帥哥，是在叫我旁邊的人。」<ref name="ETNEWS">{{cite web|url=http://star.ettoday.net/news/812656?t=%E5%91%A8%E6%B9%AF%E8%B1%AA%E8%A7%A3%E9%87%8B%E3%80%8A%E5%B8%A5%E5%88%B0%E5%88%86%E6%89%8B%E3%80%8B%E3%80%80%E6%AD%8C%E8%A9%9E%E7%8E%84%E6%A9%9F%E9%81%93%E7%94%B7%E6%80%A7%E5%BF%83%E8%81%B2%E8%B6%85%E4%B8%AD%E8%82%AF|title=周湯豪解釋《帥到分手》　歌詞玄機道男性心聲超中肯|publisher=ETNEWS|date=2016-11-16|accessdate=2017-09-07|archive-url=https://web.archive.org/web/20170913231758/http://star.ettoday.net/news/812656?t=%E5%91%A8%E6%B9%AF%E8%B1%AA%E8%A7%A3%E9%87%8B%E3%80%8A%E5%B8%A5%E5%88%B0%E5%88%86%E6%89%8B%E3%80%8B%E3%80%80%E6%AD%8C%E8%A9%9E%E7%8E%84%E6%A9%9F%E9%81%93%E7%94%B7%E6%80%A7%E5%BF%83%E8%81%B2%E8%B6%85%E4%B8%AD%E8%82%AF|archive-date=2017-09-13|dead-url=no}}</ref>他認為現在帥哥只是一個代稱，泛指所有男性<ref name="ETNEWS"/>。

《帥到分手》由周湯豪和[[崔惟楷|崔惟楷]]創作，并由周湯豪製作<ref name="kkbox">{{cite web |url=https://www.kkbox.com/hk/tc/song/FtO00x8S0QH2tloJ2tloJ0XL-index.html |title=帥到分手 |publisher=[[KKBOX|KKBOX]] |accessdate=2017-09-07 |archive-url=https://web.archive.org/web/20170907213346/https://www.kkbox.com/hk/tc/song/FtO00x8S0QH2tloJ2tloJ0XL-index.html |archive-date=2017-09-07 |dead-url=no }}</ref>。歌曲是一首[[華語流行|華語流行]]歌曲，全長3分39秒<ref name="kkbox2">{{cite web |url=https://www.kkbox.com/hk/tc/album/uE4DCS-ZESYzC0F17tMi009H-index.html |title=真 |publisher=[[KKBOX|KKBOX]] |accessdate=2017-09-07 |archive-url=https://web.archive.org/web/20170912144608/https://www.kkbox.com/hk/tc/album/uE4DCS-ZESYzC0F17tMi009H-index.html |archive-date=2017-09-12 |dead-url=no }}</ref>。歌曲以[[降B小調|降B小調]]創作<ref name="dailly">{{cite web |url=https://dailly.cc/pianosheet-168947 |title=帥到分手 鋼琴譜 |publisher=Dailly |accessdate=2017-09-07 |archive-url=https://web.archive.org/web/20170912191242/https://dailly.cc/pianosheet-168947 |archive-date=2017-09-12 |dead-url=no }}</ref>，[[拍號|拍號]]為4/4拍<ref name="dailly"/>，每分鐘80拍<ref name="dailly"/>。歌曲以「犀利直白」的方式唱出現代男生愛情的價值觀和無言窘態<ref name="suneg">{{cite web |url=http://www.suneg.com/2016/09/30/%E5%91%A8%E6%B9%AF%E8%B1%AA-nickthereal-%E6%9C%80%E6%96%B0%E5%96%AE%E6%9B%B2%E3%80%8A%E5%B8%A5%E5%88%B0%E5%88%86%E6%89%8B%E3%80%8B%E6%95%B8%E4%BD%8D%E4%B8%8A%E6%9E%B6/ |title=周湯豪 NICKTHEREAL 最新單曲《帥到分手》數位上架 |publisher=[[太陽娛樂文化|太陽娛樂文化]] |accessdate=2017-09-07 |archive-url=https://web.archive.org/web/20170910220110/http://www.suneg.com/2016/09/30/%E5%91%A8%E6%B9%AF%E8%B1%AA-nickthereal-%E6%9C%80%E6%96%B0%E5%96%AE%E6%9B%B2%E3%80%8A%E5%B8%A5%E5%88%B0%E5%88%86%E6%89%8B%E3%80%8B%E6%95%B8%E4%BD%8D%E4%B8%8A%E6%9E%B6/ |archive-date=2017-09-10 |dead-url=no }}</ref>，其中歌詞唱道「我帥到分手，安全感不夠，你選擇向錢走，現實將我爆頭」，指出很多人還是著重在實際的生活，選擇向錢走，已經無關帥或不帥<ref name="ETNEWS"/>。

==發行==
《帥到分手》于2016年9月30日由[[太陽娛樂文化|太陽娛樂文化]]發行至臺灣的[[數位音樂下載|數位下載]]，並成爲專輯的首發單曲<ref name="suneg"/>。歌曲同時成爲三立華語電視劇《[[飛魚高校生|飛魚高校生]]》的片頭曲發行<ref name="setn">{{cite news|url=http://www.setn.com/News.aspx?NewsID=205993|title=華賞／周湯豪開場表演《飛魚高校生》片頭真的〈帥到分手〉|publisher=三立新聞|date=2016-12-10|accessdate=2017-07-12|archive-url=https://web.archive.org/web/20161211141737/http://www.setn.com/News.aspx?NewsID=205993|archive-date=2016-12-11|dead-url=no}}</ref>。
