《'''惡魔城 蘇生的水平'''》（{{lang|en|Castlevania: Resurrection}}）是[[科樂美|科樂美]]集團旗下遊戲部門所開發的[[惡魔城|惡魔城]]系列[[動作遊戲|動作遊戲]]，這是在[[Dreamcast|Dreamcast]]上的第一作，同時也是系列作中第三次3D化的作品。於2001年因[[Dreamcast|Dreamcast]]全球停產而中止開發，遊戲劇情更隨[[PlayStation_2|PS2]]《[[惡魔城_無罪的嘆息|惡魔城 無罪的嘆息]]》於2003年發行而被全盤否定<ref name="GamerGuides.com2015">{{cite book|author=GamerGuides.com|title=Castlevania: Lords of Shadows - Strategy Guide|url=https://books.google.com/books?id=QqrTCgAAQBAJ&pg=PT9|date=29 October 2015|publisher=Gamer Guides|isbn=978-1-62776-369-1|pages=9–11}}</ref><ref>{{cite book|author=Gianni Molinaro|title=Castlevania: Le manuscrit maudit|url=https://books.google.com/books?id=TqpXDwAAQBAJ&pg=PT80|date=2018-04-24|publisher=Third Editions|isbn=978-2-37784-044-1|page=80}}</ref>。

== 劇情 ==
（總結截至中止開發前[[科樂美|科樂美]]集團所公佈的劇情）<br>

公元1666年，貝爾蒙多一族因沒有能使用聖鞭「吸血鬼殺手」的成年人，因此沒有對抗德古拉的力量。黑暗女公爵乘機令德古拉復活。正義勢力（截至中止開發前[[科樂美|科樂美]]集團沒公佈身分）從過去和未來召喚了各一位貝爾蒙特家族的吸血鬼獵人，分別為開山始祖索妮亞‧貝爾蒙多和家族後裔維克托‧貝爾蒙多。

== 角色 ==
（截至中止開發前[[科樂美|科樂美]]集團所公佈的角色）<br>

; 索妮亞‧貝爾蒙多(ソニア・ベルモンド)
: 《[[惡魔城_～漆黑前奏曲～|惡魔城 ～漆黑前奏曲～]]》的主角。
: 本作設定為貝爾蒙多一族開山始祖。
: 被正義勢力召喚至1666年。
: 此角色的開山始祖設定被《[[惡魔城_無罪的嘆息|惡魔城 無罪的嘆息]]》的里昂‧貝爾蒙多全盤否定。

; 維克托‧貝爾蒙多（ビクター・ベルモンド）
: 本作設定為貝爾蒙多一族後裔，活躍於1800年代。
: 於1800年代已改投軍隊，但仍持有聖鞭，掌握了許多當時先進的科學和作戰方法。
: 被正義勢力召喚至1666年。
: 於2008年發行的《[[惡魔城_被奪走的刻印|惡魔城 被奪走的刻印]]》定性為1797年－1897年間，貝爾蒙多失去消息，在莫理斯一族出山前，沒有人能完全掌握聖鞭「吸血鬼殺手」而須靠不同組織鑽研對抗德古拉手段，因此此角色持有聖鞭設定被否定。

; 黑暗女公爵（Countess）
: 德古拉的情人，趁貝爾蒙多一族沒有力量對抗德古拉時令德古拉復活。此角色直到[[科樂美|科樂美]]集團決定中止開發後也沒有任何正式名字。

; 德古拉（ドラキュラ）
: 君臨黑暗的帝王，被黑暗女公爵復活。
