'''霞浦县第七中学'''，简称'''霞浦七中'''，是一所位于[[福建省|福建省]][[霞浦县|霞浦县]]的二级达标中学<ref name =ref2>{{Cite web |url=http://jyt.fujian.gov.cn/jyyw/jyt/201604/t20160420_2999282.htm|title=我省新增15所二级达标高中|author= |date=2016-04-20|publisher=福建省教育厅 |language= |accessdate= 2018-10-21 |quote= }} </ref>，创办于1987年，是福建省首批体育（足球）特色学校之一<ref name =ref3>{{Cite web |url=http://jyt.fujian.gov.cn/jglb/tywsysjyyyywzc/xxty/200905/t20090531_3578517.htm |title= 福建省教育厅福建省体育局关于公布全省首批中学体育特色学校与传统体育学校的通知|author= |date=2009-05-31|publisher=福建省教育厅 |language= |accessdate= 2018-10-21 |quote= }} </ref>。<ref>{{cite web |url=http://www.fjtv.net/folder546/folder841/2016-04-20/643415.html|title=福建15所“二级达标高中”名单出炉啦！|last= |first= |date=2016-04-20|website= |publisher=福建网络广播电视台|access-date=2018-10-25|quote=}}</ref>

== 历史 ==
霞浦县第七中学是经[[福建省教育厅|福建省教育厅]]注册登记，于1987年创办一所完全中学<ref>{{Cite web |url=http://www.caigou.com.cn/edu/15822.shtml|title=霞浦七中|author= |date=|publisher=中国教育装备采购网 |language=zh-hans|accessdate= 2018-10-21 |quote= }} </ref>。2009年获评福建省首批体育特色学校<ref name =ref3/>。

霞浦县第七中学于2008年被福建省教育厅确认为省三级达标高中，2016年被提升为福建省二级达标高中<ref name =ref1/><ref name =ref2/><ref name="报道1028A">{{cite web |title=霞浦七中顺利通过省三级达标高中复评 |url=http://www.xpxww.com/2014/jjxy_0225/2061.html |website=霞浦新闻网 |accessdate=2018-10-28}}</ref>。

== 环境与教学资源 ==
校园位于龙首山下，占地60多亩。建筑面积19590平方米。2003年开办[[高中|高中]]，现拥有[[初中|初中]]1～3年级各八个班，高中1～3年级各八个班，在校学生共1183人。高中部专任教师86人，其中取得高级中学教师资格证的有81人，具有中级以上专业技术职务任职资格的教师有80<ref name =ref1/>人。该校另有福建省级骨干教师1人，宁德市级骨干教师21人；福建省学科带头人培养对象13人，宁德市级以上优秀教师8人<ref name="报道1028B">{{cite web |title=凤凰池畔的美丽校园——霞浦七中 |url=http://www.xpxww.com/2016/xyfc_0508/28980.html |website=霞浦新闻网 |accessdate=2018-10-28}}</ref>。
