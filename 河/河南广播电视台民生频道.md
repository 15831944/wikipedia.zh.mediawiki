'''河南广播电视台民生频道'''，即河南广播电视台'''3频道'''，是[[河南广播电视台|河南广播电视台]]下属的一个电视频道，前身是于2001年4月1日开播的'''经济生活频道'''，2008年1月1日改为中国大陆第一家以“民生”命名的电视频道<ref name="简介">{{cite web |url = http://www.zueiai.net/tv/ha/8.html |language= |title = 河南民生频道 |publisher = 最爱视听 |date = |accessdate = 2013年1月 }}</ref>，隶属河南大象融媒体集团。

==简介==
民生频道以“民生为本”为理念，以“表述民生，畅达民意，抒发民情”为节目定位。全天播出18个小时，覆盖[[河南省|河南省]]18个地级市，节目内容涵盖民生新闻、热播剧、生活资讯、财经动态和综艺娱乐等多方面。<ref name= "简介"/>

==主要节目==
*《民生大参考》：大型新闻资讯栏目，品牌龙头；
*《香香美食》
*《老蔡汇报》
*《民生映象》
*《小莉帮忙》
*《讲述60分》
*《欢乐英雄会》
*《健康河南》
*《民生看天下》
*《民生电视门诊》<ref name= "简介" />
