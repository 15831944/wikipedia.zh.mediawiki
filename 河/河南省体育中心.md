'''河南省体育中心'''位于[[河南省|河南省]][[郑州市|郑州市]][[惠济区|惠济区]]长兴路38号<ref>{{cite web|url=http://www.dianping.com/shop/4697600 | language= | title=河南省体育中心 |publisher=大众点评 |date= |accessdate= }}</ref>。是[[中华人民共和国|中华人民共和国]]成立后，河南省为发展体育事业而建设的大型综合体育设施。体育中心占地约980亩。上级管理机构为河南省体育局<ref>{{Cite web |url=http://venue.damai.cn/venue_289.html |title=河南省体育中心介绍 |access-date=2011-09-06 |archive-url=https://web.archive.org/web/20110823133043/http://venue.damai.cn/venue_289.html |archive-date=2011-08-23 |dead-url=yes }}</ref>。

== 简介 ==
2002年9月完工，占地约653，660平方米（980亩），建筑面积86，588平方米，总投资4.9亿人民币。它是中国足球“[[中甲聯赛|中甲聯赛]]”[[河南建业|河南建业]]队的主场。

中心交通便利，位于郑州市西北角的[[郑州市综合投资区|郑州市综合投资区]]，北臨[[310国道|310国道]]，南臨北三环路，距離[[新郑国际机场|新郑国际机场]]35公里，距離最近的火車站和长途汽車站均约15公里。

== 主要场地 ==

=== 体育场 ===
建筑面积46301平方米，总投资3.5亿元。

体育场共设48000个座椅，其中贵宾席900个，记者席360个。东西长262米，南北长255米，呈椭圆形结构。地上五层为现浇[[钢筋混凝土|钢筋混凝土]]框架结构。场地南端设有一座标高25.10米的橄榄形计分牌；东西两侧观众看台上的银灰色罩棚为球穹形钢网架结构，纵向跨度273.016米，覆盖面积28590平方米，覆盖率为70%。

== 历年重要比赛与演出 ==
2002年10月在此成功举办了河南省第九届运动会开幕式和田径比赛。也承办过[[中国足球乙级聯赛|中国足球乙级聯赛]]决赛和世界传统武术大会的开幕式。
