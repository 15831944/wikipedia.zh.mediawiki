'''河正宇'''（{{lang-ko|하정우}}，{{bd|1978年|3月11日}}），本名'''金聖勳'''<ref>[http://news.nate.com/view/20120201n12529 하정우 졸업사진 봤더니 “운동 잘할 것같은 이미지”] NEWSEN{{ko}}</ref>，[[韓國|韓國]][[男演員|男演員]]。

== 經歷 ==
作為大學戲劇專業的學生，通過參與許多舞台劇累積豐富的表演經驗。2003年，首次拍攝的電影是《瑪德琳蛋糕》，陸軍出身的他陸續拍攝多部有關軍人的長短篇電影，其中包括[[中央大學_(韓國)|中央大學]]戲劇電影系同屆生{{link-en|尹鍾彬|Yoon Jong-bin}}導演的畢業作品《不可饒恕》。2006年，因主演[[金基德|金基德]] (김기덕) 導演的文藝片《時間》而備受關注，這時大家才得知原來他是著名演員[[金容建|金容建]] (김용건) 的兒子，通過自己的努力，已是具有個人表演風格的演員，也得以擺脫父親的光環。

2012年，為了加深對電影的學習，想以導演而非演員的角度接近電影，首次執導自己編劇的喜劇電影《[[繫緊你的安全帶|繫緊你的安全帶]]》<ref>[http://cn.chosun.com/big5/site/data/html_dir/2012/10/26/20121026000015.html 河正宇自編自導新電影《人與台風》]</ref>，並由同經紀公司後輩[[鄭敬淏|鄭敬淏]]擔任男主角。

== 家族 ==
父親[[金容建|金容建]](김용건)、弟弟[[車賢宇|車賢宇]](차현우；本名：金英勳 (김영훈)) 皆是演員。

== 個人生活 ==
河正宇和模特兒[[具在伊|具恩愛]](구재이) 於2008年8月經紀人介紹相識，之後發展為戀人，2012年1月分手。<ref>[http://sports.donga.com/3/02/20120306/44567882/1 하정우-구은애 결별 “헤어진 이유는…”]東亞體育{{ko}}</ref>
