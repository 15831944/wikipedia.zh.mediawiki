'''河合明日菜'''（{{lang-ja|河合あすな}}，{{bd|1998年|3月21日}}），[[日本|日本]]的[[AV女優|AV女優]]<ref name="インタビュー">{{Cite web|url=https://web.archive.org/web/20180217141739/http://www.prestige-av.com/images/kawai_asuna/profile/interview.jpg|title=（秘）女優面接記入シート|accessdate=2017-12-14|work=|publisher=プレステージ}}</ref>，[[神奈川縣|神奈川縣]]出身<ref name="PRESTIGE"/>。目前為「[[PRESTIGE|PRESTIGE]]」專屬女優<ref name="PRESTIGE"/>，所屬於「[[T-POWERS|T-POWERS]]」事務所<ref>{{Cite web |url=http://www.t-powers.co.jp/official/talent/%E6%B2%B3%E5%90%88%E3%81%82%E3%81%99%E3%81%AA/ |title=公式サイト |publisher=[[T-POWERS|ティーパワーズ]] |accessdate=2018-07-26}}</ref>。

== 人物 ==
* 興趣是看[[恐怖片|恐怖片]]，擅長玩[[手機遊戲|手機遊戲]]<ref name="PRESTIGE"/>。

* 在中學3年級的時後，她因為在意自己的[[巨乳|巨乳]]所以穿著「ナベシャツ」去上學，為了讓自己的胸部看起來沒那麼引人注目<ref name="SSHIRABEE">{{Cite web|url=https://sirabee.com/2018/01/27/20161471215/ |title=AbemaTVで前代未聞のポロリ　話題のHカップ美女・河合あすながAVデビューを発表 |date=2018-01-27 |accessdate=2018-02-20 |publisher=しらべぇ|language=ja}}</ref>。

* 第一次性經驗是在18歲的時候，因為錯過最後一班列車所以與一位交往中的同班同學進了[[愛情賓館|愛情賓館]]，那時候還有一件[[軼事|軼事]]就是她覺得對方太小了而沒有感覺<ref>2018年2月27日、日刊スポーツ即売版17面</ref>。因此她發覺自己在過程中體驗不到任何痛苦跟愉悅。後來到AV出道之前她還與三個男人發生過關係<ref name="インタビュー" />。

* 因為在[[電視節目|電視節目]]上看到[[紗倉真菜|紗倉真菜]]，還有受到[[明日花綺羅|明日花綺羅]]以及[[凰香奈芽|凰香奈芽]]的影響所以開始對成為AV女優有興趣<ref>{{Cite web|title=「平成最後の神乳（カイブツ）」河合あすな、あどけなさの残るルックスに高いセックス偏差値!! |url=https://www.menscyzo.com/2018/02/post_15766.html |website=メンズサイゾー |date=2018-02-20|accessdate=2019-04-17 |language=ja}}</ref>。

* 藝名是經紀人取的<ref>「新人 プレステージ専属デビュー 河合あすな 奇跡の天然神乳Hカップ」（2018年2月23日、プレステージ）開頭部分</ref>，這個綽號來自「ああ～いいっすな～あすな」並附於電視[[持份者|利益相關者]]<ref>{{Cite web|url=http://www.mgstage.com/product/product_detail/SIRO-3346/|title=【初撮り】ネットでAV応募→AV体験撮影 539 |publisher=MGS動画 |date=2018-01-07 |accessdate=2018-02-20}}</ref>。

* 喜歡的[[動漫|動漫]]角色是《[[東京喰種|東京喰種]]》的「[[金木研|金木研]]」還有《[[黑子的籃球|黑子的籃球]]》的「[[黑子的籃球角色列表#高尾和成|高尾和成]]」。最喜歡的[[聲優|聲優]]是[[木村良平|木村良平]]，特別是他在《[[月歌。|月歌。]]》中演出的「霜月隼」以及《[[魔鬼戀人|魔鬼戀人]]》中的「無神皓」兩位角色<ref>{{Cite web|title=
プレステージ専属女優・河合あすな、Hカップ爆乳＆むっちり太ももが映えるコスプレ！ |url=https://www.menscyzo.com/2018/08/post_75309.html |website=メンズサイゾー |date=2018-08-31|accessdate=2019-04-17 |language=ja}}</ref>。

== 簡歷 ==
2017年她進入[[寫真偶像|寫真偶像]][[辦事處|事務所]]。同年12月18日，出現在[[AbemaTV|AbemaTV]]由「{{link-ja|アルコ＆ピース|アルコ＆ピース}}」所主持的節目《{{link-ja|妄想星期一|妄想マンデー}}》上<ref>{{Cite news |url=https://www.setn.com/News.aspx?NewsID=336039 |title=D奶女星上節目被認出有拍素人AV　全因右胸的「一顆痣」 |publisher=[[三立新聞|三立新聞]] |date=2018-01-13}}</ref>。因為要在角落擺動掛在[[比基尼|比基尼]]上的計步器來計步，由於過於緊張搖得太大力所以比基尼掉了，導致胸部走光（事實上有貼[[胸貼|胸貼]]）<ref>{{Cite web |url=https://www.news-postseven.com/archives/20180120_645483.html |title=生放送で“放送事故”のグラドル 「気付きませんでした…」|publisher={{link-ja|NEWSポストセブン|NEWSポストセブン}} |date=2018-01-20 |accessdate=2018-02-20|language=ja}}</ref>，但是多虧了這件事，她獲得了節目《新乳生》的參加權。

2018年1月7日，她演出「MGS」動畫平台（MGStage）的《【初撮り】ネットでAV応募→AV体験撮影 539 あすな 19歳 テレビのお仕事》，這也是她之前在《妄想星期一》上所提到的影片，當中也包括[[裸體|裸體]]和[[性交|性交]]場面，成為她首次AV亮相<ref>{{Cite web |url=https://www.mgstage.com/product/product_detail/SIRO-3346/ |title=「シロウトTV あすな 19歳 テレビのお仕事 」|publisher=MGS動画 |date=2018-01-07 |accessdate=2018-07-26}}</ref>。

2018年2月，日本[[AV片商|AV片商]]「[[PRESTIGE|PRESTIGE]]」官方正式宣布她成為專屬女優[[AV|AV]][[出道|出道]]，片商以「[[平成|平成]]最後の神乳」作為[[廣告標語|廣告標語]]<ref>{{cite web |url=http://www.playno1.com/article-25963-1.html#.W1k-klAzaUk |title=平成時代最後的神乳！奇蹟的H罩杯！河合あすな飢渴登場！ |publisher=PLAYNO.1玩樂達人 |date=2018-01-24}}</ref>。其出道作在「[[亞馬遜公司|Amazon]]」成人排行榜還有「[[DMM.com|DMM.R18]]」等其他[[線上購物|線上購物]]網站都獲得第一名<ref>{{cite web |url=http://www.playno1.com/article-26217-1.html#.W1k-kFAzaUk |title=新一代神乳！河合あすな將成蚊香社新王牌？ |publisher=PLAYNO.1玩樂達人 |date=2018-02-24}}</ref>。

2019年3月1日，獲得《[[DMM成人獎|FANZA成人大賞]]》2019提名「最優秀新人女優獎」<ref>{{Cite web|title=FANZAアダルトアワード2019最優秀新人女優賞ノミネート発表！|url=https://news.dmm.co.jp/article/1032979/|website=[[DMM.com|FANZAニュース]] |date=2019-03-01|accessdate=2019-03-01|language=ja}}</ref><ref>{{cite web |url=http://www.playno1.com/article-29630-1.html#.XLa0xaRS-Uk |title=爭取「最強」之名！FANZAアダルトアワード開始投票！  |publisher=PLAYNO.1玩樂達人 |date=2019̟-03-20}}</ref>。
