'''河北塞罕坝国家森林公园'''（{{lang-mnc|ᠮᡠᡵᠠᠨ ‍ᡳ<br /> ᡝᠠᠪᠠ|v=muran i aba|a=muran-i aba}}<ref name="man">參見《滿漢大辭典》，安雙成 編，遼寧民族出版社1993年出版，789頁</ref>），位于[[中国|中国]][[河北省|河北省]][[承德市|承德市]][[围场满族蒙古族自治县|围场满族蒙古族自治县]]的最北端，与[[内蒙古自治区|内蒙古自治区]][[克什克腾旗|克什克腾旗]]临界，主要由塞罕坝机械林场组成。

== 历史 ==
该国家级森林公园原名“河北木兰围场国家森林公园”，1993年5月由原[[林业部|林业部]]批准设立。2007年12月5日，原“'''河北木兰围场国家森林公园'''”更名为“'''河北塞罕坝国家森林公园'''”<ref>{{cite web |url = http://www.forestry.gov.cn/subpage/content.asp?lm_Tname=stly&lmdm=2000&f_lmname=%C9%FA%CC%AC%C2%C3%D3%CE&id=257 |title = 国家林业局关于准予木兰围场国家级森林公园变更名称的行政许可决定 |accessdate = 2007-12-21 |date = 2007-12-05 |publisher = [[国家林业局|国家林业局]] |language = zh-hans }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2008年1月7日，[[国家林业局|国家林业局]]准予[[河北省|河北省]]木兰围场国有林场管理局设立“河北木兰围场国家森林公园”。原“木兰围场国家级森林公园”与今“木兰围场国家级森林公园”虽同处围场满族蒙古族自治县，唯范围有异，所辖林班不重合。

== 概况 ==
公园总面积141万[[亩|亩]]，融[[森林|森林]]、[[草原|草原]]和沼泽[[湿地|湿地]]为一身，有山地、高原、丘陵、河流、湖泊、溪流、松树林、桦树林、羊群、军马，是消暑[[旅游|旅游]]和[[摄影|摄影]]的好地方，也是影视创作的基地。

[[木兰围场|木兰围场]]（又被称为围场坝上），与[[丰宁坝上|丰宁坝上]]和[[张北坝上|张北坝上]]被称为[[坝上|坝上]]。坝上是一个俗称，主要指在[[华北平原|华北平原]]和[[蒙古高原|蒙古高原]]交界处，海拔陡然升高呈台阶状。木兰围场的平均海拔为1487米。在[[清朝|清朝]]，木兰围场是三代[[清帝|清帝]][[木兰秋狝|秋猎打围]]的猎苑，木兰是[[满语|满语]]“哨鹿”的意思<ref name="man" />，也是蒙古部族王公朝見清帝的地方，平時有人駐守防止[[盜獵|盜獵]]。范围包含今[[围场满族蒙古族自治县|围场满族蒙古族自治县]]全境，在承德市与内蒙古自治区交界处。现在成为旅游景点。

== 动植物资源 ==
塞罕坝国家森林公园是[[黑琴鸡|黑琴鸡]]、[[大鸨|大鸨]]等珍稀鸟类的主要[[栖息地|栖息地]]，除此之外还有[[豹|豹]]、[[鹿|鹿]]、[[黄羊|黄羊]]、[[狍子|狍子]]、[[天鹅|天鹅]]等动物。

主要树种有[[白桦|白桦]]、[[山杨|山杨]]、[[樟子松|樟子松]]、[[落叶松|落叶松]]、[[云杉|云杉]]，还有[[柞树|柞树]]、[[五角枫|五角枫]]等。
