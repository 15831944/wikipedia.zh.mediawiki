'''河北师范大学'''位于[[中华人民共和国|中华人民共和国]][[河北省|河北省]][[省会|省会]][[石家庄市|石家庄市]]，是一所具有百年历史和光荣传统的[[河北省|河北省]][[省属重点大学|省属重点大学]]。学校起源于1902年创建于[[北京|北京]]的[[顺天府学堂|顺天府学堂]]和1906年创建于[[天津|天津]]的[[北洋女师范学堂|北洋女师范学堂]]。1996年6月，原河北师范大学、[[河北师范学院|河北师范学院]]与创建于1952年的[[河北教育学院|河北教育学院]]、创建于1984年的[[河北职业技术师范学院|河北职业技术师范学院]]合并，组建成新的河北师范大学，是[[中华人民共和国|中华人民共和国]]较早建立、目前规模较大的高等师范院校之一。

== 起源与沿革 ==
官方认可的起源是1902年建立的[[顺天|顺天]]府学堂（北京）和1906年建立的[[北洋|北洋]]女师范学堂（天津）。

1996年6月，原河北师范大学、河北师范学院、河北教育学院（负责教师培训，建于1952年）、河北职业技术师范学院（建于1984年）合并组建成新的河北师范大学。

其中原河北师范大学即今天的河北师范大学东校区（主校区，位于石家庄市裕华路），原河北师院为今天的河北师大西区东院（西校区的主校区，位于石家庄市红旗大街和新石南路交汇处），原教育学院为河北师大西区西院（位于新石南路），原职业技术师范学院实际仍为一个独立建制的学院，位于石家庄市东岗路。

同时，河北师范大学的新校区（位于石家庄南二环东路20号），一期工程已经结束，原西校区已经搬迁到新校区，东校区的外语学院、体育学院也已经搬迁到新区。

== 优质专业 ==
河北师大[[生物学|生物学]]是[[国家一级学科|国家一级学科]]。[[细胞生物学|细胞生物学]]为国家重点专业。学校的中国科学院院士孙大业在此领域颇有建树，可认为生物学是河北师大最优秀的专业。

此外历史学、数学、物理学这三个专业与生物学一起被称为“河北省强势特色学科”，也是河北师范大学较好的专业，并均是具有[[硕士学位授予权|硕士学位授予权]]的一级学科。

其他具有硕士学位授予权的一级学科包括：政治学、马克思主义理论、心理学、体育学、中国语言文学、化学和地理学。

== 在校学生 ==
河北师大的学校现有在拥有约350名教授，550名左右的副教授。在校本专科生（含独立学院）超过三万人，研究生则在3000人以上。
是河北省规模最大的几所高校之一。
