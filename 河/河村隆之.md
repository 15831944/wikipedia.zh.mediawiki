'''河村隆之'''（日文: {{lang|ja|'''河村 たかし'''}}，{{bd|1948年|11月3日|||Kawamura, Takashi}}）。[[日本|日本]][[政治人物|政治人物]]，[[減稅日本|減稅日本]]代表。曾任[[日本眾議院|日本眾議院]][[議員|議員]]、[[裁判官|裁判官]]追訴委員會第二代理委員長、[[總務省|總務省]][[顧問|顧問]]等。現任[[名古屋|名古屋]][[市長|市長]]。

==生平==
早年就读于[[日本|日本]][[一桥大学|一桥大学]]，大学毕业后在家族产业中任职。<ref name="ios " >{{cite web |url=http://finance.ifeng.com/roll/20120229/5677453.shtml |title=河村隆之缘何频搞“小动作” |accessdate= 2012年|work=|publisher=凤凰网 |language = zh-hans}}</ref> 河村隆之立志成为[[检察官|检察官]]，十年内参加了九次[[司法|司法]]考试，但是都没有通过。<ref name="ios " /> 河村隆之从政之后，成为[[春日一幸|春日一幸]]的男秘书。<ref name="ios " /> 并且五度当选为[[众议员|众议员]]。<ref name="ios " /> 2009年，河村隆之辞职回乡参选市长，以51万张的票高票当选。<ref name="ios " />

== 争议 ==
=== 南京大屠杀言论 ===
2012年2月20日，由[[中国共产党南京市委员会|中国共产党南京市委员会]]常委兼政法委书记[[刘志伟_(中国官员)|刘志伟]]带队的[[南京市人民政府|南京市人民政府]]赴日代表团与[[日本|日本]][[名古屋市|名古屋市]]市长[[河村隆之|河村隆之]]会谈时突然提到“南京大屠杀很可能不存在”，“牺牲的人只是正常战争死亡”，“没有目击者”等言论，一经播报和刊登出来，遭到[[中国大陆|中国大陆]]、[[台湾|台湾]]等地区人士批评。<ref name="ios " /> 第二天，[[南京市|南京市]]政府暂停与[[名古屋市|名古屋市]]政府间的官方交往；[[中华人民共和国外交部亚洲司|中华人民共和国外交部亚洲司]]司长[[罗照辉|罗照辉]]在与来访的日本[[外务省|外务省]][[亚洲大洋洲局|亚洲大洋洲局]]局长[[杉山晋辅|杉山晋辅]]会谈时也就此问题表示了强烈不满。<ref name="ios " />但河村本人事後表示不會收回此發言。<ref>[http://tchina.kyodonews.jp/news/2012/02/25458.html 詳訊：名古屋市長稱願與南京繼續交流 不撤回發言]</ref>


據日本[[共同社|共同社]]報導，事後名古屋市政府所收到的民眾意見中，絕大部份支持河村的發言<ref>[http://tchina.kyodonews.jp/news/2012/02/25481.html 名古屋市收到330件民眾意見 多數贊同市長發言]</ref><ref>[http://tchina.kyodonews.jp/news/2012/02/25886.html 名古屋市長發言風波尚無平息跡象]</ref>。河村本人則重申不會收回此言論<ref>[http://tchina.kyodonews.jp/news/2012/02/25458.html 詳訊：名古屋市長稱願與南京繼續交流 不撤回發言]</ref>。

[[東京都知事|東京都知事]][[石原慎太郎|石原慎太郎]]則在2月24日表示支持河村的言論。<ref>[http://tchina.kyodonews.jp/news/2012/02/25613.html 石原慎太郎支持名古屋市長否認"南京大屠殺"]</ref>

=== 建尾张名古屋共和国言论 ===
2012年2月29日，河村隆之自己筹集资金召集了[[爱知县|爱知县]]各处政界领导人在[[名古屋市|名古屋市]]的一家小型酒店聚餐，在就餐期间，河村隆之提出了建立“[[尾张名古屋共和国|尾张名古屋共和国]]”的设想，这个“尾张名古屋共和国”，包含22个地方自治体和400万人口。<ref name="nxz " >{{cite web |url=http://news.ifeng.com/world/detail_2012_03/01/12887146_0.shtml |title=名古屋市长称要建“尾张名古屋共和国”实现独立 |accessdate= 2012年|work=|publisher=凤凰网 |language = zh-hans}}</ref> 该言论一出，引发了日本各界的不同意见和解读。

== 亲属 ==
*父亲：[[河村鈊男|河村鈊男]]，[[二战|二战]]时期[[大日本帝国陆军|大日本帝国陆军]][[第101师团|第101师团]]步兵第101旅指令部伍长。1945年[[日本投降|日本投降]]之后于8月16日被解除武装，被安排居住于[[南京|南京]][[栖霞寺|栖霞寺]]；栖霞寺方面表示这只是对俘虏的优待，而不是“特别善待”。1946年归国，1948年创立河村纸业合资会社，任初代社长。<ref>{{cite web|url=http://city.ifeng.com/cshz/nj/20120223/220753.shtml|title=日本友人松冈环：河村隆之该谢罪|accessdate=2012年|work=|publisher=凤凰网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20140714153628/http://city.ifeng.com/cshz/nj/20120223/220753.shtml|archivedate=2014-07-14}}</ref>

由[[中国共产党南京市委员会|中国共产党南京市委员会]]常委兼政法委书记[[刘志伟_(中国官员)|刘志伟]]带队的[[南京市人民政府|南京市人民政府]]赴日代表团与[[日本|日本]][[名古屋市|名古屋市]]市长[[河村隆之|河村隆之]]会谈时，河村隆之认为[[南京大屠殺|南京事件]]（即中国方面所称“[[南京大屠杀|南京大屠杀]]”）未发生过。刘志伟对河村的说法未提出反驳，同时说南京人民是热爱和平的，学习历史是为了维护和平而不是为了延续仇恨。两人在会后交换了礼物<ref>{{cite web |url = http://view.news.qq.com/a/20120221/000032.htm |title=日本名古屋市长质疑南京大屠杀真实性|publisher=腾讯网 来源：南方日报 |author=|date=2012年2月21日 |accessdate=2012-02-26 |language=Zh-hans}}</ref><ref name="a1">{{cite web |url = http://news.qq.com/a/20120221/000565.htm|title=日名古屋市长否认南京大屠杀 南京官员未抗议 |publisher=腾讯网 来源：南方日报|author=|date=2012年2月21日 |accessdate=2012-02-26 |language = zh-hans}}</ref>。
