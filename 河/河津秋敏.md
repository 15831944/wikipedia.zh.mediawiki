'''河津秋敏'''（1962年11月5日－），日本[[游戏制作人|游戏制作人]]与[[游戏设计师|游戏设计师]]，1962年出生于[[熊本县|熊本县]]。他于[[日本工业大学|日本工业大学]]学习制陶业。河津于1985年加入[[史克威尔|史克威尔]]（今[[史克威尔艾尼克斯|史克威尔艾尼克斯]]）<ref>{{cite web |url=http://www.rpgfan.com/features/creatorstalk/index1.html |title=Creator's Talk Interview #1: Akitoshi Kawazu |accessdate=2008-01-26 |author=Winkler, Chris; Eve C. |date=2003-03-02 |publisher=RPGFan}}</ref>。他创作了电子角色扮演游戏系列[[沙加系列|沙加]]和[[最终幻想水晶编年史|最终幻想水晶编年史]]，现在他是史克威尔艾尼克斯的执行制作人以及公司第二開發事業部的部長。

== 简介 ==
{{expand section|[[沙迦系列|沙加系列]]与[[最终幻想水晶编年史系列|最终幻想水晶编年史系列]]的制作|time=2013-11-03T11:18:14+00:00}}

河津在[[东京工业大学|东京工业大学]]学习制陶术，并在朋友的邀请去[[Softbank|Softbank]]出版的一个电子游戏杂志工作<ref name="motherload" />。他称他一直对图版游戏感兴趣，并于1985年加入[[史克威尔|史克威尔]]<ref name="interview">{{cite web |url=http://www.rpgfan.com/features/creatorstalk/index1.html |title=Creator's Talk Interview #1: Akitoshi Kawazu |accessdate=2008-01-26 |author=Winkler, Chris; Eve C. |date=2003-03-02 |publisher=RPGFan}}</ref><ref name="motherload">{{cite web |url=http://www.1up.com/features/deal-square-enix-akitoshi-kawazu |title=What's the Deal with Square Enix's Akitoshi Kawazu? |accessdate=2013-02-28 |author=Jeremy Parish|date=2012-12-01 |publisher=[[1UP.com|1UP.com]]}}</ref>。他在史克威尔的第一个游戏是《[[Rad赛车|Rad赛车]]》，他创作了显示玩家成就的结束过场画面<ref name="motherload" />。

他和坂口博信一直都想创作一部[[角色扮演游戏|角色扮演游戏]]，虽然二人都是[[巫术系列|巫术]]和[[创世纪系列|创世纪]]的爱好者，但却认为这类游戏不会畅销<ref name="motherload" />。在《[[勇者斗恶龙_(游戏)|勇者斗恶龙]]》在日本获得成功，潮流对PRG的渴望得到证实后，开发开始了<ref name="motherload" />。河津监督战斗场景与系统的开发，并尽可能模仿之前的[[龙与地下城|龙与地下城]]<ref name="motherload" />。开发一些部分加入了西方奇幻游戏元素，如火怪兽惧怕冰，当时的日本游戏没有这种特性<ref name="motherload" />。

河津最為人津津樂道的便是開發SAGA系列，作為SAGA系列的生父，河津賦予遊戲高自由度、多段支線與為數龐大的同伴數量，也取消等級制度，這在當時日本RPG遊戲是從來沒有的創舉，SAGA系列也成為SE旗下的經典RPG之一。

《[[最后的神迹|最后的神迹]]》為SAGA系列以外河津獨立製作的RPG，內容充滿大量SAGA系列的影子。河津还参与了《It's New Frontier》（原文如此）的开发<ref name="spelling" />。在《[[最终幻想XII|最终幻想XII]]》的开发中，游戏领导者[[松野泰己|松野泰己]]在中途离开制作，并交由河津负责<ref name="over">{{cite web | first=Rob | last=Fahey | date=October 31, 2006 | title=This Great Fantasy| url=http://www.eurogamer.net/articles/i_finalfantasyxii_ps2 | work=[[Eurogamer|Eurogamer]] | accessdate=2013-02-28}}</ref>。

[[Avalon_Hill|Avalon Hill]]和[[龙与地下城|龙与地下城]]等图版游戏极大影响了河津的开发<ref name="motherload" />。当创作新游戏时，河津会检查图版游戏元素，并以此作为工作基础<ref name="motherload" />。
