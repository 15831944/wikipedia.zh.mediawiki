'''無上皇'''，是[[中国|中国]]古代及[[东亚|东亚]]国家在君主制度中的最高[[虚位|虚位]]，位在[[太上皇|太上皇]]之上，理論上是[[太上皇|太上皇]]的父親，[[皇帝|皇帝]]的祖父。（自称孤）

君主在經過[[禪讓|禪讓]]後成為[[太上皇|太上皇]]，而其繼位者若是再退位，則繼任者昇為太上皇，而原本的太上皇則升為地位最高的無上皇，近似於女性的[[皇太后|皇太后]]與[[太皇太后|太皇太后]]。不過相較之下，在各國歷史中無上皇出現的頻率比太皇太后更少。

== 中國 ==
[[中國|中國]]有史可查的唯一一位無上皇是[[北齊|北齊]][[後主|後主]][[高緯|高緯]]。在[[承光|承光]]元年（577年），因北齊軍隊無法抵擋[[北周|北周]]的的進攻，北齊即將亡國。[[高緯|高緯]]為避禍而將帝位內禪於[[太子|太子]][[高恆|高恆]]，是為北齊幼主。高緯成為太上皇。不久，太上皇高緯再命幼主讓位給高纬的叔父任城王[[高湝|高湝]]，於是太上皇高緯成為無上皇，但不出四日，讓位的詔書還未抵達高湝處，北齊就滅亡了。

== 日本 ==
[[日本|日本]][[平安時代|平安時代]]以後，有多位[[天皇|天皇]]很早就退位成為[[太上天皇|上皇]]（稱'''院'''），並實行[[院政|院政]]。由於這些天皇大多在英年或中年時退位，因此經常出現除[[今上天皇|今上天皇]]外，仍有一兩名上皇或[[太上法皇|法皇]]共存的情形，此時為了區別，會依照即位次序來稱之。兩位上皇（或法皇）時：
*前前任天皇稱：一院（或本院）
*前任天皇稱：新院
三位上皇（或法皇）時：
*前前前任天皇稱：一院（或本院）
*前前任天皇稱：中院
*前任天皇稱：新院

== 朝鮮王朝 ==
[[朝鮮王朝|朝鮮王朝]]的君主稱國王，因此沒有存在過太上皇甚至無上皇的稱呼，但與之相似的有女性的王妃（國王正室）。朝鮮王室將君主的法定母親為[[王大妃|王大妃]]，君主的法定祖母為[[太皇太后#朝鮮|大王大妃]]，因此一般的情況下為：
*現任國王正妻；王妃（或大妃） 
*前任國王之正妻；王大妃 
*前前任國王之正妻；大王大妃 
若是有三代以上王妃共存時；
*現任國王的正室被稱為王妃 
*前任國王的正室被稱為大妃 
*前前任國王的正室被稱作王大妃 
*前前前任國王的正室被稱為大王大妃

''詳細解釋請參看[[王大妃|王大妃]]''
