'''苏州－绍兴高速公路无锡支线'''，简称'''无锡支线'''，原称'''苏锡高速公路'''，江苏省省级高速公路网编号S83，是[[苏绍高速公路|苏绍高速公路]]连接[[无锡市|无锡市]]的一条省内支线[[高速公路|高速公路]]。<ref>[http://www.jiangsu.gov.cn/jsgov/tj/bgt/201311/t20131103_404921.html 省政府办公厅关于印发江苏省省道公路网规划（2011－2020年）的通知]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>该线路北起[[无锡市|无锡市]][[无锡新区|无锡新区]]境内的[[通锡高速公路|通锡高速公路]]新安枢纽，向东南经[[苏州市|苏州市]][[相城区|相城区]]的[[望亭镇|望亭镇]]，至[[虎丘区|虎丘区]]境内的通安枢纽接入[[通锡高速公路|通锡高速公路]]，线路全长10.015公里，全线按六车道高速公路标准建设，设计时速为100公里/小时，项目初步设计批复概算投资为5.12亿元。

无锡支线高速公路于2008年1月12日开工，2009年10月17日建成通车<ref>{{Cite web |url = http://www.subaonet.com/2009/1017/391994.shtml |title = 常昆、苏锡两条高速举行通车典礼 |author = 苏州新闻网 |date = 2009-10-17 |accessdate = 2015-09-27 |deadurl = yes |archiveurl = https://web.archive.org/web/20160304104519/http://www.subaonet.com/2009/1017/391994.shtml |archivedate = 2016-03-04 }}</ref>。
