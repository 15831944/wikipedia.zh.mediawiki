'''無窮'''或'''無限'''，來自於[[拉丁文|拉丁文]]的「infinitas」，即「沒有邊界」的意思。其[[數學|數學]][[符號|符號]]為∞。它在[[科學|科學]]、[[神學|神學]]、[[哲學|哲學]]、[[數學|數學]]和日常生活中有著不同的概念。通常使用這個詞的時候並不涉及它的更加技術層面的定義。

在神學方面，根據書面記載'''無窮'''這個符號最早被用於某些秘密宗教，通常代表人類中的神性，而書寫此符號時兩圓的不對等代表人神間的差距，例如神學家[[邓斯·司各脱|邓斯·司各脱]]（Duns Scotus）的著作中，上帝的無限能量是運用在無約束上，而不是運用在無限量上。在哲學方面，無窮可以歸因於空間和時間。在神學和哲學兩方面，無窮又作為無限，很多文章都探討過無限、絕對、上帝和[[芝諾悖論|芝諾悖論]]等的問題。

在數學方面，無窮與下述的主題或概念相關：數學的[[極限_(數學)|極限]]、[[阿列夫數|阿列夫數]]、[[集合論|集合論]]中的[[類_(數學)|類]]、{{tsl|en|Dedekind-infinite set|戴德金無限集合}}、[[羅素悖論|羅素悖論]]、[[超實數|超實數]]、[[射影幾何|射影幾何]]、[[擴展的實數軸|擴展的實數軸]]以及[[絕對無限|絕對無限]]。在一些主題或概念中，無窮被認為是一個超越邊界而增加的概念，而不是一個數。

== 歷史 ==
=== 早期無限的觀點 ===
最早關於無限的記載出現在[[印度|印度]]的[[夜柔吠陀|夜柔吠陀]]（公元前1200－900）。書中說：「如果你從無限中移走或添加一部分，剩下的還是無限。」

印度[[耆那教|耆那教]]的經書《Surya Prajnapti》（c. 400 BC） 把數分作三類：「可計的」、「不可計的」及「無限」。每一類再細分成三種階：

*可計的：小的、中的與大的。
*不可計的：接近不可計的、真正不可計的、沒有方法去計的，以及無限也包括在內。
*無限：接近無限、真正無限與無窮無盡。

現代科學家解析古代羊皮卷中的阿基米德手稿（{{le|Archimedes Palimpsest}}），在殘卷《方法》命題14中，發現阿基米德開始計算[[無窮大|無窮大]]的數目。他採取近似於19世紀[[微積分|微積分]]與[[集合論|集合論]]的手法，計算了兩組無窮大的[[集合_(數學)|集合]]，以求和的方法，證明它們之間的數目是相等的。

這是在人類記載上第一次出現無限也可以分類這一個念頭。

=== 文藝復興時代至近代 ===
伽利略最先發現一個集合跟它自己的真子集可以有相同的大小。

他用上一一對應的概念說明自然數集{1, 2, 3, 4 ...}跟子集平方數集{1,4,9,16,...}一樣多。就是1→1、2→4、3→9、4→16、.....

一一對應正是用於研究無限必要的手法。

在[[實分析|實分析]]中，符號<math>\infty</math>稱為「無窮大」，代表無界[[極限|極限]]。<math>x \to +\infty</math>表示<math>x \quad</math>超出任意給定值，<math>x \to -\infty</math>表示<math>x \quad</math>最終小於任意給定值。

一函數積分的結果可能會是無限大，若對於所有的''t''，''f''(''t'') ≥ 0，則<ref>這類在積分及級數中使用無限大的例子在任一本標準的微積分教科書中都可以找到，例如{{harvnb|Swokoski|1983|loc=pp. 468-510}}</ref>
* <math>\int_{a}^{b} \, f(t)\ dt \  = \infty</math> 意思是''f''(''t'') 在<math>a</math>到<math>b</math>的範圍內，其面積是無限大。
* <math>\int_{-\infty}^{\infty} \, f(t)\ dt \  = \infty</math>意思是在''f''(''t'')以下的總面積無限大。
* <math>\int_{-\infty}^{\infty} \, f(t)\ dt \  = a</math>意思是在''f''(''t'')以下的總面積是有限的，且總面積等於<math>a</math>。

無窮大也可以用來描述無窮[[級數|級數]]：
* <math>\sum_{i=0}^{\infty} \, f(i) = a</math>意思是無窮級數的和會收斂到某一定值<math>a</math>。
* <math>\sum_{i=0}^{\infty}  \, f(i) = \infty</math>意思是無窮級數的和會[[发散级数|发散]]。

若將標記為<math>+\infty</math>和<math>-\infty</math>的點加入到實數組成的拓撲空間，就產生實數集的「兩點[[緊致化|緊致化]]」。再加入代數屬性，就得到了[[扩展的实数轴|扩展的实数轴]]。也可將<math>+\infty</math>和<math>-\infty</math>作為一個點，記作<math>\infty</math>，並得到實數的「一點[[緊致化|緊致化]]」，也就是{{tsl|en|Real projective line|實射影線}}。[[射影幾何|射影幾何]]在平面幾何上引入無窮遠線，在高維上也有類似概念。

===複變分析中的无穷===
在[[複變分析|複變分析]]中符號<math>\infty</math>是指沒有正負號的[[极限_(数学)|极限值]]。<math>x \rightarrow \infty</math>是指''x''的大小 <math>|x|</math>會超過任意給定的數值。可以在複數平面上加上[[无穷远点|无穷远点]]，變成一個[[拓扑空间|拓扑空间]]，即為複數平面的一點[[紧化|紧化]]。若完成後，所得的平面是一維的[[复流形|复流形]]或[[黎曼曲面|黎曼曲面]]，稱為[[黎曼球面|黎曼球面]]。也可以定義在其上的代數運算（不過有一個例外，無限大不能和本身相加）。另一方面，有無限大表示可以[[除以零|除以零]]，而對於任何不為0的複數''z''，<math>\frac{z}{0} = \infty</math>，因此可以將[[亚纯函数|亚纯函数]]對映到黎曼球面上，只要將極點對應到无穷远点<math>\infty</math>即可。複變函數的[[定義域|定義域]]也可以加入无穷远点，例如[[莫比乌斯变换|莫比乌斯变换]]的函數。

=== 無窮大和無窮小 ===
一般講無窮指的都是[[無窮大|無窮大]]，但是[[無窮小|無窮小]]也是一種無窮。通過<math>y=\frac{1}{x}</math>的映射即可把無窮大映射為無窮小。在微積分中，常用高階無窮小的概念。

=== 無窮遠點 ===
[[無窮遠點|無窮遠點]]是一個加在實數軸上後得到實射影直線<math>\mathbb{R}P^1</math>的點。

=== 集合論中的無窮 ===
{{Main|序数|勢的比較}}

[[File:Infinity_paradoxon_-_one-to-one_correspondence_between_infinite_set_and_proper_subset.gif|thumb]]

在[[集合論|集合論]]中對無窮有不同的定義。[[德國|德國]][[數學家|數學家]][[康托爾|康托爾]]提出，對應於不同[[無窮集合|無窮集合]]的元素的個數（[[基數|基數]]），有不同的「無窮」。

這裏比較不同的無窮的「大小」的時候，唯一的辦法就是通過是否可以建立「[[一一對應|一一對應]]關係」來判斷，而拋棄了[[歐幾里得|歐幾里得]]「整體大於部分」的看法。例如[[整數集|整數集]]和[[自然數集|自然數集]]由於可以建立一一對應的關係，它們就具有相同的[[基數|基數]]。

例如，
* [[可數集合|可數集合]]，如[[自然数|自然數集]]，[[整数|整數集]]乃至[[有理数|有理數集]]對應的基數被定義為<math>\aleph_0</math>（阿列夫零）。
* 比[[可數集合|可數集合]]「大」的稱之為[[不可數集合|不可數集合]]，如[[實數集|實數集]]，其基數與自然數的冪集相同，為<math>2^{\aleph_0}</math>。
* 由於一個[[無窮集合|無窮集合]]的冪集總是具有比它本身更高的基數，所以通過構造一系列的冪集，可以證明[[超窮基數|超窮基數]]的個數是無窮的。然而有趣的是，超窮基數的個數比任何基數都多，從而它是一個比任何無窮大都要大的「無窮大」，它不能對應於一個基數，否則會產生某種形式的[[康托爾悖論|康托爾悖論]]。

===幾何學和拓扑学===
{{Main|向量空間的維數}}

無限[[維數|維]]的空間常用在[[幾何學|幾何學]]及[[拓扑学|拓扑学]]中，尤其是在{{link-en|分類空間|classifying space}}，也就是{{link-en|Eilenberg−MacLane空間|Eilenberg−MacLane space}}。常見的例子包括無限維的{{link-en|複射影空間|complex projective space}}K(Z,2)，以及無限維的[[實射影空間|實射影空間]]K(Z/2Z,1)。

===分形===
[[File:KochFlake.svg|thumb]]

[[分形|分形]]的結構可以重覆的放大，分形可以無限次的放大，但不會變的圓滑，而且仍維持原有的結構，分形的周長是無限的，有些的面積無限，但有些的面積卻是有限。像[[科赫曲線|科赫曲線]]就是有無限周長和有限面積的例子。

===沒有無窮的數學===
[[利奧波德·克羅內克|利奧波德·克羅內克]]懷疑無限的概念，也懷疑1870年代及1880年代時數學家使用無限的方式。這種懷疑主義形成一種稱為[[有限主義|有限主義]]的[[數學哲學|數學哲學]]，是屬於[[數學結構主義|數學結構主義]]及[[數學直覺主義|數學直覺主義]]中的一種極端形式<ref>{{cite book |title=Mathematical Thought from Ancient to Modern Times |last=Kline |first=Morris |authorlink=Morris Kline |year=1972 |publisher=Oxford University Press |location= New York|isbn=0-19-506135-7 |pages=1197–1198 }}</ref>。

== 物理中的無窮 ==
在物理上，實數的近似會用在{{link-en|連續量 (量測)|Continuum (theory)|連續量}}的量測上，自然數的近似會用在[[離散|離散]]的量測上。因此科學家假設沒有可觀察量會到無窮的數值{{Citation needed|date=February 2008}}，这是因为科学家很自然的，事实上已经是默认的接受了这样的事情：即在真实的物理场景里，是不存无穷大的可观测物理量的。在例如在[[擴展的實數軸|擴展的實數軸]]上取一個無窮的值，或是需要計算某個無窮次事件的次數。因此會預設沒有任何物體會有無窮的質量或是能量。有些事物的概念和無限有關，例如無限[[平面波|平面波]]，但現今尚沒有方法可以由實驗產生無限平面波<ref>[http://www.doriclenses.com/administrer/upload/pdf/NOT_AXI_ENG_070212_doricl97_doricle_kvgwQP.pdf Doric Lenses] {{Wayback|url=http://www.doriclenses.com/administrer/upload/pdf/NOT_AXI_ENG_070212_doricl97_doricle_kvgwQP.pdf |date=20130124011604 }} - Application Note - Axicons - 2. ==電腦計算中的無窮==
[[IEEE_754|IEEE 754]][[浮點數|浮點數]]標準中定義了正無限大及負無限大，定義為[[溢位|溢位]]、[[除以零|除以零]]或其他異常程序的結果。

像[[Java|Java]]<ref>{{cite book|last=Gosling|first=James|author2=et. al.|title=The Java™ Language Specification|publisher=Oracle America, Inc.|location=California, U.S.A.|date=27 July 2012|edition=Java SE 7|chapter=4.2.3.|accessdate=6 September 2012|url=http://docs.oracle.com/javase/specs/jls/se7/html/jls-4.html#jls-4.2.3}}</ref>及[[J語言|J語言]]<ref>{{cite book|last=Stokes|first=Roger|title=Learning J|date=July 2012|chapter=19.2.1|url=http://www.rogerstokes.free-online.co.uk/19.htm#10|accessdate=6 September 2012|archive-url=https://web.archive.org/web/20120325064205/http://www.rogerstokes.free-online.co.uk/19.htm#10|archive-date=2012-03-25|dead-url=yes}}</ref>等[[程式語言|程式語言]]允許在程式中直接用類似常數的方式存取正負無限大。正負無限大可以作為[[最大元|最大元]]，因為比所有其他的數都大（或是小）。正負無限大也可以做為像[[排序|排序]]、[[搜尋|搜尋]]或[[窗函数|窗函数]]等[[演算法|演算法]]中的{{link-en|哨兵值|sentinel value}}，找到這個值時可以結束計算。

在一些沒有最大或最小元素，但允許[[關係運算子|關係運算子]][[運算子多載|多載]]的程式語言中，程式設計師也可以「創建」最大及最小元素。若語言不允許直接存取最大或最小元素，但有[[浮點數|浮點數]]的形態，也可以用特定的運算產生正負無限大，再進行其他處理。

[[微软|微软]]的 [[Visual_Studio|Visual Studio]] 用无穷大符号作为[[标志|图标]]。
