《'''无尽旋转'''》<ref>{{Cite web |url= http://www.snh48.com/zb_list_1.html |title= SNH48 唱片EP}}</ref>是中国偶像组合[[SNH48|SNH48]]的第一张[[EP|EP]]，分紅藍兩個版本推出，紅版和藍版分別在2013年6月13日<ref>{{Cite web |url= http://weibo.com/2689280541/zBfsFCj9O |title= SNH48首张正能量EP大碟《无尽旋转》今日正式预售}}</ref>和21日于SNH48網上官方商城开始预售，但尚未在一般渠道发行。<ref name="name">{{Cite web |url= http://www.snh48.com/event/s12/ |title= SNH48无尽旋转CD正式发售}}</ref><ref>{{Cite web |url= http://lady.163.com/13/0619/11/91NS7P3R00264M4F.html |title= SNH48首张唱片《无尽旋转》开始预售 }}</ref><ref>{{Cite web |url= http://n.cztv.com/wy/2013/07/2013-07-153949475.htm |title= SNH48老码头握手会 粉丝成员共战“苏力” }}</ref>

单曲的红版与蓝版除了封面不同以外，收录的曲目及附赠的DVD内容都是相同的。本张单曲三首曲目均为[[AKB48|AKB48]]歌曲的中文翻唱。

除此之外，官方另外推出了单碟珍藏版，也分为红蓝两个版本推出。与之前版本不同的地方是仅包含CD，省去了附赠MV和花絮视频的DVD，但附赠激流之战16人合照生写一张。
