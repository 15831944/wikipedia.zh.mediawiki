== 促销 ==
普加西哲想出了一个独特方式，通过推出一个在线漫画系列来推广他即将推出的社会政治片《无法避免的战争》。这个系列的第一集，《Sohble Encounters》，开始于一群由检查员Sohble举办的派对上的年轻人（不必说对Vasant Dhoble检查员的清楚挖掘）。普加西哲已经设法在他的电影背景下巧妙地进行调整。系列的第一部分可在网上阅览。<ref>{{cite web|url=http://www.bollywoodhungama.com/movies/features/type/view/id/3738|title=Chakravyuh comic-series to highlight youth issues - Latest Movie Features - Bollywood Hungama|accessdate=1 April 2015|author=Bollywood Hungama|publisher=}}</ref>
