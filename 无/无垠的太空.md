《'''太空無垠'''》（{{lang-en|'''''The Expanse'''''}}），是[[Syfy|Syfy]]播放的一部美國[[科幻|科幻]]电视剧集，由[[馬克·費格斯與霍克·奧斯比|馬克·費格斯與霍克·奧斯比]]開發，改編自[[詹姆斯·S·A·科里|詹姆斯·S·A·科里]]（此為筆名，真名是{{link-en|丹尼爾·亞伯拉罕|Daniel Abraham (author)}}及[[詹姆斯·S·A·科里|泰·弗蘭克]]）的[[蒼穹浩瀚_(小說系列)|《蒼穹-{}-浩瀚》系列科幻小說]]。故事讲述二百年后的未来，人类殖民[[太阳系|太阳系]]。由于重力的影响，太阳系的人类逐渐分成了地球人、火星人和以小行星带为主的-{谷}-神星人，彼此之间的关系由于资源的争夺而变得紧张。警探Josephus Miller（[[湯馬士·珍|湯馬士·珍]] 飾演）連同船長Jim Holden（[[史蒂文·斯崔特|史蒂文·斯崔特]] 飾演）與他的船員一起揭發了一個威脅秩序和平及人類生存的陰謀。<ref>{{Cite web |url = http://www.hollywoodreporter.com/live-feed/thomas-jane-star-syfys-expanse-718034 |title = Thomas Jane to Star in Syfy's 'The Expanse' (Exclusive) |author = Lesley Goldberg |date = 2014-07-11 |publisher = The Hollywood Reporter |language = en}}</ref>

本劇獲得影評人的正面評價，強調本劇美麗的視覺效果、刻畫完美的角色人物和毛骨悚然的政治敘述。本劇已獲得[[雨果獎|雨果獎]]的{{link-en|雨果獎最佳戲劇表現獎|Hugo Award for Best Dramatic Presentation}}，還有提名一項[[土星獎|土星獎]]。

[[愛爾康娛樂|愛爾康娛樂]]公司負責製作和融資本劇，並將本劇的三季賣給了Syfy，2018年5月Syfy取消本劇。[[亞馬遜影片|亞馬遜影片]]後來在該月後期購買了本劇的第四季。

== 製作 ==
=== 開發 ===
《太空無垠》是根據[[詹姆斯·S·A·科里|詹姆斯·S·A·科里]]的[[蒼穹浩瀚_(小說系列)|同名小說系列]]改編而成，作者姓名是作家{{link-en|丹尼爾·亞伯拉罕|Daniel Abraham (author)}}和泰·弗蘭克的筆名，他們也是本劇的編劇和監製。第一本小說《{{link-en|利維坦覺醒|Leviathan Wakes}}》（2011年）被提名[[雨果獎最佳長篇小說獎|雨果獎最佳長篇小說獎]]和{{link-en|軌跡獎最佳科幻小說獎|Locus Award for Best Science Fiction Novel}}。2014年4月11日，Syfy宣佈小說系列改編成電視劇的承諾，並且預訂了第一季每一小時長的10集。Syfy總裁Dave Howe評論「《太空無垠》會是史詩式規模和範圍，也承諾會是Syfy最野心勃勃的電視劇。」<ref>{{Cite web |url = http://tvbythenumbers.zap2it.com/network-press-releases/syfy-gives-straight-to-series-greenlight-to-the-expanse/253301/ |title = Syfy Gives Straight-to-Series Greenlight to 'The Expanse' |date = 2014-04-11 |publisher = TV by the Numbers |language = en}}</ref>

[[馬克·費格斯|馬克·費格斯]]和[[霍克·奧斯比|霍克·奧斯比]]編寫試播集，由[[泰瑞·麥克多諾|泰瑞·麥克多諾]]執導，與{{link-en|納倫·香卡|Naren Shankar}}擔任共同編劇和節目統籌。愛爾康娛樂和The Sean Daniel Company製作，主要攝影於2014年10月29日在[[多倫多|多倫多]]開始。試播集在2015年7月的[[聖地牙哥國際漫畫展|聖地牙哥國際漫畫展]]上播放。

2015年5月，在第一季播出前，已經開始著手編寫第二季，於2015年12月續訂。<ref>{{Cite web |url = http://www.blastr.com/2015-12-31/space-opera-expanse-officially-picked-second-season-syfy |title = Space opera The Expanse officially picked up for second season at Syfy |author = Trent Moore |date = 2015-12-31 |publisher = Blastr |language = en}}</ref>《太空無垠》第二季在2017年2月1日首播。<ref>{{Cite web |url = https://twitter.com/ExpanseSyfy/status/804461700699328520 |title = #TheExpanse Season 2 premiere has moved to 2.1.17. Happy now, Earthers? |date = 2016-12-01 |publisher = Twitter |language = en}}</ref>

2017年3月16日，Syfy續訂《太空無垠》第三季，共有13集，在2018年播出。<ref name="s3 renew" />

本劇的開場標題由澳大利亞工作室Breeder和其VFX團隊製作及指導。

=== 被取消和重新續訂 ===
2018年5月，Syfy不購買未來季度的版權，據報導是因為限制性發行安排的問題，而且宣布本劇被取消。愛爾康娛樂計劃尋找其他頻道來發行未來的季度。<ref>{{Cite web |url = http://deadline.com/2018/05/the-expanse-canceled-syfy-after-three-seasons-to-be-shopped-1202388026/ |title = ‘The Expanse’ To End On Syfy With Season 3, Will Be Shopped Elsewhere By Alcon |author = Nellie Andreeva |date = 2018-05-10 |publisher = Deadline |language = en}}</ref><ref>{{Cite web |url = http://www.denofgeek.com/us/tv/the-expanse/273399/the-expanse-canceled-season-4-to-be-shopped-around |title = The Expanse Canceled, Season 4 To Be Shopped Around |author = Michael Ahr |date = 2018-05-11 |publisher = Den Of Geek |language = en}}</ref>

粉絲們對這次取消事件提出抗議，在一個網上請願書上收集了超過10萬個簽名。他們遊說{{link-en|亞馬遜工作室|Amazon Studios}}和[[Netflix|Netflix]]購買本劇；眾籌支付了一架掛有「#SaveTheExpanse」橫幅的飛機在亞馬遜工作室附近飛行。一些名人包括[[威爾·惠頓|威爾·惠頓]]、[[喬治·R·R·馬丁|喬治·R·R·馬丁]]、{{link-en|派頓·奧斯瓦特|Patton Oswalt}}和{{link-en|Andreas Mogensen|Andreas Mogensen}}都支持這次活動。<ref>{{Cite web |url = https://www.independent.co.uk/arts-entertainment/tv/news/the-expanse-season-3-george-rr-martin-will-wheaton-amazon-netflix-renewal-cancelled-a8357586.html |title = The Expanse: George RR Martin, Will Wheaton, and an astronaut call on Amazon and Netflix to renew show |author = Jack Shepherd |date = 2018-05-18 |publisher = The Independent |language = en}}</ref>5月26日，在{{link-en|國際太空發展大會|International Space Development Conference}}上，[[傑佛瑞·貝佐斯|傑佛瑞·貝佐斯]]宣布亞馬遜已經正式買了本劇的另外季度。<ref>{{Cite web |url = http://deadline.com/2018/05/the-expanse-amazon-picks-up-space-drama-series-season-4-syfy-cancellation-1202398511/ |title = ‘The Expanse’ Picked Up By Amazon For Season 4 After Syfy Cancellation |author = Nellie Andreeva |date = 2018-05-25 |publisher = Deadline |language = en}}</ref>

=== 音樂 ===
本劇的音樂由Clinton Shorter作曲。完整原聲帶名為《太空無垠：第一季 - 電視原聲帶》（''The Expanse Season 1 - The Original Television Soundtrack''），由19首曲目組成，於2016年5月20日由{{link-en|Lakeshore Records|Lakeshore Records}}公司通過iTunes發行，而2016年5月26日通過Amazon發行。
