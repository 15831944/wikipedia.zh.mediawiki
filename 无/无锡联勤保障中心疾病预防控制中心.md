'''无锡联勤保障中心疾病预防控制中心'''，位于[[江苏省|江苏省]][[南京市|南京市]][[玄武区|玄武区]]中山东路293号，隶属[[无锡联勤保障中心|无锡联勤保障中心]]<ref>{{Cite web |url = http://www.xwzf.gov.cn/art/2017/3/13/art_14523_220460.html |title = 梅园新村街道举办学雷锋为民服务活动 |author =  |date = 2017-03-13 |publisher =中国南京玄武区  |language =  |accessdate =  |quote =  }}</ref>。

==简介==
2005年8月23日，'''中国人民解放军南京军区疾病预防控制中心'''成立大会在南京举行，原南京军区联勤部军事医学研究所、卫生防疫队合并为南京军区疾病预防控制中心，成立大会由南京军区联勤部卫生部部长[[曹文献|曹文献]]主持，南京军区联勤部副部长[[仇学富|仇学富]]作指示<ref>曹勇平，南京军区疾病预防控制中心成立，东南国防医药2005,7(5):348-348</ref>。南京军区疾病预防控制中心对外也称'''中国人民解放军南京军区军事医学研究所'''<ref>南京军区疾病预防控制中心:铸健康盾牌 砺打赢精兵，解放军预防医学杂志2006(4)</ref>。南京军区疾病预防控制中心是集疾病控制、卫生执法、应用研究、培训指导为一体的综合性专业技术机构，截至2011年下设疾病预防控制所、卫生监督监测所、医学防护所、消毒与媒介生物防制所、健康教育与信息情报所、流行病微生物学研究所、医药生物技术研究所、药品仪器检验所等；根据国务院和中央军委命令，该疾病预防控制中心还担负抽组国家级卫生防疫救援队和省级三防医学救援队的任务。中心主办《中华卫生杀虫药械》杂志，编辑出版《健康之友报》、《保健信息》<ref>南京军区疾病预防控制中心，江苏卫生事业管理2011(1)</ref>。

2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，转隶[[无锡联勤保障中心|无锡联勤保障中心]]，并更名为'''无锡联勤保障中心疾病预防控制中心'''<ref>{{Cite web |url = http://db.81.cn/content/2017-04/20/content_7569518.htm |title =创新领跑提升“贡献率”  |author =  |date =  2017-04-20 |publisher =  军报记者|language =  |accessdate =  |quote =  }}</ref>。
