'''无性生殖'''是指[[生物体|生物体]]不以透過[[生殖细胞|生殖细胞]]的结合方式，也就是不經由[[減數分裂|減數分裂]]來產生[[配子|配子]]，直接由母体[[細胞分裂|細胞分裂]]後产生出新个体的[[生殖|生殖]]方式。主要分为{{le|孢子繁殖|Sporogenesis}}、[[分裂生殖|分裂生殖]]、[[出芽生殖|出芽生殖]]、[[斷裂生殖|斷裂生殖]]和[[營養器官繁殖|營養器官繁殖]]、[[孢子繁殖|孢子繁殖]]等。这种生殖的速度通常都較[[有性生殖|有性生殖]]快很多。但是，這種生殖方式的生物常常會因為其後代無法適應新環境而滅絕，這也是無性生殖的缺點之一。

个别雌性[[脊椎动物|脊椎动物]]在人工圈养或濒临灭绝的情况下也可能通过无性生殖的方式繁育下一代<ref>{{cite journal|last1=Fields|first1=Andrew T.|last2=Feldheim|first2=Kevin A.|last3=Poulakis|first3=Gregg R.|last4=Chapman|first4=Demian D.|title=Facultative parthenogenesis in a critically endangered wild vertebrate|journal=Current Biology|date=2015-06|volume=25|issue=11|pages=R446–R447|doi=10.1016/j.cub.2015.04.018}}</ref>。

== 分裂生殖 ==
{{main|分裂 (生物學)}}{{wikt|:en:schizogony|schizogony}}
又叫裂殖，通常是[[單細胞生物|單細胞生物]]所形的生殖方式。該種生物直接行細胞分裂後就可產生兩個（或更多）新個體。例如：[[草履蟲|草履蟲]]、[[變形蟲|變形蟲]]以及多數的[[細菌|細菌]]等.

== 斷裂生殖 ==
{{Main|斷裂生殖}}
通常是[[多細胞生物|多細胞生物]]受外力作用斷裂成2段或多個片段後發育為新個體。例如：[[海綿|海綿]]、[[海星|海星]]、[[渦蟲|渦蟲]]
等。

== 出芽生殖 ==
{{Main|出芽生殖}}
某些生物在生長的過程中，本身的個體上會長出一個較小的芽體。這個芽體在經過不斷的細胞分裂後，會形成一個和母體相似的形體，到最後則會脫離母體，成為一個獨立的新個體。例如：[[水螅|水螅]]、[[酵母菌|酵母菌]]等。（但水螅還是可以行有性生殖。）

== 孢子繁殖 ==
有些生物能產生大量的孢子，而孢子經過各種不同的管道散播到適當的環境中後，就能夠萌發成新個體。例如：多數的[[黴菌|黴菌]]和[[菇|菇]]、[[蘚苔|蘚苔]]等。(蕨類的孢子為減數分裂而來，故不宜用)

== 營養器官繁殖 ==
大多數[[植物|植物]]的生殖方式之一，植物的[[根|根]]、[[莖|莖]]、[[葉|葉]]等[[營養器官|營養器官]]通過分根、[[嫁接|嫁接]]、[[扦插|扦插]]等方式與母體脫離後，即能夠發育成一個新的個體。例如：[[甘薯|甘薯]]利用塊根、[[馬鈴薯|馬鈴薯]]利用塊莖、[[落地生根|落地生根]]以及[[石蓮|石蓮]]利用葉、[[草莓|草莓]]利用匍匐莖、[[洋蔥|洋蔥]]利用鱗莖以及[[萬年青|萬年青]]利用莖等。

=== 古老樹木 ===
[[瑞典|瑞典]]發現{{le|最古老樹木|oldest trees}}[[:en:Old_Tjikko|:en:Old Tjikko]]，已經有9,500年壽命，至2016年仍在生長。外電引述報道，瑞典{{link-en|于默奧大學|Umeå University}}的科學家，2004年在一個國家公園做樹木普查時，發現這棵4米高[[歐洲|歐洲]][[雲杉|雲杉]]，經過[[碳14|碳14]]年代測定法檢定，發現樹木的根至少有9,500年歷史，但[[樹幹|樹幹]]相對年輕。科學家指出，這顆樹能存活這麼久，相信因為這種樹可以無性繁殖。<ref>{{cite news |title=瑞典專家稱發現9500年歷史樹木 |url=http://rthk.hk/rthk/news/expressnews/20140126/news_20140126_55_980552.htm |publisher=[[香港電台|香港電台]] |date=2014年1月26日 |deadurl=yes |archiveurl=https://web.archive.org/web/20140201194845/http://rthk.hk/rthk/news/expressnews/20140126/news_20140126_55_980552.htm |archivedate=2014年2月1日 }}{{zh-hant}}</ref>

== 組織培養 ==
將植物組織放入含植物生長所需的營養和激素的[[培養基|培養基]]容器，組織會行細胞分裂而發育為新個體。例如：[[蘭花|蘭花]]、金線蓮
