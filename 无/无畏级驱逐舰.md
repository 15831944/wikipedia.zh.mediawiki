'''無畏級驅逐艦'''（Udaloy-class destroyer，[[蘇聯|蘇聯]]賦予的原計劃名為'''大型反潛船艦計畫1155「軍艦鳥」'''（{{lang-ru|Больши́е противолодо́чные корабли́ прое́кта 1155 «Фрега́т»}}），「無畏」實際上是其[[北約代號|北約代號]]），為[[蘇聯|蘇聯]]在[[冷戰時期|冷戰時期]]快結束前建造的一種反潛專用[[驅逐艦|驅逐艦]]，為[[現代級驅逐艦|現代級]]的後型，排水量大於現代級，被俄羅斯海軍稱為「大型反潛驅逐艦」，[[蘇聯|蘇聯]]最後一艘無畏級驅逐艦於1991年服役，[[俄羅斯海軍|俄羅斯海軍]]的恰巴年科海軍上將號 （«Адмирал Чабаненко»，舷號650）則是於1999年1月28日服役[[俄羅斯海軍|俄羅斯海軍]]北海艦隊，成為該級最後一艘實質就航的作戰艦艇。

== 艦況 ==
全長163.5米、寬度19.3米、最大排水量8700噸、最高航速30節。
== 武裝 ==
無畏級驅逐艦設計的目的是在於反潛，主要的飛彈、魚雷、炸彈都是針對攻擊潛艇而設計或改進。魚雷武裝包括2座四聯裝533毫米魚雷發射管、飛彈有2座四聯裝的SS-N-14「石英」反艦/反潛用飛彈、深水炸彈為2座RBU6000型12聯裝反潛火箭發射器，艦載機則以Ka－25和Ka－27這兩種反潛直升機為主。

== 全球機動作戰 ==
無畏級驅逐艦結構設計緊湊，在防空、反潛等裝備主要集中在船艦前部，電子設備在艦艇中部，直升機平台位於船尾。這種配屬方法在蘇俄軍艦中很常見，但無畏級驅逐艦的這種裝備方法，是配合作戰時所需的攻擊程序，同時能使軍艦航行平穩，在不利的海況下航速也能超過30節，進而滿足遠洋作戰的需求。
== 配屬 ==
北莫爾斯克號、哈爾拉莫夫海軍上將號、列夫琴科海軍上將號部屬於北方艦隊。薩波什尼科夫元帥號、維諾格拉多夫海軍上將號、潘捷列耶夫海軍上將號、特里巴特斯海軍上將號部署於太平洋艦隊。
