《'''无形杀'''》（{{lang-en|'''Invisible Killer'''}}）是一部于2009年上映的中国电影。导演[[王竞|王竞]]，主演印小天、冯波、李易祥、汤嬿。<ref>{{cite web |url=http://ent.sina.com.cn/s/m/2009-10-12/15242726703.shtml |title=《无形杀》热映 印小天坦然面对网民暴力 |accessdate=2009 |publisher=新浪网 |work= |language=zh-hans}}</ref>

== 剧情 ==
该片讲述海岛刑警队长“程涛”和“高飞”、“林燕”之间的感情纠结。

== 花絮 ==
该片涉及到“[[人肉搜索|人肉搜索]]”的话题。

该片在[[浙江省|浙江省]][[定海县|定海县]]开机。<ref>{{cite web |url=http://ent.ifeng.com/movie/news/mainland/detail_2008_10/30/53771_0.shtml |title=《无形杀》聚焦“人肉搜索” 汤嬿印小天出命案  |accessdate=2008 |publisher=凤凰网 |work= |language=zh-hans}}</ref>
