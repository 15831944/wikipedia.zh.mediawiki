'''无洞贝目'''（[[學名|學名]]：{{lang|la|'''Atrypida'''}}），又名'''無窗貝目'''，是[[动物界|动物界]][[腕足动物门|腕足动物门]][[小嘴貝綱|小嘴貝綱]]的一個[[棄用生物分類|已廢棄]]的[[並系群|並系群]]分類，於[[早古生代|早古生代]][[奥陶纪|奥陶纪]][[中奥陶世|中奥陶世]]<ref name=earth_evo_05>{{cite web |url=http://www.earthsciences.hku.hk/shmuseum/chi/earth_evo_05_intro.php |title=地球的演變展廳：早古生代 距今5.42億至4.16億年 |publisher=許士芬地質博物館 |language=zh-hant }}</ref>至[[晚古生代|晚古生代]][[石炭纪|石炭纪]][[早石炭世|早石炭世]]開始出現，繁盛于[[志留纪|志留纪]]和泥盆纪，存活至[[泥盆纪|泥盆纪]][[晚泥盆世|晚泥盆世]]<ref name=earth_evo_06>{{cite web |url=http://www.earthsciences.hku.hk/shmuseum/chi/earth_evo_06_intro.php |title=地球的演變展廳：晚古生代 距今4.16億至2.51億年 |publisher=許士芬地質博物館 |language=zh-hant }}</ref>([[泥盆紀後期滅絕事件|泥盆紀後期滅絕事件]])。

== 概述 ==
有發達的[[肉莖|肉莖]]。外殼有闊有窄，大多數皆有兩面凸出的外殼。它是腕足动物當中最早出現的[[螺旋-觸手冠動物|螺旋-觸手冠動物]](the spiral-lophophorate brachiopods)物種<ref name=paleos>{{cite web |url=http://palaeos.com/metazoa/brachiopoda/atrypida.html |title=Palaeos Metazoa: Brachiopoda: Atrypida |language=en |accessdate=2013-10-01 }}</ref>。在1952年及1965年的分類中，無洞貝目跟[[無窗貝目|無窗貝目]]（Athyridida）、[[石燕貝目|石燕貝目]]（Spiriferida）以及[[莢采貝目|莢采貝目]]（Spiriferinida）這四個目本來被歸類成為較大的[[石燕貝目|石燕貝目]]以下的四個亞目<ref name=paleos />，因為這四個分類的物種皆有螺旋-觸手冠特色。現時認為這相同之處只是[[趨同演化|趨同演化]]。
