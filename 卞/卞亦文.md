'''卞亦文'''（{{bd|1972年}}<ref>{{cite web |title=著名收藏家卞亦文 |url=http://art.people.com.cn/n/2013/0527/c364509-21632877.html |website=人民网 |accessdate=2018-09-23 |archive-url=https://web.archive.org/web/20140718013856/http://art.people.com.cn/n/2013/0527/c364509-21632877.html |archive-date=2014-07-18 |dead-url=no }}</ref>），收藏家，祖籍[[江苏|江苏]][[扬州|扬州]]，[[北京|北京]]中汉拍卖有限公司董事长。于2006年成功收藏一部古抄本《[[红楼梦|红楼梦]]》。<ref>{{cite web |title=北京中汉董事长卞亦文简介 |url=http://collection.sina.com.cn/scgd/20120307/182358698.shtml |website=新浪收藏 |accessdate=2018-09-23 |archive-url=https://web.archive.org/web/20170314080457/http://collection.sina.com.cn/scgd/20120307/182358698.shtml |archive-date=2017-03-14 |dead-url=no }}</ref>

==收藏《红楼梦》抄本==
{{main|红楼梦抄本列表}}
2006年6月，卞亦文以18万元在[[上海|上海]]敬华拍卖公司拍得一部《红楼梦》残抄本。<ref>{{cite web |title=深圳藏家的“红楼遗梦” |url=http://www.huaxia.com/zhwh/wwtd/2007/07/106087.html |website=华夏经纬网 |accessdate=2018-09-23}}</ref>抄本仅存前10回及58个回目，序言落款书名眉盦，“文介私印”，题记右下角盖“上元刘氏图书之印”章。经研究确认此人为《上元刘氏家谱》中的刘文介。<ref>{{cite web |title=卞藏本确为《红楼梦》古抄本 |url=http://www.gmw.cn/guoxue/2012-11/01/content_5544572.htm |website=光明网 |accessdate=2018-09-23 |archive-url=https://web.archive.org/web/20150331041510/http://gmw.cn/guoxue/2012-11/01/content_5544572.htm |archive-date=2015-03-31 |dead-url=no }}</ref>卞藏本在与其它抄本作对比时，发现了大量异文，可见其应是一部早期抄本的残本，可能早于甲戌本。<ref>{{cite book |author1=[[馮其庸|馮其庸]],[[李希凡|李希凡]] |title=紅樓夢大辭典（增訂本） |date=2010年 |publisher=文化藝術出版社 |location=北京 |isbn=9787503941016 |pages=409-410頁}}</ref><ref>{{cite journal |author1=戴英马 |title=从异文看卞藏本《红楼梦》早于甲戌本 |journal=宁波大学学报（人文科学版） |date=2014年 |issue=04 |page=1-11页}}</ref>
