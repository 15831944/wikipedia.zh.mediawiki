== 玩法 ==
本作是一款大逃殺類寫實[[射擊遊戲|射擊遊戲]]。在每局比賽完結後，玩家能基於他們在比賽中的生存時間及傷害和殺死敵對玩家的數目，獲得相應的遊戲貨幣（Battle point），而這些貨幣能購買適用於改變人物外表的道具。

===经典模式===
每一局遊戲最多允许有100名左右玩家參與，且在這個大型死亡競賽中爭取活至最後<ref>{{Cite web |url = https://gnn.gamer.com.tw/7/143417.html |title = 《TERA》開發商與 PlayerUnknown 合作新作《戰場 Project》24 日起展開封測 |work = 巴哈姆特電玩資訊站 |date = 2017-02-10 |accessdate = 2017-05-02}}</ref>，可以選擇第一人称视角、第三人称视角，和每隊人數不同的遊戲模式。而無論在哪種情況下，生存至最後的一人或團隊都會贏得比賽。在每局比賽開始時，玩家所控制的角色都會在沒有攜帶任何東西的情況下跳傘至战场上<ref name="Bradley.A" />。著陸後，玩家便能搜索建築物和其他場所，從而獲取隨機分佈在整個地圖上的武器、車輛及其他裝備。其後，玩家需尋找更多裝備，或者追捕及殺害其他玩家且搶走他們的裝備。

在比賽開始後的每隔數分鐘，地圖上可遊玩的「安全」區域將會向著一個隨機位置縮小，而任何身處在安全區域外的玩家便會隨著時間的推移受到傷害，直至死亡而淘汰。在遊戲內，當「安全」區域縮小時，玩家會看見一面不斷閃爍的藍色牆壁收縮，非安全区在地图上会显示为蓝色<ref name="Bradley.A">{{Cite web |url = http://www.gamasutra.com/view/news/296881/Designing_the_giant_battle_royale_maps_of_Playerunknowns_Battlegrounds.php |title = Designing the giant battle royale maps of Playerunknown's Battlegrounds |author = Alan Bradley |date = 2017-04-27 |publisher = GAMASUTRA |accessdate = 2017-05-02}}</ref>。安全區域縮窄能迫使玩家在更狹窄的區域內進行遊戲，並增加與其他倖存玩家相遇的機會{{efn|俗稱「跑毒」，但實際上絕地求生的收縮壁比較偏向電離輻射，說法来源应该来自于另一款大逃殺遊戲H1Z1的毒氣<ref>{{Cite web |url = https://justsurvive.gamepedia.com/H1Z1 |title = justsurvive.gamepedia |work = gamepedia |date = 2017-02-10 |accessdate = 2017-10-11}}</ref>。}}。另一方面，在比賽過程中，地圖上的區域會隨機遭到轟炸，並對處於該區的玩家構成威胁<ref name="Livingston.C">{{Cite web |url = http://www.pcgamer.com/playerunknowns-battlegrounds-could-have-a-bright-future-in-the-battle-royale-genre-he-created/ |title = Playerunknown's Battlegrounds could have a bright future in the battle royale genre he created |author = Christopher Livingston |date = 2017-02-28 |publisher = PC GAMER |accessdate = 2017-05-02}}</ref>。在這兩種情況下，玩家皆會在這些事件發生前被警告，從而前往安全的地區。平均而言，一局比賽的時間大約為20分鐘<ref name="Batchelor.J">{{Cite web |url = http://www.gamesindustry.biz/articles/2017-04-26-playerunknown-battlegrounds-dev-i-want-to-find-the-next-me |title = PlayerUnknown Battlegrounds dev: “I want to find the next me” |author = James Batchelor |date = 2017-04-26 |publisher = gamesindustry.biz |accessdate = 2017-05-02}}</ref>。另外，如果玩家将游戏语言设定为[[简体中文|简体中文]]，那么游戏中人物受到伤害时，流出的血将会变为绿色，而不是红色。最新版本（全球）可自定紅血或是綠血、但中國大陸版本不能自定，只有綠血。

===活动模式===
游戏官方会不定期推出不同的活动，如提高特定枪支刷新率、限定枪支种类、添加其他装备、死鬥模式（在极小的范围内进行团队作战，在一定时间内达到一定分数的队伍获胜）等。活动模式不影响段位分数。

===自定义模式===
玩家可创建房间，并设定游戏的各项数据（如是否允许出现信号枪、改变枪支刷新率和安全区缩小速率等）及玩法。除正常的大逃杀模式外，还提供战争模式和僵尸模式等玩法。初期只有部分玩家拥有开启自定义房间的权限，2018年7月18日开放给所有玩家。<ref>{{Cite web|url=http://gulugulu.cn/article/414357928867405824|title=咕噜咕噜 {{!}} G站 — 咕噜咕噜游戏站 带你玩转游戏 — GuluGulu|accessdate=2018-08-08|work=gulugulu.cn}}</ref>

== 開發 ==
本作的首席設計師[[布萊登·格里尼|布蘭登·格林]]曾經製作了遊戲《[[武裝行動2|武裝行動2]]》一個名為「DayZ：Battle Royale」的模組，而此模組是在該遊戲內甚受玩家喜愛的「DayZ」模組的其中之一<ref name="Higton.I">{{Cite web |url = http://www.eurogamer.net/articles/2017-04-07-playerunknowns-battlegrounds-interview-battle-royale |title = Battlegrounds' PlayerUnknown and the future of Battle Royale |author = Ian Higton |date = 2017-04-10 |publisher = Eurogamer |accessdate = 2017-05-02 }}</ref>。當「DayZ」成為獨立遊戲後，格林便將重心轉移至新的遊戲《[[武裝行動3|武裝行動3]]》上<ref name="Marks.T">{{Cite web |url = http://www.pcgamer.com/battle-royale-modder-brendan-greene-on-his-official-h1z1-mode/ |title = Battle Royale modder Brendan Greene on his official H1Z1 mode |author = Tom Marks |date = 2015-01-21 |publisher = PC GAMER |accessdate = 2017-05-02}}</ref>。此時，[[索尼在線娛樂|索尼在線娛樂]]已對格林的工作產生興趣，並邀請他以顧問身份協助開發《[[H1Z1：殺戮之王|H1Z1：殺戮之王]]》<ref name="Marks.T" />。繼H1Z1之後，格林成為了韓國藍洞工作室的創意總監，試圖製作一款包含著他所認定的大混戰「最終版本」概念及其以前所設計的元素的遊戲<ref name="Batchelor.J" /><ref>{{Cite web |url = http://steamed.kotaku.com/the-modder-who-created-playerunknowns-battlegrounds-1794193444 |title = The Modder Behind PlayerUnknown's Battlegrounds, A Game That Just Sold A Million Copies |author = Nathan Grayson |date = 2017-04-10 |publisher = Kotaku |accessdate = 2017-05-02}}</ref>。據格林所說，這是首次有韓國的遊戲工作室聘請外國人擔任創意總監一職，而在風險方面，他表示他與藍洞管理層的關係很強，使其團隊能在最低限度的監督下自主地工作<ref name="Batchelor.J" />。

在遊戲中，格林旨在找尋到《武裝行動3》的現實模擬及《H1Z1》街機般的動作焦點之間的平衡點<ref name="Livingston.C" />，而當中一個關鍵點便是在遊戲地圖上隨機放置武器和裝備從而實現正確的分配。在這方面，藍洞工作室作為一個大型多人線上遊戲的開發者，多年的經驗使他們能透過製作電子表格和計算以確定一個良好的分配，從而鼓勵玩家作出關於如何進行遊戲的戰略決策，而不會過度懲罰一些在開局後數分鐘都可能找不到武器的玩家<ref name="Batchelor.J" />。在每局比賽開始時從飛機跳下是格林的一個新特點，這能使玩家決定不同的戰略，例如先與其他團員會合，或獨自前進以尋找更好的物資<ref name="Batchelor.J" />。

本作的首張遊戲地圖的環境由格林的團隊人手設計。格林從自己以往製作遊戲模組的經歷中得知哪一類型的遊戲地圖最為適合大混戰概念，並查明一個擁有眾多地形特徵的島嶼可以發揮最大作用，且能為玩家提供不同的戰略選擇，使他們玩的每局遊戲都是獨一無二<ref name="Bradley.A" />。此外，格林以出現於20世紀50年代蘇聯的[[粗野主義|粗野主義]]建築為藍本，設計建築物和其他結構並將它們放置於地圖上。格林的團隊在內部進行了數次迭代測試，以測試武器隨機放置等系統，其後更不斷進行調整，例如調整物品生成、改變個別物品的容量、車輛駕駛特性及游泳速度等，甚至增加了新的地域及興趣點，從而為玩家提供更廣闊的環境來進行探索及戰鬥<ref name="Bradley.A" />。

遊戲開發工作早於2016年初展開，格林希望能在一年內將遊戲準備好<ref name="Higton.I" />。由於使用了虛幻引擎，遊戲的開發時間相對較快。這是因為較多人使用虛幻引擎作開發，而且有著大量的支援，而相比起《武裝行動》及《H1Z1》，此兩款遊戲皆使用了由遊戲開發商所開發的自家引擎，且較少開發者使用，因此開發時間較使用虛幻引擎為慢<ref name="Batchelor.J" />。在封閉測試推出後，遊戲便於2017年3月開始進入早期階段，預計將持續大約六個月<ref name="Livingston.C.2" /><ref>{{Cite web |url = http://www.pcgamer.com/playerunknowns-battlegrounds-will-hit-early-access-on-march-23-and-heres-the-official-trailer/ |title = PlayerUnknown's Battlegrounds will hit Early Access on March 23, and here's the official trailer |author = Christopher Livingston |date = 2017-03-09 |publisher = PC GAMER |accessdate = 2017-05-03}}</ref>。在此階段，遊戲開發人員打算每天改善遊戯服务器的性能，每週進行錯誤修復及每月進行內容和平衡更新<ref name="Livingston.C" /><ref name="Higton.I" />。另一方面，格林預計增加的新功能包括自定義遊戲，模組支持，以及與[[Twitch.tv|Twitch]]等串流媒體合作，使遊戲擁有重播功能甚至其他功能以將本作視作[[電子競技|電子競技]]內的其中一款遊戲<ref name="Batchelor.J" />。當Windows版本完全完成後，開發團隊計劃推出[[家用遊戲機|家用機]]版本<ref name="Higton.I" />。官方也發行了絕地求生低配免費版《PUBG Lite》面向東南亞國家，以增加遊玩人數。<ref>{{Cite web|url=https://www.fighterstudiohk.com/news/%e3%80%90%e9%a3%9f%e9%9b%9e%e5%85%8d%e8%b2%bb%e3%80%91pubg%e5%b0%87%e6%8e%a8%e5%85%8d%e8%b2%bb%e4%bd%8e%e9%85%8d%e7%89%88%e6%9c%ac%ef%bc%81/|title=【食雞免費?】PUBG將推免費低配版本！|accessdate=2019-01-15|date=2019-01-14|work=【鬥士工作室】|language=zh-TW}}</ref>

== 轶事 ==
{{Wiktionary|en:winner, winner, chicken dinner}}
在游戏中，如果最终获得了第一名，会显示“Winner winner, chicken dinner”的字样，该句台词是[[美国|美国]]赌场中赌徒的常用语，相傳是指賭贏一把，晚上就可以享用雞肉晚餐了；另說這句話可能起源於[[英國|英國]][[倫敦|倫敦]]暗巷賭徒的俚語，傳至[[澳洲|澳洲]]後，一些上年紀的[[澳洲人|澳洲人]]會用「Chicken Dinner」來代替「Winner」。無論來源是[[美國|美國]]或[[英國|英國]]，這句話的盛行與电影《[[决胜21点|决胜21点]]》息息相關，该部電影帶起了全世界對於此句台词的熱潮，電影中文版译为“大吉大利，晚上吃鸡”，《绝地求生》简体中文版沿用了这一翻译，“吃鸡”也成为《绝地求生》在[[華人|華人]]玩家中的代名词<ref>{{cite web|title=整天说“吃鸡” 《绝地求生》这句话其实起源于这里|url=http://games.ifeng.com/a/20170719/44653143_0.shtml|publisher=凤凰网|accessdate=2017-08-31|date=2017-07-19}}</ref>。而在台湾繁体中文Xbox One版本 《絕地求生》中，這句台詞的翻譯曾为“胜利胜利吃烤鸡”，后改为与简体中文相同的“大吉大利，今晚吃鸡”<ref>[//wx4.sinaimg.cn/mw690/5170f26egy1fmegbnxhagj21hc0u0npe.jpg 遊戲截圖].2017-12-13.[2017-12-15].</ref>；在香港繁体中文Android/iOS版中，這句台詞被改為「贏咗有雞食」（贏了有雞吃）。在韓國、中國的明星賽，更以[[肯德基|肯德基]]或其它炸雞品牌的現金代用卷為贊助商和獎品。

=== 广电总局审查事件 ===
2017年10月底，[[中华人民共和国|中华人民共和国]][[國家廣播電視總局|國家廣播電視總局]]对于在[[斗鱼TV|斗鱼TV]]、[[虎牙直播|虎牙直播]]等中国大陆知名网络直播平台上流行的《绝地求生：大逃杀》游戏表示，此类游戏含有暴力血腥内容，且不符合[[社会主义核心价值观|社会主义核心价值观]]，不利于青少年的身心健康发展，不应该引进或研发。2017年10月30日，中国音数协游戏工委发布公告《对“大逃杀”类游戏总局业务主管部门的基本态度》，公告称“该类游戏不仅普遍存在大量血腥、暴力内容，而且其类似于[[古罗马|古罗马]][[角斗|角斗]]场式的游戏体验与生存理念的设定严重偏离我国[[社会主义|社会主义]]核心价值观和[[中华民族|中华民族]]的传统文化习惯与道德规范，不青少年消费者的身心健康。「总局明确持有否定态度，将难以获得出版运营许可。」<ref>{{Cite web|url=http://www.cgigc.com.cn/info/15640.html|title=对“大逃杀”类游戏总局业务主管部门的基本态度_游戏产业网|accessdate=2018-05-27|work=www.cgigc.com.cn}}</ref>

受此影响，目前（截至19年4月），游戏语言设置为简体中文后，游戏内的血是绿色的，且无法更改，除非改变语言设置。“kill”的翻译从“杀”变成了“淘汰”。“headshot”的翻译从“爆头”变成了和谐的“命中头部”。

===日本“旭日旗”面具与“731部队”角色事件===
2018年7月14日，国际版本的《绝地求生:刺激战场》推出新装备，其中一款用于装饰游戏中战斗机飞行员的[[面具|面具]]，被指映射[[日本|日本]]二战时期军队的军旗“[[旭日旗|旭日旗]]”；一名人工智能角色被命名为“[[731部队|731部队]]”。7月15日，蓝洞公司将“旭日旗”面具和“731部队”角色从游戏中删除，声明是误将应该删除的测试内容不小心发布，同时致歉。<ref>{{cite web|title=“吃鸡”手游现“731部队” 韩开发商下架道歉-新华网|url=http://www.xinhuanet.com/2018-07/17/c_129914900.htm}}</ref>
