《'''絕命律師'''》（{{lang-en|''Better Call Saul''}}）是一部[[美國|美國]]電視[[連續劇|連續劇]]，由[[文斯·吉利根|文斯·吉利根]]（{{lang|en|Vince Gilligan}}）以及[[彼得·古爾德|彼得·古爾德]]（{{lang|en|Peter Gould}}）共同創作和製作。這齣電視劇為'''[[絕命毒師|絕命毒師]]'''的[[衍生劇|衍生]][[前傳|前傳]]。<ref name=IGN>{{Cite news|url=http://www.ign.com/articles/2013/09/11/breaking-bad-spinoff-better-call-saul-confirmed|publisher=IGN|title=Breaking Bad Spinoff Series Better Call Saul Confirmed|first=Roth|last=Cornet|date=September 11, 2013|accessdate=September 11, 2013}}</ref> 背景年代設於2002年，講述吉米·麥吉爾（{{lang|en|James "Jimmy" McGill}}）在'''絕命毒師'''劇情事件爆發，當上關鍵主角們的律師的六年前，是如何從個默默無名的普通辯護律師轉變成知名律師索爾·古德曼 （{{lang|en|Saul Goodman}}）的故事。<ref>{{cite web |url=http://www.ew.com/article/2014/07/11/better-call-saul-photos|title='Better Call Saul': New photos, details from 'Breaking Bad' spin-off }}</ref>

本劇由[[美國|美國]]及[[加拿大|加拿大]]地區的[[有線電視|有線電視]]頻道{{lang|en|[[AMC_(電視頻道)|AMC]]}}原創和播映。首集播於2015年2月8日<ref name="tvline">{{cite web |url=http://tvline.com/2014/11/20/better-call-saul-premiere-date-amc/ |title=''Better Call Saul'' Gets Two-Night February Premiere on AMC |publisher=TVLine |date=November 20, 2014 |accessdate=November 20, 2014 |author=Roots, Kimberly}}</ref>，目前共播出10集。首集開播首日收視率高達690萬，打破了有線電視台原創劇的開播收視紀錄。<ref>http://news.mtime.com/2015/02/10/1539353.html</ref>。2014年六月，第二季（共10集）已續訂，於2016上半年播映。<ref name=numbers>{{cite web|url=http://tvbythenumbers.zap2it.com/2014/06/19/amc-releases-first-photo-from-better-call-saul/275154/|title='Better Call Saul' Renewed for Second Season by AMC; First Season Pushed Back to 2015|last=Kondolojy|first=Amanda|work=[[TV_by_the_Numbers|TV by the Numbers]]|date=June 19, 2014|accessdate=June 19, 2014}}</ref>

==製作==
===概念===
2012年七月，'''絕命毒師'''執行製作人文斯·吉利根暗示可能會有個關於索爾·古德曼的衍生劇。<ref name="comic-con interview">{{cite web |url=http://www.deadline.com/2012/07/comic-con-qa-vince-gilligan-on-breaking-bad/|title=Comic-Con Q&A: Vince Gilligan on 'Breaking Bad'|publisher=Deadline.com|date=July 15, 2012|accessdate=July 19, 2012}}</ref>在一個訪談中，他表示很喜歡「一個主要律師為了不上法庭而願意做任何事（但會照著法院基本的步驟）的律政連續劇」的概念。<ref name="deadline">{{cite web |url=http://www.deadline.com/2013/09/breaking-bad-saul-goodman-spinoff-amc-series |title=‘Breaking Bad’ Saul Goodman Spinoff A Go At AMC, Will Serve As Prequel To Original |publisher=Deadline.com |first=Nellie |last=Andreeva |date=September 11, 2013 |accessdate=September 13, 2013}}</ref>

2013年四月，吉利根與彼得·古爾德確認會執行這齣衍生劇（古爾德是寫古德曼在絕命毒師初登場那集的劇本作家）。<ref name="deadline2">{{cite web |last=Andreeva |first=Nellie |url=http://www.deadline.com/2013/04/breaking-bad-spinoff-series-bob-odenkirk-saul-goodman-amc/ |title=AMC Eyes ‘Breaking Bad’ Spinoff Toplined By Bob Odenkirk |publisher=Deadline.com |date=April 9, 2013 |accessdate=April 9, 2013}}</ref>

==絕命毒師後續劇情==
在每一季的開頭，都會以黑白畫質形式顯示索爾·古德曼在《絕命毒師》之後的故事，顯示如同他在絕命毒師最後登場前中所說過的：“我能在奧馬哈開一家肉桂卷甜品店就不錯了”。他確實在之後潛逃至奧馬哈，隱姓埋名開了一家甜品店討生活。以下是每一季開頭顯示的各個故事。
=== 第一季 ===
索爾·古德曼，曾經風光一時的風騷律師，如今畏罪潛逃至內布拉斯加州奧馬哈，改名為基恩而當一個肉桂卷甜品連鎖店的經理。他改變面部形象，留禿頂與一撮小鬍子，日復一日地為顧客服務；而每天也要提心吊膽冒著被人認出的危險。晚上時，他會一邊酗酒、一邊含著淚、一邊觀看自己當律師時的廣告錄影帶。

=== 第二季 ===
索爾有一晚關門打烊時，獨自到商場的垃圾庫裏倒垃圾，卻因為沒把住門而導致他被反鎖在垃圾庫裏。由於打開旁邊的緊急出口會通知警方，索爾沒有辦法冒險而只好在裏面關整整兩個多小時。直到最後，清掃人員進來才讓他出去，而索爾在期間因為無聊，而在旁邊的牆上用釘子刻下小字：「索爾·古德曼到此一遊」（S. G. Was Here）。

=== 第三季 ===
索爾開門經營的一天，在午餐時間時獨自出去，坐在商場的一張長椅上吃午餐。而這時一個偷影碟的小偷跑過來躲進投幣照相庫裏，索爾在保安和警察過來搜尋時，將小偷的躲藏地點暴露給他們。當小偷被逮捕時，索爾似乎因為曾經看不慣這種事情，因此突然間起身對小偷喊出：「什麼也別說！知道嗎！找個律師！」。保安和警察沒有再理會他的言行，當索爾回去工作時，因為受驚嚇所導致的[[休克|休克]]而在同事面前昏倒……
