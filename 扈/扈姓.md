'''扈姓'''是中文姓氏之一，在《[[百家姓|百家姓]]》中排第314位。在现代他是极罕见的姓氏。

== 起源 ==

扈姓有多种起源：

#出自[[姒姓|姒姓]]。《[[史记|史记]]·[[夏本纪|夏本纪]]》[[太史公|太史公]]曰：“[[禹|禹]]为姒姓，其后分封，用国为氏，故有[[夏后氏|夏后氏]]、[[有扈氏|有扈氏]]……。”据《[[通志|通志]]·[[氏族略|氏族略]]·以国为氏》载，有扈氏的封国在扈，国灭后，其后裔以“扈”为姓。
#出[[鲜卑|鲜卑]][[魏孝文帝改革|汉化]]后改姓。《[[魏书|魏书]]·[[官氏志|官氏志]]》载，鲜卑的扈地干氏随[[元宏|孝文帝]]南迁后，改为扈姓。

== 郡望 ==

*[[京兆郡|京兆郡]]：即[[长安|长安]]。
*[[河南郡|河南郡]]：[[汉朝|汉时]]置，今[[河南省|河南省]][[洛阳市|洛阳市]]一带。

==现代分布==
*[[河北遵化|河北遵化]]：在河北省，遵化市，有上，下丁甲岭，村人大多姓扈。
*[[河南长垣|河南长垣]]：在河南省，长垣县，有魏庄镇茅庐店，孟岗乡扈占，恼里镇六里庄，油坊占等大多扈姓。
*[[山东淄博市|山东淄博市]]：在山东省，临淄区，有敬仲镇扈王庄，张王村等多扈姓。
*[[山东东营市|山东东营市]]：扈家滩
*[[山东潍坊市|山东潍坊市]]：在益都有淄博临淄扈姓分支，上千人。
*[[河北保定|河北保定]]：在蠡县有南埝村、扈家营有扈姓分支，共计三千人。
*[[河北衡水|河北衡水]]：在枣强县有扈大市 北大屯 杨黄洼、扈嘉会等村庄有分布。
*[[四川泸州|四川泸州]]：在四川泸州市有此姓，主要集中在泸州市泸县海潮镇高店扈湾子，泸州其余地方的扈姓人士基本由此处迁移过去。此外，邻近泸州的宜宾也有扈姓生活。

==名人==

*[[扈云|扈云]]：[[西汉|西汉]][[车骑将军|车骑将军]]。
*[[扈蒙|扈蒙]]：[[北宋|北宋]]初年官员，历任[[中书舍人|中书舍人]]、[[翰林|翰林]][[学士|学士]]、[[户部|户部]][[待郎|待郎]]、[[工部|工部]][[尚书|尚书]]。
*[[扈载|扈载]]：北宋初年官员，曾任翰林学士，为扈蒙堂弟。
*[[扈彦珂|扈彦珂]]：宋朝官员，曾任[[太师|太师]]。
*[[扈再兴|扈再兴]]：宋朝[[抗金|抗金]]将领，官至[[神劲|神劲]][[统制|统制]]，曾大败金兵于[[枣阳|枣阳]]。

== 文学人物 ==

*[[扈三娘|扈三娘]]：《[[水浒传|水浒传]]》中女性人物，为[[一百零八将|一百零八将]]之一。
