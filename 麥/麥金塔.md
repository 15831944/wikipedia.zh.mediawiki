'''-{zh-cn:麦金塔电脑; zh-tw:麥金塔電腦; zh-hk:Mac}-'''（{{lang-en|Macintosh}}，{{IPAc-en|ˈ|m|æ|k|ɨ|n|t|ɒ|ʃ}}，或稱為'''Mac'''），是[[個人電腦|個人電腦]]系列產品，由[[苹果公司|苹果公司]]设计、开发和销售，最初目标定位于家庭、教育和创意专业市场。随着Macintosh的出现，该电脑已经发生了巨大的改變。

首款的Mac於1984年1月24日發表。是蘋果公司繼[[Apple_Lisa|Apple Lisa]]後第二款具備[[圖形使用者介面|圖形使用者介面]]的個人電腦產品，故常被認為是首款將[[GUI|GUI]]成功商品化的個人電腦。

由於蘋果公司在之後的十多年間（68k Mac系列產品銷售時間1984~1996）將其發展成為一個龐大複雜的產品系列，並且在其後的商品系列名稱中也加入Mac，故'''Mac Computers'''也可以泛指所有由蘋果公司設計生產並且運行Mac OS作業系統的個人電腦產品。

== 命名 ==
“Macintosh”是由計劃發起人[[杰夫·拉斯金|杰夫·拉斯金]]根據他最愛的蘋果品種[[旭苹果|旭苹果]]（McIntosh）命名的。拉斯金為了避免與一家音響製造商“McIntosh”（{{link-en|McIntosh Laboratory|McIntosh Laboratory}}）的公司名稱重複，因此改變McIntosh的拼法。<ref>{{cite web|url=http://mxmora.best.vwh.net/JefRaskin.html |title=Recollections of the Macintosh project |author=Jef Raskin |year=1996 |work=Articles from Jef Raskin about the history of the Macintosh. |deadurl=yes |archiveurl=https://web.archive.org/web/20070928072214/http://jef.raskincenter.org/published/holes.html |archivedate=2007-09-28 }}</ref>

至今為止，蘋果公司仍沿用“Mac”以命名其所設計生產的個人電腦。

== 狹義的“Macintosh” ==
[[File:Steve_Jobs_with_Wendell_Brown_in_January_1984,_at_the_launch_of_Brown's_Hippo-C_software_for_Macintosh.jpg|thumb]]與[[溫德爾·布朗|溫德爾·布朗]]在为蘋果電腦(Macintosh)推出Hippo-C軟件]]
指的是1984年由[[蘋果公司|蘋果公司]]發表上市，最初發表時名叫：Apple Macintosh，有著[[All-in-One_PC_系列|一體成型]]外觀，裝載9英吋黑白CRT顯示器（512x342畫素、DTP 72PPI），並配置[[Motorola_68000|Motorola 68000]][[中央处理器|處理器]]（時脈速度：8MHz），128KB[[記憶體|記憶體]]，3.5吋[[軟碟機|軟碟機]]（容量：400K），搭配單鍵式[[滑鼠|滑鼠]]以及鍵盤，並且運行System Software（現稱[[Mac_OS|Mac OS]]）的一款個人電腦產品；由於蘋果後來發表了內置512 KB記憶體的型號（Macintosh 512K，Fat Mac），蘋果就將其正式更名為：Macintosh 128K。

而在Macintosh被發展成一整個商品系列以及後續的[[Quadra|Quadra]]系列、[[Centris|Centris]]系列和[[Power_Macintosh|Power Macintosh]]系列產品上市以後，除非是為文者有針對性的論述（例如：1984年蘋果電腦公司發布了麥金塔電腦、Macintosh是第一部具有圖形使用者界面的個人電腦⋯諸如此類的用法），現今已較少為大眾使用。

== 廣義的“Mac” ==
在1994年[[Power_Mac|Power Mac]]系列產品上市之後，廣義上所有'''由蘋果公司設計生產並且運行Mac OS作業系統'''的個人電腦產品通通被統稱為Mac Computers；但使用者及經銷商為了清楚區分採用[[Motorola_68000|680x0系列處理器]]（下稱：68k）與[[PowerPC|PowerPC]]處理器（下稱：PPC）的產品之間的差異（因二系列產品可以使用的軟體有所不同，Power Mac系列產品可以運行for 68k的軟體，反之則不可），故採用68k處理器的系列產品開始被部分使用者稱為'''68k Mac'''（或 Classic Mac），而Power Mac則簡稱為'''PPC Mac'''，到了2006年蘋果將產品轉用英特爾CPU後（[[Mac_Pro|Mac Pro]]系列⋯等），部分使用者也循此慣例，統稱為'''Intel Mac'''。

部分蘋果公司生產的個人電腦系列產品在名稱中雖然沒有包含“Macintosh”（或Mac），但仍被視為麥金塔電腦（Mac Computers），例如：[[Quadra|Quadra]]系列、[[Centris|Centris]]系列⋯等；甚至連Lisa因為能運行部份的Mac軟體，也被視為“某種Mac”（注：蘋果在Lisa電腦銷售上的失敗之後，推出型號稱為Macintosh XL的低階版“Lisa”）。

而1998年上市的[[iMac|iMac]]以及後來上市的[[eMac|eMac]]，也是廣義上的'''Mac'''（[[iMac|iMac]]並沒有被歸類為Power Mac系列，從初期到現在都是一個獨立的產品分類，但是早年的iMac使用的是PowerPC處理器，2006年以後推出的iMac則使用Intel處理器）。

至於蘋果公司的筆記型電腦系列產品，也被稱為“Mac筆記型電腦”“Mac laptops”（在[[MacBook|MacBook]]系列上市以前，蘋果公司設計生產的筆記型電腦產品系列正式名稱是[[PowerBook|PowerBook]]以及[[iBook|iBook]]系列），也是因為：'''1. 蘋果公司設計生產，2. 使用Mac OS作業系統'''。

在蘋果公司授權給其他個人電腦製造商生產的“授權相容機”的部分，一般的[[个人电脑|PC]]使用者將其視為一種“Mac電腦”（因為它們可以運行Mac OS作業系統），但部分蘋果長期使用者則多表不以為然，而嚴格的指出其為“相容機”（見英文：[http://en.wikipedia.org/wiki/Macintosh_clone Macintosh clone]、日文：[http://ja.wikipedia.org/wiki/Macintosh互換機 Macintosh互換機]）。

== 架構 ==
麥金塔（Mac）系列產品過去使用由[[摩托羅拉|摩托羅拉]]（第一代）或[[AIM聯盟|AIM聯盟]]（第二代）所研製的[[微處理器|微處理器]]，現在的新機種（第三代）已使用與一般[[x86|x86]]架構的PC[[個人电腦|個人电腦]]相同架構的[[英特爾|英特爾]]（Intel）的微處理器（註：一般的PC個人电脑上也有不同品牌的微處理器，如[[AMD|AMD]])。

第一世代麥金塔（68k Mac ,1984~1996）系列產品採用16位元的[[Motorola_68000|680x0系列微處理器]]。

第二世代麥金塔（PPC Mac ,1994~2006）系列產品則改為採用32位元的[[PowerPC|PowerPC]]系列微處理器，包含後期採用64位元的[[PowerPC_G5|G5]]微處理。

目前的產品系列為第三世代（Intel Mac ,2006~ ）。
從2006年開始，蘋果公司轉用英特爾Intel的[[中央处理器|處理器]]設計生產商品，首批採用英特爾CPU的產品是 iMac 及MacBook Pro（均使用Yonah核心的英特爾双核）。2010年至当前的產品全部使用英特爾全新架构的[[中央处理器|处理器]]英特爾[[酷睿|酷睿系列]]i5,i7或M版<ref>http://www.macworld.com/reviews/product/343881/review/27inch_imac_core_i5266ghz.html</ref>。

== 作業系統 ==
{{main|macOS}}
最初Mac的預設作業系統命名為：System Software，至System 7.5.1起正式改名為[[Mac_OS|Mac OS]]。現今Mac的作業系統為第10版，原先命名為[[Mac_OS_X|Mac OS X]]，後直接命名為[[OS_X|OS X]]+版本號；從10.12 Sierra版本開始，為了與其他平台作業系統[[iOS|iOS]]、[[watchOS|watchOS]]、[[tvOS|tvOS]]的命名方式相符合，OS X又更名為[[macOS|macOS]]。最新版 [[macOS_High_Sierra|macOS High Sierra]]與2017年9月在[[Mac_App_Store|Mac App Store]]開放下載。

macOS 使用[[Apple_Darwin|Darwin]]作為系統核心，而Darwin核心是以[[FreeBSD|FreeBSD]]為範本加以改寫而成。

蘋果官方聲稱 Mac OS X 只能在使用G3或更高階處理器的Mac上運行。但實際上，透過修改，Mac OS X 亦能成功安裝並在較早期（裝有Power PC 604e）的Power Mac機種上運行；甚至有部分使用者透過 [[:PearPC|:PearPC]] 模擬器（Linux版），在更早期的（68k Mac）Centris 650（25MHz）上安裝Mac OS X 10.3，只是使用這種方式安裝 Mac OS X 並沒有多大的實用價值可言（僅系統自我檢測便得花上數天時間）<ref>[http://www.appletalk.com.au/forums/index.php?showtopic=2003 2] {{webarchive|url=https://web.archive.org/web/20060113191533/http://www.appletalk.com.au/forums/index.php?showtopic=2003 |date=2006-01-13 }}</ref>。

除了麥金塔作業系統，麥金塔电脑還可以使用以下作業系統：
# [[BeOS|BeOS]]
# [[类Unix|类Unix]]（[[Linux|Linux]]、[[NetBSD|NetBSD]]等）
# [[Windows|Windows]]（早期PPC機種可透過第三方軟體模擬運行；採用英特爾CPU的機種則可藉由OS X 10.5以後內建的Boot Camp軟體安裝運行。）

== Mac的創新 ==
初代Macintosh的創新之處（與1984年的其他消費性個人電腦相對比）包括：

* 圖型用戶介面，圖示和电脑桌面
* 使用滑鼠作為游標工具
* 使用游標工具進行「[[雙擊|雙擊]]」、「[[拖放|拖放]]」等操作
* 「[[所見即所得|所見即所得]]」的[[文字處理系統|文字處理系統]]以及圖像修改軟體
* [[長檔名|長檔名]]，可含有空格以及沒有檔案[[扩展名|扩展名]]，初代容許31個字元，Mac OS X更可用255個字元作檔案名。
* 將3.5吋[[軟碟|軟碟]]作為標準配備
* 將播音功能作為標準功能，以及在機體裝上可發人聲的喇叭
* 美觀而且合乎[[人體工學|人體工學]]的[[工業設計|工業設計]]

1987年推出分體式設計的Macintosh II；1991年推出[[PowerBook|PowerBook]]筆記型電腦與Macintosh Quadra系列電腦（Quadra代表採用摩托羅拉68040處理器的Mac電腦，因摩托羅拉68040處理器的效能較前代68030有大幅提升，所以蘋果推出Quadra時非常自信）；1994年，應英特爾推出新世代[[Pentium|Pentium]]處理器，Apple推出採用[[PowerPC|PowerPC]]處理器的[[Power_Macintosh|Power Macintosh]]，隨後PowerBook及[[Macintosh_Performa|Macintosh Performa]]電腦亦轉向PowerPC架構。

從1998年推出至今的[[iMac|iMac]]則多定位於家用，1998年推出的[[iMac_G3|iMac G3]]採用一體機、半透明彩色機殼設計，因改變了大家對個人電腦的刻板印象而大受歡迎。1999年推出低階筆記型電腦[[iBook|iBook]]。2001年推出基於[[BSD|BSD]]的全新作業系統[[Mac_OS_X|Mac OS X]]。

2006年Apple推出採用Intel處理器的Intel Mac，Mac全面從PowerPC架構轉向Intel x86架構。iBook和PowerBook則分別被[[MacBook|MacBook]]和[[MacBook_Pro|MacBook Pro]]取代。2008年推出[[MacBook_Air|MacBook Air]]。面向專業市場的[[Power_Mac|Power Mac]]則被[[Mac_Pro|Mac Pro]]/[[iMac_Pro|iMac Pro]]取代。

== 現今的產品線 ==
現今的麥金塔電腦分成以下幾條產品線：

桌上型電腦產品分別有：針對高階用戶的工作站 [[Mac_Pro|Mac Pro]]及[[iMac_Pro|iMac Pro]]，針對消費市場與教育市場的一體成形的 [[iMac|iMac]]，還有針對入門市場及家用市場的 [[Mac_mini|Mac mini]]。

[[筆記型電腦|筆記型電腦]]產品則分為：針對最高階用戶的移動式工作平台 [[MacBook_Pro|MacBook Pro]] with Retina Display，针对普通用户的13，15英寸普通型 MacBook Pro（停產），針對消費市場與商務便攜性市場的 [[MacBook_Air|MacBook Air]]，以及主打輕薄和功能創新的[[MacBook_(Retina显示屏)|MacBook (Retina显示屏)]]。

[[伺服器|伺服器]]產品則有：針對入門市場及家用市場的 [[Mac_mini|Mac mini Server]]，針對高階及企業市場的 [[Mac_Pro|Mac Pro Server]]系列，如下表所示：
