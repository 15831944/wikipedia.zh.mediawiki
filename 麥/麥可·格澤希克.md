'''迈克尔·格泽希克'''（{{lang|en|Michael Grzesiek}}，{{bd|1994年|6月2日||}}，游戏ID：'''Shroud'''）是一名加拿大的游戏主播以及前职业电子竞技选手。他的职业生涯因为在《[[反恐精英：全球攻势|反恐精英：全球攻势]]》比赛中的出色表现而受到关注。<ref>{{cite web |url=https://dotesports.com/counter-strike/news/shroud-lan-exploit-crowd-noise-15065 |title=Shroud reveals LAN exploit involving crowd noise |publisher=Dot Esports |date=2017-06-06 |accessdate=2018-03-12 }}</ref><ref>{{cite web |first=Sam |last=Bishop |url=https://www.gamereactor.eu/esports/541473/Cloud9s+Shroud+says+players+can+exploit+crowd+noise+at+LANs/ |title=Cloud9's Shroud says players can exploit crowd noise at LANs - Game reactor UK - Counter-Strike: Global Offensive |website=Gamereactor.eu |accessdate=2017-06-26 }}</ref><ref>{{cite web |url=https://dotesports.com/counter-strike/jasonr-csgo-scene-bubble-15346 |title=JasonR: “The CS:GO pro scene is a big bubble” |publisher=Dot Esports |date=2017-06-20 |accessdate=2017-06-26 }}</ref><ref name="ThorinTreatment">{{cite web |url=https://dotesports.com/counter-strike/the-thorin-treatment-shroud-will-still-try-14922 |title=The Thorin Treatment: Shroud will still try |publisher=Dot Esports |date=2017-05-29 |accessdate=2017-06-23 }}</ref><ref>{{cite web |last=Vitale |first=Anthony |url=https://realsport101.com/news/sports/esports/csgo/cloud9-decimate-expectations-in-london/ |title=Cloud9 Decimate Expectations in London - RealSport |website=Realsport101.com |accessdate=2017-06-26 }}</ref>2018年4月他宣布正式结束自己的电竞生涯，开始成为全职的游戏主播，主要直播《反恐精英：全球攻势》、《[[绝地求生|绝地求生]]》以及《[[使命召唤：黑色行动4|使命召唤：黑色行动4]]》等游戏。<ref name="ThorinTreatment"/>截止到2018年11月，他在Twitch上获得了超过480万人的关注，是该网站受关注第三高的主播。
