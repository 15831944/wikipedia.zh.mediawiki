'''麥寮鄉'''{{lang-nan-tw|Be̍h-liâu-hiong}}）位於[[臺灣|台灣]][[雲林縣|雲林縣]]西北角，西濱[[台灣海峽|台灣海峽]]，北隔[[濁水溪|濁水溪]]與[[彰化縣|彰化縣]]為鄰。地處沿海地區，濁水溪出海口，在當地有[[臺語|臺語]]「風頭水尾」之稱。<ref>[http://www.fpcc.com.tw/tc/reclamation.php 台塑石化股份有限公司－填海造陸]</ref>其地勢平坦，氣候屬[[副熱帶季風氣候|副熱帶季風氣候]]。自[[第六套輕油裂解廠|六輕]]設廠後，人口逐年上升，現已為雲林人口第三多鄉鎮，僅次於虎尾、斗六，工商業也日益發達，漸漸在雲林西北形成新的生活圈（原本雲林西北得遠至[[虎尾鎮|虎尾]]消費），在《雲林縣綜合發展計畫》中，已將麥寮劃出虎尾生活圈而獨列為雲林的三大生活圈（環形城市<ref>斗六、斗南、虎尾、西螺四鎮市形成的大生活圈</ref>、北港、麥寮）之一，同生活圈還有[[崙背鄉|崙背鄉]]、[[褒忠鄉|褒忠鄉]]、[[東勢鄉|東勢鄉]]、[[臺西鄉|臺西鄉]]<ref>[http://bprf.yunlin.gov.tw/ 雲林縣綜合發展計畫第二次通盤檢討資訊網站]</ref>。

== 歷史 ==
麥寮鄉昔日為[[洪雅族|洪雅族]]南社及[[貓兒干社|貓兒干社]]的活動範圍，（西元1278年）[[漢人|漢人]]最早的開發重心是[[海豐港|海豐港]]<ref>[https://ctext.org/wiki.pl?if=gb&chapter=925840#p35 宋史-1277年文天祥、興化]</ref><ref>[https://ctext.org/wiki.pl?if=gb&chapter=925840#p53 宋史-1278年文天祥、海豐]</ref><ref>[https://ctext.org/wiki.pl?if=gb&chapter=40999#p68 宋史-1278年文天祥、零丁洋]</ref><ref>[https://ctext.org/wiki.pl?if=gb&chapter=734873#p19 臺灣府志-南社、貓兒乾、興化(興華村)]</ref><ref>[https://ctext.org/wiki.pl?if=gb&chapter=168045#p20 臺灣府志-零丁洋、番語(興化語)]</ref><ref>[https://ctext.org/wiki.pl?if=gb&chapter=716801#p165 雲林縣採訪冊-海豐、狀元(文天祥)]</ref>，康熙年間（西元1685年）該港已成為沿岸貿易門戶之一<ref>[https://zh.m.wikisource.org/wiki/臺灣府志_(蔣志)/卷07#水稅 臺灣府志-卷七]</ref><ref>[https://ctext.org/wiki.pl?if=gb&chapter=501104#p126 臺灣府志-卷五]</ref>。

海豐港在台灣府誌中稱為海防港<ref>[https://ctext.org/wiki.pl?if=gb&chapter=452193#p444 續修臺灣府志-卷二]</ref>，雍正八年（ 西元1730年）陳、張、石三姓墾首向南社社番請墾麥寮<ref>[https://ctext.org/wiki.pl?if=gb&chapter=655281#p129 清代臺灣大租調查書-卷一]</ref>；雍正九年（西元1732年）開為島內貿易之所，昔日港灣水深，商船每避風碇泊，早形成店肆<ref>[https://ctext.org/wiki.pl?if=gb&chapter=525270#p62 重修福建臺灣府志-卷五]</ref>；乾隆末年至嘉慶初年又有泉州人向南社社番請墾[[橋頭_(雲林縣)|橋頭]]、[[沙崙後|沙崙後]]、[[施厝寮|施厝寮]]、[[雷厝|雷厝]]、[[許厝寮_(雲林縣)|許厝寮]]等地<ref>[http://docsplayer.com/29964235-南華大學數位論文.html 南華大學論文-臺西麥寮地區文學發展之研究29]</ref>。至乾隆中期新虎尾溪開河口時，被溪沖壞<ref>[http://jntnu.ord.ntnu.edu.tw/pub/PaperContent.aspx?cid=53&ItemId=506&loc=tw# 臺灣師範大學論文-清代侵台颱風災害之初步研究390]</ref>，街肆荒廢，商勢全移於東南麥寮。 當時麥寮因新虎尾溪之故，形成一麥津渡船頭（中興路、泰順路附近）<ref>[https://www.mlvillage.gov.tw/form/Details.aspx?Parser=2,9,50,115,,,27 麥寮鄉公所-村里]</ref>，與大陸貿易之貨物皆由此船頭進出。因當時麥寮盛產大麥與小麥<ref>[https://ctext.org/wiki.pl?if=gb&chapter=419323#p129 臺灣志略-卷一]</ref>，農民為運輸方便，在船頭旁搭一寮仔，專門堆放成包的麥子，當時人稱此寮為「麥仔簝」<ref>[https://ctext.org/wiki.pl?if=gb&chapter=452193#p102 續修台灣府志-卷二]</ref>。辛亥年時新虎尾溪暴漲改道，麥津渡船頭也因泥沙堆積而淤塞，後來的人就以麥仔簝代表全麥寮。之後，又衍申為麥簝街<ref>[https://ctext.org/wiki.pl?if=gb&chapter=272539#p108 彰化縣志-卷二]</ref>，最後改為麥寮。

[[大正|大正]]九年（西元1920年）台灣實施「州郡街-{庄}-制」，隸屬[[臺南州|台南州]][[虎尾郡|虎尾郡]]-{[[崙背庄|崙背庄]]}-，戰後分設麥寮鄉和[[崙背鄉|崙背鄉]]<ref>[https://www.mlvillage.gov.tw/content/index.aspx?Parser=1,9,48 麥寮鄉公所-歷史]</ref>。

== 地理 ==
麥寮背山面海位於濁水溪與新虎尾溪之間，早期以農業跟漁業為主，後有工業進駐，人口約四萬三千人，農地約四萬八千多公頃<ref>[https://www.mlvillage.gov.tw/content/index.aspx?Parser=1,9,49 麥寮鄉公所-地理]</ref>。
=== 位置 ===
麥寮鄉位於雲林縣的西北角，北倚彰化縣大城鄉，南臨為雲林縣台西鄉、東勢鄉及褒忠鄉為鄰；東鄰崙背鄉，西臨台灣海峽。

=== 地形 ===
主要為平原地形與海埔新生地形：
*東側：濱海沖積平原區－濁水溪與新虎尾溪沖積平原
*西側：海埔新生地
**海岸隆起新生地
**濁水溪入海口淤積新生地
**季風產生之沙洲新生地

=== 氣候 ===
麥寮鄉屬於亞熱帶氣候型，年雨量約1,100～1,200公釐之間。 	
年均溫約攝氏22度。最低溫1月為攝氏l5.5度，最高溫7月為攝氏28.5度。

=== 工業 ===
==== 六輕工業區 ====
{{main|第六套輕油裂解廠}}
*工業區進駐
近年來麥寮鄉因為六輕工業區的進駐，由於六輕工業區提供為數不少的工作職缺，吸引了大量人口移入麥寮鄉，加上附屬在六輕工業區底下外包商的員工，有著數萬人的勞動力。此外六輕工業區的進駐雖然令麥寮鄉從原本的農鄉轉變成工業化的新市鎮，不過在帶來優厚補助和回饋福利的同時，所產生的環境髒亂也開始突顯出來。

*影響<ref>[http://www.shs.edu.tw/works/essay/2008/03/2008033115162015.pdf 六輕工業區設立對麥寮鄉的影響]</ref>
**六輕工業區創造大量人口就業機會。
**工業區人口提高消費力。
**麥寮商圈與醫療建趨完善。
**公共大眾運輸初步發展與交通建設興建。
**商店街和房產升值。
**上下班交通路線在尖峰時刻的阻塞。
**環境[[空氣汙染|空氣汙染]]問題。
**麥寮工業專用港設立<ref>[http://www.moeaidb.gov.tw/iphw/mlharb/index.do?id=11 麥寮工業專用港-設立目地]</ref>。

== 經濟 ==
[[File:Formosa_Plastics_Group_Mail-Liao_Industrial_Complex,_Yunlin_(Taiwan).jpg|thumb]]
產業主要以[[農業|農業]]、[[漁業|漁業]]為主，近年來由於長庚醫院麥寮分院、雲林離島式基礎工業區的設立，
其工業區共分為四個區域:
麥寮區 : 2679公頃
新興區 : 1433公頃
台西區 : 1163公頃
四湖區 : 8965公頃
總面積 : 14240公頃

欲建立完整的 煉油、煉鋼、石化等基礎產業及其中、下游工業，
至今只開發建廠到 麥寮區，其定位為 大型石化產業工業區，
目前總計有兩大石化集團在此區設廠，台塑企業集團、長春企業集團。

麥寮區 : 
*[https://web.archive.org/web/20050507120417/http://www.fpg.com.tw/ 台塑集團相關企業及人才招募網站]

[[台塑集團|台塑集團]]相關企業:
台塑石化（[[第六套輕油裂解廠|第六套輕油裂解廠]]）、麥寮汽電、台灣塑膠、南亞塑膠、台灣化學纖維、台朔重工、
南中石化、台灣小松電子、台灣旭彈性、台朔光電、中塑油品、台灣醋酸化學。

*[http://www.ccp.com.tw/ 長春石化集團相關企業]：
**[https://web.archive.org/web/20150227050604/http://recruit.ccp.com.tw/job/ 長春相關企業人才招募網站]

[[長春集團|長春集團]]相關企業:
長春石油化學、長春人造樹酯、大連化學工業。

據估計麥寮工業區的員工，大約有1.5萬人左右，
再外加承包商的1.5萬員工，共計有約3萬人。

工業區設立，提供不少直接與間接就業的機會，也帶來不少外包商的進駐，同時也帶來不少的消費與商機，
讓原本的農漁業為主的麥寮地區，從農漁村生活型態逐漸地改變，促使其工商業都有明顯的成長，
生活機能也日漸齊全，使麥寮逐漸成為 雲林地區重要工業重鎮之一。
