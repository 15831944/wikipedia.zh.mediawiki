'''麥可·路易斯·戴門'''（{{lang-en|Michael Louis Diamond}}，{{bd|1965年|11月20日||}}）<ref name=rollingstone>{{cite web |url=https://www.rollingstone.com/music/artists/beastie-boys/biography |title=Beastie Boys Biography |publisher=[[Rolling_Stone|Rolling Stone]] |accessdate=May 24, 2011}}</ref>是美國饒舌歌手，知名於其藝名'''麥克·D'''（{{lang-en|Mike D}}）以及身為[[嘻哈音樂|嘻哈]]團體[[野獸男孩|野獸男孩]]的創始成員。除了饒舌與歌唱外，戴門也有打鼓的專長，並曾幫[[莫比|莫比]]與[[碧玉_(歌手)|碧玉]]做過混音。<ref name="MTV bio">{{cite web |url=http://www.mtv.com/music/artist/beastie_boys/artist.jhtml#bio |title=Beastie Boys MTV Bio |publisher=[[MTV|MTV]] |accessdate=June 3, 2009}}</ref>

== 生平 ==
身為{{link-en|聖安學校 (紐約市)|Saint Ann's School (New York City)|聖安藝校}}的學生，戴門出生於紐約市的[[犹太人|猶太]]家庭。1979年，戴門共同組建了樂團{{link-en|The Young Aborigines}}。1981年，樂團的粉絲[[亞當·佛契|亞當·佛契]]（即後來的MCA）入團並成為了貝斯手，並在當時的吉他手{{link-en|約翰·貝瑞 (音樂家)|John Berry (musician)|約翰·貝瑞}}建議之下，樂團正式更名為[[野獸男孩|野獸男孩]]。1982年，隨著貝瑞的退出與[[亞當·霍羅維茨|亞當·霍羅維茨]]（即Ad-Rock）的加入，他們的音樂風格也從[[朋克搖滾|龐克]]轉為[[嘻哈音樂|嘻哈]]。<ref>{{cite web |url=http://www.xfm.co.uk/artists/beastie-boys/biography |title=Beastie Boys Biography |publisher=[[Xfm_London|Xfm London]] |accessdate=June 3, 2009 |deadurl=yes |archiveurl=https://web.archive.org/web/20081204224459/http://www.xfm.co.uk/artists/beastie-boys/biography |archivedate=December 4, 2008}}</ref>

1992年，戴門成立了野獸男孩的自家唱片廠牌{{link-en|Grand Royal}}（現已停止營運）。他也對室內裝飾感興趣，並設計了以布魯克林為主題的{{link-en|棉質印花布|Toile}}，並被用於2013年1月重新開放的[[雀兒喜_(紐約)|雀兒喜]]華蓋夜總會（Marquee nightclub）的翻新工程。<ref>{{cite news |url=http://blog.zagat.com/2013/01/marquee-unveils-complete-remodel-on-ten.html |title=Marquee Unveils Complete Remodel on Ten-Year Anniversary |last=Mulcahy |first=James |date=January 17, 2013 |work=[[Zagat|Zagat]] |accessdate=January 24, 2013 |deadurl=yes |archiveurl=https://web.archive.org/web/20140606212907/http://blog.zagat.com/2013/01/marquee-unveils-complete-remodel-on-ten.html |archivedate=June 6, 2014}}</ref><ref>{{cite news |url=http://blogs.artinfo.com/objectlessons/2013/01/17/beastie-boy-mike-ds-designs-embellish-marquees-bathroom-walls/ |title=Beastie Boy Mike D's Designs Embellish Marquee's Bathroom Walls |last=Zara |first=Janelle |date=January 17, 2013 |work=[[Louise_Blouin_Media|Artinfo]] |accessdate=January 24, 2013}}</ref>

在2012年佛契去世一年後，戴門告訴《''滾石雜誌''》他「再次對製作新東西感到興奮」並於2013年7月發行了新歌〈Humberto Vs the New Reactionaries (Christine and the Queens Remix)〉。<ref>{{cite web |url=https://www.rollingstone.com/music/news/beastie-boys-mike-d-im-excited-about-making-new-stuff-again-20130715 |title=Beastie Boys' Mike D: 'I'm Excited About Making New Stuff Again' |website=rollingstone.com |accessdate=July 8, 2017}}</ref>2013年8月，由戴門和哈洛維茲混音的[[塑胶小野乐队|塑膠小野樂隊]]歌曲〈Bad Dancer〉在線上平台上架。<ref>{{cite web |url=https://thefutureheart.com/2013/08/05/plastic-ono-collabs/ |title=Preview New Iggy Pop Plastic Ono Split 7"; Ono Songs with tUnE-yArDs, Lenny Kravitz |first= |last=TheFutureHeart |date=August 5, 2013 |website=thefutureheart.com |accessdate=July 8, 2017}}</ref>2014年10月，戴門表示他以製作人身分與美國搖滾樂團{{link-en|葡萄牙.人|Portugal. The Man}}在錄音室工作。<ref>{{cite web |url=https://www.youtube.com/watch?v=wfp3cgvC5bo |title=A surprising home for a Beastie Boy |first= |last=CNN |date=October 13, 2014 |publisher= |accessdate=July 8, 2017 |via=YouTube}}</ref>他也曾製作英國龐克樂團{{link-en|奴隸樂團|Slaves (UK band)}}於2016年9月30日發行的第二張錄音室專輯《{{link-en|隻手遮天 (奴隸樂團專輯)|Take Control (Slaves album)|隻手遮天}}》。<ref name="HMV">{{cite web |title=Slaves announce full details of Beastie Boys produced new album Take Control |url=http://www.hmv.com/music/slaves-announce-full-details-of-beastie-boys-produced-new-album-take-control |website=HMV |accessdate=September 3, 2016 |date=August 12, 2016}}</ref>

2017年，戴門幫[[傑克·強森|傑克·強森]]的歌曲〈{{link-en|大蘇爾 (傑克·強森歌曲)|Big Sur (Jack Johnson song)|大蘇爾}}〉進行了混音。<ref>{{Cite web |url=https://itunes.apple.com/us/station/beats-1/idra.978194965?app=music |title=Connecting to the iTunes Store. |website=itunes.apple.com |access-date=August 6, 2018}}</ref>

== 個人生活 ==
1993年，戴門與電影、電視和MV導演{{link-en|塔姆拉·戴維斯|Tamra Davis}}結婚。他們育有兩個孩子，戴維斯·戴門（Davis Diamond）和希基勒·戴門（Skyler Diamond）。<ref name=huffingtonpost>{{cite web |url=http://www.huffingtonpost.com/tamra-davis |title=Tamra Davis |publisher=[[Huffington_Post|Huffington Post]] |accessdate=May 24, 2011}}</ref>夫妻倆目前為合法分居。<ref>{{Cite news|url=https://www.rollingstone.com/music/features/how-exbeastie-boy-mike-d-found-new-peace-in-malibu-w451725|title=Mike D's Endless Summer: How Ex–Beastie Boy Found New Peace in Malibu|work=Rolling Stone|access-date=2018-04-01}}</ref>
