'''麥理浩徑'''（{{lang-en|MacLehose Trail}}）是[[香港|香港]]最早啟用的一條長途遠足徑，於1979年10月26日啟用，以時任[[香港總督|香港總督]][[麥理浩|麥理浩]]命名、並且由其本人主持剪綵開幕。麥理浩徑是香港最長的遠足徑，全長100公里，共分10段。麥理浩徑共有200支標距柱，約每500米一支。它起始於[[西貢|西貢]]北潭涌(大網仔路及北潭路路口，該路口設有本徑起點標誌)，貫穿了[[西貢東郊野公園|西貢東]]、[[西貢西郊野公園|西貢西]]、[[馬鞍山郊野公園|馬鞍山]]、[[獅子山郊野公園|獅子山]]、[[金山郊野公園|金山]]、[[城門郊野公園|城門]]、[[大帽山郊野公園|大帽山]]及[[大欖郊野公園|大欖郊野公園]]，以[[屯門|屯門]]為終點(橫過[[青山公路|青山公路]]及[[杯渡路|杯渡路]]路口的輕鐵橋下)。大部分的麥理浩徑都是依山勢而建。
[[File:Tuen_Mun_Moai_at_MacLehose_Trail.jpg|thumb]]
1979年初，第25任港督麥理浩下令駐港英軍開闢此徑，只花了約九個月。麥理浩徑啟用初期，主要用作訓練英軍。據說當時的英軍必須在24小時內走畢全程，最快的[[尼泊爾|尼泊爾]]軍只用了12小時多。1984年後，麥理浩徑的民用性質才日益擴大。之後陸續開闢非軍事的[[衛奕信徑|衛奕信徑]]、[[鳳凰徑|鳳凰徑]]、[[港島徑|港島徑]]，以及多條郊遊徑。

1979年3月29日，即麥理浩徑落成前約七個月，麥理浩到北京會見鄧小平，得知中國一定會收回香港，香港問題正式出現。有人認為麥理浩徑的開闢和英國對香港前途憂慮有關。麥理浩徑是有意識地將[[新界|新界]]東西及[[九龍半島|九龍半島]]以北的各個[[制高點|制高點]]，包括[[西灣山|西灣山]]、[[牛耳石山|牛耳石山]]、[[雞公嶺|雞公嶺]]、[[馬鞍山|馬鞍山]]、[[獅子山|獅子山]]、[[大帽山|大帽山]]等連成一線，即把新界切成兩半，而第五、六段更有[[二戰|二戰]]時留下的許多仍可使用的[[碉堡|碉堡]]和軍事建設。因此有人推測麥理浩徑是一條[[軍事防線|軍事防線]]。邊界是第一線、麥理浩徑是第二線、維港是第三線。

2016年5月3日，[[國家地理頻道|國家地理頻道]]邀得二十位行山愛好者，嚴選全球最佳行山徑「夢想之路」，受訪者包括越野跑手、集團總裁和作家等，麥理浩徑獲評為全球二十條最佳行山徑之一。<ref>{{cite news|url=http://orientaldaily.on.cc/cnt/news/20160504/00176_075.html|title=全球20佳 麥理浩徑 夢想之路|newspaper=''[[東方日報_(香港)|東方日報 (香港)]]''|date=2016年5月4日}}</ref>

# 從[[北潭涌|北潭涌]]大網仔路與北潭路交界開始，沿著[[西貢|西貢]][[萬宜路|萬宜路]]繞過[[萬宜水庫|萬宜水庫]]南邊，經過[[萬宜水庫|萬宜水庫]]的西壩和東壩，最後到達[[浪茄|浪茄]]沙灘。
# 由[[西貢東郊野公園|西貢東郊野公園]]的北半部，沿[[郊野公園|郊野公園]]的邊界而行，跨過高314公尺的[[西灣山|西灣山]]，下[[吹筒坳|吹筒坳]]、[[西灣_(西貢)|西灣]]，經[[鹹田灣|鹹田灣]]，過[[大浪村|大浪村]]西走後登上大浪坳，經[[赤徑|赤徑]]抵[[北潭凹|北潭凹]]為終點。
# 此段從[[北潭凹|北潭凹]]開始，登上約400米的[[牛耳石山|牛耳石山]]後下抵300公尺的[[嶂上|嶂上]]，然後向西南走，經過[[畫眉山|畫眉山]]、[[雷打石|雷打石]]和[[雞公山_(西貢)|雞公山]]，下抵[[企嶺下|企嶺下]]通往[[西貢|西貢]]的[[西沙路|西沙路]]。
# 此段跨過[[馬鞍山郊野公園|馬鞍山郊野公園]]：穿過[[黃竹洋_(西貢北約)|黃竹洋村]]，向西攀上至[[馬鞍山_(香港山峰)|馬鞍山]]山脊（555米），抵[[昂平|昂平]]、[[茅坪|茅坪]]高地後攀上打瀉油坳及黃牛山、水牛山山腰，然後西行至位於[[飛鵝山|飛鵝山]]道的[[基維爾營地|基維爾營地]]。
# 此段是麥理浩徑最近市區的部分，比大部分段落的[[海拔|海拔]]為低。由[[基維爾營地|基維爾營地]]起，攀越[[大老山|大老山]]後西行山脊至[[沙田坳|沙田坳]]，再到達495米的[[獅子山_(香港)|獅子山]]的北面，經458米的[[畢架山|畢架山]]及[[鷹巢山自然教育徑|鷹巢山自然教育徑]]至[[大埔公路|大埔公路]]。
# 這是麥理浩徑中最短的一段，僅4.6公里，沿路為本地[[獮猴|獮猴]]最密集聚居地。由[[大埔公路|大埔公路]]開始，沿[[金山路|金山路]]到達走私坳，經[[孖指徑|孖指徑]]<ref>麥理浩理途經孖指徑路段與衛奕信徑不同</ref>至[[城門水塘|城門水塘]]。
# 這是麥理浩徑的次短段落。由[[城門水塘|城門水塘]]上[[針山|針山]]、[[草山_(香港)|草山]]，再下行至[[鉛礦凹|鉛礦凹]]。
# 這段跨越[[香港|香港]]最高的[[大帽山|大帽山]]。由[[鉛礦凹|鉛礦凹]]向西登上[[四方山|四方山]]後沿山脊而行，經四方山涼亭後再登上[[大帽山|大帽山]]山頂後下行至[[荃錦公路|荃錦公路]]。在2009年10月1日開始，大帽山道與川龍林道交界的路段不再直接經馬路，改經[[禾塘崗|禾塘崗]]山脊後到達[[荃錦公路|荃錦公路]]。
# 從[[荃錦坳|荃錦坳]]起步，一直緩緩下降，以距離[[大欖涌水塘|大欖涌水塘]]2公里的[[田夫仔|田夫仔]]為終點。
# 這是麥理浩徑最長段落。沿[[大欖涌水塘|大欖涌水塘]]北面行至西面引水道，沿此向下步至[[屯門|屯門]]目的地。由2009年10月1日起，部份路段將作永久改道。過了M174及大欖涌水塘紀念碑後，將不再經大欖涌水塘塘畔的小路，而改為繼續沿馬路前行，於一岔口左轉，經過伯公坳後，進入山路大約向西南方走，然後接上通往黃泥墩水塘的車路，下行至M186左右，接回舊路。

==毅行者==
{{main|毅行者}}

每年11月的第3個[[週末|週末]]，[[樂施會|樂施會]]均於麥理浩徑舉辦[[毅行者|毅行者]]。毅行者為[[香港|香港]]一項主要[[籌款|籌款]]活動。起點是大網仔路北潭涌閘口，終點原為於[[掃管軍營|掃管軍營]]，自2009年起更改為元朗[[保良局賽馬會大棠渡假村|保良局賽馬會大棠渡假村]]。參賽者需於48小時內行完全長100公里的麥理浩徑，最快完成時間約11小時。
