'''翁文灝'''（{{bd|1889年|7月26日|1971年|1月27日|catIdx=W翁}}），譜名存璋，[[表字|字]]詠霓，又[[表字|字]]永年，號君達，又號愨士。[[浙江省|浙江省]][[鄞縣|鄞縣]]石塘（今[[寧波市|寧波市]][[海曙區|海曙區]][[高桥镇_(宁波市)|高橋鎮]]石塘村）人，[[中華民國|中華民國]]政治人物，屬[[政學系|政學系]]，著名學者，[[輔仁大學|輔仁大學]]教授，是中國最早期的[[地質學|地質學]]家之一。對中國地質學教育、礦產開探、[[地震|地震]]研究等多方面有傑出貢獻。翁文灝曾以名學者之身份在[[國民政府|國民政府]]內任事，在[[抗戰|抗戰]]期間主管礦務資源與及生產。1948年更擔任行憲後首任[[中華民國行政院長|行政院長]]。不過在任期間推出[[金圓券|金圓券]]，引起金融混亂和惡性通脹，聲名大壞而下台。1948年12月被中共列为第12号[[战犯|战犯]]。1951年從[[歐洲|歐洲]]回到[[中國大陸|中國大陸]]，後以特別邀請人士身份成為第二届[[全國政協|全國政協]]委員等職。1971年病逝[[北京|北京]]。

== 早年經歷 ==
[[File:Ningbo_Weng_Wenhao_Guju_2013.07.28_06-23-31.jpg|thumb]]]]
[[File:Former_Residence_of_Weng_Wenhao_02_2012-09.JPG|thumb]]]]翁文灝出身紳商家庭，1902年13歲時通過鄉試中[[秀才|秀才]]。十八岁前往上海入学震旦公学，一年后得到浙江官费补助，前往欧洲留學<ref>1948年5月25日，《[[申报|申报]]》,第1张，翁文灏氏略历</ref>。1912年在[[比利時|比利時]][[天主教鲁汶大学|天主教鲁汶大学]]獲[[地質學|地質學]][[博士|博士]]。同年回到中國，在[[北洋政府|北洋政府]]農商部任事，並在地質研究所任講師、主任教授，中國首代地質工作者多出自其門下。翁文灝同時亦於[[北京大學|北京大學]]、[[清華大學|清華大學]]任教授，曾為清華地質學系主任、兼任代理校長。

翁文灝對中國[[地質學|地質學]]各方面研究皆有貢獻。他曾任[[北洋政府|北洋政府]]礦產股長，國民政府農礦部地質調查所長，研究中國各種礦產分佈。翁文灝首創多個中國第一：第一位地質學博士、第一個撰寫中國礦產志、編成第一張全國地質圖、第一位代表中国出席国际地质会议的地质学者、第一位对中国煤炭按其化学成分进行分类的学者、燕山运动及与之有关的岩浆活动和金属矿床形成理论的首创者、主導發現及開採中國第一個油田：[[玉門油田|玉門油田]]。

== 中年經歷 ==
1920年[[甘肅|甘肅]]發生8.5級地震，翁文灝亦是第一名考察研究的中國學者，事後出版中國地震區分佈圖，是中國首張地震區劃圖。中國第一個現代[[地震台|地震台]][[鹫峰地震台|鹫峰地震台]]是在翁的主事下，於1930年在[[北平|北平]][[西山_(北京)|西山]]建立。此外翁文灝對[[構造地質學|構造地質學]]、[[歷史地質學|歷史地質學]]都有研究貢獻。[[河北省|河北省]][[宛平縣|宛平縣]][[周口店|周口店]]出土[[北京猿人|北京猿人]]，亦是由翁所主導的新生物研究所所發現。翁文灝是[[中國地質學會|中國地質學會]]的創會會員之一，亦是[[中國地理學會|中國地理學會]]第一至十屆會長。翁無論在中國內外學術界皆享有聲譽，英國倫敦地質學會授與其榮譽會員，[[美國|美國]]、[[德國|德國]]、[[加拿大|加拿大]]各地大學及研究機構亦曾分別授與他榮譽學位或職位。

翁文灝早年表現不欲從政，曾被委為[[國民政府|國民政府]][[中華民國教育部|教育部]]部長但亦未受；所任政府公職皆與學術有關。1932年出任[[國民政府軍事委員會|軍事委員會]]國防設計委員會秘書長，但委員會內皆是學術界名人。1934年，翁於考察石油途中遇上嚴重車禍以致昏迷。得蔣中正關護，指令全力搶救，後來渡過危險。事後翁可能是難卻「救命之恩」，正式踏上仕途。1935年蔣自任行政院長，根据翁文灏日记自述，1936年5月23日任行政院秘书长。1938年元月三日，任经济部长。经蒋介石特别介绍，1938年4月19日填入党申请书，5月24日收到国民党党员证书，特字37254号。在抗戰期間主管中國的戰時工業生產及經濟建設。1945年當選[[中國國民黨|國民黨]]中央委員，並任行政院副院長，至1947年辭任。1947年4月23日，國民政府特任翁文灝為行政院政務委員，兼資源委員會委員長。<ref name="民國史大事記">{{cite book|editor=李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編|title=《中華民國史大事記》|location=[[北京|北京]]|publisher=[[中華書局|中華書局]]|date=2011-07}}</ref>{{rp|8340}} 

1948年榮獲中華民國[[中央研究院|中央研究院]]第一屆(數理科學組)院士<ref>{{cite news|title=逝世院士一覽表 |url=https://academicians.sinica.edu.tw/index.php?func=1-D |publisher=中央研究院|date= |author= |accessdate= |language=zh-hant}}</ref>。

1948年5月，[[蔣介石|蔣介石]]以翁文灝為[[行政院長|行政院長]]。<ref name="年表">{{cite book|authors=陳布雷等編著|title=《蔣介石先生年表》|location=台北|publisher=[[傳記文學|傳記文學]]出版社|date=1978-06-01}}</ref>{{rp|55}}5月27日，行政院長翁文灝飛至上海，專程訪晤何應欽和吳國楨等人，聽取對組閣之意見，並接洽[[中國民主社會黨|中國民主社會黨]]、[[中國青年黨|中國青年黨]]兩黨負責人；晚上，翁文灝又赴[[顧孟餘|顧孟餘]]宅，請其出任行政院副院長。<ref name="民國史大事記"/>{{rp|8610}}翁文灝任內主持貨幣改革，在8月推出[[金圓券|金圓券]]取代[[法幣|法幣]]，並以行政方法意圖控制物價，結果造成金融失調。[[翁文灝內閣|翁文灝內閣]]於11月總辭。12月1日，行政院會議通過非軍事機關分遷於重慶、廣州兩地辦公計劃。<ref name="民國史大事記"/>{{rp|8741}}

次年1月，蔣中正下野。2月，翁文灝任代總統[[李宗仁|李宗仁]]之[[總統府秘書長|總統府秘書長]]。至5月，李宗仁主事與[[中共|中共]]和談失敗，翁辭去秘書長之職，並出國至法國。

== 晚年經歷 ==
1951年，翁經[[香港|香港]]回到中國大陸，是曾經擔任[[中華民國政府|中華民國政府]]高級官員中首名回北京的官員。翁在1948年曾被[[中國共產黨|共產黨]]列為戰犯，回到大陸後初期曾因不積極批[[蔣中正|蔣]]而受壓。1954年後方獲任[[政協委員|政協委員]]，之後主要從事翻譯及學術研究。[[文革|文革]]中得特別保護而只受少許衝擊。1971年病逝於北京。

翁文灏的墓地原位于[[北京市|北京市]][[八宝山革命公墓|八宝山革命公墓]]内，后根据其家属们按照翁的生前遗愿，将其骨灰移葬于北京万安公墓中其夫人林韵秋墓侧。

==家庭==
翁文灝有四子。大兒子[[翁心源|翁心源]]是著名石油工程師，1970年在[[文革|文革]]中被迫害至死。次子[[翁心翰|翁心翰]]為[[中華民國空軍|中華民國空軍]]機師，於[[第二次中日戰爭|抗日戰爭]]中殉國。

中國[[地球物理學|地球物理學]]奠基人、[[中国科学院|中国科学院]][[院士|院士]][[翁文波|翁文波]]是翁文灝的堂弟。[[中国工程院|中国工程院]]院士[[翁心植|翁心植]]是其侄子。[[美国|美国]]钛合金专家[[翁心梓|翁心梓]]亦是其侄子。
