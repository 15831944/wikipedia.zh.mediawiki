'''翁丁村'''（[[佤语|佤语]]：am diem或om diem，意为连接之水，引申为水边的寨子）是[[云南省|云南省]][[临沧市|临沧市]][[沧源佤族自治县|沧源佤族自治县]]的一个行政村，隶属于[[勐角傣族彝族拉祜族乡|勐角傣族彝族拉祜族乡]]，附近建有[[翁丁水库|翁丁水库]]。共有村民四百多人，因[[佤族|佤族]]的聚落而闻名，村民多信仰[[赛玛教|赛玛教]]，现成为沧源县的著名旅游景区，被[[中国国家地理|中国国家地理]]称为“佤族传统文化保留最完整的村寨”。2015年，翁丁村被列入“第三批全国特色景观旅游名镇名村”。翁丁村也是一个3A级旅游景区，名称为“临沧沧源翁丁原始部落文化旅游区”。<ref name="国家地理">{{cite web |url=http://www.dili360.com/article/p54b76e1a4725535.htm |title=翁丁佤族原始部落的慢生活 |last=邱 |first=太建 |date=2015-01-16 |website=www.dili360.com |publisher=中国国家地理 |access-date=2017-10-22 |quote=}}</ref><ref>{{Citation |last=陈 |first=燕 |author-link= |year=2008 |title=沧源翁丁村佤族饮食旅游资源开发 |periodical=边疆经济与文化 |volume= |issue=第12期 |pages=P22-P23 |url=http://ishare.iask.sina.com.cn/f/37647442.html |doi= |oclc= }}</ref><ref>{{cite web |url=http://www.mohurd.gov.cn/wjfb/201507/W020150729104443.doc |title=第三批全国特色景观旅游名镇名村示范名单 |last= |first= |date=2015-08-04 |website=www.mohurd.gov.cn |publisher= [[中华人民共和国住房和城乡建设部|中华人民共和国住房和城乡建设部]] |access-date=2017-12-13 |quote=}}</ref><ref>{{cite web |url=http://www.ynta.gov.cn/Item/33089.aspx |title=云南省A级旅游景区名录 |author=李齐凡 |date=2017-07-28 |website=www.ynta.gov.cn |publisher=云南省旅游发展委员会 |access-date=2018-01-06 |quote=}}</ref>

翁丁村保留有原始的佤族建筑——[[干栏式|干栏式]]茅草顶的“叉叉房”，沿袭着上面住人，下面关猪、关牛的生活习惯，还有曾经祭祀使用的人头桩和牛头桩，当地[[猎首|砍人头]]祭旱谷的习俗直到1956年才被全部禁止。<ref name="国家地理"></ref><ref>{{cite web |url=http://iel.cass.cn/ztpd/ddlt/wyb/200810/t20081031_2763893.shtml |title=翁丁村之旅 |last=袁 |first=智中 |date=2008-10-30 |website=中国民族文学网 |publisher=文艺报 |access-date=2017-10-22 |quote=}}</ref>
