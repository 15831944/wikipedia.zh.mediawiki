'''翁布利亚'''（{{lang-it|Umbria}}，{{IPA-it|ˈumbrja}}），位于意大利中心，首府[[佩鲁贾|佩鲁贾]]。面积8,456平方公里——其中[[佩鲁贾省|佩鲁贾省]]（Perugia）6,334平方公里；[[特尔尼省|特尔尼省]]（Terni）2,122平方公里——是意大利-{面}-积第五小的大区，也是意大利唯一不临海也不与外国接壤的大区。与[[马尔凯|马尔凯]]、[[托斯卡纳|托斯卡纳]]和[[拉齐奥|拉齐奥]]大区毗邻。人口834,210（2003年统计）。其他城市有[[阿西西|阿西西]]（Assisi）、[[卡斯泰洛城|卡斯泰洛城]]（Città di Castello）、[[福利尼奥|福利尼奥]]（Foligno）、[[古比奥|古比奥]]（Gubbio）、[[诺尔恰|诺尔恰]]（Norcia）、[[奥尔维耶托|奥尔维耶托]]（Orvieto）、[[斯波莱托|斯波莱托]]（Spoleto）、[[托迪_(意大利城市)|托迪]]（Todi）等。

翁布利亚一词原指于公元前6世纪时迁徙至此的翁布利人，使用[[翁布里亚语|翁布利语]]（与[[拉丁语|拉丁语]]有语源关系）。现在所指的翁布里亚不同于[[古罗马|古罗马]]时期同一名称所指区域，后者包含今[[马尔凯|马尔凯]]北部，直至[[拉维娜|拉维娜]]；但不含[[台伯河|台伯河]]西岸、[[佩鲁贾|佩鲁贾]]地区（属[[埃特鲁利亚|埃特鲁利亚]]人）和[[诺尔恰|诺尔恰]]地区（属[[萨宾人|萨宾人]]）。

== 地理 ==
翁布里亚多山和[[丘陵|丘陵]]，[[亚平宁山脉|亚平宁山脉]]贯穿全区，意大利第四大湖——[[特拉西梅诺湖|特拉西梅诺湖]]（Lago Trasimeno）位于其中。全区最高点位于与[[马尔凯|马尔凯]]交接海拔2,476米的维托雷峰（Mt. Vettore），最低点位于[[台伯河|台伯河]]谷[[盆地|盆地]]的阿提亚诺（96米）。全区植被丰富常青，自然环境优美，素有意大利的绿色心脏之称（Il Cuore Verde d'italia，意大利诗人[[卡尔杜齐|卡尔杜齐]]语）。

== 历史 ==
最早居于此地的居民当属翁布利人和埃特鲁利亚人。公元前295年在Sentino战役后被罗马人征服。在[[第二次布匿战争|第二次布匿战争]]期间，[[汉尼拔|汉尼拔]]在特拉西梅诺湖畔打败罗马大军。随后公元前40年，在[[马克·安东尼|马克·安东尼]]和[[屋大维|屋大维]]的内战期间，埃特鲁利亚时期名城重镇[[佩鲁贾|佩鲁贾]]遭到毁灭。

在[[西罗马帝国|西罗马帝国]]灭亡之后，该地一直处于[[东哥特|东哥特]]和[[拜占庭|拜占庭]]的争夺之中，而[[伦巴第人|伦巴第人]]571年建立[[斯波莱多公国|斯波莱多公国]]（Ducato di Spoleto）。13世纪中叶，[[卡罗大帝|卡罗大帝]]征服了大部分该地区，后献给[[教皇|教皇]]，后通过兼并一些小[[僭主|僭主]]的领土，在14世纪时逐渐形成[[教皇国|教皇国]]直至18世纪。随着当时[[法国大革命|法国大革命]]席卷全[[欧洲|欧洲]]，该地成为[[罗马共和国|罗马共和国]]（1789—1799, la Repubblica Romana）的一部分，[[拿破仑|拿破仑]]称帝后，为拿破仑帝国（1809—1814）的属地，1860年并入[[意大利王国|意大利王国]]。

== 重要历史遗存 ==

在翁布里亚国立考古博物馆（Museo archeologico nazionale dell'Umbria）和位于奥利维托的“克劳迪奥·法伊娜”博物馆（Museo Clasudio Faina）中收藏有数十件翁布里亚地区史前文物，其中最为著名的是一件出土于特拉西梅诺湖畔、取名为“特拉西梅诺的维纳斯”的石雕，可上溯至旧石器时代晚期。除此之外，还该地区还曾发掘过数个墓葬，年代从青铜时代至铁器时代。

== 经济 ==
该地区主要经济支柱为工业、农业和旅游业。农业方面以[[橄榄油|橄榄油]]、[[葡萄酒|葡萄酒]]的生产为主，工业方面则以机械、建筑材料生产为大宗。

受地理环境所限，发展速度与周围其它地区相比较缓。
