'''板頭村'''位於[[臺灣|台灣]][[嘉義縣|嘉義縣]][[新港鄉|新港鄉]]的西北邊，是新港鄉轄下的行政區。面積約1.790189平方公里，包含13鄰，總人口數640人，為具有[[交趾陶|交趾陶]]特色的聚落。

== 地理 ==
[[雲林縣|雲林縣]][[北港鎮_(台灣)|北港鎮]]和[[嘉義縣|嘉義縣]][[新港鄉|新港鄉]]南港村這一帶，舊名「笨港」。1731年([[雍正|雍正]]九年)，[[福建|福建]][[總督|總督]]劉世明上奏添設[[縣丞|縣丞]]一名，因而有了笨港[[縣丞署|縣丞署]]。[[縣丞署|縣丞署]]是最早設在笨港街的磚仔窯，1734年([[雍正|雍正]]12年)遷至[[北港溪|北港溪]]南岸的板頭厝，就是現在新港鄉的板頭村。<ref>{{Cite web|url=https://www.tbocc.gov.tw/SightLib/Sight_Detail.aspx?id=a686db1e-64d4-e411-a5d3-e4115b13f301&page=1&&SearchPagSize_Data=10&SearchDisplay=close&SelectData_KeyWord=all&SelectData_Place=3b80450b-2cd1-e411-a5d3-e4115b13f301&SelectData_SightClass=all&lang=tw|title=古笨港縣丞署|accessdate=2016-04-19|author=|date=|publisher=}}</ref>

== 聚落歷史 ==
[[板頭厝|板頭厝]]車站在1911年(明治44年)日本[[製糖株式會社|製糖株式會社]]所架設的營業鐵道停靠站，調度火車往蒜頭。1982年糖廠停止客運業務，1998年停止貨運後，於2001年拆除木造車站，現在只留下空地與鐵軌。

== 教育 ==
板頭村沒有學校，小學生到福德村的新港國小就學<ref>{{Cite web|url=http://www.hkps.cyc.edu.tw/modules/tadnews/index.php?nsn=816|title=新港國小|accessdate=2015-02-26|author=|date=|publisher=}}</ref>，國中生則到福德村新港國中就學<ref>{{Cite web|url=http://ws-tm.cyhg.gov.tw/001/Upload/1348/relfile/12453/136674/%E5%98%89%E7%BE%A9%E7%B8%A3%E5%9C%8B%E4%B8%AD%E5%AD%B8%E5%8D%80%E5%8A%83%E5%88%86%E8%A1%A81050801.pdf|title=嘉義縣國民中學學區劃分表|accessdate=105-08-01|author=|date=|publisher=}}</ref>。

== 文化資源 ==
笨港長天宮，自1734年(清雍正十二年)至1817年([[嘉庆|嘉慶]]二十二年)，為笨港統治中樞所在。[[嘉慶|嘉慶]]年間(1792年-182年)，洪水氾濫，長天宮所幸留存，廟中香火鼎盛，[[笨港縣丞|笨港縣丞]]及[[清朝鳳山縣知縣列表|鳳山縣知縣]]曾共獻「山海咸寧」之匾<ref name="auto">{{Cite web|url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/shingang/1007022-CTG|title=笨港長天宮|accessdate=2012-12-01|author=|date=|publisher=}}</ref>，而長天宮牌更有「[[祺祥|祺祥]]」壬戊年(1862年)之記，其為史民所崇仰，其歷史之久遠於此可見。但由於廟宇建築有一定的年代，再加上缺乏管理維修，決定重建。<ref name="auto"/>

== 景點 ==
位於板頭村[[北港溪|北港溪]]畔河堤上的大壁畫，用在地交趾陶及剪黏藝術裝置。逐漸沒落的小村莊，但因為這三個人的努力：[[陳忠正|陳忠正]]先生、[[黃水水|黃水水]]先生、[[陳明惠|陳明惠]]先生，讓板頭社區展現出不一樣的風貌，也成為臺灣以產業帶動社區改造成功實例。<ref>{{Cite web|url=http://www.tbocc.gov.tw/SightLib/Sight_Detail.aspx?id=a686db1e-64d4-e411-a5d3-e4115b13f301&page=1&&SearchPagSize_Data=10&SearchDisplay=close&SelectData_KeyWord=all&SelectData_Place=3b80450b-2cd1-e411-a5d3-e4115b13f301&SelectData_SightClass=all&lang=tw|title=古笨港縣丞署|accessdate=2016-04-29|author=|date=|publisher=}}</ref>

== 其他特色 ==
