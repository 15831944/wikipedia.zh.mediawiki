'''板橋區'''（'''枋橋'''、'''擺接'''，{{lang-nan|Pang-kiô-khu}}；{{lang-hak|Piông-khièu-khî}}）為[[台灣|台灣]][[新北市|新北市]]的一個[[區_(中華民國)|區]]，是[[新北市政府|新北市政府]]所在地，也是新北市首善之區。板橋因地處新北市主要人口商業腹地的中心位置，在重大公共建設的投入下，近年來已由[[台北都會區|台北都會區]]主要[[衛星都市|衛星都市]]之一，進而逐漸成為新北市與[[淡水河|淡水河系]]西岸的[[CBD|政經中心]]。板橋[[林本源園邸|林本源園邸]]（林家花園）為境內頗負盛名的名勝古蹟，與近來快速發展的城市面貌相互呼應，形成新舊並蓄、宜古宜今的城市景緻。

板橋區總面積為23.14[[平方公里|平方公里]]，全境為適宜開發人居的平原地形並有河川環繞，其中河域面積廣達686[[公頃|公頃]]，行政區劃分為126[[村里|里]]、2509[[鄰|鄰]]。全區人口約55.5萬，在全國368個鄉、鎮、市、區中排名第一，甚至超越全國半數以上縣市之人口數<ref>[[臺灣行政區人口列表#區排序|臺灣行政區人口列表]]</ref>。

== 詞源 ==
板橋原為[[平埔族|平埔族]][[原住民|原住民]][[擺接社|擺接社]]聚居地，後來[[漢人|漢人]]在17世紀後逐漸遷入並形成聚落。在[[大漢溪|大漢溪]]船運興盛時，商旅貨物在「湳仔港」上岸運往「擺接社」[[閩南人|閩南人]]聚落，途中需經[[林家花園|林家花園]]後方的「公館溝」。為方便人貨通行，其上興建了一座木板橋，當地因而被稱為「'''枋橋頭(pang-kiô-thâu)'''」。後來的「枋橋城」即沿著公館溝東南岸興建城牆，在木板橋前設置西城門，約為今府中路、林園街口。近年來公館溝經整治加蓋後，橋已經不復見<ref>《板橋區志》</ref><ref>《台灣堡圖》，1904年</ref>。

[[台灣日治時期|日治時期]]時，將「枋橋」改為日式地名'''板橋'''，讀法為「{{lang|ja|いたはし}}」（itahashi）<ref>{{cite web|language=zh-tw|url=http://mapstalk.blogspot.tw/2010/09/blog-post_30.html|title=葉高華 臺灣日式地名新論}}</ref>，先為'''板橋-{庄}-'''，1929年升格為'''[[板橋街|板橋街]]'''。[[二次世界大戰|二次世界大戰]]後[[國民政府|國民政府]]延續此名，改為'''板橋鎮'''，隸屬台北縣，成為台北縣之[[縣治|縣治]]；1972年7月1日升格為[[縣轄市|縣轄市]]'''板橋市'''；2010年12月25日隨著台北縣改制為新北市而隨之改制為'''板橋區'''。

== 地理 ==
板橋區位於新北市中西部，為整個[[大臺北|大臺北]][[淡水河|淡水河]]以西的中心城市，北以[[淡水河|淡水河]]、[[大漢溪|大漢溪]]與[[三重區|三重區]]、[[新莊區|新莊區]]為界，東北與[[臺北市|台北市]][[萬華區|萬華區]]隔[[新店溪|新店溪]]相望，西接[[樹林區|樹林區]]，南與[[土城區|土城區]]、[[中和區|中和區]]毗鄰。幅員略呈東北- 西南走向，東北部較寬闊，西南部則較為狹長。總面積23.1368平方公里，河域面積廣達686公頃，人口55.2萬人，居全國市轄區及縣轄鄉、鎮、市之冠。

全區地勢平坦，無山嶽及丘陵。東南部略高，但海拔僅約10公尺，漸次向西北部趨低，至海拔約2公尺。其地質時期全屬第四紀[[完新世|完新世]]完新統之沖積層。但板橋街區及新埔、埔墘、深丘、後埔以東一帶，地勢較高，可謂之為老沖積層，形成較早，可能為地質時期的[[台北湖|台北湖]]之沈積<ref>[https://www.banqiao.ntpc.gov.tw/content/?parent_id=10065 地理生態]，板橋區公所 </ref>。 

板橋區最主要的河川為[[大漢溪|大漢溪]]與[[新店溪|新店溪]]。大漢溪為[[淡水河|淡水河]]主流上游，也是本市最大河川，流經[[浮洲|浮洲]]地區後，以弓狀形成本區西北邊界，至東北端[[江子翠|江子翠]]地區與新店溪相會。新店溪為淡水河最大[[支流|支流]]，流經板橋區東北邊界。

=== 水文 ===
[[台灣日治時期|日治時代]]以前板橋是個有許多溝渠的地方，現在已經大都覆蓋成為下水道。

[[湳仔溝|湳仔溝]]是分隔板橋與[[浮洲|浮洲]]地區的河道溝渠，湳仔即當地地名湳雅的[[閩南語|閩南語]]原文。湳仔溝本來是大漢溪之分流河道，1982年因應防洪堵絕大漢溪上游的水流入，只剩其他溝渠與地下水匯入。湳仔溝最後流向大漢溪。

[[公館溝|公館溝]]為板橋地區僅次於湳仔溝的最大溝渠，也可能是板橋一名之由來。現在已經大都成為地下排水溝。水脈發源於土城山區，19世紀曾經成為[[林本源園邸|林本源園邸]]城牆外的護城河。該溝渠約略從[[土城|土城]]青雲路進入板橋信義路、四川路、館前西路，館前西路即為[[林家花園|林家花園]]城牆之南側。沿著館前西路，流過接雲寺及[[湳雅夜市|湳雅夜市]]入口，轉往府中路、西門街，在北門街北流，離開原本城牆的範圍。最後約略經過公館街、國光路、智樂路、中正路、新海抽水站，匯入大漢溪。<ref>[https://maps.google.com.tw/maps/ms?ie=UTF8&oe=UTF8&msa=0&msid=101997132199390654494.00046be8034f8096bf545 繪製在Google Map上的公館溝河道圖。作者：吳澤山]</ref><ref>《枋橋古城遊》地圖，[[枋橋文化協會|枋橋文化協會]]監事[[莊文毅|莊文毅]]繪製，2014年。</ref>

公館街即以公館溝命名。現在只有館前西路旁一區段無覆蓋，可見到黑水溝，且縣民大道經過館前西路和華福街間之路段，有明顯隆起之橋樑特徵。

== 歷史 ==
板橋區自古即為[[凱達格蘭族|凱達格蘭族]]原住民[[武朥灣社|武朥灣社]]和[[擺接杜|擺接杜]]的生活範圍。

[[清|清]][[康熙|康熙]]年間始有[[漢人|漢人]]足跡，根據板橋「新埔賴氏族譜」記載，[[清|清]][[康熙|康熙]]六十年（1721年），[[漳州|漳州]][[平和縣|平和縣]]葛竹鄉賴天經族人，渡海至現[[板橋區_(新北市)|板橋]][[新埔里_(新北市)|新埔]]地區開墾。

[[乾隆|乾隆]]年間廖啟樁、林成祖入墾擺接平原，隨後大量[[閩南人|閩南人]]移住，人口漸增。其時為「擺接堡」之行政中心，轄地遍及今日板橋區、[[中和區|中和區]]、[[土城區|土城區]]和[[樹林區|樹林區]]、[[新莊區|新莊區]]西盛地區。

[[道光|道光]]年間[[林本源家族|林本源家族]]自新莊遷居此地，更使板橋區形成鄰近地區的宗教、商業、文化中心<ref>[https://www.banqiao.ntpc.gov.tw/content/?parent_id=10067 板橋憶往]，板橋區公所 </ref>。[[大正|大正]]9年台灣行政區改制為五州三廳，板橋為[[海山郡|海山郡]]役所的所在地，是郡之行政中心。

[[台灣日治時期|日治時期]]，為[[海山郡|海山郡]]之中心，稱海山郡板橋街。[[二次大戰|二次大戰]]結束，[[日本|日本]]撤離，[[國民政府|國民政府]]接收台灣之後改稱[[新北市|臺北縣]]板橋鎮，為縣治所在，1972年7月1日升格為[[縣轄市|縣轄市]]。

2010年12月25日，[[2010年中華民國縣市改制直轄市|五都改制]]案實行，臺北縣改制為[[新北市|新北市]]，遂稱板橋區。目前板橋區為新北市政府所在地，為新北市的政治經濟中心。

== 經濟 ==
板橋區已逐漸成為[[新北市|新北市]]的政經中心，其工商服務業相對發達，為知名連鎖餐飲、金融機構…各大服務性業種進駐新北市的首要據點。

位於板橋區中央的[[新板橋車站特定專用區|新板特區]]，約在1990年代左右開始整地開發，建立臺灣第一個[[共構與共站|三鐵共站]]複合式車站，強化板橋在[[臺北都會區|臺北都會區]]西側交通樞紐的地位。陸續將再引進數座大型複合式購物中心、商場、高級商辦、[[大台北新劇院|劇院]]、會展中心、華劇製作中心(至2014年共會興建5個)、[[CNN|CNN]]海外攝影棚<ref>《[[爽報_(台灣)|爽報]]》Charm，2013年3月1日第7版</ref>及五星級觀光飯店，與現有的新北市政府及政務機關結合，帶動消費及商務位階的提昇。

[[台北遠東通訊園區|台北遠東通訊園區T-Park]]在2007年起實質快速開發，除了將陸續牽動電信服務、科技研發產業進駐外，區內[[亞東紀念醫院|亞東醫學中心]]擴建、鄰近的[[中華電信雲端運算中心|中華電信雲端運算中心]]也將開發，整體預計將進一步帶來高達2萬名白領科技業、醫療產業的就業人口。

傳統上板橋較未發展的[[浮洲|浮洲]]地區，因有歷史悠久、藝文人才輩出的[[國立臺灣藝術大學|國立臺灣藝術大學]]落腳，且鄰近板橋市區[[林本源園邸|林家花園]]等具人文特色的歷史背景，使得文創產業的蘊育成為浮洲地區的重點發想，然而近來浮洲[[合宜住宅|合宜住宅]]政策卻轉變成快速導入大量居住人口。

== 政治 ==
[[File:Taipei_County_Hall_20100208.jpg|thumb]]]]
[[File:Taipei_County_Council_Hall_front_view_20070508.jpg|thumb]]]]

板橋區為新北市行政中心，地處新北市溪南最接近於溪北的中間區域，此一地緣特色反應出來的政治選票板塊，使得板橋歷來呈現[[泛藍|泛藍]]、[[泛綠|泛綠]]勢力相當的中間多元屬性。與新北市所轄的行政區相較，板橋不若[[三重區|三重區]]、[[新莊區|新莊區]]、[[蘆洲區|蘆洲區]]的偏綠，亦不若[[中和區|中和區]]、[[永和區|永和區]]及[[新店區|新店區]]的偏藍，加以擁有新北市南來北往的中心樞紐位置和最大票倉，向來是兵家必爭之地、競選總部的所在地。

===行政區===
板橋區有十個非正規的次分區，11個聯合里。次分區與其大致的涵蓋範圍包括[[板橋府中商圈|府中]]（範圍包括板橋市區聯合里下的5個里）、[[江子翠|江翠]]（江翠聯合里下屬的27個里與港嘴聯合里下屬的4個里）、[[新埔里_(新北市)|新埔]]（新埔聯合里下屬的11個里）、[[新板特區|新板]]（又常稱為「深丘」，範圍包括深丘聯合里下屬的8個里）、[[台北遠東通訊園區|亞東]]（又常稱為「湳子」，主要涵蓋湳興聯合里下屬的8個里）、[[埔墘|埔墘]]（埔墘聯合里下屬的15個里）、[[後埔|後埔]]（廣福聯合里下屬的15個里）、[[社後|社後]]（社後聯合里下屬的16個里）、[[浮洲|浮洲]]（浮洲聯合里下屬的11個里）、[[溪崑|溪崑]]（溪州聯合里下屬的6個里）。其中，府中、江翠、新埔、新板與亞東等幾個地區的中心位置分別都設置有台北捷運板南線的車站，是板橋區內交通較為便利的區段。

板橋全區有126個里，是全國里數最多的市轄區，也是全國下轄最多里（村）的鄉鎮市區級行政區。

== 交通 ==

板橋區為[[新北市|新北市]]政經中心，地理上亦為新北市人口密集帶的中心，都市定位及南來北往的中心地理位置，使得板橋擁有頗為發達的聯外路網及發達的大眾運具。其中有四座東西向橋樑（[[華江橋|華江橋]]、[[萬板橋|萬板橋]]、[[華翠橋|華翠橋]]、[[光復橋|光復橋]]）直通臺北市；南北向則有[[新北環河快速道路|新北環快]]、[[臺64線|臺64線]]（特一號）及[[臺65線|臺65線]]（特二號）三大[[快速道路|快速道路]]貫穿板橋市區並連結[[中山高速公路|國道一號]]及[[福爾摩沙高速公路|國道三號]]；[[板南線|捷運板南線]]在板橋區設有五站（[[江子翠站|江子翠站]]、[[新埔站_(新北市)|新埔站]]、[[板橋車站_(臺灣)|板橋站]]、[[府中站|府中站]]、[[亞東醫院站|亞東醫院站]]）與北市主要精華區（[[西門商圈|西門商圈]]、[[博愛特區|博愛特區]]、[[站前商圈|站前商圈]]、[[東區商圈|東區商圈]]、[[信義商圈|信義商圈]]、[[南港經貿園區|南港經貿園區]]）直接串連；連接臺灣西部走廊主要都會區的[[台灣高速鐵路|高鐵]]及大小都市的[[臺鐵|臺鐵]]和短中長客運中心亦在板橋設站，使得板橋區堪稱是新北市的交通樞鈕。未來串連新北市發達區的[[環狀線_(台北捷運)|捷運環狀線]]第一階段（西環段）亦以板橋為中心，並於區內設置三個捷運站（[[板新站|板新站]]、[[板橋車站_(臺灣)|板橋站]]、[[新埔民生站|新埔民生站]]）。

==== 橋樑 ====
由於境內河域廣闊，板橋區境內共有19座主要聯絡橋樑，板橋區之北、西面為大漢溪及其支流，東面為新店溪，而南面以陸地連接[[中和區|中和區]]、[[土城區|土城區]]，主要以圳溝為界，沒有橋樑。因此板橋區可說是台灣所有鄉鎮市區中，大型聯絡橋樑最多者。

板橋區的東面有四座跨越新店溪的橋樑，全部通往台北市萬華區：
*[[光復橋|光復橋]]：[[市道114號|市道114號]]，為板橋通往萬華歷史最早的橋樑，日治時期為吊橋，稱「昭和橋」，後改建為斜張橋，中山路二段→[[臺北市|台北市]][[萬華區|萬華區]]西園路二段。
*[[華江橋|華江橋]]：台三線，是繼光復橋之後為第二座興建的橋樑，今天的華江橋是第二代的橋樑，[[文化路_(板橋區)|文化路]]二段→[[臺北市|台北市]][[萬華區|萬華區]]和平西路三段。
*[[萬板橋|萬板橋]]：為第三座興建的橋樑，也是四座橋樑中高度最高的(因台北端橋下有抽水站)，但卻是交通量最少的一座，萬板路→[[臺北市|台北市]][[萬華區|萬華區]]西藏路，。
*華翠橋：原為縱貫鐵路跨越新店溪橋樑，鐵路地下化完成之後，改建為公路橋樑，本橋為汽車專用，無機車道與自行車道，至今交通量甚至勝於華江橋，[[縣民大道_(新北市)|縣民大道]]三段→[[臺北市|台北市]][[萬華區|萬華區]]艋舺大道。
西面11座：三座為區內聯絡橋樑，一座鐵路橋(高鐵與台鐵共構)，兩座通往[[樹林區|樹林區]]，一座通往[[土城區|土城區]]
*浮洲橋：[[市道114號|市道114號]]，聯絡板橋浮洲與溪洲兩地區以及樹林的交通，橋下為大漢溪主流，大觀路二段→大觀路三段(樹林陸橋)
*南興橋：[[市道114號|市道114號]]，聯絡板橋舊市區（湳子）與浮洲的交通，橋下為大漢溪支流湳子溝，館前西路→大觀路一段
*新興橋：[[市道116號|市道116號]]，又稱浮洲二橋，橋下為大漢溪支流湳子溝，聯絡四汴頭與浮洲的交通，在城林橋通車之前，亦背負聯絡土城與樹林、新莊之間的交通。
*樹林陸橋：[[市道116號|市道116號]]，聯絡樹林與板橋溪洲之間的交通，為鐵路跨越橋，但其下亦有水圳河道通過，大觀路三段50巷(浮洲橋)→[[樹林區|樹林區]]中正路
*沙崙橋：[[市道114號|市道114號]]，聯絡板橋溪洲與樹林市區之間的交通，十五座主要橋樑內為最小的一座，大觀路三段→樹林區保安街。
*城林橋：聯絡樹林區、板橋區溪洲地區與土城區的交通，板林路→溪城路→[[土城區|土城區]]城林路
*（大漢溪）鐵路橋：其實是兩座橋，跨越大漢溪，高鐵橋與台鐵橋毗鄰而立。

北面有四座橋樑，其中三座通往[[新莊區|新莊區]]，一座通往[[三重區|三重區]]：
*重翠橋：屬於快速道路橋樑，[[台64線|台64線]]和新北環快共構，與[[新北大橋|新北大橋]]直接連通，有機車道與自行車道，常被誤認為是新北大橋的一部份，通往三重區。
*大漢橋：[[市道106號|市道106甲]]，為北面通往新莊區的交通主動脈，民生路三段→新莊區思源路。
*新海橋：[[市道106號|市道106號]]，在大漢橋興建之前，為唯一的北面通往新莊、三重的聯絡橋樑。在大漢橋通車之後，逐漸被取代，但至今仍然地位重要。
*[[台65線|台65線]]大漢溪跨越橋，屬於快速道路橋樑，通往新莊區，也有機車道（板新機車聯絡道橋）與自行車道。
