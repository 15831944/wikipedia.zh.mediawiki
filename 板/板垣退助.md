'''板垣退助'''（[[天保|天保]]8年4月17日（1837年5月21日）—[[大正|大正]]8年（1919年）7月16日），[[日本|日本]][[明治維新|明治維新]]的功臣之一，也是日本[[自由民權運動|自由民權運動]]家、日本第一個[[政黨|政黨]][[自由黨_(日本1881年)|自由黨]]的創立者。[[土佐藩|土佐藩]]出身。位階勲等是[[從一位|從一位]]{{link-ja|勲一等旭日桐花大綬章|勲一等旭日桐花大綬章|勳一等}}，[[伯爵|伯爵]]爵位。

== 生平 ==

[[File:Itagaki_Taisuke_young.jpg|thumb]]

1837年5月21日，板垣退助生於[[日本國|日本國]][[土佐藩|土佐藩]]上級武士之家。[[板垣氏|板垣氏]]本姓[[乾氏|乾氏]]，據說出於[[源氏|源氏]]後裔。初諱“正躬（{{lang|ja|まさみ}}）”，后改“正形（{{lang|ja|まさかた}}）”，幼名“猪之助（{{lang|ja|いのすけ}}）”。通稱“退助（{{lang|ja|たいすけ}}）”，“無形（{{lang|ja|むけい}}）”是其号。

{{japanese|japanese=板垣 退助|kana=いたがき たいすけ|romaji=Itagaki Taisuke}}
{{Liberalism}}

1860年，板垣退助參加藩政，曾參加[[討幕運動|討幕運動]]，主張武力倒幕。[[慶應|慶應]]3年（1867年）進入明治政府。在[[征韓論|征韓論]]之爭中屬於支持征韓派，在該派失敗後於1873年離開明治政府。

1874年板垣退助與其他七位人士向政府提出民選[[議院|議院]]成立的建言書，使得追求議會制度以及憲法制定的[[自由民權運動|自由民權運動]]起跑。1881年與其同志共組[[自由黨|自由黨]]。1882年4月，板垣退助至[[岐阜縣|岐阜縣]]演說時遭人[[刺殺|刺殺]]，據稱板垣當時曾大喊“''板垣雖死，自由不滅''”（{{lang|ja|板垣死すとも自由は死せず}}）<ref>日本岐阜公園碑文曰「我岐阜公園者、為自由党總理板垣退助先生遭難之霊地。明治十五年四月六日先生遊説之途次、東濃来臨。中教院之有志、懇親會為大演説。将出玄関、突如有凶漢利刃一閃、刺先生之左胸。先生一聲大喝曰『板垣雖死自由不死』一語。凛然永照青史矣。大正六年、予等同志胥謀建設銅像。迎先生及夫人令息、擧除幕式。爾来、毎歳行遭難記念祭。誰圖為大東亞戰之犠牲遭没收之厄。遺憾、無窮這回有志協力再建焉。感喜何堪難、賦一詩記其顛末。欲自由民権、不省身東奔西走而説教綸。英姿颯爽、崋山麓立像。永留其姿眞之」。</ref>。1887年，板垣退助被封為伯爵。1900年退休。

1914年（大正3年）3月，板垣退助曾應[[林獻堂|林獻堂]]之邀赴[[台灣|台灣]]發表演說，並且參與了[[台灣同化會|台灣同化會]]的成立。1914年3月19日，板垣退助在歡迎會上致答辭說：「-{余}-想，在亞洲，究竟不能以日本一國來擔負全盤之國防（此指[[區域安全|區域安全]]）責任；是故陸上軍備可委託支那（中國），海-{面}-軍備可由日本來擔當，然後始能期其堅實。是以有和支那人締結親交之必要。而要開其端緒，（若）捨本島（台灣），實無他法可想。」後板垣南下考察，發表演講：「世界為人類之共有物。將人口由過多之地移向稀薄之地，此乃自然之法則。美國人之暴論，任誰亦不能與之相容。因此，日本人作為亞細亞的一份子，非與支那提攜共同抵禦白人不可。我台灣最接近支那，適於與其親善融和。故在台內地人（來自日本本土的人）實有尊重人種、充分保護本島人生命財產之必要。-{余}-此次之行，乃視察台灣之治績，仔細調查土人與內地人之關係，以促成充分同化為目標。」後板垣於1914年11月再度來台，1914年12月20日參加[[台灣同化會|台灣同化會]]的創立典禮，板垣被推舉為總裁。

現今，[[日本國會|日本國會]]前還有板垣退助銅像以紀念這位日本[[民權|民權]]運動家。板垣退助自稱反對世襲貴族制度，而提倡“[[一代華族論|一代華族論]]”。板垣退助逝世後，遺族伯爵位被退還。

== 逸話 ==
* 家譜上說「乾氏之太祖[[乾正信|乾正信]]者、“[[日本|日本]][[甲斐國|甲斐國]][[刺史|刺史]][[武田信玄|武田信玄]]之傅，[[板垣信方|板垣信方]]之孫也。正信之父[[板垣信憲|板垣信憲]]有故改易而后被誅、因故籠居於[[遠江国|遠江国]][[周智郡|周智郡]]-{乾}-邑，因以邑名改氏”正信是“1590年於[[遠江国|遠江国]][[掛川市|掛川市]]被[[山內一豐|山內一豐]]召抱之武士”，但-{乾}-正信是否果為板垣信方之孫則不詳。
* 板垣退助自幼便是一個[[現實主義|現實主義]]者、曾經把村裡[[稻荷|稻荷]]神社的[[護身符|護身符]]棄於茅坑以試驗有無[[天譴|神罰]]。（結果並無神罰。）
* 而同樣的、当時日本人有食物禁忌的迷信（例如「[[鰻魚|鰻魚]]與[[梅干|梅干]]」、“[[天婦羅|天婦羅]]與[[西瓜|西瓜]]”等，若一起吃下將會病死）。但他招集眾人一同試吃，結果證明無害。
* 1868年時任日本新政府[[東山道先鋒總督府|東山道先鋒總督府]]参謀。當年舊曆2月14日進兵至[[美濃国|美濃国]]大垣時曾說“本日吾祖先考[[板垣信方|板垣信方]]戦歿320年之日、因此在此復姓”。當時的[[甲府城|甲府城]]是幕府直轄的[[天領|天領]]，因而在此復姓，宣告自己是[[甲斐源氏|甲斐源氏]]支流[[板垣氏|板垣氏]]末裔以壯自己聲勢。結果得到[[甲斐国|甲斐国]]民衆的莫大支持。[[甲州勝沼之戰|甲州勝沼之戰]]得以大勝。
* 戰後，板垣退助得以親見[[甲斐国|甲斐国]][[武田氏|武田氏]]宗族之廟[[惠林寺|惠林寺]]所藏之古画[[板垣信方|板垣信方]]肖像。退助感激而直筆“是吾祖先”。
* 在[[戊辰戰爭|戊辰戰爭]]時、因說服幕軍與政府軍避免在[[德川氏|德川氏]]家廟[[日光東照宮|日光東照宮]]交戰以保護貴重文物，因此日後日光東照宮中設立了板垣退助的銅像。
* 板垣因為身為民權運動家、對世襲[[華族|華族]]制度十分消極，曾兩度拒絕了天皇的受勳。但因為旁人勸諫「若三次推辭，便是不敬了」（取[[三顧茅廬|三顧茅廬]]故事），因而在1887年5月第三次正式授與伯爵。
* 板垣退助至岐阜縣演說時遭相原尚褧暗殺而負傷。但之後板垣並沒有追究，反而上奏[[明治天皇|明治天皇]]特赦兇手。結果相原被特赦、之後更親向板垣謝罪。
* 因反對華族世襲而提倡“[[一代華族論|一代華族論]]”，更向全國華族發信尋求支持。板垣死後，遺族也把[[伯爵|伯爵]]之位還給了天皇。
* 板垣家代代信奉[[曹洞宗|曹洞宗]]，因此板垣退助的葬禮也根據遺囑以曹洞宗禮法進行。不過退助本人卻信奉[[新教|新教]]，也影響因同郷[[片岡健吉|片岡健吉]]（日本初代[[衆議院議長|衆議院議長]]）一同受洗。
*板垣退助與[[後藤象二郎|後藤象二郎]]也是日本首位擁有[[路易·威登|LV]]包的人。（據說是在1882~1883赴歐考察立憲政治時所購入的）

== 世系 ==
*'''乾氏（板垣氏）'''　本姓[[清和源氏|清和源氏]]。[[江戶時代|江戶時代]]，歴世[[土佐藩|土佐藩]]士。家格，馬廻役（[[上士|上士]]）。[[家紋|家紋]]，元來[[板垣氏|板垣氏]]者，甲斐武田氏同流。因使用於“地黒菱（陰花菱）”，雖乾氏者使用於“榧之内十文字”（戦国時代末期～明治時代中期），“土佐桐”（明治中期以降）。
