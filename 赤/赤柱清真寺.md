'''赤柱清真寺'''位於[[香港|香港]][[赤柱|赤柱]]。<ref>{{cite web|url=http://www.islam.org.hk/eng/E-address_mosque.asp |title=Mosques in Hong Kong |publisher=Islam.org.hk |date= |accessdate=2014-04-21}}</ref>是香港第四家清真寺，位于[[赤柱监狱|赤柱监狱]]。

== 历史 ==
20世纪初，有大约400来自[[巴基斯坦|巴基斯坦]]和[[印度|印度]]的[[穆斯林|穆斯林]]雇员为香港监狱署工作。监狱署的总部办公室最初在[[亚毕诺道|亚毕诺道]]。他们大多去[[些利街清真寺|些利街清真寺]]进行[[礼拜_(伊斯兰教)|礼拜]]。但总部办公室搬到更远的赤柱监狱后，为了迎合穆斯林员工的福利和宗教需求，需要在监狱区附近新建一所清真寺。1937年1月1日，赤柱清真寺在监狱内开放。

2009年12月18日，古物古迹办事处咨询委员会将该清真寺指为[[香港一级历史建筑列表|香港一级历史建筑]]。<ref>{{cite web |url=http://www.iticfhk.org/mosque.html |title=存档副本 |accessdate=2016-08-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20141018154157/http://www.iticfhk.org/mosque.html |archivedate=2014-10-18 }}</ref>

== 建筑 ==
该沙色的清真寺有一个大祈祷大厅、走廊和庭院。寺前有一个停车区。因在监狱范围内，该清真寺一般不对公众开放。
