赤铁矿，是[[氧化铁|氧化铁]]的主要矿物形式，[[铁|铁]]主要由赤铁矿冶炼。

== 冶炼 ==
Fe<sub>2</sub>O<sub>3</sub> + 3CO → 2Fe + 3CO<sub>2</sub>
或 2Fe<sub>2</sub>O<sub>3</sub> + 3C → 4Fe + 3CO<sub>2</sub>
