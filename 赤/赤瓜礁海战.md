'''赤瓜礁海戰'''，又稱'''南沙之戰'''或'''“3·14”海战'''，是1988年3月14日[[中华人民共和国|中华人民共和国]]與[[越南|越南]]為爭奪[[南中國海|南中國海]]上多个[[南沙群岛|南沙群岛]]島礁发生的一場小規模战斗。

越南方面将南沙群岛称作“长沙群岛”，因此越方将这次海战称为“'''长沙海战'''”（{{lang-vi|Hải chiến Trường Sa}}）。

本次战役以中方的完全胜利而告终，中方以1人轻伤为代价，夺取了[[南沙群岛|南沙群岛]][[赤瓜礁|赤瓜礁]]等6个重要岛礁，并几乎全歼了参战的[[越南海军|越南海军]]。

== 背景 ==
1980年代，随着中国与西方国家的关系的改善，中国的国际地位得到显著提升。1987年2月，[[联合国教科文组织|联合国教科文组织]]政府间海洋学委员会第14次会议通过《全球海平面联测计划》，[[中华人民共和国|中华人民共和国]]政府参与该项计划，并受委托建立5个海洋观测站，其中南沙群岛的1个观测站序号为“第74号”。经考察选址，中国选择南沙群岛[[永暑礁|永暑礁]]作为第74号海洋观测站的站址，并派出海军编队到达南沙群岛海域建站。期间，[[越南社会主义共和国|越南社会主义共和国]]政府外交部声明要对中国建立74号海洋观测站进行干预。

== 經過 ==
=== 对峙 ===
为了保护在南沙建设海洋观测站的安全，[[中国人民解放军海军|中国人民解放军海军]][[南海舰队|南海舰队]]奉命进入南沙群岛海域。1988年1月14日，中国海军将用于建站的部分施工器材运到[[永暑礁|永暑礁]]。1月31日，越南海军派出2艘武装运输船，运载修建高脚屋的器材开到[[永暑礁|永暑礁]]。[[中华人民共和国海军|中国海军]]担任警戒任务的南海舰队552“宜宾”号[[护卫舰|护卫舰]]编队进行拦截，并派7名官兵駕駛小艇登上[[永暑礁|永暑礁]]，插上了[[中国国旗|中国国旗]]。越南海军放弃了占礁行动。在中国海军舰队的保护下，1988年2月上旬，中国海军施工部队开始在永暑礁建设第74号海洋观测站。<ref name="official_record1">{{cite web |title = 1988.3.14：中越南沙群岛赤瓜礁海战战记(1) |url = http://news.ifeng.com/history/1/jishi/200903/0319_2663_1068627.shtml |publisher=凤凰网 |date=2008-03-19}}</ref> 

从1月15日到2月19日，越南海軍171舰队，125运输旅先后派出艦隻、人员搶佔中國軍隊所據礁盤周围的其它岛礁，中、越軍人在岛礁上相互對峙了近一個月。

2月18日，中国海军162“南宁”号[[驱逐舰|驱逐舰]]、508“西昌”号护卫舰（原舰名"衡阳"）、南拖147号[[拖船|拖船]]与越南海军851号[[扫雷舰|扫雷舰]]、614号运输船在南沙群岛[[尹庆群礁|尹庆群礁]][[华阳礁|华阳礁]]对峙。中国海军和越南海军士兵同时冲上华阳礁企图抢占礁盘，双方士兵在礁盘上形成对峙，最终越南海军撤走。<ref name="official_record2">{{cite web |title = 1988.3.14：中越南沙群岛赤瓜礁海战战记(2) |url = http://news.ifeng.com/history/1/jishi/200903/0319_2663_1068627_1.shtml |publisher=凤凰网 |date=2008-03-19}}</ref>

当时，中共中央高层确定南沙斗争的原则总结为“不主动惹事、不首先开枪、不示弱、不吃亏、不丢面子，如果敌占我岛屿，要强行把他赶走。”。2月22日，中国海军南海舰队派出护卫舰502“南充”舰、503“开源”舰编队（指挥官为南海舰队榆林基地参谋长[[陈伟文|陈伟文]]）<ref name="official_memoirs">[http://news.qq.com/a/20111222/000468.htm 陈伟文将军回忆南沙海战(1)]</ref> 前往南沙海域；3月5日，由[[东海舰队|东海舰队]]派出的护卫舰510“绍兴”舰、531“鹰潭”舰编队抵达永暑礁附近海面，此外，还有先前已经到达的护卫舰553“韶关”舰、556“湘潭”舰编队。 
经部署553舰、556舰编队，510舰、531舰编队在永暑礁、华阳礁海域巡逻。502舰、503舰编队部署在[[南薰礁|南薰礁]]附近海域巡逻，并在礁盘上修建高脚屋。<ref name="official_memoirs" />

3月12日，越南海军派遣三艘军舰（HQ505[[登陆舰|登陆舰]]、HQ604、HQ605武装运输船）运送117名武装人员前往南沙群岛[[九章群礁|九章群礁]][[赤瓜礁|赤瓜礁]]试图修建高脚屋。
3月13日6时，中国海军派遣正在[[南薰礁|南薰礁]]附近海域巡逻的护卫舰502“南充”舰前往[[赤瓜礁|赤瓜礁]]。3月13日14时25分，中国海军502舰抵达，并派出船员登上赤瓜礁。<ref name="official_record2" />
3月13日傍晚，越南海军三艘军舰抵达赤瓜礁附近海域，对中国军舰要其离开的喊话并不理会，HQ505登陆舰到[[鬼喊礁|鬼喊礁]]（赤瓜礁西北1.5海里）附近抛锚，HQ604运输船到赤瓜礁抛锚，HQ605到[[琼礁|琼礁]]（赤瓜礁东北5海里）抛锚。<ref name="official_record3">{{cite web|title=1988.3.14：中越南沙群岛赤瓜礁海战战记(3)|url=http://news.ifeng.com/history/1/jishi/200903/0319_2663_1068627_2.shtml|publisher=凤凰网|date=2008-03-19}}</ref> 
3月13日晚中国海军增派两艘护卫舰，531“鹰潭”舰、556“湘潭”舰前往支援。<ref name="official_memoirs" />

3月14日6时30分，中国海军531“鹰潭”舰抵达赤瓜礁；6时44分，中国海军556“湘潭”舰抵达，<ref name="official_log">{{Cite web |url=http://military.china.com/zh_cn/history2/03/11027561/20060317/13176166.html |title=官方详细记载的1988中越南沙赤瓜礁海战(1) |access-date=2009-04-04 |archive-url=https://web.archive.org/web/20100323042811/http://military.china.com/zh_cn/history2/03/11027561/20060317/13176166.html |archive-date=2010-03-23 |dead-url=yes }}</ref> 奉命警戒琼礁方向。
中国海军502、531、556护卫舰编队由陈伟文担任指挥员。

=== 交火 ===
[[File:053K_(531)_frigate_in_Qingdao_Chinese_Navy_Museum_2011.jpg|thumb]]
[[File:BNS_OSMAN_(F-18).JPEG|thumb]]

3月14日晨6時，越南海军利用低潮开始登上赤瓜礁，604号武装运输船派出43人。中国海军502舰派出33人，531舰派出25人，共计58人登上赤瓜礁。8時，双方人员在礁盘上形成对峙。中国502舰政委李楚群带领7名士兵，驾驶小艇绕过越南运输船驶近礁盘，将繫留越军登礁小船的缆绳砍断。<ref name="official_record3" />

在双方对峙的过程中，中国531舰反潜兵班长杜祥厚出手拔掉插在礁盤上的越南國旗。双方争夺旗帜过程中，中国502舰副枪炮长楊志亮用手推開越軍瞄准的槍支，杨志亮左臂被子弹射穿，8时47分，双方士兵发生交火。越南604船上的機槍也向礁盘上开火，8时48分，中國海军502艦亦開火還擊。4分鐘後越軍604船起火，8分钟后下沉，随后越军登礁人员40多人表示投降。<ref name="chenweiwen_memoirs">[http://news.qq.com/a/20111222/000468_1.htm 陈伟文将军回忆南沙海战(2)]</ref> 
中国海军531舰距离30[[鏈_(量度單位)|链]]与越南505登陆舰交火，中國502艦击沉越南604船后也调转炮口向越南505登陸舰射击。越南505艦舰体起火，舰上炮位被毁，为避免沉没在附近鬼喊礁上抢滩搁浅。越南海军505舰当时未被击沉主要原因是中国531舰因火炮射击震动导致全舰多起故障{{NoteTag|中国海军531舰是搭载“红旗”61型[[防空导弹|防空导弹]]的实验性军舰，海战当时未携带防空导弹，且火炮以及动力设备经受试验考核后，火炮膛蚀严重，主机使用消耗小时已到寿命，状况欠佳。}}<ref>[http://v.huanqiu.com/opinion/201206/20120628131411.shtml 中国海军原装备技术部长郑明海军少将访谈]</ref><ref name="徐友法回忆">[http://mil.news.sina.com.cn/2013-03-13/1125718366_3.html 原531舰政委谈3-14海战]，新浪军事专访。</ref>，致使主炮中止射击。<ref name="访谈录">{{cite journal |title = 6次海战亲历者 常胜将军陈伟文访谈录 |journal=现代舰船 |issue=2011年第10期}}</ref> 
在琼礁海域，越南605号放出舢板登礁，9时15分，在发出警告无效的情况下，中国海军556舰接到编队指挥员命令对越南605号武装运输船发起攻击{{NoteTag|中国海军556舰装备“上游”1型[[反舰导弹|反舰导弹]]，海战当时“没有装弹”。}}<ref name="访谈录" />，越南605船中弹起火，指挥台被击毁，船体严重倾斜，甲板上水。9时37分，中国556舰停止射击（据陈伟文少将战后回忆556舰曾被调来打越南505登陸舰）。后查证越南605船当晚沉没。
战斗中中国海军3艘军舰共消耗100毫米炮弹285发、37毫米炮弹266发。<ref>{{cite web |title = 1988.3.14：中越南沙群岛赤瓜礁海战战记(4) |url = http://news.ifeng.com/history/1/jishi/200903/0319_2663_1068627_3.shtml |publisher=凤凰网 |date=2008-03-19}}</ref>

== 結果 ==
当时[[中国人民解放军空军|中国人民解放军空军]]还不能有效支援南沙群岛海域的作战，因担心越南空军的空中支援，中国海军编队在战斗后撤离了战场。战后，[[中华人民共和国外交部|中华人民共和国外交部]]向[[越南驻华使馆|越南驻华使馆]]递交抗议[[照会|照会]]<ref>{{cite web |title = 1988年3月14日 越南武装舰船侵入南沙群岛海域  中国船只被迫进行自卫还击 |url = http://www.people.com.cn/GB/historic/0314/5913.html |publisher=人民网}}</ref>，但中国政府当时采取了低调处理方式。

中国方面统计指出，舰艇编队中仅有第二小组上礁考察队长[[杨志亮|杨志亮]]1人左臂负伤，无人死亡。<ref name="chenweiwen_memoirs" /> 

此次战斗，越南方面艦艇被擊沉2艘、被擊傷搁浅焚毁1艘（同年7月12日拖带时沉没）。中国数据指出，越軍死亡40余人。被俘40多人，其中中校军官1人。（为迎战可能出现的增援越南军舰，中国舰队收回登礁人员，向外海机动占据阵位，最初投降的俘虏也顾不上接收，再返回礁盘时俘虏只剩9人）<ref name="chenweiwen_memoirs" />。

最终，中国方面占领了赤瓜礁，越南方面则利用伤舰抢滩占领了鬼喊礁、琼礁，对赤瓜礁形成夹击之势。

== 影响 ==
{{See also|南沙群岛主权争议|沙长城}}

越南在1979年后因苏联无偿租用[[金兰湾|金兰湾]]作其后盾，加强了对南海各岛屿资源的支配，海峡两岸因长期对立，均无法或无力对南沙群岛作有效控制。至2012年，南沙群岛岛礁中有29个為越南所控制并有驻军。<ref>{{cite journal |title = 特别策划：终于盼来了三沙市 |journal=中国国家地理 |issue=总第621期 |year=2012年7月 |ISBN = 9771009633001}}</ref>

中国在此次战斗发生以前，虽然一直宣称拥有南沙群岛的主权，但实际上并没有有效控制其中任何一个[[岛屿|岛屿]]、[[珊瑚礁|珊瑚礁]]或[[沙洲|沙洲]]。1970年代以后，南中国海周边国家纷纷进入南沙群岛，单方面划定疆域，并借助外资进行资源开发，使得中方加速進駐該地區。在本次海战以后，中国控制了南沙群岛的[[永暑礁|永暑礁]]、[[华阳礁|华阳礁]]、[[东门礁|东门礁]]、[[南薰礁|南薰礁]]、[[渚碧礁|渚碧礁]]、[[赤瓜礁|赤瓜礁]]共6个重要岛礁，填补了中国对南沙群岛实际控制的空白点，成為了今日控制的[[沙长城|人工島]]前身。<ref>{{cite web |title = 赤瓜礁冲突与南沙高脚屋 |url = http://book.ifeng.com/lianzai/detail_2010_11/25/3226392_57.shtml |publisher=凤凰网}}</ref> 

此次海战也引起[[东南亚|东南亚]]各国对中国的忧虑；由于当时正处于中国与[[西方|西方]]国家的蜜月期，除[[苏联|苏联]]外，欧美各国并没有做更多评论。<ref>[http://book.sina.com.cn/z/zhongyuehaizhan/ 南沙之战的影响 新浪文化·读书 第46期]</ref>
