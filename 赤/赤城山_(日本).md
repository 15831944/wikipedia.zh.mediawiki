'''赤城山'''（{{jpn|hg=あかぎやま|j=赤城山|rm=Akagi yama}}）位於[[日本|日本]][[关东地方|關東地區]]北部的[[群馬縣|群馬縣]]境內，是一座[[休火山|休火山]]。赤城山是[[太平洋板塊|太平洋板塊]]與[[鄂霍茨克板塊|鄂霍茨克板塊]]撞擊后下沉形成的孤島型火山。赤城山與關東北部另外兩座山脈—[[榛名山|榛名山]]、[[妙義山|妙義山]]經常被合稱為[[上毛三山|上毛三山]]。其中赤城山是三山中最高的，因此被選為[[日本百名山|日本百名山]]和[[日本百景|日本百景]]之一。

赤城山是黑檜山與地藏岳的合稱，其中，黑檜山是山脈最高峰，山頂標高1828公尺，其山麓建有[[赤城神社|赤城神社]]。赤城山山頂有[[陷落火山口|陷落火山口]]（Caldera）和火山湖，山頂周圍被標高1200到1800米的群峰環繞，下面是標高800米左右的山麓高原。關東地區至少還有四、五座擁有類似地質結構的山脈。火山地形的赤城山礦藏豐富，據說赤城山[[黃金|黃金]]儲量高達400萬兩。

== 火山活動 ==
[[File:Maebashi20080227.jpg|thumb]]眺望赤城山]]
赤城山約在50万年前就開始活動，那時候形成了標高2300米左右的由[[安山岩|安山岩]]質[[熔岩|熔岩]]和火山浮石的山体。20万年前由於[[二氧化硅|二氧化硅]]的含量增加，熔岩逐漸變成[[英安岩|英安岩]]，同時爆發逐漸增多造成山體坍塌並被熔岩流沖走形成現在開擴平坦的山麓。約5萬年前形成山頂的火山口，約3萬年前的爆發噴出[[鹿沼土|鹿沼土]]，約2萬年前由於水和空氣腐蝕形成[[地蔵岳|地蔵岳]][[熔岩洞窟|熔岩洞窟]]，此後沒有大規模噴火。

== 最新的噴火 ==
[[File:Akagiyama_ono.jpg|thumb]]
紀錄日本鐮倉時期的歷史書《[[吾妻鏡|吾妻鏡]]》中有「[[建長|建長]]3年4月19日（1251年5月11日）赤木岳燒」的紀錄，據此[[氣象廳_(日本)|日本氣象廰]]把赤城山列爲[[活火山|活火山]]。然而目前尚未發現當時噴火后的堆積物，火山口（小沼血池火口）的檢出物形成於公元6世紀，而[[火山口|火山口]]本身是約2萬4千年前所形成。因此這條歷史紀錄可能是[[山火|山火]]而不是噴發，當時也的確有南麓的寺院發生火災的傳聞。由於在1萬年以内發生過噴火的火山被視爲[[活火山|活火山]]，因此赤城山是否為活火山，專家們仍然圍繞這條記錄在討論著。

== 山的構成 ==
[[File:Kakumanbuchi.jpg|thumb]]
赤城山由下述山峰組成：最高峰'''黒檜山'''（1,828米）、'''駒岳'''（1,685米）、'''地藏岳'''（1,674米）、'''長七郎山'''（1,579米）、'''鍋割山'''（1,332米）等，中央部火山口内有大沼、覺滿淵、小沼等三個[[火山湖|火山湖]]，地藏岳山頂建有通信用的赤城中繼局。

== 赤城落山風 ==
冬季[[關東平原|關東平原]]的西北風在[[群馬縣|群馬縣]][[埼玉縣|埼玉縣]]被稱爲「赤城落山風」。由於關東平原北部是山，只有東南方面對[[太平洋|太平洋]]方向有開口，因此東南方的海風氣流隨著[[東京灣|東京灣]]北上遇到北部山脈上升後下降，形成落山風。同時夏季也有這樣的現象，濕熱空氣被積蓄在山腳的平原形成局部高溫。[[群馬縣|群馬縣]][[伊勢崎市|伊勢崎市]]和[[館林市|館林市]]、[[埼玉縣|埼玉縣]][[熊谷市|熊谷市]]等地也因此成爲日本有名的高溫紀錄保持地區。

== 呼称 ==
赤城山自古在當地的稱呼中是發音爲「{{lang|ja|あかぎやま}}」（Akagiyama），可是[[國土地理院|國土地理院]]卻根據規範在[[地圖|地圖]]上使用「{{lang|ja|あかぎさん}}」（Akagisan）的發音方式。在[[群馬縣|群馬縣]]縣民多次陳情之下，官方同意將用名改成與當地習慣相同的「{{lang|ja|あかぎやま}}」（Akagiyama），但赤城山周邊城鎮有些地方的[[大字|大字]]（一種比市村町更小的行政區劃單位）如果仍包含「赤城山」這名字在其行政區劃名稱中時，目前仍然是保留「{{lang|ja|あかぎさん}}」（Akagisan）的發音方式。

== 赤城山的傳説 ==
[[日光|日光]][[男体山|男体山]]西北的[[戰場原|戰場原]]據説是男體山的山神和赤城山山神變成[[蛇|大蛇]]和[[蜈蚣|大蜈蚣]]之後大戰的地方，最後男體山山神取得勝利；而赤城山北有一座[[老神溫泉|老神溫泉]]，傳説是赤城山神戰敗後躲避、療傷的地方。

沿[[江戶川|江戶川]]往下游有[[千葉縣|千葉縣]]有[[流山市|流山市]]也有個赤城神社以及被祭祀的小山，傳説是大洪水的時候赤城山山體的一部分被沖到此處，這是「流山」地名的由來。

提到赤城山不得不說起上州[[國定忠治|國定忠治]]，他是[[江户時代|江户時代]]後期出生於赤城山下國定村的[[遊俠|遊俠]]，[[明治|明治]]、[[大正|大正]]、[[昭和|昭和]]初期在[[講談|講談]]、[[新國劇|新國劇]]中作爲題材而廣爲流傳。國定忠治有一段「赤城の山も今宵限り…かわいい子分のてめえ達とも、別れ別れになる門出だぁ」（只有今晚才能眺望到赤城山，明天我就要告别你们这些可爱的娃娃们出远门）的台詞，也因此赤城山在日本全國家喻戶曉。

== 登山 ==
從赤城山訪問中心可以攀登黒檜山和駒岳，赤城山訪問中心到黒檜山頂單程需要一個半小時。

== 以赤城山命名的事物 ==
[[第二次世界大戰|第二次世界大戰]]結束之前，[[大日本帝國海軍|大日本帝國海軍]]向來習以日本國內的各山脈作為[[戰艦|戰艦]]或[[巡洋戰艦|巡洋戰艦]]的命名，但有點不同的是[[赤城號航空母艦|赤城號]]實際上卻是一艘[[航空母艦|航空母艦]]，這是因為赤城號雖然原本是要作為巡洋戰艦而起造，但卻因為[[华盛顿海军条约|华盛顿海军条约]]的限制，而中途改成航空母艦用途，卻仍保留原本的「赤城」這山名所致。
* 舊[[大日本帝国海軍|大日本帝国海軍]][[航空母艦|航空母艦]]「[[赤城號航空母艦|赤城]]」。
* [[上毛電氣鐵道|上毛電氣鐵道]][[上毛線|上毛線]]與[[東武鐵道|東武鐵道]][[桐生線|桐生線]]的交會車站、[[赤城車站|赤城-{zh-tw:車站;zh-hk:站;zh-cn:站}-]]，位於[[群馬縣|群馬縣]][[綠市|綠市]]境內。
* [[赤城乳業|赤城乳業]]。
* [[藤岡市|藤岡市]]出身相撲選手[[赤城山晃|赤城山晃]]（[[時津風部屋|時津風部屋]]）。
* [[沼田市|沼田市]]出身相撲選手[[栃赤城雅男|栃赤城雅男]]（[[春日野部屋|春日野部屋]]）。
