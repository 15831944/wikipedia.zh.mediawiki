'''赤鱲角'''（鱲，[[粵音|粵音]]讀蠟，laap6<ref>[http://www.unicode.org/cgi-bin/GetUnihanData.pl?codepoint=9c72 Unihan data for U+9C72]</ref>；[[漢語拼音|漢語拼音]]為liè<ref>[[注音符號|注音符號]]為ㄌㄧㄝˋ，參見[http://www.zdic.net/zd/zi/ZdicE9ZdicB1ZdicB2.htm 字典中「鱲」字的解釋] zdic.net 漢典網</ref>；{{lang-en|'''Chek Lap Kok'''}}）是[[香港|香港]]西部海域的[[島嶼|島嶼]]，位於[[大嶼山|大嶼山]][[東涌|東涌]]以北，面積達14.57[[平方公里|平方公里]]<ref>[http://www.landsd.gov.hk/mapping/en/publications/hk_geographic_data_sheet.pdf 地政總署測繪處：香港地理資料]</ref>，是香港境內第三大島嶼，是[[香港國際機場|香港國際機場]]及[[港珠澳大橋香港口岸|港珠澳大橋香港口岸]]的所在地，同樣更是[[民航處|民航處]]及[[香港機場管理局|香港機場管理局]]總部所在地。

== 地名源由 ==
[[File:Chek_Lap_Kok_赤鱲角_Pre-airport,_1964.jpg|左]]
此島於十六世紀初已命名為赤鱲洲。該島名由來，相傳是由於該島黃土處處，植物稀疏，似人體赤露上身，廣州話俗稱「打赤肋」。「赤肋」與「赤鱲」粵音相近，因而得名。另一說則認為其島的漁民能捕捉到許多[[赤鱲|赤鱲]]（[[魚|魚]]類），又可能是島形似赤鱲魚而得名。以前英文地圖亦以Chu Lu Kuk、Chu Lu Kok 和 Chu Lou Kok別稱。

== 歷史 ==
[[File:A_bird's_eye_view_of_Hong_Kong_International_Airport.JPG|缩略图]]

赤鱲角島原是位於大嶼山北岸的多山小島，面積只有2.8平方公里，於石器時代已有人定居。1952年爬蟲學家[[約翰·盧文|盧文]]先生於島上發現香港獨有[[蛙類|蛙類]]品種——[[盧文氏樹蛙|盧文氏樹蛙]]，在興建新機場前送往[[南丫島|南丫島]]棲息。赤鱲角從前有[[赤鱲角村|赤鱲角村]]，現在已經遷移至[[東涌|東涌]]，名為[[赤鱲角新村|赤鱲角新村]]。

赤鱲角歷史悠久，文物十分豐富，據考古資料顯示，早在六千年前的新石器時代中期，已有先民在島上活動；歷經新石器時代晚期、青銅器時代、漢代、唐代、宋代、元代、清代，以至1990年代初，一直有人在島上聚居。島上曾發現[[新石器時代|新石器時代]]先民遺物，也曾出土過[[唐代|唐代]][[灰窰|灰窰]]。[[元朝|元朝]]時期，赤鱲角一帶亦可能有[[香港礦業|冶鐵工業]]。於赤鱲角[[蝦螺灣|蝦螺灣]]，曾被發現[[香港秦朝至元朝歷史|元朝時期]]的[[鑄爐|鑄爐]]十三座<ref>[http://www.amo.gov.hk/b5/oe_lantau_historic.php 赤鱲角島歷史源流]，[[康樂及文化事務署|康樂及文化事務署]]</ref>。早在1970年代末期，考古人員已在島上進行考古調查和試掘，到1990年，由於要在赤鱲角興建新機場，於是便進行全面性的發掘工作。出土大量的考古遺物，為研究該島的歷史提供寶貴的資料。

興建新機場時，赤鱲角大部分土地被夷平，剩下東南面[[過路灣|過路灣]]旁的[[觀景山|觀景山]]和[[白沙咀|白沙咀]]；經過[[填海|填海]]壙充約10平方公里後，成為[[香港國際機場|香港國際機場]]的所在地。另外，赤鱲角西南方的[[欖洲|欖洲]]於填海時，與赤鱲角連體。

因[[港珠澳大橋|港珠澳大橋]]工程，2011年開始在赤鱲角島的東南面填海，以興建[[港珠澳大橋香港連接綫|港珠澳大橋香港連接綫]]，其中包括[[赤鱲角路|赤鱲角路]]及連接[[港珠澳大橋香港口岸|港珠澳大橋香港口岸]]的長堤。

現在，赤鱲角島的總面積僅次於大嶼山和香港島，超越[[南丫島|南丫島]]成為香港第三大島。

== 島上建築及設施 ==
赤鱲角島上主要的設施主要是和航空有關的，當中最主要的是[[香港國際機場|香港國際機場]]、[[民航處|民航處]]及[[香港機場管理局|香港機場管理局]]。不少航空公司亦在島上設立辦事處和後勤設施，如[[國泰航空|國泰航空]]就在島上設置[[國泰城|國泰城]]為其總部。島上也有航空配套設施如燃油庫、空運貨站、航饍公司等。另外，島上亦設有[[機場警署|警署]]、消防局（包括[[機場消防隊|機場消防隊]]轄下消防局和赤鱲角消防局），以保障島上旅客和建築安全。

島上亦有兩間酒店，分別為[[富豪機場酒店|富豪機場酒店]]和[[香港天際萬豪酒店|香港天際萬豪酒店]]，以滿足旅客需要。

島上其他旅客出入境設施包括[[海天客運碼頭|海天客運碼頭]]（只限機場旅客使用）和[[港珠澳大橋香港口岸|港珠澳大橋香港口岸]]。

島上東北亦設有[[亞洲國際博覽館|亞洲國際博覽館]]，提供會議、展覽及活動場地，亦有不少中外歌手在此舉辦演唱會。

島上曾設有[[航天城高爾夫球場|航天城高爾夫球場]]，隨[[航天城_(香港)|航天城]]擴建而結業。

為保存島上古蹟，島上設有[[香港國際機場古物園|香港國際機場古物園]]和[[香港國際機場古窯公園|香港國際機場古窯公園]]。

== 未來發展 ==
{{main|香港國際機場2030規劃大綱}}
{{See also|航天城_(香港)}}
赤鱲角未來主要發展和[[香港國際機場|香港國際機場]]有關，其中包括在機場島北部填海以興建[[香港國際機場|機場]]第三跑道，在島的東部擴建[[航天城_(香港)|航天城]]，當中包括興建一個面積達二十萬平方米的[[商場|商場]]。

== 曾經存在的海灣 ==
[[File:Lantau_Island_reclamation.png|缩略图]]
赤鱲角曾為一個小島，因此周邊都有海灣。在進行香港國際機場填海工程後，大部份海灣基本上已經消失，加上其後興建港珠澳大橋連接路，現大致上只保留過路灣的天然海灣。
