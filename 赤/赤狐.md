'''赤狐'''（[[學名|學名]]：''{{lang|la|Vulpes vulpes}}''），又俗名'''火狐'''，是[[狐|狐]]的一種。
[[File:RenardCrâne.jpg|right]]

== 分布 ==
赤狐是狐狸当中分布最廣的。分布于整個北半球，包含欧洲、北美洲、亚洲草原以及北非地區。是食肉目中分布最廣者。[[冰島|冰島]]、北極島嶼、[[西伯利亞|西伯利亞]]部份地區及極端沙漠地區沒有分布。歐洲亞種在1850年代引入[[澳大利亞|澳大利亞]]，其它引入地區包含[[福克蘭群島|福克蘭群島]]與[[曼島|曼島]]等地。
因帶給許多原生種負面的衝擊，[[國際自然保護聯盟|國際自然保護聯盟]][[物種存續委員會|物種存續委員會]]的入侵物種專家小組（ISSG）列為[[世界百大外來入侵種|世界百大外來入侵種]]。

== 主要特征 ==
体长70厘米，体重4.2-7千克，尾长20-40厘米。腹部为白色，腿和耳尖是黑色的，其他的都呈红色。

== 生活環境与习性 ==
赤狐的适应能力很强，从高山到平原，从森林到草原，甚至沙漠、人类的城镇聚落都是他们的栖息地。很少自己挖洞。以鼠类、鸟类、昆虫、蠕虫和水果为食。每年初春交配，妊娠期50-60天，5月产崽，性成熟半年。尽管赤狐只能活14年，但这在犬科动物中已经是长寿的了。
