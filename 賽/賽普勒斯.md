'''賽普勒斯共和国'''（{{lang-el|'''Κυπριακή Δημοκρατία'''}}；{{lang-tr|'''Kıbrıs Cumhuriyeti'''}}），通称'''賽普勒斯'''（{{lang-gr|'''Κύπρος'''}}；{{lang-tr|'''Kıbrıs'''}}），是位於[[歐洲|歐洲]]與[[亞洲|亞洲]]之間的一個[[岛国|岛国]]，位於[[地中海|地中海]]東部，面積9,251[[平方公里|平方公里]]。

已知人類在賽普勒斯的活動足跡最早可以追溯至西元前10,000年，此一時期的遺址有[[喬伊魯科蒂亞|喬伊魯科蒂亞]]，為[[新石器時代|新石器時代]]保存至今依然完好的建築群。塞浦路斯因地處地中海進入[[西亞|西亞]]地區的要衝，從古至今已被[[西臺|西臺]]、[[亞述|亞述]]、[[埃及|埃及]]、[[波斯|波斯]]、[[四大哈里发|阿拉伯哈里發王朝]]、[[烏邁耶王朝|烏邁耶王朝]]、[[威尼斯|威尼斯]]及土耳其[[奧斯曼帝國|奧斯曼帝國]]等外來勢力侵略或佔領過。西元前333年，[[亞歷山大大帝|亞歷山大大帝]]從波斯人手中接管了此島。1878年起，為[[英國|英國]]所管理，直至於1960年獲得獨立，隔年成為[[大英國協|大英國協]]會員國。

賽普勒斯独立后的主體民族[[希腊人|希腊人]]和少數民族[[土耳其人|土耳其人]][[賽普勒斯问题|零星冲突不断]]，直至1974年爆发严重的種族流血冲突。主要居住于该岛北部的土耳其人在[[土耳其|土耳其]]的干預和支持下于当地另立政权[[北賽普勒斯土耳其共和國|北賽普勒斯土耳其共和國]]。因此賽普勒斯共和國[[de_jure|法理]]主權為賽普勒斯全島及其周圍海域（除了[[亞克羅提利與德凱利亞|一小部分地區]]因為條約分配予英國作為[[軍事基地|軍事基地]]），然而[[de_facto|事實上]]（希臘人政权）有效統治區域仅为南方，占全島面积的63%。北賽普勒斯作为政治實體目前僅為[[土耳其共和國|土耳其共和國]]所承認。賽普勒斯共和国于2004年5月1日正式加入[[歐盟|歐盟]]成为[[欧洲联盟成员国|欧洲联盟成员国]]。

== 國名 ==
[[希臘文|希臘文]]為「{{lang|el|Κυπριακή Δημοκρατία}}」（拉丁文字轉寫:「''Kypriakí Demokratía''」），簡稱「{{lang|el|Κύπρος}}」；[[土耳其文|土耳其文]]為「{{lang|tr|Kıbrıs Cumhuriyeti}}」，簡稱「{{lang|tr|Kıbrıs}}」，[[中國|中國]]官方译之為塞浦-{}-路斯共和國，[[台灣|台灣]]通常译为賽普-{}-勒斯共和國。在政治意義上，與'''[[北塞浦路斯|北塞浦路斯]]'''相對，又稱'''[[南塞浦路斯|南塞浦路斯]]'''。

== 歷史 ==
{{main|塞浦路斯历史}}
早在8000年前就已有定居的村落。前16世紀，[[希臘人|希臘人]]移居。塞浦路斯島在古時包括[[亞述帝國|亞述帝國]]時期被[[犹太人|犹太人]]稱為基提島(Kittim,כתים)。前709年－前525年間，先後被[[亞述|亞述]]、[[埃及|埃及]]和[[波斯人|波斯人]]征服。

前58年起，[[羅馬帝國|羅馬帝國]]在这里建立[[塞浦路斯行省|塞浦路斯行省]]，持續400年。395年歸屬[[拜占庭帝國|拜占庭帝國]]。1489年被[[威尼斯共和国|威尼斯共和国]]吞并。1571年由土耳其[[奧斯曼帝國|奧斯曼帝國]]占领，1573年正式割让。

1878年由[[英國|英國]]吞并塞浦路斯，英國開始對塞浦路斯長達82年的管理。1925年為[[英國|英國]]的[[直轄殖民地|直轄殖民地]]。

1959年2月，英國、[[希臘|希臘]]、[[土耳其|土耳其]]簽訂《蘇黎世－倫敦協定》，[[希臘|希臘]]、[[土耳其|土耳其]]、塞浦路斯三國簽訂《聯盟條約》。希、土、英3國保證塞浦路斯的獨立。1960年8月16日，宣佈獨立，成立塞浦路斯共和國。組成希、土兩族[[聯合政府|聯合政府]]。1963年底，希、土兩族因制憲問題發生嚴重流血衝突。

1974年7月，希臘右翼軍人政權在塞浦路斯策動政變，[[土耳其|土耳其]]以此為由出兵。

1975年2月，[[土耳其人|土耳其人]]在塞浦路斯北部自行宣佈建立“塞浦路斯土族邦”。

1983年，[[土耳其人|土耳其人]]在塞浦路斯北部自行宣佈成立[[北塞浦路斯土耳其共和國|北塞浦路斯土耳其共和國]]，但不獲國際社會的承認。

2004年4月24日，舉行全民公決，[[科菲·安南|科菲·安南]]的[[塞浦路斯问题|塞浦路斯統一]]方案結果無效。同年5月1日，塞浦路斯共和國（希臘人的區域）單獨加入歐盟，成為歐盟正式成員國。

== 政治 ==
賽普勒斯目前分裂為兩個政權：
* '''{{CYP}}'''：南部的[[希臘人|希臘人]]政權，即'''賽普勒斯共和國'''（希臘語：{{lang|el|'''Κυπριακή Δημοκρατία'''}}），得到國際社會的承認。是[[联合国会员国|联合国会员国]]和[[欧盟成员国|欧盟成员国]]。
* '''{{TRNC}}'''.：北部的[[土耳其人|土耳其人]]政權，即[[北賽普勒斯土耳其共和國|北賽普勒斯土耳其共和國]]，僅得到[[土耳其|土耳其]]和自行宣布独立的[[阿塞拜疆|阿塞拜疆]]的[[纳希切万自治共和国|纳希切万自治共和国]]的承認，目前由土耳其人有效控制。

賽普勒斯的希臘人和土耳其人2004年4月24日就[[聯合國秘書長|聯合國秘書長]][[科菲·安南|科菲·安南]]關於賽普勒斯島統一方案同時舉行[[全民公決|全民公決]]。雖然土耳其人支持了安南的方案，但南部的希臘人以絕對多數反對該方案，因而使安南的方案被否決。

2004年5月1日，賽普勒斯的[[希臘人|希臘人]]區域單獨加入[[歐盟|歐盟]]。

[[劳动人民进步党|劳动人民进步党]]候选人[[季米特里斯·赫里斯托菲亞斯|季米特里斯·赫里斯托菲亞斯]]于2008年當選總統，[[劳动人民进步党|劳动人民进步党]]首次成為執政黨，亦是[[歐盟|歐盟]]國家中首個民選產生的[[共產黨|共產黨]]總統。

== 地理 ==
{{main|塞浦路斯地理}}
塞浦路斯是地中海東部的[[島國|島國]]。[[塞浦路斯岛|塞浦路斯岛]]是[[地中海|地中海]]第三大島，仅次于[[西西里岛|西西里岛]]和[[撒丁岛|撒丁岛]]，[[海岸線|海岸線]]長537千米。北部為狹長山脈，多丘陵；西南部為山脈，地勢較高；中部是[[美索利亞平原|美索利亞平原]]。島上無[[常流河|常流河]]，只有少數[[間歇河|間歇河]]。屬[[亞熱帶地中海型氣候|亞熱帶地中海型氣候]]，[[夏季|夏季]]炎熱乾燥，平均氣溫28－35℃，[[冬季|冬季]]溫和濕潤，氣溫4－10℃。

== 經濟 ==
{{main|塞浦路斯經濟}}
塞浦路斯為傳統的農業國，並著重旅遊業的發展。人民相对富足，[[人均國內生產總值|人均國內生產總值]]高於[[歐盟|歐盟]]平均數。

2013年，受[[歐債危機|歐債危機]]困擾的塞浦路斯政府為獲得[[歐元區|歐元區]]援助而決定向銀行儲戶收取存款稅，挽救當地面臨[[破產|破產]]的銀行業。根據徵稅建議，存款10萬[[歐元|歐元]]以下儲戶徵稅3%，10萬歐元至50萬歐元之間徵收10%，超過50萬歐元按照15%的稅率徵稅。此前的方案為對存款10萬歐元以下儲戶的徵稅稅率為6.75%，超過這一水準為9.9%。此舉引起當地居民嘩然，當地多間銀行和[[提款機|提款機]]更出現[[擠提|擠提]]情況。

== 人口 ==
{{main|賽普勒斯人口}}
2003年，人口已超過80萬，希臘裔賽普勒斯人約占85.2％，土耳其裔賽普勒斯人約占11.6％。

自2000年起，大量[[外籍勞工|外籍勞工]]從[[菲律賓|菲律賓]]、[[泰國|泰國]]、[[斯里蘭卡|斯里蘭卡]]等地前來工作。

=== 媒体 ===
在2015年[[自由之家|自由之家]]的[[自由指数|自由指数]]中，塞浦路斯和[[北塞浦路斯|北塞浦路斯]]被评为“自由”。塞浦路斯的新闻自由得分为25/100，法律环境得分为5/30，政治环境得分为11/40，经济环境得分为9/30（得分越低则越好）。在2015年世界新闻自由指数排名中，[[无国界记者|无国界记者]]组织的[[新闻自由指数|新闻自由指数]]中，将塞浦路斯排名180个国家的第24位，得分为15.62。

塞浦路斯法律规定了言论自由和新闻自由，该国政府在实践中普遍尊重这些权利。独立的新闻机构、有效的司法和有效的民主政治制度共同确保言论和新闻自由。法律还禁止任意干涉隐私和家庭通信，政府在执行中通常遵守这些禁令。

[[塞浦路斯广播公司|塞浦路斯广播公司]]是塞浦路斯的国有广播电视媒体，该公司经营着三个电视频道。塞浦路斯广播公司一直致力于制作新闻、宗教、儿童、艺术纪录片和电视连续剧等各类节目，在部分时段也同样引进外国电视节目（提供希腊语字幕）。此外还有12个私人电视台，希腊ERT世界频道、[[欧洲新闻台|欧洲新闻台]]在塞浦路斯有落地播出。
