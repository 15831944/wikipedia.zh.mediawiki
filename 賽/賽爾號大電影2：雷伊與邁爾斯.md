《'''賽爾號大電影2：雷伊與邁爾斯'''》（英語：SeerⅡ），（又名《赛尔号大电影2：圣灵系的王子》）<ref>{{Cite web |url= http://games.cntv.cn/2011/seer_news_1231/72718.shtml |title= 赛尔号号外：赛尔号2圣灵系的王子 |author= 游戏台编辑  |date= 2011-12-31 |publisher= [[央视|央视]]网游戏 |language= zh-hans |accessdate=2013-07-13 |quote=  }}</ref>是由[[上海淘米网络科技有限公司|上海淘米网络科技有限公司]]运营的的儿童虚拟社区《[[赛尔号|赛尔号]]》改编的第二部同名动画电影，于2012年6月28日在[[中国大陆|中国大陆]]公映。电影首周票房破两千万[[人民币|人民币]]。<ref>{{Cite web |url= http://comic.qq.com/a/20121025/000023.htm |title= 《赛尔号2》动画电影票房破两千万 |author=   |date= 2012年7月4日 |publisher= [[腾讯|腾讯]]动漫 |language= zh-hans |accessdate=2013-07-13 |quote=  }}</ref>

==剧情==
圣灵系神兽迈尔斯在被宇宙海盗抓捕前，把珍贵的幻影宝石交给儿子蒙塔，并让他离开幻影星。 远在赫尔卡星的雷伊迅速前往了幻影星。蒙塔被海盗追击，偶然闯入[[赛尔号|赛尔号]]，并遇到先锋小队队长赛小息，队友阿铁打、卡璐璐一行人。他们决定帮蒙塔救出爸爸迈尔斯，最终成功解救。<ref>{{Cite web |url= http://news.7k7k.com/content/20120502/169712.html |title= 赛尔号大电影2精彩剧情预告 |author= shadow  |date= 2012-05-02 |publisher= 7k7k.com |language= zh-hans |accessdate=2013-07-13 |quote=  }}</ref>
