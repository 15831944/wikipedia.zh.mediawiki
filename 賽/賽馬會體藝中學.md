'''賽馬會體藝中學'''是[[香港|香港]]的一所中學，於1989年成立，是全港唯一可百分百[[自行分配學位_(中學)|自行分配學位]]之[[津貼中學|津貼中學]]，位於[[新界|新界]][[沙田_(香港)|沙田]][[火炭|火炭]][[樂景街|樂景街]]5至7號。

== 歷史 ==
體藝中學的創校理念源自已故[[港督|港督]][[尤德|尤德]][[爵士|爵士]]，他主張在香港成立一所不僅強調[[學術|學術]]表現，而是更為提供學生發揮[[體育|體育]]及[[視覺藝術|視覺藝術]]潛能的中學。1985年11月，[[布政司|布政司]]與[[英皇御准香港賽馬會|英皇御准香港賽馬會]]達成協議，由政府與賽馬會共同出資8400萬港元在[[沙田_(香港)|沙田]]建立體藝中學，並且由[[教育署|教育署]]邀請專業人士及教育工作者籌組學校管理局。體藝中學於1989年9月正式開課，由前任[[保良局董玉娣中學|保良局董玉娣中學]]校長[[張灼祥|張灼祥]]先生出任創校校長。

* 仁 (Humanity)

== 設施 ==
新翼大樓於2005年完工，除一般中學設施外，另設有室內體育館、健身室、游泳池、多元化球類運動場、體育實驗室、繪畫及素描室、雕塑室、平面設計室、陶藝室、攝影室、版畫室、展覽廳、演講廳、樂隊練習室、學生宿舍及飯堂，全校空氣調節，並已鋪設電腦網絡。

== 特色 ==
為吸納更多有體育或美術潛質的學生入讀，該校收生不受校網限制，是全港唯一可以百分百自行分配學位的津貼中學。該校重視體藝發展，中一至中三學生必須選擇體育或美術其中一科作為主修科。體育每週6節課，勞美每週7節課。該校亦設有宿舍，有宿位96個，以方便居住於較遠地區的學生。
