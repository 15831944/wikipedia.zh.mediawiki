'''尼古拉·巴甫洛维奇·普霍夫'''（{{lang-ru|'''Никола́й Па́влович Пу́хов}}'''，{{bd|1895年|1月25日|1958年|3月28日|catIdx=Nikolay Pavlovich Pukhov}}），苏联[[上将|上将]]。<ref>{{cite web|url=http://www.warheroes.ru/hero/hero.asp?Hero_id=1834|title=Пухов Николай Павлович|website=www.warheroes.ru|accessdate=2018-01-18}}</ref>

== 生平 ==
===早年生活===
1895年，生于[[卡卢加州|卡卢加州]][[巴贝宁斯基区|巴贝宁斯基区]]格里绍沃镇。

1916年，应征入帝国陆军。

===内战时期===
1918年，参加[[苏联红军|苏联红军]]，历任团副官、步兵旅参谋长和师参谋长。

===二战时期===
1941年，加入[[联共|联共]]。

1942年，任[[布良斯克方面军|布良斯克方面军]]第13集团军司令。

1943年，参加[[沃罗涅日战役|沃罗涅日战役]]，2月因战功晋升[[中将|中将]]；7月参加[[库尔斯克会战|库尔斯克会战]]；10月成为[[苏联英雄|苏联英雄]]；11月参加[[基辅战役|基辅战役]]，苏军进入反攻。

1943年-1944年，参加[[日托米尔—别尔季切夫战役|日托米尔—别尔季切夫战役]]、[[普罗斯库罗夫—切尔诺维策战役|普罗斯库罗夫—切尔诺维策战役]]、[[利沃夫—桑多梅日战役|利沃夫—桑多梅日战役]]，8月晋升[[上将|上将]]。 

1945年，在[[桑多梅日—西里西亚战役|桑多梅日—西里西亚战役]]、[[下西里西亚战役中|下西里西亚战役中]]，协助其它集团军挫败德军进军至[[尼斯河|尼斯河]][[特里贝尔|特里贝尔]]西北区域。

1945年，参加[[柏林战役|柏林战役]]、[[布拉格战役|布拉格战役]]。
===战后时期===
1948年，任[[敖德萨|敖德萨]]军区和[[北高加索|北高加索]]军区司令。

1953年，任[[西西伯利亚|西西伯利亚]]军区和[[西伯利亚|西伯利亚]]军区司令。

1958年3月28日在莫斯科去世。

== 其它 ==
2002年，俄罗斯军事学院评定“战时最有才华的33个集团军司令”，他位居第7位。
