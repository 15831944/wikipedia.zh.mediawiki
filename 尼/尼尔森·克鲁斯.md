'''尼尔森·雷蒙·克鲁斯·马丁内斯'''（'''Nelson Ramón Cruz Martínez'''，{{bd|1980年|7月1日|||Cruz, Nelson}}）出生于[[多明尼加共和国|多明尼加共和国]][[基度山省|基度山省]]，是[[美国职棒大联盟|美国职棒大联盟]][[西雅圖水手|西雅圖水手]]的[[外野手|外野手]]。

== 早年生活 ==
克鲁斯小时候曾打过[[篮球|篮球]]，还入选过多明尼加青年篮球队。{{by|1998年}}2月23日他以自由球员的身份被[[纽约大都会|纽约大都会]]签下，开始在多明尼加夏季联盟大都会所属的球队效力。{{by|2001年}}起来到[[美国|美国]]打球，当年他被交易到[[奥克兰运动家|奥克兰运动家]]，并在{{by|2004年}}被交易到[[密尔瓦基酿酒人|密尔瓦基酿酒人]]。

== 大联盟生涯 ==
在小联盟磨练了7年之后，克鲁斯终于在{{MLBy|2005年}}9月17日登上大联盟。当年他被[[美国棒球|美国棒球]]杂志评选的未来之星中名列酿酒人球团第八位。

{{MLBy|2006年}}7月28日，克鲁斯和[[卡洛斯·李|卡洛斯·李]]一起被交易到[[德州游骑兵|德州游骑兵]]。8月16日他击出职业生涯首支[[满贯全垒打|满贯全垒打]]，9月4日还击出了一支[[场内全垒打|场内全垒打]]。
[[File:DSC05837_Nelson_Cruz.jpg|thumb]]

在大联盟的前四个赛季，克鲁斯未能在大联盟站稳脚跟，常来回于大小联盟之间。直到{{MLBy|2009年}}他终于开始成为球队的主力外野手，这个赛季他的长打能力完全爆发出来，一共击出33支全垒打，名列美联第八位，并且取代了受伤的[[托利·杭特|托利·杭特]]，首次入选[[2009年美国职棒大联盟全明星赛|全明星赛]]<ref>http://fantasynews.cbssports.com/fantasybaseball/players/playerpage/10813</ref>。他还参加了[[美国职棒大联盟全垒打大赛|全垒打大赛]]，最终名列第2名。

{{MLBy|2014年}}2月22日，克鲁斯和[[巴爾的摩金鶯|巴爾的摩金鶯]]簽下1年800萬[[美元|美元]]的合約，而克魯茲在2014年球季擊出40發全壘打，幫助金鶯隊奪下睽違17年的ALEC，並一路打進ALCS。

2014年12月2日，克鲁斯和[[西雅圖水手|西雅圖水手]]簽下4年4700萬美元的合約。












== 經歷 ==
*2009年參加[[美國職棒大聯盟|美國職棒大聯盟]]全壘打大賽(Home Run Derby)，以總數21支全壘打、決賽5支全壘打，屈居於亞軍。
*2009年入選[[美國職棒大聯盟明星賽|美國職棒大聯盟明星賽]]。
*2010年5月24日，獲選[[美國聯盟|美國聯盟]]單週最佳球員。
*2011年美國職棒大聯盟季後賽，美國聯盟冠軍賽(ALCS)最佳球員。
*2013年代表[[多明尼加|多明尼加]]，參加世界棒球經典賽(WBC)，最終拿下冠軍。
*2013年入選[[美國職棒大聯盟明星賽|美國職棒大聯盟明星賽]]。
*2014年6月2日，獲選[[美國聯盟|美國聯盟]]單週最佳球員。
*2014年入選[[美國職棒大聯盟明星賽|美國職棒大聯盟明星賽]]。
*2014年9月8日，獲選[[美國聯盟|美國聯盟]]單週最佳球員。
*2015年4月20日，獲選[[美國聯盟|美國聯盟]]單週最佳球員。
*2015年4月，獲選[[美國聯盟|美國聯盟]]單月最佳球員。
*2015年入選[[美國職棒大聯盟明星賽|美國職棒大聯盟明星賽]]。
*2017年入選[[美國職棒大聯盟明星賽|美國職棒大聯盟明星賽]]。
*2018年6月24日，獲選[[美國聯盟|美國聯盟]]單週最佳球員。
*2018年入選[[美國職棒大聯盟明星賽|美國職棒大聯盟明星賽]]。
