'''尼斯'''（Nice）是[[地中海|地中海]]沿岸[[法国|法国]]南部港口[[市镇_(法国)|城市]]，位于[[普罗旺斯-阿尔卑斯-蓝色海岸|普罗旺斯-阿尔卑斯-蓝色海岸]][[大区|大区]]，为[[滨海阿尔卑斯省|滨海阿尔卑斯省]][[行政中心|行政中心]]。尼斯地处[[马赛|马赛]]和[[意大利|意大利]][[热那亚|热那亚]]之间，为主要旅游中心和[[蔚藍海岸|蔚藍海岸]]地区的首选度假地，尼斯是[[普罗旺斯-阿尔卑斯-蓝色海岸|普罗旺斯-阿尔卑斯-蓝色海岸]][[大区|大区]]第二大城市（第一大城市[[马赛|马赛]]），大都会区人口933080人(1999年[[统计|统计]])。尼斯是[[法国|法国]]第五大城市，城市中心人口347,100人。（2004年统计）虽然现在尼斯是法国的领土，但是在历史上尼斯的文化和语言和意大利几乎一模一样，反倒是距离法国比较遥远，不过因为经过了法国100多年的统治，法语和法国菜等法国的精致文化也融入了尼斯城当中，造就了独一无二的风景。

== 历史 ==
[[File:Meyers_b9_s0067b.jpg|缩略图]] 在[[罗马帝国|罗马帝国]]时代 前154年.]]
[[File:Italy_Lothar_II.svg|缩略图]] 781年]]
[[File:Italia_1494-it.svg|缩略图]] 1494.]]
[[File:Italy_1796_it.svg|缩略图]] 1796.]]
尼斯大约400,000 年前就有人类活动，从古迹''[[Terra_Amata|Terra Amata]]''可以看到对[[火|火]]的使用和房屋的建造。

尼斯城大约建造于公元前350年，很快就成为[[利古里亚|利古里亚]]海岸最繁忙的贸易站之一。

公元7世纪，尼斯加入利古里亚的城市建立的[[热那亚|热那亚]]联盟。729年，尼斯击退了[[撒拉森人|撒拉森人]]的进攻，但是在859年和880年两度被掠夺，城市也被焚烧。在10世纪的大部分时期尼斯统治着周边城市。

[[中世纪|中世纪]]时期，尼斯和大部分意大利城市一样，遭受战争的灾难。作为[[比萨|比萨]]的同盟，它是[[热那亚|热那亚]]的敌对方，[[法国|法国]]和[[东罗马帝国|东罗马帝国]]都想征服它。13世纪和14世纪，尼斯几度成为Provence公爵的领地。1388年，成为[[薩伏依公爵|薩伏依公爵]]的领地。

1543年，尼斯遭到法国和土耳其盟军的攻击，法国得到尼斯的所有权。

1600年，尼斯落入[[吉斯公爵|吉斯公爵]]手中。之后在1696年，尼斯再度被[[薩伏依公爵|薩伏依公爵]]占有。1705年又被法国夺回。之后的两百年，尼斯不停更换主人。直到1860年，法国拿破仑三世和萨丁尼亚国王签定合約，尼斯成为法国的领地，1940年二戰時期[[義大利王國|義大利]]軍隊佔領法國[[濱海阿爾卑斯省|濱海阿爾卑斯省]]，尼斯亦为義大利所控制，直至二战完結才回歸法國。

== 气候 ==
尼斯全年气候宜人，冬暖夏凉，属于[[地中海性气候|地中海性气候]]。在冬天很少有结冰的时候，几乎不下雪，有时在早春时节气温可以达到20 °C ，2007年3月2日最高气温达到26 °C。最适宜的旅游月份是五月和九月中到十月中。

== 文化 ==
[[File:Dscn0062-nice-port-castle-hill_crop_1200x600.jpg|300px]]
独特的历史造就了独特的尼斯文化。当地语言[[尼斯语|尼斯语]]（Niçard或Nissart）属于法国南部的[[奥克语|奥克语]]群，[[拉丁语族|拉丁语族]]，至今仍有少数人使用。[[意大利|意大利]]文化的强大影响和[[科西嘉島|科西嘉島]]文化较轻微的作用使它区别于其它已经失传的奥克语分支，不那么晦涩难懂，也使它保留至今。

当地的美食，包括[[尼斯洋葱塔|尼斯洋葱塔]]（''pissaladière''，[[洋蔥|洋蔥]]和[[凤尾鱼|凤尾鱼]]酱制成的一种[[比萨饼|比萨饼]]）、''socca''（[[鹰嘴豆|鹰嘴豆]]粉制成的薄烤饼）、[[馬賽魚湯|馬賽魚湯]]和其它鱼汤、[[鳕鱼干|鳕鱼干]]、尼斯填菜（''farcis niçois''，蔬菜上面洒面包屑）和尼斯沙拉（''salade niçoise''，[[西红柿|西红柿]]、青[[辣椒|辣椒]]、煮[[鸡蛋|鸡蛋]]、[[金枪鱼|金枪鱼]]或其他鱼加上[[橄榄|橄榄]]做成的色拉）。

很多其它传统文化艺术被保留下来，尤其是[[民間音樂|民間音樂]]和民间舞蹈，尼斯最著名的舞蹈是[[法朗多尔舞|法朗多尔舞]]。

以前，尼斯欢迎来自意大利人的许多移民（现在仍然是人口的大部分），还有[[西班牙|西班牙]]和[[葡萄牙|葡萄牙]]的[[移民|移民]]。然而，在过去的几十年，尼斯开始向世界上其它国家开放，很多北欧和西非、东南亚人移民到了尼斯。

== 教育 ==

[[尼斯大学|尼斯索菲尔-昂蒂波利大学]]和[[EDHEC商业学校|EDHEC商业学校]]坐落在尼斯。

尼斯有很多大学校预科学校（Classe Préparatoire），其中尼斯预科是全法国最好的10所预备学校之一。

== 交通 ==
尼斯的[[尼斯蓝色海岸机场|蔚蓝海岸国际机场]]是游客到达尼斯的主要方式。[[尼斯城站|火车站]]将尼斯和法国其它城市相连。也可以通过轮渡到达[[科西嘉島|科西嘉島]]的另外几个城市。

市内公共交通由[[天蓝线巴士公司|天蓝线巴士公司]]（Ligne d'Azur）提供服务，市内的[[尼斯電車|有轨电车]]在2007年开通。
