'''尼可勒斯菊石'''（[[學名|學名]]：{{lang|la|'''''Nicklesia'''''}}）是生存在[[白堊紀|白堊紀]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]。牠們是[[旋菊石超科|旋菊石超科]]演化成[[帶菊石超科|帶菊石超科]]過程中的一群異類，有學者將牠們和其近親獨立成[[美菊石超科|美菊石超科]]（{{lang|la|Pulchelliaceae}}），也有學者將牠們納入[[區域菊石超科|區域菊石超科]]之下。<ref name="Nicklesia">{{Cite web |url = http://liusir.pixnet.net/blog/post/248636477 |title = Nicklesia 尼可勒斯菊石 (法國) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-07-20 }}</ref>其[[化石|化石]]被發現於[[摩洛哥|摩洛哥]]、[[法國|法國]]及[[西班牙|西班牙]]等地。<ref name="Fossilworks" />

== 特徵 ==
尼可勒斯菊石的外觀相當特別，具有扁平的殼形以及非常狹小的臍部，粗大的肋條之間幾乎沒有間隙。<ref name="Nicklesia" />
