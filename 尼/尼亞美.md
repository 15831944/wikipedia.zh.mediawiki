'''尼阿美'''（法語：'''Niamey'''）是[[尼日尔|尼日尔]]的首都及最大城市，位在[[尼日河|尼日河]]河畔，市區大部分位於尼日河的左岸。它是行政、文化和經濟中心。

== 歷史 ==
尼阿美可能建立於18世紀，但是直到[[法國|法國]]在尼阿美設立了殖民站之前，一直沒有重要的地位。此後，尼阿美發展成了一個重要的中心，並於1926年12月28日成為尼日的首府。尼阿美的人口也從1930年的3千人、1960年的約3萬人、1980年的25萬人增加至2000年估計的80萬人<ref name=sahel20110701 />。因為乾旱而移入的人口成了人口增加的主要原因，也為尼日的唯一大都市<ref name=sahel20110701 />。

==地理==
[[Image:Niamey_SPOT_1101.jpg|thumb]]所拍攝)]]
尼亞美都會區面積超過250平方公里，尼日河流經西南城區。

== 經濟 ==
尼阿美位於[[花生|花生]]產區，製造業則有[[磚|磚]]、[[陶器|陶製品]]、[[水泥|水泥]]及[[紡織|紡織]]、織布、水泥、啤酒、金屬工具、食品加工、肉類冷凍、軋花、傢具、磚瓦和塑料等等。<ref name="中國大百科全書">丁登山，《[[中國大百科全書|中國大百科全書]]》－尼阿美</ref>
== 城市設施 ==
建築物以政府辦公大樓為主。設有[[尼阿美大學|尼阿美大學]]、國家博物館及[[清真寺|清真寺]]等。大部分居民居住在尼日河沿岸。[[甘迺迪橋|甘迺迪橋]]於1970年完工後，市區已由左岸向右岸擴展。<ref name="中國大百科全書"/>
== 交通 ==
尼阿美於尼日河上建有碼頭，10月到翌年3月可通航小船。有多條公路會聚於此，各公路可連通國內主要城市及[[奈及利亞|奈及利亞]]、[[貝寧|貝寧]]、[[布基納法索|布基納法索]]、[[阿爾及利亞|阿爾及利亞]]等鄰國。市郊有[[迪奥里·哈马尼国际机场|迪奥里·哈马尼国际机场]]。<ref name="中國大百科全書"/>
== 氣候 ==
尼阿美市區面積約250平方公里，海拔約200餘公尺，城區中有[[尼日河|尼日河]]流經，尼阿美屬[[熱帶草原氣候|熱帶草原氣候]]，雨季由5月開始，6至9月雨量較多，其他月份雨量較少。
