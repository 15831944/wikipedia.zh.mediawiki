'''尼古拉·維拉里耶維奇·杜洛夫'''（{{lang-ru|Никола́й Вале́рьевич Ду́ров}}，{{bd|1980年|11月21日}}）是[[俄罗斯|俄罗斯]]的[[程序员|程序员]]和[[数学家|数学家]]，[[保罗·杜洛夫|保罗·杜洛夫]]的哥哥。他们一起创办了俄罗斯社交网络[[VKontakte|VKontakte]]<ref>{{cite web|url=http://forumvk.ru/news/nikolaj_durov/2010-06-20-125|title=Forumvk 2010 Vk.com stars|publisher=|deadurl=yes|archiveurl=https://web.archive.org/web/20101122175632/http://forumvk.ru/news/nikolaj_durov/2010-06-20-125|archivedate=2010-11-22|df=}}</ref>，随后创办了[[Telegram|Telegram]]。<ref>{{cite web|url=https://techcrunch.com/2013/10/27/meet-telegram-a-secure-messaging-app-from-the-founders-of-vk-russias-largest-social-network/|title=Meet Telegram, A Secure Messaging App From The Founders Of VK, Russia’s Largest Social Network - TechCrunch|first=Catherine|last=Shu|publisher=}}</ref>

== 教育与早期经历 ==
在1996、1997、1998年，杜洛夫三度夺得了[[国际奥林匹克数学竞赛|国际奥林匹克数学竞赛]]金牌<ref>{{cite web|url=http://www.imo-official.org/hall.aspx|title=International Mathematical Olympiad|website=www.imo-official.org}}</ref>。1995至1998年的四年间，他在[[国际信息学奥林匹克|国际信息学奥林匹克]]中获得一枚金牌、三枚银牌<ref>{{cite web|url=http://stats.ioinformatics.org/people/1706|title=Nikolai Dourov|website=stats.ioinformatics.org}}</ref>。他也曾是圣彼得堡州立大学[[Association_for_Computing_Machinery|ACM ICPC]]团队的成员，该团队在2000年<ref>{{cite web|url=http://icpc.baylor.edu/past/icpc2000/Report.html|title=Final report of the ACM programming contest 2000|publisher=|deadurl=yes|archiveurl=https://web.archive.org/web/20110102184345/http://icpc.baylor.edu/past/icpc2000/Report.html|archivedate=2011-01-02|df=}}</ref>和2001年<ref>{{cite web|url=http://icpc.baylor.edu/past/icpc2001/Report.html|title=Final report of the ACM programming contest 2001|publisher=|deadurl=yes|archiveurl=https://web.archive.org/web/20110102183826/http://icpc.baylor.edu/past/icpc2001/Report.html|archivedate=2011-01-02|df=}}</ref>赢得了[[ACM国际大学生程序设计竞赛|ACM国际大学生程序设计竞赛]]。

他于2005年从[[圣彼得堡国立大学|圣彼得堡国立大学]]获得了他的第一个博士学位。并在[[波恩大学|波恩大学]]继续他的数学教育，在[[格尔德·法尔廷斯|格尔德·法尔廷斯]]的教学下获得了另一位博士学位<ref>{{cite web|url=http://www.hcm.uni-bonn.de/old/faculty/Faltings/phd-students.html|title=HCM: 404|website=www.hcm.uni-bonn.de}}</ref>。

== 职业 ==
尼古拉•杜洛夫在2013年之前担任[[VKontakte|VKontakte.ru]]开发团队的[[首席开发人员|首席开发人员]]<ref>{{Cite web|url=http://www.rg.ru/2014/04/30/nikdurov-site.html|title=Брат Павла Дурова предсказал новые увольнения из "ВКонтакте"|last=Anton|first=Blagoveshchensky|publisher=[[Rossiyskaya_Gazeta|Rossiyskaya Gazeta]]|date=April 30, 2014|accessdate=October 28, 2014}}</ref>

并在2013年创办了[[telegram|telegram]]，为它开发了[[MTProto|MTProto]]协议。

被认为是[https://drive.google.com/file/d/1lqVlrgiztnA5dkOHP7-ENDKT1FgZuCUV/view original TON (Telegram Open Network) whitepaper]的作者。
