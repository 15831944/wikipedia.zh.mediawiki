'''尼布甲尼撒二世'''（[[亚拉姆语|亚拉姆语]]：ܢܵܒܘܼ ܟܘܼܕܘܼܪܝܼ ܐܘܼܨܘܼܪ；{{lang-he-n|נְבוּכַדְנֶצַּר}} ''{{transl|he|ISO|Nəḇūḵaḏneṣṣar}}''；{{lang-grc|Ναβουχοδονόσωρ}}；''{{transl|el|ISO|Naboukhodonósôr}}''；[[阿拉伯语|阿拉伯语]]：نِبُوخَذنِصَّر ''{{transl|ar|DIN|nibūḫaḏniṣṣar}}''；约{{bd|前634年||前562年||N}}），是位于[[巴比伦|巴比伦]]的[[伽勒底帝国|迦勒底帝国]]最伟大的君主，在位时间约为公元前605年–前562年。他因在[[首都|首都]]巴比伦建成著名的[[空中花园|空中花园]]而为人赞颂，同时也因毁掉了[[所罗门圣殿|所罗门圣殿]]而为人熟知。他曾征服[[犹大王国|犹大王国]]和[[耶路撒冷|耶路撒冷]]，并流放[[犹太人|犹太人]]，《[[圣经|圣经]]》上对此也有所记载。他的事迹被记录在《[[但以理書|但以理書]]》中，并且圣经的其它章节也提到了他。

用巴比伦的语言解释他的名字意思是“皇冠的保护和继承者尼布”，或“保卫边疆者尼布”。

== 早年 ==
尼布甲尼撒的父亲是迦勒底帝国的开国君主－[[那波帕拉薩爾|那波帕拉薩爾]]，母亲是出身于[[伊朗|伊朗]]的[[米底|米底]]王国公主，这两个王国联姻后，势力大增。尔后那波帕拉薩爾将巴比伦从[[亚述|亚述]]人的属国地位中独立出来，并打垮亚述人，将亚述帝国的都城[[尼尼微|尼尼微]]夷为平地。尼布甲尼撒是他父亲的长子和继承人，前607年随父亲出征亚述北部山区，后来单独指挥作战。由于[[埃及|埃及]]曾经击败亚述人，并占领了原亚述帝国西部[[卡尔基米什|卡尔基米什]]一带，导致前亚述帝国领土被伽勒底和米底分割。前606年，他带兵任总司令，在卡尔基米什痛歼埃及军队，收复了前亚述领地。前605年8月，那波帕拉薩爾去世，尼布甲尼撒赶回巴比伦继承王位。

== 對外征戰 ==
前604年他发兵征服[[叙利亚|叙利亚]]和[[巴勒斯坦|巴勒斯坦]]部分地区。尔后雇用[[希腊|希腊]]佣兵作战，但于前601年被[[埃及|埃及]]军队打败，损失惨重，他竭力补充损失的战车和兵员，卷土重来。前597年，他率兵攻占[[耶路撒冷|耶路撒冷]]，并将[[猶大王國|猶大王國]]王－约雅斤掳回巴比伦，[[圣经|圣经]]中还记载了他在前586年对耶路撒冷的另一次进攻以及对[[推罗|推罗]]城的围攻，并将两座城市毁灭，将神殿拆毁。

[[大英博物馆|大英博物馆]]目前保存有描述他的[[楔形文字|楔形文字]]泥版文书，写到：“巴比伦国王，尼布甲尼撒，在第37年，带领军队进攻埃及，埃及国王雅赫摩斯集合军队前往应战”。

== 成就 ==
他同时也是一位卓越的外交家，曾派遣使节和[[小亚细亚|小亚细亚]]的希腊人国家吕底亚及米底周旋，解除西方的威胁，他将巴比伦建成一个著名的城市，完善输水设施和水库。

== 逝世 ==
他于公元前562年去世，由他儿子[[以未米羅達|以未米羅達]]繼位。公元前539年，他去世后尚不到25年，巴比伦就被[[波斯帝國|波斯帝國]]的[[居鲁士大帝|居鲁士大帝]]征服。
