'''尼罗鳄'''（學名：'''''Crocodylus niloticus'''''）是一種大型[[鱷魚|鱷魚]]，[[非洲|非洲]]最大的[[爬行動物|爬行動物]]，體型僅次於[[河口鱷|河口鱷]]，為全數23種[[鱷魚|鱷魚]]當中被人類研究最多的一種。

== 特征 ==
雌性尼羅鱷身長2.5至3.9米，雄性體長3.5至5.5米，雌性體重227至300公斤, 雄性500至1090公斤，平均壽命達70至100年。<ref>[http://wildnature.go2c.info/view.php?doc=/wildlife/crocodile Crocodile (Nile Crocodile)（尼羅河鱷魚）東非大自然]</ref>吻阔略呈长三角形，上颌每侧有牙齿16-19枚，下颌每侧14-15枚；躯干背面有坚固的厚鳞甲6-8纵列；四肢的外侧有锯齿缘，趾间有蹼。

体色背面为暗橄榄褐色，腹面淡黄色，幼体颜色较淡，有黑色的斑點及網狀花紋。其下顎第四齒由上顎的V字形凹陷中向外面突出。尼羅鱷非常強壯，[[尾巴|尾巴]]強而有力，有助於游泳。成年尼羅鱷的體重可以重達一[[噸|噸]]。

== 分佈 ==
尼羅鱷出現在[[非洲|非洲]]東部、中部及南部的大部分地區，以及[[馬達加斯加|馬達加斯加]]島，主要棲身於[[河流|河流]]及[[湖泊|湖泊]]之中。自公元9世紀開始，牠們已經成為[[人類|人類]]的獵殺對象，因為牠們的[[皮|皮]]於當時非常名貴。目前在[[尼羅河|尼羅河]]一帶，尼羅鱷已經絕跡。

== 習性 ==
尼羅鱷喜愛獨居，尼羅鱷夜間會在水內，日出時則會上岸[[日光浴|日光浴]]。牠們會埋伏在水池邊緣，偷襲捕食[[羚羊|羚羊]]、[[斑馬|斑馬]]、[[水牛|水牛]]等，甚至可以獵殺[[河馬|河馬]]（只食用河馬屍體，或捕食年紀衰老及生病受傷的河馬，或河馬幼崽，不會去攻擊健康強壯的成年河馬）、[[獅子|獅子]]（通常避免硬碰硬）以及[[人類|人類]]（以上三種動物也能獵殺鱷魚）。成年尼羅鱷會吞下[[石塊|石塊]]以作[[壓艙物|壓艙物]]之用，有助於水底保持平衡。在[[毛里塔尼亚|毛里塔尼亚]][[旱季|旱季]]期間，尼羅鱷會躲藏於地底之下，直到下一個[[雨季|雨季]]來臨為止。尼羅鱷身體的[[代謝作用|代謝作用]]很低，並能夠藉日光浴保持體溫，所以並不需要大量的食物，一次吃下羚羊後，可以一年內不用吃任何東西。<ref>[http://zoo.e-tobe.com/animal.php?sub=15&cata=3&id=24 新竹市立動物園-尼羅鱷]</ref>

繁殖方面，雌性尼羅鱷會於沙質的河岸挖洞造巢，每次可生25至100隻[[蛋|蛋]]。對於這種鱷魚的蛋來說，[[水災|水災]]同[[尼羅河巨蜥|尼羅河巨蜥]]是最大的威脅。
