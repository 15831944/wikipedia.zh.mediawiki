'''尼辻站'''（{{jpn|j=尼ヶ辻駅|hg=あまがつじえき|rm=Amagatsuji eki|f=y}}）位於[[日本|日本]][[奈良县|奈良县]][[奈良市|奈良市]]尼辻中町，是[[近畿日本鐵道|近畿日本鐵道]]（近鐵）[[橿原線|橿原線]]上的[[火車站|火車站]]<ref name="mainichi-np-2014-10-21" />。車站編號是'''B27'''。

[[江户时代|江户时代]]時，因[[德川家纲|德川家纲]][[征夷大將軍|將軍]]的捐献而尔后迁至法莲村（今[[奈良市|奈良市]]法莲町）的{{tsl|ja|尼寺|尼寺}}{{tsl|ja|興福院|興福院}}位於[[平城京|平城京]]{{tsl|ja|三条通り|三条大路}}和西一坊大路的交叉路口（日语中称之为“{{lang|ja|[[:wikt:辻|辻]]}}”）附近，故該地區名為“尼辻”（{{lang-ja|尼ヶ辻}}），该站亦由此得名。<ref name="mainichi-np-2014-10-21" />

== 歴史 ==
* [[1921年|1921年]]（[[大正|大正]]10年）[[4月1日|4月1日]] - {{tsl|ja|大阪電気軌道|大阪電氣軌道}}畝傍線（現橿原線）西大寺（現{{tsl|ja|大和西大寺駅|大和西大寺}}） - 郡山（現{{tsl|ja|近鉄郡山駅|近鐵郡山}}）區間開通時启用。<ref name=sone03>{{Cite book|author={{tsl|ja|曽根悟||曽根悟}}（監修） |title=週刊 歴史でめぐる鉄道全路線 大手私鉄 |date=2010-08-29 |publisher={{tsl|ja|朝日新聞出版||朝日新聞出版}} |series=週刊朝日百科 |volume=3号 近畿日本鉄道 2 |isbn=978-4-02-340133-4 |pages=11-13}}</ref>
* [[1941年|1941年]]（[[昭和|昭和]]16年）[[3月15日|3月15日]] - 由於與參宮急行電鐵合併，該站成為關西急行鐵道的車站。<ref name=sone03/>
* [[1944年|1944年]]（昭和19年）[[6月1日|6月1日]] - 由于公司合并，成为近畿日本铁道的车站。<ref name=sone03/>
* [[2000年|2000年]]（[[平成|平成]]12年）[[12月10日|12月10日]] - 站厅地下化。
* [[2007年|2007年]]（平成19年）[[4月1日|4月1日]] - 开始使用[[PiTaPa|PiTaPa]]<ref>{{Cite press release |title=平成19年4月1日から、近鉄主要路線でICカードの利用が可能になります |publisher=近畿日本鉄道 |date=2007-01-30 |url=http://www.kintetsu.jp/news/files/iccard20070130.pdf |format=pdf |accessdate=2016-02-26}}</ref>。

== 車站構造 ==
尼辻站是一座[[地上車站|地上]][[側式站台|側式站台]]车站，站台有效長度为四节長。检票口和中央大厅設在地下，站台设在地上。檢票口各設在東西兩處，出入口位於大和西大寺方向的站台兩側。

作为{{tsl|ja|天理駅#近畿日本鉄道|天理站}}管理的[[鐵路車站|有人站]]，尼辻站設置了適用於[[PiTaPa|PiTaPa]]和[[ICOCA|ICOCA]]的[[驗票閘門|自動驗票機]]以及{{tsl|ja|自動精算機|自動補票機}}（也適用於[[回数券|回数券]]及IC卡的充值）。
