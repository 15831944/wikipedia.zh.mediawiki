'''尼爾森·強森'''（{{lang-en|'''Nelson Johnson'''}}，{{bd|1948年}}）是一名[[美國|美國]]律師和作家，他最著名的暢銷書是《''Boardwalk Empire: The Birth, High Times and Corruption of Atlantic City''》。<ref name=scotsman>{{Cite web |url = http://www.scotsman.com/lifestyle/book-review-boardwalk-empire-by-nelson-johnson-1-1494086 |title = Book review: Boardwalk Empire by Nelson Johnson |date = 2011-01-28 |publisher = The Scotsman |language = en}}</ref>

==早年==
強森於美國的{{link-en|南澤西|South Jersey}}成長。<ref name=nj>{{Cite web |url = http://www.nj.com/entertainment/arts/index.ssf/2010/12/northside_a_book_review.html |title = "Northside," a book review |date = 2010-12-19 |publisher = NJ.com |language = en}}</ref>

==生涯==
1980年代期間，強森曾於[[大西洋城|大西洋城]]的規劃委員會裏工作。截至2014年，他是大西洋縣高等法院法官。<ref name=scotsman />

他的著作《''Boardwalk Empire''》被[[HBO|HBO]]改編成電視劇《[[酒私風雲|酒私風雲]]》。<ref name=nj />該書的續集是《''The Northside: African Americans and the Creation of Atlantic City''》。<ref name=nj /><ref>{{Cite web |url = http://www.pressofatlanticcity.com/life/book-review-johnson-gets-it-right-with-tale-of-the/article_4d5d32bd-1107-5865-8fc1-c4665fd8e3d6.html |title = Book Review: Johnson gets it right with tale of 'The Northside' |author = Patricia Reid-Merritt |date = 2010-11-21 |publisher = Press of Atlantic City |language = en}}</ref>2010年，他被新澤西州高等法院要求停止宣傳他自己的著作和電視劇，原因是為了維持他作為一位法官的中立立場。<ref>{{Cite web |url = http://observer.com/2010/09/boardwalk-empire-author-muzzled/ |title = Boardwalk Empire Author Muzzled |author = Michael H. Miller |date = 2010-09-15 |publisher = Observer |language = en}}</ref>

他的第三本著作是《''Battleground New Jersey: Vanderbilt, Hague and Their Fight for Justice''》，內容講述[[澤西市|澤西市]]市長{{link-en|法蘭克·赫格|Frank Hague}}及新澤西州高等法院第一位首席法官{{link-en|阿瑟·T·范德比爾特|Arthur T. Vanderbilt}}。<ref>{{Cite web |url = http://www.mycentraljersey.com/story/news/local/somerset-county/2015/01/16/judge-details-state-politics-history-book/21866329/ |title = Judge details state politics history in book |author = Katie Park |date = 2015-01-16 |publisher = MyCentralJersey.com |language = en}}</ref>
