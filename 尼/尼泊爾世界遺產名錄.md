由於[[文化遺產|文化遺產]]和[[自然遺產|自然遺產]]越來越受到破壞威脅，[[聯合國教科文組織|聯合國教科文組織]]大會於1972年10月17日至11月21日在巴黎舉行第十七屆會議<ref name=convention>{{cite web |url=http://whc.unesco.org/en/convention/ |title=The World Heritage Convention |publisher=UNESCO |accessdate=2010-09-21}}</ref>，並於11月16日在法國巴黎通過《[[保護世界文化和自然遺產公約|保護世界文化和自然遺產公約]]》。尼泊爾自1978年6月20日批准《保護世界文化和自然遺產公約》，並在1979年開始有歷史遺蹟被列入世界遺產；截至2017年，尼泊爾已有4處世界遺產，其中2處為文化遺產，另2處為自然遺產，還有15處在預選名單內。<ref name="Nepal">{{cite web |url=http://whc.unesco.org/en/statesparties/np/|title=Nepal - UNESCO World Heritage Centre|publisher=[[UNESCO|UNESCO]] |accessdate=2016-11-06}}</ref>1979年，尼泊爾的[[加德滿都谷地|加德滿都谷地]]與[[薩加瑪塔國家公園|薩加瑪塔國家公園]]一同被聯合國教科文組織列入世界遺產名錄中，而這也是該國首次有歷史遺產被列入世界遺產名錄中。<ref name="SitebyYear">{{Cite web |url=http://whc.unesco.org/pg.cfm?cid=31&l=en&action=stat&mode=table#s4 |title=Number of World Heritage properties inscribed each Year |publisher=[[UNESCO|UNESCO]] |accessdate=2011-09-08}}</ref>

然而，[[加德滿都谷地|加德滿都谷地]]在2003年時由於周邊的城市開發未加妥善，危及到一旁的歷史遺跡，導致該歷史遺址群中有6分之一的傳統元素受到損害，因而曾被列入[[處於危險的世界遺產目錄|處於危險的世界遺產目錄]]中<ref name="dangerlist">{{cite web |url=http://whc.unesco.org/en/158/ |title=World Heritage in Danger |publisher=[[UNESCO|UNESCO]] |accessdate=2010-05-28}}</ref>，在遺跡加強管理與維護後才於2007年得以解除。<ref name="removed"/>

| rowspan=7 |[[加德滿都谷地|加德滿都谷地]]位於尼泊爾的政治文化中心，此景點包含7處歷史遺址群和建築群，包括[[印度教|印度教]]和[[佛教|佛教]]建築遺址。分別散布在[[加德滿都|加德滿都]]、[[拉利特普爾_(尼泊爾)|拉利特普爾]]、[[巴克塔普爾|巴克塔普爾]]，這三座城市曾經為古代王城，目前皆屬[[巴格馬蒂專區|巴格馬蒂專區]]。這7處歷史遺址群分別為位於加德滿都的[[加德滿都王宮廣場|加德滿都王宮廣場]]、[[帕舒帕蒂納特廟|帕舒帕蒂納特廟]]、[[斯瓦揚布納特|斯瓦揚布納特]]、[[布達納特|布達納特]]，位於拉利特普爾的[[帕坦王宮廣場|帕坦王宮廣場]]與[[巴克塔普爾王宮廣場|巴克塔普爾王宮廣場]]，以及位於巴克塔普爾的[[昌古·納拉揚廟|昌古·納拉揚廟]]。這些建築有著特殊的雕刻裝飾品，或是以半球體與鍍金立方體組成的佛塔等，標誌著佛教與印度教建築藝術在尼泊爾融合並繁榮的景況，因而在1979年與[[薩加瑪塔國家公園|薩加瑪塔國家公園]]一同被[[聯合國教科文組織|聯合國教科文組織]]納入世界遺產名錄，是尼泊爾最先被列入的文化遺產。 

在2003年至2007年間，由於加德滿都谷地周邊的城市開發未加妥善，危及到一旁的歷史遺跡，導致這7處歷史遺址群中有6分之一的傳統元素受到損害，因而曾被列入[[處於危險的世界遺產目錄|處於危險的世界遺產目錄]]中，在遺跡加強管理與維護後才得以解除。2015年4月25日，尼泊爾[[廓爾喀縣|廓爾喀縣]]發生一起[[2015年尼泊爾地震|大地震]]，使得加德滿都谷地的古建築遺跡受到毀壞，不少建築不是坍塌，就是建築受到損害。
