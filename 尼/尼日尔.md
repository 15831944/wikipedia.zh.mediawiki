'''尼日尔共和国'''（{{lang-fr|République du Niger}}） 通稱'''尼日'''，是[[西非|西非]][[内陆国家|内陆国家]]之一，因[[尼日尔河|尼日尔河]]得名，首都[[尼亚美|尼亚美]]。东临[[乍得|乍得]]，南界[[尼日利亚|尼日利亚]]、[[贝宁|贝宁]]，西部与[[布基纳法索|布基纳法索]]和[[马里|马里]]毗邻，北部与[[阿尔及利亚|阿尔及利亚]]接壤，东北与[[利比亚|利比亚]]交界。边境总长5500公里。面积126.76万平方公里，是世界上[[最低度開發國家|最低度開發國家]]，亦是[[人類發展指數|人類發展指數]]排名最低的國家(根據2017年所統計的資料)。

== 历史 ==
{{main|尼日尔历史}}

公元8至16世纪，在尼日尔河中上游地区先后出现过三个强大国家：[[加纳帝国|加纳帝国]]、[[马里帝国|马里帝国]]和[[桑海帝国|桑海帝国]]。11至18世纪在[[乍得湖|乍得湖]]地区兴起了[[博尔努帝国|博尔努帝国]]。加纳王国和马里帝国的统治中心虽然不在尼日尔，但通过贸易往来而对该地区产生了很大影响。而尼日尔的西部和中部都曾经是桑海帝国的一部分。

1897年～1900年被[[法蘭西第三共和國|法国]]佔领，1904年成為法國殖民地（法屬尼日軍管地），1940年代并入[[法屬西非|法屬西非]]。

1960年8月3日獨立，[[哈马尼·迪奥里|哈馬尼·迪奧-{里}-]]成為首任總統。1974年4月，武裝部隊參謀長[[賽義尼·孔切|賽義尼·孔切]]發動政變奪權。孔切一直掌權至1987年11月去世，由[[阿里·賽義布|阿-{里}-·賽義布]]繼任。1993年3月，[[馬哈曼·奥斯曼|馬哈曼·奥斯曼]]在該國首次民主總統選舉中勝出，不過他的政府在1996年1月被軍人發動政變推翻。[[坦賈·馬馬杜|坦賈·馬馬杜]]在1999年11月當選總統，2004年12月成功連任，後於2010年2月18日被軍隊發動政變推翻。<ref>[http://www.bbc.co.uk/zhongwen/simp/world/2010/02/100218_niger_coup.shtml 《尼-{日}-尔发生兵变总统被扣》]，BBC中文網，2010年2月18日。</ref>

== 政治 ==
實行半總統制。總統為國家元首、行政首腦和武裝部隊統帥，通過兩輪多數選舉產生，任期5年，可連選連任一次。即將任滿兩屆總統任期的坦賈在2009年提出修改憲法，更改連任限制，引起爭議。

總理為政府首腦，由總統根據議會多數黨團的提名人選任命，對議會負責。

* 2004年當選的總統[[坦贾·马马杜|坦贾·马马杜]]在2010年2月的军事政变被推翻。

== 地理 ==

尼日領土面積126.7萬平方公里。国土纵横的范围，从北纬11°37′延伸到23°33′，经度上东经0°6′伸展到16°。

=== 地形 ===
尼日尔全境地势呈北高南低。按地形特征大致可分为三部分：

北部是[[艾尔高原|艾尔高原]]及周围群山，地势较高，海拔700～1000米。其中位于[[塔泽尔扎伊特|塔泽尔扎伊特]]山区的[[格雷本山|格雷本山]]海拔1997米，是尼日尔的最高点。

中部是[[萨赫勒|萨赫勒]]地区，是[[半沙漠草原区|半沙漠草原区]]，海拔500～800米。

南部是[[平原|平原]]区，地势较平坦，海拔200～300米。

=== 河流与湖泊 ===
[[尼日尔河|尼日尔河]]是最主要的河流，另东部還有一条[[科马杜古-约贝河|科马杜古-约贝河]]，流入[[乍得湖|乍得湖]]。此外还有一些[[时令河|时令河]]。总体来说，尼日尔水资源比较贫乏。

=== 气候 ===
尼日尔气候属于[[干旱热带性|干旱热带性]]，终年炎热少雨。平均气温摄氏30 °C。雨季从6月到9月，旱季从10月到第二年5月。降水量由西往东逐渐递减。

== 居民与宗教 ==

=== 人口 ===
2011年估計尼日尔全国人口1574萬。人口密度为每平方公里5人。人口主要集中在[[尼亚美|尼亚美]]及其周边地区。人口结构相对年轻，65岁以上的老人占总人口的2%。

=== 民族 ===
全國有5個主要部族：[[豪薩族|豪薩族]]（佔全國人口56%）、[[哲爾馬-桑海族|哲爾馬-桑海族]]（22%）、[[富拉尼人|富拉尼人]]（8.5%）、[[圖阿雷格族|圖阿雷格族]]（8%）和[[卡努里族|卡努里族]]（4%）。

=== 语言 ===
官方語言為[[法語|法語]]，但日常使用的人不多。各部族均有自己的語言，[[豪薩語|豪薩語]]可在全國大部分地區通用。

=== 宗教信仰 ===
尼日尔是一个世俗国家，施行政教分离制度，该制度由2010年的宪法所保障，并且规定未来的修订不得改变尼日尔是世俗国家的状态。同时，宪法也保障宗教自由。从10世纪开始广泛传播的伊斯兰教深刻影响了尼日尔的文化和人群。超過99%的居民信奉[[伊斯蘭教|伊斯蘭教]]，當中約90%是[[遜尼派|遜尼派]]，約5%是[[什葉派|什葉派]]；其餘居民信奉原始宗教、[[基督教|基督教]]等。

== 经济 ==
===經濟產業===
农业是尼日尔最基本的经济生产部门。但由于长期干旱，粮食长时间不能自给。畜牧业是尼日尔的经济支柱之一，在出口贸易中仅次于铀矿业，在国内生产总值中占15%～20%，占外汇收入的50%。此外尼日爾是鈾元素出口國之一，尼日尔的铀矿发现于20世纪50年代。现已探明储量为21万吨，占世界总储量的11%，居世界第五位。1992年，尼日尔成为仅次于[[加拿大|加拿大]]、[[澳大利亚|澳大利亚]]和[[美国|美国]]的世界第四大产铀国。此外尼日尔的采金业近年来发展迅速。主要产金地是[[萨米拉|萨米拉]]（Samira）和[[利比里|利比里]]（Libiri）。並且尼日尔有比较丰富的[[磷酸盐|磷酸盐]]，储量达12.5亿吨，居世界第四位。此外，尼日尔的[[煤炭|煤炭]]和[[石油|石油]]资源也比较重要。
===發展程度===
尼日爾的發展十分落後，甚至比其他非洲國家還要落後。根據聯合國發布的2017年人類發展指數，尼日爾排行最後，而人均收入亦十分低，在2018年尼日爾人均收入只有約1200美元(購買力平價)。有超過40%的人口每日生活低於1.9美元。

== 外交 ==
{{主條目|中華民國與尼日關係|中國－尼日關係}}
尼日尔1963年7月7日与[[中华民国|中华民国]]建交，1974年7月20日改同[[中华人民共和国|中华人民共和国]]建交，1992年7月30日尼日尔再同中华民国复交四年，直至1996年8月19日尼日尔同中华人民共和国二度复交。

== 教育 ==
尼日尔教育水準十分落后，成人识字率只有16.5%，妇女更低，只有7%。

目前尼日尔有两所大学，一所是[[尼亚美阿卜杜·穆穆尼大学|尼亚美阿卜杜·穆穆尼大学]]，另一所是设在[[萨伊|萨伊]]地区的[[萨伊伊斯兰大学|萨伊伊斯兰大学]]。前者是国立大学，后者是[[伊斯兰会议组织|伊斯兰会议组织]]开设的。
