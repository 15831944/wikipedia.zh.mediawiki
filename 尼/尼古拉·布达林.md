'''尼古拉·米哈伊洛维奇·布达林'''（{{lang-ru|Никола́й Миха́йлович Буда́рин}}，{{bd|1953年|4月29日|}}）是[[俄罗斯|俄罗斯]][[政治人物|政治人物]]，已退役[[宇航员|宇航员]]，俄罗斯第82位宇航员， 世界第326位宇航员。

==早年==
1953年出生在[[苏联|苏联]][[楚瓦什共和国|楚瓦什共和国]][[阿拉特爾斯基區|阿拉特爾斯基區]]基尔亚。1979年毕业于[[格里戈里·康斯坦丁诺维奇·奥尔忠尼启则|奥尔忠尼启则]][[莫斯科航空学院|莫斯科航空学院]]，获得[[机械工程|机械工程]]学位。自1976年以来，他一直在[[科罗廖夫能源火箭航天集团|科罗廖夫能源火箭航天集团]]担任工程师<ref name=nasabio>{{cite web |title=Nikolai Mikhailovich Budarin |url=https://web.archive.org/web/20170113030859/http://www.jsc.nasa.gov/Bios/htmlbios/budarin.html |publisher=[[NASA|NASA]] |accessdate=2019-04-18}}</ref>。
==宇航员生涯==
1989年2月，他成为测试宇航员候选人。1989年9月至1991年1月在[[加加林宇航员培训中心|加加林宇航员培训中心]]接受基础太空训练课程并通过了考验，获得测试宇航员资格。1991年2月至1993年12月参加了联盟-TM系列载人飞船与[[和平号空间站|和平号空间站]]对接任务的高级专业培训<ref name=nasabio/>。
===第一次===
1995年6月27日至9月11日，他作为飞行工程师，与[[阿纳托利·索洛维约夫|阿纳托利·索洛维约夫]]参加了前往和平号空间站的第19次远征任务。这是俄罗斯宇航员首次乘坐美国的航天飞机（[[亚特兰蒂斯号航天飞机|亚特兰蒂斯号]]，STS-71）前往空间站。7月14日，他进行了生涯中的第一次太空行走，共持续5小时34分钟<ref>{{cite book |author=Ben Evans |title=The Twenty-first Century in Space |date=2014 |publisher=Springer |isbn=9781493913077 |url=https://books.google.com/books?id=BH7ZBQAAQBAJ&pg=PA195&|page=195}}</ref>。任务期间，他一共进行了3次[[舱外活动|舱外活动]]任务，共计14小时17分。任务结束后乘坐[[联盟TM-21|联盟TM-21]]返回地球。本次任务总时长75天11小时20分<ref name=ru>{{cite web |title=Nikolai Mikhailovich BUDARIN |url=https://www.energia.ru/en/iss/iss06/budarin.html |publisher=S.P. Korolev RSC "Energia" |accessdate=2019-04-18}}</ref>。
===第二次===
1998年1月29日至8月25日，作为[[联盟TM-27|联盟TM-27]]的飞行工程师，参加了前往和平号空间站的第25次远征任务。在执行任务期间，他一共进行了5次舱外活动，共计30小时8分钟。任务结束后乘坐联盟TM-27返回地球。本次任务总时长207天12小时51分<ref name=ru/>。
===第三次===
[[File:ISS_Expedition_6_crew.jpg|thumb]]
[[格林尼治时间|格林尼治时间]]2002年11月24日0时49分，美国[[奋进号航天飞机|奋进号航天飞机]]从[[佛罗里达州|佛罗里达州]][[肯尼迪航天中心|肯尼迪航天中心]]发射升空，执行编号为[[STS-113|STS-113]]的太空飞行任务。机上载有7名机组人员，其中布达林和美国宇航员[[肯·鲍尔索克斯|肯·鲍尔索克斯]]、[[唐纳德·佩蒂特|唐纳德·佩蒂特]]是[[远征6|远征6]]任务成员<ref>{{cite journal |author=许彤 |title=第六批长住客进驻国际空间站 |journal=太空探索 |date=2003年 |issue=第1期 |issn=1009-6205}}</ref>。

2003年2月1日，[[哥伦比亚号航天飞机|哥伦比亚号航天飞机]][[STS-107|STS-107]]发射任务[[哥伦比亚号航天飞机灾难|发生空难]]，造成机上7名成员全部遇难。同日，[[美国航空航天局|美国航空航天局]]决定对哥伦比亚号航天飞机出事原因进行调查，期间暂停所有航天飞机飞行<ref>{{cite web |author=邱冰 |title=关注国际空间站3名宇航员状况 还能撑多久 |url=http://news.sina.com.cn/w/2003-02-06/0948898642.shtml |publisher=[[北京青年报|北京青年报]] |accessdate=2019-04-18 |date=2003-02-06}}</ref>。当时仍在国际空间站执行任务的他和同事的身心受到了巨大打击。2月4日，他们在空间站通过[[无线电|无线电]]通讯方式参加了遇难者悼念仪式<ref>{{cite web |author=清晨 |title=国际空间站3名宇航员将在太空参加悼念仪式 |url=http://news.sina.com.cn/w/2003-02-04/1808897676.shtml |publisher=[[新浪|新浪]]科技 |accessdate=2019-04-18 |archiveurl=https://web.archive.org/web/20041104074304/http://news.sina.com.cn/w/2003-02-04/1808897676.shtml |archivedate=2004-11-04 |location=[[莫斯科|莫斯科]] |date=2003-02-04}}</ref>。

[[格林尼治时间|格林尼治时间]]2003年5月4日2点07分（北京时间10点07分），布达林、鲍尔索克斯和佩蒂特乘坐[[联盟TMA-1|联盟TMA-1]]返回舱在[[哈萨克斯坦|哈萨克斯坦]][[阿尔卡雷克|阿尔卡雷克]]以北55[[英里|英里]]处安全降落<ref>{{cite web |author1=闻新芳 |title=美俄宇航员安全着陆 |url=http://news.sohu.com/47/81/news209028147.shtml |publisher=搜狐科技 |accessdate=2019-04-18 |archiveurl=https://web.archive.org/web/20050407115120/http://news.sohu.com/47/81/news209028147.shtml |archivedate=2005-04-07 |date=2003-05-04}}</ref>。这个降落地点偏离了预定降落地点440公里，致使俄罗斯救援人员搜寻了近两个小时<ref>{{cite web |author=赵建中 |title=俄救援小组在哈萨克斯大草原找到三名宇航员 |url=http://news.southcn.com/international/zhuanti/spaceship/new/200305040192.htm |publisher=新浪科技 |archiveurl=https://web.archive.org/web/20190418030407/http://news.southcn.com/international/zhuanti/spaceship/new/200305040192.htm |archivedate=2019-04-18}}</ref>。
