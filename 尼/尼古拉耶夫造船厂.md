'''尼古拉耶夫造船厂'''，又名'''黑海造船厂'''，是一家位于[[乌克兰|乌克兰]][[尼古拉耶夫州|尼古拉耶夫州]][[尼古拉耶夫_(乌克兰)|尼古拉耶夫]]的造船企业，成立于1895年，是[[苏联|苏联]][[红海军|海军]]的主要造船厂，曾是欧洲最大的造船企业之一<ref>{{cite web |title=尼古拉耶夫州 |url=http://odessa.chineseconsulate.org/chn/lqgk/t935075.htm |publisher=中华人民共和国驻敖德萨总领事馆 |accessdate=2018-07-05}}</ref>。

尼古拉耶夫造船厂建造了所有红海军、[[俄罗斯|俄罗斯]][[俄罗斯海军|海军]]的[[航空母舰|航空母舰]]，包括两艘[[莫斯科级直升机航空母舰|莫斯科级直升机航空母舰]]（[[莫斯科号直升机航空母舰|莫斯科号]]、[[列宁格勒号直升机航空母舰|列宁格勒号]]）、四艘[[基辅级航空母舰|基辅级航空母舰]]（[[基辅号航空母舰|基辅号]]、[[明斯克号航空母舰|明斯克号]]、[[新罗西斯克号航空母舰|新罗西斯克号]]、[[戈尔什科夫苏联海军元帅号航空母舰|戈尔什科夫号]]（原名巴库号，后转给[[印度海军|印度]]更名为[[超日王号航空母舰|维克拉玛蒂亚号]]））、[[库兹涅佐夫号航空母舰|库兹涅佐夫号航空母舰]]等，除此之外还有[[瓦良格号巡洋舰_(1983年)|瓦良格号导弹巡洋舰]]等。<ref>{{cite web |title=Крейсеры |url=http://smart-maritime.com/ru/reference-list/kreysery/ |accessdate=2018-07-05}}</ref><ref>{{cite web |title=Суда ВМФ |url=http://smart-maritime.com/ru/reference-list/suda-vmf/ |accessdate=2018-07-05}}</ref>{{来源请求}}在此建造但未完成的有[[瓦良格号航空母舰|瓦良格号航空母舰]]（后由[[大连造船厂|大连造船厂]]续建，现为[[中国人民解放军海军|中国人民解放军海军]]辽宁舰）<ref>{{cite journal |author1=海韬 |coauthors=梁嘉 |title=中国首艘航母“诞生”记 |journal=人民文摘 |issue=2011年第6期 |url=http://paper.people.com.cn/rmwz/html/2011-06/01/content_900534.htm |accessdate=2018-07-05}}</ref>、[[乌里杨诺夫斯克号航空母舰|乌里杨诺夫斯克号航空母舰]]<ref>{{cite web |title=Проект 1143.7 |url=http://smart-maritime.com/ru/reference-list/kreysery/proekt-1143-7/ |accessdate=2018-07-05}}</ref>等。

[[苏联解体|苏联解体]]后，由于乌克兰经济不景气、缺少订单等原因，尼古拉耶夫造船厂被分成总装厂、造机、鱼船、铸锻等四个独立部分，尽管有乌克兰政府的免税政策扶持，依然日渐衰落。<ref>{{cite web |title=探访黑海地区最大造船厂 |url=http://www.81.cn/jskj/2015-03/13/content_6396315.htm |publisher=中国军网 |accessdate=2018-07-05}}</ref>自2014年底起，尼古拉耶夫造船厂曾一度拖欠职工数年工资，一周只上两天班，乌克兰政府曾考虑出售乌克兰号导弹巡洋舰以清偿职工工资。<ref>{{cite news |title=“乌克兰”号巡洋舰卖给中国？外界阻挠 乌政府毫无作为 |url=http://www.cankaoxiaoxi.com/mil/20180109/2251065.shtml |accessdate=2018-07-05 |work=参考消息网 |date=2018-01-09}}</ref>

据乌克兰媒体报道，尼古拉耶夫造船厂已于2018年7月3日被当地经济法院宣布破产。<ref>{{cite news |title=Суд признал Черноморский судостроительный завод банкротом |url=https://www.mk.mk.ua/rubric/our-life/2018/07/05/56973/ |accessdate=2018-07-05}}</ref>
