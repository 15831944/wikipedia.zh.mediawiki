'''尼科西亞'''（{{lang-el|'''Λευκωσία'''}}、{{lang-tr|'''Lefkoşa'''}}），是[[塞浦路斯共和国|塞浦路斯共和国]]及[[北塞浦路斯土耳其共和国|北塞浦路斯土耳其共和国]]的首都，也同時為[[塞浦路斯島|塞浦路斯島]]上最大的都市。首都被分為南部的[[希臘人|希臘人]]及北部的[[土耳其人|土耳其人]]控制，是仅有的[[国家分裂|国家分裂]]之后南北政权皆定于此的首都，然而國際上承認尼科西亚為南塞浦路斯的首都。

尼古西亞位於[[賽普勒斯島|賽普勒斯島]]中部、臨[[派迪亞斯河|派迪亞斯河]]，北依橫越島國北岸的[[凱里尼亞山脈|凱里尼亞山脈]]，西南臨[[海拔|海拔]]約150[[米_(單位)|米]]的[[特魯多斯山|特魯多斯山]]。城市總[[面積|面積]]50.5[[平方公里|平方公里]]，[[人口|人口]]近40萬。

尼科西亞市也為[[中央政府|中央政府]]所在地及全島主要的[[商業中心|商業中心]]，同時也是[[尼科西亞區|尼科西亞區]]的[[行政中心|首府]]。此城市也被[[全球化與世界級城市研究小組與網路|全球化與世界級城市研究小組與網路]]（GaWC）指定為[[全球城市|B-級世界都市]]（Beta-）。

== 歷史 ==
尼科西亞在[[古代|古代]]是一個被稱為[[萊德拉|萊德拉]]（Ledra或Ledrae）的[[城邦|城邦]]。萊德拉的國王——[[奧納薩苟瑞斯|奧納薩苟瑞斯]]（Onasagoras）在前672年曾向[[亞述|亞述]]國王[[阿薩爾哈東|以撒哈頓]]進貢。後大約於前300年由[[托勒密一世|托勒密一世]]之子[[里夫科斯|里夫科斯]]（Lefkos）重建。在[[古希臘|古希臘]]、[[古羅馬|羅馬]]時期，萊德拉是一個毫不起眼的小鎮，在當時被稱為[[里夫科西亞|里夫科西亞]]（Lefkothea）。348年，第一次有[[基督教|基督教]]的[[主教|主教]]——[[特力菲力歐斯|特力菲力歐斯]]（Trifillios）踏入此鎮。約10世紀時，沿海城市[[帕福斯|帕福斯]]及[[瑟拉米斯|瑟拉米斯]]（Salamis）在此時受到威脅，居民開始遷往內陸，使得尼科西亞逐漸變得重要，也成為了島上的首府。城市中心保留了16世紀時期的威尼斯[[要塞|要塞]]，將[[古代|古代]]及[[中世紀|中世紀]]的部分城市給包圍了起來，目前保存完整。[[萊德拉街|萊德拉街]]（Ledra）是現在最熱門的商業街。

1192年塞浦路斯國王[[呂西尼昂|呂西尼昂]]（Lusignans）即位，1489年時成為了[[威尼斯|威尼斯]]的屬地，1570年大約兩萬名居民死於奧斯曼帝國的圍城，後來在1571年被[[奧斯曼帝國|奧斯曼帝國]]征服。

呂西尼昂國王的即位，也造成了「尼科西亞」一詞的出現。原因是[[法蘭克|法蘭克]][[十字軍|十字軍]]將里夫科西亞（Lefkothea）的發音，唸成「尼科西亞」（Nicosia），此訛傳成為了現在廣為人知的名稱之一。在這時期的法蘭西人，經由城市將文化擴展出去，並在15、16世紀，建立許多的[[宮殿|宮殿]]、[[豪宅|豪宅]]、[[教堂|教堂]]和[[修道院|修道院]]。呂西尼昂國王的陵寢位於[[聖索菲亞大教堂_(尼科西亞)|聖索菲亞大教堂]]（St. Sophia Cathedral）中，現址在北方的土耳其人佔領區，被改宗成為了[[塞利米耶清真寺_(尼科西亞)|塞利米耶清真寺]]。1821年：[[希臘人|希臘人]]以武力反抗了土耳其奧斯曼帝國統治，其後成功獨立。1835年霍亂襲擊了尼科西亞。1857年大火燒毀了城市的大片區域。

1878年[[大英帝國|大英帝國]]控制了塞浦路斯，並以塞浦路斯作為地中海的重要軍事基地，將尼科西亞設為首府。

1960年塞浦路斯脫離英國獨立時，尼科西亞遭遇大型[[暴亂|暴亂]]。

1974年因希臘所支持的政變造成[[土耳其|土耳其]]入侵，部分城市區也被[[土耳其人|土耳其人]]佔領，使首都一分為二，[[北賽普勒斯|北賽普勒斯]]對外宣稱[[北尼科西亚|北尼科西亚]]為其首都。目前佔領區的[[邊界|邊界]]由[[聯合國軍隊|聯合國軍隊]]進駐，為[[塞浦路斯聯合國緩衝區|塞浦路斯聯合國緩衝區]]。

2003年，歷經30年隔絕後開放邊境。2008年4月3日，[[莱德拉街|莱德拉街]]打開了分隔南北兩區的關卡。

[[賽普勒斯聯合國緩衝區|綠線]]的南方為國際上承認的尼科西亞首都範圍，各國不少大使館設於尼科西亞，城市的人口是31.34萬（至2009年底），而北部的人口為8萬4893人。尼科西亞有許多商店、兩座現代化購物商場、餐館和娛樂場所，是重要的商業區。這個城市是一個貿易中心，生產[[紡織品|紡織品]]，[[皮革|皮革]]，[[陶瓷材料|陶瓷]]，[[塑膠|塑膠]]製品等。城市附近有[[銅礦|銅礦]]。[[賽普勒斯大學|賽普勒斯大學]]（UCY）和其他四所大學也座落於此。

== 政治 ==
自古以來，尼科西亞就是賽普勒斯的中心，無論是在[[地理|地理]]、[[運輸|運輸]]及[[經濟|經濟]]。此有利條件及易於管理的情況下，成為賽普勒斯之首府。

=== 政府 ===
[[File:Parlanet_cyprus.jpg|thumb]]
賽普勒斯共和國的總統府就座落在尼科西亞市之中，前英國殖民的[[總督|總督]]也於此市辦公。許多[[政府機構|政府機構]]和各[[部會|部會]]也設在此。

=== 自治市 ===
[[File:NICOSIA,_11_AUGUST,_2011_114.jpg|thumb]]、[[尼科西亚市旗|尼科西亚市旗]]、[[塞浦路斯国旗|塞浦路斯国旗]]和[[希腊国旗|希腊国旗]]]]
政治上由市長管理，目前的市長是[[艾勒尼‧馬弗羅|艾勒尼‧馬弗羅]]女士（希臘語：Ελένη Μαύρου；英語：Eleni Mavrou）。

== 教堂 ==
在尼科西亞市有16世紀遺留的[[威尼斯之牆|威尼斯之牆]]，還有[[賽普勒斯大主教|賽普勒斯大主教]]的宅邸。

== 氣候 ==
尼科西亞為[[半乾旱氣候|半乾旱氣候]]<ref>http://upload.wikimedia.org/wikipedia/commons/7/7f/World_Koppen_Map.jpg</ref>，特性為長時間乾熱的夏天，和相對溫和濕潤的冬天。
