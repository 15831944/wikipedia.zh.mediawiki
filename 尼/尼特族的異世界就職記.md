《'''尼特族的異世界就職記'''》（{{lang-ja|ニートだけどハロワにいったら異世界につれてかれた}}）是日本小說家[[桂かすが|桂かすが]]創作的[[輕小說|輕小說]]，最初是發表於小說投稿網站《[[成為小說家吧|成為小說家吧]]》的網路連載小說。<ref>{{cite web |title=23歳ニートが異世界に就職！野ウサギ退治から始まる冒険ライフ、マンガ版1巻 |url=https://natalie.mu/comic/news/297334 |website=natalie |accessdate=2019-03-24}}</ref>

==簡介==
23歲的[[尼特族|尼特族]]山野·勝，找到一份月薪25萬日幣的遊戲測試的工作。在簽下就職契約時，被轉移到具有魔法的異世界。<ref>{{cite web |title=12 月輕小說新書《皇帝聖印戰記》《尼特族的異世界就職記》等 |url=https://gnn.gamer.com.tw/0/107180.html |accessdate=2019-03-24}}</ref>

==登場人物==
*以台版現行內容為主。<ref>{{cite web |author1=桂かすが |title=設定資料 |url=http://mypage.syosetu.com/mypageblog/view/userid/268339/blogkey/657624/ |accessdate=2019-03-24}}</ref>
===主要人物===
;{{Nihongo|山野·勝|山野マサル}}
:23歲，[[尼特族|尼特族]]（自宅警備員），童顏、身高160公分瘦小，被轉移到異世界的半年內一次娶了4個老婆。在解救精靈國之後，第二順位繼承權的精靈族公主成為第5個老婆。後來建立開墾村，成為領主。

;{{Nihongo|媞莉卡|ティリカ}}
:21歲，在冒險者公會裡的真偽院的3級真偽官，外貌年幼，因後天植入魔眼生長便停止了，身高152公分，後來嫁給山野·勝。

;{{Nihongo|安吉拉|アンジェラ}}
:20歲，神殿巨乳修女，使用水魔法與回復魔法，後來嫁給山野·勝。

;{{Nihongo|伊麗莎白|エリザベス}}
:17歲，B級隊伍曉之戰斧的風魔法師，帝國前貴族之女，後來嫁給山野·勝。

;{{Nihongo|莎蒂|サティ}}
:獸人，奴隸，被山野·勝買下後治好視力不良的問題，並成為山野·勝隊伍裡的冒險者。後來解放奴隸紋，嫁給山野·勝。
