'''尼尔斯·克里斯托弗·杜奈尔'''（{{lang-sv|'''Nils Christoffer Dunér'''}}，{{bd|1839年|5月21日|1914年|11月10日}}），瑞典天文学家。

== 生平 ==
1839年5月21日生于[[马尔默胡斯省|马尔默胡斯省]]比勒贝里亚。其父母是尼尔斯·杜奈尔和彼得罗尼奇（恩雷·施莱特）<ref>{{cite book |title=The Biographical Encyclopedia of Astronomers |last=Hockey |first=Thomas |year=2009 |publisher=[[Springer_Publishing|Springer Publishing]] |isbn=978-0-387-31022-0 |accessdate=August 22, 2012 |url=http://www.springerreference.com/docs/html/chapterdbid/58392.html}}</ref>。

杜奈尔于1862年在[[隆德大学|隆德大学]]获得博士学位，1864年成为该大学天文台观测员，1888年起被[[乌普萨拉大学|乌普萨拉大学]]聘任为天文学教授。

1887年他被授予拉朗德奖<ref>{{cite journal|title=Paris. Academy of Sciences, December 26, 1887|journal=Nature|date=5 January 1888|volume=37|issue=949|pages=239–240|url=https://books.google.com/books?id=-skKAAAAYAAJ&pg=PA240|doi=10.1038/037239a0 }}</ref>，1892年获拉姆福德奖章。[[月球|月球]]上的[[杜奈尔环形山|杜奈尔环形山]]就是以他的名字命名的，此外，[[斯瓦尔巴|斯瓦尔巴]][[瑞典岛|瑞典岛]]上的杜奈尔山、萨宾陆地区的杜奈尔湾以及[[熊岛|熊岛]]西端的杜奈尔岬都是以他的名字命名的<ref>{{cite web |url=http://placenames.npolar.no/stadnamn/Dunérfjellet|title=杜奈尔山（斯瓦尔巴） |publisher=[[挪威极地研究所|挪威极地研究所]] |accessdate=2014年6月4日}}</ref><ref>{{cite web |url=http://placenames.npolar.no/stadnamn/Dunérbukta|title=杜奈尔湾（斯瓦尔巴） |publisher=[[挪威极地研究所|挪威极地研究所]] |accessdate=2014年6月4日}}</ref><ref>{{cite web |url=http://placenames.npolar.no/stadnamn/Kapp_Dunér|title=杜奈尔岬（斯瓦尔巴） |publisher=[[挪威极地研究所|挪威极地研究所]] |accessdate=2014年6月4日}}</ref>。

1914年11月10日在[[斯德哥尔摩|斯德哥尔摩]]去世。
