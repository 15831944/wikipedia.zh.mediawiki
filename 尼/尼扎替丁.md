'''尼扎替丁'''（[[国际非专利药品名称|INN]]：nizatidine）是一種[[组胺|组胺]][[H2受体阻抗剂|H<sub>2</sub>受体阻抗剂]]，主要用於抑制[[胃酸|胃酸]]的分泌，並用于治疗[[消化道溃疡|消化道溃疡]]和[[胃食道逆流|胃食道逆流]]。尼扎替丁由[[礼来公司|礼来公司]]開發，並以商品名稱'''Tazac'''和'''Axid'''推出市場。

== 醫療用途 ==
{{Main article|H2受體阻抗劑}}
尼扎替丁主要用於治疗[[胃溃疡|胃溃疡]]、[[十二指肠溃疡|十二指肠溃疡]]和[[胃食道逆流|胃食道逆流]](GERD)，並用於預防{{tsl|en|Stress_ulcer|压力性溃疡}}。
<ref name="LiverTox">{{Cite web|url=http://livertox.nih.gov/Nizatidine.htm|title=Nizatidine|accessdate=2015-10-11|publisher=Livertox.nih.gov}}</ref>

== 副作用 ==
尼扎替丁的已知[[副作用|副作用]]通常是轻微而罕见的，當中包括[[腹泻|腹泻]]、[[便秘|便秘]]、[[疲倦|疲倦]]、[[昏睡|昏睡]]、[[头痛|头痛]]和[[肌肉痛|肌肉痛]]。

== 历史 ==
尼扎替丁由[[礼来公司|礼来公司]]開發，並於1987年首次上市。尼扎替丁一般被認為與[[雷尼替丁|雷尼替丁]]等效，而兩種藥物的不同之處在於尼扎替丁用[[噻唑|噻唑]]環取代了雷尼替丁中的[[呋喃|呋喃]]環。在2000年9月，[[礼来公司|礼来公司]]宣布他們將向{{tsl|en|Reliant Pharmaceuticals}}出售尼扎替丁的銷售權。<ref name="encyclopedia">[http://www.encyclopedia.com/doc/1G1-65062714.html?Q=%22axid%22] {{Webarchive|url=https://web.archive.org/web/20080526223021/http://www.encyclopedia.com/doc/1G1-65062714.html?Q=%22axid%22|date=May 26, 2008}}</ref>隨後，Reliant Pharmaceuticals在獲得[[美國食品和藥物管理局|美國食品和藥物管理局]]（FDA）的批准後，於2004年推出了尼扎替丁的[[口服|口服]]溶液。<ref name="reliant2004">[http://www.reliantrx.com/investor/press_releases/072604.htm] {{Webarchive|url=https://web.archive.org/web/20131226093839/http://www.reliantrx.com/investor/press_releases/072604.htm|date=December 26, 2013}}</ref> 但是，在一年後，Reliant Pharmaceuticals將尼扎替丁口服溶液（包括其專利<ref>{{Cite web|url=http://patft.uspto.gov/netacgi/nph-Parser?Sect1=PTO1&Sect2=HITOFF&d=PALL&p=1&u=%2Fnetahtml%2FPTO%2Fsrchnum.htm&r=1&f=G&l=50&s1=6930119.PN.&OS=PN/6930119&RS=PN/6930119|title=United States Patent: 6930119|accessdate=2015-10-11|publisher=Patft.uspto.gov}}</ref>）的權利出售給Braintree Laboratories。<ref name="reliant2005">[http://www.reliantrx.com/investor/press_releases/063005.htm] {{Webarchive|url=https://web.archive.org/web/20070814071007/http://www.reliantrx.com/investor/press_releases/063005.htm|date=August 14, 2007}}</ref>

尼扎替丁是在[[氫離子幫浦阻斷劑|氫離子幫浦阻斷劑]]出現之前，所推出的最後一種[[組胺|組胺]][[H2受體拮抗劑|H<sub>2</sub>受體拮抗劑]]。
