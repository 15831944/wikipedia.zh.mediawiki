《'''尼格爾的葉子'''》（{{lang-en|Leaf by Niggle}}）是[[J·R·R·托爾金|J·R·R·托爾金]]創作的[[短篇小說|短篇小說]]，首度發表於1945年1月的《{{le|都柏林評論|Dublin Review (Catholic periodical)}}》上<ref name=":0">{{Cite book|title=魔戒的鍛造者：托爾金傳|last=柯林·杜瑞茲|publisher=黑龍江教育出版社|year=2015年2月|isbn=978-7-5316-7894-6|location=哈爾濱}}</ref>。

== 情節概要 ==
故事主角是善良而又懶惰的[[畫家|畫家]]尼格爾，他多年來一心要完成一幅畫作，從小樹葉一直發展到位在森林之中的一棵樹。尼格爾雖知悉自己來日無多，但因許多瑣事耽擱而一直未能完成該幅畫作。一次他在風雨中病倒後，搭乘[[火車|火車]]開始了旅程，第一站是[[工廠|工廠]]，尼格爾在那裡工作了一個世紀，後來他搭上火車繼續旅程，進入他畫中的世界，與老[[鄰居|鄰居]]帕立斯重逢，兩人一同讓森林變得更加完善。最後尼格爾在嚮導的指引下，進入他嚮往已久的山內<ref name=":1">{{Cite book|title=聖經、魔戒與奇幻宗師|last=柯林·杜瑞茲|publisher=啟示出版|year=2004年6月|isbn=986-7470-03-6|location=臺北}}</ref>。

== 創作背景 ==
托爾金在撰寫《[[魔戒|魔戒]]》的停頓期間創作了《尼格爾的葉子》，這個故事的靈感主要來自於一起事件，他在[[北沼路|諾斯摩爾路]]的鄰居艾格紐老太太曾在聊天時表態希望砍掉路上的一棵[[白楊樹|白楊樹]]，因為擔心[[暴風雨|暴風雨]]時該棵白楊樹會倒在她的屋子上，熱愛大自然的托爾金便勸艾格紐老太太應該將樹保留，後來他因此有了靈感，而逐漸寫出《尼格爾的葉子》。一年之後，他有了繼續撰寫《魔戒》動力<ref name="麥可·懷特">{{Cite book|title=托爾金傳|last=麥可·懷特|publisher=聯經|year=2002年10月|isbn=957-08-2512-X|location=臺北}}</ref>。

托爾金的女兒普麗西拉認為《尼格爾的葉子》有[[自傳|自傳]]色彩，對托爾金「有慰藉、淨化、解放心靈之效」<ref name="麥可·懷特" />。托爾金本人也曾認為他的[[托爾金的傳說故事集|中土大陸神話]]就是他「自己的內心之樹」<ref name=":0" />。

== 分析 ==
學者[[柯林·杜瑞茲|柯林·杜瑞茲]]認為：畫家尼格爾象徵著托爾金本人，樹葉代表《[[哈比人歷險記|哈比人]]》、樹則代表《魔戒》和《[[精靈寶鑽|精靈寶鑽]]》這樣宏大的故事，森林代表[[中土大陸|中土大陸]]，旅程代表[[死亡|死亡]]，工廠代表[[地獄|地獄]]，尼格爾嚮往已久的山代表[[天國|天國]]<ref name=":1" />。
