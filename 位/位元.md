'''位元'''（{{lang-en|Bit}}），亦称'''二進制位'''，指[[二进制|二进制]]中的一位，是資訊的最小单位<ref>{{cite web |language =zh | author = | url =https://market.cloud.edu.tw/content/vocation/data_processing/ks_ct/ch3/3-1.htm| title = 資料儲存單位 =2016-06-01}}</ref>。'''Bit'''是'''Bi'''nary digi'''t'''（二进制数位）的缩写，由数学家John Wilder Tukey提出（可能是1946年提出，但有资料称1943年就提出了）。这个术语第一次被正式使用，是在[[克劳德·艾尔伍德·香农|香农]]著名的论文《通信的数学理论》（A Mathematical Theory of Communication）第1页中。

假设一事件以A或B的方式发生，且A、B发生的概率相等，都为0.5，则一个二进位可用来代表A或B之一。例如：
* '''二进位'''可以用来表示一个简单的正负
* 有两种状态的开关（如电灯开关） 
* [[電晶體|電晶體]]的通断
* 某根导线上[[电压|电压]]的有无
* 一个抽像的逻辑上的是否
* 一个位元有二個數變化如同一個開闗有開和關二種變化，二个位元有四個數變化，三个位元有八個數變化，以下開以1表示關以0表示

可見一個位元有2個數變化，二個位元有4個數變化，三個位元有8個數變化，而當有N個位元有多少變化呢?
可以用2<sup>N</sup>算出，如同2<sup>1</sup>=2（2^1)，2<sup>2</sup>=4(2^2)，2<sup>3</sup>=8(2^3)。
不同进位制下一位的信息量并不总是一个二进位，例如十进制的7转换成二进制后就成了111（7= [ ( '''1''' ) × 2<sup>2</sup>+ [ ( '''1''' ) × 2<sup>1</sup> ]+ [ ( '''1''' ) × 2<sup>0</sup> ]   ] =2^2+2^1+2^0=2*2+2+1=100+10+1<sub>2</sub>=111<sub>2</sub>）。

除二进位外，在電腦上常用的还有八进制、十进制和十六进制等进制的八进位、十进位和十六进位等。
* 1[[位元組|位元組]]（{{lang en|'''Byte'''}}）=8比特（{{lang en|'''bit'''}}）

由於bit位元太小了只有二種變化2<sup>1</sup>=2，而使用以 8 個位元為一個單元的位元組Byte而有2<sup>8</sup>=256變化。

在中文語境中各地稱謂有所不同。對於'''bit'''，中國大陸作比-{}-特，台灣作位-{}-元；而對於'''Byte'''，中國大陸作字-{}-节，台灣作位-{}-元-{}-組。

按照[[热力学|热力学]]与[[信息论|信息论]]的关系，[[熵_(信息论)|信息量]]和[[熵|热力学熵]]可以换算。

一比特相当于：
:<math>{k \ln 2} \mbox{ J/K}</math>

这个换算关系有很深刻的物理含义：向一个存储器存入一比特的信息会使其热力学熵降低
:<math>{k \ln 2} \mbox{ JK}^{-1}</math>

根据[[热力学第二定律|热力学第二定律]]，环境的熵至少要增加这么多，即在T[[开尔文|K]]下操作一比特，至少向环境排放
:<math>{kT \ln 2} \mbox{ J}</math>

的废热。这是信息操作带来的熵变的理论下限，实际当今最先进的电子计算机操作一个比特带来的熵变是该下限的数十亿倍。
