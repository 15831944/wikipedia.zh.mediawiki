== 種類 ==

非洲聯盟護照有以下三種：

; 一般護照
: 此類護照適宜於一般公民能因私人或工作出外旅遊。護照共有32頁，有效期為5年。

; 公務護照
: 此類護照適宜於代表政府機構因履行公務的人員。

; 外交護照
: 此類護照適宜於因公務外出的外交人員和領事，及其隨行受領人。
