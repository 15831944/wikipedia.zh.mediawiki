'''非洲冕豪豬'''（''Hystrix cristata''）是一種[[豪豬|豪豬]]。<ref>{{Citation|last = Woods|first = C. A.|coauthors = Kilpatrick, C. W.|editor-last = Wilson|editor-first = D. E.|editor2-last = Reeder|editor2-first = D. M.|contribution = Hystricognathi|title = Mammal Species of the World: a Taxonomic and Geographic Reference, 3rd ed.|publisher = Johns Hopkins University Press|date = 2005|location = Baltimore|pages = 1538-1600|url = http://www.bucknell.edu/msw3/|isbn = 0801882214}}</ref> 牠們主要分佈在[[義大利|義大利]]、[[西西里島|西西里島]]、[[北非|北非]]及[[撒哈拉以南非洲|撒哈拉以南非洲]]。

== 特徵 ==
[[File:Hystrix_cristata_MHNT.ZOO.2005.0.255_global.jpg|left]]
非洲冕豪豬約長60-90厘米。差不多整個身體都由鬃毛所覆蓋，顏色呈深褐色或黑色。牠們[[頭部|頭部]]、[[頸部|頸部]]及背部的翎毛可以豎起像冠冕一樣。兩側及身體的後背有一些約長35厘米堅硬的翎毛，呈深淺色間，是用來保護自己的。牠們的尾巴較短，末端的毛可以發聲。這些毛的末端較闊，且是空心的，震動時可以發出嘶嘶聲。

非洲冕豪豬的前腳有四趾，趾上有爪，拇趾退化，後腳則有五趾。腳掌沒有毛及有墊，呈蹲行的姿勢。牠們的[[耳朵|耳朵]]及[[眼睛|眼睛]]都很細小，頭上有長的感覺毛。其[[頭顱骨|頭顱骨]]的眶下孔很大，可以讓咀嚼肌伸延至吻的正面。另外，下顎的角突是折曲的，鼻腔很大，有很大地方與咀嚼肌連接。[[鎖骨|鎖骨]]很少，只有1隻[[門齒|門齒]]、1隻[[前臼齒|前臼齒]]及3隻[[臼齒|臼齒]]。

== 食物及消化 ==
非洲冕豪豬主要是[[草食性|草食性]]的，吃樹根、芽、農作物等。牠們有時也會吃[[昆蟲|昆蟲]]、細小的[[脊椎動物|脊椎動物]]及腐屍。為了吸收[[鈣|鈣]]及磨利[[牙齒|牙齒]]，牠們經常會咬骨頭。牠們為了覓食會行走一段很長的距離。牠們的齒冠很高，可以磨爛[[植物|植物]]，並由[[胃部|胃部]][[消化|消化]]，[[纖維|纖維]]則在較大的[[闌尾|闌尾]]及[[大腸|大腸]]被分解。

== 繁殖 ==
有關非洲冕豪豬[[繁殖|繁殖]]的資料都是從圈養的得來。雌豬每年會產一胎，每胎約有1或2隻幼豬。[[妊娠期|妊娠期]]為66日。幼豬出生時重約1公斤，約母豬的3%。出生後一星期幼豬就可以離開巢穴，其刺於此時開始變硬。大約1-2年的時間，幼豬就達至性成熟。

== 行為 ==
非洲冕豪豬是有領域的。牠們很少攀樹，但能夠游泳。牠們是夜間活動的，且是[[一夫一妻制|一夫一妻制]]的。牠們會長時間照顧幼豬，一群通常只有一對父母及幼豬。若受到騷擾，牠們會豎起刺來，讓敵人覺得牠們很大。若繼續受到騷擾，牠們踩到對方的腳上，用刺刺牠。這種攻擊曾殺死[[獅子|獅子]]、[[豹|豹]]、[[鬣狗|鬣狗]]及[[人類|人類]]。

== 分佈地 ==
非洲冕豪豬分佈在[[義大利|義大利]]、[[北非|北非]]及[[撒哈拉以南非洲|撒哈拉以南非洲]]。在[[地中海|地中海]]，牠們主要分佈在義大利大陸、[[西西里島|西西里島]]、[[摩洛哥|摩洛哥]]、[[阿爾及利亞|阿爾及利亞]]、[[突尼西亞|突尼西亞]]、[[利比里亞|利比里亞]]及沿[[埃及|埃及]]海岸。牠們曾在[[海拔|海拔]]2550米的[[安蒂阿特拉斯山|安蒂阿特拉斯山]]出沒。一直以來就相信牠們是由[[羅馬帝國|羅馬帝國]]引入義大利的，但根據[[化石|化石]]及[[亞化石|亞化石]]紀錄，牠們可能於[[更新世|更新世]]末就已經存在於[[歐洲|歐洲]]。
