
'''非我所創'''或'''NIH综合症'''（[[英文|英文]]：'''{{lang|en|Not Invented Here Syndrome}}'''），指的是[[社会|社会]]、[[公司|公司]]和组织中的一种[[文化|文化]]现象，人们不愿意使用、购买或者接受某种产品、研究成果或者知识，不是出于技术或者法律等因素，而只是因为它源自其他地方。

NIH是英文Not Invented Here（非我所創）的缩写，通常带有贬义。在[[国家|国家]]范围内的NIH综合征是[[民族主义|民族主义]]的一种形式，比如中国历史上曾经出现的[[抵制日货|抵制日货]]运动。

[[苹果公司|苹果公司]]的[[Mac_OS|Mac OS]]操作系统曾经墨守自己较早提出而被界面设计界奉为经典的''Apple Human Interface Guidelines''<ref>{{cite web |url=http://developer.apple.com/documentation/UserExperience/index.html |title=存档副本 |accessdate=2009-02-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20090311114620/http://developer.apple.com/documentation/userExperience/index.html |archivedate=2009-03-11 }}</ref>，其他操作系统中出现的许多创新只要违反或者超出这一规范，都不予采纳。

== 在计算机领域 ==

在编程领域，“NIH综合症”通常是指因为认为公司内部的开发总是会比现成的成果更安全、更适合公司、维护成本更低而去[[重造轮子|重造轮子]]（即重复发明已经发明了的东西）的倾向。

== 是我所創 ==
「是我所創」（invented here）是與「非我所創」相反的[[文化|文化]]現象，是描述人們僅因某觀念或知識是由自己國家、組織的人所創造，而對其不信任或低估其價值。這在自信不足的文化或組織中尤為常見。<ref>"... Call it the "invented here" syndrome, the opposite of the "not invented here" syndrome: Gee, it can't be worth much if someone local thought of it first. ..." 引用自[https://web.archive.org/web/20070927122840/http://gatewayva.com/biz/virginiabusiness/magazine/yr1999/april99/rebel.html]</ref>

;示例
:不必看了，那篇論文是[[中文|中文]]的。（說者為使用中文的[[華人|華人]]）

;例二
:我國不應該有[[死刑|死刑]]，不應該[[重罰|重罰]]犯罪，不應該將[[酒駕|酒駕]]比照謀殺，不應該有[[體罰|體罰]]，也不應該有[[髮禁|髮禁]]，因為這些都是我國那些思想落伍的人的主張。

;例三
:日本貨和德國貨就是好，相比之下，國貨沒有一處好的。

;例四
:留美博士能力比土博士強，因為他們好歹出過國、懂外語且學到過國外最新的知識。
