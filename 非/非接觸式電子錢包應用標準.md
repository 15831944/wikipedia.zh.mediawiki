'''非接觸式電子錢包應用標準'''（{{lang-en|Contactless e-Purse Application}}，簡寫：CEPAS）是[[新加坡|新加坡]]的非接觸式[[電子錢包|電子錢包]]及[[智慧卡|智慧卡]]之標準架構。CEPAS在整個新加坡島都有設點且自2009年10月1日起取代原先使用[[FeliCa|FeliCa]]技術的[[易通卡|易通卡]]<ref name="lta_sep">{{cite web |url=http://app.lta.gov.sg/corp_press_content.asp?start=2056 |title=SeP Ready For Implementation |author=Land Transport Authority |date=2009-12-02 |accessdate=2009-04-04 |archiveurl=https://web.archive.org/web/20110527043620/http://app.lta.gov.sg/corp_press_content.asp?start=2056 |archivedate=2011-05-27 |language=en}}</ref>。

== 功能及技術規格 ==
CEPAS提供用於[[免觸碰智慧卡|非接觸式電子錢包]]的程序和資料，允許多用途儲值卡（MPSV）的共同操作。
現行的CEPAS為2.0版本遵照{{link-en|ISO/IEC 7816-4: 2005|ISO/IEC 7816}}規格。主要的指令為：讀取錢包、偵測錢包、信用錢包。其他的功能有自動儲值、部分退款，並可以在最後一筆交易金額借支、和累積借支。在這個版本中，控制電子錢包的管理金鑰是活動式的，而原子性（atomicity，或稱不可分割性）則是用來確保跨介面傳輸可靠性的金鑰。

== 背景 ==
CEPAS是由新加坡資通訊發展管理局（{{lang-en|Infocomm Development Authority of Singapore}}，簡稱：IDA。2016年改組為{{link-en|資通訊媒體發展管理局|Info-communications Media Development Authority}}，簡稱：IMDA）領導的iN2015資訊通信總體規劃的關鍵下一世代電子支付計劃之一<ref name="ida">{{cite web |url=https://www.imda.gov.sg/industry-development/infrastructure/ict-standards-and-frameworks/specification-for-contactless-e-purse-application-cepas |title=Specification for Contactless e-Purse Application (CEPAS) |author=新加坡資訊通訊發展局 |date=2008-01-23 |accessdate=2017-02-03| |language=en}}</ref>。主要目的是希望由1張多用途存儲值卡（Multi Purpose Stored Value Card）在新加坡各種大眾運輸工具（[[新加坡巴士服務|巴士]]、[[新加坡地鐵|新加坡地鐵]]、[[新加坡輕軌列車系統|新加坡輕軌列車系統]]、[[聖淘沙捷運|聖淘沙捷運]]、計程車）、駕車（[[電子道路收費系統_(新加坡)|電子道路收費]]、停車費）之車資、規費、商店小額消費，或支付其他服務款項。

CEPAS是當時資通訊發展管理局、[[陸路交通管理局|新加坡陸路交通管理局]]及新加坡資訊通信發展管理局（ITSC）管轄的卡片和個人身份識別技術委員會（CPITC）主導，{{link-en|星網電子付款私人有限公司|Network for Electronic Transfers}}（NETS），以及易通私人有限公司（EZ-Link）2家私人企業共同研發的標準<ref>{{cite web |url=https://www.tech.gov.sg/Media-Room/Media-Releases/2006/06/Singaporeans-to-Enjoy-the-Convenience-of-One-Card |title=Singaporeans to Enjoy the Convenience of One Card for Electronic Payments|author=新加坡政府技術局 |date=2006-06-27 |language=en}}</ref> 。

CEPAS由{{link-en|新加坡標準生產力暨創新局|SPRING Singapore}}於2006年6月27日在[[新達城|新加坡國際會議展覽中心]]舉行的"CEPAS發布暨下一代電子支付研討會"期間發佈並啟用。<ref name ="ida"/>而符合CEPAS標準的易通卡於2008年12月29日開始發售<ref>{{cite web |url=http://www.ezlink.com.sg/NEWS_ez_press26Dec08.htm |title=Commencement of sale of the NEW CEPAS-Compliant EZ-Link card |author=易通私人有限公司 |accessdate=2009-04-04 |date=2008-12-26 |language=en}} {{Dead link|date=2010-10|bot=H3llBot}}</ref>。

== 現在及未來的發展方向 ==
新加坡現在的小額支付共有2套系統，分別為[[易通卡|易通卡]]及{{link-en|萬捷通卡|NETS FlashPay}}。[[易通卡|易通卡]]以大眾運輸工具資費支付為主，{{link-en|萬捷通卡|NETS FlashPay}}以道路通行（[[電子道路收費系統_(新加坡)|電子道路收費]]，停車費）為主，並拿下大部分小額支付的市場。2套系統卡片和讀卡器目前不能交互識別，所以用戶必須同時持有2個系統的儲值卡來支付費用。

CEPAS的目標是提高[[易通卡|易通卡]]小額支付的交易量並開放{{link-en|萬捷通卡|NETS FlashPay}}支付車資。

== 批評 ==
市民曾經批評，以銀行轉賬（GIRO）為CEPAS規格易通卡提供的自動儲值服務，每次儲值都要收取管理費，而且申請自動儲值的手續比之前麻煩。而原來FeliCa規格的易通卡自動儲值服務是免費的。<ref name="stforum">{{cite web  |url=http://www.straitstimes.com/ST+Forum/Online+Story/STIStory_429724.html |title=Have a card to rival EZ-link |author=Straits Times |date=2009-09-15 |accessdate=2009-10-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20090922225935/http://www.straitstimes.com/ST%2BForum/Online%2BStory/STIStory_429724.html |archivedate=2009-09-22 |language=en }}</ref>
