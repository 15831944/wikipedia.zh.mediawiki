'''非洲钳嘴鹳'''（[[学名|学名]]：''{{lang|la|Anastomus lamelligerus}}''<ref>Gill, F. and Wright, M. (2006) ''Birds of the World: Recommended English Names'', Princeton University Press. {{ISBN|978-0691128276}}</ref>，{{lang-en|African openbill}}）是[[鹳科|鹳科]]钳嘴鹳属的一个现存无危[[物种|物种]]，分布在[[撒哈拉以南非洲|撒哈拉以南非洲]]。<ref>{{cite journal |author1=Dodman, Tim |title=Estimating the size and status of waterbird populations in Africa |journal=Ostrich |date=2007 |volume=78 |issue=2 |page=475-480 |doi=10.2989/OSTRICH.2007.78.2.57.137 |language=en}}</ref>

==外形==
非洲钳嘴鹳体型较大，体长80-94[[厘米|厘米]]，[[体重|体重]]1-1.3[[千克|千克]]。腿部细长呈黑色，眼睛为灰色。全身羽毛为[[黑色|黑色]]，胸部和背部泛绿色、棕色和紫色的光泽。幼鸟羽毛颜色更加暗淡。<ref name=oiseaux-birds>{{cite web|last1= Bouglouan|first1= Nicole|title=African Openbill|url=http://www.oiseaux-birds.com/card-african-openbill.html|website=oiseaux-birds.com|publisher=oiseaux-birds|accessdate=2015-11-01}}</ref>

==栖息地和分布==
非洲钳嘴鹳分布于撒哈拉以南的[[非洲|非洲]]，北部从[[塞内加尔|塞内加尔]]东部至[[埃塞俄比亚|埃塞俄比亚]]，南部至[[南非|南非]]的[[夸祖魯-納塔爾省|夸祖魯-納塔爾省]]。<ref name = IUCN/><ref name = Iziko/>它们栖息于热带湿地，包括水田、浅海滩、河口[[湿地|湿地]]、[[淡水|淡水]]和苦咸水湖泊。<ref name = Iziko/>非洲钳嘴鹳共有2个[[亚种|亚种]]<ref>{{cite web |author1=Internet Bird Collection |title=Anastomus lamelligerus |url=https://www.hbw.com/ibc/species/african-openbill-anastomus-lamelligerus |accessdate=2018-11-23}}</ref>：

==习性==
非洲钳嘴鹳主要以水生[[蜗牛|蜗牛]]和[[淡菜類|淡菜類]]为食。<ref name = Iziko/>它们有时也吃陆生蜗牛、[[无尾目|蛙]]、[[螃蟹|螃蟹]]、小[[鱼|鱼]]、[[蠕虫|蠕虫]]和大型昆虫。在水中，它们用触觉和视觉定位食物，并利用自己尖锐的下颌骨粉碎硬壳猎物并提取它们的肉。<ref name=oiseaux-birds/>

非洲钳嘴鹳不是典型的[[迁徙|迁徙]]鸟类。它们大多居住于固定的场所。只有在干旱季节时，它们才会动身离开干旱地区。<ref name=Iziko>{{cite web|url=http://www.biodiversityexplorer.org/birds/ciconiidae/anastomus_lamelligerus.htm |title=''Anastomus lamelligerus'' (African openbill, Openbilled stork) |accessdate=2016-11-08 |publisher=Iziko Museums of South Africa |work=Biodiversity Explorer |deadurl=yes |archiveurl= https://web.archive.org/web/20111003102252/http://www.biodiversityexplorer.org/birds/ciconiidae/anastomus_lamelligerus.htm |archivedate=2011-10-03 }}</ref>

==繁殖==
由于雨天会有较多的蜗牛，非洲钳嘴鹳通常在[[雨季|雨季]]繁殖。在此期间，异性钳嘴鹳互相靠近，进行复杂的表演，头部来回摇摆，并互相啄对方的头。[[巢|巢]]由树枝和[[芦苇|芦苇]]制作而成，宽度约为50厘米。雌鸟通常会产3-4枚蛋，孵化期约为25-30天。雏鸟在出生50-55天成熟后离巢。<ref name=oiseaux-birds />
