'''非玩家角色'''或稱'''非操控角色'''（{{lang-en|Non-Player Character}}，{{lang|en|NPC}}），是指[[角色扮演遊戲|角色扮演遊戲]]中非玩家控制的角色（例如[[英雄聯盟|英雄聯盟]]中的河蟹跟癒水飛龍）。

NPC是遊戲背景中，非主角（狭义上来说还要求非敌人）的陪襯人物。玩家藉由他們與遊戲互動。在[[桌上角色扮演遊戲|桌上角色扮演遊戲]]裡，NPC是由[[遊戲主持者|遊戲主持者]]操縱，而在電腦角色扮演遊戲裡，則是由程序預設的劇本來決定NPC的反應，由遊戲的人工智能做出控制。部分NPC有時可變成可操控角色。<ref name="概论">{{cite book|author1=孙祺舜|title=《电子游戏概论》|publisher=高等教育出版社|isbn=9787040260748|pages=126~129}}</ref><ref name="设计">{{cite book|author1=胡昭民、吴灿铭|title=《游戏设计概论(第四版)》|publisher= 博硕|isbn=9789864340781|pages=233}}</ref>

== 類型 ==
NPC通常扮演以下幾種角色：
# 玩家角色的協助者：以各種方式協助玩家冒險的角色，例如商人或牧師。<ref name="概论" /><ref name="设计" />
# 玩家角色的敵對者：遊戲中玩家的[[敵人|敵人]]，打敗他們可得到[[經驗值|經驗值]]和寶物。可能是[[嘍囉|嘍囉]]或[[頭目_(角色類型)|頭目]]。嘍囉一般容易對付，頭目則較為強大。頭目通常是玩家必須擊敗的對象，打敗最終頭目通常代表遊戲結束。<ref name="概论" /><ref name="设计" />然而，这种人物通常不会被视作单纯的 NPC。
# 玩家隊伍的夥伴：在戰鬥中幫助玩家的輔助性角色，但不受玩家控制，或只能做大略的命令。<ref name="概论" />
# 玩家角色暴行的受害者（不論陣營），暴行後的反應則視游戏剧本和人工智能而定，部份遊戲會對玩家角色還擊或是就此死亡。
# 群眾角色：沒有具體背景、台詞的角色。通常是一個群體的方式參與遊戲情節的推動。<ref name="概论" />
# 引導玩家角色發生事件任務的關鍵角色：又稱情節觸發類角色，可能是推動遊戲情節的必要關鍵，或是觸發支線劇情或隱藏劇情的角色。<ref name="概论" />
# 提供事件背景的資訊：又稱為情報角色，能提供玩家關於遊戲的訊息。<ref name="概论" />
# 提供玩家使玩家獲得獎勵：又稱回饋類角色，會根據玩家在遊戲中的任務表現結果而給予獎勵。<ref name="概论" /><ref name="设计" />
这些角色并不是[[互斥|互斥]]的——一个 NPC 可能既是路人、又是某个支线任务的目标、还可能在玩家完成了其它任务后加入队伍成为伙伴，端视游戏剧本而定。
