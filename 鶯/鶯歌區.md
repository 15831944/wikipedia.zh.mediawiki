'''鶯歌區'''是[[臺灣|臺灣]][[新北市|新北市]]下的一個[[市轄區|市轄區]]，因為境內盛產窯土，因此[[陶瓷|陶瓷]]製造工業發達。

== 地名由來 ==
[[File:Yingge_2_070513.jpg|thumb]]
鶯歌北面山脈斜坡有一顆如鸚巨石，被稱為「鸚哥石」，「鷹哥石」、「鶯哥」、「鶯歌」。早年傳說此奇石常吐霧食人，而當[[鄭成功|鄭成功]]領軍經過此地時<ref>事實上，鄭成功於攻取台灣的同年病逝於今之[[台南|台南]]。因此他並沒有北上至此地。</ref>，被瘴氣所阻擾，士兵被食，於是砲打巨石，使其瘴氣散，而得以進軍，此為鶯歌人家喻戶曉的傳說，而鶯歌地名也由山上奇石而來。清代的《淡水廳志》亦載有類似的傳說：「鶯歌石在三角湧，與[[鳶山|鳶山]]對峙。相傳吐霧成瘴，偽鄭進軍迷路，斷其頸。」，然因鄭成功事實上未曾親至北台灣地區，故此傳說不可信。<ref>{{cite news|title=鄭成功沒來過台北？那「鶯歌」、「劍潭」的傳說都假的囉|url=http://www.setn.com/News.aspx?NewsID=28490|publisher=三立新聞|date=2014-06-27|accessdate=2015-09-19}}</ref>

== 歷史 ==
相傳在1684年時，鶯歌區當地已有[[客家|客家]]籍移民入墾，植[[茶|茶]]為農。[[台灣日治時期|日治時期]]編纂的《桃園廳志》則記載：相傳1685年[[泉州|泉州]]人陳瑜從南部招佃，入墾其南方的南靖厝庄，後因閩粵械鬥離去。其後，又有[[許丁|許丁]]等人入墾植茶。

日佔時期1920年設置'''鶯歌-{庄}-'''，隸屬[[臺北州|台北州]][[海山郡|海山郡]]，1940年升格為'''[[鶯歌街|鶯歌街]]'''。鶯歌街（或鶯歌-{庄}-）雖以鶯歌為名，但街役場（或-{庄}-役場）設於彭福[[大字_(行政區劃)|大字]]下的樹林[[小字_(行政區劃)|小字]]。戰後，1946年改制隸屬於台北縣[[海山郡|海山區]]'''鶯歌鎮'''（當時鎮公所設於樹林），同年8月1日，鶯歌鎮與樹林鎮分治。1947年撤銷海山區由台北縣直轄，2010年[[2010年中華民國縣市改制直轄市|五都升格]]，隨著台北縣改制為新北市而改稱'''鶯歌區'''。新北市改制時，各區區長多由原任鄉鎮市長續任，然而鶯歌因為末代鎮長[[蘇有仁|蘇有仁]]當選市議員而成為例外。

過去，鶯歌因境內盛產窯土而擁有發達的[[陶藝|陶瓷]]工業。

== 地理 ==
鶯歌區位於[[新北市|新北市]]西端，東與[[樹林區|樹林區]]、[[三峽區|三峽區]]相接，西與[[桃園市|桃園市]][[八德區|八德區]]、[[桃園區|桃園區]]、[[大溪區|大溪區]]相鄰，北邊為[[桃園市|桃園市]][[龜山區|龜山區]]。地形上東北邊為山區，南邊為大漢溪河埔地，市區地勢較為高低不平。

== 文化特色 ==
鶯歌區最著名的特產即是陶瓷工藝。相傳[[清朝|清朝]][[嘉慶|嘉慶]]年間[[吳岸|吳岸]]、[[吳糖|吳糖]]、[[吳曾|吳曾]]等人來此，發現鄰近的尖山地區盛產[[黏土|黏土]]，遂於此設[[窯|窯]]製陶。清代與日本統治時代當地居民所從事的行業以農業為主、陶瓷業為輔。[[臺灣戰後時期|二戰後]]，陶瓷業漸能與農業分庭抗禮；近一、二十年間陶瓷業已取代農業成為當地標誌，以盛產各類陶器著名。

約兩百餘年前，當時鶯歌為全台灣最大的陶瓷藝品輸出地，如今，鶯歌的陶瓷藝品更是聞名全球，該鎮更有「台灣的[[景德鎮|景德鎮]]」之稱。現今鎮內的陶瓷相關產業工廠及店家已超過千餘家、大小商店近2,300餘家，其中工廠有80%以上為陶瓷製造業，所生產的產品有屬於建築用的琉璃瓦、地磚、壁磚、衛浴設備等，亦有裝飾或日常用品的藝術花瓶、仿古花瓶，茶壺、家庭器皿、兒童玩具、裝飾品，以及工業用的精密陶瓷等；陶瓷老街也是台北縣騎警隊勤務地點之一（其他地點分別為[[淡水漁人碼頭|淡水漁人碼頭]]、[[八里左岸公園|八里左岸公園]]和板橋[[新板橋車站特定專用區|新板特區]]）<ref>《跟著偶像去旅行》，[[東森綜合台|東森綜合台]]，2010年9月3日播放</ref>。鶯歌並設有[[新北市立鶯歌陶瓷博物館|新北市立鶯歌陶瓷博物館]]及國內首座以陶瓷為主題的陶瓷公園，另外[[新北市立美術館|新北市立美術館]]亦確定設於鶯歌三鶯新生地。

=== 主要道路 ===
*中山路：從鶯歌中山高架橋起算，向西經由鶯桃路往桃園龜山方向，向東經由中正一路往樹林
*[[中正路_(鶯歌區)|中正一路、二路、三路]]：起於樹林焚化爐，向西來到鶯歌阿鴻當家社區，穿越火車鐵橋，與八德路（可前往八德、國防大學）、尖山路交會，經過鶯歌高職，中正三路的終點，便是大溪區，沿著大鶯路可往大溪。
*[[鶯桃路|鶯桃路]]：是鶯歌地區往來桃園區、八德區、龜山區重要的道路，也是鶯歌人口密集的道路之一。從鶯歌中山高架橋算起，沿著鶯桃路往西北方向，圖中經過德昌路、鳳鳴國中（分別是三鶯捷運C3與C4站），最後可往桃園火車站，鶯桃路終點即為桃鶯路的起點。
*[[建國路_(鶯歌區)|建國路]]：為鶯歌火車站周邊最主要、最繁華的道路，約從阿鴻當家算起，經過鶯歌火車站後分別與中正一路、國慶街、育英街交會，經過全聯福利中心、阿婆壽司、康是美、屈臣氏、陶瓷老街、鶯歌國小、鶯歌國中、鶯歌消防分隊。
*光明路、永明街：為平行於建國路的一條清新宜人、適宜行走道路。光明路：東起國慶街，向西經過建國國小、台北國寶社區，終止於天生贏家社區，終點即為永明街起點。永明街：起於國華路與光明街交會，也就是鶯歌興龍黃昏市場（捷運三鶯線C2-國華路站預定地），向西經過金富貴社區、歡喜就好社區、名宸大地社區、東街日式料理（原東金火鍋）店、昌福國小，終止於與鶯桃路的交會。

== 更多資訊 ==
*台語歌手[[黃克林|黃克林]]以前有首台語老歌叫做「倒退嚕」，歌詞中的第一段「耶~拜請 拜請拜請 東海岸西海岸 北投西帽山 鶯歌出土炭 草山底勒出溫泉」歌詞中有提到鶯歌出產「土炭」。
*福氣爚鶯歌-鶯歌地區土地公廟訪查實錄：為文史工作者[[呂江銘|呂江銘]]先生的著作，2016年12月出版，書籍內容詳細記載新北市鶯歌區76座土地公廟珍貴史料，為新北市鄉土教學教材提供更完整記錄鶯歌土地公廟之資料。
