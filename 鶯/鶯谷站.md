'''鶯谷站'''（{{jpn|j=鶯谷駅|hg=うぐいすだにえき|rm=Uguisudani eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[台東區|台東區]][[根岸_(台東區)|根岸]]一丁目，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）的[[鐵路車站|鐵路車站]]<ref name="zeneki"/>。京濱東北線的[[車站編號|車站編號]]是'''JK 31'''，山手線是'''JY 06'''。

== 停靠路線 ==
停靠的路線在正式軌道名稱上只有[[東北本線|東北本線]]1條路線（詳詳參見路線條目與{{link-ja|鐵道路線的名稱|鉄道路線の名称}}），而在列車系統上則有[[京濱東北線|京濱東北線]]列車與[[山手線|山手線]]列車2個系統，旅客資訊不使用「東北（本）線」。另外依據[[特定都區市內|特定都區市內]]，本站屬於「[[特定都區市內#設定區域一覧|東京都區內]]」與「[[東京山手線內|東京山手線內]]」。

== 历史 ==
* 1912年（[[明治|明治]]45年）7月11日：[[鐵道省|鐵道院]]東北本線車站開業，為明治時期最後一個開業的車站。
* 1987年（[[昭和|昭和]]62年）4月1日：[[國鐵分割民營化|國鐵分割民營化]]後由JR東日本繼承
* 1988年（昭和63年）3月13日：京濱東北線開始運行快速班次，午間京濱東北線不停靠本站。

* 2001年（[[平成|平成]]13年）11月18日：[[IC卡|IC卡]]「[[Suica|Suica]]」啟用。

== 車站構造 ==

本站是上野站的業務委託站，為[[島式月台|島式月台]]2面4線構造，出口有北口與南口。北口為地上站，南口為1927年建造的[[跨站式站房|跨站式站房]]，南口前是連結上野公園與北側市區的天橋<ref name="zeneki"/>。

由於沒有[[綠色窗口|綠色窗口]]，可在南口的[[自動售票機|指定席券販賣機]]購買指定席券。

由於站名「鶯谷」，早晨月台喇叭會撥放鶯叫聲。

== 車站周邊 ==
車站西側是[[上野恩賜公園|上野恩賜公園]]（上野山）北側，並且是高級住宅區[[上野櫻木|上野櫻木]]與[[東京藝術大學|東京藝術大學]]的最近車站。鄰近[[寛永寺|寛永寺]]、[[東京國立博物館|東京國立博物館]]、[[國際兒童圖書館|國際兒童圖書館]]、{{link-ja|大名時計博物館|大名時計博物館}}、{{link-ja|東京文化財研究所|東京文化財研究所}}。每年七夕前後，在{{link-ja|下谷七福神|下谷七福神}}與車站周邊[[商店街|商店街]]的入谷朝顏市帶動下，本站人潮眾多。

東側為旅館街，可見[[屋台|屋台]]與[[居酒屋|居酒屋]]。街道狹小。

在[[警視廳|警視廳]]的犯罪情報地圖中，本站的治安比山手線其他各站較差。警察針對上野站入谷口～鶯谷站～入谷站一帶加強巡邏與監視。鶯谷站內禁止街頭拉客等活動。

近年周邊陸續進行改建成新建公寓。有許多東京藝大與專門學校的學生、家庭入住，北口鄰近高級住宅區上野櫻木。
