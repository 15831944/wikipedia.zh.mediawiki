'''葵芳邨'''（{{lang-en|Kwai Fong Estate}}）是[[香港|香港]]的一個[[香港公共屋邨|公共屋邨]]，於1971年落成，為[[葵青區|葵青區]]內第3個[[政府廉租屋|政府廉租屋邨]]（不計現劃入[[荃灣區|荃灣區]]之[[梨木樹邨|梨木樹邨]]），並是全港首個被揭發建築用料低於標準的[[公共屋邨|公共屋邨]]（[[26座問題公屋醜聞|26座問題公屋醜聞]]）。現已經全部重建完成，並由雅居物業管理有限公司負責屋邨管理，目前住戶數目為6,200戶。<ref>{{cite web |author1=香港房屋委員會及房屋署 |title=葵芳邨 |url=https://www.housingauthority.gov.hk/tc/global-elements/estate-locator/detail.html?propertyType=1&id=2828 |website=https://www.housingauthority.gov.hk/tc/global-elements/estate-locator/detail.html?propertyType=1&id=2828 |publisher=香港房屋委員會及房屋署 |accessdate=2018-10-23}}</ref>

== 地理位置 ==
該屋邨位於[[香港|香港]][[新界|新界]][[下葵涌|下葵涌]]，全邨建築於[[醉酒灣|醉酒灣]]填海地上，故此全為平地。以街道標示則為葵富道以北、葵益道（前稱葵安道）以南、[[葵涌道|葵涌道]]以西、興芳路（前稱葵芳邨通道）以東。

本邨土地前身曾規劃為地鐵[[荃灣線|荃灣線]]車廠及附設的垃圾灣站（今[[葵芳站|葵芳站]]），但後來政府更改地鐵規劃，將車廠位置兩度更改，原來預留建廠土地成為葵芳邨。

== 屋邨資料 ==
=== 樓宇 ===
重建前的葵芳邨[[政府廉租屋邨|政府廉租屋邨]]，首批葵芳邨的樓宇在1971年至1973年建成，共有11座舊長型樓宇，建築費用約為港幣4,000萬元。早在1980年3月，葵芳邨第5座及第6座多處天花板及外牆出現嚴重[[混凝土|混凝土]]剝落，房署起初對外稱樓宇結構「無問題」<ref>樓高二十層 居民四千餘 葵芳第五座出現險情 天花板洞穿外牆剝落 居民促徹底調查以保生命安全 《大公報》 1980年3月30日 第4頁</ref><ref>落成七年之葵芳邨公屋 六個單位天花板 水坭脫落露鋼筋  房屋署進行修補 據說結構無問題 《工商日報》 1980年3月30日 第8頁</ref><ref>葵芳邨經不起時間考驗 樓宇出現裂痕 第五座外牆天花露出石屎鋼筋 居民要求當局將大廈列為危樓 房屋署稱並無危險已進行修補 《華僑日報》 1980年3月30日 第4頁</ref>。由於受影響居民擔心所住樓宇結構安全，房署先於第6座進行全面修葺工程，並於1982年6月前把該座全部居民調遷至其他屋邨<ref>葵芳邨一大廈進行維修工程 千戶人獲安排他遷 搬遷補償迄未談妥 房署以每戶計算 居民則以每人為標準 《大公報》 1982年7月11日 第4頁</ref>，第6座維修工程於1985年初完成，並於4月起入伙。

房署原本計劃是將第5、6及10座進行全面修葺工程，但因為維修第6座的費用嚴重超支，於是於1985年1月13日宣佈決定將第5座改為清拆重建<ref>葵芳邨居民昨開會 對重建搬遷表憂慮 要求公佈修葺費及保證維修樓宇安全 決定再到房署及兩局議員辦事處請願 《大公報》 1985年1月14日 第5頁</ref><ref>房署表示訂有計劃維修舊邨樓宇 葵芳邨屋結構安全 續有兩座須大修葺 居民可選擇搬遷津貼增搬電話費 《華僑日報》 1985年1月14日 第3頁</ref><ref>葵芳第六座維修費超額 第五座政府已決定重建 《華僑日報》 1985年1月16日 第7頁</ref>（現今葵仁樓），結果第5座便成為首幢被拆卸的政府廉租屋，大部分居民獲調遷至[[青衣島|青衣]][[長康邨|長康邨]]（1984年入伙的2座[[工字型大廈|H型大廈]]）。至同月23日，房署又宣佈將第8座，和相連的第9、10及11座清拆重建<ref>廖本懷在立法局表示 葵芳邨樓宇經檢驗 證實全部結構安全 只有四幢樓要拆卸重建 《大公報》 1985年1月24日 第5頁</ref><ref>葵芳邨廈維修費昂 四座改採拆卸重建 房屋司強調全邨均安全毋須擔心 日內將檢討特惠搬遷津貼 《華僑日報》 1985年1月24日 第25頁</ref>，惟推遲至葵仁樓（原第5座）及葵智樓於1987年拿匙入伙後，才宣告重新發展。

葵芳邨第5和第6座之問題經驗證後，發覺由於在建屋施工時偷工減料，令[[混凝土|混凝土]]強度大幅低於標準，以致房屋結構受損，事件亦引發了「[[26座問題公屋|26座問題公屋]]」和「[[鹹水樓|鹹水樓]]」的公屋醜聞。

由於要盡量做到原邨安置，建好一期入伙後才能拆下一期，所以重建葵芳邨時間長達20年（1982-2002），所以經分期重建後，葵芳邨擁有12幢共7款不同時期的標準設計大廈，故未能將全邨設計統一成一個整體，而每期所採用的樓宇設計、外牆顏色及設計風格則各有不同，計有[[新長型|新長型]]、[[相連長型|相連長型]]L1及L3型、[[和諧式大廈|和諧式]]的和諧1及3型、單方向設計大廈及[[新十字型大廈|新十字型]]樓宇。但全邨由一個屋邨辦事處統一管理，此辦事處已於1998年從第6座2樓搬到葵泰樓地下。

==選區劃分==
葵芳邨在立法會選舉上屬於新界西區域，而地方行政則為葵青區。全邨所有樓宇組成葵青區「葵芳選區」。該選區包括附近的[[大連排|大連排]]工業區，惟該區並無居民居住，故此選區的絕大部份居民均為葵芳邨居民。現任民選議員為[[街坊工友服務處|街坊工友服務處]]執行監督兼[[立法會|立法會]]議員[[梁耀忠|梁耀忠]]。

== 文娛康樂設施 ==
葵芳邨設計頗為特別，由[[港鐵|港鐵]][[荃灣綫|荃灣綫]]的鐵路高架橋，穿越葵芳邨社區的中軸，但該鐵路橋設有混凝土有蓋隔音屏障，以減低列車嘈音對葵芳邨居民的影響，及避免因為有人高空擲物擊中電纜導致列車服務癱瘓，而鐵路橋下及兩旁均沒有建置居住樓宇，所以區內有一個頗大的羅馬廣場，平日有些文娛表演。2005年底於香港舉行[[第六次世貿部長級會議|第六次世貿部長級會議]]期間，來自東南亞的[[反世貿|反世貿]]人士舉辦的「[[稻米節|稻米節]]」就曾在葵芳邨羅馬廣場表演。而橋下亦佈有休憩桌椅，平日有不少長者在桌椅休息及下棋。

另外邨內有設長者及青少年服務中心。提供長者服務的基督教光愛中心位於葵明樓地下；提供青少年服務的則有[[白普理|白普理]]中心及[[香港青年協會|香港青年協會]]葵芳青年空間。

但較特別的是，葵芳的[[社區會堂|社區會堂]]並非位於葵芳邨內，而是位於葵芳邨對面的智芳街、高芳街舊式私人住宅區中。

==教育設施==
葵芳邨內設有一所幼稚園暨幼兒園。同時邨內設有一所小學。本來'''葵芳邨'''在重建前設有4所小學及一所特殊學校，包括[[惠州公學|惠州公學]]、[[聖公會思晃小學|聖公會思晃小學]]、[[閩僑第二小學|閩僑第二小學]]、[[丁熊照學校|丁熊照學校]]及[[佛教林金殿紀念學校|佛教林金殿紀念學校]]。在當時教育統籌局1邨1校政策下，現時僅剩下一所林金殿紀念小學。聖公會思晃小學則遷往[[青衣島|青衣]]，易名為[[聖公會何澤芸小學|聖公會何澤芸小學]]；而[[閩僑第二小學|閩僑第二小學]]亦於1988年遷往[[青衣島|青衣]]，但已在2009年被[[地利亞(閩僑)英文小學|地利亞(閩僑)英文小學]]所取代。

==商業設施==
葵芳邨內有兩個由[[香港房屋委員會|香港房屋委員會]]興建的公營屋邨商場，邨內各式的商業設施齊備；另外亦設有一個乾濕貨市場。

屋邨同時鄰近[[葵青區|葵青區]]內的消費中心點——[[新都會廣場|新都會廣場]]及[[葵涌廣場|葵涌廣場]]。
