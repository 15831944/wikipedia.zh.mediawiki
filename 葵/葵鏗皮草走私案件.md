中國廣東省揭陽市惠來縣'''葵鏗皮草走私案件'''（1301葵鏗皮草走私案件）是汕頭市海關偵辦的走私案件，涉案皮草總值76億元人民幣，逃稅20億元人民幣<ref>[http://legal.people.com.cn/n/2013/1107/c188502-23468751.html 汕头海关破获偷逃税17亿走私皮草大案]</ref>。

==案件经过 ==
2013年初，汕頭海關根據海關總署緝私局有關線索，經過長達半年的情報經營，掌握[[其仕集團|其仕集團]]旗下揭陽市惠來縣葵鏗公司存在走私皮草的重大嫌疑。現已查明，葵鏗公司負責人林金招在境外拍賣購得水貂、狐狸等動物生皮後，利用葵鏗公司加工貿易手冊，將本應以[[一般貿易|一般貿易]]方式納稅進口的上述生皮，偽報成來料[[加工保稅|加工保稅]]貨物進口，生皮進口後直接在國內銷售或鞣制加工成熟皮後內銷，從而達到逃避海關稅收的目的。查扣涉嫌走私的皮草100多万张<ref>[http://news.sohu.com/20131108/n389782750.shtml 汕头查获重大走私皮草案 涉嫌偷逃税额达17亿元]</ref>。僅2013年1月至7月間，林氏的葵鏗集團就利用來料加工手冊，以來料加工保稅原料的名義走私動物皮草生皮225萬張，涉嫌偷逃稅款5.3億元人民幣<ref>[http://tv.cntv.cn/video/VSET100232480132/f2e0da13f49c4c8081694763373172be 《经济与法》 20161122 2016“国门利剑”在行动——狐狸皮下隐藏的秘密]</ref>

==判決==
2015年12月28日廣東省檢察院收到汕頭市中級人民法院的刑事的判決中，涉案的其仕集團旗下惠來縣葵鏗皮草二廠、葵鏗皮革皮草有限公司（香港）被列為被告單位因犯走私普通貨物罪，分別罰金7.5億元人民幣、3.5億元人民幣，其仕集團集團主席林金招等人被判有期徒刑。<ref>[https://hk.news.appledaily.com/china/realtime/article/20131107/51880494 廣東破76億元皮草走私大案 以香港為中轉站]</ref><ref>[http://www.0518meet.com/t20160111_1742415.shtml.htm “1301走私皮草案”获成功判决]</ref><ref>[http://www.shenyang.gov.cn/zwgk/system/2006/10/27/000004313.shtml 市领导会见其仕（美国）皮草有限公司董事长林金招]</ref><ref>{{Cite web |url=http://m.21jingji.com/article/20161123/herald/6c0482e58230795d99b87d5d26c9ce7f_baidunews.html |title=央视曝光惊天大案！一个工厂设了一个局，竟然偷税20亿 |access-date=2018-06-16 |archive-url=https://web.archive.org/web/20180616163357/http://m.21jingji.com/article/20161123/herald/6c0482e58230795d99b87d5d26c9ce7f_baidunews.html |archive-date=2018-06-16 |dead-url=yes }}</ref><ref>{{Cite web |url=http://m.21jingji.com/article/20161123/herald/6c0482e58230795d99b87d5d26c9ce7f_baidunews.html |title=央视曝光惊天大案！一个工厂设了一个局，竟然偷税20亿 |access-date=2018-06-16 |archive-url=https://web.archive.org/web/20180616163357/http://m.21jingji.com/article/20161123/herald/6c0482e58230795d99b87d5d26c9ce7f_baidunews.html |archive-date=2018-06-16 |dead-url=yes }}</ref><ref>[http://orientaldaily.on.cc/cnt/china_world/20131108/00178_028.html 汕頭檢76億走私皮草]</ref><ref>[http://gd.sina.com.cn/news/m/2013-11-07/135752609.html 广东汕头查扣走私皮草100万张 案值76亿]</ref><ref>[http://gd.qq.com/a/20131107/013078.htm 广东查扣走私皮草100万张 案值76亿元(图)]</ref><ref>[http://www.gd.jcy.gov.cn/jcyw/gs/dxal3/201601/t20160111_1742415.shtml “1301走私皮草案”获成功判决]</ref>
