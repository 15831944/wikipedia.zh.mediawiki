'''祖宾·梅塔'''（{{lang-en|'''Zubin Mehta'''}}，{{bd|1936年|4月29日|catIdx=Mehta}}），[[印度|印度]]籍[[帕西人|帕西裔]][[指挥家|指挥家]]。

== 职业生涯 ==
祖宾·梅塔1936年出生于印度[[孟买|孟买]]的一个音乐世家，父亲[[梅利·梅塔|梅利·梅塔]]是一位出色的小提琴手、小提琴教师和指挥家、孟买交响乐团和弦乐四重奏团的创始人。祖宾·梅塔7岁时就受到了正规的小提琴和钢琴教育，所以他很小的时候已经对欧洲音乐非常熟悉。作为父亲的学生，他16岁的时候就指挥了孟买交响乐团。可是他遵循父母的意愿，去学习医学。但上了两个学期的课之后，他就转投到音乐的怀抱中了。18岁的时候他去了[[维也纳音乐学院|维也纳音乐学院]]学习钢琴，作曲与[[低音提琴|低音提琴]]。在[[汉斯·斯瓦洛夫斯基|汉斯·斯瓦洛夫斯基]]门下完成了指挥的学业。

1958年他[[利物浦|利物浦]]国际指挥大赛上胜出，并在当地担任一乐团的副指挥一职。20多岁的梅塔就已经指挥过[[维也纳爱乐乐团|维也纳爱乐乐团]]和[[柏林爱乐乐团|柏林爱乐乐团]]。而且时至今日，他们还保持着紧密的合作关系。在美国坦格伍德指挥大赛上他获得了第二名（第一名是[[阿巴多|阿巴多]]）。这时候他认识了[[波士顿交响乐团|波士顿交响乐团]]的指挥[[查尔斯·孟许|查尔斯·孟许]]，后者对梅塔影响深远。1960年他第一次和[[纽约爱乐|纽约爱乐]]、[[费城乐团|费城乐团]]和[[蒙特利尔交响乐团|蒙特利尔交响乐团]]合作，1962到1966年他还任[[蒙特利尔交响乐团|蒙特利尔交响乐团]]的首席指挥。

祖宾·梅塔在1962年到1978年任[[洛杉矶爱乐乐团|洛杉矶爱乐乐团]]音乐总监。1969年他还成为了[[以色列爱乐乐团|以色列爱乐乐团]]的音乐顾问，1977年升为首席指挥，1981年任终身音乐总监。1978年他成为美国五大交响乐团之一[[纽约爱乐乐团|纽约爱乐乐团]]的音乐总监，历时13年，直到继任者[[库特·马苏尔|库特·马苏尔]]的到来。

歌剧方面，他在[[蒙特利尔|蒙特利尔]]、[[大都会歌剧院|大都会歌剧院]]、[[维也纳国立歌剧院|维也纳国立歌剧院]]、[[巴伐利亚国家歌剧院|巴伐利亚国家歌剧院]]、[[伦敦皇家歌剧院|伦敦皇家歌剧院]]、[[柯文特花园|柯文特花园]]指挥过歌剧演出。

1998年起，梅塔成为巴伐利亚国家歌剧院音乐总监。

== 奖项 ==
祖宾·梅塔是[[佛罗伦萨|佛罗伦萨]]的荣誉市民，1997年成为[[维也纳国家歌剧院|维也纳国家歌剧院]]荣誉会员。

2004年初他成为慕尼黑爱乐乐团荣誉会员。

2006年荣获用以表彰终身为美国文化作出贡献的杰出人士的肯尼迪中心荣誉奖（Kennedy Center Honors,也称为肯尼迪中心终身成就奖）。

== 其他 ==
1990年，1995年，1998年與2007年指挥维也纳愛樂樂團的新年音乐会，并已接到2015年维也纳新年音乐会的指挥邀请<ref>[http://wien.orf.at/news/stories/2623182/ Zubin Mehta dirigiert Neujahrskonzert 2015]</ref>。

梅塔能以[[低音提琴|低音提琴]]演奏者的身份登台，并曾与[[帕尔曼|帕尔曼]]、[[祖克曼|祖克曼]]、[[杰奎琳·杜·普蕾|杜普雷]]与[[丹尼爾·巴倫波因|巴伦博伊姆]]四人组成五重奏，演奏[[舒伯特|舒伯特]]的名作《鳟鱼五重奏》。这一历史性的音乐盛事，包括演奏前的准备花絮，被灌制成DVD纪录片发售。

2014年3月1日成为好莱坞星光大道第2434颗巨星<ref>[http://news.xinhuanet.com/2011-03/02/c_13757659.htm 新华网报道：世界指挥大师祖宾·梅塔添星好莱坞星光大道]</ref>。

2015年1月1日，第五次指揮[[維也納新年音樂會|維也納新年音樂會]]。
