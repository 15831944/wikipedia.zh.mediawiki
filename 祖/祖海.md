'''祖海'''{{BD|1976年|5月10日|||catIdx= Z祖}}出生于[[安徽省|安徽省]][[蚌埠市|蚌埠市]]，毕业于[[中国音乐学院|中国音乐学院]]，为中国大陆女歌手，被誉为“新民歌天后”。<ref>{{cite web |url=http://news.xinhuanet.com/ent/2013-08/16/c_125182090.htm |title=华语乐坛至尊榜如火如荼 歌唱家祖海强势七连冠  |accessdate=2013 |publisher=新华网 |work= |language=zh-hans}}</ref>

== 生平 ==
1976年，祖海出生在安徽省蚌埠市，父亲是电工，家人喜好[[音乐|音乐]]和[[戏曲|戏曲]]。祖海早年在蚌埠市第四高级中学读书，后考入[[中国音乐学院|中国音乐学院]]，师从[[金铁霖|金铁霖]]。祖海出道之后，2003年加入[[中国人民解放军总政歌舞团|中国人民解放军总政歌舞团]]<ref>{{cite web |url=http://www.people.com.cn/GB/wenhua/1084/2408143.html |title= 总政歌舞团特招祖海 彭丽媛有了“接班人” 近日，总政歌舞团将排名“四小民旦”之首的女歌手祖海特招入伍|accessdate=2003 |publisher=人民网 |work= |language=zh-hans}}</ref>。2008年[[退伍|轉業]]至[[中國音樂學院|中國音樂學院]]<ref>{{cite web |url=http://ent.ifeng.com/music/special/zuhaiyinlehui/content-4/detail_2010_03/16/394190_0.shtml |title=祖海艺术简历|accessdate=2016-06-27 |publisher=凤凰网 |work= |language=zh-hans}}</ref>。
