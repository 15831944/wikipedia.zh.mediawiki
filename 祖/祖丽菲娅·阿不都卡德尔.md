'''祖丽菲娅·阿不都卡德尔'''（{{bd|1960年|5月|}}）女，[[新疆|新疆]][[伊宁|伊宁]]人，[[维吾尔族|维吾尔族]]，[[中华人民共和国|中华人民共和国]]政府官员、[[全国人民代表大会|全国人民代表大会]]新疆地区代表。<ref>{{cite web|author=|title=第十一届全国人大代表|url=http://news.xinhuanet.com/ziliao/2008-03/03/content_7708230.htm?node=36628|publisher=新华网|date=2008-03-03}}</ref>

==生平==
毕业于[[新疆农业大学|新疆八一农学院]]机械化专业。后取得中共新疆维吾尔自治区委党校研究生学历。[[中共|中共]]党员。<ref>{{cite news|title=全国人大代表信息-祖丽菲娅·阿不都卡德尔|url=http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=112648|accessdate=2013-04-05|newspaper=全国人大网}}</ref><ref name=jl>{{cite news |language =  |author = |coauthors =  |url = http://news.163.com/15/0725/14/AVCHPNCU00014JB5.html |title =祖丽菲娅·阿不都卡德尔(女)任新疆自治区交通厅厅长  |publisher = 网易 |pages =  |date = 2015-07-25 |accessdate = }}</ref>1984年8月起，历任新疆维吾尔自治区道路运输管理局科技处、车管处、企管科、安检科干部，安监科副科长。2002年3月，任新疆维吾尔自治区道路运输管理局总经济师。<ref name=jl/>（其间，2003年获聘为新疆维吾尔自治区第十届人民政府特邀行政执法监督员，聘期五年。<ref name=xzb>{{cite news |language =  |author = |coauthors =  |url = http://www.110.com/fagui/law_177068.html |title = 新疆维吾尔自治区人民政府办公厅关于聘请自治区第十届人民政府特邀行政执法监督员的决定（新政办发〔2003〕134号）|publisher = 新疆维吾尔自治区人民政府办公厅 |pages =  |date = 2003-10-14 |accessdate = }}</ref>）2004年9月起，任新疆维吾尔自治区道路运输管理局党委委员、总经济师。2007年12月，任[[新疆维吾尔自治区道路运输管理局|新疆维吾尔自治区道路运输管理局]]党委委员、副局长（主持行政工作）。<ref name=jl/>2008年8月，任新疆维吾尔自治区道路运输管理局党委副书记、局长。<ref name=jl/><ref>{{cite news |language =  |author = |coauthors =  |url =  |title = 新疆维吾尔自治区人民政府干部任免通知（新政任字〔2015〕131号）|publisher = 新疆维吾尔自治区人民政府 |pages =  |date =2015年7月14日  |accessdate = }}</ref><ref>{{cite news|language =|author =|coauthors =|url =http://www.xjkunlun.cn/zzgz/gbgz/gbrm/2008/505942_1.htm|title =自治区党委管理干部近期任免|publisher =昆仑网|pages =|date =2008-11-04|accessdate =|deadurl =yes|archiveurl =https://web.archive.org/web/20161224095748/http://www.xjkunlun.cn/zzgz/gbgz/gbrm/2008/505942_1.htm|archivedate =2016年12月24日|df =}}</ref>2015年7月，任[[新疆维吾尔自治区交通运输厅|新疆维吾尔自治区交通运输厅]]厅长、党委副书记。她是新疆维吾尔自治区第十一届政协常委。第九、十、十一届全国人大代表。<ref name=jl/>
