'''祖国不会忘记'''（又名“祖国不会忘记我”），[[张月谭|张月谭]]作词，[[曹进|曹进]]作曲，歌词1992年创作于渭南市桥南镇。该歌曲歌颂了为中国和平与繁荣默默奉献的人们。

== 创作背景 ==
1992年，[[国防科工委|国防科工委]]政治部要求各部队自编自演一台节目进行录像调演。国防科工委某基地宣传处为制作调演节目征集歌曲，时任所在部队宣传处处长的张月谭创作的《祖国不会忘记我》是其所在部队参赛节目中的一首歌曲，该曲在此次调演中获得了一等奖。后曲调又经过曹进的斟酌、修改成为现在流行的版本。<ref>{{Citeweb|url=http://qiaonangongshe.blog.163.com/blog/static/26499612200791110581925/ |title=《祖国不会忘记我》歌曲的创作过程 |publisher=网易博客 }}</ref> 
== 获奖情况 ==
*2001年，该曲被选为中宣部第八届“[[五个一工程|五个一工程]]”获奖作品。
*2009年5月，该曲被选入中宣部的推荐的《[[中宣部推荐100首爱国歌曲|100首爱国歌曲名单]]》<ref>{{Citeweb|url=http://news.xinhuanet.com/politics/2009-05/25/content_11434058.htm |title=100首爱国歌曲名单公布 |date=2009年5月25日 |publisher=新华网 |language=zh-cn }}</ref>。

== 歌曲影响 ==
“祖国不会忘记”一语常用于对各行业奉献者的颂扬及肯定，如《人民日报》1999年10月25日文章《祖国不会忘记》赞扬了中国海关工作者一直以来的工作<ref>{{Citeweb|url=http://www.people.com.cn/rmrb/199910/25/newfiles/col_19991025001096_jjzk.html |title=祖国不会忘记 |publisher= 人民日报 |date=1999年10月25日 }}</ref>。
