{{中國禪宗}}
'''祖師禪'''，[[禪宗|禪宗]]術語，為[[禪|禪]]的一種上乘法門，有別於如來禪、相似禪。旨在利用參究祖師公案的方式，讓人究竟瞭解自己。

== 歷史與內涵 ==
中國在唐宋之後，禪門日漸式微，禪宗祖師乃改以古來[[公案|公案]]令學子時時刻刻在疑情中參究話頭，令其妄心不生而守安修慧。假以時日因緣成熟之際，忽地疑團粉碎，識得萬法本來面目。

[[話頭禪|話頭]]（公案）是祖師與學子之間對話的頭，亦即是對話的核心。參究公案能令學子瞭解祖師的教學用意。祖師禪就像是一把祖師的利刃，斬盡學子一切葛藤，然而學子若沒有大信心及大憤心，腳踏實地參究，疑情無法承為大疑團，如此欲[[開悟|開悟]]則遙遙無期<ref>《祖師禪－－宗門格外玄旨》，圓濟惠忠禪師著，旺文社出版。</ref>。

[[六祖惠能|六祖惠能]]傳的是南宗禪，又叫'''頓悟禪'''，與北宗漸修禪不同。北宗漸修禪靠打坐、內觀自己，以明自性，觀一念未生前的本來面目。北宗禪和南宗禪，全包括在祖師禪裡。<ref>[http://www.ss.ncu.edu.tw/~calin/other/introduction.doc 禪宗八講]</ref>但是南宗禪極為靈活，不得以才用公案，比直指（禪師舉竹篦）<ref>[https://sites.google.com/site/oxsearchhut/chan-zong-wu-men-guan-xu/shou-shan-zhu-bi 首山竹篦]：「無門曰：『喚作竹篦則觸、不喚作竹篦則背。
不得有語、不得無語、速道速道。』
」</ref>多了一個方便。

== 修行方式 ==
參究公案時，不得從[[三藏十二部經|三藏十二部經]]中尋找答案，一是如此無法瞭解禪宗祖師的心意，二是從經藏中亦無法找到答案。

參公案需具備的三個條件：
* 大信心：對法門、禪師和自己有信心。
* 大憤心：將開悟視為人生最重要的事，不半途而廢。
* 大疑團：以堅強的毅力與決心，時時刻刻守住疑情。

== 入門公案 ==
針對公案，一般較為人知的著作有《[[五燈會元|五燈會元]]》、《[[景德傳燈錄|景德傳燈錄]]》、《[[無門關|無門關]]》、《[[碧岩錄|碧岩錄]]》等。現代則有由韓國[[臨濟宗|臨濟宗]]祖師[[惠菴玄門|惠菴玄門]]禪師、[[牟峰印空|牟峰印空]]禪師、[[圓濟惠忠|圓濟惠忠]]禪師等人所編著的《海底生煙》（原名：《祖師禪要》），將公案由易至難收錄編輯，方便指導與學習。

以《海底生煙》<ref>《海底生煙》，牟峰印空禪師英譯、圓濟惠忠禪師漢譯，旺文社出版。</ref>為例，祖師禪的入門公案為〈須彌山〉：
*一個學人問[[雲門文偃|雲門文偃]]禪師：「學人不起一念，還有過也無？」 雲門禪師回答：「須彌山。」
*其中，為何雲門禪師回答「須彌山」，便是參公案者需要去回答給指導者（禪師）的問題。
