'''温福铁路'''，又称'''温福线'''，为[[东南沿海客运专线|东南沿海客运专线]]的一部分，是一条由[[中华人民共和国铁道部|中华人民共和国铁道部]][[上海铁路局|上海铁路局]]和[[南昌铁路局|南昌铁路局]]运营的连接中國[[浙江省|浙江省]][[温州市|温州市]]和[[福建省|福建省]][[福州市|福州市]]的国家I级双线电气化铁路干线，为福建省内首条[[高速铁路|高速铁路]]。温福铁路由铁道部、福建省和浙江省合资建设，于2005年8月正式开工建设，2009年7月1日正式试运行并开通货物列车<ref>[http://www1.www.gov.cn/jrzg/2009-07/01/content_1354627.htm 温福铁路7月1日起通车运营 年内开行动车组列车] {{webarchive|url=https://web.archive.org/web/20170802170949/http://www.gov.cn/jrzg/2009-07/01/content_1354627.htm |date=2017-08-02 }}</ref><!-- 计划于2009年6月30日试运行<ref>[http://www.fjsen.com/newscenter/2008-10/10/content_580874.htm 温福铁路计划于2009年6月30日开通试运行]</ref> -->，比原计划提前8个月。此铁路是中国“十一五”重点建设项目，也是“四纵四横”主通道沿海客运专线的重要组成部分。

温福铁路在2009年9月28日正式运营[[和谐号|和谐号]][[动车组|动车组]]客运列车。目前温州与福州间最快可在1小时25分钟之内实现直达<ref name="D3108时刻表">[http://chaxun.shike.org.cn/ti.asp?c=D3108&n=22147&fz=%B8%A3%D6%DD&dz=%C9%CF%BA%A3%C4%CF D3108次动车组时刻表]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

温福铁路全长298.4公里，包括浙江段69公里，福建段229.14公里（其中包括新建铁路段211.04公里及福州枢纽段18.482公里<ref>[http://www.chinabidding.com/bidDetail/2291308.html 通信、信号、牵引供电及电力供电系统集成施工招标公告]</ref><ref name="fujian" />），近期最大区段货流密度1270万吨，客车35对，运营时速250公里。最高试验时速曾达到292公里。

温福铁路通过[[马尾站|马尾站]]与既有的[[福马铁路|福马铁路]]（福州至马尾）连接，同时通过[[福州南站|福州南站]]与2005年动工的[[福厦铁路|福厦铁路]]，[[向莆铁路|向莆铁路]]支线相接。温福铁路所在区域包括低山丘陵区、山间谷地区、滨海平原区，地形复杂，桥梁和隧道占全线78%。其中浙江段有[[飞云江大桥|飞云江大桥]]、[[鳌江大桥|鳌江大桥]]和[[平阳大桥|平阳大桥]]等特大桥7座。福建段隧道共有52座，全长140.5公里<ref>{{cite web

== 项目投资 ==
温福铁路初步设计批复总概算174.8亿元（人民幣、下同），浙江段约51.51亿元。福建段总投资概算126亿元<ref>[http://www.fujian.gov.cn/zwgk/zjxx/zdxm/zdjsjb/200903/t20090327_117389.htm 重点建设简报] {{webarchive|url=https://web.archive.org/web/20101119080727/http://www.fujian.gov.cn/zwgk/zjxx/zdxm/zdjsjb/200903/t20090327_117389.htm |date=2010-11-19 }}</ref>。

[[中华人民共和国铁道部|铁道部]]还从{{link-de|德国复兴信贷银行|KfW Bankengruppe}}({{lang|de|KfW}})获得了一笔贷款，用以支付温福铁路建设的部分费用。

铁道部投资70％，省出资30％。项目业主由铁道部、浙闽两省和沿线各市县组成，铁道部投资业主为[[上海铁路局|上海铁路局]]，浙江省投资业主为由铁道部和浙江省共同组建的“沿海铁路浙江有限责任公司”，福建省投资业主为由铁道部和福建省共同组建的“沿海铁路福建有限责任公司”。这两家公司是依照铁道部与各省政府出资比例建立，因此也按比例分配董事会席位<ref>{{cite web

== 建设背景 ==
温福铁路沿线经过的地区，包括福建东部沿海及浙江东南沿海地区，属于丘陵沿海地带，陆路交通直到温福高速公路建成之前都十分不便，严重制约当地经济发展。因此在2004年福建省政府和浙江省政府就要求中央政府及铁道部修建温福铁路以改变此地区落后的交通状况。

=== 建设前阶段 ===
* 1988年，[[宁德市|宁德市]]提出修建温州至宁德间铁路的要求
* 1992年，宁德市受[[福建省|福建省]][[浙江省|浙江省]]两省委托开始该项目前期工作
* 1993年6月，福建省和浙江省计委召开对温福铁路项目可行性研究报告的审查会议
* 1995年，铁道部将该项目列入 “九五”地方铁路建设计划，要求地方为主建设
* 1998年，温福铁路争取作为国家级铁路项目，铁道部对此积极回应。
* 1999年，温福铁路作为“八纵八横”铁路的组成部分进入“十五”国家铁路建设计划，铁道部正式启动了该项目前期工作；
* 2000年，温福铁路作为“十五”开工项目列入国家“十五”计划；
* 2001年，国务院批准福建省铁路建设附加费征收期限延长近5年，所收的铁路建设附加费作为国家资本金缴入地方国库，专项用于温福铁路等项目建设。
* 2001年3月28日，铁道部长傅志寰、福建省长[[习近平|习近平]]、浙江省长柴松岳分别代表铁道部、浙江省政府、福建省政府三方共同签署了《合资建设温福铁路协议书》
* 2001年12月31日，[[中华人民共和国铁道部|铁道部]]、[[浙江省|浙江省]]政府、[[福建省|福建省]]政府联合向国家发展计划委员会提交《新建温州至福州铁路项目建议书》，要求兴建温福铁路。
* 2002年4月，中国国际工程咨询公司受国家发展计划委员会的委托组织完成对该项目的立项评估，并向国家发展计划委员会上报了《温福铁路项目建议书评估报告》。
* 2002年8月中旬，国家发展计划委员会通过了铁道部和福建省政府、浙江省政府联合报送的《温福铁路项目建议书》及《温福铁路项目建议书评估报告》，并上报提请国务院审批
* 2002年9月18日，国务院办公会批准《温福铁路项目建议书》
* 2002年10月，铁道部与浙江省政府、福建省政府向国家发展计划委员会报送《新建温州至福州铁路可行性研究报告》
* 2002年10月15日，国家发展计划委员会批复《审批新建温州至福州铁路项目建议书的请示的通知》,初定技术标准为国铁I级、单线、时速140公里
* 2003年3月至12月，[[铁道第四勘察设计院|铁道部第四勘察设计院]]三次对温福线技术标准进行补充研究，完成补充可行性研究报告并送审
* 2004年5月，补充可行性研究完成
* 2004年8月3日至8月6日，铁道部主持对温福铁路初步设计方案进行会审，提高设计标准为全线时速220公里，双线，电力牵引，工期4年半
* 2004年10月18日，国家发展和改革委员会批复了可行性研究报告。
* 2004年10月，国务院批准《国家发改委关于审批新建温州至福州铁路可行性研究报告的请示》
* 2004年11月19日，[[中华人民共和国铁道部|铁道部]]、[[浙江省|浙江省]]政府、[[福建省|福建省]]政府联合批复了温福铁路的初步设计。

== 对沿线地区产生的影响 ==
温福铁路使得温州及长三角地区通往中国南方沿海地区的交通状况将得到极大改善，预计对该地区现有的公路运输服务构成竞争<ref>[http://finance.66wz.com/system/2009/04/01/101181421.shtml 两铁路即将开通 温州“公铁赛跑”再度升温]</ref>，同时由于其为通过的宁德市的第一条铁路线<ref>[http://www.railcn.net/news/railway-express/78773.html 温福铁路架起腾飞的轨道 ]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，预计将带动宁德经济发展<ref>[http://www.xinhuanet.com/chinanews/2009-03/31/content_16115894.htm 宁德借力温福铁路谋求新一轮跨越]</ref>，而对于南端的福州市而言，温福铁路使得福州前往长三角甚至华北不再需要绕道较为难行的福建山区，因而所需时间大为缩短。
