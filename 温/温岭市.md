'''温岭市'''在[[中国|中国]][[浙江省|浙江省]]东南沿海，是[[台州市|台州地级市]]下辖和代管的[[县级市|县级市]]。

温岭全市总面积为926平方公里，（2017年）年末全市户籍总人口1220090人，其中男性人口617459人，女性人口602631人，男女性别比为102.5︰100；全市城镇人口593604人，乡村人口626486人<ref>{{cite web |author1=温岭市统计局 |title=温岭市2017年国民经济和社会发展统计公报 |url=http://www.wl.gov.cn/web/zwgk/tjxx/tjgb/201804/t20180403_216090.shtml |website=中国温岭 |accessdate=2018-10-30}}</ref>。政府驻地[[太平街道_(温岭市)|太平街道]]人民东路，政府驻地邮政编码317500。

==行政区划==
全市下辖5个街道、11个镇，97个居民区、847个行政村：
*街道办事处：太平街道、城东街道、城西街道、城北街道、横峰街道；
*镇：[[大溪镇|大溪镇]]、[[泽国镇|泽国镇]]、[[新河镇_(温岭市)|新河镇]]、[[箬横镇|箬横镇]]、[[松门镇|松门镇]]、温峤镇、坞根镇、城南镇、石桥头镇、滨海镇、石塘镇。　

== 历史沿革 ==
1469年（[[明宪宗|明宪宗]][[成化|成化]]五年）十二月，析黄岩南方岩、太平、繁昌三乡置[[太平县_(成化)|太平县]]，属[[台州府|台州府]]，是为建县之始。治所设于太平乡泉溪。1513年（[[正德_(明朝)|正德]]八年），县城造六个城门，东西各置一个水门。1552年（[[嘉靖|嘉靖]]三十一年），县城修筑2.5公里的城墙。城略成方形，故有“方城”之别称。

1914年（民国三年），因与山西、四川、安徽等省[[太平县|太平县]]同名，乃改为温岭县，县名得名于境内的[[温峤岭|温峤岭]]。

1994年3月，经[[中华人民共和国国务院|中国国务院]]批准，撤县设市，称温岭市。

== 自然地理 ==
浙江沿海中部，台州南部，北纬28°22'，东经121°21'。海岸曲折，多岛屿。

== 经济 ==
温岭市经济发达，为全国最具实力的[[县级行政区|县市]]之一，其综合竞争能力[[中国百强县#县域经济论坛评价|2010年位居浙江省县（市）的第七位，全国百强县（市）列第二十一位]]。2017年[[国内生产总值|地区生产总值]]990.36亿元，人均81283元（折合12039美元）<ref>温岭市2017年国民经济和社会发展统计公报</ref>。

中国“[[股份合作制|股份合作制]]”的发源地，民营经济很发达。在2007年全国“百强县”排名中排名15。有中国[[水泵|水泵]]之乡、中国小型空压机之乡、中国大棚西瓜之乡、中国大棚葡萄之乡、中国高橙之乡、中国果汁之乡的美称。

== 古代名人 ==
[[File:Dai_Fugu.JPG|200px]]
*[[戴复古|戴复古]]，字式之，黄岩南塘人（新河镇屏上村），生于南宋乾道三年（1167），卒于淳祐八年（1248）。曾在诗坛上负盛名达50年之久。留有《石屏诗》六卷、《石屏词》一卷。
*[[张元勋|张元勋]]（1533-1590），字世臣，号东瀛，海门卫新河所人（今温岭市新河镇人），庆隆三年（1569），因抗倭有功，擢福建副总兵。庆隆五年为总兵官镇守广东。万历三年升右都督。新河镇东门沈家墙里现存有张元勋旧居。
*[[方幹民|方幹民]]（1906—1984），油畫家，美術教育家，中國現代藝術的開拓者之一，曾經擔任浙江美術學院教務長、油畫系教授。
