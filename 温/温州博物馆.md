'''温州博物馆'''，位于[[浙江省|浙江省]][[温州市|温州市]][[鹿城区|鹿城区]]市府路，是温州市属综合性博物馆。

==简介==
1958年5月，温州博物馆在[[江心屿|江心屿]]的[[兴庆寺_(温州)|兴庆寺]]创建，[[方介堪|方介堪]]兼任首任馆长，这是温州第一家综合性博物馆。“[[文革|文革]]”期间，撤销温州博物馆，并入温州市图书馆。1981年恢复温州博物馆原建制<ref name=gao>高启新，温州区域博物馆群的建设，温州职业技术学院学报2015(2):6-10</ref>。

2003年10月初，位于世纪广场西侧的温州博物馆新馆试开馆<ref>{{Cite web |url = http://www.zjwh.gov.cn/dtxx/2007-11-11/34636.htm |title = 温州市新建博物馆即将试开馆 |author =  |date = 2007-11-11 |publisher =  浙江省文化厅|language =  |accessdate =  |quote =  }}</ref>。一个月后闭馆，重新调整并改进。2004年1月12日正式开馆<ref>{{Cite web |url =http://news.sina.com.cn/c/2004-01-12/11471565956s.shtml  |title =温州博物馆今开馆 带来视听触觉全新感受  |author =  |date = 2004-01-12 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。2008年3月起免费开放。新馆占地面积约2.34公顷，建筑面积2.6万平方米，展厅面积1.2万平方米，设历史馆、书画馆、陶瓷馆、自然馆、工艺馆、王维新铜版画陈列室、临时展馆<ref name=gao/>。新馆建筑设计灵感来自[[雁荡山|雁荡山]]，大门照壁饰以大榕树石雕，中厅[[花岗岩|花岗岩]]壁面悬挂着9幅中国传统神话铜雕壁画，分别为[[盘古|盘古开天]]、[[女娲补天|女娲补天]]、[[夸父追日|夸父追日]]、[[精卫填海|精卫填海]]、[[后羿射日|后羿射日]]、[[嫦娥奔月|嫦娥奔月]]、[[神农|神农]][[伏羲|伏羲]]、[[燧人氏|燧人取火]]、[[大禹治水|大禹治水]]<ref name=jj>{{Cite web |url =https://www.wzmuseum.cn/Col/Col8/Index.aspx  |title = 温博简介 |author =  |date =  |publisher = 温州博物馆 |language =  |accessdate =2017-07-05  |quote =  }}</ref>。2015年，历史馆启动改造，2017年1月21日完工并对外试开放，2017年5月18日[[世界博物馆日|世界博物馆日]]正式开放<ref name=sanjian>{{Cite web |url = http://zj.zjol.com.cn/news/643366.html |title = 温州博物馆新历史厅正式开放 这三件宝贝最有故事 |author =  |date =2017-05-18   |publisher =  浙江新闻|language =  |accessdate =  |quote =  }}</ref>。

本馆馆藏文物3万余件，分为青铜器、陶瓷器、彩塑、漆器、砖雕、书画等20个门类，以陶瓷器、彩塑、书画为特色。珍贵文物有宜山出土的[[东晋|东晋]]朱曼妻薛氏石买地券，金沙塔出土的[[唐代|唐代]]写经，白象塔出土的[[北宋|北宋]]彩塑，西山出土的北宋青瓷，[[南宋|南宋]][[龙泉窑|龙泉窑]]青瓷[[叶适|叶适]]墓志，[[中华民国|中华民国]]“双麒麟”脚踏弹棉机等等<ref name=jj/><ref name=sanjian/>。

2017年，温州博物馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref>。
