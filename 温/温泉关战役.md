'''溫泉關戰役'''（{{lang-el|Μάχη τῶν Θερμοπυλῶν}}）是[[波希戰爭|波希戰爭]]中的一次著名戰役，也是[[西方世界|西方]]歷史上一次重要的戰役，具有極高的歷史意義和軍事意義。[[希臘|希臘]]的[[斯巴達|斯巴達]]國王[[列奧尼達一世|列奧尼達一世]]率領三百名[[斯巴達|斯巴達]]精銳戰士與部分[[希臘|希臘]]城邦聯軍於[[溫泉關|溫泉關]]抵抗[[波斯帝國|波斯帝國]]，成功拖延波斯軍隊進攻，爭取到[[雅典|雅典]]及其他城邦準備戰役的寶貴時間，為之後[[希臘|希臘]]的勝利立下大功。但因寡不敵眾，三百名[[斯巴達|斯巴達]]戰士及殿後的志願軍全部陣亡，無人被俘或投降，列奧尼達被[[梟首|梟首]]。

== 背景 ==
公元前480年，時間可能從8月7日或9月 ，[[阿契美尼德王朝|波斯帝国]]國王[[澤克西斯一世|薛西斯一世]]為了雪耻[[马拉松战役|马拉松战役]]而征伐[[古希臘|古希臘]]，準備了一支大軍，面對這個威脅，希臘的[[城邦|城邦]][[雅典|雅典]]與[[斯巴達|斯巴達]]不計前嫌地聯合起來對抗波斯，組成了一支由斯巴達領導的聯軍，他們原先的計畫是要在[[奧林帕斯山|奧林帕斯山]]南端的一個峽谷（Vale of Tempe）阻擋波斯人，但是在得知波斯人即將繞道避開這個山谷時，希臘聯軍撤退了，附近的希臘城邦得知了聯軍即將撤出這個地區時，紛紛望風而降。為了阻止其他的城邦繼續變節，希臘人決定在溫泉關建立他們的防禦據點。溫泉關是一個易守難攻的狹窄通道，一邊是大海，另外一邊是陡峭的山壁。這個村莊附近有熱湧泉，因而得到[[溫泉關|溫泉關]]（Thermopylae）這個名字。

== 戰役經過 ==
[[File:溫泉關位置圖.jpg|thumb]]
[[斯巴達|斯巴達]]國王[[列奧尼達一世|列奧尼達一世]]以其本國精兵300人、400名[[底比斯_(希腊)|底比斯]]人和大約6000名希臘其他城邦的聯軍，利用地形優勢在溫泉關抵擋數量上遠遠超過他們的波斯軍隊長達三天。

列奧尼達的軍隊擋住了當時認為唯一能通往希臘的通道，使得波斯軍隊在頭兩天不得寸進，並且死傷慘重。但在第三天，一個[[希臘人|希臘人]]背叛希臘陣營，帶領波斯軍隊沿著山區的小徑繞到希臘聯軍的後方，導致聯軍遭到迂迴包圍。見此列奧尼達解散了希臘聯軍，留下300名斯巴達精兵與大約3000名希臘志願軍墊后。列奧尼達與三百位戰士視死如歸，堅守自己的位置，奮力頑抗，同時保護其他正在撤退中的希臘部隊，直至全軍覆沒。雖然波斯軍隊最後仍成功佔領溫泉關，但卻為此付出了沉重的代價。

面對數十倍於己的波斯軍隊，斯巴達人與希臘聯軍奮勇抵抗死守溫泉關，在第一天阻擊兩萬人的進攻，第二天阻擊了五萬人的衝鋒。這是軍事史上的一次偉大壯舉，此次戰役也完全驗證了斯巴達人的勇猛與驍勇善戰。

== 結果 ==
列奧尼達率領的部隊在溫泉關勇猛的死戰，為[[雅典|雅典]]的海軍提供了寶貴的時間，使雅典在接下來的[[薩拉米灣海戰|薩拉米灣海戰]]中能夠獲勝。雅典摧毀了大量的波斯船艦。波斯失去了海軍，無法向陸軍提供補給，這給波斯軍隊帶來了致命的影響。[[澤克西斯一世|薛西斯一世]]退回[[亞洲|亞洲]]，留下了指揮官[[马铎尼斯|马铎尼斯]]（[[:en:Mardonius|Mardonius]]），與大量的波斯軍隊，繼續在他們已經佔領的領土與希臘軍隊作戰。最後在[[普拉提亚战役|普拉提亚战役]]中，希臘聯軍擊潰了馬鐸尼斯，為[[波希戰爭|波希戰爭]]劃上句號。從此波斯再也無力入侵希臘。

== 纪念 ==

斯巴達300勇士戰死後，為了紀念他們的英勇戰績，人們在一尊獅子狀紀念碑上鐫刻下由古希腊诗人[[西蒙尼德|西蒙尼德]]所做的警句：

:{{lang|grc|Ὦ ξεῖν', ἀγγέλλειν Λακεδαιμονίοις ὅτι τῇδε}}
:{{lang|grc|κείμεθα, τοῖς κείνων ῥήμασι πειθόμενοι}}.

:''Ō ksein', angellein Lakedaimoniois hoti tēide''
:''keimetha, tois keinōn rhēmasi peithomenoi.''

:流浪人，你若到斯巴达，请报告那里的公民们，我们阵亡此地，至死犹恪守他们的命令。<ref name="Ziogas">{{cite web | last = Ziogas | first = Ioannis | title = SPARSE SPARTAN VERSE: FILLING GAPS IN THE THERMOPYLAE EPIGRAM | work = Ramus 43 (2) | publisher = Aureal Publications |date=9 November 2014| url = http://www.academia.edu/3387164/Sparse_Spartan_Verse_Filling_Gaps_in_the_Thermopylae_Epigram | accessdate=10 October 2014}}</ref>

[[美國|美國]]於2007年3月9日上映的電影《[[300壯士：斯巴達的逆襲|-{zh-hans:斯巴达三百勇士;zh-hk:戰狼300;zh-tw:300壯士：斯巴達的逆襲;}-]]》便是由敘述這段故事的[[300_(漫畫)|漫畫]]所改編。
溫泉關戰役的歷史意義深刻且深遠，雖然最後以失敗告終，但千餘年來一直被後人所津津樂道，到了文化交融的今天成為世界各民族學習的楷模。
