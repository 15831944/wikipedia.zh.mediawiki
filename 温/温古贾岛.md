'''温古贾岛'''（Unguja），通常非正式地称为'''桑给巴尔岛'''，是[[印度洋|印度洋]]上[[桑给巴尔群岛|桑给巴尔群岛]]的主岛，位于[[坦桑尼亚|坦桑尼亚]][[桑给巴尔|桑给巴尔]]，岛上人口相对稠密。

== 地理 ==
岛上地形主要以[[丘陵|丘陵]]为主，长85公里，最宽处宽30公里，面积约2600 km²。

== 人口 ==
根据2002年的人口普查，温古贾岛上人口为622459<ref>{{cite web |url=http://www.nbs.go.tz/index.php?option=com_content&view=article&id=104&Itemid=110 |title=存档副本 |accessdate=2010-05-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20100522034737/http://www.nbs.go.tz/index.php?option=com_content&view=article&id=104&Itemid=110 |archivedate=2010-05-22 }}</ref>，多在桑给巴尔市区<ref>http://www.tanzania.go.tz/census/unguja.htm 2002年坦桑尼亚普查数据</ref>。岛上最大城市是桑给巴尔市。
