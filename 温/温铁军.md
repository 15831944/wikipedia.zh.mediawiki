'''温铁军'''{{BD|1951年|5月|||catIdx= W温}}出生于[[北京市|北京市]]，籍贯[[河北省|河北省]][[秦皇岛市|秦皇岛市]][[昌黎县|昌黎县]]，为[[中国人民大学可持續发展高等研究院|中国人民大学可持續发展高等研究院]]院长、《中国改革》、《改革内参》杂志社社长兼总编，知名[[农业|农业]]专家。<ref name=" ops" >{{cite web |url=http://news.qq.com/a/20090812/002338.htm |title=中国人民大学农业与农村发展学院院长温铁军 |accessdate=2009年8月12日 |publisher=腾讯网 |archiveurl=http://news.qq.com/a/20090812/002338.htm |archivedate=2012年11月6日 |work= |language =zh-cn}}</ref>

== 生平 ==
1951年，温铁军出生在[[北京市|北京市]]，[[文革|文革]]期间中断学习，恢复高考之后，考入[[中国人民大学|中国人民大学]]新闻系，1983年大学毕业后被中国政府派送到美国[[密执安大学|密执安大学]]社会调查研究所和[[世界银行|世界银行]]调查学习，并且在美国[[哥伦比亚大学|哥伦比亚大学]]、[[康奈尔大学|康奈尔大学]]和[[南加州大学|南加州大学]]学习，回国后，温铁军在[[中国农业大学|中国农业大学]]研究生院和中国农业大学经管学院升造，曾在中央军委总政治部研究室、中央农村政策研究室、国务院农村发展研究中心联络室、全国农村改革试验区办公室、农业部农村经济研究中心、中国经济体制改革研究会等政府机构任职，现为[[中國人民大學可持續發展高等研究院|中國人民大學可持續發展高等研究院]]院长、《中国改革》、《改革内参》杂志社社长兼总编。<ref name=" ops" />

2012年起担任中国农业银行股份有限公司的独立董事。<ref name="abc">
{{cite web |url=http://www.abchina.com/cn/aboutabc/investor_relations/corporate_governance/directors/director/ |title=中国农业银行董事会 |accessdate=2018年2月27日 |work= |language =zh-cn}}</ref>

== 思想 ==
温铁军主要从事中国国情、[[农村城镇化|农村城镇化]]、[[三农问题|三农问题]]的研究。<ref name=" ops" />

== 作品 ==
*《三农问题：世纪末的反思》
*《解构现代化》
*《中国农村基本经济制度研究》
*《我们到底要什么》
*《认识与实践的对话》
*《八次危机》
