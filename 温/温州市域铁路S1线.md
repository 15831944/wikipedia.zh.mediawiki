'''溫州市域鐵路S1線'''又称'''溫州轨道交通S1線'''，是[[浙江省|浙江省]][[溫州市|溫州市]]第一条城市[[轨道交通|轨道交通]]线路，线路代表色为蓝色（#0061AE）<ref>{{Cite web|url=http://wlwz.wenzhou.gov.cn/wzshow_68562.html|title=请问S1线的标志颜色|accessdate=2018-11-07|author=温州市铁路与轨道交通投资集团有限公司|date=2018-11-04|publisher=温州网络问政平台}}</ref>。S1线为[[温州轨道交通|温州轨道交通]]市域铁路网中的重要主干线，是[[温州南站|温州南站]]、[[温州站|温州站]]、[[溫州龍灣國際機場|溫州龍灣國際機場]]等不同交通枢纽的纽带。

S1线路全长约77[[公里|公里]]，规划设站28个，平均站间距2.8公里；其中一期工程全长53.507公里，设站21个，其中预留3个<ref>{{cite web|url=http://www.wenzhou.gov.cn/art/2011/11/8/art_3597_191221.html|title=市域铁路四线交错勾勒都市新生活|publisher=温州政府网|date=2011-11-08|accessdate=2011-12-06|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120527062908/http://www.wenzhou.gov.cn/art/2011/11/8/art_3597_191221.html|archivedate=2012-05-27}}</ref>。

== 路线 ==
S1线为东西走向的郊区快线，构建未来温州大都市核心区两大中心——中心城和瓯江口新城的通勤铁路，承担都市区范围内东西向组团间快速交通联系，串联瓯海中心区、中心城区、龙湾中心与[[温州龙湾国际机场|龙湾机场]]和灵昆半岛，并服务[[温州南站|动车站]]、[[温州龙湾国际机场|龙湾机场]]。预留向洞头延伸条件。S1线路全长77.0km，平均站间距2.8km。設計時速120km/h，平均旅行速度50-60km/h。

S1线一期工程西起[[瓯海区|瓯海区]][[桐岭站|桐岭站]]，过[[甬台温铁路|甬台温铁路]][[温州南站|温州南站]]，上跨甬台温铁路至[[温州西站|温州西站]]，沿既有[[金温铁路|金温铁路]]通道至温州东站，折向西南沿[[龙湾区|龙湾区]]南洋大道、前房路至机场，出机场后向北跨越瓯江南口至[[灵昆岛|灵昆岛]]，终点为[[双瓯大道站|双瓯大道站]]，全长53.458km。为了加快S1线建设进度，一期工程以[[温州站|温州站]]为节点，分为东、西两段（温州站归入西段）分别编制设计方案。

其中西段（桐岭站至奥体中心站段）于2018年10月1日开始不载客试运行<ref>{{cite web|url=http://wlwz.wenzhou.gov.cn/wzshow_48142.html|title=高铁s1线（灵昆站）的建设进程|publisher=温州网络问政平台|date=2018-02-27|accessdate=2018-03-03|language=zh-cn}}</ref><ref>{{cite web|url=http://wlwz.wenzhou.gov.cn/wzshow_47124.html|title=S1开通时间|publisher=温州网络问政平台|date=2018-02-01|accessdate=2018-03-03|language=zh-cn}}</ref>，并于2019年1月23日起开通试运营<ref name=":1">{{cite news |author1=温州市铁路与轨道交通投资集团 |title=重磅！温州轨道交通S1线将于1月23日试运营 |url=https://mp.weixin.qq.com/s/GC76WdW-u4s--cRLitUmkA |date=2019-01-15}}</ref>。

==主要技术标准==
* 铁路等级：I级
* 正线数目：[[双线|双线]]
* 速度目标值：120km/h（CK37+000-CK39+000段限速80km/h）
* 正线线间距：4.0m
* 最小曲线半径：一般地段800m，困难地段700m。限速地段根据实际情况和速度时间曲线模拟确定
* 最大坡度：一般地段20‰，困难地段30‰
* 到发线有效长度：360m；机场站450m
* 站台长度：180m；站台宽度：岛式14.5m、侧式6m
* 行车指挥系统：分散自律调度集中（CTC）

== 车辆 ==
采用的D型<ref>[https://wlwz.wenzhou.gov.cn/source/mobile/wlwz/ask_detail.aspx?id=40576 温州市域线车辆都是统一市域动车D型车？]</ref>市域[[动车组|动车组]]基于[[中车四方|中车四方]][[Cinova|Cinova]]平台设计，结合城市轨道交通A型列车技术制造，为中国大陆自主研制的首列市域动车组。该动车组以电力牵引，[[接触网供电|接触网供电]]，采用[[动力分散式|动力分散式]]设计，初期以4辆编组运营，最大载客量1328人。同时，列车可以实现4辆、6辆、8辆灵活编组，以满足不同客流量的运营需求。每节车厢每侧设有4对车门，每对门为1.3米的大开度气密性双向[[塞拉门|塞拉门]]。在满载情况下，乘客可以在29秒以内快速完成乘降，大大减少了站停时间<ref>[https://zj.zjol.com.cn/news/599452.html 我国首列市域动车组问世 温州S1线明年底投用]</ref><ref>[http://www.sohu.com/a/131334086_355932 哇！温州市域铁路S1线首列市域动车组今天进行试乘，动车组照片抢先看！]</ref><ref>{{Cite web |url=http://www.sohu.com/a/131592909_669840 |title=厉害了！温州市域铁路S1线首列动车问世！ 关于S1线你想了解的全在这里了 |access-date=2018-01-15 |archive-url=https://web.archive.org/web/20180116004458/http://www.sohu.com/a/131592909_669840 |archive-date=2018-01-16 |dead-url=yes }}</ref>。列车采用基于[[ETCS|ETCS]]-1的点式[[ATC|ATC]][[列控系统|列控系统]]，预留升级至[[CBTC|CBTC]][[移动闭塞|移动闭塞]]的条件<ref>{{Cite journal|title=温州市域铁路交通系统关键技术分析|author=杜运国|url=|journal=铁道标准设计|issue=12|doi=|others=|year=2015|volume=|page=27-30+39|pmid=}}</ref>。

== 大事记 ==
*2011年11月11號S1线石坦隧道试验段开工建设<ref>{{cite web|url=http://news.66wz.com/system/2011/11/12/102866219.shtml|title=S1线石坦隧道开工建设 温州驶向市域铁路时代|publisher=溫州網-溫州商報|date=2011-11-12|accessdate=2011-12-06|language=zh-cn}}</ref>。
