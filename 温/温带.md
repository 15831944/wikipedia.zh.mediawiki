'''溫帶'''（{{lang-en|Temperate climate}}、{{lang-de|Gemäßigte Zone}}、{{lang-fr|Climat tempéré}}），在[[地理學|地理學]]上，是位於[[亚熱帶|亚熱帶]]和[[極圈|極圈]]之間的[[气候带|气候带]]。[[北半球|北半球]]溫帶區的範圍是從北緯23.5°的[[北回歸線|北回歸線]]到北緯66.5°的[[北極圈|北極圈]]之間。[[南半球|南半球]]溫帶區的範圍是從南緯23.5°的[[南回歸線|南回歸線]]到南緯66.5°的[[南極圈|南極圈]]之間。温带气候即包括比较温和多雨的海洋气候，也包括四季分明和比较干燥的大陆性气候。温带的大多数地区位于[[西风带|西风带]]内。

== 氣候類型 ==
[[柯本氣候分類法|柯本氣候分類法]]，屬於D類，分別是[[大陸性濕潤氣候|大陸性濕潤氣候]]（Dfa, Dwa, Dfb, Dwb）和[[副極地氣候|副極地氣候]]（Dfc, Dwc, Dfd, Dwd, Dsc, Dsd）

== 温带的气候区 ==
温带的气候区可以继续分为暖温带、涼温带和寒温带。整个温带的年平均气温为摄氏8度。广义上的温带也包括[[亚热带|亚热带]]。

=== 暖温带 ===
暖温带有两个不同的定义：
# 最热月的平均气温在摄氏20度以上
# 最热月的平均气温在摄氏10度以上，最冷月的平均气温在摄氏0度以上

按照第一个定义[[北京|北京]]属于暖温带，因为其六月至八月的平均气温在20摄氏度以上，按照第二个定义北京不属于暖温带，因为其一月的平均气温在0摄氏度以下。按照第二个定义中国[[秦嶺-淮河線|秦岭、淮河]]以南的大部分地区可以算暖温带，而以北则属涼温带。

与热带的界线一般为年平均气温不到摄氏20度，在这个年平均气温以下为温带或寒带。

=== 涼温带（中溫帶） ===
涼温带也有两个不同的定义：
# 最热月的平均气温在摄氏10度和20度之间
# 最热月的平均气温在摄氏10度以上，最冷月的平均气温在摄氏0度以下

与寒温带的区别在于年平均气温高于摄氏0度。

按照第二个定义中国华北和东北的大多数地区属于涼温带。[[波兰|波兰]]和[[俄罗斯|俄罗斯]]西部的大多数地区也属于涼温带。

=== 寒温带 ===
寒温带是年平均气温低于摄氏0度，同时最热月的平均气温高于摄氏10度的地区。与[[寒带|寒带]]的区分在于寒带的最热月的平均气温低于10摄氏度。此温度带亦称为“[[亚寒带|亚寒带]]”。

[[西伯利亚|西伯利亚]]的大多数地区和[[黑龙江省|黑龙江省]]的北部属于寒温带。

== 植被 ==
温带的森林有[[针叶林|针叶林]]、[[阔叶林|阔叶林]]和两者的混合林。大陆内部也有[[草原|草原]]、[[半沙漠|半沙漠]]和[[沙漠|沙漠]]。
