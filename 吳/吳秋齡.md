'''吳秋齡'''（{{bd|1957年|9月9日}}）是[[中華民國|中華民國]]女性政治人物，[[中國國民黨籍|中國國民黨籍]]。吳秋齡畢業於[[國立臺灣海洋大學|國立臺灣海洋大學]]，2014年起任宜蘭縣議會議員（[[羅東鎮_(台灣)|羅東鎮]]選區）。2018年，吳秋齡當選羅東鎮鎮長，也是自前任鎮長林姿妙當選以來第二位女性鎮長。<ref>{{cite web |author1=黃富溢 |title=羅東鎮長選戰！黃素琴、吳秋齡、薛呈懿三足鼎立 |url=https://news.ftv.com.tw/news/detail/2018524N04M1 |website=民視 |accessdate=2018-08-17}}</ref><ref>{{cite web |author1=羅建旺 |title=國民黨羅東鎮長初選民調 吳秋齡勝出 |url=https://udn.com/news/story/11322/2961218 |website=聯合報}}</ref><ref>{{cite web |author1=翁正杉 |title=吳秋齡專訪－談羅東交通與治水 |url=https://www.travelnews.tw/news/%E5%90%B3%E7%A7%8B%E9%BD%A1%E5%B0%88%E8%A8%AA%EF%BC%8D%E8%AB%87%E7%BE%85%E6%9D%B1%E4%BA%A4%E9%80%9A%E8%88%87%E6%B2%BB%E6%B0%B4/ |website=宜蘭新聞網 |accessdate=2018-08-17}}</ref><ref>{{cite news |title=爭羅東鎮長 縣議員吳秋齡成立工作室 |url=http://www.chinatimes.com/realtimenews/20170717002571-260407 |accessdate=2017-07-17 |agency=中時電子報}}</ref>現任[[羅東鎮_(台灣)|羅東鎮長]]。
