'''吳金川'''（{{Bd|1905年|6月23日|?||catIdx=}}<ref name=":0">{{Cite book|url=https://books.google.com.tw/books?id=jvnECgAAQBAJ&pg=PA122&dq=%E5%90%B4%E9%87%91%E5%B7%9D&hl=zh-TW&sa=X&ved=0ahUKEwiDoNvW3tjWAhUKxLwKHeLXAqYQ6AEIOjAD#v=onepage&q=%E5%90%B4%E9%87%91%E5%B7%9D&f=false|title=口述歷史(五)|last=|first=|date=2015-08-31|publisher=中央研究院近代史研究所|year=|isbn=9789860458800|location=|pages=|language=zh-TW|chapter=}}</ref>），是一位出身[[臺南|臺南]]的[[銀行家|銀行家]]。

== 生平 ==
1905年，吳金川出生於[[大日本帝國|大日本帝國]]臺灣[[臺南市_(州轄市)|臺南州臺南市]]。

後來，他就讀於[[公學校|公學校]]、[[長老教中學|長老教中學]]、臺南商業專門學校、[[臺南高等商業學校|臺南高等商業學校]]，又因[[學校成績表|學業成績]]優異被薦入{{Link-ja|東京商科大学|東京商科大学 (旧制)}}（後成為[[一橋大學|一橋大學]]）[[本科教育|本科]]。在東京商科大学，他成為{{Link-ja|高垣寅次郎|高垣寅次郎}}的學生，並專攻[[貨幣經濟學|貨幣金融]]；最終，他在該校取得[[商學碩士|商學碩士]]文憑。<ref name=":0" />

完成學業後，他前往[[滿州中央銀行|滿州中央銀行]]任職，並在該行提倡[[銀本位制|銀本位制]]<ref name=":1">{{Cite web|url=http://www.cdns.com.tw/news.php?n_id=23&nc_id=3567|title=尋根 神田光又找到2名姻親|accessdate=2017-10-05|author=林雪娟|date=2015-01-10|work=www.cdns.com.tw|publisher=中華日報新聞網}}</ref><ref name=":3">{{Cite book|url=https://books.google.com.tw/books?id=akmBAAAAIAAJ&q=%E5%90%B4%E9%87%91%E5%B7%9D&dq=%E5%90%B4%E9%87%91%E5%B7%9D&hl=zh-TW&sa=X&ved=0ahUKEwiDoNvW3tjWAhUKxLwKHeLXAqYQ6AEILDAB|title=總統的親戚: 台灣權貴家族|last=陳柔縉|first=|date=1999|publisher=時報文化出版企業股份有限公司|year=|isbn=9789571329703|location=|pages=|language=zh-TW|chapter=}}</ref>。<ref name=":0" />

1934年至1936年，他被短暫派駐於[[上海|上海]]，並曾前往[[南昌|南昌]]觀摩「[[新生活運動|新生活運動]]」。<ref name=":0" />

1940年，[[吳三連|吳三連]]前往[[中國東北地區|中國東北地區]]並拜訪他，更替他與[[楊肇嘉|楊肇嘉]]說[[媒人|媒]]；同年10月15日，他與楊湘玲在[[東京|東京]]舉行[[婚禮|婚禮]]。<ref name=":0" />

1943年，楊肇嘉在前往探視吳金川一家途中於[[新義州市|新義州]]遭押送至[[平安北道_(日本統治時代)|平安北道]]監禁，吳金川因等待甚久，隨後展開救援行動。<ref name=":0" />

1945年，他在[[中國抗日戰爭|中國抗日戰爭]]結束後，隨同[[張嘉璈|張嘉璈]]任職於[[國民政府|國民政府]]東北行轅經濟研究處主任[[研究員|研究員]]。<ref name=":0" />

1949年9月，他前往臺灣出任[[臺灣省合作金庫|臺灣省合作金庫]]<ref name=":1" />業務部[[經理|經理]]。

1952年3月，他進入[[彰化商業銀行|彰化商業銀行]]任職，歷任副總經理、[[總經理|總經理]]<ref>{{Cite web|url=http://catalog.digitalarchives.tw/item/00/17/d9/58.html|title=彰化銀行總經理吳金川報告營業概?及預算編製情形：本|accessdate=2017-10-05|work=catalog.digitalarchives.tw}}</ref>及[[董事長|董事長]]<ref name=":3" />，後於1966年屆齡退休<ref>{{Cite web|url=http://naturallybread.yam.org.tw/2006-adamanotice/oldhouse-y3/oldhouse-y3-101.htm|title=319旅行事誌----四知三惑--台中清水楊家宅群--楊金波.楊清珠.楊肇嘉.楊天賦宅|accessdate=2017-11-06|work=naturallybread.yam.org.tw}}</ref>。

== 家庭 ==
吳金川的[[妻子|妻子]]是[[楊肇嘉|楊肇嘉]]的[[女兒|女兒]]楊湘玲<ref>{{Cite book|url=https://books.google.com.tw/books?id=A5Q_AAAAMAAJ&q=%E5%90%B4%E9%87%91%E5%B7%9D&dq=%E5%90%B4%E9%87%91%E5%B7%9D&hl=zh-TW&sa=X&ved=0ahUKEwiDoNvW3tjWAhUKxLwKHeLXAqYQ6AEIMzAC|title=台灣新五大家族|last=司馬嘯青|first=|date=2005|publisher=玉山社出版事業股份有限公司|year=|isbn=9789867819208|location=|pages=|language=zh-TW|chapter=}}</ref>，他也因此與[[蔡萬才|蔡萬才]]成為[[連襟|連襟]]。<ref name=":3" />他與楊湘玲育有一子二女，長女為吳芳枝。<ref name=":0" />

== 家族 ==
吳金川的[[父親|父親]]吳鏡秋（[[表字|字]]應澄）是一名[[秀才|秀才]]<ref name=":3" />，他後來在[[臺灣總督府臺南師範學校|臺灣總督府臺南師範學校]]與[[長榮中學|長榮中學]]教授[[漢學|漢學]]（一說[[古代漢語|漢文]]）。<ref name=":1" /><ref name=":0" />
