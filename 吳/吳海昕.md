'''吳海昕'''（{{lang|en|'''Sofiee Ng Hoi Yan'''}}，{{bd|1989年|8月25日}}），本名'''吳凱欣'''，[[香港|香港]]女[[演員|演員]]及[[節目主持人|節目主持人]]。現為[[耀榮文化|耀榮文化]]旗下藝人。

== 簡介 ==
吳海昕畢業於[[保良局田家炳小學下午校|保良局田家炳小學下午校]]（現[[保良局田家炳千禧小學|保良局田家炳千禧小學]]）及[[聖公會莫壽增會督中學|聖公會莫壽增會督中學]]，曾在[[紐約電影學院|紐約電影學院]]修讀電影表演課程。她加入娛樂圈前曾為[[國泰航空|國泰航空]][[空中服務員|空中服務員]]，後於《[[2014年度香港小姐競選|2014年度香港小姐競選]]》中獲得旅遊大使獎，有「民選港姐」之稱<ref>{{cite news |language = zh-hk |url = http://www.hkcna.hk/content/2014/0902/290838.shtml |title = 民選港姐跌出三甲引質疑 邵佩詩：很公平 |publisher = 香港新聞網 |date = 2014-09-02 }}</ref>；參選港姐期間一直獲認定為冠軍大熱，傳媒亦提及決賽當晩她受到[[鄭裕玲|鄭裕玲]]、[[陳百祥|陳百祥]]、[[王祖藍|王祖藍]]、[[崔建邦|崔建邦]]和[[曾志偉|曾志偉]]五位司儀青睞<ref>{{cite news |language = zh-hk |url = http://www.metrohk.com.hk/?cmd=detail&id=241611 |title = 港姐賽果爆冷15號邵珮詩成3料冠軍 |publisher = [[都市日報|都市日報]]|date = 2014-09-01 }}</ref><ref>{{cite news |language = zh-hk  |url = http://cn.on.cc/hk/bkn/cnt/entertainment/20140831/bkn-20140831224225554-0831_00862_001.html?eventsection=hk_entertainment&eventid=40288107471040570147154447e91bc4 |title = 三強爆冷產生　大熱門齊齊落選 |publisher = [[ON.CC|東網]] |date = 2014-09-01 }}</ref><ref>{{cite news |language = zh-hk |url = http://cn.on.cc/hk/bkn/cnt/entertainment/20140901/bkn-20140901001126697-0901_00862_001.html?eventsection=hk_entertainment&eventid=40288107471040570147154447e91bc4 |title= 港姐：吳凱欣呻被王祖藍打亂陣腳 |publisher = 東網 |date = 2014-09-01}}</ref>，但結果三甲不入而只當選為旅遊大使，履行職責出席[[無綫電視|無綫電視]]多個慈善活動<ref>{{cite news |language= zh-hk |url = http://hk.apple.nextmedia.com/enews/realtime/20160728/55413731 |title =【兩年前港姐冠軍係邊位？】「民間港姐」吳凱欣大台發呆兩年 終於甩身！|date = 2016-07-28 |publisher = [[蘋果日報_(香港)|蘋果日報]]}}</ref>。

2015年卸任後，吳海昕開始參與電視劇、舞台劇、音樂錄像等各項演出，也曾於訪問裡表示因看到[[黃修平|黃修平]]執導的電影《[[狂舞派|狂舞派]]》，覺得主角[[顏卓靈|顏卓靈]]十分迷人且吸引，故此立志加入電影圈<ref>{{cite news |language= zh-hk |url = https://bka.mpweekly.com/interview/%E5%A8%9B%E6%A8%82123/20180606-116867 |title = 【吳海昕專訪】向黃修平叩門 花盡積蓄紐約追夢 |date = 2018-06-06 |publisher = [[明報周刊|明報周刊]]}}</ref>。
