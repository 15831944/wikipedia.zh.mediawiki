'''吳國楨'''（{{bd|1903年|10月21日|1984年|6月6日|吳國楨}}），字'''峙之'''、'''維周'''，湖北省[[建始縣|建始縣]]人。中華民國政治人物，曾任上海市市長、[[臺灣省主席|臺灣省主席]]等職，是塑造現代台灣的關鍵人物之一，亦以公開大膽地與[[蔣經國|蔣經國]]激烈爭執而著稱。
[[File:吳國楨2.jpg|thumb]]

== 早年 ==
1903年，吴国桢生于[[湖北省|湖北省]][[建始县|建始县]]凉水埠，自幼随任陆军部军训处长的父亲[[吴经明|吴经明]]旅居[[北京|北京]]。1914年，年仅10岁的吳國楨考入天津[[南開中學|南開中學]]就读。自南開畢業後，就讀北京[[清華大學|清华学校]]，1921年毕业後前往[[美國|美國]]留學，在[[格林內爾學院|格林內爾學院]]獲得[[經濟學|經濟學]][[碩士|碩士]]之後，1926年獲得[[普林斯頓大學|普林斯頓大學]][[政治學|政治學]][[博士|博士]]。

== 汉口时期 ==
1926年吳國楨回國後进入政界，在[[夏斗寅|夏斗寅]]任职期间，担任[[漢口|漢口市]]土地局長、財政局長，1928年出任湖北省财政厅长。1932年，吴任[[蔣中正|蔣中正]]機要秘書，同年出任漢口市長，时年29岁。1936年[[长江|长江]]发生洪水，吴国桢下令修建了巨大的堤防工程，挽救了这座城市<ref name="Time">"Man on the Dike"; Time，1950年8月7日 ([http://www.time.com/time/magazine/article/0,9171,812967,00.html])</ref>。

== 重庆时期 ==
1938年10月，日军攻占汉口，吳國楨全家逃往[[重庆|重庆]]。1939年，出任[[重慶|重慶]]市長。1942年，[[重慶大隧道慘案|重慶大隧道慘案]]发生，吴国桢被免職。1943年，吴国桢出任[[中華民國外交部|外交部]]政務次長，<!-- interacting with [[周恩来|周恩来]] as part of 抗日联合阵线. --> 1945年出任[[中國國民黨|中國國民黨]]中央宣傳部長。

== 上海市长 ==
1946年5月14日，國民政府令免[[錢大鈞|錢大鈞]]上海市市長本職，任命吳國楨為上海市市長<ref name=民/>{{rp|8052}}。5月14日，經行政院戰後第一次例會任命，吴国桢出任[[上海|上海]]市長<ref name="Time"/>。

1947年5月22日，吳國楨公佈四項緊急措施，決對煽動罷工罷課者，予以逮捕究辦<ref name="民">{{cite book |editor=李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編 |title=《中華民國史大事記》 |location=[[北京|北京]] |publisher=[[中華書局|中華書局]] |year=2011 }}</ref>{{rp|8359}}。吴国桢从政颇为勤勉，并且与半数以上被捕的共产党地下人员进行谈话<ref>吴国桢：《夜来临》，1955年</ref>。1947年7月19日，國民政府派吳國楨為上海選舉事務所主任委員；國民政府同時派定其為[[國民大會代表|國民大會代表]]，[[立法院|立法院]][[立法委員|立法委員]]<ref name=民/>{{rp|8385-8386}}。

1949年3月9日，[[蔣經國|蔣經國]]勸吳國楨“不辭市長職”<ref>蔣經國：〈危急存亡之秋〉，刊《風雨中的寧靜》，台北：正中書局，1988年</ref>{{rp|166}}。3月18日，《上海晚郵報》報導，上海市長吳國楨堅決請辭<ref name="錄">{{cite book |year=2011 |title=《1949大流亡：美國外交檔案室密錄》 |editor=王景弘編譯 |location=台北市 |publisher=玉山社出版 |isbn=978-986-294-000-6 }}</ref>{{rp|90}}。4月，吴国桢因目睹京滬杭警備司令[[湯恩伯|湯恩伯]]“吃空額”、“盜賣軍用汽油”等行為而對上海前景失去信心，並身患[[疟疾|疟疾]]，只得向蔣中正提出辭職。蔣中正准假一個月，吳國楨乃於4月17日攜家屬至臺灣。5月1日，行政院第五十七次會議議決：准上海市長吳國楨辭職，任命陳良為上海市長<ref name=民 />{{rp|8900}}。一個月后，解放军攻占上海，吳遂定居台灣<ref>{{Cite web|url=http://pylin.kaishao.idv.tw/?p=6654|title=北投埔林炳炎    » 誰敢威脅吳國禎–不沾鍋如何研究|accessdate=2016-12-21|work=pylin.kaishao.idv.tw}}</ref>。

== 臺灣時期 ==
蔣下野後，吳國楨隨蔣四處行動，作為他的幕僚，出謀劃策，並在1949年8月1日成立的中國國民黨總裁辦公室內任職，但吳國楨亦特別對[[臺灣|臺灣]][[政治|政治]]有所動作，此亦種下了[[陳誠|陳誠]]對吳不滿的原因。

1949年12月7日，[[國民政府|國民政府]]遷往[[臺北|臺北]]。12月15日，美國建議將陳誠換成吳國楨，蒋中正遂任命吴国桢接替軍人出身的[[陳誠|陳誠]]担任台湾省主席兼保安司令、[[行政院|行政院]]政務委員，以利用吳國楨“民主先生”的形象，“全力争取美援”<ref>吴国桢：《八十忆往》，香港《文汇报》1984年6月</ref>。12月16日，任命吳國楨為台灣省政府委員兼主席，[[蔣渭川|蔣渭川]]為委員兼民政廳廳長，[[任顯群|任顯群]]為委員兼財政廳廳長，[[彭德|彭德]]為委員兼建設廳廳長，[[陳雪屏|陳雪屏]]為委員兼教育廳廳長，[[徐慶鐘|徐慶鐘]]為委員兼農林廳廳長；任命吳國楨兼台灣省保安司令<ref>{{cite news|title=《總統府公報》第貳肆伍號|editors=總統府第五局|date=1949-12-31}}</ref>。吳國楨擔任臺灣省主席期間，致力于推動台湾[[地方自治|地方自治]]、農業改革，允许某些地方官员职位由普选产生，并试图减少滥用警权。1950年1月，吳國禎在一場由蔣介石召集三軍將領的會議中，提出應嚴禁軍隊[[走私|走私]]，並將茅頭直指海軍總司令[[桂永清|桂永清]]，桂要求吳拿出證據，吴隨即將海军走私材料遞呈蔣介石，並稱已逮捕海军司令部情报处长[[钟山|钟山]]以及舰长[[周齐荣|周齐荣]]。<ref>江南：〈吳國楨八十憶往〉</ref>

1950年6月25日，[[韓戰|韓戰]]爆發後，美國改為全力支持國民政府，在外交上承認台灣的國府當局，並給予[[美援|美援]]，吳國楨地位不再重要，與[[蔣經國|蔣經國]]與[[彭孟緝|彭孟緝]]的特務系統也不斷發生衝突。1952年，台湾第二次市县长和市县议会的选举在即，吴国桢在台北建立培训学校，轮番培训从各区选出的民众代表，此舉引發蔣氏父子高度不滿。<ref>吴国桢《从上海市长到“台湾省主席”》，第162页，上海人民出版社，1999年。</ref>[[臺灣火柴公司|臺灣火柴公司]]總經理[[王哲甫|王哲甫]]無辜被捕，吳下令放人。臺灣保安副司令[[彭孟緝|彭孟緝]]執行[[蔣經國|蔣經國]]命令，堅不放人，先判[[死刑|死刑]]；由於吳的反對，經蔣中正出面，改判七年徒刑。吳國楨已完全無法再與蔣經國共事，蔣中正曾派[[黃伯度|黃伯度]]傳話，許以[[行政院長|行政院長]]之職，要吳好好與蔣經國合作，願當院長兼省主席亦可。但吳一概拒絕，偕妻上[[日月潭|日月潭]]，聲言非辭職照准不下山。1952年[[復活節|復活節]]當天，吳國楨由日月潭下山，在[[臺中|臺中]]無錫飯店和僕人吃飯。一出門，就看到包姓司機臉色慘白；吳的座車前輪與主軸聯接的地方，疑似螺絲帽鬆動，幾遭不測。

1953年4月3日，吳國楨几乎被刺客杀死。4月10日，吳國楨辭去台灣省主席一職，蔣介石任命[[俞鴻鈞|俞鴻鈞]]接替吳國楨。5月24日，吳國楨夫婦得到邀請前赴[[美國|美國]]講學開會，蔣經國、陳誠到機場送行。<ref>[[茅家琦|茅家琦]]《蔣經國的一生與他的思想演變》，台北，[[臺灣商務印書館股份有限公司|臺灣商務印書館股份有限公司]]，2003年，第231-236頁。</ref>除了吳的次子吳修潢必須留在台灣作[[人質|人質]]，吴的全家都前往美国。<ref name="Kerr">''Formosa Betrayed'' by George H. Kerr; 1965, Houghton Mifflin, ISBN 978-0-306-70762-9. ([http://www.romanization.com/books/formosabetrayed/ text])</ref>1954年，当他的儿子离开台湾后，吳开始公开批评国民党政府存在的严重问题。同年，吳在杂志《Look》上发表文章“你的钱正被用来建造台湾的警察国家”。<ref name="Kerr"/>

== 吳國楨事件 ==
1954年2月，[[總統府|總統府]]秘書長-{[[王世杰_(中華民國)|王世杰]]}-被免職，蔣氏父子決定乘機削除「[[政學系|政學系]]」勢力。吳國楨和[[張群|張群]]、王世-{杰}-等人都過從甚密。1954年1月，臺灣傳出吳國楨[[貪污|貪污]]套取巨額[[外匯|外匯]]的傳聞，臺灣報紙刊出〈勸吳國楨從速回臺灣〉社論；1954年2月9日，吳國楨在臺灣大各報刊登啟事駁斥謠言：“此次來美，曾經由行政院長陳院長批准，以私人所有[[臺幣|臺幣]]向[[臺灣銀行|臺灣銀行]]購買[[美金|美金]]五千元，作為旅費，此外未由政府或政府中之任何人員批准撥給分文公款，……為國服務三十餘年，平生自愛，未曾貪污，在此國難當頭之際，若尚存心混水摸魚盜取公帑，實際自覺不儕于人類。”並且公開批評政府「一黨統治」，批評[[救國團|救國團]]、情治單位及蔣介石獨裁，並指出臺灣當時政治的六大問題：[[一黨執政|一黨專政]]、[[軍隊|軍隊]]政戰部門、[[特務|特務]]問題、[[人權|人權]]問題、[[言論自由|言論自由]]與[[思想控制|思想控制]]。1954年6月，吳在美國《Look》雜誌，用英文發表《在台灣你們的錢被用來建立一個警察國家》的文章，文中指稱目前台湾已经变成了一个[[警察国家|警察国家]]，“在台湾每年的预算中，美国人提供了30-40亿美元，用来创造一个极权国家”。美國著名報刊《[[紐約時報|紐約時報]]》、《[[芝加哥論壇報|芝加哥論壇報]]》、《[[時代週刊|時代週刊]]》、《[[新聞周刊|新聞周刊]]》等，無不爭相報導。<ref>茅家琦《蔣經國的一生與他的思想演變》，台北，臺灣商務印書館股份有限公司，2003年，第236頁。</ref>並稱「國民黨之經費，非由黨員之捐助，乃係政府，即國民之負擔，這種國庫通黨庫做法，除共產集權國家外，實為今古所無。」<ref>[http://news.ltn.com.tw/news/politics/breakingnews/2451317 他最早說出「國庫通黨庫」 在歷史課本上被KMT消失]，自由時報</ref>

針對吳國楨公開批評執政的國民黨當局，[[立法院|立法院]]長[[張道藩|張道藩]]曾三度向行政院提出三次質詢批評吳國楨，罪名包括“擅離職守，拒辦移交，私自濫發鈔票，拋空糧食；並在外匯、貿易、林產等問題的處理上，非法亂紀，專擅操縱，有意地包庇貪污、營私舞弊等”，列舉吳國楨十三條罪狀；但証據明顯不足，僅用“據說”、“據聞”、“據報”等不確定的字眼。[[陶希聖|陶希聖]]則發表〈兩把刀、殺到底〉一文，口誅筆伐。吳國楨寫了三封信給蔣介石，逐條駁斥對他的誣蔑。不久吳國楨在美刊出〈上總統書〉，批評蔣介石“自私之心較愛國之心為重，且又固步自封，不予任何人以批評建議之機會。”同時，把矛頭直指“太子”，主張將其送入“美國大學或研究院讀書”，否則會妨礙臺灣進步。3月17日，[[國民大會|國民大會]]更通過決議，要求政府除撤免吳國楨[[政務委員|政務委員]]職務並依法究辦，蔣介石總統同日發佈「總統命令」稱：“據行政院呈：本院[[政務委員|政務委員]]吳國楨於去年五月藉病請假赴美，托故不歸。自本年二月以來竟連續散播荒誕謠諑，多方詆毀政府，企圖淆亂國際視聽，破壞反共復國大計，擬請予以撤職處分。另據多方報告，該員前在臺灣省主席任內，多有違法與瀆職之處，自應一併依法查明究辦，請鑒核明令示遵等情。查該吳國楨歷任政府高級官吏，負重要職責者二十八年，乃出國甫及數月，即背叛國家，污蔑政府，妄圖分化[[國軍|國軍]]，離間人民與政府及[[僑胞|僑胞]]與祖國之關係，居心叵測，罪跡顯著，應即將所任行政院政務委員一職予以撤免，以振綱紀，至所報該吳國楨前在臺灣省政府主席任內違法與瀆職情事，並應依法澈查究辦，此令。”<ref>[[劉宜良|劉宜良]]（[[江南|江南]]）《蔣經國傳》，第221頁。</ref>將吳國楨撤職查辦，並開除吳國楨的中國國民黨籍。吳國楨還公開聲明要蔣給他扣留在臺北的兒子[[吳修潢|吳修潢]]發[[護照|護照]]，“如你30天后仍堅持拒發，我將被迫采取其它行動。”蔣介石無奈，命[[外交部|外交部]]補發吳修潢護照。[[吳修潢|吳修潢]]三個月后到了美國。<ref>茅家琦《蔣經國的一生與他的思想演變》，台北，臺灣商務印書館股份有限公司，2003年，第236-237頁。</ref>

吳國楨一連串的動作讓同在美國的[[胡適|胡適]]終於按奈不住。1954年8月3日胡適去信給吳國楨，信中以最嚴厲的字眼譴責吳國楨，批评吴“沒有常识（Common Sense），而且在若干情况下他缺乏道德感（Moral Sense）。”<ref>《胡适致吴国桢》，《胡适全集》卷二十五，第559-560页，安徽教育出版社，2003年。</ref>幾天後（7日）吳國楨回信胡適，吴国桢表示：“我后悔的是我在过去许多次向道德考虑以外的其他影响力屈服。正因为如此，所以我现在决定只根据道德考虑从事，不顾其他。如果我过去犯了错误，那因为我以前太软弱，而我的确现在正努力不再软弱。”，接著兩人在美國大打筆仗，胡甚至痛罵吳是“道德的懦夫”。8月16日，胡適在美国《新领袖》杂志以英文发表文章《台湾有多么自由》，內容提到“巡视今日的台湾，可以发现八九百万中国人在那里正受到最好的管理。”，进一步批驳吴国桢的观点。有人推断胡适在美国批评吴国桢是在作“奉命文章”，“为蒋介石做打手的痕迹已极明显”，當時台灣的自由派人士[[殷海光|殷海光]]反而支持吳國楨。<ref>李敖：《蔣介石評傳》，第459页，北京：中国友谊出版公司2006年版。</ref>1955年，人在台灣的殷海光去信胡适，提到台湾状况，胡适回信说：“台岛情形，我岂不知？”<ref>《胡适致殷海光》，《胡适全集》卷廿五，第626页。</ref>同樣在台湾的[[雷震|雷震]]写信給胡適：“先生说《自由中国》之有言论自由是它这五年争得来的，不料我个人的自由则因是而一天比一天缩减，竟至变成囚犯。”<ref>万丽娟编著《万山不许一溪奔——胡适、雷震来往书信选集》，第68页。</ref><ref>{{Cite web|url=http://pylin.kaishao.idv.tw/?p=1885|title=北投埔林炳炎    » 1949是亟需嚴厲批判的『文化恐怖』符碼|accessdate=2016-12-20|work=pylin.kaishao.idv.tw}}</ref>

== 晚年 ==
“吳國楨事件”告一段落後，吳國楨受聘-{於}-《[[芝加哥論壇報|芝加哥論壇報]]》擔任遠東問題顧問。1966年，吳國楨於美國[[佐治亚州|佐治亚州]][[薩凡納|薩凡納]]{{link-en|阿姆斯特朗大西洋州立大學|Armstrong Atlantic State University}}任東方歷史和哲學教授，直到退休。在美国期间，吳國楨著有各种著作，其中包括详细分析中国神话与早期历史的《中國的傳統》（''Chinese Heritage''）一書。

1984年6月6日，因[[心臟病|心臟病]]病故於美國佐治亚州[[薩凡納|薩凡納]]寓所，終年81歲。死訊傳到臺灣，再次引起極大震盪，《雷聲》、《[[夏潮|夏潮]]》等刊物藉機大做文章。作家[[劉宜良|劉宜良]]生前一度有意撰寫「吳國楨傳」，但是尚未動筆，劉宜良便被國防部情報局局長[[汪希苓|汪希苓]]授意[[竹聯幫|竹聯幫]]成員[[陳啟禮|陳啟禮]]、[[董桂森|董桂森]]、[[吳敦|吳敦]]等人[[暗殺|暗殺]]於舊金山，人稱[[江南案|江南案]]。

== 家庭 ==
1928年，吳國楨在任湖北省财政厅长时，结识[[汉阳铁厂|汉阳铁工厂]]技师长[[黄金涛|黄金涛]]之女，毕业于上海[[晏摩氏女校|晏摩氏女校]]的[[黄卓群|黄卓群]]。1931年在[[汉口|汉口]]结婚。吳國楨与黄卓群育有二女二子：吳修蓉(Eileen)、吳修惠(Edith)、吳修廣(H.K.)和吳修潢(Sherman)<ref name="obit">"Obituaries: Edith Huang Wu"; Savannah Morning News, 2002年8月25日  ({{cite web |url=http://www.lowcountrynow.com/stories/082502/OBITSindex.shtml |title=存档副本 |accessdate=2006-10-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20070311023200/http://www.lowcountrynow.com/stories/082502/OBITSindex.shtml |archivedate=2007-03-11 }})</ref> ，次女吴修惠嫁给[[厉鼎毅|厉鼎毅]]，是著名物理学家。2002年黄卓群去世。 

在担任上海市长时，吳國楨结识[[芝加哥论坛报|芝加哥论坛报]]的[[罗伯特·麦考密克|罗伯特·麦考密克]]夫妇。当上海局势不稳时，吴国桢将两个女儿送到[[美国|美国]][[伊利诺斯州|伊利诺斯州]]的麦考密克家。<ref>{{cite web |url=http://www.cantignypark.com/newsite/mccormickmansion2ndfloorguestbedrooms.htm |title=存档副本 |accessdate=2006-10-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20061018160644/http://www.cantignypark.com/NewSite/mccormickmansion2ndfloorguestbedrooms.htm |archivedate=2006-10-18 }}</ref>
