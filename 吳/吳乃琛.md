'''吴乃琛'''（{{bd|1878年|12月2日|1953年|}}，生於光緒四年十一月初九日），字鹿籌，號賮忱（或書作贐忱），[[浙江省|浙江省]][[嘉興府|嘉興府]][[石門縣_(浙江省)|石門縣]]（即[[崇德縣|崇德縣]]，並入[[桐乡县_(明朝)|桐鄉縣]]）人<ref name="鄉試硃卷">「吳乃琛鄉試硃卷」《清代朱卷集成（303）》，顧廷龍，第59-70頁</ref>。清朝及中华民国财政官员、经济学家。<ref name=wx>文献2007(1-4)，第52页</ref>

== 生平 ==
* 縣學優行廩膳生，民籍，前肄業[[求是書院|求是書院]]、[[南洋公學|南洋公學]]<ref name="鄉試硃卷" />。
光緒二十九年（1903年），吴乃琛中癸卯恩科第81名举人<ref name="鄉試硃卷" />。从[[上海|上海]][[南洋公学|南洋公学]]毕业后，留学美国，先后入[[加利福尼亚大学|加利福尼亚大学]]和[[威斯康辛大学|威斯康辛大学]]经济系，先后获经济学硕士、博士学位。归国后，获授法政科进士，历任[[翰林院|翰林院]]编修、[[币制局|币制局]]会办、[[邮传部|邮传部]]法律编辑员、[[京师大学堂|京师大学堂]]商科学长。1912年任[[北京大学|北京大学]]监督。不久，任[[中国银行|中国银行]]副总裁、北京政府财政部泉币司司长。1913年改署财政部参事。1914年调任财政部赋税司司长。1920年任财政部名誉顾问。<ref name=wx/>

== 家庭<ref>「吳乃琛遺留美國、香港股票、存款分配繼承案(北京市中級人民法院1987年3月判決)」，《中國近年涉外涉港澳案件》，馬守仁編著，1990</ref> ==
其先後娶有五房妻子：大妻倪逸椒、二妻周秉清、三妻曹靜怡、四妻茅詩清、五妻徐韞如。大妻(1928年去世)生有三個子女：吳樓梅、吳卓(又名吳是春)，吳竹修(1944年去世)。吳竹修去世後，留有子女劉皓郢、劉次聽、劉以伸三人。二妻(1936年去世)未生子女。三妻收養吳意靜為女；後生女吳潔，並於1936年與吳乃琛脫離夫妻關係。四妻(1978年去世)生子女吳引林、吳是寅。五妻（2013年去世）生子女吳是龍、吳戌明。

== 其他<ref name="鄉試硃卷" /> ==
* 高高祖妣張氏，是乾隆元年丙辰科進士江南壽州知州[[張大宗|張大宗]]女。
* 十七世嫡堂伯叔祖[[吳之屏|吳之屏]]，天啟二年壬戌科進士，福建巡撫、都察院右僉都御史。
* 十八世從堂伯叔祖[[吳爾纁|吳爾纁]]，崇禎十六年癸未科進士，翰林院庶吉士，贈修撰，諡文愍，崇祀鄉賢，詞著有《（氵茲）蘭堂初集》、《聶許堂遺草》、行世行載府縣志，《明史》附「史可法傳」。
* 再從從堂伯叔祖[[吳于宣|吳于宣]]，乾隆五十二年丁未科進士，刑部郎中，辛酉福建主考，直隸永平、江南揚州等府知府，護理常通海兵備道。行載府縣志。
