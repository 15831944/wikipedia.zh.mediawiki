'''吴涛'''（{{bd|1912年||1983年|11月4日}}）原名'''吴文启'''，男，[[中国蒙古族|中国蒙古族]]人，出生于[[奉天省|奉天省]][[盛京|盛京]]（今[[辽宁省|辽宁省]][[沈阳市|沈阳市]]），[[中国人民解放军开国少将|中国人民解放军开国少将]]。1935年加入[[中国共产党|中国共产党]]，是中共第九、十届中央委员会委员。

==生平==
1935年，吴涛毕业于[[北平中国大学|中国大学]]政法系。同年参加了[[一二·九运动|一二·九运动]]。1935年加入[[中国共产党|中国共产党]]。1937年[[抗日战争|抗日战争]]全面爆发前，历任中共绥西垦区工委组织部部长、宣传部部长<ref name=ml/>。

1937年抗日战争全面爆发后，吴涛历任绥西垦区抗日先锋队中队长、总队组织科科长、参谋长，[[冀热察挺进军|冀热察挺进军]]抗联支队政治部副主任，[[晋察冀军区|晋察冀军区]]10团政治处主任、政委兼中共中心县委书记，晋察冀军区平北军分区政治部副主任、主任、副政委，[[冀察热辽军区|冀察热辽军区]]热西支队政委。参加过[[百团大战|百团大战]]。1939年冬，吴涛出任晋察冀军区冀热察挺进军35团政治处主任。1940年4月，出任[[八路军|八路军]][[129师|129师]]暨鲁西军区（由343旅兼）第1军分区（太西军分区）政治部副主任。1940年10月，出任晋察冀军区冀热察挺进军平北军分区10团政治处主任。1941年12月，出任10团政委。1942年9月，10团改隶晋察冀军区第12军分区，吴涛继续任政委。1945年2月，吴涛出任晋察冀军区冀察军区第12军分区（辖10团、40团）政治部主任。1945年9月，继续任晋察冀军区冀察军区第12军分区（辖10团、40团，新6团、新7团）政治部主任<ref name=ml/>。

[[第二次国共内战|第二次国共内战]]时期，吴涛历任[[晋察冀野战军|晋察冀野战军]]8旅政委、[[佳木斯|佳木斯]]卫戍司令部政委、[[东北民主联军|东北民主联军]]炮兵纵队政治部副主任。1948年1月，东北民主联军炮兵司令部更名为[[东北人民解放军|东北人民解放军]]炮兵司令部。1948年8月15日，以炮兵司令部前方临时指挥机构以及炮兵部队组成了炮兵纵队。吴涛历任炮兵纵队政治部副主任、主任。参加[[辽沈战役|辽沈战役]]。1948年12月，东北人民解放军炮兵司令部和炮兵纵队合编成[[中国人民解放军东北野战军|中国人民解放军东北野战军]]特种兵司令部，吴涛任政治部副主任。1949年4月，中国人民解放军东北野战军特种兵司令部改编成[[中国人民解放军第四野战军|中国人民解放军第四野战军]]特种兵司令部，吴涛继续任政治部副主任<ref name=ml/>。

[[中华人民共和国|中华人民共和国]]成立后，吴涛历任中国人民解放军第四野战军炮兵政治部主任、中央军委直属政治部主任、[[中国人民解放军总参谋部政治部|中国人民解放军总参谋部政治部]]主任。1959年，调任[[内蒙古军区|内蒙古军区]]副政委。1972年，任[[北京军区|北京军区]]副政委兼内蒙古军区第一政委。1978年退出政坛<ref name=ml/>。

1955年，吴涛获授少将军衔（副军级、军直）。获二级[[独立自由勋章|独立自由勋章]]、一级[[解放勋章|解放勋章]]。吴涛是[[第四届全国人大代表|第四届全国人民代表大会代表]]，中国共产党第九、十届中央委员会委员<ref name=ml/><ref>{{Cite web |url = http://www.chnlung.cn/jq/zgjq/2016/0726/232486.html |title =  吴涛将军 文革中周恩来三次出面保护他|author =  |date =2016-07-26  |publisher = 强龙历史网 |language =  |accessdate =  |quote =  }}</ref><ref>{{cite news|title=揭秘：中共首批开国将军的级别是怎样划分的？ |url=http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml |accessdate=2012-06-15 |newspaper=凤凰网（转自摘自《党史博览》2010年第5期） |date=2010-06-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20120620091504/http://msn.news.ifeng.com/history/jishi/detail_2010_06/13/1620383_0.shtml |archivedate=2012-06-20 }}</ref>。 

1983年11月4日，吴涛在北京病逝，享年71岁<ref name=ml>{{Cite web |url = http://www.krzzjn.com/html/39305.html |title = 【晋察冀抗日功臣录】少将名录 吴涛|author =  |date = 2016-10-10 |publisher = 抗日战争纪念网 |language =  |accessdate =  |quote =  }}</ref>。
