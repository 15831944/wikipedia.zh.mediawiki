'''吳宏宇'''{{BD|1996年||2016年|5月23日||}}，[[中國|中國]][[廣東|廣東]][[揭陽|揭陽]]人，[[廣東工業大學|廣東工業大學]]土木與交通工程學院2015级學生。2016年5月23日，吳宏宇和另一名學生在駕駛[[摩托車|摩托車]]追截一名偷車嫌疑人時與嫌疑人的摩托車發生碰撞後倒地受傷，嫌疑人在逃離現場後的隔天被捕，然而吳宏宇最終經搶救無效死亡，其[[見義勇為|見義勇為]]之舉被多家媒體報導，並且獲得讚揚。

==追截偷車嫌疑人==
2016年5月23日下午4時，有人在廣東工業大學四號教學樓下偷竊摩托車，過程中被吳宏宇以及附近另一院校學生謝子哲等多名學生發現，嫌疑人隨即駕車逃跑。吳宏宇等學生主動前往圍堵，在追截過程中，嫌疑人駕駛的摩托車和吳宏宇搭載的摩托車發生碰撞，導致吳宏宇和謝某倒地受傷後被送往醫院急救，當日晚上7時，吳宏宇經搶救無效死亡，得年20歲<ref>[http://m.dayoo.com/125045/125048/201607/04/125048_47466641.htm 吴宏宇:追贼勇士——用生命与社会邪恶力量斗争] {{webarchive|url=https://web.archive.org/web/20161031212847/http://m.dayoo.com/125045/125048/201607/04/125048_47466641.htm |date=2016-10-31 }}.[[廣州日報|廣州日報]]大洋網.2016-07-04</ref>。
5月24日凌晨3時許，盜車嫌疑人在[[番禺區|番禺區]]南村鎮被抓獲<ref name="A1">[http://www.chinanews.com/sh/2016/05-27/7885791.shtml 广工大学生追截偷车贼被撞身亡：获见义勇为奖] {{webarchive|url=https://web.archive.org/web/20160912160122/http://www.chinanews.com/sh/2016/05-27/7885791.shtml |date=2016-09-12 }} .中國新聞網.2016-05-27</ref>。

==評價及榮譽==
吳宏宇的室友汪健評價吳宏宇是一個陽光開朗，樂於助人和朋友多的人，他又說吳宏宇絕不僅僅是為了一輛摩托車才去追小偷，被偷的摩托車並非吳宏宇本人的個人財產：「他身上有[[正義|正義]]感，看到這種事情肯定會出手的。」<ref name= "A2"/>

廣東工業大學校方在5月24日中午發布通報，稱吳宏宇是一名平時為人熱心，樂於幫助同學，有正義感，深受老師和同學們喜愛的好學生，表示“吳宏宇同學見義勇為的行為值得肯定”，校方亦於當日向相關部門為吳宏宇申報正式的見義勇為榮譽稱號<ref>[http://ep.ycwb.com/epaper/ycwb/html/2016-05/25/content_47350.htm 大学生飞车追贼 撞车后不幸身亡] {{webarchive|url=https://web.archive.org/web/20161031155834/http://ep.ycwb.com/epaper/ycwb/html/2016-05/25/content_47350.htm |date=2016-10-31 }}.[[羊城晚報|羊城晚報]].2016-05-25</ref>。

吳宏宇的父親在接受媒體採訪時表示：“他為保護人民財產付出了生命，我真的是感到又驕傲，又可惜。他本來應該是做好保護措施再去的，但當時情況太突然了，他為了保護財產，想都沒想就衝上去了。”他說，希望今後任何有正義感的人，在做見義勇為的事情之前，要先保護好自己的安全，理智地去做好事。<ref name = "A1"/>

2016年5月27日，[[廣州市|廣州市]]綜治辦副主任、市見義勇為基金會秘書長李作春慰問吳宏宇的父母，並頒發“廣州市見義勇為人員”證書及慰問金30萬元<ref name = "A1"/>。6月6日，廣州市見義勇為評定委員會發出公示，確認吳宏宇為見義勇為人員<ref>[http://www.gzjd.gov.cn/gzjdw/xxgk/xxgkml/tzgg/20160606/detail-328555.shtml 确认吴宏宇、谢子哲、欧志贤同志见义勇为的公示] {{webarchive|url=https://web.archive.org/web/20180408140028/http://www.gzjd.gov.cn/gzjdw/xxgk/xxgkml/tzgg/20160606/detail-328555.shtml |date=2018-04-08 }}.廣州市公安局.2016-06-06</ref>，在此前亦被授予了“廣東青年五四獎章”、“廣州市見義勇為人員”、“廣東工業大學見義勇為英雄大學生”以及“廣東工業大學青年五四獎章”稱號<ref name= "A2"/>。

==游戏中的纪念==
吳宏宇生前曾經在[[微信|微信]]朋友圈上留言表示很期待一款稱為《[[守望先锋|-{zh-hant:鬥陣特攻;zh-hans:守望先锋}-]]》的[[多人電子遊戲|多人]][[第一人稱射擊遊戲|第一人稱射擊遊戲]]，這遊戲的發售日期正是他逝世後的隔天，為了紀念他見義勇為的行為，該遊戲的發行商[[暴雪娛樂|暴雪娛樂]]在遊戲中的一個名為「漓江塔」的地圖裡为他增添了一個特别设计。在漓江塔紅方，進樓門左拐可以看到三個穿著[[太空衣|太空衣]]的人物模型，中間那件太空衣胸口處寫著吳宏宇的名字「宏宇」，腳下有一圈白花，而背後的紅牆上寫著[[漢字|漢字]]——「英雄不朽」。<ref name="A2">[http://games.sina.com.cn/o/z/overwatch/2016-07-11/fxtwihq0008155.shtml 新华社：游戏中“重生”的青年 为何赢得网友点赞？] {{webarchive|url=https://web.archive.org/web/20160712130610/http://games.sina.com.cn/o/z/overwatch/2016-07-11/fxtwihq0008155.shtml |date=2016-07-12 }}.新浪游戏.2016-07-13</ref><ref>[http://www.wxrw123.com/qy/20160706/1552714.html 《斗阵特攻》小彩蛋纪念中国玩家的勇敢事迹] {{webarchive|url=https://web.archive.org/web/20161031153114/http://www.wxrw123.com/qy/20160706/1552714.html |date=2016-10-31 }}.微信熱文.2016-07-06</ref><ref>[http://games.ifeng.com/a/20160707/41635140_0.shtml 吴宏宇被暴雪游戏收录 父亲说：还了儿子心愿] {{webarchive|url=https://web.archive.org/web/20161103215335/http://games.ifeng.com/a/20160707/41635140_0.shtml |date=2016-11-03 }}.[[南方都市报|南方都市报]].2016-07-07</ref>
