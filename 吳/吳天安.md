'''吳天安'''牧師 <!--（{{bd|1949年||||catIdx=N}}）-->（{{lang|en|Rev. Stephen Tin-On NG}}），[[中華基督教會|中華基督教會]]香港教區牧師，曾任[[香港中文大學|香港中文大學]][[崇基學院|崇基學院]]校牧（1983-1990），是崇基學院「大學講章」的創辦人。

==簡歷==

吳天安1952年入讀香港聖樂院（今為[[香港音樂專科學校|香港音樂專科學校]]），1954年轉修神學<!--，並於1970年畢業-->。1974年畢業於[[香港中文大學崇基學院神學院|崇基神學組]]。1978年遠赴英國[[牛津大學|牛津大學]]神學研究所深造神學，1983年畢業。

1983年，吳於牛津大學神學研究所畢業回香港，由[[中華基督教會|中華基督教會]]借調到[[香港中文大學|香港中文大學]][[崇基學院|崇基學院]]出任校牧一職。任內推動校牧室搬遷事工，令校牧室由眾志堂遷回崇基禮拜堂，加強主日崇拜及週會等事工。吳天安是[[香港中文大學|香港中文大學]][[崇基學院|崇基學院]]「大學講章」的創辦人，[[崇基學院|崇基學院]]仿傚英國[[牛津大學|牛津]]、[[劍橋大學|劍橋]]等校的制度，於主日崇拜推行中英雙語並用的「大學講章（University Sermon）」，邀請不同學科的教授講道，將學術、神學與信仰共冶一爐。<ref>{{ cite web |url=https://weproclaimhim.com/?p=8799 |title=建立心靈健康的校園：以香港中文大學崇基學院為例 |date=2017-11-13 |publisher= |author=龔立人}}</ref><ref>{{ cite book |url= |title=吳天安牧師講章集 |date=2008 |publisher=基督教文藝 |author=吳天安|location=香港 | pages=  |ISBN=9789881774811 }} </ref> 1989年起，吳氏在禮拜堂開始施行洗禮，共有8位領洗，由於崇基禮拜堂並非獨立堂會，領洗者歸入[[中華基督教會|中華基督教會]]堂會。<ref name="cc65">{{cite book |url=https://www.ccc.cuhk.edu.hk/images/ChungChiPublication/ChungChi_65Book.pdf |title=善彌雅集-崇基學院六十五週年校慶特刊 |page=53|publisher=崇基學院}}</ref><ref name="chaplaincy">{{ cite web |url=https://www.cco.cuhk.edu.hk/chaplaincy/chi/1983-1990-10/ |title=吳天安牧師 |date= |publisher= |author= }}</ref>

吳氏身兼[[中華基督教會|中華基督教會]]香港教區牧師及[[香港音樂專科學校|香港音樂專科學校]]校監（1977-1978，1986-2011）<ref>{{ cite web |url=http://www.hkmi.net/chairman.php |title=歷任校監及校長 |date= |publisher=香港音樂專科學校 |author= }}</ref> ，與各界教會人士及聲樂家多有聯繫，能順利邀請眾多聲樂家在禮拜堂獻唱。1984年至1990年間的「大學講章」共邀得牧長及學者共49位參與，講者名單及講題亦於《基督教週報》刊登。<ref name="cc65" /><ref name="chaplaincy" />

至1990年8月離任，吳氏擔任崇基校牧7年，是當時任期最長的崇基校牧 （這紀錄後來被[[伍渭文|伍渭文]]牧師打破） 。離任後吳氏返回中華基督教會區會及堂會繼續牧會工作。

== 個人生活 ==
吳天安夫人是周德恩女士，吳牧師夫婦均富音樂造詣。周德恩於2016年2月11日於香港逝世。<ref>{{ cite web |url=http://www.hkcccc.org/News/viewNews.php?aid=1101 |title=吳周德恩師母主懷安息 |date=2016-02-16 |publisher=中華基督教會香港區會 |author= }}</ref><ref>{{ cite web |url=https://www.cco.cuhk.edu.hk/chaplaincy/uploads/ServiceBulletin/ServiceBulletin_20160221.pdf |title=主日崇拜報告事項  |date=2016-02-21 |publisher=崇基學院禮拜堂 |author= }}</ref>
