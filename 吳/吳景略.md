'''吳景略'''（1907年2月5日-1987年8月16日<ref name="宗師">{{Cite web |url= http://info.guqu.net/guqindashi/20060729224240_555.html|title=“琴坛一代宗师”吴景略|accessdate=2014-06-26 |work= |publisher=中國古曲網|date=2006-07-29}}</ref>），名'''韜'''，别號'''縵叟'''，是中國著名[[古琴|古琴]]演奏家，創立了“[[虞山_(常熟)|虞山]]吳派”，有“琴壇一代宗師”之稱<ref name="宗師"/>。亦對古琴製作頗有研究，是現代所用的鋼絲尼龍弦的發明者之一。

==生平==
吳景略祖籍[[江蘇|江蘇]][[常熟|常熟]]，他亦出生在常熟[[西塘|西塘]]。少時從[[赵剑侯|赵剑侯]]、[[周少梅|周少梅]]、[[吴梦非|吴梦非]]等學習音律，也曾從[[蕭退庵|蕭退庵]]、[[樊少雲|樊少雲]]學習書畫。1930年吳景略開始師從[[王端璞|王端璞]]學琴，三月閒共學《[[湘江怨|湘江怨]]》和《[[醉渔唱晚|醉渔唱晚]]》二首。据他的學生吳寧回憶，吳景略曾説：“王先生是票友...他没有好的琴谱，他弹琴也没有节奏感。我与王先生只学了三个多月，以后他去吴江县任官，我就没有老师了”。此後他基本靠自學，某次在上海的古籍书店找琴谱時，從书店的夥計処得知有一名為徐少峰的医生懂得弹琴。吳景略前去拜訪，因後者行醫出門不在家而折返，但又向其寄出信件求學。徐少峰爲他所感動，因請上海琴人李明德前往常熟教他彈琴。但吳僅與李明德學過一首曲子，即《[[梅花三弄|梅花三弄]]》。按吳景略所述，原因是他衹上了一節課就基本摸清全曲，李明德稱“你根本不需要老师，你是无师自通的琴人、天才”，而後將他介紹給了[[查阜西|查阜西]]和[[李子昭|李子昭]]，查阜西又請他加入了[[今虞琴社|今虞琴社]]。他在[[抗日戰爭|抗日戰爭]]期間主持琴社事務。1950年代初在[[上海|上海]]創立“策聲琴韵室”，在上海、常熟教授古琴。<ref name="宗師"/><ref name="誕辰">{{Cite web |url= http://iom.ccom.edu.cn/cbm/html/xueshuxinwen/2013/1017/4330.html|title=“国际古琴音乐文化周——暨纪念古琴大师吴景略诞辰一百周年”即将举办|accessdate=2014-06-26 |work= |publisher=中央音乐学院网站 |date=2013-01-17}}</ref><ref name="yiguren">{{Cite web |url= http://rufodao.qq.com/a/20151117/034210.htm|title=怀念我的古琴老师吴景略先生|accessdate=2016-10-26 |author=吴宁 |publisher=騰訊儒學，轉自七弦古琴网 |date=2015-11-17}}</ref>

1953年吳景略受聘到[[中央音樂學院|中央音樂學院]]音樂研究所擔任通訊研究員，三年後成爲中央音樂學院教授、亦在天津的中央音樂學院民樂系彈撥教研室擔任主任一職。1950年代他曾改良過今虞琴絃<ref name="誕辰"/>。[[文化大革命|文化大革命]]爆發後被迫離開學校，住在上海的女兒家中<ref>{{Cite web |url= http://xmwb.xinmin.cn/xmwb/html/2013-05/03/content_28_6.htm|title=吳景略軼事|accessdate=2014-06-26 |work= |publisher=新民晚報|date=2013-05-03}}</ref>。這段時期與今虞琴社成員胡維禮是鄰居，吳景略曾教導他彈琴。據胡回憶，吳景略可以做到叼着煙彈琴而至曲終菸灰不墜<ref name="誕辰"/>。但同時因生活所迫而賣掉了許多藏琴<ref name="yiguren"/>。

文化大革命結束後回到中央音樂學院，併成爲終身教授。後來亦曾擔任中国文学艺术联合会委员、[[中国音乐家协会|中国音乐家协会]]民族音乐委员会委员、[[北京古琴研究會|北京古琴研究會]]會長。併在1970年代發明出了鋼絲尼龍琴絃<ref name="誕辰"/>。1987年8月16日因[[肺癌|肺癌]]在北京逝世，其子[[吳文光_(音樂家)|吳文光]]後來亦成爲古琴演奏家<ref name="宗師"/><ref name="yiguren"/>。
