'''吳旭智'''，[[民國黨|民國黨]]現任資訊長，現任第19屆[[新竹縣議員|新竹縣議員]]、民國黨第1任代理黨主席，曾參選臺北市立法委員落敗，同時也是時任黨籍新竹縣長候選人[[徐欣瑩|徐欣瑩]]交通大學後援會執行長<ref>{{cite news |title =交大人站出來！ 徐欣瑩後援會成軍 挺工程師從政 |url=http://news.ltn.com.tw/news/politics/breakingnews/2577043}}</ref>。曾旅居美國[[麻省理工學院|麻省理工學院]]深造，專長於電腦輔助語言學習(Computer Aided Language Learning)領域，針對對等互動對話系統進行研究與提供概念。<ref>{{cite news |title =對等互動語言學習對話系統 |url=http://newsletter.teldap.tw/news/read_news.php?nid=2295}}</ref>

== 個人背景 ==
吳旭智的爸爸小學沒有畢業、媽媽小學畢業，靠著擺路邊攤賣成衣養家。為了生活一家人搬過十餘次家。父母做生意的理念是，寧願自己吃虧也不要讓廠商、客戶虧錢。經常教育子女：「吃虧就是占便宜。」雖然父母學歷不高，但所堅持的人生哲學'''「誠信、奉獻、共好」'''的精神，讓鄉里許多人喜歡找他們排解問題，深受鄉民信賴。受到父母家庭教育的影響，在建立自己的家庭後對孩子的教育更為重視，也認為竹北其實可以有更好的教育環境。2018年，吳旭智的外公、外婆結婚超過69年，榮獲新竹縣政府重陽敬老表揚大會「鑽石婚」表揚。

吳旭智於民國92年取得交通大學資訊科學研究所取得資訊科學博士學位。就讀博士班時，執行專案計畫建立國內大型金融機構視訊安全監控系統，至今已委定運作十餘年，範圍遍布全台灣。博士論文做的是公益與私利的衝突研究，透過AI模型的模擬分析，得到的結論是：'''每個人都希望獲得自己的最大利益，而當每個人都願意花心思在公共利益的時候，反而每個人都能得到最大的個人利益'''。博士畢業後，仍經常受邀至校園及企業為學生與業界人士舉行公益講座，距今已累計達數百場。從學生時期便熱衷投入志工服務，迄今已達二十餘年，未來仍會以相同的志工精神繼續替更多人服務。任職工研院期間，獲知名出版社提供數萬本書籍的版權，透過技術轉換成有聲書，開發出'''「視障者有聲書」'''，讓視障朋友們可以從雲端下載收聽。其後亦協助非營利組織建置全球視訊連線教學系統，實現打破國界、洲際的藩籬。

2018年3月18日吳旭智於風傳媒上發表社論「『水』來救救我—大新竹水汙染議題如何解？」關心新竹縣水汙染的問題。

2017年7月31日吳旭智於風傳媒上發表社論認為"台灣金融科技已經落後？用人工智慧＋ATM殺出一條血路！"<ref>[http://www.storm.mg/article/305982 觀點投書：台灣金融科技已經落後？用人工智慧＋ATM殺出一條血路！"2017-07-31 風傳媒"]</ref>

2015年8月9日，出席「年代高峰會」談IT(Information Technology)與DT(Data Technology)的轉變，從利己的角度到利他的角度，透過共享、公開、透明的方式來推動社會的進步。

2006年至2009年間，吳旭智在《[[電腦與通訊|電腦與通訊]]》期刊發表過多篇專題，以語言系統系統為主。<ref>利用對話回合間之內文關係來降低辨識錯誤率 吳旭智 ; 李青憲 電腦與通訊 120 2007.06[民96.06 頁65-70]</ref><ref>電腦輔助華語文學習對話系統之辨識錯誤率降低方法 吳旭智 電腦與通訊 124 2008.06[民97.06] 頁57-62</ref><ref>Mi-DJ--自動產生個人化語音節目的方法 吳旭智 ; 李青憲 電腦與通訊 128 2009.06[民98.06] 頁70-75</ref>

== 從政經歷 ==


2015年9月10日，民國黨徵召吳旭智參選台北市第六選舉區立法委員，僅獲不足六千票而落敗。

2018年1月[[民國黨|民國黨]]主席[[徐欣瑩|徐欣瑩]]表示，目前有黨員、具科技背景的吳旭智將挑戰竹北選區，也傳出民國黨將在竹北提名2席以增加席次。<ref>
[http://www.chinatimes.com/newspapers/20180123000580-260118 '搶攻竹縣議員 藍綠都推新血 中國時報-政治要聞 2018-01-23']</ref>

2018年8月，獲民國黨提名參選新竹縣(竹北)縣議員。「麻省理工科學家吳旭智 代表民國黨出征新竹縣議員」

</ref>

2018年8月，正式登記參選新竹縣第19屆議員竹北選區選舉，化身寶可夢訓練師「小智」帶著皮卡丘與寶貝球，擊出三大政見主軸「爭取教育資源、升級社會福利、友善智慧城市」。「美國麻省理工科學家吳旭智 投身竹縣議員選戰」

</ref>

2018年10月，吳旭智競選總部正式成立，徐欣瑩到場力挺吳旭智，盼議會出現第二位博士議員<ref>
[https://www.storm.mg/localarticle/532213'徐欣瑩力挺吳旭智　望竹縣第二位博士候選人搶灘議會殿堂 2018-10-08']

2018年11月29日，吳旭智以3,987票，得票率為4.91%，當選新竹縣議會第19屆議員(竹北選區)，並兼任民國黨第1任代理黨主席。12月25日宣誓就職至今。


== 幼兒教育 ==
吳旭智主張竹北市應設立'''特色公園'''，一座高出生率的城市需要有足夠的親子遊憩空間，讓每戶家庭可以有假日的好去處。此外提升公園兒童遊具的品質也十分重要，設立特色遊具不僅有助於孩童感覺統合的發展，也有助於親子感情的加溫。

此外，吳旭智也相當重視孩子閱讀的重要性，新竹縣各鄉鎮圖書館尚無法實現通借通還，無法有效打破館際藩界，讓全縣的圖書資源可以共享。因此主張'''新竹縣圖書館應增設通借通還'''的功能之外，亦應建立友善的圖書資訊使用app，便利民眾使用。

2018年4月起，吳旭智受邀擔任「有智一同未來工作室」志工講師推動資訊教育，'''透過桌遊培養程式設計中運算思維的邏輯'''。

2017年7月21日，吳旭智投書風傳媒，針對於台灣的育兒百寶箱無法帶給父母幸福感，認為政府應該針對根源解決問題，<ref>[http://www.storm.mg/article/301815  觀點投書：芬蘭美談「育兒百寶箱」，為何在台灣被罵翻？" 吳旭智 2017-07-21 風傳媒"]</ref>2017年12月衛福部決定育兒百寶箱政策暫緩執行．

== 社會公益 ==
從學生時代就投入志工行列，一直以來不斷思考的是'''「如何才能從根本改善弱勢者的生活處境，並保障其權益？」'''大學時參加幼幼社，經常到孤兒院或青少年輔導中心擔任義工。博士畢業後，出任自閉症權益促進會理事，關心星兒的權益。十分關心青少年朋友，經常受邀至各大專院校、企業或人民團體擔任公益講座師資，至今已達數百場。新竹縣慈善愛心功德會會員，經常與理事長送愛心物資至新竹偏鄉幫助生活困難的鄉親。赴竹北社區發展協會擔任志工，陪伴長者快樂學習。與台灣向陽關懷協會共同舉辦「關懷守護愛無限 關懷失智症」講座，關懷失智症患者及其家屬。
<ref>
[https://www.chinatimes.com/realtimenews/20180724002662-260405 關懷守護愛無限 失智症宣導講座竹北開跑 2018-07-24']
</ref>

2017年12月11日，吳旭智提出政府應針對於長照2.0，可配合加上「[[時間銀行|時間銀行]](Time Bank)」，邀請社會各熱心志工團隊來服務，增加長期照護效果，減少看護與照護員負擔過大問題，讓台灣變成真正的「功德之島」．<ref>[http://www.storm.mg/article/369729 觀點投書：引入「時間銀行」，讓台灣變成真正的「功德之島」 "2017-12-11 風傳媒" ]</ref>
