'''吳珈慶'''{{BD|1989年|2月9日|||W吴}}，出身[[臺灣|臺灣]][[新北市|新北市]][[泰山區_(新北市)|泰山區]]，現落戶於[[廣東省|廣東省]][[深圳市|深圳市]]，-{zh-hans:[[台球|台球]];zh-hk:[[桌球|桌球]];zh-tw:[[撞球|撞球]];}-运动员，現已取得[[中華人民共和國居民身份證|中華人民共和國居民身份證]]。綽號「泰山神童」，[[中國文化大學|中國文化大學]]體育學系肄業，2005年世界花式-{zh-hans:[[台球|台球]];zh-hk:[[桌球|桌球]];zh-tw:[[撞球|撞球]];}-錦標賽及[[世界八號球錦標賽|-{zh-hant:世界八號球錦標賽;zh-cn:世界8球锦标赛}-]]雙料冠軍。和史上最年輕世界冠軍吳珈慶同為國手的柯秉逸、吳育綸，被譽為「撞壇三小龍」。

他是第一個入籍中國大陸的台灣運動員，也是[[中華民國|中華民國]]史上，第一個事實上具備中華人民共和國國籍而入[[中華民國國軍|中華民國國軍]]服役的軍人。

==生平==
6歲時在舅舅開設的-{zh-hans:台球;zh-hk:桌球;zh-tw:撞球;}-館接觸-{zh-hans:台球;zh-hk:桌球;zh-tw:撞球;}-，9歲開始正式打球，曾在2005年拿下[[新加坡|新加坡]]站亞軍及[[雅加達|雅加達]]站季軍，在同年7月10日以17比16逆轉打敗同胞選手[[郭柏成|郭柏成]]，成為-{zh-hans:台球;zh-hk:桌球;zh-tw:撞球;}-史上最年輕（16歲又5個月）的世界-{zh-hans:[[花式撞球|花式撞球]];zh-hk:[[花式桌球|花式桌球]];zh-tw:[[花式撞球|花式撞球]];}-錦標賽冠軍得主。2005年11月14日第二屆[[世界八號球錦標賽|-{zh-hant:世界八號球錦標賽;zh-cn:世界8球锦标赛}-]]在[[阿拉伯聯合大公國|阿拉伯聯合大公國]]舉辦，吳珈慶在決賽中以11：3擊敗[[荷蘭|荷蘭]]選手[[尼克·范登博格|范登博格]]，再度奪冠，成為史上第一位在同一年度獲得-{zh-hant:八號球;zh-cn:8球}-及-{zh-hant:九號球;zh-cn:9球}-雙料冠軍的-{zh-hans:台球;zh-hk:桌球;zh-tw:撞球;}-員。2005年12月27日在[[江蘇省|江蘇省]][[蘇州市|蘇州市]][[昆山市|昆山市]]舉行的兩岸花式-{zh-hans:台球;zh-hk:桌球;zh-tw:撞球;}-交流賽當中擊敗台灣選手[[張裴瑋|張裴瑋]]，獲得冠軍。

2011年4月20日，中華撞協表示，大陸當局向他們通報吳珈慶報名[[北京|北京]]公開賽，並且於2011年4月8日，取得[[中華人民共和國居民身份證|中華人民共和國居民身份證]]，落戶[[廣東省|廣東省]][[深圳市|深圳市]]，成為[[台灣史|台灣史]]上首位轉入[[中國大陸|中國大陸]]體壇的運動員<ref>{{cite web |url=http://news.chinatimes.com/sports/130512/132011042001522.html |language=zh-tw |title=吳珈慶移籍中國 體委會惋惜 |publisher=[[中央社|中央社]] |date=2011-04-20 |accessdate=2011-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20110511154800/http://news.chinatimes.com/sports/130512/132011042001522.html |archivedate=2011-05-11 }}</ref>。

因為台灣法律規定，2012年1月14日，吳珈慶以[[大陸人民回台探親證|大陸人民回台探親證]]入境台灣，至[[中華民國空軍|中華民國空軍]]服[[兵役|兵役]]，成為中華民國史上，第一個事實上具備中華人民共和國國籍而入伍服役的軍人。

之後，傳出罹患慢性[[白血病|白血病]]，以重大疾病為理由提前退伍<ref>{{cite web |url=http://tw.news.yahoo.com/%E5%90%B3%E7%8F%88%E6%85%B6%E5%82%B3%E7%BD%B9%E8%A1%80%E7%99%8C-%E6%8F%90%E5%89%8D%E9%80%80%E5%BD%B9-202412974--spt.html?bcmt_s=m |language=zh-tw |title=吳珈慶傳罹血癌 提前退役 |publisher=[[自由時報|自由時報]] |date=2013-01-06 |accessdate=2013-01-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20160317014758/https://tw.news.yahoo.com/%E5%90%B3%E7%8F%88%E6%85%B6%E5%82%B3%E7%BD%B9%E8%A1%80%E7%99%8C-%E6%8F%90%E5%89%8D%E9%80%80%E5%BD%B9-202412974--spt.html?bcmt_s=m |archivedate=2016-03-17 }}</ref>。

==國籍、戶籍及服役爭議==
《[[中華民國憲法|中華民國憲法]]》第二十條規定：「人民有依法律服兵役之義務。」吳珈慶未服畢[[中華民國自由地區|中華民國自由地區]]成年男子須服的[[義務役|義務兵役]]，即在[[大陸地區|大陸地區]]設有戶籍，依[[中華民國|中華民國]]《[[臺灣地區與大陸地區人民關係條例|臺灣地區與大陸地區人民關係條例]]》第9條之1之規定，喪失[[臺灣地區|臺灣地區]]人民身分；但根據《中華民國國籍法》第十二條第一項規定：「男子年滿十五歲，未服兵役者，不得喪失國籍。」，故依其法理，吳珈慶為曾具中華民國臺灣地區人民身分之現中華民國[[大陸地區|大陸地區]]人民。至於吳之服兵役義務，則依[[中華民國|中華民國]]《[[臺灣地區與大陸地區人民關係條例|臺灣地區與大陸地區人民關係條例]]施行細則》第十一條之規定，不因而免除。[[中華民國內政部|中華民國內政部]]役政署表示，吳依法仍須服役，2012年1月14日前必須返台入伍，否則將被列為[[通緝|通緝]]犯。<ref>{{cite web |url=http://tw.news.yahoo.com/article/url/d/a/110426/142/2qgt5.html |language=zh-tw |title=尚未服役 吳珈慶若不歸 恐通緝 |publisher=[[東森|東森]]新聞 |date=2011-04-26 |accessdate=2011-04-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20110430094553/http://tw.news.yahoo.com/article/url/d/a/110426/142/2qgt5.html |archivedate=2011-04-30 }}</ref>

{{or|《[[中華人民共和國國籍法|中華人民共和國國籍法]]》第三條規定：「[[中華人民共和國|中華人民共和國]]不承認[[中國公民|中國公民]]有[[雙重國籍|雙重國籍]]。」[[中華人民共和國|中華人民共和國]]現政府的立場並不承認目前有效管領[[臺灣|臺灣]]、[[澎湖|澎湖]]、[[金門|金門]]、[[馬祖|馬祖]]、[[東沙群島|東沙群島]]、[[太平島|太平島]]、[[中洲島|中洲島]]及其附屬島嶼之[[中華民國|中華民國]]政府，又認為[[中華人民共和國政府|中華人民共和國政府]]自[[中華人民共和國國慶節|1949年10月1日]]建立以來，已繼承原[[中華民國|中華民國]]政府對於全中國之一切合法權利及義務，原[[中華民國|中華民國]]政府業已終止存在。基於上述官方立場，吳珈慶依據《[[中華民國憲法增修條文|中華民國憲法增修條文]]》及[[中華民國|中華民國]]《[[臺灣地區與大陸地區人民關係條例|臺灣地區與大陸地區人民關係條例]]》所具有之[[中華民國國民|中華民國國民]]（[[大陸地區|大陸地區]]人民）身分與其[[中華人民共和國公民|中華人民共和國公民]]身分並無衝突，故無違背《[[中華人民共和國國籍法|中華人民共和國國籍法]]》之疑慮。}}

==比賽成績==
*2018年5月20日，美式台球雙打世界杯在上海盧灣體育館落幕，劉海濤、吳珈慶組成的中國隊在決賽中以10比3大勝衛冕冠軍奧地利隊奪冠。
*2016年9月28日，亞洲全台球錦標賽男子9球雙打決賽，吳珈慶、韓皓翔組合11-4戰勝沙特阿拉伯隊奪冠。
