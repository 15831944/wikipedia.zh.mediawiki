'''吳幾康'''（{{bd|1918年|1月9日|2002年|7月21日}}<ref name = '快報'/>），原名'''吳畿康'''，安徽歙县（今[[安徽省|安徽省]][[黄山市|黄山市]][[歙县|歙县]]）人，[[中華人民共和國|中華人民共和國]]計算機專家。<ref name = '中國之最'/>

== 生平 ==
吳幾康1943年於[[同濟大學#1937-1949年|四川同濟大學]]機電系畢業，後在上海同濟大學無線電系執教。<ref name = '大辭典'/> 1949年吳幾康來到[[丹麥技術大學|丹麥工業大學]]進修，1951年在[[哥本哈根|哥本哈根]]擔任無線電廠發展工程師。<ref name = '快報'/> 鉴於中華人民共和國於1949年成立，吳幾康決定回國參加工作。但當時正值[[抗美援朝|抗美援朝]]期間，吳幾康只能繞道回中國，途經瑞典、芬蘭和蘇聯的西伯利亞，終於在1953年回到中國。<ref name = '回鄉'/>

1953年回到中國之後，吳幾康擔任[[中國科學院數學研究所|中國科學院數學研究所]]、原子能研究所計算機小組副研究員。<ref name = '大辭典'/> 1956年，吳幾康任中國科學院計算技術研究所籌備委員會委員，和[[華羅庚|華羅庚]]、[[夏培肅|夏培肅]]、[[張效祥|張效祥]]等人一起籌建[[中國科學院計算技術研究所|中國科學院計算技術研究所]]（簡稱中科院計算所）。<ref name = '歷史'/> 同年9月，中國成立赴蘇聯計算技術考察團，吳幾康為其中一名團員，對蘇聯計算技術的科研、生產與教育進行學習。<ref name = 'IT'/>

吳幾康是中國第一台大型通用數字電子計算機104機的主要研究人員之一<ref name = '中國之最'/> ；104機起先由張效祥和吳幾康共同負責，後來吳幾康改任119機負責人。<ref name = '104機'/> 1964年，吳幾康負責的119機研制成功，是當時歷史上最快的電子管計算機。<ref name = 'csdn'/>

1972年吳幾康擔任陝西微電子學研究所副所長，1979年任中科院計算所副所長。2002年7月21日，吳幾康逝世，享年84歲。<ref name = '快報'/>
