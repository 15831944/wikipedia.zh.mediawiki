'''北京吴裕泰茶业股份有限公司'''，总部位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[交道口东大街|交道口东大街]]4-17号，是[[北京老字號|北京老字號]]茶业企业。

==简介==
[[清朝|清朝]][[光绪|光绪]]十三年（1887年），吴锡卿（祖籍[[安徽|安徽]][[歙县|歙县]]昌溪村，1930年逝世）在北京创办吴裕泰茶栈。吴氏祖上数辈都从事茶叶生意，吴家有兄弟6个，吴锡卿在其中排行第四，总管吴裕泰茶栈，先后在北京城内外开出8家茶庄，后增加到11家，分别是：[[崇文门|崇文门]]内的“乾泰聚”、“福盛”、“吴鼎裕”茶庄（“吴鼎裕”在后来的[[崇文门菜市场|崇文门菜市场]]旧址处，专营高档茶叶），[[广安门|广安门]]内的“协利”，[[西单北大街|西单北大街]]的“吴新昌”，[[东单|东单]]的“信大”，[[清河|清河]]的“吴德利”，[[通州区|通县]]及[[天津|天津]]的“乾泰聚”分号、“裕胜”茶庄等等。吴锡卿曾经担任京师茶行会会长<ref name=bjlyw>{{Cite web |url = http://www.bjlyw.com/bj/yule/meishi/laozihao/200509/12558.html |title = 百年老店吴裕泰茶庄 |author =  |date = 2005-09-03 |publisher =北京旅游网  |language =  |accessdate =  |quote =  }}</ref>。

吴裕泰茶栈以拼配茉莉花茶为特色，起初是自己窨制，后来全部在产地窨制。茉莉花茶在北京流行是因当时北京百姓喝不起名茶，吴裕泰拼配的茉莉花茶走大众化路线，所以生意兴隆<ref name=bjlyw/>。

吴裕泰茶栈自创建之日起便坐落于今[[东四北大街|东四北大街]]44号。旧时占地面积20余亩，大门可同时进出三辆大车，共8个院落100余间房屋，是花园式建筑，分成库房、客房、加工拼配室、营业店堂等等。当时主要自[[安徽|安徽]]、[[福建|福建]]、[[浙江|浙江]]等地进茶叶，经[[大运河|大运河]]到达通县，用大车运进[[东直门|东直门]]到[[北新桥|北新桥]]。当时吴裕泰茶栈的电话为：北局，1928。21世纪初，王府井吴裕泰茶文化馆展出有一位老顾客捐赠的吴裕泰茶栈茶叶筒<ref name=bjlyw/>。

吴家信佛，每年冬三月都搭粥棚接济穷人。吴家数代为安徽儒商，经商的同时学文。吴裕泰最初的牌匾是吴锡卿用五块银圆请[[祝椿年|祝椿年]]题写的“吴裕泰茶栈”。这块匾一直挂到1955年底[[公私合营|公私合营]]时，吴裕泰茶栈改为“吴裕泰茶庄”。“吴裕泰茶庄”没有牌匾。[[文革|文革]]初期，东四北大街被改为“红日路”，吴裕泰茶庄也更名为“红日茶店”，并挂有写着“红日茶店”的竖牌。1985年，恢复老字号“吴裕泰茶庄”，牌匾请[[冯亦吾|冯亦吾]]题写，为黑地金字横匾，沿用至今<ref name=bjlyw/>。

1994年，东四北大街的吴裕泰茶庄翻扩建，营业面积由50平方米增至80余平方米，店内悬挂天津炎黄画院赠的大幅[[国画|国画]]《[[陆羽|陆羽]]品茶图》。1995年，在茶庄隔壁建成“吴裕泰茶社”，[[刘炳森|刘炳森]]题写了“吴裕泰茶社”匾额。1995年秋，吴裕泰茶社开笔会，刘炳森、[[胡絜青|胡絜青]]、[[李滨声|李滨声]]、[[苏适_(书法家)|苏适]]、[[米南阳|米南阳]]题写书法<ref name=bjlyw/>。

1995年，[[中华人民共和国国内贸易部|中华人民共和国国内贸易部]]认定吴裕泰为[[中华老字号|中华老字号]]<ref name=bjlyw/>。2005年，北京吴裕泰茶业股份有限公司成立，为销售茶叶、茶具、茶衍生品的专业公司<ref>{{Cite web |url = http://www.wuyutai.com/list-13-1.html |title = 品牌之旅 |author =  |date =  |publisher = 北京吴裕泰茶业股份有限公司 |language =  |accessdate = 2017-01-24 |quote =  }}</ref>。2014年，北京吴裕泰茶业股份有限公司被北京老字号协会认定为[[北京老字号|北京老字号]]。
