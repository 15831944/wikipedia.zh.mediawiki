'''吳清友'''（{{bd|1950年|10月|2017年|7月18日|catIdx=W吴}}），臺灣企業家，[[誠品書店|誠品書店]]創辦人。2017年7月18日因病逝世，享壽68歲。<ref name="自由">{{cite news|author=楊明怡、楊媛婷、張慧雯、姚岳宏|title=改變書店文化 誠品吳清友病逝|url=http://news.ltn.com.tw/news/life/paper/1119958|publisher=自由時報電子報|language=zh-tw|date=2017-07-19|accessdate=2017-07-19}}</ref>

== 早年 ==
吳清友生於臺南縣將軍鄉（今臺南市將軍區）馬沙溝，父親為地方顯達吳寅卯。吳清友有九個兄弟姊妹，在五個男孩中排名第四。<ref name="今周刊">{{cite book|author= 林讓均、林麗娟、賴琬莉|title=闖關 吳清友的誠品傳奇|url=http://www.businesstoday.com.tw/article-content-92751-99520|magazine=今周刊|volume=815|publisher=今周出版股份有限公司 |language=zh-tw|date=2012-08-02|accessdate=2017-07-19}}</ref>

吳清友1968 年[[國立臺南高級工業職業學校|國立臺南高級工業職業學校]][[機械科|機械科]]畢業後，因患有遺傳性疾病[[馬凡氏症候群|馬凡氏症候群]]，<ref name="udn">{{cite news|author=劉嘉韻、魏忻忻|url=https://udn.com/news/story/7266/2591452|title=因馬凡氏症候群 誠品董座吳清友三度動刀|publisher=[[聯合新聞網|聯合新聞網]]|date=2017-07-18|accessdate=2017-07-25|archive-url=https://web.archive.org/web/20170718201558/https://udn.com/news/story/7266/2591452|archive-date=2017-07-18|language=zh-tw|deadurl=yes}}</ref>故免服[[兵役|兵役]]。工作幾年之後於1970年再進入[[國立臺北工業專科學校|國立臺北工業專科學校]]（今[[國立臺北科技大學|國立臺北科技大學]]）機械科就讀，1972年從二專機械科畢業。

高工畢業後，1973年先在專營飯店餐廚設備與[[咖啡機|咖啡機]]的誠建室內裝修股份有限公司當業務員，31歲（1981年）接下誠建的全部股權，成為誠建的董事長。<ref>{{Cite web|url=http://www.ntutana.org.tw/?p=1373|title=『闖蕩一種生命的可能』誠品書店董事長 吳清友 {{!}} 國立臺北科技大學校友會全國總會|accessdate=2017-11-22|work=www.ntutana.org.tw|language=zh-TW}}</ref>

== 創建誠品書店 ==
當飯店數量漸趨飽和之後，吳清友考慮開創新的企業版圖，開始涉足書店的經營。吳清友在1989年3月創辦[[誠品書店|誠品書店]]，初期專賣[[藝術|藝術]]與[[建築|建築]]類書籍，並規劃出有別於同業的經營模式（如1999年誠品敦南店首開24小時營業、結合書店與商場的複合式經營模式等），成功的經營出特殊的誠品書店文化：「尊重走進書店裡的每一個人」。

誠品書店開始經營的前15年都是賠本，吳清友說：「[[童子賢|童子賢]]先生跟我說，店儘量開，賠錢不要緊。」。由於誠品書店受到不少文人、藝術家的敬重，及許多媒體報導，使得誠品書店在台灣有相當的重要性與知名度。2011年總計有1.2億人次到訪臺灣誠品，遠遠超過[[臺灣人口|臺灣人口]]總數。<ref>{{cite web|title=誠品生活股份有限公司 民國一〇〇年度年報|publisher=誠品生活股份有限公司|url=http://www.eslitespectrum.com.tw/Investors/Files/AnnualReport/20140904141726833.pdf|date=2012-04-30|accessdate=2017-07-19|language=zh-tw}}</ref>

[[施振榮|施振榮]]夫婦、[[殷琪|殷琪]]與[[童子賢|童子賢]]是誠品個人大股東，長期投資不賺錢的誠品書店。童子賢曾說：「我樂意看到文化走入日常生活裡⋯⋯，讓每個人伸手可得到美好的生活經驗⋯⋯，應以實質力量支撐，讓它繼續茁壯。」<ref>{{cite book|author=劉德宜|title=吳清友撐起誠品帝國 搶下3大案|magazine=時報周刊|volume=1443|publisher=時報周刊股份有限公司|quote=我樂意看到誠品現象，文化走入日常生活裡，如果消失了，我會覺的很傷心；相對地，不會消失、值得高興的是，不一定要站在經營者的困難辛勞與股東的回報與財務報償，不看這些，光它替台灣帶來了文化深入日常生活，讓每個人伸手可得到美好的生活經驗，就應該鼓勵它，並以實質力量支撐，讓它繼績茁壯。}}</ref>

== 家庭 ==
吳清友已婚，與妻子洪肅賢育有一子一女。<ref name="今周刊"/>長子吳威廷於2009年因病去世。<ref name="蘋果">{{cite news|title=誠品老董獨子猝死|url=http://www.appledaily.com.tw/appledaily/article/headline/20090501/31592231/|publisher=蘋果日報|language=zh-tw|date=2009-05-01|accessdate=2017-07-19}}</ref>長女吳旻潔為英國[[谢菲尔德大学|-{zh-hant:謝菲爾德大學;zh-hans:谢菲尔德大学;zh-tw:雪菲爾德大學;}-]]廣電新聞系碩士，2004年學成歸國後進入誠品團隊，<ref name="今周刊"/>在吳清友身故後，接任誠品股份有限公司董事長一職，<ref>{{cite news|title=吳清友猝逝 愛女接班誠品|url=http://www.appledaily.com.tw/appledaily/article/headline/20170720/37721654/|publisher=蘋果日報|date=2017-07-20|accessdate=2017-07-20|language=zh-tw}}</ref>另兼任誠建室內裝修股份有限公司董事長、誠品生活股份有限公司董事暨總經理等職務。

== 逝世 ==
吳清友患有遺傳性疾病[[馬凡氏症候群|馬凡氏症候群]]，曾分別於1988年、2000年與2006年開刀診治。<ref name="udn"/><ref>{{cite web|author=張曉卉|title=吳清友的馬凡氏症Marfan syndrome是罕見疾病，有多麻煩？|url=http://www.commonhealth.com.tw/article/article.action?nid=75374|publisher=[[康健雜誌|康健雜誌]]|date=2017-07-19|accessdate=2017-07-20|language=zh-tw}}</ref>吳清友2017年7月18日晚間在臺北市信義區的辦公室突然心臟病復發，經送醫後仍然不治，享壽68歲。<ref name="自由"/>
