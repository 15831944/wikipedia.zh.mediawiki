'''吳敬恆'''（{{bd|1865年|3月25日<ref>「吳敬恆先生事略」，《中國近代學人像傳》初輯，[[江蘇廣陵古籍刻印社|江蘇廣陵古籍刻印社]]，1997年</ref>{{rp|78}}|1953年|10月30日|catIdx=W吳}}），一名'''脁'''，字'''稚暉'''，[[以字行|以字行]]，原籍[[江蘇省|江蘇省]][[常州府|常州府]][[武進縣|武進縣]]，出生於江蘇[[武進|武進]]和[[无锡|无锡]]交界處的[[雪堰橋|雪堰橋]]。[[中華民國|中華民國]]開國元老、[[中央研究院|中央研究院]]院士，一生追隨[[國民革命|國民革命]]却不入[[政府|公家]]。1947年，在原籍江蘇省武進縣當選為[[第一屆國民大會代表|第一屆國民大會代表]]。

吳稚暉工書法，著作有《荒古原人史》、《天演圖解》、《上下古今談》、《'''朏盦'''{{NoteTag|1=fěi ān}}客座談話》等。

== 早年經歷 ==
吳稚暉早年曾在南菁書院[[黃以周|黃以周]]門下學習過。光緒十七年（1891年）中式[[舉人|舉人]]。接觸[[西方文化|西方文化]]後，追求[[無政府主義|無政府主義]]，後來參與[[孫中山|孫中山]]主導的[[國民革命|國民革命]]，孫中山信仰[[太平天國|太平天國]]老兵故事即是出自吳敬恆所述。曾在[[蘇報案|蘇報案]]中被認為是可能的告密者；也是早期[[國民黨|國民黨]]內部極右派[[西山會議派|西山會議派]]成員之一。

== 中年經歷 ==
1898年到[[南洋公学|南洋公学]]任教，1901年时曾出任南洋公学附属小学堂（今[[上海市南洋模范中学|上海市南洋模范中学]]）堂长。<ref name="sjtu">{{cite web|url=http://www.sjtu.edu.cn/intro/document/recall/r05.htm|title=南洋公学和南洋模范中学|publisher=上海交通大学|accessdate=2010-05-01|deadurl=yes|archiveurl=https://web.archive.org/web/20070228132518/http://www.sjtu.edu.cn/intro/document/recall/r05.htm|archivedate=2007-02-28}}</ref>

1903年在《[[苏报|苏报]]》撰文抨击[[清廷|清廷]]，痛罵當時執政的[[慈禧太后|慈禧太后]]是「娼妓淫婦」。《苏报》案发後经[[香港|香港]]去[[伦敦|伦敦]]。1905年冬参加[[同盟会|同盟会]]。1906年在[[巴黎|巴黎]]参与组织[[世界社|世界社]]，1907年刊行《[[新世紀_(雜誌)|新世紀]]》周刊及《[[世界画刊|世界画刊]]》，鼓吹[[无政府主义|无政府主义]]。

1911年後，多從事文化運動。1913年任教育部[[读音统一会|读音统一会]]议长，提倡[[國語注音|國語注音]]與[[國語運動|國語運動]]。他積極投入的國語注音，今稱「[[注音符號|注音符號]]」，仍於[[臺灣|臺灣]]及海外老[[華僑|華僑]]間通行。除了文化運動外，他更被蔣中正之子[[蔣經國|蔣經國]]視為老師。

1913年6月参与创办《[[公论|公论]]》日刊。1913年參與孫文發起的反袁世凱政權的[[二次革命|二次革命]]，8月失败后於9月再赴[[欧洲|欧洲]]。<ref>詹瑋，《吳稚暉與國語運動》（台北市：文史哲出版社，1992），頁185-186。</ref>在這一段時間，吳稚暉還創辦[[里昂中法大學|里昂中法大學]]並發起留法[[勤工儉學運動|勤工儉學運動]]。呼籲中國青年到海外以[[半工半讀|半工半讀]]方式留學。如[[周恩來|周恩來]]、[[鄧小平|鄧小平]]等都曾參加。

1924年，吳稚暉等人在[[上海|上海]]創辦[[中華粥會|中華粥會]]。至今中華粥會仍在臺灣運作，秉持著每月集會吃粥，以紀念吳稚暉。1925年，蔣經國向吳稚暉推薦自己為可以前往中國國民黨候選成員，並且認為自己能通過[[中國共產黨中央委員會|中國共產黨中央委員會]]同意而前往[[蘇聯|蘇聯]]就讀。雖然吳稚暉在中國國民黨內是[[西山會議|西山會議派]]重要人物，反對中國共產黨之主張，後來促成中國國民黨決定發起[[四·一二事件|四·一二事件]]與蘇聯決裂，但吳稚暉並沒有嘗試勸阻蔣經國前往[[莫斯科|莫斯科]]之念頭。

== 晚年經歷 ==
1927年，任[[中国国民党|中国国民党]]中央监察委员。他認為[[中國共產黨|中國共產黨]]與[[蘇俄|蘇俄]]的密切聯繫危害[[中國|中國]]未來發展，並有破壞[[中華文化|中華文化]]之嫌，因此力倡[[清黨|清黨]]。之後，他反对中国国民党内一切反对蒋中正的力量，成为蒋的御用心腹智囊；在1927年－1950年間的國共鬥争中，扮演積極[[反共|反共]]的角色，也因此，中國共產黨對他定位與評價十分低劣。
 
1943年，[[國民政府主席|國民政府主席]][[林森|林森]]病逝[[陪都|陪都]][[重慶|重慶]]，蔣中正力邀吳稚暉為新主席，但被其推辭，按其原話，吳稚暉的推辭理由是這麼「三不」：
{{cquote|
#我平常的衣服穿得很隨便很簡單，做元首要穿[[燕尾服|燕尾服]]、打[[領結|領結]][[領帶|領帶]]，我覺得不自在；
#我臉長得很醜，不像一個大人物的模樣；
#我這個人愛笑，看到什麼會不自主地笑起來，哪天外國使節來遞[[國書|國書]]，會不由得笑起來，不雅。
}}
以後，吳稚暉也基於此三點，不任任何官職。

[[File:Delivering_ceremony_of_the_Constitution_of_ROC.jpg|right]]
1946年，[[國民政府|國民政府]]宣佈結束[[訓政|訓政]]階段，在南京召開的[[制憲國民大會|制憲國民大會]]確立了《[[中華民國憲法|中華民國憲法]]》，吳稚暉擔任[[制憲國民大會代表|制憲代表]]主席，將《中華民國憲法》遞交給蔣中正。這個歷史時刻所拍攝下的照片，至今仍然是臺灣歷史教科書裡的重要圖片。因此，很多臺灣人對他的印象是「制憲大老」。隨後蔣中正當選[[中華民國|中華民國]]第一任總統。

1947年3月17日，三民主義青年團團部聘定第二屆指導員吳稚暉、[[戴季陶|戴季陶]]等25人，評議員[[周鯁生|周鯁生]]等49人<ref name="民國史大事記">{{cite book|editor=李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編|title=《中華民國史大事記》|location=北京|publisher=[[中華書局|中華書局]]|year=2001}}</ref>{{rp|8314}}。3月23日，中國國民黨第六屆中央監察委員會舉行第三次全體會議，通過《各級監察委員會考核員政治活動辦法》，並改選常務委員，[[張繼|張繼]]、吳敬恒、[[王寵惠|王寵惠]]、[[邵力子|邵力子]]等12人當選<ref name="民國史大事記"/>{{rp|8319}}。

1949年，蔣中正派專機「[[美齡號|美齡號]]」將吳稚暉從[[廣州|廣州]]接到了[[臺灣|臺灣]]臺北。6月1日，蔣介石與蔣經國在台北拜訪吳敬恒，晤談後抵達[[岡山區|岡山]]，於傍晚偕蔣經國等視察高要塞<ref name="民國史大事記"/>{{rp|8932}}。有一次蔣經國問他：「先生，你看我們今後成功的希望怎麼樣？我們重新站起來的希望怎麼樣？」吳老先生的答覆非常簡單，他說：「你想成功，就會成功，你不想成功，就永遠不會成功。」<ref>蔣經國：〈統一思想統一步驟〉（1965年7月2日對[[金門|金門]]防衛部全體官長講），刊《只有戰鬥才能生存 只有生存才有希望》，台北：[[黎明文化事業股份有限公司|黎明文化事業股份有限公司]]，1985年</ref>{{rp|34-35}}吳稚暉又對蔣經國說：
{{quote|
「我知道有許多人，想用各種手段反對你，也有人造謠中傷你；但是這些事，想明白了，算不得什麼！為了你的父親，為了你的同事，你都必須好好的做。一個没有被人打擊過的人，是不會成人的。我覺得你所受的打擊還是太小了。你現在不但是為了自己的工作，要好好幹下去；即使為了你的各種各樣敵人，更應該好好的幹，因為任何敵人所希望的，是你放手、讓步、不幹！」<ref>{{Tsl|ja|小谷豪治郎|小谷豪冶郎}}著、[[陳鵬仁|陳鵬仁]]譯：《蔣經國先生傳》，台北：《[[中央日報|中央日報]]》出版部，1990年</ref>{{rp|232}}
}}

== 逝世 ==
1953年，吳敬恆病重住院。根據好友[[李石曾|李石曾]]在吳敬恆住院期間探望時寫下的[[日記|日記]]記錄：「稚老在醫院晤面，屢次表示欲到[[美國|美國]]看子淑微就醫情況，俟較愈再往，而卒未能成行，稚老憾事……」日記中還記錄說，如果能成行美國，也不會[[海葬|海葬]][[金門|金門]]。同年10月30日深夜，中國國民黨中央評議委員吳敬恆於臺北逝世<ref name="年表"/>{{rp|75}}，年89歲。蔣中正題詞「痛失師表」。12月1日，按照遺囑，蔣經國將他的遺骨運到金門，雇用漁船，「把它葬於廈門南面海底」<ref>《蔣總統經國先生言論著述彙編》第三集，台北：黎明文化事業股份有限公司，1983年2月</ref>{{rp|407－413}}。

== 紀念 ==
1954年，連基座高達四公尺以上的吳稚暉銅像於臺北市[[敦化北路|敦化北路]]跟南京東路交叉口圓環豎立，成為當時臺北市地標之一。

另外在金門縣[[金城鎮_(金門縣)|金城鎮]]水頭附近，還有一座[[吳稚暉公園|吳稚暉公園]]，於海葬當年建成。與水頭碼頭比鄰而座，由於公園地勢較高，水頭碼頭景緻盡收眼底。除此，還可瞭望[[小金門|小金門]]、大陸及[[金烈水道|金烈水道]]等。園內建築稚暉亭（後改為吳公亭），鑄銅像、立石碑，皆出於蔣經國經營。<ref>王禹廷著：《胡璉評傳》，台北：傳記文學出版社，1987年6月15日</ref>{{rp|326}}亭中央是蔣中正所題的碑「吳稚暉先生水葬紀念亭」。公園盡頭則是敬恆先生半身銅像。另於由蔣經國修建[[太魯閣|太魯閣]]之交通要道（即[[台8線|橫貫公路]]），以「稚暉橋」命名之。


在其身後，[[李石曾|李石曾]]、[[鄧傳楷|鄧傳楷]]等人開始籌設學校紀念他。1960年設立私立稚暉初級中學，1963年增設高中部，改稱稚暉中學。

1964年3月，吳稚暉先生百年誕辰，蔣介石撰頌詞讚揚吳氏一言而為天下法，一行而為萬世師。<ref name="年表">[[陳布雷|陳-{布}-雷]]等編著：《蔣介石先生年表》，台北：[[傳記文學|傳記文學]]出版社，1978年6月1日</ref>{{rp|106}}

1986年，稚暉中學被改為稚暉高級工業家事職業學校，1996年又被更名為[[嘉義市私立仁義高級中學|嘉義市私立仁義高級中學]]。<ref>{{cite web|title=[[教育部|教育部]]中部辦公室網站 |url=http://www.tpde.edu.tw/ap/school_view.aspx?id=201309--01&ctycod=20&type=3 |deadurl=yes |archiveurl=https://archive.is/20120805172641/http://www.tpde.edu.tw/ap/school_view.aspx?id=201309--01&ctycod=20&type=3 |archivedate=2012-08-05 }}</ref>

1990年代，該銅像因政治風氣與整頓交通等因素被拆除，銅像今置於臺北市[[士林區|士林區]][[至善公園|至善公園]]。

== 評價 ==
吴稚晖以精於[[小篆|小篆]]為著名，與操[[隶书|隶书]]的[[胡汉民|胡汉民]]、[[楷书|楷书]]的[[谭延闿|谭延闿]]、[[草书|草书]]的[[于右任|于右任]]等人，並譽為「[[中國國民黨|中國國民黨]]四大[[书法家|书法家]]」。

蔣經國：「老人（吳稚暉）嚴肅而有力的語句，帶給我很大的勇氣和力量。」<ref>{{cite book|author=蔣經國|title=《風雨中的寧靜》|location=台北|publisher=[[正中書局|正中書局]]|year=1988}}</ref>{{rp|335}}
