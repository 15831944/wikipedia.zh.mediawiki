'''株洲师范高等专科学校'''，位于[[湖南省|湖南省]][[株洲市|株洲市]][[天元区|天元区]][[泰山路|泰山路]]，为湖南省高等[[专科大学|专科大学]]。<ref name=" fish" >{{cite web |url=http://www.huaue.com/xxjj/2005227110518.htm |title=株洲师范高等专科学校 |accessdate=2005年2月27日 |publisher=华禹教育网 |work= |language =zh-cn}}</ref>

== 历史 ==
1978年，湖南省建立了“株洲教育学院”，它是株洲师范高等专科学校的前身。<ref name=" fish" />

1993年，经过[[中华人民共和国教育部|中华人民共和国教育部]]的允许，学校升格为“株洲师范高等专科学校”。<ref name=" fish" />

2006年,经教育部批准株洲师范高等专科学校并入湖南工业大学。 <ref name=" fish" />

2012年，原株洲师专校区以总价7.5亿人民币出售给中国奥园地产集团，目前株洲师专泰山路的校区，已经全部被拆除完毕。以后株洲师专的学子们，恐怕再也看不到母校的身影了。

== 学校院系 ==
株洲师范高等专科学校一共有10个系，1个中心，25个专业。<ref name=" fish" />
*系：中文系、外语系、人文社科系、策划与现代管理系、数学与计算机科学系、物理与电子工程系、化学化工系、音乐系、美术系、体育系<ref name=" fish" />
*中心：现代教育中心<ref name=" fish" />
*专业：中文教育、文秘与社区管理、英语教育、商贸英语、思想政治教育、历史教育、旅游与酒店管理、初等教育、策划与经纪、数学教育、计算机教育、计算机应用、电子商务、物理教育、应用电子技术、现代教育技术、化学教育、应用化学技术、音乐教育、音乐表演、美术教育、现代艺术设计、新闻美术与摄影、体育健康教育、社会体育<ref name=" fish" />

== 学校文化 ==
=== 期刊 ===
株洲师范高等专科学校创办了《学理追问》、《思想拷问》、《义理究问》三大类丛书。<ref name=" fish" />

== 奖项 ==
株洲师范高等专科学校先后荣获“全省普法依法治理先进单位”、“省级花园式单位”、 “档案管理省级先进单位”、“省卫生先进单位”、“文明单位”等荣誉奖项。<ref name=" fish" />
