== 登场人物 ==

; {{nihongo|伊斯卡|イスカ|Iska|CV：[[小林裕介|小林裕介]]}}
: 本作男主角。帝国所属的16岁军人少年。以史上最小年龄成为帝国军的最高战力“使徒圣”。使用着一把名为“星剑”的特殊黑白双剑。

; {{nihongo|爱丽丝莉泽·卢·涅比里斯 9世|ァリスリーゼ・ルゥ・ネビュリス9世|Aliceliese Lou Nebulis IX|CV：[[雨宫天|雨宫天]]}}
: 本作女主角。涅比里斯皇厅的17岁第二王女。使用冰系星灵力量的强大星灵使，被帝国冠以“冰祸的魔女”这一畏称。

; {{nihongo|蜜思米丝·克拉斯|ミスミス・クラス|Mismis Klass}}
: 伊斯卡所属的小队队长。虽然身躯较小还长着一张娃娃脸，但确实是一名正儿八经的22岁成熟女性。

; {{nihongo|音音·阿卡斯通|音々・ァルカストーネ|Nene Alkastone}}
: 担任小队的机械师。将伊斯卡当做兄长来敬仰，开朗的少女。

; {{nihongo|烬·休拉干|ヅソ・ツュラルガソ|Jhin Syulargun}}
: 小队的狙击手。过去曾与伊斯卡在同一师门下修行的孽缘。

; {{nihongo|燐·维斯珀斯|燐・ヴィスポーズ|Rin Vispose}}
: 爱丽丝莉泽的亲信兼女仆。使用土系星灵力量的星灵使，也很擅长暗杀术。

== 设定&用语 ==

=== 国家 ===
'''帝国 — 机械装置的理想乡'''（{{lang|en|Empire}}）

: 拥有高度发达的科学力与强大的军事力，世界第一的大国。对于皇厅使用尚未解明名为“星灵”强大的力量感到了威胁，想要将其毁灭。

'''涅比里斯皇厅 — 魔女们的乐园'''（{{lang|en|Imperial Household Agency}}）

: 运用“星灵”这一尚未解明的能量，来引出异于常人力量水平的魔女们的国度。憎恨日常迫害着她们的帝国。

=== 用语 ===
'''使徒圣''' （{{lang|en|Protecter Lord}}）

: 帝国的最强战力，天帝直属的护卫，但是也有人会接受司掌帝国议会的『八大使徒』的命令而行动。

'''星灵使'''

: 在星脉喷泉，沐浴星灵之后，拥有资格的人，身体上会出现星纹，这是成为星灵使的标志。

'''星脉喷泉''' （{{lang|en|Vortex}}）

: 产生不明能量-星灵的地方。帝国与皇厅之间引发战争的罪魁祸首。
