== 故事簡介 ==
四名女高中生在修學旅行途中，突然遭遇[[空難|空難]]而流落荒島，劫後餘生的她們為了存活，究竟如何利用島上資源、學習自我營生？

== 登場人物 ==
; {{nihongo|鬼島譽|鬼島 ほまれ（おにしま ほまれ）|Onishima Homare|聲：[[M·A·O|M·A·O]]}}
: 本作[[主人公|主人公]]，退役軍人的女兒，自幼隨父行旅各國，卻屢逢不測，所幸憑藉其父豐富的求生知識，都能化險為夷，譽也在耳濡目染下，熟諳野外技能，並以此深得三人仰賴。

; {{nihongo|鈴森明日香|鈴森 明日香（すずもり あすか）|Suzumori Asuka|聲：[[河野日和|河野日和]]}}
: 體能卓越，粗枝大葉，能為朋友兩肋插刀。心地仁慈，初次捕捉到[[蟾蜍|蟾蜍]]，竟不忍將其作為食材而蓄意放生，不過為了生存，還是學會了狠心。

; {{nihongo|天谷睦|天谷 睦（あまたに むつ）|Amatani Mutsu|聲：[[安野希世乃|安野希世乃]]}}
: 勤勉向學，行事慎重，頗擅烹飪，是譽最稱職的助手。將來夢想是[[BL_(和製英語)|BL]][[小說家|小說家]]。

; {{nihongo|九條紫音|九条 紫音（くじょう しおん）|Kujo Shion|聲：[[和氣杏未|和氣杏未]]}}
: 生於豪貴之家，過慣養尊處優的日子，經常不自覺提出[[公主病|公主病]]的要求，但也慢慢自食其力，是四人中最靈巧的一個。

; {{nihongo|鬼島父|鬼島父|}}
: 譽的父親，前軍人，具有淵博的求生知識。在救援隊之前，獨力搜查到女兒下落，卻偶遇累二人，見其狀況危急、更需救助而出手，令其成功脫險。

; {{nihongo|鳥居累|鳥居 累（とりい るい）|Torii Rui}}
: 與譽等同時遇難的女學生，身著男裝，讓宗二誤認其性別。對宗二暗生情愫，刻意妨礙他接近譽等人。

; {{nihongo|服部宗二|服部 宗二（はっとり そうじ）|Hattori Soji}}
: 與譽等同時遇難的男學生，和累兩人相依為命，欲求生卻不得綱領。
