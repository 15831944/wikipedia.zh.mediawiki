《'''喜迎十八大　走基层·百姓心声　假日特别调查：你幸福吗'''》是[[中国中央电视台|中国中央电视台]]新聞中心在2012年[[中秋|中秋]]、[[中华人民共和国国庆节|国庆]]期间推出的关于“走基层转作风改文风”的特别节目，分别在《[[新闻30分|新闻30分]]》、《[[新闻联播|新闻联播]]》等新闻栏目中作为一个子栏目呈现。内容为各路央视[[记者|记者]]在全国各地以“你幸福吗？”的问句为开场采访普通百姓。因得到如“我姓曾”“我耳朵不好”等各种风趣幽默的“神回覆”得到广泛关注<ref>{{cite web|url= http://opinion.people.com.cn/n/2012/1006/c159301-19177984.html |title=为“你幸福吗”的提问鼓次掌 |publisher=人民网 |accessdate=2012-10-06 |deadurl=yes |archiveurl= https://web.archive.org/web/20121114211726/http://opinion.people.com.cn/n/2012/1006/c159301-19177984.html |archivedate=2012-11-14}}</ref>。

== 争议对话 ==
* 2012年9月29日新闻联播中播出的走基层百姓心声中，记者问一名清徐县北营村务工人员“您幸福吗？”对方回答“我姓曾”。<ref>{{cite web |url= http://news.ifeng.com/mainland/detail_2012_09/29/18008320_0.shtml |title=央视记者问你幸福吗 务工人员回答：我姓曾 |accessdate=2012-10-06}}</ref>
* 2012年10月4日新闻联播中播出的走基层百姓心声中，记者问一拣瓶子的老人“您幸福吗？”老人回答“我耳朵不好”。<ref>{{cite web |url= http://news.qq.com/a/20121004/000848.htm?pgv_ref=aio2012&ptlang=2052 |title=央视记者问“你幸福吗” 捡瓶子老人：耳朵不好 |accessdate=2012-10-06}}</ref>
