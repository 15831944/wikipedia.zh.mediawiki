== 劇情 ==
[[迪亞馬特|迪亞馬特]][[彗星|彗星]]（{{lang|ja|ティアマト彗星}}）每逢1200年造訪[[地球|地球]]，在深山小鎮{{nihongo|[[糸守町|糸守町]]|糸守町／いともりまち|romaji=}}{{notetag|中國大陸上映的中文配音版口誤成「'''系'''守镇」<ref>{{cite web|url=http://www.weibo.com/ttarticle/p/show?id=2309404049110251485696|title=《你的名字。》译制名单（日本·长影）|work=|author=陆柏宇|date=2016-12-04|accessdate=}}</ref>。}}中家族世代經營鎮內「宮水[[神社|神社]]」的高中少女'''宮水三葉'''、以及[[東京|東京]][[四谷|四谷]]{{notetag|官方設定集指出瀧的家在[[新宿區|新宿區]][[若葉_(新宿区)|若葉]]，但[[加納新太|加納新太]]的外傳小說中是[[千代田區|千代田區]][[六番町_(千代田區)|六番町]]。}}的高中少年'''立花瀧'''，從某天開始，在睡覺的时候，就會[[靈魂|靈魂]]互換，醒來什麼都不記得，感覺像做了奇妙的[[夢|夢]]，但周围的人会讲出他们昨天有违平常。他倆很快意識到這種情形，困惑之中開始在彼此[[智能手机|智能手機]]留言，逐漸地變得習慣於身體切換，並干預對方的生活。三葉不僅樂於所嚮往的[[東京|東京]]生活，也利用女人心這點，幫助瀧更接近其打工的美女前輩奧寺，促成與她約會；而瀧以陽剛的形象，幫助柔弱的三葉在她的學校更受歡迎，雙方逐漸對彼此有好感。

有次三葉（瀧）和妹妹四葉、外祖母一起徒步至宮水神社位在鎮外山上的「[[御神體|御神體]]」，途中外祖母告訴她們土地神「{{lang|ja|産霊＝ムスビ＝結}}」，可以連結人與時間，因此宮水家族的歷代[[巫女|巫女]]都編織[[結繩|結繩]]來體現，水酒等進入人體也能形成連結。三人參拜位於[[破火山口|破火山口]]型[[窪地|窪地]]{{notetag|[[新海誠|新海誠]]筆下小說版明確用「{{lang|ja|カルデラ型の窪地}}」。外傳小說描述勅使河原想像若有火山時的情景，但周邊就真只是山而已，似乎否定火山存在。}}{{nihongo|「隱世」|カクリヨ}}上的御神體，並以[[口嚼酒|口嚼酒]]為半身獻祭。回程時當三葉（瀧）和妹妹四葉、外祖母在觀看糸守湖的夕陽時，外祖母對三葉（瀧）說：「你是不是在作夢？」，瀧聽完后從床上驚醒起来，并发觉自己在流泪。

瀧與奧寺約會時，偶然看到[[飛驒市|飛驒]]照片喚起記憶，三葉留下信息告訴瀧約會後正好可以看到彗星，瀧覺得此話十分奇怪，決定打电话詢問卻打不通，而且自那天起就再也沒有跟三葉交換身體。心中充滿疑問的他利用擅長的素描畫下片段印象，並與同學藤井司、前輩奧寺一起去飛驒探個究竟。在瀧一行人拿著[[素描|素描圖]]詢問當地居民但四處碰壁之際，遇上拉麵店老闆閒聊回憶起家鄉，得知該地名為糸守町，並在拉麵店老闆的帶領下來到已無人居住的原糸守町；由於3年前迪亚马特彗星分裂出来的隕石擊中當時正在舉行秋日[[祭_(日本)|祭典]]的宮水神社一帶，村镇被摧毀大半，超過500名鎮民、包括宮水本家及其朋友皆不幸罹難。不相信這個事實的瀧打開手機，卻發現三葉的留言全部消失，而歷史文獻亦將她列在罹難者名單之中。當天晚上和奧寺谈起瀧手上不知何來的结绳信物时，瀧想到三葉外祖母曾經說過土地神可以連結人與時間，便決定回到過去幫助三葉和糸守町居民們躲過災難，他留下字條跟藤井司、奧寺說明原因後，在拉麵店老闆的幫助下前往御神體。

瀧到達御神體並喝下三葉制作而自己当时留下的口嚼酒後便一陣暈眩、意外絆倒時，种种片段歷歷在目，等到瀧回過神來，發現自己再度與三葉交換身體。在三葉（瀧）思考如何拯救糸守町時，又被外祖母識破，并提起自己、二叶、甚至整个宫水家都有过似夢恍惚时候，但是自己已經錯過了記憶，特別告誡三葉（瀧）要好好珍惜現在這個緣份。三葉（瀧）向友人勅使河原克彦、名取早耶香說明情況並且開始策畫避難行動，查出小鎮所環繞的糸守湖也是[[隕石|隕石]]造成的[[撞击坑|撞击坑]]<ref>早耶香本來以為三葉是胡思亂想，然後敕使就說明糸守湖是如何形成。</ref>，更加深他們的信念。勅使河原克彦負責用[[水胶炸药|水胶炸药]]爆破糸守町附近的變電所造成騷動，名取早耶香則在爆炸後透過廣播疏散鎮民，三葉（瀧）則是跑去跟身為[[市長|町長]]的父親要求疏散鎮民，爭執之中町長亦從三葉舉止發現她的意識變成別人。在三葉（瀧）奔走之際，在御神體的瀧（三葉）醒來對於瀧的身體為何在御神體感到不明白，好奇的瀧（三葉）在走出御神體的山頂時見到小鎮消失，才領悟到本人已在当时觀看彗星的時候死去。当三葉（瀧）至御神體之處，在{{nihongo|「黃昏之時」|カタワレ時}}兩人得以相見。在短暫的寒暄後，為避免再次相忘，瀧提議雙方在對方手上寫下自己的名字。三葉正要在瀧手上寫下名字之際兩人卻又分離，而瀧在三叶手上寫下了“喜歡你”。回復真身的三葉繼續執行瀧未完成的計畫，造成騷動促使鎮民避難，但其同儕的行動遭町長阻止，此時三葉見到彗星分裂碎片開始墜落，趕緊再次去遊說父親……

時間跳到自飛驒回來5年期間，瀧已經步入社會，对御神體時的事蹟的種種已变得模糊，以為應是跟藤井司、奧寺吵架然後負氣跑到山頂散心；回到東京，不自覺地對糸守這個地方很在意，從新聞記錄中得知鎮民因町長以「演習」名義指示至當地高中避難而躲過一劫。即使距彗星災難8年後（2021年），大學畢業在東京四處求職的瀧，仍依稀想找尋某人，期間數次與三葉擦肩而過。在一次偶然機會下，瀧與三葉隔著不同電車的車廂，驚訝地遇見了彼此。雙方下車後焦急地想與對方見面，最終在階梯相會。此時的瀧和三葉逐漸對彼此有了熟悉的感覺，在那久別重逢的當下，眼淚卻從自己的臉龐止不住地滑下。瀧忍不住開口問：「我是不是在哪裡見過你？」，三葉回答：「我也是。」，最終，雙方對問：「你的-{}-名字是？」（{{lang|ja|君の、名前は}}）。

== 製作 ==
=== 源起 ===
[[東寶|東寶]]在映像事業部負責發行《[[言葉之庭|言葉之庭]]》之際即傳達長期合作意圖，由於[[CoMix_Wave_Films|CoMix Wave Films]]自己沒法應付更具規模作品，因此慶功宴時社長川口典孝告知下次全都交給東寶辦理，而日本全國23家戲院上映就有1.3億日圓票房佳績，讓東寶認為導演非新海誠莫屬。2013年秋企畫啟動，2014年春正式洽商相關事宜<ref name=yno>{{cite web |url=http://www.kiminona.com/production/ |script-title=ja:PRODUCTION NOTE：映画「君の名は。」とは |publisher={{lang|ja|映画『君の名は。』公式サイト}} |language=ja |date=2016-09-08 |accessdate=2016-10-16}}</ref>。只是不光尊重導演的作家性，也得考量賣座需求、該有現場能跟他議論的人才行，東寶最當紅的{{link-jp|川村元氣|川村元気}}於是加入。2014年夏開始製作<ref>{{cite web |url=http://shinkaimakoto.jp/kiminona |script-title=ja: 新作アニメーション映画『君の名は。』について |work=Shinkaimakoto.Com |language=ja |author={{lang|ja|[[新海誠|新海誠]]}} |date=2015-12-11 |accessdate=2016-01-06}}</ref><ref name=anata>{{cite web|url=http://shinkaimakoto.jp/archives/849|script-title=ja:「あなた」へ、2016年のこと。|work={{lang|ja|Other voices-遠い声-}}|author={{lang|ja|新海誠}}|date=2016-12-31|accessdate=|language=ja }}</ref>。

=== 構思 ===
作品劇情方面上，新海誠最初的方向並非以單純戀愛題材為主，而想採用「有著見面的命運但遲遲無法相見的兩人」；一種較不平常的「Boy meets girl」風格故事<ref name="VisualGuidePage58">{{cite book |script-title=ja:『君の名は。 公式ビジュアルガイド』 |publisher={{lang|ja|[[角川書店|角川書店]]}} |location=日本 |isbn=978-4-04-104780-4 |date=2016-08-27 |script-chapter=ja:［監督］新海誠 インタビュー|page=58|language=ja}}</ref>。其中過去2014年新海誠替教育機構「Z會」補習班製作的廣告動畫短片{{nihongo|《[[十字路口_(動畫)|十字路口]]》|クロスロード}}，被新海誠作為此影片的原型<ref>{{cite tweet |author={{lang|ja|[[新海誠|新海誠]]}} |author-link= |user=shinkaimakoto |number=774223435505381376 |date=2016-09-09 |script-title=ja:映画『君の名は。』の原型のひとつです。|access-date=2016-10-04 |language=ja}}</ref><ref name="as">{{cite interview|url=https://akiba-souken.com/article/27680/?page=1 |title={{lang|ja|新海誠監督・最新作品「君の名は。」公開記念特別インタビュー}} |publisher={{lang|ja|アキバ総研}} |author=|language=ja |date=2016-08-25 |accessdate=2016-10-25}}</ref>。

作品靈感之中，[[平安時代|平安時代]]的女性[[和歌|和歌]]歌人[[小野小町|小野小町]]的一首創作歌詞{{nihongo|「梦里相逢人不见，若知是梦何须醒。纵然梦里常幽会，怎比真如见一回。」|思ひつつ寝ればや人の見えつらむ 夢と知りせば覚めざらましを}}給予新海誠許多劇情上的發想<ref name="VisualGuidePage58"/><ref name="toho">{{cite news|url=http://www.toho.co.jp/movie/news/1512/04kiminonaha_hk.html |script-title=ja:「君の名は。」製作発表記者会見 |publisher=[[東寶|東寶]] |language=ja |date=2015-12-10 |accessdate=2016-10-17 }}</ref>。而其它諸如{{link-jp|押見修造|押見修造}}的《{{link-jp|我在麻理體內|ぼくは麻理のなか}}》、[[高橋留美子|高橋留美子]]的《[[亂馬½|亂馬½]]》、平安時代小說{{nihongo|《真假鴛鴦譜》|とりかへばや物語}}、澳洲小說家[[格雷格·伊根|格雷格·伊根]]短篇合輯《Oceanic and Other Stories》{{notetag|僅在日本發行的合輯。}}裡頭一篇名為〈保險庫〉（The Safe-Deposit Box）的故事等等以性別轉換、或身份交換題材漫畫小說皆有作為新海誠的參考題材<ref>{{cite book |script-title=ja:『君の名は。 公式ビジュアルガイド』 |publisher={{lang|ja|[[角川書店|角川書店]]}} |location=日本 |isbn=978-4-04-104780-4 |date=2016-08-27 |script-chapter=ja:〈COLUMN1 入れ替わりから見えてくる面白さ〉|page=48|language=ja}}</ref>。

新海誠表示出道14年間（{{as of|2016}}）不斷嘗試錯誤，像《[[追逐繁星的孩子|追逐繁星的孩子]]》或《[[言葉之庭|言葉之庭]]》等等製作中每週也會聽取周圍對劇本的看法；此次特別重視劇本，以方向性強的製作人川村元氣為中心、徹底聽從其意見，並花了半年跟製作團隊開會改良。例如影片開頭三葉的糸守生活不能多於15分鐘，就得切換到瀧那邊以免枯燥；劇情數起{{nihongo|高潮|ピーク|climax}}<!--日、英訪談所使用的同義詞-->中，黃昏之時相見於原始劇本是兩個分開場景，整合成同一橋段輪流顯現等，皆出自川村的建議。自承過去長篇作品《[[雲之彼端，約定的地方|雲之彼端，約定的地方]]》、《追逐繁星的孩子》都掌握不了兩小時片長內容，但這回頭一次有掌控的感覺。<ref>{{cite interview|url=http://animeanime.jp/article/2016/08/26/30176.html|title={{lang|ja|「君の名は。」新海誠監督インタビュー 　40代の仕事としてスタートラインにある映画になった}}|author=|publisher={{lang|ja|アニメ！アニメ！}}|language=ja |date=2016-08-26 |accessdate=}}</ref><ref>{{cite interview|url=http://mainichi.jp/articles/20161020/k00/00m/040/119000c|title={{lang|ja|新海監督「価値観訴える映画、作りたくない」}}|author=|publisher={{lang|ja|[[毎日新聞|毎日新聞]]}}|language=ja |date=2016-10-20 |accessdate=}}</ref><ref>{{cite interview|url=http://www.otakunews.com/article.php?story=2459|title=An Interview with Makoto Shinaki Director of Your Name |author=|publisher=Otaku News|language=en |date=2016-11-15|accessdate=}}</ref><ref name=ann>{{cite interview|url=http://www.animenewsnetwork.com/feature/2016-12-20/.110150|title=Interview: Makoto Shinkai |author=|publisher=[[動畫新聞網|ANN]] |language=en |date=2016-12-20|accessdate=}}</ref><ref>{{cite web |url=http://lite-ra.com/2016/12/post-2797.html |script-title=ja:新海誠に「気持ち悪いです」と言い放って怒らせた！『君の名は。』プロデューサー川村元気の仕事術 |work=LITERA |author={{lang|ja|新田樹}} |language=ja |date=2016-12-24 |accessdate=}}</ref>

起初影片的暫定片名包含有{{nihongo|《若知是夢（暫）－男女取替早物語》|夢と知りせば（仮）－男女とりかえばや物語}}{{r|yno}}{{r|anata}}{{notetag|顯然也參考平安時代小說{{nihongo|《真假鴛鴦譜》|とりかへばや物語}}名稱。「取替早物語」則是日本後世考據便於表記的漢字，並非完全如其漢字表意。}}、和{{nihongo|《分身戀人》|かたわれの恋人}}<ref name=at>{{cite interview|url=http://www.animatetimes.com/news/details.php?id=1472453958|title={{lang|ja|映画『君の名は。』新海誠監督インタビュー}}|author=|publisher={{lang|ja|アニメイトタイムズ}}||language=ja |date=2016-08-30 |accessdate=2016-10-16}}</ref>等等<ref>{{cite web |url=http://kai-you.net/article/40754 |script-title=ja:『君の名は。』監督が、時系列と観客の感情グラフを公開　大ヒット作の凄まじい考察に反響 |work=KAI-YOU.net |author={{lang|ja|ふじきりょうすけ}} |language=ja |date=2017-04-25 |accessdate=}}</ref>，影片製作人之一的[[伊藤耕一郎|伊藤耕一郎]]提議使用「你的名字」<ref>{{cite book |script-title=ja:『君の名は。 公式ビジュアルガイド』 |publisher={{lang|ja|[[角川書店|角川書店]]}} |location=日本 |isbn=978-4-04-104780-4 |date=2016-08-27 |script-chapter=ja:［監督］新海誠 インタビュー|page=59}}</ref>。雖然新海誠等人一度對於「你的名字」和同樣原名為「{{lang|ja|君の名は}}」的名影劇《[[請問芳名|請問芳名]]》撞名而無法決定，在重讀劇本下才確定此建議片名與內容相符，且加上句號要跟《請問芳名》有別。<ref name=at/>

=== 設計 ===
擔任本次作品角色設計的[[田中將賀|田中將賀]]是過去新海誠在動畫《十字路口》合作的夥伴，因當時田中將賀另需要參與別部劇場動畫《[[好想大聲說出心底的話。|好想大聲說出心底的話。]]》製作，而無法額外擔任此片作畫監督一職。新海誠隨後想到[[安藤雅司|安藤雅司]]是自己先前就覺得很好的人選，而試著與對方聯繫<ref name="filmers">{{cite web |url=http://filmers.jp/articles/2016/08/21/post_80/ |script-title=ja:『君の名は。』には、日本のアニメーションの文脈が豊かに含まれている 新海誠監督インタビュー |work=Filmers |language=ja |date=2016-08-21 |accessdate=2016-10-28}}</ref>。

隨後安藤雅司認為講究細膩動作表現的自己；與角色設計上較美型的田中將賀合作會是個有趣嘗試便接下此任務<ref name="filmers"/>。在製作時兩人也有著職務互相切換情況，如安藤雅司另處理了影片裡男女主角於身份互換情況下時；在神情與姿態模樣方面的角色設計<ref>{{cite journal |script-title=ja:INTERVIEW2_作画監督&キャラクターデザイン |journal=月刊MdN |volume=10 |publisher=MdN Corporation |pages=32 |location=日本 |date=2016-09-06}}</ref>，田中將賀亦擔任影片開頭段落部份的作畫監督<ref>{{cite journal |script-title=ja:INTERVIEW2_作画監督&キャラクターデザイン |journal=月刊MdN |volume=10 |publisher=MdN Corporation |pages=38 |location=日本 |date=2016-09-06|language=ja}}</ref>。

=== 音樂 ===
負責影片音樂的[[RADWIMPS|RADWIMPS]]為新海誠喜愛的樂團，他們是在動畫發行前2年期間接收到影片策劃{{link-jp|川村元氣|川村元気}}的電子郵件邀請而促成合作<ref name="VisualGuide78Page">{{cite book |script-title=ja:『君の名は。 公式ビジュアルガイド』 |publisher={{lang|ja|[[角川書店|角川書店]]}} |location=日本 |isbn=978-4-04-104780-4 |date=2016-08-27 |script-chapter=ja:［音楽］RADWIMPS インタビュー|page=78|language=ja}}</ref>。

初期新海誠的便提出需要26首左右曲子<ref name="VisualGuide78Page"/>，其中包含插入歌、片尾曲及其它配樂多數由主唱[[野田洋次郎|野田洋次郎]]完成，而樂團吉他手[[桑原彰|桑原彰]]及貝斯手[[武田祐介|武田祐介]]也分別擔咖啡廳場景或前往飛驒探索真相等段落情節配樂<ref name="VisualGuide80Page">{{cite book |script-title=ja:『君の名は。 公式ビジュアルガイド』 |publisher={{lang|ja|[[角川書店|角川書店]]}} |location=日本 |isbn=978-4-04-104780-4 |date=2016-08-27 |script-chapter=ja:［音楽］RADWIMPS インタビュー|page=80|language=ja}}</ref>。

新海誠是先將[[分鏡圖|分鏡圖]]交給RADWIMPS觀看來讓對方參考做出配樂，並且會特別配合對方創作的曲子內容來截短或增加動畫段落長度<ref name="VisualGuide79Page">{{cite book |script-title=ja:『君の名は。 公式ビジュアルガイド』 |publisher={{lang|ja|[[角川書店|角川書店]]}} |location=日本 |isbn=978-4-04-104780-4 |date=2016-08-27 |script-chapter=ja:［音楽］RADWIMPS インタビュー|page=79|language=ja}}</ref>。而新海誠另曾反應出之前RADWIMPS於2006年發表的單曲{{nihongo|〈兩人的事情〉|ふたりごと}}；裡頭的一句歌詞{{nihongo|「就把一生唯一的[[曲速引擎|空間穿越]]用在這吧」|一生で一度のワープをここで使うよ}}是個能與此篇動畫世界觀連結的內容，而讓RADWIMPS創作能較好拿捏方向<ref name="VisualGuide79Page"/>，雙方花了一年多在調整音樂和劇本上{{r|ann}}。

對於野田洋次郎而言，在片尾曲{{nihongo|〈沒有怎麼樣〉|なんでもないや}}中{{nihongo|「再多一點點、只要再多那一點點就好」|もう少しだけでいい　あと少しだけでいい}}此段描述即將逝去事物美感的歌詞；便是自己的歌能與動畫內容產生共同點時候。而新海誠也特別將該段歌詞以及另首插入歌{{nihongo|〈Sparkle〉|スパークル}}裡的{{nihongo|「華麗地掙脫」|美しくもがくよ}}歌詞引用到自己撰寫的動畫原作小說上<ref name="VisualGuide80Page"/>。

=== 配音 ===
在考慮到劇中男主角瀧要用女性化言詞來對談場面，新海誠想到[[神木隆之介|神木隆之介]]是能應付此情況的配音人選<ref name="toho"/>。[[上白石萌音|上白石萌音]]則是新海誠會面多位女演員後，確定能呈現出女主角三葉形象而被挑選<ref name="as"/>，而她過去在電影《{{link-jp|窈窕舞妓|舞妓はレディ}}》裡的演出也被新海誠認為是個類似三葉故事背景的角色<ref name="toho"/>。

對於神木隆之介而言，在原來身份的瀧、與內在變為三葉的瀧兩者之間，要用不同音調搭配且仍要維持住原來瀧的風格是一件挑戰<ref>{{cite book |script-title=ja:『君の名は。 公式ビジュアルガイド』 |publisher={{lang|ja|[[角川書店|角川書店]]}} |location=日本 |isbn=978-4-04-104780-4 |date=2016-08-27 |script-chapter=ja:〈神木隆之介 x 上白石萌音 対談〉|page=50|language=ja}}</ref><ref name="columnii">{{cite web |url=http://columii.jp/movie/interview/article-1187.html |script-title=ja:『君の名は。』神木隆之介＆上白石萌音＆長澤まさみ　インタビュー |work=Columii |language=ja |date=2016-08-24 |accessdate=2016-10-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20161028220150/http://columii.jp/movie/interview/article-1187.html |archivedate=2016-10-28 }}</ref>。而同樣要應付在劇中兩人交換身體時必須以男孩子的口吻說話的上白石萌音，也參考了一些男子演出影集；其中包含神木隆之介主演的《[[還有第11人！|還有第11人！]]》<ref name="columnii"/>。

電影評論網站方面，基於94條專業評論，[[爛番茄|爛番茄]]對這部電影的新鮮度高達98%，其中92條評論表示贊同，2條評論反對，平均得分為8.2/10<ref name="rottentomatoes">{{cite web |url= https://www.rottentomatoes.com/m/your_name_2016/|title= Your Name (Kimi No Na Wa.)  |date= |language=en|accessdate= 2017-08-15|work= [[爛番茄|爛番茄]]}}</ref>。[[Metacritic|Metacritic]]得分為81分（滿分是100分），基於7條評論<ref name="metacritic">{{cite web |url= http://www.metacritic.com/movie/your-name|title=Your Name |date=|language=en |accessdate=2016-11-27 |work= [[Metacritic|Metacritic]]}}</ref>。

[[吉卜力工作室|吉卜力工作室]]影片製作人[[鈴木敏夫|鈴木敏夫]]提出在觀賞這部片時，法國作家[[安托萬·德·聖-修伯里|安托萬·德·聖-修伯里]]生前曾提到的「人類到最後仍牽掛著的，就是名為『希望』這樣的毛病。」這句話好幾次在他腦海裡浮現著<ref name="natalie199275">{{cite web |url=http://natalie.mu/eiga/news/199275 |script-title=ja:新海誠「君の名は。」に鈴木敏夫や岩井俊二、秦基博が称賛コメント |work=[[Natalie_(網站)|Natalie]] |language=ja |date=2016-08-24 |accessdate=2016-10-28}}</ref>。電影導演[[岩井俊二|岩井俊二]]認為新海誠的作品可比[[雷內·馬格利特|雷內·馬格利特]]的畫作《庇里牛斯山的城堡》（The Castle in the Pyrenees），且有著更高的境界，而此片可謂他過去至今集大成的作品<ref name="natalie199275"/>。過去與新海誠在《[[言葉之庭|言葉之庭]]》合作的歌手[[秦基博|秦基博]]表示著對於不知要花上多少構思，才能表現出像本片如此內容感到震撼<ref name="natalie199275"/>。

[[岡田斗司夫|岡田斗司夫]]認為新海誠不採曲高和寡方式，用傻瓜也能看懂、傻瓜看了也會哭泣的這種眾人易理解手法而讓本片熱賣<ref>{{cite web |url=http://originalnews.nico/462 |script-title=ja:「『君の名は。』は”バカでも分かる”作品だからこそヒットした」岡田斗司夫が語る『君の名は。』ヒットの要因 |work=[[niconico|niconico]] |language=ja |date=2016-09-16 |accessdate=2016-10-28}}</ref>。新海誠對此次作品大賣深感偶然，在於年輕人有巨大需求而適時填上這空白，誠屬幸運{{r|do}}。[[NHK|NHK]]節目《[[今日焦點|今日焦點]]》調查30歲以上觀眾，從公開1週後占31.4%，到第14週時已至49.3%，顯示衰退較少、不光年輕人也受中高年齡層歡迎<ref>{{cite web |url=http://www.nhk.or.jp/gendai/articles/3898/1.html |script-title=ja:想定外!?「君の名は。」メガヒットの謎 |work={{lang|ja|[[今日焦點|クローズアップ現代+]]}} |language=ja |date=2016-11-28 |accessdate=}}</ref>。

漫畫家[[江川達也|江川達也]]批評此片以專業角度來看沒有有趣的地方、沒有表現個人特質且缺乏可賣要素；而支持此片影迷對於反方批評的鄙視行為，讓他覺得根本是一部[[法西斯主義|法西斯主義]]電影<ref>{{cite web |url=http://news.livedoor.com/article/detail/12111695/ |script-title=ja:江川達也氏　「君の名は。」に持論「プロから見ると全然面白くない」 |work=[[活力门|livedoor]] |language=ja |date=2016-10-06 |accessdate=2016-10-28}}</ref>。導演[[是枝裕和|是枝裕和]]接受專訪，反思日本電影危機則感到該片賣座要素有點堆疊過盛<ref>{{cite web |url=http://gendai.ismedia.jp/articles/-/50258 |script-title=ja:邦画大ヒットの年に是枝裕和監督が「日本映画への危機感」を抱く理由 |author=立田敦子 |work= {{lang|ja|現代ビジネス}} |language=ja |date=2016-11-28 |accessdate=}}</ref>。科幻小說家{{link-jp|山本弘|山本弘 (作家)}}在個人推特表示雖然這是部好影片，但片中對於彗星運行路徑的描述有誤，缺了一筆畫龍點睛效果<ref>{{cite web |url=http://npn.co.jp/article/detail/67951594/ |script-title=ja:大ヒット公開中の映画『君の名は。』で指摘された“演出ミス” |work=REAL LIVE. |language=ja |date=2016-10-06 |accessdate=2016-10-28}}</ref><ref>{{cite tweet |author=山本弘 |user=hirorin0015 |number=770163461011546113 |script-title=ja:『君の名は』を観た。いい映画なんだけど、いい映画なんだけど、いい映画なんだけど、ひとつだけどうしても言わせて。 |language=ja |date=2016-08-29 |access-date=2016-10-04 }}</ref>。

=== 小說 ===
由新海誠所撰寫的原作小說於電影上映前發售；是歷屆新海誠作品的首例。新海誠在書中後記表示，這是在劇本完成、電影製作尾聲時執筆的小說，可視為「小說版」、「電影小說化」、「何者為原作誠屬微妙」<ref>{{cite book |script-title=ja:《小説　君の名は。》 |publisher=[[角川書店|角川書店]] |location=日本 |isbn=978-4-04-102622-9 |date=2016-06-18 |page=254}}</ref>。原作小說在電影上映前便已銷售50萬本，在上映後的9月20日為止則達到102.9萬本<ref>{{cite pressrelease |url=http://prtimes.jp/main/html/rd/p/000002680.000007006.html |script-title=ja:『小説 君の名は。』ついに100万部突破!! 映画『君の名は。』も興行収入91億円突破！！本も映画も記録的超メガヒット!! |publisher=[[角川書店|角川書店]] |language=ja |date=2016-09-20 |accessdate=2016-10-14}}</ref>。{{link-ja|角川翼文庫|角川つばさ文庫}}還推出由{{lang|ja|ちーこ}}繪製插圖、將小說內容標上[[振假名|振假名]]，適合較低年齡層讀者的版本。
