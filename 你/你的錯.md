「'''{{lang|ja|君のせい}}'''」（{{lang|ja|きみのせい}}）是[[the_peggies|the peggies]]的第三張[[單曲|單曲]]，[[2018年|2018年]][[11月7日|11月7日]]由[[日本史詩唱片|日本史詩唱片]]發行。

== 解説 ==
封面曲「{{lang|ja|君のせい}}」是[[北澤ゆうほ|北澤ゆうほ]]為電視動畫「[[青春豬頭少年不會夢到兔女郎學姊|青春豬頭少年不會夢到兔女郎學姊]]」而寫的新作。北澤表示製作時「想要描寫（登場人物的）麻衣和咲太之間好像縮短了又好像沒有縮短的絕妙距離感」<ref name="natalie">{{cite web |url=https://natalie.mu/music/news/297215 |title=the peggies「青春ブタ野郎」OP書き下ろし、11月にニューシングルとして発表 |date=2018-08-28 |website=音楽ナタリー |access-date=2019-01-02}}</ref>。歌曲於2018年8月23日首次披露，同年10月4日開放先行下載<ref>{{cite web |url=https://spice.eplus.jp/articles/205321 |title=the peggies　新曲「君のせい」がアニメ『青春ブタ野郎はバニーガール先輩の夢を見ない』オープニングテーマに、石渡マキコ誕生日ワンマンで初披露 |date=2018-08-28 |website=SPICE |access-date=2019-01-02}}</ref><ref name="musicman">{{cite web |url=https://www.musicman-net.com/artist/79819 |title=the peggies、新曲「君のせい」MV公開 注目の12歳・中島セナが出演 |date=2018-10-03 |website=musicman-net |access-date=2019-01-02}}</ref>。

單曲以期間生產限定盤及通常盤兩種形式發售，前者封面由田村里美繪製；後者則以{{tsl|ja|小山健|小山健}}的漫畫為CD內頁<ref name="natalie"></ref>。MV於動畫舞台[[湘南|湘南]]一帶取景<ref name="musicman"></ref>。
