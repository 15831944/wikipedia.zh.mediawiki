《'''你在看我嗎'''》是台灣歌手[[張惠妹|張惠妹]]的第16張個人專輯，2011年4月5日開始預購，在4月23日正式發行。此專輯由[[林夕|林夕]]、陳鎮川聯合打造，首波主打'''都什麼時候了'''於3月30日在[[HitFM聯播網|HitFM]]首播。

== 黑白廣告宣傳 ==
[[金牌大風|金牌大風]]將都什麼時候了的MV剪輯成7支每段5秒的短版廣告在網路上播出，而這7支廣告因為製作方式前衛，也引起網友熱烈討論，甚至許多網友被廣告開場的黑白無聲影像嚇到，以為是恐怖片而不敢開啟，後來知道影片原來是張惠妹專輯的廣告，消息才陸續傳開，也造成MV廣告迴響不斷，大家也都好奇：「這究竟是『張惠妹』還是『阿密特』？！」也有不少人以為是愚人節整人片段。

== 專輯歌曲介紹 ==
# 第一波主打「都什麼時候了」，MV用黑白畫面拍攝。
# 第二波主打「我最親愛的」，是新專輯裡的一首抒情歌，許多人聽完此曲後不禁感動落淚。
# 第三波主打「渴了」，此曲為廣告[[可口可樂|可口可樂]]125週年的主題曲，阿妹也繼2010年「快樂暢開」之後繼續為[[可口可樂|可口可樂]]代言。
# 第四波同名主打「你在看我嗎」聘請多個外國名導來[[台灣|台灣]]拍攝MV。[[金牌大風|金牌大風]]對「你在看我嗎」這張專輯十分重視，邀請長年在[[英國|英國]]拍攝廣告的[[香港|香港]]導演楊龍澄專程飛來[[台灣|台灣]]為張惠妹拍攝「你在看我嗎」的MV，楊龍澄細膩且色彩濃烈的奇幻影像風格，讓不少品牌紛紛大力贊助，令「你在看我嗎」這支耗費近300萬台幣的MV呈現出國際級的視覺體驗。 
# 第六波主打「他們」，因中國[[騰訊微博|騰訊微博]]的贊助，所以此MV出現不少[[騰訊微博|騰訊微博]]鏡頭。
# 第七波主打〈你和我的時光〉，此曲是由[[蘇打綠|蘇打綠]]的[[吳青峯|吳青峯]]創作，這首歌是張惠妹為了感謝15年來支持她的歌迷而創作，這首歌是此專輯第一首錄音的歌曲。

== 你和我的慶功珍藏版 ==
*[[張惠妹|張惠妹]]第16張國語專輯《'''你在看我嗎'''》正式改版為《'''你在看我嗎（你和我的慶功珍藏版）'''》，此改版專輯多收錄了一首新歌「你和我的時光」。
*另外還新增渴了和快樂暢開的Remix版。
