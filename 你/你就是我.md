《'''你就是我'''》（{{lang-ja|君は僕だ}}）是[[前田敦子|前田敦子]]的第2張[[單曲|單曲]]，於2012年6月20日由[[King_Records|King Records]]發售。

== 概要 ==
單曲的[[廣告標語|廣告標語]]是「只要待在身邊，就能理解」（{{lang|ja|そばにいるだけで、わかる}}）。<ref>{{zh-hant icon}}{{cite web |url=http://www.gold-typhoon.com.tw/music/album/5409 |title=你就是我 Act 1 |publisher=[[金牌大風|金牌大風]] |date=2012 |accessdate=2012-06-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20120629065141/http://www.gold-typhoon.com.tw/music/album/5409 |archivedate=2012-06-29 }}</ref>歌詞中亦有一句「你就是我 只要待在身邊就能理解（{{lang|ja|君は僕だ そばにいるとわかる}}）」。

這張單曲距離上一張單曲《[[Flower_(前田敦子單曲)|Flower]]》發行約一年，為[[前田敦子|前田敦子]]於2012年發行的第1張單曲。單曲分為Act.1、Act.2、Act.3和劇場盤，共4種形式發行；其中Act.1、Act.2和Act.3都附送[[DVD|DVD]]。此外，Act.1、Act.2和Act.3在首輪發行時會附送單曲發行紀念特別演唱會參加申請抽選劵。購入劇場盤可抽選獲得上述演唱會的DVD或生寫真。

前田敦子曾於《[[MUSIC_JAPAN|MUSIC JAPAN]]》<ref>{{ja icon}}{{cite web|url=http://natalie.mu/music/news/70909 |script-title=ja:「MUSIC JAPAN」に前田敦子、柴咲コウ、BIGBANGら5組 |publisher=[[Natalie_(網站)|Natalie]] |date=2012-06-11 |accessdate=2012-06-26}}</ref>、《[[HEY!HEY!HEY!MUSIC_CHAMP|HEY!HEY!HEY!MUSIC CHAMP]]》<ref>{{ja icon}}{{cite web|url=http://natalie.mu/music/news/71286 |script-title=ja:田原俊彦、綾小路翔作詞曲引っさげ今夜「HEY!×3」出演 |publisher=[[Natalie_(網站)|Natalie]] |date=2012-06-18 |accessdate=2012-06-26}}</ref>、《[[MUSIC_STATION|MUSIC STATION]]》<ref>{{ja icon}}{{cite web|url=http://natalie.mu/music/news/71462 |script-title=ja:Mステに桑田、少女時代、KAT-TUN、前田敦子、リンキン |publisher=[[Natalie_(網站)|Natalie]] |date=2012-06-20 |accessdate=2012-06-26}}</ref>等音樂節目宣傳這張單曲，並演唱《你就是我》。

== 歌曲 ==
《你就是我》是電影《[[雅夫#電影|LOVE 雅夫Go!]]》的主題曲。<ref>{{ja icon}}{{cite web|url=http://natalie.mu/music/news/68383 |script-title=ja:前田敦子、新曲が映画「LOVE まさお君が行く！」主題歌に |publisher=[[Natalie_(網站)|Natalie]] |date=2012-04-25 |accessdate=2012-06-26}}</ref>這是前田首次有個人歌曲被用作電影主題曲。<ref>{{ja icon}}{{cite web |url=http://listen.jp/store/musicnews_39384_all.htm |script-title=ja:AKB48・前田敦子、初の映画主題歌 担当に「感無量」 |publisher=Listen Japan |date=2012-06-11 |accessdate=2012-06-26 }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>《右肩》是[[日本電視台|日本電視台]]系電視劇《[[私立馬鹿蘭高校|私立馬鹿蘭高校]]》的片尾曲。<ref>{{ja icon}}{{cite web |url=http://www.daily.co.jp/gossip/johnnys_topics/2012/04/07/0004952642.shtml |script-title=ja:キスマイ＆あっちゃんが主題歌“競演” |publisher=Daily Sports Online |date=2012 |accessdate=2012-06-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20120502025130/http://daily.co.jp/gossip/johnnys_topics/2012/04/07/0004952642.shtml |archivedate=2012-05-02 }}</ref>《繞遠路》（{{lang|ja|遠回り}}）是{{link-ja|丸美屋食品工業|丸美屋食品工業}}的廣告歌曲。<ref>{{ja icon}}{{cite web|url=http://www.kingrecords.co.jp/cs/g/gKIZM-161/ |script-title=ja:君は僕だ＜Act 1＞(マキシ＋DVD複合) |publisher=[[King_Records|King Records]] |date=2012 |accessdate=2012-06-26}}</ref>《Sunday drive》是[[富士電視台|富士電視台]]《[[AKB自動車部|AKB自動車部]]》的插曲。<ref>{{ja icon}}{{cite web|url=http://www.kingrecords.co.jp/cs/g/gKIZM-165/ |script-title=ja:君は僕だ＜Act 3＞(マキシ＋DVD複合) |publisher=[[King_Records|King Records]] |date=2012 |accessdate=2012-06-26}}</ref>

音樂網站hotexpress指《你就是我》沿用了《LOVE 雅夫Go!》的世界觀，是一首有[[AKB48|AKB48]]風格的夏日歌曲。歌詞中「你就是你 隨心所欲地活下去吧」<ref group="註">歌詞原文：{{lang|ja|君は君だ 好きなように生きよう}}</ref>、「我就是我 讓我任性一下吧 逞強的時候 總是有你在」<ref group="註">歌詞原文：{{lang|ja|僕は僕だ 勝手にさせてくれ 強がりのすぐそばに いつだって君がいる}}</ref>能讓粉絲想起對前田的想法以及她應該的位置。<ref>{{ja icon}}{{cite web |url=http://www.hotexpress.co.jp/review/3577/ |script-title=ja:AKB48前田敦子だらこそ成立する、面白くなる表現 |publisher=hotexpress |date=2012 |accessdate=2012-06-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20121226052558/http://hotexpress.co.jp/review/3577/ |archivedate=2012-12-26 }}</ref>

== 音樂影片 ==
單曲的[[音樂影片|音樂影片]]（PV）由[[犬童一心|犬童一心]]擔任導演。<ref>{{ja icon}}{{cite web |url=http://www.sanspo.com/geino/news/20120530/akb12053005040000-n1.html |script-title=ja:あっちゃん、ＭＶで漫才挑戦！相方は鶴瓶長男) |publisher=SANSPO.COM |date=2012-05-30 |accessdate=2012-06-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20120701120637/http://www.sanspo.com/geino/news/20120530/akb12053005040000-n1.html |archivedate=2012-07-01 }}</ref>前田敦子於PV中飾演一個[[漫才|漫才師]]，名字是「鍋子」。演員{{link-ja|駿河太郎|駿河太郎}}、漫才師{{link-ja|內海桂子|内海桂子}}都有參與演出，其中駿河太郎飾演漫才師，名字是「信二」。PV內容描述鍋子跟信二邂逅，然後住在一起，並組成漫才組合去參加比賽的故事。<ref>{{ja icon}}{{cite web |url=http://www.daily.co.jp/gossip/article/2012/05/30/0005095094.shtml |script-title=ja:あっちゃん、ＭＶで鶴瓶長男と夫婦漫才) |publisher=Daily Sports Online |date=2012-05-30 |accessdate=2012-06-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20120703055838/http://www.daily.co.jp/gossip/article/2012/05/30/0005095094.shtml |archivedate=2012-07-03 }}</ref>

== 反應 ==
2012年6月19日（發售日前一天），[[Oricon公信榜|Oricon公信榜]]的估計銷量約為6.6萬張，初登場取得日榜第2位，同時刷新了2012年女性個人歌手的首天銷量紀錄（當時最高紀錄是[[指原莉乃|指原莉乃]]的《[[就是喜歡妳|就是喜歡妳]]》，約5.7萬張）。不過，前田上一張單曲《[[Flower_(前田敦子單曲)|Flower]]》的首天銷量約10.2萬，即首天銷量比上作少。<ref>{{ja icon}}{{cite web|url=http://www.oricon.co.jp/news/rankmusic/2013254/full/ |script-title=ja:【オリコン】前田敦子、ソロ2作連続首位発進！ さしこ抜き2012年女性ソロシングルトップの6.6万枚 |publisher=[[Oricon|Oricon]] |date=2012-06-20 |accessdate=2012-06-26}}</ref>

其後，單曲以首周銷量約13.6萬張取得Oricon公信榜周榜第2位。該周的第1位是[[EXILE|EXILE]]的《[[ALL_NIGHT_LONG|ALL NIGHT LONG]]》。<ref>{{ja icon}}{{cite web|url=http://www.oricon.co.jp/rank/js/w/2012-07-02/ |script-title=ja:2012年06月18日～2012年06月24日のCDシングル週間ランキング（2012年07月02日付） |publisher=[[Oricon|Oricon]] |date=2012 |accessdate=2012-06-27}}</ref>
