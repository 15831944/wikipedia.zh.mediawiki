《你不會孤單》是[[黎明|黎明]]的[[音樂專輯|音樂專輯]]，於2008年7月11日推出，專輯收錄了一首國語歌曲《你不會孤單》<ref>{{Cite web |url = https://mojim.com/twy100171x78x1.htm|title = 你不會孤單歌詞-黎明|publisher =Mojim.com 魔鏡歌詞網|language = 中文}}</ref>及鋼琴演奏版《美好的心靈》。[[黎明|黎明]]以[[聯合國兒童基金會|聯合國兒童基金會]]親善大使身份<ref>{{Cite web |url = https://www.unicef.org.hk/ambassador/mr-leon-lai-bbs-mh/|title = 黎明, BBS, MH - The Hong Kong Committee for UNICEF|publisher =聯合國兒童基金香港委員會|language = 中文}}</ref>，與[[林夕|林夕]]及[[雷頌德|雷頌德]]攜手創作「四川心靈重建計劃」歌曲《你不會孤單》<ref>{{Cite web|url=https://hk.entertainment.appledaily.com/entertainment/daily/article/20080523/11142562|title =黎明、林夕、雷頌德 鐵三角譜歌救兒童| date =2008-05-23|publisher =蘋果日報|language = 中文|access-date =2019-04-12|archive-url =https://web.archive.org/web/20190312235204/https://hk.entertainment.appledaily.com/entertainment/daily/article/20080523/11142562|archive-date =2019-03-12|dead-url =no}}</ref><ref>{{Cite web |url = https://hk.entertainment.appledaily.com/entertainment/daily/article/20080609/11207948|title = 黎明行善兌現承諾|date = 2008-06-09日|author =李珮洳|publisher =蘋果日報|language = 中文}}</ref>，並製成唱片進行慈善義賣籌款<ref>{{Cite web |url = https://life.mingpao.com/general/article?issue=20080609&nodeid=1507284461878|title = 黎明單曲賑災CD金舖發售-通識素材|date = 2008-06-09|author =劉一立|publisher =明報教育網|language = 中文}}</ref>。
