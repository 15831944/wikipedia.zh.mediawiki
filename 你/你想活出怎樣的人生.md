《'''你想活出怎樣的人生'''》（{{lang-ja|君たちはどう生きるか}}）是由日本[[吉卜力工作室|吉卜力工作室]]所製作，由[[宮崎駿|宮崎駿]]執導預計在2020或2021年期間公開的[[動畫電影|動畫電影]]。作品名稱是引用自小說家[[吉野源三郎|吉野源三郎]]在1937年發表的同名著書。<ref>{{cite web |url=https://www.asahi.com/articles/ASKBX5T4ZKBXUCLV008.html |script-title=ja:宮崎駿監督、新作タイトルは「君たちはどう生きるか」 |work=[[朝日新聞|朝日新聞]] |author=小原篤 |language=ja |date=2017年10月28日 |accessdate=2018年1月22日}}</ref>

== 過程 ==
2013年期間，宮崎駿在推出動畫電影《[[風起|風起]]》時，舉辦了一場宣佈自己將不再投入長篇動畫製作的引退聲明。<ref>{{cite web |url=http://www.huffingtonpost.jp/2017/05/19/hayao-miyazaki_n_16698030.html |script-title=ja:宮崎駿監督が引退撤回　スタジオジブリ が新作スタッフ募集「今度こそ、本当に最後」 |work=[[哈芬登郵報|哈芬登郵報]] |author=安藤健二 |language=ja |date=2017年5月19日 |accessdate=2018年1月22日}}</ref>

過了一陣子後，宮崎駿開始想把所過去構思的《[[毛毛蟲波羅|毛毛蟲波羅]]》，弄成在[[吉卜力美術館|吉卜力美術館]]公開的短篇動畫，並且興起想再製作動畫電影的念頭。在宮崎駿撤回引退想法後，吉卜力工作室開始募集人手，讓先前已解散的動畫製作部門再次運作。<ref>{{cite web |url=http://www.itmedia.co.jp/business/articles/1705/19/news097.html |script-title=ja:スタジオジブリ復活　宮崎駿監督「引退撤回」で新人募集 |work=ITmedia, Inc. |language=ja |date=2017年5月19日 |accessdate=2018年1月22日}}</ref>

新作品的名稱在2017年10月底公開，並預計會花費3至4年的時間製作。<ref>{{cite web |url=http://www.sankei.com/entertainments/news/171030/ent1710300008-n1.html |script-title=ja:「君たちはどう生きるか」宮崎駿監督の新作の題名 |work=[[產經新聞|產經新聞]] |language=ja |date=2017年10月30日 |accessdate=2018年1月22日}}</ref>
