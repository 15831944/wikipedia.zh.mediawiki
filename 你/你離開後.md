'''你離開後'''（{{lang-en|'''After You've Gone'''}}）是1918年的[[流行音樂|流行歌曲]]，由{{link-en|特納·林頓|Turner Layton}}作曲，{{link-en|亨利·克萊莫|Henry Creamer}}填詞。1918年7月22日由{{link-en|瑪麗恩·哈里斯|Marion Harris}}錄製，並在{{link-en|Victor Talking Machine Company|Victor Talking Machine Company|Victor}}公司上發行。<ref>{{Cite web |url = http://www.78discography.com/vic18500.html |title = the Online Discographical Project |publisher = the Online Discographical Project |language = en}}</ref>
