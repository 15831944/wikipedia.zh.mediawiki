'''主神教'''是另一邪教组织“[[被立王|被立王]]”派生的一个教派，系基督教新教地方教会运动变种组织。1998年被[[中华人民共和国政府|中华人民共和国政府]]宣布为[[邪教|邪教]]组织加以取缔<ref>{{cite web
|url = http://www.people.com.cn/item/kxmx/kexue18.html
|publisher = [[人民日报|人民日报]]
|title = “主神”是神还是人？——揭开中国第一邪教教主的真面目
|date = 1999年}}</ref>。

== 历史 ==
“主神教”教主[[刘家国|刘家国]]，于1964年10月20日出生于安徽省[[霍邱县|霍邱县]][[牌坊乡|牌坊乡]]，1989年加入被立王组织，教名“怜悯”。1991年，[[刘家国|刘家国]]被派往[[湖南|湖南]][[湘潭|湘潭]]、[[湘乡|湘乡]]发展信徒。

1993年刘家国脱离“被立王”组织，自称“主神”，建立了“主神教”。

1995年，“被立王”被宣布为邪教，遭到取缔，其领袖[[吴扬明|吴扬明]]被处死。此后，刘家国趁机收编原“被立王”的组织，从湖南向外发展。

1996年4月，“主神”刘家国在湖南[[衡山县|衡山县]]、[[衡东县|衡东县]]的两个会场召开“第一次全国代表大会”。刘家国在会上宣布该组织的机构和人员，并对购买枪支、进行武装[[暴动|暴动]]等问题进行了讨论。最后提出的目标是采取“[[农村包围城市|农村包围城市]]”的战略，推翻当今社会的“人的国”，建立以“主神”为首的“[[神的国|神的国]]”<ref>[http://www.people.com.cn/GB/historic/1110/5226.html 1998年11月10日 邪教“主神教”覆灭]</ref>。

=== 取缔和惩处 ===
1998年初，[[中华人民共和国政府|中华人民共和国政府]]决定打击“主神教”；6月11日，[[刘家国|刘家国]]在[[云南|云南]][[曲靖|曲靖]]被捕，10月被判处[[死刑|死刑]]，10月13日被处决。同时，从犯朱爱清被判处有期徒刑17年，剥夺政治权利3年，并处罚金1万元。

中华人民共和国政府将主神教的罪行归纳为三个方面：
# 企图推翻[[中华人民共和国政府|中华人民共和国政府]]，建立“[[神的国|神的国]]”；
# 散布所谓“1999年地球末日”的预言，大肆[[诈骗|诈骗]]信徒财产；
# 奸污女信徒。

== 组织结构 ==
“主神教”共分8个等级，除“主神”[[刘家国|刘家国]]外，还有“在上主”(6人)、“[[四活物|四活物]]”(4人)、“七[[天使|天使]]”(7人)、“省权柄”、“县权柄”、同工以及一般信徒。
