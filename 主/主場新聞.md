《'''主場新聞'''》（{{lang-en|''House News''}}）是[[香港|香港]]的一個新聞網站，於2012年由[[蔡東豪|蔡東豪]]、[[梁文道|梁文道]]、[[劉細良|劉細良]]<ref name="成報">{{cite web |url= http://www.singpao.com/gw/xwa/201207/t20120723_372291.html |title= 劉細良大談港政治核心 |publisher= [[成報|成報]] |accessdate= 2012-07-23 }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>和[[宋漢生|宋漢生]]<ref>{{citenews|url=http://cablenews.i-cable.com/webapps/program_video/index.php?video_id=12135425|publisher=[[有線電視|有線電視]]|title=Money Cafe：網上報紙|date=2012年9月7日}}</ref>共同創辦<ref>{{citenews|url=http://news.sina.com.hk/news/3/1/1/2714187/1.html|publisher=[[明報|明報]]|title=劉細良梁文道有新搞作|date=2012年7月10日|deadurl=yes|archiveurl=https://archive.is/20130107164657/http://news.sina.com.hk/news/3/1/1/2714187/1.html|archivedate=2013年1月7日}}</ref>。提供的消息、博客和原創內容覆蓋[[政治|政治]]、[[財經|財經]]、[[生活|生活]]、[[文化|文化]]、[[自然|自然]]、[[科技|科技]]、[[環保|環保]]、[[LGBT|LGBT]]和一些熱門社會議題等。網站除了設一般文字報道外，亦有不少改造圖片。

《主場新聞》以「我城‧我觀點‧我主場」為口號，形式類似[[美國|美國]]的[[赫芬頓郵報|赫芬頓郵報]]<ref name="am730">{{cite web|url=http://www.am730.com.hk/article.php?article=113865&d=1804 |title= 【am專訪】新政府開局陷危機 劉細良：公眾對港人治港感憂慮 | publisher= [[am730|am730]] |accessdate= 2012-07-25 }}</ref>，「在既有的傳統新聞基礎上，收集、組織、策劃、再送到讀者眼前。」<ref name="House News">{{cite web |url= http://thehousenews.com/about/ |title= 關於我們 |3=  |deadurl= yes |archiveurl= https://web.archive.org/web/20130723040146/http://thehousenews.com/about/ |archivedate= 2013-07-23 }}</ref>2012年7月28日，香港「[[全民行動，反對洗腦，729萬人大遊行|全民行動，反對洗腦，729萬人大遊行]]」的前夕，《主場新聞》表示支持，決定把網站提早上線。<ref>{{citenews|url=http://www.am730.com.hk/article.php?article=117358&d=1822|publisher=[[am730|am730]]|title=阿牛在那霸|date=2012年8月17日}}</ref>

但到2014年7月26日下午，創辦人[[蔡東豪|蔡東豪]]突然在《主場新聞》網頁和[[facebook|facebook]]專頁表示因為他恐懼、他的家人的擔憂、主場新聞一直未達收支平衡、以及在廣告收入跟它的影響力不成比例下從生意角度看不到曙光，決定正式結束運作。<ref name="House News 主場新聞">{{

== 運作模式 ==
《主場新聞》辦公室設於[[觀塘|觀塘]]工廈，有14名全職員工，包括兩名主編、7名記者，每月開支要數十萬。由於網站每日主要靠轉載其他主流媒體報道，並將不同報道整合簡化「加工」推出，方便讀者了解事件，但就因為有時轉載別人報道及圖片時未有註明出處，引起非議。<ref>[http://hk.apple.nextmedia.com/news/art/20140727/18813470 批判政府　轉載博客文章 《蘋果日報》 (2014年7月27日)]</ref>跟其他網媒一樣，由於《主場新聞》成立初期人手緊張，博客文章對營運十分重要，《主場新聞》亦意識到博客的重要性，員工會定期發掘博客，並向在內部推薦，博客獲加盟後，主場員工會揀選、重新包裝整理其文章，再以Facebook及其他方式推介，並且定期邀請出席主場博客聚會。<ref name="信報網站">{{cite web|url= http://www2.hkej.com/instantnews/current/article/404747/%E3%80%8A%E4%B8%BB%E5%A0%B4%E6%96%B0%E8%81%9E%E3%80%8B%E5%85%AD%E4%BB%B6%E4%BD%A0%E5%8F%AF%E8%83%BD%E4%B8%8D%E7%9F%A5%E9%81%93%E7%9A%84%E4%BA%8B |title= 《主場新聞》六件你可能不知道的事 | publisher= [[信報網站|信報網站]] |accessdate= 2012-07-23 }}</ref>對於一個博客來說，雖然沒有稿費，但也能引起關注，令博客文章能接觸更多讀者。<ref>[http://www.hkdailynews.com.hk/NewsDetail/index/408972 網媒每月燒錢數十萬 《新報》 (2014年7月27日)] {{webarchive|url=https://web.archive.org/web/20140809063315/http://www.hkdailynews.com.hk/NewsDetail/index/408972 |date=2014年8月9日 }}</ref>

== 結束運作 ==
2014年7月26日，主場新聞創辦人[[蔡東豪|蔡東豪]]對當前政治鬥爭氣氛感到恐懼，網站生意前景亦看不到曙光，即日起結束運作<ref>{{cite web |url=http://inews.mingpao.com/htm/INews/20140726/gb717290.htm |title=蔡東豪：恐懼誤判 結束主場新聞 |language=zh-hant |publisher=[[明報|明報即時新聞網]] |date=2014-07-26 |accessdate=2014-07-26 }}</ref>。全文：<ref name="House News 主場新聞"/><ref>{{cite web |url=https://www.facebook.com/thehousenews/posts/551166598320991 |title= House News 主場新聞 - Facebook |language=zh-hant |author=主場新聞 |publisher=[[Facebook|Facebook]] |date=2014-07-26 |accessdate=2014-07-26 }}</ref>

{{cquote|給關心主場新聞的人：

各位，主場新聞今天要結束了！

辦主場新聞是出於簡單信念，「為香港做點事」，推動社會前進，同時希望像美國Huffington Post一樣，創造一個成功的媒體生意，打開新局面。

「為香港做點事」，背後沒什麼偉大使命，只是在自己能力範圍內儘量關心社會，做一個正常公民應做的事。我生於1964年，今年50歲，趕上嬰兒潮最後一班車，受惠香港八九十年代經濟起飛，把握了向上流動力的機會，就如普通香港人，年青時埋首事業，到略有小成，想利用自己商界的經驗、人脈及知識，走向社會。

我恐懼

原來今天的香港已經變了，做一個正常公民、做一個正常媒體、為社會做一點正當的事，實在不容易，甚至感到恐懼 — 不是陌生，而是恐懼。由於當前政治鬥爭氣氛令人極度不安，多位民主派人士，被跟蹤、被抹黑、被翻舊賬，一股白色恐怖氛圍在社會瀰漫，我亦感覺到這種壓力。還有，作為一個經常往返內地公幹的商人，我得承認，每次過境都會提心吊膽，但這是我過分疑神疑鬼嗎？那種感覺，根本不可能向外人説得清楚。

令我最不安，是家人也感受到這股壓力，終日替我擔心。隨著社會氣氛逐漸緊張，這股壓力在我身邊蔓延的程度令我日益困擾。在家吃飯，我堅持不開電視，因為我不想面對面跟家人討論社會話題，我知道他們只會愈來愈擔心。家人因我憂慮，我傷心。

我誤判

兩年前，我和幾位朋友一同創辦主場新聞，我們以理性為起點，相信包容是香港最重要價值，以博客和新聞策展為基石，創造全新媒體形式。根據最新數據，主場新聞上月平均每日「獨立瀏覽人次」(Unique Visitor)有30萬人，表現可算理想。從開始，我們當一盤正常生意來做，可是，在不正常的社會及市場氣氛下，主場新聞的廣告收入跟它的影響力，不成比例。主場新聞小本經營(很多熟悉我們的博客可作證)，但創辦至今，每月從未達至收支平衡。最大問題是在可見將來，香港社會氣氛只會更見緊張，從生意角度，主場新聞實在看不到曙光。有人問我，主場新聞有沒有出現抽廣告情況，答案是沒有，從未落，何來抽？香港不單止核心價值被扭曲，市場也被扭曲。

我愧疚

我的恐懼及誤判，源於我曾一度相信香港還是一個正常的地方，一心以為可做一個關心香港的公民、一個相信市場的商人，很明顯，我錯了。在一個不正常的社會和市場，做一個正常的公民和商人，原來竟是錯誤的幻想。

對於兩年來努力拼搏的同事，我很內疚，因為大家年中無休，義無反顧地用行動來支持主場新聞的信念。對於家人的包容，我更內疚，讓妳們為我憂慮了這麼長的時間。對於支持主場的博客、海內外讀者，你們終於發現，原來我也只不過是個普通人。我實在盡了力，也只能走到這麽遠。

主場新聞由即日起正式結束，再見！

——[[蔡東豪|蔡東豪]]，2014年7月26日
|}}

== 結束運作原因 ==
主場新聞另一名創辦人[[梁文道|梁文道]]表示，主場未能繼續營辦主要有兩大原因。首先，主場開初營運資金，大部分是來自於蔡與[[蘋果日報|蘋果日報]]財經版「金融中心」的合作計劃，其撰稿和合作費均投放於主場，涉及金額每月數十萬元，但到2014年初蘋果因被抽廣告而要削減預算5％， 「金融中心」遂由原來星期一至五刊出，變為只在周一刊登，結果投放於主場資金大減。

另一方面，主場每日瀏覽多達30 萬人，是眾多網上媒體之冠。但廣告量一直偏低，大部分是靠蔡東豪爭取回來，客戶包括[[電訊盈科|電訊盈科]]及[[大快活|大快活]]，但估計因蔡是大快活的非執行董事才找到這廣告。[[梁文道|梁文道]]認為肯定與主場的政治立場有關，而且中資機構、以至本港大財團、銀行都對網絡媒體無興趣，仍傾向在傳統媒體落廣告。<ref>[http://news.mingpao.com/20140727/gba1.htm 主場新聞長期虧蝕 宣布結束 瀏覽多廣告少 梁文道：與政治立場有關 《明報》 (2014年7月27日)]</ref>

== 事後 ==
網上媒體《主場新聞》突然宣布結業後，部分在《主場新聞》撰寫網誌的博客，隨即在facebook開設「主場新聞博客群」專頁，期望在此繼續分享文章。亦有網民在facebook發起「自己文章自己救，主場文章再現」行動，召集網民利用Google的庫存功能備份《主場新聞》文章，讓其他讀者可以重溫。<ref>[http://www.metrohk.com.hk/index.php?cmd=detail&id=238236 主場結業網民「自救」《都市日報》 (2014年7月28日)]</ref>及後成立《[[主場博客|主場博客]]》網站目前有百多位前[[主場博客|主場博客]]參與，網站沒有單一主事人，各博客在此上載的文章，文責自負。<ref>[http://thehousenewsbloggers.net/about/ 主場博客] </ref>

2014年12月23日，原創辦人蔡東豪，伙拍[[鍾沛權|鍾沛權]]、[[余家輝|余家輝]]為發起人，另外成立《[[立場新聞|立場新聞]]》，董事會還包括：[[吳靄儀|吳靄儀]]、[[方敏生|方敏生]]、[[何韻詩|何韻詩]]、[[練乙錚|練乙錚]]、周達智等人。
