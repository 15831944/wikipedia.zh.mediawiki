《'''少年花草黄'''》，中国大陆禁片，导演[[崔子恩|崔子恩]]，主演[[邓紫衣|邓紫衣]]、[[王桂峰|王桂峰]]、[[焦乐|焦乐]]。

== 剧情 ==
姐弟俩小峰和文文从小都是同居一室，但是自从暑假一开始，姐姐突然就变了，不允许弟弟小峰到房间内睡觉，这时候，文文开始恋爱了，对象小俊是一个大学生，由于恋姐情节，小峰吃醋决心拆散他们，便找了自己的好友乐乐去追求自己的姐姐文文，乐乐答应了小峰的要求，但是自己也提出了条件，都是小峰每天要给他发短信、牵手逛街，这时候，小峰才知道乐乐是同性恋。日久天长，文文开始发觉男友小俊不对劲，她跟踪后才知道小俊是自己妈妈的情人。

== 花絮 ==
该片涉及[[同性恋|同性恋]]、[[姐弟恋|姐弟恋]]、[[三角恋|三角恋]]等多边题材。

2006年10月，该片远赴[[韩国|韩国]][[釜山|釜山]]参加第11届[[釜山国际电影节|釜山国际电影节]]。<ref>{{cite web |url=http://news.qq.com/a/20061013/000839.htm |title=釜山国际电影节开幕 中韩电影颇有断背山情节 |accessdate=2006年10月13日 |publisher= 腾讯网|archiveurl=http://news.qq.com/a/20061013/000839.htm |archivedate=2012年12月27日 |work= |language=zh-cn}}</ref>

2012年，邓紫衣谈起当年拍摄该片的感受说：“这部电影开拍的时候都没有剧本，崔子恩导演就让我们自由发挥。我从未学过表演，可不知为何，第一次就那么放松，可能真的冥冥之中早有安排吧。”<ref>{{cite web |url=http://ent.163.com/12/0514/14/81FK3JJP00031H2L.html |title=邓紫衣《金鹰报》专访 “影坛乖乖女”十年蜕变 |accessdate=2012-05-14 |publisher=网易 |archiveurl= http://ent.163.com/12/0514/14/81FK3JJP00031H2L.html|archivedate=2012年12月27日 |work= |language=zh-cn}}</ref>
