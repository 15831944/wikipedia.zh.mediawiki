《'''少年毛泽东'''》（英語：Teenage Mao Zedong），(曾用名ː 韶山少年：毛泽东的故事)，是[[中华人民共和国|中华人民共和国]]第一部以[[毛泽东|毛泽东]]为人物题材的[[3D|3D]][[动画电影|动画电影]]，导演[[雷珺麟|雷珺麟]]，纪念毛泽东诞辰120周年的献礼片。<ref>{{cite web|url=http://www.hn.xinhuanet.com/2013-07/08/c_116440600.htm|title=3D动画《韶山少年—毛泽东的故事》将于年内公映|accessdate=2013年7月8日|publisher=新华网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20130717050839/http://www.hn.xinhuanet.com/2013-07/08/c_116440600.htm|archivedate=2013年7月17日}}</ref><ref>{{cite web |url= http://news.ifeng.com/mainland/special/maozedong120/content-3/detail_2013_10/12/30260593_0.shtml|title= 湖南制作3D动画电影《毛泽东的故事》年内上映(图)|accessdate=2013 |publisher= 凤凰网|work= |language=zh-hans}}</ref><ref>{{cite web |url=http://www.china.com.cn/guoqing/2013-10/12/content_30274634.htm |title=动画电影《韶山少年-毛泽东的故事》年内将上映 |accessdate=2013 |publisher= 中国网|work= |language=zh-hans}}</ref>2015年4月30日上映

==剧情==
1893年12月26日，毛泽东诞生在[[湖南省|湖南省]][[长沙府|长沙府]][[韶山冲|韶山冲]]，毛父[[毛贻昌|毛贻昌]]是一个精明、吝啬、暴躁的商人，毛母[[文素勤|文素勤]]慷慨厚道、心地善良，在毛贻昌不在的时候偷偷施舍财物给穷人。8岁时，毛泽东在韶山冲附近的一家私塾读书，一直到13岁，其启蒙老师是文正莹，他教会了毛泽东传统的《[[三字经|三字经]]》、《[[百家姓|百家姓]]》、《[[千字文|千字文]]》等，而毛泽东喜欢读[[清朝|清朝]]禁书《[[精忠传|精忠传]]》、《[[水浒传|水浒传]]》、《[[三国演义|三国演义]]》和《[[西游记|西游记]]》等，敢于打破封建礼教封建思想的束缚，毛泽东从小就从父亲那学会了反抗意识，从母亲那学会了救世济人，为日后的成就打下了基础。
