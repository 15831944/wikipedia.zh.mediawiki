'''少翁'''（{{bd|前2世纪||前117年|catIdx=S}}<ref name=dd>《史记·孝武帝本纪》《史记·封禅书》《汉书·郊祀志》都载少翁伏诛次年，武帝大赦。据《汉书·武帝纪》《资治通鉴》，武帝此次大赦为元鼎元年（公元前116年）五月事。则少翁卒于公元前117年，而王夫人则先他去世一年多。</ref>），[[西汉|西汉]]齐国人、方士。

== 生平 ==
[[汉武帝|汉武帝]][[元狩|元狩]]四年（公元前119年），少翁因善于鬼神之术得见汉武帝。<ref>《[[册府元龟|册府元龟]]》作公元前140年事。</ref>大约五年（公元前118年），汉武帝所宠幸的[[王夫人_(汉武帝)|王夫人]]过世，<ref name=dd/>少翁用[[皮影|皮影]]为汉武帝召唤王夫人的鬼魂，如王夫人生前之貌，武帝从帷帐中望见了王夫人的样子，于是拜少翁为文成将军，赏赐甚多，以客礼礼待。<ref name=SJ12>[[:s:史记/卷012|《史记》卷一十二]]</ref><ref name=SJ28>[[:s:史记/卷028|《史记》卷二十八]]</ref><ref name=HS25>[[:s:汉书/卷025|《汉书》卷二十五]]</ref><ref name=ZZTJ19>[[:s:资治通鉴/卷019|《资治通鉴》卷一十九]]</ref>

少翁向武帝进言，认为神灵未降临的原因是建筑物和服装都不像神物，与神灵不相称。于是画云气车，在胜日（古代五行家称金、木、土、水、火五行相克之日为“胜日”）驾车，辟邪恶鬼。少翁又劝武帝造甘泉宫，其中造台室，画天、地、太一诸鬼神而置祭具，以致意于天神。一年多后，道术衰退，不能引发神迹，就作帛书夹在草里喂牛，<ref>[[颜师古|颜师古]]注</ref>装作不知道，说：“此牛腹中有奇。”杀了牛剖腹，得到帛书，书中所言甚怪，武帝生疑。有人认得少翁笔迹，武帝问之，果然是伪书。于是诛杀少翁，但隐秘其事。<ref name=SJ12/><ref name=SJ28/><ref name=HS25/><ref name=ZZTJ19/>

== 典故 ==
方士[[栾大|栾大]]曾与少翁师出同门。武帝弟胶东康王[[刘寄|刘寄]]的王后是乐成侯[[丁义|丁义]]的姐姐，无子，刘寄死后庶子[[刘贤_(胶东王)|刘贤]]继位为王，丁王后有淫乱行为，和刘贤不合，互相陷害。丁王后得知少翁已死，想取悦武帝，就遣栾大通过丁义求见武帝说以方术。武帝也悔恨少翁早死，可惜没完全用尽其方术，[[元鼎|元鼎]]四年（公元前113年）二月见到栾大后，大悦。栾大说担心自己步文成将军的后尘，这样以后方士都不敢说话了。武帝因隐秘少翁死因，说：“文成是吃马肝而死的（当时人传言吃马肝会致死）。<ref>《[[史记索隐|史记索隐]]》引《[[论衡|论衡]]》：“气勃而毒盛，故食走马肝，马肝杀人。”《汉书·儒林传》：“食肉无食马肝，是也。”</ref>”<ref>[[:s:资治通鉴/卷020|《资治通鉴》卷二十]]</ref>遂成典故“文成食马肝”。后来栾大被封为五利将军，也获罪伏诛。<ref name=SJ12/><ref name=SJ28/><ref name=HS25/>

== 轶闻 ==
《[[汉武故事|汉武故事]]》将少翁记作'''李少翁'''，二百岁，貌若童子，很受武帝信任，说致意于太一就能升天达到[[蓬莱|蓬莱]]，但一年多后，道术未灵验。正逢武帝爱妃[[孝武李夫人|李夫人]]死，少翁自称能招魂，夜间张灯烛、设帷帐、陈酒肉，让武帝在其他帐中远远看见李夫人，却不得近观。[[:s:汉书/卷097|《汉书·外戚列传》]]《册府元龟》亦载少翁招魂李夫人事，[[唐朝|唐朝]]文学家[[谢观|谢观]]亦据此作《[[:s:招李夫人魂赋（以「所思逝魂，伤春荣落」为韵）|招李夫人魂赋（以「所思逝魂，伤春荣落」为韵）]]》。但李夫人去世在少翁死后多年（约公元前100年），疑即招魂王夫人事张冠李戴。

《汉武故事》还载，少翁伏诛一月有余后，从关东回来的使者在漕亭遇到了他，回去说见到了他，武帝生疑；发其棺，无所见，只见竹筒一枚。派人追寻，也无踪迹。

== 后世引用 ==
[[东汉|东汉]][[班固|班固]]《西都赋》：“骋文成之丕诞，驰五利之所刑。” 唐朝[[温庭筠|温庭筠]]《马嵬驿》诗：“甘泉不得重相见，谁道文成是故侯。”[[宋朝|宋朝]][[杨亿|杨亿]]《汉武》诗：“力通青海求龙种，死讳文成食马肝。”

[[汉成帝|汉成帝]][[永始|永始]]三年（前104年），[[凉州|凉州]][[刺史|刺史]][[谷永|谷永]]举少翁、栾大等例子劝阻成帝尊崇鬼神、方术之道，<ref>[[:s:说成帝距绝祭祀方术|:s:说成帝距绝祭祀方术]]</ref>得成帝认可。<ref>[[:s:资治通鉴/卷031|《资治通鉴》卷三十一]]</ref>

[[唐宣宗|唐宣宗]][[大中|大中]]十一年（857年）九月，谏官劝阻宣宗迎接罗浮山处士轩辕集，宣宗对宰相[[崔慎由|崔慎由]]说：“告诉谏官们，即使少翁、栾大复生，也不能迷惑我。”<ref>[[:s:旧唐书/卷018下|《旧唐书》卷一十八下]]</ref>

[[宋真宗|宋真宗]][[天禧_(北宋)|天禧]]年间，[[朱能_(宋朝)|朱能]]献《乾祐天书》。[[给事中|给事中]]、知[[垧州|垧州]][[孙奭|孙奭]]上书举少翁帛书喂牛之事，称朱能为妄言祥瑞的小人。<ref>[[:s:宋史/卷431|《宋史》卷四百三十一]]</ref>
