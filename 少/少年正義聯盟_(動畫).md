《'''少年正義聯盟'''》（{{lang-en|Young Justice}}）及其續作《'''少年正義聯盟：入侵'''》（{{lang-en|Young Justice: Invasion}}）是一部[[卡通頻道|卡通頻道]]推出的動畫系列，創建者為布蘭登·維耶蒂（{{lang-en|Brandon Vietti}}）和格雷葛·韋斯曼（{{lang-en|Greg Weisman}}）。該作與[[DC漫畫|DC漫畫]]公司所推出的同名漫畫系列《[[少年正義聯盟|少年正義聯盟]]》雖然有相同的名稱，但是劇情卻沒有很大的關聯。不過兩作都是側重於一群少年超級英雄所組成的團隊的故事<ref name="YT-GWI1">{{YouTube|id = DzoopkET0m8 |title = DC Animated Showcase: Greg Weisman Interview, Part 1}}</ref><ref name="AG-WW">{{Cite web| last =Weisman | first = Greg | title =Young Justice | work = Ask Greg | date = April 21, 2010 | url =http://www.s8.org/gargoyles/askgreg/search.php?rid=861 |accessdate=July 26, 2010}}</ref>。此外劇中並無使用少年正義聯盟這個名稱稱呼主角們組成的團隊，只有直接稱之為「The Team」（小隊）。他們的主要行動就是協助成人超級英雄所組成的團隊-[[正義聯盟|正義聯盟]]，進行一些隱密任務<ref name="DC-SDCC10">{{Cite podcast | url =http://www.dccomics.com/media/podcasts/DCComics_2010-07-23_WB_Animation_Batman_The_Brave_and_the_Bold_at_SDCC_2010.mp3 | title = WB Animation: Batman: The Brave and the Bold| website =DC Comics | publisher =DC Comics | host =Sam Register | date =July 23, 2010|accessdate=July 27, 2010}}</ref><ref name="DCSE">{{Cite podcast | url =http://www.dccomics.com/media/podcasts/DCComics_2010-07-23_DC_Nation_Special_Edition_at_SDCC_2010.mp3| title = DC NATION Special Edition| website =DC Comics | publisher =DC Comcis | host =Dan Didio | date =July 23, 2010|accessdate=August 19, 2010}}</ref>。本作的故事背景被設定發生在DC多重宇宙被闪点事件重组之前的16號地球（{{lang-en|Earth-16}}），本作亦有推出同世界觀的漫畫系列及電玩遊戲。

第二季結束於2013年3月16日後，儘管獲得粉絲積極反應，但官方仍決定就此收尾。2016年11月7日，華納兄弟宣布，該系列確定推出第三季動畫<ref name="ign-triumphant-return">{{cite news | url=http://www.ign.com/articles/2016/11/07/young-justice-returning-for-season-3 | title=Young Justice Returning For Season 3 | work=IGN | date=November 7, 2016 | accessdate=November 7, 2016 | author=Goldman, Eric}}</ref>。而且第三季將會更換播放平台，從原本的卡通頻道移至[[DC_Universe_(串流媒體)|DC Universe]]和[[Netflix|Netflix]]播出。<ref>https://geek-base.toy-people.com/?p=1285 SDCC 《少年正義聯盟：局外者》公開全新角色造型以及相關細節</ref>

== 劇情大綱 ==
一群由[[羅賓|羅賓]]及[[閃電小子|閃電小子]]和[[水少俠|水少俠]]等為首的超級英雄的小搭檔，基於不滿自己的導師對他們的過度保護。

因此，打算以實際的功績讓導師們刮目相看，卻意外救出由某個秘密實驗室製造的[[超人|超人]][[複製|複製]]體———「超級小子」。

最後，他們的實際作為獲得[[正義聯盟|正義聯盟]]的認同，並提供正義聯盟的舊基地「[[快樂港|快樂港]]」，供這群青少年組成的團隊作為訓練中心及總部使用。

最後，以「水少俠」為隊長帶領閃電小子、羅賓、超級小子及[[火星獵人|火星獵人]]的姪女火星小姐組成了最初的小隊，後來更有[[扎坦娜|扎坦娜]]和火箭等人的加入。

另一方面，反派則是有著[[雷克斯·路瑟|雷克斯·路瑟]]等勢力領袖所組成的「[[光明會|光明會]]」(The Light)擔當。

== 主要人物 ==
; 羅賓／夜翼／[[迪克·格雷森|迪克·格雷森]]（Robin / Nightwing / Dick Grayson）
: 蝙蝠俠布魯斯·韋恩的養子，在第一季以羅賓作為秘密身分（13歲），之後則改以「[[夜翼|夜翼]]」為超級英雄的代號。原本自認為是領導人，但後來自覺能力不足而將這權力讓給水少俠。第二季也有提姆·戴克（Tim Drake）接下羅賓這個秘密身分。如同蝙蝠俠他擅於配合各種道具使用的體術，同時也有高深的[[駭客|駭客]]技巧和偵查能力。

; 水少俠／[[卡德拉·漢姆|卡德拉·漢姆]]（Aqualad / Kaldur'ahm）
: 他是[[水行俠|水行俠]]亞瑟的學徒，也是亞特蘭提斯人。擁有控制水變形成武器的法術能力，背後揹著兩把握柄，可以變形成各種由水組成武器。此外個性冷靜成熟且謙虛，擁有領導天賦的他也是這團隊的領袖。

; 閃電小子／[[華利·威斯特|華利·威斯特]]（Kid Flash / Wally West）
: 他是閃電俠[[巴里·艾倫|巴里·艾倫]]的妻姪。擁有天才的化學天份的他自行複製了當初導致巴里變成閃電俠的意外獲得了神速力。他和巴里一樣有因为神速力而要消耗大量能量的副作用，導致他必須經常進食大量零食補充血糖，因此他有一个專門放零食的口袋。雖然比羅賓大兩歲，但成熟度和他一樣。常搭訕火星小姐。後來和阿提米斯談戀愛。

; 超級小子／康纳·肯特（Superboy / Conner Kent）
: 他是結合超人的氪星人及路瑟的地球人基因製造的克隆體實驗品，被救出後就住在快樂港基地。由於只有一半的[[氪星|氪星]]基因所以他無法擁有像超人一般的能力（如無法使用熱射線），但是依然保有強壯的體力並以跳躍取代飛行。他的個性一开始非常自大倔強，但後來漸漸地改變。第一季中曾和火星小姐談戀愛。

; 火星小姐／梅甘恩·摩爾茲／梅根·摩斯（Miss Martian / M'gann M'orzz / Megan Morse）
: 她是火星獵人的姪女，同樣擁有變形及心電感應等火星能力，不過她後來自白她是一个白火星人。梅根·摩斯這個地球化名原自於她喜歡的一部地球情劇喜劇中女主角的名字。第一季中曾和超級小子談戀愛。

; 阿提米斯／母老虎／阿提米斯·克羅克（Artemis / Tigress / Artemis Crock）
: 她是由[[綠箭俠|綠箭俠]]介紹而加入團隊的，並沒有公開其秘密身分。後來才自白是反派[[運動大師|運動大師]]和[[女猎手_(漫画)|女獵手]]的女兒，她還有個姊姊柴郡貓也是知名反派。她為了希望超出自己的血緣，因此向正義聯盟的成員自薦並獲准以匿名身分加入「The Team」。後來和閃電小子談戀愛。
