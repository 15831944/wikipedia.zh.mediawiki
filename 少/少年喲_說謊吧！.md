《'''少年喲 說謊吧！'''》（{{lang-ja|少年よ 嘘をつけ!}}）是[[走廊奔跑隊7|走廊奔跑隊7]]的第10張<ref group="註">這是以走廊奔跑隊計算。以走廊奔跑隊'''7'''的名字發行的單曲則是第4張。</ref>[[單曲|單曲]]，於2012年5月30日由[[波麗佳音|波麗佳音]]發售。

== 概要 ==
《少年喲 說謊吧！》距離前作《[[希望山脈|希望山脈]]》發行約6個月，是[[走廊奔跑隊7|走廊奔跑隊7]]在2012年發行的第1張單曲。單曲分為通常盤、初回盤A、初回盤B和初回盤C，共4種形式發售。初回盤A、初回盤B和初回盤C附送「走廊奔跑隊7 高級活動參加應徵抽選卷」和「走廊奔跑隊7 戲劇比賽 投票劵」。另外初回盤A附送「原創交換卡片 單人拍攝 全身Ver.」一張（共7種），初回盤B附送「原創交換卡片 單人拍攝 近拍Ver.」一張（共7種），初回盤C附送「替換封面 單人拍攝」一張（共7種）。通常盤則同樣附送「高級活動參加應徵抽選卷」，並附送「全體成員集合合照」一張（共3種）。

這是[[AKB48|AKB48]]及[[SDN48|SDN48]]的前成員[[浦野一美|浦野一美]]在2012年3月暫時加入走廊奔跑隊7後首張發行的單曲。<ref name="7-1+1" />此外，AKB48另一[[AKB48#長期組合|衍生組合]]的[[Not_yet|Not yet]]第4張單曲《[[西瓜BABY|西瓜BABY]]》同樣在5月30日發行，是[[AKB48|AKB48]]首次有衍生組合的單曲同時開始發售。<ref>{{ja icon}}{{Cite news|date=2012-05-10 |url=http://www.oricon.co.jp/news/music/2011346/full/ |title=Not yet新曲「西瓜BABY」は渡り廊下と同時発売 |publisher=[[Oricon|Oricon]] |accessdate=2012-05-10}}</ref>

== 歌曲 ==
《少年喲 說謊吧！》是電影《[[蠟筆小新_風起雲湧！我的宇宙公主！|蠟筆小新 風起雲湧！我的宇宙公主！]]》的主題曲。<ref>{{ja icon}}{{cite web|url=http://natalie.mu/music/news/67457 |title=新生渡り廊下走り隊7、映画クレしん主題歌がCD化 |publisher=natalie |date=2012-04-08 |accessdate=2012-06-04}}</ref>《你在思索》（{{lang|ja|君は考える}}）是第32屆{{link-ja|全國高等學校問答比賽|全国高等学校クイズ選手権}}的加油歌曲。<ref name="7-1+1" />

除了《少年喲 說謊吧！》和《你在思索》，所有歌曲都不是由全部7名成員演唱。<ref name="7-1+1">{{ja icon}}{{cite web|url=http://natalie.mu/music/news/69224 |title=渡り廊下新作にまゆゆ＆浦野ユニット、多田愛佳ソロ曲収録 |publisher=natalie |date=2012-05-11 |accessdate=2012-06-04}}</ref>單曲跟前作一樣收錄了一首獨唱曲，這次為收錄於通常盤的《不坦率》（{{lang|ja|へそが曲がる}}），由[[多田愛佳|多田愛佳]]演唱。

收錄在初回盤A的《姊妹飯》（{{lang|ja|姉妹どんぶり}}）因歌名有性的意味，在網上引起討論。<ref>{{ja icon}}{{cite web|url=http://www.j-cast.com/2012/05/11131902.html?p=all |title=「姉妹どんぶり」ってどんな意味？ 「AKB48」ユニットの新曲名にネット騒然 |publisher=J-CAST |date=2012-05-11 |accessdate=2012-06-04}}</ref>

== 反應 ==
單曲以首周銷量約6.7萬張取得2012年6月11日的[[Oricon公信榜|Oricon公信榜]]周榜第3位。[[Not_yet|Not yet]]《[[西瓜BABY|西瓜BABY]]》得首位，[[AKB48|AKB48]]的《[[仲夏的Sounds_good！|仲夏的Sounds good！]]》得第2位，為Oricon公信榜首次有組合再加上其衍生組合於同一周取得周榜首3位。<ref group="註">若只計算同一藝人或組合，就只有[[關西傑尼斯8|關西傑尼斯8]]曾於2010年1月4日達成此紀錄。</ref>此外，這令[[走廊奔跑隊|走廊奔跑隊]]自出道以來的10張單曲都取得周榜頭10位。<ref>{{ja icon}}{{cite web|url=http://www.oricon.co.jp/news/confidence/2012519/full/ |script-title=ja:【オリコン】AKBグループ史上初のTOP3独占　Not yet・AKB・渡り廊下が1～3位  |publisher=[[Oricon|Oricon]] |date=2012-06-05 |accessdate=2012-06-05}}</ref>
