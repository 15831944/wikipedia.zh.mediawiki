《'''少女☆歌劇Revue Starlight'''》（{{lang-ja|少女☆歌劇 レヴュー・スタァライト}}）是由[[武士道_(公司)|武士道]]和{{tsl|ja|ネルケプランニング|Nelke Planning}}共同策劃的[[跨媒體製作|跨媒體製作]]企劃，推出了[[音樂劇|音樂劇]]和[[電視動畫|電視動畫]]作品<ref>{{Cite web|url=http://www.animatetimes.com/news/details.php?id=1493535372|script-title=ja:『少女☆歌劇 レヴュー・スタァライト』始動！ミュージカル化、TVアニメ化も決定――アニメ×ミュージカルがリンクする|date=2017-04-30|website=animate Times|accessdate=2017-08-14|language=ja}}</ref>。

== 登場人物 ==
=== 主要人物 ===
; {{nihongo|愛城華戀|愛城 華恋（あいじょう かれん）|聲、演：[[小山百代|小山百代]]}}
: 聖翔音乐学园第99期演员育成科学生，学号1号。生日9月27日。对小时候看过的Revue「Starlight」感到着迷，從而走向舞台之道的舞台少女。无论何时都很有精神、声音洪亮，也有些冒失。和兒時交換了「命運」的小光一同參加了Revue選拔。夢想是「與小光一起站上舞台！」。

; {{nihongo|神樂光|神楽 ひかり（かぐら ひかり）|聲、演：[[三森鈴子|三森鈴子]]}}
: 聖翔音乐学园第99期演员育成科学生，学号29号。生日1月8日。华恋的儿时好友，故事開始前在英国的演剧学校留学，天生的舞台少女。和當年一起觀賞「Starlight」的華戀交換了「命運」。为人冷淡，十分反對華戀參加Revue選拔。夢想是「與華戀一起站上舞台」。
: 在英国留学时参加过当地的Revue選拔，并由此知道Revue選拔的后果。之后在长颈鹿的所托下，回国参加日本的Revue選拔。为了保住华恋而自愿承受Revue「Starlight」的惩罚，最终被华恋打破原来的悲惨结局所救。

; {{nihongo|天堂真矢|天堂 真矢（てんどう まや）|聲、演：[[富田麻帆|富田麻帆]]}}
: 聖翔音乐学园第99期演员育成科学生，学号18号。生日7月24日。知名舞台演员与当红歌星的女儿，演剧界的名门出身。拥有完美的身材与高昂的嗓音，虽天赋异禀仍積極进取、精益求精的孤高的舞台少女。「夢想並不是做夢，而是用來實現的」。

; {{nihongo|星見純那|星見 純那（ほしみ じゅんな）|聲、演：[[佐藤日向|佐藤日向]]}}
: 聖翔音乐学园第99期演员育成科学生，学号25号。生日10月1日。学年第一的才女，担任学级委员的优等生，熟知莎士比亞等作家的名言。判断力、分析力优秀，卻也因此容易鑽牛角尖。不顧父母的反對投入舞台，因自己与真矢等人在天分上的差距而感到烦恼的舞台少女。夢想是「想要以主角的身份，在舞台的正中央沐浴在聚光燈下！」。

; {{nihongo|露崎真晝|露崎 まひる（つゆざき まひる）|聲、演：{{tsl|ja|岩田陽葵}}}}
: 聖翔音乐学园第99期演员育成科学生，学号17号。生日5月4日。雖在舞台上能活泼地歌唱起舞，平時卻老实又畏縮的舞台少女。华恋的室友，一直憧憬着她的开朗积极，隨著华恋的儿时好友光的归来，令其感到情感的困惑与复杂。老家是北海道的自家农场，是“故乡之星”，唯一当地考进学园的学生。夢想是「華戀的搭檔！」。
: 因多次受到華戀的鼓勵與幫助而十分憧憬华恋，將華戀視為自己的閃耀，直至小光归来打破原来的关系後演变成嫉妒。最终在Revue中被华恋点明要成为开朗的自己而释怀，接受了小光的存在并同时照顾两人的生活。

; {{nihongo|大場奈奈|大場 なな（だいば なな）|聲、演：[[小泉萌香|小泉萌香]]}}
: 聖翔音乐学园第99期演员育成科学生，学号15号。生日7月12日。溫柔地包容每個人，在第99期同儕之間猶如「媽媽」般的存在。暱稱「BANANA」<ref group="註">「場奈奈」的連音。同时也因为喜欢用香蕉制作食品。</ref>。不論歌舞還是演出都相當出众，如今仍十分喜愛一年级時大家一起出演的「Starlight」。總是关心努力过度的纯那。夢想是「那個時候的舞台，能夠再來一次就好了呢」。
: 因執著於一年級時的「Starlight」，十分抗拒華戀與小光重逢後帶來的改變。第7話揭露她在無數次贏得Revue后冠後輪迴，只為重現當時無法觸及的耀眼與感動。不断轮回重复之下越發迷茫，在長頸鹿的誘導下觸發小光的归来，刺激了华恋的成長。畏懼改變的同時被嶄新的大家所吸引；最终在純那的陪伴下解開了心結并全力投身于第100期的「Starlight」制作。

; {{nihongo|西條克洛蒂娜|西條 クロディーヌ（さいじょう くろでぃーぬ）|聲、演：[[相羽亞衣奈|相羽亞衣奈]]}}
: 聖翔音乐学园第99期演员育成科学生，学号11号。生日8月1日。日法混血兒，擁有豐富的才能，過去曾為童星、登台無數的舞台少女。好胜，入學後初次認知到真矢這個強烈閃耀的存在，而後將她視作對手與追趕的目標。夢想是「超越天堂真矢，由我來成為最棒的明星！」。

; {{nihongo|石動雙葉|石動 双葉（いするぎ ふたば）|聲、演：{{tsl|ja|生田輝}}}}
: 聖翔音乐学园第99期演员育成科学生，学号2号。生日4月17日。香子的青梅竹馬，為了照顧香子而陪同入學的異端的舞台少女。嘴巴雖壞但内心温柔，抱怨之餘仍將香子視為最優先。個子小、體能佳，擅長實戰訓練（？）的武打場面。有一辆自用的摩托。夢想是「只要香子開心，怎樣都無所謂啦」。
: 母親為千華流的出師弟子。兒時曾和香子約定自己會對她予以支持，香子則答應雙葉要讓她最先看到自己在這世上最閃耀的模樣。

; {{nihongo|花柳香子|花柳 香子（はなやぎ かおるこ）|聲、演：[[伊藤彩沙|伊藤彩沙]]}}
: 聖翔音乐学园第99期演员育成科学生，学号22号。生日3月3日。日本舞名人的孫女，拥有卓越的歌谣及舞蹈才能的和风舞台少女。地道的大小姐，生活起居幾乎全靠雙葉打理；總是笑臉迎人，城府其實比誰都深。操著一口漂亮的京都口音。夢想是「世界征服……之類的」。
:實力堅強，卻一度因己身怠惰與雙葉鬧彆扭；經真矢點醒才回想起初心並意識到雙葉的陪伴承載的意義。後與雙葉重新約定兩人要一起成為世界第一。
