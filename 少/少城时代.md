'''少城時代'''，全稱'''北京少城時代文化傳播有限公司'''，是一家位於[[北京|北京]]的演藝經紀公司，业务涉及艺人经纪、唱片制作、演出宣传等领域。该公司由[[中国大陆|中国大陆]]歌手[[张靓颖|张靓颖]]和当时的男友[[冯轲|冯轲]]共同创立。<ref>{{Cite web |url = http://news.hexun.com/2009-05-26/118052794.html |title = 张靓颖承认自开公司当老板 |date = 2009年5月26日 |publisher = 和讯网}}</ref>

[[张靓颖|张靓颖]]约满后，现已退出该公司成立个人工作室。
