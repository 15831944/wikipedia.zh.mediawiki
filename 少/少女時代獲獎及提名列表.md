'''少女時代獲獎及提名列表'''主要列舉韓國女子組合[[少女時代|少女時代]]由出道至今於頒獎典禮的獲獎及提名。

== 2007年 ==
以歌曲〈[[再次重逢的世界|再次重逢的世界]]〉出道。出道曲在一週的放送數目共247次而得一位，更登上歌手搜索榜第一而引起關注。推出的首張正規專輯《[[少女時代_(專輯)|GIRLS' GENERATION]]》中，同名主打歌〈少女時代〉是翻唱歌手李承哲的高人氣的歌曲，重新編曲的〈少女時代〉發布後即引起大眾關注，獲得多個音樂節目一位。成為2007年新人歌手單曲專輯銷量中冠軍，更獲得年末多個頒獎禮的新人獎。

== 2009年 ==
經過九個月空白期後，少女時代於1月推出迷你一輯《[[Gee_(專輯)|Gee]]》，MV首日公開後瀏覽次數達100萬次，MV中穿著的緊身牛仔褲和白色T恤衫，及「螃蟹舞」在當時成為熱潮。音源發佈兩天就佔據音源排行榜一位。更在Mnet排行榜及KBS《[[Music_Bank|Music Bank]]》中分別創下連續9週獲得一位，打破了不少紀錄，在泰國音樂排行榜也是連續6週佔據一位。歌曲〈Gee〉在電台和無線電廣播、有線廣播合共總播放1,536次，更憑此曲獲得多個音樂頒獎禮的大賞榮譽。
6月再以歌曲〈[[Genie_(專輯)|說出你願望 (Genie)]]〉回歸，在各排行榜獲得一位，專輯首周銷售達5萬張（打破並遠遠超過《Gee》保持女歌手的專輯銷量），總銷量亦超過10萬張，成為過去三年唯一連續三次唱片銷量超過10萬張的女歌手。更於年末在首爾展開首次單獨亞洲巡迴演唱會。

== 2010年 ==
少女時代在2009年獲得大成功，令人氣急升，成為「國民女團」。2010年年初再以正規一輯《[[Oh!_(專輯)|Oh!]]》回歸，隨即席卷多個排行榜一位。少女時代決定8月推出日本單曲《[[Genie_(單曲)|GENIE]]》正式出道，10月再推出單曲《[[Gee_(單曲)|Gee]]》，少女時代熱潮不僅在日本大眾文化界，還受到了經濟界的關注。10月，官方網站公開新專輯概念照，以迷你三輯《[[Hoot|Hoot]]》驚喜回歸韓國樂壇。

== 2011年 ==
在韓國結束《[[Hoot|Hoot]]》的活動後，2011年大部分時間進行日本活動，發行單曲《[[Mr._Taxi/Run_Devil_Run|MR.TAXI]]》、推出日本專輯《[[Girls'_Generation_(日本專輯)|GIRLS' GENERATION]]》及進行日本巡迴演唱會；日本專輯的銷量更獲得百萬銷量認證。下半年以專輯《[[The_Boys_(專輯)|The Boys]]》回歸，更推出英文版〈[[The_Boys_(單曲)|The Boys]]〉，翌年遠赴美國參加脫口秀《大衛深夜秀》和《Live! with Kelly》宣傳，並獲得音樂界和觀眾一定的迴響，也增加了媒體對其關注度與看好。

== 2013年 ==
1月1日發行專輯《[[I_Got_a_Boy|I Got a Boy]]》，多份具有國際影響力的美國媒體相繼報導，使少女時代成為國際話題<ref>{{cite news |language = zh-tw |url = http://kpopn.com/2013/01/07/159774/ |title =Billboard 對少時新輯樂評 |publisher =kpopn |date = 2013-01-07}}</ref>。歌曲在美國舉辦的第1屆[[YouTube音樂獎|YouTube音樂獎]]上，擊敗多位國際知名歌手獲得「年度最佳MV」，成為首次獲得此項殊榮的亞洲組合。隨後多個美國媒體更以「打敗全世界人氣歌手」為題，讚揚少女時代是名副其實的韓國國家代表，是韓國的國寶組合<ref>{{Cite web |url =http://blogs.wsj.com/speakeasy/2013/11/04/why-girls-generation-and-k-pop-won-big-at-the-youtube-music-awards/ |title =Why Girls’ Generation and K-Pop Won Big at the YouTube Music Awards |date =2013-11-04 |publisher =The Wall Street Journal |language =en}}</ref> <ref>{{Cite web |url =http://edition.cnn.com/2013/11/04/showbiz/youtube-awards-girls-generation/index.html |title =K-pop group Girls' Generation beats Miley, Lady Gaga at first YouTube awards|date =2013-11-04 |publisher =CNN |language =en}}</ref>。

== 2017年 ==
少女時代在2017上半年大多以個人活動為主，成員[[徐玄|徐玄]]先於年初以Solo歌手身份出道，其後更主演電視劇；[[太妍|太妍]]也緊接回歸及舉辦個人巡迴演唱會；[[允兒|允兒]]、[[崔秀英|秀英]]也先後出演電視劇<ref>{{cite web | url=http://www.koreastardaily.com/tc/news/94508 | language=zh | title=少女時代接力！徐玄、潤娥到秀英 陸續通過小螢幕與粉絲見面 | publisher=KSD 韓星網 | date=2017-05-16}}</ref>，[[孝淵|孝淵]]也確定於6月Solo回歸<ref>{{cite web | url=http://www.koreastardaily.com/tc/news/94847 | language=zh | title=少女時代孝淵6/1發行新曲《Wannabe》 San E參與Rap feat | publisher=KSD 韓星網 | date=2017-05-25}}</ref>。組合於8月發佈十周年紀念專輯《[[Holiday_Night|Holiday Night]]》，以雙主打曲〈Holiday〉及〈All Night〉回歸。10月9日，[[Tiffany_(少女時代)|Tiffany]]、[[秀英_(少女時代)|秀英]]及[[徐朱玄|徐玄]]宣佈不再與SM娛樂續約，但不會退團。
