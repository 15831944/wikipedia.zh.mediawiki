{{engname|《少年的你》|Better Days}}，[[曾国祥|曾国祥]]导演电影，由[[周冬雨|周冬雨]]、[[易烊千玺|易烊千玺]]主演。

== 剧情概述 ==
一場高考前夕的校園意外，改變了兩個少年的命運。

陳念性格內向，是學校裡的優等生，努力複習、考上好大學是高三的她唯一的念頭。同班同學的意外墜樓牽扯出一連串不為人知的故事，陳念也被一點點捲入其中 … 在她最無助的時刻，一個叫“小北”的少年闖入了她的世界 ……

大多數的人的18歲是明媚、快樂的，而他們卻在18歲這個夏天提前嚐到了成人世界的漠然。一場祕而不宣的“戰鬥”正在上演，他們將一起守護少年的尊嚴。

== 制作 ==
制作班底来自《[[七月与安生_(电影)|七月与安生]]》中的原班团队，包括导演曾国祥、监制许月珍、编剧林咏琛、李媛、许伊萌，摄影指导余静萍、造型指导吴里璐、声音指导黄铮等。

2018年7月10日在[[重庆市|重庆市]]开机，9月10日正式杀青。

== 宣传 ==
2018年7月23日發佈男女主角的官方宣傳海报<ref>{{cite news|url=http://ent.163.com/18/0724/10/DNFLL96U000380D0.html|title=《少年的你》男女主角曝光 易烊千玺搭档周冬雨|date=2018-07-24|publisher=网易|accessdate=2018-07-24}}</ref>。

2019年1月22日发布超前预告和剧照<ref>{{cite news|url=http://ent.cri.cn/20190122/6cc44024-9b6c-e38f-9461-50ed1214b656.html|title=《少年的你》首发预告 周冬雨易烊千玺露少年锋芒|date=2019-01-22|publisher=中央广电总台国际在线|accessdate=2019-01-26}}</ref>。

== 发行 ==
影片入围了第69届[[柏林电影节|柏林电影节]]水晶熊奖新世代单元<ref>{{Cite web|url=https://www.berlinale.de/en/presse/pressemitteilungen/generation/generation-presse-detail_48660.html|title=Generation 2019: Giving the Oppressed a Voice|accessdate=2019-01-12|work=www.berlinale.de}}</ref>。

原定于2019年2月11日在第69届柏林电影节首映<ref>{{Cite web|url=https://www.berlinale.de/en/programm/berlinale_programm/datenblatt.html|title=Shao nian de ni|accessdate=2019-02-01|work=www.berlinale.de}}</ref>，但2月2日片方宣布因后期制作原因无法按时参加展映<ref>{{cite news|url=https://weibo.com/6544107560/HeQAJkcnyl|title=电影少年的你|date=2019-02-02|publisher=微博|accessdate=2019-02-02}}</ref>，原电影节官网上的上映信息也被撤下。
