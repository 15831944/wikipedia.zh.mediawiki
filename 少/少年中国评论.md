'''少年中国评论'''（{{lang-en|YoungChina}}），简称'''少中评'''（YCA），是[[中华人民共和国|中华人民共和国]]的一个具有[[新左翼|新左翼]]倾向的政经评论网站。该网站自2008年起开始运营，自称其愿景是：“和那些醒着的要进步的人们一道，向着光明的地方行进。”<ref>{{cite web |title=少年中国评论-关于我们|url=http://review.youngchina.org/about |publisher=YoungChina|accessdate=2015-05-04}}</ref> 该网站对于中医和转基因问题与乌有之乡等网站不同，并在女权问题上是持马克思主义式的激进态度。

== 概述 ==
2008年创立时，该网站撰写文章批判中国大陆左翼网站[[乌有之乡|乌有之乡]]的代表人物[[张宏良|张宏良]]，认为张宏良具有强烈的民族主义倾向。

网站在其出版的电子杂志《少年中国文萃》上经常翻译并刊载各国共产党的政治批评文章，对[[西方马克思主义|西方马克思主义]]的哲学、社会学、文学批评理论多有介绍，并十分关注各国[[左翼|左翼]]组织的活动情况。2013年，英国马克思主义经济学家[[保罗·科克肖特|保罗·科克肖特]]授权网站成员翻译其著作《走向新社会主义》。

《[[南方人物周刊|南方人物周刊]]》2015年报道认为少年中国评论试图表达一种不同于主流媒体上任何一派，也不同于外国媒体的独立的左翼论述。<ref>{{cite web |author1=杨潇 |title=左翼青年|url=http://www.nfpeople.com/story_view.php?id=4560 |website=http://www.nfpeople.com/ |publisher=南方人物周刊 |accessdate=2015-05-05 |archive-url=https://web.archive.org/web/20150505080137/http://www.nfpeople.com/story_view.php?id=4560 |archive-date=2015-05-05 |dead-url=yes }}</ref>

2015年10月，该网站的部分成员发布公告宣布开除所有编辑，并在其内部开展了“[[反修正主义|反修正主义]]斗争”。被开除的成员另建了一个“少年中国评论”网站。<ref>[http://review.youngchina.org/archives/12124 YCA与伪YCA修正主义分子的分歧] {{webarchive|url=https://web.archive.org/web/20151030044141/http://review.youngchina.org/archives/12124 |date=2015-10-30 }}</ref>
