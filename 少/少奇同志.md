《'''少奇同志'''》，中国大陆电视剧，导演孙波，主演郭连文、李克俭、王伍福、孙维民、宗利群。

== 剧情 ==
该剧讲述了从1945年到1949年[[国共内战|国共内战]]期间的刘少奇相关事迹。

== 花絮 ==
该剧是“纪念刘少奇诞生100周年”、“纪念[[中国共产党|中国共产党]]建立80周年”献礼剧。<ref name=" cat" >{{cite web |url=http://www.people.com.cn/item/lsq/newfiles/b1200.html |title=系列电视剧《少奇同志》摄制完成 |accessdate=1998 |publisher=人民网 |archiveurl=http://www.people.com.cn/item/lsq/newfiles/b1200.html |archivedate=2013年1月3日 |work= |language=zh-cn}}</ref><ref name="bee " >{{cite web |url=http://www.gmw.cn/01gmrb/1998-11/16/GB/17878%5EGM2-1612.HTM |title= 系列电视剧《少奇同志》在京首映|accessdate=1998年11月16日 |publisher= 光明日报|archiveurl=http://www.gmw.cn/01gmrb/1998-11/16/GB/17878%5EGM2-1612.HTM |archivedate=2013年1月3日 |work= |language=zh-cn}}</ref><ref name=" qwe" >{{cite web |url=http://ent.sina.com.cn/v/48472.html |title=电视剧《少奇同志》第三部明天与观众见面 |accessdate= 2001年6月29日 |publisher=新浪网 |archiveurl=http://ent.sina.com.cn/v/48472.html |archivedate=2013年1月3日 |work= |language=zh-cn}}</ref><ref name=" nba" >{{cite web |url=http://www.people.com.cn/GB/wenyu/223/5724/5727/20010623/495561.html |title=庆祝建党80周年　央视推出《少奇同志》 |accessdate= 	2001年6月23日 |publisher=人民网 |archiveurl= http://www.people.com.cn/GB/wenyu/223/5724/5727/20010623/495561.html|archivedate= 2013年1月3日|work= |language=zh-cn}}</ref>

该剧制作方是[[中国中央电视台|中国中央电视台]]影视部、[[中国银行|中国银行]]、海军政治部电视艺术中心。<ref name=" cat" /><ref name="bee " /><ref name=" qwe" /><ref name=" nba" />

== 评价 ==
中共官方重大革命历史题材影视创作领导小组：“该剧在艺术上既尊重史实，又有艺术创新，保持了它的历史价值和美学品格，朴实无华地塑造了刘少奇同志以及毛泽东、朱德、周恩来、彭德怀等中共领导人，运筹帷幄决胜千里的艺术形象。这部电视剧同时也是一部集党史、军史、中国革命史的生动教材，在反映刘少奇同志的系列影视剧创作上，具有填补空白的意义。”<ref name=" qwe" />
