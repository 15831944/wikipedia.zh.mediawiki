'''拖雷'''（{{mong|m=ᠲᠤᠯᠤᠢ|Ppp=Tului|k=Тулуй}}；意为“[[镜子|镜子]]”；{{bd|1192年|{{NoteTag|[[白寿彝|白寿彝]]《[[中国通史|中国通史]]》认为拖雷生于1192年。理由如下：拖雷生年史料缺载，唯《[[蒙古秘史|元朝秘史]]》第214节有一段记事，谓成吉思汗攻杀塔塔儿人时，有一塔塔儿人逃脱，窜入成吉思汗后营行帐中觅食，乘机劫持了五岁的拖雷，幸被部属救出。这次战争当是指金承安元年（1196）的斡里札河之役，则拖雷应生1192年，此与《睿宗传》所载“寿四十有□（此字缺）”合（按：拖雷死于1232 年），缺字当为“一”。}}|1232年||catIdx=T拖}}）又译'''图垒'''<ref>《[[续资治通鉴|续资治通鉴]]》所译</ref>，是元太祖[[成吉思汗|成吉思汗]]的幼子，排行第四，是成吉思汗最宠爱的儿子。拖雷和正妻[[唆鲁禾帖尼|唆鲁禾帖尼]]生有四子：[[蒙哥|蒙哥]]、[[忽必烈|忽必烈]]、[[旭烈兀|旭烈兀]]、[[阿里不哥|阿里不哥]]。据《[[元史|元史]]》，拖雷一共有子十一人。1227年8月25日至1229年9月13日担任[[大蒙古国|大蒙古国]]（[[蒙古帝国|蒙古帝国]]）[[监国|监国]]，历时二年。

== 生平 ==
1213年，成吉思汗分兵伐[[金朝|金]]，拖雷从其父率领中路军，攻克宣德府，再攻德兴府。拖雷与驸马[[赤古|赤驹]]先登，拔其城。即而挥师南下，拨[[涿州|涿州]]、[[易州|易州]]，残破[[河北|河北]]、[[山东|山东]]诸郡县。1219年，从成吉思汗西征，攻陷[[布哈拉|布哈拉]]、[[撒馬爾罕|撒馬爾罕]]。1221年，分领一军进入[[呼罗珊|呼罗珊]]境，陷马鲁、尼沙不儿，渡搠搠阑河，降也里。遂与成吉思汗合兵攻塔里寒寨。

按照[[蒙古|蒙古]]习俗，幼子继承父业，而年长诸子则分析外出、自谋生计。故成吉思汗生前分封诸子，拖雷留置父母身边，继承父亲所有在斡难和怯绿连的[[斡耳朵|斡耳朵]]、牧地及军队。成吉思汗留下的[[军队|军队]]共有12.9万人。其中10.1万的精銳俱由拖雷继承。

1221年拖雷[[屠杀|屠杀]][[木鹿|木鹿]]（今[[梅尔夫|梅尔夫]]）城中居民，超过100萬人。除去四百个工匠之外，其余人口被屠杀殆尽，城墙被毁。从此木鹿结束了繁荣的历史。

1227年8月25日[[成吉思汗|成吉思汗]]病逝後，由拖雷[[監国|監国]]，称'''也可那颜'''（{{mong|m=ᠶᠡᠬᠡ <br />ᠨᠣᠶᠠᠨ|k=Их ноён|Ppp=Yeqe noyan}}）；意为“大官人”）。直至两年后在选举大汗的[[忽里臺|忽里臺]]時，拖雷和[[察合台|察合台]]等宗王们在1229年9月13日一起推举[[元太宗|元太宗]][[窩闊臺|窩闊臺]]即大汗位。

1231年，与窩闊臺分道伐金，拖雷总右军自陝西鳳翔渡渭水，过宝鸡，入大散关。11月，[[蒙古|蒙古]]军假道[[南宋|南宋]]境，沿汉水而下，经兴元（今[[陕西|陕西]][[汉中|汉中]]）、洋州（今陕西[[洋县|洋县]]）在均州（今[[湖北|湖北]][[均县|均县]]西北）、光化（今湖北[[光化|光化]]北）一带，渡汉水，迂迴北上入金境。1232年初与金军在均州（今[[河南|河南]][[禹县|禹县]]）遭遇。拖雷乘雪夜天寒（有一[[康里|康里]]人作法）大败金将[[完颜合达|完颜合达]]、[[移剌蒲阿|移剌蒲阿]]、[[完颜陳和尚|完颜陳和尚]]於[[三峰山之战|三峰山]]，尽歼金军精锐。此役毕，拖雷与自[[白坡|白坡]]渡河南下的窩闊臺军会合。

1232年[[农历|农历]]九月，[[拖雷|拖雷]]在北返[[蒙古草原|蒙古草原]]途中逝世。據蒙古秘史，[[窩闊臺|窩闊臺]]在一場重要戰爭中得了重疾，為了治愈窩闊臺，拖雷決定犧牲自己。巫師認為，窩闊臺所得的惡疾的病源是由中土的水和土之靈而來。水土之靈不滿蒙古人把中土臣民趕出中土，和不滿蒙古人令中土滿目瘡痍。若以中土的土地，動物和人作祭品，只會令窩闊臺的病情更加惡化，但若是他們願意犧牲家庭成員，窩闊臺便能好過來，於是拖雷主動飮了被詛咒的飲料後死亡。另一說法，是拖雷可能因酗酒過量而死。<ref>Kahn, Paul; Cleaves, Francis Woodman. ''The Secret History of the Mongols'', p.xxvi</ref>

== 尊諡廟號 ==
1251年7月1日，[[元宪宗|元宪宗]][[蒙哥|蒙哥]]即位，[[大蒙古国|大蒙古国]]（[[蒙古帝国|蒙古帝国]]）皇位从[[窝阔台|窝阔台]]家族转入拖雷家族，[[元宪宗|元宪宗]]追尊父亲'''拖雷'''为皇帝，为拖雷追上尊谥庙号，[[庙号|庙号]]'''睿宗'''，[[谥号|谥号]]'''英武皇帝'''。<ref>《元史》卷一百一十五《拖雷传》，相关记载：“宪宗立，追谥曰英武皇帝，庙号睿宗。”节选自 [http://www.guoxue.com/shibu/24shi/yuanshi/yuas_115.htm 《元史·睿宗传》]</ref>

[[至元_(元世祖)|至元]]三年十月十八日（1266年11月16日），[[太庙|太庙]]建成，制尊谥庙号，[[元世祖|元世祖]][[忽必烈|忽必烈]]将父亲'''拖雷'''的[[谥号|谥号]]由'''英武皇帝'''改谥为'''景襄皇帝'''。 <ref>《元史》卷六《世祖本纪三》，相关记载：至元三年冬十月，“丁丑，徙平阳经籍所于京师。更敕牒旧式。太庙成，丞相安童、伯颜言：“祖宗世数、尊谥庙号、增祀四世、各庙神主、配享功臣、法服祭器等事，皆宜定议。”命平章政事赵璧等集群臣议，定为八室。申禁京畿畋猎。”节选自[http://www.guoxue.com/shibu/24shi/yuanshi/yuas_006.htm 《元史·世祖本纪三》]</ref><ref>《新元史》卷八《世祖本纪二》相关记载：至元三年，“冬十月丁丑，徙平阳经籍所于大都。太庙成，平章政事赵璧等集群臣廷议，定太庙为八室。 ”节选自[http://www.guoxue123.com/shibu/0101/00xys/007.htm 《新元史·世祖本纪二》]</ref><ref>《元史》卷七十四 志第二十五，相关记载：至元三年，“冬十月，太庙成。丞相安童、伯颜言：“祖宗世数、尊谥庙号、配享功臣、增祀四世、各庙神主、七祀神位、法服祭器等事，皆宜以时定”。乃命平章政事赵璧等集议，制尊谥庙号，定为八室。烈祖神元皇帝、皇曾祖妣宣懿皇后第一室，太祖圣武皇帝、皇祖妣光献皇后第二室，太宗英文皇帝、皇伯妣昭慈皇后第三室，皇伯考术赤、皇伯妣别土出迷失第四室，皇伯考察合带、皇伯妣也速伦第五室，皇考睿宗景襄皇帝、皇妣庄圣皇后第六室，定宗简平皇帝、钦淑皇后第七室，宪宗桓肃皇帝、贞节皇后第八室。”节选自 [http://www.guoxue.com/shibu/24shi/yuanshi/yuas_074.htm 《元史·祭祀三·宗庙上》]</ref>

[[至大|至大]]二年十二月六日（1310年1月7日），[[元武宗|元武宗]][[海山|海山]]为'''拖雷'''加上尊谥'''仁圣'''，从此之后，拖雷的[[谥号|谥号]]变为'''仁圣景襄皇帝'''。<ref>《元史》卷七十四 志第二十五，相关记载：至大二年“十二月乙卯，亲享太庙，奉玉册、玉宝。加上太祖圣武皇帝尊谥曰法天启运，庙号太祖，光献皇后曰翼圣。加上睿宗景襄皇帝曰仁圣，庙号睿宗，庄圣皇后曰显懿。”节选自 [http://www.guoxue.com/shibu/24shi/yuanshi/yuas_074.htm 《元史·祭祀三·宗庙上》]</ref>《睿宗皇帝加上尊谥册文》<ref>《睿宗皇帝加上尊谥册文》作者为刘赓，册文内容节选自：苏天爵《国朝文类》卷十《册文》。</ref>，内容如下：

== 後嗣 ==
据《元史》，拖雷有子十一人：长子[[蒙哥|蒙哥]]，次子[[忽睹都|忽睹都]]、四子[[忽必烈|忽必烈]]、六子[[旭烈兀|旭烈兀]]、七子[[阿里不哥|阿里不哥]]、八子[[拨绰|拨绰]]（[[不者克|不者克]]）、九子[[末哥|末哥]]、十子[[岁哥都|岁哥都]]、十一子[[雪别台|雪别台]]，第三子和第五子失其名。

拖雷和正妻[[唆鲁禾帖尼|唆鲁禾帖尼]]所生的四子皆有所成，[[元宪宗|元宪宗]][[蒙哥|蒙哥]]和[[元世祖|元世祖]][[忽必烈|忽必烈]]相继做过[[大元|大元]]（[[大蒙古国|大蒙古国]]）的帝王，[[蒙元|蒙元]]皇帝由拖雷一系繼承。[[旭烈兀|旭烈兀]]在西亚开创了[[伊儿汗国|伊儿汗国]]，1259年[[蒙哥|蒙哥]]去世后，[[阿里不哥|阿里不哥]]在1260年在蒙古本土的[[庫力臺|庫力臺]]大會被部分[[王公|王公]]推举即位，并和[[忽必烈|忽必烈]]争位达四年之久。

拖雷可考的女儿有二：赵国公主[[独木干|独木干]]，下嫁[[汪古部|汪古部]][[聂古得|聂古得]]；鲁国公主[[也速不花|也速不花]]，下嫁[[弘吉剌部|弘吉剌部]][[斡陈|斡陈]]。

== 金庸小說中的拖雷 ==
