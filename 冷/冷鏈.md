'''冷鏈'''（{{lang-en|cold chain, cool chain}}）全稱為'''冷凍冷藏供應鏈'''，是一項控制[[溫度|溫度]]的[[供應鏈|供應鏈]]系統，從原材料供應、食品工廠內生產、貯藏運輸至販賣銷售等物流環節，維持產品在一定的低溫範圍內，以延長和確保產品的[[保質期|保存期限]]。冷鏈貨物一般具有易腐爛的特性，常見的冷鏈產品包括：[[農產品|農產品]]、[[海鮮|水產品]]、[[冷凍食品|冷凍食品]]、[[膠捲|攝影膠捲]]、[[化學品|化學品]]、易受温度影响的电子元件以及[[藥物|藥物]]等<ref>{{cite web |title = 冷鏈介紹 |url = http://www.cclc.nkfust.edu.tw/files/11-1145-8727.php |website = [[國立高雄第一科技大學|國立高雄第一科技大學]]冷鏈認證中心 |accessdate = 2018-07-05}}</ref>。冷鏈的概念最早是由[[冷王|冷王]]（Thermo King）的創辦人{{link-en|弗雷德里克·麥金利·瓊斯|Frederick McKinley Jones}}提出，冷王也是國際最知名的冷鏈大廠。<ref>{{cite website |title = 果蔬從產地到餐桌 無縫銜接的冷鏈技術 |url = https://www.logisticnet.com.tw/newsCaseRunDetail.asp?id=148 |publisher = 物流技術與戰略雜誌 |date = 2015年5月5日 |accessdate = 2018-07-05}}</ref>

依據溫度範圍，可區分為冷藏（chilled，2℃）、冷凍（Frozen，-18℃）和速凍（deep-frozen，-29℃）冷鏈系統。<ref>{{cite journal |author = 徐志宏、吳少雄、鄒伯衡 |title = 冷鏈市場現況與未來藍海市場的發展 |url = https://www.logisticnet.com.tw/publicationArticle.asp?id=379 |journal = 物流技術與戰略雜誌 |date = 2016年4月 |issue = 80 |accessdate = 2018-07-05}}</ref>

[[冷鏈品質指標|冷鏈品質指標]]（CCQI）是冷鏈協會（CCA）所訂定的統一規範準則。<ref>{{cite website |author = 陳慧娟 |title = 推動台灣低溫鏈產業供應鏈整合 |url = http://www.talm.org.tw/uploads/newsFile/20081212-111555-0.pdf |publisher = 工業技術研究院 |website = 中華民國物流協會 |accessdate = 2018-07-05}}</ref>
