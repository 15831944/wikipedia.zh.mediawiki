《'''冷然之天秤'''》是由[[Idea_Factory|Idea Factory]]旗下少女遊戲品牌{{link-ja|Otomate|オトメイト}}開發的[[PlayStation_Vita|PlayStation Vita]]平台{{link-ja|少女遊戲|乙女ゲーム}}。2016年4月21日推出第1作《帝都幻惑綺譚》，2017年9月21日推出第2作《黑百合炎陽譚》。改編電視動畫2018年4月8日開始播出，由[[ZERO-G|ZERO-G]]負責動畫製作<ref>{{cite web|title=Nil Admirari no Tenbin Anime's Teaser Video Reveals Spring 2018 Premiere, Cast, Staff|url=https://www.animenewsnetwork.com/news/2017-09-03/nil-admirari-no-tenbin-anime-teaser-video-reveals-spring-2018-premiere-cast-staff/.120880|website=Anime News Network|language=en|accessdate=2018年1月13日|date=2017年9月3日}}</ref>。

== 故事簡介 ==
故事背景為大正25年，在大地震之後，發展著和洋相互混合的獨特文化的帝都東京。
隨著近代化的發展與印刷出版的書的數量的增加，接連發生了一系列無法解開的事件。
寄宿著作者靈魂和感情，對讀者產生巨大影響的「稀有之物」。
與收集和保護此類書籍的機構——「帝國圖書情報資產管理局」（通稱「鴟鵂」）相關聯的角色們。
他們相遇，相聯繫，命運的天秤開始擺動。

== 登場角色 ==
; {{nihongo|久世鶇|久世 ツグミ|聲：[[木村珠莉|木村珠莉]]}}
: 久世家的大小姐，待人處事都十分溫和，對於和異性相處容易感到害羞。
:因為弟弟鶲差點葬身火海，收到了刺激的啟發，因而能夠看見「稀有之物」所散發的「靈韵」。
; {{nihongo|尾崎隼人|尾崎 隼人（オザキ ハヤト）|聲：[[梶裕貴|梶裕貴]]}}
: 鶇在鴟鵂的同事。其真實身分為鶇的未婚夫，八代家的少爺八代隼人。
:常說出些奇怪的話，說話較不經過大腦。
:在平時其實能夠看出，常常在鶇的身邊守護著她。
; {{nihongo|鴻上滉|鴻上 滉（コウガミ アキラ）|聲：[[岡本信彦|岡本信彦]]}}
: 鴟鵂的一員。為烏鴉的間諜。無表情的面癱，很難看出來，其實是個很容易帶出感情的人。在鴟鵂時話十分稀少，偶爾只有在面對鶇時才會有多一點情緒。
; {{nihongo|星川翡翠|星川 翡翠（ホシカワ ヒスイ）|聲：[[逢坂良太|逢坂良太]]}}
: 鴟鵂的一員。眼睛是漂亮的的翡翠綠及紅色的異色瞳，紅色眼睛是能使用火能力的證明。母親是橫濱的藝妓，因此從小在妓院長大，一次差點遭受侵犯，而開啟了能使用火的能力。
; {{nihongo|鵜飼昌吾|鵜飼 昌吾（ウカイ ショウゴ）|聲：[[木村良平|木村良平]]}}
: 日本首相的兒子，因被靈韵影響，差點自殺身亡，被救起來後到鴟鵂休養。明顯被寵慣的大少爺，傲嬌的個性令旁人拿他沒轍。
; {{nihongo|汀紫鶴|汀 紫鶴（ミギワ シヅル）|聲：[[鈴村健一|鈴村健一]]}}
: 著名的小說作家，偶爾會跑到街上幫人賣金魚。有點謎樣的人物。因某種原因，借住在鴟鵂。
; {{nihongo|鷺澤累|鷺澤 累（サギサワ ルイ）|聲：[[櫻井孝宏|櫻井孝宏]]}}
:帝都大學的醫學生。為「迦具土」的首領。偶爾會手製一些萬花筒拿出去賣。
