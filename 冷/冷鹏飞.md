'''冷鹏飞'''（{{bd|1933年|1月||}}）[[湖北|湖北]][[浠水|浠水]]人，[[中国人民解放军少将|中国人民解放军少将]]，“战斗英雄”<ref name=shiji>{{Cite web |url =http://military.cnr.cn/gz/20170613/t20170613_523797996_10.html  |title =“八一勋章”颁授对象初步候选人简要事迹  |author =  |date =2017-06-13  |publisher = 央广军事 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1954年加入[[中国共产党|中国共产党]]<ref name=huaxia>{{Cite web |url = http://www.huaxia.com/lasd/hxrwk/ddrw/dl/2003/09/491295.html |title = 冷鹏飞 |author =  |date =2003-09-15  |publisher =华夏经纬网  |language =  |accessdate =  |quote =  }}</ref>。1956年2月入伍<ref name=shiji/>。曾任排长、连长、营长<ref name=huaxia/>。1969年3月2日，在[[中国人民解放军|中国人民解放军]]同[[苏联红军|苏联红军]]进行的[[珍宝岛事件|珍宝岛战斗]]中，营长冷鹏飞组织炮火还击，摧毁了苏军指挥车、卡车各1辆，同边防巡逻队协同驱逐了入侵的苏军。1969年3月15日凌晨4时许，苏军60多人在6辆[[装甲车|装甲车]]掩护下，企图偷袭中国人民解放军驻守[[珍宝岛|珍宝岛]]的部队。冷鹏飞奉命率1个加强排登岛，同苏军对峙。8时许，苏军猛烈进攻中国人民解放军守岛分队，营长冷鹏飞作为珍宝岛上中国人民解放军最高指挥员，指挥部队依托有利的地形，以多路兵力分割苏军，派出[[机枪|机枪]]班及[[火箭筒|火箭筒]]手抵近射击，击毁苏军2辆装甲车，击退了苏军的第一次进攻。9时许，苏军出动6辆[[坦克|坦克]]及5辆装甲车，企图将中国人民解放军守岛分队前后夹击。冷鹏飞采取“往前靠、放近打”的战法，指挥中国人民解放军守岛分队击毁苏军2辆装甲车、击伤苏军1辆坦克。冷鹏飞在战斗中负重伤，左臂被打断，但他用树枝夹绑住左臂继续坚持战斗，指挥守岛部队与苏军激战9小时，顶住了苏军的6次炮袭、3次进攻，以小代价获得了中国人民解放军的胜利<ref name=shiji/>。冷鹏飞立一等功<ref name=huaxia/>。

1969年7月，冷鹏飞被[[中共中央军委|中共中央军委]]授予“战斗英雄”荣誉称号<ref name=shiji/><ref name=huaxia/>。回到部队后，冷鹏飞先后任团长、师副参谋长、师长、副军长。由冷鹏飞钻研创新的“抗击敌坦克进攻系列战法”，对部队训练起了重要指导作用<ref name=qshb>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1771143  |title =  第78集团军看望慰问85岁“八一勋章”获得者冷鹏飞|author =  |date =2017-08-22  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。在退役前担任[[中国人民解放军陆军第二十三集团军|中国人民解放军陆军第二十三集团军]]（中国人民解放军81032部队）副军职调研员<ref name=shiji/>。1993年，冷鹏飞退休。此后20多年间，冷鹏飞先后担任数十所中小学校的课外辅导员，义务为孩子们进行爱国主义教育；自制教育宣传展板巡回各部队作报告、座谈<ref name=qshb/>。

2017年7月，中央军委主席[[习近平|习近平]]签署命令，授予冷鹏飞等10位同志“[[八一勋章_(2017年)|八一勋章]]”<ref>{{Cite web |url = http://news.ifeng.com/a/20170728/51522244_0.shtml |title = 景海鹏、程开甲、王刚等10人被授予“八一勋章” |author =  |date = 2017-07-28 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。2017年7月28日，中央军委颁授“八一勋章”和授予荣誉称号仪式在北京[[八一大楼|八一大楼]]举行，中共中央总书记、国家主席、中央军委主席习近平向冷鹏飞等10位“八一勋章”获得者颁授勋章和证书。这是“八一勋章”首次颁授。冷鹏飞同时获授“冲锋在前、英勇果敢的战斗英雄”荣誉称号<ref>{{Cite web |url = http://news.ifeng.com/a/20170728/51524452_0.shtml |title = 习近平向“八一勋章”获得者颁授勋章和证书 |author =  |date =2017-07-28   |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。2017年8月11日，[[中国人民解放军陆军第七十八集团军|中国人民解放军陆军第七十八集团军]]副军长[[毕毅|毕毅]]代表集团军党委机关到[[大连|大连]]看望了冷鹏飞<ref name=qshb/>。

冷鹏飞1988年被授予少将军衔。是第四、五届全国人大代表<ref name=huaxia/>。
