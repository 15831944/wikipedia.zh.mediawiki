'''談文南號誌站'''位於[[台灣|台灣]][[苗栗縣|苗栗縣]][[造橋鄉|造橋鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[海岸線_(臺鐵)|海岸線]]的鐵路[[臺灣鐵路管理局車站等級#號誌站|號誌站]]。

== 車站構造 ==
*往南為單線，往北為雙線至[[竹南車站|竹南站]]。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#號誌站|號誌站]]，僅供列車交會。
*原由[[談文車站|談文站]]控制，[[談文車站|談文站]]降等後改由[[竹南車站|竹南站]]控制。

== 車站週邊 ==
*[[中港溪|中港溪]]

== 歷史 ==
*1964年1月30日：設站。
