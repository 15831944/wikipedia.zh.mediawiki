== 歷史 ==
*1922年10月11日：「淡文湖驛」開業。
*1954年11月1日：改名為「談文」。<ref>{{cite book |author = 戴震宇 |authorlink =  |title = 《台灣的老火車站》 |publisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>
*1976年5月30日，北上56次對號快車與南下107次平快車於本站附近，因副線對號快車司機員疏忽未讓主線平快車先行，造成與平快車迎面對撞，共29人死亡，141人受傷，兩部機車及六輛車廂損毀。
*1991年3月15日：由三等站降為甲種簡易站，並指定由[[竹南車站|竹南站]]管理。
*2008年1月24日：轉移行車設施，已無交會列車功能。
*2008年3月21日：登錄為苗栗縣定歷史建築。
*2008年4月1日：由甲種簡易站降為招呼站。
*2015年6月30日：啟用多卡通刷卡機。
