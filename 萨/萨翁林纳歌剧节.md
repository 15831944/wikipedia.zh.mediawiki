'''萨翁林纳歌剧节'''（{{lang-fi|Savonlinnan oopperajuhlat}}）是一个每年七月份在[[芬兰|芬兰]]城市[[萨翁林纳|萨翁林纳]]举办，为期约一个月的歌剧节。歌剧演出的地点是位于[[塞马湖|塞马湖]]，始建于1475年的[[奥拉维城堡|奥拉维城堡]]。

== 歌剧节起源 ==
[[File:Macbeth_applause_at_Savonlinna_Opera_festival_in_2007_-_panoramio.jpg|thumb]]
芬兰女高音歌唱家{{tsl|en|Aino Ackté|爱诺·阿克特}}（Aino Ackté）于1912年在奥拉维城堡举办了第一届歌剧节<ref name="historia"/>，之后又接着在1913年、1914年、1916年和1930年举办了歌剧节。在此以后歌剧节活动沉寂了37年。

1965年开始了歌剧节重启的筹备活动，当时市政府和始于1955年的萨翁林纳音乐节共同成立了筹备小组。音乐节有一个附带的歌剧进修课程。1967年歌剧课程小组在奥拉维城堡演出了[[贝多芬|贝多芬]]的歌剧《[[费德里奥|费德里奥]]》。从此之后歌剧节在观众人数和知名度上稳步增长。

从1987年起歌剧节开始邀请其它国家的歌剧院前来演出。2008年[[上海歌剧院|上海歌剧院]]受邀前来演出了[[威尔第|威尔第]]的歌剧《[[奥泰罗|奥泰罗]]》、[[温德清|温德清]]的歌剧《[[赌命|赌命]]》（{{lang-fr|Le Pari}}，{{lang-en|The Wager}}）和[[冼星海|冼星海]]的《[[黄河大合唱|黄河大合唱]]》<ref name="teosluettelo"/><ref name="hs"/><ref name="晨报"/>。

== 2016年节目单 ==
2016年歌剧节的主题是意大利歌剧：[[朱塞佩·威尔第|威尔第]]的《[[奥泰罗|奥泰罗]]》、《{{tsl|en|Falstaff (opera)|法尔斯塔夫 (威尔第)|法尔斯塔夫}}》和《[[麦克白_(歌剧)|麦克白]]》，[[贾科莫·普契尼|普契尼]]的《[[波希米亚人_(歌剧)|波希米亚人]]》，[[温琴佐·贝利尼|贝利尼]]的《[[诺尔玛|诺尔玛]]》，以及[[沃尔夫冈·阿马德乌斯·莫扎特|莫扎特]]的《[[唐·乔望尼|唐·乔望尼]]》。此外还有[[莱奥什·雅纳切克|雅那切克]]的《{{tsl|en|From the House of the Dead|死屋手记}}》（{{lang-cs|Z mrtvého domu}}）<ref name="ohjelmisto2016"/>。

== 2017年节目单 ==
[[File:Olavinlinna_(2017-07-27)_02.jpg|thumb]]
2017年歌剧节在2017年7月7日至8月6日举行。歌剧节为庆祝芬兰共和国独立100周年而演出由芬兰作曲家[[奥利斯·萨利宁|奥利斯·萨利宁]]谱写的两部歌剧：《水中城堡》和《{{tsl|en|Kullervo (Sallinen)|库莱尔沃 (歌剧)|库莱尔沃}}》，其中《水中城堡》将会是在世界范围内的首演。此外歌剧节上还将演出莫扎特的《[[后宫诱逃|后宫诱逃]]》和威尔第的《[[弄臣_(歌劇)|弄臣]]》<ref name="season2017"/>。

像往年一样，歌剧节也邀请了其它国家的歌剧院来演出。这一年邀请的是[[莫斯科大剧院|莫斯科大剧院]]和{{tsl|en|Teatro Real|马德里皇家剧院}}。莫斯科大剧院演出的是[[彼得·伊里奇·柴可夫斯基|柴可夫斯基]]的两部作品：歌剧《{{tsl|en|Iolanta|Iolanta}}》和《[[叶甫盖尼·奥涅金_(歌剧)|叶甫盖尼·奥涅金]]》的音乐会版本。马德里皇家剧院则演出了[[温琴佐·贝利尼|贝利尼]]的《{{tsl|en|I puritani|清教徒 (歌剧)|清教徒}}》<ref name="season2017"/>。

== 2018年节目单 ==
[[File:Savonlinna_Opera_Festival_2018_-_Madame_Butterfly_20180709_185241.jpg|thumb]]
2018年歌剧节在2018年7月6日至8月4日举行。今年演出的歌剧有[[彼得·伊里奇·柴可夫斯基|柴可夫斯基]]的《[[黑桃皇后|黑桃皇后]]》、[[夏尔·古诺|古诺]]的《[[浮士德_(歌劇)|浮士德]]》、[[贾科莫·普契尼|普契尼]]的《[[蝴蝶夫人|蝴蝶夫人]]》和[[朱塞佩·威尔第|威尔第]]的《[[奥泰罗|奥泰罗]]》。

今年邀请的外国歌剧表演团体是意大利的[[普契尼音乐节|普契尼音乐节]]。他们带来的是普契尼的歌剧《[[托斯卡|托斯卡]]》和《[[图兰朵|图兰朵]]》。<ref name="season2018"/>

== 2019年节目单 ==
2019年歌剧节将在2019年7月4日至8月3日举行。演出的歌剧有[[焦阿基诺·罗西尼|罗西尼]]的《[[塞维利亚的理发师|塞维利亚的理发师]]》、[[朱塞佩·威尔第|威尔第]]的《[[弄臣_(歌劇)|弄臣]]》、[[沃尔夫冈·阿马德乌斯·莫扎特|莫扎特]]的《[[后宫诱逃|后宫诱逃]]》。

今年邀请的外国歌剧院是[[斯卡拉大剧院|斯卡拉大剧院]]和[[维也纳人民歌剧院|维也纳人民歌剧院]]。斯卡拉大剧院将演出威尔第的《{{le|强盗 (歌剧)|I masnadieri|强盗}}》，而维也纳人民歌剧院的剧目则是[[小约翰·施特劳斯|小约翰·施特劳斯]]的《[[蝙蝠_(歌剧)|蝙蝠]]》。<ref name="season2019"/>
