'''薩摩亞独立国'''（[[薩摩亞語|薩摩亞語]]：Malo Sa'oloto Tuto'atasi o Samoa，{{lang-en|Independent State of Samoa}}），简称'''萨摩亚'''，是一個[[南太平洋|南太平洋]][[岛国|岛国]]，約位於[[夏威夷|夏威夷]]与[[新西兰|新西蘭]]的中間、[[美屬薩摩亞|美屬薩摩亞]]的西方，為[[波利尼西亞群島|波利尼西亞群島]]的中心。1962年独立。

== 历史 ==
{{main|萨摩亚历史}}

在公元前1000年，萨摩亚便已有人定居。在第二次薩摩亞內戰後，薩摩亞被一分為二，東邊由美國統治成為現[[美屬薩摩亞|美屬薩摩亞]]，而西邊則在1899年成为[[德国|德国]]殖民地。[[第一次世界大战|第一次世界大战]]刚开始，1914年8月，[[新西兰|新西兰]]占领[[德属萨摩亚|德属萨摩亚]]，以防止[[德国|德国]]用它作为[[太平洋|太平洋]]的海军基地。岛上的德国军队没有投降，但是也没有反抗。1919年，依照[[凡尔赛条约|凡尔赛条约]]，德国把西萨摩亚让给[[新西兰|新西兰]]委任统治。

新西兰以[[国联|国联]]和[[联合国|联合国]]名义托管西萨摩亚，直到它1962年1月1日独立，称为「西萨摩亚」；1997年改名「萨摩亚」。西萨摩亚是[[波利尼西亚|波利尼西亚]]岛国中第一个在20世纪独立的国家。

2006年3月31日，萨摩亚议会选举举行，执政的[[人权保护党|人权保护党]]再次赢得大选，赢得议会49席中的30个席位。

隨著薩摩亞人來往和移民[[澳洲聯邦|澳洲]]和新西蘭增加，薩摩亞在2009年9月7日更改[[道路通行方向|道路通行方向]]，從靠右行駛改為靠左行駛，用意是方便進口汽車。2011年12月29日晚，薩摩亞更改標準時間，從[[UTC-11|UTC-11]]改為[[UTC+13|UTC+13]]（[[夏令時|夏令時]]從[[UTC-10|UTC-10]]改為[[UTC+14|UTC+14]]），以消除與[[澳大利亞|澳大利亞]]和新西蘭相差一天的商業障礙。

2017年6月，國會通過修憲，訂[[基督教|基督教]]為國教。

== 政治 ==
{{main|萨摩亚政治}}

1960年制定、1962年1月1日生效的[[宪法|宪法]]规定[[萨摩亚国家元首|国家元首]]由[[萨摩亚议会|萨摩亚议会]]选出，任期5年；首任国家元首为[[终身领导人|终身制]]。除元首外，还设立代表委员会，委员即为副元首，不超过3名；其职能是，一旦[[国家元首|国家元首]]出现空缺或不能行使职务时，代行元首的职权。议会称[[萨摩亚立法大会|立法大会]]，为[[一院制|一院制]]。萨摩亚政府首脑为[[萨摩亚总理|萨摩亚总理]]，实行[[议会制|议会制]]，
总理对议会负责。

== 地理 ==
{{main|萨摩亚地理}}

萨摩亚是一個[[南太平洋|南太平洋]]島國，約位於[[夏威夷|夏威夷]]至[[紐西蘭|紐西蘭]]之間，[[美屬薩摩亞|美屬薩摩亞]]的西方。陸地面積2,831[[平方公里|平方公里]]。各岛山峦起伏，多火山，[[萨瓦伊岛|萨瓦伊岛]]的最高峰[[西利西利|西利西利]]山（Mauga Silisili）海拔1858米。属[[热带雨林气候|热带雨林气候]]，11月至次年4月为雨季，年降水量在2500毫米以上。吹東南[[風|風]]，年均溫約29[[摄氏度|摄氏度]]。

== 行政区划 ==
[[File:Samoa_districts_numbered2.png|left]]

{{main|萨摩亚行政区划}}

萨摩亚由11个itūmālō（政治区）组成。它们是在欧洲人到达之前很早就有的传统11个区。它们分别为：
[[图阿马萨加|图阿马萨加]]（Tuamasaga）、[[阿纳|阿纳]]（A'ana）、[[艾加伊勒泰|艾加伊勒泰]]（Aiga-i-le-Tai）、[[阿图阿|阿图阿]]（Atua）、[[瓦奥福诺蒂|瓦奥福诺蒂]]（Va'a-o-Fonoti）、法塞莱莱阿加（Fa'asaleleaga）、加加埃毛加（Gaga'emauga）、加盖福毛加（Gaga'ifomauga）、韦西加诺（Vaisigano）、萨图帕伊泰阿（Satupa'itea）、帕劳利（Palauli）。

== 经济 ==
{{main|萨摩亚经济}}

以[[农业|农业]]為主，出產[[椰子|椰子]]、[[可可|可可]]、[[麵包果|麵包果]]、[[芋頭|芋頭]]、[[香蕉|香蕉]]。[[工業|工業]]有小型[[輕工業|輕工業]]及農產品加工業。[[旅游业|旅游业]]也是萨摩亚主要经济支柱之一。薩摩亞是税制与英联邦国家类似，也有外汇管制。

== 人口 ==
約20萬人。90%为[[萨摩亚人|萨摩亚人]]，属[[波利尼西亚|波利尼西亚]]語族；其餘為混血族群、歐洲人和[[萨摩亚华人|華人]]等。

== 体育 ==
萨摩亚的主要体育运动是[[英式橄榄球|英式橄榄球]]与萨摩亚板球。[[英式橄榄球|英式橄榄球]]可称作是萨摩亚的国球。[[联盟式橄榄球|联盟式橄榄球]]在萨摩亚也很流行。2000年联盟式橄榄球世界杯上萨摩亚国家队进入了四分之一决赛。

[[萨摩亚人|萨摩亚人]]在[[拳击|拳击]]、[[摔跤|摔跤]]、[[相扑|相扑]]方面也颇有成绩。
