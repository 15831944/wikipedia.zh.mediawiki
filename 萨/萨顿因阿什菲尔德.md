'''萨顿因阿什菲尔德'''是一个在[[英国|英国]][[诺丁汉郡|诺丁汉郡]]阿什菲尔德辖区的市镇，拥有约45800人口<ref>{{cite web |url=http://www.ashfield-dc.gov.uk/ccm/navigation/council--government-and-democracy/district-councillors-for-ashfield |title=存档副本 |accessdate=2014-04-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20100329050826/http://www.ashfield-dc.gov.uk/ccm/navigation/council--government-and-democracy/district-councillors-for-ashfield/ |archivedate=2010-03-29 }}</ref>。其位于[[曼斯菲爾德|曼斯菲尔德]]以西4英里（大约6.4公里），接近[[德比郡|德比郡]]边界。

== 地理 ==
从人口上讲萨顿因阿什菲尔德是{{link-en|曼斯菲尔德城市群|Mansfield urban area}}的一部分，即便其实际位于独立的阿什菲尔德市政区。{{link-en|斯科比|Skegby}}和{{link-en|斯坦顿丘|Stanton-Hill}}在其北部。

== 地标 ==
萨顿因阿什菲尔德的主要景点是欧洲最大的日晷，其位于波特兰广场的中央，靠近艾德威尔购物广场和萨顿社区学校。
国王米尔医院是一个主要的[http://www.geograph.org.uk/photo/856996 地标建筑]，位于市镇中心和[[曼斯菲尔德|曼斯菲尔德]]之间，紧邻A38公路。
萨顿因阿什菲尔德有一个大型[[Asda|Asda]]超市位于市镇中心的北部。
== 历史 ==
=== 撒克逊 ===
撒克逊人最早在此定居，撒克逊人的后缀"ton"的意思是“一块在林中空地中的圈地”。此市镇在英国土地志中被称作"sustone"。1189年的档案亦显示沃尔特德萨顿(Walter de Sutton)之子杰勒德(Gerard)曾赠与瑟嘉顿(Thurgarton)修道院两博维特(bovate)土地和位于萨顿的教堂。
