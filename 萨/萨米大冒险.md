《'''萨米大冒险'''》是一部2010年的[[比利时|比利时]][[3D|3D]]动画电影，由[[:en:Ben_Stassen|本·斯塔森]]导演，[[梅拉尼·格里菲思|梅拉尼·格里菲思]]和[[伊莎贝拉·弗尔曼|伊莎贝拉·弗尔曼]]担任配音主角，配乐则由为《[[鐵甲奇俠_(電影)|钢铁侠]]》配乐的[[:en:Ramin_Djawadi|拉民·贾瓦迪]]担任。比利时于2010年8月4日上映，[[香港|香港]]于2012年8月16日上映，[[中国大陆|中国大陆]]则于2012年9月30日上映。

影片主要讲述了一只[[海龟|海龟]]在海洋里生活的50年以来，由[[全球变暖|全球变暖]]所带来的一系列周遭事物的变化。

== 剧情 ==
一只名叫萨米的小海龟从沙滩上的蛋壳爬出，由于没有及早下水而差点被海鸟所捕食。幸存下来之后，他结识了美丽的女孩莎莉。在此后的50年里，他们乘坐破木板周游世界，见识了海洋各处的美景奇观，也经历着人类为了私欲对大自然做出的无休止的破坏……

== 评价 ==
该电影在[[烂番茄|烂番茄]]网站上的得分为44%，整体口碑褒贬不一。正面评价大多赞赏电影水下镜头的3D技术以及可爱的人物塑造，而负面评价主要是针对剧情设计的不满、以及与《[[海底总动员|海底总动员]]》的疑似雷同情节（如Nemo和萨米有许多相似之处）。

== 续集 ==
该片的续集《萨米大冒险2》于2012年8月15日在[[法国|法国]]上映，但全球公映日期尚未确定。
