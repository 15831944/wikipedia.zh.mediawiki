'''萨利占·沙基罗维奇·沙里波夫'''（{{lang-ru|Салижан Шакирович Шари́пов}}，{{lang-uz|Солижон Шокирович Шарипов}}，{{bd|1964年|8月24日|}}）是[[乌孜别克族|乌孜别克裔]]<ref>{{cite web |title=Космонавт Шарипов пытается помирить узбеков и кыргызов |url=https://rus.azattyq.org/a/sharipov_russia_students_kyrgyzstan_rzhev_/24182236.html |publisher=[[自由欧洲电台／自由电台|Радио Азаттык]] |accessdate=2019-04-15 |date=2011-05-23}}</ref>[[俄罗斯|俄罗斯]]已退役[[宇航员|宇航员]]。
==早年==
1964年8月24日出生于[[苏联|苏联]][[吉尔吉斯苏维埃社会主义共和国|吉尔吉斯苏维埃社会主义共和国]]（现[[吉尔吉斯斯坦共和国|吉尔吉斯斯坦共和国]]）奥什州[[烏茲根|烏茲根]]。1987年毕业于[[哈尔科夫|哈尔科夫]]高等空军飞行员学校。毕业后担任[[飞行员|飞行员]]教练，指导了8名学员。他的累计飞行时间超过950小时。他有驾驶[[米格-21戰鬥機|米格-21戰鬥機]]和[[L-39信天翁|L-39信天翁]]教练机的丰富经验。1994年毕业于[[莫斯科国立大学|莫斯科国立大学]]，获得[[地图学|制图学]]学位<ref name=bio>{{cite web |title=Biographical Data |url=https://web.archive.org/web/20170211214812/https://www.jsc.nasa.gov/Bios/htmlbios/sharipov.html |author=[[NASA|NASA]] |accessdate=2019-04-15}}</ref>。
==宇航员生涯==
1990年，他成为[[加加林宇航员培训中心|加加林宇航员培训中心]]的宇航员候选人，之后开始接受基础训练。1992年完成训练，获得测试宇航员资格<ref name=bio/>。
===STS-89===
1998年1月，他作为任务专家，与[[泰伦斯·威尔卡特|泰伦斯·威尔卡特]]和[[迈克尔·安德森|迈克尔·安德森]]等宇航员前往[[和平号空间站|和平号空间站]]执行[[STS-89|STS-89]]任务。这是[[奋进号航天飞机|奋进号航天飞机]]的第12次太空飞行，也是第5次和最后一次美国宇航员交换。任务共持续了8日19时47分<ref>{{cite web|url=http://science.ksc.nasa.gov/shuttle/missions/sts-89/mission-sts-89.html|title=STS-89|author=NASA|date=2001-06-29|accessdate=2010-06-15}}</ref>。

[[协调世界时|协调世界时]]2004年10月14日3时6分，沙里波夫、[[尤里·沙尔金|尤里·沙尔金]]和[[焦立中|焦立中]]乘坐[[联盟TMA-5|联盟TMA-5]]宇宙飞船从[[拜科努尔航天发射场|拜科努尔航天发射场]]起飞，执行[[远征10|远征10]]任务。经过2天的绕地飞行，宇宙飞船于协调世界时10月16日4时16分与国际空间站成功对接<ref>{{cite web|url=http://spaceflight.nasa.gov/station/crew/exp10/intsharipov.html|title=Preflight Interview: Salizhan Sharipov|author=NASA|date=2004-09-15|accessdate=2010-06-15}}</ref>。

莫斯科时间2004年12月26日2时58分（北京时间7时58分），俄“进步M－51”货运飞船与国际空间站成功自动对接，为面临食品短缺危险的沙里波夫和焦立中带去了食品、实验设备和[[圣诞节|圣诞礼物]]<ref>{{cite web |title=国际空间站宇航员有饭吃了 |url=http://www1.people.com.cn/GB/keji/1058/3081310.html |publisher=新华社 |accessdate=2019-04-15 |archiveurl=https://web.archive.org/web/20050205072246/http://www.people.com.cn/GB/keji/1058/3081310.html |archivedate=2005-02-05 |date=2004-12-27}}</ref>。

2005年2月，他和[[焦立中|焦立中]]利用пк-3装置，进行了“[[等离子体|等离子晶体]]”物理实验，实验结果能帮助研制发生[[核事故|核事故]]时除去大气中[[放射性|放射性]]有害杂质的“[[吸尘器|吸尘器]]”和[[航天器|航天器]]上的大[[功率|功率]]小型核[[电源|电源]]<ref>{{cite web |author=周道其 |title=空间站宇航员进行“等离子晶体”实验 |url=http://www.taikongmedia.com/Item/Show.asp?m=1&d=14082 |publisher=中国太空网 |archiveurl=https://web.archive.org/web/20190415115339/http://www.taikongmedia.com/Item/Show.asp?m=1&d=14082 |archivedate=2019-04-15}}</ref>。

2005年4月25日，他和[[焦立中|焦立中]]、意大利宇航员罗伯托·维托里搭乘[[联盟TMA-5|联盟TMA-5]]载人飞船返航。北京时间25日2时44分，飞船与国际空间站脱离，3个小时后进入地球[[大气层|大气层]]。北京时间25日凌晨6时8分，飞船在哈萨克斯坦[[阿尔卡雷克|阿尔卡雷克]]以北90公里的预定地点成功软着陆<ref>{{cite web |title=联盟载人飞船返回地面 焦立中平安回家 |url=http://tech.sina.com.cn/d/2005-04-25/1452592759.shtml |publisher=[[新浪|新浪]]科技 |accessdate=2019-04-15 |archiveurl=https://web.archive.org/web/20060525011135/http://tech.sina.com.cn/d/2005-04-25/1452592759.shtml |archivedate=2006-05-25 |date=2005-04-25}}</ref>。
