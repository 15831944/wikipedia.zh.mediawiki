'''萨翁林纳'''（{{lang-fi|Savonlinna}}，意为“萨沃城堡”；{{lang-sv|Nyslott}}，意为“新城堡”），是[[芬兰|芬兰]][[南萨沃尼亚区|南萨沃尼亚区]]的一个城市，成立于1639年。辖区总面积3597.70平方公里，其中陆地面积2238.21平方公里、淡水面积1359.49平方公里<ref name="面积"/>。2018年1月31日登记人口总数为34670人<ref name="人口"/>，人口密度为15.5人/平方公里。2016年底时芬兰语人口和瑞典语人口比例分别为96.3%和0.1%<ref name="母语"/>。

== 历史 ==
[[File:Savonlinna_from_air_1934.jpg|thumb]]
萨翁林纳在1639年取得获得城市地位。而这座城市的建设则始于1475年[[奥拉维城堡|奥拉维城堡]]的建造。奥拉维城堡由[[瑞典人|瑞典人]][[埃里克·阿克塞爾松|埃里克·阿克塞尔松·托特]]领导建造，目的是为了保护萨翁林纳城，使其免遭[[俄国|俄国]]的侵犯。尽管如此，萨翁林纳还是在1473年的瑞俄战争中被俄国人占领，直到1812年才被归还给[[芬兰大公国|芬兰大公国]]<ref name="LP_Finland">{{cite book |last=Symington |first=Andy |title=Lonely Planet Finland |publisher=Lonely Planet Publications |edition=5th Edition |date=2006年4月 |pages=p115-116 |isbn=1-74059-791-5 |oclc=63136816 |language=en }}</ref>。

1973年，塞明基[[芬兰市镇|市镇]]并入萨翁林纳。2009年，萨翁兰塔市镇以及萨翁林纳与萨翁兰塔之间一片面积为31.24平方公里、原属[[埃农科斯基|埃农科斯基]]的土地一起并入萨翁林纳。2013年，[[凯里迈基|凯里迈基]]市镇和[[蓬卡哈尔尤|蓬卡哈尔尤]]市镇一起并入萨翁林纳。

== 文化 ==
萨翁林纳每年7月在奥拉维城堡内举办为期四星期的[[萨翁林纳歌剧节|萨翁林纳歌剧节]]。这项节日早在1912年便首次举办，但很快由于[[第一次世界大战|第一次世界大战]]的到来而中断，直到1967年起恢复，并逐年举办<ref>{{cite web |url=http://www.operafestival.fi/In_English/Front_Page/mainmenu/The_Event/History.iw3 |title=History |publisher=萨翁林纳歌剧节 |date= |accessdate=2009年8月16日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20090715233525/http://www.operafestival.fi/In_English/Front_Page/mainmenu/The_Event/History.iw3 |archivedate=2009年7月15日 }}</ref>。此外，萨翁林纳以前还在8月份举办过萨翁林纳[[芭蕾舞|芭蕾舞]]节和世界扔手机锦标赛<ref name="LP_Finland"/>。

* [[科洛韦西国家公园|科洛韦西国家公园]]

== 教育 ==
[[约恩苏大学|约恩苏大学]]在萨翁林纳设有一个校区。该校区约有170名员工和1100名学生。<ref>{{cite web |url=http://www.joensuu.fi/general/index.html |title=University of Joensuu - General infromation |publisher=[[约恩苏大学|约恩苏大学]] |date= |accessdate=2009年8月16日 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20060427093901/http://www.joensuu.fi/general/index.html |archivedate=2006年4月27日 }}</ref>2010年，约恩苏大学和[[库奥皮奥大学|库奥皮奥大学]]合并为[[东芬兰大学|东芬兰大学]]，约恩苏大学萨翁林纳校区也是东芬兰大学的三个校区之一。

== 体育 ==
萨翁林纳冰球队SaPKo目前参加芬兰冰球联赛第二级别的比赛。足球队STPS则参加芬兰足球联赛第四级别的比赛。

== 交通 ==
[[File:Savonlinnan_uusi_rautatieasema.JPG|thumb]]
=== 公路和铁路 ===
萨翁林纳的公共交通在2010年代受到公共开支削减的影响，很多车次大为减少。市内有5条公交路线。萨翁林纳与首都[[赫尔辛基|赫尔辛基]]及周边的[[米凯利|米凯利]]、[[约恩苏|约恩苏]]、[[库奥皮奥|库奥皮奥]]等城市间有大巴通行。

萨翁林纳与[[帕里卡拉|帕里卡拉]]之间通有轻型火车，每天有四至六趟。在帕里卡拉可以转城际火车前往约恩苏和赫尔辛基。

=== 水路 ===
在夏季，萨翁林纳港口里有着繁忙的轮船交通。每天有多班观光蒸汽轮船在[[塞马湖|塞马湖]]上行驶。萨翁林纳与库奥皮奥之间开有定期轮渡。

=== 航空 ===
[[萨翁林纳机场|萨翁林纳机场]]位于市区以北约15公里处。萨沃林纳与赫尔辛基之间有定期的航班。萨翁林纳北部地区离[[约恩苏机场|约恩苏机场]]也很近。
