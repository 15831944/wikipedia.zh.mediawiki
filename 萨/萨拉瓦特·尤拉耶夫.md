'''萨拉瓦特·尤拉耶夫'''（{{lang-ba|Салауат Юлай}}，{{lang-ru|Салават Юлаев}}，{{bd|1754年|6月16日|1800年|9月26日|catIdx=Yulaev, Salavat}}），[[巴什基爾|巴什基爾]]起义领袖、诗人，被[[巴什基爾人|巴什基爾人]]奉为民族英雄。

== 生平 ==
萨拉瓦特·尤拉耶夫生于[[俄羅斯帝國|俄羅斯帝國]][[奧倫堡省|奧倫堡省]]捷克耶沃村（{{lang|ru|Текеево}}）。捷克耶沃村在1775年被烧毁，今已不存。萨拉瓦特·尤拉耶夫的出生年份一说为1754年，也有资料说他出生在1752年<ref>{{cite web |title=Салават Юлаев |url=http://hrono.ru/biograf/bio_s/salavat_yul.php |website=ХРОНОС |accessdate=2018-11-28}}</ref>。17世纪中期，沙皇政府开始侵占巴什基尔人的土地，兴建堡垒。18世纪起，政府还在巴什基尔人的土地上兴建工厂。截至18世纪初，政府已经在巴什基尔人的土地上兴建了31座堡垒，堡垒周边的土地被分给俄罗斯贵族、教会、商人、农民。17世纪后期，俄国政府几次试图让巴什基尔人皈依[[东正教|东正教]]。侵占土地、增加税收和劳役等做法激起了巴什基尔人的不满。17世纪起，巴什基尔人{{tsl|ru|Башкирское восстание|巴什基爾起義|多次起兵反抗}}。1773年，[[叶梅利扬·伊万诺维奇·普加乔夫|叶梅利扬·伊万诺维奇·普加乔夫]]发动起义，史稱[[普加乔夫起义|普加乔夫起义]]。当时仅19岁的萨拉瓦特·尤拉耶夫起兵响应，他召集士兵、组建一支3000多（一说2000多<ref>{{Cite book|title=俄国女大帝 叶卡特琳娜二世传|last=张达明编著|first=|publisher=长春：[[吉林人民出版社|吉林人民出版社]]|year=1998.02|isbn=7-206-02846-2|location=|pages=188}}</ref>）巴什基里亚人的部队，与政府军作战，指挥了一些重要的战斗，攻取了数座堡垒，包括6月17日占领了[[克拉斯諾烏菲姆斯克|克拉斯诺乌菲姆斯克]]<ref>{{Cite book|title=世界战争解焦点 100场战争|last=刘乐土编著|first=|publisher=北京：华夏出版社|year=2012.01|isbn=978-7-5080-5683-8|location=|pages=187}}</ref>。萨拉瓦特帮助普加乔夫传播告示，为普加乔夫的部队提供资金、装备、给养，帮助动员。普加乔夫起义最终被沙皇[[叶卡捷琳娜二世|叶卡捷琳娜二世]]镇压<ref>{{Cite book|title=世界史大事汇编|last=王正平主编|first=|publisher=杭州：[[浙江人民出版社|浙江人民出版社]]|year=1984.11|isbn=|location=|pages=238-239}}</ref>。1774年11月24日，萨拉瓦特被捕，随后被判处终身监禁<ref>{{Cite book|title=世界历史百科全书 人物卷|last=黑龙江大学等译|first=|publisher=北京：[[商务印书馆|商务印书馆]]|year=1992.12|isbn=7-100-01542-1|location=|pages=1095}}</ref>。1775年10月3日，被押送到[[波罗的海|波罗的海]]岁格尔维克要塞（今[[爱沙尼亚|爱沙尼亚]]的[[帕尔迪斯基|帕尔迪斯基]]）。1800年，萨拉瓦特·尤拉耶夫在狱中离世<ref name="russiapedia.rt" /><ref name="legal">{{cite news |title=欧亚走廊重镇准备迎接两大峰会召开 |url=http://news.youth.cn/gn/201507/t20150706_6829723.htm |accessdate=2018-11-26 |work=法制日报 |date=2015-07-06}}</ref><ref>{{cite news |title=令人向往的俄文化胜地—巴什科尔托斯坦共和国 |url=http://tsrus.cn/lvyou/2015/06/27/42549.html |accessdate=2018-11-26 |work=透视俄罗斯 |date=2015-06-27}}</ref>。

== 文学 ==
萨拉瓦特·尤拉耶夫的诗歌作品超过五百部，题材包括反抗压迫、歌颂家乡、巴什基爾民族、爱情<ref name="russiapedia.rt">{{cite web |title=Prominent Russians: Salavat Yulayev |url=https://russiapedia.rt.com/prominent-russians/history-and-mythology/salavat-yulayev/ |accessdate=2018-11-26 |work=RT Russiapedia}}</ref><ref>{{Cite book|title=苏联通史第2卷|last=（苏）安米潘克拉托娃主编|first=|publisher=北京：[[生活·读书·新知三联书店|生活·读书·新知三联书店]]|year=1980.04|isbn=|location=|pages=118}}</ref>。他被视为[[巴什基尔文学|巴什基尔文学]]的奠基人<ref>{{cite book |editor=高尔基世界文学研究所 |title=世界文学史 第8卷 上册 |date=2013-12 |publisher=上海文艺出版社 |location=上海 |isbn=978-7-5321-4995-7 |page=251}}</ref>。

== 纪念 ==
巴什科尔托斯坦共和国首府[[乌法|乌法]]的一些街道、商业中心、冰球队均以萨拉瓦特·尤拉耶夫命名。巴什科尔托斯坦共和国国家奖亦以尤拉耶夫命名。乌法市中心山上的[[薩拉瓦特·尤拉耶夫雕像|薩拉瓦特·尤拉耶夫雕像]]是城市的重要景点，[[巴什科尔托斯坦共和国国徽|巴什科尔托斯坦共和国国徽]]上绘有这座雕像的形象。<ref>{{cite news |title=走进“合金峰会”举办地乌法 |url=http://www.xinhuanet.com/herald/2015-07/09/c_134397572.htm |accessdate=2018-11-26 |work=国际先驱导报 |date=2015-07-09}}</ref><ref name="russiapedia.rt" />巴什科尔托斯坦共和国南部城市[[薩拉瓦特|薩拉瓦特]]以他命名。<ref>{{cite book |editor=邵献图 |title=外国地名语源词典 |date=1983-04 |publisher=上海辞书出版社 |location=上海 |page=371| CSBN=17187-86}}</ref>
