'''郡山富田站'''（{{jpn|j=郡山富田駅|hg=こおりやまとみたえき|rm=Kōriyamatomita eki|f=y}}）位于[[福岛县|福岛县]][[郡山市|郡山市]]富田町，是[[东日本旅客铁道|东日本旅客铁道]]（JR東日本]）管辖的[[磐越西线|磐越西线]]上的[[鐵路車站|鐵路車站]]<ref>{{Cite web |date=2015-9-15 |url=http://www.jr-sendai.com/wp-content/uploads/2015/09/koriyamatomita.pdf |title=磐越西線に新駅を設置いたします |format=PDF |publisher=JR東日本仙台支社 |accessdate=2015-9-15}}</ref>。

== 历史 ==
为了能够让郡山市公共交通促进地区发展、在[[自然灾害|自然灾害]]发生时可以及时运送救灾物资和受灾群众<ref>[http://www.city.koriyama.fukushima.jp/shise/citysales/sineki.html 郡山富田駅（磐越西線）の設置について] - 郡山市</ref>，当时以“郡山北”的名义，郡山市与JR东日本商讨修建新车站。2015年9月15日，JR东日本正式命名新车站为“'''郡山富田站'''”，其中新设站所花费的近20亿日元由郡山市独自承担<ref name="minpo20150916">{{Cite news |title=名称は「郡山富田駅」JR磐越西線の新駅 29年春開業へ|newspaper=福島民報|date=2015-09-16|author=|url=https://www.minpo.jp/news/detail/2015091625339|accessdate=2015-09-17|format=}}</ref>。

* 2017年4月1日 - 车站投入运营<ref>{{PDFlink|[http://jr-sendai.com/upload-images/2016/12/20161216.pdf 2017年3月ダイヤ改正および新駅開業等について]}} - JR東日本仙台支社</ref><ref>[http://www.minyu-net.com/tourist/naka/FM20170402-161303.php JR郡山富田駅が開業 県内で30年ぶり新駅] - 福島民友 みんゆうNet、2017年04月02日 08時00分、同月4日阅览</ref>。

== 车站结构 ==
[[侧式站台|侧式站台]]1台1线的[[地上车站|地上车站]]，站台长度能容纳6辆编组列车停靠。[[郡山站_(福島县)|郡山]]管理的无人站。站内设有简易候车室、车站仓库，车站手表设置有交通广场、停车场、公共厕所和人行跨线[[天桥|天桥]]。开业起便支持[[Suica|Suica]]使用。

== 使用情况 ==
