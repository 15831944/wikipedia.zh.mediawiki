'''嫦娥五号'''是[[中国探月工程|中国探月工程]]第三阶段的[[月球探测器|月球探测器]]，主要承担探月工程三期的采样返回任务<ref>{{Cite web |url = http://news.sciencenet.cn/htmlnews/2011/12/256719.shtm |title = 欧阳自远院士描绘嫦娥工程后续蓝图 |author=张巧玲 |publisher=[[科学时报|科学时报]] |date=2011年12月9日 |language = zh-hans}}</ref>。目前，嫦娥五号预计将于[[海南|海南]][[文昌航天发射场|文昌航天发射场]]发射，执行在月球表面自动采样并起飞返回的任务<ref>[http://www.apdnews.com/news/54869.html 嫦娥五号预计2017年前后发射] {{webarchive|url=https://web.archive.org/web/20131217052217/http://www.apdnews.com/news/54869.html |date=2013-12-17 }}，[[亚太日报|亚太日报]]，2013年12月17日</ref><ref name="18年发射">{{Cite news |url = http://www.chinanews.com/mil/2013/03-11/4630533.shtml |title=  探月工程副总设计师：嫦娥五号计划于2018年发射 |publisher = [[中国新闻网|中国新闻网]]引自[[中国广播网|中国广播网]] |author = 张棉棉 |date = 2013年3月11日 |accessdate = 2013年6月27日 |language = zh-cn}}</ref><ref>{{Cite web|url =http://www.morningpost.com.cn/xwzx/guonei/2012-07-25/338237.shtml|title =欧阳自远：嫦三明年发射|publisher =[[北京晨报|北京晨报]]|author =韩娜|date =2012年7月25日|language =zh-hans|deadurl =yes|archiveurl =https://web.archive.org/web/20130614012049/http://www.morningpost.com.cn/xwzx/guonei/2012-07-25/338237.shtml|archivedate =2013年6月14日}}</ref>。嫦娥五号有望实现中国开展航天活动以来的四个“首次”：首次在月球表面自动采样；首次从月面起飞；首次在38万公里以外的月球轨道上进行无人交会对接；首次带着月壤以接近第二宇宙速度返回地球<ref>{{cite news|title=六位“嫦娥”去探月 各司其职有不同|url=http://www.chinanews.com/gn/2017/03-01/8162718.shtml|accessdate=2017-04-20|agency=新华网|date=2017年3月1日}}</ref>。

嫦娥五号由轨道器、返回器、上升器和着陆器组成。嫦娥五号设计有两种取样方式，用以丰富样品类型。其一是在月面打钻，是为取得较深层的月壤；其二是用机械臂在月球表面取样，在有限范围内横扫收集月壤。两种样品的比例约为1：3，表面采取的月壤更多。随后，上升器将携带月壤样品从已落在月面的着陆器上起飞，并由轨道器抓获，将月壤样品转移到返回器里<ref>{{cite news|title=“嫦娥五号”2017年择机发射 揭秘五大看点|url=http://www.chinanews.com/gn/2017/03-02/8164097.shtml|accessdate=2017-04-20|work=www.chinanews.com|agency=中国新闻网|date=2017年3月2日}}</ref><ref>{{cite web|author1=观察者网|title=欧阳自远：嫦娥5号发射计划目前没有受长5失利影响|url=https://tech.sina.com.cn/d/s/2017-07-13/doc-ifyiakur8777372.shtml|website=新浪科技|accessdate=2017-07-13|date=2017-07-13}}</ref>。

== 科学目标 ==
开展着陆点区的形貌探测和地质背景勘察， 获取与月球样品相关的现场分析数据，建立现场探测数据与实验室分析数据之间的联系。主要包括：
* 着陆区的地形地貌探测：
** 采样点周围形貌与结构构造特征
** 撞击坑的形貌、大小与分布等

* 物质成分探测：
** 采样点的物质成分特征
** 月壤物理特性与结构
** 月壳浅层的温度梯度探测等

对返回地面的月球样品进行系统、长期的实验室研究，分析月壤与月岩的物理特性与结构构造、矿物与化学组成、微量元素与同位素组成、月球岩石形成与演化过程的同位素年龄测定、宇宙辐射与太阳风离子与月球的相互作用、太空风化过程与环境演化过程等，深化月球成因和演化历史的研究。

== 设备 ==
降落相机、光学相机、月球矿物光谱分析仪、月壤气体分析仪、月壤结构探测仪、采样剖面测温仪、岩芯钻探机和机械取样器。
