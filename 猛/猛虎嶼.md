'''猛虎嶼'''，為中國福建外海島嶼，本名'''虎仔嶼'''，1960年，中華民國方面改為現名，但中華人民共和國方面不承認中華民國方面改名，在其地圖文件等仍稱“虎仔嶼”。

該島行政區域隸屬於[[中華民國|中華民國]][[福建省|福建省]][[金門縣|金門縣]][[烈嶼鄉|烈嶼鄉]]，地處[[大膽島|大膽島]]與[[小金門|小金門]]（烈嶼）連線中點，自[[國共內戰|國共內戰]]時期起即為中華民國所管轄之戰地前線島嶼，迄今仍為[[中華民國國防部|中華民國國防部]]依據《要塞堡壘地帶法》所實施軍事管制區域。
== 命名歷史 ==
猛虎嶼原名虎仔嶼，為烈嶼當地民眾沿襲使用之島名，命名來自於東南角之附屬礁岩狀似張口俯臥之虎首。[[八二三砲戰|八二三砲戰]]期間與戰役後期數年，時任國防部總政治部主任、國防會議副秘書長、國防部長等職的[[蔣經國|蔣經國]]屢次親赴金門周邊前線島嶼據點視察，當時求振奮守軍士氣，將該地區多座島嶼另予命名（如[[覆鼎嶼|覆鼎嶼]]改[[復興嶼|復興嶼]]、[[鼠嶼|鼠嶼]]改[[獅嶼|獅嶼]]），1960年「虎仔嶼」正式改為現名。

== 地理與戰略地位==
猛虎嶼距離[[中華人民共和國|中華人民共和國]][[廈門|廈門]]之距離雖然不比[[大膽島|大膽島]]、[[二膽島|二膽島]]，但其仍相當接近廈門。該島由一主體與東南角一大型附屬礁岩所組成，面積約為0.025平方公里。猛虎嶼之主體海拔高度接近五十公尺，地勢高聳、幾乎為峭壁所圍繞，為傳統戰術考量上易守難攻之重要地點。猛虎嶼扮演扼守[[廈門灣|廈門灣]]南線水道（亦即[[廈門港|廈門港]]聯外水路）之角色時，可以發揮中程火力支援與固守補給路線之功能。

== 人文 ==
島上並無常住居民，僅有[[中華民國軍隊|中華民國軍隊]]駐守，人文活動只能從載於中華民國國軍刊物的少數報導與[[散文|散文]]發掘。島上一如金門各地長期之戰地經營結果，由[[地道|地下通道]]連結各個槍炮陣地。因軍事管制之因素，一般民眾無法登島，僅能由海上船隻遙望島上之大型標語與少數地面建物。

== 交通 ==
島上僅有簡易碼頭，[[碼頭|碼頭]]旁之峭壁刻有1973年[[端午節|端午節]]時[[參謀總長|參謀總長]][[賴名湯|賴名湯]]上將題字之「以猛虎之精神殺敵報國」大型標語。定期由中華民國軍方派遣船隻往返大小金門之各碼頭，通常僅供駐軍人員流動與物資運補之用。因地勢高聳，島上可運用各類[[無線電|無線電]]、[[旗號|旗號]]、[[通訊衛星|通訊衛星]]等與周邊軍民聯繫。
