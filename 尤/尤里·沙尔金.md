'''尤里·格奥尔基耶维奇·沙尔金'''（{{lang-ru|Юрий Георгиевич Шаргин}}，{{bd|1960年|3月20日|}}）是[[俄罗斯|俄罗斯]]已退役[[宇航员|宇航员]]，也是[[俄罗斯太空军|俄航空兵]]中第一位进入[[国际空间站|国际空间站]]的军人<ref name=renmin>{{cite web |author=张春燕 |title=俄联盟飞船成功返回 宇航员难忘空间站生活 |url=http://www.people.com.cn/GB/keji/1058/2941518.html |publisher=[[人民网|人民网]] |accessdate=2019-04-13 |date=2004-10-25|archiveurl=https://web.archive.org/web/20190413044528/http://www.people.com.cn/GB/keji/1058/2941518.html|archivedate=2019-04-13}}</ref>。
==早年==
1960年出生在[[恩格斯城|恩格斯城]]工人家庭。1982年毕业于红旗航空军事学院（现{{tsl|en|A.F. Mozhaysky Military-Space Academy|亚历山大·莫扎伊斯基军事航天学院}}）。1982年到1986年在[[拜科努尔航天发射场|拜科努尔航天发射场]]第31号航空发射队担任高级工程师；1987年1月起成为首席工程师助理；1993年4月升任首席工程师。<ref name="bio"/>
==宇航员生涯==
1996年入选宇航员，1998年起开始接受前往国际空间站的基础训练。<ref name=zx>{{cite web |title=俄十月发射飞船俄军宇航员将成第三位太空游客 |url=http://www.cas.cn/xw/kjsm/gjdt/200906/t20090608_629085.shtml |publisher=[[中国科学院|中国科学院]] |archiveurl=https://web.archive.org/web/20190413044131/http://www.cas.cn/xw/kjsm/gjdt/200906/t20090608_629085.shtml |archivedate=2019-04-13 |date=2004-08-29}}</ref>

[[莫斯科时间|莫斯科时间]]2004年10月14日7时06分（[[北京时间|北京时间]]10月14日12时06分），他和美籍[[华裔|华裔]]指挥员[[焦立中|焦立中]]和俄罗斯工程师[[萨利占·沙里波夫|萨利占·沙里波夫]]在[[哈萨克斯坦|哈萨克斯坦]][[拜科努尔航天发射场|拜科努尔航天发射场]]搭乘[[联盟TMA-5|联盟TMA-5]]号载人飞船飞往国际空间站。其中焦立中和沙里波夫将执行[[远征10|远征10]]任务，沙尔金仅是俄航空兵[[中校|中校]]身份的短期“观光客”。<ref>{{cite web |author=久亮 |title=国际空间站第10批宇航员今天踏上征程 |url=http://www.cnetnews.com.cn/2004/1014/147219.shtml |publisher=[[CNET|CNET]] |archiveurl=https://web.archive.org/web/20190413045149/http://www.cnetnews.com.cn/2004/1014/147219.shtml |archivedate=2019-04-13 |date=2004-10-14}}</ref>

莫斯科时间10月24日凌晨4时36分（北京时间10月24日8时36分），他乘坐[[联盟TMA-4|联盟TMA-4]]载人飞船的返回舱成功降落[[地球|地球]]。<ref name=renmin/>

==退役==
2008年8月退役，被调到{{tsl|en|Titov Main Test and Space Systems Control Centre|季托夫测试和航天系统控制中心}}担任研究中心副主任<ref name="bio">{{cite web |title=Шаргин, Юрий Георгиевич |url=http://www.warheroes.ru/hero/hero.asp?Hero_id=2318 |publisher=Герои страны |accessdate=2019-04-13 |language=ru}}</ref>。
