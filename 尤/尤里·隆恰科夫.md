'''尤里·瓦连季诺维奇·隆恰科夫'''（{{lang-ru|Ю́рий Валенти́нович Лончаков}}，{{bd|1965年|3月4日|}}）是哈萨克斯坦出身的[[俄罗斯|俄罗斯]][[宇航员|宇航员]]，曾两次完成常驻国际空间站任务，并两度进行太空行走。2013年9月退役。

==早年==
1965年3月4日出生于[[苏联|苏联]][[哈萨克苏维埃社会主义共和国|哈萨克苏维埃社会主义共和国]][[巴爾喀什|巴爾喀什]]。1982年毕业于[[阿克托比|阿克纠宾斯克]]第22中学，后进入[[奥伦堡|奥伦堡]]空军飞行员学校，1986年以优异成绩毕业，后在[[波罗的海|波罗的海]]海军舰队航空部队服役，历任军队任机组指挥官、中队高级飞行员和海军航空旅指挥官等职务，有[[雅克-52|雅克-52]]、[[L-39信天翁|L-39]]、[[苏-24战斗轰炸机|苏-24]]、[[A-50預警機|A-50]]、[[L-29海豚|L-29]]、[[图-134|图-134]]和[[图-16|图-16]]等多种飞机的驾驶经验，累计飞行时间超过1500小时。1995年进入[[尼古拉·叶戈罗维奇·茹科夫斯基|茹科夫斯基]]空军工程学院进修，1998年毕业<ref name=bio>{{cite web |title=Biographical Data - Yury Valentinovich Lonchakov|url=https://web.archive.org/web/20170202003528/https://www.jsc.nasa.gov/Bios/htmlbios/lonchakov.html|publisher=[[NASA|NASA]] |accessdate=2019-04-14}}</ref>。

==宇航员生涯==
1997年入选[[加加林宇航员培训中心|加加林宇航员培训中心]]宇航员候选人。1998年1月到1999年11月接受基础太空飞行训练，1999年12月获得测试宇航员资格。

2000年9月28日，他被任命为[[奋进号航天飞机|奋进号航天飞机]]STS-100任务机组人员，2000年10月至2001年4月在美国[[林顿·约翰逊太空中心|林顿·约翰逊太空中心]]接受专门训练<ref name=eng>{{cite web |title=Yury Lonchakov |url=https://www.energia.ru/en/iss/iss18/lonchakov.html |publisher=Korolev RSC "Energia" |accessdate=2019-04-17}}</ref>。
===STS-100===
协调世界时2001年4月19日18时42分，[[奮進號太空梭|奮進號太空梭]]从[[肯尼迪航天中心|肯尼迪航天中心]]LC-39A发射场升空，装载[[移动维修系统|加拿大臂2]]和包括隆恰科夫在内的7名[[STS-100|STS-100]]任务队员前往国际空间站<ref>{{cite web |title=STS-100:  Полет тяжелый,  но успешный |url=https://web.archive.org/web/20120417210122/http://www.novosti-kosmonavtiki.ru:80/content/numbers/221/07.shtml |website=novosti-kosmonavtiki.ru |accessdate=2019-04-17 |language=ru}}</ref>。2001年5月1日16时11分，载有相同任务队员的[[奮進號太空梭|奮進號太空梭]]在[[美国|美国]][[加利福尼亚州|加利福尼亚州]][[爱德华兹空军基地|爱德华兹空军基地]]成功降落<ref>{{cite web|url=http://spaceflight.nasa.gov/spacenews/reports/sts100/STS-100-27.html|title=STS-100 Mission Control Center Status Report #27|author=NASA|date=2001-05-01|accessdate=2010-06-08}}</ref>。
===联盟TMA-1/TM-34===
2002年10月30日乘坐联盟TMA-1前往国际空间站执行任务，10天后乘坐[[联盟TM-34|联盟TM-34]]于协调世界时2002年11月10日0时4分成功在哈萨克斯坦[[阿尔卡雷克|阿尔卡雷克]]东北100公里处着陆<ref>{{cite web|url=http://www.russianspaceweb.com/iss_soyuztm34.html|title=Soyuz TM-34 Mission to the ISS|publisher=RussianSpaceWeb.com|accessdate=2010-06-08}}</ref>。总飞行时间为10天20小时53分<ref name=eng/>。
===远征18===
[[File:Expedition_18_crew_portrait.jpg|远征18任务队员，从左至右依次为[[若田光一]]、芬克、[[桑德拉·马格努斯|马格努斯]]、隆恰科夫、[[格雷戈里·查米托夫|查米托夫]]|thumb]]
莫斯科时间10月12日11时1分（北京时间15时1分），[[联盟TMA-13|联盟TMA-13]]载人飞船从[[哈萨克斯坦|哈萨克斯坦]]境内的[[拜科努尔发射场|拜科努尔发射场]]顺利升空升空，前往国际空间站。飞船载有3人，其中隆恰科夫担任联盟TMA-13指令长及空间站工程师，美国宇航员[[迈克尔·芬克|迈克尔·芬克]]是随航工程师，美国电脑游戏开发商[[理查·蓋瑞特|-{zh-cn:理查德·加里奥特;zh-tw:理查·蓋瑞特;}-]]是太空游客<ref>{{cite web |author=聂云鹏 |title=俄美宇航员与太空游客飞赴国际空间站 |url=http://news.sciencenet.cn/htmlnews/2008/10/211894.html |publisher=[[新华社|新华社]] |archiveurl=https://web.archive.org/web/20190417043151/http://news.sciencenet.cn/htmlnews/2008/10/211894.html |archivedate=2019-04-17 |date=2008-10-12}}</ref>。

2008年11月18日，隆恰科夫在空间站进行了一项[[地球物理|地球物理]]试验，试验设备为[[远征17|远征17]]成员事先安装在[[星辰号服务舱|星辰号服务舱]]外部的仪器。该设备能够记录近地空间[[电子流|电子流]]及质子流的变化，这些变化与地球雷雨现象及[[地震|地震]]活动密切相关。该研究可以为地震的预测指明新方向<ref>{{cite web |author=聂云鹏 |title=国际空间站宇航员进行预测地震试验 |url=http://www.taikongmedia.com/Item/Show.asp?m=1&d=16387 |publisher=中国太空网 |archiveurl=https://web.archive.org/web/20190417043808/http://www.taikongmedia.com/Item/Show.asp?m=1&d=16387 |archivedate=2019-04-17 |date=2008-11-19}}</ref>。

2008年12月23日，他和芬克顺利完成了一次太空行走任务。包括在[[码头号对接舱|码头号对接舱]]外部安装用于监测空间站周围电磁强度的电磁测量装置，在[[星辰号服务舱|星辰号服务舱]]外安装并拆卸相关实验装置。这是国际空间站远征18任务队员完成的首次太空行走，也是空间站历史上的第119次太空行走<ref>{{cite web |author=刘洋 |title=空间站宇航员完成太空行走 |url=http://scitech.people.com.cn/GB/8577787.html |publisher=人民网 |accessdate=2019-04-17 |archiveurl=https://web.archive.org/web/20130406213132/http://scitech.people.com.cn/GB/8577787.html |archivedate=2013-04-06 |date=2008-12-25}}</ref>。

2009年3月10日19时22分（北京时间11日零时22分），他和芬克打开空间站舱门，开始了远征18任务的第二次太空行走任务。他们再次安装了一套实验装备，并对空间站俄罗斯舱段进行了修理维护<ref>{{cite web |author=聂云鹏 |title=国际空间站宇航员进行太空行走 将持续近6小时 |url=http://world.huanqiu.com/roll/2009-03/399293.html?agt=15438 |publisher=新华网 |archiveurl=https://web.archive.org/web/20190417052210/http://world.huanqiu.com/roll/2009-03/399293.html?agt=15438 |archivedate=2019-04-17 |date=2009-03-11}}</ref>。

2009年4月8日11时16分（北京时间15时16分），载有远征18任务队员的[[联盟TMA-13|联盟TMA-13]]飞船返回舱平安降落在哈萨克斯坦[[杰兹卡兹甘|杰兹卡兹甘]]市东北部草原。它原定于本月7日返程，但由于天气原因而被推迟<ref>{{cite web |author1=聂云鹏 |title=美国太空游客乘俄飞船平安返回地面 |url=http://news.eastday.com/w/20090408/u1a4297505.html |publisher=新华网 |archiveurl=https://web.archive.org/web/20190417053008/http://news.eastday.com/w/20090408/u1a4297505.html |archivedate=2019-04-17 |date=2009-04-08}}</ref>。

===相关活动===
2007年5月22日，隆恰科夫作为第十六届“人在太空”国际学术会议俄罗斯宇航员代表，与[[焦立中|焦立中]]、[[简·弗朗考斯·卡瓦略|简·弗朗考斯·卡瓦略]]和[[费俊龙|费俊龙]]等宇航员在[[清华大学|清华大学]]参加了“清华学子与中外航天员面对面”活动，以自己的真实经历向观众详细介绍了航天员鲜为人知的太空生活<ref>{{cite web |title=俄罗斯航天员:航天员生活如学生一样 天天面临考试 |url=http://www.china.com.cn/tech/zhuanti/rztk/2007-05/22/content_8290214.htm |publisher=中国网 |archiveurl=https://web.archive.org/web/20100425025336/http://www.china.com.cn/tech/zhuanti/rztk/2007-05/22/content_8290214.htm |archivedate=2010-04-25 |date=2007-05-12}}</ref>。

2011年5月14日，他和[[马克西姆·苏拉耶|马克西姆·苏拉耶]]现身[[浙江省科技馆|浙江省科技馆]]，为浙江省科协主办的“纪念人类太空飞行50周年宇航图片展”揭幕，同时做客“[[科学咖啡馆|科学咖啡馆]]”，与[[浙江|浙江]]的[[天文|天文]]爱好者进行交流<ref>{{cite web |author1=林丹 |title=俄罗斯现役宇航员逛西湖 惊叹西湖比太空还美 |url=http://www.wzrb.com.cn/article249465show.html |publisher=[[钱江晚报|钱江晚报]] |accessdate=2019-04-17 |archiveurl=https://web.archive.org/web/20110517091622/http://www.wzrb.com.cn/article249465show.html |archivedate=2011-05-17 |date=2011-05-14}}</ref>。

==退役==
2013年9月，隆恰科夫发布个人声明，表示自己获得了另一份更高薪的工作，并将不再担任宇航员。随后，辞职申请得到了指挥官的批准，2013年9月13日正式离职<ref>{{cite web |title=Юрий Лончаков покидает отряд космонавтов |url=http://novosti-kosmonavtiki.ru/news/12678/ |publisher=Новости космонавтики |accessdate=2019-04-17 |language=ru |date=2013-08-31}}</ref>。他也成为俄罗斯历史上首次出现主动请辞的宇航员<ref>{{cite web |author=钟通 |title=俄首次有宇航员主动辞职 为高薪跳槽申请已获批 |url=http://www.chinanews.com/gj/2013/09-09/5260502.shtml |publisher=中国新闻网 |accessdate=2019-04-17 |archiveurl=https://web.archive.org/web/20130910232851/http://www.chinanews.com/gj/2013/09-09/5260502.shtml |archivedate=2013-09-10 |date=2013-09-09}}</ref>。2014年3月开始担任[[加加林宇航员培训中心|加加林宇航员培训中心]]主任。

2013年11月参与了[[索契冬奥会|索契冬奥会]]火炬传递活动<ref>{{cite web |title=奥运火炬抵达阿穆尔州东方航天发射场建筑工地 |url=http://sputniknews.cn/russia/2013111843914413/ |publisher=[[俄罗斯卫星通讯社|俄罗斯卫星通讯社]] |accessdate=2019-04-17 |date=2013-11-18}}</ref>。
