'''喬治斯·「尤格」·藍西莫'''（{{lang-el|Γιώργος Λάνθιμος}}，{{lang-en|Georgios "Yorgos" Lanthimos}}，{{bd|1973年|5月27日||}}）是一名[[希臘|希臘]]電影與劇場導演，他曾是[[2004年夏季奧林匹克運動會|2004年雅典奧運]]開閉幕式的創意小組成員之一。<ref name="PressKit">{{cite web|url=http://www.festival-cannes.com/assets/Image/Direct/028999.pdf|title=DOGTOOTH - Press Kit|work=[[Cannes_Film_Festival|Cannes Film Festival]]|publisher=festival-cannes.com|accessdate=2009-05-23}}</ref>。尤格曾四次獲奧斯卡提名，分別是2010年代表希臘參選[[奧斯卡最佳外語片獎|最佳外語片獎]]（《[[非普通教慾|非普通教慾]]》）、2015年的[[奧斯卡最佳原創劇本獎|最佳原創劇本獎]]《[[單身動物園|單身動物園]]》，以及2018年的[[奧斯卡最佳影片獎|最佳影片獎]]與[[奧斯卡最佳導演獎|最佳導演獎]]（《[[真寵|真寵]]》）。

== 生平 ==
2009年，他執導的第三部長片《[[非普通教慾|非普通教慾]]》以爭議性的劇情與風格驚豔[[第62届戛纳电影节|第62屆坎城影展]]，在[[一種注目|一種注目]]單元獲得最佳影片<ref name="Prizes' announcement">{{cite web |url=http://www.festival-cannes.com/en/article/56798.html |title=Un Certain Regard Awards Ceremony|work=[[Cannes_Film_Festival|Cannes Film Festival]]|publisher=festival-cannes.com|accessdate=2009-05-23}}</ref><ref name="indiewire2009">{{cite news|url=http://www.indiewire.com/article/dogtooth_wins_top_cannes_un_certain_regard_prize/|title="Dogtooth" Wins Top Cannes Un Certain Regard Prize|last=Hernandez|first=Eugene |date=2009-05-23|work=[[IndieWIRE|IndieWIRE]]|publisher=indiewire.com|accessdate=2009-05-23}}</ref>，隔年還入圍[[奧斯卡最佳外語片獎|奧斯卡最佳外語片獎]]<ref name="Oscar Nominees">{{cite web |url=http://www.oscars.org/awards/academyawards/83/nominees.html |title=Nominees for the 83rd Academy Awards |accessdate=2011-01-25|work=oscars.org}}</ref>；2011年，他以《{{link-en|非普通服務|Alps (film)}}》獲選為[[第68届威尼斯国际电影节|第68屆威尼斯影展]]正式競賽片，並獲得最佳劇本獎。<ref>{{cite web |url=http://www.labiennale.org/en/cinema/festival/awards/ |title=存档副本 |accessdate=2012-09-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20120912001913/http://www.labiennale.org/en/cinema/festival/awards/ |archivedate=2012-09-12 }}</ref>

2015年，他以《[[單身動物園|單身動物園]]》獲得[[第68屆坎城影展|第68屆坎城影展]][[評審團獎|坎城影展評審團獎]]。2017年以《[[圣鹿之死|圣鹿之死]]》赢得[[第70届戛纳影展|第70届戛纳影展]]最佳编剧奖。

*2005年：《{{link-en|吉尼塔 (電影)|Kinetta (film)|吉尼塔}}》（Κινέτα）
*2009年：《[[非普通教慾|非普通教慾]]》（Κυνόδοντας）
