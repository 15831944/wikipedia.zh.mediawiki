'''尤溪县'''地处[[福建省|福建省]]中部，[[三明市|三明市]]所轄。东邻[[闽清县|闽清县]]和[[永泰县|永泰县]]，西连[[大田县|大田县]]和[[沙县|沙县]]，南接[[德化县|德化县]]，北毗[[南平市|南平市]][[延平区|延平区]]，有“'''闽中明珠'''”之称，[[宋明理學|理学家]][[朱熹|朱熹]]的诞生地。面积3463平方千米，居全省各县（市、区）第二位，其中山地418.5万亩、耕地34.3万亩，自然概貌约为“八山一水一分田”。县人民政府驻城关镇。[[File:Zhu_xi.jpg|thumb]]

== 历史 ==
尤溪在四千年前的[[新石器时代|新石器时代]]就有人类劳动生息。

[[唐|唐]][[开元|开元]]二十九年（公元741年）建县，以境内的“尤溪”为县名，别名'''沈溪'''。建国后属南平专区管辖，1970年改辖[[三明|三明]]地区（市）。

== 行政区划 ==
辖9个镇、6个乡：
*镇：城关镇、尤溪口镇、梅仙镇、西滨镇、新阳镇、洋中镇、管前镇、西城镇、坂面镇。
*乡：联合乡、汤川乡、溪尾乡、中仙乡、台溪乡、八字桥乡。

== 地理== 
东部为[[戴云山|戴云山]]；境内主要河流有[[尤溪_(河流)|尤溪 (河流)]]，北部[[尤溪口镇|尤溪口镇]]汇入[[闽江|闽江]]；南部与[[大田县|大田县]]、[[德化县|德化县]]交界处建有[[街面水库|街面水库]]。

== 方言 ==
尤溪县地处[[戴云山|戴云山]]脚下，处于[[闽东语|闽东语]]、[[闽中语|闽中语]]、[[闽南语|闽南语]]和[[莆仙语|莆仙语]]交界处。历史上，戴云山脉一带曾经历过多次行政区划的改变，再加上人口流动性大，使得尤溪县的方言多而且复杂。尤溪县也是三明市方言最多、且最复杂的一个县。

这些方言为：

*[[后路话|新桥话]]：通行于尤溪县西南和[[大田县|大田县]]西北，大田人称之为[[后路话|后路话]]。带有闽中语特点。
*尤溪城关话：也被称为[[尤溪话|尤溪话]]，通行于尤溪县中部，影响力最大。兼有闽中、闽南和闽东语特点。
*[[中仙话|中仙话]]：通行于尤溪县东南部。兼有闽中、闽南和闽东语特点，并具有莆仙语成分。
*[[洋中话|洋中话]]：通行于尤溪县东北部。带有闽东语特点。

以上几种方言不仅与周边各语交流困难，就连它们之间都难以通话。<ref>[http://www.fjsq.gov.cn/ShowText.asp?ToBook=196&index=8& 《福建省志·方言志》]</ref>学术界对这些方言的语言学归属存在很大争议。

== 旅游资源 ==
自[[唐|唐]][[贞观|贞观]]年间开发[[坂面|坂面]][[蓬莱山|蓬莱山]]和在顶峰观音佛像起，各地名胜景点逐渐增多。“尤溪八景”即指双峰挂日、二水[[明霞|明霞]]、东崖呼啸、西泽龙潜、玉溪清印、金鲫湛泉、牛岭耕烟和龙台钓雪，后在老八景的基础上新增八景，为文山毓哲、公字成山、源头活水、[[半亩方糖|半亩方糖]]、龙门古洞、[[汤川|汤川]]平洋、高山流水、韦斋垂柳。位于城南的[[南溪书院|南溪书院]]系[[宋|宋]][[理学|理学]]家[[朱熹|朱熹]]诞生所，因此形成了朱子文化旅游风景区。现城中有33个景点，境内还有西滨库区风景区、龙亭湖旅游风景区、倒排岩森林狩猎度假区、九阜山省级[[自然保护区|自然保护区]]、[[洋中|洋中]]镇桂峰历史文化名村、枕头山风景区、伏虎岩风景区等旅游景区。
=== 朱子故里 ===
宋建炎四年庚戌（1130年）九月十五日午时，朱熹（祖籍[[江西|江西]][[婺源|婺源]]）诞生于尤溪的[[郑氏草堂|郑氏草堂]]，因此，尤溪有“朱子故里”之称，朱熹也成为当地的“文化名片”。
=== 特产 ===
坂面[[茶籽油|茶籽油]]、[[红酒|红酒]]、[[梅仙|梅仙]]肉[[光饼|光饼]]、水东[[板鸭|板鸭]]、洋中草根汤、汤川[[小笋|小笋]]等，其中[[管前|管前]][[金桔|金桔]]更是远销各地。尤溪[[金桔|金桔]]和尤溪[[绿竹笋|绿竹笋]]为[[中国地理标志产品|中国地理标志产品]]<ref>[http://kjs.aqsiq.gov.cn/dlbzcpbhwz/bhml/200906/t20090622_118997.htm 《地理标志产品保护规定》实施以来批准的地理标志名录（2005年7月至2009年6月）] {{webarchive|url=https://web.archive.org/web/20110311120900/http://kjs.aqsiq.gov.cn/dlbzcpbhwz/bhml/200906/t20090622_118997.htm |date=2011-03-11 }}</ref>。
