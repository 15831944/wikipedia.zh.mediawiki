'''尤文图斯球場'''（{{lang-en|Juventus Stadium}}）是[[意大利|意大利]][[都靈|都靈]]的一座[[足球場|足球場]]。現被[[安联|安联保险公司]]冠名[[贊助|贊助]]稱為'''安聯球場'''（Allianz Arena）。在2009年動工，2011年完工並開始使用，可容納觀眾數41,000人。尤文图斯球場是在尤文图斯之前的主場球場[[迪爾艾比球場|迪爾艾比球場]]上建設的。尤文图斯球場是全意大利第一座由球會全權擁有的足球場。因為在意大利，足球場通常是由地方當局所擁有。

== 揭幕戰 ==
球場耗資1.05億歐元（約11.3億港元）興建而成。2011年9月10日周四正式啟用，尤文图斯更在此與英甲球隊[[諾士郡|諾士郡]]進行友賽。之所以选择该队作为新球场首场比赛的对手，是因为尤文图斯俱乐部刚刚成立时，球衣制作商误将为诺士郡队制作的黑白相间款式队服寄给了尤文图斯，而后尤文图斯俱乐部将错就错使用该款队服进行比赛，然后一直沿用至今。尤文图斯由[[盧卡·東尼|盧卡·東尼]]先開紀錄，[[李爾·曉士|李爾·曉士]]隨後為諾士郡扳成1:1完場。<ref>[http://hk.soccer.nextmedia.com/showart.cfm?sec_id=25391&art_id=15601314&iss_id=20110910 祖 記 新 球 場 開 幕] {{webarchive|url=https://web.archive.org/web/20111113001414/http://hk.soccer.nextmedia.com/showart.cfm?sec_id=25391 |date=2011-11-13 }} 波網. 蘋果日報 (香港). 2011年09月10日.</ref>

== 季票銷售 ==
尤文图斯過往在奧林匹克球場的入座率偏低，不過在2011年8月已有2.3萬球迷購入2011-12球季的季票，比起2010-11球季增長了56%，足見新球場所帶來的吸引力及不俗的季票銷售情況。<ref name="newstadiumtic">[http://news.now.com/home/sports/player?newsId=139799737 尤文图斯新主場帶挈季票熱賣] NOW news. 2011年08月03日.</ref>

== 設施 ==
尤文图斯的新球場設有41,000個座位，球場優點是不設跑步徑，令觀眾可以更接近足球場。此外，新球場亦有全意大利最完善的配套設施，包括購物商場及4,000個泊車位置。<ref name="newstadiumtic" />
