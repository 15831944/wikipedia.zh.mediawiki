'''尤里·伊万诺维奇·奥努夫里恩科'''（{{lang-ru|Юрий Иванович Онуфриенко|Yuri Ivanovich Onufriyenko}}，{{bd|1961年|2月6日|}}）是[[俄罗斯|俄罗斯]]已退役[[宇航员|宇航员]]，俄罗斯第84位宇航员，世界第342位宇航员，[[无线电台呼号|呼号]]“[[斯基泰人|斯基夫]]”（{{lang|ru|Скиф|Skif}}）<ref>{{cite web |title=Позывные космонавтов |url=http://www.gctc.ru/main.php?id=156 |publisher=GCTC |archiveurl=https://web.archive.org/web/20190421055051/http://www.gctc.ru/main.php?id=156 |archivedate=2019-04-21 |language=ru}}</ref>。他曾两次前往太空，八次进行太空行走任务。
==早年==
1961年2月6日出生于苏联[[乌克兰苏维埃社会主义共和国|乌克兰苏维埃社会主义共和国]][[哈爾科夫州|哈爾科夫州]][[佐洛喬夫區_(哈爾科夫州)|佐洛喬夫區]][[里亞斯涅_(佐洛喬夫區)|里亞斯涅]]。1974年毕业于以[[弗拉基米尔·米哈伊洛维奇·科马洛夫|弗·米·科马洛夫]]命名的[[叶伊斯克高等军事航空学校|叶伊斯克高等军事航空学校]]，毕业后在[[苏联空军|苏联空军]][[远东军区|远东军区]]服役，历任飞行员，高级飞行员<ref name=ru>{{cite web |title=Юрий Иванович Онуфриенко |url=https://www.energia.ru/en/iss/iss04/onufrienko.html|publisher=[[科罗廖夫能源火箭航天集团|SP Korolev RSC Energia]] |accessdate=2019-04-21}}</ref>。
==宇航员生涯==
1989年入选[[加加林宇航员培训中心|加加林宇航员培训中心]]测试宇航员候选人。1989年9月至1991年1月接受了基础航空飞行培训，1991年2月获得测试宇航员资格。1992年2月1日至1994年4月7日在航空航天生态环境工程学院学习，获得[[环境工程|环境工程]]学位<ref name=nasabio>{{cite web |title=Yuri Ivanovich Onufriyenko |url=https://web.archive.org/web/20170113012212/http://www.jsc.nasa.gov/Bios/htmlbios/onufriyenko.html|date=2002 |publisher=NASA |accessdate=2019-04-20}}</ref>。

1991年至1994年，他接受了“和平号空间站远征队”的专业培训。1994年5月至1995年2月通过了“和平号空间站第18次远征”任务第3备用指令长的培训。1995年2月至5月通过了“和平号空间站第19次远征”任务备用指令长的培训。1995年6月至1996年1月通过了[[联盟TM-23|联盟TM-23]]载人飞船正式指令长培训<ref name=ru/>。
===EO-21===
1996年2月21日至1996年9月2日，他担任“和平号空间站远征21”任务指令长，在[[和平号空间站|和平号空间站]]度过了193天19小时，期间一共进行了六次太空行走<ref name=ru/>。

===远征4===
2001年12月5日22时19分，[[奋进号航天飞机|奋进号航天飞机]]从美国[[佛罗里达州|佛罗里达州]]的[[肯尼迪航天中心|肯尼迪航天中心]]升空<ref>{{cite web |title=STS-108 Mission Archives |url=https://www.nasa.gov/mission_pages/shuttle/shuttlemissions/archives/sts-108.html |publisher=NASA |accessdate=2019-04-21}}</ref>，它原计划11月29日升空，然而由于一些特殊情况被推迟发射。航天飞机上共载有7名航天员，其中3人为国际空间站[[远征4|远征4]]任务成员，奥努夫里恩科担任指令长，美国宇航员[[卡尔·欧文·沃尔兹|卡尔·欧文·沃尔兹]]和[[丹尼尔·布尔施|丹尼尔·布尔施]]是随航工程师<ref>{{cite journal |author=辛航 |title=奋进号升空一波三折 |journal=国际太空 |date=2002年 |issue=2 |issn=1009-2366}}</ref>。

2002年6月19日17时58分，奋进号航天飞机降落在[[爱德华兹空军基地|爱德华兹空军基地]]，奥努夫里恩科等队员安全返航<ref>{{cite web |title=STS-111 Mission Archives |url=https://www.nasa.gov/mission_pages/shuttle/shuttlemissions/archives/sts-111.html |publisher=NASA |accessdate=2019-04-21}}</ref>。
