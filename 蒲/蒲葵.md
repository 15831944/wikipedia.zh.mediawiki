'''蒲葵'''（[[学名|学名]]：''{{lang|la|Livistona chinensis}}''）又名'''扇叶葵'''，是[[棕榈科|棕榈科]][[蒲葵屬|蒲葵屬]]的一种植物。

== 形态 ==
常绿[[乔木|乔木]]，单干直立粗大，老茎中部较粗，表面有少量棕皮和叶鞘包被。叶大，簇生于茎顶，叶柄粗壮，具相互分离的尖锐倒刺，叶有一米长左右，呈三菱状。掌状多裂，中央区不分裂联合呈扇状，前端下垂。腋生肉穗花序，多分枝而疏散，排列成圆锥花序状。雌雄异株，黄绿色花。椭圆形[[核果|核果]]，成熟后为蓝黑色，形似橄榄，9－10月果熟。

== 分布 ==
原产于[[中国|中国]]东南部、[[日本|日本]]南部以及[[臺灣島|臺灣島]]。

== 用途 ==
其叶可制为[[蒲扇|蒲扇]]、[[蓑笠|蓑笠]]、[[掃把|掃把]]等。根、叶、果均可入药。

== 變種 ==
*''Livistona chinensis'' var. ''chinensis'' 
*''Livistona chinensis'' var. ''boninensis'' Becc. ，亦被指為另一[[物種|物種]][[小笠原蒲葵|小笠原蒲葵]] (''Livistona boninensis'')，而非[[變種|變種]]
*''Livistona chinensis'' var. ''subglobosa'' (Hassk.) Becc.
