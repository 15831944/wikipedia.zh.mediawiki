《'''考槃馀事'''》四卷，是明代[[屠隆|屠隆]]的一部关于生活艺术的著作，成书于1590年。

《考槃馀事》分十五笺<ref>明屠隆著 《考槃馀事》 金城出版社 2012年 304页 ISBN 978-7-5155-0230-4</ref>
# 书笺：论书，刻地，印书，藏书，观书等。
#  帖笺：古今帖辨，戏鱼堂帖，二王帖，西岳华山庙碑，[[王涣之|王涣之]][[陀罗尼经|陀罗尼经]]，[[开元寺|开元寺]]碑，真书千文，阅古堂记等。
#  画笺：古画，唐画，宋画，元画，国朝画家，看画法，学画，裱画，挂画等。
#  [[纸|纸]]笺：唐纸，宋纸，元纸，国朝纸，高丽纸，染宋笺法，造松信笺等。
#  [[墨|墨]]笺：论墨，古制墨法，[[朱万初|朱万初]]墨。
#  [[毛笔|笔]]笺：笔法，毫，管，式，工，藏，涤，笔经等。
#  琴笺：论琴，[[古琴|古琴]]色，古琴纹，古琴灰，古琴材，琴徽，[[琴弦|琴弦]]，[[琴台|琴台]]，[[琴室|琴室]]，唐琴，宋琴，元琴，国朝琴等。
#  香笺：论香，角沉香，片速香，降真香，兰香，黑芸香，[[香炉|香炉]]，香盒，香盘等。
#  [[茶|茶]]笺：茶品，[[虎丘|虎丘]]，天池，[[六安|六安]]，[[龍井茶|龙井]]，[[天目|天目]]，采茶，日晒茶，[[花茶|花茶]]，江水，井水，丹泉，洗茶，[[茶具|茶具]]等。
#  盆玩笺：盆玩，瓶花，拟花荣辱。
#  鱼鹤笺：鹤品，金鱼品。
#  山斋笺：山斋，药室，花榭，[[佛堂|佛堂]]，[[茶寮|茶寮]]等。
#  起居器服笺：榻，短榻，神椅，坐团，被，帐，禅衣，道服，冠，道扇，披云巾等。
#  文房器具笺：笔格，砚山，笔床，笔筒，砚匣，墨匣，[[印章|印章]]，印色池，[[镇纸|镇纸]]，压尺，裁刀，[[剪刀|剪刀]]等。
#  游具笺：鱼竿，舟，[[葫芦|葫芦]]，提盒，酒樽等。
曾任英国[[维多利亚和阿尔伯特博物馆|维多利亚和阿尔伯特博物馆]]远东馆馆长的柯律格（Craig Clunas），在研究明代屠隆的《考槃馀事》及其同时代人[[高濂|高濂]]《[[遵生八笺|遵生八笺]]》和稍后[[文震亨|文震亨]]《[[长物志|长物志]]》等书后，认为屠隆的《考槃馀事》可等辑录自《[[遵生八笺|遵生八笺]]》，但屠隆此书，无论在当代和后世，影响都比较深远，此书可能是《长物志》材料来源之一<ref>Craig Clunas, Superfluous Things: Material Culture and Social Status in Early Modern China, University of Hawaii Press 2004, ISBN 0-8248-2820-8, pp. 29-30</ref>。
