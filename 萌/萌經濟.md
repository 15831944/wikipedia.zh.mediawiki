[[File:くまモン＠山口県_2012_(8363546077).jpg|thumb]]是日本萌經濟非常成功的一個典範，圖為印有熊本熊圖案的服裝]]
'''萌經濟'''是指通過有關可愛的周邊系列產品而催生的商機。自1970年代以來，“可愛”已經成為日本文化的重要要素，舉凡[[大眾文化|大眾文化]]、[[娛樂|娛樂]]、[[衣著|衣著]]、[[飲食|飲食]]、[[玩具|玩具]]、[[打扮|打扮]]、[[行為|行為]]與[[禮節|禮節]]等等，無不受其影響。外國觀察家發現日本的可愛文化很有趣，但有時又顯得怪異，因為日本人把這種文化運用在幾乎所有的生活領域中，甚至是[[人口統計學|人口統計學]]也是如此。這種現象在別的文化裡很可能代表的就是將小孩的玩意用在嚴肅的場合，讓人覺得很輕佻（例如政府出版品、公共告示牌、辦公室、軍方的廣告以及民航客機等等，不勝枚舉）。

== 萌經濟的流行 ==
[[File:Ana.b747.pokemon.arp.750pix.jpg|thumb]]波音747機身的[[神奇寶貝|神奇寶貝]]圖案]]
萌經濟在日本隨處可見，從大企業到街角的市場、國家級政府機關、病房，甚至是地方政府機關。<ref> [http://www.wired.com/wired/archive/7.12/cute.html ''Wired'', "Cute Inc."] URL accessed [[April_19|April 19]], 2006.</ref><ref> [http://www.businessweek.com/bwdaily/dnflash/jun2002/nf20020625_7574.htm ''Business Week'', "In Japan, Cute Conquers All"].</ref>許多大小公司行號運用可愛的[[吉祥物|吉祥物]]來買賣商品與提供客戶服務。舉例如下：
* 《[[精靈寶可夢系列|精靈寶可夢系列]]》的角色[[皮卡丘|皮卡丘]]被用來裝飾[[全日本空輸|全日本空輸]]的民航機機身。
* 偵探動漫[[名偵探柯南|名偵探柯南]]的主角[[工藤新一|工藤新一]]。
* 日本大型銀行[[里索那控股|理索納銀行]]用小白兔形象的[[Miffy|Miffy]]（源自荷蘭的兒童圖書）來裝飾[[ATM|ATM]]的提款卡。
* 猴子形象的{{tsl|en|Monkichi|Monkichi}}出現在[[保險套|保險套]]的包裝上。<ref>[http://www.sampsonstore.com/neovision/condom/product/details.is?cat_id=168&product_id=377 Monkichi condoms]. URL accessed [[September_30|September 30]], 2006.</ref>
* 日本的[[都道府縣|都道府縣]]各具有代表的吉祥物。
* [[日本郵政公社|日本郵政公社]]郵筒上的"Yū-Pack"吉祥物。{{fact|time=2008-4-13}}
* 日本郵政公社也將可愛吉祥物用在郵票上面。
* 日本警方有些有[[萌|萌系]]吉祥物，有時也用來裝飾派出所。
* 可愛商品在日本大受歡迎，成人小孩都很喜歡。<ref>[http://www.mynippon.com/photos/photo94.htm Kawaii or cute Japanese products]. URL accessed [[April_19|April 19]], 2006.</ref><ref>[http://www.sfgate.com/cgi-bin/article.cgi?file=/gate/archive/2004/07/14/helkit.DTL ''SFGate Asian'' "Pop: How Hello Kitty Came to Rule the World"]. URL accessed [[April_19|April 19]], 2006.</ref>日本最大的兩個專門生產這種商品的公司分別是[[三麗鷗|三麗鷗]]（設計「[[Hello_Kitty|Hello Kitty]]」的公司）以及{{link-ja|San-X公司|サンエックス}}<ref>San-X（サンエックス，San Ekkusu）株式会社</ref>(設計「[[Kogepan|Kogepan]]」、「[[Nyan_Nyan_Nyanko|Nyan Nyan Nyanko]]」和「[[Rilakkuma|Rilakkuma]]」)。 

可愛也用來形容一個人在穿著上很時髦<ref>[http://www.newyorker.com/fact/content/?020318fa_FACT ''The New Yorker'' "FACT: SHOPPING REBELLION: What the kids want"]. URL accessed [[April_19|April 19]], 2006.</ref><ref>[http://www.time.com/time/asia/arts/magazine/0,9754,131022,00.html ''Time Asia'': "Arts: Kwest For Kawaii"]. URL accessed [[April_19|April 19]], 2006.</ref>，常指幼童穿的衣服，或者是任何給人看起來就很可愛的衣服。這種衣服常常(但不總是)以[[襞襟|襞襟]]的造型呈現，各色混以粉白（例如粉紅色）為基調，輔以玩具或包包之類的小裝飾，看起來像動漫裡的人物。<ref>[http://www.wired.com/wired/archive/7.12/cute.html ''Wired'': "Cute Inc."] URL accessed [[April_19|April 19]], 2006.</ref>

== 日本人的觀感 ==
作為一種文化現象，“可愛”已經成為日本文化的一環，也成為了日本的國家象徵。《[[酷日本|酷日本]]》（日語：{{lang|ja|クール・ジャパン}}）的作者[[杉山知之|杉山知之]]相信，“可愛”早已深植在日本那種以和為貴的文化裡。[[武藏大學|武藏大學]]社會學教授[[栗田宣義|栗田宣義]]指出“可愛”在日本是一個“很有魔力的詞”，只要是人們可接受的想要的事物都是可愛的。<ref name=ap2006>Quotes and paraphrases from: {{cite news|title=Cuteness a hot-selling commodity in Japan|author=Yuri Kageyama|date=June 14, 2006|publisher=[[Associated_Press|Associated Press]]|url=http://www.washingtonpost.com/wp-dyn/content/article/2006/06/14/AR2006061401122.html}}</ref>

另一方面，對可愛抱持懷疑態度的人認為可愛其實是人類童心未泯的象徵。<ref name=ap2006>Quotes and paraphrases from: {{cite news|title=Cuteness a hot-selling commodity in Japan|author=Yuri Kageyama|date=June 14, 2006|publisher=[[Associated_Press|Associated Press]]|url=http://www.washingtonpost.com/wp-dyn/content/article/2006/06/14/AR2006061401122.html}}</ref>不過有些日本人認為“可愛”文化讓人覺得日本人始終不願意長大，逃避殘酷的現實，也危及了傳統文化。但是也有人認為可愛文化替日本賺取了大筆的外匯，甚至為日本文化外交力量（[[軟實力|軟實力]]）創造有利籌碼。<ref>[http://www.nownews.com/2009/03/13/11490-2421980.htm 本向外國推廣「卡哇伊文化」　動漫人物成文化大使], [[Nownews|Nownews]], 2009年3月13日</ref><ref>[http://news.epochtimes.com.tw/6/10/27/39668.htm 卡哇伊熱潮 扭轉日本文化]</ref><ref>[http://fashion.japan-i.jp/cht/article/go_kawaii.html “最可愛”的國度 日本 體驗東京的“卡哇伊”文化]</ref>

== 對其他文化的影響 ==
可愛商品在東亞的國家特別流行，例如韓國、台灣和香港。<ref>[http://www.wired.com/wired/archive/7.12/cute.html ''Wired'': "Cute Inc."] URL accessed [[April_19|April 19]], 2006.</ref> 其中尤以台灣受其影響最深。

在某些亞洲與西方文化當中，喜愛日本流行文化的人經常將日文的“可愛”（'''kawaii''', {{lang|ja|可愛い}}） 與其他日文單詞連用，但實際上卻造成誤用。“可愛”一般只用在[[御宅族|御宅族]][[次文化|次文化]]裡，美國歌手[[關·史蒂芬妮|關·史蒂芬妮]]卻曾經在她的[[MV|MV]]當中隨口說出這個詞。<ref>[http://www.salon.com/ent/feature/2005/04/09/geisha/print.html Salon.com: "Gwen Stefani neuters Japanese street fashion to create spring's must-have accessory: Giggling geisha!"]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}. URL accessed [[April_19|April 19]], 2006.</ref>

在北美的企業裡，可愛文化與[[漫畫|漫畫]]的影響力隨處可見。{{fact|time=2008-4-13}}[http://www.epochtimes.com/b5/12/11/23/n3736423.htm|美國梅西百貨凱蒂貓花車遊行]
