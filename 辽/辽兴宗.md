'''遼興宗耶律宗真'''（{{bd|1016年|4月3日|1055年|8月28日|catIdx=Yelu耶律宗真}}），[[辽朝|契丹]]第七位[[皇帝|皇帝]]（1031年6月25日－1055年8月28日在位），[[契丹名|契丹名]]'''只骨'''。在位24年，享年40歲，[[謚號|謚]]'''孝章皇帝'''。他是[[遼聖宗|遼聖宗]]的長子，母乃宮女[[蕭耨斤|蕭耨斤]]。

== 生平 ==

耶律宗真生于1016年，其后，由辽圣宗的皇后[[蕭菩薩哥|蕭菩薩哥]]抚养。《[[辽史|辽史]]》记耶律宗真为“圣宗长子”<ref>《[[辽史|辽史]] 第十八卷 本纪第十八》</ref>，实际上辽圣宗第六子[[耶律宗愿|耶律宗愿]]的生年是1008年或1009年，耶律宗真和同母弟[[耶律宗元|耶律宗元]]应该是辽圣宗最年幼的两个儿子。虽然年幼，但与其他四位皇子相比，只有耶律宗真兄弟的生母蕭耨斤出身于[[契丹萧氏|契丹萧氏]]，其他皇子的生母出身于汉族或不详。

[[太平_(辽朝)|太平]]元年（1021年），耶律宗真被冊立為[[皇太子|太子]]，太平十年（1030年）六月判北南院枢密院事。太平十一年（1031年6月25日）夏六月己卯，辽圣宗逝世，同时，耶律宗真繼承皇位，改[[年號|元]]'''[[景福_(辽朝)|景福]]'''。興宗繼位後，其母順聖元妃蕭耨斤自立為皇太后[[攝政|攝政]]，並把聖宗的齊天皇后迫死。並重用了在聖宗時代被裁示永不錄用的貪官污吏以及其娘家的人。

景福二年十一月，興宗上太后尊號為法天應運仁德章聖皇太后（法天太后），而興宗被群臣上尊號為文武仁聖昭孝皇帝，改元重熙。

[[重熙|重熙]]三年（1034年），法天太后企圖廢掉興宗，改立次子[[耶律宗元|宗元]]（遼史作重元），重元告訴其兄興宗，興宗發動政變，迫法天太后「躬守慶陵」。大殺太后親信。七月，興宗親政。

興宗在位時，遼國勢已日益衰落。而有興宗一朝，奸佞當權，政治腐敗，百姓困苦，軍隊衰弱。面對日益衰落的國勢，興宗連年征戰，多次征伐[[西夏|西夏]]；逼迫[[宋朝|宋朝]]多交納歲幣，反而使遼內部百姓怨聲載道，民不聊生。興宗還迷信[[佛教|佛教]]，窮途奢極。興宗曾與其弟宗元賭博，一連輸了幾個城池。

他對自己的弟弟宗元非常感激，一次酒醉時答應百年之後傳位給宗元，其子[[耶律洪基|耶律洪基]]（後來的遼道宗）也未曾封為皇太子，只封為[[天下兵馬大元帥|天下兵馬大元帥]]而已。種下了道宗繼位後，宗元父子企圖謀奪帝位的惡果。

重熙二十四年八月初四日（1055年8月28日），興宗駕崩。

== 南北朝的提法 ==
契丹辽兴宗欲易國書稱南北朝。宋臣[[梁適|梁適]]說：「宋之為宋，受之於天，不可改也。契丹亦其國名，自古豈有無名之國哉。」遂止。<ref>《宋史》（卷二百八十五·列傳第四十四）：「契丹欲易國書稱南北朝，（梁）適曰：「宋之為宋，受之於天，不可改也。契丹亦其國名，自古豈有無名之國哉。」遂止。」</ref>

== 評價 ==

* [[元朝|元朝]]官修[[正史|正史]]《[[辽史|辽史]]》[[脱脱|脱脱]]等的評價是：“兴宗即位，年十有六矣，不能先尊母后而尊其母，以致临朝专政，贼杀不辜，又不能以礼几谏，使齐天死于弑逆，有亏王者之孝，惜哉！若夫大行在殡，饮酒博鞠，叠见简书。及其谒遗像而哀恸，受宋吊而衰绖，所为若出二人。何为其然欤？至于感富弼之言而申南宋之好，许谅祚之盟而罢西夏之兵，边鄙不耸，政治内修，亲策进士，大修条制，下至士庶，得陈便宜，则求治之志切矣。于时左右大臣，曾不闻一贤之进，一事之谏，欲庶几古帝王之风，其可得乎？虽然，圣宗而下，可谓贤君矣。 ”<ref>
