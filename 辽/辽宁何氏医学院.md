'''辽宁何氏医学院'''（{{lang-en|'''He University'''}}）是一所设立于2011年的民办医学类普通高等院校。学校位于中国[[辽宁省|辽宁省]][[沈阳市|沈阳市]]。

== 基本概况 ==
辽宁何氏医学院是以医学相关专业为主体的民办普通高等院校。1999年9月，何氏眼科医院创始人[[何伟|何伟]]与[[沈阳医学院|沈阳医学院]]合作创立了沈阳医学院何氏眼科视光学院，即辽宁何氏医学院的前身，何伟任校长。学校建立初期依托沈阳医学院设施和师资办学，设置临床医学及眼视光学等专业。2007年，学校棋盘山校区建成并整体迁入，此后专业设置和招生规模逐渐扩大。2011年，学校转设为普通本科高等院校。至2017年，已经涵盖医学、理学、工学、管理学、文学、艺术学等学科门类，拥有19个本科专业，13个高职专业，在校生7000余人<ref name=":0">{{Cite web|url=http://www.huh.edu.cn/intro/info_jianjie.php|title=辽宁何氏医学院简介|accessdate=2018-04-04|author=|date=|last=辽宁何氏医学院|work=www.huh.edu.cn|publisher=}}</ref>。

== 办学理念 ==
辽宁何氏医学院的办学理念是“'''一切为了学生'''<ref name=":1">{{Cite web|url=http://www.upln.cn/index.php?m=content&c=index&a=show&catid=436&id=556|title=辽宁何氏医学院2016-2017学年本科教学质量报告|accessdate=2018-04-04|author=辽宁何氏医学院|date=|work=www.upln.cn|publisher=辽宁省普通高等学校本科教学管理平台}}</ref>”。这一理念主要突出了学生本位的教育观。但因为和同城的[[中国医科大学|中国医科大学]]，沈阳医学院等公立院校的办学水平之间存在差距，该理念并未得到全体学生的理解。

== 培养目标 ==
辽宁何氏医学院的培养目标是培养“'''具备审美意识、独立人格、自由精神、知行合一的民族复兴人'''<ref name=":1" />”。

== 历史沿革 ==

*1999年9月 何氏眼科医院与[[沈阳医学院|沈阳医学院]]共建的沈阳医学院何氏眼科视光学院成立<ref>{{Cite web|url=http://www.hsyk.com.cn/index.php?a=shows&catid=68&id=45|title=沈阳医学院何氏眼科视光学院成立|accessdate=2018-04-04|author=何氏眼科|date=|publisher=}}</ref><ref>{{Cite web|url=http://www.chinaglassesonline.com/optical_view.php?id=173|title=沈阳医学院—何氏眼科视光学院|accessdate=2018-04-04|author=中国眼镜在线|date=2006-12-15|work=|publisher=|archive-url=https://web.archive.org/web/20180405024211/http://www.chinaglassesonline.com/optical_view.php?id=173#|archive-date=2018-04-05|dead-url=yes}}</ref>。
*2004年6月 沈阳医学院何氏眼科视光学院更名为沈阳医学院何氏视觉科学学院，成为全国统招的[[独立学院|独立学院]]<ref>{{Cite web|url=http://www.moe.gov.cn/srcsite/A03/moe_634/200407/t20040701_180636.html|title=2004年具有普通高等学历教育招生资格的高等学校名单|accessdate=2018-04-04|author=教育部|date=|last=|work=www.moe.gov.cn|publisher=}}</ref>。
*2011年6月 沈阳医学院何氏视觉科学学院脱离沈阳医学院，从独立学院转设成为普通本科院校，并更名为辽宁何氏医学院<ref>{{Cite web|url=http://www.moe.gov.cn/srcsite/A03/s181/201105/t20110504_120126.html|title=教育部关于同意沈阳医学院何氏视觉科学学院转设为辽宁何氏医学院的通知|accessdate=2018-04-04|author=教育部|date=|publisher=}}</ref>。

== 对外交流 ==
2015年6月，辽宁何氏医学院与日本[[九州大学|九州大学]]之间签订了学生交换协议<ref>{{Cite web|url=http://www.isc.kyushu-u.ac.jp/intlweb/agreeview/univ-detail.php?area_id=1&country_id=7&univ_id=2248|title=何氏医学院：協定締結校｜九州大学｜|accessdate=2018-04-04|work=www.isc.kyushu-u.ac.jp|language=ja}}</ref>。

2017年7月起，辽宁何氏医学院临床医学本科专业招收国际留学生就读。

学校有部分专业开展英语（临床医学）、日语（护理学）、韩国语（医学美容）等外语强化教学。但截止2017年底，并未派出本科生交换留学。

== 校园轶事 ==
校长[[何伟|何伟]]在每年的开学典礼时会手持一块红砖进行演讲。因为演讲内容贴近生活有亲近感，从而被学生爱称为“大伯”。部分学生也因此常自称“小何”，形成独特的校园文化。

== 黑车事故 ==
2015年3月，四名学生搭乘非法运营的面包车在返回学校途中与公交车发生车祸，造成两人死亡、两人受伤的严重后果。事故发生后，学校呼吁学生选择安全合法的交通手段出行。沈阳客运集团也将168路支线公交车的运营班次增加为每小时一班。但校区出行不便的问题没有得到根本改善<ref>{{Cite news|url=http://liaoning.lnd.com.cn/htm/2015-03/09/content_4207225.htm|title=微面载4名大学生撞公交2死2伤|author=北国网|date=2015-03-09|work=|accessdate=2018-04-05|archive-url=https://web.archive.org/web/20150711043017/http://liaoning.lnd.com.cn/htm/2015-03/09/content_4207225.htm#|archive-date=2015-07-11|dead-url=yes}}</ref>。
