'''辽宁石油化工大学'''，简称'''辽石化'''、'''抚油'''，为中国辽宁省抚顺市望花区丹东路的一所[[公立大学|公办全日制普通高等学校]]。

== 校史 ==
1950年，'''大连石油学校'''创建于[[大连|大连]]，为新中国第一所石油类院校。1953年，迁至抚顺改为'''抚顺石油学校'''。1958年，改为'''抚顺石油学院'''。2000年2月由[[中国石油化工集团公司|中国石油化工集团公司]]划转为辽宁省人民政府领导

2002年2月，更名为辽宁石油化工大学。

== 校园 ==
截至2017年5月，校园占地1906亩，藏书243.4万册<ref>{{cite web | language = zh-hans | publisher = 辽宁石油化工大学官网>学校总揽>正文 | title = 学校概况 | url = http://www.lnpu.edu.cn/list2.jsp?urltype=tree.TreeTempUrl&wbtreeid=12726 | author = 来源：辽宁石油化工大学官网 | date = 2017年5月 | accessdate = 2017年5月}}</ref>。

== 院系设置 ==
辽宁石油化工大学现设有如下院系，分别为：研究生学院、信息与控制工程学院、机械工程学院、经济管理学院、马克思主义学院、体育学院、艺术设计学院、继续教育学院、民族教育学院、化学化工与环境学部、计算机与通信工程学院、石油天然气工程学院、理学院、外国语学院、矿业工程学院、国际教育学院、创新创业学院。

==提拔门==
2011年2月14日，在[[华声论坛|华声论坛]]辣眼时评栏目中，有一位名叫“鬼文子”的网友发帖爆料，称该校国际教育学院副院长王圣淇出生于1986年，在学校工作仅三年就被提拔为副院长。有网友称王圣淇为当地某高官子弟。不过有学校工作人员称是为了“培养年轻干部”<ref>{{cite web | language = zh-cn | publisher = 中国广播网 | title =24岁女孩任大学副院长 被疑为市长女儿| url = http://www.cnr.cn/zgsp/jrtj/201102/t20110216_507681571.html| date = 2011-02-16 | accessdate = 2011-02-16 }}</ref>。

== 知名校友 ==

*[[蔡希有|蔡希有]]，1982年毕业于抚顺石油学院（现辽宁石油化工大学）自动化专业本科，历任中国石化董事、高级副总裁，中国中化集团公司党组成员、总经理。2016年2月6日涉嫌严重违纪，接受组织调查。2018年12月26日蔡希有以受贿罪判处有期徒刑十二年，并处罚金人民币三百万元。
