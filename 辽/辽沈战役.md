'''辽沈战役'''是[[中国近代史|中国近代史]]中[[第二次国共内战|第二次国共内战]]的“[[三大战役|三大战役]]”之一，1948年9月12日开始，同年11月2日结束，共历时52天。[[中国共产党|中国共产党]]称之为'''辽沈-{}-战役'''，[[中國國民黨|中國國民黨]]称之为'''遼西-{}-会战'''，又作'''遼瀋會戰'''。<ref>[http://lov.vac.gov.tw/Protection/Content.aspx?i=25&c=3 行政院退輔會榮民文化網--遼瀋會戰]</ref>

中共將領[[林彪|林彪]]、[[罗荣桓|罗荣桓]]指挥[[东北人民解放军|东北人民解放军]]在[[中國東北地区|中國東北地区]]向被分割于[[沈阳|沈阳]]、[[长春|长春]]、[[锦州|锦州]]等地的[[国民党军|国民党军]]发起进攻，以伤亡6.9万人的代价，消灭及收编敌东北[[剿匪总司令部|剿匪总司令部]]总司令[[衛立煌|衛立煌]]所率领的1个剿匪总司令部、4个兵团部、11个军部、33个师部、旅部團部不計，共47.2万人，并解放了[[中国东北地区|东北]]全境<ref name="战史简编">{{cite book |author=国防大学《战史简编》编写组| title=中国人民解放军战史简编 |edition=2001年修订版 |publisher=解放军出版社|date=2002 |ISBN = 978-7-5065-0029-6}}</ref>。东北人民解放军获得完全胜利，辽沈战役后，中国人民解放军首次在兵力数量方面超越国民党军。

== 背景 ==
[[抗日战争|抗日战争]]结束后，东北被[[苏联红军|苏联红军]]占领，苏联在东北缴获日军各种火炮1836门。时局此时非常复杂，国共双方都急欲在苏联撤军后独占东北<ref>{{cite book |authors = 张文杰、郭辉 |title = 中国革命战争纪实·抗日战争·八路军卷 |year = 2007 |location = 北京 |publisher = 人民出版社 |ISBN = 978-7-01-005024-9 |pages=782-783}}</ref>。
由于[[国军|国军]]主力集中在中国西部地区，无法在短时间内到达东北，而中共在河北建有冀热辽抗日根据地，因此捷足先登。1945年9月6日，冀热辽军区第十六军分区司令员[[曾克林|曾克林]]率[[八路军|八路军]]进入[[沈阳|沈阳]]，随即成立了沈阳卫戍司令部，中共开始接收苏联缴获日本关东军的武器装备<ref>{{cite news |url=http://culture.gmw.cn/newspaper/2014-08/30/content_100262470.htm |date=2014-08-30 |title=抗战胜利日的沈阳记忆 |accessdate=2015-03-06 |author=高巍 publisher=华商晨报 |deadurl=yes |archiveurl=https://web.archive.org/web/20150725232011/http://culture.gmw.cn/newspaper/2014-08/30/content_100262470.htm |archivedate=2015-07-25 }}</ref>。在東北，蘇聯自日本關東軍手中接收轉交給中國共產黨武器裝備<ref>{{cite book|author=何應欽|title=《中國與世界前途》|publisher=正中書局|location=台北|quote=「前年（1970年）蘇聯[[塔斯社|塔斯社]]發表的數字，當屬正確。計有：步槍六八五、八九七枝；輕重機槍二九、八二二挺；手槍六〇、三七七枝；火炮一二、四四六門；步機彈一八〇、九九〇、〇〇〇發；手槍彈二、〇三五、〇〇〇發；砲彈二、〇七〇、〇〇〇發；戰車三八三輛；裝甲車一五一輛；卡車一五、七八五輛；飛機一、〇六八架；飛機油一〇、〇〇〇噸；炸彈六、〇〇〇噸；艦艇一、四〇〇艘（計五四、六〇〇噸）」|date=1974年10月}}</ref>{{rp|91－92}}。
此后，中共中央又从全国各战略区调集11万部队和2万干部进入东北加入[[东北民主联军|东北民主联军]]，并由[[林彪|林彪]]指挥前线<ref>{{cite journal |authors=钟占兴、沈兆璜 |title=东北军区暨东北野战军组织沿革|journal=军事历史|year=1994|issue=2|pages=47-51}}</ref>。11月，国军[[新1军|新1军]]、[[新6军|新6军]]从[[秦皇岛|秦皇岛]]登陆，突破伪满州国边界[[山海关|山海关]]并占领[[葫芦岛|葫芦岛]]、[[锦州|锦州]]等地，苏联未有异议；东北共军按照中共中央“让开大路，占领两厢”的指示，在背靠[[朝鲜|朝鲜]]、[[蒙古|蒙古]]、苏联的地区建立中共根据地<ref>{{cite book | author = 中央文献研究室 | coauthors =  |title = 刘少奇年谱(上) | url =  | date = 1996 | location = 北京 | publisher = 中央文献出版社 |ISBN = 9787507302967 |page = 531 }}</ref>。1月10日，由周恩来、张群在[[乔治·卡特里克·马歇尔|乔治·卡特里克·马歇尔]]调停下于陪都重庆签订[[一月停战令|一月停战令]]，蒋介石对国军下达“第一次停战令”。

至1946年3月初，东北主要城市仍在苏联占领之下，中共与国民政府被允许在部分苏占城市活动。[[国民政府|国民政府]]被允许将部分伪满军和民间武装改编为[[保安团|保安团]]，在苏占城市以维持治安名义存在，但因其历史背景，苏军限制其武器装备，并允许[[八路军|八路军]]接收部分苏联不需要的日本[[关东军|关东军]]武器装备；双方主力，[[东北民主联军|东北民主联军]]以农村为根据地，[[国军十大王牌军|国民党五大主力]]中的新1军、新6军美械军，及第13军、第52军、第60军半美械军，共5个军由美国海军海运到秦皇岛登陆，并推进至锦州伺机而动；另有2个军尚在路上。

1946年3月中旬苏联开始撤军，因2月[[反苏运动|反苏运动]]，苏军未举办交接仪式，低调一走了之。[[东北民主联军|东北民主联军]]在苏军自3月8日于[[抚顺|抚顺]]、[[吉林|吉林]]撤军后占领了部分城市，在某些地区与当地[[保安团|保安团]]发生小规模军事摩擦。国军亦于3月12日苏军自[[沈阳|沈阳]]撤军后，在次日由[[新1军|新1军]]占领沈阳，并以此为前进基地进一步深入东三省。新1军占领沈阳受周恩来认同，周恩来希望东北问题和平解决，指示中共军队不要占领沈阳，承认国民党军队有权进占苏军撤离区，但也要国民党政府承认共产党在东北的地位。随后东北民主联军撤出[[抚顺|抚顺]]，3月21日国民党52军进占抚顺。3月27日，由周恩来、张治中、吉伦军事三人组，进一步签订了[[东北停战协议|东北停战协议]]。

1946年3月31日，国民党军[[东北行辕|东北行辕]]主任[[熊式辉|熊式辉]]令[[东北保安司令部|东北保安司令部]]副司令长官[[郑洞国|郑洞国]]、[[梁华盛|梁华盛]]，集中5个军11个师的兵力由[[沈阳|沈阳]]沿[[安沈路|安沈路]]向南满、沿[[中长路|中长路]]向北满大规模挺进，指向[[本溪|本溪]]、[[四平|四平]]等[[东北民主联军|东北民主联军]]占领的城市。
4月1日，国民党军第52军第25师、新6军第14师分别从[[抚顺|抚顺]]、[[辽阳|辽阳]]出发向[[本溪|本溪]]发起进攻未果；4月7日，国民党军第52军第25师、新6军第14师和新增调的1个团，兵分3路向本溪发起第2次进攻，再次失利；4月16日，东北保安司令部司令长官[[杜聿明|杜聿明]]病愈返渖阳，并亲自部署攻打本溪；4月20日，东北民主联军第3纵队主力奉命北上参加保卫四平作战，4月28日国民党军以新6军、第52军、第71军下属的5个师共5万余人分3路第3次进攻本溪，终于5月7日完全占领本溪。随即新6军转向北上，协助攻打四平。
4月14日蘇聯于[[長春|長春]]撤軍後一個半小時，解放軍對长春發動攻勢，並在4月18日攻克長春<ref>梁敬錞譯註：《馬歇爾使華報告書簽注》，北京：[[中國社會科學院|中國社會科學院]]近代史研究所</ref>{{rp|202－203}}，国民政府宣称此舉率先違反[[1月停战令|1月停战令]]<ref>秦孝儀主編：《中華民國重要史料初編--對日抗戰時期第七編（三）》，台北</ref>。
4月18日，杜聿明撤作战不力的梁华盛，换郑洞国指挥国民党新1军第30师，开始攻打[[四平|四平]]；至5月18日中共中央决定守城部队全部撤离四平，5月19日四平被国军完全占领。
4月22日蘇军撤出[[齐齐哈尔|齐齐哈尔]]，解放軍24日占领齐齐哈尔。
4月25日蘇军撤出[[哈尔滨|哈尔滨]]，解放軍28日占领哈尔滨。

同时中國共產黨實行總動員，加緊破壞[[北寧鐵路|北寧路]]。<ref name="年表"/>{{rp|49}}中国共产党[[东北民主联军|东北民主联军]]“发动攻势”，[[林彪|林彪]][[罗荣桓|罗荣桓]]部队兵力“占据优势”，且装备精良，故一直以“攻势”为主。同年夏，關內各地國軍屡战屡败，东北地區由於防地遼闊，“兵力不足”；加以交通被解放軍所破壞，完全处于“被动”。4月，[[国军|国军]]于[[本溪|本溪]]、[[四平|四平]]与当地[[东北民主联军|东北民主联军]]展开激战。稍后，[[东北民主联军|东北民主联军]]于[[长春|长春]]、[[齐齐哈尔|齐齐哈尔]]、[[哈尔滨|哈尔滨]]击败当地[[保安团|保安团]]占领这几市。林彪部主力30餘萬，集中於[[四平街|四平街]]附近，企圖阻止國軍前進。<ref name="現代史">李守孔著：《中國現代史》，台北：[[三民書局|三民書局]]，1973年9月，ISBN 978-957-14-0663-3</ref>{{rp|154}}<ref name="現代史"/>{{rp|162}}國軍杜聿明部迫於情形，自“5月起”在東北發動“反攻”。<ref name="現代史"/>{{rp|154}} 
”5月7日，國軍攻占[[本溪|本溪]]，并激戰於[[四平街|四平街]]一帶，5月19日攻占四平。<ref name="現代史"/>{{rp|154}}[[四平战役_(1946年)|四平战役]]后，东北民主联军主动撤往[[松花江|松花江]]以北，与国军隔江对峙<ref>{{cite book|author=翁衍慶|title=林彪的忠與逆：9/13事件重探 |url = http://books.google.com/books?id=hyn6OmgWYDgC&pg=PA111|date=2012-07-01|publisher=新銳文創|isbn=978-986-6094-93-4|pages=111-112}}</ref>。5月23日，国军攻占[[长春|长春]]，5月28日攻占[[吉林|吉林]]，6月5日直逼[[哈尔滨|哈尔滨]]。<ref name="現代史"/>{{rp|154}}6月6日，在[[马歇尔|马歇尔]]壓力下，蒋介石对国军下达暂时的[[六月停战令|第二次停戰令]]，同意休战15天；后来又延长8天，于關內在华中、华北集结军队。<ref name="年表">陳布雷等編著：《蔣介石先生年表》，台北：傳記文學出版社，1978年6月1日</ref>{{rp|48}}7月，蔣徇調人之請，兩次頒令停止衝突，並主張召開五人小組，商談政治問題，與三人軍事小組同時進行，中國共產黨“堅決反對”。<ref name="年表"/>{{rp|48}}

1946年7月，鉴于国军攻势，[[中共中央东北局|中共中央东北局]]扩大会议在哈尔滨召开，通过《[[七七决议|七七决议]]》，号召干部下乡，巩固农村根据地<ref>{{cite web |url = http://dangshi.people.com.cn/GB/151935/176588/176594/10555486.html |title = 东北局扩大会议(1946年7月3-11日) |accessdate =2015-03-06 |date = |publisher = 中国共产党新闻网 }}</ref>。

1946年10月，国军通过“南攻北守，先南后北”的方针，派遣十万部队进攻南满。至11月下旬，中共南满根据地仅剩下[[临江县|临江县]]、[[长白县|长白县]]、[[抚松县|抚松县]]、[[濛江县|濛江县]]、[[辑安县|辑安县]]五座小县，中共中央东北局决定派遣陈云、萧劲光、萧华成立南满分局，陈云任分局书记。<ref>{{cite book|author=[[朱佳木|朱佳木]]、[[中共中央文献研究室|中共中央文献研究室]] |title= 陈云年谱（上卷）|year=2000|location=北京 |publisher=中央文献出版社 |ISBN = 9787507307870}}</ref>{{rp|471-476}}。11月，蔣又一次对国军下达“第三次全面停戰令”<ref name="年表"/>{{rp|49}}。12月17日，国军集中第52军第195师、第2师，第71军第91师，新一军新30师，第60军182师及暂编21师6个师下属的共10个团，对东北民主联军坚守的南满根据地临江地区发动进攻，东北民主联军胜利进行了[[三下江南、四保临江战役|三下江南、四保临江战役]]。最终东北民主联军成功守卫了临江地区，此战役的结束标志着国共双方在东北战场上攻守局面彻底改变{{r|syzs|pages=166-182}}，随后中共在东北转入反攻。

2月底，东北民主联军南渡松花江，5月初大舉發動猛攻，形成“关内小打、关外大打”局面。5月17日攻佔[[懷德|懷德]]，5月21日攻佔[[公主岭市|公主屯]]，永吉、長春、四平街乃陷於孤立<ref name="現代史"/>{{rp|162}}。6月6日，解放軍攻陷[[熱河|熱河]][[赤峰|赤峰]]<ref name="現代史"/>{{rp|162}}。6月16日，國軍被迫放棄[[安東|安東]]省會<ref name="現代史"/>{{rp|162}}。解放軍開始進攻遼寧瀋陽<ref name="現代史"/>{{rp|162}}。6月，解放軍猛攻四平街，國軍守軍[[陳明仁_(中國軍事家)|陳明仁]]几近全灭，蔣特電嘉勉。<ref name="年表"/>{{rp|51}}

1947年1月31日，解放軍延安總部公布1946年7月至1947年1月全国范围内戰爭幾項重要統計：七個月中，國軍攻佔[[解放區|解放區]]城市199座，解放軍攻佔[[國統區|國統區]]城市100座<ref name="民國史大事記"/>{{rp|8276}}。解放军共殲滅國軍56個整旅（師）、其中1月份殲國軍12個旅；斃、俘其將級軍官103名，其中俘獲87名，俘獲後逃跑4名，擊傷12名<ref name="民國史大事記"/>{{rp|8276}}。国军战绩未知。

1948年1月1日，[[東北民主聯軍|東北民主聯軍]]改稱[[東北人民解放軍|東北人民解放軍]]，原聯軍總部機關改為東北軍區兼東北野戰軍領導機關，[[林彪|林彪]]任軍區、野戰軍司令員兼政治委員，[[高岗|高岗]]、[[呂正操|呂正操]]、[[周保中|周保中]]、[[蕭勁光|蕭勁光]]任副司令員，[[羅榮桓|羅榮桓]]、[[高崗|高崗]]、[[陳雲|陳雲]]、[[李富春|李富春]]任副政治委員，[[譚政|譚政]]任政治部主任，[[劉亞樓|劉亞樓]]、[[伍修权|伍修权]]任參謀長<ref>{{cite book |authors = 张明金、刘立勤 |title=中国人民解放军历史上的200个军区 |year=2010 |publisher=解放军文艺出版社 |location=北京 |pages=343-344 |ISBN = 978-7-5033-2252-5}}</ref>。

東北野战军在[[东北1947年冬季战役|冬季攻势]]（1947年底到1948年3月）之后，为适应“大兵团、正规化、攻坚战”需要进行整编。至1948年8月，东北野战军编有12个[[步兵|步兵]]纵队、1个[[炮兵|炮兵]]纵队、1个[[铁道|铁道]]纵队、15个独立师、3个[[骑兵|骑兵]]师，共54个师约70万人，加上地方军区武装，合计103万余人{{r|syzs|page=295}}。已控制东北97%的土地和86%的人口。当时東北是全国唯一一个人民解放军兵力超过国民党军的地区，因此[[中国共产党中央军事委员会|中共中央军委]]把战略决战的第一个战场选在东北<ref>{{cite book |title=中国革命史手册 |author=陈荣华 |year=1986 |publisher=华中师范大学出版社 |page = 227}}</ref>。

1948年春，東北人民解放軍林彪部連續在東北發動七次攻勢，2月7日攻佔[[遼陽|遼陽]]，2月26日攻佔[[營口|營口]]，3月14日攻佔四平街，3月19日攻佔[[永吉|永吉]]，国民党集中兵力堅守長春、瀋陽、錦州三個互不相连的地區，補給全賴空運接濟，因運輸量有限，三地食糧燃料奇缺，[[長春|長春]]每日餓斃者不下百人。<ref name="現代史" />{{rp|162}}

國軍在冬季攻势中战事陷入不利，[[蒋介石|蒋介石]]用[[卫立煌|卫立煌]]替代[[陈诚|陈诚]]指挥东北地区国军战事。1948年1月1日，[[參謀總長|參謀總長]]兼[[國民政府主席|國民政府主席]]東北行轅主任陳誠在瀋陽發表元旦《告東北軍民書》，宣布危機已過，戰備完成，即將由[[鐵嶺|鐵嶺]]、瀋陽、新民三路（以新六軍、新三軍為右路，新編第七十一軍、新編第一軍為中路，新五軍為左路）出兵，向瀋陽西北扇形推進，與東北人民解放軍決戰<ref name="民國史大事記">{{cite book |editors = 李新 總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編 |title=中華民國史大事記|location=北京|publisher=中華書局|year=2011|ISBN = 978-7-1010-7998-2}}</ref>{{rp|8479}}。东北剿匪总司令部辖有4个兵团14个军44个师（旅），加上地方保安团共约55万人，但被分割压缩在沈阳、长春、锦州三个互不相连的地区内。总司令卫立煌驻沈阳，有[[周福成|周福成]]第八兵团、[[廖耀湘|廖耀湘]]第九兵团等共8个军（[[国民革命军第五十三军|第五十三军]]、[[国民革命军新编第一军|新编第一军]]、[[国民革命军新编第三军|新编第三军]]、[[国民革命军新编第六军|新编第六军]]、[[国民革命军第四十九军|第四十九军]]、[[国民革命军第五十二军|第五十二军]]、[[国民革命军第七十一军|第七十一军]]以及[[中国青年军|青年军]]第二〇七师{{NoteTag|[[中国青年军|中国青年军]]第二〇七师为3旅3团制，实力接近于军，隶属第六军，第六军另辖第一九五师，已归属第四十九军，第六军实际是空番号，辽沈战役被俘国军将领回忆，多直接称为第二〇七师。}}）；剿总副总司令[[郑洞国|郑洞国]]驻长春指挥第1兵团（辖[[国民革命军新编第七军|新编第七军]]、[[国民革命军第六十军|第六十军]]）；剿总副总司令兼锦州指挥所主任[[范汉杰|范汉杰]]驻锦州，有[[卢浚泉|卢浚泉]]第六兵团等共4个军（[[国民革命军第九十三军|第九十三军]]、[[国民革命军新编第五军|新编第五军]]、[[国民革命军新编第八军|新编第八军]]、[[国民革命军第五十四军|第五十四军]]），防守[[义县|义县]]至[[秦皇岛|秦皇岛]]一线，防守重点为锦州、[[锦西|锦西]]{{r|syzs|page=296}}。 由于部分[[北宁铁路|北宁铁路]]为东北人民解放军所控制，长春、沈阳通向[[山海关|山海关]]内的陆上交通已被切断。

卫立煌接任东北剿总总司令后，专注于部队整训，充实兵员，补充装备，构筑工事，把主力集中于沈阳，任解放军打到什么地方、守军如何告急都不为所动，卫立煌认为可以长期固守沈阳，亦认为长春是个包袱“是着死棋”。由于锦瀋间交通被解放军截断，为减轻运输压力，部队缺额多的师重建以及新建接受兵员的需运到山海关、锦州等地就近关内与港口整补{{NoteTag|据[[范汉杰|范汉杰]]回忆，沈阳运来锦州、山海关重建的师有：新编第六军之暂编第六十二师（锦西）、第四十九军之第二十六师（山海关）、第五十二军之暂编第五十四师（塔山）、第七十一军之第八十八师（锦州），此外还有冬季攻势中被歼灭的新编第五军军部。另有新建的暂编第五十五、暂编第五十七师等。}}。而蒋介石因为对东北国军仅靠空运补给不能负担，力主留第八兵团守瀋陽，打通锦瀋线交通，将主力调往锦州，必要时放棄補給不易的長春，防止東北共軍進入山海關，安定關內局勢，伺機轉攻收復[[關外|關外]]。卫立煌认为沈阳主力单独出辽西背水侧敌行动会被消灭，与蔣意见相左，焦点在于是否“将主力撤至锦州的问题”{{r|qlj|page=47-59}}。1948年8月3日，蔣在南京國防部召集軍事檢討會議，參加者80餘人，由[[何應欽|何應欽]]主持；會議檢討兩年來作戰方針、作戰指揮、編制裝備、新兵補充、後勤供應、士氣、情報等方面問題，並作出如下軍事決策：「東北力求穩定，華北力求鞏固、西北阻匪擴張，華東、華中則加強進剿，一面阻匪南進，一面攻打匪之主力。」<ref name="民國史大事記"/>{{rp|8653-8654}}同日，行政院會議通通委任[[馬占山|馬占山]]、[[羅卓英|羅卓英]]、[[萬福麟|萬福麟]]、[[鄭洞國|鄭洞國]]為[[東北剿匪總司令部|東北剿匪總司令部]]副司令<ref name="民國史大事記"/>{{rp|8654}}。8月20日，東北剿匪總司令部副總司令范漢杰由[[南京市_(中华民国)|南京]]飛返錦州<ref name="民國史大事記"/>{{rp|8662}}。

1948年2月，[[中共中央军委主席|中共中央军委主席]][[毛泽东|毛泽东]]致电东北野战军提出“是以封闭蒋军在东北加以各个歼灭为有利”<ref name="文集">{{cite book |title=毛泽东军事文集 |volume = 第四卷 |publisher=中央文献出版社 |author = 毛泽东 |editors = 中共中央文献研究室、中国人民解放军军事科学院 |location=北京|page = 391 |year=1993 |ISBN = 9787800217388 }}</ref>。1948年3月冬季攻势结束后，4月18日东北野战军向中共中央军委提出“先打长春”{{r|syzs|page=300}}。5月发起长春外围作战同时准备打击由沈阳出援的国军，但是战斗中发现部队对坚固设防的大城市攻坚存在问题{{r|syzs|page=301}}，而且卫立煌以是“圍城打援”圈套为由在沈阳按兵不动。于是中央军委与东北野战军在“打长春”与“南下北宁线作战”多次反复<ref>{{cite book|author=胡绳主编|title=中国共产党的七十年|location=北京|publisher=中共党史出版社 |page =221 |year=1991 |ISBN = 7-8013-6339-6}}</ref>，侧敌南进作战还有后勤补给的担忧{{r|syzs|page=303}}。8月14日，中共中央軍委批准東北野戰軍和東北軍區領導機關正式分開，林彪任東北軍區司令員兼政治委員、東北野戰軍司令員，羅榮桓任東北軍區第一副政治委員、東北野戰軍政治委員<ref name="民國史大事記"/>{{rp|8659}}。至8月中央军委对东北野战军南下作战的顾虑提出批评，随后东北野战军作出南下切斷錦州、山海关守線，截斷北寧鐵路，封閉、殲滅國軍的战役决心。9月，中共中央在[[西柏坡|西柏坡]]召开政治局扩大会议（“九月会议”），确定在大约五年内“从根本上打倒国民党的反动统治”，为此，要求各战略区在[[第二次国共内战|第二次国共内战]]的第三年准备打更大规模的歼灭战<ref>{{cite book |title=中共党史导读 |url = http://books.google.com/books?id=f_wMAAAAYAAJ |year=1991 |publisher=中国广播电视出版社 |page=966}}</ref>。9月中央军委批准东北野战军的计划，在《关于辽沈战役作战方针》中提出“要置长春、沈阳两敌于不顾，将主力使用于锦州—[[唐山|唐山]]一线……”“确立打前所未有的大歼灭战的决心，即在卫立煌全军来援时敢于同他作战。”{{r|mx|page=1336}}以求全歼卫立煌集团，完全获得东北地区这一战略[[资源|资源]]重地<ref>{{cite book |title=中国革命史（修订本）|editor=黄干周 |year=1988 |publisher=江西教育出版社 |page = 327}}</ref>。

==== 錦州 ====

1948年9月12日，东北野战军南下，向[[北宁铁路|北宁铁路]][[锦州|锦州]]至[[昌黎|昌黎]]段进攻{{r|syzs|page=307}}，遼瀋戰役開始<ref name="民國史大事記"/>{{rp|8674}}。第2兵团司令员[[程子华|程子华]]指挥部队包围[[绥中|绥中]]、[[兴城|兴城]]等地国军，国军第五十四军紧急向南增援，在兴城以北遭到解放军独立第6师阻击{{r|syzs|page=307-308}}。随后，各独立师主动撤兴城之围，集中力量进攻绥中。东北野战军第11纵队也于此时出击山海关至[[昌黎|昌黎]]各点，于14日攻占昌黎县城；紧接着又攻占[[北戴河|北戴河]]等地，并击退六十二军对昌黎的反扑{{r|syzs|page=308}}。28日，解放军攻占绥中，切断锦州至秦皇岛间北宁线，关闭国军由关内至关外的陆上通道{{r|syzs|page=308}}。

[[File:W020090827535348204915.jpg|thumb]]
9月15日，錦西國軍再次增援興城，被解放軍頑強擊退<ref name="民國史大事記"/>{{rp|8676}}。与此同时，东北野战军第4、第9纵队在9月16日突然包围[[义县|义县]]，使驻防国军无法向锦州收缩。20日前后，东北野战军3纵、2纵5师、炮纵到达义县，接替执行包围。此后，4纵绕过锦州南下，9纵继续向锦州进攻，与第8纵队向国军攻击{{r|syzs|page=308}}。9月23日，錦州第九十三軍第二十二師北援義縣，進至[[七里河镇|七里河]]附近，遭解放軍阻擊，被消滅一部，其餘退回[[葛文碑|葛文碑]]、[[薛家屯|薛家屯]]一帶防守<ref name="民國史大事記"/>{{rp|86780}}。9月24日，蔣電召衛到南京，命令衛組織西進兵團，由瀋陽馳援錦州；衛則主張應由關內出兵解錦州之圍後，瀋陽主力方可西進；蔣允由關內增兵3個軍，由錦西東進，並以海空軍增援之，惟瀋陽主力必須立即西進<ref name="民國史大事記"/>{{rp|8681}}。卫決定空运第四十九军增援锦州<ref name="民國史大事記"/>{{rp|8681}}。錦州外圍激戰，北二郎洞、舊飛機場、薛家屯、葛文碑等地均受到解放軍攻擊；東北野戰軍第4縱隊殲滅興城東北地區守敵第五十四軍1個團，切斷錦州、錦西、興城間之聯繫；第3縱隊切入錦州以北，攻克達子營、五姓屯、白老虎屯<ref name="民國史大事記"/>{{rp|8682}}。经过激烈交战，至25日，解放军攻占葛文碑、薛家屯、[[帽儿山|帽儿山]]等地，封闭义县、锦州间国军联系{{r|syzs|page=309}}。東北野戰軍第9縱隊殲敵5000餘人，占領錦州東北和北部外圍陣地，並封鎖錦州機場，完全切斷錦州至義縣交通<ref name="民國史大事記"/>{{rp|8682}}。国军增援锦州的陆上通道被切断。9月26日，衛立煌自南京返瀋陽，奉命西援錦州，參謀總長顧祝同同行，監督西援執行情況，顧、衛飛錦州上空時，電告范漢杰：立即實施空運第四十九軍援錦計劃；空軍運輸機50架，由瀋陽向錦州空運第四十九軍，至9月28日僅運到第九十七師（欠1個團），解放軍以炮火控制機場，當即擊毁飛機5架，至9月29日空運被迫停止<ref name="民國史大事記"/>{{rp|8682}}。9月28日，东北野战军第9纵队附炮兵纵队抵近锦州机场并发动炮击，致使国军仅运了第七十九师2个团就因机场被炮火封锁而停止{{r|lrh|page=274}}。

同时，9月26日，東北人民解放軍總部就先打山海關或先打錦州問題電中央軍委請示；9月27日，毛澤東覆電：如不能同時打兩處，則應先集中兵力打錦州<ref name="民國史大事記"/>{{rp|8683}}。9月27日，东北野战军第4、第7纵队攻占[[高桥|高桥]]、[[塔山|塔山]]﹐殲滅國軍暫編第六十二師和第五十四軍第一九八師各一部，截斷錦西第五十四軍與錦州之聯繫；至此，解放軍完全包圍錦州，[[北寧鐵路|北寧鐵路]]山海關以東至義縣間，除錦州、錦西兩孤立據點外，都為解放軍所控制<ref name="民國史大事記"/>{{rp|8683}}。

9月28日，衛立煌、廖耀湘再晤顧祝同，廖等提出「進軍營口，再折向錦州解圍」之方案，為顧祝同所拒絕，堅持必須先新民集中，不容再貽誤戰機；衛、廖請顧返南京向蔣陳述東北將領意見，同時表示準備向新民集結部隊<ref name="民國史大事記"/>{{rp|8684}}。

9月29日，第4纵队攻占[[兴城|兴城]]，歼灭国军4000余人{{r|syzs|page=309}}。10月1日，3纵司令员[[韩先楚|韩先楚]]统一指挥部队攻克义县，歼灭暂编第二十师，俘虏师长[[王世高|王世高]]{{r|syzs|page=310}}{{r|刘统|page=563}}。战斗结束之后，东北野战军炮兵司令员[[朱瑞_(1905年)|朱瑞]]入城时不幸牺牲，这是东北野战军在战役中损失的官阶最高者<ref>{{cite news|url=http://xh.xhby.net/mp2/html/2011-05/17/content_364753.htm|title=朱瑞：人民炮兵奠基人|date=2011-05-17|accessdate=2015-06-10|publisher=新华网|deadurl=yes|archiveurl=https://web.archive.org/web/20150610132555/http://xh.xhby.net/mp2/html/2011-05/17/content_364753.htm|archivedate=2015-06-10}}</ref><ref>{{cite book|author=《中国人民解放军军史》编写组|title=中国人民解放军军史：第三卷|year=2010|publisher=军事科学出版社|isbn=978-7-80237-381-5}}</ref>{{rp|263}}。至此，东北野战军经过20天连续作战，歼灭国军20000余人{{r|lrh|page=274}}，完成对锦州的包围{{r|syzs|page=310}}。

9月30日，蔣介石由南京飛抵[[北平|北平]]巡視，[[陸軍大學|陸軍大學]]校長[[徐永昌|徐永昌]]、空軍總司令[[周至柔|周至柔]]、海軍總司令[[桂永清|桂永清]]、聯勤總司令[[郭懺|郭懺]]、總統府參軍[[羅澤闓|羅澤闓]]、青年部長[[陳雪屏|陳雪屏]]以及[[俞濟時|俞濟時]]等隨行<ref name="民國史大事記"/>{{rp|8686}}。10月1日，蔣召開軍事會議，研究援錦方案，決定從[[华北|华北]]抽调第六十二、獨立第九十五師、第九十二军第二十一師及[[山东|山东]][[烟台|烟台]]第三十九军兩個師，海運[[葫蘆島|葫蘆島]]，會同錦西第五十四軍及暫編第六十二師，共11個師，組成東進兵團，由第十七兵团司令[[侯镜如|侯镜如]]指挥，從[[锦西|锦西]]北上，增援锦州<ref name="民國史大事記"/>{{rp|8687}}。蔣中正10月2日亲赴沈阳<ref>{{cite book |title=中国人民解放军历史简编|editor=郭清树|year=1985|publisher=辽宁大学出版社}}</ref>{{rp|190}}，令以沈阳国军主力由第九兵团司令[[廖耀湘|廖耀湘]]指挥西进，企图东西对进解锦州之围，并与东北野战军进行决战{{r|syzs|page=311}}。而卫立煌对由沈阳派出主力兵团解锦州之围持有异议{{r|刘统|page=560}}。
[[File:Final_Attack_on_Jinzhou.jpg|thumb]]

9月30日，东北人民解放军司令部、政治部組成前線指挥所，由林彪、羅榮桓率領，乘火车从[[双城|双城]]出发，南下赴遼瀋前线{{r|syzs|page=312}}。10月2日，林彪在列车上收到国军增援的消息后，一度产生犹豫，提出回师进攻[[长春|长春]]<ref>{{cite book|title=世纪风云：中国共产党史话 |id=中册|author=刘凤舞 |year=1997 |location=郑州|publisher=河南人民出版社 ISBN =7-215-04140-9}}</ref>{{rp|207}}{{r|刘统|page=565}}，受到毛泽东批评<ref>{{cite book |author = 毛泽东 |editor = [[中国人民解放军军事科学院|中国人民解放军军事科学院]] |title=毛泽东军事文选 內部本 |url = http://books.google.com/books?id=z79lAAAAIAAJ |year=1981 |publisher=中国人民解放军战士出版社 }}</ref>{{rp|476-478}}。林彪在罗荣桓劝说下{{r|lrh|page=276}}，很快又打消回师的想法<ref>{{cite book |title=辽沈战役 |editor=中国人民解放军历史资料丛书编审委员会 |location=北京 |publisher=解放军出版社|year=1993 |ISBN = 7-5065-2163-6}}</ref>{{rp|19}}。东北野战军随后调整部署，以第2、3、7、8、9纵队以及6纵第17师（该师擅长攻坚战）附炮兵纵队攻打锦州，以第2兵团指挥第4、11纵队以及冀察热辽独立第4、6师防守塔山，第1纵队两个师置于塔山与锦州之间的高桥为预备队，第5、6、10纵队以及1纵3师、内蒙古骑兵第1师置于[[新民|新民]]附近，第12纵队和长春外围12个独立师继续围困长春<ref name="战史简编" />{{r|lrh|page=281}}。攻打锦州的同时，以阻击国軍西由锦西以及東由瀋陽来的援军，令攻城軍不會陷入反包圍之險{{r|lrh|page=281}}。10月9日开始，东北野战军集结25万人扫清锦州所有外围据点包围锦州城，至13日，全部夺取锦州外围阵地{{r|刘统|page=597}}{{r|syzs|page=317}}。范汉杰指挥的国军第六兵团陷入包围{{r|qlj|page=63-78}}。

在蒋介石亲自督促下{{r|syzs|page=320}}，锦西、[[葫芦岛|葫芦岛]]守军第五十四军、海运增援的第六十二军及独立第九十五师，在指挥官侯镜如到达前，暂由第五十四军军长[[阙汉骞|阙汉骞]]指挥{{r|syzs|page=321}}，于10月10日发动攻势，在解放军第4、11纵队阻击下，攻击受挫{{r|syzs|page=322}}。11日，第十七兵团司令侯镜如到达接替指挥，因第九十二军防务无法交接仍留原地仅运到第二十一师{{r|syzs|page=322}}。侯镜如已经秘密与中共联络，宣称援救锦州打进去也出不来，故意放慢动作，推迟11天才率领部队达到{{r|刘统|page=580}}<ref>{{cite web|url=http://paper.people.com.cn/hqrw/html/2013-02/06/content_1205701.htm?div=-1 |title= 国民党中将侯镜如之子揭秘：父亲不算真正的“卧底”|author=陈晓燕、刘畅| publisher=环球人物|date=2013-2-4|accessdate=2018-4-7 }}</ref>。13日，国军调整部署再次进攻，并动用海军[[重慶號巡洋艦|重慶號巡洋艦]]火炮支援。至15日解放军攻下锦州时始终[[塔山阻击战|被阻于塔山]]{{r|qlj|page=241-253}}。而由山东海运增援的第39军拖延到16日才到达。整个塔山阻击战中，国军伤亡6千余人；东北野战军伤亡人数在致中央军委电报中提到“第4、11纵队在守备战及小战斗中各伤亡五六千人”。<ref>《林彪、罗荣桓、刘亚楼拟令第四、第十一纵队入关致中央军委电》，1948年10月29日。</ref> 作战结束后，东北野战军第四纵队12师34团获得“[[塔山英雄团|塔山英雄团]]”称号{{r|syzs|page=323}}。国军由沈阳西进的廖耀湘第九兵团被东北野战军第5、6、10纵队牵制，也畏惧被解放军包围，希望向[[营口|营口]]撤退，于10月11日出兵攻占东北野战军补给线上的[[彰武|彰武]]后，在彰武、[[新立屯镇|新立屯]]地区徘徊<ref>{{cite book|title=新编中国现代史 （1919.5-1949.10）|editor=郭绪印|year=1996|publisher=上海人民出版社|isbn=7-208-02451-0}}</ref>{{rp|473}}，等待锦西的第十七兵团东进与锦州国军会师而迟延出击{{r|qlj|page=152-188}}。对于彰武被占，东野和中共东北局的反应是从[[内蒙古|内蒙古]]进入辽宁直到[[阜新|阜新]]火速动员修成两条新的补给通道<ref>{{cite book|author=王树增|title=解放战争 下|year=2009|location=北京|publisher=人民文学出版社|isbn=978-7-02-007373-3}}</ref>{{rp|87}}。

10月14日，东北野战军向锦州发起总攻，次日晚占领锦州，此战东北野战军伤亡24,548人；歼灭守军国军第六兵团及所属第九十三军、新编第八军共10万餘人，其中死伤1.9万人、被俘虏约8万人{{r|刘统|page=604}}，俘虏东北剿总副总司令[[范汉杰|范汉杰]]，封闭國軍从陆上撤回关内的唯一通道{{r|syzs|page=319}}。10月15日，蒋介石派遣[[徐州剿匪总司令部|徐州剿匪总司令部]]副总司令兼第二兵团司令官[[杜聿明|杜聿明]]为东北剿总副总司令兼冀热辽边区司令官，事实上剥夺卫立煌的战场指挥权，而令其仅能统领在沈阳的国军<ref>{{cite book|title=杜聿明将军|author=郑洞国等|pages=79|year=1986|publisher=中国文史出版社|}}</ref>。

==== 長春 ====
{{Main|長春圍困戰}} 

与此同时，东北野战军[[长春围困战|围困长春]]5个多月，城內缺糧嚴重，国军军队士气低落，百姓餓死者数以萬計，长春景象十分悲惨<ref name="Dikötter2013">{{cite book|author=Frank Dikötter |title=''The Tragedy of Liberation: A History of the Chinese Revolution 1945-1957'' |url=http://books.google.com/books?id=4RDV12RRj8cC |date=2013-09-24 |publisher=Bloomsbury Publishing |ISBN = 978-1-62040-348-8}}</ref><ref>{{cite book |author=[[龍應台|龍應台]] |title=大江大海一九四九 |url = http://books.google.com/books?id=wRAEQgAACAAJ |year=2009 |publisher=天下雜誌 |isbn=978-986-241-049-3 }}</ref>{{rp|70}}。与原来驻长春市西半部的新编第七军相比，从吉林撤下来驻守长春市东半部的[[國民革命軍第六十軍|国军第六十军]]粮食供应更加困难，引起第六十军官兵的颇有意见，士兵虽无饿毙但亦谋兵变<ref>{{cite journal |title=起义纪实 |author=曾泽生 |journal=人物|year=1986|issue=1}}</ref>。辽沈战役开始后，蔣要求长春守军突围<ref>{{cite book |author=《萧劲光传》编写组 |title=萧劲光传 |year=2011 |publisher=当代中国出版社 |location=北京 |isbn=978-7-8017-0995-0}}</ref>{{rp|159}}{{r|刘统|page=611}}。由于粮食断绝官兵体力甚弱已无力突围，加上在锦州城破的压力下，10月17日，第一兵团副司令兼第六十军军长[[曾泽生|曾泽生]]与下属第一八二师师长白肇学、暂编第二十一师师长陇耀经过密谋，率部起义{{NoteTag|起义的[[國民革命軍第六十軍|中华民国国军第六十军]]在1949年元旦后，根据中国人民解放军总部发布命令，改编为[[中国人民解放军第50军|中国人民解放军第50军]]，曾泽生仍担任军长，下辖第148、第149、第150师。另外，隶属第六十军的原来还有第一八四师，1946年5月守海城被包围后起义，重建后1947年又被歼灭一次，再次重建后隶属第六兵团在锦州被歼灭。}}{{r|qlj|page=306-319}}，19日，新编第七军与解放军接洽放下武器投降。10月23日，解放軍攻陷長春，東北剿匪副總司令（兼國軍第一兵团司令官）[[鄭洞國|鄭洞國]]被俘<ref name="現代史"/>{{rp|162}}。东北野战军占领长春，全歼国軍守军第一兵团及所属新编第七军、第六十军等共10万餘人<ref>{{cite journal |title=放下武器 |author=郑洞国 |journal=人物 |year=1986 |issue=1}}</ref>。

=== 第二阶段 ===
{{main|黑山阻击战}}

駐防锦州和长春的国軍被东北野战军消灭后，东北局势对[[国民党軍|国民党軍]]已经极为不利。但蔣仍坚持分别由锦西和沈阳，东西对进收复锦州<ref>{{cite book |title=东北近百年史讲话|author=黑龙江省社会科学院历史研究所 |year=1984 |publisher=黑龙江人民出版社}}</ref>{{rp|246}}{{r|syzs|page=329}}。第九兵团司令官廖耀湘认为不应再继续向锦州前进，亦不愿意退回沈阳。10月16日，廖耀湘与来视察的杜聿明达成共识，计划先占领[[黑山县|黑山]]、[[大虎山镇|大虎山]]，再立即向南撤往营口，或依托营口向北进攻锦州或海运回关内，卫立煌也同意第九兵团撤往营口{{r|刘统|page=629}}。10月18日蔣飛臨瀋陽，指示東北作戰計劃，仍未能迅速挽回頹勢。<ref name="現代史"/>{{rp|162}}在说服蔣改变决策后，第九兵团于21日向黑山发起进攻以掩护兵团向营口撤退，并令第五十二军于23日由[[辽阳|辽阳]]先行进占营口{{r|刘统|page=631}}。东北野战军占领锦州以后，改变原来攻击锦西、葫芦岛的国军的计划{{r|syzs|page=331}}。10月20日令第1、3、8纵队及6纵17师为第一梯队急行军东进；以第2、7、9纵队及炮兵纵队为第二梯队随后跟进，两翼包抄围歼廖耀湘第九兵团。并以第5纵队，第6纵队两个师南下截断第九兵团往沈阳的退路，令第10纵队和1纵3师在黑山、大虎山阻击第九兵团{{r|syzs|page=332}}。

10月17日，廖耀湘令[[國民革命軍第七十一軍|第七十一軍]]军长[[向鳳武|向鳳武]]担任向黑山攻击的前线指挥官，并将[[中国青年军|青年军]]第二〇七师第三旅以及兵团直属炮兵配属给第七十一军加强攻击力量，以掩护兵团主力由[[新立屯镇|新立屯]]通过黑山以东的走廊向营口撤退。完成部署后，在争取蔣中正改变决策过程中又迟误了4天，至21日才下达命令开始发起进攻黑山的战斗{{r|syzs|page=333}}。21日中午，东北野战军电令第10纵队在黑山、大虎山一线死守，阻击国军，掩护主力到达{{r|刘统|page=635}}。东北野战军第10纵队以所属第28、第29、第30师展开防御，其中第28师防守的黑山正面高家屯、101高地、92高地和石头山一线为双方攻防的重点{{r|syzs|page=334}}。由于是短时间仓促建立防御阵地，前沿阵地先修成地面火力点，然后再加强阵地的副防御，只作好表面简陋的防御工事<ref name="梁兴初">{{cite book |url = http://www.oklink.net/lszl/jfzz/jf05.html|author=梁兴初|chapter=黑山阻击战|title=红旗飘飘 |volume = 13 |publisher=中国青年出版社 |pages=189-219| year=1995}}</ref>。23日，国军第七十一军及第二〇七師第三旅对黑山发起攻击，但无法突破东北野战军第10纵队和1纵3师{{r|lrh|page=286}}的防守，攻击受挫{{r|syzs|page=334}}。廖耀湘令[[國民革命軍新編第一軍|新編第一軍]]军长潘裕昆接替向鳳武担任指挥，23日，新编第一军投入战斗，加强对黑山的攻击，双方在前沿阵地反复拉锯，阵地多次易手，第10纵队不断发动反击，国军攻势毫无进展，无法进占黑山{{r|syzs|page=334}}。

由于东北野战军第10纵队死守黑山、大虎山一线，廖耀湘第九兵团被拖住，无法占据黑山、大虎山掩护向营口撤退的通道，丧失大量时间{{r|syzs|page=335}}。东北野战军主力赶到，对第九兵团展开穿插、合围、分割，25日，向营口撤退的道路以及向沈阳的退路被切断{{r|syzs|page=336}}。26日，东北野战军开始围攻第九兵团{{r|syzs|page=338}}，林彪命令各部队以快制快、以乱制乱<ref>{{cite book |author=欧阳青 |title=大授衔：1955共和国将帅授衔档案 |year=2011 |publisher=长城出版社 |location=北京 |ISBN = 978-7-5483-0058-8}}</ref>{{rp|372}}。在一次突然袭击中，东北野战军第3纵队7师21团3营以无畏的勇气打掉了廖耀湘位于胡家窝棚的兵团指挥部<ref>{{cite book |title=四十军征战纪实|author=王树和|year=1997|publisher=白山出版社|isbn=7-80566-542-7}}</ref>{{rp|249}}，同时新编第三军、新编第一军、新编第六军的军部也被打掉，造成整个兵团指挥中枢系統全部混乱，部队陷入瘫痪和崩溃的状态{{r|syzs|page=339}}。至10月28日，东北野战军在黑山、大虎山以东，[[绕阳河|绕阳河]]以西范围内歼灭意图逃往营口的国军第九兵团及所属新编第一、新编第三、新编第六、第四十九、第七十一军5个军12个师共约12万人，第九兵团司令官廖耀湘被俘虏{{r|syzs|page=339}}。新一軍、新六軍美式軍械裝備全部被快递到解放軍手中<ref>{{cite book|title=大決戰（下）：裂岸|author=[[程思遠|程思遠]]|chapter=〈拒蔣調兵與逼蔣求和〉|location=[[香港|香港]]|publisher=中原出版社|year=1991|isbn=9576450462}}</ref>{{rp|366}}。10月26日，蒋介石在日记中写道：“东北全军，似将陷于尽墨之命运。寸中焦虑，诚不知所止矣。”<ref>{{cite book|title=蒋介石秘录|year=1989|publisher=广西人民出版社|isbn=7-219-01035-4}}</ref>{{rp|603}}

=== 第三阶段 ===
[[File:The_Fall_of_Shenyang.jpg|thumb]]在[[九七式中战车|九七式中战车]]的掩护下向[[沈阳|沈阳]]沿线进攻]]

10月29日，遼西會戰後，瀋陽守軍失措；东北剿匪总司令部開始撤退，卫立煌偕高級官員赴[[錦西|錦西]]，瀋陽守軍交第八兵团司令官[[周福成|周福成]]指挥<ref name="民國史大事記"/>{{rp|8710-8711}}辽西作战结束后，東北野战军决定以第1、2、7、8、12纵队包围沈阳；以第9纵队进攻[[营口|营口]]，攻击驻扎营口的第52军，并命令辽宁军区组织部队在辽河上架桥{{r|syzs|page=341}}。東北解放軍攻克[[撫順|撫順]]，殲滅國軍第二〇七師一部；並在[[東陵|東陵]]、[[水泉|水泉]]等地殲滅第五十三軍暫編第三十師一部，逼近瀋陽；第1縱隊攻克[[新民|新民]]，追歼騎兵第二旅一部<ref name="民國史大事記"/>{{rp|8711}}。

10月30日上午，杜聿明奉蔣命令飛瀋陽調整該市防務，因機場混亂不能降落，遂直飛北平向蔣請示錦葫部隊行止；蔣命杜回[[葫蘆島|葫蘆島]]待命，並命衛立煌到葫蘆島指揮，蔣旋返南京<ref name="民國史大事記"/>{{rp|8711}}杜飛返葫蘆島，計劃撤退營口和葫蘆島之部隊<ref name="民國史大事記"/>{{rp|8711}}；衛立煌偕趙家驤、董文琦等由瀋陽飛抵錦西，一週後衛立煌奉蔣令飛北平<ref name="民國史大事記"/>{{rp|8712}}。東北野戰軍攻克本溪、遼陽、鞍山；第12縱隊在瀋陽西吳家荒、大幸屯等地殲滅由新民撤退之第五十三軍第一三〇團，逼近瀋陽西南，於瀋陽市及其近郊包圍瀋陽守軍<ref name="民國史大事記"/>{{rp|8712}}。10月31日，周福成電蔣：「衛立煌棄職逃走，職正調整部署，竭力奮戰，孤守瀋陽待援。」<ref name="民國史大事記"/>{{rp|8713}}東北野戰軍分路向瀋陽攻擊，各路均逼近城郊<ref name="民國史大事記"/>{{rp|8713}}。第12縱隊攻克瀋陽南蘇家屯車站，殲滅第二〇七師暫編第十團；第2縱隊在三間房殲滅第二〇七師第二旅第四團一部，並突入瀋陽市鐵西區，瀋陽以西國軍3個騎兵旅投降<ref name="民國史大事記"/>{{rp|8713}}。同時，東北野戰軍第7、8、9縱隊對營口實行合圍，以切斷瀋陽守軍南撤路線<ref name="民國史大事記"/>{{rp|8713}}。

11月1日，解放軍攻克瀋陽<ref name="民國史大事記"/>{{rp|8713}}。東北野战军第1、2、12纵队以及8个独立师向沈阳市区发起总攻<ref>{{cite book|title=刘震回忆录|author=[[刘震|刘震]]|pages=285|year=1990|publisher=解放军出版社|isbn=7-5065-0434-0}}</ref>{{r|syzs|page=342}}。是日凌晨，東北野戰軍發起總攻，第1、2縱隊為主攻，由西部和西北部突破；第12縱隊由南部突破；獨立第1、3、4師由東部和北部突破<ref name="民國史大事記"/>{{rp|8713-8714}}。10時，第1、2縱隊攻佔鐵西全區，殲滅國軍第二〇七師2個旅大部；第12縱隊攻佔城南飛機場，殲滅守軍1個團<ref name="民國史大事記"/>{{rp|8714}}。下午，沈阳守軍投降，在大東營繼續抵抗之第二〇七師第一旅第二團被消滅<ref name="民國史大事記"/>{{rp|8714}}。国军第五十三军与解放军接洽放下武器投降，仅青年军第二〇七师做局部抵抗也缴械投降，第八兵团司令官周福成被俘虏<ref>{{cite book|title=大決戰（上）：驚濤|author=江深，陳道闊|location=香港|publisher=中原出版社|year=1991|pages=317|isbn=9576450454}}</ref>。是役歼灭東北剿匪總司令部所屬1個兵团部、2个军部、6個師、3個騎兵旅，及守备、直属部队等{{r|syzs|page=343}}，共13.45万餘人，第八兵團司令官周福成以下將官106名被俘<ref name="民國史大事記"/>{{rp|8714}}。是日夜，營口國軍第五十二軍開始登船南撤，至次日，僅撤走該軍軍部及第五十二師<ref name="民國史大事記"/>{{rp|8714}}。

11月2日，東北野战军攻占营口<ref name="民國史大事記"/>{{rp|8715}}。是日上午，第7、8、9縱隊及遼南獨立第二師從三路發起攻擊，戰至10時，殲滅国軍第五十二军第二師全部，第二十五師1個團及軍部運輸團，共1.48萬餘人<ref name="民國史大事記"/>{{rp|8715}}。第五十二军撤退时损失三分之一，第五十二军残部从海上撤往葫芦岛<ref name="年表"/>{{rp|56}}。11月2日，解放軍攻陷瀋阳，衛立煌先期飛北平，部分國軍突圍撤至营口，經海運南下，东北盡失，國軍先後被歼灭47萬人。<ref name="現代史"/>{{rp|162}}國軍自瀋陽、營口撤出東北。<ref name="年表"/>{{rp|56}}辽沈战役至此结束<ref name="現代史"/>{{rp|162}}。

在沈阳被攻占后，锦西、葫芦岛地区的援锦作战的国軍第十七兵團在杜聿明的指挥下，于11月9日经海上撤向关内，次日東北野战军占领锦西、葫芦岛，东北全境解放。{{r|syzs|page=345}}。

据中共官方记录：国民党軍损失4个兵团、11个军部、32个整师、3個騎兵旅、4個守備總隊（相當於師），其他獨立團及特種部隊、直屬部隊、地方部隊，共47.2万人，其中斃、伤5.68万人，俘32.433万人（一说是30.62万人），放下武器投诚（中共方面认定与被俘性质不同）6.49万人（一说是8.3万人），倒戈（中共方面称为起义，国军方面当认为是叛变）2.6万人<ref name="民國史大事記"/>{{rp|8715}}；解放军伤亡6.9213万人<ref name="民國史大事記"/>{{rp|8715}}，其中阵亡14,009人，负伤53,328人，失踪1,874人{{r|刘统|page=674}}。据中共方面统计：空軍各型飛機143架、海軍艦艇26艘。国軍东北剿匪总司令部建制的部队仅有駐山海關的新編第五軍撤退到[[天津|天津]]，第五十二、第五十四军由葫芦岛撤退到[[上海|上海]]。

== 影响、纪念与评价 ==
辽沈战役的失败使得[[国民党政府|国民党政府]]喪失了战略地位极为重要、資源豐富的東北地区以及大量精锐部队，士气遭到重挫{{r|刘统|page=674}}。此战以后，国民党与中共产党的总兵力对比发生逆转。中共在辽沈战役的胜利使当时国共内战的军事形势出现重大转折，东北野战军成为一支战略机动力量很快被投入[[关内|关内]]战场参与[[平津战役|平津战役]]，对关内战场产生决定性影响{{r|syzs|page=353}}。1948年11月14日，毛泽东起草《中国军事形势的重大变化》一文，表示：“原来预计，从1946年7月起，大约需要五年左右时间，便可能从根本上打倒国民党反动政府。现在看来，只需从现时起，再有一年左右的时间，就可能将国民党反动政府从根本上打倒了。”{{r|mx|page=1361}}

1948年11月1日，蔣在[[中國國民黨中央政治會議|中國國民黨中央政治會議]]上説明東北戰局及其失敗原因，並謂東北戰事雖然失利，但解放軍主力尚難即予華北局勢以直接威脅，整個戰局必無可虞，應堅定信心，補救缺陷<ref name="民國史大事記" />{{rp|8714}}。11月3日，杜聿明電蔣請示錦葫守軍撤退事宜；蔣復電：第六十二軍、第九十二軍、獨立第九十五師仍歸華北剿匪總司令部建制，其他各軍撤往上海、南京<ref name="民國史大事記" />{{rp|8716}}。杜聿明即按蔣電示部署撤退<ref name="民國史大事記"/>{{rp|8716}}。

辽沈战役在国际上也取得了相当反响。[[法国共产党|法国共产党]]《[[人道报|人道报]]》评论称中国共产党在东北的胜利“乃是为独立而斗争的一切人民的胜利”。美国[[美联社|美联社]]报道称：“国民党在满洲的败北，已使南京突呈紧张，人们已在公开谈论着政府迁移的可能性。”<ref>新华社电讯：《东北胜利震动中外》，1948年11月5日</ref>1948年11月6日{{r|syzs|page=354}}，美国驻华大使[[司徒雷登|司徒雷登]]报告[[美国国务院|国务院]]，谓中国局勢惡化，任何軍事援助亦于事無補，“因此，我们非常不愿意地得到这样的结论：国民党现政府之早日崩溃是不可避免的了”<ref name="民國史大事記" />{{rp|8720-8721}}。

1948年11月30日，蔣以東北剿匪總司令部總司令[[衛立煌|衛立煌]]「遲疑不決，坐誤戎機，致失重鎮」，明令着即撤職查辦。<ref name="民國史大事記" />{{rp|8740}}

1983年8月9日，[[陈云|陈云]]總結解放军辽沈战役勝利的原因有六點：<ref>{{cite news |url = http://cpc.people.com.cn/GB/69112/83035/83317/83597/5738429.html |title = 对编写《辽沈决战》一书的意见 |date=1983-08-09 |accessdate=2014-09-30|author=陈云}}</ref>
# [[苏联红军|苏联红军]]出兵东北，打败日本[[关东军|关东军]]，为解放军能抢在国軍前面接收，改善解放军的装备创造有利条件。
# 全国各个根据地部队支援[[中国东北地区|东北]]。
# 用正规部队打擊國民黨，巩固后方。
# 在东北进行土地改革，爭取农民支持。
# 东北革命根据地经济基础雄厚，解放军可以不断扩充，而且交通方便，部队机动性强。
# 党中央正确的作战方针。

1991年，[[中國人民解放軍八一電影製片廠|中國八一電影製片廠]]出品電影《[[大決戰|大決戰之遼瀋-{}-戰役]]》，是电影“大決戰”三部曲（另外两部是《[[大決戰|大決戰之淮海-{}-戰役]]》及《[[大決戰|大決戰之平津-{}-戰役]]》）系列之一<ref>{{cite web |date=2011-05-20|accessdate=2015-07-15 |url = http://news.qq.com/a/20110520/000862.htm |title=电影《大决战——辽沈战役》|publisher=腾讯}}</ref>。
