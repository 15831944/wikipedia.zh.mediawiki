'''霽青描金游魚轉心瓶'''是18世紀中國[[清朝|清朝]][[乾隆|乾隆]]年間（西元1736－1795年）的官窯瓷器。為[[轉心瓶|轉心瓶]]，分內外兩層，握器頸旋轉，透過外瓶的[[開光|開光]]看見內瓶繪製的水草魚紋悠游於眼前，如同[[走馬燈|走馬燈]]一般。現存於[[國立故宮博物院|國立故宮博物院]]。

== 年代背景 ==
[[清朝|清朝]]（西元1644－1911年）官窯的生產由皇家主導控制，改善窯工待遇與窯場制度，因此使官窯瓷器能持續維持最佳的品質。盛清時期，[[康熙|康熙]]、[[雍正|雍正]]與[[乾隆|乾隆]]三朝，由皇帝親身指揮與督陶官的監造下，生產出釉彩、技術和紋樣皆為一時之選的官樣作品，有仿古、創新，或融合東西紋飾的裝飾風格。<ref name = "摶泥幻化">{{Cite book|title=摶泥幻化 : 院藏歷代陶瓷|last=|first=|publisher=國立故宮博物院|year=2014|isbn=9789575627126|location=台北 |pages=}}</ref>{{rp|130}}本瓶為[[乾隆|乾隆]]時期的官窯作品。

== 外觀 ==
根據典藏機構的測量，本瓶的資料如下<ref>[http://antiquities.npm.gov.tw/Utensils_Page.aspx?ItemId=314474 國立故宮博物院器物典藏資料檢索系統]</ref>：

高23.5公分、深19.8公分、口徑5.3公分、足徑8.4公分

本體的形式特徵如下：
*瓶身外觀：套瓶造型，瓶口微向外捲，長頸，豐肩，腹部往下收斂，矮圈足外撇，肩飾四個環形繫耳。

*瓶腹內層：底色為淺青湖綠色釉，以[[粉彩_(颜料)|粉彩]]畫[[金魚|金魚]] [[File:霽青描金游魚轉心瓶_金魚紋.jpg|thumb]]、水草及落花。金魚有粉紅、白、橘、黑等色。花主要是白色，少數粉紅，點綴些許淡粉花瓣，花蕊以金釉描飾。水草骨幹繪黑線，深綠釉做葉子效果。

*外層器外壁：霽青釉背景地，以[[描金|描金]]彩繪。
# 口頸：口沿外一圈連續六角形紋內外飾花朵紋，器頸部先飾雙絃紋，下繪如意鉤連紋一周，內填卷草寶相花，外綴瓔珞。
# 肩腹：通體飾以番蓮花及花葉紋。頸肩交接處有如意鉤連紋一圈，腹壁四個海棠菱形鏤空[[開光|開光]]，可見內瓶。
# 圈足：一圈變形蓮瓣紋。
# 器外底：施湖綠釉，中心留白，以[[青花|青花]]書「大清乾隆年製」六字[[篆書|篆書]]款。

== 裝飾之文化脈絡 ==
霽青是形容雨後或霜雪過後天空轉晴之色。以[[鈷|鈷]]為呈色劑的一種藍釉，起源可追溯到[[唐三彩|唐三彩]]，到了元代[[景德鎮|景德鎮]]才發明高溫鈷藍釉瓷器。明、清藍釉習稱「霽藍」，後人因其釉色藍如深海，釉面勻稱潔淨，稱「霽青」。

[[描金|描金]]為清代[[康熙|康熙]]時期開始運用的新興金彩工藝，之前用金箔貼金，容易脫落。製作方式是將金粉容於膠水中，後用繪筆描畫瓷器紋飾，優點是圖案細緻生動，不易脫落，又能與各類彩又相互搭配。清代官窯金彩器分兩類，一類為「仿金或仿古銅彩器」，另一類是金彩構圖細緻的「描金器」<ref>[http://www.360doc.com/content/11/0623/21/6115210_129129371.shtml 淺談明清官窯瓷器中的"金彩"]</ref>，此瓶屬於後者。

內瓶的游魚為[[金魚|金魚]]。魚紋在[[雍正|雍正]]之前，不論是沿襲或仿古依然在傳統窠臼中。至[[乾隆|乾隆]]時期，金魚已成為宮廷市井最主要的觀賞用魚，品種增加，色彩豐富，因而反映於瓷器中<ref>{{Cite book|title=	瓷之纹|last=|first=|publisher=故宮出版社|year=2013|isbn=9787513404433|location=北京 |pages=}}</ref>。

部分清代洋彩瓷器內壁施湖綠色釉，口緣處描金線，本瓶即是一例，推測應是模仿自銅胎畫琺瑯。[[File:霽青描金游魚轉心瓶_瓶口.jpg|thumb]]

== 結構 ==
此瓶分內外二層，可拆解成內瓶的頸部與腹部，和外瓶的肩部與腹部四個部分，各自分別燒製再組裝而成。利用內瓶底凹下的雞心槽，結合外瓶底部突起的雞心紐<ref name=teenager>{{Cite book|title=華麗的淸瓷 : 淸瓷篇|last=|first=|publisher=靑少年叢書故宫文物寶藏續編編輯委員會|year=1992|location=台北 |pages=}}</ref>
，兩者做為卡榫套接在一起，上下相連不相黏，此技法稱「交泰」。再將內瓶頸部黏接，最後外肩部以低溫釉彩烘黏固定，此處仔細觀察有較圓鼓、泥金描飾的痕跡<ref>[http://tech2.npm.edu.tw/da/3d/ch/intro.htm 國立故宮博物院3D虛擬文物展示系統]</ref>。

轉心瓶另一技法稱「玲瓏」，藉由外瓶雕鏤穿透，可從外部看內瓶的圖樣。因此將此平臥頸部旋轉，可帶動內瓶轉動，透過外瓶的開光，可見型態各異的金魚追逐落花、嬉戲悠游的景象，如同[[走馬燈|走馬燈]]一般，故名「[[轉心瓶|轉心瓶]]」。

== 研發監造者 ==
此類[[轉心瓶|轉心瓶]]為[[乾隆|乾隆]]時期御窯廠之監造官[[唐英_(清朝督陶官)|唐英]]（1682-1756）、助理老格精心研發的新品類。唐英實際赴廠監造的歷史可推至[[雍正|雍正]]時期，熟悉皇室品味，配合[[乾隆|乾隆]]皇帝藝術創作旺盛的企圖心，以豐富經驗創燒新奇的品目。特別的是，[[乾隆|乾隆]]八年四月（1743）[[唐英_(清朝督陶官)|唐英]]寫給皇帝的奏章裡，清楚的說明玲瓏交泰瓶正是他與老格共同創研的新式瓷器，由於事前未先向乾隆報備，即自行研發生產，上呈時無不誠惶誠恐希望皇帝改正指教<ref>{{Cite book|title=國寶菁華. 器物篇|last=|first=|publisher=國立故宮博物院|year=2011|isbn=9789575626075|location=台北 |pages=}}</ref>。

如〈恭進奉發及新擬瓷器摺〉奏摺中所述：「……與協造之催總老格料理開工將奉發至得各器，敬謹辦理……今自三月初二日開工之後……又擬得夾層玲瓏交泰瓶等九種，僅供摺送京呈進，其新擬各種系奴才之愚昧。」以資料來看此時期對夾層、轉心器物之製造特別熱衷，是清瓷最高技巧<ref>{{Cite book|title=淸康雍乾名瓷特展|last=|first=|publisher=國立故宮博物院|year=1988|isbn=9575620844|location=台北 |pages=}}</ref>。

[[唐英_(清朝督陶官)|唐英]]創燒與監造的[[轉心瓶|轉心瓶]]，多半以手操作，可近距離觀賞的類型；[[乾隆|乾隆]]十八年（1753）下旨生產結構更複雜的[[轉心瓶|轉心瓶]]後，也逐漸出現尺寸加大至七十公分以上者，此類型由專業人員操作，帶有表演性質。[[轉心瓶|轉心瓶]]具備的旋轉機能，且往更精密的層次發展，或許和[[乾隆|乾隆]]關注造辦處<ref>[http://www.twwiki.com/wiki/%E9%80%A0%E8%BE%A6%E8%99%95 造辦處]</ref>產造，所發展出由發條帶動人物與動植物轉動之機巧，希望運用一種源於西洋鐘錶設計的概念有密切關聯<ref name="multiple">{{Cite book|title=精彩一百 國寶總動員|last=|first=|publisher=國立故宮博物院|year=2011|isbn=9789575626228|location=台北 |pages=}}</ref>。轉心瓶僅[[乾隆|乾隆]]時期才有，數量稀少，[[乾隆|乾隆]]前後都不曾出現，可以算是一種空前絕後的精品<ref name=teenager/><ref>[https://www.academia.edu/10152368/唐英監造轉心瓶及其相關問題 唐英監造轉心瓶及其相關問題]</ref>。

== 備註 ==
[[唐英_(清朝督陶官)|唐英]]曾在上呈皇帝的奏摺中，出現有如[[乾隆|乾隆]]八年〈洋彩魚游春水瓶一對〉，和乾隆十年〈吹青寶蓮描金魚游春水四繫橄欖瓶一對〉等，應是足以和本瓶對照的記載<ref name="multiple"/>。

傳世四十公分以下之[[轉心瓶|轉心瓶]]多半收藏於[[國立故宮博物院|國立故宮博物院]]，其產造之初或因應乾隆皇帝的需求，故見成對傳世<ref name="multiple"></ref>。如國立故宮博物院即典藏有與此瓶一模一樣的作品，名為磁胎洋彩青地金花魚遊春水瓶<ref>[http://antiquities.npm.gov.tw/Utensils_Page.aspx?ItemId=314323 磁胎洋彩青地金花魚遊春水瓶]</ref>。[[北京故宮博物院|北京故宮博物院]]藏「乾隆青花粉彩鏤空魚藻紋轉心瓶」，與本瓶形式相同，僅外瓶運用[[青花瓷|青花瓷]]的裝飾技法。
