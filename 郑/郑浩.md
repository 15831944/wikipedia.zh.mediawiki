'''郑浩'''（{{bd|1959年|6月8日||}}），香港凤凰卫视节目主持人、时事评论员、新闻记者。<ref>{{cite web |title=郑浩简介 |url=http://phtv.ifeng.com/program/sskj/xxz/200712/1228_2528_346542.shtml |website=凤凰网 |accessdate=2018-11-13}}</ref><ref>{{cite web |title=郑浩 |url=http://phtv.ifeng.com/star/zhenghao/ |website=凤凰网 |accessdate=2018-11-13}}</ref><ref name="评论员">{{cite news |title=郑浩：举办北京香山论坛非常契合当前国际安全形势需要 |url=http://www.chbcnet.com/hgbd/content/2018-10/25/content_1345532.htm |accessdate=2018-11-13 |work=华广网 |date=2018-10-25}}</ref>

== 工作经历 ==
1990年－2004年，先后在香港《[[明报|明报]]》、《[[香港商报|香港商报]]》、[[亚洲电视台|亚洲电视台]]担任记者及编辑。

2004年，加入香港[[凤凰卫视|凤凰卫视]]，并参与制作专题片《凤凰号下西洋》。

2009年起，担任凤凰卫视时事评论员。<ref name="评论员"/>

2018年10月27日，担任《闽江学院建校60周年文艺晚会》主持人；10月28日，担任闽江学院凤凰数字媒体学院名誉院长。<ref>{{cite news |title=培养标准塑英才 产教融合创辉煌 |url=http://zgsc.china.com.cn/2018-10/31/content_40555631.html |accessdate=2018-11-13 |work=中国网 |date=2018-10-31}}</ref>
