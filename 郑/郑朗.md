'''郑朗'''（{{bd|？||857年|catIdx=Z}}），字'''有融'''，[[唐朝|唐朝]]官员，[[唐宣宗|唐宣宗]]年间拜相。

== 家世及早年生涯 ==
郑朗生年不详。他家世显赫，父[[郑珣瑜|郑珣瑜]]和兄[[郑覃|郑覃]]分别拜相。<ref name=BT173>[[:s:舊唐書/卷173|《旧唐书》卷一百七十三]]</ref>郑朗于[[唐穆宗|唐穆宗]][[长庆|长庆]]元年（821年）三月登进士甲科。有相士说：“你当贵，但不能凭科举进身。”当时，虽然郑珣瑜已过世，但郑覃已官居显要。但因[[段文昌|段文昌]]、[[李德裕|李德裕]]、[[元稹|元稹]]、[[李绅|李绅]]弹劾主考官[[杨汝士|杨汝士]]、[[钱徽|钱徽]]不公，郑朗和其他一些要员的亲属的考试成绩都陷入争议，其中包括[[裴度|裴度]]子[[裴譔|裴譔]]、[[李宗闵|李宗闵]]婿[[苏巢|苏巢]]、杨汝士弟[[杨殷士|杨殷士]]。结果穆宗下令重考，并贬谪了钱徽、李宗闵、杨汝士及郑朗等10名被认为在他们舞弊下通过考试的考生。<ref>[[:s:旧唐书/卷16|《旧唐书》卷一十六]]</ref><ref>[[:s:旧唐书/卷166|《旧唐书》卷一百六十六]]</ref><ref>[[:s:旧唐书/卷168|《旧唐书》卷一百六十八]]</ref><ref>[[:s:旧唐书/卷176|《旧唐书》卷一百七十六]]</ref><ref>[[:s:覆试郑朗等诏|:s:覆试郑朗等诏]]</ref>此事被认为为之后持续数十年的[[牛李党争|牛李党争]]埋下了伏笔。<ref>[[:s:資治通鑑/卷241|《资治通鉴》卷二百四十一]]：右补阙杨汝士与礼部侍郎钱徽掌贡举，西川节度使段文昌、翰林学士李绅各以书属所善进士于徽；及榜出，文昌、绅所属皆不预焉，及第者，郑朗，覃之弟；裴譔，度之子；苏巢，宗闵之婿；杨殷士，汝士之弟也。文昌言于上曰：“今岁礼部殊不公，所取进士皆子弟无艺，以关节得之。”上以问诸学士，德裕、稹、绅皆曰：“诚如文昌言。”上乃命[[中书舍人|中书舍人]][[王起|王起]]等覆试。夏，四月，丁丑，诏黜朗等十人，贬徽[[江州_(隋朝)|江州]]刺史，宗闵[[剑州_(先天)|剑州]]刺史，汝士[[开江|开江]]令。</ref>而相士在得知郑朗被贬谪后，即对他道贺。<ref name=NBT165>[[:s:新唐書/卷165|《新唐书》卷一百六十五]]</ref>

郑朗因此不能留在[[长安|长安]]开始仕途，只能在[[山南东道|山南东道]]（在今[[湖北|湖北]][[襄樊|襄樊]]）[[节度使|节度使]][[柳公绰|柳公绰]]手下当幕僚。<ref>柳公绰于823年至825年在山南东道任职，故郑朗为他效力必然在此期间。见[[s:舊唐書/卷165|《旧唐书》卷一百六十五]]。</ref><ref>[[:s:新唐書/卷163|《新唐书》卷一百六十三]]</ref>最后他还是被召回长安，任[[右拾遗|右拾遗]]。<ref name=NBT165/>

== 唐文宗年间 ==
穆宗子[[唐文宗|唐文宗]][[开成|开成]]（836年 - 840年）年间，郑朗被任为[[起居郎|起居郎]]。一次，文宗刚和宰相商议完节俭之事，看到郑朗正在履职作记录。会后，文宗要求看记录。郑朗反对，指出一旦皇帝可随意看起居郎的记录，起居郎会感到所写的东西受到限制，不能保证公正了。但文宗坚持，称看了[[起居录|起居录]]可以纠正自己的错误，郑朗便将起居录呈上。郑朗后任[[考功郎中|考功郎中]]。四年（839年），迁[[谏议大夫|谏议大夫]]。<ref name=BT173/>

== 唐武宗年间 ==
唐文宗弟[[唐武宗|唐武宗]][[会昌_(唐朝)|会昌]]二年（842年）十一月，尚在谏议大夫任上的郑朗和同僚[[高少逸|高少逸]]在阁中进谏武宗频繁游猎之事。武宗致谢并将他们的建议告知宰相们。<ref>[[:s:旧唐书/卷18上|《旧唐书》卷一十八上]]</ref><ref>[[:s:資治通鑑/卷246|《资治通鉴》卷二百四十六]]：乙卯，谏议大夫高少逸、郑朗于阁中谏曰：“陛下比来游猎稍频，出城太远，侵星夜归，万机旷废。”上改容谢之。少逸等出，上谓宰相曰：“本置谏官使之论事，朕欲时时闻之。”宰相皆贺。己未，以少逸为给事中，朗为左谏议大夫。</ref>郑朗以朝散大夫、守谏议大夫、兼弘文馆学士、[[上柱国|上柱国]]被授左谏议大夫。<ref>[[:s:授郑朗等左谏议大夫制|李德裕《授郑朗等左谏议大夫制》]]</ref>后历任[[给事中|给事中]]、[[华州|华州]]刺史、[[御史中丞|御史中丞]]、[[户部侍郎|户部侍郎]]，<ref>《全唐文》作开成年间事。</ref>判本司事。<ref name=BT173/>在华州刺史任上署[[杨假|杨假]]为从事。<ref name=BT177>[[:s:旧唐书/卷177|《旧唐书》卷一百七十七]]误将郑朗作郑覃。</ref>

== 唐宣宗年间 ==
武宗叔父唐宣宗登基后，郑朗任[[鄂岳观察使|鄂岳观察使]]，[[大中_(唐朝)|大中]]元年（847年）改浙西道观察使，以杨假为大理评事。<ref name=BT177/>加检校工部尚书。五年（851年）改检校[[户部尚书|户部尚书]]、[[汴州|汴州]]刺史、[[宣武军节度使|宣武军节度使]]、宋亳汴颍观察使，<ref>[[:s:授郑光河中节度使郑朗汴州节度使制|李顾行《授郑光河中节度使郑朗汴州节度使制》]]</ref>当年回朝任[[工部尚书|工部尚书]]，判度支。大中七年（853年）或八年（854年），又任[[御史大夫|御史大夫]]。<ref>此处据《唐刺史考全编》之<义武条><宣武条><江南东道·润州条>及[[严耕望|严耕望]]《唐仆尚丞郎表·卷二十一·辑考八上·工部尚书》；《两唐书》称其曾任[[义武军节度使|义武军节度使]]，误。</ref>十年（856年）正月，他被授[[同中书门下平章事|同中书门下平章事]]，为实质宰相。<ref>[[:s:除郑朗工部尚书同平章事制|:s:除郑朗工部尚书同平章事制]]</ref>十一月，内园使李敬寔没有避让郑朗的马，被郑朗弹劾，受到指责后自称作为供奉官按例不避，于是以私出不避宰相被剥夺官服，配役南牙。<ref>《旧唐书·宣宗纪》《册府元龟》作七年四月以御史大夫改中书侍郎、同平章事。</ref>郑朗后又被任为[[中书侍郎|中书侍郎]]、[[集贤殿大学士|集贤殿大学士]]。十一年（857年）二月，他又以通议大夫、守中书门下侍郎、兼礼部尚书、同平章事、集贤殿大学士、上柱国、赐紫金鱼袋被加监修国史。九月，他因病辞相，十月被罢为检校尚书右仆射，兼守[[太子少师|太子少师]]。<ref>[[:s:新唐书/卷8|《新唐书》卷八]]</ref><ref name=ZZTJ249>[[:s:資治通鑑/卷249|《资治通鉴》卷二百四十九]]：春，正月，丁巳，以御史大夫郑朗为工部尚书、同平章事。……中书侍郎、同平章事郑朗以疾辞位。壬申，以朗为太子太师。</ref><ref>[[:s:授郑朗太子少师制|:s:授郑朗太子少师制]]</ref><ref>《资治通鉴》作[[太子太师|太子太师]]。但当时并无[[太子|太子]]，这完全是个荣誉职位。</ref>当月卒，<ref>《旧唐书·宣宗纪》作十一月卒。</ref>赠[[司空|司空]]。<ref name=BT173/><ref>[[:s:赠郑朗司空制|:s:赠郑朗司空制]]</ref>
