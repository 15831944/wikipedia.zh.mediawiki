'''郑州四人撕毁毛泽东像事件'''是指2012年10月25日姬来松、蔺其磊、程帅帅，曹小东四人在[[河南省|河南省]][[郑州市|郑州市]][[紫荆广场|紫荆广场]][[毛泽东像|毛泽东像]]后面每人撕毁毛泽东像一张，并且把照片传在网上，随后四人身份被曝光，并且引发争论，[[左派|左派]]认为此事违反《[[中华人民共和国宪法|宪法]]》和《[[中华人民共和国刑法|刑法]]》，是[[煽动颠覆国家政权罪|煽动颠覆国家政权罪]]，须依法认罪；右派认为毛泽东发动[[反右运动|反右运动]]、[[大跃进|大跃进]]、[[大饥荒|大饥荒]]、[[文化大革命|文化大革命]]的遗毒至今没有清理干净，四人的行为是表达[[言论自由|言论自由]]和呼吁[[民主|民主]]，另外一种观点是四人撕毁毛泽东像并非[[去毛|去毛]]或拥毛，是为了出名和引起社会关注度，借机炒作自己。<ref name=" fish" >{{cite web |url=http://www.szhgh.com/html/07/n-14907.html |title= 河南四人公开撕毁毛主席画像！|accessdate= 2012年10月30日|publisher= 红歌会网|archiveurl= http://www.szhgh.com/html/07/n-14907.html|archivedate= 2012年11月13日|work= |language =zh-cn}}</ref>

== 事件 ==
2012年10月25日，在[[河南省|河南省]][[郑州市|郑州市]][[紫荆广场|紫荆广场]]毛泽东像后面，姬来松、蔺其磊、程帅帅，曹小东四人每人手拿一张毛泽东像撕毁，随后将撕毁毛泽东像的照片传到网上，旋即引发网民围观和讨论，四人身份也被迅速曝光。<ref name=" fish" /> 郑州市公安局旋即逮捕了其中三人。<ref>{{cite web |url=http://www.szhgh.com/html/12/n-15112.html |title=快讯：网传撕毁毛主席像肇事者已有三人被拘留 另一人正在抓捕 |accessdate=2012年11月3日 |publisher=红歌会网 |archiveurl=https://web.archive.org/web/20121105200226/http://www.szhgh.com/html/12/n-15112.html |archivedate=2012年11月5日 |work= |language=zh-cn |deadurl=yes }}</ref> 

11月1日，另一人曹小东被传唤随后遭到监禁。<ref name="mans " /><ref name="lio " /><ref>{{cite web |url=http://www.boxun.com/news/gb/china/2012/11/201211020022.shtml |title=艾滋维权人士曹小东被警察带往派出所|accessdate= 2012年11月2日|publisher= 博讯网|archiveurl=http://www.boxun.com/news/gb/china/2012/11/201211020022.shtml |archivedate=2012年11月13日 |work= |language =zh-cn}}</ref>

11月11日，在[[美国|美国]][[洛杉矶|洛杉矶]]的中华人民共和国驻洛杉矶总领事馆前面，50余名[[汉族|汉族]]、[[藏族|藏族]][[中国民主党|中国民主党]]党员聚集在一起，每人手拿一张毛泽东像撕毁，高呼“中国人权不能等待”，随后把撕碎的毛泽东像碎片放入信奉，交给中华人民共和国驻洛杉矶总领事馆，宣布正式成立“中国人权保护组织”。<ref name="mans " >{{cite web |url=http://www.rfa.org/mandarin/yataibaodao/la-11122012100044.html |title=洛杉矶汉藏人士撕毛像 中领馆前呼喊人权自由 |accessdate=2012-11-12 |publisher=自由亚洲电台 |archiveurl=http://www.rfa.org/mandarin/yataibaodao/la-11122012100044.html |archivedate=2012年11月13日 |work= |language =zh-cn}}</ref><ref name="lio " >{{cite web |url=http://news.rti.org.tw/index_newsContent.aspx?nid=388944&id=1&id2=2 |title=聲援曹小東 人權組織撕毛澤東像 |accessdate=2012-11-12  |publisher=  中央社 |archiveurl=http://news.rti.org.tw/index_newsContent.aspx?nid=388944&id=1&id2=2 |archivedate= 2012年11月13日|work= |language =zh-tw}}</ref>

*蔺其磊，男性。<ref name=" fish" />

*曹小东，男性。<ref name=" fish" />

== 社会舆论 ==
===正面评价===
[[中央民族大学|中央民族大学]]教授[[赵士林|赵士林]]：“撕毛像是正义之举，但也是表面之举。为了唤醒被误导的民众及未经历毛时代的年轻人，关键是公布真相。当局应为民族负责，公布毛时代真相，诸如[[反右|反右]]、[[大饥荒|大饥荒]]饿死人、[[文革|文革]]等。真相会将毛粪那一套立刻击得粉碎，民众立刻就会认识清楚孰是孰非。”<ref name=" tiger" >{{cite web |url=http://www.voachinese.com/content/young-men-ripping-up-maos-portrait-20121104/1539258.html |title= 青年撕毛像公安拘查敏感时刻评毛网战|accessdate=2012年11月4日 |publisher=美国之音 |archiveurl=http://www.voachinese.com/content/young-men-ripping-up-maos-portrait-20121104/1539258.html |archivedate=2012年11月13日 |work= |language =zh-cn}}</ref>

政治学者[[吴稼祥|吴稼祥]]、影视编剧[[谷凯|谷凯]]、作家[[土家野夫|土家野夫]]：“《[[参考消息|参考消息]]》登了一篇文章，标题：《国人愤怒：‘郑州四人公开撕毁毛泽东像’》，抱歉，我是[[中国人|中国人]]，我没有感到愤怒，别随便代表我。”<ref name=" tiger" />

政治学者[[张鸣|张鸣]]：“撕毛像的人被警方传讯，毛粉一片欢腾。是的，如果在文革时，这样的人是要被判死刑的。文革又回来了，你们高兴了。”<ref name=" tiger" />

军事家[[赵楚_(军事家)|赵楚]]：“新文革份子疯狂出动，我草草看了看，印象是，这些毛的拥护者绝对是400%的毛黑。除了不堪入目的辱骂，就是很空洞的暴力威胁，他们所展示的毛追随者的嘴脸基本流氓、无赖和心理变态者。这些人就是毛的好学生？太滑稽了。”<ref name=" tiger" />

经济学者[[郭朝晖|郭朝晖]]：“1976年，[[山东|山东]][[历城县|历城县]]教育局长用带主席像的报纸包鱼骨头、北园中学的老师用带主席像的报纸擦屁股。两人皆被打成[[现行反革命|现行反革命]]。近闻郑州警方传讯撕毁毛画像青年人，是不是文革真的又来了？”<ref name=" tiger" />

=== 负面评价 ===
作家[[司马平邦|司马平邦]]：“他们有表达看法的权利，虽方式极端，但应依法追诉。”<ref name=" tiger" />

鹤龄：“（他们）应该归咎在《[[中华人民共和国刑法|刑法]]》第一百零五条‘以造谣、诽谤或者其他方式[[煽动颠覆国家政权罪|煽动颠覆国家政权罪]]、推翻社会主义制度的’的范围之内，这种行为如果已经构成犯罪，依法可以‘处五年以下[[有期徒刑|有期徒刑]]、拘役、管制或者[[剥夺政治权利|剥夺政治权利]]；首要分子或者罪行重大的，处五年以上有期徒刑’。”<ref>{{cite web |url=http://www.szhgh.com/html/08/n-14908.html |title=鹤龄：必须依法查处姬来松等撕毁毛主席像的违法行为 |accessdate=2012年10月31日 |publisher=红歌会网 |archiveurl=https://web.archive.org/web/20121031234213/http://www.szhgh.com/html/08/n-14908.html |archivedate=2012年10月31日 |work= |language=zh-cn |deadurl=yes }}</ref>
