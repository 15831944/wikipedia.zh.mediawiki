== 历史沿革 ==

学校于1983年建校，建制为[[完全中學|完全中学]]，地址在陇海西路65号。上世纪九十年代末，郑外把办学目标确定为“文理兼长，外语突出，全面发展”。2001年作为郑州市重点工程，学校高中部外迁至高新区枫杨街新址，建设成为[[寄宿学校|寄宿制]]高中，陇海路校区成为初中部。2004年，在学生平均分班的基础上，根据学生兴趣，开始了[[理科实验班|理科实验班]]的尝试。2009年，提出“中西文化融合、智慧人格并重，本土情怀与国际视野兼备的高素质预备人才”的培养目标。<ref name=":0" />

== 办学条件 ==

=== 硬件设施 ===
郑外拥有省级示范性实验室，标准400米塑胶跑道的体育场，现代化的体育馆，郑州外国语学校在河南省属于设施齐全、设备一流的现代化窗口学校。学校位于郑州市高新技术开发区，占地164.7亩，建筑面积60000平方米。郑州外国语中学是寄宿制学校，宿舍每间6个床位，内有空调。<ref>{{Cite web|url=http://edu.dahe.cn/2013/08-20/102367556.html|title=郑州外国语中学总校-大河教育|accessdate=2018-08-18|work=edu.dahe.cn}}</ref>

=== 师资力量 ===
学校的特级教师，国家级骨干教师，省专家，省学科带头人，省骨干教师，省优秀教师，市拔尖人才，国家、省优质课一等奖教师，数、理、化、生[[全国中学生学科奥林匹克竞赛|国家级奥赛]]教练员，高级教师，省市教研室学科中心组成员占高中任课教师总数的70%，河南省[[海外高层次人才引进计划|“百千万工程”]]中首批确定的72名省级教育教学专家，我校有7名；郑州市“名师”我校多达26人、郑州市10名首届杰出教师中我校有2人；省教研室学科中心组成员中我校有11人。上述各项人数之多、覆盖学科之广均居全省各学校之首。<ref>{{Cite web|url=http://www.zzfls.com.cn/Teacher/Faculty.aspx|title=郑州外国语学校|accessdate=2018-08-18|author=|date=|work=www.zzfls.com.cn|publisher=}}</ref>

=== 办学规模 ===
学校现有48个班级，在校学生2800余人。<ref name=":0" />郑州外国语学校国际部设有中美[[進階先修課程|AP国际课程]]班、中英[[普通教育高级程度证书|A-Level]]国际课程班、中澳[[維多利亞州教育證書|VEC]]国际课程班。

== 办学成果 ==
郑州外国语学校的中招录取分数线连续九年为郑州市最高。<ref name=":0" />2014年至2018年的统录分数线分别为613分，635分，652分，655分，652分。<ref>{{cite web |title=一所“前所未有”的高中郑州外国语学校，河南人的骄傲 |url=http://js.qinxue100.com/news/201712/45933.html}}</ref>

2018年，郑州外国语学校分校郑州外国语新枫杨学校学生社团PPE社团（政哲经社）成立。现已成为郑州外国语教育集团社团活动频率最高的社团之一。<ref>{{cite web|url=http://news.dahe.cn/2015/07-22/105318671.html | language= | title=郑州中招第一批录取的19所高中录取分数线出炉 |publisher=大河网 |date=2015年7月22日 |accessdate= }}</ref>

位列2015中国高中百强榜第17位<ref>{{cite web|url=http://news.dahe.cn/2015/09-09/105602510.html | language= | title=2015中国高中百强榜出炉 |publisher=大河报微信 |date=2015年9月9日 |accessdate= }}</ref>。 郑州外国语学校在2016年中国内地高中美国留学排行榜上位居第十七。<ref>人民网 "http://gz.people.com.cn/n2/2016/0224/c358162-27802234.html" 2016-02-24 </ref> <ref>联校网排名 "{{cite web |url=http://www.collegenode.com/list.php |title=存档副本 |accessdate=2016-04-05 |0=2016-10-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20161014013045/http://www.collegenode.com/list.php |archivedate=2016-10-14 }}" </ref>2018年，郑外国际部毕业学生中，[[哈佛大学|哈佛]]、[[牛津大学|牛津]]、[[剑桥大学|剑桥]]均有录取，其中宗泽泓同学为河南首位被哈佛大学录取的高中毕业生。<ref>{{Cite web|url=http://www.zzfls.com.cn/Internation/Internation.aspx|title=郑州外国语学校|accessdate=2018-08-18|author=|date=|work=www.zzfls.com.cn|publisher=}}</ref>
