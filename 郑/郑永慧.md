'''郑永慧'''{{BD|1918年||2012年|9月9日|catIdx= Z}}原名'''郑永泰'''，籍贯[[广东|广东]][[中山市|香山]]，毕业于[[上海|上海]][[震旦大学|震旦大学]]法学院，精通[[法文|法文]]，著名翻译家。

== 名字由来 ==
郑永慧原名郑永泰，改名原因是由于妻子邓慧群陪伴他从事翻译工作通宵达旦，为感激妻子，取妻子的名字中的“慧”字，组成自己的新名字“郑永慧”。<ref>{{cite web|url=http://epaper.dfdaily.com/dfzb/html/2012-09/12/content_673708.htm|title=郑永慧|accessdate=2012年9月12日|publisher=东方早报|work=|language=zh-cn}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 生平 ==
1918年，郑永慧出生在[[法屬印度支那|法属越南]][[海防市|海防市]]，10多岁时，郑永慧到[[香港殖民地時期|英属香港]]求学，<ref>{{cite web|url=http://www.dahuawang.com/stwb/html/2012-09/18/content_359149.htm|title=郑永慧一位真正“信”的大师|accessdate=2012年9月18日|publisher=汕头特区报|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20160308173527/http://www.dahuawang.com/stwb/html/2012-09/18/content_359149.htm|archivedate=2016年3月8日}}</ref>毕业于[[上海|上海]][[震旦大学|震旦大学]]法学院，在校期间，看到[[列夫·托尔斯泰|列夫·托尔斯泰]]的长篇小说《[[安娜·卡列尼娜|安娜·卡列尼娜]]》的俄文版和法文版，却没有中文版，决心从事翻译工作，后曾指出法文翻译家[[傅雷|傅雷]]的翻译错误。<ref name="bull " >{{cite web |url=http://culture.ifeng.com/gundong/detail_2012_09/12/17538254_0.shtml?_from_ralated |title=【逝者】郑永慧 |accessdate=2012年9月12日 |publisher=凤凰网 |work= |language=zh-cn}}</ref>1952年，郑永慧的处女作译自[[巴尔扎克|巴尔扎克]]的短篇小说《钱袋》、《猫球商店》和《苏城舞会》出版。<ref name="bull " />郑永慧曾在震旦女子文理学院、国际关系学院教书。<ref>{{cite web |url=http://culture.ifeng.com/8/detail_2012_09/12/17546137_0.shtml?_from_ralated |title=94岁翻译家郑永慧去世 曾指出傅雷错译 |accessdate=2012年9月12日 |publisher=凤凰网 |work= |language=zh-cn}}</ref>1966年，[[毛泽东|毛泽东]]发起[[文化大革命|文化大革命]]，郑永慧被迫下乡到[[河北省|河北省]]接受贫下中农再教育，期间他还悄悄地关门读法文名著。<ref name="bull " />2012年9月9日早晨，郑永慧在[[北京市|北京市]]去世。<ref name="bull " /><ref>{{cite web |url=http://www.gmw.cn/xueshu/2012-12/24/content_6120139.htm |title=2012永远的怀念——华人学者大师盘点  |accessdate=2012-12-24  |publisher=光明网 |work= |language=zh-cn}}</ref>

== 译著 ==
郑永慧专门从事法文文学翻译工作，一生有翻译作品40多部，600多万字，曾翻译[[巴尔扎克|巴尔扎克]]、[[雨果|雨果]]、[[福楼拜|福楼拜]]、[[梅里美|梅里美]]、[[大仲马|大仲马]]、[[左拉|左拉]]、[[纪德|纪德]]、[[乔治桑|乔治桑]]、[[莫泊桑|莫泊桑]]、[[萨特|萨特]]、[[罗伯·葛里耶|罗伯·葛里耶]]等作家的作品，他自己最满意的翻译作品是雨果的《九三年》。<ref>{{cite web |url=http://culture.people.com.cn/n/2012/0912/c172318-18989968.html |title=郑永慧数十年翻译法文经典 最满意《九三年》 |accessdate=2012年9月12日 |publisher=人民网 |work= |language=zh-cn}}</ref>
