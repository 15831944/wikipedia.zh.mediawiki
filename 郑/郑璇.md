'''郑璇'''（{{bd|1964年|11月||}}），[[福建|福建]][[闽侯|闽侯]]人。[[中国人民解放军少将|中国人民解放军少将]]<ref name=caixin/><ref name=xhw/>。

==生平==
1976年12月，12岁的郑璇参加[[中国人民解放军|中国人民解放军]]（？？？）。后于[[中国人民解放军国防大学|国防大学]]中青年领导干部培训班全日制硕士研究生毕业。曾任[[成都军区|成都军区]][[中国人民解放军第14集团军|陆军第十四集团军]]高炮旅政委。2008年，任陆军第十四集团军某师政委。2012年，任[[中国人民解放军第13集团军|陆军第十三集团军]]副政委，并且晋升少将军衔。2013年，接替升任[[西藏军区|西藏军区]]政委的[[刁国新|刁国新]]担任陆军第十三集团军政委，升为正军级<ref name=caixin/><ref name=xhw/>。2017年1月，任[[中国人民解放军中部战区|中国人民解放军中部战区]]副政委兼[[中国人民解放军中部战区政治工作部|中部战区政治工作部]]主任<ref name=caixin>{{Cite web |url =http://china.caixin.com/2017-02-03/101049721.html  |title =13集团军政委郑璇转岗中部战区  |author =  |date = 2017-02-03 |publisher = 财新网 |language =  |accessdate =  |quote =  }}</ref><ref name=xhw>{{Cite web |url =http://news.xinhuanet.com/mil/2017-02/03/c_1120404899.htm  |title =  13集团军政委郑璇跨战区升任中部战区副政委兼政治工作部主任|author =  |date = 2017-02-03  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

郑璇曾多次执行抗震救灾等任务。2008年5月20日，[[四川省|四川省]][[阿坝藏族羌族自治州|阿坝藏族羌族自治州]][[汶川县|汶川县]]发生[[汶川大地震|汶川大地震]]，成都军区某高炮旅政委郑璇率部开到[[江油市|江油市]]北部山区抗震救灾，并向来江油市视察灾情的[[国务委员|国务委员]]兼[[中华人民共和国国防部|中华人民共和国国防部]]部长[[梁光烈|梁光烈]]汇报情况。2011年3月10日，[[云南省|云南省]][[德宏傣族景颇族自治州|德宏傣族景颇族自治州]][[盈江县|盈江县]]发生[[2011年盈江地震|盈江地震]]，成都军区驻云南某师政委郑璇率部开到平原镇拉勐村抗震救灾。2013年4月20日，[[四川省|四川省]][[雅安市|雅安市]][[芦山县|芦山县]]发生[[2013年芦山地震|芦山地震]]，第13集团军政委郑璇曾率部开赴灾区抗震救灾，郑璇所在部队某团十多位战士在赴灾区途中，所乘车辆翻下将近30米高陡坡，倒扣在[[荥经河|荥经河]]里，多位战士受伤，1位战士当场牺牲<ref name=caixin/><ref name=xhw/>。
