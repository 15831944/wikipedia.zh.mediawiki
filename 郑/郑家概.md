==生平==
1979年11月入伍，到[[河南|河南]][[许昌|许昌]][[襄城县|襄城县]][[中国人民解放军陆军第二十集团军|陆军第二十集团军]]五十八师一七三团二连服役。历任班长、排长、连长、作战参谋、作训股长、科长、一七三团团长、五十八旅参谋长、步兵第六十旅旅长、第五十八旅旅长、[[中国人民解放军陆军第二十六集团军|陆军第二十六集团军]]副参谋长、参谋长、副军长。2010年5月，任陆军第二十集团军副军长兼参谋长<ref name=wj/>。

2015年4月，任[[济南军区|济南军区]]副参谋长<ref name=wj/>。2016年，任[[北部战区陆军|北部战区陆军]]参谋长<ref name=wj/><ref name=zhengling>{{Cite web |url =http://www.thepaper.cn/www/v3/jsp/newsDetail_forward_1600530  |title =郑家概、凌焕新已调任北部战区陆军领导  |author =  |date = 2017-01-14 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。2017年9月，任[[中国人民武装警察部队参谋部|中国人民武装警察部队参谋长]]<ref name=wj/>。

2011年7月，晋升少将军衔。2017年9月后改为武警少将<ref name=wj/>。

==家庭==
据《常德晚报》报道，2011年，时任某集团军副军长的郑家概以[[郑洞国|郑洞国]]家族后代的名义为母校洞国学校捐款<ref>{{Cite web |url = http://news.ifeng.com/a/20170924/52143680_0.shtml |title = 26个月来，武警部队已三换参谋长 |author =  |date = 2017-09-24 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
