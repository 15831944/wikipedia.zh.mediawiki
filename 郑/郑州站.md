'''郑州站'''是[[中国铁路|中国铁路]]最重要的枢纽之一，位于[[河南省|河南省]][[郑州市|郑州市]][[二七区|二七区]]，[[京广铁路|京广铁路]]及[[陇海铁路|陇海铁路]]两大铁路主干线、[[郑西高速铁路|郑西高速铁路]]、[[郑洛城际铁路|郑洛城际铁路]]、[[郑焦城际铁路|郑焦城际铁路]]、[[郑新城际铁路|郑新城际铁路]]等多条线路在此交汇。

==历史==
[[File:Old_Zhengzhou_Railway_Station_1911.jpg|thumb]]
[[File:Old_Zhengzhou_Railway_Station_1954.jpg|thumb]]
郑州火车站始建于1904年，时名“郑县站”，位于现郑州站一站台偏南处。站内设有两间平房、4条股道，是[[京汉铁路|京汉铁路]]中段的业务站<ref name="est">{{cite web|title=郑州火车站东广场简介|url=http://www.sszzw.com/ming-c/cf/huoche-jian.htm|publisher=搜索郑州网}}</ref>。1911年，郑县站计两空三梯的铁质天桥投入使用。1914年，陇海铁路[[开封|开封]]—[[洛阳|洛阳]]段（[[汴洛铁路|汴洛铁路]]）的开业奠定了郑州火车站為全国交通枢纽的基础。1928年，郑州车站修造票房和候车风雨棚二个。1932年9月，郑州车站货站开工<ref name="yearlist">{{cite web|title=郑州火车站百年大事记|url=http://www.sszzw.com/ming-c/cf/huoche-dashi.htm|publisher=搜索郑州网}}</ref>。 
 
1944年4月，[[陇海铁路|陇海铁路]]郑州至开封的铁路线通车。1948年10月22日，中共占领郑州，[[中国人民革命委员|中国人民革命委员]]会派军事代表对郑州车站实行军事管制。1949年3月11日，[[中国铁路郑州局集团有限公司|郑州铁路管理局]]成立；5月，郑州铁路管理局将平汉路、陇海路郑州南北车站合并，成立郑州总站。同年11月10日，郑州站将现有8条股道增修为22条股道<ref name="yearlist"/>。 

1953年，郑州站进行了为期3年的改造暨扩建工程，分离出货运站[[圃田西站|郑州东站]]（现圃田西站），使得现時的郑州站成为了一座以运输旅客为主的大型客运站。1955年，经铁道部批准，郑州车站为特等站。次年3月8日，车站新站房工程开工。站房于1958年竣工，设京广候车区和陇海候车区，同时地道贯通4条股道，3个站台。1959年8月，于1957年开工的车站新天桥投入使用<ref name="yearlist"/>。

1962年，经铁道部批准，郑州车站新老两场分为两个车站：老场为现在的客运郑州火车站，新场则改为编组站[[郑州北站|郑州北站]]。1971年10月，郑州车站一站台-{}-风雨棚、天桥、排水改造工程动工；1972年6月，建二、三、四站台-{}-风雨棚，总覆盖面积为2160平方米<ref name="yearlist"/>。 

1988年至1999年，郑州站经历了历史上最大的一次改暨扩建工程。改造工程分为三期：一期工程为站场改造，于1992年启用；二期工程为南北配楼，于1995年建成并投入使用。1999年12月28日，具有标志性的郑州站新主站楼正式投入使用，标志着历时12年的改暨扩建工程的结束<ref name="east">{{cite news|title=郑州火车站扩建成为中国第二大客运站|url=http://news.sina.com.cn/china/1999-12-30/47362.html|publisher=新浪|date=1999-12-30}}</ref><ref name="general">{{cite web|title=百年火车站领跑郑州换新颜(组图)|url=http://news.sina.com.cn/c/p/2008-11-10/180116624857.shtml|publisher=新浪|date=2008-11-10}}</ref>。

2006年12月，郑州站西站房开建<ref name="general"/>。[[2010年1月|2010年1月]]，西站房和无柱风雨棚建成并投入使用。

==车站布局==
===站房===

郑州站总建筑面积14.4万平方米，由东站房、西站房和高架候车室组成<ref>{{cite news|title=探访郑开城际铁路:每个车站都是微缩版“火车站”|url=http://news.china.com.cn/live/2014-10/28/content_29512443.htm|publisher=中国网|date=2014-10-28}}</ref>。
;东站房
郑州站东站房又称作主站楼，于1999年12月29日作为郑州火车站改造工程的三期投入使用。主站楼高高63.3米，建筑面积4万平方米，地下局部一层，地上九层。售票大厅位于一层南侧，面积2500平方米，有68个售票窗口。进站口位于一层中央，设有4部扶梯、2部直梯及2座步行楼梯通往位于二层各候车厅及高架候车厅。东站房设有三个普通候车厅及贵宾、软席、母子、军人、团体等专用候车厅<ref name="east"/>。行包房面积1.49万平方米，日发送行包8000多件<ref name="est"/>。
;西站房
郑州站西站房于2010年投入使用，长268米，宽56米，总面积11774平方米。站房外立面通过现代的钢结构材料抽象化表现[[司母戊鼎|司母戊鼎]]的意向<ref>{{cite news|title=郑州火车站西55.4公顷将要变脸　郑州火车站西出口相关工程昨日论证(组图)|url=http://news.sohu.com/20070324/n248938961.shtml|publisher=搜狐|date=2007-03-24}}</ref>。站房整体位于郑州站站场西侧，与既有的高架候车室和两条出站地道连接<ref>{{cite news|title=郑州火车站出站口规划论证 “交通优先”(组图)|url=http://news.sohu.com/20070324/n248942302.shtml|publisher=搜狐|date=2007-03-24}}</ref>。西站房北侧一层为售票大厅，设有30个人工售票窗口和10台自动售票机<ref>{{cite news|title=郑州火车站西站房几乎没人用 乘客不知已经启用|url=http://www.dahe.cn/xwzx/zt/ss/2010cy/tl/t20100201_1741022.htm|publisher=大河网|date=2010-02-01}}</ref>，其中1号窗口为残疾人专口；6号窗口为退票窗口；7号窗口为车票签证窗口；9号窗口负责出售站台票；15号窗口为值班主任窗口。一层南侧为VIP候车厅。进站口位于一层中部，配备4个进站通道，进入站房后通过楼梯、自动扶梯或残疾人专用折叠式的无障碍升降平台进入二层高架候车区<ref>{{cite news|title=郑州火车站西出口人流少 想方便快捷不妨"走西口"|url=http://news.ifeng.com/gundong/detail_2012_06/19/15410093_0.shtml|publisher=凤凰网|date=2012-06-19}}</ref>。西广场二层新建高架候车室6186平方米，内设各式商铺，并与既有的6个候车室相连通<ref>{{cite news|title=中铁七局施工的郑州火车站西出口新建高架候车室竣工|url=http://www.sasac.gov.cn/n2588025/n2588124/c4038063/content.html|publisher=国务院国有资产监督管理委员会|date=2008-12-15}}</ref>。

郑州站东西站房间设有高架候车室横跨站场，高架候车室南北长190米，东西宽108.5米<ref name="est"/>，面积2.4万平方米，设有6个候车室，可同时容纳近2万人候车。

郑州站设有20条股道，其中供客运使用13条，4条货运线和数条外绕线<ref>{{cite news|title=郑焦铁路列车在郑州站遇“进门难”：需借道京广线但线路饱和|url=http://www.thepaper.cn/newsDetail_forward_1615892|publisher=澎湃新闻|date=2017-02-10}}</ref>。车站配备7座客运站台<ref>{{cite news|title=郑州火车站升级道岔作业33天 “术后”5年内这件事儿都很省心|url=http://news.dahe.cn/2017/10-12/108551251.html|date=2017-10-12|publisher=大河网}}</ref>。其中2-6站台为岛式站台，系高站台。1站台为岛式站台，系低站台，并设有中转换乘通道通往高架候车厅：乘坐列车到达郑州站的旅客可根据表示指引通过地道进入位于1站台的换乘点进行换乘<ref>{{cite news|title=郑州火车站设置中转换乘通道|url=http://newpaper.dahe.cn/hnrb/html/2015-08/21/content_1302783.htm?div=-1|publisher=大河网|date=2015-08-21}}</ref>。

车站1-7站台上方覆盖有无站台柱雨棚。雨棚建筑面积53036.9平方米，由钻孔桩基础、承台、钢管混凝土双肢组合柱、倒三角形钢管桁架结构组成。雨棚为白色，东西最大跨度145.55米，建筑最高处27.8米，钢结构总重4000多吨<ref>{{cite news|title=郑州火车站风雨棚投入使用 旅客告别露天候车|url=http://news.sina.com.cn/c/2010-01-22/102616975719s.shtml|publisher=新浪网|date=2010-01-22}}</ref>。

车站西北侧设有五里堡客整所，从郑州站始发的客车大多需要在五里堡整备，待临近发车时再由调机牵入站台<ref>{{cite news|title=郑州火车调车组的春运：一晚上帮30多列车“穿针引线”|url=http://new.qq.com/omn/20180206/20180206A04PAC.html|publisher=腾讯网|date=2018-02-06}}</ref>。

郑州火车站东、西广场均设有公交港湾，其中东广场设有北港湾、南港湾、一马路公交港湾，西广场南北各设一处公交港湾，共有超过50条线路途径或从这些公交港湾始发终到<ref>{{cite web |url=http://zhengzhou.8684.cn/z_f8027c0e |title=火车站西广场公交车站_郑州火车站西广场公交站点_经过火车站西广场的公交 - 郑州公交查询 |last= |first= |date=2017年12月10日 |website= |publisher=8684 |access-date= |quote=}}</ref><ref>{{cite web |url=http://zhengzhou.8684.cn/z_da859a1a |title=火车站(北港湾)公交车站_火车站(北港湾)公交站_火车站(北港湾)公交站点_经过火车站(北港湾)的公交 |last= |first= |date=2017年12月11日 |website= |publisher=8684 |access-date= |quote=}}</ref><ref>{{cite web |url=http://zhengzhou.8684.cn/z_08fa577c |title=火车站(南港湾)公交车站_郑州火车站(南港湾)公交站点_经过火车站(南港湾)的公交 - 郑州公交查询 |last= |first= |date=2017年12月10日 |website= |publisher=8684 |access-date= |quote=}}</ref><ref>{{cite web |url=http://zhengzhou.8684.cn/z_e3c2b774 |title=火车站(一马路)公交车站_火车站(一马路)公交站_火车站(一马路)公交站点_经过火车站(一马路)的公交 |last= |first= |date=2017年12月11日 |website= |publisher=8684 |access-date= |quote=}}</ref>。此外，郑州火车站东北亦设有公交站一处，游16路途径该站。<ref>{{cite web |url=http://zhengzhou.8684.cn/z_58f6a1a6 |title=火车站(二马路)公交车站_郑州火车站(二马路)公交站点_经过火车站(二马路)的公交 - 郑州公交查询 |last= |first= |date=2017年12月10日 |website= |publisher=8684 |access-date= |quote=}}</ref>

==火车站商圈==
郑州火车站商圈一直以来都是河南省乃至中国中部地区服装和小商品交易的中心。在以火车站广场为中心的数公里内，分布着大大小小十多个批发市场。为郑州市主要商业街之一的[[大同路_(郑州)|大同路]]，将这些批发市场分为两个区域：以银基商贸城为代表的服装批发区域和以中州商场、苑陵商场、河南金林市场等为代表的电子、工艺品批发区域。
==争议事件==
1998年郑州市民葛锐在郑州站高架候车室如厕时，被管理人员索要0.3元[[人民币|人民币]]的如厕费。葛锐认为火车票的价格应包含厕所使用的费用，因此将[[中国铁路郑州局集团有限公司|郑州铁路分局]]提起诉讼，要求其向他赔礼道歉，并退回0.3元如厕费。经过一审败诉后，2001年3月13日，郑州市中级人民法院在二审中认为郑州站高架候车室在郑州火车站范围之内，车站在候车室内的厕所收费违反相关行政法规。法院最后判决葛锐胜诉，车站退还收取的0.3元如厕费。但在之后相当一段时间内，郑州火车站继续向使用高架候车室厕所的旅客收取如厕费<ref>{{cite news|title=如厕官司打了两年半“铁老大”退还0．3元|url=http://www.people.com.cn/GB/shenghuo/76/123/20010316/418816.html|publisher=人民网|date=2001-03-16|author=王方杰}}</ref>。
