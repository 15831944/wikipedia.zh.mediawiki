'''郑州市白沙园区'''，原名为'''中牟产业园区'''，是[[郑汴新区|郑汴新区]]的重要组成部分，为郑州都市区的重要拓展空间、郑汴新区的建设主区域和[[中原经济区|中原经济区]]核心增长极的重要组成部分<ref name="简介"/>。白沙园区管委会位于[[郑开大道|郑开大道]]19号。<ref name="简介">{{cite web |url=http://www.zmcyyq.com/html/112/2011/20110122002539340967374/20110122002539340967374_1.html |language= |title=园区简介 |publisher=郑州市白沙园区 |date= |accessdate=2013年1月 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 概况 ==
园区总规划面积为300平方公里，西起[[京港澳高速|京港澳高速]]，东至[[开封市|开封市]]边界，北起[[连霍高速|连霍高速]]，南至[[310国道|310国道]]—[[贾鲁河|贾鲁河]]一线<ref name="致辞">{{cite web |url=http://www.zmcyyq.com/html/140/2011/20111201201841062844624/20111201201841062844624_1.html |language= |title=主任致辞 |publisher=郑州市白沙园区 |date= |accessdate=2013年1月 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

截至2011年12月，园区共开建58个基础设施项目，完成投资30亿元，建成51公里的道路，4座变电站，供水厂、污水处理厂和天然气门站各1座。累计引进82个项目，其中22个高等院校。开工建设42个项目，成功申报22个省重点项目；现代教育园区开工14所院校，入住师生2.2万人。<ref name="致辞"/>

== 规划布局 ==
园区重点布局白沙组团、绿博组团和官渡工业园区：<ref name="简介"/>
*白沙组团：紧邻[[郑东新区|郑东新区]][[CBD|CBD]]的区位优势，侧重职业教育、高新技术产业、现代服务产业；
*绿博组团：抓住[[郑州绿博园|郑州绿博园]]建设机遇，侧重文化创意、总部经济、商贸金融产业，兼备居住功能；
*官渡工业园区：主要承接沿海产业转移，发展先进[[电子|电子]]制造业。
