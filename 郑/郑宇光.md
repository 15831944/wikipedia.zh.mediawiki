'''郑宇光'''（{{lang-en|'''Jeffrey Chueng'''}}，{{bd||11月23日|||catIdx= Z}}）辽宁人，毕业于加拿大女皇大学，2006年世界男模特大赛中国赛冠军，2007年韩国江原道世界男模大赛冠军。<ref name="fish" >{{cite web |url=http://www.iweihai.cn/iweihainews/snews/2007/02/29648.asp |title=中国男模特郑宇光首次夺得世界男模大赛冠军(图) |accessdate=2007 |publisher= |work= |language=zh-hans}}</ref><ref name="bee" >{{cite web |url=http://women.sohu.com/20070607/n250449800.shtml |title=男模郑宇光获世界大赛冠军 |accessdate=2007 |publisher=搜狐网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/s/m/2007-04-28/22351538163.html |title=刘园园“遭遇”求爱 国际男模郑宇光现场索吻 |accessdate=2007 |publisher=新浪网 |work= |language=zh-hans}}</ref>

== 生平 ==
郑宇光出生在[[辽宁省|辽宁省]][[大连市|大连市]]，他母亲是一名英文翻译，曾被母亲送到[[皇后大學_(加拿大)|加拿大女皇大学]]学习，主攻计算机，辅修经济学。
