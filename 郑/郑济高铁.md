'''郑济高铁'''是[[河南|河南]][[郑州市|郑州市]]至[[山东|山东]][[济南市|济南市]]的一条正在建设的[[高速铁路|高速铁路]]。郑济高铁大致呈东北西南走向，线路全长370公里左右，自[[郑州东站|郑州东站]]引出，接入[[济南西站|济南西站]]，并设有通往[[济南站|济南站]]的联络线。郑济高铁设计时速为350公里/小时，总投资约700亿元。目前，郑济高铁河南段正在建设，山东段即将开工建设。郑济高铁意义重大，它将济青高铁，京沪高铁，京广高铁，京九高铁、郑渝高铁以及郑西高铁连接起来，大大方便沿线人民出行。

== 高铁概况 ==
郑济高铁河南段全长237.4公里，设计时速350公里,线路自[[郑州东站|郑州东站]]引出，经[[郑州|郑州]]、[[新乡市|新乡]]、[[鹤壁|鹤壁]]、[[滑县|滑县]]、[[安阳|安阳]]至[[濮阳|濮阳]]，到达豫鲁省界，然后进入山东省聊城市莘县境内。郑济高铁河南段新建线路196.871公里，总投资380.02亿元。2016年11月1日，郑济高铁河南段正式开工建设，工期4年，预计2020年河南段竣工。<br>
山东段始于聊城市莘县(豫鲁省界)，至济南西枢纽站,新建正线长度169.694公里，其中聊城市境内正线长度115.271公里，德州市境内正线长度20.48公里，济南市境内正线长度33.873公里<ref>{{cite web |title=郑济高铁选址公示，山东段拟设5个站点，这些高铁也有新进展--山东频道--人民网 |url=http://sd.people.com.cn/n2/2018/1015/c166192-32156311.html |website=sd.people.com.cn |accessdate=2018-10-15}}</ref>。山东段设计时速350公里,总投资324.5亿元。郑济高铁山东段由聊城市莘县进入山东省境内,途经聊城市下辖的莘县、阳谷县、东昌府区、茌平县，德州市下辖的齐河县，济南市下辖的长清区、市中区、槐荫区到达济南西站。<ref >{{cite web|url=http://news.sina.com.cn/o/2018-01-18/doc-ifyqtycw9692852.shtml | language= | title= 定了！郑济高铁长清段今年开工！|publisher=山东商报 |date=2018-01-30 |accessdate= }}</ref>并且，将建设联络线通往济南站并在该站于[[济青高铁|济青高铁]]相连。山东段预计2018年底开工。

== 沿途站点 ==
根据现有规划，河南段设置七站，即郑州东、平原新区、新乡东、卫辉南、滑县浚县、内黄、濮阳东和南乐站；山东段设置五站，即莘县站、聊城西站、茌平南站、长清站、济南西站（正线接入）。同时，郑济高铁还将在济南市内建设联络线通往济南站。
<br>

== 线路之争 ==
* '''豫鲁南北线路之争'''<br>
济南是山东省会，而长清和平阴为济南下辖两县。济南市基于该两县发展的考虑，希望郑济高铁从济南出发，历经长清、平阴以及梁山、菏泽到达开封，然后连接郑州（即南线方案）。此方案优点是铁路一直位于黄河南岸，无需新建黄河大桥；缺点是铁路通过东平湖区，湖水泛滥容易影响高铁安全运行。而河南省则有自己的考虑，由于濮阳作为河南省东北部城市到现在还没有通火车，河南省自然希望新修建的高铁通过濮阳，然后进入山东境内。由此，两省争执不下。最终，山东省让步，郑济高铁经濮阳进入山东（即北线方案）。此方案要在郑州与济南新建两座跨黄河大桥。两省的争执大大延缓了郑济高铁开工时间。<br>
* '''山东南北线路之争'''<br>
豫鲁线路达成协议后，郑济高铁在山东境内又陷入南线北线之争。北线走莘县-聊城-茌平-齐河-济南西，南线走莘县-聊城-茌平-长清-济南西站。南线北线问题长期争论不休。最终铁总支持南线方案，山东省也同意了该方案。2018年初，聊城市、德州市齐河县、济南长清区先后发布郑济铁路社会稳定风险分析公众参与公示，间接宣布山东境内线路之争结束。<br>山东段目前已确定采用南线方案，即经过济南市长清区且设站，过德州市齐河县南部不设站。

== 高铁意义 ==
尽管郑州到济南直线距离只有370公里，但是由于两地没有直线交通需要通过[[京沪铁路|京沪铁路]]、[[陇海铁路|陇海铁路]]绕行[[徐州|徐州]]、[[砀山|砀山]]等地，非常费时不便。两地之间唯一的一趟动车需要5个多小时，普通列车则需8到11个小时。2016年9月，[[郑徐高铁|郑徐高铁]]开通后，郑州至济南增开高铁，郑州到济南需时3个半小时。<br>
待郑济高铁开通后，郑州到济南由弯变直，无需绕行[[徐州|徐州]]，单程只需一个多小时，<ref >{{cite web|url= http://hn.chinaso.com/finance/detail/20160923/1000200032826601474588208557964461_1.html| language= | title=郑济高铁河南段挂网招标　未来郑济仅需一个多小时 |publisher=大河报 |date=2016-09-23 |accessdate= }}</ref><ref >{{cite web|url=http://news.focus.cn/puyang/2016-09-18/11140973.html | language= | title=郑济高铁郑州至濮阳段公开招标 11月1日开工建设 |publisher= 搜狐焦点网 |date=2016-09-18 |accessdate= }}</ref>从青岛至郑州的铁路客车时间也将缩短为三个多小时。郑济高铁开通后，高铁列车还可通过郑西高铁、西兰高铁、兰新高铁、郑渝高铁等快速抵达西北、西南地区。郑济高铁与济青高铁相连后,将与京沪高铁在济南形成十字交叉,使济南成为全国少有的时速350公里高速铁路十字交叉城市,强化济南的交通枢纽地位。
