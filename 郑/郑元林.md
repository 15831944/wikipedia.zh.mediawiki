'''郑元林'''（{{bd|1962年|4月|||}}），[[山东|山东]][[平度|平度]]人，[[中国人民解放军少将|中国人民解放军空军少将]]。

1978年参加[[中国人民解放军|中国人民解放军]]，大型军用运输机飞行员出身，曾任[[中国人民解放军空军|空军]]某运输航空兵团长、师参谋长、[[空军指挥学院|空军指挥学院]]战役班学员、[[广州军区空军|广州军区空军]]航空兵13师师长，2007年被评为全军优秀指挥军官<ref>{{cite web |title=2007年度全军和武警部队优秀指挥军官人才名单 |url=http://military.people.com.cn/GB/1076/52984/6664143.html |website=人民网 |accessdate=2019-01-22 |date=2007-12-17}}</ref>。2010年1月，任广州军区空军副参谋长。2010年3月，任空军参谋长助理。任内组织空军[[2010年玉樹地震|玉树抗震救灾]]行动，表现出色<ref>{{cite web |author1=妮爾硯 |title=止戈為武：空軍軍官調整凸顯軍事變革 |url=http://paper.wenweipo.com/2010/05/17/PL1005170003.htm |website=文匯報 |accessdate=2019-01-22 |date=2010-05-17}}</ref>。2012年10月，升任[[成都军区空军|成都军区空军]]参谋长<ref>{{cite web |author1=刘安红 |author2=蒋德玉 |title=昭通机场迁建选址阶段 |url=http://www.zynews.com.cn/article/show-125151.html |website=昭阳信息网 |accessdate=2019-01-22 |date=2012-10-18}}</ref>。2013年3月，改任[[广州军区空军|广州军区空军]]参谋长。2016年1月，出任新组建的[[南部战区空军|南部战区空军]]参谋长<ref>{{cite web |author1=刘理桂 |title=苏海棠率团慰问南部战区空军参谋部官兵 |url=http://www.gxylnews.com/html/news/2016/02/119471.html |website=玉林新闻网 |accessdate=2019-01-22 |date=2016-02-01}}</ref>。2018年12月，升任空军副司令员，跻身副战区级<ref>{{cite web |author1=岳怀让 |title=解放军空军总部迎来一名副司令员：南空参谋长郑元林升任 |url=https://www.thepaper.cn/newsDetail_forward_2865679 |website=澎湃新闻网 |accessdate=2019-01-22 |date=2019-01-17}}</ref>。

2010年7月，晋升空军少将军衔。
