'''郑珣瑜'''（{{bd|738年||805年|12月11日|catIdx=Z}}<ref name=BT14>[[:s:旧唐书/卷14|《旧唐书》卷一十四]]</ref>），字'''元伯'''，唐朝官员，[[唐德宗|唐德宗]]、[[唐顺宗|唐顺宗]]年间任[[宰相|宰相]]。[[郑余庆|郑余庆]]堂兄。

== 家世 ==
郑珣瑜出自[[荥阳郑氏|荥阳郑氏]]北祖第二房，祖父[[鄭長裕|鄭長裕]]曾任[[许州|许州]]刺史，父亲[[鄭諒|鄭諒]]曾任冠氏令。<ref>《新唐书》卷七十五：{{cite web |url=http://www.sidneyluo.net/a/a17/075.htm |title=存档副本 |accessdate=2010-02-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20091219203329/http://www.sidneyluo.net/a/a17/075.htm |archivedate=2009-12-19 }}{{cite web |url=http://www.sidneyluo.net/a/a17/table/form90.htm |title=存档副本 |accessdate=2010-06-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20100620182223/http://sidneyluo.net/a/a17/table/form90.htm |archivedate=2010-06-20 }}</ref>

郑珣瑜少孤，[[天宝_(唐朝)|天宝]]十四年（755年）[[安史之乱|安史之乱]]爆发后，他携母亲进入[[陆浑山|陆浑山]]赡养并抚养弟妹，不回郑州。乱平后，[[转运使|转运使]][[劉晏|劉晏]]奏请补他为[[寧陵|寧陵]]、宋城尉，[[山南節度使|山南節度使]][[張獻誠|張獻誠]]表他为[[南鄭|南鄭]]丞，他都谢绝不应。[[唐代宗|唐代宗]][[大历|大历]]六年（771年），郑珣瑜通过了諷諫主文科考试，被任为[[大理評事|大理評事]]，后为了得到俸禄奉养母亲，求得调任[[陽翟|陽翟]]丞，<ref name=CFYG>《册府元龟》</ref>因政绩出众，被任为万年（组成都城[[长安|长安]]的两县之一）尉。<ref name=NBT165>[[:s:新唐书/卷165|《新唐书》卷一百六十五]]</ref>

== 唐德宗年间 ==
唐代宗[[大历|大历]]十四年（779年），[[崔祐甫|崔祐甫]]拜相，升郑珣瑜为[[左補闕|左補闕]]，又出任[[涇原|涇原]]帥府判官，又回长安任[[侍御史|侍御史]]、刑部員外郎，后因母丧去职，期满后迁[[吏部|吏部]]任职。唐德宗[[贞元_(唐朝)|贞元]]二年（786年），德宗诏令选择十个中央官员分任长安、[[洛阳|洛阳]]地区的地方官，郑珣瑜检校本官吏部员外郎兼奉先（地近长安）令。次年五月，任为[[饶州_(隋朝)|饶州]]刺史，<ref>《[[册府元龟|册府元龟]]》卷七百零一</ref><ref>《[[太平御览|太平御览]]》卷二百五十五</ref>后又回长安任[[谏议大夫|谏议大夫]]，七年（791年）八月以[[给事中|给事中]]为[[中书舍人|中书舍人]]，八年（792年）十月，监外官考。<ref name=CFYG/>共经四次升迁为[[吏部侍郎|吏部侍郎]]。<ref name=NBT165/><ref name=BT13>[[:s:旧唐书/卷13|《旧唐书》卷一十三]]作“郑瑜”</ref>

十一年（795年）三月，受任为河南、淮南水陆转运使，<ref name=BT13/>又任[[河南尹|河南尹]]（即洛阳地区長官），尚未赴任，正逢唐德宗生日，按习惯，河南尹应该献马为寿礼。郑珣瑜手下的小吏请求挑选一匹马上献，郑珣瑜拒绝了：“还没上任就献礼，合适吗？”郑珣瑜为人严肃寡言，从不拿自己的私事拜托他人，人们也不敢打听他的爱好。到任洛陽后，清静惠下，减少赋税，增加津贴以便民。<ref name=NBT165/>当时，夏绥节度使[[韩全义|韩全义]]为蔡州四面行营招讨使正统帅十七道兵马攻打叛将[[吴少诚|吴少诚]]。<ref>韩全义伐吴少诚之役在贞元十六年（800年），大约此时郑珣瑜已任河南尹。见《[[资治通鉴|资治通鉴]]》卷二百三十六。</ref>河南地区负责将军队补给水运给韩全义，郑珣瑜将食物储存在[[阳翟|阳翟]]，减少了百姓运粮的负担。韩全义和监军宦官对河南屡有索取，但除了有诏令授权的以外，其余数百封信函郑珣瑜都不理会。有下属担心他此举会冒犯了韩全义和宦官，郑珣瑜答道：“将军们作战，常以此为由索取。如果我的拒绝是罪，作为尹，我自当承担后果。我终不会浪费一万劳工的劳动成果。”郑珣瑜迎送中使皆有固定地址，差距不出马脚三五步。<ref name=CFYG/>时人认为郑珣瑜对河南的治理堪比前任[[张延赏|张延赏]]，又比张延赏更严肃、宽厚、正直。郑珣瑜又被召回再任吏部侍郎。<ref name=NBT165/>郑珣瑜请尚书礼部考功二员外郎[[裴垍|裴垍]]考校辞判，裴垍守正不受请托，考核务实。<ref>[[:s:旧唐书/卷148|《旧唐书》卷一百四十八]]</ref><ref>[[:s:新唐书/卷169|《新唐书》卷一百六十九]]</ref>十九年（803年）十二月，他被任为[[门下侍郎|门下侍郎]]，和[[高郢|高郢]]一同被授[[同中书门下平章事|同中书门下平章事]]，为实质宰相。<ref name=NBT7>[[:s:新唐书/卷7|《新唐书》卷七]]</ref><ref name=ZZTJ236>[[:s:資治通鑑/卷236|《资治通鉴》卷二百三十六]]：十二月，庚申，以太常卿高郢為中書侍郎，吏部侍郎鄭珣瑜為門下侍郎，並同平章事。珣瑜，[[郑余庆|餘慶]]之從父兄弟也。……丁酉，諸宰相會食中書。故事，宰相方食，百寮無敢謁見者。叔文至中書，欲與執誼計事，令直省通之，直省以舊事告，叔文怒，叱直省。直省懼，入白。執誼逡巡慚赧，竟起迎叔文，就其閤語良久。杜佑、高郢、鄭珣瑜皆停箸以待，有報者云：「叔文索飯，韋相公已與之同食閤中矣。」佑、郢心知不可，畏叔文、執誼，莫敢出言。珣瑜獨歎曰：「吾豈可復居此位！」顧左右，取馬徑歸，遂不起。二相皆天下重望，相次歸臥，叔文、執誼等益無所顧忌，遠近大懼。……又以鄭珣瑜為吏部尚書，高郢為刑部尚書，並罷政事。</ref><ref name=BT147>[[:s:旧唐书/卷147|《旧唐书》卷一百四十七]]</ref><ref>《册府元龟》作十一月事。</ref>时德宗近臣嗣道王[[李实_(唐朝)|李实]]为[[京兆尹|京兆尹]]，以为了上贡而严酷抽税闻名。郑珣瑜对此不悦，讯问他：“京兆尹府的收入明细都当记载，多余的应当上交度支。你的贡品都是哪来的？”但李实正得宠，未因郑珣瑜讯问而受损。<ref name=NBT165/>

当时集贤殿学士很多，德宗诏问[[神策军|神策军]]建置始末，[[中书省|中书省]]宰相不知，学士们也都不能对答，于是宰相们访求司勋员外郎兼史职[[蒋义_(唐朝)|蒋乂]]，蒋乂旁征博引回答得很详细明白，高郢和郑珣瑜相对叹道：“集贤殿有人了！”翌日，诏蒋乂兼判集贤院事。<ref>[[:s:旧唐书/卷149|《旧唐书》卷一百四十九]]</ref><ref>[[:s:新唐书/卷132|《新唐书》卷一百三十二]]</ref>德宗想要文学之士，有人推荐校书郎[[李建|李建]]，德宗问左右，郑珣瑜说：“臣在吏部时，当补校书者八人，其他人都是仗着贵势求官的，只有李建不是。”德宗喜，擢李建为左拾遗、翰林学士。<ref>[[:s:新唐书/卷162|《新唐书》卷一百六十二]]</ref>

== 唐顺宗、唐宪宗年间 ==
[[贞元_(唐朝)|贞元]]二十一年（805年）正月，唐德宗崩，子唐顺宗继位，三月任郑珣瑜为[[吏部尚书|吏部尚书]]，相位依旧。当时顺宗近臣[[王叔文|王叔文]]权势很大，又和与郑珣瑜同列相位的[[韦执谊|韦执谊]]联盟。一次宰相们按例一起吃午饭时，王叔文闯入中书省看望韦执谊。有小吏拒绝王叔文入内，被斥退。韦执谊从座位上起身走开和王叔文说话。郑珣瑜和其他宰相[[杜佑|杜佑]]、高郢停止用餐，等候韦执谊回来。过了一会儿，他们派小吏去查看，小吏报告：“王叔文索要饭食，韦宰相已和他一起用餐了。”杜、高都没敢多言，郑珣瑜却说：“我怎么还能在这里！”看看左右，命随从取马，一回家就卧床七天不起。<ref name=NBT165/>而高郢却无所作为，时人因此认为郑珣瑜优于高郢。<ref name=BT147/>七月，顺宗病重，以[[皇太子|皇太子]][[唐宪宗|李纯]][[监国|监国]]，时任[[银青光禄大夫|银青光禄大夫]]、守吏部尚书、平章事、[[上柱国|上柱国]]的郑珣瑜和高郢都被罢相。<ref name=NBT7/>八月，顺宗传位李纯，李纯即[[唐宪宗|唐宪宗]]，<ref name=ZZTJ236/>十月，郑珣瑜卒，<ref name=BT14/>贈尚書左僕射。太常博士[[徐复|徐复]]议谥文献，兵部侍郎[[李巽|李巽]]认为二谥不合《[[春秋_(史书)|春秋]]》，请再议，徐复坚持二谥在周、汉皆有先例，且郑珣瑜是名臣，足以二谥。诏从徐复议，谥'''文献'''，后谥'''文宪'''<ref>《卢载妻郑氏墓志铭》载：“故门下侍郎平章事、文宪公讳珣瑜，夫人之从祖父也。旧相、今东都留守检校兵部尚书馀庆，夫人之叔父也。</ref>。子[[郑覃|郑覃]]、[[郑朗|郑朗]]分别在[[唐文宗|唐文宗]]和[[唐宣宗|唐宣宗]]年间拜相。<ref name=NBT165/>后又赠[[太师|太师]]。<ref>《唐荥阳郑氏女墓志铭并序》（墓主为郑朗女郑子容）</ref>

== 评价 ==
*《册府元龟》：性公直严重少言，未尝以私徇人，而人望其风亦不敢干以私，所居皆有理绩。
*[[欧阳修|欧阳修]]《[[新唐书|新唐书]]》赞曰：“王叔文虽内连姏尹，外倚奸回，以攘天权。然是时太子已长，朝无嫌罅，若珣瑜、郢与杜佑等毅然引东宫监国，执退叔文辈，其力不难。顾循嘿苟安，所谓焉用彼相者矣。珣瑜一忿卧第，与郢、佑固位，二者亦不足相轻重云。”即认为郑珣瑜、高郢、杜佑足以毅然推戴皇太子监国、罢黜王叔文等，却只是苟安，没有尽宰相之责，郑珣瑜一怒之下卧床，也比苟安于位的高郢、杜佑好不到哪里去。
