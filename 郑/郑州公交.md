'''郑州公共交通汽车系统'''，由1954年1月19日成立郑州市公共汽车公司筹备，自1954年2月1日正式运营，采用[[公车专用道|中央专用车道]]和普通道路并存的方式，以[[郑州火车站|郑州火车站]]和[[郑州东站|郑州东站]]为中心点、纵横环线交叉的网状线路网络，主要服务于郑州市区，并有连接郑州城区与郊县，总共有300多条线路。郑州公交常规公交线路条数由2010年底的222条增加到2016年的300余条，线路长度由3521.56公里增加到目前的4301.2公里。截至2016年，郑州公交日均运营里程达到80万公里，日均运送乘客280万余人，公交出行分担率为28%，在中心城区，主干线路平均发车间隔3-5分钟，市内一般不超过500米就可以坐上公交车，基本上实现了“有路就有公交车”<ref>{{Cite web |url=http://www.zhengzhoubus.com/QiYeJianJie.aspx?mid=29 |title=郑州公交大事纪 |date=  |publisher=郑州市公共交通总公司}}</ref><ref>{{Cite web |url=http://www.zgxw.org/Html/?11988.html |title=记郑州市公共交通总公司总经理巴振东 |date=2016-10-29  |publisher=中国新闻播报}}</ref>。

== 运营线路 ==
郑州公交目前共有300余条线路正在运营，其主要运营者为郑州市公共交通总公司，亦有绿城巴士公司（郑州市公共交通总公司、郑州交运集团共同建立）等公司。除日间线路外还运营有Y开头的夜班线路，市区夜班线路运营到凌晨24点，[[郑州航空港区|港区]]夜班线路Y636则运营到次日凌晨3点。同时还运营有开往市区外景区的专用旅游线路。目前郑州的公交线路配车绝大多数为[[宇通客車|宇通客車]]，少部分为[[少林客车|少林客车]]，亦曾有金陵、京华、[[东风扬子江汽车|扬子江]]、[[恒通客车|恒通]]等品牌的配车，但已全数退役。

== 票价规定 ==
郑州公交常规市内线路统一票价为1元，同时还提供绿城通公交卡以及二维码公交卡服务{{NoteTag|私营线路票价为2元，且多数线路不支持刷卡}}。其中绿城通公交卡普卡享有8折乘车优惠，持老年卡可按相关规定每月免费乘80次，学生卡可享受半价优惠。

开往郊县的线路和市区外景区的旅游线路票价为2—5元，绿城通普卡可以享受8折优惠，其他卡种不可用。

===绿城通公交卡===
{{Main|绿城通}}
绿城通，是由郑州城市一卡通有限责任公司负责设计、运营，用于中国河南省郑州市公交、地铁等公共服务领域支付的智能卡及其相关系统，于2013年12月23日起正式发售。

===二维码支付===
郑州公交可以使用[[支付宝|支付宝]]乘车二维码和郑州公交[[微信小程序|微信小程序]]进行扫码支付。
