'''郑州自然博物馆'''位于[[郑州市|郑州市]]文化北路与开元路交叉口<ref name="联系">{{cite web |url=http://www2.zztc.com.cn/zmnh/lxwm/5915.html |language= |title=联系我们 |publisher=郑州自然博物馆 |date=2009-12-24 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20120504214442/http://www2.zztc.com.cn/zmnh/lxwm/5915.html |archivedate=2012-05-04 }}</ref>，是由[[郑州市|郑州市]]政府专项资金支持、[[郑州师范高等专科学校|郑州师范高等专科学校]]承建的自然类[[博物馆|博物馆]]，建立于2009年10月<ref>{{cite web |url=http://www2.zztc.com.cn/zmnh/bggk/gzzc/5875.html |language= |title=馆长致辞 |publisher=郑州自然博物馆 |date=2009-12-23 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20120504214359/http://www2.zztc.com.cn/zmnh/bggk/gzzc/5875.html |archivedate=2012-05-04 }}</ref>。主要收藏展览动物、植物、古生物、矿物、土壤、水资源等领域的标本，集教学、科研、科普于一体。<ref name="简介">{{cite web |url=http://www2.zztc.com.cn/zmnh/bggk/bwgjj/5863.html |language= |title=郑州自然博物馆简介 |publisher=郑州自然博物馆 |date=2009-12-23 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20150603135242/http://www2.zztc.com.cn/zmnh/bggk/bwgjj/5863.html |archivedate=2015-06-03 }}</ref>

== 简介 ==
展厅面积3,000平米，共设9个常设展区：
*一层设：[[嵩山|嵩山]]之春、南国夏风、大河金秋、冰雪风情、水生世界；
*二层设：[[昆虫|昆虫]]世界、生命支撑、绿色畅想、造化神秀。

展馆开辟了实验天地，为互动式探索自然奥秘的科普教育活动场所。展馆收藏、展示近万件[[标本|标本]]，其中有[[扬子鳄|扬子鳄]]、[[巨蜥|巨蜥]]、[[丹顶鹤|丹顶鹤]]、[[金雕|金雕]]、[[金钱豹|金钱豹]]等国家珍贵动物标本近百件。

此外，博物馆每年会推出若干个临时性专题展览。<ref name= "简介"/>
