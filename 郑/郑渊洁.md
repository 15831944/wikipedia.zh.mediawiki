'''郑渊洁'''（{{bd|1955年|6月15日|catIdx=Zheng郑}}），[[中国大陆|中国大陆]][[作家|作家]]、[[慈善家|慈善家]]，1955年出生于中国[[河北省|河北省]][[石家庄市|石家庄市]]，其父[[郑洪升|郑洪升]]。一个人写一本月刊（《[[童话大王|童话大王]]》）的[[世界纪录|世界纪录]]保持者，有“'''童话大王'''”之称。1979年发表第一篇童话作品《黑黑在诚实岛》。笔下的知名人物有皮皮鲁、鲁西西、舒克、贝塔、大灰狼罗克等。代表作有《皮皮鲁外传》、《鲁西西外传》、《旗旗号巡洋舰》、《舒克贝塔历险记》等。同时还从事慈善活动。

==经历==
1955年6月15日，郑渊洁出生于[[河北|河北省]][[石家庄|石家庄市]][[白求恩国际和平医院|白求恩国际和平医院]]。

1961年随父母迁入[[北京|北京]]。曾就读于北京市[[马甸小学|马甸小学]]（现在已改名为民族小学），1964年加入[[中国少年先锋队|中国少年先锋队]]。四年级时因“[[文革|文革]]”爆发，暂时中止学校教育。

1969年11月至1970年12月随父母下放至[[河南省|河南省]][[遂平县|遂平县]][[国防科委|国防科委]][[五七干校|五七干校]]（“938”部队）劳动。在由[[下放|下放]]劳动人员组成的五七干校临时学校就读初中。由于上课玩[[拉炮|拉炮]]而再次失去在校学习机会。由其父亲郑洪升督导在家自学。

在此期间，与中国第一位原子弹将军[[张蕴钰|张蕴钰]]成为忘年交。

1970年至1976年，分别在[[江西|江西]]和[[福建|福建]]的[[空军|空军]]部队服兵役，任[[地勤|地勤]]兵。

1976年[[复员|复员]]后，在北京大华无线电仪器厂当工人，负责看管工厂的[[水泵|水泵]]。

复员当年受[[四五运动|四五运动]]的激励开始文学创作。创作风格受[[张天翼|张天翼]]的影响比较大。先后创作过诗、歌词、小说、科幻小说、童话等文学体裁。1979年9月15日，在《[[儿童文学|儿童文学]]》上发表了第一篇童话《黑黑在诚实岛》。先后在《[[中国盲童文学|中国盲童文学]]》、《[[儿童文学|儿童文学]]》等杂志社当编辑，还参与创办了《[[东方少年|东方少年]]》、《[[大灰狼画报|大灰狼画报]]》、《[[中国盲童文学|中国盲童文学]]》三本杂志。

1985年创办《[[童话大王|童话大王]]》月刊，专门刊登郑渊洁一个人的作品。1988年最高时发行量超过每月100万册。郑渊洁作品仅正版总印数逾亿册。据《[[中华读书报|中华读书报]]》报道，中国大陆[[盗版|盗版]]郑渊洁作品非法获利就超过2000万元[[人民币|人民币]]。

1990年4月4日由[[台湾|台湾]][[牛顿出版公司|牛顿出版公司]]在台湾代理出版《童话大王～郑渊洁作品月刊》（一共发行了60期）。

1990年代中期，“十二生肖童话”系列在[[韩国|韩国]]出版。

1994年在经历了一次未遂的“封笔”计划后，郑渊洁应读者强烈要求，继续童话创作。从这年起，郑渊洁开始了长达十年的“隐居”状态，潜心创作，其文风也发生了巨大的变化。

2002年1月起，中国大陆版《童话大王》暂停刊登新作，《鬼车》在《童话大王》上停止连载，而长篇小说《仇象》更是未刊登。这三件事引起很多读者的不满。不过郑渊洁也在多种场合表示这样并非出于自愿。自2004年7月起，《童话大王》重新刊登郑渊洁新作，但仍没刊登于在2002年停止连载的《鬼车》。后《鬼车》作为单行本得以发行。2005年10月号的《皮皮鲁》画册开始连载《仇象》，但一些人就郑渊洁在[[新浪|新浪]]网发表的一些言论进行批评，认为郑渊洁在《皮皮鲁》画册这种儿童刊物上登载不适合少年儿童看的内容会影响少年儿童的身心健康。{{noteTag|1= 见“[[秦尘|秦尘]]批判郑渊洁”}} 郑渊洁决定暂时停止《仇象》在《皮皮鲁》画册上的连载，停止本来准备出版的长篇小说《谜图》《反动人》《压哨球》《空镜头》等。

2004年受两位从外地赴京寻他并成功相见的书迷影响，郑渊洁结束了长达十年的“隐居”状态，频频参与社会活动。他开始参加电视台节目录制。包括人物访谈、厨艺等节目。与此同时，郑渊洁在自己的官方网站上开了“[[郑氏胡说|郑氏胡说]]”的节目。这是《童话大王》的电视版，与文本相比，更具现代气息。从第8集开始不再在网上播放视频，仅在《皮皮鲁画册》上刊载文字版。后又与长影集团合作，制作了专为电视播放的《郑氏胡说》节目。此节目在自2006年7月2日起，在吉林电视台长影频道及中国大陆多家电视台同步播出。

2005年11月在新浪网开设[[Blog|Blog]]“勃客郑渊洁”，坚持每天更新，并称要写20年Blog，“再创一个奇迹”（指《童话大王》杂志出版逾20年是第一个奇迹）。

郑渊洁还称有十几部长篇小说以及为其子编写的四百万字教科书与家庭教育学说，以及为其女讲述的一百多部低幼童话尚没有发表。他说有可能要等到中国大陆解除[[报禁|报禁]]，允许私人成立出版社的时候再发表。<ref>{{Cite web|url = http://www.damai.cn/artistintro_60302.html|title = 郑渊洁简介|accessdate = |author = |date = |publisher = }}</ref>

2010年4月，他发表文章表示退出[[中国作家协会|中国作家协会]]，并称[[中国作协全国委员会|中国作协全国委员会]]难以促进文学的发展。<ref>{{cite web|url=http://news.163.com/10/0425/13/6549VP9B000146BD.html|title=郑渊洁宣布退出中国作协 称作协难促进文学发展|publisher=网易|accessdate=2010-04-25}}</ref>

2010年4月25日上午12时05分，郑渊洁在其[[新浪|新浪]]微博上发表博文，宣布退出[[中国作家协会|中国作家协会]]。随后，微博作为一种新的信息传播平台开始在中国流行，郑渊洁借助其微博上庞大的粉丝群，在互联网上逐渐发挥越来越大的影响力。除了他为快乐男声选手[[刘心|刘心]]统计粉丝数量的一条微博短时间内即被转了220多万次，郑渊洁还在微博上进行了“郑在打卡”、为[[宠物|宠物]]征名、“通缉”在逃贪官、晒照片“监督”公交站、打拐、寻找多年前给他写过信的读者等等活动，均成为互联网热门话题，也创造着一个又一个奇迹。如今的童话大王每天都会登陆自己的微博，除了继续向全中国的孩子们推广郑氏快乐学习的理念外，郑渊洁也广泛参与到各种[[维权|维权]]活动中，积极履行其[[纳税人|纳税人]]的义务，尽自己的力量推动着[[中国|中国]][[法系|法制体系]]的发展和完善。

2010年9月，郑渊洁在其[[新浪|新浪]]微博发表博文直截了当的抨击[[中国大陆|中国大陆]][[小学|小学]][[班干部|班干部]]制度培养“[[汉奸|汉奸]]”，称“小学班干部制度就是培养叛徒的体制。老师不在时，班干部要记下‘不守纪律’同学的名字，报告给老师”。11月，郑渊洁发起一则 “关于取消小学班干部制度的调查”，选项分为“赞成”和“反对”两项，在1588张投票当中，89%网友选择“赞成”，只有11%的网友选择“反对”。<ref>{{cite web |url=http://culture.ifeng.com/whrd/detail_2011_12/14/11311016_0.shtml|title=《童话大王郑渊洁：小学班干部制度培养“汉奸”》|accessdate=2011年12月14日|work=李征 钱钰 杨玉红|publisher=凤凰网|language= zh-hans}}</ref>

2019年4月18日，大星文化和《华西都市报》等单位共同发布“第13届作家榜童书作家榜”，名列前三位的是作家[[杨红樱|杨红樱]]、[[北猫|北猫]]和[[曹文轩|曹文轩]]，而去年在作家财富榜上以2100万版税排名第三的郑渊洁却意外消失。由此引发网友质疑郑渊洁宣称的畅销有“注水”嫌疑。4月19日，郑渊洁发长微博回应网友，称在榜单发布前，他就主动向制作方表示“拒绝上榜”。原因是他认为中国童书销售市场存在极大泡沫，包括曹文轩在内的很多童书作家，是在进校园签售后才积累了海量读者，而这种行为“涉嫌违法”，所以他拒绝参与评比。<ref>{{cite web |url=https://www.yicai.com/news/100168031.html|title=郑渊洁炮轰童书富豪榜调查：名作家要求进校园须包销一万册|accessdate=2019年04月22日|work=彭晓玲 孙行之|publisher=第一财经|language= zh-hans}}</ref>

===社会活动===
1992年创办[[郑渊洁少儿用品开发有限公司|郑渊洁少儿用品开发有限公司]]。

1993年当选首届[[北京市十大杰出青年|北京市十大杰出青年]]。

2005年10月19日，郑渊洁出任北京[[府学胡同小学|府学胡同小学]]“皮皮鲁文学社”名誉社长，这是他首次授权小学使用“皮皮鲁”作为文学社名称，也时首次担任学校文学社名誉社长。

2005年11月11日，北京2008年[[奥运会|奥运会]][[福娃|吉祥物]]公布。郑渊洁是北京奥运会吉祥物评委中惟一的[[儿童文学|儿童文学]]作家。

2006年2月14日郑渊洁授权[[长影世纪城|长影世纪城]]无偿使用皮皮鲁和鲁西西的卡通肖像权。

2006年3月23日52集大型奥运主题动画片《[[福娃|福娃]]》在[[深圳|深圳]]正式启动制作，郑渊洁担任总编剧。后因“对《福娃》动画片的理解和该公司有出入”，辞去了总编剧一职。

2006年3月30日，郑渊洁在北京[[人民大会堂|人民大会堂]]公布，他的作品全面影视化，作品影视改编权全部交给[[长春电影制片厂|长影]]集团，拍摄皮皮鲁鲁西西等为主角的系列影视作品，并将重拍舒克和贝塔系列动画片。皮皮鲁、鲁西西扮演者，将通过[[中央电视台|中央电视台]][[海选|海选]]。

2009年3月16日，郑渊洁宣布彻底告别手机，原因是看了央视有关通信运营商将手机用户信息随意出售并发送[[垃圾短信|垃圾短信]]的报导，故认为“手机用户像傻子”。<ref>[http://tech.163.com/09/0317/09/54JLH8OE000915BE.html 郑渊洁自曝将弃用手机]</ref>

2010年4月20日晚，在央视的玉树地震捐赠晚会上，郑渊洁慷慨解囊，一次性捐赠100万元，捐赠数额遥遥领先于国内其他作家。晚会上，面对央视主持人董卿的话筒，他用平和的语调动情地说：“我捐出的是我的稿费，用于灾区小学的重建，我希望我捐的小学能抗8级地震，让孩子们在固若金汤的校园里安全快乐地学习”。而在此前的汶川地震上，郑渊洁也为灾区捐出38万，同样为中国国内作家捐款最多者。

2010年4月25日上午12时05分，郑渊洁在其新浪微博上发表博文，宣布退出[[中国作家协会|中国作家协会]]。随后，微博作为一种新的信息传播平台风靡中国，郑渊洁借助其微博上庞大的粉丝群，在互联网上逐渐发挥越来越大的影响力。截至目前，郑渊洁在新浪微博上的粉丝数已超过360万。

== 教子 ==
郑渊洁因“不满应试教育的摧残”，令其儿子[[郑亚旗|郑亚旗]]小学毕业退学在家自学，郑渊洁编写了10部教材教育郑亚旗。10本教材都用童话手法撰写，有[[哲学|哲学]]篇《鲁西西和苏格拉底对话录》、[[法系|法制]]篇《[[皮皮鲁和419宗罪|皮皮鲁和419宗罪]]》、[[生理|生理]]知识篇、[[道德|道德]]篇、[[数理化|数理化]]篇和[[艺术|艺术]]篇等等。总字数有四百万多字。

郑渊洁在一些电视访谈节目中表示，改编后的动画片等很难令他满意。但他仍在推进作品影视化的步伐。
