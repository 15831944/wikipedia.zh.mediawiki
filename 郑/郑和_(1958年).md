'''郑和'''（{{bd|1958年|11月|}}），[[上海市|上海市]]人。[[中国人民解放军中将|中国人民解放军中将]]。[[中国共产党第十九届中央委员会委员列表|中共第十九届中央委员]]。

==生平==
[[俄罗斯|俄罗斯]][[俄羅斯聯邦軍隊綜合軍事學院|伏龙芝军事学院]]战役战术专业毕业，研究生学历，军事学硕士学位。早年长期在[[中国人民解放军南京军区|南京军区]]服役，曾任[[中国人民解放军陆军第七十三集团军|陆军第三十一集团军]]旅长、师长。2008年6月，任陆军第三十一集团军参谋长。2011年2月，任南京军区副参谋长。2013年2月，接替[[陈照海|陈照海]]担任[[中央军委训练管理部|中国人民解放军总参谋部军训部]]第二任部长。2015年7月，任[[中国人民解放军成都军区|成都军区]]副司令员<ref>{{cite news|title=总参谋部军训部部长郑和少将升任成都军区副司令员|url=http://www.thepaper.cn/newsDetail_forward_1355912|accessdate=2016-07-08|newspaper=澎湃新闻}}</ref>。

2016年2月，任[[中央军委训练管理部|中央军委训练管理部]]部长<ref>{{cite news|title=原成都军区副司令员郑和少将任军委训练管理部部长|url=http://www.thepaper.cn/newsDetail_forward_1419670|accessdate=2016-07-08|newspaper=澎湃新闻}}</ref>。2017年1月，接替[[蔡英挺|蔡英挺]]上将升任[[中国人民解放军军事科学院|中国人民解放军军事科学院]]院长。2017年6月，调任[[中国人民解放军国防大学|中国人民解放军国防大学]]校长<ref>{{Cite web |url = http://news.ifeng.com/a/20170620/51284757_0.shtml |title = 国防大学新“掌门人”就职 郑和出任校长 |author = |date = 2017-06-20 |publisher = 凤凰网 |language = |accessdate = |quote = }}</ref>。

2009年晋升少将军衔<ref>{{cite news|title=郑和白吕履新成都军区|url=http://m.china.caixin.com/m/2015-07-24/100832480.html|accessdate=2016-07-08|newspaper=财新网}}</ref>。2016年7月晋升中将军衔<ref>{{cite news |title =郑和中将晋升正战区级，曾任中央军委训练管理部部长|url = http://www.thepaper.cn/newsDetail_forward_1629194 |accessdate=2017-03-02 |publisher = 澎湃新闻}}</ref>。
