'''郑天亮'''{{BD|1978年|11月28日|||catIdx= Z郑}}出生于[[山东省|山东省]][[烟台市|烟台市]]，毕业于[[中国传媒大学|中国传媒大学]]，为中国中央电视台节目主持人。<ref name=" bear" >{{cite web |url=http://www.zhuchijie.com/show_208_z7_t1.html |title=郑天亮 |publisher=主持界 |accessdate=2011年 |work= |language =}}</ref>

== 简介 ==
1978年，郑天亮出生于[[山东省|山东省]][[烟台市|烟台市]]。<ref name=" bear" />

1997年，郑天亮考入[[中国传媒大学|中国传媒大学]]，在播音主持艺术学院攻读四年。<ref name=" bear" />

2001年，郑天亮被分配到[[中国中央电视台|中国中央电视台]]工作，主持《[[东方时空|东方时空]]》、《[[时空资讯|时空资讯]]》两个栏目。<ref name=" bear" />

2002年至今，郑天亮主持《[[每日资讯|每日资讯]]》、《[[新闻早报|新闻早报]]》、《[[午夜新闻|午夜新闻]]》三个新闻栏目。<ref name=" bear" />

== 个人生活 ==
郑天亮的丈夫是中国中央电视台主持人[[鲁健|鲁健]]。<ref name="duck " >{{cite web |url=http://digest.scol.com.cn/2006/11/03/20061103329534177486.htm |title=鲁健和郑天亮的“爱情直播” |publisher=文摘周报 |accessdate=2006年 |work= |language =}}</ref>

2001年，郑天亮大学毕业后被分配到中国中央电视台工作，上任之前，他们20多位新手得先去[[内蒙古|内蒙古]][[赤峰|赤峰]]电视台实习半年，他们的队长正是鲁健。在实习的半年里，鲁健渐渐地爱上了郑天亮，实习结束之后就向郑天亮吐露了心曲，但是遭到郑天亮的拒绝说她不会找同行。<ref name="duck " />

2002年，鲁健邀约郑天亮一起出去吃饭，地点在[[北京|北京]][[炸酱面|炸酱面]]餐馆，因为郑天亮身体不适，于是鲁健劝说她喝汤暖暖身子骨，就是鲁健的这一关怀的举动，俘虏了她的心，她说鲁健“实在”、“不玩花样”、“踏实”。<ref name="duck " />

2004年3月6日，鲁健和郑天亮在[[北京|北京]]登记结婚。<ref name="duck " />
