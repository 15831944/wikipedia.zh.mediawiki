'''庙沟后石牌坊'''，又名'''韩岭石牌坊'''，是中国[[浙江省|浙江省]][[宁波市|宁波市]]一座墓前神道牌坊，位于[[鄞州区|鄞州区]][[东钱湖镇|东钱湖镇]]韩岭村庙沟后山南麓。牌坊为仿木结构，形制与宋《[[营造法式|营造法式]]》可相互印证，推断建造年代为[[南宋|南宋]]至[[元代|元代]]<ref name="杨新平">{{cite journal|title=宁波东钱湖庙沟后牌坊探析|author=杨新平|journal=建筑史|issue=01|year=2003|page=112-124}}</ref>，与同属[[鄞州区|鄞州区]]的[[横省石牌坊|横省石牌坊]]形制相近，同为中国现存最早的石[[牌坊|牌坊]]之一。2001年，庙沟后石牌坊被列为[[第五批全国重点文物保护单位|第五批全国重点文物保护单位]]<ref name="nbwb-detail">{{cite web|url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=766&type=2|title=庙沟后、横省石牌坊|accessdate=2018-12-02|author=鄞州区文管办|date=2018-01-08|publisher=宁波市文物保护管理所|archive-url=https://archive.is/20130502123344/http://www.nbwb.net/pd_wwbh/info.aspx?Id=766&type=2|archive-date=2013-05-02|dead-url=yes}}</ref>。

== 方位 ==
庙沟后石牌坊位于东钱湖镇韩岭村东北，蝙蝠山南坡庙沟后<ref>{{cite web|url=http://yz.cnnb.com.cn/system/2013/02/25/010514103.shtml|title=东钱湖韩岭村|accessdate=2018-12-02|author=吴海霞|date=2013-02-25|publisher=鄞州新闻网|archive-url=https://web.archive.org/web/20181202045600/http://yz.cnnb.com.cn/system/2013/02/25/010514103.shtml|archive-date=2018-12-02|dead-url=no}}</ref>。韩岭村位于[[东钱湖|东钱湖]]东南，南有[[215省道_(浙江)|215省道]]穿过。东钱湖周边有多座南宋时期的墓葬，形成[[东钱湖墓葬群|东钱湖墓葬群]]，存留大量墓道建筑和石刻遗迹。牌坊坐东朝西，所在的山坡上有若干疑似人工修筑的台地，且周边有数块条石，说明该牌坊极有可能是墓道牌坊<ref name="杨新平" />。

== 形制 ==
庙沟后石牌坊面宽3.29米，高6.16米，为双柱仿木结构牌坊，采用[[鄞县|鄞县]]西部（今[[海曙区|海曙区]]）所产的“梅园石”砌筑。牌坊为单檐[[歇山顶|歇山顶]]，檐角飞翘上有蹲兽，山面设有[[博风板|博风板]]并有如意形悬鱼。屋面刻有[[筒瓦|筒瓦]]，[[瓦当|瓦当]]刻有兽面纹，滴水为重唇板瓦。檐下有斗拱，其中补间辅作二朵，柱头辅作两朵。第一跳为偷心造，第二、三跳为计心造，分别为华拱和令拱，偷心、计心结合的形式为斗拱发展中的过渡形式。转角辅作和柱头辅作形制相近，角下昂下无华拱，而采用上昂式斜撑，这种上昂为南方特有的构件，现存很少。立柱与阑额组成的开间宽3.29米，高3.28米，立柱为正方形，边长44厘米，柱下无基座，下方变小，直接插入土中。阑额与斗拱间有普柏枋，和阑额均伸出柱外。牌坊构件上无文字，且传世文献中无记载，因而其确切建造年代及墓主信息均不可考<ref name="杨新平" />。

庙沟后石牌坊的结构追求与木结构一致，二柱一楼形制与《[[清明上河圖|清明上河圖]]》、《[[平江图|平江图]]》等宋画中出现的二柱一楼木牌坊形制类似，而这一形制到明代数量逐渐减少。因而庙沟后石牌坊为木牌坊向石牌坊转化阶段的实例<ref name="杨新平" />。

== 保护 ==
{{external media|width=250px|image1=[http://www.nbwb.net/UploadFile/images/2018012216393938.JPG 庙沟后石牌坊保护区划图]，原始内容[https://web.archive.org/web/20181202045240/http://www.nbwb.net/UploadFile/images/2018012216393938.jpg 存档]于2018年12月02日}}
庙沟后石牌坊于1986年5月列为鄞县文物保护单位，当时认定牌坊建造于明代嘉靖年间<ref name="shizhi-transport">{{cite book|title=宁波市志|publisher=中华书局|year=1995|isbn=978-7-101-01507-2|chapter=第四十一卷 文物古迹}}</ref>{{rp|2486}}。1997年8月，庙沟后石牌坊被列入[[浙江省文物保护单位|浙江省文物保护单位]]<ref>{{cite book|title=宁波年鉴（1998）|publisher=中华书局|year=2000|isbn=7-101-02505-6|editor=宁波年鉴编辑部}}</ref>{{rp|459}}。2001年6月25日，庙沟后石牌坊与鄞州区五乡镇新发现的南宋牌坊[[横省石牌坊|横省石牌坊]]一起被列入[[第五批全国重点文物保护单位|第五批全国重点文物保护单位]]，年代认定为宋元时期<ref name="nbwb-detail" />。牌坊的保护范围被确定为牌坊本体向东12.5米，向西8米，向南10米，向北11米<ref>{{cite web|url=http://www.nbwb.net/info_bhqh.aspx?id=41674|title=庙沟后石牌坊：保护区划|accessdate=2018-12-02|date=2018-01-22|publisher=宁波市文物保护管理所|archive-url=https://web.archive.org/web/20181202045600/http://www.nbwb.net/info_bhqh.aspx?id=41674|archive-date=2018-12-02|dead-url=no}}</ref>。自2016年起<sub>，</sub>[[宁波东钱湖旅游度假区|东钱湖管委会]]社会事务管理局对庙沟后石牌坊开展变形监测<ref>{{cite web|url=http://www.zjww.gov.cn/culture/2017-12-15/1216031628.shtml|title=东钱湖对庙沟后石牌坊展开第三次变形监测|accessdate=2018-12-02|author=宁波文化网|date=2017-12-15|publisher=浙江省文物局|archive-url=https://web.archive.org/web/20181202160143/http://www.zjww.gov.cn/culture/2017-12-15/1216031628.shtml|archive-date=2018-12-02|dead-url=no}}</ref>。
