'''庙街'''（{{lang-mnc|ᠮᡳᠶᠣᠣ ᡤᠠᡧᠠᠨ|v=miyoo gašan}}），俄語全稱為'''阿穆爾河畔尼古拉耶夫斯克'''（{{lang-ru|'''Никола́евск-на-Аму́ре'''}}），中國称'''庙街'''（又作廟爾、廟屯），现为[[俄羅斯|俄羅斯]]城市，是[[遠東聯邦管區|遠東聯邦管區]][[哈巴羅夫斯克邊疆區|哈巴羅夫斯克邊疆區]][[庙街县|庙街县]]行政中心，邻近[[黑龍江|黑龍江]][[出海口|出海口]]，[[俄罗斯帝国|俄罗斯]]於1850年建市，2014年的人口統計有20,774人。本市距離[[黑龍江|黑龍江]]口約80[[公里|公里]]，距離[[伯力|伯力]]約977公里、[[阿穆尔河畔共青城|共青城]]火車站約582公里。

== 歷史 ==
该地在[[元朝|元朝]]被征服與管轄，屬[[遼陽等處行中書省|遼陽行省]]，并在[[黑龙江|黑龙江]]下游设置了[[征东元帅府|征东元帅府]]，其治所[[奴儿干城|奴儿干城]]即与庙-{}-街相去不远。到[[明朝|明朝]]永樂年间明廷設置[[奴兒干都指揮使司|奴兒干都指揮使司]]管轄奴兒干地區的軍事建制機構，其駐地亦为元朝征东元帅府旧址。1413年（永樂十一年）明朝宦官亦失哈在巡視奴兒干都指揮使司所轄地區時，在奴兒干都司官衙所在地附近原有觀音堂的基礎上所修建的一座供奉觀世音菩薩的佛寺，名為[[永寧寺_(黑龍江)|永寧寺]]。[[明朝|明朝]]滅亡後屬[[清朝|清朝]]所擁有，[[满語|满語]]名为[[廟街|廟街]]。1689年[[清朝|清朝]]與[[俄罗斯沙皇国|沙皇俄国]]簽定的《[[尼布楚條約|尼布楚條約]]》中也明確規定廟街屬[[清朝|清朝]]。不過，由于清政府在1860年代前曾长期对内地[[汉族|汉族]]封禁东北，禁止[[汉人|汉人]]进入[[东北|东北]]开发。而同时因为[[满族|满族]][[清军入关|入关]]的缘故，[[东北|东北]][[满族|满族]]和[[通古斯|通古斯]][[土著|土著]]的人口非常的低，导致[[东北|东北]][[边防|边防]]十分空虚。随着[[清朝|清朝]]的衰落，[[清政府|清政府]]在[[康乾盛世|康乾时代]]定下的[[巡视东北边境|巡视东北边境]]的制度，也逐渐荒废。

[[第一次鸦片战争|第一次鸦片战争]]爆发后，[[俄罗斯帝国|俄罗斯帝国]]开始重新向[[东北|东北]]渗透。1850年8月13日，俄國人[[根納季·伊萬諾維奇·涅韋爾斯科伊|根納季·涅韋爾斯科伊]]在當地開設尼古拉耶夫斯基驛站，為[[俄罗斯帝国|俄国]]至當地的武装移民提供[[郵政|郵政]]服務。其後，[[俄罗斯帝国|俄國]]在當地大幅[[殖民|殖民]]，並在當地興建[[教堂|教堂]]、[[警察局|警察局]]、[[賭場|賭場]]、[[金庫|金庫]]等各種[[基礎建設|基礎建設]]。在1856年，[[俄罗斯帝国|俄国]]单方面成立[[濱海邊疆州|濱海邊疆州]]，此城就被俄国承認其城鎮的地位，並改為現在的俄語名稱。[[第二次鴉片戰爭|第二次鴉片戰爭]]之後，根據1858年所簽訂的《[[璦琿條約|璦琿條約]]》及1860年所簽訂的《[[中俄北京條約|中俄北京條約]]》而割讓予[[俄罗斯帝国|俄罗斯帝国]]。

直到1880年之前，它一直都是[[濱海邊疆州|濱海邊疆州]]的行政中心，直到[[濱海邊疆州|濱海邊疆州]]一分為二為止，其行政中心的地位分別被[[海參崴|海參崴]]及[[伯力|伯力]]所取代。直到現在，它亦一直都是[[俄羅斯联邦|俄羅斯联邦]][[遠東區|遠東區]]的重要[[港口|港口]]。1920年，[[廟街|廟街]]發生了「[[廟街事件|廟街事件]]」，然後曾遭到大面积的[[轟炸|轟炸]]，[[郵電局|郵電局]]及各種[[通訊設備|通訊設備]]都被完全摧毀。[[第二次世界大戰|第二次世界大戰]]之後經過重建，最終回復昔日的光彩。

== 名稱 ==
该城市的[[中国|中国]]古称为“庙街”，在[[滿語|滿語]]當中miyoo是[[廟|廟]]的意思，gašan為[[村莊|村莊]]的意思，因此地曾有明朝所設立的[[永寧寺_(黑龍江)|永寧寺]]因而得名。划归[[俄罗斯帝国|俄国]]后，其[[俄語|俄語]]名音译为“尼古拉耶夫斯克”，用來纪念俄國皇帝[[尼古拉一世_(俄國)|尼古拉一世]]。[[中华人民共和国|中华人民共和国]]出版的地图将其标记为“尼古拉耶夫斯克（庙街）”，但是各地华人仍然普遍使用“庙街”作为名称。

== 廟街事件 ==
1920年，此地發生[[日军|日军]]和[[日本裔移民|日本裔移民]]，[[俄國白军|俄國白军]]和[[俄羅斯裔居民|俄羅斯裔居民]]被[[無政府主義者|無政府主義者]][[雅科夫·伊萬諾維奇·特里亞皮岑|雅科夫·特里亞皮岑]]的軍隊杀害的事件，史稱[[廟街事件|廟街事件]]。
