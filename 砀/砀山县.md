'''-{zh-cn:{{ruby-py|砀|dàng}}; zh-tw:{{ruby-zy|碭|ㄉㄤˋ}};zh-hans:砀;zh-hant:碭;}-山县'''位于[[中华人民共和国|中国]][[安徽省|安徽省]]最北端，是[[宿州市|宿州市]]下辖的一个[[县|县]]。2007年初被列为安徽省直管县试点，享有省辖市经济管理权限和部分社会事业管理权限，是长三角地区的西北门户。

== 历史沿革 ==
[[战国|战国]]时属[[宋国|宋国]]，為砀邑。後楚滅宋，改属楚。

[[秦始皇|秦始皇]]二十二年（公元前225年），灭[[魏国|魏国]]，置[[砀郡|砀郡]]（治砀县，縣城在今[[河南|河南]][[商丘|商丘]][[永城市|永城市]]芒山西麓），为[[三十六郡|三十六郡]]之一，在今碭山县境设下邑县。碭县东南有砀山（今属[[永城市|永城市]]），故名。

[[劉邦|汉王]]元年（公元前206年）四月，今县境仍设下邑县（治今砀山县城东1公里之毛李庄），属[[项羽|项羽]][[西楚|西楚]]国（都[[彭城|彭城]]，今[[徐州市|徐州市]]）砀郡（仍治故砀城）。[[汉高祖|汉高祖]]五年（公元前202年）正月，砀郡改属[[彭越|彭越]]的[[梁國_(漢朝)|梁国]]（治今毛李庄）。六年春，砀郡归汉。十一年三月丙寅，砀郡改属刘恢梁国（都[[睢阳|睢阳]]，今[[商丘|商丘]]市區南）。[[呂后|高后]]七年（前181年）二月，改属吕产的吕国。八年八月，改属梁国。後九月，砀郡又归汉。文帝前元二年（前178年）二月乙卯，砀郡改属刘楫梁国。十二年，梁国封給劉武。元封五年（前106年）四月，梁国改属[[豫州刺史部|豫州刺史部]]。

[[新莽|新莽]]时（公元9年），改梁国为陈定郡,治所砀县改为「节砀县」，改下邑县为「下治县」，仍属豫州刺史部。

[[东汉|东汉]]章帝建初四年（公元79年），恢復砀郡（仍治砀故城），下邑县改名[[夏邑縣|夏邑縣]]（仍治下邑故城），仍属豫州（治[[谯|谯]]，今[[安徽省|安徽省]][[亳州|亳州]]市区）。同年，梁国迁都下邑故城，夏邑县属之。

[[曹魏|曹魏]]延康元年（公元220年)十月，魏代汉，今县境仍设夏邑县（仍治今毛李庄），為[[豫州_(漢朝)|豫州]]（初治谯，後多次移治）[[梁郡|梁郡]]之郡治。青龙四年（236年），夏邑县兼梁国国都，後又廢梁國為[[梁郡|梁郡]]。

[[西晋|西晋]]武帝泰始元年（265年），復置梁国，砀县并入下邑县。

[[南朝宋|南朝宋]]时，复置砀县。[[南齐|南齐]]時（479年）废。

[[北魏|北魏]][[孝昌|孝昌]]二年（526年）复置砀郡，轄砀县、安陽縣。

[[北齐|北齐]]（555年）废砀郡、砀县，併入安阳县。

[[隋|隋]][[开皇|开皇]]十八年（598年），改安阳县为砀山县。大业六年（610年），割[[彭城縣|彭城縣]]、[[雎陽縣|雎陽縣]]之地置[[永城县|永城县]]，芒砀山以北为砀山县辖区。

[[唐朝|唐朝]]，砀山县属[[宋州|宋州]]（雎陽郡）。光化二年（899年），朱全忠以砀山縣、[[单父縣|单父縣]]、[[虞城縣|虞城縣]]、[[曹州|曹州]]之[[成武縣|成武縣]]表置[[辉州|辉州]]。

[[後唐|後唐]]，砀山县属[[单州|单州]]。

[[宋朝|宋朝]]，属单州。

[[金朝|金朝]][[兴定|兴定]]元年（1217年），砀山县改属[[归德府|归德府]]（府城在今河南省商丘市）。兴定五年（1222年），永城县升为[[永州_(金朝)|永州]]，辖下邑、砀山、郏三县，[[芒砀山|芒砀山]]一带又属砀山县辖 。[[宋理宗|宋理宗]]绍定年间（1228—1233年）砀山县城为洪水荡没，迁治保安镇。

[[元宪宗|元宪宗]]七年（1257年），复置碭山县，还旧地，属[[东平路|东平路]]。[[至元|至元]]二年（1265年），降永州为永城县，芒砀山一带再属永城县。是年，砀山县并入单父县，三年，复置砀山县，属[[濟州_(後周)|济州]]，八年，属[[济宁路|济宁路]]。

[[明朝|明朝]][[洪武|洪武]]四年（1371年），砀山县随[[臨濠府|臨濠府]][[徐州_(三國至明朝)|徐州]]改属[[中书省|中书省]]。六年九月，改[[临濠府|临濠府]]为[[中立府|中立府]]（治中立县，由[[鍾離縣|鍾離縣]]改來）。七年八月，中立府改为[[凤阳府|凤阳府]]（治[[临淮县|临淮县]]，由中立县改來）。八年十月，凤阳府設[[凤阳县|凤阳县]]（府城即今[[府城鎮_(鳳陽縣)|府城镇]]）。十三年正月，凤阳府[[南直隸|直隶六部]]。十四年十一月，砀山县改属[[徐州直隶州|徐州直隶州]]。
[[永乐|永乐]]元年（1403年），徐州屬[[南直隶|南直隶]]。
[[嘉靖|嘉靖]]四十一年（1562年）水患，迁砀山县治于小神集（今砀山县城关镇东南陈寨北门外小神集）。四十四年，复旧治（今毛李庄）。
[[萬曆|萬曆]]二十六年（1598年）在旧城西1公里之秦家堂築新縣城（今城关镇）。萬曆二十八年竣工，为夯土砖堞城垣。

[[清朝|清朝]][[雍正|雍正]]八年（1730年）升徐州为[[徐州府|徐州府]]，砀山县属徐州府。

[[中華民國|中華民國]]元年属[[江蘇省_(中華民國)|江苏省]][[徐海道|徐海道]]；後属江苏省銅山行政督察区；民国二十四年属江苏省行政督察第九区所辖。

1949年元月，属[[中原行政区|中原行政区]][[商丘专署|商丘专署]]。1949年10月，属[[皖北人民行政公署|皖北人民行政公署]][[宿县专区|宿县专区]]。
1950年2月，砀山县[[民主政府|民主政府]]改称砀山县[[人民政府|人民政府]]（1954年，地方人民政府改稱人民委員會）。
1952年11月15日，砀山县划归江苏省。
1953年3月6日，复属[[江苏省|江苏省]][[徐州专区|徐州专区]]（驻[[徐州市|徐州市]]）。
1955年2月21日，将砀山县划入安徽省[[宿县专区|宿县专区]]。
1956年1月12日，撤销宿县专区，砀山县划属[[蚌埠专区|蚌埠专区]]。
1959年3月22日，撤销[[萧县|萧县]]、砀山县，合併為萧砀县，属蚌埠专区。
1961年4月7日，萧砀县改属宿县专区；12月15日，撤销萧砀县，恢复砀山县，仍属宿县专区。
1968年8月，砀山县[[革命委員會_(文革)|革命委员会]]成立。
1971年，宿縣專區改為宿縣地區。
1981年12月30日，砀山县革命委员会改称砀山县人民政府，至今未变。
2000年4月，宿縣地區改設地級[[宿州市|宿州市]]，砀山县属宿州市。

== 地理 ==
砀山县位于安徽省最北端，地处苏、鲁、豫、皖四省七县(市)交界处。境内几乎全为平原。县城以北一二十公里处为[[宿州黄河故道|黄河故道]]，沿河多为沙土土质，适合酥梨种植。

==民族==
砀山县民族构成主要由汉族、回族构成、其他少数民族较少。回族虽然在人口结构中占据少数部分，但对砀山民风有着不同程度的影响。回族主要为聚居，主要居住地为砀山县城的南关、曹庄镇的回民许庄等地。

== 特产 ==
砀山素有“梨都”之称，以盛产酥梨闻名于世，近百万亩连片果园年产各类水果30亿斤。砀山酥梨年产量15亿斤左右，占全国梨总产量的八分之一，系全国水果生产10强县之一，是全国水果加工第一大县。2010年申请世界吉尼斯记录——基尼斯中国种植梨树规模最大的县。

== 饮食 ==
砀山 饣它汤（也写作sha汤）、木炭烧饼和豆浆粥是当地的特色小吃。 饣它汤味道香醇，为早点中的重要角色，其制作方法很少被人们了解。一般 饣它汤中主料为羊肉丝，小麦仁，辅料有葱，姜，胡椒面，小茴香，大茴香，丁香，花椒，桂皮等。枣糕亦很有名，当年，胡耀邦视察砀山，品尝后很是赞赏。木炭烧饼是砀山籍游子的最想，有些在外地工作的砀山人经常通过长途客车捎带，除自己食用，也赠同乡解馋。
