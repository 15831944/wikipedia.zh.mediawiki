'''敬一丹'''{{BD|1955年|4月27日|||}}，出生于[[黑龙江省|黑龙江省]][[哈尔滨市|哈尔滨市]]，毕业于[[北京广播学院|北京广播学院]]（今[[中国传媒大学|中国传媒大学]]），[[中国中央电视台|中国中央电视台]]著名[[节目主持人|节目主持人]]，[[中华全国妇女联合会|中华全国妇女联合会]]会员<ref name="mix">{{cite web|url=http://www.chinaqking.com/rm/2008/19994.html|title=敬一丹：女人到八十还是女人|accessdate=2008年|work=|publisher=中国期刊网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20131122120140/http://www.chinaqking.com/rm/2008/19994.html|archivedate=2013-11-22}}</ref>。敬一丹也是中国广播电视学会播音主持委员会副理事长、中国电视艺术家协会主持人委员会副会长、北京大学电视研究中心特聘研究员、中国传媒大学兼职教授、第九、十届[[全国政协|全国政协]]委员以及第十一届[[全国人大|全国人大]]江西地区代表。

== 简介 ==
1955年，敬一丹出生于[[黑龙江省|黑龙江省]][[哈尔滨市|哈尔滨市]]。[[文革|文革]]时期的[[上山下乡运动|上山下乡运动]]，她在农村当了[[知青|知青]]。1976年[[文革|文革]]结束后，敬一丹回到[[黑龙江|黑龙江]]，考入[[北京广播学院|北京广播学院]]。1979年在[[北京广播学院|北京广播学院]]毕业后，被分配到家乡[[黑龙江人民广播电台|黑龙江人民广播电台]]当播音员。1983年，敬一丹考取了[[北京广播学院|北京广播学院]]的研究生，毕业后留在学校里执教。1988年，敬一丹出任[[中央电视台|中央电视台]][[记者|记者]]和[[节目主持人|节目主持人]]<ref name="mix" />，先后在经济部（现[[中国中央电视台财经频道|财经频道]]节目中心）、新闻评论部（现隶属于央视新闻中心）工作<ref>{{cite web |url=http://news.sina.com.cn/c/2007-02-12/175812299009.shtml|title=央视主持人敬一丹简介(图) |date= 2007年2月12日|accessdate= 2015年4月21日|publisher=新浪网 |language = zh-hans}}</ref>。

2015年4月30日，年满60岁的敬一丹在主持完个人最后一期《焦点访谈》之后，正式退休<ref>{{cite web |title=60岁敬一丹最后一次主持画面曝光 今日正式退休
|url=http://ent.qq.com/a/20150501/010222.htm|date=2015-05-01|accessdate=2015-05-01|publisher=腾讯 |language =zh-hans}}</ref>。

== 爱情婚姻 ==
敬一丹的丈夫[[王梓木|王梓木]]是她在考场上认识的，当时敬一丹考取了[[北京广播学院|北京广播学院]]，王梓木考取了[[清华大学|清华大学]]经济系。两人于1985年10月结婚。敬一丹曾这样形容自己的丈夫：“宽容和自信结合到同一个男人身上的时候，这个男人就称得上是一个完美的男人、一个很有魅力的男人了。”<ref name="ojs">{{cite web|url= http://www.sznews.com/culture/content/2008-10/18/content_3315053_4.htm|title= 敬一丹考场上认识老公幸福走过20多年|accessdate= 2008年|work= |publisher= 深圳新闻网|language= zh-hans|deadurl= yes|archiveurl= https://web.archive.org/web/20150924112719/http://www.sznews.com/culture/content/2008-10/18/content_3315053_4.htm|archivedate= 2015年9月24日|df= }}</ref>王梓木于1994年担任[[国家经贸委|国家经贸委]]综合司副司长。1996年辞去公职后，创立[[华泰保险|华泰保险]]并担任董事长<ref>{{cite web |url=http://culture.people.com.cn/GB/106905/17550087.html|title=揭敬一丹老公亿万身家背景 曝漂亮女儿美照 |date=2012年3月31日|accessdate= 2015年4月21日|publisher=人民网 |language = zh-hans}}</ref>。

敬一丹与王梓木养有一女，名叫“王尔晴”，高中毕业后被父母送到国外深造，并以优异成绩完成学业。2011年与来自[[苏格兰|苏格兰]]的小伙Nick结婚<ref>{{cite web |url=http://ent.ifeng.com/a/20150123/42219605_0.shtml#p=1|title=敬一丹28岁女儿与洋丈夫恩爱照曝光 |date=2015年1月23日|accessdate= 2015年4月21日|publisher=人民网 |language = zh-hans}}</ref>。

== 奖项 ==
敬一丹连续三届获得中国电视节目主持人[[金话筒|金话筒]]奖。<ref>{{cite web|url=http://nvxing.xooob.com/nr/200811/357708_1039252.html |title=连续获得3届“全国十佳电视节目主持人金话筒奖的敬一丹 |accessdate=2008年 |work= |publisher=新榜网 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20140222171619/http://nvxing.xooob.com/nr/200811/357708_1039252.html |archivedate=2014-02-22 }}</ref>

== 评价 ==
* 敬一丹自谓：“面对真正的坏人，我斗不过，面对好人无心犯了错，我又不忍心。就我个人而言，从小就比别人笨，也比别人慢，上学时从来没有什么出类拔萃的风光。我身上传统的东西很多，因袭的负担也很重。我可能缺少了某种新锐之气，但我确实已经很难一惊一乍的了。”<ref name="smex " >{{cite web |url=http://www.people.com.cn/GB/14677/21965/22071/2358269.html |title=敬一丹：不以“媚”取人 |accessdate= 2004年|work=|publisher=人民网 |language = zh-hans}}</ref>
* [[杨澜|杨澜]]：“一般女主持人，都难脱一个‘媚’字，而在我认识的众多女主持人中，敬一丹是不以‘媚’取人的。”<ref name="smex " />
* [[北京广播学院|北京广播学院]]教授：“敬一丹对生活中的许多事情都感受得很浓很深，可在面对观众时，她又能把事情表达得很平和很清淡。”<ref name="smex " />
* [[白岩松|白岩松]]《[[痛并快乐着|痛并快乐着]]》：“我很庆幸，能与敬一丹、[[水均益|水均益]]、[[方宏进|方宏进]]、[[崔永元|崔永元]]成为搭档……我们习惯称敬一丹为敬大姐，这不仅是因为她在我们5个人中年岁最大，还因为她的确是大姐的样子。”，<ref name=" mix" />“敬大姐的心很软，即使是批评性的报道她也是商量性的口吻，而在采访需要提一些尖锐性问题时，敬大姐总是狠不下心来，这使得敬大姐在我们这个经常流露出‘尖酸刻薄’的团队中多少显得有点与众不同。”<ref name=" mix" />
