'''纳闽'''（[[马来语|马来语]]：'''{{lang|ms|Wilayah Persekutuan Labuan}}'''；{{lang-en|'''Federal Territory of Labuan'''}}；[[爪夷文|爪夷文]]: لابوان；簡稱'''{{lang|ms|Labuan}}'''或'''纳闽'''）是[[马来西亚|马来西亚]]东部一个[[聯邦直轄區|聯邦直轄區]]，位于[[沙巴|沙巴]]西南部，是[[南中国海|南中国海]]中的[[岛屿|岛屿]]。纳闽在[[马来语|马来语]]是“良港”的意思。纳闽也是一个自由贸易港及旅游胜地。纳闽自古以来是沙巴一部分，但自1984年成為[[聯邦直轄區|聯邦直轄區]]后，纳闽属于马来西亚独自的行政区划，从此受联邦政府直接管辖，不再由沙巴州政府管制。纳闽的首府是[[维多利亚_(马来西亚)|维多利亚]]。

纳闽于1990年成为[[自由港|自由港]]和[[离岸金融中心|國際離岸金融中心]]<ref>{{cite web |url=http://www.pl.gov.my/sejarah.htm |title=存档副本 |accessdate=2008-01-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20071230032919/http://www.pl.gov.my/Sejarah.htm |archivedate=2007-12-30 }}</ref><ref>http://www.hotelreservation.com.my/hotel/labuan/history.htm{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，获[[世界级城市|世界级城市]]评为'''高度自足'''级（High Sufficiency level cities）城市。

== 歷史 ==
納閩原本是[[汶莱|汶莱]]王朝的一部分，但於1846年被汶莱蘇丹割讓給[[英國|英國]]，並將之改名為“維多利亞島”（{{lang|en|Victoria Island}}）。於1848年成為英國皇家殖民地，並於1890年1月1日被英國合併為英屬[[北婆羅洲|北婆羅洲]]的一部分。1906年10月30日，被合併為[[海峽殖民地|海峽殖民地]]的一部分。1942年，[[太平洋戰爭|太平洋戰爭]]爆發，[[馬來半島|馬來半島]]、[[新加坡|新加坡]]、北婆羅洲和[[砂拉越|砂拉越]]被[[日軍|日軍]]所佔領，納閩被改名為“前田島”。1945年，日本投降，英國殖民地政府取回統治權，重新受[[海峽殖民地|海峽殖民地]]政府管治。1946年7月15日，納閩重歸英屬北婆羅洲。於1963年7月15日脫離英國的統治組成[[馬來西亞|馬來西亞]]，但歸[[沙巴|沙巴]]管治。於1984年4月16日正式成為[[聯邦直轄區|聯邦直轄區]]，並於1990年成為[[自由港|自由港]]及[[离岸金融中心|國際離岸金融中心]]<ref>{{cite web |url=http://www.pl.gov.my/sejarah.htm |title=存档副本 |accessdate=2008-01-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20071230032919/http://www.pl.gov.my/Sejarah.htm |archivedate=2007-12-30 }}</ref><ref>{{cite web |url=http://www.hotelreservation.com.my/hotel/labuan/history.htm |title=存档副本 |accessdate=2008-01-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20080330104319/http://www.hotelreservation.com.my/hotel/labuan/history.htm |archivedate=2008-03-30 }}</ref>。1992年3月，马来西亚政府正式成立纳闽发展局。主要任务包括：促进、加强纳闽作为国际离岸金融中心、旅游点和免税区的地位<ref>{{Cite book|title=国际离岸金融市场|last=王雪冰主编|first=|publisher=北京：[[中国金融出版社|中国金融出版社]]|year=1995.12|isbn=7-5049-1533-5|location=|pages=158-159}}</ref>。

=== 郵政 ===
[[File:Stamp_Labuan_1885_2c.jpg|thumb]]
納閩的郵政局從1864年開始投入服務。一些郵件使用當時英屬印度或英屬香港的郵票，不過通常都是用於私人的郵件。當時的郵件都需要經由新加坡再轉遞到別的地方。1867年時納閩的郵政局使用海峽殖民地的郵票，直到1879年5月才出現本土發行的郵票。

== 人口 ==
根据2000年的统计，纳闽的人口是77700人，其中土族占62594人、华人占10014人、及其他种族占14100人。
