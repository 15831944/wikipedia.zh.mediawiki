'''纳克什班迪教团军'''（{{lang-ar|جيش رجال الطريقة النقشبندية}} ''Jaysh Rijāl aṭ-Ṭarīqa an-Naqshabandiya''），是一支活跃在[[伊拉克|伊拉克]]西部和中部的[[恐怖组织|恐怖组织]]。

== 历史 ==
该恐怖组织大约成立于2003年底，前身是[[萨达姆|萨达姆]]的共和国卫队，由于[[美国|美国]]解散了萨达姆的军队并将[[逊尼派|逊尼派]]排挤出局，伊拉克前军队就组织恐怖团伙展开报复活动，连续制造多起袭击。

== 发展 ==
2014年初，[[伊拉克和黎凡特伊斯兰国|伊拉克和黎凡特伊斯兰国]]联合[[纳克什班迪教团|纳克什班迪教团]]军里应外合，夺取了伊拉克西部的[[安巴尔省|安巴尔省]]。并在六月和七月先后夺取[[摩苏尔|摩苏尔]]和[[萨拉赫丁省|萨拉赫丁省]]，如今在与[[什叶派|什叶派]]为主的伊拉克政府军和[[库尔德|库尔德]]武装展开拉锯战。战术上经常是伊拉克和黎凡特伊斯兰国在前面攻城，纳克什班迪教团军就在周围负责夺取重要的交通要道和桥梁。
