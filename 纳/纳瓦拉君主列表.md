== 波旁王朝 ==
{{main|波旁王朝}}
* 1572年－1610年：[[亨利四世_(法兰西)|恩里克三世]]
* 1610年－1620年：[[路易十三|路易斯二世]] 

1589年纳瓦拉国王恩里克三世继承了[[法国|法国]]王位，成为法国国王[[亨利四世_(法兰西)|亨利四世]]，領有兩國國王的[[寶座|寶座]]，1620年[[下纳瓦拉|下纳瓦拉]]正式并入法国，但各代法国国王继续使用'''纳瓦拉国王'''的称号直至1791年[[法国大革命|法国大革命]]推翻了波旁王室。在1814年—1830年[[波旁王朝复辟|波旁王朝复辟]]期间，[[路易十八|路易十八]]和[[查理十世_(法兰西)|查理十世]]又再次使用了这个头衔。

== 现時自称的纳瓦拉国王 ==
# 卡拉布里亞公爵[[佩德羅_(兩西西里)|佩德羅]]是纳瓦拉王位按非萨利克法的继承人。他的祖母{{link-en|波旁-帕尔马的阿利希亚|Infanta Alicia, Duchess of Calabria}}公主（卡拉布里亚公爵夫人、西班牙公主）是法国王孫[[亨利五世_(法兰西)|亨利]]的姐姐的后裔；她对纳瓦拉王位的要求在其子、佩德羅的父親卡洛斯的官方网站证实。
# 巴黎伯爵、法兰西公爵[[奧爾良的亨利_(巴黎伯爵)|亨利亲王]]，法兰西和纳瓦拉的奥尔良派王位继承人。
# 安茹公爵[[路易·阿方索_(安茹公爵)|路易斯·阿尔方索·德·波旁亲王]]，西班牙国王[[胡安·卡洛斯一世|胡安·卡洛斯一世]]的伯父塞尔戈维亚公爵[[海梅_(塞戈維亞公爵)|海梅]]的孙子，也是保王党拥立的法国王位继承人。
# 西班牙国王[[費利佩六世|費利佩六世]]，他已经把纳瓦拉国王的称号加入头衔之中，毕竟纳瓦拉的大部分都已被他的先祖[[斐迪南二世_(阿拉贡)|阿拉贡的费尔南多二世]]所征服，现正由他继承。
