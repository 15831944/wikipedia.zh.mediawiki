《'''纳粹疯狼'''》（{{lang-it|''La Bestia in calore''}}）是一部1977年意大利[[剝削電影|剝削電影]]，由[[路易吉·巴采拉|路易吉·巴采拉]]执导、编剧和剪辑。<ref name="archivio" />

== 情节 ==
一位美丽而邪恶的[[党卫队|党卫队]]高级女军官/医生（马哈·马加尔饰）创造了一种变异人类（半人半兽）。野兽性欲极强，她用它来折磨女性囚犯，而[[纳粹主义|纳粹分子]]在旁观看。这个[[侏儒症|侏儒]]野兽饮食中包含大量[[春药|春药]]。除了野兽外，囚犯（无论男女）还被剥光衣服，被迫忍受集体[[审讯|审讯]]、[[觸電|电击]]、蓄意[[强奸|强奸]]和殴打。与此同时，当地村庄的一群意大利[[游击战|游击队员]]正准备袭击纳粹集中营。

== 制作 ==
大多数（如果不是全部）袭击德国人的游击队员战斗场景都来自之前由巴采拉执导的战争电影。{{sfn|Shipka|2011|p=168}}

== 风格 ==
《纳粹疯狼》属于20世纪70年代早期开创的[[剝削電影|剝削電影]]的子类型，即[[纳粹剥削电影|纳粹剥削电影]]。{{sfn|Richardson|2012|p=44}}这些电影主要在美国和意大利制作。{{sfn|Richardson|2012|p=44}}在这些电影中，尤其是意大利制作的电影，如《[[纳粹疯淫史2之间谍军妓|纳粹疯淫史2之间谍军妓]]》、《[[第三帝国最后的纵欲|第三帝国最后的纵欲]]》和《纳粹疯狼》经常含有[[硬調色情|硬調色情]]镜头，但作家迈克尔·D·理查森称其“更应归类为恐怖电影，因为有着过多的、生动的暴力镜头”。{{sfn|Richardson|2012|p=44}}{{sfn|Richardson|2012|p=45}}

== 发行 ==
该片于1977年6月10日通过了意大利审查机构的审查，与巴采拉的另一部纳粹主题电影《纳粹女魔王》一起于1977年上映，在美国上映时经过了大量剪辑。这部电影曾以《党卫队地狱营》、《党卫队实验第二部分》和《党卫队末日的恐怖实验》的名字发行。{{sfn|Paul|2005|p=316}}
