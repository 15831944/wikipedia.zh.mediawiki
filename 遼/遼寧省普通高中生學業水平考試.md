'''遼寧省普通高中生學業水平考試'''為[[遼寧省教育廳|遼寧省教育廳]]舉辦的用來測測驗遼寧省內普通高中學生學術水平的測驗，以衡量在校學生是否達到[[普通高中|普通高中]]畢業標準，其前身為「會考」。自2012年開始實行，每年定期舉辦一次。該考試將作為[[中國大學|中國大學]]錄取遼寧省考生的參考條件之一。<ref>{{cite news|author=朱晓梅|title=河北：高中“学业水平考试”将代替“会考”|url=http://www.ln.chinanews.com/html/2011-02-22/195589.html|accessdate=2012-04-03|newspaper=遼寧新聞網|date=2011-02-22|deadurl=yes|archiveurl=https://web.archive.org/web/20140309190705/http://www.ln.chinanews.com/html/2011-02-22/195589.html|archivedate=2014-03-09}}</ref><ref>{{cite web|title=辽宁省高中会考将取消 学业水平考试与高考挂钩|url=http://www.sylgu.com/gaokao/93.html|accessdate=2012-04-03}}</ref><ref name="辽宁省普通高中学生学业水平考试实施方案">{{cite web|title=辽宁省普通高中学生学业水平考试实施方案|url=http://www.lnen.cn/secpage.htm?actionType=viewjsp&id=13172|publisher=遼寧省教育廳|accessdate=2012-04-03|deadurl=yes|archiveurl=https://web.archive.org/web/20140309191259/http://www.lnen.cn/secpage.htm?actionType=viewjsp&id=13172|archivedate=2014-03-09}}</ref> 

测试分為文化科目、通用技術、藝術、綜合實踐和實驗操作。

== 功能 ==
該考試將考生參考各科目以等級形式進行呈現，分為A（100~90分，優秀）、B（89~75，良好）、C（74~60分，及格）、D（59分以下，不及格）四个等级，考察科目成績不合格的考生，允許下一年再參加考試，但等級不會超過B級<ref name="辽宁省普通高中学生学业水平考试实施方案" />。

遼寧省教育廳稱，自2013年起將會把遼寧省考生的考試成績，在中國大學進行[[普通高等学校招生全国统一考试|高考]]錄取時提交給考生所報考學校，成績運用由所報考學校自行決定<ref>{{cite news|title=高中学业水平考试下月底进行 与高考“软”挂钩|url=http://ln.sina.com.cn/news/lnyw/2012-02-25/16432.html|accessdate=2012-04-03|newspaper=沈阳晚报|date=2012-02-25}}</ref>。

此外，無高中學歷的各種在職從業人員、中專、技校畢業生或其他社會人員等亦可通過該考試獲得高中學歷。

== 考試內容 ==
文化科目分為：<ref name="辽宁省普通高中学生学业水平考试实施方案" />
#語文（包括漢語文和民族語文）
#數學
#外語（包括英語、日語、俄語）
#思想政治
#歷史
#地理
#物理
#化學
#生物
#信息技術

此外還將考察通用技術、藝術（包括音樂、美術）、綜合實踐活動（包括研究性學習、社區實踐、社會實踐）和理化生實驗操作。

== 首次考試 ==
遼寧省於2012年3月30日至4月1日舉行首次考試，遼寧省人民政府官方稱此次考試難度小於高考，試題易、中、难比例為7∶2∶1，但考試嚴格程度與高考相同<ref>{{cite news|title=首届高中学业水平考试明日开考|url=http://www.ln.gov.cn/zfxx/jrln/wzxw/201203/t20120329_842292.html|accessdate=2012-04-03|newspaper=遼寧省人民政府|date=2012-03-29}}</ref>，据統計[[遼寧省|遼寧省]]省會城市[[瀋陽市|瀋陽市]]約有3.7萬考生參加此次考試<ref>{{cite news|author=王晓波|title=今起沈阳近3.7万考生参加学业水平测试|url=http://news.lnd.com.cn/htm/2012-03/30/content_2235924.htm|accessdate=2012-04-03|newspaper=時代商報|date=2012-03-30|archive-url=https://web.archive.org/web/20140808210709/http://news.lnd.com.cn/htm/2012-03/30/content_2235924.htm#|archive-date=2014-08-08|dead-url=yes}}</ref>，部分家長特地請假3天陪同孩子考試。由於此次考試不分文理，許多理科考生表示對文綜擔心，但考試後考生普遍表示文綜「还挺简单的」。另據考生透露，此次考試語文科目作文為[[史蒂夫·乔布斯|喬布斯]]說的話。<ref>{{cite news|author=张倍博|title=辽宁首次学业水平考试昨开考部分家长请3天假陪考|url=http://fushun.nen.com.cn/74874555713191936/20120331/2602105.shtml|accessdate=2012-04-03|newspaper=撫順新聞網|date=2012-03-31}}</ref>
