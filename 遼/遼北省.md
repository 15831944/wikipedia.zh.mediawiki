'''遼北省'''，簡稱「洮」，是[[中華民國|中華民國]]時期新置的12[[省|省級]]行政區之一，為[[東北九省|東北九省]]之一。

== 名稱由來 ==
由於位置在[[遼寧省|遼寧省]]北部，故名遼北。

== 地理 ==
全省位於東北[[松遼平原|松遼平原]]西部，轄境相當於今[[遼寧省|遼寧省]]彰武、法庫、康平、昌圖，[[吉林省|吉林省]]四平、梨樹、雙遼，[[內蒙古|內蒙古]][[通遼市|通遼市]]大部、[[興安盟|興安盟]]的小部，面積12萬1624.17平方公里，人口490萬4399人。全境以[[松遼平原|松遼平原]]為主體，西北部為[[大興安嶺|大興安嶺]]南端，是[[蒙古人|蒙古人]]的[[牧區|牧區]]。

周圍鄰省分別為北界[[興安省|興安省]]、[[嫩江省|嫩江省]]，東界[[吉林省_(中華民國)|吉林省]]、[[安東省|安東省]]，南界[[遼寧省_(中華民國)|遼寧省]]，西界[[熱河省|熱河省]]、[[察哈爾省|察哈爾省]]。

== 歷史沿革 ==
民國三十四年（1945年）日軍投降，國民政府於9月上旬，在重慶宣布任命劉翰東為遼北省政府主席，主持在四平市組建遼北省政府，並在重慶成立遼北省政府籌備處開始策劃接收工作。而[[中共|中共]]在[[蘇聯|蘇聯]]的協助下，率先進入東北，在[[四平|四平]]成立'''遼北省'''自治政府，民國三十五年（1946年）1月10日，遼北省政府在四平正式成立並開始辦公。隨著蘇聯軍隊撤離東北，國軍和東北民主聯軍在東北地區展開激烈爭奪。3月13日，蘇聯軍隊從四平撤出。17日，中共調集軍隊1萬餘人攻克四平市，擊斃500餘人，俘獲遼北省省主席[[劉翰東|劉翰東]]及其各廳、處官職人員150餘人和警察總隊官兵3000餘人，遼北省政府頃刻瓦解。25日，中共釋放劉翰東、[[徐鼐|徐鼐]]等遼北省政府要員。劉翰東等人於27日飛赴錦州後，即令警務處長[[王泰興|王泰興]]率領部分人員赴瀋陽建立遼北省政府辦事處。

4月至5月，國軍主力[[國民革命軍新一軍|新一軍]]（[[孫立人|孫立人]]部）、[[國民革命軍新六軍|新六軍]]（[[廖耀湘|廖耀湘]]部）、[[國民革命軍七十一軍|七十一軍]]（[[陳明仁|陳明仁]]部）共約十萬之眾，北進開原、昌圖，與解放軍在四平街，長春均爆發了激烈戰鬥，5月下旬，國軍在四平重創[[東北民主聯軍|東北民主聯軍]]部隊。19日，國軍控制四平。翌日，遼北省政府進駐四平。國軍從遼北全線追擊解放軍到[[松花江|松花江]]畔逼近[[哈爾濱|哈爾濱]]。同一時間在美國特使[[馬歇爾|馬歇爾]]的壓迫下，國民政府於6月6日下達了第二次停戰令，東北戰事稍息。

民國三十六年（1947年）5月，解放軍發起夏季攻勢，先後攻克懷德、公主嶺、昌圖、開原、山城鎮、東豐、梅河口、西安、西豐、清源、雙山、鄭家屯、伊通、雙陽、樺甸、輝南等地。6月，解放軍除在吉林省的永吉、長春開火外，並將主力圍攻由[[陳明仁|陳明仁]]率領的七十一軍部隊防守的四平，在國軍奮戰拼命到底及中央及時抽調[[國民革命軍新六軍|新六軍]]、[[國民革命軍九十三軍|九三]]、[[國民革命軍五十三軍|五三]]等軍赴援，解除四平之圍，史稱「[[四平街會戰|四平街會戰]]」。四平戰役後，國軍在東北完全處於劣勢。1947年7月19日，國民政府派劉瀚東為遼北選舉事務所主任委員；國民政府同時派定其為[[國民大會代表|國民大會代表]]，[[立法院|立法院]][[立法委員|立法委員]]。<ref name="民國史大事記">{{cite book|editor=李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編|title=《中華民國史大事記》|location=[[北京|北京]]|publisher=[[中華書局|中華書局]]|date=2011年7月}}</ref>{{rp|8385-8386}}民國三十七年（1948年）2月，解放軍以強大兵力第四次圍攻四平，經過血戰於3月13日攻克四平，殲滅國軍1萬9千多人。中華民國在遼北省的統治結束。

== 行政區劃 ==
=== 省會 ===
民國三十六年（1947年）6月，國民政府公布東北九省的行政區域，遼北省省會為遼源縣（今吉林省雙遼市），因遼源為中共所控，因此省會實際位於四平。

=== 縣、市、設治局 ===
民國三十六年（1947年）6月，國民政府公布東北九省的行政區域，全省轄有1市、18縣、6旗，由於省境較小，不設行政督察區。由於本省因抗戰結束後，大部分地區己被東北民主聯軍控制，故國民政府公佈的行政區劃大多未得到執行。新設的四平市，在滿州國時期已經存在，沿革過程，詳見[[四平省|四平省]]。
