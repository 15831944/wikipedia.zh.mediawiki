'''譚凱邦'''（{{lang-en|'''Tam Hoi Pong，Roy'''}}，{{bd|1980年|5月17日}}），現任[[香港|香港]][[荃灣區議會|荃灣區議會]][[馬灣|馬灣]]選區議員。他是[[香港|香港]][[環保觸覺|環保觸覺]]創辦人、現任總幹事及前主席及[[新民主同盟|新民主同盟]]成員。

他雖然強調[[香港本土化運動|務實本土]]路線，但仍不時就環保議題發表較激進的評論，亦支持減少[[單程證|單程證]]配額因此常於香港各大媒體曝光。

==簡歷==
譚凱邦曾於[[華英中學|華英中學]]任教一年期合約的綜合科學。他先前亦任教於新界屯門安定邨的[[順德聯誼總會梁銶琚中學|順德聯誼總會梁銶琚中學]]<ref>{{cite web|language=zh-hk|publisher=[[順德聯誼總會梁銶琚中學|順德聯誼總會梁銶琚中學]]|title=教職員|url=http://www.lkkc.edu.hk/info/index4.html|author=|date=|accessdate=2009-08-24|deadurl=yes|archiveurl=https://web.archive.org/web/20090414180314/http://www.lkkc.edu.hk/info/index4.html|archivedate=2009-04-14}}</ref>，香港著名社運界人士、[[香港中文大學學生會|香港中文大學學生會]]會長[[張秀賢|張秀賢]]就讀該校時曾是他的學生<ref>[http://www.mingpaovan.com/htm/News/20120927/HK-gfj2_er.htm?m=0  明報新聞網海外版 - 社運師徒] {{webarchive|url=https://web.archive.org/web/20140715012756/http://www.mingpaovan.com/htm/News/20120927/HK-gfj2_er.htm?m=0 |date=2014-07-15 }}</ref>，於2012年離職。<ref>http://hk.news.yahoo.com/%E8%AD%9A%E5%87%B1%E9%82%A6%E6%9C%89%E9%8C%A2%E7%84%A1%E8%81%B2%E5%94%94%E9%81%B8%E7%AB%8B%E6%9C%83-212807747.html</ref>他是該中學環保學會負責人。
他分別教授[[電腦|電腦]]、[[科學|綜合科學]]、[[生物|生物]]及[[通識|通識]]四個科目。他亦曾於2006年任教[[仁愛堂田家炳中學|仁愛堂田家炳中學]]<ref>{{cite web|language=zh-hk|publisher=仁愛堂田家炳中學|title=已離職教師|url=http://www.yottkp.edu.hk/yottkp_fck/image/ex_teachers.pdf|author=|date=|accessdate=2009-08-24}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>及2004年於[[佛教大雄中學|佛教大雄中學]]任職實習老師。譚凱邦另經營網上售賣環保產品。

==關注環保議題==
早年就讀[[中華傳道會安柱中學|中華傳道會安柱中學]]，於2003年畢業於[[香港中文大學|香港中文大學]]環境科學系，並加入環保學會「綠色天地」，推動大學生的環保意識。其後與一班志同道合的同學創辦中文大學第一份環保報章「綠色報」，並擔任第一至三期的總編輯。<ref>{{cite web |language=zh-hk |publisher= |title=綠色報創刊號 |url=http://logic.itsc.cuhk.edu.hk/~z044125/985800000/page0_0.html |author= |date=2001-03-29 |accessdate=2009-08-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20090819014938/http://logic.itsc.cuhk.edu.hk/~z044125/985800000/page0_0.html |archivedate=2009-08-19 }}</ref>2002年創辦環境教育團體[[自然足印|自然足印]]，於2002年至2004年出任主席<ref>{{cite web |language=zh-hk |publisher= |title=自然足印網頁 |url=http://www.footprint.org.hk/ |author= |date= |accessdate=2009-08-24 }}</ref>，致力於中、小學推行環境教育。2004年組成綠色團體[[環保觸覺|環保觸覺]]<ref>{{cite web|language=zh-hk|publisher=|title=環保觸覺網頁|url=http://www.greensense.org.hk |author= |date= |accessdate=2009-08-24}}</ref>，關注各方面的環保問題。

譚凱邦關注多項[[環保|環保]]議題，並不時因此出現在報章或電視[[新聞|新聞]]，議題包括豐樂圍、[[屏風樓|屏風樓]]<ref>{{cite web |language=zh-hk |publisher=雅虎新聞 |title=油街「地王」建築物或降高限 |url= http://hk.news.yahoo.com/article/090621/3/ct6d.html |author=明報 |date=2009-06-21 |accessdate=2009-08-24}}</ref>、新[[立法會綜合大樓|立法會綜合大樓]]擴建<ref>{{cite web |language=zh-hk |publisher=新浪網 |title=新立會大樓擴建環團斥荒謬 |url=http://www2.news.sina.com.hk/cgi-bin/nw/show.cgi/2/1/1/1217987/1.html |author=東方日報 |date=2009-08-01 |accessdate=2009-08-24 }}</ref>、酷熱[[天氣|天氣]]下的[[節約能源|節約能源]]措施<ref>{{cite web |language=zh-hk |publisher=香港商報 |title=熱爆黃大仙高溫37度 |url=http://www.cnwnc.com/content/2009-08/03/content_2356546.htm |author= |date=2009-08-03 |accessdate=2009-08-24 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、開[[冷氣|冷氣]][[打邊爐|打邊爐]]<ref>{{cite web |language=zh-hk |publisher=新浪網 |title=環團轟火鍋店開冷氣打邊爐加劇暖化 |url= http://news.sina.com.hk/cgi-bin/nw/show.cgi/3/1/1/1234214/1.html |author=明報 |date=2009-08-17 |accessdate=2009-08-24}}</ref>、樓盤售樓書<ref>{{cite web |language=zh-hk |publisher= 雅虎新聞 |title=售樓書涉美化 議員促規管 |url=http://hk.news.yahoo.com/article/090712/4/d5lq.html |author=明報 |date=2009-07-13 |accessdate=2009-08-24 }}</ref>、[[廣深港高速鐵路|廣深港高速鐵路]]規劃<ref>{{cite web |language=zh-hk |publisher= 雅虎新聞 |title=環保團體反對廣深港鐵路規劃 |url=http://hk.news.yahoo.com/article/090628/3/cx9x.html |author=星島日報 |date=2009-06-28 |accessdate=2009-08-24 }}</ref>、[[公共空間|公共空間]]<ref>{{cite web |language=zh-hk |publisher=新浪網 |title=公共天橋變酒家等位區 |url=http://news.sina.com.hk/cgi-bin/nw/show.cgi/2/1/1/1240732/1.html |author=明報 |date=2009-08-23 |accessdate=2009-08-24 }}</ref>等。

2007年，譚凱邦所屬的[[環保觸覺|環保觸覺]]在[[麥當勞|麥當勞]]推出「無飲管日」後，2009年8月更呼籲政府實施每件[[港幣|港幣]]五角的「即棄餐具徵費」等議題。<ref>{{cite news |language=zh-hk |author= |coauthors= |url= http://hk.news.yahoo.com/article/090823/3/dumh.html |title=環團籲即棄餐具徵費 |work=星島日報 |publisher=雅虎新聞 |pages= |date=2009-08-24 |accessdate=2009-08-27}}</ref>由於其創辦的[[環保觸覺|環保觸覺]]建議要求他人跟從其環保路向，時事評論員[[蔡子強|蔡子強]]曾喻其為「環保[[塔利班|塔利班]]」。

譚凱邦經常於上課時播放有關環保的影片給其學生觀看，有家長投訴影響其子女學習進度。<ref>{{cite web |language=zh-hk |publisher=雅虎新聞 |title=譚凱邦﹕回應蔡子強先生 ——〈小心莫變成環保塔利班〉一文 |url=http://hk.news.yahoo.com/article/090817/4/dr3t.html |author=明報 |date=2009-08-18 |accessdate=2009-08-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20090821204202/http://hk.news.yahoo.com/article/090817/4/dr3t.html |archivedate=2009-08-21 }}</ref>譚凱邦曾於社交網站呼籲抵制世界自然基金會及呼籲停止捐款。  <ref>http://property.mpfinance.com/cfm/pa3.cfm?File=20130520/paa01/gha2.txt</ref>譚凱邦希望港鐵不要設扶手電梯，他提到，醫學會建議每天應步行8,000步，消耗150卡路里，而在短短步程加設電梯不利市民健康，也耗用電力。 <ref>[http://hk.epochtimes.com/8/6/27/84276.htm]</ref>

譚凱邦曾呼籲暫停[[屯門至赤鱲角連接路|屯門至赤鱲角連接路]]工程，[[中九龍幹線|中九龍幹線]]押後上馬 <ref>[http://the-sun.on.cc/cnt/news/20141203/00407_060.html 貪曾濫基建超支640億] 2014-12-03</ref>

譚凱邦曾於社交網站批評「冰桶挑戰」、「水炮演習」<ref>[http://the-sun.on.cc/cnt/news/20140909/00408_003.html 政壇：譚凱邦暗寸「水炮演習」浪費] 2014-09-09</ref>

==從政生涯==
譚凱邦於2008年7月24日報名參選[[2008年香港立法會選舉|2008年香港立法會選舉]]，在[[九龍西|九龍西]]出選，他的口號是「綠色香港 來自你一票」（Vote for a green Hong Kong）。[[民主黨_(香港)|民主黨]]新界東支部主席[[陳竟明|陳竟明]]、[[公民黨_(香港)|公民黨]]表示支持譚參選。<ref>[http://www1.appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20080623&sec_id=4104&subsec_id=15333&art_id=11264941&cat_id=6962706&coln_id=6796044 蘋果日報 譚凱邦獲民主黨核心力撐 公民黨歡迎新人上場] {{webarchive|url=https://web.archive.org/web/20080626081218/http://www1.appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20080623&sec_id=4104&subsec_id=15333&art_id=11264941&cat_id=6962706&coln_id=6796044 |date=2008-06-26 }}</ref>他在6月底籌獲20萬元選舉經費，全期目標為60至100萬<ref>九龍西選區的競選經費上限是250萬</ref>。被問及雙普選議題時，譚自稱「不是泛民主派也不是保守派」 ，主張2012年普選特首，2016年普選立法會。<ref>[http://www.metrohk.com.hk/news.php?startDate=23062008&newscat=1&newsid=80685 都市日報 環保人士譚凱邦非陪跑冀爭第5席 ]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>最後低票落選，僅得1,603票，得票率只為0.78%。
譚凱邦道出他的參選原因：「過去香港過度經濟掛帥，引致多項環保及社會問題，包括用電量過去10年大增4成、空氣污染、屏風樓問題、工時長休息不足、子女對菲傭時間多於父母、陪伴子女時間不足、缺乏家庭教育、手機引致焦慮、並失去私人空間、市民睡眠不足、連嬰兒也不例外」<ref>[http://roytam.wordpress.com/]</ref>

他曾以[[環保觸覺|環保觸覺]]身份參與[[2008年香港立法會選舉|2008年香港立法會選舉]]但以低票落選。曾有意參選[[2012年香港立法會選舉|2012年香港立法會選舉]]（[[新界西|新界西]]），後來以聲帶勞損，擔心病情惡化為理由放棄參選。

==抗議香港被中國化==
2013年8月底，譚凱邦與[[香港立法會|立法會]]議員[[范國威|范國威]]（[[新民主同盟|新民主同盟]]）、[[毛孟靜|毛孟靜]]（[[公民黨_(香港)|公民黨]]）、[[張超雄|張超雄]]（[[工黨_(香港)|工黨]]）、保衛香港自由聯盟發言人[[韓連山|韓連山]]及多名社運人士在[[Facebook|Facebook]]發起「[https://www.facebook.com/events/574631195936103/ 抗融合．拒赤化．反盲搶地 一人$100換特首 還香港人一個家]」行動，由數百位港人集資於9月3日在《[[明報|明報]]》及《[[都市日報|都市日報]]》刊登「反梁」廣告，批評梁振英漠視港人利益，讓自由行陸客「肆虐」香港，破壞香港的文明和秩序，香港人基本生活需要不斷被剝削，已讓港人忍無可忍，表明「換特首是出路」，促請中國政府停止干涉香港內部事務；並在[[臺灣|臺灣]]《[[自由時報|自由時報]]》刊登「香港面對嚴重中國化，請台灣引以為鑑」廣告，讓[[臺灣人|臺灣人]]明白香港引入中國新移民及旅客後港人所面對的苦況，不要讓臺灣被中國客攻陷。<ref>[http://hk.apple.nextmedia.com/realtime/news/20130902/51695489 「一人$100換特首」明日港台兩地登廣告]，[[蘋果日報_(香港)|蘋果日報 (香港)]]，2013年09月02日</ref><ref>[http://thehousenews.com/politics/%E6%B8%AF%E4%BA%BA%E6%8A%97%E8%9E%8D%E5%90%88%E6%8B%92%E8%B5%A4%E5%8C%96%E8%81%B2%E6%98%8E-%E5%8F%B0%E7%81%A3%E7%99%BB%E5%A0%B1%E7%B1%B2%E4%BB%A5%E6%B8%AF%E7%82%BA%E9%91%91/ 港人抗融合拒赤化聲明 台灣登報籲以港為鑑] {{webarchive|url=https://web.archive.org/web/20130904192235/http://thehousenews.com/politics/%E6%B8%AF%E4%BA%BA%E6%8A%97%E8%9E%8D%E5%90%88%E6%8B%92%E8%B5%A4%E5%8C%96%E8%81%B2%E6%98%8E-%E5%8F%B0%E7%81%A3%E7%99%BB%E5%A0%B1%E7%B1%B2%E4%BB%A5%E6%B8%AF%E7%82%BA%E9%91%91/ |date=2013-09-04 }}，[[主場新聞|主場新聞]]，2013-9-3</ref><ref>[http://www.ettoday.net/news/20130903/265383.htm 「請台灣引以為鑑」　港網友登報抗議陸客、移民肆虐]，[[ETtoday|ETtoday]]，2013年09月3日</ref><ref>[http://www.libertytimes.com.tw/2013/new/sep/4/today-p1.htm?Slots=P 香港中國化 港人盼台灣引以為鑑] {{webarchive|url=https://web.archive.org/web/20130904163040/http://www.libertytimes.com.tw/2013/new/sep/4/today-p1.htm?Slots=P |date=2013-09-04 }}，[[自由時報|自由時報]]，2013-9-4</ref>並要求台灣齊聲要求減少自由行，譚凱邦自己認為廣告成功引起台灣注意，但廣告在台灣登出後數星期，台灣的交通部觀光局表示，最近大陸旅行團及自由行大增，配額不足，未來擬放寬自由行人次<ref>{{Cite web |url=http://news.on.cc/cnt/china/20130924/bkn-20130924180446624-0924_00922_001.html |title=台擬放寬大陸自由行  |access-date=2013-10-01 |archive-url=https://web.archive.org/web/20131004215258/http://news.on.cc/cnt/china/20130924/bkn-20130924180446624-0924_00922_001.html |archive-date=2013-10-04 |dead-url=yes }}</ref>。

2014年2月，譚凱邦用「人口政策關注組」名義與「數百位本土真香港人」每人集資$300登廣告，題為《自由行政策出錯 政府應深切反省 不應諉過「驅蝗行動」》於都市日報及明報刊登廣告批評特首及多位高官不正視「驅蝗行動」遊行者訴求<ref>[https://www.facebook.com/events/473132262812197/#!/events/473132262812197/]</ref>

==支持台獨==
2014年3月，譚凱邦發文: 「香港都想獨了 台灣幹嘛不獨」指出台灣應該獨立，以保存各項優勢，包括民主制度及人文氣息。觀乎香港一國兩制的失敗，請台灣人民以香港為鑑，不應向中國靠攏。香港已禮崩樂壞，台灣應該要有危機意識，拒絕大陸化。<ref>[http://www.taiwancon.com/?p=105416 譚凱邦: 香港都想獨了台灣幹嘛不獨] 2014-03-06 [[自由時報|自由時報]]</ref>
