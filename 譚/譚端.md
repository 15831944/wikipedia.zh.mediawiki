'''譚端'''(1971年-)

== 教育 ==
* [[英國|英國]][[謝菲爾德大學|謝菲爾德大學]]（University of Sheffield）新聞碩士。

== 新聞業 ==
* 記者，作家，播客。

== 文史界 == 
* 2010年參與拍攝兩岸歷史紀錄片《最後島嶼：台灣防衛戰1950-1955》，講述1949年~1955年[[國共內戰|國共內戰]]由大陸延燒至大陸東南沿海島嶼的真實故事。本片榮獲中華民國[[行政院新聞局|行政院新聞局]]補助製作高畫質電視節目，並獲得2012年[[第47屆金鐘獎|第47屆金鐘獎]]最佳文化教育類節目獎。
* 2011年策劃執行[[一江山戰役協會|一江山戰役協會]]主辦之《守望民國的歲月：看見老士官生命故事展》，本案獲得中華民國[[行政院文化建設委員會|行政院文化建設委員會]]補助支持。
* 2011年出版[[口述歷史|口述歷史]]圖書《烽火、離亂、老士官》，本書收錄20位[[中華民國國軍|中華民國國軍]]低階士兵在亂世中輾轉落腳台灣的故事。
* 2014年與[[視納華仁|視納華仁]]（CNEX）合作，在講述中日戰爭期間中國空軍飛行員的的紀錄片電影《冲天》（The Rocking Sky）擔任故事策劃。
* 2015年出版紀實文學《天空的情書》：紀錄片《冲天》電影紀事（天下文化）。

== 文化創意產業 ==
* 2018年底，因景氣長期低迷，結束偵探書屋實體店面，轉入網路經營。
* 偵探書屋目前轉化為播客網站《[https://agathas.ink/ 昨日世界]》
* 創辦臺灣第一家專營[[偵探小說|偵探小說]]的書店「偵探書屋」<ref>{{Cite web|url=http://www.chinatimes.com/newspapers/20140511000298-260115|author=林欣誼|language=zh-tw|title=偵探書屋進駐大稻埕|publisher=[[中國時報|中國時報]]|date=2014-05-11|accessdate=2014-05-11}}</ref>。
*  作品《烽火、離亂、老士官》（2011），《天空的情書》（天下，2015）
*  譯有偵探小說《石猴子》（新星出版社，2008）、魔幻小說《王者之眼》（新星出版社，2010）、合譯《推理寫作祕笈》（馬可孛羅，2019）。
