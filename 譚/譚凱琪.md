'''譚凱琪'''（{{lang-en|'''Zoie Tam Hoi Ki'''}}，{{lang-ja|'''スーイ'''}}，{{bd|1981年|10月10日|catIdx=T譚}}），[[香港|香港]]女歌手、主持及演員，現為[[無綫電視藝員列表|無綫電視經理人合約]]藝員<ref>[http://artiste.tvb.com/main/artistedetail/tamhoiki 譚凱琪 Zoie Tam - TVB藝人資料 - tvb.com]</ref>。

== 簡歷 ==
譚凱琪父母在她唸小學時分開<ref>{{web reference |URL=http://dailynews.sina.com/bg/ent/hktwstar/sinacn/20100615/18011537223.html |title=譚凱琪迎父親節費盡心思 為父親學做壽包 |work=[[新浪網|新浪網]] |date=2010年6月15日}}</ref>，自此便跟母親一起生活<ref name=Zoie>[https://topick.hket.com/article/2234645/%E3%80%8A%E5%A4%A7%E5%B8%A5%E5%93%A5%E3%80%8B%E8%AD%9A%E5%87%B1%E7%90%AA%E5%85%A5%E8%A1%8C18%E5%B9%B4%E6%9B%BE%E6%AD%B7%E9%9B%B6%E6%94%B6%E5%85%A5%E3%80%80%E8%AD%9A%E5%87%B1%E7%90%AA%E4%B8%8D%E8%A6%BA%E8%B5%B0%E5%86%A4%E6%9E%89%E8%B7%AF%EF%BC%9A%E6%A9%9F%E6%9C%83%E4%B8%8D%E6%98%AF%E5%BF%85%E7%84%B6 《大帥哥》譚凱琪入行18年曾歷零收入　譚凱琪不覺走冤枉路：機會不是必然]，[[香港經濟日報|香港經濟日報]]，2018年12月20日</ref>。她早年於[[九龍|九龍]][[觀塘區|觀塘區]]生活，曾就讀[[港澳信義會慕德中學|港澳信義會慕德中學]]（中一至中三）<ref>{{cite news | title=譚凱琪細細粒比校花可愛 | url=http://orientaldaily.on.cc/cnt/entertainment/20110508/00282_038.html | format= | publisher=東方日報 | date=2011-5-8 | accessdate=2019-1-2 }}</ref>，畢業於[[九龍灣|九龍灣]][[威靈頓英文中學|威靈頓英文中學]]（中四至中五；商科），也曾赴[[日本|日本]]留學，亦是日本音樂家[[小室哲哉|小室哲哉]]弟子之一；2000年簽約其公司 [[ROJAM|ROJAM]]，並到[[東京|東京]]受訓4個月，之後推出首張唱片《全力愛》，但後來曝光率越來越少<ref>[http://www.recruit.com.hk/Resources/ArticleDetail.aspx?articleId=25768 譚凱琪　踏實的快樂]，Recruit.com.hk，2010年1月12日</ref>，且嘗試往[[台灣|台灣]]發展，卻受到語言及機會所限，最終選擇回港發展<ref name=Zoie/>。直至2006年，她才開始恢復幕前演出，2007年加盟[[驕陽星藝館|驕陽星藝館]]成為旗下藝人；而2007至2013年間曾是[[香港有線電視|香港有線電視]]經理人合約藝員（正式加入前曾擔任[[有線衛視新知台|有線衛視新知台]]客席主持<ref name=Zoie2>[https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/278549/%E5%A4%A7%E5%B8%A5%E5%93%A5-%E5%B0%88%E8%A8%AA-%E8%AD%9A%E5%87%B1%E7%90%AA%E5%85%A5%E8%A1%8C%E5%BB%BF%E5%B9%B4%E7%B5%82%E6%86%91%E5%A4%A9%E5%AC%8C%E5%BD%88%E8%B5%B7-%E5%8F%AF%E8%83%BD%E4%BB%A5%E5%89%8D%E5%94%94%E5%A4%A0%E5%8A%AA%E5%8A%9B?utm_medium=Social&utm_source=fbpost_link&utm_campaign=entertainment&fbclid=IwAR1shqlKjrEPhS_WbX_ZTVb12CWPdoxvlLNPPPVSMXCez1fHfgYS7zz90tM 【大帥哥．專訪】譚凱琪入行廿年終憑天嬌彈起：可能以前唔夠努力]，香港01，2019年1月4日</ref>），2013年11月則轉投無綫電視<ref>[http://www.menclub.hk/girls/6248 近日爆紅水晶梨 譚凱琪]，[[MenClub|MenClub]]，2016年3月26日</ref><ref name=Zoie2/>。

2016年，譚凱琪在該台劇集《[[EU超時任務|EU超時任務]]》內飾演「李姿婷（水晶梨）」一角獲網民支持<ref>[http://www.bastillepost.com/hongkong/4-%E5%A8%9B%E5%9C%88%E4%BA%8B/1087530-34%E6%AD%B2%E3%80%8C%E6%B0%B4%E6%99%B6%E6%A2%A8%E3%80%8D%E8%AD%9A%E5%87%B1%E7%90%AA%E5%87%BA%E9%81%9316%E5%B9%B4%E7%B5%82%E4%B8%8A%E4%BD%8D 34歲「水晶梨」譚凱琪出道16年終上位]，[[巴士的報|巴士的報]]，2016年4月2日</ref><ref>[http://www.orientalsunday.hk/%e6%9c%80%e6%96%b0%e5%a8%9b%e8%81%9e/%e8%ad%9a%e5%87%b1%e7%90%aa/ 正到動 L《EU》譚凱琪 cute 爆真身]，東方新地，2016年3月31日</ref>。此外，她於同年台慶劇集《[[致命復活|致命復活]]》裡飾演「郭卓詩（Emma）」也備受注目<ref>[https://www.hk01.com/%E5%A8%9B%E6%A8%82/58428/-%E8%87%B4%E5%91%BD%E5%BE%A9%E6%B4%BB-%E8%AD%9A%E5%87%B1%E7%90%AA%E9%B5%AA%E9%B6%89%E5%A6%B9%E6%89%AE%E8%87%AA%E4%BF%A1-%E8%A3%B8%E8%83%8C%E6%8C%91%E6%88%B0%E8%87%AA%E6%88%91-%E5%B9%BE%E5%A4%A7%E9%83%BD%E8%A6%81%E5%81%9A%E8%90%BD%E5%8E%BB 【致命復活】譚凱琪鵪鶉妹扮自信　裸背挑戰自我：幾大都要做落去]，香港01，2016年12月8日</ref>，更首度獲提名《[[TVB_馬來西亞星光薈萃頒獎典禮2016|TVB 馬來西亞星光薈萃頒獎典禮2016]]》「最喜愛 TVB 飛躍進步女藝員」、「最喜愛 TVB 電視角色」，以及《[[萬千星輝頒獎典禮2016|萬千星輝頒獎典禮2016]]》「最佳女配角」。2017年，她分別在《[[不懂撒嬌的女人|不懂撒嬌的女人]]》和《[[踩過界|踩過界]]》中，分別飾演單親媽媽「陳日澄（Annie）」與形體藝術家「魏雨珍」<ref>{{cite web |url=http://orientaldaily.on.cc/cnt/entertainment/20170626/00282_020.html | title=譚凱琪全裸遇鹹豬手 |publisher=[[東方日報_(香港)|東方日報]] |date=2017年6月26日}}</ref>，出色表現使其奪得《[[TVB_馬來西亞星光薈萃頒獎典禮2017|TVB 馬來西亞星光薈萃頒獎典禮2017]]》「最喜愛 TVB 飛躍進步女藝員」殊榮。

2018年，譚凱琪於《[[大帥哥|大帥哥]]》裡首次飾演反派角色 —— 日本間諜「葉天嬌（舞川奈伊）」，演技再次獲觀眾讚賞<ref>[https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/272769/%E5%A4%A7%E5%B8%A5%E5%93%A5-%E5%A4%A9%E5%AC%8C-%E8%AD%9A%E5%87%B1%E7%90%AA%E7%A3%A8%E5%8A%8D18%E5%B9%B4-%E5%85%A5tvb%E7%AD%89%E6%8C%91%E6%88%B0-%E5%B8%8C%E6%9C%9B%E7%B9%BC%E7%BA%8C%E9%80%B2%E6%AD%A5 【大帥哥】「天嬌」譚凱琪磨劍18年　入 TVB 等挑戰：希望繼續進步]，香港01，2018年12月18日</ref><ref>{{cite web |url=https://hk.on.cc/hk/bkn/cnt/entertainment/20181223/bkn-20181223173447982-1223_00862_001.html | title=天嬌挑機大帥哥！譚凱琪轉戲路奸到噴汁 |publisher=東方日報 |date=2018年12月23日}}</ref>。

== 感情生活 ==
譚凱琪出道初期曾與[[丁子高|丁子高]]拍拖，但兩人於2002年分手<ref>[https://hk.lifestyle.appledaily.com/nextplus/magazine/article/20071101/2_10360704/%E5%A4%B1-%E8%B1%AC-%E5%A5%B3-%E5%8F%8B-%E6%95%B8-%E8%87%AD-%E4%B8%81-%E5%AD%90-%E9%AB%98-%E6%83%85-%E5%A0%B4-%E8%80%81-%E5%8D%83-%E5%8D%83-%E5%8D%83-%E5%AC%85 失豬女友數臭丁子高　情場老千千千嬅]，[[蘋果日報_(香港)|蘋果日報]]，2007年11月1日</ref>。2015年，她再跟餐廳少東 Enrico 拍拖<ref>[https://news.mingpao.com/ins/instantnews/web_tc/article/20170131/s00007/1485853502243 【拍拖2年】譚凱琪認戀餐廳少東]，[[明報|明報新聞網]]，2017年1月31日</ref><ref>[http://orientaldaily.on.cc/cnt/entertainment/20180131/00282_009.html 譚凱琪富貴男友錫晒送靚車]，東方日報，2018年1月31日</ref>。

== 軼聞 ==
* 譚凱琪在家中四姊妹裡排行第三。大姊[[谭凯欣|谭凯欣]]曾是廣告模特兒及藝員，二姐[[谭凯韻|谭凯韻]]曾為[[有線新聞|有線新聞]]主播，四妹[[譚凱倫|譚凱倫]]亦為 [[ViuTV|ViuTV]] 藝員，大姊姐夫則是前香港跳舞組合[[風火海|風火海]]成員[[朱永棠|朱永棠]]<ref>[http://news.stheadline.com/dailynews/headline_news_detail_columnist.asp?id=293437&section_name=wtt&kw=289 那些年 —— 譚凱琪尋開心]，[[頭條日報|頭條日報]]，2014年7月14日</ref>。
* 她因拍攝《[[平安谷之詭谷傳說|平安谷之詭谷傳說]]》而與[[陳凱琳|陳凱琳]]、[[龔嘉欣|龔嘉欣]]及[[沈卓盈|沈卓盈]]等自組「少婦聯盟」<ref>[http://hk.on.cc/hk/bkn/cnt/entertainment/20170628/bkn-20170628123051896-0628_00862_001.html 少婦晒腿打友誼波陳凱琳露腰跌膊搶鏡]，[[ON.CC|東網]]，2017年6月28日</ref>。
