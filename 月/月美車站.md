'''月美車站'''位於[[台灣|台灣]][[臺東縣|臺東縣]][[關山鎮_(台灣)|關山鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]（現已廢止）的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[島式月台|島式月台]]一座。

== 利用狀況 ==
*廢止前為招呼站。
*2013年6月27日零時，[[關山車站|關山站]]＝[[瑞和車站|瑞和站]]間截彎取直完成軌道切換後，本站裁撤廢站走入歷史。

== 車站周邊 ==
*[[卑南溪|卑南溪]]
*加能溪
*[[台東縣月眉國民小學|月眉國小]]

== 歷史 ==
*1922年5月10日：開業，初名'''月野驛'''。
*1962年12月15日：因行政區域屬[[關山鎮_(臺灣)|關山鎮]]月眉里，改稱為'''月眉'''。（係與[[保安車站|保安]]、[[菁桐車站|菁桐]]、[[嶺腳車站|嶺腳]]、[[十分車站|十分]]、[[林榮車站|林榮]]（已廢止，計畫復站）、[[南平車站_(臺灣)|南平]]等車站同時改稱）
*1963年5月1日：因與中部糖鐵月眉站同名，改名'''月美'''。
