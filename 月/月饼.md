'''月饼'''，或叫'''中秋饼'''，是[[大中華地區|大中華地區]]、[[越南|越南]]、[[琉球|琉球]]的[[中秋節|中秋節]]食品。月餅最初为形如[[满月|满月]]的饼，外观为[[圆形|圆形]]，后来有[[方形|方形]]或其他形状。月饼外层通常以[[小麦|小麦]]粉为皮，其内有馅，馅料多种多样。

== 簡要 ==
[[中秋节|中秋节]]吃月饼是[[中華传统文化|中華传统文化]]的风俗。中式月饼以[[广式月饼|广式月饼]]、[[潮式月饼|潮式月饼]]、[[苏式月饼|苏式月饼]]、[[京式月饼|京式月饼]]此四大派别最為普遍；此外中國各地尚有各式各样的月饼，如:[[火腿月饼|火腿月饼]]（[[云南|云南]]）、[[椒葱月饼|椒葱月饼]]（[[江西|江西]]）、水晶月饼（[[潮汕|潮汕]]）、[[薄酥月饼|薄酥月饼]]（[[湖南|湖南]]）、冬蓉月饼（[[台山|廣東台山]]）、德懋恭水晶饼（[[陕西|陕西]]）、三白月饼（[[黑龙江|黑龙江]]）、海味月饼（[[山東|山東]]）、鸡丝月饼（[[清真|清真]]式）、酥皮芋泥月餅（[[潮汕|潮汕]]、[[福建|福建]]、[[臺灣|臺灣]]），港台有[[蛋黃酥|蛋黃酥]]和[[綠豆椪|綠豆椪]]（[[臺灣|臺灣]]）、[[冰皮月餅|冰皮月餅]]（[[香港|香港]]）等<ref>[http://www.shiro-bara.com/goods/cake/36 月餅] {{webarchive|url=https://web.archive.org/web/20130921060509/http://www.shiro-bara.com/goods/cake/36 |date=2013-09-21 }}</ref>。

== 溯源 ==
古代月饼被作为祭品于中秋节所食，現已成為中秋节食品和礼品。中秋節吃月餅的習俗出現於[[唐朝|唐朝]]中期，根據《[[洛中見聞|洛中見聞]]》，[[唐僖宗|唐僖宗]]曾在中秋節當日命令御膳房用紅綾將餅賞賜給新科[[進士|進士]]。[[北宋|北宋]]之時，該種餅被稱為「宮餅」，在宮廷內流行，但也流傳到民間，當時俗稱「小餅」和「月團」。[[蘇東坡|蘇東坡]]的《留别廉守》曾經提及：「小餅如嚼月，中有酥和飴」，當中的「小餅」也即是月餅。

「月饼」一词最早见于[[南宋|南宋]][[吴自牧|吴自牧]]《[[梦梁录|梦梁录]]》中，那时的月饼是[[菱花饼|菱花饼]]一類的饼形食物，后来才演变成圆形，寓意团圆美好。

[[明代|明代]][[田汝成|田汝成]]《[[西湖游览記|西湖游览記]]》提及：“八月十五谓之中秋，民间以月饼相馈，取团圆之意。”，可見月餅於當時流行于民间。[[清代|清代]]已有详细记述月饼制作方法的书籍。清[[杨光辅|杨光辅]]写道：“月饼饱装桃肉馅，雪糕甜砌蔗糖霜”。

== 派别 ==
*广式月饼主要流行于华南，因为容易保存和运输，在全中国范围也最为常见。特点是外层皮薄而松软，馅料细滑而种类繁多，表面有印花。
*苏式月饼主要流行于华东地区，外层为千层酥皮，形状为饱满半球形。馅料分甜咸两种，甜可为豆沙百果，咸可为葱油鲜肉。
*潮式月饼主要流行于潮汕、闽南和台湾等闽语区。特点是外层为螺旋状的酥皮，馅料也多为细滑类的芋泥、豆沙或绿茶。
*京式月饼主要流行于中国北方，外形多呈球形或半球形。外层酥松但相对较为硬实。自来红月饼馅料多为干果和果仁辅以麻油甜面，自来白月饼则多为豌豆、枣泥等细腻馅料。

== 品尝与存放 ==
由于月饼是高[[盐|盐]]分、高[[脂肪|脂肪]]和高[[胆固醇|胆固醇]]的食物，即使是冰皮月饼也含有高[[糖|糖]]分，所以食用要适量。患有[[循环系统|循环系统]]、[[消化系统|消化系统]]疾病的人士，更应避免进食过量而加重病情。

传统月饼保质期约为1个月，如果月饼包装已经开启，就要放入4[[摄氏度|摄氏度]]的[[冰箱|冰箱]]冷藏。

冰皮月饼由于未经烘烤，所以由制造、运输到零售点，都要保持冷藏。消费者在购买后，应尽快放入零下18摄氏度的冰箱保存，并应只在进食前才开启月饼的包装袋<ref name="fehd-1">{{cite web|url=http://www.fehd.gov.hk/sc_chi/safefood/food-safety-express/safefood_tips/mooncake_2003.html|title=注意月饼安全 欢渡中秋佳节|last=|first=|work=|publisher=食物环境卫生署|date=2006-10-20|accessdate=2013-09-11}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。在进食前约两小时，可把月饼移到4摄氏度的冰箱（例如双门冰箱的下层），令月饼略为软化以便切开。当冰皮月饼解冻后便要尽快食用，在室温下摆放超过两小时后便应弃置，也不应把已经解冻的冰皮月饼反复冷藏。

== 制作 ==
[[File:Lotus_seed_paste.jpg|thumb]]
[[File:Vietnam_Grilled_moon_cake_2.JPG|thumb]]
傳統月饼的制法，以水油面团或酥油面团摘剂作皮，内包馅成扁圆形生胚，再入模具压使表面呈凸凹花纹，经烘烤成熟后食用。其皮因制酥法多种，有多样酥皮；其馅因用枣泥或五仁、豆沙、松仁、火腿及香料，而成多种不同风味的馅心。

冰皮月饼的外皮主要使用[[糯米|糯米粉]]、[[粘米|粘米粉]]和砂糖，经[[蒸|蒸煮]]后的面团做成，部分冰皮月饼的外皮，会加入[[巧克力|巧克力]]、[[咖啡|咖啡]]和果汁等材料，除了可增添外皮的味道外，还可使外皮产生不同颜色。冰皮月饼的内馅，除了可使用红豆蓉、绿豆蓉外，还可使用水果蓉、奶酪、巧克力做出不同风味的馅料。更有部分冰皮月饼，使用果酱做夹心或使用脆米增加口感。当内馅调制完成后，便使用外皮包裹，并利用[[模具|模具]]压制成型，再放入冰箱冷藏。

因为冰皮月饼不经烘烤，不能使用高温杀灭细菌，而且进食前也无须加热，直接冷吃。所以制作冰皮月饼的厂房，必须保持环境卫生，制作人员也要严格遵守制作食物的卫生程序。由于冰皮月饼对于生产程序的卫生条件有较高要求，除了提高生产程序的自动化，减少人手接触外，还要设立预防性的食物安全管理系统，例如[[危害分析重要管制點|食物安全重点控制系统（HACCP）]]，防止冰皮月饼在制作过程中被污染，或因为温度控制不当，造成微生物能够大量滋生<ref name="cfs-1">{{cite web|url=http://www.cfs.gov.hk/sc_chi/programme/programme_haccp/programme_haccp_mooncakes1.html|title=冰皮月饼制造指引|last=|first=|work=|publisher=食物安全中心|date=2009-11-10|accessdate=2013-09-15}}</ref>。

== 包裝 ==
月饼在古時一般只以[[布|布]]裹，使用月餅罐或紙盒包裝是到了20世紀因為月餅被[[商品化|商品化]]才出現。由於[[衛生|衛生]]的考慮，現代月餅通常都會用透明[[塑膠|塑膠]]袋作獨立包裝，而鐵罐印花亦於60年代迄今取代了戰前的紙盒包裝。

广式月饼的包装通常是四个为一单位，19世纪70年代时，四个月饼叠在一起，用的是[[玉扣纸|玉扣纸]]起成一个圆柱体，通常竖立摆放，称为一筒月饼，外表贴上红色[[蜡纸|蜡纸]]，注明品种。80年代后，采用盒装，初期还是会把蜡纸放在盒内。

冰皮月饼由于未经烘烤，不能利用高温杀灭[[细菌|细菌]]，所以对包装和保存有较高要求<ref name="ZhongHuaWang-1">{{cite web|url=http://food.china.com/news/11116379/20120815/17374641.html|title=冰皮月餅安全隱患多|last=|first=|work=|publisher=中华网|date=2012-08-15|accessdate=2013-09-11|deadurl=yes|archiveurl=https://web.archive.org/web/20170112080637/http://food.china.com/news/11116379/20120815/17374641.html|archivedate=2017年1月12日|df=}}</ref>。冰皮月饼需要保持冷藏，防止微生物大量滋生。大部分冰皮月饼都会使用[[塑料袋|塑料袋]]单独或两小个一起包装，消费者可在食用时才按食用分量，个别开启包装袋。很多冰皮月饼的包装袋内，还会加入一包[[干燥剂|干燥剂]]，用以减慢袋内空气的[[湿度|湿度]]上升，达到防止微生物大量滋生作用。

由于近年来出现很多昂贵的月饼盒包装的月饼，有些还附加各种高级[[礼品|礼品]]，被一些人用于送礼甚至[[行贿|行贿]]，中国政府在2005年出台法律，规定从2006年起，月饼的包装成本不得超过月饼价值的25%<ref>中华人民共和国国家标准GB19855－2005月饼，2005年9月2日发布，2006年6月1日实施，中华人民共和国国家质量监督检验检疫总局、中国国家标准化管理委员会发布</ref>。

另一方面，香港有環保組織則關注月餅包裝過多的問題，其中[[香港地球之友|香港地球之友]]便發起行動表揚少包裝的月餅品牌的「嫦娥行動」<ref>[http://www.foe.org.hk/welcome/gettc.asp?id_path=1,%207,%2026,%202937 重點項目：常哦行動]</ref>。亦有人關注到月餅售價高昂，反映貧富懸殊加劇，因此有人在[[Facebook|Facebook]]設立群組，對月餅實行抵制<ref>[http://www.facebook.com/group.php?gid=116304431749973 請支持2010年9月22日(農曆八月十五)中秋饑饉無月餅日 FACEBOOK群組]</ref>。

每年一度的中秋节，都会因为月饼的包装物料，而产生大量节日废物，近年已引起公众关注。香港的月饼生产商经过政府部门的协调后，已经同意把月饼的包装，在维持衛生、保質、運輸、外觀和品牌形象等因素下，尽量減少不必要的包裝物料或使用可[[循環再造|循環再造]]的包装物料<ref name="HuanBao-1">{{cite web|url=http://www.epd.gov.hk/epd/tc_chi/environmentinhk/waste/eco_responsibility/gts_packaging.html|title=自願性月餅包裝管理協議|last=|first=|work=|publisher=香港环境保护署|date=2013-03-18|accessdate=2013-09-15}}</ref>。

== 節日後的處理 ==
在大陸由於月餅是時令食品，過了中秋就無人問津，因此於中秋節後，市面上的月餅多會作低價拋售。由于月饼的保质期最长不过30天，所以中秋过后许多大厂家都回收月饼进行销毁，避免出现陈馅月饼，保证品牌质量。於是[[香港|香港]]有人會想出各種方法處理積壓家中的月餅，包括將月餅餡取出用來包湯圓等。每年香港政府都會勸籲市民注意不要進食太多月餅吸收太多糖份影響健康，而市面上亦有廠家生產低糖月餅供糖尿病患者食用。大陆第三军医大学新桥医院营养科主任王建表示：只要是月饼，都会用淀粉做皮，其馅料和皮仍含有很高的糖分和油脂，并非健康食品，故不宜多吃，而糖尿病和高血脂患者最好不要吃月饼<ref>[http://news.xinhuanet.com/food/2013-09/17/c_125398937.htm 专家：“养生月饼”是商家的营销噱头]</ref>。此外，近年每年香港均會有環保團體於中秋過後舉行月餅罐回收活動，以免浪費資源。

在台灣，月餅並不只在中秋時食用（外地傳入之月餅除外），各類台式月餅如蛋黃酥和綠豆椪即使在非中秋時節仍可能買得到，就如一般糕餅一樣。

2001年9月3日，[[中国中央电视台|中央电视台]]曝光[[南京|南京]][[冠生园|冠生园]]食品有限公司回收没卖掉的月饼馅第二年重新使用的做法，而且该公司负责人说全国的月饼行业都这么做，掀起轩然大波，是为“陈馅月饼”事件，使大陆的月饼销售受到一定的影响<ref>[http://www.cctv.com/news/financial/20020322/196.html 南京冠生园：信誉的破产]</ref>。
