'''月球大氣與粉塵環境探測器'''（{{lang-en|'''Lunar Atmosphere and Dust Environment Explorer'''}}，縮寫：'''LADEE'''）是由 [[NASA|NASA]] 轄下{{link-en|艾姆斯研究中心| NASA Ames Research Center}}和[[戈达德太空飞行中心|戈达德太空飞行中心]]主導的[[月球|月球]]探測計畫。該計畫的探測器已於2013年9月6日在{{link-en|瓦羅普斯飛行研究所|Wallops Flight Facility}}[[中大西洋地區太空中心|中大西洋地區太空中心]]以[[米諾陶五號運載火箭|米諾陶五號運載火箭]]發射<ref>{{Cite web |url=http://nssdc.gsfc.nasa.gov/nmc/spacecraftDisplay.do?id=LADEE |title=Lunar Atmosphere and Dust Environment Explorer (LADEE) |publisher=NASA |work=[[美国国家空间科学数据中心|National Space Science Data Center]] Master Catalog}}</ref>。該任務執行的7個月之間 LADEE 環繞月球的赤道，並探測[[月球大氣層|月球大氣層]]的[[散逸层|散逸层]]和周圍的塵埃。探測器上的儀器有塵埃探測器、中性[[質譜儀|質譜儀]]、紫外線－可見光光譜儀，並且該探測器也是[[雷射|雷射]][[通訊衛星|通訊]]{{link-en|技術示範|Technology demonstration}}的終端衛星<ref>{{cite web|url=http://science.nasa.gov/missions/ladee/|title=Missions - LADEE - NASA Science|publisher=NASA}}</ref>。2014年4月18日，地面控制中心操作 LADEE 使之撞毀於[[月球背面|月球背面]]結束任務<ref name="nyt20140418">{{cite news |url=http://www.nytimes.com/2014/04/19/science/space/nasa-lunar-explorer.html |title=With Planned Crash, NASA Lunar Mission Comes to End |work=[[纽约时报|The New York Times]] |first=Kenneth |last=Chang |date=April 18, 2014 |accessdate=April 18, 2014 }}</ref><ref name="abc20140418">{{cite news |url=http://abcnews.go.com/Technology/wireStory/nasas-moon-orbiting-robot-crashes-23375574 |title=NASA's Moon-Orbiting Robot Crashes Down as Planned |work=ABC News |first=Marcia |last=Dunn |date=April 18, 2014 |accessdate=April 18, 2014 }}</ref>

== 任務目的 ==
LADEE 任務目的是要達到以下三個科學目標<ref>{{cite web | url=http://www.spaceref.com/news/viewsr.html?pid=27450 | title=NASA Solicitation: Instruments for LADEE Lunar Mission | publisher=SpaceRef | date=March 25, 2008 | accessdate=July 30, 2011}}</ref>：
* 確認在未來進一步人類活動造成擾動前的月球稀薄大氣層的整體密度、組成和隨時間變化狀況。
* 確認[[阿波羅計畫|阿波羅計畫]]太空人看到的高數十公里處漫射是鈉輝光或塵埃造成。
* 紀錄月球環境中塵埃的影響程度（體積－頻率），作為未來任務工程設計的參考。

== 發射 ==
[[File:Ladee_9_4_13_nasa_edge_0.jpg|thumb]]
LADEE 的探測器已於2013年9月6日11:27 p.m. EDT在美國維吉尼亞州瓦勒普斯岛瓦羅普斯飛行研究所以米諾陶五號運載火箭發射<ref name="propsow">{{cite web |url=http://prod.nais.nasa.gov/eps/eps_data/137365-SOL-001-003.doc |title=Statement of Work - LADEE Spacecraft Propulsion System |publisher=NASA ARC |date=August 27, 2009}}</ref>。這次發射是該發射場首次發射深太空任務。自[[紐約市|紐約市]]到[[維吉尼亞州|維吉尼亞州]]大多數海岸都可見到發射過程<ref name=ladeelaunch.cfm>{{cite web |url = http://moon.nasa.gov/ladeelaunch.cfm |title = LADEE Launch |date = 25 June 2013 |author = NASA |deadurl = yes |archiveurl = https://web.archive.org/web/20130827024701/http://moon.nasa.gov/ladeelaunch.cfm |archivedate = 2013年8月27日 }}</ref>。

當探測器從火箭分離後，在反應輪處偵測到高電流並造成反應輪關閉。目前並沒有故障的跡象，將進行產生高電流原因調查<ref name=sfn-20130907>{{cite news |url=http://www.spaceflightnow.com/minotaur/ladee/130907launch/#.Uisgjn_b0_s |title=Moon mission hits snag after flawless late-night launch |author=Stephen Clark |publisher=Spaceflight Now |date=7 September 2013 |accessdate=7 September 2013}}</ref>。

== 儀器 ==
LADEE 攜帶了三個科學儀器和一個技術示範設備。

科學酬載如下：
* 中性質譜儀：該設備將進行月球大氣散逸层的現場分析試驗。它的設計受到[[好奇號|好奇號]][[火星車|火星車]]的火星樣本分析設備（SAM）影響。
* [[紫外－可见分光光度法|紫外－可見光]]光譜儀（UVS）：將進行月球塵埃和大氣層探測。設計概念繼承[[月球坑观测和传感卫星|月球坑观测和传感卫星]]（LCROSS）的紫外－可見光光譜儀。
* 月球粉塵實驗設備（Lunar Dust EXperiment，LDEX）：可直接量測月球塵埃，技術繼承自[[伽利略号探测器|伽利略号探测器]]、[[尤利西斯号|尤利西斯号]]、[[卡西尼－惠更斯号|卡西尼－惠更斯号]]。

LADEE 將作為[[光通訊|光通訊]]的技術示範平台。該探測器將搭載月球雷射通訊示範設備（Lunar Laser Com Demo，LLCD）進行太空探測器新通訊方式的測試，預期將大幅度增加資料傳輸率。

== 推進系統 ==
LADEE 的推進系統將由軌道控制系統和[[反應控制系統|反應控制系統]]組成。軌道控制系統沿著正向[[笛卡儿坐标系|Z軸]]提供大幅度的速度調整。反應控制系統將在軌道控制系統燃料燃燒時沿著三軸方向進行[[姿態控制|姿態控制]]，並且以[[反應輪|反應輪]]提供動量卸除<ref name="propsow" />。
