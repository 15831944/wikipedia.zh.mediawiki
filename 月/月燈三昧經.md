'''月燈三昧經'''（[[梵語|梵語]]：Candrapradīpasamādhi sūtra），或稱《三摩地王經》，收於[[大正藏|大正藏]]經集部。本經稱佛陀為月光童子宣說「一切諸法體性平等無戲論三昧」，謂得此[[三昧|三昧]]能速得一切法及[[無上正等正覺|無上正等正覺]]。此三昧因月光童子所問而得，故又稱月燈三昧<ref>{{cite web|title=月灯三昧|url=http://cidian.foyuan.net/%D4%C2%B5%C6%C8%FD%C3%C1/}}</ref>。

本經在中國及日本，不甚流行，然於尼泊爾，列為「九法」之一。[[中觀學派|中觀學派]]之大成者[[月稱|月稱]]《入中論》、《中論釋》，及其後繼者[[寂天|寂天]]《大乘集菩薩學論》等書，皆常引用本經，為中觀學派之重要經典<ref name="candra">{{cite web|title=月燈三昧經|url=http://buddhaspace.org/dict/index.php?keyword=%E6%9C%88%E7%87%88%E4%B8%89%E6%98%A7%E7%B6%93}}</ref>。[[噶舉派|噶舉派]]，以本經為[[大手印|大手印]]的根本心要，在西藏亦有極大的影響力<ref>{{cite web|title=月燈·序|url=http://read.goodweb.cn/news/news_view.asp?newsid=7169|deadurl=yes|archiveurl=https://web.archive.org/web/20160917044148/http://read.goodweb.cn/news/news_view.asp?newsid=7169|archivedate=2016-09-17}}</ref>。

== 題解 ==
現存梵本單題為 《Samādhirāja》（三摩地王），而梵文《大乘集菩薩學論》則稱之為《Samādhirāja-Candrapradīpa-Sūtra》（三昧王月燈經）<ref name="candra"/>。

[[三摩地|三摩地]]，意譯為等持、正心行處、心一境性，佛教術語，意指專注於所緣境，而進入心不散亂的狀態，皆可稱為三摩地，因此又可譯為「止」、「定」、「禪定」。

== 傳譯 ==
據《[[歷代三寶紀|歷代三寶紀]]》，《月燈三昧經》，由[[高齊|高齊]]「那連提耶舍」於天保八年（557）於天平寺譯出<ref name="candra"/>。

此經有別生經二種，皆劉宋沙門釋先公所譯。其一題為《月燈三昧經》，又稱《文殊師利菩薩十事行經》，相當於本經第六卷前半。其二亦名月燈三昧經，類似本經第五卷後半。本經現存梵本在一八九六年於加爾各答出版，[[藏文大藏經|藏文大藏經]]亦有本經之譯本<ref name="candra"/>。

== 內容 ==
本經記述佛陀為月光童子說由平等心、救護心、無礙心、無毒心及所修之無量三昧，成就施、戒、忍等法，則得「諸法體性平等無戲論三昧」。又說菩薩應成就善巧，住不放逸，修神通本業，行財施與捨身，具足修學身戒等，以明若具足修學身戒，則可招得相好及一切戒<ref name="candra"/>。
