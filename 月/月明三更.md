《'''月明三更'''》，是一部中国大陆佛教题材电视连续剧，导演张鑫，由[[吕良伟|吕良伟]]、[[宋佳_(1962年出生)|宋佳]]、[[韩月乔|韩月乔]]、[[郑则仕|郑则仕]]主演。<ref>{{cite web |url=http://news.163.com/11/0324/06/6VT094H600014AED.html |title=赣演员和吕良伟联袂演《月明三更》 |accessdate= 2011|publisher=网易 |work= |language=zh-hans}}</ref>本剧根据六祖惠能的生平事迹，以中国禅宗六祖惠能大师的出生、成长、隐居、弘法和叶落归根的过程为背景，通过演绎主人公以顽强的毅力和善良的品格，从一个目不识丁的樵夫，逐步成为一位杰出的思想家、哲学家以及享誉青史的杰出宗师的传奇故事。以一首“菩提本无树，明镜亦非台，本来无一物，何处惹尘埃。”法偈得五祖认可，夜授《金刚经》，密传禅宗衣钵信物，成为第六代祖<ref>{{cite web |url=http://t.m.china.com.cn/convert/c_kpU08J.html?from=timeline&isappinstalled=0&bsh_bid=1512492267&url_type=39&object_type=webpage&pos=1 |title=亚联卫视：吕良伟主演六祖电视剧《月明三更》即将播出 |date=2016-09-14|publisher=中国网 |work= |language=zh-hans}}</ref>。本剧于2011年3月殺青，2016年11月18日在于[[爱奇艺|爱奇艺]]首播。

==剧情==
{{See also|慧能}}

该剧讲述了[[中国佛教|中国佛教]][[禅宗|禅宗]]祖师“慧能”的人生旅程。

==历史差异==
剧组恶搞[[中国佛教|中国佛教]]和[[慧能|慧能]]祖师，不尊重历史和中国佛教，设计大篇幅的情节，使[[鸿利|鸿利]]出演的“蔡玉娇”在剧中爱上“慧能”。<ref>{{cite web |url=http://ent.qq.com/a/20110118/000322.htm |title= 鸿利出演电视剧《月明三更》 剧中暗恋吕良伟|accessdate=2011 |publisher=腾讯网 |work= |language=zh-hans}}</ref>
