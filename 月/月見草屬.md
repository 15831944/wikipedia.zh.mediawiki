'''月見草屬'''（[[学名|学名]]：''{{lang|la|Oenothera}}''）又称'''待宵草'''、'''晚櫻草'''，屬於[[柳葉菜科|柳葉菜科]]的植物。因為本屬植物大多數只在夜晚開花、白天凋謝，所以又名「Evening Primrose」。

== 形态 ==
多年生[[草本|草本]]。[[茎|茎]]直立，被毛；基生[[叶|叶]]丛生，有柄；茎生叶互生，有短柄或无柄，条状披针形，两边有白色短柔毛，边缘有疏锯齿；夏季开黄色花，单生于叶腋，夜开晨闭；圆柱形[[蒴果|蒴果]]，略具四棱。

== 分布 ==
原產於[[美國|美國]]南部、[[墨西哥|墨西哥]]和[[中美洲|中美洲]]地區，[[中国|中国]]东北和山东、江苏等地有栽培，或野生在河畔的沙地上。

== 用途 ==
月见草属植物有可食用的部分。据传[[美丽月见草|美丽月见草]]的幼根可食。
市场上宣称月见草油有治疗癌症的功效，但根据美国癌症协会资料，并无明显证据显示有此功效：“亚麻酸及富含亚麻酸的保健品（如月见草油）均被证实并无预防疾病或保健作用。”
2013年8月，Cochrane 发表的荟萃分析文章发现月见草油和琉璃苣油对湿疹治疗无效。
