'''月光·摩利亞'''是[[日本動漫|日本動漫]]作品《[[ONE_PIECE|ONE PIECE]]》的角色。[[恐怖三桅帆船|恐怖三桅帆船]]的「四怪人」之首，且是前任「[[王下七武海|王下七武海]]」成員之一；「頂點戰爭」結束後，世界政府的高層剝奪其七武海稱號並下令將他抹殺<ref name="ONEPIECE漫畫_581">《ONE PIECE》漫畫第581話。</ref>，但他成功逃脫。

== 設計 ==
* 月光（也譯為「凱克」）原意是[[壁虎|壁虎]]；名字的中央部份（光－摩利ko Mori）原文接近日文的[[蝙蝠|蝙蝠]]，亦可能有取諧音之意。
* 初期摩利亞的人設方面比其他七武海還要多，有巫師、雙重人格的神父，巨漢策士家等各式各樣的設定。

== 人物 ==
部下都對他十分忠心和恭敬，尊稱他為「主人」，較低級的部下甚至不敢直呼他的名字。做任何事情都喜歡倚靠別人來幫他完成，口頭禪是「交給你了」<ref name=comic_people01>{{Cite web |url=http://comic.people.com.cn/GB/238131/238181/16915040.html |title=原王下七武海 月光·莫利亚 |date=2012年1月18日 |work=人民网动漫频道}}</ref>。身形龐大，是時任七武海當中身高最高的（6.92公尺），比「暴君」[[革命軍_(ONE_PIECE)#巴索羅繆·大熊|巴索羅繆·大熊]]還要高。同時也是時任七武海當中最年長的一員。特徵是有如[[火蔥|火蔥]]般的髮型（魯夫稱他為「大火蔥頭」），長相有如惡魔，額側長有兩角，耳朵和牙齒也是尖的，頭至頸部都有類似縫合傷口的交叉狀縫線，穿著蝙蝠狀領子的襯衫和[[哥德式藝術|哥德式藝術]]的服裝，獨特笑聲是「嘿嘻嘻嘻」。

== 劇中表現 ==
童年時代的摩利亞在漫畫單行本第63集SBS專欄露面，當時的他身上配戴一隻匕首，身形並沒有像現在那麼高大，身上也沒有交叉狀的縫線，卻有隨意拆卸玩偶手腳的習慣<small><ref>《ONE PIECE》漫畫單行第63集SBS專欄</ref></small>。

摩利亞也於前傳第零話中短暫登場過，是22年前目睹「海賊王」[[哥爾·D·羅傑|哥爾·D·羅傑]]被處死的圍觀群眾中之一。當時的摩利亞外貌比現在還瘦很多。

過去的摩利亞就像魯夫一樣，是個對自己實力充滿自信並勇於追逐夢想的海賊，但在「新世界」的某次戰鬥中，摩利亞被現時四皇之一的「百獸」海道擊敗並失去所有夥伴<small><ref>《ONE PIECE》漫畫第456話。</ref></small>。

受此打擊以後，摩利亞便堅信「因為活著，才會失去」的道理，因此組織了沒有生命的「殭屍兵團」，繼續幫助他追逐「海賊王」的夢想<small><ref>《ONE PIECE》漫畫第463話。</ref></small>。當他跟草帽海賊團提到此事時，可看出他受過很大刺激且仍心有餘悸，而且他還認為草帽海賊團必然無法在新世界存活。

=== 第一部 ===
在恐怖三桅帆船上與草帽海賊團和[[翻滾海賊團|翻滾海賊團]]交戰，曾憑能力奪走草帽海賊團5名成員（[[魯夫|魯夫]]、[[索隆|索隆]]、[[香吉士_(ONE_PIECE)|香吉士]]、[[布魯克|布魯克]]、[[妮可·羅賓|妮可·羅賓]]）的影子，並將魯夫的影子注入魔人彊屍--奧茲內作為主要戰力。最終被魯夫以「橡膠巨人噴射砲彈」擊敗。

徹底戰敗後被部下赫古巴庫和阿布薩羅姆救出，後看似康復，並接受了[[海軍_(ONE_PIECE)|海軍]]本部的召集跟其餘的「王下七武海」成員進餐。其後再奉召到海軍總部「馬林福特」與其他人在「火拳」艾斯的處刑現場，與[[白鬍子海賊團|白鬍子海賊團]]開戰；期間曾以「角刀影」貫穿小歐斯，以及被「海俠」[[吉貝爾|吉貝爾]]以「鮫肌掌底·鮫瓦·正拳」重擊，後轉與白鬍子海賊團第十隊隊長克利耶爾交戰。

「頂點戰爭」尾聲，被「王下七武海」之一的多佛朗明哥以及和平主義者出手攻擊，起因是世界政府的高層判定摩利亞實力太弱，不足以繼續擔當「王下七武海」，因此剝奪其稱號並命令多佛朗明哥將他抹殺<ref>《ONE PIECE》漫畫第581話。</ref>。

在「大事件」結束以後，「鷹眼」[[喬拉可爾·密佛格|喬拉可爾·密佛格]]和[[ONE_PIECE海賊列表#培羅娜|培羅娜]]從報章中看到「摩利亞戰死」的消息，但「鷹眼」對這段報導的真實性頗為懷疑<ref name="ONEPIECE漫畫_592" />。

之後多佛朗明哥透露，摩利亞似乎用了「影子果實」的能力在他面前憑空消失<ref>《ONE PIECE》漫畫第595話。</ref>，但實際上摩利亞應該是被其部下[[ONE_PIECE海賊列表#阿布薩羅姆|阿布薩羅姆]]使用「透明果實」的能力救走並帶離現場。

=== 第二部 ===
摩利亞為了尋找不久前失聯的部下[[ONE_PIECE海賊列表#阿布薩羅姆|阿布薩羅姆]]而來到海賊島「蜂巢」，結果被從阿布薩羅姆那裡奪取了「[[惡魔果實|透明果實]]」能力的「雨」之矢龍以及使用動物系 幻獸種「[[惡魔果實|犬犬果實 九尾狐形態]]」能力的卡達莉納·戴彭圍攻，此時「黑鬍子」欲招攬摩利亞至自己旗下。

== 能力 ==
=== 惡魔果實 ===
摩利亞是超人系「[[惡魔果實|影子果實]]」能力者，影子人，擁有奪取別人影子的能力<ref name=comic_people01/>，亦能將自己與影子的位置互換。擁有一把特大的[[剪刀|剪刀]]，可將別人的影子剪掉並置入[[屍體|屍體]]中，形成擁有影子原主人能力和個性的[[殭屍|殭屍]]，但[[惡魔果實|惡魔果實]]能力無法隨之轉換到殭屍身上。若被取走影子的人受到陽光照射就會完全消失，故只能活在不見天日的地方。如果影子原主人死了，影子就會隨之消失，殭屍也會失去影子而無法行動。此外，殭屍怕火、鹽、海水。

== 人物關係 == 
[[ONE_PIECE海賊列表#恐怖三桅帆船海賊團|恐怖三桅帆船]]
: 恐怖三桅帆船是位處於魔幻三角地帶（魔の三角地帯）的一個「島嶼」；但根據布魯克敘述，這「島嶼」的真面目其實是一艘世界最大型的海賊船，來自西海。整個「島嶼」被高高的外牆包圍，入口是一道外型像巨大嘴巴的大閘；「島嶼」中央位置有著古堡建築群，住著「四怪人」月光·摩利亞及其手下。古堡上的旗幟有著摩利亞的火蔥頭髮型和一對帶火焰的蝙蝠翅膀的骷髏頭。10年前起，每年有超過100艘船隻在魔幻三角地帶失蹤，實際是被恐怖三桅帆船所捕。四周都被濃霧包圍著，早期是為了防止失去影子的人被陽光消滅。
: 在與草帽海賊團對決期間，恐怖三桅帆船遭到摩利亞、歐斯（特別殭屍）及巴索羅繆·大熊嚴重破壞，多數建築物被摧毀至倒塌；此外，船上所有依賴摩利亞能力存活的殭屍，亦遭到「淨化」或因摩利亞召回影子而滅。在摩利亞及其手下撤離後，該船目前處於廢棄狀態。
: 作者尾田榮一郎表示，在創作本篇的時候，曾參考過很多恐怖電影（包括讀者問及的《愛登士家庭》和提姆·波頓的黏土動畫電影《聖誕夜驚魂》），而且在作畫時也會比平日用上更多黑色，以營造陰森恐怖的感覺。
