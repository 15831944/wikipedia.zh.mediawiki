'''福井市'''（{{jpn|j=福井市|hg=ふくいし|rm=Fukui shi|f=y}}）位於[[日本|日本]]中部，是[[福井縣|福井縣]]最大[[城市|城市]]兼縣廳所在地，亦是[[紡織|紡織]]工業中心。日本[[施行時特例市|施行時特例市]]，位於古代的{{link-jp|足羽郡|足羽郡}}。

== 歷史 ==
自從[[柴田勝家|柴田勝家]]建設[[北之庄城|北之庄城]]後，發展成城下町，江戶時代是親藩大名的領地。而福井市的行政中心位於[[福井城|福井城]]天守一帶。1889年，實行市制，福井市成立，當時人口為38,963人，總面積達4.43平方公里。1896年[[北陸本線|北陸本線]]通車，設置[[福井站|福井站]]。城市經過與附近的町村多次的合併，現代城而的面積較設時的時候為大。最近一次合併為2006年2月1日，{{link-jp|足羽郡|足羽郡}}{{link-jp|美山町|美山町 (福井県)}}、[[丹生郡|丹生郡]]{{link-jp|越廼村|越廼村}}及{{link-jp|清水町 (福井縣)|清水町 (福井県)|清水町}}合併到福井市。

它在[[第二次世界大戰|第二次世界大戰]]中經歷轟炸，1948年又遭強烈[[地震|地震]]破壞，不久市內[[九頭龍川|九頭龍川]]決堤令舊城區損失殆盡。其後市內發表了以不死鳥之鎮宣言重建城市。

== 地理 ==
[[File:Row_of_sakura,_Asuwa_River,_Fukui.jpg|250px]]堤]]
向西面臨[[日本海|日本海]]，市內大多數河流是九龍頭川支流。市東是全市[[海拔|海拔]]最高的{{link-jp|飯降山|飯降山}}。

== 區域 ==
福井市共分八個大區域，四十七個小區。

== 經濟 ==
主要以纖維業為主，近年以化學工業為主。1977年發展市內首個商業區。以福井市登記的總行的上市公司有[[熊谷組|熊谷組]]、{{link-jp|江守商事|江守商事}}、{{link-jp|福井銀行|福井銀行}}、{{link-jp|三谷商事|三谷商事}}等。
