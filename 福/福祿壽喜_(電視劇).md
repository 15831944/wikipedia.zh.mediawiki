《'''福祿壽喜'''》是 1986 年[[台灣電視公司|台灣電視公司]]自製的[[連續劇|連續劇]]，也是[[金馬獎|金馬獎]][[影后|影后]][[歸亞蕾|歸亞蕾]]首次製作的電視劇。本劇以[[養老院|養老院]]為背景，於台北縣仁愛之家頤苑安養院 ( 今[[衛生福利部|衛生福利部]]北區老人之家[[頤苑自費安養中心|頤苑自費安養中心]] ) 實景拍攝，描寫住在養老院的[[銀髮族|銀髮族]]們的生活點滴，以及他們每個人之所以住進來的原因。本劇折射出現實工商社會中的老人問題，藉此拋磚引玉，呼籲為人子女應恪盡孝道，以及社會應重視老人問題。

然而該劇並非消極論述子女不應把父母送去養老院，而是積極宣導養老院的優點，宣稱「並非送父母去住養老院就是不孝，讓老人能與[[同儕|同儕]]交流互動，反而能使他們得到更多的快樂而不寂寞，會比住在家裡兒孫都去上班上學而孤獨一人來得更好」，提倡子女應經常來院裡探親，而非託置不管，只負責替父母繳費。

該劇雖走喜劇路線，台詞對話多詼諧風趣，尤其是老人之間的抬槓調侃，但也深具教化意義，發人深省。

該劇由[[歸亞蕾|歸亞蕾]] ( 自製自演 )、[[劉嘉芬|劉嘉芬]]、[[陶君薇|陶君薇]]、[[劉延方|劉延方]]、[[陳婉麗|陳婉麗]]主演，資深演員[[王珏|王珏]]、[[雷鳴|雷鳴]]、[[韓甦|韓甦]]、[[魯直|魯直]]、[[丁也恬|丁也恬]]等飾演養老院的老人們，[[劉德凱|劉德凱]]、[[紫薇|紫薇]] ( 第 1 集 )、[[張國榮|張國榮]] ( 第 6 集 )、[[陶大偉|陶大偉]] ( 第 21、22 集 ) 於劇中[[客串|客串]]演出，以及[[中華民國陸軍|中華民國陸軍]][[一級上將|一級上將]][[何應欽|何應欽]][[將軍|將軍]]對該劇劇名的題字墨寶。1986 年 6 月 9 日起於台灣電視公司周一至週五晚上八點首播，7 月 18 日結局。

== 劇情簡介 ==
客機降落[[中正國際機場|中正國際機場]]，沈雲平 ( [[劉德凱|劉德凱]]飾 ) 陪同母親方孝儀 ( [[歸亞蕾|歸亞蕾]]飾 ) 由機場入境。

方孝儀原本遠赴美國與兒子媳婦同住，因不習慣美國生活和跟美國媳婦 Anna 的相處，想回台灣住養老院。適逢台北厚福安養院聘請一位美國研究老人福利的專家崔安琪 ( [[陶君薇|陶君薇]]飾 ) 前來擔任院長，她是美國華裔，很年輕，才 26 歲，美麗的東方臉孔卻具有西方女子活潑豪爽的性格，說了一口帶著濃厚英文腔的華語。她與方孝儀母子恰巧搭同一班機返台，但互不認識。

厚福安養院是莊義生 ( [[劉延方|劉延方]]飾 ) 的父親創辦的，義生雖在船運公司工作，但對於安養院也盡心盡力管理。崔安琪來到厚福，組長郭毓芬 ( [[劉嘉芬|劉嘉芬]]飾 ) 為她作了簡報，介紹了整個安養院的概況，以及院裡的老人，得知這些可愛的老人們各自都有不同的原因住進來──鄭超凡 ( [[王珏|王珏]]飾 ) 是一名退役老將軍，他與莊家共同合資厚福，所以自己不去住榮民之家，一心一意住在自己投資的厚福；袁宗維 ( [[魯直|魯直]]飾 ) 以前是鄭超凡軍中的部屬，雖有家室，仍追隨鄭超凡一起住進來；-{于}-建林 ( [[韓甦|韓甦]]飾 ) 是退休的建築業老闆，雖擁有財富卻生性勤儉，因看不慣兒子媳婦的奢侈生活，負氣住進厚福；蘇中光 ( [[雷鳴|雷鳴]]飾 ) 生性風流，年輕時不腳踏實地而負債累累，年老之後又不滿兒子成就不高，不能給自己無憂無慮的生活，於是住進厚福，他在院裡仍不改其風流個性，每天裝扮時髦，到處與院裡的女性老人或年輕社工妹妹搭訕交誼；而歐陽燕 ( [[丁也恬|丁也恬]]飾 ) 生了六個子女，每位子女都藉故忙碌而將她托置在這裡，平常很少來探望，使得她大嘆兒女不孝，每每想到自己的兒女，或見到別人的兒女來院裡探親，就開始犯情緒化的歇斯底里......。簡報結束之後，沈雲平帶著母親方孝儀來報到了。方孝儀一進入院區，備感厚福的溫馨和同儕們的熱情歡迎，蘇中光一如往常的，開始傾心新來的方孝儀，經常親近她、討好她。

郭毓芬在院裡擔任社工組長，因從小與莊義生一起長大，暗戀義生很久，又住在同一棟宿舍裡，每天暗自照顧義生的生活起居，而義生卻因習慣於毓芬的照顧，對她的付出視為理所當然，不知珍惜。安琪住進宿舍之後，他們之間出現了另一人，再加上安琪也被義生樂於助人的純真性情吸引了，對他產生好感，於是漸漸形成了三角關係。義生自己也不清楚自己的感情，不知自己到底喜歡誰，於是他對兩個女生都一視同仁，逃避這樣的三角關係不想處理，曾一度造成毓芬和安琪之間的隔閡，以及三人之間的尷尬，而形同陌路。之後，安琪得知毓芬對義生的始終如一，便收斂起自己的情感而退出，不想破壞自己跟毓芬之間的友誼；直到安琪在美國的朋友唐喬治 ( [[王台慶|王台慶]]飾 ) 的出現，他對毓芬一見鍾情，展開猛烈的追求攻勢，才刺激了義生對毓芬的告白......

沈雲平的妹妹沈雲采 ( [[陳婉麗|陳婉麗]]飾 ) 得知哥哥把母親送到養老院，非常不滿，大罵哥哥不孝，毅然決然把母親接回家住，殊不知其實母親是自願想住進養老院的，硬要她去女兒家住，反倒使她增加了心理壓力，對女兒產生避之猶恐不及的尷尬場面。在女婿俞全忠 ( [[關山|關山]]飾 ) 處理岳母是否住厚福的事情之中，巧遇住在厚福的鄭超凡，一眼認出他是自己小時候的戰亂期間殺死父親的凶手，這是只知其一不知其二的一場誤會，卻帶給全忠半生的仇恨，他立誓要報殺父之仇......

故事的主軸分為兩路，一是以養老院的老人為主軸，一是以年輕輩的三角關係為主軸，但兩個主軸又互相有密不可分的關聯。
