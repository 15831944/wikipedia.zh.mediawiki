'''福州市公共交通集团有限责任公司'''是福州市的公共交通运营公司之一，为福州市政府全资拥有国有独资企业，业务以[[公共汽车|公共汽车]]运营、管理为主。福州公交集团的巴士主要运营在福州六个的市辖区和闽侯县。

== 沿革 ==
1952年1月1日年福州公交集团前身福州市公共交通公司成立，初期仅21辆汽车、两条线路，职工210人。<ref>{{cite book|author1=方炳桂|title=名城的记忆|date=2007|publisher=福建人民出版社|isbn=9787211055982|page=66|accessdate=2017-11-26}}</ref><ref name="80s">{{cite book|author1=福州市交通局|title=福州交通志|date=1988|publisher=福建人民出版社|isbn=7211006137|pages=228|accessdate=2018-01-16}}</ref>

公司在1958年后曾多次变更公司名称<ref name="80s"/>：
*[[文化大革命|文化大革命]]期间1968年1月21日公司名称变更为福州市人民汽车公司。
*1970年7月16日，公司[[军事化|军事化]]，与货运企业福州市运输公司合并，编为福建省汽车第九团。
*1973年7月26日，省汽车第九团一分为二，公司恢复原名。

1983年9月29日，公司[[无轨电车|无轨电车]]系统开通，使福州成为[[中国大陆|中国大陆]]第26个拥有无轨电车的城市，首条线路51路为台江影院到[[福州火车站|福州火车站]]，用以替代原福州2路公共汽车<ref>{{cite web|title=【海峡·特别策划】一次高架桥改造，竟让福州51路回到了无轨电车时代…|url=http://n.hxbus.net/index.php/archives/185/|website=n.hxbus.net|accessdate=2017-11-26}}</ref><ref name="80s"/>，全长7.6公里，票价2元。<ref>{{cite news|title=福州大事记（1949—1999）|url=http://www.66163.com/fujian_w/news/fzwb/990817/1_1.html|accessdate=2017-11-26|agency=福州晚报|date=1999-08-17}}</ref><ref name="fznews51">{{cite web|title=闪回1983年的福州:无轨电车诞生 少年爱做太空梦|url=http://news.fznews.com.cn/fuzhou/20150203/54d03b0018b1d.shtml|publisher=福州新闻网|accessdate=2017-11-26}}</ref>

2001年，公司无轨电车停运。<ref name="fznews51"/>

2004年7月，经福州市政府批准，福州市公共交通总公司进行体制改革，改制为福州市公共交通集团有限责任公司，由市政府所属的福州市国有资产监督管理委员会全资控股<ref>{{cite web|title=集团简介|url=http://www.fz-bus.cn/class.asp?class=0|website=www.fz-bus.cn|archive-url=https://web.archive.org/web/20110219082653/http://www.fz-bus.cn/class.asp?class=0|archive-date=2011-02-19|accessdate=2017-11-26|archiveurl=https://web.archive.org/web/20110219082653/http://www.fz-bus.cn/class.asp?class=0|archivedate=2011-02-19|deadurl=yes}}</ref>。

2013年，福州市政府对市属国有企业进行重组调整，福州公交集团被并入福州交通建设投资集团有限公司内，由其全资控股。<ref>{{cite news|title=福州公交集团并入“大交投”　东部新城辟新线路_即时信息_新闻频道_福州新闻网|url=http://news.fznews.com.cn/jsxx/2013-6-4/201364XSE3RGX9PO225022.shtml|accessdate=2017-11-26|work=news.fznews.com.cn|date=2013-06-05}}</ref>

== 运营 ==
[[File:AYC981_at_Ximen_(20160118094608).jpg|thumb]]
集团所属的公交线路运营分由下属子公司分别运营，覆盖6个市辖区、1个县。
*福州公交集团一公司 (5个车队)
*福州公交集团二公司 (4个车队)
*福州公交集团三公司 (5个车队)
*福州市公共交通集团马尾惠民巴士有限责任公司
*福州市公共交通集团闽侯惠民巴士有限公司

截至2018年1月，不论运营形式、区域，以上公司共运营有公交线路199条。

另有开设福州公交出租汽车有限责任公司进行[[出租车|出租车]]运营和福州公交[[驾校|驾校]]对外招生培训汽车驾驶员。
