'''福州民系'''（{{lang-cdo|Hók-ciŭ-nè̤ng|han=福州儂|ipa=huʔ˨˩ tsiu˥˥ nœŋ˥˧}}<sub>（福州腔讀音）</sub>），即'''福州族群'''、'''福州人'''、'''福州侬'''、'''十邑人'''，也有称为'''闽东民系'''，是发源于[[中国|中国]][[福建|福建]]东部的一个[[民系|民系]]，属[[闽民系|闽民系]]的一支。福州民系是指籍贯在[[福州十邑|福州十邑]]，以[[福州话|福州话]]为母语，认同[[福州文化|福州文化]]的汉族民系。广义的福州民系还包括籍贯在[[宁德|宁德]]地区其余县市，以广义的[[闽东语|闽东语]]为母语的[[族群|族群]]<ref>{{Cite book|title=福建六大民系|author=陈支平|first=|date=2000年8月|publisher=福建人民出版社|year=|isbn=7-211-03677-X|location=福州|pages=}}</ref>。传统上福州民系不包括该地区同说闽东语的[[福州疍民|疍民]]，也不包括近现代迁入福州的[[福州外地移民|外地移民]]。

== 分布 ==
狭义的福州民系分布于[[福州十邑|福州十邑]]，即今天福建省[[福州|福州]]地区的福州五区、[[闽侯县|闽侯县]]、[[闽清县|闽清县]]、[[长乐市|长乐市]]、[[福清市|福清市]]、[[平潭县|平潭县]]、[[连江县_(福州市)|连江县]]、[[罗源县|罗源县]]、[[永泰县|永泰县]]，以及[[宁德|宁德]]地区的[[古田县|古田县]]和[[屏南县|屏南县]]、中華民國治下的[[連江縣_(中華民國)|連江縣]]（[[馬祖|馬祖]]）。由于历史上各个时期福州人进行[[移民|移民]]的缘故，福州民系还分布在[[马来西亚|马来西亚]]、[[印度尼西亚|印度尼西亚]]、[[新加坡|新加坡]]、[[文莱|文莱]]、[[缅甸|缅甸]]、[[泰国|泰国]]、[[美国|美国]]、[[日本|日本]]、[[英国|英国]]、[[加拿大|加拿大]]、[[臺湾|臺湾]]、[[香港|香港]]等许多国家和地区。

== 形成 ==
=== 来源 ===
福州民系的来源有两个：[[中原|中原]][[汉人|汉人]]的入迁和土著[[闽越人|闽越人]]的[[汉化|汉化]]。南迁至福州的汉人与当地汉化的闽越人相互融合，形成了福州民系。福州民系有着普遍的中原认同。

中原汉民迁居福建可追溯至[[汉代|汉代]]，自[[晋代|晋代]]的[[八姓入闽|八姓入闽]]开始，见诸于文字记载的汉人入闽渐多，尤以迁居[[闽江|闽江]]下游的[[福州平原|福州平原]]为众，福州地区成为早期汉人入闽的主要聚居点。到了[[晚唐|晚唐]]和[[五代|五代]]，随着[[王潮|王潮]]、[[王审知|王审知]]入闽建立[[闽|闽]]政权，形成了一个中原汉人入闽的高潮，福州成为闽国的政治中心，得到了大力开发。<ref>{{Cite book|title=福建六大民系|author=陈支平|first=|date=2000年8月|publisher=福建人民出版社|year=|isbn=7-211-03677-X|location=福州|pages=}}</ref>

古闽越人是福建的原住居民。公元前202年，闽越王[[无诸|无诸]]定都东冶，即今天福州[[冶山|冶山]]附近。公元前110年，闽越国被[[汉朝|汉朝]]所灭，闽越国的国民被遷徙至江淮间，闽地遂空。在此后漫长的时间里，留居福建的闽越人逐渐被汉化。<ref>{{Cite book|title=福州市志(第一册)|last=|first=|date=1998年12月|publisher=方志出版社|year=|isbn=7-80122-400-0|location=北京|pages=}}</ref>
<ref>{{Cite book|title=妈祖的子民：闽台海洋文化研究|author=徐晓望|first=|date=1999年12月|publisher=学林出版社|year=|isbn=7-80616-790-0|location=上海|pages=}}</ref>

=== 形成时间 ===                                                          
对于福州民系的形成时间，陈支平认为大致在[[隋|隋]][[唐|唐]][[五代|五代]]已经形成<ref>{{Cite book|title=福建六大民系|author=陈支平|first=|date=2000年8月|publisher=福建人民出版社|year=|isbn=7-211-03677-X|location=福州|pages=}}</ref>。从语言形成的角度上看，李如龙认为闽东语形成于唐末王潮、王审知入闽时期<ref>{{Cite book|title=福建方言|author=李如龙|first=|date=1997年10月|publisher=福建人民出版社|year=|isbn=7-211-02406-2|location=福州|pages=}}</ref>。一般认为，福州民系形成于[[五代|五代]]时期。

== 语言 ==
{{main|福州话}}
福州民系使用的语言为[[闽东语|闽东语]][[福州话|福州话]]，属于[[汉语|汉语]]的[[闽语|闽语]]分支。福州人常称自己的母语为“平话”。

== 风俗 ==
{{main|福州风俗}}

=== 节日 ===

==== 特有节日 ====
===== 拗九节 =====
{{main|拗九节}}
'''拗九节'''又称'''后九节'''、'''孝九节'''和'''送穷节'''，为农历的正月廿九。该节来源于“[[目连救母|目连救母]]”的传说，在这一天，要用[[糯米|糯米]]、[[红糖|红糖]]、[[花生|花生]]、[[红枣|红枣]]、[[芝麻|芝麻]]、[[荸荠|荸荠]]、[[桂圆|桂圆]]等煮成“拗九粥”，用来祭祖或赠与亲友。出嫁的女儿要送拗九粥回娘家孝敬父母。

===== 二月二 =====
在农历二月初二，这一天，福州人要改吃用糯米、[[芹菜|芹菜]]、葱、蒜、虾米、[[牡蛎|牡蛎]]、肉丝等做成的咸粥。

==== 春节 ====
===== 筅堂 =====
[[春节|春节]]前，福州人要进行筅堂（或称“扫堂”），即选吉日进行房屋大扫除，并贴春联、門神。

===== 送年与送灯 =====
每年[[春节|春节]]来临之际，出嫁的女儿要往娘家“送年”。传统上所送的是[[年糕|年糕]]、[[豬腳|豬腳]]、[[草鱼|草鱼]]或[[鲢鱼|鲢鱼]]以及[[公鸡|公鸡]]。<br>
“送灯”是过年时由亲家舅往出嫁的女儿家裏送[[灯笼|灯笼]]，有“添丁”之意。

===== 回娘家，送年礼 =====
女儿女婿春节之前，要回娘家送年礼。

==== 元宵节 ====
元宵节要举办灯会，还有舞龙、舞狮、[[擺暝|擺暝]]、迎神遶境等民间文艺活动。

==== 清明节 ====
清明节要扫墓、压墓纸，还要用菠菠菜做成“菠菠粿”（菠菠饼）来食用。

==== 七月半 ====
七月半为“鬼节”，民间在这一天烧冥纸、冥衣等。

==== 重阳节 ====
在九月初九，福州人有重阳节登高放纸鹞的习俗。

==== 冬节 ====
在每年农历冬至日。福州人冬节有“搓𥻵”的习俗。

==== 祭灶 ====
农历十二月廿四祭奠灶神。福州民间有“官三民四[[曲蹄|曲蹄]]五”的说法，即做官人家在十二月廿三祭灶，平民家庭在廿四祭灶，而[[疍民|疍民]]家庭（福州贬称为“曲蹄”）在廿五祭灶。

=== 人生礼仪习俗 ===

==== 做寿 ====
福州人常由虚岁五十岁开始做寿，宴請賓客。寿礼常送寿桃包、线麵、烛、炮等。做寿要食“太平麵”。

==== 丧葬习俗 ====
传统上丧事要“做七”，即在亲人去世后七七四十九天内做法事等。孝男要戴孝，穿麻衣麻帽草鞋，腰扎草绳，麻帽上挂两个棉球。<ref>{{Cite book|title=福州风情|author=方炳桂|first=|date=1999年8月|publisher=鹭江出版社|year=|isbn=7-80610-622-7|location=厦门|pages=}}</ref><ref>{{Cite book|title=福建民俗|author=林蔚文|first=|publisher=甘肃人民出版社|year=2002|isbn=7-226-02751-8|location=兰州|pages=}}</ref>

== 艺术 ==

=== 戏曲音乐 ===
福州人的传统音乐是[[十番音乐|十番音乐]]，传统曲艺包括[[闽剧|闽剧]]、[[福州评话|福州评话]]、[[伬唱|伬唱]]、[[北路戏|北路戏]]等。

=== 工艺美术 ===
福州的传统工艺有[[寿山石雕|寿山石雕]]、[[脱胎漆器|脱胎漆器]]、福州纸伞、牛角梳等。

== 建筑 ==
{{main|闽东建筑}}
[[File:Eastmin_arch1.jpg|300px]]，摄于[[罗源县|罗源县]]]]
福州民系的建筑具有强烈的地域特点。民居多为三合型和四合型，注重入口门罩的处理，封火山墙造型尤其独特，比较典型的是马鞍形风火墙。风格明快、灵巧、素雅。代表性的福州民居群落如福州的[[三坊七巷|三坊七巷]]和[[闽清|闽清]]的[[宏琳厝|宏琳厝]]等。<ref>{{Cite book|title=缘与源——闽台传统建筑与历史渊源|author=林从华|first=|publisher=中国建筑工业出版社|year=2006|isbn=7-112-08456-3|location=北京|pages=}}</ref>

== 民间信仰 ==
福州人的传统信仰非常丰富。除了[[妈祖|妈祖]]等全闽普遍信仰的神祇外，还有众多的地方神祇，包括[[開閩聖王|開閩聖王]]、[[王審邽|泉安尊王]]、[[威武陳將軍|威武侯]]、[[白馬尊王|白馬尊王]]、[[五福王爺|五福王爺]]、[[法主真君|法主真君]]、[[临水夫人|临水夫人]]、[[懿德夫人|懿德夫人]]、[[武夷君|武夷真君]]、[[齐天大圣|齐天大圣]]、[[董奉|董真人]]、[[裴真人|裴真人]]、[[马天君|马天君]]、[[照天君|照天君]]、[[英烈王|英烈王]]等等，许多神祇仅仅限于某个村落或城镇。每年[[除夕|除夕]]到[[元宵|元宵]]期间，有大量的民间游神活动。<ref>{{Cite book|title=福建民俗|author=林蔚文|first=|publisher=甘肃人民出版社|year=2002|isbn=7-226-02751-8|location=兰州|pages=}}</ref>

== 基督宗教信仰 ==
清末时期，有很多福州人信仰基督宗教，特别在[[福清|福清]]，[[闽清|闽清]]，[[古田|古田]]等。今天的海外福州人社会当中，还可以看到福州基督徒把信仰带到南洋的足迹；如[[马来西亚|马来西亚]]的[[诗巫|诗巫]]，和[[实兆远|实兆远]]。直到如今，他们的子子孙孙还仍然保留着祖先们的传下来的基督宗教信仰。东马来西亚的[[砂拉越|砂拉越]]基督徒人口比例高达44%。

{{main|福州菜}}
福州人的传统菜系为[[闽菜|闽菜]]中的福州菜。

== 海外移民 ==

=== 老移民 ===
自宋代起，就有明确见诸于文字记录的福州人海外经商移民活动。[[明|明]][[洪武|洪武]]二十五年（1392年）有[[闽人三十六姓|闽人三十六姓]]到[[琉球国|琉球]]定居。明代中叶后，福州地区有不少人移居[[日本|日本]]，还有人迁往[[东南亚|东南亚]]。清末有大量的福州十邑人以各种形式移民东南亚，并形成了规模可观的福州人社区，比如马来西亚[[霹雳州|霹雳州]]的[[实兆远|实兆远]]和[[砂拉越州|砂拉越州]]的[[诗巫|诗巫]]。这个移民潮一直持续到1949年[[中华人民共和国|中华人民共和国]]成立，因为政治形势的改变，此后直到1970年代，福州人的海外移民基本停止。<ref>{{Cite book|title=福州市志(第八册)|last=|first=|date=2000年12月|publisher=方志出版社|year=|isbn=7-80122-605-4|location=福州|pages=}}</ref>

=== 新移民 ===
1970年代开始，福州人开始新一阶段的移民潮。移出地主要为[[闽江|闽江]]口的福州、长乐、福清、[[连江县_(福州市)|连江]]等市县，主要移民目的地国包括美国、日本、英国、加拿大等。这一阶段的新移民中有很大一部分采取了不合法的手段进入所在国，也有相当多的人最后取得了合法居留权。福州新移民分布极为广泛，遍布于五大洲的许多国家，在海外形成了大量新的福州人社区，其中比较著名的比如[[纽约|纽约]]的[[曼哈顿华埠|曼哈顿华埠]]。<ref>{{Cite book|title=福州市志(第八册)|last=|first=|date=2000年12月|publisher=方志出版社|year=|isbn=7-80122-605-4|location=福州|pages=}}</ref><ref>{{cite book |last= Pieke |first= Frank N. |title= ''Transnational Chinese : Fujianese migrants in Europe '' |publisher= Stanford University Press |year= 2004 |isbn= 0804749957 }}
</ref>在海外福州人社会当中，既有“长乐人在美国，福清人在日本”的说法。
