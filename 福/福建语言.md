'''闽语'''，为[[福建省|福建省]]內各種本土語言。可分为[[闽东语|闽东语]]、[[闽南语|闽南语]]、[[闽北语|闽北语]]、[[闽中语|闽中语]]、[[莆仙語|莆仙語]]。另外，在福建地区流通的[[漢語變體|漢語變體]]還有[[客家語|客家語]]、[[吳語|吳語]]和[[赣语|赣语]]等。

最狹義的'''福建话'''是指[[閩南語|閩南語]][[泉漳片|泉漳片]]，以[[廈門話|廈門話]]與[[臺灣話|臺灣話]]為代表。另也有部分人將[[閩北語|閩北語]][[建瓯話|建瓯話]]以及稱為福建話。在[[海外華人|海外華人]]社群當中，“福建”（英語：Fuguing；Hokkien)即指[[閩北語|閩北語]][[建瓯片|建瓯片]]和[[閩南語|閩南語]][[泉漳片|泉漳片]]。

== 闽语 ==
[[闽语|闽语]]通常分为[[闽东话|闽东话]]、[[闽南语|闽南语]]、[[闽北语|闽北语]]、[[闽中语|闽中语]]、[[莆仙語|莆仙語]]（[[興化語|興化語]]）等。

*[[闽东语|闽东语]]：主要分布于福建中东部的[[福州|福州]]和[[宁德|宁德]]两地市，分为南北两片。南片以[[福州话|福州话]]为代表，分布在[[福州十邑|福州十邑]]地区；北片以[[福安话|福安话]]为代表，也有学者将宁德腔归入闽东语南片中。这一片所在的地区在[[吴越国|吴越国]]时期大都属于[[两浙十三州|两浙十三州]]的福州府，因此受到吴语的一些影响。

*[[闽南语|闽南语]]：主要分布于福建南部的[[泉州|泉州]]、[[漳州|漳州]]、[[厦门|厦门]]等地市，[[臺灣|臺灣]]使用的[[臺灣話|臺灣話]]亦屬之。

*[[闽北语|闽北语]]：主要分布于福建北部的[[南平|南平]]地区，以建瓯话为代表。

*[[闽中语|闽中语]]：主要分布于福建中北部的[[三明|三明]]、[[永安市|永安]]、[[沙县|沙县]]等地区，以永安话为代表，是闽语中使用人数最少，适用范围最窄的一种。

*[[莆仙語|莆仙語]]：分[[莆田|莆田]]、[[仙游|仙游]]、福州市的[[福清|福清]]南部、[[永泰县|永泰]]部分地区， 以及[[泉州|泉州]][[泉港|泉港]]的部分地区，也常称作兴化话。以莆田话为代表。

== 客语 ==
*[[汀州话|汀州话]]：分布于[[龍巖市|龍巖市]]、[[三明市|三明市]]。以[[長汀話|長汀話]]为代表

== 吳语 ==
*[[浦城话|浦城话]]：分布于[[浦城縣|浦城縣]]北部，屬[[吴语|吴语]]的[[处衢片|处衢片]]。

== 贛语 ==
*分布于建寧、泰寧等闽北县市。以[[建寧話|建寧話]]为代表，屬[[贛语|贛语]]的[[撫廣片|撫廣片]]。
