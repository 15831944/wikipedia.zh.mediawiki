'''福樂村'''是中臺灣嘉義縣[[民雄鄉|民雄鄉]]轄下的行政區。前身為雙福村，屬埤角地段，民國71年埤角開發成民雄工業區後，內移人口遽增，於民國83年脫離雙福村，正式命名福樂村，因發展快速，目前已成為全嘉義縣人口最多的村落之一！<br />
位於嘉義縣東北方，北鄰民雄鄉雙福村，西鄰民雄鄉興南村，南鄰嘉義市東區，東接民雄鄉北斗村。面積約1.840325平方公里，行政區包含39個鄰、2727戶、人口數7651人<ref>{{Cite web|url=http://minsyong-hro.cyhg.gov.tw/Default.aspx|title=嘉義縣民雄戶政事務所|accessdate=2017-09-27|author=|date=|publisher=}}</ref>。聚落大部分位於民雄暨頭橋工業區範圍內，居民也多為因工作移入的人口，是民雄鄉鄰數、戶數及人口最多的聚落<ref>{{Cite web|url=https://www.moeaidb.gov.tw/iphw/minsyong/index.do?id=10|title=民雄暨頭橋工業區|accessdate=2017-09-26|author=|date=|publisher=}}</ref>。<br />
相傳楊姓是最早開拓埤角的先民，後來姜姓、陳姓ㄉ陸續遷來。為了填飽肚子，只好到處打工，早、慢冬稻收穫期，他們遠赴成熟期較早的屏東，一路出賣勞力，逐稻田幫割回家。牛車出門載石頭，一早出門，分文不帶，只帶一盒便當，備一竹筒的水﹙當年農夫用麻竹筒盛開水﹚，晚上八、九點回到家，一天下來，領到工資買不到一斗米。終年辛勞，還要儲錢買田，這種強韌的生命力，足讓現代人汗顏。62年頭橋工業區推出後，縣府隨即著手規劃民雄工業區。67年五月開發，於71年三月完成，總面積244公頃，公共設施48.6公頃，可供建廠用地173.4公頃。70年八月成立管理中心，舉凡工業區內土地、建築物、環境衛生、水權等之管理，大路、水溝、路燈、景觀等之維護，協助籌組醫療診所、消防防護、守望相助、職業訓練、休閒活動、求才求職、排解勞資爭議，或污水處理、水質分析化驗、進水廠水質管制等皆為其服務範圍。

== 地理 ==
本區域最早開發的地區是「埤角」聚落，然屬於灌溉較不易的的區域，據當地文史調查直至1950年代，仍未達百戶居民。然1973年頭橋工業區完成後，新建公寓大廈集合住宅，形成近期的聚落景觀。
聚落西向比鄰台一線省道，東西向為嘉166縣道貫穿。聚落南方為工業區<ref>{{Cite book|title=說咱庄頭|last=|first=|publisher=財團法人民雄文教基金會|year=2001|isbn=|location=民雄鄉|pages=158-161}}</ref>。

== 教育 ==
* 本區域原為秀林國小學區，因1999年九二一大地震摧毀校舍，2002年重建秀林國小福樂分校，2003年改名嘉義縣民雄鄉福樂國民小學<ref>{{Cite web|url=https://www.moeaidb.gov.tw/iphw/minsyong/index.do?id=10|title=福樂國小─校史|accessdate=2017-09-27|author=|date=|publisher=}}</ref>至今。
本聚落無設立國中，學區為民雄國民中學。

== 文化資源 ==
* 台灣基督長老教會頭橋教會<ref>{{Cite web|url=http://www.pct.org.tw/look4church.aspx|title=台灣基督長老教會|accessdate=2017-09-27|author=|date=|publisher=}}</ref>：為台灣基督長老教會嘉義中會下的教會，設立於1980年。

== 景點 ==
嘉義酒廠<ref>{{Cite web|url=http://event.ttl-eshop.com.tw/cy/index.aspx|title=臺灣菸酒股份有限公司─嘉義酒廠|accessdate=2017-09-27|author=|date=|publisher=}}</ref>：1999年於民雄工業區建立營運。原嘉義酒廠於目前嘉義市境內於新酒廠落成後停止營運。1916年(大正5年)設立，原建築廠房為木構造建築，1945年台灣光復後，由台灣省專賣局管理及修復。並於2003年公告為歷史建築。
