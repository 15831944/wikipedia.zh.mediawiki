'''福音主義'''（{{lang-en|evangelicalism}}），{{Fact|time=2011-01-03|為[[基督教神學|新教神學]]影響層面最為廣闊的[[神學|神學]]主張之一}}，起源于1870年代的英国<ref>Bebbington, D. W. (2008). ''Evangelicals in Modern Britain: A History from the 1730s to the 1980s'', London: Unwin, 1.</ref>。在不同的時期中，福音神學一詞經常會被立場更為開放的新神學主張作為比較的對象，故而在[[基督教|基督教]]近代史中，福音派神學等同於信仰上的[[保守主義|保守主義]]。福音神学的四大特点是：强调个人归信基督（或曰[[重生_(基督教)|重生]]）；积极地表述和[[傳福音|传播福音]]；强调[[圣经|圣经]]的权威，坚信[[圣经无错谬|圣经无错谬]]；强调与[[耶稣复活|耶稣复活]]有关的基督教教义。<ref>{{Cite web |url = http://www.wheaton.edu/isae/defining_evangelicalism.html |title=Defining Evangelicalism |accessdate=2008-03-04 |last=Eskridge |first=Larry |year=1995 |publisher=Institute for the Study of American Evangelicals }}</ref>

== 福音（派）神學名稱的形成 ==
* 現代人所稱的'''福音派神學'''是作為「[[自由主義神學|自由主義神學]]」的對立面而形成的一個論述集團，此意義上的福音派神學是一種平民化的神學主張（不使用複雜的[[知識論|知識論]]、採取較為鬆散的論點），並且在此一情況中應該要準確的將這兩個神學主張譯名為『福音派神學』與『自由派神學』，因為兩個名稱是彼此對照而來、成員更是互相排除的。

* 「[[福音派|福音派]]」一詞另曾被[[喀爾文|喀爾文]]的[[歸正宗|歸正宗]]以外的新教諸宗派用來稱呼自己（最初是[[路德宗|路德宗]]使用，後來[[宣道會|宣道會]]、[[衛理公會|衛理公會]]、[[貴格會|貴格會]]和英國[[聖公會|聖公會]]....等等宗會都使用），在此一意義上的'''福音神學'''是突顯其與[[喀爾文主義|喀爾文主義]]中極為嚴酷的「[[預選說|預選說]]」之不同，意在強調福音的普世性和教會肩負廣傳福音的使命。在此種情形中又不妨使用'''福音主義'''稱之，以免混淆。也有學者認為[[阿民念主義|阿民念主義]]屬於福音主義的濫觴。

* 雖然有至少兩個以上的'''福音神學'''名稱源流，但它們仍同屬新教神學系統，且神學內容沒有重大歧異，因此福音神學至今仍被新教神學中的保守主義者延用。

== 福音神學的定義 ==
* 福音神學中沒有[[新正統神學|新正統神學]]裡那般清晰的[[哲學|哲學]]上的知識論傳統，卻也同樣不接受自由主義神學所採納的那些不利於《[[聖經|聖經]]》信仰權威的新科學證據，因此同樣有[[反理性主義|反理性主義]]的傾向，但又不刻意發展[[神秘主義|神秘主義]]哲學以對抗[[理性主義|理性主義]]。因此也可以說：福音派神學注重的是基督教在普羅大眾中的影響力，並不像新正統神學那樣強調與知識份子的對話。

* 其強調「神愛世人」、[[基督|基督]]為世人的[[原罪|罪]]釘死在[[十字架|十字架]]上、基督的降生乃至復活便是[[上帝|上帝]]賜給人類的福音、基督升天前要求門徒將福音傳到地極....等[[新約|新約]]《聖經》的經節。強調基督徒應熱心地傳福音，主張「聖經無誤論」、「逐字聖靈啟示說」，但也沒有一致的權威釋經文本，多是各自採用「以經解經」的方式，只要不與別處《聖經》抵觸的情形之下，允許信徒用個人的經歷來解釋各別《聖經》章節。

== 與基要神學間的異同 ==
「[[基要神學|基要神學]]」乃是稱呼福音神學中最為激烈跟保守的論點。基要神學與福音派神學兩個名詞若被賦予不同意義時，通常是強調基要神學家已經發展出了嚴謹的論述体系。福音神學較之則是更大範圍的一個通稱，只要符合其基本神學要求的主張都可以被歸作福音神學──通常，立論鬆懈甚至提不出論證而僅止於信仰的基要神學主張會被視為「溫和的福音神學」。

== 與福音（派）神學有過爭論的其他神學主張 ==
* [[加爾文主義|加爾文主義]] - 宗教改革時期的神學爭議。現在的加爾文宗教派改採修正後的[[新加爾文主義|新加爾文主義]]，原先的歧異現已消弭大半。
* [[自由主義神學|自由主義神學]] - 內容又有：「[[歷程神學|歷程神學]]」、「[[進化論神學|進化論神學]]」、「[[概率有神論|概率有神論]]」與「現代主義神學」....等
* [[新正統神學|新正統神學]] - 內容又有：「[[詮釋學神學|詮釋學神學]]」、「[[辯証神學|辯証神學]]」、「[[存在主義神學|存在主義神學]]」與「[[危機神學|危機神學]]」....等
* [[靈恩派|靈恩派]] - 參閱該條
