'''福佬'''（即[[福建人|福建人]]，亦作'''學佬'''、'''鶴佬'''、'''河老'''、'''[[河洛|河洛]]'''）原為广东居民對[[閩民系|福建居民]]的稱呼<ref>台語文學者[[吳守禮|吳守禮]]以及《台灣文獻叢刊》，〈第七十三種 鳳山縣採訪冊〉，「粵稱閩人曰福老，謂福建人也」，盧德嘉，1894年</ref><ref>[http://taigi.fhl.net/dict/ 台語辭典（台日大辭典台語譯本）]，「福佬：此語本來是廣東種族對福建人輕視稱呼」，信望愛</ref>，對[[福建人|福建人]]而言，這屬於一個外來名稱。

== 詞源 ==
*目前發現的最早的「hoklo」文獻出現在美國漢學家[[衛三畏|衛三畏]]（曾在廣府中心地帶[[廣州|廣州]]生活過一段時間）於[[同治|同治]]甲戌年（1874年）所編輯的《漢英韻府》一書。
*1894年，台灣人[[盧德嘉|盧德嘉]]《鳳山縣採訪冊》出現『粵稱閩人曰福老，謂福建人也』。
*台灣學者[[吳守禮|吳守禮]]對福佬二字解釋：『「佬」字在[[廣東話|廣東話]]裏指「男子」，「福佬」二字起初可能如「福人」、「福建人」，到了客語裏就不同了。客語稱小偷叫「賊佬」稱啞巴叫「啞佬」。「福佬」自是帶出「討厭」、「壞」的貶義來』。<ref>{{cite web|title=一百年來的閩南系台灣話研究回顧#正名|author=吳守禮|date=2003-04-02|publisher=老醫之家|url=http://olddoc.tmu.edu.tw/chiaushin/shiuleh-11.htm}}</ref>{{誰|但}}台灣客家話並未稱小偷為「賊佬」、亦沒有稱啞吧為「啞佬」，所以可推論吳守禮誤用了某些資料。
*台灣學者[[洪惟仁|洪惟仁]]指出，「福佬」兩字的[[客家話|客家話]]讀音是「Fuk-ló」（洪惟仁1987，132）。
*在[[閩|閩]][[粵|粵]]邊界，閩客混居的地方，雙方互相稱呼對方為『hoklo』與『講客的』。原[[潮州府|潮州府]]（即今[[潮汕地區|潮汕地區]]）有俗語：『[[大埔縣|大埔]]無hok，[[澄海|澄海]]無客』，說明hok與客並存。早期閩南人不會自稱hoklo。後來因客家人對閩南人的俗稱漸傳漸廣，福佬人一詞才廣為流行。故近代多數閩南人亦會承認福佬人此稱呼。至於「學佬」一詞，則為廣府人粵語hoklo「學佬」近音，將之音譯為「學佬」。

== 河洛（Ho̍-lóh）釋義 ==
'''Ho̍-lóh'''人<ref>{{cite web|title=台語辭典(台日大辭典台語譯本):hô-lóh-lâng|publisher=臺語信望愛|url=http://taigi.fhl.net/dict/search.php?DETAIL=1&LIMIT=id=20380&dbname=dic&graph=2}}</ref>，是針對日常使用[[閩南語|閩南語]]者<ref>不代表這些族群皆認為自己的[[母語|母語]]是Ho̍-lóh話，如部分[[潮汕話|潮汕話]]的使用者。</ref>的族群之稱呼，是閩南民系的一支。多數的閩南族群會以居住地區或使用語言自稱，而不是血緣上的分類，比如[[雷州話|雷州話]]人口自稱「[[雷州|雷州]]人」或「講黎人」<ref>雷州話又叫做黎話。</ref>；[[潮州話|潮州話]]人口自稱[[潮州人|潮州人]]，[[泉漳片|泉漳片]][[泉州話|泉州話]]人口自稱泉州人，[[漳州話|漳州話]]人口自稱漳州人等。廣義的河洛話<ref>含[[泉漳片|泉漳片]]和[[溫州|溫州]]蒼南平陽話、[[潮汕|潮汕]]揭陽話、海豐陸豐話、隆都話、雷州話、海南文昌話。</ref>使用人口分佈世界各地。講[[臺灣話|閩南語臺灣話]]的[[閩南裔臺灣人|河洛人]]，是[[臺灣四大族群|臺灣四大族群]]之一。

== 相关历史 ==
* 台語文學家[[吳守禮|吳守禮]]研究指出，「福佬」最初為[[客家語|客家語]]中對福建人的貶義稱呼，後來「佬」字的人義淡化，稱呼福建人還必須加上「人」字成為「福佬人」，最後連福建人和閩南系[[台灣人|台灣人]]都忘記「福佬人」是他族對該族的貶稱，也自稱「福佬」<ref name="吳守禮">{{cite web|title=一百年來的閩南系台灣話研究回顧#正名|author=吳守禮|date=2003-04-02|publisher=老醫之家|url=http://olddoc.tmu.edu.tw/chiaushin/shiuleh-11.htm}}</ref>。{{誰|但}}吳守禮應該是不清楚[[廣府民系|廣府人]]跟[[客家民系|客家人]]的方言之間的差異：會大量用「佬」自蔑稱他族的貶義詞，是廣府方言而非客家方言。
* 1907年，广府人[[黃節|黃節]]編輯的《廣東鄉土地理教科書》出現了「廣東種族有曰客家、福狫族，非漢種亦非粵種」；福佬一詞變成具污蔑性的福狫。爾後彼時屬[[潮州府|潮州府]][[大埔縣|大埔縣]]的[[溫廷敬|溫廷敬]]、[[鄒魯|鄒魯]]與落籍[[廣東省|廣東省]][[海陽縣|海陽縣]]（現[[潮安縣|潮安縣]]）的[[丘逢甲|丘逢甲]]結合文人，與廣府人以文章互相批評，「福佬」一詞也開始為人所知<ref>主要是因為[[溫廷敬|溫廷敬]]所編寫的《潮州福佬民族考》一文</ref>。
* [[張壽祺|張壽祺]]認為[[粵文|粵文]]中，[[佬|佬]]字未必是個[[蔑稱|蔑稱]]<ref>張壽祺〈從民俗學考析廣州俚語「佬」字〉（羊城今古1990年第2期）</ref>
