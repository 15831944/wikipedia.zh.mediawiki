'''福州大学世界文明研究中心'''（俗称'''西观藏书楼'''）位于[[中国|中国]][[福州|福州]]，隶属于[[福州大学|福州大学]]，2003年建成，建立者为[[汉学家|汉学家]][[施舟人|施舟人]]和其妻子[[袁冰凌|袁冰凌]]，它是中国第一家西文经典图书馆，仅次于[[日本|日本]]。<ref name="fish " >{{cite web |url=http://news.sina.com.cn/c/2003-10-21/1153961314s.shtml |title= 中国首家西文经典藏书楼主人施舟人印象|accessdate=2003年10月21日 |publisher= 新浪网|work= |language =zh-cn}}</ref>

== 名字由来 ==
施舟人熟谙中国历史文化，他对东汉的[[东观藏书楼|东观藏书楼]]仰慕和钦佩不已，就把他在福州大学建立的图书馆命名为“西观藏书楼”，寓意“西方观景”，福州大学官方名称为“福州大学世界文明研究中心”。<ref name="fish " />

== 选址原因 ==
西观藏书楼始建于福州大学的原因是：其一福州的三坊七巷名人辈出，有[[林则徐|林则徐]]、[[林纾|林纾]]、[[严复|严复]]、[[冰心|冰心]]，具有深厚的文化积淀；其二施舟人的妻子袁冰凌是[[福建|福建]][[宁德|宁德]]人。<ref name="fish " />

== 历史 ==
2003年10月25日，西观藏书楼建成，福建省副省长[[汪毅夫|汪毅夫]]亲自揭牌。<ref name="fish " /><ref>{{cite web |url=http://www.chinanews.com/n/2003-10-25/26/361107.html |title=组图:福州大学“西观藏书楼”揭牌 |accessdate=2003年10月25日 |publisher=中国新闻网 |work= |language =zh-cn}}</ref>

== 藏书规模 ==
西观藏书楼藏书2.5万册，均为西方典籍和汉学著作。西观藏书楼的藏书主要是人文学科的藏书，尤其是[[欧洲|欧洲]]对于[[远东|远东]]地区的[[中国|中国]]、[[日本|日本]]、[[朝鲜|朝鲜]]的研究资料，有[[文献|文献]]、[[抄本|抄本]]、[[珍本|珍本]]、[[古籍|古籍]]、[[照片|照片]]、[[视听档案|视听档案]]等，涉及[[世界史|世界史]]、国家和地区史、艺术史和[[考古|考古]]、[[文学|文学]]、[[哲学|哲学]]、[[法律|法律]]等学科，其中1万册是施舟人夫妻的藏书，其他来自于[[荷兰|荷兰]]、[[美国|美国]]、[[法国|法国]][[政府|政府]]和国际友人的捐赠。<ref name="fish " /><ref>{{cite web |url=http://news.sina.com.cn/c/2003-10-26/1042993617s.shtml |title=中国首家西文经典图书馆西观藏书楼昨日开馆 |accessdate=2003年10月26日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

== 建筑 ==
西观藏书楼的建筑设计出自于施舟人本人，小到一个窗户，大到一个房间，均是施舟人自己亲自设计。
*阅览室
*展览室
*演讲厅

== 社会反响 ==
2011年，清祥教育基金董事长[[吴建荣|吴建荣]]和福州大学签约《关于清祥教育基金资助西观藏书楼的协议书》，出资捐赠给西观藏书楼，辅助购买珍稀典籍和文献。<ref>{{cite web|url=http://www.cndaxue.cn/news/20110925547.html|title=福州大学举行清祥教育基金资助西观藏书楼签约仪式|accessdate=2011-09-25|publisher=中国大学联盟|work=|language=zh-cn}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
