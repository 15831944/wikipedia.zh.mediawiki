'''福佑路清真寺'''，又叫'''北寺'''，是[[中国|中国]][[上海|上海]][[黄浦区|黄浦区]]的一家[[清真寺|清真寺]]。

== 历史 ==
该清真寺建成前，来自[[南京|南京]]的穆斯林已租下两间房用于祈祷。1870年，一组31人的穆斯林筹钱买下400 m<sup>2</sup>的土地，在[[清朝|清朝]][[同治|同治]]九年建造了福佑路清真寺。该寺后来在1897年、1905年、1936年和1979年分别数次重建、扩建。<ref>{{cite web|url=http://www.leadtochina.com/travel/muslim-mosques-guide/shanghai-fuyou-road-mosque-56 |title=Shanghai Fuyou Road Mosque |publisher= |deadurl=yes |archiveurl=https://web.archive.org/web/20150116101316/http://www.leadtochina.com/travel/muslim-mosques-guide/shanghai-fuyou-road-mosque-56 |archivedate=2015-01-16 |df= }}</ref>

== 建筑 ==
[[File:Fuyou_Road_Mosque_-_Prayer_Hall.jpg|thumb]]
该寺占地1,520 m<sup>2</sup>，位于一家传统清朝风格的三层建筑。建筑包含了祈祷厅、观众厅、院长室、图书馆、会议厅、浴室和其他设施。该处入口处后有一个长方形的庭院。在庭院北部是沐浴房，南部是祈祷厅。<ref>{{cite web|url=http://www.idealshanghai.com/focus/1789/|title=A window on city's Islamic heritage - iDEALShanghai|publisher=|accessdate=2015-01-15|archiveurl=https://web.archive.org/web/20150116101326/http://www.idealshanghai.com/focus/1789/|archivedate=2015-01-16|deadurl=yes}}</ref>

望月亭建在屋顶上。寺内也有各种图案风格和在屋顶横梁上画的镂空地板的装饰。<ref>{{cite web|url=http://www.muslim2china.com/china-mosques/Shanghai-Fuyou-Road-Mosque-38.html|title=Shanghai Fuyou Road Mosque|publisher=}}</ref>

== 活动 ==
该寺曾是上海穆斯林的政治、宗教和文化中心。它也是上海当代历史上第一家伊斯兰学校务本小学的诞生地。但现在它是祈祷和该城其他穆斯林活动的服务中心。它是1909年创立的上海伊斯兰董事会的所在地。
