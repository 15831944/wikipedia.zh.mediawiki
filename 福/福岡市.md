'''福冈市'''（{{jpn|j=福岡市|hg=ふくおかし|rm=Fukuoka shi|f=y}}）是位於[[日本|日本]][[九州_(日本)|九州]]的北部、[[福岡縣|福岡縣]]西部的一個都市，是福岡縣的[[縣廳所在地|縣廳所在地]]，同時為[[政令指定都市|政令指定都市]]之一<ref>{{cite web |url=http://www.pref.fukuoka.lg.jp/contents/shichoson01.html |script-title=ja:福岡県内の市町村 |accessdate=2017-08-19 |publisher={{lang|ja|福岡県の公式サイト}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170819020133/http://www.pref.fukuoka.lg.jp/contents/shichoson01.html |archivedate=2017-08-19 }}</ref>。福岡市拥有150万以上的人口，是九州最大都市與政治、經濟、交通中心，也是[[大阪市|大阪市]]以西人口最多的城市。以福岡市為中心的{{link-ja|福岡都市圈|福岡都市圏}}聚集了眾多政府機關和企業分公司，是日本人口第五多的都市圈，並和{{link-ja|北九州都市圈|北九州都市圏}}共同構成了[[北九州－福岡都市圈|北九州－福岡都市圈]]。

福岡市臨{{link-ja|博多灣|博多湾}}，古稱[[博多|博多]]，是前往東亞大陸的玄關，自中世就形成了商人的自治城市。17世紀初期[[黑田氏|黑田氏]]興建[[福岡城|福岡城]]之後，現福岡市區呈現以{{link-ja|那珂川 (九州)|那珂川 (九州)|那珂川}}為界，西側是以[[武士_(日本)|武士]]為主的[[城下町|城下町]]「福岡」，東側是以商人為主的「博多」的格局。1876年，福岡和博多合併，改稱福博（{{lang|ja|ふくはく}}），並之後再改名為福岡。在福岡設市時，兩地亦因市名而展開激烈紛爭，在議會的市名投票中兩派亦得票相同，最終因議會議長認為應繼續使用舊名而維持現狀至今<ref>{{cite web |url=http://gikai.city.fukuoka.lg.jp/wordpress/wp-content/uploads/2015/01/gikaishi1.pdf |script-title=ja:（１）市名変更問題（「福岡市」か，「博多市」か） |accessdate=2017-07-02 |publisher={{lang|ja|福岡市議会}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20160329164211/http://gikai.city.fukuoka.lg.jp/wordpress/wp-content/uploads/2015/01/gikaishi1.pdf |archivedate=2016-03-29 |title=存档副本 }}</ref>。但福岡市中心最大的車站名為[[博多站|博多站]]，博多這一地名仍相當常用。現在福岡市的[[天神_(福岡市)|天神]]、博多雙中心格局仍是[[江戶時代|江戶時代]]兩地並立格局的延續。

在[[明治維新|明治維新]]時期，[[長崎市|長崎市]]、[[鹿兒島市|鹿兒島市]]的人口均超過福岡市，且九州地方的大多數中央政府的分支機構設在位於九州中部的[[熊本市|熊本市]]，因此福岡市在九州的地位並不像現在這樣高。然而隨著1899年[[博多港|博多港]]的開港和1911年九州帝國大學（現[[九州大學|九州大學]]的前身）的設立，福岡市迅速發展，逐漸發展為九州的中心城市。1963年，由於北九州地區五市合併為[[北九州市|北九州市]]，福岡市一度是福岡縣人口第二多城市；但之後隨著重工業的低迷和服務業發展，福岡市人口在1979年再次超過北九州市，成為九州人口最多城市。現在福岡市人口仍在增加。2015年人口普查時，福岡市人口超過[[神戶市|神戶市]]，在政令指定都市中排名第五位<ref>{{cite web |url=http://mainichi.jp/articles/20160219/k00/00m/040/012000c |script-title=ja:福岡市が５位に　神戸市を抜く |accessdate=2017-07-02 |publisher={{lang|ja|毎日新聞}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228172157/http://mainichi.jp/articles/20160219/k00/00m/040/012000c |archivedate=2017-12-28 }}</ref>。

福岡市地理位置靠近[[東亞|東亞]]大陸，距[[首爾|首爾]]的直線距離和與[[近畿地方|近畿地方]]的距離相當，距[[上海市|上海]]的距離和[[東京|東京]]距離相當<ref>{{cite web |url=http://kodomo.nishinippon.co.jp/quiz/20120520/20120520_0001.shtml |script-title=ja:福岡（ふくおか）―東京間と福岡―上海（シャンハイ）間、距離が近いのは？ |accessdate=2017-07-02 |publisher={{lang|ja|西日本新聞}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228172218/http://kodomo.nishinippon.co.jp/quiz/20120520/20120520_0001.shtml |archivedate=2017-12-28 }}</ref>。福岡市自古就是日本和東亞大陸交流的窗口，在這裡發現了[[漢光武帝|漢光武帝]]贈與[[奴国|奴國]]的「[[漢倭奴國王印|漢委奴國王]]」金印。現在福岡市也有眾多前往東亞主要都市的定期航線，和東亞各國的交流十分活躍。

福岡市的市徽是在1909年10月制定的，由九個[[片假名|片假名]]「{{lang|ja|フ}}」所組成，代表福岡的「福」字<ref>{{cite web |url=http://www.city.fukuoka.lg.jp/shisei/profile/02.html |script-title=ja:市章 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170704021356/http://www.city.fukuoka.lg.jp/shisei/profile/02.html |archivedate=2017-07-04 }}</ref>。福岡市的市樹是[[鐵冬青|鐵冬青]]，廣場之樹是[[樟樹|樟樹]]。夏季之花是[[木芙蓉|木芙蓉]]，冬季之花是[[茶梅|茶梅]]。市樹、市花在1979年設市90週年紀念時制定。福岡市的山之鳥是[[草鵐|草鵐]]，海之鳥是[[紅嘴鷗|紅嘴鷗]]，在1989年設市100週年紀念時制定<ref>{{cite web |url=http://www.city.fukuoka.lg.jp/shisei/profile/08.html |script-title=ja:市の木・花・鳥 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170806090105/http://www.city.fukuoka.lg.jp/shisei/profile/08.html |archivedate=2017-08-06 }}</ref>。

=== 古代歷史 ===
福岡市自[[舊石器時代|舊石器時代]]就有人居住，博多區的諸岡遺跡<ref>{{cite web |url=http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/76 |script-title=ja:諸岡遺跡 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228232131/http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/76 |archivedate=2017-12-28 }}</ref>和西區的吉武遺跡均是舊石器時代的遺跡。糸島半島東部的大原D遺跡是[[繩文時代|繩文時代]]初期的遺跡，發現了豎穴住居<ref>{{cite web |url=http://sitereports.nabunken.go.jp/ja/17959 |script-title=ja:大原D遺跡群 |accessdate=2017-07-03 |publisher={{lang|ja|全国遺跡報告総覧}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228172113/http://sitereports.nabunken.go.jp/ja/17959 |archivedate=2017-12-28 }}</ref>。南區的柏原遺跡則發現了繩文時代初期至後期的土器<ref>{{cite web |url=http://sitereports.nabunken.go.jp/ja/17136 |script-title=ja:柏原遺跡群Ⅱ |accessdate=2017-07-03 |publisher={{lang|ja|全国遺跡報告総覧}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228232114/http://sitereports.nabunken.go.jp/ja/17136 |archivedate=2017-12-28 }}</ref>。福岡市內繩文時代中期的遺跡有早良區的有田遺跡等地<ref>{{cite web |url=http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/22 |script-title=ja:有田遺跡群 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228172246/http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/22 |archivedate=2017-12-28 }}</ref>。繩文時代後期，福岡市已出現元岡瓜尾貝塚等[[貝塚|貝塚]]遺跡<ref>{{cite web |url=http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/438 |script-title=ja:元岡瓜尾貝塚 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228172024/http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/438 |archivedate=2017-12-28 }}</ref>。

[[繩文時代|繩文時代]]後期，福岡市已出現農耕文化，在{{link-ja|板付遺跡|板付遺跡}}發現了水稻種植的痕跡<ref>{{cite web |url=http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/430 |script-title=ja:板付遺跡 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170621175236/http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/430 |archivedate=2017-06-21 }}</ref>。發現於[[志賀島|志賀島]]的漢委奴國王印更證明在1世紀時，福岡已和東亞大陸有密切交流<ref>{{cite web |url=http://museum.city.fukuoka.jp/gold/ |script-title=ja:金印 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市博物館}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170704223821/http://museum.city.fukuoka.jp/gold/ |archivedate=2017-07-04 }}</ref>。福岡市內發現了那珂八幡古墳<ref>{{cite web |url=http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/item_id:101553 |script-title=ja:那珂八幡古墳 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170902121824/http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/item_id:101553 |archivedate=2017-09-02 }}</ref>、梅林古墳<ref>{{cite web |url=http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/447 |script-title=ja:梅林古墳 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170803104304/http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/447 |archivedate=2017-08-03 }}</ref>、今宿大塚古墳<ref>{{cite web |url=http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/9 |script-title=ja:今宿古墳群（大塚古墳） |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170824201247/http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/9 |archivedate=2017-08-24 }}</ref>等{{link-ja|前方後圓墳|前方後円墳}}。

[[File:Fukuoka-jo_tenshudai_soseki.jpg|thumb]]天守台]]
在《日本書紀》中記載有527年[[磐井之亂|磐井之亂]]後，[[大和王權|大和王權]]在福岡興建那津官家以儲備糧食的記載<ref>{{cite web |url=http://museum.city.fukuoka.jp/archives/leaflet/269/index.html |script-title=ja:那津官家とその時代 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市博物館}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171204222924/http://museum.city.fukuoka.jp/archives/leaflet/269/index.html |archivedate=2017-12-04 }}</ref>。博多的總鎮守[[櫛田神社_(福岡市)|櫛田神社]]興建於8世紀中期<ref>{{cite web |url=http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/88 |script-title=ja:櫛田神社 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170621142630/http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/88 |archivedate=2017-06-21 }}</ref>。自[[奈良時代|奈良時代]]至[[平安時代|平安時代]]，博多因其地理位置而是日本和東亞大陸交流的窗口。福岡不僅是[[遣唐使|遣唐使]]的出發地之一，並且還建有接待外國使節的{{link-ja|鴻臚館|鴻臚館}}，是當時日本最大的國際交流據點<ref>{{cite web |url=http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/61 |script-title=ja:鴻臚館跡 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170621223943/http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/61 |archivedate=2017-06-21 }}</ref>。平安時代中期至鎌倉時代，福岡是[[宋日貿易|中日貿易]]的主要據點之一<ref>{{cite web |url=http://museum.city.fukuoka.jp/archives/leaflet/337/index.html |script-title=ja:平安時代の福岡－和魂漢才－ |accessdate=2017-07-03 |publisher={{lang|ja|福岡市博物館}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171204222916/http://museum.city.fukuoka.jp/archives/leaflet/337/index.html |archivedate=2017-12-04 }}</ref>。

靠近大陸的地理位置也使得福岡在古代多次遭到外敵入侵。1019年（[[寬仁|寬仁]]3年），[[女真族|女真族]]曾入侵福岡海岸地區，史稱[[刀伊入寇|刀伊入寇]]<ref>{{cite web |url=http://guides.lib.kyushu-u.ac.jp/content.php?pid=672657&sid=5570511 |script-title=ja:はじめに |accessdate=2017-07-03 |publisher={{lang|ja|九州大学}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228171845/http://guides.lib.kyushu-u.ac.jp/content.php?pid=672657&sid=5570511 |archivedate=2017-12-28 }}</ref>。蒙古帝國更在1274年（[[文永|文永]]11年）和1281年（[[弘安|弘安]]4年）兩次攻擊福岡地區。現在福岡市內仍有當時防禦蒙古軍隊工事的遺跡<ref>{{cite web |url=http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/36 |script-title=ja:元寇防塁（西新地区） |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228171906/http://bunkazai.city.fukuoka.lg.jp/cultural_properties/detail/36 |archivedate=2017-12-28 }}</ref>。

在[[室町時代|室町時代]]，博多是一座由商人（年行司）自治的自由城市，商業繁榮<ref>{{cite web |url=http://www.ncbank.co.jp/corporate/chiiki_shakaikoken/furusato_rekishi/hakata/038/01.html |script-title=ja:No.38 お櫛田さんと博多町人 |accessdate=2017-07-03 |publisher={{lang|ja|西日本シティ銀行}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228172658/http://www.ncbank.co.jp/corporate/chiiki_shakaikoken/furusato_rekishi/hakata/038/01.html |archivedate=2017-12-28 }}</ref>。但這也使得博多成為戰國大名激烈爭奪的地區。[[豐臣秀吉|豐臣秀吉]]在[[九州征伐|佔領九州]]之後，於博多進行大規模都市重劃，被稱為「太閣町割」，使得博多在經歷戰火之後得以快速復興。[[關原之戰|關原之戰]]後，[[黑田氏|黑田氏]]被分封至[[筑前國|-{筑}-前國]]並在現福岡市中心興建福岡城，福岡也因此開始出現以那珂川為界，西側是以福岡城為中心的城下町福岡，東側是商業都市博多的雙子城市格局<ref>{{cite web |url=http://www.ncbank.co.jp/corporate/chiiki_shakaikoken/furusato_rekishi/hakata/016/01.html |script-title=ja:No.16 城下町 福岡の町並み |accessdate=2017-07-03 |publisher={{lang|ja|西日本シティ銀行}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228172518/http://www.ncbank.co.jp/corporate/chiiki_shakaikoken/furusato_rekishi/hakata/016/01.html |archivedate=2017-12-28 }}</ref>。

=== 近現代歷史 ===
[[File:Seaside-momochi.JPG|thumb]]
[[廢藩置縣|廢藩置縣]]後，福岡成為福岡縣的縣廳所在地。1889年，福岡正式設[[市制_(日本制度)|市]]，是日本最早設市的城市之一。同年還開通了博多至久留米之間的鐵路<ref name="power">{{cite web |url=http://www.city.fukuoka.lg.jp/promo/magazine/history.html |script-title=ja:福岡市のできごと |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20170902153115/http://www.city.fukuoka.lg.jp/promo/magazine/history.html |archivedate=2017-09-02 }}</ref>。19世紀末期至20世紀初期，福岡市相繼出現了電燈、電話、瓦斯等新事物，象徵福岡市開始近代化<ref name="power"></ref>。1910年路面電車的開通更極大促進了福岡市內商業的發展<ref>{{cite web |url=http://www.nishitetsu.co.jp/museum/rosen/rosen06.html |script-title=ja:福岡市内線 |accessdate=2017-07-03 |publisher={{lang|ja|西日本鉄道}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170624031154/http://www.nishitetsu.co.jp/museum/rosen/rosen06.html |archivedate=2017-06-24 }}</ref>。1920年代初期，福岡市實施了西南部土地區劃整理事業，這是福岡市歷史上首個近代都市計劃，亦大幅改善了市內交通並擴大了城市規模<ref name="Hyakunen">{{cite book |author={{lang|ja|平岡昭利}}|script-title=ja:《九州 地図で読む百年》 |year=1997 |publisher={{lang|ja|古今書院}} |location=東京都 |isbn=4-7722-1665-0|language=ja}}</ref>{{rp|1}}。1927年，福岡市舉辦了東亞勸業博覽會<ref>{{cite web |url=https://www.ncbank.co.jp/corporate/chiiki_shakaikoken/furusato_rekishi/hakata/046/01.html |script-title=ja:No.46 大濠公園 |accessdate=2017-07-03 |publisher={{lang|ja|西日本シティ銀行}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228172735/https://www.ncbank.co.jp/corporate/chiiki_shakaikoken/furusato_rekishi/hakata/046/01.html |archivedate=2017-12-28 }}</ref>。1933年時，福岡市人口超過27萬，成為九州人口最多城市<ref name="power"></ref>。在[[第二次世界大戰|第二次世界大戰]]期間，福岡市也遭到美軍的空襲。1945年6月19日的{{link-ja|福岡大空襲|福岡大空襲}}造成超過900人遇難，市中心化為焦土<ref name="kushu">{{cite web |url=http://www.soumu.go.jp/main_sosiki/daijinkanbou/sensai/situation/state/kyushu_01.html |script-title=ja:福岡市における戦災の状況（福岡県） |accessdate=2017-07-03 |publisher={{lang|ja|総務省}} |language=ja}}</ref>。

福岡市在二戰之後快速開始復興，1949年時人口已超過戰前的最高值<ref name="kushu"></ref>。1972年，福岡市升格為[[政令指定都市|政令指定都市]]，成為九州第二個政令市<ref>{{cite web |url=http://toshokan.city.fukuoka.lg.jp/files/Event/Event_728_flyer_filename.pdf |script-title=ja:「政令指定都市へのあゆみ」展 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228171837/http://toshokan.city.fukuoka.lg.jp/files/Event/Event_728_flyer_filename.pdf |archivedate=2017-12-28 }}</ref>。3年之後的1975年，福岡市人口突破百萬大關<ref name="power"></ref>。同年博多站至[[東京站|東京站]]之間的[[山陽新幹線|山陽新幹線]]全線通車，使得福岡市與東京之間的陸路交通時間距離縮短至7小時以內<ref name="power"></ref>。1981年，福岡市首條地鐵通車<ref>{{cite web |url=https://subway.city.fukuoka.lg.jp/subway/about/histry.php |script-title=ja:地下鉄のあゆみ |accessdate=2017-07-03 |publisher={{lang|ja|福岡市交通局}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170928160539/http://subway.city.fukuoka.lg.jp/subway/about/histry.php |archivedate=2017-09-28 }}</ref>。1989年，為紀念設市100週年，福岡市舉辦了{{link-ja|亞洲太平洋博覽會|アジア太平洋博覧会}}，吸引了822萬人次參觀<ref name="Hyakunen"></ref>{{rp|2}}；博覽會場地之後整備為[[海濱百道|海濱百道]]開發區，這也是日本1990年代後眾多都市[[水濱|水濱]]開發計劃的先驅<ref name="Hyakunen"></ref>{{rp|2}}。2005年，福岡市發生芮氏震級達7.0的{{link-ja|福岡縣西部近海地震|福岡県西方沖地震|地震}}，市內部分地區震度達6級，是自[[阪神淡路大震災|阪神淡路大震災]]後首次有政令市記錄下震度6級的地震<ref>{{cite web |url=http://www.jma-net.go.jp/fukuoka/chosa/jikazan_saigai/20050320.html |script-title=ja:2005年3月20日の福岡県西方沖の地震 |accessdate=2017-07-03 |publisher={{lang|ja|気象庁}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228171709/http://www.jma-net.go.jp/fukuoka/chosa/jikazan_saigai/20050320.html |archivedate=2017年12月28日 }}</ref>。近年福岡市人口仍持續增加，在九州內的地位進一步提高。

== 地理 ==
[[File:Fukuoka_City_ISS045.jpg|thumb]]
福岡市位於九州北部，大部分轄區位於[[福岡平原|福岡平原]]，市轄區北至伸入{{link-ja|博多灣|博多湾}}的沙洲[[海之中道|海之中道]]及[[陸連島|陸連島]][[志賀島|志賀島]]，西至[[糸島半島|糸島半島]]。福岡市還有{{link-ja|能古島|能古島}}、{{link-ja|玄界島|玄界島}}、{{link-ja|小呂島|小呂島}}等離島<ref>{{cite web |url=https://yokanavi.com/feature/27369/ |script-title=ja:福岡から日帰りで行ける島、「能古島」と「志賀島」の観光・楽しみ方 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228172059/https://yokanavi.com/feature/27369/ |archivedate=2017-12-28 }}</ref>。為獲得平地，福岡市自20世紀開始就進行過多次大規模填海，以提供港口、住宅和商務用地<ref name="Hyakunen"></ref>{{rp|6}}。現在福岡市內大部分海岸線均是人工海岸。福岡市還在博多灣東部填海修築了人工島（{{link-ja|福岡島城|福岡アイランドシティ}}）<ref>{{cite web |url=http://island-city.city.fukuoka.lg.jp/about/shintyoku.html |script-title=ja:事業の進捗状況 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170821030651/http://island-city.city.fukuoka.lg.jp/about/shintyoku.html |archivedate=2017-08-21 }}</ref>。由於福岡市大部分地區位於福岡平原，因此地形較為平坦。但福岡市西部和西南部的部分地區屬於脊振山地，標高較高且地形起伏明顯。

[[File:Fukuoka_Viewed_From_Minamiku_Observation_Deck.jpg|thumb]]

福岡市沒有一級河川，市內的主要河川有市中心的{{link-ja|那珂川 (九州)|那珂川 (九州)|那珂川}}、{{link-ja|御笠川|御笠川}}，市區東部的{{link-ja|多多良川|多々良川}}，市區西部的{{link-ja|室見川|室見川}}<ref>{{cite web |url=http://www.city.fukuoka.lg.jp/data/open/cnt/3/29063/1/fukuoka_river.pdf |script-title=ja:福岡市の河川 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228172047/http://www.city.fukuoka.lg.jp/data/open/cnt/3/29063/1/fukuoka_river.pdf |archivedate=2017-12-28 }}</ref>。福岡市在歷史上多次遭到乾旱，1978年時曾發生限制用水日數達約300日的記錄<ref name="Hyakunen"></ref>{{rp|8}}。但另一方面，逢短時間內發生集中降雨時亦容易發生洪水。

福岡市的兩大中心地區是中央區的[[天神_(福岡市)|天神]]地區和博多站附近的博多地區，兩地之間的中洲地區則是日本著名的娛樂區。近年因填海和鐵路沿線地區的市街重劃，[[海濱百道|海濱百道]]、{{link-ja|香椎|香椎}}、{{link-ja|千早|千早 (福岡市)}}、{{link-ja|姪濱|姪浜}}等地區發展為新興商業區。由於福岡市中心有機場，因此福岡市中心對建築高度有嚴格限制，市內大多數高層建築均位於郊區<ref>{{cite web |url=http://ocab.mlit.go.jp/news/limit/pdf/08_09_05_07_fukuoka.pdf |script-title=ja:転移表面 |accessdate=2017-07-03 |publisher={{lang|ja|大阪航空局}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20180327031730/http://ocab.mlit.go.jp/news/limit/pdf/08_09_05_07_fukuoka.pdf |archivedate=2018-03-27 }}</ref>。

=== 氣候 ===
福岡市氣候溫暖，年平均氣溫約17℃，年均降水量約1,500至1,800毫米。福岡市位於日本海沿岸，並有冬季日照時間較短等日本海沿岸氣候的特徵，但由於冬季降水量明顯少於夏季，因此氣候區分上屬於[[太平洋側氣候|太平洋側氣候]]<ref>{{cite web |url=http://www.jma-net.go.jp/fukuoka/yoho/tenki_hanasi.html |script-title=ja:天気のはなし |accessdate=2017-07-03 |publisher={{lang|ja|福岡管区気象台}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20160305165553/http://www.jma-net.go.jp/fukuoka/yoho/tenki_hanasi.html |archivedate=2016-03-05 }}</ref>。福岡市的降水主要集中在[[梅雨|梅雨]]及[[颱風|颱風]]過境時期<ref>{{cite web |url=http://www.jma-net.go.jp/fukuoka/kaiyo/tenkou_main.html |script-title=ja:九州北部地方の天候の特徴（解説） |accessdate=2017-07-03 |publisher={{lang|ja|福岡管区気象台}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20160406215540/http://www.jma-net.go.jp/fukuoka/kaiyo/tenkou_main.html |archivedate=2016-04-06 }}</ref>。福岡市原本就是日本各大城市中氣溫最高的城市之一，近年隨著[[熱島現象|熱島現象]]的發展，氣溫更趨炎熱<ref>{{cite web |url=http://www.data.jma.go.jp/cpdinfo/himr/2010/chapter3.pdf |script-title=ja:３．九州北部地方のヒートアイランド現象 |accessdate=2017-07-03 |publisher={{lang|ja|気象庁}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228232156/http://www.data.jma.go.jp/cpdinfo/himr/2010/chapter3.pdf |archivedate=2017-12-28 }}</ref>。福岡市還是雷電天氣頻發的城市之一，年均雷電天氣日數達24.7日<ref>{{cite web |url=http://www.jma.go.jp/jma/kishou/know/toppuu/thunder1-1.html |script-title=ja:雷の観測と統計 |accessdate=2017-07-03 |publisher={{lang|ja|気象庁}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170724075219/http://www.jma.go.jp/jma/kishou/know/toppuu/thunder1-1.html |archivedate=2017-07-24 }}</ref>。

== 人口 ==
1889年設市時，福岡市有人口50,847人<ref name="population2">{{cite web |url=http://www.city.fukuoka.lg.jp/data/open/cnt/3/52263/1/daigonotoshini.pdf |script-title=ja:福岡市が日本第 5 の都市に！ |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170811143343/http://www.city.fukuoka.lg.jp/data/open/cnt/3/52263/1/daigonotoshini.pdf |archivedate=2017-08-11 }}</ref>。此後直到二戰，福岡市人口長期穩定增加，並在1930年代成為九州人口最多城市和日本人口第七多城市<ref name="population2"></ref>。二戰期間福岡市人口一度減少，但在戰後迅速恢復並在1955年突破50萬人大關<ref name="population2"></ref>。1972年升格為政令指定都市時，福岡市有人口約91.2萬人<ref name="population2"></ref>。1975年，福岡市人口突破百萬大關<ref name="population2"></ref>。在2010年至2015年期間，福岡市的人口增長率和增長數都是政令指定都市首位，人口數更超過京都市和神戶市，成為政令指定都市中人口第五多城市<ref name="population2"></ref>。2017年8月1日時，福岡市有人口約156.6萬人<ref name="population1"></ref>。來自九州其他地區的人口流入是福岡市人口增長的主因。由於福岡市大學眾多，市內學生人數在日本僅次於東京23區、京都市、名古屋市、橫濱市，排名第五，使得九州各地的年輕人口流入福岡市<ref name="Chishi">{{cite book |author={{lang|ja|野澤秀樹・堂前亮平・手塚章}}|script-title=ja:《日本の地誌10　九州・沖縄》 |year=2008 |publisher={{lang|ja|朝倉書店}} |location=東京都 |isbn=978-4-254-16770-2|language=ja}}</ref>{{rp|152}}。但另一方面，自九州其他地區轉入的人口亦有約三成比例在之後轉入[[首都圈_(日本)|東京地區]]<ref name="population3">{{cite web |url=http://www.city.fukuoka.lg.jp/data/open/cnt/3/49519/1/jinko-vision-soan.pdf |script-title=ja:福岡市 人口ビジョン（素案） |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170811182115/http://www.city.fukuoka.lg.jp/data/open/cnt/3/49519/1/jinko-vision-soan.pdf |archivedate=2017-08-11 }}</ref>。福岡市的[[總和生育率|總和生育率]]長期低於日本平均值，在19個政令指定都市中也只排名第16位<ref name="population3"></ref>。

2014年，福岡市的外國人人口有約2.74萬人，其中以佔比約四成的中國人最多，比例約二成的韓國人次之<ref name="population3"></ref>。

== 政治 ==

==經濟==
[[File:Canal_City_Hakata_2011.jpg|right]]

2014年度，福岡市市內生產總值達67,340億日元，人均市民所得為312萬日元。福岡市經濟的特徵之一是第三產業比重較高，擁有近八成比例。而第二產業則只有不到兩成的比例<ref>{{cite web|url=http://www.city.fukuoka.lg.jp/data/open/cnt/3/6414/1/H26gaiyou.pdf　|script-title=ja:平成26（2014）年度 福岡市民経済計算の概要|accessdate=2017-07-03|publisher={{lang|ja|福岡市}}|language=ja}}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。

=== 農業 ===
福岡市的農業屬於都市近郊型農業，在福岡市經濟總量中比例很低。福岡市耕地面積有1,860公頃，農業就業人口有2,352人。福岡市的農業產值中以雞蛋的比例最高，其次則是蔬菜<ref>{{cite web |url=http://www.machimura.maff.go.jp/machi/contents/40/130/index.html |script-title=ja:福岡県福岡市 |accessdate=2017-07-03 |publisher={{lang|ja|農林水産省}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228171926/http://www.machimura.maff.go.jp/machi/contents/40/130/index.html |archivedate=2017-12-28 }}</ref>。

=== 工業 ===
博多自古代開始就生產{{link-ja|博多人形|博多人形}}<ref>{{cite web |url=http://www.hakataningyou.com/joho/yurai.html |script-title=ja:伝統工芸品～博多人形について |accessdate=2017-07-04 |publisher={{lang|ja|人形のごとう}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170705003609/http://hakataningyou.com/joho/yurai.html |archivedate=2017-07-05 }}</ref>、{{link-ja|博多織|博多織}}<ref>{{cite web |url=http://www.hakataori.or.jp/contents/denrai.html |script-title=ja:博多織とは |accessdate=2017-07-04 |publisher={{lang|ja|博多織工業組合}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228172214/http://www.hakataori.or.jp/contents/denrai.html |archivedate=2017-12-28 }}</ref>等傳統工藝品，工業歷史深厚。但在福岡縣產業近代化的過程中，擁有煤炭資源的北九州地區成為主要的重工業地區，而福岡市則憑藉服務業得到發展，因此工業在福岡市經濟中比重較低。2014年，福岡市工業品生產額有6,365億日元，其中以佔三分之一比例的食品製造業比重最大，其次是佔約兩成比例的電子零件業、佔比14.8%的飲料·煙草·飼料業、佔比8.8%的印刷業<ref name="industry">{{cite web |url=http://www.city.fukuoka.lg.jp/data/open/cnt/3/53179/1/gaiyou.pdf |script-title=ja:平成 26 年工業統計調査結果の概要 |accessdate=2017-07-03 |publisher={{lang|ja|福岡市}} |language=ja }}{{dead link|date=2018年3月 |bot=小躍 |fix-attempted=yes }}</ref>。就地理分佈來看，博多區、西區、東區是福岡市主要的工業區，三者的工業品產值均佔福岡市總量的二成以上<ref name="industry"></ref>。

==文化==
===語言===
福岡市使用的方言主要是{{link-ja|博多方言|博多弁}}，屬於[[九州方言|九州方言]]中的{{link-ja|筑前方言|筑前方言|-{筑前方言}-}}，在重音上接近{{link-ja|東京式重音|東京式アクセント}}。福岡市是是日本方言意識最強的城市之一，當地民眾對自身方言的喜好度極高<ref>{{cite book |author={{lang|ja|陣内正敬}}|script-title=ja:《福岡県のことば》 |year=1996 |publisher={{lang|ja|明治書院}} |location=東京都 |isbn=4-625-52240-4|language=ja}}</ref>{{rp|7}}。

===飲食===
福岡市的飲食文化十分發達。[[博多拉麵|博多拉麵]]是[[日本拉麵|日本拉麵]]中豚骨口味的代表，以極細的麵條和濃厚的湯汁而著稱，並擁有獨特的途中加麵文化<ref>{{cite web |url=http://www.raumen.co.jp/rapedia/study_japan/study_raumen_hakata.html |script-title=ja:博多ラーメン |accessdate=2017-07-03 |publisher={{lang|ja|新横浜ラーメン博物館}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170704001945/http://www.raumen.co.jp/rapedia/study_japan/study_raumen_hakata.html |archivedate=2017-07-04 }}</ref>。以牛或豬的腸道為主要材料的[[腸鍋|腸鍋]]<ref>{{cite web |url=http://www.i-fukuoka.jp/gourmet/motunabe/motsunabetoha.html |script-title=ja:博多もつ鍋とは |accessdate=2017-07-03 |publisher={{lang|ja|とくなび福岡}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228232043/http://www.i-fukuoka.jp/gourmet/motunabe/motsunabetoha.html |archivedate=2017-12-28 }}</ref>和以雞肉、蔬菜為主要材料的{{link-ja|水炊|水炊き}}<ref>{{cite web |url=http://www.i-fukuoka.jp/gourmet/mizutaki/mizutakitoha.html |script-title=ja:博多水炊きとは |accessdate=2017-07-03 |publisher={{lang|ja|とくなび福岡}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170806182009/http://www.i-fukuoka.jp/gourmet/mizutaki/mizutakitoha.html |archivedate=2017-08-06 }}</ref>是福岡市發祥的火鍋類料理。[[屋台|屋台]]（露天店鋪）眾多亦是福岡市飲食文化的一大特徵<ref>{{cite web |url=https://tenjinsite.jp/yatai/history.php |script-title=ja:屋台の歴史 |accessdate=2017-07-03 |publisher={{lang|ja|天神屋台の歩き方}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170630142643/https://tenjinsite.jp/yatai/history.php |archivedate=2017-06-30 }}</ref>。 
===節日慶典===
福岡市有三大節慶活動，依序為每年5月3日、4日舉辦的{{link-ja|博多Dontaku|博多どんたく}}、每年7月前半期舉行的{{link-ja|博多祇園山笠|博多祇園山笠}}、每年9月12日至18日舉行的[[筥崎宮|筥崎宮]][[放生|放生會]]。

博多Dontaku起源於1179年開始舉行的松囃子，已有超過830年的歷史。自1962年開始，博多Dontaku成為由福岡市民舉辦的大型節慶活動，每年有超過3.3萬人直接參加和超過200萬人觀看，是日本規模最大的節慶活動之一<ref>{{cite web |url=http://www.dontaku.fukunet.or.jp/about/origin |script-title=ja:博多どんたくとは |accessdate=2017-07-03 |publisher={{lang|ja|博多どんたく}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170609091644/http://www.dontaku.fukunet.or.jp/about/origin |archivedate=2017-06-09 }}</ref>。

[[File:Hakata_gion_yamakasa_2005_01.jpg|left]]
博多祇園山笠是[[櫛田神社_(福岡市)|櫛田神社]]的祇園祭，已有超過760年的歷史，以巨大的花車而著稱，在2016年被聯合國教科文組織列入世界非物質文化遺產<ref>{{cite web |url=http://www.hakata-yamakasa.net/knowledge/history/yamakasa-history/ |script-title=ja:博多祇園山笠の由来 |accessdate=2017-07-03 |publisher={{lang|ja|山笠ナビ運営事務局}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170629124629/http://www.hakata-yamakasa.net/knowledge/history/yamakasa-history/ |archivedate=2017-06-29 }}</ref>。

筥崎宮放生會已有超過千年的歷史。在放生會舉辦期間，除了放生動物的活動之外，筥崎宮附近還聚集了約500家露天店鋪，吸引眾多市民參與，是九州秋季規模最大的節慶活動之一<ref>{{cite web |url=http://www.hakozakigu.or.jp/omatsuri/houjoya/ |script-title=ja:お祭り |accessdate=2017-07-03 |publisher={{lang|ja|筥崎宮}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170701205729/http://hakozakigu.or.jp/omatsuri/houjoya/ |archivedate=2017-07-01 }}</ref>。

[[博多港|博多港]]開港於1899年，是日本海沿岸的據點港灣之一<ref>{{cite web |url=http://port-of-hakata.city.fukuoka.lg.jp/profile/port-history/index.html |script-title=ja:博多港の歴史 |accessdate=2017-07-03 |publisher={{lang|ja|博多港}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170711125852/http://port-of-hakata.city.fukuoka.lg.jp/profile/port-history/index.html |archivedate=2017-07-11 }}</ref>。2015年，博多港的入港船舶數超過3萬艘，貨物吞吐量達3,133萬噸<ref>{{cite web |url=http://port-of-hakata.city.fukuoka.lg.jp/profile/data/pdf/h27kakutei.pdf |script-title=ja:博多港統計年報 |accessdate=2017-07-03 |publisher={{lang|ja|博多港}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228172247/http://port-of-hakata.city.fukuoka.lg.jp/profile/data/pdf/h27kakutei.pdf |archivedate=2017-12-28 }}</ref>。博多港還是日本國際旅客數最多的港口，開設有前往釜山港的定期國際航線<ref>{{cite web |url=http://port-of-hakata.city.fukuoka.lg.jp/guide/terminal/ |script-title=ja:国際旅客のご案内 |accessdate=2017-07-03 |publisher={{lang|ja|西鉄バス}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20171228232134/http://port-of-hakata.city.fukuoka.lg.jp/guide/terminal/ |archivedate=2017-12-28 }}</ref>。2016年，博多港的國際郵輪停靠次數達328次，同樣是日本首位<ref>{{cite web |url=http://www.mlit.go.jp/report/press/port04_hh_000163.html |script-title=ja:2016年の訪日クルーズ旅客数とクルーズ船の寄港実績（速報値） |accessdate=2017-07-03 |publisher={{lang|ja|国土交通省}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170329210834/http://www.mlit.go.jp/report/press/port04_hh_000163.html |archivedate=2017-03-29 }}</ref>。

[[福岡機場|福岡機場]]又名板付機場，於1945年開始使用，最初是軍用機場，在1951年開通民航航線，跑道全長2,800米<ref>{{cite web |url=http://www.fuk-ab.co.jp/gaiyou/ |script-title=ja:福岡空港について |accessdate=2017-07-03 |publisher={{lang|ja|福岡空港}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20170701151249/http://www.fuk-ab.co.jp/gaiyou/ |archivedate=2017年7月1日 |df= }}</ref>。福岡機場不僅開通有前往日本各主要機場的定期航線<ref>{{cite web |url=http://www.fuk-ab.co.jp/network_dom/ |script-title=ja:定期路線のご案内（国内線・就航路線） |accessdate=2017-07-03 |publisher={{lang|ja|福岡空港}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20170705121636/http://www.fuk-ab.co.jp/network_dom/ |archivedate=2017年7月5日 |df= }}</ref>，還開設有前往首爾、北京、上海、香港、台北、高雄、馬尼拉、曼谷、新加坡等東亞主要都市及[[丹尼爾·井上國際機場|檀香山]]的國際航線<ref>{{cite web |url=http://www.fuk-ab.co.jp/network_int/ |script-title=ja:定期路線のご案内（国際線・就航路線） |accessdate=2017-07-03 |publisher={{lang|ja|福岡空港}} |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20170704235339/http://www.fuk-ab.co.jp/network_int/ |archivedate=2017年7月4日 |df= }}</ref>。地處市中心的地理位置使得福岡機場的聯外交通極為便利，客流量也因此較多。2014年度，福岡機場的乘客數達2,000萬人次，位居日本第三<ref>{{cite web |url=http://www.pref.fukuoka.lg.jp/contents/kuko2.html |script-title=ja:空港計画課ホームページ |accessdate=2017-07-03 |publisher={{lang|ja|福岡県}} |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20170704021436/http://www.pref.fukuoka.lg.jp/contents/kuko2.html |archivedate=2017-07-04 }}</ref>。
