'''福斯托里亞'''（{{lang-en|Fostoria}}）是[[美國|美國]][[堪薩斯州|堪薩斯州]][[波特瓦特米縣_(堪薩斯州)|波特瓦特米縣]]一個[[非建制地區|非建制地區]]，距離奧爾斯堡（Olsburg）以東約{{convert|6|mi|km}}，海拔高度為1,473英尺（449米）。<ref name="GNIS">{{Cite GNIS | 476080 | 堪萨斯州福斯托里亞}}{{en}}</ref>當地的郵政局於1884年開始營業，其[[美國郵遞區號|郵遞區號]]為66426。<ref>{{cite web | title=Search Results for ZIP code 66426 | url=http://www.zipinfo.com/cgi-local/zipsrch.exe?cnty=cnty&zip=66426&Go=Go | publisher=Zip-Info.com | accessdate=2017年12月12日 | language=en}}</ref><ref>{{cite web | url=http://www.kshs.org/geog/geog_postoffices/search/county:PT | title=Kansas Post Offices, 1828-1961 | publisher=Kansas Historical Society | language=en | accessdate=2017年12月12日 | deadurl=yes | archiveurl=https://web.archive.org/web/20131009123121/http://www.kshs.org/geog/geog_postoffices/search/county%3APT | archivedate=2013年10月9日}}</ref>

夏季濕熱，冬季暖和涼快為福斯托里亞之氣候特徵。另外根據[[柯本氣候分類法|柯本氣候分類法]]系統，當地屬於[[副熱帶溼潤氣候|副熱帶溼潤氣候]]，氣候地圖上以「Cfa」標示。<ref>{{cite web | language=en | url=http://www.weatherbase.com/weather/weather-summary.php3?s=917841&cityname=Fostoria%2C+Kansas%2C+United+States+of+America&units= | title=Climate Summary for Fostoria, Kansas | publisher=Weather Base | accessdate=2017年12月12日}}</ref>
