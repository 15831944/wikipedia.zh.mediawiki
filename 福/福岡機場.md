'''福岡機場'''（{{jpn|j=福岡空港|kn=ふくおかくうこう|rm=Fukuoka Kūkō}}；{{airport codes|FUK|RJFF}}）是一個位於[[日本|日本]][[福岡縣|福岡縣]][[福岡市|福岡市]][[博多區|博多區]]的國際與國內[[機場|機場]]。因該機場曾為[[美國空軍|美國空軍]]所駐紮之板付基地，所以又曾稱為'''板付機場'''（{{lang|en|Itazuke Field}}）。

== 歷史 ==
* 1944年2月：[[日本陸軍|日本陸軍]]的席田機場開始建設。
* 1945年5月：跑道完成。
* 1945年10月：美軍接收板付基地。
* 1951年10月：開設民航航線（福岡-[[大阪|大阪]]-[[東京|東京]]）。
* 1965年9月：開設國際航線（福岡-[[釜山|釜山]]）。
* 1970年12月：機場的管轄權移交到運輸省。
* 1972年4月：變更為第二種機場。
* 1993年3月：[[福岡市地下鐵|福岡市地下鐵]][[機場線_(福岡市地下鐵)|機場線]]通車，提供轉乘服務。
* 1999年5月：新國際線客運大樓完成，舊國際線大樓改稱為第三航運大樓。
* 2004年1月：34跑道的[[儀器著陸系統|ILS]]啟用。
*2015年6月：開始改建新國內線航運大樓，第一、第二航運大樓一部分將拆除，第二大樓剩餘部分將與第三大樓合併為地上五層、地下兩層的大樓。
*2016年3月：被指定為混雜機場。
*2016年10月：第一航運大樓關閉，第二、第三航運大樓合稱為國內線航運大樓。
*2019年4月：計劃將機場運營私有化。

== 簡介 ==
[[File:FUK_airport_diagram.png|thumb]]
福岡機場到的福岡市中心交通相當便利。可以轉乘[[福岡市地下鐵機場線|福岡市地鐵機場線]]，到JR[[博多站|博多站]]約5分鐘（約4公里）。到福岡市中心[[天神車站|天神]]約11分鐘（約12公里）。利用福岡都市高速公路30分鐘左右也能到太宰府。

航運大樓共有日本國內線（舊第二、第三）和國際線共三棟，貨物大樓國內線，國際線的兩棟。國內線航運大樓位於飛行[[跑道_(航空)|跑道]]東側，1999年5月開始營運的國際線航運大樓則位於飛行跑道西側，約10分鐘有一班間隔免費的接駁巴士運行。

福岡機場雖然只有一條2800公尺的跑道，跑道的起降率跟同樣的第二種機場比較起來是最多的，每年起降次數達到14萬次（日本國內線12.6萬次，國際線1.4萬次）。

日本國內航班的福岡─[[東京國際機場|東京]]線接著[[新千歲|新千歲]]─東京線乘客人數多。大阪和名古屋的航班數也持續增加，與東海道・山陽新幹線和高速巴士競爭激烈。

國内線的航空路線有25條，1年的旅客人數約有1632萬人（乘客和降落人數合計是2004年日本第5位）。國內線和國際線的旅客人數約有1856萬人（2004年）。本機場的旅客人數是繼羽田和成田的日本第3位。

另一方面本機場亦是日本最初設置[[登機橋|登機橋]]的機場。

== 意外事件 ==
*1970年3月31日，[[淀号劫机事件|日本航空351號班機]]由東京飛往福岡，被[[日本赤軍|日本赤軍]]劫持到朝鮮。
*1989年12月16日，[[中國民航981號班機劫機事件|中國民航981號班機]]由北京飛往上海途中被劫持，最後被迫降落在福岡機場。
*1996年6月13日，[[加魯達航空865號班機|加魯達航空865號班機]]由福岡飛往雅加達，起飛時引擎發生故障，導致飛機墜毀，3人死亡。
