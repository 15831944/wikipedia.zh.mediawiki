'''福鼎市'''地处中国[[福建省|福建省]]东北部沿海，是[[寧德市|寧德市]]下轄的[[縣級市|縣級市]]，北接[[浙江省|浙江省]]。

==文化==
===语言===
福鼎大多地方通行北片[[闽东语|闽东语]]。大部地区通行闽东方言的[[桐山话|桐山话]]；前岐、沙埕、贯岭、嵛山通[[浙南闽语|闽南话]]，秦屿镇区、台山岛通行[[閩東語|閩東語]][[福州话|福州话]]。

==地理==
福鼎因其境内山[[太姥山|太姥山]]之[[覆鼎峰|覆鼎峰]]而得名。福鼎市位于福建[[宁德市|宁德市]]东北沿海，有[[福建|福建]]「北大门」之称。北纬26°52'─27°26'，东经119°55'─120°43'。面积1526.3平方公里，海域面积14959.7平方公里。山地和丘陵地面积占士地面积88.1％，平原占11.9％。现有耕地28.47万亩，山地527平方公里，森林覆盖率65％。海岸线曲折，长度432.7公里，福鼎三面环山，一面临海，地势由东北、西北、西南向东南沿海倾斜。全市最高峰青龙山海拔1141米。大小岛礁达200多个，其中岛屿81个，主要有[[福瑶列岛|福瑶列岛]]、[[七星列岛|七星列岛]]、[[台山列岛|台山列岛]]；主要港湾41个，其中[[沙埕港|沙埕港]]融商贸、渔业、军事为一体，是东南沿海重要的深水避风良埠，可建数座5-10万吨泊位的码头，在[[孙中山|孙中山]]《[[建国方略|建国方略]]》中就定为天然良港。主要河流：[[桐山溪|桐山溪]]、龙山溪、交溪、赤溪、三沙溪。人口55.85万（2001），大部地区通行闽东方言的桐山话；前岐、沙埕、贯岭、嵛山通闽南方言，秦屿镇区、台山岛通福州话。

==历史==
[[乾隆|乾隆]]四年（1739年）从[[霞浦县|霞浦县]]划出劝儒乡的望海、育仁、遥香、廉江四里置福鼎县，隶属于福宁府，县治设在桐山，故称桐城。民国二年（1913年）废府，属东路观察使，民国六年（1917年）属闽海道，民国十七年（1928年）废道，省直辖，民国二十年（1931年）隶属[[福建省|福建省]]第二督察区，民国三十二年（1943年）改称第八行政督察区。1949年6月11日解放军进入福安，受辖于[[浙江省|浙江省]]第五行政公署，1949年10月受辖于福建省第三行政督察区，1950年改称为福安专区，1970年行政公署迁往[[宁德|宁德]]，1971年福鼎改属宁德地区。1995年10月13日经国务院批准撤销福鼎县设福鼎市，仍隶属于宁德地区。2000年11月撤销了宁德地区设[[宁德市|宁德市]]，福鼎现属[[宁德市|宁德市]]。

==行政区划==
福鼎市辖：[[太姥山镇|太姥山镇]]、[[前岐镇|前岐镇]]、[[点头镇|点头镇]]、[[沙埕镇|沙埕镇]]、[[白琳镇|白琳镇]]、[[店下镇|店下镇]]、[[贯岭镇|贯岭镇]]、[[磻溪镇|磻溪镇]]、[[管阳镇|管阳镇]]、[[嵛山镇|嵛山镇]]10镇；桐山街道办事处、桐城街道办事处、山前街道办事处，3街道；[[叠石乡|叠石乡]]、[[佳阳畲族乡|佳阳畲族乡]]、[[硖门畲族乡|硖门畲族乡]]3乡；龙安1个开发区管委会。26个居委会，252个行政村。市政府驻地为：桐山（别名：桐城、桐川）。

==旅游==
福鼎市旅游风景融合了山川海岛，有登山探洞、宗教朝圣、文物考古、海岛观光、海上垂钓、九鲤漂流、沙洲戏水、休闲度假等旅游项目。境内太姥山是国家级重点旅游名胜风景区，可观赏面积92平方公里，面朝大海，山、海素被称为“山海大观”，太姥山以花岗岩峰林岩洞为其特色，包括太姥山岳、晴川海滨、九鲤瀑布、福瑶列岛、桑园翠湖五大景区和[[冷城古堡|冷城古堡]]、畲寨、[[牛郎岗|牛郎岗]]海滨度假区和[[翠郊古民居|翠郊古民居]]等景点。

福鼎市最著名的旅游景点是[[太姥山|太姥山]]——[[国家级风景名胜区|国家级风景名胜区]]，被誉为“海上仙都”，太姥山位于福鼎市境内，北距浙江省温州市150公里，南距福州市250公里，山、海素被称为“山海大观”，传说诸仙每年到太姥山聚会，故又被称为“海上仙都”。传说远古时越女修真于此山中鸿雪洞，得道成仙，人称太姥娘娘，据说是八闽大地第一位女神。因此闽人称太姥、武夷为双绝。   

其次在国内较著名的是[[大嵛山岛|大嵛山岛]]，被《中国国家地理》评为中国最美十大海岛之一，大嵛山岛，直径约5公里，面积21.22平方公里，最高处红纪洞山海拔541.4米，为[[闽东|闽东]]第一大岛。

在福鼎较著名的还有[[南溪水库|南溪水库]]，南溪因其湖光山色每年都吸引着一批徒步旅行爱好者，又因养着鱼又吸引了众多钓鱼者前来。

== 经济 ==
近年来旅游业的兴起为'''福鼎市'''发展带来了契机，2002年全市共接待游客42.98万人次，比1995年增长48.7％，实现旅游营业收入1.42亿元（游客主要前往位于[[秦屿|秦屿]]的[[太姥山|太姥山]]）。

== 自然资源 ==
福鼎山海蕴藏着丰富的资源。海域面积2243.95万亩，浅海面积45.14万亩，滩涂面积10.44万亩，名优特产众多，[[玄武岩|玄武岩]]储量5000万立方米，[[白碄|白碄]]属全国建筑石材10多个出口基地之一。福鼎白茶为[[中国地理标志产品|中国地理标志产品]]<ref>[http://kjs.aqsiq.gov.cn/dlbzcpbhwz/bhml/200906/t20090622_118997.htm 《地理标志产品保护规定》实施以来批准的地理标志名录（2005年7月至2009年6月）] {{webarchive|url=https://web.archive.org/web/20110311120900/http://kjs.aqsiq.gov.cn/dlbzcpbhwz/bhml/200906/t20090622_118997.htm |date=2011-03-11 }}</ref>。

== 桐城八景 ==

桐城八景：桐山名胜素有八景之称。关于“八景之说”有二：其一，御屏积翠，莲峰曙月，罾坪渔火，石湖春涨，玉塘秋色，龙山霁雪，双髻凌云，栖林烟雨。其二，昭明夕照，栖林烟雨、圆觉晓钟、双髻凌云、罾坪渔火、石湖春涨、玉塘秋色、龙山霁雪。通常流行的桐城八景指第二种：

*双髻凌云：双髻山
*龙山霁雪：龙山 
*园觉晓钟：圆觉寺
*石湖春涨：石湖桥
*罾坪渔火：罾排渔火
*玉塘秋色：玉塘堡，塘底堡   
*昭明夕照：昭明寺
*栖林烟雨：栖林寺
