'''福寿沟'''是中國[[江西省|江西省]][[赣州市|赣州市]]老城的一座大型地下水利工程，始建于[[北宋|北宋]]时期，由时任[[虔州|虔州]]知军[[刘彝|刘彝]]主持修建。福寿沟既是赣州城区的防洪排涝工程，也是城区污水处理的综合治理工程，属于“雨污合流制”管网系统<ref>{{cite journal|author=韩振飞|date=2012-02-14|title=宋代排水工程“福寿沟”的营造|url=http://economy.guoxue.com/?p=4904|journal=《中国社会科学报》|volume=|issue=243|pages=|doi=|access-date=2018-05-03|deadurl=no|archiveurl=https://web.archive.org/web/20170812125330/http://economy.guoxue.com/?p=4904|archivedate=2017-08-12}}</ref>。在降雨量较大的时节，赣州城得益于福寿沟的排涝作用，没有发生其他城市严重内涝的现象，福寿沟经过之地均无积水点<ref name="吴锺昊"></ref>。[[1998年中国水灾|1998年中国水灾]]中，福寿沟保护了赣州城区免遭洪水侵袭<ref>{{cite web |url=http://3g.china.com/act/culture/11170621/20170222/30272188.html |title=“江南宋城”江西赣州市长：保护好福寿沟 延续城市文脉 |author1=刘占昆 |author2=苏路程 |date=2017-02-21 |website=3g.china.com |publisher=[[中新社|中新社]]赣州 |access-date=2018-05-03 |quote=}}</ref>。

==建筑==

福寿沟分为两个部分，福沟主要排城市东南部之水，集水面积约1.7平方公里，主沟长约11.6公里；寿沟主要排城市西北部之水，集水面积约1平方公里，主沟长约1公里<ref name="吴锺昊">{{cite web |url=http://www.xinhuanet.com/politics/2015-10/16/c_1116851676.htm |title=江西赣州福寿沟：一颗跳动千年的城市“良心” }}</ref>。整个系统共有12个出水口，形成了比较完整的排水干道网，排水沟呈砖拱结构，沟顶分布铜钱状的排水孔<ref name="中国青年网">{{cite web |url=http://qiyuan.youth.cn/ttxw/201207/t20120723_2283595.htm |title=江西赣州福寿沟：正在消失的宋代先进城市排水系统 |author= ==历史==

公元885年，[[南康市|南康]][[卢光稠|卢光稠]]起兵攻占[[虔州|虔州]]，开始了大规模的城建工程，同时破坏了原始地貌阻碍城区地表水进入[[贡江|贡江]]和[[章水_(赣江)|章江]]，导致城区排水不畅<ref name="吴锺昊"></ref>。公元1068年，[[刘彝|刘彝]]任虔州知军，主持规划建设了赣州城区的街道，依据街道布局采取分区排水的原则，修建了两个排水干道系统。由于排水沟网形似古篆体的“福”、“寿”二字，故此得名福寿沟<ref>{{cite web |url=http://society.people.com.cn/n/2012/1011/c1008-19225584-3.html |title=老祖宗的智慧超越现代人？ 探访江西赣州“福寿沟” |author=刘茸 |date=2012-10-11 |website=society.people.com.cn |publisher=[[人民网|人民网]] |access-date=2018-05-03 |quote= |deadurl=no |archiveurl=https://web.archive.org/web/20121014105302/http://society.people.com.cn/n/2012/1011/c1008-19225584-3.html |archivedate=2012-10-14 }}</ref>。清[[同治|同治]]八年，赣州修复了福寿沟的6个出水口<ref name="吴庆洲"></ref>。1953年，赣州在修复下水道的工程中修复了厚德路长767.6米的福寿沟<ref name="吴庆洲"></ref>。赣州市政机关曾将部分福寿沟地段挖填处理，导致赣州出现水淹的情况，2014年开始政府部门再次重视并修复福寿沟<ref name="李津逵"></ref>。

赣州将新建福寿沟博物馆，以促进遗址保护并发展旅游<ref>{{cite web |url=http://jx.ifeng.com/a/20161123/5170661_0.shtml |title=赣州将新建福寿沟博物馆 保护千年的城市“良心” |author=赣州发布 |date=2016-11-23 |website=jx.ifeng.com |publisher=赣州发布 |access-date=2018-05-03 |quote= |deadurl=no |archiveurl=https://web.archive.org/web/20180504092914/http://jx.ifeng.com/a/20161123/5170661_0.shtml |archivedate=2018-05-04 }}</ref>。
