'''福田円'''{{BD|1980年|5月2日||}}是一名[[日本|日本]][[政治学家|政治学家]]，現任[[法政大学_(日本)|法政大學]]法学部教授。専門是東亞国際政治史、現代中国・台湾論。

== 略歷 ==
出身於[[東京都|東京都]]，畢業於[[国際基督教大学|国際基督教大学]][[教養学部|教養学部]]。[[慶應義塾大学|慶應義塾大学]]大学院政策・媒體研究科博士課程修了。

擔任[[国士館大学|国士館大学]]准教授、[[法政大学|法政大学]][[法学部|法学部]]准教授、2017年起擔任現職。

== 學歴 == 
*2003年3月　[[国際基督教大学|国際基督教大学]][[教養学部|教養学部]]国際関係学科 卒業
*2005年3月　[[慶應義塾大学|慶應義塾大学]]大学院政策・媒體研究科修士課程 修了
*2005年9月 - 2007年9月　獲得[[中華民國|中華民國]]教育部台湾獎學生到[[国立政治大学|国立政治大学]]留学
*2008年3月　慶應義塾大学大学院政策・媒體研究科後期博士課程 単位取得退学
*2011年4月　博士号取得

== 職歴 ==
*2008年4月 - 2009年3月　[[東海大学|東海大学]][[教養学部|教養学部]]国際関係学科 非常勤講師　
*2009年4月 - 2012年3月　[[国士館大学|国士館大学]]21世紀亞洲学部 専任講師 
*2010年4月 - 2011年3月　[[法政大学|法政大学]]大学院政治学研究科 兼任講師 
*2012年4月 - 2013年3月　国士館大学21世紀亞洲学部 准教授 
*2013年4月 - 2017年3月　法政大学法学部国際政治学科 准教授 
*2017年4月 -　法政大学法学部国際政治学科 教授

== 受賞 ==
