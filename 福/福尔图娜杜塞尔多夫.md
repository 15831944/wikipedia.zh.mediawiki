'''福尔图娜杜塞尔多夫'''（'''{{Audio|De-Fortuna Düsseldorf.ogg|Fortuna Düsseldorf}}'''）是位於[[德國|德國]][[北莱茵-威斯特法伦|北莱茵-威斯特法伦]][[杜塞尔多夫|杜塞尔多夫]]的[[職業|職業]][[足球會|足球會]]，現時在[[德國足球甲級聯賽|德國足球甲級聯賽]]中作賽。

== 主場球場 ==
杜塞尔多夫的主場球場是位於[[杜塞尔多夫|杜塞尔多夫]][[莱茵河|莱茵河]]河畔的'''LTU竞技场'''（LTU-Arena），耗資2億4,000萬[[歐元|歐元]]，於2004年落成揭幕，可容51,500名觀眾，球場設有可關閉的頂蓋。球場現時由當地航空公司「LTU國際」（LTU International）[[贊助|贊助]]的冠名合約於2009年6月30日結束，由7月1日開始更名為'''思捷环球竞技场'''（Esprit Arena），[[思捷環球|思捷環球]]的冠名權為期五年<ref>{{cite web |language=de |url=http://www.ltuarena.de/template_content_id.php4?contentId=1056&hlc=0_0_0 |title=Neuer Namensgeber vorgestellt |publisher=www.ltuarena.de |date=2009-03-16 |accessdate=2009-06-03}}</ref>。

LTU竞技场除用作[[英式足球|英式足球]]比賽外，於2005-07年曾成為[[歐洲美式足球聯盟|歐洲美式足球聯盟]]球隊[[萊恩火焰|萊恩火焰]]（Rhein Fire）的主場進行[[美式足球|美式足球]]比賽。由於[[德甲|德甲]]球會[[勒沃库森足球俱乐部|利華古遜]]的主場[[拜耳竞技场|拜耳竞技场]]（BayArena）進行重建，於[[2008年至2009年德國足球甲級聯賽|2008/09年球季]]下半季開始借用LTU竞技场作為臨時主場。

== 著名球員 ==
共有25名德素多夫球員曾代表[[德國國家足球隊|德國國家足球隊]]，合共上陣240場。

* [[克劳斯·阿洛夫斯|克劳斯·阿洛夫斯]]（Klaus Allofs），效力期間兩奪[[德國足協盃|德國足協盃]]，國家隊前鋒。
* [[托马斯·阿洛夫斯|托马斯·阿洛夫斯]]（Thomas Allofs），克劳斯的弟弟，同期效力。
* [[尤普·德瓦尔|尤普·德瓦尔]]（Jupp Derwall），國家隊教練。
* [[托尼·图雷克|托尼·图雷克]]（Toni Turek），[[1954年世界盃足球賽|1954年世界盃]]決賽號稱「伯爾尼奇蹟」的國家隊門將。
