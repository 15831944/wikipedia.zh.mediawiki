'''福田区'''是[[中华人民共和国|中国]][[广东省|广东省]][[深圳市|深圳市]]下辖的一个[[市辖区|市辖区]]。位于[[深圳|深圳]]经济特区中部，是[[经济特区|经济特区]]内的四区之一。福田区是[[深圳市委|深圳市委]]、[[深圳市政府|市政府]]所在地，是深圳市重点开发和建设的中心城区，将建设成为深圳市的[[行政|行政]]、[[文化|文化]]、[[信息|信息]]、国际展览和商务中心。

== 地理 ==
福田区东部从红岭路起与[[罗湖区|罗湖区]]相连，西部至[[华侨城|华侨城]]与[[南山区_(深圳市)|南山区]]相接，北到[[鸡公山_(深圳)|鸡公山]]与[[龙华区|龙华区]]相连，南临[[深圳河|深圳河]]、[[深圳湾|深圳湾]]与[[香港|香港]][[新界|新界]]的[[米埔|米埔]]、[[元朗|元朗]]相望。辖区面积78.04[[平方公里|平方公里]]，下辖10个[[街道办事处|街道办事处]]。截至2002年全区总[[人口|人口]]达88.58万人，其中户籍[[人口|人口]]35.6万人，暂住人口52.98万人。

== 历史 ==
“福田”一名的由来，有两种说法，一种说法是源于[[宋代|宋代]]所题“湖山拥福，田地生辉”一词；另一种说法是源于[[南宋|南宋]]光宗皇帝[[赵淳|赵淳]][[绍熙|绍熙]]三年（1192年），史书记载，上沙村始祖[[黄金堂|黄金堂]]的四子[[黄西|黄西]]为到松子岭南麓建村，开荒造田，块块成格，故名为“幅田”，后人谐音为福田，意为“德福于田”。福田区历史悠久，福田地域历来为宝安（新安）县地。[[明朝|明朝]]时属新安县城归城乡去都，清时属新安县官富司辖地。民国时期到解放初期，福田地域属[[宝安县|宝安县]]第二区沙头乡。1990年10月，成立福田區，為深圳市的市轄區。<ref>{{zh-cn}}{{cite web |url=http://www.szb.sz.gov.cn/ftq/ |title=福田区 |publisher=深圳市史志办公室 |accessdate=2014-05-03}}</ref>

现今福田区主要部分为原居民上埗村{{fact}}范围（北至笔架山，南至深圳河，西至华富路，东至红岭南路），因此现今福田区原名[[上步区|上步区]]，但因为普通话上埗区谐音“上不去”与当时特区建设政治气氛不相称，因此改为意头好的名字{{fact}}。

===农村城市化===
梅富<ref>{{cite web|url=http://www.szft.gov.cn/bmxx/qlhjdbsc/zwxxgk/jgzn/jdgk/ |title=莲花街道概况 |publisher=福田区政府在线 |accessdate=2014-08-09}}</ref>、布尾、上沙、下沙、沙尾、沙咀、巴登、沙埔头、新洲等13个社区在1990-2010年代仍有鲜明的农村城市化印记，表现为社区内原住民从社区所在股份公司获得年度分红收益，以及区域内密集而且高矮不一的[[农民楼|农民楼]]，在大众文化中称谓以上社区为[[城中村|城中村]]。虽然政府主导或地产参与了这些区域的旧城改造，将原住民建筑逐步更换为写字楼、商业中心、厂房，但以上农村城市化现象不会很快变更为现代化城镇。

== 功能组团 ==
=== 住宅区 ===
福田区黄木岗安置区<ref>{{cite web|url=http://gzdaily.dayoo.com/gb/content/2004-05/26/content_1556630.htm|title=深圳黄木岗安置区后天拆除(图)|publisher=广州日报|deadurl=yes|archiveurl=https://web.archive.org/web/20141007233246/http://gzdaily.dayoo.com/gb/content/2004-05/26/content_1556630.htm|archivedate=2014-10-07}}</ref>和莲花山安置区<ref>{{cite web |url=http://sztqb.sznews.com/html/2009-06/06/content_650941.htm |title=莲花山安置区，开始圆梦的地方 |deadurl=yes |archiveurl=https://web.archive.org/web/20141007172016/http://sztqb.sznews.com/html/2009-06/06/content_650941.htm |archivedate=2014-10-07 }}</ref>是深圳历史上出现过的公务员、企事业单位过渡性住房解决方案，后因城市规划、住房改革等因素而消失。后由福田区其他地点的莲花北村、益田村、彩田村、侨香村等纯住宅社区替代，这些是深圳市区政府公务员或事业单位成员的福利性住房。福田区的大型屋苑有[[福民新村|福民新村]]、[[天澤花園|天澤花園]]、[[皇御苑|皇御苑]]、[[帝濤豪園|帝濤豪園]]等。

=== 商业区 ===
福田区的商业区主要有[[华强北|华强北]]、會展中心、車公廟等；而深圳第一高樓[[平安國際金融中心|平安國際金融中心]]，以及深圳中心、深業上城等超高層摩天大廈，均建於此區。

=== 旅游区 ===
坐落在福田区的商业旅游区有[[香蜜湖水上世界|香蜜湖水上世界]]（已結業）和[[赛格观光|赛格观光]]（[[赛格广场|赛格广场]]）等，市政旅游景点有[[红树林自然保护区|红树林自然保护区]]、[[深圳市中心公园|深圳市中心公园]]、[[莲花山_(深圳)|莲花山公园]]、[[荔枝公园|荔枝公园]]、[[笔架山公园|笔架山公园]]、[[梅林公园|梅林公园]]、[[深圳园博园|深圳园博园]]等。

=== 医疗和卫生保健 ===
==== 现状 ====
区属公立医院有福田人民医院、福田区第二人民医院、福田中医院，其中福田区人民医院另行地址管辖香蜜湖分院、南园分院。民营医院有景田医院、和平医院、友谊医院、福华中西医结合医院、[[深圳希玛林顺潮眼科医院|深圳希玛林顺潮眼科医院]]，等等。疾病控制机构有福田区疾病预防控制中心、福田区慢性病防治院。保健机构有福田区妇幼保健院。

区内市级医疗机构和卫生专业机构有[[北京大学深圳医院|北京大学深圳医院]]、[[香港大学深圳医院|香港大学深圳医院]]、[[深圳市第二人民医院|深圳市第二人民医院]]、[[深圳市妇儿医院|深圳市妇儿医院]]、[[深圳市儿童医院|深圳市儿童医院]]、[[深圳市中医院|深圳市中医院]]、[[深圳市疾病预防控制中心|深圳市疾病预防控制中心]]。
