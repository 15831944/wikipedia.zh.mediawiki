'''艋舺大道'''是[[臺北市|臺北市]][[萬華區|萬華區]]的一條道路，東起[[中華路_(台北市)|中華路二段]]，西接[[華翠大橋|華翠大橋]]至[[新北市|新北市]]，下有[[臺鐵|臺鐵]][[縱貫線_(鐵路)|縱貫線]]、[[高鐵|高鐵]]經過<ref>{{Cite web |url = http://www.tlsh.tp.edu.tw/~t127/Mankahgeo/mankurroad.htm|author = |title = 
艋舺大道 下月掛牌|publisher = [[中國時報|中國時報]]|date = 2001-05-21|accessdate = 2015-02-18}}</ref>。
