'''湖州经济技术开发区'''是位于[[浙江省|浙江省]][[湖州市|湖州市]]的[[国家级经济技术开发区|国家级经济技术开发区]]。区内设湖州高新技术产业园区、浙江省湖州台商投资区、浙江省留学人员湖州创业园区。<ref name= "概述"/><ref name= "简介"/>

== 简介 ==
开发区位于[[吴兴区|吴兴区]]，成立于1992年，[[2010年3月|2010年3月]]升格为国家级开发区。规划面积80平方公里<ref name="简介">{{cite web |url=http://www.hetd.gov.cn/contents/94/307.html |language= |title=国家级湖州经济技术开发区简介 |publisher=湖州经济技术开发区 |date= |accessdate=2012年10月 |deadurl=yes |archiveurl=https://web.archive.org/web/20121002093049/http://www.hetd.gov.cn/contents/94/307.html |archivedate=2012-10-02 }}</ref>，委托管辖[[杨家埠镇|杨家埠镇]]、[[凤凰街道_(湖州市)|凤凰街道]]和[[康山街道|康山街道]]，行政管辖面积144平方公里（40个行政村，13个社区），常住人口12.5万，流动人口8万。<ref name="概述">{{cite web|url=http://unn.people.com.cn/GB/121616/199688/199752/12395000.html | language= | title=湖州经济技术开发区概述 |publisher=人民网 |date=2010年8月10日 |accessdate= }}</ref>由凤凰分区、西南分区、西塞山分区、康山分区四个区块构成。<ref name= "概述"/>

初步形成的主导产业有汽配机电、生物医药、新材料、新能源、环保设备、电子信息和食品加工业。<ref name= "概述"/>

== 著名公司 ==
有160家规模以上企业，包括美国[[陶氏化工|陶氏化工]]、日本[[三菱|三菱]]、[[英美资源股份|英美资源股份]]、德国[[巴斯夫|巴斯夫]]、台湾[[统一集团|统一集团]]等[[世界500强|世界500强]]和国际著名企业。<ref name= "概述"/>

== 交通 ==
[[318国道|318国道]]、[[104国道|104国道]]、[[长湖申航道|长湖申航道]]、[[宣杭铁路|宣杭铁路]]、[[杭湖宁高速公路|杭湖宁高速公路]]、[[申苏浙皖高速公路|申苏浙皖高速公路]]、[[申嘉湖高速公路|申嘉湖高速公路]]都在区内通过。[[杭湖宁城际轨道|杭湖宁城际轨道]]建成后，湖州到[[杭州|杭州]]只需半小时，到[[南京|南京]]只需1小时。<ref name= "概述"/>
