'''湖中之地国家休闲区'''（{{lang-en|Land Between the Lakes National Recreation Area}}）是片[[美国|美国]]的{{tsl|en|National Recreation Area|国家休闲区}}，位于[[肯塔基州|肯塔基州]]、[[田纳西州|田纳西州]]内，{{tsl|en|Lake Barkley|伯克利湖}}和{{tsl|en|Kentucky Lake|肯塔基湖}}之间。该区域在1963年被美国总统[[约翰·肯尼迪|约翰·肯尼迪]]指定为国家休闲区。该休闲区最开始由[[田納西河谷管理局|田納西河谷管理局]]所管理，但随后被转交给[[美国国家森林局|美国国家森林局]]。1991年，它被[[联合国教育、科学及文化组织|联合国教科文组织]]指定为[[人與生物圈計劃|生物圈保育地]]，<ref>{{cite web |date=2005-03-30 |title=Biosphere Reserve Information: Land Between The Lakes |url=http://www.unesco.org/mabdb/br/brdir/directory/biores.asp?mode=all&code=USA+47 |publisher=[[联合国教育、科学及文化组织|联合国教育、科学及文化组织]] |accessdate=2016-06-14}}</ref>但2017年6月14日撤出该项目。<ref>{{cite web |date=2017-06-14 |title=23 new sites added to UNESCO’s World Network of Biosphere Reserves |url=http://en.unesco.org/news/23-new-sites-added-unesco-s-world-network-biosphere-reserves |publisher=[[联合国教育、科学及文化组织|联合国教育、科学及文化组织]] |accessdate=2017-06-14}}</ref>
