'''湖南理工学院南湖学院'''，隶属于[[湖南理工学院|湖南理工学院]]，位于[[湖南省|湖南省]][[岳阳市|岳阳市]][[南湖风景区|南湖风景区]]，为[[本科大学|本科大学]]。

== 历史 ==
2002年，湖南理工学院在[[湖南省|湖南省]][[岳阳市|岳阳市]][[南湖风景区|南湖风景区]]建立了“湖南理工学院南湖学院”。

== 学校院系 ==
湖南理工学院南湖学院拥有8个系，37个专业。

*系：文学与法学系、外国语言文学系、经济与管理系、机械与电子工程系、建筑与化学工程系、音乐系、体育系、美术系

== 校训 ==
