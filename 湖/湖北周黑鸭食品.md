'''湖北周黑鸭食品有限公司'''（简称'''周黑鸭'''，{{hkex|1458}}），是一家位于[[武汉市|武汉市]]的食品加工企业，于1995年开业<ref name="“黑鸭子”是如何腾飞的！">{{cite web| url=http://www.ctax.org.cn/news/rdzt/ssxcy/sshxw/txl/t20110112_625634.html| title=“黑鸭子”是如何腾飞的！| accessdate=2012-04-26}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，主要产品为使用“周黑鸭”品牌的鸭肉加工食品，目前在武汉及[[湖北省|湖北省]]内各地市，以及[[北京|北京]]、[[天津|天津]]、[[上海|上海]]、[[广东|广东]]、[[江苏|江苏]]、[[浙江|浙江]]、[[山东|山东]]、[[湖南|湖南]]、[[江西|江西]]、[[福建|福建]]、[[安徽|安徽]]、[[河南|河南]]、[[陕西|陕西]]、[[四川|四川]]、[[重庆|重庆]]、[[河北|河北]]等地有售。<ref name="公司简介">{{cite web| url=http://www.zhouheiya.cn/culture.asp| title=公司简介| accessdate=2012-04-26}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>2011年，[[武汉地铁2号线|武汉地铁2号线]][[江汉路站|江汉路站]]被“周黑鸭”冠名。<ref name="江汉路站被“周黑鸭”冠名">{{cite web | url=http://news.cnhan.com/content/2011-12/11/content_1352039.htm | title=江汉路站被“周黑鸭”冠名 | accessdate=2012-04-26}}</ref>2012年，“周黑鸭”被国家工商总局认定为“中国驰名商标”。<ref name="武汉农产品加工产值过千亿">{{cite web | url=http://news.cnhubei.com/ctdsb/ctdsbsgk/ctdsb11/201204/t2051098.shtml | title=武汉农产品加工产值过千亿 | accessdate=2012-04-26}}</ref>

2016年11月，周黑鴨國際控股有限公司在香港進行[[公開招股|公開招股]]，招股價介乎每股5.8至7.8港元，集資最多33.11億元，集資所得主要用於開發加工設施、物流及倉儲中心、新增自營門店數目及拓展海外市場。<ref>{{cite web | url=http://www.mpfinance.com/fin/daily2.php?node=1477939411560&issue=20161101 | title=周黑鴨今起招股 沒基礎投資者 | publisher=明報 | date=2016-11-01 | accessdate=2016-12-26}}</ref>

==沽空機構質疑造假門==
有狙擊港股屢次得手(被狙擊者有部分被停至今，甚至遭除牌)的沽空機構Emerson Analytics 在2019年3 月1 日發報告指控上市公司周黑鴨國際偽造銷售數據， 給予目標價2.4港元，較報告當日收市價3.6港元有33.3%潛在跌幅，事件更於3月4日繼續發酵，惟周黑鴨當日股價未有受此沽空報告影響，報升2.5%，收報3.69港元。 Emerson Analytics 去年第三季走訪了周黑鴨收入重鎮，調查貢獻周黑鴨總收入54.2% 的華中地區，以驗證該區實際業務量，並收集區內524 間店內「銷售時點情報系統」（POS）每日營業時間最後一小時的收據。 Emerson Analytics發現，周黑鴨的湖南省和江西省分店，有店主經常在無顧客購物下開立收據，認為分店將交易量「發水」達28%；對比周黑鴨所披露區內分店日均交易量（ADSV）74 宗，而經Emerson上述調查所得為125宗。而周黑鴨當晚並沒發公告就報告的諸多指控作回應。<ref>{{cite news |language =  zh-cn |author = 方楚茵 |coauthors = |url=https://news.mingpao.com/pns/%E7%B6%93%E6%BF%9F/article/20190305/s00004/1551724901635/%E5%91%A8%E9%BB%91%E9%B4%A8%E8%A2%AB%E6%8C%87%E8%AA%87%E5%A4%A7%E9%8A%B7%E5%94%AE-%E8%82%A1%E5%83%B9%E4%B8%8D%E8%B7%8C%E5%8F%8D%E5%8D%87-%E6%8C%87%E8%82%A1%E5%83%B9%E5%83%85%E5%80%BC2-4%E5%85%83-%E6%98%A8%E4%BB%8D%E5%8D%872-5 |title = 周黑鴨被指誇大銷售 股價不跌反升指股價僅值2.4元 昨仍升2.5% |publisher =明報 |pages =B05 經濟 |date = 2019年3月5日 |accessdate = 2019年3月19日}}</ref>

周黑鴨於翌日（即3月5日）上午8時49分方才刊發需「短暫停牌」，事隔一日才發出駁斥四大核心指控的澄清公告<ref>{{Cite web |url=http://www3.hkexnews.hk/listedco/listconews/SEHK/2019/0306/LTN20190306024_C.pdf |title = 周黑鴨國際澄清公告 |author =|date = 2019年3月6日 |publisher = 港交所披露易 |language = zh-cn |accessdate = 2019年3月19日 |quote =  }}</ref> 。

2019年3月6日早上開市前，周黑鴨發公告回應上述指控，澄清集團並未將任何已取消訂單入賬作為銷售， 而該報告完全誤解了銷售小票單號的運作方式。至於捏造財務資料的虛假指控，周黑鴨已駁斥該報告中有關誇大日均客單數及客單價的所有未經證實的指控及控告，認為對虛假過往財務資料的指控完全屬無稽之談及缺乏根據。集團將保留權利，在適當或有需要的情況下以法律或任何其他途徑來維護自身權益。<ref>{{cite news |language =  zh-cn |author = |coauthors = |url=https://hk.finance.appledaily.com/finance/realtime/article/20190306/59335395 |title =【通告盤點】周黑鴨反擊指控 今復牌 |publisher =蘋果財經 |pages =|date = 2019年3月6日 |accessdate = 2019年3月19日}}</ref>周黑鴨當日亦申請復牌。

期後中金公司更發研究報告力撐周黑鴨，該研究報告認為沽空機構Emerson指控欠缺依據，維持對周黑鴨的目標價6.12元，重申「推薦」評級。<ref>{{cite news |language =  zh-cn |author = |coauthors = |url=http://www.etnet.com.hk/www/tc/news/categorized_news_detail.php?newsid=ETN290307190&category=market |title =周黑鴨獲中金撐場股價反彈，傳華為向外買零件舜宇回吐 |publisher =經濟通 |pages =|date = 2019年3月7日 |accessdate = 2019年3月19日}}</ref>

本月初發報告狙擊內地食物股周黑鴨（1458）的沽空機構Emerson，事隔一周再度出手，2019年3月14日發出周黑鴨專屬沽空報告「第二季」，報告直斥周黑鴨在澄清公告聲稱取消訂單是常態，不等於取消交易，並聲稱獲得逾400 張周黑鴨長沙分店單據，指交易取消情況不尋常，質疑涉及一批「不存在的客人」。<ref>{{cite news |language =  zh-cn |author = |coauthors = |url=https://news.mingpao.com/pns/%E7%B6%93%E6%BF%9F/article/20190314/s00004/1552502269059/emerson%E5%86%8D%E7%8B%99%E6%93%8A%E5%91%A8%E9%BB%91%E9%B4%A8-%E8%B3%AA%E7%96%91%E3%80%8C%E4%B8%8D%E5%AD%98%E5%9C%A8%E7%9A%84%E5%AE%A2%E4%BA%BA%E3%80%8D |title = Emerson再狙擊周黑鴨 質疑「不存在的客人」 |publisher =明報 |pages =B02 經濟  |date = 2019年3月14日 |accessdate = 2019年3月19日}}</ref>。而周黑鴨再就Emerson Analytics指控偽造銷售數據作出澄清，周黑鴨從未將任何無支付記錄的已取消訂單入賬紀錄為銷售或收入，並強調報告披露了其作者可能於該公司股份中擁有淡倉並因此在股份價格下跌時可實現巨額收益。有鑑於此，周黑鴨董事會特此強調，周黑鴨股東及有意投資者於閱覽報告時應採取非常謹慎的態度。<ref>{{cite news |language =  zh-cn |author =  |coauthors = |url=https://www.finet.hk/newscenter/news_content/5c8997debde0b3135ea447ed |title =周黑鴨(01458-HK)指沽空機構Emerson持淡倉 出報告可能獲巨利 |publisher =財華社 |pages = |date = 2019年3月14日 |accessdate = 2019年3月19日}}</ref>。

== 冠名江汉路地铁站争议 ==
[[武汉地铁2号线|武汉地铁2号线]]的[[江汉路站|江汉路站]]早在2011年12月就已将冠名权拍卖给了[[周黑鸭|周黑鸭]]公司，<ref name="周黑鸭地铁站引网友吐槽 站点冠名引争议(图)">{{cite web | url=http://native.cnr.cn/society/201211/t20121114_511341952.html | title=周黑鸭地铁站引网友吐槽 站点冠名引争议(图) | accessdate=2012-11-16}}</ref>
但当时并未引起广泛舆论关注，直到近一年后装修即将完成，有网友在[[新浪微博|新浪微博]]上发出一张地铁站内显示“周黑鸭·江汉路站”的站牌时才引起轩然大波。<ref name="周黑鸭冠名地铁或布局上市 商业味重网友反感">{{cite web | url=http://finance.chinanews.com/cj/2012/11-16/4333664.shtml | title=周黑鸭冠名地铁或布局上市 商业味重网友反感 | accessdate=2012-11-16}}</ref>
很多人指责地铁集团拍卖冠名权的做法不妥，但实际上武汉地铁自1号线开始便已将部分站点冠名权拍卖给了部分公司；<ref name="武汉地铁2号线贴24处周黑鸭图片遭网友吐槽">{{cite web | url= http://news.sina.com.cn/s/p/2012-11-14/095325578085.shtml| title=武汉地铁2号线贴24处周黑鸭图片遭网友吐槽 | accessdate=2012-11-16}}</ref>
也有人指责周黑鸭只是小吃，冠名地铁拉低了档次；<ref name="周黑鸭冠名地铁或布局上市 商业味重网友反感" />
但有评论认为，周黑鸭是武汉特产，如果一定要冠名，让其冠名地铁站并无不妥，对乘客利益影响微乎其微，反而对推广武汉文化有帮助。<ref name="周黑鸭地铁站伤不了市民感情">{{cite web | url=http://opinion.hexun.com/2012-11-15/147975768.html | title=周黑鸭地铁站伤不了市民感情 | accessdate=2012-11-16}}</ref><ref name="周黑鸭·江汉路，不妨碍乘客就行">{{cite web| url=http://review.jschina.com.cn/system/2012/11/15/015219900.shtml| title=周黑鸭·江汉路，不妨碍乘客就行| accessdate=2012-11-16| deadurl=yes| archiveurl=https://web.archive.org/web/20160304041914/http://review.jschina.com.cn/system/2012/11/15/015219900.shtml| archivedate=2016-03-04}}</ref>

但是在2012年11月24日，所有2号线的冠名都已取消，包括周黑鸭冠名的[[江汉路站|江汉路站]]。<ref name="武汉地铁站点冠名全部取消“周黑鸭”称没办法评论">{{cite web | url=http://news.cnhubei.com/xw/jj/201211/t2337048.shtml | title=武汉地铁站点冠名全部取消“周黑鸭”称没办法评论 | accessdate=2012-11-23}}</ref>
