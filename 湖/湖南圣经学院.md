'''湖南圣经学院,''' 原名為長沙聖經學院''','''<ref>{{Cite book|url=https://www.worldcat.org/oclc/48973971|title=宣道與中華：宣道會早期在華宣教史略|last=羅|first=腓力|last2=|date=|publisher=宣道出版社|year=1997|isbn=9789622445055|edition=|location=香港|pages=|chapter=|oclc=48973971}}</ref> 是一所基督教神学院，位于中国[[湖南省|湖南省]]省会[[长沙市|长沙市]]。

== 歷史 ==
湖南圣经学院由[[内地会|内地会]]美国传教士创办于1917年<ref>{{cite web|url=http://www.hnzj.gov.cn/hnzj/zjyx/2011-09-03/257.html|title=湖南宗教|accessdate=2012-12-11|work=|publisher=|deadurl=yes|archiveurl=https://web.archive.org/web/20121118055735/http://www.hnzj.gov.cn/hnzj/zjyx/2011-09-03/257.html|archivedate=2012-11-18}}</ref>，设于韭菜园，为美国[[:en:Bible_Institute_of_Los_Angles|洛杉矶圣经学院]]（今美国[[:en:Biola_University|拜欧拉大学]]）分校。该校建成一组西式建筑群，为当时湖南省“最讲究、最气派的建筑”<ref>{{cite web |url=http://www.people.com.cn/h/2012/0909/c25408-4212192560.html |title=韭菜园圣经学院老照片|accessdate=2012-12-11 |work=|publisher=}}</ref>。1930年代，神学家[[陈崇桂|陈崇桂]]主持该校工作。

1937年11月，[[北京大学|北京大学]]、[[清华大学|清华大学]]、[[南开大学|南开大学]]为避战火，南迁长沙，组建[[国立长沙临时大学|国立长沙临时大学]]，租借湖南圣经学院为其校址。次年2月，该校西迁云南，即[[西南联合大学|西南联合大学]]。1938年11月8日，湖南圣经学院遭到日军飞机轰炸，随即又遭遇[[文夕大火|文夕大火]]破坏。

1951年，湖南圣经学院脱离与美国差会的关系，不久被并入北京[[燕京神学院|燕京神学院]]，校舍被政府机关占用，为湖南省政府二院第三办公楼<ref>{{cite web |url=http://www.people.com.cn/h/2012/0909/c25408-4212192560.html |title=韭菜园圣经学院老照片|accessdate=2012-12-11 |work=|publisher=}}</ref>。1990年湖南圣经学校租用房屋恢复办学。1993年迁入马王堆乡校址。
