'''湖南科技大学'''（{{Lang-en|Hunan University of Science and Technology}}）是一所位于[[中华人民共和国|中华人民共和国]][[湖南省|湖南省]][[湘潭市|湘潭市]]的[[公立大学|公立大学]]，于2003年由原[[湘潭工学院|湘潭工学院]]和[[湘潭师范学院|湘潭师范学院]]合并组建而成。<ref name="moe">{{cite 湖南科技大学是一所由[[中华人民共和国应急管理部|国家应急管理部]]、[[国家国防科技工业局|国家国防科技工业局]]与[[湖南省政府|湖南省政府]]进行[[省部共建|省部共建]]的高校；<ref>{{cite “[[十三五规划|十三五规划]]”国家百所“[[中西部高校基础能力建设工程|中西部高校基础能力建设工程]]”支持高校；<ref>{{cite web|url=http://www.hnust.edu.cn/pub/news/xyyw/77194.htm|publisher=湖南科技大学|title=湖南省发改委副主任童名让一行来我校调研中西部高校基础能力建设规划项目}}</ref> == 学校概况 ==
湖南科技大学由[[湘潭师范学院|湘潭师范学院]]和[[湘潭工学院|湘潭工学院]]于2003年合并而来，<ref name="moe" /> 其前身最早可以追溯至1949年所创办的湘北建设学院。<ref name="summary" /><ref name="sohu20180711" /> 湖南科技大学位于[[中国|中国]][[湖南省|湖南省]][[湘潭市|湘潭市]]，占地面积约{{Convert|3100|mu|km2}}，建筑面积约{{Convert|1600|mu|km2}}。<ref name="summary" /> 学校现有教职员工2,577人；本科学生27,838人，硕士及博士研究生3,134人；另外其附属[[独立学院|独立学院]]——[[湖南科技大学潇湘学院|潇湘学院]]有学生5,742人。<ref name="summary" />

=== 学校历史 ===
==== 湘潭师范学院 ====
{{Main|湘潭师范学院}}

[[湘潭师范学院|湘潭师范学院]]其历史最早可以追溯至1044年所兴建的湘潭县学，目前该校遗址保存在湖南科技大学雨湖校区内，现为湖南科技大学继续教育学院所在地。<ref>{{cite book |author1=张楚廷 |author2=张传燧 |title=湖南教育史(一) |date=2008-11-01 |publisher=Beijing Book Co. Inc. |location={{Link-en|林登 (新泽西州)|Linden, New Jersey|Linden NJ}} |isbn=9787999027270 |url=https://books.google.com.tw/books?id=ENh8DwAAQBAJ |accessdate=2019-01-27}}</ref><ref name="sohu20180711" /> 1949年，[[中国人民解放军|中国人民解放军]]进入湘潭，同期[[中国共产党|中国共产党]]长沙地委成立。在长沙地委于1949年7月23日召开的第一次代表会议上，确立了筹建“湘北建设学院”的事宜。后经[[毛泽东|毛泽东]]确认，确立最终校名。<ref>{{cite book |author1=王邦杰 |title=湖南古今名胜词典 |date=2010-12-01 |publisher=Beijing Book Co. Inc. |location={{Link-en|林登 (新泽西州)|Linden, New Jersey|Linden NJ}} |isbn=9787535765758 |url=https://books.google.com.tw/books?id=0V1qDwAAQBAJ |accessdate=2019-01-27}}</ref><ref>{{cite book |author1=[[万里|万里]] |title=湖湘文化辞典（八） |date=2011-06-01 |publisher=Beijing Book Co. Inc. |location={{Link-en|林登 (新泽西州)|Linden, New Jersey|Linden NJ}} |isbn=9787999008972 |url=https://books.google.com.tw/books?id=oWJ6DwAAQBAJ |accessdate=2019-01-27}}</ref><ref name="sohu20180711" />

1949年8月18日，湘北建设学院正式开学。[[武光|武光]]任政委，罗其南任院长。<ref>{{cite web |title=罗其南 |url=http://www.library.hn.cn/hxrw/xdrw/sbjld/lqn.htm |publisher=[[湖南图书馆|湖南图书馆]] |accessdate=2019-01-27}}</ref><ref name="sohu20180711" /> 截止1951年9月结束办学，湘北建设学院共计办学七期。<ref name="sohu20180711" /><ref>{{cite web |title=湘北建设学院留下的精神财富 |url=http://www.hnust.edu.cn/pub/news/xyyw/xwtt_xyyw/35008.htm |publisher=湖南科技大学 |accessdate=2019-01-27}}</ref> 后湘北建设学院改建为长沙地委党校。<ref name="sohu20180711" />

1958年7月，湘潭师范专科学校在原湘北建设学院的校址上成立，是中国首批八所师范专科学校之一。<ref name="sohu20180711" /><ref name="statutes" /><ref>{{cite web |title=邵阳学院 |url=https://www.applysquare.com/institute-cn/cn.hnsyu/ |publisher=申请方 |accessdate=2019-01-27}}</ref>

1973年，[[郭沫若|郭沫若]]为湘潭师范专科学校题写校名。<ref>{{cite web |author1=Lattice_Xiao |title=记忆里的湘潭师专 |url=https://blog.csdn.net/qq_25116591/article/details/45576063 |publisher=[[CSDN|CSDN]] |accessdate=2019-01-27}}</ref> 随着湘潭师范专科学校的办学规模不断扩大，而校区面积受限不得不将部分专业迁至桃园路校区（即现在湖南科技大学北校区所在地）办学。<ref name="sohu20180711" /> 

1985年10月，湘潭师范专科学校升格为湘潭师范学院，成为湖南省内第二所[[师范|师范]]类[[本科院校|本科院校]]。1985年11月15日，[[胡耀邦|胡耀邦]]为湘潭师范学院题写校名。同年11月29日，[[王震|王震]]题写校训。<ref name="sohu20180711" />

2003年，湘潭师范学院与湘潭工学院合并组建了湖南科技大学。<ref name="moe" />

==== 湘潭工学院 ====
{{Main|湘潭工学院}}

1978年4月1日，[[煤炭工业部|煤炭工业部]]决定在湖南省湘潭市石码头成立隶属煤炭工业部的第13所煤炭院校，即“湘潭煤炭学院”。<ref name="sohu20180711" /><ref>{{cite web |title=1978年4月1日 国务院批准恢复和增设五十五所高校 |url=http://www.people.com.cn/GB/historic/0401/6020.html |publisher=[[人民网|人民网]] |accessdate=2019-01-27}}</ref> 

1981年，学校改名为“湘潭矿业学院”。<ref name="sohu20180711" /><ref>{{cite web |title=最被低估的10所地矿院校，绝对的实力大于名气！ |url=http://kuaibao.qq.com/s/20180618B05WWB00 |publisher=快报 |accessdate=2019-01-27}}</ref> 

1997年3月，随着新专业的增多和招生规模的扩大，学校于再度改名为“湘潭工学院”。<ref name="sohu20180711" /><ref>{{cite web |title=矿业大学报考:15所原煤炭工业部直属院校详解 |url=http://gaokao.xdf.cn/201602/10425210.html |publisher=新东方高考网 |accessdate=2019-01-27}}</ref>

2003年，与湘潭师范学院合并成湖南科技大学。<ref name="moe" />

==== 合并之后 ====
2002年初，[[湘潭师范学院|湘潭师范学院]]与[[湘潭工学院|湘潭工学院]]达成合并意向。<ref name="sohu20180711" />

2003年4月16日，[[中华人民共和国教育部|教育部]]正式批准了两校的合并；<ref name="moe" /> 同年11月18日，两校正式合并成为湖南科技大学。<ref name="gmw" />

2011年，湖南科技大学成为“[[卓越工程师教育培养计划|卓越工程师教育培养计划]]”第二批入选大学。<ref>{{cite web |title=教育部关于批准第二批卓越工程师教育培养计划高校的通知 |url=http://old.moe.gov.cn//publicfiles/business/htmlfiles/moe/s3860/201110/xxgk_125721.html |publisher=[[中华人民共和国教育部|中华人民共和国教育部]] |accessdate=2019-01-27}}</ref><ref>{{cite web |title=133所高校成为第二批卓越工程师培养计划高校 |url=http://news.sciencenet.cn/htmlnews/2011/10/254156.shtm |website=科学网 |publisher=[[新华网|新华网]] |accessdate=2019-01-27}}</ref><ref>{{cite web |title=“卓越工程师教育培养计划”高校名单(第二批) |url=http://www.chinadegrees.cn/xwyyjsjyxx/xwsytjxx/279564.shtml |publisher=中国学位与研究生教育网 |accessdate=2019-01-27}}</ref>

2012年，湖南科技大学从“[[普通高等学校招生全国统一考试#志愿填报|本科二批]]”划入“[[普通高等学校招生全国统一考试#志愿填报|本科一批]]”参加[[普通高等学校招生全国统一考试|高考]]录取。<ref>{{cite web |title=2012湖南高考一本录取率近13% 本科录取率44% |url=http://learning.sohu.com/20120824/n351431519.shtml |website=[[搜狐网|搜狐网]] |publisher=华声在线 |accessdate=2019-01-27}}</ref>

2016年，湖南科技大学成为[[国家国防科技工业局|国家国防科技工业局]]与[[湖南省|湖南省]]的[[省部共建|省部共建]]高校。<ref>{{cite web |author1=曹辉 |author2=张灿强 |title=湖南4高校获国防科工局共建 |url=http://hunan.sina.com.cn/news/2016-07-07/detail-ifxtwiht3244408.shtml |website=[[新浪网|新浪网]] |publisher=[[湖南日报|湖南日报]] |accessdate=2019-01-27}}</ref> 同年，湖南科技大学入选“[[中西部高校基础能力建设工程|中西部高校基础能力建设工程]]”第二期大学名单。<ref>{{cite web |title=这133所高校，被称为“小211工程”大学！ |url=https://k.sina.com.cn/article_6114172783_16c6edf6f00100ggub.html |publisher=[[新浪网|新浪网]] |accessdate=2019-01-27}}</ref> 并成为[[中华人民共和国教育部|教育部]]评选的“2016年度50所全国创新创业典型经验高校”。<ref>{{cite web |title=教育部办公厅关于公布2016年度全国创新创业典型经验高校名单的通知 |url=http://www.moe.gov.cn/srcsite/A15/s7063/201608/t20160816_275060.html |publisher=[[中华人民共和国教育部|中华人民共和国教育部]] |accessdate=2019-01-27}}</ref>

2017年，湖南科技大学成为[[国家安全生产监督管理总局|国家安全生产监督管理总局]]（现[[中华人民共和国应急管理部|中华人民共和国应急管理部]]）与[[湖南省|湖南省]]的[[省部共建|省部共建]]高校。<ref>{{cite web |title=省部共建 四所高校获省政府和国家安监总局共建 |url=http://kaoyan.eol.cn/nnews/201709/t20170916_1554698.shtml |publisher=中国教育在线 |accessdate=2019-01-27}}</ref>

=== 学校标识 ===
==== 校名 ====
湖南科技大学，简称为“湖科大”；英语译名为“{{lang|en|Hunan University of Science and Technology}}”，缩写为“{{lang|en|HNUST}}”。湖南科技大学中文校名在使用时为[[毛体字|毛体字]]，字体颜色为“{{Colorbox|#003B90}}深蓝色”。<ref name="statutes" />

==== 校标 ====
湖南科技大学校标以汉字“人”和英文字母“{{lang|en|T}}”（代表{{lang|en|Technology}}）为设计元素，采用三角形构图，设计成“山峰”形态。代表了“以人为本”和“科技之峰”的含义。<ref name="statutes" /><ref name="bagel">{{cite web |title=校徽·校标 |url=http://www.hnust.edu.cn/xxgk/xxbs/xh/index.htm |publisher=湖南科技大学 |accessdate=2019-02-02}}</ref>

==== 校徽 ====
湖南科技大学校徽采用圆形设计，中间为校标，外部为含有“湖南科技大学”及“{{lang|en|Hunan University of Science and Technology}}”字样的外环；中英文校名之间有“圆点”予以区隔。校徽分为正形和负形两种，正形校徽中校标、校名等采用“{{Colorbox|#00A2E6}}天蓝色”；负形则为正形的反白效果。<ref name="statutes" /><ref name="bagel" />

=== 学校领导 ===

== 人才培养 ==
湖南科技大学目前设有19个教学院及研究生院，另有续教育学院、国际教育学院和潇湘学院。湖南科技大学共计拥有覆盖11个学科门类的88个本科招生专业；此外还拥有3个[[一级学科|一级学科]][[博士后|博士后]]科研流动站，5个一级学科[[博士|博士]]点，29个一级学科和[[二级学科|二级学科]][[硕士|硕士]]学位授权，9种[[专业硕士|专业硕士]]学位授权类别。湖南科技大学拥有3个国防特色学科，1个省级优势特色重点学科，8个省级重点学科。另拥有8个国家特色专业、17个省级特色专业、1个国家专业综合改革试点专业。<ref name="summary" />

=== 本科教育阶段 ===
{{Also|本科教育|学士}}

湖南科技大学是[[中华人民共和国教育部|教育部]]本科教学工作水平评估优秀高校。<ref>{{cite web |title=湖南科技大学 |url=http://news.sina.com.cn/o/2017-06-23/doc-ifyhmtrw3628949.shtml |publisher=[[新浪网|新浪网]] |accessdate=2019-01-27}}</ref> 近年来，湖南科技大学的学生在[[紅點設計大獎|红点设计大赛]]、[[全国青少年科技创新大赛|全国青少年科技创新大赛]]、全国“挑战杯”大学生课外学术科技作品竞赛、全国大学生数学建模竞赛、全国大学生电子设计竞赛等赛事中取得优异成绩。<ref>{{cite web |title=本科生培养 |url=http://www.hnust.edu.cn/rcpy/bkspy/index.html |publisher=湖南科技大学 |accessdate=2019-01-27}}</ref>该校具有推荐优秀应届本科生免试攻读研究生权和港澳台及留学生招生权。截至2018年，已有23个国家留学生及台湾地区交换生来校学习。<ref name="summary" />

=== 研究生教育阶段 ===
{{Also|研究生|博士|硕士}}
{{See|专业硕士}}

=== 继续教育阶段 ===
湖南科技大学继续教育以高等[[函授|函授教育]]为主体，并结合行业业务培训等而形成多元化继续教育体系。湖南科技大学继续教育包括[[成人高考|成人学历教育]]、“中小学教师国家级培训计划”、湖南省中学校长和中学骨干教师培训，以及煤炭行业“现代管理领域专业技术人才知识更新工程”培训、矿长培训和煤矿安全生产培训等。<ref>{{cite web |title=继续教育 |url=http://www.hnust.edu.cn/rcpy/jxjy/index.htm |publisher=湖南科技大学 |accessdate=2019-01-27}}</ref>
