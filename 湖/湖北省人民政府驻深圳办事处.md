'''湖北省人民政府驻深圳办事处'''，是[[中国共产党湖北省委员会|中国共产党湖北省委员会]]、[[湖北省人民政府|湖北省人民政府]]驻[[深圳市|深圳市]]的直属派出机构，由[[湖北省人民政府办公厅|湖北省人民政府办公厅]]管理，与'''湖北省人民政府驻珠海办事处'''、'''湖北省人民政府驻香港办事处'''、'''湖北省人民政府驻广东办事处'''实行[[一个机构两块牌子|一个机构多块牌子]]的工作机制，是湖北省委省政府在珠三角和港澳地区唯一政务窗口和办事机构。主要工作职责包括招商引资、承接[[珠三角地区|珠三角地区]]产业转移、收集报送[[粤港澳|粤港澳]]政经消息、粤港澳地区政务商务联络与接待工作、负责湖北驻深单位党建工作等<ref>{{cite web |title=深办简介 - 湖北省人民政府驻深圳办事处 |url=http://www.hbszb.gov.cn/show_58.html |accessdate=2018-12-20}}</ref>。

湖北省人民政府驻深圳办事处位于[[深圳湖北大厦|深圳湖北大厦]]内，楼内另有办事处下辖的[[楚天大酒店|楚天大酒店]]<ref>{{cite web |title=楚天大酒店官方网站 - 酒店介绍 |url=http://www.grandchuhotel.com.cn/introduce_zh-cn.php |accessdate=2018-12-20}}</ref>。
