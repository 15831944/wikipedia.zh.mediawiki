'''湖南化工职业技术学院'''，是[[湖南省|湖南省]][[株洲市|株洲市]]的高等[[专科大学|专科大学]]，分为2个校区，北校区位于湖南省株洲市石峰区清石路2号，东校区位于湖南省株洲市荷塘区红旗南路82号。<ref name="fish ">{{cite web|url=http://www.hnhgzy.com/hnhgzy/Introduction.asp|title=学院概况|accessdate=2012年|publisher=湖南化工职业技术学院官方网站|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120629151339/http://www.hnhgzy.com/hnhgzy/Introduction.asp|archivedate=2012-06-29}}</ref>

== 历史 ==
1958年，“湖南省化学工业学校”建立，它是湖南化工职业技术学院的前身。<ref name="fish " />

1998年8月，“湖南省化学工业学校”与“湖南省资江化学工业学校”合并。<ref name="fish " />

2003年4月，学校升格为专科大学，更名为“湖南化工职业技术学院”。<ref name="fish " />

2010年，“湖南化工职业技术学院”与“湖南化工机械学校”（湖南工业高级技工学校）合并，仍用“湖南化工职业技术学院”的校名。<ref name="fish " />

== 学校院系 ==
湖南化工职业技术学院设有7个系，2个部，28个高职专业、3个技师专业、12个高级技工专业、21个中专、中技专业。<ref name="fish " />
*系部：化学工程系、应用化学系、机械工程系、自动化系、信息工程系、经济管理系、基础科学系。<ref name="fish " />
*部：技工部、思想政治课部<ref name="fish " />

== 学校文化 ==

== 奖项 ==
湖南化工职业技术学院先后荣获“省级和市级文明卫生单位”、“省级园林式单位”、“湖南省职业教育先进单位”、“全国石油和化学工业先进集体”等荣誉奖项。<ref name="fish " />
