'''湖南省广播电视大学'''（{{lang-en|''' Hunan Radio and TV University '''}}）位于[[湖南省|湖南省]][[长沙市|长沙市]][[雨花区|雨花区]][[青园路|青园路]]168号，为湖南省一所综合性大学，有本科、专科、中专、成人大学和无学历技术性培训。<ref name="fhs " >{{cite web |url=http://www.hnrtu.edu.cn/redwind/Catalog.aspx?id=32f8ed64-8622-49de-b823-5af677b460bb |title=学校简介 |accessdate= 2011年|work= |publisher=湖南省广播电视大学 |language =zh-cn}}</ref>

== 历史 ==
1979年，[[湖南省|湖南省]][[长沙市|长沙市]]建立了“湖南省广播电视大学”，接受“[[中央广播电视大学|中央广播电视大学]]”的指导。<ref name="fhs " />2010年，[[中华人民共和国教育部|中华人民共和国教育部]]颁发文件“2010年具有成人高等学历教育招生资格高校名单”，湖南省的十二所中就有“湖南省广播电视大学”。<ref>{{cite web |url=http://edu.sina.com.cn/chengkao/2010-08-27/1129265040.shtml |title=2010年具有成人高等学历教育招生资格高校名单 |publisher=新浪网 |accessdate=2010-08-27 |work= |language =zh-cn}}</ref>

== 学校院系 ==
湖南省广播电视大学开设了9个学科19类40余个专业。<ref name="fhs " />

*学科：经济学、法学、教育学、文学、理学、工学、农学、医学、管理学
