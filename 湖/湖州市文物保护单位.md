'''湖州市文物保护单位'''是中国[[浙江省|浙江省]][[湖州市|湖州市]]的市级[[文物保护单位|文物保护单位]]，其范围涉及湖州市辖区[[吴兴区|吴兴区]]和[[南浔区|南浔区]]（两区不再公布区级文物保护单位），截止到2017年共公布八批。

*第一批：1961年7月公布，数量不详。

*第二批：1984年8月公布，数量不详。

*第三批：1989年3月21日公布，共20处，含古文化遗址3处，古窑址3处，古建筑12处，古水利运输建筑1处，革命纪念物1处。

*第四批：1994年10月公布，数量不详。

*第五批：1998年3月公布，数量不详。

*第六批：2003年1月20日公布，共35处。含古遗址4处、古墓葬3处、古建筑19处、石刻1处、近现代重要史迹及代表性建筑8处。调整4处，其中与原有市级文保单位合并1处、更名3处<ref>{{cite web |title= 湖州市人民政府公布第六批市级文物保护单位 |publisher= 浙江省文化厅 |url=http://www.zjwh.gov.cn/dtxx/2007-12-10/32726.htm}}</ref>。

*第七批：2011年2月25日公布，共42处。含古遗址11处、古墓葬4处、古建筑20处、近现代重要史迹及代表性建筑7处。调整3处，其中浮玉塔塔基更名为浮玉塔塔基旧址，沈家本故居更名为沈家本故居旧址，馆驿河头王宅更名为王安申旧宅<ref>{{cite web |title=湖州市人民政府关于公布第七批市级文物保护单位的通知 |publisher= 湖州在线 |url=http://www.hz66.com/2011/0321/34982.shtml}}</ref>。

*第八批：2015年11月13日公布，共43处。含古遗址5处、古建筑18处、石窟寺及石刻1处，近现代重要史迹及代表性建筑18处、其他1处。调整2处，其中西舍古桥与原市级文物保护单位路村四桥合并为一处文物保护单位，命名为路村古桥群，诸溇桥、永济塘桥、尚义桥、述中桥与原市级文物保护单位太湖溇港古桥群合并为一处文物保护单位，文物保护单位名称不变，仍为太湖溇港古桥群<ref>{{cite web|title=湖州市人民政府关于公布第八批市级文物保护单位的通知|publisher=湖州市政府|url=http://www.huzhou.gov.cn/xxgk/jcms_files/jcms1/web2/site/art/2015/12/7/art_2482_96445.html|deadurl=yes|archiveurl=https://web.archive.org/web/20160918162052/http://www.huzhou.gov.cn/xxgk/jcms_files/jcms1/web2/site/art/2015/12/7/art_2482_96445.html|archivedate=2016-09-18}}</ref><ref>{{cite web |title=湖州公布第八批市级文物保护单位 |publisher= 湖州在线|url=http://www.hz66.com/2015/1212/245606.shtml}}</ref>。
