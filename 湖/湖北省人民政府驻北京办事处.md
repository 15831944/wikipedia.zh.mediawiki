'''湖北省人民政府驻北京办事处'''（简称'''省驻京办'''），是[[中国共产党湖北省委员会|中国共产党湖北省委员会]]、[[湖北省人民政府|湖北省人民政府]]驻[[北京市|北京市]]的派出机构，为正厅级规格，由[[湖北省人民政府办公厅|湖北省人民政府办公厅]]管理，与'''湖北省委省政府驻京群众工作办公室'''（简称'''省驻京群工办'''）实行[[一个机构两块牌子|一个机构两块牌子]]的工作机制。主要工作职责包括代表湖北省同中央、国家机关与[[北京市委|北京市委]]、[[北京市人民政府|市政府]]联系，省领导在京接待，省政府在京投资等<ref>{{cite web |title=主要职责 - 湖北省人民政府驻北京办事处 |url=http://www.hubei.gov.cn/xxgk/czyjs/zfbm/zjb/201803/t20180305_1259011.shtml |accessdate=2018-12-20}}</ref><ref>{{cite web |title=湖北省人民政府驻北京办事处简介 |url=http://bgs.ggj.gov.cn/bsccx/201002/t20100210_270111.htm?docid=270111&chnlid=2326 |publisher=[[国务院机关事务管理局|国务院机关事务管理局]]驻京办事处管理司 |accessdate=2018-12-20}}</ref>。

==内设机构==
湖北省驻京办、省驻京群工办共设有7个内设机构<ref>{{cite web |title=内设机构 - 湖北省人民政府驻北京办事处 |url=http://www.hbzjb.com/index.php?m=content&c=index&a=show&catid=126&id=950&ls=1 |accessdate=2018-12-20}}</ref>：
* 办公室（总值班室）
* 应急处
* 联络处
* 接待服务处
* 财务资产管理处
* 重大项目前期工程处
* 人事处（机关党委）

另下辖北京湖北大厦有限责任公司，拥有集写字楼、客房、餐饮、娱乐、购物于一体的四星级酒店[[北京湖北大厦|北京湖北大厦]]，内有中餐厅对外开放<ref>{{cite web |title=酒店介绍_北京湖北大厦有限责任公司 |url=http://www.bjhbds.com.cn/intro/1.html |accessdate=2018-12-20}}</ref><ref>{{cite web |title=足不出城 到各省驻京办餐厅尝遍全国美味 |url=https://www.guancha.cn/life/2014_12_01_302092_2.shtml |publisher=[[观察者网|观察者网]] |accessdate=2018-12-20}}</ref>。
