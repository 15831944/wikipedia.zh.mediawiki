'''尾張四觀音'''（日語：おわりしかんのん），是指以下[[尾張國|尾張國]]代表的四座[[觀音寺|觀音寺]]

* [[愛知縣|愛知縣]][[名古屋市|名古屋市]][[中川區|中川區]]荒子町的'''[[荒子觀音|荒子觀音]]'''
* 愛知縣[[海部市|海部市]]甚目寺的'''[[甚目寺|甚目寺觀音]]'''
* 愛知縣名古屋市[[守山區|守山區]]龍泉寺的'''[[龍泉寺_(名古屋市)|龍泉寺觀音]]'''
* 愛知縣名古屋市[[南區_(名古屋市)|南區]]笠寺町的'''[[笠覆寺|笠寺觀音]]'''

不管哪座都是開基至今有千數百年以上的古刹。

== 概要 ==
{{Location map+}}[[德川家康|德川家康]]在建築[[名古屋城|名古屋城]]時，預計從城往鬼門的方位上建設上述4寺作為鎮護。（順帶一提，大須觀音在[[江戶時代|江戶時代]]初期，因家康命令從[[美濃國|美濃]]移轉到現在地，是比較新的寺，不跟其他尾張四觀音並列。）

和每個觀音所連結成的道路形成了'''四觀音道'''<ref>{{cite book |author1=村松喬 |title=新しい鯱 |date=1964 |publisher=毎日新聞社}}</ref>，如笠寺和龍泉寺相交道路的一部分現在仍保存下來，[[千種區|千種區]]内也仍保留以之命名的地名。另外，路上也仍殘存一個明治時代所立標示的道標，上面刻有「東有八事、西有平針、南有熱田、北有瀨戶龍泉寺」<ref>{{cite web |title=名古屋市:四観音道と道標（千種区） |url=http://www.city.nagoya.jp/chikusa/page/0000000655.html |website=www.city.nagoya.jp |accessdate=2019-01-07}}</ref>。

== 在節分的四觀音 ==
因為是以[[名古屋城|名古屋城]]為中心向四方分散的位置，在每年[[節分|節分]]時期四寺會進行節分祭（會），特別是當其中一寺（以名古屋城為中心所見）位於惠方時（北北西/甚目寺觀音、東北東/龍泉寺觀音、南南東/笠寺觀音、西南西/荒子觀音），當年會舉行特別盛大的節分祭（會）。

尾張四觀音的節分惠方，會以笠寺觀音（1）、龍泉寺（2）、荒子觀音寺（3）、笠寺観音（4）、甚目寺觀音（5）的順序，5年一巡<ref name="報告書">{{cite web |title=調査団報告書No.60　「名古屋で節分の日に、もちまわりでおこなわれる有名な豆まき会があるとききました。平成30年（2018年）はどこですか？」 |url=https://www.library.city.nagoya.jp/img/reference/chosahoukoku_60.pdf |website=www.library.city.nagoya.jp |accessdate=2019-01-07}}</ref>。

按此進行尾張四觀音的惠方參拜，在名古屋風俗中據說可以得到最好的祈福效果<ref>{{cite book |author1=斎藤良輔 |title=鄉土玩具辞典 |date=1971 |publisher=東京堂出版 |page=356}}</ref>。

在昭和30年代，和名古屋城位於同樣中區位置的大須觀音，自稱為「中心惠方」，因為此說法已流通現今，所以會有惠方中心是名古屋城、還是大須觀音的疑問<ref>{{cite web |title=恵方と名古屋城（尾張の恵方について） |url=http://www.shiroyama.or.jp/setubun/ehou_nagoyajyo.html |website=www.shiroyama.or.jp |accessdate=2019-01-07}}</ref>。尾張四觀音的各寺自古以來都是以名古屋城為中心，此外不清楚大須觀音和成田山萬福院所說的「中心惠方」是什麼，但和尾張四觀音中心（名古屋城）所指的「中心惠方」是不同的，以此理解可能較為妥當<ref name="報告書"/>。
