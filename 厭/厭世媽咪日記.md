《'''厭世媽咪日記'''》（{{lang-en|''Tully''}}）是一部於2018年上映的美國{{link-en|喜劇勵志劇情片|Comedy-drama}}，由[[傑森·瑞特曼|傑森·瑞特曼]]執導、[[迪亞布羅·科蒂|迪亞布羅·科蒂]]編劇，並由[[莎莉·賽隆|莎莉·賽隆]]、[[麥坎西·黛維斯|麥坎西·黛維斯]]、[[馬克·杜普拉斯|馬克·杜普拉斯]]和[[榮·利文斯通|榮·利文斯通]]主演。電影定於2018年5月4日於美國上映，由[[焦點影業|焦點影業]]發行。

* [[馬克·杜普拉斯|馬克·杜普拉斯]] 飾 瑪洛的弟弟
* [[榮·利文斯通|榮·利文斯通]] 飾 德魯·莫羅

== 製作 ==
=== 拍攝 ===

== 發行 ==
《厭世媽咪日記》定於2018年5月4日於美國上映，由[[焦點影業|焦點影業]]發行。此前，該片的美國上映日為2018年4月20日<ref>{{cite web|url=http://deadline.com/2017/05/tully-movie-release-date-charlize-theron-jason-reitman-diablo-cody-focus-features-1202084547/|title=Focus Features Acquires Reitman-Theron-Cody Reteam ‘Tully’ For April 2018 Bow|website=[[Deadline.com|Deadline.com]]|first=Patrick|last=Hipes|date=2017-05-04|accessdate=2017-05-04}}</ref>。
