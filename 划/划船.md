'''划船'''是一项依靠{{link-en|桨|Oar}}在水中的运动来推动船只的活动，它通过划水来推动船只前进。其中的桨可以是通过机械连接在船上的，或者是手持的（没有与船的机械连接）。

本条目涉及的是更广泛的划船活动，例如用于休闲和运输，而不仅是[[赛艇|赛艇]]运动。赛艇是一项专业的竞速运动，对使用的器械有严格的规定，同时对技术的要求也较高。<ref>{{Cite web | url=http://www.speedrower.com/index.htm | title=Speed Rower, Competitive Rowing | trans-title=竞速桨手，赛艇竞技 | accessdate=2009-02-05 | language=en}}</ref>

== 划船系统的类型 ==

某些地区流行面向后的划船系统。而在其它地区，则流行面向前的系统，尤其是在拥挤的地区，例如意大利的威尼斯，以及亚洲和印度尼西亚的河流和港口。<!-- 这不是严格意义上的"非此即彼"，因为在不同情况是有用的，能够行船上面临的无论哪种方式。 目前的重点放在健康方面的划导致了一些新的机械系统正在发展，一些(例如Rantilla划船的方法)非常不同的传统划系统的过去。-->

''划船'' 涉及一座赛艇谁 ''拉'' 的两个桨或 ''桨''，附着在船上，从而可移动的小船在方向相反的，其中划船者的面孔。 在一些多座位坐船赛艇每拉在一个单一的"''扫描''"桨，通常用两只手。 船在哪赛艇是由协调一个 舵手 是被称为"coxed"对四/八。 有时候 ''滑席位，'' 用于启用的赛艇使用的腿部肌肉，大大增加了功率提供的。 一个替代的滑座，叫一个 ''滑动装配工人''，使用一个固定的座位和划船的移动oarlocks与他的脚下。 -->但在意大利使用的双体船莫斯科尼（moscone）上，桨手要站起来，并利用他的体重来增加划船的力量。<ref>{{cite web | url=http://www.settesere.it/public/parser_download/save/allegati.2007.32.pag04.e05due.pdf | title=Remando in piedi sul moscone | trans-title=在莫斯科尼上站立划船 | language=it }}{{Dead link|date=2019年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

'''面向前的系统：'''''铰接式''或''面向船首''的桨有两只桨，并使用机械传动将桨叶的方向反转，从而使得坐着的桨手可以面向前，并且采用拉的动作来划船。如果船并没有被设计为向前动作，则''推式划船''或''向后划水''使用常规船桨和推的动作来实现面朝前划船，桨手可以是坐着的，也可以是站着的。<!-- 这是一个方便的方法的操纵在一个狭窄的水道，或通过一个繁忙的港口。 "Rantilla"系统的frontrowing桨使用 ''内安装oarlocks'' 而不是扭转送到实现的向前运动的船有一个拉动桨。-->

== 设计因素的划艇 ==
许多考虑到选择一个很好的划艇。 一个设计良好的划艇将执行以及在想条件。 经典的形状的划艇反映一演进的几百年的试验和错误得到一个良好的状态。 一些因素被认为是水线的长度，速度，承载能力、稳定、偏差、体重、适航性、成本、水线束，丰满或度的结束，并修剪。

细节设计是一个妥协之间的相互竞争的因素。 如果吃水线束(宽度)是过于狭窄的船将被招标和乘员在下降的风险，如果束过广泛的船将会是缓慢和有更多的阻力波。 如果高(高度的船舷水线以上)过高，那么偏差会高和结果的船将会被抓到的风和赛艇将不能够控制该船在高风。 如果高过低，水将进入的船通过海浪。
如果船是设计为一个人然后仅只一个划船的位置是必需的。 如果赛艇携带的是一个乘客在船尾然后船会被严重和整将是不正确的。 纠正这一重可添加在蝴蝶结，或者该船可以提供的第二个划船的位置进一步推进这一目的。 对于船上有三个独立的阻挠，并有足够的空间，每一乘客，然后该船已经被一个特定的最小尺寸。 一 {{Convert|8|ft|m|abbr=out|sp=us|adj=mid}} 婴儿车的小艇能够携带4名乘客月3阻挠在平坦的海域。 理想的尺寸的排好船用于距离划是 {{Convert|5|m|ft|abbr=on}}.

总束(宽度)是重要的。 如果桨架太靠近在一起桨将难以使用。 如果桨架是太远，然后船会过大和划将效率低下、浪费一个划船者的努力。 有时候，在狭窄的、更快速的划艇的保护水域支腿被添加，为增加rowlock分离。 许多传统的划艇具有约束 {{Convert|135|cm|ft|abbr=on}}. 大多数现代划艇之间的 {{Convert|8|ft|m|abbr=on}} 和 {{Convert|15|ft|m|abbr=on}} 有一大束的 {{Convert|4|-|4+5/12|ft|abbr=on}}. 水线束是重要的稳定性。 最通常的目的划艇水线束 {{Convert|3|-|3+11/12|ft|abbr=on}}. 稳定得多的影响座高度为划船者做了一个大总重量百分比的。 最通常的目的划艇有一个主要挫败高 {{Convert|10|in|cm|abbr=on}} 以上的龙骨。 更广泛的船只可以有较高的席位。 赛艇的老年僵硬或疼痛的后背受益于 {{Convert|11|in|cm|abbr=on}} 高席位。 芬兰Savonian类型划艇是非常狭窄，速度快，具有非常尖锐的船头和船尾，与梁周围 {{Convert|110|cm|ft|abbr=on}}，但也是危险的招标和易于翻船很容易地在没有经验的手中。 大多数现代风格的划艇都大大低于传统的熟料建成的风格。 许多 {{Convert|12|–|15|ft|abbr=out|sp=us|adj=mid}} 行的船只约有 {{Convert|70|-|100|lb|abbr=on}}. 一个纯粹的划船建造速度将有一个狭窄的横梁 {{Convert|2|-|2+3/4|ft|abbr=on}} ，是非常狭窄在船尾水线，而一个划艇旨在使用一个小舷外发动机的 {{Convert|2|-|5|hp|abbr=on}} 需要一个更广泛的横梁，以支持发动机的重量和运营在船尾。 小短轴船外需要一个气窗高度的 {{Convert|15|in|cm|abbr=on}}. 通常船尾阻挠向前移动 {{Convert|12|-|14|in|abbr=on}} 从严厉得到更好的重量分配和平衡，如果该船正在运营的个与船外。
