'''-{zh-cn:蝙蝠女孩; zh-tw:蝙蝠女孩; zh-hk:蝙蝠女;}-'''（{{lang-en|Batgirl}}）是[[DC漫画|DC漫画]][[蝙蝠侠|蝙蝠侠]]中的角色。也译为'''-{zh-cn:蝙蝠女; zh-tw:蝙蝠女; zh-hk:蝙蝠女孩;}-'''。

需注意的是，-{zh-cn:蝙蝠女孩; zh-tw:蝙蝠女孩; zh-hk:蝙蝠女;}-（{{lang-en|Batgirl}}）与-{[[蝙蝠女侠|蝙蝠女侠]]}-（{{lang-en|Batwoman}}）'''不是'''同一个角色。

== 人物 ==
=== 歷代蝙蝠女孩 ===
; 贝蒂·凯恩（Bette Kane）
{{main|貝蒂·凱恩}}
: 现在是间谍组织旋中碟（Spyral）成员。

; 芭芭拉·高登（Barbara Gordon）
{{main|芭芭拉·高登}}
: 高譚市[[詹姆斯·高登_(漫画)|高登]]警長的养女（後改為親生女兒），是一位功夫天才兼高材生。
: 被[[小丑|小丑]]袭击导致脊髓受伤引退。現在以新的称号「神谕（'''Oracle'''）」、通过操控複数的超级电脑帮助蝙蝠侠。和[[迪克·格雷森|迪克·格雷森]]（第一代羅賓-夜翼）相爱，此外在[[Rocksteady_Studios|Rocksteady Studios]]開發製作的[[蝙蝠俠：阿卡漢騎士|阿卡姆騎士]]遊戲中是和[[提姆·德雷克|提姆·德雷克]]（第三代羅賓）相愛最後在故事結婚。在新52连载中通过神经芯片恢复了行走能力，重新成为蝙蝠女孩。
: 在電影《[[蝙蝠俠4：急凍人|蝙蝠俠4：急凍人]]》被改編成阿福的親戚，無意間進入艾爾福管理蝙蝠俠和羅賓裝備與武器的房間，進而化身蝙蝠女孩與蝙蝠俠和羅賓並肩作戰。

; 海伦娜·伯廷莉（Helena Bertinelli）
{{main|海倫娜·伯廷莉}}
: 幼年时家人被黑手党派的杀手杀害，后来成为了复仇的女战士女猎手（'''Huntress'''）。
: 在美国隔離政策实行，歌谭市与本土被分隔开的时期（无主之地，英文是No Man's Land）以蝙蝠女孩的名义维护和平，但与黑手党战斗时造成了人员伤亡，被此事激怒的蝙蝠侠便将她解雇。
: 現在以女猎手的身份，加入了神谕建立的'''[[猛禽行動_(漫畫)|猛禽小队]]'''(Birds of Prey)。

; 卡珊德拉·该隐（Cassandra Cain）
{{main|卡珊德拉·該隱}}
: 蝙蝠俠的敵人——刺客聯盟最頂尖的兩名殺手[[西瓦女士|西瓦女士]]和[[大衛·該隱|大衛·該隱]]的女兒，自幼被父親大衛帶走，並打算將她訓練成最完美的殺人機器。從結果來講極為成功，可以光憑肉體能力就能閃躲子彈，也打敗過武術高強的母親西瓦女士，就連蝙蝠俠也沒有信心能憑武術將她打敗。
: 代價是幾乎沒有語言和讀寫能力，在蝙蝠俠和芭芭拉的教導下才慢慢可以開口說話，但不怎麼流利。
: 把芭芭拉當成母親或姊姊來看待，也把蝙蝠俠布魯斯當成父親，在受到兩人認可後成為新任蝙蝠女孩。布魯斯更將她收為養女。
: 可是之後發生了[[蝙蝠俠之死|蝙蝠俠之死]]事件，感覺心靈失去支柱的她於是選擇引退到香港定居。

; 史蒂芬妮·布朗（Stephanie Brown）
{{main|史蒂芬妮·布朗}}
: 她是正史線中唯一一個女性羅賓，也是提姆·德雷克（第三代羅賓-紅羅賓）的女友，曾有過其他的超級英雄身分－掠奪者(Spoiler)。
: 她在提姆一次的離職後自願擔任羅賓的工作。曾經一度在一次與黑幫的衝突事件後被認為死亡，但其實是被與蝙蝠俠一家有交情的醫生為了保護她而偽造了[[死亡證明|死亡證明]]，並且瞞過所有人，包括蝙蝠俠和羅賓們。
: 後擔任蝙蝠女孩。在新52連載中，其過去的歷史被抹去。
: 於《Batman Eternal》系列中以掠奪者(Spoiler)身分重新登場。
