'''征服者-313戰鬥機'''（[[波斯語|波斯語]]：'''قاهر-۳۱۳'''，又有「'''Ghaher-313'''」、「'''Qaher-313'''」、「'''Q-313'''」、「'''F-313'''」多種稱呼與譯名）是[[伊朗空軍|伊朗空軍]]的一款研製中的單座[[隱形戰機|隱形]][[战斗机|战斗机]]，於2013年2月1日首次對外公佈<ref name = "PressTV_2013-02-01" />，其性能遽聞與美國製的[[F/A-18E/F超級大黃蜂式打擊戰鬥機|F/A-18E/F「超級大黃蜂」]][[战斗机|戰鬥]][[攻击机|攻击机]]相似。<ref>[http://news.takungpao.com.hk/military/world/2013-02/1424552.html 伊朗公布征服者313新型國產戰鬥機 外形很科幻] {{webarchive|url=https://web.archive.org/web/20130629202851/http://news.takungpao.com.hk/military/world/2013-02/1424552.html |date=2013-06-29 }}《大公報》2013-02-03 </ref>。2013年2月2日由總統[[马哈茂德·艾哈迈迪内贾德|马哈茂德·艾哈迈迪内贾德]]與國防部長[[艾哈迈德·瓦希迪|艾哈迈德·瓦希迪]]於[[伊朗伊斯蘭革命|伊斯蘭革命成功]]為期10天的紀念儀式中介紹<ref name = "PressTV_2013-02-02_a" /><ref name = "Aviationist_2013-02-02" /><ref name = "GoogleNews_2013-02-02" /><ref name = "InquirerNews_2013-02-02" /><ref name=Haaretz_b_2013-02-02/>。

==發展 ==
據伊朗政府資料，征服者-313戰鬥機由伊朗國防部下屬部門、「[[伊朗航空工業組織|伊朗航空工業組織]]」、[[伊朗空軍|伊朗空軍]]所研發與生產，研製工程由哈桑·帕法涅（Hassan Parvaneh）負責<ref name=EconomicTimes_2013-02-02/><ref name="Majumdar02Feb13" />。

遽聞設計團隊已造出兩架縮小型的飛機模型與測試，一座模型使用螺旋槳發動機，另一座據影片顯示，使用微型渦輪噴射發動機<ref>{{cite news| url=http://www.youtube.com/watch?v=dGd0EU-1uYc | work=Youtube | location=Tehran | title=Qaher-313 Inauguration Ceremony (arabic) | date=2013-02-02 | accessdate=2013-02-02}}</ref>。模型也被設計團隊公佈於影片中<ref>{{cite news| url=http://www.youtube.com/watch?v=hSIFPOaILXE | work=Youtube | location=Tehran | title=Qaher-313 (Persian) | date=2013-02-02 | accessdate=2013-02-02}}</ref>。

伊朗之外的媒體消息指出，該機可能無法滿足現有性能規格的要求，充其量只能作為一種大型的樣機或原型機<ref name = "Aviationist_2013-02-02" /><ref name=Haaretz_b_2013-02-02/><ref name=Haaretz_a_2013-02-02/><ref name=IsraelHaYom_2013-02-03/>，[[BBC新聞|BBC新聞]]則指出這架飛機看似為美軍F22的廉價仿製品<ref name = "BBC_2013-02-02" />。[[以色列|以色列]]的《[[国土报|国土报]]》表示，「伊朗聲稱該機的飛行影像極為模糊，顯示似乎並非為載人機，反倒像小型的無線電操作機。」<ref name=Haaretz_b_2013-02-02/>但之後的公布照片表明該機雖然體積比各國主力戰機明顯小，然而確實能載一名駕駛員。

==設計==




據伊朗資料該機為[[匿蹤戰機|匿蹤戰機]]，具低海拔作戰能力<ref name = "Emirates247_2013-02-02" />，還聲稱征服者以「先進材質」所造、低雷達檢測度<ref name = "BBC_2013-02-02" />，也有資料宣稱可於短跑道上進行起降，且容易維護<ref name = "PressTV_2013-02-02_b" />。

《[[全球飛行|全球飛行]]》記述道，征服者類似於[[波音|波音]]的「[[食肉鳥隱形戰機|猛禽]]」（Bird of Prey）原型機，但有更多部份類似於1970年代的[[擁藍|擁藍]]（Have Blue）原型機，後來該機發展成現已退役的[[F-117夜鷹戰鬥機|F-117夜鷹戰鬥機]]。《全球飛行》還寫道「這是一款明顯尺寸較小的飛機，發動機可能取自於通用電氣的J-85渦輪發動機逆向工程所得的變體。」<ref name="Majumdar02Feb13" />
