《'''入間同學入魔了！'''》（{{lang-ja|魔入りました！入間くん}}）是日本漫畫家{{link-ja|西修|西修 (漫画家)|}}的作品，於2017年14號出版的《[[週刊少年Champion|週刊少年Champion]]》上開始連載<ref>{{cite web|title=「お人好し少年が悪魔学校で奮闘！週チャン新連載「魔入りました！入間くん」||url=http://natalie.mu/comic/news/222022 }}</ref>。在《週刊少年Champion》2019年10號上宣布製作電視動畫<ref>{{Cite news|newspaper=[[MANTANWEB|MANTANWEB]]|url=https://mantan-web.jp/article/20190206dog00m200039000c.html|title=魔入りました！入間くん：「チャンピオン」の魔界学園コメディーがテレビアニメ化|date=2019-02-07|accessdate=2019-02-07}}</ref>。

== 故事内容 ==
無法拒絕任何拜託的老好人鈴木入間，是個從1歲開始就踏入社會、為了金錢東奔西走、在修羅場求升的十四歲少年，他的父母為了金錢將他賣給了惡魔！陰差陽錯之下他簽下了惡魔契約、進入了惡魔學院、成為了特待生，並且即將與學院第一的新生代表決鬥——

=== 惡魔小知識 ===
; 惡魔的尾巴
: 基本上女性惡魔的尾巴的尖端是心型；男性則是葵扇型。對應種族的不同等等理由下也會有不跟從基本形狀的情況，例如：歐佩拉。尾巴可以自由收納。

; 惡魔的翅膀
: 翅膀也是據個體會有差別，但基本上都擁有。因為礙事的關係所以大多惡魔會藏起來。沒有批准是不能在校內飛行。
: 對人暴露自己翅膀的根部意味著最大的敬佩與謝罪。

; 惡魔的角
: 角是遺傳，所以會和父母相似當然繼承不了的情況也會有的，這種情況通常發生在老么身上。角是無法隱藏的。雖然超級罕見也會有突然長出來的可能性。
