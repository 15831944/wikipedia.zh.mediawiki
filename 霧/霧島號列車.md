'''霧島'''（{{lang-ja|きりしま}}）是由日本的[[九州旅客鐵道|九州旅客鐵道]]公司所營運；行經[[日豐本線|日豐本線]]、[[鹿兒島本線|鹿兒島本線]]來往[[宮崎車站|宮崎車站]]與[[鹿兒島中央車站|鹿兒島中央車站]]之間地區的[[特急列車|特急列車]]。名稱取自於位處[[宮崎縣|宮崎縣]]與[[鹿兒島縣|鹿兒島縣]]之間的山峰「[[霧島山|霧島山]]」。<ref>{{cite book |script-title=ja:《完全保存版 ありがとう！北斗星＆夜行列車》 |publisher=My Navi |location=日本 |isbn=978-4-839-95595-3 |language=ja |chapter=〈{{lang|ja|列車愛称命名の由来}}〉 |page=111頁 |date=2015年7月10日}}</ref>

== 歷史 ==
1960年西鹿兒島車站（現[[鹿兒島中央車站|鹿兒島中央車站]]）與[[山川車站|山川車站]]之間開通了準急等級列車「錦江」（{{lang|ja|きんこう}}）。隔年運行路線從原先的西鹿兒島車站延伸到[[宮崎車站|宮崎車站]]。<ref>{{cite book |script-title=ja:《山川町史》 |publisher=改訂山川町史刊行會 |author=山川町史編輯委員會 |location=日本 |language=ja |page=737頁 |date=1987年}}</ref>

錦江號在1966年調升等級為[[急行列車|急行列車]]，<ref>{{cite book |script-title=ja:《国鉄準急列車物語》 |publisher=JTB Publishing, Inc. |author=岡田誠一 |location=日本 |isbn=978-4-533-08851-3 |language=ja |chapter=〈{{lang|ja|[15] 九州}}〉 |page=168頁 |date=2012年11月29日}}</ref>該年度[[特別急行列車|特別急行列車]][[日輪號、日向號列車|日輪號]]也編排從九州北方[[博多車站|博多車站]]發車行經宮崎至終點西鹿兒島車站的班次。<ref>{{cite book |script-title=ja:《国鉄・JR　悲運の特急・急行列車50選》 |publisher=JTB Publishing, Inc. |author=寺本光照 |location=日本 |isbn=978-4-533-10522-7 |language=ja |chapter=〈{{lang|ja|急行の列車番号を付けた特急列車}}〉 |page=135頁 |date=2015年6月19日}}</ref>

在[[日豐本線|日豐本線]]電氣化完成後，1980年錦江號被降級為快速列車。<ref name="RS">{{cite web |url=http://www.railstation.net/duke/ressha/rapid_kinko.html |script-title=ja:快速【錦江】 |work=Urabe Research Laboratory |author=Duke. Vigilante |language=ja |accessdate=2017年11月26日}}</ref>之後在使用率因素下，1995年4月20日起錦江號遭除名，並把日輪號終點定為宮崎車站，宮崎與西鹿兒島之間的路線則以「霧島」為名的特急列車來擔任。<ref name="RS"/><ref>{{cite book |script-title=ja:《国鉄・JR 特急列車100年》 |publisher=JTB Publishing, Inc. |author=三宅俊彥 |location=日本 |isbn=978-4-533-08852-0 |language=ja |chapter=〈{{lang|ja|8章　ブルトレの衰退と優等列車特急一本化―平成6年10月～平成24年10月}}〉 |page=167頁 |date=2012年11月15日}}</ref><ref>{{cite book |script-title=ja:《国鉄・JR　悲運の特急・急行列車50選》 |publisher=JTB Publishing, Inc. |author=寺本光照 |location=日本 |isbn=978-4-533-10522-7 |language=ja |chapter=〈{{lang|ja|急行の列車番号を付けた特急列車}}〉 |page=138頁 |date=2015年6月19日}}</ref>

== 營運狀況 ==
從1995年開始運行時，霧島號是採用[[日本國鐵485系電聯車|485系電聯車]]，到了2011年改換成[[JR九州787系電聯車|787系]]與[[JR九州783系電聯車|783系]]電力動車組。<ref>{{cite book |script-title=ja:《485系物語》 |publisher=JTB Publishing, Inc. |author=梅原淳 |location=日本 |isbn=978-4-533-08974-9 |language=ja |chapter=〈{{lang|ja|485系特急 全特急の記錄  ①}}〉 |page=81頁 |date=2013年2月22日}}</ref>

線路的來往範圍雖以宮崎-鹿兒島中央為主，不過在2004與2011年分別增加了[[國分車站|國分]]-鹿兒島中央、宮崎-[[都城車站|都城]]或[[西都城車站|西都城]]的班次。

2011年曾推行在霧島號裡進行車內販售商品的措施，但該服務在2014年秋季終止。<ref>{{cite web |url=http://toyokeizai.net/articles/-/47189 |script-title=ja:JR九州の客室乗務員は本当に不要なのか 10月から一部列車で車内販売を取りやめ |work=[[東洋經濟新報社|東洋經濟新報社]] |author=大坂直樹 |language=ja |date=2014年9月20日 |accessdate=2017年11月26日}}</ref>到了2017年夏季則又因應削減成本的問題，召開過考慮讓霧島號列車採取[[一人控制|一人控制]]方案的檢討會。<ref>{{cite web |url=http://www.sankei.com/west/news/170628/wst1706280060-n1.html |script-title=ja:宮崎－鹿児島中央もワンマン特急検討　ＪＲ九州 |work=[[產經新聞|產經新聞]] |language=ja |date=2017年6月28日 |accessdate=2017年11月26日}}</ref>
