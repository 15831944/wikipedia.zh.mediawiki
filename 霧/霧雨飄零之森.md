《'''霧雨飄零之森'''》（{{lang-ja|霧雨が降る森}}）是一款由真田誠製作的解謎類免費遊戲。遊戲對應[[Microsoft_Windows|Microsoft Windows]]平台，於2013年10月11日發行。漫畫版由名束{{lang|ja|くだん}}作畫，2014年3月起於[[月刊Comic_Gene|月刊Comic Gene]]開始連載，全書共4卷已完結。

== 故事簡介 ==
父母雙亡後變成孑然一身的主角，某天在雙親的房間發現一張標記著「阿座河村」的照片。<br />
照片中映有父母、年幼的自己，以及記憶裡完全沒有印象的爺爺。<br />
「──莫非，這裡就是父母的故鄉？」<br />
心中懷揣寂寞與期待，主角為了尋覓雙親的身影來到阿座河村。<br />
然而，那裡卻是她絕對不能前往的「約定之地」──<ref>{{Cite web |url = http://nanos.jp/hskzkrnkrn/album/1/1/ |title = 遊戲官方網站 |language = ja |accessdate = 2018-05-22}}</ref>

== 登場人物 ==

; {{nihongo|神崎詩織|{{ruby|神崎|かんざき}}シオリ|聲：[[生天目仁美|生天目仁美]]}}
: 本作主角。春夏出生。19歲生日前夕父母因車禍雙亡，為找尋爺爺與雙親過去的蹤跡來到阿座河村資料館進行調查。
: 為人友善、大膽；小時候曾與子狩鬼訂下約定，後被家人以夜光石消除記憶並帶往城市養育。
: 性格剛強，稍微有些男孩子氣。當年在森林附近偶遇迷路的須賀，以此為契機與之交好並締結了可貴的情誼。
: 運動神經佳，過去曾獨自擊退數名欺負須賀的男孩，兒時愛玩的「戰鬥遊戲」讓現在的須賀聽了都能大驚失色。<ref>{{Cite web |url = http://nanos.jp/hskzkrnkrn/blog/1/24/ |title = 「二人の力関係」 |language = ja |accessdate = 2018-07-03}}</ref>

; {{nihongo|須賀孝太郎|{{ruby|須賀|すが}}{{ruby|孝太郎|こうたろう}}|聲：[[浅沼晋太郎|浅沼晋太郎]]}}
: 阿座河村資料館管理員。夏秋出生。左撇子，手持模型刀，以筆談與他人交流。十分不歡迎詩織的到來，屢次告誡詩織不准進入森林。
: 高瘦的身型搭配一身漆黑的裝束被佐久間稱作「黑色電線桿」。腳程快速，畫風狂氣，製作的料理總是宛如黑炭。
: 實為詩織的兒時玩伴，當年進入森林尋找母親被子狩鬼捕獲，詩織即時趕到並替其與子狩鬼定下約定才逃過一劫，須賀則以自己的聲音為交換延續了詩織履行約定的期限。爾後由詩織的爺爺撫育成人並擔任管理員一職，防止亡靈跑出森林的同時避免村里的孩子們闖入森林。
: 性格較內向、愛哭。身負「那男人」的詛咒，平日進行加工夜光石的工作。因同時背負了「詛咒」與「約定」，是個唯有獲得子狩鬼原諒方可得到幸福的角色。<ref>{{Cite web |url = http://nanos.jp/hskzkrnkrn/blog/1/23/?FolderID=2 |title = 「須賀が背負ったもの」 |language = ja |accessdate = 2018-07-03}}</ref>

; {{nihongo|佐久間美夜子|{{ruby|佐久間|さくま}}{{ruby|美夜子|みやこ}}|聲：[[日高里菜|日高里菜]]}}
: 時常進出資料館的女國中生。秋冬出生。與詩織關係良好，身材嬌小，因具靈異體質與同儕關係疏遠，逃課晚歸的慣犯，平時受盡望月巡查的關照。

; {{nihongo|望月洋介|{{ruby|望月|もちずき}}{{ruby|洋介|ようすけ}}|聲：[[伊藤健太郎|伊藤健太郎]]}}
: 阿座河村新來的巡查。夏季出生。對村子仍不太熟悉，為人十分熱心。
: 相當受圍繞在佐久間身旁的幽靈們喜愛。

: 閃爍藍光的石頭，有護身符的效果。

; {{nihongo|子狩鬼|ことりおばけ|聲：[[新井美里|新井美里]]}}

== 結局 ==
本遊戲一共5種結局。
; END 1：兩人的約定
: 全員存活。子狩鬼與孩子們的亡靈得到淨化，須賀取回了自己的聲音。
; END 2：消失的約定
: 全員存活。子狩鬼仍在尋找詩織，須賀以夜光石再次消除了詩織的記憶。
; END 3：約定的看守者
: 全員存活。子狩鬼消失，須賀待在資料館以防孩子們的亡靈跑出森林，詩織回到城市生活。
; END 4：他所守護的約定
: 須賀死亡。詩織因夜光石幸免於難。
; END 5：破碎的約定，履行的約定
: 須賀死亡。詩織精神崩潰，回到森林自取滅亡。
