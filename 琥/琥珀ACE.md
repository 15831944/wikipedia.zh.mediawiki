{{日本作品|琥珀ACE|コハエース}}是[[經驗值_(漫畫家)|經驗值]]繪畫的[[日本|日本]][[漫畫|漫畫]]作品，以《[[月姬|月姬]]》琥珀與遠野秋葉為主角，回顧[[TYPE-MOON|TYPE-MOON]]歷史的漫畫，涉及TYPE-MOON的所有作品。《[[Comptiq|Comptiq]]》2011年6月號開始連載，用「TYPE-MOON 10周年記念漫畫」宣傳<ref>{{cite web|script-title=ja:漫画「コハエースEX」の「桜セイバー」がフィギュア化、桜をイメージした袴のあざやかなカラーが印象的|url=https://www.inside-games.jp/article/2014/10/09/81392.html|website=iNSIDE|accessdate=2017年6月12日|date=2014年10月9日}}</ref>。

《[[幻想嘉年華|幻想嘉年華]]》亦有該作的登場，《琥珀ACE GO》並有《Fate/KOHA-ACE 帝都聖杯奇譚》連載。

== 登場角色 ==
; {{nihongo|琥珀|琥珀（こはく）|配音：[[高野直子|高野直子]]}}

; {{nihongo|Saber|セイバー|配音：[[川澄綾子|川澄綾子]]}}
: 出自《Fate/stay night》。
: 要和其他的Saber作區分時，頭上會戴著「青」字樣。另外，頭上戴著「社」字時是以社長（[[武內崇|武內崇]]）的身分說話。

; {{nihongo|翡翠|翡翠（ひすい）|配音：[[松來未祐|松來未祐]]}}
: 出自《[[月姬|月姬]]》。
: 琥珀的妹妹。

; {{nihongo|紅Saber|赤セイバー|配音：[[丹下櫻|丹下櫻]]}}
: 出自《[[Fate/EXTRA|Fate/EXTRA]]》。

; {{nihongo|舊Saber|プロトセイバー|配音：[[櫻井孝宏|櫻井孝宏]]}}
: 出自《[[Fate/Prototype|Fate/Prototype]]》。
: 和Saber的真名相同，歷史上也是[[亞瑟王|同一個人物]]，不同之處在於這個Saber是男性。個性謙遜。

; {{nihongo|愛爾奎特·布倫史塔德|アルクェイド・ブリュンスタッド|配音：[[柚木涼香|柚木涼香]]}}
: 出自《[[月姬|月姬]]》。
: 頭上戴著「奈」字時，是以[[奈須磨菇|奈須磨菇]]的身分說話。

; {{nihongo|C狐|キャス狐|配音：[[齋藤千和|齋藤千和]]}}
: 出自《[[Fate/EXTRA|Fate/EXTRA]]》。

; {{nihongo|小莫|モーさん|配音：[[澤城美雪|澤城美雪]]}}
: 出自《[[Fate/Apocrypha|Fate/Apocrypha]]》。
: Saber的女兒，通稱紅「的」Saber，對Saber的態度就像是叛逆期中的小孩。

; {{nihongo|[[土方歲三|土方歲三]]|ひじかた としぞう|配音：[[星野貴紀|星野貴紀]]}}

: 只在[[LINE|LINE]]貼圖中登場。

: 出自《[[Fate/Grand_Order|Fate/Grand Order]]》。
