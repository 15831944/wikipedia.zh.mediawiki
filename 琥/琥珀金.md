'''琥珀金'''是一种[[金|金]]和[[银|银]]的天然[[合金|合金]]，包含极少量的[[铜|铜]]和其他金属。它已经能够人为生产，经常被称为'''绿金'''。古希腊人称之为「黄金」或「白金」，而不是「精炼黄金」。它的颜色范围由苍白到亮黄色，根据黄金和银的比例改变。

在现代西方[[安那托利亚|安那托利亚]]的天然含金量范围有70％到90％，与此相反同一地理区域的古[[吕底亚|吕底亚]]钱币的琥珀金含金量45％至55％。这表明在这一领域造币的发明是为了从[[铸币税|铸币税]]透过发行含金量低于普通循环金属的货币而增加利润。

琥珀金早在公元前三千年中的埃及的古王國時期被使用，有时作为古埃及金字塔和方尖碑顶部的[[pyramidions|pyramidions]]的外部涂层。它也被用于制造古[[饮水容器|饮水容器]] 。第一金属硬币都是琥珀金製的和追溯至公元前7世纪末或公元前6世纪初。几十年来，[[诺贝尔奖|诺贝尔奖]]的[[奖章|奖章]]都是镀金绿金製的。

琥珀金名字还被用来表示 [[镍银|德国「银」(镍银)]]，主要为其使用制作技术手段。<ref>[https://groups.yahoo.com/group/DrawingInstruments/message/2098 有关用于绘制仪器的「琥珀金」、德国银和镍银的使用术语的讨论。]</ref>

== 词源 ==
现代使用的术语白金通常是关于[[金|金]]合金的任何一个或一组合的 [[镍|镍]]中， [[銀|銀]]、[[铂|铂]] 和 [[钯|钯]] 产生一个银色的黄金。
「琥珀金/电金」这个名字是[[希腊语|希腊字]]ἤλεκτρον的[[拉丁语|拉丁语]]形式（èlektron），在''[[奥德赛|奥德赛]]''中提及的是一种金属物质，由金合金制成。同样的词也用于[[琥珀|琥珀]]，可能是因为某些品种是淡黄色的，而且来自琥珀的[[静电|静电]]性质是现代英语单词「电子」和「电」。琥珀金在古时候经常被称为「[[白金|白金]]」，但有可能更准确地描述为「淡金」，因为它的颜色通常是淡黄色或黄白色。白金术语的现代用法通常涉及与任何合金或[[镍|镍]]，[[银|银]]，[[铂|铂]]和[[钯|钯]]的组合以产生银色金。

== 成分 ==
琥珀金的主要组成是[[金|金]]和[[银|银]]但有时会找到包含极少量的[[铂|铂]]，[[铜|铜]]和其他[[金属|金属]]。该名称主要用于约20-80％的金和20-80％的银原子之间的组合物，但这些都是取决于主导因素而严格地称为金或银。分析琥珀金的组成从约公元前600年的古希腊币造币的琥珀子的组成，显示出由Phocaea发行的金币中的黄金含量约为55.5％。在古典时代早期，琥珀金的含金量范围从Phokaia的46%到[[米蒂利尼|米蒂利尼]]的43%。在这些地区的后期造币，可追溯到公元前326年，含金量平均为40%至41%。在[[希臘化時代|希腊化时代]]中，[[布匿人|布匿人]]发行的琥珀金币里黄金的比例有所下降。后来在[[君士坦丁堡|君士坦丁堡]]控制的[[拜占庭帝国|拜占庭帝国]]下，金币的纯度降低了，并且开始使用可以称为琥珀金的合金。

== 历史 ==
琥珀金在[[埃及第五王朝|埃及第五王朝]]的法老[[萨胡拉|萨胡拉]]的考察中被提到(见 [[萨胡拉|Sahure]])。还在[[老普林尼|老普林尼]]的[[博物志_(老普林尼)|博物志]]里讨论过。

琥珀金被认为已经在公元前约600年的[[阿吕亚泰斯|阿吕亚泰斯]]统治下的[[吕底亚|吕底亚]]用作硬币。
* [http://rjohara.net/coins/lydia-electrum/ 古吕底亚的琥珀金狮子硬币(约公元600年)]
* [http://www-cm.fitzmuseum.cam.ac.uk/gallery/East-West/images/Ancient_01_obv.gif 由琥珀金制成的吕底亚硬币的正面图片]
琥珀金比[[金|金]]更适合用作造币，主要是因为它是更坚硬和更耐用，但也是因为炼金技术在当时并不普及。从现代西[[安那托利亚|安那托利亚]](70-90％)到古吕底亚造币(45-55％)的含金量之间的差异表明，吕底亚人已经解决了银的炼制技术，并在介绍以下引用的纯银圆之前向当地原住民提供了一些精炼银。

在[[呂底亞|吕底亚]]，琥珀金被铸造成4.7克硬币，每一个价值1/3''[[斯塔特|斯塔特]]'' (意为「标准」)。 三个硬币(重量约为14.1 克，将近0.5盎司)共计了一斯塔特，约一名士兵一个月的薪水。 为补充斯塔特，制成了分数：''trite(''第三)，''hekte''(第六)等，包括1/24斯塔特，甚至下降到1/48和1/96斯塔特的。 该1/96斯塔特只有0.14到0.15克。 较大面额如一斯塔特硬币，也会被铸造的。

由于琥珀金的组成变化，难以鉴定每个硬币的确切价值。广泛的交易都受到这个问题的阻碍，因为谨慎的外国商人在当地的琥珀金硬币上提供差价。

这些困难在公元前570年被引进的纯[[银币|银圆]]淘汰。然而，琥珀金货币仍然一直持续到大约公元前350年。最简单的原因是因为含金量，一个14.1克的斯塔特高达十几个14.1克的银片。
