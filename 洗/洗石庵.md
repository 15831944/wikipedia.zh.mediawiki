'''洗石庵'''是一座位于[[广西壮族自治区|广西]][[桂平市|桂平市]][[西山|西山]]的[[临济宗|临济宗]]尼众道场，为[[汉族地区佛教全国重点寺院|汉族地区佛教全国重点寺院]]。

== 历史沿革 ==
洗石庵始建于[[清朝|清]][[顺治|顺治]]三年（1646年），[[康熙|康熙]]三十八年（1699年）建成。雍正、乾隆、嘉庆年间都曾重建<ref name="A">[http://www.ndcnc.gov.cn/datalib/2003/Place/DL/DL-20031210132333/ 洗石庵]</ref>。

== 建筑特点 ==

洗石庵有庵门、经堂、佛堂三进堂，为广西境内保存最完整、最华丽的寺庙之一。经堂为三开间，屋顶为金色琉璃硬山重檐。佛堂为三开间大殿，外檐为石柱、内为红漆木柱、下为宝瓶型石柱<ref name="A" />。庵前有唐建李公祠。庵后有清建的[[龙华寺|龙华寺]]及李宗仁飞阁。还有[[乳泉|乳泉]]、[[史隐洞|史隐洞]]等<ref>[http://spot.cnair.com/guangxi/guiping/list_xishian.htm 洗石庵]</ref>。

== 佛事活动 ==
洗石庵每年都举办“浴佛节”，即每年农历四月初八，纪念佛陀[[释迦牟尼|释迦牟尼]]诞辰<ref>[http://www.gx.xinhuanet.com/dtzx/2009-05/05/content_16431916.htm 桂平西山浴佛文化旅游节盛况空前] {{webarchive|url=https://web.archive.org/web/20160304122010/http://www.gx.xinhuanet.com/dtzx/2009-05/05/content_16431916.htm |date=2016-03-04 }}</ref><ref>[http://www.gxnews.com.cn/staticpages/20090501/newgx49fb124e-2028729.shtml 桂平"浴佛节"隆重开幕 千人朝拜上山]</ref><ref>[http://news.dayoo.com/society/57401/200905/03/57401_5856198.htm 广西桂平“浴佛节”大典吸引数万人观礼] {{webarchive|url=https://web.archive.org/web/20150512163639/http://news.dayoo.com/society/57401/200905/03/57401_5856198.htm |date=2015-05-12 }}</ref>。

2009年，“浴佛节”进行传灯祈愿[[汶川大地震|四川地震]]遇难亡灵<ref>[http://news.gxnews.com.cn/staticpages/20090501/newgx49fb124e-2028714.shtml “浴佛节”传灯祈愿 虔诚祈福“5·12”]</ref>。
