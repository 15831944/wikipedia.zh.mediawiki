《'''語理分析的思考方法'''》（[[英文|英文]]：'''''Logico-Linguistic Analysis - Methods of Thinking (1)'''''），是一本哲學書籍，初版由[[李天命|李天命]]於1981年完成。此書主張[[分析哲學|分析哲學]]與[[邏輯實證主義|邏輯實證主義]]之觀點，並闡述若干使用語言之方法會導致思考紊亂。

有當代哲學家（例如劉桂標、[[王偉雄|王偉雄]]、楊梓燁）嚴厲地批評此書，例如認為此書誤解了[[歐陸哲學|歐陸哲學]]，並且內容源自過時的[[邏輯實證主義|邏輯實證主義]]。
==背景==
[[艾耶爾|艾耶]]於1936年出版《[[語言、真理與邏輯|語言、真理與邏輯]]》，引起英語世界注意[[邏輯實證主義|邏輯實證主義]]。李天命亦是受到[[邏輯實證主義|邏輯實證主義]]影響，其於《語理分析的思考方法》之中，則有大量引用艾耶之邏輯實證主義。

==內容==
===釐清關鍵詞義===
李天命於此書主張用語理分析的方法來處理問題，即首先釐清問題的關鍵詞語的意思。釐清關鍵詞語的意思後，才著手進一步處理（消除或者解答）問題，<ref>{{cite book |author=[[李天命|李天命]] |date=1999|title=《語理分析的思考方法》|location=香港 |publisher=青年書屋 |page=18 |isbn=962-393-700-8}}</ref>否則根本就連在說什麼也不知道<ref>{{cite book |author=[[李天命|李天命]] |date=1999|title=《語理分析的思考方法》|location=香港 |publisher=青年書屋 |page=17 |isbn=962-393-700-8}}</ref>。

===語言陷阱===
李天命在此書將導致思考紊亂之言辭用法，稱為「語言陷阱」，並且批評「語言陷阱」惑人心思，妨害思想。<ref>{{cite book |author=[[李天命|李天命]] |date=1999|title=《語理分析的思考方法》|location=香港 |publisher=青年書屋 |page=37 |isbn=962-393-700-8}}</ref>李天命將其認為常見之「語言陷阱」分為六類：「闕義」（meaning incomplete）、「歧義」（ambiguous expression）、「含混」（vague）、「着色」（coloured expression）、「實化」（reification）、「癖義」（idiosyncratic sense）。
====闕義====
李天命認為，闕義語句，是在所在脈絡不能提示所欠缺之意義。<ref>{{cite book |author=[[李天命|李天命]] |date=1999|title=《語理分析的思考方法》|location=香港 |publisher=青年書屋 |page=40 |isbn=962-393-700-8}}</ref>李天命舉例：「怎樣的制度是最妥善的政治制度？」是一個闕義語句，因為「最妥善」這個關鍵詞，並未能指出適用對象，例如是對統治者、商人還是對工人，還是對所有人。<ref>{{cite book |author=[[李天命|李天命]] |date=1999|title=《語理分析的思考方法》|location=香港 |publisher=青年書屋 |page=39 |isbn=962-393-700-8}}</ref>
====歧義====
李天命指，歧義語句，是同一個言辭，在一個脈絡出現，但是這個言辭未能表達出明確是屬何意思，例如「一個所謂自由的人，其實都不是自由的」就是歧義語句，這一句的第一個「自由」意思是「沒有受監獄、極權制度等外在因素約束」，而第二個「自由」意思是「不被心理因素等一切因素決定」。李天命批評，這類歧義語句令人誤以為「很有深度」。<ref>{{cite book |author=[[李天命|李天命]] |date=1999|title=《語理分析的思考方法》|location=香港 |publisher=青年書屋 |page=45 |isbn=962-393-700-8}}</ref>

===認知意義===
李天命於此書主張，若且唯若，當一個語句有真假條件時，才是有認知意義，而有真假條件，就可以考察語句之真假值。李天命批評，沒有認知意義之語句說不上「真」或「假」，是連「假」的資格都無。李天命認為，沒有認知意義之語句是沒有真假值。<ref>{{cite book |author=[[李天命|李天命]] |date=1999|title=《語理分析的思考方法》|location=香港 |publisher=青年書屋 |page=90-91 |isbn=962-393-700-8}}</ref>李天命主張印證原則（confirmation principle）：如果一個語句是[[分析-綜合區別|分析語句]]，那就是有認知意義。如果一個語句是綜合語句，但不可以由經驗證據支持，那就是沒有認知意義。<ref>{{cite book |author=[[李天命|李天命]] |date=1999|title=《語理分析的思考方法》|location=香港 |publisher=青年書屋 |page=95|isbn=962-393-700-8}}</ref>李天命於此書批評，[[黑格爾|黑格爾]]的一些語句是意義不明，沒有認知意義。<ref>{{cite book |author=[[李天命|李天命]] |date=1999|title=《語理分析的思考方法》|location=香港 |publisher=青年書屋 |page=22|isbn=962-393-700-8}}</ref>李天命將沒有認知意義，稱為偽贗（pseudo）。

坊間，所謂偽命題（pseudo proposition）之意思就是沒有認知意義之命題。

==批評==
[[劉桂標|劉桂標]]指出，李天命於《語理分析的思考方法》明顯誤解[[歐陸哲學|歐陸]]哲學家的用語 。李天命認為，[[尼采|尼采]]「真理是醜惡」這一句犯了「實化」語言陷阱，因而意思不清楚。劉桂標則指，尼采的語句，並非隨意亂說，而是針對傳統哲學常常獨斷而封閉等弊處而說。劉桂標認為，李天命沒有清楚說明尼采想法之宗旨和特徵，亦不提尼采用了比喻方式：「隨意截取他的一句說話而批評其語意不清，這明顯是對尼采哲學的一種誤讀。」<ref>{{cite web |author1=劉桂標|agency=[[香港獨立媒體|香港獨立媒體]]|title=東方及歐陸哲學家語意不清嗎？ ——論李天命對其哲理的誤解|url =https://www.inmediahk.net/node/1043576|date=2016-07-21|accessdate=2019-04-01|}}</ref>

[[王偉雄|王偉雄]]指，李天命之《語理分析的思考方法》內容相當粗淺，而並非哲學傑作。<ref>{{cite web |author1=[[王偉雄|王偉雄]]|agency=魚之樂|title=天命難測|url =http://fishandhappiness.blogspot.com/2010/02/blog-post_17.html|date=2010-02-17|accessdate=2019-04-01|}}</ref>

[[楊梓燁|楊梓燁]]認為，《語理分析的思考方法》之「認知意義判準」（「印證原則」或「（廣義）檢證原則」）是源自[[邏輯實證主義|邏輯實證主義]]，而經過當代哲學家嚴厲批判，幾乎已經無哲學家接受此標準，所以不應該將「認知意義判準」納入思考之方法。<ref>{{cite web |author1=楊梓燁|agency=[[立場新聞|立場新聞]]|title=重構思考方法 (5) ：放棄過時、錯誤的認知意義判準|url =https://www.thestandnews.com/cosmos/%E9%87%8D%E6%A7%8B%E6%80%9D%E8%80%83%E6%96%B9%E6%B3%95-5-%E6%94%BE%E6%A3%84%E9%81%8E%E6%99%82-%E9%8C%AF%E8%AA%A4%E7%9A%84%E8%AA%8D%E7%9F%A5%E6%84%8F%E7%BE%A9%E5%88%A4%E6%BA%96/|date=2015-09-08|accessdate=2019-04-01|}}</ref>
