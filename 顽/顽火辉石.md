'''顽火辉石'''（{{Lang|en|enstatite}}）是一种硅酸盐矿物，是富镁的斜方[[辉石|辉石]]。

== 性质与特征 ==
化学组成为 Mg<sub>2</sub>Si<sub>2</sub>O<sub>6</sub> ，若 Fe<sup>2+</sup> 置换Mg所形成的斜方輝石称为铁珪辉石（{{Lang|en|ferrosilite}}）。顽火辉石与铁珪辉石可以任意比例互溶，形成固溶体，一般标注其相对比例，如En<sub>80</sub>Fs<sub>20</sub> 。这两种矿物曾经亦称'''古铜辉石'''（こどうきせき、{{Lang|en|bronzite}}） 、 '''紫苏辉石'''（しそきせき、{{Lang|en|hypersthene}}） 、 '''铁紫苏辉石'''（てつしそきせき、{{Lang|en|ferrohypersthene}}）现较少使用。富镁固溶体是造岩矿物之一，常见于火成岩和变质岩中。

此外，相同化学组成的[[单斜晶系|单斜晶系]]矿石称[[单斜顽火辉石|单斜顽火辉石]]。

含少量铁的顽火辉石经风化作用可呈亚金属光泽古铜色，所以有时也被称为古铜辉石，但更准确的名称应为顽火辉石。

== 鉴别 ==
可以利用光学特征鉴别斜方晶系和单斜晶系的[[辉石|辉石]]，比如双折射、多色性等。顽火辉石可以沿两个垂直的方向完美解理。顽火辉石通常呈现白、灰、绿或褐色。莫氏硬度介于5到6之间，比重约3.2-3.3 。

== 太空中的顽火辉石 ==
顽火辉石晶体被发现也存在于太阳系外，比如演化后的恒星与[[行星状星云|行星状星云]] （NGC 6302）。顽火辉石被认为是早期在太空中形成的硅酸盐晶体。

顽火辉石被认为是[[E-型小行星|E-型小行星]]主要组分。在太阳系中，如[[匈牙利族小行星|匈牙利族小行星]]。
