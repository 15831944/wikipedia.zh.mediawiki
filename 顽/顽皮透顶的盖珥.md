《'''顽皮透顶的盖珥'''》（{{lang-th|'''แก้วจอมแก่น'''}}）是一部[[泰国|泰国]]的[[儿童文学|儿童文学]]作品，由[[菀盖珥|菀盖珥]]（{{lang-th|แว่นแก้ว}}，[[诗琳通公主|诗琳通公主]]的[[笔名|笔名]]）写作，1978年初版，共有23篇有趣的故事<ref>{{cite news |author= |last= |first= |date=1983-12-31 |title=書林 （41）|url=http://eresources.nlb.gov.sg/newspapers/Digitised/Article/lhzb19831231-1.2.82.2.3 |newspaper=[[联合早报|联合早报]] |location=第39页 |access-date= |quote=}} </ref>。该小说作品改编自作者的亲身经历，讲述的是小学四年级女生盖珥和她的朋友们身边的故事。<ref>{{cite web |url=http://www.sirindhorn.net/hrh_new/s4_1_1_1.php?id_order=2009-08-21%2011:43:59&bt_id=25 |title=แก้วจอมแก่น |last= |first= |date= |website=Sirindhorn.net |publisher= |access-date=2018-04-16 |quote=}}</ref><ref>{{cite web |url=http://www.sirindhorn.net/hrh_new/e4_1_1_1.php?id_order=2009-08-21%2011:43:59&bt_id=25 |title=Kaew the Irrepressible |last= |first= |date= |website=Sirindhorn.net |publisher= |access-date=2018-04-16 |quote=}}</ref>

== 登场角色 ==
;盖珥（{{lang|th|แก้ว}}）
故事主角，其原型为作者自己，生日为4月2日<ref>{{cite news |author=林瑧 |last= |first= |date=1984-01-07 |title=盖珥——不，诗琳通公主 |url=http://eresources.nlb.gov.sg/newspapers/Digitised/Article/lhzb19840107-1.2.305 |newspaper=[[联合早报|联合早报]] |location=第38页 |access-date= |quote=盖珥也不是凭空虚构的人物，而实在确有其人——她就是玻璃镜里的泰国公主诗琳通的化身。}} </ref>。住在泰国一个简单小家庭的小学四年级女生，特征是[[双马尾|双马尾]]。泰文名意为“[[玻璃|玻璃]]”。

;凯姐（{{lang|th|พี่ไก่}}）
盖珥的姐姐，其原型为作者的姐姐[[乌汶叻公主|乌汶叻公主]]。特征是短发、戴红框圆[[眼镜|眼镜]]。泰文名为“[[鸡|鸡]]”之意。

;蕾妹（{{lang|th|น้องเล็ก}}）
盖珥的妹妹，其原型为作者的妹妹[[朱拉蓬公主|朱拉蓬公主]]。特征是披肩发。

== 译本 ==
1982年，郭宣颖准备开始翻译泰国[[诗琳通公主御作列表|诗琳通公主的御作]]《顽皮透顶的盖珥》，请儿童文学作家[[陈伯吹|陈伯吹]]作“闪光美妙的画卷”序言，同时诗琳通公主本人也在该作品的中译本上撰写了中文前言。1983年8月，该作品的中文翻译版由[[少年儿童出版社_(上海)|少年儿童出版社]]发行，属于《外国儿童文学丛书》系列之一。印制首版5000册，其余50本样本转交给诗琳通公主。<ref>{{cite web |url=http://news.ifeng.com/gundong/detail_2013_01/02/20732261_0.shtml |title=“公主”、“王子”和“皇后” |last= |first= |date=2013-01-02 |website=凤凰网 |publisher=新民晚报 |access-date=2018-04-16 |quote=}}</ref>目前中文版《顽皮透顶的盖珥》已[[绝版|绝版]]，但泰文原版在泰国仍有印制发行。

2017年2月10日，泰国的[[南美出版社|南美出版社]]（Nanmee Books）出版了[[法语|法语]]版本，名为《{{lang|fr|Kaew la joyeuse}}》。<ref>{{cite web |url=https://www.matichon.co.th/news/458785 |title=‘แก้วจอมแก่น’ ฉบับภาษาฝรั่งเศส วรรณกรรมเยาวชนสุด ‘คลาสสิก’ |last= |first= |date=2017-02-10 |website=Matichon |publisher= |access-date=2018-04-16 |quote=}}</ref>
