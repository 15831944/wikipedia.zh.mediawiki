'''棉花嶼'''，俗稱'''花矸嶼'''，為[[台灣|台灣]]附屬島嶼之一。

位於[[基隆|基隆]]外海，是由[[火山作用|火山作用]]形成的島嶼。棉花嶼名稱取自因海鳥多，人船近岸時海鳥紛飛、宛如棉絮飛揚而名之。島嶼的最東端為[[中華民國|中華民國]]實際控制的極東點，[[台灣日治時期|日治時期]]為磷礦開採區。<ref name=bhihueu>記者盧賢秀,"北方三島小檔案"{{cite web |url=http://www.libertytimes.com.tw/2010/new/aug/30/today-fo8.htm |title=存档副本 |accessdate=2010-08-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20100901163440/http://www.libertytimes.com.tw/2010/new/aug/30/today-fo8.htm |archivedate=2010-09-01 }},[自由時報],2010-8-30.</ref> 行政區劃歸於[[基隆市|基隆市]][[中正區_(基隆市)|中正區]]。與[[花瓶嶼|花瓶嶼]]、[[彭佳嶼|彭佳嶼]]合稱[[北方三島|北方三島]]。[[海洋國家公園管理處|海洋國家公園管理處]]規劃把棉花嶼、[[花瓶嶼|花瓶嶼]]和[[彭佳嶼|彭佳嶼]]這塊海域劃為海洋國家公園。

===地理位置===
棉花嶼位於東經122.06度，北緯25.3度。距離基隆東北方約42公里，航程約2小時。島東西長約300公尺，南北長約400公尺。面積約0.12平方公里，海拔最高的地方只有64公尺。由於棉花嶼和彭佳嶼地理位置相近，形狀也相似，所以有不少漁民都誤將棉花嶼當作彭佳嶼。

===歷史現況===
棉花嶼最早於[[清朝|清朝]]便有人居住，1886 年自加拿大差派來台的傳教士[[馬偕|馬偕]]於記錄到許多鳥類於此島活動；[[台灣日治時期|日治時期]]也有定期船往返棉花嶼與基隆之間；[[國民政府|國民政府]]遷台之後劃入軍事管制區而無人居住。1996年，[[基隆市政府|基隆市政府]]向[[行政院|行政院]][[農委會|農委會]]提出「[[棉花嶼、花瓶嶼野生動物保護區|棉花嶼、花瓶嶼野生動物保護區]]」的規劃，以保護島上野生動植物。棉花嶼也是台灣北部唯一的海鳥繁殖地。台灣基隆漁民表示，棉花嶼和彭佳嶼，還存有珍貴稀有的第二級保育類鳥種[[白顶玄鸥|玄燕鷗]]和[[褐翅燕鸥|白眉燕鷗]]生存繁殖。2009年基隆野鳥學會人員發現屬瀕臨絕種的第一級保育類鳥種[[遊隼|遊隼]]，在棉花嶼繁殖。 <ref>記者盧賢秀、劉信德／彭佳嶼採訪報導,"他們，孤守國境之北"{{cite web |url=http://www.libertytimes.com.tw/2010/new/aug/30/today-fo5.htm |title=存档副本 |accessdate=2010-08-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20100901162549/http://www.libertytimes.com.tw/2010/new/aug/30/today-fo5.htm |archivedate=2010-09-01 }},[自由時報],2010-8-30.</ref>

===生態植物===
島上另一種特別的植物是俗稱「[[海芙蓉|海芙蓉]]」的[[蘄艾|蘄艾]]，屬於[[菊科|菊科]]多年生草本植物，喜歡生長在海岸線邊緣的岩石上，是一種[[稀有|稀有]]的[[名貴藥材|名貴藥材]]，用來治療[[氣喘|氣喘]]、[[肺結核|肺結核]]、[[高血壓|高血壓]]及[[背脊痠痛|背脊痠痛]]。
