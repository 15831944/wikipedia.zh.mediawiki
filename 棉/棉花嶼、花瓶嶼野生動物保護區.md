'''棉花嶼、花瓶嶼野生動物保護區'''位於[[臺灣|臺灣]][[基隆市|基隆市]]外海的[[棉花嶼|棉花嶼]]、[[花瓶嶼|花瓶嶼]]及兩島周圍海域，為依據《野生動物保育法》公告之野生動物保護區。此處保有火山地質，且是臺灣北部唯一的海鳥繁殖地<ref name="林務局">{{Cite web |url=http://conservation.forest.gov.tw/ct.asp?xItem=59950&ctNode=178&mp=10 |title=棉花嶼、花瓶嶼野生動物保護區 |accessdate=2013-01-11 |publisher=農委會林務局 |deadurl=yes |archiveurl=https://web.archive.org/web/20140623133103/http://conservation.forest.gov.tw/ct.asp?xItem=59950&ctNode=178&mp=10 |archivedate=2014-06-23 }}</ref>。保護區周圍海域也是重要經濟魚種的產卵及育幼處，因此內政部計畫將擴大兩島嶼和[[彭佳嶼|彭佳嶼]]的管制範圍，甚至是成立國家公園<ref>{{Cite web|url=http://erarc.epa.gov.tw/e/165/201011211734/archive/marine.c/chinese/filesys/dlarea/45/file2.pdf|title=北方三島自然資源調查計畫成果報告書|author=戴昌鳳等人|publisher=海洋國家公園管理處|date=2008-12|accessdate=2013-01-11}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==保護區範圍==
保護區合計陸域面積16.38公頃、海域210公頃，總計226.38公頃<ref name="林務局"/>。包括棉花嶼陸域13.3公頃，由陸域低潮線向外延伸五百公尺的海域面積188公頃，總計達201.3公頃<ref>{{Cite web |url=http://conservation.forest.gov.tw/ct.asp?xItem=3348&ctNode=176&mp=10 |title=棉花嶼野生動物重要棲息環境 |accessdate=2013-01-11 |publisher=農委會林務局 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304134753/http://conservation.forest.gov.tw/ct.asp?xItem=3348&ctNode=176&mp=10 |archivedate=2016-03-04 }}</ref>。花瓶嶼陸域面積為3.08公頃，低潮線向外延伸二百公尺的海域面積22公頃，總計25.08公頃<ref>{{Cite web |url=http://conservation.forest.gov.tw/ct.asp?xItem=3349&ctNode=176&mp=10 |title=花瓶嶼野生動物重要棲息環境 |accessdate=2013-01-11 |publisher=農委會林務局 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304103332/http://conservation.forest.gov.tw/ct.asp?xItem=3349&ctNode=176&mp=10 |archivedate=2016-03-04 }}</ref>。
