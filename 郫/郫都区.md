'''{{ruby-py|郫|-{zh-cn:pí}-}}都区'''，别称'''鹃城'''，是[[中华人民共和国|中国]][[四川省|四川省]][[成都市|成都市]]下辖的一个[[区|区]]，区人民政府驻[[郫筒街道|郫筒街道]]。

郫都区原名'''郫县'''。区花为[[杜鹃花|杜鹃花]]，区鸟为[[杜鹃|杜鹃]]，皆来自于“杜宇化鹃”的故事。<ref name=简介>{{cite web|title=郫县公共信息网—县情简介|url=http://www.pixian.gov.cn/index.php?cid=62&tid=64439|accessdate=2011-06-29|deadurl=yes|archiveurl=https://web.archive.org/web/20170305003220/http://www.pixian.gov.cn/index.php?cid=62&tid=64439|archivedate=2017-03-05}}</ref>

==地理==
郫都区位于[[成都市|成都市]]西北近郊，地处[[成都平原|成都平原]]中部。城域东至[[犀浦镇|犀浦镇]]万福村万福仓库，西至新胜乡天生村走石山，南至[[德源镇|德源镇]]平成村鱼凫桥，北至古城乡指路村通腴桥。城东南与成都市郊的[[金牛区|金牛区]]接壤，西部及西南隔江安、走马河与[[温江区|温江区]]相望，西北和[[都江堰市|都江堰市]]紧邻，北部与[[彭州市|彭州市]]以蒲阳河、柏条河为界，东北与[[新都区|新都区]]毗连。

境内地形以平坝为主。西北角有一小块浅丘台地，名“横山子”，为全区唯一的山丘。其海拔630-653[[米|米]]，比周围平原高出10-20[[米|米]]；占地4.42[[平方公里|km<sup>2</sup>]]，约占总面积1%。

==历史==
公元前316年秦灭蜀后，巴蜀地区实行分封制与郡县制。公元前314年，以[[郫邑|郫邑]]作为蜀郡的属县称郫县<ref name=简介/>，在秦汉时期，郫县辖地涉及今郫县、[[温江|温江]]、[[彭县|彭县]]、[[灌县|灌县]]等数县，之后历经数次变迁。中华人民共和国成立后，1950年属[[温江专区|温江专区]]，1968年属[[温江地区|温江地区]]，1983年改属成都市。

===大跃进===
1957年，曾获亩产千斤粮食帅旗的郫县，提出“学麻城，赶孝感”的口号，在大面积丰产上要“元帅升帐”，同时在发射高产“卫星”中也要在[[四川|四川]]领先。从收早稻开始，与附近的[[新繁县|新繁县]]竞争，发射一个比一个大的“卫星”。

为了开展发射水稻“[[卫星|卫星]]”的竞赛，他们专门在城关北街十字口竖立一个长十几米，宽数米的“卫星发射台”。每次发射一颗高产“卫星”，就使用一个装饰性的彩色水稻牌悬挂在“卫星台”上，象征“卫星”升起了。水稻牌的大小会随着产量的高低而变化。随后，“卫星台”上的水稻牌越来越多，越来越大。当年，全县共发射水稻“卫星”41个。<ref>{{cite web |title=大跃进运动中郫县放水稻高产卫星的内幕 |url=http://www.cdds.chengdu.gov.cn/detail.asp?id=4356 |accessdate=2011-06-28}}</ref>

=== 筹划改为郫都区 ===
2015年12月25日，郫县第十七届人大常委会第三十一次会议中，审议通过郫县人民政府“关于撤销郫县设立成都市郫都区的报告”<ref>{{Cite web |url=http://www.pxrd.gov.cn/index.php/v/detail?tid=10745 |title=郫县第十七届人大常委会第三十一次会议纪要 |accessdate=2016-09-03 |author= |date=2015-12-31  |work= |publisher=成都市郫县人民代表大会常务委员会网站，来源：郫县人大常委会办公室 |language=Zh-hans }}</ref>。
2016年9月11日，中共成都市委第十二届七次全会审议并通过了《关于同意申报郫县撤县设区的决议》，同意申报撤销郫县，设立郫都区<ref>{{Cite web |url=http://news.163.com/16/0911/17/C0MT3GNE00014SEH.html |title=成都市委十二届七次全会举行，同意申报撤销郫县设立郫都区 |accessdate=2016-09-11 |author= |date=2016-09-11 |work= |publisher=澎湃新闻 |language=Zh-hans }}</ref>。
2016年11月24日，经国务院批准，撤销郫县，设立成都市郫都区。

==文化==
===古迹===
* 郫都区古城村有[[成都平原史前城址|成都平原史前城址]]的[[郫县古城遗址|郫县古城遗址]]，为全国重点文物保护单位，为新石器时代城址。
* 郫都[[郫筒镇|郫筒镇]]有“望丛祠”，为四川省文物保护单位。其名为纪念古蜀国的望帝[[杜宇|杜宇]]和丛帝[[开明_(蜀国)|开明]]（又称鳖灵）。[[李商隐|李商隐]]的《[[s:锦瑟|锦瑟]]》一诗中“望帝春心托杜鹃”所说的就是他们的故事。

==经济==
2015年，全区实现地区生产总值426亿元（人民币，下同），比2014年增长8.5%。其中第一产业增加值21.1亿元，增长3.8%；第二产业增加值247.8亿元，增长9.5%；第三产业增加值157.1亿元，增长7.3%，三次产业结构4.9:58.2:36.9，三次产业对经济增长的贡献率分别为2.3%、68.8%和28.9%<ref>{{Cite web |url=http://www.chengdu.gov.cn/info/00020102/2016/8/18/a1e504b957fc408596c59f346ebdccccinfo.shtml&gws_rd=cr&ei=jinWV9yrOtj8jwPDuaTQCg|title=郫县2015年国民经济和社会发展统计公报 |accessdate=2016-09-11 |publisher=郫县政府办公室|language=Zh-hans }}</ref>。
