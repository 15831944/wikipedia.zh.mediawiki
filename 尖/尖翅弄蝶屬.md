'''尖翅弄蝶屬'''（學名：'''''Badamia'''''）-{zh-cn:在台湾称'''长翅弄蝶属''';zh-hk:在台灣稱'''長翅弄蝶屬''';zh-sg:在台湾称'''长翅弄蝶属''';zh-tw:在中國稱'''尖翅弄蝶屬''';}-，是[[弄蝶科|弄蝶科]][[豎翅弄蝶亞科|豎翅弄蝶亞科]]中的一個屬，包含了兩個物種<ref>Brower, Andrew V. Z. and Andrew Warren. 2010. [http://tolweb.org/Badamia/94258/2010.10.20 Badamia Moore 1881]. Version 20 October 2010 (under construction). in [ http://tolweb.org/ The Tree of Life Web Project] {{en}}</ref>。分佈於[[東洋界|東洋界]]、[[古北界|古北界]]和[[澳新界|澳新界]]。

== 形態 ==
成蟲中到大型。前翅狹長，褐色，頂角尖出，翅面有透明斑；中室很長，長大於前翅後緣。後翅2A脈尖出，略呈尾狀，2A 脈與 Cu<sub>2</sub> 脈間凹入，臀角明顯呈瓣狀突出。觸角短，不及前翅長的二分之一。腹部有橫紋。雄蝶後足脛節有一束豎立毛簇，嵌在胸部的窩內。

== 寄主 ==
* 使君子科：[[欖仁樹屬|欖仁樹屬]]、[[風車藤屬|風車藤屬]]
* 木犀科：[[插柚紫屬|插柚紫屬]]
* 桑科：[[榕屬|榕屬]]
* 金虎尾科：[[飛鳶果屬|飛鳶果屬]]、[[驟翅果屬|驟翅果屬]]、[[三星果屬|三星果屬]]
* 蝶形花科：[[水黃皮屬|水黃皮屬]]

== 物種 ==
* [[太平洋長翅弄蝶|太平洋長翅弄蝶]] ''B. atrox'' <small>(Butler, 1877)</small>
* [[長翅弄蝶|尖翅弄蝶]] ''B. exclamationis'' <small>(Fabricius, 1775)</small>
