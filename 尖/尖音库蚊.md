'''尖音库蚊'''（学名：''Culex pipiens''），又称'''淡色库蚊'''、'''家蚊'''，是[[蚊科|蚊科]]的一种吸血蚊子。这个物种是一些疾病的[[载体_(生物学)|载体]]，如[[日本脑炎|日本脑炎]]、[[脑膜炎|脑膜炎]]、[[荨麻疹|荨麻疹]]，在美国，它传播[[西尼罗河病毒|西尼罗河病毒]]。

== 分布 ==
该物种在全球主要大洲均有分布，包括欧洲、亚洲、非洲、南美洲和北美洲。

它分布在以下国家：[[阿根廷|阿根廷]]、[[波斯尼亚和黑塞哥维那|波斯尼亚和黑塞哥维那]]、[[保加利亚|保加利亚]]、[[加拿大|加拿大]]、[[塞浦路斯|塞浦路斯]]、[[捷克|捷克]]、[[埃及|埃及]]、[[法国|法国]]、[[德国|德国]]、[[希腊|希腊]]、[[匈牙利|匈牙利]]、[[伊朗|伊朗]]、[[以色列|以色列]]、[[日本|日本]]、[[约旦|约旦]]、[[拉脱维亚|拉脱维亚]]、[[黎巴嫩|黎巴嫩]]、[[立陶宛|立陶宛]]、[[卢森堡|卢森堡]]、[[摩洛哥|摩洛哥]]、[[巴基斯坦|巴基斯坦]]、[[波兰|波兰]]、[[葡萄牙|葡萄牙]]、[[罗马尼亚|罗马尼亚]]、[[俄罗斯|俄罗斯]]、[[沙特阿拉伯|沙特阿拉伯]]、[[斯洛伐克|斯洛伐克]]、[[西班牙|西班牙]]、[[瑞典|瑞典]]、[[塔吉克斯坦|塔吉克斯坦]]、[[突尼斯|突尼斯]]、[[土耳其|土耳其]]、[[英国|英国]]、[[塞尔维亚|塞尔维亚]]、[[美国|美国]]、[[乌拉圭|乌拉圭]]、[[蒙特內哥羅|蒙特內哥羅]]。<ref>[http://www.mosquitocatalog.org/htmlfiles/genus17.htm Genus ''Culex''] {{webarchive|url=https://web.archive.org/web/20070717145405/http://www.mosquitocatalog.org/htmlfiles/genus17.htm |date=2007-07-17 }} Systematic Catalog of the Culicidae</ref>

== 描述 ==
尖音库蚊长度为3-7毫米，身体细长，翅膀狭长，腿很长。腹部为带状棕色和白色。它们有羽状触角。

== 习性 ==
雌性以吸血为生，尤其是人类的血液。雄性吸食花粉，花蜜和植物汁液。
