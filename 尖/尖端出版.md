'''尖端出版'''（Sharp Point Publishing），簡稱'''SPP'''，是[[台灣|台灣]][[城邦文化|城邦媒體控股集團]]旗下的出版事業部，以發行[[漫畫|漫畫]]、[[輕小說|輕小說]]和書籍為主。

== 歷史 ==
[[File:Sharp_Point_Press_goods_window_in_Comic_Exhibition_20140810.jpg|缩略图]]尖端出版攤位]]
[[File:Sharp_Point_Press_booth_goods_window_20160211.jpg|缩略图]]尖端出版攤位]]
*1982年7月12日，「尖端出版有限公司」成立，總部設於[[臺北縣|臺北縣]][[新店市|新店市]]復興路45號6樓，主要出版[[軍事|軍事]]、[[模型|模型]]相關書籍。[[黃鎮隆|黃鎮隆]]為創辦人之一兼負責人。
*1987年7月，[[電玩雜誌|電玩雜誌]]《[[電視遊樂雜誌|電視遊樂雜誌]]》創刊。
*1987年10月，電玩雜誌《[[電視遊樂報導|電視遊樂報導]]》（''TV Game Report''）創刊。
*1990年，動畫、[[漫畫|漫畫]]、模型情報雜誌《[[神奇地帶|神奇地帶]]》（''Magic''）創刊，正式投入漫畫出版業。
*1991年，開始發行已取得[[授權|授權]]的[[日本漫畫|日本漫畫]]，第一部漫畫是[[小學館|小學館]]授權發行的[[高橋留美子|高橋留美子]]漫畫作品《[[福星小子|福星小子]]》（{{lang|ja|うる星やつら}}）。
*1993年，電玩雜誌《勝利小子》（''VV Kids''）、《遊戲天堂》、《電玩攻略月刊》相繼創刊。
*1993年10月，[[青年漫畫|青年漫畫]]雜誌《活躍Tempo半月刊》（''Tempo''）創刊。
*1994年1月，流行生活[[占星|占星]]情報雜誌《談星》（''My Birthday''）創刊。
*1997年11月，街頭流行文化情報雜誌《COOL流行酷報》（''COOL''）創刊。
*1998年4月，[[流行音樂|流行音樂]]及藝人情報雜誌《Play流行樂刊》（''Play''）創刊。
*1999年5月，[[手機|手機]]及其周邊商品情報雜誌《Call流行通訊》（''Call''）創刊。
*1999年6月，收藏[[玩具|玩具]]情報雜誌《COOL TOYS玩具酷報》（''COOL TOYS''）創刊。同年，與[[講談社|講談社]]同步製作、同步發售[[井上雄彥|井上雄彥]]漫畫作品《[[浪人劍客|浪人劍客]]》（{{lang|ja|バガボンド}}），並邀請台灣[[書法家|書法家]][[黃明勝|黃明勝]]擔任《浪人劍客》台灣中文版封面題字。
*1990年代後期，尖端出版改組為「尖端出版股份有限公司」。
*2000年2月，[[心理測驗|心理測驗]]雜誌《心理測驗》（''Psychological Tests''）雙月刊創刊。
*2001年1月，十二星座專門雜誌《完全星座誌》（''Total Astrology''）雙月刊創刊。同年5月，[[女性|女性]]街頭流行文化情報雜誌《Choc恰女生》（''Choc''）創刊，《Play流行樂刊》改版為《Play綜藝王》（''Play''）。同年，尖端出版被[[TOM集團|TOM集團]]收購。
*2002年5月，日本流行裝扮情報雜誌《Popteen》創刊。
*2002年6月，[[機車|機車]]情報誌《機車人月刊》（''COOL BIKERS''）創刊。
*2003年，[[電子報|電子報]]《尖端數-{}-位e週報》（''Sharp Point E-Paper Weekly''）創刊，《心理測驗》雙月刊改版為《非常心理測驗》（''Psychological Tests''）月刊，集合[[網路遊戲|網路遊戲]]玩家共同撰寫的網路遊戲密技攻略雜誌《網路遊戲密技大補帖》（''Online Game Guides Monthly''）月刊創刊。
*2003年7月25日，因[[大然文化|大然文化]][[倒閉|倒閉]]事件，隨後獲得[[集英社|集英社]]刊物《[[RIBON|RIBON]]》的台灣代理發行權，發行《[[夢夢少女漫畫月刊|夢夢少女漫畫月刊]]》（''MonMon''），每月25日發行，遇假日提前、順延。
*2003年12月，[[日本|日本]]偶像情報雜誌《Junon》創刊。
*2005年1月，[[喬傑立|喬傑立]]偶像情報雜誌《傑報J★Star》（''J★Star''）創刊。
*2005年7月15日，合併解散<ref>2005年7月15日[[臺北市政府|臺北市政府]]「府建商字第0941034871號」。解散前[[統一編號|統一編號]]15918129。</ref>，成為城邦媒體控股集團旗下的[[分公司制|品牌公司]]。
*2005年9月，[http://www.cool-style.com.tw/wd/ 《COOL流行酷報》]衍生雜誌《Men's Cool》創刊。
*2006年1月，取得日本[[Apollo_Communication|Apollo Communication]]旗下雜誌《APOLLO Brand Bargain》授權，開始發行中文版的[[名牌|名牌]]商品情報雜誌《i-Brand 愛名牌》（''i-Brand''）。
*2007年8月12日，取得[[小學館|小學館]]《[[Ciao|Ciao]]》的台灣代理權，於[[漫畫博覽會|漫畫博覽會]]發行中文版《[[甜芯少女漫畫月刊|甜芯少女漫畫月刊]]》，每月1日發行，遇假日提前、順延。
*2011年，於上海設立[[辦事處|辦事處]]「尖端文創」<ref>法人登記為「上海唐码城邦咨询有限公司」，實際成立時間為2004年8月31日，由TOM PRINT MEDIA GROUP LIMITED（TOM集團成員）全資設立。</ref>。

== 出版品 ==
=== 雜誌 ===
; 漫畫雜誌
* [[夢夢少女漫畫月刊|夢夢少女漫畫月刊]] - [[集英社|集英社]]月刊少女漫畫雜誌《[[Ribon|Ribon]]》的繁體中文版。
* [[甜芯少女漫畫月刊|甜芯少女漫畫月刊]] - [[小學館|小學館]]月刊少女漫畫雜誌《[[Ciao|Ciao]]》的繁體中文版。

=== 輕小說 ===
; 浮文字
: 尖端於2006年1月開始發行的[[輕小說|輕小說]]書系<ref name="Faust">[http://www.spp.com.tw/spp2006/introduction/novel.htm 尖端出版 出版理念（2006年）]</ref>。發行初號稱「引進日本文學新星最具爆發力與想像力的新文體」<ref name="Faust"/>。同年2月，尖端開始發行日本[[講談社|講談社]]文藝雜誌《Faust》的中文翻譯版[[Mook|Mook]]《[[浮文誌|浮文誌]]》<ref name="Faust"/>。

: 最初以翻譯出版日本《Faust》系作家的作品為主，現在則發行各種日本輕小說和中文原創輕小說。

; 書盒子
: 尖端的盒裝日本輕小說書系。設計和代理書目均來自[[講談社BOX|講談社BOX]]系列。

; MoonBleu
: 尖端的[[BL|BL]]向小說書系。

; 夢小說
: 尖端的少女向小說書系。主要發行[[少女漫畫|少女漫畫]]改編小說和中文原創少女小說。
