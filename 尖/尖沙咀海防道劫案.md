'''尖沙咀海防道劫案'''發生於2018年7月1日。3名劫匪在2分鐘內劫去總值2,000萬港元的手錶及首飾。此劫案是短期內香港第三宗針對錶行的劫案。

==經過==
在7月1日早上10時45分，3名戴口罩、操香港口音的賊匪衝入[[尖沙咀|尖沙咀]][[海防道|海防道]]44號一間珠寶店，亮刀指嚇職員，並拿鐵錘打破店內一個玻璃飾櫃，搶走一批總值2,000萬港元的手錶及首飾，賊人得手後登上接應[[私家車|私家車]]逃去，全程歷時約一分鐘。<ref>{{Cite news |url=http://hk.on.cc/hk/bkn/cnt/news/20180701/bkn-20180701110540974-0701_00822_001.html |title=3匪錘劫海防道珠寶店　掠走2000萬元名錶首飾 |publisher=[[東方日報|東方日報]]}}</ref>在劫掠過程中，一名職員因正在使用手機，被劫匪懷疑其報警，遂掌摑該店員，其事後送院治理。

==調查==
劫案發生兩小時後，警方在[[落馬洲|落馬洲]]發現案中所用私家車，該車於2018年3月報失，發現時已被燒毀<ref>{{Cite news |url=https://www.hk01.com/%E7%AA%81%E7%99%BC/205559/%E4%B8%89%E5%8C%AA%E5%8A%AB%E5%90%8D%E9%8C%B6-%E6%96%B0%E7%94%B0%E7%81%AB%E7%87%92%E8%BB%8A%E6%9C%89%E5%8F%AF%E7%96%91-%E6%8E%A5%E6%87%89%E7%A7%81%E5%AE%B6%E8%BB%8A%E9%81%AD%E6%AF%80%E5%B1%8D%E6%BB%85%E8%BF%B9 |title=【三匪劫名錶】新田火燒車有可疑　接應私家車遭毀屍滅迹 |publisher=[[香港01|香港01]]}}</ref>。

在7月4至10日，警方再拘捕3男1女，分別涉嫌行劫及協助疑犯逃走。<ref name ="Topick">{{Cite web |url=https://topick.hket.com/article/2112257/%E5%B0%96%E6%B2%99%E5%92%80%E9%80%BE2300%E8%90%AC%E5%85%83%E7%8F%A0%E5%AF%B6%E5%8A%AB%E6%A1%88%E8%AD%A6%E6%8B%984%E7%94%B7%E5%A5%B3%E3%80%80%E6%83%9F%E8%B4%93%E7%89%A9%E4%B8%8B%E8%90%BD%E6%9C%AA%E6%98%8E%20%C2%A0 |title=尖沙咀逾2300萬元珠寶劫案警拘4男女　惟贓物下落未明 }}</ref><ref name = "headline">{{Cite web |url=http://hd.stheadline.com/news/realtime/hk/1283023/ |title=【2千萬械劫案】疑犯「螞蟻搬家」形式運贓物返內地 兩名涉案男子仍在逃}}</ref>至7月17及19日，深圳公安在[[龍崗|龍崗]]及[[福田|福田]]拘捕三名香港人及兩名內地人。被捕的3名港人為械劫集團骨幹成員，另兩名內地男女是協助將贓物脫手<ref name = "headline"></ref>，被捕疑犯於8月4日由深圳公安移交香港警方<ref>{{Cite web |url=http://hd.stheadline.com/news/realtime/hk/1282818/ |title=【2千萬械劫案】深圳公安向香港警方移交3疑犯 據悉兩人為主腦}}</ref>。香港警方於19日再拘捕1男3女，涉嫌處理案中贓物<ref name = "headline"></ref>。警方相信案中至今仍然有兩人在逃<ref name = "headline"></ref>。
