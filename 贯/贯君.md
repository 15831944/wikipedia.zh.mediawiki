'''贯君'''（又名：应君）真名不详，（{{bd|1980年|9月27日||catIdx=G贯}}）北京美悦玺美容技术有限公司法人，[[海航集团|海航集团]]大股东。<ref>{{cite web |url=http://cn.rfi.fr/%E4%B8%AD%E5%9B%BD/20170726-%E6%B5%B7%E8%88%AA%E7%A5%9E%E7%A7%98%E5%A4%A7%E8%82%A1%E4%B8%9C%E5%BC%83%E8%82%A1-%E5%9B%BD%E9%99%85%E5%AA%92%E4%BD%93%E8%81%9A%E7%84%A6%E7%96%91%E4%BC%BC%E7%8E%8B%E5%B2%90%E5%B1%B1%E5%AE%B6%E4%BA%BA%E8%B4%AF%E5%90%9B |title=海航神秘大股东弃股 国际媒体聚焦疑似王岐山“家人”贯君|publisher=法国广播电台 |date=2017-07-26 |accessdate=2017-08-15}}</ref>

== 持股[[海航集团|海航集团]]1年 ==
贯君在[[海航|海航]]的股份是通过以下两家分别在[[开曼群岛|开曼群岛]]和[[香港|香港]]注册的公司持有的：Pan American Aviation Holding和千江源投资(Headstreams Investment)。这两家公司此前由在香港生活的航空业商人巴拉特•拜斯(Bharat Bhise)拥有，拜斯将[[海航|海航]]介绍给其最知名的投资者[[乔治·索罗斯|乔治·索罗斯]](George Soros)。<ref>{{cite web |url=http://www.ftchinese.com/story/001073541 |title=海航股权变动引发疑问|publisher=英国《金融时报》 |date=2017-07-25 |accessdate=2017-08-15}}</ref>[[海航|海航]]行政总裁[[谭向东|谭向东]]接受英国《金融时报》访问时亦表示，贯君曾经持有过[[海航|海航]]股份。<ref>{{cite web|url=http://news.etnet.com.cn/all-jishixinwen/270726760.htm|title=海航谭向东称贯君一直在海外代持股；据报已全数转让|publisher=经济通通讯社|date=2017-07-26|accessdate=2017-08-15|deadurl=yes|archiveurl=https://web.archive.org/web/20170816060409/http://news.etnet.com.cn/all-jishixinwen/270726760.htm|archivedate=2017-08-16}}</ref>

== 被指是[[王岐山|王岐山]]家人 ==
据《纽约时报》2017年7月25日报道，[[海航集团|海航集团]]表示，前不久，它最大的股东、一个名叫贯君（音）的中国私营商人突然把他在该公司持有的30%的股份捐给了在美国注册的非营利机构。有些文件表明，近几年，其近30%的股权转向了[[巴拉特·拜斯|巴拉特·拜斯]](Bharat Bhise)控制的几家离岸公司或商人贯君。拜斯是该公司的长期顾问，经营Bravia Capital公司。<ref>{{cite web |url=https://cn.nytimes.com/business/20170725/chinas-hna-discloses-shift-of-ownership-stake-to-foundation/ |title=海航神秘大股东把所持股份捐给公司旗下基金会|publisher=纽约时报 |date=2017-07-25 |accessdate=2017-08-15}}</ref><ref>{{cite web |url=http://chinadigitaltimes.net/chinese/2017/07/%E3%80%90%E6%96%87%E8%B4%B5%E4%BC%90%E8%B5%B5%E3%80%91%E9%83%AD%E6%96%87%E8%B4%B57%E6%9C%8826%E6%97%A5%E6%8A%A5%E5%B9%B3%E5%AE%89%E7%9B%B4%E6%92%AD%E8%A7%86%E9%A2%91%EF%BC%9A%E8%B4%AF%E5%90%9B/ |title=【文贵伐赵】贯君，刘呈杰的找爹大戏！|publisher=中国数字时代 |date=2017-07-26 |accessdate=2017-08-15}}</ref>
