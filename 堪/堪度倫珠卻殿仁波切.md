'''堪度倫珠卻殿仁波切'''{{BD|1954年| |2018年|9月23日|楊}}，俗名'''楊開雲'''，出生於台中，高中就讀於[[臺北市立成功高級中學|成功高中]]，大學就讀[[政治大學|政治大學]]中文系，在1988年取得美國[[威斯康辛大學麥迪遜分校|威斯康辛大學麥迪遜分校]]社會學碩士，1992年取得[[加州大學洛杉磯分校|加州大學洛杉磯分校]]社會學博士。返回台灣後，於[[東海大學_(台灣)|東海大學]]社會學擔任副教授，及在[[弘光科技大學|弘光科技大學]]的通識教育中心擔任客座副教授。2009年接受[[第十四世達賴喇嘛|第十四世達賴喇嘛]]認證後，辭去學校工作，於[[台灣|台灣]]、[[中國大陸|中國大陸]]、[[美國|美國]]等世界各地指導弟子。<ref>{{Cite web |url = https://www.lama.com.tw/content/msg/data.aspx?id=9678|title =楊開雲教授（堪度倫珠卻殿仁波切）簡介|publisher =喇嘛網|language = 中文 |accessdate = 2018-12-9 }}</ref><ref>{{Cite web |url = https://www.facebook.com/profile.php?id=599487177|title =楊開雲臉書|language = 中文 |accessdate = 2018-12-9 }}</ref><ref>{{Cite web |url = https://www.taiwan-mth-nalanda.org/guru_khentul.html|title =堪度倫珠卻殿仁波切小傳|publisher =台灣蒙藏漢那蘭陀佛學研究會|language = 中文 |accessdate = 2018-12-9 }}</ref><ref name = ' soc ' >{{Cite web |url = http://soc.thu.edu.tw/news/20181004.pdf|title =惋悼 楊開雲老師|publisher =東海大學社會學系|language = 中文 |accessdate = 2018-12-9 }}</ref><ref name = ' chinesege ' >{{Cite web | url = http://www.chinesege.org.tw/geonline/html/page4/publish_pub.php?Pub_Sn=59&Sn=412|title =「以人為本，關懷生命」 弘光科技大學| author=方心怡|publisher =通識在線雜誌社|language = 中文 |accessdate = 2018-12-9 }}</ref>

==教學經歷==	
任教東海大學期間，開授「[[社會學|社會學]]」、「社會思想史」、「政治經濟學：[[政治學|政治學]]」、「宗教社會學專題：[[藏傳佛教|藏傳佛教]]精義」、「臺灣社會專題研究」、「中國社會思想專題：[[莊子|莊子]]與佛學」、「藏傳佛法專題：[[菩提心|菩提心]]」…等課程<ref name = ' soc '/>。2004年負責規劃弘光科技大學通識教育的「人文精神課程」，於課程中討論[[生命|生命]]、[[死亡|死亡]]、[[快樂|快樂]]、認識自我、肯定自我，及實踐環保生態體系的議題。<ref name = ' chinesege '/>
