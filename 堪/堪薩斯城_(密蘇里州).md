'''堪薩斯城'''（{{lang-en|Kansas City, Missouri}}，常縮寫成''KCMO''）为[[美国|美国]][[密苏里州|密苏里州]]的一座城市，包含[[密蘇里州|密蘇里州]][[傑克遜縣_(密蘇里州)|傑克遜]]（Jackson）、[[克萊縣_(密蘇里州)|克萊]]（Clay）、[[卡斯縣_(密蘇里州)|卡斯]]（Cass）和[[普拉特縣_(密蘇里州)|普拉特]]（Platte）四個縣，是傑克遜縣最大城市、密蘇里州最大城市，[[美國|美國]]第三十七大城市。位處[[密蘇里河|密蘇里河]]與[[堪薩斯河|堪薩斯河]]交匯處、密蘇里州及[[堪薩斯州|堪薩斯州]]交界，面對[[堪薩斯城_(堪薩斯州)|堪薩斯州堪薩斯城]]（堪薩斯城橫跨密蘇里州與堪薩斯州，因此兩州在州界兩側各設有一個同名城市）。
 
在[[美國中西部|中西部]]是第七大城市，界乎[[克里夫蘭_(俄亥俄州)|克里夫蘭]]和[[奧馬哈|奧馬哈]]之間。根據[[美國2016年人口普查|美國2016年人口普查]]，人口481,420人。加上堪薩斯州堪薩斯城人口588,411，都會區人口2,159,159。由於城市及都會區不斷成長，[[英國|英國]][[伯明翰大學|伯明翰大學]]的「全球化與世界級城市研究小組與網絡」（the Globalization and World Cities Study Group and Network），把堪薩斯城列為「具世界級城市潛力城市」之一。

==地理与气候==

堪萨斯城气候代表美国中西部温带，属[[溫帶大陸性濕潤氣候|溫帶大陸性濕潤氣候]]，四季分明。冬季寒冷，潮湿，但日照相对充足，日最高气温低于{{convert|0|C}}的平均日数为21天，日最低气温低于{{convert|-10|C}}的平均日数为18天，低于{{convert|-15|C}}的有6.1天；夏季炎热潮湿，日最高气温超过{{convert|30|C}}的日数年均有76天，{{convert|35|C}}以上的有20天。<ref name = "NWS Pleasant Hill, MO"/>最冷月（1月）均温{{convert|31.0|F|1|disp=flip}}，极端最低气温{{convert|−23|F|0|disp=flip}}（1989年12月22日与23日）。<ref name = "NWS Pleasant Hill, MO"/>最热月（7月）均温{{convert|81.0|F|1|disp=flip}}，极端最高气温{{convert|113|F|0|disp=flip}}（1936年8月14日）。<ref name = "NWS Pleasant Hill, MO"/>无霜期平均为209天（4月5日至10月30日）；可测量降雪平均期为11月27日至3月16日。<ref name = "NWS Pleasant Hill, MO"/>年均降水量约{{convert|39.1|in|sigfig=3|disp=flip}}，年极端最少降水量为{{convert|20.93|in|disp=flip}}（1953年），最多为{{convert|60.25|in|disp=flip}}（1961年）。<ref name = "NWS Pleasant Hill, MO"/>年均降雪量为{{convert|18.8|in|cm|disp=flip}}；2011–12年的降雪量最少，积累降雪量只有{{convert|3.9|in|cm|disp=flip}}，1911–12年的降雪量最多，积累降雪量为{{convert|67.0|in|cm|disp=flip}}。<ref name = "NWS Pleasant Hill, MO"/>
