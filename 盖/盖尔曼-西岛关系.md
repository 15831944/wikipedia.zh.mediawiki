|bibcode = 1953PThPh..10..581N }}</ref>首先提出，1955年由美国物理学家[[默里·盖尔曼|默里·盖尔曼]]完成的。此关系指[[强子|强子]]的[[电荷|电荷]]Q、[[同位旋第三分量|同位旋第三分量]]I<sub>3</sub>、[[重子数|重子数]]b、[[奇异数|奇异数]]S满足以下的关系：

:<math>Q = I_{3} + \frac{1}{2}(b + S)=I_{3} + \frac{1}{2} Y </math>

其中<math>Y = ( b + S )</math>称为[[超荷|超荷]]。

随着1974年发现了[[粲数|粲数]]C、1977年发现了[[底数_(物理学)|底数]]B、1995年发现了[[顶数|顶数]]T，盖尔曼-西岛关系扩展为：

:<math>Q = I_{3} + \frac{1}{2}(b + S + C + B + T)=I_{3} + \frac{1}{2} Y </math>

其中<math>Y = ( b + S + C + B + T)</math>.

目前已经发现的所有强子都满足这一关系。
