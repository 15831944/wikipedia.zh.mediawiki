'''盖如垠'''（{{bd|1953年|3月|catIdx=盖}}），[[辽宁省|辽宁省]][[东港市|东港市]]人，1970年9月参加工作，1973年6月加入[[中国共产党|中国共产党]]，[[东北大学|东北大学]]工商管理学院管理工程专业在职研究生毕业，工学硕士，工程师。曾于2009年8月至2012年1月担任[[中共黑龙江省委|中共黑龙江省委]]常委、[[哈尔滨|哈尔滨]][[市委书记|市委书记]]<ref>{{cite web |url=http://leaders.people.com.cn/GB/9907960.html |language=中文 |title=盖如垠同志简历 |publisher=[[人民网|人民网]] |date=2009年8月21日 |accessdate=2011年10月31日 |deadurl=no |archiveurl=https://web.archive.org/web/20111027210833/http://leaders.people.com.cn/GB/9907960.html |archivedate=2011年10月27日 }}</ref>。

==履历==
*1970年9月－1974年4月，[[辽宁省|辽宁省]][[沈阳|沈阳]]气压机厂工人、车间副指导员；
*1974年4月－1975年1月，[[共青团|共青团]]辽宁省沈阳市委工作人员；
*1975年1月－1978年9月，共青团辽宁省沈阳市委常委、学校部部长；
*1978年9月－1979年4月，辽宁省沈阳市工业安装公司党委办公室副主任；
*1979年4月－1982年7月，辽宁广播电视大学机械专业学生；
*1982年7月－1983年4月，辽宁省沈阳市工业安装公司一队副队长；
*1983年4月－1984年8月，辽宁省沈阳市工业安装公司一处副主任；
*1984年8月－1984年9月，辽宁省沈阳市工业安装公司一处主任；
*1984年9月－1989年11月，辽宁省沈阳市工业安装公司党委书记、副经理；
*1989年11月－1992年12月，共青团辽宁省沈阳市委书记；
*1992年12月－1993年8月，辽宁省沈阳国际经济技术合作公司副总经理；
*1993年8月－1996年1月，辽宁省沈阳市[[辽中县|辽中县]]委书记；
*1996年1月－1998年1月，辽宁省沈阳市市长助理；
*1998年1月－1998年12月，辽宁省沈阳市副市长。
*1998年12月－2002年10月，[[黑龙江省|黑龙江省]][[哈尔滨市|哈尔滨市]]副市长；
*2002年10月－2004年10月，黑龙江省[[大庆市|大庆市]]委副书记、市长；
*2004年10月－2007年4月，黑龙江省大庆市委书记；
*2007年4月－2008年2月，[[中共黑龙江省委|中共黑龙江省委]]常委、大庆市委书记；
*2008年1月－2009年8月，中共黑龙江省委常委、副省长；
*2009年8月－2012年1月，中共黑龙江省委常委、中共哈尔滨市委委员、常委、书记；
*2012年1月－2012年3月，中共黑龙江省委常委；
*2012年5月－2013年1月，黑龙江省人大常委会党组书记；
*2013年1月－2015年12月，黑龙江省人大常委会副主任、党组书记。

== 落马 ==
2015年12月8日，因涉嫌严重违纪，接受组织调查<ref name="简历">{{cite web| url=http://www.ccdi.gov.cn/xwtt/201512/t20151208_69771.html| language=中文| title=黑龙江省人大常委会党组书记、副主任盖如垠涉嫌严重违纪接受组织调查| publisher=[[中央纪委监察部网站|中央纪委监察部网站]]| accessdate=2015-12-08| deadurl=no| archiveurl=https://archive.is/20151208013653/http://www.ccdi.gov.cn/xwtt/201512/t20151208_69771.html| archivedate=2015-12-08}}</ref>。他是[[中共十八大|中共十八大]]后[[黑龙江省|黑龙江省]]第三位落马的省部级官员。据《法制晚报》报道，2014年9月，盖如垠曾因主政哈尔滨期间的“失地补偿”问题遭到哈尔滨市道外区民主镇新立村和光明村村民的联名举报。举报信称，2010年，市长盖如垠、民主乡书记黄某等领导强行按每平米38.5元低价将8300亩土地收购。转手出卖给国家每平米148元。从中谋利。举报信还称，2011年8月8日上午，在农户毫不知情的情况下，雇佣黑社会上千余人，手持砍刀、铁锹等凶器，到农田里恶意毁坏青苗。这一行为激怒村民，双方发生激烈的殴斗，造成两死一伤<ref>{{cite web|url=http://www.fawan.com/Article/fwkx/2015/12/08/100143317317.html|title=黑龙江人大副主任盖如垠落马 去年曾遭村民实名举报|publisher=法制晚报|accessdate=2015-12-08|deadurl=no|archiveurl=https://web.archive.org/web/20151208232126/http://www.fawan.com/Article/fwkx/2015/12/08/100143317317.html|archivedate=2015-12-08}}</ref>。另外，2015年12月28日，据《新京报》报道，挂职于哈尔滨市建委的盖如垠之子[[盖阔|盖阔]]（原名盖宝华）因插手工程建设、利用其父职权牟利，亦被带走调查<ref>{{cite web|url=http://news.ifeng.com/a/20151228/46854798_0.shtml|title=哈尔滨“造城书记”之子被带走 圈内人缘差|publisher=新京报|accessdate=2015-12-28|deadurl=no|archiveurl=https://web.archive.org/web/20151228000405/http://news.ifeng.com/a/20151228/46854798_0.shtml|archivedate=2015-12-28}}</ref>。因涉嫌严重违纪，2015年12月18日，黑龙江省第十二届人大常委会第二十三次会议决定接受其辞去[[第十二届全国人民代表大会代表|第十二届全国人民代表大会代表]]职务<ref>{{cite news|title=第十二届全国人民代表大会常务委员会代表资格审查委员会关于个别代表的代表资格的报告（2015年12月27日在第十二届全国人民代表大会常务委员会第十八次会议通过）|url=http://www.npc.gov.cn/wxzl/gongbao/2016-02/26/content_1987067.htm|accessdate=2016-07-04|newspaper=中国人大网|deadurl=no|archiveurl=https://web.archive.org/web/20160828194206/http://www.npc.gov.cn/wxzl/gongbao/2016-02/26/content_1987067.htm|archivedate=2016年8月28日}}</ref><ref name=2015-12a>{{cite news|author1=全国人民代表大会常务委员会|title=全国人民代表大会常务委员会公告 〔十二届〕第十六号|url=http://www.npc.gov.cn/npc/xinwen/2015-12/28/content_1957461.htm|accessdate=2016-01-09|agency=全国人民代表大会常务委员会|date=2015年12月27日|deadurl=no|archiveurl=https://web.archive.org/web/20170728120810/http://www.npc.gov.cn/npc/xinwen/2015-12/28/content_1957461.htm|archivedate=2017年7月28日}}</ref>。2016年2月3日，被开除党籍、公职。

2017年4月13日，[[天津市第一中级人民法院|天津市第一中级人民法院]]一审宣判盖如垠犯受贿罪，判处有期徒刑14年，并处罚金人民币200万元；对盖如垠受贿所得财物及其孳息予以追缴，上缴国库<ref>{{cite web |url = http://www.ccdi.gov.cn/yw/201704/t20170413_97244.html |title = 黑龙江省人大常委会原副主任盖如垠受贿案一审宣判 |accessdate = 2017年4月13日 |publisher = 中央纪委监察部网站}}</ref>。
