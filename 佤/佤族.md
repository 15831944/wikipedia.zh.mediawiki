'''佤族'''（佤文：{{lang|prk|Ba Rāog<ref>{{cite book |url=http://mall.cnki.net/reference/ref_readerItem.aspx?bid=R200610031&recid=R2006100310000264 |title=《佤汉简明词典》 |author=颜其香等 |date=1981 |publisher=[[云南民族出版社|云南民族出版社]]}}</ref> / si viex<ref>{{cite book |url=http://mall.cnki.net/reference/ref_readerItem.aspx?bid=R200610031&recid=R2006100310008392 |title=《佤汉简明词典》 |author=颜其香等 |date=1981 |publisher=[[云南民族出版社|云南民族出版社]]}}</ref> / Brāog / Vax Ba Rāog / Vāx qux}} （口语） / {{lang|prk|Vā}} （文学语），旧佤文：{{lang|prk|Praok}} / {{lang|wbm|Va}}；中国标准罗马字母：'''Va'''<ref>GB 3304-91 [http://www.moe.edu.cn/edoas/website18/22/info16422.htm 中国各民族名称的罗马字母拼写法和代码] {{webarchive|url=https://web.archive.org/web/20100503143709/http://www.moe.edu.cn/edoas/website18/22/info16422.htm |date=2010-05-03 }}</ref>）是[[亞洲|亞洲]]的一個[[跨國民族|跨國民族]]，主要分佈在[[瀾滄江|瀾滄江]]與[[薩爾溫江|薩爾溫江]]之間的山地，分屬於[[緬甸|緬甸]]與[[中華人民共和國|中國]]<ref name="sinica" />。在[[緬甸|緬甸]]境內超過60萬人，主要分佈於[[佤邦|佤邦]]，也分布於[[撣邦|撣邦]]。在中國約43萬人，主要分佈在[[雲南省|雲南省]]西南部的[[西盟佤族自治縣|西盟佤族自治縣]]和[[滄源佤族自治縣|滄源佤族自治縣]]，其餘分佈在[[瀾滄|瀾滄]]、[[孟連|孟連]]、[[雙江|雙江]]、[[耿馬|耿馬]]、[[鎮康|鎮康]]等縣。传统上佤族人信仰[[萬物有靈|萬物有靈]]，现在则多信仰[[基督教|基督教]]或[[佛教|佛教]]。有木鼓崇拜。佤族使用[[佤語|佤語]]，佤語屬[[南亞語系|南亞語系]][[孟-高棉語族|孟-高棉語族]][[佤-德昂語支|佤-德昂語支]]，有巴饒克、佤和阿佤3個方言。中國的佤族原來沒有文字，1956年創製了一套以巴饶方言为基础方言、以岩帅语音为标准音的[[拉丁字母|拉丁字母]]的拼音文字。

== 歷史 ==
佤族在2000年前就與[[布朗族|布朗族]]、[[德昂族|德昂族]]等民族的先民一起居住在[[高黎貢山|高黎貢山]]和[[瀾滄江|瀾滄江]]地區。[[先秦|先秦]]時「[[百濮|百濮]]」的一支。[[唐代|唐代]]稱「濮子蠻」，[[宋代|宋代]]稱「濮蠻」，[[明代|明代]]稱「古剌」，清代稱「嘎剌」、「哈瓦」，也被記載為「野『左犭右卡』」。佤族分為兩種，一是「卡瓦」(佧佤)，保有獵人頭的習俗，所以又稱為「野佧」；另一是「卡喇」(佧喇)，常與鄰族(尤其是傣族，舊稱「擺夷」)接觸，且已經革除獵人頭的習慣，又稱之為「熟佧」，他們受傣文化的影響而接受了南傳[[上座部佛教|上座部佛教]]信仰。<ref>{{Cite web | url=http://ethno.ihp.sinica.edu.tw/page/D-VA/D_main-2-VA1-2.htm | title=早期佤族族稱的歷史| website=村寨網}}</ref>「野佧」依獵頭方式又分為兩種：「濕頭」用殺人的方式取得人頭，「乾頭」則以購買或挖掘墳墓的方式取得人頭。<ref>{{cite book|author=Harold Mason Young|title=''Burma Headhunters''|url=https://books.google.com/books?id=hALbBQAAQBAJ&pg=PA14|date=2014-12-12|publisher=Xlibris|isbn=978-1-5035-1419-5|page=14}}{{en}}</ref>他們自稱「佤」、「巴饒克」、「阿佤」等，都有「住在山上的人」的意思。當地[[傣族|傣族]]與國外[[老族|老族]]、[[撣族|撣族]]稱其為「佧佤」，「佧」在傣語中意為奴隸。在[[中華人民共和國|中華人民共和國]]成立後，統一稱為佤族。佤族有[[猎首|猎头]]的习俗，最后一次有记录的猎头在1950年代末。

他們本来也生活在[[景棟|景棟]]，後被[[孟萊王|孟萊王]]孫兒征服，成為[[掸族|掸族]][[詔法|詔法]]的奴隶。

佤族民族學家趙明生提出了與傳統史籍相異的觀點，他認為晉朝[[永昌郡|永昌郡]]的永壽城不僅是普通縣城，而是晉朝時歸順中原政權的佤族地方民族政權，可能是民間傳說的佤族古城<ref>趙明生，永壽考一。永昌郡治永壽縣今地考</ref>。

== 佤族相关文藝作品 ==
* [[柏楊|柏楊]]於1961年出版的暢銷小說《[[異域|異域]]》講述[[泰緬孤軍|孤軍]]的故事，其中包括國軍退到佤族地區，以及反攻大陸時[[岩帅镇|岩帥]]王[[田興武|田興武]]、岩城王屈鴻齋率領大批佤族青年加入孤軍的事蹟。
* 《[[阿佤人民唱新歌|阿佤人民唱新歌]]》，佤族民歌改編的中國革命歌曲。此歌曲在改編前是佤族的祝酒歌，解放军通讯兵杨正仁在1964年改編後傳唱至今，也是[[雲南省|雲南省]][[西盟佤族自治縣|西盟佤族自治縣]]縣歌。
* 《[[孔雀飛來阿佤山|孔雀飛來阿佤山]]》，中國電影，1976年[[峨嵋電影製片廠|峨嵋電影製片廠]]攝製。電影拍攝地點為[[雲南省|雲南省]][[景洪市|景洪市]]，內容講述[[中國解放軍|中國解放軍]]在1950年代進入[[阿佤山|阿佤山]]佤族居住的山寨、趕走國軍部隊的故事。
* 佤族和众多西南地区少数族群一样不为绝大多数人所认知，值得一提的是在90年代初期，有一批佤族青年前往深圳表演佤族的歌舞，其中包括一种名叫甩发舞的舞蹈，引起了人们的兴趣。
