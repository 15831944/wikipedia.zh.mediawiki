'''椙山浩一'''（{{lang-ja|すぎやま こういち}}，本名{{lang|ja|椙山 浩一}}，{{bd|1931年|4月11日}} ）是日本音乐[[作曲家|作曲家]]，[[日本音樂著作權協會|日本音樂著作權協會]]理事成员，日本[[国家基本问题研究所|国家基本问题研究所]]董事会成员，日本[[国际双陆棋|国际双陆棋]]社团名誉主席。他以为[[史克威尔艾尼克斯|史克威尔艾尼克斯]]游戏[[勇者斗恶龙系列|勇者斗恶龙系列]]谱曲而知名，此外它还为《[[傳說巨神|傳說巨神]]》、《[[人造人009|人造人009]]》、《[[科學小飛俠|科學小飛俠]]》等日本[[动画|动画]]及[[电视节目|电视节目]]谱曲。

作为一名职业的[[古典音乐|古典音乐]]指挥者，人们认为他是[[植松伸夫|植松伸夫]]等其他游戏音乐家灵感的来源<ref>{{cite web |url=http://www.nobuouematsu.com/steffen.html |title=Nobuo Uematsu interview |author=Eric Steffens |date=February 1999 |deadurl=yes |archiveurl=https://www.webcitation.org/679rmugJL?url=http://www.nobuouematsu.com/steffen.html |archivedate=2012-04-24 }}</ref>，他还被人们成为“游戏音乐的大巨头”。此外，人们认为他是[[日本民族主义|日本民族主义]]者。

== 传记 ==

=== 早期活动与电视事业 ===

椙山浩一出生于[[日本|日本]][[东京|东京]]。在他成长时，家庭的音乐氛围激发了他的热情。在高中，他开始发现了自己的激情，并创作了各种音乐小作品<ref name="JC">{{cite web |url=http://sugimania.com/ |title=Koichi Sugiyama's Official Profile}}</ref> 。

在从1958年以全优成绩从[[东京大学|东京大学]]毕业后，他进入了[[株式會社文化放送|株式會社文化放送]]，先後待過文化放送的报导部与娱乐部<ref name="JC" />。他於1958年以音乐监制的身份加入了[[富士电视台|富士电视台]]。在1965年他离开了[[日本媒體|日本媒體]]行业并成为了自由监制人，在1968年他退出了监制界并集中精力于音乐谱写<ref name="JC" />。

在20世纪70至80年代，椙山为[[音乐剧|音乐剧]]、[[宣传片|宣传片]]、流行艺人、[[动画电影|动画电影]]和[[电视节目|电视节目]]作曲，如《[[科学小飞侠|科学小飞侠]]》、《[[天狼星的传说|天狼星的传说]]》<!-- [[:en:The_Sea_Prince_and_the_Fire_Child|:en:The Sea Prince and the Fire Child]] -->和《[[人造人009|人造人009]]》等。他还协助[[真锅理一郎|真锅理一郎]]谱写了《[[哥吉拉對黑多拉|哥吉拉對黑多拉]]》，包括谱写了唱片的一个声道以及指挥了一些声轨。

=== 勇者斗恶龙与电子游戏事业 ===

椙山最早和[[艾尼克斯|艾尼克斯]]接触是在1980年代早期，当时椙山作为爱好者写了一封关于PC[[将棋|将棋]]游戏的信。在收到椙山手写的明信片后，艾尼克斯工作人员对椙山深度的知识与游戏鉴赏能力留下了深刻印象，并决定邀请椙山为《勇者斗恶龙》作曲。

在刚到艾尼克斯工作时，椙山第一个作品是[[PC-8801|PC-8801]]的《[[Wingman_2|Wingman 2]]》。在1986年他为第二个游戏，[[FC游戏机|FC]]游戏《[[勇者斗恶龙_(游戏)|勇者斗恶龙]]》谱写了音乐。而[[勇者斗恶龙系列|勇者斗恶龙系列]]也成了他最知名的作品<ref>{{cite web | url=http://www.1up.com/news/dragon-quest-composer-reflects-24 | title=Dragon Quest Composer Reflects on 24 Years of Games: Kouichi Sugiyama on Japan's most recognized game music. |publisher=[[1up.com|1up]]|author=Gifford, Kevin |date=February 24, 2010 | accessdate=April 18, 2011}}</ref>。它为游戏的[[古典音乐|古典]]配乐被认为是[[电子游戏音乐|电子游戏音乐]]的革命<ref name="1up_dw">{{cite web|last=Gifford|first=Kevin|title=The Essential 50 Part 20 - Dragon Warrior|url=http://www.1up.com/features/essential-50-dragon-warrior|publisher=[[1UP.com|1UP.com]]|accessdate=15 May 2011}}</ref>。

椙山是第一个使用现场管弦乐队录制电子游戏音乐的作曲家。在1986年，由[[伦敦爱乐乐团|伦敦爱乐乐团]]演奏的CD《勇者斗恶龙I 交响组曲》发售。[[原声音乐|原声音乐]]的八首乐曲（序幕、城堡、城镇、旷野、迷宫、战斗、最终战、尾声）成为了大多[[電子角色扮演遊戲|角色扮演游戏]]的样板，此后上百的游戏都使用了类似的组织方式<ref>{{cite web|title=The "Eight Melodies" Template: How Sugiyama Shaped RPG Soundtracks|url=http://www.rpgfan.com/editorials/2008/11-29.html|work=RPGFan|accessdate=4 September 2011|author=Patrick Gann}}</ref>。

在1987年，他为《[[勇者斗恶龙II|勇者斗恶龙II]]》谱曲，之后他举办了世界上最早的电子游戏音乐会。该“家庭古典音乐会”的编排和指挥由椙山本人负责，并由东京弦乐联合演奏组于1987年8月20日在日本东京[[Suntory_Hall|Suntory Hall]]上演。音乐会上演奏了《勇者斗恶龙I 交响组曲》和《勇者斗恶龙II 交响组曲》<ref name="HC">{{cite web|url=http://sugimania.com/dq/index.html |archiveurl=https://web.archive.org/web/20060820065024/http://sugimania.com/dq/index.html |title=Koichi Sugiyama's Official Concert index |archivedate=2006-08-20 |deadurl=yes }}</ref>。“家庭古典音乐会”获得了很大的成功，此后椙山在日本各地举办了18次演出<ref>{{cite web |url=http://sugimania.com/family/index.html |title=Koichi Sugiyama's Official Family Classic Concerts Listing}}</ref>。

在1987年至1990年间，椙山继续为电子游戏谱曲。自1991年起，他发起了一系列的电子游戏音乐音乐会，五场音乐会合称为[[管弦乐游戏音乐演唱会|管弦乐游戏音乐演唱会]]，由东京市爱乐乐团和东京交响乐团演奏<ref>{{cite web|url=http://geocities.com/leergutdieb/bio.html |archiveurl=https://web.archive.org/web/20091027095515/http://geocities.com/leergutdieb/bio.html |title=Unofficial Koichi Sugiyama Biography |archivedate=2009-10-27 |deadurl=yes }}</ref>。其包括了十八位游戏作曲家的演出作品，如[[近藤浩治|近藤浩治]]、[[菅野洋子|菅野洋子]]、[[羽田健太郎|羽田健太郎]]、[[植松伸夫|植松伸夫]]、[[铃木庆一|铃木庆一]]以及椙山本人。这些音乐会在1991年至1996年间举办；在此期间椙山为其他的游戏进行了谱曲，并为管弦乐游戏音乐演唱会的演出而改编了一些作品。

在1995年9月，椙山创作了勇者斗恶龙[[芭蕾舞|芭蕾舞]]。其于1996年首次公演，并于1997年、1999年、2001年和2002年再次演出<ref name="JC" />。在这几年中，他依然继续发行勇者斗恶龙交响组曲，并一直工作至今。

=== 之后的职业生涯 ===

椙山还完成了其他的作品，如东京赛马场和中山赛马场中，门开启与关闭的声号。

椙山非工作性质的爱好有[[摄影|摄影]]、[[旅行|旅行]]、建造模型船<ref>{{cite web |url=http://rhapsody_1st.1up.com/ |title=Gaming's Rhapsody: First Movement |publisher=[[1UP.com|1UP.com]] |author=Nich Maragos |date=20 July 2005 |deadurl=yes |archiveurl=https://web.archive.org/web/20060329235341/http://rhapsody_1st.1up.com/ |archivedate=2006年3月29日 }}</ref>、收集旧[[相机|相机]]及[[阅读|阅读]]。他在自己的网站上开放了一个相机模块<ref>{{cite web |url=http://sugimania.com/tuhan.html |title=Koichi Sugiyama's Official camera page}}</ref>，他还于2004年6月23日建立了自己的唱片公司“SUGI Label”<ref>{{cite web |url=http://sugimania.com/sugi.html |title=Koichi Sugiyama's Official SUGI Label page |deadurl=yes |archiveurl=https://web.archive.org/web/20060820070544/http://sugimania.com/sugi.html |archivedate=2006-08-20 }}</ref>。

在2004年末，他完成了《[[勇者斗恶龙VIII_空海陆与被诅咒的公主|勇者斗恶龙VIII 空海陆与被诅咒的公主]]》的原声音乐以及《勇者斗恶龙VIII 交响组曲》。

在2005年，椙山和[[东京都交响乐团|东京都交响乐团]]在日本举办了一系列的音乐会，其中演奏了《勇者斗恶龙VIII》的音乐，其依然延续了过去的古典风格<ref>{{cite web |url=http://sugimania.com/concert.html |title=Koichi Sugiyama's Official Concert announcement page}}</ref>。其后2005年8月的欧洲[[交响游戏音乐会|交响游戏音乐会]]上，则现场演奏了他的勇者斗恶龙音乐，这也是第一次在日本以外的现场交响音乐会上演奏他的音乐<ref>{{cite web|url=http://www.vgmconcerts.com/ |title=Symphonic Game Music Concert Official website}}</ref> 。

在2006年，椙山开始参与一些不同的工作，其中之一就是为《[[勇者斗恶龙之剑：假面女王与镜之塔|勇者斗恶龙之剑：假面女王与镜之塔]]》谱曲<ref>{{cite web |url=http://www.square-enix.com/na/company/press/2006/0508_2/ |title=Square-enix North America announcement |date=2006}}</ref>。在2006年8月19日，椙山在日本电子游戏杂志《[[Fami通|Fami通]]》上公布了《勇者斗恶龙IV》的作品，并称“我不确定《勇者斗恶龙IX》何时会发布，但我看见它在一步步地完成。我个人很激动。”<ref name="famitsu"> {{cite web | url = http://www.famitsu.com/game/news/2006/08/11/103,1155293596,58611,0,0.html | title = ファンの声を反映したプログラムで、今年もドラゴンクエストコンサートが盛大に幕開け | publisher = Famitsu | date = 2006-08-11 |language=en}}</ref>

== 风格 ==

椙山的作品主题始终重复自我以保持自己的一致性，在不同的作品中都保持怀旧品质。这点在勇者斗恶龙系列上尤为凸显，每部游戏都包含几乎相同，名为“序曲”的欢快主题曲。此外，《勇者斗恶龙III》至《IX》的游戏保存界面都使用了名为“间奏曲”的简单轻松的曲调。

人们把椙山的作曲风格和后[[巴洛克|巴洛克]]与早[[古典主义音乐|古典]]风格做了对比。[[约翰·塞巴斯蒂安·巴赫|约翰·塞巴斯蒂安·巴赫]]、[[古斯塔夫·马勒|古斯塔夫·马勒]]和[[格奥尔格·弗里德里希·亨德尔|格奥尔格·弗里德里希·亨德尔]]等音乐家给了他一些启示，此外还有20世纪中叶美国电影的旋律风格。另外从他的一些试验性作品中可以听到[[阿诺德·勋伯格|阿诺德·勋伯格]]的影响，其中有《勇者斗恶龙IV》。

== 政治活动 ==

椙山以其政治活动而闻名。1970年代，他发起反对日本不同县之间选民的不平等。2010年，他與[[三橋貴明|三橋貴明]]及[[西村幸祐|西村幸祐]]建立了网站《日本媒体監督》（Media Patrol Japan，簡稱MJP），以「[[日本人|日本人]]要有日本人的媒體」（{{lang|ja|日本人による日本人のためのメディア}}）為由「监督」[[日本媒体|日本媒体]]，並為該站撰寫評論文章。他亦以反对[[美国众议院第121号简单决议案|美国众议院第121号简单决议案]]而知名，其指称现有围绕[[南京大屠杀|南京大屠杀]]和[[慰安妇|慰安妇]]的事实是有选择性的。他與同伴合組的[[歷史事實委員會|歷史事實委員會]]第一次尝试以《事实》（THE FACTS）为名刊登他們的反对观点时，[[华盛顿邮报|华盛顿邮报]]和[[纽约时报|纽约时报]]拒绝了他的请求，而最后华盛顿邮报同意了刊登<ref>{{cite web |url=http://www.jiaponline.org/documents/Jun14AdALLSignatoriesLIST.pdf |title=Signatories to the June 14th Washington Post "The Facts" Advertisement - Politicians, Professors, and Journalists |date=2007-07-25}}</ref><ref>{{cite web |url=http://www.nikkeibp.co.jp/sj/2/column/y/64/index.html |title=ワシントン・ポスト紙に「慰安婦意見広告」― その経緯と波紋 / SAFETY JAPAN [花岡 信昭氏] / 日経BP社 |deadurl=yes |archiveurl=https://web.archive.org/web/20110719012218/http://www.nikkeibp.co.jp/sj/2/column/y/64/index.html |archivedate=2011年7月19日 |df= }}</ref><ref>{{cite web |url=http://www.gamasutra.com/php-bin/news_index.php?story=24627 |title=The Complex Question |publisher=[[Gamasutra|Gamasutra]]}}</ref>。

椙山是由[[新歷史教科書編撰會|新歷史教科書編撰會]]（{{lang|ja|新しい歴史教科書をつくる会}}）分裂出來的[[教科書改善會|教科書改善會]]（{{lang|ja|教科書改善の会}}）的成員，也是[[国家基本问题研究所|国家基本问题研究所]]評議員。椙山曾為日本保守派政治人物如[[松原仁|松原仁]]、[[稻田朋美|稻田朋美]]、[[城內實|城內實]]創作[[選舉|選舉]]歌曲。2010年5月，[[藤井嚴喜|藤井嚴喜]]與西村幸祐告發[[日本首相|日本首相]][[鳩山由紀夫|鳩山由紀夫]]涉嫌違反《公職選舉法》，椙山是連署人之一。

椙山是[[日本音樂著作權協會|日本音樂著作權協會]]（JASRAC）評議員，反對[[海賊版|海賊版]]，還在自己的網站放上標語「人有[[人權|人權]]，音樂有[[著作權|著作權]]」（{{lang|ja|人には人権・音楽には著作権}}）。[[Copy_Control|Copy Control]] [[CD|CD]]（CCCD）剛上市的時候，獲得椙山的支持；但在[[Sony_BMG|Sony BMG]]爆發[[Sony_BMG_CD防拷醜聞|CD防拷醜聞]]後，椙山表示「希望很快就能看到能取代CCCD的新技術登場」（{{lang|ja|一刻も早くCCCDにかわる新技術の登場が待たれる}}）。
