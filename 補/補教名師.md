-{zh-hans:'''补教名师'''; zh-hant:'''補教狼師'''; zh-hk:'''補習天王'''，在台灣稱為補教名師; zh-tw:'''補教名師'''，在香港稱為補習天王;}-，或稱為'''補習天皇'''、'''補習天后'''、'''天王名師'''等，是[[香港|香港]]、[[台灣|台灣]]等[[中文|中文]]使用[[地區|地區]]的常見名詞，指具有高知名度、評價與高收入的[[補習班|補習班]]教師，網路風潮後，因應數位內容的崛起與各國家少子化的威脅，眾多補教名師紛紛投靠不同的雲端平台，開始在雲端進行授課<ref>[http://punnode.com/archives/21930 補習搬上雲端 學生在家拼升名校]</ref>藉以維持收入。

== 日本 ==
在[[日本電視劇|電視劇]]《[[女王的教室|女王的教室]]》裡，班裡的學生有不少在放學後亦要參加補習班。但令人出奇的是：節目內有補習學校的負責人聲稱，有不少從再培訓機構出來的老師都選擇教補習班，而非主流學校。

== 香港 ==
 {{seealso|香港補習文化 |補習班 |應試教育}}  

事源80年代末[[香港樂壇|香港樂壇]]上出現[[香港四大天王|四大天王]]，及至90年代，[[香港傳媒|香港傳媒]]將各行各業收入豐厚的人稱為天王，「補習天王」一詞源於[[香港傳媒|香港傳媒]]於90年代對連鎖補習社創辦人及導師[[史偉全|史偉全]]的報導。

香港的補習天王主要教授中四至中六的[[香港中學文憑考試|香港中學文憑考試]]科目為主。

補習名師普遍以[[考評局|考評局]][[歷屆試題|歷屆試題]]({{lang-en|past paper}}) 和評分參考({{lang-en|marking scheme}})作為教材及標準答案，分析出題走勢和佔分比重，並根據公開考試年報({{lang-en|annual report}}) 中的考生表現不滿意的題目或失分原因加以針對式應付，冠以巧立名目的字眼如「獨門技巧」、「拆題思維」等宣傳，加上風趣幽默的口訣，傳授給補習學生(顧客)。 

補習天王更會預測公開試的試題（俗稱「貼題／Tip題」），讓學生可選擇性重點溫習，若果「貼中」了，便會大肆宣傳自己，以表示自己貼題能力高。這類補習課程頗受中學生歡迎。坊間更充斥不少補習天王天后，而不少名補習老師大灑金錢，到處刊登廣告，如巴士車身廣告，大廈外牆廣告橫額等，更會標榜在公開考試中取得最高成績A級、5**級的學生人數。

部份頂級名師年薪超過1000萬港元，比香港[[政務司司長|政務司司長]]年薪高一倍<ref>[[無綫電視|無綫電視]]晚間新聞 2009-7-15</ref><ref>[http://www.epochtimes.com/b5/6/5/3/n1306447.htm 香港補習天王年薪超過一千萬港幣]</ref><ref>[http://news.sina.com.hk/cgi-bin/nw/show.cgi/15/1/1/681843/1.html 年薪過百萬 住獨立別墅駕名車]</ref>。但其實際收入是扣除了廣告宣傳費用的。而據[[遵理學校|遵理學校]]導師[[伍經衡|伍經衡]]指出，凡加入補習界的人一定要[[大學|大學]]畢業，上課內容要具趣味性以及能充分表達自己。除了以上基本條件外，他們亦要具一定的名氣。當要加入補習界時，他們要自薦，並言明個人的賣點，或者經由內部挖角形式加入。

== 台灣 ==
一般來說，台灣較出名的補教名師是以教授國高中課程為主。

== 中國大陸 ==
中國大陸的補習班有[[考研|考研]]輔導和出國考試輔導等類別。
