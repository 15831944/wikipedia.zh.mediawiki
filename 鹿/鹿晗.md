'''鹿晗'''（{{lang-en|'''Lu Han'''}}，{{bd|1990年|4月20日|||catIdx=L鹿}}）出生于[[中華人民共和国|中華人民共和国]][[北京市|北京市]][[海淀區|海淀區]]，[[中國|中國]]大陸男[[藝人|藝人]]，曾為[[韓國|韓國]]男子組合[[EXO|EXO]]及中國分隊[[EXO-M|EXO-M]]成員，隊內擔任主唱、領舞、中心擔當<ref name="EXO-M成员成长史遭起底 鹿晗kris张艺兴中枪">{{cite web | url=http://ent.163.com/13/0430/10/8TN13HJE00032DGD.html| title=EXO-M成员成长史遭起底 鹿晗kris张艺兴中枪|publisher=网易|}}</ref>。而後，與SM公司達成協議，以個人身份回國發展，並在2015年起加盟綜藝節目《[[奔跑吧兄弟|奔跑吧兄弟]]》第三季至第六季，以及在2016年參加真人秀節目《我們去上學啦2》。

== 演藝经历 ==
===早年生活===
1990年4月20日，鹿晗於[[中華人民共和國|中華人民共和國]][[北京市|北京市]][[海淀區|海淀區]]出生。小学就读北京市育英学校（1996年），初中就读北京市师达中学（2002年），高中就读北京海淀外国语实验学校（2005年）。鹿晗小时候爱玩乒乓球，有一次没抢到球桌，意外爱上了踢足球，并成为校队主力。2008年，赴韩国留学，在延世大学韩国语学堂学习韩语，并通过了韩语等级考试。2009年，考入首尔艺术大学实用音乐艺术系。留学期间在明洞逛街时被SM公司星探发掘。2010年，鹿晗通过S.M. Casting System成为旗下练习生。

=== 組合期間 ===
* 2011年12月27日，個人出道預告照、舞蹈視頻《TIME CONTROL》公佈  。 12月29日，亮相SBS歌謠大戰特別舞台《SMTown Orchestra》 。 2012年1月12日，鹿晗主唱歌曲《你的世界》公佈 。  1月30日，鹿晗主唱歌曲《What Is Love》公佈 。 4月8日以EXO組合主唱、領舞、門面擔當身份在第十二屆“音樂風云榜頒獎典禮｣正式出道 。
* 2013年7月－9月，擔任[[EXO|EXO]]正規一輯《XOXO(Kiss&Hug)》主打歌《狼與美女》及後續主打《咆哮(Growl)》劇情版MV男主角。 7月17日，個人微博[@M鹿M]被曝，122萬條微博評論記錄引發國內媒體關注，新華網統計調查2013大事件，鹿晗個人微博7月以百萬評論記錄列入熱門事件內 。
* 2014年4月20日，個人微博420萬評論創新記錄，隊內個人影響力及人氣效應持續上升 。 6月出演導演陳正道執導的中韓合拍電影《重返20歲》。 7月20日，個人微博單條評論破千萬創微博新記錄。 8月2日，鹿晗個人單條微博達1314萬，七夕1314萬封微博情書再創微博評論新紀錄 。 8月19日，金氏世界紀錄認證鹿晗單條微博13,162,859條評論創吉尼斯金氏世界紀錄，成為中國中文社交媒體的第一個對外公佈的[[金氏世界紀錄|金氏世界紀錄]]。 10月10日，鹿晗由代理律師向韓國首爾地方法院遞交起訴書，申請與[[SM娛樂|SM娛樂]]合同無效。

===個人時期===
==== 2014年 ====
* 2014年11月2日，解約後的鹿晗首次亮相，出席《[[重返20歲|重返20歲]]》在北京舉辦的定檔發布會。<ref>{{cite web|url=http://www.qlwb.com.cn/2014/1103/241137.shtml|title=《重返20岁》鹿晗解约后首次公开现身 影片改编自《奇怪的她》|website=齐鲁晚报}}</ref>

==== 2015年 ====
{{无来源章节}}
* 2015年1月8日，鹿晗出演的首部電影《[[重返20歲|重返20歲]]》正式上映。
* 2015年3月，出演懸疑電影《[[我是證人|我是證人]]》擔任男主角。 同月12日，加盟張藝謀執導的好萊塢史詩電影《長城》。
* 2015年5月，獻唱北京申辦[[2022年冬奧會|2022年冬奧會]]優秀音樂作品《請到長城來滑雪》。同月登上《福布斯》中文版封面。
* 2015年7月13日，首位在新浪微博上個人詞條瀏覽量突破300億的藝人，同月17日首次做客微訪談，一小時113萬提問量打破中国古劍奇譚劇組保持的記錄。
* 2015年9月14日，鹿晗首張個人數字專輯《Reloaded I》（重啟I）正式发售。
* 2015年9月25日，鹿晗首唱會在北京舉行。
* 2015年12月10日，[[迪士尼|迪士尼]]影業宣布鹿晗成為《[[STAR_WARS：原力覺醒|STAR WARS：原力覺醒]]》的中國宣傳大使。
* 2015年12月14日，首張實體專輯《Reloaded（重啟）》销售。

==== 2016年 ====
* 2016年1月6日，鹿晗首張實體專輯在日本發行的輸入盤一周銷量進入[[oricon|oricon]]日本公信榜周榜第30位。 1月7日，鹿晗的蠟像正式入駐北京[[杜莎夫人蠟像館|杜莎夫人蠟像館]]。 1月8日，首張個人專輯《Reloaded》發片不足十日即獲台灣五大金榜2015年終總榜第五名，成為唯一一位入榜的中國歌手。 1月15日，《Reloaded》豪華版專輯在音悅台獨家限量預售1萬套，上線89秒清空庫存，打破了此前鹿晗自己創下的平台實體銷售紀錄。
* 2016年2月14日，第三張個人迷你數字專輯《Reloaded+》預售上線後九分鐘售出10萬張，再次刷新國內唱片銷售紀錄。 2月18日，鹿晗演唱會門票開啟預訂後32秒即被搶空。
* 2016年3月18日，鹿晗成為《Vogue Me》創刊號封面人物，雜誌預售六分鐘突破三萬本銷量。
* 2016年4月10日，首張個人專輯《Reloaded（重啟）》成為中國大陸首張獲得國際唱片業協會IFPI認證的白金唱片。
* 2016年4月19日，首次以设计师身份与izzue合作推出izzue x LuHan 2016春夏联名系列。
* 2016年6月9日19：52分， 首张个人专辑《Reloaded I》夺得QQ音乐数字专辑首个殿堂金钻唱片。
*2016年7月21日，與 [[SM娛樂|SM娛樂]]長達約兩年的合約糾紛在日前正式結束，隨著法院下達和解建議的決定，合約將有效維持至2022年，將自行管理韓國與日本以外地區的活動。<ref>{{cite news|title=KRIS、鹿晗與 S.M. 協議維持合約關係|url=http://kpopn.niusnews.com/=K0v4vm36 |accessdate=2016-07-22|publisher=kpopn.com|language=zh}}</ref>。
*2016年12月23日，飾演電影《[[擺渡人_(2016年電影)|擺渡人]]》中年輕時的馬力。

==== 2017年 ====
* 2017年1月27日，登上央視雞年春晚，與[[陳偉霆|陳偉霆]]演唱《愛你一萬年》。
* 2017年2月6日，清流資本聯合新生代現象及鹿晗、新希望集團宣布，共同成立清晗基金。
* 2017年2月21日，迷你數字專輯《Venture》發布，至此，鹿晗已發布的數字專輯總銷量突破1000萬。
* 2017年4月17日，第一部主演的電視劇《[[擇天記_(電視劇)|擇天記 (電視劇)]]》開播，飾演男主角陳長生。
* 2017年4月19日，迷你數字專輯《Imagination》發布。
* 2017年5月4日，登上央視“五月的鮮花”舞台，演唱《勳章》(歌詞部分更改)，並且獲得共青團中宣頒發的“五四優秀青年”獎項
* 2017年6月27日，XXVII系列之終篇《I》正式發布。XXVII代表即是27歲在「經歷」、「探索」、「冒險」、「想象」後，是一個發現「自我」的一個年紀。也代表通過這一系列專輯，能一起完成關於我們自己的、自我的成長之旅。《I》預售10分鐘銷量突破31萬張，三天銷量即突破100萬張。
