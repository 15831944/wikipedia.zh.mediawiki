'''鹿麻產車站'''位於[[台灣|台灣]][[嘉義縣|嘉義縣]][[竹崎鄉|竹崎鄉]]，為[[行政院農業委員會林務局|林務局]][[阿里山森林鐵路|阿里山森林鐵路]][[阿里山線|阿里山線]]之[[鐵路車站|鐵路車站]]。

== 車站構造 ==
木造。
== 利用狀況 ==
於2019年4月16日起恢復定期客運，所有上、下山列車皆停靠。

== 車站週邊 ==
車站周遭約二公頃的土地結合地方文化，設有地方文物館，展示鹿滿社區村民收藏的古老用具，及早期焙菸窯，並規劃出鐵道文物展示館，未來將搭配自行車專用車道與竹崎親水公園做一條觀光新動線。

== 歷史 ==
*1910年10月1日：設站。
*1982年11月16日：本站廢止。
*2004年12月：木造站房才依舊觀重新整建，恢復昔日風貌。
*2007年7月15日：例假日行駛的檜木列車恢復固定停靠本站。
*2008年3月：檜木列車不再固定停靠本站。
*2019年4月16日：恢復定期客運業務。
