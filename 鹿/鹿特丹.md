'''鹿特丹'''（{{lang-nl|Rotterdam}} {{IPA-nl|ˌrɔtərˈdɑm|lang|Nl-Rotterdam.ogg}}）是[[荷蘭|荷蘭]][[南荷蘭省|南荷蘭省]]的城市，是[[荷兰|荷兰]]第二大城市，位于荷兰的[[南荷蘭省|南荷蘭省]]，[[新马斯河|新马斯河]]畔。其名稱来自于在市中心注入新马斯河的小河[[鹿特河|鹿特河]]、以及荷蘭語的「壩」（Dam）。

鹿特丹长期为[[欧洲|欧洲]]最大的[[海港|海港]]，以[[集装箱|集装箱]]运量计算，1980年代曾是世界上第一大港口，在2010年为世界第10。

== 历史 ==
[[File:HerdenkingVuurgrensRotterdam1940_2007_edit1.jpg|thumb]]
至少公元900年左右，在汇入沼泽的鹿特河（Rotta，由代表“泥泞”的Rot和代表“水域”的词缀a组成，意为泥泞的水域）的下游便有人居住了。1150年前后，为了抵御洪水，当地居民开始修建堤坝。1260年左右，人们在在鹿特河上建成了一座大坝，是为Dam on the Rotte或Rotterdam；此即为鹿特丹名字的起源。

1340年，鹿特丹被授予城市自治权，当时居民约有2000人。不久，一条连接鹿特丹和荷兰北方市镇的运河开通，大大促进了鹿特丹航运事业的发展，使之渐渐成为了荷兰、[[英格兰|英格兰]]和[[德国|德国]]之间重要的货运中转中心。此后，鹿特丹作为货运港的地位不断增强，并成为了[[荷兰东印度公司|荷兰东印度公司]]六处办公室的所在地之一。19世纪，在人口的急剧增加的同时，鹿特丹的海运业达到全盛，成为欧洲最重要的货运港口之一。19世纪末全欧洲最高的建筑就在鹿特丹。

1940年5月10日，德国入侵荷兰；为了迫使荷兰投降，[[德意志國防軍|德军]]对鹿特丹进行了狂轰滥炸并威胁轰炸其它城市。鹿特丹市中心的建筑物几乎全部被毁，800多人遇难。[[奥西·撒丁|奥西·撒丁]]后来为被毁后的鹿特丹设计了一座叫做“没有心的城市”（Stad zonder hart）的塑像，这座塑像今天位于城市的北部。

从1950年代到1970年代鹿特丹被重建，但整个城市依然相当空旷。1980年代中市政府开始主动规划城市建设，在市中心建立了一个以新式建筑为主的新的天幕。1990年代里城市在河南岸建立了一个新的商业中心。

2011年[[蘭斯台德|蘭斯台德]]地區有合併建省的計畫，位於南荷蘭省的蘭斯台德地區也有意加入此新的省份建置<ref>http://www.nrc.nl/nieuws/2011/02/04/randstadprovincies-onderzoeken-fusie/</ref>。

== 地理 ==
[[File:Satellite_image_of_Europoort,_Netherlands_(4.25E_51.90N).png|thumb]]
[[File:Rotterdam-plaats-OpenTopo.jpg|thumb]]
鹿特丹被[[新馬斯河|新馬斯河]]一分為二，南北兩側的連接包括（由西到東依序為）：荷比盧隧道（Beneluxtunnel），馬斯隧道（Maastunnel），[[伊拉斯謨橋|伊拉斯謨橋]]，一條地鐵隧道，威廉鐵道隧道（Willemsspoortunnel），[[威廉姆斯大橋|威廉姆斯大橋]]，皇后橋（Koninginnebrug），以及凡布里涅諾德橋（Van Brienenoordbrug）。鹿特丹市中心位於新馬斯河的北岸，後隨著都市擴張，市中心區域漸漸擴展到鹿特丹南岸的北側（De Kop van Zuid）。

=== 氣候 ===
鹿特丹的氣候屬於溫帶[[海洋性氣候|海洋性氣候]]（[[柯本氣候分類法|柯本氣候分類法]]：Cfb），是典型的荷蘭海岸城市，相對內陸地區較為濕潤。鹿特丹夏季氣溫時常突破30 ºC，冬季也常下探-5 ºC。下方表格是鹿特丹海牙機場所測得的氣候資訊。因為[[熱島效應|熱島效應]]的關係，鹿特丹市區的溫度普遍略高於機場測候站。

== 人口 ==
由于55%的市民属于低收入，鹿特丹市具有典型的城市问题。鹿特丹与其他荷兰的城市一样，原本是以基督徒为主的城市，但是近些年以来，世俗化进程加剧，鹿特丹原本的基督教人口不断减少。同时，由于北非穆斯林移民生育率超高，伊斯兰教在鹿特丹得到快速发展，穆斯林人口在鹿特丹占有很大比例，而且仍在继续上升。

鹿特丹居民約半數為無特定宗教信仰，其餘有信仰者以基督教（含天主教、新教及其他教派，36.3%）、伊斯蘭教（13.1%）和印度教（3.3%）信徒為大宗。

鹿特丹自1795年開始主持[[抗辯派|抗辯派]]自由派新教徒兄弟會的總會。1955年，[[天主教鹿特丹教區|天主教鹿特丹教區]]自[[天主教哈勒姆-阿姆斯特丹教區|哈勒姆教區]]獨立後，便一直是鹿特丹主教的主教教區。2010年，[[埃薩拉清真寺|埃薩拉清真寺]]（Essalammoskee）竣工，可容納1,500人，是荷蘭最大的清真寺。

== 行政區劃 ==
按1999年1月1日的统计数据鹿特丹304.22平方公里（其中陆地面积206.44平方公里），总人口598,923人（2004年1月1日）。

除市中心外鹿特丹还包括以下城镇：夏洛瓦（Charlois）、德夫哈芬（Delfshaven）、费耶诺德（Feijenoord）、Hillegersberg-Schiebroek、Hoek van Holland、Hoogvliet、IJsselmonde、Kralingen-Crooswijk、Noord、Overschie、佩尔尼斯（Pernis）、Prins Alexander以及以下这些工业和港口地区：博特莱克（Botlek）、爱姆哈文港区（Eemhaven）、欧洲港（Europoort）、马斯弗拉克特（Maasvlakte）、Spaanse Polder、Vondelingenplaat、威尔哈文（Waalhaven）。

== 經濟 ==
=== 港口 ===
[[File:9-028_Rotterdam_ECT.jpg|thumb]]

鹿特丹是欧洲最大的港口，它是远洋货物进出欧洲的大门。1980年代曾为世界最大的集装箱港口，但是1990年代起，随着亚洲港口的兴起，其排名不断下滑，至2010年为世界第10。

鹿特丹港最重要的部分是[[石油工业|石油工业]]和一般的[[货物|货物]]运输。它是欧洲向和从世界其它地区运输大批量货物的主要港口。从鹿特丹货物可以装入海轮、河轮、火车和汽车。从2000年开始一条连接鹿特丹与[[德国|德国]]的高速铁路开建。市区西部有很大的[[炼油厂|炼油厂]]。通过[[马斯河|马斯河]]和[[莱茵河|莱茵河]]鹿特丹与内陆的连接非常好。

在20世纪的前半叶中鹿特丹港口的活动不断向[[北海_(大西洋)|北海]]推近。从鹿特丹到北海挖了一条运河来避開比较浅的莱茵河。这条运河是1872年建成的，此后许多工业在这条运河的两岸立足。

通过在运河口建立的欧洲港和在北海内建立的马斯弗拉克特鹿特丹的港口被扩大很多。1990年代中关于建立一个第二个马斯弗拉克特的问题引起了很大的政治争议。这个扩大从2004年夏开始动工，部分费用由政府支付。

== 教育 ==
鹿特丹有一所大学（Universiteit），即[[鹿特丹伊拉斯谟大学|鹿特丹伊拉斯谟大学]]（Erasmus Universiteit Rotterdam），经常被习惯性称为[[鹿特丹大学|鹿特丹大学]]。该校是以历史上一位著名的“鹿特丹人”、荷兰十五世纪的人文主义哲学家[[伊拉斯谟|伊拉斯谟]]命名的。鹿特丹大学在世界大学各类排行榜上一直名列前茅。2007年，鹿特丹大学被《金融时报》（Financial Times）评为欧洲前十的顶尖大学。2014年，鹿特丹大学被泰晤士高等教育世界大学综合排名列为欧洲第20位，世界第73位，其中社会科学为世界第47位，临床与健康为世界第37位。<br />
鹿特丹大学设有鹿特丹管理学院 、伊拉斯姆斯经济学院、伊拉斯姆斯法学院、医学与健康科学学院、社会科学学院、国际社会科学研究所等院所。管理学院、经济学院和医学院在荷兰处于第一的位置。尤其是鹿特丹管理学院是荷兰商学院最高学府，在荷兰被称为欧洲的哈佛，处于绝对第一的位置，位列欧洲研究三强之首，在国际商学院顶级杂志发表论文数量长期欧洲第一。<br />
该校分成位于城市西部的Woudestein和城市东部的Hoboken两个校区。在全欧洲和全世界都享有声誉的鹿特丹管理学院位于Woudestein校区；医学部则位于Hoboken校区，同在一个校区还有两所附属医院，它们和鹿特丹大学的医学部共同组成“伊拉斯谟医学中心”（Erasmus MC）。<br />
鹿特丹大学还有全欧洲最大的学生联合会：STAR Study Association RSM Erasmus University。

== 文化 ==
[[File:Rotterdam_standbeeld_Erasmus.jpg|thumb]]像，位于鹿特丹]]
2001年鹿特丹被选为[[欧洲文化之都|欧洲文化之都]]。城市有它自己的[[鹿特丹交响乐团|交响乐团]]、一个大会议堂和多个剧院。市南有一个可以召开流行音乐会、展览、网球比赛等活动的中心。市北有一个[[动物园|动物园]]。市内还有一个海洋水族馆。[[約克布斯·約翰尼斯·彼得·歐得|約克布斯·約翰尼斯·彼得·歐得]]是鹿特丹的著名建筑师。

鹿特丹有一个艺术学院。

鹿特丹近年来获得了一些新生，市内的建筑项目为城市带来了新的夜生活。夏季市内举办多个多文化的艺术节。此外市内一月举办国际电影节，六月诗歌节等等。

鹿特丹的自我认识是它是一个不华浮的工人城市。它经常与被看作是荷兰的艺术首都的[[阿姆斯特丹|阿姆斯特丹]]竞争。

=== 博物馆 ===
鹿特丹有许多[[博物馆|博物馆]]。著名的博物馆有[[博伊曼斯·范伯宁恩美术馆|博伊曼斯·范伯宁恩美术馆]]，新研究所(The New Institute,由原荷兰建筑研究所NAI等机构合并而成)，[[沙博博物馆|沙博博物馆]]，[[鹿特丹历史博物馆|鹿特丹历史博物馆]]（Historisch Museum），民俗博物馆（Volkenkundig Museum），[[艺术厅_(鹿特丹)|艺术厅 (鹿特丹)]]（由[[雷姆·庫哈斯|雷姆·庫哈斯]]设计），当代艺术中心——[[Witte_Corneliszoon_de_With|Witte de With]],<ref>{{cite web |title=Witte de With museum |url=http://www.wdw.nl |accessdate=2008-05-15}}</ref> [[鹿特丹海事博物馆|鹿特丹海事博物馆]]<ref>{{cite web |title=Maritiem Museum official site |url=http://www.maritiemmuseum.nl |accessdate=2008-05-15}}</ref> 以及消防队博物馆（''Brandweermuseum''）。

其他博物馆包括税务博物馆（tax museum），[[鹿特丹自然史博物馆|鹿特丹自然史博物馆]]，历史博物馆——[[Dubbelde_Palmboom|Dubbelde Palmboom]]以及Schielandhuis。在历史上的造船厂和博物馆Scheepswerf 'De Delft，重建的[[战列舰|战列舰]]'De Delft'可供参观。<ref>{{cite web |title=Scheepswerf 'De Delft' official site |url=http://www.dedelft.com |accessdate=2008-05-15}}</ref>

== 建筑 ==
鹿特丹有荷兰最高的建筑，高160米。这里也有许多世界知名的建筑设计师和他们的公司和有名的建筑学院。连接南北鹿特丹的[[伊拉斯谟桥|伊拉斯谟桥]]是一座单塔[[斜拉桥|斜拉桥]]，全长802米。

[[File:Rotterdam_Cube_House.jpg|thumb]]），位于鹿特丹市中心]]

1898年，45米高的高层办公楼[[白房子|白房子]]（{{lang-nl|Witte Huis}}）完工，成为当时欧洲最高的办公楼。

20世纪，一些现代风格建筑兴建于鹿特丹。著名的有（Van Nelle fabriek，1929年）—现代工厂的纪念碑，由（Brinkman en Van der Vlugt）设计；[[皇家马斯游艇俱乐部|皇家马斯游艇俱乐部]]（Royal Maas Yacht Club）的[[新艺术运动|Jugendstil]]会所（Jugendstil clubhouse），由Hooijkaas jr. en Brinkman设计（1909年）；[[飛燕諾|飛燕諾]]的足球场——[[费耶诺德球場|de Kuip]]（1936年），同样由（Brinkman en Van der Vlugt）设计。建筑师（Jacobus Oud|J. J. P. Oud）那时是一位著名的鹿特丹人。在[[第二次世界大战|第二次世界大战]]初期，德国人轰炸鹿特丹市中心，破坏了许多城市中心的老建筑物。在起初决定性的重建后，鹿特丹中心已经成为雄心勃勃的新建筑荟萃之地。

鹿特丹也因其Kubuswoningen或称[[方房子|方房子]]而闻名，这是由建筑师[[皮特·布洛姆|皮特·布洛姆]]（Piet Blom）于1984年建造的。此外还有许多国际知名的建筑师事务所以鹿特丹为基地，仅略举数例如下：O.M.A（[[雷姆·库哈斯|雷姆·库哈斯]]）, [[MVRDV|MVRDV]], [[Neutelings_Riedijk_Architects|Neutelings & Riedijk]]以及[[Erick_van_Egeraat|Erick van Egeraat]]。

[[File:erasmusbrug.jpg|thumb]]]]
[[File:Euromast2.jpg|thumb]]]]

鹿特丹具有若干荷兰最高的建筑。
* [[伊拉斯谟桥|伊拉斯谟桥]]（Erasmusbrug，1996年）是一座全长802米的斜拉桥，连接鹿特丹北部和南部。它由13米带有一特有弯曲的桥塔（pylon）支撑，后者令该桥获得了其昵称“天鹅桥”（'De Zwaan'，即'the Swan')。
* 荷兰最高的住宅Montevideo Tower（160米）。
* 荷兰最高的办公楼Delftse Poort（160米），该楼是[[ING集团|ING集团]]下属荷兰国家保险公司（Nationale-Nederlanden insurance company）的所在地。<ref>{{cite web|title= ING building brief|url= http://inghouse.ing.com/intra/eng/buildings/delftsepoort.html|accessdate= 2008-05-15|deadurl= yes|archiveurl= https://web.archive.org/web/20050308113741/http://inghouse.ing.com/intra/eng/buildings/delftsepoort.html|archivedate= 2005-03-08}}</ref><ref>{{cite web|title= Sky Scraper City ING site|url= http://www.skyscrapercity.info/100.php?id=4&bid=1|accessdate= 2008-05-15}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
* [[欧洲之桅|欧洲之桅]]（Euromast），建于1960年，起初高101米；1970年代，它延长85米至现今。

通过新研究所（原[[荷兰建筑学研究所|荷兰建筑学研究所]]（Netherlands Architecture Institute））对公众的开放以及多种多样的有关建筑学和城市规划问题的优秀展览，鹿特丹具有良好的建筑学发展及教育声誉。

鹿特丹位列最好的欧洲天际线Top（European SkylineTop）中，与[[美因河畔法兰克福|法兰克福]]，[[伦敦|伦敦]]，[[巴黎|巴黎]]，[[莫斯科|莫斯科]]，[[布鲁塞尔|布鲁塞尔]]和[[华沙|华沙]]同列。超过30个新高层项目正在制定，包括{{Convert|165|m|ft|0}}高的[[马斯塔|马斯塔]]（Maas Tower），[[新奥尔良塔|新奥尔良塔]]（New Orleans Tower，约{{Convert|158|m|ft|0}}高），以及Zalmhaven Urban Tower {{Convert|195|m|ft|0}}。<ref>{{cite web |title=Sky scraper city project view|url=http://www.skyscrapercity.com/showthread.php?p=17425220#post17425220 |accessdate=2008-05-15}}</ref>

== 体育 ==
鹿特丹有四个足球俱乐部。其中[[飛燕諾|飛燕諾]]于1999年获荷兰冠军，2002年获[[欧洲联盟杯|欧洲联盟杯]]，在其大球场进行过2000年欧洲足球冠军赛决赛。

鹿特丹每年举行[[马拉松|马拉松]]赛。从1985年到1998年世界马拉松纪录是在鹿特丹建立的。

== 购物 ==
[[File:Open_market_in_rotterdam.JPG|thumb]]
鹿特丹的人行区是荷兰最早的人行区。市内有一些大的购物中心。

== 商业和工业 ==
鹿特丹拥有荷兰部分的[[联合利华|联合利华]]和世界上最大的钢铁厂[[米塔尔钢铁公司|米塔尔钢铁公司]]。

鹿特丹港是欧洲最大的港口，它与[[上海港|上海港]]的关系密切。

鹿特丹大学的管理和经济研究和教育非常强。大学位于城市的东部，周围有许多全球性的企业。
