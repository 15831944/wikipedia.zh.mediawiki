==變遷==

===興建-僑委會使用===
1960年代，許多校外機構陸續與臺大訂定契約，在臺大周邊預定地上興建與台大無關之建築物，今日之鹿鳴堂即為該時期的建築物<ref>{{cite web|url=http://homepage.ntu.edu.tw/~cpo/download/2009plan/chap_01/chap_01.pdf| title=校園規劃報告2009 | accessdate=2012-11-11}}</ref>
。當時，[[中華民國|中華民國]][[僑務委員會|僑務委員會]]為了經營海內外僑生業務，在[[美援|美援]]的經費下，興建僑生活動中心<ref>{{cite web|url=http://wfotaa.freeinterchange.org/zh_tw/journa3_2000_1_31.htm|title=世界留台校友會聯誼總會|accessdate=2012-11-11}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。總統[[蔣中正|蔣中正]]命名為'''僑光堂'''<ref>{{cite web| url=http://www.ezfreegroups.com/index.php?mod=journal&action=show&cid=13&pid=40&id=670| title=僑務委員會僑務大事記| accessdate=2012-11-11| archive-url=https://web.archive.org/web/20180709094514/http://www.ezfreegroups.com/index.php?mod=journal&action=show&cid=13&pid=40&id=670| archive-date=2018-07-09| dead-url=yes}}</ref>，設計師為創立淡江大學建築系的[[馬惕乾|馬惕乾]]<ref name=":1" />，於1967年開始興建<ref name="pictures">臺大校史館.青春、記憶、老臺大：臺大老照片的二十個故事.臺北.國立臺灣大學出版中心.2012.11.13</ref>。
1968年5月4日，舉行落成典禮，由副總統[[嚴家淦|嚴家淦]]剪綵主持<ref>{{cite web| url=http://memory.ncl.edu.tw/tm_2007/hypage.cgi?HYPAGE=all_detail.htm&subject_type=tools&did_id=8&project_id=news&xml_id=0000022702| title=僑委會籌建僑生活動中心僑光堂落成嚴副總統剪綵| accessdate=2012-11-11}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

僑光堂的一樓為中菜餐廳、會議室、貴賓招待室，二樓則為挑高的禮堂，可供會議或集會使用<ref>{{cite web |url=http://homepage.ntu.edu.tw/~buhome/file/new-051205-a.pdf| title=台大總務處| accessdate=2012-11-11}}</ref>。之後，僑光堂成為每年雙十國慶接待歸國僑胞的主要場所，另外也曾經舉辦過圖書展覽<ref>{{cite web| url=http://nrch.cca.gov.tw/ccahome/search/search_meta.jsp?xml_id=0005920539| title=第一屆全國圖書雜誌展覽揭幕| accessdate=2012-11-11}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、藝術品展覽、頒獎典禮<ref>{{cite web |url=http://memory.ncl.edu.tw/tm_cgi/hypage.cgi?HYPAGE=toolbox_vnews_detail.hpg&subject_name=&subject_url=&project_id=news&dtd_id=8&xml_id=0000020512| title=僑委會頒獎優良僑生及球隊代表| accessdate=2012-11-11}}</ref>、各式會議<ref>{{cite web |url=http://catalog.digitalarchives.tw/item/00/31/a7/ce.html| title=第一屆全國工業安全衛生會議| accessdate=2012-11-11}}</ref>、聯誼會<ref>{{cite web |url=http://catalog.digitalarchives.tw/item/00/31/a3/87.html| title=第二屆亞洲青年育樂營揭幕| accessdate=2012-11-11}}</ref>、座談會、音樂會、舞會、慶祝會等等。

===臺灣大學收回使用===
1996年1月<ref name="pictures"></ref>，僑委會遷出，台大收回使用，整修後更名為'''鹿鳴堂'''。命名引據[[詩經|詩經]]小雅「鹿鳴篇」。在中國也有以鹿鳴為名的餐宴-[[鹿鳴宴|鹿鳴宴]]。其始於唐代，是當時為考中的舉人而設的宴席。鹿鳴堂取名為此，除了有邀請眾人分享美食之意，也有恭賀考生金榜題名、未來發達的含意<ref>{{cite web |url=http://homepage.ntu.edu.tw/~cpo/plan/CS_road.pdf| title=大學校園發展的新里程--談舟山路環境改善| accessdate=2012-11-11}}</ref>。
===配合舟山路封巷鹿鳴廣場改建===
2000年7月舟山路經台北市政府核定廢巷，台大即著手這一帶的改建工程，2002年鹿鳴廣場落成，鹿鳴堂外部圍牆也隨之拆除。為活化空間，於側牆開一個門並將側邊改成木平台及增加座椅。內部空間部分，一樓原餐廳業者契約到期結束營業後，台大將此空間招標委外經營，得標後引進中式宴席、麵包店、簡餐等。二樓部分則找到新的定位，改為台大劇場，重新定義為一個「黑盒子劇場」，意即劇場本身不限定舞台及觀眾席的相互關係，提供演出者及使用者最大的可能性，約可容納觀眾 200～300 人。從2004年暑假開始，[[台大戲劇系|台大戲劇系]]師生開始積極投入臺大劇場規劃作業，並在此劇場演出畢業製作和學期製作<ref>{{cite web |url=http://homepage.ntu.edu.tw/~theatre/production.htm| title=國立臺灣大學戲劇學系| accessdate=2012-11-11}}</ref>，另外此劇場也供師生社團或校外人士租借做為藝文活動之用。
===拆除與爭議===
鹿鳴堂經過多年的使用，現在是一棟危樓，做為公共表演空間的台大劇場設備及可用性也有限，因此台大正在評估重新蓋一棟機能完整的建築，具有行政辦公空間和演藝廳等功能<ref>{{cite web |url=http://homepage.ntu.edu.tw/~cpo/committee/100/100_01.pdf |title=國立臺灣大學校務發展規劃委員會校園規劃小組 100 學年度第 1 次委員會會議紀錄 |accessdate=2012-11-11}}</ref>。

鹿鳴堂將要拆除的消息，引發台大師生的質疑。2018年7月7日，台大學生發現，在台北市文化局官上網審議鹿鳴堂文資價值的2017年9月18日第97次文資會議記錄不翼而飛，質疑鹿鳴堂文資審議不透明<ref>{{Cite news|url=https://udn.com/news/story/7266/3240846|title=獨╱鹿鳴堂文資審議紀錄被消失？ 台大生要北市府講明白|author=聯合報 記者馮靖惠|date=2018-07-08 11:20|work=聯合報|accessdate=}}</ref>。台大城鄉所則有教授認為，拆除鹿鳴堂即等於抹除台大校園空間發展歷史過程中的實體見證<ref name=":1">{{Cite news|url=https://udn.com/news/story/7266/3240754|title=台大鹿鳴堂將拆 僑光堂歷史不再 學者：抹除實體見證|author=|last=|date=|work=聯合新聞網|newspaper=|accessdate=2018-07-09}}</ref>。

=== 結合現代主義手法的仿古中式建築 ===
鹿鳴堂為淡江大學建築系創系教授[[馬惕乾|馬惕乾]]設計<ref name=":1" />，外觀上具有中國式的仿古建築元素，如圓柱、圓形柱礎、雀替、椽條、雨批、仿中國明清官式建築欄杆、大量花窗等意象，也是台大校園內唯一現存的中式建築。由於鹿鳴堂在中華文化復興運動發起隔年開始興建，因此推測可能是在此脈絡之下採用仿古造形<ref name=":2">{{Cite web|url=https://solomo.xinmedia.com/archi/140936-ActionA|title=一探台大「鹿鳴堂」的前世今生-欣建築-欣傳媒建築設計頻道|accessdate=2018-07-09|work=欣傳媒|language=zh-Hant-TW}}</ref>。

除了這些中式語彙，建築另一特別之處為正面與側面填充牆面的一顆顆正方形錐體。這些椎體與鹿鳴堂的裝飾性中國式樣單元不同。後者不具有結構意義跟使用合理性，但前者內部具有鋼筋，從而兼具裝飾與結構支撐功能，體現出現代主義式建築手法<ref name=":2" />。這也反映出建築師在仿古裝飾的要求下，融入現代主義手法，追求構造真實性<ref name=":2" />。

==鹿鳴雅舍==
鹿鳴雅舍是鹿鳴堂旁的住宿部，和鹿鳴堂同時興建於1967年，是一棟四層樓的白色建築物，提供平價的短期住宿服務。鹿鳴雅舍的一樓是校園紀念品販售商店，販賣各種台大紀念商品。

為配合「[[台大卓越聯合中心|台大卓越聯合中心]]」（即第三行政大樓）之興建，鹿鳴雅舍在2013年9月結束營業，2014年9月初拆除。
