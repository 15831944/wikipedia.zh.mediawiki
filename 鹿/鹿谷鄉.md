'''鹿谷鄉'''位於[[臺灣|臺灣]][[南投縣|南投縣]]西南部，是全臺灣唯一'''[[台大實驗林|台大實驗林]]'''佔面積過半的鄉鎮，北鄰[[集集鎮|集集鎮]]，東鄰[[水里鄉|水里鄉]]、[[信義鄉_(台灣)|信義鄉]]，西及南接[[竹山鎮_(台灣)|竹山鎮]]。鄉內的行政中心位於[[鹿谷村|鹿谷村]]。鹿谷地區古稱'''羌仔寮'''（{{lang-nan|Kiuⁿ-á-liâu}}），後於[[台灣日治時期|日治]]1920年改名「鹿谷」。<ref>《台灣地名研究》，安倍明義，173頁。</ref><ref>[http://twblg.dict.edu.tw/holodict_new/index.html 台灣閩南語常用詞辭典]</ref>

== 歷史 ==
'''鹿谷鄉'''的開發最早的文字記錄是在1741年左右( 清[[乾隆|乾隆]]六年 )，由[[漳州|漳州]]人沿[[濁水溪|濁水溪]]進入番仔寮與大坵園（今瑞田村及清水村）開墾。之後，又有[[泉州|泉州]]人自林圮埔（今[[竹山鎮_(台灣)|竹山鎮]]）進入囤墾，陸續的有初鄉-{庄}-（初鄉村）、新寮-{庄}-（鹿谷村）、坪仔頂-{庄}-（秀峰村）、車輄寮-{庄}-（廣興村）、小半天-{庄}-（竹林村、竹豐村）、[[內樹皮|內樹皮]]-{庄}-（和雅村）等各-{庄}-的形成。並於道光11年間，初鄉舉人[[林鳳池|林鳳池]]從[[福建|福建]][[武夷山|武夷山]]帶回軟枝烏龍茶栽種於[[鹿谷村|鹿谷]]、彰雅、永隆、鳳凰四村，即成為今日聞名國際的[[凍頂烏龍茶|凍頂烏龍茶]]。

==地理==
===地形===
鹿谷鄉地形大致分為為沖積地、丘陵、山地、台地等

===水文===
北勢溪：流經竹豐村、竹林村、和雅村、內湖村、廣興村、彰雅村、鹿谷村

清水溝溪：流經瑞田村、清水村、秀峰村、永隆村、鳳凰村、彰雅村(支流)、鹿谷村(支流)

===經緯度位置===
*東極：北緯二三度四二分三八秒，東經一二○度五○分二五點四秒
*西極：北緯二三度四六分一六秒，東經一二○度四二分四九點九秒
*南極：北緯二三度三八分五七點三秒，東經一二○度四八分一四點五秒
*北極：北緯二三度四九分五點三秒，東經一二○度四六分五六點二秒

==人口==
鹿谷鄉屬於山區鄉鎮，多年來因工作機會少的緣故導致有人口外流的情形；但近年來因[[觀光業|觀光業]]發展蓬勃，可望逐年減緩人口外流情況。

== 氣候與文化 ==
'''鹿谷鄉'''的行政中心位於海拔510公尺的鹿谷台地，屬於[[暖溫帶氣候區|暖溫帶氣候區]]，年均溫22℃，氣候宜人，為著名的避暑勝地。鹿谷鄉雨量豐沛，再加上終年雲霧籠罩，土壤富含有機質，十分適合茶樹生長，所產「[[凍頂烏龍茶|凍頂烏龍茶]]」已成所有[[茶葉|茶葉]]之代名詞。該鄉飲茶品茗已成一門獨到的日常生活藝術，更因價格穩定，居民享有極高生活水準。該鄉有八成的居民以[[茶葉|茶葉]]和[[孟宗竹|孟宗竹]]為主要經濟來源，比例相當高。除茶葉和孟宗竹外，[[山芹菜|山芹菜]]及[[鱒魚|鱒魚]]也是該鄉特產。 鄉內土地有60%屬[[台大實驗林|台大實驗林]]，國有林地佔8%，林相保存良好，是全台灣少有'''低開發程度'''的美麗鄉鎮，[[溪頭|溪頭]]、[[杉林溪|杉林溪]]等著名景點，每年更吸引數以百萬人次前來鄉內休閒旅遊。

== 教育 ==
鹿谷鄉的教育機構僅到[[國中|國中]]，且鄉內部分小學因人口外流，學生人數有逐漸減少的趨勢。

== 特產 ==
鹿谷鄉和雅村[[內樹皮|內樹皮]]為全台灣第一個種植[[孟宗竹|孟宗竹]]的地方，造就鹿谷鄉的孟宗竹栽植面積佔全國56%，因此[[冬筍|冬筍]]成為鹿谷鄉民重要經濟來源。
