'''鹿潔身'''（{{bd|1956年|8月||}}），[[中華民國|中華民國]]交通界人士，曾任[[臺灣鐵路管理局|臺灣鐵路管理局]]局長<ref>{{Cite web|url=http://www.railway.gov.tw/tw/CP.aspx?sn=3582|title=TRA WebSite - tw|accessdate=2017-09-14|date=2008-12-11|last=交通部臺灣鐵路管理局|work=交通部臺灣鐵路管理局}}</ref>。

== 生平 ==
鹿潔身是[[國立成功大學|成大]]交通管理科學系畢業，當年參加成大與台鐵的[[建教合作|建教合作]]計畫。民國69年，鹿潔身正式進入台鐵，長年在運務部門工作，從[[列車長|列車長]]做起，擔任過副站長、[[臺北運務段|臺北運務段]]段長、運務處副處長，對運務工作相當熟悉<ref>{{Cite web|url=https://www.myplus.com.tw/mag-article.aspx?nid=1250|title=鐵路 承載你我的情感與回憶|website=加分誌|date=2016-11-29}}</ref>。

2012年，鹿潔身升任為台鐵局副局長。

2016年9月，台鐵局局長[[周永暉|周永暉]]轉任[[交通部觀光局|觀光局]]局長，鹿潔身代理台鐵局長。10月，鹿潔身真除為台鐵局長<ref>{{Cite web|url=https://m.ltn.com.tw/news/politics/breakingnews/1846857/|title=交通部人事異動 鹿潔身真除、趙興華接高公局|website=[[自由時報|自由時報]]|date=2016-10-05}}</ref>。

2018年10月21日，台鐵局長鹿潔身因[[2018年宜蘭普悠瑪列車出軌事故|普悠瑪列車出軌事故]]提出口頭請辭。10月25日，[[中華民國交通部|交通部]]部長[[吳宏謀|吳宏謀]]以「事情暫告一段落」為由，同意鹿潔身請辭<ref>{{Cite web|url=https://tw.appledaily.com/new/realtime/20181025/1454018/|title=台鐵局長鹿潔身請辭獲准！ 普悠瑪翻車首位官員下台|website=[[苹果日报_(台湾)|蘋果日報]]|date=2018-10-25}}</ref>。11月9日，台鐵局辦理新、舊任局長交接，鹿潔身正式卸下台鐵局長一職<ref>{{Cite web|url=https://news.tvbs.com.tw/politics/1025861|title=「當個終身台鐵人」　鹿潔身退休結束台鐵40年生涯|website=[[TVBS|TVBS]]|date=2018-11-09}}</ref>。

== 軼聞 ==
鹿潔身與後任局長[[張政源|張政源]]、前任局長[[范植谷|范植谷]]為成大交管系同一屆、而且是同班同學<ref>{{Cite web|url=https://wantweekly.turnnewsapp.com/exclusive/3582.html|title=台鐵3局長 成大交管保證班|website=[[周刊王|周刊王]]|date=2019-01-04}}</ref>。
