'''鹿林天文台'''是一座設於[[嘉義縣|嘉義縣]][[阿里山鄉|阿里山鄉]]及[[南投縣|南投縣]][[信義鄉_(臺灣)|信義鄉]]交界處[[鹿林前山|鹿林前山]]的[[天文台|天文台]]，在[[玉山國家公園|玉山國家公園]]之內，海拔2862公尺，由[[國立中央大學|國立中央大學]]在1999年設立，前身為鹿林觀測站。目前由[[國立中央大學天文研究所|國立中央大學天文研究所]]管理。

== 歷史 ==
國立中央大學在1981年就在校內設有天文台。但因為該校所在的[[桃園市|桃園縣]][[中壢區|中壢市]]（今桃園市中壢區）光害逐漸嚴重，中央大學遂於1990年到1994年間在行政院國家科學委員會的支持下進行天文台選址，最後選定鹿林天文台現址<ref name="物理雙月刊-1998">{{cite journal|author1 = [[蔡文祥_(天文學家)|蔡文祥]] |author2 = 張光祥 |author3 = 黎煥欣 |title = 中大鹿林天文台介紹 |url = http://www.lulin.ncu.edu.tw/doc/LulinHistory.htm |publisher = 中華民國物理學會 |journal = 物理雙月刊 |issue = 24 |issn =  |doi =  |date = 1998-08 |accessdate = 2017-11-08 |language = zh-tw |archiveurl = https://web.archive.org/web/20171108144453/http://www.lulin.ncu.edu.tw/doc/LulinHistory.htm |archivedate = 2017-11-08 |deadurl= no}}</ref>。1997年在[[國家太空中心_(中華民國)|國家太空計畫室]]的支持下進行建設，於1999年落成。2002年啟用口徑一米望遠鏡。
