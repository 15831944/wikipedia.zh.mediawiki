'''鹿阜街道'''，全称'''鹿阜街道办事处'''，是[[中华人民共和国|中华人民共和国]][[云南省|云南省]][[昆明市|昆明市]][[石林彝族自治县|石林彝族自治县]]下辖的一个[[乡镇|乡镇]]级行政单位，也是石林彝族自治县县治所在<ref>{{PRC admin/ref sgc|{{PRC admin/parent|53/01/26/001/000}}}}</ref><ref>{{Cite book|title=常见灾害事故的预防与救助|last=周白霞主编|first=|publisher=北京：中国环境科学出版社|year=2014年8月|isbn=978-7-5111-1879-0|location=|pages=221}}</ref><ref>{{Cite book|title=2014中华人民共和国行政区划简册|last=中华人民共和国民政部编|first=|publisher=北京：中国地图出版社|year=2014年5月|isbn=978-7-5031-6304-3|location=|pages=154}}</ref>。

== 历史沿革 ==
原称鹿阜镇，在县境西部，[[巴江|巴江]]中游，东接[[维则乡|维则乡]]，南连[[板桥乡|板桥乡]]，西与[[宜良县|宜良县]][[狗街镇|狗街镇]]相邻，北与[[路美邑乡|路美邑乡]]接壤。海拔1675米。从唐朝始就成为政区治所驻地。明朝成化十三年（1477年），建州署于[[鹿阜山|鹿阜山]]，今仍为石林县治驻地，为全县政治、经济、文化中心<ref>{{Cite book|title=石林彝族自治县志 1989-2000|last=刘世生主编；石林彝族自治县志编纂委员会编|first=|publisher=昆明：云南民族出版社|year=2006年10月|isbn=7-5367-3448-4|location=|pages=22-24}}</ref>。清咸丰前名阜城乡。民国时改称鹿阜镇。1954年，改称城关镇，1958年，为路南公社，1961年为路南区，1965年复为城关镇。1988年，归鹿阜镇<ref>{{Cite book|title=中国邮政编码大全 第8卷 街道乡村卷 下|last=邮电部邮政总局主编|first=|publisher=北京：人民邮电出版社|year=1989年11月|isbn=7-115-04134-2|location=|pages=590}}</ref><ref>{{Cite book|title=昆明市2010年人口普查资料 上|last=昆明市第六次人口普查办公室，昆明市统计局编|first=|publisher=北京：中国统计出版社|year=2014年6月|isbn=978-7-5037-6786-9|location=|pages=83}}</ref>。[[彝族|彝族]]、汉族等民族杂居<ref>{{Cite book|title=路南彝族自治县志|last=昆明市路南彝族自治县志编纂委员会编|first=昆明：云南民族出版社|publisher=|year=1996年11月|isbn=7-5367-1294-4|location=|pages=71}}</ref>。2011年2月，鹿阜镇、[[石林镇|石林镇]]、[[板桥镇|板桥镇]]合并为'''鹿阜街道'''。2017年，鹿阜街道析置为鹿阜、[[石林街道|石林]]、[[板桥街道_(石林县)|板桥]]3个街道。同时将北大村和板桥村委会改设为居委会<ref name=a>{{Cite web|url=http://www.shilin.gov.cn/c/2017-01-17/1452564.shtml|title=石林县鹿阜街道一分为三|accessdate=2017-07-28|work=中国石林网-今日石林-中国石林网}}</ref>。

== 行政区划 ==
2017年1月，鹿阜街道下辖以下地区：辖龙泉、南门、东门、西北4个居委会和大屯、小乐台旧、大乐台旧、东海子、阿乌、三板桥、山冲、宏图、铺兵、上蒲草、清水塘、新宅、路美邑、大哨、小河、 所卜所、阿怒山、北山、麦地庄、堡子、鱼龙坝21个村委会，国土面积232.9平方千米，总人口105662人，驻地为龙泉居委会双龙山18号<ref>{{Cite book|title=昆明年鉴 2013|last=昆明市人民政府主办，昆明市地方志编篆委员会办公室编|first=|publisher=昆明：云南人民出版社|year=2013|isbn=978-7-222-11248-3|location=|pages=488}}</ref><ref name=a />。

== 产业 ==
当地政府与民众筹集水利工程，完成干山冲烟水工程、南门紫玉抽水站等。全镇以[[水稻|水稻]]为主要粮食作物，玉米、小麦、豆类、红薯等为辅，特产豆腐<ref>{{Cite book|title=石林年鉴 2006|last=《石林年鉴》编辑部编|first=|publisher=昆明：云南美术出版社|year=2006年11月|isbn=7-80695-453-8|location=|pages=55}}</ref>。工业有农业机械、电力、汽车修配、五金、炸药、制药等。旅游有[[雁塔山古塔|雁塔山古塔]]、[[高泽|高泽]]墓志铭碑、文庙大成殿等，风景有[[灵秀湖|灵秀湖]]、西山公园、龙江公园等<ref>{{Cite book|title=云南辞典|last=施之厚主编；《云南辞典》编辑委员会编辑|first=|publisher=昆明：云南人民出版社|year=1993|isbn=7-222-01264-8|location=|pages=115}}</ref>。

石林县鹿阜中心学校也位于鹿阜街道<ref>{{Cite book|title=云南教育大全 单位卷|last=《云南教育大全》编委会编|first=|publisher=昆明：云南教育出版社|year=2007年8月|isbn=978-7-5415-3288-7|location=|pages=104}}</ref>。
