'''不动明王'''亦称'''不动使者'''或'''不动尊'''（{{lang-sa|'''अचलनाथ''' {{IAST|acalanātha}}}}），[[佛教|佛教]][[密宗|密宗]]{{lj|五大明王|五大明王}}主尊、八大明王首座，[[毗盧遮那佛|毗盧遮那佛]]（[[大日如来|大日如来]]）的教令轮身。在镇守东西南北中五个方位的五大明王中，为镇守中央方位的明王，也是著名的[[護法神|護法神]]。

== 概述 ==
不动明王名称中的“不动”，是指[[慈悲心|慈悲心]]不变，无物可以改变撼动，“明”则乃智慧的光芒，“王”是对能操控世间万物、现象的人的尊称。不动明王其誓愿为“见我身者发菩提心，闻我名者断恶修善，聞我法者得大智慧，知我身者即身成佛。<ref>《佛学大辞典》</ref>”

不动明王被视为[[毗盧遮那佛|毗盧遮那佛]]（大日如来）的忿怒[[化身|化身]]，打败过[[大自在天|大自在天]]。相传大自在天认为自己是[[三千世界|三千世界]]之主，故心存傲慢，不肯皈依[[佛祖|佛祖]]，所以佛祖化身不动明王，前去降伏其心，不动明王用其足踏大自在天的[[首級|首級]]，使其歸順，在诸明王之中不动明王也是最高尊者。<ref>《底哩三昧耶經》</ref>

不动明王在[[汉传佛教|汉传佛教]]或[[藏传佛教|藏传佛教]]都有相当高的声誉，不动明王與[[观音菩萨|观音菩萨]]和[[地藏菩萨|地藏菩萨]]一般，乃是[[藏族|藏民]]供奉的三尊主要佛像之一。

在[[日本佛教|日本佛教]]，[[東密|東密]]的[[信徒|信徒]]多半信奉不动明王，[[高野山|高野山]]是其[[道场|道场]]。在[[江戶時代|江戶時代]]，[[幕府將軍|幕府將軍]]在[[都城|都城]][[江戶|江戶]]設置有[[五色不動|五色不動明王]]，以祈求[[政權|政權]]的和平穩固。

[[日本人|日本人]]認為不动明王能鎮守[[溫泉|溫泉]]，[[臺灣|臺灣]]經歷過[[台灣日治時期|日治時期]]，在各溫泉區多有'''不动明王'''的[[香火|香火]]，如[[臺北市|臺北市]][[北投區|北投區]]、[[嘉義縣|嘉義縣]][[中埔鄉|中埔鄉]]、[[臺南市|臺南市]][[白河區|白河區]][[關子嶺|關子嶺]]等地有寺廟奉祀。另外在[[花蓮縣|花蓮縣]][[太魯閣|太魯閣]]寧安橋下以及[[吉安鄉|吉安鄉]][[慶修院|慶修院]]、[[干城村|干城村]]西寧寺和[[壽豐鄉|壽豐鄉]][[豐田村|豐田村]][[豐田神社|碧蓮寺]]四地有奉祀明王尊像，並非[[道教|道教]]的[[火神|火神]][[火德星君|火德星君]]。而日本消防隊員也在廳舍內供奉'''不动明王'''的佛像。

== 形象 ==
不动明王周身呈现青蓝色，右手持智慧[[剑|剑]]，左手拿金刚索，右眼仰视，左眼俯视，周身火焰。一般都以愤怒的形象示人，降伏內心諸魔障。而不动明王有两臂、四臂、六臂多个形象，但大多以两臂形象示人。其动作形象的来源主要是由（大日如來）的忿怒形象而来，多做舞蹈相，立相<ref>大日经疏卷五：“此尊坐盘石座，呈童子形。顶上有七髻，辫发垂于左肩，左眼细闭，下齿啮上唇，现忿怒相，背负猛火，右手持利剑，左手持罥索，作断烦恼之姿。”</ref>。

== 佛教守护神 ==
不动明王是其佛教八大守护神之一，同时也是十二生肖的守护本尊。

佛教《大集经》记载，我们的东南西北有十二神兽镇守，此十二神兽即汉地十二生肖之起源。《法苑珠林》曰：「[[阎浮提|阎浮提]]外，四方海中，有十二兽，并是菩萨化导，人道初生，当菩萨住窟，即属此兽护持、得益，故汉地十二辰依此行也。」而不动明王对应的十二生肖中的鸡（对应其他地区的金翅鸟等），负责护佑、教化一方众生。所以在中国属鸡的人也佩戴“不动明王”法相作为守护神、本命佛。
