{{日本作品|不眠的真珠|眠れぬ真珠|}}為[[石田衣良|石田衣良]]創作的長篇小說，第13屆{{tsl|ja|島清恋愛文学賞|島清戀愛文學賞}}得獎作。

[[吉田真由美|吉田真由美]]將其漫畫化，並於{{tsl|ja|女性セブン|女性Seven}}中連載。2015年[[黑谷友香|黑谷友香]]主演配信連續劇<ref>{{Cite web |url=https://www.excite.co.jp/News/cinema/20150402/Crankin_3617703.html |title=黒谷友香×中村蒼、官能的シーン披露！ 石田衣良小説『眠れぬ真珠』ドラマ化 |date=2015-04-02 |publisher=クラインクイン！ |accessdate=2017-12-21}}</ref>。2017年[[讀賣電視台|讀賣電視台]]將其連續劇化，並由[[藤原紀香|藤原紀香]]主演<ref>{{Cite web |url=https://www.excite.co.jp/News/entertainment_g/20171220/Messy_58934.html |title=『眠れぬ真珠』鈴木伸之の色気爆発！　藤原紀香と濃厚ベッドシーンに悲鳴続出も期待大!! |date=2017-12-20 |publisher=エキサイトニュース |accessdate=2017-12-21}}</ref>。
