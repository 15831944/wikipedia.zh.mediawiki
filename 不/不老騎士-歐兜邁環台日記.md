== 紀錄片攝製來源 ==
*弘道老人福利基金會於2007年發起「挑戰八十、超越千里~不老騎士的歐兜邁環台日記」方案，期待能提升國人對老化的正面觀點與行動。<ref>[http://www.hondao.org.tw/motor_taiwan/about.htm 挑戰八十、超越千里-不老騎士-歐兜邁環台日記 弘道歐兜邁部落格] {{webarchive|url=https://web.archive.org/web/20121115211140/http://hondao.org.tw/motor_taiwan/about.htm |date=2012-11-15 }}</ref>

== 影片後製與發佈 ==
*2008年，弘道老人福利基金會先行發布短片《不老騎士-2008搶先版》。<ref>[http://blog.xuite.net/hondao/motortaiwan/27741193 暫緩《不老騎士》電影長片上映計畫 弘道老人福利基金會 2009-10-15 08:42 AM]</ref>
*2009年，弘道老人福利基金會因全力投入[[林邊鄉|林邊鄉]]光林村之[[八八水災|八八水災]]災後重建工作，原訂當年度秋天預計推出的電影長片暫緩執行。

== 劇情簡介 ==
這不僅僅是一個普通的環島旅程，也是一個圓夢之旅。團長賴清炎說：「追求夢想時，你會忘記自己幾歲。」

平均81歲的長者，心懷18歲的騎士夢。在弘道老人福利基金會的計畫下，於2007年11月13日成行。隊伍花費為期13天的時間，終於完成了這個在眾人眼中「不可能完成的夢想」。

* [[臺灣|臺灣]]「不老騎士」這股十三天機車環島壯舉，感動[[美國|美國]][[重機|重機]]愛好者，美國[[摩托車|摩托車]]雜誌（Motorcyclist）編輯、（[[探索頻道|探索頻道]]）[[製作人|製作人]]彼得.史塔（Peter Starr）於2011年時，應[[臺灣|臺灣]][[觀光局|觀光局]]駐[[洛杉磯|洛杉磯]]辦事處之邀，來臺勘查此地旅遊路線，發現臺灣環島旅遊相對路程極近，極適合機車旅遊，返美後，於其雜誌上發文報導，引發部份讀者迴響。近日又號召十位年長重機愛好者(五位平均年齡七十二歲，五位陪騎人員)。於2012年10月16日沿續臺灣「不老騎士」的出發地點集結，以四天時間尋訪[[阿里山國家風景區|阿里山]]、[[日月潭|日月潭]]、[[太魯閣|太魯閣]]等觀光景點。<ref>[http://udn.com/NEWS/NATIONAL/NAT5/7432663.shtml 美版不老騎士 11人今遊台追夢 曾懿晴／台北/聯合報 2012-10-16 02:57 am]</ref><ref>[http://www.cna.com.tw/News/aEDU/201210150091.aspx 美版不老騎士 重機遊台追夢 徐卉/台北/中央社 2012/10/15 12:24:07]</ref><ref>[http://tw.news.yahoo.com/%E4%B8%8D%E8%80%81%E9%A8%8E%E5%A3%AB-%E6%84%9F%E5%8F%AC-%E7%BE%8E%E9%87%8D%E6%A9%9F%E5%90%8C%E5%A5%BD%E9%81%8A%E5%8F%B0-050559792.html "不老騎士"感召 美重機同好遊台 林尚賢/陳建華/台中/民視新聞 2012/10/16 19:06 PM]</ref>並在2012年10月22日齊聚[[台北市|台北市]]大直美麗華戲院，觀賞「不老騎士」的長紀錄片。而於10月23日搭機離開臺灣。
