'''不可數集'''是[[無窮集合|無窮集合]]中的一種。一個無窮集合和[[自然數集|自然数]]之間要是不存在一個[[双射|双射]]，那麼它就是一個不可數集。集合的不可数性与它的[[基数|基数]]密切相关：如果一个集合的基数大于[[自然数|自然数]]的基数，那么它就是不可数的。

== 定義 ==

不可数集有许多等价的定義。一个集合<math>X</math>是不可数集，当且仅当以下任何一个条件成立：
* 不存在从<math>X</math>到自然数集合的[[单射函数|单射函数]]。
* <math>X</math>的[[基数|基数]]既不是有限的，又不等于<math>\aleph_0</math>（[[阿列夫数|阿列夫-0]]，自然数集合的基数）。
* <math>X</math>的基数严格大于<math>\aleph_0</math>。

== 性质 ==
* 如果不可数集<math>X</math>是集合<math>Y</math>的子集，则<math>Y</math>是不可数集。

== 例子 ==

不可数集的最广为人知的例子，是所有[[实数|实数]]的集合<math>\mathbb{R}</math>；[[对角论证法|对角论证法]]证明了这个集合是不可数的。对角论证法也可以用来证明一些其它的集合是不可数的，例如所有自然数的无穷[[序列|序列]]的集合（甚至是所有-{只}-由0和1所组成的无穷序列的集合），以及自然数集合的所有[[子集|子集]]所组成的集合。<math>\mathbb{R}</math>的基数通常记为<math>c</math>、<math>2^{\aleph_0}</math>，或<math>\beth_1</math>。

[[康托尔集|康托尔集]]是<math>\mathbb{R}</math>的一个不可数子集。它是一个[[分形|分形]]，其[[豪斯多夫维|豪斯多夫维]]大于零，但小于一（<math>\mathbb{R}</math>的维数是一）。这是以下事实的一个例子：如果<math>\mathbb{R}</math>的某個子集有严格大于零的豪斯多夫维，那麼它一定是不可数的。

另外一个不可数集的例子，是所有从<math>\mathbb{R}</math>到<math>\mathbb{R}</math>的函数的集合。这个集合比<math>\mathbb{R}</math>更“不可数”，因为它的基数是<math>\beth_2</math>，它比<math>\beth_1</math>还要大。

一个更加抽象的例子，是所有可数[[序数|序数]]的集合，记为<math>\Omega</math>或<math>\omega_1</math>。<math>\Omega</math>的基数记为<math>\aleph_1</math>。利用[[选择公理|选择公理]]，可以证明<math>\aleph_1</math>是最小的不可数基数。于是，实数的基数<math>\beth_1</math>，要么等于<math>\aleph_1</math>，要么严格比它大。[[康托尔|康托尔]]是第一个提出<math>\beth_1</math>是否等于<math>\aleph_1</math>的问题的人。在1900年，[[希尔伯特|希尔伯特]]把这个问题作为他的[[希尔伯特问题|23个问题]]之一。<math>\aleph_1 = \beth_1</math>的陈述现在称为[[连续统假设|连续统假设]]，現已知道它獨立于[[集合论|集合论]]的[[策梅洛-弗蘭克爾集合論#公理|ZF公理]]（包括选择公理）。
