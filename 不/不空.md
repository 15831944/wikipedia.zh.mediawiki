'''不空'''（{{lang-sa|अमोघवज्र|Amoghavajra}}，{{bd|705年||774年|catIdx=b}}），音译为'''阿目佉跋折罗'''，意译为不空金刚，[[汉传密宗|汉传密宗]]的祖师。

== 遇金刚智 ==
不空是南天竺[[斯里兰卡|师子国]]人{{ref|birth}}，自幼聪颖异常，十四岁时（718年，[[开元|开元]]六年），在[[闍婆|闍婆国]]{{ref|java}}依止[[金刚智|金刚智]]。720年（开元八年）左右，跟随金刚智来到[[洛阳|洛阳]]。724年（开元十二年），刚满二十的不空，于[[广福寺|广福寺]]依[[說一切有部|說一切有部]]律仪受[[具足戒|具足戒]]。由于不空才华出众，通晓多种语言，金刚智译经时，常请不空作译语，协助翻译。学习声论，通常要12年时间，但不空六个月就完成；背诵文殊愿，一般人需半年，而不空仅用几天时间。不空渴望得到瑜伽五部三密法，但求了三年，金刚智都没有传，于是准备远赴天竺求法，金刚智观察梦兆后，知道不空是真法器，很快就将[[五部灌顶|五部灌顶]]、[[护摩阿闍黎教法|护摩阿闍黎教法]]、[[大日经悉地仪轨|大日经悉地仪轨]]、诸佛顶部诸多真言仪轨等，悉数教授无余。

== 赴狮子国 ==
741年（开元二十九年），[[唐玄宗|唐玄宗]]准许金刚智返回天竺，8月行至洛阳时金刚智圆寂。12月，不空率弟子含光、惠辩等三十七人从[[广州|广州]]出发，过[[诃陵|诃陵国]]{{ref|mjava}}，历经艰险，于次年到达狮子国，受到国王的隆重欢迎，国王甚至亲自为不空沐浴。而后不空拜见普贤阿闍黎，恭敬供养，求十八会金刚顶瑜伽法门毗卢遮那大悲胎藏，普贤遂给不空及含光、惠辩授五部灌顶，此外还传授了其他一些密法。不空非常勤奋，「经余三岁，寝食无安」，将各种仪轨、造像、坛法，都学得很精通。通过到天竺各地游历，搜集了许多经论梵本。

== 大唐传译 ==
746年（天宝五年），带着狮子国王的国书以及各种法宝{{ref|sutra}}，不空回到了大唐首都长安，奉敕暂住[[鸿胪寺|鸿胪寺]]。不久入宫为唐玄宗灌顶。后移居[[净影寺|净影寺]]，进行翻译、传法等。因祈雨止风有神效，唐玄宗赠号"智藏"。

749年（天宝八年），唐玄宗允准他回国，但路途中不空染疾，止于[[韶关|韶州]]{{ref|ill}}，唐玄宗敕令且住，不空在这里"卷不释手，扶疾翻译"。

753年（天宝十二年），因西平王[[哥舒翰|哥舒翰]]奏请，不空赴[[长安|长安]]居[[保寿寺|保寿寺]]，一个多月后前往[[武威|武威]]城住[[开元寺|开元寺]]，灌顶传法。

755年（天宝十四年），爆发[[安史之乱|安史之乱]]。第二年[[长安|长安]]陷落，[[唐玄宗|唐玄宗]]逃往[[四川|四川]]，[[唐肃宗|唐肃宗]]在[[灵武|灵武]]登基。

757年（[[至德|至德]]二年），唐肃宗还都，不空奏表祝贺。住錫[[大兴善寺|大兴善寺]]。

758年（至德三年，即[[乾元|乾元]]元年），不空上表奏请搜求梵本，以备翻译，得到敕许{{ref|trans}}。

765年（[[永泰|永泰]]元年），[[唐代宗|唐代宗]]授特进试鸿胪卿，号大广智三藏。

766年（永泰二年），不空奏请于[[五台山|五台山]]建[[五台山金阁寺|金阁寺]]{{ref|temple}}，代宗敕准。

774年（[[大历|大历]]九年）六月十五日，不空圆寂，遗言「无殉利以辱身，勿为名而丧道」，世寿七十，[[僧腊|僧腊]]五十。代宗哀悼，辍朝三日，并[[追赠|追赠]][[司空|司空]]，諡'''大辨正'''。
