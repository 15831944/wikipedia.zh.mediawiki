'''遠見控股有限公司'''，簡稱'''遠見控股'''（'''{{lang-en|Vision Values Holdings Limited}}'''，'''{{hkex|0862}}'''），則在2009年12月14日，由'''新世界移動控股有限公司'''換取上市而來。

現時業務在[[香港|香港]]經營網絡解決方案，項目服務及物業投資<ref>[http://www.visionvalues.com.hk/chi/corp_details.asp Vision Values Holdings Limited 簡介]</ref>，現時為[[魯連城|魯連城]]為主席及總裁。

而總部位於[[香港|香港]][[九龍灣|九龍灣]][[常悅道|常悅道]]19號[[福康工業大廈|福康工業大廈]]3樓309室。

新世界移動控股曾經是[[新世界發展|新世界發展]]的子公司之一，並持有[[新世界流動電話有限公司|新世界流-{}-動電話有限公司]]，該公司主要經營提供流動電話服務的[[新世界傳動網|新世界傳動網]]；新世界移動控股在2004年7月6日，透過[[亞洲物流科技有限公司|亞洲物流科技有限公司]]借殼上市。

2006年3月，新世界流動電話有限公司與[[CSL_(電訊)|香港流動通訊有限公司]]（CSL）完成合併。新世界傳動網現成為香港流動通訊有限公司旗下流動網絡服務品牌（其餘兩個品牌為[[1010|1010]]及[[One2Free|One2Free]]），而該公司是[[Telstra|澳洲電訊]]及新移動之合營公司[[CSL_New_World_Mobility_Group|CSL New World Mobility Group]]。

2006年11月22日，新移動將其持有的CSL New World Mobility Group共23.6%股權悉數售予給母公司新世界發展。<ref>[http://www.nwd.com.hk/download/2003/a061123c_NWMH.pdf 新世界移動與新世界發展的買賣協議] {{webarchive|url=https://web.archive.org/web/20070927045340/http://www.nwd.com.hk/download/2003/a061123c_NWMH.pdf |date=2007-09-27 }}</ref> 及後，新移動亦被母公司出售。 

2010年1月15日公佈正式易名為「'''Vision Values''' Holdings Limited」。
