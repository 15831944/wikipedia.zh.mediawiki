'''遠山和葉'''（日語：{{lang|ja|遠山 和葉}}/{{lang|ja|とおやま かずは}}，Tōyama Kazuha），是[[日本|日本]][[漫畫|漫畫]]和[[動畫|動畫]]作品《[[名偵探柯南|名偵探柯南]]》中的[[名偵探柯南角色列表|虛構角色]]。

命名來自[[遠山金四郎|'''遠山'''金四郎]]和[[青山剛昌|青山剛昌]]其中一位名叫'''和葉'''的女助手。

== 人物 ==
改方學園高中部二年級的學生，是[[服部平次|服部平次]]的青梅竹馬及同班同學，父親远山银司郎是{{link-ja|大阪府警察|大阪府警察|大阪府警本部}}刑事部部長，為[[服部平藏|服部平藏]]（[[服部平次|平次]]父親）的屬下。

好友是[[毛利蘭|毛利蘭]]，曾誤以為[[毛利蘭|小蘭]]就是[[服部平次|平次]]在[[東京|東京]]交到的姓「工藤」的女朋友，因此對其態度很不友善；經[[服部平次|平次]]解釋之後，冰釋前嫌成為好友<ref group="參" name="19卷"/>。常因摸不透[[服部平次|平次]]的想法而煩惱不已，是位體貼朋友的少​​女。

== 形象 ==
性格單純可愛、大膽勇敢、活潑樂觀、充滿朝氣，很喜歡小孩子，但方向感不好，是個路痴。

擅長[[合氣道|合氣道]]，等級為二段。和[[毛利蘭|小蘭]]一樣，害怕鬼怪和恐怖傳說。相當喜歡[[服部平次|平次]]，看到他與其他女性友好就容易[[妒忌|吃醋]]。

== 情感 ==
是[[服部平次|平次]]的初戀情人（但她本人並不知道）<ref group="參">M7《[[迷宮的十字路|迷宮的十字路]]》</ref>，本人以平次的「姊姊」自居，也常和他鬥嘴。小時候兩人曾因玩手銬，卻意外被銬在一起而引起大騷動，最後甚至連上廁所、洗澡時都在一起<ref group="參" name="19卷"/>。

雖然兩人堅決否認彼此為「男女朋友」<ref>在M14《[[天空的劫難船|天空的劫難船]]》中被川口聰（和葉遠房親戚的小孩）問到時。</ref>，但實際上並非如此。自己將小時候把二人鎖在一起的手銬的碎片做成護身符送給平次，而護身符每次也都能讓遭逢危難的平次保住性命，也曾救過柯南<ref group="參" name="19卷"/>。

與平次有着戀人已滿的關係，但雙方卻未曾正式對彼此告白。和葉曾嘗試向平次告白，但平次本人卻未聽到<ref group="參">漫畫第75卷，TV 655《毒和幻的設計（Illusion）》</ref>，後來平次在緊急狀況下脫口說出「你要對我的和葉做什麼？」，但事後卻被敷衍過去<ref group="參">漫畫第83卷，TV 764《柯南與平次 戀愛的暗號（後篇）》</ref>。平次暗自決定要找一個合適的地方告白，但總是受到外來的阻饒而失敗<ref group="參"> TV 832《殭屍包圍的別墅（後篇）》、874《柯南與平次的鵺傳說（解決篇）》</ref>。
