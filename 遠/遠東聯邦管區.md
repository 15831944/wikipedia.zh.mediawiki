'''遠東聯邦管區'''（{{lang-rus|Дальневосточный федеральный округ|Dalnevostochny federalny okrug}}）是[[俄罗斯|俄罗斯]]位於[[远东|远东]]的[[俄羅斯聯邦管區|聯邦區]]。2017年的人口为618万人<ref name="2010Census">{{ru-pop-ref|2010Census}}</ref>，总统特使是{{lang|ru|{{link-ru|Олег Александрович Сафонов|Олег Александрович Сафонов}}}}。

该地区自古以来就是东亚民族的活动地区，这块土地上曾先后存在[[肃慎|肃慎]]、[[夫余|夫余]]、[[靺鞨|靺鞨]]、[[女真|女真]]等民族，其部分地区歷史上也曾被[[中國历史|中國历史]]上的[[唐朝|唐]]、[[辽朝|辽]]、[[金朝|金]]、[[元朝|元]]、[[明朝|明]]、[[清朝|清]]六朝所統治。自从该地区歸[[俄罗斯|俄罗斯]]統治后，除了俄罗斯民族外，尚有[[赫哲族|赫哲族]]、[[达斡尔族|达斡尔族]]等当地居民。另外該地原來的一些東亞民族，包括[[漢族|漢族]]、[[滿族|滿族]]、[[朝鮮族|朝鮮族]]和[[大和族|大和族]]，不是遭到後來的蘇聯政府屠殺或強制遷徙，就是被驅逐出境回到自己的母國。

== 地理 ==
俄罗斯远东地区是俄罗斯滨临[[太平洋|太平洋]]的地区，它包括[[萨哈共和国|萨哈共和国]]、[[犹太自治州|犹太自治州]]和[[堪察加半島|堪察加半島]]。联邦区的首府是[[海参崴|海参崴]]（人口59.2万）。

== 历史 ==
1920年到1922年[[远东共和国|远东共和国]]从俄罗斯独立，在[[蘇聯|蘇聯]]与[[大日本帝國|大日本帝國]]之间造成了一个缓冲区。

2018年11月5日，俄总统普京签署法令，[[布里亚特共和国|布里亚特共和国]]与[[外贝加尔边疆区|外贝加尔边疆区]]从[[西伯利亚联邦管区|西伯利亚联邦管区]]划归远东联邦管区。

== 气候和动植物 ==
俄罗斯远东地区属于[[大陆性气候|大陆性气候]]，冬季非常寒冷，夏季短暂，在南部温暖，在北部也相当冷。大多数地区由北极[[针叶林|针叶林]]覆盖或者是[[冻原|冻原]]。冻原上只有矮的灌木、草和[[苔藓植物|苔藓植物]]，地面往往终年不化。[[农业|农业]]只有在少数受保护的地方可以进行，[[林业|林业]]是重要的经济部门。
