《'''巨神兵现身东京'''》（{{lang-ja|巨神兵東京に現わる}}），是在[[:ja:東京都現代美術館|东京都现代美术馆]]于2012年7月10日开幕的展览会“'''馆长 庵野秀明 特摄博物馆从迷你模型看昭和平成的技术'''”（{{lang-ja|館長 庵野秀明特撮博物館 ミニチュアで見る昭和平成の技}}）中公开的[[特摄片|特摄]][[短片|短片电影]]<ref> {{Cite web |url= http://www.oricon.co.jp/news/confidence/2011378/|title= ナウシカの“巨神兵”が特撮短編映画化　企画は“エヴァ”の庵野秀明|date=2012-05-10|publisher= ORICON STYLE|language= ja|accessdate= 2012-11-10 }}</ref><ref> {{Cite web|url=http://asahichinese.com/article/cool_japan/AJ201207100090|title=《风之谷》巨神兵现身东京|date=2012-07-10|publisher=朝日新闻中文网|language=zh-cn|accessdate=2012-11-10|deadurl=yes|archiveurl=https://web.archive.org/web/20140801215541/http://asahichinese.com/article/cool_japan/AJ201207100090|archivedate=2014-08-01}}</ref>。

==概要==
2012年7月10日开幕的展览会“馆长 庵野秀明特摄博物馆”的馆长[[庵野秀明|庵野秀明]]负责企划了本作品，并开始在馆内上映，片长9分3秒。本作是由《[[风之谷的娜乌西卡|风之谷的娜乌西卡]]》<ref>本作的企划庵野秀明，在动画版《[[风之谷的娜乌西卡|风之谷的娜乌西卡]]》中曾负责绘制巨神兵登场画面以及发射光束炮的画面。</ref>中登场的[[巨神兵|巨神兵]]衍生出的作品。

2012年11月17日，经制片人铃木敏夫提议、庵野秀明首肯，《'''巨神兵现身东京 劇場版'''》与动画电影《[[福音战士新剧场版：Q|福音战士新剧场版：Q]]》同时上映<ref> {{Cite web |url= http://www.cinematoday.jp/page/N0047646|title= 『ヱヴァQ』同時上映に『巨神兵東京に現わる』が決定！ジブリ鈴木Pの提案で実現！|date=2012-11-09|publisher= シネマトゥデイ|language= ja|accessdate= 2012-11-10 }}</ref><ref> {{Cite web|url= http://www.khara.co.jp/news/2012/11/000250.html|title= 『ヱヴァンゲリヲン新劇場版：Ｑ』 同時上映に 『巨神兵東京に現わる 劇場版』|date= 2012-11-09|publisher= khara|language= ja|accessdate= 2012-11-10|deadurl= yes|archiveurl= https://web.archive.org/web/20130130234442/http://www.khara.co.jp/news/2012/11/000250.html|archivedate= 2013-01-30}}</ref><ref> {{Cite web |url= http://extr.b-ch.com/cgi-bin/eva_news/official/rss_det.cgi?news_c=11261&newsctg_c=1001|title= 『巨神兵東京に現わる 劇場版』 同時上映のお知らせ|date=2012-11-09|publisher= 福音战士新剧场版：Q官方网站|language= ja|accessdate= 2012-11-10 }}</ref>。剧场版在声音和画面上加入新的调整，片长为10分7秒。

本片後收錄於《[[福音战士新剧场版：Q|福音战士新剧场版：Q]]》所發行的DVD與BD版中（但台灣發行授權版本未收錄）。
