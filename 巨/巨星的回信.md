{{作品名稱|name=''Danny Collins''|hk=巨星的回信|tw=翻唱人生|cn=丹尼·科林斯}}是一部2015年的美國喜劇電影，由[[丹·福吉爾曼|丹·福吉爾曼]]編劇和執導，是他首次執導的電影。本片靈感來自民謠歌手{{link-en|史蒂夫·蒂爾斯頓|Steve Tilston}}的真實故事<ref>{{Cite web |url = https://www.telegraph.co.uk/culture/music/music-news/7946804/John-Lennon-letter-to-aspiring-folk-singer-received-nearly-four-decades-later.html |title = John Lennon letter to aspiring folk singer received nearly four decades later |author = Laura Roberts |date = 2010-08-16 |publisher = Telegraph |language = en}}</ref>。主演是[[阿爾·柏仙奴|阿爾·柏仙奴]]、[[安納·貝寧|安納·貝寧]]、[[珍妮佛·嘉納|珍妮佛·嘉納]]、[[波比·簡拿威|波比·簡拿威]]和[[基斯杜化·龐馬|基斯杜化·龐馬]]。本片於2015年3月20日在戲院上映。

阿爾·柏仙奴憑他的演技而在[[第73屆金球獎|第73屆金球獎]]中得到提名[[金球獎最佳音樂及喜劇類電影男主角|金球獎最佳音樂及喜劇類電影男主角]]。<ref>{{Cite web |url = https://www.goldenglobes.com/film/danny-collins |title = Danny Collins |publisher = Goldenglobes.com |language = en}}</ref>

== 背景 ==
故事是基於[[約翰·連儂|約翰·連儂]]和[[小野洋子|小野洋子]]在1971年寫了一封信給英國民謠歌手{{link-en|史蒂夫·蒂爾斯頓|Steve Tilston}}，但在34年後他才發現信件的真實情況。真實的信是簽署了「約翰+洋子」（John + Yoko），而電影中的信是簽署了「約翰」（John）。
