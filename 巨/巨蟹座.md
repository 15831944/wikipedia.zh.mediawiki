'''巨蟹座'''（{{lang-la|'''Cancer'''}}，[[天文符号|天文符号]]：♋），[[黄道带|黄道带]]星座之一，面积505.87平方度，占全天面积的1.226%。巨蟹座中亮于5.5等的恒星有23颗，最亮星为[[柳宿增十|柳宿增十]]（巨蟹座β），视星等为3.52。每年1月30日子夜巨蟹座中心经过上中天。巨蟹座位于[[双子座|双子座]]和[[狮子座|狮子座]]之间、北方是[[天猫座|天猫座]]、南面则是[[小犬座|小犬座]]和[[长蛇座|长蛇座]]，是一个暗淡细小的星座，没有亮于3等的恒星。中心位置：赤经8时10分，赤纬20度。

== 特征 ==

巨蟹座位于[[双子座|双子座]]和[[狮子座|狮子座]]之间、北方是[[天猫座|天猫座]]、南面则是[[小犬座|小犬座]]和[[长蛇座|长蛇座]]，是一個暗淡細小的星座，没有亮于3等的恒星。巨蟹座较亮的3颗恒星α、β、δ组成一个“人”字形结构。

这个星座有两个[[梅西耶天体|梅西耶天体]]：[[M44|M44]]位于[[鬼宿四|鬼宿四]]（巨蟹座δ）附近；[[M67|M67]]位于位於[[柳宿增三|柳宿增三]]（巨蟹座α）以西。两者均为[[疏散星团|疏散星团]]。

== 星座神話 ==

[[File:Cancer_Hevelius.jpg|left]]

當[[宙斯|宙斯]]的私生子[[海克力士|海克力士]]迎戰[[勒拿湖|勒拿湖]]的九頭大蛇（[[長蛇座|長蛇座]]）時，海克力士的死敵[[希拉|希拉]]派出巨蟹從沼澤竄出並偷偷攻擊海克力士的腳，海克力士一怒之下將其踩成碎片，事後希拉撿起巨蟹，升上天空，成為巨蟹座。

另外γ星及δ星的意思為驢子，原來是當[[克羅諾斯|克羅諾斯]]等[[泰坦|泰坦]]巨人迎戰[[宙斯|宙斯]]時，酒神[[戴歐尼修斯|戴歐尼修斯]]、火神[[赫菲斯托斯|赫菲斯托斯]]和騎驢者加入戰團，由於泰坦巨人未曾聽過驢嘶，他們以為是怪物的叫聲而溜之大吉，於是戴歐尼修斯將驢子升上天空，以示紀念。

== 恒星 ==
* [[柳宿增十|柳宿增十]]（巨蟹座β），[[视星等|视星等]]3.52等，巨蟹座最亮的恒星，距离190[[光年|光年]]。
* {{tsl|en|Zeta Cancri|水位四}}（巨蟹座ζ），四合星，子星A的视星等是5.6，子星B的视星等是6.0，子星C的视星等为6.2，而子星D则为9.7。
* {{tsl|en|R Cancri|巨蟹座R}}，[[米拉变星|米拉变星]]，变光周期为361.6日。
* [[巨蟹座55|巨蟹座55]]，[[双星|双星]]，巨蟹座55A是与[[太阳|太阳]]差不多的[[黄矮星|黄矮星]]，55B则是一颗[[红矮星|红矮星]]，至今已发现5个环绕着55A的[[太阳系外行星|太阳系外行星]]。

== 深空天體 ==
[[File:Messier_object_067.jpg|thumb]]
* [[M44|M44]]（[[鬼星團|鬼星團]]，或稱蜂巢星團），位於巨蟹座的正中的[[疏散星团|疏散星团]]，肉眼看起來是一團模糊的白色霧氣，中國古代把它叫做“積尸氣”。亮度为3.1等，距离为520光年。
* [[M67_(疏散星团)|M67]]，位於[[柳宿增三|柳宿增三]]（α）以西大而暗的[[疏散星團|疏散星團]]。它是[[银河系|银河系]]最古老的星團之一，年齡超過十億年。距离2700光年。
* [[NGC_2775|NGC 2775]]，[[漩涡星系|漩涡星系]]，亮度为10.3等。
* [[NGC_2535|NGC 2535]]和[[NGC_2536|NGC 2536]]，两个正在发生[[交互作用星系|交互作用]]的星系。
