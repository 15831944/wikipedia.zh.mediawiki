'''巨狐猴'''（''Megaladapis''）是[[馬達加斯加|馬達加斯加]]已[[滅絕|滅絕]]的一[[屬|屬]][[靈長目|靈長目]]。

== 身體特徵 ==
巨狐猴的近親是[[鼬狐猴|鼬狐猴]]，並與另外兩個屬組成了[[鼬狐猴科|鼬狐猴科]]。但是巨狐猴與其他的[[狐猴|狐猴]]有很大分別。牠的身體矮胖，像現今的[[無尾熊|無尾熊]]。牠的手臂及手指很長，適合抓樹，腳則呈八字形適合直立攀爬。

巨狐猴的[[頭部|頭部]]不像其他的靈長目，[[頭顱骨|頭顱骨]]在靈長目中很獨特，鼻的部份像[[黑犀牛|黑犀牛]]，且可能有特大的上唇。牠的[[犬齒|犬齒]]很長，[[顎骨|顎骨]]像[[牛|牛]]的，形成很長的鼻端。牠的顎肌很強壯，可以咬碎粗糙的[[植物|植物]]。牠們約重50公斤。

== 滅絕 ==
巨狐猴的攀樹能力使牠不能適應[[馬達加斯加|馬達加斯加]][[森林|森林]]的轉變。於1500-2000年前[[人類|人類]]到達後，馬達加斯加的森林被剷平成為農地。巨狐猴未能適應這種環境轉變，且成為了獵人的目標，最終於500年前[[歐洲|歐洲]]發現馬達加斯加時滅絕。

== 神秘生物學 ==
一般相信[[馬達加斯加|馬達加斯加]]的一種傳說[[生物|生物]]「tretretretre」，可能就是巨狐猴。但是這種傳說生物卻有著像[[人類|人類]]的面，似乎更像[[古原狐猴|古原狐猴]]。<ref>{{cite web |url=http://www.zoobooks.com/rainforest/extinct.html |title=存档副本 |accessdate=2008-03-04 |0=2007-10-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20071031223128/http://www.zoobooks.com/rainforest/extinct.html |archivedate=2007-10-31 }}</ref>另外，[[霍伊維爾曼|霍伊維爾曼]]（Bernard Heuvelmans）認為巨狐猴其實是另一種傳說生物「Tokandia」的參考對象。<ref>{{Cite web|title=Tokandia|url=http://www.kryptozoologie-online.de/Forum/viewtopic.php?t=289&view=previous&sid=9d5ef3913439a8db9f5f8e7d8cfc7ec0|accessdate=2014-02-16|language=en|publisher=Kryptozoologie-Online}}</ref>一些如在地上或樹上的跳躍動作，不像人的面但像人的叫聲等，都顯示這個可能性。
