'''巨爵座'''（{{lang-la|Crater}}，意思是[[杯子|杯子]]）是现代88[[星座|星座]]和[[克劳狄乌斯·托勒密|托勒密]]划分的48星座之一。不包含亮于4等的星。包含中国古代星座[[翼宿|翼宿]]。

== 神话传说 ==
在[[希腊神话|希腊神话]]中，太阳神[[阿波罗|阿波罗]]让一只乌鸦为他取水。乌鸦在路上懒惰的拖延，且在最终取得水后，带回一条水蛇作为一个借口。这个骗局被阿波罗看穿，把乌鸦、水杯和蛇一起扔到了天上。巨爵座、[[乌鸦座|乌鸦座]]和[[长蛇座|长蛇座]]甚为接近。

== 特殊星體 ==
[[巨爵座γ|巨爵座γ]]是一对临近双星，距太阳84[[光年|光年]]。

[[巨爵座R|巨爵座R]]是半规则SRb型[[变星|变星]]，周期160天，星等9.8-11.2，光谱分类M7。

[[巨爵座SZ|巨爵座SZ]]是8.1等的变星，距太阳44光年，又称Gliese 425。

[[BD-10°3166|BD-10°3166]]有一个行星。

== [[深空天体|深空天体]] ==
* [[NGC_3511|NGC 3511]]是SBbc型螺旋[[星系|星系]]，带一个不明显的棒。呈侧面观，12等，大小为4'×1'。
* [[NGC_3887|NGC 3887]]是SBc型棒旋星系, 11等，直径3.5'。
* [[NGC_3981|NGC 3981]]是SBbc型螺旋星系，带两条旋臂，星等为12等，直径3'；于1785年为[[約翰·赫歇爾|約翰·赫歇爾]]所发现。
