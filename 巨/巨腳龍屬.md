'''巨腳龍屬'''（[[學名|學名]]：''Barapasaurus''），又名'''巴拉帕龍'''或'''巨腿龍'''，是[[火山齒龍科|火山齒龍科]]下的一個[[屬|屬]]。屬名是由[[孟加拉語|孟加拉語]]的「巨腿」與[[古希臘語|古希臘語]]的「蜥蜴」而來，因牠的大型腿骨。種名則是以[[印度|印度]]詩人、哲學家[[羅賓德拉納特·泰戈爾|羅賓德拉納特·泰戈爾]]為名。

== 描述 ==
巨腳龍是已知最早的[[蜥腳下目|蜥腳下目]][[恐龍|恐龍]]之一，估計生活於[[侏羅紀|早侏羅紀]]的[[托阿爾階|托阿爾階]]，距今約1億8960萬到1億7650萬年前。這一點可以從牠的不獨特形態中得到引證。後期的蜥腳下目如[[腕龍|腕龍]]，已發展出牠們獨有的[[生態位|生態位]]及攝食策略。例如，後期的蜥腳下目已發展出空心的[[脊椎|脊椎]]，作為減輕體重的方法。而巨腳龍脊椎卻是幾乎實心的，只有少許空心的早期跡象。

巨腳龍長約18米，體重約48[[公噸|公噸]]。牠的[[臀部|臀部]]高約5.5米。

== 生活環境 ==
如同其他[[蜥腳下目|蜥腳下目]]恐龍，巨腳龍是[[草食性|草食性]]動物。但是，到目前為止都沒有發現[[頭顱骨|頭顱骨]]，所以不能確定牠的實際食性。目前已發現一些個別的[[牙齒|牙齒]]，但都不足以作出確定。

雖然巨腳龍的[[化石|化石]]是於[[印度|印度]]被發現，但卻很像其他在[[東非|東非]]發現的標本。由此估計，在[[侏羅紀|早侏羅紀]]時，這兩個土地應該是連接在一起，或是不久後分離的。

== 分類 ==
巨腳龍被分類在[[火山齒龍科|火山齒龍科]]下，而非[[鯨龍科|鯨龍科]]。這個分類並不確定，因為未有足夠的骨骼研究。而將牠分類在此科的原因，是牠擁有火山齒龍科特徵的狹窄[[薦骨|薦骨]]。[[背椎|背椎]]的[[神經弓|神經弓]]高於[[椎孔|椎孔]]，是種[[鯨龍科|鯨龍科]]的特徵。

目前已知的唯一[[種|種]]是泰氏巨腳龍（''B. tagorsi''）。

== 發現 ==
巨腳龍的骨頭首先於1960年在[[印度|印度]]被發現。但是要到1975年才作出詳細描述，並命名了[[模式種|模式種]]。自此，有五個骨骼標本在印度南部的[[戈達瓦里河|戈達瓦里河]]河谷被發現，但是沒有發現任何[[頭顱骨|頭顱骨]]或腳掌，使得巨腳龍成為了解最多的[[侏羅紀|侏羅紀]][[蜥腳下目|蜥腳下目]]，不過只有少數的文獻記述有關巨腳龍的資料。
