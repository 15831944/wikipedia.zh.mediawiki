巨蜂鳥（[[學名|學名]]：{{lang|la|'''''Patagona gigas'''''}}）是[[蜂鸟科|蜂鳥科]]'''巨蜂鳥屬'''的唯一一種。

== 特徵 ==
體型最大的蜂鳥，體重可達24公克，翼展長可達23公分，以蜂鳥來說算是非常大的。巨蜂鳥的體色成有光澤的銅色，背部則偏灰綠色。有長而分叉的尾巴，有白色的喉部。
[[File:Patagona_gigas.jpg|缩略图]]

== 分布 ==
此鳥分布於[[安地斯山脈|安地斯山脈]]。

== 習性 ==
本鳥以花蜜為主食，繁殖期會食用砂土以補充產卵時消耗的[[鈣質|鈣質]]。本鳥也會進行小規模遷徙。
