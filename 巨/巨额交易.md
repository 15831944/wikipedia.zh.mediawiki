《'''巨额交易'''》（英文名：'''''A Big Deal'''''）2011年12月2日上映于[[中国大陆|中国大陆]]的喜剧片。

== 剧情 ==
该片讲述了三个男士去[[迪拜|迪拜]]淘金而遭遇到的一系列疯狂冒险而搞笑的故事。<ref name="web100 " >{{cite web |url=http://ent.163.com/11/1027/08/7HC0UAUB000300B1.html|title=《巨额交易》贺岁上映 主演乔任梁变“三陪” |accessdate=2011-10-27|work=|publisher=网易|language=zh-hans}}</ref>

== 花絮 ==
该片于2011年贺岁档播出，导演[[马俪文|马俪文]]说出了“20亿美金，买观众一笑”的宣言。<ref name="web100 " />该片触碰了社会当前的现实“女人物质男人悲催”，反映了当下一代“究竟是爱情重要还是金钱重要”的困惑。<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2011-11-30/00073494992.shtml|title=《巨额交易》曝男女对抗特辑聚焦婚恋与金钱 |accessdate=2011年11月30日|work=|publisher=新浪网|language= zh-hans}}</ref>
