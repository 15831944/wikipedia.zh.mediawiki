《'''為時已晚的警告'''》（{{jpn|j=手遅れcaution|hg=ておくれコーション|rm=Teokure caution}}）是日本女子[[日本偶像|偶像]]組合[[=LOVE|=LOVE]]的第3張[[單曲|單曲]]，於2018年5月16日由[[SACRA_MUSIC|SACRA MUSIC]]發行。單曲中心成員是[[高松瞳|高松瞳]]<ref name="イコラブ、女子校舞台に“イケナイ恋”絡み合う 指原P絶賛「何度見ても楽しめる」"/>。

== 概要==

單曲分為Type-A、Type-B和Type-C（通常版），總共3個版本。其中，《為時已晚的警告》收錄於所有版本，其音樂錄影帶、製作特輯和《「我就知道在課外活動中能夠與你對上眼」》收錄於Type-A；《樹愛羅，我們來幫你了》和「Documentary of =LOVE』總集編」收錄於Type-B；《「我就知道在課外活動中能夠與你對上眼」》和《樹愛羅，我們來幫你了》均收錄於Type-C。Type-C以外所有版本均有全國[[握手會|握手會]]參加券和隨機[[生寫真|生寫真]]一張<ref>{{cite web|url=http://www.sonymusic.co.jp/artist/equal-love/discography/VVCL-1237|language=ja|script-title=ja:手遅れcaution【初回仕様限定盤 Type-A】|publisher=[[=LOVE|=LOVE]]|accessdate=2018-05-28|deadurl=no|archiveurl=https://web.archive.org/web/20180528092127/http://www.sonymusic.co.jp/artist/equal-love/discography/VVCL-1237|archivedate=2018-05-28}}</ref><ref>{{cite web |url=http://www.sonymusic.co.jp/artist/equal-love/discography/VVCL-1239 |language=ja |script-title=ja:手遅れcaution【初回仕様限定盤 Type-B】 |publisher==LOVE |accessdate=2018-05-28 |deadurl=no |archiveurl=https://web.archive.org/web/20180528092129/http://www.sonymusic.co.jp/artist/equal-love/discography/VVCL-1239 |archivedate=2018-05-28 }}</ref><ref>{{cite web |url=http://www.sonymusic.co.jp/artist/equal-love/discography/VVCL-1241 |language=ja |script-title=ja:手遅れcaution |publisher==LOVE |accessdate=2018-05-28 |deadurl=no |archiveurl=https://web.archive.org/web/20180528092129/http://www.sonymusic.co.jp/artist/equal-love/discography/VVCL-1241 |archivedate=2018-05-28 }}</ref>。

== 反應 ==
在《[[告示牌_(雜誌)|告示牌]]》Top Singles Sales榜中，單曲首週銷量為73,319張，排第3位<ref>{{cite web |url=http://www.billboard-japan.com/d_news/detail/63574/2 |language=ja |script-title=ja:【ビルボード】TWICE『Wake Me Up』が299,195枚を売り上げてシングル・セールス首位 |date=2018-05-21 |publisher=[[告示牌_(雜誌)|告示牌]] |accessdate=2018-05-28 |deadurl=no |archiveurl=https://web.archive.org/web/20180523172605/http://www.billboard-japan.com/d_news/detail/63574/2 |archivedate=2018-05-23 }}</ref>，而在[[日本熱門100金曲榜|日本熱門100金曲榜]]也是位列第3位<ref>{{cite web|url=http://www.billboard-japan.com/d_news/detail/63629/2|language=ja|script-title=ja:【ビルボード】TWICE「Wake Me Up」が29.9万枚を売り上げ総合首位　BTS (防弾少年団)が4曲、西城秀樹さんが3曲チャートイン |date=2018-05-23 |publisher=告示牌 |accessdate=2018-05-28}}</ref>。在[[Oricon公信榜|Oricon公信榜]]中，單曲首週銷量是62,930張，排第3位<ref>{{cite web |url=https://www.oricon.co.jp/rank/js/w/2018-05-28/ |language=ja |script-title=ja:週間　CDシングルランキング2018年05月28日付 |publisher=Oricon |accessdate=2018-05-28 |deadurl=no |archiveurl=https://web.archive.org/web/20180523095357/https://www.oricon.co.jp/rank/js/w/2018-05-28/ |archivedate=2018年5月23日 }}</ref>，銷量也超越了出道作《[[=LOVE_(单曲)|=LOVE]]》（18,746）和前作《[[我們的制服聖誕節|我們的制服聖誕節]]》（40,255）<ref>{{cite web|url=http://realsound.jp/2018/05/post-198895.html |language=ja|script-title=ja:＝LOVE『手遅れcaution』が好調の理由　新境地で掴んだグループの変化を読む |date=2018-05-28 |publisher=Real Sound |accessdate=2018-05-28}}</ref>。
