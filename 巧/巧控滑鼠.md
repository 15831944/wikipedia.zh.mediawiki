'''Magic Mouse'''是由[[蘋果公司|蘋果公司]]出品和發售的[[多點觸控|多點觸控]]（Multi-touch）[[滑鼠|滑鼠]]<ref name = "Magic Mouse" >{{cite web
    | url = http://www.apple.com/magicmouse/
    | title = Magic Mouse
    | publisher=Apple
    | accessdate = {{start date|2009|12|01}}}}</ref>，它在2009年10月20日宣佈和發售<ref name = "Engadget-announcement" >{{cite web
    | url = http://www.engadget.com/2009/10/20/apples-magic-mouse-multitouch-gestures-bluetooth-four-month/
    | title = Apple's Magic Mouse: one button, multitouch gestures, Bluetooth, four-month battery life
    | last = Topolsky
    | first = Joshua
    | date = {{start date|2009|10|20}}
    | publisher=Engadget
    | accessdate = {{start date|2009|10|25}}}}</ref>。Magic Mouse是世界上首個引入多點觸控技術的滑鼠<ref name = "Magic Mouse" />。繼[[iPhone|iPhone]]、[[iPad|iPad]]、[[iPod_Touch|iPod Touch]]和觸控板後，Magic Mouse容許用户使用不同的[[手勢識別|手勢]]，例如在滑鼠表面上輕掃和滾動，便可達到特定的效果。Magic Mouse經[[藍牙|藍牙]]技術連接，而且以[[乾電池|AA電池]]驱動。

Magic Mouse與[[Mac_OS_X_v10.5|Mac OS X 10.5.8]]及更新系统相容。它可以被設定為雙鍵、適合左撇子或右撇子的滑鼠，儘管它的預設設定是單鍵滑鼠。它使用[[鼠标#技術|鐳射追踪]]，比早期蘋果滑鼠更有準確性<ref name = "Apple store" >{{cite web
    | url = http://store.apple.com/us/product/MB829LL/A
    | title = Magic Mouse
    | publisher=Apple
    | accessdate = {{start date|2009|10|25}}}}</ref>。自它發行以來，它已經與蘋果無線鍵盤一起包括在2009年的[[iMac|iMac]]和與蘋果有線鍵盤一起包括在2010年的[[Mac_Pro|Mac Pro]]工作台中。而Magic Mouse也能單獨購買。

Magic Mouse推出初期的反應不一，正面評論有捲動功能<ref name = "Gizmodo-review" >{{cite web
    | url = http://gizmodo.com/5386202/apple-magic-mouse-review
    | title = Apple Magic Mouse Review
    | last = Chen
    | first = Jason
    | date = {{start date|2009|10|21}}
    | publisher=Gizmodo
    | accessdate = {{start date|2009|10|25}}}}</ref>，而負評則有它沒有中鍵（如果沒有額外的軟體），或沒有實體鍵來開啟[[Exposé_(Mac_OS_X)|Exposé]]、[[Dashboard_(Mac_OS)|Dashboard]]或[[Spaces_(software)|Spaces]]（而[[Apple_Mighty_Mouse|上一代]]卻有）<ref name = "Macworld-review" >{{cite web
    | url = http://www.macworld.com/article/143410/2009/10/magic_mouse.html

    | title = Add more gestures to Magic Mouse
