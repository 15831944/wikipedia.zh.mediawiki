'''士美非路體育館''' ([[英文|英文]]：'''Smithfield Sports Centre''') 是[[香港|香港]]的一間室內[[體育|體育]]中心，位於[[香港島|香港島]][[堅尼地城|堅尼地城]][[士美非路|士美非路]]12號K，[[士美非路市政大廈|士美非路市政大廈]]4至7樓。

== 開放時間 ==
全年開放，每日07:00-23:00，每月兩次的保養日除外。

== 設備 ==
* 多功能主場(可換作2個[[籃球|籃球]]場、2個[[排球|排球]]場，或者8個[[羽毛球|羽毛球]]場)
* 觀眾席看台可容438人
* [[士碌架|美式桌球]]室2個
* [[壁球|壁球]]場3個 (也可換作乒乓球室，或者小活動室)
* 兒童遊戲室
* [[康文署健身室|健身室]]
* [[舞蹈|舞蹈]]室
* 乒乓球室2間，可容7張乒乓球桌
* [[射箭|射箭]]室2間
