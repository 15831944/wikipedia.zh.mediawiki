'''常州市第一中學'''，簡稱'''常州一中'''，在[[中國|中國]][[江蘇省|江蘇省]][[常州市|常州市]][[天宁区|天宁区]]和平路边，新坊桥下的巷内，红梅公园和天宁寺附近，是国家级示范高中、江苏省首批四星级高中。<ref>{{cite web | title=常州市第一中学简介 | accessdate=2013-03-14 | url=http://www.czyz.com.cn/www/html/xiaoyuangaikuang/200810/10-98.html }}</ref> 常州市第一中学在2016年中国内地高中美国留学排行榜上位居第二十九。<ref>人民网 "http://gz.people.com.cn/n2/2016/0224/c358162-27802234.html" 2016-02-24 </ref> <ref> 联校网排名 "{{cite web |url=http://www.collegenode.com/list.php |title=存档副本 |accessdate=2016-04-05 |0=2016-10-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20161014013045/http://www.collegenode.com/list.php |archivedate=2016-10-14 }}" </ref>

初中部改名为正衡初中，现已为私立学校，并已搬至新校址。2018年9月起由于原地改造工程，将过渡校区设置在常州开放大学（原常州市田家炳中学）。<ref>{{cite web | title=常州市正衡中学简介 | accessdate=2013-03-14 | quote=2008年，由杨褔宽先生全额投资举办正衡中学，与常州市第一中学合作办学，实行管、办分离的运行模式，并迁至常州市清凉东路48号。 | url=http://www.czzhzx.com.cn/a/zhannadaohang/xuexiaojianjie/2008/1014/87.html }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==校友==
*李瑞麟：43届校友。1996年先后被评为上海市科技功臣、国家计生委科技功臣，同年被选为中国工程院院士。<ref>{{cite web | title=李瑞麟院士 | accessdate=2013-03-14 | url=http://www.czyz.com.cn/www/html/xiaoyoufengcai/200810/09-74.html}}</ref>
*[[顾冠群|顾冠群]]：56级校友，中国工程院院士，曾任东南大学校长，计算机网络专家。<ref>{{cite web | title=顾冠群院士 | accessdate=2013-03-14 | url=http://www.czyz.com.cn/www/html/xiaoyoufengcai/200810/09-73.html}}</ref>
*[[陶宝祺|陶宝祺]]：52届校友，中国科学院院士，南京航空航天大学教授，博士生导师，智能材料与结构航空科技重点实验室主任。<ref>{{cite web | title=陶宝祺院士 | accessdate=2013-03-14 | url=http://www.czyz.com.cn/www/html/xiaoyoufengcai/200810/09-72.html}}</ref>
*[[庄逢辰|庄逢辰]]：46届正衡初中校友，国防科技大学教授，博士生导师，中国科学院院士。<ref>{{cite web | title=庄逢辰院士 | accessdate=2013-03-14 | url=http://www.czyz.com.cn/www/html/xiaoyoufengcai/200810/09-71.html}}</ref>
*[[梅自强|梅自强]]：47届校友，纺织科学研究院教授级高级工程师，中国工程院院士，第八、九届全国政协委员。<ref>{{cite web | title=梅自强院士 | accessdate=2013-03-14 | url=http://www.czyz.com.cn/www/html/xiaoyoufengcai/200810/09-70.html}}</ref>
*[[蒋亦元|蒋亦元]]：46届校友，中国工程院院士，东北农业大学教授，博士生导师。<ref>{{cite web | title=蒋亦元院士 | accessdate=2013-03-14 | url=http://www.czyz.com.cn/www/html/xiaoyoufengcai/200810/09-69.html}}</ref>
*[[陆孝彭|陆孝彭]]：34届初中校友，飞机设计师，中国工程院院士。<ref>{{cite web | title=陆孝彭院士 | accessdate=2013-03-14 | url=http://www.czyz.com.cn/www/html/xiaoyoufengcai/200810/09-68.html}}</ref>
