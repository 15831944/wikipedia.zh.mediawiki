'''常德公寓'''（{{Lang-en|'''Changde Apartment'''}}），原称'''爱林登公寓'''（{{Lang-en|'''Eddington House'''}}），位于[[上海|上海]][[静安区|静安区]][[常德路_(上海)|常德路]]195号，靠近[[南京西路_(上海)|南京西路]]与[[愚园路|愚园路]]。公寓在[[民国25年|民国25年]]建于[[赫德路|赫德路]][[静安寺路|静安寺路]]，共8层，为装饰艺术派风格<ref name="dd">[http://www.shtong.gov.cn/newsite/node2/node71994/node81772/node81774/node81784/userobject1ai108983.html 特色志 >> 上海名建筑志 >> 欧美风情的沪滨广厦 >> 灿烂与孤寂的女作家张爱玲住过的爱林登公寓（常德公寓）] 上海地方志办公室</ref>。其原来是[[意大利人|意大利人]]拉乌尔·斐斯的房产，居住者旧时多是社会中上层人士<ref name="dd"/>。1994年，常德公寓入选第二批[[上海市优秀历史建筑|上海市优秀历史建筑]]。

== 张爱玲故居 ==
[[张爱玲|张爱玲]]曾于1942至1947年居住在这座公寓5层的65室，并在此期间发表了《[[倾城之恋|倾城之恋]]》、《[[金锁记|金锁记]]》等作品。在《公寓生活记趣》一文中，[[张爱玲|张爱玲]]如是描绘她的公寓生活：<ref name=s>{{Cite book |author= 潘玲 |year= 2008 |title= 《Shanghai Style:Art and Design Between the Wars》 |publisher=  Long River Press |isbn= 9781592650781 |page =217-218}}</ref> {{cquote|我喜歡听市聲。比我較有詩意的人在枕上听松濤，听海嘯，我是非得听見電車響才睡得著覺的。……公寓是最合理想的逃世的地方。}}

作家[[陈丹燕|陈丹燕]]在《[[上海的风花雪月|上海的风花雪月]]》一书中就称这栋公寓“被刷成了女人定妆粉的那种肉色，竖立在上海闹市中的不蓝的晴天下面”。

== 建筑特色 ==
常德公寓是根据地形设计的，共8层，[[钢筋混凝土|钢筋混凝土]]结构，占地面积580平方米，建筑面积2663平方米，设有电梯，为装饰艺术派风格<ref name="dd"/><ref name=s></ref>。整个建筑配有细腻的局部装饰，平面呈凹形，两翼后收。大楼东立面两侧出挑长条形阳台，中部竖线条形成横竖对比。顶部2层退台收进。其建筑利用阳台的处理所形成的强烈阴影效果，使建筑产生丰富的变化<ref name="cs">[http://www.shtong.gov.cn/newsite/node2/node4/node2249/node4412/node17436/node18417/node62251/userobject1ai6802.html 区县志 >> 区志 >> 静安区志 >> 第八编房屋建筑 >> 第三章旧有住宅 >> 第二节　公寓] {{webarchive|url=https://web.archive.org/web/20131213164126/http://www.shtong.gov.cn/newsite/node2/node4/node2249/node4412/node17436/node18417/node62251/userobject1ai6802.html |date=2013-12-13 }} 上海地方志办公室</ref>。旧时常德公寓的内部，每层3户，户型有2室户和3室户。每户都安装有壁炉。西面有长挑长廊，既作为安全通道，又兼作阳台。每套住宅上下有小楼梯连通。第八层是电梯机房和水箱等用房。
