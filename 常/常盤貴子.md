'''常盘贵子'''（{{bd|1972年|4月30日|CatIdx=Tokiwa, Takako}}），[[日本|日本]][[女演员|女演员]]，出生于[[神奈川县|神奈川县]][[橫濱市|橫濱市]]，<!--[[身高|身高]]162[[公分|公分]]，[[血型|血型]]A型，-->隸屬於[[Stardust_Promotion|Stardust Promotion]]。其出演的[[電視剧|電視剧]]广受好评，在[[1990年代|1990年代]]被日本及華語媒体称为「日剧女王」。

== 生涯經歷 ==
1991年夏天，常盤進入目前所屬的[[Stardust_Promotion|Stardust Promotion]]。當正職演員之前，曾在[[原宿|原宿]]的[[徒步區|徒步區]]擔任街頭表演的主持人。1991年，在[[TBS電視台|TBS電視台]]的單集電視劇《イブは初恋のように》之中擔任配角，為個人出道作。同一時期，常盤在經紀公司亦屬於「儲備歌手」之一。

1993年，當時仍籍籍無名的常盤被拔擢擔任夏季連續劇《{{link-ja|惡魔之吻|悪魔のKISS}}》的重要角色，飾演陷入卡債地獄而從事性工作的女大學生，在劇中的大膽詮釋和演技初次受到電視圈注目而造成話題。1995年，憑夏季連續劇《[[跟我說愛我|跟我說愛我]]》獲得[[第6回日劇學院賞|第6回日劇學院賞]]最佳女主角獎。其後，又分別以《{{link-ja|白晝之月|真昼の月}}》、《{{link-ja|理想的結婚|理想の結婚 (テレビドラマ)|理想的結婚}}》、《{{link-ja|最後之戀|最後の恋}}》和《[[美麗人生_(日劇)|美麗人生]]》獲得[[第10回日劇學院賞|第10回]]、[[第12回日劇學院賞|第12回]]、[[第14回日劇學院賞|第14回]]及[[第24回日劇學院賞|第24回日劇學院賞]]最佳女主角，加上一次女配角獎（《{{link-ja|美人 (電視劇)|美しい人 (テレビドラマ)|美人}}》），合計共6次獲獎，被媒體封為「日劇女王」。其中，以《[[美麗人生_(日劇)|美麗人生]]》一劇攀上事業頂峰，創下20年來連續劇大結局最高收視率（41.3%），七年間共主演至少十部劇集，奠定一線女演員地位。

1999年起，在[[香港|香港]]先後與[[張國榮|張國榮]]（《[[星月童話|星月童話]]》）和[[劉德華|劉德華]]（《[[阿虎|阿虎]]》）等其他亞洲地區電影人合作，為跨出日本的代表性女演員之一。
    
2006年之後重心轉向電影圈發展，與此同時，戲路也從談情說愛的妙齡女郎「兩級跳」；先是向「御姐」演變（《[[叫她第一名|叫她第一名]]》、《[[漂流教室|漂流教室]]》、《{{link-ja|戀愛偏差值|恋愛偏差値}}》），分別在片中與新派小生[[妻夫木聰|妻夫木聰]]、[[窪塚洋介|窪塚洋介]]，以及[[SMAP|SMAP]]的[[稻垣吾郎|稻垣吾郎]]有微妙的姊弟戀；進軍影壇後，她已接拍3部飾演「人妻」的電影（《{{link-ja|Get up (電影)|ゲロッパ!|Get up}}》、《{{link-ja|赤月|赤い月}}》、《[[少年馴象師|少年馴象師]]》），除愛情戲外，還須處理親子關係的演繹。

2006年有三部電影公映，包括《[[間宮兄弟|間宮兄弟]]》、《[[Brave_Story|Brave Story]]》、《[[地下鐵_(電影)|地下鐵]]》。除了電影與電視，常盤在2005年春季首次演出舞台劇，其後便不斷持續舞台表演，至今維持每年至少一部作品的產量。

2009年，出演[[NHK大河劇|NHK大河劇]]《[[天地人|天地人]]》，飾演[[妻夫木聰|妻夫木聰]]主演的[[直江兼續|直江兼續]]的妻子[[阿船|阿船]]。

== 表演風格 ==
常盤在電視劇經常扮演弱勢族群的角色。例如[[性工作者|性工作者]]（《{{link-ja|惡魔之吻|悪魔のKISS}}》、《{{link-ja|最後之戀|最後の恋}}》）、被[[性侵|性侵]]者（《{{link-ja|白晝之月|真昼の月}}》）、受虐妻子（《{{link-ja|美人 (電視劇)|美しい人 (テレビドラマ)|美人}}》）、身障人士（《[[美麗人生_(日劇)|美麗人生]]》）等，其自卑感之中帶著倔強的戲劇張力頗獲觀眾及電視評論家好評。

== 人物 ==
=== 家庭背景 ===
常盤貴子出生於[[橫濱|橫濱]]，因父親工作之故，小學四年級起搬遷至[[兵庫縣|兵庫縣]][[西宮市|西宮市]]，在當地生活至高中一年級，其後在關東就讀高中，畢業於[[關東學院女子短期大學|關東學院女子短期大學]]家政科。但自稱[[近畿地方|關西]]人，平常說話亦持[[關西腔|關西腔]]。

常盤和家人關係親密，除了父母，亦有一個大三歲從事護理相關工作的姐姐<ref>常盤本人在TBS《はなまるマーケット》節目中的訪談內容</ref>。祖父常盤稔是前日本{{link-ja|清水銀行|清水銀行}}總經理。常盤家的一部分住屋被歸劃為[[靜岡縣|靜岡縣]]文化遺產之一（[[紀伊德川家|紀伊德川家]]古蹟）。

2009年10月20日，常盤與知名舞臺劇導演{{link-ja|長塚圭史|長塚圭史}}提出結婚申請。兩人因拍攝電影《{{link-ja|Get up (電影)|ゲロッパ!|Get up}}》而認識，交往六年半後結婚。

=== 性格交友 ===
自小個性內向沉靜，搬遷至兵庫縣西宮生活近七年之後，性格才開始有了180度的轉變。與[[天海祐希|天海祐希]]、[[深津繪里|深津繪里]]交情好。圈中亦有許多演員曾表示是常盤的影迷，如[[內山理名|內山理名]]、[[鈴木惠美|鈴木惠美]]、[[真鍋香織|真鍋香織]]等人。
