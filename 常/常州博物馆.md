'''常州博物馆'''，位于[[江苏省|江苏省]][[常州市|常州市]][[新北区|新北区]]龙城大道1288号，是常州市属综合性博物馆<ref name=jj>{{Cite web |url =http://www.czmuseum.com/default.php?mod=article&do=detail&tid=1  |title = 常州博物馆简介 |author =  |date =  |publisher = 常州博物馆 |language =  |accessdate = 2017-07-05 |quote =  }}</ref>。

==简介==
常州博物馆创建于1958年10月。曾先后以[[红梅公园|红梅阁]]、[[清凉寺_(常州)|清凉寺]]作为馆址。2005年3月23日，常州博物馆、规划馆工程立项，同年5月26日开工建设，2007年4月28日正式对外开放。常州博物馆新馆位于龙城大道1288号，地处市民广场西侧，与常州市人民政府、[[常州大剧院|常州大剧院]]、[[常州奥林匹克体育中心|常州奥林匹克体育中心]]毗邻。2008年4月1日起，常州博物馆免费对外开放<ref name=yiji>{{Cite web |url =http://www.changzhou.gov.cn/ns_news/139148504664304  |title = 常州博物馆获评“国家一级博物馆” 全国百强 |author =  |date = 2017-01-22 |publisher =  中国常州|language =  |accessdate =  |quote =  }}</ref><ref name=jj/>。

常州博物馆建筑面积2.8万平方米，展区面积将近1万平方米，共设有10个展厅。分为两个基本陈列（“龙腾中吴——常州历史文化陈列”，“神奇的自然美丽的家园——自然资源陈列”）、两个专题陈列（“[[谢稚柳|谢稚柳]]艺术陈列”，“刘国钧先生捐献红木家具陈列”）、三个临时展厅<ref name=jj/>。

常州博物馆馆藏文物3万多件，其中国家一级文物51件（国宝级文物1件）、国家二级文物245件、国家三级文物2945件。包括[[良渚文化|良渚文化]]玉器、[[春秋时期|春秋]][[战国|战国]]原始青瓷器、[[宋朝|宋]][[元朝|元]]漆器及瓷器、[[明朝|明]][[清朝|清]]书画。常州博物馆内还设有江苏省内唯一的少儿自然博物馆，内藏自然标本8148件，分为海洋动物、皮毛类动物、昆虫、中草药、矿物晶体、古生物化石，其中[[圣贤孔子鸟|圣贤孔子鸟]]化石、[[金斑喙凤蝶|金斑喙凤蝶]]、[[翁戎螺|翁戎螺]]等化石和生物标本十分珍贵<ref name=jj/>。

2017年，常州博物馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref><ref name=yiji/>。
