'''常捷'''（{{bd|1929年|2月|2012年|7月29日|catIdx=}}），[[辽宁省|辽宁省]][[辽阳县|辽阳县]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中华人民共和国国务院副秘书长|国务院副秘书长]]。

==生平==
1929年2月常捷生于辽宁省辽阳县。[[中正大学|中正大学]]肄业。1948年12月参加革命工作，到[[华北大学|华北大学]]学习并加入[[中国新民主主义青年团|中国新民主主义青年团]]，1949年10月加入中国共产党。历任[[中共中央统战部|中共中央统战部]][[中南海|中南海]]办事处、交际处干事，[[中央人民政府政务院|中央人民政府政务院]]交际处干事。1950年12月到[[国家机关事务管理局|政务院机关事务管理局]]（先后改为[[国家机关事务管理局|中央人民政府机关事务管理局]]、[[国家机关事务管理局|国务院机关事务管理局]]）工作，历任交际处干事，人事处科员，党委办公室秘书、宣教科科长、组织科科长，交际处秘书、科长，办公室秘书科长、副主任、代主任。1959年受到错误处理。后予平反。1983年6月后，任[[国务院机关事务管理局|国务院机关事务管理局]]局长、党组书记。1989年7月，担任[[中华人民共和国国务院副秘书长|国务院副秘书长]]、机关党组成员兼国务院机关事务管理局局长、党组书记。1990年9月任[[国务院参事室|国务院参事室]]主任、党组书记。1998年12月离休。他是[[中共十三大|中共十三大]]代表，[[第八届全国政协委员|第八届全国政协委员]]<ref>{{cite news|title=常捷|url=http://www.counsellor.gov.cn/Item/4093.aspx|accessdate=2012-06-10|newspaper=国务院参事室|deadurl=yes|archiveurl=https://web.archive.org/web/20100711121541/http://www.counsellor.gov.cn/Item/4093.aspx|archivedate=2010-07-11}}</ref><ref name=shishi>{{Cite web |url = http://news.sina.com.cn/c/2012-08-21/033925003622.shtml |title =常捷同志逝世  |author =  |date =  2012-08-21|publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

2012年7月29日，常捷在北京病逝，享年83岁<ref name=shishi/>。
