'''常熟—合肥高速公路'''，简称'''常合高速'''，江苏段编号S38<ref>{{cite web |url=http://www.jscd.gov.cn/art/2010/7/21/art_11028_461281.html |title=存档副本 |accessdate=2015-01-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20130531102246/http://www.jscd.gov.cn/art/2010/7/21/art_11028_461281.html |archivedate=2013-05-31 }} 江苏省高速公路命名和编号对照表</ref>，安徽段编号S24<ref>{{cite web |url=http://www.ahglj.com/preview/lzxxw/gsmm/ |title=存档副本 |accessdate=2014-04-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20140415143248/http://www.ahglj.com/preview/lzxxw/gsmm/ |archivedate=2014-04-15 }} 安徽省高速公路命名和编号</ref>，是连接[[中华人民共和国|中华人民共和国]][[江苏省|江苏省]][[常熟市|常熟市]]和[[安徽省|安徽省]][[合肥市|合肥市]]的一条[[高速公路|高速公路]]。该线路起自[[江苏省|江苏省]][[常熟市|常熟市]]董浜枢纽，接[[沈海高速公路|沈海高速公路]]，途经[[常熟市|常熟市]]、[[张家港市|张家港市]]、[[江阴市|江阴市]]、[[常州市|常州市]]、[[常州市|常州市]][[金坛区|金坛区]]、[[南京市|南京市]][[溧水区|溧水区]]、[[安徽省|安徽省]][[马鞍山市|马鞍山市]]、[[巢湖市|巢湖市]]，止于[[合肥市|合肥市]]。

==过境路段==

===江苏段===
{{main|沿江高速公路}}
董浜枢纽至横林枢纽段，原属[[沿江高速公路_(江苏)|沿江高速公路]]的一部分。其中青阳互通以东段于2004年8月15日通车<ref>{{cite web |url=http://news.thmz.com/col58/2004/08/2004-08-1923932.html |title=存档副本 |accessdate=2013-04-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304185912/http://news.thmz.com/col58/2004/08/2004-08-1923932.html |archivedate=2016-03-04 }} 宁太高速江阴至太仓段试运行</ref>，青阳互通以西段于2004年12月1日通车<ref>http://news.sina.com.cn/c/2004-12-02/08374409359s.shtml 沿江高速常澄段正式通车</ref>。

横林枢纽至桂庄枢纽段，原名宁常高速公路。2007年9月30日通车<ref>http://www.yangtse.com/jspd/ttxw/200710/t20071001_360637.htm{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }} 宁常和镇溧两高速昨建成通车</ref>。

桂庄枢纽至苏皖省界段，又名溧马高速公路。其中洛家边枢纽以东段原属[[宁杭高速公路|宁杭高速公路]]江苏段一期工程的一部分，2003年9月29日通车<ref>http://news.sina.com.cn/c/2003-09-30/0741844289s.shtml 宁杭高速南京段国庆前通车</ref>，洛家边枢纽以西至苏皖省界段于2013年12月31日通车<ref>http://jiangsu.sina.com.cn/news/m/2013-12-31/074286113.html 溧马高速今日12点通车 马鞍山半小时到禄口机场</ref>。

===安徽段===
{{main|马鞍山长江公路大桥|北沿江高速公路|芜合高速公路}}
苏皖省界至姥桥枢纽段，即[[马鞍山长江公路大桥|马鞍山长江公路大桥]]及接线。2013年12月31日通车。

姥桥枢纽至巢湖枢纽段，又名马巢高速公路，与[[天潜高速公路|天潜高速公路]]重线，属[[北沿江高速公路|北沿江高速公路]]的一部分。

巢湖枢纽至陇西枢纽段，与[[芜合高速公路|芜合高速公路]]重线。已通车。
