{{Japanese|japanese= ツンデレ |romaji= Tsundere }}
'''傲嬌'''（{{lang-ja|'''ツンデレ'''}}，亦常翻譯為'''口是心非'''、'''外冷內熱'''、'''惡嬌'''、'''嬌蠻'''<ref>{{cite journal |url={{Google books |plainurl=yes |id=g3bfAwAAQBAJ |pg=GBS.PT2}} |title=第5話 |journal=[繁體中文]《純真女僕週刊 |volume=5 |author=[[柿本健次郎|柿本健次郎]] |page=GBS.PT2 |language=zh-hant }}</ref>或'''嗔羞'''）是指「平常說話帶刺、態度強硬高傲，但在一定的條件下害臊地黏膩在身邊」的人物性格。此詞本來是發源於[[日本|日本]][[ACG|ACG界]]，而在包含[[動畫|動畫]]、[[漫畫|漫畫]]的[[御宅族|御宅族]][[文化|文化]]裡廣泛形成[[萌|萌]]的對象之一種[[類型|類型]]；另一方面在[[大眾媒體|大眾媒體]]上也會被使用到，作為譬喻有魅力的人物形象、戀愛模範、話題人物等。

== 定義 ==
源於日本動漫術語「{{lang|ja|ツンデレ}}」。此日语本字合成自日文拟声表感叠词「{{lang|ja|ツンツン}}」和「{{lang|ja|デレデレ}}」：
* 「{{lang|ja|ツンツン}}」，原指物體尖銳，指人竪起一身刺，難以親近，引申為是帶刺的态度，包含以冷淡、冷漠、故意挑毛病等行为表现。中文讀法为[[:wikt:嗔|「嗔」]]（Chēn），日文读音也近似「嗔」。
* 「{{lang|ja|デレデレ}}」的意思是害羞，害臊脸红（{{lang|ja|照れる}}），指羞澀、刻意討好，類似中文的「羞答答」。

取兩者合成的「{{lang|ja|ツンデレ}}」譯作「傲嬌」，就專指人口是心非，以敵意掩飾自己的不知所措，對人事物明明有好感，偏要拒諸門外<ref>{{cite web|url=http://www.vjmedia.com.hk/articles/2015/06/22/109464|title=「傲嬌」不是「驕傲」|publisher=[[輔仁媒體|輔仁媒體]]|author=名越拉岸|date=2015-06-22|language=zh}}</ref>。这两种情感表现未必有内外前后之别，也可能同时出现。

== 起源 ==
[[Comiket|Comiket]]的參與者市川浩一認為[[福星小子|福星小子]]裡的[[拉姆_(福星小子)|拉姆]]既是第一個萌角色也是第一個傲嬌角色<ref>{{cite book|last=Galbraith|first=Patrick W.|title=[[The_Otaku_Encyclopedia:_An_Insider's_Guide_to_the_Subculture_of_Cool_Japan|The Otaku Encyclopedia: An Insider's Guide to the Subculture of Cool Japan]] |publisher=Kodansha International|year=2009|page=46|isbn=978-4-7700-3101-3}}</ref>。

== 衍生用法 ==
「{{lang|ja|ツンデレっこ}}」<ref>{{lang|ja|っこ}}（漢字可寫成寫成「子」或「娘」）為日語對女孩的[[綽號|愛稱]]。</ref>（ {{lang|en|Tsunderekko}} ）和「{{lang|ja|ツンデレラ}}」<ref>為{{lang|ja|ツンデレキャラクター}}的簡寫法之一，因為人物在某些方面擁有明顯的落差對比，與《[[格林童話|格林童話]]》中的《[[灰姑娘|灰姑娘]]》（{{lang|ja|シンデレラ}}）極具異曲同工之妙。</ref>（ {{lang|en|Tsunderella}} ）則是對擁有這個屬性的人的稱呼。另外，「{{lang|ja|ツンデレラー}}」（{{lang|en|Tsundere-er}}）多半用來稱呼喜愛這個屬性的人，而非擁有這個屬性的人。

從2006年左右開始有產生傲嬌原詞「{{lang|ja|つんでれ}}」的派生語「{{lang|ja|つんてれ}}」（為方便稱謂，暫時稱其為「怒傲嬌」），引據日本語俗語詞典可知：
怒傲嬌是傲嬌的派生語，和傲嬌一樣，指平時有點神經的、有些帶刺的女生到了和男生2個人在一起的時候變得害羞起來的特質。但是在被男生的「原來你也有挺可愛的一面呢」之類的話語刺激之后，如果變得靦腆就繼續保持傲嬌「{{lang|ja|つんでれ}}」之稱。若是吼著“笨蛋”逃開則被稱為「怒傲嬌」「{{lang|ja|つんてれ}}」。

此外亦有衍生的「傲沉」（ツンしゅん），指作出帶刺的行為後，會因為厭惡自己無法坦率起來，變得情緒低落或苦惱的類型。

「[[雙馬尾|雙馬尾]]」（{{lang|ja|ツインテール}}）與「傲嬌」兩詞的日語發音相近，所以傲嬌女性角色的髮型常被設定為雙馬尾。

== 社會影響 ==
<!--日語 {{lang|ja|定義の変遷}} 一節-->
日語的這個詞語在[[ACG|ACG]]界約於2000年左右出現，但是它對一般男女關係發展的影響潛力甚深（特別是在[[戀愛遊戲|戀愛遊戲]]中）。

[[File:Tsundere.jpg|200px]]
「傲嬌」這個共通印象的形成，與以下作品有關：
* 1997年推出的[[美少女遊戲|美少女遊戲]]《[[To_Heart|To Heart]]》中的保科智子，從原本緊閉心房的帶刺態度，慢慢地與主角融洽起來。另外在1998年的《[[ONE～光輝的季節～|ONE～光輝的季節～]]》中登場的七瀨留美，從蠻橫態度到嬌羞少女的轉變而大受歡迎。七瀨留美和後繼作《[[Kanon|Kanon]]》的澤渡真琴並稱為「[[雙馬尾|雙馬尾]]系」，對後來的《[[願此刻永恆|願此刻永恆]]》大空寺亞由（{{lang|ja|大空寺あゆ}}）與《[[秋櫻之空|秋櫻之空]]》佐久間晴姬的人物設計也造成不小的影響。
* 2000年，《[[Canvas_～茶色的主题～|Canvas ～茶色的主题～]]》的櫻塚戀，以「不坦率」的性格博得人氣。
* 2001年，《願此刻永恆》、《秋櫻之空》相繼發表，《願此刻永恆》大空寺亞由為大小姐，《秋櫻之空》佐久間晴姬為損友，為「傲嬌般的（{{lang|ja|ツンデレ的な}}）」角色下了定義。
2002年8月29日，在投稿到『{{lang|ja|{{tsl|ja|あやしいわーるど|奇异世界|あやしいわーるど＠暫定}}}}』的文章裡，首次出現以「{{lang|ja|ﾂﾝﾂﾝﾃﾞﾚﾃﾞﾚが良い}}」形容大空寺亞由，並且在同掲示板上將本語縮短為「{{lang|ja|ツンデレ}}」後用以形容佐久間晴姬。

日本著名女[[聲優|聲優]][[釘宮理惠|釘宮理惠]]也因為經常為傲嬌的角色配音而被媒體封為「傲嬌女王」<ref>{{cite news|url=http://mainichi.jp/enta/mantan/archive/news/2009/03/07/20090307mog00m200025000c.html |archiveurl=https://web.archive.org/web/20100203160859/http://mainichi.jp/enta/mantan/archive/news/2009/03/07/20090307mog00m200025000c.html |title=声優アワード：「ツンデレの女王」釘宮理恵さんが主演女優賞に　神谷浩史さん二冠達成 |publisher=[[每日新聞|每日新聞]] |date=2009-03-07 |archivedate=2010-02-03 |language=ja |deadurl=yes }}</ref><ref>{{cite news|url=http://mainichi.jp/enta/mantan/graph/anime/20071210/ |archiveurl=https://web.archive.org/web/20070514030407/http://mainichi.jp/enta/mantan/graph/anime/20071210/ |title=ツンデレカルタ：「私の気持ちも知らないで…」“女王”釘宮理恵が読み手　緊急発売へ |publisher=[[每日新聞|每日新聞]] |date=2007-12-10 |archivedate=2007-05-14 |language=ja |deadurl=yes }}</ref>。

聲優[[青山由香里|青山由香理]]因常為成人遊戲配音傲嬌角色也被封為「傲嬌之神」，甚至有粉絲將她與釘宮並稱「表里雙嬌」
