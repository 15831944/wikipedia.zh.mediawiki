'''崁頂車站'''位於[[屏東縣|屏東縣]][[崁頂鄉|崁頂鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[屏東線|屏東線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[岸式月台|岸式月台]]一座。
*原有木造站房於2001年拆除，原址改建廁所。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]，主要以鄰近之[[南榮國中|南榮國中]]為主要通勤客源。
*停靠區間車及區間快。（區間車為[[復興號列車|復興號]]車廂編組或是單組[[臺鐵柴聯車|柴聯車]]編組，區間快為2組柴聯車編組）。
*北上首班車發車時間為清晨5點34分;北上末班車發車時間為晚間11點28分。
*南下首班車發車時間為清晨5點48分;南下末班車發車時間為晚間12點16分。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1952年11月24日：設簡易站，並指定由[[潮州車站|潮州站]]管理。
*1990年11月15日：由簡易站降為招呼站。
*2016年1月1日：啟用多卡通刷卡機。
