'''齿龈颤音'''是[[辅音|辅音]]的一种类型, 在很多[[语言|语言]]中使用，如[[俄语|俄语]]、[[藏语|藏语]]、[[西班牙语|西班牙语]]、[[亚美尼亚语|亚美尼亚语]]和[[波兰语|波兰语]]、[[意大利语|意大利语]]、[[瑞典语|瑞典语]]、[[德语|德语]]、[[阿拉伯语|阿拉伯语]]、[[挪威语|挪威语]]、[[冰岛语|冰岛语]]、[[芬兰语|芬兰语]]、[[马来语|马来语]]、[[蒙古语|蒙古语]]、[[土耳其语|土耳其语]]、[[满语|满语]]、[[鄂伦春语|鄂伦春语]]、[[锡伯语|锡伯语]]、[[泰语|泰语]]、[[维吾尔语|维吾尔语]]、[[邵語|邵語]]等。[[國際音標|國際音標]]符號{{IPA|{{尖括号|r}}}}反映了[[齿音|齒顫音]]、[[齿龈音|齒齦顫音]]和[[齒齦後顫音|齒齦後顫音]]，其[[X-SAMPA|X-SAMPA]]符號也是{{尖括号|<tt>r</tt>}}。在中文環境，齒齦顫音又稱為'''大舌音'''、'''大舌顫音'''或'''彈舌音'''，與[[小舌顫音|小舌顫音]]相對。齿龈颤音是舌尖颤音中的最常见的一种。除了部分[[西南官话|西南官话]]和[[吴语|吴语]][[宣州片|宣州片]]等個別方言外，大部分地區的漢語方言及[[普通話|普通話]]均无此音。

== 主要语言 ==
大部份[[斯拉夫语族|斯拉夫语言]]都有齿龈颤音，如[[俄语|俄语]]和[[波兰语|波兰语]]，以及大部分[[罗曼语族|罗曼语言]]，如[[西班牙语|西班牙语]] (写成rr或在词首的r)、[[意大利语|意大利语]]、[[加泰罗尼亚语|加泰罗尼亚语]]和[[奥克语|奥克语]]。[[巴斯克语|巴斯克语]]也有齿龈颤音。标准巴黎[[法语|法语]]使用[[小舌颤音|小舌颤音]]，标准[[德语|德语]]使用齿龈颤音或者小舌颤音均可，实际使用中以小舌颤音居多，尤其在德国北部。[[法语|法语]]的和[[德语|德语]]的一些南方方言则用齿龈颤音。[[荷兰语|荷兰语]]、口语[[希伯莱语|希伯莱语]]也有类似的现象。通俗及标准的[[阿拉伯语|阿拉伯语]]也有该颤音，字母是{{lang|ar|ر|}}。

在一些語言，例如[[捷克語|捷克語]]、[[斯洛伐克語|斯洛伐克語]]和[[塞爾維亞-克羅地亞語|塞爾維亞-克羅地亞語]]，齒齦顫音可以作為[[成節輔音|成節輔音]] {{IPA|[r̩]}}，例如捷克語 ''krk'' (頸)或塞爾維亞-克羅地亞語 ''rt'' (海角)。成音節的{{IPA|[r̩]}}也可以讀成長音，例如斯洛伐克語 (加上[[變音符號|變音符號]]表示；/ŕ/)和塞爾維亞-克羅地亞語的 ''{{lang|sh|sŕna}}'' {{IPA|[sr̩ːna]}} (歐洲雌性狍)。一些捷克人會以捷克語的特點做成[[繞口令|繞口令]]，例如：“{{link-en|Strč prst skrz krk|Strč prst skrz krk}}”（“用手指戳穿頸”）。整句話只有{{IPA|[r̩]}}（音節化的{{IPA|[r]}}）是元音。

該音尚有一清輔音版本，符號為{{IPA|[r̥]}}。[[威爾士語|威爾士語]]有此音，寫作''rh''。[[布列塔尼语|布列塔尼語]]保留了其齒齦顫音。[[蘇格蘭蓋爾語|蘇格蘭蓋爾語]]和[[愛爾蘭語|愛爾蘭語]]一些方言有齒齦顫音，但[[齒齦閃音|齒齦閃音]]更為普遍。齒齦顫音也在[[冰島語|冰島語]]的音節末出現。在[[古典希臘語|古典希臘語]]，清和濁齒齦顫音可能是[[同位異音|音位變體]]。

[[馬拉雅拉姆語|馬拉雅拉姆語]]的一些使用者，會將其語言的兩個R音都讀成顫音。他們區分''齒齦前'' (類似齒音)和''齒齦后''顫音，即是{{IPA|[r̟]}}相對于{{IPA|[r̠]}}。

唱英語歌也會用到這個音。特別是歌劇等的古典音樂也會經常用到這個音。

[[漢語|漢語]]中，在漢江流域，包括[[襄陽|襄陽]]、[[荊門|荊門]]、[[荊州|荊州]]等地的方言中，也有這個音。

== 該音見於漢語 ==

多数[[漢語|漢語]]中的方言都沒有此音，但[[吴语|吴语]][[宣州片|宣州片]]部分方言点以及[[湖北省|湖北]]的中北部的一部分[[中原官话|中原官话]]区和[[西南官话|西南官话]]区里（即[[当阳|当阳]]、[[江陵|江陵]]，[[钟祥|钟祥]]、[[京山|京山]]一带）直至[[神农架|神农架]]北部地区，存在有颤音/r/。 宣州吴语中的/{{IPA|r}}/来自中古[[定母|定母]][d]，而官话中则多半是由词尾“子”演变而来。<ref>李宇明《鄂豫方言的颤音》，华中师范大学学报 1984</ref>。另外，某些[[普通話|普通話]]或[[官话方言|官话方言]]的民歌如[[鳳陽花鼓|鳳陽花鼓]]裡，歌詞「''得<sub>兒</sub>''叮噹飄一飄」裡，如果讀時舌肌放鬆，就會讀出清齒齦顫音。

== 该音见于英语 ==
绝大多数的英语方言都没有齿龈颤音。最突出的例外是[[苏格兰英语|苏格兰英语]]方言。对于母语中没有该音的人们来说，要发出颤音是很具挑战性的，不過只要抓到訣竅就非常簡單。

== 标音 ==
在英语、德语、法语等语言的词典中，符号[r]通常作为这些语言的R辅音之统一符号（即R[[音位|音位]]），而不管字母R在这种语言里实际发音是否确是[r]所标记的颤音。例如英语中的[[齿龈无擦通音|齿龈]]和[[卷舌无擦通音|卷舌无擦通音]]（IPA符号应为：{{IPA|[ɹ]}}和{{IPA|[ɻ]}}），和德语的[[小舌颤音|小舌颤音]]（IPA符号: {{IPA|[ʀ]}}），均在字典內用[r]标记。

==清齿龈颤音==
{{Infobox IPA|ipa symbol=r̥}}

有些语言还有'''清齿龈颤音'''，就是一种声带不震动的发音。通常伴随于之相似的音位和音素出现。有学者认为此音曾在[[古希腊语|古希腊语]]中出现，写作⟨{{lang|grc|ῥ}}⟩，而在[[现代希腊语|现代希腊语]]中并入{{IPA|/r/}}。

== 抬齒齦顫音 ==
{{Infobox IPA|ipa symbol=r̝|ipa symbol2=ɼ|sound=cs-ř.ogg}}
[[捷克語|捷克語]]有一個獨特的[[音素|音素]]，與齒齦顫音不同。其發音方法雷同，但舌頭抬起，而且有[[擦音|擦音]]的特徵。此音在捷克語以字母<ř>表示，國際音標符號是 [{{IPA|r̝}}]。此音一般的發聲類型是濁音，也有相應的清音[{{IPA|r̝˚}}]，但並非獨立音素，只作為[[同位異音|音位變體]]。此音在詞語{{lang|la|''rybá'''ř'''''}} （單數漁夫）中是清音，在{{lang|la|''rybá'''ř'''i''}} (眾數漁夫)卻是濁音。

此音[{{IPA|r̝}}]不是[[響音|響音]]，不能作為[[音節|音節]]的[[核心|核心]]。

== 音位變體 ==
在大部分印歐語言，齒齦顫音偶爾和[[齒齦閃音|濁齒齦閃音]]作為音位變體，尤其在非重音的音節。[[西班牙語|西班牙語]]和[[阿爾巴尼亞語|阿爾巴尼亞語]]卻是例外，兩個輔音都是獨立音素。

在[[瑞典語|瑞典語]]中齒齦顫音的音位變體通常是[[齒齦無擦通音|濁齒齦無擦通音]] {{IPA|[ɹ]}}。
