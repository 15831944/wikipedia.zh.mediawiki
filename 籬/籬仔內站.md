'''籬仔內站'''位於[[高雄市|高雄市]][[前鎮區|前鎮區]][[籬子內|籬子內]]地區，為[[高雄捷運|高雄捷運]][[環狀輕軌|環狀輕軌]]的捷運車站。

== 車站概要 ==
車站代碼為'''C1'''。站區位於[[凱旋路_(高雄市)|凱旋三、四路]]與[[一心路|一心一路]]、[[瑞隆路|瑞隆路]]口的南側，站名取自舊地名[[籬仔內|籬仔內]]（閩南語：Lî-á-lāi）。

採[[側式月台|側式月台]]，為環狀輕軌第一階段的起點站。預計通車初期每日約六千人次<ref group="註">此資料為最初設計時之估計，當時[[高雄捷運紅線|紅線]]、[[高雄捷運橘線|橘線]]皆尚未通車，且不包括規劃中的鳳山線，故此估計可能不符時宜</ref>。

== 車站構造 ==
*[[側式月台|側式月台]]兩座。

== 歷史 ==
* 以「一心路站」為計畫站名。
* 2015年3月29日，公布定名為「籬仔內站」<ref group="參">[http://www.thenewslens.com/post/144007/ 高雄環狀輕軌C1-C14站有名字了！最快8月通車]</ref><ref group="註">命名活動第一階段建議站名為一心瑞隆站、凱旋瑞隆站、籬仔內站；第二階段票選選項為籬仔內站、瑞祥高中站、一心瑞隆站。</ref>
* 2015年10月16日，隨著環狀輕軌第一階段籬仔內－[[凱旋中華站|凱旋中華]]完工試營運啟用<ref group="參">自由時報—[http://news.ltn.com.tw/news/life/breakingnews/1248359 高雄輕軌 力拚8月試營運]</ref><ref group="參">中時電子報—[https://tw.news.yahoo.com/%E6%B0%B4%E5%B2%B8%E8%BC%95%E8%BB%8C-8%E6%9C%88%E5%BA%95%E9%9A%8E%E6%AE%B5%E9%80%9A%E8%BB%8A-215031837--finance.html 水岸輕軌 8月底階段通車]，2015年3月2日。</ref><ref group="參">[http://travel.ettoday.net/article/573060.htm?from=fb_et_travel 確定了！「高雄輕軌」10/16開放試乘、8日起受理預約]</ref>。
