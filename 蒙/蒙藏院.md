'''蒙藏院'''，是[[中华民国|中华民国]][[北洋政府|北京政府]]时期的中央机构，负责管理[[蒙古|蒙古]]、[[西藏|西藏]]等地区的少数民族事务。

==沿革==
1911年7月，[[外蒙古|外蒙古]]四部王公、喇嘛會盟，密商[[外蒙古獨立|外蒙古獨立]]。1911年10月10日，[[武昌起義|武昌起義]]爆發，[[辛亥革命|辛亥革命]]开始。1912年1月1日，[[南京臨時政府|南京臨時政府]]成立，[[中華民國|中華民國]]誕生。2月12日，[[清朝|清朝]]皇帝[[溥仪|溥仪]]正式逊位，清朝灭亡。3月10日，袁世凱任[[中華民國臨時大總統|中華民國臨時大總統]]<ref name=bainian>{{Cite web |url = http://www.nmg.xinhuanet.com/bnfynmg/bnjs/js3.htm |title = 百年内蒙古 |author =  |date =  |publisher =新华网  |language =  |accessdate = 2017-11-29 |quote =  }}</ref>。

1912年1月15日，[[呼倫貝爾|呼倫貝爾]][[厄魯特蒙古|厄魯特]]總管[[勝福|勝福]]等人武裝攻佔呼倫城，在[[俄罗斯帝国|俄国]]策劃下宣布呼倫貝爾獨立，成立隸屬[[外蒙古|外蒙古]][[庫倫|庫倫]]方面的獨立政權[[呼倫貝爾自治政府|呼倫貝爾自治政府]]，勝福被任命為參讚大臣、駐呼倫貝爾總督。勝福与[[陳巴爾虎旗|陳巴爾虎旗]]總管[[車和扎|車和扎]]在俄軍配合下多次进攻[[胪滨府|臚臏府]]，中方為免与俄军發生武裝衝突，于2月4日將駐軍撤出臚臏府，呼倫貝爾被勝福控制。1月29日，[[喀喇沁扎萨克杜棱郡王|喀喇沁右旗札薩克郡王]][[貢桑諾爾布|貢桑諾爾布]]與[[日本|日本]]浪人[[川島浪速|川島浪速]]共同谋划組織統一[[內蒙古|內蒙古]]各旗的機關，策劃內蒙古獨立；2月，[[扎萨克冰图郡王|科爾沁左翼前旗札薩克郡王]][[棍楚克蘇隆|棍楚克蘇隆]]與部分蒙旗王公及官員共商響應[[外蒙古獨立|外蒙古獨立]]<ref name=bainian/>。

1911年底至1912年初，受内地[[辛亥革命|辛亥革命]]影响，[[西藏|西藏]]发生[[辛亥拉萨动乱|辛亥拉萨动乱]]。[[英国|英国]]为实施其侵略西藏计划，派卸任不久的英印总督[[明托|明托]]亲自到[[大吉岭|大吉岭]]同[[十三世达赖|十三世达赖]]密谈，随后达赖派亲侍[[达桑占堆|达桑占堆]]秘密潜回西藏，以达赖的名义通告西藏各地官员和寺院，组织起一支1万多人的[[藏军|藏军]]，发动武装“驱汉”。1912年3月，达桑占堆率军进攻驻[[江孜|江孜]]的清军。4月初，英国驻[[亚东|亚东]]商务委员[[麦克唐纳|麦克唐纳]]赶到江孜，以“调停”的名义迫使驻江孜清军交出武器弹药，由西藏[[噶厦|噶厦]]发给路费，经[[印度|印度]]回中国内地。随后达桑占堆率军包围驻[[日喀则|日喀则]]清军，麦克唐纳以同样手法逼走清军。接着达桑占堆在英国支持下率军包围[[拉萨|拉萨]]，与驻拉萨清军激战3个月，使驻藏清朝官兵商民遭杀害。4月22日[[中华民国临时大总统|临时大总统]][[袁世凯|袁世凯]]发布大总统令，重申西藏是中国领土的组成部分，并将按历史定制处理西藏问题，5月10日发布命令任命[[钟颖|钟颖]]为[[西藏办事长官|西藏办事长官]]。随后英国以护侨为名直接出兵拉萨干涉，又护送达赖自[[印度|印度]]返藏，7月30日英国派[[廓尔喀|廓尔喀]]驻藏官员[[噶卜典|噶卜典]]出面调停，迫使钟颖与达桑占堆军队“讲和”，驻拉萨的钟颖军队交出武器，大部分于9月1日自拉萨启程经印度回内地。钟颖及其卫队也于11月经印度返回中国内地。当时西藏[[九世班禅|九世班禅]]为首的班禅系统，以及[[哲蚌寺|哲蚌寺]]、[[丹吉林寺|丹吉林寺]]数万僧侣公开反对分裂、拥护国家统一，内地[[云南|云南]]都督[[蔡锷|蔡锷]]、[[江西|江西]]都督[[李烈钧|李烈钧]]、[[四川|四川]]都督[[尹昌衡|尹昌衡]]、署川督[[胡景伊|胡景伊]]、四川的[[熊克武|熊克武]]等地方军政领导人纷纷上书袁世凯督促中央政府早筹良策。1912年5月，西藏在英国支持下发动[[康藏边界纠纷|川边战争]]，进攻[[康区|川边]]，不久即被尹昌衡率川军击败，中央政府也在外交上同英国进行了针锋相对的斗争。1913年1月，西藏与外蒙古签订了《[[蒙藏协定|蒙藏协定]]》，相互承认独立地位。《蒙藏协定》的消息使英国非常震惊，担心俄国在西藏加强势力，同时也对中华民国中央政府同西藏[[噶厦|噶厦]]和解的动向感到忧虑，故英国一方面同西藏噶厦订立密约，预备西藏宣言独立后英国保持在西藏的影响，另一方面1913年2月16日又同俄国签订了关于西藏的秘密协约。1913年10月至1914年4月，“中、英、藏三方会议”即[[西姆拉会议|西姆拉会议]]召开，但会议最终破裂<ref name=ju>{{Cite web |url = http://lib.xzmy.edu.cn/newsinfo?id=4321&cid=106 |title =蒙藏事务局及其对藏政的管理  |author =  张羽新|date =  |publisher =中国藏学2003年01期  |language =  |accessdate =  |quote =  }}</ref>。

1912年3月25日，[[北洋政府|中華民國政府]]頒布《勸諭蒙藏令》，希望蒙藏人民率循舊俗，作西北屏藩，安心內向；4月13日頒布《勸諭漢、滿、蒙、回、藏各族聯姻令》；21日宣布取消蒙藏回疆等藩屬名稱，撤銷[[清朝|清朝]][[理藩部|理藩部]]，凡蒙藏回疆各地方行政事俱歸中華民國內務部接管；設[[綏遠城將軍|綏遠城將軍]]，監督節制[[歸綏道|歸綏道]]所屬12[[撫民廳|撫民廳]]、[[烏蘭察布盟|烏蘭察布盟]]6旗、[[伊克昭盟|伊克昭盟]]7旗以及[[土默特旗|土默特旗]]，不久歸綏道改為歸綏觀察使，所屬12撫民廳均改成縣<ref name=bainian/>。

[[中华民国|中华民国]]初年，蒙藏事务仍由清朝留下的'''[[理藩部|理藩部]]'''管理，大政方针由具临时政府性质的民国筹备处决定。清帝[[溥仪|溥仪]][[清室退位|逊位]]时，北京设临时筹备处。民国元年（1912年）三月初七日，临时大总统[[袁世凯|袁世凯]]命令：“理藩部正首领[[达寿|达寿]]，因病不克进署，恳请另委接替等语。理藩部正首领委任[[荣勋|荣勋]]兼管。此令。”袁世凯就任临时大总统后，撤销理藩部，将蒙藏政务移交内务部管理，1912年4月22日为此颁发大总统令称<ref name=ju/>：
{{引用|“现在五族共和，凡[[蒙古|蒙]]、[[西藏|藏]]、[[回疆|回疆]]各地方，同为我中华民国领土，则蒙、藏、回疆各民族，即同为我中华民国国民。此后，蒙、藏、回疆等处，自应统筹规划，以谋内政之统一，而冀民族之大同。民国政府于理藩不设专部，原系视蒙、藏、回疆与内地各省平等，将来各该地方一切政治，俱属内务行政范围，现在，统一政府业已成立，其理藩院事务，著即归并内务部接管。其隶于各部之事，仍归划各部管理。在地方制度未经划一规定以前，蒙、藏、回疆应办事宜，均各仍照向例办理。”<ref name=ju/>}}

1912年5月，內務部下設'''蒙藏事務處'''<ref name=bainian/>。蒙、藏、回疆事务归内务部后，内务部想增一位次长。1912年6月12日，[[中华民国第一届国会|参议院]]的议案“大总统交议修改官制通则”由法制股审查长报告，说明了审查会否决内务部增加次长的理由，并就设置'''蒙藏事务局'''的必要性及其规格向参议院说明<ref name=ju/>：

{{引用|“蒙藏事务的办法，以后尤要主张积极。《[[中华民国临时约法|约法]]》上规定，五族人民平等。但是，[[藏族|藏族]]的文化上、经济上、教育上，与内地恐不能一致，所以须要积极主义，方可发展蒙藏的事务，然后才可为蒙藏同胞谋幸福，才可以达到五族人民平等的目的，所以决定另设特别机关，专营蒙藏事务，并非对于蒙藏有特别的政策，实在因为注重蒙藏事务起见，不得不然。其机关取名为蒙藏事务局，直隶于[[中华民国国务总理|国务总理]]。”<ref name=ju/>}}

该议案经参议院议决通过，由参议院咨请大总统，同日大总统交参议院议蒙藏事务局官制案。参议院议决通过后，7月24日大总统令公布《蒙藏事务局官制》，正式成立蒙藏事务局，该官制第一条规定：“蒙藏事务局直隶于国务总理，管理蒙藏事务。”7月29日，政府任命[[姚锡光|姚锡光]]为蒙藏事务局副总裁，暂兼署总裁。8月5日内务部通告，概不接受关于蒙藏的文件，此为蒙藏事务脱离内务部的开始。蒙藏事务局暂设北京[[东单牌楼|东单牌楼]][[苏州胡同|苏州胡同]]，姚锡光副总裁于当天接收视事。9月4日，国务院颁发汉、蒙、藏三体文字“蒙藏事务局”新印一颗，于5日启用。9月9日大总统令任命贡桑诺尔布为蒙藏事务局总裁，他于16日到任视事。10月28日，姚锡光副总裁奉使宣抚口北。大总统令任命[[荣勋|荣勋]]署蒙藏事务局副总裁，荣勋于10月31日到任视事。蒙藏事务局于11月18日从苏州胡同迁入前理藩部旧署办公。与此同时，总裁、副总裁呈请大总统袁世凯任命了一批顾问、参事、全事等。另外依照《蒙藏事务局官制》，“蒙藏事务局附设蒙藏研究会，掌研究、调查蒙藏一切事宜”<ref name=ju/>。

1912年8月19日中華民國政府頒布《蒙古待遇條例》，不以藩屬待蒙古，保留蒙古王公原管轄權，照舊承襲世爵封號，從優支給世爵俸餉；9月20日頒《加進實讚共和之蒙古各札薩克王公封爵》令，沿襲清制；設[[熱河都統|熱河都統]]，監督節制[[卓索圖盟|卓索圖盟]]7旗，[[昭烏達盟|昭烏達盟]]12旗及境內14個府縣；設[[察哈爾都統|察哈爾都統]]，統轄察哈爾左、右翼八旗以及4[[牧群|牧群]]，監督節制[[錫林郭勒盟|錫林郭勒盟]]10旗；[[哲里木盟|哲里木盟]]10旗歸蒙藏事務局管轄，並受[[東三省|東三省]]監督節制，其中[[郭爾羅斯後旗|郭爾羅斯後旗]]、[[扎賚特旗|扎賚特旗]]、[[杜爾伯特旗|杜爾伯特旗]]受[[黑龍江省|黑龍江省]]監督節制，[[郭爾羅斯前旗|郭爾羅斯前旗]]受[[吉林省|吉林省]]監督節制，[[科爾沁|科爾沁]]6旗受[[奉天省|奉天省]]監督節制；[[阿拉善旗|阿拉善旗]]、[[額濟納旗|額濟納旗]]受蒙藏事務局管轄，同時受[[寧夏|寧夏]]護軍使節制。8月20日，[[扎萨克扎萨克图郡王|科爾沁右翼前旗札薩克郡王]][[烏泰|烏泰]]發布《東蒙古獨立宣言》且舉兵；[[科爾沁右翼後旗|科爾沁右翼後旗]][[札薩克|札薩克]][[鎮國公|鎮國公]][[拉喜敏珠爾|拉喜敏珠爾]]起兵響應。東三省官軍随即出兵讨伐，烏泰、拉喜敏珠爾敗逃外蒙古。1912年11月，蒙藏事務局呈請大總統袁世凱批准，將清朝[[景山公园|景山]]宗室覺羅三學及清理藩部所設蒙古學堂合並成立[[蒙藏學校|蒙藏學校]]<ref name=bainian/>。
