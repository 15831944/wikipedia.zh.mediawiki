
'''蒙面人謬誤'''（masked man fallacy）是一種[[形式謬誤|形式謬誤]]，是因不當代換指涉對象造成的錯誤推論。

例如一般情況下，我們可如此推理：
{{exampleH}}
* 法官判處蒙面人死刑
* 蒙面人是老張
* 因此，法官判處老張死刑
{{exampleF}}

{{exampleH}}
* 法官判處蒙面人死刑
* 法官未判處老張死刑
* 因此，老張不是蒙面人
{{exampleF}}

然而，以下推理是不適當的：
{{exampleH}}
* 法官相信蒙面人是兇手
* 蒙面人是老張
* 因此，法官相信老張是兇手
{{exampleF}}

{{exampleH}}
* 法官相信蒙面人是兇手
* 法官不相信老張是兇手
* 因此，老張不是蒙面人
{{exampleF}}

== 代換原則 ==
'''[[代換原則|代換原則]]'''或'''共指涉代換'''是在甲是乙（甲和乙有共同的指涉）時，將語句中的甲代換乙。就如大部分形式系統的規則，如果a=b，則任何式子中的a都可以代換為b。

例如根據「小美是老王的女兒」將「小明欺負小美｣代換為「小明欺負老王的女兒」，就是代換原則的應用。

然而在某些情況下代換性原則會違反一般直覺，例如根據「超人是克拉克肯特」將「小美相信超人會飛｣代換為「小美相信克拉克肯特會飛」。

諸如「小明欺負x」、「法官判處x死刑」等語句是處於[[外延|外延性]]語境，而諸如「小美相信x會飛｣、「法官相信x是兇手」之類關於信念、想望、[[模態邏輯|模態]]（可能性）等語句則是處於[[內涵|內涵性]]語境。代換原則只適用於外延性語境，內涵性語境是不適用的。

== 萊布尼茲定律 ==
根據'''[[萊布尼茲定律|萊布尼茲定律]]'''，如果兩個東西相同，則兩個東西的所有性質都相同；反之，如果兩個東西的所有性質都相同，則兩東西相同。

原來的例子「法官相信蒙面人是兇手；蒙面人是老張；因此，法官相信老張是兇手」可將「法官相信蒙面人是兇手」翻譯成「蒙面人具有『被法官相信是兇手』的性質」，此時便可套用萊布尼茲定律：
* 蒙面人具有『被法官相信是兇手』的性質
* 蒙面人是老張
* 因此，老張具有「被法官相信是兇手」的性質

然而『被法官相信是兇手』能否作為一個合法的性質，是有爭議的。
