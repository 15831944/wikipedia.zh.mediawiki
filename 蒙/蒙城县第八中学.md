'''蒙城县第八中学'''，简称'''蒙城八中'''，是[[安徽省|安徽省]][[亳州市|亳州市]]的一所[[完全中学|完全中学]]，为亳州市示范高中。

== 校史 ==
蒙城八中创建于2002年。2012年7月，学校整体搬迁至原[[亳州师专|亳州师专]]蒙城校区。2014年8月，被批准为“亳州市示范高中”<ref name=jj>{{cite web|url=http://www.mc8z.com/about.php|title=关于我们|publisher=蒙城八中|accessdate=2018-07-03}}</ref>

== 现状 ==
学校占地67980平方米，建筑面积36720平方米。现有在校学生4560人。在职教师214人，其中特级教师1人，高级职称38人，硕士研究生5人。<ref name=jj/>

== 历任校长 ==
* 杨超（2002.5—2010.8）
* 张华（2010.8—2014.9）
* 白云峰（2014.9—今）
