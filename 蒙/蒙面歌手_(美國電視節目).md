'''蒙面歌手'''（{{lang-en|''The Masked Singer''}}）是一檔[[美國|美國]][[真人實境秀|真人秀]]歌唱比賽[[電視節目|電視節目]]，該系列於2019年1月2日在[[福斯廣播公司|Fox]]首度播映，它引進[[韓國|韓國]][[文化廣播公司|MBC電視台]]的綜藝節目 - [[神秘音樂秀：蒙面歌王|神秘音樂秀：蒙面歌王]]為基礎<ref>{{cite web |title= 福克斯推出“蒙面歌手”，這是一個具有開創性的新名人競賽系列 |url= http://www.thefutoncritic.com/news/2018/08/02/fox-unveils-the-masked-singer-a-new-celebrity-competition-series-with-a-groundbreaking-twist-103311/20180802fox01/ |website= [[The_Futon_Critic|The Futon Critic]] |date= {{start_date|2018|08|02}} |language= en}}</ref><ref>{{cite web |last1= 伯恩鮑姆 |first1= 黛布拉 |title= 福克斯訂購「蒙面歌王」真人秀系列; 將由尼克·卡農主持 |url= https://variety.com/2018/tv/news/fox-masked-singer-reality-series-nick-cannon-1202893337/ |website= [[綜藝_(雜誌)|《綜藝》]] |date= {{start_date|2018|08|02}} |language= en}}</ref><ref>{{cite web |last1= 林奇 |first1= 傑森 |title= 名人在Fox的蒙面歌手預告片中偽裝成不同的怪物和動物 |url= https://www.adweek.com/creativity/celebs-are-disguised-as-monsters-and-animals-in-foxs-bonkers-trailer-for-the-masked-singer/ |website= Adweek |date= {{start_date|2018|08|03}} |language= en}}</ref><ref>{{cite web |last1= 皮斯特 |first1= 勞倫 |title= 福克斯的新節目「蒙面歌手」很神秘和完全瘋狂 |url= https://www.eonline.com/ca/news/957240/fox-s-new-show-the-masked-singer-is-very-mysterious-and-totally-bonkers |website= E!頻道 |date= {{start_date|2018|08|02}} |language= en}}</ref>。

== 賽制 ==
一共有12位名人參加了十場比賽。 每集都有一對一的對決，介紹每個蒙面歌手，提供他們的身份提示（使用合成的聲音），並讓他們表演一首預訂的歌曲。 在表演結束後，評委有時間推測歌手的身份，並向他們提出一個問題，試圖確定他們的身份。 在一對一的對決後，評委和觀眾將對哪位歌手可以繼續下週的比賽進行投票。 在這一集的結尾，對決中的輸家是由評委較早投票決定； 被淘汰的歌手隨後摘下面具，向評委和觀眾展示自己的身份。
