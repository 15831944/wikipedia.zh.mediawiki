'''蒙特克莱尔州立大学'''（{{lang-en|'''Montclair State University'''}}）始建于1908年，是一所位于[[美国|美国]][[新泽西州|新泽西州]]的[[公立大学|公立大学]]，也是新泽西州第二大大学，<ref name="montclair.edu"/> 仅次于[[罗格斯大学|罗格斯大学]]。<ref name="montclair.edu"/> 截至2011年10月，一共有18498个学生，其中，14590个在校学生和3908名研究生。<ref name="montclair.edu"/> 蒙特克莱尔州立大学一共有250多个专业。<ref>[http://www.montclair.edu/majors/index.html Montclair State University - Undergraduate Degree Programs<!-- Bot generated title -->]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 历史 ==
1903年，美国新泽西州建立了国家师范学院，这是蒙特克莱尔州立大学的前身。<ref name="montclair1908">{{cite web |url=http://www.montclair.edu/centennial/history/timeline/1908.php |title=Montclair State University - History of MSU - Timeline - 1908 |publisher=Montclair.edu |date= |accessdate=2011-08-07 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

为了建设成为一所真正意义上的大学，美国政府花费了一年时间讨论才批准允许它的扩大。<ref name="montclair1908"/>

1908年，学校命名为'''新泽西州立蒙特克莱特师范学校'''（ New Jersey State Normal School at Montclair），学校计划5年后完成。<ref name=timeline>[http://www.montclair.edu/centennial/history/timeline.php Centennial Timeline]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}, Montclair State University's official website</ref> 同年，新泽西州的[[州长|州长]]约翰·富兰克林参加了学校的建设，查尔斯·萨姆纳·查平正式在此年度担任第一责任人，负责掌管学校的事务。<ref>{{cite web |url=http://www.montclair.edu/centennial/history/leadership.php |title=Montclair State University - History of MSU - Leadership |publisher=Montclair.edu |date= |accessdate=2011-08-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20110721011648/http://www.montclair.edu/centennial/history/leadership.php |archivedate=2011-07-21 }}</ref> 学校的第一个建筑物是学校的大厅，至今依然存在和发挥作用。那时候，学校实行的是两年制，因为当初学校专门负责培训教师。当时，校园约25英亩(100000 m²)，具有8个教员和187名学生。第一个毕业班，编号为45的学生，包含1929年[[普利策奖|普利策奖]]的获得者。<ref name="montclair2">{{cite web |url=http://www.montclair.edu/centennial/history/timeline.php |title=Montclair State University - History of MSU - Timeline |publisher=Montclair.edu |date= |accessdate=2011-08-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20110721011336/http://www.montclair.edu/centennial/history/timeline.php |archivedate=2011-07-21 }}</ref>  第一个宿舍被建成五年之后，在1915年，被称为“罗素大楼”。<ref name="montclair2"/>

1924年，哈利斯·普拉格博士是成为第一任学校校长。<ref name="montclair2"/> 不久后，他就主导了学校的课程改革，学校开始开放更具包容性的课外活动，如体育，但是，学校的宗教色彩依然严重，这仍然是一个重要的教派文化学校。<ref name="montclair2"/> 1927年，学校为新泽西州培育了一大批的高中教师。1937年，学校开始设立了专门的师范学院。<ref name="montclair2"/>

1943年，[[第二次世界大战|第二次世界大战]]期间，几名学生在被允许的前提下，经过校长哈利·斯普拉格的同意，加入了[[美国海军|美国海军]]，他们作为[[志愿者|志愿者]]坐上[[火车|火车]]，远赴战场参加战争。<ref name="montclair1943">{{cite web |url=http://www.montclair.edu/centennial/history/timeline/1943.php |title=Montclair State University - History of MSU - Timeline - 1943 |publisher=Montclair.edu |date= |accessdate=2011-08-07 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 当时，学生和教师将出售战争债券来支持美国军队。<ref name="montclair1943"/>

1958年，学校体育卫生装甲学院（Panzer College of Physical Education and Hygiene）合并，成为蒙特克莱尔州立大学。<ref name="montclair1943"/> 1966年，学校转变为一所多功能的综合性大学。<ref name=timeline /> 1994年4月27日，学校开设下属分院。<ref name=timeline />

== 学校院系 ==

*人文学科与社会科学学院
*科学与数学学院
*音乐学院
*文学院
*商业学院
*研究生学院
