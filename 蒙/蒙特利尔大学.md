'''蒙特利尔大学'''（{{lang-fr|Université de Montréal}}，简称UdeM）是一所以[[法语|法语]]为教学语言的[[加拿大|加拿大]]公立大学，位于[[加拿大|加拿大]][[魁北克|魁北克]]省[[蒙特利尔市|蒙特利尔市]][[皇家山|皇家山]]脚下。学校由13个学院以及2个著名的附属学院（[[蒙特利尔工程学院|蒙特利尔工程学院]] 和 [[蒙特利尔高等商学院|蒙特利尔高等商学院]]）组成。該校於1878年成立，建校初为同省另一名校[[拉瓦尔大学|拉瓦尔大学]]在[[蒙特利尔|蒙特利尔]]的分校，随后独立发展成如今的蒙特利尔大学。该校主要以[[法语|法语]]为教学语言，除法语外也有少量以[[英语|英语]]为教学语言课程。研究生项目可以申请用英语完成。
该大学拥有[[魁北克|魁北克]]最大以及[[加拿大|加拿大]]第三大的赞助研究收入,亦是加拿大顶尖研究型大学联盟[[U15_(加拿大大学)|U15]]的成员之一，超过55,000名学生就读于本科和研究生课程使它成为[[加拿大|加拿大]]学生人数第二多的大学。<ref name=stats>[http://www.umontreal.ca/udem-aujourdhui/en/faits-et-chiffres/index.html  Université de Montréal official statistics] {{webarchive|url=https://web.archive.org/web/20120303043907/http://www.umontreal.ca/udem-aujourdhui/en/faits-et-chiffres/index.html |date=2012-03-03 }}</ref>

== 历史 ==
大学成立于1878年，成立初始是作为[[拉瓦尔大学|拉瓦尔大学]]于[[蒙特利尔|蒙特利尔]]所设立的分校而存在。直到1919年5月，教皇[[本笃十五世|本笃十五世]]授予其完全的自主权，自此它成为了一间独立的[[天主教大学|天主教大学]]，并开始使用蒙特利尔大学作为其正式校名。

== 校园 ==
[[Image:Université_de_Montréal.JPG|thumb]]
这所大学的主校区位于[[蒙特利尔|蒙特利尔]]市的Outremont（乌特蒙）和[[:yue:葛地尼殊|葛地尼殊]]行政区[[皇家山|皇家山]]的北坡。

除了主要的皇家山校区，大学也分别拥有Terrebonne（迭荷本），[[拉瓦勒_(魁北克)|拉瓦勒]]，[[隆格伊|隆格伊]]，Hyacinthe（耶桑特）和[[摩丽丝区|摩丽丝区]]五个分校区，以及蒙特利尔大学医学中心（Centre hospitalier de l'Université de Montréal）和圣贾斯汀医学中心（Centre hospitalier universitaire Sainte-Justine）两所服务于医学院教学网络的附属医院。

==附属学校==
[[蒙特利尔高等商学院|蒙特利尔高等商学院]]是蒙特利尔大学的附属商学院，是加拿大第一所专业的管理学院。[[蒙特利尔工程学院|蒙特利尔工程学院]]也隶属于蒙特利尔大学，但在管理和财政上享有自治权。

== 学术 ==
