'''蒙恬'''（{{bd|約前250年||前210年|catIdx=M}}），[[中国|中国]][[秦朝|秦朝]]名将，祖居[[齐国|齐国]]，祖父[[蒙骜|蒙骜]]和父亲[[蒙武|蒙武]]皆为[[秦國|秦國]]名将。蒙恬北防[[匈奴|匈奴]]多年，威震北方。[[秦始皇|秦始皇]]駕崩後，因為支持[[太子|太子]][[扶苏|扶苏]]，被[[趙高|趙高]]設計處決。

== 生平 ==
蒙恬最初作过掌理[[司法|司法]]文書的官吏。《史記·白起王翦列傳》记载蒙恬在前224年作为[[李信|李信]]的副手，一起带兵二十万攻[[楚国|楚]]，独自带领一支分遣队攻克[[楚国|楚国]]的寝丘（今[[安徽|安徽]][[临泉|临泉]]），李信在攻破鄢（今[[河南|河南]][[鄢陵|鄢陵]]）后，引兵向西，與蒙恬在城父（今[[河南|河南]][[宝丰县|宝丰县]]东）会合，不久李信军被楚军击破。秦始皇復用[[王翦|王翦]]为将，蒙恬父蒙武为裨将军带兵六十万攻楚。始皇二十六年（前221年）因为出身将门被[[秦始皇|秦始皇]]起用为将军，攻[[田齐|齐国]]大胜，后任[[内史_(官制)|内史]]。<ref>{{Wikisource history|史記|073|白起王翦列傳}}</ref>

统一六国后，秦始皇三十三年（前214年），率领秦军三十萬人征伐[[匈奴|匈奴]]，次年收复河南地（今[[内蒙古|内蒙古]][[河套|河套]]一带），击退匈奴七百余里，屯兵上郡（今[[陕西|陕西]][[榆林|榆林]]东南）。蒙恬吸取[[战国|战国]]时期据险防御的经验，从[[榆中|榆中]]（今属[[甘肃|甘肃]]）沿[[黄河|黄河]]至[[阴山|阴山]]构筑城塞，连接[[燕|燕]]、[[赵|赵]]、秦5000余里旧[[长城|长城]]，據陽山（陰山之北）逶迤而北。并修筑北起[[九原|九原]]、南至[[云阳|云阳]]的[[秦直道|直道]]，构成了北方漫长的防御线。蒙恬守北防十餘年，[[匈奴|匈奴]]慑其威猛，不敢再犯。

而当时蒙恬之弟[[蒙毅|蒙毅]]也在作[[上卿|上卿]]。[[蒙毅|蒙毅]]曾判处[[中车府令|中车府令]][[赵高|赵高]][[死刑|死刑]]，但赵高又被[[赦免|赦免]]，从此蒙氏兄弟和赵高结怨。赵高和秦始皇幼子[[胡亥|胡亥]]关系密切，曾私下教授[[胡亥|胡亥]][[法律|法律]]方面知识，頗受胡亥重用。

秦始皇长子[[扶苏|扶苏]]因-{}-數諫[[皇帝|皇帝]]，失帝愛，貶至[[上郡|上郡]]，成為蒙恬的[[監軍|監軍]]。始皇三十七年（前210年）秦始皇在出巡中在[[沙丘_(地名)|沙丘]]病故。[[赵高|赵高]]勾結[[李斯|李斯]]，矯[[詔|詔]]擁立[[胡亥|胡亥]]继位，[[赐死|赐死]]扶苏，史称[[沙丘之变|沙丘之变]]。因为蒙恬与扶苏关系密切，也同时赐死蒙恬。接到命令后，蒙恬覺得可疑，勸阻扶蘇：「皇帝陛下在外地，沒有立太子，派微臣率領三十萬大軍防守邊疆，又派您[[監軍|監軍]]，這是天下的重任啊。現在隨便一個使者來，就自殺？怎麼知道這不是有詐？我們應該上書請示，請示了再請示，再三請示，那時候死還不遲。」但扶蘇說：「父親要賜死兒子，有甚麼好請示的？」<ref>{{Wikisource history|史記|084|李斯列傳}}</ref>扶苏即刻自杀，蒙恬並未從死，被解除兵权关押在[[上郡|上郡]]的[[阳周|阳周]]，軍隊交由[[裨将|裨将]][[王离|王离]]指挥。

后来，由于扶苏已死，二世皇帝[[胡亥|胡亥]]想释放蒙恬，赵高不肯，称蒙恬兄弟政治上倾向扶苏，于是蒙恬、[[蒙毅|蒙毅]]兄弟被[[处決|处決]]，葬於[[綏德|綏德]]縣城西的[[馬鞍山|馬鞍山]]下。

== 家族 ==
蒙恬的祖父[[蒙骜|蒙骜]]原居[[齐国|齐国]]，在[[秦昭襄王|秦昭襄王]]在位時来到秦国，在[[秦莊襄王|秦莊襄王]]时出现在史书中，不断地为秦国征战，后来成为上卿。到[[秦始皇|秦始皇]]初年成为了秦国最重要的[[将领|将领]]之一。蒙恬的父亲蒙武作为[[王翦|王翦]]（正是后来蒙恬的裨将[[王离|王离]]的祖父）的[[裨将军|裨将军]]参加了秦灭楚之战。到蒙恬时，蒙恬已经可以率领当时秦军主力北征匈奴，连秦始皇的长子也被派到蒙恬军中作监军。蒙毅也同时官至上卿，并曾带兵，家族的地位更甚於蒙骜在世時期。弟弟[[蒙毅|蒙毅]]為得秦始皇信任的內衛大臣。但也造成了与胡亥、李斯和赵高等的矛盾和利益冲突，最终导致了蒙氏家族的灭亡。

== 评价 ==
蒙恬在后世有各种评价，[[司马迁|司马迁]]在《[[史记|史记]]》中认为：蒙恬在当时位高权重，而秦朝刚刚建立，人民刚刚脱离战乱，需要休养生息，不去劝阻秦始皇滥用民力，反而积极的修筑长城是有罪的。

传说蒙恬发明了[[毛笔|毛笔]]，《[[史記|史記]]》“恬取中山兔毛造筆”，[[晉人|晉人]][[崔豹|崔豹]]《[[古今注|古今注]]》提到：“牛享問曰：‘自古有書契以來，便應有筆，世稱蒙恬造筆，何也？’答曰：‘蒙恬始造，即秦筆耳。’”。據說蒙恬在鎮守北方時，見[[匈奴|匈奴]]以獸毛沾[[顏料|顏料]]繪圖，又同時前方戰況告急，才改篆刻[[竹簡|竹簡]]由絲卷書寫軍情。這才聲名大噪，並由史官記於史冊內。

[[考古|考古]]发现在蒙恬之前中国已有[[毛笔|毛笔]]，蒙恬應該是毛筆的改進者，1954年湖南長沙發現第一枝戰國毛筆。

揚雄認為蒙恬的忠並無值得學習：“ [[扬雄|揚子]]《[[法言|法言]]》曰：或問：「蒙恬忠而被誅，忠奚可為也？」曰：「漸山，堙谷，起臨洮，擊遼水，力不足而尸有餘，忠不足相也。」”<ref>{{Cite web|url=https://zh.wikisource.org/wiki/%E8%B3%87%E6%B2%BB%E9%80%9A%E9%91%91/%E5%8D%B7007#%E5%A7%8B%E7%9A%87%E5%B8%9D%E4%B8%89%E5%8D%81%E4%B8%83%E5%B9%B4%EF%BC%88%E8%BE%9B%E5%8D%AF%EF%BC%8C%E8%A5%BF%E5%85%83%E5%89%8D%E4%BA%8C%E4%B8%80%E3%80%87%E5%B9%B4%EF%BC%89|title=資治通鑑/卷007 - 维基文库，自由的图书馆|accessdate=2018-02-06|work=zh.wikisource.org|language=zh}}</ref>

[[司马光|司馬光]]在《[[資治通鑒|資治通鑑》]]中評：“臣光曰：秦始皇方毒天下而蒙恬為之使，恬不仁不知矣。然恬明於為人臣之義，雖無罪見誅，能守死不貳，斯亦足稱也。”<ref>{{Cite web|url=https://zh.wikisource.org/wiki/%E8%B3%87%E6%B2%BB%E9%80%9A%E9%91%91/%E5%8D%B7007#%E5%A7%8B%E7%9A%87%E5%B8%9D%E4%B8%89%E5%8D%81%E4%B8%83%E5%B9%B4%EF%BC%88%E8%BE%9B%E5%8D%AF%EF%BC%8C%E8%A5%BF%E5%85%83%E5%89%8D%E4%BA%8C%E4%B8%80%E3%80%87%E5%B9%B4%EF%BC%89|title=資治通鑑/卷007 - 维基文库，自由的图书馆|accessdate=2018-02-06|work=zh.wikisource.org|language=zh}}</ref>
