'''蒙彼利埃'''（[[法语|法语]]：Montpellier；[[奥克语|奥克语]]：Montpelhièr）是法国[[埃罗省|埃罗省]]省会，位于[[法国|法国]]南部[[地中海|地中海]]沿岸。属[[亚热带地中海气候|亚热带地中海气候]]。市内人口275,318（2014年），为法国第八大城市。<ref>[http://www.ot-montpellier.fr/cn/%E5%8F%91%E7%8E%B0/7.html 蒙彼利埃旅游局]</ref> 

该市教育事业发达，成立于1279年的[[蒙彼利埃大学|蒙彼利埃大学]]是法国第三古老的[[大学|大学]]。

== 历史 ==
[[File:Braun_Montpellier_UBHD.jpg|thumb]]
蒙彼利埃在公元8世纪时开始形成城市。[[中世纪|中世纪]]中期，蒙彼利埃因绝佳的地理位置优势，受到普罗旺斯公国的重视。[[洪诺留三世|洪诺留三世]]时期（1220年），这里创办了慈善性医学机构——蒙彼利埃医学院，这是法国最早的医科学校，也是[[蒙彼利埃大学|蒙彼利埃大学]]的前身。13世纪时，法国国王[[路易九世|路易九世]]曾授予该城在全国自由贸易的特权，因而财富大增。<ref>[http://www.yesfr.com/montpellier/ 法国在线]</ref>

[[文艺复兴|文艺复兴]]以及[[宗教战争|宗教战争]]时期，蒙彼利埃曾被严重损毁。从17世纪起，蒙彼利埃成为[[埃罗省|埃罗省]]的省会。19世纪后，随着工业和农业水平的进步，蒙彼利埃的[[葡萄酒|葡萄酒]]产量大增，质量也得到了提高，在法国名噪一时。1907年6月9日，因葡萄种植园主和农民不满当时的[[乔治·克列孟梭|乔治·克列孟梭]]政府的政策，蒙彼利埃发生了[[法兰西第三共和国|法兰西第三共和国]]历史上最大的反抗[[暴动|暴动]]，约有60万—80万左右的人参加。

[[二战|二战]]后，蒙彼利埃成为了[[朗格多克-鲁西永大区|朗格多克-鲁西永大区]]的首府。二十世纪末，美国[[IBM|IBM]]电子计算机公司在该市投资建厂，成为了蒙彼利埃提供职位数量最多的私营企业，也使得蒙彼利埃的现代化工业进一步发展。

== 教育==

蒙彼利埃是[[欧洲|欧洲]]最先出现[[大学|大学]]的地方之一，[[中世纪|中世纪]]时期[[醫學|醫學]]研究尤為盛行。[[諾斯特拉達姆士|諾斯特拉達姆士]]、[[彼特拉克|彼特拉克]]、[[拉伯雷|拉伯雷]]、[[奥古斯特·孔德|奥古斯特·孔德]]都曾在蒙彼利埃學習過。現有[[蒙彼利埃大學|蒙彼利埃大學]]和[[保羅凡萊內大學|保羅凡萊內大學]]（Paul Valéry University, Montpellier III，原蒙彼利埃第三大学）兩所大學。2014年在法国媒体评选出的最适合学习的法国城市排名中，蒙彼利埃位列第二名。

== 体育 ==

承办过的体育赛事：
* 1986年[[世界排球锦标赛|世界排球锦标赛]]
* 1998年[[法国世界杯|法国世界杯]]分赛场
* 2001年世界[[手球|手球]]锦标赛
* 2007年[[橄榄球|橄榄球]]世界杯

城市主要[[足球|足球]]队是[[蒙彼利埃足球俱乐部|蒙彼利埃-埃罗体育俱乐部]]，1974年正式成立，曾获得2012年[[法甲|法甲]]冠军。

== 交通 ==

===市内交通===
蒙彼利埃区域的交通主要由Tam公司管理

目前蒙彼利埃擁有四條[[蒙彼利埃有軌電車|輕軌]]，建構了稠密的大眾運輸網路，分別是一號藍線、二號黃線、三號綠線、四號棕線。

蒙彼利埃市区内有36条TAM公共汽车路线

蒙彼利埃同时提供自由的租自行车服务和租汽车服务
