'''黑山'''（[[黑山语|黑山语]]：{{lang|sr|Црна Гора}}，拉丁字母：{{lang|sr|Crna Gora}}）意即“黑色的山”，是位於[[巴尔干半岛|巴尔干半岛]]西南部、[[亞得里亞海|亞得里亞海]]東岸上的一個多山國家。过去多采用其英文名稱“Montenegro”的音译，如「'''门的内哥罗'''」，[[中國大陸|中國大陸]]、[[新加坡|新]][[马来西亚|馬]]、[[香港|港]][[澳門|澳]]則意譯作「'''-{黑山}-'''」; [[台灣|台灣]]則譯作「'''-{zh-hant:蒙特內哥羅;zh-hans:蒙特内哥罗;}-'''」。

黑山的東北為[[塞爾維亞|塞爾維亞]]，東為[[科索沃|科索沃]]（2008年2月17日[[科索沃独立|单方面宣布独立]]），東南為[[阿爾巴尼亞|阿爾巴尼亞]]，西北為[[波士尼亞與赫塞哥維納|波士尼亞與赫塞哥維納]]以及[[克羅埃西亞|克羅埃西亞]]，西南則為[[亞得里亞海|亞得里亞海]]（為[[地中海|地中海]]的其中一個部分）。

黑山原为1910年建立的[[黑山王国|黑山王国]]，一战后加入新成立的[[塞尔维亚人、克罗地亚人和斯洛文尼亚人王国|塞尔维亚人、克罗地亚人和斯洛文尼亚人王国]]，二战時期[[軸心國|軸心國]][[入侵南斯拉夫|入侵南斯拉夫]]，黑山成為[[意大利王國|意大利]]的[[保護國|保護國]]，後[[納粹德國|納粹德國]]佔領、[[盟軍|盟軍]]解放。二战后成为[[南斯拉夫社會主義聯邦共和國|南斯拉夫社會主義聯邦共和國]]的六个加盟共和国之一，南斯拉夫的四个加盟国（[[斯洛維尼亞|斯洛維尼亞]]、[[克羅埃西亞|克羅埃西亞]]、[[波士尼亞與赫塞哥維納|波士尼亞與赫塞哥維納]]、[[馬其頓共和國|馬其頓]]）在1990年代初独立后，黑山和[[塞尔维亚|塞尔维亚]]成为了南斯拉夫仅存的两个[[南斯拉夫聯盟共和國|加盟共和国]]。2003年，[[南斯拉夫聯盟共和國|南斯拉夫聯盟共和國]]决定放弃“南斯拉夫”的国名，改称[[塞尔维亚和黑山|塞尔维亚和黑山]]。2006年5月21日，黑山舉行[[2006年黑山獨立公民投票|公民投票]]，獨立派以55.5%的得票比例在投票中險勝；同年6月3日，黑山國會正式宣佈獨立，同時成为[[联合国会员国|联合国会员国]]。2017年6月5日，黑山加入[[北約|北約]]，成為該組織第29個會員國。<ref>{{cite news|url=http://www.bbc.com/news/world-europe-34981973|title=Nato invitation to Montenegro prompts Russia warning|work=BBC|date=2015-12-02}}</ref>

== 國名 ==
[[File:PodgoricaOverview.jpg|缩略图]]
黑山在本國官方語言[[黑山|黑山]]中稱為{{lang|sr|Црна Гора}}（[[西里爾字母|西里爾字母]]）或{{lang|sr|Crna Gora}}（[[拉丁字母|拉丁字母]]），意為“黑色的山”。西歐語言（如[[英语|英语]]、[[意大利语|意大利语]]等）中的黑山名称為Montenegro，該名稱源於[[意大利语|意大利语]]的威尼斯方言。monte即是山，negro則是黑色的意思。[[中国大陆|中]]、[[新加坡|新]]、[[馬來西亞|馬]]將該名稱意譯為「'''黑-{}-山'''」或音譯為「'''門特尼-{}-哥羅'''」；[[台灣|台灣]]則將其音譯為「'''-{zh-hant:蒙特內哥羅;zh-hans:蒙特内哥罗;}-'''」。

== 歷史 ==
{{Main|黑山歷史}}
黑山的先民為[[伊利里亞人|伊利里亞人]]，公元前3世紀時被[[古羅馬|古羅馬]]征服。羅馬帝國衰落以後，伊利里亞落入[[哥德人|哥德人]]之手，後來[[拜占庭帝國|拜占庭帝國]]（東羅馬帝國）皇帝[[查士丁尼一世|查士丁尼一世]]又將該地區重新納入版圖。

早期斯拉夫民族自6、7世紀起抵達此地，並且和當地先民融合，建立自己的國家。其中，杜克里亞（{{lang|sr|Duklja}}）公國比較接近蒙特內哥羅的版圖。該國1042年從拜占庭帝國獨立，並由羅馬教皇[[額我略七世|額我略七世]]封為王國。杜克里亞於12世紀衰落的同時，拉什卡（{{lang|sr|Raška}}）大公國（又稱塞爾維亞大公國）興起了新的[[內馬尼亞王朝|內馬尼亞王朝]]，杜克里亞成為塞爾維亞的一部分。

在[[內馬尼亞王朝|內馬尼亞王朝]]統治下，杜克里亞地區又出現了澤塔（{{lang|sr|Zeta}}）公國，14世紀中時澤塔脫離了內馬尼亞的控制。15世紀起[[奧斯曼帝國|奧斯曼帝國]]征服了巴爾幹半島大部分地區，澤塔于1498年被征服，併入奧斯曼帝國的斯庫台桑扎克（{{Lang|sr|İşkodra Sancağı}}）。1514年澤塔地區從斯庫台桑扎克分離，成為黑山桑扎克（{{Lang|sr|Karadağ Sancağı}}）。但奧斯曼帝國未能征服澤塔全境，例如[[洛夫琴山|洛夫琴山]]地區保持了獨立。 

[[Image:Montenegro1913.png|thumb]]
1516年，黑山成為以[[采蒂涅|-{采}-蒂涅]]為中心的[[政教合一|政教合一]]國家，國君為集政教權利於一身的[[親王主教|親王主教]]（{{lang|sr|Vladika}}）。1852年，黑山親王主教、彼德洛維奇-涅戈什家族的達尼洛二世（{{lang|sr|Danilo II Petrović Njegoš}}）放棄宗教頭銜，成為[[黑山公國|黑山公國]]的達尼洛一世，黑山又成為世俗公國。1860年，達尼洛二世被暗殺，[[尼古拉一世_(蒙特內哥羅)|尼古拉一世]]繼位。1876年蒙特內哥羅向[[奧斯曼帝國|奧斯曼帝國]]宣戰，並奪得大量土地。蒙特內哥羅於[[1878年柏林會議|1878年柏林會議]]中正式被承認為[[獨立國家|獨立國家]]。1910年，蒙特內哥羅大公尼古拉一世正式稱王，建立[[蒙特內哥羅王國|蒙特內哥羅王國]]。

[[第一次世界大戰|第一次世界大戰]]期間黑山加入了[[協約國|協約國]]以協助盟友[[塞爾維亞|塞爾維亞]]，隨即與[[奧匈帝國|奧匈帝國]]開戰。奧軍前兩次攻勢失敗，但在1916年1月的第三次進攻中，黑山被奧匈帝國完全佔領。尼古拉一世逃亡至義大利、法國。1918年，塞爾維亞等協約國的軍隊佔領蒙特內哥羅。

一戰結束後，在塞爾維亞的主導之下，斯洛文尼亞、克羅埃西亞聯合組成了第一個由[[南斯拉夫|南斯拉夫]]族群聯合的國家。同時，塞爾維亞控制下的黑山也召開會議，廢除本國國王、禁止國王回國，加入新成立'''塞爾維亞人、克羅埃西亞人和斯洛文尼亞人王國'''。<ref>John R. Lampe, 2002, Yugoslavia as History: Twice There was a Country. New York: Cambridge University Press, pp. 102-106.</ref>1919年，反對派發起[[聖誕節|聖誕節起義]]，但是被塞爾維亞軍鎮壓。1929年，國王[[亞歷山大一世_(南斯拉夫)|亞歷山大一世]]正式將王國更名為「[[南斯拉夫王國|南斯拉夫王國]]」，1939年[[二戰|二戰]]爆發，1941年4月[[納粹德國|納粹德國]]及其盟國[[入侵南斯拉夫|入侵南斯拉夫]]，[[意大利王國|意大利王國]]自[[阿爾巴尼亞意占時期|意屬阿爾巴尼亚]]入侵蒙特内可羅，4月17日意大利軍隊佔領黑山全境，1943年德軍取代意大利的占领，1945年[[第二次世界大戰|第二次世界大戰]]結束，德軍撤出南斯拉夫，南斯拉夫民族解放反法西斯議會制憲會議宣布南斯拉夫成立共產黨政府，由五個民族共組聯邦，罷黜君主制。<ref>江秉彝，2007，〈南斯拉夫疆界與民族衝突（1991-2006）〉，碩士論文，[[東海大學|東海大學]]政治系，頁16-22。</ref>蒙特內哥羅成為了[[南斯拉夫社會主義聯邦共和國|南斯拉夫社會主義聯邦共和國]]的加盟共和國之一。

1990年代，隨著共產黨政權倒台，南斯拉夫聯邦各國紛紛獨立，唯有[[塞爾維亞|塞爾維亞]]、黑山兩國未獨立。黑山於1992年舉行獨立公投，95.96%認為應繼續留在南斯拉夫聯邦之內。當時投票率為66%，[[穆斯林|穆斯林]]、[[天主教徒|天主教徒]]以及傾向獨立的[[黑山人|黑山人]]抵制了此次公投，他們認為該次公投的投票環境不夠民主，國營媒體過多為親統一（[[塞爾維亞|塞爾維亞]]）的一方宣傳。之後黑山共和國內在獨立議題上時有紛爭。

[[File:Milo_Djukanovic_with_Obamas.jpg|thumb]]自1991年至今五次出任總理]]
1996年，[[米洛·久卡諾維奇|米洛·久卡諾維奇]]政府宣佈和[[米洛塞維奇|米洛塞維奇]]掌權的[[塞爾維亞|塞爾維亞]]斷絕關係，從此以後黑山自行制定經濟政策，改用[[德國馬克|德國馬克]]（後來德國廢除馬克，採用[[歐元|歐元]]，蒙特內哥羅亦改為[[歐元|歐元]]）。从此[[塞爾維亞第納爾|塞爾維亞第納爾]]在黑山不是法定貨幣，只能在一些度假村使用。

由於和[[米洛舍維奇|米洛舍維奇]]掌權的[[塞爾維亞|塞爾維亞]]斷絕關係，蒙特內哥羅在接下來的南斯拉夫和[[科索沃戰爭|科索沃戰爭]]中受到損傷並不如[[塞爾維亞|塞爾維亞]]來的大，局勢也較[[塞爾維亞|塞爾維亞]]穩定許多。

2000年南聯盟大選後，執政十多年的塞爾維亞強人米洛塞維奇被迫下台，使得歐盟與南聯盟恢復外交關係，西方因此由支持蒙特內哥羅獨立，暫時轉向支持維持現有的聯邦狀態。但塞爾維亞與蒙特內哥羅間的關係並沒有因為西方國家支持維持現狀而改善，而1992年南聯盟的憲法中又沒有提及關於聯邦共和國實體獨立的權利及過程，從而引發爭論。<ref>江秉彝，2007，〈南斯拉夫疆界與民族衝突（1991-2006）〉，碩士論文，東海大學政治系，頁86。</ref>後來在歐盟的調停下，雙方於2002年3月14日在[[貝爾格萊德|貝爾格萊德]]簽署協議，決定將聯邦變為由兩個各自擁有經濟、貨幣以及關稅決定權的半獨立（semi-independent）國家所組成，並且更名為「[[塞爾維亞與蒙特內哥羅|塞爾維亞與蒙特內哥羅]]」。<ref>BBC News, 2002.3.14,“Yugoslav partners sign historic deal,”http://news.bbc.co.uk/2/hi/europe/1872070.stm.</ref>另外依據此協定通過了聯邦的新憲章，明定憲章通過三年後，任何一方都有權提出脫離聯邦。<ref>張志涵，2005.1，〈後冷戰時期族群衝突與血緣國干涉〉，碩士論文，台灣大學政治研究所，頁121。</ref>

黑山國會於2004年7月12日採用新的國旗、國慶日、以及國歌，以推進獨立進程。其中國旗基於前黑山王國的國旗，國慶日紀念1878年7月13日簽訂的[[1878年柏林條約|柏林條約]]承認黑山為獨立國家，以及1941年7月13日爆發的反法西斯起義。國歌則為黑山著名民歌。

2006年5月21日舉行[[2006年黑山獨立公民投票|公民投票]]，獨派以55.5%的微弱優勢決定終止與塞爾維亞的聯邦關係，剛剛超過55%[[歐盟|歐盟]]規定的有效結果票數。6月3日，黑山國會正式宣佈獨立，6月28日黑山正式加入[[聯合國|聯合國]]。

== 地理 ==
[[File:Budva,_view_from_Gospostina.jpg|缩略图]]]]
[[File:Biogradska_suma.jpg|缩略图]]
蒙特內哥羅境內主要是山脈、丘陵，只有沿海地區為狹長的平原，其中西北部的[[科托爾灣|科托爾灣]]為歐洲位置最靠南的[[峽灣|峽灣]]，該峽灣東岸有古城[[科托|科托]]，現已列為世界遺產。

位於中北部的[[各国最高点列表|博博托夫库克峰]]（{{lang|sr|Bobotov Kuk}}）為境內第一高峰，海拔2522米，附近的[[塔拉河|塔拉河]]有高達1300米的峽谷，該地亦已列為世界遺產。

== 政治 ==

[[File:FilipVujanovic.jpg|200px]]]]

蒙特內哥羅的[[国家元首|国家元首]]为总统，任期为5年，通过[[直接选举|直接选举]]产生。总统在海外代表本国，拥有颁布法律、向国会提议总理、宪法法院法官候选人、要求国会选举、颁发各类奖章奖项、通过国家法律特赦犯人等权力。总统官邸位于[[采蒂涅|采蒂涅]]。

总理是[[政府首脑|政府首脑]]，由国会任命，無任期限制，领导[[副总理|副总理]]和各政府部长。

国会为[[一院制|一院制]][[立法机构|立法机构]]，现共有81名议员，每名议员任期为4年。国会的作用是制定法律，批准条约，通过预算，任命总理、政府部长、各级法院法官以及由宪法确立的其他职责，并能以[[简单多数决制|简单多数决制]]通过对政府的[[不信任投票|不信任投票]]。

== 經濟 ==
黑山目前的經濟以服務業支撐。在2007年，服務業佔國民生產總值的72.4%，製造業和農業則分別佔17.6%和10%。該年國民生產總值有10.7%之升幅，但受全球金融危機影響，升幅有所放緩。

黑山的製造業主要是鋁材、鋼鐵業，其次是農產品加工。

旅遊業同時也是黑山之經濟命脈。2007年約100萬遊客抵達黑山觀光，帶來4億8千萬[[歐元|歐元]]進賬。

== 人口 ==
人口约65万（2014年估计）<ref name=CIA/>。黑山族占44.98%、塞尔维亚族28.73%，波斯尼亚克族占8.65%，阿尔巴尼亚族占4.91%。黑山72.07％人口信奉[[東正教|東正教]]，屬於[[塞爾維亞正教會|塞爾維亞正教會]]。1993年又建立了一個[[黑山正教會|黑山正教會]]，但信徒很少，而且並未得到其他東正教會的承認。19.11%的人口為[[穆斯林|穆斯林]]，其中包括[[阿爾巴尼亞人|阿爾巴尼亞人]]、[[波斯尼亞克人|波斯尼亞克人]]等。其中阿爾巴尼亞族主要居住在東南部，在東南部海港[[烏爾齊尼|烏爾齊尼]]占多數。波斯尼亞克族主要居住在北部。另外還有聚集在沿海地區（如[[科托爾灣|科托爾灣]]）的[[天主教|天主教]]徒。
