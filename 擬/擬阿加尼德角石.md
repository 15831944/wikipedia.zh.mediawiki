'''擬阿加尼德角石'''（[[學名|學名]]：{{lang|la|'''''Pseudaganides'''''}}）是一[[屬|屬]]已[[滅絕|滅絕]]的[[鸚鵡螺亞綱|鸚鵡螺類]]。牠們的[[化石|化石]]主要產於[[歐洲|歐洲]]等地，和同時期常見的[[新生角石|新生角石]]相比之下較為罕見。<ref name="Pseudaganides">{{Cite web |url = http://liusir.pixnet.net/blog/post/248636348-pseudaganides-%E6%93%AC%E9%98%BF%E5%8A%A0%E5%B0%BC%E5%BE%B7%E8%A7%92%E7%9F%B3-%28%E6%B3%95%E5%9C%8B%29 |title = Pseudaganides 擬阿加尼德角石 (法國) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-05-16 }}</ref>

== 特徵 ==
擬阿加尼德角石和新生角石的外型十分相像，都擁有內旋的殼與平坦的[[腹|腹]]側，體管較偏向腹面。兩者最顯著的差別為縫合線的差異，新生角石的縫合線側面為單純的弧線狀；而擬阿加尼德角石的縫合線側面則是呈現[[S|S]]型，氣室隔壁也更加扭曲。<ref name="Pseudaganides" />
