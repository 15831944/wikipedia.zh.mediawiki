== 車站構造 ==
*目前使用中的高架車站為[[島式月臺|島式月臺]]三座。

=== 車站樓層 ===
*二樓：月臺層，三座島式月臺
*一樓：出入口、計程車排班區、接送區、等候區、育嬰室、售票處、自動售票機、洗手間、商店（太平洋潮州驛站）、付費區

== 利用狀況 ==
配合台鐵潮州車輛基地啟用，本站為[[普悠瑪列車|普悠瑪列車]]的北高屏發車站及終點（[[潮州車站|潮州車站]]-[[高雄車站|高雄車站]]-[[台中車站|台中車站]]-[[南港車站|南港車站]]）每日北上南下各兩班。

車站1樓的空間，未來計畫引進多功能宴會廳及首創的貨櫃市集區：以BOT方式，由[[太平洋百貨|太平洋百貨]]取得經營權，將以「餐廳市集」的概念打造商場，並規畫貨櫃屋餐飲文創市集，提供特色文創商品及在地小吃，吸引年輕族群，成為潮州新地標，這也是[[高屏地區|高屏地區]]繼[[新左營車站|新左營車站]]及[[屏東車站|屏東車站]]以來，第三座台鐵火車站共構的百貨公司。另外也將設立多功能宴會廳，提供鄉親更優質的婚喜宴場地，預計2018年底開始營運。由於屏中、屏南地區並沒有電影院，未來也不排除引進電影院，為當地民眾開發新的休閒去處。<ref>{{cite news|author=|title=屏東、潮州車站商場 未來有的吃、有的玩|url=https://udn.com/news/story/7327/2556252}}</ref>

*目前為[[臺灣鐵路管理局車站等級#一等站|一等站]]。
*隨著[[高雄潮州鐵路捷運化計畫|高雄潮州鐵路捷運化計畫]]完工，本站於2015年成為臺鐵[[西部幹線|西部幹線]]的[[電氣化|電氣化]]終點，電氣化列車可行駛至本站，客運量快速成長。另外，潮州車站南端興建的[[臺鐵潮州車輛基地|潮州車輛基地]]於2015年10月15日啟用，2021年底整體計劃完工後（高雄機廠南遷潮州），潮州車站將取代[[高雄車站|高雄車站]]及[[屏東車站|屏東車站]]，成為[[西部幹線|西部幹線]]的電氣化終點，於2015年8月23日通車啟用，10月15日起改點增加班次。

*自強號除303次[[新左營車站|新左營]]至[[花蓮車站|花蓮]]、324次[[花蓮車站|花蓮]]至[[臺南車站|臺南]]與[[觀光列車|觀光列車]]1、2次未停靠外，其餘各級經此站的列車班班都有停靠。
*站內設有「鐵路之旅－小站巡禮紀念章」之戳章，可供旅客蓋章留念。
*2015年8月26日，139、141次自強號，3121、3133、3171、3203次區間車延駛至本站；108、114次自強號，3154、3168、3192、3228次區間車改為本站發車。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。<ref>{{cite news|author=|title=歸來─潮州電子票證多卡通　10月15日啟用|url=http://www.appledaily.com.tw/realtimenews/article/new/20151010/708645/|accessdate=2015-10-10|newspaper=蘋果日報|date=2015-10-10}}</ref>
*前站設立鎮內第一座有直達[[高雄國際機場|高雄國際機場]]與[[高雄捷運紅線|高雄捷運紅線]][[高雄國際機場站|高雄國際機場站]]的客運轉運站。

== 歷史 ==
*1920年2月22日：設站。
*2013年6月25日：改為高架車站。
*2015年8月23日：東西正線高架雙軌電氣化通車。
*2015年8月26日：電氣化列車由屏東站延長行駛至本站。
*2015年9月19日：升為一等站。
*2015年10月15日：路線大改點，本站成為部分西部幹線列車南部端點站；啟用多卡通刷卡機。
*2016年4月21日：改點後本站開始停靠[[普悠瑪列車|普悠瑪號]]。
