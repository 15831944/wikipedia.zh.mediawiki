'''潮州话音系'''包含[[声母|声母]]、[[韻頭|韻頭]]（[[介音|介音]]）、[[韻腹|韻腹]]和[[韻尾|韻尾]]。除了[[韻腹|韻腹]]和[[聲調|聲調]]，其他[[音節|音節]]的元素都並非必要。故此，[[韻腹|韻腹]]只要配合[[聲調|聲調]]，就可以自成音節。以下分析列出潮州话拼音、[[國際音標|國際音標]]（{{IPA|[ ]}}中者）和例字。

== 聲母 ==
正如其他閩南語方言，潮州話是少數擁有[[清濁音|濁]][[阻礙音|阻礙音]]（塞音、擦音和塞擦音）的漢語方言。然而，潮州話與[[吴语|吳語]]和[[湘语|湘語]]不同，其濁[[塞音|塞音]]並非源自中古漢語的濁阻礙音，而是來自其[[鼻音_(語言)|鼻音]]。故此，濁塞音{{IPA|[b]}}和{{IPA|[g]}}其實分別是前鼻化輔音（Prenasalized consonant）{{IPA|[ᵐb]}}和{{IPA|[ᵑg]}}；[[濁齒齦塞擦音|濁齒齦塞擦音]]{{IPA|[dz]}}是塞化的擦音{{IPA|[z]}}，两者为[[同位异音|同位异音]]。潮州话特色之一，就是没有[[唇齿音|唇齿音]]。

除了[[喉塞音|喉塞音]]，所有下列圖表列出的輔音都可以用作聲母。喉塞音只作入聲韻尾之一，以h表示。然而，有些音節沒有輔音聲母，也就是零聲母。潮州話聲母共十八個，包括零聲母。

== 韻母 ==
[[四呼|四呼]]當中，潮州話有開口、齊齒和合口呼，故此韻頭（介音）有[i]和[u]，但並沒有撮口呼。

韻腹是唯一必須存在的音節元素，故此也是音節最重要的部分。它可以是[[元音|元音]]、[[鼻化元音|鼻化元音]]或成節[[輔音|輔音]]。除了饶平北部和潮安凤凰镇，潮州話普遍沒有[[韻尾|韻尾]][n]/[t]，下列拼音中以n為尾綴的韻母其實表示[[鼻化|鼻化]]韻。元音[[韻腹|韻腹]]有四個，分別是[a]、[o]、[e]和[ɯ]。絕大部份音節以元音為韻腹，但有一些音節不含元音，而是成節輔音[[雙唇鼻音|雙唇鼻音]][m]和[[軟齶鼻音|軟齶鼻音]]{{IPA|[ŋ]}}。

韻尾通常是[[塞音|塞音]]或[[鼻音_(語言)|鼻音]]（實際上也可說是塞音，因為軟顎降低，讓空氣通過鼻腔），但是並非必要。元音韻尾共三個，分別是[i]、[o]和[u]。輔音韻尾共五個，包括塞音{{IPA|[ʔ]}}、[p]和[k]，鼻音[m]和{{IPA|[ŋ]}}，以及存在于饶平北部和潮安凤凰镇的[n]和[t]。

== 声调 ==
潮州话跟其他汉语方言一样，是[[声调语言|声调语言]]。它有八个[[声调|声调]]<ref name="fangan"/>，并有丰富的[[变调|变调]]。

調值之數字是指[[聲調輪廓|聲調輪廓]]，1為最低，5為最高；拼音中的聲調以1至8表示，置於音节右上方：
