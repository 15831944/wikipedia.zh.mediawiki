'''潮站'''（{{jpn|j=潮駅|hg=うしおえき|rm=Ushio eki|f=y}}）位於[[島根縣|島根縣]][[邑智郡|邑智郡]][[美鄉町_(島根縣)|美鄉町]]潮村，是[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）[[三江線|三江線]]的一個[[鐵路車站|鐵路車站]]。

== 歷史 ==
* 1975年（[[昭和|昭和]]50年）8月31日正式開業。
* 1987年（昭和62年）4月1日因[[國鐵分割民營化|國鐵分割民營化]]交由[[西日本旅客鐵道|西日本旅客鐵道]]營運。
* 2018年（[[平成|平成]]30年）4月1日與三江線廢線同時關閉。

== 結構 ==
[[File:三江線探索2_047.jpg|thumb]]
該站是一個在往三次方向的左邊設有1面1綫的[[車站月台|月台]]的[[地面車站|地面車站]]（[[停車場_(鐵路)#日本JR定義|停留所]]）。這個[[招呼站|無人站]]的月臺上設有等候室（由{{link-ja|濱田鐵路部|浜田鉄道部}}管理），但沒有自動販賣機等設備<ref>{{Cite web|url=http://www.jr-odekake.net/eki/top.php?id=0642518|title=潮駅│駅情報：JRおでかけネット|publisher=[[西日本旅客鐵道|西日本旅客鐵道]]}}</ref>。

== 周邊 ==
該站旁邊有[[江之川|江之川]]。

* {{link-ja|潮溫泉|潮温泉}}
* [[中國電力_(日本)|中國電力]]{{link-ja|來島水壩|来島ダム|潮發電所}}

== 使用情況 ==
該站2014年度一日平均乘車人次不足一人。另外於2004年度平均人次為七人，1994年度為十一人，而1984年度則為十六人。
