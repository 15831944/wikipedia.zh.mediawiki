'''潮音橋'''为位於[[中国|中国]][[浙江省|浙江省]][[湖州市|湖州市]][[吴兴区|吴兴区]][[朝阳街道_(湖州市)|朝阳街道]]南街中段、原[[湖州城池|湖州府城]]南门定安门内的一座三孔石拱橋，东西向跨[[霅溪|霅溪]]，始建於明[[嘉靖|嘉靖]]十八年（1539），[[萬曆|萬曆]]三十三年（1605）重建。

== 歷史 ==
此地原有渡口「潮音渡」<ref group="注">潮音渡一名既因渡口风大浪急，有如潮音，又因河东旧有慈感寺供奉观音菩萨，与普陀山的观音与潮音洞相联系。</ref>，桥因之而得名。两侧原有东西渡亭以蔽风雨，建桥后仍保留，今已不存<ref>清·罗愫、杭世骏等纂修，《光绪乌程县志·卷十一·津梁》：潮音桥，舊爲渡，在慈感寺前，以近奉觀音大士，故名。两岸皆有渡亭，嘉靖十八年知府魏济建桥，万历三十年倾，知府陈幼学重建，改亭为阁。崇正三年东亭毁，知府金廷璧葺。本朝乾隆八年八月西亭毁，南仓三井二铺居民重建。……沈儆炌《潮音桥两亭记》：“湖郡定安门纳南来诸水，灌注入城，深广病涉，故设渡以利往来，设东西亭以蔽待渡者风雨，西亭列石塔，输运快便，势如转输。嘉靖己亥郡侯魏公更渡为桥，万历乙巳圮，郡侯筠塘陈公鼎新此桥，而亭特加以层楼，峩峩对峙，几三十年。”</ref><ref name="湖州日报">{{cite web |url=http://www.zjww.gov.cn/culture/2011-06-17/330629166.shtml|title=《湖州桥文化：长虹卧波潮音桥》|accessdate=2014-04-09|work=沈鑫元|publisher=湖州日报}}</ref>。

== 構造 ==
此橋高7米，長50米，寬5米，橋兩側置石質[[吳王靠|吳王靠]]，可供行人坐息，端部各置一对石狮，桥外侧石缝中原生有[[枸杞|枸杞]]、[[石榴|石榴]]，寓意“长留人间”，2004年重修时清除<ref name="湖州日报" />。舊時橋西岸河街商店林立，西側次孔跨一人行道，以避免割斷沿河街道，橋上橋下均可通行人，謂之「橋里橋」<ref>{{cite web|url=http://wx.hz66.com/content.asp?Id=42892|title=《吴兴瑰宝之潮音桥》|accessdate=2014-04-10|work=|publisher=吴兴在线}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。因潮音橋是拱橋，不能行駛車輛，后于橋南侧新建公路橋，橋西堍直對車站路。
