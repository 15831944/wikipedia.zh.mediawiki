'''潮王路站'''位于杭州市下城区潮王路河东路，是[[杭州地铁3号线|杭州地铁3号线]]的地下车站。预计2021年开通。<ref>{{cite web |title=近距离曝光！杭州地铁3号线24个站点具体位置图来了 |url=https://www.sohu.com/a/213313794_100032554 |website=搜狐}}</ref>

== 结构 ==
潮王路站位于潮王路与河东路交叉口北侧，沿河东路南北向布置，车站长490.7m，宽21.75m，为地下二层岛式车站，设有一股存车线，有效站台宽度12.6米，总建筑面积27802平方米。<ref>{{cite web |title=杭州地铁3号线站点有哪些 |url=http://hz.bendibao.com/traffic/2014513/42452.shtm |website=杭州本地宝}}</ref>
