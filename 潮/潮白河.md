'''潮白河'''，是[[北京市|北京地区]]的一条河流，属海河水系，长458公里。

素有“北京莱茵河”之称的潮白河，是京东的第一大河，被誉为[[顺义|顺义]]的母亲河。流经顺义32公里，水域面积14.8平方公里，河面宽度平均500米，最宽水面达800米，河道平均水深2.5米。已建成5闸8桥，实现了五级梯级蓄水，水深调控自如。

==歷史==
[[清朝|清朝]]末年以前，潮白河在通州同[[温榆河|温榆河]]交汇，为[[北运河|北运河]]上源。光绪十二年（1882年）潮白河在通州平家疃漫口，入[[箭杆河|箭杆河]]。1916年北洋政府在顺义苏庄建拦河闸为北运河补水，当年即被冲毁。1922至1925年又在苏庄建拦河闸，1939年新闸又被冲毁。自此潮白河夺箭杆河，不再入北运河。<ref>{{cite book|author=北京市地方志纂委员会|title=北京志·地质矿产水利气象卷·水利志|location=北京|publisher=北京出版社|year=2000|pages=156-157|isbn=7-200-04180-7}}</ref>

== 水利设施 ==
向阳闸，河南村橡胶闸坝，柳各庄和苏庄两座橡胶坝。

== 流经方向 ==
发源于[[河北省|河北省]][[沽源县|沽源县]]，经[[赤城县|赤城县]]，于白河堡进入[[北京市|北京市]][[延庆县|延庆县]]境，东流经[[怀柔区|怀柔区]]青石岭入[[密云县|密云县]]，沿途有黑河、汤河、白马关河等支流汇入，在张家坟附近注入[[密云水库|密云水库]]。潮、白两河出库后，各自排放故道，于密云县城之西南的河漕村汇合后称潮白河。南流，经怀柔区、[[顺义区|顺义区]]，至[[通州区|通州区]]，沿途有支流[[怀河|怀河]]、[[箭杆河|箭杆河]]来汇，于通州区牛牧屯出本市入河北省境，东流汇入[[海河|海河]]而注[[渤海|渤海]]。

潮白河及其支流组成潮白河水系，本境流域面积6531平方公里，占全市面积33.4％，年均径流量10.22亿立方米，占全市水系总径流量的39.4％。两项均居全市首位。解放后，在这一水系上修建了[[密云水库|密云水库]]、[[怀柔水库|怀柔水库]]2座大型水库和5座中型水库、33座小型水库，总库容46.6867亿立方米，亦居全市各水系首位；以密云水库为引水源，通向市区的[[京密引水渠|京密引水渠]]为北京市供水主动脉。潮河、白河的河道在历史上曾多次改道。[[东汉|东汉]]以前，潮河、白河各自入海；[[北魏|北魏]]时，在今[[通州区|通州区]]东北（潞县）汇合；以后，汇流点逐步向北迁移，至[[五代|五代]]在今[[顺义区|顺义区]][[牛栏山地区|牛栏山地区]]汇流；[[明朝|明]][[嘉靖|嘉靖]]三十四年（1555年），为利用潮白河通漕，经人工治理，始于[[密云县|密云县]]西南18里之河漕村汇流合。
