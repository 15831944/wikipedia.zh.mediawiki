'''潮风电台'''（{{Lang-ja|しおかぜ}}，{{Lang-en|Shiokaze}}）是[[日本|日本]]的一個[[廣播電台|廣播電台]]，由民间团体'''特定失踪者问题调查会'''主要向[[朝鲜民主主义人民共和国|-{zh-hans:朝鲜;zh-hant:北韓}-]]进行[[短波|短波]]广播。

== 概要 ==
2005年10月开始向-{zh-hans:朝鲜;zh-hant:北韓}-广播，主要广播有关失踪的日本人的信息。后来由于受到-{zh-hans:朝鲜;zh-hant:北韓}-的干扰，不得不频繁更换频率。

== 广播语言 ==
* 第1放送（[[日语|日语]]）
* 第2放送（日语、[[朝鲜语|-{zh-hans:朝鲜;zh-hant:韓}-语]]、[[英语|英语]]、[[汉语|汉语]]）

== 历史 ==
※时间为日本时间
=== 初次建立时期 ===
* 2005年10月31日 - 于0:30在5890kHz开始广播。
* 2005年11月1日 - 广播时间在23:30开始
* 2005年12月8日 - 增加了广播时间至90分钟。
* 2006年1月30日 - [[汉语|汉语]]（呼号：'''潮风'''，日语表记：'''チャオフォン'''）、[[朝鲜语|-{zh-hans:朝鲜;zh-hant:韓}-语]]（呼号：'''바닷바람'''，日语表记：'''パダッパラム'''）广播开始。
* 2006年5月5日 - -{zh-hans:朝鲜;zh-hant:北韓}-开始干扰潮风电台。
* 2006年5月9日 - [[内阁官房长官|内阁官房长官]][[安倍晋三|安倍晋三]]在记者见面会上陈述了关于-{zh-hans:朝鲜;zh-hant:北韓}-干扰潮风电台的情况。

== -{zh-hans:朝鲜;zh-hant:北韓}-的干扰问题 ==
2007年3月29日和3月30日在第1放送的6045kHz均存在来自-{zh-hans:朝鲜;zh-hant:北韓}-的干扰，这次事件在[[朝日新闻|朝日新闻]]和[[产经新闻|产经新闻]]中均有所报道。经检测，干扰源来自平壤附近，[[国际电信联盟|国际电信联盟]]指出-{zh-hans:朝鲜;zh-hant:北韓}-的此举违反了相关规定。而2007年11月2日<ref>[http://warp.ndl.go.jp/info:ndljp/pid/286922/www.soumu.go.jp/menu_news/s-news/2007/071102_2.html {{lang|ja|「しおかぜ」への影響が懸念される電波への対応}}] {{lang|ja|総務省 報道資料 平成19年11月2日}}</ref> 和2008年4月4日<ref>[http://warp.ndl.go.jp/info:ndljp/pid/286922/www.soumu.go.jp/menu_news/s-news/2008/080404_13.html {{lang|ja|「しおかぜ」への影響が懸念される電波への対応}}] {{lang|ja|総務省 報道資料 平成20年4月4日}}</ref> 的干扰迫使了潮风电台开始从此频繁更换频率。

== 广播目的 ==
# 向-{zh-hans:朝鲜;zh-hant:北韓}-民众及时提供外界信息
# 努力营救被-{zh-hans:朝鲜;zh-hant:北韓}-绑架的日本人

== 发射台 ==
潮风电台在[[八俣发射台|八俣发射台]]使用100千瓦的短波发射机对-{zh-hans:朝鲜;zh-hant:北韓}-进行广播。不过，目前-{zh-hans:朝鲜;zh-hant:北韓}-经常对这个电台进行干扰，在朝鲜及韩国很难清晰收到该电台信号。
