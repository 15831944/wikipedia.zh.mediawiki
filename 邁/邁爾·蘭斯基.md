'''邁爾·蘭斯基'''（{{lang-en|'''Meyer Lansky'''}}，本名{{lang|pl|'''Meier Suchowlański'''}}，{{bd|1902年|7月4日|1983年|1月15日|catIdx=Lansky, Meyer}}），被稱為是「黑幫會計師」（Mob's Accountant），是一名重要的有組織犯罪人物，他與他的伙伴[[查理·盧西安諾|查理·「幸運」·盧西安諾]]一起協助美國的[[全國犯罪集團|全國犯罪集團]]發展。<ref name=obit>{{Cite web |url = http://www.nytimes.com/1983/01/16/obituaries/meyer-lansky-is-dead-at-81-financial-wizard-of-organized-crime.html |title = MEYER LANSKY IS DEAD AT 81 |date = 1983-01-16 |publisher = The New York Times |language = en}}</ref>

蘭斯基發展了一個橫跨世界的賭博帝國。據說他在[[拉斯維加斯|拉斯維加斯]]、[[古巴|古巴]]、[[巴哈馬|巴哈馬]]和[[倫敦|倫敦]]的賭場擁有點數（利潤分成）。儘管是{{link-en|猶太裔美國有組織犯罪|Jewish-American organized crime|猶太黑幫}}的成員，蘭斯基無疑對意大利黑手黨有很大的影響力，並且在鞏固的犯罪黑社會中擔當著一個很大的角色，不過這一角色的整個程度上已經成為一些爭論的話題，像他本人否認了許多對他的指控。<ref>{{Cite web |url = https://www.youtube.com/watch?v=1UNJTPtmZhI |title = Meyer Lansky Interview 1971 |publisher = YouTube |language = en}}</ref>

儘管近五十年來作為一名[[黑社會|有組織犯罪]]的成員參與者<ref>{{Cite web |url = https://vault.fbi.gov/meyer-lansky |title = Meyer Lansky |publisher = vault.fbi.gov |language = en}}</ref>，蘭斯基從未被判犯有比非法賭博更嚴重的罪行。他有一份成為美國歷史上財政最成功的黑幫之一的遺產。在邁爾·蘭斯基逃離古巴之前，據說他估計有2000萬美元（2017年為1.84億美元）。可是，當他在1983年去世時，他的家人感到震驚，他的遺產價值只有大約57,000美元。<ref>{{Cite web |url = http://www.americanmafia.com/Feature_Articles_331.html |title = The Lost Journals of Meyer Lansky |author = Gary Cohen |publisher = AmericanMafia.com |language = en}}</ref>

== 生平 ==
生於[[俄羅斯帝國|俄羅斯帝國]][[格罗德诺|格罗德诺]]，出身於[[波蘭猶太人|波蘭猶太人]]家庭。1911年，經由[[敖德薩|敖德薩]]，移民到美國[[紐約市|紐約市]]。

1983年1月15日，因[[肺癌|肺癌]]過世，享年80歲。
