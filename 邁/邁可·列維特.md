'''邁可·列維特'''，[[皇家学会#院士|FRS]]（{{lang-en|'''Michael Levitt'''}}，{{bd|1947年|5月9日||catIdx=Levitt, Micheal}}），出生於[[南非|南非]]，[[犹太|犹太]]裔[[美国|美国]]、[[英國|英國]]、[[以色列|以色列]]籍[[生物物理學|生物物理學家]]，現任美國[[史丹佛大學|史丹佛大學]][[結構生物學|結構生物學]]教授<ref>{{cite web |url=http://csb.stanford.edu/ |title=存档副本 |accessdate=2017-03-22 |0=2016-03-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20160325231210/http://csb.stanford.edu/ |archivedate=2016-03-25 }} Levitt Lab website</ref><ref>{{cite web |url=http://csb.stanford.edu/levitt/ |title=存档副本 |accessdate=2010-07-15 |0=2010-07-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20100715200937/http://csb.stanford.edu/levitt/ |archivedate=2010-07-15 }} Lab Website Profile Page</ref>，研究專長為[[計算生物學|計算生物學]]<ref>{{cite journal|url=http://www.nature.com/doifinder/10.1038/87545|pages=392–393|journal=Nature Structural Biology|volume=8|issue=5|title=The birth of computational structural biology|date=2001-05-01|language=En|accessdate=2018-04-02|doi=10.1038/87545|author=Michael Levitt}}</ref>，也是[[美國國家科學院|美國國家科學院]][[院士|院士]]。

列維特與[[馬丁·卡普拉斯|馬丁·卡普拉斯]]、[[阿里耶·瓦舍爾|阿里耶·瓦舍爾]]，因「為複雜化學系統創造了多尺度模型」，共同獲得2013年[[諾貝爾化學獎|諾貝爾化學獎]]。<ref>{{cite press release
