'''邁克·沃格爾'''（{{lang-en|'''Mike Vogel'''}}，本名'''Michael James Vogel'''，{{bd|1979年|7月17日}}）<ref>{{Cite web |url = http://mike-vogel.com/about-mike/biography/ |title = BIOGRAPHY |publisher = mike-vogel.com |language = en}}</ref>是[[美國|美國]]的一位[[演員|演員]]，之前也曾是[[模特兒|模特兒]]。他從2001年開始從事演藝事業。他出演過多部電影和電視劇，包括《[[海神號_(2006年電影)|海神號]]》、《[[有人喜歡藍|有人喜歡藍]]》、《[[貝茲旅館|貝茲旅館]]》、《[[末世凶煞|末世凶煞]]》、《[[天幕圍城|天幕圍城]]》<ref>{{Cite web |url = https://www.hollywoodreporter.com/live-feed/under-dome-cbs-cast-mike-vogel-pan-am-421346 |title = CBS' 'Under the Dome' Enlists 'Pan Am's' Mike Vogel |author = Lesley Goldberg |date = 2013-02-13 |publisher = The Hollywood Reporter |language = en}}</ref>等。
