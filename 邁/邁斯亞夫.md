'''邁斯亞夫（'''{{Lang-ar|مصياف}}，转写：{{Transl|ar|DIN|Miṣyāf}}）是[[敘利亞|敘利亞]]的城鎮，由[[哈馬省|哈馬省]]負責管轄，位於該國西部。是[[迈斯亚夫区|迈斯亚夫区]]的行政中心，距離叙利亚首府[[哈馬|哈馬]]40公里，海拔高度447米。据[[叙利亚中央统计局|叙利亚中央统计局]]的统计结果，2009年马西亚夫拥有人口37,109，绝大多数是[[什叶派|什叶派穆斯林]]。尽管1940年代[[伊斯玛仪派|伊斯玛仪派]]曾占主导地位，现如今也有很多[[阿拉维派|阿拉维派]]居民。<ref>{{Citation|first=Nicholas A.|last=Heras|title=The Potential for an Assad Statelet in Syria|url=https://www.washingtoninstitute.org/uploads/Documents/pubs/PolicyFocus132_Heras4.pdf|publisher=The Washington Institute of Near East Policy|page=25|date=2013年12月|accessdate=2015年11月26日}}</ref>

迈斯亚夫以其[[中世纪|中世纪]]的要塞著称，迈斯亚夫堡曾经是著名的[[暗杀|刺客]]组织[[阿萨辛派|阿萨辛派]]的总部。

== 名称 ==
自从该城市被[[穆斯林|穆斯林]]统治以来，当地居民对这座城市有多个不同的阿拉伯语称呼，诸如Maṣyaf（迈斯亚夫），Maṣyat（迈斯亚特），Maṣyad（迈斯亚德）。这些称呼来源于该城市的[[亚述|亚述]]名字Manṣuate（曼苏亚特）。[[东方学家|东方学家]]{{Internal link helper/en|爱德华·利平斯基|Edward Lipinsky}}认为亚述名字Manṣuate中的“nṣw”相当于阿拉伯语的“nṣṣ”，是“建立”的意思。爱德华也认为亚述名字来源于亚述词语Manṣuwatu，是“建立起的平台”的意思，指代迈斯亚夫堡。<ref>{{Cite book|title = The Aramaeans: Their Ancient History, Culture, Religion|last = Lipinsky|first = Edward|publisher = Peeters Publishers|year = 2000|isbn = 9042908599|location = |pages = 306-308|url = https://books.google.com/books?id=rrMKKtiBBI4C}}</ref>

== 历史 ==

=== 早期历史 ===
迈斯亚夫很有可能曾经是古代[[阿拉米人|阿拉米人]]于公元前8世纪建立的城市Mansuate（曼苏亚特），后来亚述人把该城市作为曼苏亚特省（现叙利亚中部）的行政中心继续使用。<ref>{{Cite book|title = The Aramaeans: Their Ancient History, Culture, Religion|last = Lipinsky|first = Edward|publisher = Peeters Publishers|year = 2000|isbn = 9042908599|location = |pages = 309|url = https://books.google.com/books?id=rrMKKtiBBI4C}}</ref>迈斯亚夫也有可能曾经是[[羅馬帝國|罗马人]]和[[拜占庭帝国|拜占庭人]]的历史学家提到过的城市Marsyas（马西亚斯）。<ref name=":0">{{Cite book|title = The Citadel of Masyaf|last = Hasan|first = Haytham|publisher = The Aga Khan Trust for Culture|year = 2008|isbn = 9782940212064|location = |pages = 7|url = http://archnet.org/system/publications/contents/5214/original/DPC1951.pdf?1384788768}}</ref>

最初记载迈斯亚夫以及它的要塞的是1099年的十字军编年史，<ref name=":1">{{Cite book|title = The Encyclopaedia of Islam: Fascicules 111–112 : Masrah Mawlid|last = Honigman|first = E.|publisher = BRILL|year = 1989|isbn = 9789004092396|location = |pages = 789|url = https://books.google.com/books?id=PvwUAAAAIAAJ|editor-last = Bosworth|editor-first = Clifford Edward|chapter = Masyad}}</ref>然而迈斯亚夫11世纪前就已存在。很有可能是{{Internal link helper/en|Hamdanid王朝|Hamdanid dynasty}}最开始在该地建造了一个用来看守山路据点。<ref name=":0">{{Cite book|title = The Citadel of Masyaf|last = Hasan|first = Haytham|publisher = The Aga Khan Trust for Culture|year = 2008|isbn = 9782940212064|location = |pages = 7|url = http://archnet.org/system/publications/contents/5214/original/DPC1951.pdf?1384788768}}</ref>随后迈斯亚夫被[[法蒂玛王朝|法蒂玛王朝]]统治。公元999年的秋天，[[拜占庭帝国|拜占庭]]皇帝[[巴西尔二世|巴西尔二世]]在征服[[安提阿|安提阿]]的途中曾经摧毁过迈斯亚夫堡。<ref name=":1" />

随后[[塞尔柱帝国|塞尔柱帝国]]统治了该地区。1099年，十字军在占领[[的黎波里_(黎巴嫩)|的黎波里]]后继续进攻迈斯亚夫，统治[[大马士革|大马士革]]的{{Internal link helper/en|Tughtakin|Tughtakin}}率兵反抗，并与十字军达成了一个短暂的协议：迈斯亚夫和Alhsn继续归穆斯林所有，然而穆斯林必须对十字军上贡。随后{{Internal link helper/en|Mirdasid王朝|Mirdasid dynasty}}统治了该地区。<ref name=":1" />1127年，Mirdasid人将它卖给了Banu Munqidh家族。<ref name=":2">{{Cite book|title = Historical Dictionary of the Ismailis|last = Daftary|first = Farid|publisher = Scarecrow Press|year = 2011|isbn = 9780810879706|location = |pages = 115}}</ref>

=== 阿萨辛派统治时期 ===
[[Image:Masyaf_-_Gesamtansicht.jpg|thumb]]
1140年，[[伊斯玛仪派|伊斯玛仪派]]的一个分支，著名的刺客组织[[阿薩辛派|阿萨辛派]]统治了迈斯亚夫。一部分阿萨辛派离开他们的总部[[阿剌模忒堡|阿剌模忒堡]]（现伊朗境内），并选中了叙利亚作为新的定居地，并成功来到[[阿勒颇|阿勒颇]]和[[大马士革|大马士革]]。但由于阿萨辛派对于十字军和穆斯林的立场左右不定，导致当地政府以及被神职人员煽动的当地居民将他们视为异端并加以迫害以及屠杀。1139年，阿萨辛派在大马士革遭到肃清，并最终离开城市选择了在沿海的山区以及其中星罗棋布的要塞中定居了下来，其中包括迈斯亚夫。依靠着迈斯亚夫堡以及其他要塞，他们与当地的穆斯林[[阿拔斯王朝|阿拔斯王朝]]结盟并抵抗十字军的入侵。1162年[[阿薩辛派|阿萨辛派]][[拉希德丁·锡南|派拉希德丁·锡南]]来到迈斯亚夫指导阿萨辛派叙利亚支部的工作。<ref>{{Cite book|title = The Eagle's Nest: Ismaili Castles in Iran and Syria|last = Willey|first = Peter|publisher = I.B.Tauris|year = 2005|isbn = 9781850434641|location = |pages = 42-46|url = https://books.google.com/books?id=RTyTn4ErwRIC}}</ref>

1152年，阿萨辛派刺杀[[的黎波里伯国|的黎波里伯国]]的君主{{Internal link helper/en|雷蒙德二世|Raymond II, Count of Tripoli}}，[[赞吉王朝|赞吉王朝]]认为阿萨辛派的威胁比十字军更大，于是转而进攻迈斯亚夫和阿萨辛派的其他要塞，但被击退。为了对抗赞吉王朝，拉希德丁于1173年派人前往[[耶路撒冷王國|耶路撒冷王国]]，同其国王[[阿马尔里克一世|阿马尔里克一世]]结为同盟。不久之后，法蒂玛王朝被萨拉丁推翻，[[阿尤布王朝|阿尤布王朝]]成立。1170年代中期，阿尤布王朝的苏丹萨拉丁率军进攻叙利亚，试图让[[遜尼派|逊尼派]]统治穆斯林世界。拉希德丁把阿尤布王朝视为比十字军更大的威胁，转而同赞吉王朝结盟抵抗萨拉丁。<ref>{{Cite book|title = The Eagle's Nest: Ismaili Castles in Iran and Syria|last = Willey|first = Peter|publisher = I.B.Tauris|year = 2005|isbn = 9781850434641|location = |pages = 47|url = https://books.google.com/books?id=RTyTn4ErwRIC}}</ref>

从1176年开始，拉希德丁两次派人暗杀萨拉丁，但都失败了。出于报复，萨拉丁率兵围攻迈斯亚夫。几天后，由于接到[[第三次十字军东征|十字军再次东征]]并进攻阿尤布王朝的紧急信息，萨拉丁选择了撤退。随后在萨拉丁的叔叔的斡旋下，拉德希丁与萨拉丁结盟共同抵抗十字军。

1249年，阿萨辛派为迈斯亚夫城镇的外围修建了一道围墙。1256年，因阿萨辛派曾经试图刺杀[[蒙古帝国|蒙古帝国]]大汗[[蒙哥|蒙哥]]，[[旭烈兀|旭烈兀]]在[[旭烈兀西征|蒙古西征]]途中派兵进攻迈斯亚夫，阿萨辛派最终于1256年11月19日投降。<ref name=":3">{{Cite book|title = The Encyclopaedia of Islam: Fascicules 111–112 : Masrah Mawlid|last = Honigman|first = E.|publisher = BRILL|year = 1989|isbn = 9789004092396|location = |pages = 790|editor-last = Bosworth|editor-first = Clifford Edward|chapter = Masyad|url = https://books.google.com/books?id=PvwUAAAAIAAJ}}</ref>

=== 马木留克王朝统治时期 ===
1260年，由于[[阿音札鲁特战役|阿音扎鲁特战役]]的失败，蒙古人撤离迈斯亚夫，[[馬木留克王朝|马木留克王朝]]占领了此地。1262年，马木留克的苏丹[[拜巴尔一世|拜巴尔一世]]让萨利马丁·穆巴拉克代替那马丁·伊斯玛尔统治此地，然而随后穆巴拉克被拜巴尔囚禁在开罗，那马丁恢复了对迈斯亚夫的统治。1270年，迈斯亚夫直接划归马木留克王朝管辖。<ref name=":3">{{Cite book|title = The Encyclopaedia of Islam: Fascicules 111–112 : Masrah Mawlid|last = Honigman|first = E.|publisher = BRILL|year = 1989|isbn = 9789004092396|location = |pages = 790|editor-last = Bosworth|editor-first = Clifford Edward|chapter = Masyad|url = https://books.google.com/books?id=PvwUAAAAIAAJ}}</ref>而伊斯玛仪派则继续在马木留克王朝的统治下居住在此地。<ref name=":2">{{Cite book|title = Historical Dictionary of the Ismailis|last = Daftary|first = Farid|publisher = Scarecrow Press|year = 2011|isbn = 9780810879706|location = |pages = 115}}</ref>迈斯亚夫成为了马木留克王朝官道的重要驿站和战略要冲，迈斯亚夫的将军直接听命于苏丹。

1320年，阿尤布王朝哈马的统治者以及历史学家[[阿布·菲達|阿布·菲达]]指出迈斯亚夫是“伊斯玛仪派教义的中心”，花园和喷泉都让它看起来非常“美丽”。旅行家[[伊本·白图泰|伊本·白图泰]]在1355年路过此地时，迈斯亚夫隶属于的黎波里省的一个区，区内还有{{Link-en|Al-Kahf|Al-Kahf_Castle}}、{{Link-en|Al-Qadmus|Al-Qadmus}}、{{Internal link helper/en|Al-Ulayqa|Aleika_Castle}}等要塞城镇。随后迈斯亚夫被划入大马士革省。15世纪中叶，苏丹[[巴尔斯拜|巴尔斯拜]]建立了一条连接迈斯亚夫和的黎波里的道路，然而官道不再经过此地。1446年，历史学家Khalil al-Zahiri形容迈斯亚夫是“一个农民的城镇，有很大的郊区”。<ref name=":4">{{Cite book|title = The Encyclopaedia of Islam: Fascicules 111–112 : Masrah Mawlid|last = Honigman|first = E.|publisher = BRILL|year = 1989|isbn = 9789004092396|location = |pages = 791|editor-last = Bosworth|editor-first = Clifford Edward|chapter = Masyad|url = https://books.google.com/books?id=PvwUAAAAIAAJ}}</ref>

=== 奥斯曼帝国统治时期 ===
[[File:PAB_Massyaf_Chateau_de_Vieux_de_la_Montagne.jpg|thumb]]

1516到1517年，[[奥斯曼帝国|奥斯曼帝国]]在战胜马木留克王朝后占领了叙利亚，迈斯亚夫被划归[[霍姆斯省|霍姆斯省]]管辖。迈斯亚夫和其他要塞城镇被要求交纳额外的税款。城镇的管理者被称之为“可汗”，并需要向奥斯曼帝国缴纳一定的费用，随后在16世纪中叶，该费用被取消。据[[苏菲派|苏菲派]]旅行家{{Internal link helper/en|Abd al-Ghani al-Nabulsi|Abd al-Ghani al-Nabulsi}}描述，1690年代前期，迈斯亚夫的统治者是阿拉伯酋长苏莱曼的后裔。<ref name=":4">{{Cite book|title = The Encyclopaedia of Islam: Fascicules 111–112 : Masrah Mawlid|last = Honigman|first = E.|publisher = BRILL|year = 1989|isbn = 9789004092396|location = |pages = 791|editor-last = Bosworth|editor-first = Clifford Edward|chapter = Masyad|url = https://books.google.com/books?id=PvwUAAAAIAAJ}}</ref>

1703年，阿拉维派控制了迈斯亚夫，8年后，在奥斯曼帝国当局的干预下，伊斯玛仪派重新获得统治。<ref>{{Cite book|title = Syrian Ismailism: The Ever Living Line of the Imamate, AD 1100–1260|last = Mirza|first = Nasser Ahmad|publisher = Psychology Press|year = 1997|isbn = 9780700705054|location = |pages = 15|url = https://books.google.com/books?id=Qiyy6skgPfoC}}</ref>1788年，迈斯亚夫的统治者穆斯塔法·伊本·伊德里斯修建了一个洗礼池，供自己以及其他伊斯玛仪派穆斯林使用。1808年，阿拉维派进攻迈斯亚夫，杀掉了伊斯玛仪派领导人穆斯塔法·米尔希姆和他的儿子，并占领了要塞，同时有大约300个伊斯玛仪派居民丧生。<ref name=":4">{{Cite book|title = The Encyclopaedia of Islam: Fascicules 111–112 : Masrah Mawlid|last = Honigman|first = E.|publisher = BRILL|year = 1989|isbn = 9789004092396|location = |pages = 791|editor-last = Bosworth|editor-first = Clifford Edward|chapter = Masyad|url = https://books.google.com/books?id=PvwUAAAAIAAJ}}</ref>迈斯亚夫的宗教领袖和其他人则逃往了霍姆斯和其他叙利亚中部地区。<ref>{{Cite book|title = The Isma'ilis: Their History and Doctrines|last = Daftary|first = Farid|publisher = Cambridge University Press|year = 2007|isbn = 9781139465786|location = |pages = 489|url = https://books.google.com/books?id=cSO9zh61AGEC}}</ref>奥斯曼帝国的大马士革省长{{Internal link helper/en|昆吉·尤瑟夫·帕夏|Kunj Yusuf Pasha}}介入此事，派出了4000到5000名士兵并试图重新夺取了城镇。在历经3个月的展透后，阿拉维派投降，伊斯玛仪派重新控制了城市并且在1810年重新修建了要塞。<ref name=":4">{{Cite book|title = The Encyclopaedia of Islam: Fascicules 111–112 : Masrah Mawlid|last = Honigman|first = E.|publisher = BRILL|year = 1989|isbn = 9789004092396|location = |pages = 791|editor-last = Bosworth|editor-first = Clifford Edward|chapter = Masyad|url = https://books.google.com/books?id=PvwUAAAAIAAJ}}</ref>

在伊斯玛仪派重新进入迈斯亚夫的2年后，瑞士旅行家{{Internal link helper/en|约翰·路德维希·伯克哈特|Johann Ludwig Burckhardt}}记录城镇有大约280户居民，其中绝大多数是伊斯玛仪派穆斯林，30户是[[基督徒|基督徒]]。伯克哈特注意到城镇的[[教堂|教堂]]，[[清真寺|清真寺]]和很多房屋都在之前的战争中遭到严重的破坏。直到19世纪结束，迈斯亚夫的人口持续的减少。<ref name=":4">{{Cite book|title = The Encyclopaedia of Islam: Fascicules 111–112 : Masrah Mawlid|last = Honigman|first = E.|publisher = BRILL|year = 1989|isbn = 9789004092396|location = |pages = 791|editor-last = Bosworth|editor-first = Clifford Edward|chapter = Masyad|url = https://books.google.com/books?id=PvwUAAAAIAAJ}}</ref>

=== 现代 ===
1917年，[[同盟国_(第一次世界大战)|盟军]]从奥斯曼帝国中夺取叙利亚。英国将军[[托马斯·爱德华·劳伦斯|托马斯·爱德华·劳伦斯]]曾经在迈斯亚夫休息，并记录到当时迈斯亚夫被当作[[貧民窟|贫民窟]]使用。<ref name=":5">{{Cite book|title = The Citadel of Masyaf|last = Hasan|first = Haytham|publisher = The Aga Khan Trust for Culture|year = 2008|isbn = 9782940212064|location = |pages = 15|url = http://archnet.org/system/publications/contents/5214/original/DPC1951.pdf?1384788768}}</ref>在[[法屬敘利亞託管地|法国治理叙利亚]]时期，迈斯亚夫被从阿拉维区域分开，划入大马士革区域管辖。然而在1929年，一些阿拉维派居住地又被划归阿拉维区域，随后在1936年融入叙利亚其他部分。1939年，迈斯亚夫由哈马省改为划归[[拉塔基亚省|拉塔基亚省]]管辖。<ref name=":6">{{Cite book|title = The Encyclopaedia of Islam: Fascicules 111–112 : Masrah Mawlid|last = Honigman|first = E.|publisher = BRILL|year = 1989|isbn = 9789004092396|location = |pages = 792|url = https://books.google.com/books?id=PvwUAAAAIAAJ|editor-last = Clifford|editor-first = Edward Bosworth|chapter = Masyad}}</ref>在1930年代，当地居民依旧居住在要塞的房间中。<ref name=":5" />

1947年，叙利亚从法国独立的第二年，迈斯亚夫记录了有3808名伊斯玛仪派穆斯林居住。<ref>{{Cite book|title = La région alaouite et le pouvoir syrien|last = Balanche|first = Fabrice|publisher = Karthala Editions|year = 2006|isbn = 2845868189|location = |pages = 92|url = http://tel.archives-ouvertes.fr/docs/00/06/30/97/PDF/Les_Alaouites_l_espace_et_le_pouvoir.pdf|language = 法语}}</ref>1965年，叙利亚宣布迈斯亚夫为毛毯纺织中心，绝大多数雇佣者都为女性。<ref name=":6" />1970年，迈斯亚夫的城镇的大部分仍在城墙之内，然而到了1998年，城镇区域已经扩展到城墙之外了。<ref name=":6" />
