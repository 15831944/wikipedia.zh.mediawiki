'''邁阿密大師賽'''，是一項於每年的三月下旬在[[美國|美國]][[佛羅里達州|佛羅里達州]][[邁阿密|邁阿密]]舉行的[[網球|網球]]賽事，屬於[[ATP世界巡迴賽1000大師賽|ATP 1000大師賽]]和[[女子網球聯合會|WTA 皇冠賽]]級別，也是[[職業網球聯合會|職業網球聯合會]]和[[女子網球聯合會|女子網球聯合會]]所屬的比賽項目中等級最高的巡迴賽之一。賽事組織由著名的經紀公司[[IMG|IMG]]負責。

邁阿密大師賽自從舉辦以來，有不同的贊助商贊助過該項賽事，並以該贊助商的名字命名該年度的比賽。知名度較廣的有「[[立頓|立頓]]國際球員錦標賽」，「[[愛立信|愛立信]]公開賽」、「[[納斯達克|納斯達克-100]]公開賽」。2007年開始更換成「[[索尼愛立信|索尼愛立信]]公開賽」，2013年開始更換成「索尼網球公開賽」2015-19年改由[[伊塔烏聯合銀行|Itaú 銀行]]贊助。

邁阿密公開賽是僅次於四大滿貫之外規模最大、觀賽人數最多的巡迴賽之一。2010年超過30萬人次到網球中心觀看賽事，成為了四大滿貫之外觀看人數最多的巡迴賽。

== 歷史 ==
[[File:Nadal_Del_Potro_Miami_2009.jpg|thumb]]
邁阿密大師賽的前身是由網球運動員Butch Buchholz創辦的一項賽事，當初他的目標是將這項比賽打造成年度第一項大滿貫比賽(那時候的[[澳網|澳網]]是在每年的12月舉辦)，並給它取了個「冬季溫布頓」的外號。Butch Buchholz前後跟ATP與WTA接洽過，並且開出一系列條件，即提供比賽的獎金，開售比賽門票，在全世界範圍內提供15年的電視轉播權等。最終兩個賽事委員會都答應了他的請求。

第一屆的比賽在1985年[[佛羅里達州|佛羅里達州]][[德拉海灘|德拉海灘]]的拉沃國際網球娛樂中心舉行(Laver's International Tennis Resort)，並找來當年常年擔任[[溫布頓網球錦標賽|溫布頓網球錦標賽]]裁判的Alan Mills任職首席裁判長，同時聘請了有名的時尚設計師Ted Tinling，負責策劃賽事的舉辦與商標等設計。另外，當年主辦方為賽事提供了一百八十萬美元作為獎金，僅次於溫布頓和[[美網|美網]]，成為獎金第三高的網球賽事。<ref>{{cite web

1986年，賽事轉移到Boca West舉行，由於賽事的成功舉辦，隨後很多人士包括WTA總裁紛紛要求Butch Buchholz將比賽轉移至 Key Biscayne舉辦，也就是現在邁阿密大師賽舉行的地方。

1999年，Butch Buchholz本人將賽事的主辦權賣給了經紀公司[[IMG|IMG]]公司。<ref>[http://articles.sun-sentinel.com/1999-06-22/sports/9906220127_1_butch-buchholz-img-atp-tour Buchholz Is Selling, But Not Giving Up, Tournament He Founded]</ref>

== 賽事特點 ==
邁阿密大師賽是除了[[四大滿貫|四大滿貫]]之外，在單打比賽的主籤表中超過64名球員的比賽之一(另一站比賽為[[印第安維爾斯大師賽|印第安維爾斯大師賽]])，單打正賽的籤表為96籤，前32的種子享受首輪輪空的待遇，而雙打比賽則為32籤，主籤表在正賽開打的前幾天公佈，加上資格賽的賽程，整站比賽用12天的時間舉行。

2006年，邁阿密大師賽成為美國第一站將[[即時回放|即時回放]]（後期被[[鷹眼系統|鷹眼系統]]所替代）功能列入比賽規則的賽事，提供給選手用於挑戰邊線球是否出界，每個球員每一盤球有兩次的挑戰機會，另外在搶七局中可增加一次挑戰機會。

== 積分與獎金 ==
作為ATP 1000大師賽和WTA 皇冠賽級別賽事，單打和雙打冠軍將會得到1000個積分，其他輪次依次作降分處理。男子項目與女子項目不同輪次所獲得的積分有所不同，女子球員獲得的積分相對而言比男子的高。截止到2014年賽季，積分和獎金作如下分佈：
