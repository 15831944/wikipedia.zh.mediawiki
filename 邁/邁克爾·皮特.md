'''米高·卡門·皮特'''（{{lang-en|Michael Carmen Pitt}}，{{bd|1981年|4月10日||}}）是一位美國男演員、音樂人和模特兒。他最著名的作品是奧斯卡金獎導演[[貝納多·貝托魯奇|貝納多·貝托魯奇]]的電影《[[戲夢巴黎|戲夢巴黎]]》（2003年），在片中出演主角。他隨後在[[葛斯·范·桑|葛斯·范·桑]]的《{{link-en|超脫末日|Last Days (2005 film)}}》（2005年）、[[邁克爾·哈內克|邁克爾·哈內克]]的《[[瘋殺遊戲|瘋殺遊戲]]》（2007年）和[[馬丁·史柯西斯|馬丁·史柯西斯]]監製與部分執導的[[HBO|HBO]]電視劇《[[海濱帝國|海濱帝國]]》中飾演[[吉米·達摩狄|吉米·達摩狄]]<ref>{{Cite web |url = http://deadline.com/2011/10/uta-says-it-fired-boardwalk-empire-actor-185710/ |title = UTA Says It Fired 'Boardwalk Empire' Actor |author = Nikki Finke |date = 2011-10-21 |publisher = Deadline Hollywood |language = en}}</ref><ref>{{Cite web |url = https://variety.com/2012/film/awards/michael-pitt-35285/ |title = Michael Pitt looks back on ‘Boardwalk Empire’ |author = Jon Weisman |date = 2012-05-29 |publisher = Variety |language = en}}</ref>，以及在[[全國廣播公司|NBC]]電視劇《[[雙面人魔|雙面人魔]]》中飾演{{link-en|梅森·韋格爾|Mason Verger}}。他也在數部電影裡有演出，例如《{{link-en|搖滾芭比|Hedwig and the Angry Inch (film)}}》（2001年）、《{{link-en|半熟少年激殺案|Bully (2001 film)}}》（2001年）、《{{link-en|異旅情絲|Silk (2007 film)}}》（2007年）、《[[癲狗喪七|癲狗喪七]]》（2012年）和《[[I型起源|I型起源]]》（2014年）。
