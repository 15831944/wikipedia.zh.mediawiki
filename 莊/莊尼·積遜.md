'''約翰·亞歷克·"莊尼"·積遜'''（{{lang-en|'''John Alec "Johnnie" Jackson'''}}，{{bd|1982年|8月15日|catIDx=Jackson, Johnnie}}）是一名[[英格蘭|英格蘭]]職業足球員，主要司職[[中場|左中場]]，亦可擔任[[中場|正中場]]或[[後衛_(足球)|左閘]]，現時效力[[英冠|英冠]]球會[[查尔顿竞技足球俱乐部|查爾頓]]，並擔任[[隊長_(足球)|隊長]]。

== 生平 ==
===球會===
;熱刺
積遜於1999年在老牌球會[[托特纳姆热刺足球俱乐部|熱刺]]開展足球事業，翌年簽署首份職業球員合約，但卻鮮有機會為一隊上陣。2004年2月於[[英超|英超]]作客4-2擊敗[[查尔顿竞技足球俱乐部|查爾頓]]取得1個入球<ref>{{cite web  |language=en |url=http://news.bbc.co.uk/sport1/hi/football/eng_prem/3456291.stm |title=Charlton 2–4 Tottenham |publisher=''[[BBC_Sport|BBC Sport]]'' |date=11 February 2004 |accessdate=25 August 2009}}</ref>，是他為母會射入的唯一入球。

;外借不同球會
逗留在[[白鹿徑球場|白鹿徑球場]]期間積遜曾被借用到多支不同的低組別球隊短暫效力，包括外借[[斯温登足球俱乐部|史雲頓]]時曾取得兩個入球，其一是在聯賽對[[北安普顿足球俱乐部|諾咸頓]]時射入<ref>{{cite web |language=en |url=http://news.bbc.co.uk/sport1/hi/football/eng_div_2/2257953.stm |title=Swindon 2–0 Northampton |publisher=''[[BBC_Sport|BBC Sport]]'' |date=21 September 2002 |accessdate=7 November 2009}}</ref>，而另外一球是於[[英格蘭聯賽錦標|英格蘭聯賽錦標]]在[[绍森德足球俱乐部|修安聯]]身上取得<ref>{{cite web |language=en |url=http://news.bbc.co.uk/sport1/hi/football/2344845.stm |title=Swindon 6–1 Southend |publisher=''[[BBC_Sport|BBC Sport]]'' |date=22 October 2002 |accessdate=7 November 2009}}</ref>。他亦曾借用到[[科尔切斯特联足球俱乐部|高車士打]]，其後更落實正式加盟。積遜於借用到[[考文垂足球俱乐部|高雲地利]]時，於對本地死敵[[西布罗姆维奇足球俱乐部|西布朗]]的賽事完場前射入絕殺對而獲得稱賞<ref>{{cite web  |language=en |url=http://news.bbc.co.uk/sport1/hi/football/eng_div_1/3314395.stm |title=Coventry 1–0 West Brom
|publisher=''[[BBC_Sport|BBC Sport]]'' |date=20 December 2003 |accessdate=25 August 2009}}</ref>。

;高車士打
2006年夏季熱刺棄用積遜，他隨即以自由身簽約加盟曾在2003年以借用身份短暫效力而剛升上[[英冠|英冠]]的[[科尔切斯特联足球俱乐部|高車士打]]。2007年2月他射入加盟後的首個入球，但球隊仍以1-2不敵西布朗。翌季積遜獲選球會的年度最佳球員，但高車士打以墊底成績降班返回[[英甲|英甲]]。

[[2008年至2009年英格蘭足球甲級聯賽|2008/09年球季]]積遜於1-1賽和[[米尔顿凯恩斯足球俱乐部|米爾頓凱恩斯]]後背傷加劇，下半季大部分時間因此而缺陣。

;諾士郡
2009年8月積遜轉投位處[[英乙|英乙]]的[[諾士郡足球會|諾士郡]]，於8月22日首次在主場上陣3-0擊敗[[达根汉姆和雷德布里奇足球俱乐部|杜根咸]]中取得加盟後的首球<ref>{{cite web |language=en |url=http://news.bbc.co.uk/sport1/hi/football/eng_div_3/8207887.stm |title=Notts County 3–0 Dag & Red |publisher=''[[BBC_Sport|BBC Sport]]'' |date=22 August 2009 |accessdate=7 November 2009}}</ref>，直到11月7日在[[英格蘭足總盃|英格蘭足總盃]]對[[布拉德福德城足球俱乐部|巴拉福特]]才取得第二個入球<ref>{{cite web |language=en 
|url=http://news.bbc.co.uk/sport1/hi/football/fa_cup/8341087.stm |title=Notts County 2–1 Bradford
|publisher=''[[BBC_Sport|BBC Sport]]'' |date=7 November 2009 |accessdate=7 November 2009}}</ref>。

;外借查爾頓
2010年2月18日積遜被借用到[[英甲|英甲]]球會[[查尔顿竞技足球俱乐部|查爾頓]]，為期1個月以填補因傷陣的[[格兰特·巴塞|格兰特·巴塞]]（Grant Basey）的左閘位置<ref>{{cite web |language=en |url=http://www.southlondon-today.co.uk/tn/Sport.cfm?id=5742&headline=Charlton%20sign%20Johnnie%20Jackson |title=Charlton Sign Johnnie Jackson |publisher=''southlondon-today.co.uk'' |date=18 February 2010 |accessdate=18 February 2010}} {{Dead link|date=October 2010|bot=H3llBot}}</ref>。

;查爾頓
2010年7月積遜正式轉投[[查尔顿竞技足球俱乐部|查爾頓]]，更於同年11月獲選為「英甲每月最佳球員」。

[[2011年至2012年英格蘭足球甲級聯賽|2011/12年球季]]積遜獲委任為[[隊長_(足球)|隊長]]，主力擔任[[中場|左中場]]，這個球季他的入球大豐收，整季轟入13球，包括憑[[自由球|自由球]]射入[[谢菲尔德|谢菲尔德]]雙雄[[谢周三足球俱乐部|錫周三]]和[[谢菲尔德联足球俱乐部|錫菲聯]]，分別各勝1-0為查爾頓取得重要勝仗，協助球隊贏得聯賽冠軍及升班返回[[英冠|英冠]]。2012年6月29日積遜與球會續約兩年<ref>{{cite web |language=en |url=http://www.cafc.co.uk/page/NewsDetail/0,,10267~2825807,00.html |title=Jackson pens new contract |date=29 July 2012 |publisher=[[查尔顿竞技足球俱乐部|''cafc.co.uk'']] |accessdate=30 June 2012 |deadurl=yes |archiveurl=https://web.archive.org/web/20120705081459/http://www.cafc.co.uk/page/NewsDetail/0%2C%2C10267~2825807%2C00.html |archivedate=2012年7月5日 }}</ref>。
