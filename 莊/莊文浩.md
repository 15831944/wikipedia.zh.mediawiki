'''莊文浩'''（'''{{lang|en|Michael David Chong}}'''，{{bd|1971年|11月22日|catIdx=C莊}}），[[荷兰裔加拿大人|荷裔]]、[[华裔加拿大人|华裔加拿大人]]，[[加拿大聯邦|加拿大聯邦]][[加拿大保守黨|保守黨]]籍政治家，現任[[加拿大國會下議院|加拿大國會下議院]]惠靈頓-哈頓山選區[[國會議員|國會議員]]。他曾於2006年2月6日至11月27日期间擔任[[加拿大總理|總理]][[史蒂芬·哈珀|史蒂芬·哈珀]]內閣的[[加拿大樞密院|樞密院]]主席和政府間事務兼體育部長，是聯邦歷史上第二位華裔內閣成員。他出生於加拿大[[安大略|安大略]]的一個移民家庭，父親來自[[香港|香港]]，母親來自[[荷蘭|荷蘭]]。他可以說英文、法文和粵語，在從政前，他曾在IT行業工作。

2016年5月，莊文浩宣布參選加拿大保守黨黨魁<ref>{{cite web |url=http://www.cbc.ca/news/politics/wherry-chong-leadership-1.3583991 | title=Michael Chong's bid for Conservative leadership starts with a story | publisher=加拿大廣播公司CBC | date=2016年5月16日 }} </ref>，成為加拿大歷史上首位參與主要黨派領袖角逐的華裔政治家。

== 早期生涯 ==
莊文浩的父親在1952年從香港移居加國，在[[曼尼托巴大学|曼尼托巴大学]]取得[[理學士|理學士]]學位後，於1956年考入[[渥太華大學|渥太華大學]]醫學院。他的父親倚靠在餐館打工和做伐木工人的薪水完成學業後成為醫生；他的母親在1960年從荷蘭移居加國成為護士。莊文浩的父親在安大略[[京士頓|京士頓]]的[[女王大學|女王大學]]醫院實習時與他的母親相識。<ref>{{cite web |url=http://www.mingpaocanada.com/van/htm/News/20160517/var1h_r.htm | title=莊文浩競逐聯邦保守黨魁 標榜父親移民故事 |publisher=明報加西網| date=2016年5月17日 }}</ref>

莊文浩於1971年在多倫多郊區[[威靈頓縣|威靈頓縣]]自家的農場中出生，是家中長子；他6歲時，母親在車禍中去世，而後1999年，他的父親在同一地點出車禍去世。莊文浩曾在[[多伦多大学三一学院|多伦多大学三一学院]]學習[[歷史|歷史]]、[[哲學|哲學]]和[[政治學|政治學]]，並取得[[文學士|文學士]]學位。大學畢業後，他與大學期間的室友和同學{{link-en|羅德亞德·格里菲斯|Rudyard Griffiths}}創辦了提供加拿大歷史教育的慈善機構{{link-en|多米尼研究所|Historica Canada}}。而後，他在[[信息技術|信息技術]]行業工作，曾任加拿大[[國家冰球聯盟|國家冰球聯盟]]{{link-en|球員工會|National Hockey League Players' Association}}的[[首席信息官|信息技术总监]]、[[多倫多皮爾遜國際機場|多倫多機場]]IT技術顧問等職位。<ref>{{cite web |url=http://www.macleans.ca/politics/we-looked-at-each-other-and-kind-of-knew/ |title=Michael Chong’s unusual path to Tory contender |publisher=麥克林雜誌 | date=2016年8月12日}}</ref>

== 從政 ==
從高中起，莊文浩就曾在代表自己家鄉選區的國會議員{{link-en|佩林·貝蒂|Perrin Beatty}}的競選團隊中做志願者，而佩林·貝蒂曾是加拿大前總理[[马丁·布赖恩·马尔罗尼|马尔罗尼]]內閣重臣。2000年，莊文浩代表[[加拿大進步保守黨|加拿大進步保守黨]]角逐{{link-en|滑鐵盧-惠靈頓選區|Waterloo—Wellington}}的國會議員，僅居第三。四年後的[[2004年加拿大聯邦大選|2004年加拿大聯邦大選]]中，莊文浩代表與[[加拿大联盟|加拿大联盟]]合併後的[[加拿大保守黨|加拿大保守黨]]參選新成立的惠靈頓-哈頓山選區國會議員並獲勝，並在2006年、[[2008年加拿大聯邦國會選舉|2008年]]、[[2011年加拿大聯邦大選|2011年]]和[[2015年加拿大聯邦大選|2015年]]大選中成功連任，擔任該選區國會議員至今。

2006年，莊文浩成為史蒂芬·哈珀第一屆內閣成員，但莊文浩不滿總理哈珀承認[[魁北克|魁北克]]是加拿大的一個“邦國”（Nation），稱該政策對其它族裔不公平，因而在該年11月27日莊文浩辭去所有內閣職務。<ref>{{cite web |url=http://news.xinhuanet.com/overseas/2006-11/28/content_5400851.htm |title=新聞人物：加政壇首位辭職的華裔內閣部長莊文浩 |publisher=新華網 |date=2006年11月28日 |deadurl=yes |archiveurl=https://web.archive.org/web/20161124093300/http://news.xinhuanet.com/overseas/2006-11/28/content_5400851.htm |archivedate=2016年11月24日 }} </ref>

莊文浩長期致力於加拿大[[議會|議會]]程序的改革，並在2014年提出了議會改革法案（Bill C-586）以增加普通議員的權力和遏制各個黨派的核心領袖的权力，該法案受到了多個黨派議員的支持並最終獲得議會兩院通過。<ref>{{cite web |url=http://ottawacitizen.com/news/politics/reform-act-house-vote-third-reading |title=Michael Chong's Reform Act wins Commons approval |publisher=Ottawa Citizen |date=2015年2月25日}}</ref><ref>{{cite web |url=http://www.theglobeandmail.com/news/politics/senators-are-showing-what-a-less-partisan-body-looks-like/article24910558/ |title=Senators are showing what a less partisan body looks like |publisher=加拿大環球郵報 |date=2015年6月10日}}</ref>

2016年5月16日，莊文浩宣布參選加拿大保守黨黨魁。
