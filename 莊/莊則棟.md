'''莊則棟'''（{{bd|1940年|8月25日|2013年|2月10日|catIdx=Z庄}}），[[江苏|江苏]][[扬州|扬州]]人，[[中國|中國]]著名[[乒乓球|乒乓球]]運動員。中国男子乒乓球运动员，生于江苏省扬州市，自幼喜爱乒乓球运动，14岁加入北京市少年宫业余体校乒乓球小组，1957年入选北京市乒乓球队，同年参加全国比赛，1959年入选中国青年乒乓球队；庄则栋曾获得第26—28届世乒赛男子单打冠军，是二十世纪60—70年代中国男子乒乓球队主力队员之一；1971年4月，在日本参加第31届世乒赛期间，冒着风险，结交美国运动员，打开了中美两国友好的大门；曾任国家体委主任，中共十届中央委员，第三、四届全国人大代表。

== 生平 ==
[[File:1965-01_徐寅生_庄则栋.jpg|thumb]]与庄则栋]]
三次蟬聯男子單打冠軍（三次決賽均戰勝國家隊隊友[[李富榮|李富榮]]），成為中國第一個在該項比赛上獲得“三連冠”的人。他亦是中美「-{[[乒乓外交|乒乓外交]]}-」中方的代表（美方的代表是[[科恩_(運動員)|科恩]]）。1974年至1976年任[[国家体委|国家体委]]主任。[[文革|文革]]结束后，被隔离审查四年，著有《闯与创》总结他的乒乓球技法及心得。

2004年科恩因[[心臟搭橋|心臟搭橋]]手術失敗病逝，享年52歲，莊則棟亦致電表示慰問。後來2007年莊則棟出訪美國並會見科恩的母親，並向其表示沒有在科恩生前與他再見為「畢生遺憾」。

莊則棟自2008年被診斷為[[肝癌|肝癌]]晚期後，在北京、上海多所醫院治療後，於2012年8月29日入住[[北京|北京]][[佑安醫院|佑安醫院]]治療，經[[新華社|新華社]]記者[[唐師曾|唐師曾]]以博客把莊的景況與費用引致的家庭困難對外曝光<ref>[http://blog.sina.com.cn/s/blog_4ab761450102e555.html 莊則棟癌症惡化，日本媳婦服侍27年]、[http://blog.sina.com.cn/s/blog_4ab761450102e563.html 佐佐木敦子、莊飆，共同照顧莊則棟]，唐師曾博客2012年9月16-17日</ref>經[[中新社|中新社]]曝光後，莊以親筆簽名信謝絕外界幫助捐贈<ref>[http://www.chinanews.com/ty/2012/10-22/4267071.shtml 莊則棟致信中新社 表達心裡「要說的幾句話」]，中新社2012年10月22日</ref>；2013年初病情惡化<ref>[http://www.chinanews.com/ty/2013/02-05/4550537.shtml 莊則棟病情再度惡化 院方采多種措施做最後一搏]，中新社2013年2月5日</ref>，終於2月10日下午17:06逝世，享年72岁<ref>{{cite web | title=唐师曾：庄则栋逝世 |url=http://e.weibo.com/1787920531/ziy5syMiD|accessdate=2013-02-10 }}</ref>，骨灰於同年[[重陽節|重陽節]]（10月13日）早上安葬於[[昌平区|昌平区]]之鳳凰山陵園。

== 家庭 ==
莊則棟與第二任妻子佐佐木敦子於1987年12月19日在北京結婚，此前前妻為[[鮑蕙蕎|鮑蕙蕎]]，有一子莊飆，一女莊嵐。

== 逸事 ==
莊則棟最為擅長的乒乓球技巧，時常與外國國賓切磋球技時顯現：發球時把球拋高，快速跑繞球桌一圈後返回自己桌面，球拍適時接球击出，令外賓大開眼界嘆為觀止。

2009年夏，庄则栋在被《[[中国体育报|中国体育报]]》记者采访时提到，[[毛泽东|毛泽东]]曾经称呼其为“我的莊爷爷”，但是在2009年7月21日《中国体育报》第三版报道中，编辑未经其同意改成“我的莊则栋”，庄则栋对此在其博客中表示不满，认为是“随意更改领袖的话，歪曲了历史，欺骗了群众”<ref>{{cite web | title=请大家来讨论，这是为什么？！| url=http://blog.sina.com.cn/s/blog_4cf7b4ec0100egww.html |accessdate=2009-07-28 |work=庄则栋}}</ref>。

== 家庭 ==
*前妻：[[鲍惠荞|鲍惠荞]]（1940年-），出生于四川省犍为县五通镇；中央乐团的钢琴演奏家；鲍惠荞与莊則棟1959年相识，1967年结婚。他们的儿子庄彪、女儿庄岚，受母亲薰陶，都喜爱音乐，常练习钢琴和小提琴。庄、鲍二人已于1985年2月离异。
*儿子：[[庄彪|庄彪]]，
*女儿：[[庄岚|庄岚]]，
*第二任妻子：[[佐佐木敦子|佐佐木敦子]]（1944年10月10日-），1944年10月10日出生于[[沈阳|沈阳]]。1985年2月前妻与庄则栋离婚。当时庄则栋刚从关押的山西落实政策，到北京市教育局下属的少年宫当体育老师。佐佐木敦子与莊則棟1985年在北京市少年宫重逢，两人开始恋爱。1987年12月19日，佐佐木敦子放弃日本国籍，加入中国国籍，领取身份证并于当天和庄先生结婚，定居北京。佐佐木敦子没有儿女。
当时北京不给庄则栋发护照，不许庄则栋出国。最终[[邓小平|邓小平]]批准，才得以成婚。最终佐佐木敦子自愿放弃日本国籍，当中国人，移民北京。
2016年，莊則棟墓碑金字被不明人士噴黑，其遺孀佐佐木敦子為日本人，其友人推測可能是反日人士所為<ref>{{cite news|title=日裔遺孀被反日者針對 乒乓國手墓遭噴黑|url=http://hk.apple.nextmedia.com/international/art/20160821/19744540|accessdate=2016-08-25|agency=香港蘋果日報|date=2016-08-21}}</ref><ref>{{cite news|title=因遺孀是日人？已故乒球名宿莊則棟墓遭塗鴉|url=http://hk.on.cc/cn/bkn/cnt/news/20160820/bkncn-20160820200253072-0820_05011_001.html|accessdate=2016-08-25|agency=東網|date=2016-08-20}}</ref>。
