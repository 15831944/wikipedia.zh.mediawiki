== 剧情 ==
[[北宋|北宋]]时期，法王寺的空空小沙弥带领自己的师兄弟，出山门，为黎民百姓，除恶扬善，弘扬佛法，激昂正气。

== 花絮 ==
在接受新浪网的采访时，导演[[都晓|都晓]]说，电视剧为了赢取儿童的注意力和收视率，而刻意选择了儿童作为主角。<ref>{{cite web |url= http://ent.sina.com.cn/v/m/2012-08-22/ba3718885.shtml|title=实录：《小空空》导演携巩汉林阎青妤做客 |accessdate= 2012年8月22日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

电视剧在情节和题材上有仿照[[日本|日本]]《[[聪明的一休|-{zh-cn:聪明的一休; zh-hk:機靈小和尚; zh-tw:一休和尚}-]]》。<ref>{{cite web |url=http://www.chinanews.com/yl/2012/08-08/4093228.shtml |title=《聪明小空空》央视将播 打造中国版“一休” |accessdate=2012年8月8日 |publisher=中国新闻网 |work= |language =zh-cn}}</ref>

因为电视剧获得了儿童的亲睐，同名小说也业已上市。<ref>{{cite web |url=http://et.21cn.com/tv/roll/2012/09/20/13049823.shtml |title=《聪明小空空》受欢迎 同名小说已问世 |accessdate=2012-09-20 |publisher=21CN |work= |language=zh-cn |archiveurl=https://web.archive.org/web/20160304124207/http://et.21cn.com/tv/roll/2012/09/20/13049823.shtml |archivedate=2016-03-04 |deadurl=yes }}</ref>

[[中国电视艺术家协会|中国电视艺术家协会]]分党组书记、驻会副主席[[张显|张显]]认为该剧挖掘了[[佛教|佛教]]的[[智慧|智慧]]和[[慈悲|慈悲]]。<ref>{{cite web |url=http://ent.sina.com.cn/r/m/2012-09-12/11193739358.shtml |title= 张显：《聪明小空空》向佛家智慧更进一步|accessdate= 2012年9月12日|publisher=新浪网 |work= |language =zh-cn}}</ref>

原[[中国儿童艺术剧院|中国儿童艺术剧院]]院长、[[中国动画学会|中国动画学会]]高级顾问、儿童艺术专家[[欧阳逸冰|欧阳逸冰]]认为该剧的口味正是老少咸宜。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2012-09-10/15233737152.shtml |title=专家评《小空空》：来之不易的老少皆宜 |accessdate=2012年9月10日 |publisher= 新浪网|work= |language =zh-cn}}</ref>

对于观众认为电视剧的名字借著名[[女优|女优]][[苍井空|苍井空]]炒作，制片方经理都小明、编剧刘麒给予否认，并且列举了该剧的大量大陆和[[香港|香港]]的明星。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2012-08-19/12133715697.shtml |title=网友质疑《聪明小空空》 称其借苍井空上位 |accessdate= 2012年8月19日 |publisher=新浪网 |work= |language =zh-cn}}</ref>
