'''沂水路'''是[[中國|中國]][[青岛市|青岛市]][[市南区|市南区]]一条城市支路，全长约400米，东北起自[[江苏路_(青岛)|江苏路]]，西南止于总督府广场（分为两叉）。该路始建于[[德国|德国]]租借时期，时名'''迪德里希路'''（{{lang-de|Diederichs Weg}}，又译'''地利路'''），[[日本|日本]]第一次统治时期更名为'''赤羽町'''（{{jpn|j=赤羽町|kyu=赤羽町|hg=あかばね まち}}），回归国统后改现名。沿路有许多德国租借时代留下的德式建筑，西南端11号正对广场的就是当年的最高政府机关——[[胶澳总督府|胶澳总督府]]，最南端是[[英国|英国]]驻青岛的[[总领事馆|总领事馆]]，总督府以东路段两侧有[[美国|美国]]驻青岛的[[总领事馆|总领事馆]]，以及一批高级别墅。东北端正对着位于高地上的[[江苏路基督教堂_(青岛)|江苏路基督教堂]]。

== 周边历史建筑介绍 ==
===北侧===
总督府以东的沂水路路段北侧集中了多座经典的德租时期别墅建筑。

====美国领事馆旧址====
{{Main|美国驻青岛总领事馆}}
沂水路1号为美国领事馆旧址，建于1912年，为建筑师[[李希德|李希德]]（P.F.Richter）的地产，同年，美国驻青岛领事馆迁入办公，直至1949年关闭<ref Name=qing>{{cite book |author=袁宾久 |title=《青岛德式建筑》|year=2009 |publisher=中国建筑工业出版社 |location=青岛 |isbn=9787112109050}}</ref><ref>{{cite web |url=http://blog.sina.com.cn/s/blog_48883220010002nu.html|title=美国驻青岛领事馆}}</ref>。该建筑为德租时期末期别墅建筑的代表作。

====格尔皮克-科尼希别墅旧址====
沂水路3号建于1899年，是青岛现存最早的德租时期建筑之一，设计师可能是马克斯·克诺普夫（Max Knopff）。德租时期该别墅为总督府地产，用以为在青德籍军政人员提供住所。第一个入住该楼的人是[[胶澳皇家法院|胶澳皇家法院]]首任大法官保罗·格尔皮克博士（Dr.Paul Gelpcke）。1902年至1907年，[[青岛督署医院|督署医院]]院长哈里·科尼希（Harry Koenig）在此居住。[[辛亥革命|辛亥革命]]后的1912年，大量逊清遗老逃亡青岛，蒙古多罗特公升允曾在此居住。1949年该楼被青岛市军管会接管，后曾为青岛日报社幼儿园，文革初期至今为日报社宿舍楼<ref Name=qing/><ref>{{cite web |url=http://blog.sina.com.cn/s/blog_48883220010003fc.html|title=格尔皮克-科尼希别墅}}</ref>。2000年列入第一批[[青岛市历史优秀建筑|青岛市历史优秀建筑]]名单。

该建筑内外装饰精美，有木制阳台、桁架、塔楼、铁艺风向标、木雕楼梯柱等德式装饰元素，亦有带有明显中式风格的云龙纹石柱<ref Name=qing/>，是德租时期早期别墅建筑的代表作。这些装饰细节虽历经多年，仍基本保存。尤其是同时被保存下来的塔楼与铁艺风向标，在青岛老建筑中实属罕见。该建筑建成几年后所建的带有铁艺大门（如今仅剩一扇）和盲拱装饰的花岗岩砌挡土墙及院门，如今成为沂水路上的一景。

====施迪克弗特住宅旧址====
沂水路5号建于1905年，最初为维林公司（C.Vering Co.）的筑港工程师约翰·施迪克弗特（John Stickforth）的别墅（施氏此前居住在今武定路北端的维林公司别墅）<ref>{{cite web |url=http://blog.sina.com.cn/s/blog_48883220010006n8.html|title=筑港工程师斯提克弗茨}}</ref>。约1910年前后，施氏离开青岛，将住宅卖给[[兰山路_(青岛)#哈利洋行青岛分行|哈利洋行]]董事汉斯·克里斯蒂安·奥古斯特森（Hans Christian Augustesen）<ref Name=qing/>。1918年，奥氏被日军驱逐，前往上海。1925年[[麦加利银行|麦加利银行]]买下了这栋住宅，并将其作为经理住宅使用。该建筑现为青岛市城市管理局、青岛市知识产权局、青岛市建筑材料工业总公司的办公楼。2000年列入第一批[[青岛市历史优秀建筑|青岛市历史优秀建筑]]名单。

先于施迪克弗特住宅在沂水路建成的沂水路3号、7号、9号均使用了装饰精美的复古风格，施迪克弗特住宅的外立面则使用了相当简洁明快的装饰风格，受到了[[新艺术运动|德国青年派]]的影响，是青岛德租时期建筑由复古转向现代的一个标志。

====迪德里希路捷成洋行别墅旧址====
沂水路7号建于1902年，最初为捷成洋行（Diederichsen Jebsen & Co. Tsingtau）经理的别墅<ref Name=qing/>。该建筑使用了红砖、花岗岩、木制桁架、塔楼等装饰元素，在外观的精美程度上丝毫不逊色于格尔皮克-科尼希别墅。约1920年代，该楼西侧扩建出一部分，扩建部分虽极力模仿原建筑的风格，但破坏了建筑外立面的美感。文革时期临街立面的木制桁架与塔楼被拆除。该建筑现与施迪克弗特住宅旧址同处一座政府办公楼大院。2000年列入第一批[[青岛市历史优秀建筑|青岛市历史优秀建筑]]名单。

====德国海军第三营营长官邸旧址====
{{Main|德国海军第三营营长官邸}}
沂水路9号建于1899年，是青岛现存最早的德国建筑之一。由[[广包公司|广包公司]]（Baufirma F.H.Schmidt）承建，设计师可能是[[马克斯·克诺普夫|马克斯·克诺普夫]]（Max Knopff）。该建筑自建成后即为德国海军第三营（III.Seebataillon）历任少校营长的官邸。现为政府办公楼<ref Name=qing/>。

===南侧===
相对于北侧，南侧的开发时间较晚，老建筑也相对较稀疏。沂水路[[江苏路_(青岛)|江苏路]]路口旁的沂水路2号有两栋建于第二次日占时期的日本现代风格住宅。

====沂水路4号====
沂水路4号为一单层{{link-en|蒙莎屋顶|Mansard roof}}加地下室的建筑，对面即为沂水路5号施迪克弗特住宅旧址。该楼据说最早为德租时期消防队车库。但由于该楼与施迪克弗特住宅同时建造，建筑风格与建筑材料也与一致，1913年地籍图显示其与原施迪克弗特住宅同为奥古斯特森的地产<ref Name=kata>{{cite web |url=http://www.tsingtau.org/kataster-karten-tsingtau-1914/|title=Katasterkarten Tsingtau}}</ref>，所以该楼更有可能为施迪克弗特住宅的附属建筑。现为某单位驻地。

====克纳珀住宅旧址====
沂水路6号建于1913-1914年左右，为一双层住宅建筑，1913年地籍图显示该楼属于一个叫克纳珀（Knappe）的人<ref Name=kata/>。现为民居。

====李经迈住宅旧址====
沂水路8号建于1913-1914年左右，为一双层蒙莎屋顶住宅，1913年地籍图显示该楼属于[[李鸿章|李鸿章]]之子[[李经迈|李经迈]]（地籍图上写作Li tsching mai）<ref Name=kata/>。现为青岛日报社发行处。

====胶澳总督府旧址====
{{Main|胶澳总督府}}
沂水路11号是胶澳总督府旧址，俗称“老市府”，建于1903-1905年，由德国建筑师弗里德里希·马尔克（Friedrich Mahlke）设计<ref Name=qing/>，是老青岛的地标之一。

====[[湖南路_(青岛)#曼弗雷德·齐默尔曼住宅旧址（英国驻青岛总领事馆旧址）|英国驻青岛总领事馆旧址]]====
沂水路14号建于1910年，由广包公司承建，最初为律师兼公证人曼弗雷德·齐默尔曼（Manfred Zimmermann）的住宅<ref Name=qing/><ref>{{cite web |url=http://blog.sina.com.cn/s/blog_48883220010003z0.html|title=英国领事馆的补充资料}}</ref>。1935-1951年，该楼曾为英国驻青岛总领事馆<ref>{{cite web |url=http://blog.sina.com.cn/s/blog_48883220010002ol.html|title=英国驻青岛领事馆}}</ref>。
