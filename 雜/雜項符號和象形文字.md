'''雜項符號和象形文字'''是一個[[Unicode區段|Unicode區段]]，包含氣象和天文符號，表情符號主要是為了與日本電話公司的[[Shift_JIS|Shift JIS]]實現兼容，以及Microsoft Windows中來自[[Wingdings|Wingdings]]和[[Webdings|Webdings]]字體的中的字符。

==繪文字修飾符==
雜項符號和象形文字區段有49個用來表示人類或身體部位的繪文字。
因此，一套「表情符號修飾符」被定義了出來。這些修飾符以{{tsl|en|Fitzpatrick scale|菲茨帕特里克級別}}（Fitzpatrick scale）為基礎，用於定義表情符號的膚色：<ref name="EmojiData"/>
:U+1F3FB EMOJI MODIFIER FITZPATRICK TYPE-1-2
:U+1F3FC EMOJI MODIFIER FITZPATRICK TYPE-3
:U+1F3FD EMOJI MODIFIER FITZPATRICK TYPE-4
:U+1F3FE EMOJI MODIFIER FITZPATRICK TYPE-5
:U+1F3FF EMOJI MODIFIER FITZPATRICK TYPE-6

建議採用這種系統來表示“[[多元文化主義|人類多樣性]]”的文件草案是由Google的Mark Davis和Apple Inc.的Peter Edberg提交的。<ref>
"The default representation of these modifier characters when used alone is as a color swatch. Whenever one of these characters immediately follows certain characters (such as WOMAN), then a font should show the sequence as a single glyph corresponding to the image for the person(s) or body part with the specified skin tone" [http://www.unicode.org/reports/tr51/tr51-2-archive.html#Diversity Draft Unicode Technical Report #51 "UNICODE EMOJI"] Version	1.0 (draft 10) eds.	Mark Davis (Google Inc.), Peter Edberg (Apple Inc.), 2015-05-08.</ref>
