'''苑桂森'''（{{bd|1955年|||catIdx=}}），[[天津|天津]]人，[[中华人民共和国|中华人民共和国]]政治人物、外交官。

1999年，接替[[陶苗发|陶苗发]]，担任[[中国驻斯洛伐克大使列表|中华人民共和国驻斯洛伐克大使]]<ref>{{cite web|title = 驻斯洛伐克共和国历任大使 |url = http://www.fmprc.gov.cn/mfa_chn/ziliao_611306/wjrw_611324/2167_611334/2172_611344/t9238.shtml |publisher = 中华人民共和国外交部 |accessdate = 2014-11-11}}</ref>。2003年，由[[黄忠坡|黄忠坡]]接任，其接替[[周晓沛|周晓沛]]担任[[中国驻波兰大使列表|中华人民共和国驻波兰大使]]<ref>{{cite web| title = 驻波兰共和国历任大使 |url = http://www.fmprc.gov.cn/mfa_chn/ziliao_611306/wjrw_611324/2167_611334/2172_611344/t9225.shtml |publisher = 中华人民共和国外交部 |accessdate = 2014-11-11}}</ref>。2011年，由[[孙荣民|孙荣民]]接任，其接替[[蔡润国|蔡润国]]，担任[[中国驻厄瓜多尔大使列表|中华人民共和国驻厄瓜多尔大使]]<ref>{{cite web| title = 驻厄瓜多尔共和国历任大使 |url = http://www.fmprc.gov.cn/mfa_chn/ziliao_611306/wjrw_611324/2167_611334/2174_611348/t9284.shtml |publisher = 中华人民共和国外交部 |accessdate = 2014-11-11}}</ref>。2013年至今，由[[王士雄_(外交官)|王士雄]]接任，其接替[[胡山|胡山]]，担任[[中国驻巴哈马大使列表|中华人民共和国驻巴哈马大使]]<ref>{{cite web| title = 驻巴哈马历任大使 |url = http://www.fmprc.gov.cn/mfa_chn/ziliao_611306/wjrw_611324/2167_611334/2174_611348/t9281.shtml |publisher = 中华人民共和国外交部 |accessdate = 2014-11-11}}</ref>。2016年6月，卸任中华人民共和国驻巴哈马大使职务<ref>{{cite web|accessdate=2016-06-06|url=http://www.thepaper.cn/newsDetail_forward_1479934|title=李华新出任中国驻沙特大使，杜伟出任中国驻乌克兰大使|publisher=澎湃新闻}}</ref>。
