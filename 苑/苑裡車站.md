'''苑裡車站'''位於[[中華民國|中華民國]][[苗栗縣|苗栗縣]][[苑裡鎮|苑裡鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[海岸線_(臺鐵)|海岸線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[島式月台|島式月台]]、[[岸式月台|岸式月台]]各一座。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]，但與一般三等站多以停靠較低等級列車為主的狀況不同，本站一天僅有一班次[[自強號列車|自強號]]不停靠外，其餘各級列車均有停靠。
*[[自強號列車|自強號]]不停靠北上6126次(週日行駛)、及南下109次(週日停駛)。
*對號列車停靠率：95.9%；所有車種停靠率：98.7%。
*是海線沿線第三大車站，僅次於[[大甲車站|大甲車站]]與[[沙鹿車站|沙鹿車站]]。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1922年10月11日：「苑裡驛」開業。
*2014年7月16日：為加強疏運需要，開始停靠[[太魯閣列車|太魯閣號]]。
*2015年6月30日：啟用多卡通刷卡機。
*2015年10月15日：原[[太魯閣列車|太魯閣號]]行駛之278、285次改由[[普悠瑪列車|普悠瑪號]]行駛。
*2016年10月20日：原[[普悠瑪列車|普悠瑪號]]行駛之278、285次改由[[太魯閣列車|太魯閣號]]行駛。
