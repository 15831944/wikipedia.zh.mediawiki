'''野生稻'''（[[學名|學名]]：''Oryza rufipogon''），又称鬼仔稻，是一个野生[[稻|稻]]种（但并非[[水稻|水稻]]的品种），目前作为人类重要食糧的[[稻米|稻米]]可能由其演化而来。在世界多处都有分布，台灣從[[桃園區|桃園]]到[[新竹市|新竹]]之間的水塘有發現的紀錄，70年代後絕種。<ref>{{cite journal|title=台灣野生稻|author=蔡國海|url=http://www.npgrc.tari.gov.tw/npgrc-web/publish/newsletter/nltr6-1.html#a2| journal=中華民國作物種原簡訊|volume=4|issue =1|publisher=農業委員會農業試驗所|date=1999年6月}}
</ref><ref>{{cite book|title=臺日大辭典|page=355|author=臺灣總督府|url=http://jdlib.ntl.gov.tw/cgi-bin/browse.cgi?bookid=bjn00172v01|deadurl=yes|archiveurl=https://web.archive.org/web/20110818213300/http://jdlib.ntl.gov.tw/cgi-bin/browse.cgi?bookid=bjn00172v01|archivedate=2011-08-18}}</ref>

中国列為國家二級保護植物，不允許隨意採集和出售，但是可以作為科學研究並培育野生稻雜交種。

== 入侵物种 ==
野生稻是一種入侵物種且被美國列為“有害雜草”。<ref>{{Cite web|url=http://plants.usda.gov/java/profile?symbol=ORRU|title=Plants Profile for Oryza rufipogon (brownbeard rice)|accessdate=2016-10-22|work=plants.usda.gov}}</ref> 它同樣被阿拉巴馬、加州、佛羅里達、馬薩諸塞、明尼蘇達、北卡羅來納、俄勒岡州，南卡羅來納州和佛蒙特州列為有害雜草。 據北美植物保護協會所稱，野生稻能很好地在[[水稻|馴化稻]]裡混雜生長，以至於他們很難被偵測到。在這種場合下，野生稻會跟馴化稻競爭空間和肥料。野生稻能在馴化稻的收割期前早一步散播種子，所以每次輪耕都總會有少數野生稻存活下來。另外，從野生稻所收割下來的稻米在外觀上與白米有所區隔，所以會被視作外來雜質而不會被消費者所食用。<ref>NAPPO - PRA / Grains Panel Pest Fact Sheet - Oryza rufipogon Griff. June / 2003, http://www.nappo.org/PRA-sheets/Oryzarufipogon.pdf {{Wayback|url=http://www.nappo.org/PRA-sheets/Oryzarufipogon.pdf |date=20070730103504 }}</ref>
