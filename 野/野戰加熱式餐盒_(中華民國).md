'''野戰加熱式餐盒'''又被稱作'''野餐餐包'''，是由[[中華民國|中華民國]][[國防部聯合後勤司令部|國防部聯合後勤司令部]]自行研發、使用的加熱式餐盒，這也是[[中華民國國軍|中華民國國軍]]首次為野外演習訓練開發的熱食{{Link-en|野戰口糧|Field ration}}。

== 歷史 ==
=== 研發試吃 ===
[[中華民國國軍|中華民國國軍]]長期將[[壓縮餅乾|壓縮餅乾]]作為野戰口糧，但儘管餅乾式野戰口糧的熱量和[[營養素|營養素]]足夠提供[[参考膳食摄入量|體力需求]]，但和其口感與民眾飲食習慣差異甚大<ref name = "王烱華 5">{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20080505/30519081/ |title = 阿兵哥打野外 改吃熱飯 |author = 王烱華 |date = 2008年5月5日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2017年5月26日 |quote =  }}</ref>。[[國防部聯合後勤司令部|國防部聯合後勤司令部]]為了提升戰時官兵生活品質、強化戰鬥持續力，設定官兵在任何情況下食用熱食的需求，與民間廠商共同研發能夠在無水、無火條件加熱，且符合民眾口味的新型野戰加熱式餐盒<ref name = "王烱華 5"/><ref name = "王烱華 4">{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20090805/31838011/ |title = 國軍野戰訓練 改洗熱水澡 |author = 王烱華 |date = 2009年8月5日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2017年5月26日 |quote =  }}</ref>。

隨後其配合2007年的[[漢光演習|漢光23號實兵操演]]，小批量製作1,220份野戰加熱式餐盒，分配給[[中華民國陸軍|中華民國陸軍]]、[[中華民國海軍|中華民國海軍]]、[[中華民國空軍|中華民國空軍]]各別遴選3個以上操演單位，提供官兵試食驗證<ref name = "NOWnews 今日新聞 1">{{Cite web |url = http://www.nownews.com/n/2009/08/04/748117 |title = 國軍野戰加熱式餐盒先推四種口味　可無水、無火加熱 |author =  |date = 2009年8月4日 |publisher = [[NOWnews_今日新聞|NOWnews 今日新聞]] |language = zh-hant |accessdate = 2017年5月26日 |quote =  }}</ref>。而在2008年的漢光24號演習，試驗擴大至10,000份，交由陸軍、海軍、空軍、後備軍事動員的實兵操演部隊官兵試吃<ref name = "NOWnews 今日新聞 1"/>。

=== 正式使用 ===
2009年8月4日，國防部聯合後勤司令部臺北聯合保修廠首先發表野戰加熱式餐盒<ref name = "王烱華 4"/>，並持續提供口味試吃<ref name = "王烱華 5"/>。其還規劃在2009年採購920,000份加熱式套餐，普遍提供參與演習訓練的官兵食用，從而逐漸取代餅乾式{{Link-en|野戰口糧|Field ration}}<ref name = "王烱華 5"/>。

2017年，[[中華民國總統|中華民國總統]][[蔡英文|蔡英文]]視導國軍在[[澎湖縣|澎湖縣]]的漢光演習實兵演練與火力展示後<ref name = "王烱華 1">{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20170525/1126221/ |title = 【有片】三軍統帥的第一次　蔡英文陪官兵吃野戰加熱餐盒 |author = 王烱華 |date = 2017年5月25日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2017年5月26日 |quote =  }}</ref>，中午便與基層官兵在帳篷內共同食用野戰加熱式餐盒和野戰口糧<ref name = "數位發展部">{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20170526/1127026/ |title = 【壹週刊】好神奇　小英吃的野戰便當曝光 |author =  |date = 2017年5月26日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2017年5月26日 |quote =  }}</ref><ref name = "呂炯昌 1">{{Cite web |url = http://www.nownews.com/n/2017/05/25/2537940 |title = 漢光演習／蔡英文校閱部隊　大讚野戰餐好吃 |author = 呂炯昌 |date = 2017年5月25日 |publisher = [[NOWnews_今日新聞|NOWnews 今日新聞]] |language = zh-hant |accessdate = 2017年5月26日 |quote =  }}</ref>，並聽取官兵介紹野戰加熱式餐盒的種類及使用，這也是三軍統帥首次食用野戰加熱式餐盒<ref name = "王烱華 2">{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20170526/37662924/ |title = 蔡挺國防改革「當最大靠山」 |author = 王烱華 |date = 2017年5月26日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2017年5月26日 |quote =  }}</ref>。另外[[中華民國國防部|中華民國國防部]]部長[[馮世寬|馮世寬]]也曾在野戰指揮所指導戰訓工作重點後，與官兵共同食用野戰加熱式餐盒，以理解野戰部隊戰時的膳食後勤措施<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20170523/1124824/ |title = 【漢光演習】馮世寬搭黑鷹　首次驗證戰備轉場飛澎湖 |author = 王烱華 |date = 2017年5月23日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2017年5月26日 |quote =  }}</ref>。

== 內容 ==
=== 餐盒設計 ===
野戰加熱式餐盒將套餐封包後<ref name = "王烱華 5"/>，可以保存18個月<ref name = "NOWnews 今日新聞 1"/><ref name = "王烱華 1"/>。野戰加熱式餐盒的設計目的，是在不需要火源和水源的條件下，不須經由烹煮而能隨時加熱<ref name = "數位發展部"/>，提供官兵食用熱食[[米飯|米飯]]餐盒<ref name = "王烱華 4"/>，且過程不會產生煙霧或散發臭味<ref name = "王烱華 1"/>。雖然其份量與外面調理包相似，不過為了應對野戰缺乏水源的環境需要，野戰加熱式餐盒亦自行準備清水，以透過最簡便的方式提供熱能<ref name="蕭照平">{{Cite web |url = http://news.rti.org.tw/news/detail/?recordId=305165 |title = 野戰廚房快狠準 加熱式餐盒更方便 |author = 蕭照平 |date = 2016年10月31日 |publisher = [[中央廣播電臺|中央廣播電臺]] |language = zh-hant |accessdate = 2017年5月26日 |quote =  }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

其中野戰加熱式餐盒藉由水和石灰混合[[放熱反應|放熱]]的化學反應原理<ref name = "蕭照平"/>，共同加熱餐盒的米飯與[[主菜|主菜]]<ref name = "呂炯昌 1"/>，以提供官兵熱食<ref name = "王烱華 5"/>。其中在野外戰地食用時，將米飯包、調理包和加熱包放進加熱袋，之後把加熱用水袋中事前準備的清水<ref name = "王烱華 3">{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20161031/978731/ |title = 野外作戰有熱食、熱水澡　官兵：好幸福哦 |author = 王烱華 |date = 2016年10月31日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2017年5月26日 |quote =  }}</ref>，倒入遇水變熱的加熱包中<ref name = "王烱華 4"/><ref name = "王烱華 1"/><ref name = "數位發展部"/>。隨後水便會與其中的加熱片發生作用，在15分鐘內可以加熱至50度<ref name = "王烱華 1"/>。

過去加熱包因為需要加熱加熱25分鐘才能食用，曾經遭到批評<ref name = "王烱華 5"/>。現在通常野戰加熱式餐盒的加熱時間需7分鐘至10分鐘<ref name = "呂炯昌 1"/>，便能夠獲得熱食食用<ref name = "王烱華 3"/>。每包野戰口糧的費用約110元至150元<ref name = "王烱華 4"/>，國防部聯合後勤司令部則希望藉由大量製作，將成本控制在100元內<ref name = "王烱華 5"/>。

=== 菜餚口味 ===
野戰加熱式餐盒設定難度較高的米飯為主食，並搭配民眾經常食用的菜色，因此需參考歐美國家、新加坡野戰口糧改進<ref name = "王烱華 1"/>。而軍方高層考量野外出操需要補充流失的鹽分，野戰加熱式餐盒的菜餚口味偏向重鹹<ref name = "NOWnews 今日新聞 2">{{Cite web |url = http://www.nownews.com/n/2009/08/04/748120 |title = 野戰加熱式餐盒好吃排行榜　黑胡椒牛腩PK素麻婆豆腐 |author =  |date = 2009年8月4日 |publisher = [[NOWnews_今日新聞|NOWnews 今日新聞]] |language = zh-hant |accessdate = 2017年5月26日 |quote =  }}</ref>，而米飯口感尚仍接受<ref name = "王烱華 5"/>。其內容參考年輕士兵一餐食量，米飯包約230公克，米飯包和主菜包共計450公克<ref name = "NOWnews 今日新聞 1"/>。

2009年，軍方最初選擇的口味有[[黑胡椒|黑胡椒]][[牛腩|牛腩]]、[[東坡肉|東坡肉]]、法式鮮菇雞、[[麻婆豆腐|素麻婆豆腐]]<ref name = "NOWnews 今日新聞 1"/>。之後根據漢光演習驗證期間的提案<ref name = "NOWnews 今日新聞 1"/>，陸續加入[[香菇|香菇]][[雞肉|雞肉]]、[[咖哩雞|咖哩雞]]<ref name = "NOWnews 今日新聞 2"/>、[[咖哩|咖哩]][[牛肉|牛肉]]<ref name = "呂炯昌 2">{{Cite web |url = http://www.nownews.com/n/2017/05/25/2538058 |title = 漢光演習／蔡英文與部隊一同用餐　大讚野戰餐好吃 |author = 呂炯昌 |date = 2017年5月25日 |publisher = [[NOWnews_今日新聞|NOWnews 今日新聞]] |language = zh-hant |accessdate = 2017年5月26日 |quote =  }}</ref>、[[紅燒牛肉|紅燒牛腩]]、紅酒牛肉、[[牛丼|日式牛肉飯]]等[[葷菜|葷菜]]菜單<ref name = "王烱華 1"/><ref name = "呂炯昌 1"/>，共計選出11種口味<ref name = "王烱華 5"/><ref name = "數位發展部"/>。在2008年漢光演習驗證期間，也曾挑選黑胡椒[[素肉|素肉]]、百菇宴等[[素食主義|素食]]品項試吃<ref name = "NOWnews 今日新聞 2"/>，最後決定僅生產素麻婆豆腐<ref name = "NOWnews 今日新聞 1"/><ref name = "王烱華 1"/>。

== 使用 ==
國軍一日份口糧規劃2盒加熱套餐（午餐和晚餐食用），及1包作為早餐的野戰口糧<ref name = "NOWnews 今日新聞 1"/>。在戰場狀況中，由食勤組人員使用野戰炊膳機具準備食物<ref name = "王烱華 3"/><ref name = "呂炯昌 2"/>。而在面臨地形限制無法順利架設野戰廚房實行野炊任務、或臨時出現熱食無法運送的情況下，食勤組便會以事前準備的野戰加熱式餐盒供應<ref name = "蕭照平"/><ref name = "王烱華 3"/><ref name = "呂炯昌 2"/>。其中由各作戰區至補給庫領取，並在6個月內配合任務集中食用<ref name = "NOWnews 今日新聞 1"/>。

另外基於颱風災情可能造成交通中斷或熱食無法追送，軍方針對防災官兵的膳食補給、及預判撤離居民的收容需求，亦會緊急運補野戰加熱式餐盒增援，以提供救災工作的官兵應急食用，例如[[颱風鮎魚_(2016年)|中度颱風梅姬]]期間<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20160926/956502/ |title = 防斷炊　國軍急送加熱式餐盒到花東 |author = 劉智維 |date = 2016年9月26日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2017年5月26日 |quote =  }}</ref>。而對於災後嚴重缺乏民生物資與糧食的情況，國防部亦會主動運補野戰加熱式餐盒和軍用口糧提供給受災居民，包括[[強烈颱風尼伯特_(2016年)|強烈颱風尼伯特]]等災情<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20160710/904863/ |title = 【更新】徒手搬大石　國軍搶通太麻里聯外道路 |author = 王烱華 |date = 2016年7月10日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2017年5月26日 |quote =  }}</ref><ref>{{Cite web |url = http://www.appledaily.com.tw/appledaily/article/headline/20160711/37302963/ |title = 「颳風像打仗」 綠島木屋全倒 |author = 林金聖 |date = 2016年7月11日 |publisher = 《[[蘋果日報_(台灣)|蘋果日報]]》 |language = zh-hant |accessdate = 2017年5月26日 |quote =  }}</ref>。

== 評價 ==
大部分官兵認為國軍的野戰加熱式套餐餐盒，比起餅乾為主的野戰口糧有著大幅改善<ref name = "王烱華 5"/>。在2007年的適用滿意度資料中，官兵滿意度達到80%以上<ref name = "NOWnews 今日新聞 2"/>。不過與美軍加熱式[[MRE口糧|MRE口糧]]相比，美國野戰口糧加熱時間短、內容豐富，亦能提供豐富熱量<ref name = "王烱華 5"/>；僅有主餐包的國軍口糧內容簡單，且紙餐盒方式不利於野戰攜帶<ref name = "王烱華 5"/>。

而口味滿意度排名部分，依序為黑胡椒牛腩（84%）、東坡肉（82.51%）、咖哩牛肉（80.67%），黑胡椒牛腩是最受到官兵歡迎的口味<ref name = "王烱華 1"/>。不過基於均衡飲食考量，由滿意度僅60%的法式鮮菇雞取代咖哩牛肉，成為首批野戰加熱式餐盒葷菜口味<ref name = "NOWnews 今日新聞 2"/>。另外，素麻婆豆腐在軍中也獲得86%滿意度<ref name = "NOWnews 今日新聞 2"/>，為官兵最喜歡的素食野戰加熱式餐盒<ref name = "王烱華 1"/>。蔡英文在食用紅燒牛腩口味的野戰加熱式餐盒後，亦曾給予正面評價<ref name = "王烱華 2"/>。
