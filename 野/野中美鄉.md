'''野中美鄉'''（{{lang-ja|野中 美郷}}，{{bd|1991年|4月20日|catIdx=Nonaka Misato}}），是日本女子團體[[AKB48|AKB48]] Team B的前成員，原研究生（6期生）。出生於[[福岡縣|福岡縣]]，現在住在神奈川縣。

== 經歷 ==
; 2009年
* 8月23日在「{{link-ja|AKB104選拔成員組閣祭|AKB48全国ツアー (2009年)}}」演唱會的晚場公演中，公布將昇格成為Team K的成員。

; 2010年
* 4月-5月期间，创下了连续进行48场剧场公演的记录。

; 2014年
* 2月24日，於{{link-ja|Zepp DiverCity Tokyo|Zepp}}舉辦的「AKB48集團大組閣祭」中宣布留任Team B<ref>{{cite web|url=http://www.akb48.co.jp/page/daisokaku/list.php?after=1&group=akb48 |script-title=ja:AKB48グループ大組閣祭り |publisher=AKB48官方網站 |language=ja |date=2014-02-24 |accessdate=2014-04-13}}</ref>。

* 2月28日，於AKB48的冠名節目[[AKB48的你、是誰？|AKB48的你、是誰？]]中宣布畢業。<ref>{{cite web|url=http://ameblo.jp/akihabara48/entry-11784022248.html |script-title=ja:お知らせ |publisher=AKB48官方Blog |language=ja |date=2014-02-28 |accessdate=2014-02-28}}</ref>

* 4月22日，於AKB48劇場舉行畢業公演<ref name="畢業">{{cite web|url=http://ameblo.jp/akihabara48/entry-11821722229.html |title=野中美郷の卒業について |publisher=AKB48官方Blog|language=ja |date=2014-04-13 |accessdate=2014-04-13}}</ref>。

* 5月31日，與經紀公司的合約結束<ref>{{cite web |url=http://akb48teamogi.jp/news/3918 |title=野中美郷　契約終了のお知らせ |publisher=AKB48TeamOgi官方網站 |language=ja |date=2014-05-08 |accessdate=2014-05-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20140508223910/http://akb48teamogi.jp/news/3918 |archivedate=2014-05-08 }}</ref>並從演藝界引退<ref>{{cite web |url=http://blog2.akb48teamogi.jp/nonakamisato/?p=175 |title=…お知らせです… |publisher=野中美郷於AKB48TeamOgi的官方Blog |language=ja |date=2014-05-08 |accessdate=2014-05-08 |archive-url=https://web.archive.org/web/20140512215849/http://blog2.akb48teamogi.jp/nonakamisato/?p=175 |archive-date=2014-05-12 |dead-url=yes }}</ref>。

* 9月23日，參加於[[橫濱國際平和會議場|橫濱國際平和會議場]]舉辦的『「勇往直前」劇場盤發售紀念大握手會，』至此以AKB48成員身分參與的活動全部結束<ref name="畢業"/>。

== 人物 ==
* [[大島麻衣|大島麻衣]]與[[佐藤亞美菜|佐藤亞美菜]]、[[中田千智|中田ちさと]]、[[前田敦子|前田敦子]]等人叫她「もなか」，而Team研究生的成員與同期的[[高城亞樹|高城亞樹]]常叫她「みちゃ」。跟[[梅田彩佳|梅田彩佳]]是同鄉。
* 與[[梅田繪理香|梅田繪理香]]（前[[℃-ute|℃-ute]]成員）唸同一所中學，交情很好。
* 18年之間都不知道自己的血型，在2009年7月26日的Team K公演中因為[[鈴木紫帆里|鈴木紫帆里]]的發言，野中才知道自己的血型是A型。本人一直以為是O型。
* AKB48总制作人秋元康于2012年3月14日，在其个人g+上公布以野中美乡命名的2012年AKB48全国47都道府县巡回演唱会的名字[野中美郷，移动。〜在47都道府县相会吧〜]
