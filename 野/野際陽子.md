'''野際陽子'''（{{lang-ja|野際 陽子}}，{{bd|1936年|1月24日|2017年|6月13日|catIdx=Nogiwa Youko}}），出生於[[日本|日本]][[富山縣|富山縣]][[富山市|富山市]]。[[日本|日本]][[女演員|女演員]]、[[女藝人|女藝人]]、[[配音員|配音員]]、[[旁白|旁白]]、[[主持人|主持人]]。前[[NHK|NHK]]主播。屬於{{tsl|ja|ラヴァンス|L'avance}}經紀公司。專長是[[英語|英語]]、[[法語|法語]]<ref name="名鑑">{{Cite web |url= http://talent.yahoo.co.jp/pf/profile/pp1416 |script-title= ja:野際陽子 - プロフィール |work= YAHOO! 人物名鑑 |publisher= 日本タレント名鑑 |language= ja |accessdate= 2011-08-26}}</ref>。

== 基本資料 ==

野際陽子出生於日本富山縣富山市，三歲以後在東京都杉並區天沼長大。<ref name="脱いでみようか">{{lang|ja|自著　「脱いでみようか」}}</ref>畢業於立教大學文學部英美文學系。大學時代參加了ESS的英語戲劇以及{{tsl|ja|劇団テアトルジュンヌ|Theatre Jeune劇團}}，演出一些作品。

大學畢業後，1958年加入[[NHK|NHK]]擔任主播，為期三個月的研修後結束後便到[[NHK名古屋放送局|NHK名古屋放送局]]赴任。<ref name="脱いでみようか"/>1962年3月從NHK退職。

1963年演出電視劇《{{tsl|ja|悲の器}}》，此為野際以女演員身分出道之作。1968年開始演出電視劇《[[龍虎群英|龍虎群英]]》，播出的五年間締造30%亮眼的收視紀錄。此外由於非常想唱歌而向導演直接提出要求<ref name="NHKラジオ深夜便">{{lang|ja|「女優が語る私の人生」 [[ラジオ深夜便|ラジオ深夜便]]、2010年8月7日放送分。}}</ref>，就這樣取得了演唱主題曲〈{{lang|ja|非情のライセンス}}〉的機會。1972年與合作演出的男演員[[千葉真一|千葉真一]]結婚。1975年生下長女真瀨樹里。

1992年在電視劇《{{tsl|ja|ずっとあなたが好きだった|一直都愛你}}》中演出非常溺愛兒子的婆婆一角，從這以後接演媽媽或婆婆的角色的機會倍增。1994年想去美國的千葉<ref name="千葉真一">{{Cite book<!--|和書-->|author = [[千葉真一|千葉真一]]|year = 2008|script-title = ja:千葉真一 改め 和千永倫道|publisher = [[山と渓谷社|山と渓谷社]]|pages = 113 - 114|isbn = 4635340228|language=ja}}</ref>跟希望住在日本的野際因為意見不合而[[離婚|離婚]]。

近年來因為在電視劇及廣告中常常是穿著[[和服|和服]]的，因此被推舉為「適合穿和服的女演員」（{{lang|ja|着物の似合う女優}}）。

日本時間2017年6月13日因肺腺癌在東京的某間醫院病逝<ref>中廣新聞網，[https://tw.news.yahoo.com/日知名女星野際陽子癌逝-享年81歲-035934398.html 日知名女星野際陽子癌逝]</ref>，終年81歲。
