坚果Pro2是[[锤子科技|锤子科技]]在2017年秋季新品发布会上发布的准旗舰手机，也是锤子的第一款全面屏手机。搭载高通骁龙660平台，1200+500万后置双摄和1600万前置单摄，支持18W有线快充，双卡双待。原生搭载[[Smartisan_OS|Smartisan OS]] 4.1，基于[[Android|Android]] N。
==设计==
坚果Pro2采用18：9的[[多点触控|多点触控]]显示屏，官方宣称“全面屏（Almost）”<ref>{{cite web |author1=Smartisan |title=坚果Pro2-设计 |url=https://www.smartisan.com/jianguopro2/design}}</ref>。坚果Pro2首次使用双面玻璃加金属中框，背面的logo和指纹识别器合二为一，正面的光距感应器和提示灯全部隐藏，双摄像头系统中的副摄像头做小且不凸起<ref>{{cite web |author1=电脑报 |title=用了坚果Pro2之后，我觉得老罗真的要成了 |url=https://baike.baidu.com/tashuo/browse/content?id=9a7035925f762e3e51cdea82&lemmaId=&fromLemmaModule=pcBottom}}</ref>。拥有碳黑色、纯白色、浅金色、酒红色四种配色。
==配置==
坚果Pro2搭载[[高通骁龙|高通骁龙]]660平台，Adreno 512 GPU，单核最高主频2.2GHz。拥有4或6GB RAM和32、64、128或256GB ROM。支持双卡双待。屏幕[[分辨率|分辨率]]为2160*1080，采用康宁第三代大猩猩玻璃。支持指纹识别和摄像头面部识别。
1200+500万后置双摄，支持4K视频拍摄。主摄采用IMX 386传感器，副摄采用 S5K4E8传感器。前摄1600万像素，支持智能美颜和1080P视频拍摄。
坚果Pro2采用3500[[毫安时|毫安时]]的电池，支持18W有线快充，不支持[[无线充电|无线充电]]。<ref>{{cite web |author1=Smartisan |title=坚果Pro2-技术规格 |url=https://www.smartisan.com/jianguopro2/specs}}</ref>
==连接性==
坚果Pro2配备一个[[Type-C|Type-C]]接口，没有3.5mm耳机孔。全系支持[[蓝牙[[5.0、5G_WiFi和[[GPS|蓝牙[[5.0、5G WiFi和[[GPS]]，6+256GB版本还支持[[NFC|NFC]]。<ref>{{cite web |author1=Smartisan |title=坚果Pro2-技术规格 |url=https://www.smartisan.com/jianguopro2/specs}}</ref>
==质量问题==
坚果Pro2的质量问题主要出在屏幕。部分机器屏幕严重[[拖影|拖影]]，边缘触控不灵敏，而且屏幕密封性差，可能进水。<ref>{{cite web |author1=知乎 |title=如何评论坚果pro2的屏幕出现了很多问题（官方有回复）？ |url=https://www.zhihu.com/question/68604927}}</ref>2018年夏天，还有机器的屏幕开胶脱落<ref>{{cite web |author1=凤凰网 |title=屏幕脱落？坚果Pro2曝质量问题！ |url=http://tech.ifeng.com/a/20180630/45044711_0.shtml}}</ref>。
