坚果R1是[[锤子科技|锤子科技]]在2018年5月15日和[[坚果TNT工作站|坚果TNT工作站]]一同发布的旗舰手机，搭载高通骁龙845处理器和1200+2000万双后置摄像头，支持QC4+18W有线快充和10W无线充电，这也是第一款支持无线充电的[[锤子|锤子]]手机，原生搭载[[Smartisan_OS|Smartisan OS]] 6.0，支持TNT。<ref>{{cite web |author1=Smartisan |title=坚果R1-技术规格 |url=https://www.smartisan.com/r1/specs}}</ref>
==设计==
坚果R1采用金属中框加双面玻璃的设计，黑色版的中框上为装饰加了一条金色的细线。有碳黑色、纯白色和孔雀蓝三个配色，指纹识别和logo二合一。<ref>{{cite web |author1=Smartisan |title=坚果R1-设计 |url=https://www.smartisan.com/r1/design}}</ref>背部的双摄像头横排放置，凸起。正面的屏幕采用“美人尖”的设计，类似于sharp crystal和essential Phone。<ref>{{cite web |author1=engadget |title=坚果R1评测-生不逢时 |url=https://cn.engadget.com/2018/07/05/smartisan-r1-review/}}</ref><ref>{{cite web |author1=Smartisan |title=坚果R1-技术规格 |url=https://www.smartisan.com/r1/specs}}</ref>
==配置==
搭载高通骁龙845SOC平台，10nm制程，Adreno 630图形处理器，搭载AI引擎。全系标配UFS 2.1[[闪存|闪存]]规格，运行内存6GB起步，存储最高1TB。蓝牙5.0，双卡双待。电池容量3600毫安时，支持18W有线快充和10W无线快充。6.17英寸LCD显示屏，采用[[康宁|康宁]]第三代大猩猩玻璃，分辨率2242*1080，支持压力感应。
1200+2000万双摄后置，支持4K视频拍摄；2400万前置单摄，可以拍摄全高清视频。配备[[线性振动马达|线性振动马达]]。支持指纹解锁和面部识别。
==软件==
坚果R1运行SmartisanOS 6.x，该操作系统基于[[Android|Android]] O进行优化。支持游戏模式和人物相册，同时增强了相机算法<ref>{{cite web |author1=Smartisan |title=坚果R1-操作系统 |url=https://www.smartisan.com/r1/os}}</ref>。不过最大的升级是支持TNT工作站。有关内容请到[[坚果TNT工作站|坚果TNT工作站]]查看。
==售价==

==质量问题==
有用户反映，坚果R1白色版的听筒存在毛刺。坚果R1的镜头容易被刮花，此问题后来被修复。坚果R1部分机器屏幕开胶，维修之后二次开胶，而且影响压感屏的使用。<ref>{{cite web |author1=腾讯新闻 |title=坚果R1已知品控问题汇总 |url=https://new.qq.com/omn/20180714/20180714A0ZQ4L.html}}</ref>
