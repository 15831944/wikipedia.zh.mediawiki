'''番石榴'''（[[学名|学名]]：''{{lang|la|Psidium guajava}}''），为[[热带|热带]]、[[亚热带|亚热带]]水果，原产[[美洲|美洲]]，现中国[[华南|华南]]地区及[[四川盆地|四川盆地]]均有栽培。

== 名稱緣由 ==
*因外型像[[石榴|石榴]]（即安石榴，屬[[安石榴科|安石榴科]]），卻產自外地因而得名。此外，還有多個名稱，包括'''番稔'''、'''番桃'''、'''吉卜賽果'''等。
*因其聞上去有一股臭味，故又名'''雞屎果'''或'''雞矢果'''。<ref>[http://www.hljh.tcc.edu.tw/teach/校園植物/番石榴/番石榴.htm 番石榴] {{webarchive|url=https://web.archive.org/web/20111130150607/http://hljh.tcc.edu.tw/teach/%E6%A0%A1%E5%9C%92%E6%A4%8D%E7%89%A9/%E7%95%AA%E7%9F%B3%E6%A6%B4/%E7%95%AA%E7%9F%B3%E6%A6%B4.htm |date=2011-11-30 }}，后里國中：植物面面觀</ref>
*[[閩南語|閩南語]]稱番石榴為'''菝仔'''、'''林菝仔'''、'''那菝仔'''或'''那菝'''（nâ-po̍at-á），[[客家語|客家語]]亦作'''拔仔'''（po̍at-á），[[潮汕|潮汕]]以'''木仔'''称呼，[[台灣|台灣]]一般俗寫為'''芭樂'''，[[澳門|澳門]]及[[珠海|珠海]]一帶舊語又叫作'''花拈'''。<ref>源出自清代澳門中文檔案詞語。</ref>
*[[金沙江|金沙江]]中游干热河谷地区称番石榴为'''白果'''或'''红心果'''<ref>[http://www.docin.com/p-618143920.html 攀枝花地区野生番石榴叶提取物中Zn、Mg、Cr、V等微量元素分析] {{webarchive|url=https://web.archive.org/web/20140202143202/http://www.docin.com/p-618143920.html |date=2014-02-02 }}</ref>，[[文革|文革]]时期亦有称为'''革命果'''。

== 分布 ==
番石榴是熱帶[[喬木|喬木]]或[[灌木|灌木]]，原產於[[中美洲|中美洲]][[墨西哥|墨西哥]]到[[南美洲|南美洲]]北部，被引入到世界各地的[[熱帶|熱帶]]和[[亞熱帶|亞熱帶]]地區取果實用來食用。在一些國家，種植、加工和出口番石榴果實形成規模龐大的產業。甜味種糖度高，果肉甜脆，適合鮮食；酸味種可加工製果汁、果醬。尚可鹽漬、甘草漬、製罐、製蜜餞、果乾和製酒；葉片和未熟果曬乾研磨成末，可製成健康食品「菝仔茶」。番石榴已經侵入牧草地和野地，排擠當地植物，部分原因是它能夠生長在各種不同的土壤和氣候。在某些官方排名，番石榴是[[入侵物種|入侵性]]最高的物種之一。番石榴可以存活30至40年。

== 特徵 ==
番石榴為灌木或小喬木，通常有10公尺高，偶爾高達20公尺，根部淺，樹皮光滑，淡紅褐色，嫩枝4角有柔毛，樹皮會大片剝落，暴露樹皮下綠色層，主幹通常直徑約25公分，最大可達60公分，外表骨質嶙峋，樹葉對生，卵形，橢圓形或長圓形，葉背面有柔毛，往往相當易脆，葉脈明顯，側脈10至20對，葉片大多7至15公分長，3公分寬，基部圓形，暗綠色，壓碎後有香味，果肉甜酸，粒狀多汁，種子為黃色，腎形。未成熟時果實硬，綠色，粘和澀。

== 營養成分 ==
番石榴為低熱量、高纖維、水分高，易有飽足感之水果，是糖尿病和減肥者的最佳選擇之一。因屬熱帶高維他命Ｃ水果，亦為天然美白的聖品，對於牙齦健康也有幫助。也是人體攝取維他命C重要的來源。同時富含維生素，纖維與抗氧化劑的番石榴更是天然的防老抗癌防癌的水果。

成熟的番石榴果实，味道香甜可口。果皮为浅绿色，脆薄，食用时一般不用削皮。果肉厚、甜。其含有较丰富的蛋白质、[[维生素|维生素]][[维生素A|A]]、[[维生素C|C]]等营养物质及[[磷|磷]]、[[钙|钙]]、[[镁|镁]]等微量元素，是非常好的保健食品。

番石榴葉萃取物中含多元[[酚|酚]]成分，能刺激脂肪細胞上[[胰島素|胰島素]]受體-{zh-hans:的;zh-hant:之;}-[[磷酸化|磷酸化]]及[[葡萄糖|葡萄糖]]轉運蛋白-{zh-hans:的;zh-hant:之;}-[[轉位|轉位]]，具有類似胰島素-{zh-hans:的;zh-hant:之;}-功能，有助於[[糖尿病|糖尿病]]-{zh-hans:的;zh-hant:之;}-控制。

== 常見病蟲害 ==
由於番石榴多栽種於熱帶與亞熱帶地區，氣候及環境適於疫病蟲害的發生與蔓延，需特別注意疫病防治。

=== 蟲害及蟎害 === 
*[[東方果實蠅|東方果實蠅]]
:果實蠅雌成蟲以產卵管將卵產於果皮內，幼蟲孵化後鑽食果肉導致腐爛，嚴重時造成落果{{sfn|何琦琛等作|2005|p=18}}。
*[[粉介殼蟲|粉介殼蟲]]
:粉介殼蟲的若蟲及成蟲性喜歡棲居於枝椏、葉背及果實等部位，吸食植株養液，分泌大量蜜露引發煙煤病，汙染葉片與果實，影響光合作用，致被害枝葉生長不良，提早落葉、落果或果味變酸{{sfn|何琦琛等作|2005|p=25}}。
*[[黑疣粉蝨|黑疣粉蝨]]
:黑疣粉蝨的成蟲大部分集中在新葉上棲息、產卵，卵孵化後，若蟲即在葉背刺吸汁液，使植株發育不良，並分泌蜜露誘發煙煤病{{sfn|何琦琛等作|2005|p=29}}。
*[[螺旋粉蝨|螺旋粉蝨]]
:螺旋粉蝨的若蟲在葉背吸食植株養液，分泌白色粉狀及絲狀物，並分泌蜜露誘發煙煤病{{sfn|何琦琛等作|2005|p=32}}。

== 大眾文化 ==
*在台湾，「芭樂」一詞被用來暱稱[[手榴彈|手榴彈]]。最常在[[第一人稱射擊遊戲|第一人稱射擊遊戲]]（例如[[反恐精英|反恐精英]]）中使用，使用方法如：請你吃芭樂<ref>{{cite news|url=http://news.ltn.com.tw/news/local/paper/788770|title=請債主吃「土豆」「芭樂」 涉恐嚇起訴|publisher=自由時報|date=2014年6月19日|accessdate=2014年9月11日|author=余瑞仁}}</ref>。
*在台湾的[[網路|網路]]上，芭樂一词被稱指為「不實的[[消息|消息]]」，因此未能兌現的[[支票|支票]]，則被稱為「[[芭樂票|芭樂票]]」。
