'''{{ruby-py|番禺|-{zh-cn:pānyú;zh-tw:ㄆㄢㄩˊ;zh-hk:pun<sup>1</sup>jyu<sup>4</sup>}-}}'''（官方音譯：{{lang|en|Panyu}}，传统外文：{{lang|en|Punyü}}<ref>中华民国《1947年广东政区及邮政式广东地名一览》，大部分根据粤语标准音广州话拼写，其他部分由潮汕、客家、官话或其他方音拼写而成。</ref>）是[[中华人民共和国|中华人民共和国]][[广东省|广东省]][[广州市|广州市]]于2000年设立的[[副地级市|副地级]][[市辖区|市辖区]]，為原'''禺南'''地區，位于广东省及广州市中南部，[[珠江|珠江]]口西北岸，处于[[珠三角|珠三角]]腹地。番禺区土地肥沃、水陆交通便利，自古以来就是[[岭南地区|岭南地区]]的鱼米之乡。旧时[[番禺_(古代)|番禺]]（番禺縣）与同样经济富庶、文化互通的[[南海区|南海]]、[[顺德区|顺德]]合称“[[南番顺|南番顺]]”，是[[岭南文化|岭南文化]]的发源地之一。番禺区的区划代码为440113，区政府驻地邮政编码为511400（其他地区各有异同）。

==地理== 
番禺全区位于[[北回归线|北回归线]]以南，[[北纬|北纬]]二十三度（23°N）横贯区境。

番禺以东面临狮子洋水道西面与[[佛山市|佛山市]][[南海区|南海区]]、[[顺德区|顺德区]]毗邻，南与[[南沙区|南沙区]]一衣带水；北部为海拔50米以下低丘，与广州市[[荔湾区|荔湾区]]芳村、[[海珠区|海珠区]]、[[黄埔区|黄埔区]]南部一江之隔。

地势主要为丘陵与平原交错之肥沃土地，由西部向东以及南部倾斜，陆地最高点为西部[[大夫山|大夫山]]之顶峰大乌岗，海拔高程约226公尺。主要河流有[[珠江|珠江]]、[[西江|西江]]及[[北江|北江]]支流等，区内河网星罗棋布。冬季时有[[咸潮|咸潮]]溯流侵袭本区。

番禺属南[[亚热带海洋性季风气候|亚热带海洋性季风气候]]，年均降水量1646.9毫米，年均温22.2℃，“无气象意义上之冬季”，年平均日照时数1807.6小时。热量充足，降水丰沛，因能达到“水热同期”亦有利于农作物生长。

== 经济 == 
===基建===

響應[[廣州市人民政府|廣州市人民政府]]之“南拓”計劃以及“廣州時尚創意都會區”方針。番禺區在建有大量巨資投入之基建。

=== 主要产业 === 
====旅游业====
番禺区旅游业繁荣兴旺，旅游资源丰富，[[长隆旅游度假区|长隆旅游度假区]]成为全国首批[[中国国家5A级旅游景区|国家级5A景区]]。石楼镇莲花山及海鸥岛常年吸引珠三角各地民众前来祈福观光及假日[[农家乐|农家乐]]之“好去处”;全区共有3个[[全国重点文物保护单位|全国重点文物保护单位]]、4个省级、10个市级文物保护单位。

====工业====
同时番禺区工业也相当发达，为广州“南拓”重要发展区。主要产业有[[珠宝设计|珠宝设计加工业]]、[[光电工程|光电制造业]]、[[汽车制造|汽车制造业]]、动漫后期编辑业、[[机电工程|机电制造业]]、[[化工业|日化工业]]等。

====产业效益====
[[File:_Panyu_Yifa_Walking_Street.JPG|thumb]]的易发步行街。]]
番禺区2017年[[国内生产总值|国内生产总值]]达1972.24亿元人民币，社会消费品零售总额1229.35亿元人民币，分别位列广州十区二市第四及第三<ref>数据来源于广州市统计局公布的《广州统计信息手册（2018年）》</ref>。

====农业====
番禺农业经近年来持续的城市化发展，大片土地被政府或财团收购，规模早已大不如前。现大多数集约农地位于东部[[石碁镇|石碁镇]]、[[石楼镇|石楼镇]]，而[[石楼镇|石楼镇]]之[[海鸥岛|海鸥岛]]是以临海渔业和基耕鱼塘为特色。

===知名企业===
番禺区内的知名企业有[[广州番禺电缆集团|广州番禺电缆集团]]、[[广东长隆集团|广东长隆集团]]、[[广州立白企业集团|广州立白企业集团]]、[[广州汽车|广汽乘用车有限公司]]、[[珠江鋼管|广州珠江钢管有限公司]]、[[箭牌|箭牌糖果(中国)]]、[[日立|广东日立电梯]]、广州香江集团、三雄极光照明、自由鸟服装等。

===商业区===
*市桥商圈：番禺区传统的商业中心区，位于[[市桥街道|市桥旧城区]]，坐落有大北路步行街、易发步行街、[[新大新公司|新大新百货]]、繁华盛世等商业地标。
*洛溪-大石（番禺新城）商圈：位于番禺区北部，与[[海珠区|海珠区]]一桥之隔，随着房地产业兴旺而发展起来的商圈，有吉祥道步行街、新地购物广场、大石城、富丽城、建华汇等商业中心。
*万博商圈：位于华南板块[[番禺大道|番禺大道]]北与汉溪大道交界处，现有[[天河城|天河城]]、海印又一城、万博中心、番禺[[万达广场|万达广场]]，在建有长隆生态旅游城、保利大都汇、奥园国际中心等大型商业综合体，是番禺区未来的商业中心。
*南站商圈：建设中的商业区，以[[广州南站|广州南站]]为依托，以商务商贸为主导功能的现代服务业集聚区，届时将建有继[[花城汇|花城汇]]後广州最大规模的地下空间商业区。

=== 房地产业 ===
[[File:PanYu_Clifford_Estate_hotels.jpg|thumb]]
番禺区由于离广州中心区较近，交通方便（有多条[[地铁|地铁]]及快速路接驳），而且没有市区那么喧闹，空气质素及绿化环境较市区好，故房地产业十分发达，建有多个超千亩的大型社区，包括星河湾、丽江花园、洛溪新城、锦绣香江、[[祈福新邨|祈福新邨]]、[[碧桂园|华南碧桂园]]、[[广州亚运城|广州亚运城]]、广州[[雅居乐|雅居乐]]花园、[[南国奥林匹克花园|南国奥林匹克花园]]、富豪山庄等，是不少广州人及香港人退休及渡假的乐园。 1997年以前屋苑的业主多是北上之港人，但现在屋苑的业主多为在广州市市区上班的白领。

== 历史 == 
{{main|广州历史|番禺 (古代)}} 
*1921年2月1日（民国十年），广州市政厅成立，广州正式设市。廣東省城內番禺县府保留運作。
*1933年（民国二十二年），番禺县府从廣東省城內外遷至禺南[[新造|新造]]。
*[[抗日战争|抗日战争]]期间番禺沦陷后，原县政府輾轉遷移去廣東[[三水区|三水]]、沙坪、威井等地，而[[汪精卫|汪精卫]]政府另立的番禺县府驻於广州市[[东山区_(广州市)|东山区]]。 
*1945年光復后，番禺县政府移治禺南[[市桥|市桥]]，初在先锋巷谢氏祠堂。
*1949年10月[[中国人民解放军|中共部隊]]進駐番禺，地方改由广东省珠江三角洲地方军事管制委员会管治。
*1950年3月至1952年11月，隶属珠江地区专员公署。
*1952年12月至1956年1月，隶属粤中行政公署。
*1956年2月起，隶属[[佛山地区|佛山]]专员公署。
*1958年12月15日至1959年6月10日，番禺、[[顺德|顺德]]合并为番顺县，县治设原順德縣的縣治[[大良街道|大良镇]]，其间一度改隶[[广州专区|广州]]专员公署。
**番禺縣地域經歷多番調劃，到1958年為止，剩原禺南地區（前沙灣司及茭塘司大部），時年轄內人口25萬，耕地65萬畝。
*1959年6月10日恢复两县建置，并将[[中山县|中山县]]属的大岗、万顷沙、南沙、黄阁等地划属番禺（萬頃沙、南沙在民國到共和國初是歸屬[[東莞|東莞]]），县府沿设[[市桥|市桥]]，確定番禺現行地域格局。
*1975年1月，改隶广州市。
*1992年5月20日，民政部批复（民行批[1992]49号）：经国务院批准，撤销番禺县，设立番禺市（县级），仍由广州市管辖。 同年6月18日，番禺举行撤县设市挂牌仪式。
*2000年5月21日，国务院同意广东省撤销番禺市设立广州市番禺区，同年7月10日，番禺区举行撤市设区挂牌仪式。
*2005年4月28日，原属番禺区的[[广州南沙经济技术开发区|广州南沙经济技术开发区]]获国务院批准新设立[[南沙区|南沙区]]，同年五月附同鄰近的[[黄阁镇|黄阁镇]]、[[万顷沙镇|万顷沙镇]]、[[横沥镇|横沥镇]]、[[灵山镇|灵山镇]]部分地区正式改为[[南沙区|南沙区]]。
*2006年1月25日，撤銷靈山鎮，其行政區域分別併入欖核鎮、大崗鎮和東涌鎮。
*2012年12月1日，原属番禺区的[[东涌镇|东涌镇]]、[[大岗镇|大岗镇]]、[[榄核镇|榄核镇]]正式划归[[南沙区|南沙区]]管辖。<ref>[http://informationtimes.dayoo.com/html/2012-12/01/content_2042393.htm 2012年12月1日大洋网-信息时报《番禺三镇今起划归南沙》] {{webarchive|url=https://web.archive.org/web/20121210201742/http://informationtimes.dayoo.com/html/2012-12/01/content_2042393.htm |date=2012年12月10日 }}</ref>

==社会 民生 ==
===主要事件===
==== 撤市设区 ==== 
共和國時代，將番禺（禺南地區）改劃为广州市辖区的构想当年在番禺民间、政府、公务员系统和国企都有极大反对声音，{{fact|民间反对声音主要为不想番禺失去经济发展的自主性，亦担心番禺施行已在广州实施但不适合番禺的政策（如停止核发摩托车牌号），而公务员反对声音主要为担心工资下调，因当时番禺公务员工资平均水平远在广州市区公务员之上}}。当时番禺政府曾组织专家学者从文化、历史、经济等各方面论证撤市设区不适合番禺，番禺人大代表于1999年广州市人大会议联名向大会提交《关于撤市改区后对番禺产生严重影响应慎重考虑的议案》<ref>[http://www.panyu.gov.cn/portal/jsp/pynj/nr.jsp?xxnrId=5879&catId=4035&isFist=true&year=2000 《番禺年鉴（2000）》] {{webarchive|url=https://web.archive.org/web/20140808053452/http://www.panyu.gov.cn/portal/jsp/pynj/nr.jsp?xxnrId=5879&catId=4035&isFist=true&year=2000 |date=2014-08-08 }}</ref>。其时广州市政府承诺以后将[[广州地铁三号线|广州地铁三号线]]通往番禺，撤市设区后给予番禺政府三年行政独立过渡，撤市设区后将重点发展番禺等等作为谈判条件。自1997年广州市提出设区设想，双方拉锯数年。直到2000年5月21日国务院下发《关于同意广东省撤销番禺市和花都市设立广州市番禺区和花都区的批覆》，番禺最终撤市设区，改劃为广州市辖区。时任广州市市长林树森称，“这一步是我在广州工作期间城建方面最难的一步”<ref>[http://news.dichan.sina.com.cn/2010/10/25/228646.html 南方都市报《新区一路向南 老城华丽转身》] {{webarchive|url=https://web.archive.org/web/20140810060127/http://news.dichan.sina.com.cn/2010/10/25/228646.html |date=2014-08-10 }}</ref>。

==== 土地利用 ==== 
1998年，由于在番禺影响盛极一时的“梁柏楠”案，使得番禺华南板块意外“诞生”。有见及此，番禺撤市设区后，广州市政府对番禺区收紧土地供应，而番禺工业多为密集式私营办厂的生产或加工，工业土地需求大，故政策制约了番禺的工业发展；广州市政府的政策禁止农村农民新增自建房，番禺很多有新增家庭人口的农民家庭的居住需求得不到满足；自合番禺设区之后，大量市区人口到番禺置业，使番禺成为广州的住宅区。另外很大部分在广州工作、生活或渡假的香港人也在番禺区置业，如祈福新村与碧桂园等楼盘有八成住户是香港人。

==== 番禺“反烧” ====
指发生在2009年下半年番禺区居民自发抵制政府兴建垃圾焚烧发电厂的公共事件。2002年，广州市番禺区人大通过了番禺区生活垃圾处理系统规划，该规划确定了未来生活垃圾设施的选址地点；2006年8月，广州市规划局批准了番禺区生活垃圾综合处理厂的选址地点。到2009年9月，番禺市政林园局表示，生活垃圾发电厂的征地工作基本完成，地点规划在番禺区大石街会江村区域；10月，番禺垃圾焚烧发电厂有关情况新闻通报会召开，提出番禺垃圾焚烧厂环评启动，公众可参与；建设垃圾焚烧厂项目引发周边居民的广泛担忧，11月23日，番禺区三百余居民自发就垃圾焚烧厂选址问题向政府上访<ref>{{cite web|author1=中国百科网|title=由番禺垃圾焚烧场选址事件引发的法学思考|url=http://www.chinabaike.com/t/31251/2014/0422/2110480.html|website=http://www.chinabaike.com}}</ref>，期间，广州当地媒体亦就此议题发动广泛的舆论问政<ref>{{cite web|author1=笑蜀|title=从垃圾危机中捕捉公民社会曙光|url=http://www.chinavalue.net/Finance/Article/2010-2-28/190552.html|website=http://www.chinavalue.net}}</ref>，事件遂发酵成一起全国性公共议题，引来广东省以外的媒体参与讨论，不少网友亦在社交媒体上对选址程序发起质疑，最终，社会舆论压力迫使番禺区政府将建厂项目延后实施，表示将重新进行垃圾焚烧厂选址。番禺“反烧”事件一般被视为大陆公民社会发育进程中的一大标志性事件，与“厦门反PX项目运动”并称。

==== 公共交通 ==== 
番禺区有番字號的地區內[[巴士|巴士網絡]]，並不鏈接广州城区巴士線網，广州城区有少量巴士線则进出番禺。番禺公共汽车有限公司发行的[[番禺莲花卡|番禺莲花卡]]，只能用于番禺公交。早年[[羊城通|羊城通]]不能用于乘坐番禺巴士，及後於2006年11月已可使用。由於使用番禺莲花卡乘搭番禺巴士价格为[[人民币|人民币]]1.50元，使用羊城通为2元<ref>http://informationtimes.dayoo.com/html/2010-04/12/content_927976.htm 番禺花都有望享羊城通优惠（内文提到广州市财政只负责中心区的公交票价优惠，故在番禺使用羊城通没有优惠）</ref>，所以番禺莲花卡在番禺被广泛使用。直至2013年1月1日，番禺区内的巴士路線才开始正式执行与广州市中心区统一的公交票价优惠政策，即普通市民持同一张羊城通卡在一个自然月内乘搭巴士或地铁累计达到15次后，从第16次开始每次享受6折优惠<ref>[http://pyrb.dayoo.com/html/2012-12/31/content_2097088.htm 2012年12月31日区党报番禺日报《番禺区明年1月1日开始实施公交票价优惠政策》] {{webarchive|url=https://web.archive.org/web/20140118225118/http://pyrb.dayoo.com/html/2012-12/31/content_2097088.htm |date=2014年1月18日 }}</ref>。番禺出租车也是独立，且不得进入广州市区载客。番禺区的车辆不能购买广州的公路隧道[[年票|年票]]，因此过洛溪大桥前往广州市区需缴交人民币10元作为次票。從2018年開始，因為番禺莲花卡跟不上全國一卡通的標準，開始逐步暫停使用，從2018年1月1日開始，暫停番禺卡充值服務。從4月1日開始，停止番禺卡刷卡服務，並且逐步進行羊城通設備更新。<ref>http://www.sohu.com/a/210424856_430226 再见了，番禺公交卡！你是选择置换退款还是保留这份“记忆”呢？</ref>

===其他===
===外来人口===
受广州市开放且限制甚少的人口政策影响，番禺区截至2014年底为止，累计有150多万名外来人员(流动人口)居住在本区。<ref name="人口">[http://gd.qq.com/zt2015/edu30/index.htm 2014年12月31日腾讯网吴岳冬对话段落]</ref>主要来自于广东邻近之省份如[[湖南|湖南]]、[[广西|广西]]、[[四川|四川]]及[[江西|江西]]等地，绝大部份为[[农民工|农民工]]及其随迁人员落地番禺工作、学习。流动人口已占番禺总人口的约53％，比例逐渐赶超本地人口。

===财政===
番禺区之财政资金基本独立于广州市财政。

===医疗===
截至2013年，番禺共有“三甲”医院2间，“二甲”医院5间，“一甲”等级（社区农村）医院138间。[[何贤纪念医院|何贤纪念医院]]、[[番禺区中心医院|番禺区中心医院]]、[[番禺区中医院|番禺区中医院]]等区内优秀医疗资源几乎全部集中在市桥城区。城镇居民医疗保障制度以及新型[[农村合作医疗|农村合作医疗]]制度在全区均可通用。

=== 广播及电视 === 
{{main|番禺区电视台}}
番禺区有自己的合营电台和电视台（番禺广播电视中心，番禺区文广新局直接管辖）。番禺有线电视在其转播的高收视率境外电视台的广告时段及广东公共频道插播自行招揽的广告和自行制作的番禺新闻，而且番禺与广州市区的有线电视传媒没有互通。<ref>[http://www.pycatv.tv/Html/tv/business/20100415160643.html 电视频道节目对照表]，番禺有线电视</ref>。
