'''番-{}-荔枝'''（學名：{{lang|la|''Annona squamosa''}}），又稱'''佛頭果'''、'''釋-{}-迦'''、'''释迦果'''、'''亚大-{菓}-子'''（指其果实）、'''亚大树'''（指果树）、'''林檎'''（[[广东|广东]][[潮汕地区|潮汕地区]]）、'''唛螺陀'''（[[广西|广西]]）、'''洋波罗'''（[[广西|广西]][[龙州|龙州]]）、'''假波罗'''（[[广西|广西]][[凭祥|凭祥]]，因其外形与结构都酷似广西特产水果[[菠萝蜜|菠萝蜜]]）、'''番鬼(紅毛)荔枝'''（[[廣東|廣東]][[粵語|粵語]]區、[[香港|香港]]、[[澳門|澳門]]）、'''麻窝楂'''（[[云南省|云南]][[德宏傣族景颇族自治州|德宏]]）<ref>{{cite web |url=http://xms.yunnan.cn/html/2011-10/14/content_459624.htm |title=麻窝楂 |author=李美斌 |date=2011年10月14日 |website=云南网 |publisher=春城晚报·新芒市 |access-date=2017年7月13日 |quote=麻窝楂是芒市人对这种水果的称呼}}</ref>等，为[[番荔枝科|番荔枝科]][[番荔枝屬|番荔枝屬]]多年生半落葉性小喬木植物。多种植做经济作物，果肉白而甜。

== 生長分佈 ==
原產於熱帶[[美洲|美洲]]的釋迦，多栽種於熱帶地區，目前全世界[[臺灣|臺灣]]栽植最多。在臺灣，釋迦大多分佈於東部及南部，尤其以[[臺東縣|臺東]]所種植的釋迦最多，品質最為優良。據[[農業|農業]]年報統計，迄2015年止，全臺栽植面積為5,800公頃，其中臺東縣約為5,750公頃，占全臺栽植總面積之99%，故臺東縣為全臺番荔枝之主要盛產地。自2005年開始，中國地區開放台灣水果外銷，釋迦一直是台灣外銷中國地區的第一名，而且數量逐年上升，農民收益明顯增加，近5年來越來越多釋迦農戶擴大生產面積，到2017年底，台東縣的釋迦生產面積又增加了200公頃。

== 型態外觀 ==
[[File:Flora_Sinensis_-_Custard_Apple.JPG|right]]著作《Flora Sinensis》（译：中国植物志）笔下的 '''亚大菓子''']]
釋迦樹莖高3—5公尺，枝葉細密且低垂。單葉，互生無托葉，葉子正面深綠色反面為灰綠色。花單生，附生於[[葉腋|葉腋]]或枝端，4至9月開放，花瓣3枚呈橢圓長形，顏色淡綠花身下垂，散發濃厚的果香。果實為聚生果，果皮表面呈圓形、心臟形、橢圓形或圓錐形，未熟果表為深綠色，成熟果實呈現淺綠色或綠黃色，形狀神似[[荔枝|荔枝]]，故原名番-{}-荔枝，又因為果皮突起形似-{zh-hans:[[释迦牟尼佛|释迦牟尼佛]]; zh-hant:[[釋迦牟尼佛|釋迦牟尼佛]]}-頭部，因此別名「釋-{}-迦」。「釋-{}-迦」之名另有一說應來自[[印尼語|印尼語]]srikaya，[[荷蘭人|荷蘭人]]自[[印尼|印尼]]引進時[[臺灣人|臺灣人]]採用[[臺灣語|臺灣語]]音譯稱呼此果。釋迦種子為黑褐色橢圓形狀，光滑堅硬；果肉雪白口感綿密，香甜又帶點微酸。

也因為此種水果果皮突起形似-{zh-hans:[[释迦牟尼佛|释迦牟尼佛]]; zh-hant:[[釋迦牟尼佛|釋迦牟尼佛]]}-頭部，[[臺灣人|臺灣人]]通常在前往佛寺或宮廟拜拜時，不會選用此種水果為供品，認為會對神明及佛菩薩大不敬。

== 習性 ==
釋迦[[原產|原產]]於[[熱帶|熱帶]][[美洲|美洲]]，故喜愛溫暖[[乾燥|乾燥]]的環境。生长习性喜光，喜温暖气候，要求年平均温度在22度以上，不耐寒；适生于深厚肥沃排水良好的砂壤土。

== 歷史 ==
===臺灣===
根據[[臺灣|臺灣]]府誌記載是由[[荷蘭|荷蘭]]人引種入臺栽培，至今有400多年[[歷史|歷史]]，原產於熱帶[[美洲|美洲]]，又因為自「番邦」引入，故又稱為「番-{}-荔枝」。釋迦於[[臺東縣|臺東縣]]栽植風土適應性良好，[[果樹|果樹]]種類乃適時更新番荔枝成為主要經濟果樹。由於採集約經營，利用修剪及栽培管理技術之改進，矮化樹型及產期調節技術之開發與應用，使產期調節為一年兩收，供果時期延長，生產量增加，果形大，品質好，已成為臺東縣一項新興特產經濟果樹，藉以促進臺東縣農業持續發展，繁榮農村經濟，番荔枝產業可謂居功厥偉。由於臺東縣主要[[經濟|經濟]]果樹種類更新為番荔枝，形成[[產業|產業]]經營，逐漸促使產業發展。

== 杂交種 ==
[[File:sygar.jpg|180px]]
釋迦可以其它同属物种杂交栽培，如[[旺來釋迦|旺來釋迦]]（又名[[凤梨释迦|凤梨释迦]]，学名：{{lang|la|''Annona'' × ''atemoya''}}）就是本种与[[秘鲁番荔枝|秘鲁番荔枝]]（''{{lang|la|A. cherimola}}''）的雜交種。

== 種植 ==
釋迦適宜生長在砂質土、[[壤土|壤土]]或構造及排水良好的粘質壤土，因其根比一般果樹為淺，但土壤深度在１公尺左右最為理想。釋迦的理想[[酸鹼度|酸鹼度]]（pH）為6.0-6.5之間，在此範圍內，土壤中有效性營養[[元素|元素]]最容易被[[植物|植物]]利用。由於釋迦喜愛溫暖乾燥的環境，秋冬低溫容易下霜雪處就不適合栽培釋迦且不利果實的產期調節；且釋迦根淺且枝葉較脆弱，容易因強風吹動使植株搖動，引起根部的損傷，忌諱栽種於向風及海邊風大處。

== 採收 ==
釋迦硬熟期適時採收的[[標準|標準]]，須檢視果實表面的[[鱗|鱗]]目、色澤及鱗溝展開程度來判定成熟度。當鱗溝微裂，呈現奶黃色時，表示已可採收。如果實成熟度不足便採下，釋迦果會無法正常軟熟，而無商品及食用價值，如果實過熟，則容易快速軟熟，造成損失。

== 營養 ==
釋迦含有大量的[[蛋白質|蛋白質]]、[[碳水化合物|碳水化合物]]及[[維生素C|維生素C]]、[[鉀|鉀]]、[[鈣|鈣]]、[[鎂|鎂]]、[[磷|磷]]等，營養價值相當高。而且因[[熱量|熱量]]極高，可以有效的補充體力，並且具有養顏美容，強健[[骨骼|骨骼]]、增強免疫力等多樣功能，所以無論正餐或是水果都非常合適。但因其[[糖分|糖分]]極高，[[減肥|減肥]]和[[糖尿病|糖尿病]]患者，不宜多食。（營養分析於[[佛羅里達|佛羅里達]]、[[菲律賓|菲律賓]]和[[新南威爾斯大學|新南威爾斯大學]]。[http://www.hort.purdue.edu/newcrop/morton/atemoya.html#Food%20Uses 參見]）

== 保存 ==
採收下的釋迦果建議保存在涼快、遮蔭、通風處，理想的[[溫度|溫度]]為的是20℃，約5—7天後開始軟熟。長程運輸需以冷藏櫃保持12℃或以[[乙烯|乙烯]]吸收劑保鮮可維持10—15天（旺來釋迦Atemoya溫度可降至5℃以上）。釋迦果未軟熟前絕不可放入冰箱冷藏，否則將變成啞巴果變黑變硬。

當釋迦開始軟即可食用，或存放在[[冰箱|冰箱]][[冷藏|冷藏]]可延長保存時間，亦可冷凍成釋迦冰，冰涼享用另有一番風味。

== 藥用價值 ==
釋迦[[種子|種子]]、[[葉|葉]]片、[[樹皮|樹皮]]均含有[[生物鹼|生物鹼]]，可治療[[赤痢|赤痢]]。釋迦種子含有黃色[[乾性油|乾性油]]，可以用來殺[[蝨|蝨]]洗髮。葉片磨成粉末可治療[[癬疥|癬疥]]及做為[[拔膿劑|拔膿劑]]。
