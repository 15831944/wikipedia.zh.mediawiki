'''番杏科'''（學名：{{lang|la|''Aizoaceae''}}）是[[雙子葉植物綱|雙子葉植物綱]]，[[石竹目|石竹目]]的一個科，約有138屬，1882種。番杏科植物主要分佈在[[非洲|非洲]]南部，在[[澳洲|澳洲]]和中[[太平洋|太平洋]]地區也有少量分佈。

== 形態 ==
多為[[草本植物|草本植物]]，稀為[[木本植物|木本]]。[[莖|莖]]為直立莖或平臥莖。[[葉|葉]]為單葉，對生或互生，葉通常為肉質，葉緣全緣（或稀為齒狀）。[[花|花]]輻射對稱，多為[[兩性花|兩性花]]，少數為[[單性花|單性花]]，花腋生，單生花或聚繖花序。[[花萼|花萼]]有萼片5片（3-8片），萼片基部通常會相互結合。無真正的[[花瓣|花瓣]]，不過有些種類有由退化雄蕊瓣化而成的線形花瓣。[[果實|果實]]為[[蒴果|蒴果]]，每一室有一或多個[[種子|種子]]遇水會打開釋放出種子。

== 應用 ==
本科少數種類（特別是原產於[[南非|南非]]的[[莫邪菊|莫邪菊]]）被廣泛的[[引進物種|引入]]其他地區栽種，而後變成了[[入侵物種|入侵植物]]。在某些乾旱地區，[[番杏|番杏]]常被栽培來做為觀賞植物，也被用來代替[[菠菜|菠菜]]做成[[沙拉|沙拉]]食用。本科植物中有很多種類的葉片肥厚多肉，葉形奇特，常被栽培做為[[多肉植物|多肉植物]]觀賞。

== 分類 ==
有些植物學家主張將番杏屬和''Tribulocarpus''這二個屬由番杏科分出，另外成立一個科Tetragoniaceae。
