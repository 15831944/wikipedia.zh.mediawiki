'''番木瓜'''（[[学名|学名]]：''Carica papaya''），又称'''木瓜'''、'''石瓜'''、'''木冬瓜'''、'''蓬生果'''，是[[十字花目|十字花目]][[番木瓜科|番木瓜科]]水果。[[果實|果實]]長於樹上，外形与原产于中国的[[木瓜|木瓜]]类似，又是从中国之外传入，故名之番木瓜。番木瓜的[[乳汁|乳汁]]含[[木瓜蛋白酶|木瓜蛋白酶]]，是製作[[鬆肉粉|鬆肉粉]]的主要成份。

果肉質厚、軟、甜、有香味。果肉可鮮食，搾汁加[[牛奶|牛奶]]，名為「[[木瓜牛奶|木瓜牛奶]]」，是[[臺灣|臺灣]]有名飲品。

== 形态 ==
木瓜一般高2－3米的大型[[灌木|灌木]]，因為其不堅果所以亦有人認為是草本植物。其樹幹直立，底層分枝較少，主要在頂部生長。樹[[葉|葉]]大，[[葉柄|葉柄]]也長；[[花|花]]乳[[白色|白色]]，有雌花、雄花和兩性花。雄花的花型小，花瓣基部筒状；两性花按雌蕊、雄蕊发育的情况又分为雌型两性花、雄型两性花和长圆形两性花等3种，以长圆形两性花发育的果实最好。因花性的不同，植株可分单性株和两性株，雌株只开雌花，花性稳定，如有两性株的花粉授粉，结果力强，亦有天然单性结实现象。两性株开各种类型的花，往往受外界条件影响，花性不稳定，结出不同的果实，称异型果现象。 
      
[[果实|果实]]长圆形、卵形或洋梨形，单果重1－2.5千克，成熟时果皮由绿变黄，肉厚，肉质软滑，橙黄色或[[红色|红色]]。[[种子|种子]]未成熟时白色，成熟时[[黑色|黑色]]，外种皮有皱纹，种皮外有一层透明胶质的假种皮包围。

== 分佈 ==
木瓜原產[[熱帶|熱帶]]的[[美洲|美洲]]，在[[墨西哥|墨西哥]]已種植了幾個世紀。現時木瓜普遍在熱帶國家都有種植，例如[[巴西|巴西]]、[[印度|印度]]、[[菲律賓|菲律賓]]和南中國等地。而亞洲又以[[廣東|廣東]]、[[台灣|台灣]]栽培最多。

== 种类 ==
番木瓜有40多個品種。較重要的有：
* [[茎花番木瓜|茎花番木瓜]]（''Carica cauliflora'' Jacq.=''Vasconcellea cauliflora''）
* [[秘鲁番木瓜|秘鲁番木瓜]]（''Carica monoica'' Desf.=''Vasconcellea monoica''）
* [[山番木瓜|山番木瓜]]（''Carica candamarcensis'' Hook.f.=''Vasconcellea pubescens''）
* [[槲叶番木瓜|槲叶番木瓜]]（''Carica quercifolia'' Benth.et Hood.=''Vasconcellea quercifolia''）
* [[五棱番木瓜|五棱番木瓜]]（''Carica pentagona'' Heilb.=''Vasconcellea × heilbornii''）
*[[台農五號番木瓜|台農五號番木瓜]]（Tai-Non No.V）是中華民國的農業試驗所以雜交的方式育成，對蚜蟲傳播的｢木瓜輪點病PRSV｣具抗性。長果型，紅肉種。

== 營養 ==
果實含有豐富木瓜酶，維生素C、B及鈣、磷及礦物質，營養豐富，果實含大量豐富的胡蘿卜素、蛋白質、鈣鹽、蛋白酶、檸檬酶等。

== 用途 ==
果實具有防治高血壓、腎炎、便秘和助消化、治胃病，對人體有促進新節代謝和抗衰老的作用，還有美容護膚養顏的功效。木瓜酶可以軟化肉類，是鬆肉粉常見的成分。

==主要病害==
*'''木瓜輪點病'''
此病使瓜肉成輪點狀斑，有異味，由[[蚜蟲|蚜蟲]]傳佈。台灣由農試所雜交成功的'''｢[[台農五號|台農五號]]｣'''是專針對此病。
*'''真菌寄生'''
