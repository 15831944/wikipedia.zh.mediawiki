'''夸夸群'''是流行于[[中国大陆|中国大陆]]的一类以称赞他人为唯一聊天内容的[[即时聊天|即时聊天]]群组，主要是[[微信|微信]]群，也有[[腾讯QQ|腾讯QQ]]群。在夸夸群中，有群组成员会发言描述自己生活中遇到的事情，多数为负面的事情，如搭错[[公交车|公交车]]，然后其他群组成员便会从另外的角度对其进行夸奖，如搭错公交车可以看到不同的风景。

== 起源与发展 ==
在以微信群为形式的夸夸群出现之前，有[[豆瓣|豆瓣]]用户在2013年1月2日建立了互相表扬群组，性质与夸夸群类似，均是以表扬他人为主题。<ref>{{cite web |title=有人点赞、有人说“假”，“夸夸群”缘何流行？ |url=http://www.xinhuanet.com/politics/2019-03/26/c_1124284266.htm |website=新华网 |publisher=半月谈 |date=2019-03-26 |access-date=2019-04-10 |archive-url=https://web.archive.org/web/20190331233442/http://www.xinhuanet.com/politics/2019-03/26/c_1124284266.htm |archive-date=2019-03-31 |dead-url=no }}</ref>

夸夸群这一性质的微信群组最初是由[[浙江大学|浙江大学]]、[[清华大学|清华大学]]、[[西安交通大学|西安交通大学]]、[[北京大学|北京大学]]和[[上海复旦大学|上海复旦大学]]等中国大学的学生建立的，建立不久后便在校内流行，人数不断增加，并新创建了很多分群。后来通过[[新浪微博|新浪微博]]等[[社交软件|社交软件]]的分享，夸夸群这一形式的群组开始被其他人所知晓，其他大学也开始效仿创建夸夸群，后来夸夸群又流行到社会上，为更多的人所知晓。<ref>{{cite web |title=浙大夸夸群火了，网友纷纷求进群 |url=http://www.xinhuanet.com/local/2019-03/17/c_1124244039.htm |website=新华网 |publisher=钱江晚报 |date=2019-03-17}}</ref>

一开始的夸夸群是大学中的学生相互称赞的群组，所以是免费的。传播到社会上后，有商家以此为商机，创建付费的夸夸群，只有支付一定的费用才能被邀请入私密群组接受称赞，且这类付费夸夸群常有限定服务时间，时间一到便会被移除出群组。这种付费服务引起了一些人的反感，有消费者表示用钱买来的称赞不真实，无法还原被称赞时的情感满足。不久后，付费夸夸群热度渐降，购买付费夸夸群服务的顾客越来越少。<ref>{{cite web |author1=郑亚文 |title=十天，夸夸群从爆红到落寞 |url=https://wallstreetcn.com/articles/3498170 |website=华尔街见闻 |publisher=卖家 |date=2019-03-21}}</ref><ref>{{cite web |author1=王莉莉 |title=付费“夸夸群”：一只短命的猪 |url=http://hznews.hangzhou.com.cn/jingji/content/2019-03/26/content_7167319.htm |website=杭州网 |publisher=杭州日报 |date=2019-03-26 |access-date=2019-04-10 |archive-url=https://web.archive.org/web/20190326135236/http://hznews.hangzhou.com.cn/jingji/content/2019-03/26/content_7167319.htm |archive-date=2019-03-26 |dead-url=no }}</ref>

== 评价 ==
=== 正面评价 ===
有评价指出，在夸夸群中求夸的用户常常是生活中遇到不如意的事情，存在精神压力，而通过其他成员的称赞后，被称赞者能够从另外一个角度看问题，从不如意中看到如意的一面，从缺点中看到闪光点。这样有利于舒缓精神壓力，满足自尊心，消除负面情绪，同时也有利于营造和谐的社群氛围。<ref>{{cite web |title=夸夸群火爆 仅仅是因为求赞的刚需吗 |url=http://www.xinhuanet.com/tech/2019-03/26/c_1124281335.htm |website=新华网 |publisher=科技日报 |date=2019-03-26 |access-date=2019-04-10 |archive-url=https://web.archive.org/web/20190326125745/http://www.xinhuanet.com/tech/2019-03/26/c_1124281335.htm |archive-date=2019-03-26 |dead-url=no }}</ref>

=== 负面评价 ===
由于夸夸群是基于网络，群组中基本上均为陌生人，所以有人指出，夸夸群的称赞过于浮夸和虚伪，是[[拍马屁|拍马屁]]。有心理学专家指出，不应该过度依赖夸夸群来发泄负面情绪，而应该多在现实生活中调节自己的情绪，如多参加现实中的社交活动，多运动等，否则可能会因为网络世界与现实生活中的巨大差异而形成心理落差甚至迷失自我。<ref>{{cite web |title=“夸夸群”在重庆高校迅速蹿红 专家:理性对待 |url=http://m.xinhuanet.com/cq/2019-03/17/c_1124244095.htm |website=新华网 |publisher=重庆晨报 |date=2019-03-17}}</ref><ref>{{cite web |title=“夸夸群”可以有，但别依赖 |url=http://www.xinhuanet.com/comments/2019-03/14/c_1124233575.htm |website=新华网 |publisher=新京报 |date=2019-03-14 |access-date=2019-04-10 |archive-url=https://web.archive.org/web/20190328102655/http://www.xinhuanet.com/comments/2019-03/14/c_1124233575.htm |archive-date=2019-03-28 |dead-url=no }}</ref><ref>{{cite journal |author1=贾亮 |title=夸夸群可以减压不能依赖 |journal=浙江法制报 |date=2019-03-18 |page=008 |url=http://kns.cnki.net/KCMS/detail/detail.aspx?dbcode=CCND&dbname=CCNDTEMP&filename=ZJFZ201903180080&uid=WEEvREcwSlJHSldRa1Fhb09jT0pjeG92VkVTV253VGVPN2xzZ0VITXNPcz0=$9A4hF_YAuvQ5obgVAqNKPCYcEjKensW4IQMovwHtwkF4VYPoHbKxJw!!&v=MjkyMzJadTl2RWlqa1U3M0pKVndRUHlmTmRMRzRIOWpNckk1TlpPc0hEQk5LdWhkaG5qOThUbmpxcXhkRWVNT1VLcmlm}}</ref>

== 衍生 ==
