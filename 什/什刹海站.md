'''什刹海站'''位于[[北京市|北京市]][[西城区|西城区]]与[[东城区_(北京市)|东城区]]交界处，是[[北京地铁|北京地铁]][[北京地铁8号线|8号线]]的一座车站。车站以当地的地名“[[什刹海|什刹海]]”命名<ref>[http://www.takefoto.cn/viewnews-58197.html 北京地铁8号线站名溯源]. 北京晚报. 2013-12-29.</ref>。于2013年12月28日随8号线二期南延长段开通投入运营。

== 位置 ==
什刹海站地处什刹海东面，位于[[地安门外大街|地安门外大街]]（南北向）东侧地下，呈南北向布置<ref name="bjghw">{{cite web |url=http://www.bjghw.gov.cn/web/static/articles/catalog_14300/article_ff80808123fa497b0124528b720f000a/ff80808123fa497b0124528b720f000a.html |title=地铁8号线二期规划方案公告 |date=2009-10-14 |accessdate=2011-12-05 |publisher=[[北京市规划委员会|北京市规划委员会]] |archive-url=https://web.archive.org/web/20101227063621/http://www.bjghw.gov.cn/web/static/articles/catalog_14300/article_ff80808123fa497b0124528b720f000a/ff80808123fa497b0124528b720f000a.html |archive-date=2010-12-27 |dead-url=yes }}</ref>。

== 结构 ==
=== 车站楼层 ===
什刹海站为地下三层车站，车站主体空间两柱三跨结构，[[岛式站台|岛式站台]]设计，站台宽12米。车站工程总长146.6米。中心顶板覆土约2.5米。<ref>{{cite web |title = 北京地铁8号线二期10标什刹海站工程 |url = http://www.jlzulin.com/news/ShowNews.asp?id=158 |accessdate = 2012-02-24 |deadurl = yes |archiveurl = https://archive.is/20130427015131/http://www.jlzulin.com/news/ShowNews.asp?id=158 |archivedate = 2013-04-27 }}</ref>

== 装饰 ==
什刹海站的装修风格贴近北京传统古建风格，设计上以北京传统市井的灰色调和青砖的肌理为背景，衬托主体色调是黄色的装饰元素，用现代的材料和手法表达北京传统建筑构件和图案为主要装饰造型<ref>[http://www.chinaasc.org/news/zonghexiaoxi/20121224/71510.html 北京地铁8号线什刹海站] {{webarchive|url=https://web.archive.org/web/20140111084845/http://www.chinaasc.org/news/zonghexiaoxi/20121224/71510.html |date=2014-01-11 }}. 中国建筑学会. 2012-12-24.</ref>。
车站站厅墙面有由不锈钢镀钛金材料制成的名为“秋荷映像”公共艺术设计<ref>[http://www.takefoto.cn/viewnews-58056.html 北京地铁什刹海站 长廊宛若皇家园林]. 北京晚报. 2013-12-27.</ref>。
