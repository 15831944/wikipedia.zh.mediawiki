'''什切青電車'''（{{lang-pl|Tramwaje w Szczecinie}}）是[[波兰|波兰]][[什切青|什切青]]的[[有軌電車|有軌電車]]網絡，歷史可以追溯至1879年<ref name="księga"/>。2009年時，原本的公營操作單位民營化，成立{{link-pl|斯塞新電車公司|Tramwaje Szczecińskie}}（Tramwaje Szczecińskie）接手系統的營運<ref>{{cite web |url=http://ts.szczecin.pl/firma |title=O firmie |website=ts.szczecin.pl |access-date=2017-08-29}}</ref>。

這種有軌電車採用[[標準鐵軌|標準鐵軌]]，1900年開始通過600[[伏特|V]]的[[直流電|直流電]]運行。<ref name="księga">{{cite web |url=https://drive.google.com/file/d/0B6OmN31hboAIOHROdTYydExDcG8/view |title=Tramwaje w Szczecinie we trzynastu księgach prozą |website=drive.google.com |access-date=2017-08-29}}</ref>{{As of|2017}}，全市共有12條有軌電車路綫、203輛電車。<ref>{{cite web |url=http://www.mkm.szczecin.pl/tabor.php?action=view |title=Księga taborowa |website=mkm.szczecin.pl |access-date=2017-08-29}}</ref>路綫全長{{convert|112.8|km|mi}}。
