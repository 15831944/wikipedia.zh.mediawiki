'''什門十哲'''是佛教名相，「什」指的是[[中國佛教|中國佛教]][[三論宗|三論宗]]初祖，中國佛教四大譯經家之一{{notetag|中國佛教四大譯經家為：鳩摩羅什、[[真諦|真諦]]、[[玄奘|玄奘]]和[[不空|不空]]三藏。}}的「''' [[鳩摩羅什|鳩摩羅什]] '''」[[三藏|三藏]]法師；「什門十哲」是指鳩摩羅什門下的十位高足，十哲分別是：[[道生|道生]]、[[僧肇|僧肇]]、道融、[[僧叡|僧叡]]、曇影、慧嚴、慧觀、僧䂮、道常、道標<ref name="中華百科全書">{{Cite web |url =http://ap6.pccu.edu.tw/Encyclopedia/data.asp?id=9511&forepage=1|title = 什門十哲|author = 李志夫|publisher =中華百科全書 |language =zh|accessdate = 2018-07-28|quote =羅什有高足十人，稱為什門十哲，即：道生、僧肇、道融、僧叡、曇影、慧嚴、慧觀、僧䂮、道常、道標等。}}</ref>。十哲是羅什法師三千門徒中的精英，譯經事業的得力助手。其中道生、僧肇、道融、僧叡四人並稱為「什門四聖」（關中四聖）<ref>{{Cite web |url =http://buddhaspace.org/dict/fk/data/%25E8%25A1%259B%25E5%25A1%259E%25E7%25AF%2580.html|title = 什門十哲|author = 慈怡法師|publisher =佛光大辭典 |language =zh|accessdate = 2018-07-28 }}</ref>；僧叡與僧肇二人，並稱為「三論宗二祖」，在漢地[[龍樹菩薩|龍樹菩薩]][[中觀學派|中觀思想]]的弘揚發展起到重要的影響。

另外還有以道生、僧叡、道融、僧肇為「什門四傑」；曇影、慧觀、道恆、曇濟為「什門四英」的說法，合四傑、四英稱為「什門八俊」<ref name="中華百科全書"/>。

== 十哲生平簡述 ==
=== 道生 ===
[[File:_仙佛奇蹤_卷七_竺道生.png|right]]
竺道生（355年－434年），[[東晉|東晉]][[劉宋|劉宋]][[鉅鹿|鉅鹿]]人，俗姓魏。幼年隨[[竺法汰|竺法汰]]出家，數年後到盧山依止[[慧遠|慧遠]]大師靜修七年，天資穎悟，一覽能誦，後遊長安依什公門下受學<ref name="慧炬">{{cite journal |author1=蕭世昌 等 |title=鳩摩羅什大師的長安譯場概況 |journal=慧炬出版社 |date=2017-04-15 |issue=605 |page=24 |url=http://www.towisdom.org.tw/03-mag/605/tow605.pdf |accessdate=2018-07-30}}</ref>。在《大般涅盤經》還未傳入[[建康|建康]]時，據《六卷涅盤經》提出「頓悟成佛」、「善不受報」、「一闡提人階得成佛」等主張，大眾聽其主張，認為是異教邪說，便被擯出僧團。於是道生大師便隱居在[[蘇州|蘇州]][[虎丘|虎丘]]山，聚石為眾，為石頭說法，在講《涅槃經》到闡提有佛性時，眾石皆應聲點頭<ref>{{cite web |title=卍續藏《東林十八高賢傳》卷1 |url=http://tripitaka.cbeta.org/X78n1543_001 |publisher=CBETA 中華電子佛典協會 |accessdate=2018-07-31}}</ref>，這就是佛教中「生公說法，頑石點頭」的公案<ref>{{cite web |author1=文心 |title=【佛語典故】頑石點頭 |url=http://www.merit-times.com.tw/NewsPage.aspx?unid=231479 |publisher=人間福報 |accessdate=2018-07-31}}</ref>。後來漢地譯出《大般涅槃經》，世人讚譽道生先見，便稱他為「涅槃聖」，為關中四聖之首。著有《二諦論》、《佛性常有論》、《佛無淨土論》<ref>[http://dev.dila.edu.tw/authority/person/?fromInner=A001502 法鼓佛教學院－人名規範檢索－竺道生]</ref>及《法華義疏》、《維摩注解》等論疏。道生是佛學中國化的第一人，也是掙脫「格義」佛教的第一人<ref name="中華百科全書"/>。

=== 僧肇 ===
僧肇（384年－414年），[[東晉|東晉]][[長安|長安]]人，俗姓張。幼年時家貧，受人雇傭抄書為業，喜讀[[道家|道家]]老莊，在讀到舊《維摩經》時深有感悟而說：「知所歸矣。」<ref name="中華百科全書"/>於二十歲時出家為僧，善學《方等》<ref>{{cite wikisource|title=方等|author=丁福保|date=|wslanguage=zh|wslink=佛學大辭典/方等}}</ref>諸經，深通[[三藏|三藏]]，在知道羅什在[[姑臧|姑臧]]後，便前往依止，參與譯事，詳定經論，解悟深義，被喻為「解空第一」，為，關中四聖之一。[[弘始|弘始]]六年（404年），鳩摩羅什譯出《大品般若經》，僧肇作《般若無知論》呈羅什，受到羅什與慧遠的讚賞，另著有：《物不遷論》、《不真空論》、《涅盤無名論》、《注維摩詰經》等，肇師的弟子將其結集合成《肇論》<ref>{{cite wikisource|title=《肇論》|author=僧肇|date=|wslanguage=zh|wslink=肇論}}</ref>。晉安帝義熙十年（414年）卒於長安，時年三十一歲<ref>[http://dev.dila.edu.tw/authority/person/?fromInner=A001598 法鼓佛教學院－人名規範檢索－僧肇]</ref>。

=== 道融 ===
道融（生卒年不詳），又名僧融，關中四聖之一，[[南梁|南梁]][[汲郡|汲郡]][[林州市|林慮]]人，十二歲出家，住[[九江|九江]]東林寺<ref>[http://dev.dila.edu.tw/authority/person/?fromInner=A002842 法鼓佛教學院－人名規範檢索－僧融]</ref>，後參與羅什譯事，善講《中論》與《法華經》，著有法華、大品般若、金光明、十地、維摩詰等經的義疏。晚年於[[彭城|彭城]]講法宣教，隨行門徒有三百人，從學問道的有千餘人，世壽七十四歲<ref name="中華百科全書"/>。

=== 僧叡 ===
僧叡（373年－439年），劉宋[[魏郡|魏郡]][[長樂郡|長樂郡]]人，十八歲出家，拜僧賢為師，聽僧朗講《放光般若經》，二十二歲時即博通經論，聲明遠播，為關中四聖之一<ref>{{cite wikisource|title=四聖|author=丁福保|date=|wslanguage=zh|wslink=佛學大辭典/四聖}}</ref>。後加入什公譯經隊伍，在解釋法華經句：「天見人，人見之」為「人天交接，兩得相見」，使羅什驚服。著有《[[大智度論|大智度論]]》、《[[中論|中論]]》、《[[十二門論|十二門論]]》、《[[大品般若經|大品般若經]]》、《[[小品般若經|小品般若經]]》、《[[法華經|法華經]]》、《[[維摩詰所說經|維摩詰所說經]]》……等經論序<ref name="中華百科全書"/>。

元嘉十六年（439年），法師身無疾病，卻忽然召集僧眾向大家告別，隨後入房沐浴淨身後，燒香禮佛，面向西方合掌坐化圓寂，世壽六十七歲，當日寺裡見有五色香煙從僧叡的僧房飄出<ref>[http://dev.dila.edu.tw/authority/person/?fromInner=A001599 法鼓佛教學院－人名規範檢索－僧叡]</ref>。

=== 慧嚴 ===
慧嚴，（363年－443年），[[劉宋|劉宋]]豫州人，俗姓范，十六歲出家，於鳩摩羅什座下受學，深解經論，善於言說。羅什圓寂後，到建康東安寺，受到宋高祖的器重，宋文帝在位時，常向法師請示佛法，著有《無生滅論》及《老子略注》等書<ref>[http://dev.dila.edu.tw/authority/person/?fromInner=A001741 法鼓佛教學院－人名規範檢索－慧嚴]</ref>。《大[[涅槃經|涅槃經]]》剛傳進中國時，與慧觀、[[謝靈運|謝靈運]]合譯，將北本《涅盤經》四十卷改為三十六卷，稱為「南本涅槃經」<ref name="中華百科全書"/>。

=== 慧觀 ===
慧觀，（366年－436年），劉宋[[清河郡|清河郡]]人，俗姓崔，先依止[[廬山|廬山]][[東林寺|東林寺]][[慧遠|慧遠]]大師，後來鳩摩羅什來到漢地以後，前往親近依止修學。慧觀法師風神秀雅，思入玄微，其所著的《法華宗要序》受到羅什三藏的稱揚讚賞。因通達[[十誦律|十誦律]]宗旨，精研律學，為羅什門中的[[佛教戒律|戒律]]學者<ref name="中華百科全書"/>。晚年住持楊都道坊寺弘揚佛法，卒於[[宋文帝|宋文帝]][[元嘉_(劉宋)|元嘉十三年]]，世壽七十一歲<ref>[http://dev.dila.edu.tw/authority/person/?fromInner=A001746 法鼓佛教學院－人名規範檢索－慧觀]</ref>。

=== 曇影 ===
曇影，（生卒年不詳），本性謙和，因家貧而志於學，多聞而學識廣博。善講《[[妙法蓮華經|妙法蓮華經]]》及《光讚般若經》<ref>{{cite wikisource|title=《光讚般若波羅蜜經》|author=竺法護|date=|wslanguage=zh|wslink=佛學大辭典/光讚般若波羅蜜經}}</ref>，每次開講說法有上千僧俗聽講<ref>[http://dev.dila.edu.tw/authority/person/?fromInner=A002223 法鼓佛教學院－人名規範檢索－曇影]</ref>。到了關中以後，受到[[姚興|姚興]]的禮遇，敕令曇影進駐[[逍遙園|逍遙園]]協助羅什譯經，著有《法華義疏》及《[[中論|中論]]註》。晚年隱棲終老，世壽七十歲。

=== 僧䂮 ===
僧䂮{{notetag|讀音：略。}}，（344年－416年）<ref>[http://dev.dila.edu.tw/authority/person/?fromInner=A002222 法鼓佛教學院－人名規範檢索－僧䂮]</ref>姚秦[[尋陽縣|潯陽]]人，俗姓傅。少年出家，依長安大寺的弘覺為師，通達內外經論，因[[戒律|戒律]]清淨，行持嚴謹。姚興任命為[[僧正|僧正]]，使正綱紀，並參與羅什大師於長安大寺的佛經翻譯工作<ref> {{Cite web |url = http://tripitaka.cbeta.org/T55n2145_014 |title =《出三藏記集》第14卷第一鳩摩羅什傳|author =釋僧祐| publisher = CBETA 中華電子佛典協會|language = zh|accessdate = 2018-07-30}}</ref>，為譯場助譯<ref name="慧炬"/>。於姚興[[文桓帝|文桓帝]][[弘始|弘始]]十八年，[[圓寂|圓寂]]於長安大寺，世壽七十三歲。

=== 道常 ===
道常，（346年－417年）又稱為道恒，[[姚秦|姚秦]]時期[[藍田縣|藍田]]人<ref>[http://dev.dila.edu.tw/authority/person/?fromInner=A002224 法鼓佛教學院－人名規範檢索－道恒]</ref>。法師幼年即顯宿世善根，八、九歲時與玩伴捏沙成佛塔。法師幼年喪父，侍奉後母至孝，二十歲後母過世後便[[出家|出家]]為僧，深入佛理，學通內外典籍。當得知鳩摩羅什入關後，便前往依止修學，他的才思清敏受到鳩摩羅什大為嘉許。協助羅什的譯出許多經典，負責筆錄其口譯經文，寫序、校正及譯文潤飾等工作<ref name="慧炬"/>。著有《釋駁論》、《百行箴》。卒於東晉晉安帝義熙十三年（417年），世壽七十二歲<ref name="道恒、道標"> {{Cite web |url = http://tripitaka.cbeta.org/T50n2059_006 |title =《高僧傳》第6卷第十二|author =釋慧皎| publisher = CBETA 中華電子佛典協會|language = zh|accessdate = 2018-07-30}}</ref>。

=== 道標 ===
道標（生卒年不詳），他的生平僅在《高僧傳》的《釋道恒》傳中有略述：「秦姚興以恆、標二人神氣俊朗，有經國之量，乃敕[[尚書令|尚書令]]姚顯敦逼恆、標署道助振王業。並親書之曰：『卿等皎然之操，實在可嘉，但君臨四海，治急須才，…望體此恆不以守節為辭也。』恆、標不從；姚復致書什公。什亦婉辭：『…伏膺佛道…，道以啟悟童稚，助化功德也，願陸下放既往之恩，縱其微志也。』道標也在羅什譯場協助翻譯<ref name="慧炬"/>，著有《舍利弗毘曇序》、《弔王喬文》，其餘生平均無記載。<ref name="道恒、道標"/>。
