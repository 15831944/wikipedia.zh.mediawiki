'''什锦燒'''（{{jpn|j=お好み焼き|kn=おこのみやき|rm=Okonomiyaki}}）<ref>{{cite web |url=https://www.welcome2japan.hk/attractions/dining/b-rank/okonomiyaki.html |title=日本國家旅遊局 / 東瀛魅力 / 購物美食 / 便宜、簡便又好吃! 日本的大眾美食(B級美食) |author=日本國家旅遊局 |authorlink=國際觀光振興機構 |deadurl=no |archiveurl=https://web.archive.org/web/20180224012149/http://www.welcome2japan.hk/attractions/dining/b-rank/okonomiyaki.html |archivedate=2018-02-24 |accessdate=2018-11-09 }}</ref>，又稱為'''御好燒'''或'''隨意燒'''，是源自[[日本|日本]]的一種[[鐵板燒|鐵板燒]][[小吃|小吃]]。製作方法是將水加入[[小麥粉|小麥粉]]攪伴形成粉漿，再加上蔬菜、肉類、魚類和貝類等材料，於鐵板上烧成煎饼，最後加上調味料進食。不同地方的什锦燒在做法和材料上各具特色，其中最具代表性的有兩大派系，即關西風什锦燒（'''大阪燒'''）和廣島風什锦燒（'''廣島燒'''），實際上兩者發展的起源也不同。關西風什锦燒的基礎上，將高麗菜改為綠蔥，加上[[牛蹄筋|牛蹄筋]]，改為醬油味的蔥燒現在也大受歡迎。由於廣受歡迎，因而風靡全球，進一步成為各地熱門美食。

== 關西風什锦燒（大阪燒） ==
[[file:Modern_yaki,_rice_and_tsukemono_by_hirotomo_in_Osaka.jpg|thumb]]
===製法===
以下是以[[豬肉|豬肉]]為主要材料的雜菜煎餅製法，按嗜好可改為[[牛肉|牛肉]]、[[墨魚|墨魚]]、[[蝦仁|蝦仁]]、[[牡蠣|牡蠣]]等：
#[[水|水]]或[[柴魚片|柴魚]]高湯混合低筋[[麵粉|麵粉]]。（注意水份多寡，麵粉和水的比例約1.5:1。）
#準備切碎的[[高麗菜|高麗菜]]、加上[[雞蛋|雞蛋]]、山藥（[[日本薯蓣|山芋]]）泥、[[蝦皮|蝦皮]]、[[炸麵花|炸麵花]]等攪拌。
#將麵糊放在加熱了的鐵板上面，整理成直徑25cm，厚度2.5cm左右的圓形。
#將薄豬肉片放在煎餅上。
#如果一面焦結了，就反向另一面，但要小心不要弄散煎餅。
#如果兩面都燒成黃褐色，就降低火溫，塗上調味汁。
#撒上[[柴魚|柴魚]]片、[[青海苔|青海苔]]。
#縱橫切塊，完成。在完成的煎餅表面上按照愛好塗上蛋黃醬。
#以後在鐵板上以自己的愛好尺寸切吃。

== 廣島風什锦燒（廣島燒） ==
廣島燒（{{lang-ja|ひろしまやき}}），是什锦燒的一个种类，主要流行于[[日本|日本]][[廣島縣|广岛]]地区。但是廣島地方並沒有將其稱呼為「広島焼き」<ref name="hatena">[http://d.hatena.ne.jp/keyword/%B9%AD%C5%E7%BE%C6%A4%AD はてなキーワード - 広島焼き]</ref>。

===歷史===
[[明治|明治時代]]在[[東京|東京]]，有了叫做「-{冬冬}-燒」的[[醬油|醬油]]簡單調味煎餅，這可能和目前的[[文字燒|文字燒]]（{{lang|ja|もんじゃ焼き}}）也有著發源關係。1910年代，日本引進的[[辣醬油|辣醬油]]開始普及，也應用到-{冬冬}-燒，改稱為[[一錢洋食|一錢洋食]]。這是[[水|水]]混和[[麵粉|麵粉]]，放在平面鐵板上烤，加上[[蔥|蔥]]、[[豆芽|豆芽]]等少量[[蔬菜|蔬菜]]的，類似[[天津煎餅|天津煎餅]]的食品。因為價廉，一錢（0.01[[日圓|日圓]]）就可以吃到的洋味食品，故稱。

到二戰後，日本的食生活也開始平穩，在原來的一錢洋食上，增加[[洋白菜|洋白菜]]等蔬菜、加上[[肉|肉]]類、煎[[雞蛋|雞蛋]]等，或還加上[[炒麵|炒麵]]，按照客戶的「御好」（嗜好）調製的豪華食品也有了。這就是廣島風什锦燒。

===材料===
基本材料有[[麵粉|小麥粉]]（薄力粉）、[[水|水]]、[[豬肉|豬肉]]、[[鸡蛋|鸡蛋]]、[[荞麦面|蕎麥面]]、[[蔥|蔥]]、醬料等。

其他，依個人喜好添加，如：[[青海苔|青海苔]]、鰹魚粉、[[豆芽|豆芽]]等。<ref name="otafuku">[http://www.otafuku.co.jp/resipi/detail/433.html オタフク公式サイト - 広島お好み焼き]</ref>

===製法===
現在的典型廣島風什锦燒製法如下：
#首先將[[雞蛋|雞蛋]]、[[水|水]]、[[湯料|湯料]]和[[麵粉|麵粉]]混合成比較稀的麵漿，放在熱鐵板上拉成圓型，灑點[[柴魚|柴魚]]粉。
#迅速擺放大量[[洋白菜|洋白菜]]絲（高麗菜絲）和[[綠豆芽|綠豆芽]]，灑[[食盐|鹽]]和[[胡椒|胡椒]]，再擺蔥花、炸麵花、[[豬肉|豬肉]]、[[墨魚|墨魚]]等，從上面灑少量麵漿。
#同時，在旁邊，用鹽、胡椒和[[清酒|清酒]]等炒麵。
#等洋白菜等蔬菜開始軟化後，將煎餅和蔬菜等一起翻倒，壓平。
#在旁邊，煎一個半熟的雞蛋，擺在炒麵上，上面再擺煎餅。
#将整體翻倒，表面塗大量帶甜味的濃辣醬油，灑柴魚絲、[[青海苔|青海苔]]粉供食。
