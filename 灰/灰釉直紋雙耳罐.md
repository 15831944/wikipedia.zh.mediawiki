'''灰釉直紋雙耳罐'''，又稱'''青瓷直條紋雙繫罐'''，高22.1公分、腹寬約34.5公分、口徑17.5公分，是[[戰國|戰國]]年間的瓷器，現存於[[國立故宮博物院|國立故宮博物院]]。

== 歷史 ==
<!--可以改寫更多 http://antiquities.npm.gov.tw/Utensils_Page.aspx?ItemId=659590-->[[File:戰國灰釉直紋雙耳罐的直條紋.png|thumb]][[春秋時期|春秋]]、[[戰國|戰國]]時期，江蘇、浙江一帶盛行燒造高溫灰釉器。這件灰釉瓷直紋雙耳罐，便是在[[戰國|戰國]]年間，西元前5世紀－西元前3世紀，以瓷土為胎，施以透明鈣灰釉，於1200度的高溫燒製而成的作品<ref>{{cite web |url = http://antiquities.npm.gov.tw/Utensils_Page.aspx?ItemId=659590 | title = 戰國 灰釉瓷直紋雙耳罐 |publisher = 國立故宮博物院器物 |website = 典藏資料檢索系統 |accessdate = 2017-11-10 |archiveurl = https://web.archive.org/web/20171110082109/http://antiquities.npm.gov.tw/Utensils_Page.aspx?ItemId=659590 |archivedate = 2017-11-10 |deadurl = no}}</ref>。

== 外觀 ==

本件灰釉直紋雙耳罐呈直口、斜肩、圓鼓腹部下收、平底。肩上有二對稱環形繫耳，耳內銜環。以瓷土為胎呈黃褐色，作品表面施以透明鈣灰釉至腹部中間，釉層較薄，並有剝釉現象。作品的肩部和上腹部的直紋瓦楞線[[刻紋|刻紋]]，刻紋成平行排列。此作品造型樸素大方，原型來自於青銅器，類似的作品在浙江德清亭子橋窯址亦可見，江蘇無錫鴻山越國貴族墓葬中也有類似作品出土，然而在為數眾多的作品中，保存完好者則不多見<ref>[http://auction.artxun.com/paimai-3010-15049690.shtml 博宝艺术品拍卖网]</ref>。
