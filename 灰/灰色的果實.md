== 概要 ==
本作是作為FrontWing創立10周年的紀念作品。標題的「Grisaia」是採自法文美術用語的「Grisaille」的變體，意思為「灰色」。

角色設計是由以動畫「[[化物語|化物語]]」而廣為人知的動畫家[[渡邊明夫|渡邊明夫]]和以FrontWing過去的作品「星之歌」聞名的{{lang|en|Fumio}}來擔任，其中フミオ所設計的人物較多，而渡邊明夫還負責擔任開頭動畫的監製。

== 故事 ==
故事的舞台是個名叫三嶋崎的濱海城鎮。在2011年5月，三嶋崎這裡有間於一年前設立的「私立美濱學園」的住宿制學校，於同月的24日，一名少年風見雄二轉入這間學校促成了故事的開端。

在外界鮮有人知、以高聳圍牆隔絕與外界的聯繫的美濱學園，本來是只有五名學生，而且全為女性的新學校，在轉學生也是唯一的男學生雄二轉入後，便在這些有著豐富個性的女學生的包圍下，過著平和又安樂的日子。

可是，在這些平淡的日子背後，美濱學園的五名女學生其實都有著無法向他人述說的晦暗過去，在雄二於學校的日子中，也逐漸接近這些女孩所抱持著的秘密。

== 用語 ==

; {{nihongo|三嶋崎|三嶋崎（みしまざき）}}
: 本作品舞台的城鎮。位於[[湘南_(日本)|湘南]]近郊的半島。從都心搭車需要兩個小時。車站前有著商店街，天音常到那裡購物。
; {{nihongo|私立美濱學園|私立美浜学園（しりつみはまがくえん）}}
: 雄二轉學進入的學校，建於三嶋崎的埋立地上，由學校法人榊學園經營的全宿舍制學園，同時也是理事長榊道昭一年前為了女兒所創立的地方。
: 雖然設備一流，但學生人數卻只有五名女學生，雄二是經由特例入學的唯一男性。校舍周圍被高聳的圍牆圍住，由美子稱之為「鳥籠」。
; {{nihongo|東濱急行電鐵集團|東浜急行電鉄グループ（とうひんきゅうこうでんてつグループ）}}
: 以關東地方私鐵網「東濱急行電鐵」為中心的巨大企業，對三嶋崎的經濟發展有極大貢獻，目前的總帥是由美子父親榊道昭。
: 在政財界有巨大的影響力，甚至跟[[國土交通省|國土交通省]]有密切關係。
; {{nihongo|「會社」|「会社」}}
: 雄二所屬的裡世界機關，本社位於美國。在日本藉由「[[美國駐日大使館|赤坂]]」介入，以「[[防衛省|市谷]]」所屬機關的名義進行活動。組織正式名稱不明，略稱為「CIRS」。
:; {{nihongo|CIRS|Central Intelligence and Research Second}}
:: 全名「日美聯合反恐組織 中央調查部諜報二科分室」，是日下部麻子、JB、雄二工作的單位。
; {{nihongo|9029|9029号（ナインオーツーナイン）}}
: 雄二在裡世界的代號。此為「會社」中最厲害的王牌工作人員代代相繼的代號。因為12年前、前任的9029（麻子）將古巴的販毒組織以一人之力一舉殲滅而成了讓裡世界成員聞風喪膽的稱號。
; {{nihongo|私立瀧園學園|私立滝園学園（しりつたきぞのがくえん）}}
: 6年前天音與一姬就讀的女子學校，學生多是家境富裕的大小姐。有設立宿舍，但也可以從家裡上學。
; {{nihongo|瀧園學園巴士翻覆事故|滝園学園マイクロバス転落事故}}
: 6年前的2005年8月14日，瀧園學園籃球社成員在夏季合宿歸途時的巴士從懸崖摔落樹海，共十四人（學生十三人、教師一人）下落不明。
: 因為樹海-{干}-擾電波，外加巴士走的並非原來的路線，導致搜查非常困難。兩週後更認為遇難者已難以存活而停止搜查。
: 事故發生的十八天後，在身體衰弱的天音自力下山後情況改變，得到天音提供位置的搜救隊找到了其他的遇難者。
: 但是其餘遇難者的屍體呈現支離破碎的狀態，此一慘狀也成了媒體報導的話題。
