'''灰原哀'''（{{jpn|j=灰原 哀|hg=はいばら あい|rm=Haibara Ai}}），即[[日本漫画|日本漫画]]和[[日本动画|动画]]作品《[[名侦探柯南|名侦探柯南]]》虛構角色'''宮野志保'''（{{jpn|j=宮野 志保|hg=みやの しほ|rm=Miyano Shiho}}）吃下毒藥身體縮小後所使用的化名。<ref>{{Cite web|url=http://ent.people.com.cn/GB/8222/86596/180338/180341/10816922.html|title=《名侦探柯南》经典人物：灰原哀·宫野志保|work=人民网娱乐频道|date=2010年1月21日|deadurl=no|archiveurl=https://web.archive.org/web/20130717114706/http://ent.people.com.cn/GB/8222/86596/180338/180341/10816922.html|archivedate=2013年7月17日}}</ref>

== 人物 ==
本名'''宫野志保'''，是日英[[混血儿|混血儿]]與「[[名偵探柯南#藥品|APTX-4869]]」研發者<ref name="multiple" />、；父亲是人称“被科学界放逐的疯狂科学家”的[[宫野厚司|宫野厚司]]，母亲是人称“堕落地狱的天使”的英國人[[宫野艾蓮娜|宫野艾蓮娜]]；两人都是[[黑暗组织|黑暗组织]]的[[科学家|科学家]]，在她出生后不久丧生<ref name=" multiple">《名侦探柯南》漫画第18期</ref>。她在组织命令下赴美留学<ref>《名侦探柯南》漫画第19期</ref>，也曾是组织的科学家，代号「'''Sherry'''」（日語：シェリー/台譯「雪莉」，取名自[[雪利酒|西班牙同名酒品]]）。在[[工藤新一|工藤新一]]失蹤後，将他的服用药物结果从「不詳」改为「死亡」<ref name="multiple" />，让黑暗组织停止寻找新一。之後她唯一的亲人——姐姐'''[[宫野明美|宫野明美]]'''（{{jpn|j=宮野明美|hg=みやの あけみ|rm=Miyano Akemi}}，化名「廣田雅美」）在一次事件中慘遭组织幹部[[黑暗组织#干部|琴酒]]杀害<ref>《名侦探柯南》漫画第2期。</ref>，於是停止药物研究以示抗議，卻也因此被監禁在組織的[[毒氣室|毒氣室]]裡。在等候審判時她吞下預藏的「APTX-4869」企圖自盡，身體卻意外的缩小而得以從垃圾口逃脱<ref name=" multiple" />。

逃出组织的她為尋找唯一可以依靠的人——「[[工藤新一|工藤新一]]」而來到他家，卻因體力不支倒在門前；而後被[[阿笠博士|阿笠博士]]發現並予以收留寄住在他家<ref>動畫第129集</ref>（劇場版開頭介紹中還提及那天是雨天）。[[阿笠博士|阿笠博士]]採用[[P·D·詹姆斯|P·D·詹姆斯]]推理小说中女侦探「寇蒂莉亚·葛蕾（Cordelia '''Gray'''）」（日語：コーデリア・'''グレイ'''，グレイ=灰）和{{link-en|萨拉·帕瑞特斯基|Sara Paretsky|萨拉·帕瑞特斯基}}作品主人翁女侦探「V·'''I'''·华沙斯基」（英語：V. I. Warshawski/日語：V・'''I'''・ウォーショースキー，I=あい（愛）），为她取化名“灰原愛”；但她坚持用同音的「哀」取代「愛」。日後也就读于[[江户川柯南|江户川柯南]]所在的帝丹小学一年B班<ref name="multiple" />及成为[[少年侦探团|少年侦探团]]一员。

本名宫野志保則是參考[[青山剛昌|青山剛昌]]一位網友的名字<ref>作者青山刚昌曾说「志保」是他认识网友的名字</ref>。

青山后来表示，“哀”或“爱”其實是源自於[[福尔摩斯探案|福尔摩斯探案]]系列中的歌剧女演员「[[艾琳·艾德勒|艾琳·艾德勒]]」<ref>{{cite web |title=黒幕、灰原哀のヒミツ… 『名探偵コナン』青山剛昌、佐藤健につられて衝撃ネタバレ！ |url=http://ddnavi.com/news/190845/ |accessdate=2014-04-18 |language=ja |deadurl=no |archiveurl=https://web.archive.org/web/20140416030241/http://ddnavi.com/news/190845/ |archivedate=2014-04-16 }}</ref><ref>{{cite book |title=名侦探柯南SDB30+ |publisher=长春出版社 |date=2008年1月 |ISBN=9789861561448 |quote=Q40：灰原哀的名字是结合了两个女侦探的名字，但是不是也参考了福尔摩斯敬佩的女性艾琳的名字呢？…… <br> A：……原来如此！（笑）}}</ref>。

== 形象 ==
個性沉著穩重，思想成熟而縝密。有著高尚的品味，常看些潮流雜誌和個人護理之類的書籍。在[[少年偵探團|少年偵探團]]中擔任緩衝角色，擁有着不遜於柯南頭腦和偵探才能，和柯南為搭檔關係，與[[服部平次|服部平次]]同為柯南查案時的夥伴，會適時輔佐他。

在博士家中是類似管家的存在，平時會注意他的飲食情況，她與博士同樣將對方視為親人般看待，二人關係有點像爺爺和孫女，也會和博士為柯南找尋案件線索及有時會與博士合作一同研究和發明新產品及陪同他參加發布會。多次在危機中保持冷靜，使得柯南一行人都能化險為夷。

== 情感 ==
對[[黑暗組織|黑暗組織]]非常恐懼，能夠直覺般地察覺他們的氣息，並引發如同恐慌症一樣的反應。

因父母突然離奇死亡，後來姐姐也慘遭組織毒手，令她欠缺家庭溫暖，導致待人較為冷漠。而在逃離組織後還有自暴自棄的傾向，因為深知組織恐怖之處的她無法擺脫被他們追殺的恐懼，認為現今的安穩總有一天會破滅，甚至想要尋短<ref>在神秘乘客篇中灰原打算放棄自己，柯南把她從正要爆炸的公車裡救出來（動畫第230~231集；漫畫第287~289話）劇場版M5<通往天國的倒數計時>中柯南與少年偵探團打算藉炸彈的衝擊波飛離大樓，灰原打算犧牲自己為大家在炸彈旁讀秒倒數，後被元太比喻成碗裡剩下的米粒抱回車上一同逃離</ref>；這些負面情感在柯南等人的陪伴下逐漸釋懷，變得較為開朗。

最初對柯南抱持著關注，因為他是第一個服下「APTX-4869」不死而縮小的人類，是個有趣的觀察對象。而在逃離組織後則與他建立起共享秘密的合作關係。在經過許多次共患難的經歷後，她與柯南已是互相信賴的夥伴，而她自身也漸漸對柯南抱有好感<ref>在東都顯像所的秘密篇中（動畫第335～336集；漫畫第417～419話）被工藤有希子發現灰原喜歡柯南</ref>。

其後亦因得到[[少年偵探團|少年偵探團]]眾人的關心及真情對待，令她原本悲觀的性格亦漸漸開始轉變。

曾有一段時間排斥與毛利蘭接觸，之後決定敞開心扉，主動上前和小蘭打招呼<ref>漫畫第313話，動畫第247集</ref>，日後也得到了[[圓谷光彥|圓谷光彥]]的「暗戀」。雖然仍喜歡柯南，但未對他表態或展開追求，也不強求發展為戀人關係<ref>漫畫第423話</ref>。

說到牽涉到她親人的話題時她會容易變得敏感和情緒化；討厭[[靜電|靜電]]。

喜歡的偶像是BIG大阪隊的足球選手比護隆佑。

非常喜歡小動物，而且她對小動物非常有愛心和溫柔，更會令她較容易卸下冷酷的一面而露出笑容。

|  APTX-4869
||外觀是印有「APTX-4869」的白、紅膠囊，為宮野志保（灰原哀）效力於黑暗組織時所研發的藥品（目前已終止研發）。藥理作用能引起細胞爆發性壞死，同時強化[[端粒酶|端粒酶]]（{{lang|ja|テロメアーゼ}}，{{lang|en|telomerase}}）活性，使[[細胞分裂|細胞分裂]]潛力增強、也能讓被害者屍體無法檢驗出毒素；但未成年人服用可能會導致身體縮小（在細胞自我破壞程序的偶發作用之下，使得神經組織外的骨骼、肌肉、內臟、毛髮等細胞退化到成長過程的幼兒時期），是一種神祕毒藥<ref>動畫第129集</ref>。

其中「APTX」為「Apoptoxin」的縮寫，是「Apoptosis」（程序性細胞死亡）和「toxin」（毒素）所組合成的混成詞；「4869」則是命名自「夏洛克（Sherlock）·福爾摩斯」的名字<ref>「Sherlock」（シャーロック）與日文中「4869」的其中一種念法「しやろく」相似。</ref>。

[[黑暗組織|黑暗組織]]曾使用[[福爾摩斯|福爾摩斯]]原本的名字「Shelling Ford」作為藥物名稱，因此「APTX-4869」亦有「殘廢的名偵探」之稱<ref>動畫第345集</ref>；根據宮野母親留給志保18歲生日的錄音帶，宮野夫婦稱呼正在研發的藥物為「銀色子彈」（Silver Bullet）<ref>漫畫第821話，動畫第702集</ref>，推測為「APTX-4869」的前身<ref>漫畫第948話</ref>。

== 人气投票排名 ==
2005年在[[周刊少年Sunday|周刊少年Sunday]]举行的“柯南”官方人气投票的最佳人物类别中排第4名，在2008年[[日本雅虎|日本雅虎]]“名侦探柯南特辑”的角色人气投票中排第4名，2011年TV版和剧场版年度15周年纪念活动人物投票中位居第4名<ref>{{cite web|url=http://www.ebookjapan.jp/ebj/special/conan/vote_kekka2.asp|title=『名探偵コナン』 人気投票結|date=12 May 2011|accessdate=11 January 2016}}</ref><ref>{{cite web|url=http://www.conan-movie.jp/camp_chara/camp15th_res.html|title=名探偵コナン「キャラクター人気投票」|date=2011|accessdate=12 August 2011|deadurl=yes|archiveurl=https://web.archive.org/web/20111028004825/http://www.conan-movie.jp/camp_chara/camp15th_res.html|archivedate=28 October 2011}}</ref>，2012年“周刊少年Sunday”800话纪念活动的角色人气投票中获得第3名<ref>{{cite web|url=http://forum.dctp.ws/index.php?topic=8024.msg651566#msg651566|title=Shonen magazine detective conan official poll results published [magazine pic]|date=28 February 2012|accessdate=29 February 2012|deadurl=yes|archiveurl=https://web.archive.org/web/20120308232438/http://forum.dctp.ws/index.php?topic=8024.msg651566#msg651566|archivedate=8 March 2012}}</ref>，在日本权威书评杂志《达芬奇》2014年5月刊中最受欢迎的角色排名中排名第3。灰原哀通常在柯南的女性角色中排名最高。2014年12月放映的“江户川柯南失踪事件 - 史上最糟糕的两天”特别篇的纪念活动的角色人气投票中排所有角色第1名。
