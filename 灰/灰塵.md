'''灰塵'''是[[大氣|大氣]]中一種[[固態|固態]]懸浮物，常態存在於[[空氣|空氣]]之中，易伴隨[[風|風]]的吹拂而四散至各處，包括家中的每一個角落。其粒徑大小有所差異，不一定能以肉眼見到。

== 定義 ==
灰塵颗粒的直徑通常小於500[[微米|微米]]。小於10微米的[[懸浮粒子|懸浮粒子]]（PM<sub>10</sub>），即被認定有害於人體；小於2.5微米的[[细颗粒物|细颗粒物]]（PM<sub>2.5</sub>），更可穿透[[肺泡|肺泡]]直達[[血液|血液]]。

== 影響 ==
[[土壤|土壤]]中的灰塵和[[沙粒|沙粒]]常伴隨[[季風|季風]]的吹拂，產生[[沙塵暴|沙塵暴]]。現今工業化地區嚴重的[[空氣污染|空氣污染]]問題，[[浮懸物|浮懸物]]亦成為測定標準之一，例如[[台灣|台灣]][[行政院環境保護署|環保署]]即定義粒徑10[[微米|微米]]以下之粒子為'''懸浮微粒'''，亦稱'''浮游塵'''，為空氣污染指數'''PSI指數'''測定標準之第一項。<ref>{{cite web |url=http://taqm.epa.gov.tw/emc/html/b0202.htm |title=存档副本 |accessdate=2007-10-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20071019011703/http://taqm.epa.gov.tw/emc/html/b0202.htm |archivedate=2007-10-19 }}</ref>懸浮微粒具有細小的特點，故容易經由[[呼吸|呼吸]]吸入肺部深處，造成呼吸道傷害。在礦坑中工作的礦工，也因環境過多的[[煤灰|煤灰]]而成為[[肺塵病|肺塵病]]的高危險族群。

另外，灰塵易形成[[塵蟎|塵蟎]]寄生之處，而除了灰塵本身導致呼吸道問題之外，[[蟎|蟎]]亦是許多呼吸道疾病的[[過敏原|過敏原]]，例如[[哮喘病|哮喘病]]。

前央視記者柴靜的紀錄片「穹頂之下」提到，PM2.5會導致哮喘、肺活量下降。它還會夾帶重金屬毒素，讓人體免疫系統中的巨噬細胞難以消化，導致巨噬細胞破裂而死，影響免疫功能。
根據台灣環保署的資料，許多流行病理學研究，已證實PM2.5對健康的危害，包括早逝、支氣管炎、氣喘、心血管疾病、肺癌等。

== 太空中的塵埃 ==
[[宇宙塵|宇宙塵]]在太空中非常普遍，其中所包含的氣體和塵埃是構成[[行星系統|行星系統]]前身的主要成分。[[黃道光|黃道光]]，是在黑暗的夜晚看見的微光，就是由圍繞著[[太陽|太陽]]的塵埃微粒所反射的陽一光產生的。[[彗星|彗星]]尾巴的微光也是從彗星釋出的電離氣體和塵埃所發射的光。塵埃也可以覆蓋在行星固體的表面，像是[[火星|火星]]上所發生的[[塵暴|塵暴]]，就幾乎可以覆蓋整個的行星表面。[[星際塵埃|星際塵埃]]被發現存在於恆星之間，濃度高的可能形成[[瀰漫星雲|瀰漫星雲]]和[[反射星雲|反射星雲]]。

從外太空帶回來的塵埃樣品或許能提供早期[[太陽系|太陽系]]的訊息。有些[[太空船|太空船]]已經發射前往蒐集塵埃和其他物質的樣本，像是[[星塵號|星塵號]]在2004年穿越過[[維爾特二號|維爾特二號]][[彗星|彗星]]的尾巴，並且在2006年一月將保存在膠囊內的一些樣本帶回地球。[[日本|日本]]的[[隼鳥號|隼鳥號]]太空船過去亦在執行收集[[小行星25143|小行星]]塵土的樣本帶回地球的任務。
