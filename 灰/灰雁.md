'''灰雁'''（學名：{{lang|en|''Anser anser''}}），又名'''大雁、沙鹅、灰腰雁、红嘴雁、沙雁、黄嘴灰雁'''，为[[雁属|雁属]][[模式种|模式种]]。

== 生态环境 ==
灰雁栖息于湖泊、河滩水域地带。他们经常选择水生植物密布的环境藏身，有时也在开阔的湖面游荡。憩息时常有一些灰雁放哨警戒。早晨、夜间或黄昏出来觅食。

== 分布地域 ==
本物种是[[古北区|古北区]]代表鸟种之一，繁殖于[[西伯利亚|西伯利亚]]南部、[[欧洲|欧洲]]北部、东部、[[冰岛|冰岛]]以及[[中国|中国]][[新疆|新疆]]西部北部、[[青海|青海]][[柴达木盆地|柴达木盆地]][[青海湖|青海湖]]、[[中国东北|东北]]的[[内蒙古|内蒙古]]、[[黑龙江|黑龙江]]等省份；越冬于[[欧洲|欧洲]]南部、[[地中海|地中海]]沿岸、[[伊拉克|伊拉克]]、[[印度|印度]]西北部、中国南方的[[江苏|江苏]]、[[湖南|湖南]]、[[广东|广东]]、[[福建|福建]]等地也有本物种越冬。

== 特征 ==
[[File:Anser_anser_at_Beijing.jpg|thumb]]
体重1900-2350克，体长770-795毫米。雌雄两性全年体色概为灰褐色，嘴基周围有狭长的白纹。下背及腰部为较深灰鼠色、颈、胸和腹部淡灰色，腹部有黑色横纹眼棕色，嘴橙黄或略带红色。在野外观察中，本物种最大的特征就是没有特征，它不具有[[豆雁|豆雁]]喙端醒目的黄色、[[鸿雁|鸿雁]]颜色对比鲜明的颈部，也不具有[[白额雁|白额雁]]和[[小白额雁|小白额雁]]引人瞩目的白色额头，因此如果在野外看到一只灰褐色的大型雁类又没有观察到上述特征，且可以肯定地观察到粉红色地喙和脚，那么就可以判断其为灰雁。
[[File:Anser_anser_MHNT.ZOO.2010.11.14.3.jpg|thumb]]

== 食物 ==
灰雁是植食性鸟类，以野草和种子为主要食物，偶尔也会取食小虾、田螺和一些[[鞘翅目|鞘翅目]][[昆虫|昆虫]]。

== 繁殖与保护 ==
灰雁的营巢地常常在水边草丛间，营巢时亲鸟在地上以[[苔藓|苔藓]]、杂草或羽毛筑巢，有时在沼泽地区建筑群巢。每窝产卵4-6枚，卵呈乳白色。

本物种未列入保护名单，但由于过度捕猎和栖息地的破坏（如在[[鄱阳湖|鄱阳湖]]、[[洞庭湖|洞庭湖]]等越冬地，受到非法捕猎和湖区破坏的威胁。本物种的数量呈下降趋势。

== 灰雁与鹅 ==
普遍认为灰雁是[[鹅|欧洲家鹅]]的祖先。
