== 種類 ==
市售蜂胶根据不同的用途，主要有以下几个类型：
# 毛胶：直接从蜂箱中取得，未经过加工，含有蜜蜂肢体的木屑，泥沙、麻布纤维等杂质，其形态为不透明的团块状或碎屑状，适于外用或用于鸡猪牛羊饲料强化剂。　 
# 原胶：毛胶经工厂化加工，杂质与[[重金属|重金属]]含量符合国际标准。形态为不规则条块状，适合于做食品、药品、化妆品原料。 　 
# 蜂胶标准溶液：是一种蜂胶的[[乙醇|乙醇]]溶液，适于外用或用于食品、药品添加剂及食品防腐剂。 
# 蜂胶水溶液：适于稀释后口服，或用于食品、药品、添加剂。 　　 
# 蜂胶油：蜂胶油溶液，适用于美容化妆品，或日常皮肤保养。 　　 
# 蜂胶乾膏：蜂胶提纯物，胶片状，适用于食品、药品、化妆品、并用于工业、农业、畜牧业、水产养殖业。 　　 
# 蜂胶制品：各种含蜂胶的食品、药品、化妆品。
# 蜜溶蜂胶：适用于稀释后口服，或外用。是由蜂蜜溶蜂胶液，不添加化学添加剂。
