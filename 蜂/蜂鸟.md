'''蜂鸟'''（[[學名|學名]]：Trochilidae）是屬於[[雨燕目|雨燕目]]的蜂鸟科，体型小，能够以快速拍打翅膀（每秒70次，取决於鸟的大小）的方式而悬停在空中，也是唯一可以向后飞的鸟。蜂鸟的飞行本领高超，也被人们称为「神鸟」、「彗星」、「森林女神」和「花冠」等<ref>{{Cite book|title = [http://www.chaoxing.com/search?sw=%E5%90%84%E5%9B%BD%E5%9B%BD%E9%B8%9F%E4%B8%8E%E4%B8%AD%E5%9B%BD%E5%90%8D%E9%B8%9F&x=0_7209 各国国鸟与中国名鸟]|last = 中国野生动物保护协会|first = |publisher = 中国林业出版社|year = 1998|isbn = 7-5038-2001-2|location = 北京|pages = 16}}</ref>。

[[吸蜜蜂鸟|吸蜜蜂鸟]]（學名：''Mellisuga helenae''）僅重1.8[[公克|公克]]，是世界上最小的鸟。还有一种典型的[[北美|北美]]蜂鸟[[红喉北蜂鸟|红喉北蜂鸟]]（''{{lang|en|Archilochus colubris}}''）重量约三公克左右，身长7.6[[公分|公分]]。蜂鸟中体型最大的是[[巨蜂鳥|巨蜂鸟]]（學名：''Patagona gigas''），其体重也僅20克。

== 蜂鸟的起源 ==
蜂鸟的体型小，骨架不易保存成为[[化石|化石]]，因此其演化史至今仍未解。现今的蜂鸟大多生活在[[中南美洲|中南美洲]]，在[[南美洲|南美洲]]曾发现100万年前的蜂鸟化石，因此[[动物学家|动物学家]]认为蜂鸟是源自[[更新世|更新世]]；然而在[[歐洲|歐洲]]，动物学家发现了[[歐洲蜂鳥|歐洲蜂鳥]]的化石，是迄今已知最早的蜂鳥化石，距今有3000多万年的历史，如此可知蜂鸟的祖先远在[[渐新世|渐新世]]的时候便出现。

== 生态环境 ==
蜂鸟居住的范围十分广阔，範圍从高达4,000米的[[安第斯山|安第斯山]]直到[[赤道|赤道]]區域的[[亚马逊河|亚马逊河]][[热带雨林|热带雨林]]，有的蜂鸟生活在干旱的[[灌木|灌木]][[丛林|丛林]]，也有蜂鸟生活在潮湿的[[沼泽|沼泽]]地区。

== 分布地域 ==
蜂鸟仅分布于[[美洲|美洲]]，从南[[加拿大|加拿大]]和[[阿拉斯加|阿拉斯加]]到[[火地岛|火地岛]]，包括[[西印度群岛|西印度群岛]]都有牠们的踪迹。[[黑颏北蜂鸟|黑颏北蜂鸟]]是[[美国|美国]]和加拿大西部最常见的种类。只有[[红喉北蜂鸟|红喉北蜂鸟]]在北美洲东部繁衍，但是其他种类的蜂鸟个别成员也可以在北美洲东部見到，有时是来自[[古巴|古巴]]或[[巴哈马群岛|巴哈马群岛]]的游民。

蜂鸟在[[歐洲|歐]][[亞洲|亞]][[非洲|非]]全境均没有分布，但有些[[天蛾|天蛾]]（如[[長喙天蛾|長喙天蛾]]）在白天活动取食花蜜时会被误认为是蜂鸟。[[太陽鳥科|太陽鳥科]]物種同樣會吃花蜜，但分佈於東亞及東南亞。兩者在外型上的近似是基於[[趨同演化|趨同演化]]，而不是在分類學上相關。

== 生理特征 ==
[[File:Colibri-thalassinus-001.jpg|thumb]]
* 形态特征
:蜂鸟的羽毛一般为蓝色或绿色，下体较淡，有的雄鸟具有羽冠或修长的尾羽。雄鸟中，绝大多数为蓝绿色，也有的为紫色、红色或黄色。雌鸟体羽较为暗淡。
* 新陈代谢
:为适应翅膀的快速拍打，蜂鸟的[[代謝|代谢率]]是所有动物是最快的，它们的[[心跳|心跳]]能达到每分钟100到1000下，依體型不同。蜂鸟每天消耗的食品远超过他们自身的体重，为了获取巨量的食物，它们每天必须采食数百朵花。有时候蜂鸟必须忍受好几个小时的饥饿，为了适应这种情况，它们能在夜里或不容易获取食物的时候减慢新陈代谢速度，进入一种像冬眠一样的状态，称为“{{link-en|蛰伏|torpor}}”。 在“蛰伏”期间，心跳的速率和呼吸的频率都会变慢，以降低对食物的需求。
* 飞行速度
:蜂鸟飞行的速度是時速90公里，如果是俯冲的话，时速可以达到100公里。
* 寿命
:对于蜂鸟寿命的研究资料比较少，大部分专家认为蜂鸟的平均寿命为三至四年，人工饲养的蜂鸟寿命可达十年之久，野外记录的蓝胸蜂鸟的寿命仅有七年。

== 食物 ==
蜂鸟喜愛[[有花植物|有花植物]]（尤其是[[红色|红色]]花朵），包括[[小蝦花_(尖尾鳳屬)|小蝦花]]、[[倒挂金钟|倒挂金钟]]（又名吊钟花）及[[钓钟柳|钓钟柳]]类等的植物，蜂鸟採食[[花蜜|花蜜]]為主食，但大多数的蜂鸟也以昆虫为食。蜂鳥因常採食花蜜，所以也是重要的传粉者，特别是对於长筒花。

蜂鸟的运动量十分巨大，为了供应體力的高消耗，蜂鸟每天会吃下约为自身体积两倍的食物（[[花粉|花粉]]），同时也因此被认为有着惊人的新陈代谢速度。

蜂鸟一次产两个非常小的白色的[[卵|卵]]，但是与鸟的身材比較起来也頗為巨大。卵的孵化期通常为15至19天，孵化後17天背部便長齊鮮豔羽毛，開始練習飛行。

== 蜂鸟与人 ==
=== 蜂鸟的餵養 ===
[[File:Lightmatter_hummingbird.jpg|thumb]]
在美加常于户外安置特制的“奶瓶”加入人工花蜜喂养蜂鸟，特别是红色等色彩鮮豔的奶瓶，用红色的花形器具都会有極好的招引效果。合适的人工花蜜由一份[[蔗糖|蔗糖]]和四份水组成，蔗糖最容易在沸水中溶化，待完全冷却后再提供蜂鸟食用。除了白糖以外也可用其他食物，但如[[蜂蜜|蜂蜜]]发酵太快，會因此而会伤到蜂鸟。

也有一些出售的蜂鸟食品，但是通常含有不需要的红色色素，有报道称色素会使蜂鸟中毒。出售的蜂鸟食品也包含少量的营养物，但是蜂鸟显然以捕食昆虫来获取营养，所以营养物較不需要，用白糖和水便可作成最適合牠們的花蜜。

蜂鸟的“奶瓶”应每周清洗和更换糖水，如果气候暖和的话要更频繁，最少一个月更换一次，或发现黑色霉菌出现时便必须更换，{{fact|“奶瓶”应在[[氯|氯]][[漂白粉|漂白粉]]溶液中浸泡。蜂鸟不愿意使用肥皂清洗过的“奶瓶”，因為牠們較不喜欢肥皂的气味。}}

=== 被困的蜂鸟 ===
蜂鸟有时会误入[[车库|车库]]被困住，因为牠们将悬挂的门闩手柄（通常为红色）误認為是花朵，但有时蜂鸟也会被不含任何红色的围栏所困住。一旦被困在里面，蜂鸟可能无法逃脱，因为牠们在遇到威胁或被困住的时候[[本能|本能]]反应是向上飞。这将威胁到蜂鸟的生命，牠们会因为体力耗尽而在短时间内死亡，其时间可能短于一个小时。如果蜂鸟被困住，你可以抓住牠并将牠释放到室外。 当牠被抓在手中时，牠会保持安静，直到被释放。

=== 蜂鸟的神话 ===
蜂鸟在[[阿兹特克神话|阿兹特克神话]]中被稱作[[維齊洛波奇特利|維齊洛波奇特利]]（[[納瓦特爾語|納瓦特爾語]]：Huitzilopochtli，又譯作「威齊洛波契特里」、「惠齊洛波契特利」、「呼茲洛波奇特里」），為[[太阳神|太阳神]]之意，同時也代表著战争之神。

=== 蜂鸟的故事 ===
[[旧金山|旧金山]]土著民族[[歐隆族|歐隆族]]（Ohlone）讲述关于蜂鸟如何把火种带到人间的故事。

=== 蜂鸟的保护 ===
由于蜂鸟的羽毛十分华丽，在19世纪时，欧美妇女常用蜂鸟的羽毛做为帽饰，还有商人收购蜂鸟皮，对于蜂鸟的生存造成很大威胁。在现代社会中，随着[[森林|森林]]的砍伐、[[耕作|耕作]]的发展，蜂鸟赖以生存的栖息地逐渐被破坏，有的蜂鸟也面临[[灭绝|灭绝]]的危险。
