'''蜂蜜檸檬'''，是在[[2018年中華民國地方公職人員選舉|2018年中華民國地方公職人員選舉]]期間出現的[[網路迷因|網路迷因]]。起因是[[無黨籍|無黨籍]][[台北市市長|台北市市長]]候選人[[吳蕚洋|吳蕚洋]]在參選期間多次提到自己飲用了[[蜂蜜檸檬|蜂蜜檸檬]]以後產生的特殊效果，並認為[[臺北市政府|臺北市政府]]應提供蜂蜜檸檬以促進民眾健康，還能藉此拯救[[長期照顧|長期照顧]]與[[全民健康保險|全民健康保險]]財務危機；相關言論引起[[觀眾|觀眾]]和[[網友|網友]]對蜂蜜檸檬的好奇，並造成[[網路爆紅|網路爆紅]]，逐漸變成[[網路迷因|網路迷因]]。

== 起因 ==
[[吳蕚洋|吳蕚洋]]在登記參選2018年的台北市長選舉後，以「平安、健康、智慧、慈悲」為競選主軸<ref name="參政"/>，並提出推廣「蜂蜜檸檬」的相關政見。他表示之所以參與選舉，是[[彌勒佛|彌勒佛]]的[[神諭|佛旨]]，想透過政治力量推廣蜂蜜檸檬的好處，讓[[眾生|眾生]]遠離病苦<ref name="自拍影片"/>。吳蕚洋在2018年11月4日由[[TVBS|TVBS]]舉辦的「107年臺北市長候選人辯論會」，同月10日由[[公共電視文化事業基金會|公共電視文化事業基金會]]主辦的「2018台北市長選舉電視辯論」，以及同月11日由台北市政府主辦的「台北市公辦政見發表會」裡，多次提到這項主張。

吳蕚洋宣稱蜂蜜檸檬是「人間功能最強的食物」<ref name="自拍影片"/><ref name="TVBS辯論2"/>，他表示自己的眼睛下方本來有瘤，手上也長了[[汗斑|汗斑]]，醫生都束手無策，但在[[素食|茹素]]並飲用蜂蜜檸檬後，這些情況都有所改善。他表示營養師是因為不懂蜂蜜檸檬的神奇之處，才會與自己抱持不同意見<ref name="公視辯論1"/>；並期望眾人能相信蜂蜜檸檬一定有效果<ref name="政見2"/>。

吳蕚洋認為，[[生老病死|生老病死]]是人生四苦，當中只有疾病帶來的苦可以自己控制；如果透過飲用蜂蜜檸檬來養生，就能讓眾生遠離疾病帶來的苦<ref name="TVBS辯論1"/>，而透過政府的力量推動，則可以讓人民更快了解這項方法。他的政見包括讓臺北市政府和廠商合作，每日配送便宜新鮮的蜂蜜檸檬供市民購買。他認為，只要透過喝蜂蜜檸檬來強身健體，就能讓[[全民健康保險|全民健康保險]]的支出下降，還能解決[[長期照顧|長期照顧]]問題<ref name="TVBS辯論2"/><ref name="公辦辯論2"/><ref name="200萬"/>。

另外，吳蕚洋兒子兼其競選總幹事吳承澔也支持父親的說法，表示自己因為喝蜂蜜檸檬，十年來都不曾看過醫生<ref name="政見1"/>。吳蕚洋於[[兆豐金控|兆豐金控]]的同事也表示，吳蕚洋在兆豐金控任職十多年間不斷推廣蜂蜜檸檬的好處，並教導同仁如何製作蜂蜜檸檬<ref name="理念"/>。

== 傳播與影響 ==
吳蕚洋這一套關於蜂蜜檸檬的主張和政見在經過辯論會的傳播後，引起廣泛迴響，在2018年11月11日14時的[[Google|Google]]的搜尋熱門程度中，「蜂蜜檸檬」這一關鍵字達到了100分滿分<ref name="松果"/>。[[全家便利商店|全家便利商店]]趁機推出買十罐蜂蜜檸檬水，再送十罐蜂蜜檸檬水的限量優惠促銷，開賣三分鐘內，超過兩萬瓶蜂蜜檸檬就被搶購一空；[[摩斯漢堡|摩斯漢堡]]推出優惠的蜂蜜檸檬蒟蒻禮盒，而電商業者「好吃市集」、「松果購物」的蜂蜜、檸檬相關商品也推出降價促銷<ref name="松果"/><ref name="電商70"/><ref name="全家1"/><ref name="全家2"/>。也有攤販業者在得知消息後趕工製造蜂蜜檸檬水，並打著「北市辯論會唯一推薦」的招牌，在[[民主進步黨|民主進步黨]][[高雄市市長|高雄市市長]]候選人[[陳其邁|陳其邁]]的造勢場合銷售<ref name="攤販"/>。

有不少藝人、[[YouTuber|YouTuber]]和政治人物都搭上這股蜂蜜檸檬的熱潮，例如發言時提及蜂蜜檸檬，或是分享自製的蜂蜜檸檬，包括[[田馥甄|田馥甄]]<ref name="電商70"/>、[[姚文智|姚文智]]<ref name="姚文智1"/><ref name="姚文智2"/>、[[柯志恩|柯志恩]]<ref name="柯志恩"/>、[[馬英九|馬英九]]、[[黃敏惠|黃敏惠]]<ref name="馬英九"/><ref name="黃敏惠"/>、[[台灣經濟研究院|台灣經濟研究院]]<ref name="台經院"/>、壹加壹<ref name="壹加壹"/>、圖佳<ref name="圖佳"/>、搞神馬<ref>{{Citation|last=搞神馬|title=【超巨大】500倍健康！巨大蜂蜜檸檬！|url=https://www.youtube.com/watch?v=qQhsRe-hO24|accessdate=2018-12-15}}</ref>等。先前參與[[湖南衛視|湖南衛視]]美食節目《[[中餐廳|中餐廳]]》的藝人[[范冰冰|范冰冰]]與[[趙薇|趙薇]]也被提及，因為她們曾於節目中表示自己的養生之道就是每天喝蜂蜜檸檬<ref name="趙薇"/>。

由於吳蕚洋的政見除了「以蜂蜜檸檬救健保和長照」，還包括「以[[鳳梨酥|鳳梨酥]]救外交」，提到送鳳梨酥給[[中華民國外交#邦交國|中华民国邦交國]]，「那些國家說不定吃了之後，覺得『很好吃』，就不會跟你[[斷交|斷交]]了」<ref>{{cite web|title= 2018-11-11他的蜂蜜檸檬夯 吳蕚洋再提「鳳梨酥」救外交|url=http://news.ltn.com.tw/news/politics/breakingnews/2609160|agency=自由時報}}</ref>。一些網友表示，吳蕚洋比起台北市長，更適合擔任[[臺北農產運銷公司|臺北農產運銷公司]]總經理<ref name="北農"/>。對於引發風潮，吳蕚洋表示，能夠讓眾人知道蜂蜜檸檬的效果，選舉保證金新台幣200萬元用得十分值得<ref name="政見1"/>。

== 醫師與營養師見解 ==
儘管引發網路熱潮，但吳蕚洋的言論並不被中西醫的醫師和營養師支持。皮膚科醫師黃毓惠、營養師李錦秋、中醫師陳潮宗、營養師黃淑惠都表示，蜂蜜檸檬水無法解決臉部小[[瘤|瘤]]或[[汗斑|汗斑]]<ref name="黃毓惠"/><ref name="李錦秋"/><ref name="陳潮宗"/><ref name="黃淑惠"/>，[[臺北醫學大學附設醫院|臺北醫學大學附設醫院]]營養室主任蘇秀悅表示飲用過量的蜂蜜檸檬會攝取過高的熱量，牙醫師賴榮毅指出檸檬的酸性會提高[[蛀牙|蛀牙]]風險<ref name="蘇秀悅賴榮毅"/>。中醫師陳盈光指出，蜂蜜檸檬並沒有奇效，寒性體質和有[[糖尿病|糖尿病]]的人不能多喝<ref name="陳盈光"/>。[[衛生福利部臺南醫院|衛生福利部臺南醫院]]腸胃科醫師鄭維鈞、[[馬偕紀念醫院|馬偕紀念醫院]]營養師趙強表示，不該迷信單一食物，應該均衡飲食<ref name="鄭維鈞"/><ref name="趙強"/>。

== 對迷因的評論 ==
公關行銷專家林偉立認為，吳蕚洋的言論天馬行空卻能在網路上爆紅，靠的是「真心」；他認為，吳蕚洋就如同2014年參選台北市長的[[趙衍慶|趙衍慶]]一樣，不同於傳統政治人物，而是想向大眾傳揚理念，因此造成支持<ref name="行銷"/>。工程師[[翟本喬|翟本喬]]認為，蜂蜜檸檬事件說明了選舉保證金制度的問題：如果有人以新台幣200萬元保證金作為廣告費，到政治場合上進行商業宣傳，[[中華民國中央選舉委員會|中央選舉委員會]]無法遏止<ref name="翟本喬"/>。精神科醫師沈政男評論，吳蕚洋如同[[武俠小說|武俠小說]]裡的隱世高手，在其他市長參選人唇槍舌戰時，以超脫的角度宣揚蜂蜜檸檬的好處<ref name="沈政男"/>。食品議題評論媒體《飴-關於食的黑與白》的寫手「在宇宙牧羊的擺渡人」撰文指出，以產品行銷角度而言，蜂蜜檸檬引發的風潮十分成功，原因包括言論具有驚奇感、商品價格親民、沒有特定政治立場<ref name="宇宙牧羊"/>。[[端傳媒|端傳媒]]有評論認為，除去網路熱潮的部分，此事可見到世代和解共同面對島嶼困局的隱喻。<ref name="徐和謙"/>
