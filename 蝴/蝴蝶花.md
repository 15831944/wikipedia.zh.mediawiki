'''蝴蝶花'''（[[学名|学名]]：{{lang|la|''Iris japonica''}}）是[[鸢尾科|鸢尾科]][[鸢尾属|鸢尾属]]的植物。

== 形态 ==
多年生[[草本|草本]]，具横生根状茎；剑形的叶子交互排列成两行；花茎高于叶子，分枝成疏总状花序，初夏开淡紫色蝶形花，故名；椭圆状卵圆形[[蒴果|蒴果]]。

== 分布 ==
分布在[[日本|日本]]以及[[中国大陆|中国大陆]]的[[甘肃|甘肃]]、[[广西|广西]]、[[安徽|安徽]]、[[福建|福建]]、[[湖南|湖南]]、[[云南|云南]]、[[四川|四川]]、[[江苏|江苏]]、[[广东|广东]]、[[浙江|浙江]]、[[贵州|贵州]]、[[陕西|陕西]]、[[湖北|湖北]]等地，生长于海拔3,000米至3,300米的地区，一般生于山坡较阴蔽而湿润的草地、疏林下或林缘草地，目前尚未由人工引种栽培。

== 别名 ==
别名：日本鸢尾（[[中国植物学杂志|中国植物学杂志]]）<ref>[http://frps.eflora.cn/frps/%E8%9D%B4%E8%9D%B6%E8%8A%B1 中国植物志 ]</ref>，开喉箭、兰花草、扁竹、剑刀草（[[湖南|湖南]]），豆豉草、扁担叶、扁竹根（四川），铁豆柴（贵州）。
