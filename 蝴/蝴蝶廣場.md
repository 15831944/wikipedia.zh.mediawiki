'''蝴蝶廣場'''（[[英語|英語]]：'''{{lang|en|Butterfly Plaza}}'''），前稱'''蝴蝶邨商場'''（[[英語|英語]]：'''Butterfly Shopping Centre'''），是一個位於[[香港|香港]][[新界|新界]][[屯門|屯門]][[蝴蝶邨|蝴蝶邨]]的購物商場，於1984年落成。商場位於在[[蝴蝶邨|蝴蝶邨]]內，主要為邨內居民提供日常需要，因而得名。除店舖及食肆外，商場於地下亦設有[[街市|街市]]，名為「蝴蝶街市」。

該商場原來由[[香港房屋委員會|香港房屋委員會]]管理，後來在2005年交由[[領展房地產投資信託基金|領展房地產投資信託基金]]（時稱領匯）管理，於2008年至2010年進行商場翻新工程，並引入不少商店、食肆和各種生活所需服務，並改名為「蝴蝶廣場」。商場另設313個車位，方便駕車人士，亦鄰近熟食檔和鮮活街市。<ref>{{cite web |author1=領展 |title=蝴蝶廣場 |url=http://www.thelink.com.hk/tc/shopCentre/bexxc2 |website=http://www.thelink.com.hk/tc/shopCentre/bexxc2 |accessdate=2018-10-10}}</ref>

== 簡介 ==
蝴蝶廣場原稱蝴蝶邨商場，樓高3層，佔地28,400[[平方米|平方米]]，翻新後主要商戶以大型連鎖商戶為主，包括[[惠康超級市場|惠康超級市場]]、[[中銀香港|中銀香港]]、[[萬寧_(零售商)|萬寧]]等。食肆包括[[麥當勞餐廳|麥當勞餐廳]]、[[大家樂|大家樂]]、[[吉野家|吉野家]]等。此外，商場設有[[街市|街市]]、蝴蝶郵政局及有310個車位的[[停車場|停車場]]。

== 翻新工程 ==
[[領展|領展]]於2016年的業積顯示，領展預估耗資3.29億元展開商場連街市翻新工程，是領展正在翻新的11商場中成本最高的。

===商場===
2008年，[[領展|領展]]為商場進行翻新工程，成為[[屯門區|屯門區]]旗下首個商場進行有關工程，不少小商戶不獲續約而結業。於2010年改稱為'''蝴蝶廣場'''。整個翻新工程於2010年年終完成，完成工程後，大堂位置的大樓梯及休息位置改建成為店鋪，商場地下及3樓的店鋪數目也有增加，商場也加入了不少[[連鎖店|連鎖店]]，包括[[日本城|日本城]]、[[惠康|惠康]]、[[美國冒險樂園|美國冒險樂園]]等。

===街市===
2015年2月28日起，[[領展|領展]]展開街市翻新工程，翻新工程期間，原有過百商戶中，只有22檔獲安排到臨時街市繼續經營。14個月後資產提升工程完成，2016年4月29日早上10點正式重開，變身為佔地30,000呎街市，為全港最大單層街市，由建華集團承包。<ref>[http://hk.apple.nextmedia.com/realtime/finance/20160608/55200770 【領展業績】蝴蝶廣場用3億超豪翻新 《蘋果日報》 2016-06-08]</ref>翻新後街市舖租高昂，魚檔月租約8萬，位於角落的燒味鋪月租7萬。所有商戶即使生意差亦不能轉場。<ref>{{cite news|title=【領展業績】屯門蝴蝶街市兩成吉舖 淡季冇得減租|url=http://hk.apple.nextmedia.com/realtime/finance/20160608/55202030|accessdate=2017-02-19|publisher=蘋果日報|date=2016-06-08}}</ref>

===冬菇亭===
到2016年3月，[[冬菇亭|冬菇亭]]同時亦進行翻新，三檔開業多年的商戶，包括已有35年歷史的發財記、儲財記，以及祥順景因領展要求檔戶須租用整個冬菇亭單位，而被迫結業。<ref>[http://www.inmediahk.net/node/1041386 領展霸權 35年歷史蝴蝶邨冬菇亭結業 香港獨立媒體 2016-03-21]</ref>
