'''蝴蝶'''（英語：Butterfly）是香港[[屯門區議會|屯門區議會]]下轄的[[選區|選區]]，代號'''L18'''，成立於1982年，現任區議員為[[民協|民協]]成員[[楊智恒|楊智恒]]。

==範圍==
選區範圍為[[蝴蝶邨|蝴蝶邨]]和[[兆山苑|兆山苑]]，與其相鄰的選區有[[富新|富新]]、[[湖景_(選區)|湖景]]和[[樂翠_(選區)|樂翠]]<ref>[http://www.elections.gov.hk/dc2011/maps/dc2011l.pdf]</ref>。

==沿革==
蝴蝶選區首次出現於[[1991年香港區議會選舉|1991年香港區議會選舉]]，由[[屯門展望社|屯門展望社]][[楊美光|楊美光]]和[[王健康|王健康]]爭奪，最後[[楊美光|楊美光]]勝出。

[[1994年區議會選舉|1994年區議會選舉]]，[[楊美光|楊美光]]沒再參選。這次有當時是[[新界西居民聯會|新界西居民聯會]]的[[蘇愛群|蘇愛群]]和[[民協|民協]][[章楚英|章楚英]]爭奪，最後蘇愛群勝出。

[[1999年區議會選舉|1999年區議會選舉]]，當時人口估算約15,381人<ref name="elections.gov.hk">{{cite web |url=http://www.elections.gov.hk/elections/dcelect99/cg12.htm |title=存档副本 |accessdate=2012-04-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20111229025617/http://www.elections.gov.hk/elections/dcelect99/cg12.htm |archivedate=2011-12-29 }}</ref>，其中有7,216位是選民<ref name="elections.gov.hk"/>。今次[[蘇愛群|蘇愛群]]自動當選，蘇當時已經加入[[民主建港聯盟|民主建港聯盟]]。

[[2003年區議會選舉|2003年區議會選舉]]，當時人口估算約14,411人<ref>{{cite web |url=http://www.elections.gov.hk/elections/dc2003/tc_chi/summary/summary_tm.html |title=存档副本 |accessdate=2012年4月14日 |deadurl=yes |archiveurl=https://web.archive.org/web/20111108185646/http://www.elections.gov.hk/elections/dc2003/tc_chi/summary/summary_tm.html |archivedate=2011年11月8日 |df= }}</ref>，其中有6,470位是選民<ref>{{cite web |url=http://www.elections.gov.hk/elections/dc2003/tc_chi/eb/eb_num_tm.html |title=存档副本 |accessdate=2012年4月14日 |deadurl=yes |archiveurl=https://web.archive.org/web/20110115013125/http://www.elections.gov.hk/elections/dc2003/tc_chi/eb/eb_num_tm.html |archivedate=2011年1月15日 |df= }}</ref>。今次[[蘇愛群|蘇愛群]]和[[民協|民協]][[鄭嘉樂|鄭嘉樂]]爭奪，最後蘇愛群勝出。

[[2007年區議會選舉|2007年區議會選舉]]，當時[[兆山苑|兆山苑]]劃入此區，人口估算增到約18,590人<ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/summaryl.html |title=存档副本 |accessdate=2012年4月14日 |deadurl=yes |archiveurl=https://web.archive.org/web/20150621171907/http://www.elections.gov.hk/dc2007/chi/summaryl.html |archivedate=2015年6月21日 |df= }}</ref>，其中有10,197位是選民<ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/num_electors_eastern.html |title=存档副本 |accessdate=2015-12-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20150621034533/http://www.elections.gov.hk/dc2007/chi/num_electors_eastern.html |archivedate=2015-06-21 }}</ref>，今次[[蘇愛群|蘇愛群]]和[[民協|民協]][[楊智恒|楊智恒]]爭奪，最後蘇愛群勝出。

[[2011年區議會選舉|2011年區議會選舉]]，當時人口估算約17,631人<ref>[http://www.elections.gov.hk/dc2011/chi/summaryc.html]</ref>，其中有10,154位是選民<ref>[http://www.elections.gov.hk/dc2011/chi/ebriefsc.html]</ref>，今次有[[蘇愛群|蘇愛群]]、[[人民力量|人民力量]]温瑞岐和前[[民協|民協]]成員，前[[兆新|兆新]](即[[富新|富新]]區)區議員[[戴賢招|戴賢招]]爭奪，最後蘇愛群勝出。

[[2015年區議會選舉|2015年區議會選舉]]，民主黨原本有意派[[溫朗鈞|溫朗鈞]]於樂翠選區接現任雙料議員[[何俊仁|何俊仁]]棒，但在前香港律師會會長[[何君堯|何君堯]]決定參選後，何俊仁留在樂翠選區競逐連任，溫朗鈞擬轉戰蝴蝶選區，但在提名期及民主黨誓師大會不見其蹤影。[[民協|民協]][[楊智恒|楊智恒]]臨危受命參選和[[蘇愛群|蘇愛群]]爭奪，最後[[楊智恒|楊智恒]]勝出，打破[[建制派|建制派]]在該區近20年的控制。
