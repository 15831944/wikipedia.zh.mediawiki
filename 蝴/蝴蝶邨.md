'''蝴蝶邨'''（{{lang-en|'''Butterfly Estate'''}}）是[[香港公共屋邨|香港公共屋邨]]，位於[[新界|新界]][[屯門|屯門]]近[[蝴蝶灣|蝴蝶灣]]，其所屬位置於發展屯門新市鎮時進行填海工程所得出的土地。於1983年開始入伙，由[[香港房屋委員會|香港房屋委員會]]管理，亦是唯一一個梯級型的公屋樓宇。由於屋邨所有單位的設計大致上是一樣的，所以邨內居民大部分是1-4人的家庭為主，截至2018年6月30日住戶數目為5300戶。<ref>{{cite web |author1=香港房屋委員會 |title=蝴蝶邨, 屯門, 新界西 |url=https://www.housingauthority.gov.hk/tc/global-elements/estate-locator/detail.html?propertyType=1&id=2730 |website=https://www.housingauthority.gov.hk/tc/global-elements/estate-locator/detail.html?propertyType=1&id=2730 |accessdate=2018-10-11}}</ref>

另邨內有一由[[領展|領展]]管理的商場，名為[[蝴蝶廣場|蝴蝶廣場]]，亦設有一[[街市|街市]]，根據2015年3月中公布，該街市正式名為'''蝴蝶鮮活街市'''，2016年4月28日完成翻新，同年4月29日早上10時正式重開。熟食檔（冬菇亭）亦在2016年3月24日正式翻新，於2017年1月完成重開。

== 設施 ==
邨內設有多個設施，包括停車場、多個兒童遊樂場、兩個足球場、兩個籃球場、羽毛球場、籃球場及乒乓球桌。蝴蝶街市對出設小廣場和水池。邨內設有多個公共設施，包括蝴蝶公共圖書館、蝴蝶郵政局及蝴蝶灣社區中心。

== 商場 ==
{{main|蝴蝶廣場}}
購物商場樓高3層，佔地28,400平方米，於1984年落成，主要為邨內居民提供日常需要，因而得名。

除店舖及食肆外，商場於地下亦設有街市，名為「蝴蝶街市」。冬菇亭於2016年進行翻新，三檔開業多年的商戶，包括已有35年歷史的發財記、儲財記，以及祥順景因領展要求檔戶須租用整個冬菇亭單位，而被迫結業。

*[[香港路德會增城兆霖學校|香港路德會增城兆霖學校]]（2005年創辦）

*香海蓮社蝴蝶邨幼稚園（已拆除）

== 選區 ==
{{main|蝴蝶 (選區)}}
蝴蝶邨屬於[[區議會|區議會]]選區分區中蝴蝶（L18）選區，該選區自[[1991年區議會選舉|1991年區議會選舉]]設立。<ref>[http://www.eac.gov.hk/pdf/distco/maps/dc2015l.pdf 區議會選區分界圖—屯門區]</ref> [[2015年香港區議會選舉|2015區議會選舉]]中，[[民建聯|民建聯]]區議員蘇愛群連任失敗，由[[民協|民協]]楊智恒當選為當區區議員。<ref> [http://www.elections.gov.hk/dc2015/chi/results_tuen_mun.html?1448254367392 2015區議會選舉結果 - 屯門區] </ref> [[香港立法會|立法會]][[地區直選|地區直選]]中屬於[[新界西|新界西]]（LC4）選區。<ref>[http://www.eac.gov.hk/pdf/legco/2016/2016boundaries/lc2016ntw.pdf 立法會地方選區分界圖—新界西選區]</ref>
