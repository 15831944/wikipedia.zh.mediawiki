'''兰山区'''是[[山东|山东]]省[[临沂|临沂]]市下辖区，[[临沂|临沂]]市政府所在地，在[[沂河|沂河]]河畔，总面积650平方千米，人口83万，辖3个办事处、8个乡镇、440个村居。古称'''启阳'''、'''开阳'''。

== 行政区划 ==
兰山区辖4个街道、10个镇：
* 街道：兰山街道、银雀山街道、金雀山街道、南坊街道
* 镇：白沙埠镇、枣沟头镇、半程镇、义堂镇、马厂湖镇、李官镇、朱保镇、新桥镇、方城镇、汪沟镇。

== 历史 ==
兰山区内的临沂老城区，始建于[[春秋时期|春秋]][[鲁哀公|鲁哀公]]三年（前492年），时称启阳邑。[[西汉|西汉]]初年置[[启阳县|启阳县]]，后来为了[[避讳|避]][[汉景帝|汉景帝刘启]]讳而改称开阳。东汉[[建初_(东汉)|建初]]五年（80年），[[开阳县|开阳县]]为[[琅琊王|琅琊王]][[刘京_(琅邪王)|刘京]]国都，此后一直为[[琅琊郡|琅琊郡]]、[[琅琊国|琅琊国]]的[[治所|治所]]。[[隋朝|隋朝]]为[[沂州|沂州]]、琅琊郡、[[临沂县|临沂县]]郡治和县治，因城东临沂水而得名临沂<ref>{{cite web|url=http://www.linyisq.gov.cn/NewsShow.asp?NewsID=465|title=兰山街道|publisher=临沂市情网|accessdate=2008-06-01}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[清朝|清朝]]复置[[蘭山縣|兰山县]]，为[[沂州府|沂州府]]治所；民国三年（1914年），改兰山县为临沂县，即今临沂市。兰山区因辖区属兰山县而得名<ref>{{Cite web |url=http://cul.dmmap.com/cul_dmwh_dimh/cul_dmwh_dimh_dmgs/20101208/60391.html |title=山东临沂市及其管辖区县的地名来历及历史沿革 |accessdate=2013-12-22 |author= |date=2010年12月8日 |work= |publisher=图骥网 |language=Zh-hans }}</ref>。

== 地理 ==
[[临沂|临沂]]市[[兰山|兰山]]区位于[[山东|山东]]省东南部，[[地理坐标|地理坐标]]为东经118°,北纬35°。东隔[[沂河|沂河]]与[[河东|河东]]区相望，西与[[费县|费县]]、[[苍山|苍山]]县接壤，南邻[[罗庄|罗庄]]区，北靠[[沂南|沂南]]县。[[兰山|兰山]]区地理位置优越，自古就有“[[齐鲁|齐鲁]]噤喉”之称，为鲁南苏北重要的交通枢纽。交通便利，327、205国道、[[兖石铁路|兖石铁路]]贯穿其中。
