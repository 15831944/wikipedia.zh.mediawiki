'''兰德尔·帕特里克·门罗'''（{{lang-en|Randall Patrick Munroe}}，{{bd|1984年|10月17日}}），美国[[网络漫画|网络漫画]]家，前[[NASA|NASA]]机器人专家<ref>{{cite news | url=http://www.wired.com/entertainment/theweb/news/2007/11/xkcd | title=Real Geek Heart Beats in Xkcd's Stick Figures | work=WIRED | date=2007-11-13}}</ref>及[[程序员|程序员]]<ref name="NYT">{{Cite news | id={{issn|0362-4331}} | last = Cohen | first = Noam | title = This Is Funny Only if You Know Unix | work = [[The_New_York_Times|The New York Times]] | accessdate = 2008-09-25 | date = 2008-05-26 | url = http://www.nytimes.com/2008/05/26/business/media/26link.html?partner=rssnyt&emc=rss }}</ref>。他创作了知名网络漫画“[[xkcd|xkcd]]”，拥有众多支持者和崇拜者。在离开NASA后，他成为一名职业网络漫画家<ref name="NYT" />。

== 网络漫画 ==
{{main|xkcd}}

[[Image:Webcomic_xkcd_-_Wikipedian_protester.png|thumb]]
“xkcd”是一个以[[计算机科学|计算机科学]]、[[科學|科技]]、[[数学|数学]]、[[科学|科学]]、[[语言|语言]]、[[流行文化|流行文化]]以及[[爱情|爱情]]为主题的火柴人漫画。

门罗最早选择xkcd作为其网络即时通讯的用户名，是因为他想要一个没有任何意义的名字，这样他就不会有一天对其感到厌倦。他注册该用户名之后一直未用，直到2005年9月开始用它发布他的画作<ref name="Red Hat" /><ref>{{cite video |people=@Google Talks (venue) |date=2007-12-11 |title=Authors@Google: Randall Munroe |url=http://www.youtube.com/watch?v=zJOS0sV2a24 |format=[[Adobe_Flash|Adobe Flash]] |medium=[[Digital_video|Digital video]] |publisher=[[Google|Google]] |location=[[Mountain_View,_California|Mountain View, California]] |accessdate=2008-09-25 |time=48:05 |quote= I'm pretty sure I started in September 2005.}}</ref>。他的漫画有众多支持者，门罗称其得到最多反馈的漫画是 [http://xkcd.com/277/ stoplights]。”

门罗使用[[創作共用授權條款|創作共用授權條款]] [http://creativecommons.org/licenses/by-nc/2.5/ attribution-noncommercial 2.5]许可授权其漫画，他说这不仅仅是为了[[自由文化运动|自由文化运动]]，在商业上也很合理<ref name="Red Hat" />。他的收入来自向其漫迷销售的产品，其中包括每月售出上千件的[[T恤|T恤]]<ref name="NYT" />。

此漫画在科幻爱好者群体中的流行，使门罗在2011年和2012年被提名[[雨果奖|雨果奖]]最佳漫迷艺术家<ref>{{cite web|url=http://www.thehugoawards.org/hugo-history/2012-hugo-awards/|accessdate=7 April 2012|title=Hugo Awards 2012 nomination|deadurl=yes|archiveurl=https://www.webcitation.org/66nnsGGRI?url=http://www.thehugoawards.org/hugo-history/2012-hugo-awards/|archivedate=2012年4月9日}}</ref>。

== 其他项目 ==

门罗还创建了网站“Best Things”<ref>{{cite web | url = http://bestthing.info | title = bestthing.info}}</ref>、"The Funniest"<ref>{{cite web | url = http://thefunniest.info | title = The Funniest | deadurl = yes | archiveurl = https://web.archive.org/web/20061205091701/http://thefunniest.info/ | archivedate = 2006-12-05 }}</ref>、“The Cutest”<ref>{{cite web | url = http://thecutest.info | title = The Cutest | deadurl = yes | archiveurl = https://web.archive.org/web/20100528145750/http://thecutest.info/ | archivedate = 2010-05-28 }}</ref>以及“The Fairest”<ref>{{cite web | url = http://thefairest.info | title = The Fairest}}</ref>。网站的网页里都给访客提供两个选项，让他们选择其中一个。 

在完成漫画“Rule 34”<ref>{{cite web | url = http://xkcd.com/305/ | title = Rule 34 | work = xkcd | first = Randall | last = Munroe}}</ref>一个月之后，他开始以开玩笑的方式创建了“WetRiffs”网站，用以补充互联网上“在淋浴里弹吉他”一类图片的不足。<ref>{{cite web |url=http://wetriffs.com/ |title=Nudity + Guitars + Showers |accessdate=2008-09-26 |last=Munroe |first=Randall |date=2008-03-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20081201162017/http://wetriffs.com/ |archivedate=2008-12-01 }}</ref> 。

他的另一个网站LimerickDB鼓励创作大家搜集旧的或创作新的[[打油诗|打油诗]]<ref>{{cite web|url=http://blag.xkcd.com/2008/02/04/limerickdbcom|title=LimerickDB.com|accessdate=2008-09-26|last=Munroe|first=Randall|date=2008-02-04|work=Blag|publisher=xkcd}}</ref>，但该网站现已无法访问。

他创建了栏目[http://wiki.xkcd.com/geohashing Geohashing wiki]，该栏目基于一个每天生成地球上一个随机坐标的[[算法|算法]]的漫画<ref>{{cite web | url = http://wiki.xkcd.com/geohashing/Main_Page | title = Geohashing Wiki}}</ref>。

他还创建了一个叫做“What If”的栏目，用户能够在网站上向他提一些与[[物理|物理]]和[[数学|数学]]相关但看起来很荒谬的问题，他会通过他的知识以及各种网上学术资源来解答问题<ref>{{cite web | url = http://what-if.xkcd.com/ | title = what if?}}</ref>。2014年，他将问答集以《那些古怪又让人忧心的问题》为书名集结出版。

除了维护他的网站，门罗还参与{{tsl|en|kite aerial photography|风筝空中摄影}}活动，利用安放在风筝上的[[照相机|照相机]]来拍摄地面和建筑的照片<ref>{{cite web | first = Ben | last = Kuchera | url = http://arstechnica.com/gaming/news/2007/07/the-joys-of-kite-photography.ars | title = The joys of kite photography | work = [[Ars_Technica|Ars Technica]] | date = 2 July 2007}}</ref>。
[[File:Radiation_Dose_Chart_by_Xkcd.png|thumb]]剂量下的放射性，从无关紧要到致命。]]

为了回应2011年大家对于[[福岛第一核电厂|福岛第一核电厂]]核辐射泄露的关注，为他称作“令人困惑”的媒体关于辐射级别的报道提供补充，门罗创作了一个比较辐射指数级别的图表。该图表很快被多个国家的印刷媒体和网络媒体的记者采用，包括被“[[卫报|卫报]]”<ref>{{cite news |last=Monbiot|first=George |authorlink=George Monbiot |title=Why Fukushima made me stop worrying and love nuclear power |url= http://www.guardian.co.uk/commentisfree/2011/mar/21/pro-nuclear-japan-fukushima |work=The Guardian |accessdate=29 March 2011 |location=London |date=2011-03-21}}</ref>及“[[纽约时报|纽约时报]]”<ref>{{cite news |last=Revkin|first=Andrew |authorlink=Andrew Revkin |title=The 'Dread to Risk' Ratio on Radiation and other Discontents |url= http://dotearth.blogs.nytimes.com/2011/03/23/the-dread-to-risk-ratio-on-radiation-and-other-discontents/?hp |work=Dot Earth blog |publisher=''The New York Times'' |accessdate=29 March 2011 |date=2011-03-23}}</ref> 的记者。在得到了将该图表再版和日语翻译的许可之后，门罗将其放在了公共网络上，但是要求在再版声明中必须注明他并非该领域专家<ref>{{cite web |last=Munroe|first=Randall |title=Radiation Chart |url=http://blog.xkcd.com/2011/03/19/radiation-chart/ |publisher=www.xkcd.com |accessdate=29 March 2011}}</ref>。
