'''兰德·保罗'''（{{lang-en|'''Rand Paul'''}}；{{bd|1963年|1月7日}}），全名'''-{zh-hk:蘭度·侯活;zh-tw:蘭德爾·霍華德;zh-cn:兰德尔·霍华德}-·「蘭德」·保羅'''（{{lang-en|Randal Howard "Rand" Paul}}），美國[[肯塔基州|肯塔基州]][[醫生|醫生]]與[[政治人物|政治人物]]；於2011年代表[[共和黨_(美國)|共和黨]]當選[[美國參議員|美國參議員]]至今。此前，兰德·保罗是一名專業[[眼科|眼科]]醫生，父親[[羅恩·保羅|羅恩·保羅]]是前[[德州|德州]][[美國眾議員|美國眾議員]]與醫生。

== 簡介 ==
兰德·保罗於[[賓夕凡尼亞州|賓夕凡尼亞州]][[匹茲堡|匹茲堡]]出生，曾就讀[[貝勒大學|貝勒大學]]；並於[[杜克大學|杜克大學]]醫學系畢業。自1993年起，他在肯塔基州Bowling Green開始行醫，並在2007年開設個人私家眼科診所。

兰德·保罗被視為[[茶黨運動|茶黨運動]]的支持者，經常對[[聯邦儲備系統|聯邦儲備系統]]發表意見；他反對[[美國國家安全局|美國國家安全局]]經常大規模監控[[美國公民|美國公民]]，也擁護裁減政府支出同埋稅收。

雖然他是[[自由意志主義|自由意志主義]]者，但他強調自己百分百支持[[生命權|生命權]]（反對[[墮胎|墮胎]]），也盡力推動立法保障平等生存權益。

2015年4月7日，兰德·保罗正式宣佈參選[[2016年美國總統選舉|2016年美國總統選舉]]<ref name="Camia">{{cite news |url=http://www.usatoday.com/story/news/politics/elections/2015/04/07/rand-paul-president-2016-announcement/25041783/ |first=Catalina |last=Camia |title=Rand Paul announces presidential run |work=[[USA_today|USA today]] |date=April 7, 2015 |accessdate=April 7, 2015}}</ref>。2016年2月3日因初选支持率过低而宣布暂停选举活动。<ref>{{Cite news|url = http://www.theguardian.com/us-news/2016/feb/03/rand-paul-drops-out-2016-presidential-election-iowa|title = Rand Paul drops out of Republican presidential race after Iowa caucuses|last = Jacobs|first = Ben|date = 2016-02-03|work = The Guardian|accessdate = 2016-02-06}}</ref><ref>{{Cite news|url = https://www.washingtonpost.com/news/post-politics/wp/2016/02/03/rand-paul-suspends-presidential-campaign/|title = Rand Paul suspends presidential campaign|last = Costa|first = Robert|date = 2016-02-03|work = The Washington Post|accessdate = 2016-02-06|last2 = Weigel|first2 = David}}</ref>其後他競逐連任參議員，最終成功連任。
