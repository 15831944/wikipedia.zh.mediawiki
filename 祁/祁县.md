'''祁县'''位于中国[[山西省|山西省]]中部、[[汾河|汾河]]中游东岸、[[太原盆地|太原盆地]]中部。

祁县有着2500余年的建[[县|县]]历史，于1994年成为中国[[国家历史文化名城|国家历史文化名城]]，并与另一座名城[[平遥|平遥]]相毗邻，县人民政府驻[[昭余镇|昭余镇]]，属[[晋中市|晋中市]]。

== 历史 ==
祁县古称昭餘，以境内昭餘祁泽薮而得名。[[春秋|春秋]]时为[[晋国|晋国]]大夫[[祁奚|祁奚]]的食邑。公元前514年设祁县，迄今已有2500餘年历史。

祁县在汉晋时代胡人内迁中曾属于匈奴五部的驻地之一。

== 名人 ==
祁县人杰地灵，自古以来名人辈出。在文人中比较出名的有[[祁奚|祁奚]]（晋国七大家族之一）、[[王允|王允]]（[[东汉|东汉]]末年的重臣、除去国贼[[董卓|董卓]]的元勋）、[[温峤|温峤]]（东晋名相）、[[温大雅|温大雅]]（《大唐起居注》作者）、[[王维|王维]]（“诗佛”）、[[温庭筠|温庭筠]]（晚唐词人。“花间派”开创者）、[[罗贯中|罗贯中]]（《[[三国演义|三国演义]]》作者）等人；武人中则有南朝名将王玄漠（[[刘裕|刘裕]]北伐之重臣）、[[王僧辩|王僧辩]]（平息“侯景之乱”，政争中败于陈太祖[[陈霸先|陈霸先]]，被杀）、[[王思政|王思政]](南北朝时期北魏、西魏的军事家)、[[王轨|王轨]]([[王允|王允]]后代，北周[[周武帝|周武帝]]宇文邕重臣)<ref>《周书·卷四十·列传第三十二》 王轨，太原祁人也，小名沙门，汉司徒允之后。世为州郡冠族。累葉仕魏，赐姓乌丸氏。父光，少雄武，有将帅才略。每從征討，頻有戰功。太祖知其勇決，遇之甚厚。位至驃騎大將軍、開府儀同三司、平原縣公。 </ref>、[[王忠嗣|王忠嗣]]（[[唐代|唐代]]边防股肱之臣，领西北四镇节度使）、[[王方翼|王方翼]]（唐代西北节度使，血战于[[吐蕃|吐蕃]]）；明清以来，又为[[晋商|晋商]]核心地带，出现了[[乔致庸|乔致庸]]、[[渠本翘|渠本翘]]等晋商名流，盛极一时。

== 祁县县城 ==
清朝末年，三晋富甲天下，有“金太谷、银祁县、铜平遥”之说，而尤以祁县执全国金融牛耳。祁县的县城可上溯至[[北魏|北魏]]时期，正中一条十字街，纵横交错三十二条道路，目前格局基本保存完好。街道、民居、商铺、寺庙等均保持明清风貌，具有较强的历史研究价值。

城内外有上千民居院落，其中以城内的[[渠家大院|渠家大院]]和城郊的[[乔家大院|乔家大院]]最出名。

*[[全国重点文物保护单位|全国重点文物保护单位]]：[[渠家大院|渠家大院]]（山西省晋商文化博物馆）
*[[全国重点文物保护单位|全国重点文物保护单位]]：[[乔家大院_(文物古迹)|乔家大院]]

== 地理 ==
[[同蒲铁路|同蒲铁路]]斜贯县境。全境土地肥沃、水源充足，盛产[[粮食|粮食]]，主要有[[小麦|小麦]]、[[玉米|玉米]]、[[高粱|高粱]]等。特产[[核桃|核桃]]和[[苹果|苹果]]，俱是远近闻名。祁县支柱产业为玻璃制品业，其吹制玻璃的产量目前居国内第一名，产品远销海外，尤其北美地区，已经形成相当规模，并有待产业提升和资本和技术注入。

== 文化 ==
祁县有着悠久而浓厚的文化氛围，由劳动人民自编自演，具有泥土风味浓厚，方言俚语流利上口，内容丰富多彩，曲调优美动听等特点的祁太秧歌，便千百年来广泛传唱于祁县、太谷一带。同时，公元1798年，最早的晋剧科班“云生班”也首先出现在祁县。

== 语言 ==
祁县方言属[[晋语|晋语]][[并州片|并州片]]，有五个声调，[[平声|平声]]不分阴阳，[[入声|入声]]分阴阳，[[文白异读|文白异读]]现象很丰富。
