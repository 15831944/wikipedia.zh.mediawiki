'''祁答院良重'''（1526年—1566年2月5日／1559年）是[[日本戰國時代|日本戰國時代]]的武將。{{tsl|ja|祁答院氏}}第13代[[當主|當主]]。父親是[[祁答院重武|祁答院重武]]。

{{tsl|ja|祁答院氏}}是[[桓武平氏|桓武平氏]]{{tsl|ja|秩父氏|秩父黨}}{{tsl|ja|渋谷氏|澁谷氏}}一族。從[[薩摩國|薩摩國]]北部擴大勢力至[[大隅國|大隅國]]，與同是澁谷一族的{{tsl|ja|東郷氏|東鄉氏}}、{{tsl|ja|入来院氏|入來院氏}}等連結，與薩摩[[守護|守護]][[島津氏|島津氏]]周旋。

== 生平 ==
[[天文_(後奈良天皇)|天文]]7年（1538年），因為父親重武死去，繼任[[家督|家督]]。天文23年（1554年），同盟{{tsl|ja|蒲生氏}}攻撃投向島津氏（與蒲生氏、祁答院氏敵對）的[[肝付氏|肝付氏]]的{{tsl|ja|加治木城}}。對此，[[島津貴久|島津貴久]]在同年9月12日，率領攻向蒲生氏的{{tsl|ja|岩剣城|岩劍城}}。此時，率兵迎擊，岩劍城是在三面都是懸崖的山上，[[島津忠良|島津忠良]]（日新齋）見到城池如此險要後，向[[島津義久|島津義久]]等三兄弟問誰敢奮死攻堅；而蒲生軍得知岩劍城被包圍後，解除加治木城的包圍並返回救援；自身則在岩劍城北部的平松（現今[[鹿兒島縣|鹿兒島縣]][[姶良市|姶良市]]）與島津軍激戰，並敗走，在這次戰鬥中，嫡子[[祁答院重經|重經]]戰死；在沒有見到援軍的情況下，與城兵一同在夜晚逃離；同年10月3日，島津軍攻佔岩劍城（在此戰中，雙方都有使用種子島（[[火繩銃|火繩銃]]），被認為是在[[日本|日本]]初次於實戰中使用[[鐵砲|鐵砲]]的戰役）。

此後，以[[帖佐城|帖佐城]]、{{tsl|ja|平安城}}為據點，繼續抵抗島津貴久的侵攻。天文24年（1555年），受[[島津義弘|島津義弘]]、[[喜入季久|喜入季久]]等猛攻，於是向祁答院撤退。之後，待到蒲生氏支援，嘗試奪還帖佐，不過被義弘軍擊敗。[[弘治_(日本年號)|弘治]]3年（1557年），蒲生氏的本據被島津軍包圍，於是為其當主[[蒲生範清|蒲生範清]]仲介，向島津貴久降伏。此時，迎範清、[[蒲生為清|為清]]父子至祁答院的[[松尾城_(薩摩國)|松尾城]]，提供保護。

失去帖佐的本城，蒲生城的救援失敗，失去最大的同盟者，失去最強盛時期的勢力，不過從姶良地方撤退，並以[[虎居城|虎居城]]為據點，與{{tsl|ja|入来院氏|入來院氏}}、{{tsl|ja|東郷氏|東鄉氏}}等澁谷一族的諸勢力連結，保持一定力量。[[永祿|永祿]]9年（1566年）正月，被妻子虎姬（[[島津實久|島津實久]]的女兒（[[島津義虎|義虎]]的姐姐），一說為島津義虎的女兒）刺殺（一說是受島津貴久侵攻，所領被奪去，並在永祿2年（1559年）被殺害）。法名是'''樹蔭得鈕大禪定門'''。

== 死後 ==
死後，{{tsl|ja|祁答院氏}}無法繼續經營領地，因為祁答院氏的家長[[大井實勝|大井實勝]]、[[高城重治|高城重治]]、[[久富木重全|久富木重全]]3人的[[連判|連判]]，祁答院氏的領地全部讓渡予[[入來院重嗣|入來院重嗣]]，於是沒落。另外，出奔到[[日向國|日向國]]飫肥的三男[[祁答院重加|重加]]被定為嫡流。繼承家督的子孫是養子，加入島津家後，在各地轉戰，子孫成為入來院氏、{{tsl|ja|喜入氏}}的家臣。

== 人物、逸話 ==
*射箭能手，亦熱心於養馬等，被認為是戰上手。不過亦有以小孩為箭靶以殺害、監禁不順從自己意見的家臣，並流放到城外等強橫的行為（這些事績記載在敵對的島津氏側的記錄，因此可能被誇大或捏造）。

*[[天文_(後奈良天皇)|天文]]9年（1540年），修築虎居的大明神。[[永祿|永祿]]4年（1561年），修築鬼丸大明神。永祿8年，修築{{tsl|ja|大願寺}}的雨華堂。
