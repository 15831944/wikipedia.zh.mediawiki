'''祁阳县'''，隶属于[[湖南省|湖南省]][[永州市|永州市]]市辖县，位于湖南省西南部，湘江中上游，永州市东北部。西接永州市零陵区和冷水滩区，东抵常宁市，南临新田县、宁远县、双牌县和桂阳县，北连祁东县。地处东经110°35′-112°14′，北纬26°02′-26°51′。

==历史==
[[三国|三国]]时期（268年）开始置县，故址在今[[祁东县|祁东县]]。[[唐朝|唐朝]]时，移至今县境，属[[永州_(隋朝)|永州]]。历五代十国、两宋。至元朝时属[[永州路|永州路]]，明清时，祁阳县属[[永州府|永州府]]。

中华民国初年，全国[[废府州厅改县|废府]]，直属[[湖南省_(中華民國)|湖南省]]。1949年，[[第二次国共内战|第二次国共内战]]进入尾声。[[衡宝战役|衡宝战役]]后，解放军攻占湖南全省指日可待。10月9日至12月5日间，在当地游击队配合下，解放军先后攻占了今永州市各县。祁阳县是最早攻占的县城。10月9日，[[中国人民解放军第40军|中国人民解放军第40军]]120师在地方武装配合下攻占祁阳县城<ref>{{Cite web |url=http://lib.huse.cn/lzy/news_view.asp?newsid=6312 |title=第十六章 国民党统治的崩溃与永州人民的解放（二）第四节永州全境的解放|accessdate=2018-06-21 |author= |date=2012-07-05 |work= |publisher=湖南科技学院图书馆——潇湘文化专题网，来源：《永州史话》|language=Zh-hans }}</ref>。其后，祁阳县历属[[永州专区|永州专区]]、[[零陵专区|零陵专区]]、[[湘南行政区|湘南行政区]]、[[衡阳专区|衡阳专区]]、[[衡阳地区|衡阳地区]]。1983年撤消衡阳地区，原有七县中，六县划入衡阳市，仅祁阳县划至[[零陵地区|零陵地区]]。1995年，零陵地区改为永州市，隶属至今。

== 地理 ==

=== 行政区划 ===

2013年，祁阳县的2个[[类似乡级单位|类似乡级单位]]——[[大江林场|大江林场]]、[[挂榜山林场|挂榜山林场]]改设为乡。2015年，祁阳县曾大幅调整乡镇区划。获得[[湖南省人民政府|湖南省人民政府]]的批准后，11月24日，永州市人民政府下发永政函〔2015〕216号文件，撤并七个乡镇（[[大江乡|大江乡]]、[[内下乡|内下乡]]、[[挂榜山乡|挂榜山乡]]、[[白果市乡|白果市乡]]、[[上司源乡|上司源乡]]、[[万宝山乡|万宝山乡]]和[[小金洞乡|小金洞乡]]）。调整后的祁阳县，共辖26个[[乡级行政区|乡级行政区]]（1个民族乡、2个乡、20个镇以及3个街道）<ref>{{Cite web |url=http://www.yzcity.gov.cn/cnyz/szfwj/201705/401c15abb12346a09891fd95dcf3f13d.shtml |title=永州市人民政府关于祁阳县乡镇区划调整的通知 |accessdate=2018-06-21 |author=|date=2015-12-03 |work= |publisher=永州市人民政府网站，永政函〔2015〕216号文件|language=Zh-hans }}</ref>，另有2个类似乡级单位。

=== 旅游景点 ===
祁阳境内文化古迹众多，风景秀丽。由浯溪碑林、文昌塔、潇湘楼、甘泉寺、大江自然风光、金洞漂流、陶铸故居七大景区构成核心风景名胜区。

*[[浯溪摩崖石刻|浯溪摩崖石刻]]：是全国第二大石刻碑林，为[[国家重点文物保护单位|国家重点文物保护单位]]，现存有唐代以来摩崖石刻505块，楷、行、草、纂诸体皆备，是全国罕见的露天碑林，由元结撰文，颜真卿大字正书的《大唐中兴颂》，自古以来称其三奇：文奇、字奇、石奇，世称“摩崖三绝”。
