'''投毒捕鱼'''（{{lang-en|'''Poison fishing'''}}）是使用有毒物品投入水中，鱼类被毒晕或毒死之后捕捞的一种捕鱼方法。<ref name="fishman" >{{cite web |url=http://ent.163.com/12/0806/17/8889PPOU00032DGD.html |title=电鱼与毒鱼 |accessdate= 2012|publisher= 网易|work= 金羊网|language=zh-hans}}</ref><ref>{{cite web |url=http://news.sohu.com/20071023/n252799095.shtml |title=桂林千米河面漂浮数万死鱼 疑为投毒药非法捕鱼 |accessdate=2007 |publisher=搜狐网 |work= |language=zh-hans}}</ref><ref>{{cite web|url=http://www.hq.xinhuanet.com/hihk/2005-05/11/content_4204439.htm|title=150艘船文昌投毒捕鱼 渔民机智擒获投毒人|accessdate=2005|publisher=新华网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20160304190600/http://www.hq.xinhuanet.com/hihk/2005-05/11/content_4204439.htm|archivedate=2016-03-04}}</ref>

==概述==
===危害===
毒鱼期间，不论大小鱼类，全部都难以逃脱被毒晕或毒死的结果，它导致鱼类[[不孕|不孕]]，鱼[[卵|卵]]、虾卵、泥鳅卵等都会被毒死，水中的[[无脊椎动物|无脊椎动物]]也受到伤害，毒鱼使得鱼类灭绝，生态系统受损，渔业资源枯竭。<ref name="fishman" />

人食用毒死的鱼类对人也有危险，残存鱼类体内的毒素会进入人体内，造成癌症等疾病。

毒鱼也影响[[水质|水质]]，毒素在水中使得人类和牲畜饮水不安全，毒死后的鱼类的尸体在水中[[腐烂|腐烂]][[变质|变质]]，造成水质的损坏，漂浮的尸体则被家禽、水鸟等食用，使得家禽、水鸟中毒而死。<ref name="fishman" />

在中国大陆农村，农民常常使用打虫的上海青等杀虫剂、蚊香、白酒搅拌在一起，然后撒入河流上游，河水把毒素往下游流，把毒晕的鱼类带到下游，人则站在下游的河中用抄网舀鱼。

==法律==
===中国大陆===
《[[中华人民共和国渔业法|中华人民共和国渔业法]]》第三十条规定：“禁止使用[[爆破捕鱼|炸鱼]]、毒鱼、[[电力捕鱼|电鱼]]等破坏渔业资源的方法进行捕捞”。<ref name="bull" >{{cite web |url= http://news.sohu.com/20050314/n224671457.shtml|title=电网捕鱼、殃及子孙 |accessdate=2005 |publisher=搜狐网 |work= |language=zh-hans}}</ref><ref name="bee" >{{cite web |url=http://www.qingdaonews.com/gb/content/2011-11/10/content_9003741.htm |title=浙江渔船来山东青岛捕鱼，所经之处鱼类灭绝 |accessdate=2011 |publisher=青岛新闻网 |work= |language=zh-hans}}</ref>

《[[中华人民共和国刑法|中华人民共和国刑法]]》：“对违反保护水产资源法规，在禁渔区、禁渔期或者使用禁用工具、方法捕捞水产品，情节严重的，处三年以下[[有期徒刑|有期徒刑]]、拘役、管制或罚金”。<ref name="mao" >{{cite web |url= http://www.xsnet.cn/xs/sh/2009/6/8/904243.shtml|title=电力捕鱼屡禁不止 |accessdate=2009 |publisher=萧山网 |work= |language=zh-hans}}</ref>
