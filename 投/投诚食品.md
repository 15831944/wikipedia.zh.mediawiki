'''投诚食品'''，是中华民国国军在2018年宣布在“反攻大陸”背景下的“心戰喊話”被取消後，台軍方引人注目的最新舉措。<ref>{{cite web |title=台灣軍方對外展示針對解放軍“投誠食品 ” |url=http://trad.cn.rfi.fr/%E4%B8%AD%E5%9C%8B/20190414-%E5%8F%B0%E7%81%A3%E8%BB%8D%E6%96%B9%E5%B0%8D%E5%A4%96%E5%B1%95%E7%A4%BA%E9%87%9D%E5%B0%8D%E8%A7%A3%E6%94%BE%E8%BB%8D%E6%8A%95%E8%AA%A0%E9%A3%9F%E5%93%81 |website=法廣 |language=zh-Hant |date=2019-04-14}}</ref>由[[中华民国国防部|中华民国国防部]]心战大队2019年4月14日在台北的文化营区与心战作业车一起展示。 <ref name="VOA"/>

“投诚食品”包括 “投诚[[方便面|方便面]]”，“投诚[[牛奶糖|牛奶糖]]”，“投诚[[朱古力|朱古力]]”等，包装上通常印有诸如“即刻享用， 安全与自由就在前方” ，以及“放下武器，双手高举”，“请保留此盒，向中华民国国军投诚”等字样，包装盒还印上最近的投降地点位置图，地点随部队驻地不同而改变。<ref name="VOA">{{cite web |title=台湾军方针对中国制作“投诚食品” |url=https://www.voachinese.com/a/Taiwan-military-using-instant-noodles-and-candies-for-psychological-warfare-20190414/4875256.html |website=美国之音 |language=zh}}</ref>

== 争议 ==
曾经担任两届立法委员的退役陆军中将[[帅化民|帅化民]]认为，现在，招降的投诚食品毫无必要，因为两岸生活差异不大，牛奶糖，方便面等心战食品根本没有投资效益。帅化民对美国之音说，台湾国防预算少，不能把国防经费虚掷在无效益的单位，他甚至主张裁掉国防部的心战单位。 <ref name="VOA"/>

在“投诚食品”见诸台湾媒体后，立即引来大陆网友的嘲笑，说“ 台军要拿一包方便面招降解放军？”、“我堂堂[[解放军|解放军]]差这一口方便面吗？”、“现在什么年代了，还用这招？”<ref name="VOA"/>台湾也有网友对此予以嘲讽，同时认为“投诚食品”是在“浪费钱”，“不知图利了哪家厂商”。<ref>{{cite news |title=台军秀“投诚方便面”欲招降解放军 网友热议 |url=http://news.dwnews.com/china/news/2019-04-14/60128833.html |accessdate=2019-04-15 |publisher=多维新闻网 |date=2019-04-14}}</ref>
