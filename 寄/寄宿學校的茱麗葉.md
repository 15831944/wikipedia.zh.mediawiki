== 故事簡介 ==
東和國與威斯特公國是兩個水火不容的國家，而在兩個國家的學生都就讀的達利亞學園中也不例外。其中住在黑犬寮的東和國男學生犬塚露壬雄卻對威斯特公國所住的白貓寮一年級的領導者茱麗葉·佩西亞產生了戀情，兩人都知道這段愛情不會被允許，因此必須過著偷偷摸摸的戀愛生活。

; {{nihongo|狗神玲音|狗神 玲音（いぬがみ れおん）}}

;  {{nihongo|犬塚朱奈|犬塚 朱奈（いぬづか しゅな）}}
: 黑犬寮的住宿生，達利亞學園高中從外部新編入的一年級女學生，犬塚家分家的少女，因家裡貧窮，被露壬雄的父親給收養，並且作為犬塚家的傭人，只要聽到有人對露壬雄口出惡言，便會痛揍一頓，被土佐稱為犬塚家最強的傭人，丸流稱之為看門狗，白貓之中有一條默認的規定「絕對不可以對犬塚朱奈出手」。

=== 白貓寮 ===
; {{nihongo|夏爾特琉·威士提亞|シャルトリュー・ウェスティア|聲：[[嶋村侑|嶋村侑]]<ref>{{Cite web|date=2018-06-08|url=https://natalie.mu/comic/news/285839/|title=アニメ「寄宿学校のジュリエット」“暴姫”シャル役は嶋村侑、デートボイスも|website=コミックナタリー|publisher=ナターシャ|accessdate=2018-06-08|archiveurl=https://web.archive.org/web/20180608095255/https://natalie.mu/comic/news/285839/|archivedate=2018-06-08}}</ref>}}
: 白貓寮的住宿生，初登場時為達利亞學園高中部一年級生，現為二年級生，與茱麗葉一樣文武雙全，擁有一雙藏著暗器的靴子，原本休學一年，看到了男裝的茱麗葉後，突然復學因為不想讓露壬雄靠近茱麗葉，而用他們暗中交往的秘密打壓露壬雄。是威斯特公國的第一王女，是個抖S的公主，擁有「暴姬」之名。茱麗葉從小的好友，喜歡茱麗葉，偷偷收藏大量茱麗葉從小到大的照片。是白貓寮唯一知道茱麗葉和露壬雄交往的人，十分討厭茱麗葉與露壬雄交往，但因為茱麗葉的關係默默支持著。另外來有飼養著一隻純白色的貓。

; {{nihongo|斯科特·佛爾德|スコット・フォールド|聲：[[神谷浩史|神谷浩史]]<ref>{{Cite web|date=2018-08-13|url=https://natalie.mu/comic/news/295197|title=アニメ「寄宿学校のジュリエット」ペルシアを敬愛する男子学生役に神谷浩史|website=コミックナタリー|publisher=ナターシャ|accessdate=2018-08-13|archiveurl=https://web.archive.org/web/20180813080752/https://natalie.mu/comic/news/295197|archivedate=2018-08-13}}</ref>、日高里菜（童年）}}
: 白貓寮的住宿生，初登場時為達利亞學園高中部一年級生，現為二年級生，雷克斯的雜務代理人，監督生候選人之一。對茱麗葉十分敬愛，時常被夏爾公主奴役。

; {{nihongo|阿比·西尼亞|アビ・シニア|聲：[[立花慎之介|立花慎之介]]<ref name="natalie180824">{{Cite web|date=2018-08-24|url=https://natalie.mu/comic/news/297096|title=アニメ「寄宿学校のジュリエット」怪力でおバカなソマリ役に喜多村英梨|website=コミックナタリー|publisher=ナターシャ|accessdate=2018-09-07|archiveurl=https://web.archive.org/web/20180907082031/https://natalie.mu/comic/news/296780|archivedate=2018-09-07}}</ref>}}
: 白貓寮的住宿生，初登場時為達利亞學園高中部二年級生，現為三年級生，凱特的雜務代理人，監督生候選人之一。與索瑪里是親密夥伴，和佩西亞在白貓派系裡是競爭關係。

; {{nihongo|索瑪里·朗海爾德|ソマリ・ロングヘアード|聲：[[喜多村英梨|喜多村英梨]]<ref name="natalie180824" />}}
: 白貓寮的住宿生，初登場時為達利亞學園高中部二年級生，現為三年級生。自稱是阿比的女朋友，十分聽阿比的話。力氣可以與露壬雄相提並論，曾經與斯科特競爭雷克斯的雜務代理人。

; {{nihongo|凱特·西|ケット・シィ|聲：[[興津和幸|興津和幸]]<ref>{{Cite web|date=2018-08-27|url=https://natalie.mu/comic/news/297096|title=アニメ「寄宿学校のジュリエット」下ネタ大好きなケット役に興津和幸|website=コミックナタリー|publisher=ナターシャ|accessdate=2018-09-07|archiveurl=https://web.archive.org/web/20180907081955/https://natalie.mu/comic/news/297096|archivedate=2018-09-07}}</ref>}}
: 白貓寮的監督生代表，達利亞學園高中部三年級男學生，雖是貴族出身卻選了勞動階級出身的阿比做為雜務代理人。總是顯得超然於世，舉手投足間充滿了自信和遊刃有餘，但卻常常做出低級的舉動。對賽貝爾有友情以上的情感，在監督生選舉大會上因賽貝爾被攻擊受傷而憤怒暴走，與前來阻止的藍瑠打在一起。

; {{nihongo|安·塞貝爾|アン・サイベル|聲：[[上坂堇|上坂堇]]<ref>{{Cite web|date=2018-08-29|url=https://natalie.mu/comic/news/297424|title=「寄宿学校のジュリエット」ツインテールのクールな監督生、サイベル役に上坂すみれ|website=コミックナタリー|publisher=ナターシャ|accessdate=2018-09-07|archiveurl=https://web.archive.org/web/20180907081859/https://natalie.mu/comic/news/297424|archivedate=2018-09-07}}</ref>}}
: 白貓寮的監督生，初登場時為達利亞學園高中部二年級生，現為三年級生，茱麗葉的指導者。性格認真、紀律至上，重視校內、寮內的紀律勝過其他任何事，但是實際上非常喜歡可愛的東西，曾被露壬雄看見穿著小熊內褲，似乎覺得胡蝶姊妹很萌。對凱特非常毒舌，常常讓凱特去死，在監督生選舉大會上因玲音的計謀受傷，導致凱特憤怒暴走。

; {{nihongo|傑尼·雷克斯|ジャーニー・レックス|聲：[[星野貴紀|星野貴紀]]<ref>{{Cite web|date=2018-08-31|url=https://natalie.mu/comic/news/297713|title=「寄宿学校のジュリエット」かわいさとワイルドさを両立、レックス役に星野貴紀|website=コミックナタリー|publisher=ナターシャ|accessdate=2018-09-07|archiveurl=https://web.archive.org/web/20180907081739/https://natalie.mu/comic/news/297713|archivedate=2018-09-07}}</ref>}}
: 白貓寮的監督生，初登場時為達利亞學園高中部二年級生，現為三年級生，斯科特的指導者。性格光明正大、質樸剛健，追求可愛與狂野並存的女裝男子漢，曾敗在丸流的傲嬌屬性下，要求斯科特和自己打扮得一樣。

;  {{nihongo|阿梅莉亞·卡爾|アメリア・カール}}
: 白貓寮的住宿生，達利亞學園高中部一年級新生領導人，白貓寮舍監的女兒，曾因為覺得茱麗葉與自己的母親一樣，把規矩掛在嘴邊而討厭茱麗葉。
