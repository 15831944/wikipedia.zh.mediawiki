'''军委办事组'''是[[文化大革命|文化大革命]]前期主持[[中共中央军事委员会|中共中央军事委员会]]日常工作的机构。

==沿革==
===背景===
'''军委办事组'''是在1967年批判[[二月逆流|二月逆流]]、[[武汉|武汉]][[七二〇事件|七二〇事件]]以及打倒[[中央军委常务委员会|中央军委常委]]、[[中国人民解放军总政治部|中国人民解放军总政治部]]主任[[萧华|萧华]]的背景下成立。1980年9月23日，《[[解放军报|解放军报]]》原总编辑[[赵易亚|赵易亚]]在一份揭发[[林彪|林彪]]、“[[四人帮|四人帮]]”罪行的材料里称：“1967年7月25日，……去[[天安门|天安门]]参加天安门大会。……[[李讷|肖力]]就把造反突击队员一一介绍给[[林彪|林彪]]。上台阶不久，林问肖力，你们的组织叫什么名字。肖答：叫造反突击队。林说：‘对，要造反，要突击，要彻底砸烂[[中国人民解放军总政治部|总政]]阎王殿！’……在吃饭前后，她（指[[叶群|叶群]]）还对我说了[[萧华|萧华]]同志很多坏话。并说，今天本来没有通知萧上天安门（7月23日[[北京西郊机场|西郊机场]]迎接[[谢富治|谢富治]]、[[王力_(政治人物)|王力]]就未让萧去），后来他闹了很久才让他上去。……”这段文字里，肖力是[[毛泽东|毛泽东]]、[[江青|江青]]夫妇的女儿[[李讷|李讷]]在解放军报社的化名，“造反突击队”是文革中解放军报社内以肖力为首的一个造反组织<ref name=wny>{{Cite web |url =http://view.news.qq.com/a/20140429/006432.htm  |title = 王年一：关于“军委办事组”的一些资料 |author =  |date = 2014-04-29 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。

1967年10月上旬，在解放军报社三楼肖力的办公室里，肖力向中国人民解放军总政治部卢前安、魏建群、杜嘉等7人传达了林彪的上述“要彻底砸烂总政阎王殿”的指示。（杜嘉后来回忆说，这7人均为总政斗批筹备处勤务组的办事组成员）。次日，卢前安在总政礼堂举行的大会上向总政治部人员传达了该指示。在此前后，[[北京市|北京市]]街头已贴满“打倒萧华”的标语和[[大字报|大字报]]（其中有些是批判萧华有生活作风问题），总政治部机关已经难以运转<ref name=wny/>。

1968年10月30日，江青在[[中共八届十二中全会|中共八届十二中全会]]上说：“军委办事组是怎么产生的呢？1967年夏天，斗争萧华的时候，出现了不严肃的现象，把一场严肃的[[阶级斗争|阶级斗争]]，变成低级下流，转移了斗争大方向，侮辱了[[中国人民解放军|中国人民解放军]]。因此，我建议成立一个看管小组。开始是4个人。后来……在军队产生了新的办事班子——军委办事组。[[杨、余、傅事件|杨、余、傅问题]]出现后，进行了改组。”（注：军委办事组起初又称'''军委看守小组'''）<ref name=wny/>

1968年6月11日，[[吴法宪|吴法宪]]在接见军委办事组全体工作人员时回忆称：“原来没有个办事组。文革高潮的时候，‘[[七二〇事件|七二〇事件]]’[[武汉|武汉]]发生了[[陈再道|陈再道]]的问题。那时王力、[[关锋|关锋]]、[[戚本禹|戚本禹]]勾结军队内部[[揪军内一小撮|揪军内一小撮]]，抢枪，冲击军队。那时[[毛主席|毛主席]]还在外地视察。[[杨成武|杨成武]]回来后，成立了军委办事组，杨担任组长。后来又相继搞了政工组、军报组、文艺组。[[中央军委办公厅|军委办公厅]]是军委的办事机构。[[中央军委常务委员会|军委常委]]、军委办事组的事情都通过那里办。军委办事组的问题经[[中央军委办公厅|办公厅]]提交军委常委讨论。……”<ref name=wny/>

===成立===
1967年8月17日，[[周恩来|周恩来]]和[[中央文革小组|中央文革小组]]指示，由吴法宪、[[叶群|叶群]]、[[邱会作|邱会作]]、[[张秀川|张秀川]]4人组成一个小组，由吴法宪负责，负责驻[[北京|京]]各机关、部队[[无产阶级文化大革命|无产阶级文化大革命]]的工作，看着中国人民解放军总政治部机关的文化大革命运动不要出偏差。1967年9月24日，周恩来和中央文革小组又决定增加杨成武为办事组成员，并指定杨成武为组长，吴法宪为副组长<ref name=wny/>。

1967年10月4日，军委办事组研究了如何开展工作的问题，认为由于总政治部及[[全军文化革命小组|军委文革小组]]实际上处于瘫痪状态，军委办事组是在中央、中央军委、中央文革小组直接领导下，负责处理军队系统驻京机关、部队无产阶级文化大革命方面的具体工作；各[[大军区|大军区]]、省军区的无产阶级文化大革命工作是由中央文革小组直接领导，军委办事组负责办理[[毛泽东|毛泽东]]（[[中共中央主席|中共中央主席]]）、[[林彪|林彪]]（[[中共中央副主席|中共中央副主席]]）、中央文革小组交办的具体工作；而部队的建设、战备工作、干部工作、行政工作等均由[[中央军委常务委员会|军委常委]]直接领导，提到军委办事组来的此类问题均呈送军委常委处理<ref name=wny/>。

1967年10月15日，军委办事组致周恩来、[[陈伯达|陈伯达]]、[[康生|康生]]、江青并中央文革小组的报告回顾了自1967年8月17日以来的上述情况，并称：“军委办事组除设有几个联络员、几个秘书外，还有一个负责对‘[[五一六反革命阴谋集团|五一六兵团]]’调查工作的调查组，和一个负责干部学习班具体工作的办公室。”（“干部学习班”是指当时中央办的一个干部学习班，由林彪主持，抽调各省、自治区、直辖市的军、政负责人来北京学习，以便解决各省、自治区、直辖市的文革问题。）“军委办事组是个临时性的，待政治工作组组成后，即将办事组的工作移交政工组负责，军委办事组即行撤销。”<ref name=wny/>

就军委办事组的该报告，1967年10月16日，[[中华人民共和国国务院总理|国务院总理]][[周恩来|周恩来]]批示：“提议在小组会议上讨论一次，再正式报告主席、林副主席。”1967年10月20日，康生批示：“此事必须先请示林彪同志，在林彪同志同意后再正式上报主席。1967年10月27日，杨成武、吴法宪报告林彪称：“关于军委办事组问题，向总理、伯达、康生、江青同志并中央文革小组写了一个报告，遵康生同志的指导先报您批示后，报主席批示。现将报告呈上，请阅批。”林彪批示：“呈主席批示。”1967年10月28日，[[中共中央主席|中共中央主席]][[毛泽东|毛泽东]]批示：“刚成立又取消，恐怕不妥。”1967年10月29日，林办（林彪办公室）批：“林彪同志嘱：送总理、伯达、康生、江青、成武、[[张春桥|春桥]]、[[姚文元|文元]]、法宪同志阅，退杨成武同志。”同日，周恩来批示：“我仍建议，此事应先在中央文革扩大会上讨论一次。请伯达同于30日例会上提出。”<ref name=wny/>

===改组===
1968年3月，“[[杨、余、傅事件|杨、余、傅事件]]”发生，兼任军委办事组组长的杨成武倒台。1968年3月25日，周恩来、陈伯达、康生、江青报告林彪称：“中央文革碰头会议讨论过新的军委办事组名单，拟了5个同志，现先送上，请考虑是否妥当，并请在您考虑后，向主席报告请示。名单：[[黄永胜|黄永胜]]组长，吴法宪副组长，叶群、李作鹏、邱会作。”同日，林彪批示：“呈主席批示。”毛泽东批示：“照办。”<ref name=wny/>

1968年3月28日，毛泽东、林彪、周恩来接见黄永胜、吴法宪、[[温玉成|温玉成]]。毛泽东问军委办事组现在有哪几人，黄永胜等人回答之后，毛泽东表示：还可以增加[[刘贤权|刘贤权]]，政工小组可以考虑取消，今后军委办事组由林彪直接捏到手里。毛泽东还说：“军委办事组要订个制度，至少一周到林彪同志那里汇报一次工作，一次谈一两个钟点。有事无事都要去，除非林彪同志身体不好。过去我们两个处在第二线，让他们去搞，他们也不汇报，搞封锁，实行隔离，隔离不反省。”林彪说：“不了解情况。”周恩来说：“军委常委不要开会了吧？”毛泽东说：“军委就是办事组。军委常委可以不开会了。”林彪说：“把它悬空起来。”毛泽东说：叶剑英说过，杨成武是接班人。现在换了黄永胜，你这个黄永胜为什么不可以接班？”<ref name=wny/>

1968年3月30日，吴法宪在军委办事组会上表示：“经主席、林副主席、中央文革批准，改组办事组，由总部、军种主要负责人组成，在主席、林副主席、中央文革直接领导下主持军队工作。办事组成员由[[黄永胜|黄]]、[[叶群|叶]]、[[李作鹏|李]]、[[邱会作|邱]]和[[吴法宪|吴]]5个人组成，将来还可以加一个刘贤权同志，黄为组长，吴为副组长。……[[李天焕|李天焕]]、[[刘锦平|刘锦平]]不参加办事组。……”<ref name=wny/>

1968年4月1日，黄永胜、吴法宪写给毛泽东、林彪、周恩来、陈伯达、康生、江青、[[姚文元|姚文元]]、[[谢富治|谢富治]]、叶群、[[汪东兴|汪东兴]]的报告称：因军委常委不再开会，军队文电今后除了发至团级文电照发各位常委外，重要文电不再抄送[[陈毅|陈毅]]、[[刘伯承|刘伯承]]、[[徐向前|徐向前]]、[[聂荣臻|聂荣臻]]、[[叶剑英|叶剑英]]等同志，[[粟裕|粟裕]]同志与工作无关文电亦不抄送。毛泽东批示：“此事待议。”<ref name=wny/>

1968年4月4日，黄永胜、吴法宪、李作鹏、邱会作将军委印章收缴。1968年4月4日，李天佑通过书面形式向黄永胜报告：文件已立即停送徐向前、叶剑英、刘伯承、聂荣臻<ref name=wny/>。

1968年6月11日晚，黄永胜、吴法宪接见军委办事组全体工作人员。黄永胜表示，军委办事组分为两个阶段，以前由杨成武任组长，揭杨（揭批杨成武）之后军委办事组改组，增加了李作鹏、刘贤权和黄永胜，加上原先的吴、叶、邱，共计6人。吴法宪说：“中央决定，军委停止工作，军委常委工作实际上已集中到办事组来了，起到过去[[中央军委办公会议|军委办公会议]]的作用。在主席、林副主席领导下，直接对主席、林副主席、中央文革负责。黄总长来了以后，权力集中了，实际上代替过去军委办公会议的职权。”<ref name=wny/>

===掺沙子===
1968年11月18日，黄永胜致毛泽东、林彪、周恩来、陈伯达、康生、江青、姚文元、谢富治、吴法宪、叶群、汪东兴、温玉成的报告称：“主席、林副主席：中央文革确定增加谢富治、温玉成两同志为办事组成员，我们非常欢迎。还要求增加[[李天佑|李天佑]]一人。……”1968年11月22日，陈伯达、姚文元批示：“是否等几天之后，征求一下江青同志的意见。”1968年11月24日，黄永胜批：“完全同意文元同志意见。”<ref name=wny/>

林彪成为写进《[[中国共产党章程|中国共产党章程]]》里的 “接班人”后，毛泽东考虑用“掺沙子”的办法抑制林彪势力的发展<ref name=qd/>。1969年4月27日，军委办事组拟了份“军委48人名单”及“军委办事组名单”，报毛泽东主席、林彪副主席。毛泽东批示：“宜加[[李德生|李德生]]。”林彪批示：“同意主席批示。”1969年5月27日，《中共中央通知》公布了[[中共九届一中全会|中共九届一中全会]]通过的军委主席、副主席、委员名单以及军委办事组成员名单。军委办事组组长黄永胜，副组长吴法宪，组员（以姓氏笔划为序）：叶群、刘贤权、李天佑、李作鹏、李德生、邱会作、温玉成、谢富治<ref name=wny/>。当时，李德生是从[[南京军区|南京军区]]副司令员、[[中共安徽省委|中共安徽省委]]第一书记的任上被调任中国人民解放军总政治部主任，参加[[国务院业务组|国务院业务组]]和军委办事组工作。这时军委办事组共10人，包括林彪的夫人叶群以及7名林彪[[第四野战军|第四野战军]]的老部下，只有李德生、谢富治原是[[第二野战军|第二野战军]]的<ref name=qd/>。

1970年夏，在[[中共九届二中全会|庐山会议]]上，毛泽东和林彪的关系陷入紧张。1970年12月，毛泽东会见[[美国|美国]]记者[[埃德加·斯诺|埃德加·斯诺]]时说：“什么‘[[四个伟大|四个伟大]]’，讨嫌！ ”并将该讲话批给全党阅。1971年1月，中共中央宣布改组[[北京军区|北京军区]]。此后，毛泽东两度批评军委办事组不争取主动，一错再错。1971年4月，毛泽东派[[纪登奎|纪登奎]]、[[张才千|张才千]]参加军委办事组，这是继1969年4月派李德生参加军委办事组之后，毛泽东再次向军委办事组“掺沙子”。同时，毛泽东表示可以让[[李先念|李先念]]参加军委办事组，并且责问为何不给陈毅看文件<ref name=qd/>。

===撤销===
1971年[[九一三事件|九一三事件]]后，1971年9月29日，中共中央发出通知：黄永胜、吴法宪、李作鹏、邱会作4人离职反省，中央已决定由军委副主席叶剑英主持军委日常工作，并筹组军委办公会议，集体领导<ref name=wny/>。

1971年10月3日，中共中央发出通知：“中央决定，撤销军委办事组，成立军委办公会议。军委办公会议由军委副主席叶剑英同志主持，并由叶剑英、谢富治、[[张春桥|张春桥]]、[[李先念|李先念]]、李德生、[[纪登奎|纪登奎]]、[[汪东兴|汪东兴]]、[[陈士榘|陈士榘]]、[[张才千|张才千]]、刘贤权10同志组成，即日成立，在中央军委领导下负责军委日常工作，特此通知。”<ref name=wny/>
