'''军队和武警部队全面停止有偿服务工作领导小组'''，是[[中央军委|中央军委]]议事协调机构，负责领导[[军队和武警部队全面停止有偿服务|军队和武警部队全面停止有偿服务]]工作。

==沿革==
2009年11月，经中央军委批准，四总部研究制定的《关于加强军队单位对外有偿服务管理若干问题的意见》下发全军和武警部队贯彻执行。《通知》中规定，严格落实管理责任，成立'''全军对外有偿服务管理工作领导小组'''，协调建立军地共管工作机制<ref>{{Cite web |url = http://www.gov.cn/jrzg/2009-11/13/content_1463432.htm |title = 解放军四总部下发意见加强军队对外有偿服务管理 |author =  |date =2009-11-13  |publisher =  中国政府网|language =  |accessdate =  |quote =  }}</ref>。

2010年7月21日，全军对外有偿服务管理工作领导小组第一次会议在北京召开，中央军委委员、[[中国人民解放军总后勤部|中国人民解放军总后勤部]]部长[[廖锡龙|廖锡龙]]在会上讲话<ref>{{Cite web |url =  http://mil.news.sina.com.cn/2010-07-22/0633601874.html|title = 廖锡龙在全军对外有偿服务管理工作领导小组会议上要求 压缩规模控制范围从严管理 确保对外有偿服务健康开展 |author =  |date =  2010-07-22|publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

2016年3月，中央军委印发《关于军队和武警部队全面停止有偿服务活动的通知》，通知提出，中央军委计划用3年左右时间，分步骤停止军队和武警部队一切有偿服务活动<ref name=yibu>{{Cite web |url = http://news.ifeng.com/a/20160507/48716616_0.shtml |title =军队全面停止有偿服务走到了哪一步？  |author =  |date = 2016-05-07 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

2016年5月7日，军队和武警部队全面停止有偿服务试点任务部署会议召开，'''军队和武警部队全面停止有偿服务工作领导小组'''成员出席了此次部署会议。这是首次公开报道该领导小组的活动<ref name=yibu/>。
