'''陽江話'''，屬[[粵語方言#高陽粵語|高陽片]][[粵語|粵語]]，通行於[[阳江市|陽江]]大部分地區。陽江話的前身與[[客家语|客語]]類似，來自古老的[[中原|中原]]，且陽江話同時融合了[[南越部落|古南越俚人]]（原住民）的語言特點。這造成了陽江話基本無法與[[客家语|客語]]及其他[[粵語|粵語]]（包括[[广州话|廣州話]]）之使用者互相通話的現狀。但由於[[阳江市|陽江]]地處粵語圈中，詞彙上仍深受[[广州话|廣州話]]的影響。<ref name="where-come-from">{{Cite web|url=http://blog.sina.com.cn/s/blog_4d4883eb0100dzms.html|title=阳江话从哪里来|accessdate=2018-12-27|author=叶柏来|date=2009-06-05|publisher=}}</ref>

陽江話相比起[[广州话|廣州話]]，保留了更多[[粵語音系學#歷史變化|古粵語]]的特徵。[[广州话|廣州話]]中自1950年代起已不再區分的 [[清齒齦擦音#清齒齦有噝擦音|[s]]]、[[清龈颚有咝擦音|[ɕ]]] 仍能與今日之陽江話中 [[清齒齦邊擦音|[ɬ]]]、[[清颚龈有咝擦音|[ʃ]]] 所對應（如「相」與「傷」）；[[广州话|廣州話]]中 [[齒齦鼻音|[n]]] 與 [[齿龈边近音|[l]]] 不分的現象在陽江話中不存在（如「你」與「李」）；[[广州话|廣州話]]中 [[軟顎鼻音|[ŋ]]] 與[[空韻|零聲母]]交替的現象在陽江話中不存在（如「餓」與「澳」）；一些已被[[广州话|廣州話]]拋棄的詞彙在[[阳江市|陽江]]仍被日常所廣泛使用（如舊粵語中表示蜘蛛的詞彙「[[:zh-yue:蠄蟧|蠄蟧]]」）；[[广州话|廣州話]]中裏已沒有濁輔音，但陽江話裏仍然存在，但與[[中古汉语|中古漢語]]的[[全浊声母|全濁聲母]]沒有關係。

== 歷史 ==
約在[[東漢末年|漢末]]、[[三国|三國]]、[[晋朝|兩晉]]時期，[[中原|中原]]仍戰亂不斷。當時，一支龐大且富有的古[[中原|中原]][[汉族|漢族]]爲了躲避戰亂，移民到了[[阳江市|陽江]]。當時的[[阳江市|陽江]]人口稀少，這批移民與作爲[[原住民|原住民]]的[[南越部落|古南越俚人]]沒有土地上的爭端，而是共謀發展利益；[[南越部落|俚人]]的生產技術並不發達，而帶來先進生產技術的[[汉族|漢人]]極大地推動了當地的經濟發展，因此受到了作爲[[原住民|原住民]]的[[南越部落|俚人]]的歡迎，從而帶動了[[南越部落|俚人]]的漢化，[[南越部落|俚人]]與[[汉族|漢人]]的語言逐漸融合。由於[[汉族|漢人]]衆多、文明程度高，且處於本地經濟的核心位置，[[中原|中原]]語言成爲了語言融合中的主導，而[[南越部落|俚人]]的一些語言特點也被保留。現今[[阳江市|陽江城]]內那些被命名爲「那、洞、麻」的地方以及陽江話中存在的邊擦音聲母 [[清齒齦邊擦音|[ɬ]]]（此聲母亦普遍存在於[[壮语|壯語]]、[[黎语|黎語]]之中）體現了[[南越部落|古南越俚人]]之語言對陽江話的影響。<ref name="where-come-from"></ref>

# 陽江話中「知」、「痴」、「屍」的[[声母|聲母]]有被視作[[舌葉音|舌葉音]]（而非[[普通话|普通話]]中[[龈腭音|舌面音]] [[清齦顎塞擦音|[tɕ]]]、[tɕʰ]、[[清齦顎擦音|[ɕ]]]）的說法<ref name="2k" /><ref name="twenty-thirteen" />，但在葉柏來的《陽江音字典》中仍被稱爲[[龈腭音|舌面音]]（儘管書中有註明這些[[声母|聲母]]「與[[汉语拼音#声母|普通話聲母]] j、q、x 不同」）<ref name="2k-four"></ref>；
# 陽江話中「窩」的[[声母|聲母]]曾被標註爲 [[濁圓唇軟顎近音|[w]]]。2000年的《陽江縣誌》中記載「[[阳江市|陽江]]人發 [[濁圓唇軟顎近音|[w]]] 時雙唇收攏得不夠圓，有點象發 [[濁唇齒擦音|[v]]] 音」，但以「從舌根抬起和從古音及其他方音比較的角度看」爲由，選擇不標註爲 [[濁唇齒擦音|[v]]]<ref name="2k" />。這兩句在2013年的《江城區誌》已被刪去，同時聲母表中「窩」的聲母亦改爲 [[濁唇齒擦音|[v]]]<ref name="twenty-thirteen" />；
# 陽江話中「瓜」與「夸」的[[声母|聲母]]向來被當作[[聲介合母|聲介合母]] [kʷ] 與 [kʷʰ] 處理<ref name="2k" /><ref name="twenty-thirteen" /><ref name="2k-four" />。2000年的《陽江縣誌》中記載「有點象 [kv]、[kʻv]（[kʰv]），嘴唇不收圓，但唇齒間摩擦現象不很明顯」，同時以「無[[复辅音|複輔音]]是[[汉语|漢語]]的特點」爲由，選擇不標註爲 [kv] 與 [kʰv]<ref name="2k" />。這兩句在2013年的《江城區誌》已被刪去，但依然聲母表中依舊標註爲 [kʷ] 與 [kʷʰ]<ref name="twenty-thirteen" />。葉柏來在《陽江音字典》註明「[[濁唇齒擦音|[v]]] 的摩擦成分很低」，但認爲它們是[[复辅音|雙輔音（複輔音）]][[声母|聲母]] [kv] 與 [kʰv]<ref name="2k-four"></ref>；
# 1960年代開始，陽江的年青人漸漸把「瓜」與「夸」的聲母唸成 [[清軟顎塞音|[k]]] 與 [[清軟顎塞音|[kʰ]]]，「瓜」與「夸」的聲母逐漸被歸併爲 [[清軟顎塞音|[k]]] 與 [[清軟顎塞音|[kʰ]]]。這是1960年代以後陽江話在語音方面最大亦較特殊的變化<ref name="2k" /><ref name="twenty-thirteen" />；
# 在陽江話中，當[[齊齒呼|齊齒呼]][[韵母|韻母]] [iŋ] 和 [ik] 自成[[音节|音節]]時，開頭會帶有[[聲門塞音|喉塞音 [ʔ]]]，如「英 [ʔiŋ]」、「益 [ʔik]」。此現象在2000年的《陽江縣誌》被簡化成[[空韻|零聲母]]<ref name="2k" />，在2013年的《江城區誌》被簡便處理作 [[硬顎近音|[j]]]<ref name="twenty-thirteen" />，但在葉柏來的《陽江音字典》中沒有被提及；
# 關於陽江話中 [[清声门擦音|[h]]] 的[[调音部位|發聲部位]]，有[[舌根音|舌根音]]<ref name="2k-four"></ref>和[[聲門音|喉音]]<ref name="2k"></ref><ref name="twenty-thirteen"></ref>兩種說法；
# 關於陽江話中 [[軟顎鼻音|[ŋ]]] 的[[调音部位|發聲部位]]，有[[舌根音|舌根音]]<ref name="2k-four"></ref>和舌根圓唇音<ref name="2k"></ref><ref name="twenty-thirteen"></ref>兩種說法；

# 陽江話中「奧」與「非」的[[韵母|韻母]]在2000年的《陽江縣誌》中分別被標註爲 [ou] 與 [ei]，書中亦註明「[o] 衹在 [ou] 一個韻母中出現」、「[e] 在韻母 [ei] 中（出現）」<ref name="2k" />。這些註明在2013年的《江城區誌》已被刪去，「奧」與「非」的[[韵母|韻母]]亦分別改爲標註 [ɔu] 與 [εi]<ref name="twenty-thirteen" />；
# 在陽江話的實際語音中，[[開口呼|開口呼]][[韵母|韻母]]的「-t」與「-g」[[韵尾|韻尾]]一般人是不作區分的，而多唸成「-g」<ref name="2k-four" />；
# 陽江話中「英」與「億」的[[韵母|韻母]]因受[[韵尾|韻尾]]的影響，實際發音是 [eŋ] 與 [ek]，且唸得較短<ref name="2k" /><ref name="2k-four" />；
# 陽江話中「瓮」與「屋」的[[韵母|韻母]]實際發音是 [oŋ] 與 [ok]，口腔都會張得比較開<ref name="2k" />；
# 上表中「ɛ類」[[韵母|韻母]]（[εi] 除外）與[[声母|聲母]]拼合時，有輕微（且短<ref name="2k" />）的流音 [[閉前不圓唇元音|[i]]]，如「削 [ɬiɛk]」、「姜 [giɛŋ]」<ref name="2k-four" />；
# 不少陽江人講話時不太能分清楚[[齒齦鼻音|前鼻韻母]]和[[軟顎鼻音|後鼻韻母]]，如 [ɔn] 與 [ɔŋ] 不分、把「民」唸成「盟」等。[[塘坪镇|塘坪]]、[[双捷镇|雙捷]]一帶的人甚至習慣將 [iŋ] 韻母的字唸成 [iɛn] 韻母的字<ref name="2k-four" />；
# 除上述47個[[韵母|韻母]]外，陽江話還有 [m] 和 [ŋ] 兩個[[韵母|韻母]]，它們既可作[[鼻音_(辅音)|鼻]]韻母的[[韵尾|韻尾]]，又可自成[[音节|音節]]，但都不能跟[[声母|聲母]]拼合成[[音节|音節]]<ref name="twenty-thirteen" />，衹作[[感叹词|嘆詞]]用<ref name="2k" />，如「嘸 [m]」、「唅 [hm]」、「唔 [ŋ]」、「啈 [hŋ]」。
