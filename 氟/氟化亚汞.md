'''氟化亚汞'''是[[汞|汞]]元素与[[氟|氟]]元素形成的[[无机化合物|无机化合物]]，化学式为Hg<sub>2</sub>F<sub>2</sub>。它是一种黄色[[立方晶系|立方]]晶体，受到光线照射时会变黑。<ref name="hand"/>

==反应==
氟化亚汞可由[[碳酸亚汞|碳酸亚汞]]与[[氢氟酸|氢氟酸]]反应制得。它与水接触立即水解，产生液态的[[汞|汞]]、[[氧化汞|氧化汞]]和[[氢氟酸|氢氟酸]]。<ref name="hand"/>
:Hg<sub>2</sub>CO<sub>3</sub> + 2 HF → Hg<sub>2</sub>F<sub>2</sub> + CO<sub>2</sub>↑ + H<sub>2</sub>O
:Hg<sub>2</sub>F<sub>2</sub> + H<sub>2</sub>O → Hg + HgO + 2HF

它还可由[[氟化钠|氟化钠]]与[[硝酸亚汞|硝酸亚汞]]反应，或[[氯化亚汞|氯化亚汞]]与[[氟化钠|氟化钠]]反应，或用稀氢氟酸溶解[[氧化亚汞|氧化亚汞]]制得。<ref>{{Greenwood&Earnshaw}}</ref>
:2NaF + Hg<sub>2</sub>(NO<sub>3</sub>)<sub>2</sub> → Hg<sub>2</sub>F<sub>2</sub> + 2NaNO<sub>3</sub>
:2NaF + Hg<sub>2</sub>Cl<sub>2</sub> → Hg<sub>2</sub>F<sub>2</sub> + 2NaCl
:Hg<sub>2</sub>O + 2HF → Hg<sub>2</sub>F<sub>2</sub> + H<sub>2</sub>O

氟化亚汞是一种较弱的氟化剂，它的反应活性与[[氟化银|氟化银]]接近，进行氟化反应时需要存在碘单质以进行[[催化|催化]]。它在[[Swarts反应|Swarts反应]]中可用作将[[卤代烷|卤代烷]]转化为氟代烷：<ref>{{cite journal|doi=10.1021/ja01272a021|year=1938|last1=Henne|first1=Albert L.|last2=Renoll|first2=Mary W.|journal=Journal of the American Chemical Society|volume=60|issue=5|pages=1060}}</ref><ref name="org">

==结构==
与其他含有直线型X-Hg-Hg-X单元的亚汞化合物一样，Hg<sub>2</sub>F<sub>2</sub>含有直线型的FHg<sub>2</sub>F单元，Hg-Hg键的键长为251 pm（Hg-Hg键在金属中为300 pm），Hg-F键的键长则为214 pm。<ref>{{cite journal|doi=10.1021/ja00543a001|title=Molecular structure of mercurous halides: Mercurous fluoride and mercurous chloride|year=1980|last1=Kleier|first1=Daniel A.|last2=Wadt|first2=Williard R.|journal=Journal of the American Chemical Society|volume=102|issue=23|pages=6909}}</ref><ref name = "Wells"> Wells A.F. (1984) ''Structural Inorganic Chemistry'' 5th edition Oxford Science Publications ISBN 0-19-855370-6 </ref>每个汞原子总体的配位几何是八面体的，因为它除了最近的氟原子以外，还与其他四个氟原子相距272 pm。<ref name = "Wells"/>这种化合物的化学式通常写成[Hg<sub>2</sub><sup>2+</sup>]·2F<sup>−</sup>。<ref name = "Cotton">{{Cotton&Wilkinson6th}}</ref>
