'''氟溴甲烷'''是一种混合[[卤代甲烷|卤代甲烷]]，可溶于[[乙醇|乙醇]]，易溶于[[氯仿|氯仿]]。

其[[标准摩尔熵|标准摩尔熵]]（''S''<sup><s>o</s></sup><sub>gas</sub>）为276.3 J/(mol·K)，[[热容|热容]]（''c<sub>p</sub>''）为49.2 J/(mol·K)。

== 制备 ==
氟溴甲烷可由[[氟甲烷|氟甲烷]]的溴化反应制备：<ref name=Bai>白占旗, 丁元胜, 黄瑛. [http://www.ixueshu.com/document/2b7c79209d63da80.html 氟溴甲烷的制备及应用][J]. 浙江化工, 2005, 36(9):30-31.</ref>
: CH<sub>3</sub>F + [[溴|Br<sub>2</sub>]] → CH<sub>2</sub>BrF + HBr
[[二溴氟甲烷|二溴氟甲烷]]的电解还原，或者通过和[[还原剂|还原剂]][[钠汞齐|钠汞齐]]或[[氢化三丁基锡|氢化三丁基锡]]的反应也能得到氟溴甲烷。

[[氟乙酸银|氟乙酸银]]和[[溴|溴]]在50~120°C反应，可以得到氟溴甲烷：<ref name=Bai />
: FCH<sub>2</sub>COOAg + Br<sub>2</sub> → CH<sub>2</sub>FBr + AgBr + CO<sub>2</sub>

[[二溴甲烷|二溴甲烷]]和[[氟化亚汞|氟化亚汞]]反应，也能得到氟溴甲烷：<ref name=Organikis>Ασκήσεις και προβλήματα Οργανικής Χημείας Ν. Α. Πετάση 1982, pp 185, §7.2.8.；pp 186, §7.3.1.</ref>
: 2 CH<sub>2</sub>Br<sub>2</sub> + Hg<sub>2</sub>F<sub>2</sub> → 2 CH<sub>2</sub>FBr + Hg<sub>2</sub>Br<sub>2</sub>

==化学性质==
氟溴甲烷可以发生[[取代反应|取代反应]]：<ref name=Organikis />
:<math>\mathrm{CH_2FBr + NaCN \xrightarrow{-NaBr} FCH_2CN \xrightarrow{+ NaCN} NCCH_2CN + NaF} </math></div>
== 应用 ==
氟溴甲烷是中间体、医药和其它化学品生产过程中的重要试剂。由于[[臭氧破坏潜势|臭氧破坏潜势]]达0.73，其使用收到限制。它的同位素标记化合物CH<sub>2</sub>Br<sup>18</sup>F可用于放射化学。
