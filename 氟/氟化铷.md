'''氟化銣'''（RbF）是[[銣|銣]]的[[氟化物|氟化物]]。它是[[八面體|八面體]]配位晶体。

氟化銣有幾種合成方法。一個是用[[氫氧化銣|氫氧化銣]]與[[氫氟酸|氫氟酸]]混合反應：<ref name="www.webelements.com">{{cite web | url = http://www.webelements.com/webelements/compounds/text/Rb/F1Rb1-13446747.html | title = WebElements |accessdate=23 February 2006}}</ref>

: RbOH + HF → RbF + H<sub>2</sub>O

另一個方法是以氫氟酸[[中和反应|中和]][[碳酸銣|碳酸銣]]：<ref name="www.webelements.com" />

: Rb<sub>2</sub>CO<sub>3</sub> + 2HF → 2RbF + H<sub>2</sub>O + CO<sub>2</sub>

還有一種方法是用銣金屬直接與氟氣反應，但因為銣金屬很昂貴，所以是最不常用的方法。此外，銣與鹵素會產生劇烈反應並燃燒：<ref name="www.webelements.com" />

: 2Rb + F<sub>2</sub> → 2RbF
