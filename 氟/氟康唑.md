'''氟康唑'''（{{lang-en|'''Fluconazole'''}}）是一种[[三唑|三唑]]，可用于醫治由[[真菌|真菌]]引起的病徵。常见的商品名称有大扶康（Diflucan）或“Trican”（[[輝瑞|輝瑞]]）。在墨西哥的药店则以“Alfumet”为名销售。

== 藥理 ==
氟康唑是抗真菌藥劑﹙口服或靜脈注射﹚。大扶康是口服藥膠囊。服食後有效成份存在於血液﹑尿﹑皮﹑汗﹑淚﹑體液等，屬全身性藥物。

它能抑制[[真菌|真菌]]／[[黴菌|黴菌]]﹙例如: [[念珠菌|念珠菌]]﹚繁殖，但不能真的把黴菌殺死。

== 用途 ==
常用於醫治婦女陰道念珠菌症。

== 注意事項 ==
有人對[[唑類|唑類]]藥物出現[[藥物敏感|藥物敏感]]反應，出現不同副作用。包括[[嘔吐|嘔吐]]﹑[[頭痛|頭痛]]﹑[[頭暈|頭暈]]﹑[[肚痛|肚痛]]﹑對[[肝臟|肝臟]]造成損害等等。因為如果對此藥敏感[[藥物過敏反應|藥物過敏反應]]嚴重可致死亡。

「氟康唑」在大部份國家屬於「須醫生[[處方藥物|處方藥物]]」，所以不宜隨便自行購買服用。

有研究指唑類藥物可能影響女性[[懷孕|懷孕]]期[[內分泌|內分泌]]而影響懷孕<ref>[http://www.tandfonline.com/doi/abs/10.1081/ERC-120015045 ASSESSING PREGNANCY RISKS OF AZOLE ANTIFUNGALS USING A HIGH THROUGHPUT AROMATASE INHIBITION ASSAY: Endocrine Research: Vol 28, No 3]</ref>。高劑量會降低 estrogen 雌激素(女性荷爾蒙)。

氟康唑與若干[[心臟病|心臟病]]用藥有化學反應，嚴重可致死亡。此藥與14種有不同程度互相影響干擾，但藥廠方面沒有研究藥物與藥物之間的相互作用干擾。<!-- 見 輝瑞 大扶康 盒內 藥物資料紙-->
