'''氟化钕'''是一种[[无机化合物|无机化合物]]，化学式为NdF<sub>3</sub>。

==制备==
氟化钕可由NdCl<sub>3</sub>或Nd<sub>2</sub>(CO<sub>3</sub>)<sub>3</sub>和40%氢氟酸反应得到：<ref>王亚军, 刘前, 索全伶, 等. 稀土氟化物的沉淀方法及组成研究[J]. 稀土, 2000, 21(1):14-18.</ref>

: NdCl<sub>3</sub> + 3 HF → NdF<sub>3</sub>↓ + 3 HCl
: Nd<sub>2</sub>(CO<sub>3</sub>)<sub>3</sub> + 6 HF → 2 NdF<sub>3</sub> + 3 H<sub>2</sub>O + 3 CO<sub>2</sub>↑

[[硝酸钕|硝酸钕]]和[[氟硼酸钠|氟硼酸钠]]在200 °C进行水热反应，也能制得NdCl<sub>3</sub>。<ref>孙元平, 贾佩云, 王芳, 等. 水热条件下稀土氟化物形貌的规律性生长[J]. 中国陶瓷, 2013(6).</ref>
