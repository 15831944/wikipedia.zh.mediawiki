'''氟化镝'''是一种[[无机化合物|无机化合物]]，化学式为DyF<sub>3</sub>。

==制备==
氟化镝可由DyCl<sub>3</sub>或Dy<sub>2</sub>(CO<sub>3</sub>)<sub>3</sub>和40%氢氟酸反应得到：<ref>王亚军, 刘前, 索全伶, 等. 稀土氟化物的沉淀方法及组成研究[J]. 稀土, 2000, 21(1):14-18.</ref>

: DyCl<sub>3</sub> + 3 HF → DyF<sub>3</sub>↓ + 3 HCl
: Dy<sub>2</sub>(CO<sub>3</sub>)<sub>3</sub> + 6 HF → 2 DyF<sub>3</sub> + 3 H<sub>2</sub>O + 3 CO<sub>2</sub>↑

[[硝酸镝|硝酸镝]]和[[氟硼酸钠|氟硼酸钠]]在200 °C进行水热反应，也能制得DyCl<sub>3</sub>。<ref>孙元平, 贾佩云, 王芳, 等. 水热条件下稀土氟化物形貌的规律性生长[J]. 中国陶瓷, 2013(6).</ref>

[[氧化镝|氧化镝]]和[[氟化氢铵|氟化氢铵]]混合并加热至300 °C使氧化物变为多孔，并继续升温至700 °C，通入氟化氢即发生反应：<ref>刘文生, 云月厚, 李国栋. 干法氟化制备高纯金属镝的工艺研究[J]. 稀有金属, 2003, 27(1):154-156.</ref>
: Dy<sub>2</sub>O<sub>3</sub> + 6 HF → 2 DyF<sub>3</sub> + 3 H<sub>2</sub>O
