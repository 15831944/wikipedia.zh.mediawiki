'''氟化氫'''（化學式：[[氫|H]][[氟|F]]）是氫的氟化物，有強烈的[[腐蝕性|腐蝕性]]，有剧毒。它是無色的[[氣體|氣體]]，在[[空氣|空氣]]中，只要超過3[[百万分率|ppm]]就會產生刺激的味道。

[[氫氟酸|氫氟酸]]是氟化氫的水溶液，可以透過[[皮膚|皮膚]]黏膜、[[呼吸道|呼吸道]]及腸胃道吸收。若不慎暴露於氫氟酸，應立即用大量清水沖洗20至30分鐘，然後以[[葡萄酸鈣|葡萄酸鈣]]軟膏或藥水塗抹，并緊急送醫處理；若不小心誤飲，則要立即喝下大量的高鈣[[牛奶|牛奶]]，然後緊急送醫處理。

== 異常特性 ==


氟化氫相對於其他鹵素氫化物具有一些異常的物理特性和化學特性。

=== 高熔點及沸點 ===
由於氟原子[[电负性|电负性]]很大，所以会与氢原子形成[[氫鍵|氫鍵]]，所以沸點較[[氯化氫|氯化氫]]、[[溴化氫|溴化氫]]、[[碘化氫|碘化氫]]等為高。

=== 弱酸性 ===
[[氫氟酸|氫氟酸]]之水溶液是一种[[弱酸|弱酸]]。一方面由於氟原子半徑小，電荷密度高，對氫原子的束縛較大，另一方面由於H-F共價鍵高度極性，故氟化氫具有分子間氫鍵，使得氟化氫具有高度能學穩定性，離解度非常低。因此，離解生成之水合氫離子濃度較低，使酸度常數K<sub>a</sub>較低。由此氟化氫為弱酸。
:HF<sub>(aq)</sub> + H<sub>2</sub>O<sub>(l)</sub> ←→ H<sub>3</sub>O<sup>+</sup><sub>(aq)</sub> + F<sup>-</sup><sub>(aq)</sub>
:K<sub>a</sub>=7X10<sup>-4</sup> mol dm<sup>-3</sup>

=== 酸度變化 ===
其他鹵化氫於水中濃度增加，酸度會下降。
:HA<sub>(aq)</sub> + H<sub>2</sub>O<sub>(l)</sub> ←→ H<sub>3</sub>O<sup>+</sup><sub>(aq)</sub> + A<sup>-</sup><sub>(aq)</sub>
對於一般氫鹵酸，鹵化氫濃度上升至某個程度，每單位體積溶液中之含水量急劇下降，令此反應之平衡位置趨向左邊，使離解度下降，酸性下降。
但在較濃的氟化氫水溶液中，酸度反而會增加。
此現象是由於氟離子之一個反應:
: F<sup>-</sup> + HF ←→ HF<sub>2</sub><sup>-</sup>
:K=5.1 mol<sup>-1</sup> dm<sup>3</sup>
:對於氫氟酸，若氟化氫濃度增加，以上反應之平衡位置趨向右邊，使氟離子濃度下降。
:HF<sub>(aq)</sub> + H<sub>2</sub>O<sub>(l)</sub> ←→ H<sub>3</sub>O<sup>+</sup><sub>(aq)</sub> + F<sup>-</sup><sub>(aq)</sub>
根據[[勒沙特列原理|勒沙特列原理]]，此化學平衡之平衡位置趨向右邊以傾向增加氟離子之濃度，故此離解度上升，酸性上升。

=== 侵蝕玻璃性 ===
[[氫氟酸|氫氟酸]]本身對[[硅酸鹽|硅酸鹽]]（-{zh-hans:硅; zh-hant:硅;}-）及[[二氧化硅|二氧化硅]]有極強的侵蝕能力。 故可腐蚀[[玻璃|玻璃]](主要成分是二氧化硅及硅酸盐)，是可腐蚀玻璃的酸，所以日常貯存要使用塑料瓶或[[铅|铅]]製容器。
:CaSiO<sub>3</sub><sub>(s)</sub> +6HF<sub>(aq)</sub>→ CaF<sub>2</sub><sub>(aq)</sub> + SiF<sub>4</sub><sub>(aq)</sub> + 3H<sub>2</sub>O <sub>(l)</sub>

== 作为溶剂 ==
氟化氢是酸性很强的溶剂，在氟化氢中：
* [[硝酸|硝酸]]、[[高锰酸|高锰酸]]、[[硫酸|硫酸]]显碱性；
* [[高氯酸|高氯酸]]显两性。
一价的氟化物均易溶于氟化氢，二价的氟化物SrF<sub>2</sub>、BaF<sub>2</sub>、PbF<sub>2</sub>溶解度稍大，CaF<sub>2</sub>、CdF<sub>2</sub>、HgF<sub>2</sub>溶解度稍小，三价的氟化物SbF<sub>3</sub>溶解度稍小，其余一至四价金属的氟化物均难溶。

一些低价过渡金属的氟化物可以将溶剂还原，如CrF<sub>2</sub>和UF<sub>3</sub>，将溶剂还原，放出[[氢气|氢气]]，自身被氧化为CrF<sub>3</sub>和UF<sub>4</sub>。

在酸性環境下，氫氟酸非常危險，因為[[氫離子|氫離子]]在[[酸性|酸性]]環境下會[[腐蝕|腐蝕]][[皮膚|皮膚]]，而[[氟離子|氟離子]]則會將體內的[[鈣|鈣]]，[[鎂|鎂]][[離子|離子]]溶出，將嚴重破壞[[生理平衡|生理平衡]]。
