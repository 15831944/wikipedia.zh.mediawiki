'''氟化钆'''是一种[[无机化合物|无机化合物]]，化学式为GdF<sub>3</sub>。

==制备==
氟化钆可由[[氧化钆|氧化钆]]和[[氟化氢铵|氟化氢铵]]加热反应得到，反应分为两步：<ref>郝占忠. 氟化氢铵氟化法合成氟化钆过程的基础研究[J]. 材料科学与工艺, 2010, 18(5):653-656.</ref><ref>郝占忠, 王斌. Gd<sub>2</sub>O<sub>3</sub>－NH<sub>4</sub>HF<sub>2</sub>系制备氟化钆机制及工艺研究[J]. 稀有金属, 2007, 31(1):97-101.</ref>
: Gd<sub>2</sub>O<sub>3</sub> + 6 NH<sub>4</sub>HF<sub>2</sub> → 2 NH<sub>4</sub>GdF<sub>4</sub> + 4 NH<sub>4</sub>F + 3 H<sub>2</sub>O
: NH<sub>4</sub>GdF<sub>4</sub> → GdF<sub>3</sub> + NH<sub>3</sub> + HF

[[氯化钆|氯化钆]]和[[氢氟酸|氢氟酸]]反应，用热水洗涤沉淀，得到GdF<sub>3</sub>·xH<sub>2</sub>O（x＝0.53）。若要获得无水物，可以将水合物和氟化氢铵混合加热，如果直接加热水合物，会有GdOF生成。<ref>郝占忠. 湿法氟化制备水合氟化钆的脱水机制及其氧的行为[J]. 有色金属(冶炼部分), 2010(5):36-39.</ref>
: GdCl<sub>3</sub> + 3 HF + x H<sub>2</sub>O → GdF<sub>3</sub>·xH<sub>2</sub>O + 3 HCl
