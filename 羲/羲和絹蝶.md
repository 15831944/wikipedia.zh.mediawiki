'''羲和絹蝶'''（學名：''Parnassius apollonius''）是屬於[[絹蝶亞科|絹蝶亞科]][[絹蝶族|絹蝶族]]的一種[[蝴蝶|蝴蝶]]，是[[絹蝶屬|絹蝶屬]]的一種。共有10個亞種。分佈於[[中亞|中亞]]至中國[[新疆|新疆]]西部一帶。被列入中國國家林業局令[[國家保護的有益的或者有重要經濟、科學研究價值的陸生野生動物名錄|國家保護的有益的或者有重要經濟、科學研究價值的陸生野生動物名錄]]。幼蟲[[寄主|寄主]]為景天科[[六瓣景天屬|六瓣景天屬]]、[[莧科|莧科]]及[[川續斷科|川續斷科]]植物<ref name=t1997>Tuzov, Bogdanov, Devyatkin, Kaabak, Korolev, Murzin, Samodurov, Tarasov, 1997 Guide to the Butterflies of Russia and adjacent territories: Hesperiidae, Papilionidae, Pieridae, Satyridae Butts. Russia adj. terr. 1, 144</ref>。

== 語源 ==
學名[[種名|種名]]源自[[古希臘語|古希臘語]] Apollōnía，源自 Apóllōn，即[[羅馬神話|羅馬神話]]中的太陽神[[阿波羅|阿波羅]]。語源跟同屬異種[[阿波羅絹蝶|阿波羅絹蝶]]一樣，為免重複，此種以中國神話中之太陽神[[羲和|羲和]]命名<ref name=mrs>Chou, 1994. Monografia Rhopalocerorum Sinensium 1: 193</ref>。

== 分佈 ==
中國（[[新疆|新疆]]、[[西藏|西藏]]）、[[烏茲別克斯坦|烏茲別克斯坦]]、[[吉爾吉斯斯坦|吉爾吉斯斯坦]]、[[塔吉克斯坦|塔吉克斯坦]]、[[哈薩克斯坦|哈薩克斯坦]]、[[巴基斯坦|巴基斯坦]]、[[阿富汗|阿富汗]]。

== 形態 ==
翅白色，亞緣有黑色圓形點列。前翅中室及端緣上有黑斑，中室外有黑斑2個，Cu<sub>2</sub>室上有1個。一後翅基部、前緣中部及M<sub>1</sub>室、Cu<sub>2</sub>室各有1個有黑邊的紅斑，雄前翅Cu<sub>2</sub>室的紅斑有時退化成小紅點；後翅Cu<sub>1</sub>室上常多出1個黑點，翅基部與後緣黑色<ref name=mrs/>。

== 寄主 ==
幼蟲的[[寄主|寄主]]植物包括<ref name=t1997/>：
* 景天科：[[六瓣景天屬|六瓣景天屬]]、[[瓦蓮屬|瓦蓮屬]]、[[紅景天屬|紅景天屬]]：[[柱花紅景天|柱花紅景天]]
* 莧科：[[豬毛菜屬|豬毛菜屬]]
* 川續斷科：[[藍盆花屬|藍盆花屬]]
