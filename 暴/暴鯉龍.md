'''暴鯉龍'''（{{lang-ja|'''ギャラドス'''}}，{{lang-en|'''Gyarados'''}}，香港舊譯'''鯉魚龍'''）是[[精靈寶可夢|精靈寶可夢]]系列登場的{{NUMBEROFPOKEMON}}種虛構[[角色|角色]]（[[怪物|怪獸]]）中的一種。

== 特徵 ==
暴鯉龍具有如[[青龍|青龍]]般的身體。這隻[[寶可夢|寶可夢]]是根據「[[鯉躍龍門|鯉躍龍門]]」這個經典故事創作的。在[[鯉魚王|鯉魚王]]的等級提升至20級後，便會一登龍門，變成暴鯉龍。

與其他一般進化的寶可夢不同的是暴鯉龍進化後身體的改變非常大，不但是動物物種的不同，而且顏色由原來的紅和黃色變成藍及淺黃色，身高和體重都比鯉魚王時分別增加7.2倍和23.5倍。此類跨物種的進化在其後登場的[[鐵炮魚|鐵炮魚]]和[[章魚桶|章魚桶]]也有類似情況。

暴鯉龍為水屬性及飛行屬性的兇惡[[寶可夢|寶可夢]]，全國圖鑑編號為130。

== 遊戲中的暴鯉龍 ==
在《精靈寶可夢》各版中的水域均可以破舊釣竿釣到[[鯉魚王|鯉魚王]]，把他訓練至[[等級|等級]]20就可以進化成暴鯉龍。又可以在各版各水域以厲害釣竿釣到暴鯉龍。《精靈寶可夢 金、銀版》中在第七個道館上方的湖更有唯一一隻的特別版紅色暴鯉龍。

== 動畫中的暴鯉龍 ==
曾多次在動畫中出現，較出名的一次是有關一條紅色的暴鯉龍被[[火箭隊|火箭隊]]捕捉一事。這段情節是出自遊戲，不同的是那條暴鯉龍原本是有閃光。它本來是一條鯉魚王，但被火箭隊強迫進化，但由於進化不完整，以致保留了原本的紅色。期間，它感到憤怒且四處破壞，但最終被小渡制服並收歸己有。

而小霞亦有一隻暴鯉龍，但開始的時候一直都很怕牠。此隻暴鯉龍比較特別，身為水屬性寶可夢，居然可以使用火屬性的噴射火焰招式，成為小霞打敗蟲屬性/草屬性寶可夢的王牌。初登場於HOSO第8話。原來是只鯉魚王，後進化，開始破壞道館，最後回到了精靈球。在ag45中亦曾出現。

== 瑣事 ==
暴鯉龍經常被誤為龍屬性的寶可夢，但事實是它是水屬性加上飛行屬性的。儘管它叫龍（會懂龍屬性的招式）卻不懂飛行。龍屬性的寶可夢是十分稀少的，有些中文名中含有龍的寶可夢也不是龍屬性的。（例如噴火龍，但是噴火龍的日文名與英文名其實都是來自蜥蜴，也只有中文版命名為龍。）
