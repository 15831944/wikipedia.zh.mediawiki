'''叶桂'''（{{bd|1667年||1747年|catIdx=Y叶}}），享壽80歲。 [[表字|字]]'''天士'''，[[号|号]]'''香巖'''，[[别号|别号]]'''南阳先生'''，[[晚号|晚号]]'''上津老人'''，[[江苏|江苏]][[吴县|吴县]]（今[[苏州市|苏州市]]）人。[[清代|清代]]名医，四大[[溫病學派|溫病學家]]之一，與[[薛雪|薛雪]]等齊名<ref>{{Cite book|title=名医方笺墨宝赏析|last=任岩东，张存悌主编|first=|publisher=沈阳：辽宁科学技术出版社|year=2013.12|isbn=978-7-5381-8369-6|location=|pages=7}}</ref>。

== 生平 ==
葉天士生于清[[康熙|康熙]]六年（1667年），出身於醫學世家，祖父[[葉時|葉時]]、父親[[葉朝採|葉朝採]]是當地名醫。叶天士自幼廣泛閱讀醫書，一生勤勉用功，善于博采众家之所长，共拜師17人，“师门深广”，在长期的从医生涯中积攒了大量的经验<ref>{{Cite book|title=话说国医 江苏卷|last=陈仁寿|first=|publisher=郑州：河南科学技术出版社|year=2017.01|isbn=7-5349-8016-8|location=|pages=56-58}}</ref>。

== 成就 ==
叶天士最大的成就在于对[[温病|温病]]理论的发展，确立了[[卫气营血辨证|卫气营血辨证]]。《[[溫熱論|溫熱論]]》一書反映了他的學術見解，所謂溫邪由口鼻侵入人體，辨證施治綱領為“衛之後，方言氣，營之後，方言血”、“在衛汗之可也，到氣才可清氣，入營猶可透熱轉氣，入血就恐耗血動血，直須涼血散血”<ref>{{Cite book|title=中医温病学讲稿|last=车念聪|first=|publisher=北京：中国中医药出版社|year=2016.11|isbn=978-7-5132-3060-3|location=|pages=7-8}}</ref>。

在脾胃學說方面，提出“脾胃有心之脾胃，肺之脾胃，肝之脾胃，腎之脾胃”的原理，主張“認清門路，[[寒|寒]][[熱|熱]][[溫|溫]][[涼|涼]]以治之，未可但言火能生土而用熱藥”。他对“胃阴”的理解，补充了李杲[[脾胃论|脾胃论]]的不足。

叶天士善抓主证，用药极精，对今天的医学工作者仍具有重要的参考价值。今日市面上廣為流行的「[[京都念慈菴|京都念慈菴]][[川貝|川貝]][[枇杷膏|枇杷膏]]」的處方也是由葉天士以枇杷葉蜜製枇杷膏得來。

== 著作 ==
叶天士一生忙于诊务，无暇著述。逝世之后，其学生和门人[[顧景文|顧景文]]、[[华岫云|华岫云]]、[[周仲升|周仲升]]整理其文稿、医案，汇编而成的《[[溫熱論|溫熱論]]》、《[[临证指南医案|临证指南医案]]》、《[[葉氏存真|葉氏存真]]》等書。

== 遺訓 ==
死前告誡子孫：「醫可為而不可為。必天資敏悟，讀萬卷書，而後可借術以濟世。不然，鮮有不殺人者，是以藥餌為刀刃也」（沈德潛《香巖傳》）。
