'''葉傲冬'''{{JP}}（{{lang-en|'''Ip Ngo Tung Chris'''}}<ref name=info>{{Cite web|url=http://www.elections.gov.hk/legco2016/pdf/intro_to_can/LC2_10.pdf|title=九龍西地方選區候選人簡介|accessdate=2016-09-14|work=2016立法會選舉}}</ref>；{{bd|1980年7月23日}}<ref>{{Cite web|url=http://hk.apple.nextmedia.com/dc2015/candidate/E02/%E8%91%89%E5%82%B2%E5%86%AC|title=區議會選舉 2015 候選人 : 葉傲冬|accessdate=2016-09-14|work=蘋果日報}}</ref>），[[民建聯|民建聯]]副秘書長。現任[[油尖旺區議會|油尖旺區議會]]主席及佐敦南區議員。同時，葉傲冬亦擔任[[西九文化區|西九文化區]]管理局董事局成員<ref>{{cite web|url=http://www.westkowloon.hk/tc/the-authority/wkcda-board/people/mr-chris-ip-ngo-tung-mr-chris-ip-ngo-tung/page/8|publisher=西九文化區管理局|accessdate=2016-12-26|title=西九文化區 - 葉傲冬先生|deadurl=yes|archiveurl=https://web.archive.org/web/20161226150138/http://www.westkowloon.hk/tc/the-authority/wkcda-board/people/mr-chris-ip-ngo-tung-mr-chris-ip-ngo-tung/page/8|archivedate=2016-12-26}}</ref>。

==經歷==
葉傲冬出身政治家族，他是前區議員[[葉國忠|葉國忠]]的兒子以及前立法會議員[[葉國謙|葉國謙]]的侄兒，母親為雷小惠。<ref>{{Cite web|url = http://news.sina.com/hk/mingpao/103-101-101-102/2008-10-05/15463338500.html|title = 民建聯葉傲冬贏區會補選|work =明報|date = 2008年10月5日}}</ref>他畢業於[[油蔴地天主教小學|油蔴地天主教小學]]<ref name="談天說藝 : 葉傲冬">[http://enews.westkowloon.hk/newsletter/201211/tc/highlight_03.html 西九文化區管理局電子通訊 2012 / 十一月號〈談天說藝 : 葉傲冬〉]</ref>（1992年小六）、[[何文田官立中學|何文田官立中學]]（1998年中五）及[[余振強紀念中學|余振強紀念中學]]（2000年中七），後於分別於[[萊比錫大學|萊比錫大學]]、[[香港浸會大學|香港浸會大學]]及[[香港中文大學|香港中文大學]]修讀德文、歐洲研究（主要研究德國）以及文化管理。

因[[劉志榮|劉志榮]]於2008年5月15日病逝，葉傲冬於[[香港特別行政區第3屆區議會補選|同年10月5日的佐敦東選區補選]]中勝出。[[2011年香港區議會選舉|2011年香港區議會選舉]]，成功連任。[[2015年香港區議會選舉|2015年香港區議會選舉]]因選區重新劃界，當選為佐敦南區議員。2016年，葉傲冬接替[[鍾港武|鍾港武]]自動當選為油尖旺區議會主席。<ref>{{Cite web|url = http://hk.apple.nextmedia.com/news/art/20160105/19439507|title = 葉傲冬接替鍾港武 掌油尖旺區會|work =蘋果日報|date = 2016年1月5日}}</ref>
