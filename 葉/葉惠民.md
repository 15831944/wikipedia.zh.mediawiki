'''葉惠民'''（'''Yip Wai-Man'''；{{bd|1959年3月18日}}），香港著名茶道家、香港茶道總會創會會長。創辦香港惠民茶道，亦是將新式茶藝館引入香港的第一人。除了茶文化事業上的成就外，葉惠民亦熱心公益，時常參與[[慈善|慈善]]活動。

1996年，葉惠民獲得「[[香港十大傑出青年|香港十大傑出青年]]」的榮譽，2009年獲選為「世界傑出華人」。

== 早年生平 ==
葉惠民生於[[印尼|印尼]][[泗水|泗水]]，因兒時遇上印尼排華，於是隨父母舉家移居[[香港|香港]]。葉惠民早年曾於澳門居住了7年，並於1972年在[[澳門|澳門]]嶺南中學（小學部）完成小學課程。之後在1978年畢業於[[香港|香港]][[嶺南中學_(香港)|嶺南中學]]（昕社校友）。葉惠民在於中五畢業後在香港明愛總部辦公室（即明愛堅道服務中心）任信差，同時修讀社會工作及心理學課程，最終得以加入社工行列，先後在明愛中心、亞洲歸主協會及突破機構從事青少年服務工作。期間因為籌辦中國文化推廣活動，無意間開始接觸中國茶，自此成為「茶癡」。

== 雅博茶坊 ==
在習茶、研茶的過程中，葉惠民有感於市面上的茶商對茶葉的季節、以至沖泡器具形製亦不敏感，於1989年開設「雅博茶坊」，希望以新的經營方式讓年輕一代認識中國茶藝，<ref>{{Cite news|url=https://hk.lifestyle.appledaily.com/etw/magazine/article/20150924/3_17632884/%E7%84%A1%E6%82%94%E4%BB%8A%E7%94%9F-%E9%9B%85%E5%8D%9A%E8%8C%B6%E5%9D%8A|title=無悔今生 雅博茶坊|author=羅佩明|date=2015-09-24|work=飲食男女|accessdate=}}</ref> 並改變香港人品茗以吃為主、以茶為輔的習慣。<ref>{{Cite news|url=http://the-sun.on.cc/cnt/news/20141101/00408_004.html|title=政壇：葉惠民搞愛心飯堂送暖|author=|date=2014-11-1|work=太陽報|accessdate=}}</ref>  

有別於傳統茶莊只賣茶葉的做法，雅博茶坊以「茶藝館」的形式、著力為大眾提供品茗的場所，此為香港之先例；除了講究茶葉，亦講究茶器、沖泡方式、場景，開展香港精緻品飲文化之先河。 此後香港陸續出現新式茶藝館。<ref>{{Cite book|title=台灣茶百科全書|last=林|first=志煌|publisher=台灣茶訊|year=2011|isbn=978-986-82676-4-0|location=台灣|pages=}}</ref>

== 多元發展 ==
除經營茶葉及茶具生意之外，葉惠民亦還積極透過跨界別合作，於香港推廣中國茶文化。他於1997年擔任香港話劇團原創話劇《誰遣香茶挽夢回》（[[杜國威|杜國威]]撰、[[汪明荃|汪明荃]]主演）顧問，<ref>{{Cite web|url=http://www.lizawang.com/music/opera/tea/tea_backstage.htm|title=誰遣香茶挽夢回：幕後榜|accessdate=|author=|date=|publisher=荃音樂}}</ref> 其後在1999年與[[無綫電視|無綫電視]]合作製作電視劇《[[茶是故鄉濃|茶是故鄉濃]]》。<ref>{{web reference|URL=http://hk.on.cc/hk/bkn/cnt/entertainment/20151120/bkn-20151120061914015-1120_00862_001.html|title=「音樂茶博士」葉惠民望拍「茶電影」|work=東網|year=|date=2015-11-20}}</ref> 此外，他亦於2001年參與了[[香港郵政|香港郵政]]「香港茗藝」郵票的製作。<ref>{{Cite news|url=http://www.xinhuanet.com/2018-07/02/c_1123068329.htm|title=香江茶人茶事：茶香傳遞中華情|author=|date=2018-07-02|work=新華社|accessdate=}}</ref> 為了進一步使大眾注意茶文化，葉惠民先於1999年11月27日在香港舉行世界最大型茶會「香江世紀茶會－－萬人泡茶迎千禧」（吸引近五千人參加，並享用了一共14,850杯茶及15,000中式糕餅）；2000年再於喜馬拉雅山海拔5416米之山峰泡中國茶，兩項活動皆刷新了兩項健力士世界紀錄。

2006年起，葉惠民嘗試將茶與音樂結合，製作以茶為主題的音樂，並參與作曲填詞。近年主要從事茶道教學。

葉惠民認為「茶是生活的藝術，最底裡的茶味，其實是人情味，做茶文化應是為了尋回失落的人情味」，<ref>{{Cite news|url=http://www.3weekly.com/common_emag/view_content.php?issue_no=762&pub_id=7&top_id=17&art_id=190443|title=茶藝有道－作曲填詞茶大師 「茶聖」葉惠民的茶間道|author=|date=|work=3週刊（第 762 期）|accessdate=}}</ref> 因此推崇「弘揚茶藝，服務社會，以茶為本，多元發展」，鼓勵習茶者以茶為始、透過茶培養德性，最終走入人群，積極參與扶貧、社區活動。

== 香港惠民茶道 ==
「香港惠民茶道」是葉惠民創立的中國茶流派，學習此流派的學生共計最少有6萬人。<ref>{{Cite journal|title=葉惠民教授：將中華茶文化進行到底|author=廖冬梅、何成保|url=|journal=華人經濟|issue=第3期|doi=|others=|year=2009年|volume=|page=|pmid=}}</ref> 不少活躍於香港茶文化圈子的茶人，亦曾於雅博茶坊習茶。

惠民茶道的特色是簡單、直接、快速，不以茶道為表演，去取枝節、以茶為本。但這並不代表放棄茶藝的美感，相反，葉惠民追求細節上盡美盡善，透過審核茶葉製作、講究沖泡步驟、製作精細茶具，完善品茗經驗。以下為惠民茶道的部份主張。

==== 沖泡程序、禮儀<ref>{{Cite web|url=http://www.teatea.net/ed.htm|title=葉惠民教授：茶藝示範|accessdate=|author=|date=|publisher=雅博茶坊（早期網站）}}</ref> ====
中國茶藝貴乎優雅、簡潔。與沖泡、欣賞茶藝無關之多餘動作，應予減省。當然，程序並非金科玉律，可以調整或改變，但應有理據，且以優雅簡潔為宗。{{HideH|泡茶程序}}

<blockquote>1. 備器</blockquote><blockquote>在泡茶的几桌上置放應用的茶具，亦可搭配淡雅的瓶花，以及播放傳統的絲竹管絃。</blockquote><blockquote>2. 煮水</blockquote><blockquote>將潔淨清水煮至微溫，然後放在茗爐上繼續烹煮。</blockquote><blockquote>3. 備茶</blockquote><blockquote>茶則須準備適量的茶葉，可先讓品茗者欣賞。</blockquote><blockquote>4. 燙壺</blockquote><blockquote>先用燙沸的水注入泡茶器 （茶壺或茶盞），主要是提高泡茶器的溫度，以免影響真正泡茶時茶葉所能吸收的水溫。其次溫度可將茶葉在存放時吸的異味揮發出來。 </blockquote><blockquote>5. 置茶</blockquote><blockquote>將適量的茶葉置放於泡茶器中，注意茶葉如果佈放不均勻，可以輕輕拍數下泡茶器。</blockquote><blockquote>6. 初泡</blockquote><blockquote>注水至泡茶器中，如水溫過高，可以先注入茶海降溫，再由茶海注入泡茶器中，浸泡數秒即將茶湯注入茶海，然後分注杯中，這一泡有人稱為「溫潤泡」、「潤杯泡」。這泡的目的是先讓茶葉吸收水份和溫度，並且將存放時吸染的異味揮發出來。 將杯中的茶湯倒進水盂中。</blockquote><blockquote>7. 潤壺</blockquote><blockquote>如果用茶壺泡茶則可以將茶湯倒在壺的外壁，日久見工夫，茶壺的顏色和光澤會變得古樸厚潤。</blockquote><blockquote>8. 賞香</blockquote><blockquote>讓客人欣賞一下杯中的「餘香」。</blockquote><blockquote>9. 正泡</blockquote><blockquote>然後重覆初泡的程序，浸泡時間一般要加長，大概三十秒至一分鐘，因茶而異，因各人口味而異，沒有硬性規定。分茶湯時要求杯杯份量相等，而且不能太滿，約八分滿，這樣客人才會拿得舒服自然。</blockquote><blockquote>10. 品茗</blockquote><blockquote>品茶要先觀顏色，再聞香氣，然後才送進口裡，細細品賞，自然領略到箇中真趣。</blockquote>
{{HideF}}

==== 茶器製作 ====
茶具應要兼顧功能與美感，要貼合不同茶葉之性質，同時為品茗者帶來良好的品飲體驗。葉認為，坊間許多茶杯設計不佳，例如茶杯邊缺乏適當彎度，未能與嘴唇吻合，忽略了飲用者的感覺。此外，飲用不同茶類時，也要使用相應的茶具，例如飲用龍井時，則可用白酒杯觀賞茶葉翻滾之態，欣賞茶在香味以外的形色之美。<ref>{{Cite news|url=http://www.3weekly.com/common_emag/view_content.php?issue_no=762&pub_id=7&top_id=17&art_id=190443|title=茶藝有道－作曲填詞茶大師 「茶聖」葉惠民的茶間道|author=|date=|work=3週刊（第 762 期）|accessdate=}}</ref>

為此設計出合符香港惠民茶道美學的茶器，葉惠民開始設計茶具、並請匠人製作：「先是拿一些精品叫他們（製壺匠）仿造，一邊解釋許多細微巧妙之處。〔⋯〕最後是按照自己泡荼的理論，對許多形制再作調整修改。然後叫壺手領會表達，務求在泥質手藝火候造型理念上，都稱心如意，才算滿足。」<ref>{{Cite book|title=朱泥壺的世界|last=|first=|publisher=壺中天地雜誌社|year=1990|isbn=9789868563728|editor-last=萬|editor-first=妙玲|editor2-last=賴|editor2-first=傑誠|location=台灣|pages=|at=葉惠民〈朱泥壺的世界一寫在淵如館〉}}</ref>

葉惠民於1991年建立「香港茶器工作坊」，設計及製作茶具，底款有「青惠堂」、「綵秀閣」、「雅博」、「香江惠民」和「淵如館」等等。其茶器作品「雅博淵如館朱泥水平壺」、「鬥彩纏枝花品茗杯」及「鬥彩纏枝花茶池」於1993年獲[[中国茶叶博物馆|中國茶葉博物館]]收藏；同年，「十二花月令茶碗」獲[[美国总统|美國前總統]][[乔治·赫伯特·沃克·布什|老布殊]]私人收藏；1994年，「雅博鬥彩纏枝花卉瓷器茶具」、「茶池水壺茶罐茶海蓋碗杯」、「煮水器品茗杯」以及「雅博淵如館朱泥壺」獲[http://www.fmsbwg.com/ 中國法門寺博物館]收藏。<ref>{{Cite web|url=http://www.teatea.net/Yip_A.html|title=葉惠民先生簡歷|accessdate=|author=|date=|publisher=雅博茶坊（早期網站）}}</ref> 葉惠民至今仍然在設計新的茶具，購買者偏佈中外。

== 香江茶道制服團隊 ==
葉惠民教授創立香江茶道制服團隊，是香港歷史上首個以茶道為主題，並以推行中國傳統文化為宗旨的制服團隊，至今已建立了20團。團隊界別分為幼兒團，少年團，青年團，傷健團，領袖團及長者團。團隊主張以茶文化為本，多元發展。學習茶道，不單能提昇個人的修養，亦講究增進人與人之間的溝通，以及與大自然的契合。團隊保留了制服團隊中互相協作，建立團隊精神的重要環節，並致力透過活潑而多元化的考章制度，鼓勵團員循序漸進地掌握茶藝知識和注入現代氣息的傳統文化精蘊，從而達至弘揚中華優秀文化的宗旨。
