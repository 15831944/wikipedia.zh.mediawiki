'''葉翠翠'''（{{lang-en|Tracy Ip Chui Chui}}，{{bd|1981年|9月10日|catIdx=Y葉}}），[[香港|香港]][[藝人|藝人]]，2005年度[[香港小姐競選|香港小姐競選]]冠軍，前[[無綫電視|無綫電視]]基本藝人合約藝員。目前擔任[[演員|演員]]及節目[[主持人|主持人]]。

==簡介==
葉翠翠的父親任職[[消防員|消防員]]，她自小在[[元朗|元朗]][[八鄉|八鄉]]的宿舍長大，有一兄一姊<ref name="港姐的新衣_壹週Plus"/>。她早年就讀旅港增城同鄉會兆霖學校，在[[元朗|元朗]][[中華基督教會基朗中學|中華基督教會基朗中學]]就讀中一至中三，高中就讀位於[[英国|英国]]蘇里的[[愛德華國王學校|愛德華國王學校]]（King Edward's School），畢業後返港入讀[[香港理工大學|香港理工大學]][[時裝設計|時裝設計]]系。其後，曾擔任兼職[[模特兒|模特兒]]、在[[寵物|寵物]]診所任職助護<ref>{{Cite web |url=https://zh-hk.facebook.com/Tracychuichui/posts/463757880625925 |title=葉翠翠 ip chui chui facebook專頁的帖文 |date=2017-06-25}}</ref>及在[[香港賽馬會|香港賽馬會]]當接待員<ref name="港姐的新衣_壹週Plus">{{Cite web |url=https://hk.lifestyle.appledaily.com/nextplus/magazine/article/20070913/2_10143852/%E6%B8%AF-%E5%A7%90-%E7%9A%84-%E6%96%B0-%E8%A1%A3-%E8%91%89-%E7%BF%A0-%E7%BF%A0 |title=港姐的新衣　葉翠翠 |date=2007-09-13 |publisher=壹週Plus}}</ref>。

她於當年參選香港小姐期間曾表示其志願是成為一個成功的時裝設計師。不過卻曾被指“離譜[[整容|整容]]”，一直是媒體的“話題王”而被視為奪冠大熱門，並最终獲得冠軍及「完美愛心小姐」獎。

2016年12月，葉翠翠表示被[[無綫電視|無綫電視]]要求她減薪三分之一，雙方曾同意提早結束合約<ref>{{Cite news |url=http://hk.apple.nextmedia.com/entertainment/art/20161218/19869019 |title=無綫減人工兼停Job 35歲葉翠翠劈炮 |newspaper=蘋果日報 |location=香港 |date=2016-12-18}}</ref>，最終於2017年12月底正式結束合約。

==感情生活==
葉翠翠曾與歌手[[李逸朗|李逸朗]]拍拖，又曾與[[香港先生|香港先生]][[高鈞賢|高鈞賢]]傳過緋聞，2008年跟中國內地富豪[[黄有龙|黄有龙]]（[[趙薇|趙薇]]現任丈夫）秘密拍拖。至2015年7月10日，正式與建築公司副董事周曉東（Raymond）結婚<ref>{{cite news|url=http://skypost.ulifestyle.com.hk/article/1648923/|title=葉翠翠泊碼頭 閃嫁平民董事 網民笑釣錯金龜|date=2015-02-18|work=晴報}}</ref>。

2016年11月7日上午在[[微博|微博]]公布怀孕消息，預產期為2017年4月底至5月初。2017年4月11日，在[[社交網站|社交網站]]宣布誕下重七磅半的兒子周津羲（Jake）<ref>{{cite news|url=http://hk.on.cc/hk/bkn/cnt/entertainment/20170411/bkn-20170411141615118-0411_00862_001.html|title=35歲葉翠翠誕7磅半囝囝|work=東網|date=2017-04-11}}</ref>。2018年6月30日，再誕下9磅小兒子Triton<ref>{{Cite web |url=https://hk.entertainment.appledaily.com/enews/realtime/article/20180630/58384823 |title=葉翠翠誕9磅細仔 再入產房緊張顫抖 |date=2018-06-30 |publisher=蘋果日報}}</ref>。
