'''葉永鋕事件'''是台灣關於[[多元性別|多元性別]]議題的校園事件。葉永鋕生前就讀於[[屏東縣立高樹國民中學|屏東縣高樹國中]]，因其與眾不同的[[性別氣質|性別氣質]]而遭到同學[[霸凌|霸凌]]<ref >{{Cite web|url=http://163.21.249.63/joomla/upload/cht/article/5-4-d-3.pdf|title=反霸凌與性別平等教育|author= 游美惠|publisher=高雄師範大學性別研究所|language=zh-tw|accessdate=2019-04-20}}</ref><ref >{{Cite web|url=http://www.admin.ltu.edu.tw/public/Download/6777/201805011001421.pdf|title=玩耍?霸凌?國小高年級男童性霸凌者經驗之探究|author=楊幸真、 張榮顯|publisher=教育學誌|language=zh-tw|accessdate=2019-04-20}}</ref>，不敢在下課時間去上廁所。2000年4月20日早上，葉永鋕在接近下課時，提前離開教室去上廁所，後來被發現傷重倒臥血泊中，送醫後不治死亡<ref>{{Cite news|url=http://news.chinatimes.com/forum/0,5252,11051401x112010042000427,00.html|title=〈那條校園性別平等坎坷路〉|author=畢恆達|newspaper=《[[中國時報|中國時報]]》|language=zh-tw|date=2010-04-20|accessdate=2011-06-10|deadurl=yes|archiveurl=https://web.archive.org/web/20100729040734/http://news.chinatimes.com/forum/0,5252,11051401x112010042000427,00.html|archivedate=2010-07-29}}</ref>。此事件引起臺灣社會對於性別教育的討論<ref name=bb>{{Cite news |url = http://gsrat.net/news/newsclipDetail.php?ncdata_id=3089 |title=同學欺凌致上課如廁　地板濕滑校方有責|newspaper=蘋果日報 |author=王吟芳 |language=zh-tw |date=2006-09-14 |accessdate=2016-10-29}}</ref>，使得原《[[s:兩性平等教育法|兩性平等教育法]]》在2004年修訂為《性別平等教育法》，教育政策也從傳統二元的兩性教育延伸轉化成為更具普遍性的性別平等教育<ref name=aa>{{Cite journal |url = http://www.bp.ntu.edu.tw/hdbih/new_page_47.htm |title=從葉永鋕的死檢視男性特質 |issue=第13期 |journal=兩性平等教育季刊 |location=[[臺北市|臺北市]] |publisher=教育部性別平等委員會 |language=zh-tw |date=2000-11-07 |accessdate=2016-10-29}}</ref>。

== 背景 ==
根據葉永鋕的母親陳君汝描述，葉永鋕自小「太女性化」，「很喜歡玩扮家家酒的遊戲，玩煮菜的玩具」。國中時也因為他的性別氣質而遭同學欺負，例如國一和國二曾被數位同學強行脫下褲子「驗明正身」<ref name="蘇芊玲">{{Cite journal |url = http://readopac2.ncl.edu.tw/nclJournal/GetPDF?tid=A07034455&jid=69301585&eid=c4eecc588fd8dac5569025dd75a24226 |author=蘇芊玲 |title=〈我的兒子葉永鋕－專訪葉媽媽陳君汝女士〉|issue=第38期 |journal=兩性平等教育季刊|publisher=教育部性別平等委員會 |location=[[臺北市|臺北市]]|language=zh-tw| year=2006-12 |accessdate=2011-06-10}}</ref>。陳君汝回憶說：「永鋕就讀國中後，曾經跟我說同學在學校會抓著他，要脫他的褲子。」儘管他曾向學校反應，但是情況並未改善，以至於葉永鋕下課時間不敢上廁所<ref name="蘇芊玲" />。根據調查，當時葉永鋕被迫以四種方式如廁：提早幾分鐘下課、找要好的男同學陪同、上課鐘響後使用女生廁所，以及使用教職員廁所等<ref name="蘇芊玲" />。

== 事件經過 ==
2000年4月20日，屏東高樹國中三年二班的葉永鋕，在第四節[[音樂|音樂]]課下課前五分鐘舉手向老師表示想上廁所，經老師同意後離開教室，但一直到下課都沒有返回教室，之後被同學發現在廁所裡，倒臥在血泊中<ref>{{Cite book |url = http://nhuir.nhu.edu.tw/retrieve/21636/100NHU05665002-001.pdf |title=糾葛於生活事件與系統之間---國中教師處理校園霸凌之實踐邏輯 |publisher=南華大學 |language=zh-tw |date=2012-03-19 |author=簡雅慧 |accessdate=2016-10-29}}</ref>。葉永鋕被送往當地[[同慶醫院|同慶醫院]]，後轉醫到[[屏東基督教醫院|屏東基督教醫院]]急救，但隔天仍然去世<ref name=葉永鋕案>{{Cite web|url=http://diversity.tgeea.org.tw/joomla/index.php?option=com_content&view=category&id=38&Itemid=70|title=葉永鋕案大事記|publisher=社團法人台灣性別平等教育協會|language=zh-tw|accessdate=2016-10-29}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

== 死因 ==
2000年6月10日，[[法務部法醫研究所|法務部法醫研究所]]驗屍鑑定書認為葉永鋕身上無護身防禦之擦傷，解剖時發現葉生患有高度氣管性肺炎，因此判定葉生是如廁後意識上較缺乏反應能力下跌倒造成頭顱骨之骨折，並且認為是其自身疾病導致昏倒<ref>{{Cite web |url=http://jirs.judicial.gov.tw/FJUD/FJUDQRY03_1.aspx?id=1&v_court=PTD+%E8%87%BA%E7%81%A3%E5%B1%8F%E6%9D%B1%E5%9C%B0%E6%96%B9%E6%B3%95%E9%99%A2&v_sys=M&jud_year=89&jud_case=%E8%A8%B4&jud_no=371&jud_no_end=&jud_title=&keyword=&sdate=&edate=&page=&searchkw=&jmain=&cw=0|title=臺灣屏東地方法院裁判書刑事類89年刑訴字371號|accessdate=2016-10-29 |author=包梅真 |date=2001-01-19 |publisher=臺灣屏東地方法院|language=zh  }}</ref>。

2001年，臺灣高等法院高雄分院以假設性問題函詢[[國立台灣大學|國立台灣大學]]醫學院附設醫院高度氣管性[[肺炎|肺炎]]是否會引起患者昏倒，該院回函表示，一般而言『[[支氣管性肺炎|支氣管性肺炎]]』應不致產生暈倒現象，除非併發其他疾病，如腦部或[[心臟|心臟]]。<ref>{{Cite web |url=http://jirs.judicial.gov.tw/FJUD/FJUDQRY03_1.aspx?id=1&v_court=KSH+%E8%87%BA%E7%81%A3%E9%AB%98%E7%AD%89%E6%B3%95%E9%99%A2+%E9%AB%98%E9%9B%84%E5%88%86%E9%99%A2&v_sys=M&jud_year=90&jud_case=%E4%B8%8A%E8%A8%B4&jud_no=455&jud_no_end=&jud_title=&keyword=&sdate=19110101&edate=99991231&page=&searchkw=&jmain=&cw=1|title=臺灣高等法院高雄分院裁判書刑事類90上訴字455號|accessdate=2016-10-29 |author=張明松 江泰章 陳吉雄|date=2001-08-21 |publisher=臺灣高等法院高雄分院|language=zh  }}</ref>。

2005年9月6日，行政院[[中華民國衛生福利部|衛生署]]醫事審議委員會鑑定書認為，死者於屍體解剖發現有氣管性肺炎及[[心肌|心肌]]病變，因此死者仍有可能因心性猝死造成暫時性意識喪失，於昏倒時頭部撞擊發生顱內出血，依据法院所提供之資料，此點仍無法排除其可能性，且死者之弟供稱死者曾於家中夜間如廁時發生昏倒，由於該實例有記載其弟之描述，故小便時所引發的[[迷走神經|迷走神經]]性昏厥，於昏倒時造成重度頭部外傷進而死亡，無法排除其可能性。<ref>{{Cite web |url=http://jirs.judicial.gov.tw/FJUD/FJUDQRY03_1.aspx?id=1&v_court=KSH+%E8%87%BA%E7%81%A3%E9%AB%98%E7%AD%89%E6%B3%95%E9%99%A2+%E9%AB%98%E9%9B%84%E5%88%86%E9%99%A2&v_sys=M&jud_year=93&jud_case=%E4%B8%8A%E6%9B%B4(%E4%B8%80)&jud_no=224&jud_no_end=&jud_title=&keyword=&sdate=19110101&edate=99991231&page=&searchkw=&jmain=&cw=1|title=臺灣高等法院高雄分院刑事判決93年度上更(一)字第224號|accessdate=2016-10-29 |author=王憲義 李璧君 張盛喜|date=2005-10-31 |publisher=臺灣高等法院高雄分院|language=zh  }}</ref>

== 影響及爭議 ==
{{Wikisource|性別平等教育法}}
葉永鋕的死因在各方證詞與未經證實的猜測下有不同版本，部分人士認為此事肇因於[[學校|學校]]未重視他的[[性別特質|性別特質]]，也沒有教授多元性別教育有關，因而引起[[性別平權|性別平權]]教育人士、[[社會學家|社會學家]]和[[社會工作者|社會工作者]]關注<ref>{{Cite journal|author=畢恆達|title=〈從葉永鋕的死檢視男性特質－從兩性平等到性別平等：記葉永鋕 〉|journal=《兩性平等教育季刊》|url=http://sa.hwh.edu.tw/ezfiles/6/1006/attach/18/pta_973_601465_19206.pdf|year=2000|publisher=中華民國教育部|location=臺北市|volume=13|pages=125-132|format=pdf|language=zh-tw|accessdate=2011-06-10}}</ref>。

事件發生後不久，[[中華民國教育部|教育部]]由兩性平等教育委員會委員[[紀惠容|紀惠容]]、[[王麗容|王麗容]]、[[蘇芊玲|蘇芊玲]]、[[畢恆達|畢恆達]]等組成調查小組，他們在將事件與後續處置等製成記錄後，於報告書中呼籲教育部重視校園性別問題<ref name=葉永鋕案/>。

2000年10月，教育部發起「新校園運動：反性別暴力」活動，強調除了尊重傳統兩性外，也應尊重不同性傾向和性別特質的人，並破除刻板印象、消除暴力<ref name=葉永鋕案/>。

2000年12月16日，教育部宣佈「兩性平等教育委員會」更名為「性別平等教育委員會」，教育政策從兩性教育延伸轉化成為[[多元性別|多元性別]]教育<ref name=aa/>。

2000年，同志[[紀錄片|紀錄片]][[導演|導演]][[陳俊志|陳俊志]]籌畫拍攝葉永鋕紀錄片<ref name=葉永鋕案/>。

2001年，教育部推動性別教育的主題訂為「多元性別、校園安全」<ref name=葉永鋕案/>。

2004年，《性別平等教育法》制定時更注重[[性傾向|性傾向]]、性別特質、[[性別認同|性別認同]]等內容<ref>{{Cite book|author=|title=〈一個溫柔早逝的生命，改變了什麼？－葉永鋕事件五年回顧記者會〉|publisher=台灣性別平等教育協會|location=臺北市|language=zh-tw|date=2005-12-17|accessdate=2011-06-10|url=http://diversity.tgeea.org.tw/joomla/index.php?option=com_content&view=article&id=52:20051217-&catid=40:2010-12-29-09-10-46&Itemid=77}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

2007年，教育部拍攝紀錄片《[[玫瑰少年|玫瑰少年]]》。

2009年，《[[玫瑰少年|玫瑰少年]]》發行，送至全國高中作為教材<ref>{{Cite news|title=〈教部籌資拍片，葉永鋕入教材〉|newspaper=《台灣立報》|author=嚴文廷 |publisher=[[世新大學|世新大學]] |url=http://www.lihpao.com/?action-viewnews-itemid-83718 |location=[[臺中市|臺中市]]|language=zh-tw |accessdate=2011-06-10}}</ref>。
