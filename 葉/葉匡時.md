'''葉匡時'''（{{bd|1957年|1月7日||||}}），[[臺灣|臺灣]]管理學家及政治人物，[[美國|美國]][[卡內基梅隆大學|卡內基梅隆大學]]博士，現任[[高雄市|高雄市]]副市長<ref>{{cite news|url=https://www.ettoday.net/news/20181203/1321668.htm|title=韓國瑜宣布「葉匡時接任高雄副市長」　春節後報到|publisher=ETtoday新聞雲|date=2018-12-03}}</ref>，曾任中華民國[[中華民國交通部|交通部]]部長、[[國立中山大學|國立中山大學]]企業管理學系教授、[[國立政治大學|國立政治大學]]科技管理與智慧財產研究所教授。現為[[長風文教基金會|長風文教基金會]]執行長、亞太[[公私合夥建設|公私合夥建設]]發展協會理事長。

==生平==
1957年出生於臺灣[[花蓮市|花蓮市]]的[[大陳新村|大陳新村]]並度過童年。父親是1949年移居台灣的[[浙江|浙江]][[溫嶺|溫嶺]]人，母親是1955年撤退台灣的[[大陳島|大陳島]]居民<ref>[http://www.cna.com.tw/news/aipl/201406170055-1.aspx 中央社：葉匡時憶童年 學校禁說大陳話 ]</ref>。曾就讀私立若瑟小學、復興國小、花師附小。國中就讀花蓮美崙國中，後轉至台北仁愛國中。高中就讀[[師大附中|師大附中]]。1979年取得[[台灣大學|國立台灣大學]]政治系學士，1984年取得美國[[德拉瓦大學|德拉瓦大學]]公共行政碩士，1990年取得美國卡內基梅隆大學組織理論博士。

==學界經歷==
1991年起任教於[[高雄|高雄]][[國立中山大學|國立中山大學]]企業管理學系，曾任企管系主任、傳播管理研究所（現[[國立中山大學行銷傳播管理研究所|行銷傳播管理研究所]]）所長、醫務管理研究所所長以及EMBA執行長等多項職務；並曾至[[上海復旦大學|上海復旦大學]]、[[北京清華大學|北京清華大學]]任訪問教授<ref>{{Cite web |url = http://bm.nsysu.edu.tw/tutorial/ksyeh/career1.htm |title = 葉匡時中山大學個人網頁：個人簡介 |accessdate = 2014-12-23}}</ref>；發表過三十多篇具有評審制度的期刊論文<ref>{{Cite web |url =  http://bm.nsysu.edu.tw/tutorial/ksyeh/phesis.htm |title = 葉匡時中山大學個人網頁：評審論文 |accessdate = 2014-12-23}}</ref>。葉匡時也曾擔任[[經濟日報|經濟日報]]、[[天下雜誌|天下雜誌]]等書報雜誌的專欄作家，並著有《總經理的新衣》、《總經理的內衣》、《總經理的面具》、《EMBA的第一門課》、《當代中國工商環境與企業管理》、《狐狸與狼：企業管理的個性與群性》等企業管理專書，三度榮獲[[經濟部|經濟部]]中小企業處金書獎。<ref>{{Cite web |url = http://bm.nsysu.edu.tw/tutorial/ksyeh/otherpaper.htm |title = 葉匡時中山大學個人網頁：專書 |accessdate = 2014-12-23 }}</ref>

在2015年卸任交通部長後，葉匡時轉任政治大學科技管理與智慧財產研究所教授<ref>{{cite news|url=http://www.storm.mg/article/59772|title=找葉匡時運籌帷幄 洪秀柱將打政策牌對決小英|author=羅暐智|date=2015-08-04|access-date=2015-08-05|newspaper=風傳媒}}</ref>，之後於2018年退休。<ref>{{cite news|title=打算參選台北市長？葉匡時提前自政大教職退休|date=2018-02-01|publisher=自由時報|url=http://news.ltn.com.tw/news/politics/breakingnews/2329015|access-date=2018-02-15}}</ref>

==業界經歷==
葉匡時曾擔任燦坤實業與百略醫學科技兩家公司之獨立董事，以及太平洋聯網科技營運長，並與友人先後創辦了[[易遊網|易遊網]]、[[燦星網|燦星網]]兩家線上網路旅遊公司。<ref>{{Cite web |url = http://bm.nsysu.edu.tw/tutorial/ksyeh/career1.htm |title = 葉匡時中山大學個人網頁：個人簡介 |author = 葉匡時 |accessdate = 2014-12-23}}</ref>

國際上，交通部觀光局在2013年的最後一天，達成全年800萬來台旅客目標，締造歷史新高<ref>[http://www.cna.com.tw/news/aipl/201401020073-1.aspx 葉匡時向觀光業者喊話 籲加薪],[[中央社|中央社]],2013.12.31</ref>。[[聯合國|聯合國]]世界觀光組織公布2013年全球前十名成長最快速的旅遊地點，台灣以9.7%的年成長率排名第十<ref>[http://www.chinatimes.com/realtimenews/20140207002401-260405 世界旅遊最快成長地 台列第十名],[[中時電子報|中時電子報]],2014年02月07日</ref>。2013年，世界經濟論壇（WEF）評比140個國家及經濟體觀光競爭力，臺灣名列第33名，較上次評比2011年（第37名）上升4名。2013年6月葉匡時應[[法國|法國]]運輸、海洋部部長之邀，成為台法斷交後第一位赴法國官方訪問的交通部長，就航太發展交換意見、盼修航約增加航班、並期盼法國支持台灣加入國際民航組織ICAO<ref>{{cite news|author=汪淑芬|title=台法交長破冰 葉匡時提修航約 |url=http://www.cna.com.tw/news/ahel/201307010042-1.aspx | publisher=中央社|date=2013-07-01}}</ref>。9月應[[多明尼加|多明尼加]]觀光部長邀請，參訪多明尼加<ref>{{cite press release | url=http://www.motc.gov.tw/ch/home.jsp?id=14&parentpath=0,2&mcustomize=news_view.jsp&dataserno=201309100001&toolsflag=Y | publisher=交通部|title=葉匡時部長率團訪多明尼加 深化雙方觀光交流合作|date=2013-09-10}}</ref>。同年，[[民航局長|民航局長]]以特邀貴賓身份參加[[ICAO|ICAO]]，是42年來台灣第一次得以參加國際民航組織<ref>{{cite press release | url=http://www.mofa.gov.tw/igo/News_Content.aspx?n=3E9C6A131C420E9D&sms=B5877621755C11D4&s=5BA251A764D265C9 | publisher=外交部|title=「國際民航組織」(ICAO)邀請我出席本（102）年大會|date=2013-09-13}}</ref>。
