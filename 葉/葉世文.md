'''葉世文'''（{{bd|1948年|2月17日|||}}），生於[[彰化縣|彰化縣]][[和美鎮|和美鎮]]<ref>[http://www.cnps.org.tw/ReportItem/?aid=5d8e27503d28466cbc4a0861872f4f5c 國家公園的推手－葉世文專訪] {{webarchive|url=https://web.archive.org/web/20140606224000/http://www.cnps.org.tw/ReportItem/?aid=5d8e27503d28466cbc4a0861872f4f5c |date=2014-06-06 }}</ref>，中華民國政治人物，前[[中國國民黨|中國國民黨]]黨員、政府官員，曾任中華民國[[內政部營建署|內政部營建署]]署長、[[桃園市|桃園縣]]副縣長。

==生平==
===早年===
葉世文生於彰化，因為其父親的工作，幼年時曾居住過基隆、高雄、台南等地，其父後回到彰化銀行工作，全家也因此搬回彰化和美。

===公職經歷===
1973年，服兵役退伍後，在[[中興新村|中興新村]]擔任省政府主計處的約聘研究員，同年年底，特種考試臺灣省地方行政人員乙等考試統計人員類科及格後，成為[[臺灣省政府|臺灣省政府]]正式科員。

1981年，[[林洋港|林洋港]]由台灣省政府主席調任內政部長，葉世文隨林洋港至內政部任職機要，負責聯絡國會。1984年，[[吳伯雄|吳伯雄]]擔任內政部長，繼續任用葉世文為機要<ref>{{Cite web |url=http://udn.com/NEWS/BREAKINGNEWS/BREAKINGNEWS1/8715396.shtml|author=陳正健|language=zh|title=吳志揚識人不清 廖正井：無關伯公|publisher=聯合報|date=2014-06-03 |accessdate=2014-06-03}}</ref>。

吳伯雄在內政部長任內，陸續成立了玉山、陽明山、太魯閣與墾丁四大國家公園，在吳伯雄安排下，葉世文進入營建署國家公園管理處<ref>{{Cite web |url=http://udn.com/NEWS/NATIONAL/NATS6/8711957.shtml|author=游文寶、鄭國樑、顏彙燕|language=zh|title=海派葉世文 「依法行事」掛嘴邊|publisher=聯合報|date=2014-05-31 |accessdate=2014-06-03}}</ref>。1985年，擔任玉山國家公園管理處首任處長，任內興建了鹿林山管理站（1991年遷至[[塔塔加|塔塔加]]），完成塔塔加遊客中心與南安遊客中心，興建了南安管理站與玉山國家公園行政及訓練中心。隨後又擔任陽明山國家公園與[[太魯閣國家公園|太魯閣國家公園]]管理處處長。在[[太魯閣國家公園|太魯閣國家公園]]管理處處長任內，興建合歡山管理站，獲得行政院公共工程金質獎

2002年，葉世文進行[[合歡主峰|合歡主峰]]景觀改善工程，整修舊有軍營建物，增加景觀設施，同時增設手機基地台。但有遊客對媒體投書，認為這些土木建設，可能影響生態保育，違反國家公園設立的目的，時任行政院長[[游錫堃|游錫堃]]指派政務委員[[林盛豐|林盛豐]]前往調查，調查結果認定葉世文過度開發，專業判斷有問題，游錫堃因此下令撤換其太魯閣國家公園管理處長之職<ref>{{Cite web |url=http://news.ltn.com.tw/news/focus/paper/784110|author=江志雄、林惠琴|language=zh|title=游：曾撤換葉世文 馬執政竟高升|publisher=自由時報 |date=2014-06-02 |accessdate=2014-06-02}}</ref>，改調內政部營建署專門委員，後升任組長。隨後又升任內政部建築研究所主任秘書。

2005年，監察委員[[廖健男|廖健男]]、[[陳進利|陳進利]]、[[林時機|林時機]]在調查後，共同提起合歡山主峰景觀改善工程糾正案，糾正案中認為此次工程未先取得土地使用權同意書與施工許可逕行施工，行為違法且有過度開發、影響生態的可能，要求行政院改善<ref>[http://kids.cy.gov.tw/AP_HOME/op_Upload/eDoc/%E7%B3%BE%E6%AD%A3%E6%A1%88/94/094000022%E9%9B%AA%E9%9C%B8%E5%9C%8B%E5%AE%B6%E5%85%AC%E5%9C%92%E6%A1%88%E7%B3%BE%E6%AD%A3%E6%A1%88%E6%96%87.PDF 合觀山主峰景觀工程糾正案]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。同年，葉世文升任內政部建築研究所副所長，並取得[[國立臺灣大學|國立臺灣大學]]管理學院碩士在職專班高階公共管理組碩士學位。

===內政部營建署署長時期===
2008年，[[馬英九|馬英九]]當選中華民國總統，[[劉兆玄|劉兆玄]]出任行政院長，[[廖了以|廖了以]]任內政部長。7月31日，因原任營建署署長[[林欽榮|林欽榮]]申請退休，回交通大學任教，葉世文升任營建署署長<ref>{{Cite web |url=http://www.dongtaiwang.com/do/Q_aZ/tttLgkjxz_AhN9Lxjh/wW/I/H/Ur/n2211923.htm|author=謝佳珍|language=zh|title=營建署長林欽榮退休 遺缺葉世文接任|publisher=大紀元 |date=2008-07-31 |accessdate=2014-06-03}}</ref>。

2009年9月10日，[[吳敦義|吳敦義]]出任行政院長。9月11日，[[江宜樺|江宜樺]]接替[[廖了以|廖了以]]出任內政部部長，葉世文續任營建署署長。吳敦義宣誓將興建平價住宅（合宜住宅的前身），由葉世文負責。

2010年，葉世文提出開發計畫，採取先招標、再徵收的方式。2011年，林口捷運A7站進行公開招標，只有遠雄、皇翔、名軒、麗寶、櫻花等五家廠商投標，多數廠商因為招標公告提出的容積率，只能蓋2451戶，難以獲利，皆未投標。經計劃審査後，櫻花被刷掉，四塊基地由四個廠商參與投標。7月6日，葉世文宣布，為獎勵廠商參與開發，提高法定容積，預估戶數増加到4385戶。7月8日，開標之後，這四家廠商正好一家標到一塊基地<ref>{{Cite web|url=http://www.stormmediagroup.com/opencms/news/detail/73a85103-ebb7-11e3-aa1f-ef2804cba5a1/?uuid=73a85103-ebb7-11e3-aa1f-ef2804cba5a1|author=謝賀潔|language=zh-tw|title=A7戶數爆增 段宜康斥不要臉的髒東西|publisher=風傳媒|date=2014-06-04|accessdate=2014-06-04|archiveurl=https://web.archive.org/web/20140606215939/http://www.stormmediagroup.com/opencms/news/detail/73a85103-ebb7-11e3-aa1f-ef2804cba5a1/?uuid=73a85103-ebb7-11e3-aa1f-ef2804cba5a1|archivedate=2014-06-06|deadurl=yes}}</ref>。

完成投標之後，政府配合徵收了機場捷運A7站週邊土地。因徵收說明不足、政府財務規劃不足，引起林口當地住戶不滿，組成自救會抗爭，徵收進度一度落後。但因為支持的人少，自救會解體，林口捷運站週邊順利被徵收。2012年林口平價住宅開始銷售，2013年開始動工。被視為是吳敦義任內政績之一<ref>{{Cite web |url=http://news.ltn.com.tw/news/focus/paper/783890|author=林美芬|language=zh|title=從規劃到開工 營建署長任內完成|publisher=自由時報 |date=2014-06-01 |accessdate=2014-06-03}}</ref>。

同年，發生[[大埔事件|大埔事件]]，行政院長吳敦義於8月17日會見苗栗大埔自救會農民，承諾原地保留，葉世文陪同出席。葉世文後續完成與大埔農民間的協調。

期間，江宜樺曾收到葉世文操守有問題的檢舉，轉交政風單位調查，以查無實據結案<ref>{{Cite web |url=http://www.nownews.com/n/2014/06/01/1259684|author=|language=zh|title=葉世文涉貪　江宜樺痛心自爆曾接黑函|publisher=NOWnews |date=2014-06-01 |accessdate=2014-06-01}}</ref><ref>{{Cite web |url=http://www.nownews.com/n/2014/06/03/1261237|author=王鼎鈞|language=zh|title=江宜樺收葉世文黑函　吳敦義任閣揆時不知情|publisher=NOWnews |date=2014-06-03 |accessdate=2014-06-03}}</ref>。在2010年與2011年，政風單位兩次接獲內部檢舉，質疑葉世文操守有問題，但仍以查無實據結案<ref>{{Cite web |url=http://udn.com/NEWS/NATIONAL/NATS6/8716912.shtml|author=蔡佩芳|language=zh|title=葉世文操守 內政部曾兩度接獲檢舉 |publisher=聯合報 |date=2014-06-03 |accessdate=2014-06-03}}</ref>。

2012年2月6日，[[陳沖|陳沖]]任行院院長，2月16日，江宜樺接替陳沖，升任行政院副院長，[[李鴻源|李鴻源]]繼任內政部長。葉世文仍續任營建署署長。

3月27日，[[監察院|監察院]]查出葉世文漏報財產，葉世文遭罰款23萬元<ref>[http://hanreporter.blogspot.tw/2012/03/23.html 數位網路報 營建署長葉世文漏報財產，遭罰23萬]</ref>。

11月6日，出席[[聯合報|聯合報]]舉辦的「房產投資高峰論壇」，主張購買房地產是最佳投資方式，鼓勵以買房取代投入股市，遭立法委員[[賴士葆|賴士葆]]與[[張金鶚|張金鶚]]等多位學者批評為帶頭[[炒房|炒房]]<ref>[http://www.libertytimes.com.tw/2012/new/nov/8/today-e1.htm?Slots=T 自由時報 營建署長買房不買股說 惹爭議 陳裕璋批不恰當]</ref><ref>[http://www.nownews.com/2012/11/07/91-2870842.htm 今日新聞 營建署長指「股市投資9成賠」　陳裕璋:不恰當]</ref><ref>[http://www.nownews.com/2012/11/07/91-2870840.htm 今日新聞 葉世文澄清非炒樓　張金鶚批：撈過界，像建築業代言人]</ref><ref>[http://news.rti.org.tw/index_newsContent.aspx?nid=387958&id=2&id2=1 中央廣播電台 談房產投資惹議 葉世文:絕非配合炒房]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://udn.com/NEWS/BREAKINGNEWS/BREAKINGNEWS1/7482238.shtml 聯合晚報 葉世文不認為自己失言]</ref><ref>[http://www.appledaily.com.tw/appledaily/article/headline/20121107/34625021/ 蘋果日報 「未來10年買房最好」署長失言挨轟]</ref><ref>[http://news.rti.org.tw/index_newsContent.aspx?nid=387967&id=1&id2=1 中央廣播電台 葉世文投資房產言論 李鴻源：非內政部立場]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://money.udn.com/house/storypage.jsp?f_MAIN_ID=414&f_SUB_ID=4054&f_ART_ID=276716 經濟日報 財部：實價課稅無時間表] {{webarchive|url=https://web.archive.org/web/20100905110944/http://money.udn.com/house/storypage.jsp?f_MAIN_ID=414 |date=2010-09-05 }}</ref>。11月28日，營建署未邀請地政學者[[張金鶚|張金鶚]]教授參加在2012年第三季住宅需求動向調查記者會，張金鶚認為可能是因為他曾批評葉世文參與鼓勵炒房的批評言論所致<ref>[http://www.nownews.com/2012/11/28/320-2877130.htm 今日新聞 批評營建署長發言遭封殺? 張金鶚：我該說的還是要說]</ref>。

2013年2月18日，[[江宜樺|江宜樺]]出任行政院長。[[李鴻源|李鴻源]]續任內政部長，葉世文續任營建署署長。期間，[[潤泰集團|潤泰集團]][[尹衍樑|尹衍樑]]曾遭葉世文索賄，尹衍樑拒絕，向廉政署及行政院長江宜樺舉報此事，但以查無實據結案。葉世文原定於當年7月屆齡退休，因[[林益世索賄案|林益世索賄案]]，江宜樺要求對所有行政院官員進行風紀調查，政風單位向江宜樺回報，查到葉世文在4月至5月間曾接受廠商邀宴。江宜樺指示內政部長李鴻源，要求葉世文提前退休，但未向李鴻源說明原因。6月3日，葉世文申請提前退休，卸任營建署長<ref>{{Cite web |url=http://www.chinatimes.com/newspapers/20140606000373-260102|author=楊舒湄|language=zh-tw|title=去年獲情資 江揆要求葉提前退休|publisher=中國時報 |date=2014-06-06|accessdate=2014-06-06}}</ref><ref>{{Cite web |url=http://n.yam.com/tlt/politics/20140606/20140606337818.html|author=鍾麗華|language=zh-tw|title=沒處理葉世文 李鴻源：江宜樺留爛攤子|publisher=自由時報 |date=2014-06-06|accessdate=2014-06-06}}</ref>。因推動板橋浮洲及林口A7合宜住宅有功，且考績甲等，獲內政部表揚為模範公務人員<ref>[http://udn.com/NEWS/BREAKINGNEWS/BREAKINGNEWS1/8716450.shtml 諷刺！葉世文去年獲內政部表揚模範公僕]</ref>。

===桃園縣副縣長時期===

2014年5月30號，[[台北地檢署|台北地檢署]]檢察官懷疑葉世文受[[遠雄|遠雄]]集團董事長[[趙藤雄|趙藤雄]]賄賂，指揮廉政署搜索其住家、辦公室，查扣可疑現金1800萬元，並將葉世文帶回偵訊；隨後桃園縣長吳志揚宣布解除葉世文任職桃園縣的所有官職，而葉世文同時也遭到法院裁定羈押禁見。<ref>[https://tw.news.yahoo.com/%E6%A1%83%E7%B8%A3%E5%89%AF%E7%B8%A3%E9%95%B7%E8%91%89%E4%B8%96%E6%96%87%E6%B6%89%E8%B2%AA-%E5%90%B3%E5%BF%97%E6%8F%9A%E4%B8%8B%E4%BB%A4%E8%A7%A3%E8%81%B7-081043507.html 桃縣副縣長葉世文涉貪 吳志揚下令解職]</ref><ref>[http://news.ltn.com.tw/news/society/breakingnews/1020163 桃縣副縣長葉世文被北檢帶走 住家被搜索]</ref>

==爭議==
===涉八德合宜住宅弊案及林口A7案弊案===
{{see also|八德合宜住宅弊案}}

2014年6月，檢方查出，在桃園八德合宜住宅案中，葉世文曾向遠雄索賄2600萬，其中1600萬在葉世文辦公室、住所中查獲。葉世文坦承收賄，但僅承認刑度較輕的「不違背職務受賄罪」。因[[蔡仁惠|蔡仁惠]]轉為[[污點證人|污點證人]]，葉世文也承認在林口A7案中，也收賄600萬。但趙藤雄否認在林口A7案曾經行賄。

2015年3月20日，台北地方法院於下午5點一審宣判，依職務上行為收受、期約賄絡及財產來源不明等罪，判處葉世文有期徒刑19年，褫奪公權5年，併科罰金3317萬，另葉世文、陳麗玲被訴「淡海新市鎮案」藉勢、藉端勒索財物獲判無罪。遠雄集團創辦人趙藤雄則依行賄等罪判處有期徒刑4年6個月，併科罰金90萬。遠雄副總經理魏春雄依行賄罪判處有期徒刑4年，併科罰金50萬，[[仲介|白手套]]蔡仁惠因轉污點證人供出葉世文向趙藤雄等人索賄過程，獲判有期徒刑6個月，併科罰金20萬。

2015年12月25日，台灣高等法院依依職務上行為收受、期約賄絡及財產來源不明等罪判處有期徒刑21年，併科罰金新台幣1000萬元，褫奪公權5年，另葉世文、陳麗玲被訴「淡海新市鎮案」藉勢、藉端勒索財物獲判無罪。遠雄集團創辦人趙藤雄則依行賄等罪有期徒刑2年，併科罰金新臺幣140萬元，緩刑5年，並向公庫支付新臺幣2億元，褫奪公權2年定讞。遠雄副總經理魏春雄依行賄罪判處有期徒刑1年10月，併科罰金新臺幣130萬元，緩刑5年，並向公庫支付新臺幣1億元，褫奪公權2年定讞。[[仲介|白手套]]蔡仁惠因轉污點證人供出葉世文向趙藤雄等人索賄過程，期約賄賂部分獲判免刑，關於不違背職務行為，交付賄賂，處有期徒刑6月，併科罰金新臺幣20萬元，緩刑2年，褫奪公權2年定讞。

2016年4月21日，最高法院將原判決關於葉世文「新竹眷村改建土地案」對於職務上行為期約賄賂、「八德合宜住宅案」對於職務上行為收受賄賂及公務員財產來源不明部分撤銷，發回台灣高等法院；另將原判決關於葉世文「機場捷運A7站合宜住宅招商投資興建（用地預標售）案」論處收受賄賂罪刑部分，駁回葉世文之上訴（有期徒刑7年，褫奪公權5年）；及檢察官就葉世文、陳麗玲被訴「淡海新市鎮案」藉勢、藉端勒索財物判決無罪定讞<ref>{{cite news|author1=項程鎮|title=葉世文林口A7合宜宅收賄 判7年定讞|url=http://news.ltn.com.tw/news/society/breakingnews/1671797|accessdate=2016-04-22|agency=自由時報|date=2016-04-22}}</ref>。
