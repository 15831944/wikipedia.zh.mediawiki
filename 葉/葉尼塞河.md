'''葉尼塞河'''（{{lang-ru|Енисе́й}}），古称'''剑河'''，是流入[[北冰洋|北冰洋]]最大的河流。比[[密蘇里河|密蘇里河]]—[[密西西比河|密西西比河]]短，但流量是前者的1.5倍。起源於[[蒙古國|蒙古國]]，朝北流向[[喀拉海|喀拉海]]，其流域範圍包含了[[西伯利亞|西伯利亞]]中部大部分地區。如以[[色楞格河|色楞格河]]—[[安加拉河|安加拉河]]為源頭計算，全長5,539 公里。

上游湍急，多急流、洪水，周圍人口稀少。[[蘇聯|蘇聯]]政府利用[[勞改|勞改]]犯人在中游修築[[水壩|水壩]]，為工業提供[[電力|電力]]，遺留了嚴重的污染問題。在穿越人口稀疏的[[針葉林|針葉林]]時，無數支流匯入。最後到達無人煙的的[[凍原|凍原]]地區——每年有超過半年時間處於冰封狀態。與其他西伯利亞的河流一樣，近年流量有上升趨勢，有指是[[全球暖化|全球暖化]]所導致。由此形成北冰洋[[鹽度|鹽度]]下降，可能透過[[海流|海流]]對全球造成影響。

== 正源 ==
葉尼塞河由兩條河流取其源頭: 波塞爾河及馬尼爾河。這两条河流在[[克孜勒|克孜勒]]聚合。葉尼塞河由西沙也群山以約七百公里的長度流過一片草原直到處於美那也的大水壩。

== 貝加爾河源 ==
[[File:Bank_of_Yenisei_River.jpg|thumb]]附近的叶尼塞河]]
上安加拉河流入[[貝加爾湖|貝加爾湖]]北部，處於[[布里亞特共和國|布里亞特共和國]] 。但它最大的源頭在[[色楞格河|色楞格河]]形成東南面的三角洲。它另一支流土拉河經過[[烏蘭巴托|烏蘭巴托]]，而'''Egiin'''支流流到 [[庫蘇古爾湖|庫蘇古爾湖]]。

由馬爾拉下游兩百多公里的萨扬－舒申斯克水电站（Sayano-Shushenskaya）大壩推動俄羅斯1989建成的最大的6400MW水力發電廠。約一百公里下游的[[阿巴坎河|阿巴坎河]]流經[[阿巴坎|阿巴坎]], 是[[哈卡斯共和國|哈卡斯共和國]]的首都，鄰近[[鄂畢河|鄂畢河]]Chulym的支流直達三百公里後的[[克拉斯諾亞爾斯克|克拉斯諾亞爾斯克]]，也是[[西伯利亞鐵路|西伯利亞鐵路]]及葉尼塞河的最大城市。近著的克拉斯諾亞爾斯克港有[[克拉斯諾亞爾斯克水電站|克拉斯諾亞爾斯克水電站]]。

[[熱列茲諾戈爾斯克(克拉斯諾亞爾斯克邊疆區)|熱列茲諾戈爾斯克]]的[[保密行政區|保密行政區]]是在七十公里下游，可能是核工業區，有可能是葉尼塞河核污染的源頭。再兩百公里下游就到了安加拉地區。

== 安加拉河 ==
安加拉河流到[[貝加爾湖|貝加爾湖]]再由[[伊爾庫茨克|伊爾庫茨克]]會流1840公里直至與葉尼塞河在舒力爾加[[Strelka|Strelka]]匯合。它已有四處建了水壩，再多建水壩因可能破壞環保而遭到抗議。

== 下游 ==
加爾河Kaz與葉尼塞河在舒力爾加三百公里下游匯合。在此區域的河有半年是冰封的，有時要用炸藥使河水流動以免破壞水壩。河流出海前的城市叫杜金卡，與克拉斯諾亞爾斯克有定時的客輪服務。河口處闊至二百五十公里，很多時要出動破冰輪保持航道暢通。

== 探險 ==
第一支探險隊是澳洲及加拿大在2001年9月創出的全游探尋。 美国國家地理雜誌也為此拍成紀錄片。

== 歷史 ==
古代游牧民族的[[凯特人|凯特人]]及尤格人緣岸而處。今天約有一千的凯特族人還在河流東中部流域生活。但已在十七至十九世紀與俄羅斯在政治上合併。

古代[[突厥|突厥]]生活在葉尼塞河南部，曾向土著[[堅昆|堅昆]]人學習冶煉。 

第二次世界大戰時，[[納粹德國|納粹德國]]與[[大日本帝國|大日本帝國]]預計要在這裡會師，但此計劃因為德日兩軍自1943年起節節敗退，所以導致最後無法實行。<ref>{{cite web
