'''莙薘菜'''（''Beta vulgaris'' subsp. ''cicla''）<ref name=trop1/>，即'''叶用甜菜'''，俗语又叫'''牛皮菜'''、'''厚皮菜'''、'''猪乸菜'''，[[莧科|莧科]][[菾屬|菾屬]]的耐寒性[[一年生植物|一年生]]或[[二年生植物|二年生]][[草本植物|草本]][[葉菜|葉菜]]，是[[甜菜|甜菜]]的一個變種。常見於[[地中海料理|地中海料理]]。

莙薘菜原产[[欧洲|欧洲]]南部，公元5世纪由阿拉伯传入中国<ref>[http://www.zwbk.org/MyLemmaShow.aspx?lid=232714 见《中文百科在线》“牛皮菜”词条。]</ref>。雖然菜葉常綠，但其葉柄顏色多變，有綠色、紅色等多種顏色<ref name="New York Times">{{cite news |url=http://topics.nytimes.com/top/news/health/series/recipes_for_health/chard/index.html |title=Recipes for Health: Chard |date=2011-11-17 |publisher=[[紐約時報|The New York Times]] |language=en }}</ref>。莙薘菜的營養價值非常高，而且經過裁培後，菜的營養都聚集到葉片，而不是像甜菜那樣聚在根莖<ref name = "WiseGeek">{{cite web |url=http://www.wisegeek.com/what-is-chard.htm |title=What is Chard? |date=2011-11-17 |publisher=[[:en:Wise_Geek|Wise Geek]] |language=en }}</ref>。因此，莙薘菜被譽為是其中一種最健康的菜，常用於健康餐單裡<ref name="WHFoods">{{cite web |url=http://whfoods.org/genpage.php?tname=foodspice&dbid=16 |title=World's Healthiest Foods: Swiss chard - What's New and Beneficial About Swiss Chard |publisher=George Mateljan Foundation |date=2011-11-17 |language=en }}</ref>。

== 烹调 ==
[[File:Bledes_amb_porro_i_all9.JPG|thumb]]
莙薘菜虽然味道微苦，但在世界各地方的料理均有出现。
在[[北美|北美]]及歐洲亦有用來做[[西式餡餅|西式餡餅]]的餡料；華南地區一般都是灼熟、用蒜頭煮或煮湯<ref>[[何廣義|何廣義]]、[[康翠瑩|康翠瑩]]，《真情‧康翠瑩》，2012年12月11日13:30 EST</ref>，用以去掉苦味，口感也媲美烫[[菠菜|菠菜]]。莙薘菜的幼苗可以用来拌色拉。
