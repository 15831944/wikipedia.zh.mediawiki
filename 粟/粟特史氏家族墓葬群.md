'''粟特史氏家族墓葬群'''或'''粟特史姓墓葬群'''是位於寧夏[[固原北朝隋唐墓地|固原北朝隋唐墓地]]的六座[[隋|隋]]-[[唐朝|唐朝]]時期的[[粟特人|粟特人]]古墓葬，於1982–1995年間陸續發掘出土<ref name=多元信仰探討>{{cite web |url=http://www.xjass.cn/ls/content/2017-09/02/content_468592.htm |title=固原史姓粟特家族多元信仰探討 |author=陸薪羽 |date=2017年9月2日 |website=www.xjass.cn |access-date=2018年8月4日}}</ref>，分別是'''史射勿墓'''（609年）、'''史道洛墓'''（658年）、'''史索岩墓'''（664年）、'''史訶耽墓'''（669年）、'''史鐵棒墓'''（670年）和'''史道德墓'''（678年），墓主均為[[昭武九姓|昭武九姓]]中的[[史國|史姓]]後代。其中衹有史射勿墓是隋朝墓葬，其餘皆是唐朝墓葬。墓中出土了帶[[中亞|中亞]]甚至[[歐洲|歐洲]]色彩的文物，比如[[薩珊王朝|波斯]]銀幣和[[拜占庭帝國|拜占庭]]{{Link-en|拜占庭貨幣|Byzantine coinage|金幣}}。此外，墓葬亦出土有墓誌、壁畫、鎭墓獸、武士俑、寶石印章等。

同公元592年的隋朝[[虞弘墓|虞弘墓]]相比，可以看出史姓墓葬群的漢化程度很高。比如虞弘墓的藝術圖像中全是高鼻深目的胡人，淸一色穿着波斯[[薩珊王朝|薩珊王朝]]風格服飾，且[[祆教|祆教]]色彩之濃郁，更高於[[北周|北周]]的[[史君墓|史君墓]]和[[安伽墓|安伽墓]]。史姓墓葬群雖然保留了祆教傳統，但同時也受到其他宗教的影響，而且服飾已完全漢化，出土壁畫中已經有漢人形像，還有[[朱雀|朱雀]]這種中國的傳統文化元素，與虞弘墓的圖像風格可謂是天壤之別。

== 家族系統 ==
[[File:Family_Tree_of_the_Sogdian_Shih_Family.jpg|thumb]]
六名墓主分屬兩個史姓家族，史射勿一系包括史射勿、史訶耽、史道洛和史鐵棒，這四人是祖孫三代。史射勿最年長，為祖輩；史訶耽、史道洛為史射勿之子，兩人是兄弟，屬於父輩，其中史訶耽為兄長<ref>{{cite web |url=http://www.aihv.org/en/pdf/15-25.pdf |title=Six-lobed Tang dynasty (AD 658) glass cups recently excavated in China |author={{Link-ja|谷一尚|谷一尚|Taniichi, Takashi}} |date=2001 |website=www.aihv.org |page=p. 3 (PDF page) |access-date=2018-08-07 |language=en}}</ref>；史鐵棒是史射勿的另一名兒子史大興之子，即史訶耽和史道洛的侄子，屬於孫輩。史索岩一系包括史索岩夫妻與史道德，兩者為叔侄關係<ref name=多元信仰探討 />。

== 史射勿墓 ==
史射勿墓位於固原南郊鄉小馬莊村西北，墓葬全長29米，1987年開始發掘。該墓多次被盜，隨葬品所存很少，僅出土有水晶、銅鏡、金帶扣、金指環、白瓷鉢、靑瓷罐、波斯銀幣等文物20餘件。不過墓室內發現保存完好的壁畫18幅，色彩鮮艷，畫面淸晰，有胡人武士圖、漢人仕女圖等<ref>{{cite web |url=http://epaper.dfsc.com.cn/html/2015-07/22/content_8_1.htm |title=寧夏隋唐古墓壁畫管窺當年社會風情 |author=孫琪 |date=2015年7月22日 |website=epaper.dfsc.com.cn |access-date=2018年8月4日}}</ref>，着裝均已漢化。據墓誌所載，墓主史射勿，字槃陁。射勿是[[祆教|祆教]]神名，槃陁意為僕人，其名和字可復原為[[粟特語|粟特文]]{{transl|sog|Zrwmßntk}}，意為祆教{{Link-en|祖爾萬|Zurvan}}神之僕。史射勿名字濃郁的宗教意味，與其先輩擔任宗教職務有直接關係。他的祖上均是粟特貴族，曾祖和祖父曾任薩寶，執掌宗教事務，說明史射勿出生於祆教世家<ref>{{cite web |url=https://www.xueshu.com/xxyj/201703/29813915.html |title=從墓葬看粟特人史射勿之華化 |author=馬秀英 |date=2017 |website=www.xueshu.com |access-date=2018年8月4日}}</ref><ref>{{cite web |url=http://www.siluyou.org/index/citywwinfo/id/176.html |title=史射勿墓誌、誌蓋 |author1=楊淸越 |author2=鄭小紅 |website=www.siluyou.org |access-date=2018年8月4日}}</ref>。他自幼聰慧，性格豪爽且勇力過人。自[[北周|北周]][[天和_(北周)|天和]]元年（公元566年）起，史射勿就隨軍征戰，時年僅23歲，後來出任[[隋朝|隋朝]]正義大夫右領軍[[驃騎將軍|驃騎將軍]]。隋[[大業_(年號)|大業]]五年（609年）於家中病逝，終年66歲<ref name=粟特人在寧夏>{{cite web |url=http://sz.nxnews.net/xxxb/21/2014-08/25/20/2014082520_pdf.pdf |title=粟特人在寧夏 |author1=王俊蓓 |author2=毛文靜 |date=2014年8月25日 |website=sz.nxnews.net |access-date=2018年8月4日}}</ref>。

== 史道洛墓 ==
史道洛墓位於史射勿墓東北100米，是史道洛與其妻康氏的合葬墓，於1995年由中日合作的原州聯合考古隊發掘出土。該墓被盜嚴重，加上進水和墓葬坍塌，導致墓主遺骨和動物骨骼散落在墓葬各處。墓中出土有完整墓誌、玻璃器皿、大型彩繪胡人武士俑同人面鎭墓獸、獸面鎭墓獸各一尊，其中人面鎭墓獸的設計理念同[[古埃及|古埃及]]的[[獅身人面像|獅身人面像]]類似<ref>{{cite web |url=http://nx.duost.com/20150602/148142.html |title=沈睡固原的「洋人」們 |author=<!--Staff writer(s); no by-line.--> |website=nx.duost.com |access-date=2018年8月5日}}</ref>。據墓誌記載，史道洛曾是大唐左親衞，唐[[顯慶|顯慶]]三年（658年）與妻康氏合葬<ref name=粟特人在寧夏 />。

== 史索岩墓 ==
史索岩墓於1984年在羊坊村北發現，全長41.75米，為夫妻合葬墓。由於被盜，出土文物不多，有數件瓷器、銅鏡、玉釵、陶靴、金銀幣等。墓中發現史索岩及其妻子安娘的墓誌兩方，其上記載史索岩曾任平涼郡都尉驃騎將軍，同史射勿一樣，也是戎馬一生<ref name=粟特人在寧夏 />。其妻安娘是[[安息帝國|安息]]王室後裔。

== 史訶耽墓 ==
史訶耽墓位於南郊鄉小馬莊村，1986年發掘，也是夫妻合葬墓。墓中出土墓誌、石門、石棺牀、玻璃花、藍色圓形寶石印章等，石門上有異常精美的裝飾圖案。其中的寶石印章也很特別，印章中間刻一臥獅，獅子背部有三杈結果實的短枝，周圍有一圈字母銘文。經[[伊朗|伊朗]]學者與古代[[波斯|波斯]]地區的類似印章對比考證，認為獅子代表太陽，三杈枝代表生命樹。有學者認為上面的銘文是[[中古波斯語|中古波斯文]]，釋讀為「高貴」、「繁榮興旺」和「幸福快樂」。還有學者認為應解釋作「世界寬容！世界寬容！世界寬容！」。最後由[[德黑蘭大學|德黑蘭大學]]的三位教授{{transl|fa|Rezaaibaghbidi}}、{{transl|fa|M.B. Vosooghi}}、{{transl|fa|Amuzegar}}釋讀，他們認為銘文的意思是「虔誠的、敬神的、可嘉的、盡責的」。無論是哪一種解釋，都顯然有很濃的宗教意味。印章上的三杈果實被認為是石榴，石榴和[[祆教|祆教]]禮儀關係密切，古波斯人認為石榴是「太陽的聖樹」，有多子豐饒之意。祆教女神{{Link-en|阿什 (祆教)|Ashi|阿什}}的對應動植物就分別是獅子和石榴，傳說阿什女神還可以令「所養的馬成千倍的增長」。聯繫到史訶耽熟知六閒（馬廄），曾專門負責監管養馬業，而且其墓門門額上刻的馬匹圖案非常顯著，最終推測這枚印章上的圖案和阿什女神崇拜有關<ref>{{cite web |url=http://www.kaogu.cn/uploads/soft/2016/20160115guyuan.pdf |title=固原史訶耽夫妻合葬墓所出寶石印章圖案考 |author=郭物 |date=2015 |website=www.kaogu.cn |access-date=2018年8月4日}}</ref>。除了善於養馬，史訶耽由於精通多種外語，亦任職中書省翻譯，可稱是唐朝翻書譯語直官的代表人物，任職時間長達40多年<ref>{{cite web |url=https://www.xueshu.com/jinyangxuekan/201605/27372914.html |title=唐代的翻書譯語直官：從史訶耽墓誌談起 |author=李錦繡 |date=2016年1月28日 |website=www.xueshu.com |access-date=2018年8月4日}}</ref>。

墓誌載史訶耽是[[史國|史國]]國王的後裔，又據歷史記載，他幼年好學，靑年時就已博學多才，且品行高尙，受人敬仰。據說他身姿矯健、相貌堂堂，講話聲音「淸言直流如水之聲」，十分優美。隋朝末年，社會動盪，反隋勢力四起，史訶耽選擇赴[[長安|長安]]投唐，於唐朝歷任「左二監」（監管養馬業）及「直中書省翻譯朝會」等職務。[[唐高宗|唐高宗]]時，曾作為翻譯隨同前往泰山[[封禪|封禪]]。[[乾封|乾封]]元年（666年）敕授虢州刺史，但史訶耽因懷念往昔生活而不願接受刺史官職。高宗因其年事已高，便准予他隨自己的心願和雅致生活，並賞賜物品50段。[[總章|總章]]二年（669年），史訶耽在原州平高縣勸善里舍逝世，終年86歲。其首任妻子康氏是[[康居|康國]]人，早於[[貞觀_(唐朝)|貞觀]]四年（630年）逝世於長安延壽里。後妻張氏，漢人，授「南陽縣君」，去世後與史訶耽合葬<ref>{{cite web |url=http://www.guyuantv.com/rd/201705/t20170524_4275048.html |title=這位從固原走出的國王之後裔，明明可以靠顏値吃飯，卻偏不！ |author=固原電視臺 |date=2017年5月24日 |website=www.guyuantv.com |access-date=2018年8月4日}}</ref>。

== 史鐵棒墓 ==
史鐵棒墓位於南郊鄉小馬莊村，1986年發掘。因早年被盜，出土隨葬品較少，有墓誌及陶、銅、金等質地的文物20餘件。史鐵棒是史射勿之孫，史訶耽之侄。據墓誌記載，史鐵棒曾是皇帝貼身保鑣，後任司馭寺右十七監，是管理馬政的官員，為朝廷擔當養馬重任。他長期牧馬，經驗豐富，墓誌亦提到他牧馬的水平很高<ref name=粟特人在寧夏 />。

== 史道德墓 ==
史道德墓位於[[固原縣|固原縣]]南郊鄉王澇壩村，1982年發掘，出土有墓誌、陶器、金帶扣、金覆面、獸面金飾、動物紋圓形金飾、素-{面}-小銅鏡等文物20餘件。墓誌記載，史道德由東宮左勛衞官至蘭池正監，也是管理馬政的官員。此外，經考古發現，史道德與史索岩之妻安娘的府第在同一里坊，說明史氏粟特人有聚族而居的習慣<ref name=粟特人在寧夏 />。

== 宗教信仰 ==
[[File:ChosroesHuntingScene.JPG|thumb]]狩獵場景，其所戴帝冠，上有雙翼，雙翼上飾以新月日輪。]]
發源自古代[[波斯|波斯]]的[[瑣羅亞斯德教|瑣羅亞斯德教]]（在中國叫做祆教）是大多粟特人的傳統信仰，其次[[景教|景教]]和[[佛教|佛教]]在粟特人中也有可觀的影響力。根據史氏家族墓中出土的文物，再結合歷史文獻，可見這兩個史姓家族除了傳統的祆教信仰，亦受到佛教甚至[[儒家思想|儒家思想]]的影響。

其中史索岩和史道德雖為叔侄，但兩者卻有不同的信仰。史索岩墓出土了許多反映墓主信仰佛教的文物，如呈[[須彌座|須彌座]]形狀的棺牀、[[經幢|石幢]]，以及墓誌中記載的史索岩夫妻四名兒子的名諱法僧、德僧、德威、神義，都體現出濃郁的佛教思想。史索岩的侄子史道德墓中出土了與祆教相關的文物，如金覆面中的額飾上為新月托住日輪的形狀。發掘粟特古城[[彭吉肯特|彭吉肯特]]遺址的考古學家就注意到「[[粟特火祆教|粟特火祆教]]的特點是其保留了當地的古代祭祀元素」，即對日月的崇拜<ref name=多元信仰探討 />。新月日輪是一個極具標誌性的符號，除了常見於中亞粟特藝術，這種符號亦是以瑣羅亞斯德教為國教的波斯[[薩珊王朝|薩珊王朝]]君主帝冠上的固定裝飾。

史射勿墓中出土薩珊{{Link-en|卑路斯一世|Peroz I|卑路斯}}銀幣一枚，正面有薩珊皇帝像，背面有祆教火壇圖像。史射勿之子史訶耽墓出土一枚寶石印章，上刻圖案與祆教女神阿什崇拜有關。而史射勿之孫史鐵棒墓葬卻發現有儒家思想元素。由史鐵棒墓誌可知他的兩名兒子叫做孝忠、孝義，完全是儒家忠孝觀的體現<ref name=多元信仰探討 />。
