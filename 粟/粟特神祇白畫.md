'''粟特神祇白畫'''（[[法語|法語]]：{{lang|fr|Deux divinités féminines}}；[[英語|英語]]：{{lang|en|Sogdian Deities}}）是一幅[[唐朝|晚唐]]至[[五代|五代]]時期的紙本[[白描畫|白畫]]，描繪[[粟特|粟特]]人崇拜的兩名[[祆教|祆教]]神祇。該畫由法國探險家[[伯希和|伯希和]]在[[莫高窟|莫高窟]]發現，今藏巴黎[[法國國家圖書館|法國國家圖書館]]。

== 分析 ==
[[File:Dunhuang_mogao_mural_cave_409.jpg|thumb]]
[[饒宗頤|饒宗頤]]在他的著作《敦煌白畫》一書中提到該圖：「（編號為 {{lang|en|P.4518 (24)}} 的白畫）繪二女相向坐，帶間略施淺絳，顏微着赭色，頰塗兩暈，餘皆白描。一女手持蛇蠍，側有一犬伸舌，舌設朱色。一女捧杯盤，盤中有犬。紙本已汙損，懸掛之帶尙存。」這幅畫的風格雖已漢化，但畫裡仍有很多異於中原文化的符號，左邊神像的着裝亦明顯同當時中土流行的服飾不同<ref name=DDF />。近年在中亞[[粟特|粟特]]和[[于闐|-{于}-闐]]地區發現的祆教圖像中，常有手托小型動物（羊、駝、犬）的圖像，與該畫內容頗多關聯<ref>{{Cite web |url=http://www.izhsh.com.cn/doc/10/3175.html |title=介休祆神樓木雕「牛神」形像考辨─兼與姜伯勤先生商榷 |accessdate=2018年11月24日 |author1=王偉 |author2=張鐵梅 |date=2015年3月10日 |website=www.izhsh.com.cn}}</ref>。

這幅白畫經[[中山大學|中山大學]]歷史系教授[[姜伯勤|姜伯勤]]考證，確認是祆教圖像，畫中二女神為粟特神祇。他還認為右邊手持日月蛇蠍的四臂神像是祆教的娜娜女神（{{lang|ae-Latn|Nanā}}），他運用[[圖像學|圖像學]]及[[圖像誌|圖像誌]]的方法研究各種娜娜女神的圖像和文獻來充實這個觀點，並闡釋了敦煌[[賽祆|賽祆]]的情景同粟特祆神紙本畫像的運用{{refn|group=腳註|賽祆是一種祭祀活動，有祈福、酒宴、歌舞、幻術、化妝遊行等盛大場面，是粟特胡商「琵琶鼓笛，酣歌醉舞」的廟會式娛樂活動。祆神祭祀時的迎神曲，稱做穆護歌，隋朝樂府中已有「穆護砂」，宋人[[姚寬|姚寬]]曾說「穆護原為祆廟賽神之曲」<ref>{{Cite web |url=https://nuoha.com/book/158673/00114.html |title=敦煌文化中的祆教、摩尼教和景教 |accessdate=2018年11月24日 |author=顏廷亮 |website=nuoha.com}}</ref>。}}<ref>{{Cite web |url=http://nansha.schina.ust.hk/Article_DB/sites/default/files/pubs/Jour-03.2.B02.pdf |title=姜伯勤《中國祆教藝術史》書評 |accessdate=2018-07-24 |author=吳羽 |date=2004 |website=schina.ust.hk}}</ref>。歷史學家[[張廣達|張廣達]]認為左邊女神象徵祆教善神{{Link-en|妲瑞娜|Daena}}，右邊則代表惡神[[德弗|妲厄媧]]。姜伯勤雖接受張廣達關於妲瑞娜的說法，但他確定右邊神像乃是女神娜娜<ref>{{Cite web |url=http://big5.xjass.cn/ls/content/2013-11/22/content_304687.htm |title=敦煌與中外關係史研究三十年 |accessdate=2018年11月24日 |author=王旭送 |date=2013年11月22日 |website=xjass.cn}}</ref>。

娜娜女神是源自[[美索不達米亞|美索不達米亞]]的神祇，<ref>{{Cite book |last1=Gorshenina |first1=Svetlana |last2=Rapin |first2=Claude |date=2001 |title=''De Kaboul à Samarcande : Les archéologues en Asie centrale'' |series=Collection « [[Découvertes_Gallimard|Découvertes Gallimard]] » |volume=411 |location=Paris |publisher=Éditions Gallimard |page={{p.|107}} |chapter=Chapitre 5 : Des Kouchans à l'Islam – La peinture sogdienne |language=fr |isbn=978-2-070-76166-1 |quote=Dans la peinture on rencontre ainsi des versions du ''Panchatantra'', l'histoire de la déesse Nana originaire de Mésopotamie.}}</ref>據說她的原型是[[蘇美|蘇美]]的性愛、豐產及戰爭女神[[伊南娜|伊南娜]]，後來融入祆教神祇之列。[[薩珊王朝|薩珊王朝]]時期，對娜娜女神的崇拜被粟特商人帶入東亞，漢化後叫做「胡天南太后」<ref>{{Cite web |url=http://www.inmediahk.net/%E5%A8%9C%E5%A8%9C%E5%A5%B3%E7%A5%9E%E2%80%94%E2%80%94%E6%B3%A2%E6%96%AF%E3%80%81%E5%B8%8C%E8%87%98%E3%80%81%E5%8D%B0%E5%BA%A6%E3%80%81%E4%BC%8A%E6%9C%97%E3%80%81%E4%B8%AD%E5%9C%8B%E7%9A%84%E5%85%B1%E5%B4%87%E7%A5%9E%E7%A5%97 |title=娜娜女神——波斯、希臘、印度、伊朗、中國的共崇神祇 |accessdate=2018年11月24日 |date=2011年6月19日 |website=inmediahk.net}}</ref><ref>{{Cite web |url=https://www.edb.gov.hk/attachment/en/curriculum-development/kla/pshe/belt-and-road-initiative/Ancient_Silk_Road_Nomadic_Ethnic_Group.pdf |title=古代絲綢之路：商旅民族的前世今生 |page=72 |accessdate=2018年11月24日 |date=2016年11月9日 |website=edb.gov.hk}}</ref>。該畫女神頭部佩戴的桃形冠可能受到[[回鶻|回鶻]]頭飾的影響<ref>{{cite journal |last1=Russell-Smith |first1=Lilla Bikfalvy |date=2003 |title=Wives and Patrons: Uygur Political and Artistic Influence in Tenth-century Dunhuang |url=https://archive.org/stream/2003AAAH56401-428WivesPatrons/2003%20AAAH%2056%20401%E2%80%93428%20Wives%20%26%20Patrons#page/n10/mode/1up |journal=Acta Orientalia Academiae Scientiarum Hungaricae |volume=56 |issue=2/4 |page=411 |language=en |accessdate=2018-11-24}}</ref>。
