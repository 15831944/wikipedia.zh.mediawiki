'''陳蔚琦'''（{{lang-en|Tiffany Vicky Chan}}，{{bd|1994年|3月31日||}}）， 暱稱'''TIFF'''，是「電音實驗廚房」的創辦人兼行政主廚。2015年，她以最優等榮譽畢業於[[波士頓|波士頓]][[伯克利音樂學院|伯克利音樂學院]]，取得流行音樂創作學士學位。陳蔚琦擅長烹調茶系融合菜如抹茶甜品。 她曾於2017年以唱作新人身份眾籌推出單曲，現已轉戰飲食業為工作重心。<ref name="bigbigchannel">{{Cite web|website = BBC |author = | title = TIFF 陳蔚琦的 Big Big Channel 個人主頁|  |date = 2017年7月23日 |volume = |issue = |pages = |doi = |url=  https://mobileapp.bigbigchannel.com.hk/web/users/hk/PxnZWgxv }}{{zh}}</ref>
<ref name="oncc_20130707">{{Cite newspaper |newspaper = 東方日報 |author = |title =甜品客集資出碟　陳蔚琦恨同盧凱彤合作 |date = 2017年7月7日 |volume = |issue = |pages = |doi = |url =http://hk.on.cc/hk/bkn/cnt/entertainment/20170707/bkn-20170707155511274-0707_00862_001.html }}{{zh-hant}}</ref>

== 在學時期 ==
陳蔚琦自小學習古典音樂，於2005至2012年就讀[[香港演藝學院|香港演藝學院]]青少年音樂課程，主修[[小號|小號]]，師承[[香港管弦樂團|香港管弦樂團]]聯合首席小號手莫思卓及銅管樂講師裘德禮。<ref name="apa_201312">{{Cite web|website =HKAPA|author =|title =香港演藝學院通訊|4 =|date =2013年12月|volume =|issue =|pages =|doi =|url =https://www.hkapa.edu/f/page/5898/12news13.pdfl}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}{{zh-hant}}</ref>求學時期在多屆校際音樂節獲得獎項，包括第63屆[[香港校際音樂節|香港校際音樂節]] - 管樂協奏曲公開組冠軍。<ref name="etvonlinehk">{{Cite web|website =etvonline|author =|title =香港校際音樂節優勝者音樂會|4 =|date =2011年10月24日|volume =|issue =|pages =|doi =|url =http://www.etvonline.hk/web/musicfestival/2011/musicfestival2011_epi06.htmll}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}{{zh-hant}}</ref>她於中學時期開始對流行音樂產生興趣，於中學畢業後考上[[波士頓|波士頓]][[伯克利音樂學院|伯克利音樂學院]]，並獲取奬學金，主修流行音樂創作。<ref name="881903com">{{Cite web|website =881903.com|author =|title =周末一圈圈-午後Live House|4 =|date =2017年7月8日|volume =|issue =|pages =|doi =|url =http://www.881903.com/Page/ZH-TW/Pro881_4221.aspxl}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}{{zh-hant}}</ref><ref name="rthk">{{Cite web|website = RTHK |author =RTHK | title = 有冇攪錯 Part 3  |  |date = 2017年9月7日 |volume = |issue = |pages = |doi = |url= http://www.rthk.hk/radio/radio2/programme/RightableWrong/episode/445165l }}{{zh-hant}}</ref> 

2013年9月，陳蔚琦於美國在學期間，憑原創歌曲《Lost and Found》贏得美國{{link-en|BMI Foundation}} John Lennon Scholarship 第三名，亦是此獎項成立16年來首位得獎的外國學生。<ref name="bmi">{{Cite web|url=https://www.bmi.com/news/entry/16th_annual_john_lennon_scholarship_winners_announced_foundationl|title=16th Annual John Lennon Scholarship Winners Announced|author=|date=2013年10月11日|pages=|doi=||website=BMI|volume=|issue=}}{{en}}</ref> 2015年，她用3年時間完成4年大學課程且以最優等榮譽畢業於[[波士頓|波士頓]][[伯克利音樂學院|伯克利音樂學院]]，取得流行音樂創作學士學位。<ref name="Chef Tiff's Kitchen Official Website">{{Cite web|website = Chef Tiffs Kitchen Official Website|author = | title = About Chef Tiff's Kitchen |  |date = 2017年 |volume = |issue = |pages = |doi = |url= http://cheftiffskitchen.wixsite.com/home/about }}{{zh-hant}}</ref>及後定居於香港至今。

== 飲食事業 ==

=== 經營食品品牌 ===
2015年10月，陳蔚琦因要跟隨為健身而設的嚴格餐單而開始研究低糖甜品，經朋友鼓勵下成立了網上甜品品牌「電音實驗廚房」，以推廣抹茶和健康飲食文化。初期只屬玩票性質，卻因受到不少媒體及食物博客推介報導，品牌在短時間內漸漸建立起來。品牌以創意茶系融合菜見稱，並以抹茶甜品最為人熟悉。雜誌[[ELLE|ELLE]] HK更曾冠以陳蔚琦 '香港的「抹茶天后」' 稱號。<ref>{{Cite web|url=https://www.facebook.com/elleOnlineHK/videos/345587702661557/|title=ELLE HK - 【自製全港最好食的抹茶雪球！ 😋】香港的「抹茶天后」Tiff，所製作的抹茶雪球牛油味超香！|accessdate=2019-01-14|author=|date=|work=www.facebook.com|publisher=|language=en}}</ref>

在2017年12月，她於[[無綫電視|無綫電視]][[Big_Big_Channel|Big Big Channel]]擁有屬於自己的烹飪節目《Big Big Chef - 電音廚房》，短短播出兩個月總點撃率接近30萬。<ref>{{Cite web|url=https://cheftiffskitchen.wixsite.com/home/about|title=Chef Tiff's Kitchen Website ABOUT|accessdate=2019-01-14|author=|date=|work=home|publisher=|language=en}}</ref>為慶祝三周年誌慶，電音實驗廚房於2018年8月2至18日在[[元創方|元創方]]設立首間快閃餐廳，名為'T LAB (テ ラボ)'。快閃餐廳店名的'T'解作'TEA' （茶）及她的暱稱 'TIFF'，意旨'TIFF與茶的實驗廚房'。餐廳以推出注入本土特色的融合茶系菜單為賣點。<ref name="Chef Tiff's Kitchen Official Website" /><ref>{{Cite web|url=https://hk.lifestyle.appledaily.com/lifestyle/realtime/article/20180801/58507723|title=【PMQ甜品店】90後音樂系回流生開網店 自創抹茶雞批|accessdate=2018-08-20|work=Apple Daily 蘋果日報}}</ref> 'T LAB (テ ラボ)'最初為單次活動，但因第一季反應熱烈，最終T LAB延續為快閃系列並定期於不同地方舉行，每次也因應時令或節日推出特式融合茶系菜單。

== 音樂事業 ==

===出道及經歷===
陳蔚琦曾於2012年7月以獨立音樂人身份推出全創作[[迷你專輯|迷你專輯]]《Rockertude》。這張[[迷你專輯|迷你專輯]]的主打歌《夏日悠想》曾在台灣[[KKBOX|KKBOX]]的廣東歌日榜排名第14位。而《Rockertude》還連續三個月登上台灣KKBOX廣東唱片TOP100大榜。<ref>{{cite web |title=TIFF CHAN FACEBOOK |url=https://www.facebook.com/pg/TiffChanOfficial/about/?ref=page_internal |website=Facebook |publisher=Facebook}}</ref> 。她在中學畢業後考入[[波士頓|波士頓]][[伯克利音樂學院|伯克利音樂學院]]，其後遠赴[[美國|美國]]進修。

2015年於[[伯克利音樂學院|伯克利音樂學院]]畢業後，陳蔚琦回到香港從事幕後音樂工作，如為歌手錄音，灌錄歌曲小號部分等等。<ref name="881903com">{{Cite web|website =881903.com|author =|title =周末一圈圈-午後Live House|4 =|date =2017年7月8日|volume =|issue =|pages =|doi =|url =http://www.881903.com/Page/ZH-TW/Pro881_4221.aspxl}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}{{zh-hant}}</ref> 其間，因「電音實驗廚房」的客人知道她的的理想是做音樂而發起眾籌計劃，以食物及音樂作回饋，讓陳蔚琦能順利推出單曲及出道。<ref name="oncc_20130707">{{Cite newspaper |newspaper = 東方日報 |author = |title =甜品客集資出碟　陳蔚琦恨同盧凱彤合作 |date = 2017年7月7日 |volume = |issue = |pages = |doi = |url =http://hk.on.cc/hk/bkn/cnt/entertainment/20170707/bkn-20170707155511274-0707_00862_001.html }}{{zh-hant}}</ref><ref name="881903com">{{Cite web|website =881903.com|author =|title =周末一圈圈-午後Live House|4 =|date =2017年7月8日|volume =|issue =|pages =|doi =|url =http://www.881903.com/Page/ZH-TW/Pro881_4221.aspxl}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}{{zh-hant}}</ref> 直至2018年，陳蔚琦因飲食事業日漸繁忙，決定轉戰飲食業和以此為工作重心。她除了解除與Electone Music工作室簽下的歌手/音樂人合約外，亦於同年8月和無綫電視 Big Big Channel解除藝員合約。<ref>{{Cite web|url=https://www.instagram.com/p/BmsFYsKBaoA/?taken-by=tiffchanofficial|title=TIFF 陳蔚琦 'ティフ' 在 Instagram 上发布：“再見TVBuddy，把最後的牽掛也解脫了💨  還是很感謝年多前撿我回來的老闆Herman幫我完成音樂夢想， 這段期間和各位BBC同事及音樂達人也唱遊得很快樂，是一段值得珍藏的美好回憶！@kyjac @sunset.or.rise @lawmingka @pamsmusic…”|accessdate=2018-08-20|work=Instagram|language=zh-cn}}</ref>
