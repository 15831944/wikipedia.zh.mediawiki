'''陳福陽'''（{{lang-en|'''Tan Hock Eeng'''}}，{{bd|1953年||}}），出生在[[馬來西亞|馬來西亞]]華人聚集地[[檳城|檳城]]<ref>{{Cite news|url=http://finance.sina.com/bg/tech/sinacn/20171105/16451670591.html|title=這個低調的馬來西亞華人是半導體行業最凶猛的大鰐|author=鄭峻|date=|work=|accessdate=}}</ref>，現任[[博通有限公司|博通有限公司]]執行長<ref name = 'apple'>{{Cite news|url=https://tw.appledaily.com/headline/daily/20171107/37838041|title=出身大馬貧戶 博通CEO受封「兇猛大鱷」 {{!}} 蘋果日報|newspaper=蘋果日報|accessdate=2017-12-21|language=zh-TW}}</ref>。陳福陽與妻子Lisa Yang兩人育有三子，其中兩個小孩受自閉症所擾，於是在2017年捐給麻省理工學院兩千萬美元做研究自閉症的基金，試圖找出新的治療方案。<ref>{{Cite news|url=https://www.bnext.com.tw/article/48472/the-unorthodox-gambit-of-hock-tan-chief-executive-of-broadcom|title=收購高通的神秘華人，半導體界凶猛大鱷是他——博通CEO陳福陽｜數位時代|date=2018-03-13|newspaper=數位時代|accessdate=2018-04-22}}</ref>

== 人物經歷 ==
陳福陽畢業於[[美國|美國]][[麻省理工學院|麻省理工學院]]（MIT），擁有机械工程系的学士和硕士学位。後來又在[[哈佛大學|哈佛大學]]取得企管碩士學位。<ref>{{Cite news|url=http://k.sina.cn/article_1496813671_59379067001001wps.html|title=陈福阳 半导体并购之王|author=|date=|work=|accessdate=}}</ref>因此，陳福陽具備了理工背景外，也熟知財務管理與企業經營。他先后进入[[通用汽车|通用汽车]]、[[百事可乐|百事可乐]]等美国传统行业巨头，担任財務高管。1983年至1992年，陈福阳先后在在休姆工业和新加坡风投基金Pacven投资公司任董事总经理。1992年陈福阳离开Pacven投资公司，轉赴個人電腦製造商{{link-en|康懋達服務|commodore international}}（{{lang-en|Commodore}}）擔任公司副總裁，此时陳福陽才開始进入科技行业。

1994年，陈福阳加盟半导体解决方案公司（{{lang-en|Integrated Circuit Systems Inc.}}，简写ICS），先后担任公司高级副总裁、CFO、COO、总裁兼CEO职位。后来ICS作价17亿美元出售给了硅谷半导体公司集成设备技术公司（{{lang-en|Integrated Device Technology, Inc.}}，简称IDT），陈福阳也因此进入IDT，在2005年至2008年担任IDT董事会主席。同时，在2006年3月至2016年2月期间，陈福阳担任[[新加坡|新加坡]]類比IC廠安華高總裁。<ref name = 'apple'/><ref>{{Cite news|url=https://money.udn.com/money/story/5612/2798597|title=博通併高通／博通執行長陳福英 最會做生意的工程師|author=|last=經濟日報|date=|work=|newspaper=經濟日報|accessdate=2017-12-21}}</ref>並在2016年帶領安華高科技以370億美元併購博通科技後，重組公司裁員1900人後，又收購通訊大廠博科，果斷又強悍的性格，將公司整併為博通科技，成為全球第五大半導體廠。<ref>{{Cite news|url=https://www.bnext.com.tw/article/48472/the-unorthodox-gambit-of-hock-tan-chief-executive-of-broadcom|title=收購高通的神秘華人，半導體界凶猛大鱷是他——博通CEO陳福陽｜數位時代|date=2018-03-13|newspaper=數位時代|accessdate=2018-04-22}}</ref>並於2017年以每股70美元的價格欲收購全球最大手機晶片廠高通的“以小吃大”提案，引起世人注目。<ref>{{Cite news|url=https://news.cnyes.com/news/id/3955786|title=博通週一將提議以每股70美元收購  高通：太低了 {{!}} 鉅亨網 - 美股|date=2017-11-06|newspaper=鉅亨網|accessdate=2018-04-22|language=zh-TW}}</ref><ref>{{Cite news|url=https://gvlf.gvm.com.tw/article_content_17662.html|title=遠見華人精英論壇 - 博通再次收購高通的啟示|last=葉銀華|newspaper=遠見華人精英論壇 {{!}} Global Views Leaders Fourm|accessdate=2018-04-22|language=zh-TW}}</ref>一直在業界保持低調的陳福陽，為了此合併案進入白宮與川普見面並出席記者會，宣布將全球總部遷往美國，呼應川普的「美國優先」政策，但於2018年 3月12日被美國總統川普以「國安考量」為由，簽署行政命令叫停此合併案。<ref>{{Cite news|url=http://technews.tw/2018/03/15/broadcom-ceo-hock-tan/|title=收購高通的神祕華人，半導體界兇猛大鱷是他──博通 CEO 陳福陽|newspaper=TechNews 科技新報|accessdate=2018-04-22|language=zh-TW}}</ref>
