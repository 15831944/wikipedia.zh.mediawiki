'''陳永康'''（1951年－），生於[[臺灣|臺灣]][[臺北市|臺北]]，籍貫[[廣東省_(中華民國)|廣東]][[吳川市|吳川]]<ref name="籍貫"/>，[[中華民國海軍|中華民國海軍]][[二級上將|二級上將]]退役，[[美國海軍|美國海軍]][[戰爭學院|戰爭學院]]1997年班，與前[[行政院退輔會|行政院退輔會]]主委[[董翔龍|董翔龍]]同為[[海軍官校|海軍官校]]1974年班（63年班）唯二的[[上將|上將]]，任內大力推動[[潛艦國造|潛艦國造]]、[[中華民國海軍|海軍]]內部改革與多次訪美<ref>{{cite web |url=http://www.chinatimes.com/realtimenews/20141212000020-260401 |title=夢醒時刻 海軍司令陳永康訪美 |publisher=中時電子報 |author=劉屏 |language=zh-tw |accessdate=2014-12-12 }}</ref>。

== 簡歷 ==
曾於1999年12月至2003年1月間擔任[[駐美國臺北經濟文化代表處|駐美代表處]]軍事協調組少將組長（[[駐外武官|武官]]）<ref name="駐美">{{Cite news |title=駐美軍事協調組長 將由王明義接任 |language=zh-tw |page=第4版 |work=[[中央通訊社|中央社]]電 |date=2002-11-27 |publisher=[[聯合報|聯合報]]}}</ref>，亦曾接受{{Link-en|美國海軍學會|United States Naval Institute}}[[月刊|月刊]]《Proceedings》採訪並向[[美軍|美軍]]介紹[[中華民國海軍|中華民國海軍]]，陳永康在文章中提及，[[中華民國海軍|中華民國海軍]]正在經歷「復興」(renaissance)階段，建軍將秉持'''「3A」'''原則，3A指的是：「合理預算」(affordable)、「符合效益」(applicable)、「國人信賴」(accountable)，並承擔'''2S'''及'''2P'''任務，2S指的是：「捍衛(safeguard)主權」、「擱置(shelve)爭議」，2P指的是：「追求(pursue)和平互惠」、「主張(promote)共同開發」而聞名[[華府|華府]]與國內外軍政界<ref>{{cite web |url=https://video.udn.com/news/289841 |title=受美海軍學會月刊邀約 陳永康介紹我國海軍 |publisher=聯合影音 |author=程嘉文 |language=zh-tw |accessdate=2015-03-16 }}</ref><ref>{{cite web |url=http://www.chinatimes.com/newspapers/20150318000957-260302 |title=陳永康：3A海軍 迎未來挑戰 |publisher=中時電子報 |author=劉屏 |language=zh-tw |accessdate=2015-03-18 }}</ref><ref>{{cite web |url=http://www.cna.com.tw/news/aopl/201503160461-1.aspx |title=陳永康：海軍革新貢獻區域穩定 |publisher=中央通訊社 |author=廖漢原 |language=zh-tw |accessdate=2015-03-16 }}</ref>。曾任[[中華民國國防部|中華民國國防部]]軍政副部長<ref>{{cite web |url=http://news.ltn.com.tw/news/politics/paper/859801 |title=陳永康 接任國防部軍政副部長 |publisher=自由時報 |author=鍾麗華、羅添斌 |language=zh-tw |accessdate=2015-03-04 }}</ref>。
