'''陳建安'''（{{lang|en|'''Chen Chien-An'''}}，{{bd|1991年|6月16日||}}），出生于[[新竹縣|新竹縣]]，是[[台灣|台灣]][[桌球|桌球]]國手<ref>{{Cite web |url=http://www.g4ssport.com/en/G4S%204Teen/Meet%20Our%20Team/Chien%20An%20Chen/ |title=Chien An Chen |publisher=G4S |accessdate=2013-05-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20120717015224/http://www.g4ssport.com/en/G4S%204Teen/Meet%20Our%20Team/Chien%20An%20Chen/ |archivedate=2012-07-17 }}</ref>。2015年3月世界排名第21。

== 简介 ==
2008年，陳建安在西班牙馬德里贏得「第6屆世界青少年桌球錦標賽」男單冠軍<ref>{{Cite web |url=http://www.ittf.com/_front_page/ittf_full_story1.asp?ID=17139&Competition_ID=1732& |title=Chen Chien-An Collects Gold Medal for Chinese Taipei |author=Marius Widmer |publisher=ITTF |date=2008-12-13 |accessdate=2013-05-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20120824074845/http://www.ittf.com/_front_page/ittf_full_story1.asp?ID=17139&Competition_ID=1732& |archivedate=2012-08-24 }}</ref>，成為桌壇首位世青男單摘金的台灣選手。2013年，陳建安與[[莊智淵|莊智淵]]搭擋，為台灣代表隊首度在桌球頂級賽事[[2013年世界乒乓球錦標賽|世界桌球錦標賽]]拿下男雙冠軍<ref>{{cite web |url=http://www.cna.com.tw/News/aCN/201305200003-1.aspx |title=莊智淵陳建安勇奪世桌賽冠軍 |date=2013-05-20 |work=中央社新聞 |publisher=中央通訊社  |language=zh-tw  |accessdate=2013-05-20 }}</ref>。
