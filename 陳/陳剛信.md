'''陳剛信'''{{BD|1955年|5月31日|catIdx=Cheng陳}}是[[臺灣|臺灣]]媒體工作者，現任[[TVBS|TVBS]]副董事長兼總顧問<ref>{{Cite web|url=http://ent.ltn.com.tw/news/paper/1095913|title=陳剛信3月怒辭民視 今天進TVBS大門|date=2017-04-21|accessdate=2017-04-26|last=洪素津|work=自由時報}}</ref>，前[[民間全民電視公司|民視]]總經理，曾任[[中華電視公司|華視]]節目部經理、業務部經理，[[文化大學|文化大學]]新聞系畢業，其經營民視19年，強打戲劇置入，獲利逾6億，從接手的虧損13億9千萬到現值百億，有電視圈「財神爺」之稱號。

== 家庭狀況 ==
與妻子育有一子二女，大女兒[[陳瑩_(台灣主播)|陳瑩]]為[[東森電視|東森電視]]主播，曾主持馬英九總統的就職典禮和晚宴。2014年台灣[[太陽花學運|太陽花學運]]尾聲時，陳瑩針對民眾包圍中正警局事件在臉書上公開呼籲黑社會介入，甚至出現攻擊性字眼及髒話，此番言論引起爭議。<ref>{{Cite web|url=http://www.southnews.com.tw/newspaper/00/0438.htm|title=郭倍宏會把民視帶向何方？●曹長青／南方快報《媒體修理站》|accessdate=2017-04-26|author=曹長青|date=2016-08-06|work=|publisher=南方快報}}</ref>

二女兒陳琳於2009年嫁給在美國求學時的同學、上海紫江集團董事長兼中國共產黨員及全國政協委員[[沈雯|沈雯]]之子沈臻<ref>{{Cite news|url=https://tw.entertainment.appledaily.com/daily/20091018/32024811/|title=民視老總陳剛信次女歸寧 大陸女婿身家946億|author=李志展、陳幼英|date=2009-10-18|work=|newspaper=蘋果日報|accessdate=2017-04-26}}</ref>。號稱身價九百億台幣的上海紫江集團董事長[[沈雯|沈雯]](共產黨員，官至中共的全國政協委員)。中國媒體說，「沈雯幾乎把政協委員作為自己的第二專業。」

據原紫江集團董事、財務總裁沈國權的實名舉報：「紫江集團董事長兼總裁的全國政協委員沈雯，極其嚴重涉及數億元人民幣的金融詐騙、巨額偷逃稅款、虛構業務合同、偽造財務報表、欺詐上市、侵吞集體國有資產數百億元人民幣等罪行。」迄今仍沒被查辦<ref>{{Cite web|url=http://www.southnews.com.tw/newspaper/00/0438.htm|title=郭倍宏會把民視帶向何方？●曹長青／南方快報《媒體修理站》|accessdate=2017-04-26|author=曹長青|date=2016-08-06|work=|publisher=南方快報}}</ref>，可見其於中國勢力的強大。

== 經歷 ==
曾在[[中華電視公司|華視]]從節目部副理一路擔任到主任秘書兼華視董事，1995年[[蔡同榮|蔡同榮]]尋求立委連任時，遭[[中國國民黨|國民黨]]空降政務官[[蕭萬長|蕭萬長]]所擊退，而投入籌組無線電視台。<ref>{{Cite web|url=https://www.cmmedia.com.tw/home/articles/3264|title=民視內鬥  獨派郭倍宏驅逐統派陳剛信|accessdate=2017-04-26|author=李彥謀|date=2017-03-30|work=|publisher=信傳媒}}</ref>當時有人向蔡同榮推薦陳剛信，陳剛信由於出身華視，與蔡同榮之政治理念大相逕庭，使蔡對於與陳剛信的政治理念雖有些存疑，但仍因賞識其在電視與戲劇上的能力，故將陳剛信請到民視擔任顧問。1998年擔任民視執行常董兼總經理，2016年改任副董，2017年3月26日發聲明辭去民視副董一職，同年4月21日由TVBS召開董事會宣布陳剛信將擔任TVBS「副董事長兼總顧問」一職。

== 戲劇作品 ==
主導的民視8點檔《春天後母心》、《飛龍在天》、《長男的媳婦》、《娘家》等都創下收視佳績。

== 爭議 ==

=== 逃漏稅事件 ===
原是華視主任秘書的陳剛信，1996年12月接任民視執行董事，隔年國稅局接獲檢舉，指民視當年支付一千五百萬元非固定薪資給陳剛信，但是陳並未將這筆收入申報所得稅，分別以五張支票轉入親人帳戶，並作為私人款項使用，未將款項轉給東吉公司，應是陳的不固定薪資，應報繳所得稅。

此案經國稅局查核屬實，陳剛信因此被認定須補稅五百九十四萬元，處以兩百八十五萬元的罰鍰。<ref>{{Cite news|url=http://www.appledaily.com.tw/appledaily/article/property/20050528/1807657/|title=陳剛信涉漏稅 一審判罰879萬|author=朱慶文．林南谷|date=2005-05-28|work=|newspaper=蘋果日報|accessdate=2017-04-26}}</ref>

=== 民視董座之爭 (又稱民視統獨之爭) ===
2017年臺灣「第一家民營的無線電視臺」－[[民間全民電視公司|民間全民電視公司]]，爆發股權爭議，《民視》前副董事長、前總經理陳剛信疑似不滿董事長[[田再庭|田再庭]]將職位傳予[[郭倍宏|郭倍宏]]、再加上新任常務董事結構清一色皆為「獨立派」人士<ref>{{Cite web|url=http://millenniummag.dudaone.com/20170302-1|title=民主進步黨高層疑佈局 鬥垮《民視》董事會內「獨立派」|accessdate=2017-04-26|author=Stand Media Public Account|date=2017-04-02|work=millenniummag.dudaone.com|publisher=}}</ref>，終於2017年3月26日晚間怒辭副董事長職位，離開待了19年的《民視》。

陳剛信傳出將以15億新臺幣打造國際傳媒集團，製作新戲劇、綜藝以及新聞內容；目前《無線衛星電視》疑似已率先任其為類似「總顧問」職務，且已派人整理專屬辦公室，甚至傳出聯合[[味王|味王]]董事長、民視常務董事陳清福(背後的勢力為青果大王[[陳查某|陳查某]]之子─味王名譽董事長陳建忠，已歸化日本籍並改名為穎川建忠，亦為連戰之親家)、[[三立電視|三立電視]][[林崑海|林崑海]](俗稱海派)、美麗島電子報董事長[[吳子嘉|吳子嘉]](曾要求[[民主進步黨|民進黨]]拿掉台獨黨綱)等人計畫於5月底召開民視股東大會拔除郭倍宏董事之職務。
