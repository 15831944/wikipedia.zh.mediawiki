'''陳芷敏'''（{{lang|en|'''Elke Tan'''}}，{{bd|2=10月19日|3=|4=}}），[[香港|香港]]自由身[[模特兒|模特兒]]，前組合[[Donut|Donut]]的成員之一，近年主力做模特兒工作，亦為不少網上媒體拍短片。

== 簡介 ==
陳芷敏最初入行期間主要任職[[宣傳模特兒|展埸模特兒]]，由於她喜愛拍照作品，嘗試以個人興趣從事模特兒行列，獲得經紀公司賞識之下加入[[娛樂圈|娛樂圈]]工作，並參與香港[[少女模特兒|年輕模特兒]]组合[[Donut_(組合)|Donut]]。<ref>{{cite web |url=http://ent.sina.com.cn/s/h/2011-05-25/14023317371.shtml|title=嫩模Donut新成員亮相週梓盈陳芷敏推寫|accessdate=2011-05-25 |work=|publisher=新浪娛樂|language= zh-hans}}</ref>

基於時下年輕模特兒慣常被大眾認定需要行走性感路線，她認為年輕模特兒都可以展現出另類一面，堅持以清純形象示人，結果突圍而出，得到不少少男少女支持。2011年她參與拍攝[[香港保安局|香港保安局]][[禁毒處|禁毒處]]平面廣告，宣揚禁[[毒品|毒]]訊息「企硬！唔Take嘢」，逐漸為大眾所認識。<ref name=":0">{{cite web |url=http://orientaldaily.on.cc/cnt/entertainment/20110812/00282_088.html|title=陳芷敏清純氣質宣揚禁毒|accessdate=2011-08-12 |work=|publisher=東方日報|language= zh-hant}}</ref>

== 職業生涯 ==

=== 成名初期 ===
陳芷敏生於香港，[[福建|福建]]人。由於姓氏「陳」的英文並不是香港常用的「Chan」，而是新加坡、馬來西亞慣用的「Tan」，因而經常被人誤會是[[新加坡人|新加坡]]或[[馬來西亞人|馬來西亞人]]。<ref name=":2">{{cite web |url=http://www.voovlive.com/voov-web/share/replayVideo?anchor_id=600211895&video_id=100019706&lang=zh_HK|title=陳芷敏 VOOV 直播|accessdate=2018-08-23 |work=|publisher=VOOV直播|language= zh-hant}}</ref>

2010年跟HKModelhouse簽約，並且打破當時年輕模特兒一面倒的性感路線，以清純乖乖女的形象出現，隨後加入組合Donut成為其中一員。<ref name=":0" />

2011年拍攝[[香港保安局|香港保安局]][[禁毒處|禁毒處]]平面廣告「企硬！唔Take嘢」<ref name=":0" />，開始被一般大眾所認識。同年以第二代Donut成員身份推出寫真集《Summer Lover夏之戀人》，由於陳芷敏是成員中唯一沒有走性感路線的，使她脫穎而出，不單得到眾多少男的支持，同時亦得到不少少女的粉絲。<ref name=":1" />

2012年承接著2011年的氣勢，陳芷敏獲得不少香港及中國內地節目的垂青，參與不少電視節目，當中內地節目《[[華娛米飯|華娛米飯]]》更讓陳芷敏在內地受到注視<ref>{{cite web |url=http://www.szxf.net/news/ylbg/201206/132435.html|title=《华娱米饭》“嫩模夏令营”打响开播第一炮|accessdate=2018-08-23 |work=|publisher=深圳消费在线|language= zh-hans}}</ref>。同年更推出第二本寫真集《Donut 戀戀東京》，獲得不錯的口碑。


=== 嶄露頭角 ===
2013年7月陳芷敏率先與視帝[[黎耀祥|黎耀祥]]一同到[[雲浮|雲浮]][[郁南|郁南]]為內地樓盤剪綵，藉著這個機會，獲得不少內地登台的機會。同年陳芷敏首次在亞洲電視節目《[[慧珊時尚坊|慧珊時尚坊]]》亮相，不過隨後主力為[[有線娛樂新聞台|有線娛樂新聞台]]拍攝多個節目，成為該台生招牌。<ref>{{cite web |url=https://www.facebook.com/HKMHL/|title=HKModelhouse Models Facebook專頁|accessdate=2018-08-23 |work=|publisher=Facebook|language= zh-hant}}</ref>

除了電視節目，陳芷敏亦初嘗大銀幕的滋味，客串電影《[[超級經理人|超級經理人]]》。

2014年陳芷敏成為Donut的大師姐，帶領師妹力攻寫真界。同年陳芷敏更客串電影《[[十月初五的月光_(電影)|十月初五的月光]]》，與[[張智霖|張智霖]]及[[鄭欣宜|鄭欣宜]]有對手戲。


=== 直播新貴 ===
2015年開始，Donut的工作開始減少，後來更無聲無息地消失。不過陳芷敏並沒有因此而氣餒，適逢直播App「17」的興起，她轉戰直播平台，短時間內已經有過萬粉絲，成為直播界新貴。<ref>{{cite web |url=https://17.live/profile/r/44269|title=陳芷敏 17直播 帳號|accessdate=2018-08-23 |work=|publisher=17直播|language= zh-hant}}</ref>

2015年中，陳芷敏以自由身繼續在模特兒界發展，憑着以往累積下來的人氣，隨即成為新一代的寵兒。2016年尾騰訊推出直播App「VOOV」，陳芷敏亦加入VOOV的直播行列，並且在直播中建立了「食好西」這個環節。因為陳芷敏喜愛吃東西，被人稱為「美食焚化爐」，更因「食好西」得到不少粉絲的支持，吸引了飲食界的注意，當中更有很多老闆邀請她到自己的餐廳試食，使她成為新一代的「美食女神」。<ref>{{cite web |url=http://www.voovlive.com/voov-web/share/replayVideo?anchor_id=600211895&video_id=100019706&lang=zh_HK|title=陳芷敏 VOOV 直播|accessdate=2018-08-23 |work=|publisher=VOOV直播|language= zh-hant}}</ref>

2017年尾，陳芷敏除VOOV之外，同時加入Uplive直播，並且開始得到不少來自內地，與台灣及其他東南亞的支持者，開始向海外進軍。<ref>{{cite web |url=http://www.up.live/?gclid=EAIaIQobChMItLib5smB3QIViraWCh0n9wK6EAAYASAAEgLbG_D_BwE#/profile/base/22418512|title=陳芷敏 UPLIVE 直播|accessdate=2018-08-23 |work=|publisher=UPLIVE 直播|language= zh-hant}}</ref>

2018年尾，陳芷敏離開VOOV，加盟直播APP始祖17，並取得不錯的成績。


=== 挑戰短片 ===
除了直播，2017年尾開始，陳芷敏開始接拍不同的短片，雖然她一向走清純美女的路線，不過在短片中，她不時扮演一些破壞形象的角色，例如滿臉鬍鬚的女人，令人留下深刻的印像。

== 興趣愛好 ==
=== 食大過天 ===
向來有「美食女神」稱號的陳芷敏，不論任何食物都敢於挑戰，而且她對美食有獨特的見解，有別於時下的食評，對年青人的影響力很大，使他們相繼仿傚。<ref name=":2" />

=== 愛狗如命 ===
陳芷敏向來愛狗如命，對愛犬「叔叔」及「Dark Dark」寵愛有加，而兩頭愛犬的支持度更不下於主人，經常在陳芷敏的網上直播中「叔叔」亂入。

兩隻愛犬之中，以「叔叔」的出鏡率最高，除了在雜誌訪問中出現，2015年更在無線電視節目《[[絕世好狗|絕世好狗]]》亮相，差不多成為陳芷敏的代言人。<ref name=":2" />
