'''陳宥維'''（{{bd|1998年|7月7日}}），[[慈文影视|慈文传媒]]旗下藝人，中国内地男演员，模特及歌手，出生于[[浙江省|浙江省]][[丽水市|丽水市]]，现就读于[[浙江理工大学|浙江理工大学]]服装表演专业。2017-2019年期间參演多部電視劇，因《[[延禧攻略|延禧攻略]]》中五阿哥永琪一角为人所熟知。2019年，參與[[爱奇艺|愛奇藝]]偶像男團競演養成類真人秀《[[青春有你|青春有你]]》並作為限定九人男團[[UNINE|UNINE]]成員，以偶像身份正式出道。

== 经历 ==
=== 2016年 - 正式签约 ===
2016年，在慈文传媒举办的万人选新活动中脱颖而出，从而签约[[慈文影视|慈文传媒]]。

=== 2017年 - 登上综艺节目并参演首部电视剧 ===
2017年，以学员的身份参加进阶式培训综艺节目《超次元偶像》<ref>{{cite web |author1=网易娱乐 |title=《超次元偶像》青春活力洋溢陈宥维获封初恋脸 |url=http://ent.163.com/17/0529/13/CLJTN4J000038793.html}}</ref>。因其优秀的表现而获得了节目导师[[何炅|何炅]]、[[何冰|何冰]]与嘉宾[[撒贝宁|撒贝宁]]的高度认可，并在节目上荣获环法中国赛青年大使的荣誉称号<ref>{{cite web |author1=人民网 |title=环法中国赛青年大使陈宥维 阳光自信展魅力 |url=http://ent.people.com.cn/n1/2017/1030/c1012-29617071.html}}</ref>。

2017年6月，参演个人首部电视剧《[[延禧攻略|延禧攻略]]》，饰演五阿哥永琪<ref>{{cite web |author1=延禧攻略剧组 |title=电视剧《延禧攻略》官方微博 |url=https://www.weibo.com/u/6177491300}}</ref>。

=== 2018年 - 参演多部电视剧并持续打开知名度 ===
2018年4月23日，参演的美食题材古装言情剧《[[萌妻食神|萌妻食神]]》播出。

2018年10月22日，参演的古装电视剧《[[双世宠妃2|双世宠妃2]]》播出 。

2018年7月10日，参演的网剧《[[炮灰攻略|炮灰攻略]]》播出。

2018年11月，参演个人首部现代职场言情剧《[[浪漫星星|浪漫星星]]》，并于2018年12月5日在腾讯视频上线。

=== 2019年 -《青春有你》以偶像身份重新出发===
2018年12月，参加[[爱奇艺|爱奇艺]]偶像男团竞演养成类真人秀节目《[[青春有你|青春有你]]》。前三次排名公布均处于前四名。

同时，在与比赛并行的[[爱奇艺|爱奇艺]]衍生节目《青春艺能学院》第三、四、五期中，出任双队长（即MC或主持人）之一。

2019年3月31日，青春有你训练生第一次粉丝千人见面会，陈宥维《火种》单人focus饭拍片段<ref>{{Cite web|url=https://music.163.com/m/video?id=959697C6722BC46A84924131B8BA50ED&userid=1811065226|title=《火种》千人见面会直拍|accessdate=|author=ONEOFTHESENIGHTS_陈宥维|date=2019.03.31|publisher=}}</ref>（粉丝直接拍摄影片）流出并引发热议。

2019年4月6日，《[[青春有你|青春有你]]》决赛公演现场，主持人宣布陈宥维第八位出道。由于此位置与其一直保持前四的上位圈成绩相差甚远，以及其所在的公司慈文传媒代表提前离席，节目播出随即引起了巨大争議。
