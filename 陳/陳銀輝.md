'''陳銀輝'''(1931年8月5日-)，臺灣戰後第一代畫家、美術教師；作品以半抽象繪畫聞名，多次擔任國內美展評審，從事教育四十年，1995年於師大退休，捐出其退休金，成立「陳銀輝油畫創作獎學金」。<ref>{{cite web |language=zh-tw |url=https://www.gacc.org.tw/collections/show/5#| title= 歷年典藏-陳銀輝油畫創作展-|publisher=中華文化總會|date= |accessdate=2017-11-14}}</ref>。現為師大美術系名譽教授、臺灣藝術研究院院士。<ref>{{cite web |language=zh-tw |url=http://cn.taiwanart.org.tw/personal/CA7457 | title= 陳銀輝-臺灣藝術研究院-決策委員暨院士|publisher=臺灣藝術研究院 |date= |accessdate=2017-11-14}}</ref>。

== 生平 ==
===早年===
陳銀輝出生於[[嘉義縣|嘉義縣]][[鹿草鄉|鹿草鄉]]，父親原先是日治時期的[[保正|保正]]，因替人做擔保負債以致家道中落，舉家搬遷至嘉義市經營「柑仔店」。1938年進入嘉義白川公學校（現大同國小）
就讀，小學時期即展現繪畫天分，經常代表學校對外參加各項比賽。1944年考上日本人占極大多數的台南州立嘉義中學校初中部，並在三年後以優異成績續升高中部。中學時期正值國民政府接收臺灣，來自大陸的畫家[[席德進|席德進]]、[[吳學讓|吳學讓]]分別執教於該校初中部及高中部，由於陳銀輝美術成績表現優異而受到吳學讓的賞識，特別在課外給予指導，因此奠定他深厚的繪畫基礎。1950年，同時考取台大工學院與師範學校藝術系（台師大美術系前身）。為了繪畫夢，他毅然選擇藝術系。大學期間受前輩藝術家[[陳慧坤|陳慧坤]]、[[馬白水|馬白水]]、[[廖繼春|廖繼春]]、[[趙春翔|趙春翔]]、[[孫多慈|孫多慈]]、[[袁樞真|袁樞真]]等多位老師薰陶。
===投身教職和創作===
1957年，受師大藝術系系主任[[莫大元|莫大元]]之邀回母校擔任助教，負責管理系上倉庫，並一邊繼續鑽研繪畫。<ref name="multiple"></ref>據陳銀輝回憶，當時沒錢租房子，於是清出倉庫一角當作睡覺的地方，就住在那個擁擠、灰暗的空間長達6年。和妻子楊淑貞結婚後，夫妻倆經濟狀況不佳，依然住在倉庫。陳銀輝說，他和一樣學美術的妻子，除了作畫、研究畫冊，沒別的娛樂。兩人就在小小的空間，畫過一幅又一幅，很感謝妻子始終相隨，這段日子影響自己很深，更懂得關心學生。<ref>{{cite news |language = zh-tw |title = 退休教授陳銀輝回顧展|publisher = 聯合報 |pages = B2版 |date = 2014-03-30 |accessdate = 2017-11-17}}</ref>。1962年升等為講師，與擔任小學老師的夫人，利用課餘時間，為人設計月曆、製作圖表、畫肖像，甚至畫了不少國劇臉譜交給中華商場的店家賣給外國觀光客。1968升等為副教授。1961年升任為教授，六年後和[[陳景容|陳景容]]應美國聖若望大學之邀，赴該校中山畫廊舉行聯展，之後並赴歐洲各國考察美術。在自助旅行的三個月期間，陳銀輝參觀各大美術館，瀏覽世界名家畫作，待美術館關門後，便利用時間在外速寫，為他日後的創作累積不少寶貴素材。之後，陳銀輝除了教學外，也到臺灣四處寫生，並先後遊歷了印度、尼泊爾、印尼、韓國、中國絲路、敦煌莫高窟、張家界、黃山等名勝。除了大量完成旅遊所見的寫生作品外，陳銀輝發展出以羊和女體為題材的作品，暗喻自己和夫人的情感。<ref>{{cite book |author=鄭功賢 |date=1996-5 |title=台灣山羊腳力健壯─陳銀輝的執著性格和創作理路 |location=臺北 |publisher=典藏藝術雜誌}}</ref>1995年於師大退休，捐出其退休金，成立「陳銀輝油畫創作獎學金」。<br />




== 作品風格 ==
陳銀輝與[[藍蔭棠|藍蔭棠]]、[[吳隆榮|吳隆榮]]、[[陳正雄|陳正雄]]於1964年成立「心象畫會」(兩年後改組為[[世紀美術協會|世紀美術協會]]，陳正雄、[[楊炎傑|楊炎傑]]退出，而增加[[孫明煌|孫明煌]]、[[劉文煒|劉文煒]]、陳景容、[[潘朝森|潘朝森]]四人)，在當時[[台灣現代藝術運動|台灣現代藝術運動]]的背景下，許多畫會紛紛成立以發表對「現代」的主張。<ref> {{Cite book | author = 蕭瓊瑞 | title = 戰後台灣美術史 | location = 臺北 | publisher = 藝術家出版社 | date = 2013-04| ISBN =9789862820902 }} </ref>陳氏的創作追求「繪畫性」，個人生活和寫生經驗只是作畫的契機，目的並非自然形象表面的描摹或再現，而是藉自然的形態和色彩表現藝術家的自我。據他自述：「形體（Forme）的追求，畫面的構成，是我繪畫的首要目標；面對大自然，或身邊的一切物象，我都把它們還元[原]為「形」與「色」的組合」。陳銀輝時常持畫筆、畫刀併用作畫，有時用手指頭摸，用布擦，或用竹片刮等等，自由地隨手運用工具。在他的風景和靜物畫中，觀者看到的是線條、塊面造型、肌理、色彩的交響組合，呈現出藝術家所經營的獨特詩意。<ref>{{cite journal |author=陳銀輝 |date=1972-08 |title=《畫無止境》 |location=臺北 |publisher=雄獅美術}}</ref>
