'''陳奕恬失蹤事件'''是一名名叫陳奕恬（生於1985年3月1日）的女童於1988年6月在[[台灣|台灣]][[桃園縣|桃園縣]][[桃園市|桃園市]]走失的事件。陳奕恬3歲時，一天在家門外的巷口遊玩時失蹤，家人隔日發動親友尋回未果。根據目擊者的描述，陳奕恬是被兩名[[摩托車|機車]]騎手拐走的。陳奕恬至今已{{Age|1985|03|01}}歲，從失蹤時算起，截至2018年已走失30年。陳奕恬走失後，她的母親周慈宜一直沒有放棄找尋女兒的努力，一直在為找回女兒進行活動。<ref>{{cite web |author1=楊久瑩 |title=孩子妳在哪 愛母尋女20年 |url=http://news.ltn.com.tw/news/life/paper/615363 |website=自由電子報 |date=2012-09-16 |accessdate=2019-01-09}}</ref><ref>{{cite web |author1=許麗珍 |title=最長48年未尋獲！台灣失蹤兒　多遭隨機抱走 |url=https://tw.appledaily.com/new/realtime/20171222/1264414/ |website=蘋果日報 |date=2017-12-22 |accessdate=2019-01-09}}</ref><ref>{{cite web |author1=謝君怡 |title=愛女失蹤29年 母：「找到我斷氣為止」 |url=https://www.nextmag.com.tw/realtimenews/news/336596 |website=壹週刊 |date=2017-08-17 |accessdate=2019-01-09}}</ref><ref>{{cite web |title=失蹤協尋30年 陳奕恬媽媽仍盼女兒回家 |url=https://www.youtube.com/watch?v=Coy4dl4rrXs |website=東森新聞 |date=2018-10-19 |accessdate=2019-01-09}}</ref>
