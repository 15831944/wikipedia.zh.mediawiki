'''陳家樂'''（'''{{lang|en|Carlos Chan}}'''，{{bd|1986年|10月12日|catIdx=C陳}}），[[香港|香港]]男演員。2004年畢業於[[馬鞍山崇真中學|馬鞍山崇真中學]]{{noteTag|陳家樂是Facebook公開群組「馬鞍山崇真中學5B班 [03-04年度]」的成員。}}，自出生到小學五年級都是與祖母一起居住<ref>{{web reference|URL=https://skypost.ulifestyle.com.hk/article/1617000/is|title=惠英紅 陳家樂 互為改變的因果|work=晴報|year=2016年|date=9月2日}}</ref>。陳家樂在童年時已當過童星，於四至五歲時曾參與拍攝廣告的工作<ref>{{web reference|URL=http://www.metrodaily.hk/metro_news/beauty-%E6%98%9Ftalk-%E9%99%B3%E5%AE%B6%E6%A8%82%E7%94%B7%E4%BA%BA30/?page682=2|title=Beauty 星talk 陳家樂男人30|work=都市日報|year=2016年|date=12月23日}}</ref>。2003年參與[[英皇新秀歌唱大賽|英皇新秀歌唱大賽]]<ref>[https://www.hk01.com/%E5%A8%9B%E6%A8%82/131124/-%E5%B0%88%E8%A8%AA-%E9%99%B3%E5%AE%B6%E6%A8%82%E7%84%A1%E7%B7%A3%E8%88%87%E9%99%B3%E5%81%89%E9%9C%86%E7%B5%84Sun-Boy-z-%E5%8F%AA%E6%80%AA%E4%BB%A5%E5%89%8D%E5%94%94%E7%9F%A5%E8%87%AA%E5%B7%B1%E8%A6%81%E4%B9%9C 【專訪】 陳家樂無緣與陳偉霆組Sun Boy'z：只怪以前唔知自己要乜]</ref>，並簽約[[英皇娛樂|英皇娛樂]]。因愛好踢足球，所以不時與友人組成「遠東足球隊」參與足球運動{{noteTag|陳家樂是Facebook公開群組「遠東足球隊」的成員。}}。近年，又與[[張建聲|張建聲]]、[[游學修|游學修]]、[[王宗堯|王宗堯]]和[[麥子樂|麥子樂]]等男藝人組成「新明星足球隊」<ref>{{web reference|URL=http://www.metrodaily.hk/metro_news/%E9%83%BD%E5%B8%82%E5%B0%88%E8%A8%AA-%E8%87%AA%E4%B8%BB%E6%96%B0%E4%BA%BA%E9%A1%9E-%E6%B8%B8%E5%AD%B8%E4%BF%AE/|title=都市專訪 自主新人類 游學修|work=都市日報|year=2017年|date=5月10日}}</ref>。

2016年，陳家樂於電影《[[幸運是我|幸運是我]]》中的亮麗演出而獲得《第11屆華語青年影像論壇》的「年度新銳男演員獎」<ref>{{web reference|URL=https://hk.entertainment.appledaily.com/enews/realtime/article/20161123/55955759|title=陳家樂憑《幸運是我》 奪武漢華語青年影像論壇新銳男演員|work=香港蘋果日報|year=2016年|date=11月23日}}</ref><ref>{{web reference|URL=http://orientaldaily.on.cc/cnt/entertainment/20161124/00282_063.html|title=陳家樂奪新銳男演員獎|work=東方日報|year=2016年|date=11月24日}}</ref><ref>{{web reference|URL=http://hk.on.cc/hk/bkn/cnt/entertainment/20161205/bkn-20161205154750069-1205_00862_001.html|title=演技被肯定 陳家樂獲頒新銳男演員|work=東網|year=2016年|date=12月5日}}</ref>。

2018年，監製[[王心慰|王心慰]]邀請陳家樂拍攝TVB電視劇集<ref>{{web reference|URL=https://skypost.ulifestyle.com.hk/article/2040702/%E5%87%8D%E9%BD%A1%E7%88%BA%E7%88%BAVs%E8%80%81%E9%A1%8F%E5%AD%AB%E5%85%92%20%E3%80%8A%E9%80%86%E7%B7%A3%E3%80%8B%E9%99%B3%E5%AE%B6%E6%A8%82%E6%95%99%E5%8C%96%E8%B2%AA%E9%8C%A2%E7%A5%A5%E4%BB%94|title=凍齡爺爺Vs老顏孫兒 《逆緣》陳家樂教化貪錢祥仔|work=晴報|year=2018年|date=3月29日}}</ref>，他在劇集《[[逆緣|逆緣]]》中演技出色獲得觀眾稱讚<ref>{{web reference|URL=https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/179985/%E9%80%86%E7%B7%A3-%E5%B0%88%E8%A8%AA-%E9%99%B3%E5%AE%B6%E6%A8%82%E6%A5%B5%E7%AA%AE%E8%A2%AB%E8%BF%AB%E9%81%8E%E9%9A%B1%E9%9D%92%E7%94%9F%E6%B4%BB-%E5%86%87%E9%8C%A2%E5%88%B0%E9%A3%AF%E9%83%BD%E5%94%94%E6%95%A2%E9%A3%9F|title=【逆緣．專訪】陳家樂極窮被迫過隱青生活：冇錢到飯都唔敢食！|work=香港01|year=2018年|date=4月20日}}</ref>。

== 感情生活 ==
陳家樂的前女友為演員[[余香凝|余香凝]]。他於2018年的一個訪問中表示已認定女友余香凝為結婚對象，計劃在五年內結婚<ref>[https://bka.mpweekly.com/interview/%E5%A8%9B%E6%A8%82123/20180321-105698 【陳家樂專訪】五年內娶余香凝 演香港富豪學潮州鄉音]</ref>。不過直到2018年6月，陳家樂宣布與余香凝分手，有傳媒報道原因是女方偷情<ref>[https://www.orientalsunday.hk/%E6%9C%80%E6%96%B0%E5%A8%9B%E8%81%9E/%E9%99%B3%E5%AE%B6%E6%A8%82-%E4%BD%99%E9%A6%99%E5%87%9D-%E5%81%B7%E9%A3%9F-%E5%88%86%E6%89%8B/ 余香凝偷食致分手？ 陳家樂含淚認情已逝…]</ref>，但女方否認<ref>{{Cite web |url=https://www.hk01.com/article/199346 |title=余香凝稱分手不涉及第三者：嗰位先生係多年好友　最近一齊開餐廳 |date=2018-06-14 |publisher=香港01}}</ref>。陳家樂則在instagram上發文讚揚女方是他「最親的親人」，並稱「經歷過高低潮，經歷過生死，感謝她令我成長」<ref>{{Cite web |url=https://hk.entertainment.appledaily.com/entertainment/daily/article/20180615/20420763 |title=陳家樂分手痛哭 余香凝認保險才俊家留宿 |date=2018-06-15 |publisher=蘋果日報娛樂}}</ref><ref>{{Cite web |url=https://www.hk01.com/article/199755 |title=陳家樂發聲護前度　IG深情表白：余香凝為我最親的親人 |date=2018-06-15 |publisher=香港01}}</ref>。
