'''陳偉仁'''，是[[台灣|台灣]]知名[[軟體|軟體]][[公司|公司]][[友立資訊|友立資訊]]的創辦人，曾獲選為1988年[[中華民國|中華民國]]資訊產業十大傑出人物。他也是一名虔誠的[[基督徒|基督徒]]，曾任職[[台灣世界展望會|台灣世界展望會]]副會長，目前擔任中華基督教[[長老會|長老會]]台北信友堂[[董事長|董事長]]及該堂[[長老|長老]]、心又新關懷協會的創辦人及理事長<ref>http://www.cdn.org.tw/News.aspx?key=7189</ref>。他也是一名文字創作者及攝影家，涉獵於[[散文|散文]]、[[儿童文学|童書]]、[[新詩|新詩]]等文體創作。

== 早年生活 ==
陳偉仁於[[逢甲大學|逢甲大學]]資訊工程學系(電子計算機科學系)畢業，於1981年進入[[台灣電力公司|台灣電力公司]]服務，利用[[個人電腦|個人電腦]]在[[CP/M|CP/M]]系統上開發「圖形線上電壓與電流監視系統」。1983年到1985年期間在[[交通大學|交通大學]]進修，完成[[計算機工程|計算機工程]][[碩士|碩士]]的學位。畢業後進入資訊工業策進會服務，開發出全世界第一套在[[Microsoft|Microsoft]] [[Windows|Windows]] 1.0的黑白影像處理程式，並陸續與台灣的全友電腦、宏碁、東元科技，日本的[[Ricoh|Ricoh]]以及[[義大利|義大利]][[:en:Olivetti|Olivetti]]等公司的影像[[掃描器|掃描器]]搭售，行銷到世界各地。1988年，陳偉仁獲選為中華民國資訊產業十大傑出人物。

== 創立友立資訊 ==
1989年與另兩位夥伴陳學群及[[廖敏芳|廖敏芳]]創立友立資訊，隔年開發全世界第一套在Microsoft Windows 2.0的全彩影像處理程式La Palette，與包括[[HP|HP]]、全友電腦等許多世界知名的彩色影像掃描器搭售，並開始進入全世界的[[零售|零售]][[市場|市場]]。後來繼續參與[[:en:PhotoStyler|PhotoStyler]]、ImagePals及[[PhotoImpact|PhotoImpact]]等產品的研發。在[[影像處理|影像處理]]的[[套裝軟體|套裝軟體]]領域當中，他負責的產品都曾多次榮獲歐、美、日等知名媒體的肯定，對影像核心技術與產品定義有深入的研究。

== 生涯轉型 ==
2001年起擔任友立資訊董事長並兼任總經理，直到2005年4月友立資訊被[[美國|美國]][[:en:InterVideo|英特維數位科技]]收購後才退休離開友立資訊經營團隊。退休後完成[[台灣大學|台灣大學]][[EMBA|EMBA]]課程學業並從事志工活動，並自2006年1月起每週五晚上八點半在基督教佳音電台主持廣播節目「新詩部落格」，並出任台灣世界展望會副會長。

2011年3月離開台灣世界展望會後，目前仍繼續從事社會福利與團體組織改造工作，並出版詩集《旅行詩篇》及開攝影展。2006年11月28日出版他的第一本[[繪本|繪本]]《胖子尋寶》。
