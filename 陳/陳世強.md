'''陳世強'''{{JP}}{{BD|1955年}}，生於[[香港|香港]]，職業為[[律師|律師]]，[[客家人|客家人]]，是[[2011年香港選舉委員會界別分組選舉|2011年香港選舉委員會界別分組選舉]][[基督教|基督教]]界別當選委員及[[香港國際全備福音商人團契|香港國際全備福音商人團契]]總監。

==法律業務  ==
陳世強於1978年在[[英國|英國]][[修咸頓大學|修咸頓大學]]考獲法律[[學士|學士]]。於1979年，陳世強完成律師考試後，回港後在[[孖士打律師行|孖士打律師行]]工作至1986年。他現在是廖何陳世強行合夥人<ref>[http://www.lhc.com.hk/home.htm]</ref>，專長地產發展、商業及中國事務。他是[[中國司法部|中國司法部]]委托公證人和[[香港律師會|香港律師會]]內地法律事務委員會委員。

==社會服務 ==
陳世強是[[香港有品運動|香港有品運動]]<ref>[http://www.theotexture.com/hong-kong-character-city-movement]</ref>的主席。「香港有品」乃是一個在學校、和機構透過課程及培訓推動品格文化教育的一個非牟利機構。 陳世強亦是[[快樂人慈善基金|快樂人慈善基金]]的主席。

陳世強是[[香港專業及資深行政人員協會|香港專業及資深行政人員協會]]的創會副會長<ref>[http://www.hkpasea.org/news.asp?menuid=7321&supmenuid=7320] </ref>。專資會會員來自專業界、企業界及學術界的領袖。

在2006年，陳世強以最高票數當選為基督教界別的[[選舉委員會_(香港)|選舉委員會]]委員<ref>[http://www.christianweekly.net/4DACTION/W4D_CWREAD/13294/11/BACK]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，及後於2011年再度連任。

陳世強曾任[[中國|中國]][[深圳市|深圳市]]和[[長春市|長春市]][[政協委員|政協委員]]，也是[[深圳市歸僑僑眷企業家聯合會|深圳市歸僑僑眷企業家聯合會]]副會長<ref>{{cite web |url=http://szefroc.org/ |title=存档副本 |accessdate=2011年9月28日 |deadurl=yes |archiveurl=https://web.archive.org/web/20110925163932/http://www.szefroc.org/ |archivedate=2011年9月25日 |df= }}</ref>及深圳市僑界法律顧問。

為表揚陳世強在基督教、商業及社會之貢獻，[[美國|美國]][[芝加哥|芝加哥]]之[[奧利韋拿撒勒大學|奧利韋拿撒勒大學]]（[[:en:Olivet_Nazarene_University|Olivet Nazarene University]]）向他頒發了「榮譽文學博士學位」(D. Litt)<ref>[http://www.olivet.edu/news/newsDetails.aspx?Channel=%2FChannels%2FSite+Wide&WorkflowItemID=a2896a3c-8f6c-4cef-a4f0-45cf52e23af6 Olivet Nazarene University - News Story<!-- Bot generated title -->] {{webarchive|url=https://web.archive.org/web/20110930204848/http://www.olivet.edu/news/newsDetails.aspx?Channel=%2FChannels%2FSite+Wide&WorkflowItemID=a2896a3c-8f6c-4cef-a4f0-45cf52e23af6 |date=2011-09-30 }}</ref>。

== 香港國際全備福音商人團契涉頎小姐事件 ==
香港國際全備福音商人團契教會在會展舉行「國際全備福音商人團契25周年慶祝晚宴」，<ref>[http://hk.apple.nextmedia.com/news/art/20120618/16435774 「教徒斥包庇教會之狼」]，《蘋果日報》，2012年6月18日。</ref><ref>{{youtube|InujBEknqi0|追擊《商人團契陳世强包庇强姦犯》}}，2012年6月19日</ref>，有人發動「紫絲帶行動」声援被强奸的的頎小姐。<ref>{{youtube|AfdnXR97a00|手繫紫絲帶 心繫Ｋ小姐}}，K小姐朋友講述K小姐慘況，旺角西洋菜南街。2012年6月17日。</ref>馮X文指控在台灣被教友蔡X頎勒索五十萬。

頎小姐事件在2012年被瘋傳，雖然X先生已被逐出教會，但仍然逍遙法外，並不斷試圖把市民對事件的關注度擴大，例如6月15日他們在[[旺角|旺角]][[西洋菜南街|西洋菜南街]]發行集會和演說，當中更邀請過K小姐的朋友現身說法，同年6月19日，[[基督路小教會|基督路小教會]]教友到[[灣仔|灣仔]][[香港會展|香港會展]]入口示威 (因為商人團契在那裏舉行國際全備福音商人團契25周年慶祝晚宴)。<ref>[http://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=72846&Pid=5&Version=0&Cid=220 陳世強回應 K小姐事件 聲明指污蔑她與馮X文有超友誼關係，對她是極大的侮辱。她反駁陳世強的聲明是「睜著眼說大話」，誣衊及抹黑她，是自保式的回應和聲明；而最令她震驚的，是生意伙伴，有著嚴重的利益關係衝突，而這是她在向陈世强尋求協助時沒有向她提及過的。

== 傳媒業務 ==
2003到2006年期間，陳世強曾任[[亞洲電視|亞洲電視]]之[[星火飛騰|星火飛騰]]節目主持人。他是基督教《[[國度復興報|國度復興報]]》<ref>[http://www.krt.com.hk/]</ref>的創辦人之一。陳世強也是《[[快樂人月刊|快樂人月刊]]》(Happymen Magazine)的出版人<ref>[http://happymen.com.hk/vision.php?vol=23]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 家庭背景 ==  
陳世強出身自[[新界|新界]][[荃灣|荃灣]][[三棟屋村|三棟屋村]]，其父為[[陳流芳|陳流芳]]{{JP}}{{BBS}}，曾任兩屆[[荃灣區議會|荃灣區議會]]主席（1994年-1999年）。

陳世強在1981年結婚，育有3名女兒，和2名孫兒。
