'''陳冠中'''（{{bd|1952年|catIdx=C}}），出生于[[上海|上海]]，祖籍浙江[[宁波|宁波]]<ref >{{cite web|url=http://www.wenming.cn/book/pdjj/201307/t20130723_1362724.shtml |title=《我这一代香港人》由中信出版社出版 |publisher=中国文明网 |author=来源：[[中国青年报|中国青年报]] |date=发表时间：2013-07-23 |accessdate=2014-06-26}}</ref>，香港作家。

==簡歷==
1952年出生于[[上海|上海]]，在[[香港|香港]]長大，曾在[[台北|台北]]居住六年，現居[[北京|北京]]。

1974年於[[香港大學|香港大學]]社會學系畢業，之後往美國[[波士頓大學|波士頓大學]]進修[[新聞學|新聞學]]。[[綠色力量|綠色力量]]、[[綠田園有機農場|綠田園有機農場]]、[[香港電影導演會|香港電影導演會]]等發起人，現任[[綠色和平|綠色和平]]國際[[董事|董事]]。第一份正職工作是「The Star」記者。1976年陳冠中聯同[[丘世文|丘世文]]、[[鄧小宇|鄧小宇]]及[[胡君毅|胡君毅]]創辦生活潮流月刊《[[號外_(雜誌)|號外]]》，並擔任其出版人和總編輯共23 年。

他於1981年開始創作劇本，曾監製或策劃十多齣香港電影以及3齣美國電影，並為香港電影導演協會創辦人之一。他曾在90年代中期任《读书》海外出版人。

陳冠中曾在香港、台灣、北京，兩岸三地生活。他在[[香港|香港]]長大，1992年之前，陳冠中居於香港，1992年至1994年在北京逗留了兩年多，之後在台北生活6年。2000年後，他又搬到北京長住。<ref>{{cite web |title=【2018香港大學人物篇】陳冠中：香港文化“多面手” |date=2018-03-27 |publisher=香港大學SPACE |url=https://sea.hkuspace.hku.hk/cht/news/2018-homecoming-day-chen-guan-zhong}} </ref>

==成就==
陳冠中与[[胡恩威|胡恩威]]、[[甘国亮|甘国亮]]、[[林奕华|林奕华]]、[[张艾嘉|张艾嘉]]、[[关锦鹏|关锦鹏]]、[[岑建勋|岑建勋]]、[[刘天兰|刘天兰]]、[[梁文道|梁文道]]、[[陈士齐|陈士齐]]、[[张翠容|张翠容]]等香港文化人和媒体人一道创立[[香港牛棚书院|香港牛棚书院]]，为[[香港文化|香港文化]]艺术作出贡献<ref>{{cite web |url=http://www.oupchina.com.hk/what/academic_arch15.asp |title=存档副本 |accessdate=2013-05-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20081202013508/http://www.oupchina.com.hk/what/academic_arch15.asp |archivedate=2008-12-02 }}</ref>。他曾參與[[嚴浩|嚴浩]]、[[徐克|徐克]]合導的電影《[[棋王_(1991年電影)|棋王]]》的演出。

2012年，他被香港大學選為名譽大學院士。<ref>{{cite web  |title=2012名譽大學院士 |publisher=香港大學 |url=https://www4.hku.hk/honfellows/chit/honorary-university-fellows/mr-john-koon-chung-chan}}</ref>
2013年，他獲選為[[香港書展年度作家|香港書展年度作家]]。

2018年，陳冠中新短篇小說《馬可波囉》寫成。雖然他已走到中國內地，但仍選擇以小說來發聲者，在大中國的背景裏，擔起了小說專業的代表，寫出中國那曖昧不明的現狀，同時，又令人從中想起香港<ref>[https://bkb.mpweekly.com/cu0001/20180127-65218 【大中國時代　知識分子的曖昧處境】編者話：為什麼要讀陳冠中？]</ref>。同年1月，接受訪問時表示新短篇小說《馬可波囉》是跟自己長期關注的議題和生活接觸面有關。在北京常見到人文社科學者和文化工作者，他們現在的處境觸發他寫出這個短篇小說來<ref>[https://bkb.mpweekly.com/cu0001/20180131-65548 「再魅」的大中國時代　與陳冠中對談最新小說《馬可波囉》]</ref>。
