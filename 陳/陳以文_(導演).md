'''陳以文'''（{{lang|en|Chen Yi-Wen}}，{{bd|1966年||||catIdx=C陳}}），[[國立藝術學院|國立藝術學院]]戲劇系畢業。曾以《[[運轉手之戀|運轉手之戀]]》獲得[[第37屆金馬獎|第37屆金馬獎]]評審團大獎、台北電影節評審團大獎及最佳導演獎。

== 生平 ==

在學期間已活躍於[[劇場|劇場]]界、電視圈和[[電影|電影]]圈中，擁有[[演員|演員]]、[[劇作家|劇作家]]、[[編劇|編劇]]、[[導演|導演]]多項才能。與[[導演|導演]][[楊德昌|楊德昌]]合作過多部電影，除參與幕後製作也擔任演員，如《[[牯嶺街少年殺人事件|牯嶺街少年殺人事件]]》、《[[獨立時代|獨立時代]]》、《[[麻將_(電影)|麻將]]》、《[[一一|一一]]》及動畫長片《[[追風|追風]]》之劇本初稿共同編寫。於電影《[[獨立時代|獨立時代]]》一片中擔任[[副導演|副導演]]並演出當中「立人」一角。1994成立[[烈日工作室|烈日工作室]]，首部短片《[[暴力紀實錄|暴力紀實錄]]》，以[[新台幣|新台幣]]三千元的「極低」成本拍攝，卻從[[台灣|台灣]]及[[日本|日本]]電視台回收了新台幣六十萬元，奠定了他對自我投資電影事業的精準與企圖心。《[[暴力紀實錄|暴力紀實錄]]》獲得多數影展的肯定，且以如此高的投資報酬率，一度成為《[[財訊雜誌|財訊雜誌]]》的報導人物。<ref>{{Cite web |url=http://okapi.books.com.tw/index.php/p3/p3_detail/sn/2362 |title=《殺掉青春沒有夢》陳以文導演：電影不是夢，是活生生的現實 |accessdate=2013-09-03 |author=李屏瑤 |publisher=博客來OKAPI}}</ref>

1998年第一部自編自導劇情長片《[[果醬|果醬]]》，注入新生代創作者輕快的節奏和對都會的知性感，評論界認為融合了楊德昌都會風格和[[昆汀‧塔倫提諾|昆汀‧塔倫提諾]]的風格。2000年的《[[想死趁現在|想死趁現在]]》<ref>{{Cite web |url= http://www.imdb.com/title/tt0270271/combined |title= A Chance to Die |publisher=IMDB.com }}</ref>表現出社會邊緣人的大膽作為，受邀作為[[香港電影節|香港電影節]]之閉幕影片。2000年同年的《[[運轉手之戀|運轉手之戀]]》<ref>{{Cite web |url= http://www.imdb.com/title/tt0265107/combined |title= The Cabbie |publisher=IMDB.com}}</ref>開啟台灣喜劇電影的新一頁，並獲得當年[[第37屆金馬獎|第37屆金馬獎]]評審團大獎、[[臺北電影節|臺北電影節]]評審團大獎及最佳導演獎、[[法國|法國]][[杜維爾影展|杜維爾影展]]「最佳導演獎」，入圍[[德國|德國]][[柏林影展|柏林影展]]「青年論壇」單元，並代表[[台灣|台灣]]角逐[[奧斯卡金像獎最佳外語片獎|奧斯卡金像獎最佳外語片獎]]。<ref>{{cite book |author=陳以文 |title=〈殺掉青春沒有夢：有些事，要等電影殺青了才知道〉|url= http://www.books.com.tw/products/0010592618 |publisher= 明日工作室 |isbn= 9789862905784 |date=2013年8月}}</ref>

2001年自[[紐約|紐約]]遊學歸來後，除了[[電影|電影]]外，也同時參與[[劇場|劇場]]、[[電視|電視]]及[[廣告|廣告]]的工作，先後與[[可米瑞智|可米瑞智]]、[[公共電視台|公共電視台]]與[[大愛電視台|大愛電視台]]合作。2006年的完成的電影《[[神遊情人|神遊情人]]》（日本名《[[幻遊傳|幻遊傳]]》）被日本媒體譽為「能讓台灣電影復活的娛樂電影」，是結合[[计算机图形|CG特效]]、全新嘗試的奇幻古裝[[公路電影|公路電影]]。

2009年，除了為[[荷蘭|荷蘭]]導演[[David_Verbeek|David Verbeek]]擔任電影《[[R_U_There|R U There]]》監製外，也完成了自己監製兼編導的劇情長片《[[1689號追蹤檔案|1689號追蹤檔案]]》；2011年為電影《[[寶島漫波|寶島漫波]]》擔任監製及演出「Toro哥」一角；2013年，電影《[[戀戀海灣|戀戀海灣]]》擔任監製、編劇及導演。

2007年至2015年期間，擔任[[第44屆金馬獎|第44屆金馬獎]]和[[第49屆金馬獎|第49屆金馬獎]]評審委員、第16屆台北電影節評審委員、[[第50屆金鐘獎|第50屆金鐘獎]]「電視電影／迷你劇集類」評審委員召集人。

2015年，受[[雨果|雨果]]小說的啟發，編寫舞台劇《死刑犯的最後一天》，並擔任主要角色「死刑犯」的演出，於台北「思劇場」首演七場爆滿；於2016及2017年又在高雄「駁二正港小劇場」及台北「華山烏梅劇院」再重演共九場爆滿；「死刑犯的最後一天」劇本出版後，更在2017年入選文學創作類「文化部中小學生優良課外讀物」。

2015年後，陳以文著重於「表演」藝術創作後，也經常在不同電影、電視劇、和舞台劇中呈現不同主要人物。如下介紹......
