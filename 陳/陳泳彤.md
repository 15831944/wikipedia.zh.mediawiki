'''陳泳彤'''（{{lang-en|'''Jennifer Chan'''}}；{{bd|1998年|6月23日|CatIdx=C陳}}），原名'''陳詠桐'''，藝名'''JC'''，[[香港|香港]]女歌手，在16歳時參加歌唱比賽入行，於2016年正式出道。<ref name="All About JC"/>就讀[[嘉諾撒聖方濟各書院|嘉諾撒聖方濟各書院]]時已活躍於校內歌唱比賽<ref name ="JC學校狂掃獎">{{cite news|url=http://hk.apple.nextmedia.com/enews/realtime/20160703/55308915|title =黎明新女校服look唱歌 JC學校狂掃獎|work=[[蘋果日報_(香港)|蘋果日報]]|date=2016年7月3日|}}</ref>，現為[[香港理工大學|香港理工大學]][[市場學|市場學]]系學生。<ref name ="青春的可能"/>

== 简历 ==
2016年6月20日，发表出道作品《說散就散》，并获得当年香港[[觀看次數最多的YouTube影片列表|YouTube十大熱門音樂影片]]之一<ref name ="YouTube香港年度排名">{{cite news|url=http://hk.on.cc/hk/bkn/cnt/lifestyle/20161207/bkn-20161207170848019-1207_00982_001.html|title =YouTube香港年度排名　熊仔頭成冠軍|work=[[東方報業集團|on.cc東網專訊]]|date=2016年12月7日|}}</ref>。

2016年7月23日，在[[黎明|黎明]]澳門[[新濠影滙綜藝館|新濠影滙綜藝館]]舉行的演唱會做嘉賓，第一次公開演唱《說散就散》<ref>{{Citation|last= |title=JC - 說散就散 160723 現場版 Live Performance|url=https://www.youtube.com/watch?v=-OaKUqEvm5Y|}}</ref>，

2017年12月31日，在[[湖南卫视|湖南卫视]]2017年-2018年跨年演唱会與[[白敬亭|白敬亭]]，[[周筆暢|周筆暢]]、[[岑寧兒|岑寧兒]]共同合唱《成都》<ref>{{Citation|last= |title=2018跨年演唱會《成都》白敬亭，岑寧兒，周筆暢，陳詠桐|url=https://www.youtube.com/watch?v=9JhphZlFDZ8|accessdate=2017-12-31}}</ref>、《說散就散》<ref>{{Citation|last= |title=【18湖南衛視跨年】JC陳詠桐：說散就散|url=https://www.youtube.com/watch?v=1PAX5zKCUuk|accessdate=2017-12-31}}</ref>

2018年，參加了《[[2018中国好声音|2018中国好声音]]》，並加入[[庾澄慶|庾澄慶]]战隊，但在六強搶位戰中敗給黃稔欽而遭淘汰<ref>{{cite news |title=【中國好聲音】JC陳泳彤現身唱《說散就散》　會否第二輪被淘汰？ |url=https://www.hk01.com/%E5%8D%B3%E6%99%82%E5%A8%9B%E6%A8%82/221688/%E4%B8%AD%E5%9C%8B%E5%A5%BD%E8%81%B2%E9%9F%B3-jc%E9%99%B3%E6%B3%B3%E5%BD%A4%E7%8F%BE%E8%BA%AB%E5%94%B1-%E8%AA%AA%E6%95%A3%E5%B0%B1%E6%95%A3-%E6%9C%83%E5%90%A6%E7%AC%AC%E4%BA%8C%E8%BC%AA%E8%A2%AB%E6%B7%98%E6%B1%B0 |issue=2018-8-12 |publisher=香港01}}</ref>。
