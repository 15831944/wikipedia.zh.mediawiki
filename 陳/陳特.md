'''陳特'''教授（{{lang-en|Prof. CHEN Te}}，{{BD|1933年|9月|2002年|12月29日}}），曾任[[香港中文大學|香港中文大學]]哲學系教授、[[崇基學院|崇基學院]]學生輔導主任、應林堂舍監、利樹培堂舍監及資深導師。

==簡歷==
陳特於1933年9月生於[[中國|中國]][[廣東省|廣東省]][[梅縣|梅縣]]的書香世家，兄弟五人，陳特為長子。1949年，因戰亂移居[[香港|香港]]，入讀[[珠海書院|珠海書院]]，那時[[唐君毅|唐君毅]]在珠海兼課，成了他的哲學啟蒙老師。1954年珠海書院外文系畢業後，他便入讀[[新亞書院|新亞書院]]第二屆的[[新亞研究所|新亞研究所]]，指導他的是唐君毅、[[錢穆|錢穆]]和[[牟宗三|牟宗三]]，1958年於新亞研究所碩士畢業。1966年赴[[美國|美國]]南伊利諾大學深造，1969年獲博士學位。<ref name="12years">{{ cite book |url=http://www.ajr.com.hk/wp-content/uploads/2014/11/Chandak.pdf |title=生死徘徊十二年 |date=2002 |publisher=A.J.R.慈善基金 |author=陳特 |location=香港 | pages=  |ISBN= }}</ref><ref name="Chen-Talk">{{cite web |url=http://occr.christiantimes.org.hk/art_0107.htm |title=與陳特先生對談 |date= |publisher=基督教線上中文資源中心 |author=周保松 }}</ref>

1969年博士畢業後加入[[香港中文大學|香港中文大學]][[崇基學院|崇基學院]]哲學及宗教學系，主要任教[[哲學|哲學]]概論、[[倫理學|倫理學]]和[[存在主義|存在主義]]，並參與通識教育課程。1999年於中大哲學系退休。陳特退休後仍出任崇基學院資深導師，並繼續為哲學系兼課，亦繼續擔任舍監。<ref name="ChowPoChung2882">{{ cite web |url=http://cusp.hk/?p=2882 |title=夜闌風靜人歸時：紀念陳特先生 |date= |publisher=中大學生報 |author=周保松 }}</ref><ref name="alumni200303">{{ cite web |url=http://www.alumni.cuhk.edu.hk/magazine/200303/html/p40-48.htm |title=訃告 |date=2003-03 |publisher=香港中文大學校友事務處 |author= }}</ref><ref name="Chen-Talk">{{cite web |url=http://occr.christiantimes.org.hk/art_0107.htm |title=與陳特先生對談 |date= |publisher=基督教線上中文資源中心 |author=周保松 }}</ref>

陳特曾於1976至1981年，及1988至1990年，兩度出任崇基學生輔導主任共7年。1992年初至1999年7月，出任崇基通識教育主任七年半。並於1983至1993年出任應林堂宿舍舍監十年；1993至2002年出任利樹培堂舍監九年半。<ref name="12years" />

陳特是位相信基督教的存在主義者，有眾多存在主義哲學家中，他特別欣賞[[祈克果|祈克果]]，尤其是有關「信仰的跳躍」的說法。六十年代初曾任《中國學生周報》的編輯和社長。<ref name="ChowPoChung2882" /><ref name="Chen-Talk"/>

陳特於1990年發現身罹淋巴腫瘤，徘徊生死對抗癌症十二年，曾接受電療及化療。2001年3月，當以為癌症己康復時，發現在胰臟側有腫瘤並迅速擴散，2002年12月29日辭世，享年69歲。<ref name="alumni200303" /><ref name="Chen-Talk"/><ref name="12years" />

== 個人生活 ==
陳特妻子為伍麗卿，育有兩女，並有男女外孫各一人。

2004年12月，陳特的女兒陳孟晢（45歲），連同夫婿羅蔭權（48歲，任職香港中文大學物理學系教授），及女兒羅致平（9歲，就讀香港培正小學四年級正班）到[[泰國|泰國]][[布吉|布吉]]旅遊，遇上[[2004年印度洋大地震|南亞海嘯]]，一家三口不幸罹難。<ref>{{ cite news |url=https://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=27550&Pid=6&Version=0&Cid=150&Charset=big5_hkscs |title=校友徑上的偶遇：懷念陳特老師 |date=2005-02-02 |publisher=時代論壇 |author=和曉梅 }}</ref><ref>{{cite news|url=https://hk.news.appledaily.com/local/daily/article/20050104/4560911 |title=海嘯浩劫後港復課首天，8個學生不見了 |publisher=蘋果日報 |date=2005-01-04 }}</ref><ref>{{cite news|url=http://std.stheadline.com/archive/fullstory.asp?andor=or&year1=2005&month1=1&day1=1&year2=2005&month2=1&day2=9&category=a&id=20050104a01&keyword1=&keyword2=|title=復課少一人，學童發噩夢 |publisher=星島日報 |date=2005-01-04 }}</ref><ref>{{cite news|url=http://the-sun.on.cc/channels/news/20050104/20050104025029_0001.html |title=

==軼事==
以下的中文大學學生因受陳特的影響，決定本科轉讀哲學系：
* 劉旭東：本來主修化學系，大學三年級上完陳特的課後決定轉至哲學系，更因擔心化學系不肯放人，故意將成績考得很差。<ref name="ChowPoChung2882" />
* [[周保松|周保松]]：本來主修工商管理系，大學三年級時轉至哲學系。<ref name="ChowPoChung2882" />
