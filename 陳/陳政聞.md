'''陳政聞'''（{{bd|1977年|6月5日||}}，前任[[民進黨|民進黨]]籍[[一邊一國連線|一邊一國連線]][[高雄市議員|高雄市議員]]（高雄市第三選舉區）。

打出師徒聯手口號，[[陳其邁|陳其邁]]為師，[[陳政聞|陳政聞]]為徒，聯手競選。前者競選高雄市長，後者競選高雄市議員。
2018/11/24九合一大選結果，陳其邁以15萬票之差落選高雄市長，陳政聞的選區有7名候選人搶5個名額，陳政聞以第6名（16941票）落選，是全國最高票落選的議員候選人。

==學歷==
*[[國立中山大學|國立中山大學]]政治研究所碩士
*國立中山大學政治研究所博士班

==經歷==
*曾任2004年陳水扁總統競選連任南台灣青年總部「水龍頭俱樂部」執行長、民進黨高雄市黨部文宣組組長、民進黨高雄市黨部主任委員。
*2010年成為一邊一國連線最初成員之一。
*2018年10月17日，陳政聞於[[三立新聞台|三立新聞台]]政論節目《[[54新觀點|54新觀點]]》中，批評韓國瑜粉絲頁小編王律涵「年輕人可以做錯事，不能做壞事」，影射王律涵使用fb假帳號，遭王律涵回嗆「你們這些大人有多噁心」<ref>{{Cite web |url = https://udn.com/news/story/10958/3430421 |title = 韓國瑜22歲小編上節目 直嗆「你們大人多噁心」 |author =  |date =  |publisher = 聯合新聞網|language = zh-hant |accessdate = 2018年10月19日 |quote = }}</ref>。
*高雄市空中大學講師
*民進黨高雄市黨部第12屆主任委員
*2010年美國國務院IVLP（全球領袖菁英計畫）台灣政治人物代表
*民進黨高雄市議會黨團幹事長
*2018年11月24日，該選區7名候選人搶5個名額，陳政聞第6名落選。
