'''陳純精'''（{{lang-nan-tw|Tân Sûn-cheng}}；{{bd|1878年|12月16日|1944年|2月19日|catIdx=C陳}}），名'''舜'''，字'''純精'''，號'''學山'''，[[以字行|以字行]]。日治時期曾改[[和名|和名]]'''光谷純精'''。原名'''游慶順'''，出生後[[過繼|過繼]]給陳姓人家，遂改姓陳。宜蘭北門口人，[[臺灣|臺灣]][[台灣日治時期|日治時期]][[臺北州|臺北州]][[羅東郡|羅東郡]][[羅東街|羅東街]]首任街長（街相當於今日之[[鎮_(中華民國)|鎮]]），促使羅東成為[[太平山|太平山]]林場的木材集散地，奠定往後[[羅東鎮_(臺灣)|羅東鎮]]商業發展的基礎。

== 早年生平 ==
陳純精於光緒四年（1878年）12月16日出生於今[[宜蘭市|宜蘭市]]。自幼喪父，與母親相依為命。[[光緒|光緒]]十二年（1886年）進入[[宜蘭縣|宜蘭]]的私墊「[[省心齋|省心齋]]」研讀傳統漢學。1894年清日兩國發生[[甲午戰爭|甲午戰爭]]，清朝戰敗後，[[台灣割讓|台灣割讓]]給日本。[[明治|明治]]二十九年（1896年）陳純精結束私墊教育，進入[[宜蘭國語傳習所|宜蘭國語傳習所]]甲科生就讀，專攻國語（[[日文|日文]]）。明治三十一年（1898年）3月畢業後，受命擔任[[宜蘭地方法院|宜蘭地方法院]][[通譯|通譯]]。明治三十五年（1902）轉入宜蘭廳，直到四十一年（1908年）辭職轉任羅東區長。<ref>《臺灣大觀》，頁20。</ref>

明治四十一年（1908年）2月，陳純精被任命為羅東區-{[[街庄長|街庄長]]}-；1910年，街-{庄}-長制遭廢除，他改任羅東區長。[[大正|大正]]二年（1913年）2月，被授與紳章。<ref>《臺灣列紳傳》，台灣總督府，1916年。頁75。</ref>大正六年（1917年），陳純精創立羅東信用組合，參與地方的開發。<ref>《臺灣大觀》，頁20。</ref>

== 羅東街長 ==
大正九年（1920年）市區改正後，陳純精任命[[台北州|台北州]]協議會員，並兼任[[羅東街|羅東街]][[街長|街長]]。

他在羅東街長任內，做出許多影響地方深遠的建設，諸如營林所羅東出張所（林業辦事處）的設置，1921年陳純精捐出十八甲土地，日人於是將原先分別在宜蘭與員山的太平山林場出張所及貯木池移往羅東，使羅東成為太平山林場的木材加工轉運樞紐，進而帶動羅東的工商發展，奠定今日繁華之基礎，時到今日羅東仍是宜蘭縣的商業中心；他其它重要的建設尚有籌設自來水廠，及1930年(昭和5年)促使日本的王子製紙(台灣興業株式會社、中興紙廠前身)前來宜蘭設廠，以及羅東街役場(相當於鎮公所)的設立等。

昭和十五年（1940年），陳純精更改姓氏，改名'''光谷純精'''。<ref>《大眾人士錄－外地、海外篇》，頁65。</ref>

昭和十六年（1941年）4月21日，陳純精離開街長職務。<ref>「光谷羅東街長勇退」，《[[臺灣日日新報|臺灣日日新報]]》八版，昭和十六年4月24日。</ref>

昭和十九年（1944年）2月19日凌晨3時，陳純精因病逝於自宅。<ref>「光谷羅東街長逝世」，《[[臺灣日日新報|臺灣日日新報]]》四版，昭和十九年2月20日</ref>

== 逝世與紀念 ==
陳純精政績卓著，1944年逝世。1949年後人為感念其貢獻，特於[[羅東中山公園|羅東中山公園]]設立陳純精銅像及『陳純精翁紀念碑』。1985年完工的羅東鎮環鎮道路，亦命名為[[純精路|純精路]]，以資紀念。

== 家族 ==
陳純精是台灣政治人物[[陳文茜|陳文茜]]的曾祖父。
