'''陳國星'''，補教藝名'''陳星'''或'''陳興'''，出生於[[澎湖|澎湖]]，為[[臺灣|臺灣]][[高級中學|高級中學]][[國文|國文]][[補教名師|補教名師]]，在臺灣補教界有「北吳岳、南陳星」之稱，但也曾於北部大型補習班任課。

==學歷==

就讀[[臺北市立建國高級中學|臺北市立建國高級中學]]、[[國立臺灣大學|國立臺灣大學]]中國文學系（夜間部）畢業。

==爭議==

===學歷事件===
陳國星在補習班的招生網頁中記載畢業於中山大學文學研究所<ref>{{cite web|url=http://lazynako.pixnet.net/blog/post/37524032-台南同心-陳星國文|title=陳星國文}}</ref>。2017年5月2日網友Orzer於PTT八卦板表示在台灣博碩士論文系統中無法搜尋到陳國星的論文，並公開質疑其學歷的真實性。2017年5月3日中山大學表示，陳星曾在2006年考上該校中國文學系研究所碩士專班，但辦休學2年，2年後因仍未來上課，已在2008年被退學。<ref>{{cite web|url=https://tw.appledaily.com/new/realtime/20170503/1110581/
|title=陳星碩士學歷造假　中山大：沒上過課被退學}}</ref>

===遭指控性侵===
{{main|林奕含事件}}
2017年4月29日台灣作家[[林奕含|林奕含]][[自殺|自殺]]後，林奕含父母林炳煌、賴嘉芳發聲明指其女就是其著作[[房思琪的初戀樂園|房思琪的初戀樂園]]中的主角，是因遭補習班老師誘姦導致憂鬱症而自殺<ref>{{cite web|url=https://www.nextmag.com.tw/breakingnews/politics/338720|title=林奕含自縊後的113天　改變了許多事|date=2017-08-22|}}</ref>，並尚有其他二名被害者，其中一名甚至被黑道恐嚇、強拍裸照<ref>{{cite web|url=https://www.upmedia.mg/pj2_news_info.php?SerialNo=16631|title=「還有3名女學生受害」　林奕含雙親深夜再發聲明|date=2017-05-06|}}</ref>，惟林奕含父母數次表明不願對陳星提告，且拒絕提供其女生前所遺留之個人日記、手札文件與電腦等相關資料予檢察官。<ref>{{Cite web |url=http://www.tnc.moj.gov.tw/ct.asp?xItem=484586&ctNode=32375&mp=020|title=臺南地檢署偵辦陳O星妨害性自主等案件結案新聞稿|language=zh}}</ref>

2017年5月1日檢察官因接獲數人告發而開始偵辦，經陳國星同意由檢察事務官搜索其住處帶回相關證物解析。<ref>{{Cite news |url=https://news.pts.org.tw/article/358092|title=打破沉默 陳星聲明認與林奕含交往2個月|language=zh}}</ref>。2017年8月22日，台南地檢署調查林奕含醫療紀錄、小書原稿、閨密證詞、遺書後認定不存在有證明犯罪的具體事證，全案不起訴。且調查期間未有其他人前來報案，臺南市警察局及家庭暴力暨性侵害防治中心均無任何相關報案紀錄<ref>{{cite web|url=http://www.appledaily.com.tw/realtimenews/article/local/20170822/1187597/|title=南檢公布陳星不起訴原因　指難認林奕含自殺與遭性侵有關|publisher=蘋果日報|date=2017-08-22|}}</ref><ref>{{cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20170822/1187603/|title=林奕含案偵結　檢方：陳國星與林奕含是兩情相悅（動畫）|publisher=蘋果日報|date=2017-08-22|}}</ref>。職權聲請再議部份2017年9月6日台灣高等法院台南分院檢察署審議維持原處分，全案不起訴確定。

林奕含罹患精神疾病的時間點，根據2017年1月9日其親自受訪的報導，談到她於高二16歲（2007年）即固定到精神科接受診療、並每週2次從台南到台北看診，還差一點因為缺席太多而無法畢業<ref name=報導者>{{cite news|url=https://www.twreporter.org/a/mental-disorder|title=成為一個新人——與精神疾病共存的人生|publisher=報導者|author=張子午|date=2017-01-08}}</ref>，且檢察官調查自殺原因時所查扣資料中可見2008年12月在台大就醫病歷，易言之最遲在此之前就罹患精神方面的疾病是可以被確定的。另外，檢察官調查案件時亦調閱雙方2008年1月至2009年12月的通聯紀錄，發現雙方第一次有接觸的時間是2009年5月29號，故林奕含罹患精神疾病是在他們交往前。<ref>{{Cite web |url=http://www.tnc.moj.gov.tw/ct.asp?xItem=484586&ctNode=32375&mp=020|title=臺南地檢署偵辦陳O星妨害性自主等案件結案新聞稿|language=zh}}</ref>

==前往中國大陸開課==
2018年3月，陈星与福州一飞教育培训学校展開交流，交流结束后，陈星以陳藝為名在一飞教育培训学校开通“陈艺国文課堂”的网路师资培训平台，直播教授《乱句重组技巧》、《古诗文概说》等課程。2019年4月，陳星在福州開設課程的消息遭到中國大陸新浪微博網友曝光，引發輿論爭議。福州市教育局聞訊後介入調查，要求福州一飞外语学校关闭“陈艺国文”的网络师资培训平台，并强调“对师风师德问题采取零容忍态度”<ref>{{Cite web |url=http://news.dwnews.com/taiwan/news/2019-04-08/60127706.html|title=台美女作家自杀案 涉案教师赴陆开课被封杀|language=zh}}</ref>。
