'''陳姿伶 Amuy·Muwakay (阿慕伊·牧娃凱)'''（{{bd|1960年|1月15日}}），[[阿美族|阿美族]]人，出生於[[台東縣|台東縣]][[都蘭部落|都蘭部落]]，[[中國國民黨|中國國民黨]]平地原住民政治人物。現任桃園市平地原住民市議員，立法委員[[廖國棟|廖國棟]]辦公室主任，中華民國原住民知識經濟發展協會理事長<ref>[http://www.twedance.org/about03.aspx]中華民國原住民知識經濟發展協會官網</ref><ref>[http://www.chinatimes.com/newspapers/20150620000231-260210]原住民歌舞劇競賽圓滿</ref><ref>
[http://www.chinatimes.com/newspapers/20140816000980-260302]創新能帶動企業升級</ref>，台灣原住民族企業人協會理事長<ref>[http://tailian.taiwan.cn/n1080/n1239/n226102/n226163/n226463/n226583/1603778.html]福建省臺灣同胞聯誼會</ref>，活力E起舞動母語歌舞劇競賽執行長<ref>[https://news.sina.com.tw/article/20180525/26974950.html]2018「活力．E起舞動」第十六屆原住民族青少年及兒童母語歌謠競賽</ref>，國際蘭馨交流協會中華民國總會常務理事<ref>[http://www.siat.org.tw/]國際蘭馨交流協會中華民國總會</ref>，桃園市蘭馨關懷協會榮譽理事長，世界和平婦女會台灣總會監事<ref>[https://wfwp.eoffering.org.tw/]世界和平婦女會台灣總會</ref><ref>[https://wfwp.eoffering.org.tw/contents/text?id=14]世界和平婦女會台灣總會第九屆團隊組織</ref>，曾任台北市美倫國際蘭馨交流協會會長<ref>[http://siat.org.tw/blog/member.php?k=2&t=2&ms=98&m=761]陳姿伶接任會長交接典禮暨晚宴活動</ref>，歷年來致力推動台灣原住民族母語、樂舞、藝術文化向下紮根，恢弘復振；改善原住民弱勢族群、老人、婦女、女孩生活。
== 生平 ==
早年投入後山醫護服務，陳姿伶的專長是麻醉護理專業，自屏東[[慈惠醫護管理專科學校|慈惠護專畢]]業後，先後任職於公部門「台東衛生局」及「署立台東醫院」(現已更名為[[衛生福利部臺東醫院|衛生福利部臺東醫院]])，後來轉往「[[馬偕紀念醫院|台東馬偕醫院]]」、「[[台東基督教醫院|台東基督教醫院]]」任職護理部主任及擔任麻醉工作。之後隨丈夫廖國棟醫師在長濱衛生所擔任護理工作<ref>[https://tw.appledaily.com/forum/daily/20040917/21238679]謙卑以待、凡事謝恩</ref><ref>[http://www.ksnews.com.tw/index.php/news/contents_page/0000237127]誠心見證神蹟</ref>；期間前往「長庚護專」並在「[[長庚醫療財團法人|長庚醫院]]」帶職進修三年，<ref>[http://www.tynews.com.tw/print.php?action=news&id=84700]桃園市平地原住民議員參選人陳姿伶</ref>，直到廖醫生診所開業後擔任護理長一職。
自[[廖國棟|廖國棟]]立法委員從政後，長期在國會辦公室擔任辦公室主任，與廖委員一起深入全台原鄉部落及都會原民地區為族人提供服務，如職業訓練、原鄉長照服務、老人服務、獨居老人及中低收入送餐服務、<ref>[https://video.udn.com/news/471793] 東市新園社區日間托老揭牌</ref>等，參與各類祭典與活動，了解族人同胞的心聲與需求。帶領年輕原民學子進行兩岸文化交流<ref>[http://www.eastnews.tw/index.php?option=com_content&view=article&id=53380:2015-08-21-01-10-03&catid=36:life&Itemid=60]海星原舞團 將再讓對岸驚艷</ref><ref>[http://www.twtimes.com.tw/index.php?page=news&nid=294472]台灣貴州少數民族藝術展演東大飆戲</ref><ref>[http://new.jdonline.com.hk/index.php?m=content&c=index&a=show&catid=23&id=39927]滬台青少年科技夏令營</ref>、與廖委員出訪南島國家推動文化外交、舉辦活力E起舞動母語歌謠暨歌舞劇競賽，復振原住民母語及樂舞文化<ref>[https://news.sina.com.tw/article/20180525/26974950.html]2018活力．E起舞動</ref><ref>[https://money.udn.com/money/story/5635/2792979]2017活力．E起舞動</ref><ref>[http://www.chinatimes.com/realtimenews/20180611003792-260404]只有大海知道</ref>。

== 經歷事蹟 ==
===== 舉辦活力E起舞動歌舞劇競賽 =====
「活力E起舞動」<ref>[https://news.sina.com.tw/article/20180525/26974950.html]2018「活力．E起舞動」第十六屆原住民族青少年及兒童母語歌謠競賽</ref>」全國原住民族青少年及兒童母語歌舞劇競賽，於二○○三年開始舉辦，至今邁向第十六年。從最初的國小母語歌謠競賽，於二○○六年晉升為歌舞劇的模式，同時還增加了國中組及高中組；二○○九年更拓展大學(專)組。為了讓更多學校參賽，且考量學校參加歌舞劇的門檻較高，二○一七年再增辦不分組歌謠競賽。以吸引更多原住民學子在課業之餘學習母語，激發藝術創作才能，讓原住民族的文化得以在現有教育體制中發揚並傳承。

===== 獲頒全國幸福家庭模範表揚 =====
陳姿伶與廖國棟結縭三十三年，二○一七年獲頒「聯合國NGO世界和平婦女會台灣總會」全國幸福家庭模範的表揚。<ref>[http://titv.ipcf.org.tw/news-35614]世界和平婦女表揚幸福家庭 廖國棟夫婦入列</ref><ref>[https://www.peopo.org/news/356543]世界和平婦女會25週年慶啟動創新推展計畫及全國幸福家庭表揚</ref>

===== 協助八仙塵爆中受傷的二十二位原住民青年 =====
在擔任台北美倫分會會長時，得到美國總會的資助，協助當時八里八仙塵爆中受傷的二十二位原住民青年，以及隔年在桃園發生的原住民排灣族住宅區火燒家後的善後工作。<ref>[http://titv.ipcf.org.tw/news-15263]探視塵爆傷者 民團捐贈近30萬慰問金</ref><ref>[http://www.ksnews.com.tw/index.php/news/contents_page/0000781978]立委廖國棟與夫人陳姿伶 持續關懷八仙受害者</ref>

===== 辦理台東社區日間托老服務中心 =====
陳姿伶擔任東甯志工服務協會理事長期間，辦理經由多處台東社區關懷據點，調整成日間托老服務中心模式。服務項目包括：醫護諮詢保健、老人安養服務、獨居老人及中低收入送餐服務等工作。<ref>[http://news.ltn.com.tw/news/life/breakingnews/1661978]關懷據點轉型日托中心 台東市揭牌</ref>

===== 兩岸少數民族文化交流 =====
近二十年來，擔任立法委員廖國棟國會辦公室主任期間，帶領年輕學子進行兩岸少數民族文化交流，與大陸各界加強與台灣原住民族交流互動合作。受邀參加的交流活動如：全國台聯舉辦的「千人夏令營及冬令營」、大陸國家民委每年秋季舉辦「兩岸少數民族中秋聯歡晚會」及「農曆三月三少數民族節慶活動」、國台辦舉辦「台灣少數民族至內地各省市參訪交流活動」、「滬台兩岸中學生文藝交流活動」、陝西西安「兩岸中學生文化交流活動」……等。<ref>[http://tailian.taiwan.cn/n1080/n1125/n2058/n2752/861273.html]臺灣原住民民意代表積極參訪大陸</ref>

== 政策主張 ==
* 原力城市：爭取成立各區都會官方組織；公有地建設各區特色集會所。
* 原力經濟：打造公有原民社會企業；扶植青年創業促進特色產業發展。
* 原力就業：爭取勞工職業訓練增加就業機會；守護勞工職業保險權益。
* 原力文化：爭取公辦民營原民幼兒園；延伸國中小紮根母語文化教育。
* 原力教育：爭取育兒津貼0-6歲福利；國小至高中學雜費營養午餐全免。
* 原力就養：推動醫療及生活急難救助站；爭取建置原民科技長照中心。
* 原力居住：爭取公有地打造原住民特色示範住宅；補貼租屋者五成租金。
* 原力接軌：與中央政策無縫接軌；強力監督及推促市府原民施政效能。
