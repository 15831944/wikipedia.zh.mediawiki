'''陳世娫'''<ref name=官方認證漢字藝名>{{Cite web | language=ko |url=http://www.yonhapnews.co.kr/bulletin/2012/10/22/0200000000AKR20121022214400005.HTML?did=1179m | title=진세연 "신인인데 주연 3개..감사하죠"}}</ref>（{{kor|k='''진세연'''|hanja='''陳世娫'''|rm='''Jin Se-yeon'''}}，{{bd|1994年|2月15日}}），[[韓國|韓國]][[演員|女演員]]，本名'''金允貞'''（{{kor|k='''김윤정'''|rm='''Kim Yun-jeong'''}}），中華人民共和國的名字為常譯「陳世妍」。曾經是和Jewelry-S同經紀公司準備歌手出道的練習生<ref name="test">[http://ent.sina.com.cn/s/j/k/2012-09-28/17513755484.shtml 韓星陳世妍昔日照曝光 天生麗質搶眼球(圖)]</ref>。出道初期，官方基本資料顯示出生年為1993年，後來於節目中坦白自己比官方公布年齡還小一歲。經紀公司的原因是怕在青少年的尷尬年齡比較難被選角。

== 簡歷 ==
陳世娫在1994年於韓國首爾出生，有一位年長三歲的哥哥，小時候開始學習鋼琴，也學習了兩三年芭蕾舞，兒時的夢想是成為一名老師。陳世娫是在中學時期在街上被選中出道的，剛開始是拍攝廣告而展開活動的，而且還做了三個月左右的偶像練習生。
* 2009年陳世娫以“SK電信樸泰桓篇”廣告出道。
* 2010年演出了《[[沒關係，爸爸的女兒|沒關係，爸爸的女兒]]》正式開始其演藝事業，同年還拍攝了電影《[[白色詛咒|白色詛咒]]》。
* 2011年，年僅17歲的陳世娫便出演了[[SBS日日連續劇|SBS日日連續劇]]高收視人氣電視劇《[[我的女兒是花兒|我的女兒是花兒]]》，為其首次擔任第一女主角。憑藉《[[我的女兒是花兒|我的女兒是花兒]]》獲得<2011年SBS演技大賞女新人獎>。
* 2012年，《[[我的女兒是花兒|我的女兒是花兒]]》下檔不到1週，便已被選為[[KBS|KBS]]特別企劃電視劇《[[新娘面具|新娘面具]]》中的女主角吳牡丹一角，憑藉劇中飾演的變臉大師吳牡丹一角獲得<2012年KBS演技大賞女新人獎>，同年8月，《[[新娘面具|新娘面具]]》尚未下檔，便被選為[[SBS週末特別計劃劇|SBS周末特別企畫]]新劇《[[五指|五指]]》中洪多美一角，而劇中洪多美彈奏鋼琴的部分更是陳世娫親自演奏。同年成為韓國中央大學的學生 。
* 2013年出演話劇《Closer》，演出日本電影《愛的話語》。
* 2014年1月，陳世娫接替[[金昭誾|金昭誾]]演出KBS特別企劃電視劇《[[感激時代：鬪神的誕生|感激時代：鬪神的誕生]]》中金玉蓮一角，同年3月在《[[感激時代：鬪神的誕生|感激時代：鬪神的誕生]]》尚未下檔，陳世娫便接演SBS新月火劇《[[異鄉人醫生|異鄉人醫生]]》中宋在熙/韓勝熙一角。憑藉劇中一人分飾兩角的宋在熙/韓勝熙獲得<第三屆大田電視劇節人氣明星賞>，同年10月演出電影《[[危險的相見禮2|危險的相見禮2]]》。
* 2015年陳世娫出演了網絡電視劇《[[高品格單戀|高品格單戀]]》中柳伊玲一角，此劇在中國獲得了超過3億次的點擊觀看率<ref>{{cite web |title=轰动中国的网络剧《高品格单恋》，将在韩国播放 |url=http://korea.china.com/news/20000303/20180329/25187470.html}}</ref>，也令陳世娫在中國獲得大量人氣。
* 2016年陳世娫出演了[[MBC週末特別計劃連續劇|MBC週末特別計劃連續劇]]《[[獄中花|獄中花]]》中獄女一角，憑藉劇中飾演的獄女一角獲得<MBC演技大賞 女子特別企劃部門 優秀演技賞>，此劇以朝鮮時代的女性為主角，並且獲得了高收視。同年演出電影《[[仁川登陸戰_(電影)|仁川登陸戰 (電影)]]》。
* 2018年陳世娫出演了收費電視台[[TV朝鮮週末連續劇|TV朝鮮週末連續劇]]《[[大君－繪製愛情|大君－繪製愛情]]》中成慈賢一角，此劇刷新TV朝鮮電視劇史上最高收視記錄，以5.6%收視率落幕<ref>{{cite web |title=진세연 "조선 최고 미녀 설정+시청률 5%, 대박 기뻤죠 |url=http://news.chosun.com/site/data/html_dir/2018/05/14/2018051401458.html}}</ref>。
* 最新作品為《[[Item_(電視劇)|ITEM]]》。
