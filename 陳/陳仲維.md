'''陳仲維'''（英語：'''Chan Chung-Wai, Victor''' ，{{bd|1963年|10月14日|}}），生於[[香港|香港]]，籍貫[[廣東省|廣東]][[清遠市|清遠]]，香港前[[游泳|游泳]]代表，後來因拍攝飲品廣告獲垂青加入娛樂圈成為[[演員|演員]]，他亦於香港游泳界及慈善團體擁多項公職。

== 生平 ==
陳仲維自小便對演藝有著無形的好感。陳仲維說他十七歲那年，在一個朋友的支援下，瞞著家人和其他朋友，第一次踏上臺階演出舞台劇，自此為演藝的張力所著迷，此後每逢有演出機會，都會盡力把握。

== 游泳事業 ==
陳仲維擅長[[自由泳|自由泳]]及[[蝶泳|蝶泳]]，曾以代表資格為香港出戰[[亞運會|亞運會]]、[[英聯邦運動會|英聯邦運動會]]及[[泛亞太平洋游泳賽|泛亞太平洋游泳賽]]。2007年7月，陳仲維曾代表香港游泳協會參加[[無綫電視|無綫電視]]遊戲[[電視節目|電視節目]]《[[最緊要進步|最緊要進步]]》。

== 賽馬及足球 ==
陳仲維2008年曾替《[[香港蘋果日報|香港蘋果日報]]》[[馬經|馬經]]版挑戰賽中撰文；更於[[足球|足球]]網上版中成為至激讀者並勝出近二千元彩金。
