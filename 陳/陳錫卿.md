'''陳錫卿'''（{{bd|1907年|4月16日|1985年|3月27日}}），字'''錕鋙'''。出身[[南投|南投]][[竹山鎮_(台灣)|竹山]]望族。台灣政治人物，中國國民黨籍。為二戰後首任官派及民選[[彰化縣縣長|彰化縣縣長]]。前後共三屆、任期九年。

== 早年生平 ==
陳錫卿父親陳元龍為公學校訓導，育有六子一女，陳錫卿排行第一。[[台中一中|台中一中]]畢業後，隨後保送[[臺北高等學校|臺北高等學校]]，後進入[[臺北帝國大學|臺北帝國大學]]。原本陳錫卿有資格保送醫科，由於受到[[陳彩龍|陳彩龍]]、[[彭清靠|彭清靠]]、陳紹楨等長輩鼓勵下，認為需要有台灣人走政治，使陳錫卿改讀政治。<ref>〈陳許碧梧女士訪問記錄〉，《口述歷史（五）》。臺北：中研院近史所，1994年6月初版。頁249-250</ref>

== 滿洲時期 ==
[[昭和|昭和]]八年（1933年，[[大同_(滿洲國)|大同]]二年），陳錫卿畢業於[[臺北帝國大學|臺北帝國大學]]文政學部法學科，3月21日與陳許碧梧結婚。適逢[[滿洲國|滿洲國]]大同學院來台舉辦高等文官試驗，錄取後月給160元。相較下臺北帝國大學畢業生一個月僅三十元。<ref>〈陳許碧梧女士訪問記錄〉，《口述歷史（五）》。臺北：中研院近史所，1994年6月初版。頁250-251</ref>遂決定前往[[滿洲國|滿洲國]]，進入大同學院深造。同年10月畢業後，出任滿洲國文教部學務司教育科屬官。[[康德_(年号)|康德]]八年（1941年）6月改任[[北安省|北安省]]文教科長，一年半後調回文教部。

而後到[[上海|上海]]任[[上海特別市政府_(日占時期)|上海特別市政府]]參事，調往[[周佛海|周佛海]]市長處擔任秘書，直到[[二戰|二戰]]結束。

== 彰化縣長任內 ==
1948年，陳錫卿參加[[1948年臺灣省縣長考試|全國縣長考試]]，取得任用資格。同年4月，便奉派擔任[[彰化市|彰化市]]長。

1950年[[彰化縣|彰化縣]]成立後，陳錫卿奉派為首任縣長。在1951年首屆選舉中，當選首任民選[[彰化縣縣長|縣長]]。並在1954年、1957年選舉中連任，直到法規修改。被稱為「三元及第」縣長。<ref>[http://www.th.gov.tw/epaper/site/page/91/1235 竹山地區的名父名子－黃鍚三與黃式鴻、黃仲圖 ]，[[國史館臺灣文獻館|國史館臺灣文獻館]]。</ref>

陳錫卿在縣長任內整建[[八卦山風景區|八卦山風景區]]，並發起倡議建造遠東地區最巨大之[[佛像|佛像]]。於1956年由陳錫卿縣長主持破土奠基開工興建。1961年[[八卦山大佛|八卦山大佛]]完工。<ref>[http://www.chtpab.com.tw/front/bin/ptdetail.phtml?Category=351503&Part=about_D001 八卦山大佛-歷史沿革]，八卦山大佛。</ref>

== 卸任後 ==
卸任後，陳錫卿於1960年出任省政府民政廳長，兼[[彰化銀行|彰化銀行]]常任董事。

1967年被指派[[國民黨|國民黨]]中央黨部副秘書長，兼[[中國電視公司|中國電視公司]]常務董事、臺灣土地開發公司常務監察。1970年出任國民黨營[[齊魯公司|齊魯公司]]董事長兼臺灣省選舉委會主任委員，最後以省政府顧問、行政院顧問退休。

== 家族 ==
