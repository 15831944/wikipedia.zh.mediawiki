'''陳沛然'''（{{lang-en|'''Pierre Chan'''}}，{{bd|1976年}}），出生於[[英屬香港|香港]]。現任[[第六屆香港立法會|香港立法會議員（醫學界功能界別）]]，亦為[[律敦治醫院|律敦治醫院]][[內科|內科]]副顧問[[醫生|醫生]]。

== 早期經歷 ==
陳早年就讀[[天主教伍華中學|伍華書院中學部]]、[[保良局莊啟程預科書院|保良局莊啟程預科書院]]，畢業於[[香港大學|香港大學]][[香港大學醫學院|醫學院]]，後取得[[胃肠学|腸胃肝臟科]][[專科醫生|專科醫生]]資格。

== 政治生涯 ==
自2014至2016年間，擔任了[[香港公共醫療醫生協會|香港公共醫療醫生協會]]會長，2016年開始擔任香港醫務委員會委員<ref>{{cite web|url=https://sites.google.com/site/pierreinlegcoland/|title=Pierre-in-legcoland|date=2016-07-23|accessdate=2016-09-05|language=zh-hk}}</ref>。

自2015年起，參與了爭取公立醫院醫生薪酬機制加薪運動<ref>{{cite web|url=http://news.mingpao.com/pns/dailynews/web_tc/article/20151025/s00002/1445710456069|title=溫文醫生溫文抗爭 「爭的是道理」 陳沛然﹕場仗未算贏 未排除升級|location=[[香港|香港]]|publisher=[[明報|明報]]|date=2015-10-25|accessdate=2016-09-05|language=zh-hk}}</ref>。亦曾就香港公共醫療預算問題<ref>{{cite web|url=http://topick.hket.com/article/1387545/%E5%85%AC%E5%85%B1%E9%86%AB%E7%99%82%E9%86%AB%E7%94%9F%E5%8D%94%E6%9C%83%E6%9C%83%E9%95%B7%E9%99%B3%E6%B2%9B%E7%84%B6%EF%BC%9A%E5%95%9E%E5%AD%90%E5%90%83%E9%BB%83%E8%93%AE|title=公共醫療醫生協會會長陳沛然：啞子吃黃蓮|location=[[香港|香港]]|publisher=[[經濟日報|經濟日報]]|date=2016-03-11|accessdate=2016-09-05|language=zh-hk}}</ref>及醫管局為紓緩公立醫院壓力的議題如藥物重配<ref>{{cite web|url=http://app3.rthk.hk/special/pau/article.php?aid=1659|title=陳沛然：對醫生、護士加病人都唔係好公道|location=[[香港|香港]]|publisher=[[香港電台|香港電台]]|date=2016-03-10|accessdate=2016-09-05|language=zh-hk}}</ref>發表意見。

2016年立法會就醫委會的改革議題上，陳沛然亦代表工會提出改革意見<ref>{{cite web|url=http://www.hk01.com/%E6%B8%AF%E8%81%9E/15586/-%E9%86%AB%E5%A7%94%E6%9C%83%E6%94%B9%E9%9D%A9-%E6%A5%AD%E7%95%8C%E5%80%A1%E5%A7%94%E4%BB%BB%E9%86%AB%E7%94%9F%E8%AE%8A%E7%9B%B4%E9%81%B8%E9%86%AB%E7%94%9F-%E9%AB%98%E6%B0%B8%E6%96%87-%E5%8F%AF%E7%B9%BC%E7%BA%8C%E6%8E%A2%E7%B4%A2|title=【醫委會改革】業界倡委任醫生變直選醫生　高永文：可繼續探索|location=[[香港|香港]]|publisher=[[香港01|香港01]]|date=2016-04-08|accessdate=2016-09-05|language=zh-hk}}</ref>。

2016年7月23日，陳沛然宣佈參選[[2016年香港立法會選舉|2016年香港立法會選舉]]，表示倘若當選，最大的目標是重啟醫委會改革<ref>{{cite web|url=http://www.hk01.com/%E7%AB%8B%E6%B3%95%E6%9C%83%E9%81%B8%E8%88%89/33117/-%E7%AB%8B%E6%9C%83%E9%81%B8%E6%88%B0-%E9%99%B3%E6%B2%9B%E7%84%B6%E5%AE%A3%E5%B8%83%E5%8F%83%E9%81%B8%E9%86%AB%E5%AD%B8%E7%95%8C%E5%8A%9F%E8%83%BD%E7%B5%84%E5%88%A5-%E5%80%A1%E9%87%8D%E5%95%9F%E9%86%AB%E5%A7%94%E6%9C%83%E6%94%B9%E9%9D%A9|title=【立會選戰】陳沛然宣布參選醫學界功能組別　倡重啟醫委會改革|publisher=[[香港01|香港01]]|date=2016-07-23|accessdate=2016-09-05|language=zh-hk}}</ref>。

2016年9月4日的[[2016年香港立法會選舉|立法會選舉]]中，陳沛然結果戰勝前[[民建聯|民建聯]]成員、[[建制派|建制派]]的[[黃以謙|黃以謙]]，當選立法會議員，是立法會內唯一的[[中間派_(香港)|中間派]]議員。

== 政治立場 ==
陳沛然自認政治立場「偏黃」，他反對梁振英連任，也反對「[[袋住先|袋住先]]」方案，但卻不願意「被泛民」，定義自己為中間派，這是因為在醫學界選民中，多數為建制派。他表示若在醫療政策上有不義方案，也會[[拉布|拉布]]盡力阻止惡法通過。他雖然拒絕參加泛民議員的聯署，但卻加入了[[飯盒會|飯盒會]]，又拒絕與[[梁振英|梁振英]]單獨閉門會面，只因希望在政場上不偏左、右，堅持走在中間做實事，冀望能做到同時代表業界內的藍、黃陣營。他要求梁振英不要再推出令社會撕裂的政策，說「醫委會改革令醫患對立；政改一事令警民對立，但警察只是執行職務，事件與他們無關」。<ref>{{cite web|url=http://www.hk01.com/%E6%B8%AF%E8%81%9E/43517/-%E7%AB%8B%E6%9C%83%E6%96%B0%E4%B8%81-%E9%99%B3%E6%B2%9B%E7%84%B6%E8%AA%8D%E5%81%8F%E9%BB%83-%E6%8B%92%E6%94%BF%E6%B2%BB%E7%B6%91%E7%B6%81-%E4%BF%83%E6%A2%81%E7%89%B9%E5%8B%BF%E5%86%8D%E6%92%95%E8%A3%82%E7%A4%BE%E6%9C%83 |title=【立會新丁】陳沛然認偏黃　拒政治綑綁　促梁特勿再撕裂社會|publisher=[[香港01|香港01]]|date=2016-09-18|accessdate=2016-09-18|language=zh-hk}}</ref>

== 其他議題 ==
陳沛然除關注業界的急症室和醫管局監管問題，本身是足球迷的他亦關心香港的足運發展和樹木保育等議題。
