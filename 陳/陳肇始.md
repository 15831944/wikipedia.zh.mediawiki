'''陳肇始'''教授{{JP}}（{{lang-en|Professor Sophia Chan Siu-chee}}, JP，{{bd|1958年}}），祖籍[[廣東|廣東]][[潮州|潮州]]<ref>{{Cite web |url=https://hk.news.appledaily.com/local/daily/article/20170911/20149067 |title=隔牆有耳：娥班子最多架己冷 |date=2017-09-11 |publisher=蘋果日報}}</ref><ref>{{Cite web |url=https://www.hk01.com/article/140138/ |title=政壇諸事町 問責班子多「家己冷」　陳肇始自爆最愛鹵水鵝 |date=2017-12-09 |agency=香港01}}</ref>，[[香港大學|香港大學]]護理學教授，現任[[香港特別行政區政府|香港特別行政區政府]][[食物及衞生局|食物及衞生局]]局長。加入政府前為[[香港大學護理學院|香港大學護理學院]]院長、[[香港大學李嘉誠醫學院|香港大學李嘉誠醫學院]]助理院長。<ref>{{Cite web|url=https://www.med.hku.hk/summit2017/b5/speakers_Sophia_Chan.html|title=陳肇始教授|accessdate=31/01/2018|author=|date=|publisher=}}</ref>

== 簡歷 ==
=== 教研工作 ===
陳肇始曾就讀[[聖保祿中學|聖保祿中學]]，擁有[[英國|英國]][[曼徹斯特大學|曼徹斯特大學]]教育碩士、[[美國|美國]][[哈佛大學|哈佛大學]][[公共衛生|公共衛生]]碩士以及[[香港大學|香港大學]]哲學博士學位。早年在香港及[[伦敦|倫敦]]接受普通科及兒科護理培訓，並從事此方面的臨床工作。在1977年加入[[瑪麗醫院|瑪麗醫院]]任學護，3年後成為註冊護士。她在1995年加入香港大學[[護理學系|護理學系]]任講師從事學術研究，其後她專注教學、科研及行政管理的工作，出任護理學教授和系主任，至2004年出任香港大學醫學院助理院長。

=== 政府工作 ===
陳肇始一直專注於控[[煙|煙]]工作，研究領域為公共衞生、煙草成癮治療及保護兒童免[[二手煙|二手煙]]影響。2009年加入[[吸煙與健康委員會|吸煙與健康委員會]]，開創香港首個戒煙輔導培訓計劃。2011年，她擔任[[世界衞生組織|世界衞生組織]]顧問，為訓練專業醫護人員透過倡導及教育工作，進行戒除煙癮介入治療提供意見。<ref>{{Cite news|url=http://hk.apple.nextmedia.com/news/art/20170622/20065010|title=「知客」變局長　陳肇始「升呢」之路全面睇|date=2017-06-22|work=蘋果日報}}</ref>

2012年11月1日，她正式出任食物及衞生局副局長，<ref>{{Cite news|url=http://www.news.gov.hk/tc/categories/admin/html/2012/10/20121031_152810.shtml?pickList=ticker|title=兩副局長及五政助獲聘任|date=2012-10-31|work=香港政府|deadurl=yes|archiveurl=https://archive.is/20121218112211/http://www.news.gov.hk/tc/categories/admin/html/2012/10/20121031_152810.shtml?pickList=ticker|archivedate=2012-12-18}}</ref>期間專注於[[控煙|控煙]]及[[戒煙|戒煙]]的推廣工作。時任食物及衞生局局長[[高永文|高永文]]曾受訪指她較「柔性」，例如食衛局曾在2015年年提出修例改革醫委會，卻遇上議員「拉布」，她一度因這議題而落淚。<ref name="MP">{{Cite news|url=https://news.mingpao.com/pns/dailynews/web_tc/article/20170622/s00001/1498067557286|title=唯一女局長 陳肇始獲讚人緣好|date=2017-06-22|work=明報}}</ref>港大感染及傳染病中心總監[[何栢良|何栢良]]及醫委會改革三方平台醫生代表[[歐耀佳|歐耀佳]]接受《[[明報|明報]]》訪問時，亦同稱讚陳肇始願意聽從各方面的聲音，是一個人緣好及做事圓滑的人。<ref name="MP" />

2017年6月，她獲委任為食物及衞生局局長。她是林鄭月娥內閣中唯一的女性問責局長，<ref>{{Cite news|url=http://hk.on.cc/hk/bkn/cnt/news/20170621/bkn-20170621072730851-0621_00822_001.html|title=新班子：國務院任命　陳肇始唯一女問責局長|date=2017-06-21|work=東方日報}}</ref>亦是歷來首個由護士而非醫生出身的[[食物及衞生局|食物及衞生局局長]]。<ref name="MP" />有意見指陳肇始未必能駕馭醫生，她則回應在過去30年無論在臨牀、科研、教育以至副局長的工作上，多次與醫生及其他醫療人員合作，雙方合作無間，有信心與各界溝通。她在未來五年任期希望能推動基層醫療及社區照顧，減少醫院服務需求。<ref name="MP" />她於2017年7月1日正式就任食物及衞生局局長。

== 榮譽 ==
陳肇始曾獲以下榮譽：
* 英國皇家內科醫學院公共衞生學院頒授院士（榮譽）
* 美國護理學院院士
* 香港大學學院教學獎章（2005）
* 香港大學傑出教學獎（2009）
* [[太平紳士|太平紳士]]
