'''陳昱安殺父案'''是一起在2010年9月13日發生的一起弒親逆倫案件。[[新北市|新北市]][[新莊區|新莊區]]男子陳昱安因不滿父親叫其外出工作，持菜刀、生魚片刀瘋狂砍殺111刀致死，於2013年1月31日由[[台灣最高法院|最高法院]]依殺害直系血親尊親屬罪判處[[死刑|死刑]]定讞，於[[土城看守所|土城看守所]]等候[[槍決|槍決]]。2019年1月18日，於台北看守所內以橡皮筋勒脖自殺身亡。<ref>[https://www.cna.com.tw/news/asoc/201901185001.aspx 陳昱安殺父判死定讞6年 看守所內橡皮筋勒脖自盡],中央社，2019年1月18日</ref>

== 凶嫌 ==
陳昱安（原名陳文雄<ref name=":0">最高法院刑事判決: 一○二年度台上字第四四六號</ref>，1982-2019），[[新北市|新北市]][[新莊區|新莊區]]人。因祖父（於2001年逝世）過度寵溺，曾開玩笑說：會送他遺產<ref>https://www.youtube.com/watch?v=CDqBxYWUvEs YouTube影片：逆子百刀殺父 喪失繼承權</ref>，但由於陳男因長期失業在家長期玩[[電腦|電腦]]，不肯外出工作，常遭父親叨念，因此有了行凶動機。

在行兇前曾有多次買開山刀回家，但被其父沒收。後來，陳昱安寫下了「殺父筆記本」，開始計劃殺父，並對家人不利。筆記本，內容模仿[[韓國|韓國]]經典電影《[[朋友_(電影)|朋友]]》，完整記載殺人方式，用字遣詞充滿怨恨；而且，刀械的選用也很講究，就連「刀子的厚度也要仔細研究」，他有實際拿一根雞骨頭來做實驗，他認為，刀不用大，有6吋就能殺人，如果刀身厚度太薄，刀身就會斷裂。所以，經過屢次測試後，最終，選上生魚片刀、菜刀，然後攻擊時刀子刺中對方後，刀身還要旋轉90度，令人髮指。

2019年1月18日於北所持收集而來的橡皮筋套於脖子窒息而亡。<ref>[https://www.mirrormedia.mg/story/20190118inv002 弒父死刑犯陳昱安　台北看守所上吊身亡]</ref>

== 案發過程 ==
2010年9月6日，陳父再也無法忍受長子好吃懶做，限期他5日後搬離家中，陳嫌為此怒火中燒，決定將計畫付諸行動，隔日晚上，他先後在打鐵店及某賣場購買生魚片刀、菜刀，由於父親堅持要他搬離住處，陳嫌只好先帶著簡單行李氣憤離開。

任職紡織廠保全員的陳父，同月13日晚上11時許，要出門上大夜班，他一打開公寓大門就撞見陳嫌，陳父當場沒好氣地怒斥：「你都被趕出家門了，怎麼還有臉回來？」<ref name=":0" />豈知陳嫌突然便拿出預藏的菜刀，朝父親頸部砍去，陳父情急下撥開菜刀，還將陳嫌眼鏡也一併打落。

陳父眼見逆子喪心病狂，趕緊大聲呼喊救命，陳嫌卻沒住手，隨即抽出預藏在鐵門下的生魚片刀，使勁往父親胸部猛刺，拔出刀鋒時不忘旋轉，由於陳嫌力道過大，近23公分長的刀刃，最後卡在陳父右胸內，而殺紅了眼的他，又重拾菜刀朝父親頸部、上下肢及身軀猛砍，累了還換手行兇，警方獲報趕抵時，陳父已肚破腸流慘死血泊中。

陳昱安被捕時，對警方說：「不用太在意。」對於殺父行為還不知恥地聲稱：「後悔不存在我心中。」對於其他家人，陳嫌嗆聲表示恨之入骨，「我也想對他們行兇！」事後陳母對長子深感絕望，不願原諒，還寫信懇求法官判死，2013年1月30日，最高法院認定陳男極惡兇殘，判死定讞。<ref>https://tw.appledaily.com/new/realtime/20160926/954848/  【凶宅打卡】殺父筆記本之逆子111刀弒親案 蘋果日報</ref>

== 後續起訴審判 ==
*陳昱安雖領有身心障礙手冊，且有輕度精神分裂症，但自稱行兇時意識清楚且殺意強烈，在同年11月，遭[[板橋地檢署|板橋地檢署]]求處死刑。
*2011年6月28日，[[板橋地方法院|板橋地方法院]]依殺害直系血親尊親屬罪判處死刑，同時，同年9月，也裁定喪失遺產繼承權。
*2012年8月15日，[[臺灣高等法院|臺灣高等法院]]判處死刑、褫奪公權終身。
*2013年1月31日，[[最高法院|最高法院]]駁回上訴，所以判處死刑、褫奪公權終身定讞。<ref>http://www.appledaily.com.tw/appledaily/article/headline/20130131/34806065/applesearch/逆子111刀弒父死刑定讞  逆子111刀弒父死刑定讞by 蘋果日報</ref>
