'''陈国坤'''（'''Danny Chan Kwok Kwan'''，1975年8月1日－），人称'''小龙'''，是[[香港|香港]]男演员、导演及摄影师。原为特约演员，獲[[周星驰|周星驰]]賞識，于2000年担任电影《少林足球》的排舞师，及后因外貌神似[[李小龙|李小龙]]而成为《少林足球》里一主要演员。陳國坤曾憑《[[功夫_(电影)|功夫]]》中的“斧頭幫大佬琛哥”一角，獲[[2005年香港電影金像獎|第二十四屆香港電影金像獎]]“最佳男配角”提名。

2014年10月25日，陈国坤与香港女歌手[[黄伊汶|黄伊汶]]在[[峇里島|峇里島]]完婚。<ref>[http://ent.southcn.com/8/2014-10/27/content_110901047.htm 陈国坤、黄伊汶巴厘岛大婚 陈小春现身送祝福]</ref>

2017年10月，陳國坤接受訪問時表示最初在做電影配樂的音樂人[[劉以達|劉以達]]、[[韋啟良|韋啟良]]的錄音室中做助手，又認識了一些副導演，在不少電影做過臨時演員。而[[韋啟良|韋啟良]]負責《少林足球》預告片的音樂，陳國坤將碟送到[[周星馳|周星馳]]的公司。剛巧[[周星馳|周星馳]]看到，之後[[田啟文|田啟文]]走出來，拿張招募演員表格給他填，最後[[周星馳|周星馳]]問陳國坤想不想做演員，他說想，就做了戲中的四師兄<ref name=專訪/>。

== 背景 ==
陳國坤於[[中華人民共和國|中國]][[廣東省|廣東省]][[寶安縣|寶安縣]]（今[[深圳市|深圳市]]）出生，五歲時來港生活，住在[[黃大仙區|黃大仙區]]<ref name=專訪>{{cite web |url=https://bka.mpweekly.com/interview/shall-we-talk/20171028-88448 |title=【獨家專訪】陳國坤．浪子為愛定下來 |author=王志強|date=2017-10-28 |publisher=明周 |access-date=2018-12-18}}</ref>。他家有兩位胞兄，父母皆為漁民<ref>{{web reference|URL=http://ent.ltn.com.tw/news/breakingnews/1456673|title=《明星燒臘店》陳國坤貧困出身 因李小龍改變一切|work=自由時報|year=2015年|date=9月26日}}</ref>。其外祖父則為[[陳錫桃|陳錫桃]]（曾任[[深圳市|深圳市]]人大副主任，已逝）。陳國坤在1993年畢業於[[五旬節聖潔會永光書院|五旬節聖潔會永光書院]]。中學畢業後在眼鏡88當銷售員，後來晉升為驗光師監督。他之後當過侍應、廚房學徒、酒保、調酒師、股票經紀、地盤工人、裝修工人、三行工人以及旺角波鞋街鞋類售貨員。陳國坤在[[劉以達|劉以達]]與[[韋啟良|韋啟良]]的錄音室中做助手時將載有《少林足球》預告片音樂的光碟送到[[周星馳|周星馳]]的公司。結果[[周星馳|周星馳]]著[[田啟文|田啟文]]給陳國坤一份演員招募的表格。陳國坤填完便離去。其後，他在[[林子善|林子善]]的聯絡下到訓練班旁聽。周星馳詢問其當演員的意願後安排他在《[[少林足球|少林足球]]》中演出，並簽約成為周氏公司的旗下藝人，為期八年。直至2004年，陳國坤才成立自己的製作公司。他現時主力拍攝電影及網絡劇。監製[[宋本中|宋本中]]為其事業上的合作伙伴。<ref name=專訪/>

陳國坤於2014年迎娶歌手[[黃伊汶|黃伊汶]]<ref>{{web reference|URL=https://www.orientalsunday.hk/%E5%A8%9B%E8%81%9E/%E7%B5%90%E5%A9%9A%E5%85%A9%E5%91%A8%E5%B9%B4-%E9%99%B3%E5%9C%8B%E5%9D%A4%E6%99%92%E9%9D%9A%E7%9B%B8%E5%86%A7%E9%BB%83%E4%BC%8A%E6%B1%B6/|title=結婚兩周年 陳國坤晒靚相冧黃伊汶|work=東方新地|year=2016年|date=11月1日}}</ref>，現育有一子。兒子陳真於2016年3月出生。<ref>{{web reference|URL=https://news.mingpao.com/ins/instantnews/web_tc/article/20170801/s00007/1501586671198|title=【繼續唧都唔笑】陳真Chok過壽星老竇陳國坤|work=明報|year=2017年|date=8月1日}}</ref>
