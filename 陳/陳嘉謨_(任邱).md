'''陈嘉谟'''（{{bd|？||1927年|}}）<ref>网上有小说称其生于同治十一年（1874年），但并未给出依据。</ref>字'''峴亭'''，[[直隷省|直隷省]][[河間府|河間府]][[任丘县|任丘县]]人，[[中华民国|中华民国]]军事将领。<ref name=lai>{{Cite book<!--|和書-->|author = 来新夏等|title = 北洋軍閥史 下冊|year = 2000|publisher =  [[南開大学|南開大学]]出版社|isbn = 7-310-01517-7}}</ref><ref name=xu>{{Cite book<!--|和書-->|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref>

==生平==
陈嘉谟早年在直系领导人[[曹锟|曹锟]]所率的陸軍第3師内逐步升职。1922年（[[民国紀元|民国]]11年）8月，获授陸軍中将位。1924年（民国13年），被[[湖北省_(中華民国)|湖北]]督軍[[萧耀南|萧耀南]]任命为蒲通鎮守使兼第25師師長。1924年6月7日，获授[[北洋将军府|将军府]]儀威将軍。<ref>1924年6月8日第2950号. 《政府公报》第二〇九册.</ref>1925年（民国14年），任湖北第25師師長兼[[武漢市|武漢]]警備总司令。<ref name=lai/><ref name=xu/>

1926年（民国15年）2月，萧耀南急病而死，[[吴佩孚|吴佩孚]]任命陈嘉谟接任湖北督办兼省長。同年10月，[[中国国民党|中国国民党]]的[[北伐|北伐]]軍攻克[[武昌|武昌]]，陳嘉謨被俘虏。<ref name=lai/><ref name=xu/><ref>{{Cite book<!--|和書-->|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref>

1926年12月，国民政府由广州迁到武汉，成为[[武汉国民政府|武汉国民政府]]。1927年1月17日，[[中国国民党中央执行委员会暨国民政府委员临时联席会议|中国国民党中央执行委员会暨国民政府委员临时联席会议]]（简称“武汉临时联席会议”）第14次会议上，[[邓演达|邓演达]]提出，“武昌城所俘之[[刘玉春|刘玉春]]、陈嘉谟二贼，前决定由人民审判委员会审判……但至今并未提出审判，人民屡请审判毫无结果。在南昌方面，[[张凤岐|张凤岐]]、[[岳思寅|岳思寅]]、[[唐福山|唐福山]]等皆由人民判决枪毙，今特提议从速审判。”2月9日，武汉临时联席会议第22次会议正式审议并通过了武汉国民政府司法部制定的《反革命罪条例》。1927年2月10日，“人民审判委员会”首次运用该条例，审判了陈嘉谟和刘玉春，该案被定名为“陈刘反革命案”。当日的审判以“择日再判”结束，后来却未再审理或宣判，此案不了了之。<ref>王奇生，北伐时期的地缘、法律与革命——“反革命罪”在中国的缘起，近代史研究2010年01期</ref>
[[File:督理湖北軍務兼省長陳宴請英國退還庚款委員團攝影.jpg|thumb]]]
1927年（民国16年）陳嘉謨获得释放，居住在[[天津|天津]]，同年中逝世。<ref name=lai/><ref name=xu/>
