'''陳其寬'''（{{bd|1921年| |2007年|6月5日|catIdx=Chen陳其寬}}），生於[[北京|北京]]，[[建築師|建築設計師]]、藝術家、[[畫家|畫家]]。

== 生平 ==
1921年，陳其寬於[[北京|北京]]出生。1933年，初中就讀於南京中學、鎮江中學。1937年，高中就讀於鍾南中學、國立二中。1944年，畢業於[[國立中央大學|國立中央大學]]建築系（今[[東南大學建築學院|東南大學建築學院]]）。

1946年，到[[南京|南京]]基泰建築事務所擔任設計師。兩年後（1948年），前往[[美國|美國]][[伊利諾州立大學|伊利諾州立大學]]深造，隔年獲建築碩士學位。1950年，於美國[[加州大學洛杉磯分校|加州大學洛杉磯分校]]藝術系進修。

1951年，進入[[沃爾特·格羅佩斯|沃爾特·格羅佩斯]]主持的[[哈佛大學|哈佛大學]]建築研究所，並加入其創建的葛羅匹斯建築事務所擔任設計師。

1952年，任教於[[麻省理工學院|麻省理工學院]]建築系。

1954年，與建築師[[貝聿銘|貝聿銘]]合作設計[[台灣|台灣]]的[[東海大學_(台灣)|東海大學]]校園<ref>{{Cite book|author=[[漢寶德|漢寶德]]等|title=《建築之心：陳其寬與東海建築》|publisher=田園城市出版|location=[[臺北市|臺北市]]|year=2003|isbn=9867705238|language=zh-tw|accessdate=2011-07-18}}</ref>。

1960年，任[[東海大學建築系|東海大學建築系]]主任。

1967年4月30日，榮獲第一屆建築金鼎獎（同屆獲獎的還有[[王大閎|王大閎]]、[[修澤蘭|修澤蘭]]、[[陳仁和|陳仁和]]、[[沈祖海|沈祖海]]、[[楊卓成|楊卓成]]、[[張紹載|張紹載]]<ref>{{Cite web |url=https://www.ttv.com.tw/news/tdcm/viewnews.asp?news=0107714 |title=建築金鼎獎頒獎典禮| accessdate=2018-08-30 |work=台視新聞}}</ref><ref>{{Cite web |url=https://www.ttv.com.tw/news/tdcm/viewnews.asp?news=0106656 |title=獲建築金鼎獎的十位建築師作品:照片和模型展覽| accessdate=2018-09-04 |work=台視新聞}}</ref>。）

1969年，參與規劃[[國立中央大學|國立中央大學]]於[[中壢區|中壢]][[在台復校|復校]]的校區整體設計。

1980年，任東海大學工學院院長。

2007年，病逝於美國[[舊金山|舊金山]]，享年八十六歲。
