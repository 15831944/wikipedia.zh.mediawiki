'''陳真'''，是香港作家[[倪匡|倪匡]]為1972年電影《[[精武門_(電影)|精武門]]》創作的人物，設定是[[民國|民國]]時期武術家[[霍元甲|霍元甲]]的弟子。陳真這個角色最初是為當時功夫巨星[[李小龍|李小龍]]量身訂造的，後來被改拍成多套電影及電視劇°

== 經典場面 ==
1972年《[[精武門_(電影)|精武門]]》電影經典場面，日本人在[[霍元甲|霍元甲]]喪禮送贈「[[東亞病夫|東亞病夫]]」牌匾侮辱，李小龍飾演的陳真獨自挑戰日本武館「[[虹口道場|虹口道場]]」，以一敵眾將日本人打得落花流水，將「東亞病夫」牌匾交還，以宣揚中國人非弱者的不可欺精神。此後多套的精武門電影或電視都必加入這一幕，幾乎成了陳真電影的必備情節。此情節亦是[[倪匡|倪匡]]所撰寫，這些都由他在《[[am730|am730]]》的專欄及電台節目《[[光明頂_(廣播節目)|光明頂]]》中透露。<ref>倪匡在電臺節目「[[倪匡看世界|倪匡看世界]]」中披露，陳真是他筆下虛構的人物（25/03/2007）</ref>

== 真假問題 ==
2007年，[[倪匡|倪匡]]在[[香港電台|香港電台]]節目《倪匡看世界》中披露，陳真是他創作的虛構人物，寫著「東亞病夫」的塊匾同樣出自他筆下。<ref>[http://www.rthk.org.hk/rthk/radio1/MyWay/ 倪匡看世界] {{webarchive|url=https://web.archive.org/web/20080314224937/http://www.rthk.org.hk/rthk/radio1/MyWay/ |date=2008-03-14 }}，2007年3月25日節目</ref><ref>[http://1.andrew0668.com:8080/fcgi-bin/lb51/topic.cgi?forum=50&topic=201 古龍國 - 小說討論區 - [電台節目]倪匡看世界]</ref>

根據香港資深媒體人[[劉天賜|劉天賜]]所稱，「陳真」是倪匡借用霍元甲喪禮名單上一人的名字，將他設定為霍元甲弟子。<ref>{{cite web |url=http://mingpaomonthly.com/%E8%AA%B0%E7%BD%AE%E4%BA%9E%E8%A6%96%E6%96%BC%E6%AD%BB%E5%9C%B0%EF%BC%9F%E3%80%80%EF%BC%88%E5%8A%89%E5%A4%A9%E8%B3%9C%EF%BC%89/ |title=劉天賜：誰置亞視於死地？ |publisher=明報月刊 | date=2016-09-30}}</ref>

但據電影人[[陳嘉上|陳嘉上]]所稱，倪匡當年翻查了霍元甲的訃告，發現其中有一個弟子名叫陳真，結果寫進了《精武門》之內，但電影內陳真的故事是虛構的。<ref>{{cite news|title=真有陈真其人？甄子丹很能打？|url=http://cqcbepaper.cqnews.net/cqcb/html/2010-09/23/content_1242242.htm|publisher=重庆晨报|date=2010-09-23}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite news |first= |last=阿觀 |title=再續精武門 |url=http://www.am730.com.hk/article.php?article=25660 |publisher=am730 |date=2010-10-04 }}</ref>

此外，有說倪匡在[[亞洲電視|亞洲電視]]節目《[[今夜不設防_(亞洲電視)|今夜不設防]]》之中，說過陳真是真有其人，報紙上霍元甲的治喪名單上有著陳真這名字。<ref>[http://shelf3d.com/va7V3d4vvZc 精武英雄 Fist Of Legend (1994年) 導演陳嘉上Gordon Chan 的專訪 - 第一節]</ref>

== 形象定義 ==
2010年9日，甄子丹主演的電影《[[精武風雲·陳真|精武風雲·陳真]]》首影會上，有外國記者問及新陳真與舊版的區別，甄子丹斬釘截鐵的說，「大有不同」，他對陳真的定義也非常直接：「陳真就是李小龍。」甄子丹表示，陳真形象是由李小龍塑造的，他本人很崇拜李小龍，有一些元素是以前的李小龍版就有的，比如穿白衣中山裝，耍雙截棍，但在一些動作和造型上還是有很大不同。所以在該片中，他仍然穿白衣中山裝、耍雙截棍，吼著「中國人不是東亞病夫」。<ref>{{cite news |title=甄子丹：陈真就是李小龙 |url=http://entertainment.dbw.cn/system/2010/09/21/052738558.shtml |work=东北网 |publisher=大河网 |date=2010-09-21 }}</ref>

亦有評論說陳真事蹟是以霍元甲的徒弟[[劉振聲|劉振聲]]為原型。<ref>[http://culture.people.com.cn/BIG5/40479/40482/4130593.html 陳真和劉振聲 誰是霍元甲的真徒弟? 《今晚報》]</ref>

== 改拍的電影及電視劇 ==
1972年，由[[李小龍|李小龍]]飾演的[[嘉禾|嘉禾]]電影《[[精武門_(電影)|精武門]]》，這[[角色|角色]]設定為[[中國|中國]][[清朝|清]]末[[武術家|武術家]][[霍元甲|霍元甲]]的弟子，在為被[[日本|日本]]人害死的[[師傅|師傅]]報仇之後，被[[日本人|日本人]]殺死，這個角色後來一再被拍成[[電影|電影]]、[[電視劇|電視劇]]。

1981年，香港[[麗的電視|麗的電視]]劇集《大俠霍元甲》中，加入了陳真的角色，由[[梁小龍|梁小龍]]飾演。在大結局中，陳真踢館後被巡捕人員通緝，最後被亂槍射殺。

1982年2月，香港[[麗的電視|麗的電視]]劇集《陳真》，劇中陳真繼續由[[梁小龍|梁小龍]]飾演，劇情描寫陳真並未真正死去，是廳長受陳真愛國情意感動，刻意換上空彈頭免陳真一死。陳真自此與霍元甲之子[[霍東閣|霍東閣]]隱姓埋名過活，最終陳真與[[日本人|日本人]]佐藤同歸於盡，這部香港電視劇在當時中國大陸亦相當受歡迎。<ref>[http://minghaoliu.myweb.hinet.net/the%20fist.htm 麗的電視 陳真]</ref>

1994年，在[[李連杰|李連杰]]飾演陳真的電影《[[精武英雄|精武英雄]]》中，日本留学的陈真听闻师傅死讯后返回中国，协助师父之子霍廷恩主持精武门的业务，在為被[[日本|日本]]人害死的[[師傅|師傅]]報仇之後，再神秘離開。

1995年，香港[[亞洲電視|亞洲電視]]與[[衛星電視中文台|衛星電視中文台]]聯合製作的劇集《[[精武門_(電視劇)|精武門]]》中，由[[甄子丹|甄子丹]]飾演陳真，結局則為租界內日本駐軍射殺。

2001年，推出电视剧《陈真后传》，仍由[[梁小龍|梁小龙]]饰演陈真。

2008年，中國電視劇《精武陳真》推出，內容和人物名字都和1981年麗的電視版《陳真》接近一樣，明顯向該劇致敬，就連主題曲都用回舊版徐小明唱的《大號是中華》。

2010年，電影《[[精武風雲·陳真|精武風雲·陳真]]》延續《[[精武英雄|精武英雄]]》的情節，講述[[甄子丹|甄子丹]]飾演的陳真並未真的死亡，而是逃亡歐洲，參戰以方便匿藏，後逃回上海當間諜，再次對抗日本人、破壞日本人的陰謀。<ref>[http://www.funnewsblog.com/精武風雲-陳真-甄子丹.html 精武風雲·陳真 甄子丹搶爆鏡] {{webarchive|url=https://web.archive.org/web/20160310075637/http://funnewsblog.com/%e7%b2%be%e6%ad%a6%e9%a2%a8%e9%9b%b2-%e9%99%b3%e7%9c%9f-%e7%94%84%e5%ad%90%e4%b8%b9.html |date=2016-03-10 }}2010年9月</ref>
