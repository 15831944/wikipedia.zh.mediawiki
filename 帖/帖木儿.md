== 出身及奪取汗位 ==
帖木兒出生於[[西察合台汗国|西察合台汗国]][[渴石|渴石]]一个信奉[[伊斯兰教|伊斯兰教]]的[[突厥化蒙古人|突厥化蒙古人]][[埃米尔|埃米尔]][[巴鲁剌思氏|巴鲁剌思氏]]的家庭，是[[成吉思汗|成吉思汗]]七世的女系驸马。父親名叫[[塔剌海_（帖木儿之父）|塔剌海]]，是[[渴石|渴石]]的一名[[埃米爾|埃米爾]]，是[[巴鲁剌思氏|巴鲁剌思氏]]一族。相传其祖先源自成吉思汗的高祖父[[敦必乃|敦必乃]]，敦必乃第三子为[[合出里|合出里]]，合出里生子[[亦儿占赤|亦儿占赤]]，亦儿占赤生子[[速忽赤臣|速忽赤臣]]，速忽赤臣生子[[忽必来|忽必来]]、[[忽都思|忽都思]]、[[布鲁罕|布鲁罕]]和[[哈剌察儿|哈剌察儿]]，哈剌察儿生子[[亦连吉儿|亦连吉儿]]，亦连吉儿生子[[不儿赫勒|不儿赫勒]]，不儿赫勒生子塔剌海和哈吉，塔剌海就是帖木儿的父亲(哈吉是他叔叔)。也有学者认为成吉思汗世系之说不可信，帖木儿应是[[突厥人|突厥人]]，而非[[蒙古人|蒙古人]]<ref>勒内·格鲁塞《草原帝国》，北京商务印书馆北京2004年版。</ref>。

後人所知的帖木儿的傳奇歷史，絕大部分來自於其家傳的自傳。從其出身背景與控制能力來看，至少在帖木兒父辈时，其家族勢力已經支配整個[[渴石|渴石]]地區，並與西察合台王族通婚。1360年，河中地區大勢底定，[[禿忽魯帖木兒|禿忽魯帖木兒]]控制了大部分地區，並收帖木儿為[[參贊|參贊]]。但帖木儿實際上受到西察合台汗國王族兩個方面的羅致，他選擇投入較為弱小，但與其有姻親關係的[[忽辛|忽辛]]陣營。因其所處勢力較小，這一段時期帖木儿與忽辛可說是一面[[游擊|游擊]]並儲備反擊的力量，其最為人所熟知的腿傷亦是在此時期受創。

1363年忽辛終於扶持[[合不勒沙|合不勒沙]]为[[察合台汗國#西察合台汗国（1346年－1369年）或（1346年－1402年）|西察合台]][[可汗|可汗]]。1369年，帖木儿杀死[[結義|結義]]兄弟[[忽辛|忽辛]]，控制了西察合台可汗合不勒沙。第二年，帖木尔处死合不勒沙，自称为[[苏丹|苏丹]]，建立了帖木儿帝国。

== 擴張帝國 ==
首先，恢復西察合台汗國的秩序與疆土之後（也就是征服[[中亚河中地区|中亚河中地区]]），再七次征伐[[東察合台汗國|東察合台汗國]]之后，将其納入疆土。随后征服[[卡爾提德王朝|东伊朗]]（1381年）、[[花剌子模|花剌子模]]（1387年）。

1393年最终征服[[莫扎法爾王朝|西波斯]]；并在1391年、1394—1395年两次征伐[[金帐汗国|金帐汗国]]。但由於[[撒拉伯卡|撒拉伯卡]]（今[[伊朗|伊朗]]境內）發生異族叛亂，使其一改恢復[[蒙古帝國|蒙古帝國]]光榮之進軍方向，將征服目標由蒙古金帐汗国、察合台汗国轉向周圍各國。帖木儿原來希望恢復蒙古帝國的光榮，因此本來皆以各蒙古汗國為攻擊目標。但在之後卻發現到蒙古人不是敵人，異族較蒙古人更可能阻礙他的大業。

從此以後，他師法蒙古人的屠宰掠夺策略，將反叛之撒拉伯卡屠宰，他之後1394年、1398年征伐[[德里苏丹国|德里苏丹国]]時，也維持同樣的策略。最著名一戰是與金帳汗國[[脫脫迷失|脫脫迷失]]汗的[[昆都爾察河谷大戰|昆都爾察河谷大戰]]，徹底擊敗金帳汗國<ref>{{cite web |url=http://lib.ecit.edu.cn/guoxue/%E5%85%B6%E4%BB%96%E5%8E%86%E5%8F%B2%E4%B9%A6%E7%B1%8D/%E4%B8%93%E9%A2%98%E7%B1%BB/%E5%8E%86%E5%8F%B2/%E9%92%A6%E5%AF%9F%E6%B1%97%E5%9B%BD/qchg_x.htm |language=zh |title=钦察汗国（下） |publisher=東華理工大學圖書館 |accessdate=2014-03-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20140309155721/http://lib.ecit.edu.cn/guoxue/%E5%85%B6%E4%BB%96%E5%8E%86%E5%8F%B2%E4%B9%A6%E7%B1%8D/%E4%B8%93%E9%A2%98%E7%B1%BB/%E5%8E%86%E5%8F%B2/%E9%92%A6%E5%AF%9F%E6%B1%97%E5%9B%BD/qchg_x.htm |archivedate=2014-03-09 }}</ref>。

== 与明朝、奥斯曼帝国的關係 ==
1368年，[[明太祖|明太祖]][[朱元璋|朱元璋]]建立[[明帝国|明帝国]]，在同年的北伐之中使得蒙古人对中国的统治结束，并退居塞北，汉文史籍史称[[北元|北元]]，后在[[明太祖北伐|明太祖北伐]]的打击下灭亡。明军在[[明朝|明]][[洪武|洪武]]五年（1372年）将[[甘肃行中书省|甘肃行中书省]]纳入明帝国疆域后要求当时[[中亚|中亚]]各国“进贡”。

1387年起，帖木儿曾多次遣使[[进贡|进贡]]。

1396年，帖木儿扣押各國[[使节|使节]]，包括[[明帝国|明帝国]]與[[奥斯曼帝国|奥斯曼帝国]]使節，表示對外宣戰，開始第二階段的征战擴張。1398年征伐[[印度|印度]][[德里苏丹国|德里苏丹国]]的[[德里|德里]]；1399年—1402年西征[[小亚细亚|小亚细亚]]与[[马木留克王朝|马木留克王朝]]和[[巴格达|巴格达]]。

1398年，再次[[禁錮|禁錮]]、扣留明朝特使。

1402年7月20日在[[安卡拉战役|安卡拉战役]]大败[[奥斯曼帝国|奥斯曼帝国]]，俘其[[苏丹|苏丹]][[巴耶塞特一世|拜牙]]（即“闪电”[[巴耶塞特一世|巴耶塞特一世]]），使其帝国疆域成为从[[印度|印度]][[德里|德里]]到[[小亚细亚|小亚细亚]]、[[美索不达米亚|美索不达米亚]]的大[[帝国|帝国]]。他擊敗當時如日中天、不断擴張中的[[鄂圖曼帝國|鄂圖曼帝國]]，間接地缓解了[[伊斯兰文化|伊斯兰文化]]对[[基督宗教|基督宗教]]文化與整個[[歐洲|歐洲]]的渗透。而從[[小亞細亞|小亞細亞]]帶回的藝術家、工匠與學者，留給[[撒馬爾罕|撒馬爾罕]]無數無價的傳世建築，使該處在其孫[[兀魯伯|兀魯伯]]的經營下，成為了中亞[[伊斯蘭文化|伊斯蘭文化]]的重心。

1404年11月27日率领20万[[士卒|士卒]]进攻[[明朝|明朝]]，结果1405年2月18日在进军途中，在[[訛答剌|訛答剌]]病死，終結了其輝煌無敵的征戰歷史<ref>[http://ydcyjl.blog.163.com/blog/static/1263593420117142847194/ 《看世界》2011年第15期 撒马尔罕的王陵]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 逝去後的帝國局勢 ==
{{无来源章节|time=2018-06-16}}
帖木儿有四子，长子[[贾汉吉尔_(帖木儿之子)|贾汉吉尔]]早逝、次子[[乌马尔·沙伊赫|乌马尔·沙伊赫]]阵亡，三子[[米兰沙|米兰沙]]在帖木儿逝世后不久也去世了；帖木儿立[[太孫|太孫]][[皮儿·马黑麻|皮儿·马黑麻]]，但帖木儿的另一個愛孫[[哈利勒|哈利勒]]起兵争位，发生内讧，最终由帖木儿的四子[[沙哈鲁|沙哈鲁]]平定内讧，继位为王。

沙哈鲁一反帖木儿对[[明朝|明朝]]的的攻擊政策轉變為与明帝国友好，在永乐十一年（1413年）恢复遣使朝贡；同年[[明成祖|明成祖]]派遣中官李达、吏部验封司员外郎[[陈诚_(明朝)|陈子魯]]等9人使团出使[[帖木儿帝国|帖木兒汗國]]，行报施之礼；使節團的成就之一，就是重新與帖木儿汗國建立良好的外交關係。

帖木儿帝国也如[[蒙古帝國|蒙古帝國]]般，裂解為多个勢力，而且諸子間還经常發生鬥爭，1449年[[沙哈鲁|沙哈鲁]]長子[[兀魯伯|兀魯伯]]被杀后帝国分崩离析，不同地区统治者常年互相杀伐。

{{Commonscat|Timur|帖木儿}}
帖木儿帝国在1507年被[[突厥|突厥]]的[[乌兹别克汗国|乌兹别克汗国]]的[[穆罕默德·昔班尼|穆罕默德·昔班尼]]所灭。1526年，帖木儿的后裔[[巴布尔|巴布尔]]征服[[印度|印度]][[德里苏丹国|德里苏丹国]]建立了[[莫卧儿帝国|莫卧儿帝国]]，名义上存在到1857年。

== 帖木兒的司法制度 ==
=== 民、刑分開 ===
帖木兒將[[民事|民事]]、[[刑事|刑事]]及行政訴願分開；一部份[[司法官|司法官]]，專理刑事案件；另一部份則為廉政[[司法官|司法官]]，專理[[官吏|官吏]][[貪污|貪污]]案件。至於外國[[使者|使者]]，與民間往來，則另有禮官負責。如此，帖木兒境內司法，有條不紊，職責之劃分，極為清楚<ref name="克拉維約東使記">《[[克拉維約東使記|克拉維約東使記]]》，北京商務印書館1985年版。</ref>{{rp|p.160}}。

=== 案件判決過程 ===
帖木兒汗帳建立之處，[[法官|法官]]即就其旁，建立帳幕三座，工作其中。所有在押人犯，以及原被告，皆來此處候審。法官分別案件之性質，聽取訴辯後，立下判詞，但於執行之前，先呈帖木兒鑒核，候其裁可，再分別執行。法官倘作書面之判詞時，先由繕書記寫，登過底簿後，再由法官於判決書上蓋印。另有一吏，自法官之底簿上，將原文抄於清謄簿上，其判詞之尾，亦由法官蓋印。清謄簿係為呈帖木兒批閱者。帖木兒於此薄上用過印璽，即為批准。帖木兒用於司法案件之璽，其印文為「公正」一語，文外四邊，有小圓圈3個。此間為法官司繕寫判詞之書記，其職務極為忙碌。<ref name="克拉維約東使記"/>{{rp|p.160-161}}

== 驛站制度 ==
據曾經出使[[帖木兒帝國|帖木兒帝國]]的[[西班牙|西班牙]]人[[克拉維約|克拉維約]]所說，[[帖木兒帝國|帖木兒帝國]]境內有相當完善的驛站制度。帖木兒於其汗國內，遍設驛站，備置馬匹，以供換乘。較大之驛站上，常備良馬百匹，多至二百匹<ref name="克拉維約東使記"/>{{rp|p.101}}。為供給各驛站人員之需用及飼養馬匹起見，各驛站旁建有館舍。帖木兒之使者每至一驛，立刻更換新座騎並繼續前進，驛伕伴往下站，將原馬領回<ref name="克拉維約東使記"/>{{rp|p.101}}。

萬一馬匹在途中疲乏或倒斃，使者即自路上所遇之騎者借用坐騎，繼續前進，絕不中止。按驛站之規定，帖木兒之使者，於路上倘有需用馬匹之處，迎面遇有行人，無論地位高低，即屬[[王爵|王]][[公爵|公]][[貴族|貴族]]、抑或販夫走卒，只須提出商借之請求，對方必須立時將馬出借。不論誰對此種義務皆無推卻之餘地。倘有遲疑或竟予拒絕時，即有被[[斬首|斬首]]之危險。使者於途中借到之馬，馳至附近驛站，再更換他騎<ref name="克拉維約東使記"/>{{rp|p.101}}。

帖木兒不僅於赴[[撒馬爾罕|撒馬爾罕]]之大路上施行此項制度，在汗國其他各道上皆行此法。以此，驛站與郵傳使者隨地可見，其作用在使各地消息靈通，而帖木兒之巡吏晝夜活動於四方<ref name="克拉維約東使記"/>{{rp|p.101}}。

=== 對驛使的要求 ===
帖木兒並未限定其驛使晝夜奔馳，只規定每日行程為50程（Fusah，每Fusah約為5[[公里|公里]]）。如此已足令其使者，不敢稍怠。故途中坐騎，即使因奔馳過度而猝斃，亦所不惜，因帖木兒所最注重者為迅速<ref name="克拉維約東使記"/>{{rp|p.101}}。

== 帖木兒治軍 ==
=== 軍隊組織 ===
帖木兒軍隊中，百人為最小單位，置百人長，千人及萬人之上，皆置有千人長及萬人長。全軍置有統帥。遇有戰事發生，則徵調各級將士，按敵勢之強弱，分定徵調兵數之多寡。
帖木兒平日將軍中馬匹及牲畜，分撥與族人飼養。凡負責飼養之人，莫不加意看管。因倘有不周到之處，立刻刑罰隨至，罪重者且因之被殺。<ref name="克拉維約東使記"/>{{rp|p.161}}
=== 軍隊紀律管理 ===
帖木兒每次出征，軍中置有法官多人，隨時接受訴狀，加以裁判。<ref name="克拉維約東使記"/>{{rp|p.160}}

== 傳說 ==
帖木兒作戰沿習[[蒙古|蒙古]]習風，對於不投降者，破城之後大多[[屠城|屠城]]，以畏服各邦，在歷經屢次征伐[[俄羅斯|俄羅斯]]及[[黑海|黑海]]北部後，使當地人民留下痛苦的記憶。在俄羅斯與[[東歐|東歐]]地區流傳：帖木兒的遺骸若遭移動，將使天下遭遇大兵災。

1941年，[[斯大林|斯大林]]下令发掘帖木儿墓葬，在棺椁被打开的第2天，[[蘇德戰爭|蘇德戰爭]]爆发了。
