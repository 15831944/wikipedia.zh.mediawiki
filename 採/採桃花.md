《'''採桃花'''》（{{lang-th|จีนเด็ดดอกไม้}}），原名《'''讀孟浩然詩感興'''》<ref name="sirindhorn"/><ref name="rsu">{{cite web |url=http://www.rsu.ac.th/libarts/chinese/ตรุษจีน57/Details.pdf |title=你好！春节 —— 诗临溢辉 |last= |first= |date=2015-02-13 |website= 兰实大学 |publisher= |access-date= 2017-12-05 |quote=}}</ref>，是一首[[泰语|泰语]][[诗歌|诗歌]]，共分为4段。1990年由[[诗琳通公主|诗琳通公主]]所赋，1996年由[[北京大学|北京大学]]泰语专业教授[[裴晓睿|裴晓睿]]翻译为[[五言绝句|五言绝句]]。<ref name="sirindhorn">{{cite web|url=http://bclc-tu.com/yoksai47_page1.html|title=bclc-tu หยกใสร่ายคำ|last=|first=|date=|website=bclc-tu|publisher=|access-date=2017-12-05|quote=|archive-url=https://web.archive.org/web/20171205042042/http://bclc-tu.com/yoksai47_page1.html|archive-date=2017-12-05|dead-url=yes}}</ref>

《采桃花》最早收录于诗琳通公主所著作的《[[琢玉诗词|琢玉诗词]]》中，第一段为曾经翻译的[[孟浩然|孟浩然]]诗句《[[春晓|春晓]]》，后三段为公主自己根据原作风格而创作的诗句。<ref name="pbic">{{cite web|url=http://www.pbic.tu.ac.th/main/sites/default/files/16_Asscoc.Prof_.Zhang%20Gongchang_CH.pdf|title=诗琳通公主与中国文化的泰国传播|last=|first=|date=|website=Pridi Banomyong International College|publisher=张弓长、冯忠芳|access-date=2017-12-05|quote=|archive-url=https://web.archive.org/web/20171205093005/http://www.pbic.tu.ac.th/main/sites/default/files/16_Asscoc.Prof_.Zhang%20Gongchang_CH.pdf|archive-date=2017-12-05|dead-url=yes}}</ref>其最后一句是歌颂[[中泰关系|中泰友谊]]的祝福诗，其中“中泰手足情，绵延千秋好”流传较广<ref name="zjol" /><ref name="people">{{cite web |url=http://politics.people.com.cn/n/2013/0411/c70731-21102914.html |title=泰国公主诗琳通上海赋诗祝福中泰友谊 |last= |first= |date=2013-04-11 |website=人民网|publisher=中国新闻网 |access-date= 2017-12-05 |quote=}}</ref>。

== 詩文 ==
<blockquote>
春眠不覺曉，處處聞啼鳥；夜來風雨聲，花落知多少。

花繁須早折，勿待香韻消；花濃情亦重，君心爽且豪。

友情深如海，前程萬里遙；奼紫嫣紅日，春園競多嬌。

中泰手足情，綿延千秋好；擷花相餽贈，家國更妖嬈。
</blockquote>

== 创作背景 ==
诗琳通公主曾于1987年7月26日写给友人的一封信中写道，于[[澳大利亚|澳大利亚]]但不记得城市的博物馆，参访一间中国文物的收藏室时见墙上贴有《[[春晓|春晓]]》的诗句，然后便把这首诗译成[[泰文|泰文]]。后来，泰国[[帕特雅拉民樂團|帕特雅拉民樂團]]指揮要求为詩琳通公主赐予中国风泰国古典乐曲《采桃花》时，《春晓》成为了《采桃花》的第一段唱词。<ref name="zjol">{{cite web |url=http://jnnews.zjol.com.cn/jnnews/system/2011/04/21/013636658_01.shtml |title=国际政坛掀起汉语热 各国政要竞相"秀"中文 |last= |first= |date=2011-04-21|website=中国景宁新闻网|publisher=中国景宁新闻网 |access-date=2018-01-30|quote=}}</ref>

此后诗琳通公主又将《春晓》之后加入三篇唱词，符合《采桃花》唱词的风格，她写道“所有的好花都是表达心灵的好工具，而采花的目的是为了赠给对方，增进[[中泰关系|中泰友谊]]”。1990年5月初，诗琳通公主完成了《采桃花》唱词。1996年，[[裴晓睿|裴晓睿]]将《采桃花》的后三段诗词翻译成[[中文|中文]]。<ref name="sirindhorn"/>
