'''伶仃洋大橋'''是1983年提出連接[[香港|香港]]和[[珠海|珠海]]，跨越[[伶仃洋|伶仃洋]]的跨海大桥；计划現已搁置，由[[港珠澳大桥|港珠澳大桥]]取代。

==构思==
伶仃洋大橋計劃的構想最初在1983年由香港[[合和集團|合和集團]]主席[[胡應湘|胡應湘]]及[[珠海市|珠海市]]提出<ref>{{Cite web |url=http://www.southcn.com/news/gdnews/zhzt/yga/ygadq/200308071260.htm |title=存档副本 |access-date=2012-03-25 |archive-url=https://web.archive.org/web/20120620141735/http://www.southcn.com/news/gdnews/zhzt/yga/ygadq/200308071260.htm# |archive-date=2012-06-20 |dead-url= }}</ref>，原來只預算連接香港和珠海，由香港的[[屯門區|屯門區]][[爛角咀|爛角咀]]起始<ref>{{cite web|url=http://www.pland.gov.hk/pland_en/p_study/comp_s/cbl/CBLstudy_stage2_chin/ch05_c.htm|title=5. 后海灣海岸公路|accessdate=2011-11-03|work=規劃署|archive-url=https://web.archive.org/web/20150402163719/http://www.pland.gov.hk/pland_en/p_study/comp_s/cbl/CBLstudy_stage2_chin/ch05_c.htm|archive-date=2015-04-02|dead-url=no}}</ref>，經過[[內伶仃島|內伶仃島]]、[[淇澳島|淇澳島]]，到達珠海的[[金鼎|金鼎]]，珠海於境內興建了[[淇澳大橋|淇澳大橋]]；這個構思在中國大陸的舊版地圖仍可見到，當時建議的大橋長度亦不及現時的[[港珠澳大橋|港珠澳大橋]]長。

==計劃擱置==
[[File:HZMB_route.svg|thumb]]，連接[[香港|香港]][[屯門|屯門]]和[[珠海|珠海]]。]]
隨著香港的[[青嶼幹線|青嶼幹線]]和[[北大嶼山公路|北大嶼山公路]]的通車，新[[香港國際機場|香港國際機場]]的啟用，胡應湘又提出香港應善用[[青馬大橋|青馬大橋]]等基建設施；於是大橋香港起點改為[[大嶼山|大嶼山]]，原伶仃洋大橋計劃擱置，並再加入[[澳門|澳門]]（單Y方案）和[[深圳市|深圳市]]（雙Y方案）<ref>{{Cite web |url=http://paper.wenweipo.com/2003/01/23/NS0301230005.htm |title=存档副本 |access-date=2012-03-25 |archive-url=https://web.archive.org/web/20121108055913/http://paper.wenweipo.com/2003/01/23/NS0301230005.htm# |archive-date=2012-11-08 |dead-url= }}</ref><ref>{{cite web |url=http://www.waou.com.mo/wa/2011/02/20110214a.htm |title=存档副本 |accessdate=2012-03-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20140814150636/http://www.waou.com.mo/wa/2011/02/20110214a.htm |archivedate=2014-08-14 }}</ref>，最後選擇了單Y方案，並改名「[[港珠澳大橋|港珠澳大橋]]」。

== 時間表 ==
* 1983年：首次提出興建「香港珠海跨海大橋」方案。
* 1989年：[[珠海|珠海]]公佈「伶仃洋大橋」計劃。
* 1993年：珠海[[市委書記|市委書記]][[梁廣大|梁廣大]]提出了興建伶仃洋跨海大橋的方案。
* 1997年：伶仃洋跨海大橋方案獲[[中華人民共和國國務院|中華人民共和國國務院]]批准，但因當時的[[香港政府|香港政府]]反對而擱置。
* 2001年1月：[[淇澳大橋|淇澳大橋]]建成通車。
* 2002年：廣東省委常委、常务副省长[[歐廣源|歐廣源]]表示要修建[[深圳|深圳]]到珠海隧道。
* 2002年11月：國務院總理[[朱鎔基|朱鎔基]]訪港，表示支持興建港珠澳大橋。
* 2003年7月中：[[廣東省|廣東省]]常委表示支持興建港珠澳大橋。
* 2003年7月底：[[曾蔭權|曾蔭權]]專程前往北京商討興建港珠澳大橋計劃，伶仃洋大橋計劃由港珠澳大桥计划取代。
