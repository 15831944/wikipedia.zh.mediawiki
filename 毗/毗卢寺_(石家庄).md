'''毗卢寺'''位于中国[[河北省|河北省]][[石家庄市|石家庄市]][[新华区_(石家庄市)|新华区]][[杜北乡|杜北乡]][[上京村|上京村]]，[[石津总干渠|石津总干渠]]东北岸，1996年被列为[[第四批全国重点文物保护单位|第四批全国重点文物保护单位]]。毗卢寺占地28亩，它以存有明代精美的宗教壁画而驰名中外，同时还保存有一批西汉石刻、雕像等珍贵历史文物。

==历史==
毗卢殿中央佛台上供奉着[[毗卢遮那佛|毗卢遮那佛]]（印度语，意为光明普照），故该寺名为毗卢寺。据《[[方舆汇编|方舆汇编]]》和寺内现存碑碣记载，毗卢寺创建于唐朝天宝年间，宋、元、明各朝均曾重修，《真（正）定府志》亦有记载。原来寺院规模大、建筑多，现仅存释迦殿（前殿）和毗卢殿（后殿）。两殿内均绘有[[壁畫|壁画]]，绘制面积共200多平方米，是我国目前保存较为完好的明代重要壁画之一。<ref>{{Cite web|url=http://travel.hebnews.cn/2017-04/18/content_6433345.htm|title=石家庄毗卢寺耐人寻味的壁画传奇|accessdate=|author=河北新闻网|date=|publisher=}}</ref>

毗卢寺建造完成后，历经朝代变迁，遭受过多次劫难还能保留至今，可以说是个奇迹。其中要感谢曾任中华全国美学会长的王朝闻和洪波。1948年上京村进行[[土地改革运动|土改]]时，部分村民要求把毗卢寺所存的殿堂拆除，把木料、砖、瓦分到各户，作为当时土改工作负责人的王朝闻和洪波，一面给当时的[[华北人民政府|华北人民政府]]建议，要求政府加强对毗卢寺文物的保护工作，一面苦口婆心劝导村民，告诉他们毗卢寺的壁画是“国宝”，在二人的共同努力下，毗卢寺最终得以保存下来。<ref>{{Cite web|url=https://weibo.com/p/1001594020998780378195|title=石家庄上京：一个叫“京”的村子，有媲美敦煌的明代壁画 - 石家庄发布|accessdate=2018-10-15|author=|date=|work=weibo.com|publisher=}}</ref>

== 古树 ==
毗卢殿前两棵[[侧柏|侧柏]]，距今一千多年历史，依然生机勃勃，树干伟岸挺拔，都象西倾斜，角度一致，奇特壮观。像两位佛教信徒虔诚朝拜迎接西方佛祖，故名迎佛柏。据石家庄市园林局有关人士透露，目前石家庄城区共有古树33株，其中千年以上的古树仅有4株，分别为新华区毗卢寺的2株侧柏、振头一街关帝庙内的1株国槐，东岗头古槐胡同的老槐树也名列其中。<ref>{{Cite web|url=http://news.sohu.com/21/63/news214916321.shtml|title=石家庄几多古树几多风景|accessdate=2018-10-15|work=news.sohu.com}}</ref>

== 壁画 ==
毗卢寺壁画共200多平方米，分布在释迦殿和毗卢殿内。尤以毗卢寺殿壁画最为精彩。毗卢殿壁画有122平方米，绘有122组，508位神抵人物，每组均标榜题名。分上、中、下三层交错排列。布局巧妙自然，组合井然有序，层次分明，错落有致，具有浑然一体、壮美辉煌的宏伟气势和效果。

毗卢寺壁画是中华民族传统文化的一座宝库，壁画艺术中的一颗明珠，是我国以壁画形式绘制的[[水陆画|水陆画]]（佛教寺院中举行水陆法会时所用的神像画）重要作品。与甘肃[[敦煌壁画|敦煌]]、北京[[法海寺_(北京市)|法海寺]]、山西[[永乐宫|永乐宫]]壁画为我国最富盛名的宗教壁画，但其他三家描绘的只是某一教派的内容，惟有毗卢寺壁画集[[儒家|儒]]、[[佛教|释]]、[[道教|道]]三教于一堂，集三教人物于同一画面，气势壮观、富丽堂皇。毗卢寺前后两殿绘有水陆画共计二百多平方米，内容丰富，人物众多，比著名的永乐宫[[三清殿|三清殿]]壁画《朝元图》的人物还多近一倍，其构图、线描、着色、人物刻画等方面都达到了相当高的水平，具有较高的艺术价值，对研究[[元朝|元]][[明朝|明]]时期的社会生活、风俗习惯以及古代美术具有重要价值。

== 文物保护 ==
“这些壁画已经有五六百年的历史了，一旦消失，就会永远没有了。因此，我们请了[[敦煌研究院|敦煌研究院]]数字中心的工作人员为我们进行数字化拍摄，便于留存完整的数字化资料。” 对这里同样有着深厚感情的郑建飞院长介绍，壁画受环境尤其是气候影响非常大，每到伏天，就会非常潮湿，久而久之就会脱落。“西北部过来的雨水多，西面墙脱落得厉害一些，平时就是湿度、环境当中的粉尘，包括人呼出来的二氧化碳，对壁画都会有一定的影响。”

2012年，来自敦煌研究院数字中心的工作人员对毗卢寺正式启动了壁画数字化工作，对壁画进行全方位、高精度的信息采集，再选取每幅照片的中心变形几乎为零的部位，用6000余张照片拼接处理成整幅的全景图像，1︰1比例精确体现壁画。壁画数字化工作的负责人孙洪才表示，拍摄中灯光都用冷光，便于保持壁画色彩，也不会对壁画造成损害，每张照片都会达到很高的精确度。孙洪才说“像毗卢寺壁画这么大规模的壁画，保存得这么完好，在国内是不多见的，而且绘画技艺高超。”
