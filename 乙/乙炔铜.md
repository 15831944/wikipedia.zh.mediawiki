'''乙炔铜'''，或称'''乙炔亚铜'''，是一种[[有机铜化合物|有机铜化合物]]，[[化学式|化学式]]为'''[[铜|Cu]]<sub>2</sub>[[碳|C]]<sub>2</sub>'''。这是一种对热和冲击敏感的[[高爆炸药|高爆炸药]]，是一种[[金属乙炔化合物|金属乙炔化合物]]。它类似[[乙炔银|乙炔银]]和[[碳化钙|碳化钙]]，虽然它并不叫[[碳化物|碳化物]]。由于高灵敏度和反应导致它没有利用价值，它仍然很有趣，因为它是鲜有的几种在引爆后不释放[[气体|气体]]的爆炸物。

== 制备 ==
乙炔铜可以在[[氨气|氨气]]存在下，[[乙炔|乙炔]]气体通过[[氯化亚铜|氯化亚铜]]溶液制备：
:C<sub>2</sub>H<sub>2</sub> + 2CuCl → Cu<sub>2</sub>C<sub>2</sub> + 2HCl
反应产物是红色沉淀。该反应可用来检测乙炔。

== 性质和危害 ==
乙炔铜可以发生[[爆炸|爆炸]]，爆炸时不产生气体。化合物的阳离子在分解反应中作为氧化剂。
:Cu<sub>2</sub>C<sub>2(s)</sub> → 2 Cu<sub>(s)</sub> + 2 C<sub>(s)</sub>

潮湿的乙炔铜较稳定，可以安全地处理。干燥的乙炔铜极其敏感，摩擦即可引爆。[[摩擦感度|摩擦感度]]为0.1 N。<ref name="Explosivstoffe">Köhler, J.; Meyer, R.; Homburg, A.: ''Explosivstoffe'', zehnte, vollständig überarbeitete Auflage, Wiley-VCH, Weinheim 2008, ISBN 978-3-527-32009-7.</ref>

乙炔铜可以在[[铜|铜]]管或含铜量高的合金管道内部形成，这可能会导致剧烈的爆炸声。<ref>{{cite web |url = http://www.msha.gov/Accident_Prevention/Tips/acetylenegas.htm |title = Mine Safety and Health Administration (MSHA) - Accident Prevention Program - Miner's Tips - Hazards of Acetylene Gas |accessdate = 2008-06-08 |deadurl = yes |archiveurl = https://web.archive.org/web/20080706064005/http://www.msha.gov/Accident_Prevention/Tips/acetylenegas.htm |archivedate = 2008-07-06 }}</ref>这被认为是乙炔厂发生爆炸的原因，导致这样的工厂放弃铜作为结构中的材料。<ref>{{cite web|url=http://www.ilo.org/encyclopedia/?print&nd=857200158 |title=存档副本 |accessdate=2013-02-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20071001010729/http://www.ilo.org/encyclopedia/?print&nd=857200158 |archivedate=2007-10-01 }}</ref>[[石油化工|石油化工]]中使用的铜[[催化剂|催化剂]]在一定条件下也会有风险。<ref>{{cite web|url = http://aiche.confex.com/aiche/s06/preliminaryprogram/abstract_43768.htm |title = The Safe Use of Copper -Containing Catalysts in Ethylene Plants |accessdate = 2008-06-08}}</ref>
