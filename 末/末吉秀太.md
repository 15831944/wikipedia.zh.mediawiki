'''末吉秀太'''（{{jpn|j=末吉 秀太|hg=すえよし しゅうた}}，1986年12月11日）是日本的[[歌手|歌手]]、[[舞者|舞者]]。所屬公司[[愛貝克思集團|愛貝克思集團]]，為團體[[AAA_(團體)|AAA]]的成員之一。[[長崎縣|長崎縣]][[佐世保市|佐世保市]]出生，長崎縣[[西海市|西海市]]長大。<ref>[http://talent.yahoo.co.jp/pf/detail/pp187926 末吉秀太]Yahoo!検索（人物）</ref><ref>[http://news.mingpao.com/ins/aaa%E6%B8%AF%E9%96%8B%E9%A8%B7%20%E6%9C%AB%E5%90%89%E7%A7%80%E5%A4%AA%EF%BC%9A%E6%88%91%E6%84%9B%E4%BD%A0/web_tc/article/20150407/s00007/1428411415145 AAA港開騷末吉秀太：我愛你(22:06) ]明報新聞網</ref>

== 來歷 ==
原本是男性4人團體「FRIENDS」的副隊長、也發行了CD，歌曲的名字叫做《Be Together》。

曾就讀Star light九州學校。

2005年擔任[[鈴木亞美|鈴木亞美]]在「第56回NHK紅白歌合戰」・「CDTV」節目中的伴舞。 

2006年在[[富士電視台|富士電視台]]電視劇「他們的海・VIII」-Sentimental Journey-　飾演關祐輔。第一次擔任主演。

2012年開始每年連續與AAA的隊友宇野實彩子擔任日本青少年舞蹈大賽評委。<ref>{{cite web |url= http://www.news24.jp/entertainment/news/1624086.html |title = コピーキッズ＆ジュニア部門の優勝チームが決定！ |accessdate=2012-08-11|publisher=日テレNEWS24}}</ref><ref>{{cite web |url= http://www.news24.jp/entertainment/news/1629424.html |title =SAM驚く「ここまでレベルが上がるとは」 |accessdate=2013-08-04|publisher=日テレNEWS24}}</ref><ref>{{cite web |url= http://www.news24.jp/entertainment/news/1635260.html |title = SAMも納得「レベルの高い大会になった」 |accessdate=2014-08-17|publisher=日テレNEWS24}}</ref><ref>{{cite web|url=http://7andi-kids-dance-festival.jp/final/|title=第4回セブンアンドアイ・キッズダンスフェスティバル|accessdate=2015-03-29|publisher=[[セブン&アイ・ホールディングス|セブン&アイ・ホールディングス]]|archive-url=https://web.archive.org/web/20150911061100/http://7andi-kids-dance-festival.jp/final/|archive-date=2015-09-11|dead-url=yes}}</ref>

== 人物 ==
* 從[[古坂大魔王|古坂大魔王]]那裡取了新的暱稱「シューティ」。
* 在以前上電視通告的時候，曾經忘記穿內褲。
* 會模仿Japanet Takata公司的高田社長。命題為「シュウタNET社長たかたん（秀太NET 社長高田）」，由成員[[宇野實彩子|宇野實彩子]]扮演助理「みさこ（實彩子）」。會經常在演唱會的特別環節中出現。值得一提的是高田社長是長崎縣出身的，Japanet Takata公司都是長崎縣佐世保市的企業。
* 喜歡[[寵物小精靈系列|寵物小精靈系列]]中的[[耿鬼|耿鬼]]。
* 很重視省水。
