== 花絮 ==
2004年，编剧张永琛和历史学家王庆祥、东亚经贸新闻报社双方对峙法庭，原因是王庆祥发表文章《叹！史学家心血愤！名编剧剽窃》在该报社报纸上“诽谤”张永琛的“名誉权”，天津市高级人民法院判决王庆祥、东亚经贸新闻报社给张永琛赔偿经济损失人民币66万元，精神损失人民币1万元。<ref>{{cite web |url=http://ent.163.com/ent/editor/tv/050515/050515_409238.html |title=《末代皇妃》名誉权案告终 张永琛获赔 |accessdate=2005 |publisher=网易 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://culture.people.com.cn/GB/22219/3390291.html |title= 《末代皇妃》名誉权侵权案告终|accessdate=2005 |publisher= 人民网|work= |language =zh-cn}}</ref><ref>{{cite web |url=http://news.sohu.com/20041014/n222488733.shtml |title=《末代皇妃》编剧诉名誉权受侵 |accessdate=2004 |publisher=搜狐网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://ent.enorth.com.cn/system/2004/07/10/000817987.shtml |title= 《末代皇妃》编剧张永琛郑重声明  |accessdate=2004 |publisher=北方网 |work= |language =zh-cn}}</ref>
