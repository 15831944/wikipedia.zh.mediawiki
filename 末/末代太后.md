《'''末代太后'''》，2013年[[冬季|冬季]]开机拍摄的[[史诗片|史诗片]]，为中美合拍片。<ref name=" fish" />

== 剧情 ==
《末代太后》是《[[末代皇帝|末代皇帝]]》的姊妹篇，讲述[[清朝|清朝]][[慈禧太后|慈禧太后]]沉浮跌宕的传奇一生，重点是“[[百日维新|百日维新]]”到“走向[[共和制|共和]]”。<ref name=" fish" >{{cite web |url=http://ent.ifeng.com/zz/detail_2012_06/27/15601108_0.shtml?_from_ralated |title=中美将合拍"末代太后" "末代皇帝"制片任顾问 |accessdate=2012年6月27日 |publisher=凤凰网 |work= |language =zh-cn}}</ref><ref name="cob " >{{cite web |url=http://ent.qq.com/a/20120627/000131.htm |title= 中美将合拍“末代太后”|accessdate=2012年6月27日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>
