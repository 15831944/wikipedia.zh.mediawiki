'''鬆芝'''（[[学名|学名]]：{{lang|la|'''''Tomophagus colossus'''''}}），又名'''棺菇'''、'''血靈芝'''，为[[靈芝科|靈芝科]][[鬆芝屬|鬆芝屬]]下的一个种。最早是由[[瑞典|瑞典]]的Fries Murr.於1851年所發表，通常在[[熱帶|熱帶]]、[[亞熱帶|亞熱帶]]有[[棺木|棺木]][[土葬|土葬]]習俗的地區就會發現，[[中國|中國]]民間盛傳，棺菇對[[癌症|癌症]]、[[尿毒癥|尿毒癥]]具有神奇療效，兼可驅邪，但專家表示，目前並無醫學根據證明。<ref>{{Cite web |url = http://hottopic.chinatimes.com/20150901003971-260809 |title = 神秘！棺菇到底是什麼？ - 中時電子報  |author = CT Jennifer |date = 2015年9月1日 |language = zh-hant }}</ref>
