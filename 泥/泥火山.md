'''泥火山'''是指多種在地下液體或氣體噴發後遺留下來的物質，一般以泥漿的形式出現，又或壘積成為一個圓錐形的泥尖頂。這些泥狀殘餘物，在時間的積累下，也可能會發展成為[[泥岩|泥岩]]地區；而若有關地帶仍然有地下[[天然氣|天然氣]]由裂隙向地表散逸，噴發物往往夾帶著[[水|水]]或其他沉積物。

依噴發物的含水量多寡所造成的泥火山，會有錐狀、盾狀、盆狀等不同形態。而由於泥火山的溫度比天然氣的燃點低，爆出物並沒有火和熱。[[印度尼西亚|印度尼西亚]]的一些盾形或坑形的泥火山爆发，能形成很大的气泡，气泡爆裂时泥水四溅，声音能传到很远。世界上最大的泥火山即位于印尼[[西多尔佐|西多尔佐]]的“[[西多尔佐泥石流|露西]]”，其直径可达{{Convert|10|km|mi|0}}<ref name="Volcano Live">{{cite news | title=Lusi Mud Volcano | work=Volcano Live| url=http://www.volcanolive.com/lusi.html | accessdate=2013-05-24 | date=March 26, 2009}}</ref>。

== 成因 ==
大多数泥火山的形成都与天然气有关，比如[[阿塞拜疆|阿塞拜疆]]，台湾[[高雄|高雄]]、新疆[[乌苏|乌苏]]等地。

而美国[[黄石国家公园|黄石国家公园]]、[[冰岛|冰岛]]和[[印尼|印尼]]的一些泥火山成因则与水蒸气有关。[[地热|地热]]把地下水加热变成水蒸气，水蒸气带着泥浆喷出地表，形成泥火山。

== 成分 ==
泥火山逸出的氣體，約20%是[[甲烷|甲烷]]，並混雜有少量[[二氧化碳|二氧化碳]]或[[氮|氮]]；不過在[[台灣|台灣]][[嘉義縣|嘉義縣]][[中崙村|中崙村]][[濁水潭|濁水潭]]的泥火山比較特別，其噴發氣體以二氧化碳為主。
而泥火山噴出的泥漿並不一定全是泥土和水的懸浮混合物，當中尚有不少溶在水裡的酸性氣體或化學鹽，甚或含有液態的[[碳氫化合物|碳氫化合物]]。

== 与火山的异同 ==
泥火山外形与[[火山|火山]]相似，内部结构也几乎相同。
泥火山山口下面有泥浆通道，通常只有几十米深，通道下面有泥浆房。

泥浆喷发的原理也跟火山相似，地下泥浆承受压力，从地表[[岩层|岩层]]的薄弱处破土而出，涌出地面。
小规模喷发时，只有气泡顶着泥浆缓缓涌出。剧烈喷发时，泥浆会犹如[[沸腾|沸腾]]一样，气泡翻滚、并喷洒出大量泥浆，如果出口比较小，也只會像水管一樣噴出水來。

尽管很多方面相似，但泥火山并非火山的一种，它是自成一体的特殊[[地貌景观|地貌景观]]。

== 分布 ==
除了[[南极洲|南极洲]]，全世界各[[大洲|大洲]]都有泥火山分布，但总体数量不多。

泥火山常發生於板塊隱沒與碰撞帶沿線，地層壓力過壓地區。

[[阿塞拜疆|阿塞拜疆]]的中部地区拥有大规模的泥火山群，[[婆羅洲|婆羅洲]]也有分布。

== 形成条件 ==
首先地下要有能形成泥浆的固体物质，通常是固结程度不太高的[[泥岩|泥岩]]、[[砂岩|砂岩]]、[[页岩|页岩]]，或者大量破碎的[[黏土矿物|黏土矿物]]。
再与[[地下水|地下水]]搅合在一起，形成泥浆。
有了泥浆，喷发还需要压力，这来自于埋藏地下的[[天然气|天然气]]、[[沼气|沼气]]等气体。
最后在天然气和泥浆层之上，还要有一层不太透水、透气的岩层。而这一岩层又要恰好有缝隙，气体才能带着泥浆喷流而出。

这些条件缺一不可，这也就是泥火山数量稀少的原因。

== 开发 ==
泥火山也可能是天然气苗喷出的产物，可做寻找[[油气田|油气田]]的地表徵兆。

火山会把地层深处的[[矿物|矿物]]带到地表，所以分布在火山地区的泥火山有时因为携带[[矿物质|矿物质]]成分不同，会呈现出缤纷的色彩。
比如美国[[黄石公园|黄石公园]]的泥火山，因为含[[铁|铁]]、[[硫|硫]]等物质，会呈现鲜艳的黄色、绿色。
而这些矿物质中，有些含有对人体皮肤有益的成分，所以火山地区的泥火山可以开发[[泥浴|泥浴]]。
