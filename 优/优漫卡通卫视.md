'''优漫卡通卫视'''是[[中國|中國]]的[[江苏省广播电视总台|江苏省广播电视总台]]下属的一个少儿卫星频道，是中国首家上星播出的省级少儿频道。开播于2010年7月1日，前身是成立于2004年10月1日的江苏少儿频道。

== 荣誉 ==
前身江苏电视台少儿频道在2006-2009年被[[中国国家广电总局|中国国家广电总局]]评为“[[全国优秀少儿频道|全国优秀少儿频道]]”一等奖，并在2009年5月被评为“[[全国青年文明号|全国青年文明号]]”先进集体。主打动画栏目《[[动画天地|动画天地]]》被[[中国国家广电总局|中国国家广电总局]]评为“[[全国优秀国产动画栏目|全国优秀国产动画栏目]]”一等奖。<ref>[http://www.ymws.com/introduce.html 简介] {{webarchive|url=https://web.archive.org/web/20140714032225/http://www.ymws.com/introduce.html |date=2014-07-14 }}</ref>

== 收视情况 ==
2011年，优漫卡通卫视收视表现优异。截至2011年10月，频道白天收视位列[[江苏省|江苏省]]全省第2，仅次于[[江苏卫视|江苏卫视]]，晚间黄金时段收视全[[江苏省|江苏省]]排名第6。

主打动画栏目《[[动画天地|动画天地]]》自开播以来，[[收视率|收视率]]一直稳居[[江苏省|江苏省]]同时段同类节目第一。

== 前身 ==
2004年10月1日-2010年7月1日间，该台的前身江苏少儿频道作为[[江苏省广播电视总台|江苏广播电视总台（集团）]]下属的电视频道，在[[江苏省|江苏省]]省内为[[江苏省|该省]]1700万[[未成年人|未成年人]]提供少儿节目，受到广泛好评。频道在4-14岁目标观众中收视率排名[[江苏省|全省]]第一，是[[江苏省|江苏省]]第一少儿电视[[媒体|媒体]]。
