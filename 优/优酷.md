'''优酷'''（{{Nasdaq|YOKU}}）是[[合一集团|合一集团]]旗下的一家大型[[影片分享網站|影片分享網站]]，2010年12月8日在[[美國|美國]][[紐約|紐約]]独立上市。網站最初成立與YouTube非常相似，做為視訊分享平台出現，但是此後逐漸轉型為網路傳媒業者，特別是持有大量影音版權等專門播放權利，並涉足電影、電視劇製作領域，在經營方向上与YouTube已經有很大的不同。现时为中国五大视频网（[[爱奇艺|爱奇艺]]、[[搜狐视频|搜狐视频]]、[[腾讯视频|腾讯视频]]、[[芒果TV|芒果TV]]）之一。

而其另一個傳統的業務——短片內容則和[[56網|56網]]、[[YouTube|YouTube]]有些重疊，但逐漸以“快者为王”为产品特色，与其他播客有所不同。除了正式傳媒內容播送的优之外，与瞄準崇尚自由创意酷融合，目标人群的不一定只有原创才能登台表演，无论业余或专业，无论个人或机构，优酷提供一切以微视频形式出现的视频收藏、自创与分享，也为传统媒体的发行和推广提供新的平台。其“快速播放，快速发布，快速搜索”的产品特性，优酷现已成为互联网拍客聚集的阵营之一。

优酷这个名字是创业团队投票产生的结果，意为希望网站能够帮助用户在网络上找到最酷的视频。<ref>{{Cite web|url=http://finance.huanqiu.com/roll/2015-08/7215172.html|title=优酷土豆改名合一集团 古永锵：视频行业的风向变了|accessdate=2018-05-27|author=|date=2015-08-07|work=财经_环球网|publisher=}}</ref>

== 歷史 ==
2005年11月，原[[搜狐|搜狐]]總裁和首席運營官[[古永鏘|古永鏘]]（Victor Koo）創立了优酷的开发者——合一網絡技術有限公司<ref name="建立时间">{{cite web|title=优酷大事记|url=http://www.youku.com/about/cn/youku_milestone/|publisher=优酷|accessdate=2011-07-18|language=zh-cn|quote=2005年11月，原搜狐总裁和首席运营官古永锵(Victor Koo)创立合一网络技术有限公司，管理300万美元的“搜索资金”，在中国搜索新一代互联网的创业机会。}}</ref>。同年6月21日，優酷發布公測版網站。次年12月21日，優酷正式开始運營。

2007年5月16日，優酷與[[盛大|盛大]]達成戰略合作協議，成為盛大獨家遊戲視頻支持網站<ref>{{cite web|url=http://www.cnetnews.com.cn/2007/0516/392611.shtml|title=盛大牵手视频网站 玩家可将游戏视频传至优酷|accessdate=2010-08-02|author=|date=2007-05-16|publisher=CNET科技资讯网|language=zh-cn}}</ref>。6月5日，優酷與中文搜索引擎網站[[百度|百度]]達成戰略合作<ref>{{cite web|url=http://it.sohu.com/20070611/n250500337.shtml|title=优酷网宣布与百度达成战略合作|accessdate=2010-08-02|author=|date=2007-06-11|publisher=搜狐IT|language=zh-cn}}</ref>。同年11月，優酷獲得2500萬[[美元|美元]]融資，这也是优酷的第三次融资<ref>{{cite web|url=http://it.sohu.com/20071121/n253379695.shtml|title=优酷网获2500万美元投资 主要支付运营成本|accessdate=2008-05-03|author=|date=2007-11-21 |publisher=搜狐IT|language=zh-cn}}</ref>。

2008年3月，優酷和交友网站[[MySpace|MySpace]]結为合作夥伴關係<ref>{{cite web|url=http://tech.sina.com.cn/i/2008-03-12/08222072040.shtml|title=优酷网与Myspace结亲|accessdate=2008-05-03|author=|date=2008-03-12|publisher=新浪網|language=zh-cn}}</ref>。到了同年5月，調查顯示優酷在中國視頻分享企業竞爭力排名中名列首位<ref>{{cite web|url=http://news.ccidnet.com/art/951/20080423/1429009_1.html|title="新規"出臺 視頻網站第一陣營格局確立|accessdate=2008-05-03|author=|date=2008-04-23|publisher=賽迪網|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20080427060306/http://news.ccidnet.com/art/951/20080423/1429009_1.html|archivedate=2008-04-27}}</ref>。7月9日，優酷同時獲得了[[國家廣電總局|國家廣電總局]]頒發的“信息網絡傳播視聽節目許可證”和[[北京廣電總局|北京廣電總局]]頒發的“廣播電視節目制作經營許可證”。

2009年，有传媒称优酷屏蔽了來自日本的IP，<ref>{{cite web|url=http://www.bianews.com/news/47/n-58847.html|title=土豆网、优酷网屏蔽日本IP 或为规避版权风险|accessdate=2010-07-13|author=孟鸿|date=2009-01-05|publisher=腾讯科技|language=zh-cn|archive-url=https://web.archive.org/web/20101208040714/http://www.bianews.com/news/47/n-58847.html#|archive-date=2010-12-08|dead-url=yes}}</ref>但优酷否认。

[[北京时间|北京时间]]2010年12月8日晚10点，优酷正式在[[纽约证券交易所|纽约证券交易所]]挂牌上市，股票代码YOKU，发行价为12.8美元，成为全球首家在美独立上市的[[视频网站|视频网站]]<ref>{{cite web|url=http://tech.hexun.com/2010-12-08/126068145.html|title=优酷纽交所IPO 成全球首家在美上市视频网站|accessdate=2010-12-09|date=2010-12-08|publisher=和讯科技|language=zh-cn}}</ref>。

2012年3月12日，优酷与[[土豆网|土豆]]宣布以换股方式合并<ref>{{cite web|title=优酷与土豆宣布以换股方式合并 土豆将退市|date=2012-03-12 |publisher=新浪科技|url=http://tech.sina.com.cn/i/2012-03-12/17036827996.shtml}}</ref>，新公司命名为“[[合一集团|合一集团]]”。

2015年10月中，[[阿里巴巴集团|阿里巴巴集团]]宣布向[[合一集团|优酷土豆]]（合一集团）董事会发出非约束性要约，全面收购优酷土豆集团，協議最終在2015年11月達成。<ref>{{Cite news|url=http://news.sina.com.cn/m/wl/2015-11-09/doc-ifxkniup6240820.shtml|title=阿里与优土达成最终协议：全现金收购优酷土豆|date=2015-11-09|publisher=钛媒体|accessdate=2016-11-21}}</ref>次年4月6日，[[合一集团|合一集团]]（优酷土豆）正式完成[[私有化|私有化]]，優酷成为阿里巴巴集团的一份子。<ref>{{Cite web|url=http://it.sohu.com/20160406/n443467533.shtml|title=优酷土豆正式加入阿里巴巴集团,看看双方高层都是怎么说的|date=2016-04-06 |accessdate=2016-11-21|work=搜狐科技}}</ref>

2016年12月21日，优酷正式使用新Logo。<ref>{{Cite web|url=http://finance.sina.com.cn/roll/2016-12-21/doc-ifxyxvcr7234522.shtml|title=优酷更换新Logo迎合年轻人|date=2016-12-21|accessdate=2018-05-28|work=新浪财经}}</ref>

== 爭議 ==

=== 模仿YouTube ===
优酷网因其网站名称及其Logo与[[YouTube|YouTube]]的极其相似，从而被网友批评：“只是改了名字换了颜色，掉了包后的[[YouTube|YouTube]]”。除外，其网页排版设计与[[YouTube|YouTube]]一模一样。<ref>{{Cite web |url=https://www.zhihu.com/question/22512053 |title = 为什么人人和优酷这样的网站跟Facebook和Youtube比起来感觉很廉价？ |publisher =[[知乎|知乎]] |date=2018-04-24|language = zh-cn}}</ref><ref>{{Cite web |url=https://www.zhihu.com/question/21458823/answer/97912246 |title = 相比优酷，你认为 YouTube 具有哪些优点？ |publisher = [[知乎|知乎]] |date=2016-05-09|language = zh-cn}}</ref>

优酷网在2016年更改了其网站Logo，但依旧与YouTube网站标志一模一样，甚至被曝与香港[[viu|viu]]标志相似。

=== 自我审查 ===
优酷网允许用户上传任意长度的视频。目前优酷已经与1500多个许可证持有者合作者进行合作，其中包括中国的电视台，分销商，电影和电视制作公司。

优酷网的视频受到[[中华人民共和国政府|中华人民共和国政府]]的网络审查，其网站包括不允许用户上传被中华人民共和国当局认为不适当或令人反感的视频。

=== 限制境外观看 ===
优酷同[[土豆网|土豆网]]、[[爱奇艺|爱奇艺]]等[[中國大陆|中國大陆]]影片分享網站一樣，對來自[[港澳台|港澳台]]與海外的[[IP地址|IP地址]]進行限制观看处理<ref>{{cite web |url=http://pay.youku.com/warning.html?pstype=101 |title=抱歉，会员服务区域限制。 |publisher=优酷 |access-date=2015-02-17}}</ref>，不允許用戶瀏覽带有版權影片，致使境外用戶不得使用大陸的代理或各種方法觀看影片，俗稱「翻牆進大陸」或「逆翻牆」<ref>{{cite web|url=http://blog.ifeng.com/article/15749994.html|title=为看大陆电视剧 港台人民学“翻墙”|date=2012-01-05|publisher=鳳凰博報|deadurl=yes|archiveurl=https://web.archive.org/web/20120109085706/http://blog.ifeng.com/article/15749994.html|archivedate=2012-01-09}}</ref><ref>{{cite web|url=https://www.google.com.hk/search?safe=off&q=翻牆看優酷&cad=h|title="翻牆看優酷"的相關Google搜尋結果|accessdate=2015-06-15}}</ref>。
2015年起，香港用戶可以觀看部分影片（在香港沒有購入版權），但澳門和台灣仍然被优酷封鎖IP。
