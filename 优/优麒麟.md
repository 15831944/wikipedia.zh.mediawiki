'''优麒麟'''（{{lang-en|'''Ubuntu Kylin'''}}）是[[Ubuntu|Ubuntu]]的官方衍生版本之一 <ref>[http://mobile.ctocio.com.cn/24/12947524.shtml。国产操作系统 Ubuntu Kylin 取名优麒麟] {{webarchive|url=https://web.archive.org/web/20140512214204/http://mobile.ctocio.com.cn/24/12947524.shtml |date=2014-05-12 }}</ref>。

== 背景 ==
早在2010年，Ubuntu 官方就多次要推出Ubuntu中文定制版，并且将Ubuntu的[[簡體中文|簡體中文]]名确定为友帮拓，但一直影响力较小。<ref name="relesae-cb">{{cite news|title=Ubuntu与麒麟合作发布UbuntuKylin|url=http://www.cnbeta.com/articles/227978.htm|accessdate=2017-02-06|agency=[[cnBeta|cnBeta]]|date=2013-02-28}}</ref>

2013年3月21日，[[工信部|工信部]]软件与集成电路促进中心和[[中国人民解放军国防科技大学|中国人民解放军国防科技大学]]与Ubuntu的支持公司[[Canonical公司|Canonical]]在[[北京|北京]]联合创立“CCN开源软件创新联合实验室”，Ubuntu Kylin正式创立，第一个版本随Ubuntu 13.04同时推出<ref>{{cite news|title=Canonical与中国合作开发Ubuntu Kylin|url=http://www.solidot.org/story?sid=33949|accessdate=2017-02-06|publisher=Solidot|date=2013-03-22}}</ref>。<ref>{{cite news|title=“CCN开源软件创新联合实验室”正式成立|url=http://news.xinhuanet.com/tech/2013-03/21/c_115112452.htm|accessdate=2017-02-06|agency=新华网|date=2013-03-21}}</ref>Ubuntu的创始人[[马克·沙特尔沃思|马克·沙特尔沃思]]称将其之为 Ubuntu 中文定制版的“接班人”，将提供一系列适合中国用户的软件应用。<ref name="relesae-cb"/>

=== KylinOS ===
{{see also|KylinOS}}
虽然Ubuntu Kylin与[[KylinOS|KylinOS]]的名字相近，但并没有直接关系<ref>[http://blog.ubuntusoft.com/about-ubuntukylin.html 关于 Ubuntu Kylin（和麒麟系统没有关系）] {{webarchive|url=https://archive.is/20130630122923/http://blog.ubuntusoft.com/about-ubuntukylin.html |date=2013-06-30 }}</ref>。
Ubuntu Kylin 本质上是 Ubuntu 的官方中国定制版本，尽管有政府的介入，但是它并不能取代军用操作系统。

== Ubuntu Kylin 13.04 ==
{{newsrelease|段落|time=2016-03-17T14:23:30+00:00}}
根据官方 WIKI 的信息<ref>[http://www.wenzhai365.com/2353.html Ubuntu Kylin 13.04操作系统正式推出] {{webarchive|url=https://web.archive.org/web/20130511063728/http://www.wenzhai365.com/2353.html |date=2013-05-11 }}</ref><ref>[https://wiki.ubuntu.com/UbuntuKylin/1304-ReleaseNotesChinese Ubuntu Kylin 13.04 發行說明]</ref>，Ubuntu Kylin 13.04 主要特性如下。
*採用版本3.8的Linux核心。
*採用Ubuntu Kylin主题样式。

=== 特色应用 ===
*中国音乐搜索（Unity Music Scope for China）：在线中文音乐搜索。
*中国农历软件（Chinese Calendar）：[[农历|农历]]，包含[[公历|公历]]。
*中国天气软件（China Weather Indicator）：提供来自中国气象局的天气预报信息，包含[[懸浮粒子|PM<sub>2.5</sub>]]數據。
*Ubuntu Kylin 軟件中心（Ubuntu Kylin Software Center）：Ubuntu Kylin特有的軟體商店，可以下載並安裝新的應用程式，或是管理已下載的應用。
*小企鹅输入法（Fcitx Input Method）：默认使用[[fcitx|fcitx]]输入法，支持[[雲端運算|云]]拼音输入。
*WPS办公套件Ubuntu Kylin版（WPS for Ubuntu Kylin）：提供中国本土办公软件[[WPS_Office|WPS Office]]，兼容多种文档格式。

== Ubuntu Kylin 14.04 LTS ==
{{newsrelease|段落|time=2016-03-17T14:23:30+00:00}}
2014年4月18日，新版 Ubuntu Kylin 随 Ubuntu 及其他官方衍生版一并发布，是 Ubuntu Kylin 的首个长期支持版。
*採用版本 3.13 的 Linux 核心。
*採用版本 7.2 的 Unity 桌面。（新版本提供了下一代 Ubuntu 用户界面 Unity 8 的预览选项。）

=== Ubuntu Kylin 应用 ===
*[http://pinyin.sogou.com/linux 搜狗输入法] (Sogou Pinyin for Linux)：搜狗再次與國內 Linux 團隊合作重新開發的拼音輸入法，支持智能糾錯、長詞聯想、模糊音及皮膚等中國用戶習慣的功能。
允许字词语句自动补全，通过与具有内置互联网搜索提示功能的搜狗输入法合作。
*[https://launchpad.net/youker-assistant 优客助手]：协助Windows用户换到Ubuntu操作系统的工具。
*智能搜索（Smart Scopes）：本地化智能搜索。图片搜索来源于搜狗在线；视频搜索来源于优酷视频；音乐搜索来源于百度。
*[https://launchpad.net/chinese-calendar 中国传统农历]（Chinese Calendar）：[[农历|农历]]，包含[[公历|公历]]。
*[[WPS_for_linux|金山WPS办公软件]]：中国较受欢迎的办公套件。
*[https://launchpad.net/lotus 莲花即时通信]：由国科大开发。可以用它和亲朋好友聊天，也可以在办公环境下与同事联络。
*金山快盘云存储：提供个人云存储空间，通过网络或是兼容设备读取文件和内容。
