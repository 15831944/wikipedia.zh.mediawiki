'''樂富站'''（{{lang-en|'''Lok Fu Station'''}}）是[[港鐵|港鐵]][[觀塘綫|觀塘綫]]的一個[[鐵路車站|鐵路車站]]，位於[[黃大仙區|黃大仙區]][[橫頭磡|橫頭磡]][[樂富邨|樂富邨]]，由四大聯營公司（Dragages, Hochtief, Gammon, Sentab) 負責興建，於1979年10月1日啟用。

==結構==
樂富站共有兩層，地面為出入口及大堂，而月台則位於地底。車站共有3條扶手電梯連接大堂及月台（其中一條在非繁忙時間暫停使用），並設有1部升降機及1條樓梯來往月台及大堂之間<ref name="sta_info">{{Cite book|author=港鐵公司|title=《樂富站車站指南》 |publisher=港鐵公司|date=2009年8月}}</ref>。

===大堂===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Lok Fu Station 2019 03 part5.jpg | caption1 = 大堂
}}
現時，樂富站內提供不同類型的商店供乘客購物或進食，例如有[[便利店|便利店]]、[[麵包糕餅店|麵包糕餅店]]、[[銀行|銀行]]等[[零售業|零售業]]，以及自助服務設施，包括[[自動櫃員機|自動櫃員機]]、[[自動售賣機|自動售賣機]]、自動照相機等<ref name="sta_info" /><ref name="shops"/>。大堂[[售票機|售票機]]旁亦設有[[香港郵政|香港郵政]][[郵箱|郵箱]]<ref>{{cite web|title=鐵路四處遊－郵箱服務|url=http://www.mtr.com.hk/chi/getting_around/post.html|work=港鐵網頁|publisher=港鐵公司|accessdate=2012年1月10日|language=zh-hk}}</ref><ref>{{cite web|title=郵箱位置表－黃大仙區 |url=http://app1.hongkongpost.hk/CGI/location/box_content.jsp?region=%27Kowloon%27&lang=zh_TW#wongtaisin |work=香港郵政網頁 |publisher=香港郵政 |accessdate=2014年5月6日 |language=zh-hk |deadurl=yes |archiveurl=https://web.archive.org/web/20140427153150/http://app1.hongkongpost.hk/CGI/location/box_content.jsp?region=%27Kowloon%27&lang=zh_TW |archivedate=2014年4月27日 }}</ref>，方便乘客在車站內投寄[[郵件|郵件]]。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Lok Fu Station 2019 03 part4.jpg | caption1 = 1號月台（往調景嶺方向）
| image2 = Lok Fu Station 2019 03 part3.jpg | caption2 = 2號月台（往黃埔方向）
}}
樂富站設有1個[[島式月台|島式月台]]，並設有[[月台幕門|月台幕門]]<ref name="sta_info" />。由於樂富站大堂及扶手電梯集中在月台的東面，因此月台西面沒有一般乘客可見的建構物，並且以牆壁分隔1號及2號月台間，只有兩條連接1號及2號月台的通道<ref name="sta_info" />，並與[[石硤尾站|石硤尾站]]結構相近。在樂富站剛通車時，當時成為[[修正早期系統|修正早期系統]]中，月台為最短的車站（4卡）<ref name="MIS open book">{{Cite book|language=zh-hant|author=地鐵公司|title=《地下鐵路首日通車紀念特刊》|publisher=地鐵公司|year=1979}}</ref>。

在剛通車時，樂富站是修正早期系統中唯一有三條並列的[[電動扶梯|扶手電梯]]的車站。至於升降機則由本屬職員使用的升降機改裝成客用升降機<ref>{{cite web|url=http://www.hkelev.com/usg_ktl.htm | title=觀塘線鐵路站升降機 |publisher=[[電梯資料網|電梯資料網]]}}{{zh-hant}}</ref>，並於2015年11月16日暫停使用，以更換為較新型號，並於2016年5月底重投服務<ref>{{cite web|url=http://www.mtr.com.hk/ch/customer/services/lift_replacement.html | title=升降機翻新/更換工程 |publisher=[[港鐵|港鐵]]}}{{zh-hant}}</ref>。

==鄰接車站==
樂富站是觀塘綫的中途站，乘客可在此乘坐往[[黃埔站_(香港)|黃埔]]或[[調景嶺站|調景嶺]]列車，往返[[九龍|九龍]]及[[新界|新界]]等地。

==利用狀況==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Lok Fu Station 2017 12 part1.jpg | caption1 = 此為本站唯一一組扶手電梯
| image2 = Lok Fu Station 2017 12 part2.jpg | caption2 = 此為本站唯一往來月台至大堂的樓梯
| image3 = Lok Fu Station 2019 03 part6.jpg | caption3 = 往來月台至大堂升降機
}}
樂富站啟用初期，由於當時[[橫頭磡邨|橫頭磡邨]]正在重建，故此它的使用量在通車初期一直偏低，直到1983年橫頭磡邨重建完成後客量才上升。

另外，在1998年之前，本站是最接近當時仍在運作的[[啟德機場|啟德機場]]的地鐵站，因此亦有不少乘客使用此站轉乘巴士及小巴往返。本站鄰近[[九龍城|九龍城]]，但須沿[[聯合道|聯合道]]步行15至20分鐘。现在的使用者多數也是附近居民、[[香港浸會大學|香港浸會大學]]學生及教職員，以及在那里上班的[[房屋署|房屋署]]職員。

===命名及選址===
在早期系統時，樂富站曾考慮命名為'''老虎岩站'''，後因樂富邨建成，最後決定命名為'''樂富站'''。樂富站原址是橫頭磡邨內的足球場，1976年被徵用作樂富地鐵站地盤。1979年10月1日隨着[[香港地鐵|前地鐵]]修正早期系統正式通車<ref name="MIS open book" />。

==軼事==
本站為1988年電影《南北媽打》的拍攝場地之一。
