'''樂進'''（{{bd||？|218年|catIdx=Yue Jin}}），[[表字|字]]'''文謙'''，[[兖州刺史部|兖州]]<ref>《後漢書·郡國志三》兗州東郡條：〖卫〗公国。本观故国，姚姓，光武更名。有河牧城。有竿城。<br>又《三國志·魏武帝紀》載建安十七年曹操“割河內之蕩陰、朝歌、林慮，'''東郡之衛國'''、頓丘、東武陽、發幹，钜鹿之癭陶、曲周、南和，廣平之任城，趙之襄國、邯鄲、易陽以益[[魏郡|魏郡]]。”</ref>[[陽平郡_(曹魏)|陽平郡]]衛國縣（今[[山東省|山東省]][[莘縣|莘縣]]）人，[[東漢|東漢]]末年[[曹操|曹操]]重要將領，[[五子良將|五子良將]]中最先跟隨曹操。在袁氏討伐的戰役中表現出色，每戰都率先到達戰場，經常立下戰功；逝世后，諡曰'''威侯'''。

== 生平 ==
=== 先登驍果 ===
初為曹操的帳下吏（記錄系），後被曹操遣回所屬郡（指[[東郡|東郡]]）募兵，募得千多人，歸還後升軍假司馬、陷陳都尉。

193年至194年間，[[呂布|呂布]]乘曹操東征[[徐州|徐州]]時，偷襲[[兗州|兗州]]，曹操立即還擊呂布。後樂進戰[[張超_(東平)|張超]]於雍丘（195年）、[[橋蕤|橋蕤]]於苦（197年）時，都有先登之功，被封'''廣昌亭侯'''。

後征[[張繡|張繡]]於安眾、圍呂布與其副將於[[下邳之戰|下邳]]（198年）。又擊[[眭固|眭固]]於[[射犬|射犬]]（199年）、攻[[劉備|劉備]]於[[小沛|小沛]]（200年），皆破之，被拜為討寇校尉。

建安五年（200年）[[官渡之戰|官渡之戰]]，曾與[[于禁|于禁]]渡河攻獲嘉<ref>本傳原文：“渡河攻获嘉”及《于禁傳》：复与'''乐进'''等将步骑五千，击绍别营，从延津西南缘河至汲、'''获嘉'''二县</ref>，後作為曹操的先鋒，奪勇力戰，斬殺[[袁紹|袁紹]]將領[[淳于瓊|淳于瓊]]（《曹瞒传》作先俘后杀）。

八年（203年），再隨軍進攻[[袁紹|袁紹]]之子[[袁譚|袁譚]]與[[袁尚|袁尚]]於[[黎陽|黎陽]]，斬殺敵方大將[[嚴敬|嚴敬]]，獲加行[[遊擊將軍|遊擊將軍]]。五月，與[[張遼|張遼]]攻取陰安，遷徙當地居民到河南<ref>《張遼傳》：【《魏武紀》：建安八年五月】太祖还许，使辽与'''乐进'''拔阴安，徙其民河南。</ref>。不久去消滅[[樂安郡|樂安郡]]的黃巾餘黨。

九年（204年），隨軍圍攻[[審配|審配]]於[[鄴|鄴]]，城破。次年（205年），攻袁譚於[[南皮|南皮]]，再次先登從東門入城。袁譚敗退[[雍奴|雍奴]]，樂進與[[張郃|張郃]]<ref>《張郃傳》：别将军围'''雍奴'''</ref>率軍成功攻克之。

十一年（206年），樂進與[[李典|李典]]<ref>《魏志十八·李典傳》：从围鄴，鄴定，与'''乐进'''围高幹於壶关</ref>率軍征[[高幹|高幹]]，由北道進逼[[上黨|上黨]]的[[壺關|壺關]]<ref>《魏武紀》：春正月，公征幹。幹聞之，乃留其別將守城，走入匈奴，求救於單于，單于不受。公圍壺關三月，拔之。</ref>，在[[曹仁|曹仁]]的計策兼帶領下成功攻克之。<ref>《曹仁傳》：河北既定，从围壶关。太祖令曰：“城拔，皆坑之。”连月不下。仁言於太祖曰：“围城必示之活门，所以开其生路也。今公告之必死，将人自为守。且城固而粮多，攻之则士卒伤，守之则引日久；今顿兵坚城之下，以攻必死之虏，非良计也。”太祖从之，城降。</ref>

同年八月<ref>《魏武紀》：秋八月，公東征海賊管承，至淳于，遣'''樂進'''、李典擊破之，承走入海島。</ref>，曹操征[[管承|管承]]，軍至[[淳于|淳于]]，派樂進、李典與張郃<ref>《三國志·張郃傳》：別征東萊，討管承</ref>同擊。管承敗逃海島，海濱平定。後曹操表[[漢獻帝|漢獻帝]]稱讚樂進、于禁和張遼三人，並升樂進為[[折衝將軍|折衝將軍]]。

=== 奮強突固 ===
{{see also|赤壁之戰||江陵之戰 (208年)||合肥之戰}}
建安十三年（208年）曹操準備南下荊州，樂進被遣屯軍於[[潁川郡|潁川郡]]的陽翟縣，同時張遼和于禁分別屯兵當地的長社縣和潁陰縣，但三人彼此不和，曹操於是命[[趙儼|趙儼]]參三軍軍事，事事都有訓喻，令到三人和睦。<ref>《趙儼傳》：時于禁屯潁陰，'''樂進屯陽翟'''，張遼屯長社，諸將任氣，多共不協；使儼并參三軍，每事訓喻，遂相親睦。</ref>

後因赤壁之敗曹操留[[曹仁|曹仁]]、樂进等防守[[荊州_(古代)|荊州]]<ref>《三國志·吳主傳》：曹公遂北還，留曹仁、徐晃於江陵，'''使樂進守襄陽'''。</ref>。[[關羽|關羽]]斷襄阳守将[[曹仁|曹仁]]後路，卻被樂進、[[文聘|文聘]]暫時擊退<ref>本傳原文：“留屯襄阳，击关羽、苏非等，皆走之”及《文聘傳》：與'''樂進'''討關羽於尋口。</ref>。南郡等蠻夷皆詣降服。进討[[劉備|劉備]]手下[[臨沮|臨沮]]長[[杜普|杜普]]、[[旌陽|旌陽]]長[[梁大|梁大]]，皆戰勝。後又在[[青泥|青泥]]與關羽對峙<ref>《先主傳》書信：“又'''樂進'''在青泥與關羽相拒”</ref>。

建安十八年（213年），樂進去從征[[孫權|孫權]]，獲[[假節|假節]]。<ref>本傳記載“後從征孫權，假進節。”根據下文“太祖還，留進與張遼、李典屯合肥”，樂進的假節時間為[[濡须之战_(213年)|第一次濡須口之戰]]時。</ref>

215年，與張遼、李典一起駐屯[[合肥|合肥]]，增食邑五百，併合前所封至一千二百戶。後因樂進多次立功，再多分五百戶，封一子列侯；進遷為右將軍。218年逝世，諡為'''威侯'''。

== 特徵 ==
樂進個頭嬌小，作戰勇猛，因膽烈而跟隨曹操，多次有先登之功。

== 子孫 ==
*[[樂綝|樂綝]]，樂進之子。果斷勇敢，有樂進之風，官至[[扬州_(古代)|揚州]]（与今日之扬州市无关）[[刺史|刺史]]。[[諸葛誕|諸葛誕]]叛亂時被殺，追贈[[衛尉|衛尉]]，諡為愍侯。
*[[乐肇_(曹魏)|樂肇]]，樂綝之子。

== 藝術形象 ==
===《[[三國演義|三國演義]]》===
在小說中，樂進是曹操招賢時最早投靠的武將，可惜在作品中整體表現並不出色，常與另一魏將[[李典|李典]]共同登場，大致上與史實無差。

*如同史實擅長衝鋒陷陣，在第五十三回與曹軍與孫權軍於陣前對壘時，樂進單騎突襲孫權，小說形容樂進「一騎馬，一口刀，從刺斜裏逕取孫權，如一道電光，飛至面前，手起刀落」，更一擊將孫權侍衛[[宋謙|宋謙]]、[[賈華|賈華]]的雙戟打斷。

*箭術不俗。
**第十二回，與[[呂布|呂布]]軍交戰時，射殺呂軍健將[[成廉|成廉]]。
**第三十三回，參與討袁戰事，又射殺[[袁譚|袁譚]]下屬[[郭圖|郭圖]]。
*單挑能力不差。
**在第十一回中，樂進曾與呂布部將[[臧霸|臧霸]]大戰三十回合，不分勝負。
**又第六十八回中，與吳將[[凌統|凌統]]大戰五十回合，也不分勝負。而後卻因中了吳將[[甘寧|甘寧]]的冷箭而受傷退場。此後，樂進再沒有登場於小說中。

== 評價 ==
[[陈寿|陈寿]]《[[三國志|三國志]]》評曰：「太祖建茲武功，而時之良將，五子為先。……[[張郃|張郃]]以巧變為稱，'''樂進以驍果顯名'''，而鑒其行事，未副所聞。或注記有遺漏，未如張遼、[[徐晃|徐晃]]之備詳也。」 

[[曹操|曹操]]表'''樂進'''、于禁、張遼：「武力既弘，計略周備，質忠性一，守執節義（此句指張遼），'''每臨戰攻，常為督率，奮強突固，無堅不陷'''，自援枹鼓，手不知倦。又遣別征，統御師旅，撫眾則和，奉令無犯（此句指于禁），當敵制決，靡有遺失。論功紀用，宜各顯寵。」
*樂進的稱讚語是“每臨戰攻，常為督率，奮強突固，無堅不陷”，指其衝鋒陷陣。

建安十七年（212年），[[刘备|刘备]]寫信給[[劉璋|劉璋]]，內容為：「曹公征吳，吳憂危急。孫氏與孤本為脣齒，'''又樂進在青泥與關羽相拒，今不往救羽，進必大克，轉侵州界，其憂有甚於[[張魯|(張)魯]]。'''魯自守之賊，不足慮也。」間接證明樂進的武勇也被劉備所忌憚。（《三国志·蜀志·先主传》）

[[王歆|王歆]]：「乐进魏之名将，志列于张辽后，其勇悍猛迅，当亦其侪也。惜史书不详，便松之不得益一字。遍观其传，如日记行述，不过某某日于某某处破某某敌而已。所得者，唯别传不见之名有数，如管承、刘备临沮长杜普、旌阳长梁大等。」
