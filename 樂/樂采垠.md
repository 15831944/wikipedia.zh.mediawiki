'''樂采垠'''（{{kor|k='''락채은'''|rm='''Rock Chae-Eun'''}}，{{bd|1989年|8月29日|catIdx=R樂}}），本名'''李采垠'''（{{kor|k='''이채은'''|rm='''Lee Chae-Eun'''}}），是一名[[韓國|韓國]][[模特兒|模特兒]]、[[主播|主播]]和[[化妝|美妝]][[部落格作者|部落客]]<ref>{{cite news |url=https://tw.news.appledaily.com/life/realtime/20171216/1260358 |title=高顏質內衣廣告女神　S曲線身材超火辣 |publisher=[[蘋果日報_(台灣)|蘋果日報]] |date=2017-12-16}}</ref>。因為喜歡女星[[朴韓星|朴韓星]]和[[Wonder_Girls|Wonder Girls]]的[[安昭熙|安昭熙]]而被暱稱為「1秒昭熙」<ref>{{cite web |url=http://sports.donga.com/3/02/20131024/58424068/2 |title=락채은 섹시 화보, 터질 듯한 볼륨감에 남심 ‘빠져들어’ |publisher=스포츠동아 |date=2013-10-24 |language=ko}}</ref>。現在是「Chuu Korea」、「WingsGirl」和「K-Star」的模特兒。

== 經歷 ==
畢業於[[同德女子大學|同德女子大學]][[鋼琴|鋼琴]]系。在2012年獲得「Gaya Farm」廣告模特兒選拔賽大獎，曾於2013年參加[[韓國小姐|韓國小姐]]選拔並入圍，之後她以[[模特兒|模特兒]]身分在[[韓國國際遊戲展示會|G-Star]][[初登場|初登場]]，[[廣播|播報]]工作也佔了她生活相當多的時間。因為2013年的[[MV|MV]]活動，她在「Galaxy United」（은하연합）的歌曲《大雨警報》中飾演女主角。2014年她演出[[San_E|San E]]的MV《Body Language》來強調性感。此後，便在HASH TAG STAR NEWS上以[[主題標籤|主題標籤]]進行[[韓流|韓流]]明星的相關報導。2017年5月，與南韓模特兒李浩嚴（이호연）結婚<ref>{{cite web |url=http://news.kbs.co.kr/news/view.do?ncd=3482656 |title=[K스타] ‘락채은♥이호연’의 영화 같은 결혼식 |publisher=[[韓國放送公社|KBS]] |date=2017-05-18 |language=ko}}</ref>。
