'''樂施會'''（'''Oxfam'''）是國際發展及救援的[[非政府組織|非政府組織]]，1942年由Theodore Richard Milford [[法政牧師|法政牧師]]（1896年—1987年）在[[英國|英國]][[牛津|牛津]]成立，原名'''英國牛津饑荒救治委員會'''（Oxford Committee for Famine Relief）。組成目的是在[[二戰|二戰]]運送食糧到被[[同盟國|同盟國]]封鎖的[[納粹德國|納粹德國]]佔領的[[希臘|希臘]]。1963年，[[加拿大|加拿大]]成立了第一家海外分會。1965年起改以[[電報|電報]]地址「'''OXFAM'''」作為名稱。

== 救援 ==
[[File:Oxfam_clothing_and_shoe_bank.jpg|thumb]]
基於樂施會的最初目標是提供食物給貧窮人士以解決[[饑荒|饑荒]]。多年來，樂施會都有開展種種策略對付饑荒的「原因」。除了食物和藥物之外，樂施會也向人們提供生活、工作用具令他們能夠自給自足，同時爭取開放自由的國際貿易，使貧窮地區生產的貨品以公平的價格出售。

現在樂施會除了在救濟上，還參與以下議題的工作：

* [[國際貿易|國際貿易]]（[[公平貿易|公平貿易]]） 
* [[教育|教育]]
* 债务和援助
* 安定生活及居所 
* [[健康|健康]]
* [[HIV|HIV]]／[[愛滋病|愛滋病]] 
* 社會公平
* 性别平等
* [[戰爭|戰爭]]和[[天災|天災]]
* [[民主|民主]]和[[人權|人權]]
* 气候变化

== 樂施商店 ==
[[File:OxfamCirencester.jpg|thumb]][[赛伦塞斯特|赛伦塞斯特]]的慈善商店]]
樂施會在1947年於英國開設首家[[慈善商店|慈善商店]]。現在已有八百家商店遍佈英國及其他國家。而其中超過七十間商店是專門售賣書籍的[[樂施書店|樂施書店]]，它也是全英國最大家的二手書店。

[[香港|香港]]有一家樂施商店，位於[[港島|港島]][[中環|中環]][[怡和大廈|怡和大廈]]。

== 籌款 ==
[[File:HK_Wan_Chai_Johnston_Road_OXFAM_a.jpg|thumb]]活動]]
樂施會的經費除了在樂施商店售賣二手貨物之外，亦在英國、澳洲、香港舉辦樂施[[毅行者|毅行者]]。香港會有售賣「樂施米」的活動。

2008年在香港的樂施之友突破了10萬，籌款額為1億9千萬港元。

===性侵===
前樂施會職員海倫埃文斯（Helen Evans）周一晚接受英國[[第四頻道|第四頻道]]訪問，指出曾收到投訴指稱樂施會援助人員強迫女性以性換取救助，並曾在一個月內收到3宗兒童性侵指控。她向樂施會舉報後，對方僅回應指管理層會進行策略性審視，但最終並未獲得任何處理。<ref>{{cite web|url=https://global.udn.com/global_vision/story/8662/2984339|title=用身體換援助、吃案志工被性侵？樂施會性醜聞延燒|publisher=UDN|date=2018-02-13|quote=伊凡斯表示，像是范哈威梅倫這種於海外性剝削被援助者的「不肖員工」，並不罕見。在她任內，就多次收到海外援助團隊，向在地提出「陪睡換物資」的不當投訴；女性工作者在海外被同僚性騷擾、甚至性侵的案例，在部分國家，受害比例更超過10%。}}</ref>
===高行政費===

[[壹週刊|壹週刊]]第951期及[[明報|明報]]2008年5月20日以[[香港社會服務聯會|香港社會服務聯會]]「惠施網」資料查詢，揭發樂施會06－07年行政費為19%，屬同類[[慈善團體|慈善團體]]行政費百分比最高的團體，而樂施會對此也作出澄清，報道之19%比率，為樂施會06/07年度籌款開支(11%)及行政費用(8%)的總開支。樂施會08/09年度的行政費佔全年總開支的2.6%。2017/18年，樂施會項目支出總額為2.22億港元，佔全年總開支82.3%。管理及行政開支佔3.3%，籌款及事務開支佔13.5%。<ref>https://www.oxfam.org.hk/tc/useofdonations.aspx</ref>依[[明施慎選|明施慎選]]資料，平均每籌$100 的成本約$21。<ref>https://theidonate.com/charities?rank=&demand=&sort=rank&q=%E6%A8%82%E6%96%BD%E6%9C%83</ref>

樂施會解釋香港樂施會不像部分機構般獲國際總部資助行政費、或獲政府撥款。由於所有經費來源均靠公眾捐款，因此須花更多人力物力募捐，例如為了應付每月收入，要聘用有償員工在街頭招募每月定額捐款者。

== 被中華人民共和國政府指责 ==
2010年2月11日，北京中央民族大學網站發出[[中華人民共和國教育部|中華人民共和國教育部]]《關於防止香港「樂施會」中國分部通過互聯網在我國高校招聘「大學生志願者」的緊急通知》，指稱香港樂施會為“竭力向我内地渗透的非政府组织，且其负责人（[[盧子健|盧子健]]）是反对派骨干”，要求各地教育部門和高校“提高警惕，認識到香港「樂施會」招聘我「大學生志願者」的用心不善”，“要斷絕與其任何來往，不與其有任何形式的合作”。<ref>{{cite web|url=http://www.rthk.org.hk/rthk/news/expressnews/20100223/news_20100223_55_649120.htm|title=教育部據報勒令大學生不得參與香港樂施會|publisher=香港電台|date=2010-02-23|accessdate=2010-02-23}}</ref><ref>{{cite web|url=http://dailynews.sina.com/bg/chn/chnpolitics/mingpao/20100222/22321215662.html|title=教育部關於樂施會通知全文|work=明報|publisher=新浪北美|date=2010-02-23|accessdate=2010-02-23}}</ref>

据报道这份通知也出现在若干其他大学的网站上，但在2月23日被删除。中華人民共和國教育部新闻组在接受查询时表示，他们正在跟有关部门了解情况，有消息会在网上公布。[[中华人民共和国外交部|中华人民共和国外交部]]发言人[[秦刚|秦刚]]在回答有关提问时表示，他不了解有关情况，但强调在华境外组织和团体应该遵守中華人民共和國法律。<ref name="BBC">{{cite news|url=http://www.bbc.co.uk/zhongwen/simp/china/2010/02/100224_oxfam_update.shtml|title=乐施会遭指责“倍感困惑”|work=BBC|publisher=BBC中文网|date=2010-02-24|accessdate=2010-02-25}}</ref>

乐施会总裁施日庄（John Sayer）对媒体表示，如果他们“涉足了不该涉足的领域”，希望中華人民共和國政府直接跟他们联系。乐施会中国部负责人[[廖洪涛|廖洪涛]]则表示，他们没有直接收到中華人民共和國教育部的有关文件，而是从网上获知这个通知的。他指出，乐施会在国内的活动一直是守法的，且它在内地的合作伙伴80%是政府机构。过去4年的40多名志愿者中30%后来都到非政府组织工作。乐施会北京办公室的负责人说，他们正在了解情况，不明白教育部为何发出有关的通知以及涉及哪些内容。<ref name="BBC"/>

香港时事评论员刘锐绍认为这次事件表明中華人民共和國官员对非政府组织缺乏了解，把香港乐施会在大陆高校招志愿者看作串联反政府势力，并把英国人施日庄任总裁理解成外国势力渗透。有香港维权人士指这件事令人担忧，因为在维权界看来乐施会在与大陆当局打交道时非常温和，甚至过分谨慎。<ref name="BBC"/>
