'''樂善堂顧超文中學'''（簡稱：顧中；{{lang-en|Lok Sin Tong Ku Chiu Man Secondary School}}）為[[香港|香港]]六大善團之一[[九龍樂善堂|九龍樂善堂]]所興辦的第2所全日制中學及第2所在九龍區以外興辦的慈善事業，位於[[新界|新界]]葵涌，是一所男女校。該校學生主要來自第一組别

該校創辦於1978年，由九龍樂善堂1950年度司理、港九押業商會主席、[[九龍城|九龍城]]街坊會理事長顧超文捐款興建，因以為名。其餘捐款善翁包括啓明建業主席，前[[東華三院|東華三院]]副主席關啓明、[[孔聖堂|孔聖堂]]主席許讓成，[[周生生珠寶金行|周生生珠寶金行]]司理周君任、[[九龍巴士|九龍巴士]]總司理[[雷瑞德|雷瑞德]]、[[同珍醬油|同珍集團]]總裁[[王仲銘|王仲銘]]及前[[市政局|市政局]]議員[[楊勵賢|楊勵賢]]等。創校時，因葵盛校址未能應用，曾借用同屬九龍樂善堂，位於[[葵涌|葵涌]][[荔景邨|荔景邨]]的劉世仁學校頂層上課（現址為[[香港考試及評核局|香港考試及評核局]]評核中心），直至翌年1979年3月才正式開幕。2015年1月20日，該校一名姓陳17歲中五女生因擔心遲到而追[[香港小巴|小巴]]，卻被撞至頭部重創昏迷。學校教職員得知此事後立刻為女生急救，再召救護車送院搶救。<ref>{{cite news|title=女生趕返學捱車撞重傷|url=https://hk.news.yahoo.com/%E5%A5%B3%E7%94%9F%E8%B6%95%E8%BF%94%E5%AD%B8%E6%8D%B1%E8%BB%8A%E6%92%9E%E9%87%8D%E5%82%B7-213006830.html|accessdate=2015-01-21|newspaper=《[[太陽報|太陽報]]》|date=2015-01-21}}</ref>

== 校訓 ==
爲九龍樂善堂屬校通用之「仁愛勤誠」。

該校校訓簡釋爲：
*仁愛成德，推己及人，大公無私，愛衆親仁。
**是爲「仁」和「愛」。
*學問之道，浩瀚無涯，努力不懈，方克有成。
**是爲「勤」。
*成己成物，立己立人，䖏世行事，誠之爲貴。
**是爲「誠」。

== 校歌 ==
同樣使用九龍樂善堂屬校通用，由曾創作[[先總統蔣公紀念歌|先總統蔣公紀念歌]]的著名作曲家[[黃友棣|黃友棣]]作曲，該校創立及命名人顧超文作詞的樂善堂校歌。(以普通話唱）

== 校長 ==
#[[馮治華|馮治華]]教授（1978-1994）：前任[[香港浸會大學|香港浸會大學]]教育學系系主任
#[[岑福求|岑福求]]（署理，1984-1985）：此期間原校長馮治華赴英深造，遂由副校長岑福求暫代。
#[[許為天|許為天]]（1994年9月-2004年12月）：第一學年為署理校長，現任香港浸會大學社會科學院教育學系首席講師、香港青年旅舍與及教育評議會執委，在離開顧中後曾任[[伊利沙伯中學舊生會中學|伊利沙伯中學舊生會中學]]及[[香港兆基創意書院|香港兆基創意書院]]校長。
#[[張林麗瓊|張林麗瓊]]（署理，1999-2000）：此期原校長許爲天參與教統局局長[[羅范椒芬|羅范椒芬]]倡儀之「校長與官員互調計劃」調進教統局決策組，原[[沙田區|沙田區]]高級教育主任張林麗瓊則調任署理校長，她在顧中任期屆滿後，轉赴[[皇仁書院|皇仁書院]]出任創校百年來首位女校長。
#[[黃鑫鈞|黃鑫鈞]]（署理，2004年12月-2005年11月）：曾任樂善堂顧超文中學副校長12年，已於2009學年榮休。

== 科目 ==
===中一級===
*主要科目：中國語文、英國語文、數學、歷史、中國歴史、生活與社會及綜合科學
*其它科目：音樂、普通話、電腦、體育、公民教育、設計與科技/家政、視覺藝術

===中二級===
*主要科目：中國語文、英國語文、數學、歷史與文化、生活與社會及綜合科學
*其它科目：音樂、普通話、電腦、體育、公民教育、設計與科技/家政、視覺藝術

===中三級===
*主要科目：中國語文、英國語文、數學、地理、企會財、中國歷史、歷史、經濟、生物、化學及物理
*其它科目：音樂、普通話、電腦、體育、設計與科技/家政，視覺藝術

===高中（中四至中六級）===
*必修科目：中國語文、英國語文、數學、通識教育
*選修科目：數學延伸部分、中國歷史、歷史、地理、經濟、企業，會計及財務概論、生物、化學、物理、電腦與資訊科技及視覺藝術
*其它科目：體育
