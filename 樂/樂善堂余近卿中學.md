'''樂善堂余近卿中學'''（簡稱：余中；{{lang-en|Lok Sin Tong Yu Kan Hing Secondary School}}），創辦於1969年，為九龍樂善堂第一間政府資助男女文法中學，位於九龍橫頭磡富裕街3號，教學語言以中文為主。由於得到余近卿先生的捐助，故學校命名為「樂善堂余近卿中學」。<ref>{{Cite web|url=http://www.ykh.edu.hk/|title=樂善堂余近卿中學|accessdate=2017-09-14|work=www.ykh.edu.hk}}</ref>

== 校訓與校歌 ==
樂善堂轄下的學校均採用統一的校訓。

;校訓
九龍樂善堂屬校校訓皆為「仁、愛、勤、誠」。
* 校訓簡釋為：仁愛成德，推己及人，大公無私，愛眾親仁。是為「仁」和「愛」。
* 學問之道，浩瀚無涯，努力不懈，方克有成。是為「勤」。
* 成己成物，立己立人，處世行事，誠之為貴。是為「誠」。

;校歌
九龍樂善堂屬校校歌均為同一通用，由曾創作蔣公紀念歌的著名作曲家[[黃友棣|黃友棣]]作曲，顧超文作詞。

歌詞：

樂善仁翁創我校，莘莘學子沐春風。<br>
校舍敞明環境美，讀書之樂在其中。<br>
家國事，責任重，年雖少，志氣雄。<br>
萬丈高樓平地起，求學做人一樣同。<br>
前進！前進！向前進！誓作民族前鋒。<br>

== 班級結構 ==
初中（中一至中三）：分為四班<ref>{{Cite web|url=http://www.chsc.hk/ssp2016/sch_detail5.php?lang_id=2&sch_id=186|title=SSP Profiles 2016/2017|accessdate=2017-09-14|work=www.chsc.hk|language=zh-HK}}</ref>

高中（中四至中六）：分為四班

== 開設科目 ==
;中一：
*中國語文、英國語文、數學、普通話、科技、視覺藝術、體育、綜合人文、科學、演藝及多媒體教育科、創客實驗課

;中二：
*中國語文、英國語文、數學、普通話、科技、視覺藝術、體育、中國歷史、科學、地理、歷史、經濟、音樂、創客實驗課

;中三：
*中國語文、英國語文、數學、普通話、科技、視覺藝術、體育、地理、歷史、經濟、物理、化學、生物、音樂

;中四至中六：
*中國語文、英國語文、數學、通識教育、中國歷史、地理、歷史、經濟、綜合科學、旅遊與款待、視覺藝術、資訊及通訊科技、企業、會計與財務概論、物理、化學、生物

;其他學習經歷：
*演藝及多媒體教育科、音樂及視覺藝術

== 近年 ==
教育局公布《推動STEM教育——發揮創意潛能》報告，支援香港中、小學推行STEM教育。樂善堂余近卿中學被教育局評為四間「專業發展學校」之一，向多間學校分享經驗。[https://news.mingpao.com/pns/dailynews/web_tc/article/20170102/s00001/1483293371585]
