'''氫氧化亞錫'''是一種'''不存在'''<ref>《无机化学丛书》.第三卷 碳 硅 锗分族. 科学出版社. P<sub>316~317</sub>. 2.2.2一氧化锡、氢氧化亚锡和亚锡酸盐</ref>的無機化合物，实际上是'''水合氧化亚锡'''，只是为了表达方便，[[化學式|化學式]]写作Sn(OH)<sub>2</sub>并成为氢氧化亚锡。氫氧化亞錫是錫的2價(氧化態)化合物。氫氧化亞錫的外觀是[[白色|白色]][[结晶|结晶]][[粉末|粉末]]，有時會呈或微[[黄色|黄色]][[结晶|结晶]][[粉末|粉末]]，[[熔點|熔點]]約為[[攝氏|攝氏]]120，難溶於水，微溶[[乙醇|乙醇]]、[[乙醚|乙醚]]，易溶於[[二氯|二氯]][[烴類|烴類]]、[[丙酮|丙酮]]等[[有機化合物|有機化合物]]、遇到[[高溫|高溫]]容易[[分解|分解]]或[[脱水|脱水]]变成[[黑色|黑色]]的[[一氧化锡|一氧化锡]]。

氫氧化亞錫置於[[空氣|空氣]]中常常會被氫氧化成[[氫氧化錫|氫氧化錫]]。

== 制備 ==
純氫氧化亞錫的製備方式有很多，例如(CH<sub>3</sub>)<sub>3</sub>SnOH 和 SnCl<sub>2</sub> 在非質子傳遞的溶劑中:<ref name = "Wiberg&Holleman">{{Holleman&Wiberg}}</ref>
:2Me<sub>3</sub>SnOH+ SnCl<sub>2</sub> → Sn(OH)<sub>2</sub> + 2Me<sub>3</sub>SnCl

但得到的物質曾有過爭議，因為[[沉澱|沉澱]]的[[錫|錫]][[鹽|鹽]]Sn(OH)<sub>2</sub>被認為與[[鹼金屬|鹼金屬]][[氫氧化物|氫氧化物]]（如[[NaOH|NaOH]]）有類似的結構，但分析後發現它是[[氧化亚錫|氧化亚錫]]的[[水合物|水合物]]，即5SnO.2H<sub>2</sub>O or  3SnO.H<sub>2</sub>O,<ref>Structure of Tin(II) "Hydroxide" and Lead(II) "Hydroxide", R.A. Howie & W. Moser, Nature 219, 372 - 373 (27 July 1968); {{doi|10.1038/219372a0}}</ref>，所以純的Sn(OH)<sub>2</sub>結構是不確定的。<ref name = "Wiberg&Holleman"/>

另一種是利用錫金屬當作電極電解水，接負極的電極會產生[[氫氣|氫氣]]，另一極的錫原子失去電子變成錫離子進入水中遇到[[氫氧根離子|氫氧根離子]]而沉澱，乾燥後得到氫氧化亞錫:

錫金屬當作電極電解水陰極的[[半反應|半反應]]為:
:Sn → Sn<sup>2+</sup> + 2e<sup>-</sup>
:Sn<sup>2+</sup> + 2H<sub>2</sub>O → Sn(OH)<sub>2</sub> + 2H<sup>+</sup>

陽極的[[半反應|半反應]]為:
:2H<sup>+</sup>  + 2e<sup>-</sup> → H<sub>2</sub>

全反應
:Sn + 2H<sub>2</sub>O → Sn(OH)<sub>2</sub> + H<sub>2</sub>

由於Sn(OH)<sub>2</sub>難溶於水，所以會沉澱。

== 用途 ==
氫氧化亞錫可以用來防治[[馬鈴薯|馬鈴薯]]晚疫病、[[洋蔥|洋蔥]]黑班病、等各種[[作物|作物]]的疾病。除殺[[菌|菌]]外，也可用於防治[[水田|水田]]中的[[藻類|藻類]]和水蝸牛比如說[[福壽螺|福壽螺]]，但由於錫會造成[[重金屬|重金屬]]汙染，現在這些多用其他[[有機物|有機物]]取代了。氫氧化亞錫還有增糖、增產的作用。
