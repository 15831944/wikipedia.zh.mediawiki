'''練馬師'''（{{lang-en|Horse trainer}}、{{lang-ja|調教師}}）是一項[[賽馬|賽馬]]相關的職業，職責包括挑選有潛質的[[競賽馬|競賽馬]]匹、照顧馬匹的健康醫療和日常生活飲食、訓練馬匹競賽技巧及催谷馬匹至最佳出戰狀態等。一般而言，練馬師會在一個[[賽馬會|賽馬會]]旗下的合約僱員，並在對方的授權下「[[馬棚|開倉]]」訓練馬匹。練馬師一般是由騎師、策騎員、訓練員、馬伕或馬房助理等出身的，一般是在馬圈擁有一定人脈才會開倉，與[[馬主|馬主]]建立良好關係亦是練馬師的職責所在。練馬師的工作非常忙碌，由於馬匹的狀況需要長期觀察，練馬師可能需要日以繼夜、年中無休地照顧馬匹的。<ref name="MyUser_Gycmokco.com_August_12_2016c">{{cite web |url=http://www.gycmokco.com.hk/article4chi.htm |title=George Y. C. Mok & Co. |newspaper=Gycmokco.com |date=  |author= |accessdate= August 12, 2016}}</ref>而練馬師為自己訓練的馬匹匹配騎師，在贏馬後會與騎師和馬主及其他僱員等對分獎金，因此若其馬房擁有不少實力馬，收入是相當可觀的。

==香港==
{{參見|香港冠軍練馬師}}
在香港，若要成為練馬師是需要由[[香港賽馬會|香港賽馬會]]審核的，獲得通過後方能開倉練馬。每年贏取頭馬最多的練馬師會獲得「[[香港冠軍練馬師|香港冠軍練馬師]]」獎項。而每年的成績亦有一定標準，截至2016/17馬季，練馬師需要在一季內贏得至少16場頭馬，若僅贏14或15場便需要贏得$1,850萬港元獎金方能符合表現準則<ref name="MyUser_M.sina.com_August_12_2016c">{{cite web |url=http://m.sina.com.hk/news/article/20160728/4/40/4/%E7%B7%B4%E9%A6%AC%E5%B8%AB%E6%96%B0%E5%AD%A3%E8%B4%8F16W%E6%89%8D%E5%8F%8A%E6%A0%BC-6102170.html |title=練馬師新季贏16W才及格 - 香港新浪 |newspaper=M.sina.com |date=  |author= |accessdate= August 12, 2016}}</ref>，否則將會被馬會發出警告信，累計三次便要被收回牌照，結束其香港練馬事業。此外，香港練馬師亦有年齡上限，馬會規定練馬師一旦達65歲便須退休，2013年修例指已達退休年齡的練馬師若表現「符合要求」（馬會要求練馬師每年練馬師頭馬榜和獎金榜均列前五），便可放寬續牌三或五年。香港著名練馬師「大摩」[[約翰摩亞|約翰摩亞]]便是首位超過65歲而獲得續牌的練馬師。<ref name="MyUser_Hk.racing.nextmedia.com_August_12_2016c">{{cite web |url=http://hk.racing.nextmedia.com/art_main.php?sec_id=3400631&art_id=16976814&showdate=20130628&tt_cat=15286386 |title=壹蘋果馬網 |newspaper=Hk.racing.nextmedia.com |date=  |author= |accessdate= August 12, 2016}}</ref>

而在2013/14馬季起，便由13場頭馬升至15場頭馬，2016/17馬季起再升至16場頭馬。

2015/16馬季，練馬師[[胡森|胡森]]和[[孫達志|孫達志]]因分別在3季未能達標而被逐出香港練馬界<ref name="MyUser_Hk.apple.nextmedia.com_August_12_2016">{{cite web |url=http://hk.apple.nextmedia.com/realtime/racing/20160714/55359614 |title=在港職業生涯最後抗爭 孫達志胡森表明會上訴 |newspaper=Hk.apple.nextmedia.com |date=  |author= |accessdate= August 12, 2016}}</ref><ref name="MyUser_Hk.apple.nextmedia.com_August_12_2016c">{{cite web |url=http://hk.apple.nextmedia.com/sports/art/20160727/19711157 |title=【再有練馬師出局】 孫達志上訴駁回！馬房數目創25年新低 |newspaper=Hk.apple.nextmedia.com |date=  |author= |accessdate= August 12, 2016}}</ref>。

此外，香港亦與世界各地賽馬界中有「馬房主帥制度」，即騎師不與所屬馬會簽約而是與合約練馬師簽約，騎師的座騎全由該練馬師安排，其薪金亦是由馬主及練馬師支付。2015年，香港練馬師[[文家良|文家良]]便與著名法籍騎師[[巫斯義|巫斯義]]簽約成為馬房主帥<ref name="MyUser_Hk.racing.nextmedia.com_August_12_2016">{{cite web |url=http://hk.racing.nextmedia.com/art_main.php?sec_id=3403908&showdate=20150702&art_id=17162866 |title=壹蘋果馬網 |newspaper=Hk.racing.nextmedia.com |date=  |author= |accessdate= August 12, 2016}}</ref>，惶後來因巫氏的策騎手法經常違規而獲馬會多次停賽，文氏遂決定不再續約，令巫氏在港策騎多年後終離開香港馬壇<ref name="MyUser_Hk.apple.nextmedia.com_August12_2016c">{{cite web |url=http://hk.apple.nextmedia.com/realtime/racing/20160806/55458423 |title=全港最茅騎師？一老一嫩洋將 犯規紀錄不相伯仲 |newspaper=Hk.apple.nextmedia.com |date=  |author= |accessdate= August 12, 2016}}</ref>。
