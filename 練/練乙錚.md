'''練乙錚'''（{{lang-en|Joseph Lian}}，{{bd|1951年|8月|catIdx=練}}），[[香港|香港]][[傳媒工作者|傳媒工作者]]，中學就讀香港[[九龍華仁書院|九龍華仁書院]]，[[美國|美國]][[卡爾頓學院|卡爾頓學院]]數學高級榮譽學士，[[明尼蘇達大學|明尼蘇達大學]][[明尼阿波利斯|明尼阿波利斯]]分校經濟學博士。1992年回港前任教[[加州大學河濱分校|加州大學河濱分校]]，其後曾任[[香港科技大學|香港科技大學]]商學院副院長，並為該校首屆最傑出教學獎得主。<ref>[http://www.hkbookcity.com/showbook2.php?serial_no=141799 練乙錚文集II] </ref>

他曾任香港科技大學[[經濟學|經濟學]]系講師，著有《水清有魚》。後來他接任[[沈鑒治|沈鑒治]]，擔任《[[信報財經新聞|信報財經新聞]]》[[總編輯|總編輯]]。

1998年起，他任[[中央政策組|中央政策組]]全職顧問，與[[劉兆佳|劉兆佳]]、[[曾德成|曾德成]]等共事。2003－2004年間，因參與爭取民主集會，2004年7月5日，突然被港府革除職務。後來於2005年6月中，在《信報》書寫了17篇文章，名為《浮桴記——謀府生涯六載事與思》，記錄了他當顧問期間觀察到的問題，又披露了大量不為人知的中策組內部情況，趣味盎然。

自離開[[香港政府|香港政府]]後，在外地遊歷三年，更在英國學習帆船遠航的知識和技術，2007年11月駕「既濟號」航返香港。2007年12月起回《信報》任主筆。[[2010年1月|2010年1月]]，離任《信報》主筆，暫停每日發表政論專欄，改任特約評論員，只會不定期發表文章。

練乙錚曾於於[[日本|日本]]鄉郊的秋田[[國際教養大學|國際教養大學]]任經濟學教授。<ref>第1164期[[壹周刊|壹周刊]]</ref>

== 事件 ==
=== 被梁振英以律師信警告 ===
2013年2月，練乙錚因其時評文章《誠信問題已非要害 梁氏涉黑實可雙規》，被[[梁振英|梁振英]]以
律師信警告[[誹謗|誹謗]]<ref>{{cite news |url = http://www1.hkej.com//dailynews/article/id/656935/誠信問題已非要害+梁氏涉黑實可雙規 |title = 誠信問題已非要害 梁氏涉黑實可雙規 |newspaper=信報 |date=2013-01-29 |accessdate=2016-08-06 }}</ref><ref>{{cite web |url = http://www.open.com.hk/content.php?id=1188 |title = 誠信問題已非要害 梁氏涉黑實可雙規 |website = 開放網 OPENWEB |date=2013-02-10 |accessdate=2016-08-06 }}</ref>。《信報》立即刊聲明回應， 總編輯[[陳景祥|陳景祥]]聲明致歉對象是讀者，而非梁振英本人<ref>{{cite news |url = https://forum.hkej.com/node/98023 |title=《信報》回應聲明 |newspaper=信報 |date=2013-02-07 |accessdate=2016-08-06 }}</ref>。

2015年3月19日, 練乙錚於《信報》發表專欄文章《梁齊昕的處境不就是香港人的一個縮影嗎?》<ref>{{cite news |url = http://forum.hkej.com/node/121419 |title= 梁齊昕的處境不就是香港人的一個縮影嗎? |publisher=信報 |date=2015-03-19 |accessdate=2016-08-06 }}</ref>。梁振英隨即分別向《信報》及練乙錚發律師信，要求收回有關評論<ref>{{cite news |url = https://thestandnews.com/politics/梁振英致函-信報-批練乙錚專欄借題發揮-要求停手/ |title = 梁振英致函《信報》 批練乙錚專欄借題發揮 要求停手 |publisher=立場新聞 |date=2015-03-20 |accessdate=2016-08-06 }}</ref>。練乙錚於23日專欄文章《道歉．道歉的理論．史學．史學的實踐》中就事件致歉<ref>{{cite news |url = https://forum.hkej.com/node/121508 |title = 道歉．道歉的理論．史學．史學的實踐 |publisher=信報 |date=2015-03-23 |accessdate=2016-08-06 }}</ref><ref>{{cite web |url = http://1in99percent.blogspot.hk/2015/03/blog-post_43.html |title = Just Getting By: 練乙錚: 道歉．道歉的理論．史學．史學的實踐 |website = Blogspot.hk |date=2015-03-22 |accessdate=2016-08-06}}</ref><ref>{{cite web |url = https://www.thestandnews.com/politics/回應梁振英-練乙錚-文章非政治攻訐-向無辜當事人致最深歉意/ |title=回應梁振英 練乙錚：文章非政治攻訐 向無辜當事人致最深歉意 |website=立場新聞 |date=2015-03-23 |accessdate=2016-08-06 }}</ref>。

===信報專欄被停止===
2016年7月29日，練乙錚收到信報總編輯的電郵，表示信報改版而要停止其專欄。[[香港記者協會|香港記者協會]]及[[獨立評論人協會|獨立評論人協會]]即日發表聯合聲明，譴責《信報》因政治氣候甚至政治目的或任務而自我審查，更指「如只懂緊緊跟隨官方口徑，報刊分分鐘要「改姓」，丟失幾十年的招牌和靈魂。」<ref>{{cite web|url=https://www.hkja.org.hk/site/portal/Site.aspx?id=A1-1488|title=評協記協對《信報》停練乙錚專欄表震驚 (Chinese Only)|publisher=香港記者協會|date=2016-07-29|accessdate=2016-08-06}}</ref><ref>{{cite web|url=https://www.facebook.com/icahongkong/posts/1117120995021998|title=評協、記協對《信報》停練乙錚專欄表震驚 對言論自由情況越感憂慮|publisher=獨立評論人協會|agency=facebook|date=2016-07-29|accessdate=2016-08-06}}</ref>31日，練乙錚以「別了《信報》」為題寫了封給讀者的公開信。<ref>{{cite news|url=https://thestandnews.com/politics/別了-信報/|title=別了《信報》|publisher=主場新聞|date=2016-07-29|accessdate=2016-08-06}}</ref>8月1日，一群《信報》的前任及現職員工在facebook開設專頁，並發表給《信報》總編輯郭艷明的公開信。公開信以「守言論自由 關注練乙錚專欄被撤走」為題，要求交代終止練乙錚專欄的真正原因，以及撤回終止練乙錚專欄的決定。<ref>{{cite web|url=https://www.facebook.com/bringbackjosephlian/posts/168716603539066|title=堅守言論自由 關注練乙錚專欄被停|agency=facebook|date=2016-08-01|accessdate=2016-08-06}}</ref>
