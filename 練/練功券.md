'''練功券'''，全稱'''銀行練功專用券'''，也叫'''出納點鈔專用紙'''、'''點鈔券'''，供[[中國|中國]]的[[銀行|銀行]]人員練習點鈔技能時代替實體[[鈔票|鈔票]]使用。

== 概述 ==
[[中華人民共和國|中華人民共和國]]成立時，已存在金融機關工作人員使用廢鈔練習點鈔的情況。<ref>{{cite web|url=http://news.xinhuanet.com/collection/2003-09/28/content_1104826.htm|title=王超華：收集中外銀行練功券1600種|first=|last=|work=[[新華網|新華網]]|year=2003 [last update]|accessdate=30 April 2012|deadurl=yes|archiveurl=https://web.archive.org/web/20050319163303/http://news.xinhuanet.com/collection/2003-09/28/content_1104826.htm|archivedate=2005年3月19日}}</ref>1989年起開始印製專用券。其紙質、紙幅與真鈔基本相同，多在[[上海|上海]]、[[石家莊|石家莊]]、[[西安|西安]]、[[成都|成都]]、[[保定|保定]]等印鈔廠印製，有些甚至具備[[水印|水印]]等防偽措施，亦經常隨新版紙幣發行而出新。<ref name="练功券的收藏">{{cite journal | title=练功券的收藏 | author=小卫 | journal=山西老年 | year=2011年 | volume=第11期}}</ref>一般分為三類：各大[[金融機構|金融機構]]所印的，有其明顯標識，通稱「通用券」；各金融機構的[[省|省]]（[[直轄市|市]]）一级分支分支所印，有分支機構標識的「地方券」；印刷簡單，僅標明「點錢專用紙」、「練功專用」等字樣的「行業券」。 

因其印刷精美、品種多樣（除[[人民幣|人民幣]]外亦有外幣版本）、存量有限又價格適中，成為一些愛好者的收藏品。但一些地方（如[[香港|香港]]）的法律不容許私下收藏此類仿真貨幣，或會因此惹官非。<ref>{{Cite web |url=https://hk.news.appledaily.com/local/daily/article/20180312/20328929 |title=【一周必看】內地大媽用港幣練功券買奶粉  |accessdate=2018-03-12 |date=2018-03-12 |publisher=[[蘋果日報|蘋果日報]]}}</ref>

== 非法用途 ==
據香港媒體報導，香港曾出現來自中國内地的诈骗集團使用練功券行騙。如今此等「練功券」在[[淘寶網|淘寶網]]即有售，不過其造工差，如港元練功券只有在受害人沒精打采和燈光不足的情況，才有可能被騙。<ref>{{Cite web |url=https://hk.news.appledaily.com/local/daily/article/20180308/20325941 |title=淘寶練功券抄足港幣 百張$100賣$6.8  |accessdate=2018-03-09 |date=2018-03-08 |publisher=[[蘋果日報|蘋果日報]]}}</ref>。
