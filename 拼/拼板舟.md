'''拼板舟'''（[[達悟語|達悟語]]：{{Lang|tao|tatala<ref>http://yamibow.cs.pu.edu.tw/index_TAO.htm 達悟語線上詞典</ref>}}），為世居[[蘭嶼|蘭嶼]]的[[達悟族|達悟族]]製造、使用的器具<ref>[http://210.240.134.48/citing_content.asp?id=2649&keyword=%AB%F7%AAO%A6%E0 臺灣原住民族歷史語言文化大辭典（網路版）拼板舟條目]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，多用於[[近海漁業|漁撈活動]]使用。

==由來傳說==
依據達悟族神話與傳說<ref>[http://c.ianthro.tw/406443 達西烏拉彎·畢馬] {{webarchive|url=https://web.archive.org/web/20140201222745/http://c.ianthro.tw/406443 |date=2014-02-01 }}. 2003. 晨星出版社, 臺中.</ref>一書所採集之達悟族[[口述歷史|口述歷史]]，遠古時代，達悟族的祖先由[[地底人|地底人]]習得製造拼板舟的更精進技術與應遵守的禁忌，始知流傳世代。

==造型設計==
達悟族的拼板舟通常會在頭尾兩端插上鏤刻雕飾，並加以彩繪之木質裝飾物（達悟語:ai no tatala)

==種類==
*Pikatangian 單人船
*Pikavangan 雙人船
*Pinonongnongan 三人船
*Cinetkeran 大船 
**Atlo so avat 六人大船(原意指三對槳)
**Apat so avat 八人大船(原意指四對槳)
**Alima so avat 十人大船(原意指五對槳)
*Avang 船，大船，軍艦
