'''翼龙系列无人机'''是[[中华人民共和国|中华人民共和国]][[成都飞机设计研究所|成都飞机设计研究所]]研制的一系列侦查打击一体化多用途[[无人航空载具|无人机]]。翼龙系列无人机主要负责执行侦查、监视和对地打击等任务，具有全自主水平起降和巡航飞行能力、空地协同能力、地面接力控制能力，可按需装载多型光电/电子侦察设备以及小型空地精确打击武器。截止至2016年底，翼龙系列无人机已出口5个国家。<ref>{{cite web |url=http://jobs.zhaopin.com/73607389690250000000.htm |title=中航工业成都空天飞行器股份有限公司介绍 |publisher=成都空天飞行器股份有限公司 |date=2016-11-23 |access-date=2016-12-03 |archive-url=https://web.archive.org/web/20161220031759/http://jobs.zhaopin.com/73607389690250000000.htm |archive-date=2016-12-20 |dead-url=yes }}</ref>

== 翼龙-1 ==
翼龙-1是一型中低空、长航时、侦查/打击一体化多用途无人机系统，也是翼龙系列无人机中的首个型号。翼龙-1由李屹东任总设计师。翼龙-1为军民两用无人机，其可执行监视、侦查、[[电子战|电子对抗]]及[[对地攻击|对地攻击]]等任务，在[[维稳|维稳]]、[[反恐|反恐]]和[[边境|边境]]巡逻等方面发挥用途；也可应用于灾情监视、[[大气|大气]]研究及[[气象|气象]]观测、[[地质|地质]]勘探及[[土地|土地]]测绘、[[环境保护|环境保护]]、[[农药|农药]]喷洒和[[森林|森林]][[防火|防火]]、[[毒品|缉毒]][[走私|走私]]等民用及科学研究等领域。<ref name="UAV">{{cite web|url=http://news.xinhuanet.com/mil/2011-08/02/c_121756116.htm|title=中国先进无人机扫描|publisher=新华军事|date=2011-08-02|accessdate=2013-05-10|deadurl=yes|archiveurl=https://web.archive.org/web/20111009064731/http://news.xinhuanet.com/mil/2011-08/02/c_121756116.htm|archivedate=2011-10-09}}</ref>

国际上同类无人机中，翼龙无人机处于先进水平。<ref name="UAV"/>其总体性能及用途与[[中国航天科技集团|中国航天科技集团]]研制的[[彩虹-4无人机|彩虹-4无人机]]、[[美国|美国]][[通用原子技术公司|通用原子技术公司]]研制的[[MQ-1捕食者无人攻击机|MQ-1捕食者无人攻击机]]等相似，但侧重点有所不同。<ref name="AVIC">{{cite web |url=http://mil.huanqiu.com/aerospace/2012-11/3276528.html |title=专访中航工业：有能力实现无人机空中加油 |publisher=环球网 |date=2012-11-14 |accessdate=2013-05-10}}</ref>

翼龙-1无人机于2007年10月完成首飞。<ref name="wingloong"/>2009年，翼龙-1首次在中国以外的国家进行了[[飞行表演|飞行表演]]。<ref>{{cite web|url=http://www.avic.com.cn/cn/qywh/hkym/381536.shtml|title=记黄云之一：身先士卒 亲力亲为|publisher=中国航空报|date=2012-09-26|accessdate=2013-05-10|deadurl=yes|archiveurl=https://web.archive.org/web/20150511004751/http://www.avic.com.cn/cn/qywh/hkym/381536.shtml|archivedate=2015-05-11}}</ref>翼龙-1成功完成8架次任务载荷各不相同的表演，提升了“翼龙”无人机的国际影响力和关注度，<ref>{{cite web|url=http://www.avic.com.cn/cn/qywh/hkym/381704.shtml|title=记黄云之二：尽职尽责 全力以赴|publisher=中国航空报|date=2012-10-08|accessdate=2013-05-10}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>并带来了外方用户交付合同的签订。<ref>{{cite web|url=http://www.avic.com.cn/cn/qywh/hkym/381681.shtml|title=记黄云之三：在工作中培养激情 在激情中愉快工作|publisher=中国航空报|date=2012-10-09|accessdate=2013-05-10}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

翼龙-1无人机分别于2008年、2010年和2011年以缩比模型的形式参加参加了第七、第八届[[珠海航展|中国国际航空航天博览会]]和[[巴黎航展|巴黎航展]]。<ref>{{cite web|url=http://news.xinhuanet.com/mil/2011-07/06/c_121629799_2.htm|title=巴黎航展：非洲国家空军官员对翼龙无人机感兴趣|publisher=中国航空报|date=2011-07-06|accessdate=2013-05-10|deadurl=yes|archiveurl=https://web.archive.org/web/20161115193959/http://news.xinhuanet.com/mil/2011-07/06/c_121629799_2.htm|archivedate=2016-11-15}}</ref>2011年，成都所产品部启动翼龙-1无人机批量生产工作。<ref>{{cite web |url=http://www.cannews.com.cn/epaper/zghkb/2016/11/15/A03/story/1153489.shtml |title=记中航工业成都所事业部“翼龙”无人机售后服务团队 |publisher=中国航空报 |date=2016-11-15}}</ref>2012年11月，一架翼龙-1无人机实机参加了第九届中国国际航空航天博览会。该机[[机身|机身]]绘有15颗[[导弹|导弹]]和20颗[[红星|红星]]，分别代表该无人机成功发射的[[空地导弹|空地导弹]]数目及完成任务次数。<ref name="AVIC"/>翼龙-1无人机已入列[[中国人民解放军空军|中国人民解放军空军]]，并被称为攻击-1型高空远程战役侦察攻击无人机。2015年9月3日，[[纪念中国人民抗日战争暨世界反法西斯战争胜利70周年大会|纪念中国人民抗日战争暨世界反法西斯战争胜利70周年大会]]上，已2架攻击-1型无人机参与阅兵。<ref>{{cite web |url=http://news.xinhuanet.com/mil/2015-09/06/c_128198099.htm |title=9.3阅兵无人机均已列装部队 为全天候无人侦察机  |publisher=新华网 |date=2015-09-06}}</ref>截止至2016年，翼龙-1无人机已在多国参加多次实战，累计飞行时长达数万小时，发射实弹上千枚。

在翼龙-1的基础上，成飞又研制了翼龙-1D无人机，该无人机采用了全复合材料的机身，并进行了发动机升级，预计于2017年首飞。

== 翼龙-2 ==
翼龙-2无人机是在翼龙-1基础上研制的一款中空、长航时、侦查/打击一体化多用途无人机系统，也是翼龙系列无人机中的第2个型号。相比于翼龙-1，翼龙-2进行了全面的机体扩大及气动布局优化，并换装推力更强的中国国产发动机，实用升限、速度和续航等性能指标都有提高，可适应更复杂的使用环境。翼龙-2拥有总共可携带约480千克武器弹药的6个外挂点，此外还具备载荷约200千克的内部弹舱，可携带CCD相机、通信侦察设备、电子战设备等任务载荷。翼龙-2出口合同曾获中国军用无人机行业中史上金额最大订单。

2016年11月，翼龙-2无人机在第十一届[[中国国际航空航天博览会|中国国际航空航天博览会]]首次公开并进行实机地面展示。<ref>{{cite web |url=http://mil.huanqiu.com/observation/2016-11/9622715.html |title=总师：翼龙2拥有内部弹仓 获中国史上最大订单 |publisher=环球网 |date=2016-11-02}}</ref>

2017年2月27日，“翼龙”Ⅱ在中国西北某高原机场成功首飞。担任此次首飞任务的是中国功勋试飞员雷强，他也曾担任歼10战机等重要机型首飞任务。<ref>{{cite web |url=http://news.ifeng.com/a/20170228/50739095_0.shtml |title=翼龙II无人机成功首飞 水平和美军“死神”相当 |publisher=中国航空报 |date=2017-02-28}}</ref>
