'''探月工程三期再入返回飞行试验器'''或称'''嫦娥五号T1飞行试验器'''（CE-5/T1）、'''嫦娥五号再入返回飞行试验器'''、'''嫦娥五号探路星'''、'''嫦娥五号先导星'''，绰号“'''舞娣'''”<ref name="别名">{{Cite web |url = http://www.cnsa.gov.cn/n1081/n7529/n308608/654075.html |title = “舞娣”素描——揭秘探月工程三期飞行试验器 |deadurl = yes |archiveurl = https://web.archive.org/web/20141027025626/http://www.cnsa.gov.cn/n1081/n7529/n308608/654075.html |archivedate = 2014-10-27 }}</ref>、“'''小飞'''”<ref>{{Cite web |url = http://news.163.com/14/1102/07/AA1FV8I200014AED.html |title = 来去自如 “小飞”回家 }}</ref>，是[[中国探月工程|中国探月工程]]第三阶段的[[月球探测器|月球探测器]][[嫦娥五号|嫦娥五号]]发射前用于技术工程试验的一颗探路星，主要承担探月工程三期的绕月高速返回地球技术的实践验证等任务<ref>{{Cite web |url = http://www.shenkong.net/News/1407/-EWHTLX-YZXJY10Y24RQHFS21090049.htm |title = 嫦娥五号探路星/验证星将于10月24日前后发射 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==概要==
该飞行试验器是由服务舱和返回器两个部分组合而成；其中服务舱以嫦娥二号平台为基础改造而成，返回器为[[神舟飞船|神舟飞船]]的缩小版<ref name="别名"/>。

[[北京时间|北京时间]]2014年10月24日02时00分04.829秒从[[西昌卫星发射中心|西昌卫星发射中心]]由[[长征三号丙运载火箭|长征三号丙改进二型运载火箭]]（遥12）发射升空。三日后绕月并返回地球。11月1日服务舱与返回器脱离。返回器于同日06时42分（[[北京时间|北京时间]]）在[[中华人民共和国|中国]][[内蒙古自治区|内蒙古]][[乌兰察布市|乌兰察布市]][[四子王旗|四子王旗]][[红格尔苏木|红格尔苏木]]活福滩乡境内顺利着陆<ref>{{Cite web |url = http://www.shenkong.net/News/1408/-EWHXDTLXCE5T1YDXC-10Y24RFS10084252.htm |title = 嫦娥五号先导探路星CE5T1运抵西昌 10月24日发射 |deadurl = yes |archiveurl = https://web.archive.org/web/20141015211822/http://www.shenkong.net/News/1408/-EWHXDTLXCE5T1YDXC-10Y24RFS10084252.htm |archivedate = 2014-10-15 }}</ref><ref>{{Cite web |url = http://news.qq.com/a/20140123/013434_all.htm |title=嫦娥五号试验星计划浮出水面}}</ref>。嫦娥五号T1使中国成为继苏联的[[探测器5号|探测器5号]]、美国的[[阿波罗8号|阿波罗8号]]之后，第三个成功实施从月球轨道重返地面的航天器的国家。

而服务舱则留在地月空间，开始第一阶段拓展试验任务。11月27日转至地月L<sub>2</sub>[[拉格朗日点|拉格朗日点]][[李萨如轨道|李萨如轨道]]运行<ref>{{Cite web|url=http://www.chinanews.com/mil/2014/11-29/6828156.shtml|title=中国探月再入返回飞行器服务舱飞抵地月L2点}}</ref>。2015年1月4日，服务舱离开L<sub>2</sub>点，开始第二阶段拓展试验任务。1月11日03时16分（北京时间）成功完成第一次近月制动后，进入到了高度为300×200公里的大椭圆环月轨道。12日进行了第二次近月制动。13日03时44分，成功完成第三次近月制动后，进入高度200公里倾角43.7°的圆型环月轨道<ref>{{Cite web|url=http://zz.81.cn/content/2015-01/13/content_6306377.htm|title=探月三期再入返回飞行器服务舱进入200公里环月圆轨道}}</ref>。轨道与未来嫦娥五号的月球轨道类似。2月8日进行了调相试验以模拟嫦娥五号轨道器远程导引<ref>{{Cite web|url=http://www.chinanews.com/mil/2015/02-08/7044654.shtml|title=中国探月工程三期再入返回飞行器在轨验证嫦娥五号}}</ref>。同年3月份上旬将再进行一次在月轨的虚拟交会对接远距离导引试验以模拟嫦娥五号上升器远程导引。4月份服务舱将会逐步降低轨道至100公里、至50公里、至15公里，并为嫦娥五号预定的着月采样区（[[风暴洋|风暴洋]]）进行高清拍照。<ref>{{Cite web|url=http://news.xinhuanet.com/tech/2014-12/02/c_1113493000.htm|title=“小飞”服务舱将拍摄嫦娥五号月球采样区}}</ref>

== 附屬搭载 == 
此次航天发射除了“嫦娥5号T1”探测器，还在长征3号甲火箭的上面级搭载两个国外太空探测民间组织研制的月球探测器：
# 德国{{tsl|en|OHB System}}制造，由卢森堡{{tsl|en|LuxSpace}}公司技术管理的'''4M任务''' (''Manfred Memorial Moon Mission'')，重14kg，有两个科学仪器。第一个仪器是无线电信标测试新的飞船定位方法。第二个仪器是西班牙{{tsl|en|iC-Málaga}}公司制造的辐射测量计。<ref>{{cite web |url = http://www.space.com/27422-china-moon-mission-launch-october.html |title = China Readies Moon Mission for Launch Next Week |publisher=Space.com |date=Oct 14, 2014}}</ref><ref>{{cite web |url = http://www.space.com/27503-china-moon-mission-launch-thursday.html |title=China Poised to Launch Next Moon Mission on Thursday |publisher=Space.com |date=Oct 22, 2014}}</ref>
#“{{le|哥本哈根亚轨道|Copenhagen Suborbitals}}”(Copenhagen Suborbitals)等民间组织共同发起的一个众筹探月计划的“口袋飞船”微型试验飞行器PS86X1。<ref>{{cite web|url=https://www.space.com/27422-china-moon-mission-launch-october.html|title=China Readies Moon Mission for Launch Next Week|first1=Leonard|last1=David|first2=Space com's Space Insider Columnist ||last2=October 14|first3=2014 07:00am|last3=ET|website=Space.com}}</ref>
