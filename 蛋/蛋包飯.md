'''蛋包飯（{{jpn|j=オムライス|rm=omu-raisu}}，Omurice）'''是由[[蛋|蛋]]皮包裹白饭而成的[[飯|飯]]料理。这道菜的原型是[[法国|法国]]的[[欧姆蛋|欧姆蛋]]；但是经过[[日本|日本]]人的改造，成为一种日式[[洋食|洋食]]。

==历史==
蛋包饭的原型是法国的欧姆蛋（Omelette），欧姆蛋的英文 omelette 是一个非常老的[[英文|英文]]单字。根据艾伦·戴维森的说法，英文的 omelette 来源於法国，是一个在[[16世纪|16世纪]]中期初被发明，随後在[[宫廷|宫廷]]内广泛使用的词汇。另外，还有诸如 alumelle 和 alumete 的这些拼法，收录於 Ménagier de Paris(II, 5)这本[[1393年|1393年]]所出版的书当中。

到了19世纪的在[[明治时代|明治时代]]，日本大量接受了西方文化，因此也接受了法国文化当中的美食。在日本，蛋包饭被改造：放入蕃茄酱炒饭改造成日式蛋包饭，成为日本国内无人不晓的一到日本洋食（因为是Omelet+rice（米饭），所以被称为オムライス（Omuraisu））。

韩国的蛋包饭承袭自日本，发展出韩式蛋包饭，把番茄炒饭替换成像[[扬州炒饭|扬州炒饭]]一样的多种配料炒饭，并且把上面的[[番茄酱|番茄酱]]替换成[[韩式辣酱|韩式辣酱]]。

同样是在19世纪，印度因为英国殖民而接受欧美文化的关系，饮食也受到了蛋包饭的影响，并且把蛋包饭与印度国民美食——咖喱结合，发明出了广为流传的咖喱鸡排蛋包饭，一次性把[[炸鸡排|炸鸡排]]、[[咖喱|咖喱]]、蛋包饭三种美食合而为一。

==流行文化==
因為電影《{{link-ja|蒲公英 (電影)|タンポポ (映画)|蒲公英}}》而有名的。在炒好的飯上放一個鬆軟的蛋包，然後在上桌後將其劃開，讓半熟的蛋包將炒飯蓋住。其他變化包括不用炒飯而用白飯，使用[[咖哩|咖哩]]、{{link-en|demi-glace|demi-glace}}或[[牛腩汁|牛腩汁]]等。

在中國大陸，受到韓劇《[[屋塔房的王世子|屋塔房的王世子]]》裡面的韩式蛋包飯的影響，在中國大陸除了日式蛋包飯以外，韓式蛋包飯也非常有人氣，但是在其他華人地區，很多人幾乎沒有聽過韓式蛋包飯這一說法；而且，中國大陸很多掛名“日式蛋包飯”的蛋包飯，其實都是韓式蛋包飯，使用的不是法式歐姆蛋、而是類似於中式蛋餅的蛋皮，裡面的白飯也被替換成了炒飯。

另外，由於[[百度百科|百度百科]]的錯誤影響，導致甚至還有“吃蛋包飯是友情的象征”這樣的誤傳；或者，在中國大陸的宅男界中，蛋包飯是女仆咖啡厅的必備餐點；但是在日本這兩點完全不成立，甚至很多日本人一辈子均没听过这样的说法。最後，在中國大陸，因為缺乏對日本料理中“和食”和“洋食”的認知，所以經常有人認為日本才是蛋包飯的發源地、而且百度百科也这样以讹传讹，而不知道蛋包飯的起源其實是[[法國|法國]]。

==种类==
#'''蘑菇奶油醬蛋包飯'''——採用蘑菇白醬，再撒上羅勒葉，非常高雅的一品。
#'''傳統日式蛋包飯'''——傳統的蛋包飯作法首先將打勻的蛋汁倒入平底鍋，煎成半熟的蛋皮時，再將飯（一般是[[番茄醬|番茄醬]][[雞肉|雞肉]]炒飯）倒入，並將飯給包起來。最後盛盤後，有不少店還會再淋上番茄醬。
#'''韓式蛋包飯'''——飯採用炒飯，蛋皮非常薄。採用韓式辣醬調味，比一般的蛋包飯稍微辣一點，香甜可口。
#'''咖喱雞排蛋包飯'''——在日本料理店、咖喱店、西餐廳、牛排館都可以見到的超級常見的料理，咖喱醬取代了原本蕃茄醬的位置。並且在蛋包飯上蓋上豬排或者雞排，增添風味。
#'''麻婆豆腐蛋包飯'''——用麻婆豆腐代替蕃茄醬，用中式炒飯代替白飯。
#'''乾燒明蝦蛋包飯'''——用乾燒明蝦代替蕃茄醬的蛋包飯，可謂是西式和中式的結合。

==傳統蛋包飯簡易食譜==
材料：雞蛋2粒、胡蘿蔔半根、番茄2粒、蘑菇6朵、白飯一碗半、木耳少許、番茄醬少許、鹽油適量 

製作步驟：
# 將所有食材切丁。蘑菇和木耳洗淨汆燙切丁。

炒飯：
# 熱鍋冷油，放入番茄不停翻炒直到水被榨出後盛出。
# 熱鍋冷油，将胡蘿蔔丁倒入炒几下，加入蘑菇丁和碎木耳，翻炒一会儿后倒入炒好的番茄，再倒入米饭並炒至均匀。
# 调至小火並放入适量盐，翻炒均匀後盛起。

製作蛋皮：
# 蛋打散並放一点盐在蛋液中。一樣熱鍋冷油，将蛋液倒入平底锅中，將蛋液均匀铺开。
# 当表皮呈波浪狀時，将饭倒在半块蛋皮上，最后装盘。可依據各自喜好可以搭配番茄醬風味更好。
