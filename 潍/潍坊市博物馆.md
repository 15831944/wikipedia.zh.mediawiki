'''潍坊市博物馆'''，位于[[山东省|山东省]][[潍坊市|潍坊市]][[潍坊高新技术产业开发区|高新技术产业开发区]]东风东街6616号，是潍坊市属综合性博物馆。

==简介==
1962年，山东省潍坊市人民政府委员会批准成立潍坊市博物馆<ref name=gaikuang>{{Cite web|url =http://www.wfsbwg.com/about/?48.html|title =潍博概况|author =|date =|publisher =潍坊市博物馆|language =|accessdate =2017-07-06|quote =}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1978年，潍坊市博物馆从原馆址[[潍县城隍庙|潍县城隍庙]]迁入[[十笏园|十笏园]]<ref>{{Cite web |url = http://news.cntv.cn/20101217/110586.shtml |title = 沧桑百年十笏园 |author =  |date =2010-12-17  |publisher =中国网络电视台  |language =  |accessdate =  |quote =  }}</ref>。1999年，潍坊市博物馆新馆建成，占地面积24053.5平方米，建筑面积18669.7平方米。2007年11月，潍坊市博物馆单位类别划为公益一类[[事业单位|事业单位]]。2016年进行了科室调整<ref name=gaikuang/>。2017年，潍坊市博物馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref>。

潍坊市博物馆设有潍坊简史、民俗及民间艺术、馆藏字画、石刻艺术长廊等固定陈列。其中“潍坊简史陈列”曾获[[国家文物局|国家文物局]]“十大陈列展览精品提名奖”。临时展区曾先后举办《静谧悠远——[[黄宾虹|黄宾虹]]艺术特展》、《[[陈介祺|陈介祺]]诞辰二百周年——金石千秋展》、《艺术的原乡——[[刘国松|刘国松]]、[[莫言|莫言]]书画展》、《[[丰子恺|丰子恺]]艺术展》、《[[吴昌硕|吴昌硕]]艺术展》、《齐鲁瑰宝展》、《金石相契——金石碑帖特展》、《第十届中国摄影金像奖作品展》等展览<ref name=gaikuang/>。

潍坊市博物馆馆藏文物将近8万件，分为化石、陶器、瓷器、青铜器、玉石、古角牙木、钱币、碑拓、书画、古籍、碑刻、织绣、造像、玺印符牌、民俗服饰及饰品、近现代革命文物等33类。其中有20万年前原齿象属新种——潍坊象的化石，[[大汶口文化|大汶口文化]]薄胎高柄杯，中国现存体量最大、年代最早的[[唐代|唐代]]铁佛，[[唐玄宗|唐玄宗]]唯一的《纪泰山铭》朱砂拓，[[郑板桥|郑板桥]]的《峭壁兰图》等珍贵藏品<ref name=gaikuang/>。
