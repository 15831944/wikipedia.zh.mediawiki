'''爪哇笛'''（{{lang-th|ปี่}}，拉丁化：Pi）是[[爪哇|爪哇]]式的双簧管樂器，源于[[印度|印度]]，后传到爪哇，是[[泰拳|泰拳]]的伴奏乐器之一，现为[[泰国|泰国]]主要的传统乐器之一。从[[素可泰王国|素可泰]]时代起，爪哇笛就已常用。在泰拳开场的传统战乐中，爪哇笛与铜[[钹|钹]]、高音[[鼓|鼓]]、低音鼓一同奏响。

爪哇笛由管身（เลาปี่）和管口（ลำโพง）两部分组成，但有的爪哇笛没有特制的管口。爪哇笛与印度乐器[[:en:Shehnai|Shehnai]]外形类似。

==种类==
* Pi cha nai（{{lang|th|ปี่ไฉน}}）：是由两部分组成的爪哇笛（ปี่สองท่อน），为[[木|木]]或[[象牙|象牙]]製，源于[[印度|印度]]，从[[素可泰王国|素可泰]]时代起爪哇笛就已常用。目前，Pi cha nai常与Pi chawa在[[阅兵式|阅兵式]]音乐中配合使用。
* Pi chawa（{{lang|th|ปี่ชวา}}）：是由两部分组成的爪哇笛，外形与Pi cha nai相似但比之长，为木制或象牙制。Pi chawa与[[:en:Glong_khaek|Glong khaek]]是同一时期被引入泰国的。一些记载表明，这种乐器曾用于古代Ayudthaya的[[军队|军队]][[进行曲|进行曲]]（กระบวนพยุหยาตรา）中。这种爪哇笛主要用来伴奏[[泰拳|泰拳]]。
* Pi klang（{{lang|th|ปี่กลาง}}）：常用于电影[[配乐|配乐]]，长37 cm，宽4 cm。
* Pi Mon（{{lang|th|ปี่มอญ}}）：是由两部分组成的爪哇笛，外形与Pi chawa相似但比之长，体积更大。管身为木制，管口为金属制。Pi mon一般在[[:en:Piphat_mon|Piphat mon]]乐团或者是旧时的Piphat raman乐团中演奏。<ref>{{th}}[https://web.archive.org/web/20070116122955/http://www.student.chula.ac.th/~48467689/Thaipee.html ปี่]</ref><ref>{{th}}{{cite web |url=http://www.rayongzone.com/soontornpoo/pee.php3 |title=存档副本 |accessdate=2008-09-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20080926153835/http://www.rayongzone.com/soontornpoo/pee.php3 |archivedate=2008-09-26 }}</ref><ref>{{th}}{{cite web |url=http://funkysonglyrics.com/thailand/th/audio-sound/audio-sound-3.html |title=存档副本 |accessdate=2008-04-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20070927203941/http://funkysonglyrics.com/thailand/th/audio-sound/audio-sound-3.html |archivedate=2007-09-27 }}</ref><ref>{{th}}[http://www.siammelody.com/thaimusic/pao.htm#pao5]</ref>
* Pi nai（{{lang|th|ปี่ใน}}）：常见於{{link-en|泰国文学|Thai literature}}中，如诗人和小说家Sunthon Phu的《[[人鱼传说|人鱼传说]]》（Pra Aphaimani）。长41－42 cm，宽4.5 cm。
* Pi nok（{{lang|th|ปี่นอก}}）：体积比Pi klang和Pi nai中小，古时起就常被演奏。长31 cm，宽3.5 cm。

由[[泰北|泰北]][[蘭納|蘭納]]人最初使用的竹制自由簧片乐器Pi joom（{{lang|th|ปี่จุม}}），是与爪哇笛完全不同的一种乐器。
