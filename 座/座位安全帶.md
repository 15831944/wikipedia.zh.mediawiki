'''安全帶'''是一種交通工具上的主動式安全裝備，還會與[[安全氣囊|安全氣囊]]連動成為車輛的被動安全系統。

== 歷史 ==
* 1885年，Edward J. Claghorn在[[紐約|紐約]]申請了[[美國|美國]]首個安全帶的專利（編號312 085）
* 1958年，[[瑞典|瑞典]][[富豪汽車|富豪汽車]]工程師Nils Bohlin發明了三點式安全帶，首次安裝在量产车輛──[[富豪汽車|沃尔沃]]亚马逊上。<ref>{{cite news|title=偉大發明的傳世，三點式安全帶50歲|url=http://news.u-car.com.tw/11073.html|publisher=U-CAR|date=2009-08-24|accessdate=2014-11-03}}</ref>

== 種類 ==
* 腿帶：常見於[[飛機|飛機]]和巴士。
* 肩帶
* 二點式：即腿帶。
* 三點式：腿帶、肩帶兼用，在[[汽車|汽車]]上最常見的款式，能防止乘客向前撞和左右移動位置。
* 四點式：主要用於[[賽車|賽車]]，兩肩各有一帶穿過，中間（約肚臍的位置）用一組扣環將兩帶結合。
* 五點式：用於[[賽車|賽車]]、[[兒童安全座椅|兒童安全座椅]]，一條帶穿過雙腿；兩肩各有一帶穿過。
* 六點式：兒童安全座位與賽車，肩膀兩條、側腹兩條、胯下兩條。
兒童可使用加高座椅以配合安全帶的高度。

== 汽車三點式安全帶之工作原理 ==
三點式安全帶是[[汽車|汽車]]上非常重要的被动式安全裝置，其作用原理是偵測織帶被拉出來的[[加速度|加速度]]以鎖定織帶。假如織帶被拉出來的[[加速度|加速度]]過高，轉軸的轉速太快使得加速度偵測器感受到的[[離心力|離心力]]大於[[彈簧|彈簧]]對卡榫的[[形變|拉力]]，此時卡榫被甩出，使得轉軸被卡死而織帶無法繼續被抽出。反之，只要將織帶放鬆，卡榫會被彈簧拉回，轉軸也會被釋放，安全帶便放鬆了。

現今的汽車三點式安全帶設定鎖定的條件大致為：
# 織帶被拉出來的[[加速度|加速度]]大於0.8G。
# [[汽車|汽車]]側傾超過15度（欧洲标准为12度。
# 汽车减速度达到0.45G（美国标准为0.3g)时

大多數小型車的三點式安全帶一端安裝在車的B柱（前座）或C柱（後座），早期的高度為固定無法調整，造成身高較矮的乘客繫安全帶時的困擾，現今越來越多車款改用高低可調式安全帶。另外少部份車款曾配備自動上肩式安全帶，安全帶會沿著A柱內側邊的[[滑軌|滑軌]]在關門後自動滑到B柱並固定。

== 安全效果 ==
非常早期的車輛沒有安全帶，由於車體安全結構的概念還沒形成，人們認為甩飛出去比在壓扁的鐵罐車體內好，但後來許多案例後發現，車禍者並非被拋躑車外而碰撞死亡，反而是因為軀幹卡在車門，或是距離車輛殘骸過近，被自己駕駛的車所壓過，於是改變了安全觀念，認為固定在車內才是有效的生存要件，透過潰縮結構和強固結構組合，來保障車室空間並吸收外部衝擊，另外，安全帶絕對只能一人使用，不可多人共用。現代的車輛雖然能減少碰撞的結構變形，但若不繫安全帶，汽車撞擊時可能使乘員：
* 撞上車中的硬物，即使是安全氣囊，基於[[慣性原理|慣性原理]]，瞬間碰撞的撞擊力甚大。
* 甩飛出車外。按照不同的碰撞狀態，對乘員死亡率的影響不能確定。
* 根據[[東京大學|東京大學]]的研究，若後座乘客繫安全帶能降低前座者的八成死亡率，因為汽車撞擊時，沒繫安全帶的後座乘客會撞向前座背後。<ref>{{cite news|title=立委連署 小客車後座 強制繫安全帶|url=http://news.ltn.com.tw/news/life/paper/38015|publisher=[[自由時報|自由時報]]|date=2005-10-11|accessdate=2014-11-07}}</ref>
* 根據[[日本|日本]][[運輸省|運輸省]]的統計，沒使用加高座椅的兒童的死亡率是有使用者的八倍。<ref>{{Cite web|url=http://www.car-safety.org.tw/DesktopDefault.aspx?tabid=121&ItemId=108|title=兒童座椅的重要|accessdate=2014-11-07|publisher=車輛安全資訊網}}</ref>

== 法律 ==
大多數國家和地区均規定前座乘客必須繫安全帶。
===亞洲地區===
====香港====
* 1983年，私家車及的士前座乘客必須繫上安全帶<ref name="tdgovhk" />
* 1996年6月1日，私家車後座乘客必須繫上安全帶<ref name="tdgovhk" />
* 2001年1月1日，的士座位如果設有安全帶，乘客必須繫安全帶<ref name="tdgovhk"><!-- http://www.info.gov.hk/td/chi/publication/taxi200101.pdf 相關的資料 -->{{cite web| url=http://www.td.gov.hk/tc/road_safety/seat_belt/index.html| title=安全帶| publisher=運輸署| date=2011-07-06| accessdate=2012-01-23| deadurl=yes| archiveurl=https://web.archive.org/web/20120101182439/http://www.td.gov.hk/tc/road_safety/seat_belt/index.html| archivedate=2012-01-01}}</ref>
* 2004年8月1日，乘客乘坐[[公共小巴|公共小巴]]必須繫上在車上設有的安全帶，實施日期以後登記的公共小巴亦須裝設安全帶。<ref name="tdgovhk" />
* 小童必須使用加高座椅<ref name="tdgovhk" />
* 自2005年起新的[[巴士|巴士]]所有乘客上層第一排座椅及較危險的座位（如：對頭座椅向前的座位及尾排面向通道走廊的座位）也裝設安全帶，但乘客是否繫上安全帶均屬自願性質。<ref>{{cite news |title=探射燈：搭小巴無扣帶 罰款坐監 |url=http://orientaldaily.on.cc/cnt/news/20160821/00176_132.html |accessdate=2018-11-13 |work=東方日報 |date=2016-08-21}}</ref>
====新加坡====
*驾驶者与前後座乘客都必须系上安全带。八岁以下的孩童，不论坐在前座或后座，都必须使用孩童安全椅。所有在1993年1月1日之后注册的汽车，都必须安装后座安全带。
====台湾==== <!--這是一個重定向點-->
* 2001年起汽車前座乘客必須全程繫安全帶。在之前只規定行駛於[[高速公路|高速公路]]及[[快速道路|快速道路]]才需要繫安全帶。
* 2012年起無論在[[中華民國|中華民國]]任何道路上，前後座皆要繫安全帶。違者罰新台幣1500元以上（平面道路及一般省道1500元，快速公路及高速公路3000～6000元）<ref>http://law.moj.gov.tw/LawClass/LawSingle.aspx?PCode=K0040012&FLNO=31 《道路交通安全處罰條例》第31條</ref>。營業大客車或計程車駕駛人已盡告知義務或已在座位貼有標示提醒，乘客仍未繫安全帶時，處罰該乘客。

==== 中国大陆 ====
[[File:Bengbu_Bus_No.105_and_No.116_on_Shengli_Rd.jpg|thumb]]
2008年5月1日开始施行的《中华人民共和国道路交通安全法》，第五十一条规定：机动车行驶时，驾驶人、乘坐人员应当按规定使用安全带，摩托车驾驶人及乘坐人员应当按规定戴安全头盔<ref>{{cite web |url=http://www.falvwangzhan.cn/falv/xw6/10117.html |title=存档副本 |accessdate=2011-07-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20120119000203/http://www.falvwangzhan.cn/falv/xw6/10117.html |archivedate=2012-01-19 }} 道路交通法对安全带有规定</ref>。开车不系安全带的司机会被处以50元人民币（下同）罚款和记2分的处罚；乘客不系安全带则会被罚款50元人民币。<ref>{{cite web|title=新交规开车不系安全带扣几分？罚款多少？开车时打手机电话或抽烟怎么处罚？|url=http://s.faydo.com/100/18/70.html|publisher=飞度网|access-date=2015-07-11|archive-url=https://web.archive.org/web/20150803011949/http://s.faydo.com/100/18/70.html|archive-date=2015-08-03|dead-url=yes}}</ref>原则上，小车后座的乘客也是需要系安全带的，但是鉴于一些车的后座并没有配置安全带等原因，交警主要还是对驾驶员和副驾驶不系安全带的情况进行处罚。<ref>{{cite web|author1=北京晨报|title=公交车道将“全天”禁占用 后座乘车人也要系安全带|url=http://www.bj.xinhuanet.com/bjyw/2014-10/16/c_1112844262.htm|publisher=新华网|deadurl=yes|archiveurl=https://web.archive.org/web/20150923192416/http://www.bj.xinhuanet.com/bjyw/2014-10/16/c_1112844262.htm|archivedate=2015-09-23}}</ref>不过，[[广东省|广东省]][[深圳市|深圳市]]从2016年7月4日起，对不系安全带的后排乘客开罚。驾驶员一经被警察发现后排乘客不系安全带将被罚款，其中普通市区路段罚款200元，高速路和快速干线罚款500元。但新规只罚司机，不罚乘客<ref>{{cite web|author1=广州日报|title=深圳正式对不系安全带后排乘客开罚 94名司机被罚|url=http://gd.sina.com.cn/news/sz/2016-07-05/detail-ifxtsatn8103018.shtml|publisher=新浪广东}}</ref>。另外，根据2017年3月25日实施的新《[[上海市|上海市]]道路交通管理条例》，规定后排乘客也要系好安全带，否则将被罚款<ref>{{cite web|author1=新华社|title=上海“最严交规”施行 交通违法整治不放松 |url=http://news.xinhuanet.com/2017-03/25/c_1120694391.htm|publisher=新华网}}</ref>。

==== 日本 ====
《[[日本工業標準|日本工業標準]]》於1966年3月1日起加入汽車用安全帶的規格標準。並於1969年4月1日起規定車輛需在駕駛座安裝安全帶。1973年及1975年各規定前座助手席及後座也需裝設安全帶。1994年起規定汽車需有未繫安全帶的警示系統。2012年7月1日起規定汽車需裝有供[[兒童安全座椅|兒童安全座椅]]固定用的[[ISOFIX|ISOFIX]]扣具。

1971年6月2日修正通過的《道路交通法》規定前座駕駛人及乘客有義務繫安全帶，但無罰則。1985年9月1日修法後規定行駛高速公路必須繫安全帶，否則受罰。1992年11月起擴及一般道路。2008年6月1日起規定所有座位乘客需繫安全帶，否則受罰。

=== 美洲地區 ===
* [[美國|美國]]
** 1968年規定汽車前座者均要安裝安全帶
** 部分州規定後座乘客須繫安全帶
* [[加拿大|加拿大]]
** 規定汽車所有座者均要繫上安全帶
** 小童必須使用加高座椅
** 在[[安大略省|安大略省]]，不繫上安全帶的乘客，如本身擁有駕駛牌照，會被扣分罰緩；如本身沒駕駛牌照，只會被罰緩，惟有關車輛司機會被扣分。

===歐洲地區===
* [[英國|英國]]：1989年要求後座孩童強制繫安全帶。1991年起所有後座乘客須繫安全帶。
* [[德國|德國]]：1984年起規定，前後座乘客均須繫安全帶（而有關修訂法例於1992年強制執行）。
===大洋洲地區===
* [[澳洲|澳洲]]：前後座乘客均須繫安全帶。
