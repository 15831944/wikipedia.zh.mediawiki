'''奘房'''（[[德宏傣語|德宏傣語]]：ᥓᥩᥒᥰ，-{zh-cn:汉语拼音字母记音：zông²;zh-tw:注音字母：ㄗㆲ²;zh-hant:注音字母：ㄗㆦㆭ²}-；傣龍語：ၵျွင်း，-{zh-cn:汉语拼音字母记音：jông⁴;zh-tw:注音字母：ㄐㆲ⁴;zh-hant:注音字母：ㄐㆦㆭ⁴}-）也称'''缅寺'''<ref>{{cite web |url=http://www.ebaifo.com/fojiao-230638.html |title=傣族缅寺，德宏傣族佛寺 |last= |first= |date= |website= |publisher=中国佛教网 |access-date=2017-10-20 |quote=}}</ref>，是滇西地区对[[上座部佛教|上座部佛教]]寺院的称呼，缅寺多指临沧地区的佛寺，德宏保山更多称奘房而非缅寺，德宏奘房与临沧缅寺在建筑造型上有较大差异，[[西双版纳傣族自治州|西双版纳]]的则称作佛寺<ref>{{cite book |author=刀金安、李才忠、岩光、奚云华、岩温叫、李炳荣 |date=2006年8月 |title=西双版纳傣族自治州民族宗教志 |url=http://f.wanfangdata.com.cn/view/%E7%AC%AC%E4%B8%89%E8%8A%82%20%E4%BD%9B%E6%95%99%E5%BB%BA%E7%AD%91.aspx?ID=LocalChronicleItem_4977346 |location=[[昆明市|昆明]] |publisher=云南民族出版社 |page=P178 |isbn=7-5367-3483-2 |author-link= }}</ref>。奘房多为两层木结构[[干栏式建筑|干栏式]]建筑<ref>{{cite web |url=http://fo.ifeng.com/a/20150914/41474305_0.shtml |title=密林中的“奘房”：阿昌族南传佛教信仰探秘 |last=刘 |first=铭初 |date=2015-09-23 |website=凤凰网佛教 |publisher=中国民族报 |access-date=2017-10-20 |quote=}}</ref>。奘房除了作为僧侣生活和信徒从事宗教活动的场所外，也是当地民众聚会的娱乐场所，西双版纳地区的佛寺也是儿童学习的场所<ref>{{cite web |url=http://www.sohu.com/a/100574067_376237 |title=国内最大竹编大佛，就在云南一座美丽的奘房里！ |last= |first= |date=2016-07-01 |website=搜狐旅游 |publisher=玩转云南 |access-date= |quote=}}</ref>。奘房這個稱呼來自緬甸佛教，在緬甸，佛寺與學校都叫做“奘”（{{lang-my|ကျောင်း|kyaung:}}，Okell拼音：caùñ，-{zh-cn:汉语拼音字母记音：ja᷆ong;zh-tw:注音字母：ㄐㆯˋ;zh-hant:注音字母：ㄐㄠㆭˋ}-），與漢語“{{zy|庠|xiáng}}”同源。爲作區別佛寺全稱“彭饥奘”（{{lang-my|ဘုန်းကြီးကျောင်း}}，Okell拼音：hpoùñc̱ì caùñ；语委转写：''bhun:kri:kyaung:''），義爲“僧校”。

==滇西==
===德宏奘房===
[[德宏傣族景颇族自治州|德宏]]地区许多村寨都建有奘房，规模小的寨子则出资与邻近大寨共用奘房。德宏最早的奘房是[[瑞丽市|瑞丽市]]的芒佛寺，傣语称“雷奘相”，意为宝石山佛寺，约建于11世纪初，[[陇川县|陇川县]][[章凤镇|章凤镇]]的寺院“奘相过”也建于同一时期。1956年德宏州共有奘房632座，其中[[傣族|傣族]]奘房575座，[[阿昌族|阿昌族]]44座，[[德昂族|德昂族]]13座；到1989年全州共有奘房551座。<ref name="新编德宏风物志"></ref>

德宏的奘房从建筑形式上分为两种类型：无墙基的“干栏式”传统傣族风格建筑，俗称 “楼奘”，此类型数量较多，在农村尤为普遍；另一种是有墙基的受汉族风格影响的形式，俗称“地奘”，多分布在城镇。奘房整体布局上通常是单座式建筑，设少量亭阁和佛塔等附属建筑物。大殿布局多为坐西向东，呈纵式平面，进深常大于面阔。大殿内的装饰物、装饰画以黄、红为主色调，屋顶有草顶、铁皮顶、瓦顶三种类型。<ref name="新编德宏风物志">{{cite book |last=张 |first=方元 |date=2000年5月 |title=新编德宏风物志 |url=http://f.wanfangdata.com.cn/view/%E5%A5%98%E6%88%BF.aspx?ID=LocalChronicleItem_5921520 |location=[[昆明市|昆明]] |publisher=云南人民出版社 |page= P154-P155 |isbn=7-222-02968-0 |author-link= }}</ref>

==緬甸==
{{seealso|缅甸佛教}}
“彭饥奘”（{{lang-my|ဘုန်းကြီးကျောင်း}}，Okell拼音：hpoùñc̱ì caùñ；语委转写：bhun:kri:kyaung:）義爲'''僧庠'''，“奘”本意是庠序，代指寺院，是僧人——尤其是沙弥——启蒙学习的场所。在缅甸，这个词甚至还可以借指基督教的[[修道院|修道院]]。中国境内的缅寺既是僧侣修行和居住的场所，也是儿童和沙弥学习文化的学校，甚至供民众聚会，这项传统便来自於缅甸<ref>{{Cite book|title=中国少数民族文化大辞典 西南地区卷|last=铁木尔·达瓦买提主编|first=|publisher=北京：[[民族出版社|民族出版社]]|year=1998.08|isbn=7-105-03121-2|location=|pages=422}}</ref>。

在緬甸，寺院還可以根據[[巴利語|巴利語]]源（{{IAST|vihara}}）稱爲{{lang|my|ဝိဟာရ}}（wihara）；在泰国则叫作{{lang|th|วิหาร}}（Wihan）。另外，還有從“圍墻”（{{lang-sa|वाट|vāṭa}}）演變的傣仂文「ᩅᨯ᩠ᨰ」（waD+Dha）或「ᩅᨯ᩠ᨵ」（waD+dha）、新傣仂文「ᦞᧆ」、傣沅文“ᩅ᩠ᨯ᩶”（w+Da2）、泰語{{lang|th|วัด}}（wát）、[[老撾語|寮語]]{{lang|lo|ວັດ}} (wat)、[[撣語|傣耶文]]{{lang|shn|ဝတ်ႉ }}（wâ̰t）<ref>见[[:wiktionary:en:วัด|维基词典【วัด】]]</ref>，中文稱爲“{{le|窪寺|Wat}}”。

景棟（撣邦東部）與泰北（蘭納），不少[[漥寺|漥寺]]實則具有奘房風格。由於僧庠與漥寺在當地僅僅是[[精舍|僧院]]之意，並無語義上的區分，衹有外地人將這兩個名稱用於指代緬柬兩種不同風格的寺廟。因此這兩地很多奘房被稱爲漥寺，也有很多漥寺被稱爲僧庠。此外，兩地還多見混合風格的寺廟，如{{link-th|Srǐi-rɔɔng-mʉʉang寺|วัดศฺรีรองเมือง}}。
