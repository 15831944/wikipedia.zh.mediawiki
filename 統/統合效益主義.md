'''統合效益主義'''（[[英文|英文]]：Unified Utilitarianism），又稱'''統合效用主義'''、'''統合功利主義'''，是一種[[倫理學|倫理學]]理論類型，以增量社會價值統合[[行為效益主義|行為效益主義]]與[[規則效益主義|規則結果主義]]<ref>{{cite book |author=盛慶琜|date=2003|title= 效用主義精解：統合效用主義理論之深|location=[[臺北市|臺北市]] |publisher=臺灣商務印書館|page=136 |isbn=9570518006 |author-link= }}</ref>，由[[盛慶琜|盛慶琜]]於1991年提出<ref name = "6o">{{cite journal|author=盛慶琜|title= 統合效用主義的六個奧義特色|ISSN=1671-4318|location=[[北京|北京]]|journal=世界哲學|publisher=中國社會科學院哲學所 |volume= |issue=4 |year=2012 |pages=36-44}}</ref>。
==背景==
[[盛慶琜|盛慶琜]]認為，[[行為效益主義|行為效益主義]]有許多問題難以解決<ref>{{cite journal|author=盛慶琜|title= 效用原則之解釋問題──效用主義之重大困難及其解決|ISSN=1000-856X|location=[[南京|南京]]|journal=江海學刊|publisher=江蘇省社會科學院|volume= |issue=2 |year=2003 |pages=31-38}}</ref>，而[[規則效益主義|規則結果主義]]具有謬誤，並且已經脫離效益，不算是[[效益主義|效益主義]]<ref name = "6o"/>，所以盛慶琜於1991年提出統合效益主義。
==主要觀點==
統合效益主義所主張之「效益」，是統合哲學家、經濟學家與決策理論家之「效益」，以「喜好」界定「效益」，而最後「效益」則界定「價值」，其中「效益」是主觀，而又可以統計。統合效益主義認為，「社會效益」為所有人之終極目標,並且亦主張「效益原則」，而「效益原則」又與「自然原則」、「共存原則」同時存在。統合效益主義計算效益時，會考慮「增量社會價值」，此與[[行為效益主義|行為效益主義]]不同。另外，統合效益主義認為，行為是個別，而不是種類，此與[[規則效益主義|規則結果主義]]不同。<ref>{{cite journal|author=徐珍|title= 統合效用主義的道德哲學研究|ISSN=1003-8477|location=[[武漢|武漢]]|journal=湖北社會科學|publisher=湖北省社會科學界聯合會；湖北省社會科學院|volume= |issue=6|year=2012 |pages=115-117}}</ref>

盛慶琜認為，統合效益主義主張只有行動層面是可以計算效益，與{{le|R. M. Hare|R. M. Hare]}}之雙重思考方式相近<ref name = "6o"/>。

==評論==
[[張華夏|張華夏]]認為，統合效益主義有原創力，並且足以抗衡各種倫理學學派。<ref>{{cite journal|author=張華夏|title=統合效用主義的理論貢獻及其問題──簡評盛慶琜|doi=10.3969/j.issn.1004-2938.2001.01.015|ISSN=1004-2938 |location=[[上海|上海]]|journal=開放時代|publisher=開放時代雜誌社|volume= |issue=1 |year=2001 |pages=120-123}}</ref>[[徐珍|徐珍]]認為，對統合效益主義而言，反對效益主義之觀點是微不足道或無傷大雅，因為統合效益主義可以有說服力地解釋。<ref>{{cite journal|author=徐珍|title= 統合效用主義的價值和困境|doi=|ISSN=1003-7462 |location=[[廣州|廣州]]|journal=嶺南學刊|publisher=中共廣東省委黨校|volume= |issue=6 |year=2014 |pages=118-122}}</ref>
