{{主條目消歧義|本條目介紹的是[[國家元首|國家元首]]或类似领袖对武装力量的統帥權。關於一般軍隊最高指揮職務，参見[[總司令|總司令]]。}}

'''統帥權'''（{{lang-de|Oberbefehl}}{{NoteTag|[[德语|德语]]「{{lang|de|Oberbefehl}}」一詞見於《[[俾斯麥帝國憲法|德意志帝國憲法]]》、《[[威瑪憲法|威瑪憲法]]》，其英譯同中華民國《國防法》官譯為「supreme command」。}}；{{lang-en|supreme command}}）又譯為'''最高指揮權'''，指統帥[[武装部隊|武装部隊]]者具有的最高、全權指揮權限。盛行於19世紀末至20世紀前半[[德國|德國]][[公法|公法]]學上解釋成[[國家元首|國家元首]]指揮軍隊的權力，涉及軍政、軍令（military administration and command）體系之分野。

== 概要 ==
*當時[[德國|德國]]公法學通說：元首發布軍事命令（軍令）統率軍隊乃指揮權，不需法律授權，也不受國會監督；軍事行政（軍政）屬統治行為、依法行政，接受國會的監督。
*[[中華民國|中華民國]]、過去的[[日本|日本]]在[[普魯士|普魯士]]軍制影響下採納統帥權理論，尤其前者落實於法律條文中。
*[[國家元首|國家元首]]為軍隊統帥行使統帥權，可分名義上如[[英國|英國]]、[[奧地利|奧地利]]，或實質上如[[美國|美國]]、[[俄羅斯|俄羅斯]]、[[法國|法國]]、中華民國等。
*像[[中華人民共和國|中華人民共和國]]、[[越南|越南]]、[[朝鮮|朝鮮]]，前[[蘇聯|蘇聯]]及[[東歐|東歐]]等採[[集體領導|集體領導]]的[[共產國家|共產國家]]，並無國家元首為軍隊統帥的概念。不過這些國家大多參考[[軍事委員會|軍事委員會]]或[[國防委員會|國防委員會]]的方式，以[[政黨領袖|政黨領袖]]（[[總書記|總書記]]、[[第一書記|第一書記]]）身分兼任最高軍事機關領導人來掌握實質軍隊指揮權。

== 英國 ==
[[英國君主|英國君主]]（國王或女王）為軍隊之統帥，確立後來大多[[立憲|立憲]]國家元首為軍隊統帥的典範；惟實際指揮權依慣例在[[英国内阁|內閣]]手上。

== 德國 ==
德意志皇帝[[威廉一世_(德國)|威廉一世]]時代，皇帝為軍隊統帥，內閣下設[[德國總參謀部|總參謀部]]，授予總參謀長面奏皇帝、彙報的權力，自此皇帝即掌握軍隊指揮、人員、編制等軍令上的實質權力，內閣則負責軍隊管理、預算等軍政事務，該軍令、軍政二元體制影響日後的日本、中華民國。

[[威瑪共和|威瑪共和]]時代的[[聯邦大總統|聯邦大總統]]，仍沿襲帝國時代國家元首掌握統帥權。[[二戰|二戰]]後，德國揚棄傳統的統帥權理論，改為實質的命令和指揮權，軍令、軍政一元化，平時國防部長綜理負責，戰時交由總理指揮；[[德國聯邦總統|聯邦總統]]並無名義上的統帥或指揮權，僅聯邦領域遭受武力攻擊時，得經聯邦議會同意就[[國際法|國際法]]宣告進入防衛狀態。

== 日本 ==
日本在[[明治|明治]]時期建立現代化軍隊時深受[[德意志帝國|德意志帝國]]的影響，[[天皇|天皇]]為[[日本軍|軍隊]]統帥，下設[[參謀本部_(大日本帝國)|陸軍參謀本部]]、[[軍令部_(日本海軍)|海軍軍令部]]，並准予兩方可直接將軍務上奏天皇的「{{link-ja|帷幄上奏|帷幄上奏}}」權力，不但獨立於[[日本內閣|內閣]]之外，且造成軍令、軍政二元體制下，軍令又二元化以[[大本營|大本營]]而非[[參謀部|參謀部門]]統籌指揮的獨特情形。曾發生內閣簽訂[[倫敦海軍條約|倫敦海軍條約]]引起干犯統帥權的重大爭議。

[[二戰|二戰]]後，日本透過[[日本國憲法|戰後憲法]]放棄[[宣戰|宣戰]]與建軍權利，天皇因此無統帥權，內閣依行政[[權能|權能]]指揮[[自衛隊|自衛隊]]，[[日本內閣總理大臣|內閣總理大臣]]（首相）則為自衛隊最高指揮官。

== 中華民國 ==
[[中華民國|中華民國]]肇建之初，[[北洋政府|北洋政府]]的[[中華民國大總統|大總統]]為陸海軍大元帥，設參謀本部負責軍令事宜，即使政府更迭，仍以大元帥地位持有統帥權。1926年，[[國民政府|國民政府]]設[[國民政府軍事委員會|軍事委員會]]負責軍政，軍令則是國民革命軍總司部直隸於國府，並由國民黨主席[[蔣中正|蔣中正]]出任軍委會主席兼國民革命軍總司令。1928年，蔣中正當選[[國民政府主席|國民政府主席]]，成為實質掌握統帥權之人，直至1931年寧粵分裂下野。1932年，日本發動[[一二八事變|一二八事變]]，蔣中正復出任軍委會委員長兼參謀本部參謀總長。1935年，國民政府授階蔣中正任[[特級上將|特級上將]]，以《特級上將授任條例》明定為中華民國陸、海、空軍最高軍事長官；與他國[[大元帥|大元帥]]之不同，在於他國為榮譽職或全權委任，而特級上將由當時國府集體領導下制定法律授予統帥權，則著眼於未來與日本愈演愈烈之衝突。1943年，蔣中正接任國府主席，並修改《中華民國國民政府組織法》，規定主席為海、陸、空軍大元帥。

抗戰勝利後政府[[行憲|行憲]]，[[中華民國總統|中華民國總統]]依《[[中華民國憲法|中華民國憲法]]》統率全國陸海空軍，僅僅於1978年制定《國防部參謀本部組織法》，[[國防部參謀本部|國防部參謀本部]]置[[參謀總長|參謀總長]]一人，在統帥、行政系統分別為總統、國防部長之幕僚長，總統行使統帥權，直接經由參謀總長指揮軍隊，依循軍政、軍令二元體制。

然而國防體制與權責法制化仍嫌不足，所以在2000年制定《國防法》與修改《國防部組織法》，參考[[法蘭西第五共和國|法國第五共和]]模式，總統決定國家安全有關之國防大政方針，或為因應國防重大緊急情勢，得召開[[中華民國國家安全會議|國家安全會議]]。總統行使統帥權指揮軍隊，直接責成國防部部長，由部長命令參謀總長指揮執行之。另外，廢止不合時宜的《特級上將授任條例》；並於2001年修正《國防部參謀本部組織法》全文為《國防部參謀本部組織條例》，參謀本部依國防部命令指揮軍隊，參謀總長不再是總統之幕僚長，達成軍政、軍令一元化。

== 中华人民共和国 ==
[[中华人民共和国|中华人民共和国]]在1954年召开[[第一届全国人民代表大会|第一届全国人民代表大会]]，并颁布第一部《[[中华人民共和国宪法|中华人民共和国宪法]]》。根据《[[五四宪法|五四宪法]]》的规定，作为[[中国国家元首|中国国家元首]]代表的[[中华人民共和国主席|中华人民共和国主席]]兼任[[中华人民共和国国防委员会|国防委员会]]主席，统帅全国武装力量。不过[[中国共产党|中国共产党]]一直坚持对武装力量的绝对控制，奉行“[[党指挥枪|党指挥枪]]”的根本原则，[[中国共产党中央军事委员会|中国共产党中央军事委员会]]才是实际上的最高军事统帅机关，其负责人[[中国共产党中央军事委员会主席|中央军委主席]]才是最高统帅。1959年[[刘少奇|刘少奇]]继任中国国家主席，同时兼任国防委员会主席的职务，成为宪法上的军事统帅，不过当时[[毛泽东|毛泽东]]还继续担任[[中国共产党中央委员会主席|中国共产党中央委员会主席]]和[[中央军事委员会主席|中央军事委员会主席]]的职务，其最高的政治地位不变，也是实际上的最高统帅，毛泽东本人也[[终身领导人|终身]]担任这两个职务直到[[毛泽东之死|逝世]]。

[[文化大革命|文化大革命]]爆发后，担任[[中华人民共和国主席|国家主席]]的[[刘少奇|刘少奇]]被打倒，这个职务长期缺位，直到1975年被修宪废除。按照《七五宪法》和《七八宪法》的规定，作为[[党主席|党主席]]的[[中国共产党中央委员会主席|中国共产党中央委员会主席]]统帅全国武装力量。1982年通过的第四部宪法《[[八二宪法|八二宪法]]》恢复中国国家主席的设置，不过没有授予其统帅权，取而代之是设立[[中华人民共和国中央军事委员会|中华人民共和国中央军事委员会]]，这个机构与中共中央军委是“[[一个机构两块牌子|一个机构两块牌子]]”，两个机构的组成人员除了权力交接时期以外都是一样的，[[中华人民共和国中央军事委员会主席|国家军委主席]]被赋予宪法上军事统帅的地位。无论中华人民共和国的宪法体制如何变更，中国共产党对武装力量的绝对控制都没有变，[[中共中央军委主席|中共中央军委主席]]一直是实际的最高统帅，也被外界认为是辨认[[中华人民共和国最高领导人|最高领导人]]的指标。
