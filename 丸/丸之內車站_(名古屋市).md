'''丸之內站'''（{{jpn|j=丸の内駅|hg=まるのうちえき|rm=Marunouchi eki|f=y}}）是位於[[愛知縣|愛知縣]][[名古屋市|名古屋市]][[中區_(名古屋市)|中區]]丸之內二丁目，為[[名古屋市營地下鐵|名古屋市營地下鐵]]之車站。

== 可利用之鐵路路線 ==
*名古屋市營地下鐵
**[[鶴舞線|鶴舞線]] - 車站編號為'''T06'''。
**[[櫻通線|櫻通線]] - 車站編號為'''S04'''。

== 車站結構 ==
名城線與櫻通線各為1面2線[[島式月台|島式月台]]之[[地下車站|地下車站]]。
本站櫻通線之月台，設置於距離地面約深24[[公尺|公尺]]之處。在名古屋市營地下鐵之中，為位於最深之處的車站。
