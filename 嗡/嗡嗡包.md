'''嗡嗡包'''是自中華民國政治人物[[柯文哲|柯文哲]]擔任[[台北市長|台北市長]]以來，台北市政府每年[[10月10日|10月10日]]（[[國慶日_(中華民國)|中華民國的國慶日]]）免費發送給一般民眾的束口袋或[[背包|背包]]，自2015年至2018年已經發送四次。

嗡嗡包的名稱及設計概念出自於柯文哲經常哼唱的歌曲《[[小漢斯|小蜜蜂]]》<ref name="名稱1"/><ref name="名稱2"/>，《小蜜蜂》中描述的蜜蜂辛勤工作的樣態，嗡嗡包便以此為名<ref name="市府15-1"/><ref name="大紀元2017"/>，以期每個人都能樂在工作<ref name="市府15-4"/>。每年的嗡嗡包有不同的款式、顏色和主題，發送的數量也不盡相同。

== 第一代 ==
第一代嗡嗡包為黑色束口袋，共三款。束口袋中央印有期望人民樂在工作的蜜蜂圖案以及慶祝中華民國104年國慶日的「104國慶」字樣<ref name="大紀元2017"/>，尼龍材質，束口袋的一角為不同顏色的網狀小口袋，有紅、黃、藍3種<ref name="TVBS2015"/><ref name="市府15-5"/>。柯文哲認為嗡嗡包必須以實用為考量，而不是看一眼後就丟到垃圾桶。<ref name="市府15-2"/>

第一代嗡嗡包在2015年10月10日，即中華民國104年國慶日時於臺北市政府前廣場發放。在發放之前，柯文哲先於10月9日晚間在臉書宣傳，鼓勵民眾領取嗡嗡包<ref name="中時2015"/>。10月10日上午六點半，發放可兌換嗡嗡包的兌換券，柯文哲在參加完在[[總統府_(臺灣)|總統府]]前的國慶升旗典禮後抵達台北市政府前，民眾可憑兌換券向他領取嗡嗡包<ref name="TVBS2015"/>。嗡嗡包每款各有1000份，共3000份<ref name="TVBS2015"/>。根據北市府民政局統計，最早在10日凌晨2時許便有人開始排隊，至早上5、6點時開始湧入大量民眾<ref name="市府15-4"/>，共有近6000名群眾想索取嗡嗡包<ref name="中時2015"/>，群眾的反應相當熱烈，有些人為此不惜[[插隊|插隊]]，引發場面混亂<ref name="自由20153"/>，也有人排隊等了一個小時才得知號碼牌已經發完，引起抱怨。嗡嗡包在開始發放後一個小時內發完<ref name="中時2015"/>。柯文哲得知後表示始料未及，並引用《[[論語|論語]]》指出場面混亂的原因是「不患寡而患不均」<ref name="市府15-3"/>，北市府民政局也因此被要求檢討<ref name="市府15-9"/>，並表示未來的活動會有更妥善的規劃。<ref name="中時2015"/><ref name="市府15-4"/>

由於民眾反應熱烈，臺北市政府決定再製造嗡嗡包販售。於市府大樓西大門入口左側的紀念品中心販售，由於製作需要時間，先在10月14日上午九點提供100份，10月26日再提供900份<ref name="自由2015"/><ref name="市府15-5"/>，為了避免場面失控和大量採購，一次僅限三人進入購買<ref name="自由20153"/>，每人限購兩個<ref name="三立2015"/>。10月14日販售時，一對兄弟自凌晨三點時便開始排隊，至八點半左右有六十多位民眾排隊<ref name="自由20151"/>，開賣後19分鐘便售罄<ref name="市府15-8"/>。26日販售時，凌晨兩點便有人開始排隊。<ref name="自由20152"/>

有部份民眾在領到嗡嗡包後將其至拍賣網站上轉售，有的轉賣價格為原價十倍。柯文哲認為這是[[自由市場|自由市場]]，並引用[[侏羅紀公園系列|《侏羅紀公園》系列]]的名台詞，表示「每個生物都會找到他的出路」<ref name="自由20154"/><ref name="新頭殼2015"/><ref name="市府15-7"/>，他還表示若要推出市長簽名版，售價要高一點<ref name="中時20151"/><ref name="三立20151"/>。對於議員質疑販售嗡嗡包與行銷臺北市政府無關，柯文哲表示是此舉為反映民眾所需，並非為了獲利或個人行銷。<ref name="市府15-6"/>

== 第二代 ==
第二代嗡嗡包為灰色束口袋，共三款，上半部分別為粉綠、粉藍、粉紅色，並且為了配合推廣於2017年在台北市舉辦的[[2017年夏季世界大學運動會|世界大學運動會]]<ref name="中時20162"/><ref name="大紀元2016"/>，以台北世大運的吉祥物「[[2017年夏季世界大學運動會#吉祥物|熊讚]]」為主軸設計<ref name="市府16-1"/>，用進行運動的熊讚取代原本的蜜蜂圖案。<ref name="自由20162"/><ref name="中時20161"/><ref name="大紀元2017"/>

第二代嗡嗡包在2016年10月10日，即中華民國105年國慶日時於臺北市政府前廣場發放。當日上午六點半開始發放可兌換嗡嗡包的兌換券，並在8點時開放憑兌換券領取嗡嗡包，當日準備了5000個嗡嗡包，在10點前兌換完畢<ref name="自由20161"/>，最早排隊的人自10月9日晚間9點便在場等候<ref name="三立2016"/><ref name="中時20161"/>。為避免重蹈去年覆轍，領取嗡嗡包的動線有所調整<ref name="自由20162"/>，更配合民眾需要挪動原本用於懸掛國旗的經費，較去年多製備了2000個嗡嗡包，以免供不應求，也因此並未在國慶日後製造嗡嗡包販售。<ref name="三立2016"/><ref name="中時20162"/>

== 第三代 ==
第三代嗡嗡包的設計有別以往，為可以防潑水的雙肩背包，有摺疊釦環設計<ref name="台視2017"/>，成本也較以往更高<ref name="民視2017"/>，有黑色和綠色兩種款式，背包上印有「TAIPEI 2017」字樣，並留有較大的空白處讓民眾自行設計。這一代的嗡嗡包附一組貼紙，民眾可自由決定是否將貼紙貼在嗡嗡包上，或可彩繪、懸掛吊飾，北市府希望能以此鼓勵民眾自由創作<ref name="市府17-1"/><ref name="自由20171"/>。第三代嗡嗡包在2017年10月6日的記者會上首次亮相<ref name="自由20172"/>，並請[[臺北市立中正高級中學|臺北市立中正高中]]的美術班學生在記者會上展示自己設計的嗡嗡包<ref name="市府17-1"/>。柯文哲認為過往的嗡嗡包僅止於好用，而第三代嗡嗡包講究品質<ref name="中央2017"/>，他甚至認為這一代的嗡嗡包太豪華了<ref name="市府17-2"/><ref name="市府17-4"/>。柯文哲在10月9日於個人[[Facebook|Facebook]]頁面上貼出自己揹著第三代嗡嗡包的照片，有網友認為照片中柯文哲的姿勢很像日本小說家[[西尾維新|西尾維新]]的系列作品《[[物語系列|物語]]》中的女性角色[[物語系列角色列表|八九寺真宵]]，便用[[修飾照片|修飾照片]]技術將照片中的柯文哲加上該角色的[[雙馬尾|雙馬尾]]及髮帶，引起[[批踢踢|批踢踢]]網友評論。<ref name="自由2017"/>

第三代嗡嗡包在2017年10月10日，即中華民國106年國慶日時於[[市民廣場_(臺北市)|臺北市市民廣場]]發放<ref name="聯合20171"/>。當日上午6點半開始於北市府前發放可兌換嗡嗡包的兌換券<ref name="市府17-1"/><ref name="中時2017"/>，8點開始憑券兌換嗡嗡包和貼紙，由於經費有限，僅發送3000個<ref name="自由20171"/>，至上午9點時兌換完畢<ref name="市府17-1"/><ref name="市府17-3"/><ref name="聯合20172"/>。有人從9日下午便開始排隊，也有人從台南騎自行車北上參加<ref name="聯合20173"/>，而當日有五千餘人想領取嗡嗡包<ref name="TVBS2017"/>，之後拍賣網站上也出現了高價轉售的嗡嗡包。<ref name="台視2017"/>

== 第四代 ==
第四代嗡嗡包在2018年10月1日上午十點公布，台北市副市長[[鄧家基|鄧家基]]於市政府召開記者會，和民政局長[[藍世聰|藍世聰]]及市府吉祥物[[熊讚|熊讚]]一同說明嗡嗡包的數量、領取時間和排隊方式，並展示了第一代到第四代的嗡嗡包<ref name="蘋果20181"/>。第四代的嗡嗡包是尺寸比過去更大的雙肩背包，材質為雪花布<ref name="中時20181"/>，容量比過去的更大<ref name="蘋果20187"/>，有藍色和紅色兩種色款，每色1500個，共3000個，上面印有「2018 TAIPEI」字樣。預計於10月10日上午6點半發放號碼牌，並在8點憑號碼牌兌換嗡嗡包<ref name="蘋果20182"/>，領取民眾可自由挑色<ref name="聯合20181"/>。為了宣傳嗡嗡包，柯文哲身穿西裝，背上嗡嗡包拍攝宣傳照<ref name="今日20181"/>，他表示這一次的成本比去年更低，但看到樣品時感覺很不錯，認為這是一種進步<ref name="新頭殼"/>，他還指出很多人因為公家機關贈送的物品上經常印有政治人物的姓名或題字而不感興趣，因此這次的嗡嗡包不會出現這些東西<ref name="TVBS20184"/><ref name="蘋果20183"/><ref name="自由20183"/><ref name="今日20181"/><ref name="風傳媒2018"/><ref name="三立20181"/>。由於2018年是柯文哲在該次任期內擔任市長的最後一年，一些民眾認為這可能是最後一代嗡嗡包，柯文哲僅表示明年還是會有國慶，每年的創意會不同。<ref name="三立20182"/>

隨嗡嗡包附贈的物品是印有中華民國國旗圖案的圓形徽章，供民眾自由搭配，國民黨政治人物[[徐巧芯|徐巧芯]]推測此舉是柯文哲想拉攏泛藍選民<ref name="TVBS20182"/>。民進黨台北市長參選人[[姚文智|姚文智]]也批評，柯文哲的行為是拿市政府資源競選<ref name="蘋果20185"/>。藍世聰表示每一年都有將國旗納入設計，國旗徽章並不是想拉攏泛藍選民，和年底要進行的[[2018年中華民國直轄市長及縣市長選舉|2018年台北市長選舉]]也無關<ref name="聯合20181"/><ref name="聯合20182"/><ref name="TVBS20181"/><ref name="ETtoday20181"/><ref name="自由20181"/>。

第四代嗡嗡包於2018年10月10日，即中華民國107年國慶日於台北市政府前廣場前發放。有些人從前一天上午就開始排隊，也有人帶著雨衣和板凳，專程從屏東北上熬夜排隊<ref name="TVBS20183"/>。在10月9日就出現了排隊人潮<ref name="蘋果20184"/>，雖然原本預計在10日的6點半發放，但在凌晨2點時，排隊人數就超過三千人，號碼牌也已經發放完畢<ref name="蘋果20186"/><ref name="自由20184"/><ref name="中央社2018"/>。在領取過程中，除了有許多插隊的情形，工作人員也因為誤判，讓某些原本可以領到兌換券的人失去了機會，導致排隊民眾發生推擠衝突<ref name="ETtoday20182"/>，另外還有人持偽造的兌換券企圖兌換嗡嗡包，最後被警方和工作人員追回<ref name="聯合20183"/>。在嗡嗡包被發完後兩個小時，拍賣網站上就出現了轉售的嗡嗡包，售價比市政府公布的成本還高出數倍到十數倍。<ref name="TVBS20185"/><ref name="蘋果20187"/><ref name="自由20182"/><ref name="今日20182"/><ref name="中時20182"/>
