{{日本作品|春音Alice＊Gram|春音アリス＊グラム}}是[[NanaWind|NanaWind]]在2017年6月30日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]<ref>{{cite web|url=http://moedigi.com/game/2017/06/nanawind-haruoto-master-up/|title=恋と秘密のちょっと不思議な学園ADV。NanaWind『春音アリス＊グラム』がマスターアップ|website=moedigi|date=2017-06-07|accessdate=2019-03-20|language=ja}}</ref>。2019年3月28日由Entergram發售[[PlayStation_4|PlayStation 4]]與[[PlayStation_Vita|PlayStation Vita]]版本《'''春音Alice＊Gram Snow Drop'''》<ref>{{cite web|url=https://www.gamer.ne.jp/news/201902280063/|title=「春音アリス＊グラム Snow Drop」オープニングムービーとゲーム序盤が楽しめる体験版配信！|website=Gamer|date=2019-02-28|accessdate=2019-03-20|language=ja}}</ref>。2019年3月29日發行續作《'''白戀Sakura＊Gram'''》（{{lang|ja|白恋サクラ＊グラム}}）<ref>{{cite web|url=http://moedigi.com/game/2019/02/nanawind-sakura-master-up/|title=NanaWind『白恋サクラ＊グラム』がマスターアップ。『春音アリス＊グラム』の続編タイトル！|website=moedigi|date=2019-02-20|accessdate=2019-03-20|language=ja}}</ref>。

== 故事簡介 ==
成稜學園有學生會跟新生會兩個組織，男主角瀨真颯太朗所加入的黃昏社歸於學生會管轄並幫學生會解決問題。某一天颯太朗收到了一封神祕的信，並要颯太朗解決信上的問題<ref>{{cite web|url=https://www.tgsmart.jp/article.aspx?a=13819|title=NanaWind完全新作！ 恋と秘密の学園ADV『春音アリス＊グラム』を紹介!!|website=TG Smart|date=2016-03-25|accessdate=2019-03-20|language=ja}}</ref>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|瀨真颯太朗|瀬真 颯太朗（せま そうたろう）}}
: 成稜學園二年級生，黃昏社成員，成績優秀。
; {{nihongo|白羽優理|白羽 優理（しらはね ゆうり）|聲：[[遙空|遙空]]}}
: 成稜學園二年級生，學生會長，擅長照顧人且責任感強烈，深受學生信賴。
; {{nihongo|久遠寺一葉|久遠寺 一葉（くおんじ かずは）|聲：波奈束風景}}
: 成稜學園二年級生，風紀委員會的委員長，颯太朗的青梅竹馬。擁有「成稜的白百合之名」。
; {{nihongo|神咲艾麗莎|神咲 エリサ（かんざき エリサ）|聲：奏雨}}
: 從外國到成稜學園留學的公主，就讀二年級。好奇心旺盛，非常喜歡日本大眾文化與風俗，後來加入黃昏社。
; {{nihongo|凜堂耶耶|凜堂 耶々（りんどう やや）|聲：{{lang|ja|くすはらゆい}}}}
: 成稜學園三年級生，黃昏社的社長，理事長的女兒，擁有小惡魔的一面。
; {{nihongo|藤乃雪|藤乃 雪（ふじの ゆき）|聲：[[杏子御津|杏子御津]]}}
: 成稜學園一年級生，新生會的書記，經常到黃昏社的社團教室玩耍。家裡是歷史悠久的餐廳，有著高超的料理技術。
; {{nihongo|橘朝陽|橘 朝陽（たちばな あさひ）|聲：碧木梨（春音PC）→月讀梓（春音PS4·PSV、白戀）}}
: PS4、PSV版升格為女主角<ref>{{Cite web|url=http://www.entergram.co.jp/alicegram/point/|title=POINT|work=PS4、PSV版遊戲「春音Alice＊Gram Snow Drop」官方網站|publisher=ENTERGRAM|accessdate=2018-10-25|language=ja}}</ref>。成稜學園一年級生，學生會的副會長兼會計，和優理感情很好。
; {{nihongo|瀨真梢|瀬真 梢（せま こずえ）|聲：花寺香蓮}}
: 《白戀Sakura＊Gram》升格為女主角。成稜學園一年級生，隸屬於風紀執行部。颯太朗的妹妹，個性活潑。
; {{nihongo|天雨莉莉艾特|天雨 リリエト（あもう りりえと）|聲：{{lang|ja|白月かなめ}}}}
: 《白戀Sakura＊Gram》登場的女主角。在春假時轉進成稜學園一年C班。本身是西歐某機關派來的監察員。
