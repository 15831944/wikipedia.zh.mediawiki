'''春秋五霸'''，「五霸」又作「五伯」，是指[[東周|東周]][[春秋時期|春秋時期]]相繼稱[[霸主|霸主]]的五個[[諸侯|諸侯]]，其說首見于《[[左傳|左傳]]》<ref>《[[左傳|左傳]]·[[s:春秋左氏傳/成公#成公二年|成公二年]]》：「四王之王也，樹德而濟同欲焉；五伯之霸也，勤而撫之，以役王命。」</ref>。該詞常與'''[[戰國七雄|戰國七雄]]'''並稱。

目前主要有兩種觀點：
# [[齊桓公|齊桓公]]、[[晋文公|晋文公]]、[[楚莊王|楚莊王]]、[[宋襄公|宋襄公]]、[[秦穆公|秦穆公]]（可記為：齊晉楚宋秦）<ref>[[史記|史記]]</ref>；
# [[齊桓公|齊桓公]]、[[晉文公|晉文公]]、[[楚莊王|楚莊王]]、[[吳王闔閭|吳王闔閭]]、[[越王勾踐|越王勾踐]]（可記為：齊晉楚吳越）<ref>[[荀子|荀子]]</ref>。

== “霸”的含义 ==
“霸”，意为霸主，即是诸侯之[[領袖|領袖]]，在这个意义上，与代表[[長者|長者]]的“[[伯|伯]]”实同一词，<ref>[[王力_(语言x学家)|王力]]，《[[古汉语字典|古汉语字典]]》，[[中华书局|中华书局]]，2000.6. p1619.</ref>并无现代的“蛮横无理”的含义。職責：
*[[尊王攘夷|尊王攘夷]]
*禁抑篡弑
*制裁兼併<ref>[[錢穆|錢穆]]，[[國史大綱|國史大綱]]，1996.6. p59.</ref>

== 五霸所指 ==
五霸究竟指何五位[[國君|國君]]，历来有不同説法，目前史學界尚无任何定論，从古至今，关于“五霸”至少出现过六种说法。但是大部分中学课本通常都是用《[[史記|史記]]》或《[[荀子|荀子]]》的版本。

#'''[[齐桓公|齐桓公]]'''、'''[[晋文公|晋文公]]'''、'''[[秦穆公|秦穆公]]'''、'''[[楚庄王|楚庄王]]'''、'''[[宋襄公|宋襄公]]'''。《[[史记|史记]]》
#'''[[齐桓公|齐桓公]]'''、'''[[晋文公|晋文公]]'''、'''[[秦穆公|秦穆公]]'''、'''[[楚庄王|楚庄王]]'''、'''[[郑庄公|郑庄公]]'''。《辞通》
#'''[[齐桓公|齐桓公]]'''、'''[[晋文公|晋文公]]'''、'''[[秦穆公|秦穆公]]'''、'''[[楚庄王|楚庄王]]'''、'''[[越王勾践|越王勾践]]'''。《[[s:四子講德論|s:四子講德論]]》
#'''[[齐桓公|齐桓公]]'''、'''[[晋文公|晋文公]]'''、'''[[秦穆公|秦穆公]]'''、'''[[楚庄王|楚庄王]]'''、'''[[吴王阖闾|吴王阖闾]]'''。《[[白虎通|白虎通]]·号篇》
#'''[[齐桓公|齐桓公]]'''、'''[[晋文公|晋文公]]'''、'''[[秦穆公|秦穆公]]'''、'''[[宋襄公|宋襄公]]'''、'''[[吴王夫差|吴王夫差]]'''。《[[s:漢書/卷014|s:漢書/卷014]]·諸侯王表·[[顏師古|顏師古]]注》
#'''[[齐桓公|齐桓公]]'''、'''[[晋文公|晋文公]]'''、'''[[楚庄王|楚庄王]]'''、'''[[吴王阖闾|吴王阖闾]]'''、'''[[越王勾践|越王勾践]]'''。《[[荀子|荀子]]·王霸》

另有學者认为，所谓“五霸”应是一個[[虛數|虛數]]，指諸多位君主，并非实指五位国君。<ref>赵东玉，五霸别解，史学集刊，2000.02</ref>

《[[白虎通義|白虎通義]]》中記載兩種說法，以夏代[[昆吾氏|昆吾氏]]，商代[[大彭氏|大彭氏]]、[[豕韋氏|豕韋氏]]與東周齊桓公、晉文公為五霸。另一種說法為，齊桓公、晉文公、秦穆公、楚莊王與吳王闔閭<ref>《白虎通義》卷1：「五霸者，何謂也？昆吾氏、大彭氏、豕韋氏、齊桓公、晉文公也。昔三王之道衰，而五霸存其政，率諸侯朝天子，正天下之化，興復中國，攘除夷狄，故謂之霸也。昔昆吾氏，霸於夏者也；大彭氏、豕韋氏，霸於殷者也；齊桓、晉文，霸於周者也。或曰：五霸，謂齊桓公、晉文公、秦穆公、楚莊王、吳王闔閭也。」</ref>。
