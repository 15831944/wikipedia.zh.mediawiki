'''春日基地'''（{{jpn|j=春日基地|hg=かすがきち|rm=Kasuga Kichi}}、{{lang-en|JASDF Kasuga Air Base}}）是[[日本|日本]][[航空自衛隊|航空自衛隊]]設施，位於[[九州_(日本)|九州]]的[[福冈县|福冈县]][[春日市|春日市]]原町3-1-1。本基地為九州、[[四國|四國]]及[[本州島|本州島]][[中國地方|中國地方]]空防部隊——{{Link-ja|航空自衛隊西部航空方面隊|西部航空方面隊|西部航空方面隊}}的司令部所在處<ref name="航空總隊">{{Cite web |language=ja |script-title=ja:西部航空方面隊 |publisher={{lang|ja|{{Link-ja|航空自衛隊航空總隊|航空総隊|航空総隊}}}} |url=http://www.mod.go.jp/asdf/adc/butaisyoukai/003western/butaisyoukaiwestern.html |accessdate=2017-09-04 |archiveurl=http://archive.is/Ef4YR |archivedate= 2017-09-04}}</ref>，基地司令則由{{Link-ja|航空自衛隊西部航空警戒管制團|西部航空警戒管制団|西部航空警戒管制團}}司令兼任<ref name="司令">{{Cite web |language=ja |script-title=ja:基地司令からの挨拶 |publisher={{lang|ja|春日基地}} |url=http://www.mod.go.jp/asdf/kasuga/kichishirei/index.html |accessdate=2017-09-04 |archiveurl=https://archive.is/20170904055006/http://www.mod.go.jp/asdf/kasuga/kichishirei/index.html |archivedate=2017-09-04 |deadurl=yes }}</ref>。本基地分為司令部、隊本部等單位所在的營區，及設在[[福岡機場|福岡機場]]內部、供飛行部隊用機停放的飛行場區，而日本－[[韓國|韓國]]間的軍用[[熱線|熱線]]也分別以本基地和[[大邱|大邱]]的基地為日韓兩端端點，利於雙方進行[[軍事情報|軍事情報]]和飛航業務上的通訊。

==設施概述==
[[File:F-1_kasuga.jpg|thumb]]]]
在1954年[[自衛隊|自衛隊]]建立之前，[[春日市|春日]]一帶即曾存有軍事用途之建築設施。[[第二次世界大戰|第二次世界大戰]]發生時，當地即曾設置{{Link-ja|陸軍造兵廠|陸軍造兵廠|造兵廠}}春日分廠<ref name="春日歷史">{{Cite web |language=ja |script-title=ja:春日の歴史 |publisher={{lang|ja|[[春日市|春日市]]}} |url=http://www.city.kasuga.fukuoka.jp/towninfo/kasuga/rekisi.html |date=2017-06-29 |accessdate=2017-09-03 |archiveurl=https://archive.is/20170903141848/http://www.city.kasuga.fukuoka.jp/towninfo/kasuga/rekisi.html |archivedate=2017-09-03 |deadurl=yes }}</ref>，1945年[[日本投降|日本投降]]、進入[[盟軍佔領日本|盟軍佔領時期]]後，[[美軍|美軍]]亦曾於春日町（春日市舊制行政區劃）設置基地。1959年3月1日，日本航空自衛隊的西部航空司令所於春日基地宣告成立<ref name="50年">{{Cite web |language=ja |script-title=ja:５０年の歩み |publisher={{lang|ja|{{Link-ja|航空自衛隊西部航空方面隊|西部航空方面隊|西部航空方面隊}}}} |url=http://www.mod.go.jp/asdf/wadf/history/index.html |accessdate=2017-09-03 |archiveurl=http://archive.is/cEMuI |archivedate= 2017-09-03}}</ref>，至1961年時，航空自衛隊不僅將該司令所改編為西部航空方面隊，尚於同年7月將春日基地的{{Link-en|防空管制中心|Air Defense Direction Center}}從美軍手中收回<ref name="道下德成">{{Cite book |language=ja |script-title=ja: 自衛隊のエア・パワーの発展と意義 |journal={{lang|ja|エア・パワー}} |author={{lang|ja|{{Link-ja|道下德成|道下徳成|道下徳成}}}} |publisher={{lang|ja|芙蓉書房}} |editor={{lang|ja|{{Link-ja|石津朋之|石津朋之}}}} |page=179-180 |date=2005 |isbn=9784829503560 }}</ref>{{ref label|防空管制中心|註1|}}。

春日基地所屬的建築物呈現分散式的分布——基地正門所在的營區緊鄰春日市役所、{{Link-ja|福岡縣營春日公園|福岡県営春日公園|春日公園}}等場所，為各司令部及部隊本部所在地<ref name="地圖">{{Cite web |language=ja |script-title=ja:地図 |publisher={{lang|ja|春日基地}} |url=http://www.mod.go.jp/asdf/kasuga/map/index.html |accessdate=2017-09-03 |archiveurl=https://archive.is/20170903171153/http://www.mod.go.jp/asdf/kasuga/map/index.html |archivedate=2017-09-03 |deadurl=yes }}</ref>，而飛行部隊的駐地則位於北方的福岡機場內，亦稱「春日基地飛行場地區」<ref name="手嶋文史">{{Cite web |language=ja |script-title=ja:福岡空港の春日基地飛行場地区について 西部航空方面隊オピニオンリーダー |publisher={{lang|ja|{{Link-ja|航空自衛隊西部航空方面隊|西部航空方面隊|西部航空方面隊}}}} |url=http://www.mod.go.jp/asdf/wadf/opinion/opinionleader/opinion_teshima2.pdf |author={{lang|ja|手嶋文史}} |accessdate=2017-09-03 |archiveurl=https://web.archive.org/web/20170325022913/http://www.mod.go.jp/asdf/wadf/opinion/opinionleader/opinion_teshima2.pdf |archivedate= 2017-09-03}}</ref>或「春日基地板付地區」<ref name="米澤敬一">{{Cite web |language=ja |script-title=ja:筥崎宮と福岡空港 |publisher={{lang|ja|{{Link-ja|航空自衛隊西部航空方面隊|西部航空方面隊|西部航空方面隊}}}} |url=http://www.mod.go.jp/asdf/wadf/rocky/sampomichi/rocky4.html |author={{lang|ja|米沢敬一}} |date=2012-02-27 |accessdate=2017-09-03 |archiveurl=http://archive.is/U7cuT |archivedate= 2017-09-03}}</ref><ref name="米澤敬一目錄">{{Cite web |language=ja |script-title=ja:米沢元副司令官の趣味の歴史散策コーナー |publisher={{lang|ja|{{Link-ja|航空自衛隊西部航空方面隊|西部航空方面隊|西部航空方面隊}}}} |url=http://www.mod.go.jp/asdf/wadf/rocky/ |author={{lang|ja|米沢敬一}} |accessdate=2017-09-03 |archiveurl=https://archive.is/20170903145451/http://www.mod.go.jp/asdf/wadf/rocky/ |archivedate= 2017-09-03}}</ref>，春日基地所駐[[飛機|飛機]]、[[直升機|直升機]]的[[機庫|機庫]]也設在該機場<ref name="手嶋文史"/>。

除基地設施外，春日基地亦與韓國[[大邱|大邱]]的基地之間建有[[熱線|熱線]]，以便日韓兩國就情資交換、[[防空識別區|防空識別區]]重疊空域內的雙方[[軍用機|軍用機]]相互通報機制等事項，實施一日數次的雙邊通訊。[[日本防衛省|日本防衛省]]於2013年12月10日的大臣記者會中提及春日－大邱熱線的存在<ref name="防衛省日">{{Cite web |language=ja |script-title=ja:大臣会見概要 |publisher={{lang|ja|[[防衛省|防衛省]]}} |url=http://www.mod.go.jp/j/press/kisha/2013/12/10.html |date=2013-12-10 |accessdate=2017-09-03 |archiveurl=https://archive.is/20170903144804/http://www.mod.go.jp/j/press/kisha/2013/12/10.html |archivedate=2017-09-03 |deadurl=yes }}</ref><ref name="防衛省英">{{Cite web |language=en |script-title=en:Press Conference by the Defense Minister |publisher={{lang|en|[[防衛省|Ministry of Defense, Japan]]}} |url=http://www.mod.go.jp/e/press/conference/2013/12/10.html |date=2013-12-10 |accessdate=2017-09-03 |archiveurl=https://archive.is/20170903143004/http://www.mod.go.jp/e/press/conference/2013/12/10.html |archivedate=2017-09-03 |deadurl=yes }}</ref><ref name="日經中文">{{Cite web |language=zh-hant |title=日韓或在重疊防空識別區相互通報 |publisher=[[日經中文網|日經中文網]] |author=加藤宏一 |url=http://zh.cn.nikkei.com/politicsaeconomy/politicsasociety/7305-20131210.html |date=2013-12-10 |accessdate=2017-09-03 |archiveurl=https://archive.is/20170903142919/http://zh.cn.nikkei.com/politicsaeconomy/politicsasociety/7305-20131210.html |archivedate=2017-09-03 |deadurl=yes }}</ref>。

==部隊==
春日基地的主要部隊為西部航空方面隊，其下轄的{{Link-ja|航空自衛隊西部航空警戒管制團|西部航空警戒管制団|西部航空警戒管制團}}、{{Link-ja|航空自衛隊第2高射群|第2高射群|第2高射群}}及{{Link-ja|航空自衛隊音樂隊|音楽隊 (航空自衛隊)|音樂隊}}亦駐於該基地。而基地所駐的飛行部隊，如西部航空方面隊司令部支援飛行隊及{{Link-ja|航空自衛隊航空救難團|航空救難団|航空救難團}}下屬的春日直升機空運隊（{{lang|ja|春日ヘリコプター空輸隊}}）<ref name="航空救難團">{{Cite web |language=ja |script-title=ja:航空救難団の組織と部隊配置 |publisher={{lang|ja|{{Link-ja|航空自衛隊航空救難團|航空救難団|航空救難団}}}} |url=http://www.mod.go.jp/asdf/arw/kyuunanndannnituite/sosikiindex.html |accessdate=2017-09-03 |archiveurl=http://archive.is/xDrSE |archivedate= 2017-09-03}}</ref>，則部署在板付飛行場地區<ref name="部隊介紹">{{Cite web |language=ja |script-title=ja:部隊紹介 |publisher={{lang|ja|{{Link-ja|航空自衛隊西部航空方面隊|西部航空方面隊|西部航空方面隊}}}} |url=http://www.mod.go.jp/asdf/wadf/assigned/index.html |accessdate=2017-09-03 |archiveurl=http://archive.is/tYeJL |archivedate= 2017-09-03}}</ref><ref name="手嶋文史"/>，當地還併設[[海上保安廳|海上保安廳]][[第七管區海上保安本部|第七管區海上保安本部]]的福岡航空基地，同時駐有{{Link-ja|福岡縣警察|福岡県警察}}{{Link-en|警用航空|Police aviation|飛行隊}}和{{Link-ja|福岡市消防局|福岡市消防局}}[[空中消防|直升機隊]]<ref name="米澤敬一"/>。
