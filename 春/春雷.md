'''春雷'''是[[四川雷氏|四川雷氏]]第一代雷威所斫的一床[[唐代|唐代]][[古琴|古琴]]，現藏於[[國立故宮博物院|國立故宮博物院]]。此琴屬[[古琴琴式列表|連珠式]]，长{{convert|126|cm}}、肩寬{{convert|22.1|cm}}、尾寬{{convert|17.2|cm}}。琴面髤黑漆、有斷紋。琴徽、琴軫和琴足全部都是玉製的，七顆琴軫上全部刻有[[篆字|篆字]]。龍池為圓形，鳳沼為長方形。龍池上方為[[草書|草書]]綠字「春雷」，左右以[[隸書|隸書]]分刻「其聲沈以雄，其韻和以沖」、「誰其識之出爨中」；龍池下有一方印，但印文已經剝蝕不清。<ref name="國立故宮">{{Cite web |url= http://www.npm.gov.tw/zh-tw/Article.aspx?sNo=04001029|title=观琴至美——传世名琴鉴赏|accessdate=2015-01-19|work= |publisher=[[國立故宮博物院|國立故宮博物院]]|date=2012-04-25}}</ref>

==歷史==
[[北宋|北宋]]末年[[宋徽宗|宋徽宗]]在[[宣和内府|宣和内府]]建造了“万琴堂”，春雷為他收藏的眾多古琴中的第一品。[[金朝|金]]滅宋後春雷也被當做戰利品運到了[[金中都|中都]]，藏於承华殿。[[金章宗|金章宗]]逝世時以此琴陪葬，但十八年後其墳被掘、春雷被[[元朝|元朝]]收入宫中，後來春雷被賞賜給會彈奏古琴的[[中書令|中書令]][[耶律楚材|耶律楚材]]，耶律楚材將春雷贈給自己的老师[[万松野老|万松野老]]。此後春雷一度复归耶律楚材之子[[耶律铸|耶律铸]]。它後來有跡可循的主人為[[赵德润|赵德润]]。[[明朝|明朝]]時春雷传入明宫，[[清朝|清朝]]時是[[裕亲王|裕亲王]]的藏品。<ref name="古曲網">{{Cite web |url= http://info.guqu.net/guqinzhishi/20060707233522_172.html|title=春雷琴·连珠式古琴|accessdate=2015-01-19|work= |publisher=中國古曲網|date=2013-09-28}}</ref> 民國時陸續由[[何冠五|何冠五]]、[[汪景吾|汪景吾]]、[[張大千|張大千]]等人收藏。<ref name="國立故宮"/>
