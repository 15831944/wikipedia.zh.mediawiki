'''春木場站'''（{{jpn|j=春木場駅|hg=はるきばえき|rm=Harukiba eki|f=y}}）是一座位於[[岩手縣|岩手縣]][[岩手郡|岩手郡]][[雫石町|雫石町]]的[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[田澤湖線|田澤湖線]]的鐵路車站。

== 車站構造 ==
*[[側式月台|側式月台]]1面1線的[[地面車站|地面車站]]。不設站舍，但月台上設有候車室。[[雫石站|雫石站]]管理的[[無人站|無人站]]。

== 历史 ==
* 1964年（昭和39年）9月10日：車站啟用。
* 1987年（昭和62年）4月1日：因[[國鐵分割民營化|國鐵分割民營化]]，本站成為[[東日本旅客鐵道|東日本旅客鐵道]]的車站。

== 車站周邊 ==

* 雫石町立御明神小學
