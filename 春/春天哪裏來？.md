《'''春天哪裏來？'''》（{{jpn|j=春はどこから来るのか?|hg=はるはどこからくるのか|rm=Haru wa doko kara kuru no ka|f=y}}）是日本女子[[日本偶像|偶像]]組合[[NGT48|NGT48]]的第3張[[單曲|單曲]]，於2018年4月11日由{{Link-ja|Ariola Japan|アリオラジャパン}}發行。同名[[A面與B面|標題曲]]由[[本間日陽|本間日陽]]擔任[[AKB48組合構成#中心位置|Center]]（中心位置成員），為[[日本電視台|日本電視台]]節目《{{Link-ja|BUZZ RHYTHM|バズリズム|BUZZ RHYTHM02}}》的4月份片頭曲、《{{Link-ja|我們這麼吵鬧真是抱歉|ウチのガヤがすみません!}}》的4月份結尾曲、以及[[鬱金香電視台|鬱金香電視台]]節目《Kito kyun♥旅人with T》的4月份結尾曲<ref name="NGT48新曲のテレビタイアップ続々決定、ファンとのチーム対抗ボウリング大会も">{{cite web |url=https://natalie.mu/music/news/275221 |language=ja |script-title=ja:NGT48新曲のテレビタイアップ続々決定、ファンとのチーム対抗ボウリング大会も |date=2018-03-26 |publisher=[[natalie_(網站)|natalie]] |accessdate=2018-04-18 |deadurl=no |archiveurl=https://web.archive.org/web/20180326202949/https://natalie.mu/music/news/275221 |archivedate=2018-03-26 }}</ref>。

== 概要 ==
單曲分為Type-A、Type-B、Type-C和NGT48 CD盘，總共4個版本。其中，《春天哪裏來？》和《蒸發了的水份》收錄於所有版本，《春天哪裏來？》的音樂錄影帶則收錄於NGT48 CD盤以外的所有版本。NGT48 CD盤收錄了《反省梳打水》；《為了我》及其音樂錄影帶、影片「祝畢業 NGT48初代隊長 北原里英 〜成為了新潟的女人。〜」收錄於Type-A；《之後吧》及其音樂錄影帶、影片「NGT48劇場開張兩週年特別活動 挑戰世界紀錄！」收錄於Type-B；《Whatcha Gonna Do》及其音樂錄影帶、影片「NGT48研究生「亲密的称呼」紀錄片映像」收錄於Type-C。NGT48 CD盘以外所有版本均有全國[[握手會|握手會]]參加券和隨機[[生寫真|生寫真]]一張<ref>{{cite web |url=http://www.sonymusic.co.jp/artist/ngt48/discography/BVCL-875 |language=ja |script-title=ja:春はどこから来るのか？【Type-A 初回仕様（CD＋DVD）】 |publisher=[[NGT48|NGT48]] |accessdate=2018-04-18 |deadurl=no |archiveurl=https://web.archive.org/web/20180418203701/http://www.sonymusic.co.jp/artist/ngt48/discography/BVCL-875 |archivedate=2018-04-18 }}</ref><ref>{{cite web |url=http://www.sonymusic.co.jp/artist/ngt48/discography/BVCL-877 |language=ja |script-title=ja:春はどこから来るのか？【Type-B 初回仕様（CD＋DVD）】 |publisher=[[NGT48|NGT48]] |accessdate=2018-04-18 |deadurl=no |archiveurl=https://web.archive.org/web/20180222043927/http://www.sonymusic.co.jp/artist/ngt48/discography/BVCL-877 |archivedate=2018-02-22 }}</ref><ref>{{cite web |url=http://www.sonymusic.co.jp/artist/ngt48/discography/BVCL-879 |language=ja |script-title=ja:春はどこから来るのか？【Type-C 初回仕様（CD＋DVD）】 |publisher=[[NGT48|NGT48]] |accessdate=2018-04-18 |deadurl=no |archiveurl=https://web.archive.org/web/20180418203701/http://www.sonymusic.co.jp/artist/ngt48/discography/BVCL-879 |archivedate=2018-04-18 }}</ref><ref>{{cite web |url=http://www.sonymusic.co.jp/artist/ngt48/discography/BVCL-881 |language=ja |script-title=ja:春はどこから来るのか？【NGT48 CD盤（CD Only）】 |publisher=[[NGT48|NGT48]] |accessdate=2018-04-18 |deadurl=no |archiveurl=https://web.archive.org/web/20180418203701/http://www.sonymusic.co.jp/artist/ngt48/discography/BVCL-881 |archivedate=2018-04-18 }}</ref>。
