== 故事簡介 ==
椎名亞樹為了讓自己更有男子氣概，藉著升上中學的機會前往東京生活並入住學生宿舍「春原莊」。而在那裡等著他的，是帶有母性光輝的管理員大姐姐與三位學姐。

== 登場人物 ==
※[[聲優|聲優]]為廣播劇、紀念PV版及電視動畫版<ref>{{Cite web|url=https://twitter.com/ichijinsha_info/status/755368187210805249|script-title=ja:7/22発売／ねこうめ先生『すのはら荘の管理人さん』特装版ドラマCDサンプル（ドラマパート）をUPしました!!|date=2016-07-19|website=一迅社宣傳課Twitter|accessdate=2017-11-22|language=ja}}</ref><ref>{{Cite web|url=https://twitter.com/ichijinsha_info/status/931131532156190722|script-title=ja:11月22日発売の、ねこうめ先生『すのはら荘の管理人さん』3巻の発売告知PVを作成しました！|date=2017-11-16|website=一迅社宣傳課Twitter|accessdate=2017-11-22|language=ja}}</ref><ref>{{Cite web|url=http://sunoharasou-anime.com/character/matsuri.php|title=CHARACTER|accessdate=2018-08-19|language=ja}}</ref>。
; {{nihongo|春原彩花|春原 彩花（すのはら あやか）|聲：[[佐藤利奈|佐藤利奈]]}}
: 春原莊的管理員。胸部大而柔软，家务全能为人亲切，但却不把椎名亚树当男人看待，经常用自己的身体贴着椎名亚树使其困扰。
: 酒量很差，一喝醉就会变成接吻狂魔，管理员的工作是从祖母那里继承下来的。
; {{nihongo|椎名亞樹|椎名 亜樹（しいな あき）|聲：[[喜多村英梨|喜多村英梨]]}}
: 外表不像男性的少年，为了使自己看起来更男孩子气而从[[静冈县|静冈]]来到东京入读棚町中学。
: 入住春原庄后才发现除了自己周围都是女孩子，而自己则很少被当做异性看待，[[青春期|青春期]]的袭扰也让他很困惑。
: 喜欢可爱的东西，但因不能表达出来只能两眼巴巴。对自己有没有姐姐这个问题十分敏感。
; {{nihongo|雪本柚子|雪本 柚子（ゆきもと ゆず）|聲：[[高森奈津美|高森奈津美]]}}
: 春原莊住戶，棚町中學學生會長。身材矮小，为了不使自己身材矮小被发现而带着金属发箍。
: 成绩优秀，追随者众多，也是春原庄里唯一将亚树当男孩子看的人，不擅长谈论恋爱和恐怖话题。
; {{nihongo|月見里堇|月見里 菫（やまなし すみれ）|聲：[[瀨戶麻沙美|瀨戶麻沙美]]}}
: 春原莊住戶，棚町中學學生會副會長。身材高挑性格高冷，对小巧可爱的东西完全没有抵抗力（亦即也喜欢雪本柚子）。
: 也不把亚树当男孩子看，上厕所时喜欢拉着亚树一起去，并且会时不时教给他一些人生的经验。
; {{nihongo|風見百合|風見 ゆり（かざみ ゆり）|聲：[[宮本侑芽|宮本侑芽]]}}
: 春原莊住戶，棚町中學學生會書記。虽然总是满面笑容，但看起来像是在秘密策划着什么。
: 在看到柚子和堇在一起时总会妄想。
: 擅长服装制作，曾想让亚树强行穿女装，当然也不把亚树当作男生看待。
; {{nihongo|春原菜菜|春原 菜々（すのはら なな）|聲：[[佐倉綾音|佐倉綾音]]}}
: 彩花的妹妹，高中生。春原庄曾经的住户，和姐姐彩花一样有着高挑的身材，但皮肤是古铜色。初中时做过学生会长，成绩优秀。
: 喜欢捉弄亚树，尽管因此与姐姐形成了竞争，但姐妹关系依旧很好。
; {{nihongo|椎名茉莉|椎名 茉莉（しいな まつり）|聲：[[茅野愛衣|茅野愛衣]]}}
: 亞樹的姐姐。像妹妹一样宠爱亚树，已经达到过度保护级别，内心很孩子气。
: 路痴，曾在找到春原庄后找不到回家的路。
: 来到春原庄的目的是把亚树带回去，却在“谁能更好地照顾亚树”上输给了春原彩花。
: 转校到了春原菜菜所在的高中。
; {{nihongo|小園井舞子|小薗井 舞子（おそのい まいこ）|聲：[[中岛爱|中岛爱]]}}
:與菜菜同個高中的朋友。
; {{nihongo|內藤莓愛|内藤 苺愛（うちふじ めあ）|聲：[[長妻樹里|長妻樹里]]}}
: 與菜菜同個高中的朋友。
; {{nihongo|八穗錦|八穂 錦（やつほ にしき）|聲：[[伊藤靜|伊藤靜]]}}
: 八穗酒店的老闆。
