'''春川文化广播'''（[[朝鲜语|朝鲜语]]：춘천문화방송，[[朝鲜汉字|朝鲜汉字]]：{{lang|ko|春川文化放送}}；[[英语|英语]]：Chuncheon Munhwa Broadcasting Corporation）是位于[[韩国|韩国]][[江原道_(南)|江原道]]的[[文化广播_(韩国)|文化广播]]（MBC）的放送局。

== 所在地 ==
*200-718[[江原道_(南)|江原道]][[春川市|春川市]]水边公园街54

== 沿革 ==
*1968年4月6日 - '''江原道广播公司'''成立。
*1968年7月13日 - 使用1430kHz播音（呼号'''HLAN'''），发射功率10kW。
*1971年9月16日 - 改为现在名称。
*1972年11月10日 - 搬迁至江原日报社内。
*1978年11月23日 - 更改至774kHz播音。
*1983年1月20日 - 开通电视节目（CH 10，1kW）。
*1983年7月13日 - 开通[[MBC_FM4U|MBC FM4U]]节目（94.9MHz，1kW）。
*1983年11月11日 - 开始搬入至新的办公楼中。
*1985年6月23日 - 与[[日本|日本]][[福井电视台|福井电视台]]结为友谊关系。
*1993年12月10日 - 与[[中国|中国]][[杭州电视台|杭州电视台]]结为友谊关系。
*1994年8月30日 - 新东发射站建成。
*2000年1月5日 - [[MBC_FM4U|MBC FM4U]]改为至94.5MHz。
*2005年6月27日 - 大龍山发射台落成。
*2006年9月1日 - 大岩山发射台落成（MBC标准FM：88.9MHz，MBC FM4U：98.3MHz）。
*2007年12月21日 - 开通[[DMB|DMB]]。
*2012年10月25日 - 模拟电视停播。
