'''春珠里'''是[[中華民國|中華民國]]台灣[[嘉義縣|嘉義縣]][[太保市|太保市]]的行政區，位於太保市南方，西南方接茄苳腳，東南邊為[[梅埔里|梅埔里]]，東北方為[[後潭里|後潭里]]。

面積約4.589755平方公里，行政區包含12個鄰、439戶、人口數為1012人。

== 地理 ==
春珠里是個地勢平坦，交通便利的地方，居民多務農。位於太保市南方，西南方接茄苳腳，東南邊為梅埔里，東北方為後潭里<ref name=":0">{{Cite web|url=http://gn.moi.gov.tw/geonames/GNMap/map_Admin/map/MapMain.aspx#|title=地名資訊服務網|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。

== 聚落歷史 ==
春珠里是由「春珠」和「茄苳腳」合成，現以李、蔡為最大姓。在清朝時「茄苳腳」人口較多，達千戶，之後漸漸衰落，因嘉義人為爭取設行政於此的機會，在「秤土為憑」時，土裡摻鹽，贏走了轄權，傳說有人破壞地方風水，造成村內少壯丁，女兒紛紛外嫁，多缺人主祀的「神主牌」，於是人口更為凋零。

而「春珠」則是明、[[鄭成功|鄭成功]]來台開墾時，部將李元興由台南移墾至此，另外蔡姓人家也聚住本莊<ref name=":0" /><ref>{{Cite web|url=http://taibao.cyhg.gov.tw/cp.aspx?n=68AF3B64ECE864A9&s=ABBF62618F53F8DE|title=嘉義縣太保市春珠里辦公室|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。

== 教育 ==
本區域內目前無設立國中小，國小學區為[[太保國中|太保國小]]；國中學區為[[嘉義縣立太保國民中學|太保國中]]<ref>{{Cite web|url=http://edu.cyhg.gov.tw/News4.aspx?n=8B04CBD4471A6D45&sms=745127AC0E5C0036|title=嘉義縣政府教育處|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。

== 文化資源/文化資產 ==
春珠里有兩座道教的民間寺廟：清福宮、福春宮

清福宮：清福宮為一所道教的民間寺廟，主神為清水祖師，陪祀神明為土地公、註生娘娘，建立於1752年<ref>{{Cite web|url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/taibau/1001009-QFG|title=文化資源地理資訊系統 清福宮|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。

福春宮：福春宮為一所道教的民間寺廟，主神為池府千歲 城隍夫人，陪祀神明為土地公、註生娘娘，建立於2008年<ref>{{Cite web|url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/taibau/1001006-FCG|title=文化資源地理資訊系統 福春宮|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。

== 景點 ==
品皇咖啡觀光工廠是台灣少數以「咖啡」為展覽主題的觀光工廠，提供販售咖啡、奶茶、餅乾，也提供多種咖啡試喝<ref>{{Cite web|url=http://travel.network.com.tw/tourguide/point/showpage/104966.html|title=旅遊資訊王|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。

== 其他特色 ==
推動社區農園

春珠里於2016年推動社區農園，社區志工及長輩協力整頓，將原本是社區廟宇佈滿石粒之閒置空地，經由將其營造成社區面積大約20-30坪的自然農園，提供居民可以栽種蔬果的空間，後續收成也可成為在地化的低碳飲食<ref>{{Cite web|url=https://lcss.epa.gov.tw/LcssViewPage/Responsive/AreaDoc.aspx?CityID=10010&DistrictId=1001001&VillageID=1001001006&ActDocId=09d7f8e4-f0bb-4d6d-8bdd-6497de9bc342|title=低碳永續家園資訊網|accessdate=2017-12-17|author=|date=|publisher=}}</ref>。
