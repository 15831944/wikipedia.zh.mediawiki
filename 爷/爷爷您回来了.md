「'''爷爷您回来了'''」是指2005年前台灣的[[中國國民黨|中國國民黨]]主席[[连战|连战]][[2005年中国国民党和平之旅|访问中国大陸]]期间，由[[西安|西安]][[后宰门小学_(西安)|后宰门小学]]演出的一组诗歌朗诵与舞蹈节目，因第一句话为“爷爷，您回来啦！您终于回来啦！”而闻名。

此節目在[[現場直播|現場直播]]後引起[[華人|華人]]地區的軒然大波，因為表演形式、用詞和語調與[[文化大革命|文化大革命]]之[[樣板戲|樣板戲]]類似。此外，由于“爷爷，您回来了”知名度很高，在节目播出后不久，台灣[[电信|电信]]業者即推出改版的手机铃声。2005年12月20日，中國《[[新周刊|新周刊]]》將此選為2005年度經典語錄之一。<ref>[http://magazine.sina.com/bg/neweekly/217/20060110/213514873.html 2005 國內外經典語錄_新周刊_雜誌頻道_新浪網-北美]</ref>

== 事件经过 ==
节目中朗诵的诗歌原名《'''陽光、校園、童年'''》，是2005年4月30日時當時仍任[[中国国民党|中国国民党]]黨主席的[[连战|连战]]在[[2005年中国国民党和平之旅|中国大陸访问]]過程中、於重返位於[[西安|西安]]的母校[[后宰门小学_(西安)|后宰门小学]]时，六名小学生为迎接他而表演的一个诗朗诵节目，第一句即是「爷爷，您回来啦！您终于回来啦！」。除高八度的童音朗诵外，还带有较夸张的身体动作表演这种表达方式。

事前，媒体曾探问后宰门小学的一位教师有什么样的欢迎节目，得到“这是秘密”的答案，校长白彩玲则透露“校方将筹备一项以和平为主题的校庆活动。”<ref>[http://news.sina.com.cn/w/2005-04-24/08535735792s.shtml 连战访西安时将拜祭祖母墓访母校 西安紧急筹备]</ref>。事後，后宰门小学對事件及後續風波沒有評論及動作，{{Fact|不过在中国[[中央电视台|中央电视台]]和[[东森电视台|东森电视台]]合作的“[[岩松看台湾|岩松看台湾]]”节目中，在[[东森电视台|东森电视台]]演播室裡，中国主持人[[白岩松|白岩松]]和台湾主持人曾笑着谈论此事。|time=2013-09-05T02:48:26+00:00}}

== 反应 ==
*连战本人與其子連勝文在当场欣赏演出时，做出微笑、同时强忍住笑出声来的姿态。
*{{Fact|其后即将造访中国大陸的[[亲民党|亲民党]]向中共提出，在亲民党访期间，希望不要安排类似的节目，以防同类事件再度发生。|time=2013-09-05T02:33:40+00:00}}
*台灣学者[[龙应台|龙应台]]则专门撰文，将此事件与台湾[[戒严|戒严]]期间的同类事件进行比较，认为生于中国大陸现时代的这些小学生值得怜悯。<ref>[http://forums.chinatimes.com/report/lonin/94051301.htm 中時電子報新聞專輯：野火再燃 龍應台城市文化效應]</ref>

== 改编 ==

在原版节目推出后不久，[[台湾|台湾]]电信业者即推出「爷爷，你回来了」来电答铃，除原版之外，还有诸如“家里的米缸没米了”之类的搞笑版本。有人认为这是属于[[恶搞文化|恶搞文化]]的一种。这些铃声红遍台湾，下载量一度高居榜首，許多商業廣告也以這首詩為廣告創意。

“爷爷，你回来了”这种句式亦在互联网热传，“AA，你BB了，你终于BB了”<small>（AA可代入任何一个人物、BB則可代入任何一種动作）</small>的句式，在2005年[[中国大陸|中国大陸]]、[[台湾|台湾]]的[[网络|网络]]环境中都屡见不鲜。

台灣[[中天電視|中天電視]]《[[全民大悶鍋|全民大悶鍋]]》綜藝節目內亦有以此為題材的小單元<ref>{{YouTube|sLa_mcfpxHk|全民大悶鍋 - 連爺爺(台灣綜藝)}}{{zh-tw}}</ref>。

在台灣，國際連鎖快餐店[[肯德基|肯德基]]也製作了模仿「爷爷，你回来了」的電視廣告；不過講的是快餐店的象徵人物[[桑德斯上校|桑德斯上校]]，從[[墨西哥|墨西哥]]帶着辣椒和辣醬回到台灣，而電視片中的兒童更是以[[臺灣話|台語]]表演{{ref|1}}。
