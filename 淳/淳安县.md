'''淳安县'''位于[[中国|中国]][[浙江省|浙江省]]西部，是[[杭州市|杭州市]]下辖的一个县。

== 名称 ==
淳安县于宋代取“淳而易安”之义而得名。

== 沿革 ==
[[东汉|东汉]]建安十三年（208年）分歙县东之叶乡置始新县；[[隋|隋]]开皇九年（589年）改为新安县；[[唐|唐]]永贞元年（805年）改为青溪县；[[北宋|北宋]]宣和三年（1121年）改为淳安县。元时，淳安、遂安属建德路；民国3年（1914年）属金华道；1955年划归[[建德|建德]]专署。1958年撤建德专属，淳安属金华专属。1963年划属杭州市。

== 地理 ==

淳安县位于钱塘江支流[[新安江|新安江]]中游，东西96.8公里，南北94.4公里，是浙江省面积最大的县。

淳安县地势四面多山，中间为丘陵，略呈盆地状。主要山脉有[[白际山脉|白际山脉]]、[[千里岗山脉|千里岗山脉]]、[[昱岭山脉|昱岭山脉]]。最高点为千里岗山脉主峰[[磨心尖|磨心尖]]，海拔1523米。

淳安县境内的主要河流有[[新安江|新安江]]，最大的湖泊为[[千岛湖|千岛湖]]。新安江全长293公里，淳安境内68公里。

1961年至1980年间，淳安县年平均温度为17摄氏度，1月份平均温度5.0摄氏度，7月平均温度28.9摄氏度。年平均降雨量1430毫米，年平均日照时数为1951小时。

== 行政区划 ==
下辖11个镇、12个乡，8个社区、11个居民区、425个行政村：
*镇：千岛湖镇、汾口镇、石林镇、大墅镇、姜家镇、梓桐镇、中洲镇、文昌镇、枫树岭镇、临岐镇、威坪镇。
*乡：里商乡、屏门乡、浪川乡、安阳乡、王阜乡、瑶山乡、宋村乡、鸠坑乡、左口乡、金峰乡、富文乡、界首乡。

== 交通 ==
全县[[公路|公路]]里程2347.8公里，其中[[高速公路|高速公路]]里程13.6公里，[[省道|省道]]里程56.9公里，县道里程796.8公里。

== 矿产资源 ==
淳安县主要矿产资源有[[褐铁矿|褐铁矿]]、[[黄铁矿|黄铁矿]]、[[铅锌矿|铅锌矿]]等。
