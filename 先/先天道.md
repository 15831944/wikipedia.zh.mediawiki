'''先天道'''，亦称'''先天教'''、'''先天門'''、'''先天玄关大道'''等，属于[[齋教|齋教]]類的[[新兴宗教|新兴宗教]]，在[[中国大陆|中国大陆]]、[[香港|香港]]、[[臺灣|臺灣]]、[[泰國|泰國]]、[[越南|越南]]、[[馬來西亞|馬來西亞]]與[[新加坡|新加坡]]廣泛流傳。

== 歷史 ==
先天道於[[清朝|清朝]][[順治|順治]]年間創立於[[江西|江西]]，創教人為黃德輝，但黃本人自稱九祖，以開創[[羅教|羅教]]（無為教），同時也被[[龍華教|龍華教]]奉為祖師的[[羅思孚|羅思孚]]為師。早年先天道只在江西一帶活動，[[道光|道光]]初年開始傳至[[華南|華南]]、西南各省，並於清朝[[咸豐|咸豐]]年間由李昌晉、黃昌成二人傳入台灣，光緒年間由[[田邵邨|田邵邨]]傳入[[香港|香港]]。

先天道最初為秘密宗教，在[[清朝|清朝]]官方文書中，又稱為「'''齋匪'''」、「'''金丹道'''」<ref>[http://www.senwanture.com/religion/religion-fo%20dgiau-zai%20dgiau.%20sen%20ten%20pai.htm 齋教先天派]</ref>。

在[[中国大陆|中国大陆]]，1950年代初，[[中华人民共和国|中华人民共和国]]政府在[[取缔反动会道门|取缔反动会道门]]运动中，将先天道作为反动会道门取缔。<ref>[http://culture.cz001.com.cn/changzhou/2009/0806/176014_2.shtml 取缔反动会道门，中国常州网，2009-08-06]</ref>[[改革开放|改革开放]]后，先天道在不少省份加入了佛教协会，也有一部分加入道教协会。先天道由此得以在中国大陆继续活动。<ref>[http://www.pacilution.com/ShowArticle.asp?ArticleID=4867 孔德继、吴如加，大陆民间宗教管理变局，普世社会科学研究网，2014-04-04] {{webarchive|url=https://web.archive.org/web/20160304070046/http://www.pacilution.com/ShowArticle.asp?ArticleID=4867 |date=2016-03-04 }}</ref>

在[[台灣日治時期|台灣日治時期]]，為了管理方便，[[台灣總督府|台灣總督府]]將[[龍華教|龍華教]]、先天道、[[金幢教|金幢教]]一同列入[[齋教|齋教]]管理，不過在1915年[[西來庵事件|西來庵事件]]發生後，遭到壓制。在[[臺灣戰後時期|臺灣戰後時期]]，仍受[[國民政府|國民政府]]打壓。在台灣解嚴後，宗教風氣較自由而有復甦趨勢。

在[[香港|香港]]，部分齋堂歸屬於[[香港道教聯合會|香港道教聯合會]]管理，據統計，登計在聯合會下的道堂約有四分之一是屬於先天道體系<ref>黎志添、游子安、吳真，香港道教：歷史源流及其現代轉型，香港：中華書局，2010年，第89页</ref>。

== 源流 ==
先天道祖脈源流和[[龍華教|龍華教]]有些許不同。先天道直接視[[禪宗|禪宗]]諸大德為祖，以[[達摩祖師|達摩祖師]]為初祖，二祖[[神光|神光]]，三祖[[普庵印肅|普庵]]（[[鑒智|鑒智]]），四祖[[曹洞|曹洞]]（[[道信|道信]]），五祖[[弘忍|弘忍]]，六祖[[惠能|惠能]]，七祖白衣（[[南嶽懷讓|南嶽懷讓]]以及[[馬道一|馬道一]]或者[[白玉蟾|白玉蟾]]以及馬端陽），八祖羅蔚群 ({{tsl|ja|羅公遠|羅公遠}}的化身，據推測即為[[羅思孚|羅思孚]]或羅維行），九祖為創教人黃德輝（推測為黃廷臣），十祖吳紫祥，十一祖何若（了苦），十二祖袁志謙，十三祖楊還虛（守一）、徐還無（抱一）並稱。十四祖彭依法（水祖），十五祖林依秘（金祖），由於十五祖之後教派分裂眾多，今日先天道於十五祖後不再記有任何祖師。

先天道有「萬全堂」、「乾元堂」兩大系統，前者傳自[[四川|四川]]，後者源自[[江南|江南]]。乾元堂系統為真一祖黃道明所立，在[[光緒|光緒]]年間，由陳運榮傳至[[台灣北部|台灣北部]]。

同善社、[[青蓮教|青蓮教]]<ref>[http://www.xiao-en.org/cultural/magazine.asp?cat=34&loc=zh-cn&id=1862 青莲教下南洋：马来亚最早期的瑶池信仰 ] {{webarchive|url=https://web.archive.org/web/20141007100013/http://www.xiao-en.org/cultural/magazine.asp?cat=34&loc=zh-cn&id=1862 |date=2014-10-07 }}</ref>、歸根道、圓明聖道、[[圓玄學院|圓玄學院]]都是從先天道分支的[[新興宗教|新興宗教]]。

== 教義 ==
先天道主張[[三教合一|三教合一]]，但教義是以[[羅教|羅教]]為主幹，混合[[佛教|佛教]][[禪宗|禪宗]]和[[道教|道教]][[內丹|內丹]]思想。該教認為，無生老母是一切生命萬物的起源，老母從「無」生「有」，「有」幻化萬物，包括人類。當人類從「無」而生時，每個人都有與生俱來的「性」與「命」，還擁有三寶—「神」、「氣」與「精」。當一個人逐漸成長且成熟時，由於受到各種不正當慾望的干擾，而導致思想雜亂與沉溺於物質享受，而陷入外在的世界，「性」與「命」的力量就會分開，三寶也會逐漸削弱。為了讓一個人保持樸實無華的本性，就必須利用佛教的對外修行和道教內丹術對內的修煉，使人類回到出生前—「先天」—充滿潛力 和生命力的階段。

== 教階 ==
依職級高低分為「道」-十地祖師、「運」-發恩備夷、「永」-保恩師、「昌」-引恩師、「明」-證恩師、天恩師、齋友、德者、誠意眾生。先天道禁止[[神職人員|神職人員]]婚嫁，並主張絕對素食，戒律操守嚴格<ref>[http://cbs.ntu.edu.tw/dict/fk/data/%25E5%2585%2588%25E5%25A4%25A9%25E6%25B4%25BE.html 佛光大辭典 先天派]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
