'''先知'''（{{lang-en|Prophet}}）是一个在許多领域，尤其是多种[[宗教|宗教]]中常用的概念，指能够与[[神|神]]交流并预见未来的人。在不同宗教的信仰裡，对先知的认同并非全然一致。

== 不同宗教的表述 ==
[[巫覡宗敎|巫覡宗敎]]、[[基督教|基督教]]等宗教都有此概念。先知对未来的描述来自于[[神|神]]，这点和[[占卜|占卜]]不同。[[伊斯兰教|伊斯兰教]]中的'''先知'''多指[[安拉|安拉]]派遣到世界的多位使者，（《[[古兰经|古兰经]]》中列举并提到他们姓名的有廿五位先知，相信历代各个民族都有安拉所派遣给的先知或使者。）其中最后一位派遣给全人类的使者是[[穆罕默德|穆罕默德]]。

[[基督教|基督教]]，[[犹太教|犹太教]]的文献中，也出现了很多有关先知的阐述。虽然其中很多是对与未来的预见，但更多的是借助于先知之口，阐述[[上帝|上帝]]所制定的社会原则和教义原则。基督教中，先知又稱[[先見|先見]]，這更顯出基督教中的先知是更早看見某些未發生的事，而並非每一件事都能預知。

== 亞伯拉罕、摩西 ==
[[犹太教徒|犹太教徒]]以及後來的[[基督徒|基督徒]]與[[穆斯林|穆斯林]]均承认[[亞伯拉罕|亞伯拉罕]]（[[易卜拉欣|易卜拉欣]]）、[[摩西|摩西]]（[[穆萨|穆萨]]）這兩位[[舊約|舊約]]人物为先知；這三種宗教也被稱為[[亞伯拉罕諸教|亞伯拉罕諸教]]。

== 耶穌 ==
支持[[三位一体|三位一体]]的主流[[基督宗教|基督宗教]]教派認為[[耶穌|耶穌]]是上帝的獨生子，不單是先知，而且也是[[神|神]]。[[伊斯蘭教|伊斯蘭教]]承认[[耶穌|耶穌]]（[[爾撒|爾撒]]）與摩西、[[穆罕默德|穆罕默德]]一樣，都是先知，但不是[[神的兒子|神的兒子]]，相當於古代基督教的《[[義子說|義子說]]》。而[[犹太教|犹太教]]就认为[[耶穌|耶穌]]只是凡夫俗子，既非神，亦非先知。
