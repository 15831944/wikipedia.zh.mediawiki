'''瓜达尔港'''是一座位于[[巴基斯坦|巴基斯坦]][[俾路支省|俾路支省]][[瓜达尔|瓜达尔]]市的[[深水港|深水港]]，距離[[喀拉蚩|喀拉蚩]]西方約460公里，鄰近[[荷姆茲海峽|荷姆茲海峽]]，是巴基斯坦通往[[波斯灣|波斯灣]]和[[阿拉伯海|阿拉伯海]]的大門，[[战略|战略]]位置重要。

==历史==
現階段來自[[中华人民共和国|中国]]的投資在瓜达尔港进行了大量的建設。巴基斯坦在1993年時，開始就技術跟財務上評估修建瓜達爾港的可行性，瓜達爾港在2002年3月22日開工建造；2007年[[新加坡國際港务集团|新加坡國際港务集团]]中标來負責港口經營，2010年巴基斯坦高等法院裁决合同中土地转让无效<ref>[http://www.atimes.com/atimes/China_Business/NI05Cb01.html China set to run Gwadar port as Singapore quits]，亞洲時報，2012-09-05</ref>；2012年8月，巴基斯坦政府决定将瓜达尔港的控股权由新加坡转移给一家中国企业。一些中国军事专家认为瓜达尔港可成为中国军舰的补给港<ref>[http://world.huanqiu.com/roll/2012-09/3089368.html 环球网] {{webarchive|url=https://web.archive.org/web/20120903232852/http://world.huanqiu.com/roll/2012-09/3089368.html |date=2012-09-03 }}</ref>。

2013年2月18日，巴基斯坦将瓜达尔港的建设和运营权正式交予中国的中国海外港口控股有限公司<ref>[http://dawn.com/2013/02/19/china-given-contract-to-operate-gwadar-port/ China given contract to operate Gwadar port] - Pakistan - DAWN.COM</ref>。

2013年5月，中國國務院總理[[李克強|李克強]]訪問巴基斯坦，雙方決定建設“[[中國-巴基斯坦經濟走廊|中國-巴基斯坦經濟走廊]]”連接瓜達爾港和中國[[新疆|新疆]]<ref>{{cite news |title = 中巴決建經濟走廊李克強形容兩國友誼是「鐵哥們」 |url = http://paper.wenweipo.com/2013/05/24/CN1305240001.htm |accessdate=2015-04-24 |newspaper = [[香港文匯報|香港文匯報]] |date=2013-05-24 |language=zh}}</ref>。

2015年巴方将把瓜达尔港2000亩土地[[租赁|租赁]]给中方，为期43年，用于建设（瓜达尔港）首个[[经济特区|经济特区]]。<ref >{{cite web|url=http://news.163.com/15/0909/14/B332O90E0001124J.html | language= | title=中国获租巴基斯坦瓜达尔港2000亩土地为期43年 |publisher=中国青年网（北京） |date=2015-09-09 |accessdate= }}</ref>

瓜達爾深水港于2008年正式启用。第一艘在該港口停泊的船隻是一艘來自[[加拿大|加拿大]]的載有52000[[噸|噸]][[小麥|小麥]]的貨輪。小麥于该港口卸貨，並轉運往巴基斯坦其他地區供國內使用。這艘船抵達巴基斯坦時本來載了72,000噸的小麥，不過在[[卡拉奇|卡拉奇]]港停泊時，卸運了20000噸的小麥運往卡拉奇。

== 效益 ==
=== 打開中國西側對外出口 ===
2016年10月29日約50輛卡車組成的「橫貫中巴經濟走廊」商貿車隊從新疆喀什出發，沿著中巴經濟走廊抵達[[俾路支省|俾路支省]]，11月12日抵達瓜達爾港，巴基斯坦政府12日表示，中國車隊到達瓜達爾港是「分水嶺性的事件」。11月13日見證第一艘中國商船開航，這個港口經中巴鐵路連接新疆[[喀什|喀什]]，是中國西部和[[阿拉伯海|阿拉伯海]]之間的貿易聯繫，中國目前繞經[[馬六甲海峽|馬六甲海峽]]的石油運輸距離大幅縮短<ref>{{cite web|url=http://udn.com/news/story/7333/2106412| title=陸投資巴基斯坦港口 開航|publisher=[[經濟日報|經濟日報]] |date=2016年11月15日}}{{zh-hant}}</ref>。

瓜达尔港临近主要石油产地的[[波斯湾|波斯湾]]，对中国的能源战略有重大意义。

中国从中东进口的石油中有80%经过马六甲海峡，通过马六甲的海路运输有12,000公里之遥，而经由瓜达尔港运往新疆的陆路则只有3,000公里。再从[[新疆|新疆]]到中国内地省份也只有3,500公里。

中国在[[南中国海|南中国海]]海域与[[越南|越南]]，[[菲律賓|菲律賓]]等国存在领土争议。瓜达尔港的落成可以使中国绕过[[马六甲海峡|马六甲海峡]]从[[中东|中东]]进口石油，可以降低领土争议对中国的石油安全带来的风险。

=== 促進兩地政經發展 ===
中國西部地區的經濟比較落後，隨著瓜達爾港至新疆喀什的經濟走廊開通，將帶動中國西部地區的經濟發展，意義重大。

另外，[[俾路支省|俾路支省]]是巴基斯坦較貧窮落後的省份，分離主義思想興盛<ref>[http://orientaldaily.on.cc/cnt/china_world/20161115/mobile/odn-20161115-1115_00178_004.html 一帶一路要塞 巴國瓜達爾港開航]</ref>。而瓜達爾港的開發，有助於吸收大量當地就業人口，帶動經濟發展，能使中央與地方政府間形成良性互動，有助於當地政經情勢的穩定<ref>[http://hk.crntt.com/doc/1043/8/6/7/104386719.html?coluid=7&kindid=0&docid=104386719 瓜達爾港自由區能給中國帶來啥]</ref>。
