'''瓜拉納穆國際機場'''（{{lang-id|Bandar Udara Internasional Kualanamu}}，{{airport codes|KNO|WIMM}}）位於[[印尼|印尼]][[北蘇門答臘省|北蘇門答臘省]]瓜拉納穆的國際機場，離[[棉蘭市|棉蘭市]]39公里，以取代棉蘭波洛尼亞國際機場。機場地點前身是[[油椰子|油椰子]]種植場。這機場是繼耶加達蘇加諾－哈達國際機場後印尼第二大機場，期望成為印尼西部和蘇門答臘新的國際轉運中心。

機場於2013年7月25日開幕，取代本身於波洛尼亞國際機場的所有航班和服務。<ref>[http://news.sinchew.com.my/node/313173?tid=2 2013年07月26日- 印尼第二大機場正式啟用- 國際- 星洲网新聞]</ref>

==設施及基建==
瓜拉納穆國際機場是繼耶加達蘇加諾－哈達國際機場後印尼第二大機場，有一個佔地224,298平方米的客運大樓<ref>{{cite web|url=http://waspada.co.id/index.php?option=com_content&view=article&id=202352:ministry-urged-to-permit-land-for-kuala-namu&catid=30:english-news&Itemid=101 |title=Ministry urged to permit land for Kualanamu |publisher=Waspada.co.id |date=2011-06-27 |accessdate=2012-10-24}}</ref>，在2030年時預計能處理旅客五千萬人次。

機場有一條3,750 x 60 m 的跑道。

==接駁交通==
===鐵路===
[[File:Kualanamu_AirportRail.JPG|thumb]]
瓜拉納穆國際機場有鐵路服務與市區連接，服務由 [http://www.railink.co.id/ Railink]提供，這是印尼首個[[機場聯絡軌道系統|機場聯絡軌道系統]]、市區登機服務。<ref>{{cite web |url=http://www.waspada.co.id/index.php?option=com_content&view=article&id=248645:jalur-ka-kuala-namu-segera-beroperasi&catid=77:fokusutama&Itemid=131 |title=Jalur KA Kualanamu segera beroperasi (Indonesian)|date=June 2, 2012}}</ref>鐵路服務可以透過網上或在售票機以信用咭購買。<ref>{{cite web |url=http://www.railink.co.id/cara-beli-bayar |title=CARA BELI / BAYAR |accessdate=June 15, 2014}}</ref>火車在南韓製造。<ref>{{cite web|url = http://www.eturbonews.com/35998/medan-s-new-kuala-namu-international-airport-set-operate-july-25|date = July 8, 2013 |title = Medan's new Kualanamu International Airport set to operate July 25}}</ref>

列車服務時間，從 [[棉蘭車站|棉蘭車站]]出發， 從上午四時至夜晚八時，從機場出發，從上午5:25至晚上9:30，這是市區連接機場最快方法，車程30分鐘，但從機場至棉蘭市中心，或需47分鐘。<ref>{{cite web |url=http://www.railink.co.id/jadwal-kereta |title=Jadwal Baru Kereta Api bandara |accessdate=June 15, 2014}}</ref>

從2014年5月開始，棉蘭至機場之間鐵路雙線化展開，預計車程可以減少10分鐘，<ref>{{cite web |url=http://www.thejakartapost.com/news/2014/05/23/n-sumatra-double-track-railway-work-begins.html |title=N. Sumatra double-track railway work begins |author=Apriadi Gunawan |date=May 23, 2014}}</ref>
