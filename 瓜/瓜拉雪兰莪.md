'''瓜拉雪兰莪'''（{{lang-ms|Kuala Selangor}}）是[[马来西亚|马来西亚]][[雪兰莪州|雪兰莪州]][[瓜拉雪兰莪县|瓜拉雪兰莪县]]的[[巫金|巫金]]和县府，北有[[丹绒加弄|丹绒加弄]]，南有[[武吉罗丹|武吉罗丹]]。瓜拉雪兰莪的地名来自于雪兰莪河
意即『雪兰莪河河口』。<ref>http://www.mdks.gov.my/en/visitors/kuala-selangor-info</ref>

== 产业 ==
如同县内其他地区，这里的居民多数是農民，其中作物以[[油棕|油棕]]最多。渔業方面因沿海一带多为[[红树林|红树林]]，所以渔民多数聚集在離雪兰莪河河口不遠處的巴西不南邦（Pasir Penambang）一帶。

== 交通 ==
瓜拉雪兰莪可通过吉隆坡-瓜拉雪兰莪大道（Latar）抵达。此外，现在兴建中的西海岸大道（West Coast Expressway）也途径瓜雪市区。

== 文化 ==
这里的華人祖先多数來自[[福建|福建]]，所以说[[福建话|福建话]]居多。

== 旅遊景點 ==
[[File:Bienvenida_a_Bukit_Melawati.JPG|缩略图]]
[[File:Bukit_Melawati_-_panoramio_(1).jpg|缩略图]]

在瓜雪的雪兰莪河流域有好几个旅遊景点，例如：世界聞名的萤火虫聚居地和靠近[[马六甲海峡|马六甲海峡]]的老鹰喂食之旅。在巴西不南邦的对岸就是皇家山（{{lang-ms|Bukit Melawati}}），曾是雪兰莪苏丹的居住地，并擁有一些古蹟，例如苏丹的陵墓和斩头台等等。山上有许多猴子，游客可以向当地的摩托车小贩买点面包以喂食猴子。瓜雪以前也是雪兰莪的首府和皇城。皇家山下有瓜雪大自然公园和水族馆。Remis海滩也是主要景点。
