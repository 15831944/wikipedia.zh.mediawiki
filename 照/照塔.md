'''照塔'''，位于[[北京市|北京市]][[房山区|房山区]][[大石窝镇|大石窝镇]]塔照村东的山巅，是[[北京市文物保护单位|北京市文物保护单位]]。

==简介==
照塔建于[[辽朝|辽朝]]。坐北朝南，通高15米，是八角形七级密檐砖塔。当地传说此塔是风水塔。塔基是[[须弥座|须弥座]]，高3米，每面宽度2.2米。中部塔身高2.2米，每面宽度1.75米。塔身正面设有券门，其他各面设有假门和假窗。塔身之上是仿木砖刻额枋檐椽以及砖制一斗三升[[斗拱|斗拱]]。上面是七级叠涩檐。塔尖是攒尖刹<ref>{{Cite web |url =  http://www.bj.chinanews.com/news/2005/2005-12-29/1/8339.html|title = 北京古塔 |author =  |date = 2005-12-29 |publisher = 北京网 |language =  |accessdate =  |quote =  }}</ref>。

1995年，照塔被公布为第五批[[北京市文物保护单位|北京市文物保护单位]]<ref name="北京市第五批文物保护单位">{{cite web|title=第五批文物保护单位|url=http://govfile.beijing.gov.cn/Govfile/front/content/11995037_0.html|accessdate=2017-08-30|deadurl=yes|archiveurl=https://web.archive.org/web/20170831001438/http://govfile.beijing.gov.cn/Govfile/front/content/11995037_0.html|archivedate=2017-08-31}}</ref>。2004年4月到7月，房山区进行了[[周吉祥塔|周吉祥塔]]、[[玉皇塔_(北京)|玉皇塔]]、照塔和[[应公长老寿塔|应公长老寿塔]]四座古塔与[[万佛堂、孔水洞石刻及塔|万佛堂]]修缮工程<ref>{{cite web|title=房山四塔、万佛堂等修缮工程正在进行|url=http://www.huaxia.com/zhwh/whxx/2004/06/84363.html|publisher=华夏经纬网|date= 2004-06-17|accessdate=2017-08-21}}</ref>。
