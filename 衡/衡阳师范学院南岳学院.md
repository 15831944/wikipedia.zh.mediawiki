'''衡阳师范学院南岳学院'''位于[[湖南省|湖南省]][[衡阳市|衡阳市]][[雁峰区|雁峰区]][[黄白路|黄白路]]179号，是[[衡阳师范学院|衡阳师范学院]]的下属分校，为本科大学。<ref name="gfw ">{{cite web|url=http://nyxy.hynu.cn/news/more.asp?typeid=61&borderid=261|title=学院概况|publisher=衡阳师范学院南岳学院|accessdate=2011-05-20|work=|language=zh-cn}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

== 历史 ==
2002年，[[衡阳师范学院|衡阳师范学院]]建立了“衡阳师范学院南岳学院”，属于全日制本科独立学校。<ref name="gfw " />

== 学校院系 ==
衡阳师范学院南岳学院拥有14个系部，35个专业，8个科目。<ref name="gfw " />

=== 科目 ===
经济学、法学、教育学、文学、历史学、理学、工学、管理学<ref name="gfw " />

== 校训 ==
厚德、博学、砺志、笃行<ref name="gfw " />

== 对外合作 ==
衡阳师范学院南岳学院与[[美国|美国]]、[[英国|英国]]、[[韩国|韩国]]、[[香港|香港]]等地区的10多所学校建立了友好合作关系。<ref name="gfw " />

== 荣誉 ==
衡阳师范学院南岳学院曾先后获得“湖南省园林式单位”、“大学生社会实践活动先进单位”等荣誉称号。<ref name="gfw " />
