'''衡山县'''地处[[湖南省|湖南省]]中部偏东，为[[衡阳市|衡阳市]]代管县之一，因境内[[五岳|五岳]]之一的[[衡山|衡山]]而得名。[[县|县]][[行政中心|政府驻地]][[开云镇|开云镇]]。

== 地理 ==
衡山县位于湖南省中部偏东，湘江中游地区。地处东经112o27'-112o57'、北纬26o58'-27o28'之间，属[[亚热带|亚热带]][[季风|季风]]湿润气候。热量充足，雨量充沛，四季分明。地形内高外低，最高山峰[[祝融峰|祝融峰]]在县境中间，地势向西北、东南逐渐降低，形成两个倾斜面，也即“前山”与“后山”地区。地貌以[[丘陵|丘陵]]为主，兼有河溪、平原。境内主要河流有[[湘江|湘江]]、[[涓水|涓水]]、荆陂河、乌江等。

== 历史 ==
衡山有着1700多年的悠久历史，以境内有南岳衡山得名。
* [[秦朝|秦朝]]时属[[长沙|长沙]]郡。 
* [[三国|三国]][[吴|吴]]会稽王太平二年（257年），隶属[[衡阳县|衡阳县]]。
* [[晋惠帝|晋惠帝]][[永熙|永熙]]元年（公元290年），改衡阳县为衡山县，为衡山县名首次在历史上出现。
* [[唐|唐]]属[[衡州|衡州]]，[[宋|宋]]属[[潭州|潭州]]，[[元|元]]属[[潭州路|潭州路]]，后改为[[天临路|天临路]]。 
* [[明|明]][[清|清]]属[[衡州府|衡州府]]。
* [[中华民国|中华民国]]时废府存县，直属湖南省。
* [[中华人民共和国|中华人民共和国]]成立后，衡山县隶属于衡阳市。 
* 1966年，衡山县湘江东岸地区分出组建[[衡东县|衡东县]]。 
* 1984年，南岳镇、南岳乡及东湖、马迹、望峰三个乡镇的10个村分出组建衡阳市[[南岳区|南岳区]]。 

== 行政区划 ==
下辖开云、萱洲、长江、店门、[[白果镇_(衡山县)|白果]]、东湖、[[新桥镇_(衡山县)|新桥]]、马迹8个镇，贺家、永和、福田铺、岭坡、望峰、长青、贯塘、江东、沙泉9个乡，15个居委会、118个居民小组、326个村民委员会、3191个村民小组。

== 经济 ==
2011年，衡山县完成GDP78.58亿元，增长14%；完成财政总收入53733万元，增长32.03%；完成全社会固定资产投资45.18亿元，增长40%；实现规模工业总产值93.97亿元，增长51.35%，增加值25.08亿元，增长25.8%，规模企业达到84家，年内新增7家；完成社会消费品零售总额15.08亿元，增长17.5%；城镇居民可支配、农民人均纯收入分别达15546元、8786元，增长20%、14%
