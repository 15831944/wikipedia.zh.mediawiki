'''衡阳矿冶学院'''<ref name=f12 /><ref>{{Cite book|title=2004年中国高等学校大全|last=教育部发展规划司编|first=|publisher=北京：[[新华出版社|新华出版社]]|year=2004.09|isbn=7-5011-6811-3|location=|pages=764}}</ref><ref name=as>{{Cite book|title=中国高等学校 2001年版|last=中华人民共和国教育部发展规划司编|first=|publisher=北京：[[高等教育出版社|高等教育出版社]]；高等教育电子音像出版社|year=2001.12|isbn=7-900076-97-2|location=|pages=451}}</ref>，又称'''衡阳矿冶工程学院'''<ref name=name500 /><ref name="南华大学">{{Cite web|url=http://dag.usc.edu.cn/info/1013/1006.htm|title=南华大学历史沿革|publisher=[[南华大学_(湖南)|南华大学]]|date=2013-10-30|accessdate=2018-11-26}}</ref>，一度称'''衡阳工学院'''<ref name=name500 />，一所位于[[中國|中國]][[湖南省|湖南省]][[衡阳市|衡阳市]]的矿业为主本科院校。1959年成立，並在[[文化大革命|文化大革命]]爆发后的1969年停办<ref name=as /><ref name=f12>{{Cite book|title=中国高等学校大全 （2005年版） （上册）|last=中华人民共和国教育部发展规划司编|first=|publisher=北京：新华出版社|year=2005.08|isbn=7-5011-7214-5|location=|pages=799}}</ref>。

== 创建 ==
1955年，中华人民共和国国务院成立[[中华人民共和国第三机械工业部_(1955–1958)|第三机械工业部]]，开始着手建设中国核工业。1958年，改为[[中华人民共和国第二机械工业部_(1958-1982)|中华人民共和国第二机械工业部]]。同年第二机械工业部与[[中华人民共和国冶金工业部|冶金工业部]]协商，计划在湖南省创办一所铀矿学院，培养原子能事业人才。1959年2月17日，二机部请冶金部将[[中南矿冶学院|中南矿冶学院]]在1958年为二机部招收的200名新生和矿冶工程系二、三、四年级铀矿冶各专业的238名学生以及该系现有的师资、干部，原则上全部转至在衡阳兴办的学院。同年3月，二机部决定以中南矿冶学院的矿冶工程系为基础，组建'''衡阳矿冶学院'''，由矿冶工程系主任[[魏忠信|魏忠信]]负责建院筹办。二机部先后又调陈健、[[苏鸿伯|苏鸿伯]]、[[彭静山|彭静山]]、[[王中明|王中明]]、王治民、[[管明三|管明三]]、[[计亮年|计亮年]]<ref>{{Cite journal|title=庆祝计亮年院士80华诞专刊|author=毛宗万|url=|journal=中国科学:化学  Scientia Sinica(Chimica)|issue=4|doi=|others=|year=2014|volume=44|page=407-409|pmid=}}</ref>等参加筹建。1959年3月12日，二机部十二局负责领导衡阳矿院的基建工作，基建投资150万元<ref name=fas>{{Cite book|title=中国共产党湖南省主要事业单位组织史资料 1949.10-1995.12|last=中共湖南省组织史资料编纂领导小组办公室|first=|publisher=中共湖南省委组织部|year=2005.04|isbn=|location=|pages=196-197}}</ref>。最初二机部定学校名字为“'''衡阳工学院'''”；3月底，学院筹备处就学院名称问题向二机部反映：衡阳市青草桥黄家湾已有一所“衡阳工学院”，请求更名。同年4月，[[武汉中南设计院|武汉中南设计院]]负责学校设计总体规划任务<ref name=fas />，拟设冶金、采矿、选矿、化学分析、地质、矿山机电等6个专业，定于9月1日开学。5月，二机部派[[张克俭_(1921年)|张克俭]]到学院统一领导，魏忠信参加筹建领导。学校最初计划在长沙选址；但中南矿业公司及二机部系统很多厂矿设在衡阳附近，为便于教学生产，决定在衡阳建校，因此改名为'''衡阳矿冶工程学院'''，又称'''衡阳矿冶学院'''<ref name=asf>{{Cite book|title=南华大学史|last=熊哲琰，张新华主编|first=|publisher=北京：原子能出版社|year=2008.10|isbn=978-7-5022-4257-2|location=|pages=8-15}}</ref>。

== 教学与管理 ==
1959年10月7日，二机部党组决定由张克俭、魏忠信、[[苏鸿伯|苏鸿伯]]、陈健、[[王中明|王中明]]组成学院临时党委，张克俭任书记。10月15日，学院举行开学典礼，10月16日正式上课。当年按6个专业招收新生330人，其中地质专业30人，采矿专业60人，电子专业30人，选矿专业60人，冶金专业60人，分析化学90人，加上中南矿冶学院矿冶工程系转来的二、三、四年级学生488人，共有学生818人<ref>{{Cite book|title=大学理念：认知与践行|last=凌均卫，李忆华，李立生著|first=|publisher=海口：[[海南出版社|海南出版社]]|year=2009.06|isbn=|location=|pages=321}}</ref>。

1960年5月28日，二机部党组决定任命魏忠信为副院长。1960年11月21日，二机部党组任命[[徐其昌|徐其昌]]为副院长，并提议兼任党委副书记，呈报中共湖南省委决定。1961年5月29日，二机部党组决定任命[[严诚|严诚]]为副院长。1962年11月25日，中共湖南省委任命徐其昌为党委副书记，免去其副院长职务。1963年7月，副院长魏忠信调二机部太原第七研究所工作，与此同时，[[徐霁远|徐霁远]]调任学院党委书记，张克俭由党委书记改任院长<ref>{{Cite book|title=中国大学校长名典 下|last=中国大学校长名典委员会|first=|publisher=北京：中国人事出版社|year=1995.10|isbn=7-80076-587-3|location=|pages=454}}</ref>。1959年至1969年，学院共有学生2078人，其中1959年中南矿冶学院矿冶工程系转来学生488人，学院招收学生1590人（1959年招收330人，1960年招收420人，1961年招收210人，1962年未招生，1963年招收210人。1964年招收210人，1965年招收210人，1966年[[文化大革命|文化大革命]]开始以后未再招生）<ref name=name500>{{Cite book|title=中国高校|last=赵亮宏等主编|first=|publisher=北京：[[中国大百科全书出版社|中国大百科全书出版社]]|year=1993.05|isbn=|location=|pages=500}}</ref>。

== 停办 ==
{{See also|文化大革命|长沙文革历史}}
[[文化大革命|文化大革命]]期间，衡阳矿冶学院学生积极参加各类政治运动。1966年8月23日，衡阳矿冶学院[[红卫兵|红卫兵]]上街游行，冲击地、市委机关，这也是衡阳市群众组织把斗争矛头首次指向地、市领导干部。[[中共湖南省委|中共湖南省委]]书记[[张平化|张平化]]于9月24日发表讲话，斥责发生在全省各地的造反行动。随后，湖南造反派在[[首都三司|首都三司]]的支持下，成立了[[湘江风雷|湘江风雷]]，在北京散发大量传单。10月底，衡阳矿冶学院学生[[李瑞林|李瑞林]]率领20人成立“湖南无产阶级文化大革命考察团”抵达长沙，经郑开诚介绍认识[[叶卫东|叶卫东]]，加入[[湘江风雷|湘江风雷]]，并提出到各地发展串联，并成功在1966年11月7日，将中共湖南省委副书记[[万达|万达]]揪到衡阳批斗<ref name=cl />。11月23日，在衡阳矿冶学院体育场批斗张平化。1967年1月29日，学校红卫兵冲击衡阳市公安局。1966年12月16日，学院红卫兵查封《衡阳日报》<ref name=cl />。由于湘江风雷在湖南进行一连串活动，[[中央文革小组|中央文革小组]]在1967年2月4日作出批示，宣布湘江风雷为反革命组织。于此同时11月8日，革命造反有理军衡阳总指挥部成立，矿冶学院学生杨尚兴为司令，矿冶学院文革筹委会主任李至有负责筹备。1967年1月26日，矿院文革委员会、湘江革命造反兵团在衡阳再开夺权大会，并冲击衡阳地市机关；造反派各自之间也存在争夺与对抗<ref name=cl>{{Cite book|title=雁阵惊寒 湖南衡阳“文革”史事|last=应山红主编|first=|publisher=|year=2002.02|isbn=|location=|pages=40-45}}</ref>。

1967年，[[湖南省军区|湖南省军区]]部队参加[[三支两军|三支两军]]，并在同年3月，全面軍事管理衡阳矿冶学院、[[衡阳市第八中学|衡阳市第八中学]]等学校<ref>{{Cite book|title=城南区志 1942-2001|last=城南区志编纂委员会编|first=|publisher=北京：[[团结出版社|团结出版社]]|year=2012.09|isbn=978-7-5126-1313-3|location=|pages=368}}</ref>。1969年底，二机部决定停办衡阳矿冶学院<ref name=f12 />。11月12日，二机部军管会派干部部部长[[耿宏|耿宏]]等八人工作组到院，在全院职工大会上，宣布撤销学院的二军字597号文件。按文件规定六九届、七零届学生不作毕业分配，全部去厂矿劳动<ref name=asf />。干部和教职工全部下放二机部湖北[[五七干校|五七干校]]，工人根据工作需要调往有关单位。11月22日，二机部工作组组长、干部部部长耿宏宣布由徐其昌、徐泽辉、鲁彬、陈德录、蒋长益组成领导小组，党委副书记徐其昌任组长，率领全院干部、教职工去二机部湖北五七干校，编为五大队。[[马子厚|马子厚]]等人组成学院留守处，处理学校撤销后的遗留问题。同时，二机部决定将[[核工业第四勘察设计院|核工业第四勘察设计院]]迁入学院内<ref>{{Cite book|title=湖南年鉴 1986|last=《湖南年鉴》编辑部|first=|publisher=《湖南年鉴》编辑部|year=1987.02|isbn=|location=|pages=528-529}}</ref>。1970年，四院人员陆续从江西等地迁入衡阳<ref name=asf />。

== 恢复 ==
文化大革命结束后，1979年5月28日，第二机械工业部来函[[湖南省人民政府|湖南省人民政府]]：“为培养铀矿冶和其他科学技术人才，经征得你省同意，我部已报请国务院恢复衡阳矿冶学院”<ref name=fun>{{Cite book|title=湖南教育大事记 远古-2000年|last=张学军主编；湖南省教育科学研究院|first=|publisher=长沙：[[岳麓书社|岳麓书社]]|year=2002.08|isbn=7-80665-182-9|location=|pages=450}}</ref>。1980年1月1日，成立“衡阳矿冶学院筹备处”，待遇为地师级<ref name=fun />。1980年4月26日，二机部副部长[[苏华_(1918年)|苏华]]来院视察，并与矿院筹备处和六所领导协商院内房地产的划分问题。1981年12月29日，二机部决定调整筹建方向，改为以培养通用专业人才为主，命名改为[[衡阳工学院|衡阳工学院]]<ref>{{Cite journal|title=与核教育结缘|author=[[郑玉辉|郑玉辉]]，中国核工业经济研究所|url=|journal=中国核工业 China Nuclear Industry|issue=01期|doi=|others=|year=2009年|volume=|page=57-58|pmid=}}</ref>。1984年4月，在北京召开衡阳工学院教学计划审定会，并在同年9月正式招生，隶属于[[中华人民共和国核工业部|中华人民共和国核工业部]]（后为[[中国核工业总公司|中国核工业总公司]]）<ref>{{Cite book|title=中央国家机关所属事业单位大全|last=国家机构编制委员会办公室编|first=|publisher=北京：[[经济科学出版社|经济科学出版社]]|year=1992.04|isbn=7-5058-0460-X|location=|pages=1216}}</ref>。1993年，更名为[[南华大学_(湖南)|中南工学院]]，时任国务院副总理、中央军委副秘书长[[张爱萍|张爱萍]]两度题写校名。学院实行部省共建、部管为主的管理体制。2000年，和原[[衡阳医学院|衡阳医学院]]合并组建[[南华大学_(湖南)|南华大学]]。
