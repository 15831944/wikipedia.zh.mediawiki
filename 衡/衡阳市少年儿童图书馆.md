'''衡阳市少年儿童图书馆'''，位于湖南省衡阳市先锋路49号，为湖南省最早成立的地市级少儿图书馆<ref name=":0">{{Cite web|url=http://www.hyst.org/Article_Show.asp?ArticleID=1838|title=衡阳市少年儿童图书馆简介|accessdate=2018年7月10日|author=衡阳市少年儿童图书馆|date=2011年9月26日|publisher=衡阳市少年儿童图书馆}}</ref>。

== 历史沿革 ==
衡阳市少年儿童图书馆于1984年6月15日成立，1984年12月开馆<ref name=":0" />。2001年，因蒸阳路修建，办公楼被全拆，随后搬去原衡阳市文化局办公<ref>{{Cite news|url=http://hn.rednet.cn/c/2006/04/03/836664.htm|title=公共图书馆：陷入困境的空壳|last=吴|first=通清|date=2006年4月3日|work=红网|accessdate=}}</ref>。2009年，衡阳市图书馆开展维修改造工程，将原老书库改建为少年儿童图书馆的馆舍，少年儿童图书馆得以搬回<ref>{{Cite news|url=http://www.e0734.com/html/2009/1230/48533.html|title=衡阳市图书馆将在春节前开馆“迎宾”（图）|last=何|first=青青|date=2009年12月30日|work=中国衡阳新闻|accessdate=}}</ref>。2013年被文化部评定为国家二级图书馆<ref name=":1">{{Cite web|url=http://wtgxj.hengyang.gov.cn/bsfw/whss/201609/t20160912_567029.html|title=衡阳市少年儿童图书馆|accessdate=2018年7月10日|author=|date=2016年9月12日|publisher=衡阳市文体广电新闻出版局}}</ref>。

== 场馆建设 ==
衡阳市少年儿童图书馆的中心馆馆舍面积2000平方米，拥有馆藏藏书11万册。图书馆内设读者服务中心、低幼读物借阅室、期刊借阅室、电子阅览室、自修室、综合文献借阅室、连环画阅览室、声像资料借阅室、青少年心理咨询室9个对外服务窗口<ref name=":0" />。

== 活动 ==
衡阳市少年儿童图书馆的“全市少年儿童读书”活动和“周末快乐读书”活动纳入到衡阳市全民读书月系列活动。“周末快乐读书”活动被中国文化部评为2015年全国“基层文化志愿服务典型案例”<ref name=":1" />，还得到《图书馆报》的专题采访推广<ref>{{Cite news|url=http://news.163.com/16/0525/16/BNU5H9G100014AEE.html|title=《图书馆报》专题采访推广衡阳市少儿读书品牌项目|author=廖亚楼|date=2016年5月25日|work=红网|accessdate=}}</ref>。
