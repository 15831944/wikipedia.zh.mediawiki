'''衡阳县第一中学'''（{{lang-en|'''No.1 Middle School Hengyang County'''}}）位于[[湖南省|湖南省]][[衡阳市|衡阳市]][[衡阳县|衡阳县]]，为衡阳市[[公立学校|公立学校]]，湖南省示范性中学之一。<ref name="fish " >{{cite web |url=http://www.hyx.gov.cn/xdyzx/link2.htm |title= 湖南省衡阳县第一中学简介|accessdate= 2011年|publisher=衡阳县政府网 |work= |language =zh-cn}}</ref>

== 历史 ==
1945年，“湖南私立光华中学”建立，它是衡阳县第一中学的前身。<ref name="fish " />

[[中华人民共和国|中华人民共和国]]建立后，学校改名为“衡阳县第一中学”。<ref name="fish " />

== 学校文化 ==
校训：诚 智 勇 恒

== 奖项 ==
衡阳县第一中学先后荣获“全面贯彻党的教育方针的模范”、“湖南省文明单位”、“湖南省现代教育技术实验学校”、“湖南省安全文明校园”、“衡阳市绿色学校”、“衡阳市中小学教研教改示范校”等荣誉奖项。<ref name="fish " />
