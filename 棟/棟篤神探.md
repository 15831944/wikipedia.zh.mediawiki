== 結局爭議 ==
莫作棟（黃子華飾）在結局時因追捕一個偷竊的小偷而引至哮喘發作，最後於回程路上昏迷倒地，失去知覺。這個結局令觀眾感到震驚及悲傷，因此在結局播出當晚，香港[[廣播事務管理局|廣播事務管理局]]收到83宗投訴<ref>{{cite web |language =  |publisher = 蘋果日報 |title =《棟篤神探》新結局今晚播 |url = http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20040827&sec_id=462&art_id=4267834 |author =  |date = 2004年8月27日 |accessdate =  }}</ref>。TVB接獲投訴後在隔天晚上的[[娛樂大搜查|娛樂大搜查]]節目中播出原本被剪除的完滿結局（Happy Ending）。[https://www.bilibili.com/video/av798167?from=search&seid=4064738529804432738]
<blockquote style="border: 1px solid blue; padding: 0.5em 0.8em;">
有關節目的投訴：該喜劇的結局悲慘，不恰當和令人不安。男主角的死亡傳遞「滅罪無償」的不當訊息，令人不安及對兒童造成不良影響。
</blockquote>

首播後，該大團圓版結局再無出現。在2006年9月29日凌晨劇集時段及2011年6月1日、2016年5月12日的午間劇集時段，本劇重播至最後一集，而電視台播放的大結局版本同樣是上述的悲劇版本，而TVB推出的劇集DVD版本也是悲劇結局。

而在12年後同監製的電視劇《[[來自喵喵星的妳|來自喵喵星的妳]]》裏從史高拔口中已經確認莫作棟已殉職。
