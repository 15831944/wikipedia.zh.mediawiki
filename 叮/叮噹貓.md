《'''叮噹貓'''》是[[藤子·F·不二雄|藤子·F·不二雄]]的[[漫畫|漫畫]]作品，連載於希望之友1975年4月號到1978年5月號，單行本由[[潮出版社|潮出版社]]發售共3冊。2011年12月由[[小學館|小學館]]發行{{Link-ja|藤子‧F‧不二雄大全集|藤子・F・不二雄大全集}}收錄全一冊，臺灣由[[青文出版社|青文出版社]]發行。

在1993年4月5日開始播放電視動畫共170集<ref>[http://www.animenewsnetwork.com/encyclopedia/anime.php?id=1761 Rocky Rackat! (TV)][[Anime_News_Network|Anime News Network]]{{en}}</ref>，故事設定與原作有些不同<ref>[http://www.allcinema.net/prog/show_c.php?num_c=89038 ポコニャン！(1993)]allcinema{{ja}}</ref>。

2016年9月3日，於[[藤子·F·不二雄博物館|藤子·F·不二雄博物館]]上映短篇電影《叮噹貓&哆啦A夢「{{lang|ja|ポンポコニャンでここほれニャンニャン}}!?」》，也是自1993年的電視動畫以來睽違23年再度動畫化。同時在藤子·F·不二雄博物館發行限量版動畫插圖，收錄在《{{link-ja|藤子·F·不二雄大全集|藤子・F・不二雄大全集}}》發行。

==設定==
漫畫版設定叮噹貓是會使用機器道具的貍貓、類似[[哆啦A夢|哆啦A夢]]的設定，寄居者為小學生太郎一家三口，許多故事曾改編為哆啦A夢的動畫。但動畫版的主人為女孩木葉美琪，人際關係設定和居住城市亦有極大差別。

==故事簡述==
;漫畫版
:五年前太郎全家上山健行時發現叮噹貓，太郎將他帶回家，成為家中成員，故事圍繞在叮噹貓以超能力或製作道具解決太郎及親友的各種煩惱。
;動畫版
:小女孩美琪和她的爸爸媽媽到山上去，在森林裡，美琪意外的發現穿著藍色吊帶褲的橘色貍貓將牠帶回家去，大家都叫牠「叮噹貓」，牠會施各種的法術，雖然有時會調皮搗蛋，但仍帶給美琪和親友許多歡樂。

==漫畫==
===主要人物===
<small>使用青文出版社中譯名。</small>
;{{nihongo|叮噹貓|ポコニャン}}
:像貓又像貍貓的不可思議的動物，會使用超能力也會製造神奇的道具。起初只會說「叮噹貓」來表達自己的意見，之後會加入簡單的單詞，會寫字，曾經留書離家。將尾巴對準物品後說出「叮噹貓」就可利用超能力改變物品特性。是太郎家中一員。
;{{nihongo|太郎|太郎（たろう）}}
:就讀於山川小學，有賴床的毛病，個性有點冒失。小時候在山中遇見叮噹貓，並將他帶回家，和叮噹貓感情融洽。
;{{nihongo|太郎爸爸|太郎の父（たろうのちち）}}
:最喜歡的食物是糯米團子。曾嚴厲拒絕要重金購買叮噹貓的研究人員。
;{{nihongo|太郎媽媽|太郎の母（たろうのはは）}}
:家庭主婦，因為關心家人有時有些囉唆。
;{{nihongo|猩猩山|ヒヒ山（ヒヒやま）}}
:太郎的同學，常強佔同學的點心、玩具或叮噹貓製作的道具。
;{{nihongo|美紀(美紀子)|みきちゃん（みき子）}}
:太郎的同學，有很好的歌喉，成績比太郎好。
;{{nihongo|辣韮|ラッキョ}}
:太郎的同學，喜歡炫耀。
===道具===
叮噹貓所製作的神奇道具或是經過超能力改造的道具。
;美麗相機
:不管拍什麼都會變得很漂亮。
;{{nihongo|擴音器|もしもしメガホン}}
:能與昆蟲變成好朋友，讓昆蟲了解人類所說的話。
;抑制地心引力瓦斯
:被噴到的物體會不受地心引力影響。
;{{nihongo|模型山黏土|ドリームねんど}}
:以黏土製作模型山，將人縮小後可以感受登山的樂趣。
;時光機
:叮噹貓改造鬧鐘，可以將人或物品送到12小時之內的世界。
;捲筒衛生紙攝影機
:以空箱子和捲筒衛生紙製作成攝影機，衛生紙是底片。
;實物時間倒轉機
:改造捲筒衛生紙攝影機，可將物品恢復成原來的樣子。
;驚慌製造機
:利用立體影像及音效製造地震、火災、洪水的幻覺。
;將人變成煙霧的煙囪
:帶在頭上可變成煙霧。
;許願煙火
:在煙火消失前許願可以實現願望。
;空調模型相簿
:可將室內空氣及溫度變得跟模型一樣。
;孤獨瓦斯
:被瓦斯圍繞的人無法聽見及看見周圍的人。
;記憶帽子
:可以顯現出過去的記憶。
;魔術控制器
:被裝上控制器的任何物體都可被遙控。
;將信寄到過去的郵筒
:可將信件寄到過去的時空。
;室內雪
:不會融化也不會冷的雪。
;迷你迷你滑雪橇
:依不同重心位置來調整滑雪的速度及煞車。

==動畫==
===登場角色===
;叮噹貓
:
: {{配音2|jp|[[三田友子|三田友子]]|tw|[[孫若瑜|孫若瑜]]}}
:本作主角。從遙遠的「叮噹貓王國」出生的雄性貍貓動物，是能適用叮噹法術的狸貓。在美琪跟她父母爬山發現而當木葉家的孩子。把美琪當做姐姐般地撒嬌、極力想使用法術幫助美琪，但大多因為思考不夠周慮反幫倒忙。使出法術前會先說「叮噹法術變變變，看我的厲害！」。胃口甚大、喜歡難聽的歌聲。

;木葉 美琪
:
: {{配音2|jp|[[平松晶子|平松晶子]]|tw|[[敖冰欣|敖冰欣]]}}
:本作女主角，木葉家獨生女。小學一年級，個性好強，負責幫叮噹貓收拾善後而煩惱，但兩人感情相當要好，亦將叮噹貓當成弟弟般疼愛。

;木葉 綠
:
: {{配音2|jp|[[向井真理子|向井真理子]]|tw|[[呂佩玉|呂佩玉]]}}
:美琪的母親，家庭主婦，喜好室內網球。

;木葉 繁
:
: {{配音2|jp|[[江原正士|江原正士]]|tw|[[于正昇|于正昇]]}}
:美琪的父親，上班族，疼愛妻女。屢次嘗試料理或歌唱等才藝皆功敗垂成。

;一本義 幹助
:
: {{配音2|jp|[[鈴木琢磨|鈴木琢磨]]}}
:綠的父親、美琪的爺爺，是退休的高階刑警。個性頑固時常會說反話刁難他人。

;花田 菊之助
:
: {{配音2|jp|[[櫻井敏治|櫻井敏治]]|tw|[[于正昇|于正昇]]}}
:美琪的同學。花店老闆娘長子，對植物了解深入，並將花草當人類般照顧的小園丁。

;櫻 亞多莉
:
: {{配音2|jp|[[白石文子|白石文子]]|tw|[[許淑嬪|許淑嬪]]}}
:美琪的同學。個性文靜，擅長鋼琴以及精通塔羅牌占卜，準確率很高。

;楠木 昇
:
: {{配音2|jp|[[三木眞一郎|三木眞一郎]]|tw|[[許淑嬪|許淑嬪]]}}
:美琪的同學。動物方面的知識淵博，且善於領導。

;檜 秀作
:
: {{配音2|jp|[[柏倉勉|柏倉勉]]|tw|[[呂佩玉|呂佩玉]]|hk|[[曾秀清|曾秀清]]}}
:美琪的同學。頭腦精明、成績好，擅於操作電腦以及機器。

;檜 優
:
: {{配音2|jp|[[小林優子|小林優子]]|tw|[[許淑嬪|許淑嬪]]}}
:秀作的妹妹、幼稚園小班。常常愛調皮搗蛋捉弄叮噹貓。

;若葉 桃子
:
: {{配音2|jp|[[島本須美|島本須美]]|tw|[[呂佩玉|呂佩玉]]}}
:美琪的級任導師，未婚而已有男友。
