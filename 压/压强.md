'''壓力'''，是作用在與物體表面[[垂直|垂直]]方向上的每單位面積的[[力|力]]（Force）的大小，即是分布在特定作用面上之力與該面積的比值。壓力可用任意之力單位與面積單位進行測量，壓力的[[國際標準單位|國際標準單位]]為[[帕斯卡|帕斯卡]]（每單位[[平方公尺|平方公尺]]的[[牛頓_(單位)|牛頓]]）。在此定義時，壓-{}-力的力是作「規模、程度、一切事物所具有的效能或作用<ref>[http://dict.revised.moe.edu.tw/cgi-bin/cbdic/gsweb.cgi?o=dcbdic&searchid=W00000003447 字詞【力】，釋義［名］3。]</ref>」的字義，並非是物理學上的[[力|力]]。

[[絕對壓力|絕對壓力]]為[[相對壓力|相對壓力]]（又稱[[計式壓強|計式壓強]]）加上該地[[大氣壓力|大氣壓力]]。日常生活中用的壓力計，如車胎壓、血壓通常是測量相對壓力。<ref>[http://highscope.ch.ntu.edu.tw/wordpress/?p=70795 絕對壓力]</ref>

壓力在公式中常以英文字母「p」表示。压强與[[力|力]]和-{面}-積的關係如下：<ref name="Georgia State University, Physics & Astronomy">{{cite web |url=http://hyperphysics.phy-astr.gsu.edu/hbase/press.html |title=Pressure |accessdate=2018-06-14 }}</ref>

:<math>p = \frac{F}{S}</math>

其中:<br />
*<math>p</math>代表壓力
*<math>F</math>代表垂直作用力
*<math>S</math>代表受力面積

或写成[[微分|微分]]形式：

<math>p=-\operatorname{d}\!\mathit{F}/\operatorname{d}\!\mathit{S}</math>

其中的负号表示力（朝向物体）与法向量的方向（从物体出发）相反。

根据上述公式，可以推导出计算[[液体|液体]]壓力的公式：

:<math>p = \frac{F}{S} = \frac{G}{S} = \frac{mg}{S} = \frac{\rho Vg}{S} = \rho gh</math>

即：

:<math>p = \rho g h</math>

其中：
*<math>p</math>表示壓力
*<math>\rho</math>表示'''液体'''的[[密度|密度]]
*<math>g</math>表示[[重力加速度|重力加速度]]
*<math>h</math>表示物体表面离液体上方的距离

== 压强与能量密度 ==
在[[中子星|中子星]]的形成条件中，由于压强与能量的体积密度具有相同的[[量纲|量纲]]，有时使用压强来表示能量密度的数值。相关的数学关系可以在[[统计物理|统计物理]]学中推得。

==相關意義==
此外，發生在兩個物體接觸表面、垂直於該表面的作用力，亦可稱為壓-{}-力。
