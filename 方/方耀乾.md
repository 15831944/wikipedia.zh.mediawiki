'''方耀乾'''（Wade-Giles：Fang Yaw-chien；POJ：Png Iāu-khiân；1958年12月18日－），出生於[[台灣|台灣]][[臺南市|台南]]，籍貫為[[臺南市|臺南市]][[安定區_(臺南市)|安定區]]，[[成功大學|成功大學]][[台灣文學|台灣文學]]博士，目前擔任[[國立台中教育大學|國立台中教育大學]]台灣語文學系特聘教授兼系主任、[[教育部|教育部]]十二年國民基本教育語文領域(閩南語)課程綱要研修小組召集人、[[教育部|教育部]]國民小學師資培用聯盟本土教育學習領域教學中心主任、國家語言發展法之研究與規劃主持人、[[教育部|教育部]]本土教育委員會委員兼分組召集人、 [[教育部|教育部]]本國語言推動委員會委員等。

方耀乾集台灣文學創作者、研究者、教育者、編輯者於一身，是[[台語|台語]]與[[台灣文學|台灣文學]]的權威學者和詩人<ref> 林央敏編，《台語詩一世紀》，台北：前衛出版社，2006年03月，頁156。</ref>、作家，台語文學理論的主要建構者，也是台語教育、台語課程綱要研修及台灣本土語言師資培育的主要領航人，國家語言發展法主要起草人。其主要研究領域涵蓋[[台語文學|台語文學]]、[[台灣文學史|台灣文學史]]、[[文學史書寫理論|文學史書寫理論]]、戰後[[現代詩|現代詩]]等，其創作語言為[[台語|台語]]，以[[詩|詩]]及[[戲劇|戲劇]]為主。詩曾入選各國詩選，被翻譯成英文、西班牙文、土耳其文、蒙古文、日文、孟加拉文出版。

== 生平 ==
方耀乾出生於[[台南|台南]]，其先祖世世代代於[[曾文溪|曾文溪]]畔務農，直到祖父那一代，才從事[[漢醫|漢醫]]事業。祖父[[方榮欽|方榮欽]]（1902年－1975年）是一位[[中醫師|中醫師]]兼[[漢詩|漢詩]][[詩人|詩人]]，對[[漢詩|漢詩]]、[[南管|南管]]有所鑽研，在[[漢古典文學|漢古典文學]]耳濡目染之下，使其對文學產生濃厚的興趣。

父親[[方能安|方能安]]，曾擔任過小學教師，後來繼承家業成為中醫師。父親於其就讀大學時溘然長逝，故與母親相依為命。方耀乾在母親晚年中風照顧至其過世之際，為了紓解其對母親病情的無力感與尋求和母親對話的窗口，遂將其感情投注在[[母語|母語]]詩創作上。第一本[[台語|台語]]詩集《[[阮阿母是太空人|阮阿母是太空人]]》乃其對母親的懷念與不捨。方耀乾已婚並育有兩女。<ref>陳佩如，《方耀乾台語詩研究：1997－2008》，台東：國立台東大學華語文學系台灣語文教師碩士班碩士論文，2008年9月。</ref>

中學階段受其大哥[[方耀鴻|方耀鴻]]之影響，並大量閱讀其購置之文學書籍和剪報，而有志於[[文學|文學]]生涯。1980年進入[[中國文化大學|中國文化大學]]英國文學系，1984年於中國文化大學西洋文學研究所英文組攻讀英、美文學，在此時期方耀乾拓展其世界文學的觸角，並且奠定其文學理論與創作的基礎。

1991年方耀乾開始投入[[台語文|台語文]]之研究與運動，積極致力於台灣[[母語|母語]]的復振工作與文學創作。此外，方耀乾參與多個台語文社團，以及擔任多個台語文雜誌之編務。<ref>陳佩如，《方耀乾台語詩研究：1997－2008》，台東：國立台東大學華語文學系台灣語文教師碩士班碩士論文，2008年9月。</ref>

2004年方耀乾進入[[成功大學|成功大學]][[台灣文學系|台灣文學系]]博士班深造，於2008年以《Uì邊緣kàu多元中心：台語文學ê主體建構》論文拿到博士學位。在這篇論文裡，他提出「[[多元脈絡理論|多元脈絡理論]]」，企圖建立[[台語文學|台語文學]]的主體性，扭轉[[台灣文學史|台灣文學史]]的一元書寫，成為互為主體的[[多元中心|多元中心]]書寫。<ref>方耀乾，《Uì邊緣kàu多元中心：台語文學ê主體建構》，台南：國立成功大學台灣文學系，2008年。</ref>

== 經歷 ==
方耀乾自1988年開始從事教職，曾經任教單位計有[[崑山工專|崑山工專]]、[[台南家專|台南家專]]、台南市救國團、[[台南女子技術學院|台南女子技術學院]]、[[台南大學|台南大學]]（前身為[[台南師範學院|台南師範學院]]）、[[台南市社區大學|台南市社區大學]]、[[台南應用科技大學|台南應用科技大學]]。

方耀乾曾擔任[[台南應用科技大學|台南應用科技大學]]通識教育中心語文組副教授、[[台南女子技術學院|台南女子技術學院]]應用外語系主任兼視聽教育中心主任、《台南女子技術學院學報》執行編輯、《[[菅芒花台語文學|菅芒花台語文學]]》主編、《[[菅芒花詩刊|菅芒花詩刊]]》總編輯、《[[海翁台語文學|海翁台語文學]]》編審委員、《[[台灣e文藝|台灣e文藝]]》主編、《[[台文戰線|台文戰線]]》發行人、[[菅芒花台語文學會|菅芒花台語文學會]]理事長、[[漢家出版社|漢家出版社]]主編等。

方耀乾積極從事[[台語文學|台語文學]]創作與推廣，曾架設台語教學網站「[https://web.archive.org/web/20050103013039/http://www.twcat.edu.tw/webmaster/wwweng/t_edu/ 台語教育學院]」及台語文學網站「[http://203.68.184.155/pages/02/02-1.htm 方耀乾文學網]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}」，參與相關社團，及主辦多場台語文學學術研討會，其中「2005台語文學學術研討會」為第一次純台語的文學學術研討會。

2007年承接[[文建會|文建會]]「台語kap客語現代文學網站」計畫，並撰寫《[[台語文學發展簡史|台語文學發展簡史]]》，翻譯成[[台語|台語]]、[[客語|客語]]、[[華語|華語]]、[[英語|英語]]、[[法語|法語]]、[[日語|日語]]、[[德語|德語]]、[[越南語|越南語]]等8種[[語言|語言]]。<ref>詳見[http://140.116.10.241/NCKUTaiWeb/View/index.aspx 台語kap客語現代文學專題網站] {{webarchive|url=https://web.archive.org/web/20090207210236/http://140.116.10.241/NCKUTaiWeb/View/index.aspx |date=2009-02-07 }}</ref>

== 文學理論 ==
方耀乾致力於[[台語文學|台語文學]]研究與[[台語文學史|台語文學史]]的建構，主張「[[解構論|解構論]]」和「[[建構論|建構論]]」的交替運用，企圖以「[[多元脈絡理論|多元脈絡理論]]」建立[[少數文學|少數文學]]（尤其[[台語文學|台語文學]]、[[客語文學|客語文學]]、[[原住民語文學|原住民語文學]]）的主體性，解構台灣文學史一元書寫，建構使之成為互為主體的[[多元中心|多元中心]]書寫。<ref>方耀乾，《Uì邊緣kàu多元中心：台語文學ê主體建構》，博士論文，台南：國立成功大學台灣文學系，2008年。</ref>

其亦於2008年12月6日於[[南台科技大學|南台科技大學]]主辦的「[[台語藝術論述與表演研討會|台語藝術論述與表演研討會]]」提出「[[影音文學|影音文學]]」理論，企圖拓展文學研究的視野與領域。<ref>方耀乾，〈影音文學概論：多聲道的台語文學〉，發表於「台語藝術論述與表演研討會」，台南：南台科技大學，2008年12月6-7日。</ref>

== 作品主題 ==
方耀乾自1991年開始研究[[台語文|台語文]]，1997年開始從事[[台語文學|台語文學]]創作和台語文學研究，作品以[[詩|詩]]及[[戲劇|戲劇]]為主，以[[台語詩|台語詩]]見長，寫作主題涵蓋母子親情、夫妻愛情、鄉土景致、社會關懷、族群認同、本土歷史、[[台灣主體性|台灣主體精神]]、性別議題等面向。

== 寫作特色 ==
方耀乾目前創作以[[台語詩|台語詩]]為大宗，作品具有較大的形式變化和文學創作技巧，是個勇於嘗試的詩人，主要寫作特色有「多變的寫作形式」、「生動的修辭技巧」、「鮮明的意象營造」<ref>陳佩如，《方耀乾台語詩研究：1997-2008》，台東：國立台東大學華語文學系台灣語文教師碩士班碩士論文，2008.09，頁96。</ref>。茲分述如下：
=== 寫作形式 ===
方耀乾的台語詩具有多變的書寫形式，主要有「[[三行詩|三行詩]]」、「[[四行詩|四行詩]]」、「[[散文詩|散文詩]]」、「[[圖像詩|圖像詩]]」、「[[視覺詩|視覺詩]]」、「[[題號詩|題號詩]]」等。

=== 修辭技巧 ===
方耀乾善用修辭營造詩句的美感，作品中常常使用「排比」、「層遞」、「頂真」、「映襯」、「轉化」、「類疊」、「譬喻」、「借代」等修辭。

=== 意象營造 ===
其作品中主要呈現「自然意象」、「感官意象」和「整體意象」；「自然意象」著重在「植物類」、「自然現象類」等，「感官意象」著重在「視覺」、「聽覺」、「味覺」、「觸覺」等；「整體意象」著重在「夫妻情」、「平埔族」、「台灣人」等。

== 寫作風格 ==
方耀乾寫作風格主要受到[[中國古典詩|中國古典詩]]、[[西洋文學|西洋文學]]的影響，作品呈現多面向[[形式|形式]]和語言[[風格|風格]]，目的在於提高[[台語詩|台語詩]]的文學價值。他曾經為自己寫作風格下過這樣的註解：「我的詩是[[浪漫主義|浪漫主義]]、[[寫實主義|寫實主義]]佮[[現代主義|現代主義]]的結合，這佮大多數的[[台語詩|台語詩]]是寫實主義的，較無仝，按呢做的原因是，欲試驗台語詩會當行到啥乜地步。」<ref>黃彩如等，〈台語文學的疊磚仔師――專訪方耀乾〉，《白鴒鷥之歌》，台南：縣文化局，2001年10月，頁235-250。</ref>

方耀乾曾在一篇文章裡自述他自己可能：「是台語詩壇引介和實驗最多西方各門派技巧的詩人，其詩作運用的技巧有[[浪漫主義|浪漫主義]]、[[寫實主義|寫實主義]]、[[象徵主義|象徵主義]]、[[印象主義|印象主義]]、[[意象主義|意象主義]]、[[現代主義|現代主義]]、[[魔幻寫實|魔幻寫實]]、[[後現代主義|後現代主義]]等。」<ref>方耀乾，＜三位一╱異體：黃勁連、林央敏、方耀乾的台語詩之路＞，《詩心台灣情研討會》，高雄：國立高雄師範大學客家文化研究會，2005年6月3日，頁10。</ref>

*[[歐純純|歐純純]]：「方耀乾的詩歌長期以來一直以關懷本土見長，對於台灣的一草一木、一事一物，都有深刻的體悟與觀察，他將這些感觸形諸歌詠，變成一篇篇動人的詩章。除了情感上以在地事物為寄託外，其寫作技巧多變，將各式文學手法熔鑄一爐，並勇敢嘗試體製上的新巧變化，具備著鮮明的開拓精神……」<ref>歐純純，〈本土性的關懷，開拓性的書寫： 方耀乾詩集《將台南種佇詩裡》的寫作特色〉，吳達芸、方耀乾主編，《2005台語文學學術研討會論文集》，2005，頁443。</ref>
*[[胡長松|胡長松]]：「透過對鄉土的追尋與歌頌，方耀乾的詩歌為他的自我認同找到了一個很清晰實際的方向，這也是當代台語詩發展的重要脈絡。」「戰後，眾多台語詩人拿起筆找尋自己，並在鄉土上找到了自己。我們不能輕忽這件事，這是當代台灣人自我追尋之路的重要現象，而方耀乾的詩歌成就，是其中重要的里程碑！」<ref>胡長松，〈論方耀乾詩裡的自我追尋〉，《台文戰線》，第4號，台南：台文戰線雜誌社，2006年10月，頁6。</ref>
*[[胡長松|胡長松]]：「我們很有理由相信方耀乾的創作，乃是追求某種獨特生命形式的努力，而這個努力，或許不止於說明它做為一個台語詩人的個人特質，重要的是，他的努力背後同時意味著一整個時代的獨特的台灣人的生命形式特質：尤其是屬於台語詩人群積極追尋的，一種企圖從母語文化形式特質來區別於殖民者的文化身份認同。」<ref>胡長松，〈論方耀乾詩裡的自我追尋〉，《台文戰線》，第4號，台南：台文戰線雜誌社，2006年10月，頁11-38。</ref>
