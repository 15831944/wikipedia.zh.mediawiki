'''方樟顺'''（{{bd|1934年|10月|2007年|9月5日}}）[[浙江省|浙江省]][[淳安县|淳安县]]人，[[中华人民共和国|中华人民共和国]]官员，[[中国地震局|国家地震局]]局长、党组书记<ref name=zgdzj>{{Cite web |url = http://www.cea.gov.cn/manage/html/8a8587881632fa5c0116674a018300cf/_history/08_06/30/1214800086796.html |title =方樟顺同志遗体送别仪式在北京举行  |author =  |date = 2007-09-11 |publisher =中国地震局  |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1980年6月加入[[中国共产党|中国共产党]]。1955年自[[南京大学|南京大学]]地质系毕业后，分配到[[中华人民共和国地质部|中华人民共和国地质部]]中南地质局实习，一年后入职地质部河南省地质局，历任技术员、普查组长、大队技术负责人、河南省地质局地矿处主任工程师、河南省地质勘探公司技术总负责人、河南省地质局副总工程师、副局长。1982年10月，调任[[中华人民共和国地质矿产部|中华人民共和国地质矿产部]]科技司司长。1986年8月，任中华人民共和国地质矿产部副部长、党组成员。1988年7月，任[[国家地震局|国家地震局]]局长，1990年1月兼任中共国家地震局党组书记。1992年3月起，任[[中国人民政治协商会议|中国人民政治协商会议]]第八届、第九届全国委员会委员。此外还先后兼任[[中国地质学会|中国地质学会]]副理事长、[[中国国际减灾十年委员会|中国国际减灾十年委员会]]常务副主任、[[中国地震学会|中国地震学会]]会长、[[中国地质灾害学会|中国地质灾害学会]]副理事长。1995年9月，自国家地震局局长、党组书记职位上退下来后，继续担任全国政协委员，直到2004年10月退休<ref name=zgdzj/><ref name=shishi/>。

2007年9月5日，方樟顺在[[北京|北京]]病逝，享年72岁<ref name=zgdzj/><ref name=shishi>{{Cite web |url = http://news.sohu.com/20071103/n253032714.shtml |title =方樟顺同志逝世  |author =  |date = 2007-11-03 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。
