'''方庙站'''，为中国[[安徽省|安徽省]][[合肥市|合肥市]]的一座地铁车站，工程站名'''北二环站'''，位于[[北二环路_(合肥市)|北二环路]]与[[当涂北路|当涂北路]]交叉口，为[[合肥轨道交通3号线|合肥轨道交通3号线]]和[[合肥轨道交通4号线|合肥轨道交通4号线]]的换乘站。<ref>{{cite web |title="图书馆"易误解 合肥市民为地铁3号线站名“挑刺” |url=http://www.ahwang.cn/hefei/20190125/1848031.shtml |publisher=安徽网 |accessdate=2019-01-25}}</ref>

== 车站结构 ==
方庙站位于北二环路与当涂北路交叉口，3号线与4号线“T”型换乘，3号线车站为地下二层双柱三跨岛式站台，4号线车站为地下三层双柱三跨岛式站台，3、4号线站台同期施工。<ref>{{cite web |title=合肥地铁3号线沿线站点公布！开通运营时间定在 |url=http://wemedia.ifeng.com/44596356/wemedia.shtml |publisher=凤凰网安徽 |accessdate=2019-02-01}}</ref>
