'''方位角'''又稱'''地平經度'''（'''Azimuth'''，縮寫Az），是在[[平面|平面]]上量度物體之間的角度差的方法之一。

==用法==

* 真方位角 (True bearing)
: 所有角度以正北方設為000°，順時針轉一圈後的角度為360°。
: 因此：
:: 正北方：000°或360°
:: 正東方：090°
:: 正南方：180°
:: 正西方：270°

* 羅盤方位角 (Compass bearing)
: 正北和正南作首要方位，正東和正西為次要方位，在兩者之間加上角度。因此角度只會由 0°至 90°。
: 因此：
:: 正北方： N0°W 或 N0°E
:: 正東方： N90°E 或 S90°E
:: 正南方： S0°W 或 S0°E
:: 正西方： N90°W 或 S90°W

假若兩者加上與目標的距離，就會成為[[極坐標系|極坐標]]：[[直角坐標系|直角坐標系（笛卡爾坐標系）]]以外的另一種[[坐標系統|坐標系統]]。

==實際用途==

在空战中，'''方位角'''是你的飞机相对于敌机尾部的角度。字母“L”或“R”显示在方位角之后，指示你的飞机在目标的哪一侧。
