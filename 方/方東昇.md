'''方東昇'''（{{lang|en|'''Anthony Fong Tung Shing'''}}，{{bd|1976年|6月1日|||}}），[[香港|香港]]新聞從業員。

== 簡介 ==
方東昇在小學時期從[[中國大陸|中國大陸]]移居香港<ref>{{Cite web |url=http://tvb4life.pixnet.net/blog/post/205027207-%E9%98%BF%E5%AA%BD%E4%BF%82%E6%89%8D%E5%A5%B3-26%E6%AD%B2%E5%88%9D%E6%88%80-%E6%96%B9%E6%9D%B1%E6%98%87%EF%BC%9A%E8%80%81%E5%A9%86-%E5%90%B3%E5%85%88%E7%8E%B2-%E9%9D%9A |title=阿媽係才女 26歲初戀　方東昇：大起底！ |publisher=[[FACE周刊|FACE周刊]] 第438期 |date=2015-10-14}}</ref>，通曉[[粵語|粵語]]、[[英語|英語]]、[[日語|日語]]、[[俄語|俄語]]、[[普通話|普通話]]、[[荷蘭語|荷蘭語]]、[[德語|德語]]等七種語言<ref>{{Cite web |url=https://hk.celebrity.yahoo.com/%E9%87%91%E5%8F%A5%E7%8E%8B%E6%96%B9%E6%9D%B1%E6%98%87-%E4%B8%AD%E5%A4%A7%E6%89%8D%E5%AD%90%E8%AD%98%E8%AC%9B7%E5%9C%8B%E8%AA%9E%E8%A8%80-073600983.html |title=金句王方東昇 中大才子識講7國語言 |publisher=[[星島日報|星島日報]] |date=2019-02-21}}</ref>。他曾就讀於[[聖公會主風小學|聖公會主風小學]]和[[拔萃男書院|拔萃男書院]]<ref>{{Cite web |url=http://hk.apple.nextmedia.com/entertainment/art/20151013/19330484 |title=東昇秘聞：方東昇的五件事 |publisher=[[香港蘋果日報|香港蘋果日報]] |date=2015-10-13}}</ref>，1999年畢業於[[香港中文大學|香港中文大學]][[新亞書院|新亞書院]]新聞及傳播學系，並以一級榮譽畢業（GPA 4.0），更獲得[[新亞書院|新亞書院]]獎學金最高榮譽誠明獎，有「中大才子」的稱號<ref>{{Cite web |url=http://topick.hket.com/article/1619382/%E7%94%B1%E4%B8%BB%E6%92%AD%E8%B5%B0%E5%88%B0%E3%80%8A%E4%B8%96%E7%95%8C%E9%9B%B6%E8%B7%9D%E9%9B%A2%E3%80%8B%20%20%20%20%E6%96%B9%E6%9D%B1%E6%98%87%E7%88%9BGag%E7%8E%8B%E4%B9%8B%E8%B7%AF |title=由主播走到《世界零距離》 方東昇爛Gag王之路 |publisher=[[香港經濟日報|香港經濟日報]] |date=2017-01-10}}</ref>。

方東昇中學時擔任辯論隊成員。曾與辯論隊成員[[呂秉權|呂秉權]]及[[黄永|黄永]]等參與多次辯論比賽。1998年暑假加入[[無綫新聞|無綫新聞]]任職實習記者，實習期後留任當港聞記者至今，從未轉職。方在2000年4月開始兼任主播，過往曾長期主持[[香港早晨|香港早晨]]，其後於2007年升職為首席記者，2009年3月7日開始主持[[六點半新聞報道|六點半新聞報道]]。

方東昇主要採訪社會民生及海峽兩岸新聞，包括在2006年長駐[[加拿大|加拿大]]為新聞部追蹤報道[[中國|中國]][[遠華走私案|遠華走私案]]主犯[[賴昌星|賴昌星]]的暫緩遣返聆訊，以及於2006年至2008年間經常前往台灣暫駐，採訪當地的重要新聞，包括在2006年[[陳水扁|陳水扁]]總統第一家庭[[貪污|貪污]]案引起的倒扁效應、2007年[[中正紀念堂|中正紀念堂]]改名、[[2008年中華民國總統選舉|2008年總統大選]]等。另外，他常以幽默方式報道新聞，亦是唯一現職記者於節目《[[新聞話事·人|新聞話事·人]]》內接受[[伍晃榮|伍晃榮]]訪問。

2010年1月，方東昇獲晉升為助理採訪主任。同年8月23日，赴[[馬尼拉|馬尼拉]]現場採訪[[香港旅行團在菲律賓被挾持事件|香港旅行團在菲律賓被挾持事件]]。2011年9月14日起，他擔任《[[無綫電視晚間新聞|晚間新聞]]》主播，至2014年獲晉升為副採訪主任，其後再以幽默方式替《[[世界零距離|世界零距離]]》（半小時全紀錄）擔任旁白受到觀眾歡迎<ref>{{Cite web |url=https://www.hk01.com/%E5%A8%9B%E6%A8%82/64752/-%E4%B8%96%E7%95%8C%E9%9B%B6%E8%B7%9D%E9%9B%A2-III-%E6%96%B9%E6%9D%B1%E6%98%87%E5%B9%BD%E9%BB%98%E5%A4%A7%E7%88%86%E7%99%BC-%E6%96%B0%E9%87%91%E5%8F%A5%E7%AC%91%E5%88%B0%E6%9C%89%E8%85%B9%E8%82%8C |title=【世界零距離 III】方東昇幽默大爆發　新金句笑到有腹肌 |publisher=[[香港01|香港01]] |date=2017-01-09}}</ref>，而2015年又以同樣方式聲音導航《[[世界零距離|世界零距離II]]》（半小時全紀錄）更令他受歡迎程度急升<ref>{{Cite web |url=http://www.nmplus.hk/enterainment/%E6%96%B9%E6%9D%B1%E6%98%87-%E7%88%9Bgag/ |title=金句王方東昇 《世界零距離II》10首廣東歌爛GAG |publisher=[[新Monday|新Monday]] |date=2017-01-13}}</ref>。

2016年，方東昇晉升為策劃主任，負責海外專題節目，主持《[[世界零距離|世界零距離III]]》<ref>{{Cite web |url=http://hk.apple.nextmedia.com/entertainment/art/20170111/19893350 |title=方東昇又嚟 《世界零距離III》再爆金句 |publisher=香港蘋果日報 |date=2017-01-11}}</ref>，並能在出訪[[俄羅斯|俄羅斯]]及[[哈薩克|哈薩克]]期間，使用流利俄語溝通。2017年7月3日，他轉為報導六點半新聞，拍檔為[[黃曉瑩|黃曉瑩]]；2018年跟李曉欣和黃曉瑩合作主持第二輯《[[長命百二歲|長命百二歲]]》，曾分別以流暢荷蘭語及德語採訪[[荷蘭|荷蘭]]首相[[馬克·呂特|馬克•呂特]]和[[德國|德國]]總理[[安格拉·默克爾|安格拉·默克爾]]，再次展露其語言天賦<ref>{{Cite web |url=http://hd.stheadline.com/life/ent/realtime/1438656/ |title=曾問荷蘭首相「怕唔怕老」 方東昇識講多國語言「騷Quali」 |publisher=[[頭條日報|頭條日報]] |date=2019-02-21}}</ref>。

== 感情生活 ==
=== 婚前小插曲 ===
2006年11月，方東昇與舊同事[[吳先瑩|吳先瑩]]的胞妹吳先玲舉行婚禮，但於之前曾發生了一段小插曲。他們已預訂的酒家因租約在設宴前完結，他們認為該酒家處事失當，事件令他們表示「有壓力」要求索償，此事件亦曾經被傳媒報道<ref>{{cite news |url=http://the-sun.on.cc/channels/news/20060603/20060603025807_0000.html |title=力寶軒約滿 訂酒席爛尾 無綫主播逼改婚期告美心 |publisher=[[太陽報_(香港)|太陽報]] |date=2006-06-03}}</ref>。
