'''方树桥'''{{BD|1935年|11月|||catIdx=F方}}出生于[[河北省|河北省]][[沧州市|沧州市]][[吴桥县|吴桥县]]，毕业于沈阳市第八高级中学，为知名动画片配音演员，代表作有《聪明的一休》、《羚羊号历险记》、《时间飞船》等。

== 生平 ==
1935年，方树桥出生在[[河北省|河北省]][[沧州市|沧州市]][[吴桥县|吴桥县]]，曾在沈阳八中读书，1983年，他首次担任配音演员，为《聪明的一休》配音。他是[[辽宁人民艺术剧院|辽宁人民艺术剧院]]的一员。<ref>{{cite web |url=http://game.21cn.com/comic/talk/2005/03/08/2017087.shtml |title=[动漫资料]中国的配音演员介绍 |accessdate=2005-03-08 |publisher=21CN |archiveurl=https://web.archive.org/web/20050308054232/http://game.21cn.com/comic/talk/2005/03/08/2017087.shtml |archivedate=2005年3月8日 |work= |language=zh-cn |deadurl=yes }}</ref><ref>{{cite web |url= http://chinaneast.xinhuanet.com/2008-03/18/content_12722661_1.htm|title=用声音讲故事的辽艺 经典动漫的幕后传奇  |accessdate= 2008年3月18日 |publisher= 振兴东北网|archiveurl=http://chinaneast.xinhuanet.com/2008-03/18/content_12722661_1.htm |archivedate=2012年12月21日 |work= |language=zh-cn}}</ref>
