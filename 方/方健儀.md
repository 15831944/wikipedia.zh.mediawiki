'''方健儀'''（{{lang-en|'''Akina Fong Kin Yee'''}}，{{bd|1977年|1月28日|catIdx=F方}}），[[香港|香港]][[新聞工作者|新聞工作者]]，[[無綫新聞|無綫新聞]]前首席記者兼主播，現為藝人、職業[[司儀|司儀]]與皮革工作者{{citation needed|time=2017-07-15}}。

== 簡歷 ==
方健儀小時居住[[沙田|沙田]][[新田圍邨|新田圍邨]]，曾就讀於[[沙田官立小學|沙田官立小學]]及[[沙田官立中學|沙田官立中學]][[文科|文科]]班，[[香港高級程度會考|香港高級程度會考]]取得3A佳績<ref>{{cite news|url=http://www.scmp.com/article/181371/look-nature-students-urged|title='Look to nature' students urged|date=1997-01-13|work=[[South_China_Morning_Post|South China Morning Post]]}}</ref>，1999年畢業於[[香港中文大學|香港中文大學]]新聞與傳播學院，後加入[[香港電台|香港電台]]中文新聞部任職記者。方健儀於2004年負笈[[英國|英國]][[伯明翰大學|伯明翰大學]]修讀[[國際關係|國際關係]][[碩士|碩士]]課程，畢業後一度為[[聯合國總部|聯合國總部]]工作。曾在2005年11月19日採訪時任[[美國總統|美國總統]][[乔治·W·布什|喬治·布殊]]訪華和同年12月中採訪在[[香港|香港]]舉行的[[世界貿易組織第六次部長級會議|世界貿易組織第六次部長級會議]]<ref>{{Cite web |url=https://www.instagram.com/p/BZtP3C_ltyO/?taken-by=akinafong |title=方健儀instagram圖片 |date=2017-10-01|quote=我的戰績。透明膠已封塵，鐵扣都已生銹，但回憶絕唔會因為年月而褪色。}}</ref>。

2006年2月，方健儀加入香港[[無綫新聞|無綫新聞]]任高級記者，主要採訪重點新聞、治安、交通及衛生方面的新聞。

2007年2月起，方健儀兼任主播，主持《[[無綫電視午間新聞|午間新聞]]》及《[[無綫電視晚間新聞|晚間新聞]]》；2007年11月3日起亦開始兼任《[[六點半新聞報道|六點半新聞報道]]》的主播。2009年1月，方健儀獲晉升為首席記者。2009年2月2日，無綫新聞進行改革，方健儀開始兼任《晚間新聞》星期一至五之常規主播。方健儀於2010年10月10日與無綫新聞工程人員[[洪楚恒|洪楚恒]]結婚<ref>《[[FACE|FACE]]》週刊，第156、165期</ref><ref>{{cite news|url=http://hk.apple.nextmedia.com/news/art/20101011/14540520|title=無綫女主播 十全十美出嫁|date=2010-10-10|work=[[蘋果動新聞|蘋果動新聞]]}}</ref>，當日無綫特別借出[[Tvbuddy|Tvbuddy]]車牌的公司車輛作為當日的結婚花車。

2011年8月26日，方健儀在個人[[新浪微博|微博]]暗示自己將駐[[北京|北京]]，暫不再主持《晚間新聞》，並寫上「新聞報道完啦，晚安」。同年9月6日，方健儀首次擔任無綫新聞駐北京記者，為期約兩個月。

2012年1月31日，方健儀於同日擔任《[[六點半新聞報道|六點半新聞報道]]》及《晚間新聞》的主播，這是她於2008年12月26日後再次擔任《六點半新聞報道》的主播<ref>[http://ww1.sinaimg.cn/bmiddle/6ba17f48gw1dpm70kbxx8j.jpg 微博圖片]</ref>。2月7日，方健儀再次同樣擔任《六點半新聞報道》及《晚間新聞》的主播。

2012年2月24日週五，方健儀最後一晚擔任《晚間新聞》主播<ref name=akina>{{youtube|gOS3i1oHulU|方健儀最後一晚晚間新聞}}</ref>；於新聞完結時，方健儀如常以「新聞報道完畢」結束新聞報道後突然低頭一秒，面對鏡頭略帶緊張地以哽咽語調講出「多謝支持，後會有期，再見」<ref name=fongky>{{youtube|6PRKCQSif40|方健儀：多謝支持，後會有期，再見！}}</ref>；無綫新聞主題音樂隨即響起，燈光調暗。方健儀主播生涯與當晚《晚間新聞》同告一段落。《晚間新聞》播放完後，有同事送上寫有「最佳女主播」和她的頭像的蛋糕告別<ref>[http://ww3.sinaimg.cn/large/67b8f215jw1dqdt0sokszj.jpg 微博圖片]</ref>。2012年2月25及26日，分兩輯播出的[[警務處處長|警務處處長]][[曾偉雄|曾偉雄]]專訪，是方健儀在無綫記者生涯中的最後一個報導。

2012年4月1日，方健儀於[[廉政公署_(香港)|廉政公署]]任職高級新聞主任<ref>{{cite news|url=http://hk.apple.nextmedia.com/news/art/20120119/15999071|title=隔牆有耳：女主播方健儀過檔廉署|work=蘋果日報|date=2012-01-19}}</ref>，至2014年10月2日約滿離職。

2015年4月6日，方健儀重返「娘家」[[香港電台|香港電台]]，主持[[香港電台第一台|香港電台第一台]]節目《[[精靈一點|精靈一點]]》。同年9月主持無綫節目《[[望子成才|望子成才]]》。同年12月為[[BOMBA|BOMBA]]主持《星期七檔案-DSE》[[毅進文憑|毅進文憑]]廣告。<ref>{{youtube|lNymrzQqBoI|星期七檔案 - DSE}}</ref>

2016年1月11日，方健儀獲得[[毛記電視|毛記電視]]《[[第一屆十大勁曲金曲分獎典禮|第一屆十大勁曲金曲分獎點禮]]》香港區最受歡迎女歌星。

== 報道風格 ==
[[File:TVB_News_Live_in_Shanghai.ogv|thumb]]做《[[六点半新闻报道|六点半新闻报道]]》直播时的录影，摄于2010年4月29日]]

方健儀外貌酷似內地演員[[刘孜|刘孜]]及[[電視廣播有限公司|香港無綫電視]]女藝員[[葉翠翠|葉翠翠]]，常在報道中展示幽默的一面，例如在報道時親自示範戴上太陽眼鏡，以表達防止被太陽傷到眼睛的方法；在報道[[香港國際七人欖球賽|香港國際七人欖球賽]]時，她亦在[[香港大球場|香港大球場]]於鏡頭前身穿球衣接球；報道[[奧運馬術有限公司|奧運馬術有限公司]]招募禮儀小姐時，於外景片段中頭頂著書，一邊講對白一邊步行。

2011年7月，方健儀於報導日本虛擬氣功的新聞時，擺出《[[龍珠|龍珠]]》「[[龜波氣功|龜波氣功]]」的手勢，結果被[[高登討論區|高登討論區]]網友加上光影特效[[惡搞|惡搞]]改圖，其後方健儀得悉此事並沒有生氣，且在其[[新浪微博|新浪微博]]多謝網友的創意，表現出其「玩得」的一面。

2015年5月，《[[100毛|100毛]]》推出[[惡搞|惡搞]]網上視頻節目《[[毛記電視|毛記電視]]》，特別情商方「復出」作「特別新聞報道」，方更重演前無綫新聞主播[[龔偉怡|龔偉怡]]早年在《[[香港早晨|香港早晨]]》中被拍得在鏡頭前「偷吃」的經典一幕，繼續盡顯其「玩得」與充滿幽默感的一面<ref>{{cite web|url=https://m.facebook.com/story.php?story_fbid=688382891289361&id=273813022746352&_rdr|title=特別新聞報道 《100毛》申請免費電視牌照「毛記電視」 5月18日正式成立|work=100毛|date=2015-05-08}}</ref>。

== 演唱歌曲 ==
除了報道新聞和主持節目外，她也會在其他方面作出嘗試。如在2015年[[何韻詩|何韻詩]]「十八種香港HOCC」[[演唱會|演唱會]]上演唱舊曲新詞作品〈中東與綜〉（原曲：〈風箏與風〉，主唱：[[Twins|Twins]]），內容取材自無綫記者麥詩敏錯讀「[[中東呼吸綜合症|中東呼吸綜合症]]」事件。<ref>{{cite web|url=http://hk.apple.nextmedia.com/entertainment/art/20150821/19264205|title=方健儀領獎開金口|work=蘋果日報|date=2015-08-21}}</ref>
