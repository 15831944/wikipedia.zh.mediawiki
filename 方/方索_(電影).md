《'''方索'''》（{{lang-en|'''''Fonzo'''''}}）是一部即將上映的[[傳記片|傳記]][[犯罪片|犯罪電影]]，由[[喬什·特蘭克|喬什·特蘭克]]執導、編劇和剪接，[[湯·赫迪|湯·赫迪]]飾演惡名昭彰的黑幫[[艾爾·卡彭|艾爾·卡彭]]。<ref>{{Cite web |url = http://ew.com/movies/2018/03/26/tom-hardy-al-capone-fonzo-first-photo/ |title = Tom Hardy shares first photo as Al Capone in ''Fonzo'' biopic |author = Joey Nolfi |date = 2018-03-26 |publisher = Entertainment Weekly |language = en}}</ref>

== 概要 ==
電影集中講述[[艾爾·卡彭|艾爾·卡彭]]在監獄與癡呆症搏鬥11年後的事情。
