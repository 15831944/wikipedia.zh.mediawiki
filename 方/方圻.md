'''方圻'''（{{bd|1920年|2月15日|2018年|1月30日}}），[[安徽|安徽]][[定远县|定远]]人，生于[[北京|北京]]，[[中华人民共和国|中华人民共和国]]医学家。

==生平==
方圻祖籍安徽定远，1920年生于北京。因弟弟患[[血友病|血友病]]，方圻立志长大学医。1938年，方圻考入[[燕京大学|燕京大学]]医预系，与[[吴蔚然|吴蔚然]]、[[张金哲|张金哲]]等人是同班同学。此后又先后在[[北京协和医学院|北平协和医学院]]、[[上海圣约翰大学|上海圣约翰大学]]医学院、[[成都|成都]][[华西大学|华西大学]]医学院学习，1946年毕业，获[[医学博士|医学博士]]学位。毕业后，1946年到1948年在[[天津医科大学总医院|天津中央医院]]内科工作。1948年开始在[[北京协和医院|北京协和医院]]内科工作，历任住院医师、主治医师、副教授、内科教授；1956年到1978年任北京协和医院内科副主任；1978年到1985年任北京协和医院副院长、内科主任。后任北京协和医院名誉院长。曾任[[中华医学会|中华医学会]]副会长、内科学会主任委员、《中华心血管病杂志》主编等职。是中华人民共和国最早的[[博士生导师|博士生导师]]之一<ref name=sohu>{{Cite web |url = https://www.sohu.com/a/136327877_533511|title = 协和人的故事 阅尽沧桑都是爱的方圻教授 |author =  |date = 2017-04-25 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。1956年加入[[中国共产党|中国共产党]]<ref name=wangliu>王环增、刘玉纲，模范共产党员方圻教授，中国医院管理1986年07期</ref>。

1956年，36岁的方圻接受了为[[党和国家领导人|党和国家领导人]]做医疗保健的任务。自此，他陆续参加了党和国家领导人[[毛泽东|毛泽东]]、[[周恩来|周恩来]]、[[叶剑英|叶剑英]]、[[聂荣臻|聂荣臻]]、[[陈毅|陈毅]]等的医疗保健工作。作为[[中央保健委员会|中央保健委员会]]专家组副组长，他多次获中央颁发特殊贡献奖，并多次代表中方医生赴友好国家执行医疗保健任务。他曾五次去[[印度尼西亚|印度尼西亚]]参加[[印度尼西亚总统|印度尼西亚总统]][[苏加诺|苏加诺]]的会诊；七次到[[瑞士|瑞士]][[日内瓦|日内瓦]]参加国际医学会议；[[老挝|老挝]][[梭发那·富马亲王|富马亲王]]、[[越南|越南]][[胡志明|胡志明]]主席等病情危急时，他都代表中国进行了抢救治疗。1973年，周恩来患[[膀胱癌|膀胱癌]]住进[[中国人民解放军第三〇五医院|中国人民解放军第三〇五医院]]，方圻陪伴治疗两年，直到周恩来逝世<ref name=sohu/>。毛泽东晚年，包括方圻在内的医疗组全体医生一同照顾他。方圻是1976年9月9日宣布毛泽东死讯的4名医生（[[陶寿淇|陶寿淇]]、[[陶桓乐|陶桓乐]]、[[吴洁|吴洁]]、方圻）之一，随后医疗组全体医生在毛泽东病历上签名（签名依次为：[[胡旭东|胡旭东]]、陶寿淇、方圻、吴洁、[[姜泗长|姜泗长]]、[[高日新|高日新]]、[[翟树职|翟树职]]、[[李志绥|李志绥]]、陶桓乐、[[王新德|王新德]]、[[薛世文|薛世文]]、[[周光裕|周光裕]]、[[朱水寿|朱水寿]]、[[徐德隆|徐德隆]]、[[李春福|李春福]]）<ref name=yuyi>{{Cite web |url =http://www.zaobao.com/wencui/politic/story20180206-833176  |title =“中南海前御医”出殡 温家宝吴仪送别  |author =  |date = 2018-02-06 |publisher = 联合早报网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.chinanews.com/gj/kong/news/2007/09-26/1036616.shtml  |title = 《中华文摘》文章：“红墙医师”细述毛泽东的最后岁月 |author =  |date =  2007-09-26|publisher =  中国新闻网|language =  |accessdate =  |quote =  }}</ref>。

方圻在心血管内科水平超群。1984年北京协和医院原副院长[[艾刚阳|艾刚阳]][[心脏病|心脏病]]发作，1993年8月[[人民日报|人民日报]]社一位副总编辑突发急性[[心肌梗死|心肌梗死]]，都由方圻及时抢救回来。在协和医院教导方圻40年的[[张孝骞|张孝骞]]告诉他：“凡事要亲临病人，诊断要如履薄冰。”[[文革|文革]]爆发后的1967年，医护人员纷纷撇下病房去“闹革命”，已经升任副教授11年的方圻主动要求管病房，抢救了不少病人<ref name=sohu/>。

1983年，上级决定实行“一刀切”——年岁过线者不再担任医院主要领导，协和医院老院长告退，推荐方圻接任。但方圻却以自己年已63岁为由，推让一位更年轻的教授接任了院长。随后，方圻又辞去副院长职务，仅当内科主任。1985年1月，“一刀切”做法向下延伸，方圻又辞去内科主任，只当专职教授。1987年，方圻当选[[中共十三大|中共十三大]]代表。1993年，被[[中国医学科学院|中国医学科学院]]、[[中国协和医科大学|中国协和医科大学]]评为“协和名医”。1994年，方圻获中国医疗卫生工作者最高荣誉“[[白求恩奖章|白求恩奖章]]”<ref name=sohu/>。

2018年1月30日，方圻在北京协和医院病逝，享年98岁<ref>{{Cite web |url = http://news.sina.com.cn/o/2018-02-04/doc-ifyreuzn2631201.shtml |title = 北京协和医院名誉院长方圻逝世 享年98岁 |author =  |date = 2018-02-04 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。2月5日，遗体告别仪式在北京协和医院举行，[[温家宝|温家宝]]、[[吴仪|吴仪]]等逾千人前来送别<ref name=yuyi/>。
