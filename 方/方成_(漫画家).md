'''方成'''（{{bd|1918年|6月10日|2018年|8月22日|catIdx=F}}），原名'''孙顺潮'''，杂文笔名'''张化'''，生于[[北京|北京]]，祖籍[[广东|广东]][[香山县|香山]]，[[中华人民共和国|中华人民共和国]][[漫画家|漫画家]]、杂文家、幽默理论研究者，与[[丁聪|丁聪]]、[[华君武|华君武]]并称中国漫画界三老<ref name=pp>{{Cite web |url =https://www.thepaper.cn/newsDetail_forward_2369179  |title =著名漫画家方成逝世，享年100岁  |author =  |date =2018-08-22   |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref><ref name=rmw>{{Cite web |url = http://opinion.people.com.cn/n/2014/0508/c1003-24991559.html |title =人民日报足音：方成，幽默的符号  |author =  |date = 2014-05-08 |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref><ref name=leshan>杨乐生，方成：在乐山读“武大”的日子，文史春秋2005(5):41-43</ref><ref name=chch>{{Cite web |url =  http://www.cclycs.com/r139148.html|title = 世纪老人，漫画人生——《方成全集》自序 |author =  |date = 2017-09-04 |publisher =  川川旅游网|language =  |accessdate =  |quote =  }}</ref>。

==生平==
方成祖籍今[[中山市|中山市]][[南朗镇|南朗镇]]左步村，生在北京。因家中人多，方成四、五岁时，母亲带方成兄妹几个回到家乡左步村居住。方成九岁时，母亲托人将他带回北京的父亲家里，到离家很近的铭贤小学（由[[英国伦敦会|英国伦敦会]]开办）上学。小学毕业后，考入北平市市立第三中学校（今[[北京市第三中学|北京市第三中学]]），自该中学初中毕业后，考入[[北京师范大学实验二龙路中学|弘达中学]]（当时弘达中学二院和宿舍都在[[月坛公园|月坛]]内）<ref name=chch/><ref>{{Cite web |url = http://www.zsnews.cn/news/2018/06/09/3270840.shtml |title = 收录百岁漫画家方成80年作品，《方成全集》昨首发|author =  |date =2018-06-09  |publisher = 中山网 |language =  |accessdate =  |quote =  }}</ref>。初中三年级时，在铁路局工作的父亲得知方成喜欢画画后，为他找了[[徐燕荪|徐燕荪]]当老师，但学了不到十周，父亲失业，方成就没再跟随徐燕荪学画。后来徐燕荪在北平的朋友[[许麟庐|许麟庐]]接着教方成、[[钟灵|钟灵]]画画<ref name=ellj/>。

1935年高中二年级时，方成参加[[一二·九运动|一二·九运动]]，开始从事漫画创作。1938年夏，自江西省立南昌一中高中毕业，毕业成绩列全省第七，被保送[[清华大学|国立清华大学]]。当时因[[抗日战争|抗日战争]]全面爆发，清华大学1937年[[七七事变|七七事变]]后从[[北平|北平]]迁到[[长沙|长沙]]，1938年夏又迁到[[昆明|昆明]]。方成在[[重庆|重庆]]未能预订到去昆明的汽车票，错过了报到和开学时间。时任教育部中等教育司司长的舅公[[陈峙东|陈峙东]]建议他进[[国立武汉大学|国立武汉大学]]借读。1938年11月中旬，教育部批准了方成入国立武汉大学借读的申请，方成从重庆出发，1938年11月27日抵达[[乐山|乐山]]，12月1日在[[乐山文庙|乐山文庙]]开始了在国立武汉大学的学习。当时他的英文教师是英籍华人[[李那|李那]]，国文教师是[[叶圣陶|叶圣陶]]。在校期间，他在[[话剧|话剧]]演出中负责化装和音响，参加过歌咏队和下乡宣传抗日，学校的每期壁报他都以笔名“利巴尔”配画漫画，“利巴尔”是[[山东|山东]]方言，意为“小伙计”或“很不在行”。他还和几位同学组织黑白社，办《黑白》壁报（办了近三年，每周一期）。1942年毕业于国立武汉大学化学系。1942年到1946年，在乐山任[[黄海化学工业研究社|黄海化学工业研究社]]助理研究员。1945年[[日本投降|日本投降]]后，他从黄海化学工业研究社里保存的上海出版的英文报刊上，看到转载的[[英国|英国]]、[[美国|美国]]政治讽刺画，由此了解了这种画的画法。他受英国漫画家大卫·罗（David Low）的漫画技法影响大。1946年辞职离开乐山，到上海从事漫画工作。1947年春，任《[[观察_(杂志)|观察]]》杂志漫画版主编和特约撰稿人。1947年冬，方成回广东家乡探亲，听说上海白色恐怖加剧，不敢回上海，乃避居香港。当时《观察》被[[中国国民党|国民党]]当局查封，工作人员被捕入狱。1948年，方成在[[香港|香港]]参加人间画会，从事漫画创作。1948年到1949年，出版连环漫画《康伯》（原在《[[大公报|大公报]]》连载）<ref name=pp/><ref name=ellj>{{Cite web |url =http://comic.chinaren.com/20090219/n262349619.shtml  |title =方成：一位91岁新闻漫画家的幽默人生  |author =  |date = 2009-02-19 |publisher =搜狐  |language =  |accessdate =  |quote =  }}</ref><ref name=leshan/><ref name=cishi/>。

[[中华人民共和国|中华人民共和国]]成立后，1949年方成从香港回到北京。1950年到1951年任北京《新民报》（后改为《[[北京日报|北京日报]]》）美术编辑。1951年起，任《[[人民日报|人民日报]]》文艺部美术组的美术编辑，在该报社一直工作到1986年。1957年，方成因为写了篇叫《过堂》的杂文，讽刺教条主义，所以在[[反右运动|反右运动]]中，《人民日报》文艺部美术组姓杨的年轻领导想将方成划为“漏网右派”，幸得[[华君武|华君武]]保护，方成才躲过一劫。[[改革开放|改革开放]]后，方成从《人民日报》文艺部美术组转到国际部，国际部领导[[袁先禄|袁先禄]]同意给方成一段时间创作漫画。1979年，方成重新拿起画笔，短短三个月就创作了一百多幅作品。1980年8月，在北京[[中国美术馆|中国美术馆]]举办个人画展“方成漫画展览”，这是中华人民共和国成立后第一次个人漫画公开展，展出了一百幅讽刺漫画，后来该画展还到[[上海|上海]]、[[广州|广州]]、[[南宁|南宁]]、[[四川|四川]]、[[成都|成都]]、[[重庆|重庆]]等十多个城市巡回展览，后又到[[香港|香港]]展览。1984年，方成任[[武汉大学|武汉大学]]新闻系兼职教授。1989年，任[[郑州大学|郑州大学]]新闻系漫画专科班学术顾问。后任[[中国社会科学院|中国社会科学院]]研究生院新闻系[[硕士生导师|硕士生导师]]，[[中国广播艺术团|中国广播艺术团]]说唱团艺术顾问，[[中国美术家协会|中国美术家协会]]常务理事，中国新闻漫画研究会副会长、会长，[[美国|美国]]出版的国际漫画杂志《Witty World》编辑，《[[讽刺与幽默|讽刺与幽默]]》编委，享受[[政府特殊津贴|政府特殊津贴]]。1986年[[离休|离休]]<ref name=pp/><ref name=ellj/><ref name=chch/><ref>{{Cite web |url = http://www.zsmuseum.cn/zsmsg/11096.html |title = 《百岁方成作品展》 |author =  |date =2017-05-26  |publisher = 中山市博物馆 |language =  |accessdate =  |quote =  }}</ref>。

方成与[[相声|相声]]大师[[侯宝林|侯宝林]]是多年好友，二人常共同切磋幽默艺术的课题，试图创建“幽默学”。侯宝林逝世后，方成独自承担起创建中国式“幽默学”的使命，经常应邀到中国各大学讲授幽默学课程，并曾在[[美国|美国]]巡回做幽默学讲座<ref name=rmw/>。

2018年8月22日，方成在[[北京友谊医院|北京友谊医院]]病逝，享年100岁<ref name=pp/>。

==作品==
方成一生致力漫画事业，发表了很多有社会影响力的作品。出版漫画集、漫画理论和幽默研究、杂文集等共计30多种。方成的多幅作品被[[中国美术馆|中国美术馆]]收藏，作品《[[武大郎|武大郎]]开店》获《人民日报》1980年优秀作品二等奖，儿童连环诗画《王小青》获[[中国福利会|中国福利会]]奖<ref name=pp/>。2008年，方成将《武大郎开店》、《[[侯宝林|侯宝林]]》、《[[钟馗|钟馗]]》、《神仙也有残缺》等159幅漫画水墨手稿以及125册珍贵图书捐赠给[[中国国家博物馆|中国国家博物馆]]<ref name=ellj/>。1988年获中国漫画界最高奖——首届“中国漫画金猴奖荣誉奖”。2009年获“中国美术奖·终身成就奖”<ref name=cishi>{{Cite web |url =https://www.thepaper.cn/newsDetail_forward_2369738  |title =敢为人先，幽生活一默的百岁漫画家方成辞世  |author =  |date =  2018-08-22|publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

==家庭==
*父：孙桥芳（笑平），在北平（今北京市）平绥铁路局任科员。方成有一个弟弟，两个妹妹<ref name=chch/>。
*母：姓方。方成曾解释“方成”这个笔名的来历说：“母亲姓方，加上自己字写得不好，用‘方’字笔画少，好写些。”<ref name=leshan/>
*妻：[[陈今言|陈今言]]（1924年—1978年），漫画家，1947年和方成结婚。去世后留下三个未成年的孩子<ref name=rmw/><ref name=leshan/>。
