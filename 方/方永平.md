'''方永平'''[[教授|教授]] <!--（{{bd|  ||||catIdx=F}}）-->（{{lang|en|Prof. FONG Wing-ping}}），生物化學家。[[香港中文大學|香港中文大學]]生命科學學院教授，[[香港中文大學|中大]][[崇基學院|崇基學院]]第12任院長。

==香港中文大學==
*方永平於1983年在[[香港中文大學|香港中文大學]][[聯合書院|聯合書院]]生物化學系理學士畢業，1987年後獲[[香港中文大學|香港中文大學]]生物化學哲學博士學位，之後赴美國[[哈佛醫學院|哈佛醫學院]]任博士後研究員。1989年返回[[香港中文大學|香港中文大學]]生命科學學院任教。方氏於2011年及2015年獲頒中大理學院模範教學獎，並於國際期刊發表超過110份研究報告。 <ref>{{cite web |url=http://www.cpr.cuhk.edu.hk/tc/press_detail.php?1=1&1=1&id=2308 |title=香港中文大學委任三所書院院長 |date=2016-07-20 }}</ref>

==崇基學院==
*方永平曾任崇基走讀生舍堂首任舍監 （1996-1999）、學生宿舍委員會主席（1999-2004）、學生輔導長（2004-2011）、服務學習中心首位主任（2013-2015）及崇基65周年校慶籌備委員會主席。2013年，方氏獲學院推選為[[香港中文大學|香港中文大學]]董會的崇基代表，2014年8月獲委任為崇基學院副院長。<ref>{{cite web |url=http://www.ccc.cuhk.edu.hk/zh-tw/news-announcements/419-professor-fong-wing-ping-as-candidate-for-next-college-head-to-meet-with-staff-students-and-alumni |title=下任院長候薦人方永平教授與教職員、學生及校友會面 }} </ref>
*2016年1月，[[陳偉光|陳偉光]]教授因病辭任[[崇基學院|崇基學院]]院長，方永平開始出任崇基學院署理院長。<ref>{{cite book |url=https://www.ccc.cuhk.edu.hk/images/ChungChiPublication/ChungChi_65Book.pdf |title=善彌雅集-崇基學院六十五週年校慶特刊 }}</ref> 
*2016年7月，中大校董會通過委任方永平出任崇基學院院長，任期4年。<ref>{{cite news |url=https://topick.hket.com/article/1467507/%E6%96%B9%E6%B0%B8%E5%B9%B3%E6%95%99%E6%8E%88%E4%B8%8B%E6%9C%88%E3%80%8C%E5%9D%90%E6%AD%A3%E3%80%8D%E5%B4%87%E5%9F%BA%E5%AD%B8%E9%99%A2%E9%99%A2%E9%95%B7 |title=方永平教授下月「坐正」崇基學院院長 |publisher=經濟日報 |date=2016-07-20 }}</ref> 2016年10月，崇基學院假崇基禮拜堂舉行「感恩崇拜暨方永平院長就職典禮」，由崇基學院校董會主席李國星宣讀誓詞，委任方永平為第12任院長。<ref>{{cite web |url=http://www.takungpao.com.hk/finance/text/2016/1029/34189.html |title=中大崇基學院65周年院慶 }}</ref> <ref>*[http://www.ccc.cuhk.edu.hk/zh-tw/news-announcements/419-professor-fong-wing-ping-as-candidate-for-next-college-head-to-meet-with-staff-students-and-alumni 下任院長候薦人方永平教授與教職員、學生及校友會面.2016-05-06.]</ref>
*2016年10月，《中大通訊》專訪方永平院長，其以“科學精神──大膽假設，小心求證”鼓勵年輕人。<ref>{{cite book |url=http://www.iso.cuhk.edu.hk/chinese/publications/CUHKUPDates/article.aspx?articleid=1793 |title=專訪崇基學院新任院長方永平}}</ref> 
*2017年7月，方永平參與崇基學院籌辦的「院長探訪 北美校友活動」計劃。<ref>{{cite web |url=http://www.cuhk.edu.hk/ccc/publications/newsletter/pdf/newsletter20170901.pdf |title=崇基退休老師吳夢珍博士 }}</ref>
*2018年6月，方永平教授聯通崇基校董會主席李國星、中大校長段崇智，以及崇基發展委員會主席郭志樑主持「崇基學生發展綜合大樓」的動土儀式。<ref>{{cite web |url=http://std.stheadline.com/daily/news-content.php?id=1811357&target=2 |title=崇基綜合大樓動工 料2021年啟用 }}</ref>
