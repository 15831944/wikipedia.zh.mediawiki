'''克雷莫纳传统小提琴制作工艺'''是[[意大利|意大利]][[克雷莫納|克雷莫納]]的古代手工藝品，自16世紀以来，製作了[[小提琴|小提琴]]，[[中提琴|中提琴]]，[[大提琴|大提琴]]和[[低音提琴|低音提琴]]等[[弓弦乐器|弓弦乐器]]。

[[File:Workshop_luthier.jpg|左]]
「'''克雷莫纳传统小提琴制作工艺'''」（義大利官方稱为'''Saperi e saper fare liutario della tradizione cremonese'''）於2012年在[[保护非物质文化遗产政府间委员会|保护非物质文化遗产政府间委员会]]第七屆會議期間被[[联合国教育、科学及文化组织|教科文組織]]宣佈為[[人类非物质文化遗产代表作名录|非物質文化遺產]]。

== 技術 ==
[[File:Violini.JPG|左]]
弓弦乐器可以不同的技术製作，但克雷莫納的[[弦樂器製作師|弦樂器製作師]]開發了一種被認為是世界上獨一無二的技術。<ref>{{Cite web|url=http://www.beniculturali.gov.it/mibac/export/MiBAC/sito-MiBAC/Contenuti/MibacUnif/Comunicati/visualizza_asset.html_1855986864.html|title=La liuteria di Cremona iscritta nella Lista UNESCO del Patrimonio Immateriale|accessdate=2017-02-07|language=it|archive-url=https://web.archive.org/web/20180510185204/http://www.beniculturali.gov.it/mibac/export/MiBAC/sito-MiBAC/Contenuti/MibacUnif/Comunicati/visualizza_asset.html_1855986864.html#|archive-date=2018-05-10|dead-url=yes}}</ref>

每件樂器都是使用超過70種木制组件手工组装的。小提琴的每一部分都需要不同的技術，根据木材发出的聲響作出調整：因此小提琴是不可能完全相同的。小提琴的每一部分都使用特殊的木材製成，木材需經過自然的历练和[[弦樂器製作師|製作師]]精心的挑选，不能強以人為製作。<ref name="nomination">{{Cite web|url=http://www.unesco.org/culture/ich/doc/download.php?versionID=16813|title=Nomination file no. 00719|accessdate=2017-07-02|format=Doc 189kb|year=2012}}</ref>
[[File:Cremona-Liuteria.jpg|右]]
克雷莫那傳統小提琴不可使用任何工業或半工業部件制作，亦不可进行噴漆。樂器的的很多部位看起来是裝飾性的，但其实它们可令聲音更大：這是第一支小提琴的两个特徵。

從樹木挑选到完成成品的每一個階段的制作過程都是由小提琴製造商親自进行，因此，克雷莫納的小提琴製造商每年只能製作出3-6支小提琴。

傳統小提琴製作者需對所有天然材料和技術知識完整地了解，自16世紀以來，製作師们积累了共同的手法和習慣，當克雷莫納小提琴製作开始流行時，[[安德雷亚·阿马蒂|阿马蒂]]家族的技术得到了[[尼古拉·阿玛蒂|尼古拉·阿玛蒂]]的学生，[[斯特拉迪瓦裡|斯特拉迪瓦裡]]，{{le|朱塞佩·瓜尔内里|Giuseppe Guarneri|瓜爾內裡}}和{{le|卡爾洛·貝爾貢齊|Carlo Bergonzi (luthier)|貝爾貢齊}}等人的改進和完善。

== 保护 ==
[[File:Cremona_-_Museo_del_Violino_2.JPG|右]]
克拉莫納小提琴製作的古老傳統受到兩個代表了140多名工匠的監督機構保護：[[斯特拉迪瓦裡小提琴製造商協會|斯特拉迪瓦裡小提琴製造商協會]]和[[意大利小提琴協會|意大利小提琴協會]]。 此外，克雷莫納國際小提琴製作學校也於1938年9月12日在克雷莫納成立。<ref>{{cite web|title=Scuola di Liuteria|url=http://www.scuoladiliuteria.it/mostra_articolo_new.asp?area=164&area_news=164|website=www.scuoladiliuteria.it|accessdate=2017-07-02|language=IT|deadurl=yes|archiveurl=https://web.archive.org/web/20161029095914/http://scuoladiliuteria.it/mostra_articolo_new.asp?area=164&area_news=164|archivedate=2016-10-29}}</ref>

在教科文組織的認可之後，2013年，[[克雷莫納艺术宫廣場|克雷莫納艺术宫廣場]]開設了一家小提琴博物館，並设置了觀眾席让参观者欣赏克雷莫納新舊樂器演奏的音樂。<ref>{{cite web|title=Fondazione Stradivari – Museo del Violino|url=http://www.museodelviolino.org|website=www.museodelviolino.org|accessdate=2017-07-02|language=IT}}</ref>
