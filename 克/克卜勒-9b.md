'''克卜勒-9b'''是[[NASA|NASA]]的[[克卜勒太空望遠鏡|克卜勒太空望遠鏡]]發現的第一顆太陽系之外的行星 ([[系外行星|系外行星]])<ref name="datatable" /><ref name="Holman2010" />，它環繞著在[[天琴座|天琴座]]的[[克卜勒9|克卜勒9]]公轉。克卜勒-9b是克卜勒太空望遠鏡使用[[凌日法|凌日法]]在克卜勒9發現的三顆行星中最大的一顆；它的質量比行星中的[[土星|土星]]略少一些。克卜勒-9b和[[克卜勒-9c|克卜勒-9c]]顯現出被稱為[[軌道共振|軌道共振]]的現象，兩顆行星的引力互相牽引著，改變和穩定了另一顆行星的軌道。這顆行星的發現是在2010年8月26日公布的。

== 命名和歷史 ==

克卜勒-9b的名稱表明它是第一顆被發現軌道環繞著[[克卜勒9|克卜勒9]]公轉的行星。
這顆恆星因為在[[NASA|NASA]]發現[[類地行星|類地行星]]的[[克卜勒任務|克卜勒任務]]中被發現有行星，反過來被名為[[克卜勒9|克卜勒9]]<ref>{{cite web |url=http://kepler.nasa.gov/Mission/QuickGuide/ |title=Kepler: About the Mission |author= |year=2011 |work= |publisher=[[NASA|NASA]] |accessdate=10 February 2011}}</ref>。克卜勒9的行星是在克卜勒任務最初43天中發現總數高達700多顆的行星候選者。這顆恆星是被標示有多過一顆以上行星凌日的五大行星系之一，它也是第一批被證實有多顆行星凌日的恆星之一<ref name=NASAUno>{{cite web |url=http://www.nasa.gov/mission_pages/kepler/news/two_planet_orbit.html |title=NASA's Kepler Mission Discovers Two Planets Transiting the Same Star |author= |date=26 August 2010 |work= |publisher=[[NASA|NASA]] |accessdate=10 February 2011}}</ref>。 

這顆行星是克卜勒任務使用[[凌日法|凌日法]]確認的，這顆行星相對於地球從恆星的前面經過，使恆星的總光度稍微減弱了一些；這些微光度的變化確認了行星的一些特性，包括它的大小和與母恆星的距離<ref name=JPL>{{cite web |url=http://www.jpl.nasa.gov/news/news.cfm?release=2010-279 |title=NASA's Kepler Mission Discovers Two Planets Transiting Same Star |author= |date=26 August 2010 |work=[[噴射推進實驗室|噴射推進實驗室]] |publisher=[[NASA|NASA]] |accessdate=10 February 2011}}</ref>。

克卜勒-9b的質量估計由[[夏威夷|夏威夷]][[毛納基山|毛納基山]]的[[凱克天文台|凱克天文台]]完成。在這個過程中，科學家發現克卜勒-9b是[[克卜勒9|克卜勒9]]這個系統中巨大的兩顆氣體巨星中較大的一顆，但是它的質量仍然小於[[土星|土星]]<ref name=NASAUno />。

== 特性 ==
克卜勒-9b是一顆氣體行星，估計它的質量大約是0.252[[木星質量|M<sub>J</sub>]]；也就是質量大約是[[木星|木星]]的四分之一。它的半徑大約是0.842[[木星半徑|R<sub>J</sub>]]，或是大約木星半徑的80%。這顆行星環繞克卜勒9的週期是19.243天，與母恆星的距離是0.14[[天文單位|天文單位]]<ref>{{cite web|url=http://exoplanet.eu/star.php?st=Kepler-9|title=Notes for star Kepler-9|author=|year=2010|work=|publisher=[[太陽系外行星百科|太陽系外行星百科]]|accessdate=10 February 2011|deadurl=yes|archiveurl=https://web.archive.org/web/20110103072601/http://exoplanet.eu/star.php?st=Kepler-9|archivedate=2011年1月3日}}</ref>。相較於太陽系的行星，[[水星|水星]]與太陽的距離是0.3871天文單位，每87.97天才公轉一週<ref>{{cite web |url=http://hyperphysics.phy-astr.gsu.edu/hbase/solar/soldata2.html |title=Solar System Data |author=Kaufmann |year=2005 |work=HyperPhysics |publisher=[[喬治亞州立大學|喬治亞州立大學]] |accessdate=10 February 2011}}</ref>。克卜勒-9b是在克卜勒9的系統中第二接近母恆星的行星。

克卜勒-9b和[[克卜勒-9c|克卜勒-9c]]是系外行星之間有[[軌道共振|軌道共振]]被注意到的第一個例子。這兩顆行星 ，它們的軌道對應著1：2的比率，維持著彼此的軌道受到相互的引力牽引。克卜勒-9b的軌道逐漸增長，平均而言，每個[[軌道週期|軌道週期]]會增長4分鐘。最終，這種增加的趨勢將會反轉。隨著時間的推移，可以看見行星的軌道共振比率在1：2的上下振盪著<ref>{{cite web |url=http://cosmiclog.msnbc.msn.com/_news/2010/08/26/4974937-planets-spotted-in-changing-orbits |title=Planets Spotted in Changing Orbits |author=Alan Boyle |date=26 August 2010 |work=Cosmic Log |publisher=[[MSNBC|MSNBC]] |accessdate=10 February 2011 |deadurl=yes |archiveurl=https://web.archive.org/web/20110108114026/http://cosmiclog.msnbc.msn.com/_news/2010/08/26/4974937-planets-spotted-in-changing-orbits |archivedate=2011年1月8日 }}</ref>。[[卡內基機構|卡內基機構]]的Alycia Weinberger說克卜勒9的氣體巨星是在比目前距離更遠的位置上形成的，軌道共振現象的外觀可能有助於解釋它們向內遷徙的歷史<ref>{{cite web |url=http://news.cnet.com/8301-19514_3-20014870-239.html |title=NASA spacecraft spots multiplanet system |author=William Harwood |date=26 August 2010 |work= |publisher=[[CNet|CNet]] News |accessdate=10 February 2011}}</ref>。
