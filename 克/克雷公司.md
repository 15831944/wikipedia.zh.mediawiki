'''克雷'''（{{lang|en|'''Cray Inc.'''}}）是[[美國|美國]]的[[超級電腦|超級電腦]]製造商和銷售商，總部位於[[華盛頓州|華盛頓州]][[西雅圖|西雅圖]]。[[公司|公司]]的前身是1972年由[[計算機|計算機]][[設計師|設計師]][[希穆爾·克雷|希穆爾·克雷]]（[[:en:Seymour_Cray|:en:Seymour Cray]]）創立的'''克雷研究'''（{{lang|en|'''Cray Research'''}} , Inc，[[縮寫|縮寫]]：CRI）。1989年希穆爾·克雷將克雷研究拆分出克雷電腦公司（{{lang|en|Cray Computer Corporation}}，[[縮寫|縮寫]]：CCC），後來該公司在克雷研究被[[硅谷圖形公司|-{zh-cn:硅谷图形公司;zh-hk:矽谷圖形公司;zh-tw:視算科技;}-]]併購後的一年後，亦即1995年破產。目前的克雷公司成立於2000年，在[[Tera_Computer_Company|Tera Computer Company]]從[[硅谷圖形公司|-{zh-cn:硅谷图形公司;zh-hk:矽谷圖形公司;zh-tw:視算科技;}-]]手上買下前克雷研究的資產業務後成立。在目前的克雷公司成立前，總部也已遷移數次。

== 公司簡史 ==
在20世紀70年代至90年代期間，憑着希穆爾·克雷以及他的克雷公司新型超級電腦設計，控制整個超級電腦市場並佔據顛峰位置長達五年（1985年-1990年）。1980年代，正值[[小型電腦|小型電腦]]市場萌芽階段，大量小型對手加入競爭，到1990年代中葉，很多對手不是被擠出市場，就是被併購，而[[惠普|惠普]]（[[HP|HP]]）、[[國際商業機器|國際商業機器]]（[[IBM|IBM]]）在則超級計算機市場的崛起並不斷吞併小對手。公司在1995年破產了，1996年為[[硅谷图形公司|-{zh-cn:硅谷图形公司;zh-hk:矽谷圖形公司;zh-tw:視算科技;}-]]收購，1999年8月售予{{tsl|en|Tera Computer Company}}；2000年4月，Tera更名回「克雷」。到今天，超級電腦成了一種由像它們等大型電腦公司所特意設計的電腦，目前IBM佔有42%的市場，HP佔有31%，克雷僅佔6%，儘管如此但由於其比對手更強大的性能和更好的能效比，克雷依然是超級電腦領域的巨頭之一。截至2012年12月，[[TOP500|全球最快的超級電腦]]——[[泰坦_(超級電腦)|泰坦]]（Titan，置放於美國[[橡樹嶺國家實驗室|橡樹嶺國家實驗室]]，取代Jagaur），就是由克雷公司製造的{{tsl|en|Cray XK7}}組建。<ref name=top500>{{cite web|title=Oak Ridge Claims No. 1 Position on Latest TOP500 List with Titan|url=http://www.top500.org/blog/lists/2012/11/press-release/|publisher=[[TOP500|TOP500]]|accessdate=November 15, 2012|date=November 12, 2012}}</ref><ref name=anand>{{cite web|url=http://www.anandtech.com/show/6421/inside-the-titan-supercomputer-299k-amd-x86-cores-and-186k-nvidia-gpu-cores|title=Inside the Titan Supercomputer|publisher=[[Anandtech|Anandtech]]|first=Anand|last=Lal Shimpi|authorlink=Anand Lal Shimpi|page=1|date=October 31, 2012|accessdate=November 2, 2012}}</ref>

== 海外業務 ==
目前克雷公司除了在美國本土的業務以外還在[[加拿大|加拿大]]、[[巴西|巴西]]、[[日本|日本]]、[[澳大利亞|澳大利亞]]、[[台灣|台灣]]、[[香港|香港]]、[[中國大陸|中國大陸]]、[[韓國|韓國]]、[[印度|印度]]和[[歐洲|歐洲]]多國的多個國家和地區中設有[[辦事處|辦事處]]、[[子公司|子公司]]和研發機構。2009年4月在印度[[新德里|新德里]]成立子公司；2012年2月在中國大陸成立「克雷计算设备（北京）有限公司」，由克雷公司在香港註冊的「克雷中國有限公司」持有其全部股份。<ref>[http://www.gpuworld.cn/article/show/191.html 克雷(Cray)在中国成立新子公司] - GPU World</ref>在2009年至2012年間，中國大陸的大型電腦市場上也有一批克雷的機櫃，由Netop Computing引入。<ref>[http://www.xasun.com/article/43/1279.html CRAY 克雷超级计算机介绍]</ref>
