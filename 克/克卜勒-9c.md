'''克卜勒-9c'''是NASA的[[克卜勒任務|克卜勒任務]]發現的第一批7顆系外行星中，屬於[[克卜勒9|克卜勒9]]的兩顆行星中的一顆。經由[[凌日法|凌日]]確認克卜勒-9c和[[克卜勒-9b|克卜勒-9b]]是這顆恆星的行星<ref name="datatable"/><ref name="Holman2010"/>。在它最初的發現之後，[[克卜勒任務|克卜勒任務]]的團隊在2010年8月26日宣布了這項發現。在當時，克卜勒已經發現了700顆的行星候選者。 

對這顆行星的觀察顯示它是一顆質量略小於[[土星|土星]]的氫-氦氣體巨星，其軌道距離大約是0.225[[天文單位|天文單位]]。值得注意的是克卜勒-9c和b的模式是共享[[軌道共振|軌道共振]]，每顆行星的軌道互相影響使另一顆行星軌道穩定。由於這種效應，當這顆行星被發現時，它的[[軌道週期|軌道週期]]是38天，並且每個週期縮短39分鐘。久而久之，它的軌道會與b行星在2:1的上下反復振盪著。

== 命名和歷史 ==
與多數的系外行星相同，"克卜勒-9c"表示它是在[[克卜勒9|克卜勒9]]發現的第二顆行星。克卜勒9本身的名稱則是因為它是[[NASA|NASA]]的[[克卜勒任務|克卜勒任務]]發現有行星同面對方向[[凌日法|橫越]]過前方的恆星<ref>{{cite web |url=http://kepler.nasa.gov/Mission/QuickGuide/ |title=Kepler: About the Mission |author= |year=2011 |work= |publisher=[[NASA|NASA]] |accessdate=14 February 2011}}</ref>。 

這顆行星是克卜勒在最初43天的任務中發現的700顆行星候選者之一。它是五顆似乎是多行星系統恆星中被強調的一顆。克卜勒-9c和[[克卜勒-9b|克卜勒-9b]]是第一顆以凌日法確認與發現兩顆行星的恆星<ref name=Sponges>{{cite web |url=http://www.nasa.gov/mission_pages/kepler/news/two_planet_orbit.html |title=NASA's Kepler Mission Discovers Two Planets Transiting the Same Star |author= |date=26 August 2010 |work= |publisher=[[NASA|NASA]] |accessdate=16 February 2011}}</ref>。

最初對克卜勒-9c質量的估計，由[[夏威夷|夏威夷]][[毛納基山|毛納基山]][[凱克天文台|凱克天文台]]的凱克1號望遠鏡做了後續的確認。凱克證實克卜勒-9c翰克卜勒-9b的質量都略小於[[土星|土星]]<ref name=Sponges />。

== 性質 ==
克卜勒-9c是顆氣體巨星，它的大小和質量都略小於土星，大約是0.171[[木星質量|M<sub>J</sub>]]，或是木星的17%質量。它的半徑是0.823[[木星|R<sub>J</sub>]]，只比土星小一點 (1.5%)。這顆行星座落在與恆星平均距離0.225[[天文單位|天文單位]]之處<ref>{{cite web |url=http://exoplanet.eu/star.php?st=Kepler-9 |title=Notes for star Kepler-9 |author= |year=2010 |work= |publisher=[[Extrasolar_Planets_Encyclopaedia|Extrasolar Planets Encyclopaedia]] |accessdate=13 February 2011 |deadurl=yes |archiveurl=https://web.archive.org/web/20110103072601/http://exoplanet.eu/star.php?st=Kepler-9 |archivedate=2011年1月3日 }}</ref>。

這顆行星很可能是由[[氫|氫]]和[[氦|氦]]組成的。這顆行星的軌道和[[克卜勒-9b|克卜勒-9b]]，在[[克卜勒9|克卜勒9]]中的第二顆也是較大的一顆，在同一個平面上。在觀察這顆行星時，克卜勒小組注意到克卜勒-9b和c的軌道有著1:2的比率，克卜勒-9b每19天繞行恆星一圈，克卜勒-9c則是38天。這兩顆行星以重力相互吸引著，所知的[[軌道共振|軌道共振]]，維持著行星軌道的穩定。這是在太陽系之外首次看見這種現象。克卜勒-9c每完成一次觀測到的週期，它的軌道週期會縮短39分鐘。然而，在同一個點，這個趨勢會扭轉使它的軌道週期增加。它的軌道週期會在2:1的比率上下來回的振盪<ref>{{cite web |url=http://cosmiclog.msnbc.msn.com/_news/2010/08/26/4974937-planets-spotted-in-changing-orbits |title=Planets spotted in changing orbits |author=Alan Boyle |date=26 August 2010 |work=Cosmic Log |publisher=[[MSNBC|MSNBC]] |accessdate=13 February 2011 |deadurl=yes |archiveurl=https://web.archive.org/web/20110108114026/http://cosmiclog.msnbc.msn.com/_news/2010/08/26/4974937-planets-spotted-in-changing-orbits |archivedate=2011年1月8日 }}</ref>。
