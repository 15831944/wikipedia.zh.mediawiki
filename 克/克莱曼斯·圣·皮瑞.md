'''克莱曼斯·圣·皮瑞'''（{{lang-en|'''Clémence Saint-Preux'''}}，{{bd|1988年|11月29日|||catIdx=P}}）是一名法国女歌手和电影电视演员。

== 生平 ==
1988年，皮瑞出生在[[法国|法国]][[巴黎|巴黎]]西郊，父亲是法国音乐家圣·皮瑞，母亲是作家兼画家克莱曼斯，受到家庭良好教育环境的影响，她从小就展现了艺术天赋，对[[钢琴|钢琴]]、[[吉他|吉他]]和[[舞台剧|舞台剧]]充满热情。12岁时，她和法国摇滚歌手[[強尼·哈立戴|強尼·哈立戴]]合唱了歌曲《人人需要爱》。2005年，她来到[[中国大陆|中国大陆]]旅游，2011年出演了中法合拍电影《[[巴黎宝贝|巴黎宝贝]]》。<ref>{{cite web |url=http://news.xinhuanet.com/ent/2011-07/27/c_121725919.htm |title= 《巴黎宝贝》小莫痴恋邓超 当众重现法兰西之吻 |accessdate=2011年7月27日 |publisher= 新华网|archiveurl=http://news.xinhuanet.com/ent/2011-07/27/c_121725919.htm |archivedate=2012年12月16日 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://ent.sina.com.cn/m/p/2011-08-05/06023378171.shtml |title=图文：《巴黎宝贝》剧照-邓超克莱曼斯异国恋 |accessdate=2011年8月5日 |publisher= 新浪网|archiveurl=http://ent.sina.com.cn/m/p/2011-08-05/06023378171.shtml |archivedate=2012年12月16日 |work= |language =zh-cn}}</ref>
