'''克里特与昔兰尼加'''是罗马帝国的一个[[元老院行省|元老院行省]]，于公元前67年设立，所辖区域囊括了[[克里特|克里特]]岛、[[昔兰尼加|昔兰尼加]]地区以及今日的[[利比亚|利比亚]]。

== 阿皮翁将昔兰尼加赠与罗马 ==
{{tsl|en|Kingdom of Cyrenaica|昔兰尼王国}}是一个[[希臘化時代|希腊化]]国家。公元前96年，最后一位国王[[托勒密·阿皮翁|托勒密·阿皮翁]]临终前没有子嗣，将王国馈赠给了罗马共和国<ref name = Tyndale>{{cite web | url = http://www.tyndalehouse.com/Egypt/ptolemies/apion_fr.htm | title = Ptolemy Apion | accessdate = 2016-11-24 | publisher = Chris Bennett}}</ref>。罗马欣然接受了这份遗产，将其交由当地统治者管理，而非直接管辖。但在公元前70年代，犹太定居者的民间起义开始破坏地区稳定，[[羅馬元老院|元老院]]只得采取行动。前74年，派遣了一位财务官将昔兰尼加并为罗马省份，并重建秩序。元老院派遣如此低阶的官员，一方面显示出共和国在管辖不断成长的国家中所遇到的政治难题，另一方面也表明罗马管辖此地的便利与稳固<ref name = unrv>{{cite web | url = http://www.unrv.com/provinces/cyrenaica.php | title = Cyrenaica | accessdate = 2016-11-24 | publisher = UNRV.com}}</ref>。

== 罗马征服克里特 ==
{{tsl|en|Marcus Antonius Creticus|馬克·安東尼·克里提庫斯}}于前71年进攻[[克里特|克里特]]，但被击退。前69年，{{tsl|en|Quintus Caecilius Metellus Creticus|昆圖斯·卡埃基利烏斯·梅特魯斯·克雷提庫斯}}以3个军团的兵力，经过三年苦战，方才占领克里特。梅特鲁斯也因此得以在名中附加「克雷提库斯」，意为「克里特的」<ref name = unrv2>{{cite web | url = http://www.unrv.com/provinces/crete.php | title = Crete | accessdate = 2016-11-24 | publisher = UNRV.com}}</ref>。

== 沿革 == 
公元前67年，克里特与昔兰尼加并为一省<ref name = EB>{{cite web | url = https://www.britannica.com/place/Cyrenaica | title = Cyrenaica historical region, North Africa | accessdate = 27 December 2017 | publisher = Encyclopaedia Britannica}}</ref>，首府为克里特城市{{tsl|en|Gortyn|戈爾廷}}。因如此划分在地理上多有不便，公元298年，罗马皇帝[[戴克里先|戴克里先]]将其拆分<ref name = unrv2/><ref name = Livius>{{cite web | url = http://www.livius.org/articles/place/cyrenaica/ | title = Cyernaica | accessdate = 2016-11-24 | publisher = Livius.org}}</ref>。
