'''克雷格·费德里吉'''（{{lang-en|Craig Federighi}}）是[[苹果公司|苹果公司]]软件工程高级副总裁，主管iOS软件及Mac软件。

== 教育生涯 ==
Craig Federighi是[[加州大学伯克利分校|加州大学伯克利分校]]毕业的计算机科学[[硕士|硕士]]，研究方向是[[VOD|VOD]]系统。

== 事业生涯 ==
他早期在1994年加入[[NeXT|NeXT]]，负责[[企業物件框架|企業物件框架]]（Enterprise Objects Framework，EOF）系统<ref>{{cite web|url=http://news.csdn.net/a/20110323/294503.html|title=Bertrand Serlet退休，Craig Federighi接任}}</ref>
。这是一种先驱性的对象-数据库映射（[[ORM|ORM]]）系统，曾被很多金融企业和电信公司采用。这个项目当时在NeXT系统里十分重要。

1997年跟随乔布斯回到苹果。苹果渐渐地将史莱格负责的EOF系统移入Java，不再单独做发布和介绍。这使他极为失望，离开Apple而投靠了[[Ariba|Ariba]]，曾任副总裁，CTO等职。

2009年时因为Mac OS X软件高级副总裁[[伯特兰·赛莱特|伯特兰·赛莱特]](Bertrand Serlet)准备离职退休，于是Bertrand Serlet便把他从Ariba挖回到Apple，接替自己担任[[OS_X|OS X]]软件部门副总裁。随后他领导开发[[OS_X_Lion|OS X Lion]]并取得巨大成功。他再接再厉，用不到一年的时间开发了下一代[[OS_X_Mountain_Lion|OS X Mountain Lion]]。尽管他是部门副总裁，但他仍然与前任Bertrand Serlet一样，直接向[[CEO|CEO]]汇报工作。

2012年8月27日，在[[斯科特·福斯特尔|斯科特·福斯特尔]]離職後，升任Mac软件工程高级副总裁，主管iOS及OS X(macOS)。2012年10月30日改称软件工程高级副总裁。
