'''克里翁菊石'''（[[學名|學名]]：{{lang|la|'''''Cleoniceras'''''}}），又名'''香花菊石'''，是生存在[[白堊紀|白堊紀]][[阿爾布期|阿爾布期]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]。其化石被發現於[[歐洲|歐洲]]、[[馬達加斯加|馬達加斯加]]和[[美洲|美洲]]等地。<ref name="Fossilworks" />

== 特徵 ==
克里翁菊石的外觀和[[風神菊石|風神菊石]]十分相似，但其殼體的臍部較狹小、腹側較尖<ref name="Cleoniceras">{{Cite web |url = http://liusir.pixnet.net/blog/post/379983962 |title = Cleoniceras 克里翁菊石 (哈薩克) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-09 }}</ref>，且克里翁菊石的肋條十分突出，於靠近臍部的部分明顯隆起。<ref name="Aioloceras">{{Cite web |url = http://liusir.pixnet.net/blog/post/248637581-%E8%8F%8A%E7%9F%B3%E6%AD%A3%E5%90%8D%EF%BC%9A%E9%A2%A8%E7%A5%9E%E8%8F%8A%E7%9F%B3-aioloceras |title = 菊石正名：風神菊石 Aioloceras @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-09 }}</ref>從臍部結節延伸出的[[鐮刀|鐮刀]]狀肋條通常在外部的殼階上消失。
