'''克里斯汀·塞德爾'''（{{lang-en|'''Christiane Seidel'''}}）是一名在美國出生的德國裔丹麥女演員，居住在[[紐約市|紐約市]]。她著名的演出是在[[Netflix|Netflix]]的西部迷你劇《{{link-en|無神之境|Godless (TV series)}}》中飾演Martha，以及在[[HBO|HBO]]電視劇《[[酒私風雲|酒私風雲]]》中飾演挪威人Sigrid Mueller<ref>{{Cite web |url = https://www.glamour.com/story/boardwalk-empire-christiane-se |title = Boardwalk Empire Star Christiane Seidel Talks Costumes, Love Scenes, and Her Famous Acting-School Buddies |author = Lindy Segal |date = 2013-11-03 |publisher = Glamour |language = en}}</ref><ref>{{Cite web |url = https://sofetchdaily.com/2013/11/02/an-interview-with-boardwalk-empire-star-christiane-seidel/ |title = An Interview with ‘Boardwalk Empire’ Star Christiane Seidel |date = 2013-11-02 |publisher = So Fetch Daily |language = en}}</ref>。
