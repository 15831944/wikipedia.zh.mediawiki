'''克莉丝汀·卡修'''（'''Kristin Cashore'''，{{bd|1976年||||catIdx=B}}）是一位美国[[青年小說|青年]][[奇幻作品|奇幻]]小说作家。

== 早年生活 ==
克莉絲汀·卡修在[[宾夕法尼亚州|宾夕法尼亚州]]的乡村长大，是家里四个女儿中的老二。她拥有[[威廉姆斯学院|威廉姆斯学院]]的学士学位，2003年获得西蒙斯学院儿童文学研究中心的儿童文学硕士学位。<ref name=":1">{{Cite web|url=https://www.scholastic.com/teachers/authors/kristin-cashore/|title=Kristin Cashore Books, Author Biography, and Reading Level {{!}} Scholastic|website=www.scholastic.com|access-date=2018-05-26}}</ref>她曾是糖果工厂的包装工，编辑助理，法律助理和自由撰稿人。<ref name=":0">{{Cite web|url=https://www.penguinrandomhouse.com/authors/238529/kristin-cashore|title=Kristin Cashore {{!}} Penguin Random House|website=www.penguinrandomhouse.com|language=en-US|access-date=2018-05-26}}</ref>2017年，她开始在家外的办公室工作，她还在手写所有的小说。<ref name=":2" />

== 文学生涯 ==
她的[[處女作|處女作]]《{{tsl|en|Graceling||杀人恩典}}》（Graceling）出版于2008年10月。<ref>{{cite web

 | work = This Is My Secret (personal blog)

该书被提名{{tsl|en|Andre Norton Award||安德烈·诺顿}}和{{tsl|en|William C. Morris YA Award||威廉·C·莫里斯}}奖。

她的第二本书《{{tsl|en|Fire (Cashore novel)||Fire}}》出版于2009年10月，是《Graceling》的前传，<ref name="KC.my-books" />获得了阿梅利亚·伊丽莎白·瓦尔登图书奖。<ref name=":0" />

她的第三本书《{{tsl|en|Bitterblue (Cashore novel)||Bitterblue}}》出版于2012年5月，这三本书都是Graceling Realm系列的一部分<ref name="KC.my-books">{{cite web

 | date = 2008-02-28

她的最新作品是《Jane, Unlimited》，于2017年9月发行。<ref>{{Cite news|url=https://www.publishersweekly.com/pw/by-topic/childrens/childrens-authors/article/74817-q-a-with-kristin-cashore.html|title=Q & A with Kristin Cashore|work=PublishersWeekly.com|access-date=2018-05-27|language=en}}</ref> 

克莉丝汀还为教科书、教师版以及《号角指南》撰写专业著作。<ref name=":1" />
