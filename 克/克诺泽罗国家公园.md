'''克诺泽罗国家公园'''（{{Lang-ru|Кенозерский национальный парк}}）是一个成立于1991年12月28日，位于[[俄罗斯|俄罗斯]]北部省份[[阿尔汉格尔斯克州|阿尔汉格尔斯克州]]的[[卡爾戈波利斯基區|卡爾戈波利斯基區]]和[[普列謝茨基區|普列謝茨基區]]的[[国家公园|国家公园]]。2004年，克诺泽罗国家公园通过[[联合国教育、科学及文化组织|联合国教科文组织]][[人與生物圈計劃|人與生物圈計劃]]的认证，成为联合国教科文组织生物圈保护区。

== 历史 ==
{{来源请求|克诺泽罗历来是一个偏远地区。在19世纪，该地区被划分在[[奧洛涅茨省|奥洛涅茨省]]的普陀斯基乌耶兹基（ Pudozhsky Uyezd）的西部和奥伦特省（Kargopolsky Uyezd）的东部。[[苏联|苏联]]时期，在经历了几次行政区划改革之后，该地区最终划入了[[阿尔汉格尔斯克州|阿尔汉格尔斯克州]]。1950年代—1980年代，当地如现在俄罗斯北部一样，人口严重减少，特别是[[列克什莫泽罗湖|列克什莫泽罗湖]]和[[克洛泽罗湖|克洛泽罗湖]]之间的所有村庄都荒芜了。}}

1991年，当地政府决定在该地区建立一个国家公园。公园内的所有历史遗迹都移交给公园管理局，其中一些已经修复。该公园于1991年12月28日正式建成。起初建成时，公园的工作人员共有7人。<ref>{{cite web|url=http://www.kenozero.ru/o-parke/1991-god.html|title=1991 год|accessdate=2011-06-13|publisher=ФГУ НП «Кенозерский»|language=ru|deadurl=yes|archiveurl=https://web.archive.org/web/20120325162039/http://www.kenozero.ru/o-parke/1991-god.html|archivedate=2012年3月25日}}</ref>1992年增加到35人，<ref>{{cite web|url=http://www.kenozero.ru/o-parke/1992-god.html|title=1992 год|accessdate=2011-06-13|publisher=ФГУ НП «Кенозерский»|language=ru|deadurl=yes|archiveurl=https://web.archive.org/web/20120325162054/http://www.kenozero.ru/o-parke/1992-god.html|archivedate=2012年3月25日}}</ref>和1993年增至153人。<ref>{{cite web|url=http://www.kenozero.ru/o-parke/1993-god.html|title=1993 год|accessdate=2011-06-13|publisher=ФГУ НП «Кенозерский»|language=ru|deadurl=yes|archiveurl=https://web.archive.org/web/20120325162123/http://www.kenozero.ru/o-parke/1993-god.html|archivedate=2012年3月25日}}</ref>{{来源请求|2004年，该公园成为[[联合国教育、科学及文化组织|联合国教科文组织]][[人與生物圈計劃|生物圈保护区]]。}}

== 公园位置与地理 ==
克诺泽罗国家公园位于[[普列谢茨基区|普列谢茨基区]]西南部，[[阿尔汉格尔斯克州|阿尔汉格尔斯克州]][[卡尔戈波利斯基区|卡尔戈波利斯基区]]西北部，与[[卡累利阿共和国|卡累利阿共和国]]接壤，公园的北部以该地区最大的湖泊之一的[[克洛澤羅湖|克洛澤羅湖]]为中心，公园的总部设在奥辛尼诺村（Vershinino），位于湖的北岸。[[克洛泽罗湖|克洛泽罗湖]]是[[克納河|克納河]]的源头，也是[[奥涅加河|奥涅加河]]的主要支流。克納河的上游位于公园内，以及波查河—克洛泽罗湖的主要支流、它的源头是波乔泽罗湖。下游的波乔泽罗湖的主要支流—昂杜沙河的下游，均位于公园内。<ref>{{Cite web|url=http://www.kenozero.ru/en|title=Kenozero National Park / New official website of the Park|accessdate=2011-07-28|year=2010|publisher=FSI Kenozero|language=en, ru}}</ref>

{{来源请求|南部包括[[列克什莫澤羅湖|列克什莫澤羅湖]]和列克什马河的上游，这是[[拉查湖|拉查湖]]的支流。列克什莫澤羅湖和克洛泽罗湖之间被一些较小的湖泊分开，其中包括较大的[[纳格利莫澤羅湖|纳格利莫澤羅湖]]和[[维尔诺湖|维尔诺湖]]。}}

{{来源请求|沿卡累利阿边界的一些河流流入[[沃德拉河|沃德拉河]]流域，并最终汇入[[波罗的海|波罗的海]]。因此，这个公园也是大西洋和北冰洋盆地之间的分界线之一。}}

== 旅游业 ==
该公园包含自然和文化古迹，也面向[[生态旅游|生态旅游]]，公园里还开辟了许多小径。

公园内还有几个木制建筑纪古迹。其中之一是位于公园西面的{{le|波尔仁斯基·波哥斯特|Porzhensky Pogost}}教堂，它是圣乔治教堂的一部分（从18世纪开始），周围是带有大门和塔楼的木墙（1789年）。波哥斯特附近的村庄已经被遗弃，没有留下前往该教堂的正式道路，现在前往波哥斯特只能通过一条人行道才能到达。<ref>{{Cite web|url=http://www.kenozero-park.ru/|title=Кенозерский национальный парк / Old official website of the Park|accessdate=2011-06-13|year=2004|publisher=FSI Kenozero|language=ru}}</ref>

{{来源请求|有两条路可以通往公园。在南部，[[列克什莫澤羅湖|列克什莫澤羅湖]]以南，有一条未铺设的公路连接[[卡爾戈波雷|卡爾戈波雷]]和[[普多日|普多日]]。公园北部的另一条路从奥内日斯基铁路分岔而出，通过[[普列谢茨克|普列谢茨克]]连接卡尔戈波尔和耶梅茨克。这条路通往[[克洛澤羅湖|克洛澤羅湖]]的Pershlakhta村庄，然后连接到克洛澤羅湖周围的其他村庄。}}
