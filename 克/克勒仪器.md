'''克勒仪器公司'''（{{lang|en|Koehler Instrument Company, Inc.}}），又名克勒仪器，是一家[[石油|石油]]以及[[石油化工|石油化工]]测试仪器[[生产商|生产商]]。公司本部设在美国[[纽约州|纽约州]]的波西米亚，并在[[德克萨斯州|德克萨斯州]]的修斯顿有分部。

== 会议 ==
自1947年开始，克勒仪器成为[[美国材料实验协会|美国材料实验协会]]（[[ASTM|ASTM]]）的活跃成员。除此之外，克勒还是[[美国摩擦学家和润滑工程师协会|美国摩擦学家和润滑工程师协会]] <ref> [http://www.stle.org/resources/corporate/default.aspx 美国摩擦学家和润滑工程师协会记录] {{webarchive|url=https://web.archive.org/web/20090802232602/http://www.stle.org/resources/corporate/default.aspx |date=2009-08-02 }} </ref>和[[美国化学工程师学会|美国化学工程师学会]] <ref>[http://www.aiche.org/MemberCenter/CompanyNames.aspx 美国化学工程师学会记录] {{webarchive|url=https://web.archive.org/web/20081006235717/http://www.aiche.org/MemberCenter/CompanyNames.aspx |date=2008-10-06 }}</ref>的成员，并且成为[[国际滑油脂协会|国际滑油脂协会]]的董事会代表<ref>[http://www.nlgi.org/about_us/board_of_directors.php 国际滑油脂协会] {{webarchive|url=https://web.archive.org/web/20080512152521/http://www.nlgi.org/about_us/board_of_directors.php |date=2008-05-12 }}</ref>。
每年，克勒仪器公司都参与和石油测试行业有关的各项国际会议和会展<ref>[http://www.koehlerinstrument.com/conference-index.html 各项会议记录] {{webarchive|url=https://web.archive.org/web/20090620055939/http://www.koehlerinstrument.com/conference-index.html |date=2009-06-20 }}</ref>。近年来，克勒参与的主要会议有：

== 实验室测试 ==
克勒仪器测试提供了石油产品的测试服务，包括分析[[石油|石油]]，[[润滑油|润滑油]]，[[石油化工|石油化工]]，[[沥青|沥青]]，[[合成物|合成物]]，和其他相关产品。 实验室也参与测试仪器的研发工作。 其中部分特定[[ASTM|ASTM]]测试仪器包括[[氧化稳定性|氧化稳定性]]，[[闪点|闪点]]，[[蒸馏|蒸馏]]，和[[粘度|粘度]]。<ref>[http://www.astm.org/LABS/filtrexx40.cgi?+-P+ACCTNO+148600+template.frm 产品相关资料]</ref>

== 产品 ==
克勒仪器公司生产150多种仪器，所有都符合[[ASTM|ASTM]]，[[IP|IP]]，[[ISO|ISO]]，或者其他国际标准认证。<ref> [http://findarticles.com/p/articles/mi_qa5322/is_200606/ai_n21391274/ 国际认证标准资料] </ref> 克勒仪器的一些主要产品包括：[[粘度|粘度]]，[[闪点|闪点]]，[[磨损学|磨损学]]，[[针入度|针入度]]，[[蒸馏|蒸馏]]，和[[生物燃料|生物燃料]]。克勒其他制造的产品包括：色度计，离心机，氧化稳定性装置，自动滴定装置，和浊倾点测试装置。<ref>[http://www.thomasnet.com/profile/10021804/koehler-instrument-co-inc.html 制造产品资料]</ref>
