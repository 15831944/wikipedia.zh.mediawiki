'''克隆'''（{{lang-en|Clone}}）在廣義上是指利用[[生物技术|生物技术]]由[[无性生殖|无性生殖]]产生与原个体有完全相同[[基因组|基因组]]之后代的过程。在[[园艺学|园艺学]]上，克隆是指通过[[营养繁殖|营养繁殖]]产生的单一[[植株|植株]]的后代，很多植物都是通过克隆这样的无性生殖方式从单一植株获得大量的[[子代|子代]]个体。在[[生物學|生物學]]上，是指選擇性地複製出一段[[DNA|DNA]]序列（分子克隆）、[[細胞|細胞]]（細胞克隆）或是[[個體|個體]]（個體克隆）。

克隆一个生物体意味着创造一个与原先的生物体具有完全一样的[[遗传信息|遗传信息]]的新[[生物体|生物体]]。目前，现代生物学背景下，这通常包括了体[[细胞核|细胞核]][[器官移植|移植]]。在体细胞核移植中，卵母细胞核被除去，取而代之的是从被克隆生物体细胞中取出的细胞核，通常[[卵母细胞|卵母细胞]]和它移入的细胞核均应来自同一物种。由于，细胞核几乎含有生命的全部遗传信息，宿主卵母细胞将发育成为在遗传上与核供体相同的生物体。[[粒线体|粒线体]][[DNA|DNA]]这里虽然没有被移植，但相对来讲粒线体DNA还是很少的，通常可以忽略其对生物体的影响。

== 語源 ==
「clone」源于[[希腊语|希腊语]]的「klōn」（嫩枝）。在[[园艺学|园艺学]]中，「clon」一词一直沿用到20世纪。后来有时在词尾加上「e」成为「clone」，以表明「o」的发音是[[长元音|长元音]]。近来，随着这个概念及单字在大众生活中广泛使用，拼法已经变为「clone」。

「clone」在中文里有許多不同的翻译，因为地区或文意有所不同，在中国大陆普遍音译成「-{克隆}-」，在台湾则使用的場合，可翻译成「-{複製}-」、「-{選殖}-」或「-{群殖}-」。在香港和澳门則可翻译成「-{複製}-」或「-{轉植}-」。

== 分类 ==
=== 個體層次 ===
{{main|無性生殖}}

克隆通常是一种[[生物技术|生物技术]]，以[[人工诱导|人工诱导]]的[[无性生殖|无性生殖]]方式或者自然的无性生殖方式（例如：[[植物|植物]]），产生与原个体有完全相同[[基因组|基因组]]之后代的过程。一个克隆就是一个多细胞生物在[[遗传学|遗传]]上与另外一种生物完全一样。克隆可以是自然克隆，例如：无性生殖或是由于偶然的原因产生两个遗传上完全一样的个体（就像[[同卵双生|同卵双生]]一样）。但是，我们通常所说的克隆是指通过有意识的设计来产生的完全一样的复本。

=== 分子層次 ===
{{main|分子選殖}}
克隆一个基因是指从一个个体中获取一段[[基因|基因]]（例如：通过[[聚合酶鏈式反應|聚合酶鏈式反應]]的方法），然后将其插入另外一个个体（通常是通过载体），再加以研究或利用。克隆有时候是指成功地鉴定出某种[[顯性|顯性]]的[[基因|基因]]。所以，当某个生物学家说某某疾病的基因被成功地克隆了，就是说这个基因的位置和[[DNA|DNA]]序列被确定。而获得该基因的拷贝则可以认为是鉴定此基因的[[副产品|副产品]]。

==困難==
尽管動物的克隆技術研究取得了很大进展，目前的成功率还是相当低的：多利出生之前研究人员经历了276次失败的尝试<ref name="hkedcity">{{Cite web |url=http://www2.hkedcity.net/sch_files/a/kss/kss-lkh/public_html/coloninghuman.html |title=「複製人類」踩道德鋼線 - 複製人與道德 |accessdate=2018-01-26 |date=2001}}</ref>；70隻小牛的出生则是在9000次尝试后才获得成功，并且其中的三分之一在幼年时就死了；Prometea也是花费了328次尝试才成功出生。而对于某些物种，例如[[猩猩|猩猩]]，目前还没有成功克隆的报導。而狗的克隆实验，也是经过数百次反覆试验再得来的成果。

動物複製其他的失敗之作，均變成了畸形的動物。專家表示複製技術的失敗率高達97%。大部分科學家認為修改基因技術是超乎人類[[知識|知識]]的極限，因此必須禁止<ref name="hkedcity">{{Cite web |url=http://www2.hkedcity.net/sch_files/a/kss/kss-lkh/public_html/coloninghuman.html |title=「複製人類」踩道德鋼線 - 複製人與道德 |accessdate=2018-01-26 |date=2001}}</ref>。全球有19個國家已經禁止從事複製人的實驗。英國在2000年12月通過及修訂《人類受精及胚胎法》，准許涉及複製人類胚胎之醫學研究，不過仍禁止任何以繁殖為目的之複製研究<ref name="hkedcity">{{Cite web |url=http://www2.hkedcity.net/sch_files/a/kss/kss-lkh/public_html/coloninghuman.html |title=「複製人類」踩道德鋼線 - 複製人與道德 |accessdate=2018-01-26 |date=2001}}</ref>。

低成功率可能是克隆囊胚滋养外胚层存在的重编程异常细胞是克隆胚胎發育失败的原因，但这种观点还存在很多争议<ref>{{cite news|title=研究发现克隆动物出生率低关键原因|url=http://paper.sciencenet.cn/htmlpaper/2011481539046816060.shtm?id=16060|accessdate=2011-04-11|newspaper=科学网|date=2011-04-08}}</ref> 。

== 選殖結果 ==
多利出生後的年龄检测表明其出生的时候就上了年纪。她6岁的时候就得了一般老年时才得的[[关节炎|关节炎]]。这样的衰老被认为是[[端粒|端粒]]的磨损造成的。端粒位於[[染色体|染色体]]的末端。随着[[细胞分裂|细胞分裂]]，端粒在复制过程中不断磨损，这通常认为是衰老的一个原因。然而，研究人员在克隆成功牛後却发现它们实际上更年轻。分析它们的端粒表明它们不仅是回到了出生的长度，而且比一般出生时候的端粒更长。这意味着它们可以比一般的牛有更长的寿命，但是由于过度生长，它们當中很多都过早夭折了。研究人员相信相关的研究最终可以用来改变人类的寿命。
