'''克勞德·謝瓦萊'''（Claude Chevalley，{{IPA-fr|ʃəvalɛ|lang}}，{{bd|1909年|2月11日|1984年|6月28日}}）是一名[[法國|法國]]數學家，他對[[數論|數論]]、[[代數幾何|代數幾何]]、[[類域論|類域論]]、[[有限群|有限群論]]以及[[代數群|代數群論]]作出重要貢献。他是[[尼古拉·布爾巴基|布爾巴基學派]]{{notetag|布爾巴基學派的早期代表人物是第一次世界大戰後法國掘起的一批數學家，包擴[[昂利·嘉當|昂利·嘉當]]（H. Cartan，1904-2008）、[[安德烈·韋伊|安德烈·韋伊]]（A. Weil，1906-1998）、[[讓·迪厄多內|讓·迪厄多內]]（J. Dieudonné，1906-1992）等<ref name = ziran>{{cite journal | url = http://www.cssn.cn/zhx/201608/t20160823_3172864.shtml | title = 克劳德·谢瓦莱：布尔巴基巨匠、数学结构代言人 | journal =《自然辩证法通讯》| year = 2016年 | volume = 20162 | page = 147-154 | author = 阎晨光}}</ref>。}}的創始人之一。

== 人生經歴 ==
=== 青少年時期 ===
1909年2月11日謝瓦萊生於[[南非|南非]]首府[[約翰內斯堡|約翰內斯堡]]<ref name = 'ziran'/>，謝瓦萊的父親阿貝·謝瓦萊（Abel Chevalley）是一名法國外交官，母親是瑪格麗特·謝瓦萊（Marguerite Chevalley）。克勞德·謝瓦萊的父母一同編寫了《簡明牛津法語詞典》（{{lang-en|Oxford Concise French Dictionary}}）<ref name = 'history' >{{cite web |url = http://www-history.mcs.st-andrews.ac.uk/Biographies/Chevalley.html | title = Claude Chevalley | author =  J. J. O'Connor and E. F. Robertson| work = University of St Andrews, Scotland | accessdate = 2018-07-14 |language = en}}</ref>。

謝瓦萊17歲就進入了[[巴黎高等師範學院|巴黎高等師範學院]]<ref name="Euclid">{{cite journal |author1=Dieudonné, Jean |coauthors=Tits, Jacques |title=Claude Chevalley (1909 - 1984) |journal=Bull. Amer. Math. Soc. (N.S.) |date=1987 |volume=17 |issue=1 |pages=1 - 7 |url=https://projecteuclid.org/download/pdf_1/euclid.bams/1183553960 |accessdate=2018-07-15 | language = en}}</ref>，從師於[[埃米爾·皮卡|埃米爾·皮卡]]。巴黎高等師範學院為四年學制，前兩年學生在綜合性大學學習，後兩年學生準備中學教師全國會考，考式通過者可獲得教師資格。謝瓦萊在1929年僅用了三年就畢業，並取得了中學高級教師職銜<ref name = 'ziran'/>。畢業之後，在法國國家科研究中心的資助下<ref name = ziran/>，謝瓦萊先後來到德國的[[漢堡大學|漢堡大學]]和[[馬爾堡大學|馬爾堡大學]]，分別在[[埃米爾·阿廷|埃米爾·阿廷]]和{{link-en|赫爾穆·特哈塞}}（{{lang-de|Helmut Hasse}}）的指導下做研究。1933年謝瓦萊獲得了[[巴黎大學|巴黎大學]]的數學博士學位{{notetag|謝瓦萊的博士畢業論文的標題是：''[http://www.numdam.org/issue/THESE_1934__155__365_0.pdf Sur la théorie du corps de classes dans les corps finis et les corps locaux]''。}}<ref name = 'history'/>。

=== 職業生涯 ===
1939年，謝瓦萊被邀請至美國[[普林斯頓高等研究院|普林斯頓高等研究院]]進行訪問。不久後[[二戰|二戰]]暴發，法國駐美國大使館建議他留在美國，謝瓦萊便留在[[普林斯頓大學|普林斯頓大學]]任教直至1948年。1949年至1955年，謝瓦萊在美國[[哥倫比亞大學|哥倫比亞大學]]擔任數學教授<ref name = 'Euclid'/>，並成為了美國公民<ref name = 'history'/>。謝瓦萊在美國期間指導過的學生包擴{{link-en|利昂·埃伦普赖斯|Leon Ehrenpreis}}和{{link-en|格爾哈德·霍克希爾德|Gerhard Hochschild}}。1953至1954年間，謝瓦萊在[[傅爾布萊特計畫|傳爾布萊特獎學金]]的支助下去日本訪學一年，並在多所日本大學裏舉行了系列講座。他講座的準入門檻很高，因此對大眾學生沒有構成吸引力，但有少數天份不錯的研究生在他的指導下做研究<ref name = 'Euclid'/>。

在哥倫比亞大學期間，謝瓦萊曾經申請[[巴黎大學|巴黎大學]]{{notetag|巴黎大學在英文中亦被稱作：the Sorbonne。}}空缺的一個系主任的職位，希望能回到法國，但他遇到了許多困難<ref name = 'history'/>。直到1955年{{notetag|也有來源表示謝瓦萊1949至1957年間在哥倫比亞大學任教，1957年才來到巴黎大學（見<ref name = 'history'/>）。}}謝瓦萊才成功回到了法國，在巴黎大學擔任教授。他在巴黎大學任教至1978年退休<ref name = 'Euclid'/>。

=== 數學之外的興趣 ===
除了數學之外，謝瓦萊對藝術和政治也有興趣，他曾是法國{{link-en|二十世紀三十年代非傳統主義組織|non-conformists of the 1930s}}（{{lang-en|non-conformists of the 1930s}}）中青年組織「新秩序」（{{lang-fr|Ordre Nouveau}}）的一員<ref name = 'ziran'/>。數學家{{link-en|皮埃爾·卡地亞|Pierre Cartier}}（{{lang-fr|Pierre Cartier}}）曾表示：「謝瓦萊曾是許多前衞政治、藝術組織的成員…數學是他人生中最重要的一個部分，但是他並不在數學和其餘的生活之間畫出任何界限。」<ref>{{cite journal |author1=Cartier, Pierre |title=Claude Chevalley |journal=Notices of the American Mathematical Society |date=1984 |volume=31 |page=775}}</ref>

== 學術研究 ==
1930至1940年間，謝瓦萊主要研究局部和全局[[類域論|類域論]]。1940年後謝瓦萊將研究方向轉向了[[李群|李群]]和[[代數幾何|代數幾何]]，1941年他發表的兩篇論文{{notetag|兩篇論文分別是「可解群的拓扑結構」和「李群一條性質的一個代數證明」。}}也標致着他轉向李群和代數幾何。謝瓦萊分別在1946年、1951年和1955年出版了三卷本的《李群理论》。謝瓦萊發表了他最有影響力的論文之一的「論某些單群」，論文中他研究的特殊有限群現在被稱為「[[謝瓦萊群|謝瓦萊群]]」。<ref name = 'ziran'/>

1955年謝瓦萊回到巴黎之後，在巴黎舉辦了「謝瓦萊討論班」（{{lang-fr|Séminaire Claude Chevalley}}），前三期分別在1956-1958年度、1958年和1958-1959年度舉行，主題分別為李代數的群的分類、[[周環|周環]]及其應用和皮卡簇。
