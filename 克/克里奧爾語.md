'''克里奥尔语'''（{{lang-en|Creole Language}}，或{{lang|en|Creole}}），一種穩定的[[自然語言|自然語言]]，由[[皮欽語|皮欽語]]進一步而成。其特徵為混合多種不同語言詞彙，有時也掺雜一些其他語言文法的一種語言，也稱為'''混成語'''或'''混合語'''。

== 成因 ==
克里奥尔语是由[[皮钦语|皮欽語]]所演變而來的，在有著說不同語言的人混居的地方，因為這些人說著不同的語言，為了溝通，而會發展出一種稱之為「[[洋泾浜英语|洋涇浜]]」式的溝通方式。而這種溝通方式通常沒有文法可言，而若這些操不同語言的人因為種種因素而長期定居，他們的子女在缺乏可供參考的文法的情況下而進行[[語言石化|語言石化]]過程，自行將[[皮欽語|皮欽語]]加上文法，而產生克里奥尔语。<ref>易六德,"二语语言石化的成因及其对大学英语教学的启示"[http://www.xzbu.com/9/view-2235098.htm],《云南教育·高等教育研究》,2010年第2期.</ref><ref>柏桦/靳炎,"关于语法项目石化现象的实证研究"{{cite web |url=http://jxcg.lzjtu.edu.cn/wy04/lw/6.pdf |title=存档副本 |accessdate=2013-07-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20140201224909/http://jxcg.lzjtu.edu.cn/wy04/lw/6.pdf |archivedate=2014-02-01 }},兰州铁道学院学报(社会科学版),第22卷第5期/2003年10月.</ref>

== 特徵 ==
儘管各地的克里奥尔语的單詞不盡相同，但是彼此之間的語法往往有相似之處。
