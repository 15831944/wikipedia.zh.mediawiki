'''克力同篇'''（Crito）為[[古希臘|古希臘]]哲學家[[柏拉圖|柏拉圖]]所写的一篇著名[[對話|對話]]，內容為[[蘇格拉底|蘇格拉底]]與其追随者[[雅典|雅典]]富人[[克力同|克力同]]的對話，兩人談及[[政治責任|政治責任]]的根源及本質。对话发生於[[蘇格拉底的申辯|蘇格拉底的申辯]]後，当时蘇格拉底因蠱惑和不虔诚罪名遭判死刑，克力同於此篇對話中試圖說服蘇格拉底逃離监禁流亡。

== 克力同關於逃離的论据 ==
# 雅典的人民將认为蘇格拉底的跟隨者不肯花錢解救蘇格拉底。
# 蘇格拉底在別的城市（包括[[色萨利|色萨利]]）可以得到支持，逃亡並不是一个坏的选择。
# 當蘇格拉底有機會逃脫時不逃脫，這是在背叛自己並幫助他的敵人。蘇格拉底挑選了“較簡單的路”，而非克力同所認為的勇敢榮耀高尚的路，那便是逃離這死刑。
# 蘇格拉底的家庭，他的妻子與三個孩子，將失去他們的父親，以及他能提供的教育。

== 蘇格拉底的回應 ==
# 大眾的意見對於他的決定並不重要，因為眾人作为一个整体是不明智的。
# 背叛他自己對於他的決定也並不重要，重要的是逃亡這件事情是否正当。
# 他不會對敵人屈服，逃離死刑代表蘇格拉底懼怕他們。
# 逃避死刑將會是對他孩子有不好的榜樣。

== 法律的论据 ==
後來，蘇格拉底提供了拟人化的雅典法律四条论据，来向克力同显示他不應該逃跑。
# 我們（雅典）是你的雙親
# 我們是你的撫養者（提供教育）
# 蘇格拉底同意尊從我們（这是[[社會契約論|社會契約論]]的早期表述）
# 无论蘇格拉底去哪里，將都會被視為腐敗的力量
這些也基于先前他與克力同達成共識的论据：对他人作恶永远不是对的，即使他们已对你作恶。

根据这一准则（蘇格拉底说他自己及其追随者会遵守），对他的城市作恶将是不正当的，即便城市对他作恶。如果蘇格拉底逃跑了，他将在显示，有足够钱的人就可以逃避任何法律，这样他将对养育他的城市作恶，就象对父亲作恶一样。
