'''克欽邦'''（{{lang-my|ကချင်ပြည်နယ်}}，[[克欽語|克欽語]]：Jingphaw Mungdaw），是[[缅甸|緬甸聯邦]]東北部的[[克欽族|克欽族]]自治邦。首府在[[密支那|密支那]]。設有[[密支那|密支那]]、[[八莫|八莫]]、[[葡萄縣|葡萄]]三縣及18個鄉鎮市。

東部與[[中國|中國]][[雲南省|雲南省]][[怒江傈僳族自治州|怒江傈僳族自治州]]和[[德宏傣族景頗族自治州|德宏傣族景頗族自治州]]接壤，北部與[[中國|中國]][[西藏自治區|西藏自治區]][[昌都市|昌都市]]相鄰，西部則與[[印度|印度]]接壤，南部則與[[撣邦|撣邦]]相鄰。克欽邦的[[南坎|南坎]]和[[中國|中國]][[雲南省|雲南省]][[瑞麗市|瑞麗市]]可以有貿易的往來，且不需簽證而自由通行。

世界上超过90%的翡翠产于[[缅甸|缅甸]]克钦邦[[密支那|密支那]]地区。翡翠传入中国是在明朝，[[明|明]]、[[清|清]]两代，翡翠工艺品在中国流行。从[[云南|云南]][[永昌府|永昌]](现保山)、[[腾冲|腾冲]]至[[缅甸|缅甸]]克钦邦的[[密支那|密支那]]一线是进行宝石贸易的商路，有“玉石路”、“宝井路”之称。

== 族群 ==
克欽邦主要族群有[[克欽族|克欽族]]（景頗族）、[[傈僳族|傈僳族]]、[[載瓦人|載瓦族]]、[[緬族|緬族]]、[[怒族|怒族]]、[[掸族|掸族]]以及漢人([[果敢族|果敢族]])及少量[[藏族|藏族]]人。中國雲南的少數民族[[景頗族|景頗族]]与克欽族是同一民族。男人喜穿黑色對襟上衣，下穿圍布或短褲並配刀於身上。婦女一般穿黑色短衫和花圍裙。

== 政治、軍事 ==
克欽邦是[[缅甸|緬甸聯邦]]內以[[克欽族|克欽族]]為主體的高度自治邦。目前[[克欽獨立軍|克欽獨立軍]]控制全邦，[[緬甸國防軍|緬甸國防軍]]正在進攻克欽邦，企圖摧毀克欽民族武裝力量。

== 歷史 ==
[[File:In_farthest_Burma_-_the_record_of_an_arduous_journey_of_exploration_and_research_through_the_unknown_frontier_territory_of_Burma_and_Tibet_(1921)_(14784320135).jpg|170px]]

當地最早的居民是來自雲南的泰人，在6世紀開始泰人開始在當地建[[勐|勐]]。

1947年緬甸各邦向[[英國|英國]]爭取獨立，同年[[翁山|翁山]]將軍依據[[彬龍協議|彬龍協議]]（{{Script/shn-Mymr|{{lang|my|ပင်လုံစာချုပ်}}}}）同意將[[克欽邦|克欽邦]]、[[撣邦|撣邦]]等地劃為自治區。1948年克欽邦成立，由[[密支那|密支那]]、[[八莫|八莫]]和[[葡萄縣|葡萄]]三区构成。1962年[[奈温|奈温]]废除聯邦宪法，[[克欽獨立軍|克欽獨立軍]]和[[克欽獨立組織|克欽獨立組織]]相應而起，除了[[密支那|密支那]]等主要城鎮和鐵路沿線外，[[克欽獨立軍|克欽獨立軍]]可說實際控制了克欽邦，對外貿易主要是走私[[玉|玉]]和[[毒品|毒品]]到[[中國大陆|中國大陆]]。

1994年與緬甸政府簽訂和平協定，允許[[克欽獨立組織|克欽獨立組織]]可以控制克欽邦，但仍有少數克欽邦獨立軍和克欽獨立組織的成員仍不滿意，因此目前克欽邦的政局仍不穩定。

== 地理 ==
緬甸最高峰[[開加博峰|開加博峰]]與最大湖[[印多吉湖|印多吉湖]]都在其境內，德奈卡河流域的戶拱平原和[[邁立開江|邁立開江]]流域的葡萄平原是重要的平原，是供給[[克欽人|克欽人]]生活重要的地區。

=== 地形 ===
克欽邦面積89041平方公里，38.66%是平原，61.34%是山區，平原面積大約有250萬英畝。大約有100萬英畝地是可種植的農田，可種植[[稻米|稻米]]、[[玉米|玉米]]、[[大麥|大麥]]、[[葡萄|葡萄]]等。

== 經濟 ==
主要經濟來源為[[農業|農業]]，農業以稻米為主，有70%的可耕地，其他作物為[[高粱|高粱]]、[[玉米|玉米]]、[[甘蔗|甘蔗]]、[[油菜|油菜]]等。除農業外，主要的經濟來源是靠克欽邦生產的翡翠和柚木。翡翠分布於漢巴、龍欽等地，柚木則分布於八莫、密支那。但由於最近柚木盜採相當嚴重，使克欽邦政府蒙受相當大的損失。
