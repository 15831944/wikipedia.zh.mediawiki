'''克蕾曼絲·波西'''（{{lang-fr|Clémence Poésy}} {{IPA-fr|kleˈmɑ̃s pɔeˈzi}}，{{bd|1982年|11月30日}}），法國女演員和時尚模特兒。她從舞台劇的童星出身，畢業於戲劇學系，1999年踏入電視和電影界。她的著名角色包括《[[哈利波特—火盃的考驗_(電影)|哈利波特—火盃的考驗]]》和《[[哈利波特—死神的聖物_(電影)|哈利波特—死神的聖物]]》1、2集中的[[花兒·戴樂古|花兒·戴樂古]]，《[[殺手沒有假期|殺手沒有假期]]》中的柯兒，以及《[[127小時|127小時]]》中的拉娜。她也曾在《[[花邊教主_(電視劇)|花邊教主]]》中演出[[花邊教主角色列表|伊娃·古柏]]一角。

== 早期生活 ==

克蕾曼絲·吉查德出生在[[巴黎|巴黎]][[默東|默東]]的西南郊區，她是巴黎演員/作家/劇場導演[[艾蒂安·吉查德|艾蒂安·吉查德]]（Étienne Guichard）和法語[[老師|老師]]的女兒，波西以母親娘家的姓 (Poésy) 作為她的藝名。<ref name=IMDb>[[:imdbname:0993242|Poésy profile at IMDb]]</ref> 她就讀[[另類學校|另類學校]]。克蕾曼絲在學校不是那種"受歡迎的女孩"，並形容自己為"有點奇怪"。<ref>{{cite web |url=http://www.interviewmagazine.com/film/clemence-poesy/ |title=Clemence Poesy |first=Olivier |last=Zahm |work=[[Interview_(magazine)|Interview]] |accessdate=2011-05-14}}</ref> 小時候，她的父親就訓練她演出，然後她在14歲時開始演出。{{Citation needed|date=March 2011}} 她妹妹[[玫蕾·波西·吉乍得|玫蕾·波西·吉乍得]]也是一個女演員。<ref name="scotsman">{{cite web |url=http://living.scotsman.com/features/Interview-Clemence-Posy-French-model.6294069.jp |title=Interview: Clemence Poésy, French model and actress |first=Alice |last=Wyllie |date=May 17, 2010 |work=[[The_Scotsman|The Scotsman]] |accessdate=May 16, 2011}}</ref>
