'''克恩顿州'''（{{lang-de|Kärnten}}，{{lang-sl|Koroška}}，{{lang-en|Carinthia}}），又译作'''克恩滕州'''或'''卡林西亚州'''，是[[奥地利|奥地利]]最南面的一个州。首府[[克拉根福|克拉根福]]。

== 名称 ==

德语名称Kärnten出自[[凯尔特语|凯尔特语]]“karantos”，意为友好的人，所以州名可以理解为“友善之地”。也有说法称州名来自居住在该州境内阿尔卑斯山地区的[[凯尔特人|凯尔特人]]部落“Carni”，第三种说法称该名得自[[伊利里亚语|伊利里亚语]]“karant”，意为「岩石、峭壁」。

== 地理 ==

[[File:Laengsee-kaernten.jpg|thumb]]

克恩顿州西面和东[[蒂罗尔州|蒂罗尔州]]接壤，北和东北面为[[萨尔茨堡州|萨尔茨堡州]]以及[[施泰尔马克州|施泰尔马克州]]，南面与[[斯洛文尼亚|斯洛文尼亚]]及[[意大利|意大利]]为邻。该州地理上被划分为上克恩顿，中克恩顿，下（西）克恩顿三部分。在下克恩顿的克拉根福盆地坐落着该州美丽的首府[[克拉根福|克拉根福]]，由于被众多的湖泊环绕，是一个著名的避暑胜地。

== 经济 ==

由于该州有20%的居民都从事于旅游业，因而失业率受季节的影响很大。建于克恩顿州最大河流drau的水电站解决了整个奥地利12%的电力供应。

== 人口 ==

总人口559,404人（2001年）。绝大多数为[[日耳曼|日耳曼]][[民族|民族]]。在南部地区居住着一些[[斯洛文尼亚|斯洛文尼亚]]人（2001年：12,544人）。少数民族权利问题（双语学校，双语路标）常常引发激烈争论。

== 历史 ==

公元7世纪时为一独立王国，拉丁文名称为“Carinthia”。788年至843年为[[查理曼|查理曼]]的[[法兰克帝国|法兰克帝国]]之一部。899年至927年属于[[巴伐利亚|巴伐利亚]]公爵，随后成为[[神圣罗马帝国|神圣罗马帝国]]直辖的领地。995年，阿达尔贝罗·埃本施泰因被封为卡林西亚边境总督，该家族统治卡林西亚地区至1122年，随后转归施蒂里亚公国。由于该家族男性后代绝嗣，1335年，该地区统治权转入[[哈布斯堡|哈布斯堡]]家族手中，称为卡林西亚公爵领地（Geschichte Kärntens， Duchy of Carinthia）。1804年该地区并入统一的[[奥地利帝国|奥地利帝国]]。[[第一次世界大战|第一次世界大战]]及[[第二次世界大战|第二次世界大战]]后，[[斯洛文尼亚|斯洛文尼亚]]及其后身[[南斯拉夫|南斯拉夫]]都曾试图吞并整个克恩顿地区，但遭到当地居民的抵制。

== 政治 ==

州立法机关是拥有36名议员的州[[议会|议会]]，每5年选举一次。州议会所在地是[[克拉根福|克拉根福]]。
执行机关是由州长领导的州政府。克恩顿州政府主要由七名成员组成：州长，两名副州长，四名顾问。克恩顿州是奥地利右翼的大本营。前州长（2005）是右翼著名领袖、原自由党（FPÖ），奥地利未来联盟（BZÖ）党魁[[约尔格·海德尔|海德尔]]。
