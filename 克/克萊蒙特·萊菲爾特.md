'''克萊蒙特·萊菲爾特'''（{{lang-fr|Clément Lefert}}；{{bd|1987年|9月26日}}）是一位已經退役的[[法國|法國]]游泳運動員<ref>{{cite web |url=http://www.swimmingworldmagazine.com/lane9/news/College/31969.asp |title=French Olympic Champion Clement Lefert Retires. |deadurl=yes |archiveurl=https://archive.is/20130203071616/http://www.swimmingworldmagazine.com/lane9/news/College/31969.asp |archivedate=2013-02-03 }}</ref><ref name="usctrojans1">{{cite web |url=http://www.usctrojans.com/sports/m-swim/mtt/lefert_clement00.html |title=Clement Lefert Profile - University of Southern California Official Athletic Site |publisher=Usctrojans.com |date=30 July 2012 |accessdate=14 August 2012 |deadurl=yes |archiveurl=https://web.archive.org/web/20120802000932/http://www.usctrojans.com/sports/m-swim/mtt/lefert_clement00.html |archivedate=2012年8月2日 }}</ref>。他代表法國在2012年倫敦奧運會上奪得4乘100米自由泳的金牌和4乘200米自由泳的銀牌。
