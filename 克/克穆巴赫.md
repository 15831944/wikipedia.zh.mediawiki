'''克穆巴赫'''是[[德国|德国]]最受欢迎的[[啤酒|啤酒]]品牌之一。克穆巴赫口味圆滑香甜，并且有特殊的[[啤酒花|啤酒花]]气味。 

== 历史 ==
1803年，[[波恩哈德·沙德伯格|波恩哈德·沙德伯格]](Bernhard Schadeberg)创建了克穆巴赫品牌，其后该品牌所有权在沙德伯格家族内流传至今。

== 位置 ==
[[File:Kreuztal_Brauerei_Krombacher.jpg|thumb]]
[[克穆巴什|克穆巴什]]位于[[德国|德国]][[席根|席根]](Siegen)附近[[克劳茨陶|克劳茨陶]]的小城，位于被部分德国人称做[[席根兰|席根兰]](Siegerland)的地区。席根兰位于德国[[北莱茵-威斯特法伦|北莱茵-威斯特法伦]]的东南部。小城克穆巴什位于[[红发山脉|红发山脉]]脚下。这里正是酿酒所用泉水的产地。

== 产品 ==
克穆巴赫生产五种饮料产品：
*克穆巴赫比尔森(The Krombacher Pils)，传统的比尔森啤酒；
*莱茵醇(The Rhenania Alt)，黑啤酒；
*克穆巴赫Radler(The Krombacher Radler)，由50%比尔森和50%柠檬汁调配的酒；
*克穆巴赫无酒精(The Krombacher non-alcoholic)，无[[酒精|酒精]]啤酒。
*克穆巴赫Cab(The Krombacher Cab)，由比尔森，[[可乐|可乐]]和[[火龙果|火龙果]]香料调成的酒。
