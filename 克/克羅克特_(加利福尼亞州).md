'''克羅克特'''（{{lang-en|Crockett}}）是位於[[美國|美國]][[加利福尼亞州|加利福尼亞州]][[康特拉科斯塔縣_(加利福尼亞州)|康特拉科斯塔縣]]的一個[[人口普查指定地區|人口普查指定地區]]。

== 地理 ==
克羅克特的座標為{{coord|38|03|09|N|122|12|47|W|type:city_region:US-AL_source:GNS-enwiki|display=inline}}<ref name=GNIS />，而該地最高點為海拔高度39米（即128英尺）。

== 人口 ==
根據[[2010年美國人口普查|2010年美國人口普查]]的數據，克羅克特的面積為2.746平方千米，當中陸地面積為2.746平方千米，而水域面積為0.000平方千米。當地共有人口3094人，而人口密度為每平方千米{{formatnum: {{rounddown|{{#expr: 3094 / 2.746}}}} }}人。<ref name="GR8">{{cite web|url=http://factfinder2.census.gov|publisher=[[美国人口调查局|美国人口调查局]]|accessdate=2011-05-14|title=American FactFinder}}</ref>
