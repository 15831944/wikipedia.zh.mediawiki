'''玛丽亚·克里斯蒂娜·纳沃纳·鲁伊斯'''（{{lang-es|María Cristina Narbona Ruiz}}，{{bd|1951年|7月29日||}}）是[[西班牙工人社会党|西班牙工人社会党]]女性[[政治人物|政治人物]]。2004年4月至2008年4月[[何塞·路易斯·罗德里格斯·萨帕特罗|萨帕特罗]]内阁期间担任环境大臣。2017年6月就任西班牙工人社会党主席。

==经历==
1951年出生于[[马德里|马德里]]<ref>{{cite web |title=Soy más de leer y escribir, utilizo muy poco el ordenador|url=https://web.archive.org/web/20060623212349/http://www.elpais.es/articulo/elpportec/20060322elpepunet_2/Tes/internet/Soy/leer/escribir/utilizo/poco/ordenador |publisher=[[国家报|El País]] |accessdate=2019-02-24 |language=es |date=2006-03-22}}</ref>。12岁时和父母搬到意大利[[罗马|罗马]]。1975年获得[[罗马大学|罗马大学]][[经济学|经济学]]学位<ref>{{Cite news|url=https://www.elconfidencial.com/espana/2018-11-18/narbona-psoe-doctorado-licenciatura-italia_1648290/|title=La presidenta del PSOE hace pasar una licenciatura italiana por doctorado|work=[[机密报|El Confidencial]]|access-date=2018-11-19|language=es}}</ref><ref name="bio">{{cite web|url=http://www.el-mundo.es/documentos/2004/04/espana/nuevogobierno/cristinanarbona.html |title=Cristina Narbona|publisher=[[世界报_(西班牙)|El Mundo]]|language=es|access-date=2019-02-19}}</ref>。之后，在[[塞维利亚大学|塞维利亚大学]]担任教师，也在[[安达卢西亚政府|安达卢西亚政府]]担任过一些职位<ref name="bio2">{{cite web|url=http://www.todopolitica.com/biografias/?n=cristina_narbona|title=Biography of Cristina Narbona|publisher=todopolitica.com|language=es|accessdate=2019-02-24}}</ref>。1993年加入西班牙工人社会党<ref name="bio"/>。随后是西班牙众议院的议员<ref name=cv/>。在2004年至2008年萨帕特罗内阁期间担任环境大臣<ref name="bio"/>。2008年离任后担任[[巴黎|巴黎]][[经济合作与发展组织|经济合作与发展组织]]西班牙大使<ref name=elpais>{{cite web |author=Rafael Méndez |title=Narbona, heroína de 'Time' |url=https://elpais.com/diario/2008/10/01/agenda/1222812003_850215.html |accessdate=2019-02-24 |location=Madrid |language=es|publisher=El País |date=2008-10-01}}</ref>，同年被《[[时代_(杂志)|时代]]》杂志评为“正在改变世界的环境英雄”<ref name=elpais/>。

从1998年起，她与[[何塞·博雷利·丰特列斯|何塞·博雷利·丰特列斯]]保持男女朋友关系<ref name="bio"/>。2018年，两人结婚<ref>{{cite web |author=A. Mellado |title=La «boda discreta» de Josep Borrell y Cristina Narbona |url=https://www.abc.es/estilo/gente/abci-josep-borrell-y-cristina-narbona-casan-secreto-tras-decadas-juntos-201811221222_noticia.html |publisher=[[阿贝赛报|ABC]] |accessdate=2019-02-24 |language=es |date=2018-11-22}}</ref>。2012年12月28日起任[[西班牙核安全理事会|西班牙核安全理事会]]顾问<ref name=cv>{{cite web |title=Cristina Narbona |url=https://quienmanda.es/people/cristina-narbona |website=quienmanda.es |accessdate=2019-02-24 |language=es}}</ref>。

2017年6月被任命为[[西班牙工人社会党|西班牙工人社会党]]主席<ref>{{cite web |title=Cristina Narbona será la nueva presidenta del PSOE |url=https://www.lavanguardia.com/politica/20170610/423301300052/cristina-narbona-presidenta-psoe.html |publisher=[[先锋报|La Vanguardia]] |accessdate=2019-02-24 |language=es |date=2017-06-10}}</ref>。
==政治立场==
克里斯蒂娜·纳沃纳是坚定的环保主义者<ref>{{cite web |author=Manuel Ansede |title=Es necesario que las leyes aprobadas no se queden en papel mojado |url=https://web.archive.org/web/20081026164542/http://www.publico.es/ciencias/147736/es/necesario/leyes/aprobadas/queden/papel/mojado |website=Publico.es |accessdate=2019-02-24 |location=[[桑坦德|Santander]] |language=es |date=2008-09-02}}</ref>。她亦是[[核動力|核動力]]能源应用的反对者，支持西班牙逐步关停现有的[[核电站|核电站]]，坚持贯彻《[[京都议定书|京都议定书]]》约定内容<ref name="bio2"/>。

她在2002年接受{{tsl|en|Telemadrid||Telemadrid}}电视采访时表示，很多西班牙年轻人已经难以接受过高的[[住宅|住宅]]价格，并呼吁政府应在住房方面提供更多援助<ref>{{cite web|url=http://www.telemadrid.es/contenidos/html/elcirculo/pagina_cristinanarbona.htm|title=Interview with Cristina Narbona|archiveurl=https://web.archive.org/web/20070814141118/http://www.telemadrid.es/contenidos/html/elcirculo/pagina_cristinanarbona.htm |archivedate=2007-08-14|publisher=Telemadrid|date=2002|language=es}}</ref>。
