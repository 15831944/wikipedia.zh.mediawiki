{{日本作品|網路勝利組|ネト充のススメ}}是一部由日本漫畫家黑曜燐創作的網路遊戲題材漫畫作品，自2013年10月起開始連載，2015年時因作者身體不適而停刊，2017年改編為電視動畫，自2017年10月9日起[[東京都會電視台|東京都會電視台]]首播。

2018年7月1日，因作者身体恢复没有进展，Comico宣布本作正式完结。<ref>{{cite web|url=http://www.acgdoge.net/archives/28225|title=作者身体恢复没有进展，《网络胜利组》原作漫画决定完结|date=2018-07-01|publisher=ACGdoge}}</ref>
== 故事簡介 ==
辭掉工作成為繭居族的盛岡森子，因原本遊玩的線上遊戲停止營運，而後在新找到的線上遊戲「Fruits de Mer」中化身為玩家「林」，並遇見了化身為「莉莉」的玩家櫻井優太，故事從此展開。

== 登場人物 ==
; {{nihongo|盛岡森子|盛岡森子（もりおかもりこ）|聲：[[能登麻美子|能登麻美子]]<ref name="character">{{cite web|url=http://netoju.com/menu/character|title=CHARACTER|work=電視動畫《網路勝利組》官方網站|language=ja}}</ref>}}
: 本作的女主角，三十歲的家裡蹲，因為在職場上有不好的遭遇而選擇離開，用存了11年的錢當老本在家裡玩網路遊戲，在離開工作前是十分優秀的人。在離開工作後，打扮不修邊幅，但其實也是個美人。
: 在遊戲中的角色是'''{{nihongo|「林」|林|聲：[[鈴木崚汰|鈴木崚汰]]<ref name="character"/>}}'''是男性騎士，被戲稱「殘念系帥哥」，在公會「@home party」裡因紫丁香謠傳，被認為是正在找工作的大學生。在公會隊伍裡擔任主坦。另一個遊戲「Nanter SG」的角色是'''{{nihongo|由紀|Yuki|聲：能登麻美子}}'''。
: 在知道'''莉莉'''是優太後，和優太保持著特殊的關係。

; {{nihongo|櫻井優太|桜井優太（さくらいゆうた）|聲：[[櫻井孝宏|櫻井孝宏]]<ref name="character"/>}}
: 本作的男主角，二十八歲，金髮碧眼的英國混血兒，在梅崎商事工作的精英。
: 在遊戲中的角色是'''{{nihongo|「莉莉」|リリィ|聲：[[上田麗奈|上田麗奈]]<ref name="character"/>}}'''是女性法師。在公會「@home party」隊伍裡擔任支援。另一個遊戲「Nanter SG」的角色是'''{{nihongo|哈斯|Harth|聲：櫻井孝宏}}'''。
: 在知道'''林'''是森子後，和森子保持著特殊的關係。

; {{nihongo|小岩井誉|小岩井誉（こいわいほまれ）|聲：[[前野智昭|前野智昭]]<ref name="character"/>}}
: 櫻井優太的同事兼前輩，三十二歲，曾經因工作而和盛岡森子通話過，並稱對方為「小盛森」，社交能力極高。原本不是網路遊戲玩家，在森子的帶領下接觸網路遊戲。在遊戲中的角色是'''{{nihongo|「春海」||聲：前野智昭}}'''是高大壯碩的女性。

; {{nihongo|藤本和臣|藤本和臣（ふじもとかずおみ）|聲：[[寺島拓篤|寺島拓篤]]<ref name="character"/>}}
: 在便利商店邊打工邊找工作的大學生。在遊戲中的角色是'''{{nihongo|「神戶」|カンベ|聲：[[中村悠一|中村悠一]]<ref name="character"/>}}'''是盛森所屬的公會@home party的會長，職業是刺客。

; {{nihongo|紫丁香|ライラック|聲：[[相坂優歌|相坂優歌]]<ref name="character"/>}}
: 現實中是一名女子大學生。是公會@home party的成員之一，在遊戲中是有貓耳的咒術師，職業是魔法型咒術師。似乎对神户有好感。

; {{nihongo|高橋康明|}}
: 一名通勤上班族。在遊戲中的角色是'''{{nihongo|「波可太郎」|ぽこたろう|聲：寸石和弘<ref name="character"/>}}'''，所屬於公會@home party，職業是法師。

; {{nihongo|高橋夏生|}}
: 高橋康明的妻子，家庭主婦。在遊戲中的角色是'''{{nihongo|「希美拉達」|ヒメラルダ|聲：八木隆典<ref name="character"/>}}'''，所屬於公會@home party，職業是小丑。

; {{nihongo|Nico||聲：藤森多哉}}
: 公會@home party的成員之一，職業是魔法使。
