'''網走湖'''（{{jpn|j=網走湖|hg=あばしりこ}}），是一個橫跨[[日本|日本]][[北海道|北海道]][[網走市|網走市]]及[[網走郡|網走郡]][[大空町|大空町]]的[[湖泊|湖泊]]。是[[網走国定公園|網走国定公園]]的一部分。在約6,000年前，網走湖曾是鄂霍次克海的海灣。隨著海面後退和地盤上升，約在1200年前時，網走湖由海灣變為湖泊。

== 地理 ==
位於北海道的東北部。[[網走川|網走川]]由湖的南方流入，匯集了[[女滿別川|女滿別川]]等河流之後，再由湖的東北部流出，流入[[鄂霍次克海|鄂霍次克海]]。東岸的半島稱為呼人半島。

湖面的高度與海水同高，因此滿潮時海水會由下游逆流進入，所以在湖底聚集了[[鹽度|鹽度]]較高的水，形成了無氧層。但是在降水量較多的時期，因為湖面較高，抑制了海水的流入，因此湖水的鹽分濃度較低，無氧層也會變得較深。

== 利用 ==
網走湖[[競艇|競艇場]]有[[日本競艇協會網走湖|日本競艇協會網走湖]]公認的1000米賽程，每到夏天，日本全國的競艇選手會聚集到此處進行練習。

冬季湖面會結冰，釣客在此處垂釣[[西太公魚|西太公魚]]。網走湖的[[養殖漁業|養殖漁業]]發達，除了[[蜆|蜆]]、[[鯉魚|鯉魚]]、西太公魚的養殖之外，還進行[[鮭魚|鮭魚]]、[[鱒魚|鱒魚]]的幼魚培養。

== 生態 ==
周圍的濕地有[[日本榿木|日本榿木]]和[[水芭蕉|水芭蕉]]的群落生長，特別是在大空町一帶的湖岸的「女滿別濕性植物群落」，被指定為國家的[[天然紀念物|天然紀念物]]。
