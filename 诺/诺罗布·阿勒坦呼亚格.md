'''诺罗布·阿勒坦呼亚格'''（{{lang-mn|Норовын Алтанхуяг}}；{{bd|1958年|1月20日|||catIdx=A}}）[[蒙古族|蒙古族]]，今[[蒙古国|蒙古国]][[乌布苏省|乌布苏省]][[乌兰固木|乌兰固木]]人，[[蒙古国|蒙古国]]政治家<ref name=zgxww>[http://www.chinanews.com/gj/2012/07-15/4033326.shtml 蒙古民主党主席阿勒坦呼亚格获蒙古国总理提名，中国新闻网，2012年07月15日]</ref>。

==生平==
1958年1月20日，阿勒坦呼亚格生于蒙古[[乌布苏省|乌布苏省]][[乌兰固木|乌兰固木]]。1976年毕业于乌布苏省乌兰固木十年制中学。1981年毕业于[[蒙古国立大学|蒙古国立大学]]生物、物理教师专业。1981年到1990年，任[[蒙古国立大学|蒙古国立大学]]教师。1990年起，历任[[蒙古社会民主党|蒙古社会民主党]]书记、总书记。1996年到2000年，当选[[国家大呼拉尔|国家大呼拉尔]]委员。1998年到1999年，任蒙古国农牧业与工业部长，2004年到2006年，任蒙古国财政部长。2008年至2012年，当选为[[国家大呼拉尔|国家大呼拉尔]]委员，并担任蒙古国第一副总理<ref name=zgxww/><ref name=zaluu>{{Cite web |url = http://www.zaluu.com/read/dg6geff4 |title = Н.Алтанхуяг: Намын даргын сунгаанд оролцох шийдвэрийг тулгуу гаргалаа |author =  |date =2016-12-19  |publisher = ZALUU |language =  |accessdate =  |quote =  }}</ref>。2001年至2003年，任[[民主党_(蒙古)|民主党]]秘书长<ref name=zaluu/>。2008年至2014年，任民主党主席。2012年至2016年，当选为国家大呼拉尔委员<ref name=zgxww/><ref name=zaluu/>。

2012年6月28日，民主党在国家大呼拉尔选举中获得相对多数席位。同年7月底，民主党、[[公民意志绿党|公民意志绿党]]、[[蒙古人民革命党_(2010年)|蒙古人民革命党]]、[[蒙古民族民主党|蒙古民族民主党]]组成的“正义联盟”签订了联合组阁协议。[[蒙古人民党|蒙古人民党]]则成为反对党。2012年8月9日，国家大呼拉尔全会通过决议，任命民主党主席、国家大呼拉尔委员阿勒坦呼亚格为新一届政府总理<ref>[http://world.people.com.cn/n/2012/0809/c1002-18708925.html 蒙古国议会任命民主党主席阿勒坦呼亚格为新总理，人民网，2012年08月09日]</ref>。任至2014年<ref name=zaluu/>。

2016年，曾参加民主党主席选举，但未能当选<ref name=zaluu/>。
