'''诺托哈丁格罗机场'''（[[印尼语|印尼语]]：Bandar Udara Notohadinegoro，{{airport codes|JBB|WARE}}）是[[印度尼西亚|印度尼西亚]][[东爪哇省|东爪哇省]][[任抹县|任抹县]]的一座机场，主要服务该县的县治[[任抹|任抹]]。任抹机场的开通将泗水至任抹的旅程从4-7小时缩短到30分钟<ref name="busbandara"></ref>。机场附近有几座火山，火山活动会对机场运作造成影响，例如2015年[[拉翁火山|拉翁火山]]喷发导致诺托哈丁格罗机场一度被关闭<ref>{{cite web |url=http://www.bbc.com/zhongwen/simp/world/2015/07/150717_indonesia_volcanos_airports |title=开斋节前夕印尼火山灰造成机场关闭 |last= |first= |date=2015-07-17 |website=www.bbc.com |publisher=BBC中文网 |access-date=2017-11-11 |quote=}}</ref><ref>{{cite web |url=http://news.seehua.com/?p=72600 |title=印尼机场重开　大部分航线恢復 |last= |first= |date=2015-07-11 |website=news.seehua.com |publisher=诗华日报 |access-date=2017-11-11 |quote=}}</ref>。

2017年，[[佐科·维多多|佐科·维多多]]总统表示将拨款3,700亿盾，延长任抹机场的跑道，工程将在2018年开始，工期两年<ref name="任抹机场加长跑道"></ref>。扩建工程完成后，2019年任抹机场的旅客量将达到36万人次，每日接待旅客1,000人次，每日至少3架中型飞机起降<ref name="扩建工程"></ref>。

==历史==

诺托哈丁格罗机场始建于2003年，2005年机场落成<ref name="busbandara">{{cite web |url=http://www.busbandara.com/info-bus-damri-bandara-jember-notohadinegoro/ |title=INFO BUS DAMRI BANDARA JEMBER NOTOHADINEGORO |last= |first= |date= |website=www.busbandara.com |publisher=BusBandara |access-date=2017-11-11 |quote=}}</ref>，跑道长1,200米，前[[印尼总统|印尼总统]][[阿卜杜拉赫曼·瓦希德|阿卜杜拉赫曼·瓦希德]]到访任抹后便不再使用。2008年将跑道延长到1,560米，曾短暂使用，之后再次闲置。2014年7月16日，[[加鲁达印尼航空|加鲁达印尼航空]]开通从[[泗水_(印尼)|泗水]]飞到任抹的航班，使用机型为[[ATR_72#ATR_72-600型|ATR 72-600]]，最初每天仅有1个班次，后来增加到2班<ref>{{cite web |url=http://www2.jawapos.com/baca/artikel/9008/Penerbangan-JemberSurabaya-Dua-Kali-Sehari |title=Penerbangan Jember–Surabaya Dua Kali Sehari |last= |first= |date=2014-11-10 |website=www2.jawapos.com |publisher=JAWA POS |access-date=2017-11-11 |quote=}}</ref><ref>{{cite web |url=http://travel.kompas.com/read/2014/12/30/1439000/Garuda.Tambah.Penerbangan.Jember-Surabaya.Jadi.Dua.Kali.Sehari |title=Garuda Tambah Penerbangan Jember-Surabaya Jadi Dua Kali Sehari |last= |first= |date=2014-12-30 |website=travel.kompas.com |publisher=KOMPAS |access-date=2017-11-11 |quote=}}</ref>。2015年曾有计划在任抹机场建立一个飞行员学校<ref>{{cite web |url=http://beritajatim.com/ekonomi/228598/sekolah_pilot_banyuwangi_bangun_settle_base_di_jember.html#.VML_sq0RVoO |title=Sekolah Pilot Banyuwangi Bangun Settle Base di Jember |last= |first= |date=2015-01-14 |website=beritajatim.com |publisher=beritajatim |access-date=2017-11-11 |quote=}}</ref>。2017年8月，[[飞翼航空|飞翼航空]]也开通了飞到任抹的航班。<ref name="任抹机场加长跑道">{{cite web |url=http://www.guojiribao.com/shtml/gjrb/20170816/330270.shtml |title=总统：任抹机场加长跑道 |last= |first= |date=2017-08-16 |website=www.guojiribao.com |publisher=东爪哇新闻网任抹讯 |access-date=2017-11-11 |quote=}}</ref><ref>{{cite web |url=http://www.guojiribao.com/shtml/gjrb/20140717/157577.shtml |title=任抹Notohadinegoro机场开始运作 |last= |first= |date=2017-07-17 |website=www.guojiribao.com |publisher=国际日报 |access-date=2017-11-11 |quote=}}</ref>

==设施==

诺托哈丁格罗机场总占地面积120公顷，有一条1645米长x30米宽的跑道，另有141×18米的滑行道、68×96米的停机坪，以及920平方米的航站楼。未来扩建工程分为两个阶段，2018年扩建停机坪以容纳更大的飞机、研究延长跑道等以支撑机场运作；2019年开工的工程将把跑道扩建至2250米x45米，以供[[波音737|波音737]]客机起降，扩建航站楼及其他设备工程，另外修建5100米长x45米宽的机场检查道路，还将采购机场消防及救护服务车队。在这之后跑道又将再次被延长，达到2,500米的长度，另外还要修建朝觐者出发机场辅助设备。<ref name="扩建工程">{{cite web |url=http://www.shangbaoindonesia.com/indonesia-finance/%E4%BB%BB%E6%8A%B9%E5%8E%BF%E6%9C%BA%E5%9C%BA%E6%89%A9%E5%BB%BA%E5%B7%A5%E7%A8%8B%E7%AB%A3%E5%B7%A5%E5%90%8E%E4%BA%A4%E9%80%9A%E9%83%A8%E9%95%BF%E5%86%80%E6%97%85%E5%AE%A2%E9%87%8F%E5%A2%9E36%E4%B8%87.html |title=任抹县机场扩建工程竣工后 交通部长冀旅客量增36万人次 |last= |first= |date=2017-08-25 |website=www.shangbaoindonesia.com |publisher=印度尼西亚商报 |access-date=2017-11-11 |quote=}}</ref><ref>{{cite web |url=https://qiandaoribao.com/news/99614 |title=交通部长赴东爪哇视事 拟扩建任抹机场 |last= |first= |date=2017-08-22 |website=qiandaoribao.com |publisher=千岛日报 |access-date=2017-11-11 |quote=}}</ref>
