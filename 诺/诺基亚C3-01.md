'''诺基亚 C3-01'''（'''Nokia C3-01'''），也被称为'''C3 Touch and Type'''，是[[诺基亚|诺基亚]]出品的一款[[诺基亚C系列|C系列]]的[[触键双控|触键双控]]手机，使用的是基于[[Series_40|Series 40第六版 功能包1]]平台的[[Nokia_OS|Nokia OS]]系统。

诺基亚 C3-01分为两个版本，对应的编号分别是'''RM-640'''和'''RM-776'''(RM-776也被一些人叫做'''C3-01.5'''），其中后者的硬件配置比前者更高，两者对比的简表：<ref>{{Cite web |url= http://www.developer.nokia.com/Devices/Device_specifications/C3-01/ |title= Device details -- Nokia C3-01 |language= 英语 |accessdate= 2012年8月30日 |quote= There are 2 different hardware versions of the Nokia C3-01. The main differences between the versions are:<br/>RM-776: 1 GHz processor, 256 MB of ROM memory, 128 MB of RAM memory, Nokia Browser and Series 40 Web Apps support comes with the device firmware by default.<br/>RM-640: 680 MHz processor, 128 MB of ROM memory, 64 MB of RAM memory, WebKit Open Source Browser as the default web browser (Nokia Browser is available as a downloadable addon from Nokia Store). |deadurl= yes |archiveurl= https://web.archive.org/web/20111106031632/http://www.developer.nokia.com/Devices/Device_specifications/C3-01/ |archivedate= 2011年11月6日 }}</ref>

== 诺基亚 C3-01 黄金版 ==
'''诺基亚 C3-01 黄金版'''是豪华版的C3-01，其配置和RM-776版相同，不过它的外壳是金黄色的。黄金版并没有在中国上市。
