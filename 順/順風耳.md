'''順風耳'''，表意指「能夠聽到隨風而來之聲音的耳朵」。在中文語境中，順風耳也於被用於形容聽力好或者知道很多八卦傳聞的人。

於現代可以分為幾個解釋：
# 著名[[海神|海神]][[媽祖|媽祖]]的部將之一，以其無遠弗屆的聽力，為媽祖偵聽世上一切，另一位名喚[[千里眼|千里眼]]。
# 中國[[四大名著|四大名著]]之一《[[西遊記|西遊記]]》中所記載過的的一位[[天庭|天庭]][[神祇|神祇]]，與[[千里眼|千里眼]]齊名。
# 中國[[民間故事|民間故事]]《[[十兄弟|十兄弟]]》中十位各自擁有[[特異功能|特異功能]]的兄弟中，二兄的外號。
# 佛教[[神通|神通]]的一種，名為「天耳通」
# {{fact|[[特異功能|特異功能]]中常見的一種，能力擁有者可以聽到遙遠距離的聲音、微細聲音，或者[[人類聽力範圍|人類聽力範圍]]（[[頻率|頻率]]由20[[赫茲_(單位)|赫茲]]至20000赫茲）外的[[機械振動|機械振動]]，不過並不常見}}。

== 金精水精 ==
{{主條目|媽祖}}
在民間的神話傳說中，替媽祖察聽世情的兩大駕前護衛神分別為<br>
左手持方天畫戟，右手舉至額前做遠視狀的千里眼（又稱金精將軍）<br>
以及左手持月眉斧頭，右手舉至側耳作聽音狀的順風耳（又稱水精將軍）
