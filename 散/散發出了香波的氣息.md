《'''散發出了香波的氣息'''》（{{jpn|j=シャンプーの匂いがした|hg=シャンプーのにおいがした|rm=Shampoo no nioi ga shita}}）是日本女子[[日本偶像|偶像]]組合[[22/7_(企劃)|22/7]]的第2張[[單曲|單曲]]，於2018年4月11日由[[日本索尼音樂娛樂|日本索尼音樂娛樂]]發行。

== 概要==
單曲分為Type-A、Type-B和通常盤。其中，《散發出了香波的氣息》和《人格崩壞》收錄於所有版本。Type-A收錄了《散發出了香波的氣息》的音樂錄影帶、《溫柔的記憶》和交易卡（Type-A版本，11款中的其中一款）和活動參加券，Type-B收錄了《只能呼喊的青春》、影片「Documentary of 22/7」和交易卡（Type-B版本11款中的其中一款）；通常盤收錄了《穿梭巴士》<ref>{{cite web |url=http://www.nanabunnonijyuuni.com/discography/archive/?SRCL-9713 |language=ja |script-title=ja:シャンプーの匂いがした【初回仕様限定盤 Type-A】 |publisher=[[22/7_(企劃)|22/7]] |accessdate=2018-04-21 |deadurl=no |archiveurl=https://web.archive.org/web/20180420150731/http://www.nanabunnonijyuuni.com/discography/archive/?SRCL-9713 |archivedate=2018-04-20 }}</ref><ref>{{cite web |url=http://www.nanabunnonijyuuni.com/discography/archive/?SRCL-9715 |language=ja |script-title=ja:シャンプーの匂いがした【初回仕様限定盤 Type-B】 |publisher=22/7 |accessdate=2018-04-21 |deadurl=no |archiveurl=https://web.archive.org/web/20180420150731/http://www.nanabunnonijyuuni.com/discography/archive/?SRCL-9715 |archivedate=2018-04-20 }}</ref><ref>{{cite web |url=http://www.nanabunnonijyuuni.com/discography/archive/?SRCL-9717 |language=ja |script-title=ja:シャンプーの匂いがした |publisher=22/7 |accessdate=2018-04-21 |deadurl=no |archiveurl=https://web.archive.org/web/20180420150731/http://www.nanabunnonijyuuni.com/discography/archive/?SRCL-9717 |archivedate=2018-04-20 }}</ref>。

2018年1月23日，22/7在[[SHOWROOM|SHOWROOM]]上宣佈推出單曲<ref>{{cite web |url=https://natalie.mu/music/news/266282 |language=ja |script-title=ja:22/7、2ndシングル「シャンプーの匂いがした」発売決定 |date=2018-01-23 |publisher=[[natalie_(網站)|natalie]] |accessdate=2018-04-21 |deadurl=no |archiveurl=https://web.archive.org/web/20180228212524/https://natalie.mu/music/news/266282 |archivedate=2018-02-28 }}</ref>。2月3日，由[[堀口悠紀子|堀口悠紀子]]設計的單曲造型正式公開<ref>{{cite web |url=https://www.m-on-music.jp/0000252271/ |language=ja |script-title=ja:制服のデザインも一新！ 22/7（ナナブンノニジュウニ）、新曲「シャンプーの匂いがした」新ビジュアル公開 |date=2018-02-03 |publisher={{Link-ja|M-ON! == 反應 ==
在《[[告示牌_(雜誌)|告示牌]]》Top Singles Sales榜中，單曲首週銷量為12,764張，排第8位<ref>{{cite web |url=http://www.billboard-japan.com/charts/detail?a=sales&year=2018&month=04&day=23 |language=ja |script-title=ja:Billboard Japan Top Singles Sales |date=2018-04-18 |publisher=[[告示牌_(雜誌)|告示牌]] |accessdate=2018-04-21 |deadurl=no |archiveurl=https://web.archive.org/web/20180420150731/http://www.billboard-japan.com/charts/detail?a=sales&year=2018&month=04&day=23 |archivedate=2018-04-20 }}</ref>，[[日本熱門100金曲榜|日本熱門100金曲榜]]則排第18位<ref>{{cite web|url=http://www.billboard-japan.com/d_news/detail/62391/2 |language=ja|script-title=ja:【ビルボード】NGT48「春はどこから来るのか？」11.6万枚を売り上げ総合首位、米津玄師「Lemon」9週連続ダウンロード1位 |date=2018-04-18 |publisher=告示牌 |accessdate=2017-12-20}}</ref>。在[[Oricon公信榜|Oricon公信榜]]中，單曲首週銷量是9,691張，排第8位<ref>{{cite web |url=https://www.oricon.co.jp/rank/js/w/2018-04-23/ |language=ja |script-title=ja:週間　CDシングルランキング2018年04月23日付 |publisher=Oricon |accessdate=2018-04-21 |deadurl=no |archiveurl=https://web.archive.org/web/20180420075446/https://www.oricon.co.jp/rank/js/w/2018-04-23/ |archivedate=2018年4月20日 }}</ref>。
