'''辟穀'''（又寫作-{'''闢穀'''}-<!-- {{Citation needed}} begin -->{{fix |link=Wikipedia:列明来源|text=来源请求|class=Template-Fact }}<!-- {{Citation needed}} end -->，英文：'''inedia'''、'''breatharianism'''），意为“吸食空气”。辟穀者称作'''inediate'''或'''breatharian'''）是道教中的[[道士|道士]]用{{le|道教饮食|Taoist diet|饮食}}来修身的一种方法，通常在辟穀期间，不吃任何[[食物|食物]]、只喝[[水|水]]，但是要搭配调节[[气脉|气脉]]，须做[[太极|太极]]、[[瑜伽|瑜伽]]、[[禅定|禅定]]，才能保证身体没有饥饿感。另外，在除了中国[[道教|道教]]区域的全世界各地很多的[[宗教|宗教]]中也存在相似的修身方法，比如[[藏传佛教|藏传佛教]]、[[印度教|印度教]]；最后，[[基督教|基督教]]和[[伊斯兰教|伊斯兰教]]也有要求辟穀的特殊时期（[[伊斯兰斋月|伊斯兰斋月]]和基督教的[[禁食|禁食]]）。

==辟穀渊源==

“辟穀”源自道家养生中的“不食五穀”，是古人常用的一种养生方式。它源于[[秦|先秦]]，流行于[[唐朝|唐朝]]，又称：却穀、去穀、绝穀、绝粒、却粒、休粮……等。辟穀最早的记载源自《[[莊子_(書)#逍遙遊第一|莊子·逍遙遊]]》：“藐姑射之山，有神人居焉。肌肤若冰雪，淖约若处子，不食五穀，吸风饮露，乘云气，御飞龙，而游乎四海之外......”，作为一种延年益寿的养生法则，<ref>{{Cite web |url = https://zh.wikisource.org/wiki/%E8%8E%8A%E5%AD%90/%E9%80%8D%E9%81%99%E9%81%8A |title = 《莊子》 |author = [[戰國|戰國]]時代[[宋國|宋國]][[莊周|莊周]] |date = |publisher = |language = |accessdate = |quote = 逍遙遊第一：……不食五穀，吸風飲露。……}}</ref>辟穀在很多古书典籍中也有记载。

传统的辟穀分为服气辟穀和服药辟穀两种主要类型：
:*「服气辟穀」主要是通过绝食、调整气息（呼吸）的方式来进行。
:*「服药辟穀」则是在不吃主食（五穀）的同时，通过摄入其他辅食（坚果、中草药、每天一颗红枣等），对身体机能进行调节。
