== 概要 ==

《這是我的戰爭》是一个以[[生存|生存]]为主题的[[模擬遊戲|策略模擬遊戲]]，玩家控制一组幸存平民生存在战争中的城市。游戏的主要目标是控制角色通过各种材料和工具在战争中存活下去。大多数玩家控制的角色没有军事背景和特殊生存经验，需要玩家不断的干预以维持生命。玩家的责任是维持角色在健康、饥饿、心情等方面的正常直至随机天数后的战争结束。

白天敌方[[狙击手|狙击手]]将阻止幸存者外出探索，从而让玩家只能在避難所中使用收集到的材料制造工具、交易、升級工作臺、補強防禦工事、做饭和治疗角色。晚上玩家则有机会走出去，并收集邻近地区用于维持他们生存的宝贵资源。探索期间玩家控制的角色可能遇见其他[[NPC|NPC]]幸存者，友善的倖存者可能會尋求援助或交易，由玩家选择是否要幫助這些人以獲得報酬或是杀人越货；而部分具有明顯敵意或是由於你觸怒他們而轉為有敵意的倖存者，你只能選擇偷偷在他們的視線外取得所需的物資或是直接以武器強奪。玩家可以在他们的避难所建造一个收音机来收听实用信息例如天气预警、城市经济市场消息和关于战争的新闻以帮助玩家计划适当的升级并完成任务。<ref>''Grayson, Nathan (March 13, 2014). ''</ref>

在游戏初期，玩家可控制1-4个角色，<ref>{{Cite web |url = http://www.gamersky.com/handbook/201501/505824.shtml |title =  屬於我的戰爭开局组合分析及新手指南 |author = 何必看清 |date = 2015-01-05 |publisher = 游民星空 |language = zh |accessdate = 2016-01-03 }}</ref>玩家做出的行动会改变各个角色在最后的结局类型，每個角色各有六種結局，除了死亡、自杀、出走結局外，倖存的結局有三種。<ref>{{Cite web |url = http://www.gamersky.com/handbook/201505/577043.shtml |title = 《屬於我的戰爭》全人物结局一览 有哪些结局 |author = 1412934707 |date = 2015-05-04 |publisher = 游民星空 |language = zh |accessdate = 2016-01-03 |quote = 每名角色共6个结局：真结局，普通结局（有的没有收集到），坏结局（bed end），死亡结局，自杀结局，出走结局 }}</ref>一周目时游戏开始的角色为布鲁诺和帕夫莱，另外随机选择马尔科或卡蒂亚。每个角色有一或两个特点会帮助或妨碍玩家进行游戏。有些特点在某些时刻有帮助，另外的则长期有效。例如布鲁诺有一个名为精通厨技（Good Cook）的特点可以帮助玩家在烹饪时使用更少的食材和水。总共有12个可以供玩家使用的角色。<ref>''Zgierski, Kuba. ''</ref>

在版本1.3的更新中新增了一个场景编辑器，这个编辑器允许玩家创建和选择角色，改变环境变量和战争总时长设置。<ref>''Hillier, Brenna (April 29, 2015). ''</ref>

==主要角色==
*'''帕夫列'''（Pavle）：足球員（移動速度加成）
*'''布魯諾'''（Bruno）：名廚（煮食耗材減少）
*'''卡蒂婭'''（Katia）：記者（貿易談判加成）
*'''阿里卡'''（Arica）：街頭扒手（無聲移動、戰鬥、開鎖）
*'''羅曼'''（Roman）：叛軍逃兵（攻擊力加成）
*'''茲拉塔'''（Zlata）：音樂學院學生（團隊士氣加成）
*'''馬可'''（Marko）：消防員（拾荒速度加成）
*'''鮑里斯'''（Boris）：倉庫工人（血量、背包容量加成）
*'''馬林'''（Marin）：木匠（建造耗材減少）
*'''艾米莉亞'''（Emilia）：律師（降低被襲擊的幾率）
*'''安東'''（Anton）：數學家（陷阱效率加成）
*'''茨維塔'''（Cveta）：校長（小童角色士氣加成）

== 开发 ==

游戏的灵感来自[[波斯尼亚|波斯尼亚]]平民在从二战至今最长的围困战——1992年至1996年[[塞拉耶佛圍城戰役|萨拉热窝围城战役]]中的经历。<ref>''[https://www.youtube.com/watch?v=gotK5DLdVvI "This War of Mine Launch Trailer - The Survivor"]. 11 bit studios. ''</ref> 游戏的盗版镜像于2014年11月出现在网络上，开发商对此的应答是开放了一些序列号并鼓励下载者与朋友分享，希望他们在经济条件许可的情况下购买正版。<ref>''[http://playingdaily.pl/2014/11/17/11-bit-studios-wygrywa-internet/ "11-bit Studios wygrywa internet!" [11-bit studios win the Internet]]. ''</ref>

2015年3月10日，开发商在Steam上新增了一个慈善[[追加下载内容|DLC]]，此DLC会在游戏场景中增加一些绘画。11 bit studios承诺所有的收入将用于帮助[[伊拉克战争|伊拉克战争]]中的孩子。<ref>{{cite news |language = en |url = http://www.eurogamer.net/articles/2015-03-10-this-war-of-mine-adds-war-child-charity-dlc |title = This War of Mine adds War Child charity DLC |publisher = eurogame.net  |date = 2015-04-03 |accessdate = 2015-12-22}}</ref>根据官方于4月3日的简报，此DLC的收入在[[救援|救援]]、[[基础教育|基础教育]]和[[心理援助|心理援助]]方面帮助了350名战争中的儿童，贡献最多的玩家地区分别是[[美国|美国]]、[[德国|德国]]、[[英国|英国]]和[[俄罗斯|俄罗斯]]。<ref>{{Cite web |url = http://www.11bitstudios.com/blog/en/16/this-war-of-mine/78/summary-of-the-war-child-charity-dlc.html |title = Summary of the War Child Charity DLC |author = 11 bit studios |date = 2015-04-03 |publisher = 11 bit studios |language = en |accessdate = 2015-12-22 |deadurl = yes |archiveurl = https://web.archive.org/web/20151203073007/http://www.11bitstudios.com/blog/en/16/this-war-of-mine/78/summary-of-the-war-child-charity-dlc.html |archivedate = 2015-12-03 }}</ref>

2016年2月5日，版本2.0.2正式加入了官方的[[简体中文|简体中文]]支持并在[[春节|春节]]期间进行了折扣。<ref>{{Cite web |url = https://steamcommunity.com/games/282070/announcements/detail/652262930202988634 |title = Happy Lunar New Year! - Update 2.0.2 |date = 2016-02-05 |publisher = Steam |language = en |accessdate = 2016-03-26 }}</ref>

== 评价 ==
《這是我的戰爭》收到了许多评论家的好评，在[[Metacritic|Metacritic]]赢得了83分的专业评价。<ref name="Metacritic">''[http://www.metacritic.com/game/pc/this-war-of-mine "This War of Mine for PC Reviews"]. ''</ref>据报道，游戏在开始销售的两天内就收回了所有的开发成本。<ref>''Yin-Poole, Wesley (November 27, 2014). ''</ref>

在2015年[[独立游戏节|独立游戏节]]上，《这是我的战争》获得了塞尤玛斯·麦克纳利年度独立游戏奖和最佳叙事奖提名，<ref>{{cite news |language = en |url = http://www.igf.com/02finalists.html |title = 2015 Independent Games Festival Finalists |publisher = igf.com |date = 2015-01-07 |accessdate = 2015-12-21}}</ref>并最终赢得观众奖（Audience Award）。<ref>{{cite news |language = en |url = http://igf.com/2015/03/outer_wilds_wins_top_honors_at.html |title = Outer Wilds wins top honors at the 17th annual IGF Awards |publisher = igf.com |date = 2015-03-04 |accessdate = 2015-12-21}}</ref>
