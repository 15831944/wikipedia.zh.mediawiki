於《[[月刊Comic_Alive|月刊Comic Alive]]》2012年9月號至2013年6月號雜誌連載《這間教室被不回家社占領了》漫畫版<ref>{{cite web|url=http://www.amazon.co.jp/dp/B008HRWCDQ|script-title=ja:月刊 comic alive (コミックアライブ) 2012年 09月号|language=ja|publisher=[[日本亞馬遜|日本亞馬遜]]|accessdate=2016年6月12日}}</ref><ref>{{cite web|url=http://www.mediafactory.co.jp/comic-alive/index.php?MENU=detail&MD=backnumber&mid=94|script-title=ja:アライブ６月号は4月27日（土）発売！表紙はTVアニメ絶賛放送中!! 『断裁分離のクライムエッジ』|language=ja|publisher=Media Factory|accessdate=2016年6月12日}}</ref>，出版[[單行本|單行本]]全2卷<ref name="Media Factory"/>。2012年8月2日，[[Asgard_(企业)|BOOST ON]]发售本作改編PSP游戏《这间教室被不回家社占领了。便携版 夏日学园大战篇。》，并于2012年10月4日发售第二款《这间教室被不回家社占领了。便携版 学园Dog Year篇》<ref>{{Cite web|date=|url=http://www.4gamer.net/games/150/G015016/20120312029/|title= 「帰宅しない部」に占拠されたPSP用ソフト2本が登場。ラノベ原作者が執筆・監修した「この部室は帰宅しない部が占拠しました。 ぽーたぶる」発売|publisher=|accessdate=2012-06-28|language=ja}}</ref><ref>{{lang|ja|「学園ドッグイヤー編」}}（学园Dog Year篇）在省略长音符号之后，也可另认为是“学园Dog ear篇”，即狗耳（英语中指的是书的折角），以回应故事中的女孩们长出了狗耳。</ref>。

==故事简介==
戀妹狂的高中生柊木夕也為了拿回遺忘在學校的手機，偶然發現青梅竹馬木瀧戀子與轉學生櫻江櫻桃住宿在學校的事實，戀子為了成為死靈法師而在學校挖掘屍體，櫻桃則是為了實現與遠方朋友的約定，而以學校為地盤。可是無巧不巧，這件事卻被學生會長秋月琴音所發現，並禁止他們晚上在學校逗留。而為了能在晚上留在學校，夕也等三人遂決定成立「不回家社」，但卻受到琴音的學生會所阻擾，究竟他們能否對抗學生會，順利成為正式社團呢？

===不回家社===
; {{nihongo|柊木夕也|柊木 夕也（ひいらぎ ゆうや）|聲：[[井口祐一|井口祐一]]（廣播劇CD）}}
: 本作男主角，高中二年级。“单纯”宠爱妹妹的[[妹控|妹控]]（!?），擅长料理。

; {{nihongo|樱江樱桃|桜江 ゆすら（さくらえ -）|声：[[下田麻美|下田麻美]]}}
: 谜一般的少女。就读高中二年级，[[贫乳|贫乳]]，经常挂在嘴边的口头禅是“啊呜”，不懂得人类的各种习俗，习得狼性。自己得意的武器是匕首，不过也有令人喜爱的一面。
: 开学的时候，没有与班上任何一个人搭上话，夕也也屡次受到拒绝，在经过一次意外闯入校园事件之后，渐渐地熟悉了夕也和恋子，称呼他们是“兽群”。

; {{nihongo|木泷恋子|木滝 恋子（きたき れんこ）|声：[[堀江由衣|堀江由衣]]}}
: 夕也的[[青梅竹马|青梅竹马]]，高中一年级。短马尾发型，有时会穿有一件漆黑的外衣。被“祖母”称赞为百年一遇的奇才，自认为非常适合当“死灵法师”，有成为冥府女王的可能。只是心里十分害怕鬼怪。
: 小时候，称呼夕也为“哥哥”，在长大之后就没有再叫这种说法了。尽管如此，心底上仍一直在意夕也。

; {{nihongo|秋月琴音|秋月 琴音 （あきづき ことね）|声：[[水桥香织|水桥香织]]}}
: 夕也的表姐，高中二年级。学生会会长，秋月学园理事长的孙女。巨乳，成绩也是名列前茅。

; {{nihongo|柊木耶宵|柊木 耶宵 （ひいらぎ やよい）|声：[[小倉唯|小倉唯]]}}
: 夕也的妹妹，初中三年级。十分受到哥哥夕也的溺爱。
: 故事起点中跟随夕也的妈妈一起去寻找工作中的父亲。

; {{nihongo|志志谷左京|志々谷 左京 （ししや さきょう）|声：[[伊藤静|伊藤静]]}}
: “武装风纪委员部队”一号队队长兼学生会书记，高中二年级。与“志志谷右京”是一對双胞胎。

; {{nihongo|志志谷右京|志々谷 右京（ししや うきょう）|声：[[斋藤千和|斋藤千和]]}}
: “武装风纪委员部队”成员之一。与“志志谷左京”是一對双胞胎。

; {{nihongo|土御門沙月|土御門 沙月（つちみかど さつき）}}
: 夕也的後輩，出身於陰陽師家系的少女。生來魔力就很弱而且無法使用魔法。

===學校相關===
; {{nihongo|桐谷雅纪|桐谷 雅紀（きりや まさき）}}
: 夕也的同学兼从初中以来认识的死党。在学校里總是受到女孩欢迎的优等生。[[萝莉控|萝莉控]]。

; {{nihongo|一之瀨先生|一ノ瀬先生（いちのせせんせい）}}
