'''袋熊'''，即'''袋熊科'''（{{lang-en|Wombat}}，学名：''Vombatidae''），是[[澳洲|澳洲]]的[[有袋類|有袋類]]动物。牠們的腳短及[[肌肉|肌肉]]發達，身長約有1米，尾巴亦很短。牠們生活於澳洲東南部及[[塔斯馬尼亞州|塔斯馬尼亞州]]的[[森林|森林]]、山地及[[石楠地|石楠地]]。

== 特徵 ==
袋熊會以其类似[[齧齒目|齧齒目]]动物的[[牙齒|牙齒]]及強壯的爪來挖巢穴。雖然牠們主要是黎明、白天及黃昏活動的，但仍會在夜間或寒冷的日子出來攝食。牠們並非容易看見的[[動物|動物]]，但所到之處都可以看到牠們留下的證據。

袋熊是[[草食性|草食性]]的，主要以[[草|草]]、[[莎草科|莎草科]]、[[香草|香草]]、樹皮及樹根為食物。牠們的[[門齒|門齒]]有點像[[齧齒目|齧齒目]]的，可以用來咬粗糙的樹，或挖洞穴。就像其他的草食性動物一樣，牠們的門齒與頰齒之間有很大的縱裂。另外，牠們的糞便是方形的 ，但生物學家還不是很清楚方形的排泄物的形成原因，一般推測是跟領地有關。

[[澳洲野犬|澳洲野犬]]及[[袋獾|袋獾]]會獵食袋熊。袋熊的毛皮顏色各有不同，一般由沙色至褐色，或灰色至黑色不等。所有其下的[[物種|物種]]都約只有1米長，重20-35公斤。

雌性袋熊經過26-28日的[[妊娠期|妊娠期]]後，會在[[春天|春天]]會產下一頭幼袋熊。牠們有良好發育的[[育幼袋|育幼袋]]，可保護幼袋熊6-7個月。袋熊在出生後15個月會斷奶，在18個月會達至性成熟。<ref name=EoM>{{cite book en|editor=Macdonald, D.|author= McIlroy, John|year=1984|title=The Encyclopedia of Mammals|publisher=Facts on File|location=New York|pages=876-877|isbn= 0-87196-871-1}}</ref>

=== 生態及行為 ===
袋熊的[[新陳代謝|新陳代謝]]非常的慢，差不多要用14天的時間來完成[[消化|消化]]，這有助於生活在乾燥的環境。<ref name=EoM/>牠們一般行動得很慢，但當遇上危害時，逃走速度可以達40km/h，並維持達90秒。袋熊會保護由其巢穴為中心的疆界，對入侵者存有攻擊性。[[塔斯馬尼亞袋熊|塔斯馬尼亞袋熊]]的疆界達23公頃，而[[毛鼻袋熊屬|毛鼻袋熊屬]]的則不多於4公頃。<ref name=EoM/>

當受到攻擊時，袋熊會發揮巨大的抵抗力。例如當牠受到地底下的掠食者攻擊時，牠會破壞地底隧道，令掠食者窒息。牠主要的防禦是靠身體後部以[[軟骨|軟骨]]組成的結構。連同不怎麼長的尾巴，可以避免在逃走入隧道時被掠食者攻擊尾部。

另外，袋熊是有袋動物中少數袋子開口朝後的，可避免在挖地洞時泥土跑進袋中影響小袋熊。

== 演化 ==
袋熊是屬於[[雙門齒目|雙門齒目]]。牠們的祖先於5500-2600萬年前開始[[演化|演化]]。在進入[[冰河時期|冰河時期]]前共有11個[[物種|物種]]生成，當中幾類的體型像[[犀牛|犀牛]]（如[[雙門齒獸|雙門齒獸]]），是[[有袋類|有袋類]]中最大的。最早的[[澳洲原住民|澳洲原住民]]到達時正值雙門齒獸最為普遍的時期，這類動物的[[滅絕|滅絕]]可能是被獵殺、失去棲息地等原因所造成。

== 物種 ==
袋熊現存的只有三個[[物種|物種]]：<ref name=msw3/>

* [[塔斯馬尼亞袋熊|塔斯馬尼亞袋熊]]（''Vombatus ursinus''） 
* [[毛鼻袋熊|毛鼻袋熊]]（''Lasiorhinus latifrons''） 
* [[澳洲毛鼻袋熊|澳洲毛鼻袋熊]]（''Lasiorhinus krefftii''）

== 與人類的關係 ==
被飼養的袋熊可以很溫馴。很多[[澳洲|澳洲]]的公園、[[動物園|動物園]]及其他旅遊設施都有袋熊展覽。

但是由於袋熊外型可愛溫馴，人們容易缺乏足夠警戒心，袋熊可能在憤怒或鬧情緒時帶有攻擊性。袋熊的體重可以將一般體型的成人撞倒，利齒及爪可以造成損傷。袋熊一咬可以咬破褲子、厚羊毛襪及膠靴，並造成2厘米深的傷口。至2013年世界最老的袋熊為[[澳洲|澳洲]][[巴拉瑞特野生動物園|巴拉瑞特野生動物園]]([[:en:Ballarat_Wildlife_Park|Ballarat Wildlife Park]])27歲的派翠克(Patrick)。<ref>國際中心／綜合報導,"有育幼袋外型像熊　澳洲派翠克是世界最老的「袋熊」"[http://www.ettoday.net/news/20130820/259180.htm],ETtoday寵物動物新聞,2013年08月20日 10:41.</ref><ref>Heather Lighton,"World's oldest wombat"[http://thethousands.com.au/melbourne/stray/world-s-oldest-wombat]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }},STRAY/Three Thousand,retrieved Aug 20, 2013.</ref>
