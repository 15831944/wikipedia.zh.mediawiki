'''乌拉尔山脉'''（[[俄语|俄语]]：{{lang|ru|Ура́льские го́ры}}，簡稱{{lang|ru|Ура́л}}）是[[俄罗斯|俄罗斯]]境内大约南北走向的一座山脉，它位于俄罗斯的中西部。

== 位置 ==
[[File:Ural_Mountains_Map.gif|thumb]]
大致位于东经60度。乌拉尔山脉的西部是俄罗斯平原，东部是[[西西伯利亚平原|西西伯利亚平原]]。向北它一直伸入[[喀拉海|喀拉海]]，向南[[里海|里海]]离山脉的开始不远。这里它过渡到[[哈萨克斯坦|哈萨克]]的草原。从它的南端到北端它一共长2500公里，在它的北端它平行[[鄂毕河|鄂毕河]]向东转入海。

从[[地质|地质]]上看，喀拉海和[[巴伦支海|巴伦支海]]间的[[新地岛|新地岛]]是乌拉尔山脉的延伸。

== 亚洲和欧洲的分界线 ==

[[File:Prokudin-Gorskii-32.jpg|right]]
乌拉尔山脉和[[乌拉尔河|乌拉尔河]]是[[亚洲|亚洲]]和[[欧洲|欧洲]]的分界线。它们将[[歐亞大陆|歐亞大陆]]分成了两个大小不等的[[洲|洲]]。烏拉爾山的森林區也是[[烏拉爾語系|烏拉爾語系]]民族發源地。

== 最高点 ==
乌拉尔山脉的最高点位于其中北部的''Poznurr''，海拔1895公尺。

== 矿藏和资源 ==
乌拉尔山脉有丰富的资源，包括[[黄玉|黄玉]]和[[绿玉|绿玉]]。

乌拉尔山脉北方的[[科米原始森林|科米原始森林]]是[[世界遗产|世界遗产]]。

== 形成 ==
乌拉尔山脉是在[[石炭纪|石炭纪]]形成的，当时今天的西伯利亚是一块独立的大陆，它与当时[[盘古大陆|盘古大陆]]的一部分，今天的欧洲，相撞形成了乌拉尔山脉。至今为止欧洲与西伯利亚还连在一起。
