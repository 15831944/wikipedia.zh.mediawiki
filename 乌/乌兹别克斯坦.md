'''乌兹别克斯坦共和国'''（[[烏茲別克語|烏茲別克語]]拉丁字母：「{{lang|uz-latin|O'zbekiston Respublikasi}}」；[[西里爾字母|西-{里}-爾字母]]：「{{lang|uz-cyrillic|Ўзбекистон Республикаси}}」），通稱-{zh-hans:'''乌兹别克斯坦''';zh-hk:'''烏茲別克斯坦''';zh-tw:'''烏茲別克''';}-，是一个位于[[中亚|中亚]]的[[內陸國家|內陸國家]]，1991年从原[[苏联|苏联]]独立。是世上兩個[[雙重內陸國|雙重內陸國]]之一（另一個為[[列支敦斯登|列支敦斯登]]）。

== 历史 ==
{{main|乌兹别克斯坦历史}}
* 870年到1212年，锡尔河下游三角洲地区的[[乌古斯人|乌古斯人]]家族的阿弗剌昔牙卜家族的后裔创建的[[喀喇汗朝|喀喇汗朝]]（[[黑汗|黑汗]]）王朝統治。統治後期（12世纪至13世纪）由[[西遼|西遼]]監督統治。
* 1212年–1220年，[[花剌子模沙朝|花剌子模沙朝]]（1194年—1220年12月）在1210年代開始管理河中。
* 1220年–1388年，被[[蒙古帝国|蒙古帝国]]占领，分別劃歸[[金帐汗國|金帐汗國]]與[[察合台汗國|察合台汗國]]。後被[[帖木兒|帖木兒]]佔領。
* 16世紀初期的1506年，[[烏茲別克人|烏茲別克人]]領袖[[昔班尼|昔班尼]]南下[[河中|河中]]農業區。推翻帖木兒王朝。
* 17世紀，分裂為[[布哈拉汗國|布哈拉汗國]]与[[希瓦汗國|希瓦汗國]]。
* 18世紀，[[浩罕汗国|浩罕汗国]]獨立。
* 1873年，布哈拉汗国的[[曼吉特王朝|曼吉特王朝]]被[[俄罗斯帝国|俄罗斯帝国]]征服，成为[[附属国|附属国]]。
* 1918年，被划归[[突厥斯坦苏维埃社会主义共和国|突厥斯坦苏维埃社会主义共和国]]。1920年苏联红军侵入希瓦汗国，推翻[[希瓦汗国|阿布德·阿拉汗]]（1918-1920年在位）的统治，希瓦汗国灭亡。
* 苏联随后在其领土上建立[[花剌子模苏维埃人民共和国|花剌子模苏维埃人民共和国]]（见[[花剌子模|花剌子模]]），1924年，花剌子模苏维埃人民共和国解散，其领土并入[[乌兹别克苏维埃社会主义共和国|乌兹别克]]和[[土库曼苏维埃社会主义共和国|土库曼]]两个[[苏联加盟共和国|苏联加盟共和国]]。同年10月，原布哈拉汗国的一部分和希瓦汗国的一部分合并，成立[[乌兹别克苏维埃社会主义共和国|乌兹别克苏维埃社会主义共和国]]，并加入[[苏联|苏联]]。
* 1991年8月31日宣告独立，改称乌兹别克斯坦共和国，同年12月21日加入[[独立国家联合体|独立国家联合体]]。

== 政治 ==
{{Unreferenced section|time=2015-10-10T23:18:24+00:00}}
[[File:Stamps_of_Uzbekistan,_2010-62.jpg|缩略图]]
憲法上，烏兹别克是[[自由民主|自由民主]]體制。

然而，自[[卡里莫夫|卡里莫夫]]在1991年上臺至今，他的任期已多次獲得延長，分別在1995年、2002年、2007及2015年成功連任，至2016年9月去世。

2002年的選舉亦同時選出上、下議院的議員，但卻是在沒有反對黨之下進行。因此，國際觀察員都一般認為是次選舉不能達到[[民主|民主]]水平，在[[民主指數|民主指數]]也屬於[[獨裁政體|獨裁政體]]。

在[[執政黨|執政黨]]以外其他政黨一般情況下可以成立，但也必須獲得[[政府|政府]]的批准才能組黨。另外，[[傳媒|傳媒]]的[[新聞自由|新聞自由]]亦受到政府操控，因為政府維穩政策，新聞內容甚少報導國內[[政治|政治]]有關訊息，特別是[[反政府|反政府]]和民主內容，以免遭受牢獄之災。烏茲別克政府在國內基本上對所有反對派實施[[黨禁|黨禁]]。而國內最大反對派是[[乌兹别克伊斯蘭運動|乌兹别克伊斯蘭運動]]。

2016年12月4日，烏兹别克進行自[[卡里莫夫|卡里莫夫]]逝世後的首次總統大選，總理[[肖開提·米爾則亞耶夫|米爾則亞耶夫]]當選新任總統。

== 地理 ==
[[File:Oliy_Majlis_(Parliament_of_Uzbekistan).jpg|thumb]]
[[File:Termez,_stallholders.JPG|thumb]]
乌兹别克斯坦西南部与[[土库曼斯坦|土库曼斯坦]]接壤，南部与[[阿富汗|阿富汗]]接壤，东部与[[塔吉克斯坦|塔吉克斯坦]]和[[吉尔吉斯坦|吉尔吉斯坦]]接壤，北部和西部与[[哈萨克斯坦|哈萨克斯坦]]接壤，由於其鄰國皆為[[內陸國家|內陸國家]]，因此烏茲別克成了目前世界上僅有的兩個雙重內陸國之一。北部一部分濒临[[咸海|咸海]]。國土大部分位於[[克孜勒庫姆沙漠|克孜勒庫姆沙漠]]中。境內最高的山是海拔4,643米的[[哈兹拉特苏丹峰|哈兹拉特苏丹峰]]，她是[[吉萨尔山脉|吉萨尔山脉]]的主峰。

== 经济 ==
{{main|乌兹别克斯坦经济}}
和许多其他独联体经济体一样，乌兹别克斯坦的经济在经济转型的头几年里有所下滑而到1995年后开始复苏，这得益于政策改革的累积效应开始起作用。经济显示出强劲的增长，在1998年到2003年间每年增长4%，之后增速更是到了每年7%-8%。据国际货币基金组织估计，2008年该国的GDP几乎是1995年的2倍（以固定价格计算）。2003年以来年通胀率平均在10%以下。

乌兹别克斯坦的人均国民总收入很低（2006年以当时美元币值算为610美元，按购买力平价算为2250美元）。这排在209个国家中的第169位；在12个独联体国家中，只有吉尔吉斯斯坦和塔吉克斯坦的人均国民总收入比其低（2006年）。经济生产集中于日用品。该国现在是世界第6大棉花生产国和第2大棉花出口国，世界第7大黄金生产国。同时该国也是区域内重要的天然气、煤、铜、石油、银和铀生产国。

农业雇用了该国28%的劳动力，贡献了[[GDP|GDP]]的24%（2006年的数据）。官方统计的[[失业率|失业率]]很低，而不充分就业--特别是在农村地区--据估计至少有20%。不过，在棉花收获的季节，所有的学生和老师都被动员去田里帮忙收棉花，并且是义务性质的（相關內容可見[[烏茲別克童工問題|烏茲別克童工問題]]）。由于该国使用童工，导致一些公司，包括[[Tesco|Tesco]]、[[C&A|C&A]]、[[Marks&Spencer|Marks&Spencer]]、[[Gap|Gap]]和[[H&M|H&M]]共同抵制乌兹别克斯坦生产的[[棉花|棉花]]。

面对获得独立后种种的经济挑战，政府采取了一种渐进式的改革政策，强调国家控制，减少进口和能源自给。自1994年以来，该国国家控制的媒体反复宣称这是“乌兹别克斯坦经济模式”的成功，并且暗示这是计划经济向市场经济道路平稳转型的独特范例，这种方式能成功避免经济震荡、停滞和贫困。

近日，乌兹别克斯坦4总统米尔济约耶夫签署命令，任命原国家税务委员会副主席诺尔马特·图尔苏诺夫为塔什干市国家税务总局局长；任命原塔什干州国家税务总局局长阿利姆然·法伊济巴耶夫为国家税务委员会副主席。<ref>[http://www.ctax.org.cn/csxw/jwsx/201806/t20180619_1077594.shtml  乌兹别克斯坦总统米尔济约耶夫继续对税务系统进行人事调整 中国税务网 境外资讯]</ref>

3290万人，大多是[[烏孜別克族|烏孜別克族]]，也有[[俄罗斯族|俄罗斯族]]（4.4%）、[[卡拉卡爾帕克族|卡拉卡爾帕克族]]（2.2%）、[[塔吉克族|塔吉克族]]（4.9%）、[[柯尔克孜族|柯尔克孜族]]（1%）、[[哈萨克族|哈萨克族]]（3.9%）和[[高丽人_(中亚)|高丽人]]（0.7%）等少數民族的分布。

== 宗教 ==
{{main|乌兹别克斯坦宗教}}
烏茲別克斯坦為世俗國家，有96.3％的人口信奉[[伊斯蘭教|伊斯蘭教]]。

== 外交==
{{main|乌兹别克斯坦外交}}
===与中華人民共和国的的关系===

{{main|中國－烏茲別克斯坦關係}}

====政治====
早在两千多年前，通过[[丝绸之路|丝绸之路]]中国与乌兹别克斯坦就有往来。1992年1月，两国建交，之后中乌关系始终保持健康稳定发展，高层交往日益密切。2010年[[胡锦涛|胡锦涛]]首次访乌，2011年、2012年[[卡里莫夫|卡里莫夫]]总统两次访华后，2013年9月[[习近平|习近平]]受邀至乌兹别克斯坦進行国事访问。雙方政治互信不断加深，雙方政府坚定支持对方维护国家统一和领土完整的努力；安全合作亦卓有成效，中乌在合作重大政府活动顺利举行等方面保持了密切协作。

====經濟====
2011年，卡里莫夫訪問中國大陸時與中華人民共和國簽定一系列貸款和投資協議，中華人民共和國由此進入烏國的石油產業。过乌境的“中国－中亚”天然气管道运营顺利，近年交易量也逐年增長。

2012年開通了納沃伊（烏）－天津（中）的貨運航線，未來計劃將此運輸線延長到上海和廣州。而塔什干（烏）至烏魯木齊（中）的客運航線也恢復營運。而目前，規劃多年的中國-吉爾吉斯斯坦-烏玆別克斯坦鐵路交通興建計劃，因吉國方面在2012年的反對而暫告停頓。

2012年，中乌双边贸易额达28.75亿美元，创历史新高。

簽訂的中乌工业园等合作项目稳步推进，两国贸易结构也由以前单纯的原料资源贸易（天然氣、鈾精礦、稀土、棉纤維、化肥等等）向非资源和投资领域拓展。由中國國家開發銀行以及中國進出口銀行对乌的投资总额已超过40亿美元，中華人民共和国已成乌兹别克斯坦第二大贸易伙伴国、第一大投资国、第一大棉花买家、第一大电信设备和土壤改良设备供应国。

===文化===
人文交流日益密切，2005年建立塔什干孔子学院。第二家“孔子学院”亦將開设。烏國研究學者曾對中方人士指出，烏境曾挖掘出許多帶有明顯中國特色的絲路文物。中烏雙方的高等教育留學生人數也正逐年增加中。在新疆維吾爾自治區中約有3萬人左右的[[烏孜別克族|烏孜別克族]]生活。
