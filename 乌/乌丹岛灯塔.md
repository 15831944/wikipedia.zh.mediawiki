'''乌丹岛灯塔'''（{{lang-en|Undan Island Lighthouse}}、{{lang-ms|Rumah Api Pulau Undan}}）是[[马来西亚|马来西亚]][[马六甲|马六甲]]岸外[[乌丹岛|乌丹岛]]的一座灯塔。

==历史==
灯塔是马来西亚作为[[英国|英国]][[海峡殖民地|海峡殖民地]]时期开始建设。设立该灯塔的目的是更好的指引[[马六甲海峡|马六甲海峡]]上航行的船只。1879年[[伯明翰|伯明翰]]的{{tsl|en|Chance Brothers and Company||机会兄弟建筑事务所}}设计以及建造了灯塔。 灯塔于1880年完工，并且至今还在运作。2008年，原有灯塔傍建立了一座新的灯塔，新灯塔用混泥土建造，里头设有先进的导航仪器，辅佐原有灯塔指引航行的船只。<ref name="MarineDept"/><ref name="UNC"/><ref>{{cite web|url=https://twitter.com/S_FRamli/status/975271919783108609 |title=Restoration of Rumah Api Pulau Undan |author=Saiful Fazli Ramli |publisher=Twitter |date=2018-03-18 |accessdate=2018-08-15}}</ref>

==位置==
灯塔位于乌丹岛顶部，离开马六甲安外{{convert|25|km|mi}}。因此灯塔不能通过陆路交通到达，灯塔职员只能乘坐快艇前往灯塔。<ref name="UNC"/>

==建筑风格==
原有的灯塔是一座[[八边形|八边形]]灯塔，灯塔顶部设有照明灯。灯塔旁边有一间看守员宿舍。建筑都是[[砌体结构|砌体结构]]似建筑。新增设的圆筒性灯塔比起原来灯塔高3倍，也舍利在原来的灯塔的旁边，与原来的灯塔有鲜明的不同之处。<ref name="UNC"/><ref name="MarineDept"/>
