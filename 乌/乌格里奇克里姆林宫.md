'''乌格里奇克里姆林宫'''是一座位于[[俄罗斯|俄罗斯]][[烏格里奇|烏格里奇]]历史悠久的[[城堡|城堡]]。该城堡建造于10世纪，是乌格里奇城镇中最古老的部分。宫殿位于[[伏尔加河|伏尔加河]]的右岸高侧，还有另外伏尔加河的支流围绕着城堡，人们将小河扩大，以此为宫殿修建了护城河。<ref>Louis Réau: l'art russe des origines à Pierre le grand; Paris,Henry Laurens éditeur 1921; page 328</ref>
[[克里姆林|克林姆林宫]]的城墙原本是用石头与木质材料建成的，但是几百年间城墙不断的倒塌损毁。在1660-1662年，人们最后一次修复城墙、瞭望塔和护城河桥。但由于总是损毁，在一百年后他们终于还是将这些建筑都拆除了，以前修建的防御工事里唯一剩下的只有护城河。<ref name="Paris page 329">[[Louis_Réau|Louis Réau]] : l'art russe des origines à Pierre le grand; Paris,Henry Laurens éditeur 1921; page 329</ref>

昔日的城堡中的建筑被保存下来的还有：宫殿、[[德米特里王子滴血教堂|德米特里王子滴血教堂]]、耶稣复活教堂及其旁边的钟楼、城市杜马。
== 宫殿 ==
乌格利奇克里姆林内的宫殿在1481年建造，是中世纪俄罗斯少数使用砖块建起的民用建筑。这座宫殿是在俄罗斯伊万三世的兄弟安德烈·瓦西列维奇·巴尔奇伊（Andre Vasilevich Balchoy）时代修建的，他还在克里姆林宫建造了救世主大教堂。考古发掘表明，宫殿的其他部分是用木头建造的，只有这座保留至今的宫殿主要部分建筑是用砖砌成的。<ref name="rusarch">[http://rusarch.ru/liukonen1.htm Е. А. Лиуконен Ансамбль Угличского кремля]</ref>宫殿的主要部分由没有经过绘画与装饰的柱子组成的拱形大厅。墙壁很厚实，房间在低矮的地下，被黑暗所覆盖。这幢建筑分为两层，由装饰的楼梯相连。<ref name="Paris page 329"/>

== 德米特里王子滴血教堂 ==
[[File:Dmitry_church_uglich_2.JPG|左]]
[[察列維奇|察列維奇]][[烏格里奇的德米特里|烏格里奇的德米特里]]于1591年在[[乌格利奇|乌格利奇]]“滴血教堂”旧址的木质教堂离奇死亡。为了纪念这位小王子，1692年在旧址上重建了石质教堂，也就是今日的“滴血教堂”。<ref>{{Cite web|url=https://www.russia-online.cn/CityGuide/CitySummary/citys_13.html|title=乌格里奇|accessdate=|author=俄罗斯旅游中文网|publisher=russia-online.cn}}</ref>

教堂蓝色的洋葱顶与星星代表着天堂，是人们对小王子能够在天堂中平安喜乐地生活的祝愿。血红的墙体，蓝色的屋顶，装饰搭配白色的飞檐和壁柱。红色代表鲜血、白色代表纯洁、教堂蓝色的洋葱顶与星星代表着天堂，是人们对小王子能够在天堂中平安喜乐地生活的祝愿。<ref>{{Cite web|url=http://www.sohu.com/a/65076267_376225|title=俄罗斯有三座“滴血教堂”，你知道吗？|accessdate=|author=俄罗斯旅游中文网|publisher=搜狐旅游}}</ref>

{{来源请求|教堂还保留了18世纪下半叶的画作，画作描绘了德米特里王子被害，以及因为小王子报仇杀死歹徒而被镇压的群众。教堂餐厅的壁画记载了[[圣经|圣经]]的[[创世纪|创世纪]]篇：[[亚当|亚当]]和[[夏娃|夏娃]]被逐出[[伊甸园|伊甸园]]。壁画描绘的是[[超现实主义|超现实主义]]版本裸体的亚当和夏娃，而这也是教堂里最早突破世俗的作品之一。}}

== 耶稣复活教堂与钟楼 ==
[[File:Угличский_Кремль._Спасо-Преображенский_собор.JPG|缩略图]]
