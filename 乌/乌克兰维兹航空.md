'''乌克兰维兹航空'''（{{lang-ua|Товариство з обмеженою відповідальністю „Авіалінії Візз Ейр Україна”}}）是一家[[乌克兰|乌克兰]][[廉价航空公司|廉价航空公司]]，以[[基辅国际机场|基辅国际机场]]和[[利沃夫国际机场|利沃夫国际机场]]为主要运营基地。该航空公司原为[[匈牙利|匈牙利]][[维兹航空|维兹航空]]的分公司，于2015年4月20日停止运营<ref name="shutdown"/>。

== 历史 ==
乌克兰维兹航空于2008年7月11日开始运营[[乌克兰|乌克兰]]境内的国内航班，此后逐渐开始拓展国际航班业务。在其国际航班业务兴起后，乌克兰维兹航空便不再继续运营国内航班。

2013年10月，乌克兰维兹航空开始以[[顿涅茨克国际机场|顿涅茨克国际机场]]作为其第二个运营基地，由1架飞机运营6条新航线<ref>{{cite web |url=https://wizzair.com/en-GB/about_us/news/wizen178 |title=存档副本 |accessdate=2015-03-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20150402093840/https://wizzair.com/en-GB/about_us/news/wizen178 |archivedate=2015-04-02 }}</ref>，不过遗憾的是，[[顿涅茨克|顿涅茨克]]由于2014年4月开始的[[乌克兰危机|乌克兰危机]]而被迫暂停作为其运营基地<ref>http://www.bbj.hu/business/after-donetsk-closure-wizz-air-opens-new-riga-base_78783</ref>。

2015年3月26日，乌克兰维兹航空宣布由于[[乌克兰危机|乌克兰危机]]将于2015年4月20日[[倒闭|倒闭]]。8条从[[基辅|基辅]]始发的航线，其中半数飞往[[德国|德国]]，将继续由其[[母公司|母公司]]——[[匈牙利|匈牙利]][[维兹航空|维兹航空]]来运营。而余下的从[[基辅|基辅]]和[[利沃夫|利沃夫]]飞往[[意大利|意大利]]航、[[西班牙|西班牙]]、[[俄罗斯|俄罗斯]]和[[格鲁吉亚|格鲁吉亚]]的航线则完全暂停运营<ref name="shutdown">{{cite web|url=https://wizzair.com/en-GB/about_us/news/wizzen303 |title=存档副本 |accessdate=2015-04-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20150402145229/https://wizzair.com/en-GB/about_us/news/wizzen303 |archivedate=2015-04-02 }}</ref>。乌克兰维兹航空全部两架飞机——[[空中客车A320|空中客车 A320-200]]交由[[匈牙利|匈牙利]][[维兹航空|维兹航空]]接管。其中一架继续留在[[基辅|基辅]]，另一架则被安排到了[[斯洛伐克|斯洛伐克]][[科希策国际机场|科希策国际机场]]<ref>http://www.aero.de/news-21430/Wizzair-wickelt-Ukraine-Tochter-ab.html</ref>。

== 航点 ==
{{main|乌克兰维兹航空航点}}

在结束运营前，乌克兰维兹航空运营自[[基辅|基辅]]和[[利沃夫|利沃夫]]出发的共18条飞往不同国家的国际航线，主要航点位于[[德国|德国]]和[[意大利|意大利]]<ref name="shutdown"/>。
