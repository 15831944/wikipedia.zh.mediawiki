{{Infobox military person|姓名=乌尔金·加尔玛耶夫|图像=Уржин Гармаев.jpg|外文名=Уржин Гармаев （Urjin Garmaev）|出生日期=[[1888年|1888年]]（[[1889年|1889年]]）|出生地点=俄罗斯帝国外贝加尔|逝世日期=[[1947年|1947年]][[3月13日|3月13日]]|逝世地点=苏联莫斯科|allegiance=白军，满洲国军|军衔记录=陆军[[大将|上将]]}}'''乌尔金·加尔玛耶维奇·加尔玛耶夫 '''([[蒙古语|蒙古語]]: ᠭᠠᠷᠮ᠎ᠠ ᠶᠢᠨ ᠥᠷᠵᠢᠨ，汉字: '''乌尔金 ''', 俄语:Уржин Гармаевич Гармаев， [[1888年|1888年]]/[[1889年|1889年]] - [[1947年|1947年]][[3月13日|3月13日]])是[[滿洲國軍|满洲国军]]军人，为[[布里亚特人|布里亚特蒙古人]]。[[俄国内战|俄国内战]]时为[[俄国白军|俄国白军]]效力，随后移居满洲国，曾任兴[[安北省警备军|安北省警备军]]司令、第10军管区司令等职。最高军衔为陆军[[上将|上将]]。

== 经历 ==
1918年，乌尔金作为[[阿金草原|阿金草原]]僧俗领导人的代表，在满洲拜會[[格里戈里·谢苗诺夫|格里戈里·谢苗诺夫]]并建立了个人关系。随后他进入谢苗诺夫在[[赤塔|赤塔]]开设的士官学校学习，毕业后参加了驻扎于[[恰克图|恰克图]]的[[克里米亚师团|克里米亚师团]]，在一支由布里亚特人组成的部队任军官。1919年[[徐树铮|徐树铮]]率部占领恰克图，克里米亚师团投降。为了避免受到俄国革命的波及，1921年乌尔金带领家人和部下逃亡满洲，在[[海拉尔|海拉尔]]郊外的[[锡尼河|锡尼河]]畔定居放牧。 <ref name="tanaka155-156">田中（2009年）、155-156頁。</ref>当时在周边地区居住的布里亚特人共计三千左右，[[满洲国|满洲国]]建立之后为了笼络这些人，特别任命乌尔金为兴安北省警备军少将司令。 <ref name="makinami183-184">牧南（2004年）、183-184頁。</ref>

1935年1月满洲国和[[蒙古人民共和国|蒙古人民共和国]]之间发生了名为[[哈爾哈廟戰鬥|哈尔哈庙事件]]的边境冲突，乌尔金亲自赶赴现场处理事件。在随后召开的旨在解决国界问题的[[满洲里会议|满洲里会议]]中，乌尔金也作为满洲国代表团的二号人物参加了第一次会议。 <ref name="tanaka99">田中（2009年）、99頁。</ref>1936年4月，满洲国代表团团长，兴安北省省长[[凌陞|凌陞]]因为通敌嫌疑被日本宪兵队逮捕处决（凌陞事件），作为二号人物的乌尔金也被卷入，被满洲国最高顾问[[佐佐木到一|佐佐木到一]]传唤。由于乌尔金此前加入谢苗诺夫的军队与共产党作战的经历，佐佐木最终打消了他的嫌疑，加上北警備軍顧問{{link-ja|寺田利光|寺田利光}}的幫助，对他的问话仅仅持续了5分钟<ref>田中（2009年）、114-115頁。</ref><ref name="makinami41">牧南（2004年）、41頁。</ref>。满洲里会议第二次会议以后，乌尔金便成为了满洲国的首席谈判代表 <ref name="tanaka117">田中（2009年）、117頁。</ref>。

1939年的[[诺门罕战役|诺门罕战役]]中，时任第10军管区中将司令的乌尔金率领[[兴安北警备军|兴安北警备军]]（统称：乌尔金部队，含兴安骑兵第1、2、7、8、9团）参加了第一、二次战役。在第二次战役中，乌尔金部被部署在[[第23师团_(日本陆军)|日军第23师团]]的最右翼。由于作战勇敢，乌尔金战后被授予感谢状和1万日元的奖金。 

1945年1月乌尔金被晋升为上将，同时调往[[陆军兴安学校|陆军兴安学校]]任校长。 [[蘇日戰爭|苏联对日宣战]]后，负责防卫的满洲国军第2师旗下教导团发动了叛乱，乌尔金乘车追上叛军努力劝说其归正，但最终未果。 <ref name="makinami182">牧南（2004年）、182頁。</ref>苏联占领满洲国后，乌尔金亲自前往新京的苏联占领军司令部报道时当即被捕，随后被押往莫斯科。1947年3月13日以反革命分子的名义被枪决。[[苏联解体|苏联解体]]后的1992年2月23日乌尔金又被恢复了名誉。 <ref name="tanaka113">田中（2009年）、113頁。</ref>

== 个人 ==
[[File:Гармаев_ба_ганжууржав.jpg|右]]（右）。]]
乌尔金是身高两米，体重100公斤的巨汉。 他是虔诚的佛教徒。 <ref name="tanak158">田中（2009年）、158頁。</ref>哈尔哈庙事件时，外蒙军撤退后他走入寺庙探访，目睹寺庙被毁时不禁泪流满面，为共产国际扫除佛教的政策而哀叹。 <ref name="tanaka87-88">田中（2009年）、87-88頁。</ref>另外正珠尔扎布曾回忆，在第10军管区司令任上时乌尔金曾与顾问[[田古里直|田古里直]][[中佐|中佐]]就佛与天照大神孰高孰低的问题产生过争执 。<ref name="makinami185-186">牧南（2004年）、185-186頁。</ref>

烏爾金只有一個女兒{{lang|ru|Долгор}}；而Долгор也只有一個女兒奇米德（{{lang|ru|Чимид}}）。奇米德有三個孩子：休倫（{{lang|la|Хулэн}}）、貝爾（{{lang|la|Бэир}}）及亞歷山大（{{lang|la|Александр}}），他們一家四口目前居住在[[烏蘭烏德|烏蘭烏德]]<ref>{{Cite web |url=http://www.centrasia.ru/person2.php?news=form&st=1252829029# |title=存档副本 |access-date=2018-02-15 |archive-url=https://web.archive.org/web/20160304120738/http://www.centrasia.ru/person2.php?news=form&st=1252829029# |archive-date=2016-03-04 |dead-url=yes }}</ref>。
