'''乌鲁木齐南站'''（{{ug|ئۈرۈمچی جەنۇبىي بېكىتى|lat=Ürümchi jenubiy békiti}}），曾用名'''乌鲁木齐火车站'''，位于[[中华人民共和国|中国]][[新疆维吾尔自治区|新疆维吾尔自治区]][[乌鲁木齐市|乌鲁木齐市]][[沙依巴克区|沙依巴克区]]南站路135号，于1962年启用，为[[乌鲁木齐铁路局|乌鲁木齐铁路局]]管辖的客货运[[一等站_(中鐵)|一等站]]，是乌鲁木齐市内的主要客运枢纽。经过铁路有[[兰新铁路|兰新铁路]]、[[南疆铁路|南疆铁路]]、[[北疆铁路|北疆铁路]]和于2014年通车的[[兰新铁路第二双线|兰新铁路第二双线]]。并有发往[[哈萨克斯坦|哈萨克斯坦]][[阿拉木图|阿拉木图]]和[[阿斯塔纳|阿斯塔纳]]的国际联运旅客列车。

== 历史 ==
[[File:Old_urumqi_sta.,urumqi-city,Xinjiang,china.jpg|thumb]]
* 1962年12月3日：乌鲁木齐站建成运营。候车大楼建筑面积为2896平方米。
* 1963年1月15日：乌鲁木齐站开通客运。
* 1984年：乌鲁木齐站旅客发送量为122.3万人。新建建筑面积为8296.31平方米候车大楼一座，由上、下2层、3个候车大厅、3间软席候车室、3个母子候车室，能同时容纳旅客3000余人。旅客站台由原来的747米延至884米，面积由9190平方米扩大到10835平方米。站台新建两座风雨棚共9940平方米，并有地道通站外。此外还建站前广场40000平方米。
* 1985年：站线由6条增加至9条。货场货物线共6条，总有效长1194米，可容车79辆。
* 2000年：乌鲁木齐站旅客发送量猛增到398.7万人，日均发送1.09万人，高峰时达1.4万余人。
* 2002年5月18日凌晨2時：乌鲁木齐站旧[[站房|站房]]以[[爆破|爆破]]方式拆除。<ref>{{cite web |author = 新华网 |title = 历时40年的乌鲁木齐火车站站房成功整体爆破 |url = http://news.xinhuanet.com/newscenter/2002-05/18/content_398884.htm |publisher = 新华网 |accessdate = 2007-07-28 |date = 2002-05-18}}</ref>
[[File:Urumqi_Railway_Station_entrance.jpg|thumb]]题字。<ref>{{cite web |author = 巩亮亮 |title = 郭沫若题字“乌鲁木齐”承载城市情怀 |url = http://www.cssn.cn/zt/zt_zh/xwzt/zgxj/dmxjzmjp/rwmkyzdxj/201411/t20141114_1399134.shtml |work = 乌鲁木齐晚报 |publisher = 中国社会科学网 |accessdate = 2014-11-19 |date = 2014-11-14}}</ref>]]
* 2004年4月25日：新站房建成启用。新客运大楼总建筑面积为26185平方米。包括进站、出站通道、售票厅、普通候车厅、软席候车厅、贵宾候车厅等，可一次性容纳旅客7000人。乌鲁木齐站总占地面积20余万平方米。车站设有旅客高站台2座，低站台1座，一、二、三站台各设风雨棚1座，站台风雨棚总面积26059.4平方米。站内有客货到发线10条；候车厅6个，其中包括软席候车厅、国际列车候车厅各一个；人工售票厅、自动售票厅各一个。在岗职工454人，共11个民族，其中男职工243人。年发送旅客449.9万人，完成货物发送42.26万吨，运输收入完成8.82亿元。
* 2014年4月30日：发生[[2014年乌鲁木齐火车站袭击案|爆炸恐怖袭击]]。
* 2014年9月1日：乌鲁木齐站更名为乌鲁木齐南站<ref>[http://news.sina.com.cn/c/2014-08-19/051030706770.shtml 9月1日起乌鲁木齐火车站更名为乌鲁木齐南站]</ref>。
* 2018年9月1日：所有进出疆列车不再停靠乌鲁木齐南站<ref>[http://xj.people.com.cn/n2/2018/0903/c186332-32007994.html 进出疆列车本月起不再停靠乌鲁木齐南站]</ref>。

== 使用情况 ==
乌鲁木齐南站是[[兰新铁路|兰新铁路]]、[[南疆铁路|南疆铁路]]及[[北疆铁路|北疆铁路]]的客货运[[一等站_(中鐵)|一等站]]。2000年旅客发送量398.7万人次，日均旅客发送量1.09万人次。现乌鲁木齐南站安检十分严格，进入车站广场需过第一道安检；售票厅及自动售票厅各有一道单独安检；进入车站大楼（候车室）需过第三道安检。

== 旅客列车目的地 ==
{{动态列表}}
在2016年8月1日前，多数出疆普速列车都以乌鲁木齐南站为起点，也包括前往哈萨克斯坦的国际列车。目前原先在本站始发终到的所有动车组列车、多数出疆普速列车均延长至[[乌鲁木齐站|乌鲁木齐站]]始发终到，而本站办理的列车客运业务多以过路车为主，2018年9月1日起，所有进出疆列车不再停靠乌鲁木齐南站，乌鲁木齐南站只有疆内车继续停靠。
