'''烏爾都語'''（{{Lang-ur|اُردُو|translit=Urdū}}）是属于[[印欧语系|印欧语系]][[印度-伊朗语族|印度-伊朗语族]]的[[印度-雅利安语支|印度-雅利安语支]]。從使用人數來看，乌尔都语大约排名世界第20名，是[[巴基斯坦|巴基斯坦]]的国语，也是[[印度|印度]]的24种[[印度国家语言列表|规定语言]]之一。如果从宏观角度来看，乌尔都语可看成是[[印度斯坦语|印度斯坦语]]的一部分，所有印度斯坦语言构成世界上第四大的语言。

在1200年到1800年，[[南亚|南亚]]在[[德里蘇丹國|德里蘇丹國]]和[[莫卧儿帝国|-{zh-cn:莫卧儿帝国; zh-tw:蒙兀兒帝國}-]]的统治下，乌尔都语的受到[[波斯语|波斯语]]、[[突厥语|突厥语]]和[[阿拉伯语|阿拉伯语]]的影响。

== 使用者与其地理分布 ==
乌尔都语母语使用者大约有6-8千万人，其中5千2百万在印度（2001年），占当时该国人口的6%；1千3百万在巴基斯坦（2008年），占该国人口的8%；另有散布在世界各地的母语者。

在[[巴基斯坦|巴基斯坦]]的城市中，乌尔都语在大多数人中能通用，其中包括[[卡拉奇|卡拉奇]]、[[伊斯兰堡|伊斯兰堡]]、[[拉合尔|拉合尔]]、[[拉瓦品地|拉瓦品地]]、[[白沙瓦|白沙瓦]]、[[奎达|奎达]]、[[海德拉巴|海德拉巴]]、[[古吉兰瓦拉|古吉兰瓦拉]]、[[费萨拉巴德|费萨拉巴德]]、[[木尔坦|木尔坦]]和[[苏库尔|苏库尔]]。乌尔都语是巴基斯坦所有省份的官方语言，尽管该国有93%的人口不以它为母语。在同时使用英语和乌尔都语的学校中，直至高中前，乌尔都语是强迫性学习的语言。这样就使数以百万计以[[旁遮普语|旁遮普语]]、[[信德语|信德语]]、[[普什图语|普什图语]]、[[克什米尔语|克什米尔语]]、[[俾路支语|俾路支语]]、[[西莱基语|西莱基语]]、[[布拉灰语|布拉灰语]]等为母语的使用者的人，都能使用乌尔都语，乌尔都语也成为了巴基斯坦国家团结的象征，并成为巴基斯坦的主要交际语。它混合了巴基斯坦不同地区的词汇；同样，巴基斯坦不同地区的语言也受到乌尔都语的词汇影响。5百万来自不同民族（如[[普什图族|普什图族]]、[[塔吉克族|塔吉克族]]、[[乌兹别克族|乌兹别克族]]、[[哈扎拉族|哈扎拉族]]、[[土库曼族|土库曼族]]等）的[[阿富汗|阿富汗]]难民，在巴基斯坦居住了超过25年以后，都能操流利乌尔都语。这样可推论出，使用乌尔都语的中心，已由印度的德里和勒克瑙，转而到巴基斯坦的卡拉奇和拉合尔。

而在[[印度|印度]]，乌尔都语在[[北方邦|北方邦]]、[[克什米尔|克什米尔]]、[[德里|德里]]、[[班加罗尔|班加罗尔]]、[[海得拉巴|海得拉巴]]、[[孟买|孟买]]和中部其他地区使用。一些印度学校以乌尔都语作为第一语言，并有其课程和考试。在印度的[[伊斯兰学校|伊斯兰学校]]同时教授[[阿拉伯语|阿拉伯语]]和乌尔都语。一些报章如Daily Salar、Paasban，在班加罗尔等城市发行。

乌尔都语亦有在[[阿富汗|阿富汗]]的市区使用。而在[[南亚|南亚]]以外，亦有为数甚多的劳工，在[[波斯湾|波斯湾]]国家和[[沙特阿拉伯|沙特阿拉伯]]的主要城市中使用。在[[英国|英国]]、[[美国|美国]]、[[加拿大|加拿大]]、[[挪威|挪威]]和[[澳大利亚|澳大利亚]]的大城市，也有乌尔都语移民和他们的后裔在使用此语言。

在[[香港|香港]]，由於南亞裔人士眾多，不少的[[國際學校|國際學校]]都有专门開設[[烏爾都語|烏爾都語]]的課程。

=== 官方地位 ===
乌尔都语是巴基斯坦的[[官方语言|官方语言]]（英语也是巴基斯坦的官方语言）。虽然英语在[[精英|精英]]的圈子内使用，[[旁遮普语|旁遮普语]]也有大量的母语使用者，乌尔都语作为[[交际语|交际语]]被广泛使用。乌尔都语也是[[印度|印度]]、印度控制的克什米尔地区和[[安得拉邦|安得拉邦]]的官方语言。虽然政府学校和大多数邦份着重使用标准[[印地语|印地语]]，但在[[勒克瑙|勒克瑙]]和[[海得拉巴|海得拉巴]]的[[大学|大学]]，乌尔都语有被使用，并被看成是种有威望的[[语言|语言]]。

== 分类和相关语言 ==
乌尔都语属于[[印度雅利安语族|印度雅利安语族]]，也是[[印欧语系|印欧语系]]的一支。类似的方言分布在南亚从[[旁遮普|旁遮普]]到[[孟加拉|孟加拉]]的广泛区域中。这些语言有类似的语法结构和大部相同的词汇。[[旁遮普语|旁遮普语]]就很类似于乌尔都语：如果用[[梵文字母|梵文字母]]书写的话，操旁遮普语的人不难看懂乌尔都语。但是旁遮普语口语发音同乌尔都语却有很大差别。同乌尔都语最接近的是[[印地语|印地语]]。

== 方言 ==
烏爾都語有四種官方認可的方言，Dakhini、Pinjari、Rekhta和Modern Vernacular Urdu。

Modern Vernacular Urdu是目前最被廣泛傳播及使用的一種[[方言|方言]]，並且在：[[德里|德里]]、[[勒克瑙|勒克瑙]]（Lucknow）、[[卡拉奇|卡拉奇]]（Karachi）和[[拉合尔|拉合尔]]（Lahore）等地區使用。

[[Dakhini|Dakhini]]（也被称为Dakani、Deccani、Desia或Mirgan）在印度的[[马哈拉施特拉邦|马哈拉施特拉邦]]（Maharashtra）及其四周[[海得拉巴|海得拉巴]]（Hyderabad）使用。這種方言所使用的阿拉伯語及波斯語借詞較標準烏爾都語少。

== 语法 ==
{{main|印地語-烏爾都語語法}}

烏爾都語的[[名詞|名詞]]分[[阳性|阳性]]和[[阴性|阴性]]。不過，有一些[[字詞|字詞]]的[[性_(語法)|文法性]]分類比較特別，通常是從[[英語|英語]]來的新字詞，它們並無性別。
烏爾都語的名詞還有分[[單數|單數]]和[[眾數|眾數]]。
