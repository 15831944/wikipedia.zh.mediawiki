'''乌拉草'''（[[学名|学名]]：''{{lang|la|Carex meyeriana}}''），也称“靰鞡草”，是[[莎草科|莎草科]][[薹草属|薹草属]]的植物；曾被誉为中国[[东北三宝|东北三宝]]之一。

== 形态 ==
多年生[[草本|草本]]，紧密丛生，灰绿色，地上秆直立，三棱形，纤细坚硬；线性叶片，边缘外卷粗糙；小穗2～3个，聚生在茎的上部，顶生1个圆柱形雄性小穗，侧生卵形雌性小穗，苞片最下部的刚毛状，上部的鳞片状。

== 分布 ==
主要生长于中国的[[辽宁省|辽宁省]]、[[吉林省|吉林省]]、[[黑龙江省|黑龙江省]]和[[内蒙古自治区|内蒙古自治区]]境内的[[呼伦贝尔市|呼伦贝尔市]]、[[兴安盟|兴安盟]]、[[通辽市|通辽市]]和[[赤峰市|赤峰市]]及[[外兴安岭|外兴安岭]]以南(包括[[库页岛|库页岛]])的[[森林|森林]]、草甸或[[沼泽|沼泽]]地区。<ref>{{cite web| url =http://www.plant.ac.cn/latin/Cyperaceae/Carex-meyeriana-Kunth.htm| title =乌拉草| work =《中国高等植物数据库全库》| publisher =中国科学院微生物研究所| author =昆明植物研究所| accessdate =2009-12-05| deadurl =yes| archiveurl =https://archive.is/20121127153038/http://www.plant.ac.cn/latin/Cyperaceae/Carex-meyeriana-Kunth.htm| archivedate =2012-11-27}}</ref>

== 用途 ==
乌拉草具有保暖防寒的作用，可以用来填充在靰鞡鞋中。
烏拉草一般是用來當清熱利濕的草藥。主治淋症，濕疹，尿道炎，乳靡尿，腎臟炎，結石...等泌尿系統發炎及其相關的疾病，有些人誤以為是壯陽藥，這一點需要訂正。
