'''乌有之乡'''是[[中华人民共和国|中华人民共和国]]一个[[极左翼|极左翼]]的[[中国民族主义|中国民族主义]]及[[毛泽东思想|毛泽东思想]]政经评论网站<ref name="秦晉">{{cite book |author=秦晉 |title=求索與守望--中國民運江湖回望錄 |url = http://books.google.com/books?id=Oiu-RLPiq5AC&pg=PT151 |accessdate=24 December 2012 |publisher=秀威資訊科技股份有限公司 |ISBN = 978-986-5915-24-7 |pages=151–2 }}</ref>。該站于2003年由北京乌有之乡文化传播有限公司开设，创办人包括[[范景刚|范景刚]]、[[韩德强|韩德强]]等人。网站曾在2012年4月12日薄熙来被调查后被中国当局關閉，后重新开始运营。网站反对中国在[[改革开放|改革开放]]后采取的经济政策，认为[[毛泽东思想|毛泽东思想]]更适应中国发展。

== 历史 ==

2003年，[[北京航空航天大学|北京航空航天大学]]教师韩德强与学生展开网站筹备活动，北京乌有之乡文化传播有限公司成立。这时主要的活跃人士有[[杨帆|杨帆]]、[[左大培|左大培]]、[[高粱|高粱]]、[[杨斌|杨斌]]等人。<ref>{{cite web |url = http://www.zaobao.com/wencui/2012/03/hongkong120329k.shtml |title = 「乌有之乡」分崩 |language = en |website = zaobao.com |date= |accessdate = 2017-08-31 }}</ref>

乌有之乡在中国具有一定的社会影响力，其内容来源有网民投稿、学者文章和网站采集三个渠道；对该网站的评价，因浏览者立场不同而两极分化。根据2011年[[Alexa|Alexa]]的数据，乌有之乡的网站流量排在中国大陆第2373名。<ref>[http://www.alexa.com/search?q=www.wyzxsx.com&r=home_home&p=bigtop - Alexa排名和网站流量信息]，查阅于2011年12月28日。</ref>

2012年3月14日，[[中华人民共和国国务院总理|中华人民共和国国务院总理]][[温家宝|温家宝]]在[[两会|两会]]结束后的[[记者招待会|记者招待会]]上的发言号召加快政治体制改革，在回答[[路透社|路透社]]记者提问时更从侧面否定了[[重庆模式|重庆模式]]<ref>{{cite web |url = http://news.ifeng.com/mainland/special/2012lianghui/detail_2012_03/14/13190724_0.shtml |title=温家宝2012年3月14日答中外记者问(全文) |date=2012-03-14 |accessdate=2012-03-14 |website = 新华网 |agency = 新华社 }}</ref>。2012年3月15日，原[[中共重庆市委|中共重庆市委]]书记[[薄熙来|薄熙来]]和原[[重庆市人民政府|重庆市副市长]][[王立军|王立军]]被免职当天，乌有之乡以服务器维护为由，将主站主页换成公告<ref>[http://www.chinese.rfi.fr/中国/20120316-“乌有之乡”闭声-微博实名制上路 “乌有之乡”闭声 微博实名制上路]</ref>。2012年3月19日，乌有之乡首页恢复访问，但其“热点专题之重庆经验”栏目文章未显示。2012年4月6日，乌有之乡网站再次被当局以“妄议[[中国共产党第十八次全国代表大会|十八大]]”的原因关闭<ref>[http://www.chinese.rfi.fr/中国/20120406-乌有之乡网站因“妄议十八大”被关闭整顿一个月 法国广播电台：乌有之乡网站因“妄议十八大”被关闭整顿一个月 ]</ref>。

2012年4月6日12:00起，乌有之乡再次不能访问。原因是4月6日上午[[国务院新闻办公室|国务院新闻办公室]]九局、北京市网管办、[[北京市公安局|北京市公安局]]网络安全总队三家联合约谈网站负责人，认为网站存在大量发布违反《[[中华人民共和国宪法|中华人民共和国宪法]]》、恶意攻击国家领导人，妄议十八大的文章信息，要求网站关闭一个月并自我整顿。與此同時，「[[毛澤東旗幟網|毛澤東旗幟網]]」、「民聲網」、「[[四月网|四月青年網]]」等左派網站無法正常打開<ref>{{cite web |url = http://news.sina.com.hk/news/9/1/1/2628960/1.html |title=攻擊領導人「烏有之鄉」封站整改 |date=2012-04-07 |accessdate=2012-04-11 |work=星岛日报 |publisher=新浪香港 |deadurl=yes |archiveurl = https://archive.is/20120717120906/http://news.sina.com.hk/news/9/1/1/2628960/1.html |archivedate=2012-07-17 }}</ref>。

2012年4月11日，在中共中央正式宣布调查薄熙来后，乌有之乡网站以与以往红色设计风格不同的黑底白字显示“网站正在维护中”。12日起，被關閉的烏有之鄉網站以白底紅字顯示“公告  无论怎么关闭屏蔽，乌有之乡都会一直支持薄熙来！！！ 乌有之乡网站 2012年4月12日”；不久此页面消失，其官方[[新浪微博|新浪微博]]称遭到[[黑客|黑客]]攻击。

2012年4月16日，乌有之乡官方新浪微博及[[腾讯微博|腾讯微博]]被删除<ref name="谷歌网页缓存">{{cite web |url = http://www.1688.com.au/site1/news/cn/2012/04/13/188551.shtml |title=乌有之乡敏感时期“大喊”挺薄 |author=多维新闻 |date=2012-04-06|publisher=澳洲中文报业 |accessdate=2012-04-13 |deadurl=yes |archiveurl = https://web.archive.org/web/20121001020443/http://www.1688.com.au/site1/news/cn/2012/04/13/188551.shtml |archivedate=2012-10-01 }}</ref>，原網站亦沒有如期於整頓後重開；現只存烏有之鄉書社。

2012年新版网站于2012年2月底开始筹备，4月初完成初始版，因故间隔了一段时间，6月初复工完成第一个内测版。9月初开始第二个内测版的开发，至今开始内部测试。<ref>[http://www.szhgh.com/html/60/n-15760.html 快讯：乌有之乡网站2012新版开通测试 正邀请广大网友参与测试] {{webarchive|url=https://web.archive.org/web/20121128010409/http://www.szhgh.com/html/60/n-15760.html |date=2012-11-28 }}，红歌会网</ref>

目前，原域名wyzxsx.com已被改成了与“乌有之乡”无关的“无忧游戏网”，而“乌有之乡网刊”网站（wyzxwk.com）可以正常访问<ref>{{cite web |title=欢迎您访问乌有之乡网站 |url = http://www.wyzxsx.com/ |publisher=乌有之乡 |accessdate=2017-11-13 |deadurl=yes |archiveurl = https://web.archive.org/web/20140921053010/http://www.wyzxsx.com/ |archivedate=2014-09-21 }}</ref>。

== 自我定位 ==
乌有之乡网站自称是“宣传[[爱国主义|爱国主义]]、[[社会主义|社会主义]]和进步思想的[[政治|政治]]、[[经济|经济]]、[[文化|文化]]、历史评论”、“爱国主义和社会主义的话语平台”的网站。<ref>{{Cite web|url=http://www.wyzxsx.com/Article/Class22/201006/157002.html|title=乌有之乡到底是一个什么样的网站？|author=中国向左转|date=2010-06-07|publisher=|accessdate=|deadurl=yes|archiveurl=https://web.archive.org/web/20100701050946/http://www.wyzxsx.com/Article/Class22/201006/157002.html|archivedate=2010-07-01}}</ref>

=== 观点 ===
乌有之乡的多数文章自认为其代表社会中下层百姓的利益，希望中国能实现[[毛泽东|毛泽东]]时代的发展道路。对毛泽东充满敬意，视之为精神支柱，反对右派对毛泽东的批评及负面评价<ref>[http://www.wyzxsx.com/Article/Special/maols/Index.html 乌有之乡毛泽东历史真相专题] {{webarchive|url=https://web.archive.org/web/20120117014742/http://www.wyzxsx.com/Article/Special/maols/Index.html |date=2012-01-17 }}</ref>），比如署名“江淮碧玉”的文章中批评：（中国共产党）“在毛泽东去世后，立马变了面孔，对毛泽东反戈一击，可见，这一类的反毛者们是多么的猥琐、卑劣与龌龊……反毛者必将被钉上[[历史|历史]]的耻辱柱！”<ref>{{Cite web|url=http://www.wyzxsx.com/Article/Class22/201011/193835.html|title=反毛者必将被钉上历史的耻辱柱！|author=江淮碧玉|date=2010-11-06|publisher=|accessdate=|deadurl=yes|archiveurl=https://web.archive.org/web/20110112073410/http://www.wyzxsx.com/Article/Class22/201011/193835.html|archivedate=2011-01-12}}</ref>
 
在经济层面上，该网站反对中国在[[改革开放|改革开放]]后采取的“[[新自由主义|新自由主义]]”经济，认为这些政策导致了[[中美国|中美国]]现象。并认为执行它们的一大批现任官员是[[买办|买办]]性质的，出卖了中国利益，并认为经济上的[[自由化|自由化]]给人民生活带来了诸多苦难。在公开政策上反对邓小平主导的[[中国特色社会主义|中国特色社会主义]]，主张走“社会主义性质的[[改革开放|改革开放]]道路”，主张发展内需、产业独立，提倡公平正义、人民民主、[[四大自由|四大自由]]。外交上反对邓小平提出的韬光养晦政策，主张积极强势，反对美国的[[霸权主义|霸权主义]]。认为真正的社会主义只能是毛泽东模式。

这些文章中的一些认为毛泽东时代社会正义、社会平等，乃至认为建设成就均好于改革开放后的时代。社会中的新问题来自毛泽东领导的社会主义道路被[[修正主义|修正主义]]转变。比如该网站“网友之声”栏目中署名“张宏良”的《中华民族再次到了最危险的时候》，其中直接表示[[神化|神化]]毛，写道：“中国人也同样经历……是从否定“神”开始的。”<ref>{{Cite web|url=http://www.wyzxsx.com/Article/Class22/201003/139991.html|title=中华民族又一次到了最危险的时刻|author=张宏良|date=2010-03-25|publisher=|accessdate=|deadurl=yes|archiveurl=https://web.archive.org/web/20100331225633/http://www.wyzxsx.com/Article/Class22/201003/139991.html|archivedate=2010-03-31}}</ref>

乌有之乡对“[[毛泽东思想|毛泽东思想]]”持肯定、支持的态度，该网站其“时代观察”栏目署名“[[张宏良|张宏良]]”的文章称：“毛泽东根据中国国情，根据东方文化，根据人类未来的发展要求，为人类社会开辟出一条新的政治斗争模式，仅这份悲悯情怀，就已经超越了耶稣和释迦牟尼”<ref>{{Cite web |url= http://www.wyzxsx.com/Article/Class4/200901/64422.html |title= 高天滚滚寒流急，大地微微暖气吹——毛泽东诞辰115周年侧记 |author= 张宏良 |date= 2009-01-03 |publisher=  |accessdate=  |deadurl= yes |archiveurl= https://web.archive.org/web/20130525020052/http://www.wyzxsx.com/Article/Class4/200901/64422.html |archivedate= 2013-05-25 }}</ref>。

乌有之乡亦对中国社会思想向“资产阶级自由化”方向发展不满，强调要追求“社会主义教育”。在署名“奚兆永”的文章中写道：“为什么郭、杨两教授不以现成的《[[中华人民共和国宪法|中华人民共和国宪法]]》所明确规定的指导思想去教育学生……其原因显然是他们……要对这个社会制度和这个社会的价值观念进行公开的反对，不仅自己要站在对立面进行‘批评’，而且要让学生们也接受他们的这种反对态度。”<ref>{{Cite web|url=http://www.wyzxsx.com/Article/Class14/201005/153597.html|title=教我们的大学还像社会主义大学吗？--驳法大反毛演讲|author=奚兆永|date=2010-05-24|publisher=|accessdate=|deadurl=yes|archiveurl=https://web.archive.org/web/20100702155710/http://www.wyzxsx.com/Article/Class14/201005/153597.html|archivedate=2010-07-02}}</ref>

== 乌有之乡与重庆模式 ==
关于重庆课题，据2010年1月《新华网》报导，中国特色社会主义在重庆的实践课题有12项，主持人[[王伟光|王伟光]]、[[方滨兴|方滨兴]]、[[胡鞍钢|胡鞍钢]]等。“人民大学的重庆课题组编著《重庆新事》，认真总结重庆经验，透视重庆经验的核心价值。”<ref>{{Cite web |url= http://cd.qq.com/a/20100108/001767.htm |title= 重庆启动“中国特色社会主义在重庆的实践”系列研究 |author= 新华网 |date= 2010-01-08 |publisher= 腾讯大成网|accessdate=}}</ref>

据《[[悉尼先驱晨报|悉尼先驱晨报]]》2012年5月23日报道，中国的改良左派聚集在前中共重庆市委书记[[薄熙来|薄熙来]]的身后，指责当局捏造事实对付薄熙来，让国家（政权）面临解体的风险。在薄熙来因“严重违纪”被暂停了中共中央政治局委员职务后六周，乌有之乡给全球的传媒发送了一份声明，挑战当局，呼吁给薄熙来平反。乌有之乡称，“乌有之乡相信，薄熙来与王立军案是改革开放以来最大的政治冤案。”乌有之乡在声明中，轰炸当局的“邪恶方式”，其中包括编造薄熙来的犯罪和腐败指控。该声明称，“因此，这些可能被勾结西方敌对势力的国内卖国势力所用，来煽动社会动乱，分裂国家，把国家带入深渊。”声明还称“[[毛泽东思想|毛泽东思想]]万岁，重庆路线万岁。”<ref>{{Cite web |url= http://www.smh.com.au/world/maoists-accuse-authorities-of-concocting-the-case-over-fallen-hero-bo-20120522-1z36g.html |title= 毛派乌有之乡给全球传媒发声明 轰炸中央 |author= John Garnaut |date= 2012-05-23 |publisher= 《悉尼先驱晨报》|accessdate= }}</ref>
