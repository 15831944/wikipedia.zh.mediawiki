'''碘化镧'''是一种[[无机化合物|无机化合物]]，[[化学式|化学式]]为LaI<sub>3</sub>。

==制备==
* 碘化镧可以由单质直接化合得到：
:: <math>\mathsf{2La + 3I_2 \ \xrightarrow{\Delta}\ 2LaI_3 }</math>

* 也可由氧化镧和碘化铵加热反应得到：
:: <math>\mathsf{La_2O_3 + 6NH_4I \ \xrightarrow{350^oC}\ 2LaI_3 + 6NH_3 + 3H_2O }</math>
