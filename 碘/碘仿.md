'''碘仿'''，即'''三碘甲烷'''，是化学式为[[碳|C]][[氢|H]][[碘|I]]<sub>3</sub>的化合物，属于[[卤仿|卤仿]]的一员。它是淡黄色的挥发性晶体，有甜味和刺激性气味。微溶于[[甘油|甘油]]、[[石油醚|石油醚]]和[[醇|醇]]（78 g/L, 25 °C），有限可溶于[[氯仿|氯仿]]、[[乙酸|乙酸]]和[[乙醚|乙醚]]（136 g/L, 25 °C），易溶于[[苯|苯]]和[[丙酮|丙酮]]（120 g/L, 25 °C）。

== 物理性质 ==
35~92°C时，碘仿的升华焓为69.9kJ.mol<sup>−1</sup>。其分子参数为：d(C-I) = 2.12 ± 0.04 Å、d(I-I) = 3.535 ± 0.01 Å及d I-C-I = 113°，[[偶极矩|偶极矩]]1D，空间群''P6<sub>3</sub>''，[[晶胞参数|晶胞参数]]''a'' = 6.83 Å，''c'' = 7.52 Å。

5.63MPa时，碘仿的[[临界点_(热力学)|临界点]]为584.85°C。折射率1.786（D, 20 °C）。

== 合成 ==
它首先由Georges Serrulas于1822年合成，[[分子结构|分子结构]]则由[[杜马|杜马]]（Jean-Baptiste Dumas）在1834研究清楚。

碘仿可由[[卤仿反应|卤仿反应]]合成，原料是碘、氢氧化钠和以下任意一种化合物：
* 甲基[[酮|酮]]：CH<sub>3</sub>COR，R是有机基团
* [[乙醛|乙醛]]：CH<sub>3</sub>CHO
* [[乙醇|乙醇]]：CH<sub>3</sub>CH<sub>2</sub>OH
* 甲基二级[[醇|醇]]：CH<sub>3</sub>CHROH，R是烷基或芳基

[[File:Iodoform_synthesis.png|File:Iodoform synthesis.png]]

因[[碘|碘]]在鹼性環境中易發生[[歧化反應|歧化反應]]而分解，故有時候會以漂白水(含有強氧化力且為鹼性的[[次氯酸鈉|次氯酸鈉]])直接加入含[[碘離子|碘離子]]及上述[[有機化合物|有機化合物]]的混合溶液中製備碘仿。

以上的碘仿反应常用于检验[[甲基酮|甲基酮]]的存在，称为[[碘仿试验|碘仿试验]]。若反应后出现黄色碘仿沉淀，则可证明试样中存在甲基酮。

== 反应 ==
有些试剂（比如[[氢碘酸|氢碘酸]]）可将碘仿转化为[[二碘甲烷|二碘甲烷]]。

也可将碘仿转化为[[二氧化碳|二氧化碳]]：碘仿与[[硝酸银|硝酸银]]反应先生成[[一氧化碳|一氧化碳]]，再经[[硫酸|硫酸]]和[[五氧化二碘|五氧化二碘]]氧化。

== 用途 ==
20世纪初开始，碘仿被用作伤口[[防腐药|防腐药]]，现已被更好的物质代替。
歷來作為[[止血|止血]]首選，美軍隨身包必備[[碘仿紗布|碘仿紗布]]密封包。
