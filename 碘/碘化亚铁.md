'''碘化亚铁'''是[[无机化合物|无机化合物]]，化学式为FeI<sub>2</sub>。<ref>[http://ull.chemistry.uakron.edu/erd/Chemicals/26000/24496.html University of Akron Chemical Database] {{webarchive|url=https://archive.is/20121212135447/http://ull.chemistry.uakron.edu/erd/Chemicals/26000/24496.html |date=2012-12-12 }}</ref>

== 制备 ==
* 直接用[[碘|碘]]与[[铁|铁]]反应：
: <math>\mathsf{Fe + I_2 \ \xrightarrow{}\ FeI_2 }</math>

* 加热碘与[[硫化铁|硫化铁]]：
: <math>\mathsf{FeS + I_2 \ \xrightarrow{T}\ FeI_2 + S }</math>

* 加热分解Fe(CO){{sub|4}}I{{sub|2}}：
: <math>\mathsf{Fe(CO)_4I_2 \ \xrightarrow{T}\ FeI_2 + 2CO }</math>

== 物理性质 ==
红棕色[[三方晶系|三方晶体]]，[[空间群|空间群]]'''''P {{overline|3}}m1'''''，其中''a'' = 0,404 nm，''c'' = 0,675 nm，Z = 1。

易溶于水。

可形成FeI<sub>2</sub>·''n'' H<sub>2</sub>O，其中''n'' = 2, 4, 6, 9。

== 化学性质 ==
* 在空气中加热时，被[[氧气|氧气]]氧化：
: <math>\mathsf{4FeI_2 + 3O_2 \ \xrightarrow{T}\ 2Fe_2O_3 + 4I_2 }</math>

== 用途 ==
在有机反应中作为催化剂使用。
