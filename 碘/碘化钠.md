'''碘化鈉'''是一種白色[[晶体|晶体]]，實驗式為[[鈉|Na]][[碘|I]]，用於輻射偵測、治療[[碘缺乏症|碘缺乏症]]及作為[[Finkelstein反應|Finkelstein反應]]的反應物。
无臭，味咸而微苦。有潮解性。在空气和水溶液中逐渐析出碘而变黄或棕。1g溶于0.5ml水，约2ml 乙醇，1ml 甘油，溶于 丙酮。加热到64.3℃能溶于自身的结晶水中。有刺激性。

==物理性质==
碘化钠是一种白色固体，易溶于水。
==化学性质==
碘化钠是一种强[[还原剂|还原剂]]，可以将Fe<sup>3+</sup>、Cu<sup>2+</sup>、MnO<sub>4</sub><sup>-</sup>、[[卤素|卤素]]等还原，如：
: 2 FeCl<sub>3</sub> + 2 NaI → 2 FeCl<sub>2</sub> + 2 NaCl + I<sub>2</sub>
: 2 NaI + Br<sub>2</sub> → 2 NaBr + I<sub>2</sub>
和其他[[碘化物|碘化物]]一样，碘化钠可以和[[碘|碘]]作用，形成I<sub>3</sub><sup>-</sup>：
: NaI + I<sub>2</sub> → NaI<sub>3</sub>
碘化钠在水中可以解离出I<sup>-</sup>，参与沉淀反应：
: Ag<sup>+</sup> + I<sup>−</sup> → AgI↓
: Bi<sup>3+</sup> + 3I<sup>−</sup> → BiI<sub>3</sub>↓

碘化鈉通常被用來治療和預防[[碘缺乏症|碘缺乏症]]。

碘化鈉也用於[[聚合酵素連鎖反應|聚合酵素連鎖反應]]（PCR）。其丙酮溶液也用於Finkelstein反應（将[[氯代烃|氯代烃]]转化为[[碘代烃|碘代烃]]），机理是[[氯化鈉|氯化鈉]]不溶於丙酮而碘化钠可溶，从而促使反應进行。

: R-Cl + NaI → R-I + NaCl

碘化鈉結晶中添加[[鉈|鉈]]，NaI(Tl)，當受到电离輻射时，會放出[[光子|光子]]（[[閃爍|閃爍]]（scintillation）），因此應用於[[閃爍探測器|閃爍探測器]]、傳統的[[核醫學|核醫學]]、[[地球物理學|地球物理學]]、[[核物理學|核物理學]]和環境測量等方面。NaI(Tl)是使用最廣泛的閃爍材料，而且具有最高的光輸出。碘化鈉晶體通常附有[[光電倍增管|光電倍增管]]（photomultiplier tube），並密封於密封的裝置中，因為碘化鈉具[[潮解性|潮解性]]。微調部分參數，如[[輻射硬度|輻射硬度]]（radiation hardness）、餘輝（afterglow）及透明度等，可獲得不同條件的晶體生長。結晶中添加物的比率較高時，可用於高光譜品質的[[X光|X光]]探測器。在此應用中，碘化鈉可同時作為單晶體和多晶體使用。
