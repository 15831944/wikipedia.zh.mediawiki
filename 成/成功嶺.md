'''成功嶺營區'''是[[中華民國國軍|中華民國國軍]]位於[[臺中市|臺中市]][[烏日區|烏日區]][[大肚臺地|大肚臺地]]上的[[軍事基地|軍營]]，為[[國防部後備指揮部|國防部後備指揮部]]中部地區後備部隊訓練中心、[[陸軍步兵第一O四旅|陸軍步兵第一O四旅]]（常山部隊）、[[陸軍步兵第三O二旅|陸軍步兵第三O二旅]]（虎威部隊）、<!--[[國防部憲兵指揮部|憲兵]]排、已裁撤 待查-->[[陸軍後勤指揮部|陸軍後勤指揮部]][[第五地區支援指揮部|第五地區支援指揮部]]衛生營、彈藥分庫以及[[內政部役政署|內政部役政署]][[替代役_(中華民國)|替代役]]訓練班構成的營區。

== 歷史 ==
*[[臺灣日治時期|日治時期]]，成功嶺原為日本[[臺灣賽馬|賽馬場]]（競馬場）及[[高爾夫球場|高爾夫球場]]<ref>{{cite web|url=http://taiwanairpower.org/blog/?p=9689|title=昭和時期台灣的競馬場空照圖Aerial Photos of Horse Racing Tracks on Formosa  @  TaiwanAirBlog|website=taiwanairpower.org}}</ref>。
*1934年12月1日，設立「[[遊園地前車站|遊園地前車站]]」，僅停[[汽油車|汽油車]]。
*1942年，因[[太平洋戰爭|太平洋戰爭]]時汽油短缺，汽油車停駛。
*[[臺灣戰後時期|臺灣戰後時期]]，[[中華民國國民政府|中華民國國民政府]]接收為成功基地，之後成為「成功嶺訓練中心」，是[[中華民國陸軍|中華民國陸軍]]新兵訓練與大專生集訓的地方；大專集訓為臺灣1980年代以前出生的男[[大專|大專]]生共同回憶。40年間，曾有130多萬名大專生在此接受軍事洗禮<ref>[http://www.taiwan-panorama.com/show_issue.php?id=199928802110C.TXT 告別成長年代──成功嶺邁入歷史] （光華雜誌、1999年2月號110頁）</ref>。
[[File:Uniforms_of_Universitiy_and_College_Students_Military_Training_in_Chengkungling_20131012.jpg|thumb]]
*1970年1月，執行「嘉禾二號專案」，陸軍第四十一師（常山部隊）移駐成功嶺營區，改編為陸軍預備第四師（學一師），執行大專生與新兵訓練任務<ref name="mnd.gov.tw">{{Cite web |url=http://www.mnd.gov.tw/Publish.aspx?cnid=67&p=56124 |title=成功嶺新訓旅移編　陸軍步兵一０四、三０二旅編成 |date=2013-01-09|access-date=2014-10-09 |archive-url=https://web.archive.org/web/20141014100443/http://www.mnd.gov.tw/Publish.aspx?cnid=67&p=56124 |archive-date=2014-10-14 |dead-url=yes }}</ref><ref>{{cite web|url=http://m.blog.sina.com.tw/tel336358/article.php?entryid=639060|title=仙山聖地英雄島::常山部隊-隊史::新浪部落|first=|last=仙山聖地英雄島|publisher=}}</ref>。
*1976年8月11日，陸軍預備第四師改編為陸軍步兵第一O四師<ref name="mnd.gov.tw"/>。
*1984年執行，「陸精四號專案」陸軍步兵第一五一師（黃龍部隊）移駐成功嶺營區，此時成功嶺營區有二個師。
*1986年任務調整，陸軍步兵第一五一師與陸軍步兵第三O二師，部隊不動番號軍旗互換（黃龍部隊改稱虎威部隊）。<ref>{{cite web |url=http://museum.mnd.gov.tw/UserFiles/Image/afm/photo_9303_1_32.jpg |title=陸軍步兵第三O二師 |accessdate=2015-04-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20170317054757/http://museum.mnd.gov.tw/UserFiles/Image/afm/photo_9303_1_32.jpg |archivedate=2017-03-17 }}</ref><ref>{{cite web |url=http://museum.mnd.gov.tw/UserFiles/Image/afm/photo_9303_1_22.jpg |title=陸軍步兵第一五一師黃龍部隊 |accessdate=2015-04-12 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304203643/http://museum.mnd.gov.tw/UserFiles/Image/afm/photo_9303_1_22.jpg |archivedate=2016-03-04 }}</ref>
*1999年大專集訓廢除後，成功嶺以負責陸軍新兵訓練的工作為主。在大專集訓的空檔時間與大專集訓廢除後，成功嶺均負責[[中華民國陸軍|陸軍]]新兵訓練的工作，即所謂「新兵訓練中心」。在臺灣各處有都有類似的陸軍新訓中心，如：[[宜蘭縣|宜蘭]][[金六結|金六結]]、[[新竹縣|新竹]][[關西鎮_(臺灣)|關西]]、[[苗栗縣|苗栗]][[斗煥坪|斗煥坪]]、[[嘉義縣|嘉義]][[中坑|中坑]]、[[臺南市|臺南]][[官田區|官田]]、[[臺南市|臺南]][[大內區|大內]]以及[[花蓮縣|花蓮]][[北埔|北埔]]等。一般新訓中心僅為一個旅的編制，惟成功嶺佔地比一般新訓中心廣大，所以當時設有二個師，分別是陸軍步兵第一O四師（常山部隊，隊徽為獅頭，下轄第三一O、三一一、三一二步兵旅及砲兵指揮部）和陸軍步兵第三O二師（虎威部隊，隊徽為虎頭，下轄第九O四、九O五、九O六步兵旅及砲兵指揮部）。
*1999年8月1日，因應「[[精實案|精實案]]」配合「精簡高層，充實基層」政策執行，師級單位裁編成旅級單位，陸軍步兵第302師解編成陸軍步兵第102旅，陸軍步兵第104師解編成陸軍步兵第104旅<ref>[https://pic.pimg.tw/andro0918/1434564077-345820687_n.jpg 沿革史]</ref><ref>{{cite web|url=http://m.xuite.net/blog/liu6534588/twblog/169248147|title=變革中的台灣陸軍 （二）從精實案到精進案 (20101230新增文) H0t @|website= Liu653 的影音部落格 :: 隨意窩 Xuite日誌|date=2010-08-18}}</ref>。
*2004年4月1日，因應「[[精進案|精進案]]」配合「國防二法」實施，「常後分立」政策啟動，「常備打擊、後備守土」，陸軍步兵102旅改銜為後備第904旅，陸軍步兵第104旅改銜為後備第905旅，移編至[[國防部後備司令部|國防部後備司令部]]。
*2006年8月9日，國防部後備司令部設立[[中臺灣|中部地區]]後備部隊訓練中心，於成功嶺營區舉行編成授旗典禮。
*2008年，[[中華民國內政部|內政部]]實施《[[替代役實施條例|替代役實施條例]]》，[[內政部役政署|內政部役政署]]設立[[替代役_(中華民國)|替代役訓練班]]。
*2008年1月3日，因應「[[精進案|精進案]]」配合精簡後勤單位，業務外包交由民間業者經營，許可[[便利商店|便利商店]]進駐成功嶺營區，為國軍營區內首家[[便利商店|便利商店]]（目前已有2間分店進駐）。
[[File:FamilyMart_in_No.188_Armed_Forces_Camp_Supply_Station_20121006.jpg|thumb]]
*2010年3月4日，中部地區後備指揮部「模擬訓練館」正式啟用。
*2013年1月1日，因應「[[精粹案|精粹案]]」，配合「國防組織六法」實施，後備司令部裁編為[[國防部後備指揮部|後備指揮部]]，原先所屬後備新訓旅全部移編至陸軍，後備第905旅改銜為陸軍步兵第104旅，後備第904旅改銜為陸軍步兵第302旅，隸屬[[國防部陸軍司令部|國防部陸軍司令部]]。

== 營區設施 == 
=== 營門 ===
*一號門：[[烏日區|烏日區]][[中山路_(烏日區)|中山路三段]]，旁邊有[[OK便利店_(臺灣)|OK便利商店]]左轉到底就是一號門。
*二號門：一號門前右轉成功東路到底就是二號門。
*三號門：[[烏日區|烏日區]][[學田路|學田路]]，旁邊有[[全家便利商店|全家便利商店]]和[[臺糖|臺糖]]加油站。
*四號門：[[南屯區|南屯區]]同安西巷，旁邊有成功副供站，由陸軍步兵第三O二旅步兵第三營管理（不對外開放）。
