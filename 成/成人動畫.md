'''成人動畫'''，是以[[成人|成人]]為對象而製作，含有重度[[暴力|暴力]]、[[獵奇|血腥]]、[[突發驚嚇|驚嚇]]或[[性交|性交]]等內容的[[动画|动画]]。各地對以上標準各有不同，而且各地的成人歲數定義也不同，不過大多國家將成年歲數定為18歲。成人動畫是禁止[[未成年人|未成年人]]購買或觀看的，也有部分國家完全禁止成人動畫在市面發售。中文语境下，成人动画一般又被称为“里番”（一般向动画为“表番”）<ref group="註">需要注意的是，中文语境下的“里番”与日文中的“{{link-ja|裏番組|裏番組}}”（{{lang|ja|裏番組}}）概念完全不同。日语“裏番組”一般指同一黃金时段其他电视台的节目，或是人气相对较低的节目。</ref>

== 現狀 ==
[[File:Toys_Otaku_Buy_3.jpg|200px]]
=== 日本 ===
[[日本|日本]]的成人動畫（{{lang|ja|アダルトアニメ}}）又叫「18禁動畫」、「情色動畫」（{{lang|ja|エロアニメ}}），劇情內容以[[性交|性交]]為主。最早成人動畫是1932年《{{lang|ja|すヾみ舟}}》，1984年2月21日發售最早成人[[OVA|OVA]]《{{lang|ja|ロリータアニメ 雪の紅化粧 〜少女薔薇刑〜}}》。<ref>{{cite web |title=ジブリのエロアニメ!?『バルテュス ティアの輝き』 OVA創世記の大傑作！天空のエロ・ラピュタ！ |url=https://otapol.jp/hentai/2016/12/post-208.html |website=おたぽる@hentai！！|language=ja |date=2016/12/02}}</ref>後來除原創作品外也開始有改編自[[成人漫畫|成人漫畫]]、[[日本成人遊戲|成人遊戲]]、[[輕小說|輕小說]]的作品，以OVA形式發售。

成人動畫製作公司非常多，同时政府对动画产业支持，使得日本成为世界上產量最多的國家。日本成人動畫主要是销往[[歐美|歐美]]國家，[[無修正|無碼]]動畫在本土禁止銷售，只允許在本土銷售經過局部[[馬賽克_(影像處理)|馬賽克處理]]的成人動畫片。

=== 中國大陆 ===
在[[中國大陸|中國大陸]]，[[中華人民共和國法律|中華人民共和國法律]]禁止發售成人動畫。

=== 港澳臺 ===
港澳臺部分影音店有成人動畫出售或出租，不過這些動畫多數是有碼動畫，無碼的只能從外國訂購或從網上下載。臺灣曾經存在的成人動畫代理商有[[凡華影視|凡華影視]]、[[新潮館|新潮館]]等。

=== 歐美 ===
在[[歐美|歐美]]等國家甚少有專門製作成人動畫的公司，主要是從日本進口。但政府對此監管非常嚴格，銷售或出租錄像帶、光碟等需要核實顧客的年齡。

== 世界觀設定 ==
* 真實世界
: 真實世界是類似現實社會的[[性犯罪|性犯罪]]，[[性侵犯|性侵犯]]或[[偷拍|偷拍]]威脅是最常用的手段。
* 幻想世界
: 幻想世界中，性侵犯並不是犯罪，[[性交|性交]]、[[口交|口交]]是隨意的，如同[[握手|握手]]一樣僅作為一種社交禮儀。甚至擁有科幻世界觀的設定，也可能出現一些不存在的[[機器|機器]]或[[生物|生物]]。

== 聲優 ==
日本成人動畫的女聲優通常是專接成人動畫的工作，一般動畫的工作機會不多，即使有也只能是路人的角色。另外因為成人動畫的高薪，也有一般動畫的三、四線女聲優願意接成人動畫的工作，但如《天使之劍》使用知名聲優的情況並不多見。有知名聲優會使用另一藝名配成人動畫角色。相比之下男聲優幾乎都接成人動畫的工作，不少男聲優都是從成人動畫而知名。

== 語言 ==
動畫的語言往往根據發行地而定，日本製作的動畫通常只配[[日語|日語]]，而出口至歐美地區則會被代理商重新配上[[英語|英語]]或其他歐洲語言，香港也曾發售[[粵語|粵語]]配音的成人動畫VCD、影带。台灣地區也有國語配音。
