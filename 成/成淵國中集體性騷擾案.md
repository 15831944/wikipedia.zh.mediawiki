'''成淵國中集體性騷擾案'''，是臺灣臺北市立成淵國民中學（今[[臺北市立成淵高級中學|臺北市立成淵高級中學]]）在民國84年（1995年）發生的男學生[[性騷擾|性騷擾]]女學生案件。依據校方案發後作成的調查報告顯示，8名男學生對10名女同學，於不同時間點在教室、廁所等地強行撫摸、碰撞、窺看[[胸部|胸部]]及[[陰道|下體]]，構成性騷擾行為。<ref name="邱淑宜1">{{Cite news|author=邱淑宜|title={{單雙書號轉換|校長報告:受害女生有10人 涉案男生有八人 六人將轉學、兩人校規處分}}|work=《[[聯合晚報|聯合晚報]]》|publisher=[[聯合報系|聯合報系]]|location=臺北市|language=zh-tw|date=1995-12-18|accessdate=2017-04-23}}</ref>案件經過媒體、議員公開後，引起社會輿論譁然；輿論除要求追究加害者及校方法律責任外，也要求改革校園安全體系，是臺灣國民教育史上重要的校園性犯罪案件之一。<ref name="秦夢群">{{Cite book|author=秦夢群|title=《教育行政實務與應用》|chapter={{單雙書號轉換|教育行銷與危機處理}}|pages=436-439|publisher=五南文化|location=臺北市|year=2014|language=zh-tw}}</ref>

== 案情 ==
成淵國中是一所男女合校的市立[[國民中學|國民中學]]，位於臺北市[[大同區_(臺北市)|大同區]]。案件公開時，此案的加害、被害學生都就讀該校2年8班。<ref name="監察院公報">{{Cite journal|title=院臺壹丙字第四五五八號|url=http://twinfo.ncl.edu.tw/tiqry/hypage.cgi?HYPAGE=search/merge_pdf.hpg&sysid=D9603292&jid=79006615&type=g&vol=85050800&page=%E9%A0%811042-1046|format=pdf|journal=《監察院公報》|issue=2052|pages=1042-1046|year=1996|publisher=[[監察院|監察院]]|location=臺北市|language=zh-tw|accessdate=2017-04-24}}</ref>{{備註|[[九年一貫課程|九年一貫課程]]實施前，國中生分三個年級，稱為「一年級」（國一）、「二年級」（國二）、「三年級」（國三）。}}時任[[中華民國教育部|中華民國教育部]]部長的[[郭為藩|郭為藩]]於案發後，在[[立法院教育及文化委員會|立法院教育及文化委員會]]陳述案情：民國84年（1995年）9月，幾個男學生一起看電視臺放映的[[色情片|色情片]]、並在學校裡公開討論色情劇情，進而在同年10月27日第7節下課時，於教室性騷擾1名女同學。<ref name="立法院公報">{{Cite journal|author=立法院公報處|title=教育委員會紀錄|url=http://npl.ly.gov.tw/npl/report/841227/2.pdf|format=pdf|journal=《立法院公報》|volume=85|issue=3|pages=273|year=1996|publisher=[[立法院|立法院]]|location=臺北市|language=zh-tw|accessdate=2017-04-23}}</ref>該班學生於同年11月初向[[級任導師|級任導師]]舉發此事，但導師並未處理。11月10日第2節下課，幾名男同學闖入女生廁所，開啟廁所門後離開。11月15日第5節，自習課期間，又有4名男學生性騷擾1名女同學。<ref name="立法院公報"/>更早之前，根據[[監察院|監察院]]的調查報告，加害、受害學生就讀國中一年級時，加害學生們就曾在工藝課教室，將女同學壓在桌上猥褻撫摸胸部、下體，可見這類行為對涉案學生來說，並非突發個案。<ref name="監察院公報"/>

另外，《[[民生報|民生報]]》也報導在同年12月14日，該班5、6名男學生在全班同學面前，強脫2、3名女同學的衣物，猥褻撫摸其胸部、下體。<ref name="林麗雪">{{Cite news|author=林麗雪|title={{單雙書號轉換|國中男生集體猥褻女同學 北市成淵爆發男生性侵犯案 議員關切 教局善後}}|work=《[[民生報|民生報]]》|publisher=聯合報系|location=臺北市|language=zh-tw|date=1995-12-16|accessdate=2017-04-23}}</ref>當晚受害女學生回家告訴家長，隔日（12月15日）家長偕同[[新黨|新黨]]籍臺北市議員[[楊鎮雄|楊鎮雄]]及媒體等20餘人到校追究責任，事件才因此公開。<ref name="監察院公報"/><ref name="林麗雪"/>

== 調查 ==
事件公開後隔日（12月16日），[[臺北市政府教育局|臺北市政府教育局]]長[[吳英璋|吳英璋]]赴成淵國中指揮調查；<ref name="立法院公報"/>同時，該校訓導處也將加害、被害學生所在班級的全部學生帶離教室，於校園某處個別詢問學生案情，並由輔導室進行[[心理諮商|心理輔導]]。<ref name="邱淑宜2">{{Cite news|author=邱淑宜|title={{單雙書號轉換|成淵國中性侵犯案 同班男生哭了 發誓保護女生}}|work=《聯合晚報》|publisher=聯合報系|location=臺北市|language=zh-tw|date=1995-12-17|accessdate=2017-04-23}}</ref>在多方介入下，加害、受害學生的家長於同年12月16日召開協調會，受害家長對出席協調會的教育局長吳英璋提出6項訴求，分別為：

{{quote|
對加害者部分：<ref name="邱淑宜1"/>
# 以校規處分，並且轉學，並接受輔導和追蹤管理。
# 校方要負起責任，做好亡羊補牢。
# 家長要出具書面保證。

對受害者部分：<ref name="邱淑宜1"/>
# 尊重受害者意願，調班、轉學，教育局必須秘密安排，不受[[學區|學區]]的限制。
# 應有專業人員輔導課業、心理。
# 保護受害人不再受到二次傷害。}}

根據《[[聯合晚報|聯合晚報]]》報導，受害者家長表示能同理加害者家長的心情，也不希望加害者在[[標籤效應|標籤效應]]和法律追究下從此墮落，只是加害者曾揚言報復，才提出前述訴求。<ref name="邱淑宜2"/>吳英璋則表示同意所有訴求，並表示教育局已尋求專業輔導人員介入輔導學生。<ref name="邱淑宜1"/>不過，受害者家長仍對成淵國中校方處置作為表示不滿，質疑校方調查報告避重就輕、隱瞞事實。<ref>{{Cite news|author=劉時榮|title={{單雙書號轉換|重大情節不提 校方說可「意會」 家長指責避重就輕 至少六名女生受害}}|work=《[[聯合報|聯合報]]》|publisher=聯合報系|location=臺北市|language=zh-tw|date=1995-12-17|accessdate=2017-04-23}}</ref>當時代理[[臺北市長|臺北市長]]的[[陳師孟|陳師孟]]{{備註|當時的臺北市長[[陳水扁|陳水扁]]出國訪問，將市政委由副市長陳師孟代理。}}也認為校方早就知情，卻放任加害者侵犯女同學，嚴重失職，要求市府教育局必須懲處校方人員。<ref>{{Cite news|author=秦富珍|title={{單雙書號轉換|陳師孟:校方失職 一定懲處}}|work=《聯合晚報》|publisher=聯合報系|location=臺北市|language=zh-tw|date=1995-12-18|accessdate=2017-04-23}}</ref>

協調會後，加害學生由家長辦理轉學或休學，其行為涉及公然猥褻、妨害性自主等罪嫌，為公訴罪，由[[臺北市政府警察局|臺北市少年警察隊]]偵辦調查。<ref>{{Cite news|author=詹三源、董智森、張錦宏|title={{單雙書號轉換|成淵受害學生如申請國賠 陳師孟說北市府接受}}|work=《聯合報》|publisher=聯合報系|location=臺北市|language=zh-tw|date=1995-12-19|accessdate=2017-04-23}}</ref>此外，成淵國中校長陳添丁也因處置不當，遭[[臺北市政府|臺北市政府]]停職並移付懲戒，由教育局督學林信耀代理校長職務。<ref>{{Cite news|author=楊金嚴、劉時榮|title={{單雙書號轉換|學生性侵犯案 北市市政會議決定:成淵國中校長陳添丁停職 移付懲戒}}|work=《民生報》|publisher=聯合報系|location=臺北市|language=zh-tw|date=1995-12-20|accessdate=2017-04-23}}</ref>

== 影響 ==
=== 涉案關係人 ===
民國85年（1996年），[[監察院|監察院]]以成淵國中有多項行政和教育疏失，違反《[[s:國民教育法|國民教育法]]》、《國民教育法施行細則》及《[[s:公務員服務法|公務員服務法]]》規定，通過[[彈劾|彈劾]]成淵國中前校長陳添丁、教務主任陳瑞明、訓導主任林克雄、輔導主任王美玉等學校主管。<ref name="監察院公報"/>至於涉案學生，根據人本教育基金會在案發後隔年（1996年）的追蹤調查，6位涉案情節較重的加害學生全都離開成淵國中，其中1名更遠走國外，而受害學生則留在學校繼續就讀，不願多談此案。<ref name="秦夢群"/>

=== 輿論反應 === 
成淵國中集體性騷擾案引發社會輿論高度矚目，[[臺北市議會|臺北市議會]]教育審議委員會要求成淵國中校長、主任到議會接受質詢；<ref>{{Cite news|author=楊金嚴、劉時榮|title={{單雙書號轉換|國中男生集體猥褻女同學}}|work=《民生報》|publisher=聯合報系|location=臺北市|language=zh-tw|date=1995-12-16|accessdate=2017-04-23}}</ref>其後市議會更召開「由台北市成淵國中事件看青少年校園性侵害問題座談會」，與會議員、學者、社工皆主張應落實法治教育，重視青少年教育問題。<ref>{{Cite news|author=林永勝|title={{單雙書號轉換|座談會：化解校園性侵害　議員、專家強調教育著手}}|work=《民生報》|publisher=聯合報系|location=臺北市|language=zh-tw|date=1995-12-20|accessdate=2017-04-23}}</ref>[[臺灣女性學學會|臺灣女性學學會]]理事長劉毓秀、[[臺北市女性權益促進會|臺北市女性權益促進會]]理事[[黃淑英|黃淑英]]及[[世新大學|世新大學]]社會心理學教授羅燦煐都指出，臺灣社會對男生的[[性教育|性教育]]嚴重不足，是導致此案男學生性觀念偏差的主因。<ref>{{Cite news|author=詹三源|title={{單雙書號轉換|落實兩性平權教育 可消弭校園性暴力}}|work=《聯合報》|publisher=聯合報系|location=臺北市|language=zh-tw|date=1995-12-21|accessdate=2017-04-23}}</ref>

此外，[[臺北市婦女救援基金會|臺北市婦女救援基金會]]則主張在性犯罪案件中，每個當事人都是受害者，強調教育當局應仔細與該校學生討論案情，落實[[機會教育|機會教育]]，導正觀念。<ref>{{Cite news|author=詹三源|title={{單雙書號轉換|性侵犯 別當小事情處理 婦援會呼籲落實學生身體安全教育}}|work=《民生報》|publisher=聯合報系|location=臺北市|language=zh-tw|date=1995-12-19|accessdate=2017-04-23}}</ref>[[電視文化研究委員會|電視文化研究委員會]]則因加害男學生曾收看電視臺播放的色情片，而要求[[行政院新聞局|行政院新聞局]]全面檢討[[電視分級制度|電視分級制度]]，加強管理[[有線電視|有線電視]]節目和廣告。<ref>{{Cite news|author=楊起鳳|title={{單雙書號轉換|避免色情節目影響青少年身心發展 電研會 赴新聞局遞交建議文}}|work=《民生報》|publisher=聯合報系|location=臺北市|language=zh-tw|date=1995-12-24|accessdate=2017-04-23}}</ref>

=== 政府改革 ===
成淵國中集體性騷擾案凸顯學生輔導體系的不足，[[人本教育基金會|人本教育基金會]]和立法委員[[翁金珠|翁金珠]]、[[周荃|周荃]]曾舉辦「建立校內支援系統，社工人員進入校園－讓成淵國中事件不再重演」公聽會，倡議社會工作人員進駐校園。<ref name="林萬億">{{Cite journal|author=[[林萬億|林萬億]]|title={{單雙書號轉換|校園欺凌與臺灣學校社會工作的發展}}|journal=《社區發展季刊》|issue=135|pages=6|year=2011|language=zh-tw|accessdate=2017-04-23}}</ref>在輿論壓力下，教育部在民國85年（1996年）推出「國民中學試辦設置專業輔導人員計畫」，讓專業輔導人員進駐國中校園，卻被[[行政院|行政院]]定位為「試辦」性質，輔導人員名額也限縮至81人。<ref name="林萬億"/>直到民國103年（2014年），立法院通過《[[s:學生輔導法|學生輔導法]]》，在校園既有的輔導教師之外，增置具備[[心理師|心理師]]或[[社工師|社工師]]執照的「專任輔導人員」一職；成淵國中集體性騷擾案時，輿論要求增置輔導人員的訴求才在法律上落實。<ref>{{Cite journal|author=林郁倫、陳婉真、林耀盛、王鍾和|title={{單雙書號轉換|心理師校園駐區服務的困境、需求與挑戰－由台北市國中輔導人員之觀點}}|journal=《諮商與輔導》|volume=36|issue=1|pages=40-41|publisher=天馬文化|location=臺北市|language=zh-tw|year=2014|accessdate=2017-04-17}}</ref>
