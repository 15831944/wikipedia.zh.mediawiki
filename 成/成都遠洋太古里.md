'''成都遠洋太古里'''（{{lang-en|'''Sino-Ocean Taikoo Li Chengdu'''}}），位於[[中国|中国]][[成都市|成都市]][[锦江区|锦江区]]商业零售核心地段，并与春熙路购物商圈接壤，是一個总楼面面积逾11.4万平方米（逾123万平方英尺）的开放式、低密度的街区形态购物中心，由[[太古地產|太古地產]]联合[[远洋地产|远洋地产]]發展。此項目於2015年4月下旬開幕，有約300間零售及餐飲商舖，及約800個車位。<ref>{{Cite web |url=http://paper.wenweipo.com/2015/04/25/FI1504250021.htm |title=成都遠洋太古里租出九成 |accessdate=2017-06-30 |date=2015-04-25 |publisher=文匯報}}</ref>該項目毗邻千年古刹[[大慈寺|大慈寺]]、六座古建築<ref>{{Cite web |url=http://sichuan.scol.com.cn/amsc/content/2014-10/31/content_9507447.htm |title=成都远洋太古里正式亮相 街区形态购物中心开启购物新体验 |accessdate=2017-06-30 |date=2014-10-31 |publisher=四川在线}}</ref>及都会风尚酒店[[博舍|博舍]]<ref>{{Cite web |url=https://money.udn.com/money/story/6714/1795842 |title=四川成都 舊城見新歡 |accessdate=2017-06-30 |author=項貽斐 |date=2016-06-30 |publisher=聯合財經網 |deadurl=yes |archiveurl=https://web.archive.org/web/20171029173314/https://money.udn.com/money/story/6714/1795842 |archivedate=2017年10月29日 |df= }}</ref>与42间客房的服务式公寓。寫字樓部分與酒店皆為英國[[:en:MAKE_Architects|MAKE Architects]]設計作品，寫字樓命名為[[睿東中心|睿東中心]]。而此寫字樓則於2013年8月7日由發展商宣佈以約人民幣21億元售出。<ref>{{Cite web |url=http://www.swirepacific.com/tc/media/presseach.php?f=p130807.htm |title=成都大慈寺文化商業綜合體辦公樓以人民幣21億元售出 |accessdate=2017-06-30 |date=2013-08-07 |publisher=太古公司}}</ref>整个商业综合体楼面面积约26.9万平方米（逾290万平方英尺）。综合体与[[成都地铁|成都地铁]][[成都地铁2号线|2号线]]及[[成都地铁3号线|3号线]]的换乘站[[春熙路站|春熙路站]]直接连通。<ref>{{Cite web |url=http://www.chinatimes.com/realtimenews/20160727004112-260405 |title=旅途中的「家」——成都10家值得嘗試的特色酒店 |accessdate=2017-06-30 |date=2016年7月27日 |publisher=中時電子報}}</ref>
