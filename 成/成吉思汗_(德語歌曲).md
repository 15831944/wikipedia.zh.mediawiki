《'''成吉思汗'''》（{{lang-de|Dschinghis Khan}}）是[[德国|德国]]（[[西德|西德]]）乐队“[[成吉思汗_(乐队)|成吉思汗]]”专辑“成吉思汗”中的一首歌曲，该曲曾被翻成各种语言版本，粤语版由[[鄭國江|鄭國江]]填詞，[[林子祥|林子祥]]演唱，国语版则由[[张蝶|张蝶]]演唱。

== 德语原版 ==
1979年出版。

== 粤语版 ==
该曲曾被收录在[[华纳|华纳]]、[[百代|百代]]两大唱片公司合作制作的[[林子祥|林子祥]]精选专辑集《Show Off 3CD 超强版》中，位于CD2中的第2曲<ref>{{Cite web |url= http://www.dzwww.com/yule/yinyuefengqing/huangjincd/200404280709.htm |title= 《Show Off》·林子祥 |author=  |date=2004-04-28  |publisher= 大众网 |language= zh-hans |accessdate=2011-6-1导入 |quote=}}</ref>。其明快節奏使其成為[[林子祥|林子祥]]其中一首熱門的歌曲，《成吉思汗》亦是林子祥少數非主要改編自英語歌曲的廣東歌。
