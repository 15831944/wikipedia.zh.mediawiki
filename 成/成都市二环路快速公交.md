'''成都市二环路快速公交'''，是成都市第一条[[成都快速公交|快速公交]]线路，是运行在[[中华人民共和国|中国]][[四川省|四川省]][[成都市|成都市]]二环路高架路的公交专用道的[[巴士快速交通系统|快速公交系统]]，二环路快速公交K1，K1A，K2，K2A自2013年5月31日起开始免费试运行，2013年6月11日起正式运行。<ref name="基本信息"></ref>车站建筑结构分为地面层、站厅层、站台层共三层，每一个站点都有人行天桥可以过街{{NoteTag|光华村街口站除外}}。站台层采用路中侧式布局。<ref>{{cite web|title=新二环 速度舒适度媲美地铁|url=http://www.cdrb.com.cn/html/2012-08/16/content_1661752.htm|publisher=成都日报|date=2012年8月16日|deadurl=yes|archiveurl=https://web.archive.org/web/20160818101957/http://www.cdrb.com.cn/html/2012-08/16/content_1661752.htm|archivedate=2016年8月18日|df=}}</ref>快速公交采用18米铰接式双侧开门{{NoteTag|左侧2个车门、右侧3个，但在正常运营情况下只使用左侧的2个车门。}}公交车。

该系统的车辆均提供中国联通3G [[Wi-Fi|Wi-Fi]]，乘客只需搜索名为“BUS-FREE-WIFI”的热点信号，注册登录后即可免费使用无线网络。<ref>{{cite web|title=二环快速公交免费WiFi全覆盖|url=http://www.cdgjbus.com/Article.aspx?Pid=71&ID=4053|publisher=成都市公共交通集团公司宣传部|date=2014年6月17日|language=zh-cn}}</ref>

== 线路情况 ==
=== K1线 / K1A线 ===
成都快速公交K1线自金沙公交站起始，途径二环路快速公交专用道后回到金沙公交站终止；K1A线自府南新区站起始，途径二环路快速公交专用道后回到府南新区站继续行驶。

=== K2线 / K2A线 ===
成都快速公交K2线自金沙公交站起始，途径二环路快速公交专用道后回到金沙公交站终止；K2A线自光华村街口站起始，途径二环路快速公交专用道后回到光华村街口站继续行驶。

== 乘车须知 ==
# 成都二环路快速公交为全程高架，公交站点分为三层：底层为地面层；第二层为站厅层及过街人行天桥；第三层为站台层。
# 乘车市民经[[电动扶梯|电动扶梯]]（或[[楼梯|楼梯]]步行）至站厅层，按导视牌指引选择乘车方向。
# 在[[驗票閘門#門檔式閘門|闸机]]上刷卡或投币，待扇形门打开后进站，使用[[电动扶梯|电动扶梯]]（或[[楼梯|楼梯]]步行）上站台层排队候车，待车辆进站安全门打开后按两侧上中间下的方式乘车。
# 下车后请迅速走向站台内侧，使用楼梯下至站厅层，待扇形门打开后出站。
# 老、弱、病、残及行动不能自理的人群及学龄前儿童须由成年人陪护进站上车。
# 为了您和他人的安全，请勿携带易燃、易爆等危险品进站。<ref>{{cite book | title=成都市二环路快速公交乘车指南 | year=2013年 | author=成都公共交通集团公司 | authorlink=成都公共交通集团公司 | language=zh-cn }}</ref>

== 接驳线路 ==
[[成都公共交通集团公司|成都公交集团]]为了方便[[成都市|成都市]]社区居民接驳二环快速公交、[[成都地铁|地铁]]及[[成都市公交线路列表#成都市市区公交线路|骨干公交线路]]，解决最后一公里出行不便，在成都快速公交开通后不久还开行了60余条的社区巴士线路，详情请参见[[成都市公交线路列表#社区巴士|成都市公交线路列表]]。

== 安全 ==
=== 安全检查 ===
成都快速公交的安检主要采用抽检的方式。<ref>{{cite web|title=成都二环快速公交安检升级|url=http://www.cdgjbus.com/Article.aspx?Pid=71&ID=3985|publisher=成都市公共交通集团公司宣传部|date=2014年5月16日|language=zh-cn}}</ref>

=== 站台安全装置 ===
[[File:Chengdu_BRT_Platform_Fire_Fighting_Equipment.jpg|thumb]]
每个站点配有两台危险液体检测仪、两根防暴棍和两根防暴叉。<ref>{{cite web|title=二环路快速公交站厅配备新型防暴器械|url=http://www.cdgjbus.com/Article.aspx?Pid=71&ID=3980|publisher=成都市公共交通集团公司场站公司|date=2014年5月13日|language=zh-cn}}</ref>每侧站台还配有消防推车，如遇站点或站间道路上车辆发生火险，站点工作人员能够方便、快速移动推车至二环高架路上最近的消防水栓，对火灾实施扑救。<ref>{{cite web|title=快速公交站厅配置消防推车|url=http://www.cdgjbus.com/Article.aspx?Pid=71&ID=3871|publisher=成都市公共交通集团公司场站公司|date=2014年3月21日|language=zh-cn}}</ref>

== 车票 ==
成都快速公交全线采用单一票价制，乘客单次乘车在车站付费区内可以停留的最长时间为4小时。{{NoteTag|具体执行的最长可停留时间，由公交集团根据快速公交运行线网规模变化适时公布。}}<ref name="Token卡">{{cite web|author=华西都市报|title=成都二环快速公交今免费试乘 IC卡乘客需刷卡出站|url=http://www.sc.xinhuanet.com/content/2013-05/31/c_115978101.htm|publisher=新华网四川频道|date=2013年5月31日|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20130911002238/http://www.sc.xinhuanet.com/content/2013-05/31/c_115978101.htm|archivedate=2013年9月11日}}</ref>

每名乘客可免费携带一名身高不足1.1米的儿童，超出的需按超出人数购票；义务兵和残疾退伍军人可免费乘坐快速公交。

=== 单程票 ===
[[File:Chengdu_BRT_ticket.jpg|thumb]]]]
在2013年6月11日至2018年9月19日，二环路快速公交单程票使用圆形Token卡形式。乘客可通过人工售票处或自动售票机购票，每人每次2元（22:00后加收一元），快速公交采用刷卡入闸、插卡出闸方式，出站时车票将被回收。Token卡购买后若未使用，24小时内仍有效。<ref name="Token卡"></ref>

自2018年9月20日起，单程票改为直接投币进站，出站亦无须刷卡或投币，自动放行。<ref>{{Cite news|url=http://sc.people.com.cn/n2/2018/0918/c379471-32067328.html|title=成都新增两条快速公交　K3、K11本周将开通|author=田程晨|date=2018-09-18|work=成都日报|accessdate=}}</ref>

=== [[天府通|天府通]]卡 ===
普通次数卡、学生次数卡、老年次数卡每次乘车扣除次数2次；电子钱包扣除1.8元。使用天府通次数卡乘坐成都快速公交仍然享受成都公交集团提供的“两小时内免费换乘3次的乘车优惠”。
