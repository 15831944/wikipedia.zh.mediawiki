'''成都石室中学'''，创办于汉景帝后元二年（西元前141年），其前身“'''文翁石室'''”是中国最早的地方官办学堂，<ref>{{cite book | title=成都 | publisher=五洲传播出版社 | author=王少雄, 黄忠莹, 成都市对外文化交流协会 |year=1998 | pages=56 | isbn=9787801134271}}</ref>也是世界上最古老的地方官办学校。<ref>{{cite web | url=http://www.omg-facts.com/History/The-Oldest-School-In-The-World-Founded-B/53370 | title=The oldest school in the world, founded between 143 and 141 BC in China is still in use today! | publisher=OMGFACTS | accessdate=2013-04-28 | deadurl=yes | archiveurl=https://web.archive.org/web/20130503162236/http://www.omg-facts.com/History/The-Oldest-School-In-The-World-Founded-B/53370 | archivedate=2013-05-03 }}</ref>

现在，石室中学是省级一级示范性普通高中<ref>{{cite web|title=成都市39所学校获批省级（一级、二级）示范性普通高中|url=http://sichuan.scol.com.cn/fffy/content/2014-01/02/content_6695415.htm|publisher=四川在线|accessdate=2014-01-05}}</ref>，拥有两个校區，文廟校區位于[[四川省|四川省]][[成都市|成都市]][[青羊区|青羊区]]文庙前街93号，北湖校区位於[[四川省|四川省]][[成都市|成都市]]龙潭寺。

学校校徽是“石室”二字小篆体组成的钟状的标志，校庆日为11月11日。

===古代===
{{参见|文翁}}
{{quote|大城南门曰江桥，桥南曰万里桥，西上曰夷里桥，下曰笮桥，南岸道东有文学，始文翁为蜀守，立讲堂，作石室于南城。永初后，学堂遇火，后守更增二石室。|《水经注·卷三十三》}}
[[汉景帝|汉景帝]]末年（约西元前143年-前141年），当时的蜀郡太守[[文翁|文翁]]，于[[成都|成都]]城南修筑石室，创立了蜀郡郡学。<ref>{{cite book |title=水經注(七)長江之水| publisher=台灣書房出版有限公司 | author=陳橋驛, 酈道元, 葉光庭, 葉揚 | year=2002 | pages=23 | isbn=9570414847|id=ISBN13  9789570414844}}</ref>这所地方官办学堂的开设，为当时国内首创。由于文翁办学有方，成绩卓著，蜀郡的文化教育大为发展，人材辈出。司马相如、杨雄、王褒等词赋家、学者相继出现。所以，西元前124年，[[汉武帝|汉武帝]]下令全国效仿[[文翁|文翁]]兴办学校。<ref name="历史积淀"/><ref>[http://www.cnki.net/KCMS/detail/detail.aspx?QueryID=0&CurRec=7&recid=%20&filename=JYYJ199509015&dbname=CJFD9495&dbcode=CJFQ&pr=%20&urlid=%20&yx=%20&v=MTkzNTMzcVRyV00xRnJDVVJMNmZiK1Z1RnluZ1U3dkxMelRTWkxLeEY5VE1wbzlFWVlSOGVYMUx1eFlTN0RoMVQ=%20 文翁石室的办学特色及其对后世的影响]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }} 徐敦忠,《教育研究》1995(09) </ref><ref>[http://www.cnki.net/kcms/detail/detail.aspx?filename=WSZI200305001&dbcode=CJFQ&dbname=CJFD2003&v=%20 巴蜀文化与石室中学] 查有梁,《文史杂志》2003(05)</ref>

[[州夺郡学|州夺郡学]]事件后，石室被辟为[[益州|益州]]州学。西元994年[[后蜀|后蜀]]开始在石室刊刻石经，[[北宋|北宋]]时继续补刻，直到[[宣和|宣和]]年间才将十三经全部刻完。明末清初，[[张献忠|张献忠]]据蜀，石室俱成瓦砾。1661年当局在文翁石室旧址建立府学。有清三百载，四川学校实际以此为首。<ref name="历史积淀"/>

===近现代===
[[File:石室中学逸夫艺术楼.JPG|thumb]]
[[File:石室中学_文物保护单位牌.JPG|thumb]]

[[康熙|康熙]]年间，四川[[按察使|按察使]][[刘德芳|刘德芳]]在成都府学建锦江书院，后合并演变为[[四川大学|四川大学]]。至1902年，四川总督[[岑春煊|岑春煊]]改成都府学为成都师范学堂，[[癸卯学制|癸卯学制]]施行后又改成都府中学堂。[[辛亥革命|辛亥革命]]后改称成都联合中学。1940年2月，学校更名四川省立成都石室中学，1948年年被评为全国优良示范中学。1950年改称川西石室中学。中华人民共和国成立后的1952年9月，学校再更名为成都第四中学。[[文化大革命|文化大革命]]中，石室中学第三次遭到毁灭性破坏，清代修建的所有建筑均不复存在。1983年4月恢复石室中学旧名。<ref name="历史积淀"/>

2011年，“望子成龙”补习学校申请注册“石室”商标，石室中学提出异议,<ref name="商标0">[http://news.chengdu.cn/content/2013-03/11/content_1175061_2.htm 培训学校申请注册“石室”商标 掀起商标大战]四川新闻网-成都商报[2013-03-11]</ref>2013年4月，石室中学启动“石室”商标申报工作<ref name="商标3">[http://www.wccdaily.com.cn/shtml/hxdsb/20140107/181164.shtml “石室”从此成商标 蹭名校的小心吃官司] {{webarchive|url=https://web.archive.org/web/20140221170119/http://www.wccdaily.com.cn/shtml/hxdsb/20140107/181164.shtml |date=2014-02-21 }}《华西都市报》[2014-01-07]</ref>，2014年初，“石室”被确认为四川省著名商标。“石室”也是四川省第一个由非经营、无利税单位申报并获得认定的著名商标。<ref name="商标1">[http://www.cdshishi.net/its/show.aspx?id=1921&cid=135 四川首例,“石室”获授省著名商标] {{Wayback|url=http://www.cdshishi.net/its/show.aspx?id=1921&cid=135 |date=20140222155608 }}《成都商报》,[2014-01-07]</ref><ref name="商标2">[http://www.cdedu.gov.cn/news/show.aspx?id=41328 “石室”成全省首个公办学校著名商标]《四川日报》[2014-1-7]</ref><ref name="商标3">[http://www.wccdaily.com.cn/shtml/hxdsb/20140107/181164.shtml “石室”从此成商标 蹭名校的小心吃官司] {{webarchive|url=https://web.archive.org/web/20140221170119/http://www.wccdaily.com.cn/shtml/hxdsb/20140107/181164.shtml |date=2014-02-21 }}《华西都市报》[2014-01-07]</ref>

==教育教学==
学校的教育教学质量长期保持着成都市和四川省的一流水平：高考升学率一直保持100%；一大批学生参加全国学科竞赛年年获奖；每年都有一批学生获得保送名牌大学的资格。近年来，共有百多名学生考入北京大学和清华大学，千余名学生进入“全国前十强”著名大学。并且多次考出四川省高考文理科状元。<ref>{{cite web |url=http://www.cdshishi.net/its/ |title=石室概况 |deadurl=yes |archiveurl=https://web.archive.org/web/20150626070856/http://www.cdshishi.net/its/ |archivedate=2015-06-26 }}</ref>

===“三大组织”===
该校“三大组织”指[[共青团|团委]]、[[学生会|学生会]]和社团联合会，其中社团联合会为该校独有，负责组织上一节所述的社团活动<ref>{{cite web |url=http://www.cdshishi.net/wm/show.aspx?id=2675&cid=115 |title=千载石室何者续辉煌，三大风采谁人拭锋芒——我校举行2014-2015学年度三大学生组织招新活动 |access-date=2014-10-31 |archive-url=https://web.archive.org/web/20141031122545/http://www.cdshishi.net/wm/show.aspx?id=2675&cid=115 |archive-date=2014-10-31 |dead-url=yes }}</ref>（北湖校区为“四大”，包括初中的少先队<ref>{{cite web |url=http://www.cdshishi.net/its/trans.aspx?id=2270 |title=我校四大学生组织招新工作顺利完成 |access-date=2014-10-31 |archive-url=https://web.archive.org/web/20141031122805/http://www.cdshishi.net/its/trans.aspx?id=2270 |archive-date=2014-10-31 |dead-url=yes }}</ref>）。成员为学生选出，对校园进行管理并组织运动会、艺术节等活动。<ref>{{cite web |url=http://www.cdshishi.net/wm/show.aspx?id=2737&cid=123 |title=敬业合作，确保运动会圆满举行——记穿梭在会场的裁判老师和三大干事 |access-date=2014-10-31 |archive-url=https://web.archive.org/web/20141031122542/http://www.cdshishi.net/wm/show.aspx?id=2737&cid=123 |archive-date=2014-10-31 |dead-url=yes }}</ref>
==校园活动==
[[File:成都石室中学操场.jpg|thumb]]
* 运动会：石室中学运动会在每年十月国庆后举行，为期2到3天。运动会开幕式文艺表演由每个班学生和老师精心编排，独具特色。<ref>{{cite web |url=http://news.163.com/12/1028/13/8ETJ5UPR00014AEE.html | title=成都中学运动会 师生穿黑丝跳“江南style”(图)}}</ref>
* 艺术节：艺术节在每年五月举行，包括美术作品、书法作品、摄影作品、主持人大赛、器乐专场比赛、舞蹈专场比赛等项目。 <ref>{{cite web |url=http://www.cdshishi.net/wm/show.aspx?id=1472&cid=123 |title=“阳光下成长”——我校举行第25届艺术节开幕式 }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

* “吼楼”：众多学生在石室中学的方形教学楼内的楼道上一齐大声喊叫或拍手的行为。据当事人称，第一次吼楼行为发生在2003年高考前，此行为过后受到了校方的严厉批评。“吼楼”是学生自发组织的活动，多发生在第一节和第二节晚自习之前的休息时间，被学生们视作发泄心中压力的一种方式，也成为了石室中学学生的一大传统。<ref>{{cite web |url=http://cd.qq.com/a/20090608/003014.htm | title=成都考生用“吼楼”庆祝09年高考圆满结束}}</ref>但校方一直明令禁止此行为。<ref>{{cite book | title=石室中学学生手册}}</ref>
