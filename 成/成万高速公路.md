'''成万高速公路'''是连接四川省[[成都市|成都市]]和[[万源市|万源市]]的高速公路，规划总里程457公里，由成彭高速、成绵复线高速、绵万高速组成。
 
== 成都—彭州段 ==
{{main|成彭高速公路}}
'''成彭高速公路'''于2004年10月1日建成通车，全长20.94公里，双向四车道，起于[[成都绕城高速公路|成都绕城高速公路]]大丰立交桥，经[[新都区|新都区]][[新繁镇|新繁镇]]，止于[[彭州市|彭州市]]成青公路绕城段（牡丹大道）。
== 彭州—绵阳段 ==
成都至绵阳段又称'''成绵（复线）高速'''、'''成绵高速复线'''、'''成什绵高速公路'''，技术标准为双向四车道高速公路，设计速度100公里/小时，路线全长约88.3公里，起于[[成彭高速公路|成彭高速公路]]，经[[彭州市|彭州市]]、[[什邡市|什邡市]]、[[绵竹市|绵竹市]]、塔水，止于磨家，接[[成绵高速公路|成绵高速公路]]。

== 绵阳绕城高速南环线段 ==
{{main|绵阳绕城高速公路南环线}}
'''绵阳绕城高速南环线'''全长33.6公里，双向四车道，设计速度为80km/h，于2014年12月29日试通车。
== 绵阳—万源段 ==
{{SC Future infrastructure}}
成万高速公路绵万段将起于绵阳，经[[梓潼县|梓潼]]、[[苍溪县|苍溪]]、[[巴中市|巴中]]、[[通江县|通江]]，止于[[万源市|万源]]。规划双向四车道，设计时速80公里。<ref name="ref">{{cite|title=四川省高速公路网路线命名和编号方案|url=http://imgs.sc.gov.cn/DocAnnex/2017/7/5/8d32966ccb674a3aaa594f32f0e9d191.pdf}}</ref>
