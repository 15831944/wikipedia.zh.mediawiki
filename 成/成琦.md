==生平==
成琦是[[清朝|清朝]][[道光|道光]]、[[咸丰|咸丰]]时人<ref name=manzu/>。《清代人物生卒年表》记其生於[[嘉慶|嘉慶]]二十二年十二月十二日（1818年1月18日）<ref name=ziliao/>。道光三十年（1850年）进士，由[[通政使司副使|通政使司副使]]迁[[詹事府|詹事府]][[詹事|詹事]]，不久改[[户部左侍郎|户部左侍郎]]、[[仓场侍郎|仓场侍郎]]<ref name=manzu/>。1861年，中方代表成琦与俄方代表卡札凯维奇（又译“卡扎克维赤”）签订《[[中俄勘分东界约记|中俄勘分东界约记]]》<ref name=yu>{{Cite web |url = http://jds.cass.cn/ztyj/zwgxs/201605/t20160506_3327254.shtml |title = 吴大澂恢复中国图们江出海权再探讨 |author = 于逢春 |date =  |publisher = 学问2014(6):3-10 |language =  |accessdate =  |quote =  }}</ref>。咸丰十一年（1861年）[[辛酉政变|辛酉政变]]后，以“为[[载垣|载垣]]等之党援”罪名而革职<ref name=manzu/>。
