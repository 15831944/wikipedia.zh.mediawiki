'''成龍'''{{SBSHK}}{{MBE}}，<small>[[联邦直辖区授勋及嘉奖制度|PMW]]</small>（'''Jackie Chan Kong-sang'''，{{bd|1954年|4月7日|catIdx=Chan, Jackie}}，原名'''陳港生'''），本名'''房仕龍'''，前藝名'''元樓'''。國際著名武打演员、导演和電影監製。其生父為[[房道龍|房道龍]]，其妻子[[林鳳嬌|林鳳嬌]]為前[[台灣|台灣]]演員，兩人育有一子[[房祖名|房祖名]]。另有情人[[吳綺莉|吳綺莉]]，私生女[[吳卓林|吳卓林]]。<ref>{{Cite web |url = http://ent.ltn.com.tw/news/breakingnews/1268347 |title = 房祖名見吳卓林泛淚 「好可愛，像我像到死」 |date = 2015/03/26 |publisher = 自由時報 |language = zh-hant }}</ref>

成龍曾經長時間是[[橙天嘉禾|嘉禾電影]]有限公司的旗下藝人，於1979年加盟，填補突然逝世的[[李小龍|李小龍]]，期間他拍攝多部電影屢創佳績。直至1998年，[[何冠昌|何冠昌]]突然逝世，成龍暫時離開嘉禾。同年年底，他經嘉禾集團創辦人[[鄒文懷|鄒文懷]]的邀請之下，正式回巢[[嘉禾|嘉禾]]，拍攝[[玻璃樽_(電影)|玻璃樽]]和[[特務迷城|特務迷城]]。到2001年初，成龍被[[英皇集團|英皇集團]]挖角。最後，他在同年正式離開嘉禾，加盟[[英皇集團|英皇集團]]附屬的[[英皇娛樂|英皇娛樂]]和[[英皇電影|英皇電影]]，一直為此公司工作至今。成龙还担任中华人民共和国第十二届[[全国政协委员|全国政协委员]]，中華人民共和国禁毒宣传形象大使。2016年获得奥斯卡终身成就奖，为首次获此殊荣的华人。

== 生平 ==
[[File:Jackie_Chan_star.JPG|thumb]]上手印]]
成龍本名'''陳港生'''，後改名為'''房仕龍'''。其父[[房道龍|房道龍]]為[[安徽省|安徽省]][[和县|和县]][[沈巷镇|沈巷镇]]（今属[[芜湖市|芜湖市]]）人，曾在安徽結婚，生有二子房仕德、房仕勝。1947年其父獨自至香港發展，從母姓（即成龍的祖母），改名「陳志平」，在香港另娶陳月榮，生下成龍。在小時候其父向成龍說其本籍安徽。2001年，由其父親房道龍處知道其祖籍為安徽[[和县|和县]]，其本姓房。2003年，其子陳祖明回歸本姓，改名房祖名。2013年8月31日，成龍回到安徽蕪湖沈巷镇，認祖歸宗，改名'''房仕龍'''<ref>{{cite news|title=成龍回蕪湖認祖歸宗 原名房仕龍會同父異母哥哥 |url=http://news.xinhuanet.com/gangao/2013-09/03/c_125305306.htm|newspaper=京華時報|date=2013-09-03}}</ref>。

成龍在中國節目自爆父母身世，表示父親房道龍當年在海關當警察，遇上偷運鴉片的母親[[陳月榮|陳月榮]]，不過其父當時見母親新寡，背上一個女兒，手裏又拖一個女兒，一時心軟放她過關，兩人亦由此相識。

成龙沒有受過正統教育，在六歲時認[[于占元|于占元]]為義父，寄居於于占元位於[[九龍|九龍]][[尖沙咀|尖沙咀]][[美麗都大廈|美麗都大廈]]的[[中国戏剧学院|中国戏剧学院]]，學習[[京劇|京劇]]與[[功夫|功夫]]，組成[[七小福_(演員)|七小福]]戲班演出，藝名'''元樓'''，為[[元家班|元家班]]之一。由於京劇式微，于占元為了徒弟的前途，鼓勵徒弟參與電影演出。

1976年，拍攝《[[新精武门_(1976年电影)|新精武門]]》時，因[[李小龍|李小龍]]風潮，改藝名為成龍。

=== 七小福时期 ===
[[七小福_(演員)|七小福]]是一齣京劇，也是一個戲班，更是[[梨園|梨園]][[名伶|名伶]]于占元在1960年代在香港開辦“中國戲劇研究學院”時所收的得意門生總稱。于占元在一次京劇表演中，選了[[元龍|元龍]]（洪金寶）、元樓(成龍)、[[元彪|元彪]]、[[元奎|元奎]]、[[元華|元華]]、元武和元泰7人擔任七小福的主角。後來因為演出非常成功，于占元於是藉此組成七小福戲班。七小福實際人員並不固定，也不只7人，于占元師傅根據不同的演出安排组合，多的時候達三四组七小福同時在[[荔園|荔園]]等不同的地方演出。

七小福戲班成員共有十多人，除去剛才提到的7人外，尚有[[元德|元德]]、[[元俊|元俊]]、[[元振|元振]]、[[元寶|元寶]]及女學員[[元秋|元秋]]等，他們後來都進入香港電影界，對功夫片貢獻尤大。據統計，香港電影金像獎成立至今一共發出22個最佳武術指導獎盃，其中12個落在了七小福手上。臺灣電影金馬獎1990年代以後開始設最佳動作指導獎，迄今有13屆，七小福得了7次。

=== 崛起 ===
1976年，成龍在澳洲期間，經過經理人[[陳自強|陳自強]]的關係，被當時的電影人[[羅維|羅維]]看中，隨即返港拍攝《[[新精武门_(1976年电影)|新精武門]]》，正式取名成龍。

直到1978年[[吳思遠|吳思遠]]開拍由[[袁和平|袁和平]]執導的《[[蛇形刁手|蛇形刁手]]》，向羅維借用成龍出任主角，同年拍攝的《[[醉拳|醉拳]]》，大破各地票房紀錄，取得空前的成功。隨後更自導自演拍了多部包括《笑拳怪招》、《師弟出馬》等動作電影。這時羅維的幕後支持者[[嘉禾電影|嘉禾電影]]的[[鄒文懷|鄒文懷]]安排成龍到[[好萊塢|好萊塢]]拍攝《殺手壕》、《炮彈飛車》、《威龍猛探》一系列的低成本西片，未有作為。

1981年回歸香港拍攝《龍少爺》重振聲威，接著同師兄[[洪金寶|洪金寶]]拍攝一系列「福星片」。1985年成龍更憑著《[[警察故事|警察故事]]》獲[[香港電影金像獎|香港電影金像獎]]最佳故事片獎。1992年以《[[警察故事3|警察故事3]]》榮獲台灣[[金馬獎|金馬獎]]最佳男主角，1993年再度憑《[[重案組_(電影)|重案組]]》連莊金馬獎影帝。

=== 走向國際 ===
成龍早在1980年時便開始打入好莱坞市場，但他邁向國際之路並不順遂；他首次進軍國際的作品是《[[殺手壕|殺手壕]]》，票房失利，令成龍要相隔多年才再闖好莱坞。

真正令成龍打入國際市場是1994年拍攝的《[[紅番區|紅番區]]》，在[[美國|美國]]上映時創下高票房紀錄，進而接下[[好萊塢|好萊塢]]電影《[[尖峰時刻|尖峰時刻]]》，亦獲得極高的票房，登上《時代》雜誌。但是接下來的好萊塢影片如《[[环游世界八十天_(2004年电影)|環遊世界八十天]]》，在全球票房普遍不佳。

成龍曾經表示，其實好萊塢並不是他的天下，只有回到香港才是如魚得水的<ref>[http://news3.xinhuanet.com/ent/2004-09/03/content_1940392.htm 〈成龍心聲：好萊塢不是我的天下〉] {{webarchive|url=https://web.archive.org/web/20071109170539/http://news3.xinhuanet.com/ent/2004-09/03/content_1940392.htm |date=2007-11-09 }}新華網 2004年9月3日</ref>。

== 政治立场 ==
1986年，[[英國君主|英女皇]][[伊利沙伯二世|伊利沙伯二世]]訪問香港，成龍第一個登臺表演，{{Fact|表達對女王的敬愛和對英國的忠誠|time=2019-02-28T17:14:13+00:00}}。因此{{or}}，英國授予成龍勳章。但是{{or}}，成龍到中國發展之後卻{{or}}公開表示：「我們生在香港，不中不西。拿着[[英國護照|英國護照]]，但是去英國要蓋章，要拿visa。到那邊還要問你來英國幹什麽。英國不是我的國家嗎？不是。你是一個被遺棄的小孩子，我擁有你，但是我又不管你。」<ref>[http://www.nfpeople.com/News-detail-item-4019.html 南方人物周刊：很多恩愛無比的離婚了 我們反而過得更好——對話成龍] {{webarchive|url=https://web.archive.org/web/20130110060412/http://www.nfpeople.com/News-detail-item-4019.html |date=2013-01-10 }}</ref><ref>資料來源：中國內地《[[南方人物周刊|南方人物周刊]]》<[http://hk.news.yahoo.com/成龍-應規定什麼不能遊行-撐特首-不信每個都要害香港-210657777.html]</ref>他更不讳言提及自己年轻时拍戏时面对黑社会的“疯狂”事迹。他声言当时不少艺人被黑社会欺负，连[[刘德华|刘德华]]也被枪指着要求拍戏，故成龙自己当时亦每天携枪出外，甚至用钱聘用“穿草鞋的”对付[[黑社会|黑社会]]{{clarify|reason=非政治立场，与章节标题无关}}。

成龙2004年3月28日在上海演讲时，脱稿批评[[2004年中華民國總統選舉|台湾选举]]所发生的[[三一九槍擊事件|三一九槍擊事件]]是一个“天大的笑话”，并认为“不管统一不统一，最要紧是要和平相处，整个地方，不管是香港、台灣，如果我们统一之后我们一定会是世界最强的一个国家。”。

事隔两年多，他在2006年9月19日出席香港电影《[[宝贝计划|宝贝计划]]》的首映会上，被媒体问到对于台湾倒扁活动的看法，说：“所以我讲的话是对的，两年前我讲‘天大的笑话’，两年后今天已经是……，国际性的笑话，不是，宇宙的笑话，我看了觉得很可怜，两边也觉得很可怜。”他说：「不会收回两年前曾说过“天大的笑话”的发言，因为台湾政局越来越混乱」。<ref>[http://taiwanheart.ning.com/group/heart10/forum/topics/cheng-long-bu-huan-ying-ni 成龍：「台灣總統選舉是個天大的笑話！」] {{webarchive|url=https://web.archive.org/web/20140822121214/http://taiwanheart.ning.com/group/heart10/forum/topics/cheng-long-bu-huan-ying-ni |date=2014-08-22 }}</ref><ref>[http://appledaily.atnext.com/template/apple/art_main.php?iss_id=20090419&sec_id=4104&subsec_id=11866&art_id=12656399 成龍：「中國人是需要管的」] {{webarchive|url=https://web.archive.org/web/20140819083939/http://appledaily.atnext.com/template/apple/art_main.php?iss_id=20090419&sec_id=4104&subsec_id=11866&art_id=12656399 |date=2014-08-19 }}</ref><ref>[http://www.epochtimes.com/b5/9/4/21/n2501697.htm 隨鄉：成龍侮辱中國人] [[大紀元|大紀元]]</ref>

成龙在2007年公開表示「中國人是需要管的」。2009年他于海南博鳌亚洲论坛又公开表示：「香港和台湾因太自由而很乱，所以中国人一定是需要管的，否则便会为所欲为」。2012年他接受《[[南方人物周刊|南方人物周刊]]》訪問时，就“中国人是否需要文化自由”的问题再次重申「中國人一定要管」。<ref>[http://you.joy.cn/video/430914.htm 成龍：「香港和台湾因太自由而很乱，所以中国人一定是需要管的，否则便会为所欲为」] {{webarchive|url=https://web.archive.org/web/20140316135249/http://you.joy.cn/video/430914.htm |date=2014-03-16 }}</ref>

2012年12月11日，成龍接受中國《[[南方人物周刊|南方人物周刊]]》訪問時表示，以往香港受[[英国|英国]]管治镇压，令港人喜欢自由，却把香港变成现在的「遊行之都」，故應該規定「什麼可以遊行、什麼不能遊行」。成龍並出言捍衛特首[[梁振英|梁振英]]，指「每一個上台都被罵，我就不相信每個特首上來都是要害香港的」。並认为，香港已回归中国，不应经常责骂中国领导人，「你们谁有本事谁来管」。称不认同特首上台市民就批评的行為。

== 公開語錄 ==
{{wikiquote}}
2014年10月9日，香港[[雨傘革命|占中活动]]持续至第12天，成龙在[[微博|微博]]写道：“看新闻得知这次香港的经济损失高达3500亿，真的很着急。我相信每一个香港人都爱香港，都希望香港好！香港美好的明天需要所有人的支持与努力，《国家》的歌词里说“没有强的国，哪有富的家”，我愿意与大家一起努力，回归理性，面对未来，爱我们的国家，爱我们的香港。”<ref >{{cite web|url=http://gd.people.com.cn/n/2014/1011/c123932-22569380.html | language= | title=成龙谈“占中”：回归理性 爱国家爱香港 |publisher=人民网-港澳频道 |date=2014年10月11日 |accessdate= }}</ref><ref >{{cite web|url=http://ent.ifeng.com/a/20141009/40325648_0.shtml | language= | title=成龙发声谈香港“占中”事件：没有强的国 哪有富的家 |publisher=凤凰网 |date=2014年10月9日 |accessdate= }}</ref>

論香港遊行文化：「現在香港變成『遊行之都』，全世界以前是在講[[韓國|韓國]]，現在是香港。應該規定什麽可以[[遊行|遊行]]、什麽不能遊行。現在很多無理取鬧的遊行都來了，有些人沒事就遊行、有假期就遊行。我就每天都看見遊行。我們今天回歸中國，怎麽可以老罵中國的領導人？你們誰有本事誰來管，你們管不了，只是在罵。」

中國人要被管、只买日本的，不買國產貨：2009年[[博鰲論壇|博鰲論壇]]，成龍在分組會議被問對文化自由的看法時表示如果太自由，就會像香港那样“为所欲为”、台湾那样“太乱了”，所以“中國人還是需要被管的”，尤其是在中国大陆。並公开表示「我只买日本的，一定不买中国的，因为国产电视机会爆炸」。<ref>[http://you.joy.cn/video/430914.htm 要买就买日本的 因为国产电视机会爆炸] {{webarchive|url=https://web.archive.org/web/20140316135249/http://you.joy.cn/video/430914.htm |date=2014-03-16 }}</ref>

論[[菲律賓人質事件|菲律賓人質事件]]：在[[社交網站|社交網站]]發表的言論说“香港是多元种族的社会，别担心，我们没有憎恨”，又为菲警方营救人质解释，“如果警方一开始就击毙挟持者，人们会说，怎么不先谈判？但如果警方先谈判，人们又会问，怎么不早些杀了挟持者？令人难过。”菲律宾总统府旋即回应，称希望香港民众也能像成龙一样，向他们伸出友谊之手，事后成龙被媒体批评为“成聋”及“没资格代表香港人”<ref>[http://news.yule.com.cn/html/201009/79265.html 成龙：我们没有憎恨]，2010年08月27日</ref>。

批評美國是全世界最大的貪污國：2012年12月12日，成龍日在[[鳳凰衛視|鳳凰衛視]]《[[鏘鏘三人行|鏘鏘三人行]]》節目中批評[[美國|美國]]是世界最大的[[貪污|貪污]]國家，搞得世界最大的崩潰，並表示「現在我們中國強盛，所有人都拿我們中國了做問題，我們自己人都不支持自己的國家，誰支持自己的國家？我們知道我們自己國家很多問題，我們關起來講，對外我們國家是最好的。」<ref>[http://v.ifeng.com/news/society/201212/a362685e-ba17-4630-a1e9-bf7b6924aada.shtml 2012年12月12日鳳凰衛視《鏘鏘三人行》]</ref><ref>[https://www.youtube.com/watch?v=TEEmx7M7VTs 大砲成龍批美國 「全球最大貪汙國」]</ref>

想見到海嘯、地震的發生：2013年10月，成龍在[[美國|美國]][[加州|加州]]為電影《[[十二生肖_(電影)|十二生肖]]》宣傳時接受[[美聯社|美聯社]]訪問，表示「有時我真想見到有些國家遭遇災難，好像是大[[海嘯|海嘯]]、好像是大[[地震|地震]]，在發生大災難時，你會見到地球上每個人都飛到那國家幫忙。沒有海嘯、沒有地震，沒事可做，就會有政治問題！」此言論再度引來撻伐聲浪。<ref>[http://www.nownews.com/2013/10/24/340-2999259.htm#ixzz2ie2RQvVl 成龍失言想見海嘯地震　瞎扯災難論籲團結引撻伐]</ref><ref>[http://hk.apple.nextmedia.com/entertainment/first/20131024/18476715 嘉玲斥唔應該希望有災難 成龍又亂噏：想有海嘯地震 ]</ref>

有钱人需要抽空[[坐牢|坐牢]]：2015年8月，看到房祖名坐牢反而令父子关系有所增进，成龙接受[[美联社|美联社]]采访时表示“太多有钱人，好像我的一些朋友，他们一年抽空10天或1个月，到深山修心养性。”“我认为我有时需要抽时间放假坐牢，而且要强迫有些有钱人，甚至我自己去坐牢。现在这个时代，人人都需要这样做，因为大家都太忙了。”<ref >{{cite web|url=http://news.163.com/15/0806/00/B09VDK8K00014AEE.html | language= | title=成龙:富人坐牢不是坏事 自己也要抽空坐牢 |publisher=参考消息网(北京) |date=2015-08-06 |accessdate= }}</ref>
