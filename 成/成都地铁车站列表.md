'''[[成都地铁|成都地铁]]'''是服务于[[中华人民共和国|中国]][[四川省|四川省]][[成都市|成都市]]的[[地铁|地铁]]系统，为成都市[[城市轨道交通|市域铁路系统]]网络的组成部分，于2010年9月27日开始运营<ref>{{cite web| url=http://news.xinhuanet.com/society/2010-09/27/c_13531938.htm| title = 成都开通我国西部地区首条地铁 | publisher= 新华网 | date= 2010 年 9 月 27 日}}</ref>，是[[中国大陆|中国]]中西部地区投入运营的第一个地下城市轨道交通系统。截至2018年12月，共有6条线路投入运营，171座<ref group="注">换乘站重复计算。如果不重复计算换乘站，则为156座车站。</ref>运营车站，运营里程达227千米。

以下按线路列出[[成都地铁|成都地铁]]已投入运营的车站。

== 1号线 ==
{{main|成都地铁1号线}}
[[File:Luomashi_Station_of_Chengdu_Metro.jpg|thumb]]站台]]
成都地铁1号线全长40.99 km，共35座车站：
1号线一期全长18.5 km，是成都米字型骨干网络中的北-南向线路，共17座车站，于2007年12月29日开工，2010年9月27日开放试运营，是中西部地区第一条地铁。该线路北起[[成华区|成华区]][[升仙湖站|升仙湖]]，途径[[金牛区|金牛区]]、[[青羊区|青羊区]]、[[锦江区|锦江区]]、[[武侯区|武侯区]]，南至[[世纪城站|世纪城]]。[[锦城广场站|锦城广场站]]在2013年6月8日开通试运行。
1号线二期全长5.42 km，设5座车站，于2015年7月25日开通运营，北起1号线一期南段[[世纪城站|世纪城站]]，南至[[双流区|双流区]][[广都站|广都站]]，其中[[四河站|四河站]]预留1号线三期分岔运营。
1号线三期南段线路长14.332公里，支线长1.21公里，北段长1.465公里，于2018年3月18日开通运营，北起1号线二期南段[[四河站|四河站]]，南至[[天府新区|天府新区]][[科学城站|科学城站]]。支线北起[[四河站|四河站]]，南至[[五根松站|五根松站]]。北段增设[[韦家碾站|韦家碾站]]。

== 2号线 ==
{{main|成都地铁2号线}}
[[File:成都地铁2号线人民公园站台.jpg|thumb]]站台]]
成都地铁2号线全长42.32km，是成都市轨道交通米字型骨干网络中的西北-东南向线路，共32座车站：
一期工程于2007年12月29日开工建设，2012年9月16日开放试运营，共20座车站，22.47km。
二期西延线于2013年6月8日开通，共6座车站，8.74km。地铁2号线向西延伸到[[郫都区|郫都区]]犀浦镇。
二期东延线于2014年10月26日开通，共6座车站，11.12km。地铁2号线西起[[郫都区|郫都区]]犀浦镇，途经[[金牛区|金牛区]]、[[青羊区|青羊区]]、[[锦江区|锦江区]]、[[成华区|成华区]]、[[龙泉驿区|龙泉驿区]]，东至[[龙泉驿区|龙泉驿区]]龙泉驿音乐广场。

== 3号线 ==
{{main|成都地铁3号线}}
[[File:Platform_of_Line_3_in_Sichuan_Gymnasium_Station.JPG|thumb]]站台]]
3号线一期工程全长20.35公里，是成都市轨道交通米字型骨干网络中的东北-西南向线路。
起于[[军区总医院站|军区总医院站]]，途径[[金牛区|金牛区]]、[[成华区|成华区]]、[[锦江区|锦江区]]、[[武侯区|武侯区]]、止于[[太平园站|太平园站]]，设17座车站，沿途串联[[成都大熊猫繁育研究基地|成都大熊猫繁育研究基地]]、成都动物园、[[春熙路|春熙路]]商圈、成都[[武侯祠|武侯祠]]等热门区域。
于2012年4月28日开工建设；2016年7月31日开通试运营。成都地铁3号线二期工程（南延线）连接双流，起于[[成贵客专|成贵客专]][[双流西站站|双流西站]]，沿藏卫路、商都路、双楠大道、武侯大道、三环路敷设至佳灵路接一期工程起点，线路全长约17.2km，全为地下线，设车站11座,于2015年底开工。3号线三期于2015年底开工，线路基本呈南北走向，主要沿川陕公路、大件路、蓉都大道及新都大道走行。线路全长12.5km，其中高架及过渡段长8.37km，地下段长4.112km。共设置9座车站，其中高架站3座，地下站6座。成都地铁3号线二三期于2018年12月26日开通试运营。

== 4号线 ==
{{main|成都地铁4号线}}
[[File:Platform_of_Yushuang_Road_Station2.JPG|thumb]]站台]]

== 7号线 ==
{{main|成都地铁7号线}}
[[File:Platform_of_Shizishan_Station01.jpg|thumb]]站台]]
成都地铁7号线为串联成都市轨道交通米字型骨干网络的[[环线|环线]]。位于二、三环之间居住用地最密集地带，并串联了国铁[[成都站|成都站]]、[[成都东站|成都东站]]、[[成都南站|成都南站]]三大交通枢纽，并与多条轨道交通放射线路相交。
成都地铁7号全长38.61 km，设车站31座，规划远期换乘站19座。7号线为成都市首个获批采用A型车6辆编组制式的轨道交通项目，建设工期为4年2个月，2017年12月6日开通试运营。<ref>{{cite news|title=刚刚！成都地铁7号线开通了！！|url=http://www.thepaper.cn/newsDetail_forward_1893673|date=2017-12-06|accessdate=2017-12-06}}</ref>

== 10号线 ==
{{main|成都地铁10号线}}
[[File:Platform_of_Line_10_in_Taipingyuan_Station.jpg|thumb]]站台]]
成都地铁10号线一期全长10.9km，采用地下线的敷设方式，设地下车站6座。从中心城区3号线太平园站始发，连接双流机场，一期工程以机场线功能为主，利用地铁网的集散功能，方便机场客流通过10号线快速进出中心城。
成都地铁10号线一期于2014年5月12日在省发改委获得正式建设批示，2016年9月7日实现洞通，2017年9月6日开通试运营。
