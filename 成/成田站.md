'''成田站'''（{{jpn|j=成田駅|hg=なりたえき|rm=Narita eki|f=y}}）是位於[[日本|日本]][[千葉縣|千葉縣]][[成田市|成田市]]花崎町，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[成田線|成田線]]的[[鐵路車站|鐵路車站]]。車站編號'''JO 35'''。

成田線[[佐倉站|佐倉站]]起往{{link-ja|佐原站|佐原駅}}、[[銚子站|銚子站]]方向的本線、往[[成田機場站|成田機場站]]的支線以及往[[我孫子站_(千葉縣)|我孫子站]]方向的支線在此站分岔。此站軌道往4個方向延長，但是全部是同一路線，是相當罕見的狀態。

== 歷史 ==
* 1897年（[[明治|明治]]30年）1月19日 - [[成田鐵道|成田鐵道]]（初代）的車站開業。旅客、貨物取扱。
* 1897年12月29日 - 成田鐵道當站～[[滑河站|滑河]]間延伸開業。
* 1901年（明治34年）2月2日 - 成田鐵道當站～[[安食站|安食]]間延伸開業。
* 1911年（明治44年）7月1日 - 千葉縣營鐵道[[成田鐵道多古線|多古線]]開業。
* 1920年（[[大正|大正]]9年）9月1日 - 成田鐵道（初代）被[[鐵道敷設法|買收]]，[[日本國有鐵道|國有鐵道]]之車站。
* 1927年（昭和2年）4月1日 - 千葉縣營鐵道多古線[[千葉交通|成田鐵道（2代）]]讓渡。
* 1944年（昭和19年）1月11日 - 成田鐵道多古線因[[不要不急線|不要不急線]]而休止。
* 1946年（昭和21年）10月9日 - 成田鐵道多古線[[廢線|廢線]]。
* 1977年（昭和52年）7月 - 站舍建替工事開始。
* 1979年（昭和54年）4月 - 站舍建替工事完成，現在為橋上站舍。
* 1983年（昭和58年）8月9日 - 貨物扱廢止。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]，為[[東日本旅客鐵道|JR東日本]]之車站。
* 1991年（[[平成|平成]]3年）3月19日 - 成田線支線的當站～[[成田機場站|成田機場]]間開業。

== 車站結構 ==
[[側式月台|側式月台]]1面1線與[[島式月台|島式月台]]2面4線，合計3面5線的[[地面車站|地面車站]]。4號月台是沒有月台的通過線（3、5號月台之間）。車站構內設有若干留置線，是夜間等的車輛留置地，同時也是保線用設施。
