'''成知勳'''（{{lang-ko|성지훈}}，{{Bd|1962年|10月26日|||idx=成}}），是韩流乐坛的[[音訊工程師|音讯工程师]]，LIVE[[工程师|工程师]]和[[词曲作者|词曲作者]]。
[[Category:音訊工程|Category:音訊工程]] 
[[Category:韓國男歌手|Category:韓國男歌手]]

== 生涯 ==
1980年，他为[[作曲家|作曲家]]出道，以后移民到[[美国|美国]]。在[[美国|美国]][[新泽西州|新泽西州]]生活，往来[[韩国|韩国]]活跃。2009年，放弃[[美国|美国]]公民权以后入[[韩国|韩国]]籍。

== 经历 ==
从1980年代初，作为[[词曲作者|词曲作者]]和[[音樂製作人|音乐制作人]]活跃，兼[[录音|录音]]和[[混音|混音]]工程师。以后活跃为PA工程师。

现在他是JFS Mastering的总工程师。1990年[[申海澈|申海澈]]的个人出道专辑，[https://en.m.wikipedia.org/wiki/015B 015B]出道专辑，1992年[[尹鍾信|尹钟信]]出道专辑，1993年[[嚴正化|严正花]]出道专辑等，由他担任许多受欢迎的歌手的出道专辑的[[音樂製作人|音乐制作人]]。

1994年，他担任三人男歌手组合[https://en.m.wikipedia.org/wiki/Solid_(band) Solid]的[[音樂製作人|音乐制作人]]，正式开始作为专门[[音訊工程師|音讯工程师]]。

1992年，发布自己个人专辑《在傍晚去旅行》。

2009年8月，他举办JFS Mastering以后, 为[[PSY|PSY]]、[[Rain|Rain]]、[[Red_Velvet|Red Velvet]]、[[Girl's_Day|Girl's Day]]和[[朴孝信|朴孝信]]以及许多位歌星做母带处理的工作，担任去世的[[申海澈|申海澈]]的遗作歌曲的[[音乐制作|音乐制作]]、[[編曲|编曲]]、[[混音|混音]]和母带处理<ref>{{Cite web|title=김장훈-故 이보미양 ‘거위의 꿈’ 듀엣 영상 화제|url=http://www.fnnews.com/news/201407261716316452|work=www.fnnews.com|date=2014-07-26|accessdate=2019-02-13|first=파이낸셜뉴스 입력: 2014 07 26 17:16|last=수정: 2014.10.24 21:44}}</ref><ref>{{Cite web|title=[SS뮤직]고(故)신해철 1주기,세상에 '희망의 메시지' 전한다...신곡 '웰컴 투더 리얼 월드' 내용은?|url=http://www.sportsseoul.com/news/read/306480|work=스포츠서울|date=2015-10-19|accessdate=2019-02-13}}</ref>。

== 所属 ==
现在他是JFS Mastering的总经理。

JFS Mastering Studio在[[首爾|首尔市]][[江南_(首爾)|江南区]]论岘洞自己名下一栋大楼，他在2009年8月29日开业了。
