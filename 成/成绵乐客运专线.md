'''成绵乐客运专线'''，又称'''成绵乐城际铁路'''，是[[中国|中国]]一条连接[[四川省|四川省]][[绵阳市|绵阳市]]、[[德阳市|德阳市]]、[[成都市|成都市]]、[[眉山市|眉山市]]、[[乐山市|乐山市]]的[[城际铁路|城际]][[客运专线|客运专线]]，为[[中国西南地区|中国西南地区]]第一条城际客运专线，于2014年12月20日正式通车。<ref name=background>{{cite news|title=成绵乐客专效应　市长来成都开会也坐高铁|url=http://sc.people.com.cn/n/2015/0129/c345458-23722242.html|accessdate=2015-01-31|newspaper=人民四川|date=2015年1月29日}}</ref>成绵乐城际铁路只是工程用名，根据2013年11月22日成都铁路局的会议纪要，成绵乐铁路[[江油站|江油]]至[[成都东站|成都东]]的线路命名为[[西成客运专线|西成客运专线]]（江成段），成绵乐铁路成都东至[[乐山站|乐山]]的线路命名为[[成贵客运专线|成贵客运专线]]（成乐段），成绵乐铁路乐山至[[峨眉山|峨眉山]]的线路命名为成贵客专峨眉山支线。<ref>[http://c.blog.sina.com.cn/profile.php?blogid=64e140b289000b8j 关于明确成绵乐铁路线名、站名和运营里程的会议纪要 ]</ref>

== 概述 ==
成绵乐客运专线正线长318.4公里，客运专线，双线电气化，设计速度300km/h，线路实际运营速度250km/h，设计年单向旅客输送能力8000万人次。

2007年4月7日，由[[中华人民共和国铁道部|中华人民共和国铁道部]]和[[四川兴蜀铁路有限责任公司|四川兴蜀铁路有限责任公司]]投资组建的成绵峨城际铁路有限责任公司正式成立，负责成绵乐客运专线的建设，其中四川兴蜀铁路公司由成都、绵阳、德阳、乐山、眉山五市政府共同投资组建。<ref>[http://www.scol.com.cn/NSICHUAN/bsxw/20070524/2007524145232.htm 城际列车搅动成德绵财富版图]</ref>

2008年12月30日全线正式开工，工程投资估算总额为392亿元，建设工期4年。<ref name="xlw">[http://news.sina.com.cn/c/p/2008-12-30/090016947685.shtml 四川成绵乐铁路客运专线正式开建(组图)] 新浪网</ref>全线新建桥梁116座，总长度151.85公里；隧道4座，总长度12.969公里。全线桥梁隧道比例为52.1％。

2011年，环保部批复修改方案，在峨眉站之后新增[[峨眉山站|峨眉山站]]，该站距峨眉天下名山景区约2公里。

2012年12月2日上午全线开始铺轨，2014年9月10日开始联调联试<ref>[http://news.ifeng.com/a/20140912/41965094_0.shtml 成绵乐城际快铁试验列车开始试跑] 凤凰网</ref>。2014年12月20日全线通车<ref>[http://www.sc.chinanews.com/news/2014/1220/0925278507.html 成绵乐城际高速铁路客运专线今日通车] {{webarchive|url=https://web.archive.org/web/20141220111317/http://www.sc.chinanews.com/news/2014/1220/0925278507.html |date=2014-12-20 }} 中新网</ref>，但[[旌阳站|旌阳]]、[[双流西站|双流西]]、[[新津南站|新津南]]、[[峨眉站|峨眉]]（城际场）未同期投用。

== 车站 ==
全线设车站21座，其中既有车站4个、高架车站3个、地下车站1个。线路在成都东站可与[[成都地铁2号线|成都地铁2号线]]、[[成都地铁7号线|成都地铁7号线]]、在成都南站可与[[成都地铁1号线|成都地铁1号线]]、[[成都地铁7号线|成都地铁7号线]]、在双流机场站可与[[成都地铁10号线|成都地铁10号线]]实现无缝换乘。

线路北起绵阳市境内既有[[宝成铁路|宝成铁路]][[江油站|江油站]]，经[[青莲站|青莲]]、[[绵阳站|绵阳]]、[[罗江东站|罗江东]]、[[德阳站|德阳]]、[[旌阳站|旌阳]]<ref>[http://e.chengdu.cn/html/2014-12/21/content_502749.htm 成绵乐“公交化”运行何时能实现？] {{webarchive|url=https://web.archive.org/web/20141222212820/http://e.chengdu.cn/html/2014-12/21/content_502749.htm |date=2014-12-22 }} 成都商报 2014-12-21</ref>、[[广汉北站|广汉北]]、[[青白江东站|青白江东]]、[[新都东站|新都东]]、[[成都东站|成都东]]、[[成都南站|成都南]]、[[双流机场站|双流机场]]、[[双流西站|双流西]]、[[新津站|新津]]、[[新津南站|新津南]]、[[彭山北站|彭山北]]、[[眉山东站|眉山东]]、[[青神站|青神]]、[[乐山站|乐山]]、[[峨眉站|峨眉]]，南至[[峨眉山市|峨眉山市]]的新建[[峨眉山站|峨眉山站]]。

== 运营 ==
[[中国铁路成都局集团有限公司|中国铁路成都局集团有限公司]]

成绵乐客运专线采用国铁制式运营，开通时运营速度200km/h，2017年12月[[西成客运专线|西成客运专线]]通车后提速至250km/h，凭身份证[[实名制|实名制]]购票上车，其中车票与座位一一对应。另外，成绵乐客运专线亦支持[[中铁银通卡|中铁银通卡]]乘车与刷身份证乘车。

成绵乐客运专线本线列车为C（[[城际铁路_(消歧义)|城际]]）编号，其中[[江油站|江油]]与[[峨眉山站|峨眉山]]间运行的全程列车编为“C63xx”，如C6301次列车；在[[成都东站|成都东站]]与南北两端车站间运行的列车编号则为“C62xx”。

成绵乐客运专线初期运用[[和谐号CRH1型电力动车组|CRH1A]]型动车组运营，后期主要运用车辆改为[[和谐号CRH380A型电力动车组|CRH380A]]动车组与[[和谐号CRH380D型电力动车组|CRH380D]]动车组，西成高速铁路开通后，成绵乐本线C字头列车大量改用[[和谐号CRH3A型电力动车组|CRH3A动车组]]开行，跨线运营的“[[复兴号电力动车组|复兴号]]”[[CR400AF|CR400AF]]、[[CR400BF|CR400BF]]也首次出现在本线上。2018年下半年，[[复兴号CR300AF型电力动车组|CR300AF]]、[[复兴号CR300BF型电力动车组|CR300BF]]“复兴号”动车组前往成绵乐南段进行试验。在高峰时段，则会采取两列车重联的方式编为16节编组运营。
