'''成珠樓'''，始於1746年（乾隆十一年），建於[[廣州市|廣州市]][[海珠區|海珠區]]南華中路{{link-yue|漱珠桥|漱珠橋}}畔，是廣州現存可考最古老的[[茶樓|茶樓]]<ref name = "古老茶樓">{{cite book |editor=甄人 |others= |title=廣州之最 |year=1993 |month= |publisher=廣東人民出版社 |location=廣州 |isbn=7-218-01149-7 |pages=第300-301頁 |chapter=廣州最古老的茶樓 }}</ref>，曾有多家分店<ref name = "古老茶樓"/><ref name = "成珠樓">{{Cite web |url=http://eat.gd.sina.com.cn/news/2006-03-05/2262750_2.html |title=风风雨雨 有起有落 |accessdate=2012-07-27 |work=廣州文化局 |publisher=新浪網 |date=2006-03-05 |quote= }}</ref>。於2000年歇業拍賣，僅余成珠餅家延續其業務，2006年茶樓主體拆卸。<ref name = "260成珠樓">{{Cite web |url=http://news.sina.com.cn/o/2006-03-03/09418351105s.shtml |title=260歲成珠樓停業要拆了 |accessdate=2012-07-27 |work=廣州日報 |publisher=新浪網 |date=2006-03-03 }}</ref>有最著名的特產[[雞仔餅|成珠小鳳餅]]，現於成珠餅家仍存。<ref>{{Cite web |url=http://gd.news.sina.com.cn/local/2006-03-05/2262254.html |title=滄桑260年 廣州最古老茶樓拆遷在即 |accessdate=2012-07-27 |work=廣州市文化局 |publisher=新浪網 |date=2006-03-05}}</ref>，而[[广州市政府|广州市政府]]颁发的“广州老字号”牌匾及“小凤”牌鸡仔饼的商标招牌仍存。<ref>{{Cite web |url=http://news.sina.com.cn/o/2006-03-03/09418351105s.shtml |title=260歲成珠樓停業要拆了 |accessdate=2012-07-27 |work=廣州日報 |publisher=新浪網 |date=2006-03-03}}</ref><ref name = "260成珠樓"/><ref name = "惋惜"/>

== 歷史 ==

* 成珠樓前身名為'''成珠館'''，為廣州豪門伍氏{{notetag|1=當時廣州五大家族：潘、卢、伍、叶、周}}家主[[伍紫坦|伍紫坦]]府內待客之私地。<ref name = "名揚千里">{{cite news |author=大變革 |title=名扬千里成珠楼 |url=http://epaper.nfdaily.cn/html/2010-01/21/content_6814244.htm |work=南方日報 |publisher= |date=2010-01-21 |accessdate=2012-07-27 }} </ref>
* 伍家有客临门，但点心师出外，婢女小凤端出一点心，即是后来於咸丰五年名动一方的成珠小凤饼。<ref name = "古老茶樓"/><ref>{{cite book |editor=李禎蓀 |title=名城廣州小百科 |year=1992 |publisher=中山大學出版社 |location=廣州 |pages=第200頁|isbn=7-306-00553-7 }}</ref>
* 伍氏衰落，成珠馆数易其主，曾改名'''成珠餅家'''，兼营多种业务而少有起色。<ref name = "前世今生">{{Cite web |url=http://eat.gd.sina.com.cn/news/2006-03-05/2262750.html |title=成珠楼的前世今生 |accessdate=2012-07-27 |work=廣州文化局 |publisher=新浪網 |date=2006-03-05 |quote= }}</ref>
* [[光緒|光緒]]年間，成珠館成為梁氏[[福和堂|福和堂]]物業，時任家主[[梁殿華|梁殿華]]將其易名'''成珠樓'''。<ref name = "前世今生"/>
* 1928年，廣州市地政局勒索成珠樓，索要2.5万[[銀元|銀元]]，否則新開馬路將從成珠樓正中穿過。成珠樓不理睬，最後果然被“腰斬”。<ref name = "古老茶樓"/>被“腰斬”後，馬路南樓擴建營業而北樓作工場生產[[小鳳餅|小鳳餅]]。<ref name = "成珠樓"/>
* 1931年，成珠小鳳餅參加展覽，屢獲獎章，成為[[廣州|廣州]]代表[[手信|手信]]。<ref name = "小鳳餅">{{Cite web |url=http://eat.gd.sina.com.cn/news/2006-03-05/2262750_1.html |title=“一餅驚人”之小鳳餅 |accessdate=2012-07-27 |work=廣州文化局 |publisher=新浪網 |date=2006-03-05 }}</ref>
* 1937年，政府開闢馬路將成珠樓分開成兩邊<ref>{{Cite web|url=http://www.gzlzh.com.cn/zh-CN/displaynews.html?newsID=94293|title=成珠楼：浴火小凤，艰难重生|accessdate=|author=|date=2013-07-10|publisher=廣州老字號協會}}</ref>。
* [[抗日戰爭|抗日戰爭]][[廣州淪陷|廣州淪陷]]其間歇業後復業，至1947年曾設3家分店於西关第十甫成的珠茶楼、成珠饼家及澳门支店<ref name="260成珠樓" /><ref name="成珠樓" />。
* 1946年，成珠樓200週年。<ref name = "小鳳餅"/>
* 1959年，成珠樓公開小鳳餅配方。<ref name = "名揚千里"/>
* 1985年10月9日，成珠樓南樓當晚2樓擺喪酒，3樓擺喜酒，位於2樓的發電機起火引起火災，超過100人罹難<ref>{{cite web|url=http://yes-news.com/yespick/134456/%E5%BB%A3%E5%B7%9E%E5%8D%81%E5%A4%A7%E9%82%AA%E5%9C%B0%E4%B9%8B%E9%A6%96-%E6%88%90%E7%8F%A0%E9%85%92%E6%A8%93-~-%E7%82%BA%E4%BA%86%E8%B2%AA%E9%8C%A23%E6%A8%93%E6%93%BA%E7%B5%90%E5%A9%9A%E9%85%922%E6%A8%93%E6%93%BA%E5%96%AA%E9%85%92%E7%B5%90%E6%9E%9C |title=廣州十大邪地之首：成珠酒樓 ~ 為了貪錢3樓擺結婚酒，2樓擺喪酒，結果......}}</ref>。
* 1990年，成珠樓原址重建，易名'''成珠酒家'''，最旺盛時有[[南华东路|南华东路]]分店和[[荔福路|荔福路]]分店和[[新洲食品廠|新洲食品廠]]<ref name = "260成珠樓"/>。
* 2000年，成珠樓結業，同時老東主老員工重開成珠餅家。<ref name = "260成珠樓"/><ref name = "惋惜"/>
* 2002年，[[東江飲食集團|東江飲食集團]]拍得成珠樓物業，更名鴻星海鮮酒家。<ref name = "惋惜">{{Cite web |url=http://gd.news.sina.com.cn/local/2006-03-05/2262254.html |title=滄桑260年 廣州最古老茶樓拆遷在即 |accessdate=2012-07-27 |work=廣州市文化局 |publisher=新浪網 |date=2006-03-05 }}</ref>
* 2006年3月1日，時名“鴻星海鮮酒家”的成珠樓正式結業，原址開發房地產。<ref name = "260成珠樓"/>
