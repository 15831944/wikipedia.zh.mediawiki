'''成追線'''是指[[縱貫線_(鐵路)|縱貫線]]從[[成功車站|成功]]至[[追分車站|追分]]間，由[[臺灣鐵路管理局|臺灣鐵路管理局]]（台鐵）經營的傳統[[鐵路|鐵路]]幹線。其主要功能是[[臺中線|山線]]與[[海岸線_(臺鐵)|海線]]之間的[[聯絡線|聯絡線]]。目前成追線雙軌化工程施工中，預計2020年完工<ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/1845975|title=成功到追分鐵路雙軌化 中央全數補助|author= 蘇金鳳|date=2016-10-04 |publisher=自由時報|language=zh  |archiveurl=|archivedate=}}</ref>。

== 路線資料 ==
*經營管轄：[[臺灣鐵路管理局|臺灣鐵路管理局]]
*路線距離：成功＝追分間 2.2 公里<ref>{{cite web|language=zh-tw|url=http://www.railway.gov.tw/Upload/intro/file/99Year/pdf/t2.pdf|title=營業里程－ 按區間別分|publisher=台灣鐵路管理局|date=中華民國101年底|accessdate=2014-05-16}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
*[[軌距|軌距]]：1,067 毫米
*車站數：2（含起訖車站；2006年5月）
*開業時間：1922年10月11日全線開通
*[[電氣化鐵路|電化]]區間：全線（[[交流電|交流]]25KV）
*目前為單線，正在進行雙線化工程（已在2019年3月30日晚上進行雙線化切換工程）

== 運行形態 ==
[[File:TRA_Taichung_Line_(left)_and_Chengzhui_Line_(Right).JPG|thumb]]
*每日有數班行駛於山線與海線之間的成追線直通列車。
*經成追線發車站有[[竹南車站|竹南車站]]、[[苗栗車站|苗栗車站]]、[[銅鑼車站|銅鑼車站]]、[[三義車站|三義車站]]、[[后里車站|后里車站]]、[[豐原車站|豐原車站]]、[[臺中車站|臺中車站]]、[[通霄車站|通霄車站]]、[[大甲車站|大甲車站]]、[[清水車站_(台灣)|清水車站]]、[[沙鹿車站|沙鹿車站]]、[[龍井車站|龍井車站]]、[[大肚車站|大肚車站]]、[[追分車站|追分車站]]、[[新烏日車站|新烏日車站]]、[[烏日車站|烏日車站]]等站。
*從山線經過本線直接前往海線（或反之由海線轉山線），將使整組列車從南下變成北上。也會使得列車編組標示的車序（第1車、第2車等）與車站月臺所標示相反，造成旅客困擾。
*本線在1987年之前，時刻表曾有[[對號列車|對號列車]]行經此處，但皆從[[臺中車站|臺中]]經海線往返北部，之後有一段很長時間，本線沒有任何旅客列車行駛，僅在遇有災害或事故造成山線路線中斷，而仍欲使原經山線高級列車進出臺中站，才會臨時繞經本線。
*近年來因應捷運化需要，臺中市[[山線|山線]]與[[海線|海線]]地區彼此也有往返之交通需求，甚至延伸至[[苗栗縣|苗栗縣]]，開始恢復旅客列車行經本線，免除須至[[彰化車站|彰化]]換車之不便，也減少12公里的路程。
*目前僅行駛[[臺鐵區間車|區間車]]。

== 簡介 ==
本線的興築，與[[海線|海線]]有密切關係。1917年發生[[滯貨事件|滯貨事件]]，令[[總督府|總督府]][[鐵道部|鐵道部]]決定興築海線。1920年12月15日，自王田驛（今[[成功車站|成功車站]]）到[[清水車站_(台灣)|清水]]的路線通車，當時稱為「王田支線」。之後隨著於1922年10月11日通車至竹南，同時新設追分驛（今[[追分車站|追分車站]]），由追分延伸鐵路至[[彰化車站|彰化]]，海線取代原先經台中的路線成為[[縱貫線_(鐵路)|縱貫線]]，而王田＝追分間的路段，則納入[[臺中線|臺中線]]。

海線完工之初，大部分快車即改走海線，但為了因應臺中站客運需求，部分快車會利用海線與本路段行駛到臺中，此種模式稱為「循迴追分」；後來為了運轉調度上方便，漸漸改由，在彰化聯掛或解聯部分車廂，再開行臺中彰化間連絡列車，因此本路段的使用率大減。隨著臺鐵大量購入EMU500型通勤用電車，以及日益增加的山海線間通勤需求，開始開行經此線往來山海線之區間車。開行之際，因電腦售票程式不及更新，臺中車站特別申請[[埃德蒙森式鐵路車票|名片式車票]]應急。

由於此線與山海線形成[[三角線|三角線]]，因此偶爾有需要轉向的車輛，會利用此線完成轉向工作。

本線兩端站名「追分成功」，具備祝福考生高分上榜的吉祥意義，常有[[升學|升學]]考試的應試生在考前購買成功、追分兩站發售的名片式車票以求順利；又「分」字的聲符「ㄈ」唸成「ㄏ」（[[臺灣國語|台灣國語]]），則近似「婚」字，故亦常被情侶們當成求婚的護身符。這些原因使得「追分成功」的名片式車票成為熱賣商品。
