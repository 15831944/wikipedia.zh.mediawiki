'''成均館'''（{{lang-ko|'''성균관'''}}）是[[朝鮮半島|朝鮮半島]]歷代王朝最高學府，地位同等於[[中國歷史|古代中國]]的[[國子監|國子監]]、[[太學|太學]]。[[高麗王朝|高麗王朝]]的成均館位於現今[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]][[開城|開城]]，而[[朝鮮王朝|朝鮮王朝]]的成均館位於現今[[大韓民國|大韓民國]][[首爾|首爾]]。現成均館為三星集團旗下企業大學。


2013年6月23日，[[聯合國教科文組織|聯合國教科文組織]]在柬埔寨舉行的第37屆世界遺產大會，公佈把開城歷史古蹟（包括開城區、滿月台、開城南大門、成均館、嵩陽書院、王建皇陵和恭愍皇陵）列入世界文化遺產。

== 建築 ==
* 佔地約1萬平方公呎，共有18棟建築物；
* [[明倫堂|明倫堂]]：位於中央，用作講堂；
* [[東齋|東齋]]及[[西齋|西齋]]：位於明倫堂的兩旁，它們是集體[[宿舍|宿舍]]；
* 已生長了1000多年的[[櫸|櫸]]樹及[[銀杏|銀杏]]樹

== 成立歷史 ==
成均一詞源於《[[周禮|周禮]]·春官·大司樂》：「大司樂掌成均之法，以治建國之學政，而合國之子弟焉。」。原為周代的貴族大學之一。[[鄭玄|鄭玄]]注：「[[董仲舒|董仲舒]]曰：五帝名大學曰成均。」。成均泛稱官設的最高學府。

在古代[[朝鮮半島|朝鮮半島]]的[[高麗|高麗]]時代（公元918年至1392年），為了培養人才為國家的未來發展，在公元992年建立了最高教育機關[[國子監|國子監]]，後來改組成為成均館。隨著高麗王朝被朝鮮王朝取代、定都漢陽（今首爾特別市），朝鮮太宗[[李芳遠|李芳遠]]將新成均館建立於都內，也就是現今的首爾[[成均館大學|成均館大學]]。

== 當時的職責 ==
[[高麗|高麗]]政府委任飽讀中國的[[儒家思想|儒家思想]]學者來經營，對學生傳授：[[儒家思想|儒家思想]]、[[高麗|高麗]][[法律|法律]]、[[數學|數學]]及[[書法|書法]]等科目。

== 成就 ==
* 製造了[[國際|國際]]聞名的高麗[[青瓷|青瓷]]；
* 製造了顏色雪白、質地堅韌的優質[[紙張|紙張]]；
* 除[[中國|中國]]外，當時世界上唯一使用活字印刷技術；
* 研究出在[[朝鮮半島|朝鮮半島]]栽種[[棉花|棉花]]的方法。

== 現況 ==
在-{zh-hant:南北韓;zh-hans:朝韩}-雙方政府的保護歷史文物政策下，各建築物都保存良好，現今分別為北韓[[高麗成均館輕工業綜合大學|高麗成均館輕工業綜合大學]]與南韓[[成均館大學|成均館大學]]。南韓[[成均館大學|成均館大學]]以[[三星集團|三星集團]]為贊助財團，並為韓國前五大學之一。
