'''成都中医药大学'''位于[[中国|中国]][[成都市|成都市]]，前身是1956年8月创建的'''成都中医学院'''，是中国最早创办的中医学院之一，首任校长为[[李斯炽|李斯炽]]，现任校长为[[余曙光|余曙光]]。成都中医药大学是一所以中医药学科为主的高等学校，也是首批[[雙一流|雙一流]]高校，综合实力位居同类高校前列，在行业内被称为“老四校”之一。

学校占地面积1800亩，有16个学院、33个本科专业，全日制在校学生2.2万人，其中研究生2000余人。

== 历史 ==

* 1936年，在成都成立[[四川国医学院|四川国医学院]]。
* 1956年，在[[周恩来|周恩来]]等党和国家第一代领导集体的亲切关怀下，经国务院批准建立，是我国最早的四所中医药高等院校之一。
* 1995年，经[[国家教委|国家教委]]批准更名为成都中医药大学。
* 2006年，原[[四川省卫生管理干部学院|四川省卫生管理干部学院]]、[[四川生殖卫生学院|四川生殖卫生学院]]并入成都中医药大学。

== 学科建设 ==

学校是全国首批[[中医药|中医药]]学博士、硕士学位授权点，首批[[临床医学|临床医学]]（硕士、博士）专业学位试点单位。现有一级学科博士学位授权点3个、二级学科博士学位授权点20个，有一级学科硕士授权点7个、二级学科硕士学位授权点49个，有博士后流动站3个，博士后工作站4个。
学校有国家级重点学科4个（[[中药学|中药学]]、[[中医五官科学|中医五官科学]]、[[针灸推拿学|针灸推拿学]]、[[中医妇科学|中医妇科学]]），省部级重点学科44个。有国家级特色专业6个、省级特色专业7个，国家级精品课程5门、省级精品课程31门，国家级优秀教学团队3个、省级优秀教学团队5个。有国家级人才培养模式创新实验区1个，省级人才培养模式创新实验区2个等。

== 学术研究 ==

“十二五”以来，新上各类科研项目1254项，其中[[国家科技重大专项|国家科技重大专项]]、[[国家重点基础研究发展计划|国家重点基础研究发展计划]]（[[973计划|973计划]]）、[[国家自然科学基金|国家自然科学基金]]、[[社会科学基金|社会科学基金]]等国家级项目186项，获得[[国家科技进步奖|国家科技进步奖]]二等奖等各级各类科技奖励53项，其中国家科技进步二等奖3项，省科技进步特等奖1项。
现有各级各类实验室66个，其中有[[省部共建国家重点实验室培育基地|省部共建国家重点实验室培育基地]]，[[国家级实验教学示范中心|国家级实验教学示范中心]]，[[中药饮片炮制国家地方联合工程研究中心|中药饮片炮制国家地方联合工程研究中心]]，[[国家中药材种质资源库|国家中药材种质资源库]]（四川），[[教育部重点实验室|教育部重点实验室]]，[[教育部工程研究中心|教育部工程研究中心]]，[[国家中医药管理局重点研究室|国家中医药管理局重点研究室]]，[[国家中医药管理局中医药科研实验室|国家中医药管理局中医药科研实验室]]（三级）、[[四川省重点实验室|四川省重点实验室]]等。
