'''成田湯川站'''（{{jpn|j=成田湯川駅|hg=なりたゆかわえき|rm=Narita-yukawa eki|f=y}}）是位於[[日本|日本]][[千葉縣|千葉縣]][[成田市|成田市]]松崎，屬於[[京成電鐵|京成電鐵]][[成田機場線|成田機場線]]（成田Sky Access）<!--京成公式HPおよび各報道機関でも愛称名である「成田スカイアクセス」と案内されているため、愛称名を先に記載する　※議論中-->的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''KS43'''。

== 概要 ==
車站興建在[[成田市|成田市]]西部的{{link-ja|成田新市鎮|成田ニュータウン}}北端。以前有千葉交通成田營業所湯川車庫。過去末班車在晚上十點時段，2017年10月改點時將上行末班車延後至晚上11點時段。2012年8月1日至31日曾在早晨、深夜開設臨時列車。

[[東日本旅客鐵路|東日本旅客鐵路]]（JR東日本）的[[成田線|成田線]]（我孫子支線）從車站東側通過，但未設站。該線最近的車站是西北方約1.5公里的[[下總松崎站|下總松崎站]]。
本站採上下分離管理模式，車站、路軌等由成田高速鐵道Access公司建設及擁有，而[[京成電鐵|京成電鐵]]負責運行及管理。列車運行方面，現階段只有機場接駁的Access特急（不需特急料金）列車停站，[[北總線|北總線]]列車皆以[[印旛日本醫大|印旛日本醫大]]為終點站。

由於該站與[[成田線|成田線]]交匯，成田線（我孫子－成田段）活性化推進協議會等居民組織曾爭取JR東日本增設新站，但JR東日本沒有增設新站的計畫。
== 歷史 ==
*2010年7月17日 - 開業

== 站名由來 ==
規畫時的臨時站名為「成田新市鎮北站」。

在針對成田市內居民與工作、就學者進行調查後，選擇以當地熟悉的「湯川」冠上知名度高的「成田」，命名為「成田湯川」<ref>[http://www.npf-airport.jp/news/airportnews/news_0906/news/news_data090603.htm成田新高速鉄道の新駅名称が決定]</ref>。

== 車站結構 ==
[[File:Naritayugawa_station_kounai.JPG|thumb]]
該高架車站有兩個[[側式月台|側式月台]]及兩條通過線。站內設有6台扶手電梯及3部升降機。

車站東側部分是[[單線|單線]]路段，合流部分設有日本最大的38號路軌轉轍器。印旛日本醫大側有橫渡線。

大廳有[[花窗玻璃|花窗玻璃]]作為裝飾。
