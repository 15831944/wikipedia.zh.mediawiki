'''成吉思汗勋章'''，是[[蒙古国|蒙古国]]最高荣誉勋章<ref name=ub>{{Cite web|url = http://theubpost.mn/2016/10/24/artist-g-ariunbaatar-to-receive-order-of-chinggis-khaan/|title = Artist G.Ariunbaatar to receive Order of Chinggis Khaan|author = |date = 2016-10-24|publisher = UB POST|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20161230085722/http://theubpost.mn/2016/10/24/artist-g-ariunbaatar-to-receive-order-of-chinggis-khaan/|archive-date = 2016-12-30|dead-url = yes}}</ref>。

==简介==
2002年，经[[蒙古国总统|蒙古国总统]][[那楚克·巴嘎班迪|那楚克·巴嘎班迪]]倡议，[[国家大呼拉尔|国家大呼拉尔]]通过了第21号决议，决定设立成吉思汗勋章，授予为巩固蒙古国家独立自主及国家团结、深入研究和传承及让世界了解蒙古民族历史文化传统作出特殊贡献的人士，每年7月11日庆祝[[那达慕|那达慕]]之际颁发。后来，对该决议进行了修订，改在[[成吉思汗|成吉思汗]]诞辰日即“蒙古民族自豪日”颁发成吉思汗勋章<ref name=xunzhang>{{Cite web |url = http://www.weixinyidu.com/n_2555839  |title =蒙古国著名诗人格·孟德奥耀荣获“成吉思汗”勋章  |author =  |date =2015-11-24  |publisher = 头条易读|language =  |accessdate =  |quote =  }}</ref>。历年获得成吉思汗勋章的人士是：

*2005年：[[彭萨勒玛·奥其尔巴特|彭萨勒玛·奥其尔巴特]]，蒙古国首任总统<ref name=xunzhang/>
*2006年：[[宾巴苏伦·沙拉布|宾巴苏伦·沙拉布]]，音乐家<ref name=xunzhang/>
*2009年：[[达希·宾巴苏伦|达希·宾巴苏伦]]，前总理<ref name=xunzhang/>
*2010年：[[拉德那苏木贝尔勒·贡其格道尔吉|拉德那苏木贝尔勒·贡其格道尔吉]]，前[[国家大呼拉尔|国家大呼拉尔]]主席<ref name=xunzhang/>
*2011年：[[那楚克·巴嘎班迪|那楚克·巴嘎班迪]]，蒙古国第二任总统<ref name=xunzhang/>
*2012年：[[达木丁·登贝尔勒|达木丁·登贝尔勒]]，国家大呼拉尔主席<ref name=xunzhang/>
*2013年：[[杜马·索德诺姆|杜马·索德诺姆]]，蒙古国第14任总理<ref name=xunzhang/>
*2014年：[[哈布特盖·那木斯来|哈布特盖·那木斯来]]，物理学家<ref name=xunzhang/>
*2015年：[[贡布扎布·门德奥约|贡布扎布·门德奥约]]，诗人<ref name=xunzhang/>
*2016年：[[钢巴特尔·阿伦巴特尔|钢巴特尔·阿伦巴特尔]]，歌唱家<ref name=ub/><ref>{{Cite web |url =http://www.sohu.com/a/117886717_508040  |title =蒙古国功勋演员钢.艾润巴特尔荣获国家最高奖“成吉思汗勋章"总统亲自颁奖  |author =  |date = 2016-11-01 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>
