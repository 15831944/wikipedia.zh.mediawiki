'''矢乃波波木神'''（{{lang-ja|ヤノハハキノカミ}}）為[[日本|日本]][[神道|神道]]的[[神祇|神祇]]，也是日本民間信仰中可掃除厄運的帚神，亦可寫作'''波比祇神'''（{{ja}}ハヒキノカミ）。

== 概要 ==
據《[[古語拾遺|古語拾遺]]》記述，原本宮中[[神祇官|神祇官]]在西院祭祀坐摩巫（{{ja}}イカスリノミカンナギ），而《[[延喜式神名帳|延喜式神名帳]]》則提到坐摩巫祭神共有五位神明，其說明如下：
# 生井神（{{ja}}イクヰノカミ）：具有生命力的井水之神。
# 福井神（{{ja}}サクヰノカミ）：幸福與繁榮的井水之神。
# 綱長井神（{{ja}}ツナカヰノカミ）：深且清澈的井水之神。
# '''波比祇神'''（{{ja}}ハヒキノカミ）：房屋庭院之神、帚神、竈神。
# 阿須波神（{{ja}}アスハノカミ）：竈神、旅人之神。

== 解說 ==
矢乃波波木神是日本民間信仰中的帚神，神格與[[灶君|竈神]]相當。神名裡的「波波木（{{ja}}ハハキ）」與「母木」同音，意謂「孕育生命之木」，故也被信徒視為安胎順產之神。因為[[帚|掃把]]能清除髒亂，亦能去除霉厄，保佑胎兒的平安。

在日本同樣被視為帚神而信仰的，尚有高忍日賣命（{{ja}}タカオシヒメノミコト，為[[愛媛縣|愛媛縣]][[伊予郡|伊予郡]]松前町之高忍日賣命神社主祭神）、[[神功皇后|神功皇后]]等。

== 信仰 ==
在日本主祭矢乃波波木神的[[神社|神社]]除[[伊勢神宮|伊勢神宮]]的內宮外，另有下列：
* 坐摩神社（[[大阪府|大阪府]][[大東市|大東市]]）：稱為「波比岐大神」。
