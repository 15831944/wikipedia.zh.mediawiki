'''矢野茜'''（{{lang-ja|矢野 茜}}，{{bd|1992年|2月24日}}<ref>{{Cite web |url=https://twitter.com/if_224/status/967344633045950464 |title=矢野茜 on Twitter: "26さいになりました🌟( ´罒` )… " |language=ja |accessdate=2018-05-20 |date=2018-02-24}}</ref>），[[日本|日本]]女[[原画师|原画师]]，[[作画监督|作画监督]]。出身於[[千叶县|千叶县]]。{{tsl|ja|東京アニメーター学院|东京动画学院专科学校}}毕业。{{link-ja|日本編劇聯盟|日本脚本家連盟}}會員。网上盛称为“美少女作监”。<ref>{{cite news |publisher=[[週刊Playboy|週刊Playboy]] |date=2018-02-20 |accessdate=2018-03-18 |language=ja |url=http://wpb.shueisha.co.jp/2018/02/20/100044/ |title=２５歳にして総作画監督！ 注目の女性アニメーター・矢野茜が“アニメのおしごと！”を志した理由 |author=加藤真大（文）、市村圆香（摄）}}</ref><ref>{{cite news |publisher=週刊Playboy |date=2018-02-21 |accessdate=2018-03-18 |language=ja |url=http://wpb.shueisha.co.jp/2018/02/21/100060/2 |title=ルックスも話題の女性アニメーター・矢野茜は完全実力派！「贔屓されてやっていると思われたくない」 |author=加藤真大（文）、市村圆香（摄）}}</ref>

== 来历 ==
最初的志愿并非是成为一位画师而是想成为一名[[声优|声优]]。

在学生时期曾经深受[[高桥留美子|高桥留美子]]的漫画作品和动画《[[机动战士高达SEED|机动战士高达SEED]]》影响，成为了她入行的契机。在声优的学习课程中表现出色，曾通过学校内试音会，也去过动画录制现场配过路人。第二学年期间，曾为了要选择声优或是动画师而苦恼。因为感受到了动画制作者们“从一开始创作一个世界”的魅力而选择了专攻动画方向。最初加入动画公司feel，也是想做出《[[缘之空|缘之空]]》这样色气满满的作品。加入没多久后就离职成为了自由原画师。

自2013年以来多担任第二原画，后在《[[伪恋|伪恋]]》的第一集成为原画，在最后一话成为作画监督。得到了《伪恋》的制作公司[[SHAFT|SHAFT]]的社长久保田光俊的赏识。<ref>{{cite web |publisher=Anitama |language=zh-cn |date=2016-11-16 |accessdate=2018-03-18 |title=差点成为偶像声优的美少女作画监督《线上游戏的老婆不可能是女生？》{{!}}人物设计矢野茜访谈 |url=http://www.anitama.cn/article/d0a35739edabb909 |author=III}}</ref><ref>{{cite web |publisher=Anitama |language=zh-cn |date=2018年02-24 |accessdate=2018-03-18 |title=总作监：不靠颜值的工作！{{!}}作画监督矢野茜访 |url=http://www.anitama.cn/article/2b7474fc7af3b9f5 |author=izumi}}</ref>
