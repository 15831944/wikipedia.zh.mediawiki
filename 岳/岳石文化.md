'''岳石文化'''是[[中国|中国]][[黄河|黄河]]下游地区的[[新石器时代|新石器时代]]文化，得名于[[山东|山东]][[平度|平度]][[东岳石遗址|东岳石遗址]]。岳石文化晚于[[山东龙山文化|山东龙山文化]]而早于商代[[二里冈文化|二里冈上层文化]]，年代大约在公元前1900～前1600年间，属于[[城邦国家|城邦国家]]发展时期。

岳石文化的分布范围比较明确，以泰沂山为中心，北起[[山东|鲁北]][[河北|冀中]]及辽东半岛南端，向南越过[[淮河|淮河]]，西自[[山东省|山东省]]最西部、[[河南省|河南省]]的[[兰考|兰考]]、[[杞县|杞县]]、[[淮阳|淮阳]]一线，东至[[黄海|黄海]]之滨。

牟平照格庄遗址出土的[[青铜锥|青铜锥]]以及泗水尹家城遗址的青铜[[镞|镞]]说明岳石文化已进入[[青铜时代|青铜时代]]。

== 发现 ==
从岳石文化遗存的发现到对其认识和命名，经历了一个比较长的时期，实际上，岳石文化是和[[龙山文化|龙山文化]]同时发现的。1959年修淄阳水库时，在大泽山东岳石村附近岸边发现[[东岳石遗址|东岳石遗址]]。考古工作者在此发掘出大量的[[石器|石器]]、[[陶器|陶器]]、[[骨器|骨器]]和[[蚌器|蚌器]]，经考证为[[东夷|东夷]]部落创造的一种古老文化。岳石文化是[[山东|山东]]省境内继“[[龙山文化|龙山文化]]”（1930年在[[章丘|章丘]]龙山发现）、“[[大汶口文化|大汶口文化]]”（1959年在[[宁阳|宁阳]]大汶口发现）两种最重要的考古“文化”之后的又一次具有划时代意义的考古发现。<ref>{{ cite web|url=http://digitalmuseum.zju.edu.cn/front.do?methede=showTheme&oid=8a8691a6272ccb7f01272d15b3820084&schoolid=49|title=岳石文化|publisher=山东大学考古数字博物馆|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20161201015130/http://digitalmuseum.zju.edu.cn/front.do?methede=showTheme&oid=8a8691a6272ccb7f01272d15b3820084&schoolid=49|archivedate=2016-12-01}}</ref>

== 遗址 ==
岳石文化的研究始于70年代末期，在此以后，随着人们追寻龙山文化去向问题的展开，有关岳石文化的发掘工作迅速增多，规模也不断扩大。

岳石文化所发现的遗址数量不如[[龙山文化|龙山文化]]多，经过发掘的有30余处。其中比较重要的有[[山东省|山东省]]的[[平度|平度]][[东岳石遗址|东岳石]]、[[菏泽|菏泽]][[安邱堌堆遗址|安邱堌堆]]和[[河南省|河南省]]的[[杞县|杞县]][[鹿台岗遗址|鹿台岗]]等。

目前经过发掘的遗址主要有：
* [[平度|平度]][[东岳石遗址|东岳石遗址]]
* [[牟平|牟平]]照各庄遗址
* [[青州|青州]]郝家庄遗址
* [[章丘|章丘]]王推官庄遗址
* [[泗水|泗水]]尹家城遗址
* [[菏泽|菏泽]][[安邱堌堆遗址|安邱堌堆遗址]]
* [[杞县|杞县]][[鹿台岗遗址|鹿台岗遗址]]

== 文化特征 ==
=== 基本特征 ===
古朴典雅、厚重使用是岳石文化[[陶器|陶器]]的主要风格，而夹砂陶草率粗糙，泥制陶古朴精致，两者反差甚大，又是岳石文化陶器的重要特征，推测两者是出自不同作坊的产品。其他诸如盛行子母口、凸棱、唇边外凸或叠唇、器物转折圆钝以及器底周缘外凸的作风，均构成岳石文化区别于其他文化的特征与风格。

=== 分期 ===
主要根据尹家城、[[安丘堌堆遗址|安丘堌堆]]、照格庄和丁公等遗址的资料，将岳石文化划分为四期。

=== 典型陶器的演化 ===
*[[甗|甗]]，岳石文化的主要炊器，数量甚多，已发现者多为腰以下部分，完整者甚少；
*子母口[[罐|罐]]，此类器型在岳石文化中数量不多，特征鲜明，并与龙山文化同类器的传承关系较为清楚；
*平底[[尊|尊]]，是岳石文化指征性器类之一，可分三式；
*平底盒，均为泥质陶，高子口，折腹处突棱外伸，可分为3式；
*蘑菇纽器盖，是岳石文化的指征性器类之一，造型特征为顶部有空心或实心蘑菇形纽，子母口，可分为3式。

=== 年代 ===
大约在距今3950～3500年之间。其上限有可能延伸到距今4000年，个别地区的下限有可能还会稍晚一些。

== 地方类型 ==
*照格庄类型：其分布区域以[[胶莱河|胶莱河]]以东的[[胶东半岛|胶东半岛]]为主，包括周围沿海岛屿。
*郝家庄类型：分布区域以[[弥河|弥河]]和[[潍河|潍河]]流域为主，东可抵[[胶莱平原|胶莱平原]]西部，向南进入[[沂山|沂山]]北麓一带，西到[[淄河|淄河]]流域。
*土城类型：以[[沂河|沂河]]、[[沭河|沭河]]中上游地区为主，并且包括[[日照|日照]]、[[赣榆|赣榆]]等东部沿海地区在内的沂蒙地区。
*王推官类型：分布于[[泰山|岱北]]地区，东起[[孝妇河|孝妇河]]流域，西到[[聊城|聊城]]西部，北界可能已抵[[河北|冀东南]]一带。
*尹家城类型：包括[[泰山|岱南]]的[[汶河|汶河]]和[[泗河|泗河]]中上游地区，西界约在[[京杭大运河|京杭大运河]]两岸一带。
*[[安邱堌堆遗址|安邱堌堆]]类型：主要分布在[[山东|鲁西南]]、[[河南|豫东]]和[[安徽|皖北]]地区，与[[龙山文化|龙山文化]][[王油坊遗址|王油坊]]类型的分布区域基本一致，两者之间有继承发展关系。

== 社会经济 ==
*农业：岳石文化农业生产的发展，主要表现在[[农具|农具]]的改进创新和农具在全部工具中比例的上升两方面。
*冶金业：[[青铜|青铜]]冶铸业是岳石文化时期成就最突出的手工业部门，此时已进入早期[[青铜时代|青铜时代]]。
*建筑业：岳石文化的建筑技术水平，首先体现在城市建设方面；房屋建筑技术主要是继承了龙山文化的传统。

[[夏代|夏代]]，[[山东半岛|海岱地区]]的经济比龙山文化时期进一步发展了。从岳石文化的遗存来看，生产工具中主要是用[[石器|石]]、[[骨器|骨]]、[[角器|角]]、[[蚌器|蚌]]制作的[[农具|农具]]，也出土了少量青铜锥形器。在[[梁山|梁山]]青堌堆的岳石文化遗存中，发现了石制的[[斧|斧]]、[[刀|刀]]、[[铲|铲]]、[[锤|锤]]等；[[骨器|骨器]]有[[镞|镞]]、[[凿|凿]]、[[锥|锥]]、[[针|针]]；[[角器|角器]]有[[镞|镞]]、[[锥|锥]]和带倒刺的[[鱼镖|鱼镖]]；[[蚌器|蚌器]]有[[镞|镞]]与[[镰|镰]]。此外，还有一些陶纺轮和网坠。青堌堆遗址还出土了不少淡水厚壳蚌和一些鹿角。从上述生产工具和动物遗骸，可以看出采集与渔猎经济在氏族的经济生活中仍占有一定地位。

岳石文化的[[制陶技术|制陶技术]]比起龙山文化时期的鼎盛阶段，种类和器形都有较大变化，显出明显的衰落。大多数陶器为泥质灰胎[[黑皮陶|黑皮陶]]和夹砂[[红褐陶|红褐陶]]。泥质陶的胎壁厚，种类少，不过都采取了先进的轮制方法，火候也比较高，普遍饰有凸棱数周而形成独特的风格。这一时期的陶器主要有[[尊|尊]]、子母口[[罐|罐]]、三足[[簋|簋]]、圈足[[簋|簋]]、[[豆|豆]]、[[盂|盂]]、[[鼎|鼎]]、[[盆|盆]]、[[杯|杯]]及大量蘑菇状钮的器盖。一些泥质陶器上还绘有朱色图案，也有用红、白、黄多彩兼施的[[云纹|云纹]][[彩绘陶|彩绘陶]]。夹砂的红褐陶器表面抹光，有敞口的大型罐和盆，还发现了袋足肥大的陶[[鬶|鬶]]，这些陶器的颈、腰、裆部常有附加堆纹。上述因素均不见于典型龙山文化，而与中原的[[夏代|夏代]]文化有一定关系。在大汶口文化和龙山文化中十分盛行的[[鬶|鬶]]，至岳石文化时期已很少见到了。

[[河南龙山文化|河南龙山文化]]晚期阶段，已进入[[夏代|夏代]]统治时期。[[河南|豫东]][[龙山文化|龙山文化]]的[[造律台遗址|造律台]]类型，与[[鲁西平原|鲁西平原]]的[[梁山|梁山]]青堌堆等岳石文化遗址所反映的面貌是一致的，说明[[夏代|夏代]][[二里頭文化|二里頭文化]]迅速东进，其中主要是[[商部落|商部落]]势力向东的发展。受到商部落的排挤，岳石文化的一些氏族向[[淮河|淮河]]一带迁徙，在[[涡河|涡河]]、[[浍河|浍河]]、[[沱河|沱河]]、[[唐河|唐河]]的上游与那里的氏族相融合，成为[[商代|商代]]的[[淮夷|淮夷]]；还有一些氏族继续南下，在[[南京|南京]]、[[镇江|镇江]]一带发展成后来的“[[湖熟文化|湖熟文化]]”。[[泰山|泰山]]、[[沂蒙山|沂蒙山]]以东则继续发展着岳石文化。

[[淮夷|淮夷]]的形成与发展，与岳石文化一些氏族南下有关，也与原在[[河南|豫东]]地区的[[有虞氏|有虞氏]]向东南迁移有一定关系。这种大规模的迁移，一方面是因为[[中原|中原]]和[[山东|山东]]两地新石器时代末期诸部落集团的矛盾冲突，另一方面受到[[黄河中下游|黄河中下游]]及[[淮河|淮河]]流域的巨大洪水灾患的直接影响。[[安徽省|安徽省]][[安徽|淮北地区]]的[[宿县|宿县]]、[[淮北|淮北]]、[[蚌埠|蚌埠]]一带，发现了从[[新石器时代|新石器时代]]到[[商代|商]][[周代|周]]时期的50多处古遗址，从这些遗址的面貌可以看出，当中原[[夏代|夏王朝]]的势力抵达[[宿县|宿县]]以西地区时，[[宿县|宿县]]以东尚属[[东夷|东夷]]部落的岳石文化势力范围。[[夏代|夏王朝]]确立政治统治地位不久，在[[河南|豫东]]一带活动的[[有虞氏|有虞氏]]即举族向东南迁移，并定居在[[江淮|江淮]]间的[[洪泽湖|洪泽湖]]、[[高邮湖|高邮湖]]一带，他们与陆续到达这一地区的岳石文化诸氏族汇聚，并与当地土著居民结合，成为[[淮夷|淮夷]]。

与[[安徽|皖北]]交界的[[河南省|河南省]][[鹿邑县|鹿邑县]]栾台遗址，是一处历经[[仰韶文化|仰韶文化]]、[[龙山文化|龙山文化]]、[[夏代|夏代]]、[[商代|商代]]、[[西周|西周]]直至[[战国|战国]]初期的聚落。从这个堆积厚达8米的重要遗址中，可以看出[[中原文化|中原文化]]和[[山东|山东]]古文化互相影响、势力消长的情况。遗址的堆积从最深处往上，底下第一层堆积属于[[大汶口文化|大汶口文化]]晚期，出土了[[鲁西平原|鲁西平原]]常见的陶[[鼎|鼎]]、[[鬶|鬶]]、[[盉|盉]]、[[壶|壶]]、[[罐|罐]]、高柄[[杯|杯]]等；往上第二期遗存，早期阶段的陶器尚有[[大汶口文化|大汶口文化]]的基本面貌，演变发展关系清楚，但晚期阶段则增加了[[中原文化|中原文化]]的因素，整体面貌与[[河南|豫东]][[龙山文化|龙山文化]]的造律台——王油坊类型较为一致；第三期又为岳石文化所影响，成为[[夏代|夏王朝]]时期[[东夷|东夷]]的属地；第四期的出土遗物则完全变成了[[商代|商代]]的风格。这个遗址，正是中原与东夷交往与征战的缩影。

[[中原|中原]][[夏代|夏王朝]]与岳石文化的这种交往，在[[淮河|淮河]]流域有广泛的影响，淮河南岸的[[城东湖|城东湖]]、[[瓦埠湖|瓦埠湖]]一带，也分布着类似栾台遗址文化面貌的一些遗址，其中[[霍丘县|霍丘县]]城南的红墩寺夏代文化堆积中，既有中原[[二里头文化|二里头文化]]的陶器，又有岳石文化的明显因素，表明不同文化的交流与融合。

岳石文化对江淮中部的影响南界便是在江淮分水岭南侧一带，即六安—合肥
—全椒一线<ref>張小雷《略论安徽地区的岳石文化遗存》</ref>。

[[淮河|淮河]]流域的[[夏代|夏代]]文化遗存，其复杂的内涵深刻地表现出夏代正处于新的社会结构正在建立与完善，新的生产力正在形成与发展的历史时期。不同文化的交流与融合，十分有利于农业和手工业生产技术的交流与推广，极大地促进了各地经济的发展。这种交流与融合，自人类社会开始时起就已存在，随着社会的进步与历史的发展，越来越显示出其积极的作用。

== 社会性质 ==
从大的方面看，社会形态仍然是[[龙山文化|龙山文化]]时期社会形态的延续，即属于[[城邦国家|城邦国家]]的发展时期，以家族所有制为主体的生产关系日益巩固，建立在家族基础之上的分散王权得到普遍加强和发展，军事力量不断扩大，以[[青铜器|青铜器]]为代表的新生产力的出现，暗示着新的生产关系正在酝酿之中。

== 与同时期相邻诸文化的关系 ==
*与[[双砣子二期文化|双砣子二期文化]]的关系：包括了三种文化因素，即当地土著文化、对当地[[龙山文化|龙山文化]]因素的继承和来自[[胶东|胶东]]的岳石文化因素。
*与[[夏家店下层文化|夏家店下层文化]]的关系：是一支分布于[[辽宁|辽西]]、[[内蒙古|内蒙古]]东南部和[[环渤海地区|京津冀北]]地区的早期青铜文化，其时代约当[[夏代|夏代]]和[[商代|早商]]，约略和岳石文化的时代相当。
*与[[二里头文化|二里头文化]]的关系：以岳石文化对二里头文化的影响为主，典型的岳石文化因素不仅深入到二里头文化分布区的腹地，并且在墓葬内出现，当有深层上的原因。相反，二里头文化对岳石文化的影响，在地域仅限于[[河南|豫东]]一带，似未进入[[山东|鲁中南]]等岳石文化腹地，在影响趋向上处于次要地位。
*关于[[郑州|郑州]]南关外下层遗存：1955年，在郑州旧城南关发掘了一处包含有[[战国|战国]]和[[商代|商代]]文化的遗址。其中商代地层又分为上、中、下三层，上、中层分别与二里冈上、下层相当，下层则是一种时代较早的新类型。
*与[[湖熟文化|湖熟文化]]的关系：岳石文化和湖熟文化有诸多相似之处，两者之间的传播与影响也显而易见。湖熟文化中的岳石文化因素比较明显，在岳石文化中，也有少量湖熟文化的因素。以岳石文化对湖熟文化的影响为主。
