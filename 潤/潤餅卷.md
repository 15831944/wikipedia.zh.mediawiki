'''潤餅卷'''，又稱'''潤餅'''、'''嫩餅菜'''、'''薄饼'''，臺灣中南部地區亦稱為春捲，是一種用薄饼卷食其他肉菜等的[[食品|食品]]。状似[[春卷|春卷]]，但捲好熟食后直接食用，無須[[油炸|油炸]]。润饼作为[[閩南|閩南]]饮食文化的一部分，流行於[[福建|福建]]南部、[[潮汕|潮汕]]、[[臺灣|臺灣]]和[[印度尼西亞|印度尼西亞]]、[[菲律賓|菲律賓]]等[[東南亞|東南亞]]地區。[[脍卷|越南春卷]]在形式上也比較類似于潤餅。

[[台閩地區|台閩地區]]一帶的家庭於[[尾牙|尾牙]]、[[春節|春節]]以及[[寒食|寒食]]、[[清明|清明]]時，會以潤餅皮來[[祭祖|祭祖]]，之後家族成員圍據一桌，食用者挑選自己喜愛菜色加上[[花生|花生]]粉與[[砂糖|砂糖]]粉以潤餅皮包裹後食用，是閩台家族[[清明|清明]]聚會的重要飲食。泉州各个县市主要在清明期间食用润饼，作为祭拜祖先之前家庭聚会的一种常见食品。臺灣北部地區家庭则多以在農曆年前時段包潤餅（[[同安|同安]]作法）又稱之為「家庭尾牙」；臺灣中南部地區（[[泉州三邑|泉州三邑]]做法）則是在清明期間<ref>{{Cite web|url=http://qz.fjsen.com/2015-02/06/content_15655259.htm|title=泉州十二道年味：阿母的润饼菜 除夕的团圆饭|accessdate=2016-11-12|author=林继学、彭思思、田米|date=2015-02-06|publisher=东南网}}</ref>。

== 製作 ==
「潤餅」餅皮是將麵粉加水，反覆攪打成質韌的溫麵團，抓在手中，在文火小平鍋上旋烙，拭成其薄如紙的餅皮，名曰「拭餅」。

臺灣地區：餡料通常用[[胡蘿蔔|胡蘿蔔]]、[[高麗菜|高麗菜]]、[[芹菜|芹菜]]、[[豆芽菜|豆芽菜]]、[[香腸|香腸]]、[[麵|油麵]]（南部特有）、[[豆干|豆干]]、[[豬肉|豬肉]]、煎[[雞蛋絲|雞蛋絲]]、[[蝦仁|蝦仁]]、[[皇帝豆|皇帝豆]]等，分盤上席。輔料方面，可有[[糖|糖]]拌[[花生|花生粉]]、[[海苔|海苔]]、[[蔥|蔥]]、[[蒜|蒜]]、[[芫荽|芫荽]]等。食時以薄餅皮包裹餡料和輔料，捲成圓筒狀，雙手握著就食，亦可再把餅皮表面稍微油煎兩面以增加口感和香氣。

閩南地區：餡料通常用[[胡蘿蔔|胡蘿蔔]]、[[高麗菜|高麗菜]]、[[荷蘭豆|荷蘭豆]]、[[牡蠣|石蚵]]、[[抱子甘藍|菜球]]、[[豆干|豆干]]、[[五花肉|五花肉]]、[[筍|桶筍]]、[[薑|嫩薑]]、[[蒜苗|蒜苗]]等切絲，依食材熟透時間不同下鍋，炒成大雜燴。食時以薄餅皮包裹餡料和輔料，捲成圓筒狀，雙手握著就食<ref>{{Cite web|url=http://news.ijjnews.com/system/2016/04/01/010947242.shtml|title=清明食俗 “十全十美”的润饼菜|accessdate=2016-11-12|author=陈子汉|date=2016-04-01|publisher=晋江新闻网}}</ref>。

== 起源 ==
遠在[[中國|中國]][[春秋戰國|春秋戰國]]開始<sup class="noprint">[[[Wikipedia:列明來源|來源請求]]]</sup>，就有在春天以[[五辛盤|五辛盤]][[祭祀|祭祀]]春神的禮俗，[[五辛|五辛]]盤內容為[[大蔥|大蔥]]、[[小蒜|小蒜]]、[[韭菜|韭菜]]、[[香菜|香菜]]、[[芸苔|芸苔]]，再[[祭祀|祭祀]]春神後食用這五辛，以求開五臟、去伏氣的保健效果，但因[[五辛|五辛]]味道辛辣，單吃難以入口，於是後來逐漸發展以麵餅包裹[[五辛|五辛]]成為春餅來食用，春餅也就成了潤餅和春卷的前身。自[[東晉|東晉]]開始中原數度戰亂使[[中原|中原]]人士大舉南遷閩避難，以致今日台閩一帶吃[[潤餅|潤餅]]還保有古代春餅之俗，桌面無論菜色多少，內容一定必須具備[[五辛|五辛]]在內，僅芸苔以[[浒苔|浒苔]]代替。吃[[潤餅|潤餅]]在閩台一帶是家族一年中重要的聚會禮節，大多為家族成員食用，一般不會用來招待外人。但近來也出現潤餅專賣店、賣潤餅專車，尤以[[夜市|夜市]]、菜市場等地最常見。

== 傳說 ==
相傳潤餅由[[明代|明代]][[福建|福建]][[泉州府|泉州]][[同安縣|同安縣]][[金門島|金門島]]才子[[蔡復一|蔡復一]]的夫人李氏首創，相傳是蔡夫人因為蔡復一公務繁忙，无暇用餐，特意為他製作，可以邊辦公邊享用<ref>{{Cite web|url=http://zz.mnw.cn/news/1140436.html|title=清明吃润饼你知道来历吗？ 美味润饼菜咋做？|accessdate=2016-11-12|author=闽南网|date=2016-04-01|publisher=漳州之窗}}</ref>。

故事版本眾多，姑取其中最盛行者如下<ref>{{Cite web|url=https://news.tvbs.com.tw/world/717401|title=照過來！清明到呷潤餅　別顧美食不懂由來}}</ref>。

民間流傳，蔡復一是明朝[[萬曆|萬曆]]年間[[進士|進士]]，但身體[[殘障|殘缺]]得很厲害，獨眼、跛腳又駝背，但是他熟讀[[四書五經|四書五經]]，不僅能作[[詩|詩]]、[[賦|賦]]、[[八股文|八股文]]；才華橫溢，兩手能同時寫字，在[[殿試|殿試]]上大出鋒頭，皇帝原本想點其為一甲（前三名），但蔡態度狂傲，皇帝又改其為二甲。不過放榜之後，皇帝憐才之意又起，常對蔡加以賞賜，四個奸臣於是過來消遣他，欲挫其銳氣。

四奸臣前來，第一人取笑他是[[獨眼龍|獨眼龍]]，蔡說：“一眼觀天下。”第二人取笑他是瘸子，蔡回應：“一[[腳|腳]]登龍門。”第三人取笑他是駝子，蔡回應：“一背藏千卷。”最後一人又質疑他兩手同時寫字是假的，蔡就回答“兩手作萬言。”立成一首詩。四個奸臣無趣而返，於是告訴[[皇帝|皇帝]]，蔡復一說天下文書都抄寫得成，「兩手作萬言」，皇帝很生氣，覺得蔡太狂妄了，故要求蔡一個月內抄寫[[朝廷|朝廷]]歷年來的邊防文書，如不從，將以「欺君」論處。

為了完成任務，蔡只好廢寢忘食，晝夜雙手齊書，幾乎不吃飯，蔡夫人看在眼裡，急在心裡。有一天，她想出了一個妙法：麵粉加水，攪成糊狀，在熱鍋上輕輕一抹，做成一張張薄薄的麵餅皮，再把各種菜切細，炒成燴菜，然后用餅皮把燴菜卷成圓筒狀。每天用餐時，蔡夫人雙手捧著“薄餅”餵正在抄寫的蔡復一，這樣既不影響工作，又不耽擱用餐。在夫人的照料和協助下，蔡復一如期完成使命，他抄的文件足足放滿了九個櫃子。

皇上認為蔡是個能吃苦、有智謀的人，故大力提拔蔡，蔡後來作到[[湖南|湖南]]、[[湖北|湖北]]、[[貴州|貴州]]、[[雲南|雲南]]、[[廣西|廣西]]五省[[經略|經略]]。從此，蔡夫人做餅的故事在當地傳為美談，因“潤餅”是蔡夫人所創，所以民間又有“'''美人薄餅'''”或“'''夫人薄餅'''”的說法。
