==制备与结构==
2007年三位化学家在《科学》杂志上发表文章，报道他们用过量的金属[[钾|钾]]在[[甲苯|甲苯]]中还原Mg<sup>II</sup>的[[碘|碘]]配合物，制得了两种稳定的Mg(I)化合物，两种物质的化学式都可以写成L<sub>2</sub>Mg<sub>2</sub>，其中L代表一个大[[阴离子|阴离子]][[配体|配体]]，该反应使用的大阴离子配体如下：<ref name = "Green"/>。

* 胍基配体（priso），[(Ar)NC(NPr<sup>i</sup><sub>2</sub></sub>)N(Ar)]<sup>–</sup>（其中 Ar = 2,6-双（异丙基）苯基、Pr<sup>i</sup> = 异丙基），其形成的低价镁化合物为无色。
* {{le|酮亚胺|NacNac}}配体（nacnac），{[(Ar)NC(Me)]<sub>2</sub>CH}<sup>−</sup>,- （其中 Ar = 2,6-双（异丙基）苯基、Me = 甲基），其形成的低价镁化合物为黄色固体。

[[X射线晶体学|X射线晶体学]]研究发现，其中Mg-Mg键的键长分别是285.1pm和284.6pm。<ref name = "Green"/>理论研究表明他们实际上是离子化合物，化学式为Mg<sub>2</sub><sup>2+</sup>(L<sup>−</sup>)<sub>2</sub>。<ref name = "Green"/>[[碱土金属|2族元素]]形成的Mg<sub>2</sub><sup>2+</sup>离子与[[12族元素|12族元素]]形成的Hg<sub>2</sub><sup>2+</sup>离子（存在于[[氯化亚汞|氯化亚汞]]等物质中）、Cd<sub>2</sub><sup>2+</sup>离子（存在于[[四氯铝酸镉(I)|四氯铝酸镉(I)]]中）类似。

==性质==
目前制得的两种低价镁化合物容易与空气和水反应，但是有着较好的热稳定性，其热分解温度分别为170℃和300℃<ref name = "Green"/>。
