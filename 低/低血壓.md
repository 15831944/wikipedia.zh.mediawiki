'''低血壓'''在[[生理學|生理學]]及[[醫學|醫學]]上是指[[血壓|血壓]]不正常地低。比起病症，低血壓較適合稱作一種生理狀況。目前[[世界衛生組織|世界衛生組織]]沒有訂定低血壓的標準，但如果一般成人[[肱動脈|肱動脈]][[血壓|血壓]]小於（90/60mmHg）時，可能會被判斷有低血壓。縱然沒有明確指明，但它一般都與[[休克|休克]]有關聯。

== 自然生理學 ==
[[血壓|血壓]]是由身體的[[自主神經系統|自主神經系統]]不斷的調節，透過使用精細的感應器、[[神經|神經]]及[[激素|激素]]來平衡傾向

== 成因及結構 ==
[[低血容積|低血容積]]是最普遍導致低血壓的原因。這可以是由於[[出血|出血]]或失血、飢餓時液體攝取量不足、或是過度失去水份，如[[痢疾|痢疾]]或[[嘔吐|嘔吐]]等。低血容積經常會因使用[[利尿劑|利尿劑]]等而引發，不同的[[藥物|藥物]]使用亦會引起此情況。

儘管在正常的血容積下，因嚴重[[心臟衰竭|充血性心臟衰竭]]、大型的[[心肌梗死|心肌梗死]]或[[心動過緩|心動過緩]]而造成[[心輸出量|心輸出量]]的減少，同樣會引起低血壓，並可以急劇發展成[[休克|休克]]。[[心律不齊|心律不齊]]亦同樣經常會造成低血壓。[[乙型交感神經阻斷劑|乙型交感神經阻斷劑]]在使用後會令[[心跳|心跳]]下降及降低心肌供血的能力，亦是造成低血壓的另一成因。

因[[腦部|腦部]]或[[脊髓|脊髓]]受損或[[自律神經失調|自律神經失調]]引起的[[交感神經|交感神經]]的輸出下降或[[副交感神經|副交感神經]]活躍化，這會導致過度的[[血管舒張|血管舒張]]或不足的血管阻力，造成低血壓。其他引起過度血管舒張的原因有[[敗血病|敗血病]]、[[酸中毒|酸中毒]]或使用如[[鈣離子阻斷劑|鈣離子阻斷劑]]或[[血管緊張素轉化酶抑制劑|血管緊張素轉化酶抑制劑]]等藥物。很多麻醉藥及技術，包括[[脊椎麻醉|脊椎麻醉]]及大部份的[[吸入式麻醉|吸入式麻醉]]都會造成明顯的血管舒張。

==综合症==
[[姿位性低血壓|體位型低血壓]]，又稱「[[姿勢性低血壓|姿勢性低血壓]]」，是一種普遍的低血壓型態。它會在改變身體位置時，尤其是從坐下或躺下的姿勢改為直立的姿勢時出現。它一般都是暫時性的低血壓，是因[[自主神經系統|自主神經系統]]延誤了正常的調節功能。在[[低血容積|低血容積]]時會出現此情況。而當使用不同[[藥物|藥物]]時，尤其是上述那些令血壓降低的藥物或是精神科藥物，如[[抗抑鬱藥|抗抑鬱藥]]等，都會有此副作用。在躺下、坐下或站立時簡單量度[[血壓|血壓]]及[[心跳|心跳]]都能確定體位型低血壓的情況。

[[神經心源性暈厥|神經心源性暈厥]]是[[自律神經失調|自律神經失調]]的一種，尤以在直立時會出現不正常的血壓下降。[[傾斜試驗|傾斜試驗]]可以用作測試此情況。神經心源性暈厥與[[血管迷走性暈厥|血管迷走性暈厥]]有所關聯，因兩都是由於[[副交感神經|副交感神經]]的[[迷走神經|迷走神經]]增加活動所引起。

另一種較少見的是[[餐後低血壓|餐後低血壓]]，在用餐後的30-75分鐘出現。由於在[[消化|消化]]時大量的[[血液|血液]]會被送往[[小腸|小腸]]，身體必須增加心跳及[[血管舒張|血管舒張]]來抵消局部的[[血容積|血容積]]改變。一般相信這種形式的低血壓是因失調或自主神經系統老化未能作出適當的調節而起的。

== 指標 ==
對於大部份的人來說，健康的[[血壓|血壓]]介乎於115/75mmHg。血壓少量的下降，儘管少如20mmHg，都會有短暫的低血壓。

== 徵狀 ==
低血壓的主要徵狀是[[頭昏及全身無力|頭昏及全身無力]]。如果[[血壓|血壓]]低至某個程度，可能會出現[[昏厥|昏厥]]，甚至[[癲癇|癲癇]]等情況。低血壓一般都會有以下徵狀，但多是與低血壓的成因有關，而非低血壓本身的影響：
*[[胸痛|胸痛]]
*[[呼吸困難|呼吸困難]]
*[[心律不齊|心律不齊]]
*[[發熱|發熱]]高於38.3℃或101℉
*[[頭疼|頭疼]]
*[[頸部僵硬|頸部僵硬]]
*嚴重的[[背痛|背痛]]
*[[咳嗽|咳嗽]]帶有[[痰|痰]]
*長期的[[痢疾|痢疾]]或[[嘔吐|嘔吐]]
*[[吞嚥困難|吞嚥困難]]
*[[排尿困難|排尿困難]]
*[[尿|尿液]]帶有惡臭
*使用[[藥物|藥物]]後的不良反應
*激烈或危害生命的[[過敏|過敏]]反應
*眩暈
*癲癇
*[[昏迷|昏迷]]
*[[疲勞|疲勞]]
