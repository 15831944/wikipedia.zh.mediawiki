'''低音巴松管'''（英文: Contrabassoon, double bassoon）是[[巴松管|巴松管]]的變形樂器，屬於雙簧樂器，音域較巴松管低1個[[八度|八度]]，是交響樂隊木管樂器組中音域最低的樂器。演奏技巧除了少部份的指法外，大致和巴松管相近。在現代的交響樂團中，一般都有至少一位巴松管手同時演奏巴松管及低音巴松管。而演奏現代作品或者需運用較大樂隊規模的樂曲（如[[古斯塔夫·馬勒|馬勒]]、[[理察·史特勞斯|理察·史特勞斯]]、[[蕭士塔高維契|蕭士塔高維契]]等）時，更需要一位獨立的低音巴松管手。

==和巴松管的分別==
低音巴松管的簧片都比一般巴松管為大及長，長度約為65～75毫米。指法上，特高音域的指法會和巴松管所用的指法有所出入。而各指孔間的隔距亦比巴松管為闊。樂器的總長度約是其高度的兩倍半。由如它巨大的形態及重量，在樂器的底部裝有金屬腳棒支撐管體，並如銅管樂器一樣, 裝有排水鍵, 能將積存在樂器內的凝結水氣排走。
大部份的低音巴松管在出廠都是預先裝嵌好，並不能如其他木管樂器般可以隨意地拆開或結合。只有吹咀及部份型號的喇叭口能拆開。

== 音域 ==
低音巴松管的音域大約為3個八度，由 B{{music|b}}<sub>0</sub> (鋼琴的最低A音高半度) 起計。個別製造的低音巴松管能吹出A{{music|b}}<sub>0</sub>，比鋼琴的最低A音還要低半度。在樂隊中是繼大號之後，第二能演奏最低音樂器。最高音可以到達 A<sub>3</sub> (比中音 C 低小三度)，不過一般作曲家很少會運用上 D<sub>3</sub> 以上的音。

== 記譜 ==
低音巴松管採用移調記譜的方法，較常採用低音譜號，間中亦會採用次中音譜號但不常見。實際發音比記譜音低純八度。
