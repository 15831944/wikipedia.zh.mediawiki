'''慶良間諸島國立公園'''（假名：{{lang|ja|けらましょとう こくりつこうえん}}），是位在[[日本|日本]][[沖繩縣|沖繩縣]][[慶良間諸島|慶良間諸島]]及其周邊海域之一部的[[日本國立公園|國立公園]]。

== 概要 ==
由[[座間味島|座間味島]]、[[渡嘉敷島|渡嘉敷島]]兩島為主的慶良間諸島及周圍的[[珊瑚礁|珊瑚礁]]海域所構成。

擴張後的面積是陸域3,520ha、海域約82,185ha（之內的[[海域公園|海域公園]]地區約8,290ha），海域公園超越[[西表石垣國立公園|西表石垣國立公園]]，面積是國內最大。

== 年表 ==
* 1978年12月9日 - [[慶良間諸島|慶良間諸島]]編入[[沖繩海岸國定公園|沖繩海岸國定公園]]。
* 2014年3月5日 - 沖繩海岸國定公園削除慶良間地域，新指定為慶良間諸島國立公園。

== 面積 ==
* 陸域 - 3,520ha
* 海域 - 82,185ha
