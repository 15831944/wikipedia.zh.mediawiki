'''姚晨难民事件'''或称'''微博难民事件'''<ref>{{Cite web |url=https://baijiahao.baidu.com/s?id=1571449859184384 |title=微博难民事件始末，网友立场统一，姚晨被抵制 |accessdate=2017-09-06 |author=圆周 |date=2017-06-30 |work= 百度百家号|publisher=|language=Zh-hans }}</ref>是发生于[[2017年6月|2017年6月]]的网络事件，因传言女星[[姚晨|姚晨]]呼吁中国接受[[难民|难民]]而引发，一些中国[[网民|网民]]就此或借此表达反对接受[[难民|难民]]（[[移民|移民]]）的观点<ref>{{Cite web |url=https://baijiahao.baidu.com/s?id=1571398269982514 |title=姚晨难民事件，更多是群众借这件事向社会发出声音 |accessdate=2017-09-06 |author= |date=2017-06-28|work=百度百家号 |author=泛娱短评|publisher=|language=Zh-hans }}</ref>。

== 背景 ==
[[中华人民共和国|中华人民共和国]]（以下简称中国）是人口大国，[[中华人民共和国人口|人口为世界之最]]，但自1970年代起，中国政府推行[[中华人民共和国计划生育|计划生育政策]]，普通民众尤其是主体民族——[[汉族|汉族]]的[[生育权|生育自由]]被剥夺。同时，中国一直未被视为传统的[[移民国家|移民国家]]，[[归化|归化]][[中华人民共和国国籍|中国国籍]]极为困难。1979年[[1979年中越战争|中越開戰]]后，大批[[越南华僑|越南华僑]]遭越南政府驱逐出境，直接導致八十年代共有28万越籍难民逃亡中国。此后数十年內，中国陸續接受了共30多万难民，大部分即是此批被政府称为[[印支难民|印支难民]]的華僑難胞<ref name="dw1">{{Cite web |url=http://www.dw.com/zh/反难民情绪激荡中国-亲善大使姚晨受辱骂/a-39418053?&zhongwen=simp |title=反难民情绪激荡中国 亲善大使姚晨受辱骂 |accessdate=2017-09-06 |author=李鱼 |date=2017-06-26 |work= |publisher=在线报导|language=Zh-hans }}</ref>。1990年代，随着[[朝鲜|朝鲜]]经济状况的恶化，[[脱北者|脱北者]]进入[[中国东北地区|中国东北地区]]，给当地的治安、经济带来巨大的压力。中国政府遣返脱北者的举动，亦受到国际社会的强烈譴責。

2010年底，随着[[阿拉伯之春|阿拉伯之春]]带来的[[中东难民|中东难民]]问题，引发旷日持久的[[欧洲移民危机|欧洲移民危机]]，中东难民问题成为世界关注的焦点。2016年，全球[[难民|难民]]人数达到6千余万人，为历史最高。同年，浙江省[[义乌市|义乌市]]（人口约百万，为外贸、小商品集散地）发放的近万份[[中华人民共和国外国人居留许可|短期居留许可]]中，超过四成发给中东战乱、动荡国家的公民<ref name="dw1"/>。

2016年5月，[[国际特赦組織|国际特赦組織]]发布“难民受欢迎指数”，称根据调查，认为中国民众是最欢迎难民的人，愿意接纳逃离战争和迫害的人士，超过半数中国受访者“愿意在自己家中接收难民”。中国之外，最欢迎难民的国家是[[德国|德国]]、[[英国|英国]]。此项调查结果被国际特赦用以指责有关国家政府在难民问题上的表现。相关报道转引[[路透社|路透社]]报道，大赫国际秘书长{{link-en|萨里尔·谢蒂|Salil Shetty|萨里尔·谢蒂}}称：“国民对接收难民表示欢迎，政府却采取不人道的应对，偏离了[[民意|民意]]”<ref name="a1"/>。国际特赦东亚地区主任林伟（Nicholas Bequelin）在声明中表示，中国民众一般倾向于提供政府希望他们给出的答案，但无论如何这非常令人鼓舞，中国政府应该在难民问题上做更多<ref name="a2"/>。

中国方面对国际特赦的调查结果及结论有质疑。《[[环球时报|环球时报]]》下属的[[环球网|环球网]]举办的在线投票中，90.3%的投票者表示“不愿意在自己家中接收难民”，79.6%的投票者反对“在自己所在城市，或与难民为邻”<ref name="a1">{{Cite web |url=http://new.qq.com/cmsn/20160520013125 |title=“中国人最欢迎难民”调查遭质疑 九成网友不愿接收难民 |accessdate=2018-03-25 |author=记者：白云怡、甄翔 |date=2016-05-20 |work= |publisher=腾讯网，来源：环球时报|language=Zh-hans }}</ref>。美国《[[大西洋月刊|大西洋月刊]]》所属网络杂志《石英（Quartz）》的文章则指出，国际特赦调查并不可靠。一是中国受访者来自大城市，二是翻译的误导。中国受访者可能认为，调查中所指的难民是本国遭受[[自然灾害|自然灾害]]的[[灾民|灾民]]，而不是外籍难民。中国受访者表达的可能是在家中接待本国灾民的意愿，而不是外籍难民。当调查者在调查中明确指出“别国难民”时，中国受访者的赞同度在全部27个国家中排名第19位。调查中的另一个问题“你的国家政府是不是应该做更多来帮助战争难民和受迫害难民？”，中国受访问者回答“是”的比例为全部调查国家第一位<ref name="a2">{{Cite web |url=http://www.guancha.cn/society/2016_05_20_360998.shtml |title=大赦国际民调报告称中国民众最欢迎难民 网友质疑 |accessdate=2018-03-25 |author= |date=2016-05-20 |work= |publisher=观察者网站|language=Zh-hans }}</ref>。

== 当事人 ==

[[姚晨|姚晨]]是中国知名女星，有微博女王之称<ref>{{Cite web |url=https://www.nownews.com/news/20170627/2580209 |title=娛樂報報／姚晨微博淪陷　從微博女王到眾矢之的她經歷啥 |accessdate=2017-09-06 |author= |date=2017-06-27  |work= |publisher=今日新闻网站|language=Zh-hant }}</ref>。2010年，姚晨被[[联合国难民署|联合国难民署]]任命为中国区代言人。美国女星、联合国难民署亲善大使——[[安吉丽娜·朱莉|安吉丽娜·朱莉]]亦是她的效仿对象。她曾表述“我不认为有了孩子会影响工作，而且到时候我还能带着孩子一起去做难民署的工作呢，多好啊”，以此表达对难民署工作的认可<ref>{{Cite web |url=http://hb.qq.com/a/20100621/000908.htm|title=出任联合国难民署中国区代言人 姚晨：想带孩子去探望难民|accessdate=2017-09-06 |author=记者：施培琦（摄）、林艳雯 |date=2010-06-21  |work= |publisher=腾讯网，来源：上海青年报|language=Zh-hans }}</ref>。2011年《[[南方周末|南方周末]]》刊文：《姚晨：也许有一天，我们也可能沦为难民》。2013年6月20日，姚晨升任联合国难民署中国区亲善大使<ref name="fhw1"/>。

== 事件经过 ==

2017年6月7日，姚晨续约[[联合国难民署|联合国难民署]]中国区亲善大使两年。当天刚刚抵达北京，进行首次对华访问的联合国难民署负责人[[菲利普·格兰迪|菲利普·格兰迪]]出席了姚晨与难民署的续约仪式<ref>{{Cite web |url=http://news.163.com/17/0607/23/CMC7AP8N000187VI.html |title=姚晨续约联合国难民署中国区亲善大使两年|accessdate=2017-09-08 |author= |date=2017-06-07 |work= |publisher=网易，来源：中青在线(北京)|language=Zh-hans }}</ref>。6月间，“中国接收了31万难民”的讯息在微博引发关注<ref name="dw1"/>。

[[6月20日|6月20日]]，[[世界难民日|世界难民日]]期间，联合国难民署在北京举办活动，向难民[[致敬|致敬]]，姚晨亦参与活动。22时49分，姚晨在[[新浪微博|新浪微博]]转发联合国难民署的文章。姚晨等公众人物的支持引发网络争议，引起网友对中国是否接受难民进行讨论。同日，微博网友发起“是否愿意接受难民”的投票活动。至22日，仅2%的网友投票支持接受难民。在普遍反对接受难民情绪之下，此前姚晨与难民相关的报道被挖出，认为她呼吁、鼓吹中国接受难民的舆论观点形成<ref name="apple1">{{Cite web |url=http://hk.apple.nextmedia.com/realtime/china/20170622/56862622 |title= 【躺着中槍！】中國收容30萬中東難民？ 女星姚晨被網民鬧爆|accessdate=2017-09-06 |author= |date=2017-06-22 |work= |publisher=苹果日报网站|language=Zh-hant }}</ref>。网民表达了对姚晨的反感，极端者对其进行了攻击。同时，亦有评论认为她呼吁中国接受难民并非“罪不可恕”<ref>{{Cite web |url=http://dy.163.com/v2/article/detail/CNKSTMUK0517BDKK.html |title=公知姚晨再次被网友围攻，接收难民真的罪不可恕吗？ |accessdate=2017-09-06 |author= |date=2017-06-23 |work= |publisher=网易|language=Zh-hans }}</ref>。

23日，在[[黎巴嫩|黎巴嫩]][[贝鲁特|贝鲁特]]访问的[[中华人民共和国外交部部长|中国外交部长]][[王毅|王毅]]与黎巴嫩外交与侨民部长{{link-en|纪伯伦·巴西勒|Gebran Bassil|}}共同会见记者，应询代表中国政府就中东难民问题表态<ref>{{Cite web |url=http://news.163.com/17/0625/07/CNOSSDMM0001899N.html |title=王毅就难民问题阐明中方立场：难民不是移民 |accessdate=2017-09-06 |author= |date=2017-06-25 |work= |publisher=网易，来源：央视|language=Zh-hans }}</ref>。王毅主张加快热点问题解决，缓解难民危机漫延。难民不是移民，应为难民创造重返家园的条件，通过发展营造难民回归的环境<ref>{{Cite web |url=http://www.fmprc.gov.cn/web/wjbzhd/t1472830.shtml |title=王毅与黎巴嫩外交部长巴西勒举行会谈 |accessdate=2017-09-06 |author= |date=2017-06-23 |work= |publisher=中华人民共和国外交部网站|language=Zh-hans }}</ref>。

25日，姚晨转发王毅关于难民问题的文章，认同他的观点。26日，姚晨发文表示：从未在任何场合表达过“呼吁中国接受难民”的观点<ref name="fhw1">{{Cite web |url=http://news.ifeng.com/a/20170626/51323354_0.shtml |title=呼吁中国接受难民？姚晨今日发声明：这是谣言 |accessdate=2017-09-06 |author= |date=2017-06-26 |work= |publisher=凤凰网|language=Zh-hans }}</ref>。其后，[[袁弘|袁弘]]、[[宋佳|宋佳]]等知名演员和新浪微博大V{{noteTag|1=新浪微博大V指新浪微博上获得个人认证的用户，通常是明星、企业家、政治家等[[知名人士|知名人士]]。参见：[[新浪微博#认证制度|新浪微博#认证制度]]。}}纷纷转发姚晨26日的微博，以示支持。对于转发，姚晨给予了感谢的回复。其中，一位名为“李海鹏”的微博大V发言“从头到尾莫名其妙”，姚晨回复“欲加之罪”。而袁弘等人及微博大V的转发依旧被网友批评<ref>{{Cite web |url=http://www.sohu.com/a/152351783_650027 |title=袁弘站错队，姚晨“难民”言论洗不清了！|accessdate=2017-09-08 |author= |date=2017-06-27 |work= |publisher=搜狐网|language=Zh-hans }}</ref>。同时，姚晨在26日的微博中称“时常庆幸自己生长在一个和平安定的国家”。海疆在线网站认为，这个评价与此前她在[[2014年昆明火车站暴力恐怖袭击事件|2014年昆明火车站暴力恐怖袭击事件]]时对中国的评价——“恶之花绽放的土地”前后矛盾，故该声明不值得采信<ref>{{Cite web |url=http://www.haijiangzx.com/2017/0627/1896838.shtml |title=姚晨，你的觉悟连难民都不如！ |accessdate=2018-01-01 |author=林爱玥 |date=2017-06-27 |work= |publisher=海疆在线网站|language=Zh-hans }}{{需要更好来源}}</ref>。
