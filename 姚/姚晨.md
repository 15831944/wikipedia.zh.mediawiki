'''姚晨'''（{{bd|1979年|10月5日||}}），[[中国大陆|中国大陆]]女[[演員|演員]]，籍贯[[福建|福建]][[泉州|泉州]][[石狮市|石狮市]]。代表作《[[潜伏_(电视剧)|潜伏]]》获得巨大反响，2013年6月20日，升任[[联合国难民署|联合国难民署]]中国区亲善大使。<ref>{{cite web |url=http://ent.sina.com.cn/s/m/2013-06-20/09523946241.shtml |title=姚晨升任联合国难民署亲善大使 |language=zh-hans}}</ref>2014年，著名刊物[[美國|美國]]《[[时代_(杂志)|时代]]》雜誌將姚晨列为「百大最有影響力人物」之一。<ref>{{cite web |url=http://www.bbc.co.uk/zhongwen/simp/china/2014/04/140425_time100_2014.shtml |title=姚晨、习近平并列时代周刊全球百强榜 |accessdate=2014-4 |publisher=BBC |work= |language=zh-hans}}</ref>2018年10月5日，其主演的电影《找到你》在中国大陆上映。<ref>{{Cite web|title=《找到你》首映 姚晨马伊琍分享母爱故事-新华网|url=http://www.xinhuanet.com/ent/2018-09/18/c_1123445486.htm|work=www.xinhuanet.com|accessdate=2019-04-02}}</ref>2019年，主演家庭伦理剧《都挺好》播出。<ref>{{Cite web|title=《都挺好》姚晨倪大红携手“寻回心中的家”-新华网|url=http://www.xinhuanet.com/ent/2019-02/26/c_1124162793.htm|work=www.xinhuanet.com|accessdate=2019-04-02}}</ref>

== 生平 ==
1993年到1997年，姚晨就读于[[北京舞蹈学院|北京舞蹈学院]]附中(民间舞专业)。1999年到2003年，就读于[[北京电影学院|北京电影学院]]表演系（99级本科），与[[叶静|叶静]]、[[凌潇肃|凌潇肃]]、[[于波|于波]]为同班同学。

她以2005年[[尚敬_(导演)|尚敬]]导演的情景喜剧《[[武林外传|武林外传]]》中[[郭芙蓉|郭芙蓉]]一角成名。2008年她与[[孙红雷|孙红雷]]合作的谍战电视剧《[[潜伏_(电视剧)|潜伏]]》获巨大成功，并为她赢得[[华鼎奖|华鼎奖]]最佳电视剧女主角。

2013年產子後，於年尾復出拍攝投資1.3億的時裝律政情感大戲《[[离婚律师|離婚律師]]》。2014年4月殺青後，8月4日於中國內地四大衛視首播，收視率跟網上點擊率、話題量都居高不下，令其人氣再度急升，成為她的代表作。

== 个人生活 ==
2004年，姚晨与相恋多年，同为演员的大学同学[[凌潇肃|凌潇肃]]结婚。2011年1月28日傍晚，姚晨和凌潇肃发布离婚声明。

2012年8月15日，摄影师[[曹郁|曹郁]]在自己的生日宴会上向姚晨求婚成功。<ref>{{cite news|publisher=新華網|title=曹鬱生日趴向女友姚晨求婚 女方感動不已流淚答應 |url=http://news.xinhuanet.com/ent/2012-08/16/c_123589981.htm|accessdate=2012-08-16}}</ref> 2012年11月17日，两人于[[新西兰|新西兰]][[皇后镇_(新西兰)|皇后镇]]举行婚礼。<ref>{{cite news|publisher=时光网|title=姚晨曹郁新西兰完婚|url=http://news.mtime.com/2012/11/18/1501553.html|accessdate=2012-11-18}}</ref>2013年7月15日产下一子“小土豆”<ref >{{cite web|url=http://news.xinhuanet.com/overseas/2013-07/19/c_125033253.htm | language= | title=姚晨儿子照片首曝光 老公曹郁称：嘴大像妈妈/图 |publisher=光明网 |date=2013年7月19日 |accessdate= }}</ref>。2016年11月9日生下一女“小茉莉”<ref>{{Cite web|url=http://d.youth.cn/shrgch/201611/t20161110_8833182.htm|title=姚晨昨日二胎产女 儿女成双凑成“好”|accessdate=2017-02-26|author=|date=2016-11-10|publisher=中国青年网}}</ref>。

=== 离婚事件 ===
向来在中国大陆的观众视为“模范夫妻”的姚晨、凌潇肃，传出两人不和的信息。2011年1月28日，互联网上盛传姚晨、凌潇肃两人离婚的消息，称两人正在西安办理离婚手续。

2014年9月17日，网友“[[巨春雷|巨春雷]]”发表长篇文章《姚晨和凌潇肃那段被歪曲的往事》，爆料姚晨在婚内出轨4次，继而导致二人离婚。<ref>{{cite news|publisher=凤凰网|title=姚晨疑回应多次出轨传闻：满纸荒唐 物是人非 |url=http://ent.ifeng.com/a/20140917/40310710_0.shtml|accessdate= }} </ref>

== 争议事件 ==

=== 恶之花绽放的土地 ===

[[2014年昆明火车站暴力恐怖袭击事件|2014年昆明火车站暴力恐怖袭击事件]]发生后的次日（3月2日）凌晨2时11分，姚晨在[[新浪微博|新浪微博]]发文：“'''难过的无以言表，恶之花绽放的土地，愿真相早日到来，告慰无辜的亡灵...'''”不久，微博被删除。此微博是被新浪微博管理者删除，或姚晨删除，无从得知{{efn|name=ab1|1=新浪微博发表后“被删除”。新浪管理者删除，发表者删除是两大删除原因。}}。10时38分，姚晨再度发文：“默哀”<ref name="gczw1">{{Cite web |url=http://www.guancha.cn/FaZhi/2014_03_02_210038.shtml |title=昆明火车站暴力恐怖案件后大V微博评论 姚晨称是恶之花绽放的土地 |accessdate=2017-09-06 |author= |date=2014-03-02 |work= |publisher=观察者网|language=Zh-hans }}</ref>。

当时，中国普通网民群情激愤，要求严惩恐怖袭击的凶手。而[[2014年昆明火车站暴力恐怖袭击事件#西方媒体报道争议|境外媒体，主要是西方媒体的报道则引发了争议]]，他们未将事件定性为恐怖袭击。与西方媒体类似，部分[[公知|公知]]和新浪微博大V集体<ref>{{Cite web |url=http://www.ntdtv.com/xtr/b5/2014/03/02/a1072309.html |title=昆明砍人事件 大陸被刪微博集錦 |accessdate=2017-09-06 |author= |date=2014-03-02 |work= |publisher=新唐人电视台|language=Zh-hant }}</ref>表达了对凶手的同情和理解，指责[[中华人民共和国政府|中国政府]]的“暴政”是造成恐怖袭击的原因，更有大V为被击毙的凶手们祷告，呼吁网民反对政府击毙暴徒<ref name="gczw1"/>。一批同情凶手的微博在发表后即“被（新浪）删除”{{efn|name=ab1}}，姚晨被删除的微博亦被认为属于此类<ref>{{Cite web |url=http://hk.aboluowang.com/2014/0303/376669.html |title=大V評論昆明血案被刪集 看姚晨被刪是戳了誰的心肝！|accessdate=2017-09-06 |author= |date=2014-03-02 |work= |publisher=阿波罗新闻网|language=Zh-hant }}</ref>。

“恶之花绽放的土地，原真相早日到来，告慰无辜的亡灵”语句本身有歧义{{efn|1=“难过的无以言表，恶之花绽放的土地，愿真相早日到来，告慰无辜的亡灵”，可以有两种理解：<br/>一是，“难过的无以言表，恶之花（中国政府暴政<ref name="a2"/>）绽放的土地，愿真相（反抗中国政府暴政和对维吾尔族民族压迫的行动<ref name="a1"/>）早日到来，告慰无辜的亡灵（反抗暴政、民族压迫的抵抗者）”<br/>二是，“难过的无以言表，恶之花（恐怖袭击）绽放的土地，愿真相（恐怖袭击内情）早日到来，告慰无辜的亡灵（袭击死难者）”}}。舆论将姚晨说的“愿真相早日到来”，理解为她否认昆明事件是恐怖袭击，而是另有真相。认为她与西方媒体观点一致，将恐怖袭击事件定性为新疆维吾尔族民众因中国政府暴政、民族压迫而产生的反抗。3月4日，新疆地方警察机关在新浪微博的帐号以“这位女星”指代，对她质疑事件性质，发出不点名的批评<ref name="a1"/>。

在中文中，“恶之花”本身并非特有名词，有多种用法。其中，或用以形容中国共产党及中国政府{{efn|name=ab2|}}行政的阴暗面<ref name="a2"/>，即所谓“暴政”。“恶之花绽放的土地”被认为是指昆明<ref name="a2">{{Cite web |url=http://www.mingbaoxinwen.com/ylzx/2916.html |title=姚晨恶之花事件是怎么回事 姚晨为什么叫恶之花 |accessdate=2019-04-06 |author= |date=2018-01-12 |work=责任编辑：阿凡 |publisher=明报新闻网|language=Zh-hans }}</ref>、中国<ref name="gczw1"/>。2017年，在[[姚晨难民事件|难民事件]]中饱受舆论批评的姚晨，称“时常庆幸自己生长在一个和平安定的国家”。她对中国前后相互矛盾的评价，依旧被批评<ref>{{Cite web |url=http://www.haijiangzx.com/2017/0627/1896838.shtml |title=姚晨，你的觉悟连难民都不如！ |accessdate=2018-01-01 |author=林爱玥 |date=2017-06-27 |work= |publisher=海疆在线网站|language=Zh-hans }}</ref>。2019年上半年，姚晨凭借《[[都挺好|都挺好]]》中苏明玉一角再度翻红。[[人民网|人民网]]对她的采访中提及，她对剧中角色找苏明玉道歉一幕“感到非常动容”。网友则表示没有等到她为“恶之花”的道歉。更称之为“污点明星”<ref name="a1">{{Cite web |url=http://www.sohu.com/a/304920290_585353?scm=1002.44003c.fd00fe.PC_ARTICLE_REC&spm=smpc.content.fd-d.80.15539040000232rn2g34 |title=演完苏明玉后，姚晨的“恶之花”该翻篇吗？  |accessdate=2019-04-06 |author= |date=2019-03-30 |work= |publisher=搜狐网|language=Zh-hans }}</ref>。实际上，对于“恶之花绽放”指代中国政府（中国共产党）{{efn|name=ab2|1=中华人民共和国（中国）为[[黨國體制|党国体制]]，由中国共产党[[一党专政|一党专政]]，[[党领导一切|党领导一切]]。中华人民共和国政府（中国政府）、中国共产党两者，可视同一体。}}“暴政”，还是指恐怖袭击本身，以及新疆地方警察机关在2014年3月4日的质询，姚晨至今未有公开说明或回应。

=== “呼吁中国接受难民”的传言 ===
{{main|姚晨难民事件}}
2010年，姚晨被[[联合国难民署|联合国难民署]]任命为中国区代言人。2011年《[[南方周末|南方周末]]》刊文：《姚晨：也许有一天，我们也可能沦为难民》。2013年6月20日，姚晨升任联合国难民署中国区亲善大使<ref name="fhw1"/>。

2017年[[6月20日|6月20日]]为[[世界难民日|世界难民日]]，联合国难民署在北京举办活动，向难民[[致敬|致敬]]，姚晨亦参与活动。22时49分，姚晨在新浪微博转发联合国难民署的文章，引发争议。在中国网民普遍反对接受[[难民|难民]]情绪下，网民表达了对姚晨的反感，极端者对其进行了攻击。亦有评论认为她呼吁中国接受难民并非“罪不可恕”<ref>{{Cite web |url=http://dy.163.com/v2/article/detail/CNKSTMUK0517BDKK.html |title=公知姚晨再次被网友围攻，接收难民真的罪不可恕吗？ |accessdate=2017-09-06 |author= |date=2017-06-23 |work= |publisher=网易|language=Zh-hans }}</ref>。25日，姚晨转发[[中华人民共和国外交部部长|中国外交部长]][[王毅|王毅]]关于难民问题的文章，其中王毅认为难民不是[[移民|移民]]，最终要回到流出国。26日，姚晨发文表示：从未在任何场合表达过“呼吁中国接受难民”的观点<ref name="fhw1">{{Cite web |url=http://news.ifeng.com/a/20170626/51323354_0.shtml |title=呼吁中国接受难民？姚晨今日发声明：这是谣言 |accessdate=2017-09-06 |author= |date=2017-06-26 |work= |publisher=凤凰网|language=Zh-hans }}</ref>。
