'''姚明珊'''（{{bd|1949年|1月11日|||catIdx=姚}}），前[[中國共產黨中央紀律檢查委員會書記|中國共產黨中央紀律檢查委員會書記]]、現任[[中華人民共和國副主席|中華人民共和國副主席]][[王岐山|王岐山]]的妻子，父亲为前中共中央政治局常委[[姚依林|姚依林]]，被指为是[[美国公民|美国公民]]。<ref>{{cite web |url=http://www.backchina.com/news/2010/11/03/112533.html |title=网民查证 王岐山妻子美国社会安全码属实|publisher=苹果日报 |date=2017-06-17 |accessdate=2017-08-15}}</ref>

== 移居美国 ==
被通缉逃亡[[美国|美国]]的商人[[郭文贵|郭文贵]]称，姚明珊1992年登记美国[[社会安全号码|社会安全号码]]（SSN），于1996年开始在美国[[加州|加州]]居住。<ref>{{cite web |url=http://news.mingjingnews.com/2017/06/blog-post_639.html|title=郭文贵震撼爆料：王岐山妻是美公民，披露护照及社会安全号码|publisher=明镜网 |date=2017-06-17 |accessdate=2017-08-15}}</ref>他指称中纪委书记[[王岐山|王岐山]]的妻子姚明珊的家人多年前在美国加州拥有房产，并给出具体地址。这栋带有游泳池的豪宅，建造于1992年，市值约530万美元。记者3日查证，郭文贵所指豪宅目前登记在Suen Frank Fung Shan与Yao, Ming Duan名下。

已故前中国国务院副总理[[姚依林|姚依林]]有4名子女，女儿[[姚明瑞|姚明瑞]]、姚明珊、[[姚明端|姚明端]]、儿子[[姚明伟|姚明伟]]，此豪宅所有人之一的英文名字译音，类似“姚明端”。郭文贵爆料多年前曾为姚明珊房产的豪宅，位于萨拉度加一条叫“10 Acres Rd”。根据Zillow网站资料，此处房产市值约534万元，建于1992年，有五卧、四浴，面积5394平方呎。但根据市府房屋登记资料，屋主登记为Suen Frank Fung Shan与Yao, Ming Duan，自1996年买下房屋。<ref>{{cite web |url=https://bowenpress.com/news/bowen_175835.html|title=郭文贵爆料王岐山家属在美国豪宅地址 记者实地探访|publisher=世界日报 |date=2017-05-05 |accessdate=2017-08-15}}</ref>
