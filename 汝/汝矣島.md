'''汝矣島'''（{{kor|諺=여의도|漢=汝矣島|羅=Yeoui do|f=y}}）是位於[[大韓民國|韓國]][[漢江_(韓國)|漢江]]上的一個小島，面積8.4[[平方公里|平方公里]]，現屬[[首爾|首爾特別市]][[永登浦區|永登浦區]]。汝矣岛是[[韓國交易所|韓國證券交易所]]（现为[[韩国交易所|韩国交易所]]的一部分）和{{link-en|韩国金融投资协会|Korea Financial Investment Association}}的所在地，为首尔的金融与投资中心，被誉为“韩国的华尔街”<ref name=jrj>{{cite web |url =http://travel.jrj.com.cn/2008/03/251145660368.shtml  |title=韩国首尔:春花烂漫之旅 |publisher =金融界 |date=2008-03-25}}</ref><ref name=xww>{{cite web |url =http://www.chinanews.com/life/2011/03-14/2904861.shtml  |title=到首尔与樱花约会 |publisher =中国新闻网 |date=2011-03-14}}</ref><ref>{{cite web|url=http://www.cqwb.com.cn/cqwb/html/2014-03/19/content_384190.htm|title=去韩国看《星星》拍摄地|publisher=重庆晚报|date=2014-03-19}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，是韩国地价最贵的地方<ref>{{cite web |url =http://news.xinhuanet.com/fortune/2013-03/12/c_124446650.htm  |title=首尔：汉江奇迹的“金融Style” |publisher =新华网 |date=2013-03-12}}</ref>。该岛也是许多韩国大公司，比如[[LG集团|LG集团]]、[[韓國放送公社|KBS]]的总部所在地。此外[[韓國國會|韓國國會]][[国会议事堂_(韩国)|議事堂]]、[[63大廈|大韓生命63大廈]]、{{link-en|首尔国际金融中心|International Finance Center Seoul}}以及全球最大的[[基督教|基督教]][[教堂|教堂]][[汝矣島純福音教會|汝矣島純福音教會]]亦設於該島上<ref>{{cite web|url=http://you.ctrip.com/sight/seoul234/138737.html|title=汝矣岛|publisher=携程旅游|date=2015-03-22}}</ref>。

汝矣岛是韩国著名的[[樱花|樱花]]观赏地。汝矣岛轮中路经国会议事堂至韩国放送公社前的大道两旁种满了樱花，被称为“樱花道”，长达10公里。早春时节樱花怒放时，“樱花道”会形成一条壮观的“樱花隧道”<ref name=xww /><ref>{{cite web|url=http://jx.youth.cn/2015/0409/1096710.shtml|title=韩国首尔汝矣岛国会被盛开樱花“包围”|publisher=中国青年网|date=2015-04-09|deadurl=yes|archiveurl=https://web.archive.org/web/20150415231643/http://jx.youth.cn/2015/0409/1096710.shtml|archivedate=2015-04-15}}</ref><ref>{{cite web|url=http://mt.sohu.com/20150323/n410172566.shtml|title=与韩国首尔汝矣岛樱花的浪漫约会|publisher=搜狐网|date=2015-03-23}}</ref>。岛上建有包括[[汝矣岛公园|汝矣岛公园]]、[[漢江公園|漢江公園]]在内的5个公园，亦是休闲赏花的好去处<ref>{{cite web|url=http://www.koreaxin.com/article-53985-1.html|title=可以在江边与绿色自然亲密接触的汉江市民公园汝矣岛地区|publisher=韩国新网|date=2010-07-15|deadurl=yes|archiveurl=https://web.archive.org/web/20150413231110/http://www.koreaxin.com/article-53985-1.html|archivedate=2015-04-13}}</ref>。

==历史==
汝矣岛在[[朝鲜王朝|朝鲜王朝]]时期还是一个无人居住的小岛。[[漢江_(韓國)|汉江]]发洪水的时候，小岛的大部分地方都会被淹没，故名“汝矣岛”。1916年起，汝矣岛开始被日本人用来当做[[飞机跑道|飞机跑道]]<ref name=ktimes>{{cite web |url =http://www.koreatimes.co.kr/www/news/issues/2015/04/363_106946.html  |title=An Chang-nam: First Korean pilot who became national hero |publisher =Koreatimes |date=2012-03-14}}</ref>。从日本学成归国的韩国首位飞行员[[安昌男|安昌男]]曾于1922年12月10日在汝矣岛进行了历史性的试飞，成为民族英雄<ref>{{cite web |url=http://world.kbs.co.kr/chinese/archive/program/program_koreanstory.htm?no=111076&current_page=4  |title=在天空中飞出独立之梦的 安昌男 |publisher =KBS |date=2013-01-17}}</ref><ref name=ktimes />。1924年，汝矣岛被扩建成首爾的首座[[機場|機場]]<ref name="yeongdeungpo.seoul.go.kr">{{cite web|url=http://yeongdeungpo.seoul.go.kr/Dong/04/sub01/c01.asp|title=동현황|work=Yeouido-dong website (Korean)|accessdate=2006-04-16}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。汝矣岛机场起落国际、国内航班也是军事机场，同时也是一所航空学校的所在地<ref name="yeongdeungpo.seoul.go.kr">{{cite web|url=http://yeongdeungpo.seoul.go.kr/Dong/04/sub01/c01.asp|title=동현황|work=Yeouido-dong website (Korean)|accessdate=2006-04-16}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

[[File:Korea_Seoul_Sprting_Flowers_20140401_15_(13556834514).jpg|thumb]]
1970年5月，連接汝矣島及[[麻浦區|麻浦區]][[龍江洞|龍江洞]]的[[麻浦大橋|麻浦大橋]]作为[[朴正熙|朴正熙]]政府汉江工程的一部分正式竣工<ref>{{cite web|url=http://www.hanguoyou.org/newcont-959.html?xyz=0.048511948669329286|title=韩国首尔五大最美大桥|publisher=韩国旅游|date=2015-03-22}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1971年，汝矣岛机场停用，取而代之的是林立的高楼大厦<ref name=ktimes />。1975年，[[韓國國會|韓國國會]][[国会议事堂_(韩国)|議事堂]]自[[太平路_(首爾)|太平路]]遷至該島上（原址現改為首爾特別市市議會）。1979年，[[韓國交易所|韓國證券交易所]]亦由[[明洞|明洞]]搬移至此<ref>{{cite web |url=http://jp.hjenglish.com/riyuxiaozhandian/p419815|title=不可不知的韩国概况：不可不知的韩国地名|publisher=沪江日语|accessdate=2015-04-09}}</ref>。
