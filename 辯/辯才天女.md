'''辯才天女'''（[[天城體|天城體]]：सरस्वती，[[IAST|IAST]]：{{IAST|sarasvatī}}，{{lang-bo|དབྱངས་ཅན་མ།}}），簡稱'''辯才天'''，又稱'''妙音天女'''{{noteTag|是另一個常見意譯，與主譯名皆出自佛經。}}，音譯為'''薩拉斯瓦蒂'''、'''娑羅室伐底'''，是[[婆羅門教|婆羅門教]]、[[印度教|印度教]]的一個重要[[女神|女神]]，代表著醫療、子嗣、財富、智慧、美貌、音樂；傳統上她被認為是主神[[梵天|梵天]]的[[妻子|妻子]]。同時也是[[佛教|佛教]]重要的[[護法|護法]]和[[本尊|本尊]]，在東亞地區，最早是由佛教介紹進入[[漢字文化圈|漢字文化圈]]。

而辯才天女與[[拜火教|拜火教]]的水之女神[[阿娜希塔|阿娜希塔]]，被學者推定同源。辯才天女有时与[[財寶天女|財寶天女]]相互混同；但多數時候，兩者都是不同的神靈。

== 印度教 ==
薩拉斯瓦蒂（即辯才天女）是在[[梨俱吠陀|梨俱吠陀]]中薩拉斯瓦蒂河的[[水神|水神]]（[[薩拉斯瓦蒂河|薩拉斯瓦蒂河]]的人格化），而此河在[[公元前|公元前]]3000年-[[公元前2000年|公元前2000年]]間就已經完全枯竭，現今只存在薩拉斯瓦蒂河的遺跡，據考證薩拉斯瓦蒂河曾經流動於[[喜馬拉雅山脈|喜馬拉雅山脈]]到印度西海岸之間，寬八公里。

而梨俱吠陀時代的[[婆羅門|婆羅門]]白印度人幾乎沒提及[[恒河|恒河]]，吠陀中提到的河流主要位於今[[巴基斯坦|巴基斯坦]]和[[印度|印度]]西北部，多是[[印度河|印度河]]的支流。看來薩拉斯瓦蒂河在上古印度人當中的地位相當重要，她被稱為河流七姐妹（都是她的支流）中最偉大者。<ref>梨俱吠陀，第6卷，LXI</ref>梨俱吠陀裡有三首頌歌專門獻給她（梨俱吠陀6.61，7.95和7.96）。在這些頌歌中列舉了一些讚頌薩拉斯瓦蒂的詞語，並說她能賜人以子孫、財富和長壽。<ref>《梵語文學史》，30頁</ref>她還治癒過受傷的[[因陀羅|因陀羅]]，甚至有一句頌詩認為是她殺死了因陀羅的死敵[[弗栗多|弗栗多]]。<ref>梨俱吠陀，第6卷，LXI，第7節</ref>總之，薩拉斯瓦蒂在吠陀時代極受尊敬，相當於後來恒河的地位。該河對於許多的西方學者了解上古印度人的歷史中具有重大的意義。

後來辯才天女逐漸從河神發展為司各種智慧和知識的女神。從[[梵書|梵書]]開始辯才天女被認為是梵天的女兒，同時又是他的妻子。史詩[[摩訶婆羅多|摩訶婆羅多]]也說辯才天女為梵天之妻。[[梵語|梵語]]被認為是她所創造。<ref>《摩訶婆羅多》</ref>她更是文藝和科學的保護神。梵語中“語言”一詞本身就成了辯才天女的一個稱號。<ref>《古代印度文藝理論文選》，39頁</ref>

辯才天女的形象通常為一個有四個手臂的美麗女郎，她皮肤白皙，身披白衣，坐騎是一隻[[天鵝|天鵝]]（有時是[[孔雀|孔雀]]）。各手中一手持經典，一手持[[維納琴|維納琴]]，一手持[[念珠|念珠]]或[[蓮花|蓮花]]，其餘一手撥琴或持裝[[聖水|聖水]]的[[寶瓶|寶瓶]]。

中古時期的印度文藝理論著作在開頭處經常儀式性地讚頌辯才天女的名字。如，[[檀丁|檀丁]]的《[[詩鏡|詩鏡]]》，[[曼摩吒|曼摩-{吒}-]]的《[[詩光|詩光]]》等；<ref>《古代印度文藝理論文選》，22頁，78頁</ref>更有一本綜合性的文學理論書名為《[[辯才天女的頸飾|辯才天女的頸飾]]》。崇拜辯才天女的節日是[[薩拉斯瓦蒂節|薩拉斯瓦蒂節]]（在印曆的星月舉行）。

== 神话中的辩才天女 ==
辩才天女普遍被认为是从梵天的左手大拇指上诞生，梵天见她美貌非凡，欲娶她为妻，辩才天女觉得不妥，拒绝了梵天的求爱，于是辩才天女转到梵天的左面、右面、后面甚至飞到上面，以避开梵天的目光，梵天長出了許多個頭顱，所以辩才天女最终还是逃不出他的视线而成为了他的妻子。辩才天女是梵天的白色“[[性力|性力]]”（沙克提）的化身，是梵天力量的体现，其创造力和能量均来自他的行动。

有时，辩才天女与[[財寶天女|財寶天女]]相互混同；而在一些[[傳說|傳說]]中，辩才天女与財寶天女一同为[[梵天|梵天]]的妻子，两人因争宠而势不两立。<ref>《古代印度神话》（魏庆征/编、北岳文艺出版社/山西人民出版社）p.721 ISBN 7-5378-1727-8</ref>传说她们曾经还发生过激烈地争吵；因为古代印度人认为智慧和财富无法并存，文学和幸运不能共容，所以財寶天女和辩才天女的关系不合。<ref>《印度神话》（杨怡爽/著、陕西人民出版社）p.77 ISBN 9787224089899</ref>另有一则神话是说[[毗湿奴|毗湿奴]]曾经有三个妻子，即辩才天女、[[財寶天女|財寶天女]]和[[恒河女神|恒河女神]]，辩才天女以天性喜爱與人[[辯論|辯論]]而出名，她们三人在一起总是爱争吵，毗湿奴厌倦了这种争吵，于是把他的其中两个妻子送给了自己的同伴，将辩才天女送给梵天，恒河女神送给[[湿婆|湿婆]]，自己则留下財寶天女。<ref>《古代印度神话》（魏庆征/编、北岳文艺出版社/山西人民出版社）p.800 ISBN 7-5378-1727-8</ref>

辩才天女不但容貌極美，也重視梳妝打扮。有一次，梵天打算举办隆重的祭祀仪式，当一切准备就绪时，梵天发现辩才天女还未到场，梵天一问侍女才知道辩才天女还在梳妆打扮，让他再等等。梵天非常着急，因为这个仪式必须要有他的妻子在场才行，于是梵天在一怒之下另娶一妻，娶了一个普通牧女伽耶特利（吟诵女神的化身）为妻，诸神大声欢呼，赞同梵天的决定。仪式正要开始时，辩才天女走进会场，愤怒地诅咒梵天每年只能享受到一天的祭拜。<ref>《印度神话：永恒的轮回》（刘晓辉 杨燕/译、中国青年出版社）p.89 ISBN 7-5006-5062-0</ref><ref>《印度神话》（杨怡爽/著、[[陕西人民出版社|陕西人民出版社]]）pp.77-78 ISBN 9787224089899</ref>

== 佛教 ==
[[佛教|佛教]]的辯才天女形像，將她視為增長智慧、增長福德的女性本尊。<ref>《金光明最勝王經》：“若人欲得最上智，應當一心持此法，增長福智諸功德，必定成就勿生疑。若求財者得多財，求名稱者得名稱，求出離者得解脫，必定成就勿生疑。”</ref>辯才天女也是[[藏傳佛教|藏傳佛教]]的「財寶十八尊」之一。辯才天女的形象更進一步傳至[[日本|日本]]，稱為“[[辯才天|辯才天]]”、“辨財天”或“辯（辨）天”，為日本民間信仰的[[七福神|七福神]]中唯一的[[女性|女性]]。
