{{Engname|《超强音浪》|''Super Music''}}，是[[中国大陆|中国大陆]][[山东卫视|山东卫视]]与引力传媒联合推出的一档集音乐、游戏与访谈于一体的室内综艺节目，由主持过《[[中国梦之声|中国梦之声]]》的知名主持人林海担任节目的总策划与主持人<ref>{{Cite web|url=http://ent.sina.com.cn/tv/zy/2016-11-01/doc-ifxxfuff7472342.shtml|title=山东卫视资源会选秀冠军打榜《超强音浪》|website=新浪娱乐|date=2016-11-01|accessdate=2017-08-20}}</ref>。本节目与其他音乐类综艺节目最大的不同之处在于，其他节目都是注重从平民中挖掘草根明星，亦或是让已经成名的歌手进行竞赛，而本节目则注重于回归音乐本身，主打“音乐+游戏”的差异化玩法，再加上依托大数据分析为嘉宾量身定制的提问访谈环节，为音乐增添了更多娱乐元素<ref>{{Cite web|url=http://ent.163.com/17/0102/08/C9OSTVLT00038FO9.html|title=《超强音浪》首播 胡彦斌谈“搬家”落泪|website=网易娱乐|date=2017-01-02|accessdate=2017-08-20}}</ref>。节目于2017年1月1日起每週日21:20播出，并于12月31日的「跨年特别节目」播放完毕后完结。歌曲版权由[[QQ音乐|QQ音乐]]独家取得，并通过腾讯音乐娱乐集团旗下的QQ音乐、[[酷狗音乐|酷狗音乐]]、酷我音乐联合发行。

== 播出时间 ==
《超强音浪》的首播频道为山东卫视，2017年1月1日正式开播，首播时间为每週日21:20，每集时长70分钟，按季播出，季与季之间无缝连接。

此外，当遇到一些重大事件时，该节目会适时地作出让步。例如，2017年1月29日因播出《2017山东卫视春节联欢晚会语言类节目集锦》而停播一期；2017年5月14日因转播“[[一带一路|一带一路]]”国际合作高峰论坛欢迎晚宴及文艺晚会而延后至21:50播出；2017年9月24日因晚间十点档安排播出纪录片《辉煌中国》而提前至21:00播出。

== 节目形式 ==
每期将会邀请一到几位知名歌手或演员来到节目现场，为观众带来充满情怀与回忆的表演，并与音浪小分队进行各种别开生面的游戏互动。此外，节目组还会依托大数据分析为嘉宾量身定制提问访谈环节，充分展现嘉宾的音乐个性，唤醒观众的情感共鸣。

== 节目主持人 ==
《超强音浪》的主持团体被称为“音浪小分队”，队长为主持过《[[中国梦之声|中国梦之声]]》的知名主持人林海，每期固定队员为4－6人。
* [[李祥祥|李祥祥]]，中国大陆男歌手，[[中国梦之声_(第一季)|《中国梦之声》第一季]]总冠军。
* [[林采欣|林采欣]]，台湾女歌手，《中国梦之声》第一季总决赛第14名。
* 阿速，本名吕素鹏，山东电视生活频道《生活帮》首席主持人。
* 张美琪，山东广播电视台融媒体资讯中心主持人。
* [[黄龄|黄龄]]，中国大陆女歌手，代表作《High歌》、《痒》。
* 东东，[[上海广播电视台|上海电视台]]主持人。
* 虎虎，本名刘锐，中国大陆男歌手，偶像组合“青鸟飞鱼”成员。
* 王子子，中国大陆女演员。
* 郝雷，中国大陆音乐制作人。
