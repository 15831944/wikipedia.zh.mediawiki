{{Vgname|超级马力欧派对|ja=スーパー マリオ パーティ|en=Super Mario Party|official-twhk=超級瑪利歐派對}}是由[[Nd_Cube|Nd Cube]]开发，[[任天堂|任天堂]]发行的[[聚会游戏|聚会游戏]]；于2018年10月5日在[[任天堂Switch|任天堂Switch]]平台发售；包含了多个适合多人同乐的聚会小游戏。本作是[[瑪利歐派對系列|马力欧派对系列]]第十一部主遊戲，也是首次加入线上“联网竞奖”（Online Athlon）模式的作品。<ref>{{Cite web|url=https://www.vgtime.com/topic/951192.jhtml|title=没朋友也别担心 《超级马力欧聚会》包含线上多人模式 - vgtime.com|accessdate=2018-09-21|work=www.vgtime.com}}</ref>

任天堂確認本作無法於[[任天堂Switch|任天堂Switch]]掌機模式使用雙邊[[Joy-Con|Joy-Con]]控制器運行，僅可使用家用機模式或單邊[[Joy-Con|Joy-Con]]控制器才可運行。<ref name="MSN">{{cite web |url=https://www.msn.com/zh-tw/money/other/%E3%80%8A%E8%B6%85%E7%B4%9A%E7%91%AA%E5%88%A9%E6%AD%90%E6%B4%BE%E5%B0%8D%E3%80%8B%E4%B8%8D%E6%94%AF%E6%8F%B4-switch-%E7%9A%84%E6%89%8B%E6%8F%90%E6%A8%A1%E5%BC%8F%EF%BC%8C%E9%80%99%E6%9C%83%E6%98%AF%E4%BB%BB%E5%A4%A9%E5%A0%82%E7%9A%84%E6%96%B0%E5%95%8F%E9%A1%8C%E5%97%8E%EF%BC%9F/ar-BBNRlnv |archiveurl=https://web.archive.org/web/20181015130820/https://www.msn.com/zh-tw/money/other/%E3%80%8A%E8%B6%85%E7%B4%9A%E7%91%AA%E5%88%A9%E6%AD%90%E6%B4%BE%E5%B0%8D%E3%80%8B%E4%B8%8D%E6%94%AF%E6%8F%B4-switch-%E7%9A%84%E6%89%8B%E6%8F%90%E6%A8%A1%E5%BC%8F%EF%BC%8C%E9%80%99%E6%9C%83%E6%98%AF%E4%BB%BB%E5%A4%A9%E5%A0%82%E7%9A%84%E6%96%B0%E5%95%8F%E9%A1%8C%E5%97%8E%EF%BC%9F/ar-BBNRlnv |title=《超級瑪利歐派對》不支援 Switch 的手提模式，這會是任天堂的新問題嗎？ |last= |first= |website=[[MSN|MSN]] |date=2018-10-03 |accessdate=2018-10-03 |archivedate=2018-10-03 |deadurl=no}}</ref>

== 游戏玩法 ==
=== 线上模式 ===
本作为马力欧派对系列中首次提供在線多人模式的作品。玩家可在该“联网竞奖”（Online Athlon）模式中可与全球玩家游玩80款游戏。此外，在Mariothon模式中，玩家可参加5款随机迷你游戏。<ref name="Nintendo Life">{{cite web |url=http://www.nintendolife.com/news/2018/06/super_mario_party_will_feature_online_play_with_friends_but_only_for_minigames |archiveurl=https://web.archive.org/web/20180615130820/http://www.nintendolife.com/news/2018/06/super_mario_party_will_feature_online_play_with_friends_but_only_for_minigames |title=Super Mario Party Will Feature Online Play With Friends, But Only For Minigames |last=Craddock |first=Ryan |website=[[Nintendo_Life|Nintendo Life]] |date=2018-06-15 |accessdate=2018-06-15 |archivedate=2018-06-15 |deadurl=no}}</ref>

== 销售 ==
游戏在发售两天内，日本销量已達到142,868 份，其销量已超過了其前輩作品「[[瑪利歐派對10|瑪利歐派對10]]」和「[[馬里奧聚會9|瑪利歐派對9]]」。<ref>{{cite web|url=https://gematsu.com/2018/10/media-create-sales-10-1-18-10-7-18|title=Media Create Sales: 10/1/18 – 10/7/18|work=Gematsu|first=Sal|last=Romano|date=2018-10-10|accessdate=2018-10-10}}</ref> 
截至2019年1月31日，游戏總銷量超過530萬份，遠遠超過任天堂的預期销量，是自「[[瑪利歐派對6|瑪利歐派對6]]」以來銷量最快的馬里奧派對系列，並成為謹次於「[[瑪利歐派對DS|瑪利歐派對DS]]」和「[[馬里奧聚會8|瑪利歐派對8]]」以來第三個暢銷的馬里奧派對系列。
