{{nihongo|《'''超能力魔美'''》|エスパー魔美|Esupā Mami}}是一部由[[藤子·F·不二雄|藤子·F·不二雄]]在1977年至1983年連載之[[漫畫|漫畫]]，后于1987年被製成動畫。

《超能力魔美》於1977年至1978年在[[小學館|小學館]][[漫畫雜誌|漫畫雜誌]]《マンガくん》連載。《マンガくん》改為《少年ビッグコミック》後，《超能力魔美》在《少年ビッグコミック》不定期連載至1983年完結。

香港[[亞洲電視|亞洲電視]]1989年播出動畫的粵語配音版。


== 劇情 ==
佐倉魔美是明月中學2年級學生，但不久她即發現自己擁有超能力。因為她擁有超能力，她可以幫助身邊的人及她的同班男同學兼好朋友高畑和夫。她不久即開始使用[[瞬間移動|瞬間移動]]及[[心電感應|心電感應]]來幫助別人，而當任何人需要幫助時，她即使用「心型變身器」到達現場。

== 角色 ==
; {{nihongo|佐倉魔美|佐倉 魔美（さくら まみ）（台：左魔美<ref>第一卷《磨練超能力》p.44</ref>）||配音員：[[横澤啟子|横澤啟子]]（日本）／[[楊凱凱|楊凱凱]]（台灣）／[[林錦燕|林錦燕]]（香港），演員：[[笹岡莉紗|笹岡莉紗]]}}
: 本作主角，14歲，血型O型。身高153cm、體重42kg、胸圍75cm・腰圍56cm・臀圍80cm。
: 就讀於東京都佐間丘陵的明月中學，繼承父方曾祖母法國血統的的紅髮少女。<ref>第一卷《還需要用功》p.72、73</ref>因為一次的巧合而發現自己擁有超能力，並將之用於幫助別人上。個性開朗有正義感，但她的課業與廚藝較為遜色。由於父親是畫家，魔美經常以裸體模特兒賺取零用錢<ref>第一卷《誰是超少女？》p.21</ref>，而她的目標也是繼承她父親之能力—成為畫家。有幾次她想向她父親表示她具有超能力，但她得知法籍祖先因被當成巫師慘遭處刑，因此決定保密。<ref>第一卷《還需要用功》p.79</ref>

; {{nihongo|高畑和夫|高畑 和夫（たかはた かずお）（台：高和夫<ref>第一卷《誰是超少女？》p.16</ref>）||配音員：{{tsl|ja|柴本浩行}}（日）／[[陳永信|陳永信]]（香港），演員：[[上條誠|上條誠]]}}

: 故事開始時因被能力尚未覺醒的魔美以超能力救助，誤以為自己具有超能力，並和魔美進行超能力訓練。直到後來才發現魔美是真的超能力者，曾不想理會魔美，<ref>第一卷《友情因噴嚏而消失了》p.122～124</ref>最後成為她的知心好友兼超能力開發研究夥伴。

; {{nihongo|コンポコ|コンポコ（台：小波<ref>第一卷《誰是超少女？》p.11</ref>）||配音員：[[日比野朱里|日比野朱里]]}}
: 魔美的雄性寵物犬，但長得像[[狸貓|狸貓]]，極度討厭別人誤認牠為[[狸貓|狸貓]]，否則就會咬人<ref>第一卷《誰是超少女？》p.13</ref>。喜歡吃炸豆腐，經常發出"Fuyanfuyan"的叫聲，聽得懂人類說的話。鄰居陰木家的雌犬瑪莉很喜歡牠。<ref>第一卷《友情因噴嚏而消失了》p.130</ref>

; {{nihongo|佐倉十朗|佐倉 十朗（さくら じゅうろう）（台：左志明<ref>第一卷《名畫與鬼爸爸》p.84</ref>）||配音員：[[增岡弘|增岡弘]]（日）／[[龔祖澤|龔祖澤]]（香港），演員：[[草刈正雄|草刈正雄]]}}
: 魔美父親，是位畫家。經常舉辦私人畫展，同時也是市立高中藝術科教師。<ref>第一卷《磨練超能力》p.53</ref>作畫時常吸菸，個性老實，喜歡吃醃漬蘿蔔。其祖母為法國人<ref>第一卷《還需要用功》p.72</ref>，因此經常稱呼女兒"魔美公（爵）"。由於戰後日本的仇視歐美情結，因此成長過程常受到不友善對待，曾畫下[[山梨市|山梨市]]小孩在[[第二次世界大戰|第二次世界大戰]]下的殘破景況。在動畫版結局中實現前往法國深造的夢想。

; {{nihongo|佐倉菜穂子|佐倉 菜穂子（さくら なおこ）/佐倉 花枝（電視劇）||配音員：[[榊原良子|榊原良子]]（日）／[[龍寶鈿|龍寶鈿]]（香港），演員：[[涼風真世|涼風真世]]}}
: 魔美母親，工作於[[讀賣新聞|讀賣新聞]]國際新聞部（台譯朝賣新聞外信部<ref>第一卷《磨練超能力》p.53</ref>），是個工作家事皆精通、公私分明的能幹女子，個性好相處。

; {{nihongo|陰木|陰木さん（いんき）||配音員：[[京田尚子|京田尚子]]}}
: 佐倉家鄰居，丈夫病弱、孩子失蹤，過著灰暗的生活，因此變得陰險多疑並將憤怒矛頭轉向幸福的佐倉家。後在魔美幫助下回復正常的生活。

; {{nihongo|竹長悟|竹長 悟（たけなが さとる）||配音員：[[佐佐木望|佐佐木望]]}}
: 魔美同學。高畑和夫的球友。新聞社社員，似乎和幸子互有好感。正義感強烈，絕不向惡勢力妥協。

; {{nihongo|桃井紀子|桃井 のり子（ももい のりこ）||配音員：[[淵崎有里子|淵崎有里子]]}}
: 魔美朋友，血型B型。爱称是"Non"。個性樂觀。擅長烹飪，經常煮東西請魔美及高畑和夫吃。

; {{nihongo|間宮幸子|間宮 幸子（まみや さちこ）||配音員：[[江森浩子|江森浩子]]}}
: 魔美朋友。爱称是"Sacchan"。個性沉靜。

; {{nihongo|番野兆治|番野 兆治（ばんの つよし）||配音員：[[鹽屋翼|鹽屋翼]]}}
: 校內的反派人物。魔美的超能力也因為他欺負高畑和夫而被發現。魔美曾意外的將他拋開。

; {{nihongo| 富山高志|富山 高志（とみやま たかし）||配音員：{{tsl|ja|平野義和}}}}
: 魔美同學。帶著眼鏡且對於[[古典音樂|古典音樂]]有極高之造詣。

; {{nihongo| 有原成宏|有原 成宏（ありはら なりひろ）||配音員：[[鈴置洋孝|鈴置洋孝]]}}
: 明月中學電影欣賞社社長。曾籌備電影然後以魔美的超能力為主軸，並邀請魔美當女主角，但最後因為該片需魔美裸身演出但被魔美拒绝而失敗。

; {{nihongo| 黒沢庄平|黒沢 庄平（くろさわ しょうへい）||配音員：[[飛田展男|飛田展男]]}}
: 明月中學電影欣賞社副社長。在電影拍攝時，他發現了魔美能使用超能力的秘密。之后他开始以此要挟魔美并试图追求魔美。父親是一家一流製片場社長。

; {{nihongo| 黒雪妙子|黒雪 妙子（くろゆき たえこ）（台：黒雪妙<ref>第一卷《磨練超能力》p.44</ref>）||配音員：[[鶴弘美|鶴弘美]]}}
: 高畑和夫的童年時期好友，動畫設定則提及為表姊。<ref>第2集《看穿超能力》</ref>和夫通常稱她為「小妙」，年齡近二十歲，喜歡騎摩托車、菸酒及[[迪斯可|迪斯可]]，一開始見到魔美時認為她是個粗暴的女生而跟她互相懷有敵意。

== 魔美會的超能力 ==
魔美主要使用之超能力是[[念力|念力]]。同時她也會使用[[漂浮術|漂浮術]]（在許多集數裡，她可以使許多人浮起來），[[心電感應|心電感應]]及[[瞬間移動|瞬間移動]]（原作：一次距離可移動600m），不過在一開始她還沒有控制[[瞬間移動|瞬間移動]]之能力（需靠物體撞擊自己）。直到高畑和夫發明心型變身器之後<ref>第一卷《去死吧！評論家！》p.141</ref>，她才有辦法控制[[瞬間移動|瞬間移動]]之能力（裏頭有仁丹可當移動時要用的物體）。
