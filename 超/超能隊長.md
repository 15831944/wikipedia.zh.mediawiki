{{Vgname|超能隊長|ja=キャプテンスピリット|en=The Awesome Adventures of Captain Spirit|diff=yes|tw=超能隊長的奇異冒險|cn=超能隊長}}是由[[Dontnod娱乐|Dontnod娱乐]]開發的[[電子角色扮演遊戲|電子角色扮演遊戲]]，預定于2018年6月26日在[[PlayStation_4|PlayStation 4]]、[[Xbox_One|Xbox One]]和[[Microsoft_Windows|Microsoft Windows]]開放免費下載。

== 遊戲玩法 ==
遊戲方式與《[[奇妙人生|奇妙人生]]》相同，採用劇情冒險玩法，玩家將扮演寂寞的10歲小男孩克里斯（Chris），透過與角色的對話以及對環境物件的探索與互動，來作出各種抉擇，引導故事劇情的發展。

== 登場人物 ==
* 克里斯·埃里克森
: 配音：Chandler Mantione
: 本作的主角。是個充滿想像力的男孩，夢想成為超級英雄。
* 查爾斯·埃里克森
: 配音：Nick Apostolides
: 克里斯的父親。

== 開發 ==
《超能隊長》首次於2018年6月22日，在微軟XboxE3發表會上亮相，並且宣布，將在6月26日[[PlayStation_4|PlayStation 4]]、[[Xbox_One|Xbox One]]和[[Microsoft_Windows|Microsoft Windows]]開放免費下載<ref>{{Cite web|url=https://gnn.gamer.com.tw/8/163838.html|title=【E3 18】《奇妙人生》團隊新作《心靈隊長》26 日免費下載 化身小男孩展開大冒險|date=2018-06-22|website=巴哈姆特|language=zh|accessdate=2018-06-25}}</ref>。
遊戲使用[[虚幻引擎|虚幻引擎4]]開發。
