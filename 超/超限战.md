'''超限战'''是中国军旅作家[[乔良_(军人)|喬良]]和前[[中国人民解放军空军|中国人民解放军空军]]大校[[王湘穗|王湘穗]]在其所著的同名书籍中提出的概念<ref>{{cite web | url = http://www.chinatimes.com/newspapers/20150502000737-260309 | title = 喬良《超限戰》界定新型態戰爭 | accessdate = 2017-01-17 | author = 陳君碩 | date = 2015-05-02 | publisher = [[中時電子報|中時電子報]]}}</ref>。在[[911事件|911事件]]发生之前两年半就作出了预测，根据他们的论述，超限战是与[[传统战争|传统战争]]不同的新的战争手段，是以一切手段，超越传统战争手段范围的新型战争形式。它包括了传统的战争手段，同时也包括了[[贸易战|贸易战]]、新[[恐怖主义|恐怖主义]]及[[生态战|生态战]]。

==概論==
超限战可以通过不流血手段达到传统战争可以、甚至不可能达到的效果，它是立足于现代暴力冲突的演变与现代经济、文化、科技领域的高速发展上的。它强调了技术在未来战争中的地位，但是同时也提出军事思想仍是现代军队的重要组成部分。《超限战》一书立足于美国在[[越战|越战]]後的历次战争，特别是[[第一次海湾战争|第一次海湾战争]]与[[科索沃战争|科索沃战争]]，并且与网络攻击、[[亚洲金融风暴|亚洲金融风暴]]、国际极端恐怖主义相结合，认为未来的战争将是无处不在的，包括金融、貿易、網路駭客、媒體與國際法等範圍，何时何地都将是战场。<ref>[http://www.chinatimes.com/newspapers/20140901001595-260301 中時-未來超限戰]</ref><ref name = "超限戰_2004"> {{ cite web | title = 超限戰 | url = https://www.books.com.tw/products/0010277940 | publisher = 左岸文化 | author = 喬良、王湘穗 | isbn = 9867854799 | language = zh-tw | date = 2004-12-01 }} </ref>

華盛頓郵報將其稱為40年來中國在西方影響最大的兵書，[[西點軍校|西點軍校]]作為推薦讀物<ref>[https://www.youtube.com/watch?v=ymGS75DwlZQ 鳳凰衛視-喬良做客]</ref>{{Third-party-inline}}，義大利步軍總監米尼上將稱其為當代軍事名著。台灣學者，優勢戰略管理研究中心主任刘振志<ref>[http://risechina.blogspot.tw/2013/02/blog-post_3558.html 刘振志：《超限战》的历史传承及对霸权主义的警示]</ref>評論為「以中国共产党的背景，加上中国固有的军事文化，益之以转化过了的辩证法，搅拌上几十年的战争冶炼，才造就了《超限战》的思想温床，试问，美国和台湾怎么学？不论怎么学，也都只是东施效颦。」

== 超限戰與反超限戰 ==
2016年8月，乔良推出17年後的新修訂版「超限戰與反超限戰」，除了原本內容外還擴增了十多年來國際新發展的整理和新理論。乔良指出戰爭的「泛化」是未來必然的結局，網路戰、資源戰、媒體戰、金融戰、文化戰，這些領域都將是未來激烈白熱的戰場，戰爭已經遠遠超出穿軍裝的軍人和飛機大砲的範圍，中國必須將所有的領域都軍事化看待，<ref>[https://www.youtube.com/watch?v=9-grjwS7oxQ 北京衛視-超限戰新編書發表]</ref>並接受大量不穿軍裝的非軍事人才是超限與反超限的關鍵，政府必須儘快介入所有的無形戰爭領域預做準備。同時也提到其實超限戰對於中國共產黨並不是陌生的新名詞，在[[國共內戰|國共內戰]]當時並沒有這名詞但就運用了很多超限戰概念雛形，去對抗蔣介石的傳統軍服戰爭。<ref name = "超限戰與反超限戰：中國人提出的新戰爭觀美國人如何應對_2016"> {{ cite web | title = 超限戰與反超限戰：中國人提出的新戰爭觀美國人如何應對 | url = https://www.books.com.tw/products/CN11368914 | publisher = 長江文藝出版社 | author = 喬良等 | isbn = 9787535487407 | language = zh-cn | date = 2016-08-01 }} </ref>
