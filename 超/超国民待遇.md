'''超国民待遇'''指一国政府给予外籍人士或特定人士、以及外资机构超过[[国民待遇|国民待遇]]的特权，常被认为是对本国公民的一种[[逆向歧视|逆向歧视]]。

== 中国 ==
[[晚清|晚清]]时期，[[清朝政府|清朝政府]]给予外籍人士政治、经济、司法特权。实际政务中，各级官员“于是不问是非、不顾顺逆、不分黑白、不论短长，一惟媚远人、私远人、听命远人是务”，故有“一等洋人二等官”的俗语<ref>{{Cite web |url=http://www.sohu.com/a/136490345_469537 |title=一等洋人二等官！晚清为何有那么多官员对“洋大人”卑躬屈膝？ |accessdate=2017-08-19 |author= |date=2017-04-26|work= |publisher=搜狐网|language=Zh-hans }}</ref>。

在[[改革开放|改革开放]]初期，政府为吸引外资来华投资，对外商投资企业、外资企业和外籍人士，在[[税收|税收]]、土地使用、行政审批等各方面，提供便利和优惠。这种让利行为在1990年代达到高峰。而自1994年起，政府逐步在税收领域统一外资、内资待遇。2012年12月1日，政府开始对外商投资企业、外资企业和外籍人士征收[[城市维护建设税|城市维护建设税]]和[[教育费附加|教育费附加]]，被认为是在税收领域结束了外籍的超国民待遇<ref>{{Cite web |url=http://finance.sina.com.cn/review/20101201/11519038504.shtml |title=外企超国民待遇终结 专家称其应适应新环境 |accessdate=2017-08-19 |author=记者：刘欢、韩洁、王建 |date=2010-12-01|work= |publisher=新浪网，来源：新华网|language=Zh-hans }}</ref>。但部分人認为政府在司法、政治、经济等方面<ref>{{Cite web |url=http://news.ifeng.com/mainland/detail_2012_05/19/14662324_0.shtml |title=公安大学教授：我们确实给了外国人所谓的超国民待遇|accessdate=2017-09-28 |author= |date=2012-05-19|work= |publisher=凤凰网，来源：央视《新闻周刊》|language=Zh-hans }}</ref>对外籍人员的让利、照顾并未停止。即使是来自贫穷国家的国民，仍会受到官员巴结<ref>{{Cite web |url=http://www.sohu.com/a/154141133_99912621 |title=津巴布韦产妇产子，副市长忙活啥？ |accessdate=2017-08-22 |author= |date=2017-07-03|work= |publisher=搜狐网|language=Zh-hans }}</ref>，称之为“量中华之物力，结与国之欢心”。这种“巴结”与[[中华人民共和国民族政策|中华人民共和国民族政策]]交织一起，产生了“一等洋人二等官，三等少民四等汉”之类的讽刺性说法<ref>{{Cite web |url=http://tie.163.com/gt/14/0716/10/A193EADT00304IND.html |title=洋人来了 |accessdate=2017-08-19 |author=步轩 |date=2014-07-16|work= |publisher=网易|language=Zh-hans }}</ref>。

[[中华人民共和国司法体制|中华人民共和国司法体制]]在处理涉外案件，依据中国外交原则——“外事无小事”，涉外案件无小案<ref name="a1">{{Cite web |url=http://news.ifeng.com/mainland/detail_2012_05/19/14662324_1.shtml |title=公安大学教授：我们确实给了外国人所谓的超国民待遇|accessdate=2017-09-28 |author= |date=2012-05-19|work= |publisher=凤凰网，来源：央视《新闻周刊》|language=Zh-hans }}</ref>。因此日常生活中，外籍人士获得[[中华人民共和国人民警察|警方]]优待的新闻屡见不鲜，由此产生“外国人代报案”的[[都市传说|都市传说]]<ref>{{Cite web |url=http://news.ifeng.com/gundong/detail_2013_10/20/30483761_0.shtml |title=“老外代报案”让谁难堪 |accessdate=2017-09-26 |author= |date=2013-10-20|work= |publisher=凤凰网，来源：云南信息报|language=Zh-hans }}</ref><ref>{{Cite web |url=http://news.163.com/13/1020/09/9BKCMERI00014AEE.html |title=别把“老外代报案”当谣言去查处 |accessdate=2017-09-26 |author=郭元鹏 |date=2013-10-20|work= |publisher=网易，来源：东方网|language=Zh-hans }}</ref>，宣称“[[老外|老外]]报案能够百分百破案”<ref>{{Cite web |url=http://news.xinmin.cn/world/2013/10/22/22385013.html |title=媒体评老外代报案破案率高：信息不对称惹的祸|accessdate=2017-09-26 |author= |date=2013-10-22|work= |publisher=新民网，来源：中国新闻网|language=Zh-hans }}</ref>。同时，司法机关对涉外案件包括外籍人士犯罪案件时，亦有“不会管、不愿意管、不敢管”——三不管的处理状态。有认为“外事无小事”的原则在此被错误的使用。具体案件中，法院不敢处理外籍人士，直接损害本国公民的合法权益<ref name="a1"/>。

另出于[[統戰|統戰]]的需要，[[港澳台|港澳台]]人士亦属于享受超国民待遇。政府官员在[[对台工作|对台工作]]时，不仅单独方面向台湾让利，甚至公开要求中国大陆民众为台湾人士放弃自己权益<ref name="a4">{{Cite web |url=http://news.cnr.cn/native/gd/20170307/t20170307_523642425.shtml |title=人大代表张兆民：对台胞“高看一等、厚爱一筹” 平潭发展“三多一快” |accessdate=2017-08-19 |author=记者：张子亚、郭婕妤 |date=2017-03-07|work= |publisher=央广网|language=Zh-hans }}</ref>。
