'''超級大空'''（{{lang-ja|スーパーおおぞら}}）是由日本鐵路公司[[北海道旅客鐵道|北海道旅客鐵道]]所營運，用於往來[[北海道|北海道]]裡[[札幌|札幌]]與[[釧路|釧路]]之間的[[特急列車|特急列車]]。

== 概要 ==
此特急列車於1997年3月22日正式啟用，作為替代先前同樣往來札幌與釧路間的[[大空號列車|大空號列車]]。超級大空在車輛款式上採用由[[富士重工業|富士重工業]]與[[北海道旅客鐵道苗穂工廠|北海道旅客鐵道苗穂工廠]]打造的[[JR北海道Kiha_283系柴聯車|JR北海道Kiha 283系柴聯車]]，<ref>{{cite book |script-title=ja:JR新幹線&特急列車ファイル―新幹線 電車特急 気動車特急 寝台特急 夜行列車 |date=2008年7月 |publisher=交通新聞社  |isbn=978-433-000-608-6 }}</ref>在2014年8月30日起採用的編排班次上；於往來札幌釧路兩地三百多公里間最快可在3小時58分裡到達。<ref>{{Cite web |url=http://www.jrkushiro.jp/jikoku.html |script-title=ja:札幌→帯広・釧路特急スーパーおおぞら／特急スーパーとかち/ |work=[[北海道旅客鐵道|北海道旅客鐵道]] |accessdate=2015年1月22日 |deadurl=yes |archiveurl=https://web.archive.org/web/20150202105704/http://www.jrkushiro.jp/jikoku.html |archivedate=2015年2月2日 }}</ref>

== 車輛編制 ==
在車款規格上全採用[[JR北海道Kiha_283系柴聯車|JR北海道Kiha 283系柴聯車]]，其中以釧路運輸機廠發送的列車以六節車廂組成、札幌機廠的則是七節車廂為主，另曾在旅客眾多期間加派到十一節車廂過。<ref>{{cite book |script-title=ja:列車編成席番表 2009夏・秋 |date=2009年7月 |publisher=交通新聞社 |author={{lang|ja|ジェーアールアール}} |isbn=978-433-008-109-0 }}</ref>

超級大空號在2006年開始採用將部分車廂內；改成綠色車廂規格的豪華指定席，之後於2008年3月全部導入完成。<ref>{{Cite web|url=http://www.jrhokkaido.co.jp/press/2006/061213-1.pdf |script-title=ja:特急列車の指定席にグレードアップ座席が新登場！ |publisher=[[北海道旅客鐵道|北海道旅客鐵道]] |format=[[PDF|PDF]] |date=2006年12月13日 |accessdate=2015年1月22日}}</ref>後續在2011年做出進一步升級，將座椅材質、空間調整，並增加照明用[[LED|LED]]座燈和可讓[[行動電話|行動電話]]使用的插座。<ref>{{Cite web|url=http://www.jrhokkaido.co.jp/press/2011/111013-2.pdf |script-title=ja:グリーン車の座席をリニューアル！ |publisher=[[北海道旅客鐵道|北海道旅客鐵道]] |format=[[PDF|PDF]] |date=2011年10月13日 |accessdate=2015年1月22日}}</ref>

因超級大空號會行經野外山林區域，較易發生與如[[蝦夷鹿|蝦夷鹿]]等誤闖鐵道的北海道野生動物相撞、而造成延遲情況的發生。

另嚴重一次的意外是在2011年5月27日發生之{{nihongo|石勝特急列車脫軌火災事故|石勝線特急列車脱線火災事故}}，該意外除造成旅客及工作人員共30多人受傷，<ref>{{Cite web|url=http://www.jrhokkaido.co.jp/press/2011/110528-1.pdf |script-title=ja:石勝線 清風山信号場構内で発生した列車脱線事故について |publisher=[[北海道旅客鐵道|北海道旅客鐵道]] |format=[[PDF|PDF]] |date=2011年5月27日 |accessdate=2015年1月22日}}</ref><ref>{{Cite web |url=http://www.tokachi.co.jp/news/201105/20110529-0009334.php |script-title=ja:部品脱落し脱線か 無残な姿見せる石勝線特急 |work=十勝毎日新聞 |date=2011年5月29日 |accessdate=2015年1月22日 |deadurl=yes |archiveurl=https://web.archive.org/web/20110601235444/http://www.tokachi.co.jp/news/201105/20110529-0009334.php |archivedate=2011年6月1日 }}</ref>，也影響[[北海道旅客鐵道|北海道旅客鐵道]]考慮往後安全及維護方面因素，於2013年9月期間決定讓超級大空號在可使用的最高時速限制從每小時130公里調低到每小時110公里。<ref>{{Cite web|url=http://www.jrhokkaido.co.jp/press/2013/130904-1.pdf |script-title=ja:安全性向上に向けた輸送サービス抑制へのご理解について” |publisher=[[北海道旅客鐵道|北海道旅客鐵道]] |format=[[PDF|PDF]] |date=2013年9月4日 |accessdate=2015年1月22日}}</ref>

== 虛構作品 ==
2014年日本[[吉卜力工作室|吉卜力工作室]]在夏季推出、以[[北海道|北海道]]為舞台的動畫《[[回憶中的瑪妮|回憶中的瑪妮]]》（{{lang|ja|思い出のマーニー}}），有將超級大空號列車採用在劇情前段的部份場景中。<ref>{{cite book |script-title=ja:思い出のマーニー ロマンアルバム |date=2014年9月5日 |publisher=[[德間書店|德間書店]] |isbn=978-4-19-720396-3 |page=24頁}}</ref>
