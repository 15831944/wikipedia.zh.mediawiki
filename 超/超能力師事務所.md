《'''増山超能力師事務所'''》是一部由[[譽田哲也|誉田哲也]]所編寫的小説。[[2013年|2013年]][[7月16日|7月16日]]由[[文藝春秋|文藝春秋]]出版。[[2017年|2017年]]1月被改編成電視劇，同年4月、在沖縄国際映画祭展出了電視劇的分拆作品。

== 故事概要 ==
這是一個存在超能力者的世界。在日暮里的「增山超能力師事務所」，聚集了很多超能力者。在所長增山率領下，這些性格各異、能力不同的超能力師們，為了不同的委託而忙碌著。

== 登場人物 ==
=== 主要人物 ===
* 増山圭太郎：[[田中直樹|田中直樹]]（香港配音：[[招世亮|招世亮]]）
: 增山超能力師事務所所長，一級超能力師。能力深不可測，口頭蟬是 "真麻煩啊"。原本在高鍋研究所任職，離開後創辦增山超能力師事務所。
* 高原篤志：[[淺香航大|淺香航大]]（香港配音：[[黃積權|黃積權]]）
: 增山超能力師事務所偵探，二級超能力師。實力在團隊中最弱，考了6年才通過超能力師等級試，可以透過接觸物件而獲得上面的信息。
* 住吉悦子：[[中村友理|中村友理]]（香港配音：[[成瑤孆|成瑤孆]]）
: 增山超能力師事務所偵探，二級超能力師。擅長放火術，同時亦擁有超強的讀心能力，經常扮作其他角色作調查。於增山出差時會暫代社長工作。
* 中井健：[[柄本時生|柄本時生]]（香港配音：[[麥皓豐|麥皓豐]]）
: 增山超能力師事務所偵探，二級超能力師。擅長讀心術，曾經因為無法屏蔽別人心聲而十分痛苦，後來在增山幫助下建立心牆。在事務所經常負責尋找走失動物案件。
* 大谷津朋江：{{tsl|ja|平田敦子}}（香港配音：[[伍秀霞|伍秀霞]]）
: 增山超能力師事務所會計，沒有任何超能力。
* 宇川明美：{{tsl|ja|ルウト}}（香港配音：[[何寶珊|何寶珊]]）
: 增山超能力師事務所實習生，能力很強但不懂得正確控制自己的超能力。外表是一個女生，但實際上是個男生。
