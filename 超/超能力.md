'''超能力'''（{{lang-en|Super-powers}}；{{lang-de|Psi-Phänomen}}，Psi來自希臘詞{{lang|el|ψυχή}}〔Psyche〕，義為“心靈”、“靈魂”、“精神”，即英文“精神病”psycho的詞源），又称为'''特異功能'''，是指[[心灵感应|心靈感應]]、[[透視_(超感官知覺)|透視]]、[[預知|預知]]、[[念力|念力]]、意影、特殊体质（身體著[[火|火]]、[[發電|發電]]）等的超自然能力。研究這類現象的學科稱爲[[超心理学|超心理學]]（{{lang-en|Parapsychology}}）。

[[錢穆|錢穆]]則以[[史書|史書]]記載如[[扁鹊|扁鵲]]可隔牆見物等諸多異事，以及孩提時的親眼見證，認為特異功能不可妄加否定其存在的可能性。根據他觀察到諸多特異功能的案例，發現超能力者中，以幼童及[[女性|女性]]佔多數，並指出「[[西|西方]]科學心物相異，偏重[[物质|物質]][[空间|空間]]，此等事誠屬怪異。[[中國人|中國人]]向主心物和合成體，則外物移動，未嘗與心無關。亦可謂此等同屬心理現象，不必偏向物質上探求。但亦非專屬心理學，須心理[[物理学|物理]]混為一體求之，庶可得解。觀此，知此等乃人心之本有功能，亦可謂是人心之[[自然|自然]]功能。及其漸長，多在人事上歷練，則此等功能漸失去。但經特殊訓練，年長後，仍保有此功能者，亦可有之。此等事，[[中國|中國]][[社会|社會]]常見不鮮。」<ref>錢穆《現代中國學術論衡》之〈略論中國心理學 二〉－東大出版社、三民出版社、蘭臺出版社</ref>

在虛構作品中，常以在少青年時期發揮能力的形象與[[发展心理学|發展心理學]]並談。在超能力、超常現象關連用語中常見的[[前綴|前綴]]「psy-」是源自於[[希腊语|希臘語]]表示心、[[灵魂|魂]]意思的「psyche」。

== 超能力的種類 ==
; 超感官知覺（ESP）
* [[透視_(超感官知覺)|透視]]及[[遙視|遙視]]（Clairvoyance），[[佛經|佛經]]中稱作「[[天眼通|天眼通]]」或俗稱「千里眼」
* 遠隔聽覺（Clairaudience），[[佛經|佛經]]中稱作「[[天耳通|天耳通]]」或俗稱「順風耳」
* [[心灵感应|心靈感應]]（Telepathy），[[佛經|佛經]]中稱作「[[他心通|他心通]]」
* [[接觸感應|接觸感應]]（Psychometry）
* [[預知|前瞻]]（Precognition），預見未來，俗稱「預知」
* [[動物傳心|動物傳心]]
* [[後瞻|後瞻]]（Postcognition），回知過去，[[佛經|佛經]]中稱作「[[宿命通|宿命通]]」
* 催眠能力（Hypnosis），用意志對他人加屬暗示，讓他人信以為真（現已列入一般[[心理學|心理學]]研究）
* [[通靈|通靈]]
* [[靈魂出竅|靈魂出竅]]
* [[信仰治療|信仰治療]]
* [[前世回溯|前世回溯]]
* 意識轉移，類似附身他人、靈魂憑身
* 具現能力，俗稱「幻想實現術」
* {{tsl|en|Eidetic memory|瞬間記憶力}}（Photographic memory），俗稱「過目不忘」
* 情緒控制（Emotional control），俗稱「控制喜怒哀乐」
* 心靈控制（Mind control），俗稱「觀心者」
* 夢境控制（Dream control），俗稱「築夢師」 
; 意志念動（PK）
* [[念力|念力]]（Psychokinesis），又稱意念致動和念力移物，即用超能力使物體移動
* [[瞬間移動|瞬間移動]]（Teleportation），[[佛經|佛經]]中稱作「[[神足通|神足通]]」
* [[念寫|念寫]]（Thoughtography），也有透過電視將想法或意志轉化成影像與聲音的「念聽」
; 特殊體質
* [[人體發火|人體發火]]
* 人體生電，少數人可點亮燈泡或日光燈管
* 精神治癒（{{lang|en|Spirit Healing}}）、在不借助藥物或外科手術等醫療行為的前提下，用手或聲音使病患痊癒，泛稱[[神蹟|神蹟]]或[[灵疗|灵疗]]
* 人體生磁
* [[免疫體質|免疫體質]]，可免疫較強的毒藥
* 肉體強化，少數人可達到堅不可摧的程度
* 行動預知
* 肉體再生
* 石化
* 穿牆破壁
* 隱形
* 行走岩漿

== [[神話|神話]]或[[宗教|宗教]]故事中相當於超能力的表現 ==
在古代沒有“超能力”這種稱呼，因為一般人對於[[傳說|傳說]]和[[史詩|史詩]]中的[[英雄|英雄]]和[[聖人|聖人]]的能力深信不疑，而且在建制化的宗教也一直在宣揚教主的神力，如一神教的“[[奇蹟|奇蹟]]”或道教、佛教、婆羅門教中的“[[神通|神通]]”。
