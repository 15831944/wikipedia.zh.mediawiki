== 超級焰星 ==
{{As of|2000}},已經發現的焰星有9顆，有些與我們的[[太陽|太陽]]相似<ref name="RS2000" />。這種閃焰釋放出的能量是太陽最大的[[日冕物質拋射|日冕物質拋射]]能量的100倍至1000萬倍<ref name="RS2000" />。

== 詳情 ==
超級閃焰會使恆星的亮度增加為正常亮度的20倍，而[[光度|光度]]增加1000倍，它們可能最少持續幾個小時到一個星期。如果太陽釋放出一個超級閃焰，這個超級閃焰將使冬天溫暖的像夏天。地球的[[臭氧層|臭氧層]]可能會被這個閃焰所攜帶的帶電粒子摧毀，並且即使遠離太陽在[[木星|木星]]距離上的衛星，在白天這一側表面的冰也會被融化，而在閃焰漸漸減弱後又會再凍結。沒有任何證據顯示[[太陽系|太陽系]]在過去曾經發生過超級閃焰<ref name="RS2000" />。

== 在科幻 ==
在影片，[[末日預言|末日預言]] (2009年)，依據不嚴謹的聖經故事，世界是被超極閃焰毀滅的。

在拉瑞尼文的短篇故事"[[無常月亮|無常月亮]]"中，超級閃焰將白天這一側的地球毀於一旦。
