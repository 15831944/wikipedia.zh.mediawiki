《'''霹靂酷樂貓'''》（{{lang-ja|サイボーグクロちゃん}}）是由[[橫內尚樹|橫內尚樹]]所作的[[日本漫画|日本漫画]]，後被改编为[[电视动画|电视动画]]和[[遊戲軟體|遊戲軟體]]。電視動畫由1999年10月至2001年1月于[[东京电视网|东京电视网]]播放。

== 故事概要 ==
本作講述的是關於一隻住在日本櫻町富士井家中的混種家貓酷樂的故事。一天，想統治世界的反派剛博士把牠改造成[[生化人|生物機械]]後，過著熱血、瘋狂和搞笑的生活。

== 登场角色 ==
; {{nihongo|酷樂|クロ}}　
:[[聲優|聲]]：[[坂本千夏|坂本千夏]]（日本）；[[魏晶琦|魏晶琦]]（台灣）；[[林丹鳳|林丹鳳]]（香港）
: 由一對老爺爺和老奶奶所飼養的貓，在去找心上狗寶莉時被剛博士捉去，而改造成機器貓，在回到家裡後，開始了一段「不正常地過正常生活」。個性我行我素，平時會懶洋洋，但一有危機時，會為了解決事件到處破壞而變成「破壞王子」。舊名'''奇德'''。在遇到老爺爺和老奶奶前曾渡過一段流浪的歲月。

; {{nihongo|小M|ミー}}
:聲：[[手塚千春|手塚千春]]（日本）；[[楊凱凱|楊凱凱]]（台灣）；[[沈小蘭|沈小蘭]]（香港）
:喵喵軍團之首、酷樂的宿敵。初期是剛博士偶遇的小貓，兩者同被天童勇間接所害，在報仇的過程中成為生死之交。
:個性比較女性化，擅長料理。一開始與酷樂為敵但後期慢慢轉變成如摯友般的關係。
:還是幼貓時就已會使用雙手作菜和雙腳走路，與機器人28號去一同為母親報仇時被天童的基因改造犬打成重傷，機器人28號也因此無法再修復，及時趕到的剛博士便馬上把小M與機器人28號帶到自己所就讀的大學，利用大學的設備與機器人28號的零件將小M改造成機械貓，雖然一時沒了心跳，但就在剛博士要被天童的基因改造犬殺害時小M及時趕到並解決了基因改造犬為母親報了大仇。

; {{nihongo|獨眼龍|マタタビ}}
:聲：[[大本真基子|大本真基子]]（日本）；[[符爽|符爽]]（台灣）；[[袁淑珍|袁淑珍]]（香港）
: 在幼時與酷樂相識的虎紋貓，性格非常獨立，能夠做出像人的動作（雙足步行、拿筷子等）因而被稱作妖貓。最後當格治（一作葛奇）的王國被敵對貓幫暗算時，酷樂一氣之下，在與獨眼龍打架時誤將其右眼挖出，隨即多茨插手，令兩貓各自分途。之後獨眼龍獨自修行和尋酷樂復仇。現在的他自稱「在下」。木工做的很好，斗蓬內藏有極多武器。找到酷樂後似乎慢慢消除內心對酷樂的復仇之心，後來也住在老爺爺和老奶奶的家，酷樂不在時會幫他保護兩位老人家。
:值得注意的是獨眼龍雖為血肉之軀，但其實力已足以與酷樂和小M這兩隻被改造的機械貓相抗衡。 

; {{nihongo|娜娜|ナナ}}
:聲：[[綱掛裕美|綱掛裕美]]（日本）；[[楊凱凱|楊凱凱]]（台灣）；[[鄭麗麗|鄭麗麗]]（香港）
: 小太郎試作的家用女僕，將座桌電燈製成機械人。在與小太郎爭吵而被追著拆毀時，與酷樂第一次邂逅，並愛上了他。努力嘗試成為稱職的酷樂女朋友中。

; {{nihongo|剛萬太郎|剛万太郎}}
:聲：[[古澤徹|古澤徹]]（日本）；[[沈光平|沈光平]]（台灣）；[[林保全|林保全]]（香港）
: 研究生物機械的（邪惡）科學家。曾經因貧窮和誤解而被人厭棄，繼而想統治世界以「拆去人與人之間的圍牆」，成立喵喵軍團，但在改造酷樂後漸漸被他的性格所影響而忘記了大業。與小M是生死之交。痛恨天童勇。
: 除了喵喵軍團和酷樂外，還改造了為了自由而逃出動物園最終被人射殺的獅子丹克。

; {{nihongo|小太郎|コタロー}}
:聲：[[小松里賀|小松里賀]]（日本） ；[[林美秀|林美秀]]（台灣）；[[陸惠玲|陸惠玲]]（香港）
: 擁有[[智商|IQ]]200和四歲力氣的天才少年，酷樂的忠實支持者。認為普通的人生（小學生活）是個很無意義的[[角色扮演遊戲|RPG遊戲]]，對將來沒有信心，經過黑咪的鼓勵後想成為科學家，拜剛博士為師。
:經常穿著一件酷樂造型的布偶裝，還養著一個機器獅子丹克。
:有一個爸爸，因為經常流浪讓父子不常相聚，其頭腦和技術不輸給小太郎。

; {{nihongo|鈴木一郎|鈴木 一郎}}
:聲：[[森川智之|森川智之]]（日本）；[[符爽|符爽]]（台灣）；[[鄺樹培|鄺樹培]]（香港）
: 櫻町小學的體育教師，在暴走公路決鬥事件中遇上酷樂，而成為其徒弟，但在學生中似乎很沒威嚴。
:是個喜歡組裝[[GUNDAM|GUNDAM]]模型的[[御宅族|御宅族]]，又有一個外號叫作吉姆(來自GUNDAM中的一種量產機，因為性能不佳，所以有派不上用場的意思)

; {{nihongo|惠美|めぐみ}}
:聲：[[吉田小百合|吉田小百合]]（日本）；[[梁少霞|梁少霞]]（香港）
: 櫻町的女消防員，個性就像火一樣的熱血。因為與鈴木一樣喜歡[[GUNDAM|GUNDAM]]而一拍即合，但是她不管遇到火焰(不管火勢大小)時一定要撲滅的可怕執念。

; {{nihongo|爺爺、奶奶|ジーサン、バーサン}}
:聲：[[菅原淳一|菅原淳一]]、[[興梠里美|興梠里美]]（日本）；[[黃子敬|黃子敬]]、[[譚炳文|譚炳文]]（公公代配）、[[區瑞華|區瑞華]]、[[黃鳳英|黃鳳英]]（婆婆代配）（香港）
: 櫻町中的一對老夫老妻，非常樂觀且天然呆，號稱比酷樂還硬。酷樂不想他們知道自己被改造為生物機械，所以常要不斷藏起自己的金屬身體。
:夫妻兩人很容易陷入危機中，常讓酷樂疲於奔命的去救他們。
:兩人都是大路痴，但又喜歡旅行，所以老是跑到和目的地不一樣的地方。

; {{nihongo|騾密歐|ロミオ}}
:聲：[[岡野浩介|岡野浩介]]（日本）；[[符爽|符爽]]（台灣）；[[陳卓智|陳卓智]]（香港）
: 男性燈柱形機械僕人，豬麗葉的情人。過去是外星人的僕人，後來酷樂在南極打倒他的主人後，而回復自由。
:經常做一些令人討厭的事，所以讓酷樂把它視為像瘟神一樣恐怖。

; {{nihongo|豬麗葉|ジュリエット}}
:聲：[[吉田小百合|吉田小百合]]（日本）；[[盧素娟|盧素娟]]（香港）
: 女性燈柱形機械僕人，騾密歐的情人。和騾密歐同樣為外星人的僕人並且同時墜落到地球上，之後被酷樂他們在學校後山的金字塔所救後，而回復自由。

; {{nihongo|喵喵軍團|ニャンニャンアーミー}} 
: 剛博士被世人所驅罵時，將曾經救回的重傷小貓改造成生物機械，以向他人報仇。小M為隊長。第一集中被酷樂收拾之後只剩下四位。

; {{nihongo|天童勇|天童 勇}} 
:聲：[[龍田直樹|龍田直樹]]（日本）；[[李錦綸_(配音員)|李錦綸]]（香港）
: 剛博士在大學時的對手，基因[[遺傳學|遺傳學]]專家。有基因改造犬「帕多拉修」，為殺害M仔生母的兇手。

; {{nihongo|惡魔|デビル}}
:聲：[[飛田展男|飛田展男]] （日本）；[[招世亮|招世亮]]（香港）
: 某次剛博士不小心放出來、500年前的惡魔，最後被重新封印。

; {{nihongo|白俄博士（懷特伍茲）|ドクターホワイトウッズ}} 
:聲：[[千葉一伸|千葉一伸]]（日本）；[[張炳強|張炳強]]（香港）
: 來自美國的邪惡科學家，以生物機械狗軍團殺害野貓作試驗、以行走要塞作基地。觸發酷樂的怒火，剛博士在酷樂誘惑下改造幼野貓們，並迎戰白俄博士。

; {{nihongo|獨眼外星人|一つ目宇宙人・地底人}} 
: {{配音2|JP|[[千葉一伸|千葉一伸]]}}
: 羅密歐和茱麗葉的主人的外星人族，在古代時已欲侵略地球，生性野蠻兇暴。
