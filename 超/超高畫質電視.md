[[File:UHDTV_camera_fuji0000.jpg|right]]
'''超高畫質電視'''（UHDTV, Ultra High Definition Television），又稱為'''Ultra HDTV'''、'''4320p'''、'''8K解析度'''及'''UHDV'''（Ultra High Definition Video）。是一種實驗中的[[數位視訊|數位視訊]]標準，由[[日本放送協會|日本放送協會]]（NHK）、[[英國廣播公司|英國廣播公司]]（BBC）及[[義大利廣播電視公司|義大利廣播電視公司]]（RAI）等機構所倡議推動。NHK提議將超高畫質電視稱為'''Super Hi-Vision'''（'''SHV'''）。

2018年12月1日，[[NHK_BS8K|NHK BS8K]]正式開播，成為世界上第一個開播的8K超高畫質電視頻道。

== 概要 ==
NHK首先於2005年的[[愛知博覽會|愛知博覽會]]展示此技術，並於同年11月以[[光纖|光纖]][[網路|網路]]及[[波分复用|DWDM]]傳輸技術將UHDTV轉播至260公里外的地方。2006年底的[[NHK紅白歌合戰|NHK紅白歌合戰]]（{{lang-ja|紅白歌合戦}}）藉由[[TCP/IP協議|IP網路]]將UHDTV畫面從[[東京|東京]]傳送到500多公里遠的[[大阪|大阪]]，在一面450英吋（11.4公尺）的[[螢幕|螢幕]]實況播出<ref>{{cite web|url = http://www.nhk.or.jp/digital/en/super_hi/03_2_super.html|title = Super Hi-Vision live relay over IP|accessdate = 2008-01-15|date = 2007-04-03|publisher = [[NHK|NHK]]|archiveurl = https://archive.is/20120526074017/http://www.nhk.or.jp/digital/en/super_hi/03_2_super.html|archivedate = 2012-05-26|deadurl = yes}}</ref>。未壓縮的20分鐘UHDTV節目會佔用4[[TB|TB]]的儲存空間。

UHDTV的最終目標是能普及於每一家庭，預計於2016年至2020年間可達到，但還要視科技進展而定，如傳輸高頻寬的技術進展、處理UHDTV視訊的能力等。<ref>http://www.satmagazine.com/cgi-bin/display_article.cgi?number=243539090</ref>

== 技術細節 ==
===分辨率===
超高畫質定義了兩種分辨率：
*4K UHDTV（2160p）的寬高為3840×2160。總像素數是FULL HD的4倍。
*8K UHDTV（4320p）的寬高為7680×4320。總像素數是FULL HD的16倍。
