'''超临界流体萃取'''（SFE，简称'''超临界萃取'''）是一种将[[超临界流体|超临界流体]]作为[[萃取剂|萃取剂]]，把一种成分（萃取物）从另一种成分（基质）中分离出来的技术。其起源于20世纪40年代，70年代投入工业应用，并取得成功。<ref name=yanjiujinzhan/>使用这种技术时基质通常是固体，但也可以是液体。SFE可以作为分析前的样品制备步骤，也可以用于更大的规模，从产品剥离不需要的物质（例如脱[[咖啡因|咖啡因]]）或收集所需产物（如[[精油|精油]]）。[[二氧化碳|二氧化碳]]（CO<sub>2</sub>）是最常用的超临界流体。

== 基本原理 ==
[[File:SFEschematic.jpg|thumb]]
{{further|超临界流体}}
超临界萃取的基本原理是在高于[[临界温度|临界温度]]和[[临界压力|临界压力]]的条件下，用[[超临界流体|超临界流体]]溶解出所需的化学成分，然后降低流体溶液的压力或升高流体溶液的温度，使溶解于超临界流体中的溶质因其密度下降溶解度降低而析出， 从而实现特定溶质的萃取。<ref name=yanjiujinzhan>{{cite journal|author=陈文伟，王震宙|title=超临界萃取技术的应用研究进展|journal=西部粮油科技|year=2003年06期|accessdate=2012-09-16}}</ref><ref name=general_chemistry>{{cite book|author=徐端钧 聂晶晶 刘清|title=新编普通化学（第二版）|publisher=科学出版社|isbn=978-7-03-034038-2|pages=8}}</ref>超临界流体是处于临界温度和临界压力以上的高密度流体。既不是气体，也不是液体，性质介于气体和液体之间，特点是具有优异的[[溶剂|溶剂]]性质。流体处于超临界状态时，其密度接近于液体密度，并且随流体压力和温度的改变发生十分明显的变化，而溶质在超临界流体中的[[溶解度|溶解度]]随超临界流体密度的增大而增大。超临界萃取正是利用超临界流体的这一性质而进行。<ref name=yanjiujinzhan/>

== 萃取剂 ==
可作为超临界萃取中萃取剂的物质很多，如[[超臨界二氧化碳|二氧化碳]]、[[氧化亚氮|氧化亚氮]]、[[六氟化硫|六氟化硫]]、[[乙烷|乙烷]]、[[甲醇|甲醇]]、[[氨|氨]]和[[水|水]]等。但用超临界萃取方法提取天然产物时，一般用二氧化碳（CO<sub>2</sub>）作萃取剂。<ref name=yingyongyanjiu>{{cite journal|author=陈岚，满瑞林|title=超临界萃取技术及其应用研究|journal=现代食品科技|year=2006年01期|pages=199-202|accessdate=2012-09-16}}</ref>因为CO<sub>2</sub>的临界温度（31℃）接近室温，对易挥发或具有生理活性的物质破坏较少。同时，CO<sub>2</sub>安全无毒，萃取分离可一次完成，无残留，适用于食品和药物的提取。CO<sub>2</sub>液化压力低，临界压力（7.31MPa）适中，容易达到超临界状态也是重要原因。<ref name=general_chemistry />

== 技术特点 ==
超临界萃取技术的特点与优势有以下几点：<ref name=yanjiujinzhan/>
* 可在接近常温下完成萃取工艺，适合对一些对热敏感、容易[[氧化|氧化]]分解、破坏的成分进行提取和分离。
* 在最佳工艺条件下，能将提取的成分几乎完全提出，从而提高产品的收率和资源的利用率。
* 萃取工艺简单，无污染，分离后的超临界流体经过精制可循环使用。

== 应用 ==
在[[食品工业|食品工业]]中，超临界萃取技术用于[[茶叶|茶叶]]、[[咖啡豆|咖啡豆]]脱[[咖啡因|咖啡因]]；食品[[脱脂|脱脂]]；[[酒花|酒花]]有效成分提取；[[植物色素|植物色素]]的萃取；植物及动物[[油脂|油脂]]的萃取。<ref name=yanjiujinzhan/>

在[[医药工业|医药工业]]中，超临界萃取技术用于[[酶|酶]]、[[维生素|维生素]]等的精制；动植物体内药物成分的萃取；医药品原料的浓缩、精制；[[糖类|糖类]]与[[蛋白质|蛋白质]]的分离以及脱溶剂脂肪类混合物的分离精制等。<ref name=yanjiujinzhan/>

在[[化妆品工业|化妆品工业]]中，超临界萃取技术用于[[天然香料|天然香料]]的萃取；[[合成香料|合成香料]]的分离精制；化妆品原料的萃取、精制。<ref name=yanjiujinzhan/>
