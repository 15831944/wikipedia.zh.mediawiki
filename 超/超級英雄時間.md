《'''超級英雄時間'''》（'''{{lang|ja|スーパーヒーロータイム}}'''，'''{{lang|en|Super Hero Time}}'''），簡稱為『SHT』，是由[[東映|東映]]和[[朝日|朝日]]共同製作的[[特攝|特攝]]作品「[[超級戰隊系列|超級戰隊系列]]」、「[[假面騎士系列|假面騎士系列]]」連播的時間。

== 歷史 ==
: 1997年4月6日『[[電磁戰隊百萬連者|電磁戰隊百萬連者]]』至第八話後從原本的每周五17:30，移動到每周日早上7:30播放，與當時每周日早上八點的『[[鐵甲小寶|鐵甲小寶]]』成為早上7:30至8:30全一個小時的特攝連映。然後就在2000年1月30日同樣為早上8:00的『[[燃燒吧！！小露寶|燃燒吧！！小露寶]]』結束後，由平成假面騎士系列新節目『[[假面騎士空我|假面騎士空我]]』接替。於是每周日早上的'''超級戰隊系列'''和'''假面騎士系列'''的兩大英雄的節目連續放映體制就此成型。進一步的在2003年9月28日『[[爆龍戰隊暴連者|爆龍戰隊暴連者]]』、『[[假面騎士Faiz|假面騎士Faiz]]』播放的這天，東映和朝日將兩這兩大系列節目綜合在一個小時內放映的時間命名這定為『'''超級英雄時間'''』。

: 而原本『'''超級英雄時間'''』的口號設定只到2003年的12月底，直到2005年前的同一個時間並沒有規律的設定出現。

: 直到2005年的新節目『'''[[魔法戰隊魔法連者|魔法戰隊魔法連者]]'''』開始才又開始使用『'''超級英雄時間'''』的設定，並且片頭都多了'''超級英雄時間（{{lang|ja|スーパーヒーロータイム}}）'''的字樣。

: 由於該時段的前面及後面接著都是朝日電視台的重要動畫時段，因此也會影響前後動畫的收視及發展。前面動畫時段7：00至7：30自1998年2月7日開始播放至2017年9月24日19年，其著名節目為[[Battle_Spirits|Battle Spirits]]系列；後面動畫時段8：30至9：00從1984年10月7日開始至今己超過30年，其著名節目為[[光之美少女系列|光之美少女系列]]。以上時段在2007年3月4日至2011年1月30日期間被[[朝日電視台|朝日電視台]]稱為『'''星期日上午兒童節目時段'''』。

: 截至2017年9月的超級英雄時間為7:30至8:00時段播出「[[超級戰隊系列|超級戰隊系列]]」，8:00至8:30播出「[[假面騎士系列|假面騎士系列]]」。

: 2017年10月1日起，超級英雄時間將會作出播放調動。9:00至9:30時段首先播出「[[假面騎士系列|假面騎士系列]]」，然後9:30至10:00時段播出「[[超級戰隊系列|超級戰隊系列]]」，有別于以往「先戰隊後騎士」的播映時間。<ref>{{Cite web|title=『宇宙戦隊キュウレンジャー』放送時間変更のお知らせ｜東映［テレビ］|publisher=[[東映|東映]]|date=2017-07-03|url=http://www.toei.co.jp/release/tv/1210070_963.html|accessdate=2017-07-04}}</ref><ref>{{Cite web|title=仮面ライダーシリーズ　放送時間変更のお知らせ｜東映［テレビ］|publisher=[[東映|東映]]|date=2017-07-03|url=http://www.toei.co.jp/release/tv/1210075_963.html|accessdate=2017-07-04}}</ref>

: 不過因為[[光之美少女系列|光之美少女系列]]還是維持當前的8：30至9：00，但也會因為此變更也隨之產生收視上的改變<ref group="注">由於新節目是5:50播至8:30，因此該時段節目不變。但是由此開始節目階段由原本維持近14年的的7:00動畫時段 → 超級戰隊 → 假面騎士 → 光之美少女 → 其他節目模式，變成了新節目 → 光之美少女 → 假面騎士 → 超級戰隊 → 其他節目，那麼收視效果也會改變。</ref>。

== 海外情況 ==
=== 香港 ===
:1973年從『[[假面騎士|假面騎士]]』開始的假面騎士系列，一直至『[[假面騎士顎門|假面騎士顎門]]』都在[[亞視本港台|亞視本港台]]播出，之後2004年9月25日從『[[假面騎士龍騎|假面騎士龍騎]]』至『[[假面騎士鎧武|假面騎士鎧武]]』在[[電視廣播有限公司|無綫電視]][[翡翠台|翡翠台]]播放假面騎士系列作品，播畢後該台亦無播放任何『假面騎士系列』片集作品，而此段空白期官方已經在『[[假面騎士EX-AID|假面騎士EX-AID]]』推出本作相關商品、播放電視廣告、連載該作資訊及漫畫，『'''假面騎士Drive'''』至『'''假面騎士Ex-aid'''』未在電視播放，直至『[[假面騎士Build|假面騎士Build]]』登陸[[香港電視娛樂|香港電視娛樂]]旗下的[[ViuTV|ViuTV]]為止。

: 超級戰隊系列1978年佳藝電視台播出『[[秘密戰隊五連者|秘密戰隊五連者]]』（『'''五勇士'''』之名，後因佳藝電視停業而遭腰斬）。之後1981年[[亞視本港台|亞視本港台]]播出『[[JAKQ電擊隊|JAKQ電擊隊]]』（『'''皇牌突擊隊'''』之名）。1989年[[無綫翡翠台|無綫翡翠台]]播出『[[光戰隊覆面人|光戰隊覆面人]]』（『'''超能戰士'''』之名）、『[[高速戰隊渦輪連者|高速戰隊渦輪連者]]』（『'''雷霆煞星'''』之名）、『[[恐龍戰隊獸連者|恐龍戰隊]]』。1995年亞視本港台繼續播出『[[五星戰隊大連者|五星戰隊]]』至『[[超力戰隊王連者|超力戰隊]]』。1998年無綫翡翠台繼續播出『[[地球戰隊五人組|地球戰隊五人組]]』（『'''星空戰隊'''』之名）、『[[鳥人戰隊噴射人|鳥人戰隊噴射人]]』（『'''飛鳥戰隊'''』之名），2000年更改時段播出『[[電磁戰隊百萬連者|電磁戰隊百萬連者]]』（『'''數碼戰隊'''』之名）、『[[星獸戰隊銀河人|星獸戰隊]]』，2003年從『[[百獸戰隊牙吠連者|百獸戰隊]]』開始一直在無綫翡翠台繼續播放超級戰隊系列作品（『[[天裝戰隊護星者|天裝戰隊]]』及 『[[動物戰隊獸王者|動物戰隊]]』除外）。因爲『[[宇宙戰隊九連者|宇宙戰隊]]』播完後日本仍在播放後作『[[快盜戰隊魯邦連者VS警察戰隊巡邏連者|快盜戰隊VS警察戰隊]]』，所以由『[[花生動畫|花生動畫]]』暫代3周。直至『[[快盜戰隊魯邦連者VS警察戰隊巡邏連者|快盜戰隊VS警察戰隊]]』於日本播完後香港便會播出粵語配音版。

: 無綫翡翠台播放時，假面騎士都在週六11:15播出，超級戰隊都在週日09:30播出，所以超級英雄時間在香港是名副其實的未形成，結束後該作在香港接檔播放由[[卡通頻道|卡通頻道]]作品的《[[Ben_10_再顯神威|少年勇將之再顯神威]]》，播畢後一個月接檔由[[孩之寶|孩之寶]]作品的《[[彩虹小馬_(2010年電視劇)|小馬寶莉動畫版]]》時段對調播映，週六11:15時段播畢後改播其他節目，假面騎士改由ViuTV播放。

: 該時段前後時段的動畫[[Battle_Spirits|Battle Spirits]]系列（2012年3月25日播出第一季，至2015年6月19日播畢第四季後沈寂）以及[[光之美少女系列|光之美少女系列]]（2006年7月16日播出第一季，至2017年4月7日播畢第十季後沉寂），也曾在當時時段同時在無綫翡翠台播出，同時每一個系列到都在此頻道播出過。

=== 台灣 ===
: 從『[[假面騎士BLACK|假面騎士BLACK]]』開始的假面騎士系列，曾先後在[[衛視中文台|衛視中文台]]、[[八大戲劇台|八大戲劇台]]、[[華視|華視]]、[[東森幼幼台|東森幼幼台]]、[[MOMO親子台|MOMO親子台]]、[[卡通頻道|卡通頻道]]播出（『[[假面騎士顎門|假面騎士顎門]]』、『[[假面騎士Wizard|假面騎士Wizard]]』未播出），『[[假面騎士鎧武|假面騎士鎧武]]』和『[[假面騎士EX-AID|假面騎士EX-AID]]』由[[東森綜合台|東森綜合台]]播出（『[[假面騎士Drive|假面騎士Drive]]』、『[[假面騎士Ghost|假面騎士Ghost]]』未播出）。

: 而原本的超級戰隊系列從1996年首華卡通台播出『[[超新星閃光人|超新星閃光人]]』、1997年播出『[[光戰隊覆面人|光戰隊覆面人]]』。1998年衛視中文台播出『[[忍者戰隊隱連者|忍者戰隊隱連者]]』（『'''影子神兵'''』之名）。之後2003年10月5日至2011年9月18日，從『[[百獸戰隊牙吠連者|百獸戰隊牙吠連者]]』一直至『[[炎神戰隊轟音者|炎神戰隊轟音者]]』都在八大戲劇台播出，2011年10月9日從『[[侍戰隊真劍者|侍戰隊真劍者]]』至『[[獸電戰隊強龍者|獸電戰隊強龍者]]』改由東森幼幼台播出（『[[天裝戰隊護星者|天裝戰隊護星者]]』未播出），『[[烈車戰隊特急者|烈車戰隊特急者]]』開始由東森綜合台播出（『[[手裏劍戰隊忍忍者|手裏劍戰隊忍忍者]]』、『[[宇宙戰隊九連者|宇宙戰隊九連者]]』未播出）。『[[快盜戰隊魯邦連者VS警察戰隊巡邏連者|快盜戰隊魯邦連者VS警察戰隊巡邏連者]]』繼續由東森幼幼台播出。

: 東森幼幼台在2012年由於拿到『[[假面騎士W|假面騎士W]]』的播映版權，加上在2011年就開始播放的『侍戰隊真劍者』，並訂於2012年7月1日開始每週日17:00至18:00兩大英雄片連放一小時，所以東森幼幼台將此時段命名為『'''超級英雄特區'''』。後自2014年2月15日開始，將假面騎士播放的時段由每週日17:30至18:00，改為每週六16:30至17:00播放，但播放的時間名稱仍然維持不變。

: 東森幼幼台在2014年5月3日『[[假面騎士OOO|假面騎士OOO]]』播畢後，沉寂了4個月，直到同年9月7日才開始播放新一部的假面騎士『[[假面騎士Fourze|假面騎士Fourze]]』，然而在8月24日『[[特命戰隊Go_Busters|特命戰隊Go Busters]]』播畢之後，也沉寂了一週，直到9月7日才開始播放新一部的超級戰隊『獸電戰隊強龍者』，並調整超級英雄特區的時間為每週日16:30至17:30。有趣的是，台灣所播的新一部的假面騎士及超級戰隊的首集為同日一同首播，而假面騎士Fourze及獸電戰隊強龍者的集數剛好同48集，所以就在2015年8月2日一同結束。據說這是為考量孩子們開學而來的想法，也是為了保持超級英雄特區的完整性。

: 但是上面兩支作品播畢後，超級戰隊方面沉寂了9個月，東森綜合台於2016年4月30日起，每週六08:00-08:30播出『烈車戰隊特急者』。而假面騎士方面沈寂了一年一個月，東森綜合台於2016年9月3日起，每週六08:30-09:00跳過『假面騎士Wizard』而直接進入『假面騎士鎧武』，因此睽違一年一個月的超級英雄特區終於再度形成，同樣是東森電視台拿到播映權，只是更換頻道而已。有趣的是，這兩支作品2017年1月21日至東森幼幼台一同於每週六15:00至16:00重播，而兩支作品集數剛好同47集，所以就在同年12月9日一同結束，雖然也是超級英雄特區，但其實是重播。

: 但是《[[動物戰隊獸王者|動物戰隊獸王者]]》和《[[假面騎士EX-AID|假面騎士EX-AID]]》分別在2018年2月17日和7月14日播畢後，超級戰隊方面沉寂了一年兩個月，直到2019年4月27日才跳過《宇宙戰隊九連者》而直接進入《快盜戰隊魯邦連者VS警察戰隊巡邏連者》，繼續由東森幼幼台播出。至於假面騎士系列目前官方還在安排中，不是繼續播出《[[假面騎士Build|假面騎士Build]]》就是直接進入《[[假面騎士ZI-O|假面騎士ZI-O]]》。

: 該時段前後時段的動畫[[Battle_Spirits|Battle Spirits]]系列（2010年8月11日播出第一季，至2016年1月7日播畢第四季後沈寂）以及[[光之美少女系列|光之美少女系列]]（2005年12月31日播出第一季，至2016年6月8日播畢第十季後沉寂），也曾在當時時段同時在東森幼幼台播出，同時每一個系列到都在此頻道播出過。

=== 中國大陸 ===
:從《'''假面騎士ZI-O'''》開始，由[[騰訊視頻|騰訊視頻]]購入假面騎士系列且緊貼日本播放，在《'''假面騎士ZI-O'''》播放期間，騰訊視頻全集上架前作《'''假面騎士Wizard'''》至《'''假面騎士Build'''》的雙語版本（除《'''假面騎士鎧武'''》已於2018年8月上架外）。

: 超級戰隊系列方面，中國大陸只播出《[[百獸戰隊牙吠連者|百獸戰隊牙吠連者]]》，該作也是中國大陸唯一引進的一部超級戰隊系列並在電視台播出的作品。
