《'''超級機器人大戰UX'''》（{{lang-ja|スーパーロボット大戦UX}}）為[[南夢宮萬代遊戲|南夢宮萬代遊戲]]以[[帕布雷斯特|帕布雷斯特]]名義發售的[[超級機器人大戰系列|超級機器人大戰系列]]遊戲。簡稱「機戰UX」「SRWUX」。口號為「'''所有的可能性，聚集於此吧'''」。

== 概要 ==
本作為[[Nintendo_3DS|Nintendo 3DS]]上第1款超級機器人大戰系列遊戲（連派生作在內是第8款）。亦是首次於[[任天堂|任天堂]]的[[攜帶型遊戲機|攜帶型遊戲機]]上有為參戰作品配音，在戰鬥畫面中運用到主機N3DS的裸眼3D功能。另外，本作亦是首款引入[[追加下載內容|追加下載內容]]（DLC）的機戰作品，玩家可以通過付款購買小遊戲來贏取道具。

'''故事全52話（連序章分支共74關）'''，劇情亦由三部份構成，各有每部的標題。本作標題的「UX」為故事中傭兵組織 Unknown-Xtrikers的縮寫。

=== 解說 ===
連關連作在內共16個作品。系列初參戰有《'''機神咆吼'''》、《'''蒼穹之戰神 HEAVEN AND EARTH'''》、《'''麟光之翼'''》、《'''「電腦戰機 VIRTUAL-ON」系列 飛燕HD'''》、《'''劇場版 機動戰士GUNDAM00 -A wakening of the Trailblazer-'''》、《'''SD GUNDAM三國傳 BraveBattleWarriors'''》、《'''魔神皇帝SKL'''》、《'''HEROMAN'''》8個作品。

《武裝機甲》雖然版權上顯示為電視動畫版，但從官方網站特別標記它為「（原作漫畫版）」可見這次參戰是以原作漫畫版的角色及機體設定為基準。只是因為遊戲內角色的聲優及BGM的關係才使用電視動畫版版權。

因《SD GUNDAM三國傳 BraveBattleWarriors》的參戰，使得本系列第一次有[[SD_GUNDAM系列|SD GUNDAM系列]]與（搭乘型的）GUNDAM系列作品同場共演。

《「電腦戰機 VIRTUAL-ON」系列 飛燕HD》是《電腦戰機 VIRTUAL-ON》系列及語音合成軟件[[初音未來|初音未來]]而展開的合作角色，推出過插圖及動態figure。只有單一角色參戰的登場作品參戰是系列首次。[[世嘉|世嘉]]的《電腦戰機 VIRTUAL-ON》系列製作人：亙重郎將負責飛燕HD的故事線。

《機神咆吼》為[[日本成人遊戲|18禁遊戲]]《[[斬魔大聖|斬魔大聖]]》的全年齡動畫改編作，因此本作亦是首部18禁遊戲改編作品參戰的系列作。這亦是繼《[[超級機械人大戰MX|超級機械人大戰MX]]》的《[[冥王計劃傑歐萊馬|-{zh-cn:冥王计划志雷马; zh-tw:冥王計劃傑歐萊馬; zh-hk:冥王計劃志雷馬; }-]]》和《[[超級機械人大戰L|超級機械人大戰L]]》的《[[戰鬥吧！！伊庫薩1|戰鬥吧！！伊庫薩1]]》再有18禁原作改編動畫參戰。

《[[魔神Z|-{zh-hans:魔神Z;zh-hk:鐵甲萬能俠;zh-tw:無敵鐵金剛;}-]]》系列的主角兜甲儿第一次缺席超级机器人大战系列版权作。

== 系統 ==
本作繼承了「[[超級機器人大戰K|超級機器人大戰K]]」和「[[超級機器人大戰L|超級機器人大戰L]]」的小隊系統，玩家可使用2架機體組成小隊作戰，小隊成員可以作出援護攻擊和援護防禦。至於單人機體亦可以使用連續攻擊和全體攻擊。

另外，本作亦有以下系統：
; 戰術指揮
: 和「[[第2次超級機器人大戰Z|第2次超級機器人大戰Z]]」中[[Code_Geass角色列表#主要人物|-{zh-hans:鲁路修;zh-hk:魯魯修;zh-tw:魯路修;}-]]（ZERO）的能力名稱相同，但實際內容有異。玩家可以在整备画面中选定特定的角色作為战术指挥官，发动特殊效果。每名战术指挥官都會有獨特的特殊效果，而这个效果基本都是覆盖我方全员的。

; 特殊行动指令
: 特定的机体拥有能在战术地图上给周围机体带来特殊效果的「特殊行动指令」，重現原作中的特殊演出。使用指令之后能带来各种各样的效果，例如提高我方命中率、使範圍內的敵人停止行動一回合等。如果能熟练掌握，就会使战斗进行的更加有利。

; 技能芯片
: 过关或者击退特定的强敌之后，能够取得名为「技能芯片」的消耗型物品。通过使用技能芯片，可以提升驾驶员擅长的能力。

=== 原創機体 ===
;{{nihongo|Riot B|ライオット B}}
: 駕駛者：艾利亞斯·貝魯祖
: 地球聯邦軍的次期主力兵器，也是最終測試中的機動兵器「Riot」上搭載了進戰用裝備的狀態，B 則是有「領導型」的意思。機上搭載被稱為「陽子發電機」的新型動力引擎，有著其他機動兵器所沒有的安定性與高出力。

;{{nihongo|萊拉斯|ライラス}}
: 駕駛者：沙雅·克魯格
: 在 Unknown-Xtrikers 中是以一次性的支援機來作運用。似乎被設計成與歐爾菲斯同時運用，與歐爾菲斯同樣搭載著「輕子引擎（レプトン・ベクトラー）」。雖然是支援機但性能高，是一架當作戰鬥爆擊機來運用也可充分發揮性能的機體。

;{{nihongo|歐爾菲斯|オルフェス}}
: 駕駛者：理查德·克魯格
: Unknown-Xtrikers 的先鋒，來源不明的機動兵器。搭載跟陽子發電機同系統的「輕子引擎」動力引擎，控制上的難度比較高。另外，與萊拉斯之間的連攜技「Hell Stringer（ヘル・ストリンガー）」雖然有著一擊必殺的破壞力，但使用時必須兩架的輕子引擎同時達到臨界點，因此很要求操縱技術。
