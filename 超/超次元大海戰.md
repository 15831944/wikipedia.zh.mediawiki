《'''超次元大海戰'''》為[[中華人民共和國|中國]][[江蘇甲子網路科技|江蘇甲子網路科技]]於2018年研發的{{Link-ja|艦船擬人化|艦船擬人化|艦娘}}主題手機遊戲。本遊戲於2017年開始開發<ref>{{cite web |url=http://news.4399.com/xinwen/ccydhz/m/787793.html |title=GO 舰娘军！可萝可御《超次元大海战》视频首曝 |author=蘋果 |date=2017-08-30 |website=4399手機遊戲網 |publisher= |access-date=2019-03-10 |quote=}}</ref>，歷經[[2018年中國大陸遊戲版號凍結|2018年中國大陸遊戲版號凍結]]後，正式於2019年1月22獲得遊戲版號<ref>{{cite web |url=https://gamebiz.jp/?p=229901 |title=中国政府当局、オンラインゲームの新規認可タイトルを発表　テンセントやネットイースは確認できず |author=<!--Staff writer(s); no by-line.--> |date=2019-01-23 |website=Social Game Info|publisher=Social Game Info |access-date=2019-03-10 |quote=}}</ref>，並規劃在同年3月進行最終上線前測試<ref>{{cite web |url=https://www.taptap.com/app/91077 |title=超次元大海戰 |author=<!--Staff writer(s); no by-line.--> |date=2019-01-22 |website=[[TapTap_(遊戲社區)|TapTap]] |publisher=次元遊戲社 |access-date=2019-03-10 |quote=}}</ref>。

本遊戲在2019年2月28日起，搶先於日本實施{{Link-en|事前登錄|Pre-order}}，並於2019年4月26日在日本正式上市<ref>{{cite web |url=https://twitter.com/syugopro/status/1101007011746000896 |title=事前登録開始 |author=<!--Staff writer(s); no by-line.--> |date=2019-02-28 |website=Twitter |publisher=Willoo Entertainment |access-date=2019-03-10 |quote=}}</ref>。另外除了預定發行簡體中文版、日文版以外，遠程將規劃發行國際英文版、港澳台繁體中文版<ref>{{cite web |url=http://www.87g.com/zixun/87571.html |title=《超次元大海战》正式签约欧美和港澳台地区 |author=才宝网络 |date=2018-12-13 |website=87G.com |publisher= |access-date=2019-03-10 |quote=}}</ref>。

== 簡介與特色 ==
玩家除了在本遊戲操作各式各樣的艦艇，以[[戰略角色扮演遊戲|SRPG]]的方式與敵艦交戰外，也結合了卡牌蒐集、基地建造等特色<ref>{{cite web |url=https://news.qoo-app.com/post/122274/ |title=TpGS 2018：《超次元大海戰D》、《精靈王座》兩款作品現身 QooApp攤位 |author=<!--Staff writer(s); no by-line.--> |date=2018-01-13 |website=Twitter |publisher=QooApp |access-date=2019-03-10 |quote=}}</ref>，一邊攻城掠地、一邊厚植實力，壯大玩家的海上戰力與與聲勢。

另外，玩家也可以將戰鬥過程中所蒐集到的各項道具和武器，進行高度自由化的艦艇改裝，配備強大火力和頂尖科技殲滅敵艦<ref>{{cite web |url=https://twitter.com/guardian_materi/status/1102872667164565505 |title=カッシングはマハン級駆逐艦だから マハンと似てるんだよ～！ |author=<!--Staff writer(s); no by-line.--> |date=2019-03-05 |website=Twitter |publisher=Willoo Entertainment |access-date=2019-03-10 |quote=}}</ref>。

== 故事大綱 ==
本遊戲設定在一個大型災難肆虐過後的世界，人類為求生存，藉由結合[[人工智慧|人工智慧]]和[[機器人|機器人]]技術產生出人形戰艦；但其中部分人形戰艦的人工智慧開始失控，對著創造他們的人類展開反撲……<br/>
因此倖存的人類為喚醒了一群稱作「'''守護者'''」、仍忠誠於人類的的人形戰艦，挺身保衛人類的世界免受機器人的威脅……
