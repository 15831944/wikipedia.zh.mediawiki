{{vgname|莎木 一章 橫須賀|ja=シェンムー 一章 横須賀|en=Shenmue|diff=yes}}是由[[SEGA|SEGA]]開發的[[動作冒險遊戲|動作冒險遊戲]]<ref name="mobygames">[http://www.mobygames.com/game/dreamcast/shenmue ''shenmue''] game overview at [[MobyGames|MobyGames]]</ref>，製作人為[[铃木裕|铃木裕]]。本游戏是[[莎木系列|莎木系列]]的首作，于1999年12月29日在[[Dreamcast|Dreamcast]]平台发行。<ref name="gamefaqs">{{cite web | url = http://www.gamefaqs.com/dreamcast/198621-shenmue/data| title = Release information | publisher = [[GameFAQs|GameFAQs]]| date = | accessdate = 2015-09-11}}</ref>

2018年8月21日，SEGA發行了《莎木 I&II》的HD版合輯，共計有[[PS4|PS4]]、[[Xbox_One|Xbox One]]與[[个人电脑|PC]]版本，並有官方中文化支援<ref>{{cite web |url=https://gnn.gamer.com.tw/7/161367.html |title=確定中文化！《莎木 I & II》經典合輯 2018 年登陸 PS4、Xbox One 與 PC 平台 |publisher=[[巴哈姆特電玩資訊站|巴哈姆特 GNN新聞]] |date=2018-04-14 |accessdate=2018-04-15 |archiveurl=https://archive.today/20180415131734/https://gnn.gamer.com.tw/7/161367.html |archivedate=2018年4月15日 |deadurl=no |language=zh-Hant |df= }}</ref><ref>{{cite web |url=https://www.polygon.com/2018/4/13/17237006/shenmue-1-2-hd-rerelease-ps4-xbox-one-pc |script-title=en:Shenmue and Shenmue 2 coming to PS4, Xbox One and PC |trans_title=莎木與莎木II將登錄PS4、Xbox One以及PC平台 |publisher=[[Polygon|Polygon]] |date=2018-04-13 |accessdate=2018-04-15 |archiveurl=https://web.archive.org/web/20180415132117/https://www.polygon.com/2018/4/13/17237006/shenmue-1-2-hd-rerelease-ps4-xbox-one-pc |archivedate=2018-04-15 |deadurl=no |language=en }}</ref>。

== 故事簡介 ==
故事起始於1986年11月，玩家操控的主角為高中生芭月涼，自小和武道家的父親學習[[柔術|柔術]]；涼在18歲生日當天親眼目睹了父亲遭到神秘的武術高手殺害，主線劇情即以追兇為中心展開。

== 評價 ==
遊戲採用了一種製作團隊命名為FREE (Full Reactive Eyes Entertainment)的新遊玩模式，這創意即為現今常見「[[開放世界|開放世界]]」型態遊戲的先驅者<ref>http://www.hkgnews.com/32805/</ref>，諸多如實時性、天氣系統等元素至今在其他作品中也依然可見。遊戲中的多種互動性物件與擁有各自行動模式的大量[[NPC|NPC]]架構了一個更具探索感的空間；而超過300人的NPC皆有人聲配音<ref>https://jp.ign.com/shenmue12/23814/feature/iii</ref>，這在當時的遊戲開發上實屬極為少見的方式。

莎木的許多遊戲時間是在[[开放世界|开放世界]]内进行探索、检查物品和探問NPC来获得线索，進而推動情節的發展；而主線之外也提供諸多小遊戲與街機讓玩家打發時間。另外，[[QTE|QTE]]系統的名稱即來自本作，雖然此遊玩方式並非莎木原創，但在作品中有了長足的進步與推廣，故本作命名的QTE名稱得以成為通稱。

本作在發售當時為電子遊戲史上製作費用最為昂貴的作品，製作人鈴木曾表明開發費達50億日圓（4700萬美元）程度<ref>https://game.watch.impress.co.jp/docs/news/431147.html</ref>，此紀錄維持了超過8年後才遭刷新。游戏收到的正面评价表扬了3D图像、音樂、现实感等部分，但也有著对推進节奏緩慢、部分配音品質的批评。

《莎木 一章 橫須賀》全球累計售出120万份<ref>[https://news.microsoft.com/2001/10/11/microsoft-announces-leading-sega-games-for-xbox/ Microsoft Announces Leading Sega Games for Xbox]</ref>，但这个數字並無法回收投入的成本，即使有諸多創新的遊戲系統得到了好評，在商業上《一章》仍被視為是一个失败之作。
