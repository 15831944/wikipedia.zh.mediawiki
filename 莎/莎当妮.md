'''霞多丽'''（法语：'''Chardonnay'''，发音：[ʃaʁ.dɔ.nɛ]），亦译'''莎当妮'''，属欧亚种，原产自[[法国|法国]][[勃艮第|勃艮第]]，现在在全世界范围内广泛种植号称葡萄皇后。可以酿造变化多端的白葡萄酒。

== 历史－发展 ==
该品种历史悠久,在勃艮第南部的马贡产区（maconnais）被认为是霞多丽的原产地，古时候人们错误的认为其亲本为黑比诺和melon de bourgogne，也有人认为是白比诺的变种，直到20世纪末美国加州大学戴维斯分校的研究团队通过DNA技术才认定出它的真正亲本是黑比诺和gouais（一种白葡萄），并且它属于Noiriens一族。

截至1994年霞多丽共培育出340个克隆系。

== 别名 ==
莎当妮是其音译名称广泛应用于港澳地区。大陆统称霞多丽。臺灣多稱夏多內。在国外他的名称较复杂：arboisier、aubain、auxois、blanc de champagne、pinot blanc  chardonnay。

== 种植 ==
嫩叶呈铜色，成叶5瓣、色深、较大、叶裂浅。果实较小、圆形。适宜短剪枝。

对霜霉病抗性差、成熟后对灰霉较敏感。

成熟早（一期），在冷凉气候下种植通常表现出温带水果气味，如[[梨|梨子]]。在炎热气候种植通常会出现热带水果味，如[[水蜜桃|水蜜桃]]。

在石灰岩、火山岩较适宜、喜欢肥沃土壤，不耐旱。

== 酿造 ==
霞多丽多变性强，适宜陈酿、可做干酒、起泡酒、甜酒。

传统酿造干白的时候霞多丽常常通过苹果酸乳酸发酵增加复杂度并降低酸度、再放入橡木桶中陈酿增加厚实感。

现代一些清新风格的霞多丽则会更多保留果香，不苹乳发酵也不入桶陈酿。

霞多丽也是香槟产区法定品种之一，可以酿造blanc de blanc （白中白）香槟。

== 香气 ==
我们形容霞多丽为一块白布，你把它种到哪里就会涂上当地的特色，所以霞多丽香气复杂多变能很好的突出风土和气候特色。

果香为主[[柠檬|柠檬]]、[[香蕉|香蕉]]、[[桃子|桃子]]、[[芒果|芒果]]，如果入桶的还会有[[杏仁|杏仁]]、[[胡桃|核桃]]、[[奶油_(消歧义)|奶油]]、[[香草_(消歧义)|香草]]、[[蜂蜜|蜂蜜]]等香气

== 世界各地 ==
在法国霞多丽的主要种植区市香槟区和勃艮第，近年来种植面积不断扩大从1958年的7300公顷到2000年的30000公顷。

德国瑞士意大利北部都有种植。

美国的种植面积在25000公顷。

澳大利亚超过5000公顷。

南非、智利、阿根廷、新西兰或多或少都有霞多丽的身影。

於1951年由[[匈牙利|匈牙利]]引入[[中国|中国]]，2014年在[[宁夏回族自治区|宁夏]][[贺兰山东麓产|贺兰山东麓产]]区随着法国[[酩悅·軒尼詩－路易·威登集團|酩悦轩尼诗]]集团投资的起泡酒厂的投产，霞多丽在宁夏已成为最著名的白葡萄品种。

是目前全世界最受欢迎的白色[[酿酒葡萄|酿酒葡萄]]。

Chardonnay的魅力在于其多变的风格和广泛的适应性，无论在任何气候条件下，她的质量还都不错。从寒凉的[[法国|法国]]北部[[沙布利|沙布利]]到炎热的[[阿根廷|阿根廷]]Mendoza和南[[澳洲|澳洲]][[巴羅莎山谷|巴羅莎山谷]]，霞多丽可以呈现从带有柑橘类水果到充满热带水果和异国情调的不同风姿。她也是[[香槟|香槟]]和氣泡酒调和成分中最重要的一份子。
