'''衛理中學'''（{{lang-en|The Methodist Church Hong Kong Wesley College}}）為一間由[[香港基督教循道衛理聯合教會|香港基督教循道衛理聯合教會]]在1977年開辦的[[學校|學校]]，始於[[北角|北角]]，是政府津貼全日制男女校。於1991年獲[[教育署|教育署]]遷校到校現址[[小西灣道|小西灣道]]33號。學校於1998年轉為[[母語教學|母語教學]]，2010學年開始於中一改回主要以英語教學，初中每級有三班以全英語教學（彈性班），一班主要為[[母語教學|母語教學]]（非彈性班）。

== 教學語言 ==
2010年之前

初中：
* 所有科目（中文、英文、普通話、中國歷史除外）大部份內容以中文教授，少部份內容以英文教授
* 中文、普通話、中國歷史以中文教授
* 英文以英文教授
高中：
* 中文、通識以中文教授
* 英文以英文教授
* 數學及所有選修科均讓學生自行選擇教學語言（中文或英文）

2010年或之後

初中：
* 其中三班以全英語教學（中文、中國歷史除外）
* 另一班主要為母語教學
高中：
* 中文、通識以中文教授
* 英文及數學以英文教授
* 所有選修科均讓學生自行選擇教學語言（中文或英文）

== 校園環境 ==
全校建築面積5600平方米，校舍屬標準型中學，樓高7層，有標準課室27間、特別室17間，並設有兩個禮堂、小食部、學生會辦公室、英文屋(WE Lab)、中文教室、音樂室、2間電腦室、多媒體教學中心、家政室、設計室、地理室、生物實驗室、綜合科學實驗室、物理實驗室、化學實驗室、圖書館及自修室等設施。校舍對面有[[福建中學(小西灣)|福建中學(小西灣)]]，左側則有[[嶺南衡怡紀念中學|嶺南衡怡紀念中學]]。
校舍面向將軍澳對外海域，環境優美，讓學生在一個優良的環境下學習。2012年暑假進行粉飾外牆等工程。

== 行政長官《施政報告》學生諮詢 ==
2008年9月26日，中學生施政報告諮詢會選址小西灣衛理中學。除衛理中學外，並邀請區內三間中學，分別[[嶺南衡怡紀念中學|嶺南衡怡紀念中學]]、[[福建中學(小西灣)|福建中學(小西灣)]]及[[中華基金中學|中華基金中學]]的四百多名師生就民生、教育、環保及經濟議題上向該屆行政長官曾蔭權發表意見。
