'''衛仲樂'''（1909年3月21日-1997年4月7日<ref>{{Cite web |url= http://www.shtong.gov.cn/newsite/node2/node4/node2249/node75735/node75777/node75840/node75851/userobject1ai91843.html|title=【卫仲乐】（1909.3～1997.4.7） |accessdate=2014-06-28 |work= |publisher=上海市地方誌辦公室}}</ref><ref name="民族音樂">{{Cite web|url=http://www.ewen.cc/sephnew/bkview.asp?bkid=23128&cid=39033|title=民族音乐家卫仲乐先生|accessdate=2014-06-28|work=|publisher=上海教育出版社|date=2002-08-09}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>）原名'''秉濤'''、又名'''崇福'''，是中國著名[[民族樂器|民族樂器]]演奏家，擅[[琵琶|琵琶]]，被稱爲“琵琶大王”，此外還通[[古琴|古琴]]、[[二胡|二胡]]、[[簫|簫]]及[[笛|笛]]，也工於[[小提琴|小提琴]]。衛仲樂也擔任過二至六届[[全国政协委员|全国政协委员]]。

==生平==
衛仲樂於[[宣統|宣統]]元年二月廿一（1909年3月21日）出生於[[上海|上海]]，生父姓殷，是一名碼頭工人。他是父親九個子女中的老幺，因家境貧寒，其生父將他賣給了一名姓衛的寡婦，因而隨母姓。後者給他取名秉濤、又名崇福。衛家亦不富庶，因而自小就被送出去當學徒。然而即使在這種境地中，衛秉濤依然堅持在夜間抽空自學簫和笛。<ref name="民族音樂"/>

1929年上海[[大同樂會|大同樂會]]招收成員，衛秉濤被選中。後來他得到當時大同樂會的樂務主任[[鄭覲文|鄭覲文]]的賞識，授其古琴，並爲他改名'''仲樂'''。1933年在[[上海商會|上海商會]]籌資、[[明星影片公司|明星影片公司]]拍攝的紀錄片中以琵琶演奏《[[十面埋伏_(樂曲)|十面埋伏]]》，又以琵琶領奏《春江花月夜》。同年5月21日的上海[[大光明電影院|大光明電影院]]開幕晚會上，再次琵琶獨奏《十面埋伏》。1935年2月，鄭覲文逝世，衛仲樂以副主任之職接任大同樂會樂務主任。同年加入[[沈知白|沈知白]]等建立的[[中國音樂研究會|中國音樂研究會]]。<ref name="古曲網">{{Cite web |url= http://info.guqu.net/pipamingjia/20060709212030_384.html|title=“琵琶大王”卫仲乐|accessdate=2014-06-28 |work= |publisher=中國古曲網|date=2006-07-09}}</ref><ref name="民族音樂"/>

[[抗日戰爭|抗日戰爭]]爆發後，衛仲樂在1938年7月受[[香港妇女救济兵灾会|香港妇女救济兵灾会]]邀請，與[[許光毅|許光毅]]、[[罗松泉|罗松泉]]等一起到香港義演，8月6日至7日連續演出兩場，所得款項皆賑濟災民。次年10月應[[國際紅十字會|國際紅十字會]]成員谭霭芙之邀加入[[中國文化劇團|中國文化劇團]]，與[[孫裕德|孫裕德]]、許光毅一起前往美國義演，籌集善款以賑濟中國災民。歸國之前由[[Musioaff|Musioaff]]公司錄製了四張八首琵琶曲，在美國發行。<ref name="民族音樂"/>

1940年2月下旬回國，通過沈知白的介紹，衛仲樂成爲[[滬江大學|滬江大學]]國樂教授。此時大同樂會因鄭覲文之子郑玉荪和戰爭干擾，已不復存在，因而衛仲樂在其原址建起“仲樂音樂館”。1941年與[[金祖禮|金祖禮]]、许光毅等人一起成立[[中國管絃樂隊|中國管絃樂隊]]。1949年成爲[[上海音樂學院|上海音樂學院]]教授，1956年成爲該校民樂系副主任，後來又成为主任<ref name="民族音樂"/>。1962年擔任[[中国音乐家协会|中国音乐家协会]]上海分会的副主席<ref name="古曲網"/>。

[[文化大革命|文化大革命]]爆發之後，衛仲樂被當作“反動學術權威”打倒，其弟子[[屠偉剛|屠偉剛]]買通看守人員才得以保證其正常起居。<ref>{{Cite web |url= http://www.people.com.cn/GB/paper39/3470/438108.html|title=一门琵琶音  留伴彩云飞|accessdate=2014-06-28 |work= |publisher=人民網|date=2001-05-31}}</ref>
