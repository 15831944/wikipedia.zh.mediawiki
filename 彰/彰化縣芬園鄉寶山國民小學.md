'''彰化縣芬園鄉寶山國民小學'''簡稱為'''寶山國小'''，位於[[臺灣|臺灣]][[彰化縣|彰化縣]][[芬園鄉|芬園鄉]]，乃該鄉之縣立[[小学|國民小學]]。

== 校史沿革 ==
* 1947年：8月1日自[[彰化縣芬園鄉芬園國民小學|芬園國民學校]]增設'''芬園國民學校溪頭分校'''。
* 1959年：8月7日受[[八七水災|八七水災]]影響，芬園國民學校受災嚴重。張水林、陳金桔等地方人士發動捐款，加上鄉公所補助80萬元、[[臺灣省政府|省政府]]補助30萬元，購得現在校地以興建校舍。
* 1960年：2月1日獨立為'''彰化縣寶山國民學校'''。
* 1968年：6月1日因實施[[台灣九年國民義務教育|九年國民義務教育]]，校名改成'''彰化縣芬園鄉寶山國民小學'''。

== 歷任校長 ==
* 2011-2017 鄭孟忠
  2017-    陳泰佑

== 學區 ==
寶山國小的學區包含該鄉之縣庒村、溪頭村、圳墘村等。
