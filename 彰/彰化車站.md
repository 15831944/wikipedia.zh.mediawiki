'''彰化車站'''位於[[台灣|台灣]][[彰化縣|彰化縣]][[彰化市|彰化市]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[西部幹線|西部幹線]]的[[鐵路車站|鐵路車站]]，也是[[縱貫線_(南段)|縱貫線南段]]、[[海岸線_(臺鐵)|海線]]、[[臺中線|山線]]的起訖車站。本站為配合[[臺鐵捷運化|臺鐵捷運化]]政策，2011年9月26日[[人行天橋|跨站人行天橋]]正式啟用。高架化改建規劃中<ref>{{Cite web|url=http://www.howone.com.tw/railway/news.php?uid=12&id=16163|title=彰化站高架化 地方達成高架共識|accessdate=2014-11-30|publisher=台灣鐵道網|date=2011-09-23|deadurl=yes|archiveurl=https://web.archive.org/web/20110926003146/http://www.howone.com.tw/railway/news.php?uid=12&id=16163|archivedate=2011-09-26}}</ref><ref>{{Cite web|url=http://www2.chcg.gov.tw/main/main_act/main.asp?main_id=4266&act_id=26|title=彰化鐵路高架化|accessdate=2014-11-30|publisher=彰化縣政府|date=2014-10-16}}</ref>。

== 車站構造 ==
*車站大門往山側（東向）開門。
*旅遊服務中心。
*目前於站房南側有天橋通往後站，並於2010年2月在站房北側興建新的跨站式天橋。
*站內有[[島式月台|島式月台]]兩座（第二，三月台）、[[側式月台|側式月台]]一座（第一月台），共三座月台，其間以天橋相連接；
**第一月台與第二月台中間包夾一股道（[[西班牙式月台佈局|西班牙式月台佈局]]），大部分停靠南下列車，但由於台鐵無法同時開啟兩側車門，列車門只開第一月台側。
**第二月台與第三月台中間包夾二股道，第二月台常作為海線列車發車、待避、誤點調度或是供行包專車使用，第三月台主要為北上列車或到發列車使用。

== 歷史 ==
[[File:Changhua_yi.jpg|thumb]]
*1905年3月26日：增設「彰化驛」。
*1918年：第二代站房完工。
*1922年：興建[[彰化扇形車庫|扇形車庫]]與跨站天橋。
*1958年12月10日：現今彰化站房完工<ref>{{cite book |author = 戴震宇 |authorlink = |title = 《台灣的老火車站》 |publisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>。
*1959年6月15日：現今彰化站房啟用，同時在站內附設鐵路餐廳（已停業）。
*2004年1月19日：指定扇形車庫為縣定古蹟。
*2005年5月27日：扇形車庫整修完畢。
*2008年2月25日：為加強疏運需要，開始停靠[[太魯閣列車|太魯閣號]]。
*2011年9月26日：跨站人行天橋(無障礙電梯[[File:Handicapped_mark.svg|20px]])正式啟用<ref>{{cite news|title=彰化車站跨站人行天橋　月底驗收後正式啟用|url=http://www.nownews.com/n/2012/01/08/435273|publisher=今日傳媒|date=2012-01-08|accessdate=2014-11-30}}</ref>。
*2014年4月29日：配合太魯閣號維修需要，開始停靠[[普悠瑪列車|普悠瑪號]]。
*2015年6月30日：啟用多卡通刷卡機。
*2015年10月15日：改點後普悠瑪號取代原來太魯閣號，也開始停靠普悠瑪號。
*2016年4月21日：改點後太魯閣號再次開行西部幹線。
