'''彰化縣立西勢國民小學'''（{{lang-en|Sishih Elementary School}}），位在[[台灣|台灣]][[彰化縣|彰化縣]][[福興鄉_(臺灣)|福興鄉]]中央北端，北鄰[[福鹿溪|福鹿溪]]、快速道路，西側面臨員鹿路。學區包含：西勢村、同安村、福興村、橋頭村全部及社尾村7—10鄰，福南村7—14鄰。<ref name="introduction">{{Cite web |url = http://163.23.104.38/%E8%A5%BF%E5%8B%A2%E5%9C%8B%E5%B0%8F2.htm |title = 學校簡介 |author =  |date =  |publisher =  |language = zh-tw |accessdate = 2016-10-04 |quote =  }}</ref>

學校除基本硬體設施外，還設有中央監控系統、視聽教室、樂活運動站<ref name="sports_station" />和鄉土文物館<ref name="local_cultural_relics_museum" />。另外還設置有志工隊<ref>{{Cite web |url = http://cvris.chcg.gov.tw/chvol/home.jsp?serno=200904110005&mserno=200904110005&menudata=VolMenu&level2=Y&language=chinese&contlink=ap/volTeam_view.jsp&assnserno=200912300004 |title = 彰化縣志願服務團體 - 彰化縣志願服務與資源整合資訊網 |author =  |date =  |publisher =  |language = zh-tw |accessdate = 2016-10-05 |quote =  }}</ref>、晨光媽媽等，在上下學時協助指揮交通、早自修時間陪伴小孩子，以及進行社區服務等等來回饋社會。

==沿革==
* 民國38年（1949年）6月15日：奉府令（府教字第一一六七四號）准設立[[彰化縣福興鄉管嶼國民小學|管嶼國民學校]]西勢分班
* 民國38年（1949年）10月11日：遷入現址上課
* 民國39年（1950年）4月10日：奉令附設幼稚班開課
* 民國40年（1951年）8月21日：奉縣府令准予獨立為西勢國民學校
* 民國40年（1951年）9月10日：奉令啟用校鈴並派施天興先生為首任校長
* 民國40年（1951年）9月15日：本校開校典禮（中秋日）
* 民國52年（1963年）7月3日：奉令附設幼稚班停辦
* 民國57年（1968年）8月1日：奉令改制為彰化縣福興鄉西勢國民小學
* 民國64年（1975年）8月1日：奉令學校行政組織改為教務、訓導、總務三處
* 民國78年（1989年）3月29日：新校門落成
* 民國78年（1989年）8月1日：奉令設特殊教育班一班
* 民國79年（1990年）6月30日：後門及圍牆整建完成
* 民國80年（1991年）11月24日：教室、廁所、網球場<ref>{{Cite web |url = http://iplay.sa.gov.tw/Gym/Index/13319 |title = 西勢國小網球場(館) 全國運動場館資訊網-iPlay |author =  |date =  |publisher =  |language = zh-tw |accessdate = 2016-10-06 |quote =  |deadurl = yes |archiveurl = https://web.archive.org/web/20160829093015/http://iplay.sa.gov.tw/Gym/Index/13319 |archivedate = 2016年8月29日 |df =  }}</ref>落成
* 民國83年（1994年）2月15日：增設中央監控系統及飲用水設備
* 民國84年（1995年）10月23日：增設視廳教室及主控室
* 民國85年（1996年）11月1日：自動化電腦控制廁所用水設施完成
* 民國89年（2000年）6月2日：縣府補助改善教學環境及設備-PU跑道工程全部完成
* 民國92年（2003年）4月：本校辦理92年度福興鄉兒童博覽會活動
* 民國95年（2006年）3月24日：[[中華民國內政部|內政部]]補助本校城鄉新風貌-鄉土文物館及周邊景觀改善工程完工<ref name="local_cultural_relics_museum">{{Cite web |url = http://www.chcg.gov.tw/ch/03news/01print.asp?bull_id=34365 |title = 西勢國小５５週年校慶暨鄉土文物館啟用典禮 |author =  |date =  |publisher =  |language = zh-tw |accessdate =  |quote =  }}</ref>
* 民國97年（2008年）1月16日：[[中華民國教育部|教育部]]補助本校設置樂活運動站落成完工<ref name="sports_station">{{Cite web|url =http://www.lohas-sport.org.tw/school_K_002.html|title =樂活運動站 - 學校運動站介紹 /彰化縣西勢國小|author =|date =|publisher =|language =zh-tw|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20161005115110/http://www.lohas-sport.org.tw/school_K_002.html|archivedate =2016-10-05}}</ref>
