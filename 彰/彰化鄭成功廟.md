'''彰化鄭成功廟'''位於臺灣[[彰化縣|彰化縣]][[彰化市|彰化市]]，於民國一百零一年（2012年）5月23日公告為彰化縣歷史建築{{r|文}}。該寺廟主祀[[鄭成功|鄭成功]]，有部分建築構件（如三川殿、兩廊的楹柱、石珠等）是原彰化[[白沙書院|白沙書院]]拆解後拿來再利用{{r|文}}。臺灣早期的鄭成功廟（如臺南[[延平郡王祠|延平郡王祠]]等）大多已改建，相較之下彰化鄭成功廟仍大致保有原貌{{r|彰縣史建}}。

== 沿革 ==
彰化鄭成功廟的歷史可追溯至日大正六年（1917年），當時鄭龍與鄭清標二人隨[[彰化天后宮|彰化天后宮]]前往[[湄州天后宮|湄州天后宮]]進香後，又到福建南安石井鄭成功家廟參拜並迎回神像一座，回臺後暫祀於彰化天后宮{{r|鄭}}{{r|彰縣史建}}。後來鄭維仁等人發起成立宗親會，並打算興建祠堂，獲宗親多人的一致贊同{{r|鄭}}{{r|彰縣史建}}。於是該祠堂（即鄭成功廟）經募款籌建後，於昭和3年（1928年）開始動工，落成於昭和13年（1938年）{{r|文}}{{r|彰縣史建}}。

二次大戰之後，於民國六十八年（1979年）成立財團法人彰化鄭成功廟管理組織予以管理，之後廟宇曾於民國七十五年（1986年）修建三川殿，十一年後（1997）年整修屋頂，而民國九十七年（2008年）又再次進行整修{{r|文}}。

== 建築 ==
彰化鄭成功廟座西向東，廟前有寬廣廟埕{{r|彰縣史建}}。廟宇面寬七開間，格局相當於兩進的四合院{{r|彰縣史建}}。該廟正殿供奉有鄭成功神像，此外還有財神爺與鄭家歷代祖先神位{{r|彰縣史建}}。東西兩廡則是供有鄭姓宗族長生祿位、功德牌位，而三川門內兩側供有甘輝將軍與萬禮將軍{{r|彰縣史建}}。
