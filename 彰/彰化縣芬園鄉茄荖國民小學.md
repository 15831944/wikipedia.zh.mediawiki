'''彰化縣芬園鄉茄荖國民小學'''簡稱為'''茄荖國小'''，乃位於[[臺灣|臺灣]][[彰化縣|彰化縣]][[芬園鄉|芬園鄉]]之縣立[[小学|國民小學]]。

== 校史沿革 ==
* 1941年：經地方仕紳洪身長<ref>此字左身右長，因電腦無字而以此表示。</ref>耳等人爭取，4月1日芬園公學校（今[[彰化縣芬園鄉芬園國民小學|彰化縣芬園鄉芬園國民小學]]）增設'''下茄荖分教場'''。
* 1946年：2月1日下茄荖分教場獨立成'''茄荖國民學校'''。
* 1968年：6月1日因實施[[台灣九年國民義務教育|九年國民義務教育]]，校名改成'''彰化縣芬園鄉茄荖國民小學'''。

== 歷任校長 ==
* 2010年-2013年：許聖明
* 2013年迄今：石玲惠

== 學區 ==
茄荖國小的學區包含該鄉之茄荖村、嘉興村等。

== 畢業校友 ==
* 洪慶章：曾任芬園鄉鄉長。
