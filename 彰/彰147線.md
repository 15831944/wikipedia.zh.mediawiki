'''彰147線 舊館－同安宅'''，是位於[[彰化縣|彰化縣]]的一條鄉道，北起[[埔心鄉|埔心鄉]]南館村舊館聚落南側鄉道[[彰53-1線|彰53-1線]]岔路，南至[[永靖鄉|永靖鄉]]同安村、敦厚村交界處的鄉道[[彰142-1線|彰142-1線]]岔路，全長2.731公里。{{r|thb}}

== 路線說明 ==
*[[埔心鄉|埔心鄉]]：'''舊館'''（起點，鄉道[[彰53-1線|彰53-1線]]岔路）→ 新館路 → '''新館'''→ 新館路 →鄉界
*[[永靖鄉|永靖鄉]]：鄉界→ 東畔路 → '''同安宅'''（終點，鄉道[[彰142-1線|彰142-1線]]岔路）

== 沿線設施 ==
*朝南宮
*埔心鄉新館律動公園（原埔心鄉第五公墓）
*同霖宮

== 舊線 ==
===1976年===
在1976年的臺灣省公路資料統計表上，彰147線名稱為「東埔心－太平」，起點為東埔心（[[縣道148號|縣道148號]]22.260K），終點太平（鄉道[[彰82線|彰82線]]1.813K），全長1.591公里。<ref>[http://twinfo.ncl.edu.tw/tiqry/hypage.cgi?HYPAGE=search/merge_pdf.hpg&sysid=E0814369&jid=79002356&type=g&vol=66091600&page=%E9%A0%816-10 《彰化縣鄉道公路資料統計表（民國六十五年底）》，臺灣省政府公報六十六年秋字第六十七期，第9頁]</ref>

該路線在1983年二次公路總清查時，編號已變成[[彰85-1線|彰85-1線]]，名稱也微調為「埔心－太平」，全長則修正為1.560公里。<ref>[https://twroad.org/road/gazette/83ch/85-1.jpg 《台灣省鄉道公路埔心 太平路線全線略圖》，1983年12月，公路邦轉載]</ref>

其後彰85-1線起點因縣道148號改道而南移，全長也因而減少為1.548公里。

===1983年===
1983年第二次公路總清查時，彰147線名稱及路線與今相同。<ref>[https://twroad.org/road/gazette/83ch/53.jpg 《台灣省鄉道公路陝西 加錫路線全線略圖》，1983年12月，公路邦轉載]</ref>
