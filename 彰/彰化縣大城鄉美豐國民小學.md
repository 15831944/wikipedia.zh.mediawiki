'''彰化縣大城鄉美豐國小'''位於[[臺灣|臺灣]][[彰化縣|彰化縣]][[大城鄉|大城鄉]]，創校於西元1952年，當時名為'''西港國民學校美豐分校'''。該校於1964年獨立為'''美豐國民學校'''，1986改為'''豐國民小學'''並沿用至今。其學區範圍有：[[大城鄉|大城鄉]][[豐美村|豐美村]]及[[三豐村|三豐村]]。<ref>{{cite web|url= http://163.23.109.129/eweb/module/system_modules/index.php?home=mfes&sn=18&sn_ids=17&free_ids=18|title= 美豐國民小學: 學校沿革}}</ref><ref>{{cite web |author1=鐘武達 |title=大城偏鄉美豐國小陶藝傳愛 |url=https://www.chinatimes.com/realtimenews/20181009004169-260405 |website=中時電子報 |accessdate=2018-12-28}}</ref><ref>{{cite web |author1=陳冠備 |title=體驗潮間帶 山海城學童共學 |url=http://news.ltn.com.tw/news/local/paper/1091242 |website=自由電子報 |accessdate=2018-12-28}}</ref>
