'''季華園站'''是[[廣佛地鐵|廣佛地鐵]]的一座車站<ref>[http://gd.xinhuanet.com/newscenter/photo/2010-10/15/content_21135692.htm 亞運30個工作日 廣佛線免費乘坐] {{webarchive|url=https://web.archive.org/web/20131110132215/http://gd.xinhuanet.com/newscenter/photo/2010-10/15/content_21135692.htm |date=2013-11-10 }}</ref>，位於[[佛山市|佛山市]]汾江南路影荫路口南侧的地底，毗鄰[[季华公园|季华公园]]。本站在2010年11月3日啟用。

==车站结构==

===站廳===
季华园站站厅中南侧被划分为收费区，周围被行人通道以及车站商店包围，站厅收费区内设有专用电梯（[[升降机|升降机]]）、[[扶手电梯|扶手电梯]]和楼梯供乘客前往车站月台。

商店及自助服务方面，本站设置了中国银行自動櫃員機、售卖机以及好易自助终端等设施。

季华园站目前设有2个出入口，均位于汾江南路西侧的人行道上，因此需要来往汾江南路东侧的乘客仍需要过马路。同時由於車站出口附近並沒有斑馬線等設施可供橫過汾江南路，需要往南或往北繞道數百米不等的距離，為不少市民所詬病。

位於汾江南路東側的兩個出入口目前已動工，預計2018年啟用。

== 邻近车站 ==
季华园站是廣佛地鐵的中途站，乘客可以乘搭往新城東或燕崗列車，來往廣州市區和佛山市區。

== 利用情况 ==
本站周围有不少住宅区，亦临近季华公园，故本站客流主要以来往早晚通勤以及游览季华公园的客流为主，早晚客流会较为庞大，其余时间本站的客流较为稀少，令本站显得有点冷清。此外，亦有西至佛山大道中，东至岭南大道北一带的居民借助公交接驳本站。

另外，每逢農曆正月十五當晚，佛山將會舉辦傳統「行通濟」的活動，為配合當晚的交通和人流管制，前往通濟橋的乘客須利用本站或[[普君北路站|普君北路站]]出站，再沿指定路線前往通濟橋。

== 历史 ==
本站的命名来源于附近的季华公园。

2002年廣佛地鐵的走向初步確定，当时被称为'''绿景路站'''的季华园站確定為廣佛地鐵佛山段的其中一個中途站。2007年6月28日，廣佛地鐵首通段正式動工。

2010年10月28，29日，本站隨廣佛地鐵首通段全線開放給廣佛兩市的市民進行試乘。11月3日，廣佛地鐵首通段開通試運營，本站正式啟用。廣佛地鐵在開通當日起全線實施進站安檢措施，直至12月23日全線取消安檢為止。

== 未来发展 ==
{{main|佛山地铁4号线}}
佛山地鐵4号线将设置有季华园站，但广佛地铁建设之初并未预留任何換乘條件。

而根据佛山市公共资源交易中心发布的《佛山市城市轨道交通4号线一期、11号线工程防洪评价专题项目（第二次招标）》公告，4号线的季华园站将在季华公园西北角设站，和现有季华园站有一段相當可觀的距離，同時車站被標識為非換乘站，因此有可能無法與廣佛線直接實現換乘，需要出站步行再進站，或繞經其它線路，較為不便<ref>{{cite web|title=地铁4号线一期、11号线站点首次披露|url=http://www.citygf.com/fstt/201710/t20171019_109660.html|publisher=佛山新闻网|accessdate=2017-10-20}}</ref>。
