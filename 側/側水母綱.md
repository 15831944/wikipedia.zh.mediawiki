'''側水母綱'''（Dipleurozoa）是[[盾形動物門|盾形動物門]]的一個已[[絕滅|絕滅]]的[[綱|綱]]。

== 物種 ==
側水母綱其下皆為[[化石|化石]]物種，其中主要以[[狄更遜水母|狄更遜水母]]為代表，其身體呈橢圓形，兩側對稱，固著生活，主要以濾食水中的養分維生。因具其兩側對稱的特性，所以可能是[[蠕形動物|蠕形動物]]或原始[[節肢動物|節肢動物]]，但也有人認為牠和[[櫛水母動物|櫛水母動物]]有親緣關係。<ref>{{Cite web |url = http://www.baike.com/wiki/%E4%BE%A7%E6%B0%B4%E6%AF%8D%E7%BA%B2 |title = 側水母綱_互動百科language = 簡體中文 |accessdate = 2017-10-24 }}</ref>
