「'''叱咤樂壇我最喜愛的歌曲大獎'''」是[[叱咤樂壇流行榜頒獎典禮|叱咤樂壇流行榜頒獎典禮]]每年的重點獎項，由第一屆即1988年起已有設立。

此獎項賽果完全由樂迷投票決定，樂迷可由全年在叱咤樂壇流行榜曾在周榜上榜的作品中挑選1首作投票，歷經數個回合，淘汰較低票數之作品，最後由典禮當晚之現場觀眾投票選出大獎得主。

皆因此獎項是香港樂壇各頒獎禮上'''唯一一個由完全由聽眾票選的歌曲獎項'''；加上近年獎項透明度逐漸提升，會'''完全公開最後回合之投票記錄'''，所以此獎項在香港樂壇被視為極為重要的音樂獎項。

'''歌手[[謝安琪|謝安琪]]主唱的〈[[囍帖街_(歌曲)|囍帖街]]〉是史上最高終回票數的得獎歌曲，當年獲現場觀眾共3868票當選。'''

==現行投票規則==
「'''叱咤樂壇我最喜愛的歌曲大獎'''」與「'''我最喜愛的男歌手、女歌手、組合'''」獎項並列為[[香港商業電台|香港商業電台]]年度叱吒樂壇頒獎禮上四個「我最喜愛的」系列獎項。

而「'''大獎'''」的前期投票通常在每年的12月進行，屬「我最喜愛的」系列獎項中第1、2、3階段的投票（第4、5階段為「我最喜愛的男歌手、女歌手、組合」獎項投票期）。選民必須登記為商業電台叱吒903頻道官方網站——my903——認証會員（2018年起整合為881903會員），透過my903.com、Hong Kong Toolbar（手機版/網上版）或「商台節目重溫」之手機應用程式進行投票。〉<ref>[http://www.my903.com/my903/usc2014/web/last5.jsp  2014年度叱咤樂壇流行榜頒獎典官方網站 - 叱吒五強叱吒我最喜愛的] {{webarchive|url=https://web.archive.org/web/20150901011756/http://www.my903.com/my903/usc2014/web/last5.jsp |date=2015-09-01 }}</ref>

*第1階段接受聽眾從該年度903專業推介所有曾上榜的歌曲中選出最後二十強，進入第2階段投票。
*第2階段則從最後二十強歌曲中由聽眾一人一票選出最後十強，進入第3階段投票。
*第3階段由聽眾從最後十強歌曲中一人一票選出最後五強。
*最後於頒獎典禮當晚由現場觀眾即場票選最終結果。
*獲獎歌曲會歷經數個階段，而得出最後結果：

所有上榜歌曲 → 二十強 → 十強 → 五強 →「'''叱咤樂壇我最喜愛的歌曲'''」（1999－2010、2014－2015、2017）

全年播放率最高的20首上榜歌曲 → 五強 →「'''叱咤樂壇我最喜愛的歌曲'''」 （2011）

所有上榜歌曲 → 二十強 → 五強 →「'''叱咤樂壇我最喜愛的歌曲'''」（2012－2013）

所有上榜歌曲 → 二十強 → 十強 → 五強 → 兩強 →「'''叱咤樂壇我最喜愛的歌曲'''」（2016）

所有上榜歌曲 → 十強 → 五強 →「'''叱咤樂壇我最喜愛的歌曲'''」（2018）

==史上紀錄==
;最高終回票數之獲獎歌曲（只計算有公布票數之年份）
*「叱咤樂壇我最喜愛的歌曲大獎」經由網上的多回合投票得出「最喜愛歌曲5強」，交由典禮當晚現場觀眾進行終回票選，得出結果。
* 2016年度頒獎禮曾修訂終回票選準則，典禮現場觀眾將預定進行兩輪現場票選，倘若首輪票選已有作品過半數，則立即當選；否則，最高票數之2首需進行第二輪投票。唯2017年度改回原來制度。
