'''鑫茂大厦'''，位于[[北京市|北京市]][[西城区|西城区]][[金融大街|金融大街]]15号、甲15号，是[[北京金融街|北京金融街]]中的高层写字楼，是[[中国银行保险监督管理委员会|中国银行保险监督管理委员会]]的办公地点。

==简介==
鑫茂大厦项目是[[北京金融街|北京金融街]]B1地块的二期房产项目，是[[金融街控股|金融街控股]]2004年[[A股|A股]]增发募集资金的投资项目之一，分为南楼和北楼<ref name=yin/>。鑫茂大厦由中建二局三公司承建，获得2006年[[鲁班奖|鲁班奖]]，并获2008年国家级工法<ref>{{Cite web |url = http://www.xinhuanet.com/politics/2016-07/11/c_129135950.htm |title = 建筑工匠陈小茹：巾帼不让须眉 非凡楼宇筑起不平凡的人生 |author =  |date = 2016-07-11 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。

2005年，[[中国银行业监督管理委员会|中国银行业监督管理委员会]]（中国银监会）出资8.78亿元，从金融街控股手中购买了鑫茂大厦南楼（当时称西楼），合同约定金融街控股应在2006年3月31日前将完成消防验收并达到精装修标准的房产交付中国银监会使用<ref name=yin>{{Cite web |url = http://finance.sina.com.cn/stock/s/20050704/0944174121.shtml |title =中国银监会相中金融街房产项目  |author =  |date =2005-07-04  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。2006年7月28日，中国银监会发布迁址公告称，定于2006年7月28日至8月4日从北京市[[东城区_(北京市)|东城区]][[工人体育场西路|工人体育场西路]]1号和西城区[[成方街|成方街]]甲33号院两地迁入北京市西城区金融大街甲15号鑫茂大厦<ref>{{Cite web |url = http://www.gov.cn/gzdt/2006-07/28/content_348240.htm |title =  银监会迁入北京市西城区金融大街甲15号鑫茂大厦|author =  |date = 2006-07-28 |publisher =  中国政府网|language =  |accessdate =  |quote =  }}</ref>。

2005年，[[中国保险监督管理委员会|中国保险监督管理委员会]]（中国保监会）决定迁入鑫茂大厦北楼办公<ref>西蒙布线助力中国保监会，计算机网络世界2005(2):38-38</ref>。此外，2005年5月30日，中国保险监督管理委员会北京监管局迁入金融大街15号鑫茂大厦北楼10层的新址办公<ref>{{Cite web |url = http://finance.sina.com.cn/stock/y/20050530/074790681.shtml |title = 关于中国保险监督管理委员会北京监管局迁址的公告 |author =  |date = 2005-05-30 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。2018年中国银监会、中国保监会合并为[[中国银行保险监督管理委员会|中国银行保险监督管理委员会]]。
