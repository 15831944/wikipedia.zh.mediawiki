'''香港中央圖書館'''（{{lang-en|Hong Kong Central Library}}），是[[香港公共圖書館|香港公共圖書館]]系統的中樞，成立於2001年，隸屬香港特別行政區政府康樂及文化事務署，是香港公共圖書館體系中面積最大、服務範疇最廣和館藏數量最多的公共圖書館。<ref>{{cite web|url=https://www.hkpl.gov.hk/tc/hkcl/aboutus/index.html}}</ref>

==概述==
本館位於[[香港|香港]][[香港島|香港島]][[銅鑼灣|銅鑼灣]][[高士威道|高士威道]]66號，[[港鐵|港鐵]][[天后站|天后站]]與[[銅鑼灣站|銅鑼灣站]]之間，北鄰[[維多利亞公園|維多利亞公園]]；樓高12層，館藏超過2,500,000項，為香港公共圖書館系統總館藏14,350,000項的約5分之1；佔地9,400[[平方米|平方米]]，景觀可以遠望[[九龍西|九龍西]]及[[獅子山_(香港)|獅子山]]；[[總樓面面積|樓面面積]]33,800平方米；於2001年5月17日啟用，取代[[大會堂公共圖書館|大會堂公共圖書館]]成為香港公共圖書館的「總館」，以及[[灣仔區|灣仔區]]第3間公共圖書館。此外，香港中央圖書館將於2020年，在深水埗[[海達邨|海達邨]]公共圖書館/體育館綜合大樓內，另設一個7層高備用書庫連特備展覽館，樓面面積則為6730平方米，並將由房委會代為興建。

== 歷史 ==
[[File:Poster_hkcl.jpg|thumb]]
1990年代，[[市政局_(香港)|市政局]]決定興建一座[[中央圖書館|中央圖書館]]作為香港的旗艦圖書館，香港中央圖書館的籌建計劃因而應運而生。在[[圖書館學|圖書館學]]中，「中央圖書館」（Central Library）的界定和命名須以服務社區的人口數量為依據；但為突顯「香港中央圖書館」的地位，[[康樂及文化事務署|康樂及文化事務署]]將當時5間中央圖書館——香港大會堂中央圖書館、九龍中央圖書館、沙田中央圖書館、荃灣中央圖書館及屯門中央圖書館——改名為[[大會堂公共圖書館|大會堂公共圖書館]]、[[九龍公共圖書館|九龍公共圖書館]]、[[沙田公共圖書館|沙田公共圖書館]]、[[荃灣公共圖書館|荃灣公共圖書館]]及[[屯門公共圖書館|屯門公共圖書館]]，與分區圖書館同稱。

香港中央圖書館的所在地銅鑼灣[[摩頓台|摩頓台]]，現址過去曾用作存放扣留車輛，1980年代初期改建成港島區唯一露天[[排球場|排球場]]。由於周日在銅鑼灣一帶聚集的[[印尼|印尼]]傭工愈來愈多，這裡成為了他們聚集及遊玩的地方。後來政府決定在這裡興建中央圖書館，曾經引起地區人士爭議，擔心外傭會因此缺乏聚會場所。

中央圖書館大樓之建築費為6.9億港元，受到[[香港立法會|立法會]]財務委員會議員批評：加設[[羅馬式裝飾柱|羅馬式裝飾柱]]設計被指浪費公帑，成了[[香港科技大學|香港科技大學]]翻版。1997年，[[市政總署|市政總署]]署長[[鍾麗幗|鍾麗幗]]與市政局主席[[梁定邦|梁定邦]]就香港中央圖書館外觀設計爆發罵戰，鍾麗幗被指擅自更改中央圖書館的外觀設計而被降職。

工程於1998年3月展開，1999年12月22日舉行落成典禮，原訂於2000年12月啟用，卻因電腦系統問題而延遲。延至2001年5月17日由[[香港特別行政區行政長官|香港特別行政區行政長官]][[董建華|董建華]]主持揭幕儀式，5月18日正式啟用。在開放當日，單是上午已吸引了8,000多人入場，全日共有五萬名市民入場。

香港中央圖書館大樓設計曾經引起爭議，1997年市政局主席[[梁定邦_(醫生)|梁定邦]]與市政總署署長[[鍾麗幗|鍾麗幗]]就中央圖書館的設計公開罵戰，雖然鍾所提出的高科技感設計因工程已經展開而被否決，但是「仿古典風格」、「仿羅馬風格」的建築設計遭市政局議員譏為「[[中國大陸|中國大陸]]的洋房別墅」。

由地面至平台的梯級上，刻有古今中外典籍、名人的佳句，被指是對典籍和先人的侮辱。

香港中央圖書館內外隨處可見的由[[圓形|圓形]]、[[正方形|正方形]]和[[三角形|三角形]]結合而成的幾何圖案，有「天圓地方」以及「知識累積成塔」的喻意。而由於原來設計是市政局轄下的中央圖書館，後來演變成康樂及文化事務署轄下之中央圖書館，大樓的升降機門及大門扶手都可見市政局的徽號。雖然香港市政局已經解散，但當局以徽號本身亦是香港市花[[洋紫荊|洋紫荊]]的抽象圖案為由予以保留。

== 館內設施 ==
館內設有觀景[[升降機|升降機]]來往地面至10樓，另設多條[[扶手電梯|扶手電梯]]連接各樓層，另有兩部觀景[[升降機|升降機]]由地面直上10樓及11樓，館內亦設有[[洗手間|洗手間]]供讀者使用。圖書館範圍內除7樓外均設有[[香港政府WiFi通|香港政府WiFi通]]，合供63個[[無線接取器|無線接取器]]供市民免費連線。<ref>[https://apps.ofta.gov.hk/apps/clr/content/print_public_detail.asp?rec_id=4907 電訊管理局 WiFi熱點的登記位置詳情]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

圖書館大樓每層地臺均採用網絡式地板設計，方便電源及資料系統聯繫，及新資訊科技的應用。館內亦添置不少電腦設備及設施，包括亞洲第3間圖書館採用的電腦輸送系統，將書籍分送到各層，以減低人手及提高效率。館內有600台電腦及提供900張書桌，供市民攜同手提電腦插線免費上網。<ref>科技設施首次曝光 中央圖書館落成明年底啟用 明報 1999-12-23</ref>

唯逾期雜誌/報章及參考書樓層只設黑白影印機，只有地圖圖書館及藝術資源中心才設彩色影印機。

==保安措施==
圖書館入口及各樓層均派遣多名保安維持秩序，場內各層多個位置設多個[[閉路電視|閉路電視]]，多處亦貼上「禁止拍攝」告示，以提防有人拍攝及錄影。而圖書館亦不容許普通市民進行攝影申請。

== 服務 ==
作為香港最大型的圖書館，香港中央圖書館除了提供超過2,500,000項館藏供借閱及參考，還提供多種服務，包括：

* 自助借還書籍終端機
*: 設於圖書館各層，方便讀者以自助形式借還書籍。 
* 24小時還書箱服務
*: 設於圖書館外，方便讀者於圖書館開放時間以外歸還資料。
* 資訊終端機及熒幕屏
*: 設於圖書館各層的資訊終端機，及設於大堂的熒幕屏，均介紹圖書館及康文署之其他文化資訊。 
* 多媒體資訊系統
*: 於電腦資訊中心提供，附設數碼圖書館功能。
* 聯機公眾檢索目錄
*: 有助讀者查閱資料於各館之借還狀況。
* 國際組織收藏圖書館
: 於中央參考圖書館提供，包括[[亞洲開發銀行|亞洲開發銀行]]、[[歐洲聯盟|歐洲聯盟]]、[[國際海事組織|國際海事組織]]、[[聯合國|聯合國]]、[[聯合國教科文組織|聯合國教科文組織]]、[[世界銀行|世界銀行]]、[[世界貿易組織|世界貿易組織]]及[[世界糧食計劃署|世界糧食計劃署]]之參考文獻。
* 電動組合書架系統
*: 閉架部份安裝密集書架系統，部份放置珍貴資料之書架更備有淨化系統，用作過濾空氣及消滅可破壞圖書館資料之微生物。 
* 電腦輸送系統（已停用）
*: 圖書館各部門由一個電腦控制之輸送系統聯繫，把書籍放入金屬箱內運用吊載方式送至各層。

== 事件 ==
2009年7月8日近下午5時，一名身穿藍衣約50歲的中年男子從圖書館六樓攀過圍欄躍下並倒臥1樓大堂，其後圖書館女職員及[[救護員|救護員]]分別為傷者進行[[心肺復蘇|心肺復蘇]]，頭部嚴重創傷，其後其被送院治療後，證實不治。警察在該名男子身上搜獲一包[[香煙|香煙]]，無任何[[身份證明文件|身份證明文件]]，是香港中央圖書館自啟用以來首次有人在館內死亡。<ref name="appledaily20141023"/>

2014年10月22日下午1時許，一名年約60歲的中年男子於六樓扶手電梯旁攀過圍欄躍下並倒臥1樓大堂自動借書處對開地上前，多人目擊。其後圖書館女職員為傷者進行[[心肺復蘇|心肺復蘇]]，其後其被送院治療後，證實不治。警察在圖書館上疑檢獲遺書，該名男子身上無任何[[身份證明文件|身份證明文件]]。<ref name="appledaily20141023">{{cite news|url=http://hk.apple.nextmedia.com/news/art/20141023/18909407 | title=中央圖書館內 男子跳樓亡 |publisher=[[蘋果日報_(香港)|蘋果日報]] |date=2014年10月23日 }}{{zh-hant}}</ref>
