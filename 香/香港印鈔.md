'''香港印鈔有限公司'''（{{lang-en|Hong Kong Note Printing Limited}}）是[[香港公營機構|香港公營機構]]，負責印製[[香港紙幣|香港]]及[[澳門幣|澳門鈔票]]。香港印鈔是一家由[[香港政府|香港政府]]持有多數股份、但是自負盈虧運作的[[公營事業|公營事業]]，主要收入是為香港三家發鈔銀行印製鈔票<ref>{{cite news|title=陳德霖帶隊參觀印鈔廠，保安嚴密插翼難飛|url=http://www.etnet.com.hk/www/tc/news/categorized_news_detail.php?category=latest&newsid=ETN250505486|publisher=經濟通|date=2015-05-05}}</ref>。現任主席為[[香港金融管理局|香港金融管理局]]總裁[[陳德霖|陳德霖]]。

== 歷程 ==
香港印鈔的前身是[[英國|英國]][[德拉魯|德拉魯集團]]於1984年在香港[[大埔|大埔]]設立的印鈔廠房。1996年4月，[[港英政府|港英政府]]利用[[外匯基金|外匯基金]]購入該廠，並且改名為香港印鈔。香港印鈔的廠房位於[[大埔工業邨|大埔工業邨]]，佔地面積約7,500平方米，年產量大約為3.2億張。香港印鈔的產能在2015年已接近飽和，單是2014年已經印製3.72億張切好的鈔票<ref>{{cite news|title=金管局研新鈔加強防偽|url=https://hk.news.appledaily.com/local/daily/article/20150506/19137494|publisher=蘋果日報|date=2015-05-06}}</ref>，包括為[[香港上海匯豐銀行|香港上海匯豐銀行]]印製150港元特別紀念鈔<ref>{{cite news|title=香港印鈔：今年將印製逾4億張鈔票|url=http://www.finet.hk/Newscenter/news_content/55486eb1e4b0e8ded29d020a|publisher=財華社|date=2015-05-05}}</ref>。

== 持股 ==
香港印鈔的大股東是[[香港特別行政區政府|香港特別行政區政府]](55%控股權)，其他股東包括[[中國印鈔造幣總公司|中國印鈔造幣總公司]](15%控股权)及香港三家[[香港紙幣|發鈔銀行]]──[[中銀香港|中國銀行（香港）]]、[[香港上海滙豐銀行|香港上海滙豐銀行]]及[[渣打銀行_(香港)|渣打銀行]](各佔10%股權)。
