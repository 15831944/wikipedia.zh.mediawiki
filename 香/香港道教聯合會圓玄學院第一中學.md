'''[[香港道教聯合會圓玄學院第一中學|香港道教聯合會圓玄學院第一中學]]'''（{{lang-en|HKTA The Yuen Yuen Institute No. 1 Secondary School}}，簡稱圓玄一中、YY1）是[[香港|香港]]一所由[[香港道教聯合會|香港道教聯合會]]主辦的政府[[津貼|津貼]]男女[[文法中學|文法中學]]，成立於1979年，校訓為「明道立德」。{{Fact|校舍佔地12,500[[平方米|平方米]]。}}報稱位於[[葵青區|葵青區]][[和宜合道|和宜合道]]42號，在[[麗晶中心|麗晶中心]]對面，現任[[校長|校長]]為簡偉鴻先生。

== 辦學宗旨    ==
香港道教聯合會圓玄學院第一中學秉承母會「以道為宗，以德為化，以修為教，以仁為育」的辦學宗旨，推行「道化教育」，並以「明道立德」為校訓，透過品德和學業兼備的全人教育，使學生在德、智、體、群、美各方面都得到全面發展。   
除了學習和體育成績外，學校還秉承校訓「明道立德」，十分重視道德教育，透過早會、週會、班主任課和生活教育科向學生灌輸正確的道德觀，以及教導學生整潔儀容、禮儀、謙恭和節儉的美德，讓學生在現今的社會潮流中不會隨波逐流。 為了提高德育成效，學校還加強與家長的合作。 學校和另外四間香港道教聯合會屬下中學從優質教育基金取得撥款，以在校內推廣道德教育。

== 歷任校長 ==

# 尤漢基博士（任期：1979年 - 2009年），為香港道教聯合會圓玄學院第一中學的創校校長，擔任校長一職三十年，至2009年8月榮休，現為該校[[法團校董會|法團校董會]]的辦學團體校董之一。
# 文靜芬女士（任期：2009年 - 2018年），於該校任職36年，至2009年9月起接任榮休的尤漢基博士出任該校的第二任校長，至2018年8月榮休。現為該校[[法團校董會|法團校董會]]的辦學團體校董之一。
#簡偉鴻先生（任期：2018年至今），於該校任職18年，至2018年9月起接任榮休的文靜芬校長出任該校的第三任校長。

== 設施 ==
學校的[[建築物|建築物]]形成一個[[方形|方形]]，總佔地達12000平方米，中間是[[籃球場|籃球場]]暨[[操場|操場]]，除[[教室|教室]]外亦設有互動學習中心/音樂室、禮堂、飯堂、雨天操場、看台、演講廳、體適能訓練中心、電腦室、資訊及通訊科技中心、語文自學中心、英語角、普通話角、語文角、通識角、智能教室、圖書館、舞蹈室、視藝室、地理室、銀樂室、活動室、攀石牆、學生輔導室、升學及就業輔導室、學習支援室、學生食堂、學生會室、醫療室、影音控制室、錄音室、烹飪室、縫紉室、綜合科技室、科學室（組合科學）、生物實驗室、物理實驗室、化學實驗室、綜合科學實驗室、禮堂及家長教師會資源中心。天台有有機耕種的種植地點，全校均已安裝空調及無線寬頻網絡（WiFi)，全部課室已安裝實物投映機、投映機及電腦設備，以便發展資訊科技教學。升降機現正興建中，預計將於2018年尾落成。

== 歷史 ==
{{CJK-New-Char|44EA|ver=A}}
1979年，第一中學的[[校舍|校舍]]工程尚未完成，因此學校暫借位於[[葵盛邨|葵盛邨]]第十九座的[[香港道教鄧顯紀念學校|香港道教鄧顯紀念學校]]上課。學校於9月3日開課，當時共有6班中一，學生252人，[[教師|教師]]8人，由尤漢基先生擔任校長。第一中學採用[[文法学校|文法中學]]課程，學生須修讀[[漢語|中文]]、[[英語|英文]]、[[數學|數學]]、綜合科學、[[中國歷史|中史]]、[[地理|地理]]、[[世界歷史|世史]]、經濟及公共事務、[[美術|美術]]設計、[[體育|體育]]各科。 課外活動逢星期六舉行，有交通安全隊、國樂團等興趣小組。並成立「明」「道」「立」「德」四社進行社際活動，計有[[拔河|拔河]]、[[乒乓|乒乓]]、[[籃球|籃球]]等比賽。為方便居住於上[[葵涌區|葵涌區]]學生到葵盛上課，故辦理校車服務，乘搭[[學生|學生]]達173人。 

1980年，學校已辦至中二級，中一八班中二六班，學生572人，教師19人；惟因校舍仍未竣工，故仍暫借[[香港道教鄧顯紀念學校|香港道教鄧顯紀念學校]]上課。為貫徹會方辦校宗旨，重視道德教育，從本年度開始編訂周訓大綱，並舉辦活動以配合發展。 增設科目計有：音樂，家政和工藝。體育科則編排游泳課。 為瞭解學生社群情況：本校從本年度起每年均為每班舉行社群調查，以加強群育工作。增加課外活動組別，愛丁堡獎勵計劃，男童軍，少年警訊。大型課外活動有水運會、冬至聯歡會、樂韻播萬千音樂會及暑期活動。

1981年，學校首次參加[[初中派位辦法|初中派位辦法]]，153個中三[[學生|學生]]有92%獲派資助中學學位（當年全港派位率為65%）。第一屆[[陸運會|陸運會]]假葵涌運動場[[舉行|舉行]]。此外學校為強化學校與[[家長|家長]]之間[[關係|關係]]，1982年3月14日舉辦第一屆家長座談會。

1989年，[[學校|學校]]10[[周年|周年]][[校慶|校慶]]，蒙總[[校監|校監]][[湯國華|湯國華]]蒞臨主禮，隨即舉行為期兩天之[[開放日|開放日]]。 

1999年20周年[[校慶|校慶]]，由[[行政會議成員|行政會議成員]][[王䓪鳴|王{{僻字]]博士及[[教育局|教育局]][[高級助理署長|高級助理署長]][[莊國傑|莊國傑]]先生主禮。當日有兩文三語、[[太極|太極]]、[[舞蹈|舞蹈]]、[[沿繩下滑|沿繩下滑]]等表演。翌日舉行[[開放日|開放日]]，有各項展覽及遊戲活動，學生、家長、舊生及各界參觀者眾。已完成[[網際網路|網際網路]]管道、節點和[[伺服器|伺服器]]室的建設[[工程|工程]]。建立了以[[Linux|Linux]]為運行系統的[[內聯網|內聯網]]。 
多媒體學習中心於6月底落成，可供43人一起[[學習|學習]]，並於7月為教師舉行操作[[訓練|訓練]]。為配合[[資訊|資訊]][[科技|科技]][[設備|設備]]所需[[電力|電力]]，開展了加建電力[[變壓|變壓]]房的[[工程|工程]]，電力公司為學校額外增加了一條臨時[[電纜|電纜]]，以應付未建成[[變壓|變壓]]房之前的需要。本年度《荳芽集》就[[香港藝術發展局|香港藝術發展局]]資助出版。 

尤漢基校長於2009年退休，職位由文靜芬繼任。同年，第一中學正式推行[[三三四學制|三三四學制]]。高中學生可選讀兩至三個選修科，上課時間改為雙星期十日制。學校斥資改建實驗室，並加建三個小型課室，以便進行小組學習。2009年11月，第一中學在荃灣悅來酒店舉行「三十周年聯歡晚宴」。

直至2018年，簡偉鴻副校長、李彬老師及郭俊廷老師獲頒發「[[行政長官卓越教學獎|行政長官卓越教學獎]]」。
