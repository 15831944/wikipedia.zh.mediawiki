'''香港歸英運動'''，簡稱'''歸英'''，是主張推動香港脫離[[中華人民共和國|中華人民共和國]]，再次成為[[英國海外領土|英國海外領土]]一部份的一項主張。該主張的支持者認為中國大陸近年種種干預香港內政，應被視為中方單方面違反[[中英聯合聲明|中英聯合聲明]]，在此情況下，英國有權宣告[[中英聯合聲明|中英聯合聲明]]失效，並可廢止條約<ref name = "D01">{{cite news|title=親英團體請願促回歸英國|url=http://hk.on.cc/hk/bkn/cnt/news/20140604/bkn-20140604193906177-0604_00822_001.html|agency=東方日報}}</ref>，令香港返回香港主權移交前之狀態。他們亦認為，倘若香港特區政府無力管治香港，而[[中國政府|中國政府]]有意違反聯合聲明而直接管治香港，香港在憲法上的地位應該回到1997年7月1日前，英國政府因中國違法國際協議而重新擁有香港主權。

歸英論支持者認為香港可參考[[直布羅陀|直布羅陀]]，成為具有民選政府和國會，擁有真正自治權的[[英國海外領土|英國海外領土]]。理論支持者認為，在民主國家英國的主權下，香港在有機會實行真正自治，並提供安穩空間與港人思考香港的未來，如獨立成為英聯邦王國等選擇<ref name = "D01" />。

== 網上研究 ==
2013年，[[南華早報|南華早報]]在網上進行民意研究，問題為「如果有可能，香港人是否願意投票回歸成為英國海外屬土」，表示認同成為[[英國海外領土|英國海外領土]]裝上投票者一度佔93%，不認同歸英者約佔7%。<ref>{{cite news|title=願香港回歸英国？|url=http://hk.apple.nextmedia.com/news/art/20130315/18195312|agency=蘋果日報}}</ref>

2014年3月20日，前[[香港总督|香港总督]][[彭定康|彭定康]]來香港出席牛津大學與[[香港海事博物館|香港海事博物館]]合辦的活動時，有部分人士手持[[香港旗|香港旗]]，並高唱英國國歌[[天佑女王|天佑女王]]，提出“香港回归英国”的请求。彭定康和相关人士握手，在離開時無直接回應，并表示“尊重香港的意見多元化”。<ref>{{cite news|title=末代港督访港 港独集体高潮|url=http://www.imastv.com/news/article.php?lang=cht&id=37981|agency=澳亚卫视|deadurl=yes|archiveurl=https://web.archive.org/web/20150722140528/http://www.imastv.com/news/article.php?lang=cht&id=37981|archivedate=2015-07-22}}</ref><ref>{{cite news|title=彭定康撐新聞自由　學術自主 {{!}} 蘋果日報 {{!}} 要聞港聞 {{!}} 20140321|url=http://hk.apple.nextmedia.com/news/art/20140321/18663733|agency=苹果日报}}</ref>

英國一貫之政策是協助其殖民地成為主權獨立之國家。但由於英國向中國租借土地的情況複雜，獨立並不適用於香港。英國理解中國對香港主權歸屬的立場，自1972年11月2日[[聯合國大會|聯合國大會]]批准《关于准许殖民地国家及民族独立之宣言》之適用地方名單剔除「香港」後，英國從未照會聯合國秘書長列入「香港」於該名單。香港人知悉中國反對[[香港獨立|香港獨立]]，因此英國無法安排香港獨立。香港92%的土地經《[[展拓香港界址專條|展拓香港界址專條]]》租借而來，以99年為限，期滿後英國將喪失該92%土地的一切權利。<ref>{{cite web| url=http://fc95d419f4478b3b6e5f-3f71d0fe2b653c4f00f32175760e96e7.r87.cf1.rackcdn.com/820924%201030%20MT-Deng%20(962-69).pdf| title=Prime Minister's Visit to Peking| date=1982-09-24}}</ref>不論獨立與否，香港將無法單憑這8%剩下的土地而運作，英國須與中國商討香港前途。<ref>{{cite web| url=http://fc95d419f4478b3b6e5f-3f71d0fe2b653c4f00f32175760e96e7.r87.cf1.rackcdn.com/830310%20CC(83)%208th%20Annex%20(128-77-12).pdf| title=Most Confidential Record| date=1983-03-10}}</ref>

== 有關行動 ==
