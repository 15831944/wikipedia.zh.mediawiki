'''香港四大天王'''是指1990年代[[香港樂壇|香港樂壇]]四位最受歡迎的男歌手：[[張學友|張學友]]、[[劉德華|劉德華]]、[[郭富城|郭富城]]、[[黎明|黎明]]（按年齡順序排列，排名不分先後），四人自1992年開始冒起，取代了八十年代中期以來[[譚詠麟|譚詠麟]]和[[張國榮|張國榮]]在香港樂壇的地位，成為香港流行文化的其中一個現象。他們在音樂、影視、舞蹈、商業、慈善等領域都取得傑出的成就，在華人社會具有廣泛的影響力和知名度<ref>{{Cite web |url = http://news.cri.cn/gb/inet/other/sdtw.htm |title =四大天王經典回顧| publisher =國廣國際線上網路（北京）有限公司|language =中文}}</ref>。

== 入行經過 ==
*張學友（Jacky Cheung Hok-yau），1961年7月10日在香港出生，高中畢業後曾於[[香港貿易發展局|香港貿易發展局]]及[[國泰航空公司|國泰航空公司]]工作，1983年參加《[[歡樂今宵|歡樂今宵]]》的《盡顯才華Sing聲星》比賽，但未能獲獎<ref>{{Cite web |url = https://www.cyberctm.com/zh_TW/news/detail/923554#.XGfN0qIzaUk|title =娛聞中錯誤之處| publisher =澳門日報|date =2015年9月7日|language =中文}}</ref>，其後於1984年參加[[十八區業餘歌唱大賽|十八區業餘歌唱大賽]]，憑[[關正傑|關正傑]]原唱的《大地恩情》一曲獲得冠軍，同年與電視廣播有限公司及[[寶麗金唱片公司|寶麗金唱片公司]]簽約，成為藝人及歌手<ref>{{Cite web |url = http://www.epochtimes.com/b5/2/6/15/n196711.htm|title =回首張學友星路歷程：歌神就是這樣“煉”成的| publisher =大紀元|date =2002年6月15日|language =中文}}</ref>。

*劉德華（Andy Lau Tak-wah），1961年9月27日在香港出生，小時候在父親的店舖工作，售賣小食及雜貨，因經常往片場送外賣而對演藝工作產生興趣，中學畢業後於1981年入讀第10期[[無綫電視藝員訓練班|無綫電視藝員訓練班]]，畢業後任職演員<ref>{{Cite web |url = https://hk.entertainment.appledaily.com/entertainment/daily/article/20030815/3475079|title =17歲四眼仔　當年肥騰騰 華仔偷步入行證據確鑿| publisher =蘋果日報|date =2003年8月15日|language =中文}}</ref>。在拍攝電影期間認識了歌手[[林子祥|林子祥]]，在林子祥的鼓勵下嘗試發展歌唱事業<ref >{{cite web|url=http://book.hexun.com.tw/2009-11-11/121651789.html | language= | title=誰是第一個發現劉德華唱歌才華的人(圖) |publisher=和訊讀書 |date=2009-11-11 |accessdate= }}</ref>，1985年與[[華星唱片|華星唱片]]簽約及推出唱片，成為歌手。

*郭富城（Aaron Kwok Fu Shing），1965年10月26日在香港出生，中學畢業後在景福金行工作。1984年因做[[一字馬|一字馬]]受傷請假太久而遭金行辭退，同年成功投考無綫電視藝員招募班，成為舞蹈員<ref>{{Cite web |url = http://www.businesstoday.com.tw/article/category/154685/post/201402060006/%E9%83%AD%E5%AF%8C%E5%9F%8E%EF%BC%9A%E6%88%91%E5%8F%AF%E4%BB%A5%E8%BC%B8%20%E4%BD%86%E4%B8%8D%E8%83%BD%E8%AA%8D%E8%BC%B8?utm_source=%E4%BB%8A%E5%91%A8%E5%88%8A&utm_medium=autoPage|title =郭富城：我可以輸 但不能認輸| publisher =今周刊|date =2014年2月6日|language =中文}}</ref>。1987年入讀無綫電視藝員訓練班，成為藝員，因不獲重視而到台灣發展。1990年在台灣拍攝機車廣告，其後獲邀簽約福隆傳播公司，同年9月推出首張唱片，成為歌手<ref> {{Cite book | author =王強, 汪麗珍| title =香港演藝圈| location =香港| publisher =時事出版社| date =1996年|ISBN = 7800092941 | url = https://books.google.com.hk/books?id=Erx8AAAAIAAJ&q=%22%E9%83%AD%E5%AF%8C%E5%9F%8E%22+%22%E7%A6%8F%E9%9A%86%E5%82%B3%E6%92%AD%E5%85%AC%E5%8F%B8%22&dq=%22%E9%83%AD%E5%AF%8C%E5%9F%8E%22+%22%E7%A6%8F%E9%9A%86%E5%82%B3%E6%92%AD%E5%85%AC%E5%8F%B8%22&hl=zh-TW&sa=X&ved=0ahUKEwj1i_rhkcDgAhVIE4gKHZ-tBNsQ6AEIKDAA | language =中文}} </ref>。

*黎明（Leon Lai Ming），1966年12月11日在北京出生，六歲定居香港，十四歲往英國就讀高中<ref>{{cite news |language = 中文 |author =鄧明儀|url =https://www.facebook.com/elainetang.6mama/photos/a.1070316199696446/2057028964358493/?type=1&theater|title =黎明的氣質與教養：當北京遇上倫敦|publisher = 鄧明儀facebook專頁|date =2018年10月4日|}}</ref>。1985年返港後獲星探邀請參演電影，其後曾入讀[[亞洲電視藝員訓練班|亞洲電視藝員訓練班]]及任職電話推銷員。1986年參加[[新秀歌唱大賽|新秀歌唱大賽]]，憑[[蔡楓華|蔡楓華]]原唱的《絕對空虛》一曲獲得季軍，簽約華星唱片，曾灌錄電視劇主題曲，但未有推出唱片。1989年獲[[戴思聰|戴思聰]]推薦到寶麗金唱片公司試音，成為旗下的歌手，1990年7月推出首張唱片<ref>{{Cite web|url=http://dailynews.sina.com/bg/ent/hktwstar/kwongwah/20100719/00071664608.html|title =捧星有一手 戴思聰甘為黎明坐牢| publisher =光華日報|via= dailynews.sina.com |language = 中文}}</ref>。

== 成名過程 ==
隨著[[譚詠麟|譚詠麟]]、[[張國榮|張國榮]]等八十年代知名歌手淡出香港流行樂壇，[[劉德華|劉德華]]、[[張學友|張學友]]、[[李克勤|李克勤]]、[[草蜢_(組合)|草蜢]]和[[Beyond|Beyond]]樂隊等歌手於1990年開始冒起。1991年，[[黎明|黎明]]因參演電視劇而火速竄紅，唱片銷量高踞IFPI香港唱片銷量大獎前列（當中專輯《[[是愛、是緣|是愛、是緣]]》更成為1991年IFPI最高銷量唱片專輯），而同屬寶麗金唱片公司的李克勤之人氣顯得相形見絀，而作為「實力派」的張學友則保持強勁的聲勢，黎明和劉德華的形象同屬「偶像派」，曾被稱為「雙子星」，當時有傳媒將他們三人統稱為'''「三劍俠」'''。[[郭富城|郭富城]]於1990年尾在[[台灣|台灣]]以一機車廣告開始受到關注，隨後灌錄了三張國語大碟，其中《對你愛不完》及《我是不是該安靜的走開》等歌曲紅遍[[港澳臺|港澳臺]]，1992年初以歌星身份回香港發展。

1992年，《[[東方日報|東方日報]]》一篇描寫香港流行樂壇的[[打油詩|打油詩]]登出後，被市民討論，後來香港“演唱會之父”[[張耀榮|張耀榮]]（另一說法為[[方逸華|方逸華]]）將郭富城封為第四位天王，於是這四位當時最受歡迎的男歌手從此便合稱為“四大天王”，並得市民廣泛接受，一直沿用至今。另一說法則是前[[香港電台|香港電台]]助理廣播處長[[張文新|張文新]]在1992年香港電台「[[太陽計劃|太陽計劃]]」活動上，邀請四人同台表演並即場向傳媒封四人為「四大天王」<ref>{{Cite web|url=http://hk.apple.nextmedia.com/enews/realtime/20151110/54411540|title =【四大天王10個小八卦】名從何來？ 誰是天王殺手？第五天王？| publisher =蘋果日報|language = 中文}}</ref>。

== 影響力 ==
1990年，許多人認爲張學友和劉德華分飾在1980年代譚詠麟和張國榮的角色，以樂壇實力派代表和偶像派代表的姿態稱霸香港流行樂壇。不過，到了1991年，黎明憑演出熱播電視劇《今生無悔》男主角程朗及主唱插曲《對不起，我愛你》，人氣如火箭急升，且憑同年推出的唱片專輯《是愛、是緣》成為香港全年最高銷量專輯，成績婓然，因此在1991年張學友、劉德華及黎明一度被香港傳媒稱為「三劍俠」以代表三人為最受歡迎的三位男歌手。及後到了1992年，連同由台灣竄紅再回港發展的郭富城被傳媒統稱為「四大天王」。在1990年代，四大天王競爭激烈：張學友憑藉備受肯定的歌唱實力和最高的[[唱片銷量|唱片銷量]]被譽為[[歌神|歌神]]接班人；黎明配合[[廣告|廣告]]宣傳歌曲，取得極成功的反應，開始了香港流行樂壇的廣告歌熱潮；劉德華於1990年代初在港的巔峰期後，全力轉戰海外的[[華語流行音樂|國語歌]]市場，獲得很大的成功，其不少國語歌曲均在兩岸華語地區耳熟能詳，但1990年代後期在香港流行樂壇的[[粵語流行音樂|粵語歌]]發展則開始放缓，被張學友和黎明拉開距離；郭富城則主力勁歌熱舞，憑精湛舞藝獲得觀眾與演藝界的認同。

四人的歌曲在1990年代高頻出現於兩岸三地的電臺、電視、頒獎禮，對於其他歌手的發展帶乃至給華語流行樂壇的發展來了極負面的影響，其他歌手的生存空間被擠壓得很少。1994年「搖滾新勢力」在[[香港體育館|香港體育館]]演出之前，中國內地的[[何勇|何勇]]在北京接受訪問時說「香港只有娛樂，沒有音樂，『三大天王』除了張學友還算是個唱歌的，其他都是小丑」。這句說話在香港引起軒然大波，很多媒體和藝人都對何勇嚴詞批評，[[梅豔芳|梅豔芳]]說「對香港歌星這樣一個口吻，你算什麽？」，「[[中國火|中國火]]」的海報、地鐵廣告都被砸，後來何勇在香港發佈會上解釋「香港這個繁華的地方需要『四大天王』」才將事情平息。事實上前年（1993年）意外身亡的[[黃家駒|黃家駒]]亦曾說過「香港只有娛樂圈，沒有樂壇」，但並沒有批評「三大天王」。有人說“三大天王”是香港娛樂圈壟斷的一種標誌，尚有其他明星被封爲“三小天王”或“第五天王”，但無一例外被“四大天王”給湮沒。“四大天王”是香港娛樂圈壟斷時代的産物，也是香港娛樂圈黃金時代的象徵，而“四大天王”稱號的重要性在1990年代後期下降（特別是1997年[[香港回歸|香港回歸]]後），剛好反映粵語歌壇在1997年後的衰落。

1999年後期，隨著黎明宣佈不再角逐香港流行樂壇的獎項後<ref>{{Cite web|url=http://blog.sina.com.cn/s/blog_741f5dde010120qe.html|title =轉載【LEON】不領獎宣言| date = 2012年5月23日|via=新浪博客|language = 中文}}</ref>，張學友、郭富城、劉德華也相繼淡出樂壇，「四大天王」的現象逐漸冷卻<ref>{{cite news |language =中文|author =譚綺文| title =黎明唔攞獎樂壇震動 Fans贊成 : 四大天王年代告一段落|publisher =星島日報| date =1999年12月14日}}</ref>，香港樂壇亦開始進入衰落時期<ref> {{Cite book | author =朱耀偉| title =歲月如歌－詞話香港粵語流行曲|location =香港|publisher =三聯書店鋪（香港）有限公司| date =2019年9月|pages =第151至163頁|ISBN = 9789620428807|language =中文}} </ref>。

==合作演出==

=== 綜藝節目 ===
* 1992年，四大天王一同演出 [[香港小姐競選|1992年度香港小姐競選]]決賽
* 1992年，四大天王一同演出[[香港電台|香港電台]]－太陽計劃音樂會
* 1992年，四大天王一同演出1992年[[區域市政局|區域市政局]]节开幕巡游
* 1993年，四大天王一同成為華東水災慰問團香港代表於北京[[人民大會堂|人民大會堂]]演出
* 1997年，四大天王一同演出慶祝1997年香港回歸晚會
* 2003年，四大天王一同[[第22屆香港電影金像獎|第22屆香港電影金像獎]]演出悼念[[張國榮|張國榮]]
* 2007年，四大天王一同演出香港回歸的10周年慶典晚會

===電影===
* 1988年，張學友和劉德華與共同主演《[[旺角卡門|旺角卡門]]》
* 1990年，張學友和劉德華在電影《[[阿飛正傳|阿飛正傳]]》分別飾演超仔和歪仔。
* 1991年，劉德華在他投資的《[[九一神鵰俠侶|九一神鵰俠侶]]》中請來當時尚未列為天王的郭富城任要角銀狐。同年上映的《[[五億探長雷洛傳II之父子情仇|五億探長雷洛傳II之父子情仇]]》、《[[机BOY小子之真假威龙|机BOY小子之真假威龙]]》亦為劉德華及郭富城合作之電影。
* 1991年，《[[豪門夜宴|豪門夜宴]]》，為賑濟1991年華東水災的香港演藝界忘我大電影，是惟一一套四位天王全參與的電影（只有張學友主演，其他客串演出）。
* 1992年，黎明和張學友合演《[[明月照尖東|明月照尖東]]》及《[[妖獸都市_(1992年電影)|妖獸都市]]》。
* 1992年，黎明和劉德華合演《[[龍騰四海|龍騰四海]]》。
* 1993年，《[[超級學校霸王|超級學校霸王]]》，有劉德華和張學友、郭富城（客串）三位天王參與，是在2003年《[[金雞2|金雞2]]》（劉德華、黎明和張學友）上映之前惟一一套有三位天王參與的電影。
* 1998年，《[[安娜瑪德蓮娜|安娜瑪德蓮娜]]》中，張學友客串飾演軍裝警察，郭富城飾演游永富。
* 2003年，張學友和劉德華在[[曾志偉|曾志偉]]的牽頭下共同主演電影《[[江湖_(電影)|江湖]]》。
* 2003年，黎明和劉德華合演《[[無間道3之終極無間|無間道3之終極無間]]》。
* 2009年，《[[建國大業|建國大業]]》中，黎明客串[[蔡廷鍇|蔡廷鍇]]，劉德華客串[[俞濟時|俞濟時]]。
* 2009年，《[[十月圍城|十月圍城]]》中，張學友客串，黎明有演出。
* 2012年，《[[寒戰|寒戰]]》中，劉德華客串飾演陸明華，郭富城飾演劉傑輝。
* 2015年，張學友和劉德華在[[王晶|王晶]]的牽頭下共同主演2016賀歲電影《[[賭城風雲III|賭城風雲III]]》。

===電視劇===
* 1989年，當時仍未發展歌唱事業的黎明和郭富城合演《[[天涯歌女_(無綫電視劇)|天涯歌女]]》，此劇亦是唯一有超過一位天王參演的電視劇。
* 2004年，張學友和郭富城客串無綫電視劇《[[赤沙印記@四葉草.2|赤沙印記@四葉草.2]]》。

===音樂===
* 1994年的[[香港電台|香港電台]][[十大中文金曲頒獎典禮|十大中文金曲頒獎典禮]]上，四人共同在舞臺上演唱了[[譚詠麟|譚詠麟]]的《朋友》。
* 1999年1月，黎明最後一次參加頒獎晚會，四個人亦同時坐在了台下的第一排，第一排的位置並沒有其他男歌手，顯示了四大天王的地位，至此之後，由於黎明的退出和張學友逐漸淡出頒獎典禮，四人亦很少有合作的機。
* 2003年，樂壇巨星[[張國榮|張國榮]]剛於4月1日去世，與張國榮份屬好友的四人出席在4月6日舉行的[[第22屆香港電影金像獎|第22屆香港電影金像獎]]典禮上，為紀念張國榮而同台合唱生前作品《當年情》（電影《[[英雄本色_(1986年電影)|英雄本色]]》主題曲）。
* 2007年，[[香港特別行政區成立十周年紀念|香港特別行政區成立十周年紀念]]的晚會中，四人亦联袂演出。

==其他天王稱號==
*[[台灣|台灣]]封有四小天王：[[林志穎|林志穎]]、[[吳奇隆|吳奇隆]]、[[金城武|金城武]]、[[蘇有朋|蘇有朋]]。（命名自1992年，但代表性不足）
*[[香港|香港]]「第五天王」：「第五天王」意指[[1990年代香港|1990年代]]期間最有機會問鼎四大天王以外第五人的男歌手人選，實際上此角色從未正式得到承認，天王仍然只有張劉黎郭四位。
:[[李克勤|李克勤]]（曾被看好及於[[1980年代香港|1980年代]]末被稱[[譚詠麟|譚詠麟]]接班人，可惜1992年開始人氣下跌及人氣被同公司的張學友和剛冒起的黎明遠遠拋離，未能成為四大天王，曾被稱最有機會成為第五天王，離開[[寶麗金|寶麗金]]後開始後勁不繼），當時最接近「第五天王」的地位。<br />
:[[周華健|周華健]]（1994-1995年間由台灣回流香港發展並一度取得極大成功，當時曾被香港傳媒稱為「天王殺手」, 兩年間其人氣、獎項及唱片銷量更直迫四大天王，惜其後兼顧不到台港兩地市場而專注台灣及中國內地發展）。<br />
:[[鄭伊健|鄭伊健]]（1996-1998年間在香港歌影人氣甚高的偶像派歌手，一度取代李克勤與四大天王同時入選年度最受歡迎男歌手五強）

*1990年代初：台灣民间亦將四位本土歌手封号為四大王王：[[齊秦|齊秦]]、[[王傑|王傑]]（被封為亞洲[[歌聖|歌聖]]）、[[童安格|童安格]]、周華健（被封為國民[[歌王|歌王]]，周華健曾被香港傳媒稱作是「天王殺手」，而「天王」則意指香港四大天王，而周華健于2004年接受中國中央電視臺電視節目《[[藝術人生|藝術人生]]》訪問時表示，當年自己的銷量僅次於張學友，與他既是好友亦將他作為推動自己工作一個動力，每次回到公司的第一件事就是要翻查張學友的最新唱片銷量記錄，想張學友的時間比想自己老婆還要多。）。及後於2011年，台灣傳媒封[[華語流行曲|華語歌壇]]新四大天王：[[周杰倫|周杰倫]]、[[王力宏|王力宏]]、[[林俊傑|林俊傑]]、[[羅志祥|羅志祥]]。

*1995年12月由中視「歡樂傳真」節目主辦、威聚唱片企劃執行的「四大天王就是你」模仿大賽中羅志祥和[[歐漢聲|歐漢聲]]、[[陳顯政|陳顯政]]、[[陳中威|陳中威]]組成偶像團體，命名「四大天王」。不过1998年陳顯政與陳中威分別因不適應演藝圈的環境和兵役問題退出娱乐圈，羅志祥和歐漢聲其後改組「[[羅密歐_(組合)|羅密歐]]」。

===新四大天王===
2004年在香港樂壇亦曾發生「新四大天王」命名爭議事件，於2004年11月，譚詠麟在接受傳媒訪問時，封[[古巨基|古巨基]]、[[許志安|許志安]]、[[梁漢文|梁漢文]]及[[李克勤|李克勤]]為「新四大天王」，但這一“封號”並未獲得廣泛認同，甚至引發各方廣泛的不滿。當中，最惹起爭議的是流行程度甚高、同屬[[環球唱片|環球唱片]]的[[陳奕迅|陳奕迅]]沒被稱為天王，而當時在[[英皇娛樂|英皇娛樂]]唱片公司的[[謝霆鋒|謝霆鋒]]亦有相當高的知名度。該年年底，[[譚詠麟|譚詠麟]]在[[新城勁爆頒獎禮|新城勁爆頒獎禮]]上，把本來只有3個名額的新城勁爆男歌手得獎名單撕毀，放入外衣衣袋，四人與劉德華同時得獎（即共有五人獲獎，但該頒獎禮一向代表性不足）<ref>{{Cite web |url = http://ent.sina.com.cn/2004-11-15/0922565204.html|title =譚詠麟冊封“新四大天王”引發多方爭議| publisher =金羊網－新快報|date =2004年11月15日|language =中文}}</ref>。
