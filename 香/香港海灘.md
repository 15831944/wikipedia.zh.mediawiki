{{香港地理}}

[[香港|香港]][[海岸線|海岸線]]長而彎曲，令香港境內有很多[[海灣|海灣]]和[[海灘|海灘]]。

==地理環境==
由於香港以山地為主，因此很多海灣都被高山作為天然屏障。這對阻隔[[海風|海風]]有很大的幫助，減低[[海浪|海浪]]的威力。

== 管理 ==
目前，[[康樂及文化事務署|康樂及文化事務署]]一共管理41個[[泳灘|泳灘]]。其中12個位於[[香港島|香港島]]，主要集中於[[南區_(香港)|南區]]東部；29個位於[[新界|新界]]及[[離島區|離島區]]，主要集中於[[西貢區|西貢區]]、[[荃灣區|荃灣區]]、[[屯門區|屯門區]]、[[大嶼山|大嶼山]]南部、[[長洲_(香港)|長洲]]及[[南丫島|南丫島]]。另外亦有一些非政府[[刊憲|刊憲]]海灘，吸引市民前往嬉水、游泳。

== 設施 ==
大部分由[[康樂及文化事務署|康樂及文化事務署]]管理的泳灘設有[[救生員|救生員]]及裝有[[防鯊網|防鯊網]]，而部份更設有[[小食亭|小食亭]]、[[燒烤設施|燒烤設施]]、[[更衣室|更衣室]]、[[淋浴設施|淋浴設施]]及[[浮台|浮台]]等，方便遊人。

== 泳灘水質監測計劃 ==

== 警告訊號 ==
[[File:Red_flag.svg|thumb]]

;紅+黃旗
* 表示海灘有救生員當值

;紅旗
* 表示在泳灘游泳會有危險

;鯊魚旗
* 表示該海灘有[[鯊魚|鯊魚]]出沒

;根據政府指引，當紅旗和[[鯊魚|鯊魚]]旗發出時，請勿下水。
* 當康文署發出紅旗和[[鯊魚|鯊魚]]旗，會在電台發出廣播，呼籲泳客勿前住此地方下水。
