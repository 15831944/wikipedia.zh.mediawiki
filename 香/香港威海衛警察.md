'''香港威海衛警察'''（又稱'''香港魯警'''，俗稱'''山東差'''）是[[香港警察隊|香港警察隊]]成立初期時從[[山東|山東]][[英租威海卫|英租威海卫]]招募的警察人員。香港警隊一直以[[廣東人|廣東人]]為主，當中以來自[[東莞|東莞]]、[[客家|客家]]及[[潮州|潮州]]較多；相比之下，[[山東|魯籍]]人員被視為另類華籍警員。

== 起源 ==
1914年8月1日[[第一次世界大戰|第一次世界大戰]]爆發，至同月4日，[[英國|英國]]對[[德國|德國]]宣戰，[[香港|香港]]作為[[英國殖民地|英國殖民地]]亦不能幸免被捲入戰爭的旋渦中。根據[[香港政府|香港政府]]統計，至少264名[[香港居民|香港居民]]被徵召遠赴英國及參與戰爭，當中包括不少[[香港警務人員|香港警務人員]]，因而對香港[[警察力量|警察力量]]造成一定影響。[[威海衞|威海衞]]作為[[英國|英國]]的[[租借地|租借地]]，基於原籍[[兵力|兵力]]匱乏，遂招募山東人組織[[僱傭軍|僱傭軍]]，負責威海衞的防衛任務。期間山東人的表現出色，對待同胞亦不手軟，先後多次鎮壓同胞所發起的抗爭，包括於1900年協助[[八國聯軍|八國聯軍]]剿滅[[義和團|義和團]]，事後兵團指揮官巴恩斯讚賞這批山東人僱傭軍兵，認為他們表現出與其指揮官堅定地站在同一陣線，即使與自己人民對抗亦不猶豫。

1922年[[香港海員大罷工|香港海員大罷工]]事件發生後，香港政府持續地希望穩定[[香港社會|香港社會]]，以確保其管治不受威脅。由於在過去多場工潮中都顯示，抱有強烈民族意識的[[香港華人|香港華人]]警務人員未有嚴厲地對待同胞示威者，香港政府基於對香港華人不信任<ref>[http://hk.apple.nextmedia.com/news/art/20140728/18814076 梁振英父山東來港當警察] 《蘋果日報》 2014年7月28日</ref>，又希望藉以華制華的方式去控制局面，遂於同年9月派遣一名外籍教官、3位華籍教官和一名通曉[[膠遼官話|山東話]]和[[英語|英語]]的翻譯人員遠赴威海衛，招募第一批威海衛警察，共約50名。首批魯籍人員於1923年2月抵達香港，同年再有兩批陸續到埗。1924年8月來到香港的一批為人數最多的一批，有102名警員和5名翻譯員<ref>[http://www.police.gov.hk/offbeat/683/021_c.htm 軍人加入香港警隊概況（三）] 《警聲》 第683期</ref>。招募初期，魯籍人員被調遣往[[旺角警署|旺角警署]]及[[舊赤柱警署|舊赤柱警署]]接受基本訓練。

== 招募條件 ==
在招募魯籍人員的要求比招募[[香港人|香港人]]更為嚴格，應聘者身高最少達5[[呎|呎]]7[[寸|寸]]（170[[公分|公分]]），而且體格尤其健碩。這批人員在當地受訓半年後，於隔年3月被派駐香港協助維持[[公共安全|公共安全]]。當時，威海籍警察被香港警隊特別編為D隊，以茲悉別。直至1950年代末，由於警隊編制逐漸擴大，警隊方把地區編隊隊名取消<ref>[http://www.police.gov.hk/offbeat/919/chi/n24.htm 香港威海衞警察 威名遠播] 《警聲》 第919期</ref>。

據一些退役人員憶述，其魯籍外祖父參加警察招募面試時，須把雙手伸出，讓招募[[警官|警官]]觸摸。據說手粗的候選人後來多能獲得聘用，原因是考官傾向挑選吃過苦頭的人出任警員<ref>[http://www.metrohk.com.hk/index.php?cmd=detail&id=183400 梁振英 獅子山下的獅子] 《都市日報》2012年3月27日</ref>。

== 執勤情況 ==
當時的魯籍人員多不諳[[英語|英語]]或是[[粵語|粵語]]，對[[香港社會|香港社會]]了解有限。因此，魯籍人員往往被編成同一組別，其人員編號亦有別於一般[[廣東人|廣東人]]。由於言語障礙，魯籍人員較少擔任接觸市面的[[巡邏|巡邏]]勤務，而多成為衛哨部隊，被派往[[新界|新界]]，逐步取代[[摩羅差|印度籍警員]]、[[交通總部|交通部]]、[[衝鋒隊_(香港)|衝鋒隊]]或是[[香港島|香港島]]山頂[[警區|警區]]以及保衛[[港督府|港督府]]。

其中位於港島山頂，開埠初期主要由[[英國人|英國人]]聚居，基於歷史因素，粵籍人員一般不願往該處工作，有趣的是當時[[洋人|洋人]]亦不太喜歡粵籍人員，反而對身材魁梧和品格單純的魯籍人員較具好感。因此當時中西警區及山頂範圍常可看見魯籍人員執勤。當時來往香港及中國的英國商船，為提防[[中國|中國]]沿海[[海盜|海盜]]搶掠，會向香港當局僱用休更人員駐船，而被派遣上船的正是那些體格強健和飽經風浪的魯籍人員。

== 生活情況 ==
遠道來香港服役的魯籍人員身處異鄉，與本地華人的關係比較為疏離。當中不少熱衷學習[[英語|英語]]，以爭取晉升機會，當年不少魯籍人員在日後晉升至[[香港警長|警長]]職級。

在服務條件中，魯籍人員享受特別待遇，包括優先入伙剛落成的警察宿舍（例如[[荷李活道已婚警察宿舍|荷李活道已婚警察宿舍]]）、在香港服務滿3年後可以獲准休假一季及免費乘搭輪船返回家鄉探親。當中不少人會回鄉[[結婚|結婚]]，並且攜眷回來香港落地生根。此外，魯籍人員於退休後，可以獲得海外僱員津貼。

警隊為了應付魯籍人員的飲食習慣，當年有魯籍人員駐守的警署均聘有[[山東人|山東人]][[廚師|廚師]]，以供應山東口味的膳食。魯籍人員的食量非常驚人，有些人吃了兩份牛肉炒麵後，仍然會說「打了個底」而已。另外，每逢大時大節，在荷李活道已婚警察宿舍生活的魯籍人員的家庭均會包餃子，因此該處在節慶日子均會傳出剁肉聲音。

== 招募終結 ==
1949年[[中華人民共和國|中華人民共和國]]成立後，魯籍人員招募活動從此結束。可是至今，不少魯籍人員在港出生的後代，都繼續投效於香港警務處<ref>[http://www.police.gov.hk/offbeat/829/chi/h13.htm 香港警務處退役同僚協會專欄] 《警聲》 第829期</ref>。

== 軼事 ==
前[[香港特別行政區行政長官|香港特別行政區行政長官]][[梁振英|梁振英]][[籍貫|籍貫]][[山東|山東]][[威海|威海]]，其父[[梁忠恩|梁忠恩]]為當年被招募來香港的一份子，曾經駐守於[[港督府|港督府]]及[[太平山_(香港)|太平山]]。<ref>[http://weihai.sdnews.com.cn/2007/7/1/115221.html 香港高官梁振英的威海故土情—身居香港怀桑梓] 《威海晚报》 2007年7月1日</ref>梁忠恩晉升為[[警長|警長]]後，曾經駐守鴨巴甸街警察宿舍。
