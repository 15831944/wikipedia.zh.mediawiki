'''10號幹線'''（{{lang-en|'''Route 10'''}}）是[[香港|香港]]最新的主要公路幹線，東角頭-藍地段于2007年7月1日通车，亦是唯一一條全線是快速公路，以及部分路段在香港境外範圍的幹線（起點位於[[深圳市|深圳市]][[南山區_(深圳市)|南山區]]東角頭），包括[[深圳灣公路大橋|深圳灣公路大橋]]（[[深港西部通道|深港西部通道]]）及[[港深西部公路|港深西部公路]]（前稱后海灣幹線）、連接[[屯門|屯門]][[藍地|藍地]]的[[元朗公路|元朗公路]]及[[深圳市|深圳市]][[南山區|南山區]][[蛇口|蛇口]]的[[深圳灣口岸|深圳灣口岸]]港方口岸區，全長10.9公里。幹線全線均為三線雙程分隔行車路。

== 歷史 ==
[[File:No10.jpg|thumb]]亦屬10號幹線]]
最初於1997年提出的10號幹線的較詳細計劃，是由[[香港島|香港島]]的[[堅尼地城|堅尼地城]]開始，經過曾經倡議的[[青洲_(中西區)|青洲]]發展區、青洲連接路（連接[[交椅洲|交椅洲]]的隧道）<ref>[http://www.policyaddress.gov.hk/pa97/chinese/cptb.htm 一九九七年政策綱領]，[[運輸及房屋局|運輸局]]</ref>、位於[[交椅洲|交椅洲]]及[[竹篙灣|竹篙灣]]的擬建[[大嶼山|大嶼山]]港口、橫跨[[馬灣海峽|馬灣海峽]]至[[青龍頭|青龍頭]]的大橋（後稱[[青龍大橋|青龍大橋]]）<ref>{{cite web |url=http://www.pland.gov.hk/pland_en/p_study/comp_s/swnt/digest/civ.htm |title=主要規劃事項 |work=新界西南發展策略檢討 |accessdate=2018-11-30}}</ref>、青龍頭至[[掃管笏|掃管笏]]的公路（後稱大欖涌隧道）、掃管笏交匯處（連接[[屯門公路|屯門公路]]），最後經隧道（後稱藍地隧道）與近[[天水圍|天水圍]]的[[元朗公路|元朗公路]]交匯（後來改為藍地交匯處），並連接當時計劃的后海灣連接路（今[[港深西部公路|港深西部公路]]）直達深圳。<ref>[http://www.legco.gov.hk/yr97-98/chinese/panels/plw/papers/pl15012a.htm 十號幹線-北大嶼山至元朗公路的討論文件]，臨時立法會交通事務委員會及規劃地政及工程事務委員會，1998年1月15日</ref><ref>[http://www.legco.gov.hk/yr97-98/chinese/fc/pwsc/papers/pw131172.htm 519TH ─ 十號幹線 ─ 北大嶼山至元朗公路]，臨時立法會財務委員會工務小組委員會討論文件，1997年11月12日</ref>

此後數年，隨著規劃的更改、市民及議員的反對，以及交通需求預測放緩，10號幹線多項設計被修改，甚至被擱置。政府曾於2000年宣佈開展進行10號幹線（元朗至北大嶼山段南段）（即[[北大嶼山|北大嶼山]]至掃管笏）的工程，年間共收到500多份反對書。[[香港特別行政區政府|政府]]曾修訂路線，刪除小欖連接路，並在2002年6月28日刊憲。<ref>[http://www.hk-place.com/view.php?id=319 道路及鐵路－香港幹線公路(二)]，香港地方</ref>由於政府預測交通需求增長放緩，道路飽和情況比最初估計輕微，加上[[大欖隧道|大欖隧道]]的營運者反對，最終不獲立法會議員支持興建，因此興建計劃需要押後。而政府於2004年作出「二零零四年新界西北交通及運輸基建檢討」後，亦建議對元朗至北大嶼山段作出多項檢討。<ref>[http://www.legco.gov.hk/yr04-05/chinese/panels/tp/papers/tp0318cb1-1096-7c.pdf 二零零四年新界西北交通及運輸基建檢討]，立法會交通事務委員會討論文件，2005年3月18日</ref>

至於堅尼地城至大嶼山段，由於青洲填海區計劃被擱置，而原定興建[[葵青貨櫃碼頭#碼頭|10號貨櫃碼頭]]的填海用地已被改為興建[[香港迪士尼度假區|香港迪士尼度假區]]，而且大嶼山東南發展及人口增長比預期放緩，因此這一段相信也會擱置興建。<ref>於《二零零四年新界西北交通及運輸基建檢討》內，已沒有提及興建此段。</ref>

然而，為配合[[深港西部通道|深港西部通道]]的通車，后海灣幹線（現稱為港深西部公路）率先於2003年動工興建，亦已於2007年7月1日通車。而這兩條道路亦納入10號幹線，成為10號幹線首段通車的道路。如果計劃中路段落實，10號幹線亦會和4號幹線及8號幹線連接。

隨著[[中華人民共和國交通運輸部|中華人民共和國交通運輸部]]啟動全國高速公路統一命名和編號工作，10號幹線位處港深海界以北長約1.6公里的段落（即深圳灣大橋深圳段）也列為[[廣深沿江高速公路|廣深沿江高速公路]]的一部份，並在屬深圳區域但設為港方口岸區的開始點設有一以[[新字形|新字形]]標示「粵港澳高速公路」終點路牌，<ref>[http://blog.big5.voc.com.cn/blog.php?do=showone&uid=2703&type=blog&itemid=303653 通往香港的新大橋]</ref><ref>[http://www.gdcx.gov.cn/web/gov/JourneyResource/HighWayNumChange.aspx 广东省高速公路网命名与编号调整] {{webarchive|url=https://web.archive.org/web/20130225101221/http://www.gdcx.gov.cn/web/gov/JourneyResource/HighWayNumChange.aspx |date=2013-02-25}} [[广东省交通运输厅|广东省交通运输厅]]</ref>代表進入香港領域。

== 未來發展 ==
===元朗至香港島段（已改為11號幹線）===
{{see also|東大嶼都會|香港11號幹線}}
政府亦曾計劃興建元朗至大嶼山段及大嶼山至香港島段，使10號幹線成為一條連接[[香港島|香港島]]、[[大嶼山|大嶼山]]和[[新界西|新界西]]的幹線公路，全長29.5公里。路段由[[藍地隧道|藍地隧道]]、[[掃管笏交匯處|掃管笏交匯處]]、[[大欖涌隧道|大欖涌隧道]]、[[青龍大橋|青龍大橋]]、[[竹篙灣連接路|竹篙灣連接路]]及[[青洲隧道|青洲隧道]]組成。當中掃管笏至大嶼山段原已於2002年刊憲，惟受立法會反對下而被擱置。

直至2017年1月，政府宣佈重新研究興建一段連接[[北大嶼山|北大嶼山]]至[[元朗|元朗]]之公路幹線，惟相關路段將會被納入未來的[[香港11號幹線|11號幹線]]。<ref>{{cite news|title=政府將申請數千萬進行11號幹線可行性研究|publisher=[[商業電台|商業電台]]|date=2017年1月18日|url=http://www.881903.com/Page/ZH-TW/newsdetail.aspx?ItemId=918510&csid=261_341|accessdate=2017年1月19日}}</ref>

===元朗至赤鱲角段（興建中）===
政府於2007年宣佈興建[[屯門至赤鱲角連接路|屯門至赤鱲角連接路]]和[[屯門西繞道|屯門西繞道]]，有關路段將取代前述由元朗經掃管笏往大嶼山的走線，成為10號幹線的延長段。其中[[屯門至赤鱲角連接路|屯門至赤鱲角連接路]]於2013年動工，預計2020年通車；[[屯門西繞道|屯門西繞道]]至今仍在檢討走線，未有動工時間表。

元朗至赤鱲角段全段通車後，10號幹線將會與[[香港8號幹線|8號幹線]]連接，同時接通[[深圳灣口岸|深圳灣口岸]]和[[港珠澳大橋香港口岸|港珠澳大橋香港口岸]]，[[珠海|珠海]]和[[澳門|澳門]]可以經香港往返[[深圳|深圳]]。

== 路線 ==
[[File:Kong_Sham_Western_Highway_reserved_joint_to_route_10.jpg|thumb]]連接10號幹線主線預留位]]
[[File:Guangdong_–_Hong_Kong_–_Macau_Highway_End_Point_sign.jpg|thumb]]
10號幹線起始於深圳市的[[深圳灣口岸|深圳灣口岸]]香港口岸區，沿深港西部通道跨過[[后海灣|后海灣]]（深圳灣），然後經港深西部公路到達[[元朗公路|元朗公路]]近[[藍地交匯處|藍地交匯處]]，與9號幹線連接。

由廈村交匯處以西的一段10號幹線為封閉道路，只限持有通行許可證的車輛及[[香港的士|市區、新界的士]]進入。 <ref>[http://www.td.gov.hk/tc/transport_in_hong_kong/access_to_shenzhen_bay_port/index.html 前往深圳灣口岸]，[[運輸署|運輸署]]</ref>
