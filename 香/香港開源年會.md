'''香港開源年會'''（{{lang|en|Hong Kong Open Source Conference}}，簡稱 HKOSCon），是在[[香港|香港]]舉辦的開放源碼年度[[研討會|研討會]]，旨在推動[[開放源碼|開放源碼]]開發，促進社群交流，並鼓勵工商業界使用開放源碼[[軟件|軟件]]及方案。

活動每年都會就不同主題邀請業界代表參與，分享[[開放源碼|開放源碼]]趨勢及技術發展，資訊科技業界及其他技術組織亦參與協辦，再加上大專院校、熱心支持者、贊助商捐贈和支援，造就每年一次活動。由最初的演講、講座、{{link-en|閃電講|Lightning Talk}}，拓展至加入工作坊、[[非會議|非會議]]、[[聚會|聚會]]和開放源碼方案推廣攤位的規模。

== 年會 ==
=== 2013 ===
於 2013 年[[開源香港|開源香港]]與開放系統研用協會及[[香港Linux用家協會|香港Linux用家協會]]合辦第一屆香港開源年會。會議於10月19日 <ref>{{cite web |title=Announcement of Hong Kong Open Source Conference 2013 (Agenda v1) - 第一屆香港開源年會 (HKOSC) 2013 v1 議程公佈 |url=http://www.linux.org.hk/archive/20130914-012639.html |website=Hong Kong Linux User Group |accessdate=2013-09-24}}</ref> 在[[香港城市大學|香港城市大學]]舉行，全日三軌會議超過 150 人參與，主講題目包括[[開放數據|開放數據]]、開放互聯網、[[開源硬件|開源硬件]]及探討開放源碼在程式設計、教育、遊戲及政策層面所扮演的角色。在徵求提案階段大會收到遠自歐洲的講者投稿，最終採納 [https://en.wikipedia.org/wiki/Free_Software_Initiative_of_Japan Free Software Initiative of Japan]、台灣[[中央研究院|中央研究院]]及[[臺灣大學|臺灣大學]]的稿件，連同香港本地代表，合共有 20 份講稿被納入議程。

=== 2014 ===
於 2014 年接受[[香港互聯網協會|香港互聯網協會]]舉辦的 World Internet Developer Summit ( WiDS ) 邀請作為三日活動當中其中一項活動並於3月29日 <ref>{{cite web |title=Hong Kong Open Source Conference 2014 |url=https://opensource.hk/hong-kong-open-source-conference-2014/ |website=Open Source Hong Kong |accessdate=2014-12-31}}</ref> 在[[香港科學園|香港科學園]]舉行，主辦方為[https://cota.hk/ 香港創意開放科技協會]，並由[[開源香港|開源香港]]及[[香港Linux用家協會|香港Linux用家協會]]協辦。當日舉辦了 23 場會議，其中 18 位講者來自外地，13 個位講者來自本地，最終參與人數為 228 人。

=== 2015 ===
從 2013 及 2014 會期一天演變為兩天的活動，由香港創意開放科技協會、[[香港Linux用家協會|香港Linux用家協會]]、[[開源香港|開源香港]]及[[香港科技園公司|香港科技園公司]]共同主辦，於6月26及27日 <ref>{{cite web |title=HKOSCon 2015 |url=http://2015.opensource.hk/ |website=HKOSCon 2015}}</ref> 在[[香港科學園|香港科學園]]舉行。是年會議口號為 Connect Global Open Source Technology in Hong Kong.，以[[雲端|雲端]]、[[自造者|自造者]]、社群、新創企業 ( Cloud // Maker // Community // Startup ) 四大主題為基礎，27 位海外講者及 25 位本地講者在兩日五軌會議合共舉辦 52 節演講及工作坊，並由參展商及社群設立多個展覽攤位，參與人數總共為 538 人。<ref>{{cite web |title=Hong Kong Open Source Conference (HKOSCon) 2015 |url=https://cota.hk/events/hong-kong-open-source-conference-hkoscon-2015/ |website=香港創意開放科技協會 |accessdate=2015-08-07}}</ref>
是年有來自美國的 [[LWN.net|LWN.net]] 創辦人 Jonathan Corbet 及來自德國的 [[GNOME|GNOME]] 基金會董事會前成員 Tobias Mueller 的主題演講。來自俄羅斯國立托姆斯克大學及 [https://en.wikipedia.org/wiki/ALT_Linux Alt Linux] 的失明開發者 Michael Pozhidaev 分享開發給失明及視障人士使用的開源方案，多位來自歐洲各地、日本及香港的自造者分享不同講題，例如西班牙的廢物再造開源飛行器，意大利的開源汽車，保加利亞的 [[Internet_of_Things|Internet of Things]]、以及香港的開源教育三維列印機等等。<ref>{{cite web |author1=Editor |title=香港開源年會 2015 擁抱全球開放源碼專案 |url=https://www.linuxpilot.com/hkos-con-2015 |website=Linuxpilot |accessdate=2015-08-31}}</ref>

=== 2016 ===
是年開始香港開源年會固定由香港創意開放科技協會、[[香港Linux用家協會|香港Linux用家協會]]、[[開源香港|開源香港]]共同主辦，於6月24-26日 <ref>{{cite web |title=HKOSCon 2016 香港開源年會 |url=http://2016.opensource.hk/ |website=HKOSCon 2016 香港開源年會}}</ref> 在[[香港數碼港|香港數碼港]]舉行。口號為 Go Beyond Open-source，圍繞著[[Internet_of_Things|物聯網]]、[[大數據|大數據]]、[[雲端|雲端]]和 DevOp ( Internet of Things // Data // Cloud // DevOps ) 四大主題，三天四軌會議有 23 位外籍講者及 14 位本地講者參與 ，參與人數總共為 355 人。<ref>{{cite web |title=Hong Kong Open Source Conference (HKOSCon) 2016 |url=https://cota.hk/events/hong-kong-open-source-conference-hkoscon-2016/ |website=香港創意開放科技協會}}</ref>
主題演講包括來自澳洲的系統工程師 Joshua Rich 介紹 [[Elasticsearch|Elasticsearch]] 開源搜尋引擎 <ref>{{cite web |author1=Editor |title=Elastic 系統工程師 Joshua Rich 專訪 |url=https://www.linuxpilot.com/interview-joshua-rich-elastic-system-engineer |website=Linuxpilot |accessdate=2016-07-06}}</ref> 及台灣知名 Linux 核心開發高手黃敬群 ( Jserv ) 主講《回顧台灣自由開源軟體發展》。<ref>{{cite web |author1=Ken Li |title=推廣本地開源文化　HKOSCon 2016 香港開源年會雲集技術精英交流 |url=https://unwire.pro/2016/06/16/hkoscon-2016/news/ |website=Unwire.Pro |accessdate=2016-06-16}}</ref>

=== 2017 ===
是年會議於6月9-10日 <ref>{{cite web |title=HKOSCon 2017 |url=https://hkoscon.org/2017/ |website=HKOSCon 2017}}</ref> 在[[香港數碼港|香港數碼港]]舉行，口號為 Embracing Open Source's Success，專注示範及推廣世界最新開放源碼軟件及硬件技術。第一天主題為商業及教育，第二天為社群交流，活動亦吸引企業到場招聘。<ref>{{cite web |author1=林炳坤，梁逸風 |title=港企業不重視IT　辦開源年會自強「唔希望搞到叻人去哂外國」 |url=https://www.hk01.com/%E7%A4%BE%E6%9C%83%E6%96%B0%E8%81%9E/95930/%E6%B8%AF%E4%BC%81%E6%A5%AD%E4%B8%8D%E9%87%8D%E8%A6%96it-%E8%BE%A6%E9%96%8B%E6%BA%90%E5%B9%B4%E6%9C%83%E8%87%AA%E5%BC%B7-%E5%94%94%E5%B8%8C%E6%9C%9B%E6%90%9E%E5%88%B0%E5%8F%BB%E4%BA%BA%E5%8E%BB%E5%93%82%E5%A4%96%E5%9C%8B |website=HK01 |accessdate=2017-06-06}}</ref>

=== 2018 ===
