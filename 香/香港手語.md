'''香港手語'''特指[[香港|香港]]定居的聾人和弱聽人士使用的語言。香港手語在1950年代發展，由逃避戰亂的[[大陆人|大陆人]]帶來。現今的香港手語，是1950年代的香港手語和南京/上海的[[中國手語|中國手語]]混合發展而成。<ref>http://muse.jhu.edu/login?auth=0&type=summary&url=/journals/sign_language_studies/v013/13.2.sze.pdf{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 簡史 ==
手語一樣有語法、有詞彙；因歷史文化的演變，更發展出多套獨特的語言系統<ref>{{cite web |language=zh-hk | url=http://hk.apple.nextmedia.com/news/art/20140917/18869185 | title= 恐怖日佔回憶　用手講更血淋淋 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] | date=2014-09-17 | accessdate=2014-09-18}}</ref>
。
* 1880年，國際聾人教育會議廢除手語教學，認為它會妨礙聾人融入社會。自此130多年，手語都被遏制。
* 1960年代，香港聾童學校以教讀唇為主，聾童卻自行發展出一套日用手語，引致不同群體使用不同手語的問題。
* 2010年，國際聾人教育會議向聾人道歉，認為他們都擁有使用手語的權利。
* 2011年，有網站將香港手語列入瀕危語言，當時數字表示僅剩9,000個使用者，最新數據更指只剩餘3,900人。

== 自然手語與中文手語的比較 ==
現在愈來愈多[[香港人|香港人]]有興趣學習手語，由於香港流通的手語主要有兩種，一種是中文手語，另一種是自然手語。身為手語導師[[朱芷茵|朱芷茵]]接受訪問時指出，中文手語多數是用於與其他來自以中文為母語的地區的聾人交流，但未必所有聾人都有一定的語文水平，以中文手語交流難免有一定的障礙，她建議有興趣學習手語的香港人，以自然手語為學習對象，更易促進健聽人士與聾人的溝通<ref>{{Cite news|language=zh-hk |url=https://hk.lifestyle.appledaily.com/lifestyle/special/daily/article/20170210/19923414|title=【潮語手語】考吓你「你識條鐵」　點樣打？|publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]]|date=2017-02-10|accessdate=2018-06-23}}</ref>。

== 香港手語復興運動 ==
目前香港手語正在受嚴峻的威脅。根據[[香港統計處|香港統計處]]出版的2014年綜合住戶統計調查數據指出<ref>政府統計處《[http://www.statistics.gov.hk/pub/B11301622014XXXXB0100.pdf 第六十二號專題報告書]》，詳見第78頁的第6.9段。</ref>，155,000人有不同程度的聽覺困難，其中4,300人已全聾。直到2011年，有網站將香港手語列入瀕危語言，當時數字表示僅剩9,000個使用者，最新數據更指只剩餘3,900人。有慈善機構「[[龍耳|龍耳]]」推廣「兩文四語」，兩文為[[中文|中文]]和[[英文|英文]]，四語為[[粵語|粵語]]、[[英語|英語]]、[[普通話|普通話]]和香港手語，也促請[[香港政府|香港政府]]增撥資源支持手語教育。此外，培訓了10位以手語作為母語的聾人，擔任手語導師，並提供手語班，不但為聽覺有困難人士提供無障礙的社會，也藉此希望挽救香港手語的瀕危語言<ref>{{cite web | language=zh-hk | url=http://www.singpao.com/xw/gat/201202/t20120205_324524.html | title=「龍耳」推廣兩文四語 | publisher=[[成報|成報]] | date=2012-02-05 | accessdate=2014-09-28 | deadurl=yes | archiveurl=https://archive.is/20130428002451/http://www.singpao.com/xw/gat/201202/t20120205_324524.html | archivedate=2013-04-28 }}</ref>。

== 香港手語研究及推廣 ==
[[香港中文大學手語及聾人研究中心|香港中文大學手語及聾人研究中心]]獲政府資助，成功研發香港首本「手語字典」─“香港手語瀏覽器”，更是全亞洲首個全面記錄各種手語變體的資料庫<ref>{{cite web |language=zh-hk | url=http://paper.wenweipo.com/2013/01/24/ED1301240010.htm | title= 中大「手語字典」 網上短片教公眾 |publisher=[[文匯報|文匯報]] | date=2013-01-24 | accessdate=2015-09-11}}</ref>。瀏覽器內含逾1,100個日常香港手語詞彙，並設有手語短片供公眾學習<ref>{{cite web | language=zh-hk | url=http://news.sina.com.hk/news/20130124/-2-2879398/1.html | title=中大推網上手語字典 | publisher=[[明報|明報]] | date=2013-01-24 | accessdate=2015-09-11 | deadurl=yes | archiveurl=https://web.archive.org/web/20151225191351/http://news.sina.com.hk/news/20130124/-2-2879398/1.html | archivedate=2015-12-25 }}</ref>。該中心希望籍此推廣香港式聾人手語，消除香港大眾對手語的誤解，以促進香港社會“聾健共融”<ref>{{cite web |language=zh-hk | url=http://www.hkcd.com.hk/content/2013-01/24/content_3131515.htm | title= 推廣香港式聾人手語 香港首個手語瀏覽器誕生 |publisher=[[香港商報|香港商報]] | date=2013-01-24 | accessdate=2015-09-11}}</ref>。

== 香港手語翻譯員名單 ==
{{main|香港手語翻譯員名單}}
[[香港社會服務聯會|香港社會服務聯會]]、[[香港復康聯會|香港復康聯會]]和[[勞工及福利局|勞工及福利局]]康復諮詢委員會於2015年共同設立《[[香港手語翻譯員名單|香港手語翻譯員名單]]》，列載在香港可提供手語翻譯服務的人士。手語翻譯員只要在申請日起計，過去兩年內提供不少於200小時的手語翻譯服務，並持有由受僱機構簽發的證明文件，便可把個人資料加入此名單中，與註冊的翻譯員看齊，為聾人提供翻譯服務。聾人機構「[[龍耳|龍耳]]」創辦人[[邵日贊|邵日贊]]希望政府增撥資源，培訓更多手語翻譯員及發展視像翻譯服務<ref>{{cite web |language=zh-hk | url=http://hk.on.cc/hk/bkn/cnt/news/20160317/bkn-20160317121015706-0317_00822_001.html | title= 聾人的一天　接駁有聲世界困難重重 |publisher=東網電視 | date=2016-03-17 | accessdate=2016-03-31}}</ref><ref>{{cite web |language=zh-hk | url=http://hk.on.cc/hk/bkn/cnt/news/20160318/bkn-20160318103553931-0318_00822_001.html | title= 手語不統一政府不支援　翻譯認可路難走 |publisher=東網電視 | date=2016-03-18 | accessdate=2016-03-31}}</ref>。
