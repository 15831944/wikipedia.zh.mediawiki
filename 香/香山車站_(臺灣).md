'''香山車站'''位於[[臺灣|臺灣]][[新竹市|新竹市]][[香山區|香山區]]。為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(北段)|縱貫線]]的[[鐵路車站|鐵路車站]]。[[台灣日治時期|日治時期]]當局興建[[縱貫鐵路|縱貫鐵路]]，[[明治|明治]]三十五年（1902年）[[新竹車站|新竹]]～中港（今[[竹南車站|竹南]]）間通車，同時設置香山車站。現今站體於[[昭和|昭和]]三年（1928年）完工，目前為[[新竹市文化資產#市定古蹟|市定古蹟]]。<ref name="走讀台灣－新竹市">{{cite book |author = [[韓國棟|韓國棟]] |title = 《走讀台灣－新竹市》 |publisher = [[國家文化總會|國家文化總會]] |year = 2009 |isbn =  9789866573156 }}</ref>{{Rp|168}}<ref name="台灣的古蹟：北台灣"/>{{Rp|50}}

== 歷史 ==
明治三十二年（1899年）11月8日[[臺灣總督府|臺灣總督府]]成立[[臺灣總督府鐵道部|臺灣總督府鐵道部]]{{NoteTag|1=[[臺灣總督府交通局鐵道部|臺灣總督府交通局鐵道部]]於[[大正|大正]]十三年（1924年）前直屬於[[臺灣總督府|臺灣總督府]]，稱為臺灣總督府鐵道部。}}，同時開始鋪設[[縱貫線_(鐵路)|縱貫線]]的計畫。三十五年（1902年）8月10日[[新竹車站|新竹]]～中港（今[[竹南車站|竹南]]）間通車，同時設置香山車站。[[昭和|昭和]]三年（1928年）1月現今站體完工<ref name="台灣的老火車站">{{cite book |author = [[戴震宇|戴震宇]] |title = 《台灣的老火車站》 |publisher = 遠足文化 |year = 2005 |isbn = 957304935X }}</ref>{{Rp|28}}，同時建有[[倉庫|倉庫]]與[[宿舍|宿舍]]等附屬建築<ref name="風城遊">{{cite book |author = [[陳銘磻|陳銘磻]] |title = 《風城遊》 |publisher = 愛書人雜誌 |year = 2004 |isbn = 9867430050}}</ref>{{Rp|215}}。五年（1930年）[[新竹車站|新竹]]~香山間雙軌化完工。六年（1931年）香山~[[崎頂車站|崎頂]]間雙軌化完工。1990年3月1日停辦路料之外的貨運業務<ref name="臺灣區鐵道古今站名詞典">{{cite book|author=[[楊鵬飛|楊鵬飛]] |title=《臺灣區鐵道古今站名詞典》|year=1999 |publisher=[[正中書局|正中書局]] |isbn=9789579736466 }}</ref>{{Rp|38}}。1991年10月1日降為甲種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]，並由新竹車站管理<ref name="臺灣鐵路車站圖誌：全臺294個現有車站完整記錄">{{cite book |author = [[蘇昭旭|蘇昭旭]] |title = 《臺灣鐵路車站圖誌：全臺294個現有車站完整記錄》 |publisher = 人人 |year = 2000 |isbn = 9867916093 }}</ref>{{Rp|32}}。2007年2月12日臨時[[天橋|天橋]]完工。2013年5月30日啟用[[多卡通|多卡通]]刷卡機。2013年9月站房開始進行整修，同時去除外牆油漆，於2014年7月竣工。

== 車站構造 ==
香山車站現役站房建於[[昭和|昭和]]三年（1928年）1月<ref name="臺灣鐵路車站圖誌：全臺294個現有車站完整記錄"/>{{Rp|32}}，占地約44[[坪|坪]]<ref name="風城遊"/>{{Rp|215}}，為木造[[日式建築|日式建築]]，以[[阿里山|阿里山]]的[[檜木|檜木]]打造，是[[臺灣|臺灣]]僅存的檜木造車站<ref name="文化科技花園城市：新竹市觀光手冊">{{cite book |author = [[姚霞芬|姚霞芬]]、[[何儀琳|何儀琳]]、[[林柏成|林柏成]] |title = 《文化科技花園城市：新竹市觀光手冊》 |publisher = [[新竹市政府|新竹市政府]] |year = 2005 }}</ref>{{Rp|139}}<ref name="走讀台灣－新竹市"/>{{Rp|168}}。屋頂覆[[日本黑瓦|日本黑瓦]]<ref name="台灣的老火車站"/>{{Rp|28}}，兩側具[[入母屋造|入母屋造]]樣式{{NoteTag|1=入母屋造頂自中國[[歇山頂|歇山頂]]發展而來，是[[日式建築|日式建築]]建築常用的屋頂形式。由一條正脊，四條垂脊，四條斜脊構成<ref name="台灣的古蹟：北台灣"/>{{Rp|151}}，在屋頂左右兩端的[[山牆|山牆]]處形成梯形的[[屋簷|屋簷]]。}}。站房外側另加一圈迴廊，並以木構架支撐<ref name="台灣的古蹟：北台灣"/>{{Rp|50}}。入口位於站房端部，設有[[玄關|玄關]]，上置三角形[[博风板|破風]]<ref name="台灣的老火車站"/>{{Rp|28}}，相同的設計尚有[[縱貫線_(南段)|縱貫線]][[保安車站|保安車站]]、[[集集線|集集線]][[集集車站|集集車站]]與[[阿里山森林鐵路|阿里山森林鐵路]][[北門車站_(嘉義市)|北門車站]]<ref name="臺灣鐵路車站圖誌：全臺294個現有車站完整記錄"/>{{Rp|32}}。玄關木柱以雕花[[牛腿|牛腿]]支撐<ref name="台灣的古蹟：北台灣"/>{{Rp|50}}，破風下另開兩扇小窗，並以木條排成格式[[窗櫺|窗櫺]]。2001年前破風曾被大型招牌遮擋，後更換為較小的招牌，整修後已拆除招牌。

值班站長室內尚保存[[名片式車票|名片式車票]]票櫃（包括售票櫃、去回票櫃與儲票櫃）、[[天虎工業|天虎工業]]製的名片式車票日期戳章與[[檜木|檜木]]製[[辦公桌|辦公桌]]等古物。<ref name="走讀台灣－新竹市"/>{{Rp|169}}

===月臺===
香山車站擁有[[島式月台|島式月台]]兩座，曾一度無[[遮雨棚|遮雨棚]]<ref name="台灣的老火車站"/>{{Rp|29}}。於2010年完成月臺加長工程。

== 鄰近車站 ==
===廢止營運模式===
