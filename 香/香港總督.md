'''香港總督'''（簡稱'''港督'''，民間俗稱“督爺”，尊稱'''督憲'''或'''督憲閣下'''），前身為'''香港行政官'''（共有3任），是[[香港殖民地時期|香港殖民地時期]]（1841－1997），由[[英國|英國]]派駐[[香港|香港]]的[[英国君主|英国君主]]代表，共歷28任，当中第26任港督尤德爵士是唯一一位在任期间逝世的港督。

早期港督都是由英國殖民地部（後稱「[[外交及聯邦事務部|外交及聯邦事務部]]」）的官員出任，1971年以後港督大多是[[外交官|外交官]]出身，[[麥理浩|麥理浩]]勳爵是第一位；而最後一任港督[[彭定康|彭定康]]則是國會[[議員|議員]]出身，曾任[[英國保守黨|保守黨]]主席。

香港在[[英屬香港|英治時期]]共28位總督中，除第1至2、3至4任以外，其餘每個正任總督的交接時間皆不是完美交接，而是中間隔著一至兩位[[署理|署理]]總督。署理總督早期則由[[香港副總督|香港副總督]]暫代，該職位是由[[駐港英軍|香港駐軍]]司令擔任；1870年代香港戰略地位減退，故此於1870年至1902年該職位為非常設職位，1902年後香港副總督的職位正式被撤銷。自此，署理總督多由[[香港布政司|輔政司]]（1976年改名為布政司）擔任。

== 象徵 ==
香港總督的象徵主要為[[香港總督旗|香港總督旗]]及[[香港盾徽|香港盾徽]]；香港總督旗曾用過[[阿群帶路圖|阿群帶路圖]]，經過一次修改，而香港總督專用座駕是以[[聖愛德華皇冠|聖愛德華皇冠]]<ref>英國國徽為[[英國皇家徽章|英國皇家徽章]]</ref>作為[[香港車牌#無字母及數字|車牌]]，以示其代表英國君主。

== 權力 ==
{{main|英皇制誥 (香港)|皇室訓令}}
[[File:HK_History_of_HK_Museum_Uniform_of_HK_Governor_Edward_Youde.JPG|right]]藏]]

1843年，《[[英皇制誥_(香港)|英皇制誥]]》頒佈，宣佈設置香港殖民地。英王是香港的最高統治者，總督則是英王的全權代表，兼任[[駐港英軍|駐香港三軍]]總司令，而在1841年至1857年期間，香港總督更兼任[[英國駐華大使|英國駐清國全權公使]]。總督權力極大，主持香港的行政機關[[行政局_(香港)|行政局]]和立法機關[[立法局|立法局]]，並委任兩局的議員。香港最高行政權在於[[總督會同行政局|總督會同行政局]]，而行政局與立法局有別，全部議員一直由總督所委任。兩局在總督施政上，只會擔任顧問及輔助角色。而總督亦為立法局（至1993年止）及行政局主席。在1985年前，所有立法局議員皆由港督委任，直至1991年立法局才第一次以[[直選|直選]]選出部份議員；而局上所討論的一切[[議案|議案]]或[[法案|法案]]最終需要得到總督同意並簽署才可通過。換言之，總督對所有議案和法案皆有[[否決權|否決權]]<ref>《[[英皇制誥_(香港)|英皇制誥]]》第8至12條</ref>。

除了兩局議員外，總督亦有委派[[法官|法官]]和[[太平紳士|太平紳士]]的權力和對政府所有[[香港公務員|公務員]]（除[[輔政司|布政司]]、[[財政司|財政司]]、[[律政司司長|律政司]]和[[駐港英軍#歷任英國駐軍司令|英國駐港三軍司令]]）進行紀律行動（如罰款）甚至解除其公職。此外，總督亦有[[赦免|赦免]][[囚犯|囚犯]]（包括[[死囚|死囚]]）和免除罰款的權力。更甚的是，香港作為英國殖民地，所有[[官地|官地]]屬英王所有，而因為總督為英王駐港代表，所以總督有權將官地授予他人及機構<ref>《[[英皇制誥_(香港)|英皇制誥]]》第13條</ref>（如[[保良局|保良局]]等）。

總括而言，香港總督比起其他英國屬土的總督有更大的行政管治權力<ref>《澳洲SBS電台"知法守法"節目 - 誰是總督》[http://www.sbs.com.au/yourlanguage/cantonese/zh-hant/content/%E3%80%90%E7%9F%A5%E6%B3%95%E5%AE%88%E6%B3%95%E3%80%91%E8%AA%B0%E6%98%AF%E7%B8%BD%E7%9D%A3%EF%BC%9F?language=zh-hant]</ref>。

1992年末任港督彭定康上任後大幅改變以往做法，包括不接受[[爵士|爵士]]封銜、就職典禮不身穿傳統白色羽毛禮服，只穿[[西裝|西裝]]、不再兼任立法局主席、取消委任[[立法局|立法局]]、[[市政局_(香港)|市政局]]、[[區域市政局|區域市政局]]、[[香港區議會|區議會]]議員，又加設立法局答問大會、市民答問大會，親自答覆議員、市民質詢等，很大程度將自己「下降」為香港的[[政府首腦|行政首長]]{{efn|法理上，香港政府首長布政司，主管[[布政司署|布政司署]]。}}。

== 義務 ==
[[File:HK_Governor's_salary_of_1860.jpg|right]]
香港總督義務擔任社會上一些團體或組織的名譽負責人，如自[[梅含理|梅含理]]起成為[[香港童軍總會|香港童軍總會]]總領袖，自[[盧吉|盧吉]]起出任[[香港大學|香港大學]]校監，其後延伸至香港所有大學。

== 福利及待遇 ==
*會獲編配[[官邸|官邸]][[香港禮賓府|總督府]]；在[[粉嶺|粉嶺]]也有[[行政長官粉嶺別墅|別墅]]。
*坐駕沒有[[香港車輛登記號碼#無字母及數字|車牌]]，以[[聖愛德華皇冠|聖愛德華皇冠]]代替。
*於[[香港特別行政區排名表#香港排名表（1997年7月1日前最後版本）|香港排名表]]排名第一。
*薪水固定為[[布政司|布政司]]月薪之125%。

== 中文譯名由來 ==
{{see also|英國政治人物香港譯名列表|漢化譯名}}
英國早期的華裔移民都是來自[[香港圍村|香港圍村]]的移民。在[[第二次世界大戰|戰]]前，[[香港居民|香港居民]]都享有英國的居留權。不少圍村居民為改善生計，都移民到英國謀生。為服務這群不太會英語的香港移民，以及為香港當地的傳媒服務，由[[司徒拔|司徒拔]]爵士起，英國的殖民地部會為全國所有議員和官員編寫一份官方譯名表。由於香港人普遍使用[[粵語|粵語]]，官方譯名也以粵語的[[廣州音|廣州音]]作標準，並且會儘可能將譯名漢化，變成漢人常用的二字、三字名稱（如[[羅富國|羅富國]]、[[楊慕琦|楊慕琦]]、[[彭定康|彭定康]]等），使香港人有親切感，並期望香港華人認識並接受英國人的統治。
