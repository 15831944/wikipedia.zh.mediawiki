'''香港女子足球聯賽'''（[[英文|英文]]：'''Hong Kong Women League'''）成立於1986年，原本由香港女子足球總會主辦。於2012年復辦，並由[[香港足球總會|香港足球總會]]接辦女子足球聯賽，結果[[公民足球隊|公民]]贏得復辦後首屆聯賽冠軍。

== 賽事歷史 ==
香港女子足球聯賽最初於1986年由香港女子足球總會主辦，第一屆賽事共有六支球隊參賽，冠軍為[[加山|加山]]。及後，由於香港女子足總組織紊亂，令女子足球聯賽斷斷續續地舉行。

於[[2009年東亞運動會|2009年東亞運動會]]，[[香港奧運足球代表隊|香港奧運男子足球代表隊]]歷史性為[[香港|香港]]奪得首項大型[[足球|足球]]賽事冠軍後，[[香港政府|香港政府]]推行《鳳凰計劃》以改革香港足球，女子足球亦被列進計劃之中<ref>{{cite web |title=香港足球的發展「我們是香港」 – 敢於夢想（最後報告） |url=http://www.hkfa.com/upload/20092010/consultancy/masterc.pdf |publisher=[[香港足球總會|香港足球總會]] |format=PDF}}</ref>。於2012年，[[香港足球總會|香港足球總會]]正式接辦香港女子足球聯賽，重新組織一個具規模的女子聯賽<ref>{{cite web |url=http://www.hkfa.com/zh-hk/load_page.php?pid=205 |title=2012-2013香港足球總會女子聯賽 |publisher=[[香港足球總會|香港足球總會]] |date=2012年12月18日}}</ref><ref>{{cite news |url=http://www.metrohk.com.hk/index.php?cmd=detail&id=198089 |title=女足聯賽下月中開鑼 |publisher=[[都市日報|都市日報]] |date=2012年10月30日}}</ref>。

復辦的首屆賽事，共有 10 支球隊參賽，其中以[[卜公體育會|卜公]]與[[公民足球隊|公民]]兩隊的實力最強<ref>[http://hk.apple.nextmedia.com/sports/art/20130117/18136938 卜公　公民女足最強]《[[蘋果日報|蘋果日報]]》</ref>。結果，[[公民足球隊|公民]]贏得復辦後首屆女子聯賽冠軍，亞軍為[[卜公體育會|卜公]]，季軍為[[車路士足球學校_(香港)|車路士足球學校 (香港)]]。其後公民女足更於2013至17年締造聯賽五連霸紀錄，直至[[傑志體育會|傑志]]榮獲[[2017–18年香港女子足球聯賽|2017–18球季]]冠軍。

從[[2018–19年香港女子足球甲組聯賽|2018–19球季]]開始，香港女子足球聯賽分成甲、乙組進行，每組最多 8 隊，故聯賽改以雙循環賽制進行。完成聯賽後，甲組榜末球隊及[[2018–19年香港女子足球乙組聯賽|乙組]]首名球隊將會以「一場過」附加賽角逐下季甲組席位。<ref>[https://www.upower.com.hk/article/236520 【香港女足聯賽】大改革﹗新季比賽分甲乙組-設升降制聯賽採雙循環]《UPower》</ref>

== 賽事制度 ==
聯賽以單循環形式進行，勝方得 3 分，賽和各得 1 分，負分得 0 分，最後以積分多寡來定出排名。排名第 1–5 位球隊可以競逐季後賽盃賽，而排名第 6–9 位球隊則競逐季後賽碟賽。

女子聯賽賽事每場為 90 分鐘，上、下半場各為 45 分鐘，每場比賽各球隊只可以派出 3 名非本地球員出場，另外每隊有 7 個替換球員名額。<ref>{{cite web |title=
二零一二/一三年度女子聯賽章則　【二零一二/一三年度版】|url=http://www.hkfa.com/upload/20122013/womenleague/rules.pdf |publisher=[[香港足球總會|香港足球總會]] |format=PDF}}</ref>
