'''香港國際互聯網交換中心'''（{{lang-en|'''Hong Kong Internet eXchange'''}}，[[縮寫|縮寫]]：'''HKIX'''）是一家位於[[香港|香港]]的[[互聯網|互聯網]]交換中心，由[[香港中文大學|香港中文大學]]資-{}-訊科技服務處負責<ref>[http://www.hkix.net/hkix/whatishkix.htm HKIX - What is HKIX]</ref>，[[總部|總部]]位於[[香港中文大學碧秋樓|香港中文大學碧秋樓]]。

現時逾99%香港境內的互聯網訊息交換在此直接交换傳輸而無需要繞經海外，從而節省時間和成本。<ref>[http://www.hkix.net/hkix/apricot-apan2011.pdf Hong Kong Internet Exchange (HKIX) Updates @APRICOT-APAN 2011] {{webarchive|url=https://web.archive.org/web/20130805081420/http://www.hkix.net/hkix/apricot-apan2011.pdf |date=2013-08-05 }}</ref>

==歷史==
香港互聯網交換中心前身為香港中文大學於1991年9月接駁美國的64Kbps[[專線|專線]]，由於互聯網於美國發展迅速，大學對於海外學術資料交流增加，於1992年年初，香港各大學亦相繼加入使用專線。同年年中，正式交由[[香港學術網絡|香港學術網絡]]（HARNET）接管，及由香港各大學聯合組成JUCC統籌管理有關香港互聯網有關規劃。後期[[網際網路服務提供商|網際網路服務提供商]]亦陸續以間接方式加入，至1995年4月，香港互聯網交換中心正式開始與各網際網路服務提供商直接連線。

2004年，位於[[葵涌|葵涌]][[葵福路|葵福路]]93號中信電訊大廈<ref>[http://www.hkix.net/hkix/contactus.htm HK Internet eXchange - Site Address: ]</ref>的香港互聯網交換中心第二站址（HKIX2）啟用。<ref>[http://www.apan.net/meetings/kaohsiung2009/presentations/medical/hk.ppt Hong Kong  Network Updates - Interconnections in HK]</ref>於2016年被[[中信國際電訊|中信國際電訊]]重新收購，成為其轄下專屬數據中心。

== 用戶 ==
直至2015年8月，有225家網際網路服務提供商、網絡機構以及大型網站，例如[[Google香港|Google香港]]、[[Facebook|Facebook]]、[[雅虎|雅虎]]和[[微軟|微軟]]與HKIX連線<ref>[http://www.hkix.net/hkix/connected.htm HKIX - Member List - Licensed Members] {{webarchive|url=https://web.archive.org/web/20100311040045/http://www.hkix.net/hkix/connected.htm |date=2010-03-11 }}</ref><ref>[http://www.hkix.net/hkix/npconnected.htm HKIX - Member List - Non-Licensed Members] {{webarchive|url=https://web.archive.org/web/20100324022712/http://www.hkix.net/hkix/npconnected.htm |date=2010-03-24 }}</ref><ref>[http://www.hkix.net/hkix/FTNSconnected.htm HKIX - Member List - FTNS Providers]</ref>。
