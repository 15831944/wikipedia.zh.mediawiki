'''香港粵語'''（{{lang-en|Hong Kong Cantonese}}）亦稱'''港式粵語'''、'''港式廣東話'''、'''香港話'''，為流通於[[香港社會|香港社會]]的[[粵語|粵語]]，香港人一般慣稱為「廣東話」。香港粵語源於[[廣州|廣州]]，故與[[西關口音|西關口音]][[廣州話|廣州話]]互通，不過與粵語其他分支則有較大分別。隨著各地交流頻密，香港粵語中之部份詞彙被借用至其他粵語使用地區，包括回流廣州，互相影響。香港粵語以廣州話為基礎，加入[[英式英語|英式英語]]，結合香港自身日常文化而成；其特色是加入及融合了不少英式英語及其他[[外來詞|外來詞]]，亦有香港特色的[[口語|口語]]、詞彙、[[俚語|俚語]]、[[俗語|俗語]]及[[髒話|粗口]]，以及各行各業術語等。

{{香港人口及文化}}

根據2001年到2011年[[香港人口普查|香港人口普查]]，廣州話為香港的壓倒性通用語言，89.5%人以此為常用語言，次位為[[英語|英語]]，佔3.4%，使用者多為[[香港華人|香港華人]]以外的[[族群|族群]]；第四位的福建話（閩南話）為1.1%；第五位、第六位及第八位分別為[[客家話|客家話]]、[[潮州話|潮州話]]和[[上海話|上海話]]分别為0.9%、0.7%、0.3%；[[普通話|普通話]]為1.4%，由第六位升至第三位；第七位的[[他加祿語|他加祿語]]和第九位[[印尼語|印尼語]]多為外籍傭工使用。<ref>{{Cite web |url=http://www.census2011.gov.hk/tc/main-table/A111.html |title=存档副本 |access-date=2015-02-26 |archive-url=https://web.archive.org/web/20150923202012/http://www.census2011.gov.hk/tc/main-table/A111.html |archive-date=2015-09-23 |dead-url=yes }}</ref>

== 歷史 == 
{{Main|香港語文|香港早期農村社會|粵語歷史}}
[[香港開埠|香港開埠]]前，[[香港原居民|原居民]]有四大民系，分別為[[廣府民系|廣府]][[圍頭人|圍頭人]]、水上[[蜑家人|蜑家人]]、[[客家人|客家人]]、[[閩南民系|福佬人]]，各族群皆有自己的語言（即[[圍頭話|圍頭話]]、[[蜑家話|蜑家話]]、[[客家話|客家話]]、[[鶴佬話|鶴佬話]]），其中圍頭話和蜑家話亦是粵語分支，香港一詞的英語譯名「Hongkong」就是取自蜑家話發音。

香港成為英國殖民地後，中國大陸其他地方的人如廣州人、四邑人、東莞人、潮州人、閩人陸續入境，各自有不同的語言，如廣州話、四邑話及閩南話等。到[[清朝|清朝]]末年，[[土客械鬥|土客械鬥]]，大批廣州以及江門平民湧進香港，使粵語（如[[四邑話|四邑話]]）人口增加。香港本為小漁港，早期到港的人口大多數為苦力以及建築工人，亦有商人，加上廣州離香港最近而人口最密集，香港的工人商人亦大多為廣府人。不少廣府人在香港以及廣州之間打工，香港市區和省城廣州來往頻繁，廣府人是香港最大的群體，此外亦有不少其他族群，族群之間很少溝通，但部份或以[[官話|官話]]或簡單英語溝通。此情況到第一次世界大戰前已經基本定型。因此在香港市區，廣州話一直是市民的主要語言。

[[第二次世界大戰|第二次世界大戰]]後，[[香港人口|香港人口]]約50萬，當時香港社會上流通多種方言，包括廣州話、客家話（非[[新界原居民|本地客家人]]）、[[閩南話|閩南話]]、[[閩東話|閩東話]]、[[四邑話|四邑話]]、[[潮州話|潮州話]]及[[東莞話|東莞話]]等；在[[新界|新界]]則有[[圍頭話|圍頭話]]及[[客家話|客家話]]，以及[[水上人|水上人]]的[[閩南話|閩南話]]和[[蜑家話|蜑家話]]。[[國共內戰|國共內戰]]後，大量人口湧入香港，此時期移民都是以來自廣東為主，另外亦有不少來自福建和上海等省市，社會增添[[上海話|上海話]]、[[北方官話|北方官話]]使用者，至1950年，香港人口上升至220萬，語言使用情況複雜，各族群都以自家族群的語言交流。之後，隨著大陸[[逃港潮|逃港潮]]，香港人口以每10年100萬之速度大幅增長，當中多為廣東人。為方便溝通，香港在教育、廣播以及政府機構使用粵語廣州話。戰後出生一代絕大多數在粵語環境中成長，粵語成為香港華人慣用語言以及通用語。

[[六七暴動|六七暴動]]後，[[英屬香港#政治|香港政府]]欲統一中文語文。由於當時[[香港市區|港九市區]]較多使用粵語廣州話，而省城[[廣州|廣州]]也是廣州話社會，便於通商。加上戰後至[[香港主權移交|主權移交]]前的中國大陸移民羣內，較多人使用廣州話（佔當時人口四成），其次是潮州話和客家話，故港府遂以廣州話作為香港唯一的中文語言。原本在[[香港電台|香港電台]]的[[潮州話|潮州話]]和[[客家話|客家話]]新聞節目，也在1960年代後期取消<ref>《香港年報1958年》至《香港年報1967年》有關「廣播-香港電台」的章節，當時的香港電台還有潮州話、客家話和[[國語|國語]]節目。</ref>，而電視台的中文頻道也[[清一色|清一色]]以廣州話廣播。在[[新界原居民|新界原居民]]鄉村，原本以圍頭話或客家話教學的[[私塾|私塾]]，也被迫令要使用粵語廣州話教學。自此，香港粵語成為了香港[[官方語言|官方語言]]<ref>{{Cite journal |url= http://www.ln.edu.hk/mcsln/46th_issue/key_concept_01.shtml |title= 粵語的政治 |author= 池偉添 |publisher= [[香港嶺南大學|香港嶺南大學]]文化研究系 |issue=46 |date= 2015-05 }}</ref>。

=== 九聲簡易圖解 ===
香港粵語一般分作「九聲六調」，「九聲」是陰平、陰上、陰去、陰入（高陰入）、中入（低陰入）、陽平、陽上、陽去、陽入。其中陰入、中入、陽入三個聲調，調值高低分別與陰平、陰去、陽去一樣，所以是「六調」。但由於「聲調」含義包括「抑揚」和「頓挫」，我們並不能說香港粵語只有六個「聲調」。

只有「陽聲韻」，即是以鼻音{{IPA|-m}}、{{IPA|-n}}、{{IPA|-ŋ}}做尾音的韻母，才會有相應的「入聲韻」，即以{{IPA|-p̚}}、{{IPA|-t̚}}、{{IPA|-k̚}}作結尾音的韻。

此外，香港粵語有不少變調。例如「老陳」的「陳」、「何伯」的「何」、「字帖」的「帖」、「飛碟」的「碟」、「紅衫魚」的「魚」、「花園」的「園」等、「圖畫」的「畫」、「各位」的「位」等字，經常會變讀成陰上聲的調值。「玻璃」的「璃」、「烏蠅」的「蠅」、「捉伊人」的「人」、「一粒」的「粒」、「勞碌」的「碌」、「親暱」的「暱」、「白雪雪」的「雪雪」、「光脫脫」的「脫脫」，也經常變讀成陰平聲或陰入聲的調值。

=== 變調 ===
書面讀音同口語讀音會有不同。部份香港粵語的口語會將低聲調的字讀成高聲調，叫變調。
* 陽平變陰平。 如「玻璃」的「璃」，「滅蚊」的「蚊」，「豬欄」的「欄」。
* 陽平變陰上。 如「陳伯」的「陳」、「上樓」的「樓」、「還錢」的「錢」。
* 陽上變陰上。 如「書友」的「友」、「[[老母|老母]]」的「母」、「仔女」的「女」。
* 陰去變陰上。 如「魚片」的「片」、「金舖」的「舖」、「影相」的「相」。
* 陽去變陰上。 如「笑話」的「話」、「囚犯」的「犯」、「開會」的「會」。
* 中入變陰上。如「請帖」的「帖」、「麻雀」的「雀」、「填鴨」的「鴨」。用{{IPA|-p̚}}、{{IPA|-t̚}}、{{IPA|-k̚}}收音，口語混合陰上聲同入聲。
* 陽入變陰上。如「蝴蝶」的「蝶」、「書局」的「局」、「長頸鹿」的「鹿」。用{{IPA|-p̚}}、{{IPA|-t̚}}、{{IPA|-k̚}}收音，口語混合陰上聲同入聲。
* 中入變陰入。如「光脫脫」的兩個「脫」字。
* 陰平變陽平。如口語「爸爸」﹑ 「媽媽」中的第一個字。

==借詞==
香港中西文化薈萃，結合各地文化，從外語[[借詞|借詞]]，大量借詞在香港粵語中出現。同時，香港粵語亦影響其他地方[[文化|文化]]，香港的詞彙被借用到[[中国大陆|中国大陆]]、[[臺灣|臺灣]]和[[新加坡|新加坡]]等。[[澳門|澳門]]社會亦長期受到[[香港文化|香港文化]]、潮流用語及網絡傳播的影響，兩地交流密切，故此香港粵語的大部份詞彙、用語、口音等通用於澳門。

此外，來自[[港式英語|港式英語]]之借詞較多，尤其是生活用詞。這些借詞大多是由諧音轉變而成的。由諧音為基準轉化成的詞，英語之外還有很多，其中韓語和日語比較多，例如中文「受歡迎」現今社會會使用日語「高人氣」取代了。這些借詞的出現主因是娛樂媒體的盛行，以及[[韓流|韓流]]文化、日本文化的流入，其次是因為這兩種語言都有漢字的系統，雖然它們的漢字跟中文的漢字意思是不一樣的。這些借詞多源自網絡上，先是以[[諧音|諧音]]（也有人叫[[空耳|空耳]]）為準的網絡用語，之後應用到現實生活中，漸漸就加入了粵語字府了。

===來自中國各方言===
*晒冷、一條冷（「冷」借自[[閩南語|閩南語]]、[[潮州語|潮州語]]，指人）——展示自己的全部[[資產|資產]]，物品或人數等，也可以指炫耀。<ref name="bobi">{{cite web | url=http://www.huayuqiao.org/articles/yuwenjianshetongxun/8105 | title=香港粵語的貶義借詞 | accessdate=2011-09-22 | author=劉鎮發}}</ref>
*啪針、啪丸——（「啪」借自[[潮州語|潮州語]]、[[閩南語|閩南語]]的「拍」）[[注射|注射]][[毒品|毒品]]、進丸狀的毒品。<ref name="bobi" />
*架己冷（借自[[潮州語|潮州語]]）──自家人，同常是[[潮州人|潮州人]]向[[同鄉|同鄉]]的一種表達方法。
*[[打冷|打冷]]（借自[[潮州菜|潮州菜]]）——'''[[打冷|打冷]]'''一詞出自香港市民表示去食潮州菜意思。
*搭爹（「爹」借自[[閩南語|閩南語]]的「茶」）——從形容人話多的「口水多過茶」演變而成，意同「搭訕」。
*劈友（「劈」借自[[四邑話|四邑話]]的「打」）——是「用刀砍人」的意思。

== 慣用語 ==
===源出[[香港歷史|香港歷史]]因素===
*去[[差館|差館]]——去[[警署|警署]]。
*俾[[差人|差人]]拉——被[[警察|警察]]帶走（引申為[[拘捕|拘捕]]）<ref>[http://big5.huaxia.com/zhwh/whrd/whrdwz/2010/07/1993916.html 大城市繞不開“說話”關:方言普通話外語如何共生《華夏經緯網 》2010年07月20日]</ref>。
*[[摩羅差|摩羅差]]——本指[[香港差役|香港差役]]中的[[印度|印度]][[差人|差人]]，廣義上指印巴籍人士。
*咕哩──[[苦力|苦力]]。

===新潮口語===
*[[擢樣|擢樣]]──耍出使人窒息的面部容貌。
*'''淆底'''——一般是指該人沒有膽色、害怕、退縮、驚惶的意思。

===行業術語===
*[[火記|火記]]──原為[[香港警察|香港警察]]術語，意思是出生入死的軍旅同火（亦即[[伙伴|伙伴]]之意），是故多誤寫為[[伙記|伙記]]。餐廳[[侍應|侍應]]，亦被慣常稱為夥計，與伙記同音。
*飛砂走奶：[[茶餐廳|茶餐廳]]用語，咖啡或奶茶飛砂走奶指飛走（不加）[[砂糖|砂糖]]及[[淡奶|淡奶]]，取而代之加[[煉奶|煉奶]]。
*齋啡: 茶餐廳用語，指純黑咖啡。
*茶走：[[奶茶|奶茶]]不加[[糖|糖]]，改加[[煉奶|煉奶]]。
*走青: 茶餐廳用語，一般指[[粥|粥]]或[[麵|麵]]不加[[蔥花|蔥花]]。
*行街：茶餐廳用語，指外帶／外賣。

== 中英混杂 ==
{{main|中英夾雜}}
香港人講話時普遍習慣夾雜英語單字，主要是名词及连接词，例如：「hard disk」、「email」及「notebook」等等<ref>[http://news.sina.com.hk/cgi-bin/nw/show.cgi/2/1/1/1267210/1.html 名人教英文﹕中英夾雜]，《明報》2009年9月17日</ref><ref>[http://paper.wenweipo.com/2011/11/09/ED1111090012.htm 流行文化錄：我們一起穀、穀、穀！]，《文匯報》2011年11月9日</ref>，部份學者及專欄作家認為是香港的文化特色<ref>[http://edu.singtao.com/lib/earthquake/article_detail.asp?id=476&issid=%20 中英混雜《星島日報》2007年4月17日]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 語音發展 ==
=== 懒音问题 ===
{{Main|粵語懶音}}
現在，香港不少人的粵語發音出現了稱為「[[懶音|懶音]]」的現象，並被部分香港本土精英分子視之為問題。{{fact|長者因為舌頭機能問題，會出現懶音，基於[[人口老化|人口老化]]，這部份的人口正漸漸增加。另外，香港現今年輕一輩，主要因為外地語言的傳入以及對非本地文化的認同，對粵語的重視不如上一輩，所以懶音問題尤其嚴重。}}這種現象的較常見例子是「廣」「港」、「角」「國」、「男」「藍」不分等，這些字在標準粵語發音中雖然近似但並不相同而可作區分，但在使用懶音的情況下相關區分會被丟失，因此聆聽者將不能區分相關單字的讀音。以下列出部分不少香港人在日常生活中會出現的「懶音」口音現象：
* 不區分{{IPA|/n/}}聲母與{{IPA|/l/}}聲母，將鼻音{{IPA|/n/}}唸作舌邊音{{IPA|/l/}}。
* 不區分{{IPA|/ŋ/}}聲母與零聲母，將軟腭音{{IPA|/ŋ/}}唸作零聲母。有部份人則反過來，將零聲母唸作{{IPA|/ŋ/}}聲母。
* 不區分{{IPA|/k/}}聲母與{{IPA|/kw/}}聲母，將圓脣音{{IPA|/kw/}}唸作{{IPA|/k/}}。同時也不區分{{IPA|/kʰ/}}聲母與{{IPA|/kʰw/}}聲母，將圓脣音{{IPA|/kʰw/}}唸作{{IPA|/kʰ/}}。
* 不區分{{IPA|/n/}}韻尾與{{IPA|/ŋ/}}韻尾，將{{IPA|/ŋ/}}韻尾唸作{{IPA|/n/}}韻尾。
* 不區分{{IPA|/t̚/}}韻尾與{{IPA|/k̚/}}韻尾，將{{IPA|/k̚/}}韻尾唸作{{IPA|/t̚/}}韻尾。

在廣州，此情況比較香港、九龍半島、新界輕微，比如{{IPA|/n/}}和{{IPA|/l/}}聲母，某些地區的粵語依然清晰且嚴格地區分{{IPA|/n/}}和{{IPA|/l/}}聲母。

==何文匯正音運動==
{{main|粵語正音運動}}

1970年代起，受到由[[何文匯|何文匯]]提倡及領導的[[粵語正音運動|粵語正音運動]]所影響，香港粵語部份詞彙的發音偏離了粵語廣州口音，與[[珠江三角洲|珠江三角洲]]粵語系地區的發音脫節。何文匯「正音」是根據距今一千年的[[宋朝|宋朝]]《[[廣韻|廣韻]]》一書中的切韻方法，將很多與《廣韻》所載不同的粵音重新訂立標準，因此被不少香港教師認為是一種矯枉過正的學說。現時，支持和反對此運動的兩大陣營尚在對壘。在學術界的朗誦及教學方面，粵語正音運動的支持者不少；大眾傳播媒介包括[[無綫電視|無綫電視]]及[[亞洲電視|亞洲電視]]亦支持之；然而在香港市民的日常生活中，其學說並未獲得很大的支持。

== 評價 ==
曾经担任香港大学客座教授的台湾学者[[龙应台|龙应台]]（後來為[[中華民國文化部|中華民國文化部]]長）说过，香港人有一怪现象，与中国大陆人、台湾人对话，喜欢劈头一句便说：“对不起，我的普通话不好”，来了香港两年仍然未有学好粤语的龙应台笑说：“该是我道歉才对！”她質疑，別人來到香港，不懂說粵語，為什麼香港人要反過來賠不是？龍應台說：「語言是平等的。粵語文化，有其深刻動人之處，是香港最珍貴的寶藏，不應被取代，更非英語所能涵蓋。粵語是香港人的資產，不是負資產，香港人應加以珍惜。」<ref>[http://paper.wenweipo.com/2005/07/27/RW0507270001.htm 龍應台 如果我是香港人]，香港文汇报 2005-07-27。</ref>
