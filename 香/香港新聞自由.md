{{不是|香港自由新聞}}
'''香港新聞自由'''受到《[[中華人民共和國香港特別行政區基本法|中華人民共和國香港特別行政區基本法]]》規範。《基本法》第二十七條規定：「香港居民享有言論、'''新聞'''、出版的自由，結社、集會、遊行、示威的自由，組織和參加工會、罷工的權利和自由。」<ref>[http://www.basiclaw.gov.hk/tc/basiclawtext/chapter_3.html 基本法與你]</ref>。[[香港特區政府|香港特區政府]]宣稱其一直致力維護香港的[[新聞自由|新聞自由]]<ref>[http://www.news.gov.hk/tc/record/html/2015/04/20150427_135314.shtml 致力維護新聞自由]</ref>。

有評論認為[[香港回歸|香港回歸]]前[[港英政府|港英]]打壓新聞自由，回歸後《基本法》保障了新聞自由，但部份[[新聞媒體|新聞媒體]]卻濫用新聞自由，惡意抹黑、造假、炒作<ref>[http://www.storm.mg/article-page/38235/6 反滅聲vs反假聲 香港新聞自由大辯論]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2012年，[[香港大學民意研究計劃|香港大學民意研究計劃]]的民意調查結果顯示，59%受訪者認香港新聞自由被濫用<ref>[http://paper.wenweipo.com/2012/10/25/YO1210250017.htm 59%市民指新聞自由遭濫用]</ref>。2016年3月，[[香港記者協會|香港記者協會]]發表的2015年「香港新聞自由指數」的評分都比2014年及2013年下跌<ref>{{cite web | title = 蘇鑰機：香港新聞自由指數：下跌和原因 | url = http://news.mingpao.com/pns/dailynews/web_tc/article/20160331/s00012/1459360230120 |website = 明報新聞網 | date = 2016-03-31 }}</ref><ref>[http://news.mingpao.com/pns/%E8%98%87%E9%91%B0%E6%A9%9F%EF%B9%95%E9%A6%99%E6%B8%AF%E6%96%B0%E8%81%9E%E8%87%AA%E7%94%B1%E6%8C%87%E6%95%B8%E4%B8%8B%E8%B7%8C/web_tc/article/20150402/s00012/1427912934704 蘇鑰機﹕香港新聞自由指數下跌]</ref>。

2018年4月，[[無國界記者|無國界記者]]發表該年度「[[新聞自由指數|新聞自由指數]]」，香港在180個國家或地區排名中名列70名，遠遠高於排在第176位（倒數第5）的中國大陸，但低於台灣的第42位。<ref name="眾新聞0425">{{Cite web |url=https://www.hkcnews.com/article/11687/%E7%84%A1%E5%9C%8B%E7%95%8C%E8%A8%98%E8%80%85-%E6%96%B0%E8%81%9E%E8%87%AA%E7%94%B1%E6%8E%92%E5%90%8D-11687/%E7%84%A1%E5%9C%8B%E7%95%8C%E8%A8%98%E8%80%85 |title=無國界記者新聞自由排名　台灣42香港70　批中國「愈來愈接近當代版極權主義」 |date=2018-04-25 |publisher=眾新聞}}</ref><ref>{{Cite web |url=https://www.facebook.com/HKJA.official/posts/10156151953505309 |title=無國界記者組織2018年世界新聞自由指數報告, 對亞太區情況的分析 |date=2018-04-24 |publisher=香港記者協會 Hong Kong Journalists Association facebook}}</ref>報告認為這年度香港排名提升的原因包括記者遭暴力對待的事件幾乎絕跡、多家網上媒體冒起而且爭取多年獲港府認可，以及網媒有組織抵抗北京的干預<ref>{{Cite web |url=https://news.mingpao.com/pns/dailynews/web_tc/article/20180426/s00002/1524679338318 |title=報告﹕獨立網媒助抗北京干擾 |date=2018-04-26 |publisher=明報}}</ref><ref>{{Cite web |url=http://news.rthk.hk/rthk/ch/component/k2/1393116-20180425.htm |title=世界新聞自由指數排名　香港升3位排70位 |date=2018-04-25 |publisher=香港電台}}</ref>

2019年無國界記者「新聞自由指數」裏，香港在180個國家或地區排名中名列73名。報告形容香港主流傳媒「早已迅速遵守北京的命令」，其中香港外國記者會副主席[[馬凱_(記者)|馬凱]]因舉辦「北京不喜歡的活動」而被拒續發護照及《逃犯條例》修訂都受關注，亦對媒體工作造成很大威脅。<ref name="明報0419">{{Cite news |url=http://www.mingpaocanada.com/tor/htm/News/20190419/HK-gha1.htm?m=0 |title=新聞自由排名 港跌至73 主流傳媒被指「早迅速遵守北京命令」 |date=2019-04-19 |work=明報 |accessdate=2019-04-19}}</ref>

==事件==
[[File:1937年香港新聞檢查.jpg|thumb]]
1858年，署理輔政司[[布烈治|布烈治]]以誹謗罪控告《[[華友西報|華友西報]]》東主德倫（W. Jarrant），開創了香港政府控告媒體的先例。

1907年，香港政府禁止中文報刊刊登煽動反清新聞。<ref name=" 香港歷史系列">[http://hkhistory.etvonline.hk/2015/ep4.php 香港報業第四集]</ref>

1925年，因應[[省港大罷工|省港大罷工]]，香港政府成立新聞檢查處，審查中文報紙內容。<ref name=" 香港歷史系列"/>

1929年至1930年，共產黨報《香港小日報》被兩度查封，最後停刊。<ref name="楊國雄2013">{{cite book|author=楊國雄|title=香港戰前報業|url=http://books.google.com/books?id=2V9wBgAAQBAJ&pg=PA210|date=1 October 2013|publisher=三聯書店(香港)有限公司|isbn=978-962-04-3449-5|pages=210–}}</ref>

1937年[[中國抗日戰爭|中國抗日戰爭]]爆發後，香港政府禁止報刊刊登反日新聞，「日寇」、「汪逆」、「敵軍」、「漢奸」等被視為敏感字。<ref name=" 香港歷史系列"/>

二戰後，香港放寬新聞管制。<ref name=" 香港歷史系列"/>

1967年，商台播音員[[林彬|林彬]]被燒死。《香港夜報》及《新午報》社長[[胡棣周|胡棣周]]、《田豐日報》社長[[潘懷偉|潘懷偉]]被捕。《香港夜報》、《田豐日報》和《新午報》等報館被查封。<ref>[http://www.nshk.org.hk/html/d_50cel/c50_0401.htm 香港報業大事記(1954年~2004年) ] {{webarchive|url=https://web.archive.org/web/20160304120835/http://www.nshk.org.hk/html/d_50cel/c50_0401.htm |date=2016-03-04 }}</ref>

2004年，[[香港廉政公署|香港廉政公署]]搜查《東方日報》、《蘋果日報》、《太陽報》、《星島日報》、《南華早報》、《信報》及《大公報》7大報館，引起社會關注。其後法官[[夏正民|夏正民]]為保障新聞自由，撤銷廉政公署的搜查令。<ref>[http://hk.apple.nextmedia.com/news/art/20040811/4231571 官判撤銷搜查令搜報館廉署犯錯]</ref>

2014年，《明報》總編輯[[劉進圖|劉進圖]]被撤換，其後被刀手襲擊。商台時事節目主持人[[李慧玲|李慧玲]]被解僱。<ref>[http://www.cna.com.tw/news/firstnews/201402130038-1.aspx 香港商台解僱名嘴 新聞自由掀波]</ref>

[[File:香港七大傳媒團體抗議港大禁制令損害新聞自由.jpg|thumb]]
2015年[[香港大學副校長任命事件|香港大學副校長任命事件]]中，[[香港大學|香港大學]]申請禁制令，阻止[[商台|商台]]公開港大校委會會議內容，引起香港記者協會、香港攝影記者協會、香港電台節目製作人員工會、壹傳媒工會 、明報職工協會、獨立評論人協會及大專新聞教育工作者聯席抗議。<ref>[http://www.dbc.hk/radio2/news-detail/Id/102701/type/10/%E3%80%90%E6%B8%AF%E5%A4%A7%E9%A2%A8%E6%B3%A2%E3%80%91%E8%A8%98%E5%8D%94%E7%AD%89%E5%A4%9A%E5%80%8B%E5%9C%98%E9%AB%94%E6%8A%97%E8%AD%B0%E6%B8%AF%E5%A4%A7%E7%94%B3%E8%AB%8B%E7%A6%81%E5%88%B6%E4%BB%A4 記協等多個團體抗議港大申請禁制令]</ref>

2018年發生了香港外國記者會副主席[[馬凱_(記者)|馬凱]]工作簽證到期而不獲續期事件，記協調查顯示分別有69%和97%公眾和傳媒人認為做法對新聞自由有損害。同年度，市民對香港新聞自由評分只有45分，是自2013年以來新低，他們普遍認為傳媒在批評內地和香港政府時有所顧忌，其次是中央政府和傳媒老闆向員工施壓。<ref>{{Cite news |url=https://hk.news.appledaily.com/local/daily/article/20190417/20657913 |title=七成公眾指馬凱事件重創傳媒 新聞自由指數創新低 |accessdate=2019-04-17 |date=2019-04-17 |work=蘋果日報}}</ref>
