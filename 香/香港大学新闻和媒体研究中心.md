'''香港大学新闻和媒体研究中心'''（{{lang-en|'''Journalism and Media Studies Centre'''}}；英文缩写'''JMSC'''）1999年9月在[[香港大学|香港大学]]建立，为香港大学社会科学学院的一部分，教育项目包括[[研究生|研究生]]、[[本科生|本科生]]课程、研讨会、培训班等。

== 学术项目 ==
*报告和写作计划
*媒体法律程序
*中国项目
*商业和金融新闻节目

== 教员 ==
*陈婉莹，教授兼主任。

== 中国传媒研究计划 ==
中国传媒研究计划是香港大学在2003年建立的。
