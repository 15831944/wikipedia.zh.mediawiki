== 加入資格 ==
香港手語翻譯員只需符合以下條件，便可申請加入這名單<ref name="手語不統一政府不支援">{{cite web |language=zh-hk | url=http://hk.on.cc/hk/bkn/cnt/news/20160318/bkn-20160318103553931-0318_00822_001.html | title= 手語不統一政府不支援　翻譯認可路難走 |publisher=[[東網|東網]] | date=2016-03-18 | accessdate=2018-09-22}}</ref>：
* 在申請當天起計，過去兩年內合共提供超過200小時的手語翻譯服務
* 持有由受僱機構簽發的證明文件

== 香港現況 ==
香港現時手語翻譯員不足，這名單大約有十多人是全職工作。手語訓練需時，惟很多人視為義務工作，令翻譯員未能取得相應的待遇<ref>{{cite web |language=zh-hk | url=http://www.mingpaocanada.com/tor/htm/News/20171022/HK-gga2_r.htm | title= 傳譯員：訓練費時 待遇不高 |publisher=[[明報|明報]] | date=2017-10-22 | accessdate=2018-09-22}}</ref>。而翻譯服務及手語課程遠落後歐美國家，令聾人與健聽人士在生活及溝通上出現問題和困擾<ref>{{cite web |language=zh-hk | url=http://hk.on.cc/hk/bkn/cnt/news/20170327/bkn-20170327132553528-0327_00822_001.html | title= 團體推手譯寶應用程式　助聾人打破隔膜 |publisher=[[東網|東網]] | date=2017-03-27 | accessdate=2018-09-22}}</ref>。聾人機構「[[龍耳|龍耳]]」希望政府增撥資源，培訓更多手語翻譯員及發展視像翻譯服務<ref name="手語不統一政府不支援" />。
