== 歷史背景 ==
[[File:Acquisition_of_Hong_Kong_ZH.svg|thumb]]
<!--[[File:Hkcolonyprocess.jpg|thumb]]-->
香港在殖民統治前，前身是[[大清|大清]][[廣東省_(清)|廣東省]][[廣州府|廣州府]][[寶安縣|新安縣]]。英國外務大臣[[第三代巴麥尊子爵亨利·坦普爾|彭瑪斯頓勳爵]]認為英僑繳出[[鴉片|鴉片]]，無異被迫納贖命金，因此要求中國與英國訂立貿易條約，以平等地位通商，或割讓一個小島給英國，使英僑能在其國家蔭庇下，安居樂業<ref name="元">{{cite book |url=http://www.yearbook.gov.hk/1997/cindex.htm |title=《香港——邁進新紀元（1997年年報）》 |location=香港 |publisher=[[政府新聞處|政府新聞處]] |year=1998 }}</ref>{{rp|319}}。為爭取上述要求，英國顯示實力，派遣遠征軍於1840年6月（清[[道光|道光]]二十年）開抵中國，遂爆發[[第一次鴉片戰爭|第一次鴉片戰爭]]（1840至1842年）<ref name=元/>{{rp|319}}。[[林則徐|林則徐]]在雙方初步談判失敗後遭撤職，由琦善接任[[欽差大臣|欽差大臣]]<ref name=元/>{{rp|319}}。雙方且談且戰，直至1841年1月20日[[義律|義律]]與[[琦善|琦善]]達成協議《[[穿鼻草約|穿鼻草約]]》，琦善割讓香港給英國；[[英國海軍|英國海軍]]於1841年1月26日在[[水坑口街|水坑口]]登陸升旗，[[香港島|香港]]正式成為[[英國屬土|英國屬土]]<ref name=元/>{{rp|319}}。是年6月，義律分段出售土地，香港開始有移民居留<ref name=元/>{{rp|319}}。兩國政府均不承認《穿鼻草約》：[[道光帝|道光帝]]震怒，認為割地是奇恥大辱，清廷遂撤琦善職，解京查辦<ref>[[s:清史紀事本末/卷四十四|s:清史紀事本末/卷四十四]]：草約成，以聞，帝不許，至是再下宣戰之諭，命奕山爲將軍，楊芳、隆文爲參贊大臣，馳赴廣東，逮琦善入京</ref>；英國方面，[[第三代巴麥尊子爵亨利·坦普爾|彭瑪斯頓勳爵]]也大表不滿，認為香港是「一個杳無人煙的荒蕪小島」，不足以代替通商條約<ref name=元/>{{rp|319}}。[[第三代巴麥尊子爵亨利·坦普爾|彭瑪斯頓勳爵]]嚴厲譴責義律漠視他的訓令，召回義律，另派[[砵甸乍|砵甸乍]]爵士接任；1841年8月，砵甸乍爵士抵達香港，決心以武力解決問題，一年後揮軍溯長江而上南京，威脅[[京杭大運河|京杭大運河]]<ref>[http://home.chuhai.hk/~cwchau/conthistory/0203.pdf 鴉片戰爭]. 珠海書院. </ref>，雙方終於在1842年8月29日（清道光二十二年七月二十四日）訂立《[[南京條約|南京條約]]》，戰事始告平息<ref name=元/>{{rp|319-320}}。其時[[英國維新黨|英國維新黨]]政府下台，新上任的[[保守黨|保守黨]]外務大臣阿巴甸勳爵於1841年修改給砵甸乍爵士發出新訓令，撤回割地要求；砵甸乍爵士於冬天停戰時返抵香港，對香港發展甚為滿意，在訂立《南京條約》時違背英廷訓令，不僅訂立通商條約，還要求割讓香港<ref name=元/>{{rp|320}}。清政府還必須開放廣州等5個地點為通商口岸，該通商條約其後併入1843年10月締訂《虎門善後條約》，容許華人自由到香港經商<ref name=元/>{{rp|320}}。

中英兩國對前訂條約詮釋各執一辭，大清水師登上英艇「亞羅」號搜捕海盜，因而發生衝突，引發[[英法聯軍之役|英法聯軍之役]]（1856至1858年），結果兩國於1858年簽訂《天津條約》，英國得派遣外交代表到中國，戰事暫告平息<ref name=元/>{{rp|320}}。第一任使節為香港首任[[布政司|布政司]][[卜魯斯|布魯司]]爵士，他奉命前往北京呈遞國書，但途經[[大沽|大沽]]突遭炮轟，於是1859年至1860年間，戰火再起；英軍擬保留[[九龍半島|九龍半島]]駐防，經英國駐廣州領事[[白加士|白加士]]爵士與[[兩廣總督|兩廣總督]]交涉後，獲得九龍半島南端（北至[[界限街|界限街]]）連同[[昂船洲|昂船洲]]之租借權<ref name=元/>{{rp|320}}。1860年，戰爭結束，訂立《[[北京條約|北京條約]]》，九龍半島正式割讓予英國<ref name=元/>{{rp|320}}。

1895年[[甲午戰爭|甲午戰爭]]結束後，曾為中國說項的[[德國|德國]]、[[法國|法國]]、[[俄國|俄國]]更有藉口提出租地要求；其後局勢緊張，英國認為如要防衞香港，必須取得鄰近土地的控制權<ref name=元/>{{rp|320}}。根據1898年6月9日在北京簽訂《[[展拓香港界址專條|展拓香港界址專條]]》，大清同意租借九龍界限街以北直至[[深圳河|深圳河]]的[[新界|新界]]地域，以及235個島嶼予英國，為期99年；英國此舉僅在針對法俄兩國，而非對付中國<ref name=元/>{{rp|320}}。大清保留[[九龍寨城|九龍寨城]]之行政權力，仍有權使用[[九龍城碼頭|九龍城碼頭]]，「惟不得與保衛香港之武備有所妨礙」；1899年12月27日，[[英國君主|英女皇]]會同[[樞密院|樞密院]]議決撤銷這項條款，由英國單方面接管九龍城<ref name=元/>{{rp|320}}。在此之前，英國人已於1899年4月接管[[新界|新界]]，接管初期尚有零星反抗，但後來終告停止，遂成為香港一部分，但與市區分開管理<ref name=元/>{{rp|320}}。

===[[第二次世界大戰|第二次世界大戰]]===

1941年12月8日，[[日本帝國|日本]][[偷襲珍珠港|偷襲珍珠港]]，[[太平洋戰爭|太平洋戰爭]]爆發，當天[[日軍|日軍]]空襲‍[[啟德機場|啟德機場]]，日軍自深圳入侵香港，香港保衛戰開始；日軍攻破醉酒灣防線，[[醉酒灣防線|英軍被逼撤離新界和九龍]]，退守港島，與[[香港義勇軍|香港義勇軍]]苦戰一周後，以眾寡懸殊，難扭劣勢，終在[[聖誕日|聖誕日]]投降<ref name=元/>{{rp|322}}。
[[香港日治時期|日本佔領香港]]共3年8個月<ref name=元/>{{rp|322}}。1942年10月28日，英國駐華大使{{link-en|薛穆|Horace James Seymour}}奉命與中華民國[[重慶國民政府|重慶國民政府]]外交部開始簽訂新約舉行談判<ref name="錄">{{cite book |author=謝永光 |title=《香港戰後風雲錄》 |location=香港 |publisher=[[明報|明報]]出版社 |year=1996 }}</ref>{{rp|2}}。由於英國並沒有完全放棄在華全部特權之誠意，談判相當艱難<ref name=錄/>{{rp|2}}。在談判過程中出現香港問題，英國不肯歸還香港（雖然當時香港還在日本佔領下），這項談判因而拖延<ref name=錄/>{{rp|2-3}}。當中英談判時，英國戰時內閣首相[[邱吉爾|邱吉爾]]和[[安東尼·艾登|艾登]]認為，根據1842年《[[南京條約|南京條約]]》，香港已永久割讓給英國，根據1860年《[[北京條約|北京條約]]》，北至界限街之九龍半島南端連同昂船洲也已永久割讓給英國<ref name=錄/>{{rp|3-4}}。原擬定在1942年公布之中英、中美平等新約，結果延至1943年公布，英、美取消在中國之治外法權及租界，但是不提香港地位<ref name=錄/>{{rp|3}}。中國迫於無奈，惟有不再堅持九龍租借地交還問題<ref>{{cite web |url=http://ir.ndhu.edu.tw/bitstream/987654321/4427/1/7-123-150.PDF |title=蔣介石對中英新約的態度（1942-1943） }}</ref>。

1943年1月11日，重慶[[國民政府外交部|國民政府外交部]]長[[宋子文|宋子文]]與英國駐華大使薛穆分別代表本國政府在重慶簽署《{{link-en|中英關於取消英國在華治外法權及其有關特權條約|Sino-British New Equal Treaty}}》，史稱《中英新約》<ref name=錄/>{{rp|3}}。重慶國民政府以外交部長宋子文之名義向英國駐華大使薛穆發出正式照會，對新界「租借地」問題保留重提權利<ref>{{cite web |url=http://www.mac.gov.tw/ct.asp?xItem=45567&CtNode=6076&mp=1 |title=中華民國政府對「九七」香港情勢的立場與政策說帖 }}</ref>：「關於交還九龍租借地問題，英國政府不以現時進行談判為宜，本代表認為憾事！一八九八年六月九日許予英國租借九龍條約之早日終止，實為中國國民所企望。而本日簽訂條約之意義，為開兩國邦交之新紀元，中國政府以為若該約能於此時終止，則新紀元之精神更顥著，因此之故，本代表通知閣下，中國政府保留日後重行提請討論此問題之權。」<ref name=錄/>{{rp|4}}11月23日，中、英、美三國首在開羅舉行會議，[[美國總統|美國總統]][[富蘭克林·德拉諾·羅斯福|羅斯福]]問蔣介石對香港如何打算，邱吉爾請大家注意香港是英國領土；蔣說英國以暴力入侵中國，與清廷所訂的不平等條約，國民政府概不予承認，戰後隨時可以收回香港<ref name=錄/>{{rp|6-7}}。羅斯福亦曾認為英國政府應該放棄統治香港，並歸還給中國發展為國際自由港<ref name=sbr-in-20c/>。

1945年8月15日，[[日本投降|日本無條件投降]]，二戰結束。引起了香港的歸屬問題：中國能否收回香港<ref name="sbr-in-20c">{{cite book |language=zh-cn |title=《20世紀的中英關係》 |author=薩本仁，潘興明 |publisher=上海人民出版社 |year=1996 |isbn=9787208021952 |pages=243,278,282頁 |accessdate=2009-02-23 }}</ref><ref name="hk-20c">{{cite book |title=《20世紀的香港》 |author=余繩武，劉蜀永 |publisher=[[中國大百科全書出版社|中國大百科全書出版社]] |year=1995 |isbn=9787500056782 |accessdate=2009-02-23 }}</ref><ref name="hkhistory">{{cite book |language=zh-cn |title=《香港史話》 |author=劉蜀永 |publisher=社會科學文獻出版社 |year=2000 |accessdate=2009-02-23 }}</ref>。中國政府亦希望能收回香港，但因[[國共內戰|國共內戰]]日漸激烈而無暇南顧<ref name=sbr-in-20c/>。另一方面，英國則強烈希望戰後保持所有遠東殖民地，包括作為英國遠東海軍基地及商業中心的香港<ref>{{cite book |language=zh-hant |title=《日佔時期的香港》 |author=關禮雄 |publisher=三聯書店（香港） |year=1993 |isbn=9789620411052 |accessdate=2009-02-24 }}</ref>。在英國的強硬態度及戰勝國之間的利益關係下，香港的歸屬問題並沒有在戰後的國際會議上提出過<ref>{{cite book |author=[[高添強|高添強]] |title=《香港日佔時期》 |publisher=三聯書店（香港） |isbn=9620412540 }}</ref>。

== 香港前途問題緣起 ==
1949年[[中华人民共和国|中华人民共和国]]成立後，基於國際考慮，中共雖然不承認英方對香港的殖民「主權」，但是1950年向英國表明短期暫時不會「收回」香港<ref>{{cite book |language=zh-cn |title=《當代中國與它的外部世界: 第一屆當代中國史國際高級論壇論文集》 |author=朱佳木 |publisher=當代中國出版社 |year=2006 |isbn=9787801704702 }}</ref>。1950年1月6日，英國向中国遞交照會，承認[[中华人民共和国|中华人民共和国]]政府為中国之合法政府<ref>{{cite web |url=http://paper.people.com.cn/rmrbhwb/html/2014-12/13/content_1509682.htm |title=新中国建交内情（十二）（礼仪漫谈）——与英国建交（上） }}</ref>。同時英國在軍事上也做出部署，包括使用核武打擊和阻止中国大陸武力收復香港的可能<ref>{{cite web |url=http://news.bbc.co.uk/2/hi/uk_politics/5130524.stm |title=UK pondered China nuclear attack |publisher=[[BBC|BBC]] |date=2006-06-30 }}</ref>。1972年3月13日，中華人民共和國和英國發表聯合公報，英國承認[[中华人民共和国政府|中华人民共和国政府]]是中國的唯一合法政府<ref>{{cite web |url=http://www.people.com.cn/GB/historic/0313/773.html |title=1972年3月13日 我国与英国外交关系升为大使级 }}</ref>。

1952年10月，鑒於世界範圍內的「[[去殖民化|去殖民化]]」同「[[民主化|民主化]]」進程，[[葛量洪|葛量洪]]與[[殖民地部|殖民地大臣]]分別在香港和[[英國國會|英國國會]]同時宣布，香港在這個時期推行大規模政制改革，不合時宜；「但是我們己決定在香港的[[市政局_(香港)|市政局]]進行改革，市政局內將會有兩位民選的議席。」<ref name=錄/>{{rp|150}}[[英國國家檔案館|英國國家檔案館]]公開一些文件顯示，從[[1950年代|1950年代]]開始，[[香港總督|香港總督]]尋求推行民眾選舉，但迫於[[中共领导人|中共领导人]]壓力等多方因素，最終放棄：檔案顯示，主管香港事務的中国大陸高官廖承志曾在1960年表示，「我們將毫不猶豫採取積極行動，解放香港、九龍和新界。」；另一份文件記述早前兩年一次會議上，中國大陸國務院總理[[周恩来|周恩来]]告訴英國一名軍官，任何在香港引入哪怕一點點自治的嘗試，都會被視作“非常不友善的舉動”和“陰謀”，表示這種做法會被認為是讓香港走上獨立道路的手段<ref>{{cite web |url=http://cn.nytimes.com/china/20141028/c28hongkong/zh-hant |title=北京曾多次反對港英政府引入選舉 |publisher=紐約時報中文網國際縱覽 |date=2014-10-28 }}</ref>。人類重視本身利益，在政治立場上態度反覆多變，他們作出所有政策決定，都會回頭看看以前政府所作出之決定<ref name=錄/>{{rp|150-151}}。

1971年[[聯合國大會2758號決議|聯合國大會2758號決議]]，中华人民共和国取代中華民國的[[聯合國|聯合國]]「中國」合法席位和[[聯合國安全理事會|聯合國安全理事會]]常任理事國後，就開始為香港與澳門的主權問題採取外交行動。1972年3月8日，通過駐聯合國代表[[黄华_(政治人物)|黃华]]致信[[聯合國非殖民化特別委員會|聯合國非殖民化特別委員會]]主席，聲明：「香港、澳門屬於歷史遺留下來的[[帝國主義|帝國主義]]強加於中國的一系列不平等條約的結果，香港和澳門是被英國和葡萄牙當局佔領的中國領土的一部分，解決香港、澳門問題完全是屬於中國主權範圍內的問題，根本不屬於通常的所謂殖民地範疇。因此，不應列入殖民宣言中適用的殖民地區的名單之內。我國政府主張，在條件成熟時，用適當的方式和平解決港澳問題，在未解決之前維持現狀」<ref>{{cite web |url=http://www.chinanews.com/zhonghuawenzhai/200001/new/1_copy(37).html |title=中國三代領導人決策澳門回歸 |publisher=中國新聞網 |date=2000-02-21 |accessdate=2009-02-24 }}</ref>。3月10日，黃華又在[[聯合國大會|聯合國大會]]上再度重申中國大陸當局的這一主張；11月8日，第27屆聯合國大會投票通過非殖民化特別委員會關於同意中華人民共和國意見的決議，規定香港澳門不屬於殖民地範疇，港澳問題屬於中國主權問題，聯合國和他國無權干涉，應由中華人民共和國和英國、葡萄牙交涉談判解決香港澳門問題<ref name="波">{{cite web |url=http://view.news.qq.com/zt2012/am/index.htm |title=1975年“澳门归还风波” |accessdate=2017-07-02 }}</ref>。聯合國大會通過決議，確認中華人民共和國對香港立場和要求，排除其他國家（包括聯合國）參與解決香港問題（如國際共管）之可能<ref>{{cite web |url=http://www.ipm.edu.mo/cntfiles/upload/docs/research/common/1country_2systems/2010_5/p76.pdf |title=香港問題的提出與“一國兩制”的定型 |accessdate=2017-07-02 }}</ref>。香港和澳門往後無法以殖民地身份獨立，也未能透過聯合國非殖化的努力改革社會制度並及早推進民主，此舉為中華人民共和國接收香港與澳門的主權製造了背景條件<ref name="law1">{{cite book |language=zh-cn |title=中國內地、香港法律制度研究與比較 |author=香港法律教育信託基金 |publisher=北京大學出版社 |year=2000 |isbn=9787301046302 }}</ref>。中國大陸在1972年公開否定掉香港、澳門殖民地性質，但中國大陸人民瞭解詳情者不多，所以，1997年3月17日，香港回歸前夕，《人民日報》曾特意刊登文章《為什麼說香港不是殖民地》，對此做出解釋：「迄今為止，英國在香港實行的是典型的殖民式統治，但並不等於香港就是殖民地。因為，通常意義上的殖民地主要是指因外國統治、管轄而喪失了主權的國家。香港是中國領土的一部分，所以，殖民地概念不適用於香港。」<ref name=波/>1960年代至1970年代中國大陸國內政局不穩定，以致延遲到1980年代才開始和英國談判歸還香港問題<ref name="後">{{cite book |author=黎秀石 |title=《日本投降的前前後後》 |location=香港 |publisher=明報出版社 |year=1995 }}</ref>{{rp|150}}。

大部份香港市民在中英開始詳細談判前，鮮有考慮香港主權的前途問題。新界的《[[展拓香港界址專條|展拓香港界址專條]]》租約將在少於20年屆滿<ref>{{cite web |url=http://marketing2.mingpao.com/htm/foreign_affairs/content.cfm?Path=p6_28.htm |title=中英展拓香港界址專條 |publisher=明報 |accessdate=2017-07-01 }}</ref>。新界租借期於1997年6月30日屆滿亦促成中英兩國政府在[[1980年代|1980年代]]初開始談判關於[[#香港前途問題緣起|香港前途問題]]：[[香港政府|香港政府]]有權就香港島和九龍界限街以南土地批出跨越1997年6月30日的租契，在[[1970年代|1970年代]]初起，在香港籌劃多項大型基建項目，投資回本期往往跨越1997年6月30日，需要向國際籌集資金；中國大陸官方對香港前途問題的立場，以及對新界的土地契約安排，對這些大型基建都有十分關鍵影響，香港政府必須即時處理關於香港在1997年6月30日後政治前途的問題<ref>{{cite web |url=http://hkilt.com/news/68_%E6%96%B0%E7%95%8C%E5%9C%9F%E5%9C%B0%E7%BA%8C%E6%9C%9F%E6%A2%9D%E4%BE%8B%EF%BC%8D%E4%BD%95%E5%86%A0%E9%A9%A5%E5%8D%9A%E5%A3%AB-pdf.pdf |title=從《新界土地契約（續期）條例》的解釋看香港雙語法例的問題 |accessdate=2017-07-01 }}</ref>。1979年3月，[[香港總督|香港總督]][[麥理浩|麥理浩]]爵士首度訪問中國大陸，並首次向當時的[[中華人民共和國國務院副總理|中國國務院副總理]][[鄧小平|鄧小平]]提出香港前途問題<ref name="crn2007">{{cite web |url=http://news.sohu.com/20070620/n250670966.shtml |title=衛奕信：當年想續租新界被拒絕 |publisher=搜狐新闻 |date=2007-06-20 |accessdate=2017-06-30 }}</ref>。

== 開始談判 ==
[[香港總督|香港總督]][[麥理浩|麥理浩]]爵士的官方訪問提醒中英雙方要為香港前途作進一步的討論及外交接觸，也為[[英國首相|英國首相]][[戴卓爾夫人|戴卓爾夫人]]1982年的首次中國大陸訪問鋪路<ref>《香港回歸歷程：鍾士元回憶錄》</ref>{{rp|19-21}}。1980年代初，英國向中华人民共和國提出分拆香港的「主權」及「治權」：主權歸還中國、英國保留治權<ref name="fmcoprc1">{{cite web |url=http://www.fmcoprc.gov.hk/chn/sgwj/zyjh/t162295.htm |title=中英關於香港問題的談判——王桂生大使在新中國外交五十五週年座談會上的發言 |publisher=[[中華人民共和國外交部駐香港特別行政區特派員公署|中華人民共和國外交部駐香港特別行政區特派員公署]] |date=2004-09-27 |accessdate=2009-02-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20090517100419/http://www.fmcoprc.gov.hk/chn/sgwj/zyjh/t162295.htm |archivedate=2009-05-17 }}</ref>。1982年9月24日，[[鄧小平|鄧小平]]會見[[戴卓爾夫人|戴卓爾夫人]]，此前中國總理（[[趙紫陽|趙紫陽]]）同她舉行會談；戴卓爾夫人堅持《南京條約》、《北京條約》及《展拓香港界址專條》仍然有效，提出如果中國同意英國1997年後繼續管治香港，英國可以考慮中國提出之主權要求<ref name="鄧3">{{cite book |title=《鄧小平文選》第三卷 |location=北京 |publisher=人民出版社 |year=1993 }}</ref>{{rp|388}}。但鄧小平明確表明主權問題沒有妥協的空間，雙方未能達成共識<ref>{{cite book |language=zh-cn |title=《全球化與國家主權: 比較分析》 |author=程琥 |publisher=清華大學出版社 |year=2003 |isbn=9787302071532 |accessdate=2009-02-23 }}</ref>{{rp|117}}。在記者會上，戴卓爾夫人重申三條條約的合法性，並指出所有國家都須要尊重國際條約，否則任何談判皆無法進行<ref>{{cite book |title=《鄧小平傳》 |author=楊碧川 |publisher=一橋出版社 |year=1995 |isbn=9579932506 }}</ref>，惟中华人民共和国未持有三條條約正本。她堅持按照《南京條約》、《北京條約》及1898年的《展拓香港界址專條》，英國享有香港島及鴨脷洲和九龍半島及昂船洲的主權<ref name="lufanzhi1">{{cite book |language=zh-hant |title=《香港:從殖民地到特別行政區》 |author=魯凡之 |year=1982 |accessdate=2009-02-23 }}</ref>{{rp|221}}。傳統英國資金（英資）的商業力量，如[[香港上海匯豐銀行|香港上海匯豐銀行]]、[[怡和洋行|怡和洋行]]，亦曾參與游說工作。1983年5月，香港新華社組織商界青年領袖包括李鵬飛，組成「青年才俊團」前往北京。其中一些成員提出有關建議，但遭鄧小平否定<ref>{{cite book |title=《香港回歸歷程：鍾士元回憶錄》}}</ref>{{rp|47-48,54-55}}。

=== 英國立場 ===
{{Commons|File:关于准许殖民地国家及民族独立的宣言.pdf|关于准许殖民地国家及民族独立的宣言}}
英國一貫之政策乃協助其殖民地恢復主權成為獨立國家，但這並不適用於香港，基於新界部份是從中國租借所得。英國理解中國大陸對香港主權歸屬的立場，自1972年11月2日[[聯合國大會|聯合國大會]]批准《关于准许殖民地国家及民族独立之宣言》之適用地方名單剔除「香港」後，英國從未照會聯合國秘書長列入「香港」於該名單。香港人知悉中國大陸反對[[香港獨立|香港獨立]]，因此英國無法安排香港獨立。香港90%的土地經《展拓香港界址專條》租借而來（指新界土地），以九十九年為限，九十九年過去後英國將喪失該90%土地的一切權利<ref>{{cite web |url=http://fc95d419f4478b3b6e5f-3f71d0fe2b653c4f00f32175760e96e7.r87.cf1.rackcdn.com/820924%201030%20MT-Deng%20(962-69).pdf |title=Prime Minister's Visit to Peking }}</ref>。不論獨立與否，香港將無法單憑剩下的10%土地而運作，因此英國須盡快與中國大陸商討香港前途<ref>{{cite web |url=http://fc95d419f4478b3b6e5f-3f71d0fe2b653c4f00f32175760e96e7.r87.cf1.rackcdn.com/830310%20CC(83)%208th%20Annex%20(128-77-12).pdf |title=Most Confidential Record to CC(83) 8th Conclusions, Minute 2 }}</ref>。

=== 中華人民共和國立場 ===
鄧小平於[[1970年代|1970年代]]末首先提出「[[一國兩制|一國兩制]]」構思<ref name=元/>{{rp|4}}。官方說法是“对香港恢复行使主权”<ref name=元/>{{rp|v}}，或者“政权交接”<ref name=元/>{{rp|3}}。1982年9月24日，邓小平會見英國首相戴卓爾夫人時談話〈我們對香港問題的基本立場〉稱：“……关于主权问题，中国在这个问题上沒有回旋余地。坦率地讲，主权问题不是一個可以討論的問題。……”<ref name=鄧3/>{{rp|12}}中方在「收回主權」同時，容許香港擁有獨立的經濟及政治制度<ref name=":0">{{cite web |url=http://www.info.gov.hk/info/sar5/c12.htm |publisher=香港政府 |title=一國兩制 }}</ref><ref>{{cite web |url=http://www.info.gov.hk/gia/general/200306/14/0614174.htm |publisher=香港政府 |title=署理保安局常任秘書長出席於香港大學舉行的國家安全立法研討會致辭全文 |date=2003-06-14 }}</ref>，並由香港人自行管理，即所謂「港人治港、高度自治」，同樣政策亦適用於澳門，「澳人治澳，高度自治」<ref>{{cite web |url=http://big5.taiwan.cn/zt/szzt/2008lhtbbd/gzbg/zfgzbg/200803/t20080306_600267.htm |publisher=新華網 |title=堅定不移地貫徹"一國兩制""港人治港""澳人治澳"、高度自治的方針 |date=2008-03-06 }}</ref>。鄧小平同時希望能借此辦法解決[[台灣問題|台灣問題]]<ref>{{cite book |language=zh-hant |title=中共《反分裂國家法》的戰略意涵 |author=陳亦偉 |url=https://books.google.com.hk/books?id=GNm9-37SZ5YC&pg=PA84&dq=%E4%B8%AD%E5%85%B1%E3%80%8A%E5%8F%8D%E5%88%86%E8%A3%82%E5%9C%8B%E5%AE%B6%E6%B3%95%E3%80%8B%E7%9A%84%E6%88%B0%E7%95%A5%E6%84%8F%E6%B6%B5&hl=zh-TW&sa=X&ved=0ahUKEwigmOaj8eXUAhWKXbwKHbOhBI8Q6AEIJDAA#v=onepage&q=%E4%B8%AD%E5%85%B1%E3%80%8A%E5%8F%8D%E5%88%86%E8%A3%82%E5%9C%8B%E5%AE%B6%E6%B3%95%E3%80%8B%E7%9A%84%E6%88%B0%E7%95%A5%E6%84%8F%E6%B6%B5&f=false |publisher=秀威出版 |isbn=9789866732218 |accessdate=2009-02-24 }}</ref>{{rp|23}}。

=== 中華民國立場 ===
在英國與中共談判過程中，[[中華民國|中華民國]]即表示不承認的立場。1983年10月7日，[[行政院院長|行政院院長]][[孫運璿|孫運璿]]在答覆立委[[徐亨|徐亨]]、張寬、湯煥暉質詢時，表示中華民國政府極為關切香港五百萬同胞自由與福祉。<ref>{{cite news |title=〈政府關切香港同胞福祉 全力支持維護自由活動〉〈孫院長並盼自由國家羣起聲援 重申英匪談判我一概不予承認〉 |publisher=《[[中央日報|中央日報]]》頭版 |date=1983-10-08 }}</ref>並嚴正聲明香港是[[滿清|滿清]]割讓，而中華民國推翻滿清，繼承滿清法統，任何涉及香港問題時須以中華民國為談判對象，英國與中華人民共和國有關香港一切談判與協議，中華民國一概不承認。<ref>{{cite news |title=〈政府關切香港同胞福祉 全力支持維護自由活動〉〈孫院長並盼自由國家羣起聲援 重申英匪談判我一概不予承認〉 |publisher=《[[中央日報|中央日報]]》頭版 |date=1983-10-08 }}</ref>

1997年6月27日香港移交前三日，[[中華民國外交部|中華民國外交部]]部長[[章孝嚴|章孝嚴]]和[[國立故宮博物院|國立故宮博物院]]院長[[秦孝儀|秦孝儀]]召開關於《中英南京條約》特展的記者會，記者會上都特別強調外交部特展所展出的《南京條約》是唯一的正本，中國大陸所擁有的，只是清廷欽差大臣[[耆英|耆英]]和英方全權代表[[砵甸乍|砵甸乍]]所簽署的副本，經兩國政府首長簽名核准的《南京條約》正本，由中華民國外交部所保存，而英國為求割讓九龍而簽訂的《北京條約》和租借新界的《展拓香港界址專條》的正本都同樣正由中華民國持有，因此中華民國才是香港主權的合法擁有者。

然而英國依《展拓香港界址專條》規定將九龍租借地，連同依其他不平等條約取得之香港及九龍割讓地交還予同屬[[中華民族|中華民族]]之政權，結束香港的殖民統治，當時由[[中國國民黨|中國國民黨]]主政的中華民國政府在「中華民國政府對『九七』香港情勢的立場與政策說帖」中對此表示欣慰，並希望香港今後能持續自由、民主、繁榮的發展，以及保持香港在[[國際社會|國際社會]]之地位<ref>{{cite web |url=http://www.mac.gov.tw/ct.asp?xItem=45567&CtNode=6076&mp=1 |title=中華民國政府對「九七」香港情勢的立場與政策說帖 }}</ref>。

1997年之前，港生赴台讀書即可取得[[中華民國國民身分證|中華民國國民身分證]]，其後政策取消<ref>{{Cite article |url=https://www.yzzk.com/cfm/blogger3.cfm?id=1459147184510 |title=香港與中華民國 |author=蕭偉基 |publisher=《[[亞洲週刊|亞洲週刊]]》 |date=2016-03-28 }}</ref>。

== 中英談判障礙 ==
[[File:WPPeoplesHerosMonument.jpg|thumb]]

中英兩國隨後發表外交公報，同意繼續循外交途徑商討<ref>{{cite book |language=zh-cn |title=《鄧小平"一國兩制"思想研究》 |author=程林勝 |publisher=遼寧人民出版社出版 |year=1992 |accessdate=2009-02-23 }}</ref>{{rp|501}}。戴卓爾夫人離開北京隨即訪問香港，成為首位在任內訪港的英國首相。[[1983年英國大選|1983年英國大選]]，戴卓爾夫人領導[[保守黨_(英國)|保守黨]]乘[[福克蘭戰爭|福克蘭戰爭]]之勢勝出，繼續執政。

1983年7月12日至7月13日中英兩國政府代表團舉行第一輪會談，由於英方仍然堅持1997年後英國繼續管治香港，直至第四輪會談毫無進展<ref name=鄧3/>{{rp|388}}。中方團長為中國副外長[[姚廣|姚廣]]<ref>{{cite book |language=zh-cn |title=《鄧小平外交》 |author=張植榮 |publisher=海南出版社 |year=1996 |isbn=9787806178003 |accessdate=2009-02-23 }}</ref>，英方團長則是英國駐華大使[[柯利達|柯利達]]爵士，香港總督[[尤德|尤德]]爵士亦以英方成員身份參加談判<ref>{{cite book |language=zh-hant |title=風雨三十年：李鵬飛回憶錄 |page=53 |author=李鵬飛 |location=香港 |publisher=TOM (Cup Magazine) Publishing Limited |year=2004 |isbn=9789889775476 |accessdate=2009-02-23 }}</ref>。英方提出「以主權換[[治權|治權]]」，名義上承認中國擁有香港的主權，以換取英國繼續管治香港的權利<ref name="1c2sti">{{cite book |title=《“一國兩制”的理論與實踐》 |series=鄧小平理論研究叢書 |author=[[王鳳超|王鳳超]] |publisher=經濟科學出版社 |year=1998 |isbn=9787505815773 }}</ref>。在兩日會談後，新聞公報以「有益，有建設性」形容，但談判實際上並不順利。7月25日－7月26日，談判繼續，但新聞公報只說「有益」，但「有建設性」的字眼欠奉<ref name="cuhk30years">{{cite book |language=zh-hant |title=邁進中的大學: 香港中文大學三十年, 1963-1993 |author=吳倫霓霞 |publisher=香港中文大學出版社 |year=1993 |isbn=9789622016057 |accessdate=2009-02-23 }}</ref>{{rp|125}}。英方未放棄「以主權換治權」的構想，中方則堅持主權治權不可分開<ref name=1c2sti/>。中英雙方其後在8月2日至8月3日、9月22日至9月23日分別進行第三、四輪談判，兩次會談的新聞公報只表示「進行了一次會談」，沒有「有益，有建設性」的字眼，顯示雙方未能達成共識之餘，期間更傳出談判破裂的消息，觸發香港市民及投資者對前途的憂慮<ref>《風雨三十年：李鵬飛回憶錄》</ref>{{rp|75-76}}。

1983年9月鄧小平會見訪華的前英國首相[[希思|希思]]時說，英國想用主權換治權是行不通的，勸告英方改變態度，以免出現到1984年9月中國不得不單方面公布解決香港問題方針政策之局面<ref name=鄧3/>{{rp|388}}，而英方一度針鋒相對試圖以經濟牌對應<ref>{{Cite web |url=https://www.sohu.com/a/35935346_239524 |title=历史上中英对于香港回归的谈判过程 |accessdate=2017-11-14 |date=2015-10-16 |work=www.sohu.com}}</ref>。9月16日至9月24日，港元曾在一個星期內下跌20%，由7.9港元兌1[[美元|美元]]跌至9.6港元兌1美元的歷史低點<ref>{{cite book |language=zh-cn |title=《「一國兩制」與香港》 |author=施漢榮 |publisher=廣東人民出版社 |year=1995 |isbn=9787218020686 |accessdate=2009-02-23 }}</ref>{{rp|243}}。同一時期，物價一度飛漲，市面出現搶購糧食等情況<ref name=cuhk30years/>；[[恆生指數|恆生指數]]亦告大幅下跌150點，跌至780點，導致[[1981年香港股災|股災]]的出現<ref>《香港回歸歷程：鍾士元回憶錄》</ref>{{rp|49-51}}。促使香港政府在1983年10月15日起實施[[香港聯繫匯率制度|聯繫匯率制度]]，以7.80港元兌1美元的匯率與美元掛鈎，來維繫港元穩定<ref>{{cite web |url=http://www.info.gov.hk/hkma/chi/public/hkmalin/02.pdf |publisher=香港政府 |title=香港的聯繫匯率制度 |accessdate=2009-02-23 }}</ref>。

1984年1月，香港政府增加[[香港的士|的士]]牌費<ref>{{cite web |url=http://www.legco.gov.hk/yr83-84/english/lc_sitg/hansard/h840111.pdf |title=Official Report of Proceedings - 11 January 1984 |accessdate=2010-05-30 |author=Hong Kong Legislative Council |language=en }}</ref>{{rp|387-395}}，在九龍區引起[[騷亂|騷亂]]（[[1984年香港的士罷駛事件|1984年香港的士罷駛事件]]）。1984年3月14日，[[香港立法會|香港立法局]]首席非官守議員[[羅保|羅保]]，在立法局會議上提出動議，所有任何有關香港前途之建議，在未達成最後協定前，必須在立法局辯論<ref>{{cite web |url=http://www.legco.gov.hk/yr83-84/english/lc_sitg/hansard/h840314.pdf |title=Official Report of Proceedings - 14 March 1984 |accessdate=2010-05-30  |author=Hong Kong Legislative Council |format=pdf |pages=702-704 |language=en }}</ref>。在香港前途未明朗下，3月28日，英資的[[怡和洋行|怡和洋行]]宣佈遷總公司至[[百慕達|百慕達]]<ref>{{cite web |title=Jardines History 1980-1999 |publisher=Jardines Matheson Web Site |accessdate=2017-06-30 |url=http://www.jardines.com/the-group/history/1980-1999.html |deadurl=yes |archiveurl=https://www.webcitation.org/6BiZeClm1?url=http://www.jardines.com/the-group/history/1980-1999.html |archivedate=2012-10-27 }}</ref>。1994年3月，[[怡和控股|怡和控股]]和怡和策略宣佈取消在香港上市地位，其他[[怡和集團|怡和集團]]成員亦在9月宣佈同一決定<ref>{{cite web |title=香港股市大事紀要 |accessdate=2017-07-01 |deadurl=yes |url=https://www.hkex.com.hk/chi/stat/statrpt/factbook/documents/28.pdf |archiveurl=https://web.archive.org/web/20160304104445/https://www.hkex.com.hk/chi/stat/statrpt/factbook/documents/28.pdf |archivedate=2016-03-04 }}</ref>。

在香港回歸20年後2017年中旬解放軍國防大學教授[[金一南|金一南]]著書披露了當年中英會談的內幕，其實1992年10月，時任軍委副主席[[劉華清|劉華清]][[上將|上將]]把解放軍接防進香港的方案遞給[[鄧小平|鄧小平]]看。鄧小平的批示只有兩個字：軟了！鄧小平的文後解釋也很簡短，兩點說明：
*軍隊的香港接受方案只有和平接受方案，沒有武力接收方案，不行。
*1997年7月1日，香港必須收回。不管英方用什麼借口拖延，如果出現拖延回歸，解放軍就開進香港，一錘子砸死。

金一南書中說明事後這一簽呈蓄意透過管道洩漏給外界，據說英國人對中國大陸這個方案是相當震驚的，但也顯示當年會談中英國態度確實有不順暢之處<ref>{{cite web |url=http://www.chinatimes.com/realtimenews/20170805004281-260408 |title=鄧小平：英軍如拖延 解放軍就開進香港一錘砸死！ |date=2017-08-05 |publisher=中時電子報 |accessdate=2018-10-28}}</ref>。

== 中英取得共識 ==
1984年1月25日至1月26日舉行的第八輪談判，中英雙方各自更換團長，中方由外交部部長助理[[周南_(新華社社長)|周南]]擔任，英方則由[[伊文思_(英國外交官)|伊文思]]爵士擔任團長，並兼任駐華大使職務。從1984年4月第十二輪會談後，雙方轉入討論過渡時期香港之安排和有關政權移交事項<ref name=鄧3/>{{rp|389}}。4月英國[[外交及國協事務大臣|外相]][[賀維|賀維]]訪華，鄧小平會見他<ref name=鄧3/>{{rp|389}}。賀維就香港主權問題讓步，會談取得重大突破。4月20日，賀維在香港發表聲明，宣佈1997年7月1日以後，英國不再擁有香港的主權和治權。

1984年6月22日、23日，鄧小平分別會見香港工商界訪京團和香港知名人士[[鍾士元|鍾士元]]等談話〈一個國家，兩種制度〉稱：
{{quote|
「中國政府為解決香港問題所採取的立場、方針、政策是堅定不移的。我們多次講過，我國政府在一九九七年恢復行使對香港的主權後，香港現行的社會、經濟制度不變，法律基本不變，生活方式不變，香港自由港的地位和國際貿易、金融中心的地位也不變，香港可以繼續同其他國家和地區保持和發展經濟關係。我們還多次講過，北京除了派軍隊以外，不向[[香港特區政府|香港特區政府]]派出[[幹部|幹部]]，這也是不會改變的。我們派軍隊是為了維護國家的安全，而不是去干預香港的內部事務。我們對香港的政策[[五十年不變|五十年不變]]，我們說這個話是算數的。我們的政策是實行『[[一個國家，兩種制度|一個國家，兩種制度]]』，具體說，就是在中華人民共和國內，十億人口的大陸實行[[社會主義|社會主義]]制度，香港、台灣實行[[資本主義|資本主義]]制度。……所謂香港人沒有信心，這不是香港人的真正意見。目前中英談判的內容還沒有公布，很多香港人對中央政府的政策不了解，他們一旦真正了解了，是會完全有信心的。我們對解決香港問題所採取的政策，是國務院總理在第六屆全國人民代表大會第二次會議的政府工作報告中宣布的，是經大會通過的，是很嚴肅的事。如果現在還有人談信心問題，對中華人民共和國、對中國政府沒有信任感，那末，其他一切都談不上了。我們相信香港人能治理好香港，不能繼續讓外國人統治，否則香港人也是決不會答應的。未來香港特區政府的主要成分是[[愛國主義|愛國者]]，當然也要容納別的人，還可以聘請[[外國人|外國人]]當[[顧問|顧問]]。什麼叫愛國者？愛國者的標準是，尊重自己[[民族|民族]]，誠心誠意擁護[[祖國|祖國]]恢復行使對[[香港|香港]]的[[主權|主權]]，不損害香港的繁榮和穩定。只要具備這些條件，不管他們相信[[資本主義|資本主義]]，還是相信[[封建主義|封建主義]]，甚至相信[[奴隸|奴隸]][[主義|主義]]，都是愛國者。我們不要求他們都贊成中國的[[社會主義|社會主義]][[制度|制度]]，只要求他們愛祖國，愛香港。……」<ref name=鄧3/>{{rp|58-61}}
}}

1984年7月底，賀維再度訪華。7月31日，鄧小平會見賀維時稱：
{{quote|
「……根據香港和台灣的歷史和實際情況，不保證香港和台灣繼續實行資本主義制度，就不能保持它們的繁榮和穩定，也不能和平解決祖國統一問題。……」<ref name=鄧3/>{{rp|67}}
}}

1984年8月1日，賀維在香港舉行記者會，透露協議的十項主要條款。其後的會談進一步就香港在1997年後的土地、民航、國籍等問題進行討論，並達成多項共識。9月18日雙方就全部問題達成協議，並對協議的文字措辭反覆磋商，並於9月26日草簽《[[中英聯合聲明|中英聯合聲明]]》和三個附件<ref name=鄧3/>{{rp|390}}。草案在北京[[人民大會堂|人民大會堂]]，由周南及伊文思代表簽署。至此，為時兩年之中英兩國政府關於香港問題之談判圓滿結束<ref name=鄧3/>{{rp|390}}。至草案簽署為止，雙方共進行了22次的談判。10月3日，鄧小平會見港澳同胞國慶觀禮團時稱：
{{quote|
「……現在有些人就是擔心我們這些人不在了，政策會變。感謝大家對我們這些老頭子的信任。今天我要告訴大家，我們的政策不會變，誰也變不了。因為這些政策見效、對頭，人民都擁護。既然是人民擁護，誰要變人民就會反對。聯合聲明確定的內容肯定是不會變的。我們中央政府、中共中央即使在過去的動亂年代，在國際上說話也是算數的。講信義是我們民族的傳統，不是我們這一代才有的。這也體現出我們古老大國的風度，泱泱大國嘛。作為一個大國有自己的尊嚴，有自己遵循的准則。我們在協議中說五十年不變，就是五十年不變。我們這一代不會變，下一代也不會變。到了五十年以後，大陸發展起來了，那時還會小裡小氣地處理這些問題嗎？所以不要擔心變，變不了。再說變也並不都是壞事，有的變是好事，問題是變什麼。中國收回香港不就是一種變嗎？所以不要籠統地說怕變。如果有什麼要變，一定是變得更好，更有利於香港的繁榮和發展，而不會損害香港人的利益。這種變是值得大家歡迎的。如果有人說什麼都不變，你們不要相信。我們總不能講香港資本主義制度下的所有方式都是完美無缺的吧？即使資本主義發達國家之間相互比較起來也各有優缺點。把香港引導到更健康的方面，不也是變嗎？向這樣的方面發展變化，香港人是會歡迎的，香港人自己會要求變，這是確定無疑的。我們也在變。……再一個是有些人擔心干預。不能籠統地擔心干預，有些干預是必要的。要看這些干預是有利於香港人的利益，有利於香港的繁榮和穩定，還是損害香港人的利益，損害香港的繁榮和穩定。現在看起來，香港從現在到一九九七年會有秩序地度過十三年，十三年之后，會有秩序地度過五十年。這我是有信心的。但切不要以為沒有破壞力量。這種破壞力量可能來自這個方面，也可能來自那個方面。如果發生動亂，中央政府就要加以干預。由亂變治，這樣的干預應該歡迎還是應該拒絕？應該歡迎。所以事物都要加以具體分析。……所以不能籠統地反對參與，也不能籠統地反對干預。港人治港不會變。由香港人推選出來管理香港的人，由中央政府委任，而不是由北京派出。選擇這種人，左翼的當然要有，盡量少些，也要有點右的人，最好多選些中間的人。這樣，各方面人的心情會舒暢一些。處理這些問題，中央政府從大處著眼，不會拘泥於小節。一個是怕變，一個是怕干預，還怕什麼？有人說怕亂。亂就得干預，不只中央政府要干預，香港人也要干預。總會有人搗亂的，但決不要使他們成氣候。……我講過中國有權在香港駐軍。我說，除了在香港駐軍外，中國還有什麼能夠體現對香港行使主權呢？在香港駐軍還有一個作用，可以防止動亂。那些想搞動亂的人，知道香港有中國軍隊，他就要考慮。即使有了動亂，也能及時解決。……某種動亂的因素，搗亂的因素，不安定的因素，是會有的。老實說，這樣的因素不會來自北京，卻不能排除存在於香港內部，也不能排除來自某種國際力量。……只要香港同胞團結起來，選擇好的政治人物來管理香港，就不怕變，就可以防止亂。即使發生亂，也不會大，也容易解決。一九九七年以後，台灣在香港的機構仍然可以存在，他們可以宣傳「三民主義」，也可以罵共產黨，我們不怕他們罵，共產黨是罵不倒的。但是在行動上要注意不能在香港制造混亂，不能搞「兩個中國」。他們都是中國人，我們相信，他們會站在我們民族的立場，維護民族的大局，民族的尊嚴。在這樣的基礎上，進行他們的活動，進行他們的宣傳，在香港這種情況下是允許的。總之，協議簽訂後會遇到很多新情況。我們過去講過，要了解新情況，解決新問題，這就是新情況、新問題。坦率地講，將來會出現什麼問題，我們也不清楚，但問題出現了我們會合情合理地處理。……」<ref name=鄧3/>{{rp|72-76}}
}}

1984年12月19日，中英兩國在北京簽署關於香港問題的《[[中英聯合聲明|中英聯合聲明]]》<ref name=元/>{{rp|4}}。中英兩國政府首腦在北京正式簽署關於香港問題之聯合聲明<ref name=鄧3/>{{rp|390}}。[[中國國務院總理|中國國務院總理]][[趙紫陽|趙紫陽]]與[[英國首相|英國首相]][[戴卓爾夫人|戴卓爾夫人]]在[[中共中央軍委主席|中共中央軍委主席]][[鄧小平|鄧小平]]、中國國家主席[[李先念|李先念]]和英國外相[[賀維|賀維]]等人見證下，分別作為中英代表在[[人民大會堂|人民大會堂]]正式簽署聯合聲明：「中華人民共和國政府聲明：收回香港地區（包括[[香港島|香港島]]、[[九龍|九龍]]和『[[新界|新界]]』，以下稱香港）是全中國人民的共同願望，中華人民共和國政府決定於1997年7月1日對香港恢復行使主權。」「聯合王國政府聲明：聯合王國政府於1997年7月1日將香港交還給中華人民共和國。」《聲明》第三段阐述了中国政府对香港的基本方针政策，其具体说明由《香港特别行政区基本法》予以规定，并在50年内不变。1985年5月27日，兩國互換批准書，《聯合聲明》隨即生效<ref name=元/>{{rp|4}}。1985年6月12日，中英兩國政府將《聯合聲明》送交[[聯合國|聯合國]]登記<ref name=元/>{{rp|4}}。

== 過渡期 ==
=== 起草《基本法》 ===
《[[香港特別行政區基本法|香港特別行政區基本法]]》是香港特別行政區憲制性法律文件，以法律形式把「一國兩制」概念規定下來；基本法起草委員會和基本法諮詢委員會於1985年成立，負責起草《基本法》，以及徵求公眾對《基本法》之意見<ref name=元/>{{rp|4}}。1985年4月，中華人民共和國第六屆全國人民代表大會三次會議決定成立[[香港特別行政區基本法起草委員會|香港特別行政區基本法起草委員會]]，成員包括香港和中國大陸人士。1988年4月，《基本法》草案首稿公布，並隨即進行公眾諮詢工作，為期5個月；[[全國人民代表大會|全國人民代表大會]]常務委員會在1989年2月批准了草案第二稿，以便作進一步諮詢，草案反映很多第一輪諮詢所得意見<ref name=元/>{{rp|4}}。第二輪諮詢工作在1989年10月結束，起草委員會根據該輪諮詢結果，進一步修改《基本法》草案第二稿；最後，全國人民代表大會於1990年4月，正式通過《基本法》草案第二稿與特別行政區區旗、區徽圖案<ref name=元/>{{rp|4}}。《基本法》於1997年7月1日生效，是香港特別行政區憲制性法律文件，把「一國兩制 」創新概念以法律形式規定下來，清楚確定香港實行不同於內地之社會制度、經濟制度和政治體制，也保障香港市民生活方式、自由和權利，確保香港司法機構獨立運作，並讓香港全權處理外交和國防以外一切內部事務<ref name=元/>{{rp|4}}。香港也可以享有全面財政自主權和擁有獨立貨幣制度；維持香港特別行政區獨立關稅區地位，以及直接與世界各國攜手合作，打擊戰略商品交易、販毒和非法轉運，並且保護知識產權<ref name=元/>{{rp|4}}。

由1984年12月19日《中英聯合聲明》簽署、至1997年7月1日香港回归期間的12年，被稱為過渡期。期間香港經歷多番起跌，包括[[1987年香港股災|1987年香港股災]]、1989年[[六四事件|六四事件]]及所引起的[[香港移民潮|香港移民潮]]、[[香港機場核心計劃|香港機場核心計劃]]的爭議、1990年代中英爭議以至主權移交前的歌舞昇平等。

=== 移民潮 ===
[[File:Gastown.jpg|thumb]]
{{main|香港移民潮}}
中英達成協議，香港於1997年7月1日移交中华人民共和國，一些曾經在中國大陸經歷政治動盪的人（如知名作家[[倪匡|倪匡]]）等開始計劃[[移民|移民]]。1989年發生[[六四事件|六四事件]]，更使香港人感到悲觀和恐懼，1990至1994年出現了開埠以來最大規模之移民潮，五年共有約30萬人移民外地<ref>{{cite report |url=https://www.statistics.gov.hk/pub/B1010005012008XXXXB1300.pdf |title=香港的發展(1967-2007)-統計圖表集 |publisher=香港政府統計處 |work=香港保安局 |date= 2008-09 }}</ref>{{rp|21}}。[[加拿大|加拿大]]、[[澳洲|澳洲]]和[[美國|美國]]是當時不少港人移民的熱門之選。在移民潮高峰，非州島國[[佛得角|佛得角]]也刊登廣告宣傳[[投資移民|投資移民]]<ref>{{Cite news |url=https://www.am730.com.hk/column/財經/移民佛得角-57166 |title=移民佛得角 |author=李鴻彥 |publisher=[[am730|am730]] |date=2014-07-08 }}</ref><ref>{{Cite news |url=https://www.thestandnews.com/media/我在電視台的日子-三十三-佛得角-移民中轉站/ |title=我在電視台的日子（三十三）佛得角 — 移民中轉站 |author=蔡錦源 |publisher=[[立場新聞|立場新聞]] |date=2016-04-28 }}</ref>。

不少港人移民至加拿大的[[溫哥華|溫哥華]]，而位於溫哥華南邊，同屬[[大溫哥華區域局|大溫哥華]]的[[列治文市|列治文市]]又有「小香港」之稱<ref>{{cite web |url=http://www.ccas.com.cn/Article/HTML/6691.html |title=中國烹飪大師獻藝溫哥華中華美食節報導之二 |publisher=中國烹飪協會 |date=2008-03-24 |accessdate=2009-02-27 }}</ref>，其他熱門的地区有[[多倫多|多倫多]]、[[墨爾本|墨爾本]]、[[悉尼|悉尼]]、[[奧克蘭_(紐西蘭)|奧克蘭]]、[[新加坡|新加坡]]、[[台灣|台灣]]等。期間，香港有不少人才外流，及至主權移交前一兩年，一些移民-{回}-流香港。

=== 居英權計劃 ===
{{main|居英權計劃}}
1990年，英國政府宣佈選擇性給予50,000個家庭英國居留權，主要給予曾經從事敏感職位的香港華人政府公務員及各行各業精英，包括最有能力及動機申請移民的人士，如[[政治部_(香港警察)|警隊政治部]]、駐中国大陆記者怕被「秋後算賬」<ref>{{cite news |url= https://hk.news.appledaily.com/local/daily/article/20140517/18723128 |title=記協主席堅決留守　拒棄船 |publisher=蘋果日報 |date=2014-05-17 }}</ref>。居英權持有人可隨時前往英國定居，而無需在英國逗留數年以取得居留權，俗稱「坐移民監」，而持有人的子女可在英國就讀公立中學。當局希望通過此舉使獲得居英權的香港華裔精英安心在香港生活。一旦香港出現社會問題，他們可選擇隨時前往英國，從而減輕香港人才及資金的流失問題<ref>{{cite web |url=http://www.legco.gov.hk/yr90-91/chinese/lc_sitg/hansard/h901010.pdf |publisher=香港立法局 |date=1990-10-10 |title=立法局會議過程正式紀錄 |accessdate=2009-02-27 }}</ref>。但中方對居英權計劃表示不滿，而《[[中華人民共和國國籍法|中華人民共和國國籍法]]》亦對居英權不予承認，香港一般華裔人士被視為中華人民共和國公民。

=== 拆卸九龍寨城 ===
[[File:Kowloon-Walled-City-1898.jpg|thumb]]
[[九龍寨城|九龍寨城]]對於中英雙方是特殊的地方，因為根據《[[北京條約|北京條約]]》和《[[展拓香港界址專條|展拓香港界址專條]]》，九龍寨城是[[大清|大清]]的[[外飛地|外飛地]]，會定期派官員駐守。[[第二次世界大戰|第二次世界大戰]]結束後，[[中華民國|中華民國]]無力管理九龍寨城，香港政府卻沒有法律權力去管理<ref>{{cite book |url=http://www.districtcouncils.gov.hk/kc/tc_chi/links/files/KCD_Heritage_Txt.pdf |title=九龍城區風物志 |date=2005年11月 |publisher=九龍城區議會 |work=香港大學中文系 }}</ref>{{rp|16-17}}。於是，九龍寨城成了充斥“黃、賭、毒”的所謂「三不管地帶」（即中國政府不管，英國政府不管，香港政府不管<ref>九龍城寨史話 三聯書店(香港)出版 ISBN 978-962-04-0682-9</ref>）。1987年，兩國共同決定拆毀寨城、遷徙居民，而九龍寨城亦於1987年和1989年上半年分兩期進行遷徙，並且於1990年和1992年上半年完成。九龍寨城最終在1993年拆卸，改建成[[九龍寨城公園|九龍寨城公園]]。

=== 拆卸調景嶺 ===
1949年[[中華民國政府|中華民國政府]]遷移[[台灣|台灣]]後，香港政府將部份未及遷台的[[國民革命軍|國民革命軍]]退役老兵及家眷安置於新界東南地理位置偏僻的[[調景嶺|調景嶺]]。當時調景嶺沒有陸路交通可達，自成一局，成為[[寮屋|寮屋]]區。故此，調景嶺具有很濃厚的政治色彩，居民長期懸掛[[青天白日滿地紅旗|青天白日滿地紅旗]]，每年10月10日慶祝[[中華民國國慶|中華民國國慶]]等，香港政府並沒有干預，但隨著香港和澳門兩地主權即將移交中華人民共和國以及政府發展[[將軍澳新市鎮|將軍澳新市鎮]]，調景嶺寮屋區在1996年拆卸，並於1999年開始興建地鐵（今[[港鐵|港鐵]]）[[將軍澳綫|將軍澳綫]]連接[[東九龍|東九龍]][[油塘|油塘]]和[[港島|港島]][[北角|北角]]。所有居民在政府補償下遷徙至其他地區，當中大部份人都遷進了將軍澳的[[厚德邨|厚德邨]]。

=== 興建新機場 ===
{{main|香港機場核心計劃}}
[[File:Tsing_Ma_Bridge_(1).jpg|right]]]]
1989年[[六四事件|六四事件]]後，香港政府為了穩定民心，[[香港總督|香港總督]][[衛奕信|衛奕信]]爵士在《[[香港施政報告|施政報告]]》發表「港口及機場發展策略」，宣佈興建新機場。計劃包括在[[大嶼山|大嶼山]][[赤鱲角|赤鱲角]][[填海|填海]]興建全新的[[香港國際機場|香港國際機場]]及[[青馬大橋|青馬大橋]]等相關設施，共耗資超過1200億港元<ref>{{Citation | title=總督施政報告 |periodical=立法局會議過程正式紀錄 |date=1989-10-11 | page=24 |url=http://www.legco.gov.hk/yr89-90/chinese/lc_sitg/hansard/h891011.pdf }}</ref>。當時的香港總督[[衛奕信|衛奕信]]爵士宣佈此項計劃時，描述到計劃落成後的香港將會像「[[玫瑰園|玫瑰園]]」一般美好，因此被傳媒稱為「玫瑰園計劃」。但由於此計劃耗資龐大，被中國政府質疑英方打算在主權移交前耗盡儲備之嫌。1991年9月，[[英國首相|英國首相]][[馬卓安|馬卓安]]親身到北京，成為[[六四事件|六四事件]]後首位訪華的[[西方國家|西方國家]]領袖，與當時的中國國務院總理[[李鵬|李鵬]]簽署《[[香港機場核心計劃的新機場諒解備忘錄|香港機場核心計劃的新機場諒解備忘錄]]》，工程才得以順利開展，英方承諾留下至少250億港元財政儲備<ref>{{cite book |title=《香港回歸歷程：鍾士元回憶錄》 |author=[[鍾士元|鍾士元]] |publisher=[[中文大學|中文大學]]出版社 |year=2001 |isbn=9629960192}}</ref>{{rp|138-141}}，最終留下3700億<ref>{{Cite book |url=http://www.hksar20.gov.hk/chi/pdf/photobook/The_Facts.pdf |title=慶祝香港特別行政區成立20周年 - 紀念圖冊 - 香港便覽 |publisher=香港特區政府 |year=2017年6月 }}</ref>。

=== 臨時立法會 ===
由於對“[[新九組|新九組]]”等問題爭議，中方拒絕承認由彭定康在主權移交前、被指以「三違反」方案產生的立法局，單方面決定「另起爐灶」，取消[[代議政制|代議政制]]「[[直通車_(政治術語)|直通車]]」過渡1997年的安排。1996年3月24日，[[香港特別行政區第一屆政府推選委員會|香港特別行政區第一屆政府推選委員會]]決議成立[[臨時立法會|臨時立法會]]<ref name=元/>{{rp|11}}。中方決定「另起爐灶」，以在主權移交後取代原有的立法局。臨時立法會的任期不得超越1998年6月30日<ref name=元/>{{rp|11}}。12月21日，推選委員會選出60位臨時立法會（簡稱臨立會）議員<ref name=元/>{{rp|11}}。根據香港港英政府法例，此會是非法組織。1997年1月25日，臨時立法會在[[深圳|深圳]]舉行首次會議，其後臨時立法會及其屬下委員會所有會議均在深圳舉行，直至6月30日<ref name=元/>{{rp|11}}。英國管治時代的立法局在7月1日零時解散，未有進入臨立會的議員即時失去議員身份，當中部份在6月30日晚上至凌晨在立法局大樓上向市民發表宣言，揮手道別。

臨立會在7月1日即時開會，除了訂立確認主權移交後「必不可少」之法例以外，亦立即推翻多條由前立法局通過、被認為「對特區政府運作做成障礙」的法例<ref>{{cite book |url=https://books.google.com.hk/books?id=Az0ZAwAAQBAJ&pg=PT131&lpg=PT131 |title=中國大風險 |author=劉曉嵐 |publisher=明鏡出版社 |year=2014 }}</ref>。臨立會重新修訂早期由港英政府通過但迴歸2年前才廢止的爭議性的《[[公安條例_(香港)|公安條例]]》，規定超過指定人數（數額在港英時代常有變動）公眾集會及遊行必須向[[警務處處長|警務處處長]]申請「[[不反對通知書|不反對通知書]]」後方可合法進行。然該法例自港英時代即一向廣受爭議，被指剝奪集會自由、干涉打壓言論自由等人權<ref>{{cite web |title=存档副本 |accessdate=2006-01-24 |deadurl=yes |url=http://www.hkfs.org.hk/declarations/020914.html |archiveurl=https://web.archive.org/web/20060108013245/http://www.hkfs.org.hk/declarations/020914.html |archivedate=2006-01-08 }}</ref>。

== 重要官員 ==
=== 英方成員 ===
多位英方官員起著重要作用，包括英國駐華大使兼英方談判團長[[柯利達|柯利達]]爵士，及接任的伊文思爵士。香港總督[[尤德|尤德]]爵士亦是英方重要成員之一，他不幸於1986年12月任內期間，在訪問北京時病逝。接任的港督[[衛奕信|衛奕信]]爵士，在中英談判期間是英方重要成員。他熟悉中國，與中方保持着良好關係，同時亦成為影響後期香港施政的原因之一<ref name="crn2007" />。

1991年，開始傳出英國有意撤換有「中國通」之稱的香港總督衛奕信<ref name="minford2001" >{{cite book |language=zh-hant |title=《含英咀華》|author=John Minford, Joseph S. M. Lau, 陳虹莊 |publisher=中文大學出版社 |year=2001 |isbn=9789622019102 |page=142 |accessdate=2009-02-23 }}</ref>。1992年[[元旦|元旦]]，英國政府宣佈衛奕信在任期完結前提前退休；1992年7月，曾為[[英國保守黨|英國保守黨]]領袖的[[彭定康|彭定康]]正式出任第28任香港總督，亦成為最後一任港督。彭定康是唯一一位專業政治家出身的港督，又打破了多項前任香港總督的慣例，包括拒絕接受[[英國君主|英國女王]][[伊莉莎白二世|伊莉莎白二世]]頒授的[[爵士|爵士]]勳銜及沒有穿上殖民地官服就職。他擔任香港總督期間作風親民，時常到各區巡視。他巡視[[中環|中環]]時品嚐[[泰昌餅家|泰昌餅家]]的[[蛋撻|蛋撻]]、在[[灣仔|灣仔]]喝[[涼茶|涼茶]]，成為一時佳話。發表[[香港施政報告|施政報告]]後亦親自出席[[電台|電台]]節目接受提問，並主持答問大會面對質詢。

末代香港總督彭定康在政移交前拋出之[[1995年香港立法局選舉|政改方案]]，顯然是步[[楊慕琦計劃|楊慕琦方案]]軌跡加以修改而制訂，是當年夭折政治改革之延續<ref name="錄" />{{rp|151}}。但彭定康的施政未受到中方歡迎，興建新機場的財務安排已惹來北京多番反對，最嚴重的分歧是提出政改方案。方案建議新增九個[[功能組別|功能組別]]（「[[新九組|新九組]]」），被中方認為是鑽《基本法》漏洞，令彭定康被當時的[[中華人民共和國國務院港澳事務辦公室|港澳辦]]主任[[魯平|魯平]]稱作「千古罪人」。這導致主權移交前最後一屆[[香港立法局|香港立法局]]無法以「直通车」形式過渡1997年，中方「另起爐灶」成立[[臨時立法會|臨時立法會]]<ref>{{cite web |url=http://www.360doc.com/content/13/1227/15/6220327_340554731.shtml |title=魯平：堅持彭定康是陰謀顛覆中國的千古罪人 }}</ref>。

=== 中方成員 ===
中英談判期間中方的重要成員包括中方談判團長兼中國外交部部長助理[[周南_(新華社社長)|周南]]，當時的[[國務院港澳事務辦公室|國務院港澳事務辦公室]]主任[[姬鵬飛|姬鵬飛]]。根據曾參與談判的國務院港澳辦主任[[魯平|魯平]]憶述，談判中所有重要的決定，實際上是由中國共產黨領導人[[鄧小平|鄧小平]]親自定案<ref>{{cite book |title=《魯平口述香港回歸》 |page=34-37 }}</ref>。

==== 香港特別行政區主要官員 ====
[[董建華|董建華]]是香港已故船王[[董浩雲|董浩雲]]的長子，大學畢業後，曾於美國公司任職，後回到家族公司[[東方海外|東方海外]]工作。1992年，彭定康上任港督後，他被委任為[[香港行政局|香港行政局]]成員。[[1996年香港特別行政區行政長官選舉|1996年香港特別行政區行政長官選舉]]中，董建華擊敗了[[楊鐵樑|楊鐵樑]]、[[吳光正|吳光正]]和[[李福善|李福善]]而當選，同年接受[[中華人民共和國國務院|中華人民共和國國務院]]委任為首任行政長官。

為保持香港在政權移交前後的穩定，中方同意讓香港政府主要官員順利過渡至特區政府，但部分名稱則有所更改，詳見[[香港特別行政區政府部門|香港特別行政區政府部門]]條目。

香港特區政府首任司級官員如下：
* [[香港特別行政區政務司司長|香港特別行政區政務司司長]]（前稱'''布政司'''）：[[陳方安生|陳方安生]]
* [[香港特別行政區財政司司長|香港特別行政區財政司司長]]（前稱'''財政司'''）：[[曾蔭權|曾蔭權]]
* [[香港特別行政區律政司司長|香港特別行政區律政司司長]]（前稱'''律政司'''）：[[梁愛詩|梁愛詩]]

== 交接過程 ==
[[香港經濟|香港經濟]]在[[1990年代香港|1990年代]]中期有較理想的發展，市民開始以較樂觀態度面對即將來臨的主權移交。[[香港股市|香港股市]]在1997年初多次創出歷史高位。1997年4月，已卸任的前[[英國首相|英國首相]][[戴卓爾夫人|戴卓爾夫人]]再度到訪香港，主持[[青嶼幹線|青嶼幹線]]啟用儀式。

=== 港督府告別儀式 ===
1997年6月30日也舉行一些儀式，標誌156年英國管治的終結<ref name="元" />{{rp|3}}。下午末代港督[[彭定康|彭定康]]告別[[港督府|港督府]]開始港督府告別儀式，舉行了降旗儀式。當天下午4時30分，前港督府舉行香港最後一任港督彭定康告別其官邸的簡單儀式<ref name="元" />{{rp|3}}。[[香港總督旗|香港總督旗]]在《[[最後崗位|最後崗位]]》的軍號聲中隨隨降下，隨後在[[香港警察樂隊|皇家香港警察銀樂隊和風笛隊]]演奏的《[[高地教堂_(歌曲)|高地教堂]]》樂曲聲中，降旗隊伍將總督旗幟轉交港督副官白樂仁，再由港督副官將旗幟轉贈予彭定康。隨後彭定康、夫人[[林穎彤_(彭定康夫人)|林穎彤]]及其三位女兒和隨行人員登上專車在港督府前的花園繞行三圈（但是实际上只繞行了两圈），寓意日後可以重臨香江。下午4時39分，港督彭定康所乘坐的專車在警察電單車的開路下，正式告別港督府<ref>{{youtube|Q4InozQ-D04|港督府告別儀式}}</ref>。

=== 英方告別儀式 ===
[[File:Tamar_site,_Hong_Kong.jpg|thumb]]

英方告別儀式在6月30日晚上6時15分在[[添馬艦_(香港)|添馬艦]]露天場地舉行，主禮嘉賓包括王儲[[威爾斯親王|威爾斯親王]][[查理斯_(威爾斯親王)|查理斯]]、[[英國首相|英國首相]][[貝理雅|貝理雅]]、[[外交及聯邦事務大臣|外交大臣]][[郭偉邦|郭偉邦]]、前首相[[戴卓爾夫人|戴卓爾夫人]]及卸任在即的[[港督|港督]][[彭定康|彭定康]]<ref>{{youtube|AAff-XtOy5Q|英方告別儀式}}</ref>。

添馬艦的告別儀式在特為該活動而建造的場地上舉行，查理斯王子和總督彭定康在儀式上先後致辭<ref name="元" />{{rp|3}}。告別儀式包括傳統的[[英軍|英軍]]{{link-en|日落告別儀式|Colours, standards and guidons#Royal Navy}}，[[港督|港督]][[彭定康|彭定康]]發表臨別致詞，之後由王儲查理斯代表[[英女王|英女王]]致告別詞。其他節目包括名為《香港精神》的團體舞蹈、男高音[[莫華倫|莫華倫]]歌唱等表演，有超過2,000名表演者參與。儀式舉行時一直下著滂沱大雨，令嘉賓及表演者甚為狼狽並添上幾分傷感。日落時份，[[英國國旗|英國]]和[[香港旗|香港]]的旗幟徐徐降下<ref name="元" />{{rp|3}}。晚上7時45分，英國國旗及港英旗在風笛伴奏的樂曲《{{lang|en|Sunset}}》中徐徐降下。

6月30日的英國告別晚宴，由[[英國外交及聯邦事務大臣|英國外交及聯邦事務大臣]][[郭偉邦|郭偉邦]]宴請，有4,000位顥要和貴賓參加<ref name="元" />{{rp|4}}。英國外相郭偉邦為香港未來祝酒，中國國務院副總理兼外長[[錢其琛|錢其琛]]繼而祝酒答謝<ref name="元" />{{rp|4}}。

6月30日午夜，工人在[[中環|中環]][[中區政府合署|中區政府合署]]拆下以[[龍|龍]]和[[獅|獅]]為標記的舊[[香港紋章|香港紋章]]、以及代表英女王[[伊利沙白二世|伊利沙白二世]]的[[皇冠|皇冠]]徽號和「E II R」字樣。另外裝上[[港督府|港督府]]大門的[[英國君主|英國女王]]的[[皇冠|皇冠]]徽號和「E II R」字樣亦同時拆除，並且由[[中華人民共和國國徽|中華人民共和國國徽]]和[[香港特別行政區區徽|香港特別行政區區徽]]取代。

=== [[香港回歸儀式|交接儀式]] ===
[[File:Wan_Chai_Convention_Exhibition_Centre.JPG|thumb]]新翼是香港主權交接儀式的舉行所在地]]
1997年6月30日，交接儀式在晚上11時30分開始，在[[香港會議展覽中心|香港會議展覽中心]]新翼前廳舉行，中英兩國都派出儀仗隊和軍樂隊，[[中華人民共和國主席|國家主席]][[江澤民|江澤民]]和[[查理斯_(威爾士親王)|查理斯王子]]分別致辭<ref name=元/>{{rp|3}}。標誌著中華人民共和國正式對香港行使主權。大禮堂內放置中英兩國國旗及四支旗杆。<BR>

中國主席團由[[中國共產黨中央委員會總書記|中共中央總書記]]兼國家主席江澤民、[[中華人民共和國國務院總理|國務院總理]][[李鵬|李鵬]]、[[中華人民共和國國務院副總理|國務院副總理]]兼外交部长[[錢其琛|錢其琛]]、[[中華人民共和國中央軍事委員會副主席|中央軍委副主席]][[張萬年|張萬年]]上將及首任[[香港特別行政區行政長官|香港特別行政區行政長官]][[董建華|董建華]]代表。<BR>

[[英國|英國]]主席團方面則由[[查理斯王子|查理斯王子]]、[[首相|首相]][[貝理雅|貝理雅]]、[[外交及聯邦事務大臣|外交大臣]][[郭偉邦|郭偉邦]]、最後一任[[香港總督|香港總督]]彭定康及英軍參謀總長格思理代表。<BR>

在即將踏入午夜時，[[英國國旗|英國]]和[[香港旗|香港]]的旗幟徐徐降下<ref name=元/>{{rp|3}}。英國國旗及香港旗在右面旗杆上徐徐降下，同時奏起[[英國國歌|英國國歌]]《[[天佑女王|天佑女王]]》，象徵香港英治時期正式結束。午夜12時正，[[中華人民共和國國旗|中國國旗]]和[[香港特區區旗|香港特別行政區區旗]]升起<ref name=元/>{{rp|3}}。零時過後，香港特區區旗及中國國旗在左面旗杆升起，並奏起[[中國國歌|中國國歌]]《[[義勇軍進行曲|義勇軍進行曲]]》。交接儀式在4,000位應邀出席的嘉賓見證下進行，其中有中國和英國的高層代表、40多個國家的部長和40多個國際組織的代表<ref name=元/>{{rp|3}}。

約6,500位本地與海外傳媒的代表在現場報道儀式的盛況，並傳送給全世界數以億計的電視觀眾<ref name=元/>{{rp|3}}。交接儀式完成後，香港會議展覽中心內隨即進行[[香港特別行政區政府|香港特別行政區政府]]成立暨宣誓儀式。凌晨1時30分，中國副總理錢其琛主持就職儀式<ref name=元/>{{rp|3}}。在儀式中，中國國家主席江澤民向在座4,000多名嘉賓正式宣布中華人民共和國香港特別行政區政府成立<ref name=元/>{{rp|3}}。行政長官（董建華）、主要官員（由[[政務司司長|政務司司長]][[陳方安生|陳方安生]]帶領）、[[行政會議|行政會議]]成員（由召集人[[鍾士元|鍾士元]]帶領）、[[臨時立法會|臨時立法會]]議員（由主席[[范徐麗泰|范徐麗泰]]帶領）、[[香港終審法院|終審法院]]和[[高等法院|高等法院]]法官（由[[首席法官|首席法官]][[李國能|李國能]]帶領）等宣誓就職<ref name=元/>{{rp|3}}。儀式完成後，臨時立法會隨即舉行在香港特別行政區的第一次會議，制定《香港回歸條例》<ref name=元/>{{rp|3}}。

英方主要官員並無出席[[香港政府高級官員|香港政府高級官員]]宣誓儀式。彭定康與家人陪同皇儲查理斯，在[[添馬艦_(香港)|添馬艦]]與[[香港市民|香港市民]]揮手告別，並登上英國皇家遊艇[[不列顛尼亞號|不列顛尼亞號]]離開香港，並由[[駐港英軍|駐港英軍]]末任司令[[鄧守仁|鄧守仁]]少將乘搭英國軍艦[[漆咸號|漆咸號]]隨行護送。貝理雅及其他英國官員則乘飛機返回英國。

7月1日上午10時，香港特別行政區成立慶典在香港會議展覽中心新翼舉行，應邀出席的喜賓逾4,600名<ref name=元/>{{rp|3}}。首先由中國國家主席江澤民發表講話，重申「一國兩制」、「港人治港」和「高度自治」將維持50年不變；繼由行政長官發表就職演辭<ref name=元/>{{rp|3}}。儀式中，香港賣地所得之[[土地基金|土地基金]]移交給[[香港特別行政區政府|香港特別行政區政府]]；儀式結束後，還有盛大的文藝表演，其中一項演出節目是世界首演的交響曲1997《天地人》<ref name=元/>{{rp|3}}。

7月1日下午4時，香港會議展覽中心新翼再度熱鬧起，約5,000名各國顯要和中外貴賓出席香港特別行政區政府招待酒會，冠蓋雲集<ref name=元/>{{rp|3}}。

除官方儀式外，香港本地團體和社區組織也舉行各式各樣活動，其中包括煙花匯演、電視綜合表演、海港花艇燈飾巡遊、文藝表演和各式宴會<ref name=元/>{{rp|3}}。

== 軍事及外交 ==
* '''[[中國人民解放軍駐香港部隊|中國人民解放軍駐香港部隊]]'''：1997年6月30日，509名先遣士兵由[[熊自仁|熊自仁]]少將率領，在晚上9時進入香港<ref name="元" />{{rp|4}}。《基本法》規定，中央人民政府負責管理香港特別行政區防務，駐軍費用由中央人民政府負擔，派駐香港特別行政區的軍隊不得干預特別行政區地方事務，香港特別行政區政府負責維持特別行政區社會治安；中國人民解放軍駐香港部隊由1997年7月1日0時起，正式進駐香港特別行政區，負責香港防務<ref name="元" />{{rp|9}}。[[中國人民解放軍|中國人民解放軍]]沿海、陸、空三路正式進駐香港，軍車經過市區道路抵達各軍營。中英兩國的軍隊在威爾斯親王軍營舉行一個防務交接儀式；7月1日0時，所有軍營均舉行升旗儀式，象徵着防務責任正式移交香港駐軍<ref name="元" />{{rp|4}}。解放軍駐香港部隊由海、陸、空三軍組成，體現中華人民共和國對香港特別行政區海、陸、空主權，並由中央軍事委員會領導；駐香港部隊分別派駐香港特別行政區內14處軍事用地，也有一些屬駐香港部隊海、陸、空三軍部隊人員留駐內地<ref name="元" />{{rp|9}}。軍艦則在7月1日的早上從[[深圳|深圳]][[媽灣|媽灣]]碼頭經過[[維多利亞港|維多利亞港]]等香港水域到達[[昂船洲|昂船洲]]海軍基地。7月1日上午6時，由海、陸、空三軍部隊組成的駐軍主體部隊進駐香港的14個軍營<ref name="元" />{{rp|4}}。解放軍以[[中環|中環]][[添馬艦_(軍艦)|添馬艦]][[威爾斯親王大廈|威爾斯親王大廈]]作為駐港總部，並接收了原[[駐港英軍|駐港英軍]]的各項軍事設施<ref>{{youtube|G5dxVNZcSSM|解放軍進駐香港片段 (香港拍攝)}}</ref><ref>{{youtube|koH6lGlcbn8|解放軍進駐香港片段 (中國內地拍攝)}}</ref>。
* '''[[中國外交部駐港特派員公署|中國外交部駐港特派員公署]]'''：根據《基本法》，[[中华人民共和国国务院|中央人民政府]]負責管理與香港特別行政區有關的外交事務；中華人民共和國外交部在香港設立機構處理外交事務；中央人民政府授權香港特別行政區依照《基本法》自行處理有關的對外事務<ref name="元" />{{rp|8}}。有鑑於此，外交部駐香港特別行政區特派員公署（簡稱「駐港公署」）於1997年7月1日成立，與香港特區政府緊密合作，處理香港特區的外交事務；駐港公署依照《基本法》及 「一國兩制」的原則履行其職責，不會干預香港特區享有高度自治的事務<ref name="元" />{{rp|8}}。中國外交部駐港特派員公署取代[[香港英治時期|香港英治時期]]的[[英國駐香港高級專員公署|英國駐香港高級專員公署]]同時兼備[[領事館|領事館]]性質服務，包括處理中國居民換領、遺失護照或其他旅遊証件及替外國人辦理中國簽證等服務。香港特區政府與駐港公署互相合作之事務主要涉及香港參加國際組織和國際會議、談判簽訂協議，以及處理有關外國領事機構和其他官方或半官方機構之事宜；舉例來說，香港特區可以「中國香港」名義參加不以國家為單位參加的國際組織和國際會議<ref name="元" />{{rp|8}}。辦事處位於[[金鐘|金鐘]][[堅尼地道|堅尼地道]]。香港特區須透過駐港公署向中央人民政府申請批准以中華人民共和國代表團成員或中央人民政府和有關國際組織或國際會議允許之身分，參加以國家為單位的之國際組織<ref name="元" />{{rp|8-9}}。此外，香港特區可在經濟、貿易、金融、航運、通訊、旅遊、文化、體育等適當領域以「中國香港」名義，單獨地同世界各國、各地區及有關國際組織保持和發展關係，簽訂和履行有關協議<ref name="元" />{{rp|9}}。不過，香港特區須得到中央人民政府具體授權，才可與世界各國談判簽訂民用航空運輸協定、司法互助關係協定及互免簽證協議；駐港公署也與香港特區政府合作處理各國在香港特區設立領事機構事宜<ref name="元" />{{rp|9}}。由於中國同英國一樣為單一制國家政體，香港特區主權隸屬於中國中央政府。駐港公署負責之外交事務，包括有關各國在香港特區設立或關閉領事機構、領事任命確認及賦予領事特權和豁免的事宜；香港特區政府則負責處理有關領事機構之日常事務，包括簽發香港領事身分證、實際提供特權和豁免，以及領事機構辦公地點之保安工作<ref name="元" />{{rp|9}}。

== 影響 ==
{{main|香港特別行政區史}}
[[File:QueenVictoria_HK_Statue.jpg|thumb]]的[[維多利亞女王|維多利亞女王]]銅像]]
香港回归是香港歷史上重要的一頁，對香港政治和经济等各方面影響深遠。主權移交前，香港人對[[中共當局|中共當局]]的恐懼帶來了移民潮，曾一度造成人才及資金外流；另一方面，主權移交後加速了中國內地和香港兩地在經濟及社會上的融合。香港回归也標誌著英國自[[維多利亞時代|維多利亞時代]]以來最光輝的「[[日不落帝國|日不落帝國]]」時代的終結。

主權移交也為香港本身帶來不少變化，除殖民地時期的名稱需要更改外，1997年7月1日後，所有政府機構改為懸掛[[中華人民共和國國旗|中華人民共和國國旗]]和[[香港特別行政區區旗|香港特別行政區區旗]]。英國女王[[伊利沙伯二世|伊利沙伯二世]]的肖像從[[港元|硬幣]]、[[郵票|郵票]]和政府辦公室除去，但主權移交前的所有紙幣及硬幣至今仍然流通，大量市民曾經收集「女皇頭」硬幣而引起硬幣荒；郵票上則改印「[[中國香港|中國香港]]」。所有政府機構及組織皆除去「皇家」的稱號，例如[[皇家香港警察|皇家香港警察]]及[[英皇御准香港賽馬會|英皇御准香港賽馬會]]，只有[[皇家香港遊艇會|皇家香港遊艇會]]之英文名稱保留原名。原香港總督的官邸[[港督府|港督府]]改名為[[香港禮賓府|香港禮賓府]]，但首任行政長官未有遷入居住，至[[曾蔭權|曾蔭權]]才遷入居住。在[[法律|法律]]文件中，本來用上「皇家」（Crown）的地方，改稱「國家」（State）；被委任為[[御用大律師|御用大律師]]（Queen's Counsel）的[[大律師|大律師]]，則改稱[[資深大律師|資深大律師]]（Senior Counsel），[[法院|法院]]名稱也作了一些改動。除此以外，幾乎全部以英國王室成員、政府官員或地方命名的香港地方、街道及建築物，例如：[[皇后大道|皇后大道]]、[[維多利亞港|維多利亞港]]、[[彌敦道|彌敦道]]等，則仍然保留舊有名稱，[[維多利亞公園|維多利亞公園]]亦依舊保留了[[維多利亞女王|維多利亞女王]]雕像。香港政府公函的英文本來用上[[女王陛下政府_(術語)|On Her Majesty's Service]]，改成On Government Service，中英文排列次序亦由「英中」改成「中英」。

一些與英國和香港有關的公眾假期，例如：[[英女皇壽辰|英女皇壽辰]]、[[香港重光紀念日|香港重光紀念日]]被取消，新加上的假期有[[中華人民共和國國慶日|中華人民共和國國慶日]]、[[香港特別行政區成立紀念日|香港特別行政區成立紀念日]]、[[國際勞動節|國際勞動節]]、[[佛誕|佛誕]]等。[[抗日戰爭勝利紀念日|抗日戰爭勝利紀念日]]假期亦於1999年取消。

[[File:HK_Citybus_10th_Anniversary_Outdoor_ads.JPG|thumb]]

首任[[香港行政長官|香港行政長官]][[董建華|董建華]]在慶祝中華人民共和國香港特別行政區成立大會上致詞以及[[1997年度香港行政長官施政報告|首份施政報告]]中表示積極推動「[[母語教學|母語教學]]」政策。1998年新學年開始，大部份公營和資助中學改以母語中文（[[粵語|粵語]]授課）作為教學語言，當局只同意保留114間[[香港英文授課中學|英文授課中學]]，但[[英語|英語]]仍然是中小學校的必修科，而部份學校亦開始於部分學科上使用[[普通話|普通話]]教授。可是這種做法遭到香港很多家長的反對，擔心以「母語教學」會使香港學童的英文水平有所下降。<ref>{{cite web|url=http://hknews.hksyu.edu/index.php/%E5%90%8C%E6%99%82%E5%AD%B8%E7%BF%92%E4%B8%AD%E8%8B%B1%E6%96%87%E6%88%90%E6%95%88%E5%B7%AE_%E6%AF%8D%E8%AA%9E%E6%95%99%E5%AD%B8%E5%BF%85%E7%84%B6%E5%A4%B1%E6%95%97|title=同時學習中英文成效差 母語教學必然失敗 - 香港新聞網 - 樹仁新傳系學生實習習作|author=|date=|website=hknews.hksyu.edu|accessdate=2018-10-28}}</ref><ref>{{cite web |url=http://home.netvigator.com/~fyw12541/lang_crit/motherton_%20english.htm |title=論「母語教學」與「英文水平下降」的關係 |deadurl=yes |archiveurl=https://archive.fo/20120630212652/http://home.netvigator.com/~fyw12541/lang_crit/motherton_%20english.htm |archivedate=2012-06-30 }}</ref>亦有人表示，香港作為東西方交流的樞紐，雙語教學可以使香港人在國際上更有競爭力。

香港在主權移交後一些地方也因《[[香港特別行政區基本法|香港特別行政區基本法]]》<ref name=":0" />維持不變，享有高度自治。例如，香港繼續享有多方面的獨立地位，包括獨立的司法系統，繼續沿用[[英美法系|普通法]]，中文及英文同時是法定語言，並繼續單獨以地區身份成為[[世界貿易組織|世界貿易組織]]成員。

香港與中國大陆的陸路邊界線基本不變，但深圳河北面水域、[[后海灣|-{后}-海灣]]（中方稱深圳灣）北面水域、[[沙頭角|沙頭角]]海北面水域和[[大鵬灣_(南海)|大鵬灣]]北面水域移交[[廣東省|廣東省]][[深圳市|深圳市]]管理。中國大陆將[[大嶼山|大嶼山]]西面部份領海（今[[分流_(水文学)|分流]]西灣）；[[索罟群島|索罟群島]]、[[蒲台島|蒲台島]]以南部份水域劃入香港水域以作補償。兩地繼續維持獨立的出入境政策，中國大陆居民必須申請[[往来港澳通行证|往来港澳通行证]]才可進入香港。主權移交後，中國大陆為香港另設[[香港特別行政區護照|香港特別行政區護照]]來取代[[英國屬土公民護照|英國屬土公民護照]]，可於多國包括[[歐盟|歐盟]]及[[日本|日本]]等享有免[[簽證|簽證]]待遇，出入境程序較中國大陆居民簡單快捷。香港的道路沿用英國的左上右落即[[右駕|右駕]]，有別於中國大陆的右上左落的[[左駕|左駕]]，中國大陆與香港的車輛均需分別申請牌照才可使用對方道路。

在體育方面，香港仍可獨立以「中國香港」的名義參加國際體育賽事，包括[[奧運會|奧運會]]等，由[[中國香港體育協會暨奧林匹克委員會|中國香港體育協會暨奧林匹克委員會]]負責，亦擁有獨立的[[香港足球代表隊|香港足球代表隊]]。由於主權移交後香港脫離[[英聯邦|英聯邦]]的行列，在1997年之後便沒有參加[[英聯邦運動會|英聯邦運動會]]。

=== 中國大陸賀禮 ===
1997年7月1日上午10時，香港特別行政區成立慶典儀式中，中華人民共和國中央人民政府和31個省、自治區和直轄市給香港特別行政區致送禮物<ref name=元/>{{rp|3}}。賀禮包括刺繡、陶瓷、漆器、掛毯、木雕和石雕等。為慶祝香港回歸中國15周年，[[香港文化博物館|香港文化博物館]]於2012年6月20日至2013年2月25日展出其中五件的回歸賀禮，包括：中央人民政府贈送的雕塑「永遠盛開的紫荊」、海南省贈送的貝雕「天涯共此時」、吉林省贈送的松花硯「松花紫荊情繫根」、黑龍江省贈送的福桃花瓶「紫荊歸春」、山西省贈送的「應縣木塔模型」<ref>{{cite web |url=http://www.heritagemuseum.gov.hk/archive/chi/exhibitions/Past_Exhibitions_191.html |title=香港文化博物館-禮重情隆：回歸賀禮精選 }}</ref>。

== 國際反應 ==
{{Wikisourceshas|香港澳門關係條例|護照條例 (民國89年)}}{{Wikibooks|中華民國法律註解/護照條例/2000年}}
在臺灣，1997年3月18日，[[中華民國|中華民國]][[立法院|立法院]]制定《香港澳門關係條例》62條，同年4月2日[[中華民國總統|中華民國總統]]公布<ref>{{cite web |url=http://lis.ly.gov.tw/lghtml/lawstat/version2/01839/0183986031800.htm |title=《香港澳門關係條例》首次版本，立法院法律系統 |deadurl=yes |archiveurl=https://web.archive.org/web/20131002164111/http://lis.ly.gov.tw/lghtml/lawstat/version2/01839/0183986031800.htm |archivedate=2013-10-02 }}</ref>。6月19日，[[行政院|行政院]]令發布，《香港澳門關係條例》涉及香港部分，定於7月1日施行<ref>{{cite web |url=http://law.moj.gov.tw/LawClass/LawHistory.aspx?PCode=Q0010003 |title=《香港澳門關係條例》沿革，中華民國法務部全國法規資料庫 }}</ref>。2000年4月28日，立法院修正《護照條例》全文時，第9條的立法理由四，用「香港及澳門均已由中共接管」敘述此事<ref>{{cite web |url=http://lis.ly.gov.tw/lghtml/lawstat/reason2/0130389042800.htm |title=護照條例民國89年立法理由，立法院法律系統 |deadurl=yes |archiveurl=https://web.archive.org/web/20170305001837/http://lis.ly.gov.tw/lghtml/lawstat/reason2/0130389042800.htm |archivedate=2017-03-05 }}</ref>。

[[美國-香港政策法|美國-香港政策法]]<ref>{{cite web |url=http://www.wenxuecity.com/news/2004/09/02/17021_print.html |title=美國會醞釀重訂《香港法》，星島日報，2004-09-02 }}</ref>是一部由[[美國國會|美國國會]]在1992年通過、關於[[英屬香港|香港]]在1997年7月1日結束[[英國殖民地|英國殖民地]]管治及成為[[中華人民共和國|中華人民共和國]]轄下的一個[[特別行政區|特別行政區]]後，[[美國政府|美國政府]]重新釐定對港政策的[[法案|法案]]。根據此法案，美國政府將繼續把香港視作一個在[[政治|政治]]、[[經濟|經濟]]、[[貿易|貿易]]政策方面與[[中國大陸|中國大陸]]完全不同的地區，並在對外政策上把香港特區政府與中華人民共和國政府區別對待。

美國國務卿[[马德琳·奥尔布赖特|马德琳·奥尔布赖特]]1997年4月15日演講表示，「全世界將以希望與關切的複雜情緒，注視著香港重回中國的版圖。美國支持此一基於一九八四年中英聯合宣言的轉變，其條件是香港仍保持高度自治及目前的生活方式與基本自由」，以及「決定應英國與[[中華人民共和國政府|中國政府]]的邀請，代表美國參加七月的交接典禮。」<ref>[[美國在台協會|美國在台協會]]：[http://www.ait.org.tw/zh/officialtext-bg9720.html 文獻篇: 美國國務卿瑪德蘭．歐布萊特一九九七佛萊斯特講座：美國在東亞的原則與目標美國海軍官校馬里蘭州安納波里斯市一九九七年四月十五日]</ref>
