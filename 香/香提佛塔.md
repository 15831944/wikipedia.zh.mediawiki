'''香提佛塔'''（Shanti Stupa）是一个[[佛教|佛教]]白色圆顶[[窣堵坡|窣堵坡]]，位于[[印度|印度]]北部[[查谟和克什米尔邦|查谟和克什米尔邦]][[拉达克|拉达克]]列城区Chandspa村的一个山顶上<ref name="Temples"/>，由日本僧人兴建于1983年-1991年，以促进世界和平和繁荣，以及纪念佛教2500年<ref name="Temples"/><ref name = "bhasin"/>它被认为日本人民和拉达克之间联系的标志<ref name = "bhasin"/>。

香提佛塔的设想由藤井日達在1914年提出<ref name = "bhasin"/>，要在世界各地建造和平佛塔和寺庙，并尝试在印度复兴佛教。<ref name = "bhasin"/>

香提佛塔建设始于1983年4月，[[第十四世达赖喇嘛|第十四世达赖喇嘛]]于1991年8月为香提佛塔揭幕<ref name="Temples">{{cite web|url=http://www.buddhist-temples.com/buddhist-monastery/ladakh/shanti-stupa.html|title=Shanti Stupa|publisher=Buddhist-temples.com|accessdate=2009-10-19}}</ref><ref name = "bhasin"/><ref name="bajwa"/>。

[[File:Shanti_stupa_main_buddha.jpg|thumb]]
香提佛塔的基座有[[释加牟尼|释加牟尼]]的[[遗物|遗物]]<ref>{{cite web |title= Leh|url=http://www.ndtv.com/ndtvtravels/travel_xtopspot.aspx?topspot=Leh |date= |publisher=[[NDTV|NDTV]] |page=}}</ref><ref name = "bhasin"/>。佛塔为两层建筑，第一层中央为浮雕[[法轮|法轮]]，两侧为鹿。第二层的浮雕描绘佛的诞生、圆寂和击败魔鬼。

自建成以来，香提佛塔已成为受欢迎的旅游景点。《[[印度教徒报|印度教徒报]]》称它是列城周围“最著名的旅游胜地”，虽然它的建筑风格不同于拉达克风格。<ref name="Temples"/><ref>{{cite news|url=http://www.hindu.com/2005/08/21/stories/2005082100341100.htm|title=Ladakh monuments cry for renovation |last=Luv Puri|date= 2005-08-21|publisher=[[印度教徒报|印度教徒报]]|accessdate=2009-12-01}}</ref> 香提佛塔海拔高度3609米<ref name="bajwa">{{cite book|last= Jagir Singh Bajwa, Ravinder Kaur|title=Tourism Management|pages=117|url=https://books.google.com/books?id=TVXCiwdwPxsC&pg=PA117&dq=shanti+stupa+Ladakh&lr=&as_brr=3&client=firefox-a#v=onepage&q=shanti%20stupa%20Ladakh&f=false | isbn=978-81-313-0047-3 | year=2007 | publisher=APH Publishing}}</ref>，距离拉达克的前首都[[列城|列城]]5公里 - 面对[[列城王宫|列城王宫]]的陡峭的山上<ref name = "bhasin">{{cite book|last= Bhasin|first=Sanjeev Kumar|title=Amazing land Ladakh: places, people, and culture|pages=119–20|chapter=Shanti Stupa|url=https://books.google.com/books?id=8IZloNzI8BgC&pg=PA120&dq=shanti+stupa+Ladakh&as_brr=3&client=firefox-a#v=onepage&q=shanti%20stupa%20Ladakh&f=false | isbn=978-81-7387-186-3 | year=2006 | publisher=Indus Publishing}}</ref>香提佛塔的位置，使之可以看到列城和周围山脉的全景。<ref>{{cite book|title=Frommer's India|publisher=Wiley Publishing Inc.|location=Hoboken, NJ, USA|pages=524|url=https://books.google.com/books?id=qMqBvtL_nI0C&pg=PT534&dq=shanti+stupa+Ladakh&as_brr=3&client=firefox-a#v=onepage&q=shanti%20stupa%20Ladakh&f=false | isbn=978-0-470-16908-7}}</ref><ref name="visitladakh">{{cite web|url=http://www.visitladakh.com/places-to-see/leh-.html|title=Leh: Places to see|work=VISITLADAKH.COM|accessdate=2009-12-01}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 从香提佛塔可以看到极美的日出和日落。<ref name = "tour"/>佛塔从上午5点到下午9：00向游客开放。在晚上用灯光照亮。<ref name="tour">{{cite web|url=http://www.buddhist-tourism.com/countries/india/temples-monuments/shanti-stupa.html|title=Shanti Stupa|year=2007|work=Buddhist-Tourism.Com|accessdate=2009-12-01}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>
