'''香港崇正總會'''又名'''崇正會館'''，於1921年9月29日成立，由當年的[[香港大學|香港大學]][[香港大學中文學院|中文學院]]首任系主任[[賴際熙|賴際熙]]，以及多名僑商包括[[李瑞琴|李瑞琴]]、[[黃茂林|黃茂林]]、[[廖新基|廖新基]]、[[徐仁壽|徐仁壽]]、[[古瑞庭|古瑞庭]]等創立，後經[[胡文虎|胡文虎]]、[[陳承寬|陳承寬]]和[[張發奎|張發奎]]等客家聞人的奔走努力，得以發揚光大。是[[香港|香港]]的歷史悠久的[[客家|客家]]人組織，也是世界上創辦最早、影響力大的客家社會組織之一。香港崇正總會的現任總會長為黎錦文。<ref name = "香港崇正总会:崇德崇义再谱客家正气歌">{{cite web | language = zh | author = 陈嘉良 |publisher = 梅州網| title = 香港崇正总会:崇德崇义再谱客家正气歌| url = http://www.meizhou.cn/news/0909/26/090926001.html | date =  2009-09-26 | accessdate = 2015-02-28 }}</ref>

==總會宗旨==
[[File:香港崇正總會樓.jpg|right]]
聯絡各地客家人士，考證客家源流，交換知識，振興工商事業，教學育才，共謀利益，互勉同人，發揚志業，建立全球客家崇正聯合總會組織。<ref name = "崇正同人系譜">{{Cite book | author = 賴際熙 | title = 崇正同人系譜 | location = 香港 | publisher = 香港崇正總會 | date = 1995 | pages = | ISBN = | accessdate = 2015-02-28 | url = http://books.google.com.my/books/about/%E5%B4%87%E6%AD%A3%E5%90%8C%E4%BA%BA%E7%B3%BB%E8%AD%9C.html?id=qmxBnQEACAAJ&redir_esc=y | language = zh | quote = }}</ref>

==概況==
香港崇正總會原名“旅港崇正工商总会”，1926年徐仁寿先生建议，总会冠以“工商”二字范围太狭窄，不利于团结广大客家乡亲，遂决议称为“香港崇正总会”剛創辦時崇正總會是以工商交流為主，後來辦有慈善、教育及研究客家源流工作，崇正之名乃代表崇尚正義、正本清言之意義。<ref name = "香港崇正總會簡介">{{cite web | language = zh | author = 楊達詳 |publisher = 中山客家網| title = 香港崇正總會簡介| url = http://www.zskjr.com/xxgk/ShowArticle.asp?ArticleID=318 | date = 2008-04-10 | accessdate = 2015-02-28 }}</ref>

==社會公益==
中國抗戰期間，香港崇正總會除了成立救濟難民委員會，撥款捐資救助難民外，還捐錢支持抗日戰爭。

另外，香港崇正總曾會在1922年至1940年期間，先後在香港創辦了4間崇正義學。1957年又創辦崇正中學和崇正小學<ref name="崇正中學">{{cite web| language = zh| publisher = 崇正中學官方網頁| title = 崇正中學簡介| url = http://www.ttms.edu.hk/index2.php?mid=3| date = 2011| accessdate = 2015-02-28| archive-url = https://web.archive.org/web/20150402114841/http://www.ttms.edu.hk/index2.php?mid=3| archive-date = 2015-04-02| dead-url = yes}}</ref>。除此之外，崇正總會也大力支持中國內的教育發展，如：捐助中國農業大學做教育基金、資助全國婦聯培訓婦女師資、捐助華東師範大學設立客家學系、在陝西丹鳳創辦了崇正希望小學等等。<ref name = "香港崇正總會簡介" />每當中國發生天災時，崇正總會也不留餘力地捐助救災活動。

==世界客屬懇親大會==
1971年九月二十九日﹐該會舉行五十周年金禧慶典時﹐召開了世界首次客屬崇正懇親大會。全球五大洲、一百多個社團組織、眾多客家領袖雲集一處，出席盛典，「被譽為海外華人團結新里程碑」。

客屬崇正懇親大會後來也被稱為[[世界客屬懇親大會|世界客屬懇親大會]]，從1971年到2014年已經舉辦了27屆。

==與其他客屬團體==
受香港崇正總會的影響，世界許多客屬會館紛紛改名為『崇正會』，如紐約的客家人會館改名為『旅美紐約崇正會』，檀香山、三藩市、古巴、荷蘭、利物浦等地的客屬組織亦更名為『崇正會』等等。

南洋華僑商人[[胡文虎|胡文虎]]曾領導香港崇正總會支持抗日戰爭。胡文虎也是新加坡南洋客屬總會的創辦人兼首任會長。兩會交流頻繁，關係密切。

==近況==
2013年，香港崇正總會發生內部糾紛。星島新聞集團前主席胡仙、嶺南大學前副校長饒美蛟、理事長黎錦文聯同20名會員入稟高等法院，要求法庭聲明香港崇正總會為慈善組織；另外要求頒令香港崇正總會在律政司司長監督下，重選監理事會，以獲取更好管理。<ref name = "香港崇正總會疑爆內訌">{{cite web | language = zh | publisher =蘋果日報| title = 香港崇正總會疑爆內訌| url = http://hk.apple.nextmedia.com/news/art/20130828/18397994| date = 2013-08-28| accessdate = 2015-02-28 }}</ref> 2015年香港法院已取消傳票 <ref name = "取消傳票" {{https://hk.news.yahoo.com/%E9%A6%99%E6%B8%AF%E5%B4%87%E6%AD%A3%E7%B8%BD%E6%9C%83%E7%8D%B2%E5%87%86%E5%89%94%E9%99%A4%E5%82%B3%E7%A5%A8-092800023.html}}

==聞人==
崇正總會歷年來不乏香港及[[台灣|台灣]]的知名人物，包括[[陳濟棠|陳濟棠]]的後人、[[薛岳|薛岳]]以及[[張發奎|張發奎]]等人。

* [[賴際熙|賴際熙]]，崇正總會的主要創辦人，第一屆至第六屆會長，編纂《赤溪縣志》、《增城縣志》等地方志書。也是香港大學的首任中文系主任。<ref name="賴際熙的港大歲月">{{Cite book |author=方駿 |title=賴際熙的港大歲月 |location=台灣 |publisher=東亞漢學研究學會第三屆國際學術會議 |date=2012 |pages= |ISBN= |accessdate=2015-02-28 |url=http://sinologicalresearch.web.fc2.com/paper/no3/papers/27.pdf|language=zh |quote=}}</ref><ref name="香港崇正總會的緣起與發展">{{Cite book |author=載謝劍、鄭赤琰 |title=香港崇正總會的緣起與發展：一個族群會館的政治適應力的個案研究|location=香港 |publisher=《國際客家學研討會論文集》 |date=1994年 |pages=691-700|ISBN= |accessdate=2015-02-28 |url=http://hakkalib.ncl.edu.tw/pda/cp.jsp?f=lp&xmlid=0000017541&bURL=bHAuanNwP3BhZ2Vfc2l6ZT0xMCZwYWdlPTI3|language=zh |quote= }}</ref>
* [[張發奎|張發奎]]，[[中華民國|中華民國]]陸軍上將，曾任香港崇正總會的會長、榮譽會長。也是首屆[[世界客屬懇親大會|世界客屬懇親大會]]的發起人。
* [[薛岳|薛岳]]，[[中華民國|中華民國]]陸軍上將，香港崇正總會的發起人之一，曾任世界客屬總會理事長。
* [[胡文虎|胡文虎]]，[[緬甸|緬甸]]華僑，新加坡南洋客屬總會的創辦人兼首任總會長，曾領導香港崇正總會捐助抗日戰爭。其養女[[胡仙|胡仙]]也曾擔任崇正總會的會長。
