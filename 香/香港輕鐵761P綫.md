'''輕鐵761P綫'''（{{lang-en|Light Rail Route 761P}}）是連接[[天逸站|天逸站]]和[[元朗站_(輕鐵)|元朗站]]的[[香港輕鐵|香港輕鐵]]路綫，屬[[港鐵|港鐵]]一部分。

== 歷史 ==
來往天水圍至[[元朗市中心|元朗市中心]]的輕鐵服務，早於1993年初輕鐵[[香港輕鐵#天水圍支線（第三期發展路線）|天水圍支線]]通車時一併開始服務；當時的路線編號為'''721'''，來往[[天瑞站|天瑞]]至元朗。

*1993年1月10日，天水圍支線[[天瑞站|天瑞]]至[[坑尾村站|坑尾村站]]啟用，'''721'''綫開辦，由[[天瑞站|天瑞]]往返元朗，取代接駁巴士654綫。
*1995年3月26日，天水圍支線[[天瑞站|天瑞站]]至天水圍總站（今[[天榮站|天榮站]]）的延長段啟用，'''721'''延長至天水圍總站。
*2003年12月7日，天水圍第4期支線和預留區支線通車，'''721'''由'''761'''取代。'''761'''綫從天榮站出發，途經天悅、天恆和頌富，後經天瑞站前往元朗。後來加開'''761P'''綫，路線與'''761'''相同，但只行走[[天逸站|天逸]]至元朗。
*2006年10月8日，'''761P'''加強至全日服務，'''761'''取消。自此，往來天水圍和元朗的輕鐵列車服務終點站縮短至天逸，維持至今。

== 使用情況 ==
本綫是輕鐵唯一一條連接元朗及天水圍的路綫，不少天水圍居民使用本綫前往元朗上班上學及購物（距離比屯門近以及車費較低）等，因此本綫客量從早到晚都是維持高水平。由於本綫客量是輕鐵眾多條路線中最高的，所以本綫列車全部都是拖卡（除了清晨及深夜時分外）。

== 特別班次（已取消） ==
由於761P擴展至全日服務和761的取消，某些原有'''761'''行走的地區（如[[天恆邨|天恆邨]]、[[俊宏軒|俊宏軒]]等屋苑）變相失去了往來元朗的輕鐵服務。在這項重組實施前，當區居民非常不滿此重組路線安排，發起激烈抗爭，走進輕鐵路軌，阻止輕鐵運行，部份輕鐵車站服務中斷<ref>[http://www.rthk.org.hk/rthk/news/expressnews/20061001/news_20061001_55_346415.htm 相關新聞1] 香港電台</ref><ref>[http://www.rthk.org.hk/rthk/news/expressnews/20061001/news_20061001_55_346426.htm 相關新聞2] 香港電台</ref>

當時九鐵採納了部份地區團體的建議，於每日約上午10時至下午3時的非繁忙時段，安排輕鐵761P特別車行走，除了行駛原有761P外，並途經[[天恆站|天恆站]]、[[濕地公園站|濕地公園站]]、[[天秀站|天秀站]]、[[天悅站|天悅站]]，以[[天榮站|天榮]]為終點站，每30分鐘一班。

天水圍方向的特別車的總站顯示屏會照常顯示「天逸」，但會在車頭的玻璃上方掛上黃色牌，顯示以下字句：

== 車序 ==
本線的[[車序|車序]]是'''091-099'''（部份曾是[[香港輕鐵614綫|614]]的車序編號）、'''191-199'''。'''100'''、'''196-200'''為後備車序，正常情況不會使用。

== 安全問題 ==
由於輕鐵大部份路段都設在路面，只有一些高架的路段，輕鐵和行人經常出現爭相過路的情況。「[[元朗大馬路|元朗大馬路]]」（[[水邊圍站|水邊圍]]↔[[元朗站_(輕鐵)|元朗]]），是輕鐵四條路線相交的地方，輕鐵的長途線，[[香港輕鐵610綫|610]]、[[香港輕鐵614綫|614]]、[[香港輕鐵615綫|615]]、761P都經過，大廈商店林立，行人路和輕鐵過路處十分擠迫，當輕鐵行經時，行人也站至近路軌之處，險象環生。
