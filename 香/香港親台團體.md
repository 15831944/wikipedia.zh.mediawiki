'''親臺團體'''，泛稱'''民國派'''、'''親臺派'''、'''親國民黨派'''（{{lang-en|Pro-Taiwan camp、Pro-Taipei camp、Pro-Republic of China camp、Pro-Kuomintang camp或Pro-Taiwan groups}}），用以泛稱成立於[[香港|香港]]、在政治立場上親近[[中華民國|中華民國]]或[[中國國民黨|中國國民黨]]的[[社會團體|社會團體]]或政治團體，反對[[中國共產黨|中國共產黨]]及[[中華人民共和國政府|中華人民共和國政府]]，同時反對，立場貼近[[泛民主派|泛民主派]]，並以推動[[海峽兩岸|海峽兩岸]][[自由|自由]][[民主|民主]]及傳播[[臺灣經驗|臺灣經驗]]為信念。

早年香港的[[親中華民國|民國派]]團體的政治理念都均趨近中國國民黨的[[右派|右派]]，緊隨當年[[中華民國總統|總統]][[蔣中正|蔣中正]]在[[台灣|台灣]]號召之[[光復大陸|光復大陸]]、[[三民主義統一中國|三民主義統一中國]]的口號<ref name="ettoday2007">{{cite web |url = http://www.ettoday.com/2007/07/01/162-2119505.htm |title = 香港回歸10年／親台團體萎縮　部分轉型關注民生議題| date = 2007-07-01}}</ref>。後來受到兩岸形勢的遽變及1997年[[香港主權移交|香港主權移交]]影響使其理念也有所變化，近年來的民國派團體以支持香港實行[[普選|公民普選]]、[[維權|維權運動]]、推動[[兩岸|兩岸]]和平發展為主。除了關注臺港事務、[[兩岸四地|兩岸四地]][[民主|民主]]發展及[[中華民族|民族]]事業外，亦積極透過香港的地區服務，以爭取更多[[香港人|香港人]]認同。

== 概況 ==
成立於1948年的[[港九工團聯合總會|港九工團聯合總會]]是香港早期成立的親臺團體之一。由於正值[[中國國民黨|國民黨]]敗退，大量傾向國民黨的[[中華民國國軍|國軍]]移居香港，成為當時親臺團體的主要成員，並一度活躍於[[香港政治|本土政治領域]]。

自1988年[[蔣經國|蔣經國]]去世及[[李登輝|李登輝]]上台後，國民黨的兩岸政策出現改變，國民黨減少了對海外親臺團體的財政支持，導致香港方面的國民黨勢力逐漸消退；1997年[[香港回歸|香港回歸]]之後，考慮到[[北京當局|北京當局]]的壓力及主流[[意識形態|意識形態]]的轉變，親臺團體變得更為低調；到了2000年[[民主進步黨|民進黨]]執政期間，為因應[[台灣本土化運動|台灣本土化運動]]的大趨勢，[[臺灣|臺灣]]方面對[[港澳|港澳]]親臺團體的支持與關注更是大大減低，使得團體的發展能力受到嚴重打擊。於是部份親臺團體改變策略，不再太過強調[[政治|政治]][[口號|口號]]，反而多參與香港本地的[[民生|民生]]議題，並融入[[社區|社區]]，希望可以吸引新血投入，讓[[香港市民|當地市民]]感受到他們對[[香港社會|香港社會]]的貢獻<ref name="ettoday2007"/>。

==政治立場及活動==
[[File:P1020964.JPG|thumb]]中，親臺團體帶著[[青天白日滿地紅旗|青天白日滿地紅旗]]參與遊行。]]
香港早期的親臺團體大約是在1940年代至1950年代時成立。當時由於有許多支持[[中國國民黨|中國國民黨]]的人士來港，故早期的親臺團體大部分偏向[[右派|右派]]，與親[[中國共產黨|中國共產黨]]的[[左派|左派]]相互抗衡<ref>{{cite web |url=http://andrewffy.sinatown.com/geu0412e.htm |title=從零開始：近代香港的誕生 |deadurl=yes |archiveurl=https://web.archive.org/web/20080330092802/http://andrewffy.sinatown.com/geu0412e.htm |archivedate=2008-03-30 }}</ref>。在當時社會上充斥[[反共|反共]]及恐共氣氛，右派力量堅實，由文教新聞影劇、工商團體、同鄉宗親團體、工會、街坊會等都有不少影響力，尤以1950年代及[[左派|左派]][[六七暴動|六七暴動]]後，[[右派|右派]]支持香港政府恢復社會秩序，曾經有過熱鬧的時期，然而1980年代開始，支持[[國民黨|國民黨]]的勢力逐漸式微，[[泛民主派|泛民主派]]勢力崛起，因為南來[[香港|香港]]的第二、三代土生土長的[[香港人|市民]]，開始認同[[香港核心價值|香港核心價值]]，關注本地事務，而遠離國共兩者的意識型態，加上李登輝上台後，國民黨對兩岸政策的改變、[[臺灣本土化|臺灣本土意識]]及[[民主化|民主化]]，國民黨臺灣本土化及改組，撤出[[香港|香港]]，令親臺團體的政治立場基本上也產生了變化。

親臺團體一向反對[[民主進步黨|民進黨]]和[[臺獨|臺獨]]，1999年，時任[[中華民國總統|總統]][[李登輝|李登輝]]發表「[[兩國論|兩國論]]」，臺灣和中國大陸的關係跌至谷底，2000年，臺灣第一次[[政黨輪替|政黨輪替]]和[[陳水扁|陳水扁]]上台後，親臺團體要求[[李登輝|李登輝]]辭去[[中國國民黨主席|國民黨主席]]一職。2008年，總統[[馬英九|馬英九]]上任後[[兩岸關係|兩岸關係]]有新突破。

親臺團體的立場至今仍支持[[中華民國|中華民國]]，在參與集會遊行時，攜帶[[青天白日滿地紅旗|青天白日滿地紅旗]]及[[孫中山|孫中山]]先生畫像亮相，為其重要特色之一，甚至還多次舉行紀念孫中山先生、[[中華民國國慶日|中華民國國慶日]]（雙十節）及[[元旦|元旦]]開國紀念日等中華民國節慶的活動，亦多次表態支持中華民國加入國際組織，中華民國建國一百年及抗日戰爭暨臺灣光復七十年舉辦多項活動,以正視聽。親臺團體對於[[香港政治|香港政治]]的訴求，則多半支持[[民主|民主]]及[[普選|普選]][[香港特別行政區行政長官|行政長官]]和[[立法會議員|立法會議員]]，並期待香港能及早和臺灣同樣實施[[普選|公民普選]]。1997年香港主權移交後，特別是[[2003年香港七一遊行|2003年香港七一遊行]]後，為響應港人要求實施民主及實現普選的呼聲，香港親臺團體也多次透過遊行集會表達其對普選的訴求<ref>{{cite web |url = http://www.chinayouth.org.hk/20070318/ |title = 神州青年服務社參加318爭普選大遊行}}</ref>。

除了爭取普選之外，親臺團體亦高度關注兩岸議題及臺灣相關事務，除了時常表達對[[中國共產黨|中共]]打壓[[臺灣|臺灣]]的強烈譴責之外，還多次發動反對《[[反分裂國家法|反分裂國家法]]》<ref>{{cite web |url=http://www.chinayouth.org.hk/no-anti-separation/ |title = 反「反分裂國家法」公開信}}</ref>、兩岸和平請願運動、支持中華民國參加[[世界衛生組織|世界衛生組織]]<ref>{{cite web |url=http://www.epochtimes.com/b5/6/5/24/n1328549.htm |title=香港多個團體抗議中共阻攔中華民國參加世衛大會}}</ref>等抗議中共措施的集會。

親臺團體反對[[台獨|台獨]]，公開抗議[[民進黨|民進黨]]的[[去中國化|去中國化]]政策及[[去蔣化|去蔣政策]]<ref>{{cite web |url=http://chiang2006.blog.edoors.com/index.php?op=ViewArticle&articleId=28749/ |title=蔣公中正香港協會等十二個團體示威強烈抗議民進黨政府廢止蔣公紀念日 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web|url=http://www.chinayouth.org.hk/20070201/|title=要求教育部長杜正勝下台抗議行動}}</ref>。此外亦積極參與總統選舉等助選、造勢後援會、赴臺觀摩等活動，籍此引起香港人對民主的關注。對於中華民國主權事宜，也公開表達訴求，包括指[[南沙群島|南沙群島]]主權、[[釣魚台列嶼|釣魚台列嶼]]主權屬於中華民國，抗議[[菲律賓|菲律賓]][[廣大興案|廣大興案]]，[[2014年越南排華暴動|越南排華暴動]]，香港童軍總會月刊矮化臺灣童軍以至近期康文署去國立事件，友中華民國團體均會積極表達意見，亦對臺灣發生天災表達關懷及捐款賑濟，如[[九二一大地震|九二一大地震]]，[[八八水災|八八水災]]，[[高雄氣爆|高雄氣爆]]，[[2016年高雄美濃地震|高雄地震]]等。

香港親臺團體對[[中華人民共和國政府|中華人民共和國政府]]對中國大陸的專制極權管治大都持反對態度，強烈譴責中共箝制[[媒體|媒體]]，以粗暴手段壓制國內一切異見聲音，毋視基本[[人權|人權]]和[[社會公義|社會公義]]。

1997年[[香港主權移交|香港主權移交]]以後，親臺團體的政治生態有所改變，他們的政治地位全面被香港更為反共的[[泛民主派|泛民主派]]吸收。2000年時，由留臺校友組成的香港親臺團體[[一二三民主聯盟|一二三民主聯盟]]，因資源問題解散，在[[2003年香港區議會選舉|2003年香港區議會選舉]]當中，親臺團體紛紛組成選舉聯盟，如荃灣選舉聯盟等在地方參選，其中如香港臺灣大專院校校友會總會長[[麥業成|麥業成]]領軍的[[元朗天水圍民主陣線|元朗天水圍民主陣線]]在[[元朗區議會|元朗區議會]]一度成為除鄉事派以外第二大政治勢力，但在[[2007年香港區議會選舉|2007年香港區議會選舉]]中因為泛民主派的不協調，導致大規模敗陣，僅剩麥業成在[[元朗區|元朗區]]一席，就連另一領袖人物[[梁漢華|梁漢華]]也在[[深水埗區|深水埗區]]因為選區重疊，要與[[香港民主民生協進會|民協]]的區議會副主席[[梁欐|梁欐]]在同一選區對撼，而讓建制派漁翁得利。之後，親臺團體大多投靠由國民黨背景深厚的[[黃毓民|黃毓民]]擔任主席的[[社會民主連線|社會民主連線]]，例如在[[2008年香港立法會選舉|2008年香港立法會選舉]]中，親臺團體曾組織「香港泛藍撐毓民、撐大舊（社民連[[新界西|新界西]]候選人[[陳偉業|陳偉業]]）大遊行」，當中[[梁漢華|梁漢華]]則是黃毓民、[[李偉儀|李偉儀]]（社民連[[九龍西|九龍西]]候選團隊）中的主要輔選成員，而麥業成更當選社民連第三屆行政委員。而後來黃毓民另組[[人民力量|人民力量]]，頗多泛藍中人跟隨移轉，至[[2012年香港立法會選舉|2012年香港立法會選舉]]，人民力量中的泛藍團體退出，而麥業成以藍營[[民主陣線|民主陣線]]（藍陣）名義參與立法，部分香港泛藍中人亦認為國民黨的政策傾向中國大陸及中共，轉向支持民進黨及同情台獨。因此親台團體也在產生變化，大多務實地支持臺港文化經貿交流，尊重民主選舉制度，而在香港的政治影響力已式微。

== 打壓 ==
[[香港主權移交|香港主權移交]]後，在[[香港|香港]]展示[[青天白日滿地紅旗|青天白日滿地紅旗]]變得較以前困難。[[港九工團聯合總會|港九工團聯合總會]]主席[[李國強_(工團)|李國強]]指出，在1997年前港英時期，[[香港街道|香港街道]]、[[建築物|建築物]]、[[住宅區|住宅區]]、[[彌敦道|彌敦道]]兩旁都掛滿[[中華民國國旗|中華民國國旗]]和雙十節標誌。回歸後展示[[中華民國國旗|國旗]]經常遇阻撓，批評[[香港主權移交|主權移交]]後[[自由|自由]]收窄，至近幾年[[兩岸關係|兩岸關係]]好轉，[[青天白日滿地紅旗|青天白日滿地紅旗]]被警方敵視情況才有所減少。<ref>{{cite news|title=【黃安之亂】入台證有青天白日旗 親台工會：不等如支持台獨|url=http://hk.apple.nextmedia.com/realtime/news/20160116/54654204|accessdate=2016-01-16|agency=蘋果日報 (香港)|date=2016-01-16}}</ref>

[[神州青年服務社|神州青年服務社]]亦指出在公眾場所活動中揮舞國旗常遭遇打壓，如在[[2008年中華民國總統選舉|2008年臺灣總統大選]]之時，團體在[[旺角|旺角]]行人專用區直播大選開票情況，但老社員事前被[[香港中聯辦|香港中聯辦]]施壓，表明不希望看到青天白日滿地紅旗，「連佈景板遣詞用字也欲知道，不太接受佈景板印有『中華民國』字眼。」<ref>{{cite news|title=【黃安之亂】港親台團體揮青天白日旗也受干預 商討反撃行動|url=http://hk.apple.nextmedia.com/realtime/news/20160116/54653956|accessdate=2016-01-16|agency=蘋果日報 (香港)|date=2016-01-16}}</ref>目前，位於[[屯門|屯門]]的[[青山紅樓|青山紅樓]]是香港少數仍可長時間公開展示[[青天白日滿地紅旗|青天白日滿地紅旗]]的地方，每逢雙十節慶、蔣公誕等中華民國節日，親臺團體皆會到該處舉行升旗活動，並唱[[中華民國國歌|中華民國國歌]]、[[中華民國國旗歌|國旗歌]]。自2018年的雙十節，紅樓業主突然不准他們入內進行升旗禮，被批評受政治壓力。
