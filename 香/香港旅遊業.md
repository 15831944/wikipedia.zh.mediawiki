'''香港旅遊業'''為[[香港經濟|香港經濟]]4大支柱產業之一，是香港政府重要稅收來源。[[香港|香港]]為世界的其中一座[[經濟|經濟]]、[[金融|金融]]、[[航空|航空]]及[[貨運|貨運]]的樞紐城市，位置具策略性，加上其[[東方|東方]][[傳統|傳統]]與[[西方|西方]][[文化|文化]]共冶一爐，新舊事物相互交融，締造出香港獨特的[[香港文化|都會文化]]，風貌多樣的名勝景點、郊野景致，其中「東方之珠」、「[[購物|購物]]天堂」和「美食天堂」的美譽更是享負盛名，為全球最受歡迎的旅遊勝地之一。

==概況==
1994年，訪港旅客人次有930萬人，至1995年攀升至1,000萬人次，並創下新紀錄<ref>{{cite book|author=施漢榮|title=《香港放大鏡——細看昨天、今天、明天》|date=1996年|publisher=香港教育圖書公司|isbn=962-290-972-8|pages=頁44}}</ref>。

根據2008年統計，香港約有19.7萬人從事旅遊相關行業，佔香港總就業人數的5.6%，旅遊業只佔本地生產總值2.8%，在2010年上升到4.4%。<ref name="tourism.gov.hk1">[http://www.tourism.gov.hk/tc_chi/statistics/statistics_perform.html 2010年旅遊業表現]旅遊事務署 - 旅遊業統計資料 最近修訂日期:2011年4月7日</ref><ref>{{cite web|url=http://www.gov.hk/tc/about/abouthk/factsheets/docs/tourism.pdf|title=香港便覽|publisher=香港特別行政區政府新聞處出版|date=2010年8月|language=zh-hk}}</ref>

2010年，全年到訪香港的旅客人數為3,000萬人次，是1995年時候的三倍，[[中國大陸|中國大陸]]佔了當中的2,270萬人次居首，有1,420萬大陸旅客人次使用了[[港澳個人遊|自由行]]簽注到訪香港，其次是[[台灣|台灣]]，旅客總數達220萬人次，以及來自[[日本|日本]]和[[韓國|韓國]]、[[南亞|南亞]]及[[東南亞|東南亞]]地區的旅客。[[美國|美國]]是香港最主要的長途客源市場。2010年，約有120萬美國旅客到訪香港，[[歐洲|歐洲]]合共有約180萬人次訪港，另外，如[[印度|印度]]、[[中東|中東]]及[[俄羅斯|俄羅斯]]等新興市場亦是到訪香港旅客的主要來源。根據2010年統計，過夜旅客人均消費額為6,728港元，入境旅遊相關的總消費為2,100億港元。<ref name="tourism.gov.hk1"/>2011年，香港與入境旅遊相關的總開支達2,530億[[港元|港元]]，全年到訪香港的旅客人數為4,192萬人次。2012年，全年到訪香港的旅客人數逾4,861萬人次，比較前年增加約16%<ref>[https://archive.is/20130428190457/hk.news.yahoo.com/%E7%9B%A7%E7%82%B3%E6%9D%BE-%E9%A6%99%E6%B8%AF%E7%B6%93%E6%BF%9F%E6%8C%81%E7%BA%8C%E7%99%BC%E5%B1%95-%E5%8A%A0%E5%A4%A7%E5%8A%9B%E5%BA%A6%E6%94%AF%E6%8C%81%E6%97%85%E9%81%8A%E6%A5%AD-211414121.html 盧炳松﹕香港經濟持續發展 加大力度支持旅遊業] 《明報》 2013年1月15日</ref><ref>[http://hk.news.yahoo.com/去年本港旅客增16-內地客佔7成-093000512.html 去年本港旅客增16% 　內地客佔7成]</ref>。

[[世界旅行及旅遊理事會|世界旅行及旅遊理事會]]估計，旅遊業於2011年對香港經濟的直接及間接貢獻，達同香港[[國民生產總值|本地生產總值]]約15.2%。根據世界旅行及旅遊理事會的統計數字，於香港從事旅遊業或相關工作人士逾463,000人，佔了香港勞動人口的12.8%<ref>{{cite web|url=http://www.wttc.org/research/economic-impact-research/country-reports/h/hong-kong/|title=世界旅行及旅遊理事會香港經濟影響報告2013|website=WTTC|deadurl=yes|archiveurl=https://web.archive.org/web/20140303073632/http://www.wttc.org/research/economic-impact-research/country-reports/h/hong-kong/|archivedate=2014-03-03}}</ref>。根據[[香港政府|香港政府]]統計，旅遊業佔2012年[[本地生產總值|本地生產總值]]4.7%；旅遊業就業人數（僅包括直接相關的行業）約250,900人，佔全港總就業人數6.9%<ref>http://www.gov.hk/tc/about/abouthk/factsheets/docs/tourism.pdf</ref>。

於2012年，[[國際|國際]]旅遊網站[[TripAdvisor|TripAdvisor]]根據數以百萬計用戶的投票，香港獲選為[[亞洲|亞洲]]第1位及全球第10位好去處<ref>[http://hk.news.yahoo.com/港膺亞洲最佳去處-220920978.html 港膺亞洲最佳去處] 《星島日報》 2012年1月25日</ref>。同年，由[[萬事達卡|萬事達卡]]公布的2012年全球最佳旅遊城市報告中，香港在全球132座城市中排名第6<ref>[https://archive.is/20120710100626/hk.news.yahoo.com/%E5%85%A8%E7%90%83%E6%9C%80%E4%BD%B3%E6%97%85%E9%81%8A%E5%9F%8E%E5%B8%82%E9%A6%99%E6%B8%AF%E7%AC%AC%E5%85%AD-050025392.html 全球最佳旅遊城市香港第六] 《明報》 2012年6月12日</ref><ref>[http://hk.news.yahoo.com/全球最佳旅遊城市-港排第六-223000861.html 全球最佳旅遊城市 港排第六] 《星島日報》 2012年6月13日</ref><ref>[https://archive.is/20120714062103/hk.news.yahoo.com/%E6%9C%80%E4%BD%B3%E6%97%85%E9%81%8A%E5%9F%8E%E5%B8%82-%E6%B8%AF%E7%AC%AC6%E5%BE%AE%E8%B7%8C-215354332.html 最佳旅遊城市 港第6微跌] 《明報》 2012年6月13日</ref>。


隨著[[啟德郵輪碼頭|啟德郵輪碼頭]]於2013年6月建成，為香港旅遊業增加發展機會。2018年9月，通往[[中國內地|中國內地]]28座主要城市的[[廣深港高速鐵路香港段|廣深港高速鐵路香港段]]通車；同年，融匯[[珠江三角洲|珠江三角洲]]的[[港珠澳大橋|港珠澳大橋]]亦啟用，通往[[澳門|澳門]]、[[珠海|珠海]]和[[珠三角|珠三角]]西部地區的車程將更為快捷，將能夠進一步鞏固香港作為[[亞洲國際都會|亞洲國際都會]]的地位。

香港現時有兩個郵輪碼頭，分別是[[海運大廈|海運大廈]]和[[啟德郵輪碼頭|啟德郵輪碼頭]]。在2016年，停靠香港的郵輪總數有142船次，乘客總流量（只計算旅客）達74.6萬人次<ref>{{Cite web |url = http://www.discoverhongkong.com/tc/about-hktb/annual-report/annual-report-20152016/strategic-focus/cruise-experience |title = 郵輪體驗 2015/16年報 |publisher = 香港旅遊發展局 }}</ref>。

==旅遊景點及設施==
{{Main|香港旅遊景點}}

香港擁有很多[[香港旅遊景點|旅遊景點]]，[[香港政府|香港政府]]亦持續地計劃及發展多元化的旅遊景點，藉以增強香港作為旅遊勝地的吸引力，同時於現有的旅遊點進行改善計劃，務求提高旅遊景點的持續吸引力。

===維多利亞港===
{{main|維多利亞港}}

香港以海港聞名，維多利亞港是香港經濟發展之源，附近兩岸有多個旅遊景點。

;幻彩詠香江
旅遊事務署於2004年1月推出[[幻彩詠香江|幻彩詠香江]]，深受旅客及[[香港市民|香港市民]]歡迎。匯演於2005年11月被列入《-{[[健力士世界紀錄大全|健力士世界紀錄大全]]}-》，成為全球最大型燈光音樂匯演。這項紀錄是根據當時港島20幢參與建築物於每晚的同一時段，以互動燈光及音樂效果，展現維多利亞港夜景的動感而評定的。匯演於2005年12月擴展至九龍區，涵蓋維港兩岸共44幢建築物。
;金紫荊廣場
[[金紫荊廣場|金紫荊廣場]]位於[[灣仔|灣仔]][[香港會議展覽中心|香港會議展覽中心]]新翼人工島上，為了紀念[[香港回歸|香港回歸]]而設立。廣場上放有一座金紫荊雕像，[[香港警察|香港警察]]每日早上8時都會於該處舉行升[[中國國旗|中國國旗]]及升[[香港區旗|香港區旗]]儀式，吸引不少[[中國大陸|中國大陸]]遊客欣賞。
;香港星光大道
[[香港星光大道|香港星光大道]]位於[[尖沙咀|尖沙咀]]東海濱長廊，面對維多利亞港。由香港政府撥款進行美化，於2004年4月27日開幕，為了表揚[[香港電影|香港電影]]界別傑出人士，大道上放有被選出人士的掌印，亦放置了已故巨星[[李小龍|李小龍]]的[[銅像|銅像]]，及[[香港電影金像獎|香港電影金像獎]]的放大版獎座。<ref>{{Cite web |url = http://www.discoverhongkong.com/tc/see-do/highlight-attractions/themed-attractions/avenue-of-stars.jsp |title = 星光大道 |publisher = 香港旅遊發展局}}</ref>
;尖沙咀海濱花園
[[尖沙咀海濱花園|尖沙咀海濱花園]]位於尖沙咀海旁，一條沿海而建、約兩公里長的觀景步道，從該處可以欣賞[[維多利亞港|維多利亞港]]的風景。海濱花園附近有[[前九廣鐵路鐘樓|前九廣鐵路鐘樓]]、[[香港文化中心|香港文化中心]]及[[香港太空館|香港太空館]]等多個景點。<ref>{{Cite web |url = http://www.discoverhongkong.com/tc/see-do/highlight-attractions/harbour-view/tsim-sha-tsui-promenade.jsp |title = 尖沙咀海濱花園 |publisher = 香港旅遊發展局}}</ref>

===太平山頂===
{{main|太平山 (香港)}}

太平山是香港的一個主要旅遊景點，每年吸引逾600萬人次的香港及外地遊客前往。在太平山頂可以俯瞰[[香港島|香港島]]、[[九龍半島|九龍半島]]兩岸及[[維多利亞港|維多利亞港]]的風景，世界知名，被選為人生必到的景點之一。[[凌霄閣|凌霄閣]]及[[山頂廣場|山頂廣場]]為遊客提供購物及飲食的選擇。歷史悠久的[[山頂纜車|山頂纜車]]亦吸引不少遊客乘搭，別有一番滋味。<ref>{{Cite web |url = http://www.discoverhongkong.com/tc/see-do/highlight-attractions/harbour-view/the-peak.jsp |title = 太平山頂 |publisher = 香港旅遊發展局}}</ref>

===香港海洋公園===
{{main|香港海洋公園}}
香港海洋公園是一個以[[海洋|海洋]]為主題的[[主題公園|主題公園]]，位於[[黃竹坑|黃竹坑]]，於1977年1月10日啟用，每年有逾500萬人次入場。在[[港鐵|港鐵]][[南港島綫|南港島綫]]通車前，[[城巴|城巴]]設有專線巴士由不同地方直達公園，為遊客提供便捷的交通。[[港鐵|港鐵]][[南港島綫|南港島綫]]設有[[海洋公園站|海洋公園站]]，有天橋連接海洋公園正門。香港海洋公園設有多款刺激的機動遊戲、[[海洋生物|海洋生物]]表演及其他動物等，園內的水母館、鯊魚館和海洋館更是遊人必到之處。香港海洋公園亦會於節日，例如暑假、萬聖節及聖誕節舉行特別主題活動。園內的登山電梯是全世界第2長的戶外[[電動扶梯|電動扶梯]]。

===香港迪士尼樂園度假區===
{{main|香港迪士尼樂園度假區}}
香港迪士尼樂園度假區位於[[大嶼山|大嶼山]][[竹篙灣|竹篙灣]]，佔地126公頃，於2005年9月12日正式開幕。[[港鐵|港鐵]]設有專用鐵路──[[迪士尼綫|迪士尼綫]]來往[[欣澳站|欣澳站]]及[[迪士尼站|迪士尼站]]，[[城巴|城巴]]及[[龍運巴士|龍運巴士]]也提供專線巴士直達度假區，為遊客提供方便及快捷的交通。目前度假區內包含兩座[[香港迪士尼樂園度假區酒店|主題酒店]]及一座[[主題公園|主題公園]]，它是[[亞洲|亞洲]]第2個[[迪士尼度假區|迪士尼度假區]]，也是[[中國|中國]]首個迪士尼度假區。樂園內包含不同種類的遊樂設施及大型表演，亦會在節日例如[[農曆新年|農曆新年]]、[[暑假|暑假]]、[[萬聖節|萬聖節]]及[[聖誕節|聖誕節]]舉行特別慶祝活動。為期5年的擴建計劃亦陸續竣工，有助香港重新定位為亞洲區內旅遊的首選目的地，預期在未來30-40年為香港帶來約1,500億港元的淨收益。

===昂坪360===
{{main|昂坪360}}
昂坪360是[[香港|香港]]一個連接[[大嶼山|大嶼山]][[東涌|東涌]]及[[昂坪|昂坪]]的[[索道|纜車]][[系統|系統]]──昂坪纜車及其相關的旅遊設施[[昂坪市集|昂坪市集]]等。昂坪纜車是[[世界|世界]]上營運距離最長的吊掛式纜車系統，亦是香港繼-{[[柏架山吊車|柏架山吊車]]}-及[[香港海洋公園纜車|香港海洋公園纜車]]後第三個纜車系統，於2006年9月18日正式通車。於2009年4月，昂坪360獲[[ISO9001|ISO9001]]:2008認證；同月，昂坪360推出全球首創的「水晶車」，深受[[旅客|旅客]]歡迎。

===蘭桂坊===
{{main|蘭桂坊}}
蘭桂坊位於[[中環|中環]]，是一個聚集大小酒吧、俱樂部、餐廳與零售商鋪的中高檔消費區，深受中產階級、外籍人士及遊客的歡迎，它更獲得由中國大陸十大主流媒體合辦的「網友最喜愛的香港品牌評選」之「我最喜愛的香港蒲點」三甲。

==住宿==
香港的[[酒店|酒店]]、[[旅館|旅館]]及[[飯店|飯店]]長期維持著優良的服務水準，並且提供完善的設施。香港的住宅選舉繁多，故此價格差距亦很大，由豪華酒店的逾2,000港元一晚至廉價酒店的200港元一晚都有。對於要求比較低的遊客可以住宿於旅店、[[賓館|賓館]]及[[青年旅舍|青年旅舍]]，設施不及飯店，然而價錢相宜，加上客房基本設施足夠，而且清潔整齊。

==政府參與==
[[香港旅遊事務署|香港旅遊事務署]]是香港政府專責旅遊事務的部門，於1999年5月成立，負責統籌政府內部各項發展旅遊業的工作，並提供更佳的政策支援，以推動香港的旅遊業。該署的任務是確立和促進香港作為亞洲首要的[[國際大城市|國際大城市]]及世界級的度假和商務旅遊目的地。

[[香港旅遊發展局|香港旅遊發展局]]是專責在世界各地宣傳和推廣香港作為商務及消閒旅遊目的地的法定機構。其前身是香港旅遊協會，2001年4月1日根據《香港旅遊發展局條例》進行改組並易名而成。其主要任務，是向世界各國遊客宣傳，吸引他們到香港旅遊，並透過提高設備及服務水準，令旅客在香港有更好的旅遊體驗。旅遊發展局亦舉辦多項活動，包括「香港—樂在此，愛在此！」全球[[廣告|廣告]]宣傳活動、「亞洲盛事之都」及[[香港購物節|香港購物節]]等活動。亦推行[[優質旅遊服務計劃|優質旅遊服務計劃]]以提升香港旅遊業的服務質素。

政府與旅遊業界一直保持緊密合作。由政府、香港旅遊發展局和業內各界代表組成的旅遊業策略小組，專責就旅遊業發展事宜，從策略性角度向政府提出建議。

香港政府繼續透過[[入境事務處|入境事務處]]確保到訪香港的旅客出入境方便快捷。現時約170個國家的國民可以在無需要[[簽證|簽證]]的情況下進入香港境內旅遊7至180日不等。從[[中國大陸|中國大陸]]到訪香港旅客方面，自2002年1月起，香港遊的[[配額|配額]]被取消。指定可以承辦香港遊的中國大陸旅行代理商數目因此大幅度地增加。[[港澳個人遊|港澳個人遊]]計劃於2003年7月28日實施，並且逐步擴展。現時該計劃於全[[廣東省|廣東省]]21座城市，[[北京|北京]]、[[上海|上海]]、[[天津|天津]]、[[重慶|重慶]]、[[南京|南京]]、[[蘇州|蘇州]]、[[無錫|無錫]]、[[杭州|杭州]]、[[寧波|寧波]]、[[台州|台州]]、[[福州|福州]]、[[廈門|廈門]]、[[泉州|泉州]]、[[成都|成都]]、[[濟南|濟南]]、[[大連|大連]]、[[瀋陽|瀋陽]]、[[南昌|南昌]]、[[長沙|長沙]]、[[南寧|南寧]]、[[海口|海口]]、[[貴陽|貴陽]]、[[昆明|昆明]]、[[石家莊|石家莊]]、[[鄭州|鄭州]]、[[長春|長春]]、[[合肥|合肥]]及[[武漢|武漢]]49個城市實行。至2005年8月31日，到訪香港的中國大陸個人遊旅客達到逾860萬人次。

==業界組織==
[[香港旅遊業議會|香港旅遊業議會]]是香港旅遊業界的主要組織，成立於1978年，主要任務是保障旅行社的利益為宗旨。1985年政府制訂《旅行代理商條例》，規定所有經營外遊業務的旅行社必須領取牌照。

香港法例規定，任何公司必須成為香港旅遊業議會會員，才可以申領旅行代理商牌照經營離港外遊服務。旅遊業是[[香港立法會|香港立法會]]的功能組別之一，擁有一個議席。

==批評==
[[File:Hong_Kong_Night_Skyline.jpg|thumb]][[維多利亞港|維多利亞港]]夜景]]
有評論指從前香港成為「[[東方之珠|東方之珠]]」，是因為本地經濟發展自然而成的璀璨[[夜景|夜景]]，而非刻意為旅客做成的「[[幻彩詠香江|幻彩詠香江]]」，並且將一整條街變成金行和化妝品店，是將香港當成一間沒有生活特色及風土人情的[[免稅店|免稅店]]，長遠來說不利旅遊業<ref>勇先. [http://hk.apple.nextmedia.com/news/art/20130130/18150260 景點不是為遊客而設].</ref>。

==問題==
一些在香港經營，承接內地[[旅行團|旅行團]]的香港接待旅行社被揭發出不單以零團費來承接內地團，更以負團費的方式經營。以致旅行社施壓給香港領隊去鼓勵，甚至強迫內地團友消費，以賺取[[佣金|佣金]]以彌補團費。再加上內地旅行社經常透過「判上判」出團，當內地客抵港後由香港入境接待旅行社接手，已經是第三、四判，令到香港接待旅行社難有利可圖。縱合這些因素，最後導致2006年[[五一黃金周|五一黃金周]]成績強差人意，為香港旅遊業敲響警鐘。

[[2006年8月|2006年8月]]，[[廣州旅行社協會|廣州旅行社協會]]召集了20多間舉辦出境遊的廣州旅行社，簽訂《香港一日遊最低成本價合作備忘錄》，協議香港一日遊的團費不低於98元、不可以向遊客收取零團費、不可以逼使遊客[[購物|購物]]，並且按[[國家旅遊局|國家旅遊局]]規定，一日只可以到一個購物點，時間不可以逾45分鐘，從而打擊有損旅客利益的[[零團費旅行團|零團費旅行團]]。

2010年5月底，劏客事件再現，間接導致大陸遊客〈前[[乒乓球|乒乓球]]國手〉[[陳佑銘|陳佑銘]][[失救|失救]][[死亡|死亡]]。事源陳佑銘參加了[[旅行團|旅行團]]到香港，懷疑被香港[[導遊|導遊]]強迫購物，並且發生了[[衝突|爭執]]，陳佑銘最後[[猝死|猝死]]。事後，[[香港旅遊業議會|香港旅遊業議會]]指出該女導遊是[[黑工|黑工]]，[[行使假文件|行使假文件]]任職。<ref>[http://hk.news.yahoo.com/article/100602/18/idoo.html 內地旅客疑同假導遊爭執後猝死警方跟進] {{webarchive|url=https://web.archive.org/web/20100606004014/http://hk.news.yahoo.com/article/100602/18/idoo.html |date=2010-06-06 }}</ref>[[香港立法會議員|香港立法會議員]]認為，事件顯示香港旅遊業議會無能力監管香港旅行社，業界自律機制並不可行，建議由[[香港政府|香港政府]]直接監管。<ref>[http://hk.news.yahoo.com/article/100602/18/idnx.html 旅發局指內地旅客猝死事件非常嚴重及不可接受] {{webarchive|url=https://web.archive.org/web/20100606031105/http://hk.news.yahoo.com/article/100602/18/idnx.html |date=2010-06-06 }},雅虎香港</ref>此外，亦偶爾出現假貨問題。被批評影響旅客對香港貨的信心。<ref>{{cite news |url=http://news.bbc.co.uk/chinese/simp/hi/newsid_6530000/newsid_6534300/6534377.stm |title=香港海關續打假 內地遊客紛退貨 |agency=BBC中文網新聞 |date=2007-04-07}}</ref>
