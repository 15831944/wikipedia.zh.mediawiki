'''香港漫畫星光大道'''（[[英文|英文]]：'''Hong Kong Avenue of Comic Stars'''）位於[[香港|香港]][[九龍|九龍]][[油尖旺區|油尖旺區]][[尖沙咀|尖沙咀]][[九龍公園|九龍公園]]功夫閣和藝趣坊之間，長約百米，為世界上首條以[[漫畫|漫畫]]為主題的[[星光大道|星光大道]]。香港漫畫星光大道上[[香港漫畫家|香港漫畫家]]的[[手印|手印]]，星光大道兩旁有24座巨型的[[香港漫畫|香港漫畫]]經典角色的彩繪[[雕塑|雕塑]]。此外，設立有漫畫歷史及發展廊及漫畫教育廊，展示香港漫畫發展歷史、漫畫製作過程、[[工具|工具]]、[[漫畫家|漫畫家]]工作室情景及漫畫[[原稿|原稿]]等等。香港漫畫星光大道於2012年9月27日開幕，初步的營業運作時期為3年，目標為以營造[[香港社會|香港社會]]上的[[創意|創意]][[氣氛|氣氛]]，向香港及海外推廣香港漫畫，並且擴大香港漫畫的市場的規模。

主辦[[單位|單位]]──[[香港動漫畫聯會|香港動漫畫聯會]]，每年都會在香港漫畫星光大道舉辦大型活動，並且定期推出特別為到[[學生|學生]]和[[遊客|遊客]]而設立的導賞團及[[工作坊|工作坊]]。

== 歷史 ==
香港漫畫星光大道的建造籌備了逾兩年時間，由香港動漫畫聯會主要辦理，由[[香港生產力促進局|香港生產力促進局]]、[[香港數碼娛樂業支援中心|香港數碼娛樂業支援中心]]、[[康樂及文化事務署|康樂及文化事務署]]及[[油尖旺區議會|油尖旺區議會]][[文化藝術活動統籌委員會|文化藝術活動統籌委員會]]協助辦理。香港漫畫星光大道獲得[[香港政府|香港政府]]的[[創意香港|創意香港]][[資助|資助]]5,372,800[[港元|港元]]興建<ref>[http://www.hkpc.org/index.php?option=com_content&view=article&id=3821%3A2012-05-22-03-47-10&catid=149%3Anews-flash&Itemid=319&lang=zh_TW 「香港漫畫星光大道」今夏揭幕 推廣本地漫畫文化]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }} 香港生產力促進局 2012年5月22日</ref>。此外，香港動漫畫聯會亦有發展[[手提電話|手提電話]][[應用程式|應用程式]]，讓參觀人士能夠進一步了解香港漫畫星光大道所展示的漫畫角色及背景等[[資料|資料]]<ref>[https://archive.is/20130105155803/hk.news.yahoo.com/%E6%B8%AF%E7%94%A2%E6%BC%AB%E7%95%AB%E5%B7%A8%E6%98%9F%E4%B9%9D%E9%BE%8D%E5%85%AC%E5%9C%92%E7%8F%BE%E8%BA%AB-214935952.html 港產漫畫巨星九龍公園現身] 《明報》 2012年9月29日</ref>。

香港動漫畫聯會[[會長|會長]]黃玉郎表示，[[香港市民|香港市民]]和外國[[旅客|旅客]]都是香港漫畫星光大道的招攬對象，尤其希望更多[[中國大陸|中國大陸]]旅客到此一遊，從而推廣香港漫畫至中國大陸市場。他表示，每座彩繪雕塑均有表達香港特色。黃玉郎舉例子表示，王小虎等動作角色，代表「打不死」精神，反映[[香港人|香港人]]永不言敗；而老夫子就代表昔日香港人的[[阿Q精神|阿Q精神]]等。此外，他表示會於1年後檢討香港漫畫星光大道的運作內容，包括訪客人數等等，長遠希望將此項目永久落戶於[[西九文化區|西九文化區]]區內<ref>[https://archive.is/20130105100829/hk.news.yahoo.com/%E6%BC%AB%E7%95%AB%E6%98%9F%E5%85%89%E5%A4%A7%E9%81%93-%E9%95%B7%E9%81%A0%E7%9B%BC%E5%85%A5%E8%A5%BF%E4%B9%9D-211514228.html 漫畫星光大道 長遠盼入西九] 《明報》 2012年5月22日</ref>。

== 導賞團 ==
[[File:HK_Kln_Park_Hong_Kong_Avenue_of_Comic_Stars_Exhibition_blue_banner_Oct-2012.JPG|thumb]]提供三種[[語言|語言]]的[[導賞|導賞]][[服務|服務]]。]]
香港動漫畫聯會[[理事|理事]][[溫紹倫|溫紹倫]]表示，首期學生導賞團由2012年10月至12月，共50團導賞，已經被50所小學及中學訂定了，下一輪需要等待至2013年3月至6月。至於特別為到遊客而設立的導賞團，遊客可以於2012年10月開始於現場即時報名，設有[[粵語|粵語]]、[[普通話|普通話]]及[[英語|英語]]導賞<ref name=singtao></ref>。

== 藝趣坊 ==
逢[[星期日|星期日]]及[[公眾假期|公眾假期]]，香港漫畫星光大道旁邊設有由[[康樂及文化事務署|康樂及文化事務署]]設立的藝趣坊，專門售賣[[手工藝品|手工藝品]]及香港漫畫的主題產品，以及提供[[攝影|攝影]]、[[繪畫|繪畫]]、[[漫畫|漫畫]]、[[書法|書法]]及[[人像素描|人像素描]]等[[藝術|藝術]]服務<ref>[http://www.youth.gov.hk/apps/tc/EventView.do?id=E12082400044 香港漫畫星光大道]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。 

== 週年活動 ==
香港動漫畫聯會表示，為期3年內，他們每年會至少於香港漫畫星光大道舉行一次大型的週年活動<ref>[http://hk.news.yahoo.com/漫畫星光大道開幕-070100437.html 漫畫星光大道開幕] 《星島日報》 2012年9月28日</ref>，包括考慮於2013年7月，即[[動漫節|動漫節]]期間作出相應配合的活動。

== 維修 ==
香港動漫畫聯會理事溫紹倫指出，單位有預算經費為到彩繪雕塑進行定期的修維及翻新，如果發現有彩繪雕塑破損或者殘舊，單位都會即時搶救維修<ref name=singtao>[http://hk.news.yahoo.com/漫畫星光大道-今開幕-221249949.html 「漫畫星光大道」今開幕] 《星島日報》 2012年9月28日</ref>。
