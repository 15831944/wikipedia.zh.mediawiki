'''重光紀念日'''（{{lang-en|'''Liberation Day'''}}）是紀念[[同盟國_(第二次世界大戰)|盟軍]]在[[第二次世界大戰|第二次世界大戰]]中擊敗[[大日本帝國|日本]]取得勝利，結束長達三年零八個月的[[香港日佔時期|香港日佔時期]]，恢復由英國統治的日子。紀念日曾經[[香港公眾假期|香港公眾假期]]之一，定於每年8月最後的星期一及之前的星期六<ref>{{cite web |url= https://www.elegislation.gov.hk/hk/cap149!zh-Hant-hk/sch0|title= 香港法例第149章 《公眾假期條例》 附表|date= 1997年6月30日|website= |publisher= |author=|access-date= |quote=(n) 8月最後一個星期一之前的星期六；(o) 重光紀念日，即8月最後一個星期一；}}</ref>。

== 歷史背景 ==
=== 香港淪陷及重光 ===
1941年12月8日，[[日本帝國|日本帝國]]發動[[太平洋戰爭|太平洋戰爭]]，[[日軍|日軍]]突襲美國海軍基地[[珍珠港事件|珍珠港]]數小時後執行[[南方作战|南方作战]]攻打東南亞，日軍入侵香港，[[香港保衛戰|香港保衛戰]]爆發。

1941年12月25日，經歷18天激戰後，香港守軍在[[香港島|香港島]]的英軍最後防線逐漸瓦解，[[香港總督|香港總督]][[楊慕琦|楊慕琦]]決定投降，香港淪陷進入[[香港日佔時期|日佔時期]]。

1945年8月15日，[[日本廣播協會|日本廣播協會]]播放[[昭和天皇|昭和天皇]]預錄的[[終戰詔書|終戰詔書]]，日本接受中英美蘇發出的[[波茨坦公告|波茨坦公告]]並無條件投降，第二次世界大戰结束亦代表長達三年零八個月的[[香港日佔時期|香港日佔時期]]終告結束，在同盟國及日本政府代表於9月2日簽署[[日本投降書|日本投降書]]前，由[[英國皇家海軍|英國皇家海軍]][[夏慤|夏慤]]少將指揮的艦隊於8月30日抵達[[維多利亞港|維多利亞港]]，宣告[[香港重光|香港重光]]，英國恢復在香港的統治。9月16日，夏愨少將代表[[中國戰區|中國戰區]]及[[英國|英國]]接受日本陸軍岡田梅吉少將和海軍藤田类太郎中將投降，並在中國代表潘華國少將、美國代表威廉臣上校及加拿大代表凱氏上校見證下，於[[香港總督府|香港總督府]]簽署香港的受降文件。

1946年，[[港英政府|港英政府]]將8月30日定為「重光紀念日」（曾譯作'''光復紀念日'''），並列入[[香港公眾假期|香港公眾假期]]，以紀念[[英國|英國]]於1945年8月30日正式重新管治香港，並悼念在[[香港保衞戰|香港保衞戰]]為香港捐軀的將士。1968年，紀念日改定於8月第一個星期一及8月最後的星期一。1983年，紀念日改定於8月最後的星期一之前一個星期六、及8月最後的星期一。

=== 主權移交後的安排 ===
在1997年7月1日[[香港主權移交|香港主權移交]]給[[中華人民共和國|中華人民共和國]]之前，[[香港特別行政區籌備委員會|香港特別行政區籌備委員會]]已在1997年4月，安排在7月1日才運作的[[臨時立法會|臨時立法會]]暫停重光紀念日及其他與英國皇室相關的公眾假期<ref>{{cite web |title=假日（1997年及1998 年）條例 |url=https://www.legco.gov.hk/yr97-98/chinese/bills/bills01/cbills01.htm |publisher=臨時立法會}}</ref>。

香港主權移交後，重光紀念日公眾假期取消，新增性質類似的[[抗日戰爭勝利紀念日|抗日戰爭勝利紀念日]]為公眾假期，設於8月第3個星期一，但只實行了1997年及1998年兩年。

1998年9月9日，通過《1998年假期（修訂）條例草案》包含廢除[[:抗日戰爭勝利紀念日|抗日戰爭勝利紀念日]]公眾假期（即是1997年7月1日前稱的「香港重光紀念日」公眾假期）<ref>{{cite web|url=http://www.legco.gov.hk/yr98-99/chinese/counmtg/hansard/980909fc.pdf|title=會議過程正式紀錄P.63-88|publisher=香港立法會|date=1998-09-09}}</ref><ref>{{cite web |title=《1998年假期(修訂)條例草案》委員會報告條例草案》委員會報告 |url=https://www.legco.gov.hk/yr98-99/chinese/bc/bc51/reports/b109c.pdf |publisher=香港立法會}}</ref>。

1999年，政府新增[[國際勞動節|勞動節]]和[[佛誕|佛誕]]2天公眾假期，為免公眾假期太多而造成經濟衝擊，重光紀念日不再是公眾假期，該法案的主要目的是由1999 年起，以勞動節（5月1日）和佛誕（農曆四月八日）取代1997年及1998年的其中兩天公眾假期，即抗日戰爭勝利紀念日（8月第三個星期一）及10月2日（國慶日翌日），重光紀念日名義上與重陽節「合併」，並改於每年11月第2個星期日的[[和平紀念日_(香港)|和平紀念日]]，在[[中環|中環]][[皇后像廣場|皇后像廣場]]的[[和平紀念碑_(香港)|和平紀念碑]]舉行紀念活動，並與[[第一次世界大戰|第一次世界大戰]]結束同時紀念，而紀念「為保衞香港而捐軀之人士」的儀式改在[[重陽節|重陽節]]舉行。

== 儀式 ==
重光紀念日紀念儀式隆重，現役與退伍軍人會參與，政府部門、英聯邦成員國、軍隊代表分別獻花圈，之後吹[[號角|號角]]及[[默哀|默哀]]兩分鐘。

現時，重光紀念儀式由義勇軍會及第二次世界大戰退伍軍人會負責，在最近8月15日（日本宣佈投降）的星期日舉行，有獻花，無鳴槍。政府則在[[重陽節|重陽節]]在[[香港大會堂紀念花園|香港大會堂紀念花園]]紀念為香港戰死的人，有獻花及鳴槍<ref>{{cite news |title=我們丟失的　不只是一天假期 遺忘了的重光紀念日 |url=https://hk.news.appledaily.com/local/daily/article/20120826/18000529 |work=蘋果日報 |date=2012-08-26}}</ref>。

== 其他紀念日 ==
1945年9月2日，在包括[[中華民國|中華民國]]在內的[[同盟國_(第二次世界大戰)|同盟國]]代表見證下，日本簽署[[降伏文書|降伏文書]]，而翌日（9月3日）被[[中華民國政府|中華民國政府]]定為[[第二次世界大战对日战争胜利纪念日|對日戰爭勝利紀念日]]，1955年起作為[[軍人節|軍人節]]。[[中華人民共和國|中華人民共和國]]將9月3日定為抗日戰爭勝利紀念日。

2014年，香港特區政府宣佈會於每年9月3日及12月13日舉行官方儀式，分別紀念「中國人民抗日戰爭勝利」及悼念[[南京大屠殺|南京大屠殺]]死難者<ref>{{cite news |date=2014年8月26日 |title=港府突宣布辦抗日紀念活動 跟中央路線 |newspaper=[[信報財經新聞|信報財經新聞]] |url=http://www2.hkej.com/instantnews/market/article/887970/%E6%B8%AF%E5%BA%9C%E7%AA%81%E5%AE%A3%E5%B8%83%E8%BE%A6%E6%8A%97%E6%97%A5%E7%B4%80%E5%BF%B5%E6%B4%BB%E5%8B%95+%E8%B7%9F%E4%B8%AD%E5%A4%AE%E8%B7%AF%E7%B7%9A |accessdate=2015-07-09}}</ref>，但前後兩者概念有別，亦與紀念[[香港日佔時期|香港日佔時期]]結束的「抗日戰爭勝利紀念日」前稱「重光紀念日」，以及紀念兩次大戰結束並悼念戰爭死難者的[[和平紀念日_(香港)|和平紀念日]]性質不同<ref>{{cite news |title=香港重光紀念73年　團體在大會堂紀念公園獻花悼念 |url=https://www.hk01.com/%E7%A4%BE%E6%9C%83%E6%96%B0%E8%81%9E/221742/%E9%A6%99%E6%B8%AF%E9%87%8D%E5%85%89%E7%B4%80%E5%BF%B573%E5%B9%B4-%E5%9C%98%E9%AB%94%E5%9C%A8%E5%A4%A7%E6%9C%83%E5%A0%82%E7%B4%80%E5%BF%B5%E5%85%AC%E5%9C%92%E7%8D%BB%E8%8A%B1%E6%82%BC%E5%BF%B5 |work=香港01 |date=2018-08-12}}</ref>。
