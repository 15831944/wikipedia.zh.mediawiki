'''香港足球總會'''（英文：'''The Hong Kong Football Association Limited'''）是[[香港|香港]]的[[足球|足球]]管理機構，是[[中國香港體育協會暨奧林匹克委員會|中國香港體育協會暨奧林匹克委員會]]轄下74個體育總會及體育組織之一，負責[[香港足球代表隊|香港足球代表隊]]、[[香港奧運足球代表隊|香港奧運足球代表隊]]、其他不同年齡的香港足球代表隊、[[香港五人足球代表隊|香港五人足球代表隊]]及、[[香港聯賽選手隊|香港聯賽選手隊]]的挑選及[[訓練|訓練]]，亦負責各級別[[香港足球聯賽|香港足球聯賽]]的[[足球會|足球隊]]與[[足球|足球]][[運動員|運動員]]的[[註冊|註冊]]、青少年足球運動員訓練及[[發展|發展]]、[[教練|教練]]訓練及發展、[[裁判員|裁判員]]及[[紀律|紀律]]等事宜。

== 宗旨 ==
香港足球總會的宗旨是「推動及發展[[香港足球|香港足球]]，包括職業足球、青少年足球及業餘足球，從而提高香港足球的形象和世界排名，為[[香港市民|香港市民]]提供高質素的足球賽事，令足球成為[[香港|香港]]最受歡迎的[[體育|體育]][[運動|運動]]。」

== 歷史 ==
香港足球總會於1914年成立，起初稱為'''香港腳球總會'''<small><ref>[http://www.lcsd.gov.hk/b5/ppr_release_det.php?pd=20060620&ps=05 康樂及文化事務署新聞公報-歷史博物館介紹足球的歷史與發展] {{webarchive|url=https://web.archive.org/web/20150402150108/http://www.lcsd.gov.hk/b5/ppr_release_det.php?pd=20060620&ps=05 |date=2015-04-02 }} </ref></small>，其實香港足球總會於1908年底已開始組成，惟運作了一年會務便陷於停頓，到了1913年才再重組，於1914年獲得[[英格蘭足球總會|英格蘭足球總會]]正式承認為屬會，故此便以1914年作為正式成立年份。

成立初期，行政和管理人員全屬義務性質。其後租用[[中環|中環]]廣東銀行大廈4字樓55號半間寫字樓作辦公室，而另一半為印務館租用。

[[第二次世界大戰|第二次世界大戰]]後，足球總會在中環[[太子大廈|太子行]]2字樓211室租用了一間約500多平方呎辦公室，義務秘書改為受薪聘用，加添兩名職員協助秘書的工作。

在1954年，香港足球總會註冊成為[[有限公司|有限公司]]。同年，相繼加入[[國際足球協會|國際足球協會]]及[[亞洲足球協會|亞洲足球協會]]成為會員，為亞洲足球協會12位創會成員之一，時任會長[[羅文錦|羅文錦]]擔任[[亞洲足協|亞洲足協]]首任會長。為了進一步推廣足球，足球總會租用位於[[跑馬地|跑馬地]][[體育路|體育路]]剛擴建完成的[[香港足球會球場|香港足球會球場]]，辦公室設於球場一號看台下的地庫，租期為15年。

在1972年，香港足球總會向[[港英政府|政府]]申請撥地興建永久會所。於1974年獲得通過，獲得一幅位於[[何文田|何文田]][[佛光街|佛光街]]的土地。經過5年籌建，會所於1979年9月10日正式啟用，揭幕禮由[[國際足球協會|國際足球協會]]會長[[若昂·阿维兰热|夏維蘭治]]主持。會所樓高3層，面積19,000平方呎，並且可舉行大型會議，一直沿用至今。2005年，獲得國際足球協會「GOAL Project」資助，於同年進行大規模的重修工程，於2006年1月28日舉行揭幕啟用禮。

在2014年2月12日，足球總會[[行政總裁|行政總裁]][[薛基輔|薛基輔]]率領香足球總會代表團到訪[[韓國|韓國]]取經，參觀當地的訓練中心及[[經典K聯賽|經典K聯賽]][[足球會|足球會]]。[[香港足球代表隊|香港足球代表隊]]主教練[[金判坤|金判坤]]透露，此舉將會就聯賽、青年訓練及發展等不同範疇取經，為於明年啟辦的[[香港職業足球聯賽|香港職業足球聯賽]]準備。

== SMART足球計劃 ==
香港足球總會於2009年11月26日，與[[香港賽馬會|香港賽馬會]]運動醫學及健康科學中心簽訂為期兩年的「SMART足球計劃」合作備忘錄，中心將為各級代表隊 (包括：[[香港足球代表隊|香港足球代表隊]]、[[香港奧運足球代表隊|香港奧運隊]]、5人代表隊及各級青年代表隊)提供包括季前身體檢查、體適能測試、運動創傷診斷及治療、體適能訓練、預訪運動創傷訓練等運動醫學及科學服務。中心將為香港足球總會建立詳細的球員資料庫，紀錄球員生理及病歷資料。與此同時，中心亦會派出球隊醫生、物理治療師及體適能專家隨隊參與本地及海外賽事。

[[香港中文大學醫學院|香港中文大學醫學院]]香港賽馬會運動醫學及健康科學中心執行總監[[容樹恒|容樹恒]]醫生表示：「SMART除有醒目的含意外，也是運動醫學及康復治療（Sports Medicine And Rehabilitation Therapy）的英文縮寫；這正是香港足球一直所缺乏的專業支援。我們除在醫療和體適能方面提供專業支援外，也會和足總共同進行運動醫學及運動科學研究，並為足球界人士舉辦各類學術活動，以推廣運動醫學及運動科學。」

中心將會整合所有測試數據及檢查結果，協助足總成立一個詳細的球員數據庫，當中記錄球員個人生理、體能及病歷資料。容樹恒醫生補充：「建立數據庫是『SMART足球計劃』重點項目。中心將根據所收集的數據，分析出香港球員獨有的體能特點，從而研發更新更有效的體能訓練方法。另外，分析結果亦有助醫護人員制定治療及復康計劃，中心更希望藉此提升足球界人士對球員健康的重視程度，帶領香港球壇走向專業化。」礙於本港健體場地及設施不足，球壇一直缺乏合適且有效的力量訓練系統。中心正就「SMART足球計劃」與足總教練商討，制定一套適合本港地理環境及球員體能的場內力量訓練系統，好讓球員不受場地及設施所限，能於場內隨時進行針對性的力量訓練。

足球科學化是整個球壇的大趨勢，為達致這個目標，足總的角色尤其重要。往後足總及中心將會定期合作，為教練、體適能專家、醫生及物理治療師等專業人士舉辦講座及工作坊，藉以推廣足球運動醫學及運動學知識。此外，足總球員更會參與中心進行的預防運動創傷臨床研究，共同為香港科研而努力。是次簽約儀式由香港足球總會副主席貝鈞奇先生及香港中文大學醫學院香港賽馬會運動醫學及健康科學中心執行總監容樹恒醫生主禮。大會亦邀請東亞運代表隊成員於簽約儀式後示範中心所提供的體適能測試。
