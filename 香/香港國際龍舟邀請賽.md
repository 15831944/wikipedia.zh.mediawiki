'''香港國際龍舟邀請賽'''（Hong Kong International Dragon Boat Regatta），是在香港舉行的國際性[[賽龍舟|賽龍舟]]活動，於1976年首次舉行。賽事於每年6月[[端午節|端午節]]後之星期六、日舉行，是同類最早的國際性[[龍舟|龍舟]]活動及最具代表性的比賽之一。

==歷史==
1976年首次舉行香港國際龍舟邀請賽，由[[香港旅遊協會|香港旅遊協會]]主辦，於[[筲箕灣|筲箕灣]][[避風塘|避風塘]]角逐邀請賽錦標，除了9支香港漁民[[龍舟|龍舟]]隊外，[[日本|日本]][[長崎|長崎]]的漁民隊亦有組隊來港參加。賽事於1978年開始在[[尖沙咀東部|尖沙咀東部]]對開的[[維多利亞港|維多利亞港]]舉行，吸引更多市民及遊客參觀。於1985年慶祝十週年，賽事分為兩日舉行，同年亦首次舉行國際女子錦標賽。1996年由於尖東海旁進行工程，移師[[沙田|沙田]][[城門河|城門河]]舉行。

2000年[[香港旅遊協會|香港旅遊協會]]退出主辦香港國際龍舟邀請賽，由[[中國香港龍舟總會|中國香港龍舟總會]]獨力主辦。

2002年為慶祝[[香港特別行政區|香港特別行政區]]成立五周年，龍舟邀請賽被列為慶祝活動之一，首次於[[中環|中環]]／[[灣仔|灣仔]]海旁舉行。

2003年香港受[[嚴重急性呼吸系統綜合症|沙士]]侵襲而停辦一屆。

2004年再假沙田城門河舉行香港國際龍舟邀請賽，由於經費不足，導致需要延遲舉行，其後得有心人支持，最後仍可延期一個月續辦。但因時間緊迫而未能邀請美、加隊伍參賽，全靠[[中國內地|中國內地]]的強隊包括[[天津|天津]]、[[上海|上海]]、[[廣州|廣州]]、[[番禺|番禺]]、[[南海区|南海]]、[[荊州|荊州]]等和[[菲律賓|菲律賓]]的隊伍參與是次賽事<small><ref>[http://paper.wenweipo.com/2004/08/01/HK0408010035.htm 國際龍舟賽今演重頭戲]</ref></small>。

2005年再次回歸[[尖沙咀海旁|尖沙咀海旁]]舉行香港國際龍舟邀請賽。

2006年返回沙田城門河舉行，由於主辦單位出現安排混亂、場地欠佳、宣傳不足及缺乏溝通等而被狠批「搞得一年差過一年」<small><ref>[http://the-sun.on.cc/channels/news/20060605/20060605034432_0000.html 安排混亂 場面冷清 一年差過一年 國際龍舟賽失禮香港]</ref></small>。

2007年香港國際龍舟邀請賽被列入慶祝[[香港主權移交十週年|香港特別行政區成立十週年]]活動之一，於2007年6月23日及24日於尖東海旁舉行，賽道只有300米。國際賽事之各項錦標賽名額為48隊，除海外隊伍當然進入國際錦標賽外，餘額由本地賽隊組合而成。賽事分金盃賽及銀盃賽兩組進行，大會根據各隊伍的入圍時間排名，並分為兩組，每組24支隊伍。24支時間最快的隊伍參加第一組賽事（國際錦標賽金盃賽），其次的24支隊伍將參加第二組賽事（國際錦標賽銀盃賽）。[[澳門|澳門]]男子隊及[[南海区|南海]][[西樵鎮|西樵]]女子隊分別奪得國際龍舟邀請賽公開錦標賽及女子錦標賽金盃賽冠軍。

2016年5月宣佈，香港國際龍舟邀請賽移師到中環新海濱舉行<ref>[http://hk.on.cc/hk/bkn/cnt/news/20160524/bkn-20160524161208606-0524_00822_001.html 龍舟嘉年華首移師中環海濱 賽道縮短100米]，東方報業集團網站，2016年5月24日</ref><ref>[http://orientaldaily.on.cc/cnt/news/20160525/00176_013.html?pubdate=20160525 龍舟嘉年華首移師中環海濱]，東方日報，2016年5月25日</ref><ref>[http://hk.apple.nextmedia.com/realtime/news/20160524/55141229 龍舟嘉年華移師中環海濱 賽道縮短避填海工程]，蘋果日報，2016年5月24日</ref>。

==2008香港國際龍舟邀請賽==
2008香港國際龍舟邀請賽由[[中國香港龍舟總會|中國香港龍舟總會]]主辦，[[康樂及文化事務署|康樂及文化事務署]]資助，將於2008年6月14日及15日於[[沙田|沙田]][[石門_(香港)|石門]]安景街香港龍舟協會訓練中心城門河舉行。龍舟比賽賽程為500米，每艘龍舟最多可載最多20名或不少於16名-{划}-手，另有鼓手及舵手各1名，賽事分為：

# 公開錦標賽
# 女子錦標賽
# 混合錦標賽
# 漁民錦標賽
# 紀律部隊錦標賽
# 商行錦標賽
# 專業團體錦標賽
# 大專錦標賽
# 青少年錦標賽
# 壯年錦標賽
# 本地團體挑戰盃（贊助賽事）
# 傳媒邀請賽（小龍賽）

;國際賽事

# 國際女子錦標賽
# 國際混合錦標賽
