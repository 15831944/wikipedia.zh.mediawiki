《'''香港城邦論'''》，是一本於2011年出版的暢銷書<ref>{{cite news|title=城邦論民族論書籍熱賣 |url=http://hk.apple.nextmedia.com/news/art/20150308/19068007 |agency=[[蘋果日報_(香港)|蘋果日報]] |date=2016年3月8日}}{{zh-hant}}</ref>，由[[香港|香港]]學者[[陳雲_(學者)|陳雲]]整理[[Facebook|Facebook]]之貼文、留言分為八章及由學生黎志恆整理第一、三、七、八章之文稿，再由陳雲增刪修訂及自負文責<ref name ="a1"/>。

本書主要針對自2010年以來的[[中港矛盾|中港矛盾]]問題，探討香港將來[[本土|本土]]政治應該採取的發展方向。書中指香港目前應採取中港區隔的措施，捍衛本土利益，維護自英治時代留下的典章制度，最終達致與中國政府互惠互利，甚至改革中國政治體制，使之逐漸[[民主化|民主化]]的效果<ref name="香港城邦論I">{{cite book |author=[[陳雲|陳雲]] |coauthors= |title=香港城邦論 |year=2011 |publisher=天窗出版社有限公司 |location= |isbn= }}</ref> {{rp|55}}。

== 出版經過 ==

作者於前言中表示本不想做拯救[[香港|香港]]之事，以一人敵一國，幾乎令其家破人亡。奈何2010年底，香港政府參與[[環珠江口宜居灣區建設重點行動計劃|宜居灣區規劃]]，向[[廣東|廣東]]割讓香港[[自治權|自治權]]，經[[Facebook|Facebook]]朋友勸告再三，才撰文反對。2011年2月9日，陳雲率先於Facebook發文，後寫成文章，題為「香港城邦 中國門戶」，2011年3月22日於《[[am730|am730]]》專欄刊出，以[[歐洲|歐洲]]城邦觀念，重認香港的城邦身份。<ref name ="a1"/>

刊登之後，陳雲[Facebook朋友成立群組，探討香港城邦自治之事，[[天窗出版社|天窗出版社]]約陳雲寫書。Facebook討論6個月之後，於2011年6月中旬草擬《香港城邦自治總綱》，於在7月1日的遊行影印若干份派發。於同年8月整理Facebook之貼文及留言，分為八章，9月動筆撰寫，並聘請學生[[黎志恆|黎志恆]]君協助根據Facebook之貼文及留言，整理第一、三、七、八章的文稿，事後陳雲再增刪修訂，文責由陳雲自負<ref name ="a1"/>。

== 內容 ==

作者陳雲在書中闡述道，香港自從[[香港主權移交|香港主權移交]]（香港回歸）後，社會在政治上出現兩大迷思，港人往往認為香港必需以一個憲政民主的中國為依歸<ref name="香港城邦論I"/>{{rp|20}}，而且需在中國民主化後香港才能有民主。他斥責這種思想不實際，並指出這種思想是香港民主派政黨於英殖時代遺留下來的。當時，民主派政黨以中國大一統意識與英殖政府角力，本來沒有問題。然而在[[香港主權移交|香港主權移交]]後抱持這種思維，政黨往往就會為保全中國大陸的利益而犧牲香港的本土利益<ref name="香港城邦論I"/>{{rp|24-25}}；而自六四事件後，香港民主派提出「民主統一論」，期待中國能逐漸民主化，卻無視中共本身無意民主化、甚至干擾香港民主進程的事實<ref name="香港城邦論I"/>{{rp|41-42}}，而且民主派爭取中國民主的方法，只是每年[[六四燭光晚會|六四燭光晚會]]不斷重覆平反六四的口號，缺乏實質效力<ref name="香港城邦論I"/>{{rp|53}}，就算真正投入改進中國民主，最終只會耗費香港大量人力物力，徒勞無功<ref name="香港城邦論I"/>{{rp|21}}。

陳雲在書中列舉了部分[[大陸人|中國大陸人]]的劣行<ref name="香港城邦論I"/>{{rp|43-46}}，又推斷中國倘若急速民主化，由於「國際屈辱、受害者情意結、生存空間緊迫、喪失領土的憤恨、對國際道義的不信任、工業紀律社會、[[中產階級|中產階級]]的心靈狹窄、大企業家的愛國衝動、企業產力迅速增長但大量青年就業困難」九大因素，容易成為如[[納粹主義|納粹主義]]的極權主義的溫床。<ref name="香港城邦論I"/>{{rp|51-52}}他以此勸告香港人不要插手大陸事務，並探討香港族群意識，指出不少從中國大陸來香港的新移民不服膺[[香港核心價值|香港核心價值]]，又佔用香港公共資源，認為香港應有權審批大陸來港移民。<ref name="香港城邦論I"/>{{rp|49-50}}

作者亦認為香港地位特殊，根據[[基本法|基本法]]，香港除了國防及外交由中國負責外，內政一律自行管理，享有充份自治權，仿如歐洲古代的[[城邦|城邦]]。所謂城邦，陳雲將之定義為「以城市為核心範圍的自治體，有時是主權獨立，但更多時候是託庇於一個主權體制下…至少在名義上受到王族或[[帝國|帝國]]的外交保護及外交領導<ref name="香港城邦論I"/>{{rp|67}}。」，並形容城邦為「人才、資金、財貨、文化風俗和學術知識的聚散地<ref name="香港城邦論I"/>{{rp|62}}」「在國族建立的過程中，城邦充當政治動員、資金援助、文化薰陶、人才輸送和制度奠定之功能。<ref name="香港城邦論I"/>{{rp|66}}」他認為香港正正繼承了城邦開放共融的精神，加上經濟多元，又擁有強大之文化保存及傳播力量<ref name="香港城邦論I"/>{{rp|78}}，容許市民參與政治與創造文化<ref name="香港城邦論I"/>{{rp|93}}，足以成為中西交流的橋頭堡。陳雲亦在此提到中國大陸、[[台灣|台灣]]、香港及[[澳門|澳門]]締結為「[[中華聯邦主義|華夏邦聯]]」的構想。<ref name="香港城邦論I"/>{{rp|79}}

*美國加州州立大學哲學教授[[王偉雄|王偉雄]]指出《香港城邦論》主要是一連串的政治文宣 ，欠缺嚴謹論證，部份重要聲稱的資料來源闕如，明顯不是學術著作。他評價道：「城邦論也許能滿足一些人的政治幻想，而對於提高香港人的本土意識和開展這方面的討論，此書也的確功不可沒，可是，若依照書中提出的路線來追求香港本土的長遠利益，恐怕會是死路一條。」王偉雄亦認為，《香港城邦論》是與中共勾結，並且對中共都有太大信心，所以王偉雄指出「與其（按照陳雲所說的消極計畫）繼續靜待下去而最終「得個吉」，不如現在就主動爭取，齊心合力，審慎樂觀而努力不懈，還有成功的希望；總好過將來悔恨當年太天真太傻，錯信中共，以致不但害了自己，還對不起下一代。」王偉雄也由此得出，[[本土派_(香港)|本土派]]和城邦派是不同：「反對城邦論，不是反對由本土開始爭取民主，而只是反對與中共勾結。」<ref>{{cite web|title=總評《香港城邦論》兼澄清孔誥烽教授之誤會|url=http://fishandhappiness.blogspot.hk/2013/07/blog-post.html|agency=魚之樂|accessdate=2013-07-01|accessdate=2019-02-24}}</ref>

*香港人文學會會長[[劉桂標|劉桂標]]認為香港城邦論是泛民主派一嶄新而有其獨立貢獻的論述，使泛民的政治光譜進一步擴闊。他一方面認同《香港城邦論》指出中港兩地急促融合是對本地民主的一種嚴重而緊急的危害，一義下可說拉?了本地泛民的民主抗爭的層面；而指出香港作為城邦的現實，為香港民主發展提供歷史和法理上的依據，可說是對本地民主發展有很大的貢獻。但另一方面[[劉桂標|劉桂標]]認為《香港城邦論》的一大敗筆在於批判泛民的民主建國論，因為中國民主與香港民主應是相輔相成而非互相衡突；而且[[陳雲|陳雲]]以偏蓋全地歸納論證中國人民的"賊性"，是欠缺充分理據的負面道德判斷，不尊重人的人格尊嚴，是對人的一種歧視。 <ref>{{cite web|title=陳雲著《香港城邦論》述評|url=http://www.hkshp.org/mhcomment/2012-07-17lau_kwai_piu.htm|agency=香港人文學會網頁|accessdate=2016-05-22}}</ref>

*台灣社會學家[[吳叡人|吳叡人]]在為[[香港民族論|香港民族論]]撰文時，認為香港城邦自治論是一種弱小民族[[民族主義|民族主義]]的變形，當中預設了一個香港的公民共同體的存在。整個論證是在不挑戰，乃至接受現代主權民族國家普遍秩序的前提下，有意識地選擇以不完整主權之前現代非民族國家政治形式作為香港逃避大國兼併吸收的保護殼。他指[[陳雲|陳雲]]迴避民族主義概念，借用{{Link-en|公民共和主義|civic republicanism}}的語言來證成香港主體性與高度自治，其公民共和主義建基於奇妙（某種意義上謬誤）的歷史論證<ref name = lilliputian>{{Citation| last=WU| first=RWEI-REN| year=2016| title=The Lilliputian dreams: preliminary observations of nationalism in Okinawa, Taiwan and Hong Kong| periodical=Nations and Nationalism| volume=| issue=| pages=13| url=http://onlinelibrary.wiley.com/doi/10.1111/nana.12251/abstract| doi=10.1111/nana.12251| oclc=}}</ref>：香港是城邦（polis or city-state）－－也就是公民共和主義思想的起源領域－－的一種當代形式。吳叡人認為陳雲將香港（與新加坡等）作為殖民地貿易港市的歷史連結到現代主權國家體系興起前的古典/中世紀城邦國家與歐陸自由市的傳統之上，「發明」了一個從古希臘、文藝復興到現代「殖民城邦」的城邦系譜，從而將古典城邦與自由市才擁有的自治傳統順帶挪用(appropriated)到事實上缺乏自治傳統的城市殖民地香港之上，以這個想像的城邦自治傳統作為香港自治的論據，也就是香港自治的正當性基礎不是香港民族的自治權，而是香港城邦的歷史與其固有的自治特質。在這個歷史論證之上，陳雲再將原本被公民共和主義視為共和政體存在前提或存續條件的{{Link-en|公民德行|civic virtue}}轉化為界定香港認同的準則，於是香港人或香港市民成為一種基於價值而非血緣而形成的政治範疇。陳雲宣稱他的主張是一種[[現實政治|現實政治]]的論證，因為他不支持[[香港獨立|香港獨立]]，而且他的香港自治目的在於防衛與保護香港固有的主體性，不在挑戰中國主權或介入中國政治。[[吳叡人|吳叡人]]因此認為陳雲之所以拒絕使用民族主義概念也是出於現實主義之考量，以避免與中國官方民族主義直接衝突。不過吳氏亦批評這個主張充滿夢想成份，因為陳雲所發明的香港城邦自治傳統，其實是一種尚未實現的未來願景，香港人的香港認同離古典共和主義預設的高門檻城邦認同條件仍有一段距離。<ref name="香港民族論">{{cite book |author=	2013年度香港大學學生會學苑 |title=香港民族論 |year=2015 |publisher=香港大學學生會 |isbn=9789881363107 }}</ref>{{rp|87-89}}吳氏亦提醒讀者，指陳雲的共和理論由文化理論支持。其文化理論將香港人描述為傳統華夏文化的真正傳人，而中國人則在中國共產黨的統治下受[[共產主義|共產主義]]及[[資本主義|資本主義]]的污染，因而退化為蠻夷。事實上，這種看似弔詭的論點是常見策略，文化上被同化的邊緣民族往往試圖以此將自身與衰落的文化中心區別開來。典型例子包括於滿清滅明後日本的[[華夷變態|華夷變態]]論以及朝鮮的[[小中華思想|小中華思想]]。另外一些例子包括[[愛國者_(美國革命)|美國革命者]]。他們追求獨立是出於對真正英國自由價值的支持。<ref name="lilliputian"/>{{rp|13}}
