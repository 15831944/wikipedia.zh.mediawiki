'''香港共和國'''，或稱'''香港民國'''<ref>http://www.hkfront.org/declarationfile.htm</ref>（[[英語|英語]]：'''Republic of Hong Kong'''），是指[[香港獨立運動|香港獨立運動]]支持者主張[[香港|香港]]脫離[[中華人民共和國|中華人民共和國]]後建立的一個[[主權國家|主權國家]]<ref>{{cite web|url=https://www.hknationalparty.com/|title=香港民族黨官方網站|}}</ref><ref>{{Cite web |url = http://www.localpresshk.com/2016/03/hong-kong-national-party-established/ |title = 香港民族黨成立 主張成立香港共和國|publisher=本土新聞|date=2016-03-28}}</ref><ref>{{cite web|url=https://www.facebook.com/pg/studentlocalism/about/?ref=page_internal|title=學生動源Facebook專頁簡介|}}</ref><ref>{{Cite web |url = http://www.hkfront.org/20180522ch.htm |title = 五星旗下香港的法治危殆 槍桿子裡香港共和國誕生|publisher=我是香港人連線|date=2018-05-22}}</ref>。在香港獨立運動中，主張採取這個[[建國|建國]]路線的人，又被稱為[[港獨派|港獨派]]<ref>{{Cite web |url = https://www.inmediahk.net/node/1058519 |title = 唇亡齒寒，避免切割，本土派民主派旗幟同處飄揚 獨派學生堅拒改變主張|publisher=香港獨立媒體網|date=2018-07-24}}</ref>。

香港現是[[中華人民共和國|中華人民共和國]]的實際統治地域。[[香港共和國|香港共和國]]只屬[[香港獨立運動|香港獨立運動]]的其中一個構想，此政體實際並不存在，也被大多数香港市民反对，认为独立没必要性，也不可行的。
