'''香港特別行政區第五屆政府'''是[[林鄭月娥|林鄭月娥]]出任第五屆[[香港特別行政區行政長官|香港特別行政區行政長官]]後，其領導下的[[香港特別行政區政府|香港特別行政區政府]]，故又稱'''林鄭月娥政府'''，簡稱'''林鄭政府'''。

2017年3月31日，[[中华人民共和国國務院總理|中华人民共和国國務院總理]][[李克強|李克強]]簽署[[中华人民共和国國務院令|國務院令]]任命林鄭月娥為[[香港|香港]]第五任行政長官<ref>{{cite web|title=中华人民共和国国务院令 第678号|url=http://www.gov.cn/zhengce/content/2017-04/01/content_5182820.htm|date=2017-04-01|publisher=中华人民共和国中央人民政府|author=李克强}}</ref>。林鄭月娥于同年4月11日在[[北京|北京]][[中南海|中南海]]拜會李克強，并在其手上接過任命狀。<ref>{{cite news|title=【林鄭當選】國務院任命林鄭月娥為港特首 李克強：團結帶領各界人士|url=http://hk.apple.nextmedia.com/realtime/china/20170331/56506559|date=2017-03-31|newspaper=[[蘋果日報_(香港)|蘋果日報 (香港)]]}}</ref>林鄭月娥於2017年7月1日就任，開始領導香港特別行政區政府部門，以正常時間為根據，該任期會至2022年6月30日為止。

按政鋼原文及林鄭向傳媒透露的內容，其具體施政項目將包括：

; 政治、行政及司法方面<ref name="林鄭月娥競選政綱PDF檔版本"></ref>
*修訂《[[防止賄賂條例|防止賄賂條例]]》，擴大第3及第8條的適用範圍至行政長官在內的人；
*以「[[2016年及2017年香港政治制度改革#.E3.80.8C831.E6.B1.BA.E5.AE.9A.E3.80.8D|八三一]]」框架為前提，「營造有利推動政改的社會氛圍」；
*為[[香港特別行政區基本法第二十三條|《基本法》第23條]]立法，「創造有利立法的社會環境」；
*經廣泛諮詢、[[法律改革委員會|法律改革委員會]]提交報告後，「積極跟進」[[檔案法|檔案法]]的訂立。

; 經濟方面<ref name="林鄭月娥競選政綱PDF檔版本"></ref>*<ref name="林鄭承諾新風格管治班子注新血"></ref>*<ref name="【林鄭當選】林鄭月娥政綱懶人包　助上樓、改稅制、檢視教育制度"></ref>
*提出「稅務新方向」，引入兩級制[[利得稅|利得稅]]，將目前首200萬港元利潤稅率的16.5%調低至10%，餘下的利得稅則維持於16.5%，以減輕[[香港中小企業|中小]]及[[初創企業|初創企業]]的稅務負擔；
*為企業研發引入額外稅務寛免，並同時適用於企業在[[環保|環保]]、[[文化|文化]]、[[藝術|藝術]]及[[設計|設計]]等開支，以鼓勵企業研究及開發。
*研究建立聯結[[中國大陸|中國大陸]][[債券|債券]]市場的「[[債券通|債券通]]」，提升中國大陸貨幣—[[人民幣|人民幣]]的產品的流通性；
*爭取與中國大陸政府簽訂「[[一帶一路|一帶一路]]」全面合作協議，誘使香港[[企業家|企業家]][[投資|投資]]及[[融資|融資]]該計劃，並寄望各[[跨國企業|跨國企業]]可使用香港的[[保險|保險]]及[[風險管理|風險管理]]平台；
*成立由政府高官、各金融監管機構和[[金發局|金發局]]組成、由[[財政司司長|財政司司長]]主持的「金融領導委員會」，以建立商議金融政策的恆常機制。

; 運輸及房屋方面<ref name="林鄭承諾新風格管治班子注新血">{{cite news |language = 中文 |author =  |coauthors =  |url = http://news.now.com/home/local/player?newsId=215485 |title = 林鄭承諾新風格管治班子注新血 |publisher = [[now新聞|now新聞]] |pages =  |date = 2017-03-26 |accessdate = 2017-05-20}}</ref>*<ref name="【林鄭當選】林鄭月娥政綱懶人包　助上樓、改稅制、檢視教育制度">{{cite news |language = 中文 |author =  |coauthors =  |url = https://www.hk01.com/港聞/80467/-%E6%9E%97%E9%84%AD%E7%95%B6%E9%81%B8-%E6%9E%97%E9%84%AD%E6%9C%88%E5%A8%A5%E6%94%BF%E7%B6%B1%E6%87%B6%E4%BA%BA%E5%8C%85-%E5%8A%A9%E4%B8%8A%E6%A8%93-%E6%94%B9%E7%A8%85%E5%88%B6-%E6%AA%A2%E8%A6%96%E6%95%99%E8%82%B2%E5%88%B6%E5%BA%A6 |title = 【林鄭當選】林鄭月娥政綱懶人包 助上樓、改稅制、檢視教育制度 |publisher = [[香港01|香港01]] |pages =  |date = 2017-03-26 |accessdate = 2017-03-26}}</ref>
*增加「[[綠置居|綠置居]]」單位供應量，協助具經濟能力的[[公屋|公屋]]住戶購買居屋，以騰出公屋單位；

*研究可讓仍未[[補地價|補地價]]的[[居屋|居屋]][[業主|業主]]租出其買入卻非居住的丟空單位；
*成立「全面檢視土地來源」的專責小組，探究諸如[[填海|填海]]、[[市區更新|舊區重建]]、[[棕地|棕地]]及[[郊野公園|郊野公園]]的開發等增加土地房屋的方式；
*研究運用政府每年從[[香港鐵路有限公司|香港鐵路有限公司]]所收取約40億[[港元|港元]]的[[股息|股息]]，「補貼」居住在偏遠地區居民的長途車費開資，以減輕基層市民長途車費的負擔；
*要求港鐵公司設計非繁忙時間的折扣優惠，疏導[[繁忙時間|繁忙時間]]大量的鐵路人流。

; 教育方面<ref name="林鄭承諾新風格管治班子注新血"></ref>*<ref name="【林鄭當選】林鄭月娥政綱懶人包　助上樓、改稅制、檢視教育制度"></ref>
*新增約50億港元的教育恆常開支，以處理教育界需要額外資源解決的問題，例如將短期合約教席轉為常額教席、增加[[融合教育|融合]]及[[特殊教育|特殊教育]]的支援、確立[[幼稚園|幼稚園]]教師薪級表等；
*在「政策檢討完成」前暫時擱置小學三年級的[[基本能力評估|基本能力評估]]（英文縮寫BCA）。

=== 當選 ===
2017年3月26日，林鄭在[[2017年香港特別行政區行政長官選舉|2017年香港特別行政區行政長官選舉]]中當選後即發表勝選[[宣言|宣言]]，指自己上任後首要工作將會是修補社會撕裂和解開鬱結，團結[[香港市民|香港市民]]。她聲稱要為香港特區政府的管治「注入新風格」，並建立與[[香港立法會|立法會]]議員的恆常溝通機制，為改善[[行政機關|行政]]與[[立法機關|立法機關]]間的關係打好基礎。她又提出透過尊重不同聲音，讓政府、[[公民社會|公民社會]]和民間力量合力解決香港的各種問題，逐漸修補香港社會撕裂。<ref name="林鄭月娥勝選宣言全文：選舉過後，團結向前">{{cite news |language = 中文 |author =  |coauthors =  |url = https://thestandnews.com/politics/%E6%9E%97%E9%84%AD%E6%9C%88%E5%A8%A5%E5%8B%9D%E9%81%B8%E5%AE%A3%E8%A8%80%E5%85%A8%E6%96%87-%E9%81%B8%E8%88%89%E9%81%8E%E5%BE%8C-%E5%9C%98%E7%B5%90%E5%90%91%E5%89%8D/ |title = 林鄭月娥勝選宣言全文：選舉過後，團結向前 |publisher = [[立場新聞|立場新聞]] |pages =  |date = 2017-03-26 |accessdate = 2017-05-16}}</ref>

同年5月4日，林鄭在[[記者會|記者會]]上被問及其在競選期間的支持率較其他候選人為低下當選，她可如何回應市民對其修補社會撕裂能力的質疑時，表示自己在嚴格遵照《[[香港基本法|基本法]]》和法例下當選，足以反映自己作為下任行政長官的認受性，是「不容置疑」，重申自己會盡力團結社會各界，將香港重導至「正常的發展方向」。<ref name="反駁低民望 林鄭：嚴格遵照法例下當選，認受性不應受質疑">{{cite news |language = 中文 |author =  |coauthors =  |url = http://news.memehk.com/posts/17884 |title = 反駁低民望 林鄭：嚴格遵照法例下當選，認受性不應受質疑 |publisher = [[謎米香港|謎米香港]] |pages =  |date = 2017-05-04 |accessdate = 2017-05-16}}</ref>
