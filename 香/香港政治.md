{{香港政治}}
'''香港政治'''由[[香港開埠|香港開埠]]開始，社會發展便同時受臨近的[[中國大陸|中國大陸]]和[[英國|英國]]，形成有別於中國和英國而自成的[[香港政治制度|香港政治制度]]。

== 概要 ==
{{香港}}
大英帝國的構成是由英國作為中心，由[[東印度公司|東印度公司]]統治的[[印度|印度]]，以及各殖民地大體複製英國政治制度架構共同組成。英國有三個法律主權機構分支，各殖民地各自也有三個法律主權機構分支，英國有一個國家元首，各殖民地各自也有一個元首，英國有自己的政治社會和政治社群，各殖民地也各自有自身的一套政治和社會風貌。

香港在[[二戰|二戰]]前的政治制度參與者主要是洋人，不過華人上流社會人士也有組織起來尋求政治權力。當時的政治組織有[[香港憲制改革協會|香港憲制改革協會]]和[[九龍居民協會|九龍居民協會]]。事件有[[海員大罷工|海員大罷工]]、[[省港大罷工|省港大罷工]]、拘捕[[胡志明|胡志明]]、[[楊慕琦計劃|楊慕琦計劃]]等。

香港在二戰後，港英政府開始大幅開放權力，推進本土化，華人政治參與也逐漸增加，由中上層及中產階層，教師和大學生為重要的參與者。早期的政治組織有[[香港民主自治黨|香港民主自治黨]]、[[聯合國香港協會|聯合國香港協會]]，後期則有[[教協|教協]]、[[港同盟|港同盟]]。事件有[[雙十暴動|雙十暴動]]、[[天星小輪加價暴動|九龍騷亂]]、[[六七暴動|六七暴動]]、[[香港前途談判|香港前途談判]]、[[八八直選|八八直選]]、[[全球華人大遊行|一百萬人遊行]]等。社會運動有[[中文運動|中文運動]]、[[保釣運動|保釣運動]]、[[香港民主運動|民主運動]]。

== 主权移交后 ==
1997年[[香港主權移交|香港主權移交]]後，因為《[[中英聯合聲明|中英聯合聲明]]》承諾[[五十年不變|五十年不變]]，所以制度大體沿用港英殖民地時期，继续實行港英殖民地法律，政治社會和政治社群仍不同於[[中國大陸|中國大陸]]。

政治參與普及至[[草根階層|草根階層]]和[[中學生|中學生]]。[[左翼|左翼]]思潮與[[本土派_(香港)|本土主義]]興起<ref>{{Cite web |url= http://m.liberalstudies.hk/blog/ls_blog.php?id=2865 |author= 邱兆麟 |title= 政治光譜2016 |publisher=香港電台 通識網 |date= 2016-05-25 }}</ref>。[[社會運動|社會運動]]有[[香港本土運動|香港本土運動]]、[[新媒體|新媒體]]運動等。大型[[示威|示威]]有[[2003年七一大遊行|2003年七一大遊行]]、[[雨傘革命|雨傘革命]]、[[2016年農曆新年旺角騷亂|2016年農曆新年旺角騷亂]]。個人政治事件有[[斯諾登事件|斯諾登事件]]、[[銅鑼灣書店事件|銅鑼灣書店事件]]、[[林榮基事件|林榮基事件]]等等。政治議題有[[西九龙内地口岸区#轄權爭議|西九内地口岸区爭議]]、[[香港特别行政区基本法第二十三条|基本法二十三条立法]]、[[国民教育科|国民教育科]]、[[普教中|普教中]]<ref>{{Cite news |url= https://hk.news.appledaily.com/local/daily/article/20130717/18337786 |title= 香港政治的深層次問題 |author= 李永康 |publisher= 蘋果日報 |date= 2013-07-17 }}</ref>、[[香港政治審查立法會議員參選人事件|禁止參選立法會事件]]等<ref>{{Cite web |url= https://www.thestandnews.com/politics/香港政治權利大躍-退/ |author=王松蓮 |work=人權觀察 |title= 香港政治權利大躍「退」 |publisher=立場新聞 |date=2018-02-01 }}</ref>。

[[社會問題|社會問題]]有[[貧富懸殊|貧富懸殊]]、[[文化差異|世代衝突]]、[[中港矛盾|中港矛盾]]、[[警民衝突|警民衝突]]、[[双非问题|双非问题]]、[[2015-2017年香港學生連續自殺事件|學童自殺潮]]等等。
