'''8號幹線'''（{{lang-en|'''Route 8'''}}）是[[香港|香港]]第二長的[[香港幹線編號系統|主要幹線]]，亦是香港兩條路標由西向東遞增的高速幹線之一(另一條為[[香港10號幹線|10號幹線]])。它連接[[新界|新界]][[大嶼山|大嶼山]][[赤鱲角|赤鱲角]]和[[沙田_(香港)|沙田]]，1997年5月首段開通時只有赤鱲角至[[青衣島|青衣島]]一段（包括[[北大嶼山公路|北大嶼山公路]]和[[青嶼幹線|青嶼幹線]]）。及後在2008年3月，[[青沙公路|青沙公路]]沙田至[[長沙灣|長沙灣]]段開通（包括[[尖山隧道及沙田嶺隧道|尖山隧道及沙田嶺隧道]]、[[大圍隧道|大圍隧道]]）；2009年12月20日，青沙公路長沙灣至青衣段開通（包括[[昂船洲大橋|昂船洲大橋]]及[[南灣隧道|南灣隧道]]），標誌著8號幹線全線通車。

這條幹線當中包括兩條收費路段：[[尖山隧道及沙田嶺隧道|尖山隧道及沙田嶺隧道]]（合併收費）、[[青嶼幹線|青嶼幹線]]。它是連接[[沙田|沙田]]、[[長沙灣|長沙灣]]、[[青衣|青衣]]及[[大嶼山|大嶼山]]的主要幹道。當中[[北大嶼山公路|北大嶼山公路]]是全港車速限制最寬鬆的路段，限速為每小時110公里。

== 歷史 ==
現時8號幹線路段於1990年規劃，配合1989年的[[機場核心計劃|機場核心計劃]]。赤鱲角至青衣一段稱為9號幹線；青衣至沙田一段則稱為[[香港16號幹線|16號幹線]]。前者於1997年5月22日通車，包括[[北大嶼山公路|北大嶼山公路]]、[[汲水門大橋|汲水門大橋]]及[[青馬大橋|青馬大橋]]。

2002年11月，16號幹線（今8號幹線青衣至沙田段）開始動工興建。2004年1月31日，根據[[香港幹線編號系統|第三代幹線編號系統]]，9號幹線及16號幹線合併成8號幹線。其中南灣隧道早於2005年2月25日貫通，但須配合昂船洲大橋落成才一併通車。

2008年，8號幹線長沙灣至沙田段竣工，並被命名為青沙公路。[[公益金|公益金]]曾於2月24日在該路段舉行新界區[[百萬行|百萬行]]。青沙公路於3月21日通車。

2009年，8號幹線長沙灣至青衣段竣工，公益金於11月15日在昂船洲大橋連同南灣隧道舉行百萬行。青沙公路餘下路段於12月20日通車，同日亦標誌着全長33.1公里的8號幹線全線通車。

== 主要路段 ==
8號幹線途經路段有：（沙田方向顯示）[[北大嶼山公路|北大嶼山公路]]、青嶼幹線（當中包括[[汲水門大橋|汲水門大橋]]、[[馬灣高架道路|馬灣高架道路]]、[[青馬大橋|青馬大橋]]三段）、[[青衣西北交匯處|青衣西北交匯處]]、[[長青公路|長青公路]]、青沙公路（當中包括西青衣高架路、南灣隧道、東青衣高架路、昂船洲大橋、[[昂船洲高架路|昂船洲高架路]]、[[荔枝角高架路|荔枝角高架路]]、尖山隧道、青沙收費廣場（轉車站）、沙田嶺隧道及大圍隧道）。同時分別與3號幹線、7號幹線和9號幹線連接。

這條幹線的沙田至長沙灣段由荔枝角高架路、尖山隧道、青沙收費站、沙田嶺隧道、大圍隧道組成，長約6.7公里，耗資58億港元。沙田、[[大圍|大圍]]直出[[九龍西|九龍西]]只需10分鐘，車程大為縮短；前往[[香港國際機場|機場]]亦只需25分鐘，比取道[[城門隧道|城門隧道]]、[[德士古道|德士古道]]和[[青衣北岸公路|青衣北岸公路]]快5分鐘。這條幹線全線通車，亦可緩減青馬管制區[[青葵公路|青葵公路]]的擠塞情況。<ref>{{cite web|archiveurl=https://web.archive.org/web/20080228142657/http://hk.news.yahoo.com/080223/12/2pcnm.html|title=八號幹線4層天橋創紀錄 下月通車 10分鐘沙田出西九龍|author=明報|date=2008年2月24日|archivedate=2008-02-28|url=http://hk.news.yahoo.com/080223/12/2pcnm.html|deadurl=yes}}</ref><ref>{{cite web|archiveurl=https://web.archive.org/web/20110127171235/http://www.hyd.gov.hk/chi/major/road/projects/6711TH/index.htm|title=路政署－主要工程|archivedate=2011-01-27|url=http://www.hyd.gov.hk/chi/major/road/projects/6711TH/index.htm|deadurl=yes}}</ref>

該路段在[[西九龍公路|西九龍公路]]上加建兩層天橋，由於該位置已有2層天橋，幹線建成後便出現4條天橋重疊的結構，連同[[連翔道|連翔道]]地面的馬路疊成5層車路，僅次於藍田[[觀塘繞道|觀塘繞道]]、[[將軍澳道|將軍澳道]]和[[鯉魚門道|鯉魚門道]]交匯處。由於高架路下面本身已有3層車路（連翔道時速限制達80公里、西九龍公路更達100公里）和港鐵東涌綫及機場快綫路軌，參照觀塘繞道使用傳統的搭棚建橋方法不可行，因為高架天橋下的道路和鐵路必須並非繁忙要道才可使用。西九龍公路、連翔道和港鐵東涌綫及機場快綫正是繁忙要道，加上週邊地區已陸續發展，因此只能用[[預製組件|預製天橋組件]]，故工程人員先把組件運到工地，再以[[吊樑車|吊樑車]]把組件吊到天橋上組合，組件均重達70至120噸，以鋼筋穿好連接。由於高架路穿越多條繁忙道路如[[長沙灣道|長沙灣道]]，平日難以封路進行工程，工程人員只能在深宵工作，每日工作約4小時，最終花了9個月，才把新天橋建成。工程期間，曾有一輛[[吊樑車|吊樑車]]在行駛時，吊臂不慎擊中高懸的天橋組件，結果工程人員要鑿開組件，換掉鋼筋、水泥以保安全。而這條幹線經過的地區亦是高度發展地區，因此工程為遷就地下設施如電話線、水管等，需要更改橋墩位置，或改變地下設施的線路。

[[路政署|路政署]]在這條幹線引入多項全新的防火和環保設計，如[[隧道|隧道]]頂長達數公里的熱力感應器，其形狀如一條橙色電線，一旦隧道發生火警，[[隧道|隧道]][[交通督導員|交通督導員]]就可立時知悉；在高架路段兩個行車方向之間，亦新設緊急閘口，在緊急時可讓消防車駛過。

== 爭議 ==
新建的[[尖山隧道|尖山隧道]]及[[沙田嶺隧道|沙田嶺隧道]]需要收費，政府曾公佈私家車每程要收取12元，比[[城門隧道|城門隧道]]貴7元。[[民間監管公共事業聯委會|民間監管公共事業聯委會]]批評新幹線收費過高，只顧收回成本，無法疏導[[獅子山隧道|獅子山隧道]]擠塞，恐怕會重演三條[[海底隧道|海底隧道]]流量不平均的局面<ref>[http://hk.apple.nextmedia.com/news/art/20070630/7272330 八號幹線沙田長沙灣段收$12 比同區隧道貴 恐難疏導擠塞]</ref>。但到2008年1月中，卻有不少報章報導[[運輸及房屋局|運輸及房屋局]]考慮改為把收費定於任何車輛都只需8元的水平，跟[[獅子山隧道|獅子山隧道]]一致<ref>{{cite web|archiveurl=https://web.archive.org/web/20080122111730/http://hk.news.yahoo.com/080118/12/2ndxf.html|archivedate=2008-01-22|title=8號幹線擬劃一降價至8元 沙田至長沙灣段|author=明報|date=2008年1月19日|url=http://hk.news.yahoo.com/080118/12/2ndxf.html|deadurl=yes}}</ref>。該建議普遍較受歡迎，認為有助吸引司機使用，分流沙田往[[九龍|九龍]]的車輛。2008年1月22日，[[香港特別行政區行政會議|行政會議]]通過[[青沙管制區|青沙管制區]]劃一收費8元<ref>{{cite web|archiveurl=https://web.archive.org/web/20080327174848/http://hk.news.yahoo.com/080122/12/2nkuw.html|archivedate=2008-03-27|title=青沙管制區劃一收8元|author=明報|date=2008年1月22日|url=http://hk.news.yahoo.com/080122/12/2nkuw.html|deadurl=yes}}</ref>。

8號幹線於青沙公路收費廣場建造大型巴士轉車站，不過由於幹線出口不利於使用獅子山隧道的巴士路線改經，該轉車站有近7年的時間近乎荒廢，通車首年更沒有專利巴士途經。轉車站於2015年1月10日，即尖山隧道及沙田嶺隧道通車後近7年，才正式啟用。現時有[[沙田第一城居民巴士88R線|城巴88R線]]（NR88，為[[新界|新界]]單向前往香港島的晨早特別班次） 、[[九龍巴士70S線|九巴70S線]]、[[九龍巴士240X線|九巴240X線]]、[[九龍巴士249X線|九巴249X線]]、[[九龍巴士270B線|九巴270B線]]、[[九龍巴士270D線|九巴270D線]]、[[九龍巴士270P線|九巴270P線]]、[[九龍巴士270S線|九巴270S線]]、[[九龍巴士272P線|九巴272P線]]、[[九龍巴士272X線|九巴272X線]]、[[九龍巴士280X線|九巴280X線]]、[[九龍巴士286C線|九巴286C線]]、[[九龍巴士286P線|九巴286P線]]、[[九龍巴士286X線|九巴286X線]]、[[九龍巴士287X線|九巴287X線]]、[[龍運巴士A47X線|A47X線]]、[[過海隧道巴士980A線|980A線]]、[[過海隧道巴士980X線|980X線]]、[[過海隧道巴士981P線|981P線]]、[[過海隧道巴士982X線|982X線]]、[[過海隧道巴士985線|985線]]及[[過海隧道巴士307C線|307C線]]，合共22條巴士線作為途經尖山隧道及沙田嶺隧道的公共交通。直至2010年7月26日，第一條途經青沙公路的全新巴士線——九巴[[九龍巴士249X線|249X]]線終於開辦，來往[[沙田市中心|沙田市中心]]及[[青衣鐵路站|青衣鐵路站]]，成為首條雙向行走青沙公路的巴士線，唯只於星期一至六早晚繁忙時間服務。2012年8月下旬起再有2條巴士途經青沙公路。第一條路線是2012年8月27日從舊有[[過海隧道巴士182P線|182P線]]改組成的[[過海隧道巴士982X線|982X線]]，唯該路線只於星期一至六早上繁忙時間服務。於2012年12月1日，九巴[[九龍巴士286X線|286X]]線起投入服務，來往[[顯徑邨|顯徑]]及[[深水埗|深水埗]]，成為首條全日行經青沙公路（沙田段）往來的專利巴士路線。而於2014年8月23日，九巴[[九龍巴士287X線|287X]]線起投入服務，來往[[博康邨|博康邨]]、[[佐敦|佐敦]]及[[旺角|旺角]]，成為第二條全日行走青沙公路之專利巴士路線。2015年1日4日，九巴[[九龍巴士280X線|280X]]線起投入服務，來往[[穗禾苑|穗禾苑]]及[[尖沙嘴|尖沙嘴]]，成為第三條全日行走青沙公路之專利巴士路線。

{{Or|有批評指青沙公路於[[荔枝角|荔枝角]]的出入口設計失敗，對來往西九龍及沙田的駕駛者造成不便，令疏導獅子山隧道車流的效用大打折扣，並造成上述巴士轉乘站荒廢之問題。西行出口1C（Slip Road D）並非連接[[香港7號幹線|7號幹線]][[呈祥道|呈祥道]]東行，而是連接呈祥道西行，只方便駕駛者前往[[荔景|荔景]]、[[葵青區|葵青]]和[[元朗|元朗]]；東行方向入口亦須由呈祥道東行進入，西行駕駛者需於7號幹線出口14E繞圈返回東行方向，才能前往沙田，此設計未能吸引來往九龍塘、深水埗及沙田之車輛改經沙田嶺隧道，令獅子山隧道的擠塞情況未有大幅度改善。同時青沙公路西行沒有任何直接與[[香港3號幹線|3號幹線]][[西九龍公路|西九龍公路]]的出口，情況比呈祥道之連接更差劣，車輛須經[[連翔道|連翔道]]或[[貨櫃碼頭南路|貨櫃碼頭南路]]，才能間接與西九龍公路連接，未能完全方便車輛由尖山隧道直達港島西或尖沙咀一帶；反之，東行亦沒有從西九龍公路直達的入口，車輛同樣要改經連翔道或葵涌貨櫃碼頭，再取道[[荔灣交匯處|荔灣交匯處]]，延長駕駛者由[[西區海底隧道|西區海底隧道]]或[[尖沙咀|尖沙咀]]前往沙田之時間。公路更完全沒有轉入[[香港5號幹線|5號幹線]][[西九龍走廊|西九龍走廊]]的出口，由沙田往深水埗一帶需要在荔灣交匯處左轉[[連翔道|連翔道]]再轉入[[興華街|興華街]]，才能前往[[荔枝角道|荔枝角道]]；而由西九龍走廊或美孚前往沙田的車輛，亦要取道[[青山公路－葵涌段|青山公路－葵涌段]]，才能使用設於蝴蝶谷之入口，前往尖山隧道。|time=2019-01-05T07:59:49+00:00}}

2018年5月29日，立法會政府帳目委員會就8號幹線事宜展開聆訊，荔枝角高架道路的工程較原定完工日遲約廿四個月，承建商認為負責設計的顧問公司需承擔責任，向政府提出申索高達十億元，雙方調解不成功後展開仲裁。路政署最終要支付二億七千三百萬元予承建商解決紛爭，但亦成功向顧問公司追討一億三千三百萬元，即政府實際帳面損失約一億四千萬元。路政署署長[[鍾錦華|鍾錦華]]在會上否認部門「先斬後奏」。立法會議員[[林卓廷|林卓廷]]認為事件中有人失職，批評政府監管不足。運輸及房屋局局長[[陳帆|陳帆]]表示，同意審計報告內容，已責成相關部門作出檢討。<ref>{{cite news|url=http://orientaldaily.on.cc/cnt/news/20180530/00176_030.html|date=2018年5月30日|title=八號幹線 路署擅賠2.7億 被轟先斬後奏|newspaper=''[[東方日報_(香港)|東方日報 (香港)]]''}}</ref>
