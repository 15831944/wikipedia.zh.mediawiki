'''香港資訊及通訊科技獎'''（{{lang_en|'''Hong Kong ICT Awards'''}}）是香港的一系列重點創新科技奬項，每年由[[香港政府|香港政府]][[資訊科技總監辦公室|資訊科技總監辦公室]]舉辦。<ref> {{cite web |url= https://www.hkictawards.hk/ |title= 香港資訊及通訊科技奬官方網站 |last= |first= |date= |website= |publisher= 香港政府資訊科技總監辦公室|access-date= |quote=}}</ref>

== 簡介 ==
該獎項分為商業方案、數碼娛樂、金融科技、資訊科技初創企業、智慧生活、智慧市民奬、智慧出行奬、學生創新八個類別，每個類別分別頒發金、銀、銅奬。獎項每年均獲主要媒體報道。<ref> {{cite web |url= https://unwire.hk/2017/02/03/hong-kong-ict-awards-2017/learning/ |title=「2017 香港資訊及通訊科技獎」(Hong Kong ICT Awards) 頒獎典禮將於四月舉行 |last= |first= |date= |website= |publisher= Unwire.hk|access-date= |quote=}}</ref><ref> {{cite web |url= http://hk.on.cc/hk/bkn/cnt/lifestyle/20171102/mobile/bkn-20171102100045677-1102_00982_001.html |title= 2018香港資訊及通訊科技獎 現接受報名 |last= |first= |date= |website= |publisher= 東方日報|access-date= |quote=}}</ref><ref> {{cite web |url= https://www.pcmarket.com.hk/2016/08/10/%E6%B8%AF%E9%9D%92%E5%B0%91%E5%B9%B4%E3%80%80%E5%89%B5%E6%96%B0%E7%A7%91%E6%8A%80%E6%88%90%E6%9E%9C%EF%BC%88%E4%B8%89%EF%BC%89/ |title= 港青少年 創新科技成果 |last= |first= |date= |website= |publisher= PCM電腦廣場|access-date= |quote=}}</ref> <ref> {{cite web |url= https://ezone.ulifestyle.com.hk/article/1963981/2018+香港資訊及通訊科技獎%E3%80%80+智慧生活+%E2%80%A7+創科先行 |title= 2018 香港資訊及通訊科技獎 智慧生活 ‧ 創科先行 |last= |first= |date= |website= |publisher= U週刊 ezone|access-date= |quote=}}</ref>
