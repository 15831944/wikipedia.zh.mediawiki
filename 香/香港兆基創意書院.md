'''香港兆基創意書院'''（{{lang-en|HKICC Lee Shau Kee School of Creativity}}）是[[香港|香港]]一間[[直接資助計劃|直資學校]]<ref>www.creativehk.edu.hk</ref>，而其簡稱為HKSC或創意書院，也是香港第一所以研習藝術、設計、文化為主的高中學校，位於[[九龍城區|九龍城區]][[樂富|樂富]]以南的[[九龍仔|九龍仔]]，由[[李兆基基金會|李兆基基金會]]贊助及[[香港當代文化中心|香港當代文化中心]]主辦，校監為香港當代文化中心創辦人[[黃英琦|黃英琦]]。

==歷史==
香港兆基創意書院於2006年創立，是香港第一所藝術高中。開校前由現任[[浸會大學教育學系|浸會大學教育學系]]首席講師[[許為天|許為天]]擔任校長，籌備開校的工作。開校後的第一任校長為現任教育界[[香港立法會|立法會議員]][[葉建源|葉建源]]，第二任校長為獨立文化藝術工作者，{{citation_needed|[http://www.aco.hk/#!who-chi/cpt9 艺鵠]海豚[[馮美華|馮美華]]，別號 「微瘋」}}。第三任校長是陳婉芬，第四任校長是謝國駿，第五任校長是劉天明。現任校長由陳婉芬擔任。

以前的校舍是借用政府位於[[紅磡|紅磡]]青洲街的[[九龍船塢紀念學校|九龍船塢紀念學校]]舊址，現在位於[[九龍|九龍]][[九龍仔|九龍仔]]聯合道135號的校舍則是永久校舍，由知名建築師[[嚴迅奇|嚴迅奇]]設計。

==特色==
該校注重培養學生藝術文化修養，學習氣氛自由。

其自由度是：

* 學校雖然有校服，由設計師陳米記設計，但學生可以選擇搭配。如有POLO-shirt、T-shirt、恤衫等等，而且有不同的顏色可供選擇。
* 基本上只要不令校服損壞和尊重設計師設計，學生可在校服上做任何設計。
* 該校容許塗鴉。
* 學生可選擇稱呼老師其名。
* 學生可自由舉辦任何學會。
* 班別可設計有自己風格的教室。

該校也是比較民主的一間學校，有關學生的事務也會和學生討論，如學生應否畫枱、遲到缺席問題、使用電子產品問題等

==課程結構<ref>{{cite web |url=http://www.creativehk.edu.hk/new/chi/prog_nss.html |title=存档副本 |accessdate=2014-12-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20141210104135/http://www.creativehk.edu.hk/new/chi/prog_nss.html |archivedate=2014-12-10 }}</ref>==
香港兆基創意書院自於2014年開始，進行雙軌課程，即同時提供「香港中學文憑」及校本「創意藝術文憑」課程。<br />

;香港中學文憑課程：
必修科：中文、英文、數學、通識<br />
選修科：視覺藝術、設計與應用科技、音樂、倫理與宗教、中國文學、資訊與通訊科技<br />

;創意藝術文憑：<ref>{{cite web |url=http://www.creativehk.edu.hk/new/DCA/DCA_booklet.pdf |title=存档副本 |accessdate=2014-12-10 |deadurl=yes |archiveurl=https://web.archive.org/web/20141210104532/http://www.creativehk.edu.hk/new/DCA/DCA_booklet.pdf |archivedate=2014-12-10 }}</ref>
課程已於2014年7月通過[[香港學術及職業資歷評審局|香港學術及職業資歷評審局]]的評審，頒授資歷為[[資歷架構|資歷架構]]第三級。<br />
學習內容包括：
# 設計與視覺傳意 DVC
# 電影與錄像藝術 FDA
# 表演藝術 MMPA
# 空間研習 ESS
# 創意藝術基礎 CPOP

# 研究方法
# 人文通識（藝術史、哲學、香港電影）
# 實用中、英文

==設施==
位於九龍城的永久校舍由國際知名的本地建築師[[嚴迅奇|嚴迅奇]]先生設計，理念具體地配合建構創意學習社群的願景，空間佈局開放流動，教學區域和社區平台兩個部份的組成強調了師生之間、以及與社區和業界的溝通和互動。其中一個設計的特徵是南北走向的創意大道，連結了校園的各部份，亦將社區的資訊交流和動力引入校園。

香港兆基創意書院的校舍設計源於對教育願景，校園內設有專業水平的文化藝術場地。場地作為教育、交流、展覽及演出的平台，在配合學生和老師的學習和創作之餘，且作為本地文化藝術及產業界的培育中心，為實驗和創作主動地推動和支援。

==活動==
由2009年起，逢12月設文化活動「九龍城書節」，以當年牛棚書節的模式而舉辦。設書商、創意地攤、講座、工作坊、讀書會和電影放映等活動，每年吸引不少市民參與。<ref>{{cite news|title=周末逛九龍城書節 地攤尋寶搜羅童書|url=https://hk.news.yahoo.com/%E5%91%A8%E6%9C%AB%E9%80%9B%E4%B9%9D%E9%BE%8D%E5%9F%8E%E6%9B%B8%E7%AF%80-%E5%9C%B0%E6%94%A4%E5%B0%8B%E5%AF%B6%E6%90%9C%E7%BE%85%E7%AB%A5%E6%9B%B8-221118895.html|accessdate=2016-12-04|publisher=星島日報|date=2016-12-01}}</ref>
