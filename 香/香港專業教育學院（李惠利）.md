'''香港專業教育學院（李惠利）'''位於[[香港|香港]][[新界東|新界東]][[調景嶺|調景嶺]][[景嶺路|景嶺路]]3號，前身為'''李惠利工業學院'''，成立於1979年，原址位於[[九龍塘|九龍塘]][[聯福道|聯福道]]30號（[[香港高等教育科技學院|香港高等教育科技學院]]九龍塘分校所在地），由著名[[香港工業家|香港工業家]][[李惠利_(企业家)|李惠利]]捐助興建。

== 歷史 ==
[[2010年9月|2010年9月]]，李惠利分校與新成立的[[香港知專設計學院|香港知專設計學院]]一併遷往[[調景嶺|調景嶺]]，以改善學習環境。

2007年6月22日[[香港立法會|立法會]]財務委員會通過撥款10億630萬港元以供興建新校舍<ref>[http://www.legco.gov.hk/yr06-07/chinese/fc/pwsc/papers/p07-24c.pdf 立法財務委員會工務小組委員會在2007年6月6日所提出的建議 - 為職業訓練局興建位於調景嶺的新校園]</ref><ref>[http://www.legco.gov.hk/yr06-07/chinese/fc/fc/results/fc_r070622.htm 立法會財務委員會 2007年6月22日會議紀錄]</ref>。新校舍在2007年8月動工，2010年9月啟用<ref>[http://lwsua.vtc.edu.hk/lib/info/2010summer.htm 李惠利分校圖書館的暑期搬遷安排] {{webarchive|url=https://web.archive.org/web/20100111055610/http://lwsua.vtc.edu.hk/lib/info/2010summer.htm |date=2010-01-11 }}</ref>；然而遷入新校舍的學科數目有所變動。

==校舍==
===九龍塘舊校舍===
[[File:IVE_LeeWaiLee.jpg|thumb]]的舊校舍]]
[[File:IVE_Lee_Wai_Lee_Hall_2016.jpg|thumb]]
李惠利分校遷往調景嶺後，坊間普遍估計騰出的舊址可供予毗鄰的[[香港浸會大學|香港浸會大學]]發展，惟根據[[香港傳媒|香港傳媒]]2009年報道，浸會大學早前申請於該地皮興建大學校舍及宿舍的計劃被[[香港政府|香港政府]]拒絕<ref>[http://hk.news.yahoo.com/article/091221/4/frga.html 浸大生：三人房慘過住籠屋，《明報》，2009年12月22日]</ref>。

在遷校後，本校舍曾用作[[香港理工大學|香港理工大學]]及浸會大學的臨時延伸校舍，以暫時遷置兩校因「334新學制」而進行教學大樓重建工程的受影響學系，直至2012年上述工程完成後再次回遷。

2012年12月，香港政府正式提出計劃將部份用地作為住宅更改興建用途<ref>[http://hk.news.yahoo.com/吳克儉-政府支持浸大擴建但要按需要規劃-103700827.html 吳克儉：政府支持浸大擴建但要按需要規劃]</ref><ref>[http://hk.apple.nextmedia.com/financeestate/art/20121221/18109034 蘋果日報：李惠利校一拆二發展]，2012年12月21日</ref><ref>[http://hk.news.yahoo.com/吳克儉指大學擴建要按實際需要-064110008.html 吳克儉指大學擴建要按實際需要]</ref>。浸大校長陳新滋當日隨即發表聲明，表示閱報得悉事件「感到非常震驚，亦極度失望……我們不會放棄爭取前李惠利學院地段整幅土地作為大學永久用途的決心，並會繼續與政府商討，希望政府重新考慮浸大的迫切需要和對這用地的合理要求」。<ref>[https://www.facebook.com/HKBUSU 第45屆浸大學生會Facebook專頁]，2012年12月21日</ref>翌日傳媒作出跟進報道，指城規會審議時，有委員提出應再徵詢教育局，決定延期審議。<ref>[http://news.sina.com.hk/news/20121222/-3-2854216/1.html 明報：屬意宿舍地擬改劃住宅　浸大反對] {{webarchive|url=https://web.archive.org/web/20130125231942/http://news.sina.com.hk/news/20121222/-3-2854216/1.html |date=2013-01-25 }}，2012年12月22日</ref>

2014年3月10日，[[城市規劃委員會|城規會]]通過將九龍塘前李惠利校舍南面地皮，還原作「政府、機構及社區」用地的用途。<ref>{{Cite news|url = http://www.881903.com/Page/ZH-TW/newsdetail.aspx?ItemId=707037&csid=261_341|title = 城規會通過還原前李惠利地皮為政府機構社區用途|last = |first = |date = |work = |accessdate = }}</ref>

2014年10月31日，[[教育局_(香港)|教育局]]向浸大表示，政府撤回改劃決定，將南面用地當中約1.18萬平方呎土地增撥予浸大，以增加興建1600個資助宿位和約5.4萬平方呎的教學設施，以整幅李惠利前校舍地皮（16.3萬平方呎）計算，浸大總共獲撥49％即約8萬平方呎土地，餘下南部約0.77公頃（8.3萬平方呎）將用作特殊教育用途（智障人士學校），浸大對安排表示十分歡迎。<ref>{{Cite news|url = http://www.mingpaocanada.com/Van/htm/News/20141101/HK-gob1_r.htm|title = 前李惠利用地增撥浸大|last = |first = |date = |work = |accessdate = }}</ref>

2015年1月至2017年12月期間，職業訓練局再次使用本大樓，作[[香港高等教育科技學院|香港高等教育科技學院]]的臨時校址之一，直至柴灣永久校舍在2018年1月啟用後已遷出，將會空置直至拆卸重建工程開展為止。

===調景嶺新校舍===
[[File:LWL_Campus_Stair_View_201608.jpg|thumb]]新校舍樓梯]]
由於2010年開學初期，新校舍部份設施或課室裝修仍然進行中，故李惠利分校及[[香港知專設計學院|香港知專設計學院]]學生的上課時間曾較正常的少，在裝修完畢後已回復正常。由於兩間院校坐落在同一地方，為節省資源及加快行政效率，兩間院校共用學生事務處、飯堂、圖書館、學生會、學會等設施，所以李惠利分校及[[香港知專設計學院|HKDI]]和[[香港高等科技教育學院|Thei]]形同一院校。另外，校園地圖除顯示HKDI校舍的Block A - D外 ，亦會顯示Block LW（李惠利大樓），內部分校簡稱亦為DILWL。由此可見，兩間院校硬件上有密切的關係。
