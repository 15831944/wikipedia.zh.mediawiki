'''香蘭車站'''位於[[台灣|台灣]][[臺東縣|臺東縣]][[太麻里鄉|太麻里鄉]]，曾為[[臺灣鐵路管理局|臺灣鐵路管理局]][[南迴線|南迴線]]（現已廢止）的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*島式月臺一座，設有地下道連接站房，現已用木板封閉。

== 利用狀況 ==
*目前已廢止。

== 車站周邊 ==
*[[太平洋|太平洋]]
*臺東縣香蘭村新香蘭部落
*臺東縣香蘭國民小學
*[[臺9線|臺9線]]
*香蘭產業道路
*[[空軍志航基地|空軍志航基地]]太麻里靶場

== 歷史 ==
*1989年7月25日：設站。
*1997年2月19日：廢止。
