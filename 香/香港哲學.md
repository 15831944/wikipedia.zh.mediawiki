'''香港哲學'''（{{lang-en|Hongkonger Philosophy}}），是一種哲學傳統或區域哲學<ref name = "zyutai">{{cite web |author1=楊梓燁|title=香港哲學及主體性怎樣成為可能？|url =https://chitchitphilosophy.blogspot.com/2018/08/blog-post_29.html|agency=書生百用|date=2018-08-29|accessdate=2019-04-11|}}</ref>，於[[香港|香港]]產生或延續。香港哲學不屬[[中國哲學|諸子百家哲學]]與[[西方哲學|西方哲學]]，但兩者皆對香港哲學影響甚大，加上其他文化影響，香港哲學因而產生<ref name = "zyutai"/>。
==新儒家==
[[新儒家|新儒家]]於香港將[[儒家|儒家]]與[[康德|康德]]學說結合，表述心性義理，發揮外王大義，疏導中國哲學，消納西方哲學，會通中西哲學。新儒家主義者包括，[[唐君毅|唐君毅]]、[[牟宗三|牟宗三]]。

==存在主義==
二十世紀五十年代末至六十年代初，香港流行[[存在主義|存在主義]]<ref name = "ngau cam"/>。存在主義於香港，與於歐洲不同，香港主要以報刊、雜誌發展存在主義。存在主義強調「個人價值」，不與強調「[[民族國家|民族國家]]」之人為伍。由於存在主義可於戰後，滿足精神需要，所以於香港流行<ref name = "daaizoungmanfaa"/>。

當時，香港之文學雜誌與報刊（例如《好望角》、《文藝新潮》、[[劉以鬯|劉以鬯]]主編之《香港時報·淺水灣》）大量翻譯存在主義之作品，包括[[祈克果|祈克果]]、[[尚-保羅·沙特|沙特]]、[[卡繆|卡繆]]之作品。一些當時之香港文學雜誌（例如《好望角》）中，有文章詮釋存在主義之著作（例如鍾榮期之《談沙爾特的小說》），並且亦有一些作品（例如小說）彌漫存在主義，例如[[張健_(詩人)|汶津]]之《開往嘉義的吉普》、[[戴天|戴天]]之《化石》、[[崑南|崑南]]之《大風起兮》<ref name = "ngau cam">{{cite book |date=2009|title=香港文學展顏 第十八輯|author=<nowiki>楊彩杰</nowiki>|work=〈「存在」的<nowiki>鉤</nowiki><nowiki>沈</nowiki>──以《好望角》所載小說為例，探討六十年代香港文學對西方存在主義思潮的思考〉 |editor=王惠屏|location=香港 |publisher=香港公共圖書館 |page=127-148 |isbn=978-962-8899-11-1}}</ref>。

汶津於《開往嘉義的吉普》中，發展沙特之觀點──「拒絕歷史責任」與「存在之孤獨感」。《開往嘉義的吉普》之主角，意識到自己是自由，並且認為上一代所賦予之歷史責任是強行加上，所以主角則拒絕歷史責任。但是，當拒絕歷史責任之後，主角則變得孤立無援，不知何去何從，失去存在意義<ref name = "ngau cam"/>。

戴天之《化石》與卡繆之《[[異鄉人|異鄉人]]》相似。《異鄉人》中，主角對一切漠不關心，不關心母親逝世，不關心被判死刑。《化石》中，主角從對身邊事物充滿感情，變成不在乎一切，並且認為自己不能融入世界，因此主角成為局外人，就像卡繆對於人處身於世界之觀點──「演員與舞臺之間疏離」<ref name = "ngau cam"/>。

二十世紀六十年代初之後，香港仍有一些存在主義作品，包括[[西西|西西]]<ref name = "daaizoungmanfaa"/>與劉以鬯<ref name = "deoidou"/>之作品。西西於其小說，例如《東城故事》、《草圖》中，主角都認為「生命似乎沒有意義」。西西認為，此類作品皆屬[[存在主義|存在主義]]<ref name = "daaizoungmanfaa">{{cite journal|author=鄭蕾|title= 存在主義在香港──一種「大眾文化」的解讀|doi=|ISSN= 0257-0254|location=[[上海|上海]]|journal=文藝理論研究|publisher=中國文藝理論學會；[[華東師范大學|華東師范大學]]|volume= |issue=3 |year=2012 |pages=138-144}}</ref>。

劉以鬯於小說《[[對倒_(小說)|對倒]]》（長篇版）於二十世紀七十年代完成，其於《對倒》中展示出存在主義。當時，香港面臨英國與中國爭奪統治權，前景難測，香港人心靈極空無、焦慮。劉以鬯在《對倒》之第一段形容香港是「無根的樹」、「時間是借來的」、「躺在帆船甲板上的睡狗」，以寫出香港是虛無。《對倒》之男主角常回憶往事消除孤獨，而女主角則常幻想來消除寂寞，展示出現實與人疏離。劉以鬯於《對倒》亦描述了[[荒謬主義|荒謬]]，例如表達出香港人過於重視物質：炒股、炒樓、賭馬，只<nowiki>着</nowiki>重自己利益，而不幫助其他人<ref name = "deoidou">{{cite journal|author=鄺銳強|title= 存在主義對劉以鬯《對倒》的影響|doi=|ISSN= |location=[[香港|香港]]|journal=香江文壇|publisher=香江文壇編輯部|volume= |issue=38 |year=2005 |pages=44-46}}</ref>。

[[譚家博|譚家博]]於2017年將[[祈克果|祈克果]]之存在主義，應用於重新「認同香港文化」，將香港文化問題視為一個哲學問題處理<ref>{{cite web |author1=譚家博|title=讀哲學要讀很多書嗎？|url =http://kowloondaily.com/2017/10/09/%E8%AE%80%E5%93%B2%E5%AD%B8%E8%A6%81%E8%AE%80%E5%BE%88%E5%A4%9A%E6%9B%B8%E5%97%8E%EF%BC%9F/|agency=九龍叢報|date=2017-10-09|accessdate=2019-04-11|}}</ref>。根據[[格拉斯哥大學|格拉斯哥大學]]之資料，譚氏之研究注重將祈克果哲學應用於東亞文化脈絡之下。在[[黑格爾|黑格爾]]及[[赫特|赫特]]哲學的影響下，[[京都學派|京都學派]]及[[新儒家|新儒家]]皆以建構日本文化精神或中國文化精神為文化主體重構之方向；然而，譚氏認為此舉難免鼓吹國族主義，壓抑個人主體性，故引入祈克果之個人主體性、社區及文化作為熱情等概念，重構香港文化主體。<ref>{{cite web |author1=譚家博|title=MR KA POK TAM|url=
 https://www.gla.ac.uk/schools/critical/staff/kapoktam/#/researchinterests}}</ref>

==知識論==

===邏輯實證主義===
[[李天命|李天命]]受到[[艾耶爾|艾耶]]之《[[語言、真理與邏輯|語言、真理與邏輯]]》影響，而於1981年完成《[[語理分析的思考方法|語理分析的思考方法]]》，主張[[邏輯實證主義|邏輯實證主義]]。李天命於此書主張，若且唯若，當一個語句有真假條件時，才是有認知意義，而有真假條件，就可以考察語句之真假值。李天命批評，沒有認知意義之語句說不上「真」或「假」，是連「假」的資格都無。李天命認為，沒有認知意義之語句是沒有真假值。<ref>{{cite book |author=[[李天命|李天命]] |date=1999|title=《語理分析的思考方法》|location=香港 |publisher=青年書屋 |page=90-91 |isbn=962-393-700-8}}</ref>李天命主張印證原則（confirmation principle）：如果一個語句是[[分析-綜合區別|分析語句]]，那就是有認知意義。如果一個語句是綜合語句，但不可以由經驗證據支持，那就是沒有認知意義。<ref>{{cite book |author=[[李天命|李天命]] |date=1999|title=《語理分析的思考方法》|location=香港 |publisher=青年書屋 |page=95|isbn=962-393-700-8}}</ref>李天命於此書批評，[[黑格爾|黑格爾]]的一些語句是意義不明，沒有認知意義。<ref>{{cite book |author=[[李天命|李天命]] |date=1999|title=《語理分析的思考方法》|location=香港 |publisher=青年書屋 |page=22|isbn=962-393-700-8}}</ref>李天命將沒有認知意義，稱為偽贗（pseudo）。
==倫理學==
===自由主義===
[[周保松|周保松]]主張[[自由主義|自由主義]]，肯定自由是可貴，並且認為各人皆可自由選擇自己幸福人生，社會制度亦應如此，以容許社會各人可充分選擇與追求自己之理想人生<ref>{{cite web |author1=|title=風雨中抱緊自由？|url =https://www.rthk.hk/tv/dtt31/programme/philosophynight2018/episode/493282|agency=[[香港電台|香港電台]]|date=2018-04-09|accessdate=2019-04-11|}}</ref>。周保松主張，人人皆為自己之主人，自己之身體與思想，不受其他人支配，不是其他人之工具，並且認為如此，方可以達到人之最高利益
<ref>{{cite book |author=[[周保松|周保松]]|date=2015|title=《政治的道德：從自由主義的觀點看》（增訂版）|location=香港 |publisher=中文大學出版社 |page=24-25|isbn=978-962-996-664-5}}</ref>。周保松支持，[[約翰·斯圖爾特·密爾|約翰·斯圖爾特·密爾]]於《[[論自由|論自由]]》提出之「傷害原則」（harm principle）
：每個人只要不傷害其他人，就應該擁有自由，選擇自己之生活方式。周保松由「傷害原則」推論出，政府與社會應該寬容人之觀點，即使難以接受那些人之觀點<ref>{{cite book |author=[[周保松|周保松]]|date=2015|title=《自由人的平等政治》（新版）|location=香港 |publisher=中文大學出版社 |page=121|isbn=978-962-996-679-9}}</ref>。
