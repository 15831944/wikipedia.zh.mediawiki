'''[[香港|香港]][[郊野公園|郊野公園]]'''是指由[[香港政府|香港政府]]將市郊未開發地區的地方劃出，作為康樂及保育用途的[[公园|公園]]，地位與[[國家公園|國家公園]]相若。

香港政府在1976年制定《郊野公園條例》（[[香港法例|香港法例]]第208章）[https://www.elegislation.gov.hk/hk/cap208!zh-Hant-HK]，並於同年12月3日劃定首3個郊野公園。目前全港已劃定了24個郊野公園，連特別地區總面積達44,300[[公頃|公頃]]，佔香港約四成（39.98%）的土地面積。2014年，郊野公園吸引了1,120萬人次遊客到訪<ref>[http://www.afcd.gov.hk/tc_chi/country/cou_lea/the_facts.html 香港便覽 - 郊野公園及自然護理]，漁農自然護理署</ref><ref name="mp070829a">[http://hk.news.yahoo.com/070829/12/2emx4.html 郊野公園轉型 變探索自然場所]，《明報》，2007年8月29日{{dead link}}</ref>。

== 源起 ==
早在1844年，[[香港殖民地時期#香港政府|香港政府]]頒布《良好秩序及潔淨條例》（{{lang|en|Good Order and Cleanliness Ordinance}}）的法例，禁止損害[[树|樹木]]和[[灌木|灌木]]；1913年，[[烏蛟騰|烏蛟騰]][[瀑布|瀑布]]附近被劃定為樹林護理區；1925年，[[大埔滘|大埔滘]]部分林地劃定為大埔樹林護理區。1928年，[[港島東|港島東]][[哥連臣山|哥連臣山]]一帶列為[[植林|植林]]禁區，成為香港首個法定保護地區，但當時香港尚未存在法定的郊野公園。 	

1940年代的[[香港日治時期|香港日治時期]]，香港的樹林大多遭砍伐殆盡以用作燃料。重光的香港郊區，很多地方都變成寸草不生的荒野。當時政府雖然進行植林避免[[水土流失|水土流失]]，但有欠系統<ref name="mp070829b">〈[http://hk.news.yahoo.com/070829/12/2emx5.html 郊野公園30歲]〉，《明報》，2007年8月29日</ref>。隨著1950年代至1960年代初，從[[中國大陸|中國大陸]]不斷有[[移民|移民]]湧入，在[[逃港|逃港潮]]引致人口劇增下，水塘的水源和周邊的山野便顯得有保護的必要<ref>{{cite news|title=郊野公園回顧 懶人包 郊野公園的前世今生|url=http://s.nextmedia.com/magazine/r.php?a=2_542408_1&i=20170826|date=2017-08-26|agency=蘋果日報|publisher=壹週Plus}}</ref>。1965年，[[美國|美國]]環境科學專家、後來擔任[[美國喬治梅森大學|美國喬治梅森大學]]（George Mason University, USA）環境科學及政策學系客座教授的[[戴爾博|戴爾博]]（{{lang|en|Lee M. Talbot}}）<!--按《郊野叁十年》譯名-->應[[香港殖民地時期#香港政府|香港政府]]邀請，偕同妻子（{{lang|en|Marty H. Talbot}}）前來，實地考察香港郊野環境。他從1950年代開始在[[東亞|東亞]]進行環境研究工作<ref name="hknature">[http://www.hknature.net/IUCN_news/iucn_news.htm 香港自然保育的起源 - 環境科學專家湯博立 (Prof. Lee M. Talbot)] {{webarchive|url=https://web.archive.org/web/20060615071522/http://hknature.net/IUCN_news/iucn_news.htm |date=2006-06-15 }}，香港自然網</ref>，而在一次香港考察期間，[[皇家香港輔助空軍|皇家香港輔助空軍]]以兩架單引擎「{{lang|en|Auster}}」飛機接載他們，但在[[大嶼山|大嶼山]]以北的上空發生輕微爆炸，飛機直衝大海，幸而未有人死亡，翌日報章笑稱這次為「香港首次水底空中勘察」。<ref>《郊野叁十年》，第38頁</ref>　

香港政府漁農處顧問戴南博博士及戴瑪黛夫人於1965年4月合撰一份《香港保存自然景物問題簡要報告及建議》，提出成立專責自然護理的組織，管理郊區；成立法定的保護地區，防止人為破壞山野；進行植林工作，營造一個多樣性的生態系統；展開科學研究，加強對香港生態環境的認識等等。<ref name="hknature" />

== 歷史 ==
1967年3月[[香港總督|香港總督]][[戴麟趾|戴麟趾]][[爵士|爵士]]委任「郊區的運用和保存臨時委員會」，於翌年發表《郊野與大眾》報告作出康樂和自然保育需求的整體檢討，建議成立「郊野議會」向政府提供有關意見。1971年由「戴麟趾爵士康樂基金」撥出款項於[[城門水塘|城門水塘]]開設郊遊和燒烤試驗區。而《郊野公園條例》最終於1976年生效，翌年6月24日劃定第一批受法律保護的郊野公園，包括城門、金山、大潭、獅子山及香港仔。同年設計香港仔、大埔滘、大尾篤及新娘潭自然教育徑。1979年大尾篤及香港仔郊野公園遊客諮詢中心開始啟用。1985年位於[[北潭涌|北潭涌]]的西貢郊野公園遊客中心啟用。1987年修建第一條「家樂徑」。1992年設立「樹木研習徑」。至2007年3月，香港已劃定了23個郊野公園及17個[[香港特別地區|特別地區]]（其中11個位於郊野公園之內），總面積超過41,521公頃，每年遊客約1200萬人次。闢設郊野公園的目的，是為了保護大自然，以及向市民提供郊野的康樂和戶外教育設施。至於闢設特別地區，主要目的是為了保護自然生態。

1996年頒布的[[海岸公園|海岸公園]]及海岸保護區規例則更將保護區域包括了[[香港海岸公園|香港海洋環境]]。

漁農自然護理署於2000年代起加強宣傳及教育，以配合郊野公園轉型為供市民探索[[大自然|大自然]]的場所，如近年出版大量有關[[香港生態|香港生態]]的書籍，又把郊野公園內的黑白資料牌換成彩色，以及更新郊野公園遊客中心的資料等<ref name="mp070829a" />。

== 設施 ==
郊野公園內設有具備桌椅的[[野餐|野餐]]地點、[[燒烤爐|燒烤爐]]、[[兒童遊戲設備|兒童遊戲設備]]、[[涼亭|涼亭]]、[[營地|營地]]和[[廁所|廁所]]等。上述設施的設計與天然環境互相配合。此外，山間主要[[山徑|山徑]]設置了[[路標|路標]]，也有[[行人小徑|行人小徑]]及[[家樂徑|家樂徑]]，供遊人在山林中漫步，欣賞大自然景色。

;其他設施
* 具有教育意義的自然教育徑
* 4條長途遠足徑
* 健身徑
* 緩跑徑
* 傷殘人士而設的輪椅徑、傷健樂園
* 晨運園地
* 兒童歷奇樂園
* 郊野公園遊客中心

== 生態 ==
在郊野公園內，有各式各類的動植物可供欣賞。[[植物|植物]]方面有本土和外來的品種，如[[樟樹|樟樹]]、[[楠樹|楠樹]]、[[木荷|木荷]]、[[台灣相思|台灣相思]]、[[愛氏松|愛氏松]]及[[紅膠木|紅膠木]]等。[[動物|動物]]則有[[獼猴|獼猴]]、[[野豬|野豬]]、[[豹貓|豹貓]]、[[穿山甲|穿山甲]]、[[箭豬|箭豬]]及[[松鼠|松鼠]]等。[[雀鳥|雀鳥]]則有[[毛雞|毛雞]]、[[大擬啄木鳥|大擬啄木鳥]]、[[白頭翁|白頭翁]]、[[八哥|八哥]]、[[珠頸斑鳩|珠頸斑鳩]]及[[黑耳鳶|黑耳鳶]]等。此外，並有種類繁多的[[昆蟲|昆蟲]]，其中[[蝴蝶|蝴蝶]]便多達240種。

== 使用情況及影響 ==
香港的郊野公園深受市民歡迎。2001年統計，前往郊野公園的遊人約1,110萬人次，在 2012年，前往郊野公園的遊人約有1290萬人次，他們所舉行的活動包括[[漫步|漫步]]、[[健身|健身]]、[[遠足|遠足]]、[[燒烤|燒烤]]以至家庭[[旅行|旅行]]及[[露營|露營]]。

== 面對問題 ==
=== 山火 ===
*香港秋冬[[旱季|旱季]]天氣清涼乾爽期間，郊野公園容易發生[[山火|山火]]，平均每年有百多次山火，多為人為所致。其中原因多為遊人不小心遺下火種所致。由於不少[[新界原居民|新界原居民]]先人的墓地位於郊野公園內，在[[清明節|清明節]]和[[重陽節|重陽節]]等時節前後因掃墓生火而引起的山火，情況尤其嚴重。

;城門郊野公園
*1986年1月8日，城門、大欖、大帽山郊野公園及大埔滘自然護理區曾發生歷時34小時的大火，焚毀樹木13萬株，受波及的郊區面積約達900[[公頃|公頃]]，並要暫時封閉以上的郊野公園和地區。<ref>《郊野叁十年》，第110頁</ref>

;八仙嶺
{{seealso|1996年八仙嶺山火}}
*1996年2月10日八仙嶺發生五級[[山火|山火]]，導致[[香港中國婦女會馮堯敬紀念中學|香港中國婦女會馮堯敬紀念中學]]3名[[學生|學生]]及2名[[老師|老師]]死亡，13名學生受傷。
*2009年1月18日，八仙嶺又發生山火<ref>[八仙嶺山火圍困11人 為逃命跑至腳抽筋_星島環球網_港澳 {{cite web |url=http://www.stnn.cc:82/hongkong/200901/t20090118_964192.html |title=存档副本 |accessdate=2010-06-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20130314183721/http://www.stnn.cc:82/hongkong/200901/t20090118_964192.html |archivedate=2013-03-14 }}]</ref>，不過嚴重程度不及1996年山火。

;大欖郊野公園
*2006年11月2日，大欖發生嚴重山火，4部門撲救46小時才撲熄。400公頃、約65,000棵喬木被燒毀，是一香港的生態災難。

=== 垃圾 ===
大量[[垃圾|垃圾]]為另一問題。在2001年，郊野公園共被清理出約3,850[[公噸|公噸]]垃圾。

=== 罪案 ===
偷渡客及劫匪非法砍伐[[羅漢松|羅漢松]]及[[土沉香|土沉香]]。

=== 爭議 ===
'''沙螺洞事件'''

1979年一間名為「沙螺洞發展有限公司」的發展商開始在[[沙羅洞|沙羅洞]]收購土地，提出將該處發展為[[高爾夫球|高爾夫球]]場及住宅項目，包括一個18洞的高爾夫球場、66幢低密度房屋、約200個公寓或單位、160幢新村屋等，共佔地85公頃；其中32公頃在[[八仙嶺郊野公園|八仙嶺郊野公園]]之內。計劃於1989年獲郊野公園管理局批准。1992年有環保團體向[[高等法院|高等法院]]提出[[司法覆核|司法覆核]]，獲勝訴推翻批-{准}-發展的決定；[[漁農自然護理署|漁農處]]承認批准是技術錯誤，因為有關的土地位於已有的郊野公園範圍之內。

'''三號幹線'''{{Main|香港3號幹線}}
三號幹線中的郊野公園段原計劃修建於地面，將[[大欖郊野公園|大欖郊野公園]]一分為二，嚴重影響生態環境及破壞公園景致。經漁農自然護理署、路政署和運輸署力爭，才將這路段深埋地下，成為一條3.8[[公里|公里]]長的[[大欖隧道|大欖隧道]]連接汀九和元朗，大欖郊野公園才得以保持完整。

'''郊野公園土地建屋問題'''

2017年1月18日，時任行政長官[[梁振英|梁振英]]發表施政報告，建議利用[[郊野公園|郊野公園]]內小量生態價值不高、公眾享用價值較低、位於邊陲地帶的土地用作公營房屋。[[本土研究社|本土研究社]]成員陳劍青指出，做法違背《[[跨越2030年的規劃遠景與策略|跨越2030年的規劃遠景與策略]]》（簡稱《2030+》）意向，皆因《2030+》中，政府從未提及任何郊野公園發展，而時任發展局局長陳茂波曾承諾，任期內不會「考慮」發展任何郊野公園，批評一旦發展郊野公園，會觸發骨牌效應。<ref name=":0">{{Cite news|url=http://www.inmediahk.net/node/10471036|title=【施政報告】郊野公園土地建屋  本土研究社憂觸發骨牌效應 {{!}} 獨媒報導 {{!}} 香港獨立媒體網|newspaper=香港獨立媒體網|accessdate=2017-12-09}}</ref>

陳劍青亦指出，施政報告提到將佔地500公頃的[[紅花嶺|紅花嶺]]納入為郊野公園，但其實紅花嶺即使不劃為郊野公園，本身已是具生態價值的山林資源，而用作起樓的郊野公園邊陲卻會受到實質破壞。<ref name=":0" />

== 未來發展 ==
漁農自然護理署在2007年8月表示，只會為提供郊野公園「簡單而實際」的設施，因此否決市民提議加設如沐浴室及茶水亭等設施的建議<ref name="mp070829a" />。然而，署方有意繼續推廣電燒烤爐，繼於大帽山郊野公園設置後，現正計劃為香港仔郊野公園加設同類裝置<ref name="mp070829a" />。

而最新劃定的[[北大嶼郊野公園（擴建部份）|北大嶼郊野公園（擴建部份）]]位於[[大嶼山|大嶼山]]北部，佔地達2,360[[公頃|公頃]]。公園已於2008年11月7日根據《2008年郊野公園（指定北大嶼郊野公園（擴建部分））令》正式劃定。該郊野公園是繼[[龍虎山郊野公園|龍虎山郊野公園]]後最新的一個郊野公園（即香港第24個郊野公園），[[香港主權移交|香港主權移交]]後第二個劃定的郊野公園，並成為大嶼山面積第二大的郊野公園。擴建[[北大嶼郊野公園|北大嶼郊野公園]]的原因是為了補償[[香港機場核心計劃|香港機場核心計劃]]和新界西區其他基礎建設對環境造成的破壞。現時該公園的基建工程尚在進行中，預計需時一至兩年。

== 管理 ==
香港郊野公園由[[漁農自然護理署|漁農自然護理署]]負責管理一切有關事務，而[[郊野公園及海岸公園委員會|郊野公園及海岸公園委員會]]則作為郊野公園及海岸公園管理局總監的諮詢團體。

;郊野公園之友會
[[郊野公園之友會|郊野公園之友會]]（{{lang|en|Friends of the Country Parks}}，前稱協護郊野會，簡稱郊友會）成立於1992年，是非牟利機構，為郊野公園活動最為重要的非政府伙伴，由現任或已退任之郊野公園及海岸公園委員會委員和熱心人士組成，以推動香港郊野公園的發展及鼓勵市民善用郊區自然資源為目的，近年致力出版一系列「放眼大自然」的書籍。
