'''香港瑰麗酒店'''（{{lang-en|Rosewood Hong Kong}}），是{{link-en|瑰麗酒店集團|Rosewood Hotel Group}}旗下的[[甲級高價酒店|甲級高價酒店]]，位於[[九龍|九龍]][[尖沙咀|尖沙咀]][[梳士巴利道|梳士巴利道]]18號[[Victoria_Dockside|Victoria Dockside]]24樓至40樓、鄰近[[香港星光大道|香港星光大道]]，室內和室外總面積達33,880平方呎。由{{link-en|瑰麗酒店|Rosewood Hotels & Resorts}}管理，開幕日期由2018年9月延至2019年3月18日開幕<ref>{{cite web |title=【香港好去處】尖沙咀香港瑰麗酒店開幕 |url=https://www.weekendhk.com/911416/lifestyle/weekend-leisure/%E5%B0%96%E6%B2%99%E5%92%80-%E9%A6%99%E6%B8%AF%E7%91%B0%E9%BA%97%E9%85%92%E5%BA%97-%E7%B6%AD%E6%B8%AF%E6%B5%B7%E6%99%AF/ |website=新假期 |accessdate=2019-03-21 |date=2019-03-18}}</ref>。

== 設施 ==
酒店提供413間客房、8家餐廳及酒廊、健身中心、游泳池。包括宴會大禮堂和四樓「麗府」，並設可觀看[[維多利亞港|維多利亞港]]海景的戶外草坪、7間會議廳和1,990平方呎附設私人露台的尊尚套房。

== 婚宴 ==
瑰麗酒店是現時香港最昂貴的[[婚宴|婚宴]]場所，酒店有多個宴會場地，Grand Ballroom是其中最大一間，面積大約達9200呎，最多可承辦60圍，容納約600至700人。酒店的三樓和四樓提供共 3,200 平方米 (34,450 平方呎) 的會議及活動空間。賓客可由地下白色樓梯拾級而上；在宴會廳外亦有可飽覽海景的草地，賓客可於私人花園活動場所，俯瞰[[維港|維港]][[夜景|夜景]]<ref>{{cite web |title= 【2019香港婚宴場地大比拼 酒店篇】酒席價格+新人評價+場地資料|url= https://www.bespoke-wedding.com/blog/20474-婚宴場地-酒店-香港-評價 |website=BESPOKE|accessdate=2019-03-28 |date=2019-03-28}}</ref>。

泰國前總理[[他信|他信]]幼女貝東丹（Paetongtarn Shinawatra）於2019年3月22日在瑰麗酒店舉辦婚禮<ref>{{cite news|author1=鄭嘉如 |author2= 陳潤南|title=【他信嫁女】尖東超豪婚宴最後準備　婚紗相獨家曝光|url=https://www.hk01.com/article/308973|accessdate=2019-04-07|agency=[[香港01|香港01]]|date=2019-03-21}}</ref>。
