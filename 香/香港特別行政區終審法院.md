'''香港特別行政區終審法院'''（{{lang-en|The Court of Final Appeal of the Hong Kong Special Administrative Region}}）是[[香港特別行政區|香港特別行政區]]法庭制度內的最高[[上訴法院|上訴法院]]<ref>{{cite web|url=http://www.hkcfa.hk/tc/about/overview/index.html|title=終審法院簡述 |website=香港終審法院|accessdate=2017-09-26}}</ref>，相當於其他國家的[[最高法院|最高法院]]。終審法院聆訊來自[[香港高等法院|香港高等法院]]（包括[[上訴法庭|上訴法庭]]及[[香港特別行政區高等法院原訟法庭|原訟法庭]]）的[[民事|民事]]及[[刑事|刑事]][[上訴|上訴]]案件，對香港[[司法管轄權|司法管轄權]]範圍內的訴訟有最終審判權。

香港終審法院從1997年7月1日至2015年9月在[[中環|中環]][[炮台里|炮台里]]的[[前法國外方傳道會大樓|前法國外方傳道會大樓]]辦公<ref>{{Cite web|url=http://www.amo.gov.hk/b5/monuments_37.php|title=香港法定古蹟 - 香港島|accessdate=2018-08-21|author=|date=2016-09-13|publisher=古物古蹟辦事處}}</ref>。當[[香港立法會|香港立法會]]2011年遷往[[立法會綜合大樓|立法會綜合大樓]]、而[[終審法院大樓|舊最高法院大樓]]經過復修後，終審法院於2015年9月7日遷往舊最高法院大樓<ref>{{cite news|url=http://www.info.gov.hk/gia/general/201508/20/P201508190667.htm|title=昃臣道八號終審法院大樓於九月七日啟用|date=2015-08-20|publisher=香港政府新聞網}}</ref>，而原址亦將移交其他部門或[[香港聖公會|香港聖公會]]使用。<ref>{{cite news|url=http://www.mpinews.com/htm/INews/20090416/gb41531c.htm|title=終院幾年後遷立法會大樓|publisher=明報即時新聞|date=2009-04-16|deadurl=yes|archiveurl=https://web.archive.org/web/20090417180059/http://www.mpinews.com/htm/INews/20090416/gb41531c.htm|archivedate=2009-04-17}}</ref>

== 沿革 ==
1997年[[香港主權移交|香港主權移交]][[中華人民共和國|中華人民共和國]]之前，香港的司法終審權屬於[[英國|英國]][[倫敦|倫敦]][[樞密院司法委員會|樞密院司法委員會]]<ref>{{Cite web|url=https://www.hkcfa.hk/tc/faqs/role_cfa/index.html|title=終審法院的角色|accessdate=2018-08-21|author=|date=|publisher=香港終審法院}}</ref>。為體現[[一國兩制|一國兩制]]之下香港的獨立'''終審權'''<ref name=":0">《基本法》第八十二條</ref>，香港於1997年7月1日成立了自己的終審法院。

根據《[[香港特別行政區基本法|香港特別行政區基本法]]》，香港保留原有法律，繼續實行[[普通法|普通法]]制度。[[法院|法院]]判案時可以引用其他普通法地區（如[[英格蘭|英格蘭]]、[[澳洲|澳洲]]、[[紐西蘭|紐西蘭]]、[[加拿大|加拿大]]）的案例<ref>《基本法》第八十四條</ref>，亦可聘用其他普通法地區的司法人員<ref name=":0" />。

== 釋法權 ==
{{see also|全國人大常委會解釋香港基本法}}
香港終審法院對除了《[[香港特別行政區基本法|香港特別行政區基本法]]》外的[[香港法律|香港法律]]具有最終解釋權<ref>{{Cite web|url=http://www.info.gov.hk/gia/general/199912/12/1212134.htm|title=司法獨立及普通法制度受《基本法》保障|accessdate=|author=|date=1999-12-12|publisher=香港特區政府}}</ref>。基本法第158條規定，基本法解釋權屬於中華人民共和國[[全國人民代表大會常務委員會|全國人民代表大會常務委員會]]。

全国人大常委会授權香港法院自行解釋基本法關於香港[[自治|自治]]範圍內的條款。然而，凡涉及中央人民政府管理的事務，或中央和香港特別行政區關係的條款，則必須先由終審法院提請全國人大常委會作出解釋。<ref>《基本法》第一百五十八條</ref>

由於香港終審法院解釋基本法的權力源自全国人大常委会，全国人大常委会對基本法的解釋可推翻香港終審法院對基本法的解釋，但對在解釋前已作出的終審判決不受影響。

== 爭議 ==
{{expand-section|time=2015-09-28}}
{{main|香港居留權爭議|全國人大常委會解釋香港基本法}}
1999年1月29日，香港終審法院就[[香港居留權爭議|香港居留權爭議]]作出的裁決，導致第一次[[全国人大常委会解释香港特别行政区基本法|全国人大常委会解释香港特别行政区基本法]]，有人認為釋法大大衝擊了香港的法治，香港司法界發起黑衣遊行抗議人大釋法，此次釋法後，這亦意味著日後具爭議的案件在終審法院作出判決後，除終審法院外，香港政府亦可以提請人大釋法，甚至由全国人大常委会直接主動進行釋法。

== 法官 ==
=== 委任 ===
根據《[[香港基本法|基本法]]》及《香港終審法院條例》，終審法院法官須根據[[司法人員推薦委員會|司法人員推薦委員會]]推薦，由[[香港行政長官|行政長官]]任命。行政長官就終審法院法官的任命或免職，須徵得[[香港立法會|立法會]]同意及報[[全國人大常委會|全國人大常委會]]備案。

===資格===
# 任何以下人士均有資格獲委任為首席法官：
#* 常任法官；
#* 高等法院首席法官、上訴法庭法官或原訟法庭法官；或
#* 在香港以大律師或律師身分執業最少10年的大律師。
# 任何以下人士均有資格獲委任為常任法官
#* 高等法院首席法官、上訴法庭法官或原訟法庭法官；
#* 在香港以大律師或律師身分執業最少10年的律師。 
# 任何以下人士均有資格獲委任為非常任香港法官，不論他是否通常居住於香港
#* 已退休的高等法院首席法官； 
#* 已退休的終審法院首席法官；
#* 已退休的終審法院常任法官；
#* 現職或已退休的上訴法庭法官；或
#* 在香港以大律師或律師身分執業最少10年的大律師。
# 任何符合以下條件的人士均有資格獲委任為其他普通法適用地區法官
#* 屬其他普通法適用地區的民事或刑事司法管轄權不設限的法院的現職或已退休法官者；
#* 他通常居住於香港以外地方；
#* 他從未在香港擔任過高等法院法官、區域法院法官或常任裁判官。

此外，首席法官須由在外國無居留權的[[香港永久性居民|香港永久性居民]]中的[[中國公民|中國公民]]擔任。

===任期===
在獲委任時未年滿65歲的首席法官及常任法官須於年滿65歲時離任，但任期可由行政長官分別根據司法人員推薦委員會或首席法官的建議延期不超過2次，每次3年。而在獲委任時已達65歲的首席法官及常任法官，其任期為3年，其後可再延續3年。

非常任法官沒有指定退休年齡，他們的任期為3年，但行政長官可根據首席法官的建議，將任期不限次數延續，每次續期為3年。
