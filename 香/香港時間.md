'''香港時間'''或稱為'''香港標準時間'''（縮寫'''HKT'''）是[[香港|香港]]所採用的時間標準，由-{[[香港天文台|香港天文台]]}-負責管理。香港標準時間全年比[[國際協調時|國際協調時]]快8小時，所以香港的標準時間與同樣位於[[UTC+8|UTC+8]]時區的[[國家|國家]]及[[地區|地區]]相同，當中包括[[中國標準時間|北京]]（CST）、[[澳門標準時間|澳門]]（MOT）、[[台北時間|台北]]（NST）、[[馬來西亞標準時間|吉隆坡]]（MYT）及[[新加坡標準時間|新加坡]]（SGT）。香港在1904年使用[[格林威治標準時間|格林威治標準時間]]，1972年採用[[國際協調時|國際協調時]]。當國際協調時間為凌晨0時0分時，香港是上午8時整。香港時間比[[泰國時間|曼谷]]的（[[UTC+7|UTC+7]]）快1小時，較[[日本時間|東京]]、[[平壤時間|平壤]]及[[韓國標準時間|首爾]]的（[[UTC+9|UTC+9]]）慢1小時。

== 歷史 ==
香港時間的授時服務，是香港天文台從1883年成立至今的主要職責。早期香港天文台使用[[赤道儀|赤道儀]]及中星儀，透過觀測星象測量時間。當時香港時間是[[地方平時|當地平均時間]]（LMT）UTC+7:36:42（準確值為UTC+7:36:41.8842）。香港天文台最早於1885年1月1日對公眾授時<ref>[http://www.hko.gov.hk/gts/time/History_of_HK_Time_Servicec.htm 香港授時服務歷史]，香港天文台，2014年2月25日</ref>，當年香港天文台於九龍尖沙咀警署設置桅杆，以升降時間球的方式對外發布時間。1885年1月1日，香港天文台於中午12時50分把時間球升到桅杆頂端，然後於下午1時正，首次把時間球降下，成為香港首次報時訊號，並作為香港時間的標準。用於報時的時間球訊號塔，後來於1933年因為電台報時的開展而拆除。1904年10月30日，香港時間正式確定為[[格林威治標準時間|格林威治標準時間]]快8小時（GMT+8）<ref>{{cite web|title=香港政府憲報|url=http://sunzi.lib.hku.hk/hkgro/view/g1904/488317.pdf|accessdate=2018-05-11|date=1904-10-21}}</ref>。2004年，天文台安裝了一套高準確度授時系統，利用[[全球定位系統|全球定位系統]]共視方法，向國際度量衡局提供天文台的原子鐘時間數據，參與訂定協調世界時。天文台亦根據國際度量衡局提供的時間數據調校原子鐘，使其準確度保持在一百萬分之一秒以內。現時，香港天文台以[[銫|銫]][[原子鐘|原子鐘]]報時系統作為香港時間的標準，誤差僅為每日1[[微秒|微秒]]之內。香港天文台設有互聯網時間[[伺服器|伺服器]]，為互聯網的用戶提供準確的時間校正服務<ref>[http://www.info.gov.hk/gia/general/200309/04/0904109.htm 天文台原子鐘服務市民和電子商貿]，香港政府新聞公告，2003年9月4日</ref>。

== 夏令時間 ==
香港曾經採用[[夏令時間|夏令時間]]制度<ref>[http://www.hko.gov.hk/gts/time/Summertimec.htm 香港夏令時間]，香港天文台，2014年6月19日</ref>，即是[[夏季|夏季]]期間時鐘顯示的時間會較正常時間快一小時，並於[[冬季|冬季]]前的指定日子，把時鐘調校回又稱冬令時間的正常時間。香港的夏令時間制度最早於1941年4月1日至9月30日期間實施。香港於1942年至1945年被日本佔領期間，日本在香港全年實施夏令時間，令香港的時間與[[日本標準時|東京]]的時間一致<ref>[http://hk.apple.nextmedia.com/realtime/news/20131102/51864199 1952年香港仲有夏令時間]，蘋果日報，2013年11月02日</ref>。日本戰敗後，香港恢復使用比格林威治時間快8小時作為標準時間。從1946年起，香港再次於夏季採用夏令時間<ref>[http://cablenews.i-cable.com/ci/index.php/VideoPage/program/114381/%E6%96%B0%E8%81%9E%E9%80%9A%E8%AD%98/%E9%A6%99%E6%B8%AF%E6%9B%BE%E7%B6%93%E5%AF%A6%E8%A1%8C%E5%A4%8F%E4%BB%A4%E6%99%82%E9%96%93 香港曾經實行夏令時間]，cablenews.i-cable.com，2010-2-10</ref>，夏令時間一般設定於每年的4月至10月期間（除1973-74年在12月30日開始），1977年及1978年則沒有採用夏令時間，1979年5月至10月再次使用夏令時間，也是香港最後一次採用夏令時間。1980年至今，香港再沒有實施夏令時間。由於[[香港地理|香港地理]]位置所處的[[緯度|緯度]]不高，所以香港冬夏兩季的日照時間，與長年實施夏令時間的[[歐洲|歐洲]]及[[美加|美加]]地區相比，日照時間的長度差距並不顯著。雖然香港在[[夏至|夏至]]的日照時間比[[冬至|冬至]]長了約2小時43分鐘<ref>[http://www.timeanddate.com/astronomy/hong-kong/hong-kong Today's Sun Position in Hong Kong]，www.timeanddate.com，2015-12-20</ref>，但執行夏令時間須要每年調校時鐘兩次，容易造成混亂，把夏令時間的效益抵消。

== 電視及電台的報時 ==
香港的[[電視廣播有限公司|無綫電視]][[翡翠台|翡翠台]]和[[明珠台|明珠台]]及已停止大氣電波服務的[[亞洲電視|亞洲電視]][[本港台|本港台]]和[[亚洲电视国际台|國際台]]在每日數個時段為觀眾報時，包括星期日下午播出[[Keroro軍曹_(動畫)|KERORO軍曹]]，午間新聞及六點半新聞前。[[香港電台|香港電台]]及商業電台會每半個小時報時一次，通常在整點和30分發出報時廣播。[[港台電視31|港台電視31]]會以不定期播出時間而讀出報時時段。[[ViuTV|ViuTV]]和[[奇妙電視|奇妙電視]]開台至今無提供報時。另外,Now新聞台亦會不定時讀出報時.
