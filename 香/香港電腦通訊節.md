'''香港電腦通訊節'''（{{lang-en|Hong Kong Computer and Communication Festival}}，[[縮寫|縮寫]]：HKCCF），[[香港電腦商會|香港電腦商會]]主辦。[[深水埗區議會|深水埗區議會]]協辦直至到2009年。每年8月份[[香港會議展覽中心|香港會議展覽中心]]一號展覽廳舉行，目的是介紹最新的電腦科技，協助[[中小企|中小企]]發展[[外貿|外貿]]。於同一地點同時舉辦[[香港電競節|香港電競節]]和[[軟件及創意產業展覽|軟件及創意產業展覽]]。

== 小註 ==
香港電腦通訊節2008原為2008年8月22日至2008年8月25日舉行，但2008年8月22日香港受[[颱風鸚鵡_(2008年)|颱風鸚鵡]]影響，[[香港天文台|香港天文台]]發出[[九號烈風或暴風風力增強信號|九號烈風或暴風風力增強信號]]，大會曾一度宣佈會場如常開放，為吸引人流，免費入場，但最終大會於該天下午3時宣佈立即關閉會場，已入場的遊人需立即離開會場，並將展期延長至8月26日。

2017年8月27日香港受到[[強烈熱帶風暴帕卡_(2017年)|強烈熱帶風暴帕卡]]影響，香港天文台於早上發出[[八號烈風或暴風信號|八號烈風或暴風信號]]，下午1時40分改發[[三號強風信號|三號強風信號]]，大會於下午3時40分重新開放會場，為吸引人流，將原定關閉會場時間推延到晚上11時，2017年8月28日關閉時間推延到晚上7時。

同一地點同時舉辦的[[香港電競節|香港電競節]]和[[軟件及創意產業展覽|軟件及創意產業展覽]]為香港電腦通訊節一部份，惟最後一天軟件及創意產業展覽提早一小時關閉。

== 爭議 ==
雖然展覽為「電腦通訊節」，但近年不少攤位均在促銷散貨，而非介紹或銷售最新科技產品，而且亦有攤位售賣非電腦類產品（如食物、[[巴士|巴士]][[模型|模型]]、[[鍋|煮食鍋]]、[[電飯煲|電飯煲]]、[[按摩椅|按摩椅]]等等），甚至有非電腦業組織設立攤位，推廣的是非電腦類產品，包括推廣[[信用卡|信用卡]]的[[銀行|銀行]]、[[慈善組織|慈善組織]]等等<ref>[https://www.enterpr1se.info/2017/08/hong-kong-computer-communications-festival-2017/ 企業號航行網誌]</ref>。

== 香港電腦通訊節與香港電腦節的比較 ==
{{See Also|香港電腦節}}
香港電腦通訊節承接香港電腦節的成功，主辦及協辦（香港電腦商會及[[深水埗|深水埗]][[區議會|區議會]]）於每年8月假借香港會議展覽中心一號展覽廳，一般是由星期五舉辦至翌週的星期一，為期連續舉行四天的活動。除舉辦地點和時間外，會場的面積也較大，而且較多推廣新穎的電腦和通訊的科技，更設有科技交流會議。

[[香港電腦通訊節|香港電腦通訊節]]需要購買[[門票|門票]]進場，業界人士可憑[[名片|名片]]免費入場。2011年前，[[香港電腦節|香港電腦節]]的入場費則全免。2011年起，康文署租場合約上要求，開始收取租場合約上要求最低入場費。2009年前，香港電腦通訊節有部份時間只讓業界人士進場，但香港電腦節則沒有此安排。

在香港電腦通訊節舉行期間，大會也會舉行一些活動，分別是全港IT代言人大賽、IT精英問答比賽、香港電腦商會-IT業界產品大獎頒獎禮、座談會和電腦產品拍賣會。2009年前，[[深水埗|深水埗]][[區議會|區議會]]停止協辦前，會場設立的「深水埗數碼之源」展覽，介紹和推廣深水埗。
