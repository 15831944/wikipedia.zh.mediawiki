'''香港起飛'''（英語：'''Inspiration'''，[[航空器註冊編號|航空器註冊編號]]：'''B-KOO'''）是首架在[[香港|香港]]組裝和註冊的小型飛機<ref>{{Cite news|url=http://www.info.gov.hk/gia/general/201702/09/P2017020900491.htm||title=科學館展出首架香港組裝小型飛機「香港起飛」|author=|language=zh-hk|publisher=香港政府新聞公報|date=2017-02-09|accessdate=2017-04-12}}</ref>，也是香港首架完成環球飛行的民間組裝飛機。經歷八年時間的組裝後，「香港起飛」於2015年11月15日在[[香港國際機場|香港國際機場]]試飛成功，經過七個月的測試後取得飛行認證，隨即於2016年8月從香港起飛展開環球之旅，途經20個國家和地區，於2016年11月在香港降落，成功完成為期78天，航程達五萬公里的環球飛行旅程<ref>{{Cite news|url=http://bma.singtao.com/yesterday/loc/1114ao18.html|title=「香港起飛」環球五萬公里歸來|author=|language=zh-hk|publisher=《星島日報》|date=2016-11-14|accessdate=2017-04-12}}</ref>。

== 背景 ==
「香港起飛」是首架成功環球飛行的香港飛機，飛機的製造及環球飛行計劃皆由民間籌辦，計劃的目標是推廣「不怕艱辛、鍥而不捨」的香港精神，以及啟發香港大眾對航空科技的興趣<ref>{{Cite news|url=http://www.news.gov.hk/tc/city_life/html/2017/02/20170209_170832.shtml|title=首架香港組裝小型飛機明展出|author=|language=zh-hk|publisher=香港政府新聞網|date=2017-02-09|accessdate=2017-04-12}}</ref>。環球飛行計劃由鄭楚衡（Hank）發起，他在[[國泰航空|國泰航空]]擔任客機機長。鄭楚衡自小已有夢想以自己製造的飛機環球飛行，他於是進入[[安柏瑞德航空大學|安柏瑞德航空大學]]就讀[[航空航天工程|航空航天工程]]，並參加國泰航空的見習機師培訓計劃，於2001年加入國泰航空成為二級副機長，後來晉升為[[波音777客機|波音777客機]]的機長。鄭楚衡雖然當上民航機的機長，但繼續追尋夢想，於是在2008年到美國完成修讀RV小型飛機裝嵌課程後，毅然從美國訂購一架{{tsl|en|Van's Aircraft RV-8|RV-8}}小型飛機的組件在香港自行組裝<ref>{{Cite news|url=https://www.hk01.com/%E7%A4%BE%E5%8D%80/38410/-%E9%80%99%E4%B8%80%E5%A4%A9%E4%BB%96%E5%80%91%E6%9C%83%E9%A3%9B-%E6%B8%AF%E9%A6%96%E6%9E%B6%E8%A3%9D%E5%B5%8C%E9%A3%9B%E6%A9%9F%E7%92%B0%E7%90%83%E9%81%8A-%E5%A4%A9%E5%8F%B0%E6%98%AF%E7%AC%AC%E4%B8%80%E5%80%8B-%E5%9F%BA%E5%9C%B0-|title=港首架裝嵌飛機環球遊 天台是第一個「基地」|author=|language=zh-hk|publisher=HK01|date=2016-08-21|accessdate=2017-04-12}}</ref>，期望造出首架在香港試飛及環球飛行的飛機<ref name="cx-2015-11-15">{{Cite news|url=https://www.cathaypacific.com/cx/zh_HK/about-us/press-room/press-release/2015/hong-kongs-first-homebuilt-and-registered-aircraft-takes-off.html#|title=香港首架本地裝嵌並獲認證的飛機順利啟航|author=|language=zh-hk|publisher=國泰航空|date=2015-11-15|accessdate=2017-04-12}}</ref>。

== 建立團隊 ==
鄭楚衡是一名[[飛行員|飛行員]]，對飛機有一定了解，可是即使他自掏腰包購入飛機組件，但是單憑一人之力也難以完成他自製飛機環球飛行的夢想，當中涉及組裝飛機的場地、人手、測試和認證等問題，而環球飛行更須要大筆資金<ref>{{Cite news|url=https://topick.hket.com/article/993489/%E6%A9%9F%E5%B8%AB%E8%B3%A3%E6%A8%93%E7%A0%8C%E5%B0%8F%E5%9E%8B%E6%A9%9F%E3%80%80%E4%B8%80%E9%A3%9B%E6%B2%96%E5%A4%A9%E5%9C%93%E5%A4%A2|title=機師賣樓砌小型機 一飛沖天圓夢|author=|language=zh-hk|publisher=《香港經濟日報》|date=2017-04-09|accessdate=2017-04-12}}</ref>。飛機的組裝場地是首要面對的問題，鄭楚衡必須尋找一處可供組裝飛機及停放飛機組件的場所，由於購買飛機組件已經耗用大筆款項，於是他嘗試向多所學校請求借出場地，但都遭到拒絕，後來經朋友介紹向位於[[銅鑼灣|銅鑼灣]]的[[聖保祿學校_(香港)|聖保祿學校]]申請借出場地，終於獲得校方答應。聖保祿學校是一所女校，該校除了借出場地外，鄭楚衡還得到該校的學生協助組裝飛機，而校方也嘗試為組裝飛機的計劃申請[[優質教育基金|優質教育基金]]的資助，可是基金委員會拒絕批出資助，所以鄭楚衡須要自己籌集資金，他當時向採訪的媒體表示「計劃不會因為錢而半途而廢」，有信心成功籌集足夠資金完成計劃<ref>{{Cite news|url=http://dailynews.sina.com/bg/chn/chnpolitics/mingpao/20090528/1501305189.html|title=15女生裝嵌首架港產飛機飛機師出錢出力 斥300萬料3年完工|author=|language=zh-hk|publisher=新浪新聞|date= 2009-05-28|accessdate=2017-04-12}}</ref>。

聖保祿學校借出場地及學生的參與雖然解決了場地及基本人手的問題，但飛機如要成功在香港申請試飛，就必須有專業人士參與測試，以便取得[[民航處|民航處]]的批准可在香港飛行。由於向民航處申請飛機試飛的批文涉及不少文書程序，都是作為機師的鄭楚衡不熟悉的，所以他須要一位飛機工程師協助。2008年有媒體採訪鄭楚衡將會與學生在聖保祿學校組裝香港首架私人飛機，當時是飛機工程師的達仁（Gary）留意到有關報導，於是寫信給聖保祿學校的校長表示願意提供協助。達仁曾於大學修讀[[機械工程|機械工程]]學位，並於安柏瑞德航空大學考獲私人飛機駕駛執照<ref name="cx-2015-11-15" />。達仁的參與不但為組裝飛機的計劃提供專業支援，達仁還通過在飛機工程公司工作的人脈關係，聯繫其他飛機工程相關的專業人員參與這個自行組裝飛機完成環球之旅的計劃<ref>{{Cite news|url=http://hk.apple.nextmedia.com/realtime/news/20160821/55526283|title=成功男人背後的男人 工程師拍檔全程支援|author=|language=zh-hk|publisher=《蘋果日報》|date=2016-08-21|accessdate=2017-04-12}}</ref>，逐漸組建出超過500人的團隊<ref>{{Cite news|url=https://www.hk01.com/%E6%B8%AF%E8%81%9E/39879/-%E6%9C%89%E7%89%87-%E9%80%99%E4%B8%80%E5%A4%A9%E6%88%91%E5%80%91%E6%9C%83%E9%A3%9B-%E9%A6%96%E6%9E%B6%E6%9C%AC%E5%9C%B0%E8%A3%9D%E5%B5%8C%E9%A3%9B%E6%A9%9F%E5%87%BA%E7%99%BC%E7%92%B0%E9%81%8A%E4%B8%96%E7%95%8C|title=這一天我們會飛！首架本地裝嵌飛機出發環遊世界|author=|language=zh-hk|publisher=HK01|date=2016-08-28|accessdate=2017-04-12}}</ref>。

== 原型及改裝 ==
[[File:B-KOO_HKSM_P2.jpg|thumb]]的等身比例模型，並展示飛行員配備的救生用品]]
=== 原型設計 ===
「香港起飛」的原型是美國{{tsl|en|Van's Aircraft}}設計的{{tsl|en|Van's Aircraft RV-8}}，這是一款單引擎雙座位小型螺旋槳飛機，RV-8於1995年進行首次試飛，至2012年已有1,143架RV-8完成組裝順利飛行<ref>{{Cite news|url=http://archive.am730.com.hk/article-125393|title=港嵌飛機 鬧市蓄勢待發|author=|language=zh-hk|publisher=《AM730》|date=2012-10-15|accessdate=2017-04-12}}</ref>。RV-8是Van's Aircraft發展自前代{{tsl|en|Van's Aircraft RV-4}}的機型，但長度和翼展都較前代大，可增加飛機的內部空間及載重量<ref name="WDLA11">Bayerl, Robby; Martin Berkemeier; et al: ''World Directory of Leisure Aviation 2011–12'', p. 125. WDLA UK, Lancaster UK, 2011. {{ISSN |1368-485X}}</ref>。RV-8依據起落架的設計分為兩種基本型號，分別是採用後三點式起落架的RV-8，以及採用前三點式起落架的RV-8A<ref name="rv8-general">{{Cite news|url=https://www.vansaircraft.com/public/rv8specs.htm|title=RV–8/8A Specifications|author=|language=zh-hk|publisher=Van's Aircraft|date=|accessdate=2017-04-12}}</ref>。RV-8採用傳統的單翼設計，飛機容許的最大過載是+6G和-3G。RV-8的動力來自裝設於機頭的活塞式汽油引擎，原裝版的引擎主要採用{{tsl|en|Lycoming|萊康明}}的{{tsl|en|Lycoming O-360|IO-360}}水平對臥四氣缸[[汽油引擎|汽油引擎]]，最大輸出馬力為200匹。

=== 進行改裝 ===
「香港起飛」雖然是以RV-8的組件裝嵌，但因為以完成環球飛行為目標，為適應長時間飛行，所以須要進行修改。飛機的引擎改用{{tsl|en|Engine Components Inc.}}供應的泰坦IOX-360汽油引擎，最大馬力為180匹<ref>{{Cite news|url=http://www.enginehistory.org/HOAE/EngineComponents.html|title=Engine Components, ECi, TITAN (US)|author=|language=zh-hk|publisher=|date=|accessdate=2017-04-12|archive-url=https://web.archive.org/web/20170810171704/http://www.enginehistory.org/HOAE/EngineComponents.html|archive-date=2017-08-10|dead-url=yes}}</ref>。「香港起飛」的[[螺旋槳|螺旋槳]]由位於美國[[史塔克頓_(加利福尼亞州)|史塔克頓]]的Catto Propellers供應，螺旋槳以[[木材|木材]]製造，表面有[[碳纖維|碳纖維]]覆蓋，可抵擋氣流帶來的震蕩，減輕飛行員的疲勞<ref>{{Cite news|url=http://hk.apple.nextmedia.com/realtime/news/20160922/55680069|title=橫跨5.5萬公里的螺旋槳 走進加州製造商窺探誕生過程|author=|language=zh-hk|publisher=《蘋果日報》|date=2016-09-22|accessdate=2017-04-12}}</ref>。由於RV-8的原型並非用於長途飛行，飛機原設計每次只能飛行約500海里，約為香港到台灣的距離。「香港起飛」為應付環球旅程中跨越[[太平洋|太平洋]]，從美國[[夏威夷|夏威夷]]到[[聖荷西_(加利福尼亞州)|聖荷西]]長達2,000海里的最長航程<ref name="apple-20160822a">{{Cite news|url=http://hk.apple.nextmedia.com/news/art/20160822/19745406|title=機尾髹獅子山圖案|author=|language=zh-hk|publisher=《蘋果日報》|date=2016-08-22|accessdate=2017-04-12}}</ref>，所以需要增加燃料搭載量，於是駕駛艙的後座改為三個油箱，使飛機的燃料存量增加近兩倍至460公升，令飛機的航程顯著增加，但這個超出原廠設計的修改，也令駕駛座的後方就是油箱，所以須要加裝防漏油的設施，減低飛行期間面對的風險<ref name="takungpao-20160829">{{Cite news|url=http://news.takungpao.com.hk/hkol/topnews/2016-08/3362554.html|title=籌備八載環球三月香港首架自行組裝飛機啟航|author=|language=zh-hk|publisher=《大公報》|date=2016-08-29|accessdate=2017-04-12}}</ref>。

「香港起飛」為因應將會進行的越洋飛行，也加強[[導航系統|導航系統]]及[[無線通訊|通訊系統]]的配備，即使飛機飛到世界各地，也可把飛行速度、高度、位置及剩餘燃料等數據，傳送到設於香港的地面運作中心進行分析，如有發生異常情況，可以及早對飛行員作出警報<ref name="apple-20160828b">{{Cite news|url=http://hk.apple.nextmedia.com/realtime/news/20160828/55563569|title=最強地面運作中心 支援港產飛機環球創舉|author=|language=zh-hk|publisher=《蘋果日報》|date=2016-08-28|accessdate=2017-04-12}}</ref>。RV-8小型飛機的基本造價約為150萬港元<ref name="apple-20160828c">{{Cite news|url=http://hk.apple.nextmedia.com/realtime/news/20160828/55563812|title=夢想成真！港產飛機成功起飛展開環球之旅|author=|language=zh-hk|publisher=《蘋果日報》|date=2016-08-28|accessdate=2017-04-12}}</ref>，但「香港起飛」需要進行多項改裝增加航程及加強航電配備，所以鄭楚衡除了自資購買飛機的組件外，也尋求廠商的贊助。「香港起飛」的[[航電|航電系統]]得到{{tsl|en|Dynon Avionics}}的贊助，導航軟件及配套的數據則由[[傑普遜公司|傑普遜（Jeppesen）]]贊助提供<ref name="sponsors">{{Cite news|url=http://www.inspiration.bkoo.hk/sponsors|title=Sponsors and Supporting Organizations|author=|language=zh-hk|publisher=Inspiration Team|date=|accessdate=2017-04-12}}</ref>。

由於「香港起飛」需要爭取飛行高度節省燃料消耗，但為免高空空氣稀薄導致飛行員[[缺氧|缺氧]]，所以駕駛艙備有氧氣面罩，飛行員在飛行期間會使用儀器持續量度血液的含氧量，以便適時戴上氧氣面罩防止缺氧，同時機艙設有[[尿液|尿液]]收集器以便長時間飛行<ref name="apple-20160911a">{{Cite news|url=http://hk.apple.nextmedia.com/realtime/news/20160911/55628198|title=靠飛行高度控制機艙氣溫 特製玻璃過濾紫外光防曬傷|author=|language=zh-hk|publisher=《蘋果日報》|date=2016-09-11|accessdate=2017-04-12}}</ref>。

== 組裝 ==
裝嵌「香港起飛」的RV-8小型飛機組件是向美國{{tsl|en|Van's Aircraft}}購入，在香港組裝的大部分時間都是在位於[[銅鑼灣|銅鑼灣]]的[[聖保祿學校_(香港)|聖保祿學校]]校舍內進行<ref name="《AM730》-20121015">{{Cite news|url=http://archive.am730.com.hk/article-125393|title=港嵌飛機 鬧市蓄勢待發|author=|language=zh-hk|publisher=《AM730》|date=2012-10-15|accessdate=2017-04-12}}</ref>。鄭楚衡從2008年起訂購RV-8飛機的組件，並於2009年開始進行飛機組裝工作。最早期的組裝工場位於聖保祿學校六樓一個細小的工具室，鄭楚衡一邊帶領該校學生組裝飛機，一邊通過網絡及電話向美國的組件生產商購入飛機零件。隨著大型組件的組裝，機身、主翼及尾翼也逐漸成形<ref name=mingpao-20151116">{{Cite news|url=http://news.mingpao.com/pns/dailynews/web_tc/article/20151116/s00002/1447610173922|title=7年籌備 機師圓夢 土產「香港起飛」起飛了|author=|language=zh-hk|publisher=《明報》|date=2015-11-16|accessdate=2017-04-12}}</ref>，須要更大的組裝空間，所以組裝工作改到學校的一樓進行，並需要打通兩個標準課室來擺放飛機組件<ref name="《AM730》-20121015" />，單是聖保祿學校就有超過200名學生參與組裝飛機的工作<ref name="metrodaily-20161216">{{Cite news|url=http://www.metrodaily.hk/metro_news/%E9%A6%99%E6%B8%AF%E8%B5%B7%E9%A3%9B-%E6%A9%9F%E5%B8%AB%E7%A0%94vr%E6%8A%80%E8%A1%93-%E9%A3%9B%E5%BE%80%E4%B8%80%E5%B8%B6%E4%B8%80%E8%B7%AF%E5%9C%8B%E5%AE%B6/|title=香港起飛 機師研VR技術 飛往一帶一路國家|author=|language=zh-hk|publisher=《都市日報》|date=2016-12-16|accessdate=2017-04-12}}</ref>。

「香港起飛」在聖保祿學校進行五年組裝後，飛機的大型組件，包括機身、主翼及尾翼都逐漸完成，但受制於空間及後期測試須要大量專業儀器，因此飛機的總裝及地面測試則須要另覓場地。由於參與組裝「香港起飛」的專業人員，包括擔任飛機工程師的達仁，大多是來自[[港機工程|港機工程]]的工程人員，他們成功通過人脈關係，得到港機工程的高層支持「香港起飛」的製作計劃，[[港機工程|港機工程（HAECO）]]決定借出飛機庫的3,000平方呎空間供「香港起飛」進行總裝<ref name="casid-20140618">{{Cite news|url=http://www.casid.org.tw/NewsView01.aspx?NewsID=3853e4b8-ebb0-4b78-ade3-c639daa01064|title=民航處開綠燈 港嵌飛機准飛|author=|language=zh-tw|publisher=經濟部航空產業發展推動小組|date=2014-06-18|accessdate=2017-04-12}}</ref>。

隨著「香港起飛」的大型組件成形，要把飛機的組件運到位於[[香港國際機場|香港國際機場]]的維修基地飛機庫也遇到困難，學校的校舍沒有足夠空間讓「香港起飛」的半成品從課室運出學校。於是聖保祿學校的校方決定把一樓用作飛機製作室的課室，面向大街的一排共20個玻璃窗全數拆除，再在窗外與大街之間搭建一條臨時橋樑，以便飛機組件順利滑出學校，再經貨車運載到港機工程位於[[赤鱲角|赤鱲角]]的飛機庫完成各大型組件的結合<ref name="《AM730》-20121015" />。「香港起飛」的後期製作，包括飛機的總裝、裝配引擎及電子設備，以及進行地面測試，都是在[[香港國際機場|香港國際機場]]的維修基地，港機工程所屬的飛機庫內進行<ref>{{Cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20160828/937382/|title=香港夢想起飛 港產飛機挑戰環球之旅|author=|language=zh-hk|publisher=《蘋果日報》|date=2016-08-28|accessdate=2017-04-12}}</ref>。「香港起飛」的組裝工作以引擎的裝嵌最為困難，「香港起飛」所用的是[[往复式发动机|活塞引擎]]，但香港各家航空公司所使用的大型民航機都採用[[噴射引擎|噴射引擎]]，所以工程人員都缺乏活塞引擎的裝嵌經驗，只能依靠說明書和原廠技術支援才可完成<ref name=apple-20140618">{{Cite news|url=http://hk.apple.nextmedia.com/news/art/20140618/18764804|title=土產飛機明年衝上雲霄 機師6年砌出夢想|author=|language=zh-hk|publisher=《蘋果日報》|date=2014-06-18|accessdate=2017-04-12}}</ref>。「香港起飛」由2008年只有散件開始組裝，到2014年底組裝工作大致完成，即將進入地面測試及飛行認證程序。組裝完成後的「香港起飛」以白色為基本塗裝，垂直尾翼髹上象徵香港的[[獅子山_(香港)|獅子山]]圖案<ref name="apple-20160822a" />，而機頭發動機罩的兩側漆有各贊助機構的標誌<ref name="sponsors" />。

== 飛行測試 ==
過往在香港飛行的私人小型飛機，都是在外國取得飛行認證後，才可在香港境內飛行，當中包括懸掛於[[香港國際機場一號客運大樓|香港國際機場一號客運大樓]]天幕的「沙田精神號」<ref name="sina-20090528">{{Cite news|url=http://dailynews.sina.com/bg/chn/chnpolitics/mingpao/20090528/1501305188.html|title=申請在港試飛 須符航空條例|author=|language=zh-hk|publisher=新浪新聞|date=2009-05-28|accessdate=2017-04-12}}</ref>。這架仿古飛機是由一家位於美國德州的承造商製造，於1997年完成建造及試飛，並獲得[[民航處|民航處]]及[[香港機場管理局|香港機場管理局]]批准，在[[赤鱲角|赤鱲角]]新機場開幕前進行一場飛行表演，慶賀香港國際機場的啟用。「香港起飛」的情況並不相同，這是首架於香港境內由散件組裝而成的私人小型飛機，雖然飛機的原型設計來自RV-8，但「香港起飛」經過增加航程的改裝，「香港起飛」也未曾在外地取得飛行認證<ref name=apple-20140618" />。因為香港從未有本地總裝的私人飛機申請在香港進行試飛，所以民航處對於「香港起飛」在香港空域進行試飛及申請認證的態度甚為審慎<ref name="on-20160606">{{Cite news|url=http://hk.on.cc/hk/bkn/cnt/news/20160606/bkn-20160606003006507-0606_00822_001.html|title=民航處刁難 禁首架港嵌飛機香港試飛|author=|language=zh-hk|publisher=《東方日報》|date=2016-06-06|accessdate=2017-04-12}}</ref>。由於香港國際機場航班頻密，要安排飛機試飛並不容易<ref name="ezone-20151111">{{Cite news|url=http://www.e-zone.com.hk/channelnews.php?id=9847|title=港產小型飛機周日試飛喇！|author=|language=zh-hk|publisher=e-zone|date=2015-11-11|accessdate=2017-04-12}}</ref>。民航處前任處長林光宇認為，香港的空域難以容納小型飛機，又建議在[[石崗機場|石崗機場]]進行試飛，獲民航處批准的機會較大<ref name="metrohk-299747">{{Cite news|url=http://www.metrohk.com.hk/pda/pda_detail.php?id=299747|title=環球夢無期？民航處未批 「香港起飛」飛唔起？|author=|language=zh-hk|publisher=《都市日報》|date=|accessdate=2017-04-12}}</ref>。由於預期民航處不會輕易批准小型飛機在香港進行試飛，「香港起飛」的飛機工程師達仁早在飛機組裝開始後不久，便於2009年向民航處提交計劃書，希望民航處批准「香港起飛」在香港進行試飛及申請認證，但民航處一直未有就「香港起飛」的飛行測試作出回應，直至2013年終獲民航處回覆批准飛機在香港試飛<ref name="metrohk-299747" />。為免影響香港國際機場的航班運作，試飛會於早上8時前進行<ref name="metrohk-299747" />。

=== 首次試飛 ===
2015年11月15日早上，「香港起飛」在香港國際機場的北跑道進行首次試飛，由計劃的發起人鄭楚衡親自駕駛<ref>{{Cite news|url=http://fashion.ettoday.net/news/597858|title=歷經7年組裝 首架港產小型飛機「香港起飛」試飛成功|author=|language=zh-tw|publisher=東森新聞|date=2015-11-16|accessdate=2017-04-12}}</ref>。試飛歷時15分鐘，在空中盤旋三圈，飛機一度爬升至三千呎，其後順利降落<ref name="on-20151116">{{Cite news|url=http://orientaldaily.on.cc/cnt/news/20151116/00176_025.html|title=港產飛機 首航成功|author=|language=zh-hk|publisher=《東方日報》|date=2015-11-16|accessdate=2017-04-12}}</ref>，「香港起飛」成為首架在完全香港組裝並成功試飛的小型飛機。

=== 飛行認證 ===
「香港起飛」首次試飛成功後，便須要就申請飛行認證進行更多測試。香港的民用飛機認證主要分為適航證和特許飛行證兩種，其中適航證是國際通行的證件，而特許飛行證只可於香港境內飛行。香港產製的飛機只能申請特許飛行證，如要取得長期特許飛行證則要完成多次試飛<ref name="metrohk-299747" />。「香港起飛」雖然首次試飛成功，但如要取得飛行認證，便須要進行25小時的飛行測試，民航處卻一直沒有批准，並指出容許「香港起飛」進行首次試飛僅為酌情批准，並表示香港缺乏合適場地進行長時間的飛行測試<ref name="on-20160606" />。由於「香港起飛」自首次試飛成功後，近半年時間沒有再獲批准試飛，為免影響原定在2016年內展開的環球飛行計劃，「香港起飛」的製作團隊於是在2016年5月初把飛機送往澳洲進行試飛，以另一途徑證明飛機性能及安全達標，試飛需時約為一至兩個月，並由當地民航局發出飛行證，然後再於香港向民航處提交澳洲試飛文件作為佐證，申領香港的特許飛行證<ref name="on-20160606" />。民航處終於在2016年8月中旬批准飛機在香港註冊<ref name="hkgov-20160828">{{Cite news|url=http://www.news.gov.hk/tc/categories/admin/html/2016/08/20160828_121330.shtml|title=香港起飛開展環球之旅|author=|language=zh-hk|publisher=香港政府新聞網|date=2016-08-28|accessdate=2017-04-12}}</ref>，[[航空器註冊編號|航空器註冊編號]]為'''B-KOO'''，並名為「香港起飛 Inspiration」<ref name="apple-20160828c" />。

== 環球飛行 ==
=== 準備工作 ===
「香港起飛」取得民航處的飛行認證後，隨即於2016年8月底展開環球飛行之旅<ref>{{Cite news|url=http://hk.apple.nextmedia.com/realtime/news/20160821/55526282|title=港產飛機環球夢啟航：香港人 沒有不可能|author=|language=zh-hk|publisher=《蘋果日報》|date=2016-08-21|accessdate=2017-04-12}}</ref>。由於預期旅程會有不少風險，所以「香港起飛」的團隊在等候飛行認證的同時，也在香港籌備支援是次環球飛行的地面運作中心。「香港起飛」的地面運作中心位於香港國際機場[[國泰城|國泰城]]的一處角落。運作中心由六台電腦組成，支援團隊的成員自行編寫[[電腦程式|電腦程式]]，使電腦能夠在同一視窗顯示多項重要的即時資訊，並投射在房間的巨型[[顯示屏|顯示屏]]上，方便團隊緊密監察飛機狀況。在環球飛行期間，地面運作中心會為駕駛「香港起飛」的鄭楚衡提供資料分析及氣象預報。「香港起飛」已安裝[[全球定位系統|全球定位系統]]，每隔10分鐘便會將身處位置傳送回運作中心，中心當值人員可監察飛機是否有偏離預定航道，並判斷飛行速度及高度是否有異常，機上也裝有[[衛星電話|衛星電話]]可以和運作中心即時通話。由於「香港起飛」在強風下飛行，實際飛行速度或會比機上儀器顯示的讀數為慢，令完成航程的飛行時數增加，通過全球定位系統的追踪，並與機上儀表的讀數進行比較，令駕駛飛機的鄭楚衡能夠及早判斷是否有足夠燃料完成航程<ref name="apple-20160828b" />。由於環球之旅的大部分時間都在海上飛行，民航處於是提出在飛機加裝緊急定位信標（Emergency Locator Beacon）及攜帶自動充氣[[救生艇|救生艇]]，並要求負責駕駛「香港起飛」的鄭楚衡熟習遇上墜海時的求生程序<ref name="takungpao-20160829" />。

== 展出 ==
「香港起飛」於2017年2月10日至4月19日期間，在[[香港科學館|香港科學館]]名為「香港起飛．夢成真」的展覽中展出<ref>{{Cite news|url=http://www.info.gov.hk/gia/general/201702/09/P2017020900491.htm|title=科學館展出首架香港組裝小型飛機「香港起飛|author=|language=zh-hk|publisher=香港政府新聞公報|date=2017-02-09|accessdate=2017-04-12}}</ref>，市民除可以參觀這架完全在香港組裝的小型飛機，還可觀看機師的飛行日誌、裝嵌飛機的工具和配件，以及飛機上的儀器及緊急救生用品等裝備，而「香港起飛」團隊的成員，包括機師鄭楚衡、工程師達仁，以及地面支援中心人員，會於特定時段與在場參觀的市民進行經驗分享<ref>{{Cite news|url=http://hk.on.cc/hk/bkn/cnt/news/20170206/bkn-20170206120348726-0206_00822_001.html|title=環繞地球後新任務 香港起飛科學館展出|author=|language=zh-hk|publisher=《東方日報》|date=2016-02-06|accessdate=2017-04-12}}</ref>。

== 「香港起飛」諸元 ==
「香港起飛」的原型為{{tsl|en|Van's Aircraft RV-8}}，由於「香港起飛」須要因應環球飛行而進行修改，引擎改用{{tsl|en|Engine Components Inc.}}的IOX-360，原位於駕駛席後方的一人乘客座位可改為加裝油箱，因此以下的性能資料只供參考。

資料來源：Van's Aircraft<ref name="rv8-general" />，Inspiration<ref>{{Cite news|url=http://www.inspiration.bkoo.hk/about-us|title=Inspiration - About Us|author=|language=zh-hk|publisher=Inspiration Team|date=|accessdate=2017-04-12}}</ref>

*'''全長：'''6.40 m（21 ft 0 in）
*'''翼展：'''7.32 m（24 ft 0 in）
*'''全高：'''1.7 m（5 ft 7 in）
*'''翼面積：'''10.8 m²（116 sq ft）
*'''空重：'''508 kg（1,120 lb）
