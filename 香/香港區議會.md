{{香港政治}}
{{Otheruses|subject=香港各區的區議會|other=與區議會同名的其他主題|區議會 (消歧義)}}

'''香港區議會'''（{{lang|en|'''District Council'''}}，簡稱'''區會'''）是[[香港|香港]][[香港行政區劃|地區]]層次的[[地方議會|地方議會]]和區域組織，就市民日常生活事務向[[香港特別行政區政府|政府]]提供意見。現時香港[[香港行政區劃|18區]]各有其所屬的區議會，合共458名[[區議員|區議員]]，每屆區議會任期4年。

區議會在香港政治架構中只是區域諮詢組織，並無法定權力，尤其不具有現代議會最重要的兩項權力：[[立法|立法]]和審批政府[[公共開支|公共開支]]的權力（相關權力由立法會行使）。香港並沒有區域性的[[法律|法律]]及[[稅收|稅收]]，只有在立法會個別授權下才有部分立法權<ref>例如2007年《吸煙及健康（修訂）條例》，[[香港立法會|立法會]]授權區議會自行決定區內公園的禁煙範圍</ref>。現時[[建制派|建制派]]於全港十八區區議會佔有優勢，並佔有全部十八個區議會主席席位及十七個副主席席位。

香港的地區政策為第一層全港整體規劃，第二層市區城區規劃，然後第三層分區規劃大綱<ref name="reginaip">[http://www.reginaip.hk/b5_legco_details.php?id=137 葉劉淑儀議員於立法會會議上就"提升區議會的地區規劃權責"議案發言]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，香港政府只下放少量監察權力和責任給區議會<ref name="reginaip"/>，一個區議員的月薪有約三萬一千港元<ref name="reginaip"/>，連同實報實銷營運開支津貼標準，每月可動用的資金約七萬港元（以上兩個金額皆為2018年的數字）。

== 香港區議會的演變 ==
全港首個區議會是[[觀塘區議會|觀塘區議會]]，當時區議會之英文名稱為「'''District Board'''」，而首屆區議會選舉在[[1982年香港區議會選舉|1982年]]舉行。

在首屆區議會選舉以前，區議會議員都是委任的；包括[[市政局|市政局]]議員、[[鄉事委員會|鄉事委員會]]主席、區內之分區委員會主席以及地區管理委員會之主要委員<ref name="書8"/>。其時區議會的官守議員和非官守議員人數大致相若，而民選議員和委任非官守議員人數也接近。當中市政局議員出任市區區議會的議員，鄉事委員會主席出任新界區議會的議員，區議會主席由當區政務專員擔任。因應[[地方行政白皮書|地方行政白皮書]]建議，區議會選舉的選民包括21歲或以上同時居港至少滿7年的人，而候選人須居港滿10年。選區按人口和地理因素來劃分，有單議席及雙議席選區，以「票數領先者取勝」選出當選人。

[[1985年香港區議會選舉|1985年區議會選舉]]以後，議席因應人口增長而小量增加，區議會主席由各區區議員互選產生。所有官員不再出任區議會議員，改以列席形式回答議員的提問。民選議員和委任議員的整體比例仍然維持大約二比一。1985年9月，立法局56名議員中，12名是由市政局、臨時區域議局及區議會成員分組組成的選舉團選出，選舉團成員須已登記為選民，並已在香港居住最少10年。

1986年[[區域市政局|區域市政局]]正式成立，新界各個區議會各自選出一名成員出任區域市政局議員新界。直到1989年，市政局議員停止出任市區區議會當然議員，改為每個市區區議會各選出一名成員出任市政局議員，與新界區議會看齊。

1993年起，區議會負責監督[[社區會堂|社區會堂]]的管理工作及推薦有關人士擔任所屬地區多個組織的職務。

[[1994年香港區議會選舉|1994年香港區議會選舉]]中，除27名鄉事委員會主席保留新界區議會當然議席之外，373名議員幾乎全部均由地方選區以「[[單議席單票制|單議席單票制]]」和「票數領先者取勝」的方式選出。投票年齡由21歲降低至18歲。

由於英方未能與中方達成協議，所以在1997年7月1日至1999年12月31日成立臨時區議會取代1994年區議會選舉中組成的區議會，但與立法局在1997年6月30日解散不同，原有區議會議員得到行政長官[[董建華|董建華]]委任，全部過渡至臨時區議會，並加入四分一新成員<ref>{{cite web|url=http://www.gov.hk/tc/about/abouthk/factsheets/docs/district_admin.pdf|title=香港便覽 地區行政|year=2015}}</ref>。

《區議會條例草案》於1998年12月16日提交立法會，取代《臨時區議會條例》。當中增加民選議席數目之餘，亦保留委任及當然議席。於1999年3月10日立法會會議上獲得通過。<ref>立法會年報(Page.19) 1998 至 1999 年度 http://www.legco.gov.hk/general/chinese/sec/reports/arpt.htm</ref>

[[1999年香港區議會選舉|1999年香港區議會選舉]]，是[[香港主權移交|香港主權移交]]後的首次區議會選舉，[[建制派|建制派]]憑藉委任議席支持下取得大部分區議會的控制權。

2000年[[香港特別行政區政府|特區政府]]實行區域組織政治改革，廢除原有的[[市政局|市政局]]、[[區域市政局|區域市政局]]、康體發展局及兩個市政總署。當時政府承諾向區議會增撥資源、增加其職責，以鼓勵市民參與地區公共事務；同時將區議會英文名稱改為「District Council」，以示其享有「立法議會」（Council）的同等地位，與原來市政局（Urban Council）、區域市政局（Regional Council）看齊。不過特區政府並未有實現承諾，讓區議承繼兩個市政局及市政總署原有的權力。由於沒有實質權力，很多人認為區議會已淪為只能讓議員大放厥詞，討論多於成效的「口水會／吹水會」。

2005年10月，[[香港特別行政區行政長官|行政長官]][[曾蔭權|曾蔭權]]在任內首份的[[施政報告|施政報告]]宣佈<ref>http://www.policyaddress.gov.hk/05-06/chi/p19.htm</ref>，宣佈將會增加區議會權力，讓其負責參與管理區內的[[圖書館|圖書館]]、[[社區會堂|社區會堂]]、[[游泳池|游泳池]]等文娛康樂設施，但實際上仍由[[康樂及文化事務署|康樂及文化事務署]]執行。此改變於2007年在[[屯門區|屯門]]、[[灣仔區|灣仔]]、[[黃大仙區|黃大仙]]和[[西貢區|西貢]]四區試行，隨後於2008年第三屆區議會推展到全港十八區。

2013年5月22日，立法會通過《2013年區議會（修訂）條例草案》，在2016年1月組成新一屆區議會時取消所有委任議席，但保留由鄉村間選的27個當然議席，並把香港島區議會議席進行重組<ref>http://rthk.hk/rthk/news/expressnews/news.htm?expressnews&20130522&55&923632</ref>，[[灣仔區議會|灣仔區議會]]增加2席，[[東區區議會|東區區議會]]減少2席，並把最大區議會地位轉讓予觀塘區議會。

==區域沿革==
1982年香港政治制度實行之初，香港政府將全港劃分為18個區議會，分別是香港島4區：中西區、東區、南區及灣仔區；九龍6區：九龍城區、觀塘區、深水埗區、黃大仙區、[[油尖區|油尖區]]及[[旺角區|旺角區]]（其中油尖區及旺角區以[[登打士街|登打士街]]為界，而[[油麻地避風塘|油麻地避風塘]]屬油尖區，其餘區界與現時油尖旺區區界一樣）；新界8區：離島區、北區、西貢區、沙田區、大埔區、荃灣區（包括現時葵青區的範圍）、屯門區、元朗區。

到1985年，整個荃灣區人口已達60萬，相比當時全港不足500萬的人口，實在過於龐大。因此政府以[[德士古道|德士古道]]、[[青山公路|青山公路]]荃灣段、[[國瑞路|國瑞路]]及[[和宜合道|和宜合道]]為界，由荃灣區分拆出葵涌及青衣，成立區議會，使香港分為19區。但分拆後葵青區人口40多萬，而荃灣區人口不足20萬，形成「仔大過乸」（粵語俚語，即兒子比母親大）的現象。

由於全港人口在1980年代起向新界[[香港新市鎮|新市鎮]]遷移，九龍區人口大大下降。1994年，旺角區議會包括委任議員後仍只有11名議員，政府以善用資源為由，將[[油尖區議會|油尖區議會]]及[[旺角區議會|旺角區議會]]合併，由[[油尖旺區議會|油尖旺區議會]]代替原有兩個區議會，使香港回復為18區，形成現時局面。

2006年政府提議將[[荔景山路|荔景山路]]以南的[[盈暉臺|盈暉臺]]劃入深水埗區，消除盈暉臺「一屋苑兩區」的問題<ref>http://www.legco.gov.hk/yr05-06/chinese/counmtg/hansard/cm0621-translate-c.pdf</ref><ref>http://www.legco.gov.hk/yr05-06/chinese/hc/papers/hc0616cb3-571-c.pdf</ref><ref>http://www.legco.gov.hk/yr05-06/chinese/hc/minutes/hc060616.pdf</ref><ref>http://www.legco.gov.hk/yr05-06/chinese/hc/papers/hc0616ls-75-c.pdf</ref>。其後2013年，將東區內的[[天后|天后]]及[[維多利亞公園|維園]]選區劃入灣仔區，並且把東區區議會範圍縮至炮台山，以平衡兩區的議員數目<ref>http://www.legco.gov.hk/yr13-14/chinese/subleg/brief/sc52_brf.pdf</ref><ref>http://www.legco.gov.hk/yr13-14/chinese/counmtg/papers/cm0122cb3-324-c.pdf</ref><ref>http://www.legco.gov.hk/yr13-14/chinese/counmtg/voting/v20140122.pdf</ref>。

2010年代，[[九龍城區|九龍城區]]內的[[啟德|啟德]]發展區北部的[[啟晴邨|啟晴邨]]與[[德朗邨|德朗邨]]相繼入伙，加上位於舊[[啟德機場|啟德機場]]跑道南部的[[啟德郵輪碼頭|啟德郵輪碼頭]]啟用，以及[[啟德發展計劃|啟德發展計劃]]陸續實行，引起[[黃大仙區|黃大仙區]]、[[觀塘區|觀塘區]]與[[九龍城區|九龍城區]]爭奪此地管轄<ref>[http://orientaldaily.on.cc/cnt/news/20121130/00176_088.html 三區會爭奪啟德]</ref>，但政府仍決定以[[九龍城區|九龍城區]]繼續管轄[[啟德|啟德]]發展區，並設立啟德北及啟德南選區<ref>http://www.eac.gov.hk/pdf/distco/maps/dc2015g.pdf</ref>。

綜合以上事件，反映到[[香港特別行政區政府|香港特別行政區政府]]在區議會的設立和取消方面，較少向市民徵詢。另一方面區界劃分亦追不上時代步伐，如把鄰近[[荔枝角|荔枝角]]而遠離[[葵青區|葵青區]]其他部分的荔景山路以南地區分由[[深水埗區|深水埗區]]及[[葵青區|葵青區]]管理<ref>http://ssp.had.pbase.net/ssp_d/chinese/doc/report.of.survey/Publicity/Report-Chi.pdf{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；2016年之前，以[[告士打道|告士打道]]為界把[[銅鑼灣|銅鑼灣]]分劃為灣仔區及東區（在八十年代的區議會選舉中灣仔區設有銅鑼灣中選區，而[[東區|東區]]則設有銅鑼灣南及銅鑼灣北選區）<ref>http://www.eac.gov.hk/pdf/distco/maps/dc2007b.pdf</ref><ref>http://www.eac.gov.hk/pdf/distco/maps/dc2007c.pdf</ref>；在交通道路網絡完成後，西貢北部仍屬大埔區（原由[[街渡|街渡]]連接[[馬料水|馬料水]]至[[西貢北|西貢北]]各鄉村，但除[[東平洲|東平洲]]等島嶼外，皆有道路連接西貢）<ref>http://www.eac.gov.hk/pdf/distco/maps/dc2007p2.pdf</ref>；以及[[馬灣|馬灣]]和大嶼山東北部，如[[竹篙灣|竹篙灣]]、[[欣澳|欣澳]]等地（原由街渡連接青龍頭，但在[[青嶼幹線|青嶼幹線]]通車之後，因客量不足已取消）屬於荃灣區<ref>http://www.eac.gov.hk/pdf/distco/maps/dc2007k.pdf</ref>；此外選區劃界亦牽涉政治因素，例如部分建制派在相關地區經營多年，若重劃和合併部分選區，將會導致他們失去議席<ref>[http://www.inmediahk.net/node/1026029 劃界助建制 公屋撥私樓溝淡泛民選區]</ref><ref>[http://www.inmediahk.net/20140723c 選管會疑針對泛民 助建制收復劉江華老巢]</ref>。相對八九十年代的分拆[[荃灣區|荃灣區]]、成立[[油尖旺區|油尖旺區]]及目前港島區議會架構重組，近年的改動規模較少，未有引起香港市民的關注。

==職能==
區議會職權範圍很廣，主要就在該區居住和工作之人士福祉攸關之事宜，向香港政府提供意見<ref name="書8">{{cite book|title=《綠皮書：一九八七年代議政制發展檢討綠皮書》|chapter=第二章 歷史背景|publisher=香港政府印務局|page=8|year=1987}}</ref>。

區議會的職能就以下項目向政府提供意見：
*影響地區人士的福利的事宜
*區內公共設施及服務的提供和使用
*政府為各區制定的計劃是否足夠及施行的先後次序
*各區就地區公共工程和社區活動獲得的撥款的運用

在就有關目的獲得撥款的情況下，承擔：
*區內的環境改善事務
*促進區內文娛康樂活動的工作
*區內的社區活動。

==地方行政==
除了被咨詢外，區議會還會主動參與管理地區事務。每個區議會的主席和副主席都是該區[[地區管理委員會|地區管理委員會]]的成員。地區管理委員會成員包括負責管理地方事務之政府部門人員，分別聽取各區議會之意見，和參照所得意見行事<ref name="書8"/>。地區管理委員會由當區的[[民政事務專員|民政事務專員]]擔任主席，負責協調該區各個政府部門，如[[房屋|房屋]]、[[交通|交通]]、[[衞生|衞生]]和教育等工作，並向區議會匯報。

==議員數目及組成==
現時，區議員按產生辦法可以分為兩類：
; 民選議員
: 透過地區直選選出, 採用[[單議席單票制|單議席單票制]]。根據《區議會條例》，民選議員數目由[[選舉管理委員會_(香港)|選管會]]制定，而2016－2019年度區議會民選議員的數目以每16,964人口設一議席的比率為基礎，現時共有431名民選議員。
; 當然議員
: 只在[[新界|新界]]的9個區議會出現，分別是每一區內各個[[鄉事委員會|鄉事委員會]]的主席。在[[香港總督|香港總督]][[彭定康|彭定康]]推行政改方案時，[[泛民主派|泛民主派]]曾以私人[[草案|草案]]方式要求香港政府取消區議會的當然議員議席，但未獲通過，故得以保留至今。現時共有27名當然議員。
; 委任議員 (已取消)
: 在[[香港回歸|主權移交]]前[[1994年香港區議會選舉|最後一屆]]區議會，因彭定康放棄行使《區議會條例》賦予委任議員的權利，該屆的相關議席取消。1998年起，政府欲加強對區議會的控制，自此恢復委任議員議席，依例最多可委任102名議員，2012年時共有68名委任議員。2013年5月22日，立法會通過《2013年區議會(修訂)條例草案》，於2015年取消所有區議會委任議席。<ref>http://rthk.hk/rthk/news/expressnews/news.htm?expressnews&20130522&55&923632 立法會通過下屆區議會選舉 取消所有委任議席] 2015-05-22 [[香港電台|香港電台]]新聞</ref>
; 官守議員 (已取消)
: 在初初成立及1982-1985年的區議會，設有官守議員，包括當區各政府部門的負責人。1985年起，政府為鼓勵市民對地區行政的投入，所有官員不再出任區議會議員，改以列席形式回答議員的提問。
