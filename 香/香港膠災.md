'''香港膠災'''是2012年发生在[[香港|香港]]附近海域的一次海洋污染事件。2012年7月23日，[[強颱風韋森特_(2012年)|強颱風韋森特]]吹襲香港，邻近海域普遍颳起[[颶風|颶風]]；导致一艘於香港南面水域的[[中海集運|中海集運]][[貨櫃船|貨櫃船]]的7個貨櫃墮海，當中6個貨櫃裝滿由[[中國石化|中國石化]]生產的[[聚丙烯|聚丙烯]]膠粒，並在風暴過後於香港南面海域漂流。部分貨櫃遭海浪擊毀，使櫃內膠粒進入海洋，成為[[海洋垃圾|海洋垃圾]]，並隨水流散佈於香港南部海域及岸灘，造成廣泛環境及生態污染，同時影響本地漁業。

膠粒在風暴襲港翌日已陸續沖上海岸，並由[[愉景灣|愉景灣]][[非政府組織|非政府組織]][[DB_Green|DB Green]]成員率先發現。膠災之初，香港主流媒體基本没有进行报导。清理工作基本由DB Green、香港[[海洋守護者協會|海洋守護者協會]]及其他非政府組織主導，並透過社交網站以及網絡媒體傳播消息。這些消息逐步在民間引起強烈反響，大量香港普通市民決定自發到海岸清理。

事發近兩週後，香港報章及電視媒體開始大幅度報導事件。[[海事處|海事處]]及[[食物環境衛生署|食物環境衛生署]]等[[香港政府|香港政府]]部門一直積極與民間組織協作清理，但政府從未向公眾發佈相關消息。到8月5日，開始有局長級官員向公眾交代。膠粒貨主中國石化公司以及船運公司中海集運都在7月底開始與非政府組織合作清理。但膠粒污染的責任問題尚未獲釐清。

== 事件經過 ==
=== 貨櫃墮海及初期清理 ===
2012年7月23日晚至翌日凌晨，[[強颱風韋森特_(2012年)|強颱風韋森特]]於香港西南方掠過，[[香港天文台|香港天文台]]在24日午夜發出[[十號颶風信號|十號颶風信號]]。當晚一艘[[中海集運|中海集運]]貨船在香港南面海域向東航行，前往[[果洲群島|果洲群島]]外海停泊。航行期間，船上先後有7個長40呎貨櫃被強風吹入大海，當中6個裝有約168噸由[[中國石化|中國石化]]生產的[[聚丙烯|聚丙烯]]膠粒，而另一個則裝有玻璃工藝品。<ref>{{cite news|title = 10號波「膠災」 168噸膠粒淹9泳灘|work = [[香港經濟日報|香港經濟日報]]|location = 香港|page = A20 |date = 2012-08-04|language = 中文}}{{cite news|title = 百噸化學膠粒飄港海 環團：魚類誤吞積毒恐進食物鏈|work = [[明報|明報]]|location = 香港|page = A10|date = 2012-08-04|language = 中文}}</ref>

事後中海集運先後聯絡[[海事處|海事處]]及[[打撈|打撈]]公司，以搜尋遺失貨櫃，但政府並未對外公佈事件。7月25日，環保組織[[DB_Green|DB Green]]成員Tracy Read率先在愉景灣北灘發現膠粒，並與香港[[海洋守護者協會|海洋守護者協會]]成員Gary Stokes等開始組織清理。次日DB Green聯絡中石化，並向[[南華早報|南華早報]]及[[英文虎報|英文虎報]]報告事件；而海洋守護者協會則開始號召香港人到海灘清理。中石化在27日隨即派員與DB Green巡視愉景灣海灘，並在[[三白灣|三白灣]]及[[稔樹灣|稔樹灣]]等地發現更多膠粒，但當日只有南華早報在內版作小篇幅報導。而民間人士及食環署則在同日先後通知非政府組織，指[[梅窩|梅窩]]及[[坪州|坪州]]各處均發現膠粒。<ref>{{cite news|last= Cheung|first= Chi-fai|title =Typhoon dumps plastic pellets on beach|work = South China Morning Post（[[南華早報|南華早報]]）|location = 香港|page = City 3 |date = 2012-07-27|language = en}}{{cite news|last= Lam|first= Lana|title =DB residents urged to help clear pellets|work = South China Morning Post（[[南華早報|南華早報]]）|location = 香港|page = Editorial 5 |date = 2012-07-29|language = en}} [http://www.garystokesphotography.com/blog/?p=1245 Plastic Beach Day 2 Gary Stokes Photography]，[http://www.garystokesphotography.com/blog/?p=1255 Plastic Beach Day 3 Gary Stokes Photography]</ref>

7月28日後，膠災的具體情況開始為部分非政府組織所知。中石化准許民間人員到[[葵青貨櫃碼頭|葵青貨櫃碼頭]]點算貨櫃損失，以評估墮海的膠粒數量。此時民間才得悉一共有6個膠粒貨櫃墮海，而海事處及中海集運先後於[[南丫島|南丫島]]東面、北面及西北面海域，與及[[大鴉洲|大鴉洲]]以東海域，尋回4個貨櫃。4個貨櫃洩漏了相當於1.2個貨櫃的膠粒，而當時尚有兩個貨櫃仍然失蹤。同日多個非政府組織聯絡香港各大媒體，要求報導事件，但全無回音。<ref>[http://www.garystokesphotography.com/blog/?p=1280 Plastic Beach Day 4-6 Gary Stokes Photography]、[http://www.garystokesphotography.com/blog/?p=1306 Plastic Beach Day 7 Gary Stokes Photography]、[http://hongkongcoastwatch.blogspot.hk/2012/08/plastic-nurdle-spill-hong-kong-where.html Hong Kong Coast Watch:Plastic Nurdle spill–Hong Kong–where were the press?]</ref>

7月31日起，非政府組織與政府及中石化的合作開始增加。當日海洋守護者協會罕有地成功主持一聯合會議，參與者包括政府食環署及海事處等部門代表、中石化高層以及中海集運代表。會後貨櫃船主代表同意出資租用直升機，供非政府組織偵察並拍攝海岸，以評估膠粒污染實況。守護者協會在次日於[[螺洲|螺洲]]西南海域尋回第5個貨櫃，海事處及食環署人員則緊接到場清理。此時香港南面的[[石澳|石澳]]、[[赤柱|赤柱]]、南丫島[[深灣_(南丫島)|深灣]]、[[長洲_(香港)|長洲]][[東灣_(長洲)|東灣]]、[[觀音灣|觀音灣]]及[[芝麻灣|芝麻灣]]均有膠粒發現報告，而民間組織開始在社交網站[[Facebook|Facebook]]發佈消息，以收宣傳及統籌之效。<ref>[http://www.garystokesphotography.com/blog/?p=1306 Plastic Beach Day 7 Gary Stokes Photography]、[http://www.garystokesphotography.com/blog/?p=1318 Plastic Beach Day ???? == 漁業、環境與生態問題 ==
[[File:Plastic_pellets_in_Ngong_Chong_Beach.JPG|thumb]]
香港膠災的膠粒由[[聚丙烯|聚丙烯]]製成，其[[塑膠分類標誌|塑膠分類標誌]]為5號。聚丙烯的原料是[[丙烯|丙烯]]，由[[石油|石油]][[裂解|裂解]]而來。化工廠接著可採用多種方法將丙烯及其他原料混合，將其聚合成塑膠，然後再用於生產不同產品，包括膠粒。

聚丙烯膠粒本身並無即時毒性，人類及魚類在吞食膠粒後，並不會將[[聚合物|聚合物]]聚丙烯吸入人體。再加上香港海域經常有塑膠垃圾漂浮，魚類進食不同種類膠粒常有出現，只是事件一直鮮為人知，才在膠災後引發極大反響。膠災曝光後，分銷商一度擔心吞食膠粒的海魚已經中毒，而停止向本地魚排購貨；而市民對本地魚的信心動搖，令本地魚在街市銷量急降，使漁民蒙受損失。<ref>{{cite news|title =高官齊撲火 強調膠粒無毒|work = [[香港經濟日報|香港經濟日報]]|location = 香港|page = A20|date = 2012-08-08|language = 中文}}{{cite news|title =揭「膠災」組織指政府積極環團冷淡|work = [[明報|明報]]|location = 香港|page = A8|date = 2012-08-10|language = 中文}}{{cite news|title =漁民每日蝕三萬元　政府允設恩恤補助|work = [[香港蘋果日報|香港蘋果日報]]|location = 香港|page = A8|date = 2012-08-09|language = 中文}}{{cite news|title =魚腸現膠粒 魚排擬索償|work = [[星島日報|星島日報]]|location = 香港|page = A8|date = 2012-08-08|language = 中文}}</ref>

聚丙烯一直廣泛使用於日常生活之中。由於聚丙烯耐熱、耐酸鹼、耐化學物質及耐碰撞，難以出現[[生物分解|生物分解]]，故此聚丙烯常用於食物容器，包括可放入[[微波爐|微波爐]]加熱的食物盒，延長食物保存時間。即棄餐具及商品包裝亦經常使用到聚丙烯。不過根據[[太平洋垃圾帶|太平洋垃圾帶]]的經驗，塑膠若在一年內長期受陽光[[紫外線|紫外線]]照射，便會出現[[光解作用|光解作用]]（Photodegradation），在保持聚合物狀態之餘，其體積逐步分解成為微小的[[分子|分子]]，繼續在海面漂浮。隨著塑膠體積繼續分解，製造塑膠時可能用到的[[雙酚A|雙酚A]]等化學品，便會釋出海洋；與此同時，微細的塑膠亦會吸附海面的所有化學物質，包括人類排入大海的致癌物，變成海面的一層「毒湯」。光解後的塑膠可輕易被魚類及海鳥等生物消化系統攝取，從而使多種有毒化學物進入[[食物鏈|食物鏈]]，對生態系統造成難以預計的破壞。體積較大的塑膠製品，亦可能堵塞海魚及海鳥消化系統，使其營養不良而死。<ref>{{cite web|title=Plastic Breaks Down in Ocean, After All – And Fast| first=Carolyn| last=Barry| work=[[國家地理雜誌|國家地理雜誌]] |publisher= [[國家地理學會|國家地理學會]]|url=http://news.nationalgeographic.com/news/2009/08/090820-plastic-decomposes-oceans-seas.html| date=2009-08-20| accessdate=03-09-2012}}</ref>

事實上，香港膠災的膠粒一度覆蓋南丫島[[深灣_(南丫島)|深灣]]。該處為瀕危海洋生物[[綠海龜|綠海龜]]的產卵地，每年6月至10月該處均會限制遊人入內，以免阻礙綠海龜產卵。膠災恰好發生於綠海龜產卵期內，而深灣更一度遭厚一呎膠粒覆蓋。<ref>{{cite news|title =拯救生態戰料持續幾個月 膠災兩元凶仍迴避責任|work = [[香港蘋果日報|香港蘋果日報]]|location = 香港|page = A4|date = 2012-08-08|language = 中文}}</ref>

據[[香港大學|香港大學]]地球科學系和太古海洋研究中心所做的一項研究發現，超過六成香港海水含有[[塑膠微珠|塑膠微珠]]，在7個地點獲取的逾100個海水樣本中，含量最高的[[北角|北角]]樣本每平方公里海域有38萬粒微珠，將造成[[塑料顆粒水污染|塑料顆粒水污染]]，海洋生物吸食塑膠微珠後將可引致生態破壞，導致海洋生物死亡。此項研究只篩選了0.3毫米至1毫米大小的塑膠微珠，估計實際微珠含量遠大於此數目，因為部分日用產品使用的微珠可少於0.3毫米。研究發現的塑膠微珠大多為透明顆粒狀，成分多為[[聚乙烯|聚乙烯]]，研究員估計這些塑膠微珠來自日常[[香港人|香港人]]所使用的個人護理或[[美容|美容]]產品，如面部磨砂膏和[[牙膏|牙膏]]等。<ref>{{Cite web |url=https://news.mingpao.com/ins/instantnews/web_tc/article/20180703/s00001/1530616894423 |title=港大：逾六成港海水樣本含塑膠微珠　每平方公里高達38萬粒　料來自個人護理產品 |date=2018-07-03 |publisher=明報即時新聞}}</ref><ref>{{Cite web |url=http://hk.on.cc/hk/bkn/cnt/news/20180703/bkn-20180703140350110-0703_00822_001.html |title=逾6成香港海水含塑膠微珠　料來自美容護理產品 |date=2018-07-03 |publisher=東網}}</ref><ref>{{Cite web |url=https://hk.news.appledaily.com/local/realtime/article/20180703/58393945 |title=逾60%港海水樣本含微膠粒 元兇可能係你！|date=2018-07-03 |publisher=蘋果日報即時新聞}}</ref><ref>{{Cite web |url=http://paper.wenweipo.com/2018/07/04/ED1807040007.htm |title=逾六成海水樣本含塑膠微珠|date=2018-07-03 |publisher=文匯報}}</ref>

[[綠色和平|綠色和平]]所做的調查顯示，[[城門河|城門河]]在夏季無降雨日子每天最少有超過48,000件塑膠沖出[[吐露港|吐露港]]，逢降雨後更多1倍，平均每分鐘有56件塑膠流出大海。這些塑膠廢物主要為即棄塑膠製品，如餐盒、飲管、餐具和膠袋等。另外，在[[颱風山竹|颱風山竹]]過後香港各區包括[[杏花邨|杏花邨]]，發現有約20年前的快餐店[[發泡膠|發泡膠]][[飯盒|飯盒]]碎片，綠色和平認為本地垃圾同樣是污染海洋的元兇。<ref>{{Cite news |url=https://news.mingpao.com/ins/instantnews/web_tc/article/20181018/s00001/1539845174637 |title=綠色和平：城門河每年逾1750萬件塑膠流入海　本地垃圾同為污染元兇  |date=2018-10-18 |work=明報}}</ref>
