{{香港政治}}
'''行政會議'''（簡稱'''行會'''，{{lang|en|'''Executive Council'''，'''ExCo'''}}），[[香港主權移交|香港主權移交]]前稱為'''行政局'''，是[[香港特別行政區|香港特別行政區]]的[[內閣|政府內閣]]<ref>{{cite web|url=https://www.gov.hk/tc/about/abouthk/facts.htm|website=GovHK 香港政府一站通|title=香港概況}}</ref>、於[[香港特別行政區政府|香港政府]]內協助[[香港特別行政區行政長官|行政長官]]決策的最高級機構，由行政長官主持以及自行任命成員。

現時行政會議有33位成員，包括行政長官（主席）、16位[[問責制|問責官員]]（[[官守議員|官守議員]]）及16位非官方人士（[[不管部长#_香港特別行政區|非官守議員]]，包括召集人）<ref name='ExCo'> {{cite web |url = http://www.ceo.gov.hk/exco/chi/index.htm |title = 行政會議 |accessdate = 2010-07-06 |publisher = 香港特別行政區政府 |language = zh-hk}}</ref>，成員均屬[[公職|公職]]並支薪<ref>[http://www.info.gov.hk/gia/general/201501/16/P201501161183.htm 行政會議成員酬金回復至原有水平] 政府新聞公布 2015年1月16日</ref>。行政會議每周在[[香港行政長官辦公室|香港行政長官辦公室]]舉行一次會議，通常為星期二早上（[[香港節日與公眾假期|公眾假期]]除外）。

《[[香港特別行政區基本法|基本法]]》規定，行政長官在作任何重要決策前，均須徵詢行政會議的意見。<ref name='ExCo'/>行政會議實行保密制及[[集體負責制|集體負責制]]。

== 歷史 ==
[[香港殖民地時期|英國殖民地]]時期，行政局（成立初期稱為'''議政局'''）由[[香港總督|香港總督]]主持，邀請知名社會人士擔任成員（當時稱為議員），通常有十多人，重要決策均需由[[總督會同行政局|總督會同行政局]]通過後才能執行。行政局有四名當然議員，包括[[駐港英軍|駐港英軍]]三軍司令（至1993年）、[[政務司司長|布政司]]、[[財政司司長|財政司]]和[[律政司司長|律政司]]；其餘議員由香港總督委任，分為「官守」（即正在擔任官職之意）和「非官守」兩類。

行政局議員需遵守和[[英國內閣|英國內閣]]相同的「保密制」及「[[集體負責制|集體負責制]]」：即是所有議員均需對會議過程保密，政策決定後各議員對外均需支持該政策。這制度在行政會議中仍然保留，但不少人、包括現任成員均認為這制度有需要檢討；而近年亦有少數例子，例如身兼立法會議員或非官守的行政會議成員公開反對政府政策，違反集體負責制<ref>[http://www.skypost.hk/%E6%B8%AF%E8%81%9E/%E6%96%B0%E8%81%9E%E9%A0%AD%E6%A2%9D/20131115/001/%E6%99%82%E4%BA%8B%E9%9B%99%E5%B0%8D%E8%AB%96%20%3A%20%E8%A1%8C%E6%9C%83%E3%80%8C%E9%AC%BC%E6%89%93%E9%AC%BC%E3%80%8D%EF%BC%9F/116994 時事雙對論: 行會「鬼打鬼」？] {{webarchive|url=https://web.archive.org/web/20151125104910/http://www.skypost.hk/%E6%B8%AF%E8%81%9E/%E6%96%B0%E8%81%9E%E9%A0%AD%E6%A2%9D/20131115/001/%E6%99%82%E4%BA%8B%E9%9B%99%E5%B0%8D%E8%AB%96%20%3A%20%E8%A1%8C%E6%9C%83%E3%80%8C%E9%AC%BC%E6%89%93%E9%AC%BC%E3%80%8D%EF%BC%9F/116994 |date=2015-11-25 }}晴報 2013年11月15日</ref>。

[[香港主權移交|香港主權移交]]後，行政局改稱行政會議，大致繼續它的職能，但不設當然議員，所有行政會議非官守議員的任命和取消都由行政長官決定，但「保密制」及「[[集體負責制|集體負責制]]」維持至今。

2002年7月行政長官[[董建華|董建華]]推行[[高官問責制|高官問責制]]以後，在行政會議中，司局長往往佔絕大多數（三位司長加上十一局長的官守議員合共十四位，而非官守成員只有五至七人），被指令非官守議員無法對行政長官決定政策產生影響力。

2005年[[曾蔭權|曾蔭權]]接任行政長官後，增加行政會議非官方成員的人數，非官守議員及官守議員達至一比一。行政會議的官守議員除了三位司長會出席所有會議外，其他官守成員（即各決策局局長）可以選擇只在會議涉及本身負責的事務時才出席。

== 功能 ==      
主權移交前，行政局是港英政府醞釀政策的地方，香港總督在決定重要決策或向[[香港立法會|立法局]]提交[[法案|法案]]前均先由[[香港總督會同行政局|香港總督會同行政局]]通過；由於[[1991年香港立法局選舉|在1991年立法局部分議席改為直選前]]，立法局議員大多數均由香港總督委任或間接選舉產生，而且不少立法局議員均兼任行政局議員，立法局實際只是「[[橡皮圖章_(政治)|橡皮圖章]]」，不會遇到立法阻力。如果香港總督不同意行政局大多數人決議時，他需要把原因交往[[英國外交部|英國外交部]]備案<ref>《[[皇室訓令|皇室訓令]]》第12條</ref>，因此行政局對香港總督決策有相當影響力。

主權移交後，行政會議只會向行政長官一人負責；行政長官如果不同意行政會議大多數人的決議時，只須將理由記錄在案。<ref> {{cite web |url = http://www.basiclaw.gov.hk/tc/basiclawtext/chapter_4.htm |title = 基本法 |accessdate = 2010-07-06 |publisher = 香港特別行政區政府 |language = zh-hk |quote = 香港基本法第五十六條第三款：「行政長官如不採納行政會議多數成員的意見，應將具體理由記錄在案。」 }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

前行政長官董建華只當行政會議為智囊團，行政會議失去原有作用；2002年董建華推行問責制後，所有問責司局長自動成為行政會議成員，作用減少。

[[曾蔭權|曾蔭權]]上任後，恢復行政會議功能，增加官守和非官守成員，部分[[建制派|建制派]]立法會議員加入行政會議（一般為建制派政黨代表），使身兼行政會議成員的立法會議員成為政府在立法會的代表，被批評是向立法會箝票，確保法案獲得足夠票數支持。

== 現任成員 ==
根據《[[香港特別行政區基本法|基本法]]》，行政會議並不如香港主權移交前的行政局般有當然議員。行政會議所有成員任免皆由行政長官決定；他們可來自行政機關的主要官員、立法會議員和社會人士，任期不得超過任命他們的行政長官的任期（即最多五年）。
