《'''香水'''》（{{lang-de|'''''Parfum'''''}}，{{lang-en|'''''Perfume'''''}}）是一部替 {{link-en|ZDFneo|ZDFneo}} 頻道製作的德國電視劇，於2018年11月14日播出。本劇意念取自[[徐四金|徐四金]]的[[香水_(小說)|同名小說]]及2006年電影《[[香水_(電影)|-{zh-hk:香水：一個殺人者的故事;zh-tw:香水;zh-cn:香水;}-]]》，此片同樣是由康斯坦丁影業製作。[[Netflix|Netflix]] 獲得國際播放版權<ref>{{Cite web |url = https://www.hollywoodreporter.com/news/berlin-netflix-buys-german-perfume-series-1086463 |title = Berlin: Netflix Buys German 'Perfume' Series |author = Scott Roxborough |date = 2018-02-20 |publisher = The Hollywood Reporter |language = en}}</ref>，於2018年12月21日播放。<ref>{{Cite web |url = https://www.forbes.com/sites/erikkain/2018/11/27/heres-whats-coming-to-netflix-in-december-2018-and-what-to-watch/#1f808210173f |title = Here's What's Coming To Netflix In December (2018) And What To Watch |author = Erik Kain |date = 2018-11-27 |publisher = Forbes |language = en}}</ref>而普遍國際評價對於此劇是正面的。

== 故事概述 ==
在萊茵河下游發現了一名紅頭髮女性的屍體，她的陰部和腋下被割走。警方調查員娜嘉·西蒙 (Nadja Simon) 和馬提亞斯·科勒 (Matthias Köhler) 連同與娜嘉有婚外情的檢察官約亨·格林貝格 (Joachim Grünberg)，要面對從求學時就認識受害者的五名前寄宿學校同學，他們當時受小說《[[香水_(小說)|香水]]》的啟發下，對人類的氣味進行實驗。此外，當年未解決的失踪男孩案件成為關注焦點。當他的屍體被發現時，他與第一位受害者有著同樣的傷痕。甚至一名妓女被謀殺，她亦以相同的方式被殘害。與此同時，娜嘉發現格林貝格令她懷孕了，而格林貝格沒打算離開他的妻子，還要求娜嘉墮胎。當她拒絕時，他結束了這段關係。
