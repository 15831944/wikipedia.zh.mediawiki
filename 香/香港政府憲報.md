'''《香港政府憲報》'''（1841年5月1日—1997年6月30日），[[香港主權移交|香港主權移交]]後改稱《'''香港特別行政區政府憲報'''》（1997年7月1日—），是[[香港政府|香港政府]]發佈公告、啟事及法例等的官方出版物。憲報由[[政府物流服務署|政府物流服務署]]發行，受命於政府物流服務署署長及[[財經事務及庫務局|財經事務及庫務局]]局長。香港的所有[[香港法例|條例]]及{{tsl|en|By-law|附屬法例}}，都要在憲報刊登（通稱「刊憲」）才生效。

曾有[[訴訟|訴訟]]因相關法律未有刊憲，[[疑犯|疑犯]]因而脫罪<ref>[http://legalref.judiciary.gov.hk/lrs/common/search/search_result_detail_frame.jsp?DIS=43172&QS=%2B&TP=JU HKSAR v. CHAN CHIU KWOK]</ref>。

==歷史==
早於1841年[[英國|英國]][[水坑口|佔領香港]]後，便已經有《香港憲報》的出現，當時中文譯為《'''[[轅門|轅門]]報'''》或《'''香港轅門報'''》。第一份《香港憲報》出版於1841年5月1日，內容公佈了[[拍賣|拍賣]][[香港土地利用|香港土地]]的詳情。

==概述==
《香港憲報》通常於逢星期五出版。若果有需要時，《憲報》可以在星期五以外的日期出版，稱為「憲報[[號外|號外]]」。

《香港憲報》分為7部分：
# 「政府公告」，內容是政府官方委任令、法定公告及政府[[招標|招標]]等； [http://www.gld.gov.hk/cgi-bin/gld/egazette/gazettefiles.cgi?lang=c&year=2006&month=8&day=18&vol=10&no=33&header=0&acurrentpage=10&df=0&agree=1&gaz_type=mg]
# 「第1號法律[[副刊|副刊]]」，香港條例的新增或修訂公告，例如《2006年為僱員權益作核證(中醫藥)(雜項修訂)條例》； [http://www.gld.gov.hk/cgi-bin/gld/egazette/gazettefiles.cgi?lang=c&year=2006&month=7&day=7&vol=10&no=27&header=0&acurrentpage=10&df=0&agree=1&gaz_type=ls1]
# 「第2號法律副刊」，法律規例，例如法定[[巴士|巴士]]路線； [http://www.gld.gov.hk/cgi-bin/gld/egazette/gazettefiles.cgi?lang=c&year=2006&month=7&day=7&vol=10&no=27&header=0&acurrentpage=10&df=0&agree=1&gaz_type=ls2]
# 「第3號法律副刊」，[[法案|條例草案]]，例如《非應邀電子訊息條例草案》； [http://www.gld.gov.hk/cgi-bin/gld/egazette/gazettefiles.cgi?lang=c&year=2006&month=7&day=7&vol=10&no=27&header=0&acurrentpage=10&df=0&agree=1&gaz_type=ls3]
# 「第4號特別副刊」，註冊/牌照人士名單，例如[[地產代理|地產代理]]牌照名單 [http://www.gld.gov.hk/cgi-bin/gld/egazette/gazettefiles.cgi?lang=c&extra=%20&year=2006&month=8&day=4&vol=10&no=31&header=1&acurrentpage=10&agree=1&df=0&gaz_type=ls4&level=2&pid=18981&gn=18981&newfile=0]、註冊醫生 [http://www.gld.gov.hk/cgi-bin/gld/egazette/gazettefiles.cgi?lang=c&year=2006&month=7&day=21&vol=10&no=29&header=0&acurrentpage=10&df=0&agree=1&gaz_type=ls4] 及幼兒中心等名單 [http://www.gld.gov.hk/cgi-bin/gld/egazette/gazettefiles.cgi?lang=c&year=2006&month=7&day=14&vol=10&no=28&header=0&acurrentpage=10&df=0&agree=1&gaz_type=ls4] ；
# 「第5號特別副刊」，草案擬本、[[行政命令|行政命令]][http://www.gld.gov.hk/cgi-bin/gld/egazette/gazettefiles.cgi?lang=c&extra=e&year=2006&month=8&day=9&vol=10&no=24&header=0&acurrentpage=10&df=0&agree=1&gaz_type=ls5]、國務院令及國際協定等； [http://www.gld.gov.hk/cgi-bin/gld/egazette/gazettefiles.cgi?lang=c&year=2006&month=4&day=21&vol=10&no=16&header=0&acurrentpage=10&df=0&agree=1&gaz_type=ls5] 及 
# 「第6號副刊」，公共啟事，例如[[有限公司|有限公司]][[清盤|清盤]]會議通知、業務轉讓、拆夥及[[商標|商標]][[註冊|註冊]]申請等。 [http://www.gld.gov.hk/cgi-bin/gld/egazette/gazettefiles.cgi?lang=c&year=2006&month=8&day=18&vol=10&no=33&header=0&acurrentpage=10&df=0&agree=1&gaz_type=ls6]
