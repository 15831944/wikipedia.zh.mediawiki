'''香积寺路站'''（工程名为东新园站）位于杭州市下城区东新路香积寺路口，是[[杭州地铁5号线|杭州地铁5号线]]的地下车站，也是未来与规划中8号线的换乘站<ref>{{cite web |title=杭州地铁二期主城区线路和站点公示 |url=http://hznews.hangzhou.com.cn/chengshi/content/2014-07/09/content_5354325.htm |website=杭州网}}</ref>。预计2018年底前完成车站主体结构，2019年随5号线开通投入使用。<ref>{{cite web |title=盘点杭州14条地铁线完整站点名单，有经过你家门口吗？ |url=https://www.sohu.com/a/223309557_692410 |website=搜狐 |date=2018-02-20}}</ref><ref>{{cite web |title=地铁5号线香积寺路口站正式更名为东新园站！再行路站不再是3号线换乘|url=http://zzhzbbs.zjol.com.cn/thread-17388735-1-1.html |website=口水杭州}}</ref>

== 结构 ==
香积寺路站沿东新路布置<ref>{{cite web |title=杭州地铁5号线一期、二期38座车站拟定站名方案征求意见 |url=http://www.hzmetro.com/notice_show.aspx?ID=7871 |website=杭州地铁}}</ref>。车站全长192.95m，深度16~18m。<ref>{{cite journal |author1=杭州市轨道办 |title=简报2018年第27期 |date=2018年7月24日}}</ref>，车站主体围护结构采用地下连续墙，主体结构为地下两层双柱三跨箱形框架结构，采用半盖挖顺做法施工。<ref>{{cite web |title=杭州地铁5号线一期工程土建施工SG5-10标段技术标 |url=http://www.njliaohua.com/lhd_6xa1a2rjig85bn68adez_3.html |website=南京廖华 }}{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>车站共设2个风亭和4个出入口，总建筑面积12322平方米<ref>{{cite web |title=杭州地铁5号线东新园站规划公示查看 |url=http://www.hzplanning.gov.cn/DesktopModules/GHJ.PlanningNotice/PublicityInfoPQGS.aspx?GUID=12085 |website=杭州市规划局}}</ref>，此外还设置有地下商业服务及停车空间共计3.8万平方米<ref>{{cite web |title=东新街道沈家村：建设全域中央商务区 |url=http://hznews.hangzhou.com.cn/chengshi/content/2017-07/27/content_6612344.htm |website=杭州网}}</ref>。

== 历史 ==
2017年11月3日，车站开工，是5号线最晚开工的车站之一。<ref>{{cite web |title=东西向骨干线地铁3号线一期开始绿化迁移-浙江新闻-浙江在线 |url=http://zjnews.zjol.com.cn/zjnews/hznews/201711/t20171119_5732682.shtml |website=浙江在线}}</ref>
2019年3月20日，车站主体结构封顶。<ref>{{cite web |title=地铁5号线东新园站车站主体结构封顶 |url=http://www.hzmetro.com/build_4_show.aspx?ID=8470#midc |website=杭州地铁}}</ref>
