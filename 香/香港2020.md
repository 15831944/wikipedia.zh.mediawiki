'''香港2020'''（[[英文|英文]]：'''Hongkong2020'''）前身為'''民間策發會'''，於2013年4月24日由[[陳方安生|陳方安生]]成立，為一個[[香港政治|香港政治]][[组织_(社会学)|組織]]，旨在推動香港[[民主|民主]][[香港政治制度改革|政治制度改革]]，提供平台收集[[香港社會|香港社會]]對[[雙普選|雙普選]]的意見，再向[[香港政府|香港政府]]提出，並且爭取「[[真普選|真普選]]」<ref>[http://hk.news.yahoo.com/陳方安生成立新組織推動真普選-100000155.html 陳方安生成立新組織推動真普選]</ref>；表明最終方案必須堅守[[聯合國|聯合國]]制定的《[[公民權利和政治權利國際公約|公民權利和政治權利國際公約]]》及《[[基本法|基本法]]》普及平等原則，期望於同年之內提出政治制度改革方案。

== 成員 ==
香港2020成員包括[[資深大律師_(香港)|資深大律師]][[李志喜|李志喜]]、資深大律師及[[香港大學法律學院|香港大學法律學院]]院長[[陳文敏|陳文敏]]、前[[立法局議員|立法局議員]][[李鵬飛|李鵬飛]]、前高級官員[[高德禮|高德禮]]和前副[[憲制事務司|憲制事務司]][[布簡瓊|布簡瓊]]等。另外由[[民主黨_(香港)|民主黨]]前[[立法會議員|立法會議員]][[李永達|李永達]]出任研究總監，由[[公民黨|公民黨]][[余冠威|余冠威]]出任研究主任。

== 歷史 ==
2013年4月24日，陳方安生將其於2008年成立的[[民間策發會|民間策發會]]改組為香港2020。創辦人陳方安生表示自己一直支持雙普選，惟香港社會仍然未對雙普選有實質的建議及共識，而[[香港行政長官|香港行政長官]]又無所作為，所以她成立此組織以推動民主發展<ref>[http://hk.news.yahoo.com/陳方安生設新組織推動民主-072100415.html 陳方安生設新組織推動民主] 《星島日報》 2013年4月24日</ref>。她又表示，香港正在處於落實全面雙普選的重要時刻，未來數月對政治制度改革所作出的決定，除了影響時下大眾，亦會影響未來數代[[香港人|香港人]]。她表示對推動雙普選充滿熱誠，認為只有透過「[[真普選|真普選]]」，才能夠改善香港的管治及保留[[香港核心價值|香港核心價值]]<ref>[http://hk.news.yahoo.com/陳方安生成立-香港2020-組織推動政改-072000648.html 陳方安生成立「香港2020」組織推動政改]</ref><ref>[http://hk.news.yahoo.com/香港2020-沒接受外來支援-091800573.html 「香港2020」沒接受外來支援] 《星島日報》 2013年4月24日</ref>。她表示香港2020將會特別著力與商界溝通<ref>[http://hk.news.yahoo.com/陳方安生拉攏商界商討普選方案-104540841.html 陳方安生拉攏商界商討普選方案]</ref>。

2013年4月25日，陳方安生在[[香港電台|香港電台]]節目訪問時表示，成立香港2020組織是希望聽取商界及年青人對雙普選的意見，又指出行政長官[[梁振英|梁振英]]遲遲未啟動諮詢程序，所以決定需要帶動討論<ref>[http://hk.news.yahoo.com/陳太冀新組織凝聚政改共識-012000679.html 陳太冀新組織凝聚政改共識] 《星島日報》 2013年4月25日</ref>。陳方安生強調政治制度改革諮詢必須公開及透明、包容香港社會上的不同意見，以及真正反映大多數香港人的關注及訴求，最終方案亦必須堅守聯合國制定的《公民權利和政治權利國際公約》及《基本法》普及平等原則，又認為現時的[[選舉委員會_(香港)|選舉委員會]]是小圈子，將來的提名委員會必需提升其認受性。陳文敏亦解釋，《公民權利和政治權利國際公約》提及的提名限制必須是客觀合理，例如[[國籍|國籍]]及[[年齡|年齡]]，而不應該建基於參選人的[[政治立場|政治立場]]。陳方安生表示，香港2020將會在[[互聯網|互聯網]]及[[社交網站|社交網站]]上收集年青人的意見。又指自己近年在逾100間[[中學|中學]]和[[專上學院|專上學院]]發表[[演講|演講]]，涉及[[一國兩制|一國兩制]]與民主等題目，發現年青人漸漸留意有關議題，而最近反對[[德育及國民教育科|德育及國民教育科]]等事件均由年青人發起，表揚他們做得非常好。陳方安生又說，商界的聲音很有影響力，商界不能夠置身事外，因為政治制度改革將會影響香港政府的管治能力、香港社會的狀態及商業營運環境。她說自己接觸的商界人士中，不少對政黨存有戒心<ref>[http://hk.news.yahoo.com/陳太搞-香港2020-提拔政治領袖-220409808.html 陳太搞「香港2020」 提拔政治領袖] 《星島日報》 2013年4月25日</ref>，對於為政治制度改革發聲時有所顧忌，是因為生怕與[[泛民主派|泛民主派]]關係太過密切，影響其在[[中國大陸|中國大陸]]的投資及生意<ref>[http://hk.news.yahoo.com/陳太重視商界對政改意見-013400737.html 陳太重視商界對政改意見] 《星島日報》 2013年4月25日</ref><ref>[https://archive.is/20130630172248/hk.news.yahoo.com/%E9%99%B3%E5%A4%AA-%E7%89%B9%E9%A6%96%E6%94%BF%E6%94%B9%E9%A0%98%E5%B0%8E%E7%84%A1%E6%96%B9-004942220.html 陳太：特首政改領導無方] 《明報》 2013年4月25日</ref>。故此，香港2020將會以非政黨形式聆聽意見<ref>[http://hk.news.yahoo.com/陳方安生-特首遲遲未啟動諮詢-決帶動討論-015300626.html 陳方安生：特首遲遲未啟動諮詢　決帶動討論]</ref><ref>[http://hk.news.yahoo.com/陳方安生-組新組織聽取商界對雙普選意見-012300751.html 陳方安生：組新組織聽取商界對雙普選意見]</ref><ref>[http://hk.news.yahoo.com/香港2020-成立-望年內提政改方案-210649997.html; 「香港2020」成立 望年內提政改方案] 《明報》 2013年4月25日</ref>。

== 財政 ==
香港2020接受外界捐款，但不會透露資金來源及金額，陳方安生指從未接受來自外國的資金。
