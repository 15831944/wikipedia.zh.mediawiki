# 香港對頒授本地專上學歷的大專院校主要分為四種規管模式。「法定院校」是根據所屬香港法例成立的法定機構，其校董會組成及法定權力皆由所屬條例所訂明，而所屬條例的訂立及條文的修訂需經由[[香港立法會|香港立法會]]通過，法定院校除包括各所法定大學外，[[香港演藝學院|香港演藝學院]]亦為法定院校。「法定組織轄下院校」是受所屬香港法例規管的法定組織所設立的院校，這類院校包括[[職業訓練局_(香港)|職業訓練局]]的成員院校，[[香港藝術中心|香港藝術中心]]設立的[[香港藝術學院|香港藝術學院]]，以及法定大學營運的獨立學院。「認可專上學院」是根據香港法例第320章《專上學院條例》註冊的院校，都是已獲[[行政長官會同行政會議|行政長官會同行政會議]]批准開辦本地學士學位課程的院校。「一般本地院校」是根據香港法例第279章《教育條例》向[[教育局_(香港)|教育局]]註冊及提供經評審本地專上課程的院校，這類院校可提供副學位或以下程度的課程，在香港開辦本地課程的機構如不屬於法定院校或根據《專上學院條例》註冊的認可專上學院，都須要根據《教育條例》註冊。
#「非本地院校」是根據香港法例第493章《非本地高等及專業教育（規管）條例》向教育局註冊及在香港提供非本地專上課程的教育機構，這類課程的學術資格是由海外院校頒授，並由課程在香港的營運機構承擔註冊及管理責任<ref>{{cite web |title=非本地高等及專業教育課程 - 供學生參閱的資料 |url=https://www.edb.gov.hk/tc/edu-system/postsecondary/non-local-higher-professional-edu/info-for-student/index.html |publisher=香港教育局}}</ref>，只是在香港提供由海外院校頒授學歷的非本地課程而沒有開辦本地學歷課程的教育機構屬於「非本地院校」，至於同時開辦本地學歷課程及提供由海外院校頒授學歷的非本地課程的教育機構，則需要同時就其營辦的本地學歷及非本地學歷分別按條例註冊。根據《非本地高等及專業教育（規管）條例》只有獲豁免的非本地課程無須註冊，而獲得豁免的課程均由已取得大學名稱的法定大學或認可專上學院合辦<ref>{{cite web |title=獲豁免註冊課程為與指定的本地高等教育機構合辦之非本地課程 |url=https://www.edb.gov.hk/tc/edu-system/postsecondary/non-local-higher-professional-edu/course/exempted-course.html |publisher=香港教育局}}</ref>，因此即使是法定院校、認可專上學院或按教育條例註冊的院校，如在香港提供海外院校課程而又不屬於獲豁免機構，均要依據《非本地高等及專業教育（規管）條例》為其開辦的非本地課程註冊。
# [[香港大學|香港大學]]、[[香港中文大學|香港中文大學]]及[[香港科技大學|香港科技大學]]成立時已擁有大學名銜。這三所大學並非由學院升格為大學，其中[[香港大學|香港大學]]及[[香港中文大學|香港中文大學]]成立時，[[教資會|教資會]]尚未成立，而[[香港科技大學|香港科技大學]]則在籌備期間通過學術評審及立法程序成為法定大學。
# [[大學教育資助委員會|大學教育資助委員會（教資會）]]是一個非法定諮詢組織，負責就受其資助的大學在發展及所需經費方面，向香港政府提供意見<ref>{{cite web|title=教資會與政府有何關係？
|url=http://www.ugc.edu.hk/big5/ugc/faq/q106.html|publisher=大學教育資助委員會}}</ref>。教資會資助的院校都是根據本身的法定條例而成立，獲授予自主權，在課程管理、甄選學生、進行研究、聘任教職員和訂定聘用條款等多方面都擁有很大的自由度，但教資會可就各院校的資源運用提供意見，確保來自政府的撥款用得其所<ref>{{cite web|title=教資會與受資助大學有何關係？|url=http://www.ugc.edu.hk/big5/ugc/faq/q105.html|publisher=大學教育資助委員會}}</ref>。
# [[香港學術及職業資歷評審局|香港學術及職業資歷評審局（評審局）]]前稱「香港學術評審局」或簡稱「學評局」，是根據香港法例第1150章《香港學術及職業資歷評審局條例》成立的法定機構，以獨立的法定地位為香港高等教育機構開辦的課程進行評審，並根據香港法例第592章《學術及職業資歷評審條例》負責管理[[香港資歷架構|香港資歷架構]]的資歷名冊<ref>{{cite web|title=本局介紹|url=https://www.hkcaavq.edu.hk/zh/about-us/about-hkcaavq|publisher=[[香港學術及職業資歷評審局|香港學術及職業資歷評審局]]}}</ref>。香港城市大學、香港浸會大學、嶺南大學、香港公開大學、香港理工大學在升格為大學前，都曾經接受學評局就其開辦的課程進行學術評審<ref>{{cite web|title=二十週年紀念 - 主要里程碑|url=https://www.hkcaavq.edu.hk/publication/AR09-10/tc/index.htm|publisher=香港學術及職業資歷評審局}}</ref>。
# 香港高等院校所開辦的本地認可全日制學士學位課程，主要分為受教資會資助院校所開辦的資助學位課程，政府通過指定專業／界別課程資助計劃資助的學位課程，政府通過免入息審查資助計劃資助的自資學位課程，以及教資會資助院校所開辦的自資學位課程。就每學年所提供的學士學位第一年學額而言，受教資會資助的學額上限約為15,000個，指定專業／界別課程資助計劃的資助學額於2019/20學年為3,236個<ref>{{cite news |title=6院校38個指定專業課程獲資助　護理學額增五成 |url=http://hk.on.cc/hk/bkn/cnt/news/20180822/bkn-20180822160739351-0822_00822_001.html?eventsection=hk_news&eventid=402883474e4e5218014e88352c033b74 |work=東方日報 |date=2018-08-22}}</ref>，免入息審查資助計劃的資助學額主要由該學年已獲認可的自資課程數量而定，教資會資助院校的自資學位課程學額由各開辦院校自行決定及發佈。
# 「副學位」屬於[[香港資歷架構|香港資歷架構]]的第四級別，「副學士」及「高級文憑」都屬於副學位學歷<ref>{{cite web|title=資歷名銜計劃|url=https://www.hkqf.gov.hk/filemanager/tc/content_16/HKQF_ATS_C_2012_10.pdf|publisher=[[香港教育局|香港教育局]]}}</ref>。副學士較注重培養學術水平，作為升讀學士學位課程的準備，香港有多所院校以自負盈虧方式開辦副學士課程。高級文憑較注重職業實務的訓練，為就業作出準備，[[香港專業教育學院|香港專業教育學院]]是高級文憑課程的最主要開辦院校。
# 大學教育資助委員會資助的大專院校，大部分都有開設以自負盈虧方式營運的專上學院，雖然這些專上學院是附屬於受教資會資助的院校，但這些附屬學院卻不屬教資會的資助範圍。因此該等學院所開辦的[[副學位|副學位]]課程，包括[[高級文憑|高級文憑]]及[[副學士|副學士]]，大部分都不受教資會資助。受教資會資助的副學位課程，都是經由[[大學聯合招生辦法|大學聯合招生辦法]]招收學生。此外，有部分教資會資助院校在大學本部開辦自資學士、碩士及博士課程，同樣不屬教資會資助課程。
# 大學教育資助委員會資助的大專院校，大部分都有設立附屬學院營辦自資課程，由於法定大學均具有自行評審資格及可頒授大學學位，而自資課程並不屬教資會的資助範圍，所以開設自資課程無須經由教資會檢定，具有自行評審資格的院校開辦新課程亦無須事先提交香港學術及職業資歷評審局進行學術評審，因此該等附屬學院所開辦的課程僅由其所屬大學作質素保證，而無須經由教資會或學評局進行檢定及作定期校外評審<ref>{{cite news |title=八大自資附屬機構與大學「分家」 |url=http://std.stheadline.com/daily/news-content.php?id=1825775&target=2 |work=星島日報 |date=2018-06-26}}</ref>。
# [[指定專業／界別課程資助計劃|指定專業／界別課程資助計劃]]是香港政府於2014年起推行的資助計劃，為有殷設需求專才行業開辦學士學位課程的自資院校提供資助，於推出時的受計劃資助院校包括[[香港公開大學|香港公開大學]]、[[恒生管理學院|恒生管理學院]]、[[珠海學院|珠海學院]]、[[東華學院|東華學院]]、[[明愛專上學院|明愛專上學院]]及[[香港高等教育科技學院|香港高等教育科技學院]]<ref>{{cite web|title=指定專業／界別課程資助計劃 (SSSDP)|url=https://www.cspe.edu.hk/tc/sssdp/index.html|publisher=[[香港教育局|香港教育局]]}}</ref>。
# [[免入息審查資助計劃|免入息審查資助計劃]]是香港政府於2017年起推行的自資學位資助計劃，為合資格的中學教育文憑或副學位畢業生報讀自資院校所開辦的全日制認可學士學位課程提供資助，包括職業訓練局轄下的[[香港高等教育科技學院|香港高等教育科技學院]]在內，在計劃開始時共有15所自資院校的課程獲納入資助，但教資會資助院校及其轄下學院，以及[[香港演藝學院|香港演藝學院]]均未獲納入該項計劃的資助範圍。受計劃資助的自資院校於2018年7月增至16所<ref>{{cite web |title=合資格院校開辦的合資格全日制經本地評審本地及非本地自資首年學士學位及銜接學位課程名單(二〇一八／一九學年入學) |url=https://www.cspe.edu.hk/resources/pdf/tc/NSS/NSS_Prog1819_TC.pdf |publisher=香港教育局 |accessdate=2018-08-05}}</ref>。
# 除了私立院校外，公立院校也可營辦自負盈虧的自資專上課程；例如受政府經常性資助的[[職業訓練局|職業訓練局]]，其轄下院校所提供的高級文憑課程，有部份是政府資助課程，但亦有部份是不受資助的自資課程；至於該機構屬下的[[香港高等教育科技學院|香港高等教育科技學院]]則是開辦本地自資學士學位課程的自資院校。
# 香港的高等院校需確保其開辦的課程符合課程質素保證機制，已取得全面自行評審資格的院校可開辦任何新課程而無需事先通過外評，取得學科範圍評審資格的院校開辦指定學科範圍的新課程可無需事先通過外評，未取得以上兩者資格的院校，在開辦新課程前應先通過[[香港學術及職業資歷評審局|香港學術及職業資歷評審局]]的評審<ref>{{cite web|title=質素保證|url=https://www.hkqf.gov.hk/tc/qa/index.html|publisher=資歷架構}}</ref>。
# 「自行評審資格」僅授予設有完善管理架構，並獲信納可自行確保新辦課程達到指定水平的院校。根據香港第592章《學術及職業資歷評審條例》，屬於自行評審營辦者的院校，包括[[香港中文大學|香港中文大學]]、[[香港城市大學|香港城市大學]]、[[香港浸會大學|香港浸會大學]]、[[香港教育大學|香港教育大學]]、[[香港理工大學|香港理工大學]]、[[香港科技大學|香港科技大學]]、[[嶺南大學_(香港)|嶺南大學]]、[[香港公開大學|香港公開大學]]和[[香港大學|香港大學]]<ref name="cop592-2018">{{cite news|title=自行評審營辦者|url=https://www.elegislation.gov.hk/hk/cap592!zh-Hant-HK.assist.pdf|work=香港法例|date=2018-04-01}}</ref>。
# 「學科範圍評審資格」是指任何已經通過[[香港學術及職業資歷評審局|香港學術及職業資歷評審局]]課程評審的香港教育機構，在相關學科範圍內通過了至少兩次課程覆審，並在該學科範圍內有足夠的確認往績，便可在相關學科範圍申請「學科範圍評審」，在有效期內於指定學科範圍及資歷級別內籌辦新課程，並將所頒授的資歷登記在[[香港資歷架構|資歷名冊]]上，而無須事先就個別課程申請評審。提交「學科範圍評審」的課程，並不一定是學位課程，也可以是文憑或證書課程。截至2017年3月，就所開辦的個別學科送交「學科範圍評審」並獲得通過的教育機構，包括香港樹仁大學、香港演藝學院、珠海學院、恒生管理學院和職業訓練局屬下的部分培訓機構<ref>{{cite news |title=已通過學科範圍評審之營辦者名單 |url=https://www.hkcaavq.edu.hk/files/services/accreditation/List_of_Operators_awarded_PAA_Status_20170324_c.pdf |publisher=香港學術及職業資歷評審局 |date=2017-03-24}}</ref>。
# 「學科範圍評審資格」類似「自行評審資格」，但在學術範圍和時限上有所限制，並須接受定期檢討<ref>{{cite web|title=資歷架構相關評審|url=https://www.hkcaavq.edu.hk/zh/services/Accreditation|publisher=香港學術及職業資歷評審局}}</ref>。具有學科範圍評審資格的院校，可在指定的時限內，就其獲授予學科範圍評審的學科，開辦指定學術等級的課程，而無須先由香港學術及職業資歷評審局評審，但有關院校如要開辦不在指定學科範圍和指定學術等級的課程，該等課程仍要交由香港學術及職業資歷評審局評核。香港樹仁學院升格為[[香港樹仁大學|香港樹仁大學]]後的評審資格亦屬學科範圍評審資格<ref>[http://www.legco.gov.hk/yr06-07/chinese/panels/ed/papers/ed0108-embmpecr44204183-c.pdf 立法會參考資料摘要 香港樹仁學院更改名稱]</ref>。
# 「本地資歷」是指由香港課程營辦者在香港舉辦課程所頒授的學歷資格，按現行規定只有「法定院校」及「認可專上學院」具有頒授本地大學學位的資格。「非本地資歷」是指由香港以外之課程營辦者，通常是透過與香港的營辦者合辦並在香港舉辦課程所頒授的學歷資格。在香港沒有頒授學位資格的院校可通過引入由海外院校頒授學位的課程，並就其與海外院校合辦的課程向[[香港學術及職業資歷評審局|香港學術及職業資歷評審局]]申請學術評審，有關課程的評審如獲得學評局通過，將可登記在資歷名冊。因此沒有頒授本地學士學位資格的院校，都可與海外院校合辦本地認可的學士學位或以上資歷的課程<ref name="hkqr-qa">{{cite web|title=本地與非本地資歷的質素保證|url=http://www.hkqr.gov.hk/HKQRPRD/web/hkqr-tc/about/FAQ/QA/|publisher=[[香港教育局|香港教育局]]}}</ref>。
# 香港有部分本地院校與海外院校合作辦學，亦有海外院校在香港設立分校，該等院校所開辦的課程如純粹由海外院校頒授學歷資格，都屬於非本地學歷。由於香港未有規定頒授非本地學歷的海外課程必須送交並通過[[香港學術及職業資歷評審局|香港學術及職業資歷評審局]]的學術評審，所以該等課程不一定得到香港本地認可。一般而言，有關海外課程如能通過學評局的學術評審，並登記入[[香港資歷架構|香港資歷架構]]的資歷名冊，其海外課程頒授的非本地學歷將被視為等同於資歷名冊內註明的香港本地學歷程度<ref name="hkqr-qa">{{cite web|title=本地與非本地資歷的質素保證|url=http://www.hkqr.gov.hk/HKQRPRD/web/hkqr-tc/about/FAQ/QA/|publisher=[[香港教育局|香港教育局]]}}</ref>。雖然海外課程並不強制須要送交及通過香港學術及職業資歷評審局的本地學術評審，惟未經學評局評審的海外課程，其頒授的非本地學歷資格可能在香港不獲承認，在申請政府教育資助及進修貸款等方面，都會較已通過本地評審的課程受到較多限制。

# [[高峰進修學院|高峰進修學院]]是[[職業訓練局|職業訓練局]]的成員院校，該校開辦有不同程度的課程，包括以自負盈虧方式與海外院校合作，由海外院校頒授學歷資格的大學學位課程。該校部分學士學位及深造課程已通過香港學術及職業資歷評審局的學術評審及列入資歷名冊。由於該校與海外院校合辦的課程不屬於全日制學士學位課程，所以該校的課程均未獲納入免入息審查資助計劃的資助範圍。
# [[香港藝術學院|香港藝術學院]]是[[香港藝術中心|香港藝術中心]]的附屬院校，香港藝術中心並非由政府成立，而是由業界於1977年自行籌資設立的民營藝術空間<ref>{{cite news |title=藝術中心40年 香港第一代真文青聖地 洗脫文化沙漠污名 |url=https://bkb.mpweekly.com/cu0002/feature-藝文部落/20171017-55375 |work=明周文化 |issue=2017-10-17}}</ref>，政府於1986年通過《香港藝術中心條例》對其加以規管。香港藝術學院可開辦副學位程度的本地課程，至於其營辦的大學學位課程均屬合辦課程。該校部分非本地學士學位課程已經提交香港學術及職業資歷評審局進行評審，部分已通過學評局評審並在核證有效期內的非本地學士學位課程，獲納入為免入息審查資助計劃的資助範圍。
# [[香港科技專上書院|香港科技專上書院]]可開辦副學位程度的本地課程，至於其營辦的大學學位課程屬於與海外大學合辦的非本地學位課程，該校曾將部分由海外院校頒發學位的課程提交香港學術及職業資歷評審局作學術資格評審，部分已通過學評局評審並在核證有效期內的非本地學士學位課程，獲納入為免入息審查資助計劃的資助範圍。
# [[薩凡納藝術設計學院|薩凡納藝術設計學院]]屬下在香港註冊的有限公司“SCAD Foundation (Hong Kong) Limited”，於2012年在[[公司註冊處|公司註冊處]]加注中文名稱為「薩凡納藝術設計（香港）大學有限公司」，學院亦對外改稱為「薩凡納藝術設計大學（香港）」，由於該校的學歷由其所屬海外院校頒授，屬於非香港本地學歷資格，所以沒有違反香港對大學名銜使用的規定<ref>{{cite web|title=自稱「大學」無王管 薩凡納「升格」惹爭議|url=http://paper.wenweipo.com/2013/01/14/YO1301140017.htm|publisher=文匯報|accessdate=2016-06-13|date=2013-01-14}}</ref>。該校部分非本地學士學位課程已經提交香港學術及職業資歷評審局進行評審，部分已通過學評局評審並在核證有效期內的非本地學士學位課程，獲納入為免入息審查資助計劃的資助範圍。
# 芝加哥大學布思商學院（香港）是[[芝加哥大學|芝加哥大學]]於2014年在香港設立的分校，在香港開辦非本地學位課程。該校的行政人員工商管理碩士課程，已通過香港學術及職業資歷評審局的學術評審，但碩士課程並不屬於免入息審查資助計劃的資助範圍。
# [[薩凡納藝術設計學院|薩凡納藝術設計大學（香港）]]及[[芝加哥大學|芝加哥大學布思商學院（香港）]]於2016年7月獲[[中國教育部|中國教育部]]同意，就其已通過香港學術及職業資歷評審局評審的指定非本地學位課程取錄中國內地學生，於2016/17至2020/21學年期間入學<ref>{{cite news|title=兩院校非本地課程獲准招收內地學生|url=http://www.info.gov.hk/gia/general/201607/04/P201607040560.htm|publisher=香港政府新聞網|date=2016-07-04}}</ref>。雖然中國教育部已經同意這兩所海外院校在香港取錄中國內地學生，但是未有被列入《內地與香港關於相互承認高等教育學位証書的備忘錄》的互認學位的院校名單附件。
