'''香港甲組聯賽'''（簡稱'''港甲'''；{{lang-en|'''Hong Kong First Division'''}}）成立於2014年，是[[香港足球總會|香港足球總會]]於全新[[香港足球聯賽系統|香港足球聯賽系統]]中第二級別，首個球季於2014年開始。

== 成立背景 ==
2014年，醞釀多時的「[[鳳凰計劃|鳳凰計劃]]」正式推行，[[香港足球總會|香港足球總會]]為[[香港足球聯賽系統|香港足球聯賽系統]]重新定位。全新的[[香港超級聯賽|香港超級聯賽]]取代舊有的[[香港甲組足球聯賽|香港甲組足球聯賽]]成為[[香港|香港]]頂級聯賽，而舊有的低組別的聯賽，亦會作出相應更改。以往第二級別的[[香港乙組足球聯賽|香港乙組足球聯賽]]，易名為「香港甲組聯賽」。

== 賽事制度 ==
香港甲組聯賽一共有16隊參賽，以雙循環形式比賽，實行「升二降二」升降級制度，即聯賽冠軍及亞軍可獲得下屆升往[[香港超級聯賽|香港超級聯賽]]資格，而榜尾兩隊則需要降級至[[香港乙組聯賽|香港乙組聯賽]]。至於外援制度方面，甲組聯賽准許「註四出三」。<ref>{{cite web|url=http://www.hkfa.com/upload/rules/123divleague1415.pdf |title=香港足球總會 二零一四/一五年度甲、乙、丙組聯賽章則 【二零一四/一五年度版】|publisher=香港足球總會|format=PDF}}</ref>
