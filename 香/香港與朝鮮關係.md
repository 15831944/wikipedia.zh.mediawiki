'''香港與朝鮮關係'''是指[[中华人民共和国|中华人民共和国]][[香港特别行政区|香港特别行政区]]和[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]之間的{{link-en|双边关系|Bilateralism}}。

==  歷史 ==
從朝鮮民主主義人民共和國建立後一直到1997年香港成為[[特别行政区|特别行政区]]這段時間，兩地毫無官方关系。<ref name="NanHua01" />在[[朝鲜战争|朝鲜战争]]和[[冷战|冷战]]時，双方都在敵對的两侧。[[香港|香港]]作为[[英國|英國]]的[[殖民地|殖民地]]，屬於[[美國|美國]]領導的[[资本主义|资本主义]]陣營，而[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]則是屬於[[苏联|苏联]]領導的[[社會主義|社會主義]]陣營。香港在20世纪上半叶開始針對[[東方集團|東方集團]]實施[[禁運|禁運]]，不允许朝鮮派[[貿易|貿易]]代表團到香港。

[[香港回歸|香港回歸]]後，朝鮮憑藉其與北京的[[外交關係|外交關係]]在2000年2月於[[香港島|香港島]][[灣仔|灣仔]]設了一個[[领事|總領事館]]。<ref name="NanHua01" />

== 脫北者 ==
香港曾多次公開庇護[[脫北者|脫北者]]。[[主權|主權]]移交中國前的在1996年，一個16人的家庭經由中國從朝鮮逃往香港，當中一名孕婦和5名小孩聲言尋求[[庇護權|政治庇护]]，而香港政府當時協助他們前往南韓。<ref name="LiaoYunSi" />

主權移交後首次則是在2016年七月底。一名朝鮮派至[[香港科技大學|香港科技大學]]參加[[国际奥林匹克数学竞赛|国际奥林匹克数学竞赛]]的代表逃到了位在[[夏慤道|夏慤道]][[遠東金融中心|遠東金融中心]]5樓的[[大韓民國駐香港總領事館|大韓民國駐香港總領事館]]。<ref name="NanHua02" /><ref name="MingBao" />這名脫北者於香港逗留80餘日後，在2016年9月24日前往南韓。<ref name="XieMingMing" />

== 經濟關係 ==
香港和朝鮮在國際上會合作有關於社會和文化方面的事務。儘管香港是一個[[自由市場經濟|自由市場經濟體]]<ref name="Heritage" />，朝鮮則採[[計劃經濟|計劃經濟]]，但香港成功的經濟被朝鮮視為典範，用來建設[[新義州特別行政區|新義州特別行政區]]，自1997年以來，這個50平方英里的地區正在被[[朝鲜政府|朝鮮政府]]改造為一個[[自由貿易區|自由貿易區]]。许多人認為新義州將成為朝鮮未來的香港。<ref name="Demick" />

== 民間關係 ==
朝鮮與香港的民間關係十分友好，2018年3月，[[香港足球代表隊|香港足球代表隊]]支持者就獲准在[[平壤|平壤]][[金日成競技場|金日成競技場]]為作客的港隊打氣，在[[平壤|平壤]]，作客球隊球迷一般只准觀賽，不准為其支持的球隊打氣，但香港隊則屬例外。<ref>{{Cite news|url=https://www.hk01.com/%E5%8D%B3%E6%99%82%E9%AB%94%E8%82%B2/173554/%E4%BA%9E%E7%9B%83%E5%A4%96-%E6%8A%95%E7%A8%BF-%E6%B8%AF%E8%B6%B3%E8%BF%B7%E6%92%90%E9%A6%99%E6%B8%AF%E9%9A%8A%E6%92%90%E5%88%B0%E6%9C%9D%E9%AE%AE-%E8%A6%AA%E6%AD%B7%E7%90%83%E5%A0%B4%E5%85%A7%E8%A2%AB-%E7%8B%99%E6%93%8A|title=【亞盃外．投稿】港足迷撐香港隊撐到朝鮮　親歷球場內被「狙擊」|last=香港01記者|date=2018-03-30|newspaper=香港01|accessdate=2018-06-19|language=zh-HK}}</ref>

朝鮮駐港總領事Jang Song Chol近年開始應邀出席每年[[佛誕|佛誕]]在[[長洲_(香港)|長洲]]舉行的[[太平清醮|太平清醮]]活動。
