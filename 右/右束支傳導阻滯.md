'''右束支傳導阻滯'''（Right bundle branch block，簡稱'''RBBB'''）是一類[[心臟電傳導系統|心臟電傳導系統]]{{tsl|en|heart block|心阻斷|阻滯疾病}}<ref name="urlConduction Blocks 2006 KCUMB">{{cite web |url=http://courses.kcumb.edu/physio/blocks/index.htm |title=Conduction Blocks 2006 KCUMB |work= |accessdate=2009-01-20}}</ref>。乃肇因於心臟的右束支傳導阻斷，進而造成電訊號無法經由此途徑傳入右心室，而必須由來自左心室的訊號活化。

==機轉==

右束支傳導阻滯乃肇因於心臟的右束支傳導阻斷，進而造成電訊號無法經由此途徑傳入右心室，而必須由來自左心室的訊號活化。但來自左心室的電訊號必須經過{{tsl|en|myocardium|心肌}}傳入，而此途徑的傳遞速度較原先的{{tsl|en|Bundle of His|希氏束}}-[[浦金氏纖維|浦金氏纖維]]路徑要慢，因此在心電圖上，患者的{{tsl|en|QRS complex|QRS複合波}}會較寬。同時，由於左心室的去極化速度快於右心室，因此在QRS波會產生心軸偏移的狀況。


大多是患者具有其他病因，但也有1.5-3%的在整體上相當健康的患者<ref>{{cite journal|pmc=1122450 | pmid=11872557 | volume=324 | issue=7336 | title=Bradycardias and atrioventricular conduction block |date=March 2002 |vauthors=Da Costa D, Brady WJ, Edhouse J | journal=BMJ | pages=535–8 | doi=10.1136/bmj.324.7336.535}}</ref>。

右束支傳導阻滯的診斷需依賴[[心电图|心电图]]，有下列幾個診斷標準：
* 心室節律必須來自心室之上（即[[窦房结|窦房结]]、[[心房|心房]]，或[[房室结|房室结]]）
* QRS複合波時間需長於100毫秒（非完全阻斷），或120毫秒（完全阻斷）<ref name="urlLesson VI - ECG Conduction Abnormalities">{{cite web |url=http://library.med.utah.edu/kw/ecg/ecg_outline/Lesson6/index.html#RBBB |title=Lesson VI - ECG Conduction Abnormalities |work= |accessdate=2009-01-07}}</ref>
* V1導極有末期R波（如：{{tsl|en|QRS complex|QRS複合波|R、rR'、rsR'、rSR'，或qR}}等波形）
* 第一導極和V6導極的S波寬淺

=== 相關變化 ===

* RBBB右胸前導極（V1、V2、V3）可能會有正常的T波倒置和ST段下降<ref name=":0">{{Cite news|url=https://lifeinthefastlane.com/ecg-library/basics/right-bundle-branch-block/|title=Right Bundle Branch Block (RBBB) LITFL ECG Library|newspaper=LITFL • Life in the Fast Lane Medical Blog|accessdate=2018-06-08|language=en-US}}</ref>，若T波無倒轉反而可能是心肌缺血或[[心肌梗塞|心肌梗塞]]。

==病因==
可能導致RBBB的疾病包含：[[心房中隔缺损|心房中隔缺损]]<ref name=Cecil>{{cite book|last=Goldman|first=Lee|title=Goldman's Cecil Medicine|publisher=Elsevier Saunders|location=Philadelphia|year=2011|isbn=1437727883|pages=400–401|edition=24th}}</ref>、[[布魯格達氏症候群|布魯格達氏症候群]]、{{tsl|en|right ventricular hypertrophy|右心室肥大}}、[[肺栓塞|肺栓塞]]、[[冠狀動脈疾病|缺血性心臟病]]、[[風溼熱|風溼熱]]、[[心肌炎|心肌炎]]、[[心肌症|心肌病變]]，或[[高血壓|高血壓]]等等<ref name=":0" />。

==流行病學==
RBBB的盛行率隨年紀增加。

== 治療 ==
若有潛在原因，需治療其潛在原因，如高血壓或糖尿病等等。若是因為冠狀動脈阻塞，則可進行[[血管再成形術|血管再成形術]]，來改善RBBB的現象<ref>{{Cite web|title = Right Bundle Branch Block|url = https://www.symptoma.com/en/info/right-bundle-branch-block|website = www.symptoma.com|accessdate = 2015-08-13}}</ref>。
