'''栎属'''（又稱'''麻櫟屬'''，俗名'''橡树'''、'''櫟樹'''或'''柞树'''）是[[壳斗科|壳斗科]]的一个[[属|属]]。栎属有600个种，其中450种来自[[栎亚属|栎亚属]]和150则是[[青剛櫟亞屬|青剛櫟亞屬]]。主要分布在北半球与哥伦比亚的安第斯山脉。 <ref > {{Cite web |url=https://docs.google.com/gview?url=http://frps.plantphoto.cn/pdf/22/213y.pdf&embedded=true |title=中國植物誌 |accessdate=2011-08-26 |work= |publisher= |date= }}</ref>栎树可以生长在从海平面高度到海拔4000米的地方，不同海拔生长着不同的特征品种，如[[常绿植物|常绿树]]和半常绿树品种。

== 描述 ==

栎树属于硬木，生长缓慢，树干外有厚而粗糙的树皮，富含[[鞣质|丹宁酸]]。这种天然产生的化学物质被用于制革、酿酒。因为单宁酸的存在，栎木木材成了绝佳的建筑材料，因为它可以防虫防腐烂。栎树树心的纤维细胞有极厚的[[細胞壁|细胞壁]]，提高了栎树的强度。

栎树的原产地可能是[[亚洲|亚洲]]，并且在5500年前被广泛传播到世界各地。现在栎树的分布情况反映出了最近一个[[地质年代|地质年代]]所发生的生态与气候变化。栎树是风媒植物，因此，和其他许多植物一样，[[传粉|授粉]]成功并发芽是一个偶然事件，[[松鼠|松鼠]]对栎树的传播也起到了一定的帮助作用，它们常常将[[橡子|橡果]]藏在地底，之后便忘记了藏东西的地方（或自身遭受意外），大多数栎树品种需要在温和的气候条件和肥沃湿润的土壤中生长。当生长条件适宜的时候，栎树的树龄可达几百年，并长成参天大树。

== 文化与历史 ==
在原始的宗教，神话和传说中，栎树是神圣的。在不同的文明里，栎树被看作不同的天神。不同的栎树品种也有其不同的作用。

[[夏櫟|夏栎]]（''Q. robur''）原产自[[英国|英国]]和[[北歐|欧洲北部]]大部分国家，是最受欢迎的一个品种。由于在金属船身出现之前，用于造船的[[木材|木材]]需要满足以下几个要求：木材需要来自粗壮的[[樹幹|树干]]，制成的木板要具有延展性，轻便且防水。栎树满足上述的两个条件，所以夏栎一直是建造船只的选择，这也帮助英国取得了海上霸权。因此，自17世纪以来，主要生长在英国的栎树被出口至[[斯堪的纳维亚半岛|斯堪的纳维亚]]和其他[[波罗的海国家|波罗的海国家]]，以满足他们建造船只房屋，家具以及使用木柴的需求。

最常见的美洲品种为白橡（''Q.alha''），它是马萨诸塞海湾公司的摇钱树。马萨诸塞海湾公司用船将木材运回英国，并帮助船上的移民横渡大西洋。另一个美洲品种为弗吉尼亚栎（''Q. virginia- na''），被誉为“活栎树”，是存活时间最长的品种，现在较少作为商业木材。

另一个栎木品种为栓皮栎（''Q. Suber''），是制作[[軟木塞|软木塞]]的原料，深受[[红葡萄酒|红酒]]爱好者的青睐。这种半常绿植物的树皮极厚，古罗马人用它做隔热材料，鞋锚浮标及瓶塞，这些用法至今仍很常见。栓皮栎树皮本身经过进化，具有一定的防火性。栓皮栎的树龄可达100年以上，每10年可以剥去它的树皮加以利用，但每次剥树皮必须要等到上次剥去的树皮再生长出来。

加在20世纪之前，橡果一直是美洲[[美洲原住民|印第安人]]的食物（在去除丹宁酸之后，橡果是可食用的），特别是在[[加利福尼亚州|加利福尼亚]]，现在很少有人食用橡果。但是在盛产火腿的[[意大利|意大利]]与[[伊比利亚半岛|伊比利亚]]，橡果仍然是猪饲料的主要原料。<ref>{{Cite book|title=Wood: Craft, Culture, History|last=Green|first=Harvey|publisher=Penguin|year=2007|isbn=978-0143112693|location=|pages=}}</ref>
