'''栎社国际机场站'''，建设时曾称'''机场站'''，位于[[浙江省|浙江省]][[宁波市|宁波市]][[海曙区|海曙区]][[石碶街道|石碶街道]]，横跨栎高线，是[[宁波轨道交通|宁波轨道交通]]的一座地下车站，[[宁波轨道交通2号线|2号线]]的西南端终点站。车站于2010年12月开始建设，2015年9月26日启用<ref name="main">{{cite web|url=http://news.cnnb.com.cn/system/2010/12/23/006790463.shtml|title=2号线车站设计人性化而有宁波味 效果图抢先看|author=陈锦源|publisher=中国宁波网|date=2010-12-23|accessdate=2011-01-21}}</ref>。车站与[[寧波櫟社國際機場|寧波櫟社國際機場]]2号航站楼南部的机场交通中心相连，可直达2号航站楼。<ref name="airport">{{cite web|url=http://news.cnnb.com.cn/system/2015/12/02/008437543.shtml|title=宁波机场T2航站楼开工 4年后坐地铁赶飞机无需接驳了|author=郭静|publisher=中国宁波网|date=2015-12-02|accessdate=2019-02-17}}</ref>

== 车站形制 ==
栎社国际机场站长350米，标准段宽19.3米，深16.81米，设3个出口，4组风井。地下一层为站厅，二层为[[岛式车站|岛式车站]]，宽12米，东段包含交叉渡线<ref name="main" />。

== 车站装饰 ==
[[File:Lishe_International_Airport_Station,_2015-10-01_09.JPG|thumb]]
栎社国际机场站为2号线装修重点站之一。车站装修主题为天空，使用蓝色作为主色调，同时采用云纹作为吊顶装饰<ref name="decoration-1">{{cite web|url=http://auto.zjol.com.cn/system/2014/12/17/020415591.shtml|title=宁波地铁2号线栎社机场站亮相 装修玩起中国风|publisher=钱江晚报|date=2014-12-17|accessdate=2014-12-17}}</ref>。车站文化墙主题为“空港筑梦”。文化墙材质为铝板，通过世界地图和以宁波为中心的同心圆和虚实结合的线条形成的光学特效形成风的效果，寓意宁波通过机场连接世界<ref>{{cite web|url=http://www.nbmetro.com/news_dt.php?info/961|title=2号线一期6大车站文化墙亮相|publisher=宁波轨道交通集团|date=2015-09-23|accessdate=2015-09-27}}</ref>。

== 交通接驳 ==
栎社国际机场站接驳公交共任站<ref name="query" />。同时，B出口可乘坐[[宁波栎社国际机场|宁波栎社国际机场]]提供的免费接驳巴士往返于机场<ref>{{cite web|url=http://www.ningbo-airport.com/kgzx/kgdt/2147.jhtml|title=轨道交通2号线机场免费大巴接驳方案公布|publisher=栎社国际机场|date=2015-09-25|accessdate=2018-01-16}}</ref>。
