'''密克罗尼西亚联邦'''（{{lang-en|Federated States of Micronesia}}），通稱'''密克羅尼西亞'''，是一个西[[太平洋|太平洋]]岛国，属于[[加罗林群岛|加罗林群岛]]，全国陆地面积702平方公里。全国有607个岛屿，[[波纳佩岛|波纳佩岛]]为最大的岛屿，首都[[帕利基尔|帕利基尔]]位于该岛上。

== 历史 ==

密克罗尼西亚群岛上原住民（[[密克罗尼西亚|密克罗尼西亚人]]）约4000年前就在該處生活。最早到達島上的歐洲人是[[西班牙|西班牙]]人，時為16世纪，並於1885年被佔領。之后在1899年，西班牙人把卡罗林群岛转让给[[普魯士|普魯士]]。

[[第一次世界大戰|第一次世界大戰]]後德國在太平洋的殖民地被安排給予[[大日本帝国|日本]]作[[國際聯盟託管地|託管地]]，[[第二次世界大戰|第二次世界大戰]]後由[[美国|美国]]佔領。1947年，[[联合国|联合国]]将密克罗尼西亚交美国託管，現今密克羅尼西亞聯邦範圍為[[太平洋群島託管地|太平洋群島託管地]]六區的其中三區（[[科斯雷|科斯雷]]於1977年自[[波纳佩|波纳佩]]獨立成為第四區）。1965年1月成立议会，此后要求自治。

1979年，由聯合國和美國籌劃的「密克羅尼西亞憲法」公投在託管地六區舉行（[[北馬里亞納群島|北馬里亞納群島]]已於1976年成為美國的[[自由邦_(美国)|自由邦]]）。憲法公投在[[帛琉|帛琉]]和[[馬紹爾群島|馬紹爾群島]]兩區未能通過，剩餘四區便組織成為「密克羅尼西亞聯邦」。

1986年11月3日，与美国签订的《[[联系邦|自由聯合國家條約]]》正式生效，密克罗尼西亚联邦获得内政、外交自主权，但[[國防|國防]]仍由美国负责（密克羅尼西亞公民也可加入美軍）。

1990年12月，联合国安理会召开会议，通过了终止託管决议，正式结束了密克罗尼西亚联邦的託管地位，并于1991年9月17日接纳密克羅尼西亞聯邦为联合国正式会员国。

== 政治 ==
{{main|密克羅尼西亞聯邦政治}}

根据1979年5月10日通过并生效的[[宪法|宪法]]规定，[[密克罗尼西亚联邦总统|联邦总统]]为国家元首，也是政府首脑。正副總統由國會議員當中互選產生，然後在兩人所屬州份再舉行特別選舉填補空缺。

國會為一院制，由来自4个州的14名议员组成，其中10名议员任期2年，其他4名为任期4年的“全任期”议员。

== 人口 ==
{{main|密克羅尼西亞聯邦人口}}

人口大約有108,500人（2005年）。其中密克罗尼西亚人占97%。
按美、密兩國條約規定，密聯公民可自由在美國居住、就讀和工作，無需簽證（犯事被驅逐者除外）。密聯公民定居美國者超過一萬人，其中以[[夏威夷|夏威夷]]、[[關島|關島]]和[[北馬里亞納群島|北馬里亞納群島]]最集中。

全國語言繁多（主要属于[[南島语系|南島语系]]），每一小島群都有自己的語言。四個大島的語言是州級的官方語言。英語則是州際交流的國家官方語言，也是學校的必修語言。

== 宗教信仰 ==
[[天主教|天主教]]徒佔50%，基督[[新教|新教]]徒佔47%，其他教派和沒有信仰者佔3%。

由於歷史上受歐洲的影響，密克羅尼西亞聯邦西部（近菲律賓）天主教徒較多，東部則新教徒較多。

== 经济 ==
{{main|密克羅尼西亞聯邦經濟}}

密克羅尼西亞聯邦[[經濟專屬海域|經濟專屬海域]]（EEZ）海域面积大，渔业资源丰富，尤以[[金枪鱼|金枪鱼]]著名。其他粮食及生活日用品均靠进口，经济较为落后。

[[GDP|GDP]]：3.18亿美元（2015年估计），人均[[GDP|GDP]]：3,072美元（2015年估计）。

密克羅尼西亞聯邦使用美元，進口貨也多依賴美國。根據美、密兩國間的《自由聯繫條約》，密克羅尼西亞聯邦在多項政府開銷、經濟發展項目都獲美國政府資助。[[美國郵政|美國郵政]]也負責密克羅尼西亞聯邦州際和對外郵遞，郵資與美國國內一致。

== 教育 ==
{{main|密克羅尼西亞聯邦教育}}

政府重视发展教育事业，宪法规定对5～14岁儿童实行强制性义务教育制，政府每年在教育上投入经费占年度预算的20%左右。

全國最高學府是相等於美國兩年制社區學院的「密克羅尼西亞學院」（College of Micronesia），除首都本部外還在其餘三大島設有分校。四年制大學教育方面，密國學生在關島大學可獲關島居民學費待遇，在夏威夷大學只需付比夏威夷州民高50%的學費。

== 旅游 ==
密克罗尼西亚旅游资源较为丰富，热带风光秀丽。旅游业是其经济发展的一个重要项目，但仍处于开发阶段。2003年接待游客18154人次，美国游客最多。

由於人口少，交通也頗為落後。四大島設有國際機場，但只有約每日一個航班。來往外島得靠中、小型船舶，可能好幾天才有一班。

== 外交 ==
{{Details|密克罗尼西亚联邦建交列表}}

截至2017年，密克羅尼西亞聯邦已與80个国家建交，但只有在幾個亚太地區國家和世界大國设有[[大使馆|大使馆]]，包括[[澳大利亞|澳大利亞]]、[[美國|美國]]、[[中華人民共和國|中華人民共和國]]和[[日本|日本]]等。
