'''密云区'''是[[中華人民共和國|中国]][[北京市|北京市]]东北部的一个[[市辖区|市辖区]]，西、南与[[怀柔区|怀柔区]]及[[顺义区|顺义区]]相邻，东、北与[[河北省|河北省]]相接。2015年11月，经[[中国国务院|中国国务院]]批准，撤销密云县，设立密云区<ref>{{cite news|title=北京撤销密云县、延庆县 设立密云区、延庆区|url=http://www.chinanews.com/gn/2015/11-17/7627861.shtml|accessdate=2015-11-17|agency=中国新闻社|location=北京|language=zh-hans}}</ref><ref>{{cite web|author1=北京市人民政府|title=北京市人民政府关于撤销密云县、延庆县设立密云区、延庆区的通知|url=http://zhengwu.beijing.gov.cn/gzdt/gggs/t1411195.htm|website=|publisher=首都之窗|accessdate=2015-11-17|location=北京|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20151118130817/http://zhengwu.beijing.gov.cn/gzdt/gggs/t1411195.htm|archivedate=2015年11月18日|df=}}</ref>。

建有[[密云水库|密云水库]]；[[古北口|古北口]]为历史上交通重镇。

== 地理概况 ==
地处华北平原北端，为华北平原与燕山山脉之交接。全区面积2226.5平方千米，现有人口47.8万（2014年末），大部分从事农业，主要居住在原[[燕落盆地|燕落盆地]]周边的[[冲积平原|冲积平原]]地区。

全区土地基本上处于山区半山区，农业用地很少，有“八山一水一分田”的说法。

密云区林木众多，森林覆盖率超过70%。

由于处于华北、京津的北方门户，密云自古就是华北的战略要冲。绵延的[[燕山|燕山]]山脉阻断了北方的交通，只在[[古北口|古北口]]一带留有狭小通路。

== 历史 ==
最早的人类活动历史可以追溯到[[原始社会|原始社会]]的[[旧石器时代|旧石器时代]]（亦有[[新石器时代|新石器时代]]的看法），距今10万年。

因其良好的[[冲积平原|冲积平原]]位于气候恶劣的北方，古时有“燕国天府”的称呼。

=== 史书记载 ===
==== 原始部落时期 ====
山安口村出土的石器、陶器，充分证明早在[[新石器时代|新石器时代]]，密云区不老屯地区就有人类居住。燕落村南[[密云水库|密云水库]]下的[[共工城|共工城]]遗址，距今4100多年，是原始部落时期尧的臣子[[共工|共工]]（掌管手工业的大臣）流放居住的土城，是[[北京地区|北京地区]]第一座古城。《[[史记|史记]]·五帝本纪》载：“舜请流共工于幽陵。”而《[[括地志辑校|括地志辑校]]》则-{云}-：“故共工城在檀州燕落县界”。

==== 战国 ====
[[战国|战国]]时期为[[燕|燕]]地，名[[渔阳郡|渔阳郡]]。《[[顺天府志|顺天府志]]》-{云}-：“[[燕王喜|燕王喜]]十二年置渔阳郡和[[渔阳县|渔阳县]]”。行政机构位于现在的统军庄南。

==== 秦 ====
公元前221年秦[[统一中国|统一中国]]，实行[[郡县制|郡县制]]，[[秦始皇|始皇]]二年设[[渔阳郡|渔阳郡]]，为[[三十六郡|三十六郡]]之一。[[兩漢|兩漢]]及[[曹魏|曹魏]]沿用。

==== 南北朝、隋唐、五代 ====
[[北魏|北魏]]时开始使用密云为的称谓。《魏书》载：“[[皇始_(北魏)|皇始]]二年置[[密云郡|密云郡]]、[[密云县|县]]，治提携城，领密云、[[白檀县|白檀]]、[[要阳县|要阳]]三县”。

《[[隋书|隋书]]》记载：“密云后魏置密云郡，领白檀、要阳、密云三县。后齐废郡及二县入密云。又有旧[[安乐郡_(北魏)|安乐郡]]，领[[安市县|安市]]、[[土垠县|土垠]]二县，后[[北齐|齐]]废土垠入安市，后[[北周|周]]废安市入密云县。开皇初郡废。有长城。有桃花山、螺山。有渔水。”

==== 辽史 ====
檀州，武威军，下，刺史。本燕渔阳郡地，汉为白檀县。
密云县。本汉白檀县，后汉以居憕奚。元魏置密云郡，领白檀、要阳、密云三县。

=== 战事 ===
由于特殊的地理位置，密云历来成为兵家必争之地，可记述的著名战事如下：

==== 秦开对[[东胡|东胡]] ====
[[燕昭王|燕昭王]]时，将自己大将[[秦开|秦开]]主动送给胡人作为人质，保证不会进犯，实际是稳定对方。
不过送秦开的计谋还取得的更多收获，秦开在胡期间受到礼遇，且探察了当时胡人的军事、地理。
燕昭王二十六年（公元前286年），命秦开进攻东胡。秦开大败胡人，直到[[大兴安岭|大兴安岭]]，俘虏众多。
后秦开请示燕昭王，仅留下马匹供军用，将妇女、儿童和其他物资送还。
后设渔阳郡，辖：[[顺义|顺义]]、[[怀柔|怀柔]]、密云、[[滦平|滦平]]、[[隆化|隆化]]、[[围场|围场]]、[[蓟县|蓟县]]等地。

==== 安史之乱 ====
公元755年，[[安禄山|安禄山]]率密云在内的共九郡15万，向唐王朝发动军事进攻。[[白居易|白居易]]的《[[长恨歌|长恨歌]]》中名句“渔阳鼙鼓动地来，惊破霓裳羽衣曲”即指此事。八年后，安禄山、[[史思明|史思明]]战死，[[安史之乱|安史之乱]]结束。

==== 民国之后的战事 ====
1933年，日本以[[熱河省|熱河省]]地方官員表示歸附[[滿洲國|滿洲國]]為由，與[[滿洲國軍|滿洲國軍]]隊進軍熱河，省長[[湯玉麟|湯玉麟]]不戰而逃；之後日軍進攻[[山海關|山海關]]、[[長城|長城]]各[[隘口|隘口]]與熱河，[[國民政府|國民政府]]派遣[[宋哲元|宋哲元]]、[[關麟徵|關麟徵]]與[[黃傑|黃傑]]與[[日本軍|日本軍]]在[[長城戰役|義院口、冷口、喜峰口、古北口、羅文峪、界嶺口各處]]鏖戰。
