
'''密勒日巴'''（{{bo|z=རྗེ་བཙུན་མི་ལ་རས་པ|p=Jêbzün Milaräba}}，{{bd|1052年||1135年|}}），又譯為'''米拉日巴'''，[[噶舉派|噶舉派]]上師，根本上師為[[馬爾巴|馬爾巴]]，其本尊為[[勝樂金剛|勝樂金剛]] <ref>{{Cite web |url = http://www.thrangu.org/teachings/kagyu-lineage/golden-rosary/milarepa/ |title = 密勒日巴 - 創古仁波切官網 }}</ref>。家族本名密勒聞喜，為西藏最著名的密教修行者，也是西藏最偉大詩人，是西藏許多民間傳說的主角。
[[File:Jetsun_Milarepa.jpg|thumb]], [[奧特根巴亚尔·额尔舒|奧特根巴亚尔·额尔舒]]]]

== 生平 ==

密勒日巴生在西藏芒域（今[[吉隆县|吉隆县]]）贡堂（{{bo|z=མང་ཡུལ་གུང་ཐང༌།|p=Mang'yügungtang}}）一個富裕的家庭中，幼時名為聞喜（{{bo|z=མི་ལ་ཐོས་པ་དགའ།|p=Mila Töbäga}}）。因為父親早死，家產被伯父跟姑姑據為己有，從富家少爺頓時成淪為受虐小童工。聞喜成年後，母親要求依承諾歸還家產未果，他與母親、妹妹被趕出去，孤苦無依。他母親送他去學習咒術，回來報仇；他學成後，以法術咒死他貪婪的親戚，奪回家產，並且降下冰雹來懲罰他鄉里的人。但也因此犯下惡業，被鄉里的人所恐懼與厭惡。

因為良心不安，他決心至[[馬爾巴|馬爾巴]]上師處，學習正法密法，以消除他的惡[[業|業]]。馬爾巴給他許多艱苦的磨練，不斷考驗他的心志，最後認同他的努力，將口訣傳承給他，並為他取法名為喜笑金剛(Shepa Dorje)。

密勒日巴在多處山洞中長期的禪坐修苦行，最終得到證悟而解脫，成為藏傳佛教中著名的密宗上師。

== 弟子 ==
他最著名的弟子有[[惹瓊巴|惹瓊巴]]與[[岡波巴|岡波巴]]。
