《'''密道追踪之阴兵虎符'''》（英語：Tomb Robber），是中国大陆首部[[3D|3D]][[冒险片|冒险电影]]，剧情改编自网络写手蛇从革（笔名：老蛇）的盗墓力作《[[密道追踪|密道追踪]]》，导演俞岛，主演李炳渊、母其弥雅、唐文龙、李涛，该片于2014年12月5日在中国大陆上映。<ref>{{cite web |url=http://ent.ifeng.com/a/20140917/40310679_0.shtml |title= 《密道追踪》曝概念海报 阴兵虎符霸气露峥嵘|accessdate=2014 |publisher=凤凰网 |work= |language=zh-hans}}</ref>导演赢得中美电影节“金天使奖”最佳新晋导演奖。<ref>{{cite web |url=http://ent.qq.com/a/20141107/038097.htm#p=1 |title= 《密道追踪》导演俞岛获中美电影节最佳新导演|accessdate=2014 |publisher= |work= 腾讯网|language=zh-hans}}</ref>

==剧情==
该片讲述一群盗墓者围绕着阴山下的无价之宝“阴兵虎符”展开争夺宝贝之战的故事。
