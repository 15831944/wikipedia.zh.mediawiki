'''密室之不可靠岸'''（英文名：'''Lost In Panic Cruise'''），2011年10月27日上映于[[中国大陆|中国大陆]]的悬疑片。

== 剧情 ==
该片讲述[[侦探|侦探]]柳飞云参加好友董磊的在轮船上的婚礼，并智破轮船上发生的杀人案。<ref>{{cite web |url=http://ent.qq.com/d/movie/36/35105/ |title=密室之不可靠岸 (2011) |accessdate=2011年|work=|publisher=腾讯网 |language = zh-hans}}</ref>

== 幕后 ==
该片拍摄异常艰辛，主演[[苏有朋|苏有朋]]说他们拍摄的主要时间都是在一艘轮船上虽然可以每天观赏[[大海|大海]]的美丽风情，却在[[吊威亚|吊威亚]]上付出了辛勤的汗水，少数累的日子里睡眠都不足三小时。<ref name="txs " >{{cite web |url=http://ent.qq.com/a/20110624/000041.htm |title= 《密室之不可靠岸》继续惊悚 苏有朋演侦探|accessdate= 2011年|work=|publisher=腾讯网 |language = zh-hans}}</ref>

该片主创作团队认为该片将获得一亿元的票房收入，主演[[苏有朋|苏有朋]]承诺：如果达到了，将裸游。<ref>{{cite web |url=http://ent.qq.com/a/20110629/000301.htm  |title= 《密室之不可靠岸》发海报 冲击一亿主创有信心|accessdate= 2011年|work=|publisher=腾讯网 |language = zh-hans}}</ref>
