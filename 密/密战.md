'''密战'''（{{lang-en|''Eternal Wave''}}），是於2017年上映的中国谍战动作电影。改编自电影《[[永不消逝的电波|永不消逝的电波]]》，以中国共产党党员[[李白_(中共党员)|李白]]为原型<ref>{{Cite web |url=http://www.liuyangshi.cn/liuyangxinwen/lhsd/2017/165939.html |title=电影《密战》主角原型竟是这位浏阳人！他的故居就在...|accessdate=2017-12-17 |author= |date=2017-11-22 |work=|publisher=浏阳之窗|language=Zh-hans }}</ref>。由北京南海影业有限公司、北京风骨文化传播有限公司出品，钟少雄执导，[[郭富城|郭富城]]、[[赵丽颖|赵丽颖]]、[[张翰|张翰]]等主演<ref>{{Cite web |url=http://ent.sina.com.cn/m/c/2017-09-02/doc-ifykpysa2687180.shtml |title=谍战片《密战》造型公开 郭富城和赵丽颖演夫妇|accessdate=2017-12-17 |author= |date=2017-09-02 |work=罗罗|publisher=新浪娱乐|language=Zh-hans }}</ref>。2017年11月3日于中国大陆地区上映<ref>{{Cite web |url=http://ent.sina.com.cn/m/c/2017-09-14/doc-ifykymue5976039.shtml |title=《密战》定档11月3日 郭富城赵丽颖张翰生死暗战|accessdate=2017-12-17 |author= |date=2017-09-14 |work=Koyo|publisher=新浪娱乐|language=Zh-hans }}</ref>。
