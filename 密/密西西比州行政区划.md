美国[[密西西比州|密西西比州]]共有82个[[县_(美国)|县]]。州内大部份的县名都是以一些傑出的政治人物来命名的。除此之外，也有少數的县是以同名的河流、湖泊来命名的。

根据[[2010年美国人口普查|2010年美国人口普查]]的数据，密西西比州平均每个县拥有36,448人，其中[[海恩茲縣_(密西西比州)|海恩茲縣]]以245,285人高居榜首，[[伊薩奎納縣_(密西西比州)|伊薩奎納縣]]则仅有1,406人，是全州人口最少的县<ref name="Census FactFinder">{{cite web |title = American FactFinder |url = http://factfinder2.census.gov/faces/nav/jsf/pages/index.xhtml |publisher = U.S. Census Bureau |accessdate = 2013-11-18 |archive-url = https://www.webcitation.org/65WtHj6J5?url=http://factfinder2.census.gov/faces/nav/jsf/pages/index.xhtml |archive-date = 2012-02-17 |dead-url = no }}</ref>。平均每个县的面积为1,530[[平方公里|平方公里]]。最大的是2,419平方公里的[[亞祖縣_(密西西比州)|亞祖縣]]，最小的是1,039平方公里的[[奧爾康縣_(密西西比州)|奧爾康縣]]<ref name="Census Gazetteer">{{cite web |title = 2010 Census U.S. Gazetteer Files |url = http://www.census.gov/geo/www/gazetteer/gazetteer2010.html |publisher = U.S. Census Bureau |accessdate = 2012-10-21 |deadurl=yes |archiveurl = https://web.archive.org/web/20121021075704/http://www.census.gov/geo/www/gazetteer/gazetteer2010.html |archivedate = 2012-10-21}}</ref>。

以下表格会在每一个县的名称后面列出其[[联邦资料处理标准|联邦资料处理标准]]代码（简称{{lang|en|FIPS}}代码），该代码是联邦政府用来区分各州和各县的唯一识别码<ref>{{cite web |url = http://www.itl.nist.gov/fipspubs/fip6-4.htm |title = FIPS Publish 6-4 |publisher = National Institute of Standards and Technology |accessdate = 2013-11-18 |archiveurl = https://web.archive.org/web/20130929074056/http://www.itl.nist.gov/fipspubs/fip6-4.htm |deadurl = yes |archivedate = 2013-09-29 }}</ref>。密西西比州的代码是28，结合任何一个县的代码则为“{{lang|en|28XXX}}”。所有的{{lang|en|FIPS}}代码将链接至该县最近的一次人口普查数据<ref name="epa">{{cite web|url=http://www.epa.gov/enviro/html/codes/al.html |title=EPA County FIPS Code Listing |accessdate=2013-11-18 |work=US Environmental Protection Agency |archivedate=2012-10-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20121008031339/http://www.epa.gov/enviro/html/codes/al.html }}</ref><ref name="FIPS">{{cite web|url= http://www.epa.gov/enviro/html/codes/mn.html|title= EPA County FIPS Code Listing|publisher= EPA|accessdate= 2007-04-09|archive-url= https://web.archive.org/web/20040930210232/http://www.epa.gov/enviro/html/codes/mn.html|archive-date= 2004-09-30|dead-url= no}}</ref>。
