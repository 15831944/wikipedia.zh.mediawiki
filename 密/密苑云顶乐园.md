'''密苑云顶乐园'''，位于[[河北省|河北省]][[张家口市|张家口市]][[崇礼区|崇礼区]][[四台嘴乡|四台嘴乡]]太子城村梧桐大道，是以滑雪为主题的度假中心<ref name=zjkxx>{{Cite web |url =  http://www.zjknews.com/lvyou/zjkjingdian/201505/21/107525.html#n=1|title =  密苑云顶乐园|author =  |date = 2015-05-21 |publisher = 张家口新闻网 |language =  |accessdate =  |quote =  }}</ref>。

==简介==
密苑云顶乐园位于[[太行山|太行山]]、[[燕山|燕山]]交会处的[[大马群山|大马群山]]中，年平均气温3.3℃，积雪时间150天<ref name=zjkxx/>。

2011年，密苑云顶乐园试营业。2012年12月16日，密苑云顶乐园正式开业。密苑云顶乐园是由[[马来西亚|马来西亚]]卓越集团开发，马来西亚云顶集团投资运营，是密苑度假区的第一期业态，包括滑雪中心、云顶大酒店两个项目。这是云顶集团在[[中国内地|中国内地]]参与开发的第一个项目<ref>{{Cite web |url =http://sports.sina.com.cn/outdoor/2012-12-20/09443409.shtml  |title =崇礼密苑云顶乐园开业：国际高端冰雪体验  |author =  |date = 2012-12-20 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。

2012年起，密苑云顶乐园为国际雪联（FIS）认证的多项赛事提供场地<ref name=feixue>{{Cite web |url = http://sports.sina.com.cn/others/winter/2015-12-02/doc-ifxmifzh4307367.shtml?from=wap |title = 密苑云顶乐园首次披露冬奥战略 成沸雪官方训练营 |author =  |date =2015-12-02  |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。2014年，密苑云顶乐园签约成为2022年冬奥会申办委员会的赞助企业<ref>{{Cite web |url = http://news.163.com/14/0829/11/A4QGGHIR00014SEH_all.html |title = 密苑云顶乐园成为2022年北京冬奥申委赞助企业（全文） |author =  |date =2014-08-29  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。2015年中国成功申办[[2022年冬季奥林匹克运动会|2022年冬季奥林匹克运动会]]，密苑云顶乐园被定为张家口赛区主赛场，作为云顶滑雪公园场地A和B，承担[[自由式滑雪|自由式滑雪]]、[[单板滑雪|单板滑雪]]比赛<ref>{{Cite web |url = http://news.xinhuanet.com/sports/2015-03/22/c_127609030_2.htm |title =  22日至29日 京张申冬奥迎“考官”|author =  |date =2015-03-22  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。密苑云顶乐园将完善2022年冬奥会主赛场的设计方案并由[[国际奥委会|国际奥委会]]批准，其中包括U型场地、空中技巧中心、单板和自由式滑雪赛道等，计划于2016/2017年冬季完成施工<ref name=feixue/>。

2016年12月17日，U型池在密苑云顶乐园开幕，这是中国首个奥运规格的U型池，将作为国家单板和双板U型池队训练基地<ref>{{Cite web |url = http://sports.sina.com.cn/others/winter/2016-12-18/doc-ifxytqax6523497.shtml |title =密苑云顶U型池场地正式投入使用 点燃冬奥激情  |author =  |date =2016-12-18  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。2016年，还建成了奥运会比赛规格的空中技巧场地，以及滑雪山地教学公园等场地和设施<ref>{{Cite web |url = http://www.chinadaily.com.cn/micro-reading/2016-11/21/content_27445637.htm |title = 密苑云顶乐园2016-17雪季开板 首滑派对精彩纷呈 |author =  |date = 2016-11-21 |publisher = 中国日报网 |language =  |accessdate =  |quote =  }}</ref>。

截至2015年，密苑云顶乐园建成35条雪道、3条缆车，规划开发88条雪道、22条缆车<ref name=feixue/>。

2015年1月18日，密苑云顶乐园举办了“1·18国际儿童滑雪比赛”<ref>{{Cite web |url = http://news.sina.com.cn/o/2015-01-19/024031414847.shtml |title =张家口冬奥赛场赛后拟开放使用  |author =  |date = 2015-01-19 |publisher =  新浪|language =  |accessdate =  |quote =  }}</ref>。2015年12月1日，密苑云顶乐园宣布与沸雪（Air & Style）达成战略合作伙伴关系，成为“沸雪官方训练营”，同时LAAX风格的“青年公寓”项目将在2017年落户密苑云顶乐园<ref name=feixue/>。2017年1月19日，密苑云顶乐园8D雪道冠名“斯阔谷”（Squaw Valley）。2017年5月21日，[[1960年冬季奥林匹克运动会|1960年冬季奥林匹克运动会]]赛事举办地——[[美国|美国]][[加利福尼亚州|加利福尼亚州]]斯阔谷滑雪场一条雪上技巧赛道冠名“密苑雪道”<ref>{{Cite web |url = http://sports.163.com/17/0525/12/CL9GJDM700050HE7.html |title =  密苑云顶乐园冠名美国斯阔谷滑雪场赛道|author =  |date = 2017-05-25 |publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。

密苑云顶乐园内的云顶大酒店设有客房、游泳馆、室内健身房、SPA等设施，主要为游客提供服务<ref>{{Cite web |url = http://money.163.com/13/1128/17/9EPJIB3O00254TI5.html |title =密苑云顶乐园雪季正式开始 30条高质量雪道已建成  |author =  |date =  2013-11-28|publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。
