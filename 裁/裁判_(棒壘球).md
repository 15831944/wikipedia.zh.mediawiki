在[[棒球|棒球]]及[[壘球|壘球]]，'''裁判''' ('''{{lang|en|umpire}}''') 是負責主持一場比賽的人士，職責包括開始及結束比賽、執行比賽及場地規則、對雙方攻防動作做出判決、及依規則執行處罰<ref name="Official rules">{{cite web |title=Official Rules: 9.00 The Umpire |url=http://mlb.mlb.com/mlb/official_info/official_rules/umpire_9.jsp |publisher=[[Major_League_Baseball|MLB]] |accessdate=2007-05-05}}</ref>。在美國裁判常被以縮短的 '''ump''' 稱呼，又因裁判制服多為藍色系而常被稱為 '''blue'''，但在職業比賽球員及教練則多直呼裁判名字。雖然早期比賽多由一位裁判主持比賽，但20世紀後多由多位裁判所組成的裁判組來負責主持比賽。

==責任和位置==

外野裁判通常只在重要賽事使用，如[[美國職棒大聯盟明星賽|美國職棒大聯盟明星賽]]、及根據層級不同全部或部分的季後賽：[[美國職棒大聯盟|美國職棒大聯盟]]在所有的季後賽均派出六人裁判組，但在較低層級，如大學級[[NCAA|NCAA]]賽事則只在冠軍賽使用六名裁判<ref name="Special Event selection">{{cite web |title=Special Event selection |url=http://mlb.mlb.com/mlb/official_info/umpires/feature.jsp?feature=event_selection |publisher=[[Major_League_Baseball|MLB]] |accessdate=2007-05-05}}</ref>。通常由距離打擊出去球最近的裁判作出與其相關的判決。

===判決===
裁判所做的判決是最後的決定，除非在與場中其他裁判開會討論後決定改判<ref name="Official rules" />。2014年起，[[美國大聯盟|美國大聯盟]]開始容許總教練經由錄影重播挑戰判決，假如挑戰成功可以再度挑戰<ref>{{cite web|title=Replay Review Regulations|url=http://mlb.mlb.com/mlb/official_info/official_rules/replay_review.jsp|work=MLB|publisher=MLB|accessdate=20 April 2014}}</ref>。假如一位裁判可能引用規則錯誤而誤判，在某些聯盟，如大聯盟，可以針對這個判決抗議<ref name="Official rules" />。假如裁判堅持判決無誤，聯盟高層人員可做出最後裁決。
