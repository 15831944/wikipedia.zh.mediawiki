'''匪我思存 '''（{{bd|1978年|12月26日|catIdx=A}}），本名'''艾晶晶'''，女，[[湖北|湖北]][[武汉|武汉]]人，曾用笔名「'''思存'''」、「'''费小存'''」「'''离馅三千里'''」。中国当代[[愛情小說|言情小说]][[作家|作家]]，湖北省作家协会会员<ref>{{Cite web|url = http://www.ce.cn/culture/gd/201501/01/t20150101_4245719.shtml|title = 网络作家匪我思存加入湖北省作协|accessdate = 2015年1月1日|author = |date = |publisher = }}</ref>，2018年当选湖北省作家协会副主席<ref>{{Cite web|url=http://hb.people.com.cn/n2/2018/0722/c194063-31843364.html|title=湖北省文联、省作协新一届主席团选举产生--湖北频道--人民网|accessdate=2018-09-02|last=L_104249|work=hb.people.com.cn}}</ref>。在已经出版的23部作品中有多部改编成电视剧或电影。其作品深受《[[红楼梦|红楼梦]]》、[[张爱玲|张爱玲]]等的影响，被誉为“悲剧女王”和“电视剧女王”。<ref>{{cite web |url=http://news.163.com/10/0831/03/6FCRBBHI00014AED.html |title=匪我思存《来不及说我爱你》小说走红拍电视 |accessdate=2010-08-31 |publisher=网易 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://hb.sina.com.cn/news/b/2012-08-10/14279467.html |title=武汉才女作家匪我思存：微博常给人意外感动 |accessdate=2012年8月10日 |publisher=新浪网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://news.cnhubei.com/hbrb/hbrbsglk/hbrb09/201205/t2085184.shtml |title=图文：言情女王匪我思存 |accessdate=2012-05-30  |publisher= 荆楚网|work= |language =zh-cn}}</ref><ref>{{cite web |url=http://sports.163.com/12/0115/23/7NRLMNRT0005227R.html |title=匪我思存：写作的低门槛 其实是对文学创作的鼓励 |accessdate=2012-01-15 |publisher=网易 |work= |language =zh-cn}}</ref>

== 生平 ==
匪我思存为畅销书作家，为人低调，很少接受采访，她曾接受采访时表示：“No care about fame and wealth, I like being absorbed in literature just like a reader.”（不关心名气和财富，我只是喜欢沉浸在文学中就如同一个读者。）
