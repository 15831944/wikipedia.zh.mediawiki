{{死刑}}
'''斬首'''，亦稱'''砍頭'''、'''殺頭'''或'''開刀問斬'''，是[[死刑|死刑]]的一種，即將人的[[頭部|頭部]]砍去。

== 簡史 ==
斬首的歷史相當長，古代多國均有斬首的刑罰，與[[絞刑|絞刑]]一類有相當長歷史的刑罰一樣。

常用的斬首[[工具|工具]]有：[[斧|斧]]、[[劍|劍]]、[[刀|刀]]。在[[法国大革命|法國大革命]]時，當時的人發明了專門用來斬首的[[机器|機器]]：[[斷頭台|斷頭台]]。

在古代的[[中國|中國]]、[[英國|英國]]、[[法國|法國]]等多個國家，斬首均會公開進行，民眾會圍觀[[劊子手|劊子手]]如何執行有涉[[刑罰|刑罰]]。由於中國傳統上會在[[午時|午時]]三刻問斬，並將首級懸掛在城池的[[午門|午門]]上，又稱「午門問斬」。

古代中國的斬首，劊子手必須接受過嚴格訓練，原因有二，首先斬首之時必須斬在骨節之間，另有一說指劊子手必須完全砍斷而不可讓首級與身體的皮尚有連接。

== 現狀 ==
{{rellink|extraclasses=boilerplate seealso noprint|参见：[[伊斯兰的斩首|伊斯兰的斩首]]及{{link-en|伊斯兰国斩首事件列表|ISIL beheading incidents}}}}
斬首的刑罰現在已經很少使用，原因是其場面過於血腥，且難於清理，亦有不少人認為違反[[人權|人權]]。

[[中國歷史|中國歷史]]上，[[清|清]]末的[[北洋軍|北洋軍人]]受到[[西化|西化]]影響，就流行以[[槍決|槍決]]代替斬首，[[民國|民國]]建立後，1912年[[法律|法律]]廢除斬首。

[[法國|法國]]從[[法國大革命|法國大革命]]起，直到1981年廢除死刑前，死刑都是以[[斷頭臺|斷頭臺]]執行，除非是犯下危害國家罪行的罪犯，才會被槍決。法國最後一個死囚在1977年用斷頭臺斬首。

現時只有[[沙特阿拉伯|沙特阿拉伯]]以斬首作為處決囚犯的其中一種合法方式。雖然合法，但並不常用。

到了21世紀，斬首被部份人士，如[[恐怖分子|恐怖分子]]等當作私下處決[[人質|人質]]或[[俘虜|俘虜]]的方式，甚至成為[[恐怖攻擊|恐怖攻擊]]的手段。而且恐怖份子喜以[[小刀|小刀]]或[[匕首|匕首]]，慢慢將人[[頭顱|頭顱]]鋸下，使得被害人受盡痛苦而死，毫無人道，嚴格而言應為「割首」。

2004年便有多名外國人，多數是一些[[伊拉克戰爭|伊拉克戰爭]]參戰國家的[[平民|非武裝人員]]，在[[伊拉克|伊拉克]]被[[恐怖分子|恐怖分子]]擄為人質，並在退兵要求得不到該人質之國家政府允諾後，在[[錄影機|錄影機]]前以割頭處決。

2014年起，[[極端組織|極端組織]][[伊斯蘭國|伊斯蘭國]]連續以割頭方式處決了多名外籍人士，以恫嚇[[美國|美國]]等國家停止對於伊斯蘭國的[[空襲|空襲]]行動。

在[[2015年11月巴黎襲擊事件|2015年11月巴黎襲擊事件]]發生後，極右派政治家[[讓-馬里·勒龐|讓-馬里·勒龐]]要求重新啟用斷頭台用來斬首[[恐怖份子|恐怖份子]]。<ref>{{Cite web |url= http://www.thedailybeast.com/articles/2015/11/25/monsieur-le-pen-wants-to-guillotine-terrorists-in-france.html|title= Monsieur Le Pen Wants to Guillotine Terrorists in France|accessdate=2016-07-02 |work= |publisher= The Daily Beast|date= 2015-11-25}}</ref><ref>{{Cite web |url= http://www.express.co.uk/news/world/621098/Reinstate-French-death-penalty-after-Paris-attacks-says-former-NF-leader-Jean-Marie-Le-Pen|title= Far-right French politician wants to bring back death penalty and BEHEAD terrorists|accessdate=2016-07-02 |work= |publisher= Daily Express|date= 2015-11-21}}</ref>

== 用語 ==
[[File:Beheading_(the_Qing_Dynasty,_China).jpg|thumb]]
=== 棄市 ===
一般說法是「杀之于市，与国人共弃之。」即在[[市場|市場]]上公開處死。另說則是死後，需在市場陳屍數日，不許殮葬，以儆效尤。

=== 枭首 ===
[[File:LeonardGSiffleet.jpg|thumb]]游擊的[[澳大利亚|澳大利亚]]战俘{{Link-en|隆納· 西弗利特|Leonard Siffleet}}被[[日军|日军]]捕獲，並斩首。]]
在[[中國|中國]]、[[日本|日本]]、[[朝鮮半島|朝鮮半島]]等東方國家，犯人的[[首級|首級]]有時會被插在高竿上，懸於公眾地方展示，以儆效尤，遠望人頭像是[[夜梟|夜梟]]站立於樹端，稱為'''枭首'''。

在中國，梟首於[[秦朝|秦朝]]、[[漢朝|漢朝]]時用諸夷族。[[六朝|六朝]]時見於律，[[隋朝|隋朝]]時廢。[[明朝|明朝]]起復見於律，初時用於殺父母者，後用於[[強盗|強盗]]。清襲明制，至[[沈家本|沈家本]]修律時廢。

其他地方亦有類似枭首的行動，但通常只發生在戰場上；日本的[[梟首|梟首]]刑是把人犯綁在[[十字架|十字架]]，以[[矛|矛]]自[[脖|脖]]內端割下頭，略異於斬首。日本的梟首有別於中國懸於高處，而是將斬罪之人仕置以刀斬首後掛在獄門臺上，詳見徳川幕府刑事圖譜。

=== 函首 ===
用匣子裝首級，並且遞送出去之意。中國邊軍通常在殺死敵軍重要將領後，將該人[[首級|首級]]取下，簡易防腐後，郵遞回到朝廷，供朝廷驗証。

=== 斬立決、斬監候、陪斬 ===

在[[中國|中國]]，自[[明朝|明朝]]起，斬首的判決必需經過[[刑部|刑部]]的覆核。[[斬立決|斬立決]]者刑部批准即時處斬，[[斬監候|斬監候]]就會到[[三法司|三法司]]再行審議，然而並非所有情況都經刑部，例如[[王陽明|王陽明]]平定[[寧王之亂|寧王之亂]]時，明廷賜其「便宜行事」，可直接處決大逆不道之人。此外因明朝[[大臣|大臣]]與[[士大夫|士大夫]]視[[尚方寶劍|尚方寶劍]]為一種榮耀，朝廷也樂於賜與，發放的尚方寶劍之多，甚至連朝廷也沒有統計數量，但很少有人以尚方寶劍名義直接[[處決|處決]]他人，最著名例子為崇禎年間[[袁崇煥|袁崇煥]]假尚方寶劍，斬殺與自己不睦的[[東江|東江]][[總兵|總兵]][[毛文龍|毛文龍]]，但其實文龍自己也有一把尚方寶劍。而[[崇禎帝|崇禎帝]]對此事耿耿於懷，最後也成為袁崇煥被[[凌遲|凌遲]]的理由之一。

[[清朝|清朝]]時，「大逆」、「大盗」一類重犯，若被判[[斬立決|斬立決]]，需待刑部覆批，然後即著正法。其他一般獲死罪的犯人為[[斬監候|斬監候]]，一律暫緩執行。需待次年[[秋天|秋天]]，將所有文案交由三司秋審或朝審。待斷定無錯案後，名單交予[[皇帝|皇帝]]勾決。於[[霜降|霜降]]至[[冬至|冬至]]期間行刑。 [[行刑|行刑]]時，[[斬監候|斬監候]]囚犯一起被押赴刑場，[[監斬官|監斬官]]從犯人中挑出被勾決[[犯人|犯人]]處決。赴刑場而未被勾決者稱為[[陪斬|陪斬]]。陪斬後或緩勾（繼續監禁，等待下次秋決），或獲[[減罪|減罪]]。但一般多以王命旗牌直接處決，少有機會上告京城的，形成「法律軍事化」的現象，原還要請示皇帝，到乾隆朝以後「恭請王命」多半是「先斬後奏」已經執行，而不「拘泥請旨」。但有清一代真正有辦法上到三法司的案件，跟明代相同大部分都以[[流刑|流刑]]代替。

=== 割頭 ===
[[英語|英語]]中，所有切除對方頭部的行為均稱beheading或decapitation，但[[华文|华文]]的「砍頭」或「斬首」含有以利器'''快速'''切斷死者[[頸|頸]]部的意思。近年恐怖分子所採用的處決很多時都是'''割斷'''而非'''切斷'''受害人的頸部，用意在於增加受害者的痛苦及對觀看者造成的恐怖感，如果根據一些中國古代[[小說|小說]]（如[[水滸傳|水滸傳]]）的用語，恐怖份子處決人質的方式應稱為'''割頭'''。
