{{Engname|《[[爸爸去哪儿|爸爸去哪儿]]》第六季|''Where Are We Going, Dad? 6''}}，是[[中国大陆|中国大陆]][[芒果TV|芒果TV]]于2018年第三季度推出的一档亲子互动真人秀节目。节目版权引进自[[大韩民国|韩国]][[文化广播_(韩国)|MBC电视台]]，概念参考自MBC节目《[[爸爸_你去哪儿|爸爸 你去哪儿]]》，是该系列节目的第六季，由制作人周山及其团队制作。节目于2018年7月开始录制<ref>{{Cite news|url=http://www.chinatimes.com/realtimenews/20180710002896-260404|title=《爸爸去哪兒6》開錄 捕獲陳小春Jasper驚喜現身|publisher=中时电子报|author1=蔡晓婷|date=2018-07-10|accessdate=2018-08-15}}</ref>，计划于近期上线播出<!--{{efn|每週四12:00会员抢先看，每週五12:00免费观看。}}-->，首播平台为芒果TV与[[爱奇艺|爱奇艺]]<ref name="苹果日报">{{Cite news|url=https://tw.appledaily.com/new/realtime/20180809/1407824/|title=《爸爸6》Jasper回歸！網友敲碗...曝光播出時間是...|publisher=苹果日报|author1=陈嘉婉|date=2018-08-09|accessdate=2018-08-15}}</ref>。海外方面，[[马来西亚|马来西亚]]的[[Astro全佳HD|Astro全佳HD]]计划于近期同步跟播。

== 制作 ==
第六季的制作人仍然为周山及其团队，并由卫视版[[爸爸去哪儿_(第三季)|《爸爸去哪儿》第三季]]的制作人单丹霞担任制作顾问。本季节目中，曾参加过[[爸爸去哪儿_(第五季)|第五季]]的[[陈小春|陈小春]]父子再度回归参与录制<ref name="苹果日报" />，同时也是《[[爸爸去哪儿|爸爸去哪儿]]》开播六季以来的首组返场嘉宾。而在节目内容和嘉宾组成上，本季较第五季也作出了一定的调整，参加节目的固定爸爸和孩子由五组恢复为[[爸爸去哪儿_(第四季)|第四季]]时的六组，组成形式则由原来的「3+3」调整为「4+2」，即四位父亲携带自己的亲生孩子，外加两位实习父亲分别携带两位素人萌娃的形式，而实习父亲的头衔在本季则被更名为「实习哥哥」。

本季原参与录制的两位实习哥哥分别为《[[偶像练习生|偶像练习生]]》选手[[范丞丞|范丞丞]]和[[毕雯珺|毕雯珺]]<ref>{{Cite news|url=https://tw.appledaily.com/new/realtime/20180710/1388829/|title=又有Jasper可以看了！《爸爸6》人氣萌娃還有她|publisher=苹果日报|author1=郭妤|date=2018-07-10|accessdate=2018-08-15}}</ref>，素人萌娃为双胞胎兄妹李昀轩和李昀桐。但由于[[国家广播电视总局|广电总局]]对网络选秀节目出身的艺人参与上星综艺节目及亲子类综艺节目均作出了严格的限制，故自第二站起由[[何猷君|何猷君]]取代范丞丞和毕雯珺实习哥哥的位置<ref>{{Cite web|url=http://ent.sina.com.cn/tv/zy/2018-07-23/doc-ihftenhz0845944.shtml|title=范丞丞毕雯珺下车《爸爸6》？节目组暂未正面回应|website=新浪娱乐|date=2018-07-23|accessdate=2018-08-15}}</ref><ref>{{Cite web|url=https://ent.sina.cn/2018-08-15/detail-ihhtfwqr7881918.d.html|title=何猷君回应上《爸爸6》：上节目是因为妹妹|website=新浪娱乐|date=2018-08-15|accessdate=2018-08-15}}</ref>，并需要同时照顾李昀轩和李昀桐兄妹二人，从而在事实上恢复了第五季的「4+1」模式。

== 宣传 ==
在2018年8月4日举行的「青春芒果夜」主题晚会上，[[杨迪|杨迪]]和[[林更新|林更新]]作为种草代表向观众种草推荐了好暖心综艺——《爸爸去哪儿》第六季<ref>{{Cite web|url=http://ent.163.com/18/0805/05/DOE35SJD00038FO9.html|title=朱一龙首次开嗓惊艳 凌潇肃再现《回家的诱惑》主题曲|website=网易娱乐|date=2018-08-05|accessdate=2018-08-15}}</ref>。随后，本季嘉宾代表[[包贝尔|包贝尔]]、包可艾父女与[[爸爸去哪儿_(第五季)|第五季]]的萌娃代表小山竹共同上台与观众互动，标志着本季节目正式官宣。

本季节目原定于2018年8月16日起每週四12:00上线播出<ref>{{Cite web|url=http://ent.ynet.com/2018/08/15/1366395t1254.html|title=《爸爸6》定档8月16日 集体宣传片催泪上线|website=北青网娱乐|author1=刘文思|date=2018-08-15|accessdate=2018-08-15}}</ref>。然而在正式上线22分钟前，节目组通过官方微博宣布本季节目将延期上线<ref>{{Cite web|url=http://ent.sina.com.cn/tv/zy/2018-08-16/doc-ihhvciiv7065374.shtml|title=《爸爸去哪儿》节目组发声明：第六季将延期播出|website=新浪娱乐|date=2018-08-16|accessdate=2018-08-16}}</ref>。

== 主题曲 ==
本季节目的主题曲依旧由参加节目的五位爸爸和小孩共同演唱，歌曲音频于2018年8月15日全网同步上线<ref>{{Cite web|url=https://y.qq.com/n/yqq/song/004KFEw81z0ysI.html|title=爸爸去哪儿 (《爸爸去哪儿第六季》主题曲) - 陈小春/Jasper/杨烁/杨雨辰/包贝尔/包可艾/耿乐/乐那多/何猷君/李昀轩/李昀桐|website=QQ音乐|date=2018-08-15|accessdate=2018-08-15}}</ref>。与往季节目不同的是，本季节目的主题曲为一首全新作品，由唐恬作词，叶圣涛作曲并担任制作人，孔凡东编曲，但依旧以《爸爸去哪儿》为题。而本季主题曲与往季主题曲的联系则在歌词部分，前奏中的四句歌词分别为第一至四季主题曲中的四句经典歌词，第一、三、五节副歌的第一句和第三句沿用第一至四季主题曲中的「老爸，老爸，我们去哪里呀」与「宝贝，宝贝，我是你的大树」。
