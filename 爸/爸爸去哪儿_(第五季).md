{{Engname|《[[爸爸去哪儿|爸爸去哪儿]]》第五季|''Where Are We Going, Dad? 5''}}，是[[中国大陆|中国大陆]][[芒果TV|芒果TV]]于2017年第四季度推出的一档亲子互动真人秀节目。节目版权引进自[[大韩民国|韩国]][[文化广播_(韩国)|MBC电视台]]，概念参考自MBC节目《[[爸爸_你去哪儿|爸爸 你去哪儿]]》，是该系列节目的第五季，由制作人周山及其团队制作。节目于2017年8月开始录制，2017年9月7日12:00播出“先导篇”，2017年9月14日起每週四12:00上线播出{{efn|每週四12:00会员抢先看，每週五12:00免费观看。}}，首播平台为芒果TV与[[优酷网|优酷网]]。

本季节目的台湾播映权由[[中天电视|中天电视]]购得，[[中天综合台|中天综合台]]于2018年1月14日21:30播出「先导篇」，2018年1月21日起每週日20:00播出；香港播映权由[[無綫電視|無綫電視]]购得，通过[[myTV_SUPER|myTV SUPER]]上架点播，并于2018年3月31日起每週六在[[综艺台|综艺台]]播出，2018年7月22日起每週日18:30在[[J2|J2]]播出{{efn|综艺台首播版本为普通话原声播出，J2首播版本设有粤语配音。}}。海外方面，[[马来西亚|马来西亚]]的[[Astro全佳HD|Astro全佳HD]]于2017年9月9日起每週六21:00同步跟播，[[新加坡|新加坡]]的[[HUB都会台|HUB都会台]]亦于2017年11月16日起每週四20:00播出。

== 制作 ==
第五季的制作人仍然为周山及其团队，并由卫视版[[爸爸去哪儿_(第三季)|《爸爸去哪儿》第三季]]的制作人单丹霞担任制作顾问。与第四季相比，本季在节目内容上作出了一定的调整，参加节目的固定爸爸和孩子由六组调整为五组，组成形式也由原来的“3+3”调整为“4+1”，即四位父亲携带自己的亲生孩子，外加一位实习父亲携带素人萌娃的形式。

== 宣传 ==
本季节目开播前夕，恰逢[[湖南广播电视台|湖南广播电视台]]向[[中国共产党湖南省委员会|中共湖南省委]]公布2017年巡视整改情况。整改情况通报中称，[[湖南卫视|湖南卫视]]《爸爸去哪儿》为吸引观众眼球，存在过度消费明星子女之嫌，不利于青少年健康成长。湖南广播电视台按照[[国家新闻出版广电总局|广电总局]]要求，取消了湖南卫视《爸爸去哪儿》节目2017年播出计划<ref>{{cite web|title=中共湖南广播电视台委员会关于巡视整改情况的通报|url=http://www.sxfj.gov.cn/news/10972138.html|publisher=三湘风纪网（[[中国共产党湖南省纪律检查委员会|中国共产党湖南省纪律检查委员会]]、[[湖南省监察厅|湖南省监察厅]]）|date=2017-08-30|accessdate=2017-09-02}}</ref>。然而这一消息却被微博娱乐营销号及电子媒体错误解读为本季节目即将停播<ref>{{Cite web|url=http://news.ifeng.com/a/20170906/51895542_0.shtml|title=《爸爸去哪儿》停播门的背后 是亲子类综艺的裹足不前|website=凤凰网|date=2017-09-06|accessdate=2017-09-11}}</ref>。事实上，通报中所指的节目为卫视版《爸爸去哪儿》，而《爸爸去哪儿》自2016年3月广电总局出台“限童令”之后便已转型为芒果TV网综，所谓的“停播”仅仅只是不再在湖南卫视上星播出而已，并不会影响到本季节目的播出计划。在此期间，本季《爸爸去哪儿》第二站的节目也在正常录制当中。然而自“停播门”事件发生后，节目组及芒果TV方面并未对此作出任何回应，《爸爸去哪儿》节目官方微博也暂停更新，直到开播前一天才恢复。

== 主题曲 ==
本季节目的主题曲依旧由参加节目的五位爸爸和小孩共同演唱，歌曲音频于2017年9月8日全网同步上线<ref>{{Cite web|url=https://y.qq.com/n/yqq/song/0000LqOR4B94zn.html|title=爸爸去哪儿（《爸爸去哪儿第五季》主题曲） - 华语群星|website=QQ音乐|date=2017-09-08|accessdate=2017-09-11}}</ref>。与往季节目不同的是，在本季节目中，节目组除了对主题曲进行了重新编曲之外，歌词部分较以往也有了较大改动。除每节主歌的前两句，每节副歌的第一句“老爸，老爸，我们去哪里呀”与第五、六节副歌为原版歌词外，其他部分皆进行了重新填词。
