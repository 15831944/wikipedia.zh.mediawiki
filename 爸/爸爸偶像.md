《'''爸爸偶像'''》（{{lang-ja|パパドル!}}）是自2012年4月19日在[[TBS电视台|TBS电视台]]播出的[[TBS週四9時連續劇|木曜9剧集]]。主演为[[锦户亮|锦户亮]]。

==概要==
* 本作為[[錦戶亮|錦戶亮]]初次主演黃金時段的連續劇，也是繼《[[雙頭犬_(電視劇)|雙頭犬]]》（{{lang|ja|オルトロスの犬}}）（2009年7月）以來再度出演TBS的戲劇。共演的[[優香|優香]]則是在《20歲的結婚》（2000年7月）之後睽違12年主演TBS的連續劇。
* 制作發表時暫定的名稱為「{{lang|ja|パパはアイドル！}}」，是以[[中山美穗|中山美穗]]1987年主演的戲劇《[[媽媽是偶像|媽媽是偶像]]》（{{lang|ja|ママはアイドル!}}）為原案，更換性別設定後重製的作品。
* 劇中故事主要描寫錦戶亮以現實生活中偶像團體[[関ジャニ∞|關西傑尼斯8]]的成員身分與優香所飾演的單親媽媽秘密結婚所引起的騷動。

==人物介紹==
===花村家===
* '''錦戶亮'''（[[關西傑尼斯8|關西傑尼斯8]]）
: 偶像團體[[關西傑尼斯8|關西傑尼斯8]]的成員。十分疲累的時候偶然間在便利商店遇見遙，便喜歡上她。飾演角色為他本人。
* '''花村遥''' - [[優香|優香]]
: 有三個小孩的媽媽。17歲時因懷了芽衣結婚，但在8年前離婚後成了單親媽媽。靠著白天在果菜市場，晚上在便利商店工作養家。
* '''花村芽衣'''  - [[川島海荷|川島海荷]]
: 花村家的長女，就讀都立高中1年級。總是率先做家事，希望母親能得到幸福。雖然贊成母親再婚，但因為覺得「偶像沒有信用」而反對媽媽跟錦戶結婚。
* '''花村悠斗'''  - [[今井悠貴|今井悠貴]]
: 花村家長男，國中2年級生。說話方式像是自言自語般的嘟噥。有[[網路成癮症|網路成癮症]]，對人沒什麼興趣。對於偶像的錦戶抱持無感的態度。每天會在[[Twitter|Twitter]]上發表花村家的狀況。
* '''花村佳奈''' - [[谷花音|谷花音]]
: 次女，小學3年級生。非常喜歡時尚的女孩子。嵐的歌迷，其中最喜歡的是[[櫻井翔|櫻井翔]]。不會考慮對方的心情，想說的話就會直接說出口。有時候會幫芽衣說出忍住不說的話。
* '''花村奈津''' - [[高畑淳子|高畑淳子]]
: 遙的母親。石川遼的球迷。看見女兒能夠找到新戀情很開心。
* '''花村小八'''
: 花村家養的狗。名字的由來是因為「生日是8月8日」，跟「關西傑尼斯8」沒有關係。

===傑尼斯事務所===
* '''脇坂正平''' - {{link-ja|江成和己|えなりかずき}}
: [[關西傑尼斯8|關西傑尼斯8]]的現場經紀人兼司機。27歲入社半年的新人。工作做不來、意外地常常失敗。被說教時不會看現場氣氛絲毫沒在反省。
* '''間宮義春''' - {{link-ja|八嶋智人|八嶋智人}}
: 關西傑尼斯8的團體經紀人。從出道時就一直支持著關西傑尼斯8，夢想有一天能將關西傑尼斯8培育成世界第一的偶像。
* '''朝比奈元子''' - [[財前直見|財前直見]]
: 關西傑尼斯8的主要經紀人。散發出的氣場會震懾對方。給擅自結婚的錦戶的懲罰是「三個月內引退或者離婚」。
* '''關西傑尼斯8'''
: 錦戶所屬團體的成員。飾演角色為團員本人。
* '''岡島充''' - [[城島茂|城島茂]]（[[TOKIO|TOKIO]]）
: 原偶像，「串燒みつる」的老闆。過去以名為「銀河隊GALAXY」的團體隸屬於[[傑尼斯事務所|傑尼斯事務所]]。因女性問題被解雇。

===FIRE RESCUE 119===

* '''早川琢''' - [[鈴木亮平|鈴木亮平]]
