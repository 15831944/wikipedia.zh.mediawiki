'''库尔特·雨果·施耐特'''（{{lang-en|'''Kurt Hugo Schneider'''}}，{{bd|1988年|9月7日}}）美国视频编辑师、音乐家、歌手和作词者，主要在[[YouTube|YouTube]]上发布音乐[[视频|视频]]。他帮助[[山姆·徐|山姆·徐]]制作了音乐视频，包括山姆·徐一个人饰演多个角色的音乐混合视频。一些其他音乐家及歌手欣赏他的音乐混合技术而找他合作。

== 背景 ==
施耐特成长于[[宾夕法尼亚州|宾夕法尼亚州]]费城郊区的{{Link-en|布魯貝爾|Blue Bell, Pennsylvania}}，他曾经住的地方离山姆·徐只隔了一条街，他上了維沙希康高中才认识了他。他继承了父母亲的[[德国|德国]]和[[奥地利|奥地利]]血统。库尔特有一个姐姐。他在大学期间主修[[数学|数学]]，在2010年从[[耶鲁大学|耶鲁大学]]毕业。库尔特也是一个[[国际象棋|国际象棋]]高手，他参加了耶鲁大学的国际象棋队。

== 事业 ==
施耐特和徐的合作从高中就开始了，之后一直延续到他们俩都到了耶鲁。在施耐特入学前的一年，他制作了徐的翻唱MV并且后期制作了混合效果，后来，他还制作了一系列在线音乐（大学音乐片）也同时制作了很多原创歌曲。

他们第一次受到关注是因为制作了翻唱[[迈克尔·杰克逊|迈克尔·杰克逊]]的歌曲串烧MV。这个视频在施耐特的YouTube频道的浏览量超过了2,800万。总共有3.45亿观看者和超过百万的订阅者。这个混合MV是施耐特利用视频编辑，创造了由一个人演唱无伴奏的合唱表演的幻象，就像是徐主唱，有很多个一样的人完成多重唱（其实都是徐一人完成）。他们这个混合MV描述了同一时间好似几个克隆人在合唱。这个视频就像病毒一样传播到一些国家电视台。包括奥普拉脱口秀和艾伦德杰尼勒斯秀。2011年2月3号，施耐特制作了徐翻唱《Hold It Against Me》的MV，宋唱歌和演奏钢琴，施耐特使用敲打乐器。这个视频则上了小甜甜布兰妮的官方主页。

制作MV受到的超高人气鼓舞了他们去其他项目发展。拍摄独立的电影、学校音乐剧，以学校音乐剧为主题的一系列电影，已经在2010年的夏天拍摄完成正在后期制作中。一系列的幕后画面，录音以及旁白都由尼克·尤哈斯（Nick Uhas）制作，已经发布在YouTube了。2010年9月之后，施耐特和宋开始发布一些他们原创的歌曲。截至2019年3月17日，頻道訂閱數超過1192萬。頻道觀看數也達到2,885,812,910人。

== 其他工作 ==
施耐特也和其他一些音乐人合作音乐MV，一些比较受欢迎的MV有和克里斯蒂娜·葛利米（Christina Grimmie）合作翻唱麥莉·希拉（Miley Cyrus）的歌曲集錦，山姆·徐和克里斯蒂娜·葛利米一起翻唱《Just A Dream》（KurtHugoScheider播放次数最多的视频）模仿过暴雪主题的视频。

最近他还模仿了臭名昭著的「星期五」被称为是「无效射线」。此外，他还和[[蒂芬妮·沃德|蒂芬妮·沃德]]（Tiffany Alvord）合作演唱[[泰勒·斯威夫特|泰勒·斯威夫特]]（Taylor Swift）的歌曲集錦。和[[馬提塞|馬提塞]]（Matisse）合作演唱[[蕾哈娜|蕾哈娜]]（Rihanna）的歌曲集錦。他还和[[馬克斯·施耐德|馬克斯·施耐德]]（Max Schneider）合作过多次。馬克斯·施耐德（Max Schneider）也是一位制作包括山姆·徐和其他一些耶鲁学生在内的耶鲁大学音乐入学视频的助理制片人。
