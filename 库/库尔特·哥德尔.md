'''库尔特·弗雷德里希·哥德尔'''（{{lang-de|Kurt Friedrich Gödel}}，{{bd|1906年|4月28日|1978年|1月14日|catIdx=Gödel}}），出生於[[奧匈帝國|奧匈帝國]]的[[數學家|數學家]]、[[邏輯學家|邏輯學家]]和[[哲學家|哲學家]]，[[维也纳学派|维也纳学派]]（维也纳小组）的成员。其最杰出的贡献是[[哥德尔不完备定理|哥德尔不完备定理]]和[[连续统假设|连续统假设]]的相对协调性证明。

==生平==
=== 學術 ===
哥德爾的數學天分極高。他在[[維也納大學|維也納大學]]時本來修讀[[理論物理|理論物理]]和[[基础数学|基础数学]]，後來又在[[漢斯·哈恩|漢斯·哈恩]]的指導下研習[[數理邏輯|數理邏輯]]、[[集合論|集合論]]。二十四歲時，完成[[哥德爾完備性定理|哥德爾完備性定理]]，並獲得同校[[博士|博士]]學位。但1940年代中就將注意力投放在[[哲學|哲學]]上。

大學時，哥德爾曾參加[[石里克|石里克]]小组的聚會。

1930年9月7日，他正式宣布其[[哥德尔不完备定理|哥德尔不完备定理]]，引起當時重要數學家如[[冯·诺伊曼|冯·诺伊曼]]和[[希爾伯特|希爾伯特]]等的重視。後來又鑽研[[連續統假設|連續統假設]]，但未能完全解決該問題。

===逝世===
哥德爾自幼多病，而且從小就患了[[强迫症|强迫症]]（[[疑病症|疑病症]]）。他還患過[[抑鬱症|抑鬱症]]。後來他在[[普林斯頓|普林斯頓]]的醫院[[絕食|絕食]]而死，因為他認為那些食物有毒。

== 性格 ==

哥德爾是個要求嚴格的人。因此，他很多的想法在生前都沒有正式發表甚至記錄，要逝世後從其手稿找出。

哥德爾不喜歡談論自己或受到注目，亦討厭[[旅行|旅行]]。哥德爾曾要求[[王浩_(数学家)|王浩]]在死後才可以發表一篇有關他的[[傳記|傳記]]。他在學術研究之外的東西，都不公開發表意見。

== 社交 ==
哥德爾的妻子 Adele Nimbursky 比哥德爾大六歲。哥德爾21歲兩人認識時，Adele 已婚且在夜總會 Der Nachtfalter 工作。他們的婚姻遭到哥德爾家人反對，但有情人終成眷屬，在1938年9月20日結婚。他們沒有小孩。

他和家人感情不壞，哥德爾去了[[美國|美國]]後還常常跟他們書信，之後接他們到美國。但其家人似乎對他了解不深：讀大學時，哥德爾的兄長研習[[醫學|醫學]]，從其他人口中才知道他在數學方面頗有名氣。

在普林斯頓時，哥德爾和[[愛因斯坦|愛因斯坦]]成了很好的朋友。後人常將他們比較。哥德爾和愛因斯坦都在自己的範疇有極為重大的貢獻，很聰明，有好奇心，直率。但愛因斯坦性格開朗外向，這點和哥德爾大相径庭。愛因斯坦的死對哥德爾的情緒有很大打擊。

== 國籍 ==
雖然他的傳記列出很多國家，他通常被視為[[奧地利|奧地利]]人。他出生在[[奧匈帝國|奧匈帝國]]的布爾諾，在十二歲时成為[[捷克斯洛伐克|捷克斯洛伐克]]公民，在二十三歲时成為奧地利公民。當[[希特勒|希特勒]][[德奧合併|吞併奧地利]]时，哥德尔自動成為[[德國|德國]]人。[[第二次世界大戰|第二次世界大戰]]後，他再次成為奧地利公民，而且取得美國公民權利。

== 其他 ==
哥德爾的大部分手稿使用[[加貝爾斯貝格速記法|加貝爾斯貝格速記法]]寫成。格言：「有些事實被認知為真，但不必然可證。」
