'''库施'''（'''Kush'''）又譯為'''古實'''（'''Cush'''），是古代[[北非|北非]]地区的一个[[文明|文明]]，其地域大致位于今日[[苏丹共和国|苏丹共和国]]。
== 建國前 ==
早在[[古埃及|古埃及]][[埃及第一王朝|第一王朝]]出现之前，库施已经有了成熟的社会形态。前25世纪开始，古埃及人开始沿着[[尼罗河|尼罗河]]向南扩张，遇到了库施，今天我们所能得到的关于库施的信息，多半来自古埃及人的记述。

在[[中王国时期|埃及中王国]]覆灭后，古埃及人的扩张停顿了一个时期，当他们重新开始南侵时，意外地遭到了有组织的抵抗。历史学家们并不清楚这种抵抗是由一个统一的国家还是一个城市联盟进行的，也不清楚这种国家或者联盟的形态是否受到了古埃及文明的影响。不论如何，到[[图特摩斯一世|图特摩斯一世]]时，古埃及征服了库施，将其变成了属地。

== 建國 ==
前11世纪，古埃及发生内乱，库施乘机重新获得独立，建立了自己的政权，中心在[[纳帕塔|纳帕塔]]（Napata）一带。前8世纪中期，[[阿拉拉_(库施国王)|阿拉拉]]（Alara）统一了库施，他被公认为库施王国的创始人。
== 對埃及的征服及統治 ==
公元前750年，库施国王卡什塔（Kashta）將國境沿尼羅河北推至[[阿斯旺|阿斯旺]]一带，繼卡什塔統治库施的[[皮耶|皮耶]]（Painkhy）將整個上埃及地區征服，公元前712年，皮耶的兄弟[[沙巴卡|沙巴卡]]登上库施王位，[[沙巴卡|沙巴卡]]將整個下埃及也征服并將首都從纳帕塔遷至[[孟菲斯_(埃及)|孟菲斯]]，是为埃及历史上[[埃及第二十五王朝|第二十五王朝]]的建立。<br>

公元前674年，库施王[[塔哈尔卡|塔哈爾卡]]在與亚述的交戰中勝利，公元前671年，[[亚述|亚述]]為了報復，大舉入侵埃及，并將库施首都孟菲斯攻克，[[塔哈尔卡|塔哈爾卡]]多次反攻也未能奪回孟菲斯。<br>
公元前664年，塔哈爾卡去世，其子[[坦沃塔玛尼|坦沃塔瑪尼]]繼位。<br>
公元前661年，亚述攻占上埃及的首府[[底比斯_(埃及)|底比斯]]，至此努比亞人的库施王國被逐出埃及。

== 退出埃及後 ==
库施王國退出埃及後，將首都重新遷回[[纳帕塔|纳帕塔]]。
公元前591年，[[埃及第二十六王朝|埃及第二十六王朝]][[法老|法老]][[普萨美提克二世|普萨美提克二世]]洗劫了纳帕塔，标志着库施重新征服埃及的努力归于失败。公元前530年，库施的都城南迁至[[麦罗埃|麦罗埃]]（Meroë）。<br>
在麦罗埃時期，库施利用当地丰富的森林资源，大力发展了炼[[铁|铁]]业，并与[[古希腊|古希腊]]的商人通过[[红海|红海]]开展贸易，他们还发展了自己的文字[[麥羅埃文|麥羅埃文]]，不再使用古埃及的[[圣书体|圣书体]][[象形文字|象形文字]]。库施也將注意力轉向南方，向南方擴張至[[喀土穆|喀土穆]]一带。<br>
公元前3世紀至公元前1世紀是库施王國的強盛時期，在首都麦罗埃建立了很多金字塔，而且當時[[麦罗埃|麦罗埃]]的煉鐡業被後世考古學家稱為「非洲的[[伯明罕|伯明罕]]」，很多學者認為[[麦罗埃|麦罗埃]]的煉鐡技術後來經過草原地區傳入西非及南部非洲。

== 衰落 ==
公元前23年，羅馬帝國入侵库施，將纳帕塔攻克并夷為平地，經過努比亞人2年的抵抗，羅馬與库施簽署和約後退出库施。<br>

直到[[罗马帝国|罗马帝国]][[尼禄|尼禄]]的时代，库施王国仍然是一支强大的力量，尼禄曾经向其派出过一个使团。但是根据考古结果，[[金字塔|金字塔]]的建造早在前4世纪便已停止，新的王家陵墓规模和精美程度大大缩小。有人推测是[[阿克苏姆|阿克苏姆]]的入侵导致了库施的衰落，但是并未得到确认。根据罗马史学家的记载，[[纳巴提|纳巴提]]人在罗马军队撤出[[努比亚|努比亚]]地区后入侵当地，取代了库施等当地政权。<br>

公元2世紀，納巴提人（Nubas）占領了库施北部地區。<br>

公元350年，阿克苏姆攻克麦罗埃并將其摧毀，库施王國滅亡<br>

6世纪时，原麦罗埃控制的区域已经建立了一些小国，这些国家使用[[科普特字母|科普特字母]]，而不是库施文字。
== 考古 ==
考古學家在蘇丹北部發現三十五座小型金字塔，疑是約二千年前庫施帝國的遺迹。該批金字塔的排列異常緊密，最大的只有廿二呎闊；最小的僅三十吋闊，疑是供兒童殯葬。但大部分金字塔損毀嚴重，並曾遭盜竊。
