'''库车县'''（[[维吾尔语|维吾尔语]]：كۇچار ناھىيىسى ），是中国[[新疆维吾尔自治区|新疆维吾尔自治区]][[阿克苏地区|阿克苏地区]]的县。古代[[龟兹|龟兹]]王国之地。称为歌舞之乡。油气资源丰富，[[西部大开发|西部大开发]]的重要基地。总面积1.52万平方公里，总人口49万，其中，城镇人口23万。县城库车镇距[[乌鲁木齐|乌鲁木齐]]市公路里程750千米。

== 地理 ==
位于[[天山|天山]]南麓、[[塔里木盆地|塔里木盆地]]北缘，新疆维吾尔自治区中部。占阿克苏地区东部，西与该地区[[新和|新和]]县、[[沙雅|沙雅]]县相邻，东与[[巴音郭楞蒙古自治州|巴音郭楞蒙古自治州]][[轮台|轮台]]县、[[和静|和静]]县、[[尉犁|尉犁]]县接壤。

地势北高南低，自西北向东南倾斜。地形可分为北部天山山地、冲积扇形砾石戈壁地和南部[[冲积平原|冲积平原]]三部。主要河流有库车河、渭干河、拉依苏河、二八台河等。属温带大陆性气候，冬季干冷，夏季酷热，昼夜温差大，年均气温为11.4℃，年均降水量为64.5毫米，无霜期长达266天，是全国年平均晴天最多的县市。

== 历史 ==
古[[龟兹|龟兹]]王国是[[西域|西域]]三十六国中的大国之一，[[唐|唐]]设[[安西都护府|安西都护府]]在龟兹，领辖龟兹、[[焉耆|焉耆]]、[[于闐|于闐]]、[[疏勒|疏勒]]四镇驻军，是当时[[西域|西域]]的政治、经济、军事、文化和商贸中心。是[[中亞|中亞]][[雅利安人|雅利安人]]文化的極致。这里也是以前新疆佛教文化最著名的地方。[[小乘|小乘]]人口较多，后来被[[秃忽鲁帖木儿|秃忽鲁帖木儿]]的[[和卓|和卓]][[额什丁|额什丁]]攻击，成为[[伊斯兰|伊斯兰]]新据点。

库木吐拉千佛洞是全国重点文物保护单位，其它还有默拉纳额什丁墓、可可沙炼铁遗址、龟兹古城遗址、林基路纪念馆、高山湖泊大龙池等。

== 经济 ==
库车水土光热资源丰沛。特别是塔里木石油天然气开发和[[西气东输|西气东输]]工程的重要基地。

[[天然气|天然气]]储量为2万亿立方米，占塔里木盆地探明储量的90％以上，[[原油|原油]]储量7亿吨，占探明储量的70％以上。

已探明的油气田有雅克拉、塔河、东河塘、牙哈、伊奇克里克、大涝坝、依南、依深、迪那2、亚肯背斜等。

探明的[[煤炭|煤炭]]资源储量达15.6亿吨。[[岩盐|岩盐]]储量达36亿吨。

全县耕地面积达79万亩，可垦荒地约350万亩。

== 交通 ==
库车已成为南疆集航空、铁路、公路相交织的客、货运集散地。

[[库车龟兹机场|库车龟兹机场]]直达[[乌鲁木齐|乌鲁木齐]]、[[阿克苏|阿克苏]]。[[南疆铁路|南疆铁路]]横穿县域，并与石油、煤炭铁路专用线连接。

国道314线贯穿全县5镇1乡1场，国道217线贯通天山南北，终至库车与314线相连。
