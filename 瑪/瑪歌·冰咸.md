'''瑪歌·冰咸'''（'''Margot Bingham'''，{{bd|1987年|12月6日}}）是一名美國女演員和歌手。她最著名角色是在[[HBO|HBO]]電視劇《[[酒私風雲|酒私風雲]]》中飾演爵士樂歌手Daughter Maitland<ref>{{Cite web |url = http://www.rollingstone.com/tv/news/boardwalk-empire-actress-margot-bingham-dishes-on-daughter-maitland-20131115 |title = 'Boardwalk Empire' Actress Margot Bingham Dishes on Daughter Maitland |author = Sarene Leeds |date = 2013-11-15 |publisher = Rolling Stone |language = en}}</ref>。2015年，冰咸開始於[[美國廣播公司|ABC]]電視劇《[[陌生血親|陌生血親]]》中飾演主要角色之一。
