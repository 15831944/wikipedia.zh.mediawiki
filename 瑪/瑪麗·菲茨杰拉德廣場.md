'''瑪麗·菲茨杰拉德廣場'''（{{lang-af|Mary Fitzgerald Square}}，{{lang-en|Mary Fitzgerald Square}}）是位於[[南非|南非]][[約翰內斯堡|約翰內斯堡]]{{le|新城 (約翰內斯堡)|Newtown, Johannesburg|新城}}的一個廣場。其為一個以{{le|瑪麗·菲茨杰拉德 (工會會員)|Mary Fitzgerald (trade unionist)|瑪麗·菲茨杰拉德}}來命名的公共場所，她是南非第一位工會女會員。

此前，該廣場稱為「亞倫的地面」，並於1939年以菲茨杰拉德來命名，因為其為20世紀早期罷工者集會的常用地點<ref>{{cite web
 |url=http://www.joburg.org.za/content/view/1715/188/

 |accessdate=2009-03-24 
}}</ref>。

該廣場設有法國照明工程師{{le|帕特里克·里莫|Patrick Rimoux}}設計的燈光和雕刻木頭<ref>{{cite web
 |url=http://www.jda.org.za/newtown/projects.stm

該廣場是{{le|新城 (約翰內斯堡)|Newtown, Johannesburg|新城}}[[市區更新|市區更新項目]]的核心，其周圍環繞著對城市有重要意義的建築，包括{{le|市場劇院 (約翰內斯堡)|Market Theatre (Johannesburg)|市場劇院}}－該劇院在20世紀80年代上演了眾多反種族隔離戲劇，稱為“鬥爭劇場”、{{le|非洲博物館|MuseuMAfricA}}、舊的渦輪大廳以及工人圖書館。

2005年7月2日，瑪麗·菲茨杰拉德廣場舉辦由[[鮑勃·格爾多夫|鮑勃·格爾多夫]]組織的[[現場八方|現場八方]]系列音樂會約翰內斯堡站。

自2012年以來的每年10月第一個星期五，廣場都會舉辦「In The City」活動<ref name = "InTheCity">http://www.vodacom.co.za/vitc2013/ITC</ref>。
