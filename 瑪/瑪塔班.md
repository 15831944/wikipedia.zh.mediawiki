'''瑪塔班'''（{{lang-so|''Matabaan''}}），或'''瑪塔班加洛雷'''（{{lang|so|Matabaan Garoore}}），是[[索馬利亞|索馬利亞]]的一個城鎮，位於該國中部的[[希蘭州|希蘭州]]。

== 簡介 ==
瑪塔班距離[[希蘭州|希蘭州]]首府[[貝蘭德文|貝蘭德文]]約75公里遠。根據地方史，這個城鎮約在19世紀時建立。當地居民現時約有20,000人。

== 行政區域 ==
瑪塔班下轄以下的村落：
# Xariir yaale 
# Xaashi indho buur
# Neeshinka
# Barakaat ama Hormuud 
# Suuqa xoolaha 
# Baliga
# Filla baadiyo
# ceelka biyaha ama Matoorka 9
# Qabuuraha.
# C/salaan
# Matabaan yare
