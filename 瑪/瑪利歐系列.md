{{Vgsname|瑪利歐系列|ja=マリオ|en=Mario|cn=马里奥|tw=瑪利歐|hk=瑪利奧|神游科技和官方简体中文版本译作“马力欧”}}是日本電子遊戲製造商[[任天堂|任天堂]]開發的[[電子遊戲|電子遊戲]]系列，其中自[[FC游戏机|红白机]]上的經典《[[超级马里奥兄弟|超级马里奥兄弟]]》開創的動作遊戲系列，官方稱之為《[[超級瑪利歐系列|-{zh-cn:超级马力欧系列; zh-tw:超級瑪利歐系列;}-]]》，其他還有賽車、運動、RPG等系列存在。迄今所有系列的合計銷量已超過5億份<ref>http://www.vgchartz.com/article/250637/top-10-in-sales-mario-franchise-500m-games-sold/</ref>，是史上[[电子游戏系列销量排行列表|销量最高的电子游戏系列]]。本系列已成为[[任天堂|任天堂]]的招牌系列。

== 主要人物 ==

所有人物主要在競賽遊戲或外傳作品出場，但正篇遊戲至今大多只能以[[瑪利歐|瑪利歐]]、[[路易_(游戏角色)|路易]]進行遊戲。

; {{nihongo|[[瑪利歐_(角色)|-{zh-cn:马力欧;zh-tw:瑪利歐;}-]]|マリオ|Mario}}
: 初出作品：[[大金剛|大金剛]]（1981年）
: 配音：[[查爾斯·馬爾蒂內|查爾斯·馬爾蒂內]]
: 本系列正篇作品的主人翁，[[義大利裔美國人|義大利裔美國人]]的水管工，現在定居於蘑菇王國。身材微矮胖，性格勇敢無懼，總是義不容辭拯救陷入危機的蘑菇王國與對抗庫巴。

; {{nihongo|[[路易吉|路易吉]]|ルイージ|Luigi}}
: 初出作品：[[瑪利歐兄弟|瑪利歐兄弟]]（1983年）
: 配音：[[Julien_Bardakoff|Julien Bardakoff]]（瑪利歐賽車64  日版、瑪利歐派對、瑪利歐派對2、瑪利歐賽車Advance）、[[查爾斯·馬爾蒂內|查爾斯·馬爾蒂內]]（1997年－至今）
: 瑪利歐的雙胞胎弟弟，從第二玩家（2-Player）衍生來的角色，亦是住在蘑菇王國的水管工。身材較高瘦，但性格較為溫順膽小。

; {{nihongo|[[碧姬公主|碧姬公主]]|ピーチ姫|Princess Peach}}
: 初出作品：[[超級瑪利歐兄弟|超級瑪利歐兄弟]]（1985年）
: 配音：[[春花亜沙子|春花亜沙子]]（瑪利歐賽車64  日版、瑪利歐派對、瑪利歐派對2、瑪利歐賽車Advance）、[[Leslie_Swan|Leslie Swan]]（超級瑪利歐64、瑪利歐賽車64 海外版）、[[Jen_Taylor|Jen Taylor]]（1999–2007）、{{le|萨曼莎·凯利|Samantha Kelly}}（2007年－至今）
: 蘑菇王國的統治者，金髮碧眼的美麗公主，性格謙遜有禮。在正篇作品經常被庫巴覬覦綁架，並被瑪利歐所救，視瑪利歐為英雄，也對瑪利歐有好感。

; {{nihongo|[[奇諾比奧|奇諾比奧]]|キノピオ|Toad 或 Kinopio}}
: 初出作品：超級瑪利歐兄弟（1985年）
: 配音：[[丸野智子|丸野智子]]（瑪利歐賽車64 日版、瑪利歐派對、瑪利歐派對2、瑪利歐派對3、瑪利歐賽車Advance）、[[Isaac_Marshall|Isaac Marshall]]（瑪利歐賽車64 海外版）、[[Jen_Taylor|Jen Taylor]]（1998年－2006年）、{{le|萨曼莎·凯利|Samantha Kelly}}（2007年－至今）

== 历史 ==

=== 瑪利歐形象的誕生 ===

[[瑪利歐|瑪利歐]]在遊戲界的登場可追溯到1980年，瑪利歐與年方27歲的製作人[[宮本茂|宮本茂]]的初試啼聲之作《大金剛》（《Donkey Kong》）原是為店頭大型電玩所設計的遊戲，初始的遊戲系統則修改自任天堂1980年代初紅極一時的掌上型電玩[[GAME&WATCH|GAME&WATCH]]系列中的名作《[[大力水手|大力水手]]》；但由於任天堂與漫畫商恰談《大力水手》[[版權|版權]]失敗，宮本茂不得不臨時修改，改以一個穿著紅色連身工作裝與滑稽大鼻子的人物代替[[卜派|卜派]]的形象。同時，由於當時的[[點陣圖|點陣圖]]技術無法良好的表現頭髮與嘴巴，宮本茂索性再為角色加上了帽子與鬍子。日後成為家用電玩代名詞的瑪利歐，就在此時誕生了。马里奥最早出現在[[大金剛|大金刚]]游戏裡，其名原因是一位[[意大利|意大利]]裔員工表示，電玩人物類似其叔叔马里奥。

宮本茂原本為這個角色定名為"Mr. Videogame"。至於為何改名為瑪利歐，除「義大利裔員工叔叔說」外尚有一說（本說的可信度似乎更高一些）：當首批《大金剛》大型電玩運抵美國時，遊戲機版的倉儲與安裝全都在任天堂美國分公司在[[西雅圖|西雅圖]]租賃的倉庫進行；正當包括任天堂美國分公司總裁[[荒川實|荒川實]]（Minoru Arakawa）在內的所有員工都忙於出貨時，一位不速之客——身材矮胖、破囉嗓的義大利裔美國籍房東[[馬里奧·西加列|馬里奧·西加列]]（Mario Segale）——跑來催繳房租；或許是這位房東的形象恰巧與大金剛遊戲的主角頗有雷同之故，任天堂美國分公司的員工隨即把他的大名「瑪利歐」套到了這款稍後在美國大紅大紫的遊戲主角身上。

== 影響 ==

=== 在流行文化裡 ===

[[香港|香港]][[流行曲|流行曲]][[歌手|歌手]][[古巨基|古巨基]]於2003年11月28日發行的[[广东话|广东话]]大碟《[[遊戲_基|遊戲 基]]》中的《任天堂流淚》是以瑪利歐為主題描寫一痴情男和一多情女愛情的一首流行歌曲。[[陳奕迅|陳奕迅]]在2007年發行的廣東大碟[[Listen_to_Eason_Chan|Listen to Eason Chan]]中 , [[瑪利奧派對|瑪利奧派對]]一曲也表達了玩瑪利歐遊戲時的心情。

=== 世界紀錄 ===

英國一項調查發現，超級瑪利歐是自1980年代電子遊戲普及後，最受歡迎的電子遊戲。<ref>http://hk.news.yahoo.com/article/080728/4/7ent.html 孖寶兄弟膺最受歡迎電玩</ref>
直至2008年，瑪利歐分別在不同的機種登陸在116種[[電子遊戲|電子遊戲]]，當中包括瑪利歐自己為主角的遊戲和其他如[[任天堂明星大亂鬥|任天堂明星大亂鬥]]等遊戲上，成為了[[吉尼斯世界紀錄大全|吉尼斯世界紀錄大全]]中，發行最多款[[電子遊戲|電子遊戲]]的[[電子遊戲角色|電子遊戲角色]]。
