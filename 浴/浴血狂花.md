《'''浴血狂花'''》（英語：Revenge）是一部2017年美國和法國合拍的驚悚動作片。由柯洛里·法吉特執導，由[[瑪蒂爾達·魯茨|瑪蒂爾達·魯茨]]及[[凱文·詹森斯|凱文·詹森斯]]主演。

== 劇情 ==
三名好友進行一年一度的狩獵旅行，理察也帶上自己的情婦珍妮佛同行，不料其中一人色慾薰心，強姦了珍妮佛，讓原本的放縱之旅變調成致命殺機。被凱文推下懸崖而身受重傷的珍妮佛甦醒後決心利用方法報復他們……

== 製作 ==
=== 拍攝 ===
[[主體拍攝|主體拍攝]]於2017年2月在[[摩洛哥|摩洛哥]]進行，並在同年3月完成拍攝。

== 發行 ==
2018年1月21日，{{tsl|en|Neon (distributor)|霓虹娛樂}}取得電影在美國的發行權。

《浴血狂花》定於2018年7月6日在[[香港|香港]]發行DVD<ref>{{cite web|url=https://www.yesasia.com/global/血色攞命花-2017-dvd-香港版/1067388512-0-0-0-zh_TW/info.html|series= |title=浴血狂花香港版發行資料}} </ref> ，香港和美國則分別於2018年5月10日及11日正式上映。

該片的官方預告片於2018年3月31日發布。

== 評價 ==
[[爛番茄|爛番茄]]根據88條評論給予92%的新鮮度（平均評分為8.1分，滿分為10分）。該片在[[Metacritic|Metacritic]]上持有81分（基於23位評論家），顯示「普遍好評」。
