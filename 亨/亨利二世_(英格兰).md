'''亨利二世（短斗篷）'''（Henry II Curmantle，{{bd|1133年|3月5日|1189年|7月6日|catIdx=H}}）是[[英格兰|英格兰]]国王（1154年—1189年在位），他也是法国的[[诺曼底|诺曼底]]公爵（1150年起）、[[安茹|安茹]]伯爵（1151年起）和[[阿基坦|阿基坦]]公爵（1152年起）。他所创立的[[金雀花王朝|金雀花王朝]]是英格兰[[中世纪|中世纪]]最强大的一个封建王朝。从他开始的几位国王也被称作[[安茹|安茹]]国王，因为英格兰只是他们那没有多少实际统一性的巨大领地的一部分。

== 出生 ==

亨利是英格兰国王[[亨利一世_(英格兰)|亨利一世]]之女儿[[马蒂尔达皇后|马蒂尔达皇后]]的儿子，生于[[法国|法国]][[曼恩-卢瓦尔省|曼恩-卢瓦尔省]]的[[勒芒|勒芒]]。他的父亲是[[安茹|安茹]]伯爵[[若弗鲁瓦五世_(安茹)|傑弗瑞五世]]。因此亨利几乎可以說是法国人。

由於[[亨利一世_(英格兰)|亨利一世]]死後並沒有留下子嗣，只留下了兩個女兒。他曾經想讓女兒即位，先是將二女兒[[马蒂尔达皇后|马蒂尔达]]嫁給了德皇[[亨利五世_(神聖羅馬帝國)|亨利五世]]，1114年1月7日两人在[[沃尔姆斯|沃尔姆斯]]举行盛大婚礼成婚，但是不幸的[[亨利五世_(神聖羅馬帝國)|亨利五世]]死於1125年，瑪蒂爾達皇后成為一個沒有子女的23歲寡婦。[[亨利一世_(英格兰)|亨利一世]]本來安排她在英國接任女皇，但是歐洲當時的氛圍並不允許；於是亨利將[[马蒂尔达皇后|马蒂尔达]]改嫁[[安茹|安茹]]伯爵[[若弗鲁瓦五世_(安茹)|傑弗瑞五世]]，並產下了亨利，即將來的亨利二世。

== 早年生涯 ==
[[亨利一世_(英格兰)|亨利一世]]另有一外甥[[斯蒂芬_(英格兰国王)|史蒂芬]]，是[[亨利一世_(英格兰)|亨利一世]]的親妹妹嫁給一法國伯爵生的兒子。由於這位法國伯爵於十字軍東征戰死，所以在1106年便將史蒂芬帶在身邊撫養，受宮廷教育並視為親生兒子，受封大量的領地與爵位。1135年[[亨利一世_(英格兰)|亨利一世]]死後，史蒂芬第一時間赶到[[英格兰王国|英格兰]]宣布继承王位，兼领[[诺曼底公爵|诺曼底公爵]]爵位，並受到教皇的加冕，即是[[諾曼王朝|諾曼王朝]]第四位君主。

於是以亨利一世之女[[马蒂尔达皇后|马蒂尔达]]為代表及以史蒂芬為代表的雙方展開將近十年的王位爭奪，兩方各有輸贏，[[亨利二世|小亨利]]在內戰過程中於法國逐漸成長。

19歲時，[[亨利二世|亨利]]已經繼承了母親[[马蒂尔达皇后|马蒂尔达]]的[[諾曼第|諾曼第]]，從父親[[若弗鲁瓦五世_(安茹)|傑弗瑞五世]]處繼承了[[安茹|安茹]]並成為[[安茹伯爵|安茹伯爵]]，最重要的是，他還继承了父親的瀟灑和倜儻。並與[[阿基坦的埃莉诺|阿基坦的埃莉诺]]（30岁）結婚{{NoteTag|[[阿基坦的埃莉诺|埃莉诺]]當時仍是法國王后；為了改嫁給亨利，她曾多次嘗試令她和法国国王[[路易七世_(法兰西)|路易七世]]的婚姻被宣佈無效。[[路易七世_(法兰西)|路易七世]]最後是因為[[阿基坦的埃莉诺|埃莉诺]]未能生出兒子，而答應取消婚姻的。}}，於是亨利又得到了[[阿基坦的埃莉诺|埃莉诺]]的嫁妝[[阿基坦|阿基坦]]。这使他实际上成为法国最大的领主，也造成了他与路易七世的宿怨。在亨利绝大部分生命裏，他企图以武力或智力战胜路易七世。

== 登基英王 ==

亨利第一次来到英格兰是在1142年，以支持其母对英格兰王位的要求。但玛蒂尔达最终未能从[[斯蒂芬_(英格兰)|布鲁瓦的斯蒂芬]]手中夺得王位。不过根据斯蒂芬与玛蒂尔达的一项协定，亨利于1154年加冕为英格兰国王。但一道受亨利一世的私生子阻止，但其後他們二人私下解決。最終他的大哥（亨利一世的私生子）下落不明，亨利二世最終夺得王位。

亨利二世登基後，即進入[[安茹王朝|安茹王朝]]，由於其父親安茹伯爵[[若弗鲁瓦五世_(安茹)|傑弗瑞五世]]最經常的打扮是在帽沿別上一朵[[金雀花|金雀花]]，所以[[安茹王朝|安茹王朝]]又稱[[金雀花王朝|金雀花王朝]]。

== 司法改革 ==

亨利二世的司法改革中，有兩件事情影響最大，其一是加強王室法庭的權力。當時的法律規定，普通的案件都是在發生當地處理，處理的辦法也是遵循當地領主的規矩；改革後的法律規定，如果不服當地法庭的判決，可以上訴到王室法庭裁決。這樣各封建領主的法律管轄權就逐漸收斂回中央，有利於英國司法權的統一。
其二，亨利二世創立了陪审制度<ref>{{Harvnb|丸山|1990|p=8}}、{{harvnb|U.S. Courts|2008}}。</ref>並廢除了當時荒謬的[[神明裁判|神裁法]]<ref>{{harvnb|ABA|2008|p=1-2}}。</ref>。

這一套司法改革，前者為收斂各領主在司法上的干涉，後者則針對教會對司法權的干涉。亨利將司法權從神權中剝離，並將英國司法推向統一的大道，因而成為西方法律之父。這一套制度即為[[英美法系|英美法系]]被歐美國家沿用至今。

== 與教會的矛盾 ==

在教会的司法权也收回的過程中，也受到了阻礙。
1164年，他与昔日的密友和顾问，[[坎特伯雷大主教|坎特伯雷大主教]][[托马斯·贝克特|托马斯·贝克特]]发生了冲突。[[托马斯·贝克特|贝克特]]请求[[教皇|教皇]]的干预，触怒了亨利二世。1170年12月29日，四个男爵骑士，在[[坎特伯雷大教堂|坎特伯雷大教堂]]刺杀了贝克特。教皇[[亚历山大三世_(教宗)|亚历山大三世]]立刻宣布[[托马斯·贝克特|托马斯·贝克特]]封聖，並威胁要对亨利二世处以[[绝罚|绝罚]]。亨利不得不屈服，他赤身走到了[[坎特伯雷|坎特伯雷]]，讓所有的僧侶鞭打自己。最後亨利走到[[托马斯·贝克特|托马斯·贝克特]]死去的地方，跪下懺悔。并表示要领导[[十字军|十字军]]去[[巴勒斯坦|巴勒斯坦]]。

從這時候起，英國以國王代表的君權及以教會代表的神權，埋下在將來激化矛盾中的種子。

在亨利二世晚年时候，他的几个儿子争权夺地，发生叛乱。亨利二世认定是自己的妻子怂恿儿子们反对他。

== 夫妻失和 ==

王后[[阿基坦的埃莉诺|埃莉诺]]比亨利大了整整十一歲，在[[阿基坦的埃莉诺|埃莉诺]]年近50的時候，發現亨利有一名情婦[[罗莎蒙德·克利福德|罗莎蒙德·克利福德]]（[[:en:Rosamund_Clifford|Rosamund Clifford]]）。亨利為了[[:en:Rosamund_Clifford|罗莎蒙德·克利福德]]在海邊建了[[罗莎蒙德凉亭|罗莎蒙德凉亭]]，並為了隱瞞[[阿基坦的埃莉诺|埃莉诺]]甚至修建了迷宮。王后最終還是發現了[[:en:Rosamund_Clifford|罗莎蒙德·克利福德]]，並逼其自盡。  事發之後，亨利與王后[[阿基坦的埃莉诺|埃莉诺]]分居，並遂秘密请求教皇允许他与埃利诺妻子离婚，未获同意。後來也把[[阿基坦的埃莉诺|埃莉诺]]監禁。這些舉動導致了亨利二世晚年一系列的父子反目的戰爭。

== 父子反目 ==

[[亨利二世|亨利二世]]與王后[[阿基坦的埃莉諾|埃莉諾]]生了4男3女。四個兒子中長子早夭，另外三位分別是[[幼王亨利|小亨利]]、理查和[[若弗鲁瓦二世_(布列塔尼)|傑弗瑞]]。根據當時的世襲法則[[幼王亨利|小亨利]]成為太子，並受封大部分的領土，並包括法國的[[諾曼第|諾曼第]]、[[安茹|安茹]]等地。理查則受封母后[[阿基坦的埃莉諾|埃莉諾]]的土地[[阿基坦|阿基坦]]；[[若弗鲁瓦二世_(布列塔尼)|傑弗瑞]]則受封英王[[亨利二世|亨利]]早年占領的布列塔尼區。

分封既定，[[亨利二世|亨利二世]]與王后[[阿基坦的埃莉諾|埃莉諾]]於1167年又得一子[[約翰_(英格蘭國王)|約翰]]，由於領土已分封結束，[[亨利二世|亨利二世]]要求其他兒子將自己領地中的一個部分分給[[約翰_(英格蘭國王)|約翰]]，導致父子反目。

1172年，因為夫妻失和(亨利二世认定自己的妻子怂恿儿子们反对他)，[[幼王亨利|小亨利]]、理查和[[若弗鲁瓦二世_(布列塔尼)|傑弗瑞]]三兄弟站在母親一邊對父親宣戰。父子之戰長達十餘年，[[幼王亨利|小亨利]]和[[若弗鲁瓦二世_(布列塔尼)|傑弗瑞]]先後病逝，1189年理查則與法蘭西王[[腓力二世|腓力二世]]結盟。戰爭期間雙方曾試圖和解，可惜[[亨利二世|亨利二世]]的談判條件依然是為小[[約翰_(英格蘭國王)|約翰]]爭取權益，甚至提出讓[[約翰_(英格蘭國王)|約翰]]成為新的王儲，導致雙方談判沒有結果。兩位王子病死後，[[亨利二世|亨利二世]]身體狀況下滑，於是要求兩方休兵；就在這個瞬間，[[約翰_(英格蘭國王)|約翰]]倒戈向理查與[[腓力二世|腓力二世]]，於是[[亨利二世|亨利二世]]開始絕食並拒絕醫治，只求速死。兩天後，[[亨利二世|亨利二世]]含恨而死。[[亨利二世|亨利二世]]臨死前，命令畫匠畫了一幅畫，內容是四隻小鷹和一隻老鷹互相撕咬，其中最小的鷹停在老鷹的肩膀上，利嘴正在啄老鷹的眼睛。[[亨利二世|亨利二世]]命令他死後，這一幅畫掛在[[威斯敏斯特宮|威斯敏斯特宮]]的大廳裡。亨利二世在悲痛中去世于法国[[希农|希农]]，葬在法国。其子理查則於同年7月即位，稱理查一世。 

== 創立大學 ==

當時有意求學的學子都到了歐洲的大學求學，尤其是到法國巴黎。英法關係一直以來就十分不合，英王、法王室永世的宿敵，那些在巴黎求學的學子們長期處於不安全感之下，最後一批學者回到了英國。當時亨利二世經常在[[牛津大學|牛津]]一帶出沒，並且在[[牛津大學|牛津]]建立了供娛樂性質的宮殿，於是就將其改制成[[牛津大學|牛津大學]]。後來一幫牛津的學者脫離了牛津，創立了[[劍橋大學|劍橋大學]]，為世界的高等學府樹立了兩大標竿。

== 稅制改革 ==

亨利二世在英格兰进行一系列改革。他是第一个征收[[盾牌钱|盾牌钱]]的国王。他把大部分司法权力集中于国王手中，实际上亨利二世对英国[[普通法|普通法]]（Common law）的形成起到了很大作用。

== 愛爾蘭的糾紛 ==

1155年英格兰籍的教皇[[艾德里安四世|艾德里安四世]]在一份文件中把[[爱尔兰|爱尔兰]]赠给亨利二世。这份文件后来被认为无效。这很可能是亨利的政治阴谋，从此以后爱尔兰的命运就与英格兰紧紧相连。
