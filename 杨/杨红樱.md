'''杨红樱'''（{{bd|1962年|catIdx=Zheng杨|5月12日}}），[[四川|四川]][[成都|成都]]人，[[中国|中国]]大陆[[儿童文学|儿童文学]]作家，[[中国作家协会|中国作家协会]]会员，四川省作协副主席，成都文联副主席<ref>{{Cite web|url = http://www.chinawriter.com.cn/2013/2013-10-14/177322_12.html|title = 著作论述的15位作家的基本情况|accessdate = 2013年10月14日|author = |date = |publisher = }}</ref>。，其作品销量已超过5000万册，并被翻译为[[英文|英文]]、[[韩文|韩文]]、[[泰文|泰文]]、[[德文|德文]]、[[西班牙文|西班牙文]]等语言。杨红樱所著的[[繪本|繪本]]《[[淘气包马小跳|淘气包马小跳]]》系列自2003年出版以来，受到了大陆小学生的喜爱，累计销量超过3000万册<ref>{{Cite web|url = http://gb.cri.cn/27824/2013/02/06/5190s4014947.htm|title = 儿童作家杨红樱：自己的发迹拜在台湾获奖所赐|accessdate = 2015-08-03|author = |date = 2013-02-06|publisher = 国际在线|language = zh-cn}}</ref><ref>{{Cite web|url = http://cul.china.com.cn/renwu/2011-11/09/content_4609943.htm|title = 杨红樱：成长的最佳状态是快乐|accessdate = 2015-08-03|author = |date = 2011-11-09|publisher = 文化中国|language = zh-cn}}</ref>。其作品一直坚持“教育应该把人性关怀放在首位”的理念，在中小学校产生了广泛的影响，被誉为“中国童书皇后”<ref>{{Cite web|url = http://hb.sina.com.cn/wenyu/whzx/2014-12-29/1033212096.html|title = 杨红樱来汉亲述“中国童书皇后”是怎样练成的|accessdate = 2015-08-03|author = |date = 2014年12月29日|publisher = 新浪湖北|language = zh-cn}}</ref>。

== 生平 ==
1962年出生于四川成都，1982年参加工作，历任成都人民北路小学语文教师，成都出版社编辑，成都《青年作家》杂志副编审。

1984年开始发表作品，1998年加入中国作家协会<ref>{{Cite web|url = http://www.dajianet.com/people/view/144228.shtml|title = 杨红樱简介|accessdate = 2015-08-03|author = |date = |publisher = 大佳网|language = zh-cn}}</ref>。

2000年，她出人意料地停止了童话写作，开始运用日记体小说形式，记录下自己女儿告别童年，由小女孩成长为少女的生命图景<ref>{{Cite web|url = http://cd.qq.com/a/20100916/000382.htm|title = 滋养了无数个童年 杨红樱煲出心灵鸡汤|accessdate = 2015-08-03|author = |date = 2010年9月16日|publisher = 大成网|language = zh-cn}}</ref>。

2007年，作品《淘气包马小跳》系列被哈珀·柯林斯集团购买，在全球发行多语种版。

2008年，作品《笑猫日记》系列被哈珀·柯林斯集团购买，在全球发行的英语、法语、意大利语、西班牙语和韩语的著作版权<ref>{{Cite web|url = http://www.cdwb.com.cn/html/2009-11/08/content_737931.htm|title = 全球热卖 “马小跳”跳进双年展|accessdate = 2015-08-03|author = |date = 2009年11月8日|publisher = 成都晚报|language = zh-cn}}</ref>。

2010年，获第五届[[中國作家富豪榜|中国作家富豪榜]]首富<ref>{{Cite web|url = http://news.sina.com.cn/c/sd/2010-11-15/060621471402.shtml|title = 媒体发布2010年中国作家富豪榜 杨红樱居首|accessdate = 2015-08-02|author = |date = 2010年11月15日|publisher = 新浪网|language = zh-cn}}</ref>。

== 评价 ==

=== 正面评价 ===
*[[中国作家协会|中国作家协会]]副主席[[高洪波_(作家)|高洪波]]认为，杨红樱是一位有潜质的儿童文学作家，一位对社会有爱心、对民族有责任感、对所写的孩子有认同感的作家，期待杨红樱能够更加努力地探索为孩子写作的秘密，写出更多更好的作品，不但给孩子带来“阅读快感”，也能给一代儿童留下“阅读记忆”<ref name=":0">{{Cite web|url = http://www.chinawriter.com.cn/bk/2008-10-16/33571.html|title = 在“多维视野中的杨红樱”学术论坛上，专家提出——应多角度|accessdate = 2015-08-03|author = |date = 2008年10月16日|publisher = 中国作家网|language = zh-cn}}</ref>。

*中国青少年研究中心副主任[[孙云晓|孙云晓]]认为，杨红樱的作品十分关注儿童的人格塑造，探析成人世界与儿童世界之间的隔膜、误区，倡导理解与沟通，让孩子拥有健康、和谐、完美的童年，具有重要的教育学价值<ref name=":0" />。

=== 负面评价 ===
*童话作家林一苇认为，杨红樱的作品看不出丝毫语言美。并且质疑其写作是为迎合孩子们的需求，而童话是需要引导的<ref>{{Cite web|url = http://ent.sina.com.cn/x/2007-06-14/01281596531.shtml|title = 专访童话作家林一苇:童话更多的是读给成年人|accessdate = 2015-08-03|author = |date = 2007年6月14日|publisher = 新浪娱乐|language = zh-cn}}</ref>。
