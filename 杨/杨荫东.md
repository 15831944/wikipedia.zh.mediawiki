'''杨荫东'''（{{bd|1922年||2005年|11月7日}}）[[陕西|陕西]][[合阳|合阳]]人，[[中国共产党|中国共产党]]官员、情报人员，[[国务院台湾事务办公室|中央对台工作领导小组办公室]]主任、[[黄埔军校同学会|黄埔军校同学会]]副会长。

==生平==
杨荫东是合阳县同家庄乡同家庄村人。幼年丧母，由曾祖母赵氏抚养。6岁进[[私塾|私塾]]学习。13岁进入百良镇小学学习。[[抗日战争|抗日战争]]爆发后，1937年9月加入[[中华民族解放先锋队|中华民族解放先锋队]]。同年11月赴[[延安|延安]]，在[[陕北公学|陕北公学]]学习<ref name=heyang>{{Cite web |url =http://www.heyang.gov.cn/info/1017/9894.htm  |title =杨荫东  |author =  |date = 2009-04-12 |publisher = 中国合阳 |language =  |accessdate =  |quote =  }}</ref>。

1938年夏，奉[[中共中央|中共中央]]派遣，加入[[杨虎城|杨虎城]]所部[[国民革命军|国民革命军]]，在[[王屋山|王屋山]]、[[中条山|中条山]]南麓接受军事教育，参加抗战。同年9月加入[[中国共产党|中国共产党]]。在[[中国国民党|中国国民党]][[国共摩擦|第一次反共高潮]]中，中共三十八军工委贯彻中共中央“荫蔽精干，长期埋伏，积蓄力量，以待时机”的方针，派杨荫东考入中国国民党[[中央陆军军官学校|中央陆军军官学校]]第七分校（[[西安|西安]]分校）第十六期。1942年，又派杨荫东考入中国国民党步兵学校。1944年，中国国民党发动[[国共摩擦|第三次反共高潮]]，[[蒋介石|蒋介石]]为瓦解经中共政治改造的国民革命军陆军第三十八军，调该军军长[[赵寿山|赵寿山]]到[[胡宗南|胡宗南]]第三集团军任总司令。经[[毛泽东|毛泽东]]同意，中共三十八军工委派杨荫东任赵寿山的随从参谋，到[[甘肃|甘肃]][[武威|武威]]就任，在赵寿山和[[孙蔚如|孙蔚如]]（[[抗日战争第一战区|第一战区]]副司令长官兼第四集团军总司令，不久改任[[抗日战争第六战区|第六战区]]司令长官）身边从事中共地下工作。其间，杨荫东在[[重庆|重庆]]陆军大学“甲级将官班”连续学习两期，习得大兵团作战知识，并且结识一批中国国民党高级将领<ref name=heyang/><ref name=hpjx>{{Cite web |url =  http://www.huangpu.org.cn/lrfhz/201209/t20120905_3042173.html|title =  杨荫东|author =  |date = 2012-09-05 |publisher = 黄埔军校同学会 |language =  |accessdate =  |quote =  }}</ref>。

1946年1月，为应对[[第二次国共内战|全面内战]]，原中共三十八军工委书记[[蒙鼎钧|蒙鼎钧]]奉[[周恩来|周恩来]]指示，派杨荫东在[[西安|西安]]成立“军事情报组”，在胡宗南部核心刺探战略情报。1947年1月，杨荫东打入[[西安绥靖公署|西安绥靖公署]]第七补给区司令部，任司令办公室少校参谋，后来升为中校参谋。1947年4月，借助杨荫东提供的情报，[[西北野战军|西北野战军]]取得[[蟠龙战役|蟠龙战役]]胜利，受西北野战军司令部嘉奖。1947年8月，陈谢大军[[太岳兵团|太岳兵团]]进攻豫西，杨荫东提供了胡宗南作战计划情报，[[中共中央副主席|中共中央副主席]]周恩来奖给杨荫东所在情报组40两黄金。1948年2月，在[[瓦子街战役|瓦子街战役]]中，因提供情报及时准确，获得西北野战军司令员[[彭德怀|彭德怀]]嘉奖。1949年12月，[[余秋里|余秋里]]在[[四川|四川]][[剑阁|剑阁]]找到杨荫东，并电[[中共中央西北局|中共中央西北局]]派人将杨荫东接回西安，分配到中共中央西北局社会部工作，不久调到[[中共中央调查部|中共中央调查部]]工作<ref name=heyang/><ref name=hpjx/>。

1955年，中共中央成立[[中央对台工作领导小组|对台工作小组]]及其[[国务院台湾事务办公室|办公室]]（办公室简称对台办）。对台办设在中共中央调查部，办公室主任由[[周恩来|周恩来]][[中华人民共和国国务院总理办公室|总理办公室]]主任[[齐燕铭|齐燕铭]]兼，杨荫东负责具体工作。“[[文化大革命|文化大革命]]”后期，杨荫东兼任对台办副主任，任内配合中央有关部门参加了[[中华人民共和国特赦|特赦]]战犯及[[台湾|台湾]]武装特务，清理监狱，宽大释放国民党县团以上人 员，落实投诚起义人员政策，处理[[反右运动|右派]]等工作，持续五年。1979年底，[[邓小平|邓小平]]亲自主持对台工作，[[邓颖超|邓颖超]]、[[廖承志|廖承志]]负责[[中央对台工作领导小组|中央对台工作领导小组]]，对台办被列为中央直属单位，杨荫东任对台办副主任。1982年春升任主任，1985年9月离职。1989年，杨荫东加入[[黄埔军校同学会|黄埔军校同学会]]，任副会长兼秘书长，1993年辞去秘书长职务，专任副会长<ref name=heyang/><ref name=hpjx/>。

2005年11月7日，黄埔军校同学会副会长杨荫东在[[北京|北京]]病逝，享年83岁。2005年11月21日，杨荫东同志遗体告别仪式在[[北京八宝山革命公墓|北京八宝山革命公墓]]举行<ref>苑生，沉痛悼念杨荫东同志，黄埔2006年01期</ref><ref name=hpjx/>。
