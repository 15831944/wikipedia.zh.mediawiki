'''杨绛'''{{noteTag|1={{標音|s=绛|t=絳|拼音=Jiàng|注音=ㄐㄧㄤˋ|同音字=降}}}}（{{bd|1911年|7月17日|2016年|5月25日|Y杨}}<ref name="thepaper">{{cite news|url=http://www.thepaper.cn/newsDetail_forward_1474184|title=杨绛今日凌晨去世，享年105岁|work=澎湃新闻|date=2016-05-25}}</ref>），本名'''杨季康'''<ref>{{cite web|author1=北京市第二中级人民法院|title=杨季康与中贸圣佳国际拍卖有限公司、李国强侵害著作权及隐私权纠纷一案【（2013）二中民初字第10113号】|url=http://bj2zy.chinacourt.org/public/paperview.php?id=1309575|website=北京市第二中级人民法院网站|publisher=北京市第二中级人民法院}}</ref>，小名'''阿季'''。祖籍江蘇無錫，1911年於北京出生，2016年於北京病逝。

== 生平 ==
[[File:QuanjiafuYangjiang.jpg|thumb]]
杨绛，本名杨季康，小名阿季，行四。[[楊蔭杭|楊蔭杭]]之女。祖籍[[江苏|江苏]][[无锡|无锡]]。1911年7月17日出生于北京。她的命名是因为排行老四，“季康”被兄弟姐妹们嘴懒叫得吞了音，变成了减缩版的“绛”。这也是她剧本上演时自己取的笔名。<ref>[http://history.eastday.com/h/2014717/u1a8226593.html 来源:2014-7-17 东方早报 作者:石剑峰]</ref>

1928年，杨绛是苏州[[振华女中|振华女中]]的本年度毕业生，也是苏州十全街旧校址上老振华女校毕业的最后一届学生。<ref>{{cite web |url=http://www.chinanews.com/cul/2011/07-18/3188246.shtml|title=杨绛：“清水芙蓉”铸就不服输性格|accessdate=2011年7月18日|work=|publisher=中国新闻网|language= zh-Hans}}</ref><br />1928年，杨绛17岁，她一心要报考清华大学外文系。清华招收女生，但南方没有名额，杨绛只得转投[[苏州市|苏州]][[东吴大学_(苏州)|东吴大学]]。[[费孝通|费孝通]]与杨绛在中学和大学都同班。<ref name=":0">{{Cite web|url=http://culture.ifeng.com/a/20160525/48843705_0.shtml|title=105岁杨绛今晨逝世 “我们仨”终成绝响|accessdate=2016-05-25|work=culture.ifeng.com}}</ref>

1932年在苏州东吴大学毕业，之後入讀[[北京|北京]][[清华研究院|清华研究院]]做外国语研究生，並认识[[錢鍾書|錢鍾書]]。二人結成夫婦，一同前往[[英国|英国]][[牛津大学|牛津大学]]求学，後轉往[[法國|法國]][[巴黎大學|巴黎大學]]進修。女儿[[钱瑗|钱瑗]]（1937-1997）在英国留学中出生。1938年，楊絳隨錢鍾書帶著一歲的女兒回國。[[File:Yang_Jiang_1941.jpg|thumb]]
1945年，[[夏衍|夏衍]]看了杨绛的沿袭英国式喜剧《弄真成假》、《游戏人间》及悲剧《风絮》，为之赞不绝口：“你们都捧钱锺书，我却要捧杨绛！”<ref name="multiple1">{{cite web |url=http://news.sina.com.cn/c/sd/2011-05-30/173222556328_2.shtml|title=杨绛百年淑子映月泉清|accessdate=2011年5月30日|work=李乃清|publisher=新浪网|language=zh-Hans}}</ref><br />1953年，杨绛任北京大学文学研究所、中国科学院文学研究所、中国社会科学院外国文学研究所的研究员。<ref name=":0" />

1958年，47岁的杨绛决定“偷空自学西班牙语”，从[[西班牙语|西班牙语]]原文翻译《堂吉诃德》。她无师自通，坚持不懈地完成了这项艰巨的翻译工作。但是书稿未完全翻译就历尽磨难，“黑稿子”在“文革”中“被没收、丢弃在废纸堆里”，“九死一生”，才总算逃过文革劫难。<ref name="multiple1"/>

1966年文革开始后，杨绛也如同其他知识分子一样受到迫害和打压，杨绛在外国文学研究所作为“反动学术权威”被“揪出来”，并且被剃了“阴阳头”，<ref name="multiple3">{{cite web|url=http://www.jfdaily.com/a/2263266.htm|title=杨绛这一百年  死者如生，生者无愧|accessdate=2011-08-05|work=|publisher=解放牛网|language=zh-Hans|deadurl=yes|archiveurl=https://web.archive.org/web/20110910203101/http://www.jfdaily.com/a/2263266.htm|archivedate=2011-09-10}}</ref>她和丈夫钱锺书一起被划分为“牛鬼蛇神”，<ref>{{cite web |url=http://www.wenming.cn/cslm/tt_cs/201012/t20101222_35122.shtml|title=青松老人杨绛|accessdate=2010-12-21|work=高莽|publisher=中国文明网|language= zh-Hans}}</ref>她被安排去扫厕所，杨绛爱卫生，在打扫厕所时，她把厕所打扫得干干净净，物品摆放整齐，而且通风透气。红卫兵来整她时，她就躲入女厕所避难。<ref>{{cite web |url=http://news.ifeng.com/history/special/38funvjie/renwuziliao/detail_2010_03/04/360047_0.shtml|title=杨绛：走在人生边上|accessdate=2010年3月4日|work=|publisher=凤凰网|language= zh-Hans}}</ref>

1969年11月，钱锺书被下放到河南[[信阳|信阳]]地区罗山县“五七干校”，不久，随“五七干校”迁至淮河边上的河南息县东岳。1970年7月12日，杨绛也被下放到那，并且被分配在菜园干活。因为两人宿舍不过十分钟路程，故两人经常私密相会。<ref name="multiple3"/>

杨绛本身通晓[[英语|英]]、[[法语|法]]两国语言，为使《堂吉诃德》翻译得更准确，她又学习了[[西班牙语|西班牙语]]，並最终在文革中将书稿保护下来。楊絳《堂吉诃德》譯本被公認為優秀的翻譯佳作，迄今已累計發行70萬冊，是該書中譯本當中發行量最多的译本。1978年《堂吉訶德》中譯本出版時，正好[[西班牙|西班牙]]國王[[胡安·卡洛斯一世|胡安·卡洛斯一世]]訪問中國，[[鄧小平|鄧小平]]把它作為禮物送給了西班牙國王。1986年10月，西班牙国王奖给75岁的杨绛一枚“{{link-es|智慧国王阿方索十世勋章|Orden de Alfonso X el Sabio|智慧国王阿方索十世十字勋章}}”，表彰她对传播西班牙文化所作的贡献。<ref name="multiple1"/>

八十年代以后，杨绛不再将精力放在大作品上，而是偶作回忆性的散文。这些文章虽然都比较短小，但从平淡文字中透露出来的一种朴素而真挚的情感，从大白话中显现出的智慧，至今还被喜爱她的读者津津乐道。

90年代后因为钱锺书先生和钱瑗相继生病去世，杨绛基本停止了各种工作，在家整理丈夫留下的大量手稿包括讀書筆記。《錢鍾書手稿集》分三部。即《容安館札記》、《中文筆記》、《外文筆記》，分別於2003、2011、2016年出版。用她老人家自己的話是：「鍾書他們逃走了，我也想，但我不能走，得留下清理現場。」而整理的結果是，把錢鍾書先生一生中研究學習成就，留給我們子孫後代！

在父女两人相继去世后，她更是深居简出。一直到她92岁高龄之时，才重新提笔，打开尘封多年的记忆，写出了她目前最为脍炙人口的散文集《我们仨》。其中不仅仅用梦幻的方式讲述了钱锺书和錢瑗生前最后一段时光，而且回忆了很多当年夫妇二人游学欧洲，以及中華人民共和國建立后的种种往事。

2001年9月7日，杨绛以全家三人的名义，与[[清华大学|清华大学]]签订了《信托协议书》，成立“好读书奖学金”（与钱先生任“中央图书馆”英文总编纂时主编的馆刊同名）。2001年捐献的现金是72万元；到2010年春，已是800万元，并翻译英国诗人[[兰德|兰德]]的诗歌《生与死》表明心志：“我和谁都不争，和谁争我都不屑；我爱大自然，其次就是艺术；我双手烤着生命之火取暖；火萎了，我也准备走了。”<ref>{{cite web |url=http://www.21ccom.net/articles/rwcq/article_2010071913637.html|title=舒展：杨绛的魅力|date=2010-07-19|accessdate=2010-07-19 |work=舒展|publisher=[[共识网|共识网]]|language= zh-Hans|archiveurl=https://web.archive.org/web/20160625162931/http://www.21ccom.net/articles/rwcq/article_2010071913637.html|archivedate=2016-06-25|deadurl=yes}}</ref>

2011年，百岁老人杨绛查出患有心衰，但她依旧乐观豁达，每天读书写作从不间断，晚上一点半睡觉，早上六点半起床，中午休息两小时。吃得很清淡。健身运动就是室内转圈走动7000步。<ref>{{cite web|url=http://www.ycwb.com/ePaper/ycwb/html/2011-08/10/content_1182611.htm|title=笑眯眯的杨绛|accessdate=2011年8月10日|work=陶然|publisher=羊城晚报|language=zh-Hans|deadurl=yes|archiveurl=https://web.archive.org/web/20111126195318/http://www.ycwb.com/ePaper/ycwb/html/2011-08/10/content_1182611.htm|archivedate=2011年11月26日}}</ref>

2013年7月17日，[[北京|北京]][[清华大学|清华大学]]校长[[陈吉宁|陈吉宁]]在2013年本科生毕业典礼暨学位授予仪式上的讲话上建议：“今天，刚好也是清华老学长杨绛先生102岁生日。杨先生与清华同岁，走过了百年沧桑，历尽了人生坎坷，却始终保持着高贵、生动和优雅的品质，把她的温馨、温暖和安宁传递给我们。前几天我去看她，先生特意叮嘱说，学校很忙，生日时就不要来了，替她吃一碗长寿面就行。我提议，今天我们都去吃一碗长寿面，为我们的老学长贺寿，也为你们的明天祝福。祝愿你们在坚持中成长、在选择中成熟，像杨绛先生那样生命之河长流、事业之树长青！”

2016年5月25日凌晨一时，杨绛在[[北京协和医院|北京协和医院]]病逝，享壽105岁。<ref name="thepaper"/>其遗体于5月27日火化，上午9时从北京协和医院起灵，前往[[八宝山革命公墓|八宝山革命公墓]]。杨绛家中所藏存珍贵文物字画，已于生前全部无偿捐赠[[中国国家博物馆|中国国家博物馆]]；而她的书籍、手稿以及其他财产等，亦捐送给国家有关单位，并指定了遗嘱执行人<ref>[http://www.cna.com.tw/news/firstnews/201605255017-1.aspx 錢鍾書夫人楊絳北京病逝 享壽105歲]中央社，105年5月25日</ref><ref>{{cite web |url=http://media.china.com.cn/cmyw/2016-05-27/741577.html|title=杨绛遗体今火化 手稿、文物字画均捐赠国家|accessdate=2016年5月27日|publisher=中国网|language= zh-Hans}}</ref>。楊絳過世後，網路盛傳一段話「我們曾如此渴望命運的波瀾，到最後才發現，人生最曼妙的風景，竟是內心的淡定和從容。我們曾如此期盼外界的認可，到最後才知道，世界是自己的，與他人毫無關係」，後被證實並非出自楊絳。<ref>[http://www.cna.com.tw/news/acn/201605270096-1.aspx 喝錯心靈雞湯 網傳楊絳語錄是偽作]中央社，105年5月27日</ref>

楊絳著有《風》、《窗簾》、《[[干校六记|干校六记]]》、《[[洗澡_(长篇小说)|洗澡]]》、《[[我们仨|我们仨]]》、《杂忆与杂写》、《[[走到人生边上：自问自答|走到人生边上：自问自答]]》等；剧本《称心如意》、《弄真成假》；译著有《[[堂吉诃德|堂吉诃德]]》、《斐多》、《小癩子》等。

== 评价 ==
[[File:Memorial_to_Yang_Jiang_at_Majiapu_Station_(20160527073824).jpg|thumb]][[马家堡站|马家堡站]]内纪念杨绛的主题公益广告“我们再见了，您们团圆了”]]
钱锺书婶婶对杨绛评价说：“上得厅堂，下得厨房；入水能游，出水能跳。”<ref>{{cite web |url=http://www.people.com.cn/h/2011/0720/c25408-2942649658.html|title=阅读百岁老人杨绛|accessdate=2011年7月20日|work=|publisher=人民网|language= zh-Hans}}</ref>

中国小说学会副秘书长[[卢翎|卢翎]]评价杨绛说：“杨绛的散文平淡、从容而又意味无穷。可谓‘不着一字，尽得风流’。读她的散文更像是聆听一位哲人讲述些烟尘往事，在平静、平淡、平凡中有一种卓越的人生追求。”“（杨先生的）这些散文是我上中国当代文学史课时必讲的篇目。我常对学生们说，先生的作品有一种洞悉世事的深刻，有一种知识分子于乱世固持的良知与操守，还有一种悲天悯人的情怀。这对于当下在浮躁而喧嚣的世界中前行的知识分子独具意义，起码可以使他们理解自己、理解他人，面对宿命更具一种从容、旷达的姿态”。<ref>{{cite web |url=http://news.sohu.com/20110717/n313648185.shtml|title=百岁杨绛：她的坚忍、坚贞与固守|accessdate=2011年7月17日|work=|publisher=搜狐网|language= zh-Hans}}</ref>

哲学家[[周国平|周国平]]评价杨绛说：“这位可敬可爱的老人，我分明看见她在细心地为她的灵魂清点行囊，为了让这颗灵魂带着全部最宝贵的收获平静地上路”。<ref name="multiple2">{{cite web |url=http://www.zaobao.com/wencui/2011/07/liaowang110714a01.shtml|title=杨绛的百岁风华|accessdate=2011-07-14|work=于玉蓉|publisher=联合早报网|language=zh-Hans}}</ref>
