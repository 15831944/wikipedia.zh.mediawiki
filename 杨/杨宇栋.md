'''杨宇栋'''（{{bd|1968年|11月||}}），[[辽宁|辽宁]][[绥中|绥中]]人，1994年5月参加工作，1987年6月加入[[中国共产党|中国共产党]]，[[北京交通大学|北京交通大学]]交通学院交通运输规划与管理专业毕业，研究生学历，工学博士学位，高级工程师<ref name=yyd>{{Cite web |url =  http://cpc.people.com.cn/gbzl/html/161200004.html|title =杨宇栋  |author =  |date =  |publisher = 中国领导干部资料库 |language =  |accessdate =2017-11-15  |quote =  }}</ref>，曾任[[中华人民共和国交通运输部|交通运输部]]副部长。[[中共十九大|中共十九大]]代表，[[中国共产党第十九届中央纪律检查委员会|中国共产党第十九届中央纪律检查委员会]]委员。

==生平==
1987年9月至1991年9月，在[[北方交通大学|北方交通大学]]运输系铁路运输专业学习。1991年9月至1994年5月，为北方交通大学运输管理工程系铁路运输专业硕士研究生。1994年5月至1999年8月，任[[中华人民共和国铁道部|铁道部]][[北京铁路局|北京铁路局]]运输处总调度室见习生、助理工程师、调度员，综合科工程师。1999年8月至2001年8月，任铁道部北京铁路局办公室秘书科秘书，总值班室秘书、副主任。2001年8月至2001年12月，任铁道部北京铁路局北京分局[[丰台西站|丰台西站]]副站长。2001年12月至2003年8月，任铁道部北京铁路局北京分局丰台西站站长。2003年8月至2003年12月，任铁道部北京铁路局北京分局副分局长。2003年12月至2004年11月，任铁道部北京铁路局大同分局副分局长。2004年11月至2005年3月，任铁道部北京铁路局[[大秦铁路|大秦铁路]]股份有限公司副总经理、党委常委<ref name=yyd/>。

2005年3月至2006年3月，任铁道部[[太原铁路局|太原铁路局]]副局长、党委委员。2006年3月至2007年11月，任铁道部[[柳州铁路局|柳州铁路局]]常务副局长、党委常委。2007年11月至2007年12月，任铁道部[[南宁铁路局|南宁铁路局]]常务副局长、党委常委。2007年12月至2011年6月，任铁道部[[中铁快运股份有限公司|中铁快运股份有限公司]]董事长、总经理、党委副书记。其间，2003年9月至2009年1月在北京交通大学交通学院交通运输规划与管理专业在职研究生学习，获工学博士学位。2011年6月至2013年1月，任铁道部[[呼和浩特铁路局|呼和浩特铁路局]]局长、党委副书记。2013年1月至2013年3月，任铁道部北京铁路局局长、党委副书记。2013年3月至2014年8月，任[[中国铁路总公司|中国铁路总公司]]北京铁路局局长、党委副书记。2014年8月至2016年12月，任中国铁路总公司副总经理、党组成员<ref name=yyd/>。

2016年12月起，任[[中华人民共和国交通运输部|交通运输部]]党组成员、副部长，[[国家铁路局|国家铁路局]]局长<ref name=yyd/><ref>{{Cite web |url = http://news.ifeng.com/a/20161217/50431199_0.shtml |title = 杨宇栋任国家铁路局局长 |author =  |date = 2016-12-17 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。2019年1月不再担任交通运输部党组成员、副部长。
