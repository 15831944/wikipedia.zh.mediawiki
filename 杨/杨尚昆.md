'''杨尚昆'''（{{bd|1907年|8月3日|1998年|9月14日|catIdx=Y杨}}），[[四川省|四川省]][[遂宁县|遂宁县]][[双江镇_(潼南县)|双江镇]]人（今屬[[重慶市|重慶市]][[潼南縣|潼南縣]]），[[中國共產黨|中國共產黨]]和[[中華人民共和國|中華人民共和國]]的[[黨和國家領導人|主要領導人]]之一，[[中国人民解放军|中国人民解放军]]高级将领；曾任[[中華人民共和國主席|中華人民共和國主席]]，[[中共中央政治局|中共中央政治局]]委員、[[中共中央軍委|中共中央軍委]]副主席兼秘書長等職，是[[邓小平|邓小平]]主政时期的「[[中共八大元老|中共八大元老]]」之一。杨尚昆被认为是务实派，是[[邓小平|邓小平]][[改革开放|改革开放]]政策的坚定支持者，政治倾向和邓小平的[[新自由主义|新自由主义]]和[[新权威主义|新权威主义]]极为接近。

== 生平 ==
[[清|清]][[光緒|光緒]]三十三年（1907年）6月25日<ref>[http://cpc.people.com.cn/GB/66888/77791/6064304.html 族谱帮大忙，确定杨尚昆诞辰日为8月3日] 人民网</ref>{{NoteTag|出生于1907年8月3日，而不是先前的1907年7月5日（楊尚昆 著.《楊尚昆回憶錄》.中央文獻出版社,2001.第1頁），这跟历史的记忆有关，在战争年代，记忆有可能不太准确。}}，生于[[四川省_(清)|四川省]][[遂宁县|遂宁县]][[雙江鎮|雙江鎮]]（今属重庆潼南）。1920年，考入成都高等师范学校附小；后升入附中；在校期间接触[[马克思主义|马克思主义]]。1925年毕业后，开始参加革命工作；1926年，由[[中國共產主義青年團|中國共產主義青年團]]團員轉為中國共產黨党员；入[[上海大学_(民国)|上海大学]]学习，并参加了[[上海工人三次武装起义|上海工人第一次、第二次武装起义]]的筹备工作。

1926年11月，受中共委派赴[[苏联|苏联]]，入[[莫斯科中山大学|莫斯科中山大学]]，学习马克思列宁主义理论，是“[[二十八个半布尔什维克|二十八个半布尔什维克]]”之一。1929年，与[[李伯钊|李伯钊]]在苏联[[莫斯科|莫斯科]]结婚。第二年，考入苏联中国问题研究院，做研究生，同时兼任职工国际中国代表的翻译。1931年回国后，相继担任[[中华全国总工会|中华全国总工会]]宣传部部长和[[中共中央宣传部|中共中央宣传部]]部长等职；参与工人运动和抗日救亡运动的组织领导工作。

=== 红军时期 ===
[[File:Yang_Shangkun1936-2.jpg|thumb]]、杨尚昆、[[聂荣臻|聂荣臻]]、[[徐海东|徐海东]]。后排左起：[[罗瑞卿|罗瑞卿]]、[[程子华|程子华]]、[[陈光|陈光]]、[[邓小平|邓小平]]。]]

1933年初，杨尚昆到达[[中共中央|中共中央]][[革命根据地|革命根据地]]——[[江西|江西]][[瑞金|瑞金]]；先后但任[[中央苏区|中央苏区]][[中央局|中央局]]宣传干事，[[新华社|红色中华通讯社]]（新华社前身）总负责人，[[马克思|马克思]][[共产主义|共产主义]][[大学|大学]]副校长等职；并参与编辑党的机关报刊《[[新华社|红色中华]]》和《斗争》；同年6月起，参加[[中国工农红军|中国工农红军]]的工作，担任红一方面军政治部主任，随[[朱德|朱德]]、[[周恩来|周恩来]]等转战前方。1934年10月，杨尚昆和[[彭德怀|彭德怀]]率领[[红三军团|红三军团]]参加[[长征|长征]]，任红三军团政治委员；并在中共六届五中全会上，当选为中央候补委员。在[[红军|红军]]到达[[陕北|陕北]]后，杨尚昆历任西北革命军事委员会总政治部副主任，[[中国工农红军|中国工农红军]]抗日先锋军总政治部主任，[[抗大|中国抗日红军大学]]政治部主任，红军前敌总指挥部政治部主任，中央革命军事委员会总政治部副主任等职；曾长时间担任中共军队政治工作的领导职务，并参加了直罗镇、东征和山城堡等战役。<ref>[http://cpc.people.com.cn/GB/69112/89303/89304/index.html 杨尚昆光辉战斗的一生]</ref>

===抗日战争和国共内战===
[[File:Chinese_Communist_Party_Army_Leaders_at_Xiaoyi_County.jpg|thumb]][[孝义|孝义]]碾头村合影。前排左起：[[张友清|张友清]]、[[郭明秋|郭明秋]]、[[林月琴|林月琴]]、[[王秉璋|王秉璋]]，中排左起：[[林枫|林枫]]、杨尚昆、[[罗荣桓|罗荣桓]]，后排居中者为115师343旅旅长[[陈光|陈光]]]]

[[抗日战争|抗日战争]]爆发后，杨尚昆出任[[中共中央北方局|中共中央北方局]]副书记，协助[[刘少奇|刘少奇]]创建“华北敌后抗日根据地”。1938年11月，接任中共中央北方局书记，全面负责中共华北敌后抗日的任务；曾参与组织“[[百团大战|百团大战]]”。1941年回到[[延安|延安]]，协助[[王稼祥|王稼祥]]管理中央华北委员会的工作；并经历了“整风运动”运动的考验。[[國共內戰|國共內戰]]时期，杨尚昆历任中共中央军委秘书长，中央外事组副组长，中央警卫司令员，中央后方委员会副书记，中央办公厅主任，中央副秘书长等职；协助[[周恩来|周恩来]]等处理[[中共中央|中共中央]]和[[中央军委|中央军委]]的日常工作。自1945年10月，杨尚昆一直担任[[中共中央办公厅|中共中央办公厅]]主任一职，直到“[[文革|文革]]”开始；前后执掌中共中央枢要长达20年。

===1949年后===
[[中华人民共和国|中华人民共和国]]成立后，杨尚昆继续担任中共中央副秘书长、中央办公厅主任，同时兼任中央军委秘书长、中直机关党委书记等职。1956年9月，当选中共中央委员；并在中共八届一中全会上，当选[[中共中央书记处|中共中央书记处]]候补书记。1961年，楊捲入[[竊聽器事件_(1961年)|竊聽器事件]]。1963年7月6日至20日，杨尚昆与[[邓小平|邓小平]]为团长，[[彭真|彭真]]为副团长的中共代表团，同[[康生|康生]]、[[刘宁一|刘宁一]]、[[伍修权|伍修权]]在[[莫斯科|莫斯科]]与苏共中央举行了9次会谈，[[苏斯洛夫|苏斯洛夫]]为苏方代表团团长，[[鮑里斯·尼古拉耶維奇·波諾馬廖夫|波诺马廖夫]]和[[安德罗波夫|安德罗波夫]]也参加辩论，双方就[[国际共产主义运动|国际共产主义运动]]中出现的问题开展论争<ref>{{cite web |title=1963年7月5日 中共代表团抵达莫斯科 |url=http://cpc.people.com.cn/GB/4162/64165/67447/67506/4558868.html |website=人民网 |accessdate=2019-01-05}}</ref><ref>{{cite book |language=zh-hans |authors=陈继安、刘金田 |title=《邓小平与20世纪政治人物》 |year=1996 |location=北京 |publisher=北京出版社 |ISBN=978-7-200-03084-6 |ref=russ2 }}</ref>{{rp|329-332}}<ref>{{cite web |title=1963年7月6日 中苏两党关系恶化 |url=http://cpc.people.com.cn/GB/4162/64165/67447/67562/4562887.html |website=中国共产党新闻网 |accessdate=2018-12-07 }}</ref>

1964年至1965年，曾带队到[[陕西省|陕西省]][[长安县|长安县]]（今[[西安市|西安市]][[长安区|长安区]]）斗门公社牛角大队搞“[[四清运动|四清运动]]”。1965年，因竊聽器事件，楊改任廣東省書記處書記。[[文革|文革]]开始后，杨尚昆即被列为“[[彭真|彭真]]、[[罗瑞卿|罗瑞卿]]、[[陆定一|陆定一]]、杨尚昆反党集团”的成员；被撤销一切职务，受到「監護審查」，直到[[中共十一届三中全会|中共十一届三中全会]]后，方才平反獲釋<ref>[http://news.ifeng.com/history/1/renwu/200807/0718_2665_660804.shtml 非常岁月：1965年杨尚昆被贬离京 坐牢12年]</ref>。

===文革期间===
{{Wikisource|中共中央关于陆定一同志和杨尚昆同志错误问题的说明}}

1966年6月27日，[[北京时间|北京时间]]下午4时，國家主席[[劉少奇|劉少奇]]在[[人民大会堂|人民大会堂]][[安徽|安徽]]廳与中共中央召集的民主人士座談會上，讨论彭真、羅瑞卿、陸定一、楊尚昆幾個同志的問題。“彭、羅、陸、楊他們的互相關係是不正常的，到底是什麼關係，達到何種程度，我們組織了審查委員會，正在進行審查。他們共同特點是反對毛主席，反對毛澤東思想，都是搞地下活動的。他們的企圖不是個人要點什麼東西，而是企圖根本改變我們黨和國家的根本路線、根本政策，要按照他們的路線辦事，要按他們的面貌改造党，改造全中國，企圖在中國實現修正主義政策，也就是復辟資本主義的政策，如果他們的企圖得逞，就可能實現政變。”“彭、羅、陸、楊事件是有發生政變的可能的，這是激烈的國際、國內階級鬥爭在我們黨內領導機關的反映。”<ref>[http://www.singtaonet.com:82/global/wg/wg10/t20060515_213078.html  劉少奇批彭羅陸楊（1966）] {{webarchive|url=https://web.archive.org/web/20121026023254/http://www.singtaonet.com:82/global/wg/wg10/t20060515_213078.html |date=2012-10-26 }}</ref>

1978年12月至1980年底，杨尚昆相继担任[[中共广东省委|中共广东省委]]第二书记、广东省副省长，[[中共广州市委|中共广州市委]]第一书记、广州市[[革命委员会|革命委员会]]主任，兼任[[广东省军区|广东省军区]]第一政委、党委第一书记；是邓小平“[[改革开放|改革开放]]”政策的坚定执行者之一。

=== 当选国家主席 ===
{{See also|六四事件|杨家将事件}}

在1982年和1987年召开的中共第十二次、第十三次全国代表大会上，杨尚昆当选为中共中央委员，在十二届一中全会上当选为[[中共中央政治局|中共中央政治局]]委员，继任[[中共中央军委副主席|中共中央军委副主席]]兼秘书长<ref>{{cite book |authors = 张明金、赵功德 |title = 中国人民解放军历史上的70个军 |year = 2006年 |publisher = 解放军文艺出版社 |ISBN = 978-7-5033-1940-2 |page = 94 }}</ref>。1988年4月，在[[第七届全国人民代表大会|第七届全国人民代表大会]]第一次会议上，杨尚昆接替[[李先念|李先念]]当选为[[中华人民共和国主席|中华人民共和国主席]]。1990年5月，杨尚昆在当选国家主席的第二个年头，将首次出访的目的地定为[[墨西哥|墨西哥]]、[[巴西|巴西]]、[[乌拉圭|乌拉圭]]、[[阿根廷|阿根廷]]和[[智利|智利]]五国。这是[[中华人民共和国国家元首列表|中国国家元首]]首次访问[[拉丁美洲|拉丁美洲]]。1992年8月24日，[[1992年中韓建交|中华人民共和国与大韩民国建交]]，9月28日，国家主席杨尚昆在[[北京|北京]]会见了来华访问的韩国总统[[卢泰愚|卢泰愚]]。

1992年春，曾陪同已經退休的[[鄧小平|鄧小平]]作[[邓小平南巡|南方巡視]]。经过[[杨家将事件|杨家将事件]]后<ref name="東方明1997">{{cite book |author = 東方明 |title=《鄧小平垂簾聽政》|url = http://books.google.com/books?id=rUc7AAAACAAJ |year = 1997年 |publisher=天安門出版社 |ISBN = 978-962-977-001-3 }}</ref>，1992年10月和1993年3月，杨尚昆先后从中共中央政治局委员、中央军委副主席和国家主席的领导岗位完全退下来。1998年9月14日，楊尚昆因病在[[北京|北京]]逝世，享年92岁。他[[党和国家领导人#官方评价|被官方稱為]]“'''伟大的无产阶级革命家、政治家、军事家，坚定的马克思主义者，党、国家和人民军队的卓越领导人'''”。杨尚昆去世后，[[中央文献出版社|中央文献出版社]]出版了《杨尚昆日记》，涵蓋时期为1949年1月1日至1965年12月10日，是研究[[中共党史|中共党史]]和[[中华人民共和国历史|中华人民共和国历史]]的重要资料<ref>{{cite book |title = 杨尚昆日记 |url = http://books.google.com/books?id=rSKgQgAACAAJ |year = 2001年 |publisher = 中央文献出版社 |ISBN = 978-7-5073-1062-7 }}</ref><ref>{{Cite news |url = http://news.ifeng.com/history/zhuanjialunshi/gaohua/detail_2009_12/03/320268_0.shtml |author=[[高华|高华]] |title = 初读《杨尚昆日记》|date = 2009年12月3日 |website = 凤凰网 }}</ref>。

==家庭==
{{二十八个布尔什维克}}
*妻子[[李伯钊|李伯钊]]，原[[中央戏剧学院|中央戏剧学院]]院长。
*長子楊紹京，又名「紹軍」，原[[国防科工委|国防科工委]]某研究所长。已故，有兩女。
*次子楊紹明，著名攝影家，世界华人摄影学会会长、当代摄影学会主席，现任[[中国宋庆龄基金会|中国宋庆龄基金会]]副主席。有兩女。
*女兒[[楊李|楊李]]，总参谋部少将，中国友联会长助理。其夫王小朝，历任澳门兴龙行有限公司董事兼总经理、保利科技有限公司董事兼副总经理、保利通信公司董事长、保利南方集团有限公司董事长、海南洋浦土地开发有限公司总经理。现任中国保利集团公司党委副书记、董事，兼任中国保利集团公司一级子公司监事会主席。

楊尚昆共有十三名兄弟姐妹，自己排行第五。
*大哥楊尚荃（楊劍秋），[[同盟會|同盟會]]會員，曾參加護國運動、護法運動，後興辦實業，1937年去世。
*二哥楊尚麟（楊衡石），同盟會會員，後加入中國共產黨。
*四哥楊尚述（[[楊闇公|楊闇公]]），中國共產黨黨員，著名烈士，積極參加國民革命和北伐。1927年在[[三·三一慘案|三·三一慘案]]中被四川軍閥逮捕，被割舌，捥目，斷手，最後犧牲就義。
*六妹楊尚朋，早年參加中國共產黨，文革中被迫害致死。
*七妹楊尚友，體育界人士。

*九弟楊尚直。
*十弟楊尚明，畢業於南京[[中央軍校|中央軍校]]第十四期。
*十一妹楊尚智，不詳。

*十三妹楊白琳，中國共產黨黨員，[[廖汉生|廖汉生]]的妻子，原在教育部机关工作。
