'''楊傳堂'''（{{bd|1954年|5月|catIdx=Y}}），[[山东省|山东省]][[禹城市|禹城市]]人。1972年6月进入[[中国人民解放军|中国人民解放军]]服役，1976年6月加入[[中国共产党|中国共产党]]。[[山东师范大学|山东师范大学]]中文系干部专修科毕业，[[中共中央党校|中共中央党校]]经济管理专业本科学历。[[中国共产党第十六届中央委员会候补委员列表|中共第十六届中央候补委员]]、[[中国共产党第十六届中央委员会委员列表|中央委员]]（十六屆七中全會遞補），[[中国共产党第十七届中央委员会委员列表|第十七届]]、[[中国共产党第十八届中央委员会委员列表|十八届中央委员]]。曾任[[青海省人民政府|青海省人民政府]]省长、[[中国共产党西藏自治区委员会|中共西藏自治区党委]]书记、[[中华人民共和国交通运输部|中华人民共和国交通运输部]]部长等职务。现任第十三届[[全国政协副主席|全国政协副主席]]、党组成员，中华人民共和国交通运输部党组书记。

== 早年 ==
1972年6月，楊傳堂18岁时，参军入伍，在[[中国人民解放军济南军区|濟南軍區]]當戰士。1975年复员，被分配到山東黃河農場当工人。后进入山東勝利石化總廠工作；历任第二化肥廠工人，團委幹事，車間黨委書記，二化肥廠車間書記，廠黨委副書記。1978年起，参与建设[[中国石油化工集团|齊魯石化公司]]乙烯生產项目。1981年2月至1983年1月，曾在[[山东师范大学|山东师范大学]]中文系幹修科学习。进修结束后，回到齊魯石化公司工作。1984年12月，任齊魯石化公司黨委副書記。1987年8月，任[[中国共产主义青年团|共青團]]山東省委副書記，开启仕途。

==两次入藏==
1987年至1992年间，任共青團山東省委書記。1992年2月，任中共[[德州市|德州]]地委副書記、德州地区行署專員。1993年11月，当选[[中国共产党山东省委员会|中共山东省委]]常委，当月即调任[[中国共产党西藏自治区委员会|中共西藏自治區黨委]]常委、自治區副主席。后任中共西藏自治区党委副书记、政府常务副主席，中共西藏自治区党委常务副书记兼党校校长。2002年11月，当选[[中国共产党第十六届中央委员会候补委员列表|中共第十六届中央候补委员]]。

2003年10月，接替[[赵乐际|赵乐际]]，任[[中国共产党青海省委员会|中共青海省委]]副書記、副省長、代省长；2004年1月，在青海省人大十届二次会议上，正式当选青海省省长。

2004年12月，再次奉调入藏，任[[中国共产党西藏自治区委员会|中共西藏自治區黨委]]書記。

==回京养病==
2005年11月27日，[[中国共产党中央委员会|中共中央]]突然宣布由[[张庆黎|张庆黎]]代理[[中国共产党西藏自治区委员会|中共西藏自治区党委]]书记。外界有报导称，当年9月，楊傳堂因为突发[[脑溢血|脑溢血]]导致重度昏迷，被接到[[北京市|北京]]救治<ref>{{Cite web|url=http://news.creaders.net/china/2005/11/27/418839.html|title=原西藏书记杨传堂传脑溢血|accessdate=2017-09-14|author=|date=2005-11-27|work=|publisher=万维读者网}}</ref>，但是官方没有证实。

2006年5月29日，楊傳堂被正式免去[[中国共产党西藏自治区委员会|中共西藏自治区党委]]书记，任[[中华人民共和国国家民族事务委员会|國家民族事務委員會]]副主任，保留正部長級待遇。此后五年多的时间里，楊傳堂一直流转在不甚重要的岗位上。2007年10月12日，被递补为[[中国共产党第十六届中央委员会委员列表|中央委员]]。在随后举行的[[中国共产党第十七次全国代表大会|中共十七大]]上，继续当选第十七届中央委员。2011年8月，任[[中华全国供销合作总社|中华全国供销合作总社]]理事会主任、党组书记。

==再次复出==
[[中国共产党第十八次全国代表大会|中共十八大]]召开前夕，58岁的楊傳堂重归一线正部级职位。2012年7月，任[[中华人民共和国交通运输部|中华人民共和国交通运输部]]党组书记。后在8月31日结束的全国人大常委会上，正式任交通运输部部长<ref>{{Cite web|url=http://news.sina.com.cn/c/2012-08-31/122525075824.shtml|title=杨传堂被任命为交通运输部部长|accessdate=2017-09-14|author=|date=2012-08-31|work=|publisher=新浪}}</ref>。

2016年9月3日，杨传堂不再担任[[中华人民共和国交通运输部|中华人民共和国交通运输部]]部长<ref>{{cite web|accessdate=2016-09-02|url=http://www.thepaper.cn/newsDetail_forward_1523581|title=李小鹏任交通运输部部长|publisher=澎湃新闻}}</ref>，仍担任交通运输部党组书记<ref>{{cite web|accessdate=2016-09-04|url=http://www.thepaper.cn/newsDetail_forward_1523944|title=杨传堂担任交通运输部党组书记，李小鹏任部长、党组副书记|publisher=澎湃新闻}}</ref>。2018年3月14日，64岁的杨传堂当选第十三届[[全国政协副主席|全国政协副主席]]<ref>{{cite web|title=张庆黎等当选全国政协副主席|url=http://news.ifeng.com/a/20180314/56733773_0.shtml|website=凤凰网|accessdate=2018-03-14}}</ref>，晋升副国级。

==家庭==
楊傳堂的哥哥[[杨传升|杨传升]]曾任[[中国共产党中央纪律检查委员会|中共中央紀律檢查委員會]]委員、[[中共山东省委秘书处旧址|中共山東省委]]常委、山東省紀委書記。2007年，與楊傳堂雙雙當選[[中国共产党第十七次全国代表大会|中共十七大]]代表。

==名言==
2015年3月12日在接受[[京华时报|京华时报]]记者采访时表示：“私家车不许当专车，[[出租車|出租車]]无政府垄断”<ref>{{cite web |url = http://news.xinhuanet.com/politics/2015-03/13/c_127575768_2.htm |title=交通部长：私家车不许当专车 出租车无政府垄断 |accessdate=2017-09-14 |date=2015-03-13 |website=新华网 }}</ref>。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web |url = http://sports.163.com/18/0126/06/D92BRKK400051C89.html |title = 中国人民政治协商会议第十三届全国委员会委员名单 |accessdate=2018-01-26 |website = 网易 }}</ref>。
