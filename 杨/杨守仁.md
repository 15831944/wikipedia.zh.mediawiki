'''杨守仁'''（{{bd|1871年||1911年|8月5日}}），原名'''毓麟'''，字'''笃生'''。[[湖南省|湖南省]][[长沙县|长沙县]][[高桥镇|高桥镇]]白石园人。中国民主革命家。<ref name=cswb>[http://cswb.changsha.cn/CSWB/20090522/Cont_1_19_99324.HTM 长沙志士英国蹈海以身报国，长沙晚报，2009-05-22]</ref>

== 早年經歷 ==
杨守仁15岁上学。1897年，杨守仁中[[举人|举人]]，1898年，他中[[进士|进士]]，分发[[广西|广西]]知县，但未赴任。此后，他入[[湖南|湖南]][[时务学堂|时务学堂]]，和[[梁启超|梁启超]]、[[唐才常|唐才常]]、[[谭嗣同|谭嗣同]]等人提倡变法，推动湖南[[维新运动|维新运动]]。[[戊戌变法|戊戌变法]]失败后，他避居乡里几个月。<ref name=cswb/>

== 中年經歷 ==
1902年，湖南官紳開始選派公費生東渡，並將留日血費生之貧苦者亦悉化為公費。<ref name="吳相湘">[[吳相湘|吳相湘]]：《[[宋教仁|宋教仁]]：中國民主憲政的先驅》，[[台北|台北]]：[[傳記文學|傳記文學]]出版社，1985年9月15日新版</ref>{{rp|8}}是年春，他留学日本，入[[宏文学院|宏文学院]]，後考入[[早稻田大学|早稻田大学]]。[[黄兴|黄兴]]、[[楊度|楊度]]等既至[[東京|東京]]，時與日本師友接觸，問學研討之餘，深感救國之道，必先有理論，而後有事實，有[[學術|學術]]而後有[[政治|政治]]；尤須破除過去定於一尊之學風，始能盡除保守濡滯之積習；而介紹東西洋自由之學說，尤為開啟新機運之前提。<ref name="吳相湘"/>{{rp|9}}1902年10月，因之相與集議创刊《游学译编》。<ref name="吳相湘"/>{{rp|9}}。

1903年，[[沙俄|沙俄]]向清朝提出7項要求，企圖侵佔[[中國東北地區|中國東北地區]]，此舉觸及日本，東京《[[朝日新聞|朝日新聞]]》首先刊載，留日學生遂於[[千代田區|神田]]錦輝館召開學生大會。<ref name="教育與傳承">梁操雅、羅天佑：《教育與承傳：歷史文化的視角》，[[商務印書館（香港）|香港教育圖書公司]]，2011年1月20日</ref>{{rp|21}}4月29日，留日學生決定組織義勇隊抗俄以為[[國民|國民]]倡導。<ref name="吳相湘"/>{{rp|16}}據統計，各省留學生到會者500餘人，通過成立拒俄義勇隊，推陸軍士官學校[[藍天蔚|藍天蔚]]為隊長，成員有黃興、[[楊守仁|楊守仁]]、[[陳天華|陳天華]]等，日日操練，意欲隨時開赴前線。<ref name="教育與傳承"/>{{rp|21}}當中，尤以青年會成員最為積極，他們大都參加「拒俄義勇隊」，130人表示願赴戰場者，50人留東京辦事者，有女留學生願擔任看護。<ref name="教育與傳承"/>{{rp|21－22}}留學生正欲改名學生軍，在清朝壓力下，日本政府勒令解散。<ref name="教育與傳承"/>{{rp|22}}5月10日，留日學生惟有在學生軍集會上，議決改名為「军国民教育会」，以便跟進事態發展。<ref>[[張玉法|張玉法]]：《清季的革命團體》，台北：[[中央研究院|中央研究院]]近代史研究所，1982年</ref>{{rp|257－273}}<ref>〈軍國民教育會會約〉，1903年，刊〈紀事〉，《蘇報》1903年第二期</ref>其精神則不變，且公推代表二人回國，請[[袁世凱|袁世凱]]主戰。<ref name="吳相湘"/>{{rp|16}}北京大學堂及上海教育界均覆電贊成，並紛紛集會，擴大宣傳力量。<ref name="吳相湘"/>{{rp|16}}是年6月初，黃興自日本回國，在湖南、湖北醞釀新行動。<ref name="吳相湘"/>{{rp|17}}6月21日，清朝命令嚴厲緝拿倡言革命之學生，意圖以高壓手段鎮壓風潮，然而大勢所趨已不可遏阻，留日學生各種刊物均廢光緒年號，直書[[黃帝|黃帝]]紀元。<ref name="吳相湘"/>{{rp|17}}其间，楊守仁加入[[兴中会|兴中会]]。<ref name=cswb/>

是年[[夏天|夏天]]，受[[歐榘甲|歐榘甲]]主張[[廣東獨立|廣東獨立]]的《[[新廣東|新廣東]]》鼓舞，楊守仁復署名“湖南之湖南人”，刊行《新湖南》一小冊，大力主張湖南獨立，遍告湖南中等社會，以恥舊湖南人之甘為奴者。<ref name="吳相湘"/>{{rp|14}}楊守仁慷慨激昂<ref>痛論：「湖南之公敵在官場巨紳……未來之湖南，猶樹也，溉之以頑官劣紳劬民瘁士之血而後生長焉。」見吳相湘：《宋教仁：中國民主憲政的先驅》，台北：傳記文學出版社，1985年9月15日新版，第14頁</ref>，因之大力提倡開闢新局面，組織獨立之機關，規劃獨立之地方自治；而欲達此破壞及建設之目的，則組織黨會，固其大前提。<ref>此篇據日本高材世雄所論，而增其言：「今世立國於地球之上，不能無以黨會為基礎，不能無以黨會為基礎也。且夫以黨人各占其會黨之一部份，則會黨立；以會黨各占湖南之一部份，則湖南立。……吾自為之，他人亦自為之者，可與他人合為之，亦可與他人分而為之……必使各分省自任一部之位置，各分省發見其獨立之親和力……」見吳相湘：《宋教仁：中國民主憲政的先驅》，台北：傳記文學出版社，1985年9月15日新版，第14頁</ref>[[黃興|黃興]]至[[長沙|長沙]]應聘明德學堂主講席，課餘則秘密進行革命活動。<ref>劉揆一：《黃興傳記》，台北，1952年</ref>{{rp|1－2}}

{{wikisource|新湖南}}

黃興、楊守仁留日時即主張暗殺清廷大吏，奪取要府以為革命手段，嘗研究爆炸物10餘種。<ref name="吳相湘"/>{{rp|20}}1904年夏，黃興回到湖南，楊守仁即偕兩位[[同志|同志]]由日本攜帶炸藥至[[北京|北京]]，約張繼由湖南、何海樵由上海前來，設機關於[[天津|天津]]，圖謀炸死[[慈禧太后|慈禧太后]]於[[紫禁城|紫禁城]]內城宮殿及[[颐和园|颐和园]]以震動天下人耳目，然潛居京城數月，無隙可乘，乃失意南歸。其间，他因为研制炸药而受伤，一只眼睛失明。同年秋，黄兴、[[刘揆一|刘揆一]]、宋教仁等人策划在长沙起义。得知華興會在湖南起義計劃，楊乃留在上海，與章行嚴等密設愛國協會，以為接濟，並結合留日各省同，遙為聲援。<ref name="吳相湘"/>{{rp|20}}杨和[[蔡元培|蔡元培]]、[[杭辛斋|杭辛斋]]等加以配合。后来起义事泄，他改名“杨守仁”，流亡日本。<ref name=cswb/>后来他回到长沙，与黄兴和[[宋教仁|宋教仁]]一道。至11月，遂於長沙創立[[華興會|華興會]]。<ref name="吳相湘"/>{{rp|18}}

1906年，他在[[东京|东京]]加入[[中国同盟会|中国同盟会]]。1907年，他到上海，和[[于右任|于右任]]等人等创办《[[神州日报|神州日报]]》，并任总主笔，任职80天，以“寒灰”为笔名发表了社论和时评。不久，《神州日报》因火灾而暂停出版。此后，他到[[欧洲|欧洲]]，并于1909年入[[苏格兰|苏格兰]][[阿伯丁大学|爱伯汀大学]]学习[[英语|英语]]。其间，他会见了流亡[[伦敦|伦敦]]的[[孙中山|孙中山]]，并建议孙中山在欧洲设[[通讯社|通讯社]]，获得孙的赞同。此后，他作为[[中国同盟会|中国同盟会]]驻英国联络员，不断为《[[民主报|民主报]]》、《[[中兴日报|中兴日报]]》投稿，并且秘密购买炸药运回中国，提供给革命党。<ref name=cswb/>

== 身亡 ==
1911年4月，[[广州|广州]][[黄花岗起义|黄花岗起义]]失败，杨守仁十分悲痛。不久，他读报纸时得知列强企图再次瓜分中国，非常气愤。1911年8月5日，他在[[利物浦|利物浦]]的[[大西洋|大西洋]]海湾投海身亡。<ref name=cswb/>

== 安葬 ==
其遗体被英国渔民打捞上岸，[[杨昌济|杨昌济]]等人闻讯后也迅速赶到了其遇难处。经过同家属商量，他的遗体被安葬在[[利物浦|利物浦]]的[[安菲尔德公墓|安菲尔德公墓]]。1912年，孙中山在[[南京|南京]]为'''杨守仁'''、[[陈天华|陈天华]]、[[吴樾_(革命家)|吴樾]]、[[熊成基|熊成基]]、[[郑先声|郑先声]]、[[杨卓林|杨卓林]]等六位烈士建立了[[六烈士祠|六烈士祠]]。<ref name=cswb/>
