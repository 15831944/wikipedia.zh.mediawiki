'''杨信'''{{BD|1950年|5月|||catIdx= Y}}[[黑龙江|黑龙江]][[绥化|绥化]]人，1969年9月加入中国共产党，1968年11月参加工作，专业技术职务为研究员级高级农艺师，中央党校经济管理专业、[[哈尔滨师范大学|哈尔滨师范大学]]经济管理专业毕业，中央党校在职研究生。曾任大庆市人民政府市长，齐齐哈尔市人民政府市长，中共齐齐哈尔市委书记，齐齐哈尔市人大常委会主任。2014年9月，因涉嫌“严重违纪违法”被[[中纪委|中纪委]]调查。

==生平==
1950年，杨信出生在[[黑龙江省|黑龙江省]][[绥化市|绥化市]]。杨信曾在中共中央党校经济管理专业、哈尔滨师范大学经济管理专业学习。<ref name="fish" >{{cite web |url=http://news.163.com/14/0925/18/A70PS96M00014SEH.html |title=齐齐哈尔市委原书记杨信涉嫌严重违纪违法被调查 |accessdate= 2014|publisher=网易 |work= |language=zh-hans}}</ref>

1968年11月，杨信参加政治工作，1969年9月，他加入[[中国共产党|中国共产党]]。<ref name="fish" />

1968年到1980年8月，杨信在绥化县（绥化市）工作，曾担任公社干部、党委委员、组织部干事。<ref name="fish" />

1980年8月到1983年5月，杨信调到[[青冈县|青冈县]]，最后担任该县副县长。<ref name="fish" />

1983年5月到1985年4月和1986年4月到1986年8月，杨信调回绥化地区（绥化市）任职，两度担任“整党办公室副主任”。<ref name="fish" />

1985年4月到1986年4月，杨信担任黑龙江省[[海伦县|海伦县]]县委副书记。<ref name="fish" />

1986年8月到1989年7月，杨信担任黑龙江省[[庆安县|庆安县]]县委书记。<ref name="fish" />

1989年7月到1992年11月，杨信调回绥化地区（绥化市），担任行署副专员。<ref name="fish" />

1992年11月到1996年11月，杨信出任[[大庆市|大庆市]]副市长，随后担任大庆市委常委、常务副市长，1997年11月，升任大庆市委副书记、市长。<ref name="fish" />

2000年3月到2000年8月，杨信调到[[哈尔滨|哈尔滨]]，出任黑龙江省农委主任、党组书记。<ref name="fish" />

2000年8月到2001年2月，杨信担任[[齐齐哈尔市|齐齐哈尔市委副书记]]、代市长、党组书记，2001年2月，杨信得到连任，并且正式行使市长职权，2003年3月，杨信第三次得到连任。<ref name="fish" />2003年6月，杨信被擢升为中共齐齐哈尔市委书记、市人大常委会主任、党组书记。<ref name="fish" />2004年2月，杨信连任中共齐齐哈尔市委书记，2009年2月，杨信连任市委书记并且兼职市人大常委会主任。<ref name="fish" />2010年2月到2012年1月，杨信再度担任齐齐哈尔市人大常委会主任。<ref name="fish" />

2014年9月25日，因涉嫌“严重违纪违法”接受组织调查。<ref>{{cite web |url=http://www.chinadaily.com.cn/hqsj/shbt/2014-09-26/content_12449044.html |title= 齐齐哈尔原市委书记被查|accessdate= 2014|publisher=中国日报网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://news.xinhuanet.com/legal/2014-09/25/c_1112630855.htm |title= 齐齐哈尔市委原书记杨信接受组织调查 |accessdate=2014 |publisher=新华网 |work= |language=zh-hans}}</ref>2015年11月，被开除党籍<ref>{{cite news|url=http://www.thepaper.cn/newsDetail_forward_1397199|title=齐齐哈尔市人大原主任杨信被开除党籍，曾违规买卖股票|accessdate= 2015-11-15|publisher=澎湃新闻}}</ref>。
