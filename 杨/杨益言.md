'''杨益言'''（{{bd|1925年|11月18日|2017年|5月19日}}）原籍[[四川省|四川省]][[武胜县|武胜县]]，生于[[重庆市|重庆市]]，[[中华人民共和国|中华人民共和国]]作家<ref name=zgzjw>{{Cite web |url =http://www.chinawriter.com.cn/zxhy/member/6082.shtml  |title =杨益言 (1925～)  |author =  |date =  |publisher =中国作家网  |language =  |accessdate =2017-05-19  |quote =  }}</ref><ref name=caixin>{{Cite web |url = http://culture.caixin.com/2017-05-19/101092690.html |title = 《红岩》作者杨益言逝世 享年92岁 |author =  |date = 2017-05-19 |publisher =  财新网|language =  |accessdate =  |quote =  }}</ref>，《[[红岩|红岩]]》的主要作者之一。

==生平==
杨益言先后在四川省武胜县、重庆市完成初中、高中学业。[[抗日战争|抗日战争]]时期，[[同济大学|同济大学]]自上海内迁至重庆。1944年，杨益言考入重庆的同济大学工学院电机系。抗日战争胜利后，随同济大学迁返上海。1948年初，因为参加“[[反饥饿、反内战、反迫害运动|反饥饿、反内战、反迫害运动]]”而被逮捕。同年获释后回到重庆，在中国铅笔厂职工夜校当教员<ref name=caixin/><ref name=sunshu>孙曙，是“见到的”还是编造的——评杨益言《我见到的“中美合作所”》，社会科学论坛2003(11):81-85</ref>。

1948年8月，杨益言遭[[中国国民党|中国国民党]]特务逮捕，关押在重庆[[渣滓洞|渣滓洞]]集中营。在狱中，杨益言结识了[[江竹筠|江竹筠]]、[[罗广斌|罗广斌]]等人。狱友们曾秘密举行追悼会、集体绝食等活动。1949年[[春节|春节]]，狱友们开联欢会，其中杨益言表演[[二胡|二胡]]，罗广斌戴着铁脚镣表演了[[踢踏舞|踢踏舞]]<ref name=caixin/>。1949年4月，杨益言的家属出钱，通过杨益言在重庆市警察局任局员的堂兄花钱托关系，将杨益言释放出狱<ref name=sunshu/>。

1949年[[中华人民共和国|中华人民共和国]]成立后，杨益言历任重庆《团刊》编辑、科长，[[中国共青团|共青团]]重庆市委常委、办公室主任，[[中共重庆市委|中共重庆市委]]《支部生活》总编辑，专业作家。第四、五届[[四川省政协|四川省政协]]委员，[[中国文联|中国文联]]第四届全委会委员，[[中国作协|中国作协]]第三、四届理事及第五、六、七届名誉委员，四川省重庆市文联副主席、作协副主席。1951年开始发表文学作品，1963年加入中国作家协会<ref name=zgzjw/>。1952年，经[[刘德彬|刘德彬]]等人介绍，杨益言加入[[中国共产党|中国共产党]]<ref name=sunshu/>。

中华人民共和国初期，罗广斌、刘德彬、杨益言合作创作了报告文学《圣洁的血花》，记述了他们在狱中对敌斗争的经历，1950年7月1日起在重庆《大众文艺》上发表<ref name=caixin/>。1958年初，三人在《红旗飘飘》第六集上合作发表革命回忆录《在烈火中永生》。1959年2月，[[中国青年出版社|中国青年出版社]]出版了增订后的单行本回忆录《在烈火中永生》<ref name=zhangyu>张羽，《红岩》成书之前，编辑之友1986年第3期</ref><ref name=caixin/>。

1950年代，杨益言、罗广斌同在共青团重庆市委任职，经常一同做报告以开展革命传统教育<ref name=caixin/>。1958年11月，[[共青团中央|共青团中央]]常委、中国青年出版社党委书记、社长、总编辑[[朱语今|朱语今]]抵达重庆，认为革命者在渣滓洞、白公馆的狱中斗争是开展革命传统教育的良好题材，乃向当时在共青团重庆市委任职的罗广斌、杨益言约写长篇小说<ref name=wch/>。中共重庆市委对此大力支持，由中共重庆市委常委、组织部长[[萧泽宽|萧泽宽]]具体负责，指定罗广斌、杨益言二人共同执笔创作，并为二人专门安排了两间平房<ref name=wch/><ref name=caixin/>。自1959年开始，二人白天访问老革命者、搜集材料，晚间写作。当时正值“[[三年困难时期|三年困难时期]]”，粮食紧缺，二人因为承担该写作任务而可每日分得一个小馒头<ref name=caixin/>。中共重庆市委准许二人查看敌特档案，采访在押的敌特分子。二人在敌特档案中发现了一整套时长15年的特务日记，这为在小说中塑造[[军统|军统]]高层特务形象提供了素材。在小说出版前，中共重庆市委决定，要二人拜作家[[沙汀|沙汀]]为师，并到[[北京|北京]]学习参观<ref name=wch/>。1961年，二人完成了小说《[[红岩|红岩]]》，同年12月由[[中国青年出版社|中国青年出版社]]出版，此后数十年间印数超过一千万册。《红岩》出版后受到热烈欢迎，并被译为多种文字，还被改编为电影《烈火中永生》以及[[豫剧|豫剧]]《江姐》等<ref name=caixin/>。

[[文化大革命|文化大革命]]期间，罗广斌、杨益言被诬陷为“叛徒”，遭到残酷迫害（罗广斌于1967年2月10日去世）<ref name=wch>王春华，《红岩》作者罗广斌之死，廉政瞭望2008年第7期</ref><ref>杨益言，叛徒江青为什么扼杀《红岩》，人民日报1977年10月29日</ref>。1979年，杨益言出席中国文学艺术工作者第四次代表大会，并当选为中国文学艺术界联合会委员。1980年，当选为中国作家协会四川分会副主席<ref name=caixin/>。后来退休<ref name=sunshu/>。

1960年代，杨益言将自己收到的《红岩》稿费全部作为党费上交。2001年，杨益言将《红岩》手稿捐赠[[中国现代文学馆|中国现代文学馆]]<ref name=caixin/>。

2017年5月19日，杨益言在重庆病逝，享年92岁<ref name=caixin/>。
