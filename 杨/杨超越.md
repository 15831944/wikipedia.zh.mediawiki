'''杨超越'''（{{bd|1998年|7月31日||}}），出生于[[中华人民共和国|中国]][[江苏省|江苏省]][[盐城市|盐城市]][[大丰区|大丰区]]，中国大陆知名[[女歌手|女歌手]]及[[演员|演员]]。最初为闻澜（上海）文化传媒旗下「[[CH2女团|CH2女团]]」成员。<ref name="CH2">{{cite web |title=CH2高校偶像女团介绍 |url=http://www.ch2.cn/index.php?m=home&c=home&a=about |website=CH2官方网站 |accessdate=2018-08-02}}{{dead link|date=2019-04-22}}</ref>2018年4月参加[[腾讯视频|腾讯视频]]选秀节目《[[创造101|创造101]]》后在网上引发全网热议，最终在第十期以第三名的成绩正式出道，成为限定组合“[[火箭少女101|火箭少女101]]”成员。<ref>{{cite web|url=http://www.sohu.com/a/237478800_114941|title=“火箭少女101”正式成团！孟美岐获C位杨超越第三 |website= 搜狐娱乐|date=2018-06-24}}</ref>

== 经历 ==
===早年生活===
杨超越生长于江苏盐城的一个农村家庭。12岁时，父母离异，便开始与父亲相依为命<ref name="新浪娱乐"/>。初中毕业后开始打工以减轻家庭负担，曾从事过服务员，缝纫工等工作。<ref name="人民网">{{cite news |title=中国版《创造101》单集播放超4亿 普通女孩杨超越励志逆袭 |url=http://sh.people.com.cn/n2/2018/0528/c134768-31636487.html |work=人民网 |date=2018-05-28}}</ref>

===演艺事业===
2016年，杨超越在上海被2000块人民币包吃包住的广告所吸引，报名参加了由闻澜文化举办的全国高校偶像选拔赛，她被主办方选为40强接受为期1个月的集训。<ref>{{cite web |title=“球球宝贝”40强诞生 强IP欲打造中国本土偶像女团 |url=http://games.qq.com/a/20161031/027691.htm |website=腾讯游戏 |date=2016-10-31}}</ref>杨超越在决赛中并没有进入10强，但是因为前10名中有三人退出，于是杨超越因为人气第一而作为替补选手入选，与其他人组成临时组合「球球宝贝」，开始了她的练习生生涯。<ref name="Qieyou">{{cite web |url=http://m.qieyou.com/qqdzz/201611/23246.html |title=2016球球宝贝总决赛圆满落幕 现场有神秘嘉宾降临 |language=Zh-cn |work=切游网 |author=小仓鼠1973 |date=2016-11-27 |deadurl=no |archiveurl=https://web.archive.org/web/20180802041125/http://m.qieyou.com/qqdzz/201611/23246_2.html |archivedate=2019-04-22 |accessdate=2019-04-22 }}</ref><ref name="CH2女团诞生">{{cite web |title=CH2高校偶像女团诞生 互动养成由你参与 |url=http://dj.sina.com.cn/article/fycaafm5455824.shtml |website=新浪电竞 |date=2017-03-07}}</ref>
2017年2月15日，「球球宝贝」中的8名成员正式组成「[[CH2女团|CH2高校偶像女团]]」，并开始发行单曲但此时她任未正式出道。成团以后，除了日常的唱跳训练以外，杨超越也会和团里其他姐妹一起来参加一些游戏的推广、网络平台的直播、街头或年会的走秀表演和小型综艺节目的录制。<ref name="CH2女团诞生"/><ref name="CH2"/>

2018年，参加了[[腾讯视频|腾讯视频]]举办的《[[创造101|创造101]]》。在首期节目中，因称自己是全村的希望，被[[黄子韬|黄子韬]]称为村花。在当期节目中的首次等级评定考核中经导师商议以“有观众缘”的由将其划分为C组。<ref name="中时"/>
2018年6月23日，她在《创造101》总决赛中取得了总排名第三的成绩，并以「[[火箭少女101|火箭少女101]]」组合成员的身份出道。9月23日，发表第一首个人单曲《跟着我一起》，这首歌曲是[[京都念慈庵|京都念慈庵]]的推广曲。<ref>{{cite web |title=中国神药赞助村花杨超越，今年美国流感遭疯抢，一瓶涨十倍卖443元 |url=http://www.aicaijing.com.cn/article/1537752153532/ |website=AI财经社 |accessdate=2019-03-08 |date=2018-09-24}}</ref>同年11月，杨超越参加[[腾讯视频|腾讯视频]]体育竞技真人秀节目《超新星全运会》，最终收获了两枚金牌和一枚银牌。<ref>{{cite news |title=杨超越《全运会》斩获二金一银 回应锦鲤因为大家谦虚 |url=http://ent.ifeng.com/a/20181112/43136223_0.shtml |accessdate=2019-03-08 |work=凤凰网 |date=2018-11-12}}</ref>12月31日，与[[火箭少女101|火箭少女101]]队友参加《湖南卫视跨年演唱会》。

===关于昵称“锦鲤”===
杨超越曾因为唱跳能力相对较差却在总决赛中以第三名的成绩出道，而因此被网友在社交网络上被调侃“转发这个杨超越，不用努力也可以得第三”，借此拿来讽刺她不努力没实力却能走运。<ref>{{cite web |title=王思聪调侃杨超越 配图杨超越"锦鲤照" |url=https://news.china.com/socialgd/10000169/20180809/33528107.html |website=中华网 |date=2018-08-09}}</ref>后来，人们发现杨超越在参加101节目之前吹过的的愿望都慢慢实现了，于是有少量网友感叹其很幸运把她称之为“活的[[锦鲤|锦鲤]]”，与此同时网友们还以杨超越的过往形象为基础制作了一系列以“转发这个杨超越”文字表情包、杨超越与锦鲤为主体的表情包和带杨超越许愿动作的金色光圈动图，该系列图片（或者说表情包）与“转发这个杨超越”偏向嘲讽的语句在中文社交平台中开始传播。<p>2018年8月9日，火箭少女成员的第一，第二名队友宣布退出火箭少女组合，于是网友猜测组合的“C位”{{注|“C”一说为英文单词center的缩写，意为中央、中心。“C位”一般指舞台中央或艺人在宣传海报的中间位置，后被引申为各种场合中最重要、最受关注的位置。}}会顺延到原本排名第三的杨超越。8月11日，在张杰演唱会后的庆功宴上，杨超越抽中了2万元红包，另外，还有一个工作人员在拜完杨超越后抽中了5000人民币，更有一位在手机里存了很久杨超越财神图的工作人员抽中了5万人民币。于是在之前的嘲讽性的图文基础上派生出一系列“转发这个杨超越，不用努力也可以得第三，第一第二还会立刻转学”的表情包，从而引发大量网友在抱着娱乐的心态跟风转发祈求转运。而之后逐渐有部分网友表示自己的愿望真的实现了使其“锦鲤”的形象逐渐加深。<ref>{{cite web |title=转发这个杨超越什么梗？杨超越锦鲤壁纸分享 |url=http://pcedu.pconline.com.cn/1159/11597484.html |website=太平洋电脑网 |date=2018-08-14}}</ref><ref>{{cite web |title=杨超越在张杰鸟巢庆功宴抽中两万元大奖，莫非是天选的锦鲤 |url=https://new.qq.com/omn/20180812/20180812A03D53.html |website=腾讯网 |date=2018-08-12}}</ref> 此后“锦鲤”一词成为她即被称为“村花”后又一被大众广泛认同的昵称。有关她的图片和微博也常常成为网友祈求好运时的转发对象之一。<ref name="TVBS">{{cite web |title=幸運女孩楊超越又中大獎　網友：快轉發「錦鯉」求好運 |url=https://news.tvbs.com.tw/sports/972247 |website=TVBS新聞網 |date=2018-08-12}}</ref><p>2019年《哈哈农夫》第六期中播出，杨超越首度正式回应被叫成锦鲤一事，直言都是巧合，没有科学根据，“抓得住机会 ，你就是自己的锦鲤，抓不住就还需更加努力。不是所有的成功只是靠运气才能达到这一步，不能一概而论的用幸运这个词来抹平别人所有的努力”。<ref>{{cite web |url=http://www.hxnews.com/news/yl/ylxw/201903/12/1721158.shtml |title=杨超越首度回应被叫锦鲤说了什么？杨超越为什么被叫做锦鲤 |language=Zh-cn |work=海峡网 |date=2019-03-12 |accessdate=2019-04-22 }}</ref>
