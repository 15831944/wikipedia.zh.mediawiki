'''杨钟'''（{{bd|1932年|1月|2012年|6月11日|catIdx=}}），[[四川|四川]][[西充|西充]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中国共产党第十二届中央委员会|中国共产党第十二届中央委员会]]候补委员。

==生平==
1932年1月生于四川西充。1950年4月加入[[中国新民主主义青年团|中国新民主主义青年团]]，1952年10月加入[[中国共产党|中国共产党]]。1950年8月后，任中共[[南充|南充]]地委减租退押工作团岳池县坪滩区工作队副队长，白庙区副区长，中共南充地委研究室干事、秘书，中共[[南充市|南充市]]郊区区委书记，中共南充市委农工部副部长、部长，中共[[广安县|广安县]]委副书记，中共[[武胜县|武胜县]]委副书记、代理书记等职。“[[文化大革命|文化大革命]]”中受到批斗和审查。1974年后，任中共广安县委副书记、书记。1979年12月，任[[四川省人民政府|四川省人民政府]]副省长。1982年4月后，任[[中华人民共和国林业部|中华人民共和国林业部]]部长、[[党组|党组]]书记，兼中央绿化委副主任、[[国务院“三北”防护林建设领导小组|国务院“三北”防护林建设领导小组]]常务副组长等职<ref name=shishi>{{Cite web |url = http://cpc.people.com.cn/n/2012/0706/c87393-18455745.html |title =  杨钟同志逝世|author =  |date = 2012-07-06 |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。1987年，因[[大兴安岭火灾|大兴安岭火灾]]而被撤销林业部部长职务<ref>{{cite book|author=全国人大常委会办公厅硏究室|title=论人大及其常委会的监督权|year=1998年|publisher=法律出版社|isbn=9787503603327|pages=69-70}}</ref>。

1988年后，任[[中共中央政策研究室|中央农研室、国务院农研中心]]研究员，农业部党组成员，[[国务院扶贫开发领导小组|国务院贫困地区经济开发领导小组]]副组长兼[[国务院扶贫开发领导小组办公室|办公室]]主任、[[国务院扶贫开发领导小组|国务院扶贫开发领导小组]]成员等职。2003年退休<ref name=shishi/>。

杨钟是[[中国共产党第十二届中央委员会|中国共产党第十二届中央委员会]]候补委员，[[第九届全国政协委员|第九届全国政协委员]]<ref name=shishi/>。
