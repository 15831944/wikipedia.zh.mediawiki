'''杨利伟'''（{{bd|1965年|6月21日|catIdx=Y杨}}），[[辽宁省|辽宁省]][[葫芦岛市|葫芦岛市]][[绥中县|绥中县]][[绥中镇|绥中镇]]人，是[[中华人民共和国|中国]]第一位进入太空的[[太空人|太空人]]，现为[[中国人民解放军航天员大队|中国人民解放军航天员大队]][[特级航天员|特级航天员]]，曾任[[中国载人航天工程办公室|中国载人航天工程办公室]]主任，[[正军职|正军职]]，[[中国人民解放军少将|少将]]军衔。[[中國共產黨|中國共產黨]]黨員，[[中国共产党第十七届中央委员会候补委员|中共第十七届中央委员会候补委员]]。他是中国培养的第一代航天员。2003年10月15日[[北京时间|北京时间]]9时，杨利伟乘由[[长征二号F火箭|长征二号F火箭]]运载的[[神舟五号|神舟五号]]飞船首次进入太空。他和技术专家的创举使得中国成为第三个掌握[[载人航天|载人航天]]技术的国家。

== 简历 ==
父亲杨德元、母亲魏桂兰是绥中县第三初级中学的教师，父亲杨德元后调县土产公司改行任业务股股长。姐姐杨利君比杨利伟大四岁，是绥中县文化旅游局的工作人员；弟弟杨俊伟从部队复员后在当地水利局工作。<ref name="nbwb">[http://www.cnnb.com.cn/gb/node2/newspaper/nbwb/2003/10/node15708/node15718/userobject7ai669783.html 小镇“飞”出华夏英雄———探访杨利伟成长历程，《宁波晚报》 2003年10月16日] {{webarchive|url=https://web.archive.org/web/20031019165313/http://www.cnnb.com.cn/gb/node2/newspaper/nbwb/2003/10/node15708/node15718/userobject7ai669783.html |date=2003年10月19日 }}</ref>

曾在绥中第一幼儿园，1978年考入县第一初中，1981年7月<ref>[http://news.sina.com.cn/c/2003-10-15/23511928053.shtml  东北新闻网：电话连线--杨利伟的亲友在家乡接受记者采访]</ref><ref>[http://www.cnhubei.com/200405/ca460142.htm 《飞天骄子——杨利伟》，作者：周建新（[[葫芦岛市|葫芦岛市]]文联副主席） 中国广播电视出版社，2004年版 ]</ref>进入县二高中就读。在学校里，杨利伟的成绩一般，但是理科成绩优秀；爱好[[游泳|游泳]]、[[滑冰|滑冰]]以及[[田径|田径]]项目。

杨利伟的妻子张玉梅是绥中县[[小庄子乡|小庄子乡]]孤家子村人，1983年从[[绥中县一高中|绥中县一高中]]毕业考入[[锦州师范学院|锦州师范学院]]，毕业后被分配到小庄子乡中学当老师。1989年元旦杨利伟从青岛疗养后回家探亲，经母亲的一位同事介绍二人相识，1990年12月成婚。婚后杨利伟妻子作为空勤家属按照政策参军提干，调到西安工作。1998年随杨利伟调到[[中国人民解放军|解放军]]总装备部载人航天部队军官。两人有一个儿子杨宁康，1995年出生。<ref name="nbwb" />

1983年6月，杨利伟18岁，从绥中第二高中（2003年11月改名为[[绥中县利伟高中|绥中县利伟高中]]）高中二年级三班毕业（当时实行小学中学十年制学制，2003年该中学把二年三班命名为“杨利伟班”），高考考进[[中国人民解放军空军第二飞行基础学校|空军第二飞行基础学校]]，次年进入[[中国人民解放军空军第八飞行学院|空军第八飞行学院]]。1987年毕业，获得学士学位。分配至[[兰州军区空军|兰州军区空军]]强击航空兵第45师第134团（驻甘肃省白银市[[靖远县|靖远县]][[打拉池机场|打拉池机场]]）任[[强五|强五]]飞机飞行员。1988年被授予空军中尉军衔。1992年晋升空军上尉。1992年夏，杨利伟所在部队到新疆某机场驻训，杨利伟驾机超低空飞行时遭遇单发空中停车险情，最终成功爬升高度翻越了天山安全在机场降落，为此荣立三等功。1992年10月，因苏联已经解体，解放军全军精简整编，大量驻三北（东北、华北、西北）部队被撤销解散，空军航空兵第45师也被撤编。空45师所属的航空兵第134团的25名飞行员编入空军航空兵第99团，1993年3月杨利伟也随之到重庆的[[梁平县|梁平县]]的空军机场，改飞[[歼六|歼六]]歼击机。1996年晋升空军少校。

1996年初夏参加中国航天员选拔。1998年1月成为中国首批航天员。在开始航天员生涯前，杨利伟有过1350小时的飞行经历。

1998年成为航天员后，由于中国航天员隶属总装备部编-{制}-，在2000年以前进入总部编-{制}-的军人一律改为陆军改穿陆军军服，故杨利伟军衔由空军少校改为陆军少校。

2000年晋升[[中校|中校]]。2003年因提前一年晋升[[上校|上校]]，晋升军衔命令在执行神舟五号飞行任务前已经下达，但在杨利伟返回地面后才通知他本人。2004年春节前夕破格晋升[[中国人民解放军大校|大校]]，[[2008年7月|2008年7月]]晋升[[中国人民解放军少将|少将]]。2018年1月，当选[[中国人民政治协商会议第十三届全国委员会委员名单|第十三届全国政协委员]]<ref>{{Cite web|url=http://sports.163.com/18/0126/06/D92BRKK400051C89.html|title=中国人民政治协商会议第十三届全国委员会委员名单|accessdate=2018-01-26|last=|work=网易}}</ref>。

2010年5月21日，被任命為[[中國載人航天工程辦公室|中國載人航天工程辦公室]]副主任。2018年4月23日担任中国载人航天工程办公室主任<ref>http://www.cmse.gov.cn/art/2018/4/23/art_19_32447.html</ref><ref>{{Cite web|url=http://china.caixin.com/2018-04-24/101238275.html|title=杨利伟任中国载人航天工程办公室主任  |accessdate=2018-04-25|author=|date=2018-04-24|work=|publisher= 财新网 }}</ref>。

2018年7月12日，中国载人航天工程办公室原副主任[[郝淳|郝淳]]担任中国载人航天工程办公室主任。<ref>http://www.cmse.gov.cn/art/2018/7/12/art_19_32566.html</ref>

2018年11月19日，杨利伟作为中国载人航天工程办公室代表赴莫斯科参加国际空间站20周年纪念活动，作大会发言。<ref>http://www.cmse.gov.cn/art/2018/11/20/art_19_32763.html</ref>

2019年1月17日，香港《明报》声称杨利伟因涉及[[徐才厚|徐才厚]]、[[张阳|张阳]]等人而遭撤职。杨利伟曾于2018年4月23日由中国载人航天工程办公室副主任提升为正主任，由副军升正军。但不到3个月的7月12日，杨利伟便被替换。而在2018年12月12日的一场公开活动中，杨利伟仍佩戴副军职资历章。另外，2018年12月公布的改革开放杰出贡献百人名单中，航天员[[景海鹏|景海鹏]]取代了杨利伟<ref>{{cite web|url=http://news.dwnews.com/china/news/2019-01-16/60112791.html|title=港媒称杨利伟因涉腐败遭撤职 靠一头衔自保|publisher=多维新闻|date=2019-01-17}}</ref>。1月19日，中国载人航天工程办公室通过其官方[[新浪微博|微博]]否定此消息，称杨利伟的职位变动属于达到领导干部任职最高年限后的正常转岗<ref>{{cite web|url=http://3g.163.com/news/article/E5SVC85B00018AOR.html|title=官方：杨利伟同志卸任系正常转岗，个别人在网络上恶意诋毁|publisher=网易新闻|date=2019-01-19}}</ref>。2019年3月召开的全国两会上，杨利伟仍然以全国政协委员的身份参加会议并接受采访，从人民网微博发布的视频<ref>{{Cite web|title=围观群众杨利伟“专业”点评《流浪地球》|url=https://www.weibo.com/2286908003/HjM33opF8?filter=hot&root_comment_id=0&type=comment|work=人民网的微博|accessdate=2019-03-11}}</ref>可见杨利伟仍佩戴少将肩章、副军级别、33年军龄资历章和中央军委装备发展部臂章。

== 航天员生涯 ==
[[File:重庆杜莎夫人蜡像馆-杨利伟.jpg|thumb]]的杨利伟神舟五号航天服造型蜡像。除重慶外，[[香港杜莎夫人蠟像館|香港杜莎夫人蠟像館]]也竖立了楊的蠟像]]
杨利伟于1998年被选为航天员，从那时开始为执行太空任务训练。他在最终的十四名候选人中被选为中国首次载人航天任务的航天员。执行任务后，2003年10月20日，任务前担任[[中校|中校]]的杨利伟被晋升为[[上校|上校]]并佩戴上校军衔肩章出席了神舟五号庆功大会，但随后在春节前夕又迅速晋升[[大校|大校]]军衔。[[中国青年报|中国青年报]]曾提到晋升的决定是在神舟五号任务之前作出的，但杨利伟并不知情。

2003年10月15日[[北京时间|北京时间]]9时的[[酒泉卫星发射中心|酒泉卫星发射中心]]，[[长征二号F火箭|长征二号F火箭]]将杨利伟乘坐的[[神舟五号|神舟五号]]飞船送入太空。在发射前，中国航天员的人选并不公开；杨利伟被选为神舟五号航天员的新闻直到发射前一天才公布给媒体。

在任务过程中，杨利伟按时向地面报告情况，直到返回大气层中断通讯之前都与地面保持通话，告知一切正常。在神舟五号开始第八次环绕地球轨道时，杨利伟告诉地面上的妻子“在太空感觉很好，太空的景色非常美。杨利伟在太空吃了三顿“还挺丰富的”“做得有航天特点的”饭。他在太空的食物包括“一些小[[月饼|月饼]]”、[[宫保鸡丁|宫保鸡丁]]、[[蒜泥白肉|蒜泥白肉]]、[[八宝饭|八宝饭]]、[[绿茶|绿茶]]，“还有一些肉，还有一些[[鱼丸|鱼丸]]子这些东西”。据杨利伟介绍，所有的食物都是“必须一口能吃得下去的，防止残渣去飘”。考虑到神舟五号有可能降落在无人区，他的太空舱里备有枪、刀以及帐篷等基本生存工具。

在任务中途，杨利伟向摄像机镜头展示了[[中华人民共和国国旗|五星红旗]]以及[[联合国旗|联合国旗]]。在环绕地球轨道十四周，航行了超过六十万千米后，神舟五号于北京时间2003年10月16日早晨6时30分在[[内蒙古|内蒙古]]主着陆场成功着陆，实际着陆点与理论着陆点相差4.8千米，返回舱完好无损。杨利伟在降落十五分钟后离开了太空舱，[[温家宝|温家宝]][[中华人民共和国国务院总理|总理]]在现场祝贺。

一直有传闻说[[长城|万里长城]]是唯一在太空可以看见的地面人造物。但是，杨利伟曾在中央电视台的采访中提到他在太空没看到长城<ref>{{cite news|url=http://news.sina.com.cn/c/2003-10-17/0921935621s.shtml|title=“我没有看到我们的长城”|work=[[南方都市报|南方都市报]]|date=2003-10-17|accessdate=2019-03-06|quote=看地球景色非常美丽，但是我没有看到我们的长城。}}</ref>。其实有很多城市在太空都可以辨认出，但是长城由于相对过窄，是无法辨认的。<ref>{{cite news |first= |last= |authorlink= |author=齐柳明 |coauthors= |title=宇航员在太空能否看到长城  |url=http://www.gmw.cn/01gmrb/2004-04/30/content_19819.htm |format= |work=光明日报 |publisher=光明网 |location= |id= |pages= |page= |date=2004-04-30  |accessdate=2013-06-29 |language=zh-hans |quote=不过，当中国首位宇航员杨利伟返回地球后，有记者求证是否看到长城，杨利伟根据自己的亲身经历，实事求是地说没有看到。 |archiveurl= |archivedate= }}</ref>

2004年杨利伟开始在[[清华大学|清华大学]]攻读[[公共管理博士|公共管理博士]]学位。

== 获得的荣誉 ==
[[File:杨利伟航天功勋奖章.jpg|thumb]]杨利伟是第一个进入太空的[[中国公民|中国公民]]。在他之前，[[美国|美国]]有四位华裔航天员曾进入太空，他们分别为[[王赣骏|王赣骏]]（1975年获得美国国籍，1985年执行了{{lang|en|[[STS-51-B|STS-51-B]]}}任务）、[[张福林|张福林]]、[[焦立中|焦立中]]和[[盧傑_(太空人)|卢杰]]。此外，曾执行[[阿波罗8号|阿波罗8号]]的美国宇航员[[威廉·安德斯|威廉·安德斯]]在[[香港|香港]]出生，美国女航天员[[珊农·露茜德|珊农·露茜德]]也出生于中国[[上海|上海]]。

2003年10月31日杨利伟访问了香港，进行了为期六天的访问。11月5日上午，杨利伟又前往澳门继续访问。11月7日，杨利伟在[[人民大会堂|人民大会堂]]从时任[[中央军委|中央军委]]主席的[[江泽民|江泽民]]手中接受了“'''航天英雄'''”[[荣誉称号|荣誉称号]]的证书和奖章。2004年12月9日，杨利伟获[[香港中文大学|香港中文大学]]颁发荣誉理学博士学位。此后，杨利伟仍不时获准出访各地各国，公开出现在各种集会场合。

2007年10月在[[中共十七大|中共十七大]]上當選為[[中共中央候补委员|中共中央候补委员]]。

2007年，[[小行星21064|小行星21064]]以杨利伟命名。

2008年7月22日，被授予[[中国人民解放军少将|中国人民解放军少将]]军衔。

2008年11月17日[[海南省|海南省]]文昌市[[文昌中学|文昌中学]]百年校庆期间，杨利伟受聘文昌中学荣誉校长。[[文昌市|文昌市]]现为[[中国文昌航天发射场|中国文昌航天发射场]]所在地。2018年11月23日，杨利伟受邀出席文昌中学110年校庆，并题字勉励学子<ref>{{cite news |title=海南文昌中学建校110周年 名誉校长杨利伟寄语学子 |url=http://www.chinanews.com/sh/2018/11-23/8684219.shtml |accessdate=2018-12-17 |agency=中国新闻网}}</ref>。

2010年5月10日，出版自傳《天地九重》。由解放軍出版社出版，共20萬字，還有150幅珍貴圖片，憶述他從童年開始的成長歷程，並詳細描述他在太空繞地球飛行14圈的經歷。在該書中楊利偉也披露了自己在返回艙著陸時嘴角受傷的事，而此前的媒體報導中並沒有披露這一事實。<ref>{{cite book |author=楊利偉 |title=天地九重 |publisher=解放軍出版社 |url = http://book.people.com.cn/GB/69399/107429/190411/11599914.html}}在「归途如此惊心动魄」一章節中：「飞船落地时我感到嘴上一麻，心想坏了，肯定磕破了。我把面窗打开，伸手一摸，血流下来。手边没有别的东西，我就开始用里面的布手套在那里擦，擦了半天也没能止住。」</ref>

2017年10月，杨利伟获得由[[联合国教科文组织|联合国教科文组织]]颁发的“[[空间科学奖章|空间科学奖章]]”。首批仅四位航天员获得该奖章，杨利伟成为获奖人之一<ref>http://www.cmse.gov.cn/art/2017/10/31/art_19_31920.html</ref><ref>{{cite web |author=|title=中国航天员杨利伟荣获联合国教科文组织“空间科学奖章”|publisher=人民网 |url = http://world.people.com.cn/n1/2017/1030/c1002-29616760-2.html}}</ref>。

2018年1月25日，和其他11名航天员一同被[[中共中央宣传部|中共中央宣传部]]授予航天员群体“时代楷模”荣誉称号。<ref>http://politics.people.com.cn/n1/2018/0125/c1001-29786869.html</ref>

2019年1月16日，杨利伟被[[中国科协|中国科协]]聘任为“科普中国”形象大使<ref>{{cite web |author=|title=官宣：科普中国聘请六位形象大使|publisher=科普中国网 |url = http://www.kepuchina.cn/more/201901/t20190116_923105.shtml}}</ref>。
