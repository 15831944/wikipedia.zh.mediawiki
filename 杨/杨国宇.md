'''杨国宇'''（{{bd|1914年|9月|2000年|4月27日}}）[[四川省|四川省]][[仪陇县|仪陇县]][[大仪镇_(仪陇县)|大仪镇]]人，[[中国人民解放军开国少将|中国人民解放军开国少将]]<ref name=shishi>杨国宇同志逝世，解放军报2000年5月11日，第04版</ref><ref name=rmw>{{Cite web |url = http://dangshi.people.com.cn/GB/165617/165618/166491/167909/12279204.html |title =杨国宇  |author =  |date =  |publisher =  人民网|language =  |accessdate =2017-08-31  |quote =  }}</ref><ref name=ylx>{{Cite web |url = http://www.yilong.gov.cn/show/2017/07/27/40664.html |title = 杨国宇 |author =  |date = 2017-07-27 |publisher = 仪陇县人民政府 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1933年7月参加[[中国工农红军|中国工农红军]]，1935年5月加入[[中国共产主义青年团|中国共产主义青年团]]，同年8月转入[[中国共产党|中国共产党]]。[[第一次国共内战|第一次国共内战]]时期，历任[[红四方面军|红四方面军]]总部宣传员、[[红三十一军|红三十一军]]九十一师二七四团排长、红四方面军总部军部译电员等职，参加了[[川陕苏区|川陕苏区]]反“围攻”作战，[[田水堡战役|田水堡战役]]、[[山城堡战役|山城堡战役]]等战役以及[[二万五千里长征|二万五千里长征]]<ref name=shishi/><ref name=rmw/><ref name=ylx/>。

1937年[[七七事变|七七事变]]后，援西军改为[[八路军|八路军]]一二九师，他跟随[[刘伯承|刘伯承]]、[[邓小平|邓小平]]首长做机要工作，历任译电组长、科长，随部队参加了[[七亘村战斗|七亘村战斗]]、[[马山村战斗|马山村战斗]]、[[神头岭战斗|神头岭战斗]]、[[香固城战斗|香固城战斗]]、[[响堂铺战斗|响堂铺战斗]]、[[百团大战|百团大战]]以及[[白晋路破袭战役|白晋路破袭战役]]、[[平汉路破袭战役|平汉路破袭战役]]等战役、战斗<ref name=shishi/><ref name=rmw/><ref name=ylx/>。1944年入[[延安|延安]][[中央党校|中央党校]]学习，参加了[[延安整风运动|延安整风运动]]<ref name=ylx/>。

[[抗日战争|抗日战争]]胜利后，历任[[晋冀鲁豫军区|晋冀鲁豫军区]]司令部军政处长，[[中原军区|中原军区]]司令部交通处长、运输司令部副政治委员等职。1949年2月，调任[[中国人民解放军第二野战军|中国人民解放军第二野战军]]第十一军参谋长。他先后参加了[[蒙城战斗|蒙城战斗]]、[[侯马战斗|侯马战斗]]、[[新绛战斗|新绛战斗]]、[[曲沃战斗|曲沃战斗]]等战斗和[[邯郸战役|邯郸战役]]、[[晋南战役|晋南战役]]、[[进军大别山|进军大别山]]、[[淮海战役|淮海战役]]、[[渡江战役|渡江战役]]、进军西南等战役<ref name=shishi/><ref name=rmw/><ref name=ylx/>。

[[中华人民共和国|中华人民共和国]]成立后，1950年3月被派驻[[胡宗南|胡宗南]]部队十八兵团兼任军事代表，负责收编和改造工作。同年10月，调任[[中国人民解放军海军青岛基地|海军青岛基地]]参谋长。1956年9月入[[中国人民解放军军事学院|中国人民解放军军事学院]]海军系学习。1958年奉命组建海军训练基地，10月任该基地副司令员，1959年3月兼任该基地参谋长。[[文化大革命|文化大革命]]期间，1967年5月到1974年10月，被派往[[中华人民共和国第七机械工业部|中华人民共和国第七机械工业部]]军管会任第一副主任，同[[钱学森|钱学森]]一道参与领导航天工业、尖端武器的研制工作。1975年3月被调回海军，历任海军副参谋长、参谋长、副司令员等职。1980年，在中国向南[[太平洋|太平洋]]发射运载火箭试验中，担任副指挥。1985年任[[南极|南极]]考察委员会副主任，参加了[[中国南极长城站|中国南极长城站]]落成典礼<ref name=shishi/><ref name=rmw/><ref name=ylx/>。

1987年离休后，曾任《中国当代海军》主编、《第二野战军史》编辑组副组长、长城学会副会长、中国干部学会副会长以及历史巨片《[[大决战|大决战]]》军事顾问等职<ref name=shishi/><ref name=rmw/><ref name=ylx/>。

1955年，获三级[[八一勋章_(1955年)|八一勋章]]、一级[[解放勋章|解放勋章]]。1961年晋升少将军衔。1988年获一级[[红星功勋荣誉章|红星功勋荣誉章]]。是[[中共七大|中共七大]]代表，第五、六届[[全国人民代表大会|全国人民代表大会]]代表<ref name=shishi/><ref name=rmw/><ref name=ylx/>。

2000年4月27日，杨国宇在北京病逝，享年86岁<ref name=shishi/><ref name=rmw/>。
