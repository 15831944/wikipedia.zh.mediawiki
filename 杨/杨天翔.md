'''杨天翔'''（{{bd|1990年|8月14日}}），中国配音演员。毕业于[[中国农业大学|中国农业大学]]。代表作有电影《[[妖猫传|妖猫传]]》的空海<ref>{{Cite web |url=http://ent.sina.com.cn/m/c/2017-12-25/doc-ifypyuva8761445.shtml |title=《妖猫传》空海中文说得好？其实配音是中国人！|accessdate=2018-03-04 |author= |date=2017-12-25 |work=Koyo |publisher=新浪娱乐|language=Zh-hans }}</ref>，电视剧《[[谈判官|谈判官]]》中的谢晓飞<ref>{{Cite web |url=http://www.sohu.com/a/221520979_131816 |title=《谈判官》杨幂和黄子韬谈恋爱，想看（星星眼）|accessdate=2018-03-05 |author=时尚COSMO |date=2018-02-27 |work= |publisher=搜狐|language=Zh-hans }}</ref>、《[[老九门|老九门]]》的二月红、《[[九州·天空城|九州·天空城]]》的风天逸，《[[古劍奇譚|古劍奇譚]]》的方兰生，动漫《[[狐妖小红娘|狐妖小红娘]]》中的白月初、《[[全职高手|全职高手]]》的蓝河等。现为配音团队[[729声工场|729声工场]]成员<ref>{{Cite web |url=http://ent.sina.com.cn/hr/2017-05-18/doc-ifyfkqiv6520134.shtml |title=男神组团加盟《吴江锦时书》 实力挑战多变角色|accessdate=2018-03-18 |author= |date=2017-05-17 |work=Blue |publisher=新浪娱乐|language=Zh-hans }}</ref>。
