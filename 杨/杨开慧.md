'''杨开慧'''（{{bd|1901年|11月6日|1930年|11月14日|杨开慧}}），号'''霞'''，字'''云锦'''，出生于湖南省[[长沙县|长沙县]][[开慧乡|板仓乡]](现[[开慧乡|开慧乡]])；[[杨昌济|杨昌济]]之女，[[毛泽东|毛泽东]]的第二任妻子，育有三個兒子，分別為[[毛岸英|毛岸英]]、[[毛岸青|毛岸青]]及[[毛岸龍|毛岸龍]]。

==生平==
杨开慧出生不久父亲即出国留学，直到1913年才回国。7岁，杨开慧入读[[长沙|长沙]]第四十初级小学。1918年夏，其父杨昌济应聘为[[北京大学|北京大学]]教授，举家北迁。同年，杨昌济的学生[[毛泽东|毛泽东]]北上北京谋生，经杨昌济介绍，在北京大学图书馆做管理员，并抽空旁听北大课程。在此期间杨被毛泽东追求。1920年1月，父病逝，杨随母亲回湖南长沙，进美国基督教教会開辦的[[福湘女中|福湘女中]]。同年末，与毛泽东结婚，是他的第二任妻子。

1920年，杨开慧加入[[共青团|社会主义青年团]]，1921年加入[[中国共产党|中国共产党]]。曾在[[中共湘区委员会|中共湘区委员会]]负责机要兼交通联络工作。1922年，毛泽东抄写[[唐朝|唐朝]]诗人[[元稹|元稹]]的诗歌《菟丝》{{efn|1=[[唐朝|唐朝]][[元稹|元稹]]《菟丝》，主旨是：无论人和人之间有多么亲近或亲密，独立的人格是非常重要的，不要过于依赖他人，这是自尊，自重，也是自在的前提。 }} 给杨开慧，“人生莫依倚，依倚事不成。君看菟丝蔓，依倚榛和荆。下有狐兔穴，奔走亦纵横。樵童砍将去，柔蔓与之并。”暗喻杨开慧就像菟丝一样粘他，杨开慧随即恼怒说：“你在做事，我做的不是事呀？”杨开慧认为自己既要支持毛泽东的革命工作，又要做贤妻良母，怎么成了“菟丝”来反问毛泽东，毛泽东顿时哑口无言。<ref>{{cite web |url=http://news.163.com/11/0822/11/7C2EFOE000014AED.html|title=《寻找杨开慧》|accessdate=2011-08-22|work=|publisher=网易|language= zh-hans}}</ref>这件事情深深伤害了杨开慧，毛泽东数次向杨开慧解释，杨开慧仍然一直没有释怀。<ref name=" 毛泽东做了什么曾经深深刺伤了杨开慧">{{cite web |url=http://news.ifeng.com/history/zhuanjialunshi/gaohua/detail_2012_01/01/11712272_1.shtml|title=《毛泽东做了什么曾经深深刺伤了杨开慧》|accessdate=2012年1月1日|work=高华|publisher=凤凰网|language=zh-hans}}</ref>1923年，[[毛泽东|毛泽东]]奉[[中共中央|中共中央]]的命令，前往[[上海|上海]]转[[广州|广州]]，这次远行，杨开慧都没有去送行。<ref name=" 毛泽东做了什么曾经深深刺伤了杨开慧" />1923年至1927年随毛泽东在[[上海|上海]]、[[韶山|韶山]]、[[武汉|武汉]]等地开展政治活动。1927年，毛泽东准备组织[[秋收起义|秋收起义]]，后上[[井冈山|井冈山]]，杨开慧携带儿子回长沙板仓乡下老家躲避。1928年毛泽东与[[贺子珍|贺子珍]]在[[井冈山|井冈山]]结婚。1930年，在毛泽东率领中共[[红军|红军]]第二次进攻长沙后，杨开慧在1930年10月被湖南省政府主席[[何鍵|何鍵]]的部下逮捕；最终，她拒绝退党或声明与毛泽东脱离关系，并称“死不足惜，惟愿润之革命早日成功”<ref> 可信性有多高？</ref><ref>{{cite web |url=http://news.ifeng.com/history/zhongguojindaishi/detail_2011_12/16/11376063_0.shtml|title=《杨开慧临终遗言：死不足惜 愿润之革命早日成功》|accessdate=2011年12月16日|work=毛岸青 邵华|publisher=凤凰网|language= zh-hans}}</ref>，传闻[[蔡元培|蔡元培]]向[[何键|何键]]发电要求保释，何回复蔡元培诡称杨已处决，并下令立刻处决，11月14日杨被刑决。当时在[[江西|江西]]指挥[[反围剿|反围剿]]的毛泽东得知后，寄信给杨家：“开慧之死，百身莫赎。”

1982年，[[中共|中共]][[湖南省|湖南省]]文物保护委员会修葺杨家故居，从旧墙夹缝中拆出杨开慧的遗稿，其中有五言长诗《偶感》，分为五章。第一章：“天阴起朔风，浓寒入肌骨。念兹远行人，平波突起伏。足疾已否痊，寒衣是否备？孤眠谁爱护，是否亦凄苦？书信不可通，欲问无人语。恨无双飞翮，飞去见兹人。兹人不得见，惆怅无已时。”{{efn|1=此诗是怀念[[毛泽东|毛泽东]]的，“远行”、“足疾”、“寒衣”、“孤眠”、“书信”，儿女情英雄泪历历可见。 }} 第二章：“家有一秀妹，前兹为我亲。忆昔自京归，同榻共晨昏。偶去三五日，适有冰人至，狂跳盼我归，急切如燃眉。迩后入福湘，伊自往岳州，住岳不数月，仍复归长沙。急切思若人，至复得相亲，当时各陈迹，历历在吾心。风云诚莫测，人情亦复如。追索伤我怀，五内相煎熬，愿将金石意，感尔故人情。朋情至可贵，无可相比伦。”{{efn|1= 此诗是怀念[[杨开秀|杨开秀]]的，杨开秀的丈夫[[箫宗仁|箫宗仁]]是[[湘西|湘西]]中共创建及武装暴动领导人之一。出门，即已经许了人家了，“冰人”即媒人。这里把旧家庭姐妹间的感情生动地描绘出来了。}} 第三章：“沪有一纯姊，思伊伤我怀。能识我衷肠，能识我贤愚。”{{efn|1= “一纯”，[[李一纯|李一纯]]，她是杨开慧的“闺蜜”，两个“能识”排比句生动、鲜活地刻画出小儿女情态。}} 第四章：“城中有友妹，不知伊处居。爱我尚无变，情怀相永依。”{{efn|1=“友妹”，指[[杨开英|杨开英]]。 }} 第五章：“良朋尽如此，数亦何卿卿。念我远方人，复乃数良朋。心怀长郁郁，何日复重逢。”<ref name="杨开慧手稿：我不能忍了，我要跑到他那里去 ">{{cite web |url=http://news.qq.com/a/20110506/000579.htm|title=《杨开慧手稿：我不能忍了，我要跑到他那里去》|accessdate=2011年5月6日|work=朱华|publisher=腾讯网|language=zh-hans}}</ref>

[[張戎|張戎]]所著的《毛澤東：鮮為人知的故事》则认为毛泽东的部队在进攻长沙时可以带上杨开慧或者通知其离开，事实上杨开慧一直没有回毛泽东那里，可以说是环境恶劣和对贺子珍的关系的顾及。

==手迹==

*即使他死了，我的眼淚也要纏住他的屍體。他丟棄我了，一幕一幕地，他一定是丟棄我了

*他是很幸運的，能得到我的愛，我真是非常愛他的喲。不至於丟棄我，他不來信一定有他的道理！父愛是一個謎，他難道不思念他的孩子嗎？我搞不懂他。我要吻他一百遍，他的眼睛，他的嘴，他的臉頰，他的額，他的頭，他是我的人，他是屬於我的，只有母愛是靠得住的。

==失骄杨==
1957年春节，[[李淑一|李淑一]]写信给毛泽东，在信中谈到了毛诗，并附了在1933年听到其夫[[柳直荀|柳直荀]]牺牲时写的《[[菩萨蛮|菩萨蛮]]》。后来毛回信也附了首蝶恋花。
{{quote|我失骄杨君失柳，杨柳轻飏直上重霄九。问讯吴刚何所有，吴刚捧出桂花酒。寂寞嫦娥舒广袖，万里长空且为忠魂舞。忽报人间曾伏虎，泪飞顿作倾盆雨。-毛泽东 《蝶恋花 答李淑一》}}

杨开慧的祖父杨象洛，共有三个儿子即长子杨宏棠（早逝）、次子[[杨昌济|杨宏芬]]（字怀中，号华生，册名昌济）、三子杨宏启（字兆秀，号瑞生）。杨宏启在[[岳州|岳州]]厘金局做事，接济杨昌济留学东洋、西洋。

向振熙，杨开慧生母。1870年生。平江县向家镇山陂村人。1888年与小1岁的表弟杨昌济结婚。生育杨琼（夭折）、[[杨开智|杨开智]]、杨开慧三子女。1920年代与女儿杨开慧、女婿毛泽东住在一起，支持毛泽东的革命工作。杨开慧牺牲后，亲自抚养毛岸英、毛岸青、毛岸龙3个外孙，直至因担心何健的湖南省当局报复仇杀而由党组织安排把3个外孙送至上海的[[毛泽民|毛泽民]]处。1950年春，毛泽东对儿子[[毛岸英|毛岸英]]说：“你外婆是个贤德高尚的人，她全力支持我和你妈妈的事业，我很怀念。”1950年5月，向振熙八十大寿，毛岸英代表毛泽东亲赴长沙看望。毛泽东与江青联袂写了贺寿信：

1962年11月，向振熙92岁高龄在长沙谢世，毛泽东致信杨开智并寄去500元钱作悼仪，信中写道：杨老夫人葬仪“可以与杨开慧同志我亲爱的夫人同穴。我们两家同是一家，是一家，不分彼此”。1969年，向振熙与女儿杨开慧合穴并骨，新建的合葬墓落成，楷书碑文“杨开慧烈士之墓”。其下镌刻毛泽东撰写“杨老夫人与开慧烈士同穴”的题记。

杨昌济子女除夭折者外，有子[[杨开智|杨开智]]；有女'''杨开慧'''。

*'''杨展'''，杨开智与李一纯的女儿，杨开慧的亲侄女。1920年10月生。1937年12月在长沙[[周南女子中学|周南女子中学]]加入中国共产党。1938年赴延安。1939年，从[[陕北公学|陕北公学]]栒邑分校高级班结业后，随[[华北联合大学|华北联合大学]]师生跋山涉水数月，于1939年深秋抵达晋察冀抗日根据地，任校部组织科、教育科干事。1941年秋日军对晋察冀抗日根据地的大扫荡中，于河北省平山县寨北乡滚龙沟西坡与200余同志突围途中坠崖牺牲。安葬于平山县城西北方向40公里的寨北乡主投沟村。毛泽东在《[[论持久战|论持久战]]》提到有个学生从湖南写信：“在乡下一切都感到困难。单独一个人做宣传工作，只好随时随地找人谈话。对象都不是无知无识的愚民，他们多少也懂得一点，他们对我的谈话很有兴趣。可是碰了我那几位亲戚，他们总说：‘中国打不胜，会亡’。讨厌极了。好在他们还不去宣传，不然真糟。农民对他们的信仰当然要大些啊!”(《毛泽东选集》第二卷第441页。) 这是指杨展的来信。1949年8月10日，长沙解放后的第5天，毛泽东致信杨开智：“展儿于8年前在华北抗日战争中为国光荣牺牲，她是百万牺牲者之一，请你们不必悲痛。”<ref>[《毛泽东书信选集》，中央文献出版社2003年版第330页]</ref>

杨昌济亲弟杨宏启有一子三女。子杨开明，三女为杨开云（即杨开仁）、杨开秀、杨开英。

*[[杨开明|杨开明]]，为杨开慧堂弟。1924年毕业于湖南省立甲种工业学校，受毛泽东、杨开慧的影响，参加革命。1928年7月，湖南省委任命其为中共湘赣边界特委书记，与湖南省委代表[[杜修经|杜修经]]贯彻执行湖南省委要求井冈山红军向湘南发展的命令，造成湘赣边界和湘南两地[[八月失败|八月失败]]负有主要责任。1929年9月，任中共湖北省委鄂西特派员。1930年除夕夜，在[[汉口|汉口]]被捕，正月初一押回长沙，严刑逼供下坚强不屈。1930年2月22日在长沙识字岭壮烈就义，年仅25岁。有《开明烈士日记》

*'''杨开英'''，为杨开慧堂妹，二人长像非常相似。1930年10月杨开慧在家乡被捕时，杨开英、毛岸英、杨开慧的保姆等人一同被捕。后杨开慧就义，其他人获释。建国后，杨开英在大连的育英中学做副教导主任，1956年12月16日毛泽东致信并寄钱给杨开英治疗肺病。后杨开英调北京第四十三中学任教导主任。1960年4月25日，毛泽东致信杨开英托其为向振熙寿辰送上200元寿礼。
