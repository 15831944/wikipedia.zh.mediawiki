'''杨春光'''（{{bd|1954年|2月||}}）[[山东|山东]][[苍山|苍山]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员<ref name=jj>{{Cite web |url = http://district.ce.cn/newarea/sddy/201512/14/t20151214_7549169.shtml |title =国家公务员局原副局长杨春光任中国海外控股集团党委书记  |author =  |date =2015-12-14  |publisher =  中国经济网|language =  |accessdate =  |quote =  }}</ref><ref name=cpc>{{Cite web |url =http://cpc.people.com.cn/GB/64162/123659/123779/7320299.html  |title =  杨春光同志简历|author =  |date =  |publisher =人民网  |language =  |accessdate =  2018-03-20|quote =  }}</ref>。

==生平==
1969年3月参加工作。1969年到1976年，为山东省临沂沂蒙冶炼厂工人，临沂新华印刷厂工人。1976年到1978年，在[[山东大学|山东大学]]中文系学习，大学普通班学历。1978年5月加入[[中国共产党|中国共产党]]。1978年到1984年，历任[[山东省|山东省]][[济南日报|济南日报]]社校对、记者、编辑<ref name=jj/><ref name=cpc/>。

1984年到1986年，任[[中共山东省委|中共山东省委]]组织部办公室干部。1986年到1989年，任中共山东省委组织部正科级巡视员。1989年到1990年，任中共山东省委组织部副处级巡视员。1990年到1995年，任中共山东省委高校工委组织处副处长，中共山东省委高校工委机关党委书记（正处级）。其间，1994年到1995年在[[中组部|中组部]]组织局帮助工作<ref name=jj/><ref name=cpc/>。

1995年到1996年，任中组部组织局六处干部、正处级调研员。1996年到1998年，挂职担任[[新疆维吾尔族自治区|新疆维吾尔族自治区]][[巴音郭楞蒙古自治州|巴音郭楞蒙古自治州]]党委副书记。1998年到2000年，任[[中共宁夏回族自治区党委|中共宁夏回族自治区党委]]组织部副部长。2000年到2002年，任中共[[石嘴山市|石嘴山市]]委副书记。2002年到2003年，任中共石嘴山市委书记。2003年到2007年，任中共石嘴山市委书记，市人大常委会主任。2007年3月，任中共宁夏回族自治区党委宣传部部长。2007年6月到2012年1月，任中共宁夏回族自治区党委常委、宣传部部长<ref name=jj/><ref name=cpc/>。

2012年1月到2014年9月，任[[国家公务员局|国家公务员局]]党组成员、副局长（副部长级）。2015年11月，任[[中国海外控股集团|中国海外控股集团]]党委书记。他是[[第十二届全国政协委员|第十二届全国政协委员]]<ref name=jj/><ref name=cpc/>。
