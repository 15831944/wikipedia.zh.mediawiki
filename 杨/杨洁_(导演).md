'''杨洁'''（{{bd|1929年|4月7日|2017年|4月15日|catIdx=Y}}）<ref name=qingdao>{{Cite web |url=http://zaobao.qingdaonews.com/html/2009-05/05/content_2100045.htm |title=《西游记》导演首播，青岛早报2009年5月5日，第06版 |access-date=2010-11-10 |archive-url=https://web.archive.org/web/20160304080253/http://zaobao.qingdaonews.com/html/2009-05/05/content_2100045.htm |archive-date=2016-03-04 |dead-url=yes }}</ref><ref>{{cite web|url=http://ent.ifeng.com/a/20170417/42910663_0.shtml|title=86版《西游记》导演杨洁逝世 十天前才过完88岁生日|publisher=凤凰网|date=2017-04-17}}</ref>，女，汉族，出生于[[中国|中国]][[四川省|四川省]][[营山县|营山县]]，中國大陸第一代电视艺术家，[[中国电视剧制作中心|中国电视剧制作中心]][[制片人|制片人]]、[[导演|导演]]。<ref>[http://ent.sina.com.cn/s/2010-03-15/14272897761.shtml 资料：导演杨洁个人简介，新浪娱乐，2010年3月15日]</ref>

==生平==
杨洁是[[杨伯恺|杨伯恺]]、[[危淑元|危淑元]]夫妇的长女。杨伯恺是[[中国共产党|中国共产党]]党员，后又参加[[中国民主同盟|中国民主同盟]]，任中国民主同盟中央委员。1947年6月2日被[[国民政府|国民政府]]逮捕，关押于[[成都|成都]]将军衙门的[[四川省特种委员会|四川省特种委员会]]监狱，1949年12月8日和其他[[中共|中共]]、[[民盟|民盟]]、[[民革|民革]]等党派的政治犯一起被屠杀于成都十二桥，即[[成都十二桥惨案|成都十二桥惨案]]<ref>[http://www.zgdazxw.com.cn/NewsView.asp?ID=7053 胡剑，铁骨铮铮的杨伯恺，档案大观2009年7月24日425期，第2版]</ref>。

杨洁从小就喜欢看电影，还在《[[华西日报|华西日报]]》上发表过小说《禁闭室的女性》和《他有什么罪》<ref name=jiumeng/>。1945年被父亲杨伯恺委托[[張友漁_(法學家)|张友渔]]送到[[陕甘宁边区|陕甘宁边区]]首府[[延安|延安]]<ref>[http://book.sina.com.cn/longbook/1074234602_houyuan/53.shtml 六小龄童，杨洁“用兵”如神，载六小龄童·猴缘，北京：京华出版社，2004年]</ref>。后杨洁转赴[[张家口|张家口]]，进入[[华北联合大学|华北联合大学]]，并参加了[[中国人民解放军|中国人民解放军]]，被组织分配到文工队当演员<ref name=jiumeng>[http://www.qikan.com.cn/Article/bgwx/bgwx200802/bgwx20080201.html 杨洁，旧梦重温，报告文学2008年第2期]</ref>。此后她先后在晋察冀新华广播电台、陕北新华广播电台、济南广播电台担任播音员，曾负责济南解放的播音任务。[[中国人民解放军青岛市军事管制委员会|青岛市军管会]]成立“胶研会”后，组建了“广播电台班组”，杨洁被调入该班组。1949年6月2日[[中国人民解放军|中国人民解放军]]攻占[[青岛市|青岛市]]，当天即将原青岛广播电台更名为[[青岛人民广播电台|青岛人民广播电台]]，杨洁担任该台播音员。1949年6月2日晚，杨洁向青岛市人民发出了第一条“青岛解放”的消息<ref name=qingdao/>。

1953年底，杨洁因[[肺病|肺病]]住进医院。1954年她从青岛人民广播电台调至设在[[北京|北京]]的[[中央人民广播电台|中央人民广播电台]]，既为养病，也为照顾住在北京的母亲<ref name=qingdao/>。

1958年，杨洁由中央人民广播电台调到北京电视台（即[[中国中央电视台|中国中央电视台]]的前身）。自1961年起，她专门负责电视戏曲节目。她执导的[[京剧|京剧]]《[[香罗帕|香罗帕]]》曾于1981年被评选为全国优秀电视艺术加工文艺节目。1976年曾参加过[[中国中央电视台|中国中央电视台]]参与的[[为毛主席录制传统戏曲曲艺节目|为毛主席录制传统戏曲曲艺节目]]行动中的录像部分，在[[湖南省|湖南省]]拍摄过[[湘剧|湘剧]]传统戏《[[追鱼记|追鱼记]]》等上百个湘剧剧目<ref>[http://cpc.people.com.cn/GB/68742/106364/106365/7535018.html 黄禹康，“观世音”与毛泽东的忘年交——记著名湘剧表演艺术家左大玢，于2010-11-10查阅]</ref><ref name=jiumeng/>。

1980年，杨洁开始导演[[电视剧|电视剧]]。1981年，中央电视台副台长[[洪民生|洪民生]]在文艺部会议上宣布由[[王扶林|王扶林]]、杨洁分别负责《[[红楼梦_(1987年电视剧)|红楼梦]]》和《西游记》的开拍计划。此后杨洁完成了电视连续剧《[[西游记_(1986年电视剧)|西游记]]》25集的拍摄。该剧在1980年代由中国中央电视台分集陆续播出，并在1988年全播出，获得空前成功。但是，杨洁为了此剧的顺利拍摄，多次顶撞中央电视台领导，中央电视台乃在此后很长时间排挤杨洁。並因為出國表演一事與劇組四人關係陷入冰點，1994年，劇組希望能夠將當時的西遊記重拍完整，故1998年至1999年，杨洁又导演了《西游记》第二部<ref name=kongli>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1664338 |title = 杨洁导演最喜欢的不是《西游记》，而是《司马迁》 |author = 孔鲤 |date =  2017-04-17 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

2017年4月15日，杨洁因[[心脏病|心脏病]]在[[北京|北京]]逝世，享年88岁<ref>{{Cite web |url = http://news.163.com/17/0417/11/CI7JJNLF00018AOR.html |title = 86版西游记导演杨洁因心脏病在京逝世 |author =  |date = 2017-04-17 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。
