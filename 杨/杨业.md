'''杨业'''（{{bd|923年||986年||}}），又名'''繼業'''，初名'''重貴'''，[[并州|并州]][[太原|太原]]（今[[山西|山西]][[太原|太原]]）人，[[北宋|北宋]]抗辽名将。以[[云州|云州]][[观察使|观察使]]判[[代州|代州]]，在[[雍熙北伐|雍熙北伐]]中力尽被俘，绝食殉国，[[追赠|追赠]][[检校太尉|检校太尉]]、[[大同军|大同军]][[节度使|节度使]]；与其子[[杨延昭|杨延昭]]、孙[[杨文广|杨文广]]三代并称名将，号为“[[杨家将|杨家将]]”。

== 生平 ==

杨业本名重贵，[[并州|并州]][[太原|太原]]（今[[山西|山西]][[太原|太原]]）人<ref>《[[宋史·杨业传|宋史·杨业传]]》：“杨业，并州太原人。”</ref>，[[北宋|北宋]]抗辽名将。年少时便擅长骑射，立志成为名将<ref>《[[宋史|宋史]]·杨业传》：“业幼倜傥任侠，善骑射，好畋猎，所获倍于人。尝谓其徒曰：‘我他日为将用兵，亦犹用鹰犬逐雉兔尔。’”</ref>。杨业少时倜傥任侠，善于骑射，喜好打猎，猎获总比他人多。读书不多，但忠烈武勇，甚有智谋。北汉建立后，年仅弱冠的杨业即追随[[北汉|北汉]]世祖[[刘崇|刘崇]]，任保卫指挥使，以骁勇远近闻名。后屡立战功，迁升建雄军节度使，国人号为"无敌"。[[宋太宗|宋太宗]]素闻杨业之名，于北汉灭亡后，遣使召见杨业，授右领军卫大将军。累迁[[代州|代州]][[刺史|刺史]]兼三交驻泊兵马部署。<ref>《[[旧五代史|旧五代史]]·汉高祖纪》：“（后晋天福）六年七月，授北京留守、河东节度使。”《[[旧五代史|旧五代史]]·僭伪列传》：“刘崇，太原人，汉高祖之从弟也。……汉祖镇并汾，奏为河东步军都指挥使。逾年，授麟州刺史。”</ref>，授保卫指挥使，以骁勇闻名<ref>《[[宋史|宋史]]·杨业传》：“弱冠事刘崇，为保卫指挥使，以骁勇闻。”</ref>。[[后周|后周]][[广顺|广顺]]元年（951年），[[后汉|后汉]][[北京留守|北京留守]]兼[[中书令|中书令]]<ref>《[[旧五代史|旧五代史]]·隐帝纪》：“（后汉乾祐元年丙辰）北京留守、检校太尉、同平章事刘崇，领宋州节度使兼侍卫亲军马步军都指挥使、检校太尉、同平章事史宏肇，并加检校太师、兼侍中。……（二年九月乙卯）北京刘崇加兼中书令。”</ref>[[劉旻|刘崇]]在[[太原|太原]]称帝，建立[[北汉|北汉]]。杨业效力[[北汉|北汉]]凡三十年，赐姓刘氏，改名继业<ref>《[[续资治通鉴长编|续资治通鉴长编]]》：“（宋太祖开宝元年）继业本名重贵，姓杨氏，重勋之兄。幼事北汉世祖，遂更赐以姓名。”</ref>，官至[[侍卫亲军|侍卫亲军]][[马步军都虞候|马步军都虞候]]<ref>《[[续资治通鉴长编|续资治通鉴长编]]》：“（开宝元年）继元始立，宋师已入其境，乃亟遣使上表于辽，且请兵为授。又遣侍卫都虞候刘继业、马进珂领军扼团柏谷，以将作监马峰为枢密使，监其军。”</ref>，领[[建雄军|建雄军]][[节度使|节度使]]<ref>《[[宋史|宋史]]·杨业传》：“累迁至建雄军节度使，屡立战功，所向克捷，国人号为‘无敌’。”</ref>，被称为“太原骁将”<ref>《宋史·党进传》：“开宝二年，太祖师临晋阳、置砦四面，命进主其东偏。师未成列，太原骁将杨业领突骑数百来犯。”</ref>。[[北宋|北宋]][[太平兴国|太平兴国]]四年（979年），[[宋太宗|宋太宗]]平定[[北汉|北汉]]，素闻杨业骁勇，便遣使召见，温言抚慰，命其復姓杨氏，单名业，授[[左领军卫大将军|左领军卫大将军]]，还朝后授[[郑州|郑州]][[防御使|防御使]]<ref>《[[续资治通鉴长编|续资治通鉴长编]]》：“（太平兴国四年）初，刘继业为继元捍太原城，甚骁勇。及继元降，继业犹据城苦战。上素知其勇，欲生致之，令中使谕继元俾招继业。继元遣所亲信往，继业乃北面再拜，大恸，释甲来见。上喜，慰抚之甚厚，復姓杨氏，止名业，寻授左领军卫大将军。丁巳，以业为郑州防御使。”</ref>。太平兴国五年三月，杨业在[[雁门关|雁门关]]大破辽军，威震契丹。

[[宋太宗|宋太宗]]认为杨业熟悉边防，不久即命杨业知[[代州|代州]]兼[[三交|三交]][[驻泊|驻泊]][[兵马部署|兵马部署]]，临行前赏赐甚厚<ref>《[[续资治通鉴长编|续资治通鉴长编]]》：“（太平兴国四年）上以郑州防御使杨业老于边事，洞晓敌情。癸巳，命业知代州兼三交驻泊兵马部署。上密封囊装，赐予甚厚。”</ref>。次年（980年），[[宣徽南院使|宣徽南院使]]、[[开府仪同三司|开府仪同三司]]、兼[[三交都部署|三交都部署]][[潘美|潘美]]<ref>《[[宋史|宋史]]·潘美传》：“太平兴国初，改（宣徽）南院使。三年，加开府仪同三司。四年，命将征太原……命兼三交都部署，留屯以捍北边。”</ref>自三交口巡抚至代州<ref>《[[宋史|宋史]]·潘美传》：“美尝巡抚至代州，既秣马蓐食，俄而辽兵万骑来寇，近塞，美誓众衔枚奋击，大破之，封代国公。”</ref>，遇[[辽国|辽国]][[应州|应州]][[彰国军|彰国军]][[节度使|节度使]]、[[驸马都尉|驸马都尉]]、[[侍中|侍中]][[萧咄李|萧咄李]]率部南侵<ref>《[[宋史|宋史]]·李重诲传》：“重诲尝为其应州马步军都指挥使，太平兴国五年，潘美出师御寇，重诲从其节度使萧咄李迎战于代州北岭，大败。美斩咄李，擒重诲以献。”</ref>，即命杨业率麾下数百骑兵抄小路从背后夹攻，大败[[辽军|辽军]]，[[萧咄李|萧咄李]]死于阵中，[[应州|应州]][[马步军都指挥使|马步军都指挥使]][[李重诲|李重诲]]被生擒，缴获大量铠甲马匹<ref>《[[续资治通鉴长编|续资治通鉴长编]]》：“（太平兴国五年三月）癸巳，潘美言自三交口巡抚至代州，会敌十万众侵雁门，令杨业领麾下数百骑自西陉出，由小陉至雁门北口南向与美合击之，敌众大败，杀其节度使、驸马、侍中萧咄李，生擒马步军都指挥使李重诲，获铠甲革马甚众。”</ref>。[[潘美|潘美]]以功封[[代国公|代国公]]，杨业升[[云州|云州]][[观察使|观察使]]，仍判[[代州|代州]]事。杨业一介降将，未及一年便立功边境，遭到[[潘美|潘美]]嫉恨，秘密向[[宋太宗|宋太宗]]上书进谗。[[宋太宗|宋太宗]]眷遇正隆，将谗书封送杨业，以示信任<ref>《[[续资治通鉴|续资治通鉴]]》：“（太平兴国五年十二月）丁丑，以杨业领云州观察使，知代州事。业自雁门之役，辽人畏之，每望见业旗，即引去。主将屯边者多嫉之，或潜上谤书，斥言其短；帝皆不问，封其奏以付业。”</ref>。

[[雍熙|雍熙]]三年（986年），[[宋太宗|宋太宗]]发兵三路进行北伐，[[检校太师|检校太师]]、[[忠武军|忠武军]][[节度使|节度使]][[潘美|潘美]]为[[云应路行营都部署|云应路行营都部署]]（主将），[[云州|云州]][[观察使|观察使]]杨业为副都部署（副将），从西路连下云、应、寰、朔四州，因主力东路军战败奉命退还[[代州|代州]]<ref>《[[宋史|宋史]]·杨业传》：“（宋太宗）雍熙三年，大兵北征，以忠武军节度使潘美为云应路行营都部署，命业副之，以西上阁门使、蔚州刺史王侁，军器库使、顺州团练使刘文裕护其军。诸军连拔云、应、寰、朔四州，师次桑乾河，会曹彬之师不利，诸路班师，美等归代州。”</ref>。不久，[[宋太宗|宋太宗]]命[[潘美|潘美]]出兵掩护四州百姓内迁，时[[辽国|辽国]][[北院枢密使|北院枢密使]][[耶律斜轸|耶律斜轸]]、[[彰国军节度使|彰国军节度使]][[萧挞凛|萧挞凛]]已越过云、应二州攻陷[[寰州|寰州]]，直逼[[朔州|朔州]]。出兵前，杨业建议说：“现在辽军锐气正盛，最好避免与之决战。朝廷目的在于取得数州百姓，我们应当从大石路出兵应州（示意切断辽军后路），出发之前派人悄悄通知云州百姓先南下（到应州），如果辽军回应州保护后路，则朔州百姓立即南下石碣谷（撤往代州），派强弩千人守住谷口，再派骑兵作为机动部队居中救援，（而我军在应州与辽军相持，辽军也不敢越过我军袭取云州、应州撤退的百姓）这是保护三州百姓的万全之策。”[[西上阁门使|西上阁门使]]、[[蔚州|蔚州]][[刺史|刺史]]、[[云应路行营兵马都监|云应路行营兵马都监]]（监军）[[王侁|王侁]]立即反对，要求从西边往[[朔州|朔州]]寻求与辽军进行主力决战，另一[[监军|监军]]（史书未载其差遣，可能是云应路行营兵马副都监）[[军器库使|军器库使]]、[[顺州|顺州]][[团练使|团练使]][[刘文裕|刘文裕]]也赞成王侁的提议。杨业力执不可，受到王侁讥讽：“君侯素号无敌，如今逗留不进，莫非有他意？”杨业本是降将，最忌讳别人说他有二心，为明心志自请为先锋出战<ref>《[[宋史|宋史]]·杨业传》：“未几，诏迁四州之民于内地，令美等以所部之兵护之。时契丹国母萧氏与其大臣耶律汉宁、南北皮室及五押惕隐领众十余万，复陷寰州。业谓美等曰：‘今辽兵益盛，不可与战。朝廷止令取数州之民，但领兵出大石路，先遣人密告云、朔州守将，俟大军离代州日，令云州之众先出。我师次应州，契丹必来拒，即令朔州民出城，直入石碣谷。遣强弩千人列于谷口，以骑士援于中路，则三州之众保万全矣。’侁沮其议曰：‘领数万精兵而畏懦如此。但趋雁门北川中，鼓行而往。’文裕亦赞成之。业曰：‘不可，此必败之势也。’侁曰：‘君侯素号无敌，今见敌逗挠不战，得非有他志乎？’业曰：‘业非避死，盖时有未利，徒令杀伤士卒而功不立。今君责业以不死，当为诸公先。’”</ref>。

宋军抵达[[朔州|朔州]][[陈家谷|陈家谷]]，辽军也已赶到附近。杨业敦请主将[[潘美|潘美]]在陈家谷口布阵，自己作为先锋继续前进<ref>《宋史·杨业传》：“将行，泣谓美曰：‘此行必不利。业，太原降将，分当死。上不杀，宠以连帅，授之兵柄。非纵敌不击，盖伺其便，将立尺寸功以报国恩。今诸君责业以避敌，业当先死于敌。’因指陈家谷口曰：‘诸君于此张步兵强弩，为左右翼以援，俟业转战至此，即以步兵夹击救之，不然，无遗类矣。’”</ref>。辽军主将[[山西诸路兵马都统|山西诸路兵马都统]][[耶律斜轸|耶律斜轸]]派副将[[山西兵马副部署|山西兵马副部署]][[萧挞凛|萧挞凛]]设伏，自己与杨业交战后假装败退，杨业中伏大败<ref>《辽史·耶律斜轸传》：“太后亲帅师救燕，以斜轸为山西路兵马都统。……斜轸闻继业出兵，令萧挞凛伏兵于路。明旦，继业兵至，斜轸拥众为战势。继业麾帜而前，斜轸佯退。伏兵发，斜轸进攻，继业败走，至狼牙村，众军皆溃。”</ref>。监军王侁派人在高台眺望战况，看到杨业追击辽军，便率兵前去争功，潘美不能阻止王侁，于是一起沿着河边行军二十里，得知杨业战败后又立即撤退而不救援<ref>《宋史·杨业传》：“侁使人登托逻台望之，以为契丹败走，欲争其功，即领兵离谷口。美不能制，乃缘交河西南行二十里。俄闻业败，即麾兵却走。”</ref>。杨业杀出重围后且战且退，到达陈家谷口只剩[[淄州|淄州]][[刺史|刺史]][[王贵|王贵]]和儿子[[杨延玉|杨延玉]]等残部百余人，发现[[潘美|潘美]]已经撤退，伤心欲绝，对部下说：“你们都有父母妻儿，俱死无益，可归报天子。”但大家都慷慨赴死，无人逃难<ref>《[[宋史|宋史]]·杨业传》：“朔州之败，麾下尚百余人，业谓曰：‘汝等各有父母妻子，与我俱死，无益也，可走还，报天子。’众皆感泣不肯去。淄州刺史王贵杀数十人，矢尽遂死，余亦死，无一生还者”</ref>。杨业藏匿于树林中，被[[右皮室|右皮室]]（辽军精锐部队名）[[详稳|详稳]]（部队主官名）[[耶律奚底|耶律奚底]]射中，坠马被擒。辽军主将[[耶律斜轸|耶律斜轸]]责骂杨业擅开边衅，杨业但称有死而已，又叹息说：“受皇帝厚恩，本想立功边境以报答，却受奸臣逼迫而战败，无颜苟活人世。”于是绝食三天而死<ref>《[[宋史|宋史]]·杨业传》：“业力战，自午至暮，果至谷口。望见无人，即拊膺大恸，再率帐下士力战，身被数十创，士卒殆尽，业犹手刃数十百人。马重伤不能进，遂为契丹所擒，其子延玉亦没焉。业因太息曰：‘上遇我厚，期讨贼捍边以报，而反为奸臣所迫，致王师败绩，何面目求活耶？’乃不食，三日死。”</ref>。

[[宋太宗|宋太宗]]闻讯震悼，将[[潘美|潘美]]削秩三等，责授[[检校太保|检校太保]]<ref>《宋史·潘美传》：“会辽兵奄至，战于陈家谷口，不利，骁将杨业死之。美坐削秩三等，责授检校太保。明年，复检校太师。”</ref>；[[王侁|王侁]]、[[刘文裕|刘文裕]]除名（削去官籍），王侁[[配隶|配隶]][[金州|金州]]，刘文裕配隶[[登州|登州]]；赠杨业[[检校太尉|检校太尉]]、[[大同军|大同军]][[节度使|节度使]]，赐其家布帛千疋，粟千硕<ref>《宋史·杨业传》：“帝闻之，痛惜甚，俄下诏曰：‘执干戈而卫社稷，闻鼓鼙而思将帅。尽力死敌，立节迈伦，不有追崇，曷彰义烈！故云州观察使杨业诚坚金石，气激风云。挺陇上之雄才，本山西之茂族。自委戎乘，式资战功。方提貔虎之师，以效边陲之用。而群帅败约，援兵不前。独以孤军，陷于沙漠；劲果猋厉，有死不回。求之古人，何以加此！是用特举徽典，以旌遗忠。魂而有灵，知我深意。可赠太尉、大同军节度，赐其家布帛千匹、粟千石。大将军潘美降三官，监军王侁除名、隶金州，刘文裕除名、隶登州。’”案五代两宋武将升迁之序，一般初授节度使加检校官，不得便遽为真太尉。</ref>；以其子[[供奉官|供奉官]][[杨延昭|杨延昭]]（时名杨延朗）为[[崇仪副使|崇仪副使]]，[[殿直|殿直]][[杨延浦|杨延浦]]、[[杨延训|杨延训]]为供奉官，[[杨延瑰|杨延瑰]]、[[杨延贵|杨延贵]]、[[杨延彬|杨延彬]]为殿直<ref>《宋史·杨业传》：“业既没，朝廷录其子供奉官延朗为崇仪副使，次子殿直延浦、延训并为供奉官，延瑰、延贵、延彬并为殿直。”</ref>。

== 家庭 ==
妻：史书未载。自清以来，方志或-{云}-[[府州|府州]][[折氏|折氏]]。如[[清朝|清]][[光緒|光緒]]年間續修《[[岢嵐州|岢嵐州]]志》卷九《人物·節婦》條記載：「楊業『娶折德扆女』，『折性敏慧，嘗佐業立戰功，號楊無敵』」。案《[[宋史|宋史]]·折德扆傳》，折德扆父「[[折從阮|折從阮]]自[[晉|晉]][[漢|漢]]以來獨據[[府州|府州]]」。折家歸附[[後周|後周]]之後，「父子俱領節鎮，時人榮之」。折、楊兩家同为地方豪強，麟州、府州又相邻，两家结姻亲之好并非全无可能。

子：[[杨延昭|杨延昭]]、[[杨延浦|杨延浦]]、[[杨延训|杨延训]]、[[杨延瑰|杨延瑰]]、[[杨延贵|杨延贵]]、[[杨延彬|杨延彬]]、[[杨延玉|杨延玉]]。《宋史·楊業傳》记载为七子，《續資治通鑒長編》言楊業僅五子，而《東都事略》中則只提到楊延昭一個兒子。

== 历史影响 ==

楊業与其子[[楊延昭|楊延昭]]、孙[[楊文廣|楊文廣]]三代为将，颇負盛名。北宋著名文學家[[歐陽修|歐陽修]]，稱讚楊業、楊延昭「父子皆名將，其智勇號稱無敵，至今天下之士至於裡兒野豎，皆能道之。」宋[[元|元]]的民間藝人把楊家將的故事編成戲曲，搬上舞台。到了[[明代|明代]]，民間又把他們的故事編成《[[楊家將演義|楊家將演義]]》、《[[楊家將傳|楊家將傳]]》，用小說評書的形式在社會民間廣泛傳播。
