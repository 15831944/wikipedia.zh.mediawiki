'''杨凯莉'''（{{bd|1997年|7月11日}}），网名'''莉哥'''<ref>{{cite web |author1=羊驼先生 |title=“莉哥”杨凯莉，走红后出场费一次几十万，以前的职业这么苦逼？ |url=http://wemedia.ifeng.com/70355700/wemedia.shtml |website=凤凰网 |accessdate=2018-09-19 |archive-url=https://web.archive.org/web/20180919094334/http://wemedia.ifeng.com/70355700/wemedia.shtml |archive-date=2018-09-19 |dead-url=no }}</ref>，中国内地女歌手、网络主播。<ref name="杨凯莉乙" /><ref name="杨凯莉甲" />凭借一首《让我做你的眼睛》走红网络。<ref>{{cite web |title=抖音莉哥成新晋烟嗓女神？一首《让我做你的眼睛》惊艳观众 |url=http://games.ifeng.com/a/20180903/45150068_0.shtml |website=凤凰网 |accessdate=2018-09-19 |archive-url=https://web.archive.org/web/20180919061811/http://games.ifeng.com/a/20180903/45150068_0.shtml |archive-date=2018-09-19 |dead-url=no }}</ref>

== 演艺经历 ==
2016年，毕业于丰台职高的杨凯莉成为一名淘宝美工，主要负责P图工作。但因工作压力大，她转行做一名网络红人，先后入驻了[[快手|快手]]、[[抖音短视频|抖音]]视频平台，以分享小视频、发起网络直播为主。<ref>{{cite web |title=她从草根变身家5千万女主播 因戏唱国歌被拘 恐赔虎牙2.6亿 |url=http://www.aicaijing.com.cn/article/1539651782661/ |website=AI财经社 |accessdate=2019-02-12}}</ref>

2018年5月底，杨凯莉先前在[[抖音短视频|抖音]]发布的一个自己演唱歌曲《让我做你的眼睛》{{NoteTag|此歌曲改编自另一首歌曲《唱腔另类》}}的视频在网络上走红。她的知名度得到了较大的提高，微博、抖音粉丝也大幅度增长。<ref>{{cite web |title=抖音莉哥在什么平台哪里直播 莉哥因一首唱腔另类歌曲火了 |url=http://www.mnw.cn/keji/internet/2004100.html |website=闽南网 |accessdate=2018-12-08 |archive-url=https://web.archive.org/web/20181209123805/http://www.mnw.cn/keji/internet/2004100.html |archive-date=2018-12-09 |dead-url=no }}</ref>

2018年6月7日，推出首张个人EP《让我做你的眼睛》<ref>{{cite web |title=专辑：让我做你的眼睛 |url=https://y.qq.com/n/yqq/album/003krDdQ22WvtH.html#stat=y_new.singer.index.album_name |website=QQ音乐 |accessdate=2018-09-18 |archive-url=https://web.archive.org/web/20180918123641/https://y.qq.com/n/yqq/album/003krDdQ22WvtH.html#stat=y_new.singer.index.album_name#stat=y_new.singer.index.album_name |archive-date=2018-09-18 |dead-url=yes }}</ref>；
7月11日，发行歌曲《念念不忘》。<ref>{{cite web |title=专辑：念念不忘 |url=https://y.qq.com/n/yqq/album/000wUdYP0c7uXo.html#stat=y_new.singer.index.album_name |website=QQ音乐 |accessdate=2018-09-18 |archive-url=https://web.archive.org/web/20180918160530/https://y.qq.com/n/yqq/album/000wUdYP0c7uXo.html#stat=y_new.singer.index.album_name#stat=y_new.singer.index.album_name |archive-date=2018-09-18 |dead-url=yes }}</ref><ref>{{cite web |author1=搜狐音乐 |title=杨凯莉新歌《念念不忘》上线 深情演绎备受好评 |url=http://music.yule.sohu.com/20180711/n543164041.shtml |website=搜狐 |accessdate=2018-09-19 |archive-url=https://web.archive.org/web/20180918123605/http://music.yule.sohu.com/20180711/n543164041.shtml |archive-date=2018-09-18 |dead-url=no }}</ref>

2018年7月中下旬，参加[[中央电视台综艺频道|中央电视台综艺频道]]节目《暑期好味到》，担任嘉宾。<ref>{{cite web |title=《2018暑期好味到》 20180825 足球真奇妙（人物出场时间：06：18） |url=http://tv.cctv.com/2018/08/25/VIDENYtOwAz3WAeDkE5vBnTe180825.shtml |website=CCTV |accessdate=2018-09-18 |archive-url=https://web.archive.org/web/20180917181958/http://tv.cctv.com/2018/08/25/VIDENYtOwAz3WAeDkE5vBnTe180825.shtml |archive-date=2018-09-17 |dead-url=yes }}</ref><ref>{{cite web |title=“莉哥”上央视演唱，刚一开口... 观众：耳朵已怀孕！ |url=http://www.eastyule.com/news/yl/n2018082838464.html |website=东方娱乐网 |accessdate=2018-09-19 |archive-url=https://web.archive.org/web/20180919062126/http://www.eastyule.com/news/yl/n2018082838464.html |archive-date=2018-09-19 |dead-url=yes }}</ref>{{Notetag|该期节目的录制时间为2018年7月20日前后，播出时间为2018年8月25日。但由于杨凯莉被封杀，该期节目的视频现已下架}}

==争议==
2018年10月7日晚，杨凯莉在[[虎牙直播|虎牙]]TV直播时公开戏唱[[中华人民共和国国歌|中华人民共和国国歌]]，并将国歌作为自己的“网络音乐会”的“开幕曲”，有网友举报其行为。網友表示，這樣歌唱的行為是不尊重国歌的表現；10月9日，虎牙平台將其账号做了封禁处理；随后，杨凯莉本人也在[[新浪微博|新浪微博]]發表道歉声明，承认自己的行为低级愚蠢<ref>{{cite news |title=虎牙主播戏唱国歌账号被封后致歉:行为低级且愚蠢 |url=http://news.163.com/18/1010/09/DTOENHOO0001875P.html |accessdate=2018-10-10 |date=2018-10-10 |work=网易新闻 |archive-url=https://web.archive.org/web/20181010135048/http://news.163.com/18/1010/09/DTOENHOO0001875P.html |archive-date=2018-10-10 |dead-url=no }}</ref><ref>[https://tw.appledaily.com/new/realtime/20181011/1445463 中國民主] 中國民主？正妹唱國歌被批不愛國　遭全面封殺出版時間：2018/10/11 蘋果日報</ref>；10月13日晚，[[上海市公安局|上海市公安局]]静安分局在官方微博发布消息，表示已依法对杨凯莉处以[[行政拘留|行政拘留]]5日。<ref>{{cite news |title=上海警方：女主播侮辱国歌被行政拘留5日 |url=http://legal.people.com.cn/n1/2018/1014/c42510-30339320.html |accessdate=2018-10-14 |date=2018-10-14 |work=人民网 |archive-url=https://web.archive.org/web/20181014091322/http://legal.people.com.cn/n1/2018/1014/c42510-30339320.html |archive-date=2018-10-14 |dead-url=no }}</ref>目前，杨凯莉的[[抖音短视频|抖音]]短视频作品以及音乐作品已全网下架。
