'''杨亚人'''（{{bd|1926年|||}}），[[江苏|江苏]][[宜兴|宜兴]]人，[[中华人民共和国|中华人民共和国]]工艺美术家<ref name=bobao>{{Cite web |url = http://news.artxun.com/yang-1575-7874935.shtml |title =杨亚人  |author =  |date = 2011-01-19 |publisher = 博宝艺术网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1949年自[[苏州美术专科学校|苏州美专]]毕业<ref name=bobao/>。1959年任[[人民大会堂|人民大会堂]]厅室装潢、布置、设计组组长。长期在[[国务院机关事务管理局|国务院机关事务管理局]]（简称国管局）工作。[[文革|文革]]爆发后，[[红卫兵|红卫兵]]在“[[破四旧|破四旧]]”中提出要将[[人民大会堂|人民大会堂]]等处的艺术品当做“四旧”通通砸烂。1966年9月，[[周恩来|周恩来]]在人民大会堂召见国管局的杨亚人，交代人民大会堂的“红色革命化”布置工作。随后由杨亚人负责该工作。同期杨亚人遵照周恩来及国管局领导指示，还负责“红化”布置[[北京饭店|北京饭店]]、[[中南海|中南海]]几个大门（不包括[[新华门|新华门]]）、[[紫光阁|紫光阁]]、西花厅和[[北京宋庆龄故居|宋庆龄住地]]等，用时近两年，直到1969年初杨亚人被下放到[[宁夏|宁夏]]的[[五七干校|五七干校]]劳动。后他又下放[[青海省|青海省]]，在青海的高原患病导致左眼失明。1976年调回北京，被分配到[[文物出版社|文物出版社]]任美术编辑<ref name=yangsi/>。

1978年，[[中共中央办公厅|中共中央办公厅]]将杨亚人借调到[[人民大会堂管理局|人民大会堂管理局]]工作，负责将人民大会堂的布置全面复原<ref name=yangsi>{{Cite web |url =http://news.163.com/11/0514/14/7418NIO300014AED.html|title =“破四旧”运动中人民大会堂文物何以保存|author =杨亚人  |date =  |publisher =百年潮2011年04期  |language =  |accessdate =  |quote =  }}</ref>。后来他历任[[中国美术家协会|中国美术家协会]]展览部主任、国务院机关事务管理局美术顾问、高级工艺美术师、[[国务院办公厅|国务院办公厅]]特约高级美术顾问<ref name=bobao/>。他在国管局[[离休|离休]]。杨亚人1952年递交第一份入党申请书，经过长期努力，直到离休后的2011年才被批准加入[[中国共产党|中国共产党]]，并于2012年10月21日转为正式党员<ref>龚喜跃，衷心向党 一生追求——记85岁离休干部杨亚人，中国机关后勤2013年第7期</ref>。

杨亚人装帧设计的作品有《[[宋庆龄|宋庆龄]]画册》、《[[陈嘉庚|陈嘉庚]]》、《[[周恩来|周恩来]]》画册等百余种。曾负责[[中南海|中南海]]和人民大会堂总体布置设计，国家礼品——中南海纪念盘的设计，中国美术家协会[[齐白石|齐白石]]奖评选委员会齐白石奖证书设计等<ref name=bobao/>。
