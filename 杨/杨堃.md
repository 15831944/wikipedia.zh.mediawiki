'''杨堃'''（{{bd|1901年|10月8日|1998年|7月26日|catIdx=Y杨}}），男，[[河北|河北]][[大名府|大名]]人，曾用名'''杨赤民'''，笔名'''张好礼'''、'''杨念基'''，著名[[民族学|民族学]]家、[[民俗学|民俗学]]家、教授。早年曾留学法国，归国后在[[河北大学|河北大学]]、[[中法大学|中法大学]]、[[燕京大学|燕京大学]]、[[北京大学|北京大学]]、[[云南大学|云南大学]]等校任教，1979年进入[[中国社会科学院|中国社会科学院]]任研究员、研究生院教授博士生导师。其研究方向为：民族字、民俗学、社会学、原始社会史、原始宗教、民间宗教。
== 早年和留法期间 ==
1901年10月8日出生于河北省大名县一个农民家庭，是家中独子，12岁进入大名县第二高等小学，15岁进入大名中学，学习成绩优异。16岁时因家庭[[包办婚姻|包办婚姻]]，与当地一账房先生栗景召之女[[栗梅|栗梅]]结婚，后有一女，但杨堃不喜欢这个包办妻子，[[新文化运动|新文化运动]]的影响和[[五四运动|五四运动]]新思潮的涌入让杨堃坚决放弃了这段婚姻。1920年考入保定直隶省立农业专门学校留法预备班，1921年暑期预备班结业，以名列甲等第三的成绩被保送至法国[[里昂中法大学|里昂中法大学]]。

留法期间，杨堃热衷于政治，加入了[[中国共产主义青年团|共青团]]并改名杨赤民。[[第一次国共合作|第一次国共合作]]时期，加入[[中国国民党|中国国民党]]，任国民党驻法总支部里昂支部宣传委员，到处宣传中国革命。同时，他在学业上也没有耽误，获普通数学、微积分、天文学三门课合格证书，取得了理科硕士学位，1926年转入文科。1927年蒋介石发动[[四一二事件|四一二事件]]后，他支持[[武汉国民政府|武汉国民政府]]反蒋，七月[[宁汉合流|宁汉合流]]后退出国民党并解散国民党驻法总支部和里昂支部，之后亦退出共青团。期间经同乡[[郭隆真|郭隆真]]介绍，认识了同在法国读书的[[张若名|张若名]]，二人后于1930年5月在法国[[里昂|里昂]]结婚。

此后杨堃来到[[巴黎大学|巴黎大学]]进修，师从著名汉学家、社会学家、神话专家[[葛兰言|葛兰言]]教授，攻读民族学、体质人类学、语言学和史前考古学。撰写了《在法国怎样学社会学》、《法国社会学家访问记》等文章，确定了民族学和民俗学的研究方向，于1930年底获得文科博士学位。

== 回国任教 ==
1931年回国，先后在[[河北大学|河北大学]]农学院、[[国立北平大学|国立北平大学]]法商学院、[[国立北平大学|国立北平大学]]女子文理学院、[[国立北平师范大学|国立北平师范大学]]文学院社会学系、[[中法大学|中法大学]]孔德学院社会科学系、[[清华大学|清华大学]]社会学系等任教，主讲社会进化史、社会学、普通人类学和民族学。1934年，杨堃等人发起成立了[[中国民族学会|中国民族学会]]，任民族学会《民族学报》编委。1937年，赴[[燕京大学|燕京大学]]社会学系任教，讲授原始社会学、当代社会学学。1941年因太平洋战争爆发，到到北平中法汉学研究所作民俗学专任研究员，1947年至[[云南大学|云南大学]]任社会学系教，暑假加入了[[新民主主义者联盟|新民主主义者联盟]]。

== 新中国成立后 ==
中华人民共和国成立后，杨堃继续从事民族学研究。1953年，任历史系教授和云南民族史研究室主任，参加了云南省的民族识别工作，并进行云南少数民族社会历史调查，撰写了《凉山彝族的手工业》、《云南红河哈尼族彝族自治州元阳县哈尼族的宗教生活》等报告。[[反右运动|反右运动]]时期，妻子张若名自杀，儿子杨在道也被打成右派分子被劳教。[[文化大革命|文化大革命]]被打成“反动资产阶级学术权威””，四次被抄家，屡遭批斗、关牛棚，大部分图书资料、读书卡片和手稿都遗失，但杨堃依然坚持学术研究。1978年，在[[胡耀邦|胡耀邦]]的过问下，任[[中国社会科学院|中国社会科学院]]研究员，1985年担任博士生导师，学生有贾芝、李星华、陈永龄、李慰祖、王辅世、严汝娴、詹承绪、杜玉亭等，兼在[[北京大学|北京大学]]、[[北京师范大学|北京师范大学]]、[[中央民族学院|中央民族学院]]授课，并任[[中国民俗学会|中国民俗学会]]副理事长，[[中国民族学会|中国民族学会]]、[[中国社会学会|中国社会学会]]、[[中国人类学会|中国人类学会]]、[[中国民间文艺家协会|中国民间文艺家协会]]、[[中国少数民族经济研究会|中国少数民族经济研究会]]、[[中国神话学会|中国神话学会]]顾问。1998年7月26日病逝。
