'''杨福家'''（{{bd|1936年|6月|catIdx=Y杨}}），中国[[核物理|核物理]]学家，[[中国科学院院士|中国科学院院士]]和[[第三世界科学院|第三世界科学院]]院士，[[英国|英国]][[诺丁汉大学|诺丁汉大学]]第六任[[校監|校監]]。

==生平==
1936年出生于[[上海|上海]]，原籍[[浙江|浙江]][[镇海|镇海]]，就读上海格致中学并在1958年毕业于[[复旦大学|复旦大学]]，杨福家曾先后担任过[[中国科学院上海应用物理研究所|中国科学院上海应用物理研究所]]所长（1987年－2001年）、[[上海市科学技术协会|上海市科学技术协会]]主席（1992年－1996年）和[[复旦大学|复旦大学]]校长（1993年－1999年）等职务。

杨福家曾在[[丹麦|丹麦]][[哥本哈根大学|哥本哈根大学]][[尼尔斯·玻尔研究所|尼尔斯·玻尔研究所]]、[[拉特格斯大学|-{zh-hans:罗格斯大学; zh-hant:羅格斯大學;}-]]、[[纽约州立大学石溪分校|纽约州立大学石溪分校]]和[[东京大学|东京大学]]作[[访问学者|访问学者]]。同时，他还是[[东京|东京]][[创价大学|创价大学]]、[[纽约州立大学|纽约州立大学]]、[[香港大学|香港大学]]、诺丁汉大学和[[康涅狄格大学|康涅狄格大学]]的名誉博士。
2000年12月12日,在杨福家卸任复旦大学校长一年之际,英国诺丁汉大学董事会宣布:选举中国科学家杨福家先生为下一届校长。从2001年1月1日起,任期3年<ref>{{Cite web |url = http://elite.youth.cn/zt/201209/t20120907_2419652.htm |title = 中国青年报：做人第一，修业第二——专访英国诺丁汉大学前校长杨福家|publisher = 中国青年报  }}</ref>。
2001年7月4日，杨福家被正式任命为[[英国诺丁汉大学|英国诺丁汉大学]]第六任校監，他也是第一位担任英国大学校監的中国人。
2004年开始，兼任[[寧波諾丁漢大學|寧波諾丁漢大學]]校長<ref>{{Cite web |url = http://cpc.people.com.cn/n/2015/1023/c83084-27732294.html |title = 中国教育打败了英国教育吗——对话宁波诺丁汉大学校长杨福家|publisher = 解放日报  }}</ref>。
2012年年底，[[英国诺丁汉大学|英国诺丁汉大学]]4届校长任期期满，杨福家卸任[[英国诺丁汉大学|英国诺丁汉大学]]校长，继续担任[[寧波諾丁漢大學|寧波諾丁漢大學]]校長至今<ref>{{Cite web |url = https://www.nottingham.edu.cn/cn/news/2013/yang-fujia-interview.aspx |title = 杨福家　出发,是为了回归 |author = 杨晓钗  |publisher = 中华儿女报刊社  }}</ref>。
