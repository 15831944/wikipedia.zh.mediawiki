'''杨紫'''（原名：'''杨旎奥'''，{{bd|1992年|11月6日|catIdx=Y杨}}）出生于北京市，[[中國內地|中國內地]]女演员，毕业于北京电影学院。童星时期，因出演古装历史情感剧《[[孝庄秘史|孝庄秘史]]》开始崭露头角。2004年，因主演情景喜剧《[[家有儿女|家有儿女]]》“夏雪”一角而被观众所熟悉和喜爱，也因此被称为“国民女儿”。2012年，初长成挑战大银幕《[[守株人|守株人]]》，更凭借该电影获得第14届[[中国电影金凤凰奖|中国电影金凤凰奖]]新人奖。凭借《[[花非花雾非雾|花非花雾非雾]]》、《[[战长沙|战长沙]]》等多部影视作品告别童星华丽转型。后因相续出演《[[欢乐颂|欢乐颂]]》、《[[青云志|青云志]]》、《[[龙珠传奇|龙珠传奇]]》、《[[天乩之白蛇传说|天乩之白蛇传说]]》、《[[香蜜沉沉烬如霜|香蜜沉沉烬如霜]]》等电视剧人气急升。

==演藝經歷==
===早期經歷===
==== 1997年至2005年 ====
1997年，5歲的楊紫受到《[[還珠格格|還珠格格]]》的影響，開啟了演藝之路，出演了人生第一部作品《春天的狂想》，當時通知楊紫去拍戲，楊紫的爸爸就抱著她，在凌晨群演排隊。<ref>{{cite web|url=https://www.weibo.com/ttarticle/p/show?id=2309404174110186984186&mod=zwenzhang |title=楊紫星空演講講稿 |publisher=Weibo.com |date= |accessdate=2018-12-11}}</ref>

1999年，參演第一部電視劇《如次出山》，在劇中飾演周瓊，楊紫正式進入演藝圈。

2002年，參演的《黨員馬大姐》以及《如次出山》播出。

2003年，客串演出大型古裝劇《[[孝莊秘史|孝莊秘史]]》宛如(年少)一角播出而嶄露頭角，開始當群演、當客串，出演了電視劇《大宅門》、《少年康熙》與電影《警察有約》。

2004年，出演的《案發現場》與《家庭檔案》兩部電視劇播出，同時主演的《[[家有兒女|家有兒女]]》開始拍攝；5月15日，主演的電影《女生日記》上映，入圍童牛獎最佳小演員。

2005年，2月12日在《[[家有兒女|家有兒女]]》北京電視台首播，在劇中飾演成績優秀的高中生，楊紫憑藉此劇成為家喻戶曉的童星，讓觀眾們認識了楊紫，成為當時大家心目中的榜樣女兒；11月28日參與演出的《緣分》在北京電視台四套首播，在劇中飾演魏司晨，隨後主演的電視劇《女生日記》接著播出。
===求學時期===
==== 2006年至2012年 ====
2006年，播出的《[[家有兒女|家有兒女(第二部)]]》仍然大受好評，但因要準備中考辭演第三部<ref>{{cite web|url=http://eladies.sina.com.tw/getnews.php?newsid=163613 |title=《家有兒女》12年感情不變！張一山和楊紫仍在鬥嘴，開口仍叫宋丹丹“媽” |publisher=Eladies.sina.com.tw |date=2016-07-22 |accessdate=2018-12-11}}</ref>；參與演出的《無限生機》、《不想長大》也在今年播出。

2007年至2011年間參與多部作品，參與電視劇《溫暖》、《活著，真好》、《珍寶》、《不想長大》、《青春舞台》、《回家》、《女孩沖沖衝》、《純真歲月》、《吳承恩與西遊記》、《男生日記》、《幸福來敲門》的演出與電影《三班五班》、《喋血孤城》的演出以及為《淘氣包馬小跳》、《尋找狗托邦》兩部動畫片配音。

2010年7月17日，楊紫透過微博與大家分享收到[[北京电影学院|北京电影学院]]的錄取通知書<ref>{{cite web|url=http://news.163.com/10/0719/10/6BUUGLES00014AED.html |title=楊紫收到北影錄取通知書 |publisher=News.163.com |date= |accessdate=2018-12-11}}</ref>，成為2010級的表演系本科班學生。

2012年5月3日，參與演出的《[[心術|心術]]》首播，在劇中飾演懷揣夢想、性格善良正直的小護士張曉蕾；5月16日，主演的《[[老爸回家|老爸回家]]》在北京開機拍攝；7月14日，參與演出的《[[花非花霧非霧|花非花霧非霧]]》開機拍攝；8月23日，主演的電影《守株人》上映，劇中飾演杀戮者「守株人」佳佳，並憑藉該電影獲得中國電影金鳳凰獎新人獎；10月16日，主演的《[[戰長沙|戰長沙]]》開機拍攝。

===演藝經歷===
==== 2013年至2016年 ====
2013年2月23日，主演的《[[胭脂霸王|胭脂霸王]]》在湖南衛視(上午檔)首播，劇中飾演在一聲驚雷中誕生的一個特殊的女孩雷兒，並為該劇獻聲片尾曲《胭脂花開》；4月8日，主演的《[[老爸回家|老爸回家]]》在江西衛視首播，劇中飾演極具個性的“毒舌”文藝青年林然然；4月底，主演的《[[天生要完美|天生要完美]]》在深圳開機拍攝；8月6日，《[[花非花霧非霧|花非花霧非霧]]》在湖南衛視首播，為華心四朵花之一，在劇中與[[鄧倫|鄧倫]]搭檔飾演情侶；11月25日，主演電影《鬥茶》在北京開機拍攝。

2014年3月8日，主演的《[[戰長沙|戰長沙]]》在[[上海電視台|上海電視台]]首播，在劇中飾演戰地的出色護士胡湘湘，並為該劇配唱片尾曲《我會記得你》，該劇為豆瓣電影2014年度評分最高的大陸電視劇；4月13日做為固定成員參與江蘇衛視青春勵志真人秀節目《[[花樣年華_(電視節目)|花樣年華第二季]]》拍攝，於貴州雷山縣烏冬村開拍；5月17日，《[[花樣年華_(電視節目)|花樣年華第二季]]》播出，在節目中展現了不拘一格的個人魅力；5月22日於[[北京电影学院|北京电影学院]]表演系本科班畢業；6月28日主演的青春愛情電影《時間都去哪了》在青島開機拍攝；7月27日，《大秧歌》在北京開機拍攝；9月11日，楊紫主演的電影《鬥茶》上映，在劇中飾演已故鬥茶高人之女謝小香。

2015年1月11日，主演的現代軍旅劇《[[天生要完美|天生要完美]]》在央視八套首播，劇中飾演性格直爽且熱情大方的軍校學員安小惠；4月19日，參加的東方衛視明星旅行真人秀《花樣姐姐》播出；8月7日，主演的青春愛情電影《時間都去哪了》上映，飾演鋼琴家林雨桐；10月14日，領銜主演的紅色年代傳奇劇《[[大秧歌|大秧歌]]》於[[上海電視台|上海電視台]]首播，劇中飾演可愛善良漂亮的吳家大小姐吳若云；10月30日，參演的懸疑驚悚網劇《[[靈魂擺渡|靈魂擺渡]]2》在[[愛奇藝|愛奇藝]]開播，分別出演《紅月》和《鬼妻》單元。

2016年3月18日，其參演的喜劇冒險片《[[洛杉磯搗蛋計劃|洛杉磯搗蛋計劃]]》上映，她在片中飾演失踪藝人王妮娜；4月18日，主演的都市職場女性劇《[[歡樂頌_(第一季)|歡樂頌]]》開播，在劇中飾演單純熱心又有些迷糊的職場新人邱瑩瑩，並為該劇配唱主題曲《總有幸福在等你》及插曲《開得比花香》，並憑藉該劇提名第23屆上海電視節白玉蘭獎最佳女配角；6月29日，其領銜主演的都市情感劇《[[大嫁風尚|大嫁風尚]]》首播，劇中飾演性格真誠又浪漫非凡的金牌婚禮策劃師夏燃；7月31日，主演的古裝仙俠劇《[[青雲志|青雲志]]》播出，劇中飾演冷傲癡情的陸雪琪，並為該劇配唱插曲《若只如初見》，而楊紫則憑藉該劇提名第22屆華鼎獎古裝題材電視劇最佳女演員；8月26日，她主演的青春愛情電影《在世界中心呼喚愛》登陸中國院線，在劇中飾演成績優異的三好學生夏葉；9月，在南都娛樂周刊發起的“90後”女藝人綜合性評選活動中，楊紫被評為“90後四小花旦”之一；12月8日，主演的《[[青雲志|青雲志]]2》繼續登錄騰訊視頻。

==== 2017年至今 ====
2017年1月15日，青春神話劇《[[天乩之白蛇傳說|天乩之白蛇傳說]]》開機拍攝；1月27日，楊紫登上央視春節聯歡晚會的舞台，與[[劉濤|劉濤]]、[[蔣欣|蔣欣]]、[[王子文|王子文]]、[[喬欣|喬欣]]及[[TFBOYS|TFBOYS]]共同演唱開場歌曲《美麗中國年》；5月8日，主演的傳奇清裝劇《[[龍珠傳奇|龍珠傳奇]]》播出，飾演俏皮可愛的前朝公主李易歡；5月11日，楊紫繼續主演都市女性勵志劇《[[歡樂頌2|歡樂頌2]]》，並為該劇配唱主題曲《我們》及人物主題曲《蚯蚓》；6月14日，加盟主演的古裝神話仙俠劇《[[香蜜沉沉燼如霜|香蜜沉沉燼如霜]]》開機拍攝；10月23日，與[[張家輝|張家輝]]合作主演的犯罪動作電影《[[沉默的證人|沉默的證人]]》開機拍攝，飾演實習法醫喬琳；而在這一年的福布斯中國名人榜中因多部作品獲得關注的楊紫最終排名第73位。

2018年2月15日，楊紫再次登上了中央電視台春節聯歡晚會的舞台，出演了小品《為您服務》；4月10日，客串的都市女性鏡像劇《[[北京女子圖鑑|北京女子圖鑑]]》在優酷獨播，飾演了護士苗苗；隨後，領銜主演的甜萌夢幻愛情劇《[[蜜汁燉魷魚|蜜汁燉魷魚]]》開機拍攝，飾演活潑軟萌的學霸蘿莉、擅長編程的天才少女佟年；4月28日，作為固定嘉賓加盟偶像對抗挑戰真人秀《[[高能少年團|高能少年團]]2》；7月9日，《[[天乩之白蛇傳說|天乩之白蛇傳說]]》在[[愛奇藝|愛奇藝]]播出，劇中飾演癡情小白蛇白夭夭，並為該劇配唱插曲《幾生歡》；8月2日，《[[香蜜沉沉燼如霜|香蜜沉沉燼如霜]]》在[[江蘇衛視|江蘇衛視]]首播，飾演花神之女錦覓，並為該劇配唱插曲《情霜》與《天地無霜》，該劇自播出就穩坐全國收視率冠軍，網播突破百億，CSM52城平均收視率也破1.3%，是2018年暑假收視冠軍；8月28日，《[[我的莫格利男孩|我的莫格利男孩]]》在上海開機拍攝，該劇為中國第一部“環保主義”的電視劇，飾演城市女凌煕。12月24日，楊紫被選為“2018最美表演”的十位年度優秀演員之一，以作品《前夕》為觀眾呈現她的最美表演“一分鐘”。<ref>{{cite web|url=http://ent.sina.com.cn/s/m/2018-11-26/doc-ihmutuec3696131.shtml |title=2018最美表演首曝揭秘海報 陳偉霆楊紫打頭陣 |publisher=新浪娛樂 |date=2018-11-26 |accessdate=2018-12-24}}</ref>

==感情生活==
2017年2月16日，杨紫在微博承认与[[秦俊杰|秦俊杰]]的恋情。<ref>{{cite web|url=http://ent.qq.com/a/20170216/046200.htm |title=“小雪”楊紫公佈戀情 作為官配的張一山這麼說 |publisher=Ent.qq.com |date= |accessdate=2018-12-11}}</ref>

2018年8月12日，杨紫和秦俊杰工作室先后发布声明，宣布两人已和平分手，结束恋人关系，做回朋友，「二人曾攜手度過一段美好的時光，做出這個決定也是二人經過了慎重地考慮，在此過程中無關他人、沒有背叛，之後他們也將對彼此懷著真摯的祝福。」<ref>{{cite web|url=http://ent.ifeng.com/a/20180812/43086378_0.shtml?_zbs_baidu_bk |title=杨紫秦俊杰宣布分手：无关他人没有背叛 |publisher=Ent.ifeng.com |date=2018-10-26 |accessdate=2018-12-11}}</ref>

== 慈善公益活動 ==
===公益行動===
楊紫有著強烈的社會責任感，她善用自己的影響力回饋社會，在助學、公益電影、學生健康等諸多公益項目中都默默貢獻著自己的力量。
*2008年5月，楊紫與爸爸共同參與汶川大地震之賑災活動，給予鼓勵與關懷。<ref>{{cite web|url=https://new.qq.com/omn/20180512/20180512A0U1EN.html#p=1 |title=十年前汶川地震，楊紫、霍建華等志願者曾低調赴震區救災！ |publisher=New.qq.com |date=2018-05-12 |accessdate=2018-12-11}}</ref>
*2008年9月，楊紫出席歐美同學會企業家聯誼會“中信國安”慈善之夜，並獲得傳播百年奧運精神愛心行動最具慈善精神新人獎。<ref>{{cite web|url=http://gongyi.sohu.com/20081201/n260958799.shtml |title=歐美同學會企業家聯誼會在京舉辦慈善晚宴 |publisher=Gongyi.sohu.com |date= |accessdate=2018-12-11}}</ref>
*2009年7月，楊紫參加童星公益巡演，並提取該巡演的部分款項捐贈給中國關心下一代健康體育基金會和中國西部雛鷹助學網用於助學計劃。 <ref>{{Cite web |url=https://finance.qq.com/a/20090719/000094.htm |title=楊紫助陣童星公益巡演 |access-date=2018-09-05 |archive-url=https://web.archive.org/web/20180905214940/https://finance.qq.com/a/20090719/000094.htm |archive-date=2018-09-05 |dead-url=yes }}</ref>
*2010年1月，楊紫受邀出席由某慈善機構舉辦的“希望健行—給我一個家”活動，為眾多福利院的少年兒童送去了溫暖與新年禮物。 <ref>{{cite web|url=http://yule.sohu.com/20100118/n269681236.shtml |title=楊紫投身公益溫暖寒冬 與福利院兒童暢聊夢想 |publisher=Yule.sohu.com |date= |accessdate=2018-12-11}}</ref>
*2011年3月，楊紫擔任“心系女童”公益活动的爱心大使，活动旨在关爱全中国千万家庭中青春期女童的成長。<ref>[http://yule.sohu.com/20110330/n305124303.shtml 楊紫擔任愛心大使 心系女童詮釋美麗]</ref>
*2011年5月，楊紫擔任身障兒童「我們共有一片藍天」的公益慈善活動的形象代言人，身體力行關注並幫助更多的身障兒童。 <ref>[http://yule.sohu.com/20110506/n306880625.shtml 張一山楊紫關注身障兒童 北影發起“愛心接力”]</ref>
*2012年4月，楊紫參加第八屆北京青少年公益電影節並擔任形像大使。 <ref>[http://yule.sohu.com/20120423/n341331997.shtml 第八屆北京青少年公益電影節 楊紫張一山任大使]</ref>
