'''杨百翰大学电视台'''（{{Lang-en|BYU TV}}；特别字体：{{lang|en|BYUtv}}）是一家[[美国|美国]]非商业化[[电视频道|电视频道]]，成立于2000年1月1日，由[[杨百翰大学|杨百翰大学]]所有并运营。<ref name="about">{{cite web |title=About BYUtv |url=https://www.byutv.org/about |publisher=杨百翰大学电视台 |accessdate=2019-03-27}}</ref> 杨百翰大学电视台通过[[地面电视|地面电视]]、[[有线电视|有线电视]]和[[卫星电视|卫星电视]]等形式覆盖美国，制作了许多原创系列及纪录片，特别是喜剧、历史、生活、音乐和戏剧方面。同时杨百翰大学电视台也会播出一些[[華特迪士尼影業集團|华特迪士尼公司]]的真人电影和来自电影公司的经典影片，以及自然纪录片、医务剧、罪案剧和宗教节目；其宗教节目的提供方与杨百翰大学的宗教赞助方一致，均为[[耶稣基督后期圣徒教会|耶稣基督后期圣徒教会]]。<ref name="about" /><ref>{{cite web |author1=Jamshid Ghazi Askar |title=If he builds it, will they come? Scott Swofford talks BYUtv |url=https://www.deseretnews.com/article/865577390/If-he-builds-it-will-they-come-Scott-Swofford-talks-BYUtv.html |publisher={{Link-en|德律撒新闻报|Deseret News}} |accessdate=2019-03-27}}</ref> 此外，杨百翰大学电视台体育频道（{{lang|en|BYUtv Sports}}）是{{Link-en|杨百翰大学美洲豹队|BYU Cougars}}的主要体育转播方，仅在2012年就播出了超过125场的直播赛事。<ref>{{cite web |title=Ryan Teeples: BYU sports is for BYUtv, not the other way around |url=https://www.deseretnews.com/article/865581885/Ryan-Teeples-BYU-sports-is-for-BYUtv-not-the-other-way-around.html |publisher=德律撒新闻报 |accessdate=2019-03-27}}</ref> 杨百翰大学电视台获得多次[[艾美奖|艾美奖]]，<ref>{{cite web |title=洛基山地区艾美奖 |url=http://rockymountainemmy.org/awards/emmy-awards/ |publisher=艾美奖 |accessdate=2019-03-27}}</ref> 并且其原创系列中的部分节目还获得了国家级电视评论家的好评。<ref>{{cite news |last=Garvin |first=Glenn |url=https://www.miamiherald.com/2013/04/07/3328291/quirky-granite-flats-a-charmer.html |title=Quirky ‘Granite Flats’ a charmer; ‘Kalman’ plumbs Holocaust loss |work={{Link-en|迈阿密先驱报|Miami Herald}} |date=2013-04-07 |deadurl=yes |archiveurl=https://web.archive.org/web/20131021190258/https://www.miamiherald.com/2013/04/07/3328291/quirky-granite-flats-a-charmer.html |archivedate=2013-10-21 |accessdate=2019-03-27 }}</ref><ref>{{cite web |author1=Ryan Morgenegg |title=Season 3 of 'Studio C' on BYUtv |url=https://www.deseretnews.com/article/865583597/Season-3-of-Studio-C-on-BYUtv-on-the-way.html |publisher=德律撒新闻报 |accessdate=2019-03-27}}</ref><ref>{{cite web |author1=Neil Genzlinger |title=If It’s Aliens, at Least They Won’t Be Naked |url=https://www.nytimes.com/2013/04/06/arts/television/granite-flats-on-byutv-from-brigham-young-university.html |publisher=[[纽约时报|纽约时报]] |accessdate=2019-03-27}}</ref>

杨百翰大学电视台在其官方网站提供[[流媒体|流媒体]]服务，全球观众可以透过该服务收看杨百翰大学电视台的全部原创节目和其他大部分节目。同时杨百翰大学电视台也透过{{Link-en|KBYU-TV|KBYU-TV}}的{{Link-en|数字子频道|Digital subchannel}}向[[盐湖城|盐湖城]]及由[[犹他州|犹他州]]大部分地区播出高清电视信号。{{Link-en|KBYU-TV|KBYU-TV}}是[[公共广播电视公司|美国公共广播电视网]]位于[[普若佛|普若佛]]的加盟会员台，同时也隶属于[[杨百翰大学|杨百翰大学]]所有。杨百翰大学电视台是杨百翰大学广播事业部门负责运营的几个频道之一，它们还拥有以下频道：面向全球播出的杨百翰大学电视台环球频道（{{lang|en|BYUtv Global}}）和杨百翰大学电台（{{lang|en|BYUradio}}）；以及专注[[拉丁美洲|拉丁美洲地区]]的{{Link-en|杨百翰大学国际频道|BYU Television International}}。有许多明星艺人在杨百翰大学电视台出演特别节目，例如：[[謎幻樂團|梦龙乐队]]和[[霓虹树合唱团|霓虹树合唱团]]曾在《{{Link-en|音乐档案|Audio-Files}}》中出演；[[丽娅·萨隆加|丽娅·萨隆加]]、{{Link-en|霍华德·琼斯 (英国音乐人)|Howard Jones (English musician)|霍华德·琼斯}}、{{Link-en|邓肯·谢科|Duncan Sheik}}、[[蓮兒與啷噹六便士|啷当六便士合唱团]]等曾在《改变我人生的歌曲（{{lang|en|The Song That Changed My Life}}）》中出演；以及[[肖恩·布拉德利|肖恩·布拉德利]]、{{Link-en|钢琴达人|The Piano Guys}}、{{Link-en|国家伴侣|Mates of State}}等在《{{Link-en|C号演播厅|Studio C}}》中出演。此外，[[史提夫·扬|史提夫·扬]]、{{Link-en|蒂·德默|Ty Detmer}}、[[吉默·弗雷戴特|吉默·弗雷戴特]]等主要运动员也在杨百翰大学电视台体育频道的节目中登场。<ref>{{cite web |author1=Faith Heaton |title=BYUtv's AUDIO-FILES creating underground buzz |url=https://www.ksl.com/article/19928075 |publisher={{Link-en|KSL广播网|KSL (radio network)}} |accessdate=2019-03-27}}</ref><ref>{{cite web |author1=Cody Clark |title=The Song that Changed My Life on BYUtv |url=https://www.heraldextra.com/lifestyles/the-song-that-changed-my-life-on-byutv/article_9d60f394-0da8-58ed-ae83-ca4487d064ff.html |publisher={{Link-en|每日先驱报 (阿灵顿高地)|Daily Herald (Arlington Heights)|每日先驱报}} |accessdate=2019-03-27}}</ref><ref>{{cite web |title=Shoulder Angel Meets Shawn Bradley |url=https://www.youtube.com/watch?v=PHM01zr1MuU |website=[[YouTube|YouTube]] |publisher={{Link-en|C号演播厅|Studio C}} |accessdate=2019-03-27}}</ref><ref>{{cite web |title=On BYUTV right now with Lavell and Steven Young and other QB's |url=https://www.cougarboard.com/board/message.html?id=10785503 |publisher=BengalCougar |accessdate=2019-03-27}}</ref>
