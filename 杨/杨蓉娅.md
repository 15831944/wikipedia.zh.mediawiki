'''杨蓉娅'''（{{bd|1957年||catIdx=Yang楊}}），山西[[兴县|兴县]]人，[[汉族|汉族]]。[[中共|中共]]党员。[[共和国卫士|共和国卫士]]、[[全国三八红旗手|全国三八红旗手]]。[[北京军区总医院|北京军区总医院]]副院长<ref>{{cite web|title = 全国人大代表信息-杨蓉娅 |url = http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=112841 |accessdate=2013-04-05 |publisher = 全国人大网}}</ref>。

毕业于[[第三军医大学|第三军医大学]]皮肤病性病专业。1988年，进入北京军区总医院工作。曾以[[访问学者|访问学者]]身份赴美国[[纽约大学|纽约大学]]医学中心学习两年<ref>{{Cite web |url=http://elite.youth.cn/zt/201402/t20140214_4663925.htm |title=杨蓉娅　“美女少将”的非凡人生|accessdate=2015-03-30 |author=记者：陈晰，编辑：凤栖 |date=2014-02-14 |work= |publisher=中华儿女杂志，来源：中华儿女报刊社|language=Zh-hans }}</ref>，1995年归国。2008年起任[[第十一届全国人民代表大会|第十一届全国人民代表大会]]解放军[[全国人民代表大会代表|代表]]<ref>{{cite web |title = 第十一届全国人大代表 |url = http://news.xinhuanet.com/ziliao/2008-03/03/content_7708230.htm?node=36628 |publisher=新华网 |accessdate=2014-01-07}}</ref>。
