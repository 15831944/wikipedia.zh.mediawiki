'''杨刚'''（{{bd|1953年|1月5日|||catIdx=Y杨}}），[[河北省|河北省]][[藁城区|藁城县]]人。[[中华人民共和国|中华人民共和国]]政府官员，曾任[[中国共产党新疆维吾尔自治区委员会|中共新疆维吾尔自治区党委]]副书记，[[新疆维吾尔自治区|新疆维吾尔自治区]]常务副主席，[[中华人民共和国国家质量监督检验检疫总局|国家质量监督检验检疫总局]]党组副书记、副局长，[[中国人民政治协商会议全国委员会经济委员会|全国政协经济委员会]]副主任。<ref name=wy>{{cite web|accessdate=2012-05-01|url=http://news.163.com/11/0104/12/6PI92SFC00014JB6.html|title=原新疆自治区副书记任杨刚任质监总局副局长|date=2011-01-04|publisher=网易}}</ref><ref name=ifeng>{{cite news|title=财政部原副部长丁学东被任命为国务院副秘书长|url=http://finance.ifeng.com/news/people/20100618/2323131.shtml|date=2010-06-18|accessdate=2012-06-09|newspaper=凤凰网}}</ref><ref name=zgxww>{{cite news|title=国务院任免国家工作人员 宫蒲光任民政部副部长|url=http://www.chinanews.com/gn/2013/07-23/5072868.shtml|accessdate=2013-07-27|newspaper=中国新闻网|date=2013-07-23}}</ref><ref name=ifeng2>{{cite news|title=原新疆副书记、原质检总局副局长杨刚被查|url=http://news.ifeng.com/mainland/special/fanfu/content-3/detail_2013_12/27/32522415_0.shtml|date=2013-12-27|accessdate=2013-12-27|newspaper=凤凰网}}</ref>

==生平==
1969年2月至1974年12月，杨刚为[[新疆生产建设兵团|新疆生产建设兵团]]农八师150团农工。1974年12月至1978年9月，任新疆铁工局三团测量员、团委副书记、局保卫干事。其间，1976年4月加入[[中国共产党|中国共产党]]。1978年9月至1982年8月，在[[新疆大学|新疆大学]]政治系政治理论专业学习。<ref name=ifeng2/><ref name=wy/><ref name=ifeng/><ref name=zgxww/>

1982年8月至1984年4月，杨刚任中共新疆生产建设兵团党委组织部干事。1984年4月至1985年7月，任共青团新疆生产建设兵团委员会办公室主任。1985年7月至1991年3月，任共青团新疆生产建设兵团委员会副书记。其间，1986年9月至1989年7月，在[[中共中央党校|中共中央党校]]研究生班政治理论专业学习，获在职研究生学历。<ref name=ifeng2/><ref name=wy/><ref name=ifeng/><ref name=zgxww/>

1991年3月至1991年12月，杨刚任新疆生产建设兵团农八师副师长。1991年12月至1993年10月，任共青团新疆维吾尔自治区委员会副书记、党组书记。1993年10月至1994年6月，任中共吐鲁番地委副书记（正厅级）。1994年6月至1997年7月，任中共吐鲁番地委书记。其间，1996年9月至1997年7月，在[[中共中央党校|中共中央党校]]一年制中青年干部培训班学习。<ref name=ifeng2/>

1997年7月至1999年11月，杨刚任中共新疆维吾尔自治区党委组织部副部长，[[新疆维吾尔自治区人事厅|新疆维吾尔自治区人事厅]]党组书记、副厅长，[[新疆维吾尔自治区机构编制委员会办公室|新疆维吾尔自治区机构编制委员会办公室]]主任。<ref name=ifeng2/>

1999年11月至2000年12月，杨刚任中共乌鲁木齐市委书记。2000年12月至2001年3月，任中共新疆维吾尔自治区党委常委，乌鲁木齐市委书记。2001年3月至2004年12月，任中共新疆维吾尔自治区党委常委，乌鲁木齐市委书记，[[新疆生产建设兵团|新疆生产建设兵团]]农十二师党委第一书记、第一政委。2004年12月至2006年10月，任中共新疆维吾尔自治区党委常委，乌昌党委书记，乌鲁木齐市委书记，新疆生产建设兵团农十二师党委第一书记、第一政委。2006年10月至2006年11月，任中共新疆维吾尔自治区党委副书记，乌昌党委书记，乌鲁木齐市委书记。2006年11月至2007年5月，任中共新疆维吾尔自治区党委副书记，中共新疆维吾尔自治区党委党校校长。2007年5月至2010年12月，任中共新疆维吾尔自治区党委副书记，[[新疆维吾尔自治区人民政府|新疆维吾尔自治区人民政府]]常务副主席，中共新疆维吾尔自治区党委党校校长。2010年12月23日，新疆维吾尔自治区第十一届人民代表大会常务委员会第二十三次会议作出决定，接受杨刚因工作变动而辞去新疆维吾尔自治区副主席职务的请求，报新疆维吾尔自治区第十一届人民代表大会第四次会议备案。<ref name=ifeng2/><ref name=wy/><ref name=ifeng/><ref name=zgxww/>

2010年12月至2013年3月，杨刚任[[中华人民共和国国家质量监督检验检疫总局|国家质量监督检验检疫总局]]党组副书记、副局长。2013年3月至2013年7月，任国家质量监督检验检疫总局党组副书记、副局长，第十二届全国政协经济委员会副主任。2013年7月起，仅任第十二届全国政协经济委员会副主任。<ref name=ifeng2/><ref name=zgxww/>2013年12月，第十二届全国政协经济委员会副主任杨刚因涉嫌严重违纪违法，正在接受[[中国共产党中央纪律检查委员会|中共中央纪律检查委员会]]调查。<ref name=ifeng2/><ref>[http://www.apdnews.com/news/57803.html 杨刚被查] {{webarchive|url=https://web.archive.org/web/20131230221406/http://www.apdnews.com/news/57803.html |date=2013-12-30 }}，亚太日报，2013年12月30日</ref>2013年12月30日，据[[中共中央组织部|中共中央组织部]]有关负责人证实，“杨刚涉嫌严重违纪，中央已决定免去其全国政协经济委员会副主任职务，并撤销十二届全国政协委员资格，现正在按程序办理。”<ref>[http://www.hn.xinhuanet.com/2013-12/30/c_118766748.htm 中组部：杨刚涉嫌严重违纪被免职，新华网，2013年12月30日] {{webarchive|url=https://web.archive.org/web/20131230233222/http://www.hn.xinhuanet.com/2013-12/30/c_118766748.htm |date=2013年12月30日 }}</ref>2014年7月11日被双开<ref name=july11>{{cite news|title=全国政协经济委员会原副主任杨刚被双开 与他人通奸|url=http://www.chinanews.com/fz/2014/07-11/6375389.shtml|work=中国新闻网|date=2014年7月11日}}</ref>。2016年1月20日，[[北京市第三中级人民法院|北京市第三中级人民法院]]公开宣判，杨刚犯受贿罪，判处有期徒刑十二年，并处没收个人财产人民币一百万元<ref>{{cite web |url = http://www.chinacourt.org/article/detail/2016/01/id/1794196.shtml |title = 全国政经委原副主任杨刚一审获刑12年 受贿逾千万 |accessdate = 2016年1月20日 |publisher = 中国法院网}}</ref>。

杨刚是[[中国共产党第十六次全国代表大会|中共十六大]]、[[中国共产党第十七次全国代表大会|十七大]]代表，中共第十六、十七届候补中央委员，[[第十一届全国人民代表大会代表名单|第十一届全国人大代表]]。<ref name=ifeng2/>
