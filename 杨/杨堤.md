'''杨堤'''（{{bd|1924年|9月|2017年|10月6日}}），曾名'''王上达'''、'''王敏'''，[[江苏|江苏]][[青浦|青浦]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员，[[中国共产党第十二届中央委员会|中国共产党第十二届中央委员会]]委员<ref name=gaobie/>。

==生平==
1938年9月参加[[中国共产党|中国共产党]]领导的革命工作，1939年5月加入中国共产党。1938年9月到1939年7月，任[[皖南|皖南]][[新四军|新四军]]教导总队青年队学员、排长。1939年7月到1941年2月，任新四军政治部组织部科员、军法处二科科员。1941年2月到1944年3月，任苏北解放区盐阜地区保安处科员，中共[[淮安县|淮安县]]委、[[建阳县|建阳县]]委社会部部长兼保安科科长。1944年3月到1947年11月，任苏北中共[[盐城县|盐城县]]委社会部部长兼公安局局长，盐阜专署公安局科长，中共盐城县委敌工部部长兼区委书记。1947年11月到1949年8月，任华中公安处、苏南行政公署公安局科长。1949年8月到1952年9月，任[[江苏省|江苏省]][[无锡市|无锡市]]公安局副局长、局长。1952年9月到1956年2月，任[[上海市公安局|上海市公安局]]政保一处处长。1956年2月到1960年4月，任[[中共上海市委|中共上海市委]]政法工作部办公室主任。1960年4月到1967年1月，任上海市对外贸易局党委副书记<ref name=huaxia/><ref name=gaobie/>。

“[[文化大革命|文化大革命]]”中遭到迫害。1976年8月到1977年4月，任上海市副食品领导小组办公室负责人。1977年4月到1979年12月，任上海市邮电管理局党委书记、局长。1979年12月到1981年7月，任[[上海市人民政府|上海市人民政府]]副市长兼基本建设委员会主任、党组书记。1981年7月到1983年3月，任上海市人民政府副市长兼公安局局长、党组书记。1983年3月到1985年5月，任中共上海市委书记（当时设有第一书记、第二书记）。1985年5月到1989年8月，任中共上海市委副书记<ref name=huaxia>{{cite news|title=杨堤|url=http://www.huaxia.com/lasd/hxrwk/ddrw/dl/2003/11/499896.html|date=2003-11-18|newspaper=华夏经纬网}}</ref><ref name=gaobie/>。

杨堤是[[中国共产党第十二届中央委员会|中国共产党第十二届中央委员会]]委员，[[中共十二大|中共十二大]]、[[中共十三大|中共十三大]]代表，第七届、第八届全国政协常委<ref name=gaobie>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1819524 |title = 杨堤同志遗体告别仪式今天上午在上海龙华殡仪馆举行 |author =  |date = 2017-10-10 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

2017年10月6日，杨堤在[[上海华东医院|上海华东医院]]病逝，享年93岁<ref>{{Cite web|url=http://cpc.people.com.cn/n1/2017/1007/c87393-29574064.html|title=杨堤同志逝世--新闻报道-人民网|date=2017-10-07|last=2288|work=人民网}}</ref>。2017年10月10日，杨堤同志遗体告别仪式在[[上海市龙华殡仪馆|上海市龙华殡仪馆]]举行<ref name=gaobie/>。
