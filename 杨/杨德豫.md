'''杨德豫'''{{BD|1928年|12月12日|2013年|1月23日|catIdx=Yang杨德豫}}[[北京|北京]]人，籍贯[[湖南|湖南]][[长沙县|长沙县]]，毕业于[[湖南大学|湖南大学]]、[[南京大学|南京大学]]、[[清华大学|清华大学]]，著名诗歌翻译家、编辑，代表作有《华兹华斯诗选》、《柯勒律治诗选》、《拜伦诗选》、《朗费罗诗选》。<ref name="fish " >{{cite web |url=http://culture.ifeng.com/whrd/detail_2013_01/24/21545064_0.shtml |title=著名翻译家杨德豫去世 专家：他是中国英译诗高峰 |accessdate=2013年1月24日 |publisher=凤凰网 |work= |language=zh-cn}}</ref>

== 生平 ==
1928年，杨德豫出生在[[北平|北平]]，父亲是教育家[[杨树达|杨树达]]，母亲张家祓，还有2个兄弟杨德庆、杨德嘉。杨德豫早年爱好[[美术|美术]]，曾在[[湖南大学|湖南大学]]、中央大学（今南京大学）、[[清华大学|清华大学]]学习外国语言。<ref name=" tiger" >{{cite web |url=http://culture.ifeng.com/whrd/detail_2013_01/23/21518991_0.shtml?_from_ralated |title=知名翻译家杨德豫先生去世 |accessdate= 2013年1月23日|publisher= 凤凰网|work= |language=zh-cn}}</ref><ref name="shark " >{{cite web |url= http://www.china.com.cn/book/txt/2009-06/29/content_18030399.htm|title= 杨德豫：人鬼龙蛇八十秋|accessdate= 2009-06-29 |publisher=中国网 |work= |language=zh-cn}}</ref>1949年，杨德豫加入[[中国人民解放军|中国人民解放军]]。<ref name=" tiger" />更名为“江声”<ref name=" tiger2" >{{cite web |url=http://epaper.dfdaily.com/dfzb/html/2013-02/03/content_733311.htm |title= 怀念七伯杨德豫 |accessdate= 2013年2月3日|publisher= 东方早报|work= |language=zh-cn|id=[http://libdigital.umac.mo/was5/digital/academic_researchView.jsp?id=26336 備份]}}</ref>。1958年，[[反右|反右]]运动，[[上山下乡|上山下乡]]，他被打成[[右派|右派]]，被下放湖南大通湖农场。<ref name=" tiger" /><ref name="shark " />1978年，[[邓小平|邓小平]]执政之后，杨德豫被召回，担任[[湖南人民出版社|湖南人民出版社]]编辑。<ref name=" tiger" /><ref name="shark " />1990年，杨德豫担任[[湖南文艺出版社|湖南文艺出版社]]编辑。<ref name=" tiger2" /><ref name="shark " />2013年1月23日中午，杨德豫因[[食道|食道]]疾病在武汉161医院去世。<ref name="fish " />

== 风格 ==
杨德豫主要以翻译诗歌为主，将[[英文|英文]][[格律诗|格律诗]]翻译成中文格律诗。<ref name="fish " />

== 个人生活 ==
杨德豫有一女，叫杨小煜。<ref name="fish " />他与与[[卞之琳|卞之琳]]、[[屠岸|屠岸]]私交甚笃。<ref name="fish " />

杨德豫的作息时间有规律，清晨7点起床，晚上10点就寝，只看《[[新闻联播|新闻联播]]》和《[[天气预报|天气预报]]》，其他时间读书和翻译诗歌。<ref>{{cite web |url=http://news.sohu.com/20130124/n364511816.shtml |title=翻译大家杨德豫在汉去世 70多岁时还在背单词 |accessdate=2013年1月24日 |publisher=搜狐网 |work= |language=zh-cn}}</ref>
