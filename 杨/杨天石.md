'''杨天石'''{{noteTag|1= 杨天石的名字中“天石”两字的来历：南朝诗人徐陵小时候就很聪明，有人摸着他的脑袋说“此天上石麒麟也”。后来这就形成了一个典故——用“天上石麟”夸一个小孩特别聪明。}} （1936年2月15日—）笔名苏人、江东阳、吴之民、梁之彦等，[[江苏|江苏]][[东台|东台]]人，[[大陆|大陆]]著名[[现代史|现代史]][[史学家|史学家]]，[[中國大陆|中國大陆]]研究[[蒋介石|蒋介石]]第一人<ref>{{cite web |url=http://news.ifeng.com/history/zhuanjialunshi/yangtianshi/detail_2012_01/07/11816493_0.shtml|title=《蒋介石为何欣赏科学技术人才：这些领域他不懂》|accessdate=2012年1月7日|work=|publisher=凤凰网|language= zh-hans}}</ref>。

== 简介 ==
1955年，杨天石怀揣着[[作家|作家]]梦，从[[江苏|江苏]][[无锡|无锡]]考入[[北大|北大]]中文系。中学时，杨天石阅读了许多[[苏联|苏联]]作家的作品，最喜欢[[西蒙诺夫|西蒙诺夫]]和[[法捷耶夫|法捷耶夫]]。大学分专业时，他报了汉语言专业。<ref name="杨天石：追求信史和新知 ">{{cite web |url=http://finance.ifeng.com/opinion/xzsuibi/20100814/2513032.shtml|title=《杨天石：追求信史和新知》|accessdate=2010年8月14日|work=|publisher=凤凰网|language=zh-hans}}</ref>

1957年，大学里第二次分专业：语言专门化和文学专门化。杨天石填写个人志愿分到了文学专门化。在大学里，他因为写诗而受到打击，他把作品《我走了，故乡》和《让我再看一次你露水般的眼睛》交给诗社的刊物发表，但是在北大团委审稿的时候，认为这两首诗歌的感情不健康，于是封杀了。此后，杨天石就极少写诗，直到1988年被聘为[[中央文史馆|中央文史馆]]馆员，才恢复写诗。<ref name="杨天石：追求信史和新知 "/>

1958年，[[中国大陆|中国大陆]]搞学术批判，[[拔白旗、插红旗|拔白旗、插红旗]]。白旗是教授，并且是资产阶级白旗，而杨天石是学生中的白旗。在这年暑假，杨天石被分到了[[隋|隋]][[唐|唐]][[五代|五代]]文学组，并且开始撰写[[中国文学史|中国文学史]]。写完文学史初稿，文学组的领导觉得杨天石爱捣乱，就把他支到[[丰台|丰台]]桥梁厂编厂史。作家[[阿英|阿英]]看到文学史的初稿，提了一个意见，说[[清末|清末]]民初有一个叫[[南社|南社]]的文学团体要写进去。当时近代文学组已没有人力了，所以杨天石就被临时借调到[[近代文学|近代文学]]组，补写[[南社|南社]]这个章节。[[南社|南社]]研究让杨天石接触到[[清末|清末]]民初的一批[[作家|作家]]和[[历史|历史]]。[[中国文学史|中国文学史]]由[[人民文学出版社|人民文学出版社]]出版了，盛极一时，被认为是新军新史，革命的新生力量，但是此书连杨天石自己都不愿意看、引以为羞耻，此书却被[[康生|康生]]、[[陈毅|陈毅]]写信祝贺，还成为全国先进社会主义集体，又出了精装本，送到[[莱比锡|莱比锡]]的国际书展去展览。<ref name="杨天石：追求信史和新知 "/>

大学毕业后，杨天石在[[北京|北京]]南苑[[八一农业机械学校|八一农业机械学校]]工作。1962年的2月，[[北京师大附中|北京师大附中]]来挑教员，杨天石就到了师大附中当语文教师，一当就是16年，直到1978年到社科院近代史所。当杨天石在师大附中时，他一边教书，一边继续做业余研究。此时农村、城市开始“搞四清”，杨天石被内定为“新生的资产阶级知识分子”。<ref name="杨天石：追求信史和新知 "/>

1964年，批资开始，杨天石还在《新建设》杂志上发表了一篇关于[[明朝|明朝]][[泰州学派|泰州学派]]的文章，文章批评了历史家[[侯外庐|侯外庐]]等人的观点，引起争议。<ref name="杨天石：追求信史和新知 " />

70年代初，[[社科院近代史所|社科院近代史所]]已经恢复业务，决定编[[中华民国史|中华民国史]]，先编资料，其中一个题目叫“南社资料”。近代史所的[[王学庄|王学庄]]发现杨天石在“文革”前《新建设》杂志上发表过一篇文章，叫《论辛亥革命前的国粹主义思潮》，里面有一段专门讲南社。于是邀请杨天石加入。<ref name="杨天石：追求信史和新知 " />

1972年，杨天石出版了一本 《王阳明》，发行了30万2000册的印量。

1978年4月，杨天石正式调入近代史研究所民国史研究组（后改为室），从事专业研究工作。1979年评定为助理研究员。1983年，被授予近代史研究所先进工作者称号。1983年晋升为副研究员。1988年晋升为研究员。

1994年，杨天石被聘任为中国社会科学院研究生院博士生导师。
