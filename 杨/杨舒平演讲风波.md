'''杨舒平演讲風波'''是指2017年5月21日[[马里兰大学学院市分校|马里兰大学学院市分校]]毕业典礼上，来自[[中國大陸|中國大陸]][[昆明|昆明]]的留学生杨舒平在发表毕业演讲时，以[[美国|美国]]的学习、生活的自身经历為主軸的演講，因提及在昆明『每次外出都必须戴口罩』<ref>{{cite news|title=留学生演讲质疑家乡空气 昆明官微：这锅不背|url=http://3g.china.com/act/news/10000159/20170522/30554977.html|accessdate=2017-05-22|publisher=[[中華網|中華網]]|date=2017-05-22}}</ref>，以及将中美之间的空气、民主和言论自由作对比所引發的風波。

演讲内容由昆明、马里兰空气质量的对比，从中美城市环境的差异，呼吁为「民主、自由」的清新空气而奋斗，最后表达「自由是氧气，自由是激情，自由是爱」的观点。此講演內容引起中国的留學生群體以及中國媒體和民眾的強烈反應，在難以預料的輿論压力下，她在微博上道歉<ref name=globaltimes1>{{cite news|title=她说美国的空气“又甜又鲜”，台下美国人起立鼓掌，中国留学生气炸了！|url=http://world.huanqiu.com/article/2017-05/10712215.html|accessdate=2017-05-22|publisher=[[环球网|环球网]]（[[环球时报|环球时报]]）|date=2017-05-22|archiveurl=https://web.archive.org/web/20170522152217/http://world.huanqiu.com/article/2017-05/10712215.html|archivedate=2017-05-22}}}</ref><ref>{{cite news|title=马里兰大学中国留学生毕业演讲涉嫌辱华：我在美国吸到的空气都是甜的|url=http://www.sohu.com/a/142451883_170104|accessdate=2017-05-22|publisher=[[搜狐网|搜狐网]]|date=2017-05-22}}</ref>。

== 起因 ==
中国大陆媒體《[[环球时报|环球时报]]》認為演讲“诋毁祖国”、“融入美国”、“内容完全枉顾真的事实”和“天大的讽刺”<ref name=globaltimes1/>：
* 開頭以美国的新鲜空气為引子，称呼吸美国的空气有种“奇妙的奢侈感”；并与中国大陆的空气质量作了对比。
* 以空氣引出對美國大學學術自由的欣賞，将其比喻為新鲜空气; 提到美國總統大選，認同並欣賞自由民主價值觀和公民行使投票權利; 認同美國大學的批判性思维教育及多元文化，對师生表达政治观点的热情表示贊許，并与其在中国大陆学习《独立宣言》时的死记硬背作了对比；
* 以上方為鋪墊，總結全部內容並引出結尾。演讲末尾称“民主和自由是值得我们为之奋斗争取的新鲜空气”。

演讲结束后，主持毕业典礼的马里兰大学校长[[陆道逵|陆道逵]]称赞她说：“你的演讲非常鼓舞人心。我也是自己选择来到美国的移民，你的演讲也表达了我对这个国家最深刻的感受。”<ref name=zaobao1>{{cite news|author1=薛之白|title=热点综述：美国“甜美空气”让女留学生引爆辱华风波|url=http://www.zaobao.com.sg/news/china/story20170523-763426|accessdate=2017-05-23|publisher=[[联合早报|联合早报]]|date=2017-05-23}}</ref>

== 发展 ==
马里兰大学[[中国学生学者联合会|中国学生学者联合会]]在微信群呼吁学生们采取行动。前主席朱力涵發表言論: “希望这位小姑娘好自为之，脸丢光了的话在美国出门也得戴口罩了”，“以诋毁祖国的方式博眼球是坚决不能容忍的”；一些在中国的马里兰大学毕业生認為杨舒平的演讲是“一种为了融入美国社会的策略”；《环球时报》等媒体对此次演讲进行了大篇幅的宣传报道，並且表示其為辱華的行为<ref name="globaltimes1" />；在微博、微信等社交媒体平台上，杨舒平被打上了“辱华”、“卖国求荣”、“汉奸”的标签，还有网友表示杨舒平称其戴口罩是因为“人长得太丑”。事后，杨舒平迅速遭到了网友们的[[人肉搜索|人肉搜索]]，其个人信息被爆出<ref name="zaobao1" />，随即大量网友指责杨的演讲不实，因其家乡云南省[[昆明|昆明]]已经是中国大陆空气质量排行榜上少数几个空气质量优良率最好的城市之一<ref>{{cite news|author1=董宇虹|title=2014年昆明市环境状况公报发布 空气质量354天优良|url=http://www.weather.com.cn/yunnan/tqyw/06/2331785.shtml|accessdate=2017-05-31|agency=昆明日报|publisher=中国天气网|date=2015-06-05}}</ref><ref>{{cite news|author1=浦美玲|title=去年空气质量优良率为98.9% 昆明成为全国唯一进入排名前十的内陆城市|url=http://yn.yunnan.cn/html/2017-02/12/content_4726799.htm|accessdate=2017-05-31|agency=云报|publisher=云南网|date=2017-02-12|archive-url=https://web.archive.org/web/20170212045151/http://yn.yunnan.cn/html/2017-02/12/content_4726799.htm|archive-date=2017-02-12|dead-url=yes}}</ref>；有的自称杨高中同学的网友宣称杨并没有天天带口罩出門，表示杨的演讲并非事实。雖有部份評論並未將事實偏差放入考量範圍；也有评论表示昆明的空气质量比不上马里兰<ref>{{cite news|author1=横河|title=自由——从空气到言论|url=http://www.soundofhope.org/gb/2017/05/27/n866611.html|accessdate=2017-07-07|publisher=[[希望之声国际广播电台|希望之声国际广播电台]]|date=2017-05-27}}</ref>，甚至尚不如美国空气质量最差的[[休士顿|休士顿]]<ref>{{cite news|title=九鼎茶居：怎样读解杨舒平毕业演讲？|url=http://www.rfa.org/cantonese/features/teahouse/callin0605-06052017133628.html|accessdate=2017-07-07|publisher=[[自由亚洲电台|自由亚洲电台]]|date=2017-06-05}}</ref>，然而此演講的負面觀感已經在大陸網絡上形成。

在難以預料的巨大輿論壓力下，杨舒平在微博上发表致歉声明，表示将从事件中“吸取教训”<ref>{{cite news|author1=洪沙|title=中国留学生毕业演讲夸赞美国空气新鲜被批"辱华"|url=http://www.dw.com/zh/%E4%B8%AD%E5%9B%BD%E7%95%99%E5%AD%A6%E7%94%9F%E6%AF%95%E4%B8%9A%E6%BC%94%E8%AE%B2%E5%A4%B8%E8%B5%9E%E7%BE%8E%E5%9B%BD%E7%A9%BA%E6%B0%94%E6%96%B0%E9%B2%9C%E8%A2%AB%E6%89%B9%E8%BE%B1%E5%8D%8E/a-38942048?&zhongwen=simp|accessdate=2017-05-22|publisher=[[德国之声|德国之声]]|date=2017-05-22}}</ref><ref>{{cite news|title=【留美女生演講】嘲中國冇民主空氣差被轟辱華 受壓致歉：沒貶低國家|url=http://hk.apple.nextmedia.com/realtime/china/20170522/56726104|accessdate=2017-05-22|publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]]|date=2017-05-22}}</ref>；马里兰大学则发表声明，对杨舒平表示支持<ref name=zaobao1/>。而《[[大纪元时报|大纪元时报]]》的文章则质疑，为何大陆的负面评价都集中于'空气'污染程度真实与否，而对赞赏言论自由和公民投票权等只字不提，并质疑此次风波幕后操作者的真实目的。并有网友类比此次事件与二战后少数日本老兵由于'辱日'(对华道歉或揭露南京大屠杀)后在日本国内遭到的围攻谩骂，以及德国反思二战罪行后所赢得的世界尊重，指出杨舒平的演讲并不是辱华，而中国留学生的表现及官媒的评价才是真真正正的'辱华'，损害了中国在国际上的形象。<ref>http://www.epochtimes.com/b5/17/5/23/n9175415.htm</ref>

== 争议 ==
本演講美國學生的相關反應未知，然而在內容上存在以下爭議：
* 在毕业典礼的场合下，杨的演讲难以被在场中国学生接受。根据校方统计<ref name=isss>{{cite news|author1=ISSS|title=ISSS 2016 FALL STATISTICS|url=https://globalmaryland.umd.edu/sites/default/files/ies/Fall2016Stats.pdf|accessdate=2016|publisher=[[ISSS|ISSS]]|date=2016}}</ref>，近四年来马里兰大学录取的中国学生超过9000名。出席毕业典礼一部分中国学生难以接受毕业之际受到针对特定国籍的负面评价<ref name=tang>{{cite news|author1=唐迪|title=留美女毕业典礼谈〝自由〞遭围攻 演讲全文公开（视频）|url=http://www.ntdtv.com/xtr/gb/2017/05/23/a1325788.html|accessdate=2017-05-22|publisher=[[新唐人|新唐人]]|date=2017-05-22}}</ref>。

* 有论者认为杨的叙述与经验不符，以到达美国机场后准备戴上口罩的行为<ref name=tang>{{cite news|author1=唐迪|title=留美女毕业典礼谈〝自由〞遭围攻 演讲全文公开（视频）|url=http://www.ntdtv.com/xtr/gb/2017/05/23/a1325788.html|accessdate=2017-05-22|publisher=[[新唐人|新唐人]]|date=2017-05-22}}</ref>和对呼吸到甜美空气的深感意外<ref name=tang>{{cite news|author1=唐迪|title=留美女毕业典礼谈〝自由〞遭围攻 演讲全文公开（视频）|url=http://www.ntdtv.com/xtr/gb/2017/05/23/a1325788.html|accessdate=2017-05-22|publisher=[[新唐人|新唐人]]|date=2017-05-22}}</ref>，暗示普通的中国学生在前往美国留学之前可能对于国内的环境问题或对美国空气状况一无所知<ref name=aiwei>{{cite news|author1=艾薇资讯|title=杨舒平关闭Facebook，马里兰大学官方Twitter未回应Shuping Yang事件|url=http://www.aiweinews.com/archives/19398.html|accessdate=2017-05-23|publisher=[[艾薇资讯|艾薇资讯]]|date=2017-05-23}}</ref>，这与空气污染问题在中国受到广泛的关注和报道的实际情况并不相符<ref name=zhongguoxinwen>{{cite news|author1=中国新闻网|title=前4个月74城空气质量最差10城：河北占6席 邯郸最差|url=http://www.chinanews.com/gn/2017/05-16/8225012.shtml|accessdate=2017-05-06|publisher=[[中国新闻网|中国新闻网]]|date=2017-05-06}}</ref><ref name=yuxin>{{cite news|author1=李雨昕|title=京津冀及周边将出现空气污染过程 短时重度污染|url=https://www.sohu.com/a/141446729_123753|accessdate=2017-05-17|publisher=[[中国新闻网|中国新闻网]]|date=2017-05-17}}</ref> 。

* 中文媒体对演讲内容的批评还包括迎合大众对于中国环境问题和公民权利现状的刻板印象<ref name=xu>{{cite news|author1=徐一凡|title=马里兰大学中国留学生毕业演讲引发大争议|url=http://news.uschinapress.com/2017/0522/1106958.shtml|accessdate=2017-05-23|publisher=[[侨报网|侨报网]]|date=2017-05-23}}</ref>。以中国学生的形象通过对中国现状进行负面评价的方式赞赏美国的空气质量、言论自由和公民投票权<ref name=tang>{{cite news|author1=唐迪|title=留美女毕业典礼谈〝自由〞遭围攻 演讲全文公开（视频）|url=http://www.ntdtv.com/xtr/gb/2017/05/23/a1325788.html|accessdate=2017-05-22|publisher=[[新唐人|新唐人]]|date=2017-05-22}}</ref>。

==回應==
===中国外交部===
[[中華人民共和國外交部|中國外交部]]於2017年5月24日的外交部發言人例行記者會上對此事發表了回應。從全文來看，外交部的發言焦點，在於此事所反映出的對於如何在不造成負面效應的情況下發表講話內容的警示。回應如下：<ref name=ministryofforeignaffairs>{{cite news|author1=[[中華人民共和國外交部|中華人民共和國外交部]]|title=2017年5月24日外交部发言人陆慷主持例行记者会|url=http://www.fmprc.gov.cn/web/wjdt_674879/fyrbt_674889/t1464839.shtml|accessdate=2017-05-24|publisher=[[中華人民共和國外交部|中華人民共和國外交部]]|date=2017-05-24}}</ref>

{{Squote|問：中國留學生楊舒平週日在美國馬里蘭大學發表了一篇有爭議的演講，在中國國內引發憤怒情緒。中國政府是否認為中國留學生、特別是赴美留學生有義務發表負責任的涉華言論？中國駐美使館是否介入此事？中國政府是否就中國留學生能說什麼、不能說什麼發布指導性意見？

答：關於第一個問題，我想說，任何中國公民、對於任何事的表態都應該負責任，不僅是有關中國的問題，對任何問題都應該如此。

關於你提到中國駐美使領館是否介入此事，以及中國政府是否因此發布針對留學生的指導性意見，我沒有聽說這樣的情況。

事實上，你的問題不是一個外交問題，但既然你問到了，網上也有很多評論，我可以說兩句。我想可能很多網民都認為，任何國家都有自己值得驕傲的一面，同時每個國家在發展過程中也都存在一些發展中的問題，一個公民在評論自己國家時是在什麼樣的場合、以什麼樣的方式作評論，相信所有人都不難從中感受出他/她對祖國是什麼樣的感情。

我也注意到，你提到的這位學生後來又作出公開表態、或者說是一個道歉。她說自己熱愛祖國，也願意在學成之後回國為祖國做出自己的貢獻。如果這一消息屬實，我想說，當前中國在國外有很多留學生，這些年輕學子出國以後對一些事情的看法和認識可能都會經歷一個發展和變化的過程。但只要他們最終還是從心底熱愛自己的國家，願意為祖國做出貢獻，我相信中國政府是鼓勵、支持和歡迎。}}
