'''杨洁篪'''<small>（[[普通话|普通话]]：{{lang|la|{{Audio|杨洁篪 (普通话).ogg|Yáng Jiéchí}}}}，「[[篪|篪]]」字指一種[[笛子|笛子]]）</small>（{{bd|1950年|5月1日|}}）是一名[[中华人民共和国|中华人民共和国]]职业[[外交官|外交官]]和[[中国共产党|中国共产党]]高级[[党和国家领导人|领导人]]。他來自[[上海人|上海]]，於[[南京大学|南京大学]][[历史学|历史学]]系[[世界史|世界史]]专业毕业，後来成為一名[[在职研究生|在职研究生]]，為历史学[[博士|博士]]。现任[[中国共产党中央政治局|中共中央政治局]]委員，[[中央外事工作委员会|中央外事工作委员会]]秘书长兼[[中央外事工作委员会办公室|办公室]]主任，[[中央對臺工作領導小組|中央對臺工作領導小組]]秘书长，[[中央港澳工作协调小组|中央港澳工作协调小组]]副組長。曾经担任[[中华人民共和国|中华人民共和国]][[国务委员|国务委员]]、[[中华人民共和国国务院_(第十二届)|第十二届国务院]]党组成员<ref>{{Cite web|url=http://news.xinhuanet.com/2013lh/2013-03/16/c_115050223.htm|title=会议投票表决决定各部部长、各委员会主任、中国人民银行行长、审计长、秘书长|accessdate=2018-03-29|date=2013-03-16|publisher=新华网}}</ref>。

1968年9月参加工作，1971年12月加入[[中国共产党|中国共产党]]，是[[中国共产党第十六届中央委员会候补委员列表|中共第十六届中央候补委员]]，[[中国共产党第十七届中央委员会委员列表|第十七]]、[[中国共产党第十八届中央委员会委员列表|十八届中央委员]]。曾经历任[[中华人民共和国外交部|中华人民共和国外交部]]部长助理兼[[中华人民共和国外交部北美大洋洲司|中华人民共和国外交部北美大洋洲司]]司长、副部长、[[中国驻美国大使列表|驻美国大使]]。外交部常务副部长、[[中国外交部长列表|部長]]。

在其职业外交生涯中，曾经三度被派往海外工作，并且每次都派驻[[美国|美国]]，前后在[[中国驻美国大使馆|中国驻美国大使馆]]工作过12年，与布什家族建立了良好关系。有媒体披露，布什家族在某些情况下，会选择与杨洁篪私下沟通，作为與中国政府沟通的管道，杨洁篪在[[美国外交|美国外交]]场上的[[绰号|绰号]]“'''老虎杨'''”（{{lang|en|Tiger Yang}}），就是[[美国总统|美国前总统]][[乔治·赫伯特·沃克·布什|老布什]]取的。

== 早年经历 ==
1963年，杨洁篪考入[[上海外国语大学附属外国语学校|上海外国语大学附属外国语学校]]，与日后的工作搭档[[王光亚|王光亚]]是室友。受[[文化大革命|文化大革命]]的影响，于1968年辍学后，进入上海浦江电表厂当工人。在中美正式建交之前，外交部挑选一批外语方面的人才前往[[英国|英国]]进修；杨洁篪被选中，曾先后就读于[[巴斯大学|巴斯大学]]和[[倫敦政治經濟學院|伦敦经济学院]]；与其同期被派赴英国进修的，还有后来成为杨洁篪夫人的乐爱妹、中学室友[[王光亚|王光亚]]及其夫人陈姗姗，以及[[沙祖康|沙祖康]]、[[周文重|周文重]]、[[龙永图|龙永图]]等人。

1975年杨洁篪毕业后回国，被分配到[[中华人民共和国外交部|外交部]]翻译室担任专职翻译。1977年，[[乔治·赫伯特·沃克·布什|老布什]]应邀访问中国时，杨洁篪被派为全程陪同翻译。<ref>{{Cite news|url=http://news.bbc.co.uk/chinese/simp/hi/newsid_6590000/newsid_6598200/6598229.stm|title=杨洁篪接替李肇星出任中国外长|date=2007-04-27|work=BBC中文网|accessdate=2018-03-29}}</ref>

== 外交生涯 ==
[[File:Secretary_Hillary_Clinton_Meets_With_Chinese_Foreign_Minister_Yang_Jiechi.jpg|thumb]][[希拉里·克林顿|希拉里]]]]

1983年起，杨洁篪历任[[中国驻美国大使馆|中国驻美国大使馆]]二秘、一秘、[[参赞|参赞]]，[[中华人民共和国外交部北美大洋洲司|北美大洋洲司]]参赞、兼处长、副司长。1993年任驻美国使馆[[公使|公使]]，1995年任外交部部长助理兼美大司司长。1998年2月，48岁的杨洁篪升任[[中华人民共和国外交部|外交部]]副部长。

2001年2月16日，时任[[中华人民共和国主席|中华人民共和国主席]][[江泽民|江泽民]]根据[[全国人民代表大会常务委员会|全国人大常委会]]的决定，任命51岁的杨洁篪接替[[李肇星|李肇星]]为中华人民共和国第七任[[中国驻美国大使|驻美国大使]]。这也是[[中美关系|中美建交]]22年来，中国派出的最年轻的大使，比其前任[[李肇星|李肇星]]年轻了10岁。

2004年12月，杨洁篪任满回国，再次出任外交部副部长。外界认为他是下一任外长的有力竞争者，但是有媒体爆出其健康状况不佳的消息。报道称杨洁篪心脏不好，曾经做过[[冠状动脉旁路移植|搭桥手术]]。2006年，杨洁篪获得[[南京大学|南京大学]][[历史学|历史学]]系[[世界歷史|世界史]]专业[[博士|博士]]学位。

2007年4月，[[第十届全国人民代表大会常务委员会|第十届全国人大常委会]]第27次会议决定，根据[[温家宝|温家宝]]总理提名，杨洁篪接替已到退休年齡的[[李肇星|李肇星]]任[[中国外交部长列表|外交部长]]，2008年3月获得连任。

== 中央高层 ==
2013年3月16日，[[第十二届全国人民代表大会#第一次会议|第十二届全国人大第一次会议]]决定，根据[[李克强|李克强]]总理的提名，任命杨洁篪为[[国务委员|国务委员]]，成为[[党和国家领导人|党和国家领导人]]，[[国务院台湾事务办公室|国务院台湾事务办公室]]原主任[[王毅|王毅]]接任外交部长。同年8月起，杨洁篪兼任[[中央外事工作委员会办公室|中央外事工作领导小组办公室]]（中央外事办，2018年3月改组为中央外事工作委员会办公室）主任。

2017年10月25日，杨洁篪当选为[[中国共产党第十九届中央委员会委员列表|中国共产党第十九届中央委员]]、[[中国共产党中央政治局|中央政治局]]委员，分管外交、侨务、台湾、港澳事务，2018年3月19日卸任[[国务委员|国务委员]]职务。

== 评价 ==
“温文尔雅、有儒家风范”这是中國大陸外交业人士对杨洁篪的性格评价。2001年4月，就在杨洁篪出任驻美大使不久，[[中美南海撞机事件|中美南海撞机事件]]发生，他曾经一日之内四次奔走于[[美国国务院|美国国务院]]、[[美国国会|国会]]与[[中国驻美国大使馆|中国驻美国大使馆]]之间，表达中方的立场，全力化解国会方面对中国的反弹敌意。还曾经一天两次出现在美国电视上，阐述中方看法。这场危机最终得以化解，而杨洁篪也赢得了个人威望。

美国前总统[[乔治·赫伯特·沃克·布什|老布什]]称杨洁篪为“老虎杨”（Tiger Yang）。原因有两个：其一，杨洁篪是1950年出生的，属[[虎|虎]]。其二，他名字中的“[[篪|篪]]”字里正好藏了一个“虎”字。
