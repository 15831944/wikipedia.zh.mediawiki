'''杨春光'''（{{bd|1961年|3月||}}）[[安徽|安徽]][[阜阳|阜阳]]人，[[中国共产党|中国共产党]]及[[中华人民共和国|中华人民共和国]]官员。

==生平==
1982年11月加入[[中国共产党|中国共产党]]，1978年11月参加工作。1978年11月自[[中国民用航空飞行学院|中国民用航空飞行专科学校]]飞机驾驶专业毕业，[[中央党校|中央党校]]大学学历。曾任民航北京管理局油料处化验员、团支部书记（1983年3月到1985年3月为[[民航管理干部学院|民航管理干部学院]]党政干部专业学员），[[中国民航局|中国民航局]]政治部干部、副科级干事，[[中共中央组织部|中共中央组织部]]干部调配局副科级干事、正科级干事，中共中央组织部党政外事干部局正科级干事（其间1989年8月到1992年6月在中央党校函授学院经济管理专业学习）、副处长（其间1996年12月到1998年11月在[[中国社会科学院研究生院|中国社会科学院研究生院]]投资管理专业研究生课程班学习）、正处级调研员，中共中央组织部干部三局处长、副局级调研员、副巡视员、副局长（其间2013年5月到2014年2月任[[中央党的群众路线教育实践活动领导小组|中央党的群众路线教育实践活动领导小组]]办公室联络二组组长），中共中央组织部干部监督局巡视员兼部巡视工作联络办公室主任、[[中央巡视工作领导小组|中央巡视工作领导小组]]办公室副主任。2016年10月任[[国家公务员局|国家公务员局]]党组成员、副局长<ref>{{Cite web|url = http://www.scs.gov.cn/jldjs/fjz/201611/t20161103_6028.html|title = 杨春光同志简历|author = |date = 2016-11-03|publisher = 国家公务员局|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20180203211113/http://www.scs.gov.cn/jldjs/fjz/201611/t20161103_6028.html|archive-date = 2018-02-03|dead-url = yes}}</ref><ref>{{Cite web |url =http://renshi.people.com.cn/n1/2016/1103/c139617-28832113.html  |title =朱建平、张义全、杨春光任国家公务员局副局长  |author =  |date = 2016-11-03 |publisher = 中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。2018年4月任[[国家邮政局|国家邮政局]]副局长。<ref>{{Cite web |url = http://www.gov.cn/xinwen/2018-04/03/content_5279557.htm |title =国务院任免国家工作人员（2018年4月3日）|author =  |date = 2018-04-03|publisher=中国政府网|language =  |accessdate =  |quote =  }}</ref>
