==生平==
1988年7月加入中国共产党，1989年7月参加工作。1986年9月，考入[[山西大学|山西大学]]哲学系[[马克思主义哲学|马克思主义哲学]]专业学习。同时选修物理系理论物理专业课程。1989年7月提前毕业获学士学位，并考取山西大学哲学系硕士研究生。1989年7月，到[[山西省|山西省]][[长治县|长治县]]苏店中学任教。1990年9月，回山西大学攻读硕士学位研究生，研究方向是马克思主义哲学[[认识论|认识论]]。1993年9月，考入中共中央党校攻读哲学博士学位，师从[[崔自铎|崔自铎]]，研究方向是认识论<ref name=yangshzh/>。

1996年7月自中共中央党校研究生部毕业，获[[哲学博士|哲学博士]]学位，被分配到[[中共中央办公厅调研室|中共中央办公厅调研室]]工作。1997年3月，到[[山东省|山东省]][[临沂市|临沂市]][[罗庄区|罗庄区]]罗庄镇挂职锻炼一年，任镇党委副书记。1998年2月，回中共中央办公厅调研室工作。2009年4月起，任中共中央办公厅调研室五组组长。2013年起，任中共中央办公厅调研室综合调研组组长<ref name=yangshzh/>。

2015年4月调离中共中央办公厅调研室。此后任[[国家互联网信息办公室|中央网络安全和信息化领导小组办公室]]（国家互联网信息办公室）政策法规局局长兼[[中国网络空间研究院|中国网络空间研究院]]院长<ref>{{Cite web |url = http://www.81.cn/2017wlmtlt/2017-11/24/content_7840992.htm |title = 杨树桢 |author =  |date =2017-11-24  |publisher =中国军网  |language =  |accessdate =  |quote =  }}</ref>。
