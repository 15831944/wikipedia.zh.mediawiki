'''杨魏玲花'''（{{bd|1980年|12月20日}}），[[内蒙古自治区|内蒙古自治区]][[鄂尔多斯|鄂尔多斯]][[蒙古族|蒙古族]]人，歌手。

==名字==
“杨魏玲花”是一个[[蒙古族人名|蒙古族人名]]的汉字音译，意思是“天上的彩云”。她并不姓杨或者姓魏。

==生平==
杨魏玲花青少年时期在[[内蒙古|内蒙古]]大草原上成长，出名后{{who|time=2017-03-28|被誉为}}新一代的“[[乌兰牧骑|乌兰牧骑]]”。她曾是在[[广州|广州]]打工的一名电器[[销售员|销售员]]，并在此其间结识[[曾毅|曾毅]]二人组成演唱组合。<ref name="杨魏玲花老公曝光 风流倜傥的低调CEO ">{{cite web |url=http://www.zhiyin.cn/2011/0321/117903.html|title=《杨魏玲花老公曝光 风流倜傥的低调CEO》|accessdate=2011-03-21|work=|publisher=知音网|language=zh-hans}}</ref>

=== 感情生活 ===
杨魏玲花的丈夫[[徐明朝|徐明朝]]是[[百人娱乐|百人娱乐]]的[[CEO|CEO]]。两人于2011年3月在内蒙古[[鄂尔多斯|鄂尔多斯]]结婚。杨魏玲花的密友[[李玉刚|李玉刚]]、[[刀郎_(歌手)|刀郎]]、[[徐千雅|徐千雅]]等出席婚礼为玲花庆祝。<ref name="杨魏玲花老公曝光 风流倜傥的低调CEO " />

== 风格 ==
杨魏玲花的歌声高亢嘹亮而激情四射，富有草原豪壮气势，歌声有着强烈的感染力，带领人走进神秘的草原世界，张扬出生生不息的原始生命力。<ref name="杨魏玲花老公曝光 风流倜傥的低调CEO " />

== 奖项 ==
*2004年第十一届[[全国青年歌手电视大奖赛|全国青年歌手电视大奖赛]]专业组第七名。
