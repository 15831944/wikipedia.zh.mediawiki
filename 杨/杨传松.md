'''杨传松'''（{{bd|1957年|10月||}}）[[山东|山东]][[泰安|泰安]]人，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
1976年3月参加[[中国人民解放军|中国人民解放军]]。历任战士，干事，教导员，团政治处主任，团政治委员，旅政治委员，[[中国人民解放军火箭军总医院|第二炮兵总医院]]政治委员。2009年任[[中国人民解放军火箭军研究院|第二炮兵装备研究院]]政治委员。2013年7月任[[中国人民解放军火箭军第六十七基地|第二炮兵第二十二基地]]政治委员。2013年10月任[[中国人民解放军火箭军工程大学|第二炮兵工程大学]]政治委员。2013年12月任[[中国人民解放军总参谋部军训部|总参谋部军训部]]政治委员。2014年12月任[[中国人民解放军总参谋部政治部|总参谋部政治部]]主任。2016年1月任[[中央军委联合参谋部政治工作局|中央军委联合参谋部政治工作局]]主任。2017年6月任[[中央军委联合参谋部|中央军委联合参谋部]]参谋长助理，2018年卸任。他是[[中共十九大代表|中共十九大代表]]，[[第十三届全国政协委员|第十三届全国政协委员]]<ref name=xieren>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_2025805|title =杨传松少将卸任军委联合参谋部参谋长助理职务  |author =  |date = 2018-03-12 |publisher =  澎湃新闻|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://army.81.cn/content/2017-07/04/content_7662487.htm |title =推动军队政法工作创新发展，他们怎么说  |author =  |date =2017-07-04  |publisher = 中国陆军网 |language =  |accessdate =  |quote =  }}</ref>。

2010年7月晋升少将军衔<ref name=xieren/>。
