'''杨崇勇'''（{{bd|1955年|11月||catIdx=Y杨}}），[[云南省|云南]][[昆明市|昆明]]人。男，[[满族|满族]]。1971年4月参加工作，1980年8月加入[[中国共产党|中国共产党]]。[[云南大学|云南大学]]中文系汉语言文学专业毕业，大学学历。[[中国共产党第十八届中央委员会候补委员列表|中国共产党第十八届中央委员会候补委员]]。
== 任职云南 ==
曾长期在云南省任职，历任中共[[云南省|云南省]][[宜良县|宜良县]]委副书记、书记；[[云南省人民政府|云南省人民政府]]副秘书长、秘书长；中共[[玉溪市|玉溪]]地委书记，中共玉溪市委书记、市长；[[中国共产党云南省委员会|中共云南省委]]常委、省委秘书长、中共[[昆明市|昆明市]]委书记<ref>{{cite web|url=http://news.sohu.com/20060914/n245329338.shtml|title=杨崇勇任昆明市委书记|accessdate=2012-04-20|date=2006-09-14|publisher=[[搜狐|搜狐]]}}</ref>等职。
== 任职河北 ==
2007年12月，调任[[中国共产党河北省委员会|中共河北省委]]常委<ref>{{cite web|url=http://www.clzg.cn/dssbpap/2007-12/29/content_1121949.htm|title=仇和任省委常委、昆明市委书记　杨崇勇调任河北省委常委|accessdate=2012-04-20|date=2007-12-29|publisher=都市时报|archiveurl=https://web.archive.org/web/20130602213420/http://www.clzg.cn/dssbpap/2007-12/29/content_1121949.htm|archivedate=2013年6月2日|deadurl=yes|df=}}</ref>，2008年1月当选[[河北省人民政府|河北省人民政府]]副省长、党组副书记<ref>{{cite web|accessdate=2012-04-20|url=http://news.sina.com.cn/c/2008-01-21/020613295031s.shtml|title=杨崇勇任河北省副省长|publisher=[[新浪网|新浪网]]}}</ref>。2011年12月，任中共河北省委常委、[[河北省人民政府|河北省人民政府]]常务副省长，兼任河北行政学院院长。2016年1月，辞去河北省人民政府副省长职务<ref>{{cite web|accessdate=2016-01-05|url=http://district.ce.cn/newarea/sddy/201601/05/t20160105_8073627.shtml|title=杨崇勇辞去河北省副省长职务|publisher=中国经济网}}</ref>，当选为河北省第十二届人大常委会副主任、党组书记<ref>{{cite web|url=http://district.ce.cn/newarea/sddy/201601/13/t20160113_8236167.shtml|title=杨崇勇当选河北省人大常委会副主任 接替宋恩华任党组书记|accessdate=2016-01-13|publisher=中国经济网}}</ref>。
== 落马 ==
2017年4月11日，[[中央纪委国家监委网站|中纪委监察部网站]]发布消息，河北省人大常委会党组书记、副主任杨崇勇涉嫌严重违纪，正接受组织审查<ref>{{cite web|url=http://www.ccdi.gov.cn/xwtt/201704/t20170411_97161.html|title=河北省人大常委会党组书记、副主任杨崇勇涉嫌严重违纪接受组织审查|accessdate=2017-04-11|publisher=中央纪委监察部网站}}</ref>。

杨的落马使[[昆明市|昆明市]]追平了[[淮南市|淮南市]]创造的连续四任市委书记落马的全国纪录（杨崇勇、[[仇和|仇和]]、[[张田欣|张田欣]]、[[高劲松|高劲松]]）。7月4日，中纪委通报杨被双开，移送司法机关<ref>{{cite web |url = http://www.ccdi.gov.cn/jlsc/zggb/djcf_zggb/201707/t20170704_102217.html |title = 河北省人大常委会原党组书记、副主任杨崇勇严重违纪被开除党籍和公职 |accessdate = 2017年7月4日 |publisher = 中央纪委监察部网站}}</ref>。

2018年6月12日，杨崇勇受贿一案在[[天津市第二中级人民法院|天津市第二中级人民法院]]开庭审理，检方认定，杨崇勇利用职务之便，为他人提供便利，收受贿赂共计折合人民币2.06亿元。杨崇勇当庭表示认罪悔罪<ref>{{Cite web|url=http://news.163.com/18/0612/17/DK48NOT30001899O.html|title=河北人大常委会原副主任杨崇勇一审被指受贿2.06亿|accessdate=2018-06-12|date=2018-06-12|publisher=网易}}</ref>。该金额相当于前[[党和国家领导人|党和国家领导人]][[周永康|周永康]]、[[令计划|令计划]]受贿金额的总和<ref>{{Cite web|url=http://news.sina.com.cn/c/2018-06-12/doc-ihcwpcmp6666006.shtml|title=周永康令计划收的钱加起来 才跟这个官员一样多|accessdate=2018-06-12|date=2018-06-12|publisher=新浪}}</ref>。

[[天津市第二中级人民法院|天津市第二中级人民法院]]于2018年9月27日公开宣判杨崇勇受贿案，杨崇勇以受贿罪被判处[[无期徒刑|无期徒刑]]，[[剥夺政治权利|剥夺政治权利]]终身，并处没收个人全部财产；对杨崇勇受贿所得财物及其孳息予以追缴，上缴国库，不足部分，继续追缴<ref>{{cite web|url=http://www2.hkej.com/instantnewsMob/current/article/1954464/河北人大常委原副主任楊崇勇受賄囚終身|title=河北人大常委原副主任楊崇勇受賄囚終身|date=2018-09-27|accessdate=2018-09-27|publisher=信报}}</ref><ref>{{cite web|url=http://www.chinanews.com/wap/detail/zw/gn/2018/09-27/8637751.shtml|title=河北省人大常委会原副主任杨崇勇受贿超2亿被判无期|date=2018-09-27|accessdate=2018-09-27|publisher=中国新闻网}}</ref>。
