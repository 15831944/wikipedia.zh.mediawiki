'''杨宇军'''（{{bd|1970年|||}}），[[北京|北京]]人，[[中国传媒大学|中国传媒大学]]媒介与公共事务研究院院长，[[中华人民共和国国防部|中华人民共和国国防部]]新闻局原局长，退役前为[[中国人民解放军|中国人民解放军]]大校<ref name=gfb>{{Cite web |url = http://www.mod.gov.cn/info/2017-02/23/content_4621521.htm |title = 国防部新闻发言人杨宇军大校 |author =  |date =2017-02-23   |publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref><ref name=ifeng>{{Cite web |url = http://news.ifeng.com/a/20141118/42505583_0.shtml |title = 国防部发言人杨宇军晋升大校军衔 系70后 |author =  |date = 2014-11-18 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1970年，杨宇军生于北京。1993年自[[外交学院|外交学院]]毕业后参加中国人民解放军。历任学员、参谋、中华人民共和国国防部外事办公室综合局政策研究室主任等职。曾在[[英国|英国]][[伦敦玛丽王后大学|伦敦玛丽王后大学]]学习，获得公共政策专业硕士学位。2010年4月，任中华人民共和国国防部新闻事务局副局长、国防部新闻发言人，同时晋升上校军衔。2014年11月，晋升大校军衔。2015年6月升任国防部新闻事务局局长、国防部新闻发言人。2016年1月改任国防部新闻局局长、国防部新闻发言人<ref name=gfb/><ref name=ifeng/>。

2017年8月9日，杨宇军以“自主择业”的方式退役。杨宇军在接受《[[环球时报|环球时报]]》采访时表示：“经过慎重思考，确实很难做到家庭事业两头兼顾，最终决定拿出更多时间陪伴家人。”<ref>{{cite news|title=国防部新闻发言人变更:吴谦升任局长 杨宇军退役|url=http://military.people.com.cn/n1/2017/0809/c1011-29460123.html|accessdate=2017-08-10|publisher=人民网|date=2017-08-09}}</ref>

2018年3月29日，杨宇军出任中国传媒大学媒介与公共事务研究院院长<ref>{{cite news|author1=郭媛丹|title=国防部原新闻发言人杨宇军证实履职中国传媒大学媒介与公共事务研究院院长|url=http://china.huanqiu.com/article/2018-03/11718581.html|accessdate=2018-03-29|publisher=环球网|date=2018-03-29}}</ref>。
