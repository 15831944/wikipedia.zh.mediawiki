'''泉州广播电视台新闻综合频道'''，又名“泉州台一套”、“泉视一套”。是[[泉州市|泉州市]]的第一个自办[[电视频道|电视频道]]，旧称'''泉州电视台'''。1988年春节试播，1988年9月1日正式开播。亦是1991年12月31日，[[泉州广播电视台都市生活频道|泉州有线电视台]]开播前，泉州市区唯一的自办电视频道。

== 历史 ==
*1986年5月，泉州市决定筹办新闻综合频道前身“泉州电视台”。
*1987年7月1日，第一次试播《泉州新闻》
*1988年9月1日，正式开播，发射功率为1[[千瓦|千瓦]]。初期每周播出7次，时间5[[小时|小时]]。初期的自办节目有：《泉州新闻》、《侨乡周末》、《交通警报台》等
*1990年12月，自办播出时间增加至3小时/日。<ref>[http://www.fjsq.gov.cn/frmBokkList.aspx?key=14F77A06AE1A4E36BD629D1564C20AC3 福建广播电视志 泉州电视台篇]</ref>
*1994年10月1日，《周末报道》、《一周要闻》开播。同日，《泉州新闻》首播由每周三次调整为每日播出，并采用普通话及[[闽南语|闽南语]]交叉播出。<ref>[http://www.fjsq.gov.cn/frmBokkList.aspx?key=6010431E510E452895BFE0E49E89066D 《福建新闻志》4-7 泉州电视台]</ref>
*1996年11月，美食类节目《学烧泉州菜》开播。后于1998年7月，更名为《泉州美食》并沿用至今。
*1997年11月1日，第一个民生新闻栏目[[新闻广角|《新闻广角》]]，以及评讲类节目《泉州讲古》开播，同日起，《一周要闻》停播。<ref>《泉州电视台 建台10周年》第13页-泉州电视台建台十年大事记</ref>
*1997年12月28日，该频道迁入广电路1号（现称 田淮街66号）泉州广电大厦播出。
*2001年7月1日，兼并泉州有线电视台后，频道呼号由“泉州电视台”更名为“泉州电视台1套”。<ref>{{Cite web |url=http://www.qzt.gov.cn/szfzg/nj/201605/t20160504_3732.htm |title=《泉州年鉴2002》第747页 |access-date=2018-01-13 |archive-url=https://web.archive.org/web/20171014233646/http://www.qzt.gov.cn/szfzg/nj/201605/t20160504_3732.htm |archive-date=2017-10-14 |dead-url=yes }}</ref>同年，《法治连线》开播，交通警报台栏目并入法制连线。<ref>[http://www.xueshu.com/xwyjdk/201617/28446207.html 地方台法制节目创作瓶颈与改善策略——以泉州广播电视台《法治连线》栏目为实例]</ref>
*2002年4月15日，《泉州新闻》播出时间延长至15分钟/次，并开播《方言报道》，初期播出时间为12:00<ref>[http://www.qzwb.com/gb/content/2002-04/15/content_462481.htm 泉州电视台今起调整节目]</ref>
*2002年9月30日，泉州电视台1套更名为新闻综合频道。并开办《泉视早新闻》，后更名为《新闻早报》。
*2004年1月5日，开办晚间新闻栏目《新闻茶座》，初期播出时间为22:00。
*2004年8月1日，开办午间新闻栏目《午间新闻网》，方言报道播出时间因此调整至12:40，原新闻茶座更名为《[[今晚播报|今晚播报]]》，播出时间调整至22:30。<ref>[http://news.sohu.com/20040926/n222246962.shtml 泉州电视台推出7档新闻节目 市民DV作品登上荧屏]</ref>
*2012年8月16日，因泉州电视台更名为泉州广播电视台，频道呼号再次更名为“泉州广播电视台新闻综合频道”<ref>[http://www.chinanews.com/qxcz/2012/08-16/4112362.shtml 侨乡泉州整合成立广播电视台]</ref>
*2015年6月1日，更换为现有台标，频道呼号格式变更为“QZTV-1”。7月，《方言报道》停播，原播出时段由《新闻相拍报》替代。
*2018年4月12日，因二套《天使爱美丽》停播。新闻综合频道亦取消重播该节目，收台时间提前至1:40。

== 播出 ==

*开台
该频道每日6:00开台，6时整自动播放约2[[分钟|分钟]]的开台曲，开台曲播放完毕显示由白底和台标组成的“泉州广播电视台”版权页

周一至周六，开台曲播出后的第一个节目是重播前一天傍晚的《泉州美食》（闽南语节目），而周日则是重播本周的《刺桐花》（少儿节目）。
*收台
该频道收台时间不固定，由于每日19:00转播[[中央电视台|中央电视台]][[新闻联播|新闻联播]]的缘故，其后的各档夜间节目播出时间，受新闻联播时长的影响较大

一般情况下，该频道收台时间大约在次日1:30-2:00之间，若因新闻联播时间延长，其后的各档节目播出时间皆会延迟。这种情况下，收台时间将延迟至次日2:30-3:00。

每日收台前的最后一个节目为《新闻广角》，该节目播出完毕后，切入时长约1分钟的收台曲。收台曲播放完毕后，亦显示约3秒左右的版权页，然后切换为[[PM5544|PM5544]]型[[测试卡|测试卡]]
*检修
该频道每周二下午有例行的停机检修时段。持续时间约2小时。

14:00前后，重播《泉州财经报道》结束，播放时长约5分钟的广告。检修开始前，播放频道宣传动画，并显示新闻综合频道专属的蓝色版本版权页。

尔后检修开始，同收播一样，检修期间的测试卡也是PM5544。该频道停机检修时间较短，大约持续1小时55分钟左右。16:00检修结束，开台后继续播放广告、电视剧
