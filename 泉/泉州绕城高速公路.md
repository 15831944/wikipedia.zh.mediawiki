'''泉州绕城高速公路'''（原编号：'''G15<sub>02</sub>'''）,是环绕[[中国|中国]][[泉州市|泉州市]]的一条高速公路。原称'''泉州环城高速公路'''。
[[File:Nanhui_EXPWY_of_Zayton_city.jpg|300px]]

== 概况 ==
*泉州绕城高速由五段组成，分别是：'''南惠高速'''、'''南石高速'''、'''[[泉州湾跨海大桥|泉州湾跨海大桥]]'''以及'''晋石高速'''、'''泉厦漳城市联盟高速'''泉州段组成。目前除泉厦漳城市联盟高速泉州段正在施工外，其余各段均已通车。
*泉州绕城高速分为小环及大环。小环与大环的区别在于，小环与大环在西锦互通分离。小环在[[沈海高速|沈海高速]]西锦互通至内坑互通间，与沈海高速共线，通过内坑互通转入晋石高速。而大环过西锦互通后，继续南下，在石井接入泉厦漳城市联盟高速，与联盟高速共线至樟井互通后，再转入晋石高速。

== 历史 ==
*2005年9月20日，泉州市召开市环城高速公路工程预可行性研究报告审查会<ref>{{cite web|url=http://hc360.com/2005/09/21094467316.shtml|title=串起大泉州 百亿打造环城高速公路|accessdate=2005-09-21}}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
*2006年11月7日，省发改委批复南惠高速（原名 泉三高速泉州支线）可行性报告<ref>{{cite web |url=http://news.sina.com.cn/o/2006-11-08/000010439050s.shtml|title=南安至惠安斗尾将通高速|accessdate=2006-11-08}}</ref>
*2007年12月28日，绕城高速一期，南惠高速奠基。标志着泉州绕城高速正式开工。<ref>{{cite web |url=http://news.sina.com.cn/c/2007-12-29/081213165128s.shtml|title=泉州环城高速动工|accessdate=2007-12-29}}</ref> 
*2009年12月31日，绕城高速二期，晋石高速开工。<ref>{{cite web |url=http://www.chinanews.com/cj/news/2010/01-01/2049956.shtml|title=泉州“百大”重点项目开（竣）工仪式迎新年|accessdate=2010-01-01}}</ref> 
*2010年8月，绕城高速南安-石井段开工。<ref>{{cite web |url=http://news.sina.com.cn/c/2010-06-05/160317615192s.shtml|title=泉州环城高速公路建设全速展开 南石高速8月动工|accessdate=2010-06-05}}</ref> 
*2011年7月19日，南惠高速建成通车<ref>
{{cite web|url=http://news.66163.com/2011-07-19/534563.shtml|title=南惠高速公路今日正式通车 南安到惠安小车20元|accessdate=2011-7-19}}{{Dead link|date=2019年3月 |bot=InternetArchiveBot |fix-attempted=yes }}
</ref>
*2011年8月8日，绕城高速三期，泉州湾大桥开工。<ref>{{cite web |url=http://qz.fjsen.com/2011-08/09/content_5526820_2.htm|title=泉州湾跨海大桥主桥工程昨日开工 工期约三年|accessdate=2011-08-09}}</ref> 
*2012年12月31日，晋石高速建成通车。<ref>{{cite web |url=http://news.163.com/12/1231/07/8K1MSENS00014JB6_all.html|title=金安高速晋石高速今日通车 泉州实现县县通高速|accessdate=2012-12-31}}</ref>
*2013年12月31日，南石高速建成通车。<ref>{{cite web |url=http://news.163.com/14/0101/10/9HGDO2J200014Q4P.html|title=南石高速莆永高速泉州段昨通车(组图)|accessdate=2014-01-01}}</ref>
*2015年5月12日，泉州湾大桥建成通车。该段通车后，泉州绕城高速（小环)实现全线通车<ref>{{cite web |url=http://www.xinhuanet.com/politics/2015-05/12/c_127792211.htm|title=泉州湾大桥建成通车 为福建最长跨海交通工程|accessdate=2015-05-12}}</ref>
*2015年12月30日，泉厦漳城市联盟高速泉州段开工，该段为绕城高速（大环）最后一段未通车的路段。<ref>{{cite web |url=http://fjnews.fjsen.com/2015-12/31/content_17123566.htm|title=泉厦漳城市联盟高速泉州段开工 闽南将再添一快速通道|accessdate=2015-12-31}}</ref>
