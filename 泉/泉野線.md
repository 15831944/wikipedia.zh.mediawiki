'''泉野線'''（{{jpn|j=いずみ野線|hg=いずみのせん|rm=Izumino sen|f=y}}）是[[日本|日本]][[相模鐵道|相模鐵道]]的鐵路線，連結[[神奈川縣|神奈川縣]][[橫濱市|橫濱市]][[旭區_(橫濱市)|旭區]][[二俁川站|二俁川站]]與神奈川縣[[藤澤市|藤澤市]][[湘南台站|湘南台站]]。車站編號使用的路線記號為'''SO'''。

此外，此路線已經取得延伸至[[平塚市|平塚市]]的許可，然而仍未有延伸前景（詳細參見[[#往平塚方向的延伸計劃|後述]]）。

== 路線資料 ==
* 路線距離：11.3公里
* [[軌距|軌距]]：1067毫米
* 站數：8個（包括起終點站）
* 複線路段：全線
* 電氣化路段：全線（直流1500V）
* 最高速度：100公里/小時（南萬騎原→綠園都市）
* 保安裝置：[[自動列車停止裝置#ATS-P形（デジタル伝送パターン形）|ATS-P]]
* 數碼列車無線頻道：5ch

== 概要 ==
泉野線由二俁川站開始即通過丘陵地帶，[[隧道|隧道]]很多是其特徵，南萬騎原站至彌生台站前後皆為隧道。泉野站開始變成南向的高架鐵路，經過夢丘站後與[[橫濱市營地下鐵藍線|橫濱市營地下鐵藍線]]合流，於地下行走進入湘南台站。 

== 運行形態 ==
大部份列車於二俁川站通過[[相鐵本線|相鐵本線]]直通到達[[橫濱站|橫濱站]]。列車種別分為[[各站停車|各停]]，[[快速列車|快速]]和[[特別急行列車|特急]]。
== 历史 ==
* 1976年4月8日 二俁川 - 泉野間開業。
* 1990年4月4日 泉野 - 泉中央間開業。
* 1999年3月10日 泉中央 - 湘南台間開業。

== 女性専用車 ==
* 平日早上7：30～9：30到達橫濱站的全列車第4卡（實施區間為全區間）
* 平日黃昏18：00以後橫濱站發車的下行全列車第4卡（實施區間為全區間）
