'''泉州台商投资区'''由位于[[中华人民共和国|中国]][[福建省|福建省]][[惠安县|惠安县]]的[[洛阳镇_(惠安县)|洛阳镇]]、[[东园镇_(惠安县)|东园镇]]、[[张坂镇|张坂镇]]、[[百崎回族乡|百崎回族乡]]、惠南工业园区组成，于2010年成立<ref>{{Cite web|url=http://www.qzts.gov.cn/About.aspx?c=3&f=2|title=中国·泉州台商投资区|accessdate=2017-11-14|last=泉州台商投资区|work=www.qzts.gov.cn}}</ref>。泉州台商投资区是福建开发建设的第五个台商投资区。2012年中华人民共和国国务院批准设立。

== 历史 ==
1996年泉州市人民政府开始筹划建立泉州台商投资区。

2009年8月10日泉州市人民政府成立泉州台商投资区筹委会，廖小军为筹委会主任。

2010年3月8日前历史沿革同惠安县（在此之前隶属惠安县）。

2010年3月8日福建省人民政府同意成立泉州台商投资区。

2010年3月25日成立泉州台商投资区管理委员会，吴汉宗为管委会主任。<ref>{{Cite book|title=泉州台商投资区年鉴|publisher=|year=|isbn=|location=|pages=}}</ref>

== 地理 ==
区北邻惠安县涂寨镇和黄塘镇，东邻惠安县山霞镇，南面泉州湾，与石狮市隔海相望，西面洛江，与洛江区、丰泽区隔江相望。面积200平方公里。

== 语言 ==

== 人口 ==
人口约22.8万人（2010年数据）。
