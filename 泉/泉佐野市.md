'''泉佐野市'''（{{jpn|j=泉佐野市|hg=いずみさのし|rm=Izumisano shi|f=y}}）是位於[[日本|日本]][[大阪府|大阪府]]南部的行政區劃，[[關西國際機場|關西國際機場]]位於其海岸外的人工島，該人工島的北部及與本土相連的聯絡道路皆屬於泉佐野市的轄區。

市名「泉佐野」為本地的舊地名「佐野」加上舊國名[[和泉國|和泉國]]（簡稱「泉州」）組合而成。

本地主要產業包括毛巾工廠、食品加工廠。

== 歷史 ==

現在的泉佐野市轄區在令制國時代屬於[[和泉國|和泉國]][[日根郡|日根郡]]賀美鄉，過去曾設有「佐野莊」，最初的地名本來是「狹野」，意思指本地為「狹窄的原野」，經過長時間的演變而成為「佐野」。<ref name="Brief">{{Cite web| url=http://www.city.izumisano.lg.jp/city_profile/gaiyou.html| title=市的概要| publisher=泉佐野市政府| accessdate=2018-04-03| language=ja}}</ref>

在江戶時代大部分區域屬{{Tsl|ja|岸和田藩}}的領地，另有部分區域為幕府領，此後本地由於漁業及造船業而發展為港口。

19世紀末明治維新後，此地先於1871年被併入[[堺縣|堺縣]]，1881年再被整併至[[大阪府|大阪府]]；1889年實施[[町村制|町村制]]時，現在的泉佐野市轄區在當時分屬'''佐野村'''、{{tsl|ja|北中通村}}、{{tsl|ja|日根野村}}、{{tsl|ja|長滝 (泉佐野市)|長瀧 (泉佐野市)|長瀧村}}、{{tsl|ja|上之郷 (泉佐野市)|上之鄉 (泉佐野市)|上之鄉村}}、{{Tsl|ja|南中通村}}、{{tsl|ja|大土村}}共7個行政區劃。

佐野村於1911年改制為'''佐野町'''，北中通村先在1937年併入佐野町後，佐野町於1948年升格為市，但由於當時在[[栃木縣|栃木縣]]已有「佐野市」<ref name="Brief"/>，因此冠上舊國名的別稱成為「'''泉佐野市'''」。升格為市後，昭和大合併期間，其餘町村也於1954年併入泉佐野市成為現在的轄區。

1994年關西國際空港啟用，同時開始發展的新市鎮計畫－[[臨空鎮|臨空鎮]]為本地帶來大型商店及旅館的進駐，但由於當時日本景氣持續的低迷，反而讓本地的財政狀況惡化。

== 交通 ==

[[关西国际机场|关西国际机场]]位於轄區外海的人工島上，該人工島上的行政區劃轄區，是以原海岸線上各行政區劃延伸而出，因此關西國際機場的東北側約三分之一區域屬於泉佐野市的轄區，也因為[[關西國際機場聯絡橋|關西國際機場聯絡橋]]是從該島上東北端連結本土，因此整座聯絡橋都被劃入泉佐野市轄區，泉佐野市也向使用橋梁的車輛徵收每輛100日圓的橋樑使用費。

也因為該機場的因素，轄內所有鐵公路交通都設有延伸支線可通往機場，包括[[南海電氣鐵道|南海電氣鐵道]]的[[機場線_(南海電氣鐵道)|機場線]]、[[西日本旅客鐵道|西日本旅客鐵道]]的[[關西機場線|關西機場線]]，高速公路也有[[關西機場自動車道|關西機場自動車道]]。

== 教育 ==
[[File:Osaka_Prefecture_University_Rinku_Campus.JPG|thumb]]

[[關西國際機場|關西國際機場]]開始營運後，大阪府同時也在此沿海區域利用填海造地的區域規劃為[[臨空鎮|臨空鎮]]新市鎮，[[大阪府立大學|大阪府立大學]]也在此設立臨空校區，以生命環境科學及獸醫學為主。同時日本[[國土交通省|國土交通省]]轄下負責培訓航空管制及保安人員的[[航空保安大学校|航空保安大学校]]也從東京遷至此地。
