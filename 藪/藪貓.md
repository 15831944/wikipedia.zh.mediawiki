'''藪貓'''{{notetag|{{標音|字=藪|注音=ㄙㄡˇ|拼音=sǒu|同音字=叟}}}}（学名：{{lang|la|''Leptailurus serval''}}）是产于[[非洲|非洲]]的中型[[猫科|猫科]]动物，为'''藪貓屬'''的唯一成员。藪貓体长85厘米，尾长40厘米。平均寿命在12-20年。与其他猫科动物相比，它体型修长，腿长而尾短，耳朵又高又圆，距离很近。斑点有许多变种。

藪貓的主要棲息地是非洲大草原，有時也出現在山地。它們會爬樹和游泳，但很少這麽做。一般以群體為單位聚居，數量由於棲息地的破壞和偷獵而減少。

藪貓被列入《[[瀕臨絕種野生動植物國際貿易公約|瀕臨絕種野生動植物國際貿易公約]]》（CITES）<ref>{{cite web |url=http://www.cites.org/eng/app/index.shtml |title=存档副本 |accessdate=2011-02-18 |0=2011-02-18 }}</ref>。

== 捕食对象 ==
藪貓的食物基本上是[[嚙齒類|嚙齒類]]動物，但它們有時也捕食[[野兔|野兔]]、[[蹄兔|蹄兔]]、[[鳥類|鳥類]]、[[爬行類|爬行類]]、[[昆蟲|昆蟲]]、[[魚|魚]]和[[蛙|蛙]]。<ref name = awf>{{Cite web
| url = http://www.awf.org/content/wildlife/detail/serval
| title = Serval
| accessdate = 2007-03-13
| publisher = African Wildlife Foundation
}}</ref>  還有報導稱，藪貓可能捕食體型較大的動物，例如小[[羚羊|羚羊]]，但是藪貓90%的獵物體重都不超過200克。<ref name = catsurvivaltrust>{{Cite web
| url = http://www.catsurvivaltrust.org/serval.htm
| title = The Serval
| accessdate = 2007-03-13
| publisher = Cat Survival Trust}}</ref>藪貓的進食速度非常快。

藪貓的行動以垂直奔跳爲主，它們主要通過大大的耳朵聆聽草原上嚙齒類動物的行動（尤其是挖穴活動）。藪貓的捕獵效率很高，成功率超過50%，而一般貓科動物只有10%左右。<ref name = catsurvivaltrust/>

每胎產1-5仔，以2-3仔爲主，產仔一般利用[[土豚|土豚]]的洞穴，或者灌木地下的空地。藪貓有時被[[豹|豹]]捕食，不過它們最大的威脅來自人類。由於其皮毛美麗而昂貴，有不少人爲此偷獵藪貓。現在[[西非|西非]]和[[東非|東非]]的藪貓數量還比較多，但[[南非|南非]]和[[撒哈拉沙漠|撒哈拉沙漠]]以北的藪貓已經非常稀少，瀕臨滅絕。

== 驯养 ==
在很多國家，尤其是[[美國|美國]]，藪貓被當作[[寵物|寵物]]馴養。一些具有馴養經驗的人提供的報告表明，藪貓比[[家貓|家貓]]更加聰明、活潑。

此外，有一些雄性藪貓被用來與家貓雜交，以培育出新的寵物貓品種──[[熱帶草原貓|熱帶草原貓]]。

== 文化 ==
在[[意大利|意大利]]，藪貓是托馬西家族的象徵動物。

2017年1月，以野生動物[[萌擬人化|萌擬人化]]為主題的日本動畫《[[動物朋友|動物朋友]]》（{{lang|ja|けものフレンズ}}）播出，藪貓是動畫中的主要角色之一。
