'''滕'''是一個[[中國人|中國人]]的[[姓氏|姓氏]]，在《[[百家姓|百家姓]]》中排名第73位。

== 起源 ==
'''滕'''姓起源甚早，可追溯的源頭有三：

#出自「[[姬姓|姬姓]]」，以國名為氏。根據《姓氏考略》、《[[廣韻|廣韻]]》、《[[通志|通志]]·[[氏族略|氏族略]]》所載，「[[周|周]]」克[[商|商]]滅[[紂|紂]]後，「[[周文王|周文王]]」第十四子「錯叔繡」，受其兄「[[武王|武王]]」封於「滕」（今[[山東|山東]][[滕縣|滕縣]]西南），為「滕侯」，建立「[[滕國|滕國]]」。後世子孫以國為氏，並尊「錯叔繡」為其始祖。
#亦出自「姬」姓。根據《[[史記索隱|史記索隱]]》、《[[萬姓統譜|萬姓統譜]]》所載，「[[黃帝|黃帝]]」有二十五子，為四母所生，黃帝把他們分成十二個胞族，賜給他們十二個姓。滕姓就是其中之一。《國語》中記載了黃帝之子十二姓中排在第五位的即是滕姓（姬、酉、祁、己、'''滕'''、箴、任、荀、僖、佶、缳、依）。
#出自他族。明[[洪武|洪武]]年中，[[蒙古|蒙古]]人瓚住獲賜姓滕<ref>《[[明史|明史]]》（卷156）：“滕定，父瓚住，元樞密知院。洪武中，來降。授會州衛指揮僉事，賜姓滕。從燕起兵，進燕山右衛指揮使。卒，定嗣官，屢從出塞，有功，進至都督僉事。宣德四年封奉化伯，祿八百石。正統初卒。子福嗣，為指揮使。”</ref>。今土家、[[苗族|苗]]、[[蒙古|蒙古]]等民族均有此姓。

== 繁衍播遷 ==
滕立國約一千年，當中史冊有記載的國君有「[[滕昭公|滕昭公]]」、「滕定公」及其子「[[滕文公|滕文公]]」。滕國為[[齊|齊]]所滅，子孫紛紛離開得姓的原居地 — 今山東[[滕州|滕州]]，而散居於山東、[[河南|河南]]等地。[[東漢|東漢]]時，滕姓在[[北海郡|北海郡]]繁衍昌盛，族大人眾，名人輩出。這段時期滕姓登於史冊的有三人，分別為：[[滕延|滕延]]、滕咨和[[滕撫|滕撫]]，都是[[北海郡|北海郡]]人，可見在[[東漢|東漢]]時滕姓在北海郡已成為望族。漢末動亂時，當地的滕胄、滕耽等避亂渡江，歸屬[[孫吳|孫吳]]；值得一提的是滕胄與[[孫權|孫權]]結為親家，壯大了北海郡滕姓之盛名。吳末時期，北海人滕牧之女嫁與[[孫皓|孫皓]]而貴為皇后，滕牧後來卻因失寵而被遣居[[蒼梧|蒼梧]]，即今[[廣西|廣西]][[梧州|梧州]]；吳滅亡後，孫皓與滕皇后遷于[[洛陽|洛陽]]。兩晉時期，繁衍於[[南陽郡|南陽郡]]、[[開封縣|開封縣]]之滕族昌盛浩大，名家輩出，成為該地之名門望族。兩晉[[南北朝|南北朝]]至[[隋朝|隋朝]]，北方持續的動盪不安，加劇了滕姓南移，加上之前漢末已南遷的氏族，現今[[安徽|安徽]]、[[江蘇|江蘇]]、[[浙江|浙江]]、[[江西|江西]]、[[湖北|湖北]]、[[廣西|廣西]]、[[湖南|湖南]]等地均有了滕姓人家。[[唐代|唐代]]及其以後，浙江[[金華|金華]]的滕家一支獨秀，出了不少傑出人物，有[[白居易|白居易]]作詩為證：「身著錦衣兒戲繡，東陽門外數滕家」。另據《安遠滕氏族譜》、《石城縣誌》所載，「唐有滕文紀，家居江西[[撫州|撫州]]臨川縣滕家州，生子有三，其三子學珠自臨川徙居[[福建|福建]][[汀州|汀州]][[寧化|寧化]]招得裏南橋嶺，至宋時，傳八世，榜二居邵武，榜遂遷延平，榜儀移[[廣東|廣東]]」。宋[[靖康之恥|靖康之恥]]後，北方滕姓有遷居浙江臨安的，其中應天宋城（今河南商丘）人滕康因仕宦居湖南永州，其孫滕宬徙居吳縣，即今江蘇蘇州。宋末元初的動亂，再加元末的動亂，使滕姓分佈於南方各省份，今[[四川|四川]]、[[雲南|雲南]]均有滕姓人落籍。明初，山西滕姓作為「洪洞大槐村」遷民姓氏之一，被分遷于山東、河南、江蘇等地。明中葉時，滕學珠之裔有遷居江西石城豐山、高田等地。明末[[張獻忠|張獻忠]]屠川後，有湖南、湖北之滕姓徙至四川。清康、乾年間以後，有冀魯豫地之滕姓闖[[關東|關東]]進入[[中国东北|東三省]]，亦有沿海之滕姓遷居[[臺灣|臺灣]]以及東南亞等地。如今，滕姓在全國分佈較廣，尤以[[廣西|廣西]]、[[黑龍江|黑龍江]]、[[遼寧|遼寧]]、[[湖南|湖南]]多此姓，上述四省之滕姓約占全國[[漢族|漢族]]滕姓人口的百分之七十。

== 歷史名人 ==
* [[滕文公|滕文公]]：名弘(或作'''宏''')，戰國時「滕國」國君，公元前599年繼位，乃滕國諸君中史書記載最多之國君
* [[滕撫|滕撫]]：漢順帝時期名將
* [[滕胄|滕胄]]：漢獻帝時期著名文人，滕胤之父；博學善文，孫權待以賓禮。曾任京兆尹
* [[滕耽|滕耽]]：漢獻帝時仕吳之「右馬司」，滕胤之伯父
* [[滕胤|滕胤]]：三國時期吳國人；少有節操，剛成年即娶公主為妻；孫權稱王後，封「都亭侯」；歷任丹楊太守、吳郡太守和會稽太守；孫亮繼位後，出任太常、大司馬
* [[滕延|滕延]]：漢桓帝時北海郡人，後來曾任「京兆尹」，有政績，世人稱之為長者
* [[滕牧|滕牧]]：三國時期吳國人；官至「衛將軍」：其女嫁與吳王孫皓而貴為皇后
* [[滕皇后|滕皇后]]：滕牧之女，嫁與吳國末代王帝孫皓
* [[滕修|滕修]]：晉代名將；先任「廣州刺史」，晉為「安南將軍」，封「武當侯」
* [[滕含|滕含]]：滕修之孫，東晉大臣；功封「夏陽縣開國侯」；歷任「平南將軍」、「廣州刺史」
* [[滕子豚|滕子豚]]：滕修之孫，滕含之弟，為「交州刺史」
* [[滕恬之|滕恬之]]：滕修之曾孫，歷任「晉陵太守」、「黎陽太守」等，官至「龍驤將軍」
* [[滕峻|滕峻]]：為「桓溫都護」
* [[滕演|滕演]]：晉「秘書監」
* [[滕令琮|滕令琮]]：唐代開元初「國子」「司業公」
* [[滕昌佑|滕昌佑]]：五代十國時期名畫家；繼承唐代尚華麗之風，善畫花鳥魚蟲，尤以畫鵝鴨稱
* [[滕子京|滕宗諒]]：字「子京」，宋代岳州（今湖南岳陽）知府，即范仲淹之《岳陽樓記》中開首曰:「慶曆四年春，『滕子京』謫守巴陵郡。越明年，政通人和，百廢具興，乃重修岳陽樓……」而廣為人知；
* [[滕茂實|滕茂實]]：字「秀穎」，北宋進士
