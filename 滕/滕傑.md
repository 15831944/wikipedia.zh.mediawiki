== 早年經歷 ==
滕傑自[[黄埔軍官学校|黄埔軍官学校]]第4期步兵科毕业後，赴[[日本|日本]][[留学|留学]]，入[[明治大学|明治大学]]学习政治经济。归国後参加[[国民革命軍北伐|国民革命軍北伐]]，任[[国民革命軍|国民革命軍]]总司令部政治部副主任，後来改任[[国民政府軍事委員会|国民政府軍事委員会]]委員長侍从室第4組組長。1932年（[[民国紀元|民国]]二十一年）3月1日，滕傑協助籌組的 [[三民主義力行社|三民主義力行社]] (力行社) 正式成立，[[蔣中正|蔣中正]]被推為社長 (統稱領袖)，騰傑任常務幹事 (另四人為：[[賀衷寒|賀衷寒]]、[[康澤|康澤]]、[[桂永清|桂永清]]、[[潘佑強|潘佑強]]) 兼首任書記。1933年3月8日，旗下成立第三層級 (負責執行) 的 [[中華復興社|中華復興社]] (復興社)，騰傑兼任書記。<ref>滕傑先生訪問紀錄, 近代中國出版社 (國民黨-黨史委員會)(1993)</ref> 1933年（民国二十二年），任[[中央陸軍軍官学校|中央陸軍軍官学校]]政訓处处長。1934年（民国二十三年）赴[[欧洲|欧洲]]考察政治经济。1935年（民国二十四年）夏之后，历任[[湖南省政府|湖南省政府]]参議兼軍管区参謀長、国民政府軍事委員会战時幹部訓練团第一团政治部主任、国民政府軍事委員会政治部办公厅主任。<ref name=Xu>徐主編（2007）、2419頁</ref><ref name=Liu>劉国銘主編（2005）、2431頁。</ref>

== 中年經歷 ==
1941年11月，滕傑被任命為準備階段的[[三民主義青年團|三民主義青年團]]中央幹事會幹事。1943年2月，三民主義青年團正式成立，出任中央幹事會幹事，並晉升陸軍[[少將|少將]]。1945年，任[[抗日戰爭第三戰區|第三戰區]]長官部政治部主任。1946年，任[[徐州市|徐州]][[綏靖公署|綏靖公署]]政治部總務廳中將廳長兼第二廳廳長。同年11月，當選[[制憲國民大會|制憲國民大會]]代表。1947年3月，出任[[立法院|立法院]][[立法委員|立法委員]]。1947年11月，[[1947年中華民國國民大會代表選舉|當選]][[行憲國民大會|行憲國民大會]]代表，並任[[中國國民黨|中國國民黨]]國民大會黨部書記長。同年中，出任[[中國陸軍總司令部|陸軍總司令部]]徐州司令部秘書長兼中央訓練團徐州分團教育長。1948年12月22日，[[中國國民黨中央常務委員會|中國國民黨中央常務委員會]]批准南京市市長[[沈怡|沈怡]]辭職，由滕傑繼任。<ref name="民國史大事記"/>{{rp|8758}}12月24日，蔣介石特任命滕傑為[[南京市_(中華民國)|南京市]]市長兼黨部主任委員，原任沈怡准免本職。<ref name="民國史大事記">李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編：《中華民國史大事記》（全十二卷，共十二冊），[[北京|北京]]：[[中華書局|中華書局]]，2011年7月</ref>{{rp|8759}}[[第二次國共內戰|第二次國共內戰]]中國國民黨敗北，他隨政府到[[臺灣|臺灣]]。<ref name=Xu/><ref name=Liu/>成為中華民國最後一位南京市市長。

滕傑到臺灣後，擔任中國國民黨國民大會黨部書記長、第9屆至第15屆中國國民黨中央評議委員。此外，他還歷任[[國防研究院|國防研究院]]講座、[[中央信托局|中央信托局]]理事會主席、[[光復大陸設計研究委員會|光復大陸設計研究委員會]]委員、[[国防大学政治作战学院|政治作戰幹部學校]]教授。

== 晚年經歷 ==
1981年，滕傑任臺灣軍火輸出貿易公司董事長。1990年，任[[中國全民民主統一會|中國全民民主統一會]]會長。

1990年2月底，作為[[1947年中華民國國民大會代表選舉|未經改選已逾42年之久]]的[[第一屆中華民國國民大會代表|第一屆國大代表]]之一的滕傑，突然結合少數國大代表宣佈推選司法院長[[林洋港|林洋港]]與[[蔣緯國|蔣緯國]]參加該任[[1990年中華民國總統選舉|正副總統選舉]]<ref>[https://www.nownews.com/news/20130414/293595 二月政爭誤信李登輝　林洋港曾被開除國民黨籍]，NOWnews，2013年4月14日</ref>。

2004年（民國93年）7月4日，滕傑逝世，享年99歲。<ref>[[勞政武|勞政武]]：「同儔諸葛 國體之材―敬悼滕傑先生」，《傳記文學》507號，2004年8月，第40－52頁 </ref>
