'''鉀鎂煌斑岩'''（Lamproites）是{{tsl|en|Ultrapotassic igneous rocks|超鉀質火成岩|超鉀質}}[[地函|地函]]性[[火成岩|火成岩]]，可能是[[喷出岩|喷出岩]]或{{tsl|en|Subvolcanic rock|半深成岩}}。鉀鎂煌斑岩具有低含量[[氧化鈣|CaO]]、[[氧化铝|Al<sub>2</sub>O<sub>3</sub>]]、[[氧化钠|Na<sub>2</sub>O]]、高含量K<sub>2</sub>O/Al<sub>2</sub>O<sub>3</sub>，以及相對高含量的[[氧化镁|MgO]]，並含有許多[[不相容成分|不相容成分]]。

鉀鎂煌斑岩分布範圍很廣，但是體積都不大。鉀鎂煌斑岩不像[[金伯利岩|金伯利岩]]僅僅存在於上古大陸，而是廣泛分布在不同年齡的地塊，從[[太古宙|太古宙]]的[[澳大利亞|澳大利亞]]西部，到[[元古宙|元古宙]]和中生代的的[[西班牙|西班牙]]南部。岩石的年齡也差異頗大，從[[元古宙|元古宙]]到[[更新世|更新世]]都有，目前已知年輕的樣品約有56,000±5,000歲。

鉀鎂煌斑岩的形態學相當多樣，可以{{tsl|en|Diatreme|碎礫火山筒|筒狀}}、[[火山渣錐|柱狀]]，或錐狀表現。

== 岩石學 ==
鉀鎂煌斑岩是由地下超過150公里深的一部分熔融地函形成。熔融的物質被推到地表面形成[[火山筒|火山筒]]，同時在{{le|harzburgite|斜方輝石橄欖岩|斜方輝石}}[[橄欖岩|橄欖岩]]或[[榴輝岩|榴輝岩]]中夾帶一些{{tsl|en|Xenolith|異質捕虜岩|捕虜岩}}和[[鑽石|鑽石]]，其中的鑽石相對穩定。

最近的研究和鉛-鉛{{tsl|en|Isotope geochemistry|同位素地球化學}}顯示，鉀鎂煌斑岩可能源自[[隱沒帶|隱沒帶]][[岩石圈|岩石圈]]，在衝入過渡區後受熱融化，熔岩則被困在地函基部。這種觀察也使熔融深度與特殊的地球化學相一致，這是最容易通過在深地幔條件下熔化的原始材料來解釋的。

==  礦物學 ==
鉀鎂煌斑岩在礦物學上是被奇特的地球化學所控制，具有罕見優勢的稀有缺二氧化矽礦物， 以及稀有地函產生礦物。
 		
典型的鉀鎂煌斑岩礦物包括︰鎂質[[橄欖石|橄欖石]]、多鐵[[白榴石|白榴石]]、多鈦少铝[[金雲母|金雲母]]、含[[钾|钾]]多钛[[錳閃石|錳閃石]]、少铝[[透輝石|透輝石]]和多鐵[[透長石|透長石]]。出現各種罕见的微量礦物质。該岩石钾含量高，具有6至8%的氧化钾。[[鉻|鉻]]和[[鎳|鎳]]含量高也是典型的。該岩石通常會有滑石粉，並摻雜碳酸盐或蛇纹石、绿泥石、磁鐵礦等。也可能出現[[沸石|沸石]]和[[石英|石英]]。

==  經濟上的重要性 ==
隨著1979年發現[[西澳大利亚州|西澳大利亞]]的{{tsl|en|Argyle diamond mine|阿蓋爾鑽石礦}}，鉀鎂煌斑岩的經濟價值變得越來越高。這一發現導致了全世界其他已知鉀鎂煌斑岩的深入研究和重新評估；以前只有[[慶伯利岩|金伯利岩]]管被認為是經濟上可行的[[钻石|鑽石]]來源。
