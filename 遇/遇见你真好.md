'''遇见你真好（{{lang-en|Nice To Meet You}}）'''是一部2018年中国大陆青春校园喜剧电影，由首映时代、欢喜传媒、[[华谊兄弟|华谊兄弟]]、和禾和文化传媒等出品，[[顾长卫|顾长卫]]执导，[[白客|白客]]、[[蓝盈莹|蓝盈莹]]、[[张海宇|张海宇]]、[[周楚楚|周楚濋]]、[[曹骏|曹骏]]、[[王玉雯|王玉雯]]、[[李晨晖|李晨晖]]、[[蒋易|蒋易]]主演，[[范明|范明]]、[[蒋雯丽|蒋雯丽]]特别出演<ref>{{Cite web |url=http://ent.sina.com.cn/m/c/2017-08-29/doc-ifykiurx2639002.shtml |title=蓝盈莹《飞火流星》杀青 引青春回忆浪潮|accessdate=2017-11-21 |author= |date=2017-08-29 |work=YY|publisher=新浪娱乐|language=Zh-hans }}</ref>。2017年5月22日于广州紫荆中学开机<ref>{{Cite web |url=http://ent.sina.com.cn/m/c/2017-06-01/doc-ifyfuzmy0701767.shtml |title=顾长卫新片《飞火流星》开机 青春校园喜剧|accessdate=2017-11-21 |author= |date=2017-06-01 |work=YY|publisher=新浪娱乐|language=Zh-hans }}</ref>，2017年10月13日杀青<ref>{{Cite web |url=http://tech.sina.com.cn/roll/2017-11-06/doc-ifynmnae2150374.shtml |title=历经多次问询仍不放弃，长城影视拟12亿收购顾长卫家族控股公司|accessdate=2017-11-21 |author=徐露 吴丽仟 杨柳青 |date=2017-11-06 |work=高庆秀|publisher=新浪科技|language=Zh-hans }}</ref>，于2018年3月29日中国上映<ref>{{Cite web |url=http://ent.huanqiu.com/movie/yingshi-neidi/2018-03/11643966.html |title=《遇见你真好》创意“杯子舞”定档3.29|accessdate=2018-03-15 |author= |date=2018-03-06 |work=张玮|publisher=环球网|language=Zh-hans }}</ref><ref>{{Cite web |url=http://news.iqilu.com/shandong/kejiaoshehui/20180325/3867950.shtml|title=《遇见你真好》路演主创见面会爆棚 青春期最动人的“相遇”引观众共鸣|accessdate=2018-03-26|author= |date=2018-03-25 |work=|publisher=齐鲁网|language=Zh-hans }}</ref><ref>{{Cite web |url=https://www.xuehua.us/2018/03/25/电影《遇见你真好》发布会在上海举行，距开播还/|title=电影《遇见你真好》发布会在上海举行，距开播还有4天|accessdate=2018-03-26|author= |date=2018-03-26 |work=|publisher=雪花新闻 |language=Zh-hans }}</ref>。

== 剧情简介 ==
当遇见爱情的时候才会发现，它就像万有引力，苹果总归会落地。在被称为“高四”的紫荆复读学校是一所爱情集中营，梦想成为作家的张文生和珊妮在互损中萌生情愫，好友陈奇误认为文生横刀夺爱，与“坏学生”青龙在小酒馆相遇并商议各自的复仇计划。不料陈奇带去的巨型烟花被青龙向酒馆发泄中引发火灾，带出人命。于是，这所学校里的一个又一个神秘搞笑的爱情故事浮出水面，一群热血而欢乐的男女演绎了另类的青春和爱情……<ref>{{Cite web|url=https://movie.douban.com/subject/26967920/|title=遇见你真好 (2018)|accessdate=2018-03-26|author=|date=|work=|publisher=douban|language=Zh-hans }}</ref>
