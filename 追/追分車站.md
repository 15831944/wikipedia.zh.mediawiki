'''追分車站'''位於[[臺灣|臺灣]][[臺中市|臺中市]][[大肚區|大肚區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[海岸線_(臺鐵)|海岸線]]、[[成追線|成追線]]的[[鐵路車站|鐵路車站]]。「追分」（{{jpn|j=追分|hg=おいわけ|rm=oiwake|f=y}}）為[[日文漢字|日文漢字]]，為「分岐路」之意<ref name="追分車站">《台灣的老火車站》2003年  戴震宇著 遠足文化出版 第52頁</ref>，因早期縱貫線行至追分這裡之後路線開始分歧，其中一邊繼續沿[[海岸線_(臺鐵)|海岸線]]通往[[彰化車站|彰化]]，另一邊則經[[成追線|成追線]]通往[[臺中線|山線]]之[[成功車站|成功車站]]，形成[[三角線|三角線]]而得名，而在日本也有多個因為相同緣故而命名為「追分」的車站存在<ref name="追分車站"/>。除了站名的淵源之外，本站也因販售「[[吉祥語車票|追分－成功]]」紀念車票而聞名，其餘票證一律使用電腦製票。

== 車站構造 ==
*[[島式月台|島式月台]]兩座。
*追分站內並無設置任何的行人立體穿越設施，因此欲進出[[月台|月台]]需在站務人員的指揮下直接跨越鐵軌。
*木造車站，曾為臺中縣縣定古蹟，今為臺中市市定古蹟。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]，停靠車種以[[臺鐵區間車|區間車]]為主。
*第1月台：B側海線上行或成追線
*第2月台：A側往彰化、B側往大肚
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]、[[icash|icash 2.0]]及[[HappyCash|HappyCash]]付費。

== 歷史 ==
*1920年12月15日：海岸線（海線）自臺中線（山線）王田驛（今[[成功車站|成功車站]]）首先通車至[[清水車站_(台灣)|清水]]，稱為「王田支線」。
*1922年10月11日：海岸線（海線）全線通車，另設'''追分驛'''（今'''追分車站'''）直通[[彰化車站|彰化]]。「王田驛＝追分驛」間的路段，即成為今天的成追線。
*2015年6月30日：啟用多卡通刷卡機。
