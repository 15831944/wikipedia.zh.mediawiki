'''輔大聖心高級中學'''（{{lang-en|
Keelung Fu Jen Sacred Heart Senior High School}}<ref>{{cite web |url=http://www.trust.fju.edu.tw/minutes_PDF_files/minutes-18.pdf |title=存档副本 |accessdate=2014-07-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20131226234757/http://www.trust.fju.edu.tw/minutes_PDF_files/minutes-18.pdf |archivedate=2013-12-26 }}</ref>），簡稱'''輔大聖心'''、'''輔大聖中'''<ref>http://210.59.51.36/bestschool/</ref>或'''聖心高中'''，是一所位於[[台灣|台灣]][[基隆市|基隆市]][[中山區_(基隆市)|中山區]]飛鳳山麓的[[天主教|天主教]][[完全中學|完全中學]]，也是[[輔仁大學學校財團法人|輔仁大學學校財團法人]]所屬[[學校|學校]]之一，接受[[輔仁大學|輔仁大學]]教育學院垂直支援。

輔大聖心擁有[[高級中學|高中部]]（普通科、[[職業學校|技術類科]]）、[[國民中學|國中部]]等學制，與[[輔大聖心國民小學|聖心小學]]共享同一校園，並毗鄰[[耶穌聖心堂_(基隆市)|基隆耶穌聖心堂]]，構成了[[幼稚園|幼稚園]]至[[高中|高中]]階段的一貫教育體系，世稱「堂校合一」、「四代同堂」。

為有別於[[新北市|新北市]][[八里區|八里區]]的「[[新北市私立聖心女子高級中學|聖心女中]]」，一般又將該校稱為'''基隆聖心'''。

== 沿革 ==
聖心高中由原籍[[浙江省|浙江]][[乐清市|樂清]]的[[姚宗鑑|姚宗鑑]][[蒙席|蒙席]]來台後創立，同一校區內的聖心小學、以及校區側旁的[[基隆耶穌聖心堂|聖心天主堂]]皆由其創立。姚宗鑑蒙席亦曾在1970年代於基隆市[[南榮公墓|南榮公墓]]對面的山坡地購地以成立[[技術學院|技術學院]]，但當時未獲[[中華民國教育部|中華民國教育部]]核准而作罷。2012年8月1日連同[[校區|校區]]內的聖心小學正式歸屬於輔仁大學。

*1969年，姚宗鑑蒙席在基隆市飛鳳山麓籌設'''聖心高級工商職業學校'''（簡稱'''聖心工商'''），並經[[台灣省政府|台灣省政府]]教育廳核准立案。
*1971年，聖心工商開始招生，當時設有機械、電機、商業三科。
*1972年，聖心工商附設補習學校成立。
*1990年，增設[[普通科_(台灣高中)|高中普通科]]，並改名為'''聖心高級中學'''。
*1995年，奉-{}-准成立國中部，並於同年開始招生。
*2009年，[[天主教輔仁大學|天主教輔仁大學]]董事會同意將聖心高級中學併入輔大<ref>http://www.trust.fju.edu.tw/minutes_PDF_files/17-05minutes.pdf{{dead link|date=十月 2017 |bot=InternetArchiveBot }}</ref>。
*2012年8月1日，聖心高級中學更名為'''輔大聖心高級中學'''<ref>[http://mag.udn.com/mag/campus/storypage.jsp?f_MAIN_ID=13&f_SUB_ID=104&f_ART_ID=393324 聖心高中、小學 將併入輔大 - 聯合報 2012.06.02]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 聖心校區內各設施 ==
聖心中學和聖心小學，有時會合稱為「聖心中小學」，這也是台灣許多學校家族的通稱方法，如[[再興中學|再興]]、[[立人高中|立人]]、[[復興中小學|復興]]等私校皆是。

為了避免聖心校區內不同學制的彼此干擾，並促進校園內資源分享，聖心學校另有設立「教務督導委員會」，各級學校有重大事務及活動時，由該委員會協調及處理。而學校旁的[[耶穌聖心堂_(基隆)|聖心天主堂]]與學校關係密切，聖心中小學與聖心天主堂在昔日號稱「堂校合一」、「四代同堂」。

== 與輔仁大學合併案 ==
2008年起，聖心中小學傳出將歸併於[[天主教輔仁大學|天主教輔仁大學]]的傳言。2008年6月20日，校長[[池易釧|池易釧]]發表《給聖心中學家長的一封信》公開信，證實聖心中小學正與輔大討論合併事宜。經過近兩年的協商與準備後，輔仁大學董事長[[劉振忠|劉振忠]]總主教與聖心中學暨小學董事長姚宗鑑蒙席在2010年8月11日正式簽約，確定將進行輔仁大學、聖心中學、聖心小學三方之[[財團法人|財團法人]]合併。三校原有之[[董事會|董事會]]將解散，籌組「輔仁聖心董事會」，原有教職員工將全部結算年資重新聘用<ref>[http://www.pro.fju.edu.tw/press_show.asp?press_id=721 輔大與基隆聖心高中暨國民小學三方財團法人合併簽約 共創新未來 - 輔大新聞稿 2010/8/11]</ref><ref>[http://www.libertytimes.com.tw/2010/new/aug/17/today-taipei1.htm 聖心、輔大董事會 簽約合併 - 自由時報 2010/8/17] {{webarchive|url=https://web.archive.org/web/20110111121428/http://www.libertytimes.com.tw/2010/new/aug/17/today-taipei1.htm |date=2011-01-11 }}</ref>。至2011年8月1日、也就是[[民國紀年|民國]]101學年度開始後，聖心中學與聖心小學正式成為輔仁大學的附屬[[學校|學校]]，其校名之前也同步加上「輔大」兩字。
