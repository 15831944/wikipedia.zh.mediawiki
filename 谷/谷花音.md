'''谷花音'''（{{bd|2004年|5月4日}}）是出生於[[日本|日本]][[埼玉縣|埼玉縣]]的[[童星|童星]]，從事[[女演員|女演員]]、[[主持|主持]]、[[歌手|歌手]]活動，隸屬於[[:ja:テアトルアカデミー|THEATRE ACADEMY]]，官方身高136公分，體重38公斤。

== 概歷 ==
*2007年由[[:ja:テアトルアカデミー|THEATRE ACADEMY]]簽下，2008年開始演藝活動至今。
*2011年4月，演出[[富士電視台|富士電視台]]日劇《[[失去名字的女神|媽媽們的戰爭]]》進藤羅羅一角，同年7月演出《[[全開女孩|全開女孩]]》櫻川日向一角，加上同時期放送的NEC電視廣告「花音的公司訪問」篇，集結了注目度。
*2011年7月，在拍攝《[[全開女孩|全開女孩]]》時，開設了期間限定的富士電視台專屬[[微網誌|微網誌]][[:ja:イマつぶ|imatsubu]]<ref>{{Cite web|url=http://imatsubu.jp/profile/2102857|title=イマつぶ|author=谷花音|date=2011年7月 - 2011年9月|deadurl=yes|archiveurl=https://web.archive.org/web/20111230130934/http://imatsubu.jp/profile/2102857|archivedate=2011-12-30}}{{Jp icon}}</ref>，由本人親自更新。
*2012年3月，出版個人第一本服裝寫真書《谷花音−Kanon's Book− 〜ステキな女の子になる51の方法》。
*2012年4月，和童星[[鈴木福|鈴木福]]一起搭檔主持[[:ja:BSフジ|BS富士台]]招牌兒童節目《[[:ja:beポンキッキーズ|beポンキッキーズ]]》，於每週一到週五早上7:30 - 8:00播出。
*2012年8月，和童星{{link-ja|小林星蘭|小林星蘭}}組成日本最年少女子團體[[:ja:すたーふらわー|Star Flower]]（すたーふらわー）發行單曲《年下の男の子》。

== 個人資料 ==
=== 個人 ===
*對自己的自稱是のんちゃん（nonchan）。
*有美人子役<ref>{{Cite web |url= http://www.news-postseven.com/archives/20110829_29453.html |title= 『全開ガール』のツンデレ役で注目　7才の美人子役・谷花音 | publisher= 女性セブン2011年9月8日号}}{{Jp icon}}</ref>、小悪魔系子役<ref>{{Cite web |url= http://www.sponichi.co.jp/entertainment/news/2011/08/20/kiji/K20110820001447590.html |title= “小悪魔系子役”谷花音、将来の夢は「アイスクリーム屋さん」 | publisher= SPORTS NIPPON NEWSPAPERS 2011年8月20日}}{{Jp icon}}</ref>（小惡魔系童星）的封號。
*家庭成員為父、母、年差5歲的妹妹，從埼玉搭電車往來於攝影現場拍戲，記台詞的方式是前一晚就寢前由媽媽唸劇本的對白。
*擅長的表演項目是哭戲，落淚時是藉由想像媽媽死掉的情景。
*曾公開表達自己是[[嵐|嵐]]、[[Becky_(日本藝人)|Becky]]、[[:ja:マツコ・デラックス|松子DELUXE]]的忠實粉絲。

=== 人物、交友 ===
*與同事務所的[[鈴木福|鈴木福]]在3歲就認識<ref>於2012年8月18日播出的《[[:ja:メレンゲの気持ち|メレンゲの気持ち]]》裡，鈴木福與谷花音兩人3歲時的影片錄像所提及。</ref>，一起上表演課至今。亦曾在節目上透露送出情人節巧克力給[[鈴木福|鈴木福]]<ref>於2012年4月8日播出的《[[:ja:行列のできる法律相談所|行列のできる法律相談所]]》裡，谷花音本人提及。</ref>。
*與同事務所的{{link-ja|小林星蘭|小林星蘭}}交情甚篤，除了一起演出日劇《[[彩虹閃耀夏之戀|閃耀夏之戀]]》《[[失去名字的女神|媽媽們的戰爭]]》《[[澪之料理帖|澪之料理帖]]》，也共同組成女子團體[[:ja:すたーふらわー|Star Flower]]（すたーふらわー）。
*與[[新垣結衣|新垣結衣]]在合作《[[全開女孩|全開女孩]]》後關係良好<ref>{{Cite web |url= http://www.cinematoday.jp/page/N0034749 |title= ガッキーもメロメロ！女優らしさと子どもっぽさが同居した注目子役・谷花音との関係語る！ | publisher= CINEMATODAY 2011年8月20日}}{{Jp icon}}</ref>，皆稱被對方治癒。
*與[[川島海荷|川島海荷]]、[[今井悠貴|今井悠貴]]在合作《[[爸爸偶像|爸爸偶像]]》後關係良好，除了共演者Blog的證實<ref>{{Cite web |url= http://ameblo.jp/sato-jiro/entry-11283337421.html |title= ムリヤリ脈測りシリーズ～今井悠貴くん・谷花音ちゃん編～| publisher= 佐藤二朗 2012年6月21日}}{{Jp icon}}</ref>，亦有谷花音與[[今井悠貴|今井悠貴]]一同搭電車回[[埼玉縣|埼玉]]的[[Twitter|Twitter]]目擊照片。
