'''谷智鑫'''{{BD|1978年|8月24日|||catIdx=G谷}}出生于[[辽宁省|辽宁省]][[丹东市|丹东市]]，毕业于[[中央戏剧学院|中央戏剧学院]]2000級表演系，为中国大陆知名电影电视演员。

== 生平 ==
1978年，谷智鑫出生在辽宁省丹东市，家中有一个哥哥和姐姐，儿时在永昌小学、丹东二十一中、丹东一中读书，后在中央戏剧学院表演系学习。<ref>{{cite web |url=http://data.ent.sina.com.cn/star/18043.html |title=谷智鑫 |accessdate=2012年9月18日 |publisher=新浪网 |work= |language =zh-cn}}</ref>
