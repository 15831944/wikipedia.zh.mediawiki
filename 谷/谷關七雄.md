'''谷關七雄'''，係指[[臺灣|臺灣]]中部[[谷關|谷關]]風景區一帶的七座[[中級山|中級山]]，包含[[八仙山|八仙山]]、[[馬崙山|馬崙山]]、[[屋我尾山|屋我尾山]]、[[波津加山|波津加山]]、[[東卯山|東卯山]]、[[白毛山|白毛山]]和[[唐麻丹山|唐麻丹山]]。<ref name="谷關七雄">[http://www.trimt-nsa.gov.tw/cht/unit_04_1_2.aspx?subsiteID=1&hotID=412&chk=%20&showlabel=1#B1 交通部觀光局參山國家風景區管理處]網站。</ref>[[林務局|林務局]][[東勢林區管理處|東勢林區管理處]]於曾2012年6月1日舉辦「谷關七雄完登」活動，民眾只要於2013年8月31日前完成谷關七雄登頂，即可獲得東勢林區管理處所頒發的證書及徽章。<ref name="谷關七雄完登">[http://dongshih.forest.gov.tw/ct.asp?xItem=60846&ctNode=1964&mp=320 「谷關七雄完登」活動。]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 八仙山步道 ==
[[八仙山|八仙山]]標高2,366公尺，為谷關七雄之首（海拔高度，以下同），位於[[八仙山森林遊樂區|八仙山森林遊樂區]]內。登山步道爬升高度1,366公尺，路程非常遙遠，來回約需9至10個小時，登山口位位於[[八仙山森林遊樂區|八仙山森林遊樂區]]內，靜海寺旁，單程6.3公里，兩側多為針葉林。沿途展望不佳，走在路上不容易曬到太陽，空氣出奇的好，很適合健行，由登山口開始便是一連串位於二葉松林的之字形陡上坡，雖然步道指示牌上指出，八仙山步道勞而不累，但根據走過此步道之不少山友表示，八仙山步道是全谷關七雄之中難度排名第一高的登山步道。

== 馬崙山步道 ==
[[馬崙山|馬崙山]]標高2,305公尺，為谷關七雄老二。登山步道爬升高度1,255公尺，單程7公里，路程遙遠，來回約需8個小時。登山口在省道[[台8線|台8線]]36.2公里附近。登山路徑寬大好走，之字形上登，沿途高大松樹清涼有勁。滿地松針落葉一路相伴，走來賞心悅目，即使是盛夏來此登山，也是清爽舒坦，最佳避暑勝地。

== 屋我尾山步道 ==
[[屋我尾山|屋我尾山]]標高1,796公尺，為谷關七雄老三。登山口有兩處：北入口在大雪山林道約31.2公里處，可以輕鬆登頂，來回約需3小時。南入口在台8線31.2公里處，全程陡上坡，路程遙遠，來回約需8小時。

== 波津加山步道 ==
[[波津加山|波津加山]]標高1,772公尺，山頂擁有一顆編號6583之三等三角點，為谷關七雄老四。登山口在谷關溫泉區，捎來步道上的涼庭旁，捎來步道擁有兩處入口：
* 郵局旁及四季溫泉會館。自郵局至波津加登山口，約0.8公里，距離雖長，但坡度較緩
* 自四季溫泉會館至波津加登山口，約0.5公里，距離雖短，但坡度較陡。
波津加山步道全長3.6公里，爬升高度962公尺，每0.5公里設置一里程木椿，來回約需 7 個小時。步道陡峭，直到最後一段稜線稍緩。與山名台語諧音（坡真陡）相呼應，可測試腳力。沿途展望良好，可見[[大甲溪|大甲溪]]南岸山巒腰台地聚落，沿途標示清楚。

== 東卯山步道 ==
[[東卯山|東卯山]]標高1,690公尺，為谷關七雄老五。步道全長6公里，若從大道院下方停車場走林道至東卯山全長7.55公里，爬升高度910公尺，來回約需6至7個小時。登山口在台8線25.2K，谷關大道院下方停車場旁。由登山口起沿途「Z字」緩坡而上，每500公尺有一木椿標記。
登頂三等三角點基點，山頂有一片水泥廣場及三座發射板且視野一級棒360度群山環繞，整體路況良好山徑明顯。

== 白毛山步道 ==
[[白毛山|白毛山]]標高1,522公尺，為谷關七雄老六。步道爬升高度720公尺。
由白鹿吊橋至白毛山第二登山口2.7公里，至白毛山2.2公里。白毛山步道單程4.9公里，來回約需6至7個小時。
原登山口在台8線19.9K[[白鹿吊橋|白鹿吊橋]]<ref>{{Cite news|url=https://www.youtube.com/watch?v=8m6iI73vt2s|title=和平白鹿吊橋封很久待修|author=大台中新聞|date=2017-04-24|work=|accessdate=2017-07-24}}</ref>，但白鹿吊橋於2012年5月23日封閉。登山客一般改走其他兩處：
* [[白毛山林道|白毛山林道]]主線終點。需注意白毛林道無車輛管制，但林道年久失修，崎嶇不平，一般車輛難以進入。一般登山客於林道前段停車後，步行進入。
* 台8線21k處右轉下馬鞍壩，經壩頂過大甲溪後，接上白鹿部落聯外道路，沿水泥路上行直達登山口。

攀登途中可俯瞰大甲溪，東卯山和唐麻丹山分峙左右，麗陽、松鶴分佈在大甲溪兩岸，遠處右邊八仙山、馬崙山以及後頭雪劍稜脈，左邊[[稍來山|稍來山]]、[[鳶嘴山|鳶嘴山]]稜脈也都一清二楚。

==唐麻丹山步道==
[[唐麻丹山|唐麻丹山]]標高1,305公尺，為谷關七雄老么。登山口有兩處：
* 台8線29公里處的松鶴部落
* 台8線26公里處的裡冷部落
步道爬升高度640公尺，單程2.7公里，來回約需6個小時。
為谷關七雄輕鬆路線，可順遊蝴蝶谷瀑布，景色優美，展望佳。附近至[[白冷山|白冷山]]上下約150分，至頭拒山約30分。
