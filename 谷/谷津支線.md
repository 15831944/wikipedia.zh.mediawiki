'''谷津支線'''（{{jpn|j=谷津支線|hg=やつしせん|rm=Yatsu shisen|f=y}}）是[[千葉县|千葉县]][[船橋市|船橋市]]的花輪站（現・[[船橋競馬場车站|船橋競馬場车站]]）至[[習志野市|津田沼町]]的谷津遊園地站曾经连接的[[京成電鉄|京成電氣軌道]]的[[铁路线|铁路线]]（[[廃线|廃线]]）。

== 路線資訊 ==
*路線距離（{{link-ja|營業距離|営業キロ}}）：1.2公里
*[[軌距|軌距]]：1372毫米
*站數：2個（沒有中間站）
*複線路段：沒有（全線單線）
*電氣化路段：全線（1928年3月為止直流600V、以後1200V）

== 路線概要 ==
该线路为1925年所开业的[[谷津遊園|谷津遊園]]而建设。此时的花轮站被设立作为谷津支线的一个分支站，营运初期有从[[押上站|押上站]]有直通运输业务，之后由于谷津海岸站（現[[谷津站|谷津站]]）始发的道路開通而停止营运。现已弃置已久，谷津海岸站也改称为「谷津遊園站」。

由于营运时期很短，路线废弃已久而周边发展迅速，目前已经很难再找到当时的痕迹。另一方面，由于当时的[[地图|地图]]并未及时作出修改将其收录，所以可以说是很难推测谷津游乐园站的位置。

== 历史 ==
*1927年（昭和2年）8月21日 開業
*1931年（昭和6年）10月27日 停止营运
*1934年（昭和9年）6月22日 全線废弃
