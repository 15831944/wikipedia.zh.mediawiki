'''-{谷}-氨酰胺'''（Glutamine）亦被稱作'''麩醯胺酸'''，為人體中含量最豐富的非必需[[胺基酸|胺基酸]]，且是唯一一種可直接通過[[腦血管障壁|腦血管障壁]]（BBB）的[[胺基酸|胺基酸]]。在人體中儲存於[[骨骼肌|骨骼肌]]或[[血液|血液]]中。當受傷或患病時，谷氨酰胺可能需要藉由攝取含Gln的食物來獲得足夠的量。[[生物体|生物体]]通过[[谷氨酰胺合成酶|谷氨酰胺合成酶]]催化[[谷氨酸|谷氨酸]]和[[铵盐|铵盐]]反应生成谷氨酰胺。

== 性質 ==
对应的密码子：CAA 或者 CAG
=== 食物中含量 ===
除了[[味精|味精]]以外，[[左旋麩醯胺酸|左旋谷胺醯酸]]可藉由攝取[[牛肉|牛肉]]、[[牛奶|牛奶]]（奶製品）、[[雞肉|雞肉]]、[[雞蛋|雞蛋]]、[[魚肉|魚肉]]、[[穀類|穀類]]、[[甘藍菜|甘藍菜]]、[[甜菜|甜菜]]根、[[豆類|豆類]]、[[菠菜|菠菜]]和[[西洋芹|西洋芹]]等獲取。果菜汁或[[發酵|發酵]]食物（如：[[味噌|味噌]]）中也有少量的左旋谷氨醯酸。

=== 功能 ===
在生化領域有多種應用，包括:
# 合成DNA的基質之一。
# 在合成蛋白質扮演重要角色。
# 為腸道細胞的主要能量來源。
# 為快速分裂的免疫細胞的前導物質，故可增強免疫力。
# 在腎臟中藉由生成氨來達成酸鹼值平衡。
# 腦細胞的替代能量來源，並可阻止由[[皮質醇|皮質醇]]（可的松）導致的蛋白質的分解。

此外，針對不能進食的外科病人，手術後給予的靜脈營養液裡就含有豐富的谷氨醯胺（以及[[精氨酸|精氨酸]]、ω-3[[脂肪酸|脂肪酸]]等等），除了增強免疫力避免感染，也加快病人的復原。
