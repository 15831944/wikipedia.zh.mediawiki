'''谷正綱'''（{{bd|1902年|4月30日|1993年|12月11日|catIdx=Ku Cheng-kang}}），字'''叔常'''，[[贵州省|贵州省]][[安顺府|安顺府]]人。[[中國|中华民国]]政治人物。早年属於[[中国国民党|中国国民党]][[改組派|改組派]]，晚年号称「反共鐵人」。谷氏三兄弟（兄[[谷正倫|谷正倫]]，弟[[谷正鼎|谷正鼎]]）同時擔任中國國民黨及中华民国政府要职。<ref name=zx>{{Cite book|author = 東亜問題調査会|title = 最新支那要人伝|year = 1941|publisher = [[朝日新聞社|朝日新聞社]]|isbn = }}</ref><ref name=xu>{{Cite book|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref><ref name=kmt>{{Cite book|author = 劉国銘主編|title = 中国国民党百年人物全書|year = 2005|publisher = 团結出版社|isbn = 7-80214-039-0}}</ref>

==生平==
===改組派要人===
谷正綱生於[[清|清]][[光緒|光緒]]二十八年（1902年）3月23日。1922年（民国11年），与弟[[谷正鼎|谷正鼎]]共赴[[德国|德国]]留学，入[[柏林洪堡大学|柏林大学]]。留学期间，1924年兄弟共同加入[[中国国民党|中国国民党]]。1925年，和[[谷正鼎|谷正鼎]]、[[王仲裕|王仲裕]]同赴[[苏联|苏联]]，在[[莫斯科中山大学|莫斯科中山大学]]从事研究。<ref name=zx/><ref name=xu/><ref name=kmt/>

1926年（[[民国紀元|民国]]15年）归国。1928年（民国17年），加入[[陳公博|陳公博]]、[[顧孟餘|顧孟餘]]等中国国民党人士结成的「国民党改組同志会」（俗称「[[改組派|改組派]]」）。1931年（民国19年），[[蒋介石|蒋介石]]软禁[[胡漢民|胡漢民]]，反蒋派在[[广州|广州]]召开非常会議，谷正綱参加，在[[北平|北平]]、[[天津|天津]]活動。<ref name=zx/><ref name=xu/><ref name=kmt/>

同年，[[九一八事变|九一八事变]]爆发，蒋派与反蒋派实现大团結。1931年12月，谷正綱当选中国国民党第四届候補中央執行委員。1934年（民国23年）12月，被任命为[[国民政府|国民政府]]实业部常務次長。1935年11月，升任中国国民党第五届中央執行委員、常務委員、組織部副部長。1937年（民国26年）[[抗日战争|抗日战争]]爆发後，被任命为国民政府軍事委員会第五部副部長。1938年6月，任[[三民主義青年团|三民主義青年团]]中央臨時幹事会常務幹事。此外还任[[中国国民党浙江省党部|中国国民党浙江省党部]]主任委員兼[[抗日战争第三战区|第三战区]]政治部主任。<ref name=zx/><ref name=xu/><ref name=kmt/><ref name=liu>{{Cite book|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = 中華書局|isbn = 7-101-01320-1}}</ref>

===社会部部長===
1939年（民国28年）11月，谷正綱被任命为[[中国国民党中央社会部|中国国民党中央社会部]]部長，负责战時的社会福祉事業。1940年（民国29年）10月，该部移交国民政府管理，设国民政府社会部，谷正綱继续担任国民政府社会部部長，此後一直任至1949年（民国38年）3月。<ref name=zx/><ref name=xu/><ref name=kmt/><ref name=liu/>

1941年，日軍[[重庆大轰炸|轟炸重慶]]，谷正綱處理救助傷患事宜。1944年，日軍在[[豫湘桂会战|豫湘桂会战]]中，在[[廣西|廣西]]、[[貴州|貴州]]邊境的进攻造成大量中国軍民傷亡，谷正綱親赴前線撫輯流亡，歷時两個多月。<ref>中國災胞救助總會編，谷正綱先生與救總，台北市：中國災胞救助總會，1991年</ref>

1945年（民国34年）5月，谷正綱当选中国国民党第六届中央執行委員兼常務委員，兼任中国国民党中央農工委員会委員長。同年7月至8月，暂任国民政府農林部部長。<ref name=zx/><ref name=xu/><ref name=kmt/><ref name=liu/>

[[第二次国共内战|第二次国共内战]]期間，谷正綱担任中国国民党中央社会部部長。1947年4月23日，國民政府准免社會部部長谷正綱本職；特任谷正綱為行政院政務委員，兼社會部部長。<ref name="民國史大事記">{{cite book|editor=李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編|title=《中華民國史大事記》|location=[[北京|北京]]|publisher=[[中華書局|中華書局]]|date=2011年7月}}</ref>{{rp|8340}}4月25日，中國國民黨中常會決議組織中央合作指導委員會，定陳立夫、谷正綱、趙仲容、王世穎、陳果夫等為該會當然委員，吳鐵城、馬超俊、蔣經國、蕭錚、錢天鶴、樓桐蓀、于樹德、陳仲明為委員，即依組織規程成立，並推定陳立夫為主任委員，王世穎為總幹事。<ref name="民國史大事記"/>{{rp|8341}}5月1日，上海總工會在跑馬廳舉行「上海市勞工節慶祝大會」，社會部部長谷正綱、[[上海市市長|上海市市長]]吳國楨、警備司令[[宣鐵吾|宣鐵吾]]等出席。<ref name="民國史大事記"/>{{rp|8346}}

谷正綱历任最高经济委員会委員、綏靖区政務委員会委員、[[制憲国民大会代表|制憲国民大会代表]]、[[行憲国民大会|行憲国民大会]]代表。1949年（民国38年）1月，蒋介石被迫下野。2月17日，谷正綱與[[張道藩|張道藩]]赴「[[溪口鎮_(宁波市)|溪口]]」<ref>[[蔣經國|蔣經國]]〈危急存亡之秋〉，刊《風雨中的寧靜》，台北，[[正中書局|正中書局]]，1988年，第153頁。</ref>。3月，谷正綱辞任社会部部長。8月27日，蔣與谷正綱談[[雲南|雲南]]、[[貴州|貴州]]政情。<ref>蔣經國〈危急存亡之秋〉，載《風雨中的寧靜》，台北，正中書局，1988年，第239頁。</ref>此后轉進[[台湾|台湾]]。<ref name=zx/><ref name=xu/><ref name=kmt/><ref name=liu/>

===台灣時期===
到台湾後，1950年（民国39年）1月，谷正綱被任命为[[中華民国内政部|内政部]]部長，兼任[[中國大陸災胞救濟總會|中國大陸災胞救濟總會]]理事長。<ref name=zx/><ref name=xu/><ref name=kmt/>谷正綱担任[[中國大陸災胞救濟總會|中國大陸災胞救濟總會]]理事長，任内處理了[[韓戰|韓戰]][[反共義士|反共義士]]、[[大陳義胞|大陳義胞]]、北越僑胞、泰北難民等诸多事宜。<ref>中國災胞救助總會編，谷正綱先生與救總，台北：中國災胞救助總會，1991年</ref>　1950年8月，谷正綱任中国国民党中央改造委員会委員兼特別党務組組長，对中国国民党的組織进行改革。1951年（民国40年），获聘为[[总統府国策顧問|总統府国策顧問]]。1952年（民国41年），当选中国国民党第七届中央委員，此後至第十二届连任。<ref name=zx/><ref name=xu/><ref name=kmt/>

1954年（民国43年）1月，谷正綱被任命为[[中華民国国防部|国防部]]参謀次長。1954年8月，任[[世界反共聯盟|亚洲自由国家聯合反共聯盟]]中国总会理事長。1959年（民国48年）10月，任国民大会秘書長兼憲政研討委員会秘書長。1966年（民国55年），任国民大会憲政研討委員会副主任委員。1967年（民国56年），任[[世界反共聯盟|世界人民反共聯盟]]主席，翌年改任荣誉主席。<ref name=zx/><ref name=xu/><ref name=kmt/>

1980年（民国69年），谷正綱转任[[总统府資政|总统府資政]]。在中国国民党内，1988年（民国77年）任第十三届中央評議委員，第十四届连任。<ref name=zx/><ref name=xu/><ref name=kmt/>

1993年（民国82年）12月11日，谷正綱在[[台北市|台北市]]病逝。享耆壽92岁（满91歲）。<ref name=zx/><ref name=xu/><ref name=kmt/>

==家庭==
生有五子，分別為[[谷家泰|谷家泰]]、谷家華、谷家嵩、[[谷秀衡|谷秀衡]]、五子[[谷家恒|谷家恒]]。

長子[[谷家泰|谷家泰]]，台灣國立成功大學電機工程學士，後至美國留學，於[[羅格斯大學|羅格斯大學]]取得電機工程碩士，[[匹茲堡大學|匹茲堡大學]]取得電機工程博士。曾任[[王安電腦|王安電腦]]台灣區總經理、吉悌電信公司總裁，並曾擔任[[遠傳電信|遠傳電信]]董事長[[徐旭東|徐旭東]]特別助理。長孫[[谷有為|谷有為]]，與[[寶成集團|寶成集團]]總裁[[蔡其瑞|蔡其瑞]]之女[[蔡佩君|蔡佩君]]結婚。
五子[[谷家恆|谷家恆]], 台灣國立台灣大學機械工程學士, 後至美國聖母大學取得機械工程碩士,博士學位。於1993年奉[[教育部|教育部]]創設[[國立高雄技術學院|國立高雄技術學院]],並擔任校長一職。1998年[[國立高雄技術學院|國立高雄技術學院]]改名為[[國立高雄第一科技大學|國立高雄第一科技大學]]擔任首任校長。現為[[中國科技大學|中國科技大學]]校長。其妻為[[陳田錨|陳田錨]]之女陳怡君。
