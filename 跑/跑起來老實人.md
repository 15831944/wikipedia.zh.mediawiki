《'''跑起來老實人'''》（{{lang-ja|走れ正直者}}）是[[日本|日本]]男性[[歌手|歌手]][[新御三家|西城秀樹]]的第66張[[單曲|單曲]]1991年4月21日由[[VERMILLION_RECORDS|D.O.G HOUSE]]／[[日本索尼音樂娛樂|BMG勝利（今日本索尼音樂娛樂）]]發行。

== 解說 ==
* 同名標題曲「跑起來老實人」是[[富士電視台|富士電視台]][[電視動畫|電視動畫]]《[[櫻桃小丸子|櫻桃小丸子 (第1期)]]》的第2代片尾主題曲。至於該歌曲會請[[新御三家|西城秀樹]]擔任主唱的原因是，同名漫畫的原作者[[櫻桃子|櫻桃子]]本身就是西城的歌迷，因此西城爽快的答應了，並實現了櫻桃子從小的心願。
* 「跑起來老實人」後來另外有收錄在西城的名曲集大成[[CD|CD]]中。
* 發片廠商雖然是由[[日本索尼音樂娛樂|BMG勝利]]（現已併入日本索尼音樂娛樂），但{{link-ja|原盤權|原盤権}}至今仍歸[[Being_(公司)|Being]]所有。從這首歌曲之後，只要是由[[長戶大幸|長戶大幸]]擔任執行製作人的其它歌曲之原盤版權都歸Being所有。
* 「跑起來老實人」也是在日本國內創舉的[[斯卡曲風|斯卡]]名曲。但是負責作曲和編曲由[[織田哲郎|織田哲郎]]後來在接受專訪時表示，事實上在錄製這首歌的時候，斯卡和[[雷鬼音樂|雷鬼]]類音樂在當年的日本流行音樂沒有很流行<ref>{{Cite web|date=|title=【織田哲郎長篇專訪第13回｜織田哲郎Project2007】TETSURO ODA Official Web Site|url=http://www.aspect.co.jp/oda2007/interview/13.html|work=|publisher=織田哲郎的官方網站|accessdate=2017-06-21|language=ja}}</ref>。

== 時事 ==
* 該單曲發行3星期之後，西城他在5月11日於{{link-ja|東京厚生年金會館|東京厚生年金会館|東京·厚生年金會館}}舉辦的歌壇出道20週年記念演奏會『{{link-ja|HIDEKI SAIJO CONCERT TOUR '91 FRONTIER ROAD|HIDEKI SAIJO CONCERT TOUR '91 FRONTIER ROAD}}』中有獻唱該歌曲。

== 逸事 ==
「跑起來老實人」在被選為《[[櫻桃小丸子|櫻桃小丸子]]》第1期動畫片尾曲之前，曾在同名動畫的筆頭贊助廠商[[日本電信電話|日本電信電話]]（當時廠名叫NTT）的廣告作為『{{link-ja|call waiting|コールウェイティング}}』短片歌曲使用。同時，《櫻桃小丸子》的主角小丸子有在這部廣告短片中出場。歌詞方面，廣告短片版的「跑起來老實人」歌詞中的「{{lang|ja|インディアン}}（中譯印第安人）」在動畫片尾曲時改成「{{lang|ja|ソーセージ}}（中譯小香腸）」（歌詞中唱到「印第安」或「小香腸」之前，一樣都以{{link-ja|Rinrin & Ranran|リンリン・ランラン|樂句}}表現）。
