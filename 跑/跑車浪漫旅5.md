{{vgname|跑車浪漫旅5|ja=グランツーリスモ 5|en=Gran Turismo 5|cn=GT赛车5|tw=跑車浪漫旅5}}是[[PlayStation_3|PlayStation 3]]平台獨佔發行的[[GT赛车系列|GT赛车系列]][[竞速游戏|竞速游戏]]第五部正统作品，由[[Polyphony_Digital|Polyphony Digital]]开发，[[索尼电脑娱乐|索尼电脑娱乐]]发行。本游戏发行于2010年11月24日，可以认为是此前发售的《[[跑车浪漫旅5_序章版|跑车浪漫旅5 序章版]]》的完整版本。

2014年5月20日，索尼关闭了《GT赛车5》的的线上服务。<ref>{{Cite web |url = http://www.ali213.net/news/html/2013-12/93314.html |title = 今后只能玩单机！索尼关闭多款PS3游戏服务器 |author = 乐乐 |date = 2013-12-27 |publisher = 游侠网 |language = zh-cn |accessdate = 2019-02-13 }}</ref><ref>{{Cite web |url = http://www.ali213.net/news/html/2014-1/94084.html |title = SONY宣布旗下《抵抗》1-3系列服务器将停运 |author =  |date = 2014-01-07 |publisher = 游侠网 |language = zh-cn |accessdate = 2019-02-13 }}</ref><ref>{{cite web |url=http://www.eurogamer.net/articles/2014-01-06-sony-turning-off-gran-turismo-5-resistance-servers |title=Sony turning off Gran Turismo 5 and Resistance servers |author=Robert Purchese |date=2014-01-06 |website=[[Eurogamer|Eurogamer]] |publisher=Gamer Network |language=en|accessdate=2019-02-13}}</ref>

== 玩法 ==
{{Contradict|章节|time=2016-04-26}}

本游戏遵循系列一直以来的原则，尽可能在游戏中实现真实驾驶的体验。游戏中收录車型有1031款（其中約有200款為精細車型，800款為普通車型），26个比赛地点，和总计71条赛道（包含变化的赛道）。游戏封面主題車是2009年的[[平治SLS_AMG|平治SLS AMG]]。

游戏GT Life和有两种模式，一种是親自扮演駕駛員操縱方向盤遊玩，另一种为扮演教練培育[[人工智慧|AI]]駕駛員。

相对于本系列以往作品，本作有了很多革命性的进步。包括：

* 游戏加入了新汽車、賽道，例如[[丰田FT-86|丰田FT-86]]以及[[Top_Gear|Top Gear]]賽道，也包含夜间赛事。
* 首次加入了线上赛事，支持最多16人参加的比赛。
* 收录的车辆分为“标准模型”和“精緻模型”两大类。
* 首次加入损毁系统，根据车辆本身是属于 “标准模型”或“精緻模型”，损坏时有不同程度的表现。
* 若干PS3平台带来的新特性，比如具有奖杯系统，支持1080p显示，3D电视支持，以及通过网络下载补丁和扩展包。

=== 车型与赛道 ===

本作收錄了超過1088款車輛<ref>{{Cite web|url=http://eu.gran-turismo.com/local/gb/data1/products/gt5/carlist_en.html|title={{En icon}}官方車種列表|publisher=gran-turismo.com}}</ref>與31條賽道共81種路線<ref>{{Cite web|url=http://www.gran-turismo.com/local/jp/data1/products/gt5/courselist_en.html|title={{En icon}}官方賽道列表|publisher=gran-turismo.com}}</ref>（截至2013年1月16日，包括下載項目，即是DLC），車型包括已經收錄在跑車浪漫旅PSP版以及跑車浪漫旅5序章的車型，當中大約200款為精緻車型，大約800款為普通車型。精緻車型乃全新製作，著重車身及車廂細節，設有車內視點，頭燈可調整亮度；普通車型則源自歷代舊作並加以改良，車內視點簡化及沒有可調整的頭燈。由於精緻車型的製作時間長達半年，所以數量並不多。

新加入的車型有[[迈凯伦F1|迈凯伦F1]]、[[法拉利458_Italia|法拉利458 Italia]]、[[Tesla_Roadster|Tesla Roadster]]、[[捷豹|捷豹XJ13]]、[[標緻汽車|標緻908 HDi FAP]]、以及在不受賽車規限影響，与一级方程式冠军车队[[红牛车队|红牛车队]]合作以現有技術研製而成的超級賽車[[Red_Bull_X2010|Red Bull X2010]]及其後繼車款Red Bull X2011。

《跑車浪漫旅5》收錄了31條真實和原創賽道共81種路線。其中部分賽道可呈現時間及天候的變化。

== 開發歷程 ==

2005年的E3中，Polyphony Digital公開了名為Vision Gran Turismo的影片，基本上是在跑車浪漫旅4中加入更多車子以及更多人。直到2006年底，跑車浪漫旅HD Concept公開下載，並於2007年底推出跑車浪漫旅5序章。

山內一典曾經表示，本作的開發團隊達到140人，投資額達到6千萬美元，由於不論軟件還是硬件上都是相當複雜，收錄的車型極多，外型與機械規格上需要與其他車廠協調，過程亦遇到很多問題需要解決，所以開發時間相當長，足足花了2000日時間才完成，在人力物力上的付出為1997年初代的50倍。

2009年的E3中，首個跑車浪漫旅5正式版的預告影片發放，收錄NASCAR、世界拉力錦標賽以及Super GT等車型，並且有車身損毀的顯示，但未公布發售日期。2009年的[[東京電玩展|東京電玩展]]中，Polyphony Digital表示於2010年3月在日本及亞洲發售，但索尼於2010年1月13日突然宣佈延期，最後在[[E3|E3 2010]]上公佈北美上市日期為2010年11月2日，後來公佈日本以及歐洲上市日期為2010年11月3日。在2010年10月13日於官方網站宣佈再次延期至年底<ref>{{Cite web|url=http://www.jp.playstation.com/info/release/nr_20101013_gt5.html|title={{Ja icon}}日本索尼發佈之延期新聞稿|publisher=www.jp.playstation.com}}</ref>，一個月後，即是11月13日則公佈北美及歐洲上市日期為2010年11月24日，日本上市日期為2010年11月25日。

遊戲上市後，製作小組仍然繼續製作新車型、新賽道以及新內容，並透過網絡進行更新。本作的大型更新於2011年10月發布，稱為Spec 2.0，並加入付費下載內容。

2011年10月开始，各地区版本先后推出了扩充游戏内容的收费DLC下载升级包。和前作《[[跑车浪漫旅4|跑车浪漫旅4]]》相同，本作的[[香港|香港]]版本和[[台湾|台湾]]版本支持中文语言，台湾限定版本附赠的官方Apex指导手册也被译为中文。

12月17日，跑車浪漫旅5計時挑戰賽在PlayStation Network公開下載，只限登入PlayStation Network方可進行遊戲，車型為日產370Z，賽道為印第安那波里賽車場。此乃2010年GT學院的首部曲，創下最快圈速的玩家有機會駕駛真實的賽車參與真實的賽事。

== 遊戲樂曲 ==

遊戲開場動畫之鋼琴樂曲由[[郎朗|郎朗]]演奏，曲目為[[:en:Piano_Sonata_No._7_(Prokofiev)|Piano Sonata No. 7 in B flat major, Op. 83]]。<ref>{{Cite web|url=http://www.gran-turismo.com/tw/news/d11086.html|title={{zh-tw icon}}『Gran Turismo 5 (跑車浪漫旅 5)』開場影片配樂由郎朗所演奏|publisher=gran-turismo.com|deadurl=yes|archiveurl=https://web.archive.org/web/20101130135843/http://www.gran-turismo.com/tw/news/d11086.html|archivedate=2010-11-30}}</ref>

開場動畫的歌曲為美國樂團[[我的另類羅曼史|我的另類羅曼史]]的Planetary (GO!)，日本版例外，以歷代正章主題音樂Moon Over The Castle（由安藤正容演繹）重新編曲的Knight's Song ~Generation 3~（由安藤正容領導的演奏樂團T-Square演繹）修改版作為音樂，稱為Moon Over The Castle GT5 Version。
