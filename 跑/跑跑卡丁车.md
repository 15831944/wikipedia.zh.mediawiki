《'''跑跑卡丁车'''》是[[韩国|韩国]][[NEXON|NEXON]]公司出品的一个休闲类赛车竞速游戏，亦是[[泡泡堂|泡泡堂]]的衍生游戏，在[[中国大陸|中国大陸]]的代理是[[世纪天成|世纪天成]]，在[[臺湾|臺湾]]及[[香港|香港]]则是[[游戏橘子|游戏橘子]]代理。

《[[跑跑飛天車|跑跑飛天車]]》則是由本遊戲衍生出的一款遊戲。

== 历史 ==

*2004年3月Nexon公司《Kart Rider》在韩国发布 ，6月公测
 
*2004年12月《Kart Rider》打破《星际争霸》长达六年的统治地位，成为在韩国网吧最受欢迎的游戏 
 
*2005年2月 《Kart Rider》韩国在线人数突破20W 实名注册人数超过总人口1/3，成为韩国史上的五款“国民级”游戏

*2006年3月17日 中国大陸版《POP KART》公测，后在线人数破70万，打破由盛大网络代理的NEXON另一款“泡泡堂”的60万记录。12月18日於台灣[[封測|封測]]<ref>{{cite web |url=https://gnn.gamer.com.tw/6/25396.html |title=《跑跑卡丁車》18 日封測 韓國國民遊戲台灣試身手 |publisher=巴哈姆特電玩資訊站 |date=2006-12-13}}</ref>。

*2007年1月3日於台灣公測<ref>{{cite web |url=https://gnn.gamer.com.tw/1/25591.html |title=《跑跑卡丁車》正式登台 搶攻上班族與女性玩家市場 |publisher=巴哈姆特電玩資訊站 |date= 2007-01-03}}</ref>。

*2008年7月30日在巴哈姆特遊戲大賞榮獲「年度最佳線上遊戲(其它)金賞」<ref>{{cite web |url=https://gnn.gamer.com.tw/0/31700.html |title=第 1 屆「巴哈姆特遊戲大賞」台灣年度遊戲網路票選活動結果出爐 |publisher=巴哈姆特電玩資訊站 |date=2008-07-30}}</ref>。隔年在巴哈姆特遊戲大賞榮獲「人氣線上遊戲(其它)銅賞」<ref>{{cite web |url=https://gnn.gamer.com.tw/7/38467.html |title=2009 巴哈姆特遊戲大賞 13 家廠商榮獲 25 項大獎 |publisher=巴哈姆特電玩資訊站 |date=2009-08-04}}</ref>。

*2010年7月1日，在巴哈姆特遊戲大賞榮獲「年度最佳線上遊戲(其它)銀賞」<ref>{{cite web |url=https://gnn.gamer.com.tw/1/45271.html |title=巴哈姆特 2010 遊戲大賞名單揭曉 各款人氣遊戲幕後功臣喜獲肯定  |publisher=巴哈姆特電玩資訊站 |date= 2010-07-01}}</ref>。

==手機遊戲==
===《跑跑向前衝》===
{{vgname|跑跑向前衝|en=Kartrider Rush}}是將《跑跑卡丁車》移植至ios平台的賽車遊戲，採取免費下載內含付費機制，自2011年3月上市後的三個月已突破300萬下載量<ref>{{cite web |url=https://gnn.gamer.com.tw/3/55803.html |title=NEXON 宣布《跑跑卡丁車向前衝》突破 300 萬次下載量 |publisher=巴哈姆特電玩資訊站 |date=2011-07-22}}</ref>。隔年2月9日，韓國 NEXON 宣布下載量突破700萬並即將登陸[[Facebook|Facebook]]遊戲平台<ref>{{cite web |url=https://gnn.gamer.com.tw/4/62314.html |title=NEXON 宣布今年上半年將在 Facebook 推出《跑跑卡丁車向前衝》 |publisher=巴哈姆特電玩資訊站 |date=2012-02-09}}</ref>。5月14日《跑跑向前衝》正式登陸Facebook遊戲平台<ref>{{cite web|url=https://gnn.gamer.com.tw/9/65689.html |title=NEXON 旗下 FB 新作《跑跑卡丁車向前衝》今日公測 |publisher=巴哈姆特電玩資訊站 |date=2012-05-14}}</ref>。

===《迷你賽車站》===
{{vgname|迷你賽車站|en=Kart Rider Pit}}是NEXON公司在ios平台推出的另一款改編《跑跑卡丁車》的手機遊戲，但不同的是，遊戲玩法為反應排程類[[模擬遊戲#經營模擬|經營模擬遊戲]]並採取免費下載內含付費機制<ref>{{cite web |url=https://gnn.gamer.com.tw/4/69674.html|title=跑跑卡丁車專用！《迷你賽車站》iOS 版正式開張 |publisher=巴哈姆特電玩資訊站 |date=2012-08-17}}</ref>。
