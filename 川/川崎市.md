'''川崎市'''（{{jpn|j=川崎市|hg=かわさきし|rm=Kawasaki shi|f=y}}）是位於[[日本|日本]][[神奈川县|神奈川县]]東北端的都市，位處[[東京都區部|東京]]、[[横滨市|橫濱]]兩大都市之間，為[[政令指定都市|政令指定都市]]。其為面積最小的政令指定都市，總人口數則為151萬，位居日本全國都市的第8位，同時是日本除[[都道府縣廳所在地|都道府縣廳所在地]]以外人口最多的都市。

== 历史 ==
川崎市西北部的[[丘陵|丘陵]]地区，已经确认自[[绳文时代|绳文时代]]开始便有人居住。随着公元七世纪的[[律令体制|律令体制]]的制定，该地区成为[[武藏国|武藏国]]的一部分。[[平安时代|平安时代]]，该地区由稻毛氏一族控制。自公元1128年，寺院川崎大师（平间寺）建立後，从[[镰仓时代|镰仓时代]]到[[战国_(日本)|战国时代]]，历经了一些小领主的统治，最终被纳入北条氏的统治范围。[[江户时代|江户时代]]，随着[[东海道|东海道]]以及[[中原街道|中原街道]]重要性的日益提升，[[幕府|幕府]]在现在的[[川崎站|川崎站]]地区附近设置了[[川崎宿|川崎宿]]，并于1623年将它指定为[[东海道五十三次|东海道五十三次]]中的最后一个正式[[宿场|宿场]]。

到了近代，随着日本第一条铁道（现[[东海道本线|东海道本线]]）于1872年开业，[[川崎站|川崎站]]诞生。此后，以该车站为中心，邮政，电力等公共事业也随之展开。并且在[[明治|明治]]、[[大正|大正]]年间，以川崎站为中心的[[城市化|城市化]]加速。与此同时，西北部丘陵地区，仍然保持着原有的农村模样。到了[[昭和|昭和]]初期，已经形成了工业区聚集在[[南武线|南武线]]沿线，住宅区集中在各[[私铁|私铁]]沿线的基本格局。二战期间，川崎市于1945年4月15日遭到[[盟军|盟军]]轰炸，川崎站周边以及临海地区的工业地带遭受到巨大损失。

战后，随着日本经济复苏，因为作為紧挨着[[东京|东京]]的城市，川崎市西北部地区作为良好的住宅区而獲得良好聲譽，吸引了很多在東京工作的人入住。1972年4月1日，川崎市被指定为[[政令指定都市|政令指定都市]]。1974年，人口超过了100万，并保持繁荣至今。

== 地理 ==
* 川崎市位于[[神奈川县|神奈川县]]东北部。市的北部与[[东京都|东京都]]的交界处流淌着[[多摩川|多摩川]]、東部则是宽广的[[东京湾|东京湾]]。市的西南部是被称作为[[多摩丘陵|多摩丘陵]]的丘陵地帶。因为市的东部是平坦地，自古以来多用于工业用地及住宅，特别是东京湾岸在大規模的填海造地之后众多的重化学工业进出此处。一方面，中部、西部则是商业地、住宅地集中区。
* 流过市内的河川主要是[[多摩川|多摩川]]以及[[鶴見川|鶴見川]]。
** 多摩川水系：[[三澤川_(多摩川水系)|三澤川]]、山下川、[[五反田川|五反田川]]、[[二領本川|二領本川]]、[[平瀨川|平瀨川]]、多摩川
** 鶴見川水系：片平川、麻生川、真福寺川、有馬川、江川、澀川、[[矢上川|矢上川]]、鶴見川

=== 区域划分 ===
川崎市目前由七个区构成。另外，川崎市的行政区名与其他的[[政令指定都市|政令指定都市]]的行政区名或者是[[东京|东京]]的行政区名无一相同。在日本，都市中使用最多的区名是"西区"。

# 幸区 - 离区公所最近的车站是[[矢向站|矢向站]]。
# 中原区 - 离区公所最近的车站是[[武藏小杉站|武藏小杉站]]。该区有许多大企业的办公楼或者工场，比如[[NEC|NEC]]、[[富士通|富士通]]
# 高津区 - 离区公所最近的车站是武蔵溝ノ口駅。
# 多摩区 - 离区公所最近的车站是向ヶ丘遊園駅。[[藤子·F·不二雄|藤子·F·不二雄]]生前就住于此。
# 宫前区 - 离区公所最近的车站是宫前平駅。
# 麻生区 - 离区公所最近的车站是新百合ヶ丘駅。

== 经济 ==
川崎市的产业结构主要以[[重工业|重工业]]以及[[服务业|服务业]]为主。[[明治|明治]]初期曾一度兴盛的[[农业|农业]]随着战后[[城市化|城市化]]的推进，所占比重逐渐减少。

由于川崎市位于[[京滨工业地带|京滨工业地带]]中间，自[[明治维新|明治维新]]以来，重工业就一直在不断增长。目前，市内仍然存有诸如[[JFE控股|JFE]]、[[富士通|富士通]]、[[NEC|NEC]]、[[东芝|东芝]]等大企业的工场，以及诸多中小企业的工场。另一方面，临海地区的化学工业相关产业，则在不断萎缩中。近年来，川崎市内集中了许多尖端科技的研究所，{{link-ja|川崎科技工业园区|かながわサイエンスパーク}}的建设也在实施中。

川崎市的商业活动主要以[[川崎站|川崎站]]为中心展开。比如川崎站西口的{{link-ja|LAZONA川崎广场|ラゾーナ川崎プラザ}},以及川崎站东口的{{link-ja|Azalea地下街|川崎アゼリア}}。另外，在各个区的主要车站，也分别建设了很多商业中心。如[[武藏溝之口站|武藏溝之口站]]，[[新百合丘站|新百合丘站]]等车站。另外，川崎市还值得一提的是它的[[风俗业|风俗业]]，川崎站附近的[[堀之内|堀之内]]是[[关东地区|关东地区]]仅次于[[吉原|吉原]]的[[风俗街|风俗街]]。

=== 巴士 ===
川崎市的巴士线路主要由[[川崎市市营巴士|川崎市市营巴士]]、[[东急巴士|东急巴士]]、[[小田急巴士|小田急巴士]]及[[川崎鹤见临港巴士|川崎鹤见临港巴士]]等公、私营巴士企业负责运营。巴士統一採用前门上车，后门下车的運作方式。

== 教育 ==
截至2009年川崎市擁有市立[[小學|小學]]114所，市立[[初級中學|初中]]51所，全日制[[高級中學|高中]]25所（縣立14所，市立5所，私立6所），定時制高中7所（皆為市立）<ref>{{cite web|language=ja|title=平成21年度 学校統計要覧|url=http://www.pref.kanagawa.jp/osirase/40/4020/toukei/index.html|deadurl=yes|archiveurl=https://web.archive.org/web/20100412005403/http://www.pref.kanagawa.jp/osirase/40/4020/toukei/index.html|archivedate=2010-04-12}}</ref>。此外，擁有8所[[大學|大學]]（含大學的校區），4所[[短期大學|短期大學]]。

在社會教育方面，川崎市各個區都設立有市立[[圖書館|圖書館]]以及相應的分館，總計7所本館、5所分館以及1處閱覽所<ref>{{cite web|language=ja|title=川崎市立図書館　図書館一覧|url=http://www.library.city.kawasaki.jp/riyou_libmap/libmap.html}}</ref>。此外，神奈川縣縣立圖書館在川崎市也設立有神奈川縣立川崎圖書館，主要收藏自然科學，工學，產業信息等分類的資料。

== 体育 ==
=== 主要体育设施 ===
* '''川崎球场'''
位于川崎市富士见公园内。1952年启用。曾经是职业棒球联赛的赛场。现今只是作为美式足球以及一些临时赛事的场地使用。

* '''川崎等等力球场'''
位于中原区等等力绿地内。可容纳5000名观众。是高中棒球预选赛的赛场，以及美式足球的正式赛场。

* '''等等力陆上竞技场'''
位于中原区等等力绿地内。可容纳25000名观众。1964年启用，经历若干修整后于1993年被正式认定为[[J联赛|J联赛]]的主要赛场。

* '''古市场陆上竞技场'''
原本是幸区的古市场。

* '''川崎国际生田绿地高尔夫球场'''
位于多摩区的丘陵地带。1952年作为乡村俱乐部而开业。

* '''川崎竞轮场'''
[[File:Kawasakinight.jpg|thumb]]
位于川崎市川崎区。是一个自行车赛的场地，于1949年启用。

* '''川崎赛马场'''
位于川崎市川崎区。1949年启用，现在作为神奈川县川崎赛马联合会举办赛马的赛场。

* '''川崎市体育馆'''
位于川崎市川崎区。主要举办[[职业摔角|职业摔角]]等室内赛事。

* '''川崎等等力Arena'''
位于中原区等等力绿地内。主要举办国际排球赛事以及一些演唱会。
