== 簡介 ==
本名：'''永野繁代'''（{{lang|ja|永野 繁代／ながの しげよ}}）。丈夫[[永野護|永野護]]為[[機械設計_(動畫)|機械設計師]]、[[漫畫家|漫畫家]]。其藝名是在參演配音出道作《[[聖戰士DUNBINE|聖戰士鄧拜音]]》時，由擔當的動畫導演[[富野由悠季|富野由悠季]]取名。

為人熟悉的代表配音作品有《[[聖戰士DUNBINE|聖戰士鄧拜音]]》的恰姆·法烏、《[[重戰機|重戰機L-Gaim]]》女主角莉莉絲·法烏&加烏·哈·蕾茜、《[[機動戰士Z_GUNDAM|機動戰士Z鋼彈]]》的{{link-ja|貝托蒂嘉·伊爾瑪|ベルトーチカ・イルマ}}、《[[機動戰士GUNDAM_逆襲的夏亞|機動戰士鋼彈 逆襲的夏亞]]》的珂絲·帕拉雅、《[[秀逗魔導士_(動畫)|秀逗魔導士]]》系列的[[白蛇娜卡|白蛇娜卡]]、《{{link-ja|貓咪也瘋狂|ホワッツマイケル}}》的波波、《{{link-ja|烏龍少爺|おぼっちゃまくん}}》的大小姐沙麻代、《{{link-ja|藤子不二雄A的夢魔子|藤子不二雄Aの夢魔子}}》主角夢魔子、《[[羅賓漢大冒險|羅賓漢大冒險]]》的溫妮芙蕾德·杭丁頓、《[[我的長腿叔叔|我的長腿叔叔]]》的卡蓮·帕特森、《[[崔普一家物語|崔普一家物語]]》的海德薇希·馮·崔普、《[[十二生肖爆烈戰士|十二生肖爆烈戰士]]》的塔露朵&歐拉公主、《[[新世紀福音戰士|新世紀福音戰士]]》的惣流·今日子·齊柏林、《[[羅密歐的藍天|羅密歐的藍天]]》的安潔麗塔·蒙特巴尼、《[[蒙面俠蘇洛傳_(日本動畫)|蒙面俠蘇洛傳]]》的羅莉塔·普里多、《[[灰姑娘物語|灰姑娘物語]]》主角辛蒂蕾拉、《[[機械女神#機械女神J|機械女神J]]》的玉三郎、《[[機動神腦|機動神腦]]》的希金絲·薩斯、《[[神奇寶貝_(1997-2002年動畫)|神奇寶貝]]》的[[精靈寶可夢動畫角色列表#四天王·冠軍|科拿]]、《[[詩∽片|詩∽片]]》的木暮沙耶、《{{link-ja|遙遠時空 ～八葉抄～|遙かなる時空の中で-八葉抄-}}》的席琳、《[[無限地帶23|無限地帶23]]》的高中由唯、《[[飛越巔峰|飛越巔峰]]》的容格·弗洛伊德、《{{link-ja|妙齡女大亨|クレオパトラD.C.}}》主角克婁巴特拉·科內斯、《{{link-ja|星座宮神話|アリーズ}}》主角貝爾瑟弗尼／佐倉亞理沙、《{{link-ja|白鳥麗子是也！|白鳥麗子でございます!}}》主角白鳥麗子、《{{link-ja|究極超人R|究極超人あ〜る}}》的西園寺瑪麗、《[[五星物語|五星物語]]》的拉克西絲、《[[花之詩女_GOTHICMADE|花之詩女 GOTHICMADE]]》主角貝琳·阿潔麗；遊戲《[[女神側身像|女神戰記]]》系列的芙蕾、《{{link-ja|問答七彩夢 虹色町的奇跡|クイズなないろDREAMS 虹色町の奇跡}}》的[[森永製菓|森永]]惠、《[[波波羅古洛伊斯物語|波波羅古洛伊斯]]》系列的娜露西亞王妃等。

== 經歷 ==
川村從初中時進入文藝社開始，因當時文藝社與話劇社合併，於是成為了體驗話劇的女演員。{{link-ja|富士見中學高等學校|富士見中学高等学校|富士見高校}}入學後，進入漫畫研究會參演廣播劇<ref>川村萬梨阿《私{{lang|ja|も悩んでおとなになった}}》，{{link-ja|POPLAR社|ポプラ社}}，1998年，第44～52頁。</ref>。此外，川村本身是[[江古田|江古田]]咖啡店「漫畫畫廊」的粉絲，因此和{{link-ja|小牧雅伸|小牧雅伸}}、[[結城正美|結城正美]]、{{link-ja|蛭兒神建|蛭児神建}}等人都是這裡的常客<ref>吉本松明《{{lang|ja|おたくの}}起源》，NTT出版，2009年。</ref>。並曾經在漫畫畫廊友人的慫恿之下兼差當[[動畫雜誌|動畫雜誌]]的編輯<ref name="out83-7">「{{lang|ja|声}}優{{lang|ja|インタビュー}} 川村{{lang|ja|万}}梨阿{{lang|ja|さんの巻}}」《{{link-ja|月刊OUT|月刊OUT}}》1983年7月號，{{lang|ja|みのり}}書房，第95～100頁。</ref>。

1980年高中畢業之後，川村成為[[東映|東映]]演技研修所的研修生，第一份工作是擔當{{link-ja|歪斗秀|ワイドショー}}節目《{{link-ja|小川宏Show|小川宏ショー}}》的記者，演員的出道作是特攝影集《[[太陽戰隊太陽火神|太陽戰隊太陽火神]]》（飾演紳士服店店員。本名川村繁代名義）。之後作為演員活動參加的還有[[東映|東映]]製作[[超級戰隊系列|超級戰隊系列]]和《{{link-ja|特搜最前線|特捜最前線}}》、《[[猛龍特警隊|G-MEN'75]]》<ref name="out83-7" /><ref>《My Anime》1984年8月號的專訪記事，[[秋田書店|秋田書店]]。</ref>。東映研修所畢業後，立即成為研修所附屬事務所東映演研製作（<small>{{lang-ja|東研プロダクション}}</small>）所屬。

就在川村幫朋友進行動畫導演[[富野由悠季|富野由悠季]]的取材專訪時，被受邀請參加新作動畫的試鏡錄音{{Refnest|group="注"|在電話中，富野說：“喔喔，這是我想要的聲音，這個好這個好”<ref>天本伸一郎編「第4章／異世界{{lang|ja|へ アンケート}} 富野由悠季{{lang|ja|とその}}仲間{{lang|ja|たち}}CHAP.04 川村{{lang|ja|万}}梨阿」《富野由悠季 全仕事》，[[電影旬報社|Kinema旬報社]]〈Kine旬Mook〉，1999年6月9日，第199頁 ISBN 4-906649-00-9</ref>。}}。1983年，她以電視動畫《[[聖戰士DUNBINE|聖戰士鄧拜音]]》（飾演恰姆·法烏）在配音業界出道<ref>《私{{lang|ja|も悩んでおとなになった}}》，第68～73頁。</ref>。《聖戰士鄧拜音》完結之後，接著通過次年之接檔時段作品《[[重戰機|重戰機L-Gaim]]》女主角加烏·哈·蕾茜的試鏡，從此下定決心全面投入配音工作，並從東映演研製作轉往加入剛成立的聲優經紀公司[[ARTSVISION|ARTSVISION]]<ref>《私{{lang|ja|も悩んでおとなになった}}》，第85～91頁。</ref>。之後她又在富野由悠季執導、接前作重戰機L-Gaim時段播出的《[[機動戰士Z_GUNDAM|機動戰士Z鋼彈]]》（聲演{{link-ja|貝托蒂嘉·伊爾瑪|ベルトーチカ・イルマ}}）繼續出演。

川村經常出演高傲、氣勢強人系統的角色，因此與同世代的[[鶴弘美|鶴弘美]]、[[松井菜櫻子|松井菜櫻子]]及[[富澤美智惠|富澤美智惠]]都博得了很高的人氣而為人熟知。

[[日高範子|日高範子]]以前在電台廣播節目中，曾經跟擔任來賓的川村與[[松井菜櫻子|松井菜櫻子]]共3人一起互相進行[[女王|女王大人]]的聲音PK。還有跟松井、後輩[[林原惠|林原惠]]3人一起被稱之為看到她們就「抬不起頭來的聲優夥伴」之一。

2001年，川村因罹患急性疾病必須休業，此時不得不卸下正在播出的[[NHK教育頻道|NHK教育頻道]]幼教節目《{{link-ja|Goo Choco Lantan|ぐ〜チョコランタン}}》角色小湯匙的配音，由與她親近的後輩[[橘光|橘光]]繼任。

2003年，川村離開出道公司ARTSVISION，從此以[[自由職業|自由聲優]]身份持續活動。2010年12月起，現在擔任數家經紀公司部份營業業務的委託、提攜。

2011年自《{{link-ja|HobbyJAPAN|月刊ホビージャパン|月刊HobbyJAPAN}}》（從{{link-ja|Charano!|キャラの!|Charano！}}移籍）10月號開始，進行新書《川村萬梨阿與[[椋本夏夜|椋本夏夜]]的淑女日常（<small>{{lang-ja|川村万梨阿&椋本夏夜の淑女的日常}}</small>）專欄執筆。

2014年5月10日，啟用期間限定官方[[Twitter|Twitter]]。

2014年6月18日，發行歌手活動31週年紀念專輯BOX「ΟΔΥΣΣΕΙΑ（中譯奧德賽）」。同年10月2日，於官方網站公開作為出道紀念私人製作的“時裝娃娃momoko"『川村萬梨阿momoko』的2種拍攝照片<ref>{{Cite web|url=http://automaticflowers.ne.jp/maria/marianow/archives/others/index.html|script-title=ja:【お知らせ、その他】Automatic Flowers MARIA KAWAMURA|work=川村萬梨阿官方個人網站|accessdate=2015-01-15|language=ja}}</ref><ref>{{Cite web|url=http://www.petworks.co.jp/doll/pwmomoko/ae/ae_20.html|script-title=ja:【川村万梨阿が「花の詩女 ゴティックメード」劇場公開時に舞台挨拶やイベン】PW-momoko「MARIA KAWAMURA momoko」官網|work=|accessdate=2015-01-15|language=ja}}</ref>。

== Cosplay ==
上面已經提到，川村有作為動畫迷的活動經歷，多次在公開活動場合進行[[Cosplay|Cosplay]]。
* 1981年2月22日，《[[機動戰士GUNDAM|機動戰士鋼彈]]》動畫電影上映，於[[新宿站|新宿東口]]的活動「動畫新世紀宣言」作為「粉絲代表」扮成{{link-ja|拉拉·遜|ララァ・スン}}的造型登場。
* 《[[福星小子|福星小子]]》動畫紀念活動上，扮成作品女主角[[拉姆_(福星小子)|拉姆]]，並作為寫真偶像彩色照片有刊登在《[[週刊少年Sunday|週刊少年Sunday]]》雜誌上。
* 《[[聖戰士DUNBINE|聖戰士鄧拜音]]》本是川村的自身配音出道作，但她以寫真偶像扮成自己聲演的角色恰姆·法烏穿著韻律服、背上還有添加翅膀的照片刊登在《{{link-ja|animec|アニメック}}》雜誌上。
* 前往美國參加的動畫粉絲招待活動時，扮成《[[秀逗魔導士_(動畫)|秀逗魔導士]]》的[[白蛇娜卡|白蛇娜卡]]現身舞台獻唱「能{{lang|ja|あるライバルは}}爪{{lang|ja|を隠す}}」。並講了一句自己設定的台詞「娜卡來到美國了」。

== 交友關係 ==
川村在出演由[[富野由悠季|富野由悠季]]執導、自身的動畫配音出道作《[[聖戰士DUNBINE|聖戰士鄧拜音]]》時，被富野被給予「萬梨阿」的漢字藝名<ref>《私{{lang|ja|も悩んでおとなになった}}》，第74頁。</ref>。在此之前，川村曾提出用「{{lang|ja|マリア}}（中文音譯“瑪麗亞”）」這個名字。

與漫畫家[[新谷薰|新谷薰]]是高中時期的熟人，他的漫畫《{{link-ja|妙齡女大亨|クレオパトラD.C.}}》決定製作OVA動畫時，川村當初是以史恩的配音人選參加。由於原作者新谷的要求，主人翁克婁巴特拉的配音因此允諾由她擔任<ref>出自《妙齡女大亨D.C 完整DVD》附錄小冊子「QUEEN OF FORTUNE」原作者 新谷薰專訪。</ref>。

自身配音出道作《聖戰士鄧拜音》跟她一起共演的主角飾演者[[中原茂|中原茂]]，在錄音的空檔因為經常受到他的照顧，至今2人也是持續進行交流、並以「小茂（<small>{{lang-ja|茂ちゃん}}</small>）」和「小萬梨阿（<small>{{lang-ja|万梨阿ちゃん}}</small>）」互相用暱稱稱呼的事業夥伴關係。

== 模特兒角色 ==
根據動畫雜誌《The Anime》的記載，《[[戰鬥裝甲Xabungle|戰鬥機甲薩芬格爾]]》的瑪麗亞·瑪利亞是以川村為模特兒。但這是川村作為專業配音員出道之前還是「粉絲代表」的逸事了。而該角色的配音是由[[島本須美|島本須美]]擔任。

漫畫家[[結城正美|結城正美]]因為跟川村熟識，所以漫畫《{{link-ja|究極超人R|究極超人あ〜る}}》登場的角色西園寺瑪麗是以她為模特兒。之後決定錄製成廣播劇CD和製作OVA動畫時，西園寺瑪麗的配音同樣由川村擔任。此時，川村發牢騷的說「託這個角色的福，除了高傲千金的角色之外都不來了」，但工作人員卻稱讚她說，沒有比這個最受歡迎角色的了（來自廣播劇CD的歌詞卡）。

[[矢野健太郎|矢野健太郎]]漫畫《{{lang|ja|インジュカーシス}}》的一些專有名詞來自現實存在的地名，或者是人名的羅馬拼音反向拼寫，如「{{lang|ja|アイラム}}={{lang|ja|アルマワック}}（AIRAM-ARUMAWAK）王家」、登場人物之中一位「{{lang|ja|ウロマモ}}={{lang|ja|ナガン}}（UROMAMO-NAGAN）」等等，都是將川村（或永野）她的姓氏作為使用的由來。

== 家族 ==
川村與[[設計師|設計師]]、[[漫畫家|漫畫家]][[永野護|永野護]]交往12年之後，於1991年11月舉行婚禮。[[富野由悠季|富野由悠季]]擔任證婚人<ref>《Newtype別冊 漫畫GENKi》1992年1月號，[[角川書店|角川書店]]。</ref>。

上面提到「動畫新世紀宣言」活動當時的永野Cosplay扮成[[夏亞·阿茲納布爾|夏亞·阿茲納布爾]]做大聲朗讀聲明<ref>{{link-ja|冰川龍介|氷川竜介}}、{{link-ja|藤津亮太|藤津亮太}}編「第三章 映{{lang|ja|画}}版{{lang|ja|への}}志 解{{lang|ja|説}} ～新世紀宣言～」《來自鋼彈現場 富野由悠季發言集》，Kinema旬報社，2000年10月16日，第167頁 ISBN 4-87376-537-4</ref><ref>{{link-ja|小牧雅伸|小牧雅伸}}《{{lang|ja|アニメックの}}頃… 編集長{{lang|ja|ま}}奮闘記》，[[NTT出版|NTT出版]]，2009年，第219頁。</ref>。然而，他在後來的活動曾經公開表示，聽到從台下站在最前面的粉絲正在討談論著說（在1980年代中期）“那可是個禁句”。

在大多數情況下，川村以不同的方式主要參加跟永野作品相關的動畫：
* 在《[[五星物語|五星物語]]》的形象專輯中，她負責聲樂主唱。並唱了[[朋克搖滾|龐克搖滾]]、[[搖滾樂|搖滾]]、[[流行_(音樂類型)|流行樂]]、[[敘事曲|敘事曲]]等等各種音樂。此外，當丈夫的漫畫《五星物語》決定製作動畫電影時，女主角[[拉刻西斯|拉刻西斯]]的配音由妻子擔任。
* 由永野擔任導演的第一部動畫電影作品《[[花之詩女_GOTHICMADE|花之詩女 GOTHICMADE]]》中，川村擔任主角貝琳·阿潔麗的配音、主題歌「空之皇子 花之詩女（<small>{{lang-ja|空の皇子 花の詩女}}</small>）」的填詞和主唱。
