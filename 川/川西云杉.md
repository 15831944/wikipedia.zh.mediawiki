'''川西云杉'''（[[学名|学名]]：{{lang|la|''Picea likiangensis'' var. ''balfouriana''}}）为[[松科|松科]][[云杉属|云杉属]][[丽江云杉|丽江云杉]]的[[变种|变种]]。

== 形态 ==
常绿[[乔木|乔木]]，高可达40米。淡黄灰色小枝，有密毛；条形叶片有气孔线；圆柱形[[球果|球果]]，成熟前为紫红色或紫黑色，斜方状卵形薄种鳞。

== 分布 ==
分布于[[中国大陆|中国大陆]]的[[青海|青海]]、[[西藏|西藏]]、[[四川|四川]]等地，生长于海拔3,000米至4,000米的地区，常生长在针叶树混交林，目前尚未由人工引种栽培。

== 别名 ==
西康云杉（中国树木分类学），水平杉（中国裸子植物志）
