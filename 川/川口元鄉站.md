'''川口元鄉站'''（{{jpn|j=川口元郷駅|hg=かわぐちもとごうえき|rm=Kawaguchi-motogō eki|f=y}}）是一個位於[[埼玉縣|埼玉縣]][[川口市|川口市]]{{link-ja|元鄉 (川口市)|元郷 (川口市)|元鄉}}一丁目，屬於[[埼玉高速鐵道線|埼玉高速鐵道埼玉球場線]]的[[鐵路車站|鐵路車站]]。車站編號是'''SR 20'''。

== 歷史 ==
* 2001年（[[平成|平成]]13年）3月28日 - 開業。
* 2002年（平成14年）11月10日 - 站前廣場完工，站前廣場側新設升降機。

== 車站結構 ==
島式1面2線的[[地底車站|地底車站]]。裝設了保安用的[[月台閘門|月台閘門]]系統。車站顏色是紫色。

地下2樓是閘口層，地下3樓是月台層。出入口設有2個，[[扶手電梯|扶手電梯]]只設於兩個出入口的上行方向。

== 使用情況 ==
