<strong>川田利明 ( 1963年12月8日 - )</strong>   日本的男子[[職業摔角|職業摔角]]選手<ref>{{Cite web|url=https://ja.wikipedia.org/wiki/%E5%B7%9D%E7%94%B0%E5%88%A9%E6%98%8E|title=|accessdate=|author=|date=|publisher=}}</ref>。同時也是摔角手、企業家、料理家、藝人。

他出道於1982年，以踢技聞名。跟同樣是摔角手的三澤光晴是高中學長學弟關係的好友，近來淡出摔角界重心轉往自創品牌-"麺ジャラスK 店長" 的拉麵店。

== 生平 ==
川田利明出生於梔木縣[[都賀町|都賀郡]][[大平町|大平町]]（現：[[栃木市|栃木市]]）。 高中時加入[[摔跤|角力]]社與大一歲的三澤光晴相識，1982年高中畢業後由三澤光晴介紹加入全日本摔角，陸續加入超世代軍、勝鬼軍等，與[[三澤光晴|三澤光晴]]、[[小橋建太|小橋建太]]、[[田上明|田上明]]並稱為全日四天王。

== 主要經歷 ==

=== 1982年 ===
10月4日於[[千葉|千葉]]・大原町消防署前的大會對冬木弘道一戰後出道。

=== 1985年 ===
赴派往海外修行。

=== 1987年 ===
* 加入[[天龍源一郎|天龍源一郎]]的「Revolution」。
* 與參孫冬木( 後改為冬木弘道 ) 組成「フットルース」

=== 1990年 ===
* 加入三澤光晴的「超世代軍」。

=== 1992年 ===
* 與田上明組成「勝鬼軍」。

=== 2010年 ===
* 「麺ジャラスK」 [[拉面|拉麵]]店開幕。


=== 全日本職業摔角 ===

* 三冠重量級王座：第12、18、21、28、32代。
*世界雙打王座：第17、20、22、26、28、31、35、42、52代。
* 亞洲雙打王座：第45代、第47代、第49代。
* 冠軍嘉年華優勝：1994年、1997年。
* 世界最強雙打決定賽 : 1992年。

=== ZERO1 ===

* 世界重量級王座 : 第6代。

== 軼事 ==
* 常被稱為「名勝負製造機」。
* 出生時是[[產婆|產婆]]來家裡接生的。
* 自傳裡曾提到本身是一個喜歡說話人，常會滔滔不絕的說不停。
* 自己表示過除了摔角外喜歡唱歌、演戲等的表演。
* 摔角選手[[佐佐木健介|佐佐木健介]] 曾表示川田的踢擊是地表最強。

== 常見招式 ==
* '''顔面踢'''  （以踢擊攻擊頭部、臉部，另有對手在角落時踩第2根繩索並做出頭部踢擊的方式）
*'''跳躍式顏面踢'''  （跳躍後以踢擊攻擊頭部、臉部，此踢擊比一般顏面踢更為強大，常作為決勝技 )
* '''炸彈摔'''  （因為壓制時常會整個人壓下，所以此技也被稱為川田式炸彈摔）
* '''垂直落下技'''  （以吊車翻摔技的方式舉起對手，卻是以垂直落下的方式落下）
* '''垂岩石落下技'''
* '''垂顏面固定+身體扭轉'''
* '''金臂勾'''
* '''岩石落下技'''
* '''垂直式炸彈摔   (''' 此技為1991年11月22日對三澤光晴三冠戰時，所意外發明。三澤原欲對川田所使出的炸彈摔做夾腳翻摔反擊，但卻被川田識破而動作中斷，川田也就索性以垂直方式摔下)
* '''足球式顏面踢'''  （對手呈坐姿時，以助跑方式並作出[[足球|足球]][[射门|射門式]]的踢擊攻擊對手的頭部或側頭部 )
* '''連續式顏面踢'''
* '''各種踢技'''
