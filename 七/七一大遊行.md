'''七一大遊行'''是[[香港|香港]]自[[香港回歸|主權移交]]以來最為持續的大型活動之一。一般認為，七一大遊行源於[[2003年香港七一遊行|2003年香港七一遊行]]，然則，自1997年[[香港回歸|香港主權移交]]後，香港[[民間人權陣線|民間人權陣線]]等社會運動組織已發起有關活動，但規模較小。

2003年，由於[[香港特别行政区政府|香港特别行政区政府]]施政惡劣：時任財政司司長[[梁錦松|梁錦松]]「偷步買車」風波、時任[[保安局局長|保安局局長]][[葉劉淑儀|葉劉淑儀]]強推《[[香港特別行政區基本法第二十三條|基本法第二十三條]]》立法、時任衞生福利及食物局局長[[楊永強|楊永強]]在「[[非典型肺炎|沙士]]」事件中的失職，激起民怨，七一大遊行由是應運而生。50萬香港人在這一年的7月1日回歸慶典中上街，但仍在運動中保持了守秩序、非暴力的作風。當年的七一大遊行是香港繼1989年5月28日150萬人參加[[全球華人大遊行|全球華人大遊行]]及1989年5月21日[[八九民運|八九民運]]100萬人大遊行後最大型一次遊行活動。最終，基本法第二十三條立法被擱置，而七一大遊行亦成為了香港民主抗爭的精神象徵，每年舉行，由民間人權陣線主辦。[[泛民主派|泛民主派]]政治團體藉遊行匯眾支持者並進行籌款。

== 1997年至2002年 ==
自1997年至2002年，[[香港市民支援愛國民主運動聯合會|香港市民支援愛國民主運動聯合會]]（簡稱支聯會）都會於7月1日舉行遊行，由[[銅鑼灣|銅鑼灣]][[維多利亞公園|維多利亞公園]]（簡稱維園）遊行到[[中環|中環]][[中區政府合署|中區政府合署]]。

1997年7月1日下午，支聯會以「愛國愛港民主大遊行」的名義冒雨遊行，稱要建設民主中國。遊行開始時有800多個市民參加，但隊伍去到[[灣仔|灣仔]]時就則增加至2,300人，警方估計人數有4,000人（支聯會估計有3,000人），多於支聯會預先通知的2,000人，便口頭警告支聯會。支聯會成員[[張文光|張文光]]表示，警方以前從來未有警告支聯會，希望只是巧合事件，而警方就表示期望主辦團體可以更準確估計遊行人數。遊行隊伍最後抵達政府總部門外，由支聯會主席[[司徒華|司徒華]]代表遞交意見書給[[香港特別行政區行政長官|香港特別行政區行政長官]][[董建華|董建華]]代表之後和平散去。

== 2003年 ==
[[File:Article23demonstration.jpg|thumb]][[軒尼詩道|軒尼詩道]]|left]]
{{Main|2003年香港七一遊行}}
2003年7月1日的「七一遊行」，是[[八九民運|八九民運]]以來最浩瀚澎湃的遊行示威，受到國際輿論重視，由於[[香港基本法第二十三條|香港基本法第二十三條]]的立法程序，加上[[SARS事件|SARS事件]]導致香港社會不景氣，引起大量香港市民的不滿。遊行主題為「反對廿三，還政於民」，大會呼籲市民穿黑色衣服參與遊行，以表達對政府的不滿。雖然當日香港天氣炎熱，但有大量人士參與遊行。主辦團體[[民間人權陣線|民間人權陣線]]（簡稱民陣）估計遊行人數超過50萬，而警方則公佈當日截至下午六時，由起點到終點之間共有35萬人。[[支聯會|支聯會]]主席[[司徒華|司徒華]]在回憶錄中提到，他收到消息，警方內部獲悉的遊行人數為67萬5千人。遊行所引發的[[七一效應|七一效應]]，加上自由黨撤回對立法的支持，使香港政府停止對廿三條立法程序，並使[[建制派|建制派]]在同年的[[2003年香港區議會選舉|香港區議會選舉]]大敗。

[[路透社|路透社]]、[[美聯社|美聯社]]、[[法新社|法新社]]、[[美國有線新聞網|美國有線新聞網]]、[[英國廣播公司|英國廣播公司]]以及[[德新社|德新社]]等西方通訊社，圖文並茂報道香港的大遊行。美聯社形容，示威人士是「憤怒和憂慮的香港人」；路透社稱：「示威者來自社會各階層，包括商人、退休人士，也有年輕夫婦推嬰兒車，與著名的民主派人士一起遊行。」。<ref>[http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20030702&sec_id=4104&subsec_id=12731&art_id=3386625 港人遊行　舉世矚目]蘋果日報 2003年07月02日</ref>

== 2004年 ==
[[File:71marches-2004-CWB.JPG|thumb]]
[[File:71marches-2004-hsbc.JPG|thumb]]
2004年，當天是有紀錄以來最高溫的7月1日，幾十萬市民會師[[銅鑼灣|銅鑼灣]][[維園|維園]]，參加7.1大遊行，爭取2007年普選行政長官、2008年立法會全面直選。人群早在下午二時半已擠滿維園六個足球場，主題為「爭取07、08[[真普選|普選]]」，大會呼籲市民穿白色衣服參與遊行，以表達對普選的期望。另一方面，有親政府團體呼籲市民穿紅色衣服，參加另外一個要求民主的靜坐集會，大遊行於晚上八時在平靜中完結，行政長官[[董建華|董建華]]於九時半後發表講話，重複會根據人大釋法的決定，循序漸進推動民主，最後董建華分別以中、英文朗讀聲明一次之後，卻以「時間太晚」為由拒絕回答在場中外記者的提問。<ref>[http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20040702&sec_id=4104&subsec_id=11866&art_id=4148185 中國人的驕傲　香港53萬人上街] [[蘋果日報_(香港)|蘋果日報 (香港)]] 2004年7月2日</ref>

雖然當日香港天氣炎熱，但2004年的“七一”遊行人數依然高企，主辦單位指人數估計有高達53萬，較2003年的50萬人還要多，警方指只有20萬人，統計數字引起了爭議，因為多個統計都是不足20萬，香港大學統計及精算學系高級講師[[葉兆輝|葉兆輝]]指出，遊行人數最多有19萬2千人。<ref>{{cite news|url=http://news.sina.com.hk/cgi-bin/news/show_news_f2.cgi?type=focus&name=20040628&id=443103|title=【今年七一遊行相關新聞】 葉兆輝：電腦分析 19.2萬人遊行|date=2005-06-28}}</ref>其後甚至有資料顯示，民陣的計算中犯上了小學生常犯的「植樹問題」，即把兩輪多一點的遊行人數，錯計成三輪；亦即單靠這算術錯誤，就使報稱的遊行人數增大了十多萬。<ref>{{cite news|url=http://hkupop.hku.hk/chinese/columns/columns105.html|title=遊行集會人數不應各說各話}}</ref>縱使如此，遊行人數始終並非少數。

== 2005年 ==
[[File:71marches-2005.JPG|thumb]]
[[File:71marches-2005-EthnicMinority.JPG|thumb]]
[[File:71marches-2005-child.JPG|thumb]]
2005年7月1日，儘管經濟已經復甦，董建華已經下台，仍然有萬人參加七一遊行，主題為「反對官商勾結，爭取全面普選」，大會再次呼籲市民穿白色衣服參與遊行，以表達對[[普選|普選]]的期望。此外，民陣公投小組計劃於2005年七一遊行時在維園進行「七一模擬[[公投|公投]]」，公投議題是「〇七、〇八年普選行政長官及立法會」。

但與往年不同的是，這次遊行有不少爭取其它訴求為主的人士加入，如爭取原工資水平的海外家庭傭工、反對[[香港大學醫學院|香港大學醫學院]]冠上贊助10億港元的富商[[李嘉誠|李嘉誠]]之姓名的教學人員和校友、爭取平等權益的[[同性戀者|同性戀者]]、爭取居港權人士以及一些[[法輪功|法輪功]]的修練者等。

這一次遊行之前發生了一些插曲，保守派基督教團體[[明光社|明光社]]高調反對民陣容許在同性戀權利團體帶領隊伍下一同遊行，呼籲教徒不要參與，引起了社會不少議論。

根據香港警方在維園出發點的統計數字，約有11,000名市民參與遊行，而民陣起初表示有45,000人參加，及後修正遊行人數為21,000人。然而，有遊行人士懷疑警方把人數大減，比起當天上午的慶祝回歸巡遊還要少（警方估計有2萬多人參與）。<ref>[http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20050702&sec_id=4104&subsec_id=11866&art_id=5017921 香港換了特首民間各表訴求 2.1萬人遊行]蘋果日報 2005年07月02日</ref>

此外，2004年人大釋法決定2007年及2008年不實行普選，董建華下台，新政府剛上任，也是遊行人數減少因素之一。

== 2006年 ==
[[File:Anson_Chan_at_July_1_march_2006.jpg|thumb]]
[[File:Hong_Kong_July_1_Marches.jpg|thumb]]外之天橋）]]
這年的七一以「平等公義新香港，民主普選創希望」為主題，之前07/08的政改方案被泛民主派在立法會否決。舉行前，前政務司長[[陳方安生|陳方安生]]發言支持香港實行普選，表示會參加遊行，並呼籲市民上街表達訴求。陳太此舉一度引來外間揣測她為參選下一屆特首鋪路，當時陳太對此的回應不置可否，及後至同年九月陳方安生正式宣布無意參選特首。

一如往年，當日上午有親中社團主辦的慶祝香港回歸大巡遊，下午則有民陣發起的遊行。民陣的遊行在下午三時出發，路線和以往一樣，由維園至中區政府合署。主辦單位估計有58,000人參加了這次的遊行，比上年的多近1倍。遊行大約在七時和平結束。

== 2007年 ==

這年的七一以「爭取普選，改善民生」為主題。遊行前20日左右，正式向[[香港警方|香港警方]]申請，而警方表示希望遊行於傍晚6時前完成遊行，理由是晚上在[[維多利亞港|維多利亞港]]發放煙花匯演，並擔憂民陣讓長者及傷殘人士排在隊伍前列會拖慢人流速度。但此舉引起泛民主派的不滿，指出警方擔憂行進緩慢但只開放一條行車線並不合理<ref>[http://www1.appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20070618&sec_id=4104&subsec_id=11867&art_id=7225866 7.1遊行警提多項無理要求 建議非長者帶隊 輪椅不可逾10部] {{webarchive|url=https://web.archive.org/web/20070620233723/http://www1.appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20070618&sec_id=4104&subsec_id=11867&art_id=7225866 |date=2007-06-20 }}，[[蘋果日報|蘋果日報]]，2007年6月18日</ref>，而且認為警方歧視傷殘人士<ref>[http://appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20070621&sec_id=4104&subsec_id=11867&art_id=7237474 7.1前 民陣成員遭竊聽 孔令瑜指03年來最嚴重 擬周六遊行抗議] {{webarchive|url=https://web.archive.org/web/20070626031058/http://www1.appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20070621&sec_id=4104&subsec_id=11867&art_id=7237474 |date=2007-06-26 }}，蘋果日報，2007年6月21日</ref>，遂提出司法覆核<ref>[https://web.archive.org/web/20070626132330/http://hk.news.yahoo.com/070620/12/29vsl.html 不滿7．1遊行限制 泛民擬申司法覆核] 《明報》2007年06月21日 星期四 05:05AM</ref>。6月26日，公眾集會及遊行上訴委員會裁定，遊行需於4小時內結束，要提早至2時半開始，6時半要結束<ref>[http://www.rthk.org.hk/rthk/news/expressnews/news.htm?expressnews&20070626&55&411015 民陣七一遊行被裁定可於四小時內完結]，[[香港電台|香港電台]]，2007年6月26日</ref>。此外，警方亦需開放3條行車線予遊行人士使用。民陣對此表示滿意<ref>[http://www.rthk.org.hk/rthk/news/expressnews/news.htm?expressnews&20070626&55&411021 民陣滿意七一遊行上訴結果]，香港電台，2007年6月26日</ref>。

== 2009年 ==
這年的七一遊行仍由民間人權陣線發起。大會估計有7.6萬人參加，但警方表示在維園出發時約有2.6萬人。由於當時市區氣溫高達33度，有不少網民表示為免身體不適而在中途插入或中途離開。

民陣發起的七一大遊行，主題是「施政失誤、貧富懸殊、還政於民、改善民生」，隊伍於下午3時半在維園出發，目的地是中區政府合署，龍尾遲至近五時才出發，晚上近8時抵達政府總部。其後，包括立法會議員[[梁國雄|梁國雄]]在內約一百名自發人士，在中區政府合署繼續集會，故遊行在7月2日凌晨1時仍未結束。警方在現場增加警力，使到位於政府總部的[[警察機動部隊|警察機動部隊]]人員逾百名，比集會人士多<ref>商業一台 1：30新聞簡報</ref>。警方其後在凌晨2時清場，把示威人士勸諭離開或抬走。

同日亦有其他團體分別表達不同訴求，如雷曼苦主大聯盟、捍衛人格尊嚴協會、反泛民大聯盟等，但規模比民陣小得多。

== 2010年 ==
香港主權移交踏入13周年，本年度亦有多個團體或組織在當日進行慶祝或抗議遊行。

民陣發起的遊行於下午3時半在維園出發，今年主題是「七一向前走，香港前途在我手」，目的地是中區政府合署，晚上近八時抵達。同日遊行中亦有其他團體分別表達不同訴求，包括民間爭取最低工資聯盟。民陣責備香港政府今年遊行申請限制比往年多。今年以「七一向前走，香港前途在我手」為遊行主題。遊行過後，召集人[[李偉儀|李偉儀]]直言政改在七一前表決對遊行人數有影響，有52,000人參加<ref>[http://news.sina.com.hk/cgi-bin/nw/show.cgi/3/1/1/1772716/1.html/ 明報2010年7月2日新聞via新浪香港]</ref>。

2012年政改方案在2010年6月23日通過，多個泛民主派組織主要口號強調廢除[[功能界別|功能組別]]。今年七一遊行最特別的地方是[[民主黨_(香港)|民主黨]]因支持[[香港政府|香港政府]]提出經民主黨同意改良的政改方案，決定「行尾」減少與示威者發生衝突，在遊行期間，不斷被反對政改的示威人士辱罵和滋擾，當中包括[[社會民主連線|社民連]]黨員。身兼支聯會主席、患病的民主黨元老[[司徒華|司徒華]]更被投擲陰司紙。今年的七一籌款數字亦由去年的30萬元暴跌至今年的45,000元，劇減了85%。遠遜於反對政改、各籌得逾20萬元的公民黨及社民連。<ref name="disturb">[http://news.sina.com.hk/cgi-bin/nw/show.cgi/32/1/1/1772387/1.html 星島日報2010年7月1日新聞via新浪香港]</ref>

== 2011年 ==
[[File:2011_Hong_Kong_July_1_march_6070.JPG|250px]]外牆上]]
香港主權移交踏入第14年，口號為「還我[[2012雙普選|2012雙普選]]，打倒[[地產霸權|地產霸權]]，[[曾蔭權|曾蔭權]]下台」。七年以來第一次以曾蔭權下台為口號，另外亦有大量市民要求[[唐英年|唐英年]]為施政失誤下台，[[曾俊華|曾俊華]]為[[2011年至2012年度香港政府財政預算案|財政預算案]]和[[上網學習支援計劃|政治干預]]下台，[[林瑞麟|林瑞麟]]為[[立法會遞補機制爭議|立法會遞補機制爭議]]下台。主辦單位民間人權陣線本年度之召集人為[[新民主同盟|新民主同盟]]輪任召集人[[范國威|范國威]]。

遊行前警方在不反對通知書上的限制，包括<ref>[http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20110624&sec_id=4104&art_id=15371070 禁玩樂器　不准籌款　遊行完即解散 警陰招打壓 7.1]</ref>：
* 民陣須確保遊行人士到達[[中區政府合署|中區政府合署]]特定範圍後隨即解散。
* 民陣須確保參與者未獲發許可證情況下不得擅自作任何籌款，賣旗等活動，否則可能被控。
* 任何人無合法權限或解釋在公眾或道路上奏玩樂器即屬犯罪。
* 須確保遊行隊伍以正常速度前進，遊行過程中要避免任何令遊行隊伍速度減慢的行動，如籌款、販賣及簽名活動。
* 須確保所有遊行參者在行政署長批准時間內解散，不會組織或鼓勵任何參加者繼續以集體方式前往其他目的地舉行公眾活動。

藝術公民發言人鄧小樺指，不少藝術界和年輕人有市民形容警方禁止奏樂是向市民宣戰<ref>[http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20110624&sec_id=4104&subsec_id=12731&art_id=15371071 「拿樂器又不是拿武器」]</ref>民陣提出上訴，公眾集會及遊行上訴委員會裁定，民陣只須協助警方而不是確保遊行人士離開，遊行人士也可以使用樂器，所以警方要修改「不反對通知書」內的字眼。<ref>[http://news.sina.com.hk/news/2/1/1/2368436/1.html 警﹕七一遊行可奏樂器] {{webarchive|url=https://web.archive.org/web/20110702063305/http://news.sina.com.hk/news/2/1/1/2368436/1.html |date=2011-07-02 }}</ref>。

== 2012年 ==
[[File:July_1_march_in_Hong_Kong_in_20120701.jpg|right]]
[[File:July_1_march_in_Hong_Kong-Bankruptcy_of_Lehman_Brothers_2012.jpg|right]]
香港主權移交踏入第15年，遊行主題為「踢走黨官商勾結，捍衛自由爭民主」，寓意[[梁振英|梁振英]]為首的黨官商利益集團，只會繼續拖延民主進程和維持特權利益輸送。因梁振英被指強權統治，有人呼籲香港市民更應提高警覺，繼續以行動捍衛自由，爭取民主普選。

主辦單位指出今年大量市民上街遊行主要因為內地民運人士[[李旺陽|李旺陽]]離奇死亡事件，中聯辦被指介入香港事務，以及特首梁振英的僭建問題等因素致令大量市民上街遊行。

多名雷曼苦主亦參加遊行，批評立法會就雷曼事件發表的調查報告沒有全面解決有關問題。<ref>[http://orientaldaily.on.cc/cnt/news/20120702/00176_023.html 市民訴求 百花齊放]，東方日報，2012年07月02日</ref>

香港主權移交踏入第21年，遊行繼續由民陣發起，今年遊行主題為「結束一黨專政，拒絕香港淪陷」。警方今年繼續要求遊行者在維園中央草坪作起點，[[警務處處長|警務處處長]]稱可以確保安全，避免出現公共秩序隱憂和混亂<ref>{{Cite web |url=http://std.stheadline.com/daily/news-content.php?id=1824617&target=2 |title=一哥：七一維園起步 可保秩序安全 |date=2018-06-24 |publisher=星島日報}}</ref>，維園足球場也與往年一樣由慶回歸活動佔用，民陣提出的以[[銅鑼灣|銅鑼灣]][[東角道|東角道]]起步建議遭警方否決<ref>{{Cite web |url=https://www.inmediahk.net/node/1057977 |title=七一遊行維園草坪起步 民陣籲市民避免衝突 歡迎沿途加入 |date=2018-06-28 |publisher=香港獨立媒體}}</ref><ref>{{Cite web |url=http://news.rthk.hk/rthk/ch/component/k2/1404590-20180701.htm |title=七一遊行維園中央草坪起步　民陣冀約6萬人參與 |date=2018-07-01 |publisher=香港電台}}</ref>。遊行期間，警方在銅鑼灣[[崇光百貨|崇光百貨]]外架起鐵馬封閉斑馬線，以阻止市民中途加入隊伍，警方指中途加入屬「非法集會」。有市民遂改在[[希慎廣場|希慎廣場]]外加入隊伍，也有市民稱新起步地點不方便，所以在希慎廣場外插隊<ref>{{Cite web |url=http://news.rthk.hk/rthk/ch/component/k2/1404642-20180701.htm |title=市民中途加入遊行　形容警方稱屬非法集會「離譜」 |date=2018-07-01 |publisher=香港電台}}</ref><ref>{{Cite web |url=http://news.rthk.hk/rthk/ch/component/k2/1404629-20180701.htm |title=民陣七一遊行隊伍抵政總　有市民響應呼籲中途加入 |date=2018-07-01 |publisher=香港電台}}</ref>。這一年警方公佈的遊行人數是[[2003年香港七一遊行|2003年七一大遊行]]以來最低，只有9,800人，也是16年以來首次低於10,000人<ref>{{Cite web |url=https://news.mingpao.com/ins/instantnews/web_tc/article/20180701/s00001/1530447586069 |title=警稱高峰時9800人 成03年以來最低 16年來首跌穿五位數|date=2018-07-01 |publisher=明報}}</ref><ref>[http://std.stheadline.com/instant/articles/detail/754891-%E9%A6%99%E6%B8%AF-%E3%80%90%E4%B8%83%E4%B8%80%E9%81%8A%E8%A1%8C%E3%80%91%E6%B0%91%E9%99%A3%EF%BC%9A%E6%BB%BF%E6%84%8F5%E8%90%AC%E4%BA%BA%E5%8F%83%E8%88%87+%E8%AD%A6%E6%96%B9%E6%8C%87%E6%9C%80%E9%AB%98%E5%B3%B09800%E4%BA%BA%E5%89%B5%E6%96%B0%E4%BD%8E 【七一遊行】民陣：滿意5萬人參與 警方指最高峰9800人創新低]，星島日報，2018-07-01</ref><ref>[http://hk.on.cc/hk/bkn/cnt/news/20180701/bkn-20180701144605246-0701_00822_001.html?eventid=4028810746c809070146d7527adc00e0&eventsection=hk_news 七一遊行民陣稱5萬人參加 警指最高峰僅9800人]，on.cc東網，2018-07-01</ref><ref>[https://hk.news.appledaily.com/local/daily/article/20180702/20437716 5萬人7.1上街：林鄭月娥 咪當我儍]，蘋果日報，2018-07-02</ref>。

=== 人數點算爭議 ===
==== 2012年 ====
[[香港警務處|香港警務處]]公佈了兩組數字，分別是有5.5萬人從維園起步和最高峰時期遊行者有6.3萬人。香港大學民意研究在[[軒尼詩道|軒尼詩道]]及[[軍器廠街|軍器廠街]]交界的[[天橋|天橋]]點算人數，約6.8萬人經過了該處。根據2011年的數字，遊行人士中有65.6%經過該處，故此推算2012年的總遊行人士在乎於9.8萬至11.2萬人之間。香港大學社會工作及社會行政學系教授[[葉兆輝|葉兆輝]]團隊則在靠近起點的銅鑼灣以及靠近終點的金鐘分別派遣人員進行統計，點算出約5萬人經過了銅鑼灣、約6.9萬人經過了金鐘，同時又派遣人員到金鐘訪問約1,000名遊行人士，發現當中46%名受訪者沒有經過銅鑼灣，即是於中途加入遊行隊伍，故此以此推算整體遊行人數約8.2萬人。

於2003年擔任民主人權陣線召集人的[[蔡耀昌|蔡耀昌]]表示，2012年的七一遊行人數遠多於2011年（大會稱有21.8萬），但是與2003年仍然有所差距。他稱2003年遊行人士逼滿整座[[維多利亞公園|維多利亞公園]]，包括球場和草地，並且在晚上7時許龍尾才離開公園，2012年則只是逼滿了球場，龍尾近6時就離開了公園<ref>[http://hk.news.yahoo.com/港大民研-介乎9-8萬至11-2萬-213933096.html 港大民研﹕介乎9.8萬至11.2萬] {{webarchive|url=https://web.archive.org/web/20121102010712/http://hk.news.yahoo.com/%E6%B8%AF%E5%A4%A7%E6%B0%91%E7%A0%94-%E4%BB%8B%E4%B9%8E9-8%E8%90%AC%E8%87%B311-2%E8%90%AC-213933096.html |date=2012-11-02 }} 《明報》 2012年7月2日</ref>。

2012年七一遊行後，有[[網民|網民]]自行製作圖表，列出由2004年至2012年的各方估算遊行人數，藉此發現主辦單位[[民間人權陣線|民間人權陣線]]的估算數字對比[[學者|學者]]的估算數字的差距，從2005年的1.5倍升至2012年的4.5倍，質疑民陣估算的誇大成份愈來愈高。民主人權陣線七一遊行總負責人[[麥德正|麥德正]]稱，在[[金鐘|金鐘]]、[[灣仔|灣仔]]及[[銅鑼灣|銅鑼灣]]有工作人員一邊遊行一邊統計人數，工作人員以遊行人士所覆蓋的路面的密度來推算總遊行人數，同時亦都會將周邊的[[行人|行人]]計算入內。此種做法將所有在行人路上、馬路上、在中途插隊、在中途離隊的人數都計算在內，經常將非遊行人士計算入內。而且工作人員邊遊行邊統計，亦可能令到統計流於主觀。麥德正承認民間人權陣線是以最寬鬆的尺度作為統計，方法未夠科學化，數字也只屬於估算<ref>[http://hk.news.yahoo.com/%E9%81%8A%E8%A1%8C%E4%BA%BA%E6%95%B8%E5%90%B9%E5%99%93%E5%AE%B3%E6%B8%AF-%E7%A7%91%E5%AD%B8%E7%B5%B1%E8%A8%88%E7%90%86%E6%80%A7%E6%B1%82%E7%9C%9F-224743770.html 遊行人數吹噓害港 科學統計理性求真] {{webarchive|url=https://web.archive.org/web/20120709004308/http://hk.news.yahoo.com/%E9%81%8A%E8%A1%8C%E4%BA%BA%E6%95%B8%E5%90%B9%E5%99%93%E5%AE%B3%E6%B8%AF-%E7%A7%91%E5%AD%B8%E7%B5%B1%E8%A8%88%E7%90%86%E6%80%A7%E6%B1%82%E7%9C%9F-224743770.html |date=2012-07-09 }} 《經濟日報》 2012年7月6日</ref>。基於主辦單位表示遊行人數為40萬人，警務處表示為63,000人，參與人數多寡再度成為爭議。[[警務處副處長|警務處副處長]]（行動）[[洪克偉|洪克偉]]表示，警務處統計遊行人數目標純粹為疏導人群，因此着眼於最高峰期的遊行人數，避免有過多人聚集在終點處，堵塞後方而至的人，令到遊行隊伍不能夠順暢地到達目的地。多年來，他為了是否錯數人數問題，一直嘗試以不同的方法來計算。他透露於同年6月，公眾就[[李旺陽事件|李旺陽事件]]到[[中聯辦|中聯辦]]請願，主辦單位表示當日有25,000人參與，警務處表示最高峰期為5,400人。洪克偉表示，由於兩者所得出的人數相距甚遠，於翌日，他使用[[香港報章|香港報章]]從高處攝影中聯辦門前的示威群眾的照片，安排人員計算照片中的人群數目，以數人頭方法逐個點算，發現照片只有逾2,000人。如此類推，倘若根據主辦單位公布的人數，其餘的逾20,000人人龍理應排隊至中環，惟事實並非如此。對於流動人群，警察則會安排數組人員，在制高點計算經過人頭，每數分鐘換人一次，將所得出的人流通過人數為準，以計算遊行參與人數<ref>[http://hk.news.yahoo.com/示威人數屢爭拗-警方計法科學化-220541531.html 示威人數屢爭拗 警方計法科學化] 《星島日報》 2012年11月12日</ref>。
