'''七條底線'''，又称'''鲁七条'''，是2013年8月15日閉幕的“中國互聯-{}-網大會”上發出的倡議。2013年8月10日，[[国家互联网信息办公室|国家互联网信息办公室]]主任[[鲁炜|鲁炜]]召集微博、网络名人进行座谈，提出“七条底线”的共识，并敦促名人遵守该七条原则<ref name="fish " >{{cite web |url=http://news.163.com/13/0825/09/9745L89300014Q4P.html |title=“秦火火”们玩火自焚网上哪些言论会犯法？ |accessdate=2013年8月25日 |publisher=网易 |work=华商晨报 |language=zh-hans}}</ref>。
同时，鲁炜希望名人承担社会责任，作为[[正能量|正能量]]，为实现[[中国梦|中国梦]]而出力<ref>{{cite web |url=http://news.china.com.cn/live/2013-08/13/content_21659942.htm |title=微博"大V"与官员探讨坚守底线--网管释放由堵到疏信号 |accessdate=2013年8月13日 |publisher= 中国网|work=光明网 |language=zh-hans}}</ref>。当日参加会议的名人有：[[纪连海|纪连海]]、[[廖玒|廖玒]]、[[陈里|陈里]]、[[潘石屹|潘石屹]]、[[古永锵|古永锵]]、[[陈彤|陈彤]]、[[周小平|周小平]]、[[薛蛮子|薛蛮子]]、[[徐世平|徐世平]]、[[齐向东|齐向东]]、[[孙健|孙健]]、[[胡延平|胡延平]]、[[张国庆|张国庆]]、[[高龙|高龙]]、[[徐小平|徐小平]]、[[李未柠|李未柠]]等<ref name=" bee" >{{cite web |url=http://news.xinhuanet.com/2013-08/12/c_132621768.htm |title=国信办主任鲁炜与网络“大V”畅谈社会责任  |accessdate= 2013年8月12日|publisher=新华网 |work=新华网 |language=zh-hans}}</ref>。

== 背景 ==
2013年7月，著名女歌手[[吴虹飞|吴虹飞]]在新浪微博称“炸建委”被罚行政拘留10日<ref name="fish " />。

==发起==
8月10日，国家互联网信息办公室主任鲁炜与纪连海、廖玒、陈里、潘石屹、薛蛮子等十多位网络名人举行座谈交流。鲁炜在座谈时就网络名人社会责任提出六点希望，也与在场的网络名人们达成了坚守“七条底线”的共识<ref>[http://bbs1.people.com.cn/post/1/1/2/132304637.html 《強國論壇》:虚拟世界同样要担当 网络名人应坚守“七条底线”]</ref>。

==具体條文内容==
#[[法律|法律]][[法规|法规]]底线：有法可依、有法必依、执法必严、违法必究，任何时候，无论是网上网下，都将始终做到违法必究；
#[[社会主义制度|社会主义制度]]底线：为我们全面建成小康社会提供了有力地制度保障，我们要积极拥护社会主义及社会主义制度；
#[[国家利益|国家利益]]底线：作为国家公民，时刻维护我们伟大祖国的利益，这也是宪法赋予我们每位公民的光荣义务；
#[[公民|公民]]合法权益底线：我们在网络反腐的同时，切记不能以“艳照”等不健康、不正当甚至违法手段对别人进行，否则不仅触犯法律，也侵犯了无辜者的合法权益；
#社会公共秩序底线：网络世界必须也要遵循一定的秩序规则，唯有如此大家才能营造一个良好健康的网络环境；
#[[道德|道德]]风尚底线：崇尚美德在我国延续几千年的优秀传统，网络空间里也要讲道德，不做有违道德之事；
#[[信息|信息]]真实性底线：要求我们在上网时一定要实事求是，而不能以讹传讹、散发谣传，积极宣传政府部门发布的真实信息。<ref name="fish " />

==民间反应==
七條底線发布当日，被大陆各大门户网站放上首页。大部分网民极力反对，指中国政府公然用“制度”钳制言论自由，部分人在大陆[[凯迪社区|凯迪社区]]和个人博客等网站公开表示对第二条“社会主义制度底线”的讥讽<ref>[http://club.kdnet.net/dispbbs.asp?id=9438307&boardid=1 社会主义底线和官员贪赃枉法]</ref><ref>[http://yangyi.bokerb.com/493614.html “社会主义制度底线”，请问什么是“社会主义”？]</ref>。

但是不少网站受到当局压力，被迫删除大量留言和跟帖，出现了几万人留言，只留下区区一百条的怪相<ref>[http://comment.news.163.com/news3_bbs/96BA26A300014JB5.html 话题：互联网协会关于传播正能量坚守"七条底线"倡议书]</ref>。

[[潘石屹|潘石屹]]声称响应国家号召，但在转发“七条底线”时“[[敏感词|敏感词]]太多，发了一天也发不出去”<ref>{{cite news|title=媒体札记：七条底线|url=http://www.ftchinese.com/story/001051923|accessdate=2013-09-09|date=2013-08-12}}</ref> 也有网友表示[[央视|央视]]、[[人民日报|人民日报]]、[[环球时报|环球时报]]常违反信息真实性底线，望其率先自律。<ref>{{cite web|title=两头不讨好的“七条底线|url=http://www.dw.de/%E4%B8%A4%E5%A4%B4%E4%B8%8D%E8%AE%A8%E5%A5%BD%E7%9A%84%E4%B8%83%E6%9D%A1%E5%BA%95%E7%BA%BF/a-17022839|accessdate=2013-09-09|date=2013-08-15}}</ref>。

== 实施 ==
2013年8月20日，中华人民共和国[[公安机关|公安机关]]开始集中打击网络有组织制造传播[[谣言|谣言]]等违法犯罪专项行动，[[秦火火|秦火火]]等人因网络造谣而被逮捕<ref name="fish " />。
