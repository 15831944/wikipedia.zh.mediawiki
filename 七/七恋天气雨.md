'''七恋天气雨'''（{{Lang|ja|うつりぎ七恋天気あめ}}）是日本[[Caramel_Box|Caramel Box]]公司制作的一款[[成人游戏|成人游戏]]，2007年2月23日[[初回限定版|初回限定版]]发售<ref>{{cite web
|url=http://www.caramel-box.com/products/unaten/index.html|title=うつりぎ七恋天気あめ／CARAMEL-BOX|publisher = CaramelBox|quote="2007年2月23日更新 「うつりぎ七恋天気あめ」本日発売！"|language=ja|accessdate=2011-07-02}}</ref>。[[数字发行|DL]]版於2009年9月18日販售。<ref>{{cite web|url=http://dlsoft.dmm.co.jp/detail/hobc_0175/|title=うつりぎ七恋天気あめ|publisher=[[DMM.com|DMM.com]]|accessdate=2016年1月24日|language=ja}}</ref>

==故事==
八千古岛莲是八阿多学院的高二学生，他是学校的“七大奇迹委员会”的成员之一。一个初夏的夜晚，他遇到了一个起着扫帚和女巫袍子的美丽女孩。她自称是来自外国某个魔法学院的学生，因为一些事情要暂住在莲的家里。

==人物==
; 八千古島 蓮（やちこじま れん）
: 莲是整个故事的主人公，他与经营着一个名为“八千”甜品店的自己的父母相处很好。与他的义妹华和三只能听懂人类语言的狐狸亦是如此。
; 艾伦・吉蝶儿（エリン・グィディル）
: [[声優|声]] - [[青山由香里|青山由香里]]

; 明松 月砂（かがり かずさ）
