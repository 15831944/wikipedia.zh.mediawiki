'''七帶豆娘魚'''（[[学名|学名]]：{{lang|la|''Abudefduf septemfasciatus''}}），又稱'''七帶雀鯛'''，'''七纹豆娘鱼'''，俗名'''厚殼仔'''、'''立身仔'''<ref>{{cite web | url=http://fishdb.sinica.edu.tw/chi/species.php?id=382309 | work=台灣魚類資料庫 | title=Abudefduf septemfasciatus | accessdate=2009-04-24}}</ref>，為[[輻鰭魚綱|輻鰭魚綱]][[鱸形目|鱸形目]][[隆頭魚亞目|隆頭魚亞目]][[雀鯛科|雀鯛科]]的一[[種|種]][[鱼类|鱼类]]。

== 分布 ==
本魚分布於[[红海|红海]]、[[印度洋|印度洋]]至[[太平洋|太平洋]]中部<ref name='id7874'/>
，包括[[東非|東非]]、[[馬達加斯加|馬達加斯加]]、[[馬爾地夫|馬爾地夫]]、[[印度|印度]]、[[斯里蘭卡|斯里蘭卡]]、[[安達曼海|安達曼海]]、中国[[南海诸岛|南海诸岛]]、[[台湾|台湾]]、[[日本|日本]]、[[澳洲|澳洲]]、[[菲律賓|菲律賓]]、[[印尼|印尼]]、[[馬來西亞|馬來西亞]]、[[新幾內亞|新幾內亞]]、[[密克羅尼西亞|密克羅尼西亞]]、[[所羅門群島|所羅門群島]]、[[帛琉|帛琉]]、[[馬紹爾群島|馬紹爾群島]]、[[斐濟|斐濟]]、[[萬納杜|萬納杜]]、[[法屬波里尼西亞|法屬波里尼西亞]]等海域。

== 深度 ==
水深0~30公尺。

== 特徵 ==
本魚體側橫帶於生活時明顯，死亡時易於消失。背鰭前鱗片可達眼眶末端。體側有七條深色橫帶，橫帶間隔小於橫帶寬度。背鰭有硬棘13~14枚、軟條12~13枚；臀鰭硬棘2枚；軟條13~14枚。體長可達25公分。
[[File:Sergeant_major_school.jpg|thumb]]

== 生態 ==
本魚棲息於沿岸較淺的岩礁區、潟湖區、或礁湖，屬雜食性，以動物性[[浮游生物|浮游生物]]和[[藻類|藻類]]為食。

== 經濟利用 ==
為食用性魚類，[[油炸|油炸]]使[[骨酥|骨酥]]則美味。亦可飼養於[[水族箱|水族箱]]中。
