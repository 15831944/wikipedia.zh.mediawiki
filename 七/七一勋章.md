'''七一勋章'''是[[中国共产党|中国共产党]]授予党员的一种勋章，是现行[[中华人民共和国勋章奖章纪念章列表|中国共产党功勋荣誉制度]]中最高的一级，由[[中国共产党中央委员会|中国共产党中央委员会]]决定，[[中国共产党中央委员会总书记|中国共产党中央委员会总书记]]授予，于2017年7月22日起启用。<ref name="tixi"/>

七一勋章至今尚未颁发，而且仍未制定勋章图案。

=== 资格 ===
{{Wikisource|中华人民共和国国家勋章和国家荣誉称号法}}

2017年7月，由[[党和国家功勋荣誉表彰工作委员会|党和国家功勋荣誉表彰工作委员会]]制定的《中国共产党党内功勋荣誉表彰条例》、《“七一勋章”授予办法》，已经[[中共中央|中共中央]]批准实施。根据《中国共产党党内功勋荣誉表彰条例》及《“七一勋章”授予办法》，“七一勋章”授予在中国特色社会主义伟大事业和党的建设新的伟大工程中作出杰出贡献的党员。这是由中共中央决定、[[中共中央总书记|中共中央总书记]]签发证书并颁授的党内最高荣誉<ref name=tixi>{{Cite web |url =http://news.xinhuanet.com/mil/2017-07/27/c_129665561.htm|title =我国确立中国特色功勋荣誉表彰制度体系|author =  |date = 2017-07-27 |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref><ref name=zhishi>{{Cite web |url = http://news.xinhuanet.com/politics/2016-05/18/c_1118890603.htm |title =习近平对党和国家功勋荣誉表彰工作作出重要指示  |author =  |date = 2016-05-18 |publisher =新华网  |language =  |accessdate =  |quote =  }}</ref>。
