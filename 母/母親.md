'''母親'''簡稱'''母'''，或稱'''媽媽'''，是一種[[親屬關係|親屬關係]]的[[稱謂|稱謂]]，是[[子女|子女]]對[[雙親|雙親]]中的[[女性|女性]]的稱呼。母親和子女是重要的直系親屬關係之一，通常具有[[親密關係|親密關係]]。

一般指生育小孩的女人（即生母）；但在有些狀況下，僅[[養母|養育小孩]]，或[[繼母|再婚配偶]]、[[義母|結誼長輩]]，或是僅提供[[卵子|卵子]]讓[[精子|精子]]與之結合（例如[[捐卵|捐卵]]等{{link-en|辅助生殖技术|Assisted reproductive technology}}），但並未親自生育小孩的女人也可如此稱呼。

== 定義 ==
在[[生物學|生物學]]上，子女體細胞中成對的[[染色體|染色體]]，有一半是由母親的[[卵子|卵子]]的提供，因此可藉由[[脱氧核糖核酸|DNA]]分析來辨別親屬關係，且[[父親|父親]][[精子|精子]]與卵子結合時，只有提供細胞核的遺傳物質，因此子女細胞中[[粒線體|粒線體]]的DNA皆來自母親，可由此來判別母系祖譜。

在[[社會學|社會學]]上，母親可能代表了養育與教養子女成長的女性，在[[法律|法律]]上，女性也可以經由合法的管道，領養子女，或與有子女的男性結婚，進而成為該子女的法定母親。經領養而成為母親的稱為養母，與有子女男性結婚而成為母親的則稱為[[繼母|繼母]]、[[後母|後母]]或[[晚娘|晚娘]]。

== 母親的稱呼 ==
在日常口語中，小孩對母親有許多不同的稱呼，包括'''媽'''、'''媽媽'''，'''妈咪''' 有些地區子女（特別是古代）對母親的稱呼是'''娘'''、'''阿娘'''或'''娘親'''，又有'''阿母'''、'''老媽'''、'''老媽子'''（[[粵語|粵語]]，北方話「老媽子」是指中年或老年[[女僕|女僕]]）、'''老母'''等稱呼，滿語稱'''額娘'''（{{lang-mnc|ᡝᠨᡳᠶᡝ|z=eniye}}），時下更有親暱地稱為'''母后'''<ref>[http://tw.weibo.com/1764123974/4070485836789234 陳奕ChenI 祝福我的母后～ 生日快樂！]</ref><ref>[https://huilamli.wordpress.com/2009/09/11/%E7%A5%9D%E6%AF%8D%E5%90%8E%E7%94%9F%E6%97%A5%E5%BF%AB%E6%A8%82/ 祝母后生日快樂]</ref><ref>[http://blog.xuite.net/keturah/twblog/120368558-%E6%AF%8D%E5%90%8E%E5%A4%A7%E4%BA%BA%E7%9A%84%E7%94%9F%E6%97%A5 母后大人的生日]</ref>。

對別人稱自己的母親有'''家母'''、'''家慈'''；稱已死去的母親為'''先慈'''、'''先妣'''<ref>《禮記·曲禮下》:「生曰父、曰母、曰妻，死曰考、曰妣、曰嬪。」</ref>；對別人母親的尊稱是'''令堂'''或'''令壽堂'''（粵語口語中另稱'''[[伯母|伯母]]'''）。要特別注意，有時候父親或母親的丈夫（繼父）也會用兒女對母親的稱呼——“媽媽”或“孩子他（她）媽”——代表他的妻子。

子女在不同的年龄阶段对其母亲的称呼也有所不同。幼儿及少儿时期，子女通常用一些较亲切的称呼，如'''妈妈'''等。青年之后，称呼通常为'''妈'''等。

在古代中國的漢族皇族中，[[皇后|皇后]]所出的[[皇子|皇子]]和[[公主|公主]]稱母親為'''母后'''，[[嬪御|嬪御]]所出的子女須認皇后為嫡母，稱皇后為母后，稱生母為'''母妃'''或与其他庶母一样称为'''（阿）姨'''（有的庶出皇子成为皇帝后，尊在世生母为[[皇太后|皇太后]]，即给予生母父亲正妻的名分），皇子、親王的子女亦稱母親為母妃。[[清|清]]代的皇子和公主則稱皇后作'''皇額娘'''、生母稱'''額娘'''。[[英語|英語]]「母后」（Queen Mother）一詞是指現任[[國王|國王]]或女王的母親，即'''王太后'''，但王太后包括並非現任君主母親的前任王后。

在[[中國|中國]]的[[一夫多妻|一夫多妻]]家庭中，父親的[[正妻|正妻]]相對於[[庶出|庶出]]子女而言是[[嫡母|嫡母]]，庶出子女稱嫡母為「大媽」、「大娘」。父親的[[妾|妾]]相對於正室或其他妾所生的子女而言是[[庶母|庶母]]，子女會把庶母稱為「小媽」、「○媽」（○為在妻妾中的排行）、「阿姨」、「姨娘」等，「阿姨」、「姨娘」的稱呼是源於一些地區對母親姊妹的稱呼。

在臺灣[[鄒族|鄒族]]稱母親為'''伊娜'''（Ina）<ref name="番族慣習調查報告書">{{cite book |author=臺灣總督府臨時臺灣舊慣調查會|title=《番族慣習調查報告書》第四卷‧鄒族 |date=2001|publisher=中央研究院民族學研究所|location=臺灣 |language=zh-tw|isbn=9576717981}}</ref>。

== 母親角色 ==
[[File:On_Mangyong_Hill_05.JPG|220px]]一個母親]]

在任何一個社會文化中，都有對於母親應扮演角色職責的期望和規範，此即母親角色（Motherhood），例如在傳統中國，母親被期望扮演為一個盡心照顧公婆、操持家務、輔助丈夫、教育子女的重要角色。

在[[美國|美國]]1960年代的第二波婦女運動中，[[女性主義|女性主義]]者開始對於西方傳統價值觀中對於母親角色的假設有所反省，認為許多對於母親角色的期望（照顧小孩、負責家務），其實是父權制度的產物。

== 著名的母親 ==
*中國古代[[孟母三遷|孟母三遷]]的故事，是描述儒家學者[[孟子|孟子]]幼時，母親為了讓他習得良好的品行，將住家搬到學堂旁，並激勵他努力向學，終於成為一位著名的學者。
*中國古代[[岳母刺字|岳母刺字]]的故事：[[岳飛|岳飛]]的忠勇事蹟廣布民間，其中-{岳}-母刺字的故事也是家喻戶曉。

== 衍生用法 ==
在許多語言中都會將自己所屬的事物以「母」稱之，例如將自己的祖國稱為母國，畢業的學校稱為母校，出生成長所學的第一種語言為[[母語|母語]]。
