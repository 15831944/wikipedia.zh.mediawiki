{{engname|《母與子》|'' She’s not my mother ''}}，又名《巨星單元劇場之母與子》或《巨星TVB單元短劇黎明篇－母與子》，是[[電視廣播有限公司|電視廣播有限公司]]在1986年播出的單元電視劇，由[[梁材遠|梁材遠]]編劇、[[王天林|王天林]]執導，[[黎明|黎明]]、[[柳影虹|柳影虹]]、[[許紹雄|許紹雄]]、[[羅蘭_(香港)|羅蘭]]、[[林帆|林帆]]、鄧汝超、[[駱應鈞|駱應鈞]]、[[李慧芬|李慧芬]]等演出<ref>{{Cite web |url= https://movie.douban.com/subject/20395337/|title =巨星單元劇場－母與子| publisher =豆瓣電影|language = 中文}}</ref><ref>{{Cite web |url= http://www.ihkmusic.com/detail.asp?Product_id=vcd020075&txtParam=0&txtCat=VCD|title =巨星單元劇場－黎明篇 Epiaodic Drama| publisher =現代音像﹝國際﹞有限公司|language = 中文}}</ref>。

==劇情簡介==
此劇以親情故事為題材，劇情講述青茂從小在離島居住，父親伯尼患有肝病，需要長期休養，無法出外工作，青茂的母親梅英因感到寂寞而紅杏出場。伯尼病情惡化，臨終前叮囑青茂不要與梅英為伴，梅英沒有經濟能力為伯尼辦理殮葬事宜，使到青茂更為痛恨母親，離家出走投靠世叔。多年後，青茂大學畢業，梅英希望與青茂見面，青茂起初拒絕，後來在女友玫中的斡旋下答應與梅英見面，見到當天卻因為梅英的現任丈夫搞破壞，導致青茂憤怒離開。青茂到父親的墳前拜祭，想了很久，覺得人死後，什麼事情都結束了，不應該痛恨自己的母親。青茂決定到外國升學，起程當天，梅英與玫中前來送行，青茂非常感動，終於與母親冰釋前嫌<ref> {{Cite book |title =大眾電影| location =中國大陸| publisher =大眾電影雜誌社| date =1992年|url = https://books.google.com.hk/books?hl=zh-TW&id=lmOwAAAAIAAJ&dq=%22%E6%AF%8D%E8%88%87%E5%AD%90%22+%22%E9%BB%8E%E6%98%8E%22&focus=searchwithinvolume&q=%E6%AF%8D%E8%88%87%E5%AD%90| language =中文}} </ref><ref>{{Cite web |url =https://eastweek.my-magazine.me/main/22983|title =不覊舞台|date = 2017年6月5日 |publisher =東周網|language = 中文}}</ref>。

==主要角色==
* [[黎明|黎明]]－青茂，是一名大學畢業的年青人。
* [[柳影虹|柳影虹]]－梅英，青茂的母親。
* [[許紹雄|許紹雄]]－伯尼，青茂的父親。
* [[林帆|林帆]]－玫中，青茂的女朋友。
