'''庆祝中国人民解放军建军90周年阅兵'''是[[中国人民解放军|中国人民解放军]]在[[中国人民解放军建军纪念日|建军]]90周年之际，2017年7月30日于[[中国人民解放军陆军朱日和合同战术训练基地|朱日和合同战术训练基地]]举行的[[阅兵|阅兵]]仪式。这是[[中华人民共和国|中华人民共和国]]成立以来，中国人民解放军全軍層級首次以庆祝建军节为主题阅兵（2004年[[中國人民解放軍駐香港部隊|駐港部隊]]有過一次小規模的建軍節閱兵），也是中国人民解放军在[[深化国防和军队改革|深化国防和军队改革]]后的全新亮相<ref>{{Cite web |url = http://news.163.com/17/0730/09/CQJ6JFI000018AOR.html |title = 解放军首次以庆祝建军节为主题阅兵 系军改重塑后的全新亮相 |author =  |date = 2017-07-30  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

==过程==
阅兵开始于2017年7月30日9时。[[中共中央总书记|中共中央总书记]]、[[中华人民共和国主席|国家主席]]、[[中央军事委员会主席|中央军委主席]][[习近平|习近平]]检阅部队并发表讲话。10时12分阅兵结束<ref>{{Cite web|url=http://military.people.com.cn/n1/2017/0729/c1011-29436790.html|title=庆祝中国人民解放军建军90周年阅兵将于7月30日举行|date=2017-07-29|accessdate=|publisher=人民网}}</ref><ref>{{Cite web |url = http://news.ifeng.com/a/20170730/51530473_0.shtml |title =  习近平：军队有能力打败一切来犯之敌|author =  |date =2017-07-30  |publisher =  凤凰网|language =  |accessdate =  |quote =  }}</ref>。本次阅兵是野战化实战化阅兵<ref name=zhibo/>。

7月30日9时，习近平来到标有“1927—2017”字样和[[中国人民解放军军徽|中国人民解放军军徽]]的检阅台，[[中共中央政治局委员|中共中央政治局委员]]、[[中央军委副主席|中央军委副主席]][[范长龙|范长龙]]宣布阅兵开始。一辆野战检阅车缓缓向受阅部队驶去，习近平站在检阅车中央。阅兵总指挥、[[中国人民解放军中部战区|中国人民解放军中部战区]]司令员[[韩卫国|韩卫国]]上将向习近平报告：“主席同志，受阅部队准备完毕，请您检阅！” 习近平命令“开始！”《检阅进行曲》随即响起。习近平乘检阅车向东驶去，依次检阅了护旗方队、地面作战群和人员方阵<ref name=zhibo/>。其中，8100名官兵的受阅人员方阵，由代表不同方向9个900人的方队组成，既表示向“八一”建军节献礼，又代表人民军队走过90年历程，受阅人员方阵指挥员为[[李广泉|李广泉]]少将<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1758844 |title =建军90周年阅兵将军领队李广泉已担任第82集团军副军长  |author =  |date =  2017-08-10|publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。随后习近平回到检阅台。开始阅兵分列式<ref name=zhibo>{{Cite web |url = http://news.ifeng.com/mainland/special/jfjjj90znyb/ |title =凤凰直播：建军90周年朱日和沙场大阅兵  |author =  |date =2017-07-30  |publisher =  凤凰网 |language =  |accessdate =  |quote =  }}</ref><ref name=shachang/>。

中央军委副主席范长龙宣布分列式开始。阅兵分列式按照作战编组，以空地一体的形式接受检阅。参加分列式的部队依次是<ref name=zhibo/><ref name=shachang/>：

*护旗方队：由陆军、海军、空军、火箭军官兵组成，护卫着[[中国共产党党旗|中国共产党党旗]]、[[中华人民共和国国旗|中华人民共和国国旗]]、[[中国人民解放军军旗|中国人民解放军军旗]]<ref name=zhibo/>。
*纪念标识梯队：17架直升机组成“八一”标识，24架直升机绘出“90”字样，象征人民军队走过90年历程<ref name=zhibo/>。 ​
*陆上作战群
**空中突击梯队：10架武装直升机开辟空中通道，实施侦察预警、火力压制。8架武装直升机掩护18架[[直-8|直-8]]运输直升机突入作战区域。18架运输直升机抵达各自战位，呈战斗队形快速降落，8架武装直升机在一侧实施支援掩护，突击队员跃出，向预定地域集结。
**坦克方队：领队为[[姚旺|姚旺]]少将。22辆坦克呈箭形前进。中国自主研发的主战坦克[[ZTZ-99主战坦克|99A坦克]]体现了陆战装备新水平。
**步战车方队：领队为[[赵宇_(陆军)|赵宇]]少将。由[[ZBD-04步兵战车|04A履带式步战车]]、[[ZBL-08步兵战车|08式轮式步战车]]混合编成。这两款新型步战车是机械化、信息化复合式发展的陆上作战代表性装备。
**自行火炮方队：领队为[[柳林_(少将)|柳林]]少将。目前的中国人民解放军炮兵精度更高、火力更猛、机动性更强。
**反坦克导弹方队：领队为[[黄桃益|黄桃益]]少将。受阅的[[红箭-10反坦克导弹|红箭-10反坦克导弹]]是中国人民解放军目前装备的最新式反装甲武器。
*信息作战群
**信息支援方队：领队为[[李宗利|李宗利]]少将。这支新型作战力量在[[深化国防和军队改革|深化国防和军队改革]]中诞生，正加快融入全军联合作战体系，并在关键领域实现跨越发展。
**电子侦察方队：领队为[[易建设|易建设]]少将。受阅电子侦察车组能为掌握复杂电磁环境下战场主动权提供有效支撑。
**电子对抗方队：领队为[[吴亚非|吴亚非]]少将。由雷达干扰车、通信侦察干扰车组成。新型电子装备列装部队，使中国人民解放军电子对抗能力获长足发展。
**无人机方队：领队为[[毛明荣|毛明荣]]少将。由通信干扰无人机、雷达干扰无人机、反辐射无人机组成，能够有效压制敌方预警探测和指挥通信体系。
*特种作战群
**特种兵方队：领队为[[张建刚|张建刚]]少将。特种兵乘坐全地形车接受检阅。
**特战装备方队：领队为[[吴爱民|吴爱民]]少将。由轮式轻型侦察车、防护型突击车组成。
*防空反导作战群
**预警雷达方队：领队为[[孟宪章|孟宪章]]少将。受阅雷达装备具有高机动、高性能、高可靠的特点。
**地空导弹方队：领队为[[刘明豹|刘明豹]]少将。[[红旗-9中远程防空导弹|红旗-9B]]地空导弹和[[红旗-22|红旗-22]]地空导弹，是中国自行研制的中远程新一代地空导弹，极大提高了信息化条件下的防空反导能力。
**弹炮系统方队：领队为[[董玉江|董玉江]]少将。由[[PGZ-09自行高射炮|09式自行高炮]]、[[红旗-6|红旗-6]]弹炮导弹发射车和跟踪高炮车组成。
*海上作战群
**海军陆战队方队：领队为[[孔军|孔军]]海军少将。由[[海军陆战队|海军陆战队]]男兵和女兵组成。
**海军导弹第1方队：领队为[[魏文徽|魏文徽]]海军少将。由[[红旗-9中远程防空导弹|海红旗-9B]]舰空导弹、[[鷹擊12反艦飛彈|鹰击-12A]]舰舰导弹组成。海红旗-9B主要担负舰艇区域防空任务，鹰击-12A主要担负大中型水面舰艇打击任务。
**海军导弹第2方队：领队为[[刘子柱|刘子柱]]海军少将。由[[鹰击83反舰导弹|鹰击-83K]]空舰导弹、[[鹰击62反舰导弹|鹰击-62A]]岸舰导弹组成。
*空中作战群：领队为[[夏维宴|夏维宴]]空军少将。
**预警指挥机梯队：[[空警-2000|空警-2000]]预警机、[[空警-500|空警-500]]预警机、运干-8指通机、运干-8干扰机，在8架歼-10B战机护卫下接受检阅。
**轰炸机梯队：9架[[轰-6|轰-6K]]编成3个三机楔队。
**运输机梯队：由1架[[运-20|运-20]]和2架[[运-9|运-9]]运输机组成。运-20是中国自主研发的新一代军用大型运输机，运-9是中国国产中型战术运输机。
**加受油机梯队：由轰油-6加油机、[[歼-10|歼-10]]C战机组成。
**舰载机梯队：由7架[[歼-15|歼-15]]组成。
**歼击机梯队：3机楔队为[[歼-20|歼-20]]战机，5机楔队为[[歼-16|歼-16]]战机，7机楔队为[[歼-11|歼-11]]B战机组成。歼-20是中国自主研制的第四代超音速隐身战斗机，此次是继亮相[[中国国际航空航天博览会|珠海航展]]后，首次以战斗姿态展示。6架歼-11B发射了红外干扰弹。
**空降兵方队：领队为[[孙向东|孙向东]]空军少将。空降兵在[[深化国防和军队改革|深化国防和军队改革]]中得到了进一步加强。
*综合保障群
**信息通信保障方队：领队为[[杨毅_(少将)|杨毅]]少将。受阅装备具备全时通、全域通、全频通保障能力。
**工程防化保障方队：领队为[[李志忠_(少将)|李志忠]]少将。由综合扫雷车、装甲工程车、轮式冲击桥和喷洒车组成。
**后勤保障方队：领队为[[殷志红|殷志红]]少将。新组建的[[中央军委联勤保障部队|联勤保障部队]]正在按照打仗要求加紧构建新一代后勤保障体系。
**装备保障方队：领队为[[付文化|付文化]]少将。由修理拆装车、装甲抢修车、装甲抢救车、坦克抢救牵引车组成。
*反恐维稳群
**武警特警方队：领队为[[徐平_(少将)|徐平]]少将。由[[猎鹰突击队|猎鹰突击队]]编成，168名武警特战队员乘坐反恐突击车接受检阅。
*战略打击群
**火箭军部队方队：领队为[[吴祚宝|吴祚宝]]少将。由火箭军部队360名官兵组成。
**核常兼备导弹方队：领队为[[赵秋领|赵秋领]]少将。由[[东风-26|东风-26]]导弹组成。东风-26导弹可实施快速核反击和常规中远程精确打击。
**常规导弹第1方队：领队为[[兰吉银|兰吉银]]少将。由[[东风-21|东风-21]]丁导弹组成。东风-21丁导弹是打击舰船目标的陆基弹道导弹。
**常规导弹第2方队：领队为[[张明国|张明国]]少将。由[[东风-16|东风-16]]改常规导弹组成。东风-16改常规导弹能够对敌要害目标实施精确打击。
**核导弹方队：领队为[[潘吉慧|潘吉慧]]少将。由两型固体洲际战略核导弹组成<ref name=shachang>{{Cite web |url =  http://news.cyol.com/content/2017-07/30/content_16343945.htm|title =  习近平沙场阅兵并发表重要讲话：有信心有能力打败一切来犯之敌！|author =  |date =  2017-07-30|publisher =中青在线  |language =  |accessdate =  |quote =  }}</ref>。

中部战区负责本次阅兵的组织筹划工作，担任阅兵指挥任务。本次阅兵成立了由军兵种组成的阅兵联合指挥部，以及由军兵种指挥员参加的阅兵联合指挥部党委。阅兵联合指挥部总指挥是中部战区司令员韩卫国上将，副总指挥是中部战区参谋长助理[[樊建荣|樊建荣]]大校等人<ref>{{Cite web |url = http://china.huanqiu.com/article/2017-07/11055429.html |title = 惊人细节！阅兵这天，五大战区的这些调动不容忽视 |author =  |date =2017-07-31  |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1761784  |title =樊建荣大校任中部战区参谋长助理，曾任原北空导弹某师师长  |author =  |date =2017-08-13  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

==争议==
阅兵的行列中，[[中国共产党党旗|中国共产党党旗]]领先于[[中华人民共和国国旗|中华人民共和国国旗]]<ref name=shachang/><ref>{{Cite news|url=http://thediplomat.com/2017/08/chinas-military-parade-reaffirms-communist-partys-absolute-control-over-army/|title=China’s Military Parade Reaffirms Communist Party’s Absolute Control Over Army|last=Diplomat|first=Charlotte Gao , The|newspaper=''The Diplomat''|accessdate=2017-08-14|language=en-US}}</ref>。[[美国之音|美国之音]]指出，此举违反《[[s:中华人民共和国国旗法|中华人民共和国国旗法]]》第十五条第二款规定：“列队举持国旗和其他旗帜行进时，国旗应当在其他旗帜之前。”<ref>{{Cite news|url=https://www.voachinese.com/a/china-parade-20170730/3965254.html|title=中国阅兵中党旗先于国旗，网民批违反国旗法|last=丁力|newspaper=美国之音|accessdate=2017-08-01|language=zh}}</ref>而观察网的评论家黎阳则认为此举并无不妥之处。他认为因《国旗法》的第十条规定-{“}-军事机关、军队营区、军用舰船，按照中央军事委员会的有关规定升挂国旗-{”}-，且该阅兵-{“}-属于军事行为-{”}-，应当-{“}-按照[[中央军事委员会|中央军委]]的规定办事-{”}-<ref name="察网黎阳">{{cite web|author1=黎阳|title=黎阳：朱日和大阅兵之际，法律小丑玩弄《国旗法》|url=http://www.cwzg.cn/politics/201708/37537.html|publisher=察网|accessdate=2017-12-19}}</ref>
