'''庆安火车站枪击事件'''是指2015年5月2日发生在[[中国大陆|中国大陆]][[黑龙江省|黑龙江省]][[庆安站|庆安火车站]]候车室的一起枪击案件，中年男子徐纯合和车站执勤民警发生冲突，并抢夺警械，然后警察开枪将其击毙。<ref>{{Cite web |url = http://news.hexun.com/2015-05-06/175562635.html |title = 黑龙江庆安枪击案 |author =  |date = 2015-05-06 |publisher = [[和讯网|和讯网]] |language = zh-hans |accessdate = 2015-05-14|quote = }}</ref>由于徐纯合曾有[[上访|上访]]经历，因此引发媒体质疑案件是否因违法[[维稳|截访]]而引发。<ref>{{Cite web|url =http://www.dqdaily.com/shishi/2015-05/09/content_2661924.htm|title =媒体三问黑龙江庆安枪案：是否因违法截访引发|date =2015-05-09|publisher =大庆网|language =zh-hans|accessdate =2015-05-14|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20150518095223/http://www.dqdaily.com/shishi/2015-05/09/content_2661924.htm|archivedate =2015-05-18}}</ref>

== 当事人 ==
'''徐纯合'''，男，1970年出生，[[中国|中国]][[黑龙江省|黑龙江省]][[庆安县|庆安县]]人。

2015年5月2日，疑因其在[[上访|上访]]途中，因庆安县火车站不允许其上火车，因而和车站执勤警察发生争执，被警察开枪击毙<ref>{{cite web|title=《纽约时报》：中国警察枪杀访民，官方调查难以服众|url=http://cn.nytimes.com/china/20150517/c17police/print/dual/}}</ref>。

==过程==
2015年5月2日，徐纯合与其母亲及3个孩子准备在庆安站搭乘[[庆安县|庆安]]至[[金州|金州]]的K930次列车，因为徐纯合曾经的上访经历，所以检票员极力阻止他们上车。9点58分，徐纯合购买了前往大连的火车票，并随后前往周边的餐馆用餐。12时许，徐纯合在候车室安检口处拦截旅客进站乘车，哈尔滨铁路公安局的执勤[[民警|民警]]发现后前往制止，不久与徐纯合发生冲突，徐被警察开枪击毙，之后120人员赶到现场后，确认其死亡。此过程没有其他旅客受伤。事后警方称，开枪是因为徐抢走民警携带的警具，并试图抢夺枪支，为确保现场旅客生命安全，民警才开枪将其击倒。但央视只字未提起冲突原因。<ref>{{Cite web |url = http://www.21ccom.net/plus/wapview.php?aid=124497 |title = 庆安枪声：别让真相淹没在血泊中 |author = [[谭敏涛|谭敏涛]] |date = 2015-05-08 |publisher = [[共识网|共识网]] |language = zh-hans |accessdate =2015-05-14 |quote = |archiveurl=https://web.archive.org/web/20160427200153/http://www.21ccom.net/plus/wapview.php?aid=124497|archivedate=2016-04-27|deadurl=yes}}</ref>

5月14日，在网友发布的警察打人视频被审查后，[[央视新闻频道|央视新闻频道]]发布现场监控视频，显示徐纯合封堵火车站通道阻止乘客进站，并有抢夺防暴棍并拳击民警的行为，徐还曾将自己6岁的女儿举起抛摔，后来民警取出佩枪对其口头警告无效后开枪。{{what|开枪后，徐母还拿起警棍打了徐几下。}}<ref>{{youtube|s524hEU4P1E|央视公布庆安枪击案监控 展现徐纯合被击毙全过程}}</ref><ref>{{Cite web |url =http://video.sina.com.cn/news/spj/hljqaqja/?opsubject_id=top1#138051258 |title = 黑龙江庆安枪击案专题 |language = zh-hans |publisher = 新浪 |accessdate = 2015-05-14}}'''（视频）'''</ref><ref>{{cite web|url=http://cn.nytimes.com/china/20150517/c17police/ |language=zh-hans |title=中国警察枪杀访民，官方调查难以服众 |publisher=[[纽约时报|纽约时报]]中文网 | date=2015-05-17 |accessdate=2015-05-26 }}</ref>

==后续==
死者家属表示，警方在事后给了他们一笔抚慰金，提出让家属签字之后不再追究责任，他们虽然感觉无奈，但也只能接受。在徐纯合被击毙的第二天，庆安县副县长董国生立即出面肯定杀人行为合法，并称徐纯合为“歹徒”，但不久他就被网友曝光“中专还没毕业，大学学历是假文凭”，其妻子也被曝光在政府部门吃空饷，之后庆安县[[纪委|纪委]]介入调查。<ref> {{Cite web|url =http://finance.gucheng.com/201505/3015922.shtml|title =庆安枪击案拿钱封口 现场视频私下流出真相曝光|author =|date =2015-05-07|publisher =股城网|language =zh-hans|accessdate =2015-05-14|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20150514084306/http://finance.gucheng.com/201505/3015922.shtml|archivedate =2015-05-14}}</ref>

央视视频发布后，有评论指出视频经过剪辑，证据力不足。<ref>{{Cite web|url=http://money.163.com/15/0515/12/APLHK0NJ00253B0H.html|title = 陶舜谈庆安枪案：人民不需要剪辑的真相|publisher =网易财经|language = zh-hans |date = 2015-05-15|accessdate = 2015-05-27}}</ref>
