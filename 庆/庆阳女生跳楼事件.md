== 事件经过 ==
6月20日下午15时左右，在西峰区某服装店担任导购的李奕奕爬上百货公司8楼玻璃幕墙外直播，期间不少网民冷嘲热讽，催促其快跳。<ref>{{Cite web|url=http://news.dwnews.com/china/big5/news/2018-06-25/60066489.html|title=被老師猥褻后抑郁 女生直播跳樓遭網友起哄后墜亡|accessdate=2018-06-28|author=多维新闻|date=|work=news.dwnews.com|publisher=}}</ref>而围观群众有不少人起哄大喊，“你到底跳不跳啊”“要跳就跳，果断一点”“楼下好热，快跳啊，你到底跳不跳？”。<ref>{{Cite news|url=http://news.ltn.com.tw/news/world/breakingnews/2467693|title=中國正妹遭老師性騷   悲憤跳樓圍觀群眾竟叫好|author=自由时报电子报|date=|work=|accessdate=2018-06-28}}</ref><ref>{{Cite web|url=http://news.sina.com.cn/s/2018-06-25/doc-ihencxtt9111784.shtml|title=甘肃19岁女孩跳楼:参与救援的消防员曾救过她一次|accessdate=2018-06-28|work=news.sina.com.cn}}</ref>19时15分许，李奕奕挣脱了消防员的手，向消防员道谢后跳楼身亡。<ref>{{Cite web|url=https://3g.163.com/all/article/DLA70LQL0001875P.html|title=甘肃跳楼女生父亲:女儿坠亡前吃救援者买的爆米花_手机网易网|accessdate=2018-06-28|author=手机网易网|date=2018-06-27|work=3g.163.com|publisher=}}</ref><ref>{{Cite news|url=http://www.epochtimes.com/b5/18/6/24/n10508481.htm|title=甘肅19歲女生遭老師猥褻 投訴無門跳樓自殺 - 大紀元|date=2018-06-25|newspaper=大紀元 www.epochtimes.com|accessdate=2018-06-28|language=zh-TW}}</ref>

== 事后处理 ==
25日晚10时许，[[庆阳市公安局|庆阳市公安局]]西峰分局召开“西峰6·20女孩跳楼死亡事件”媒体通气会，警方介绍称，已有两名围观群众因妨碍救援被[[行政拘留|行政拘留]]，4月24日又确定6名，事件正进一步调查中。<ref>{{Cite web|url=http://views.ce.cn/view/ent/201806/27/t20180627_29540997.shtml|title=妨碍救援行动，自作就该自受 _中国经济网——国家经济门户|accessdate=2018-06-28|last=评论频道|first=中国经济网|work=views.ce.cn}}</ref><ref>{{Cite web|url=http://video.sina.com.cn/p/news/s/doc/2018-06-25/140968902736.html?cre=videopc&mod=sl&loc=3&r=0&doct=0&rfunc=100&tj=none|title=视频{{!}}甘肃女孩跳楼围观者起哄 官方：已经拘留好几个了{{!}}时事{{!}}甘肃{{!}}拘留{{!}}庆阳{{!}}女生_新浪视频|accessdate=2018-06-28|work=video.sina.com.cn}}</ref><ref>{{Cite web|url=http://www.xinhuanet.com/legal/2018-06/27/c_1123042397.htm|title=甘肃庆阳女生跳楼事件最新进展：涉案人教师资格被取消-新华网|accessdate=2018-06-28|work=www.xinhuanet.com}}</ref>

== 自杀原因 ==
据报道，死者在就读庆阳市第六中学期间，于2016年9月5日遭到班主任吴永厚[[强奸|强奸]]未遂。事后吴永厚向李奕奕道歉，李不接受，将吴诉至法院。但法院认为情节轻微不构成犯罪，2018年3月1日，作出不予起诉的决定<ref>{{Cite web|url=https://3g.163.com/all/article/DLB5F6JN0001875P.html|title=甘肃女生跳楼事件检察院为何不起诉?专家:有依据_手机网易网|accessdate=2018-06-28|date=2018-06-27|work=3g.163.com}}</ref><ref>{{Cite web|url=http://culture.dwnews.com/renwen/big5/news/2018-06-26/60066867.html|title=甘肅女生跳樓：誰在爭当壓死她的最后一根稻草|accessdate=2018-06-28|work=culture.dwnews.com}}</ref><ref>{{Cite news|url=http://www.epochtimes.com/b5/18/6/24/n10508481.htm|title=甘肅19歲女生遭老師猥褻 投訴無門跳樓自殺 - 大紀元|date=2018-06-25|newspaper=大紀元 www.epochtimes.com|accessdate=2018-06-28|language=zh-TW}}</ref>。之后，李奕奕被诊断患上[[抑郁性障碍|抑郁症]]和[[創傷後壓力症候群|创伤性应激障碍]]，四次自杀未遂。但猥亵和抑郁症之间是否有直接因果关系，抑郁症是否是促使李某奕走上八楼并最终跳下来的直接诱因，警方表示还在继续调查中。<ref name=":0">{{Cite web|url=http://www.xinhuanet.com/legal/2018-06/27/c_1123042397.htm|title=甘肃庆阳女生跳楼事件最新进展：涉案人教师资格被取消-新华网|accessdate=2018-06-28|work=www.xinhuanet.com}}</ref>

2017年7月23日[[庆阳市教育局|庆阳市教育局]]党委作出决定，降低吴永厚岗位等级，并将其调离教学岗位。跳楼案发生后，吴被注销教师资格证<ref name=":0" />，其家门口被不明人士喷红漆，个人信息和家庭住址也被[[人肉搜索|人肉到网上]]<ref>{{Cite news|url=http://hk.on.cc/cn/bkn/cnt/news/20180627/bkncn-20180627224542100-0627_05011_001_cn.html|title=甘肃猥亵女生老师家门遭喷漆　同校老师遭谩骂|newspaper=on.cc东网|accessdate=2018-06-30|language=zh-hk}}</ref><ref>{{Cite web|url=http://www.sohu.com/a/238019225_665455|title=老师“亲吻”学生不够开除？那要怎么“整顿师风”|accessdate=2018-06-30|date=2018-06-27|work=www.sohu.com}}</ref>。
