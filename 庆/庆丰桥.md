'''庆丰桥'''是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]]一座跨[[甬江|甬江]]桥梁，载有[[通途路|通途路]]，连接[[鄞州区|鄞州区]]和[[江北区_(宁波市)|江北区]]。始建于2004年3月，2008年11月建成通车，是甬江上继[[甬江大桥|甬江大桥]]后第二座连接鄞州区和江北区的桥梁。<ref name="cons">{{cite web|url=http://china.qianlong.com/4352/2005/10/18/155@2848781.htm|title=庆丰桥露出地面 北仑、江东、江北、海曙将一线通|date=2005-10-18|accessdate=2010-10-11|deadurl=yes|archiveurl=https://web.archive.org/web/20070529085612/http://china.qianlong.com/4352/2005/10/18/155@2848781.htm|archivedate=2007年5月29日|df=}}</ref>

== 桥梁形制 ==
庆丰桥为三跨双塔双索面自锚式悬索桥，是宁波建设的首座悬索桥。桥梁全长3654米，其中主跨280米，桥面载有双向8车道<ref>{{cite web|url=http://news.cnnb.com.cn/system/2008/11/01/005851308.shtml|title=庆丰桥主线建成力争在11月初通车|date=2008-11-01|accessdate=2010-10-11}}</ref>。在江北引桥一侧，设有4条匝道，分别连接人民路、东草马路、大庆南路和湖东路。<ref name="auxlane">{{cite web|url=http://news.cnnb.com.cn/system/2010/07/13/006597684.shtml|title=庆丰桥匝道本周五开通 行车时速不能超40公里|date=2010-07-13|accessdate=2010-10-11}}</ref>

== 事故与争议 ==
庆丰桥原本属于规划中的通途路快速路，不允许非机动车通行。但是，由于甬江大桥拉索腐蚀需要大修，建成后的庆丰桥不得不开通非机动车道以方便非机动车过江。这一临时性措施由于桥梁坡度较陡等原因造成非机动车难以控制车速，带来了了大量交通事故。这也使得2008年底竣工的庆丰桥匝道迟迟未开通，引起了不少机动车驾驶员的质疑。<ref>{{cite web|url=http://news.163.com/09/1231/12/5RS5N72U0001124J.html|title=宁波庆丰桥下的大庆南路为何不开通？|date=2009-12-31|accessdate=2010-10-11}}</ref>2010年7月16日，庆丰桥匝道开通之后，由于匝道与非机动车道并未完全分离，非机动车常在匝道上与匝道-主线出入口与机动车发生冲撞。对此，有专家认为，是大桥设计中存在的问题导致事故频发。<ref>{{cite web|url=http://zjnews.zjol.com.cn/05zjnews/system/2010/10/11/016991829.shtml|title=宁波庆丰桥交通事故频发 专家:大桥设计欠周到|date=2010-10-11|accessdate=2010-10-11}}</ref> 另据报道，在[[外滩大桥|外滩大桥]]通车之后，庆丰桥将禁止非机动车上桥。<ref>{{cite web|url=http://www.nbzx.gov.cn/art/2010/2/5/art_9542_444408.html|title=行在路上，能否让人舒畅——政情交流会侧记|date=2010-02-05|accessdate=2010-10-11|deadurl=yes|archiveurl=https://web.archive.org/web/20160304094552/http://www.nbzx.gov.cn/art/2010/2/5/art_9542_444408.html|archivedate=2016-03-04}}</ref>

== 轶事 ==
庆丰桥桥塔使用[[繁体中文|繁体中文]]“-{zh-hans:慶豐橋; zh-hant:慶豐橋}-”进行标注。部分市民认为，繁体字使得整座桥梁显得大气，而简体过于单薄。也有人指出，这种做法不符合规范用字的要求。<ref>{{cite web|url=http://www.zj.xinhuanet.com/newscenter/2008-11/04/content_14819960.htm|title=宁波“-{zh-hans:慶豐橋; zh-hant:慶豐橋}-”难倒小学生 繁体字命名惹争议|date=2008-11-04|accessdate=2010-10-11|deadurl=yes|archiveurl=https://web.archive.org/web/20160304140825/http://www.zj.xinhuanet.com/newscenter/2008-11/04/content_14819960.htm|archivedate=2016-03-04}}</ref>
