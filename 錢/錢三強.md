'''钱三强'''（{{bd|1913年|10月16日|1992年|6月28日|catIdx=Q}}），原名'''钱秉穹'''，[[原子核物理学|原子核物理学]]家，中国原子能事业的主要奠基人和组织领导者之一<ref>[http://news.xinhuanet.com/ziliao/2003-01/17/content_694704.htm 中国原子能事业的奠基人——钱三强] {{webarchive|url=https://web.archive.org/web/20070320233450/http://news.xinhuanet.com/ziliao/2003-01/17/content_694704.htm |date=2007-03-20 }}</ref>。

== 生平 ==
钱三强，[[浙江|浙江]][[湖州|湖州]]人，其父[[钱玄同|钱玄同]]是中国近代著名的语言文字学家。钱三强1913年10月16日出生于[[浙江|浙江]][[绍兴|绍兴]]。1936年钱三强毕业于[[清华大学|清华大学]]物理系。在短期担任[[北平研究院|北平研究院]]物理研究所[[严济慈|严济慈]]所长的助理后，1937年留学[[法国|法国]]，在法国[[巴黎大学|巴黎大学]]居里实验室和[[法兰西学院|法兰西学院]]原子核化学实验室从事[[原子核物理学|原子核物理]]研究工作，获博士学位，师从诺贝尔奖获得者[[伊雷娜·约里奥-居里|伊雷娜·约里奥-居里]]及其丈夫[[弗雷德里克·约里奥-居里|弗雷德里克·约里奥-居里]]。自此期间，在研究铀核三裂变中取得了突破性成果。1946年获[[法国科学院|法国科学院]][[亨利·德巴微|亨利·德巴微]](Henri de Parville)物理学奖金。

1948年回国后，他先后担任了[[中国科学院近代物理研究所|中国科学院近代物理研究所]]（后改名原子能研究所）的副所长、所长，计划局局长，[[二机部|二机部]]副部长、[[中国科学院|中国科学院]]副院长。他作为原子能学科的学术带头人和科技计划地制定者，对中国原子能事业、“[[两弹一星|两弹一星]]”计划做出了巨大贡献。

晚年的钱三强，担任了[[浙江大学|浙江大学]]校长、[[中国科协|中国科协]]副主席、[[中国物理学会|中国物理学会]]理事长、[[中国核学会|中国核学会]]名誉理事长等职务。

1992年6月28日逝世。1999年，他被追授了由515克纯金铸成的“[[两弹一星功勋奖章|两弹一星功勋奖章]]”，以表彰其贡献。

== 家庭 ==
與同是核物理學家的妻子[[何泽慧|何泽慧]]育有兩女一男，錢祖玄、錢民協、錢思進，均為科學家。

其父[[钱玄同|钱玄同]]是中国近代语言文字学家。

钱三强和[[钱学森|钱学森]]、[[钱伟长|钱伟长]]并称为中国科技界的“[[三钱|三钱]]”。<ref>{{Cite web |url=http://news.xinhuanet.com/politics/2010-10/21/c_12684639.htm |title=“三钱”的称谓出自周恩来总理之口 |publisher=新华时政 |date=2010年10月21日}}</ref>
