== 故事概要 ==
此剧讲述患有[[帕金森病|帕金森病]]的独居老人钱老实（[[朱厚任|朱厚任]]饰）拥有百万洋房，原本宁静的生活却被侄子钱仁杰（王沺裁饰）和侄女钱美玲（钟琴饰）扰乱。大哥钱仁杰和妻子罗舒萍（[[童冰玉|童冰玉]]饰）面临财务困难，走投无路下，带着女儿钱宁宁（罗美仪饰）住进大伯的洋房。不料，妹妹钱美玲因老公吴国伦（姚玟隆饰）理财不慎，无家可归的他们也带了女儿吴希希（[[雅慧|雅慧]]饰）和儿子吴佳明（陈昱志饰）搬入大伯家。

两兄妹觑觎钱老实的财富，不断对老人家献殷勤。钱老实都看在眼里，所以经常故意闹脾气，搞得大家精疲力竭，家里鸡犬不宁。剧中人物的共通毛病是不善于理财，因此搞得落魄不堪，不得不寄人篱下。

== 軼事 ==
*此劇演員王沺裁與[[童冰玉|童冰玉]]曾在《Missy先生》中飾演情侶，在此劇中他們則飾演夫妻。
*此劇是鐘琴與姚玟隆繼《對對碰》后，再度飾演夫妻。
*此劇是[[雅慧|雅慧]]與包勋评繼《[[燒賣_(電視劇)|燒·賣]]》后，再度合作的電視劇。
*此劇是陈昱志首部參演的劇集。

== 與史實不合之處 ==
*第12集里，钱美玲梦到吴国伦在酒吧搞外遇，吴国伦在梦里骂她是个“死八婆”两次；在这时，中文字幕没有显示，但是，他更正她是个“黄脸婆”时，字幕却显示了。其他方面，第10集钱美玲对付高丽莘后骂她是个“死八婆”，第20集Bobby被捕时骂钱美玲“臭三八”、“死八婆”，中文字幕也没有显示。
