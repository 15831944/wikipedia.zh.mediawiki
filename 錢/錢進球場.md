《'''錢進球場'''》（{{lang-ja|グラゼニ}}）是由[[Cozy城倉|森高夕次]]原作、[[足立金太郎|足立金太郎]]（太平洋聯盟篇連載之前筆名「足立刑事（<small>{{lang|ja|アダチケイジ}}</small>）」<ref name="natalie">{{Cite news|url=https://natalie.mu/comic/news/275709|script-title=ja:PEACH-PIT「二科てすら」新シリーズ&「グラゼニ」パ·リーグ編がモーニングで|newspaper=[[Natalie_(網站)|Comic Natalie]]|publisher=株式會社Natasha|date=2018-03-29|accessdate=2018-03-29|language=ja}}</ref>）負責作畫以[[日本職棒|日本職棒]]題材的[[日本漫畫|日本漫畫]]作品，自2011年起於《[[Morning_(漫畫雜誌)|Morning]]》上連載，2014年完結，同年推出第二部《錢進球場 ～東京巨蛋篇～》，連載至2018年止並推出第三部《錢進球場 ～太平洋聯盟篇～》。作品以重視金錢的職棒中繼左投凡田夏之介為主角，描述職業棒球員的職業生涯和職場生態。作品名稱「{{lang|ja|グラゼニ}}」是一個作者原創的混成詞，指「{{lang|ja|グラウンド}}」（球場）和「{{lang|ja|ゼニ}}」（金錢），是將日本棒球運動員[[鶴岡一人|鶴岡一人]]的名言「{{lang|ja|グラウンドにはゼニが落ちている}}」縮減而成<ref>{{cite web|url=http://www.sankei.com/premium/news/151122/prm1511220005-n3.html|script-title=ja:【スポーツ異聞】 名将·鶴岡一人は言った…「グラウンドには銭が落ちている」 神聖な職場を「賭場」にした残念な選手たち|accessdate=2017-03-25|date=2015年11月27日|publisher=[[產經新聞|產經新聞]]|work=產經News|language=ja}}</ref><ref>{{cite web|url=https://ddnavi.com/news/130896/a/|script-title=ja:プロ野球開幕！ 『グラゼニ』にみる野球界のシビアで過酷な現実|accessdate=2017-03-25|date=2013年3月29日|publisher=[[KADOKAWA|KADOKAWA]]|work={{lang|ja|ダ·ヴィンチ ニュース}}|language=ja}}</ref><ref>出自《週刊棒球》9月30日號，第30頁的「{{lang|ja|名将の言葉に見る美学、哲学}}」專題。</ref><ref>出自《東京體育》2013年2月6日「{{lang|ja|マネー金言珍言}}」。</ref>。

== 登場角色 ==
;{{nihongo|凡田夏之介|凡田 夏之介（ぼんだ なつのすけ）|聲：[[落合福嗣|落合福嗣]]}}
: 本作主角。投手，[[左投左打|左投左打]]。背號：'''57'''→'''39'''→'''40'''→'''41'''。開始連載時26歲。出身：[[山梨縣|山梨縣]]。178公分、79公斤。效力球隊；神宮Spiders（原型為[[中央聯盟|中央聯盟]]的[[東京養樂多燕子|東京養樂多燕子]]）。
: 年薪：1800萬(連載開始時)→2600萬→3000萬+300萬激勵獎金。
: 高中畢業之後隨即加入職業棒球界，左[[中繼投手|中繼投手]]和[[一人左投|一人左投]]，在職業棒球界已經打滾了八年。由於球探曾對他說球員生涯的短暫，必須要賺到退休後足以生活的費用，否則退休後的年收入可能連普通上班族的一半都不到，因此在球場上對錢非常執著，常把「球場上埋著錢」掛在嘴邊。
: 喜歡研究年鑑，對於球員年薪有著極大興趣，遇到年薪比自己低的球員會輕視，即使是右打者也能輕蔑的將對方三振。相反的，對於遇到年薪比自己略高的球員，容易讓自己陷入迷惘而表現不佳，但是當年薪超過一億日元時思考反而會當機，卻因此能發揮不錯的成績。
: 連載初期擔任球隊的中繼左投，在系列賽表現平平，卻在季後賽大放異彩而得到加薪的機會。加入職棒第九年，在球團考量缺乏優秀的先發左投手的情況下，從[[牛棚|牛棚]]中繼升格為先發，但先發上場時隊友打擊火力全都未開張，進而累積大量的壓力想回牛棚繼續擔任中繼。雖然最後找到了先發投手應有的直覺跟態度，並創下「[[完投|完投]]九局零失分」的壯舉，但最後擔任先發的首年還是留下了「13戰0勝」的不光彩成績，而被踢到二軍作為回歸中繼投手的冷卻期。
: 在生涯第九年的季後賽時大顯神威，以極少的用球數在短局數中多次重挫對手士氣，即使球隊以勉強擠進第三名的成績進季後賽，仍然在中央聯盟冠軍賽橫掃對手，跌破記者與球評的眼鏡，並拿下季後賽MVP。
: 然而在[[日本大賽|日本大賽]]中與第四棒強打丸金千太郎，因為過於緊張而表現失常，使球隊開賽就吞下三連敗，遭到球迷的怒罵。認為對於加薪完全沒有幫助的日本大賽中，還要不斷出醜、遭人謾罵非常不划算，便破罐子破摔以放棄的心態隨意投球，卻因此反而讓球隊起死回生，在三連敗後拿下四連勝奪得日本冠軍，凡田雖然與日本大賽MVP擦肩而過，卻也拿下最有價值投手獎。
: 樣板球員類似於[[東京養樂多燕子|東京養樂多燕子]]的左投手{{link-ja|佐藤賢|佐藤賢}}<ref name="shuube">出自《週刊棒球》2012年2月6日號，第16頁刊登森高與上原浩治的接受專訪。</ref>，身為左投手有罕見的極高控球能力，使用[[側肩投法|側肩投法]]，擅長[[滑球|滑球]]與[[指叉球|指叉球]]。球速約135公里，快速球曾有過148公里的紀錄，在精準控球的同時能維持球速是其個人特色。學生時代眼界比起其他高中生更廣，對於職業生涯也想得比普通高中生更遠、更現實，與棒球社的學長兼王牌投手：西浦菜津樹，互看不順眼，雖然在控球上力壓隊上的王牌，但在投球的氣勢跟球速卻被對方壓制，只有球探跟教練看出夏之介的潛能。

; {{nihongo|小雪|ユキちゃん|聲：[[市道真央|M·A·O]]}}
: 在銀座惠比壽味平食堂擔任侍應，初登場時24歲，大學畢業後轉讀烹飪學校，成為食堂裡的菜單設計。長相甜美，有不少男性顧客為了他而上門，凡田的暗戀對象。本來身為大阪人而支持大阪Tempters，從而對神宮Spiders的中繼投手凡田非常反感，但即使如此也依然沒有對常來食堂用餐的凡田留下印象。在季後賽結束後的球迷回饋活動中，跟凡田有極近距離的接觸，之後凡田轉任先發開始有知名度時，才知道常來餐廳用餐的陌生人是職業球員，慢慢開始對凡田與神宮Spiders隊有好感。

; {{nihongo|松本秀夫|松本 ひでお（まつもと ひでお）|聲：松本秀夫}}
: 唯一實際人物。棒球球評，比賽之外同時在音羽電台有主持節目。他的現實搭檔也會客串出演。

; {{nihongo|湯瑪斯·赫普博|トーマス·ホットポッパー|聲：[[小西克幸|小西克幸]]}}
: 來自美國的洋將，左[[中繼投手|中繼投手]]，力量和速度型投手。身高超過兩米，脾氣暴躁，很怕因為洋將名額滿額，而再被送到二軍 （Minors）。第一次跟凡田參觀[[靖國神社|靖國神社]]時祈求美日不會再次陷入戰爭。

; '''{{nihongo|樹六破|樹 六破（いつき ろっぱ）|聲：[[高木涉|高木涉]]}}'''
: 第二季的配角。右投左打，右外野手／後備打者。26歲。性格隨性，對棒球只是興趣的延伸，喜歡二軍節奏較慢的比賽。因此，每當他在一軍的重要時刻常常發生防守失誤，或未能推進隊友。可是當他發現妻子肚中胎兒胎位不正時，被凡田訓話的他終於發憤圖強，離開埼玉，搬到東京自住後，連續三場打進3分打點，包括三支全壘打。在協助蜘蛛隊在聯賽勝掃帚隊（原型為[[中央聯盟|中央聯盟]]的[[讀賣巨人|讀賣巨人隊]]，即日本聯盟最成功，和歷史最悠久的球隊），也拿下該球賽的MVP。他接著在廣島，以先發[[指定打擊|指定打手]]，協助凡田拿下第四勝，也因掃帚隊和大阪誘惑者隊（原型為[[中央聯盟|中央聯盟]]的[[阪神虎|阪神虎]]隊）當天敗陣，蜘蛛隊升上聯盟第一。但連續6日（3場）後，為了探望接近臨盆的妻子而回家後，表現突然嚴重滑落，其實是敵隊投手已發現他不擅長打變速球，已作出應對措施。在對陣大阪誘惑者隊的主場戰當日，妻子臨盆，剖腹誕下男孩，但他因為沒有呼吸，要到深切治療部。可是，連岳父都看不下去而訓斥他，叫他在拿下聯盟首位前不准回家。當晚，他拯救凡田，在第9局下擊出追平比數陽春全壘打，使球隊最後打完12局和局收場，提早拿下聯盟冠軍，進入季後賽。他承認：對於妻兒不穩定的狀況，反而讓他忘卻棒球賽輸贏、壓力或後續影響。可因此，他挽救了凡田的名聲和未來。
