'''{{Ruby|萘|-{zh-cn:nài; zh-hk:nài; zh-tw:ㄋㄞˋ}-}}'''（{{lang-en|Naphthalene}}），旧称'''艿'''，俗稱焦油腦，是一种[[多环芳香烃|多环芳香烃]]。

== 来源 ==
可从[[煤焦油|煤焦油]]分离和[[煉油廠|石油提煉]]製得。

== 物理性质 ==
萘为无色或白色有光泽的鳞片状[[单斜|单斜]]结晶，有溫和芳香氣味，粗萘有特殊的煤焦油样臭味；[[熔点|熔点]]80.2℃，[[沸点|沸点]]218℃；难溶于[[水|水]]，易溶于[[有機溶劑|有機溶劑]]中，如[[乙醇|乙醇]]及[[乙醚|乙醚]]。能挥发并易升华，能[[水蒸氣蒸馏|水蒸氣蒸馏]]。与空气形成[[爆炸性混合物|爆炸性混合物]]；[[爆炸极限|爆炸极限]]0.9％－5.9％（体积）。

== 化学性质 ==

正常情况下，萘比[[苯|苯]]更易发生典型的[[芳香亲电取代反应|芳香亲电取代反应]]（如[[卤化|卤化]]、[[磺化|磺化]]、[[硝化|硝化]]）以及[[氧化|氧化]]、[[加氢|加氢]]、[[加氯|加氯]]等反应。硝化和卤化反应主要发生在α位（1位）上。磺化反应在低温时生成反应[[活化能|活化能]]低的[[α-萘磺酸|α-萘磺酸]]，但磺化反应是可逆的，随着反应温度升高，产物逐渐转变为[[空阻|空阻]]较小的[[β-萘磺酸|β-萘磺酸]]。

== 用途 ==

萘用于生产[[染料|染料]]、[[树脂|树脂]]、[[溶剂|溶剂]]、[[炸药|炸药]]、[[消毒剂|消毒剂]]、[[杀虫剂|杀虫剂]]、[[防腐剂|防腐剂]]、防蛀剂和[[碳|碳]]化照明气。萘的催化氧化可以制得[[苯酐|苯酐]]：

== 危险性 ==

遇火、高熱時可燃，放出有毒的刺激性煙霧，為可能[[致癌|致癌]]因子，粉體與空氣可形成爆炸性混合物。

萘如果由呼吸道和[[消化道|消化道]]进入[[人|人]]体，会引起毒性症状。萘在体内的代谢产物[[萘酚|萘酚]]和[[萘醌|萘醌]]可引起血管内溶[[血|血]](特別是[[蠶豆症|蠶豆症]]患者)，内脏出血，[[肝|肝]]中毒。由于萘可以储存入[[脂肪|脂肪]]中，而脂肪转化成热量的同时可以释放萘引起中毒，所以萘中毒不会在隔绝接触后短时间内消失。

萘在体内首先被氧化为1,2-[[环氧|环氧]]萘。然后环氧萘可以通过三种途径代谢：

# 重排为[[1-萘酚|1-萘酚]]，与[[硫酸|硫酸]]或[[葡糖醛酸|葡糖醛酸]]结合。
# 在环氧化物水解酶的催化下水解为反-1,2-二氢-1,2-二羟基萘，失水为[[2-萘酚|2-萘酚]]，或者脱氢为[[邻萘醌|邻萘醌]]或[[对萘醌|对萘醌]]。
# 与[[谷胱甘肽|谷胱甘肽]]结合，并最终以[[巯基尿酸|巯基尿酸]]的形式排出。
