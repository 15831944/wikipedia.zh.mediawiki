'''萬達鎮－巴生線'''（{{lang-en|'''Bandar Utama-Klang Line'''}}），簡稱「BKL」，計畫名稱「LRT3」<ref name="autogenerated1">{{cite news|author=Minderjeet Kaur|title=Sultan's nod for LRT Line 3|publisher={{tsl|en|New Straits Times||New Straits Times}}|date=17 April 2015}}</ref>或「第三轻快铁」，早期則以「莎阿南線」稱之，是[[巴生谷|巴生谷]]第十一条列车线路，也是马来西亚第三条[[轻快铁|轻快铁]]线。这条路线属于[[陆路公共交通委员会|陆路公共交通委员会]]（SPAD）的大吉隆坡/[[巴生谷综合运输系统|巴生谷综合运输系统]]。路线编号为11，而颜色则为天蓝色。

本綫由[[快捷通軌道|快捷通軌道]]營運<ref>{{cite web |url=http://www.thestar.com.my/Business/Business-News/2013/10/21/Study-on-LRT-3-nears-completion.aspx|title= Study on LRT from Kelana Jaya to Klang via Shah Alam nears completion |publisher=The Star Online |accessdate=2013-10-21}}</ref><ref>{{cite web |url=http://manage.mmail.com.my/story/third-lrt-boost-port-klang-66899 |title=Third LRT to boost Port Klang |publisher=The Malay Mail |accessdate=2013-10-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20131213152438/http://manage.mmail.com.my/story/third-lrt-boost-port-klang-66899 |archivedate=2013-12-13 }}</ref>。此輕快鐵系統共長37[[公里|公里]]，並是马来西亚第一条绿色环境铁路。<ref>http://www.sinchew.com.my/node/1668950/</ref>本計畫由{{tsl|en|Prasarana Malaysia|馬來西亞國家基建}}於2013年4月24日宣布。<ref>{{cite web |url=http://www.malaysia-chronicle.com/index.php?option=com_k2&view=item&id=91022:shah-alam-lrt-link-in-pipeline&Itemid=3#axzz2nMbOGznV |title= Shah Alam LRT link in pipeline |publisher=Malaysia Chronicle |accessdate=2013-04-24}}</ref><ref>{{cite web |url=http://www.bidocean.asia/Asia-tender-business-news/99293-MY--Malaysia-to-Add-a-New-LRT-Line-to-Klang.html |title=MY – Malaysia to Add a New LRT Line to Klang? |publisher=Bid Ocean Network |accessdate=2013-05-30 |deadurl=yes |archiveurl=https://archive.is/20131213142701/http://www.bidocean.asia/Asia-tender-business-news/99293-MY--Malaysia-to-Add-a-New-LRT-Line-to-Klang.html |archivedate=2013-12-13 }}</ref>
