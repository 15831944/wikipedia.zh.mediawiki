'''万安桥站'''（工程名为'''平海路站'''）是[[杭州地铁5号线|杭州地铁5号线]]建设中的地下车站，，位于[[杭州市|杭州市]][[上城区|上城区]][[建国中路|建国中路]]解放路交叉口以北。

==结构==
万安桥站为地下二层岛式站台，沿建国中路布置，在站后设有两道存车线。车站总长约478.4米，标准段宽21.3米，深约16.5米，设4个出入口、2组风亭。 

==历史==
2017年1月3日，车站开工。<ref>{{cite web |title=关于地铁5号线平海路站主体结构施工期间调整交通组织措施的通告|url=http://www.hzpolice.gov.cn/Html/201701/06/0bf04b75-91ef-491e-9545-c71ac69870d7.html |website=杭州市公安局交通警察局}}</ref>

== 事故 ==
*2017年6月27日16点30分，万安桥站工地施工单位在围挡范围内进行集水坑施工时挖机不慎将直径500MM的燃气管挖破。影响1家公建用户。民用气不受影响。地铁集团与相关单位接报后立即启动应急预案，关闭燃气阀门，进行抢修。事故没有造成人员伤亡。<ref >{{cite web|url=http://weibo.com/1750349192/F9Mve75Ze?refer_flag=1001030103_&type=comment#_rnd1498628617962 | language= | title=杭州建国中路解放路口燃气管道被挖破 |publisher=FM93浙江交通之声 |date=2017年6月27日 |accessdate= }}</ref>

*2019年3月7日上午7时，万安桥水泵站D1800出水管因万安桥站施工受损，出现渗漏现象，建国北路南向北车道路面发生下沉，污水进入地铁基坑内，事故暂未造成人员伤亡。<ref>{{cite web |title=杭州建国中路出现路面塌陷 目前仍在应急抢险中(图) |url=http://zj.sina.com.cn/news/m/2019-03-07/detail-ihsxncvh0538317.shtml |website=新浪浙江}}</ref>
