'''万胜围站'''是[[广州地铁|广州地铁]][[广州地铁4號綫|4號綫]]及[[广州地铁8號綫|8號綫]]的換乘站，同时是8号线现时的东端终点站。车站位于[[海珠区|海珠区]][[新港东路|新港东路]]琶洲新馬路（原規劃新滘南路）路口的地底。2005年12月26日随着2号线调整工程和4号线大學城專線的开通而启用。

本站是广州地铁第一个啟用后即為換乘站的車站。萬勝圍站上方的[[萬勝廣場|萬勝廣場]]，亦是[[廣州地鐵|廣州地鐵集團有限公司]]總部、運營管理指揮中心、廣州地鐵博物館的所在地。

== 車站結構 ==
===車站樓層===
本站共有2層，地面為新港東路、琶洲新馬路、[[万胜围站_(有轨电车)|有轨电车万胜围站]]、萬胜廣場及市郊地區，地下一層為南北站厅以及8號綫月台；地下二層為4號綫月台以及非付費區站厅联络通道。
地下一、二層之間設有扶梯連接8號綫月台及4號綫月台。

===站厅===
万胜围站分為南北兩個站廳，其中南站厅被划分为西侧扩建部分以及中部原有部分，均位于地下一层，兩者被8號線路軌相隔。南北站廳在非付費區通过位于中部站台一侧的地下二层的联络通道連接。同时，南站厅中部与西侧的付费区靠8号线的4號站台联络。中部站厅收费区内设有专用电梯（[[升降机|升降机]]）、[[扶手电梯|扶手电梯]]可供乘客来往4号线月台。

由於此站與4号线大学城专线同步興建開通，因此本站是8号线唯一一座使用玻璃幕板+马赛克组合裝修的车站。

另外，万胜围站內的北站厅非付费区設有[[面包糕饼店|面包糕饼店]]以及[[洗衣店|洗衣店]]，另外万胜围站也提供自助照相机以及[[自動售賣機|自動售賣機]]、自助售卡/充值机、“好易”自助服务等自助服务。

===月台以及换乘===
本站8號綫為側式月台，設於新港東路地底，4號綫為島式月台則設於原规划新滘南路（現時琶洲新馬路）之地底。兩線月台組成一個X字型。8號綫月台為上層，4號綫月台在下層。值得一提的是，4号线月台因为扶梯、升降機等設備的佈置需要一定空間，月台实际比4节L車编组所需要的要长，所以在两个月台的头端的月台幕门经过特别设计，方便司机在到站停车打开驾驶室门后，便可推开对应位置的月台幕门观望月台情况。相应位置上亦同时装设有闭路电视方便司机观看。

另外，本站4號線站台南端設有一條渡綫。當4號綫的北段出現故障時，往黃村站的列車將會通過此渡線折返，以本站為臨時總站。此時往車陂南及黃村站的乘客需轉乘其它交通工具，過江往市區的乘客則需要改用8號線再換乘其它路線抵達。在车陂南站随5號線開通前，所有4號線列車皆使用該渡線折返。

8號線站台東端則設有一組交叉渡線，現時8號線所有列車皆使用該組渡線折返。在早晚高峰過後及晚上收車前夕，部分列車在4號月台清客後會退出服務，然後折返至3號月台開出，沿途不停站返回[[赤沙車輛段|赤沙車輛段]]。屆時3號月台會發出廣播提醒乘客不能上車。

换乘方面，由于本站為8號線終點站，以及客流组织的需要，4号线月台北端扶梯連接8號線的3號月台，供4号线乘客换乘至8号线；4号线月台南端的扶手电梯則連接8號線的4號月台，仅供8号线乘客换乘至4号线。如果8号线乘客到达本站后需要乘搭回头车时，由于地下二层之联络通道被劃入非收费区，乘客须经4号线月台前往。

== 利用状况 ==
本站作为4號線首通段（大學城專線）開通時與市區線（舊2號線）的唯一換乘車站，自此承担了来往[[广州大学城|广州大学城]]乘客的换乘任务。随后4号线南延伸至金洲站，北延伸至黄村站后，8号线沿线的乘客若要前往[[广州科学城|广州科学城]]、天河智慧城，或者南沙区一带，均需要在本站換乘，因此全天換乘客流都屬高水平。每逢週日（含节假日最后一天）下午及大學城各高校開學前夕，學生集中返校更會導致本站換乘客流進一步上升，更要實施客流控制，減緩8號線乘客換乘至4號線的速度。

自2016年4月22日起，本站八号线每周五晚高峰17:30-19:30实施常态化客流控制<ref>[http://news.dayoo.com/guangzhou/201604/08/139995_47369678.htm 广州地铁新增万胜围站为常态化客流控制站点] {{webarchive|url=https://web.archive.org/web/20160424183100/http://news.dayoo.com/guangzhou/201604/08/139995_47369678.htm |date=2016-04-24 }}</ref><ref>{{Cite web |url=http://news.ycwb.com/2016-04/08/content_21767821.htm |title=广州两地铁站客流控制有变化 新增万胜围站为常态化客流控制站 |access-date=2018-03-10 |archive-url=https://web.archive.org/web/20180311081120/http://news.ycwb.com/2016-04/08/content_21767821.htm |archive-date=2018-03-11 |dead-url=yes }}</ref>。

另外，由於本站接駁海珠有軌電車THZ1的同名車站，附近亦有黄埔古村、廣州地鐵博物館等本地旅遊景點及文化设施，甚至本站是海珠区前往[[长洲岛_(广州)|长洲岛]]最近的一座地铁车站。因此每逢天氣狀況良好的周末，万胜围站都會湧現大量本地或外地旅客，利用地铁換乘有軌電車，或接驳公交及水上巴士前往黄埔古村以及长洲岛。

==历史==
万胜围站规划最早提出于1997年出現的《广州市城市快速轨道交通线网规划研究（最终报告）》的方案A中，2号线拆解後的東西向路線則稱為5號線，繼續沿昌崗路向西過江至西塱，東段則在琶洲後分成兩支，一支沿現有四號綫的路線北上至奧體中心附近地區，另一支則繼續東進至新洲一帶，当时亦设有名為'''琶洲塔站'''<ref>[http://news.dayoo.com/china/gb/content/2002-11/15/content_813653.htm  二号线将东延至琶洲塔] {{webarchive|url=https://web.archive.org/web/20150402120220/http://news.dayoo.com/china/gb/content/2002-11/15/content_813653.htm |date=2015-04-02 }}</ref>的中途站。而在2000年的《近期線網規劃實施調整方案》方案中，当时的5号线支線被獨立为4号线，并北延伸到科學城內，同時由于大学城的落实兴建并南延至[[广州大学城|广州大学城]]，本站亦改为2、4号线的换乘车站，车站位置与现时基本一致。

2003年1月28日，本站正式开工，标志着2号线调整工程及4號線首通段（大學城專線）正式开工<ref>[http://news.sohu.com/82/69/news206046982.shtml 广州地铁四号线开始建设 计划于2006年建成使用]</ref>。2005年12月26日，本站随四號綫首期大學城專線段與2號線調整工程投入服務时使用。

从万胜围站启用至2010年9月21日，本站一直都是舊2號線的始发站之一。同年9月22日至24日，本站2號線月台隨舊2號線一同停運，進行線路拆解。最終該月台於2010年9月25日重新啟用，並改屬新的8號線，同樣作為其東端總站至今。

===车站命名===
车站命名方面，由于当时车站附近多为農田，没有任何高楼建筑，故本站被地铁官方命名为'''万胜围站'''，“万胜围”是解放前一个地主对自己所拥有的几十亩土地的命名<ref>{{Cite web |url=http://www.ycwb.com/xkb/2007-01/22/content_1358499.htm |title=个别地铁站名让人糊涂 |access-date=2014-09-19 |archive-url=https://web.archive.org/web/20160105104251/http://www.ycwb.com/xkb/2007-01/22/content_1358499.htm |archive-date=2016-01-05 |dead-url=yes }}</ref>。很多乘客誤以为万胜围站附近有“万胜围村”，不过现实中并没有这座村。

由于车站命名並非原生地名，乘客對於其过于陌生，因此存在很多争议，有部分人建议使用原來的名稱“琶洲塔站”。不过[[琶洲塔|琶洲塔]]距离本站有一公里以上，并不适合命名为“琶洲塔站”。

===C出入口旁出现沉降===
本站C出入口旁在2006年3月初出现沉降，地铁公司称是本站在启用之前，相关市政单位在对车站出入口周围的地面进行回填土施工时基础夯得不够牢固，致使近期每次下雨该处都有不同程度地沉陷。由于该出入口外地面的沉降是市政地面施工造成的，不会影响车站结构的安全。出现这种情况后，地铁公司采取了临时性措施，待雨季结束后进行永久加固<ref>[http://news.sohu.com/20060329/n242532205.shtml 因市政单位回填土层不实　二号线万胜围站出口外地面沉陷]</ref>。

==未來發展==
===8号綫东延段===
根據规劃，8号线將由万胜围站往東伸延至番禺区东部的[[广汽基地站|广汽基地站]]，屆時万胜围站將從8号线东端終點站變為中途站。目前該段工程仍在規劃階段，沒有建造時間表。
