'''万源市'''位于[[四川省|四川省]]东北部，与[[陕西省|陕西省]]及[[重庆市|重庆市]]接壤，为[[达州市|达州市]]代管[[县级市|县级市]]，全市面积4065平方公里。

== 历史 ==
*秦置巴郡宕渠县。
*由汉到明代，属达州通川郡东乡县。
*明正德十年（1515年）析東鄉太平里所置。
*清代初屬[[夔州府|夔州府]]，雍正六年，升[[達州|達州]]為[[直隸州|直隸州]]，以[[東鄉縣_(四川省)|東鄉縣]]、[[太平縣_(四川省)|太平縣]]二縣往隸。升[[達州直隸州|達州直隸州]]為[[綏定府|綏定府]]。
*清乾隆六十年（1795）全县分为十保。嘉庆七年（1802）升县为直隶厅。道光二年（1822）由太平直隶厅复置太平县，原太平厅七八九保设城口厅，割巴州（今巴中）之太平、长乐两乡及通江之安仁里、长宁里、麻巴里各甲拨入太平县，时人称“三乡”为新太平，原七保为老太平，时七保三乡共辖119甲，51个场镇，其中九乡黄钟堡设巡检分署。
*[[民国|民国]]三年（1914年），因與山西、安徽、浙江、江蘇等省太平縣同名，又較置於後，即以「縣東北有萬頃池，鄰邑之水多源於此」，改為萬源。然萬源境內只有千頃池，確無萬頃池；萬頃池即今巫溪縣西之紅池壩；萬源之名得來，是歷史的大誤會。<ref>『萬源名由　歷史誤會』，趙洪賢，《達縣師範高等專科學校學報》，1999第1期</ref>
*1933年至1935年，[[红四方面军|红四方面军]]建立了[[川陕革命根据地|川陕革命根据地]]，先后在县内建立万源、红胜、[[城口|城口]]三个县[[苏维埃政府|苏维埃政府]]。
*1949年10月1日，中华人民共和国建立，12月29日，中国人民解放军进驻县城，全县被[[中國共產黨|中國共產黨]]控制。
*1990年7月3日，由46节航空汽油槽车和9节货车编组的0201次列车，从陕西安康行至万源县境内的“梨子园”隧道内发生爆炸，列车长等4人当即死亡，14人受伤，造成襄渝铁路停运24天。
*1993年7月由原万源县和白沙工农区合并建立万源市，属于省辖县级市，由达州市代管。

== 地理 ==
=== 气候 ===
万源属亚热带季风气候，四季分明。降雨充足，集中在夏季。1月平均气温3.9℃，7和8月平均气温24.8℃，全年均温14.7℃。

== 人口 ==
总人口54.95万人，其中农业人口46.46万人。

== 经济 ==
万源市为一典型山区农业市，经济以农业为主，工业及第三产业比较落后。2004年，全市生产总值达到25.24亿元，人均生产总值4520元，地方财政收入4044万元。
