《'''万家灯火'''》（{{lang-en|'''Glittering Days'''}}）中国大陆电影，导演[[安战军|安战军]]，主演刘桦、辛柏青、冯千、刘琳、吴越、金雅琴、刘金山、黄海冰、王长立。<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2010-05-06/14352949713.shtml |title=《万家灯火》入围大影节 黄海冰应邀参加颁奖礼 |accessdate= 2010 |publisher=新浪网 |work= |language=zh-hans}}</ref>该片根据[[北京人艺|北京人艺]]的同名话剧改编而成。

==剧情==
该片讲述[[北京市|北京市]][[北京南城|南城]]金鱼池地区[[胡同|胡同]]大杂院和[[筒子楼|筒子楼]]的拆迁和安置，主要叙述了“何氏”三兄弟和母亲因为“房子”纠纷的生活趣事。

==花絮==
该片是2009年，[[中华人民共和国|中华人民共和国]][[国家广电总局|国家广电总局]]庆祝建国60周年的献礼片，旨在塑造[[中国共产党|中国共产党]]为人民群众安居乐业而奋斗的光辉形象。

2010年，该片入围“第十七届大学生电影节”。<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2010-04-17/20382932647.shtml |title=《万家灯火》亮相大影节：看似平常最奇崛 |accessdate=2010 |publisher=新浪网 |work= |language=zh-hans}}</ref>

2012年，该片亮相柏林中国文化中心。<ref>{{cite web |url=http://www.scio.gov.cn/zhzc/3/2/Document/1102611/1102611.htm |title=电影《万家灯火》在柏林中国文化中心上映 |accessdate=2012 |publisher=中华人民共和国国务院 |work= |language=zh-hans}}</ref>
