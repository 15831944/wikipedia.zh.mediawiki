'''锡'''是一种[[化学元素|化学元素]]，其[[化学符号|化学符号]]是'''Sn'''（[[拉丁语|拉丁语]]''Stannum''的缩写），它的[[原子序数|原子序数]]是50。它是一种[[主族金属|主族金属]]。纯的锡有银灰色的金属光泽，它拥有良好的伸展性能，它在[[空气|空气]]中不易[[氧化|氧化]]，它的多种[[合金|合金]]有防[[腐蚀|腐蚀]]的性能，因此它常被用来作为其它金属的防腐层。锡的主要来源是它的一种氧化物矿物[[锡石|锡石]]（[[二氧化锡|SnO<sub>2</sub>]]），盛產於[[中華人民共和國|中国]][[雲南|雲南]]、[[馬來西亞|馬來西亞]]等地。

== 主要特征 ==
=== 物理特性 ===
[[File:Candlestick_made_of_Tin_by_Royal_Selangor.JPG|thumb]][[皇家雪兰莪|皇家雪兰莪]]制作的锡製烛台]]
锡是一种[[延展性|可延展的]]、[[延展性|柔软的]]、高度[[晶体|结晶状]]的、银白色的金属。当锡棒被弯曲时，由于锡晶体是[[孪晶|孪晶]]，可以听到被称之为[[锡鸣|锡鸣]]的爆裂声。<ref name="Hol1985">{{cite book|publisher = Walter de Gruyter|year = 1985|edition = 91–100|pages = 793–800|isbn = 3-11-007511-3|title = Lehrbuch der Anorganischen Chemie|first = Arnold F.|last = Holleman|coauthors = Wiberg, Egon; Wiberg, Nils;|chapter = Tin| language = de}}</ref>

=== 同素异形体 ===
[[File:Sn-Alpha-Beta.jpg|thumb]]

在常压下锡有2种[[同素异形体|同素异形体]]。以'''灰锡'''（α锡）、'''白锡'''（β锡）的状态存在。

在室溫，是常见的银白色金属'''白锡'''，富有延展性。白锡的晶体为正方晶结构，呈金属性，比重較灰錫重。当温度回降到低于13.2℃的话，它会慢慢变为粉末状的'''灰锡'''。灰锡的晶体结构是与[[钻石|钻石]]、[[硅|硅]]和[[锗|锗]]类似的[[钻石结构|鑽石型]][[晶体结构|晶体结构]]。灰锡因为其[[原子|原子]]间形成了共价结构，[[电子|电子]]不能自由转移，故而没有任何金属性。灰锡是暗灰色的粉狀物，除了一些非常特殊的[[半导体|半导体]]应用，日常用途不大。<ref name="Hol1985"/>在低温下，白锡转化为灰锡的现象常被称为{{Link-en|锡疫|tin pest}}。{{#tag:ref|这种转化被称为''锡病''或''锡疫''。锡疫在18世纪的北欧是一个特别的问题，因为管风琴的琴管是由锡合金制成，在漫长的冷冬有时会发生锡疫。有些来源说，在拿破仑的1812年的俄国战役时，由于温度太低以至于士兵们的军服上的锡纽扣都被冻坏，是他们战败原因之一。<ref>{{cite book|last = Le Coureur|first = Penny|coauthors = Burreson, Jay|title = Napoleon's Buttons: 17 Molecules that Changed History|place = New York|publisher = Penguin Group USA|year = 2004}}</ref>|group="註"}}這個現象最早由[[亞里士多德|亞里士多德]]發現，稱[[錫疫|錫疫]]。未染上「锡疫」的锡板，一旦和有「锡疫」的锡板接触，也会产生灰色的斑点而逐渐「腐烂」掉。这个过程的轉化溫度會因存在杂质如[[铝|铝]]和[[锌|锌]]而降低。在锡中加入[[锑|锑]]或[[铋|铋]]可以防止锡的退化，促进锡的延展性。<ref name="Schwartz">{{cite book|first = Mel|last = Schwartz |title = Encyclopedia of Materials, Parts and Finishes|edition = 2nd|chapter = Tin and Alloys, Properties|publisher = CRC Press|year = 2002|isbn= 1-56676-661-3}}</ref>這是由於铋原子中有多余的电子可供锡的结晶点阵，使錫的晶體結構稳定化，完全消除「锡疫」的可能性。

锡的另外两种合金γ锡和σ锡，只存在于温度高于161℃ (322℉)和压力大于几个[[帕斯卡|GPa]]的环境中。<ref>{{cite journal|first = A. M.|last = Molodets|coauthors = Nabatov, S. S.|title = Thermodynamic Potentials, Diagram of State, and Phase Transitions of Tin on Shock Compression|journal = High Temperature|volume = 38|issue = 5|year = 2000|pages = 715–721|doi = 10.1007/BF02755923}}</ref>

=== 化學性質 ===
{{main|有機錫化合物}}
锡受[[水|水]]的锈蚀的影响很小，但易被[[酸|酸]]和[[碱|碱]]腐蚀。锡可以被高度的抛光，可被用作其他金属的保护层。<ref name="Hol1985"/>锡表面会形成保护性的氧化层以防止进一步的氧化。白锡和其他的锡合金表面都会形成这样的保护层。<ref>{{cite book| url = http://books.google.com/books?id=KXwgAZJBWb0C&pg=RA1-PT126 | page = 126| title = Handbook of corrosion data| isbn = 978-0-87170-518-1| author1 = Craig| first1 = Bruce D| last2 = Anderson| first2 = David S| last3 = International| first3 = A.S.M.| date = 1995-01}}</ref>当氧气存在于溶液中时，锡会变成一种[[催化剂|催化剂]]，加速化学腐蚀。<ref name="Hol1985"/>

在空气中加热后锡可以形成SnO<sub>2</sub>。SnO<sub>2</sub>是弱酸性的。与熔融碱反应可以形成[[锡酸盐|锡酸盐]]。它可以直接与[[氯|氯]]和氧反应。在稀酸中它可以取代[[氢|氢]]离子。

由於處於[[金屬|金屬]]和[[非金屬|非金屬]]中間，同時具有兩者的特性。

*作為金屬，可氧化至二價錫和較穩定的四價錫。
:*二價錫是[[兩性_(化學)|兩性]]的，在[[酸|酸]]中，可形成 [SnX<sub>3</sub>]<sup>−</sup>[[離子|離子]]，X為F<sup>−</sup>,Cl<sup>−</sup>等；在[[鹼|鹼]]中，會形成[Sn(OH)<sub>3</sub>]<sup>−</sup>等離子。
:*四價錫也是[[兩性_(化學)|兩性]]的，在[[酸|酸]]中，可形成 [SnX<sub>6</sub>]<sup>2−</sup>[[離子|離子]]，X為F<sup>−</sup>,Cl<sup>−</sup>等；在[[鹼|鹼]]中，會形成[Sn(OH)<sub>6</sub>]<sup>2−</sup>, [SnO<sub>3</sub>]<sup>2−</sup>等離子。四價錫的[[溶液|溶液]]都傾向[[水解|水解]]出SnO<sub>2</sub>[[沉澱|沉澱]]。

*作為非金屬，錫可和[[碳|碳]]，[[氧|氧]]等非金屬組成份子。尤其是碳，可和錫形成很多[[有機錫化合物|有機錫化合物]]，應用很廣泛。

=== 同位素 ===
{{Main|锡的同位素}}
锡有9种稳定的同位素，是所有化学元素中稳定同位素最多的。此外锡还有28种不稳定同位素。

== 用途 ==
锡很容易与[[铁|铁]]结合，它被用来做[[铅|铅]]、[[锌|锌]]和[[钢|钢]]的防腐层。涂锡的钢罐多用于贮藏食物，这是金属锡的一个重要市场。

其它用途：
* 锡是一些重要[[合金|合金]]如[[青铜|青铜]]、[[巴氏合金|巴氏合金]]等的组成部分。
* [[氯化锡|氯化锡]]在[[印刷|印刷]]术中被用作一种[[还原剂|还原剂]]和[[媒染剂|媒染剂]]。锡盐喷在玻璃上可以形成导电的涂层。这些涂层被用在防冻玻璃上。
* 一般玻璃板是将熔化的玻璃浇在锡板上形成的，来保证玻璃面的平坦和光滑。
* 焊锡含锡用来连接管道和电子线路，此外锡还被用在多种化学反应中。
* 锡纸常用来包装[[食物|食物]]或[[药品|药品]]。
* 可镀于铜和铁上，镀锡的铁片称为[[马口铁|马口铁]]；可防鏽、製作[[罐頭|罐頭]]容器。
* 有機錫可作為有機化合物的合成的[[試劑|試劑]]，作用包括還原[[官能團|官能團]]、造成[[自由基|自由基]]及令有機份子重新排列。

在3.75K的低温下，锡晶體可成为[[超导体|超导体]]，也是最早被发现的超导体之一，超导体的一个特别特征[[迈斯纳效应|迈斯纳效应]]便是首先在锡晶体中被发现。由于[[铌|铌]]-锡-混合物（Nb<sub>3</sub>Sn）拥有较高的[[临界温度|临界温度]]（18K）和较高的[[临界磁场|临界磁场]]（25特斯拉），它常被用来制作商用[[超导电磁铁|超导电磁铁]]的導线，一个数千克重的超导电磁铁可以产生一个数吨重的普通电磁铁所产生之磁场。

全球年锡用量约为30万吨，其中约35%用作焊锡，30%用为锡片和30%用为化学原料或颜色。由于越来越多的锡-铅-[[銲錫|焊锡]]被无铅焊锡（含95%以上的锡）代替，全球的锡用量每年约提高10%。2003年在伦敦金属交易所，锡的价格为每吨五千美元左右，2004年的价格达每吨八千到一万美元。

自1992年起，美国立法禁止包装产品中使用[[铅|铅]]、[[汞|汞]]、[[镉|镉]]及[[六价铬|六价铬]]这四种有毒金属。全球著名的红酒封套生产商拉蒙丁（RAMONDIN）公司便开始使用锡作为以上有毒金属的替代品，此种錫封套是当今红酒封套制造业中的领先技术且无法被輕易复制，因此成为保护红酒品牌及产品的一种有效途径。

== 历史 ==
锡是人类知道最早的金属之一，从古代开始它就是[[青铜|青铜]]的组成部分之一。早在前36世纪锡就被用来硬化[[铜|铜]]。约从前7世纪开始人类认识到纯的锡。於[[战国_(中国)|戰國時期]]就開始用來作武器的主要材料，[[無錫|無錫]]即以此命名，相傳無錫於戰國時期盛產錫，到了錫礦用盡之時，人們就以無錫來命名這地方，作為天下沒有戰爭的寄望。

今天有时锡这个词也被用在没有锡或只有很少锡的物体上。比如许多“[[锡纸|锡纸]]”实际上是铝纸。大多数锡罐实际上是钢罐，上面涂有一层非常薄的锡。

== 来源 ==
全世界约有35个国家产锡。几乎每个[[大洲|洲]]都有重要的产锡国。

在[[地壳|地壳]]中锡比较稀少，只占[[地壳|地壳]]的百万分之二，{{Fact|今天的锡矿的产量可能还可以维持约35年|time=2018-05-16T11:09:06+00:00}}。80%锡矿是[[沉积岩|沉积岩]]，至少半数的锡来自[[东南亚|东南亚]]（从[[中華人民共和國|中国]]中部经泰国到印度尼西亚）。

最重要得锡矿石是[[锡石|锡石]]（SnO<sub>2</sub>），锡矿中的原矿石含约5%的锡，首先矿石要被粉碎和用不同的方式提纯，提纯后的锡矿含75%的锡。在[[精炼炉|精炼炉]]中锡从其矿物中用[[碳|碳]]还原出来。精炼炉中的温度稍高于锡的熔点，而杂质的熔点比这个温度要高，因此还原的锡可以从炉中流出。少量锡来自它的硫化物如[[圆柱锡石|圆柱锡石]]、[[硫银锡矿|硫银锡矿]]和[[硫锡铅矿|硫锡铅矿]]等。越来越多的锡是回收来的。

[[中華人民共和國|中国]]的锡产量占全世界的30%，其它东南亚国家一起占30%，南美洲占20%。

== 生理作用 ==
金属锡即使大量也是无毒的，简单的锡化合物和锡盐的毒性相当低，但一些有机锡化物的毒性非常高。尤其锡的[[三烃基|三烃基]]化合物被用作船的漆来杀死附在船身上的微生物和贝壳。这些化合物可以摧毁含硫的[[蛋白质|蛋白质]]。
