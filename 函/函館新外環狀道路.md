{{jp|c='''函館新外環狀道路'''|j=函館新外環状道路|hg=はこだてしんそとかんじょうどうろ||lead=yes}}是位於[[日本|日本]][[北海道|北海道]][[函館市|函館市]]的[[地域高規格道路|地域高規格道路]]。起於函館市[[桔梗町|桔梗町]]，終於函館市[[古川町|古川町]]，長約15公里{{Sfn|再評価原案準備書説明資料|2013|p=1}}，是連接[[函館機場|函館機場]]的機場道路，部分已通車，而仍有部分興建中{{Sfn|再評価原案準備書説明資料|2013|p=1}}。

函館新外環狀道路的興建，是用以緩和函館都市圏外環道路[[北海道道100號函館上磯線|北海道道100號函館上磯線]]（產業道路）的壅塞路況，與縮短到達[[道南|道南]]其他地區的時間{{Sfn|再評価原案準備書説明資料|2013|pp=19-27}}{{Sfn|函館新外環状道路|2015}}。2015年，[[赤川交流道|赤川交流道]]通車後，函館上磯線（產業道路）與周邊道路的車流量降低，事故也減少。<ref>{{Cite press release |url=http://www.hk.hkd.mlit.go.jp/press/H27/160316_1.pdf |title=函館新外環状道路「函館IC〜赤川IC」開通後1年 〜渋滞の緩和､事故の減少､観光等地域活性化に寄与〜 |date=2016-03-16 |format=PDF |publisher=[[北海道開發局|北海道開發局]] |accessdate=2016-07-26 }}{{Dead link|date=2018年1月}}</ref>。

== 計畫路段 ==
函館新外環狀道路目前分為機場道路與古川道路兩段。機場道路自北海道函館市桔梗町起，至同市上湯川町止，長10公里，路幅12.0公尺（全部完工後將有20.5 m），[[暫定2車線|暫定2車線]]（全部完工後將有4線道），每車道寬3.5公尺。其[[道路構造令|構造規格]]為第1種3級，速限為時速80公里。<ref name=kuko-road>{{Cite web |url=http://www.hkd.mlit.go.jp/topics/singi/h25_3/h251101_2_4_1.pdf |title=函館外環状道路（一般国道278号）空港道路 再評価原案準備書説明資料 |publisher=北海道開發局 |accessdate=2017-01-06 |archiveurl=https://web.archive.org/web/20150319064209/http://www.hkd.mlit.go.jp/topics/singi/h25_3/h251101_2_4_1.pdf |archivedate=2015年3月19日}}{{Dead link|date=2018年1月}}</ref>古川道路則是從上湯川町至古川町，長5.0 km，2000年已勘測，但興建時間未定。<ref>{{Cite web |url=http://www.ehako.com/news/news2015a/8490_index_msg.shtml |title=函館 - 赤川IC 3月14日開通 |date=2015-01-30 |work=函館市・道南地域ポータルサイト e-HAKODATE |accessdate=2016-07-26}}</ref>。
