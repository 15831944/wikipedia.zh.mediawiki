'''蘭芳園'''（{{lang-en|'''Lan Fong Yuen'''}}），是[[香港|香港]]一間歷史悠久的[[茶餐廳|茶餐廳]]，總店位於[[香港島|香港島]][[中環|中環]][[結志街|結志街]]2號及4A-6號，早年以[[大牌檔|大牌檔]]形式經營，現時除設於大廈的舊店及新店兩間店舖外，位於街邊的檔口仍然保留，是香港僅餘的大牌檔之一。

蘭芳園於1952年由[[林木河|林木河]]創辦，並首創港式飲料「[[鴛鴦_(飲料)|鴛鴦]]」，首創以[[茶袋|茶袋]]沖製、俗稱[[絲襪奶茶|絲襪奶茶]]的香港式濃滑奶茶<ref>[https://hk.news.appledaily.com/local/daily/article/20061013/6401756 地道文化 絲襪奶茶始祖 公開沖茶秘技] {{webarchive|url=https://web.archive.org/web/20061201143201/http://www1.appledaily.atnext.com/template/apple/art_main.cfm?iss_id=20061013&sec_id=4104&subsec_id=11867&art_id=6401756 |date=2006-12-01 }} 蘋果日報，2006年10月13日</ref>，其[[奶茶|奶茶]]使用來自[[斯里蘭卡|斯里蘭卡]][[科倫坡|科倫坡]]的「季後[[茶葉|茶葉]]」及[[馬來西亞|馬來西亞]]的[[植脂奶|植脂奶]]沖製。

除奶茶外，蘭芳園供應[[傳統|傳統]]茶餐廳食品如[[多士|多士]]、[[三文治|三文治]]等，其他食品包括[[蔥油|蔥油]][[雞扒|雞扒]]撈[[出前一丁|丁]]（撈[[出前一丁|出前一丁]]麵）、[[奶油|奶油]][[豬仔包|豬仔包]]、[[豬扒包|豬扒包]]、[[番茄|番茄]][[薯仔|薯仔]][[湯|湯]][[通粉|通粉]]等<ref>[http://big5.viewcn.com/sh/msj/2005/00353275.html 舊式茶餐廳魅力不減] {{webarchive|url=https://web.archive.org/web/20071219133643/http://big5.viewcn.com/sh/msj/2005/00353275.html |date=2007-12-19 }} 廣州日報</ref>。每日吸引不少名人、[[中環|中環]][[上班族|上班族]]及[[海外|海外]][[旅客|旅客]]慕名光顧，其門外常見排隊等候的人龍。 

這裡也是[[撈丁|撈丁]]的始創地。<ref>[http://www.youtube.com/watch?v=sb0LjDs1U2Q RTHK-香港故事(第19輯)07-速食速決-2012-4-02]</ref>

==分店==
2009年9月9日，蘭芳園於[[九龍|九龍]][[尖沙咀|尖沙咀]][[重慶大廈|重慶大廈]]地庫[[活方商場|活方商場]]開設分店<ref>[https://hk.lifestyle.appledaily.com/lifestyle/food/daily/article/20090910/13191832 蘭芳園絲襪奶茶過尖嘴] 蘋果日報，2009年9月10日</ref>。其水吧模仿中環蘭芳園總店第一代街邊鐵皮檔的設計。推出[[牛柳|牛柳]]及[[西冷牛扒|西冷牛扒]]等扒餐於晚上供應。

2011年5月10日，蘭芳園於[[香港|香港]][[上環|上環]][[信德中心|信德中心]]商場3樓開設分店。

== 争议 ==
2016年6月，[[浙江香飘飘食品公司|浙江香飘飘食品公司]]宣稱和蘭芳園签署协议，授权蘭芳園商标在内地的使用权。<ref>{{cite web |url=http://finance.ce.cn/stock/gsgdbd/201808/28/t20180828_30137169.shtml|title=转型液态奶茶遇阻 香飘飘巨量产能如何消化 |website=中国经济网 |publisher=[[时代周报|时代周报]] |access-date=2018-08-30 |date=2018-08-29}}</ref><ref>{{cite web |url=http://www.eeo.com.cn/2017/1202/318122.shtml|title=香飘飘的资本圈：掌门人拆解奶茶历史|publisher=[[经济观察报|经济观察报]] |access-date=2018-08-30 |date=2017-12-02}}</ref>但蘭芳園二代老闆林俊業在接受采访时表示，蘭芳園現時只在香港開設店鋪，除香港以外並無其他分店。同時，他們也親口向客人解釋，或張貼告示提醒，「我們根本在國內是沒有經營茶餐廳，希望人們知道，不要被騙。」<ref>{{cite web |url=https://nextplus.nextmedia.com/news/latest/20180524/588976|title=【山寨蘭芳園遍佈】大陸人無恥認抄襲 老闆無奈：我冇錢同佢打官司|publisher=[[壹週刊_(香港)|壹週刊]]}}</ref><ref>{{cite web |url=https://thestandnews.com/society/%E5%A4%A7%E9%99%B8%E5%86%8D%E7%8F%BE-%E5%B1%B1%E5%AF%A8%E5%BA%97-%E6%B8%AF%E8%98%AD%E8%8A%B3%E5%9C%92-%E5%BE%9E%E6%B2%92%E6%8E%88%E6%AC%8A%E5%85%A7%E5%9C%B0%E9%96%8B%E5%88%86%E5%BA%97/|title=大陸再現「山寨店」　港蘭芳園：從沒授權內地開分店|publisher=[[立場新聞|立場新聞]]}}</ref>
