'''蘭陽溪口水鳥保護區'''位於[[臺灣|臺灣]][[宜蘭縣|宜蘭縣]]境內的[[蘭陽溪|蘭陽溪]]、[[宜蘭河|宜蘭河]]及[[冬山河|冬山河]]出海口，即[[台2線|台2線]]北部濱海公路噶瑪蘭大橋以東一帶溼地<ref>{{cite web|url=http://wetland-tw.tcd.gov.tw/WetLandWeb/wetland.php?id=153|title=蘭陽溪口濕地|publisher=內政部營建署|accessdate=2014-10-31}}</ref>。

1972年，[[交通部觀光局|交通部觀光局]]劃為「雁鴨保護區」；1984年，[[中華民國內政部|內政部]]劃設「蘭陽海岸保護區」；1996年，[[行政院農業委員會|農委會]]依《野生動物保育法》公告為「'''宜蘭縣蘭陽溪口野生動物重要棲息環境'''」<ref>{{Cite web |url=https://conservation.forest.gov.tw/0000154|title=宜蘭縣蘭陽溪口野生動物重要棲息環境|accessdate=2019-01-27|publisher=農委會林務局}}</ref>，宜蘭縣政府同年公告為「蘭陽溪口水鳥保護區」<ref>{{cite web|url=https://conservation.forest.gov.tw/0000139|title=蘭陽溪口水鳥保護區|publisher=農委會林務局|accessdate=2019-01-27}}</ref>。

本區位於[[東北角暨宜蘭海岸國家風景區|東北角暨宜蘭海岸國家風景區]]內，為[[宜蘭平原|宜蘭平原]]上的知名賞鳥據點，歷年來共紀錄到鳥類236種，[[鷗|鷗]]科鳥類終年可見，是臺灣鷗科種類最多的地方<ref>{{Cite web |url=http://www.necoast-nsa.gov.tw/user/Article.aspx?Lang=1&SNo=04000506 |title=蘭陽溪口 |publisher=交通部觀光局東北角暨宜蘭海岸國家風景區管理處 |accessdate=2013-01-11}}</ref>。
