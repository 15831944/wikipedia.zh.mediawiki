'''威廉·蘭斯·柏克曼'''（{{lang-en|William Lance Berkman}}, {{bd|1976年|2月10日|}}），綽號「Big Puma」，為前美國職棒大聯盟的外野手及一壘手。曾待過太空人、洋基、紅雀與遊騎兵等隊。生涯入選過6次明星賽，拿過一次世界大賽冠軍。

柏克曼也很常做慈善公益，2012年，富比世將他列入世界上30位最慷慨的名人名單中<ref>{{cite web |author1=BILL POLLOCK |title=伯克曼為福布斯列出了30位最慷慨的名人 |url=https://www.missourinet.com/2012/04/18/berkman-makes-forbes-list-of-30-most-generous-celebrities/ |publisher=Missourinet |accessdate=2019-03-13}}</ref>。他成立了一個團體叫做Berkman's Bunch。他會在每個星期日與50名貧困的小孩見面，並與他們玩遊戲、給禮物。2013年，德州發生[[美國德州化肥廠爆炸事件|美國德州化肥廠爆炸事件]]，柏克曼還捐贈了一台消防車到德州韋斯特。
==職業生涯==
===休士頓太空人===
====1999年–2004年：生涯前期====
1999年7月16日，柏克曼登上大聯盟。由於太空人陣中已經有Jeff Bagwell坐鎮一壘了，因此柏克曼被移往外野。太空人陣中擁有Bagwell與[[克雷格·比吉歐|克雷格·比吉歐]]兩名優秀的老將，再加上其他球員表現也不遜色，使得太空人在當時是一支很有競爭力的隊伍。該年柏克曼出賽34場就被下放3A繼續磨練。

2000年球季，柏克曼重返大聯盟，2成97打擊率，21轟67分打點。隔年打擊率更是上升至3成31，34轟126分打點，還有55支二壘安打。他也入選生涯首次明星賽，在MVP票選第5名。<ref name="br">{{cite web |title=Lance Berkman statistics & history |work=Baseball-Reference.com |url=https://www.baseball-reference.com/players/b/berkmla01.shtml |accessdate=August 9, 2010}}</ref>

2002年球季，他的打擊率稍微下降至2成92，42轟128分打點。他二度入選明星賽，並在MVP票選中排第3名。

2003年，他的打擊率為2成88，25轟93分打點。<ref>[http://thefastertimes.com/sportschat/2010/06/04/lance-berkman-on-instant-replay-we-have-the-technology Lance Berkman wants instant replay. "We have the technology,"] {{Webarchive|url=https://web.archive.org/web/20100608085539/http://thefastertimes.com/sportschat/2010/06/04/lance-berkman-on-instant-replay-we-have-the-technology/ |date=June 8, 2010 }} ''[[The_Faster_Times|The Faster Times]]''</ref>2004年5月，柏克曼7成85的長打率與24分打點讓他首度獲選為國聯單週最佳球員。<ref>{{cite web |last=Footer |first=Alyson |title=Berkman NL Player of the Month |publisher=MLB.com |date=June 2, 2004 |url=http://mlb.mlb.com/news/article.jsp?ymd=20040602&content_id=759307&vkey=news_mlb&fext=.jsp&c_id=null |accessdate=August 9, 2010}}</ref>他三度入選明星賽，並參加該年的全壘打大賽，不過他在決賽不敵[[米格爾·特哈達|米格爾·特哈達]]拿下亞軍。<ref>{{cite web |url=http://www.baseball-almanac.com/asgbox/hrderby.shtml|title=Home Run Derby (2000–2008)|accessdate=September 22, 2008 |work=baseball-almanac.com |publisher= |date= }}</ref>該年他打擊率為3成16，30轟106分打點。
====2005年–2010年====
2005年3月，柏克曼與太空人簽下6年8500萬美元的合約。<ref>{{cite web |url=http://www.mlb.com/news/article.jsp?ymd=20050319&content_id=971432&vkey=spt2005news&fext=.jsp&c_id=hou |title=Astros, Berkman reach six-year deal |last=Footer |first=Alyson |date=March 19, 2005 |publisher=[[MLB.com|MLB.com]] |accessdate=August 1, 2010 }}{{Dead link|date=August 2018 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>該年他擊出24轟並帶有82分打點。

2005年國聯分區賽第4場，柏克曼在第8局擊出滿貫砲。最終勇士與太空人鏖戰18局，太空人靠著再見全壘打淘汰勇士隊。雖然太空人在世界大賽遭到白襪四戰橫掃，不過柏克曼在世界大賽的打擊率高達3成85。

2006年9月13日，柏克曼成為繼[[米奇·曼托|米奇·曼托]]後第一位單季達成40轟的左右開弓打者。<ref>{{cite web |url=https://query.nytimes.com/gst/fullpage.html?res=9C00E6DD1231F937A2575AC0A9609C8B63 |title=BASEBALL: NATIONAL LEAGUE ROUNDUP; Peavy Is Dominant As Padres Rout Reds  |date=September 14, 2006 |work=The New York Times |accessdate=August 1, 2010}}</ref>該年他打擊率為3成15，45轟，而136分打點也成功超越1997年的Jeff Bagwell，成為隊史單季打點王。<ref>{{cite web |url=https://www.baseball-reference.com/teams/HOU/leaders_bat.shtml|title=Houston Astros Batting Leaders |accessdate=September 22, 2008 |work=[[Baseball-Reference.com|Baseball-Reference.com]]|publisher= |date= }}</ref>該年他在國聯MVP票選中排名第三，僅次於[[萊恩·霍華德|萊恩·霍華德]]與[[亞伯特·普荷斯|亞伯特·普荷斯]]。<ref>{{cite web|last=Molony|first=Jim|title=Berkman finishes third in MVP voting|publisher=Major League Baseball|date=November 20, 2006|url=http://mlb.mlb.com/news/article.jsp?ymd=20061120&content_id=1744310&vkey=news_mlb&fext=.jsp&c_id=mlb|accessdate=August 9, 2010}}</ref>

2007年開季，柏克曼陷入低潮，<ref>[http://blogs.chron.com/sportsjustice/archives/2007/05/revo_gives_it_t.html Lance Berkman's long slump. Letterman books a real star.] SportsJustice, by [[Richard_Justice|Richard Justice]]</ref>打擊率僅2成61。不過他在球季末手感漸漸加溫，該年他打擊率2成78，34轟102分打點。

2008年4月，他的打擊率為3成85。他拿下2次國聯單週最佳球員與國聯單月最佳球員。該季他打擊率3成12，29轟106分打點。該年他在國聯MVP票選中排名第五。<ref>{{cite news|first=Matthew |last=Leach |url=http://mlb.mlb.com/news/article.jsp?ymd=20081117&content_id=3681885&vkey=news_mlb&fext=.jsp&c_id=mlb |title=Crowning achievement: Pujols NL MVP | MLB.com: News |publisher=[[MLB.com|MLB.com]] |date=November 17, 2008 |accessdate=August 3, 2010}}</ref>

2009年6月13日，柏克曼擊出生涯第300轟。<ref>{{cite news|last=Ortiz|first=Jose de Jesus|title=Berkman, Tejada put milestones to good use|work=Houston Chronicle|date=June 14, 2009|url=http://www.chron.com/disp/story.mpl/headline/sports/6477782.html|accessdate=August 9, 2010}}</ref>
===紐約洋基===
2010年7月31日，太空人將柏克曼交易至洋基隊，換來Jimmy Paredes與[[馬克·梅蘭康|馬克·梅蘭康]]。<ref>{{cite web|url=http://yankees.lhblogs.com/2010/07/31/yankees-make-berkman-trade-official |title=Yankees make Berkman Deal Official |publisher=Yankees.lhblogs.com |date=July 31, 2010 |accessdate=July 31, 2010}}</ref><ref>{{cite news |first=Brian |last=McTaggart |url=http://pressbox.mlb.com/news/article.jsp?ymd=20100730&content_id=12811944&vkey=news_mlb&fext=.jsp&c_id=mlb |title=Astros finalize Berkman deal with Yankees |publisher=Pressbox.mlb.com |date=July 30, 2010 |accessdate=July 30, 2010 |archive-url=https://www.webcitation.org/6CsnDODy5?url=http://pressbox.mlb.com/news/article.jsp?ymd=20100730&content_id=12811944&vkey=news_mlb&fext=.jsp&c_id=mlb |archive-date=December 13, 2012 |dead-url=yes |df=mdy-all }}</ref>柏克曼在洋基主要擔任指定打擊和替補一壘手。[[2010年美國聯盟冠軍賽|2010年美國聯盟冠軍賽]]，洋基一壘手[[馬克·塔克薛拉|馬克·塔克薛拉]]受傷，柏克曼成為洋基先發一壘手，不過最終洋基2勝4敗遭到遊騎兵隊淘汰。

10月27日，洋基宣布不與柏克曼續約，柏克曼成為自由球員。<ref>[http://yankees.lhblogs.com/2010/10/27/yankees-decline-options-on-berkman-wood-johnson Yankees decline options on Berkman, Wood, Johnson] The LoHud Yankees Blog</ref>
===聖路易紅雀===
====2011年：表現回春與拿下冠軍====
成為自由球員後，柏克曼與紅雀隊簽約。7月5日，柏克曼擊出生涯第350轟。

該年柏克曼在打擊率、全壘打與打點紀錄上都是全隊之冠，回春表現也讓他獲選國聯東山再起獎。

[[2011年世界大賽|2011年世界大賽]]第6場，柏克曼先敲出他在世界大賽的首轟。接著在第9局靠著[[大衛·佛里斯|大衛·佛里斯]]的關鍵安打跑回追平分。<ref>[http://mlb.mlb.com/news/article.jsp?ymd=20111028 Thriving Berkman marvels at Series' drama] ''MLB.com''</ref>第10局更是擊出關鍵安打再度幫助紅雀追平比數。最終紅雀完成逆轉，並拿下世界大賽冠軍。
====2012年====
4月22日，柏克曼因小腿受傷而進入傷兵名單。5月21日，柏克曼又因右膝受傷再次進入傷兵名單。8月3日，柏克曼膝蓋發炎，三進傷兵名單。該年紅雀還是順利闖進季後賽，不過在國聯冠軍戰不敵巨人。
===德州遊騎兵===
