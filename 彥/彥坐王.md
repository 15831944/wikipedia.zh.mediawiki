'''彥坐王'''（{{jpn|j=彦坐王|hg=ひこいますのみこ{{Sfn|彦坐王命（古代氏族）|2010年}}/ひこいますのおう<ref>[https://kotobank.jp/word/%E5%BD%A6%E5%9D%90%E7%8E%8B-1103280 「彦坐王」]『デジタル版 日本人名大辞典+Plus』（朝日新聞社コトバンクより）。</ref>|rm=Hikoimasunomiko/Hikoimasunō}}，{{bd|？||？||catIdx=Hikoimasunomiko}}）是日本古代[[記紀|記紀]]記載的[[皇族|皇族]]。在《[[日本書紀|日本書紀]]》中稱為彥坐王，《[[古事記|古事記]]》稱為'''日子坐王'''，其他文獻則表記為'''彥坐命'''和'''彥今簀命'''。

彥坐王是第9代[[開化天皇|開化天皇]]的第三[[皇子|皇子]]，亦是第12代[[景行天皇|景行天皇]]的曾祖父。留傳下來的記載不多，但是《古事記》有詳細記載其系譜。

== 系譜 ==
按《日本書紀》開化天皇紀記載，彥坐王是第9代開化天皇與{{Link-ja|和珥氏|和珥氏|和珥臣}}遠祖姥津命之妹姥津媛命之子{{Sfn|彦坐王命（古代氏族）|2010年}}。另外按同書[[垂仁天皇|垂仁天皇]]紀記載，[[丹波道主命|丹波道主命]]是其子（異說則指丹波道主命是彥湯產隅王之子）。

按《古事記》記載，彥坐王是開化天皇與丸邇臣（即和珥臣）之祖日子國意祁都命之妹意祁都比賣命之子{{Sfn|彦坐王命（古代氏族）|2010年}}。其子女則如下表所記載。

== 紀録 ==
《日本書紀》並沒有記載其事蹟，只是記載了其子丹波道主命作為{{Link-ja|四道將軍|四道将軍}}的其中一人被派遣前往[[丹波國|丹波國]]。

《古事記》[[崇神天皇|崇神天皇]]段則記載了，他奉天皇之前往旦波國（丹波國），討伐玖賀耳之御笠（{{lang|ja|くがみみのみかさ}}）{{Sfn|彦坐王命（古代氏族）|2010年}}。

== 墓葬 ==
彥坐王的墓位於[[岐阜縣|岐阜縣]][[岐阜市|岐阜市]]岩田西，由[[宮內廳|宮內廳]]管轄，稱為日子坐命墓<ref>宮内省諸陵寮編[{{NDLDC|1237123/8}} 『陵墓要覧』]（1934年、国立国会図書館デジタルコレクション）8コマ。</ref>。其墓緊接於{{Link-ja|伊波乃西神社|伊波乃西神社}}，神社有其相關的傳說。

== 後裔 ==
=== 氏族 ===
按《古事記》記載，彥坐王的後裔有伊勢之品遲部君、伊勢之佐那造、比賣陀君、當麻勾君、佐佐君、日下部連、葛野之別、近淡海蚊野之別、若狹之耳別、三川之穗別、近淡海之安直、長幡部連、吉備品遲君和針間阿宗君等氏族。

按《[[新撰姓氏錄|新撰姓氏錄]]》，彥坐王的後裔如下：
* 左京皇別 治田連 - 四世孫的彥命征北夷有功，獲賜[[近江國|近江國]][[淺井郡|淺井郡]]為領地，子孫賜姓為治田連。
* 左京皇別 輕我孫 - 治田連同氏。四世孫白髮王獲賜姓阿比古，後改稱輕我孫姓。
* 左京皇別 鴨縣主 - 治田連同祖。
* 右京皇別 大私部
* [[山城國|山城國]]皇別 日下部宿禰
* 山城國皇別 輕我孫公 - 治田連同祖。彥今簀命的後裔。
* 山城國皇別 堅井公
* 山城國皇別 別公
* [[大和國|大和國]]皇別 川俁公 - 日下部宿禰同祖。
* [[攝津國|攝津國]]皇別 日下部宿禰
* 攝津國皇別 依羅宿禰 - 日下部宿禰同祖。
* 攝津國皇別 鴨君
* [[河內國|河內國]]皇別 日下部連 - 狹穗彥命的後裔。
* 河內國皇別 川俁公 - 日下部連同祖。
* 河內國皇別 豐階公 - 川俁公同祖。彥坐命之子澤道彦命的後裔。
* [[和泉國|和泉國]]皇別 日下部首 - 日下部宿禰同祖。
* 和泉國皇別 日下部 - 日下部首同祖。

=== 國造 ===
按《古事記》記載，彥坐王的後裔中有三人是[[國造|國造]]，分別是甲斐國造（[[甲斐國|甲斐國]]<ref>『国造制の研究 -史料編・論考編-』（八木書店、2013年）p. 178。</ref>）、本巢國造（[[美濃國|美濃國]][[本巢郡|本巢郡]]<ref>『国造制の研究 -史料編・論考編-』（八木書店、2013年）p. 290。</ref>）、多遲摩國造（但遲麻國造）。

此外，按《[[先代舊事本紀|先代舊事本紀]]》國造本紀記載，彥坐王的後裔如下。
* {{Link-ja|淡海國造|淡海国造}} - 志賀高穴穗（[[成務天皇|成務天皇]]）在位時，任命為彥坐王三世孫大陀牟夜別為國造，範圍為近江國附近<ref>『国造制の研究 -史料編・論考編-』（八木書店、2013年）p. 195。</ref>。
* {{Link-ja|三野前國造|三野前国造}} - 春日率川（開化天皇）在位時，任命彥坐王之子八瓜命為國造，範圍為美濃國西部附近<ref>『国造制の研究 -史料編・論考編-』（八木書店、2013年）p. 196。</ref>。
* 但遲麻國造 - 志賀高穴穗（成務天皇）在位時，任命竹野君同祖的彥坐王五世孫船穗足尼為國造，範圍為[[但馬國|但馬國]][[朝來郡|朝來郡]]和[[養父郡_(兵庫縣)|養父郡]]附近<ref>『国造制の研究 -史料編・論考編-』（八木書店、2013年）p. 223。</ref>。
* 稻葉國造 - 志賀高穴穗（成務天皇）在位時，任命為彥坐王之子彥多都彥命為國造，範圍為[[因幡國|因幡國]][[法美郡|法美郡]]、[[高草郡|高草郡]]和[[八上郡_(鳥取縣)|八上郡]]附近<ref>『国造制の研究 -史料編・論考編-』（八木書店、2013年）p. 226。</ref>。

== 考証 ==
從《古事記》所見，彥坐王與春日、沙本、山代、淡海和旦波等[[豪族|豪族]]均有血緣關係{{Sfn|彦坐王命（古代氏族）|2010年}}。因此，有些說法認為其系譜流傳於和珥氏和{{Link-ja|息長氏|息長氏}}為中心的[[畿內|畿內]]的北部豪族{{Sfn|彦坐命（古代史）|2006年}}，並且指出畿內北部可能曾經存在廣域的聯合政權{{Sfn|彦坐王命（古代氏族）|2010年}}。

垂仁天皇在位期間，由於有狹穗彥王（沙本毘古王）曾經叛亂的傳說，相對於「崇神 - 垂仁」，也可能存在「彥坐王 - 狹穗彥」的皇統{{Sfn|彦坐命（古代史）|2006年}}。
