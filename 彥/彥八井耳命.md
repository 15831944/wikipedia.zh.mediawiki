'''彦八井耳命'''（{{lang|ja|ひこやゐみみ の みこと}}，'''彥八井命'''、日子八井命，生、歿年不詳）為[[日本|日本]]第一代[[神武天皇|神武天皇]]與[[媛蹈鞴五十鈴媛命|媛蹈鞴五十鈴媛命]]所生的[[皇子|皇子]]，其兄弟有[[神八井耳命|神八井耳命]]、神沼河耳命（《[[日本書紀|日本書紀]]》作神渟名川耳命，即[[綏靖天皇|綏靖天皇]]），異母兄弟則為[[手研耳命|當藝志美美命]]。別名有'''彥八井耳命'''、'''彥八井命'''、'''武國龍命'''、'''草部吉見神'''、'''高知保神'''等，為茨田連（{{lang|ja|まむた の むらじ}}）、手島連之祖。

== 概要 ==
《[[日本書紀|日本書紀]]》並未記錄彥八井耳命之名，《[[新撰姓氏錄|新撰姓氏錄]]》和《阿蘇氏系圖》卻提到他是[[神八井耳命|神八井耳命]]的[[兒子|兒子]]，屬於茨田宿禰、茨田連、下家連、江首、尾張部、豐島連的祖先。

=== 古事記之記載 ===
據《[[古事記|古事記]]》上卷之記載，庚申年八月[[神武天皇|神武天皇]]欲立正妃，廣求世族豪門，聽聞[[媛蹈鞴五十鈴媛命|比賣多多良伊須氣余理比賣]]貌美絕色，即迎娶之。神武天皇即位元年，立為皇后，生'''日子八井命'''、[[神八井耳命|神八井耳命]]（{{lang|ja|かむやゐみみ の みこと}}）、神沼河耳命（{{lang|ja|かむぬなかはみみ の みこと}}，即為後來的[[綏靖天皇|綏靖天皇]]）。

神武天皇[[駕崩|駕崩]]後，同父異母的兄長[[手研耳命|當藝志美美命]]為了奪下皇位，娶其後母[[媛蹈鞴五十鈴媛命|媛蹈鞴五十鈴媛命]]為妻，並計畫暗殺日子八井命、神八井耳命、神沼河耳命等三位異母兄弟。但這三位兄弟反倒先下手為強，由神沼河耳命手持兵器殺死手研耳命。後者死亡後，日子八井命（茨田連、手島連之祖）、神八井耳命雖為兄長，但因下手的是弟弟神沼河耳命，所以把皇位讓給他，成為第二代[[綏靖天皇|綏靖天皇]]。

== 信仰 ==
現今主祭彦八井耳命的[[神社|神社]]主要有下列：
* 堤根神社（[[大阪府|大阪府]][[門真市|門真市]]）：與[[菅原道真|菅原道真]]共同被祭祀。
* 野口神社（[[奈良縣|奈良縣]][[御所市|御所市]]）
* 草部吉見神社（[[熊本縣|熊本縣]][[阿蘇郡|阿蘇郡]]高森町）
