'''妄想'''（英語：'''delusion'''）是一种不受相反事实和逻辑推理所纠正但坚信不疑的[[信念|信念]]。它包括错误的[[判断|判断]]与[[逻辑|逻辑]][[推理|推理]]。即使把事实或已经被完全论证的[[理论|理论]]摆在妄想者的面前，也很难动摇他的信念，妄想大都出现在[[精神病|精神病]]状态下，如[[精神分裂症|精神分裂症]]。

== 妄想的种类 ==
妄想可以按内容分为以下几类：
* [[被害妄想型|被害妄想型]]：患者在没有任何事实依据的情况下，坚信有人要加害于他。
* 抑郁妄想：患者过分贬低自己的[[价值|价值]]，此种妄想包含[[负罪妄想|负罪妄想]]、[[疑病妄想|疑病妄想]]、[[贫穷|贫穷]]妄想等。
* [[自大型妄想|自大型妄想]]：患者认为自己聪明过人、认为自己是[[天才|天才]]，创造的成果可以改变[[人类|人类]]的命运，或者在[[名誉|名誉]]、地位、权势上加以妄想，也可以在自我担心的某些问题上加以妄想，夸大此事的危险性。
* [[軀體型妄想|軀體型妄想]]：患者认为自己的身体发生了奇异的变形。
* [[宗教型妄想|宗教型妄想]]：患者认为自己有通靈、或在神秘、宗教、上帝的領域中，有獨特的經驗。
* [[情愛妄想型|情愛妄想型]]：患者在没有任何依据的情况下，认为某人爱上了他。
* 被控制妄想：即异己体验，患者认为自己的身体、[[思维|思维]]、[[意识|意识]]被某种外力控制住了，也就是说患者认为自己在与某人共享身体、思维，例如患者正在思考某个问题时突然就停止了，患者体验到了自己要想的事情被外力（头脑外的思想，而非自己的思想）夺走了（思维剥夺），或患者体验到了外力强行把不属于自己的事情放到了自己的大[[脑|脑]]里（[[思维|思维]]插入）等。
* [[嫉妒妄想型|嫉妒妄想型]]

以上几种妄想多发生在精神分裂症状态下。

== 妄想的特征 ==
妄想的最大特征是妄想内容的核心完全涉及[[自我|自我]]，如“他人要加害于我”、“我是有罪的”、“那件事情会毁掉我的”等。如内容的核心不涉及自我，就算是一些没有依据的且荒谬离奇的想法，在精神病学的评价标准上，也可以不被视为妄想。其次妄想是个人独有的，与所处的文化环境不符，但同时妄想的内容受患者接受的文化教育有关。比如相信存在神灵在某些文化团体中是正常的信念，但认为自己就是神佛多为不正常的。在中国的文化环境中，患者的妄想内容常常涉及鬼怪，而西方国家患者的妄想内容常常和外星人有关。

== 注意 ==
妄想很多时候是目的性的。先入为主的设定一个或多个自认为正确的目标——如婆媳关系中婆婆对自己不好。在日常生活中她就通过他人的言谈举止中的某个字句，某件事，与这一目标建立联系，来证明自己的正确。这时她听到的往往不能称为幻听，是真实的。只是她不自觉的把其余内容过滤掉了，只留下对自己“有用”的材料。通过这样的方法，通过蜘蛛网式的思维，把周围所有人的话语织成一张天衣无缝的大网，而他就是被困在网中央的猎物。严重时，包括他自己最亲近的人也是他不信任的对象。

某些妄想是继发于[[幻觉|幻觉]]之后的，如在[[幻听|幻听]]、幻视等的诱因下所导致的继发性妄想，这类妄想与原发性妄想在病理上是有所不同的，在此与原发妄想需要区别。
