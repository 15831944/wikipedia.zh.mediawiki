'''黏丝盘虫'''（''Trichoplax adhaerens''）是1883年由德國生物學傢[[Franz_Eilhard_Schulze|Franz Eilhard Schulze]] (1840-1921)在[[奥地利|奥地利]]Graz大学的水族馆发现的<ref>F. E. Schulze "''Trichoplax adhaerens'' n. g., n. s.", ''Zoologischer Anzeiger'' (Elsevier, Amsterdam and Jena) 6 (1883), p. 92.</ref><ref name=IZ>{{cite book |author= Barnes, Robert D. |year=1982 |title= Invertebrate Zoology |publisher= Holt-Saunders International |location= Philadelphia, PA|pages= 84–85|isbn= 0-03-056747-5}}</ref>。目前在[[扁盤動物門|扁盤動物門]]中僅確認此一種，一般稱絲盤蟲即指此種。

== 形态 ==
丝盘虫是已知最简单的多细胞动物之一。丝盘虫为扁平薄片状，直径不超过4毫米，体形经常改变，边缘不规则，无对称性，但有恒定的背腹面。虫体由上下两层细胞构成，无[[体腔|体腔]]，无[[消化腔|消化腔]]，无[[神经系统|神经系统]]。

== 运动 ==
[[File:Trichoplax_movement.webm|thumb]]
丝盘虫以体表细胞上的[[鞭毛|鞭毛]]摆动进行运动。因为缺少[[神经系统|神经系统]]，其躯体各部分的运动方向不能受到协调，有可能躯体两部分同时向两个相反方向运动。其运动方向随机，属于[[布朗运动|布朗运动]]。其运动速度受养料浓度影响，养料浓度高的时候，躯体变得扁平，移动变慢，这样有利于扩大吸收面积，有更多时间吸收养料。养料浓度低的时候躯体面积变小，运动加快，以更快的找到新的养料来源。

== 捕食 ==
丝盘虫捕食微小[[原生动物|原生动物]]，以腹面包裹食物，并进行部分[[体外消化|体外消化]]。

== 生殖 ==
丝盘虫的[[生殖|生殖]]方式为[[出芽生殖|出芽]]和[[分裂生殖|分裂]]，虽然也存在[[有性生殖|有性生殖]]，但对其[[胚胎|胚胎]]发育知之甚少。

== 其他 ==
目前已确认的扁盘动物只有黏絲盤蟲（''Trichoplax adhaerens''）一种。1893年意大利人Franzesco Saverio Monticelli描述了另外一种扁盘动物''Treptoplax reptans''，据称是在那不勒斯的水域里找到的。但从1896起''Treptoplax reptans''就被认为是消失了的物种，目前大部分的动物学家都怀疑其存在。
