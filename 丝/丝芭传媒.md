'''上海丝芭文化传媒集团有限公司'''（简称'''丝芭传媒'''，{{lang-en|'''Shanghai Star48 Culture Media Group Co.LTD'''}}）是[[中华人民共和国|中国]]的一个综合性娱乐产业集团，业务涵盖艺人培育、影视、音乐、综艺及互联网粉丝社区等，2010年5月19日成立。

== 简介 ==
丝芭传媒总部位于上海，集团拥有文化经纪许可证、商业演出许可证、影视节目制作许可证等经营资质，同时拥有艺人经纪团队及专业设施，包括大型专属剧场、大型专业级摄影棚、专业级录音棚等，还提供艺人包装和立体化营销渠道的业务<ref name="about">{{cite web | url=https://www.48.cn | title=关于丝芭 | publisher=丝芭传媒官方网站 }}</ref>。

集团业务包括偶像养成和粉丝经济运营、O2O演艺、音乐创作及唱片发行、影视剧网剧及电影制作、综艺节目策划和制作、移动直播及粉丝社区、时尚媒体及网星电商、LBE文化娱乐地产策划及运营等<ref name="mina">{{cite web | url=http://www.snh48.com/html/allnews/zixun/2017/1011/2595.html | title=丝芭传媒集团成功并购米娜 打造亚洲最大规模造星平台 | publisher=SNH48官方网站 | date=2017-07-29 }}</ref>。集团旗下还拥有丝芭影视、米娜时尚杂志及新媒体，以及男子偶像团体业务的火核传媒、米娜艺人工作室、韩国{{link-en|Zanybros|Zanybros}}公司（兼丝芭传媒韩国培训中心）等多家分支子公司及关联企业，共签约约400名艺人，为亚洲地区规模最大的艺人经纪企业之一<ref>{{cite web | url=http://ent.qianlong.com/2018/0629/2666372.shtml | title=SNH48运营方开启偶像艺人全球招募 打造娱乐新旗舰 | publisher=千龙网 | date=2018-06-29 }}</ref>。

== 历史 ==
* 2010年5月19日，上海久尚演艺经纪有限公司成立。

* 2012年4月21日，久尚经纪与[[AKS|AKS]]合作宣佈[[SNH48|SNH48]]成立。

* 2015年1月13日，上海丝芭影视有限公司成立。7月25日，公司正式宣佈更名為「上海絲芭文化傳媒有限公司」<ref>{{cite web | url=http://ent.qq.com/a/20150726/009483.htm#p=1 | title=SNH48第二届总选举落幕 正式进入“百人时代” | publisher=腾讯娱乐 | date=2015-07-26 }}</ref>。

* 2016年4月14日，[[黎瑞刚|黎瑞刚]]名下的[[华人文化产业投资基金|华人文化]]入股丝芭传媒<ref>{{cite web | url=https://36kr.com/p/5045918.html | title=华人文化入股女团 SNH48 运营方丝芭传媒，粉丝经济将如何改变造星方式？ | publisher=36kr | date=2016-04-14 }}</ref><ref>{{cite web | url=http://tech.qq.com/a/20160414/041258.htm | title=大叔看中萝莉 黎瑞刚入股SNH48运营方丝芭传媒 | publisher=腾讯科技 | date=2016-04-14 }}</ref>。4月20日，于北京举行发布会，宣布[[BEJ48|BEJ48]]、[[GNZ48|GNZ48]]成立，并成立三支新SNH48子团体，分别为[[7SENSES|7SENSES]]、[[Color_Girls|Color Girls]]、[[电眼少女队|电眼少女队]]<ref>{{cite web | url=http://ent.people.com.cn/n1/2016/0422/c1012-28297812.html | title=SNH48公布北京广州姐妹团 发力音乐内容原创 | publisher=人民网 | date=2016-04-22 }}</ref>。6月10日，與AKS结束合作关系<ref>{{cite web|url=https://www.akb48.co.jp/news/detailpage/5499387|script-title=ja:AKB48グループにおける海外姉妹グループの運営について|publisher=AKB48官方网站|language=ja|date=2016-06-09}}</ref><ref>{{cite web|url=http://www.snh48.com/html/allnews/activity/2016/0610/1746.html|title= SNH48官方声明 |publisher=SNH48官方网站|date=2016-06-10}}</ref>。7月30日，宣布[[SHY48|SHY48]]成立<ref>{{cite web | url=http://music.yule.sohu.com/20170104/n477707283.shtml | title=SNH48沈阳姐妹团SHY48成立 1月12日正式出道 | publisher=搜狐 | date=2017-01-04 }}</ref>。

* 2017年1月7日，宣布[[N2M|N2M]]成立<ref>{{cite web | url=http://www.dzwww.com/yule/yulezhuanti/mtcbg/201701/t20170109_10901498.htm | title=丝芭传媒打造进化系男团 N2M成员全球招募启动 | publisher=大众网 | date=2017-01-09 }}</ref>。5月9日，丝芭传媒集团完成C轮数亿元融资，由招银国际资本领投，原有股东华人文化产业投资基金及君联资本跟投<ref>{{cite web | url=http://tech.sina.com.cn/i/2017-05-09/doc-ifyeycfp9389058.shtml | title=SNH48母公司获数亿元投资 招银国际领投 | publisher=新浪科技 | date=2017-05-09 }}</ref><ref>{{cite web | url=http://stock.jrj.com.cn/2017/05/09214122452777.shtml | title=“SNH48”运营方丝芭文化传媒完成C轮融资 | publisher=金融界 | date=2017-05-09 }}</ref><ref>{{cite web | url=http://money.163.com/17/0510/11/CK2Q5TS3002580S6.html | title=丝芭文化传媒完成C轮融资 华兴资本担任独家财务顾问 | publisher=网易 | date=2017-05-10 }}</ref>。6月2日，宣布[[CKG48|CKG48]]成立<ref>{{cite web | url=http://ent.sina.com.cn/y/yneidi/2017-06-03/doc-ifyfuzmy1435252.shtml?t=1496459789522 | title=SNH48 GROUP开启总决选 重庆姐妹团招募启动 | publisher=新浪 | date=2017-06-03 }}</ref>。7月29日，宣布完成对上海米娜时尚传媒及新媒体的合并手续，正式将其纳入集团旗下<ref name="mina"/>。

* 2019年1月19日，宣布开启海外练习生计划；[[IDOLS_Ft|IDOLS Ft]]成立；對SNH48 Group北上廣以外分團開啟授權特許經營模式。

== 子公司 ==
