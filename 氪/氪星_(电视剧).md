'''《氪星》'''（{{Lang-en|'''''Krypton'''''}}）是[[大衛·S·高耶|大卫·高耶]]和{{Link-en|达米安·金德勒|Damian Kindler}}为[[Syfy|Syfy电视台]]开发的一部[[科学幻想|科幻]]题材[[电视剧|电视剧]]。这部电视剧发生[[氪星|氪星]]这颗[[行星|行星]]上，故事时间大约在[[超人|凯尔·艾尔{{\}}超人]]出生约200年前。本电视剧的主角是凯尔·艾尔的祖父——西格·艾尔。

《氪星》已于2018年3月21日首播，第一季共有10集。

== 剧情简介 ==
《氪星》主要讲述了[[超人|超人]]的祖父西格·艾尔的故事，他所在的{{Link-en|艾尔家族|Superman dynasty}}在故事的开头因为“叛国罪”而被剥夺了荣誉和身份。西格·艾尔通过战斗赎回了属于自己家族的荣光，并从混沌手中保护了自己所心爱的世界。<ref name="PilotOrder">{{cite news|last1=Sirani|first1=Jordan|title=KRYPTON: SYFY ORDERS PILOT FOR STORY ABOUT SUPERMAN'S GRANDFATHER|url=http://www.ign.com/articles/2016/05/09/krypton-syfy-orders-pilot-for-story-about-supermans-grandfather|accessdate=2016-05-09|publisher=[[IGN|IGN]]|date=2016-05-09|deadurl=no|archiveurl=http://archive.wikiwix.com/cache/20160509043627/http://www.ign.com/articles/2016/05/09/krypton-syfy-orders-pilot-for-story-about-supermans-grandfather|archivedate=2016-05-09}}</ref>

Goldberg|伊恩·戈德堡}}参与[[编剧|编剧]]和[[执行制作|执行制作]]的工作。<ref name="TheHollywoodReporterKryptonSeries">{{cite web|url=http://www.hollywoodreporter.com/live-feed/syfy-david-goyer-developing-superman-755060|title=Syfy, David Goyer Developing Superman Origin Story 'Krypton'|last=Goldberg|first=Lesley|publisher=The Hollywood Reporter|date=2014-12-08|accessdate=2014-12-08|deadurl=no|archiveurl=https://web.archive.org/web/20141223081805/http://www.hollywoodreporter.com/live-feed/syfy-david-goyer-developing-superman-755060|archivedate=2014-12-23}}</ref> 2016年5月，根据{{Link-en|TVLine|TVLine}}的报道，[[Syfy|Syfy]]已经确认为该剧订购了官方版的试播集，并且{{Link-en|达米安·金德勒|Damian Kindler}}将参与到制作团队中，与大卫·高耶一同担任执行制片的工作。<ref>{{cite news|url=http://tvline.com/2016/04/21/krypton-series-prequel-syfy-superman-pilot/|title=Krypton Prequel From David S. == 反响评论 ==
=== 相关评论 ===
基于31人的评分，《氪星》在[[烂番茄|烂番茄]]网站的新鲜度为58%；平均得分为6.33/10。烂番茄上目前共识最高的评论点评道，“氪星原本的怪异感消失在剧中愚蠢的叙事中，没有达到一个有前途的开头。”<ref>{{cite web |url = https://www.rottentomatoes.com/tv/krypton/s01 |title = Krypton: Season 1 (2018) |website = Rotten Tomatoes |publisher = Fandango Media |accessdate = 2018-03-22 |deadurl = no |archiveurl = https://web.archive.org/web/20180329112351/https://www.rottentomatoes.com/tv/krypton/s01 |archivedate = 2018-03-29 }}</ref> 在[[Metacritic|Metacritic]]网站上，基于14位评论家的打分，本剧的加权分数为53/100，总体评价为“毁誉参半”。<ref>{{cite web |url=http://www.metacritic.com/tv/krypton |title=Krypton Reviews |work=Metacritic |publisher=CBS Interactive |accessdate=2018-03-22 |deadurl=no |archiveurl=https://web.archive.org/web/20180322130144/http://www.metacritic.com/tv/krypton |archivedate=2018-03-22 }}</ref> 而在[[豆瓣|豆瓣]]网上，基于191人的评价，《氪星》的得分为6.7/10。获得较高认同的评论说道，“剧情是预计之内的中规中矩，原预告所有内容都包括在试播集内，虽说是syfy最大投资的剧集，但也还达不到一线电影的效果，……，男女主的人选都不行，演不出使命感与厚重感。……”<ref>{{cite web|title=《氪星》|url=https://movie.douban.com/subject/26279828/|website=[[豆瓣|豆瓣]]|publisher=北京豆网科技有限公司|accessdate=2018-03-29|deadurl=no|archiveurl=https://web.archive.org/web/20180329112347/https://movie.douban.com/subject/26279828/|archivedate=2018-03-29}}</ref>
