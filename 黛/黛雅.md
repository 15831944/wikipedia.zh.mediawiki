'''格蕾丝·马丁·坦登'''（{{lang-en|Grace Martine Tandon}}，{{bd|1998年|10月24日}}<ref>{{Cite web|url=http://www.billboard.com/artist/6683644/daya/biography|title=Daya - Biography {{!}} Billboard|website=www.billboard.com|access-date=2017-06-12}}</ref>），艺名'''黛雅'''（{{lang-en|Daya}}），是来自美国[[宾夕法尼亚州|宾夕法尼亚州]][[匹兹堡|匹兹堡]]的唱作歌手<ref name="106.9 The Q">{{cite web|url = http://www.1069theq.com/artist/daya/|title = Daya 106.9 The Q|access-date = October 5, 2015|work = 106.9 The Q|archive-url = https://web.archive.org/web/20151007045941/http://www.1069theq.com/artist/daya/|archive-date = 2015-10-07|dead-url = yes}}</ref><ref>{{cite web|url = http://pittsburgh.cbslocal.com/video/3267445-daya-hide-away/|title = Daya – Hide Away – CBS Pittsburgh|access-date = October 5, 2015|work = CBS Pittsburgh}}</ref>，Artbeatz、Z Entertainment和{{le|相对论娱乐发行有限公司|RED Distribution}}旗下艺人。2015年9月4日，与其个人同名的{{le|黛雅 (EP)|Daya (EP)|首张}}[[迷你专辑|迷你专辑]]发行，其中单曲《{{le|Hide Away (黛雅歌曲)|Hide Away (Daya song)|Hide Away}}》在[[公告牌百强单曲榜|公告牌百强单曲榜]]的最高排名为第23位<ref>{{cite web|url = http://zentertainmentla.com/managment.html|title = Z Entertainment Management|access-date = October 5, 2015|work = Z Entertainment Management|deadurl = yes|archiveurl = https://web.archive.org/web/20151006034126/http://zentertainmentla.com/managment.html|archivedate = 2015年10月6日|df = }}</ref><ref name="iTunes 1">{{cite web|url=https://itunes.apple.com/us/album/daya-ep/id1028772922 |title=iTunes (U.S.) – Music – Daya – Daya – EP |access-date=October 5, 2015 |date=September 4, 2015 |work=iTunes (U.S.) |deadurl=yes |archiveurl=https://web.archive.org/web/20160901145450/https://itunes.apple.com/us/album/daya-ep/id1028772922 |archivedate=September 1, 2016}}</ref>。2016年10月16日，个人首张录音室专辑《{{le|美麗甜姐兒 (專輯)|Sit Still, Look Pretty (album)|美麗甜姐兒}}》发行。

== 早年 ==
黛雅生于[[匹兹堡|匹兹堡]]，在宾夕法尼亚州{{le|黎巴嫩山|Mount Lebanon, Pennsylvania}}长大。先祖父是从[[印度|印度]][[旁遮普邦|旁遮普邦]]移民到美国的[[旁遮普人|旁遮普]][[印度裔美国人|印度裔美国人]]，她的四姐妹叫蕾切尔（Rachel）、玛丽安娜（Mariana）、席丽雅（Celia）和艾芙莉（Avery）。艺名黛雅（Daya）出自[[梵语|梵语]]，意为“同情”或“善良”<ref>{{cite web|url=http://www.post-gazette.com/ae/music/2015/10/29/Music-preview-Mt-Lebanon-singer-Daya-has-shot-at-pop-fame/stories/201510290006|title= Music preview: Mt. Lebanon singer Daya has shot at pop fame|last=Mervis|first=Scott|website=post-gazette.com|date=October 29, 2015|accessdate=March 21, 2016}}</ref>。小学就读于圣伯纳德学校（St. Bernard School），高中毕业于<ref name="South Hills Living">{{cite web|url = http://www.southhillsliving.net/article/20160920/PUBLICATIONS07/160929992|title = Mt. Lebanon's Daya is a rising pop star who remains grounded|access-date = September 21, 2016|date = September 20, 2016|author = Kelly Washere|work = South Hills Living}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>黎巴嫩山高中<ref name="Lebo Mag">{{cite web|url = http://lebomag.com/23527/a-lebo-pop-princess/|title = A Lebo Pop Princess|access-date = October 5, 2015|date = June 4, 2015|work = Lebo Mag}}</ref>。黛雅自3岁起学习钢琴，7岁转而学习爵士钢琴。到那时，她已经学会吉他、尤克里里、萨克斯和长笛<ref name="Lebo Mag" />。

11岁，达雅就读于由克里斯蒂娜·齐鲁姆博洛（Christina Chirumbolo）<ref>{{cite web|title = Accelerando :|url = http://www.accelerandopittsburgh.com|website = www.accelerandopittsburgh.com|accessdate = January 7, 2016}}</ref>在匹兹堡开办的一所音乐专修学校，期间结识曾到学校开讲座、齐鲁姆博洛的同事、词曲作者、制作人吉诺·巴雷塔（{{lang|en|Gino Barletta}}）<ref name="106.9 The Q" />。齐鲁姆博洛和巴雷塔成立了音乐训练营INSIDE ACCESS，两人和黛雅合作，最终于2015年2月邀请她到洛杉矶录制素材。

== 音乐生涯 ==
黛雅的职业生涯开始于父母陪她到洛杉矶和INSIDE ACCESS的创办人克里斯蒂娜·齐鲁姆博洛与吉诺·巴雷塔、{{le|派拉蒙唱片|Paramount Records}}的{{le|布雷特·麦克劳林|Brett McLaughlin}}、布里顿·纽比尔（Britten Newbill）和吵闹城堡三世（Noisecastle III）共事。一次创作会议上，她的处女作《Hide Away》被写下并录制，巴雷塔随后将黛雅介绍给Z Entertainment的史蒂夫·扎普（Steve Zap）。黛雅当时还是位青少女，她表示自己一时半会还没有考虑到将来会发生的事情<ref name="South Hills Living" />，第二天便回去高中上学。扎普贵为电台营销资深人士，非常喜欢这首歌，有意推广这位歌手，于是和巴雷塔成立独立厂牌Artbeatz。

2015年4月22日，黛雅发行歌曲《Hide Away》<ref name="YouTube">{{cite web|url = https://www.youtube.com/watch?v=Giv7Jg1CdoM|title = Daya's 'Hide Away' on iTunes! www.theofficialdaya.com - YouTube|access-date = September 21, 2016|date = April 22, 2015|work = YouTube}}</ref>。歌曲在网上获得如潮好评，获得{{le|泰勒·奥克利|Tyler Oakley}}和[[佩雷斯·希尔顿|佩雷斯·希尔顿]]等知名博客主李剑。希尔顿表示“达雅的声音非常特别”<ref>{{cite web|url = http://perezhilton.com/2015-06-05-daya-hide-away#.Vfo5jfZVhBc|title = Listen To This: Fly As A Mother – Perez Hilton|access-date = October 5, 2015|date = June 5, 2015|work = Perez Hilton}}</ref>。《[[公告牌_(杂志)|公告牌]]》的杰森·利普舒茨（Jason Lipshutz）也将这位歌手放到他们的官方网站上，标榜歌曲是“非常棒的处女作”<ref>{{cite web|url = http://www.billboard.com/articles/columns/pop-shop/6663792/summer-pop-playlist-2015-beck-marina-santigold-health|title = Pop Playlist Summer 2015: 15 Awesome Songs You Might Have Missed – Billboard|access-date = October 5, 2015|work = Billboard}}</ref>。

2015年8月21日，黛雅以{{le|埃尔维斯·杜兰|Elvis Duran}}{{le|埃尔维斯·杜兰月度艺人|Elvis Duran's Artist of the Month|月度艺人}}的身份在《[[今天_(NBC)|卡希·李和荷达今日秀]]》上首次亮相电视荧屏，她演唱了歌曲《Hide Away》<ref>{{cite web|url = https://www.youtube.com/watch?v=vQOV57Untl4|title = Daya Performing "Hide Away" on The Today Show|access-date = October 5, 2015|date = September 11, 2015|work = YouTube}}</ref>。2016年6月28日，她重返《今日秀》，为夏季系列演唱会宣传<ref>{{cite web|url = https://www.youtube.com/watch?v=Rvs7JgQZ-Yw|title = Daya - Sit Still, Look Pretty - Live on Today Show|access-date = September 28, 2016|date = June 28, 2016|work = YouTube}}</ref>。

一鸣惊人后，黛雅于2015年9月4日发行同名迷你专辑《{{le|黛雅 (EP)|Daya (EP)|黛雅}}》，包含《Hide Away》在内的六首歌曲<ref name="iTunes 1" />。专辑初期在《公告牌》全天候首播，在[[公告牌二百强专辑榜|公告牌二百强专辑榜]]位列第161位，将《Hide Away》在公告牌流行歌曲榜的排名往前推到第40位<ref>{{cite web|url = http://www.billboard.com/articles/columns/pop-shop/6685792/daya-ep-premiere-hide-away-stream|title = Daya EP Premiere: Stream – Billboard|access-date = October 5, 2015|date = September 3, 2015|work = Billboard}}</ref><ref name="Billboard 200">{{cite web|url = http://www.billboard.com/artist/6683644/daya/chart?f=305|title = Billboard – Daya – Chart History – Billboard 200|access-date = October 5, 2015|date = September 26, 2015|work = Billboard}}</ref><ref>{{cite web|url = http://headlineplanet.com/home/2015/09/09/dayas-hide-away-reaches-bubbling-under-hot-100-chart/|title = Daya's "Hide Away" Debut On Billboard|access-date = October 5, 2015|date = September 9, 2015|work = Headline Planet}}</ref>。

2015年10月10日，同名迷你專輯实体版在[[目标百货|目标百货]]上架<ref>{{cite web|url = http://www.target.com/p/daya-daya/-/A-50246488#prodSlot=medium_1_1&term=daya|title = Daya – Daya (Only at Target)|access-date = November 9, 2015|date = October 30, 2015|work = Target}}</ref>。

2016年，黛雅为{{le|流行说唱|Pop-rap}}二人组{{le|杰克二人帮|Jack & Jack}}美国巡演开场<ref>{{cite web|title=Q92|url = https://www.facebook.com/Q92radio/posts/10153822783482932|access-date = April 6, 2016| date= November 13, 2015|work = facebook}}</ref>。2016年2月，她客串[[烟鬼组合|烟鬼组合]]单曲《[[别让我失望|别让我失望]]》，该曲在百强榜最高排至第3位，成为她第二首跻身40强、第一首跻身10强的作品。EP的第二首单曲《{{le|美丽甜姐儿|Sit Still, Look Pretty}}》随后发行，于第100名空降百强榜，最高排名为第28位，成为她个人第三首40强作品。

黛雅受邀在2016年[[白宫|白宫]]复活节滚彩蛋活动上表演，她和家人们会见了时任总统[[贝拉克·奥巴马|贝拉克·奥巴马]]及[[美国第一夫人|第一夫人]][[米歇尔·奥巴马|米歇尔·奥巴马]]<ref name="South Hills Living" />。

2016年11月15日，黛雅发行《{{le|美丽甜姐儿 (专辑)|Sit Still, Look Pretty (album)|美丽甜姐儿}}》的第三首单曲《{{le|Words|Words (Daya song)}}》。

2016年12月6日，《[[别让我失望|别让我失望]]》获[[第59届格莱美奖|第59届格莱美奖]]提名<ref>{{Cite news|url=http://www.billboard.com/articles/news/grammys/7604105/daya-grammy-nomination-reaction-interview-dont-let-me-down|title=Daya Reacts to First Grammy Nomination: 'This is Insane'|newspaper=Billboard|access-date=2017-02-15}}</ref>。2017年2月12日，歌曲获得[[格莱美奖|格莱美奖]]{{le|格莱美奖最佳舞曲录制奖|Grammy Award for Best Dance Recording|最佳舞曲录制奖}}<ref name= spin/>。2017年11月11日，达雅透过[[新視鏡唱片|新視鏡唱片]]发行单曲《New》<ref name="AllAccess">{{cite web|archive-url = https://web.archive.org/web/20171011170149/https://www.allaccess.com/top40-mainstream/future-releases|url = https://www.allaccess.com/top40-mainstream/future-releases|title = Top 40/M Future Releases - Mainstream Hit Songs Being Released and Their Release Dates|archive-date = October 11, 2017|access-date = October 11, 2017|date = October 17, 2017|publisher = AllAccess Music Group}}</ref>。
