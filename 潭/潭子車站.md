'''潭子車站'''位於[[臺灣|臺灣]][[臺中市|臺中市]][[潭子區|潭子區]]中山路與潭興路口，為[[臺灣鐵路管理局|臺灣鐵路管理局]]（臺鐵）[[臺中線|臺中線]]的[[鐵路車站|鐵路車站]]。除了臺中線外，潭子在過去也曾是已廢線的臺鐵貨運支線[[神岡線|神岡線]]之起點站。

== 車站構造 ==
配合[[臺中都會區鐵路高架捷運化計畫|臺鐵台中高架捷運化]]政策，已改建為兩座月台、四線軌道的高架車站。

== 歷史 ==
<gallery>
File:TRA TanZih Station.jpg|1969年建造之第二代站房（潭子街側）
File:TRA TanZih West Station.jpg|2007年增設的西出口站房（中山路側）

*1905年5月15日：設「潭子墘駅」。
*1920年10月1日：改為「潭子車站」。
*1969年11月：第二代站體啟用。
*1987年配合豐原=彰化間雙軌化工程，站內軌道重配置，增建第二月台，及第一、二月台水泥雨篷。
*1999年7月1日：神岡支線因貨運業務量縮減，加上功能逐漸由公路取代，宣告停駛，2002年拆除完畢，並改建為[[潭雅神自行車道|潭雅神自行車道]]。
*2004年年中：原日治木造站長宿舍群拆除，改建為停車場，可供15輛汽車，300輛機車停放。
*2005年10月10日：首度增停復興號，北上132車次，南下109車次。
*2006年2月19日：首度增停莒光號。北上南下共4個車次。
*2006年5月31日：首度增停自強號，北上2056車次，南下2051車次，為豐原=高雄=台東的南迴線列車。
*2007年8月1日：西出口（或稱西站）安檢通過，正式啟用，首開由地方政府出資為台鐵興建站房的先例，站房由縣長黃仲生及鄉長簡文祥題字落款。
*2009年底：豐原=大慶間高架化捷運化工程開工，豐原=太原興建臨時軌，設置臨時月台及臨時車站。
*2012年1月3日：路線切換至臨時軌使用，啟用臨時月台及改至臨時車站營業。東站停用，並拆除站體，進行高架化站區工程。
*2013年9月25日：應因新購普悠瑪號列車首批7列正式加入東幹線營運，及配合台中鐵路高架化施工而調整時刻表，取消了2班自強號列車，南下101車次、北上156車次。
*2014年3月25日：高架化站區橋樑工程及月台基礎工程完工，目前進入站體鋼樑結構搭建工程。
*2015年6月30日：啟用多卡通刷卡機。
*2016年10月15日：平面軌站場營運最後一天<!--，有許多鐵道迷前來攝影做文史記錄，當中也有人到潭子街平交道道班工房採訪最後的道班工作，潭子街平交道是台鐵山線最後設有兩組道班工房的平交道，也是台鐵沿線中路路幅最大的平交道之一，均走入歷史-->。
*2016年10月16日：高架化完工切換通車，<!--各級列車改行駛於高架軌道，-->設有兩座島式月台/四股道，<!--同時也重新恢復因高架化施工而停止的列車待避作業，-->是第一階段切換高架的5座車站中（豐原、潭子、太原、台中、大慶），高架工程全數完工的車站。同日上午8點起發售「台中鐵路高架化第一階段通車紀念月台票」，上下午月台票成1套合售，計500套<!--，吸引民眾在前一晚就有2、30人前來排隊佔位子，到了上午8點開賣，隊伍已大排長龍。最多只排到第二輪，於9點即已售光-->。另有刻印「高架化通車紀念章」供民眾蓋章，是潭子站首次在窗口發售紀念月台票。
*2016年10月20日：配合台中鐵路高架化第一階段完工通車，進行大改點，雙向停靠區間快車計1列（1543次七堵=二水），區間車計100列，並取消停靠對號列車。
*2017年3月30日：平面臨時站房，月台及軌道全數拆除完畢。

== 營運狀況 ==
[[File:潭子車站出入口.jpg|300px]]
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]，僅停靠所有[[臺鐵區間車|區間車]]和[[臺鐵區間車|區間快車]]。
*站內設有「鐵路之旅 - 小站巡禮紀念章」及「鐵路高架化紀念章」之戳章，可供旅客蓋戳留念。
*近年來車站西側[[台3線|台3線]]之發展已超過舊市街，故於2007年8月1日增設西站出口，目前東站因應高架化工程拆除，僅餘一地下道(部分為原先東站通往舊月台的地下道)連通西站。高架化通車後，待平面軌拆除前，仍保留地下道做為西出口通往位於潭子街的高架新站。而潭興路於平面軌拆除後，在2017年1月25日貫通。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]、[[icash|icash 2.0]]及[[HappyCash|HappyCash]]付費。
