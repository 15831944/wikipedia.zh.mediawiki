《未发现的中国》（{{lang-en|'''Untamed China'''}}），2011年在[[英国|英国]]上映的关于[[中华人民共和国|中华人民共和国]]的[[动物|动物]]纪录片，由英国爬行动物学家[[奈杰尔·马文|奈杰尔·马文]]担任主演，摄影实纳森·雷德勒（Nathan Ridler）、录音师马太·莱特（Matthew Wright）和灯光师阿列克斯·明顿（Alex Minton）。<ref>{{cite web |url=http://www.chinanews.com/cul/2011/09-15/3330805.shtml |title=奈吉尔·马文：在中国的奇幻冒险 |accessdate=2011年9月15日  |publisher=中新网 |work= |language=zh-hans}}</ref>

== 影集 ==
*高山之颂（[[大理|大理]]）
*神秘西域（[[新疆|新疆]]）
*热带宝藏（[[西双版纳|西双版纳]]）
*天府之国（[[四川|四川]]）
*壮美草原（[[内蒙古|内蒙古]]）
*高原瑰宝（[[青藏高原|青藏高原]]）

== 花絮 ==
[[云南卫视|云南卫视]]邀请奈杰尔·马文为云南省做“打造绿色传媒”品牌宣传，特意制作了“马文探奇云南”和“马文穿越恐龙谷”两档节目。<ref>{{cite web |url=http://ent.sina.com.cn/v/u/2010-11-19/14363151513.shtml |title=云南卫视和奈杰尔-马文打造马文探奇动物王国 |accessdate=2010年11月19日 |publisher=新浪网 |work= |language=zh-hans}}</ref>
