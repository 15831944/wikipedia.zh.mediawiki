== 品牌起源 == 
2003年，Terrans Force 的創辦人曹斌剛從[[蘇州大學|蘇州大學]]畢業，他熱愛電腦遊戲，有感當時的電腦配備不足應付日新月異的遊戲系統要求，決心要打造出高性能的電競電腦。曹斌於2005年透過與[[台灣|台灣]]電腦公司的合作，完成了第一批親自設計的筆記型電競電腦。2006年，適逢[[戴爾|戴爾]]收購[[Alienware|Alienware]]的品牌，曹斌把[[Alienware|Alienware]]的產能整合到旗下，並建立高性能電腦品牌「Terrans Force」。 <ref>[http://www.szdushi.com.cn/news/201407/201481990.shtml 曹斌：曾經別人眼裡的“瘋狂” 終將證明是高瞻遠矚]</ref><ref>[http://news.2500sz.com/news/szxw/2017-3/30_3091935.shtml 蘇州80後小夥曹斌 築起中國電競F1平臺]</ref>

== 里程碑 ==
* 2005年：品牌「Terrans Force」正式成立。
* 2006年：發布{{fact|業界首款}}Intel Xeon伺服器處理器[[手提電腦|手提電腦]]。
* 2010年：發布{{fact|業界首款}}電子快門式3D螢幕[[手提電腦|手提電腦]]。
* 2010年：發布價值10萬元人民幣的高性能[[手提電腦|手提電腦]]。
* 2012年：發佈價值21萬元人民幣的[[手提電腦|手提電腦]]。<ref>[http://nb.hqbpc.com/html/2013/11/28076.html 售價21萬的外星人 未來人類P57筆記型電腦]</ref>
* 2013年：{{fact|獲中國政府頒發六項著作權證書。}}
* 2014年：舉辨NTF電子競技大賽<ref>[http://www.szdushi.com.cn/news/201510/2015163776.shtml 蘇州本土的電競賽事 NTF電子競技嘉年華30日開戰]</ref><ref>[http://news.163.com/15/1029/06/B72T2JUD00014AED.html 2015 NTF電子競技嘉年華明日開幕]</ref>。
* 2015年：舉辦NTF女神杯電子競技嘉年華。
* 2016年：成立未來人類魔鬼魚女子戰隊<ref>[http://digi.163.com/16/0530/14/BOAPO21Q001628BV.html TRG戰隊與DR魔鬼魚女子隊決戰紫禁冠軍之夜]</ref>。
* 2016年：品牌進入香港市場。
