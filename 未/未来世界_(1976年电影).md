《'''未来世界'''》（{{lang-en|'''''Futureworld'''''}}）是一部1976年美国科幻[[惊悚|惊悚]]电影。本片是1973年电影《[[西部世界_(1973年电影)|西部世界]]》的续作

本片由[[上海电影译制厂|上海电影译制厂]]译制后在[[上海电影译制厂作品列表_(1970年代)#1979年|1979年的“美国电影周”]]时期在华公映，是改革开放后第一部引进入华的美国科幻片。<ref name="vice">{{Cite web |url = http://www.vice.cn/read/west-world-and-future-world-are-the-earliest-scifi-movies-that-come-to-china |title = 也许你的父母比你还要早就看过《西部世界》和《未来世界》了  |author = 张贼贼 |date = 2016-11-02 |publisher = 异视异色 |language = zh-cn |accessdate = 2017-09-10 }}</ref>而前作《西部世界》则于1989年在华公映。
