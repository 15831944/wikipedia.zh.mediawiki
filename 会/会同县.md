'''会同县'''位于[[中國|中國]][[湖南省|湖南省]]西南边缘、[[怀化|怀化市]]南部，为怀化市辖[[县|县]]。是共和国第一大将[[粟裕|粟裕]]的故乡，也是驰名全国的“广木之乡”。辖域面积2,245平方公里；[[国内生产总值|国内生产总值]]161,924万元（2004年）；人口345,482人（2004）。[[县治|县治]][[林城镇_(会同县)|林城镇]]。

== 历史 ==
春秋战国时，属[[楚国|楚国]]地；两汉、三国为镡成县地，属[[武陵郡|武陵郡]]；东晋为舞阳县地，梁为龙檦县地，唐[[贞观|贞观]]八年，析龙标县置朗溪县，属巫州；[[北宋|北宋]][[崇宁|崇宁]]元年置'''三江县'''，次年（公元1103年）改名'''会同县'''，属[[靖州|靖州]]。<ref>[http://www.huitong.gov.cn/Article/intro/ 基本概况] {{webarchive|url=https://web.archive.org/web/20110124002029/http://www.huitong.gov.cn/Article/intro/ |date=2011-01-24 }}</ref>县名取自《周礼》：“时见曰会，殷见曰同。”

== 地理 ==
会同地处[[雪峰山脉|雪峰山脉]]和[[云贵高原|云贵高原]]边缘。相邻[[县级行政区|县级行政区]]有，西邻[[贵州|贵州]][[天柱县|天柱县]]，南界湖南[[靖州苗族侗族自治县|靖州苗族侗族自治县]]，东与[[绥宁县|绥宁县]]接壤，北部和[[洪江市|洪江市]]相连。

[[渠水|渠水]]（位于县境西部）和[[巫水|巫水]]（位于县境东部）两大水系由南向北纵贯全境。分别在洪江市托口镇和[[洪江区|洪江区]]汇入[[沅江|沅江]]。

主要山峰有[[雪峰山|雪峰山]]1477.4米，[[金龙山|金龙山]]1075米，[[八仙山|八仙山]]744米，[[山盆坡|山盆坡]]，海拔935米。

== 经济 ==
工业以森工、建材、机械、电力、化工、采矿和食品加工为支柱。经过50年的建设，会同的工业由解放初期从事手工业生产的几个行业部门，发展到拥有机械、电力、森工、建材、酿酒、化工、皮革、服装、食品、矿山开采等工业门类。商业贸易也日趋繁荣。目前，全县拥有工商企业6200多家（其中乡以上工业企业 132家）、商业网点2800多个、规模较大的集贸市场30多个。

== 风景名胜 ==
会同地处山林地区，山美水秀。境内拥有丰富的旅游资源，有“华夏同始祖，天下共连山”的[[炎帝|炎帝]]故里；有粟裕故居和纪念馆；有被誉为“古民居建筑活化石、古村落发展建筑史书、耕读文化完美典范、江南第一古村”的国家重点文物保护单位高椅古村；有堪称“中国南方生物基因库”、“小张家界”的鹰嘴界国家级自然保护区等等。
