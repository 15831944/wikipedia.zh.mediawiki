== 節目內容 ==
節目針對近期新聞時事進行討論，並邀請相關來賓進行對談，但節目走「犀利風格」，並強調「發掘真相、不饒過任何人」<ref>{{Cite web|title=周玉蔻主持電視政論 犀利風格挖出「辣新聞」|url=https://tw.news.yahoo.com/video/%E5%91%A8%E7%8E%89%E8%94%BB%E4%B8%BB%E6%8C%81%E9%9B%BB%E8%A6%96%E6%94%BF%E8%AB%96-%E7%8A%80%E5%88%A9%E9%A2%A8%E6%A0%BC%E6%8C%96%E5%87%BA-%E8%BE%A3%E6%96%B0%E8%81%9E-115357106.html|work=tw.news.yahoo.com|accessdate=2019-03-16|language=zh-Hant-TW|author=|date=2019-03-13|publisher=民視新聞}}</ref>。

主持人周玉蔻強調，自己的立場非[[中國統一|統]]非[[台灣獨立運動|獨]]，是「[[中華民國在台灣|中華民國台灣派]]」，甚至曾在《[[放言|放言]]》投書批評民視董事長[[郭倍宏|郭倍宏]]。但郭倍宏表明尊重她的媒體人專業，並給予多元空間<ref>{{Cite web|title=周玉蔻辣新聞 民視犀利登場|url=http://ent.ltn.com.tw/news/paper/1273293|work=ent.ltn.com.tw|accessdate=2019-03-16|last=|author=賀靜賢|date=2019-03-12|publisher=自由時報}}</ref>，令他相當敬佩。

== 主持人 ==

* [[周玉蔻|周玉蔻]]：資深媒體人。

== 事件 ==
2019年3月14日，於節目開播前夕，民視最大股東「民間投資股份有限公司」召開董事會，其一董事提出民投董事長郭倍宏不適任，並通過將其撤換。有一說是因郭倍宏上任後大開政論節目，導致民視「轉盈為虧」。《辣新聞》也是郭倍宏力邀周玉蔻所誕生的節目。面對節目可能遭到波及，製作人強調沒有收到節目更動的消息，第一集會如期開播，並以「當天錄製當天播出」為走向<ref>{{Cite web|title=民視變天！周玉蔻新節目存亡堪憂 製作人強調「一定播」|url=http://ent.ltn.com.tw/news/breakingnews/2727874|work=ent.ltn.com.tw|accessdate=2019-03-16|last=|author=洪素津|date=2019-03-15|publisher=自由時報}}</ref>。
