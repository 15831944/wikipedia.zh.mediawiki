'''仁伯爵綜合醫院'''（俗稱'''山頂醫院'''；{{lang-pt|'''Centro Hospitalar Conde de São Januário'''}}，[[縮寫|縮寫]]：'''{{lang|pt|CHCSJ}}'''）是[[澳門|澳門]]三家[[醫院|醫院]]之一，也是澳門現時唯一的公立地區綜合醫院。醫院位於[[東望洋山|松山]][[若憲馬路|若憲馬路]]，佔地30,300平方米，建築面積達67,535平方米，由4幢[[建築物|建築物]]相連而成，設置[[直升機|直升機]][[停機坪|停機坪]]。仁伯爵綜合醫院始建於1874年1月6日，於1989年11月29日經擴建後啟用。

== 名稱的問題 ==
仁伯爵綜合醫院原本是一所軍人醫院，有病床100張。啟用時正適逢[[聖若憲|聖若憲]][[伯爵|伯爵]]（Conde de São Januário）管治澳門，因此當局隨即就醫院命名的巧合作出了澄清，強調醫院引用的是[[聖人|聖人]]的名義而非[[澳門總督|總督]]的名字。隨後便將1857年在[[崗頂前地|崗頂]]興建的舊軍人醫院的財產和病人轉移到這所新醫院來。

另外所位於的若憲馬路也是同一個原因，然而中文名稱為何稱作為「仁伯爵」就不得而知了。
== 歷史 ==
1874年1月6日，'''仁伯爵綜合醫院'''正式啟用。

醫院先後於1958年及1980年代重建，現時使用的大樓，除了急診大樓外，均為1980年代時重建而成。

== 現時狀況 ==
醫院是由[[澳門衛生局|澳門衛生局]]管理及其唯一轄下醫院，現任院長是郭昌宇。2004年佔地30,300平方米，建築面積67,535平方米。由四幢建築物組成 （傳染病大樓在建），並設[[直昇機|直昇機]]停機坪。<ref name="">澳門特別行政區衛生局仁伯爵綜合醫院，《仁伯爵綜合醫院130周年紀念特刊》，2004年1月6日。</ref>有病床471張，分22個科室，53個分科。門診方面有53個分科，至2001年底有[[醫生|醫生]]210人和[[護士|護士]]494人。另外有急診部、兩個手術部門、深切治療部及其他科室。住院流動率方面，符合此類醫院單位的醫療數值，近年病床佔用率增長達正常水平(71.52%)，平均住院日數為9.2天。

[[澳門理工學院|澳門理工學院]]高等衛生學校的學生一般都安排在此實習。
