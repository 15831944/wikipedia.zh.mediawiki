'''仁德車站'''位於[[臺南市|臺南市]][[仁德區|仁德區]]，是[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線的鐵路車站]]，配合[[臺鐵捷運化|臺鐵捷運化]]政策，所新設的[[通勤鐵路車站|通勤鐵路車站]]<ref name = "TRAnews" />。

== 歷史 ==
原站體為1900年啟用為[[保安車站|保安車站]]前身'''車路墘停車場'''。1909年，[[臺灣製糖株式會社|臺灣製糖株式會社]]於臺南廳仁德南里田厝庄（今仁德區田厝里）興建
「[[仁德糖廠|臺南製糖株式會社車路墘製糖所]]」，為配合工廠預定地，而將舊車站北遷1.5公里（今保安車站）。

2009年，為配合[[臺鐵捷運化|臺鐵捷運化]]政策需要，在第一代車路墘停車場舊址重設車站，站名訂為仁德。

*1900年11月29日：臺南↔打狗（今[[高雄港車站|高雄港車站]]）鐵路通車，設'''車路墘停車場'''。
*1909年11月10日：配合「臺南製糖株式會社車路墘製糖所」糖運，北遷1.5公里為今[[保安車站|保安車站]]。
*2009年7月14日：本站動工。
*2014年1月10日：本站正式啟用<ref name = "TRAnews" />。

== 站名由來 ==
古地名'''車路墘'''，第一代[[保安車站|保安車站]]（當時稱為「車路墘停車場」）就位在本站址。車路墘在[[閩南語|閩南語]]意思是「車路（大馬路）旁邊」。

本站位於[[保安工業區|保安工業區]]，並不位於仁德市區（即仁德區公所、[[臺南交流道|臺南交流道]]一帶）。縱貫線經過仁德區外圍，並設保安、仁德、[[中洲車站|中洲]]等3站，但3站都距仁德市區有5公里以上距離。

== 車站構造 ==
*平面車站
*[[側式月台|側式月台]]兩座。
*站房位於東側（非[[跨站式站房_(台鐵)|跨站式站房]]），設有無障礙電梯及月臺聯絡天橋。
*本站位於縱貫線基隆起362.0公里處，距保安站1.4公里，距中洲站2.6公里，指定由[[中洲車站|中洲車站]]管理。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#簡易站|簡易站]]，並指定由[[中洲車站|中洲站]]管理。
*本站僅停靠[[臺鐵區間車|區間車]]及[[沙崙線|沙崙線區間車]]，不停靠區間快車及對號列車。
* 本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]、[[Icash|icash2.0]]以及[[有錢卡|HappyCash]]進出站。
