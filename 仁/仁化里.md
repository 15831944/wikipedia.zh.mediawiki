'''仁化里'''位於[[台中市|台中市]][[大里區|大里區]]東南端，境內有[[大里工業區|大里工業區]]，因為人口密集，分劃出仁化、仁德兩里。

== 里名由來 ==
仁化里古名'''番仔寮'''，[[臺灣戰後時期|二戰後]]，建村之時，因政府規定以二字命名，而「番仔寮」名稱不雅，故由當時的村長[[林重|林重]]命名，取「仁德感化」的「仁化」二字為命名為「仁化村」。

==地理==
仁化里劃分為34鄰，3,051戶，總人口10309人。本里東與[[太平區_(臺中市)|太平區]]德隆里接壤，東南臨[[健民里_(大里區)|健民里]]，西、北兩側有[[頭汴坑溪|頭汴坑溪]]流經，與[[立德里_(大里區)|立德]]、[[新仁里_(大里區)|新仁]]二里對望，南以仁化路界[[仁德里_(大里區)|仁德里]]。

== 歷史沿革 ==
仁化里於清代時隸屬藍興堡番仔寮莊；明治35年（1902年）時，改隸臺中廳大里杙區番仔寮庄，至大正9年（1920年）則隨行政區調整，隸屬大屯郡大里庄番仔寮大字。二次大戰結束後，國民政府接收臺灣，以原番仔寮大字範圍被劃歸為'''健民村'''與'''仁化村'''，隸屬臺中縣大里鄉。民國45年（1956年）以後，臺灣工商業發達，仁化路的鞋廠林立，[[中國衣車|中國衣車]]等等數十家大企業，進駐仁化村，後來自全省各個族群都有遷入者，約佔全里百分之七十的外來人口。民國82年（1993年）大里鄉升格為縣轄市'''大里市'''，隨之改制為'''仁化里'''。民國91年（2002年），，由於人口過度密集，而實施里鄰調整，析南側置「'''仁德里'''」，仁德里取其「仁德教化」之意，以對照昔日原住民群集之地。所以基本上生活圈和舊有的[[仁德里|仁德里]]重疊。

== 經濟發展 ==
設有[[大里工業區|大里工業區]]和[[仁化工業區|仁化工業區]]，工業區鄰近台中市，也鄰近[[台中軟體園區|台中軟體園區]]，成為大里地區重要的中小企業集散地。主要的產業為機械、工具、五金業。通勤範圍內有[[太平區_(臺中市)|太平區]]、[[霧峰區|霧峰區]]，與[[台中生活圈四號道路|台中生活圈四號道路]]之元提交流道鄰近，使得生活圈與市區更為緊密。
