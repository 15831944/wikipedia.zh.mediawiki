'''仁大工業區（Renda Industrial Park）'''是由相鄰的'''仁武工業區'''與'''大社工業區'''所組成，位在[[台灣|台灣]][[高雄市|高雄市]]的[[仁武區|仁武區]]及[[大社區|大社區]]，鄰近[[高雄煉油廠|高雄煉油廠]]，並與[[台塑仁武廠|台塑仁武廠]]相隔於[[後勁溪|後勁溪]]兩岸。其仁武部分完成於1972年<ref name = "moe2"/>，占約21公頃，屬於綜合性工業區，區內有化學、塑膠、金屬、電子零件、機械設備廠商<ref name = "moe"/>；大社部分則於1971年及1975年分別完成兩期開發<ref name = "moe2">[http://www.moeaidb.gov.tw/iphw/renda/park/Park1.jsp 設立緣起]</ref>，共有109公頃，以[[石油化學|石油化學]]工業為主<ref name = "moe">[http://www.moeaidb.gov.tw/iphw/renda/park/Star.jsp 園區特色]</ref>。兩園區由仁大工業區服務中心管理，該中心同時也掌管位於[[梓官區|梓官區]]的[[海洋放流管制中心|海洋放流管制中心]]<ref name="moe3">[http://www.moeaidb.gov.tw/iphw/renda/park/Position1.jsp 地理位置與交通狀況]</ref>。

== 工安事件 ==
大社工業區曾在1978年11月24日發生[[氰化物|氰化物]]外洩事件，造成工人及附近[[楠梓區|楠梓區]]五常里及惠楠里居民共400多人送醫，其中一名工人死亡<ref>{{cite journal|journal=[[科學月刊|科學月刊]]|author=呂世宗|url=http://210.60.226.25/science/content/1982/00070151/0007.htm|title=台灣環境管理與評估|volume=151|date=1982年7月|deadurl=yes|archiveurl=https://archive.is/20120707103525/http://210.60.226.25/science/content/1982/00070151/0007.htm|archivedate=2012-07-07}}</ref>。

== 交通 ==
仁武工業區與大社工業區分別位在仁武區[[水管路_(仁武區)|水管路]]的南北兩側；中間另有南北向的[[鳳仁路_(仁武區)|鳳仁路]]（屬[[File:TW_CHW183.svg|23px]][[縣道183號|縣道183號]]）通過。同時該工業區也位在[[File:TWHW1.svg|23px]][[中山高速公路|國道一號]]與[[File:TWHW10.svg|23px]][[國道十號|國道十號]]之間，附近有[[楠梓交流道|楠梓交流道]]等出口。[[台灣高速鐵路|台灣高速鐵路]]在此工業區沿水管路進入[[左營區|左營區]]內的[[新左營車站|高鐵左營站]]。
