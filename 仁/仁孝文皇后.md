'''仁孝文皇后'''（{{bd|1362年||1407年||}}），[[徐姓|徐姓]]，[[本名|本名]]无记载，明成祖[[朱棣|朱棣]]嫡-{后}-，[[濠州|濠州]]人，明朝开国功臣[[徐达|徐达]]之嫡长女。

== 燕王妃 ==

徐皇后生于元朝[[至正|至正]]二十二年（1362年），比成祖小兩岁。她的父亲是明朝开国功臣[[徐达|徐达]]，母謝氏（[[謝再興|謝再興]]次女）。徐皇后“幼贞静，好读书，称女诸生”。明太祖[[朱元璋|朱元璋]]听说徐达之女贤淑，与徐达订下婚约。朱元璋曾亲至徐家，为儿子[[朱棣|朱棣]]聘为妃。

[[洪武|洪武]]九年正月二十七日，十五岁的徐氏嫁给十七岁的燕王[[朱棣|朱棣]]为王妃。第二年六月，生下朱棣长女[[永安公主_(明成祖)|朱玉英]]。第三年，又生朱棣长子[[明仁宗|朱高炽]]（即明仁宗），共生三子四女。徐氏姿色秀麗，温和賢慧，[[马皇后_(明太祖)|马皇后]]不只一次当众称赞这位儿媳妇。洪武十三年三月，燕王妃随成祖就藩[[北平|北平]]。

后来，[[建文帝|建文帝]]削藩，建文元年（1399年）七月五日，燕王朱棣宣布起兵'''靖难'''，在朱棣前往大寧“借”兵之际，面对[[李景隆|李景隆]]五十万大军兵临城下，燕王妃本人亲自登城督战，与世子朱高炽成功保卫了北平城。建文四年六月十三日，历时整整三年的[[靖难之役|靖难之役]]终于结束，明王朝的京城南京被燕王军攻陷，燕王妃再次踏进皇宫的时候，已经是[[明朝|明朝]]皇后了。

== 中宮 ==
建文四年七月成祖登基为帝，十一月封王妃徐氏为'''皇后'''<ref>《明太宗實錄》卷14：徐氏，中山武宁王徐达之女为朕正妃，内助藩国二十余年，朕躬行天讨无内顾之忧，济朕艰难同勤开国，今寰宇肃清朕登大宝允赖相成宜正位号，今特遣使奉金册金宝立尔为皇后，以奉神灵之统母仪天下。</ref>。

做为妻子，在朱棣竭力为自己“正名”的时候，徐皇后也倾尽全力相助明成祖。 永乐元年（1403年）正月，一部《梦感佛说第一希有大功德经》颁行天下。她还编写了《内训》二十篇，《劝善书》一部，都颁行天下，这些文字旨在推行针对女性的教育，并倡导修德劝善，为自己更为丈夫赢取民心。 除此之外，徐皇后还曾经向朱棣要求召见大臣们的妻子，并对她们说：“女人侍奉丈夫，并不仅仅是关心他们的衣食起居而已，应该对他们的前途事业也有所助益。朋友的劝告，不易被男人采纳，同样的话妻子来说，就容易入耳得多了。我与皇上朝夕相处，从不以私欲开口，所说的一切都以生民为念。希望你们也能以此自勉。” 

可惜，徐皇后年寿不永，她只做了四年皇后，就在永乐五年（1407年）的七月离开了人世，時年四十六歲。临终前，她最后一次劝谏朱棣，让他爱惜百姓，广求贤才，恩礼宗室，不要骄惯自己的娘家。她还叮嘱太子朱高炽说：“我一直惦记着当年在‘靖难之役’初起时，为守住北平城而应命作战的将士妻子，感念她们的功劳和付出的伤亡。想要趁着皇帝日后北巡的机会，亲自向她们以及她们的家人赠予嘉奖抚恤。只可惜我再也无法完成这个宿愿，这是我此生唯一的恨事。”

== 去世 ==
徐皇后去世时年四十六岁，朱棣对结发妻子的去世非常悲恸<ref>《明太宗實錄》卷69：后崩，上哭恸群臣奉慰，上曰皇后仁明贤淑汉马氏唐长孙之伦也，虽处中宫其一念惟在仁民继今朕入宫不复闻直言矣</ref>，他为徐皇后上谥号曰'''仁孝'''，并从此不再立皇后。

永乐七年（1409年），朱棣北巡，回到了盛满往事的北平，并着手迁都事宜。于是命礼部尚书赵羾和江西术士廖钧卿等前往北京一带卜选陵址，随即选得吉地于昌平县天寿山，得到朱棣的认可后，即降旨圈地80里，作为陵区禁地，并指示建成后将仁孝皇后安葬于此。同年，动工营建。

四年后永乐十一年（1413年）二月长陵地下玄宫落成，朱棣将仁孝皇后的梓宫从南京迁至北京葬于[[长陵|长陵]]。自永乐五年皇后去世时至永乐十一年下葬，这期间徐氏的棺椁一直被置于南京皇宫内未葬<ref>《明太宗实录》</ref>，徐皇后也是入葬明十三陵的第一人。

十五年后，即永乐二十二年（1424年）七月十八日，明成祖朱棣病逝于征討漠北的途中，享年六十四岁。同年十二月，朱棣与徐皇后'''合葬长陵'''。 

永乐二十二年（1424年）九月仁宗上尊母亲仁孝皇后谥号曰'''仁孝慈懿诚明莊献配天齐圣文皇后'''。祔太庙。

=== 子女 ===

朱棣共有四子五女，除最小的儿子[[朱高爔|朱高爔]]和女儿[[常宁公主|常宁公主]]外，俱为徐皇后所生。前三子分别为太子[[朱高炽|朱高炽]]（明仁宗）、汉王[[朱高煦|朱高煦]]及赵王[[朱高燧|朱高燧]]。除长子高炽性情温和好学之外，汉王高煦、赵王高燧则都生性骄横，她主张立高炽为太子，曾屡次向朱棣进言，指出高煦高燧性格不好，不但不能重用，还得为他们选择敢于监督劝诫的僚属。

== 注释 ==
'''徐皇后的名字'''

据史料记载，徐皇后未曾留下名字，“徐仪华”、“徐妙云”、“徐锦曦”皆为后人杜撰。

'''徐皇后与朱棣的特殊关系'''
朱元璋唯一的亲侄子[[朱文正_(明朝宗室)|朱文正]]之妻（姐）与[[徐达|徐达]]之妻（妹）是亲姐妹，同是谢再兴的女儿，所以朱文正与徐达是[[连襟|连襟]]，朱文正的儿子[[朱守谦|朱守谦]]（生于1361年）与徐达的女儿徐皇后（生于1362年）是表兄妹，朱守谦又是朱棣的侄子，所以徐皇后未婚时又可以称朱棣为四皇叔。

'''徐皇后与[[徐妙锦|徐妙锦]]'''

根据朱元璋为徐达撰写的神道碑，徐达有四子四女。史书仅记载徐达三个女儿的生平，即长女仁孝文皇后、次女代王朱桂妃、三女安王朱楹妃<ref>《弇山堂别集》、《明史》</ref>，一女生平不详。据野史记载，徐皇后在1407年去世后，朱棣曾想娶徐皇后之妹徐妙锦为皇后，可是徐妙锦嫌弃姐夫的人品毅然出家为尼<ref>《[[罪惟录|罪惟录]]》</ref>。[[嘉靖|嘉靖]]年间，[[郑晓|郑晓]]所著《[[今言|今言]]》称：徐皇后生母谢夫人当面拒绝了朱棣的请求<ref>[[郑晓|郑晓]]《[[今言|今言]]·卷二》</ref>。而根据相关史料，此时谢夫人至少已逝世二十年。徐皇后父亲徐达亦已逝世二十二年。即使是徐达[[遗腹女|遗腹女]]，此女年龄也远大于当时女性[[初婚年龄|初婚年龄]]，徐皇后十五岁即成婚。

'''徐皇后与其他妃嫔'''
史书中直到徐皇后去世两年后才开始记载朱棣的其他妃嫔，其中最出名的也仅有两位，一位是苏州的王贵妃，一位是朝鲜的权贤妃，都是在永乐七年册封的，不过王贵妃最初封的品级仅是昭容，何时被晋升为贵妃就不得而知了。

[[昭献贵妃|王贵妃]]得宠的原因是，因她以皇后为楷模所以深得朱棣的信任<ref>《明史》:妃有贤德，事仁孝皇后恭谨，为帝所重。</ref>。[[恭献贤妃|权贤妃]]得宠的原因是，以姿色得宠<ref>《明史》:朝鲜贡女充掖庭，妃与焉。姿质穠粹，善吹玉箫。帝爱怜之。</ref>，可权氏去世后的待遇却远不及王贵妃，更没有获得陪葬的资格。

徐皇后生性仁厚，为人处事有原则，待妃嫔们有恩，能容人过，有威信。处理后宫事物肃然有条。并且时常告诫妃嫔要和睦相处，待人要恭敬。<ref>《明书》</ref>

而徐皇后去世后朱棣未有再立皇后的打算。<ref>《明史》：七年营寿陵于昌平之天寿山，又四年而陵成，以后葬焉，即长陵也。帝亦不复立-{后}-。</ref>
