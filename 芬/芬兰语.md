'''芬兰语'''（{{lang|fi|suomi}}，{{audio|Fi-suomi.ogg|發音}}）是[[芬兰|芬兰]]大部分（88.3%，2016年底）居民的母语<ref name="ly_bilv">{{cite web | url = http://pxnet2.stat.fi/PXWeb/pxweb/en/StatFin/StatFin__vrm__vaerak/statfin_vaerak_pxt_010.px/?rxid=a16b50be-a334-4b1c-816b-6c43d900d446 | title = Language according to age and sex by region in 1990 to 2016 | language = en | publisher = Statistics Finland | date = 2016-12-31 | accessdate = 2018-02-22 }}</ref>，也被境外芬兰裔侨民所用。它是芬兰的两种[[官方语言|官方语言]]之一，也是[[瑞典|瑞典]]的一种法定少数族裔语言。

芬兰语属于“[[烏拉爾語系|烏拉爾語系]]”的“[[芬蘭-烏戈爾語族|芬蘭-烏戈爾語族]]”，是一种[[黏着语|黏着语]]。[[名词|名词]]和[[形容词|形容词]]都因应词语在[[句子|句子]]内的角色而有格的变化。由于和其他[[欧洲|欧洲]]语言的语系都不同，词根和绝大部分其他欧洲语言差别很大，所以经常被欧美人认为是比较难学的语言。也因為如此，以芬蘭語為母語的人民往往会学习兩三種其它語言，以利交流。

== 历史 ==
有学者推测波罗地-芬兰语支由一种原始芬兰语演变而来，而北方[[薩米人|薩米人]]（舊稱拉普人）的[[薩米語|薩米語]]可能早在公元前1500年-前1000年已由原始芬兰语分支出来。依学者的推测，原始芬兰语分北方、南方、东方三种方言，而波罗地-芬兰语支各种不同语言的形成大约在公元1世纪左右。

[[File:Agricola-Mikael-1982.jpg|thumb]]而发行的邮票]]
芬兰语最早的书写方式是在16世纪由芬兰的主教[[米卡埃尔·阿格里科拉|米卡埃尔·阿格里科拉]]发明的，书写方式依据当时芬兰的官方语言[[瑞典语|瑞典语]]、[[德语|德语]]和[[拉丁语|拉丁语]]制订。芬兰文的书写方式在之后被许多其他人修改过。

16世纪的基督教宗教复兴运动,确实标志着芬兰文的开始，许多芬兰作家——包括米卡埃尔·阿格里科拉主教——用芬兰语也创造了不少文学的成就。到了17世纪，在芬兰，已有重要的著作以芬兰文、[[丹麦文|丹麦文]]、[[挪威文|挪威文]]、[[拉脱维亚语|拉脱维亚文]]、德文、以及瑞典文写成；然而，最重要的文献还是用拉丁语写作的。瑞典语直至17世纪末还是芬兰的唯一官方语言；它和芬兰语，在当时都只是不重要的小语种。

=== 阿格里科拉主教工作的成果 ===
标准芬兰文的多种书写规则——尤其是拼字规则——都源自阿格里科拉主教的努力。由于他以芬兰西部的方言作为芬兰文的基础，西部方言的发音也成为了芬兰文的标准拼法。

阿格里科拉用''dh''和''d'' 代表浊舌齿擦音（相当[[英语|英语]]中在this 中的''th''），用''tz'' 和''z'' 代表清舌齿擦音（相当[[英语|英语]]中在thanks中的''th''），后来语音有所变化，舌齿擦音已经在芬兰语中消失，在东部方言中''dh''变成了''j''和''v''，在西部方言中变成了''r''或''l''，''tz''变成了''ht'' 或 ''tt''。

在翻译《[[新约|新约]][[圣经|圣经]]》的过程中，阿格里科拉创造了一些新词，当中部分例如<span lang=fi>「armo」</span>（慈悲）和<span lang=fi>「vanhurskas」</span>（正义的）的一些词语，现在仍然通用。他总共用了大约8500个单词，现在还在使用中的有60%。

他用''Ch''， ''c'' 或 ''h'' 代表清软腭擦音，现在这个音已经变成喉音''h''了；他用''gh'' 和''g'' 代表浊软腭擦音，现在这个音也已经消失。

== 表音方式 ==
芬兰语的书写字母和发音基本是一一对应的，所以比较容易学习。

一般规则如下：
* “B”，“C”，“F”，“Z” 基本上只用来拼写外来词；“G”通常跟在“N”之后一起发“ng”音，其它情况也只用来拼写外来词。
* “Q”，“X”只用于从[[瑞典语|瑞典语]]或其它外语而来的名字。
* “W”和“V”发音基本一样，类似英语中“V”的音，“W”在芬兰语中只是保留在古代文献和姓名中；
* 长音用双字母表示，比较清楚；
* “NK”中的“N”类似于英语中的<span lang=en>“ng”</span>，“K”照常发音；
* “H”在[[辅音|辅音]]前比在[[元音|元音]]前发音更明显；
* “Ä”和“Ö”是和“A”、“O”不同的音，类似德语中的“Ä”和“Ö”的音，但不只是变音符号，是不同的字母；
* 芬兰语中没有浊辅音z，所以实际上“z”，“š”或“ž”发音都和s差不多，只是书写表达时用。

== 地理分布 ==
使用芬兰语的人大约有六百万，主要分布在芬兰、瑞典、[[挪威|挪威]]、[[爱沙尼亚|爱沙尼亚]]和[[俄罗斯|俄罗斯]]，还有部分在[[美国|美国]]的移民作为日常用语，主要聚居在[[密西根州|密西根州]]。

== 方言 ==
芬兰语有两种方言，东部方言和西部方言。西部方言又有西南、南方、中南、中部和北方方言的区别。东部方言分布较广，主要有东部和东南部两种，东南部方言主要在[[俄罗斯|俄罗斯]]的南卡累利阿和圣彼得堡周围地区。

== 语言形式 ==
{{Main|芬兰语语法}}
芬兰语类似[[阿拉伯语|阿拉伯语]]，有两种形式，一种是通用形式''yleiskieli''，主要用于正式场合，如教堂弥撒、政治演讲以及新闻报道等，其书面形式叫''kirjakieli''，几乎所有的出版物都是用这种语言；另一种是口语''puhekieli''，用于普通广播电视节目和一般人际交流、书信和网上聊天。

口语是自然语言发展变化产生的，通用语则是依据文学作品和文献，保留严谨的语法变格规则，这些规则有很多已经从口语中消失了，像许多代词和后缀都逐渐在口语中被弱化或省略了。在学校教学中教通用语，有许多孩子由于经常读书，已经将通用语作为他们的第一语言。

== 词汇 ==
芬兰语经常而且很大程度上采用粘着，核心字汇较例如英语的语言小，后缀的使用程度也较大。例如“书”这个词，<span lang=fi>''kirja''</span>，可以用后缀演变成<span lang=fi>''kirjain''</span>（字母）、<span lang=fi>''kirje''</span>（信）、<span lang=fi>''kirjasto''</span>（图书馆）、<span lang=fi>''kirjailija''</span>（作者）、<span lang=fi>''kirjallisuus''</span>（文学）、<span lang=fi>''kirjoittaa''</span>（写）、<span lang=fi>''kirjoittaja''</span>（写字的人）、<span lang=fi>''kirjallinen''</span>（书写形式的）、<span lang=fi>''kirjata''</span>（写下、登记、记录）、<span lang=fi>''kirjasin''</span>（字体）等。

下列一些比较普遍的后缀。请注意一对后缀时，使用哪一个后缀是根据[[元音和谐律|元音和谐]]现象而定。后缀的添加会引起[[辅音交替|辅音交替]]

* <span lang=fi>-ja/jä</span>：作的人 (如 <span lang=fi>''lukea''</span> = 读 → <span lang=fi>''lukija''</span> = 读者)；
* <span lang=fi>-lainen/läinen</span>：住在某地的人 (可以做[[名词|名词]]也可以做[[形容词|形容词]]) <span lang=fi>''Englanti''</span> = 英国 → <span lang=fi>''englantilainen''</span> = 英国人或英国的东西； <span lang=fi>''Helsinki''</span> = 赫尔辛基 → <span lang=fi>''helsinkiläinen''</span> = 赫尔辛基人；
* <span lang=fi>-sto/stö</span>：集合 (<span lang=fi>''kirja''</span> = 书 → <span lang=fi>''kirjasto''</span> = 图书馆； <span lang=fi>''laiva''</span> = 船 → <span lang=fi>''laivasto''</span> = 海军、舰队)
* <span lang=fi>-in</span>：部分或工具 (<span lang=fi>''kirja''</span> = 书 → <span lang=fi>''kirjain''</span> = (一个)字母； <span lang=fi>''vatkata''</span> = 掸 → <span lang=fi>''vatkain''</span> = 掸子，搅拌器)
* <span lang=fi>-uri/yri</span>：做的人或物 (<span lang=fi>''kaivaa''</span> = 挖 → <span lang=fi>''kaivuri''</span> = 挖土机； <span lang=fi>''laiva''</span> = 船 → <span lang=fi>''laivuri''</span> = 发货人、水手)
* <span lang=fi>-os/ös</span>：表示动作的结果 (<span lang=fi>''tulla''</span> = 来 → <span lang=fi>''tulos''</span> = 结果； <span lang=fi>''tehdä''</span> 作 → <span lang=fi>''teos''</span> = 作品)
* <span lang=fi>-ton/tön</span>：无 (<span lang=fi>''onni''</span> = 幸福 → <span lang=fi>''onneton''</span> = 不幸福； <span lang=fi>''koti''</span> = 家 → <span lang=fi>''koditon''</span> = 无家可归)
* <span lang=fi>-llinen</span>：性质 (<span lang=fi>''lapsi''</span> = 孩子 → <span lang=fi>''lapsellinen''</span> = 孩子气； <span lang=fi>''kauppa''</span> = 商店 → <span lang=fi>''kaupallinen''</span> = 商业性的)
* <span lang=fi>-kas/käs</span>：性质 (<span lang=fi>''itse''</span> = 自己 → <span lang=fi>''itsekäs''</span> = 自私； <span lang=fi>''neuvo''</span> = 劝告 → <span lang=fi>''neuvokas''</span> = 随机应变)
* <span lang=fi>-va/vä</span>：正在做或有某事物 (<span lang=fi>''taitaa''</span> = 能够、可能  → <span lang=fi>''taitava''</span> = 有技术的； <span lang=fi>''johtaa''</span> = <span lang=fi>领导→ ''johtava''</span> = 领导的)

== -{借}-词 ==
许多世纪以来，芬兰语从外来语中吸收了不少-{借}-词，估计真正从古代芬兰语中遗留下的词根只有300个左右。有的专家认为芬兰语几个世纪吸收的-{借}-词几乎相当[[达罗毗荼语系|达罗毗荼语系]]几千年中吸收的-{借}-词水平。例如''kuningas'' = 国王来自<!--[[德语|德语]]-->日耳曼語的 ''kuningaz''、''morsian'' = 新娘， ''armas'' = 亲爱、''äiti'' = 母亲(äiti據信源自哥德語的aiþei)等，但源于芬兰语的表示母亲的词根''emä''仍然保留在一些复合词中，已经转化为一种比喻形式，如''emälaiva'' =母舰、''emävale''= 大谎言（谎言的母亲）等。

由于从12世纪至1809年，芬兰一直属于瑞典王国，现在芬兰还有6%的人口是[[芬兰瑞典族|瑞典族]]，1809年后属于沙皇[[俄罗斯|俄国]]，所以从[[瑞典语|瑞典语]]来的-{借}-词最多，也有一些[[俄语|俄语]]词。例如''Raamattu''=[[圣经|圣经]]是从俄语来的，还有一些宗教词汇也是从俄语来的。

在现代，从[[英语|英语]]来的-{借}-词增加了不少，主要在[[文化|文化]]方面，从估计商贸、[[音乐|音乐]]、[[电影|电影]]、[[文学|文学]]以及互联网带来不少词汇。有的跨国公司如芬兰的[[诺基亚|诺基亚]]已经将英语作为公司中的第一语言，新的-{借}-词不仅排挤原来的芬兰词汇，而且排挤早先的-{借}-词，如从瑞典语来的词汇''treffailla''=约会，已经被新的来自英语的''deittailla''取代。

== 新造词 ==
也有的词汇是根据固有词汇合成的，如
:<span lang=fi>''puhelin''</span> =[[电话|电话]]（意思是“用以说话的东西”）<br /><span lang=fi>''tietokone'' =[[计算机|计算机]]（意思是“知识机器”）<br />''levyke''</span> =[[软盘|软盘]]（<span lang=fi>''levy''</span> =碟 + <span lang=fi>''-ke''</span> = -子）<br /><span lang=fi>''sähköposti''</span> =[[电子邮件|电子邮件]]（意思是“电邮”）
在口语中还有不同来源的词。例如，<span lang=fi>levyke</span>固然是-{zh-hans:软盘;zh-hant:軟碟}-，但是在口语中大-{zh-hans:软盘;zh-hant:軟碟}-卻称<span lang=fi>''lerppu''</span>（来自英语<span lang=en>floppy</span>），而小-{zh-hans:软盘;zh-hant:軟碟}-则称<span lang=fi>''korppu''</span>（来自芬兰语“饼干”，因为其形状像饼干）。至于光盘<span lang=fi>''romppu''</span>则是1990年代初<span lang=en>CD-ROM</span>开始流行时，在由一份叫《<span lang=fi>Suomen Kuvalehti</span>》（芬兰画报）的杂志所举办的一个比赛中创作出来的。
