'''芬蘭化'''指的是一个弱小的国家近乎無底線地聽命於强大的邻国的政策决定，以保持[[主权|主权]]及领土完整，基本上屬貶意詞彙。類似[[冷戰|冷戰]]時[[芬蘭|芬蘭]]和[[蘇聯|蘇聯]]兩國之間的關係，[[第二次世界大战|第二次世界大战]]之前的[[丹麦|丹麦]]对[[纳粹德国|纳粹德国]]也是如此。此詞出現於1960年代後期的[[西德|西德]]，為當時西德的保守派批評西德總理[[威利·勃蘭特|威利·勃蘭特]]所新造的詞。[[威利·勃蘭特|威利·勃蘭特]]為當時重視同共產主義國家尤其东欧阵營對話的西德總理。

== 历史 ==
在1939年簽訂的[[蘇德互不侵犯條約|蘇德互不侵犯條約]]中，納粹德國承認芬蘭為蘇聯的勢力范圍。後蘇聯對芬蘭發起了[[冬季戰爭|冬季戰爭]]，最終芬蘭以割讓本國十分之一的領土為代價同蘇聯停戰。1941年[[蘇德戰爭|蘇德戰爭]]開始後，芬蘭對蘇聯發動了以收復失地為目的的[[繼續戰爭|繼續戰爭]]。後因軸心國轉入劣勢，因而芬蘭同蘇聯停戰，并攻擊駐扎在芬蘭國內的德軍。因其戰敗國的身份，加之地理位置而難以期望獲得西方世界支援的芬蘭（不過[[美國|美國]]曾秘密援助過芬蘭），在1948年同蘇聯簽訂了互助條約，蘇联承诺保障及不侵犯芬兰的主权。

[[冷战|冷战]]期间，芬兰沒加入[[北约|北约]]，保持[[中立国|中立国]]的外交路线，維持獨立主权，以及[[半总統制|半总統制]][[議會民主制|議會民主制]]和資本主義市场经济[[福利國家|福利國家]]，但在國際事務上，芬蘭事實上经常站在了蘇聯一側。大眾媒體也進行了自我審查，冬季戰爭中蘇聯的侵略等有可能被解释为“反苏”的言论也成為了禁忌的話題。鄰国[[瑞典|瑞典]]也奉行類似外交政策，这与加入北約的[[丹麦|丹麦]]及[[挪威|挪威]]略为不同，這些情況到了[[蘇聯解體|蘇聯解體]]，芬蘭加入[[歐盟|歐盟]]後不再存在。不过，不變的是芬兰仍奉行独立的外交政策，維持不加入北約並與美國保持距離。

== 美國 ==
「芬蘭化」说法一直受到爭議。而從另一方面來說，[[冷戰|冷戰]]時期[[美國|美國]]領導的反共陣營中，不少反共的資本主義國家如[[日本|日本]]、[[大韓民國|大韓民國]]及[[菲律賓|菲律賓]]等有美軍駐防的國家中，並無獨立自主的外交政策。

== 台灣 ==
[[中華民國|中華民國]]自1927年[[國民政府|國民政府]]成立、以至[[中華民國政府|中華民國政府]]1949年之後遷至[[臺灣地區|臺灣地區]]以來，一直奉行親[[美國|美國]]的外交政策。在台灣的[[蔣介石|蔣介石]]與和在中國大陸的[[毛澤東|毛澤東]]各自成為分離的政治實體，彼此敵對大約三十年。但兩人在1970年中期相繼去世之後，北京開始關注國內的經濟發展，後來北京停止砲擊台灣的金馬外島，並歡迎台灣「同胞」到大陸旅遊、投資及探親。

另一方面，在1988年開始主政的[[李登輝|李登輝]]總統也認可開啟這種交流，雙方的海峽交流協會於1993年在新加坡舉行首度直接[[汪辜會談|汪辜會談]]，展開雙方的「首度和解」。然而當美國核發簽證給李登輝來訪問[[康乃爾大學|康乃爾大學]]時，這個「首度和解」在1995年就遽然中斷。隨後歷經1995年和1996年中國對台海發射飛彈、華盛頓派遣航空母艦與雷達艦前來台海、以及台灣人民在2000年選出獨派的[[民主進步黨|民進黨]]籍[[陳水扁|陳水扁]]擔任總統，兩岸關係急遽惡化，這是兩岸關係的「再度凍結」。

兩岸關係的「二度和解」始於2005年時任中共中央總書記[[胡錦濤|胡錦濤]]淡化統一要求的談話。而[[中國國民黨|中國國民黨]]籍的[[馬英九|馬英九]]在2008年就任[[中華民國總統|中華民國總統]]主政後的和解政策，使台灣愈來愈走向「芬蘭化」，具體例證包括[[莫拉克颱風|莫拉克颱風]]後，台灣雖讓西藏精神領袖[[達賴喇嘛|達賴喇嘛]]赴台為災民祈福，卻拒絕維族領袖[[熱比婭|熱比婭]]入境，馬英九本人在[[六四事件|六四事件]]紀念日時不再批評北京，只輕描淡寫地以歷史傷痛帶過，都是不折不扣「芬蘭化」的表現。<ref>[https://www.wufi.org.tw/%E5%92%8C%E8%A7%A3%E6%94%BF%E7%AD%96%E4%B8%8D%E9%9C%80%E8%A6%81%E8%8A%AC%E8%98%AD%E5%8C%96/ 和解政策不需要芬蘭化] [[台灣獨立建國聯盟|台灣獨立建國聯盟]]</ref>

2008年，[[台灣|台灣]]第二次[[政黨輪替|政黨輪替]]後，[[台灣|台灣]][[亲中媒體|亲中媒體]]避免提及[[法輪功|法輪功]]、[[六四事件|天安門事件]]等，避免直接批評中國大陸，被認為是「媒體芬蘭化」<ref>[[鍾年晃|鍾年晃]]，《我的大話人生：「[[大話新聞|大話新聞]]」停播始末&我所認識的鄭弘儀》，2012，台北，前衛出版社，ISBN 9789578016996</ref>。
