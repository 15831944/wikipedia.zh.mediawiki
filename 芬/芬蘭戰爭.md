'''芬蘭戰爭'''（{{lang-sv|Finska kriget}}，{{lang-ru|Финляндская война}}，{{lang-fi|Suomen sota}}）是在1808年2月至1809年9月期間於[[芬蘭|芬蘭]]地區爆發的一場戰爭，參戰國家為[[瑞典|瑞典]]及[[俄羅斯|俄羅斯]]。最後瑞典戰敗，其東部的三成領土被割讓，成為附庸於[[俄羅斯帝國|俄羅斯帝國]]的[[芬蘭大公國|芬蘭大公國]]。

== 背景 ==
俄羅斯[[亚历山大·雅罗斯拉维奇·涅夫斯基|亞歷山大]]與[[拿破仑一世|拿破崙]]簽署了《[[提爾西特條約|提爾西特條約]]》後，他建議瑞典國王古斯塔夫四世應加入大陸封鎖。

與此同時，英國皇家海軍襲擊了[[哥本哈根战役_(1801年)|哥本哈根]]，展開了英俄戰爭。英國照1780年和1800年的條約，要求古斯塔夫關閉波羅的海所有外國軍港。但是瑞典國王回答說，只要法國人控制著[[波罗的海|波羅的海]]主要港口之前，他們就不可能達到目的。

== 1808年2月 - 5月 ==
[[File:The_Finnish_war_map1.svg|thumb]]
[[File:The_Finnish_war_map2.svg|thumb]]
1808年2月21日，在弗里德里·威廉·馮·布克霍夫登的帶領下24 000名俄軍士兵越過邊界，奪取了[[海门林纳|塔瓦斯胡什]]。瑞典對這次襲擊毫無準備，直到四月才宣戰。約有19000名瑞典軍駐紮在芬蘭的各個堡壘中，而其餘的軍隊由於擔心遭到丹麥的襲擊而無法離開瑞典南部。

3月，俄羅斯人攻陷了[[庫奧皮奧|庫奧皮奧]]、韃靼風琴、[[雅各布斯塔德|雅各布斯塔德]]、{{le|斯瓦尔特霍尔姆城堡|Svartholm fortress|斯瓦爾特霍爾姆}}、[[赫尔辛基|赫爾辛夫]]、[[漢科|漢科]]與[[哥得蘭島|哥德蘭]]和[[奧蘭群島|奧蘭群島]]。 

4月，俄羅斯人受到挫敗，來自哥德蘭和奧蘭群島的居民支持的瑞典，迫使俄國的維希上校及其駐軍投降。5月26日，英國艦隊進入[[哥德堡|哥德堡]]，但與瑞典國王交惡。

== 1808年8月 - 9月 ==
[[File:The_Finnish_war_map4.svg|thumb]]
[[File:The_Finnish_war_map3.svg|thumb]]
俄羅斯人獲得了大量的增援後，他們的人數增至55,000人，而瑞軍則達36,000人。8月14日，俄軍發起新的進攻。

俄國的11000強軍隊於9月1日在[[庫奧爾塔內|庫爾丹]]（9月1日）、{{le|薩爾米|Salmi (rural locality)}}（9月2日）和{{le|奧拉瓦伊斯|Oravais}}（9月14日）取得了更重要的勝利。在芬蘭東部，芬蘭游擊隊運動逐漸消失。

== 1809年冬季 ==
[[File:The_Finnish_war_map5.svg|thumb]]
當時俄羅斯軍隊已經佔領了芬蘭全境。11月19日，簽署《弗里德里克斯哈姆條約》，瑞典軍隊被迫離開芬蘭。
