根據《[[s:三國志/卷35#諸葛亮|s:三國志/卷35#諸葛亮]]傳》，建安十二年（207年）冬，[[徐庶|徐庶]]受駐軍[[新野|新野]]的劉備器重，推薦劉備尋訪「臥龍」[[諸葛孔明|諸葛孔明]]，劉備「凡三往，乃見」，前兩次到[[隆中|隆中]]（今[[襄陽市|襄陽市]][[古隆中|古隆中]]或[[南陽市|南陽市]][[臥龍崗|臥龍崗]]）拜訪諸葛亮都沒見到，第三次終於得見。《隆中對》中，諸葛亮為劉備分析了天下形勢，因荊、益二主皆弱，構想戰略計策，先取[[荊州_(古代)|荊州]]為家，再取[[益州|益州]]成鼎足之勢，結好[[孫權|孫權]]以為援，繼而待天下有變圖取中原，成霸業，興漢室。劉備聽完表示「善」。[[三顧茅廬|三顧茅廬]]之後，諸葛亮出山成為劉備的[[軍師|軍師]]，劉備集團之種種攻略皆基於此。

== 發生時間 ==
《[[s:三國志/卷35#諸葛亮|s:三國志/卷35#諸葛亮]]》記載《[[前出師表|前出師表]]》有言「爾來二十有一年矣」。[[裴松之|裴松之]]據此認為：「劉備以建安十三年敗，遣亮使吳，亮以[[建興_(蜀漢)|建興]]五年抗表北伐，自傾覆至此整二十年。然則備始與亮相遇，在敗軍之前一年時也。」《[[s:後漢紀/卷30#十二年|s:後漢紀/卷30#十二年]]》及《[[s:資治通鑒_(胡三省音注)/卷065|s:資治通鑒_(胡三省音注)/卷065]]》亦記三顧茅廬在建安十二年冬，即西元207年末。

== 後世評論 ==
[[崔浩|崔浩]]論說：「委棄荊州，退據巴蜀……此策之下者也。」<ref>[[s:魏書/卷43#毛脩之|s:魏書/卷43#毛脩之]]。《世說新語》</ref>，[[蘇洵|蘇洵]]稱：「棄荊州而取西蜀，吾知其無能為也。」<ref>蘇洵《[[s:權書/10#項籍|s:權書/10#項籍]]》</ref>[[田余慶|田余慶]]在《<隆中對>再認識》指出：「後人評論諸葛亮及其《隆中對》，由於種種原因，往往出現溢美和回護之詞。《隆中對》中睿智的判斷，有時被賦予先驗的色彩﹔瑜中之瑕，甚至也被認為是美玉的本色。」<ref>田余慶《〈隆中對〉再認識》</ref>有專家稱《隆中對》忽視了「荊州在揚州上游，關係吳國的安顧，孫權對荊州是勢所必爭的，否則便不能有吳國」。<ref>朱大渭：〈武將群中獨一人——關羽人神辨析〉</ref>「《隆中對》所言的自秦川、宛、洛北伐，若不是書生議政，紙上談兵，就只能是以退為進，虛張聲勢。」<ref>黎冷：〈《三國志》研究新論〉</ref>

== 地點爭議 ==
{{main|諸葛亮躬耕地之爭}}
《[[s:漢晉春秋|s:漢晉春秋]]》曰：亮家於南陽之[[鄧縣|鄧縣]]，在襄陽城西二十里，號曰隆中。《隆中对》的发生地自[[清代|清代]]开始存在争议，[[河南省|河南省]][[南阳市|南阳市]]的[[卧龙岗|卧龙岗]]和[[湖北省|湖北省]]襄樊市（已改名为[[襄阳市|襄阳市]]）的[[古隆中|古隆中]]都声称是诸葛亮躬耕之地。《隆中对》在2003年被重新选入中国大陆中学课本，文章中有“隆中，山名，在现在的湖北襄樊”的注释，这引起南阳方面的不满。有人认为《隆中对》能再次进入历史课本，襄樊市政府私下做了很多秘密工作，河南省南阳市教育界认为，教科书的编写应该以历史为根据，不能受到地方势力的影响，他们对襄樊市的小动作表示遗憾。<ref>{{cite web|url=http://www1.peopledaily.com.cn/GB/jiaoyu/1055/2043074.html|title=教科书能否并且应该承载带动地方经济发展的重任，《隆中对》复现教科书的背后}}</ref>。
