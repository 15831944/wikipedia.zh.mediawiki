'''棕背伯劳'''（[[学名|学名]]：{{lang|la|''Lanius schach''}}）为[[伯劳科|伯劳科]][[伯劳属|伯劳属]]的[[鸟类|鸟类]]，俗名海南鵙、大红背伯劳。是一種[[東洋界|東洋界]]很常見的[[留鳥|留鳥]]。

== 分布 ==
棕背伯劳分布的區域從[[哈薩克|哈薩克]]起，穿過[[阿富汗|阿富汗]]、[[巴基斯坦|巴基斯坦]]與[[印度半島|印度半島]]（除了東邊的國家<ref name=grewal>
{{cite book|title = Photographic guide to birds of India |last = Grewal |first = Bikram
|authorlink = :en:Bikram Grewal |coauthors = Bill Harvey and [[:en:Otto_Pfister_(naturalist)|Otto Pfister]]
|publisher = Periplus editions / [[:en:Princeton_University_Press|Princeton University Press]] |location = Hong Kong |year = 2002
}} p. 242</ref>），到[[紐幾內亞|紐幾內亞]]。[[中国|中国]]、泰国、缅甸、印度、斯里兰卡、越南、[[台灣|台灣]]、菲律宾和马来半岛等地都有分布。主要栖息于常见于低海拔地区的疏林地，例槐、枫杨、泡桐、榆、柳、刺槐等乔木的顶端。该物种的模式产地在中国。<ref name='id2780'/>

== 亚种 ==
* '''棕背伯劳台湾亚种'''（[[学名|学名]]：{{lang|la|''Lanius schach formosae''}}）。分布于[[台灣|台灣]]以及[[中国|中国]]境内。该物种的模式产地在台灣本島。<ref name='id2781'>{{cite web| url =http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=2781| title =棕背伯劳台湾亚种| work =《中国动物物种编目数据库》| publisher =中国科学院微生物研究所| author =中国科学院动物研究所| accessdate =2009-04-04| deadurl =yes| archiveurl =https://web.archive.org/web/20160305041255/http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=2781| archivedate =2016-03-05}}</ref>
* '''棕背伯劳海南亚种'''（[[学名|学名]]：{{lang|la|''Lanius schach hainanus''}}）。分布于[[中国|中国]]的[[海南|海南]]等地。该物种的模式产地在海南岛。<ref name='id2782'>{{cite web| url =http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=2782| title =棕背伯劳海南亚种| work =《中国动物物种编目数据库》| publisher =中国科学院微生物研究所| author =中国科学院动物研究所| accessdate =2009-04-04| deadurl =yes| archiveurl =https://web.archive.org/web/20160305010148/http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=2782| archivedate =2016-03-05}}</ref>
* '''棕背伯劳指名亚种'''（[[学名|学名]]：{{lang|la|''Lanius schach schach''}}）。在[[中国|中国]]，分布于长江以南地区、北至甘肃、[[陕西|陕西]]、西至四川、[[云南|云南]]，偶见于山东等地。该物种的模式产地在中国。<ref name='id2783'>{{cite web| url =http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=2783| title =棕背伯劳指名亚种| work =《中国动物物种编目数据库》| publisher =中国科学院微生物研究所| author =中国科学院动物研究所| accessdate =2009-04-04| deadurl =yes| archiveurl =https://web.archive.org/web/20160305074903/http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=2783| archivedate =2016-03-05}}</ref>
* '''棕背伯劳西南亚种'''（[[学名|学名]]：{{lang|la|''Lanius schach tricolor''}}）。分布于尼泊尔、泰国、缅甸、印度以及[[中国|中国]]的[[云南|云南]]、[[西藏|西藏]]等地。该物种的模式产地在尼泊尔。<ref name='id2784'>{{cite web| url =http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=2784| title =棕背伯劳西南亚种| work =《中国动物物种编目数据库》| publisher =中国科学院微生物研究所| author =中国科学院动物研究所| accessdate =2009-04-04| deadurl =yes| archiveurl =https://web.archive.org/web/20160305042821/http://www.bioinfo.cn/db05/BjdwSpecies.php?action=view&id=2784| archivedate =2016-03-05}}</ref>
