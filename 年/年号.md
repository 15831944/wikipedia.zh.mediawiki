'''年号'''是[[中国歷史|中国歷史]]君主時代帝王[[纪年|纪年]]所立的名号<ref name="重編字典">《[[重編國語辭典修訂本|重編國語辭典修訂本]]》，2015年11月[[臺灣學術網路|臺灣學術網路]]第五版試用版，[[中華民國|中華民國]][[教育部國語推行委員會|教育部國語推行委員會]]</ref>，緣起于[[西漢|西漢]][[漢武帝|漢武帝]]時期，后来[[朝鲜_(称谓)|朝鮮]][[新羅|新羅]]在6世紀、[[日本|日本]]在7世纪后期、[[越南|越南]]在10世纪都因为中国的影响，开始使用年号<ref>{{Citation| author=朱勇 | contribution=中国法律的传统与现代化| title= ‘93 中国法律史国际硏讨会论文集| editor=张晋藩| publisher=中国民主法制出版社| place=| pages=25| date=1996| year=| id= | contribution-url=http://books.google.cn/books?id=EOy4AAAAIAAJ&q=%E5%B9%B4%E5%8F%B7&dq=%E5%B9%B4%E5%8F%B7&lr=&cd=12| format=| accessdate=2010-01-29 }}</ref>；近代[[大韩帝国|大韩帝国]]时期的朝鲜以及[[高丽|高丽]]、[[蒙古国|蒙古国]]建国初年受到中国影响，都還使用过年号，目前仍使用年号的是仍保持[[君主制|君主制]]的日本。[[中國|中國]]、[[朝鮮半島|朝鮮半島]]和[[越南|越南]]因為已經廢除君主制而全面廢止年號。

值得一提，[[中華民國|中華民國]]所用的[[民國紀年|民國紀年]]、以及[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]使用的[[主体纪年|主体纪年]]，常被误认为是年号，實際上僅是單純的紀年[[曆法|曆法]]。

== 中國 ==
{{main|中国年号列表}}
=== 中国歷史上的年号起源 ===
{{see|前元|中元|後元}}
[[File:漢武帝.jpg|thumb]]
在[[中国歷史|中国歷史]]上，第一个年号出现在[[西汉|西汉]][[汉武帝|汉武帝]]时期，年号为[[建元_(西汉)|建元]]（前140年—前135年）。此前的帝王只有年数，没有年号。據[[清朝|清朝]][[趙翼|趙翼]]的《[[二十二史劄記|二十二史劄記]]》考證<ref>《廿二史札记》卷2：是帝至元狩始建年号。从前之建元、元光等号，乃元狩后重制嘉号，追纪其岁年也。不然，则武帝六年，即应云建元六年，其下所云明年又明年，皆可书元光几年、元朔几年，岂不简易明白？而乃云明年后年耶！</ref>，年號紀年是在漢武帝十九年首創的，年號為「元狩」。《漢書》上記載說，前122年十月，漢武帝出去狩獵，捉到一隻獨角獸白麟，群臣認為這是吉祥的神物，值得紀念，建議用來記年，於是立年號為「[[元狩|元狩]]」，稱該年為元狩元年，并追認[[元狩|元狩]]前的年號[[建元_(西汉)|建元]]、[[元光_(汉朝)|元光]]和[[元朔|元朔]]<ref>《漢書·郊祀志》：「有司言元宜以天瑞命，不宜以一二數。一元曰『建』，二元以長星曰『光』，三元以日月復始曰『朔』，四元以郊得一角獸曰『狩』云」</ref>。可是，過了六年，又在山西汾陽地方獲得一只三個腳的寶鼎，群臣又認為這是吉祥的神物，建議用來紀年，於是改年號為「[[元鼎|元鼎]]」，稱那年為元鼎元年，後來，人們把這記錄年代的開始之年稱为「[[紀元|紀元]]」，改換年號（或[[帝王紀年|帝王紀年]]時代改稱元年）叫做「[[改元|改元]]」。首實行改元（改稱元年）者為[[漢文帝|漢文帝]]，但未取年号。此后，每次新皇帝登基，常常会改元，并同时改变年号。一般改元从下诏的第2年算起，也有一些从本年年中算起。

辛德勇《改订西汉新莽纪年表》以為，[[太初_(西漢)|太初]]改元之初可能還沒使用年號紀年，而是在五月與改正朔、易服色的改制措施同時實行。

=== 改元 ===
一个皇帝在位时，也可以多次改元。[[明朝|明朝]]以前的皇帝多数都改元两次以上，一个皇帝的年号也可以有多个。例如[[汉武帝|汉武帝]]有11个年号，[[武则天|武则天]]在位21年有18个年号。也有皇帝在即位时使用前一代皇帝的年号，例如[[五代|五代]]时期[[后梁|后梁]]的“-{[[乾化|乾化]]}-”年号、[[后晋|后晋]]的“天福”年号、[[后周|后周]]的“显德”年号。明朝以后採用[[一世一元制|一世一元制]]，扣除[[復辟|復辟]][[政變|政變]]者（如[[明英宗|明英宗]]），大致上都是一个皇帝只用一个年号，因此也常常用年号来称呼皇帝，例如[[嘉靖皇帝|嘉靖皇帝]]、[[崇祯皇帝|崇祯皇帝]]、[[康熙皇帝|康熙皇帝]]等。

=== 年号的使用 ===
年号被认为是帝王正统的标志，称为“奉正朔”。一个政权使用另一个政权的年号，被认为是藩属、臣服的标志之一。这种现象主要发生在中国分裂的时期。[[五代十国|五代十国]]时，[[闽国|闽国]]、[[楚国|楚国]]使用后梁、[[后唐|后唐]]年号，[[吴越国|吴越国]]使用[[唐|唐]]、后梁、后唐、后晋、[[后汉|后汉]]、后周和[[北宋|北宋]]的年号，另外[[朝鮮王朝|朝鮮]]、[[琉球|琉球]]為中國藩屬時，亦用中國帝王年號。也因此，许多地方割据势力、少数民族政权，以及人民起义也常常自立年号纪年。

中国年号的使用情况非常复杂。同一时期并存的政权，往往各有年号。还有的政权一年之中数次改元，几个年号重叠使用。也有政权自己不建年号，而沿用前朝或其他政权的年号。例如[[后晋|后晋]]的[[天福_(后晋)|天福]]年号用至九年，改为[[开运_(后晋)|开运]]元年。3年后，[[后汉|后汉]][[刘知远|刘知远]]称帝，不自建年号，也不沿用开运年号，而是追承天福十二年。还有许多年号在不同时期重复使用。例如[[建元|建元]]就有5个时期在使用。还有因为[[避讳|避讳]]或者其他原因，一个年号有不同写法，例如[[李重茂|唐殇帝]]的[[唐隆|唐隆]]年号，又写作[[唐元|唐元]]、[[唐安|唐安]]、[[唐兴|唐兴]]。

有些年号是根据两个前代年号合併而成，如[[贞元_(唐朝)|贞元]]取自[[贞观_(唐朝)|贞观]][[开元|开元]]，[[隆兴|隆兴]]取自[[建隆|建隆]][[绍兴|绍兴]]，[[淳熙|淳熙]]取自[[淳化|淳化]][[雍熙|雍熙]]，[[绍熙|绍熙]]取自[[绍兴|绍兴]][[淳熙|淳熙]]，[[庆元_(南宋)|庆元]]取法[[慶曆|慶曆]][[元祐_(年号)|元祐]]、[[开禧|开禧]]取自[[开宝|开宝]][[天禧_(北宋)|天禧]]等{{fn|1}}。

=== 年号的废止 ===
[[辛亥革命|辛亥革命]]后，[[中华民国|中华民国]]废除年号纪年的做法，而改用[[民国纪年|民国纪年]]。虽然在[[袁世凯|袁世凯]][[洪宪称帝|称帝]]时使用过“[[洪宪|洪宪]]”的年号，而[[愛新覺羅·溥儀|愛新覺羅·溥儀]]在擔任[[滿洲國|滿洲國]]執政和[[皇帝|皇帝]]時分別使用過[[大同_(满洲国)|大同]]和[[康德_(年号)|康德]]的年號，但是通常不为中国正统史书所承认，而認爲[[中国皇帝|中国皇帝]]的最后一个年号为清末的“[[宣统|宣统]]”。

[[中华人民共和国|中华人民共和国]]成立后，使用西方[[公元纪年|公元纪年]]。而也有少数人提出为了传承中华传统文化，丰富世界纪年文化多样性，保护中国[[非物质文化遗产|非物质文化遗产]]而恢复年号的建议<ref>{{cite journal|title=让纪元再现中国元素——关于恢复年号的倡议|journal=社会科学论坛：B版|date=2007|author=周到|volume=|issue=9|pages=143-151|id= |url=|format=|accessdate= }}</ref>。

== 臺灣 ==
{{main|台灣年號列表}}
臺灣在[[明郑时期|明郑时期]]時雖然[[南明|南明]][[永历帝|永历帝]]已死，但仍繼續使用[[永历_(南明)|永历]]年号。在清朝統治時期則使用清朝各皇帝之年號，而在一些民變事件中，起事領袖（例如[[朱一貴|朱一貴]]、[[林爽文|林爽文]]等）曾經使用過自己的年號。[[甲午战争|甲午战争]]后，反对日本占领臺湾的[[臺灣民主國|臺灣民主國]]則一度以[[永清_(年号)|永清]]為年號，而在日治時期則使用日本天皇的年號（依序是[[明治|明治]]、[[大正|大正]]、[[昭和|昭和]]）。

== 蒙古 ==
蒙古高原在[[5世纪|5世纪]][[柔然|柔然]]政权时，一度使用中国式的年号。[[蒙古帝国|蒙古帝国]]在[[1260年|1260年]][[忽必烈|忽必烈]]即位后，建元[[中統_(年號)|中统]]，随后建立了统治全中国的[[元朝|元朝]]。年号一直沿用到[[北元|北元]]时代。[[1388年|1388年]]之后，蒙古年号不可考，仅知[[瓦剌|瓦剌]][[也先|也先]]称汗后，改元[[添元|添元]]。清朝灭亡后，[[博克多格根|博克多格根]]在撤销独立前使用[[共戴|共戴]]年号。

== 朝鮮 ==
{{main|朝鮮半島年號列表}}
朝鮮是中國國外第一個使用年號的地方，[[新羅國|新羅國]]君主[[新羅法興王|新羅法興王]]使用的年號[[建元_(新罗法兴王)|建元]]（건원）為[[朝鮮半島歷史|朝鮮半島歷史]]上首個年號，與[[中國歷史|中國歷史]]上第一個年號一樣（漢朝漢武帝的[[建元_(西汉)|建元]]）。但相比[[日本|日本]]和[[越南|越南]]兩國，朝鮮半島自創的年號比較少，主要是因為朝鮮自[[新羅|新羅]][[真德王|真德王]]四年（[[650年|650年]]）至[[大韓帝國|大韓帝國]]成立（[[1897年|1897年]]）之間的一千二百餘年內，都是中國的附屬國，採用的都是中國年號。

== 日本 ==
{{main|日本年号列表}}
[[孝德天皇|孝德天皇]]接觸[[唐朝|唐朝]]之後，開始效仿中國，於[[646年|646年]]使用年號。[[大宝_(日本)|大寶]]元年（[[701年|701年]]）開始正式使用年號，直到今日。

另外日本天皇的年號的特色之一就是部分的單一年號可跨越前後兩名在位天皇。日本在[[明治維新|明治維新]]之前跟中國[[明朝|明朝]]以前一樣，一位天皇可有多個年號，而在明治維新後就跟中國[[明朝|明]][[清朝|清]]時代的君主一樣，一位天皇一個年號。

== 越南 ==
{{main|越南年号列表}}
[[越南|越南]][[丁部領|丁部領]]於[[968年|968年]]建立[[丁朝|丁朝]]，[[970年|970年]]開始建元[[太平_(丁朝)|太平]]，直到[[1945年|1945年]][[阮朝|阮朝]]滅亡，廢止[[保大|保大]]年號。
