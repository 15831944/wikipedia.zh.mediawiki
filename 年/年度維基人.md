'''年度維基人'''（{{Lang-en|Wikipedian of the Year}}）又稱作'''神保獎'''（Jimbo Award），是[[維基百科|維基百科]]創辦人[[吉米·威爾斯|吉米·威爾斯]]在2011年設立的年度獎項<ref name = "Kevin Morris 01">{{Cite web |url = https://www.dailydot.com/news/wikipedian-of-the-year-jimmy-wales-prize-money/ |title = Winners of Wikipedia’s biggest award still haven’t received prize money |author = Kevin Morris |date = 2013年4月27日 |publisher = 《{{Link-en|每日點擊|The Daily Dot}}》 |language = en |accessdate = 2018年4月13日 |quote =  }}</ref>。該獎項依慣例在每年的[[維基媒體國際會議|維基媒體國際會議]]上頒發，用來凸顯在[[維基媒體計劃|維基媒體計劃]]中有著重大成就的個別[[維基人|維基人]]。

年度維基人的獲獎者包括有：{{Link-en|拉文·肯哲克罕努利|Rauan Kenzhekhanuly}}、「丹米」（Demmy）、[[雷米·馬蒂斯|雷米·馬蒂斯]]、[[伊霍爾·柯斯登柯|伊霍爾·柯斯登柯]]、[[埃米莉·坦普爾-伍德|埃米莉·坦普爾-伍德]]、{{Link-en|羅西·斯蒂芬森-戈德奈特|Rosie Stephenson-Goodknight}}、{{Link-en|費利斯·納提|Felix Nartey}}、以及一名「{{Link-en|默存心中|In pectore}}」的匿名編輯。而除了主要獎項外，威爾斯還曾授予5人「榮譽獎勵」。

== 獎項 ==
[[File:Wikimania_2014_closing_ceremony08.jpg|thumb]]在[[維基媒體國際會議|維基媒體國際會議]]閉幕典禮上宣布[[伊霍爾·柯斯登柯|伊霍爾·柯斯登柯]]為年度維基人。]]

最初的年度維基人獎項並不存在正式的意義、亦未公布贏得獎項的標準，它實際上也不是[[維基百科|維基百科]]或[[維基媒體計劃|維基媒體計劃]]正式主辦的獎項，完全是由維基百科共同創辦人、[[維基媒體基金會|維基媒體基金會]]理事[[吉米·威爾斯|吉米·威爾斯]]自行決定獲獎人選<ref name = "Kevin Morris 01"/><ref name = "Iryna Savchuk 01">{{Cite web |url = https://www.kyivpost.com/article/guide/people/lifestyle-blog-ukrainian-from-the-heavenly-hundred-became-a-wikipedian-of-the-year-360551.html |title = Lifestyle Blog: Ukrainian from the Heavenly Hundred becomes Wikipedian of the Year |author = Iryna Savchuk |date = 2014年8月15日 |publisher = 《{{Link-en|基輔郵報|Kyiv Post}}》 |language = en |accessdate = 2018年4月13日 |quote =  }}</ref>。最初威爾斯曾表示每位得主能獲得5,000美元，但並未有所下文<ref name = "Kevin Morris 01"/>。而除了主要獎項外，威爾斯還會授予「榮譽獎勵」<ref name="Joe Sutherland 01">{{Cite web |url = https://blog.wikimedia.org/2015/07/31/wikipedians-of-the-year-2015/ |title = 2015 Wikipedians of the Year unveiled in Mexico |author = Joe Sutherland |date = 2015年7月31日 |publisher = [[維基媒體基金會|維基媒體基金會]] |language = en |accessdate = 2018年4月13日 |quote =  |deadurl = no |archiveurl = http://archive.wikiwix.com/cache/20160625170502/https://blog.wikimedia.org/2015/07/31/wikipedians-of-the-year-2015/ |archivedate = 2016年6月25日 }}</ref>。

2011年，威爾斯在該年度的[[維基媒體國際會議|維基媒體國際會議]]的「維基現況」（State of Wiki）演講中，首次宣布這個獎項；最初該獎項被稱作「全球年度維基人」（Global Wikipedian of the Year）<ref>{{Cite web |url = https://blog.wikimedia.org/2011/09/08/wikimedia-foundation-report-august-2011/ |title = Wikimedia Foundation Report, August 2011 |author = Tilman Bayer |date = 2011年9月8日 |publisher = [[維基媒體基金會|維基媒體基金會]] |language = en |accessdate = 2018年4月13日 |quote =  }}</ref>。不過首屆年度維基人得主{{Link-en|拉文·肯哲克罕努利|Rauan Kenzhekhanuly}}在獲獎時，並不知道此事<ref name = "Kevin Morris 01"/>。而在2012年維基媒體國際會議上，該獎項另外一個名稱「神保獎」（Jimbo Award）首次提出<ref name = "Kevin Morris 01"/>。

儘管在肯哲克罕努利得獎時，年度維基人獎項曾經以新聞稿和報紙文章宣傳該新聞，《{{Link-en|每日點擊|The Daily Dot}}》仍形容年度維基人是網際網路上陌生的獎項之一，且得主並不一定會認識該獎項<ref name = "Kevin Morris 01"/>。不過該獎項的宣布往後成為年度維基媒體國際會議閉幕演講的慣例，後者是世界各地的[[維基媒體運動|維基媒體運動]]編輯每年參與的大型會議<ref name = "Joe Sutherland 01"/>。
