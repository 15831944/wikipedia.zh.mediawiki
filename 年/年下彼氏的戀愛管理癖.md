《'''年下彼氏的戀愛管理癖'''》（{{lang-ja|年下彼氏の恋愛管理癖}}）是[[日本漫畫家|日本漫畫家]][[櫻日梯子|櫻日梯子]]所繪的[[日本漫畫|日本漫畫]]作品。2012年10月於[[竹書房|竹書房]]的漫畫雜誌《Qpa》上首次刊載<ref>{{Cite book|title=桜日梯子FAN BOOK|last=|first=|publisher=東立出版社|year=2017|isbn=978-9-8647-0198-8|location=台灣|pages=156}}</ref>，後來移至[[Libre出版社|Libre出版社]]的雜誌《MAGAZINE BE×BOY》連載，並在2017年11月重新出版新裝版漫畫單行本。<ref>{{cite web|url=https://natalie.mu/comic/news/256271|title=桜日梯子「年下彼氏の恋愛管理癖」が新装版で登場、スピンオフも同時収録|language=ja|date=2017年11月10日}}</ref>本作也是櫻日梯子的首部出版漫畫書籍，與《請把手給我，王子殿下♥》、《獻給你的施虐傾向》有[[共同世界|共同世界]]。

此作獲得2015年日本書店店員票選《BL漫畫排行榜》第二名<ref>{{Cite web|url=https://news.gamme.com.tw/1080267|title=日本書店店員票選《BL漫畫排行榜》你要年下攻還是戴著兔子頭的攻？|language=zh|date=2015-03-03}}</ref>

== 劇情簡介 ==
阿理是一名研究生，時常埋首於研究之中且會認真到渾然忘我的境界。而阿泉則是阿理的學弟，同樣在研究室裡，是公認同時也是私認的阿理學長的「飼主」，負起照顧阿理學長的責任。原本只是單純地照顧著阿理學長，但阿泉在知道阿理學長被別人上下其手之後，心中便對阿理學長產生了不同的感覺……

== 登場人物 ==
<div class="notice metadata" id="spoiler"><small>'''[[聲優|聲優]]：廣播劇CD'''</small></div>

'''{{nihongo|小野塚理|小野塚 理（おのづか さとし）|SATOSHI ONOZUKA|聲：[[平川大輔|平川大輔]]}}'''<ref name=":0">{{Cite web|url=https://wikiwiki.jp/blcd/%E5%B9%B4%E4%B8%8B%E5%BD%BC%E6%B0%8F%E3%81%AE%E6%81%8B%E6%84%9B%E7%AE%A1%E7%90%86%E7%99%96|title=年下彼氏の恋愛管理癖|accessdate=2019-01-28|author=|date=2018-09-18|publisher=}}</ref>
: 研究生，基本上是個純真的人，但偶爾老舖旅館兒子身上流的商人血液會讓他展現出貪心的一面。會在把熱水倒進泡麵碗後等待泡好的時間中睡著，老是把麵泡爛。喜歡吃鮟鱇魚肝跟蟹膏（喝酒時）。常常被阿泉髮根的黑色部分萌到。日常受阿泉的照顧，自己也說「多虧有阿泉照顧，才有辦法平安地活到今天。」很不會剝三角飯糰的包裝。

'''{{nihongo|平坂泉|平坂 泉（ひらさか いずみ）|IZUMI HIRASAKA|聲：[[前野智昭|前野智昭]]}}'''<ref name=":0" />
: 大四，胸懷「男人就該冷酷又精明」而努力的[[橫濱|橫濱]]人☆被哥哥惡作劇染了頭髮後，反而覺得金髮跟自己很搭，就一直維持金髮的造型。外表的輕浮被他認真的生活中和不少。AB型巨蟹座。很會照顧阿理，常常給熬夜的阿理帶「飼料」，被西島教授稱為是阿理的飼主。

'''{{nihongo|安生高夫|安生 高夫||聲：[[森川智之|森川智之]]}}'''

: 穿著條紋襯衫，青春偶像劇客座教授。本業是為企業服務的建築顧問。不同於外表的輕浮，在工作上相當有原則，且備受肯定。應該說，他本人的個性就是個認真、超級認真的乖小孩。對阿理有興趣。在青春期遭受感情上的創傷。AB型巨蟹座。和某H坂同學（橫濱人）一樣。

'''{{nihongo|西島拓朗|西島 拓朗||聲：[[小池謙一|小池謙一]]}}'''

: 環境設計研究室的主人。擁有近似動物的直覺，能洞察人心。在阿泉本人發現前就發現「啊──這傢伙愛上我們家的拉車馬了。」發現後還是放任不管，偶爾以在旁煽風點火為樂。

'''小野塚夫婦：{{nihongo|小野塚弘文|小野塚 弘文||聲：[[各務立基|各務立基]]}}、{{nihongo|小野塚稔|小野塚 稔||聲：[[まつだ志緒理|まつだ志緒理]]}}'''

: 弘文是到旅館取材的攝影師助手，在阿稔對他一見鍾情，展開熱烈追求後，兩人共結連理，他就以養子名義入贅了。由於他能夠適時制止豬突猛進的阿稔，小野塚家的每個人都相當感謝他。

'''{{nihongo|百崎小梅|百崎 小梅||聲：[[吉田麻実|吉田麻実]]}}'''

: 大四，跟阿泉在同一間居住研討室。從[[和歌山县|和歌山]]來[[东京|東京]]念書。正在追求打工咖啡廳內的店主。和在東京的高中念書的弟弟兩人住在一起。兩姊弟都同樣戴黑框眼鏡。弟弟：林太郎，對興趣相當狂熱。

'''{{nihongo|百崎林太郎|百崎 林太郎||}}'''

: 高三，文藝系文學少年，喜歡的作家是團⚫六、谷崎⚫一郎、寺山⚫司等等。喜歡的A片是昭和的浪漫情色系列。看過和歌山老家倉庫中的春畫收藏後便喜歡上日本畫。尊敬的畫家是伊藤⚫雨。喜歡的類型是「心有所屬的人」。疼愛可悲的人讓他覺得很開心。雖然會看氣氛，卻會徹底無視當下的氛圍。常會找超市櫃檯收銀員這類的打工工作，因為能遇到比較多的人妻。最近發現了新的獵物。

'''{{nihongo|太田幹久|太田 幹久||}}'''

: 居住學系。小木研究室成員，阿泉的學弟。是在寬廣的北海道大地成長茁壯的男人。天生運氣不佳，每件他認為是好的事情，最後都會反過來咬自己一口。活得非常努力。

'''{{nihongo|小木文子|小木 文子||}}'''

: 居住研討室的負責老師。是位有氣質的美麗熟女。雖然年輕時遊戲人間，現在也是三個男孩的母親了。
