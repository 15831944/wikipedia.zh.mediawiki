'''永隆里'''，為[[臺灣|臺灣]][[臺中市|臺中市]][[大里區|大里區]]轄下的一個里<ref>{{cite book|title=臺灣全志: 政治志·建置沿革篇|year=2007|publisher=國史館臺灣文獻館}}</ref>，位於大里區西北側，為1990年代發展的市地重劃區。

== 里名由來 ==
為求「永遠興隆」，故命名為永隆里。

== 地理 ==
永隆里劃分為24鄰，3577戶，總人口11367人（台中縣大里市戶政事務所民國101年10月底戶口統計）。本里東隔[[國光路|國光路]]與[[大明里_(大里區)|大明]]、[[東興里_(大里區)|東興]]二里為界，西為[[樹王里|樹王里]]，南以[[永隆二街|永隆二街]]臨[[新里里|新里里]]，北隔[[旱溪|旱溪]]與[[南區_(臺中市)|南區]]對望。

== 行政區劃沿革 ==
東興里於清代時隸屬藍興堡涼傘樹莊；明治35年（1902年）時，改隸臺中廳樹仔腳區涼傘樹庄，至大正9年（1920年）則隨行政區調整，隸屬大屯郡大里庄涼傘樹大字。二次大戰結束後，國民政府接收臺灣，原先涼傘樹大字範圍被劃歸為'''樹王村'''與'''鷺村'''，隸屬臺中縣大里鄉。民國59年（1970年），為因應眾多的外來人口，鷺村被劃分為'''祥鷺村'''和'''東興村'''；民國82年（1993年）隨大里鄉升格為縣轄市'''大里市'''而改制為'''東興里'''。民國91年（2002年），行政區再次調整，析東興里西側置「'''永隆里'''」。
