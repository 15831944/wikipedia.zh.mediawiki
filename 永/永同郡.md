'''永同郡'''（{{kor|諺=영동군|漢=永同郡|羅=Yeongdong gun|f=y}}）[[大韩民国|大韩民国]][[忠清北道|忠清北道]]的一个郡。其英文名称与[[嶺東地方|岭东地区]]易混淆。南部与[[全羅北道|全羅北道]]、東部与[[慶尚北道|慶尚北道]]、西部与[[忠清南道|忠清南道]]接壤。盛产[[柿子|柿子]]、[[葡萄|葡萄]]、[[苹果|苹果]]、[[梨|梨]]等水果。

[[朝鲜战争|朝鲜战争]]期间，此地曾发生[[老斤里事件|老斤里事件]]。

== 象征 ==
郡鸟：[[鸽子|鸽子]]。郡花：[[金达莱|金达莱]]。郡树：[[柿树|柿树]]

== 历史 ==
1940.11.01根据府令第221号，永同面升级为邑，永同郡管辖1个邑和10个面至今。

1991.07.01根据郡条例第1332号，黄金面改称秋风岭面。

== 地方行政单位 ==
永同邑与龍山、深川、楊江、陽山、龍化、鶴山、黄澗、梅谷、上村等10个面。

== 教育 ==
大学：[[永同大学校|永同大学校]]。有多所高中初中。

== 交通 ==
鉄道有[[京釜線|京釜線]]高速公路有[[京釜高速国道|京釜高速国道]]。
