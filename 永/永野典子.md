'''永野典子'''（{{lang-ja|永野のりこ}}，{{bd||7月5日}}）<ref>{{cite web|url=https://dictionary.goo.ne.jp/word/%E6%B0%B8%E9%87%8E%E3%81%AE%E3%82%8A%E3%81%93/|title=永野のりこ(ながののりこ)の解説 - goo人名事典|website=goo辞書}}</ref>，[[日本漫畫家|日本漫畫家]]，早期筆名為「{{lang|ja|永野のり子}}」。

== 經歷 ==
[[法政大學_(日本)|法政大學]][[文學|文學]]部[[哲學|哲學]]科畢業<ref>{{cite web|url=https://www.minkou.jp/university/school/senior/20287/|title=法政大学出身の有名人|website=みんなの大学情報}}</ref>，大學時曾加入漫畫研究會。結婚生子後，永野為了增加家中收入<ref name="【永野先生への30の質問】">{{cite web|url=http://www.manga-gai.net/column/shitumon/shitumon_nagano.html|script-title=ja:【永野先生への30の質問】 }}</ref>，持稿拜訪出版社，獲[[德間書店|德間書店]]錄用，1985年以《Sci-Fiもーしょん!》出道<ref name="【永野先生への30の質問】"/>。代表作為《GOD SAVE THEすげこまくん！》。

2018年榮獲第49屆[[星雲賞|星雲賞]]的美術部門賞<ref>{{cite web|url=http://www.sf-fan.gr.jp/awards/2018result.html|title=2018年 第49回星雲賞|website=www.sf-fan.gr.jp}}</ref>

== 作品 ==
永野典子目前唯一有正式授權中文譯本的作品是《怪獸聖天使》。
