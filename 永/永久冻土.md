'''永冻土'''（[[英語|英語]]：{{lang|en|'''Permafrost'''}}，又譯'''永冻层'''、'''永冻土层'''或'''多年冻土'''）是一個[[地質學|地質學]]的名詞，指當[[凍土層|凍土層]]（frost soil）處於水的[[熔点|結冰點]]以下超過兩年的狀況。

== 結構 ==
永冻层一般分布在地下30~40公分处，通常又分为上下两层，上层夏季融化，下层仍处于冰冻状态。

永久凍土地帶的泥土不一定有水份，例如在[[无孔基岩|无孔基岩]]（不透水的岩石）裡就不可能有水；但在大多數情況下，水不單存在，而且更超過地表物料的[[潜在水饱和量|潜在水饱和量]]。

一般的[[凍土層|凍土層]]在氣候回暖或受到強壓時，[[凍土|凍土]]內的冰會溶解成為水；但永久凍土的所在之處，即使在天氣回暖之時，氣溫仍然在[[熔点|結冰點]]以下，使凍土內的冰不能再次溶化成為水份，因而使[[凍土的組成|凍土的組成]]不變。其持续冰冻时间可长达1000年以上。

== 形成的兩個必要條件 ==
* [[气温|气温]]要很低，如果[[年平均气温|年平均气温]]高于零度，虽然可以形成[[季节性冻土|季节性冻土]]，却无法形成永久冻土。
* 持续时间较长。现在发现，在[[北极|北极地区]][[冻土层|冻土层]]一般都有几百米的厚度<ref>永冻层的厚度随纬度的增高而增厚，随纬度的降低而逐渐消融，通常以北纬48度为地理分界线。</ref>，而[[大气|大气]]的低温要传导到地表以下几百米的深度，经过几百、几千年甚至更长的时间才能完成。

永久凍土地帶的水份長期結凍，一般植物難以生長。大多數永久凍土都位於[[高緯度|高緯度]]的地區<ref>一般存在于北纬48度以上的地区。</ref>，例如：[[北極|北極]]和[[南極|南極]]附近的陸土，唯一的例外是位於[[青藏高原|青藏高原]]的[[凍土帶|凍土帶]]，是因為高度而使土地變成永久冻土。

由於氣候的轉變，永久凍土的範圍亦會隨氣候而有所增減。現時地球上有20%的土地被凍土覆蓋。這些凍土，有連綿不斷的[[永久凍土地帶|永久凍土地帶]]，亦有孤立的[[不連續永久凍土|不連續永久凍土]]，以及被[[冰河|冰河]]所覆蓋的地方。

[[冻土层|冻土层]]上复是所谓[[活动层|活动层]][[季节性融化|季节性融化]]，在夏季，唯一可以在活动层内支持植物生命，可以完全融化了一部分的一年。活动层厚度不一，但一般年份和地点0.6-4米（212英尺）厚。在[[连续多年冻土|连续多年冻土]]区和[[多年冻土|多年冻土]]深度严酷的冬天厚达440米（1330英尺）。[[阿拉斯加|阿拉斯加]]最厚达726米（2382英尺）。在[[加拿大|加拿大]][[北极群岛|北极群岛]]和高达1493米（4510英尺）在北部[[莱娜|莱娜]]和[[雅娜|雅娜]]在[[西伯利亚|西伯利亚]]流域。

== 影響 ==
永冻层对环境可产生有益的影响，如分解[[二氧化碳|二氧化碳]]，减少二氧化碳的排放等。
