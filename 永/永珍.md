{{地區用詞3|zh-tw='''永珍'''|equiv-hk=tw|equiv-sg=tw|equiv=mo=tw|start=（{{老撾語|lao=ວຽງຈັນ|latin=Vieng Chan}}，|zh-cn='''万象'''}}，[[老挝人民民主共和國|老挝人民民主共和國]]的首都和最大城市。以前叫文單或雍田，即是[[撣族|撣族]]之城。在[[塞塔提腊|塞塔提腊]]王时成为老挝首都。

== 含義 ==
ວຽງຈັນ[[拉丁轉寫|拉丁轉寫]]為Vieng Chan, [[國際音標|國際音標]][ʋíəŋ tɕàn]，[[法國|法國]]殖民者按[[法語|法語]]習慣拼寫成Vientiane。ວຽງ Vieng[ʋíəŋ]是“有城墙的城”的意思。因文字简化的关系，ຈັນChan[tɕàn]在今天老撾語中通常的意思是[[月亮|月亮]]，所以一些人理解為“月城”，但其实是老挝文字简化带来的误解。ຈັນChan[tɕàn]在老撾語中還有一個意思是[[檀木|檀木]]，与老撾文相似的泰文沒有文字簡化，在泰文中，月亮จันทร์和檀木จันทน์拼法不同，根据泰文เวียงจันทน์和柬埔寨文វៀងច័ន្ទន៍的拼法，可判定ວຽງຈັນ Vieng Chan的本意是檀城。此外，Chan来自梵语“candanam”中国古书中汉译为“栴檀”，而垣在汉语中指墙，也可以指城，因此有学者将这一城市的名称翻译为“垣栴”。<ref>{{cite web|author1=杨舟|title=调查-老挝应该读“老抓”？|url=http://news.xinhuanet.com/world/2015-07/20/c_128038729.htm|publisher=新华网|accessdate=2017-10-15|ref=1}}</ref>

== 歷史 ==
永珍早於公元前4世纪已被建立。自公元14世纪[[瀾滄王國|瀾滄王國]]立國以來，永珍在十六世纪是寮國的首都和經濟中心（之前是[[瑯勃拉邦|瑯勃拉邦]]为避[[缅甸|缅甸]]遷都）。

當1707年瀾滄王國滅亡，永珍變成寮國其中一個分裂小國。1779年[[暹羅|暹羅]]入侵寮國，永珍變成暹羅之附庸。

19世紀初永珍叛亂，卻被暹羅平定，永珍被夷為平地，所有文物包括佛像，皆被洗劫一空。

20世紀初永珍被到來之[[法國|法國]]人修繕，成為法國殖民時代之首府。至二次大戰後寮國獨立，永珍依舊是寮國首都。

== 地理 ==
永珍是全球少數設於國界附近的首都之一，位於[[湄公河|湄公河]]中游，距离湄公河河口不到1600公里，大致於北緯17'57°，東經102'36°，由一湄公河沙洲相隔著寮國和[[泰國|泰國]]。屬[[熱帶季風氣候|熱帶季風氣候]]，雖然如此，但在冬天的氣溫因[[地勢|地勢]]關係顯得相當極端，最低氣溫可低至攝氏12度，但日間可高至攝氏31度，特徵與[[熱帶荒漠氣候|熱帶荒漠氣候]]相似。

永珍是寮国的政治、经济、交通中心。市中心区域約70萬人（2005年），包含週邊區域約73萬人。市区集中了面向外国人的游客中心。

== 對外交通 ==
; 航空
* 擁有一個由日本人興建及管理的[[瓦岱國際機場|瓦岱國際機場]]，距離市中心的西部約3公里，它設有現代化的航站大樓及一條10,499英呎長的跑道，跟[[昆明|昆明]]、[[河内|河内]]、[[曼谷|曼谷]]、[[清邁|清邁]]、[[吉隆坡|吉隆坡]]、[[新加坡|新加坡]]、[[广州|广州]]等地點通航。
; 鐵路
* 使用[[泰國國家鐵路|泰國國家鐵路]]的「東北線」的「[[廊開|廊開]]」（Nong Khai）車站，能夠連通[[泰國|泰國]]、[[馬來西亞|馬來西亞]]及[[新加坡|新加坡]]。
; 公路
*距离万象市中心东南方24公里处有一座横跨[[湄公河|湄公河]]通往[[泰國|泰國]][[廊開|廊開]]的友谊桥（Friendship Bridge），能够通过公路直接到达[[曼谷|曼谷]]，每天均有往来于泰国和老挝的游客从这里进出关口。

== 教育 ==
