'''永康工業區車站'''（暫訂名稱，亦被稱為「車行車站」）位於[[臺灣|臺灣]][[臺南市|臺南市]][[永康區|永康區]]，是一個配合[[臺鐵捷運化|臺鐵捷運化]]政策，而計劃於[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]][[新市車站|新市車站]]與[[永康車站|永康車站]]間增設的通勤[[鐵路車站|鐵路車站]]。

==位置==
*本站預定位置在[[臺南市|臺南市]][[永康區|永康區]]。
*緊臨[[File:TW_PHW1.svg|23px]][[台1線|台1線]]及永康工業區、永康科技園區、永康區體一運動公園

== 車站構造 ==
*簡易通勤站。

== 車站樓層 ==
== 利用狀況 ==
*主要停靠[[臺鐵區間車|區間車]]。
==進度==
*尚未排定興建期程。
== 車站周邊 ==
永康區體一運動公園、永康工業區、永康科技園區
