'''永久居留權'''指個人得到許可永久居留於某地的[[權利|權利]]，但不一定享有[[公民|公民權]]。擁有永久居留權的人稱作「'''永久居民'''」。由于[[美国永久居民|美国永久居民]]的身份证称为「[[綠卡|綠卡]]」，故绿卡在不少地區成為永久居留許可證的代名詞。

==臺灣==
[[臺灣|臺灣]]，政治與歷史情況特殊，1895年，[[大清|大清]]割讓台灣給日本，台灣人在[[台灣日治時期|日治時期]]（[[第二次世界大戰|第二次世界大戰]]結束前），屬於日本國籍，與舊日本法律相同。而在二次世界大戰後有了新的制度。二戰日本戰敗，台灣由中華民國接管。[[1945年|1945年]][[8月14日|8月14日]]，日本[[昭和天皇|昭和天皇]]發表《[[終戰詔書|終戰詔書]]》接受[[波茨坦公告|波茨坦公告]]，[[大日本帝國|大日本帝國]]無條件投降，盟軍將日本帝國全部領土分區佔領，本土由[[同盟國軍事佔領日本|美國為首]]佔領，[[朝鮮半島|朝鮮半島]]由[[美國|美國]]及[[蘇聯|蘇聯]][[盟軍託管朝鮮時期|佔領]]，[[樺太廳|庫頁島南部]]及[[千島群島|千島群島]]由[[蘇聯|蘇聯]]佔領，[[臺灣|臺灣]]、[[澎湖|澎湖]]及[[新南群島|新南群島]](南沙群島)由[[盟軍最高統帥|盟軍最高統帥]][[麥克阿瑟|麥克阿瑟]]元帥發佈《[[一般命令第一號|一般命令第一號]]》，命令在臺日軍向[[蔣介石|蔣介石]]元帥投降，由蔣介石指派代表佔領。[[臺灣人|臺灣居民]]中，[[本省人|華裔]]及[[原住民|原住民]]被自動賦予[[中華民國|中華民國]][[國籍|國籍]]，其餘[[臺灣人|臺灣居民]]如設籍在臺灣之[[日本人|日本人]]、[[朝鮮人|朝鮮人]]、[[琉球人|琉球人]]不算在內。臺灣人民[[雙重國籍|雙重國籍]]，直至[[1951年|1951年]][[9月8日|9月8日]]，[[日本|日本]]與[[同盟國|同盟國]]簽署《[[舊金山和約|舊金山和約]]》，1952年4月28日生效。《舊金山和約》生效後，同盟國正式結束對[[日本本土|日本本土]]的軍事佔領，日本恢復原有的[[國家權力|國家權力]]及[[主權|主權]][[獨立|獨立]]地位。而原[[大日本帝國|大日本帝國]][[國民|國民]]的[[臺灣人|臺灣居民]]喪失日本籍，僅存[[中華民國|中華民國]][[國籍|國籍]]至今日。统治臺灣至今之[[中華民國|中華民國]]也允許外籍人士[[中華民國居留證|永久居留]]。但因[[兩岸關係|兩岸關係]]的歷史與政治現實，台灣地區和大陸地區人民在公民權與國籍問題情形特殊；中華民國政府對中華人民共和國政府管治下之大陸地區人民採用特殊制度與方法。

==香港及澳門==

[[香港|香港]]和[[澳門|澳門]]於[[香港回歸|1997年]]和[[澳門回歸|1999年]]主權移交至[[中华人民共和国|中华人民共和国]]後，實施了新的制度。於原殖民地“華裔人士”的居留權，被《[[中華人民共和國香港特別行政區基本法|香港基本法]]》、《[[中華人民共和國澳門特別行政區基本法|澳門基本法]]》確認。從而所有華裔人士或[[土生葡人|土生葡人]]不論意願均被賦予[[中華人民共和國國籍法|中國國籍]]。而非華裔人士（如英裔和葡裔）的居留權則改為了“永久性居民”的說法。事實上，該永久性居民只要是在當地通常連續居住，即可繼續保留。如該非中國籍永久居民不再以當地為永久居住地且並未通常連續居住，則居留權和永久居民身份喪失。但是，港澳地區的居留權被剝奪後依然還有“入境權”，即可以像永久居民那樣隨時入境。但永久居民不會因任何理由被遣送離境，而沒有居留權，僅有入境權的人士，可能因違法犯罪被遣送離境。而只需要在當地通常連續居住一段時間，就會重新獲得居留權。

非中國籍港澳居民，無法申請[[中華人民共和國香港特別行政區護照|香港]]和[[中華人民共和國澳門特別行政區護照|澳門]]的“特區護照”，通常也無法申請[[香港簽證身份書|香港簽證身份書]]（DI）和[[澳門電子旅行證|澳門電子旅行證]]。但無法申請本國證件的居民可以申請上述的DI和電子旅行證。上述兩種證件無法像特區護照一樣享受免簽證待遇。

== 權利 ==
 
永久居民與公民擁有的權利大致相同，除了以下：
*選出[[公職人員|公職人員]]的[[投票權|投票權]]（部分國家容許）
*出任[[公務員|公務員]]、參選公職人員（不常見）
*出任涉及國家安全的職位
*持有該國[[護照|護照]]（少部分国家允许）
*受[[領事館|領事館]]保護（部分國家容許）
*不被遣返的權利

== 義務 ==
永久居民有時需要符合一定條件才能取得永久居留權，例如從事某種行業，或經營特定生意。

部分國家要求永久居民服兵役。新加坡規定所有屆滿18歲的男性居民，包括永久居民及公民，都要服2年兵役。之後要每年參加軍營訓練及隨時候命，直至40歲。在美國，所有年齡介乎18及26歲的男性公民及永久居民，必須登記在軍隊徵召名冊內。此法規亦適用於非法居留者。<ref>http://www.sss.gov/QA.HTM#quest7 "Are illegal alien males required to register?," Selective Service System: Registration Information (accessed August 14, 2006)</ref>未能符合規定者或會被拒批出永久居民資格。

== 喪失資格 ==
永久居民可能會喪失資格，如果：
* 未能符合永久居留的法規和要求
* 犯罪而可能被遞解出境

== 成為公民 ==
永久居民在一段時間後可以申請[[歸化|歸化]]為公民，但必須符合一定條件。在美國，因婚姻而取得永久居留權的人，可在成為永久居民後3年申請歸化，但其他人必須為5年。曾參軍人士可以通過[[緊缺人才征兵計劃|緊缺人才征兵計劃]]1年後提出申請。<ref>http://www.uscis.gov/graphics/services/natz/general.htm</ref>

== 自動權利 ==
以下國家之間自動互頒永久居留權：
* [[愛爾蘭|愛爾蘭]]和[[英國|英國]]
* [[北歐理事會|北歐理事會]]各國及其[[自治领|自治领]]

[[歐盟|歐盟]]條例賦予的權利並不屬永久居留權，但兩者差別不大。而根據旅遊協議，澳洲和新西蘭的公民在對方國家可享近乎永久居住的權利。

== 證明 ==
過去，不少國家僅在護照上蓋印以茲證明。現時多數國家會向永久居民發出證明卡或信件，如[[美國|美國]]的[[綠卡|綠卡]]、[[加拿大|加拿大]]的[[枫叶卡|枫叶卡]]、[[香港|香港]]的[[香港身份證|永久性居民身份證]]、日本的[[櫻花卡|櫻花卡]]等<ref>{{cite web|title=雙重國籍的問題|url=http://tw.knowledge.yahoo.com/question/question?qid=1610092107043|accessdate=2012-05-21}}</ref> ，又或者在永久居民的護照上貼有居留簽證，如[[澳洲永久性居民|澳洲永久性居民簽證]]。

持有遺失證明文件或被盜證明文件在不少國家均屬違法。
