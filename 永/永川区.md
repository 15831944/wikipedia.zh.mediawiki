'''永川区'''是[[中国|中国]][[重庆市|重庆市]]下辖的一个[[市辖区|市辖区]]，为重庆西部的[[金融|金融]]、[[通讯|通讯]]、[[能源|能源]]中心。因城内三河汇流，形如篆文的“永”字，故取名永川。2006年10月22日经[[中国国务院|中国国务院]]批准，由[[县级市|县级市]]成为重庆市的市辖区。永川区是重庆市一小时经济圈和城市发展新区的组成部分，成渝城市群重要节点城市。

== 历史 ==
古为昌州属地。
*776年（[[唐|唐]][[大历|大历]]十一年），割[[泸州|泸州]]、[[璧山|璧山]]两县地始置县，因县城三河汇流，形如篆文的“永”字，故取名永川，至今已有1200多年历史。
*1960年[[四川省|四川省]][[江津地区|江津地区]]行署迁至永川。
*1981年更名为永川行署。
*1983年[[地市合并|地市合并]]隶属[[重庆市|重庆市]]。
*1992年5月撤[[县|县]]建立[[县级市|县级市]]。
*2006年10月22日，由县级市成为重庆市的市辖区。
永川历史悠久，人杰地灵。蕴育了清朝台湾知府[[黄开基|黄开基]]、[[红四军|红四军]]军委书记[[刘安恭|刘安恭]]、世界知名[[画家|画家]][[陈子庄|陈子庄]]、[[微生物学|微生物学]]家[[陈文贵|陈文贵]]、[[地理学|地理学]]家[[徐近之|徐近之]]、微刻艺术家[[刘声道|刘声道]]、诗人[[钟代华|钟代华]]、音乐人李琳。

== 地理 ==
*永川区位于[[四川盆地|四川盆地]]东南部，重庆市西南部，[[长江|长江]]上游北岸，东靠[[璧山区|璧山区]]、[[江津区|江津区]]，南接[[合江县|合江县]]、[[泸县|泸县]]，西接[[荣昌区|荣昌区]]、[[大足区|大足区]]，北与[[铜梁区|铜梁区]]毗邻。
*全市地貌分为[[低山|低山]]、[[丘陵|丘陵]]、[[缓丘平坝|缓丘平坝]]三大类。地质构造为西南-东北走向的一些列平行山脉分割的缓丘平坝。自东向西这些平行山脉是：云雾山（与江津区、璧山县的分界线）、箕山-黄瓜山、巴岳山-云龙山/英山（阴山）。在上述山脉，分别建有国有林场：云雾山林场（石笋山旅游风景区）、箕山林场（茶山竹海国家森林公园）、黄瓜山林场（黄瓜山森林公园）、红旗林场（巴岳山）、英山林场（云龙山森林公园）。
*中[[亚热带湿润季风气候区|亚热带湿润季风气候区]]，日气温年平均为17.8℃，年均降雨量1036毫米。
*历为川东南物资集散中心和陆路交通枢纽，是重庆西部经济重镇。

=== 歷史轄區的變化<ref>「永川歷史上的幾塊飛地」，屈景益，中國人民政治協商會議重慶永川區委員會文史資料編輯委員會.《永川文史資料選輯第二十四輯》</ref> ===
據清光緒《永川縣誌》記載，永川縣有三塊跨縣飛地：
* 一是西部石盤鋪。由永川縣太平鄉出縣境，跨大足縣郵亭鋪而深入榮昌縣境內。據清光緒《永川縣誌》記載：石盤鋪，轄區內有三縣橋、唐家橋。設鋪兵3名。上(西)至榮昌峰高鋪15里，至榮昌縣城45里；下(東)至大足縣郵亭鋪15里，至永川縣城東底塘鋪75里。因中隔大足縣郵亭，不易管理，民國初年劃歸給榮昌縣，現屬榮昌縣峰高區轄地。
* 二是永川縣西南大磨場界外的銅鑼山一帶，轄區內有銅鑼山、描兒山、30餘戶，田土500餘挑。被瀘縣、江津縣、合江縣隔斷，孤懸於外，民國初年劃給瀘縣。
* 三是縣南的官溪子。民國26年時，為松溉鎮聯保所屬的第五十六保。位於松溉鎮長江南岸距江邊約兩里處，面積東西約一里半，南北約兩里。共6甲，51戶，256人，田1130挑。因四周均為江津縣石蟆場所轄，亦系歷史上造成的插花飛地，不易管理，民國26年7月，永川縣政府呈請將官溪子劃歸江津，同年9月7日省政府批准，12月26日兩縣代表在五十六保辦公處正式辦理交接手續。

1974年2月1日，因[[瀘縣|瀘縣]]三溪水庫淹沒永川縣部分土地。永川縣將永瀘公社淹沒之天堂、樂天大隊之7隊、臨江大隊之6隊，共15個生產隊602戶、2875人及其土地劃歸瀘縣。瀘縣將寶峰公社之保衛大隊、太平大隊8個生產隊，中心大隊五個生產隊，新華大隊3個生產隊，計26個生產隊和寶峰街村1093戶、5187人及其土地劃歸永川縣<ref>瀘州市地方誌編纂委員會編纂,《瀘州市志》,方志出版社,1998.08,第100頁</ref>。

== 人口 ==
2015年末，全区户籍人口113.1万人，常住人口109.6万人，城镇常住人口71.2万人，常住人口城镇化率为64.9%。中心城区面积65.6平方公里、人口65.87万人。

==经济==
永川区在重庆全市内比较，是除[[重庆主城区|重庆主城区]]外经济较发达地区。2015年，全区[[GDP|GDP]]为570.34亿人民币，位于重庆市辖区县第10位；人均GDP为人民币52034元，位居重庆市辖区县第12位，[[重庆主城区|重庆主城区]]外区县第3位；地方公共财政收入46.5亿人民币，公共财政支出80.3亿人民币。永川区是成渝城市群发展规划中的重庆重要的制造业基地，2015年全区工业总产值1200亿人民币，固定资产投资720.1亿人民币。年产原煤200万吨左右，主要传统工业产品为光学仪器、金属卷带工业等，近年来主要工业发展方向为自动化装备制造、电子信息及汽车零部件制造产业，建有国家新型工业化产业示范基地——装备制造（机器人）·重庆永川工业园区，软件与信息技术外包产业在重庆市有一定地位。农业特产为[[豆豉|豆豉]]、[[松花皮蛋|松花皮蛋]]、[[茶叶|茶叶]]。永川区是[[重庆主城区|重庆主城区]]外商贸业最发达的区县之一，为成渝城市群发展规划的区域商贸物流中心，2015年实现社会消费品零售总额231.7亿元，位居重庆主城区外区县第2位，人均社零额位居重庆主城区外区县第1位，已建成渝西广场、人民广场两个主要零售商圈，运营中的商业综合体有光华渝西世纪广场、俊豪中央大街、帝都红旗广场、名豪购物广场、华创渝西新天地、协信星光广场、[[万达广场|万达广场]]等，在建的有海亮国际广场、华贸国际中心、宝利阳文化广场、韩国城（截止2018年10月11日仍处于烂尾状态）等，另有中国永川商贸城、永川汽摩机电城、吉之汇国际农贸物流城、奥韵家博城、天骄银座装饰城、永川家电批发市场等专业市场。本土商超百货企业代表为重庆名豪实业（集团）百货有限公司，已引入[[重庆百货|重庆百货]]、[[新世纪百货|新世纪百货]]、[[人人乐|人人乐]]、[[永辉超市|永辉超市]]、[[沃尔玛|沃尔玛]]、[[乐天玛特|乐天玛特]]、梅西百货（泸州）、[[苏宁电器|苏宁电器]]、[[国美电器|国美电器]]、[[紅星美凱龍|红星美凯龙]]等商贸企业。永川区是[[重庆主城区|重庆主城区]]外挂牌五星级酒店最多的区县，高级酒店有江鸿国际大饭店、名豪国际酒店、两江假日野生世界大酒店、金盆湖别墅度假酒店、斯特拉福精品酒店、中华茶艺山庄等。永川区是重庆西部的金融中心，[[中国人民银行|中国人民银行]]在永川区设立中心支行，[[中国银行业监督管理委员会|重庆银监局]]在永川设立分局，辖区包括永川区及附近的[[大足区|大足区]]、[[荣昌区|荣昌区]]、[[双桥区_(重庆市)|双桥经济技术开发区]]，已引入的银行机构有[[中国农业发展银行|中国农业发展银行]]、[[中国工商银行|中国工商银行]]、[[中国农业银行|中国农业银行]]、[[中国银行|中国银行]]、[[中国建设银行|中国建设银行]]、[[中国邮政储蓄银行|中国邮政储蓄银行]]、[[交通銀行|交通银行]]、[[中信银行|中信银行]]、[[兴业银行|兴业银行]]、[[浦发银行|浦发银行]]、[[平安银行|平安银行]]、[[恒丰银行|恒丰银行]]、[[成都银行|成都银行]]、[[重庆银行|重庆银行]]、[[重庆农村商业银行|重庆农村商业银行]]、[[重庆三峡银行|重庆三峡银行]]等，其中部分银行在永川区设立片区级机构；有村镇银行1家、小额贷款公司5家，担保公司2家，基金管理公司3家，证券公司2家，保险公司28家。

==交通==
永川区是是重庆规划建设的区域综合交通枢纽。[[成渝铁路|成渝铁路]]在永川城区设[[永川站|永川站]]，高速铁路[[成渝客运专线|成渝客运专线]]在永川新城设[[永川东站|永川东站]]，另规划有[[渝昆高速铁路|渝昆高速铁路]]永川南站、[[重庆市郊铁路|重庆市郊铁路]]合川—铜梁—大足—永川段。全区公路总里程达到2802公里，公路网密度达到每百平方公里178公里；永川区亦是重庆市内少有的拥有本区绕城高速的区县，[[成渝高速公路|成渝高速公路]]、[[重庆三环高速公路|重庆三环高速公路]]在永川区过境并设有15个道口，在建有九永高速连接[[九龙坡区|九龙坡区]]与永川区，以缓解既有[[成渝高速公路|成渝高速公路]]重庆城区至永川段的通行压力，另规划有永泸高速公路连接永川区与四川省[[泸州市|泸州市]]。境内[[长江|长江]]21.5公里，内河航道73.3公里，重庆港永川港区设有松溉、朱沱2个作业区。永川城区距重庆江北国际机场约90公里，有高速客运大巴接驳旅客；在建大安通用航空机场位于永川城区东约10公里的[[大安街道_(重庆市)|大安街道]]，为一类[[通用航空|通用航空]]机场，[[飞行区等级|飞行区等级]]为2B，跑道长度1000米，宽30米。永川区现有一级公路客运站2座，为永川汽车总站、永川客运中心，在建的瓦子铺客运综合枢纽站位于永川新城。城区公交线路26条，公交车227辆，部分车型为电气混动，主要车型品牌为[[恒通客车|恒通]]、[[郑州宇通客车|宇通]]，区内发行的永川区智能公交卡与重庆[[宜居畅通卡|宜居畅通卡]]可在对方运营区域内的公共交通上互刷使用；区内出租车540辆，车型为蓝色和橘色相间涂装的[[天然气|天然气]]动力[[长安铃木|长安铃木]][[SX4|天语SX4]]轿车。[[滴滴出行|滴滴出行]]在永川区设有运营机构。2013年，全区机动车保有量为166066辆，由于城区地形较平坦，[[电动自行车|电动自行车]]亦为市民主要的通勤工具，[[摩托车|摩托车]]则在乡村地区广受欢迎。区内建有渝西（永川）物流中心、和旺物流园等物流设施。

==环境==
永川区为重庆市内唯一的[[国家森林城市|国家森林城市]]。2015年末，全区森林覆盖率46.5%，城区绿地率40.2%，绿化覆盖率42.1%，人均公共绿地面积23.3平方米。永川区亦为国家园林城区，已建成的城市公园有永川植物园、新泰桂山市民公园、北山公园、望城公园、桃花山公园、探花公园、中央公园、棠城公园、兴龙湖公园、神女湖公园、凤凰湖公园、望贤公园、观音山公园等。2015年空气质量优良天数达345天。城区饮用水源水质达标率达100%；集镇饮用水源水质达标率92%。城区区域环境噪声平均值58分贝；城区主干道交通噪声平均值67.8分贝。城区垃圾无害化处理率达98.0%，集镇垃圾无害化处理率达71.0%。

== 旅游 ==
辖区内旅游资源丰富，昔有[[桂山秋月|桂山秋月]]、[[竹溪夜雨|竹溪夜雨]]、[[铁岭夏莲|铁岭夏莲]]、[[八角攒青|八角攒青]]、[[石松百尺|石松百尺]]、[[圣水双青|圣水双青]]、[[龙洞朝霞|龙洞朝霞]]、[[梳妆台|梳妆台]]昌州八景，为游人游览留连之地。今北有风光旖旎的茶山竹海国家森林公园，南有山青水秀的卫星糊、四季飘香的国家农业生态示范园--黄瓜山百里水果长廊和野趣十足的重庆野生动物世界；有全国首例发现的肉食[[恐龙|恐龙]][[化石|化石]]——上游永川龙；有全国著名金石微刻艺术家[[刘声道|刘声道]]的作品——[[三教镇|三教镇]]石龙山摩崖石刻；有[[杜甫|杜甫]]所书的“万年松化石”；有奇山怪石组成的男、女石笋山，有大文豪[[苏东坡|苏东坡]]留连之地——来苏梳妆台，以及[[宋代|宋代]]石刻——[[佛岩寺|佛岩寺]]等风景名胜。其中以“[[茶|茶]]、[[竹|竹]]、[[石|石]]”三大特色旅文化游资源尤为引人注目。同时进行着大量的旅游项目建设，每年举行中国重庆(永川)国际茶文化旅游节。

== 科教 ==
