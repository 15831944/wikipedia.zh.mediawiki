'''永兴岛'''（{{lang-en|Yongxing Island / Woody Island}}）是[[西沙群岛|西沙群岛]]之[[宣德群岛|宣德群岛]]的[[岛屿|岛屿]]，是西沙群島面積最大的島嶼，同時也是[[中华人民共和国|中华人民共和国]][[海南省|海南省]][[三沙市|三沙市]]人民政府驻地。越南政府聲稱對該島擁有主權，並稱之為“富林島”。

== 简介 ==
永兴岛位于[[南海|南海]][[西沙群岛|西沙群岛]]东部的[[宣德群岛|宣德群岛]]（西沙群岛西部称[[永乐群岛|永乐群岛]]）的中部。该岛由[[珊瑚沙|珊瑚沙]]堆积而形成，岛的东西长约1950米，南北宽约1350米，原始面積為2.13平方公里，[[填海|填海]]造陸後扩大到2.6平方公里，平均[[海拔|海拔]]高度为5米<ref>[http://www.hnszw.org.cn/data/news/2009/06/43638/ 海南史志网：第一节 西沙群岛的主要岛礁]</ref>，四周沙堤环绕，岛中部低地是[[潟湖|潟湖]]干涸而成。岛上植被茂盛。永兴岛是西沙群岛中最大的岛屿<ref>{{Cite news|url=http://news.dichan.sina.com.cn/2012/03/29/463830.html|title=我国年内将开通海南到西沙永兴岛旅游航线|newspaper=《[[新快报|新快报]]》|publisher=[[新浪网|新浪网]]|date=2012-03-29}}</ref>。中国渔民俗称“猫注”<ref>{{Cite journal|title=中国古代对南海诸岛的命名|author=刘南威|url=http://218.189.123.38/videoplayer/d20121209124439546.pdf?ich_u_r_i=9fe284f90649c48a323d281a3c9242d5&ich_s_t_a_r_t=0&ich_e_n_d=0&ich_k_e_y=1845078923751463492489&ich_t_y_p_e=1&ich_d_i_s_k_i_d=9&ich_s_e_q=1821277935&ich_u_n_i_t=1|journal=地理科学|issue=2|doi=|others=|year=1994|volume=14|page=105|pmid=}}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>（疑为“妈祖”之变音<ref name=":0" />）、“猫峙”或“巴注”。
1946年，[[中华民国|中华民国]]政府派遣海军“永兴”号军舰接收西沙群岛，并以舰名命名岛名以示纪念。永兴岛是[[中华人民共和国|中华人民共和国]][[海南省|海南省]][[三沙市|三沙市]]人民政府的驻地，成为[[西沙群岛|西沙群岛]]、[[中沙群岛|中沙群岛]]、[[南沙群岛|南沙群岛]]的行政中心，是西沙群岛的交通中心。永兴岛上还有[[永兴边防派出所|永兴边防派出所]]、[[西沙雨水班|西沙雨水班]]。
[[File:PhuLam-HoangSa.jpg|缩略图]]
永兴岛拥有[[南海诸岛|南海诸岛]]（包括西沙、中沙、南沙、东沙）数个第一：原始面积最大、最大机场、最大港口、最多人口、海拔最高点（原本最高的石岛填海后成为永兴岛的一部分）。

== 历史 ==

1932年，[[法屬印度支那|法屬印度支那]]政府佔據了西沙群岛，期间，越南人和法國人合謀，将中国渔民在永兴岛修建的猫注娘娘庙（妈祖娘娘庙）篡改为[[黄沙寺|黄沙寺]]<ref name=":0">{{Cite web |url=http://dao.qq.com/a/20170518/029414_4.htm|title=南海诸岛庙宇史迹，基本现状及其变迁辨析|author=陈进国|date =2017-05-18|access-date=2017-07-30 |publisher=腾讯网，来源：《世界宗教文化》|language=zh-hans }}</ref>。这是越南古籍记载的一座古寺名稱，越南人的篡改行为试图证明永兴岛为越南方面所有。1935年，中华民国水陆地图审查委员会公布的岛屿名称为「茂林岛」，乃政府人员在不知道永兴岛已有既有名称的情形下，直译其英文名称“Woody Island”之结果。<ref>{{Cite journal|title=文化生态学视角下的南海诸岛地名文化|author=赵静|url=http://www.rddl.com.cn/CN/abstract/abstract2806.shtml|journal=熱帶地理|issue=6|doi=|others=|year=2016|volume=36|page=1046|pmid=}}</ref>[[第二次世界大戰|第二次世界大戰]]，[[南海諸島|南海諸島]]一度被[[日本|日本]]奪得，1939年3月20日日本出兵占领西沙群岛。

日本投降後，根据《[[開羅宣言|開羅宣言]]》和《[[波茨坦公告|波茨坦公告]]》的精神，包括西沙、东沙、中沙和南沙群岛在内的南海諸島一併歸還中國，1946年9月[[中華民國政府|中華民國政府]]派[[中華民國海軍|中國海軍司令部]]海事處上尉參謀[[張君然|張君然]]三次下南海，四次到達西沙群島，並於1946年11月23日，聯同進駐西沙群島的艦隊副指揮官[[姚汝钰|姚汝钰]]，乘{{enwiki|AM-226|「永興」號巡邏艦}}(原[[美国海军|美国海军]]{{enwiki|Admirable-class_minesweeper|钦佩级扫雷舰}}AM-226号)登臨這個島，岛名也因此而得。永兴岛上至今仍立有當年張君然（被任命为第一任海军[[西沙群島管理處|西沙群島管理處]]主任）立下一方水泥紀念碑，正面碑文爲「南海屏藩」四個大字，背面刻有「[[海軍收復西沙群島紀念碑|海軍收復西沙群島紀念碑]]」，旁署「[[民國紀年|中華民國三十五年]]十一月二十四日 [[張君然|張君然]]立」。<ref name="洪致文p284">{{Harvnb|洪致文|2007|p=284}}</ref>

1950年[[海南岛战役|海南岛战役]]之后，永興島的中華民國國軍於5月8日撤離，解放軍則於5月13日進駐永興島，中華人民共和國的武裝漁民在解放軍的支持下逐漸控制了包括永興島在内的[[宣德|宣德]]群島（西沙群島的東半部分），由於兩岸的中國政府在相當長的一段時間内無暇顧及西沙諸島，[[越南共和國|越南共和國]]（[[南越|南越]]）政府在美國的幫助下逐漸侵佔了西沙[[永樂群島|永樂群島]]，和中華人民共和國佔據的[[宣德群島|宣德群島]]隔海對峙。在其後的近20年中，雙方多次爆發衝突。1974年，解放軍和越南共和國海軍在西沙爆發武裝衝突，中華人民共和國擊敗越南共和國控制了整个西沙群島。

2006年7月15日，[[中华人民共和国交通部|中华人民共和国交通部]]西沙救助基地在永兴岛正式启用，派驻一艘“南海救111”号救助船。<ref>{{Cite news|url=http://news.xinhuanet.com/newscenter/2006-07/15/content_4837891.htm|title=交通部西沙救助基地15日在我国南海永兴岛启用|publisher=[[新华网|新华网]]|date=2006-07-15}}</ref>

2012年6月21日经国务院批准，撤销[[西沙群岛、南沙群岛、中沙群岛办事处|西沙群岛、南沙群岛、中沙群岛办事处]]，设立地级[[三沙市|三沙市]]，管辖西沙群岛、中沙群岛、南沙群岛的岛礁及其海域，三沙市人民政府驻西沙永兴岛。

== 地理位置 ==
永兴岛位于[[西沙群岛|西沙群岛]]东部[[宣德群岛|宣德群岛]]的[[宣德群礁|宣德群礁]]中。位於[[北緯|北緯]]16°50′，[[東經|東經]]112°20′。距其西北方向的海南岛[[榆林港|榆林港]]180海里。

== 气候 ==
地處[[北回歸線|北回歸線]]以南，屬[[热带季风气候|热带季风气候]]，由于周围被海包围且陆地面积小，有[[海洋性气候|海洋性气候]]的特点，雨量充沛，終年高溫，高濕，高鹽。加上空氣潔凈，陽光明媚，紫外線格外強烈。太陽直射時間多、日照長，年平均氣溫攝氏26.5度，一月最冷，平均氣溫攝氏23度，六月最熱，平均氣溫攝氏29度；日氣溫最高攝氏31度，最低攝氏21度；雨季為每年5至6月份。

== 居民及生活 ==
在島上生活的人員包括海南省三沙市的工作人員，[[中国人民解放军海军|中国人民解放军海军]]驻岛部队、[[中国人民武装警察部队|中国人民武装警察部队]]海南省边防总队驻岛部队、少量的定居渔民和大量的过路渔民、来岛建设的工人及零散的观光旅客。岛上设有[[永兴社区|永兴社区]]。岛上是典型的熱帶風光，[[椰樹|椰樹]]成行，風光旖旎。每月補給船到達永興島的時候，永興島居民都會放假兩天，到碼頭協助卸下運補物資，這兩天就是永興島的節日。駐島官兵每過兩年換班一次，工作人員則是半年就換班一次。永興島上的生活用水是經過處理的雨水，分為飲用水和洗滌用水。

== 设施 ==
* [[永兴岛机场|西沙军用/民用机场]]于1991年建成，經過後續填海造陸後，現在拥有一条{{convert|3,000|m|ft yd}}的混凝土跑道，可起降[[波音737|波音737]]飞机<ref name="洪致文p284"/>。在2005年，主要接待民航包机，使用60座位的[[运七|运七]]型螺旋桨飞机，通航[[海南省|海南省]][[海口市|海口市]]的[[美兰国际机场|美兰国际机场]]；2016年底开通往返于[[海口美兰国际机场|海口美兰国际机场]]的每日包机，由[[波音737-800|波音737-800]]机型执飞。

* 岛上的行政中心设在北京路，该路有“西沙王府井”之称，这里建有[[中国工商银行|中国工商银行]]西南中沙群岛支行，人民[[医院|医院]]，粮食站，西南中沙群岛邮电局，百货商店,水产公司，人行道旁是整齐的椰树，樹下有碧綠的草坪，環境相當整潔。
* [[西沙海洋博物馆|西沙海洋博物馆]]。
* [[西沙軍史館|西沙軍史館]]。

* 公安派出所。
* 西沙海事局。
* [[氣象台|氣象台]]。

* 2016年2月，中国军方在永兴岛上部署了两个营的[[红旗-9防空导弹|红旗-9防空导弹]]系统<ref>{{cite web|url=http://news.ifeng.com/a/20160217/47464194_0.shtml|title=美媒：中国已在永兴岛部署红旗-9防空导弹系统|publisher=凤凰网|accessdate=2016-02-17}}</ref>。

== 人文景观 ==
島上的人文景觀有二战时日军占领期间留下的炮樓、[[中華民國政府|中華民國政府]]設立的“[[海军收复西沙群岛纪念碑|海军收复西沙群岛纪念碑]]”（1946年）和[[中華人民共和國政府|中華人民共和國政府]]設立的“[[南海諸島紀念碑|南海諸島紀念碑]]”（1974年）。西部有一片被稱為“[[西沙将军林|西沙将军林]]”的椰林，這是中华人民共和国[[党和国家领导人|党和国家领导人]]以及100多位將軍先後栽種的，每一棵上刻著栽種者的名字。島上有兩間博物館：“守島部隊軍史館”及“[[西沙海洋博物馆|西沙海洋博物馆]]”，以及[[三沙市地名碑|三沙市地名碑]]、[[中国西沙主权碑|中国西沙主权碑]]、[[西沙精神纪念墙|西沙精神纪念墙]]。
