'''永春县'''（{{lang-nan|永春縣|poj=Éng-chhun koāiⁿ|bp=Yǐngcūn Gnuâi}}），古稱'''桃源'''，位于[[中华人民共和国|中华人民共和国]][[福建省|福建省]]东南部、[[晋江|晋江]]东溪上游，是[[泉州市|泉州市]]下辖的一个县。境内拥有4A级景区牛姆林

== 历史 ==

[[西周|西周]]至[[春秋战国|春秋战国]]属七闽、闽越地。秦代属闽中郡。汉初属闽越国。汉元鼎五年废闽越，元封元年徙其民于江淮间，以其地属会稽郡。[[东汉|东汉]]建安初，属侯官县。[[三国|三国]]吴永安三年属东安县。梁天监（504––519）中至陈属南安郡

[[隋朝|隋朝]]开皇九年（589年）析南安县西北乡置桃林场（治所在今石鼓镇桃场村）。后唐长兴四年（933年）升为桃源县，以其众水会于桃溪一源，故曰桃源。后晋天福三年（闽通文三年）（938年）因与湖南桃源县重名，遂以境内草木繁盛，四季如春，改名永春县，治所在上场堡（今桃场村）。

[[清朝|清朝]]雍正十二年（1734年）升永春为直隶州，辖大田、德化两县。1913年废州为县，属南路道（厦门道）。

1985年经国务院批准，被列为闽南金三角经济开放县<ref>http://www.fjyc.gov.cn/zjyc/ycgk/ 永春概况</ref>

== 特产 ==
永春[[芦柑|芦柑]]、永春[[佛手|佛手]]及永春[[篾香|篾香]]为[[中国地理标志产品|中国地理标志产品]]<ref>[http://kjs.aqsiq.gov.cn/dlbzcpbhwz/bhml/200906/t20090622_118997.htm 《地理标志产品保护规定》实施以来批准的地理标志名录（2005年7月至2009年6月）] {{webarchive|url=https://web.archive.org/web/20110311120900/http://kjs.aqsiq.gov.cn/dlbzcpbhwz/bhml/200906/t20090622_118997.htm |date=2011-03-11 }}</ref>。

永春老醋被称为“中国四大名醋”

永春县独有的工艺品[[纸织画|纸织画]]曾与杭州[[丝织画|丝织画]]、苏州[[刺绣|刺绣]]、四川[[竹帘画|竹帘画]]并称为中国四大家织<ref>{{cite web|title=永春纸织画|url=http://www.qzt.gov.cn/qzgk/ctmc/200711/t20071116_766.htm|publisher=泉州市地方志编纂委员会办公室|accessdate=2010-09-12|date=2007-11-16}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

漆籃是當地一大特產，使用竹子、漆等原料經過三十多到工序做成，製作精美，舊時是女子陪嫁的嫁妝。

永春榜舍龟、[[咯摊|咯摊]]、麻佬等是当地的著名小吃。

==海外华人==
在[[马来西亚|马来西亚]]及[[新加坡|新加坡]]，很多华裔[[祖籍|祖籍]]为福建永春，其中马来西亚的[[马六甲州|马六甲州]]有约15万人为永春华裔，还有[[柔佛州|柔佛州]]的[[昔加末县|昔加末县]]有六成以上的华人是永春华裔。
