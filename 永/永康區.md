'''永康區'''為[[臺灣|臺灣]][[臺南市|臺南市]]的[[市轄區|市轄區]]，位於臺南市中心略偏西北，東臨[[新化區|新化區]]、東南臨[[歸仁區|歸仁區]]、南臨[[仁德區|仁德區]]、西南臨[[東區_(臺南市)|東區]]、西臨[[北區_(臺南市)|北區]]、西北臨[[安南區|安南區]]、北則與[[新市區_(臺南市)|新市區]]相接。
舊稱臺南縣'''永康鄉'''，1993年5月改制為'''永康市'''，2010年因配合[[2010年中華民國縣市改制直轄市|臺南縣市合併改制]]而改制為市轄區至今。區境內因工商業與交通發達導致人口與都市規模成長相當快速；現人口數為23.4萬，為臺南市人口最多的轄區，也是台南市區繁華的精華地帶。

另知名企業─[[統一企業|統一企業]]、大成長城企業總部皆設於本區；臺灣前20名銀行及各大知名餐飲業者皆有在本區設立多個據點。

== 歷史 ==
永康開發頗早，史前文化有進入[[鐵器時代|鐵器時代]]的蔦松文化。[[臺灣荷西統治時期|荷治時期]]亦有荷人在此築橋的紀錄<ref>[http://ithda.ith.sinica.edu.tw/?action=news&id=87 大橋的故事] {{Wayback|url=http://ithda.ith.sinica.edu.tw/?action=news&id=87 |date=20150714112022 }}，翁佳音</ref>，[[鄭成功|鄭成功]]死後也葬於境內的洲子尾<ref>{{cite news|url=http://news.pchome.com.tw/society/commonsdaily/20090518/index-12425914132379844002.html
|title=永康鹽行天后宮辦媽祖文化祭典
|publisher=[[民眾日報|民眾日報]]
|date=2009-05-18|accessdate=2009-05-26}}</ref>。但永康在歷史上一直是臺南府城東側的僻地，直至戰後臺灣邁入工商業社會才急速成長，今已成台南都會區之一部份。
1920年臺灣地方改制，於此地設置'''-{[[永康庄|永康庄]]}-'''，名稱取自[[清代|清代]]保里「[[永康上中里|永康上中里]]」，隸[[臺南州|臺南州]][[新豐郡|新豐郡]]、設-{庄}-役場於埔姜頭（屬於永康[[大字_(行政區劃)|大字]]）；[[二次大戰|二次大戰]]後，改設鄉。

戰後初期永康仍是以第一、二類型產業為主之傳統鄉鎮。隨著台灣經濟起飛永康區台一線沿線出現了不少中小企製造業聚集，更奠定了今日永康製造業工業帶的基礎。1977年人口67,763並超越縣治新營鎮（今[[新營區|新營區]]），成為臺南縣第一大鄉。1978年[[中山高速公路|中山高速公路]]全線通車，永康設有交流道一處，除交通地位提升外，亦配合同時期的臺南市區向東北擴張，吸引臺南縣各鄉鎮甚至其他縣市移居，進入快速成長期，在接下來的卅年間人口增為三倍。1993年突破15萬，並在同年五月一號升格縣轄市，2002年達成20萬人口，2015年8月人口突破23萬。

2010年12月25日隨著[[2010年中華民國縣市改制直轄市|五都改制]]，改為[[直轄市_(中華民國)|直轄市]]轄區，為臺南市永康區。

== 鯽魚潭<ref>{{Cite web|url=http://163.26.173.1/cyberfair2011/m1.htm|title=古鯽魚潭的形成|accessdate=2018-03-09|work=163.26.173.1}}</ref> ==
'''古鯽魚潭的形成、範圍及水源'''

台南市永康區在昔日的台江內海之中，潭、橋甚多，其中最著名，出現在文學作品之中，最多的便屬鯽魚潭，鯽魚潭在荷鄭時期的古地圖即出現，到清代才有較多的文字記載。最早的記載是在康熙24年（1685年）蔣毓英的『台灣府志』─「鯽魚潭，在永康里東南，周圍大十里餘，深不可測，潭生鯽魚甚多，年有徵稅。」是距今約3500年前由內海演變為潟湖而成的水潭，最後因潭水改道淤積而變成良田，今僅存低窪地的排水渠道稱鯽魚潭溪，在網寮橋以南稱三爺宮溪，網寮橋以北稱大灣中排，並留有鯽魚潭橋為歷史見證。

鯽魚潭形成與台南層台南台地的地形息息相關，據林朝棨教授的研究，台南期海侵之頂部大致海拔30~35公尺，達至新化丘陵之山麓線，堆積了台南層 （台南台地、虎頭埤海層、深坑子海層、大湖海層），至海退時而形成海岸平原，平原受二層行溪、新港溪（今之鹽水溪）及支流三爺宮溪、許縣溪及其分流等之切割，而形成細長的「大灣低地」，又因背斜、斷層之構造運動，而分化出台南台地。台南層屬於全新世低度固結海相堆積層，根據吳東錦碳十四定年研究，台南台地東半部台南層之海進堆積層，堆積年代距今約7200年~5800年間，台南台地西半部台南層之海進堆積層年代，距今約4800年~4600年左右，台地西緣尚有約1700年的海進堆積層年代。又據成大鍾廣吉教授在大灣低地試掘一井，有海貝出土，測其年代約7000年~8000年。因此確定大灣低地過去是海，西鄰台南台地並與東邊新化丘陵形成一內海，因受到地盤不等量上升，在大湖期之海退時而形成潟湖，隨著潟湖淤塞漸與外海隔離，又經台南台地、新化丘陵的堆積而演變為潭；明鄭以後因潭生鯽魚甚多，故稱鯽魚潭或鯽仔潭，位府治之東，又名東湖，為旱禱雨之處，又名龍潭。                                                                                                                  

王必昌的『重修台灣縣志』記載：「鯽魚潭在永康、長興、廣儲三里之界，匯納眾流，修而不廣，計長可三十餘里，多生鯽魚，年有徵稅，三里之田，資以灌溉，一名龍潭，旱時禱雨於此，位府 治之東，又名東湖，最宜霽月，為邑治八景之一」，鯽魚潭與南邊的蓮花潭所成的絲線過脈，亦是府城龍脈水系分水嶺，鯽魚潭水北流經蔦松溪入海，蓮花潭水南流經二層行溪入海，「蓮花潭在文賢里，周圍里許，與龍潭（鯽魚潭）相對，龍潭在北，蓮花潭在南，二潭夾一府，龍入首也」。

鯽魚潭範圍約位在現今永康區，北自烏竹里（烏鬼橋），南至洋仔下（仁德區），東為王田、北灣（王厝）、西灣（頭前李）、南灣（潭墘─巷口）…崙仔尾，西邊沿著台南台地東側的角秀（角宿）、龍潭（蜈蜞潭）、永康、土虱堀、網寮、建國、仁德鄉下洲仔田、石碑仔、崁腳白崙等附近之範圍。水源來自台南台地的坑谷泉水、許縣溪水，與許縣溪以西歸仁以北的泉水、雨水及夏秋季雨水，潭水供耕地灌溉，剩餘則向北流，合許縣溪水流入蔦松溪，西行後注入海，並具有調節新港溪與許縣溪水量之功能。

'''古鯽魚潭開墾的先期文化'''

鯽魚潭為內海時，西邊的台南台地就有先民居住，台南台地北有鹽水溪（新港溪）；南有二仁溪（二層行溪）支流三爺宮溪環繞，東側以斷層崖（後甲里斷層）與大灣低地相接，西側則緩緩降低坡度，銜接安平平原，台地面東高西低，主稜線約25~30公尺，構成北北東─南南西走向之分水嶺。在台南台地的東邊，網寮、清風莊、牛稠子、十三甲、六甲頂…等多處出土史前遺物，其中最具代表的以網寮、牛稠子所出土的文物，屬新石器時代的牛稠子文化，距今約4500年~3500年，此為台南台地迄今所發現最早期的史前文化，亦即鯽魚潭為內海時的先期文化。

網寮遺址出土的陶質有夾砂陶、泥質陶、繩紋陶，也有少量的灰黑陶及陶環，以紅色夾砂陶為主，除陶環作手鐲用之外，餘大多為盛物或儲藏之用的陶容器。遺址的石材也相當多樣，有些也源自外地，如農、獵具的石斧（鋤）、石錛之橄欖石玄武岩類來自澎湖，石斧、石刀、石簇之板岩類來自中央山脈；石網墜、砥石類則在遺址附近採石製作，砥石在石器數量中佔相當大的比例，遺址石器群除網墜外少見打製石器，大多以砥石磨製。石器中用於農耕活動的生計工具為多數，推測當時以農業活動為主，貝塜及網墜等漁具，顯示漁撈亦是生活中的重要一環，而源自澎湖的橄欖石玄武岩層，也可顯示鯽魚潭先期人類，已從事海上的貿易或交換，航海技術已相當發達，網寮遺址開啟永康歷史文化之先河。

相關史載明鄭時期大量漳、泉人隨鄭軍來台，發現鯽魚潭附近土地肥沃、水利灌溉方便，於是就分布在四周定居，開始墾殖及捕魚。「鯽魚潭可饒千金之利，打鼓澳能生三倍之財」。鯽魚潭不僅具有豐富的漁業、水利資源供民需，鯽魚潭的採捕小船與港潭徵稅亦是政府重要的財政稅收。

== 交通 ==
=== 主要道路 ===
[[File:Yongkang_Road_Map.svg|thumb]]
[[File:Yongkang_IC_on_No1_National_Highway.JPG|thumb]]。]]

*[[中正南路_(永康區)|中正南路]]、[[中正北路_(永康區)|中正北路]]：部分路線屬於省道[[台1線|台1線]]。北與[[新市區_(台灣)|新市區]]中山路銜接，南與[[台南市北區|北區]]公園路接壤。沿線經車行、蔦松等聚落，行經[[中山高速公路|國道一號]][[永康交流道|永康交流道]]後進入蔦松，兩旁工業聚落氣息較強烈，各式機械、食品加工廠、塑料製品等工廠林立，永康工業區亦在其沿線上。
*[[中山南路_(永康區)|中山南路]]、[[中山北路_(永康區)|中山北路]]：全線皆屬於省道[[台20線|台20線]]，西起與北區開元路銜接，至東與[[新化區|新化區]]中山路接壤。自開元路進入[[大橋_(臺南市)|大橋]]、網寮、二王、永康、西勢等地區，沿線住宅區與商業活動混合交錯，臺南市立圖書館總館、永康區公所、[[臺南高工|臺南高工]]、[[國立臺南大學附屬高級中學|臺南大學附中]]等文教與行政機構亦在此線上。
*[[中華路_(永康區)|中華路]]：部分路段屬於省道台1線，是臺南市區環形道路的一部分。北與北區[[中華北路_(台南市)|中華北路]]連結，且和省道[[台19線|台19線]][[安南區|安南區]]安和路交會，南與[[台南市東區|東區]][[中華東路_(台南市)|中華東路]]相連接。沿中正南路至六甲頂地區後往南沿續台1線之路段，沿線經過大橋地區，是本區商業活動最密集且最發達的區域，往南即可進入臺南市區。
*[[永大路|永大路]]：南與[[仁德區|仁德區]]中正路相連，北止於中正南路。沿途經[[大灣_(臺南市永康區)|大灣]]、西勢、永康、龍潭等聚落。南可通往仁德區，北可快速通往永康交流道或蔦松地區。部分路段屬於[[市道177號|市道177號]]，可快速通往[[歸仁區|歸仁區]]。
*[[復興路_(永康區)|復興路]]、[[大灣路_(永康區)|大灣路]] 、崑大路、大灣東路：[[小東路|小東路]]、[[復興路_(永康區)|復興路 (永康區)]]、崑大路、大灣東路皆屬於[[市道180號|市道180號]]，復興路西起與東區[[小東路|小東路]]連接，往東至國道一號[[大灣交流道|大灣交流道]]後沿復興路 、崑大路、[[大灣路_(永康區)|大灣路]]進入大灣地區，沿線住宅區高度發展，商業活動也因[[崑山科技大學|崑山科技大學]]設立於此而蓬勃旺盛。近期因大灣交流道的通車使本路段交通地位提升，是本區與臺南市區互相聯絡的重要替代道路之一。
