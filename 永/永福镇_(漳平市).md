'''永福镇'''是位于[[中华人民共和国|中国]][[福建省|福建省]][[龙岩市|龙岩市]][[漳平市|漳平市]]西南部的一个[[镇|镇]]，是漳平市最大的镇<ref> {{cite web |url=http://www.zp.gov.cn/old/xzzc/yfz.htm |title=乡镇之窗——永福镇 |accessdate=2008-05-29 |date=2006-01-04 |publisher=漳平市人民政府 |language=zh-hans }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

== 概况 ==
永福镇地处福建省最南端的高山[[盆地|盆地]]，全镇土地面积535.5平方公里。

永福镇种花历史悠久，是盛产各种名贵花卉的生产基地。2000年，永福镇被[[国家林业总局|国家林业总局]]、全国花卉协会评为"中国[[杜鹃花|杜鹃花]]之乡"<ref> {{cite web|url=http://guoshu.aweb.com.cn/2008/0326/104627190.shtml |title=福建漳平永福台湾农民创业园 |accessdate=2008-05-29 |date=2008-03-26 |language=zh-hans }}</ref>。

== 行政区划 ==
永福镇辖1个居委会、27个村委会：永福居委会、福里村、李庄村、西山村、清源村、吕坊村、后盂村、新坑村、龙车村、石洪村、洪坑村、大坂村、秋范村、蓝田村、紫阳村、封侯村、文星村、同春村、桂洋村、箭竹村、佳山村、和丰村、颖水村、陈村村、元沙村、岭下村、适榕村、古溪村。
