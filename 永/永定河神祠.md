'''永定河神祠'''，位于[[北京市|北京市]][[大兴区|大兴区]][[庞各庄镇|庞各庄镇]]赵村以南的[[永定河|永定河]]大堤下。

==简介==
永定河神祠位于大兴区庞各庄镇赵村南1公里、永定河左堤下的缓坡处。此处于[[清朝|清朝]][[乾隆|乾隆]]三十五年（1770年）决口，合龙后，[[清高宗|清高宗]]命建永定河神祠。乾隆三十八年，清高宗视察永定河时，题御制碑。[[中华人民共和国|中华人民共和国]]时期，此处神祠已无存，仅存立在祠外的御制碑。该碑于1989年被列为[[北京市大兴区文物保护单位|大兴县（今大兴区）文物保护单位]]。该碑是1770年决口的唯一见证，也是关于永定河灾害史的珍贵文物。石碑三面修有挡土墙。石碑下为方座，雕有龙纹，碑高2.9米、宽0.94米，重量1至2吨，上刻乾隆御制诗<ref name=tengxun>{{Cite web |url =http://new.qq.com/cmsn/20130820014807  |title =北京永定河神祠遗址乾隆御制碑碑座被盗  |author =  |date =2013-08-20  |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。

2013年6月8日，该碑碑座被盗。当地居民发现后报警，但警方一直未能追回该碑座。幸存的碑身随后被文物部门运走<ref name=tengxun/>。2017年11月15日，中国被盗（丢失）文物信息发布平台上线，截至11月23日共发布157条被盗文物信息，其中包括永定河神祠碑碑座<ref>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1879118 |title =被盗文物信息显保护短板，有文物所40件唐代泥塑佛头像被盗  |author =  |date = 2017-11-25 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。
