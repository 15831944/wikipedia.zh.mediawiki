'''永和區'''（[[臺羅|臺羅]]：'''Íng-Hô-Khu'''）是[[台灣|台灣]][[新北市|新北市]]下轄的一個[[區_(中華民國)|區]]，為[[臺北都會區|臺北都會區]]的一部分，隔[[新店溪|新店溪]]與[[臺北市|臺北市]]的[[中正區_(臺北市)|中正區]]、[[萬華區|萬華區]]、[[文山區|文山區]]相鄰，南與新北市[[中和區|中和區]]緊密相連，境內居住大量往來臺北市區的通勤人口。

永和區為[[雙北|雙北]]面積最小的[[行政區|行政區]]，區域型態以住商混合為主。

除了早期居住於此的[[台灣原住民|台灣原住民]]、[[泉州裔臺灣人|泉州裔臺灣人]]，還有隨[[國民政府|國民政府]]遷台的[[軍公教|軍公教]][[外省人|外省人]]、[[臺灣大陳人|大陳人]]後裔，以及70、80年代遷入的[[韓國華僑|韓國華僑]]。人口密度高達每平方公里39,100人以上，現為台灣人口密度最高的[[鄉鎮市區|鄉鎮市區]]<ref>參見[[臺灣行政區人口密度表#直轄市／省轄市區排序|臺灣行政區人口密度表]]</ref>。

== 詞源 ==
[[File:YH-DOVE.jpg|thumb]]
在1950年代以前，今永和區所轄地區原隸屬[[中和區|中和鄉]]。1949年[[中華民國政府|中華民國政府]]遷台後，大批大陸各省之[[中華民國國軍|中華民國國軍]]官兵及其眷屬，以其地近[[臺北市|臺北市]]區，移住此區，人口快速增加。1958年，遂獨立成為一個[[鎮_(台灣)|鎮]]。本來命名為「中興」，因易與當時[[台灣省政府|台灣省政府]]所在地的[[南投市|南投市]][[中興新村|中興新村]]相混淆，故需重新命名。由於早年曾發生[[漳泉械鬥|漳泉械鬥]]，地方耆宿楊仲佐先生因而命名為「'''永和'''」，以祈求各族群「永久和平」相處，共創地方繁榮<ref>[https://www.yonghe.ntpc.gov.tw/content/?parent_id=10028 歷史沿革]，永和區公所</ref>。值得一提的是，當時中和鄉也存在名為[[永和_(大字)|永和的地名]]，但範圍與劃出的永和鎮不同，永和鎮的範圍為中和鄉的[[龜崙蘭溪洲|龜崙蘭溪洲]]、[[潭墘_(新北市)|潭墘]]、[[秀朗|秀朗]]。

== 地理 ==
永和區位於[[台北盆地|台北盆地]]的中央偏南，北隔[[新店溪|新店溪]]與[[臺北市|台北市]]的[[中正區_(臺北市)|中正區]]、[[萬華區|萬華區]]、[[大安區_(臺北市)|大安區]]、[[文山區|文山區]]相鄰，南則大致以[[瓦磘溝|瓦磘溝]]西段、中和路、安樂路、[[瓦磘溝|瓦磘溝]]東段、自立路2巷、秀朗路3段50巷向東延伸線與[[中和區|中和區]]相鄰。由於中、永和昔日同屬一個行政區，分治後交界線的東段房舍緊密相連，中西段的瓦磘溝也只是一條小型排水溝，區隔並不明顯，故一般民眾常以「[[雙和|雙和]]」地區合稱之。

永和區全區為新店溪的沖積平原，地勢以東南略高，向西北逐漸降低，其間並無任何山丘。其間地勢較低處在與中和區交界的潭墘一帶，為古新店溪河床地，後因泥沙汙積導致河流改道，但因地勢較鄰地為低而曾經積水成潭。

[[新店溪|新店溪]]為永和區北邊與台北市的界河，也是[[淡水河|淡水河]]水系的最大支流，其上游地區雨量豐沛且季節分配均勻，故終年流量大穩定。雖然名稱為「溪」，但實際上具「河」的型態。早年頗富航運之利，往上游可達[[景美|景尾]]、[[新店區|新店]]等港口，往下游可達淡水河的[[艋舺|艋舺]]、[[大稻埕|大稻埕]]等港口。後來因泥沙汙積、河川污染及陸運發達等因素之衝擊，航運早已終止。

[[File:由瓦磘里休閒廣場看過去的瓦磘溝20100126.JPG|thumb]]]]
[[瓦磘溝|瓦磘溝]]又名潭墘溝，為新店溪支流[[南勢角溪|南勢角溪]]的主要支流之一，發源於[[中和區|中和區]]尖山腳，其上游一小段及中、下游為永和區與中和區的界河，全長4公里<ref>[https://www.yonghe.ntpc.gov.tw/content/?parent_id=10027 細說永和]，永和區公所</ref>。早年下游仍可行小舟，現已被整治成為一條典型的大排水溝。

永和區氣象站位於新北市永和區永平路205號(永平高中)，年平均降雨量為2111毫米，平均溫度為攝氏21.7度。

== 歷史 ==
在[[台灣史前時期|台灣史前時期]]，[[台北盆地|台北盆地]]南部屬於[[平埔族|平埔原住民]][[凱達格蘭族|凱達格蘭族]]分支－[[雷朗族|雷朗族]]的活動範圍，其中有兩個部落居住於今永和地區。1641年，[[荷蘭|荷蘭]]人擊敗[[西班牙|西班牙]]人進駐[[台灣北部|台灣北部]]後，其戶口冊表始見「Cournangh」、「chiron」之部落名稱，即分別為「龜崙蘭社」、「秀朗社」之譯音<ref>《Kaartje van Tamsuy en omleggende dorpen, zoo mede het eilandje Kelang》（《淡水與其附近村社暨雞籠島略圖》，又稱《大臺北古地圖》），1654年（[[荷蘭語|荷蘭語]]）</ref><ref>[http://twstudy.iis.sinica.edu.tw/oldmap/doc/Taipei/Taipei03.htm 《第三章 新店溪與大漢溪流域：一九號至三十號地圖解讀》]</ref>。

1661年（[[明|明]][[永曆|永曆]]15年），[[鄭成功|鄭成功]]驅逐荷蘭人後，開啟[[明鄭時代|明鄭時代]]。當時台灣北部隸屬[[天興縣|天興縣]]，[[漢族|漢族]]雖已開始進入[[台北盆地|台北盆地]]開墾，但僅止於盆地北緣一小部分，本地區仍屬未開化之地。

1683年8月（明永曆37年、[[清|清]][[康熙|康熙]]22年），清軍擊敗明鄭。次年4月，台灣正式納入清朝版圖，台灣北部隸屬[[福建省|福建省]][[台灣府|台灣府]][[諸羅縣|諸羅縣]]。1709年（康熙36年），[[福建|福建]][[泉州|泉州]]人團體[[陳賴章墾號|陳賴章墾號]]獲准開墾台北盆地，其中也包括秀朗一帶，為本地區漢人開墾之始。

1723年（[[雍正|雍正]]元年），由[[諸羅縣|諸羅縣]]拆分出[[彰化縣|彰化縣]]與[[淡水廳|淡水廳]]，台灣北部改隸淡水廳。雍正年間，[[漳州|漳州]]人大舉遷台。由於位於台北盆地中央的[[大加蚋|大加蚋]]地區幾乎已被[[泉州|泉州]]人開墾殆盡，只得渡過[[新店溪|新店溪]]，開墾溪南土地，包括今[[板橋區_(台灣)|板橋]]、[[中和區|中和]]、永和及部分[[土城區|土城]]地區，當時稱為“枋橋十三-{庄}-”。1875年（[[光緒|光緒]]元年），設立[[台北府|台北府]]，本地區改隸隸屬台北府[[淡水縣|淡水縣]][[擺接堡|擺接堡]]。

1895年（清光緒21年、日[[明治|明治]]28年），清朝[[日本|日本]]兩國發生[[甲午戰爭|甲午戰爭]]，清朝戰敗後，於[[馬關條約|馬關條約]]中將[[台灣割讓|台灣割讓]]予[[日本|日本]]。6月，日本佔領台北地區後，擺接堡隸屬[[臺北縣_(日治時期)|台北縣]][[淡水支廳|淡水支廳]]。今永和地區所轄地區，約略為擺接堡轄下的龜崙蘭溪洲、潭墘、秀朗3庄。

1897年6月，日本更改地方制度，廢除「支廳」而改設轄區較小的「辦務署」，本地改隸台北縣新莊辦務署。1901年5月，新莊辦務署裁撤，本地因而改隸。1901年11月，廢縣、辦務署而改設廳、支廳，本地改隸[[台北廳|台北廳]]枋橋支廳。1909年10月，今中、永和地區改制制為枋寮區，其範圍即為日後之[[中和區|中和鄉]]。據1919年（[[大正|大正]]8年）日本人《臺北廳志》的記載，枋寮區下轄漳和、中坑、四十張、外員山、永和（今中和區中原里）、龜崙蘭溪洲、潭墘、秀朗、南勢角、員山仔等十大-{庄}-，大-{庄}-之下又轄十九土名村落。其中龜崙蘭溪洲、潭墘、秀朗三庄範圍與今永和區轄區大致相仿。

1920年廢廳、區而改為州、郡、街庄，本地隸屬於[[臺北州|台北州]][[海山郡|海山郡]][[中和庄|中和-{庄}-]]。1937年（[[昭和|昭和]]12年），川端橋（今[[中正橋|中正橋]]）建成<ref name="建設發展計畫">[http://gisapsrv01.cpami.gov.tw/CPIS/cprpts/taipei_county/county/txt/yungho.html 《台北縣地區綱要計畫－永和區建設發展計畫》] {{webarchive|url=https://web.archive.org/web/20041119193742/http://gisapsrv01.cpami.gov.tw/cpis/cprpts/taipei_county/county/txt/yungho.html |date=2004-11-19 }}</ref>，本地與台北之交通往來大為便利。

1945年[[太平洋戰爭|戰]]後，初期仍循舊制。次年地方制度改制，廢州、郡、街-{庄}-而改為縣、區、鄉鎮，本地隸屬台北縣[[海山郡|海山區]][[中和區|中和鄉]]，共有6村，人口僅有7,723人<ref name="建設發展計畫"/>。1950年8月日廢區而由台北縣直轄。

1949年[[國共內戰|國共內戰]][[中共建政|中共建政]]後，許多追隨[[國民政府|國民政府]]遷台之軍民遷入本地，台北市區考慮防空疏散問題將本區劃為疏散地區，以致人口急速增加。1957年時已增為8村，人口共24,983人，於是分鄉設鎮之議遂起。1958年4月1日，本地自中和鄉拆出獨立成為「永和鎮」。不過由於某些村莊對於要屬中和鄉還是永和鎮的意見一直無法統合，以致於中和與永和之邊界直至1960年10月才確定，由縣政府派員會同雙方鄉鎮長及民政課長，於沿鄉鎮分界處豎立水泥樑柱以為界碑，自此中、永和正式分治<ref>盛清沂，《重修中和鄉誌》</ref>。分治後的永和鎮面積共5.59平方公里，置8個里112鄰，人口38,727人<ref name="建設發展計畫"/>。

1979年1月1日，因地方人口、建設及經濟快速發展，依「台灣省各縣市實施地方自治綱要」升格為[[縣轄市|縣轄市]]。

2010年12月25日，因[[五都改制|五都改制]]施行，原台北縣升格為直轄市，永和市同時改制為[[市轄區|市轄區]]。

== 經濟 ==
由於永和沒有[[工業區|工業區]]或[[工廠|工廠]]，產業結構僅為[[商業|商業]]、[[服務業|服務業]]，加上1950年代因大量軍、公、教人員遷入，人口急速增加，逐漸發展為住商混合，並成為[[台北都會區|台北都會區]]的一部分。幅員不大，發展趨於飽和，目前除[[新店溪|新店溪]]河堤外的[[高灘地|高灘地]]之外，幾無任何較具規模的用地。

== 都市規劃 ==
[[File:俯視永和市仁愛公園20100214.JPG|thumb]]
永和自中和分出前的1955年即擬定公告都市計畫。原本係根據[[英國|英國]][[埃比尼泽·霍华德|埃比尼泽·霍华德]]所提出[[花園城市|花園城市]]概念，仿效英國郊區花園城市，將英國都市近郊適應起伏地形的環狀規劃移植至永和區。這個構想中的花園城，原本擁有七個大型公園，公園佔九分之一面積，計畫人口3萬人，狹窄的道路則在公園綠地之間迂迴。

但因早期永和與快速發展的台北市僅一橋之隔，原始都市規劃沒有落實及過度開發，湧入了遠超出原先設計的人口，導致公園預定地幾乎多數淪為未經合理管控的高密度住宅用地。最初設計的公園，僅一號公園一部分開發為公園（即今仁愛公園），另四號公園因在中永和分治時劃入中和境內而得以幸運保存之外，其餘綠地均被改為學校及住宅用地，形成人口密度非常高，路狹人稠的擁擠狀況。

*三號公園：永和區公所等機關用地、住宅用地
*四號公園：[[中和公園|中和公園]]（今屬中和區，內設有[[國立臺灣圖書館|國立臺灣圖書館]]。）
*五號公園：[[秀朗國小|秀朗國小]]、住宅用地
*六號公園：住宅用地

== 交通 ==
[[File:永和接駁公車E線.JPG|thumb]]
由於新店溪的阻隔，古時候與[[臺北市|台北市]]的交通都是靠[[渡船|渡船]]來維繫，彼時較大的[[渡口|渡口]]是網溪渡，聯絡溪洲與台北市，碼頭即現今之光復街底，道路兩旁遍植[[黃槴花|黃槴花]]，至花開時節香氣怡人。如今藉由[[中正橋|中正橋]]、[[福和橋|福和橋]]及[[永福橋|永福橋]]等交通要道，以及[[台北捷運|台北捷運]][[台北捷運中和線|中和線]]的[[頂溪站|頂溪站]]及[[永安市場站|永安市場站]]（位於中永和交界處，本站地址位於中和）兩座捷運站，幫助民眾往返[[臺北市|台北市]]與永和區兩地。

永和區與中和區雖然曾經屬於同一個行政區，但分治後道路各自命名，因此如同台灣其他相鄰的行政區，相同路名通常指不同的道路。但也有少數例外，例如兩地的中和路、中山路皆各自指向同一條路；尤其中山路還兩區共用一個編號系統，即永和部分編為「一段」，中和部分編為「二段」，所以連門牌號碼都沒有重覆。

比較特殊的是，中和區的永和路是中和鬧區通往中原里（位於華中橋西邊）的一條蜿蜒小道，因中原里舊稱「永和-{庄}-」而得名。該道路形成已有數百年之久，與後來的永和區命名毫無關係。不知其歷史背景者以為道路命名不當，實屬誤解<ref>台灣堡圖，1904年</ref>。

由於兩區之間的界限並不明顯，又有一些道路名稱的特殊情形，常讓不熟悉該地區的人迷路，故網路流傳一首[[打油詩|打油詩]]《中和永和鬼打牆》，描述此一問題：

{{Cquote|
永和有永和路，中和也有永和路，<br>
中和有中和路，永和也有中和路；<br>
中和的中和路有接永和的中和路，<br>
永和的永和路沒接中和的永和路；<br>
永和的中和路有接永和的永和路，<br>
中和的永和路沒接中和的中和路。<br>

永和有中正路，中和也有中正路，<br>
永和的中正路用景平路接中和的中正路；<br>
永和有中山路，中和也有中山路，<br>
永和的中山路直接接上了中和的中山路。<br>
永和的中正路接上了永和的中山路，<br>
中和的中正路卻不接中和的中山路。<br>

中正橋下來不是中正路，但永和有中正路；<br>
秀朗橋下來不是秀朗路，但永和也有秀朗路。<br>
永福橋下來不是永福路，永和沒有永福路；<br>
福和橋下來不是福和路，但福和路接的卻是永福橋。}}

=== 市區街道 ===
*永和路－貫穿永和的主要道路，略呈北北東－南南西走向。北經中正橋通往[[台北市|台北市]]，南接中和路聯絡[[中和區|中和區]]，中途經過樂華夜市，本路段[[捷運頂溪站|捷運頂溪站]]周邊是永和最熱鬧的商圈所在；底下有[[台北捷運中和線|台北捷運中和線]]通過，亦為[[市道111號|市道111號]]。
*中和路－連接永和區與中和區鬧區的道路，為中永和的邊界道路，東南側屬中和，西北側屬永和。其中東半段底下有台北捷運中和線通過，公路編號為市道111號。
*中正路－位於永和東側；智光商職坐落於此。
*中山路－永和新闢的道路，是通往中和最重要的道路，[[比漾廣場|比漾廣場]]坐落於此。
*福和路－經永福橋連接台北市公館地區的主要道路。
*自由街－連接本區中和路與福和路。
*竹林路－永和區公所坐落於此，並且和永貞路、文化路連結，環繞永和一圈。
*林森路－經福和橋連接台北市公館地區的另一條主要道路。
*成功路－秀朗地區主要道路，有匝道可以連接上福和橋。
*環河路－永和沿著築堤開闢的道路，新北環快沿著本路開闢。
*得和路－起自四號公園垂直交中正路.民族路.民權路.永元路.民生路.止於成功路其上有秀朗國小。
*永貞路－西起仁愛公園，東至竹林路、福和路口，為永和最重要的東西向幹道。

=== 歷史建築 ===
*[[網溪別墅|網溪別墅]]：市定[[古蹟|古蹟]]，位於博愛街七號，中和八景之一{{noteTag|1=「中和八景」命名時，永和尚未成為獨立的行政區，因此當時均通稱為中和。}}，現為[[楊三郎美術館|楊三郎美術館]]。
*[[中正橋|川端橋]]：中正橋舊名，連結永和與台北市；今日中正橋之主結構為1972年7月改建。
*[[店仔街福德宮|店仔街福德宮]]：主祀福德正神。
*[[保福宮|保福宮]]：創建於清道光2年（1822年），為一道教廟宇。主祀保生大帝，位於仁愛路202巷。

== 景點去處 ==
*[[世界宗教博物館|世界宗教博物館]]：位於永和區中山路與保生路口。
*[[樂華夜市|樂華夜市]]：位於永和路一段與永平路、永貞路一帶，是[[台灣|台灣]]著名的觀光[[夜市|夜市]]之一，也是居民生活中心。
*[[永和豆漿|永和豆漿]]：原指位於永和路二段中正橋頭的多家24小時營業的豆漿店，1970年代開始逐漸聞名全台後，台灣許多縣市都有名為永和豆漿的早點店。

==特色街道 ==
*[[韓國街|韓國街]]：位於中興街，沿線以許多[[韓國|韓國]]舶來品店家為主要特徵。
*藝術街：位於博愛街，因楊三郎美術館之故，整理為人行步道，假日有時有藝文活動表演。
*美工材料街：位於秀朗路， 因復興商工美工設計相關科系之故，開了頗為密集的店家。 
*家具街：位於中正路與中和區交界處(中和區宜安路以南、中正路2巷以北)，與中和區安樂路共同組成許多平價的家具店。
*民俗街：位於[[店仔街福德宮|店仔街福德宮]]前廣場。
