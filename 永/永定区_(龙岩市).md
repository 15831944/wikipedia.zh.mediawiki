[[File:The_Downtown_of_Yongding_County.jpg|thumb]]
'''永定区'''是[[中华人民共和国|中国]][[福建省|福建省]][[龙岩市|龙岩市]]下辖的一个[[区|区]]，位于[[福建省|福建省]]西南部，北纬24°23′-25°06′，东经116°25′-117°05′。原为'''永定县'''，2015年2月9日正式撤销，改为龙岩市永定区。<ref name="撤县设区2015"/>

== 城市建设 ==
永定城区曾经十分狭小，走完全城只要一个钟。近年发展较快，城市面积快速增长，兴建较多高楼大厦。城区有天子温泉、客家圣殿等景点。南山公园、凤山公园两座公园分别位于城区南北两端。南山山顶建有恢弘壮观的南山塔，在城区大多数地方可看到。凤山公园则为城区最好的公园，有寺庙及独具特色的亭台、石马等，目前正进行整体规划改造，准备扩大面积，兴建景点。目前正在修建栈道。

== 历史 ==

[[唐朝|唐]]初属[[晋安郡|晋安郡]][[新罗县|新罗县]]，[[宋朝|宋]]为[[上杭县|上杭县]]属地。[[明朝|明]][[成化|成化]]十四年（1478年）析上杭县地置[[永定县_(明朝)|永定县]]，历属[[汀州府|汀州府]]、[[汀州道|汀州道]]。1929－1934年为[[中央苏区|中央苏区]]辖县。1949年9月1日，解放军再度攻占永定县。历属[[龙岩专区|龙岩专区]]、[[龙岩地区|龙岩地区]]、[[龙岩市|龙岩市]]。2014年12月13日，国务院同意撤销“永定县”，设立“龙岩市永定区”的决定。<ref>[http://202.108.98.30/description?dcpid=2014 2014年县级以上行政区划变更情况]</ref>2015年2月9日上午9时30分许正式撤销“永定县”，同时设立“龙岩市永定区”。并同时在保留的一些原有机构更名的基础上设立一些新机构（如“龙岩市永定区土楼消防队”等）；同时将城建、土地、城乡规划、社会经济等划归龙岩市的有关部门管理和编制（简称“多规合一”），实现由县域经济单位向城区单位转型；其余不变。<ref name="撤县设区2015"/>

== 地理 ==

东连[[南靖县|南靖县]]，东南与[[平和县|平和县]]交界，西南与[[广东省|广东省]][[大埔县|大埔县]]、[[梅州市|梅州市]][[梅县区|梅县区]]接壤，西北与[[上杭县|上杭县]]相连，东北与[[新罗区|新罗区]]毗邻。

以山地、丘陵为主。最高峰[[茫荡洋|茫荡洋]]，海拔1447米。

主要河流有[[汀江|汀江]]、[[永定河|永定河]]，最大水库为[[棉花滩水库|棉花滩水库]]。

==语言==
全区通行永定话，属于[[客家语|客家语]]的一个方言。

== 交通 ==
*城区交通
中心城区有东大街、南门街、九一路、环城路、银河路等主干路，由于经济快速增长，车辆增多，高峰期常出现严重拥堵。近年兴建了由北环路、东环路。西环路组成的[[绕城公路|绕城公路]]，大大便利了交通，缓解了交通压力。中心城区有书院大桥、南门大桥、东门大桥、龙凤大桥、龙角大桥等五座公路桥横跨永定河，另有一座人行桥风景桥。
汽车站及火车站分别位于城区东部东大街南北两端，有列车班次到[[厦门|厦门]]、[[昆明|昆明]]、[[深圳|深圳]]等大城市。
城区有公交车线路。1路车从古镇到龙凤花园，经过火车站、汽车站、中心市场、区政府等地，贯穿全城。
*铁路：[[file:China_Railways.svg|15px]][[漳龙坎铁路|漳龙坎铁路]]、[[file:China_Railways.svg|15px]][[梅坎铁路|梅坎铁路]]
[[永定火车站|永定火车站]]是全区唯一一座客运火车站。仙师火车站、象牙村火车站、坎市站、富岭站等小站也曾为客运站，列车提速后取消，现大多数为货运站。
*公路：203、309省道，[[双永高速公路|双永高速公路]]
永定北为最靠近中心城区的高速出口，永定东出口则离城区有一定距离。
位于永定区东部的永定汽车站为中心车站，另大多数乡镇有汽车站，城际及乡村[[巴士|巴士]]线路发达，有巴士往返永定下辖各乡镇。另有大巴或中巴往返于各大城市及邻县。
*水路：[[汀江|汀江]]航道，有航班从坝头至洪山乡，另有旅游航班。

==著名景点==
永定客家土楼是福建省“八大旅游品牌”之一 <ref>{{cite web| language =zh-cn| publisher =福建省人民政府办公厅| title =福建省人民政府门户网站——龙岩市| url =http://www.fujian.gov.cn/bmdd/sxgk/ly/| date =| accessdate =2012-12-08}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，主要土楼群有[[洪坑土楼群|洪坑土楼群]]、[[南溪土楼群|南溪土楼群]]、[[高北土楼群|高北土楼群]]、[[初溪土楼群|初溪土楼群]]等。2008年7月6日，“[[福建土楼|福建土楼]]”被列入《[[世界遗产名录|世界遗产名录]]》。

此外，永定区还有[[国家森林公园|国家森林公园]][[王寿山|王寿山]]、[[龙湖影视城|龙湖影视城]]等其他景点。

== 特产 ==
永定特产之一的[[永定万应茶|永定万应茶]]被列入[[中国国家级非物质文化遗产列表|中国国家级非物质文化遗产列表]]。
下洋镇的'''客家牛肉丸'''，泡鸭爪鲜香爽口，颇受当地民众及游客欢迎。
另有笋干、菇类及牛肉干等。永定芋子包、牛杂等独特美食也深受游客欢迎。
