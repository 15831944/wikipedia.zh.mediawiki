== 历史背景 ==
1205年（宋宁宗开禧元年），辛弃疾66岁，为[[镇江府|镇江府]][[知府|知府]]，一日他登上位于今[[江苏省|江苏省]][[镇江市|镇江市]]的北固亭。面对[[南宋|南宋]]王朝江河日下、国将不国的情形，辛弃疾痛心疾首、悲愤不已，此时，南宋王朝投降派、议和派占据上风，他写作了这首词来表述自己抗金救国、恢复中原的壮志雄心，同时也警告当政大臣[[韩侂胄|韩侂胄]]吸取刘义隆的教训。

== 诗词鉴赏 ==
全词采用借古讽今的艺术手法，用历史事实来说明现实问题。全词上阕登高望远，抚今追昔，追慕英雄[[孙权|孙权]]和[[刘裕|刘裕]]。下阕引用典故来抒发感情，引用了[[刘义隆|刘义隆]][[元嘉|元嘉]]年间被[[北魏|北魏]][[拓跋焘|拓跋焘]]打败的典故警醒南宋王朝。<ref name=" duck" >{{cite book |author= 张岚 |title=《高中文言文译著及赏析》|year=2005年 |publisher=长春出版社 |location=长春市 |pages=135到136页 |isbn=7-80604-287-3 |language = zh-cn}}</ref>

全词表达了辛弃疾对南宋朝廷的不满和怀才不遇的感慨。<ref name=" duck" />
