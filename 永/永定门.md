'''永定门'''是[[北京|北京]]外城的正门，位于外城南垣正中，[[北京中轴线|北京中轴线]]上。

== 历史 ==
始建于[[明|明]][[嘉靖|嘉靖]]三十二年（1553年），寓意为“永远安定”。城楼为两层，面阔五间（24米），进深二间（10.5米），绿[[琉璃|琉璃]]剪边灰筒瓦重檐[[歇山顶|歇山顶]]。城楼下为城台，前为1544年增建的[[瓮城|瓮城]]，东西宽42米，南北深36米<ref>《[[酌中志|酌中志]]·卷之十七一大内规制经纪略》第一小节：“皇城外层，向南者曰大明门，与正阳门，永定门相对者也。稍东而北，过公生左门，向东者曰长安左门。再东过玉河桥，自十王府西夹道往北，向东者曰东安门，再南，过灵济宫，灰厂向西，曰长右门。红栅之内，门之北，则登闻鼓院在焉。此外围文六门，墙外周围红铺七十二处也。”</ref>。清[[乾隆|乾隆]]三十二年（1767年）曾重建，将城台加高至26米，重建瓮城，在瓮城正面增建单层[[箭楼|箭楼]]（辟两层箭窗）。箭楼规制甚小，面阔仅三间（12.8米），进深一间（6.7米），单檐歇山顶灰瓦。瓮城为方形，东西宽42米，南北深36米，外侧两端抹为圆角。箭楼辟两层箭孔，南面每层7个，东西每层3个。箭楼城台下辟单孔券门。

1900年[[八国联军|八国联军]]占领北京后，拆除永定门西侧部分城墙，将原在城外马家堡的[[京津铁路|京津铁路]]终点站移至[[天坛|天坛]]。1901年修建[[京奉鐵路正陽門東車站|前门火车站]]时又在城门东侧增开豁口，铺设新路轨。1951年为改善永定门交通，拆除了瓮城和东西两侧的城墙。城门及箭楼于1957年被拆除。

== 重建 ==
2003年，在北京市人大代表的提议下，为了恢复[[北京中轴线|北京中轴线]]的城市规划和景观，在原址<ref>引用《古都城門》一書的說法：“……對於城樓是北移40米或原址重建，最終確定克服城樓與舊橋根基重合的困難，在‘原址’重建。據北京市文物古建工程公司經理李彥成介紹，這只是復建永定門城樓要保持原真性的開始……”）</ref>重建了永定门城楼。2005年，完成了城楼的彩绘，永定门采用[[旋子彩画|雅五墨旋子彩画]]。2009年9月，永定门南广场落成，总占地面积2.2公顷，跨越北京护城河和南二环<ref>{{cite web |url = http://news.bjnews.com.cn/2009/0930/45867.shtml |title = 新京报网：永定门南广场落成开放 占地面积2.2公顷 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

永定门城楼于2005年9月复建后即由东城区管理使用并且对外开放。自2007年起，由东城区文化委员会、东城区旅游局、永定文化交流促进会共同使用。复建后的永定门城楼建筑面积522平方米。2011年12月，永定门城楼被列为[[北京市东城区普查登记文物|北京市东城区普查登记文物]]。截至2013年，永定门城楼90%以上的使用面积对外开放，承担旅游咨询、游客观光、中轴文化展陈的功能<ref>{{Cite web |url = http://epaper.bjnews.com.cn/html/2013-03/27/content_420710.htm?div=-1 |title = 文物局：未发现永定门城楼办会所证据 |author =  |date =  2013-03-27|publisher = 新京报 |language =  |accessdate =  |quote =  }}</ref>。
