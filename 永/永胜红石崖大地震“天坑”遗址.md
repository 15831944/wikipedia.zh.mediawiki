'''永胜红石崖大地震“天坑”遗址'''，是一个位于[[中国|中国]][[云南省|云南省]][[丽江市|丽江市]][[永胜县|永胜县]]的[[国家级典型地震遗址|国家级典型地震遗址]]。该遗址位于永胜城关坝西北部红石崖，地处[[西关山|西关山]]与[[芮官山|芮官山]]之间，包括天坑、槽谷、崩塌、滑坡、裂缝、断层、水系断错等古地震地貌和历史地震遗迹，具有科学研究和旅游开发价值<ref name="报道1"/>。

红石崖“天坑”遗址是[[1515年北胜地震|1515年北胜地震]]的地质断层遗址，遗址形似马蹄，由上部开口深200余米的坑和一个深约300米的狭窄山谷复合而成，总深度约500米<ref name="论文0"/><ref name="报道2"/>。

== 地质背景 ==
红石崖“天坑”遗址所在的永胜县处于[[欧亚板块|欧亚板块]]和[[印度洋板块|印度洋板块]]交界处，位于[[青藏高原|青藏高原]]与[[横断山脉|横断山脉]]的结合部，居于云南省西北部，属[[金沙江|金沙江]]东北岸，境内山峦起伏，沟壑纵横，两山对峙，一江环绕<ref name="报道1"/>。县城[[永北镇|永北镇]]海拔2140米，年平均气温14.7℃，年降雨量837.9毫米<ref name="报道1"/><ref name="报道2"/>。

丽江地区位于特提斯一喜马拉雅构造域弧形构造的转折部位，以[[金沙江断裂|金沙江断裂]]和[[丽江一安顺场断裂|丽江一安顺场断裂]]为界，该区由西到东分别属于三江褶皱系、松潘一甘孜褶皱系和扬子准地台。丽江地区受青藏高原大幅度隆升的影响开始整体掀斜，同时[[中甸—永胜断裂|中甸—永胜断裂]]、[[红河断裂|红河断裂]]等北西向断裂为右旋错动<ref name="论文1"/>。1515年北胜地震发震的中甸—永胜断裂具有较强的活动性，沿断裂带地震活动频繁，自1890年共记录到震级大于等于4级地震20余次，断层活动以右旋走滑为主<ref name="论文2"/>。
== 遗址历史 ==
{{Quote box| width = 200px | align = right | quoted = true | bgcolor = #B9D3EE | salign = right | quote = 澜沧城西红石崖，<br>芙蓉两畔云中开。<br>嵯峨万仞丹砂泼，<br>倒影一线青天来。 | source=清代永北知县[[罗俊民|罗俊民]]<ref name="报道1"/>}}
明正德六年农历五月六日（公历1511年6月11日），[[北胜州|北胜州]]西北侧发生了震级8级的强烈地震，震中烈度达[[中国地震烈度表|中国地震烈度表]]X(10)度<ref name="报道1"/>。明朝万历《[[云南通志|云南通志]]》卷十七记载称：“北胜州，正德六年五月六日地大震，城倾西北，民居圮者千五百余家<ref name="报道1"/>。”清朝乾隆《[[永北府志|永北府志]]》卷二十四则写道，“正德六年五月初六日，地震，城倾西北，民房倒塌千五百余间；近屯西山下，田陷成湖者百余顷<ref name="报道1"/>。”乾隆《永北府志》卷五又载：“西山草海，在近屯西山下，明初原系军民田地，正德六年，地震累月，遂成为湖<ref name="报道1"/>。”

地震造成[[永北镇|永北镇]]北部的胜利、中和、兴营等村境内受到严重破坏，方圆9平方公里，波及范围达100公里<ref name="报道1"/>。是次地震致使逾20平方公里的盆地陷沉，地势向西下降20米左右<ref name="报道1"/>。地震导致西关山与芮官山间的鞍部山岭撕裂，形成了红石崖“天坑”遗址<ref name="报道1"/>。
2005年，[[中国地震局|中国地震局]]批准将永胜红石崖大地震“天坑”遗址设立为国家级典型地震遗址<ref name="论文0"/>。中国地震局在批复中指出，“永胜红石崖国家级地震遗址，包括天坑、槽谷、崩塌、滑坡、裂缝、断层、水系断错等古地震地貌和历史地震遗迹，是宝贵的自然遗产和科普宣传教育场所，具有极高的科学研究和旅游开发价值”<ref name="报道1"/>。

== 环境地貌 ==
西关山与芮官山间的鞍部山岭因震撕裂，形成一个巨大天坑和深约500米、长达9公里的大裂谷<ref name="报道1"/>。该裂谷平面直径长轴方向约2公里，短轴方向约1公里，由一个上部开口深200余米的坑穴和一个深约300米的狭窄山谷复合组成，总深度约500米<ref name="报道2"/>。“天坑”两侧崖壁颜色有所区别，北侧崖壁为红色，南侧崖壁为青色。因此民间分别将其称为红石崖和青石崖<ref name="报道2"/>。
