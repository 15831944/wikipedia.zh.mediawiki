'''永济市'''是[[中华人民共和国|中华人民共和国]][[山西省|山西省]]的一个[[县级市|县级市]]，由[[运城市|运城市]]代管。位于山西西南隅，[[同蒲铁路|同蒲铁路]]穿越市境。

== 历史 ==
古称[[蒲坂|蒲坂]]，相传为[[舜|舜]]帝之都。[[战国|战国]]时属于[[魏国_(战国)|魏国]]，称[[蒲邑|蒲邑]]。[[秦朝|秦]]置[[蒲坂县|蒲坂县]]，[[王莽|王莽]]时改[[蒲城|蒲城]]。[[东汉|东汉]]復名蒲坂县。[[隋朝|隋朝]]开皇十六年（596年）析蒲坂县置[[河东县|河东县]]，[[大业_(年号)|大业]]二年（606年）省蒲坂入河东县。至[[明朝|明朝]][[洪武|洪武]]二年（1369年），省河东县入[[蒲州_(北周)|蒲州]]。[[清朝|清朝]][[雍正|雍正]]六年（1728年）升为[[蒲州府|蒲州府]]，并设[[附郭|附郭]][[永济县|永济县]]，以境内永济渠命名。

1912年废府存县，1958年与[[安邑县|安邑县]]、[[解虞县|解虞县]]、[[临猗县|临猗县]]合并为[[运城县|运城县]]（现运城市）。1961年析运城县复置永济县。1994年1月撤县设市，目前隶属于地级[[运城市|运城市]]。市中心建有舜都广场，纪念舜帝建都于此的传说。

== 地理 ==
位于山西西南隅，南倚[[中条山|中条山]]，西临[[黄河|黄河]]，与[[陕西|陕西]]省隔河相望。

== 经济 ==
永济的工业以[[轻工业|轻工业]]为主，农业主产[[小麦|小麦]]、[[棉花|棉花]]、[[淡水鱼|淡水鱼]]、芦笋等。

高技术行业方面，有[[中国中车集团|中国中车集团]]旗下的[[永济电机|永济电机]]公司，负责供应[[牵引电动机|牵引电动机]]予全国铁路车辆。

== 物产 ==
*矿产有磷、石英砂、石灰岩、白云石、铁、铅锌、金、铜、红玛瑙等。
*野生动物有野猪、野兔、野羊等。
*土特产主要有桑落酒、蒲柿、黄河大鲤鱼。

== 风景名胜 ==
* [[全国重点文物保护单位|全国重点文物保护单位]]：[[蒲津渡|蒲津渡]]与[[蒲州故城遗址|蒲州故城遗址]]，蒲津渡为古代黄河上的著名渡口，其附近曾发现四尊每个重达60吨的[[唐代|唐代]]铁牛，是中国现存最大的铁铸文物。
* [[山西省文物保护单位|山西省文物保护单位]]：[[石庄遗址|石庄遗址]]、[[高市村汉墓群|高市村汉墓群]]、[[小朝村汉墓群|小朝村汉墓群]]、[[杨博墓|杨博墓]]、[[万固寺|万固寺]]、[[普救寺塔|普救寺塔]]、[[韩楫墓|韩楫墓]]、[[栖岩寺塔群|栖岩寺塔群]]、[[赵杏古墓群|赵杏古墓群]]、[[叔夷伯齐墓|叔夷伯齐墓]]、[[扁鹊庙|扁鹊庙]]、[[解梁古城|解梁古城]]、[[孟桐墓|孟桐墓]]、[[杨瞻墓|杨瞻墓]]、[[董村戏台|董村戏台]]
* [[永济市文物保护单位|永济市文物保护单位]]
* [[五老峰|五老峰]]：[[道教|道教]]名山，史称东华山。它和附近的古蒲州共同构成了一处[[国家重点风景名胜区|国家重点风景名胜区]]。
* [[鹳雀楼|鹳雀楼]]：古代名楼，始建于五代，因[[王之涣|王之涣]]的《登鹳雀楼》闻名。最近一次毁于清朝，后于1997年重建。
