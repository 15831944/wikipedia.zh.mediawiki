'''永樂車站'''位於[[台灣|台灣]][[宜蘭縣|宜蘭縣]][[蘇澳鎮_(台灣)|蘇澳鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[北迴線|北迴線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
本站站內設有[[島式月台|島式月台]]、[[岸式月台|岸式月台]]各一座，月台與站房之間以天橋連通。

本站主要還是以貨運為主要的業務項目。此地為[[台灣水泥|台灣水泥]]之[[水泥|水泥]]貨運基地，是台泥蘇澳廠的主要進出門戶，因此設置有寬廣的場站區與多條供貨運列車停靠與通過的側線，經常有散裝水泥貨車停在站區。

== 營運狀況 ==
*本站是台鐵的[[臺灣鐵路管理局車站等級#三等站|三等站]]之一，僅有[[臺鐵區間車|區間車]]停靠，是沿線客運旅次較低的車站之一。車站內設有供旅客蓋戳留念的「鐵路之旅 - 小站巡禮紀念章」戳章。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費<ref>[http://www.railway.gov.tw/tw/CP.aspx?sn=1312&n=1333 臺鐵多卡通電子票證乘車營運規則]</ref>。

== 歷史 ==
*本站是在1980年2月1日，伴隨[[蘇澳新站|蘇澳新站]]至[[花蓮港車站|花蓮新站]]之間的北迴線全線通車，所一同啟用的車站之一。車站初設時原為純貨運用途，車站站房也與今日的位置不同，是在相同站區內較為南側、靠近水泥廠的路段上。1989年因應附近居民的請求，增建了一座客運用月台，並自9月26日開始停靠客運列車。
*之後，為了配合北迴線的雙軌化暨電氣化改建工程本站附近的路段進行路線修改，原有的月台停用，改在站區較北端設置新的月台並將客運服務業務遷至新位置開始營運。至於對應的站房構造則是在新月台啟用後繼續施工，直2003年8月時新站房完工啟用，行車管理的功能都移至新站之後，再將原有的舊站房拆除。
*由於本站的站房與月台區北移，導致與隔鄰的招呼站[[永春車站|永春]]距離太近，於是台鐵在2002年12月18日時裁撤了永春車站。
*2016年3月1日：啟用多卡通刷卡機。
