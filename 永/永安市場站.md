永安市場站位於[[台灣|台灣]][[新北市|新北市]][[中和區|中和區]]、[[永和區|永和區]]交界處，為[[台北捷運|台北捷運]][[中和新蘆線|中和新蘆線]]（[[中和線|中和線]]）的[[捷運車站|捷運車站]]。

== 車站概要 ==
車站位於中和路下方，車站代碼為'''O03'''，在尚未推行2016年車站代碼前則採用工程代碼'''O17'''。

== 車站構造 ==
本站站體由[[台北市政府捷運工程局|台北市政府捷運工程局]]完工後，辦理後續招標，站體上方則由得標廠商[[冠德建設|冠德建設]]興建與站體共構的[[集合式住宅|集合式住宅]]，於市場上販售，而捷運局亦依回饋比例分回部分房產，公開辦理標售(建築案名為美麗永安)。

本站採取地下四層車站，月台型式採[[側式疊式月台|側式疊式月台]]，本站開挖深度約30公尺，兩個[[側式月台|側式月台]]分別位於地下第二層及第四層兩月台間上下平行重疊，僅有一個出入口，本站設有[[半高式月台門|半高式月台門]]。

== 利用狀況 ==
本處除捷運站外，中和路上亦設有「捷運永安市場站」公車站牌，有多路公車行經。此外也為了防止人員蓄意跳軌等因素，本站將於2016年增設[[半高式月台門|半高式月台門]]來保護民眾與行車的安全。

== 歷史 ==
* 永安市場站的位置，早期是公路一村的一部份。早期有[[台汽|台汽]]直達車、[[福和客運|福和客運]]與[[台北客運|台北客運]]合營的57路、以及[[欣欣客運|欣欣客運]]的[[台北聯營公車|台北聯營公車]]238 （已停駛）有在該地設站。
* 1998年12月24日：隨著[[台北捷運淡水線|淡水線]]「[[台北車站|-{台北}-車站]]－[[中正紀念堂站|中正紀念堂]]」段、[[台北捷運新店線|新店線]]「中正紀念堂－[[古亭站|古亭]]」段與中和線正式通車而啟用。
* 2016年9月20日：半高式月台門正式啟用。
