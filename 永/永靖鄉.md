'''永靖鄉'''（[[臺灣閩南語|臺灣閩南語]][[永靖腔|永靖腔]][[臺羅拼音|拼音]]：'''Éng-tsēng-hiong''', [[Help:臺灣話國際音標|國際音標]]：[{{IPA|ʔɛŋ˦˦.t͡sɛŋ˩˩.hi̯ɔŋ˦˦}}]）位於[[台灣|台灣]][[彰化縣|彰化縣]]中部偏東南，地勢平坦，無山岳、湖澤。北緣[[員林市|員林市]]、[[埔心鄉|埔心鄉]]，東鄰[[社頭鄉|社頭鄉]]，西濱[[溪湖鎮|溪湖鎮]]，南接[[田尾鄉|田尾鄉]]。為台灣本島人口密度最高的鄉。

== 地名的由來 ==
永靖地名自1813年（[[嘉慶|嘉慶]]18年）由[[楊桂森|楊桂森]]命名後，曾經歷數次改動，除歷史可稽外，還有不同於歷史記錄的說法：

一、[[廣東省|廣東省]][[潮州|潮州]]府的墾民於1813年（嘉慶18年）在地建立街市，當時的墾民命名為'''永靖'''<ref name="張瑞和">[張瑞和，1990]，引《清代臺灣大租調查書》第二冊，第221頁。台灣省文獻叢刊，第152種：「（嘉慶18年）地方人士在關帝廳-{庄}-前，置田一十六甲建為街市....蒙縣主[[楊桂森|楊氏桂森]]，錫以永靖嘉名」。</ref>，意在期勉本地區與附近居民能和平共處（當時[[閩粵械鬥|閩粵械鬥]]事常發生），'''永'''久安'''靖'''。永靖二字本為永靖街市的名稱，大約是今日的永東、永西村，後來以街市名為鄉鎮名。

二、[[台灣日治時期|日治時代]]初期改為'''關帝廳'''，代表本區之地名。關帝廳之名在1813年（嘉慶18年）<ref name="張瑞和" />與[[道光|道光]]年間均有文獻（道光版《[[彰化縣志|彰化縣志]]》第十一卷/雜記志/兵燹：「[[員林市|員林]]一帶粵人，紛紛搬入[[埔心鄉|大埔心-{庄}-]]及'''關帝廳'''等處...」）提及。

三、1920年（[[大正|大正]]9年），[[日本|日本]]實行[[地方行政區域改革|地方行政區域改革]]，將三字地名改為二字，命名回舊名'''永靖'''，但寓意已改為「[[水患|水患]]頻仍，以求止息水患，永保安寧」<ref name="張瑞和2">[張瑞和，1990]，引《環華百科全書》永靖鄉。第20冊: 513頁。環華事業出版社。</ref>

四、本地區以前[[火災|火災]]頻繁，舊屋常被燒毀而蓋新屋。當地居民苦於火災之患，故改地名為「永靖」，其意為「永久安靖」，並於永靖街與中山路的交叉口種榕樹，以及在永靖街中間建「永安宮」兩端則建立「永奠宮」及「永福宮」，火災才逐漸減少。

== 歷史 ==
*[[臺灣清治時期|清領時期]]，永靖在行政區域上屬於[[武西堡|武西堡]]。
*1813年（清嘉慶十八年），[[廣東省|廣東省]][[潮州|潮州]]府的墾民於此地建立街市。
*[[台灣日治時期|日治]]初期，永靖大部份歸屬[[關帝|關帝]]廳區；永靖西區歸屬羅厝區或海豐崙區。
*1920年，重新劃分區域為[[臺中州|台中州]][[員林郡|員林郡]]『-{[[永靖庄|永靖庄]]}-』，永靖的行政區就此定型下來。
*1945年，改為[[臺中縣|台中縣]]員林區『永靖鄉』。
*1950年，改為彰化縣永靖鄉至今。

以民族學角度來看，世居於永靖者約有七成來自[[廣東|廣東]][[潮州|潮州]]府[[饒平縣|饒平縣]]的[[客家人|客家人]]，但因居於闽南人數居多的彰化平原，大多數居民已[[福佬客|闽南化]]。依據[[日本|日]]籍學者-{庄}-司久孝在1937年（昭和十二年）以人口資料作為台灣人口的地理學研究中，發現永靖鄉人口密度每平方公里超過五百人，居全島農村之首。

== 本土語言 ==
{{seealso|饒平客家語|臺灣閩南語|臺灣客家語拼音方案}}

雖然居民已經[[福佬化|福佬化]]，客籍族群也從[[饒平客家語|饒平客家語]][[語言轉移|語言轉移]]到[[臺灣閩南語|臺灣閩南語]]，但仍遺留若干客語口音，形成特殊的「[[永靖腔|永靖腔]]」。常以下列句子來表現出永靖人臺語發音的不同，即永、靖、冰、冷、𠕇（[[台灣閩南語推薦用字|推薦用字]]，意及俗寫硬）等ing（[[國際音標|國際音標]]：/iŋ/）韻腳皆轉如eng （/eŋ/）。如以下句子：「永靖枝仔冰，冷冷𠕇𠕇（硬硬）」（永靖腔台羅：Éng-tsēng ki-á-peng, léng-léng tēng-tēng）<ref name="腔">台灣優勢腔台羅：Íng-tsīng ki-á-ping, líng-líng tīng-tīng。本句經常被唸成[[打油詩|打油詩]]般的長句：「永靖枝仔冰，冷冷硬硬，中國兵仔拿大槍」，常被用來調侃永靖人，當地人把兵、槍等字亦發音成'''eng'''韻。</ref>。事實上持此口音的人還包括[[埔心鄉|埔心鄉]]、[[員林市|員林市]]、[[田尾鄉|田尾鄉]]、[[大村鄉_(臺灣)|大村鄉]]部份村及[[社頭鄉|社頭鄉]]部份村。

除了ing轉eng韻，彰化縣福佬客地區也會將「it / ip / ik」等入聲韻轉為「et / ep / ek」韻，如永靖鄉的[[竹子村|竹子村]]（竹仔跤）發音「tek-á-kha」。迫發音為「pik」轉「pek」，得發音為「tit」轉「tet」，級發音為「kip」轉「kep」，而「ih」韻則不會轉「eh」韻。<ref>涂文欽,"粵籍移民在彰化縣之分佈與其語言特色"[http://www.uijin.idv.tw/ogawa/history/6th/6-3.pdf],台中縣豐原市葫蘆墩國小,2010.</ref>

而該處曾使用的客語[[饒平腔|饒平腔]]的迫音「bèd」（[[客家語拼音方案|臺灣客拼]]，下同），得音「dedˋ」，冰音「benˇ」，冷音「lenˇ」，頂音「denˋ」<ref>{{cite web|title=台灣客家語常用詞辭典|url=http://hakka.dict.edu.tw|access-date=2018-10-29|archive-url=https://web.archive.org/web/20100901135502/http://hakka.dict.edu.tw/|archive-date=2010-09-01|dead-url=yes}}</ref>，皆與永靖腔相似。

== 地理 ==
彰化縣北邊為彰化隆起海岸平原，南邊為[[濁水溪|濁水溪]][[沖積平原|沖積平原]]，北邊為坡度5%至55%的八卦山脈，永靖鄉居於這三者之中的彰化平原。地勢與[[八堡圳|八堡圳]]溪流流向相同，由東南向西北緩降。永靖鄉平均海拔約在二十公尺上下。土壤以粘板岩老沖積土為主，少部份為粘板岩新沖積土。土質呈中性反應。

因濁水溪沖積平原地下水豐富，飽和層離地面甚近，地面與地下水位的深度只二至四公尺，致使鄉內較低窪的地區有湧泉現象，往往土質過於鬆軟，農民雙足陷入泥中，深及至膝蓋或大腿，耕作不便。

永靖鄉主要灌溉的水源是-{[[十五庄圳|十五庄圳]]}-，於1721年（康熙六十年）由武西堡大埔心-{庄}-（今埔心鄉）黃仕卿倡築。源頭在二水鼻子頭附近（八堡圳口左側），引濁水溪入圳，使水順勢西流，灌溉圳岸兩側的村莊。流經永靖鄉的村落有湳港西莊、關帝廟（廳）-{庄}-、獨鰲-{庄}-、同安宅-{庄}-、崙子尾-{庄}-、福興-{庄}-及竹仔腳-{庄}-等。

1862年（同治初年），民眾出資搶修十五-{庄}-圳，並推舉代表人吳郡山、魏尚瑩（日治時期曾任永靖-{庄}-長）、魏精仁（魏尚瑩之子）輪流管理。

1907年（明治四十年）十五-{庄}-圳併於八堡圳，原八堡圳改稱為八堡圳第一圳，十五-{庄}-圳改稱為八堡圳第二圳。現今永靖鄉內主要的水道，如：湳港舊圳、浮圳、陳厝厝排水路、湳港西排水路、崙子尾排水支圳、四塊厝支圳等均屬於第二圳。

永靖鄉的水源來自於[[合歡山|合歡山]]南麓的濁水溪，水流含有多量的「粘板岩」，故溪水呈黑灰色。
