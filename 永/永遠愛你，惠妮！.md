{{作品名稱|name=''Whitney''|hk= |tw=永遠愛你，惠妮！|cn=惠特尼}}，是一部2018年英國紀錄片，由[[凱文·麥克唐納|凱文·麥克唐納]]執導。入選[[第71屆坎城影展|第71屆坎城影展]]午夜單元<ref name="THR">{{cite web|url=https://www.hollywoodreporter.com/review/whitney-cannes-2018-1112681 |title='Whitney': Film Review Cannes 2018 |work=The Hollywood Reporter |accessdate=2018-05-17}}</ref><ref name="VF">{{cite web|url=https://www.vanityfair.com/hollywood/2018/05/whitney-houston-documentary-abuse |title=Whitney Houston’s Dark Family Secret Uncovered in New Documentary |work=Vanity Fair |accessdate=2018-05-17}}</ref>。

== 劇情 ==
該片記錄美國歌手[[惠特妮·休斯顿|-{zh-tw:惠妮·休斯頓; zh-cn:惠特妮·休斯顿; zh-hk:雲妮·侯斯頓;}-]]的人生。

== 製作 ==
英國導演[[凱文·麥克唐納|凱文·麥克唐納]]製作該部紀錄片長達兩年時間，重現出[[惠特妮·休斯顿|-{zh-tw:惠妮·休斯頓; zh-cn:惠特妮·休斯顿; zh-hk:雲妮·侯斯頓;}-]]聚光燈外的真實人生<ref>{{Cite web |url = http://www.epochtimes.com/b5/18/5/17/n10402652.htm |title = 奧斯卡名導團隊打造 紀錄片《惠妮》坎城首映 |author = 梁夢竹 |date = 2018-05-17 |publisher = 大紀元 |language = zh-hant |accessdate = 2018-05-17 }}</ref>。
