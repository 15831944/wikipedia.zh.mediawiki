'''永州职业技术学院'''（{{lang-en|'''YongZhou Vocational and Technical College'''}}）位于[[湖南省|湖南省]][[永州市|永州市]]，是湖南省一所高等[[专科大学|专科大学]]。<ref name="jj " >{{cite web |url=http://www.hnyzzy.com/s/1/t/237/p/1/c/134/d/146/list.jspy |title=永州职业技术学院学校简介 |publisher=永州职业技术学院 |accessdate=2011 |work= |language =zh-cn}}</ref>

== 历史 ==
2000年7月，永州职业技术学院建立，它的前身是[[零陵卫校|零陵卫校]]和[[零陵农校|零陵农校]]。<ref name="jj " />

== 学校院系 ==
永州职业技术学院分为6大系：医学系、艺术系、农学系、信息系、商贸系、理工系，共计40个专业。<ref>{{cite web|url=http://www.hnyzzyzs.com/special.php/109.html|title=院系介绍|publisher=永州职业技术学院|accessdate=2011|work=|language=zh-cn}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 校训 ==
笃学守信、志存高远

== 荣誉 ==
学院先后荣获全国职业教育先进单位、全国高职高专人才培养水平评估优秀学院、国家教学成果一等奖、湖南省文明高校、湖南省职业教育先进单位、湖南省普通高校就业工作先进单位、湖南省普通高等学校党建工作先进单位等20多项重大荣誉。2010年10月，学院顺利通过国家示范性高等职业院校验收，跻身全国高职院校100强。<ref>{{cite web |url=http://www.hnyzzy.com/s/1/t/237/p/1/c/134/d/150/list.jspy |title=学校荣誉 |publisher=永州职业技术学院 |accessdate=2011 |work= |language =zh-cn}}</ref>
