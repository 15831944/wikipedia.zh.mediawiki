'''永瀨未萌'''（{{lang-ja|永瀬みなも}}，{{bd|1998年|2月12日}}），[[日本|日本]]的[[AV女優|AV女優]]，[[靜岡縣|靜岡縣]]出身<ref name="PRESTIGE"/>。目前為「[[PRESTIGE|PRESTIGE]]」專屬女優<ref name="PRESTIGE"/>。

== 人物 ==
* 興趣是[[追星|追星]]，特技是不管在哪裡都可以睡著<ref name="PRESTIGE"/>。

* 2018年9月29日，在「MGS」動畫平台（MGStage）以《【初撮り】ネットでAV応募→AV体験撮影 768 みなも 20歳 元グラビアアイドル》首次亮相<ref>{{Cite web |url=https://www.mgstage.com/product/product_detail/SIRO-3607/ |title=「【初撮り】ネットでAV応募→AV体験撮影 768 みなも 20歳 元グラビアアイドル」|publisher=MGS動画 |date=2018-09-29|language=ja}}</ref>。

* 2018年10月7日，在網路電視「[[AbemaTV|AbemaTV]]」的節目《給與明細》中透露自己AV出道作拍攝兩天的酬勞為300萬[[日圓|日圓]]（折合約[[新台幣|新台幣]]81萬3000元）<ref>{{cite web |url=http://www.playno1.com/article-28149-1.html#.XGZq5rhS-Uk |title=金字塔的頂端！最強新人永瀬まなも的月薪是…  |publisher=PLAYNO.1玩樂達人 |date=2018-10-08}}</ref><ref>{{cite web |url=http://www.tvlife.jp/entame/192373 |title=『給与明細』元アイドル・永瀬みなものセクシー女優デビューに密着 |publisher=TVLIFE web |date=2018-10-09 |language=ja}}</ref><ref>{{cite web |url=https://www.chinatimes.com/realtimenews/20181011003398-260404 |title=一出道薪資直接登頂 G奶AV「頂級新人」幹2天天價片酬曝光！  |publisher=[[中時電子報|中時電子報]] |date=2018-10-11}}</ref>。

* 16歲時曾以'''田邊飛鳥'''名義擔任靜岡地區[[偶像|偶像]]團體「マーブルメイプル」的一員<ref>{{cite web |url=http://www.playno1.com/article-28877-1.html#.XGZq4bhS-Uk |title=鄉親啊！永瀬みなも不能說的秘密揭曉了！ |publisher=PLAYNO.1玩樂達人 |date=2018-12-22}}</ref>，大約有一到兩年的時間。之後開始挑戰[[寫真偶像|寫真偶像]]，也曾考慮過要成為[[歌手|歌手]]或者[[演員|演員]]<ref>{{cite web |url=https://sirabee.com/2018/12/07/20161905653/?FROM_WEBVIEW |title=初回ギャラ「破格の300万」の新人セクシー女優・永瀬みなも　しみけんも絶賛の「最終兵器」とは |publisher=しらべぇ |date=2018-12-07 |language=ja}}</ref>。
