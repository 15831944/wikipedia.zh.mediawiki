'''駁二藝術特區'''（{{lang-en|'''The Pier-2 Art Center'''}}）是一個位於[[臺灣|臺灣]][[高雄市|高雄市]][[鹽埕區|鹽埕區]]的藝術園區，以'''前衛'''、'''實驗'''、'''創新'''為理念來打造國際藝術平台。

== 簡介 ==
「駁二」係指第二號接駁碼頭，位於[[高雄港|高雄港]]第三船渠內，建於1973年6月12日，原為一般的港口倉庫。2000年[[高雄市政府|高雄市政府]]因尋找[[中華民國國慶煙火|中華民國國慶煙火]]施放場所，偶然發現這個具有實驗性的場域。但因年久失修，進駐單位針對舊建物的狀態進行空間的各項整建工程，於2002年3月24日完工。

在藝術家及地方文化工作者推動之下，結合[[文建會|文建會]]閒置空間再利用的專案資源，駁二藝術特區為高雄市[[社區總體營造|社區總體營造]]的代表性作品。歷經'''高雄駁二藝術發展協會'''與'''[[樹德科技大學|樹德科技大學]]發展地方藝術工坊'''經營，駁二藝術特區成為[[臺灣南部|臺灣南部]]的實驗創作場所。2006年，駁二藝術特區由[[高雄市政府文化局|高雄市政府文化局]]接手經營。

駁二藝術特區由[[高雄市政府文化局|高雄市政府文化局]]接手經營後，便舉辦一系列的高雄設計節、好漢玩字-{}-節、鋼雕藝術節、貨櫃藝術節及高雄人來了大公仔等藝文展覽。2010年，高雄市政府文化局邀請[[這牆音樂|這牆音樂]]進駐月光劇場，於每週末舉辦流行及獨立樂團音樂演唱會。2015年，高雄市政府文化局接手營運管理月光劇場及C10 Live Warehouse。

2010年[[索尼電腦娛樂|索尼電腦娛樂]]進駐新開闢的九號倉庫，設置數位產業中心，進行遊戲軟體研發及測試；2012年8月，該址由兔將創意影業股份有限公司進駐，進行3D轉換以及視覺特效服務。2011年，C3倉庫委外由帕莎蒂娜國際餐飲公司規劃為倉庫餐廳。2016年1月29日，委外由[[in89豪華數位影城|in89豪華數位影城]]經營的「in89駁二電影院」（映捌玖駁二電影院）正式開幕。

為滿足逐年成長的參訪者，與舉辦更多元的藝文展覽，2012年高雄市政府文化局開始租用[[台灣糖業公司|台灣糖業公司]]的蓬萊倉庫，使得駁二藝術特區的倉庫建築擴充至14棟，並於2012年5月4日至5月16日在新場域順利舉辦2012[[青春設計節|青春設計節]]展覽活動。大義倉庫群於2013年底啟用，駁二總計有25棟展場。

鑒於由高雄市政府文化局直接經營之因素，區內工作人員組成模式與其它私人機構稍微不同。除了公務行政管理、常設展覽正職以及特殊展覽工讀之外，亦有[[替代役|替代役]]文化服務役役男在此服役；同時，也於每年暑假招募實習生，讓更多青年學子得以有機會接觸到第一線的展場工作。

== 爭議 ==
[[File:The_Pier-2_Swimming_Pool_Artwork_201507.JPG|thumb]]
在2015年夏天於園區內的C8、C9大義倉庫屋頂上，蓋了兩座隱藏版公共藝術「泳池天台」，並打造寬闊平台可在高處近看港邊各式船舶與不遠處[[壽山_(高雄市)|柴山]]美景。其中C8屋頂上的泳池供遊客走入水面下的泳池內恣意遊玩，或與岸上友人互動及拍照；而C9屋頂上的泳池則是在水面下放置鏡子，在水中鏡射天空雲影與觀者<ref>{{cite news |title = 【更新】金澤21世紀美術館泳池　高雄駁二也有 |publisher = 蘋果新聞 |url = http://www.appledaily.com.tw/realtimenews/article/new/20150718/650693 |date = 2015-07-18 |author= 周昭平 |accessdate = 2015-07-21 |language = zh-hant}}</ref>。但因設計創意明顯和日本[[金澤21世紀美術館|金澤21世紀美術館]]的泳池裝置藝術雷同，引發外界質疑抄襲<ref>{{cite news |title = 【更新】金澤21世紀美術館泳池　高雄駁二也有 |publisher = 民視 |url = http://news.ftv.com.tw/NewsContent.aspx?ntype=class&sno=2015720U04M1 |date = 2015-07-20 |author= 鄭博暉、陳凱茂 |accessdate = 2015-07-21 |language = zh-hant}}</ref>。駁二介紹此公共藝術作品時也說到，「與藝術家[[林德羅·厄利什|林德羅·厄利什]]創作的《泳池》有異曲同工之妙」<ref>{{Cite web  |title = 史政府帶頭違法？　高駁二「泳池天台」被爆抄襲 |publisher = 東森新聞 |url = http://www.ettoday.net/news/20150719/537301.htm |date = 2015-07-19 |author= 江貞億 |accessdate = 2015-07-22 |language = zh-hant}}</ref>。高市文化局解釋，兩處泳池作品雖有雷同處，但「泳池天台」非抄襲之作，而是重新打造出不同巧思的泳池作品，讓民眾不用出國也能近距離欣賞公共藝術的無限可能<ref>{{cite news |title = 〈南部〉創意新視界 駁二藝區玩透視 |publisher = 自由時報 |url = http://news.ltn.com.tw/news/local/paper/899247 |date = 2015-07-19 |author= 黃佳琳 |accessdate = 2015-07-21 |language = zh-hant}}</ref>。

在事件爆發後，駁二坦承是援引原作概念，雖有不同手法，並強調並非藝術家創作的公共藝術，為駁二公共設施的一部分<ref>{{Cite web  |title = 駁二泳池天台抄襲名作　無限期關閉 |publisher = 蘋果日報 |url = http://www.appledaily.com.tw/realtimenews/article/new/20150719/651170/ |date = 2015-07-19 |author= 周昭平 |accessdate = 2015-07-22 |language = zh-hant}}</ref>。另外，高市文化局表示，「泳池天台」以「倉庫整修工程」名義公開招標，是駁二公共設施而非「公共藝術」，雖然也經委員會審議，卻不是依《公共藝術設置辦法》辦理，所以設計該案者不是「藝術家」，審查者也不是視覺藝術專家<ref>{{Cite web  |title = 泳池爭議 駁二致歉關閉 |publisher = 自由時報 |url = http://news.ltn.com.tw/news/supplement/paper/899498 |date = 2015-07-20 |author= 楊明怡、黃佳琳 |accessdate = 2015-07-24 |language = zh-hant}}</ref>。被抄襲的[[阿根廷|阿根廷]]藝術家林德羅·厄利什將此事全權交給他在亞洲的日本經紀人處理<ref>{{Cite web  |title = 泳池天台爆爭議 無限期關閉 |publisher = 中國時報 |url = http://www.chinatimes.com/newspapers/20150720000403-260115 |date = 2015-07-20 |author= 吳垠慧、吳江泉 |accessdate = 2015-07-21 |language = zh-hant}}</ref>。卻又在當日晚間，高雄市文化局長[[史哲|史哲]]透過媒體公開發表致歉文，分別向原創者、高雄市民以及藝文界工作者道歉<ref>{{Cite web  |title =【更新】駁二搞山寨泳池　高市文化局道歉 |publisher = 蘋果日報 |url = http://www.appledaily.com.tw/realtimenews/article/new/20150719/651170/ |date = 2015-07-22 |author= 石秀華 |accessdate = 2015-07-24 |language = zh-hant}}</ref>。
