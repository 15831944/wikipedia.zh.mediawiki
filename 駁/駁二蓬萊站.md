'''駁二蓬萊站'''位於[[高雄市|高雄市]][[鹽埕區|鹽埕區]]，為[[高雄捷運|高雄捷運]][[高雄捷運環狀輕軌|環狀輕軌]]的車站。

== 車站概要 ==
車站代碼為'''C13'''。站區位於[[西臨港線自行車道|西臨港線自行車道]]、駁二特區，瀨南街路口，定位為中間站；月台形式為[[側式月台|側式月台]]。

== 車站構造 ==
*[[側式月台|側式月台]]兩座。

== 歷史 ==

===高雄環狀輕軌第一段C1-C14車站全民參與命名活動===
* 命名活動第一階段（2014年10月18日至11月20日）建議站名：香蕉碼頭站、漁人碼頭站、蓬萊站
* 命名活動第二階段（2015年2月16日至3月6日）票選選項：駁二蓬萊站、香蕉碼頭站、高雄港站
* 命名結果（2015年3月29日公告）：駁二蓬萊站<ref>[http://www.thenewslens.com/post/144007/ 高雄環狀輕軌C1-C14站有名字了！最快8月通車]</ref>
