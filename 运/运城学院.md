'''运城学院'''，是中國[[教育部|教育部]]批准在原[[运城|运城]]高等[[专科|专科]]学校基础上独立设置的一所地方性综合普通[[本科|本科]]院校，地处[[山西|山西]]南部[[运城市|运城市]]，隶属山西省人民政府领导，[[山西|山西]]省教育厅管理。

==历史沿革==
# 1978年5月，运城[[师范|师范]][[专科|专科]]学校筹建并招生。
# 1983年7月，运城教育学院成立。
# 1985年3月，河东大学开始筹建。
# 1989年12月，经原国家教委批准，[[山西省|山西省]]人民政府决定，运城师范专科学校与运城教育学院、河东大学(筹)合并组建运城高等专科学校。
# 1999年6月，运城高等专科学校启动“强校升本”工程；2000年8月，[[山西省|山西省]]人民政府正式向国家[[教育部|教育部]]申报以运城高专为基础，建立运城学院。
# 2002年1月9日，全国大学设置评议委员会在[[广西|广西]][[桂林|桂林]]举行会议，投票通过以运城高专为基础建立运城学院。
# 2002年2月22日，国家[[教育部|教育部]]通知[[山西省|山西省]]人民政府，正式批准运城学院建立。
# 2002年4月5日，[[山西省|山西省]]人民政府正式行文批准独立设置运城学院。
# 2002年6月，“运城学院”[[印章|印章]]正式启用。
# 2003年3月10日，运城学院党政领导班子宣布成立。
