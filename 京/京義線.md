'''京義線'''（{{kor|諺=경의선|漢=京義線|羅=Gyeong'ui seon|f=y}}），為[[韓國鐵道公社|韓國鐵道公社]]路線，連結[[首爾特別市|首爾特別市]][[龍山區_(首爾)|龍山區]][[首爾站|首爾站]]與[[京畿道|京畿道]][[坡州市|坡州市]][[都羅山站|都羅山站]]。

原先首爾至新義州間可直通并進入[[中國|中國]]。但是{{link-ko|韓半島分裂|한반도 분단}}以後，所謂的「京義線」只剩下[[首爾站|首爾站]]前往[[三八線|軍事分界線]]以南的[[都羅山站|都羅山站]]為止（[[首爾站|首爾站]]至[[汶山站|汶山站]]有[[首都圈電鐵|首都圈電鐵]]運行，[[汶山站|汶山站]]至[[都羅山站|都羅山站]]有一般列車運行），而北朝鮮境内的京義線，現已以平壤為界，分為[[平義線|平義線]]及[[平釜線|平釜線]]。

==歷史==
京義線名稱由來，是取路線起點，[[大韓帝國|大韓帝國]]京城的「京」字，及終點[[新義州市|新義州]]的「義」字作名稱。<ref name="Gyeongui-jp">{{cite web |title=京義線 |url=http://ja.wikipedia.org/wiki/京義線 |publisher=wikipedia |language=ja |accessdate=2013-09-18}}</ref>19世紀末，[[大韓帝國|大韓帝國]]原打算自行興建京義線鐵路，由於資金問題，計畫未能展開。直至1902年，由[[首爾|漢城]]至[[開城特級市|開城]]的一段正式動工。<ref>[http://s03.megalodon.jp/2009-0606-1103-03/japanese.yonhapnews.co.kr/misc/2009/04/16/9000000000AJP20090416003200882.HTML 今日の歴史（5月8日）] 聯合ニュース 2009/05/08付  閲覧</ref>。

[[甲午戰爭|甲午戰爭]]後，[[大日本帝國|日本]]希望控制[[京釜線|京釜線]]，以便從日本運送物質往[[朝鮮半島|朝鮮半島]]。1901年日本取得興建權並開始[[京釜線|京釜線]]工程。此時[[俄羅斯帝國|俄國]]在中國開始擴張鐵道網，包括把中國的[[中東鐵路|中東鐵路]]向東北伸延至俄國的[[海參崴|海參崴]]、南面的滿州支線由大連伸延至旅順。出於對朝鮮半島的影響及日後可能與俄國發生軍事衝突的考慮，日本對京釜線及京義線十分重視。<ref name="imperial-japan">{{cite web |author=Nakano Akira |title=Korea's Railway Network the Key to Imperial Japan's Control |url=http://www.japanfocus.org/-Nakano-Akira/2533 |publisher=The Asia-Pacific Journal: Japan Focus |language=en |accessdate=2013-09-16}}</ref>1904年[[日俄戰爭|日俄戰爭]]爆發，[[大韓帝國|大韓帝國]]被逼與日本簽訂協議，同意讓日本在軍事行動時控大韓帝國的鐵路及工程，京義線由日本軍隊建築。1905年日俄戰爭結束，日本取得原本由俄國控制於中國東北南部的鐵道權益，亦取得大韓帝國所有鐵道的權力<ref name="imperial-japan" />，並繼續鐵道餘下的工程，至1906年4月3日京義線全線通車。<ref name="lineop">{{cite web |url=http://info.korail.com/2007/kra/ope/ope08000/w_ope08606.jsp |title=경영원칙 > 경영공시 > 영업현황 > 영업거리현황 |publisher=Korail |language=ko |accessdate=2010-12-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20110722192217/http://info.korail.com/2007/kra/ope/ope08000/w_ope08606.jsp |archivedate=2011-07-22 }}</ref>

此時京義線以[[龍山站_(首爾)|龍山站]]為起點，經[[汶山站|汶山]]、[[開城站|開城]]、[[沙里院青年站|沙里院]]、[[平壤站|平壤]]、[[新安州青年站|新安州]]、[[定州站|定州]]、[[鹽州郡|南市]]等站，至位於中朝邊境[[鴨綠江|鴨綠江]]南岸的[[新義州青年站|新義州]]為終點。由於京釜線於1905年1月已完成及運作，<ref name="imperial-japan" />日本可以從[[釜山市|釜山]]經京釜線及京義線到達中國邊境。日本方面再計劃興建[[鴨綠江大橋|鴨綠江大橋]]，以便利用鐵道到達[[中國東北|中國東北]]。有關大橋於1911年完工，使日本可以經1910年8月已被吞併的朝鮮半島，從[[釜山|釜山]]一直通過[[滿洲國|滿洲]]及[[蘇聯|蘇聯]]直達歐洲。1945年3月全線完成[[複線鐵路|複線鐵路]]工程。

由於[[朝鮮戰爭|朝鮮戰爭]]使南北分治，現時在[[大韓民國|南韓]]方面的路線，只餘下首爾市內到[[京畿道|京畿道]]西北的[[都羅山站|都羅山站]]。在[[朝鮮民主主義人民共和國|北韓]]方面，從[[開城|開城]]至[[都羅山站|都羅山站]]的線段被廢止了。餘下的線段，從[[開城|開城]]北上至[[平壤|平壤]]的一段，被重新命名作[[平釜線|平釜線]]，而從[[平壤|平壤]]到[[新義州|新義州]]的一段，則被稱為[[平義線|平義線]]。

雖然京義線因為南北分治而斷開了，但南北雙方現時正在商議，希望可以重新開通從都羅山到開城的一段鐵路。

=== 京義線復元歷程 ===
* 2000年6月15日 - 南北共同宣言。
* 2002年4月11日 - 都羅山站開業。
* 2002年9月18日 - [[三八线|三八线]]（[[朝韩非军事区|朝韩非军事区]]）的鐵路工程開展。
* 2003年6月14日 - 京義線/東海線連結儀式。(南北雙方一起舉行)
* 2003年12月31日 - 南側區間完工。
* 2007年5月17日 - 汶山—開城試運行，兩国代表團参加。
* 2007年12月11日 - 都羅山站與北韓方面的板門站之間開始作定期貨物列車運輸，以方便把原材料運往開城工業區及把裝成品運回南韓，每週開一班車。
* 2008年11月28日 - 因南北關係惡化，跨越三八線的貨物列車停止運行。
* 2008年12月1日 - 因北韓「12・1」措施，跨越三八線的貨物列車正式停止運行。
* 2009年8月25日 - 北韓取消「12・1」措施，跨越三八線的貨物列車恢復運行。
* 2013年4月3日 - 北韓宣佈不准南韓人員、車輛及貨運列車進入開城工業區，三八線的貨物列車不能進入開城工業區，只能返回都羅山站。
* 2013年4月10日 - 因北韓單方面宣佈暫時關閉開城工業區，跨越三八線的貨物列車再次停止運行。
* 2013年9月16日 - 朝鮮半島局勢緩和，開城工業區重開，跨越三八線的都羅山站至開城的貨物列車恢復運行。

===鐵道提升與發展===
當北韓以平釜線及平義線的名稱取代境內的京義線後，京義線僅餘南韓境內由首爾市到都羅山全長55.8公里的一段，而且只有單軌。由於首爾地區的經濟發展，來往汶山與首爾之間工作的人數眾多，亦因為發展西北部鄉郊[[衛星城市|衛星城市]]如[[高陽市|高陽市]]及[[坡州市|坡州市]]，南韓政府決定把京義線與鄉郊鐵路規格統一並進行升級。2005年[[韓國建設交通部|韓國建設交通部]]計畫把這段鐵道改為[[複線鐵路|複線鐵路]]、電氣化及把部份路段拉直，<ref name="KRNA_2010">{{cite web |url=http://www.krnetwork.or.kr/service/condition/RouteMapView.do?PA_MAP_ID=12419 |title=용산∼문산 복선전철 |publisher=Korea Rail Network Authority |accessdate=2013-09-16}}</ref>最初預計在2009年完工，2010年運作。<ref name="donga-20050822">{{cite web|title=首爾江南～龍山地鐵將被建成雙線地鐵|url=http://china.donga.com/gb/srv/service.php3?bicode=020000&biid=2005082231358|language=zh-cn|publisher=[[東亞日報|東亞日報]]|date=2005-08-22}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>工程於1999年11月展開，最初預算為₩19,700億。<ref name="rgi-korailplans">{{cite web |url=http://www.railwaygazette.com/news/single-view/view/10/south-koreas-growing-network.html |title=South Korea's growing network |date=2008-09-08 |publisher=[[國際鐵路公報|Railway Gazette International]] |accessdate=2013-09-16}}</ref>到2009年7月1日，由[[數碼媒體城站|數碼媒體城]]至[[汶山站|汶山]]的工程已完成，餘下[[加佐站|加佐]]至[[孝昌站|孝昌]]將以地下鐵路形式興建。此時已耗用了預算的74%，預算最後被修正為₩21,532.71億，<ref name="KRNA_2010"/>整段鐵道完工日期為2014年。<ref>{{Cite web |title =[수도권II] 경의선 효창정거장 부지 220m 때문에 |publisher=[[朝鮮日報|朝鮮日報]] |date=2011-08-25 |url=http://news.chosun.com/site/data/html_dir/2011/08/24/2011082402671.html |language=ko | accessdate = 2013-09-16}}</ref>因地鐵化而留下的地面空間，會用作興建公園。<ref>{{cite web |url=http://news.chosun.com/site/data/html_dir/2009/11/09/2009110900025.html |title=폐철로 위에 핀 '녹색문화공원' |language=ko |date=2009-11-09 |publisher=[[朝鮮日報|朝鮮日報]] |accessdate=2010-12-30}}</ref>

2010年9月，南韓政府頒佈計劃，於2020年把全國路程2小時以下的路線縮短百分之五行駛時間。<ref name="plans2020">{{cite web |title=Bullet trains coming to a town near you by 2020 |url=http://www.rapid-rail.com/go/rapidrail/_dbe,news,_auto_7019817.xhtml |date=2010-09-01 |publisher=Rapid Rail |accessdate=2010-09-15 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>按此計劃，京義線未來需提速至每小時230公里。

===現況===
京義線已電氣化並納入[[韓國首都圈電鐵|韓國首都圈電鐵]]，現在運作於[[大韓民國|南韓]]境內的京義線，若以無民用限制且能夠乘電鐵系統直接到達的，是由[[首爾特別市|首爾特別市]][[中區_(首爾)|首爾中區]]或[[龍山區_(首爾)|龍山區]]開往[[京畿道|京畿道]][[汶山站|汶山]]的一段。

京義線由[[龍山站_(首爾)|龍山站]]至[[加佐站|加佐站]]，亦即稱為[[龍山線|龍山線]]<ref name="Yongsan_Line" />的路段，現-{只}-有[[孔德站|孔德站]]至[[加佐站|加佐站]]的一段在運作。預計2014年可啟用接駁[[首爾地鐵6號線|6號線]]的[[孝昌站|孝昌站]]，並延伸至現有的龍山站，整段長度變為7.0公里，龍山也重新成為京義線的起點。龍山站現可轉乘[[首都圈電鐵1號線|1號線]]、[[中央線_(韓國)|中央線]]、[[韓國高速鐵道|湖南高速線]]，並預計於2018年與與建中的[[新盆唐線|新盆唐線]]第二期延伸線連接。<ref name="Shinbundang">{{cite web |title=新盆唐線 |url=http://zh.wikipedia.org/wiki/新盆唐線 |publisher=wikipedia |accessdate=2013-09-14}}</ref>其餘各站，[[孔德站|孔德站]]可轉乘[[首爾地鐵5號線|5號線]]、[[首爾地鐵6號線|6號線]]及[[仁川國際機場鐵路|機場鐵路]]；[[弘大入口站|弘大入口站]]可轉乘[[首爾地鐵2號線|2號線]]及[[仁川國際機場鐵路|機場鐵路]]。龍山段由[[孔德站|孔德]]開往京義線終點的[[汶山站|汶山]]平均每小時10班，上班尖峰時段每小時更達20班。<ref name="Seoulmetro">{{cite web |title=Seoul Metro Traffic Center |url=http://dmzap1.seoulmetro.co.kr/station/eng/linemap.action |publisher=Seoul Metro |language=en |accessdate=2013-09-14 }}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

京義線由[[加佐站|加佐]]至[[首爾站|首爾站]]的一段長5.8公里<ref name="Seoulmetro" />，均建於地面。在首爾站可轉乘[[首爾地鐵1號線|1號線]]、[[首爾地鐵4號線|4號線]]、[[仁川國際機場鐵路|機場鐵路]]、[[京釜線|京釜線]]及[[韓國高速鐵道|京釜高速線]]。但此段實為京義線的首爾支線。由[[首爾站|首爾站]]開出的列車，多以[[數碼媒體城站|數碼媒體城]]為終點，直接往[[汶山站|汶山]]方向的每小時平均-{只}-有一班，上班尖峰時段每小時亦-{只}-有3班，<ref name="Seoulmetro" />遠比龍山線<ref name="Yongsan_Line" />疏落，因此常需要於[[數碼媒體城站|數碼媒體城]]或[[加佐站|加佐]]轉乘京義線[[汶山站|汶山]]方向的列車。<ref name="Gyeongui">{{cite web |title=Gyeongui line |url=http://en.wikipedia.org/wiki/Gyeongui_Line |publisher=wikipedia |language=en |accessdate=2013-09-14}}</ref>

龍山線<ref name="Yongsan_Line" />與首爾支線於[[加佐站|加佐站]]匯合後，由[[數碼媒體城站|數碼媒體城]]起至[[汶山站|汶山]]的各站均建於地面，<ref name="Gyeongui" />由[[加佐站|加佐站]]至[[汶山站|汶山站]]全長40.5公里。<ref name="Seoulmetro" />在[[數碼媒體城站|數碼媒體城]]可轉乘[[首爾地鐵6號線|6號線]]及[[仁川國際機場鐵路|機場鐵路]]外，於[[幸信站|幸信站]]可轉乘[[韓國高速鐵道|京釜高速線]]及[[韓國高速鐵道|湖南高速線]]，[[大谷站_(京畿道)|大谷站]]亦可轉乘[[首爾地鐵3號線|3號線]]。

京義線雖屬於[[韓國首都圈電鐵|韓國首都圈電鐵]]，但-{只}-有孝昌至加佐才以地鐵行走，全長不到京義線的3%，有別於其他首都圈地鐵線。這除了因為沿用近百年前興建的鐵道外，亦因為在龍山線<ref name="Yongsan_Line" />與首爾支線之後，大部分路線所經的都是鄉郊、田野、林地及農地，還有很多經過地區公路的鐵道交叉口，需要鐵道員工執勤控制。

至於由汶山經[[雲泉站_(京畿道)|雲泉]]、[[臨津江站|臨津江]]至[[都羅山站|都羅山]]一段，全長9.8公里。該段主要是為住在汶山至臨津江站的居民而設，雲泉站更是只有一個月臺的單線車站，並可讓京義線的列車在本站掉頭。在京義線電氣化前，由首爾開往臨津江除一輛[[無窮花號|無窮花號]]列車外，都是以[[通勤型電力動車組|通勤車]]行駛。京義線電氣化後，通勤車-{只}-行走汶山至臨津江，部分會繼續往接近[[朝鮮民主主義人民共和國|北韓]]邊境的都羅山。若由臨津江繼續往都羅山，要通過檢查才可進入。都羅山站是2002年才啟用的民用管制區，內有北韓的出入境管制站及海關，到此站需經旅行社或親自於汶山站或臨津江站申請。<ref>{{cite web |title=都羅山駅 |url=http://ja.wikipedia.org/wiki/都羅山駅 |publisher=wikipedia |language=ja |accessdate=2013-09-14}}</ref><ref>{{cite web |title=【非軍事區】往北之路迢迢 都羅山車站 |url=http://synnwang.blogspot.com/2011/06/dorasan-station.html |publisher=Synn |date=2011-07-06 |language=zh |accessdate=2013-09-14}}</ref>現除了用作運送物資往北韓[[開城工業地區|開城工業地區]]外，每天尚有4班由[[首爾站|首爾站]]開出，運載旅客往都羅山站觀光的列車。<ref>http://info.korail.com/servlets/renew.sta.sta02000.sw_sta02207_v1Svt?code=0403&name=%B5%B5%B6%F3%BB%EA{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

今日南韓境內的京義線長度，汶山至都羅山的9.8公里，加上龍山至汶山的47.3公里，或是首爾至汶山的46.3公里，<ref name="Seoulmetro" />相對於1945年7月朝鮮半島南北分裂前原京義線的499.3公里，也不過是全長的11%左右。<ref name="Gyeongui-jp" />
