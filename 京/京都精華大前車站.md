'''京都精華大前車站'''（{{jpn|j=京都精華大前駅|hg=きょうとせいかだいまええき|rm=Kyōto-seikadai-mae eki|f=y}}）是位於[[日本|日本]][[京都市|京都市]][[左京區|左京區]]的[[叡山電鐵|叡山電鐵]][[叡山電鐵鞍馬線|鞍馬線]]的車站。為叡山電鐵之中為是最新的車站。車站編號為'''E12'''。

== 車站結構 ==
擁有[[相對式月台|相對式月台]]2面2線的[[無人車站|無人車站]]。兩月台的全體皆被遮蓬覆蓋，且兩邊的遮蓬、柱子、牆壁的設計並不相同。兩月台皆有各自的出入口。

基本上為無人車站，但在大學繁忙的時期（開學典禮或入學考時）或早上通勤時間則有配置工作人員。因此在兩個出入口的旁邊有辦公處。另外往鞍馬車站方向的月台有[[驗票閘門|驗票閘門]]，運作時間以學校假日以外的白天為主。大學校內的畫籤堂有發售一部份的回數券。

兩個月台被由京都精華大學所建設的，由[[義大利|義大利]][[建築師|建築師]]、[[安德烈亞·帕拉弟奧|帕拉弟奧]]設計的桁架橋連結，橋位於車站靠出町柳一側不遠處。

下行月台～住宅區、上行月台～大學正門附近的路段有斜坡等無障礙設施，但要到鐵軌另一側則只能靠「帕拉弟奧橋」，並沒有站內或鄰接的平交道，使用輪椅的話得繞一大段路。

== 車站周邊 ==
位於[[京都精華大學|京都精華大學]]所座落的山的斜面的山麓處。上行月台和一部份山的斜面相連。

就如同站名，本站是離京都精華大學最近的車站。從上行月台爬上階梯、或從下行月台渡過帕拉弟奧橋之後的位置有大學的側門，離校園中心部比大學正門還要近。可說是正是為了大學而設立的車站。

在反方向一側則是位於狹窄山谷間的農耕地。從2000年左右住宅開始增加，但仍然是以農耕地和停車場為主。

* [[圓通寺_(京都市)|圓通寺]]（車內廣播會提及）
* 賴光橋
* [[京都大學|京都大學]]實驗林
* [[綜合地球環境學研究所|綜合地球環境學研究所]]

== 歷史 ==
*1989年（平成元年）9月21日 開通。當初在現在上行鐵軌的位置設有臨時月台。
*1990年（平成2年）9月28日 包含本站、[[岩倉車站_(京都府)|岩倉車站]] - [[二軒茶屋車站_(京都府)|二軒茶屋車站]]之間再次複線化。成為現在的姿態。
