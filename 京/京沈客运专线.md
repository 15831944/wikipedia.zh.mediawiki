'''京瀋客运专线'''是[[中国|中国]]一条建设中连接[[北京市|北京市]]与[[辽宁省|辽宁省]][[沈阳市|沈阳市]]的[[高速铁路|高速铁路]]，是[[京哈客运专线|京哈客运专线]]的一部分。京瀋客运专线与已经投入运营的[[哈大客运专线|哈大客运专线]]构成了沈阳及中国[[东北地区|东北地区]]进[[关内|关内]]的第二条快速铁路通道。客运专线起于[[沈阳站|沈阳站]]，经[[新民|新民]]、[[阜新|阜新]]、[[朝阳|朝阳]]、[[承德|承德]]进入[[北京|北京]]，线路全长699.864千米，其中新建铁路693.369千米<ref name="xinhuanet21558304">{{cite web |url=http://www.bj.xinhuanet.com/bjpd_sdzx/2010-12/06/content_21558304.htm |title=京沈客运专线在京设五站点 最快2014年通车 |publisher=新华网 |date=2010-12-06 |accessdate=2010-12-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20121028193806/http://www.bj.xinhuanet.com/bjpd_sdzx/2010-12/06/content_21558304.htm |archivedate=2012-10-28 }}</ref>，设计时速350公里，估算投资961.6亿元人民币，2010年预计2014年竣工通车，但由于开工时间一再推迟导致通车时间也一并延后。2014年3月5日，京沈客运专线正式开工建设<ref>{{cite news|url=http://crh.gaotie.cn/jingshen/2014-05-07/157855.html|title=北京至沈阳铁路客运专线开建|publisher=高铁网|date=2014-05-07|accessdate=2014-05-08}}</ref>，[[承德南站|承德南站]]至[[沈阳站|沈阳站]]段于2018年12月29日载客运营<ref name="people">{{cite news|url=http://politics.people.com.cn/n1/2016/0918/c1001-28719622.html|title=京沈高铁辽宁段开始铺轨|publisher=经济日报|date=2016年9月18日|accessdate=2016年9月18日}}</ref><ref>{{Cite news|url=https://mp.weixin.qq.com/s/rVmtwszI4Gbw6JCTwDWlvQ|title=2019年1月5日京沈高铁承德至沈阳段正式开通运营！|author=|date=|work=世界轨道交通资讯网|accessdate=2018-12-03}}</ref>，[[北京站|北京站]]至承德南站段预计于2020年底开通运营。

== 概况 ==
京瀋客运专线位于[[中国|中国]][[东北地区|东北]]和[[华北地区|华北]]，连接[[环渤海经济圈|环渤海]]和东北工业基地两个经济区域，全线纵贯[[北京市|北京]]、河北的[[承德市|承德]]、[[辽宁省|辽宁]]的[[朝阳市|朝阳]]、[[阜新市|阜新]]、[[锦州市|锦州]]{{#tag:ref|京沈客运专线经过锦州市辖黑山县北部，设黑山北站|group="注"}}、[[沈阳市|沈阳]]三省六市。

客运专线所建设的地方大部分在历史上属于[[热河省|热河省]]，地貌多为山地和丘陵。1956年，原属热河省的三个市和各个旗县被分别归属了[[河北省|河北省]]、[[内蒙古自治区|内蒙古自治区]]和[[辽宁省|辽宁省]]。50多年来，由于此地各市在各个省区内所处地理位置偏远，地理环境复杂，在工程技术落后的情况下这一带交通建设十分滞后。铁路网络大部分还是[[满洲国|满洲国]]时期兴建的单线铁路，如锦承线，魏塔线等，早已不能满足现代物流运输的发展。而本地区所处的区位重要性也在下降，使得经济发展缓慢。 

京瀋客运专线作为中国“四纵四横”客运专线网的重要组成部分，是在[[承德市|承德市]]提出的京承快速铁路，辽宁省提出的沈朝客运线的设计基础设上，将线路延伸联通[[承德市|承德市]]与辽宁朝阳、阜新，向东北连接沈阳、[[長春市|长春]]、[[哈爾濱市|哈尔滨]]。这是[[中華人民共和國鐵道部|铁道部]]2009年开展的第一个重点踏勘项目。它是国家实施扩大内需战略的重要内容，也是充分考虑把阜新、朝阳、承德等城市从经济发展二线纳入一线的客观需要。与此同时此线路缓解了目前入关唯一一条通道的运载力问题和战略压力。

因计划在北京市区使用原[[京包铁路|京包铁路]]旧线路，2012年12月9日，有部分居住在北京市[[朝阳区_(北京市)|朝阳区]]的居民担心高铁建成后会影响他们的正常生活以及担心自己的房价因此下跌<ref>{{cite web|url=http://finance.sina.com.cn/china/20130417/005915169144.shtml|title=京沈高铁环评黑幕：同意有礼品不同意就不发卷|publisher=新浪网}}</ref>，而举行了在北京罕见的抗议活动，这次抗议得到了北京相关部门的默许，并未像其它类似活动那样受到阻扰，此类抗议活动以及北京政府部门的冷淡态度致使京沈客运专线开工一再延后。<ref>{{cite web|url=http://www.reuters.com/article/2012/12/09/china-protest-idUSL4N09J0B120121209 | title=Rare Beijing protest takes aim at high speed rail project |date=2012-12-09 |accessdate=2012-12-09|language=en}}</ref>

项目业主单位共两个：
*京沈铁路客运专线辽宁有限责任公司,于2009年8月7日在沈阳成立。承担京沈（辽宁境内段）、[[沈丹客运专线|沈丹]]、[[盘营客运专线|盘营]]三条客运专线建设任务。
*京沈客运专线工程(京冀)公司。地址：北京市东城区老钱局胡同16号。

本项目投资估算总额1245亿元，其中工程投资1203亿元，[[动车组|动车组]]购置费42亿元。其中项目资本金为总投资的50%计622.5亿元，由[[中国铁路总公司|中国铁路总公司]]出资365.5亿元，使用[[铁路建设基金|铁路建设基金]]和企业自有资金；北京市、河北省、辽宁省负责境内段征地拆迁工作及费用(计约257亿元)，由地方政府采用财政性资金等出资；资本金以外的50%使用银行贷款。

== 智能高铁试验<ref>{{Cite web|url=https://weibo.com/ttarticle/p/show?id=2309404233022881856336#_0|title=中国首次开展智能高速铁路关键技术综合试验|accessdate=2018/04/26|author=|date=2018/04/26|publisher=}}</ref><ref>{{Cite web|url=http://www.peoplerail.com/rail/show-456-377853-1.html|title=铁路总公司高速铁路智能关键技术综合试验暨京沈高速综合试验动员会召开 - 行业 - 人民铁道网 - 中国官方铁路门户|accessdate=2018-04-27|work=www.peoplerail.com}}</ref>==
2018年3月份以来，[[中国铁路总公司|中国铁路总公司]]在京沈高铁辽宁段，全面启动“高速铁路智能关键技术综合试验”，推进中国智能高铁建设，试验将一直持续到2018年9月底。这是中国首次进行智能高铁试验，时速350公里“[[复兴号电力动车组|复兴号]]”长编组动车组，就是试验的主要内容之一。本次试验成果将应用于[[京张城际铁路|京张高铁]]、[[京雄铁路|京雄城际]]等智能铁路的建设。同时可以使高速铁路运营更加安全高效、更加绿色环保、更加便捷舒适。

试验包括时速350公里“[[复兴号电力动车组|复兴号]]”长编组动车组试验、智能高铁调度集中系统CTC功能试验、铁路LTE-R宽带移动通信系统试验等27项内容；试验是对智能高速铁路的列车编组、牵引控制系统、调度指挥系统、通讯信息系统和安全保障系统的一次综合集成试验。

== 车站 ==
全程共拟设置20座车站，其中北京设4站（[[星火站|星火]]、[[顺义西站|顺义西]]、[[怀柔南站|怀柔南]]、[[密云站|密云]]<ref>{{cite web|url=http://www.bcactc.com/home/gcxx/tdzbgg_show.aspx?fid=2421&publicsource=%d6%d0%b9%fa%cc%fa%c2%b7%d7%dc%b9%ab%cb%be|title=新建北京至沈阳铁路客运专线北京段（不含动车运用所）施工监理资格预审公告（代招标公告）|publisher=[[中国铁路总公司|中国铁路总公司]]|date=2015-08-28|accessdate=2015-08-31}}</ref>），承德市设5站（[[兴隆西站|兴隆西]]、[[安匠站|安匠]]、[[承德南站|承德南]]、[[承德县北站|承德县北]]、[[平泉北站|平泉北]]），辽宁设11个车站（[[牛河梁站|牛河梁]]、[[喀左站|喀左]]、[[奈林皋站|奈林皋站]]、[[朝阳站_(辽宁省)|朝阳]]、[[北票站|北票]]、[[乌兰木图站|乌兰木图]]、[[阜新站|阜新]]、[[黑山北站|黑山北]]、[[新民北站|新民北]]、[[沈阳西站|沈阳西]]、[[沈阳站|沈阳]]）<ref name="xinhuanet21558304" /><ref name="BJJ01">{{cite web|url=http://www.bcactc.com/home/gcxx/tdzbgg_show.aspx?fid=1040&publicsource=%b1%b1%be%a9%cc%fa%c2%b7%be%d6|publisher=北京市建设工程信息网|date=2013-07-30|accessdate=2013-08-05|title=新建铁路北京至沈阳客运专线（京冀段）地质勘察监理招标公告}}</ref>。

==歷史==
京瀋客运专线已于2009年10月获批<ref>{{cite web |url=http://www.fuxin.gov.cn/center/fxxw/zwyw/2010/01/04/96134.html |title=城市向北百姓目光北移——瞭望玉龙新城 |publisher=中国阜新政府门户网站 |date=2010-01-04 |accessdate=2010-01-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20100829122455/http://www.fuxin.gov.cn/center/fxxw/zwyw/2010/01/04/96134.html |archivedate=2010-08-29 }}</ref>，当时预计2010年开始建设。
*2013年2月，開始開工前測評工作。<ref>{{cite web|url=http://crh.gaotie.cn/jingshen/2013-02-09/61897.html|title=京沈客运专线年内正式开工|publisher=高铁网}}</ref>
*2013年7月30日，《新建铁路北京至沈阳客运专线（京冀段）地质勘察监理招标公告》发布<ref name="BJJ01" />。
*2013年12月可行性研究获批<ref>[http://www.china-railway.com.cn/xwc/tlxw/201312/t20131228_40993.htm 中国铁路总公司启动京沈客专施工准备工作]</ref>。
*2014年3月5日，辽宁段控制工程三棱山隧道开工，拉开了京沈客运专线建设的序幕<ref>{{cite news|url=http://www.ln.chinanews.com/Item/869103.aspx|title=北京至沈阳铁路客运专线开建|publisher=中国新闻网|date=2014-05-04|accessdate=2014-05-09|deadurl=yes|archiveurl=https://web.archive.org/web/20140508170358/http://www.ln.chinanews.com/Item/869103.aspx|archivedate=2014-05-08}}</ref>。
*2014年3月28日，京沈客运专线承德段开工建设<ref>{{cite news|url=http://www.aastocks.com/tc/stocks/analysis/china-hot-topic-content.aspx?id=200000328966&type=17&catg=2|title=京沈客運專線承德段29日開工,承德北京1小時可達}}</ref>。
*2015年10月，北京段施工承包单位确定，项目指挥部同时进驻。<ref>{{cite web|url=http://www.bcactc.com/home/gcxx/tdzbgs_show.aspx?ID=1190|title=新建北京至沈阳铁路客运专线北京段站前工程（不含动车运用所）施工总价承包招标中标候选人公示|publisher=北京工程建设交易信息网|date=2015-10-12|accessdate=2015-11-08|archive-url=https://web.archive.org/web/20160304121540/http://www.bcactc.com/home/gcxx/tdzbgs_show.aspx?ID=1190|archive-date=2016-03-04|dead-url=yes}}</ref>
*2015年12月30日，北京段正式开工<ref>{{cite web|url=http://news.sohu.com/20160106/n433583205.shtml|title=京沈高铁北京段正式开工 设4车站|author=吴为|date=2016-01-05|publisher=搜狐网}}</ref>.
*2016年9月17日，辽宁段正式开始铺轨<ref name="people"/>。
*2017年1月10日，辽西隧道贯通。
*2017年2月16日，三棱山隧道贯通。
*2017年8月12日，辽宁段完成无砟轨道板的铺设<ref>{{Cite web|url=http://www.gov.cn/xinwen/2017-08/12/content_5217409.htm#1|title=京沈高铁辽宁段轨道板铺设完成_图片新闻_中国政府网|accessdate=2018-02-06|last=刘杨|work=www.gov.cn}}</ref>。
