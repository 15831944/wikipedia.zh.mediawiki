'''京津冀煤改气'''是指[[中华人民共和国中央政府|中国中央政府]]和京津冀三地政府通过推进各项市政工程，最终以[[中华人民共和国天然气|天然气]]取代[[中华人民共和国煤炭政策|煤炭]]的[[中华人民共和国天然气政策|能源政策]]。由于三地同处[[京津冀城市群|京津冀城市群]]，经济、政治关系紧密，通常将三地政策合并描述<ref name="a1"/><ref>{{Cite web |url=http://news.163.com/17/1113/18/D351KL06000187VE.html |title=本年度京津冀“煤改气”工程即将完工，共涉及约190多万户|accessdate=2017-12-04 |author=|date=2017-11-13  |work= |publisher=网易，来源：澎湃新闻(上海)|language=Zh-hans }}</ref>。

对于这一政策，除全国[[煤改气|煤改气]]政策本身三方面{{noteTag|1=席菁华文章提及的三点质疑：1.经济性争议，2.气源保障难题，3.治霾效果存疑。}}的质疑外。有认为在无政府补贴的情况下，河北农村地区居民的收入无法负担煤改气后冬季采暖费用<ref>{{Cite web |url=http://www.sohu.com/a/124991830_463997 |title=再谈煤改气|accessdate=2017-12-04 |author=|date=2017-01-23 |work= |publisher=搜狐网|language=Zh-hans }}</ref>。2017年进入供暖季，在天然气价格不断上涨和严禁燃煤之下，京津冀地区与山西、山东、河南等地一起陷入[[2017年中国北方供暖危机|供暖危机]]。

== 背景 ==

== 进程 ==

=== 北京市 ===

北京市自2009年推行煤改气<ref name="a1"/>{{noteTag|1=席菁华文章原文：北京“煤改气”至今已推行七年。}}。

2017年4月6日，[[北京市燃气集团|北京市燃气集团]]召开“决胜2017煤改气行动”誓师动员大会，全面启动北京市2017年煤改气工作。此次煤改气工程量为历年平均的五倍以上，被认为是北京市历史上最大规模的煤改气工程<ref>{{Cite web |url=http://money.163.com/17/0407/10/CHDOA68E002580S6.html |title=京津冀环境标准提升 北京启动史上最大规模煤改气 |accessdate=2017-12-04 |author=|date=2017-04-07 |work= |publisher=网易，来源：证券市场周刊·红周刊(北京)|language=Zh-hans }}</ref>。整个工程预计5月全面施工，9月完成实体工程，10月中旬通气<ref>{{Cite web |url=http://news.k618.cn/society/201704/t20170406_10886537.html |title=北京市2017年煤改气全面启动 |accessdate=2017-12-04 |author=记者：戴琪，编辑：未闻 |date=2017-04-06 |work= |publisher=未来网，来源：千龙网 |language=Zh-hans |archive-url=https://web.archive.org/web/20171204171113/http://news.k618.cn/society/201704/t20170406_10886537.html# |archive-date=2017-12-04 |dead-url=yes }}</ref>。要求在2017年冬采暖季之前，[[丰台区|丰台区]]、[[通州区_(北京市)|通州区]]、[[大兴区|大兴区]]、[[房山区|房山区]]的农村地区实现[[无煤化|无煤化]]。[[北京六环路|六环]]以内，在2017年底之前全部取消燃煤锅炉<ref>{{Cite web |url=http://beijing.qianlong.com/2017/0407/1580393.shtml |title=北京全面启动2017“煤改气”工作 打好蓝天保卫战 |accessdate=2017-12-04 |author=作者：叶晓彦，责任编辑：鲁路（QM0002）|date=2017-04-07 |work= |publisher=千龙网，来源：北京晚报|language=Zh-hans }}</ref>。

=== 河北省 ===

河北省农村地区80%的散煤燃烧用于冬季取暖。《京津冀大气污染防治强化措施（2016—2017年）》，将与北京市接壤的[[保定市|保定市]]、[[廊坊市|廊坊市]]下辖的十八个[[县级行政区|县级行政区]]为禁煤区<ref>{{Cite web |url=http://news.hebei.com.cn/system/2017/03/05/017911584.shtml |title=散煤归零，河北省18县市区打造禁煤区 |accessdate=2017-12-04 |author=作者：段丽茜|date=2017-03-05 |work= |publisher=河北网，来源：河北日报 |language=Zh-hans }}</ref>，而河北省与北京市交接地区本就处于[[环首都贫困带|环首都贫困带]]。河北省将在2017年10月前，完成[[保定市|保定市]]、[[廊坊市|廊坊市]]的禁煤区建设。而河北省颁布的《河北省农村散煤治理专项实施方案》，预计在2020年前，在农村地区禁止散煤燃烧<ref>{{Cite web |url=http://hb.jjj.qq.com/a/20170407/013010.htm |title=河北：今后在农村做饭取暖不再烧煤 |accessdate=2017-12-04 |author=崔虹|date=2017-04-07 |work= |publisher=燕赵晚报|language=Zh-hans }}</ref>。
