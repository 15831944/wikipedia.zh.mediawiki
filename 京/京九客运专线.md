'''京九客运专线'''，亦稱'''京港高铁'''，北起[[中國|中國]]首都[[北京市|北京]]的[[北京西站|北京西站]]，南至[[香港|香港]][[西九龍|西九龍]]的[[香港西九龙站|香港西九龙站]]，跨越京、[[河北省|冀]]、[[河南省|豫]]、[[湖北省|鄂]]、[[湖南省|湘]]、[[广东省|粤]]、港等地，是连接[[环渤海经济圈|环渤海]]和[[珠江三角洲地区|珠三角]]的重要[[高速鐵路|高速鐵路]]<ref>{{cite web  |url=http://news.163.com/15/1010/15/B5J0A8P400014AEE.html | title=京九高铁走向确定 看看都经过哪些城市(组图)
}}</ref>。全线由多条线路接驳而成，分段建设，主要分为[[京雄铁路|北京至雄安新区段]]、[[雄商铁路|雄安至商丘段]]、[[商杭客运专线|商丘至阜阳段]]、阜阳至九江段（分东线和西线，[[合安九客运专线|合安九高铁]]和[[阜冈九高铁|阜冈九高铁]]）、[[昌九城际线|南昌至九江段]]、[[昌赣客运专线|南昌至赣州段]]、[[赣深客运专线|赣州至深圳段]]，經[[广深港高速铁路香港段|香港段]]至香港西九龙站為終點站。

== 争论 ==
京九高速铁路雄安至商丘段，沿线濮阳、范县、梁山等地民众为高铁走向发生争论，引发高铁“争夺大战”，该段线路具体走向至今未能完全确定。阜阳至九江段，潢川和固始也发生了类似的争论。对于各地政府在高铁规划中的“争夺大战”，全国政协委员、交通部党组书记杨传堂回应：“理解各地争取高铁设站的心情，但高铁不可能建到所有的县和乡，最终还是要按照规划建设。”<ref>{{cite web |author1=周宇 |title=“高铁不可能建到所有的县和乡” |url=http://politics.rmlt.com.cn/2018/0309/513219.shtml |publisher=北京青年报 |access-date=2018-06-29 |archive-url=https://web.archive.org/web/20180629074402/http://politics.rmlt.com.cn/2018/0309/513219.shtml |archive-date=2018-06-29 |dead-url=yes }}</ref>
