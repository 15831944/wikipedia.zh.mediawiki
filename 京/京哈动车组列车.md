'''京哈动车组列车'''是[[中国铁路|中国铁路]]运行于中华人民共和国首都[[北京市|北京市]]及[[黑龙江省|黑龙江省]]省会[[哈尔滨市|哈尔滨市]]之间的[[普通动车组列车|动车组列车]]，自2007年4月18日起开行，现由[[沈阳局集团|沈阳局集团]]沈阳客运段及[[哈尔滨局集团|哈尔滨局集团]]哈尔滨客运段负责客运任务。列车使用[[和谐号CRH5型电力动车组|和谐号CRH5A型电力动车组]]<ref>[http://news.sina.com.cn/c/p/2007-04-16/224012792186.shtml 图文：京哈线和谐号动车组列车首次公开亮相] [[新浪网|新浪网]] 2007年04月16日</ref>，运行于[[京哈铁路|京哈铁路]]，途经[[沈阳北站|沈阳北站]]、[[长春西站|长春西站]]等车站，全长1241公里，运营时速为160-200公里，最高运营时速为250公里。目前每日共开行2对，每日最早开车时间为7点05分，最晚开车时间为15点15分。其中[[北京站|北京站]]至[[哈尔滨西站|哈尔滨西站]]下行使用车次为D27次、D101次（单号）；哈尔滨西站至北京站上行使用车次为D28次、D102次（双号）。运行时间为7小时39分至8小时03分不等<ref>[http://tour.jschina.com.cn/system/2012/11/27/015344947.shtml?COLLCC=1168530319& 哈大高铁调运行图 北京增开哈尔滨长春方向动车] {{webarchive|url=https://web.archive.org/web/20140808053947/http://tour.jschina.com.cn/system/2012/11/27/015344947.shtml?COLLCC=1168530319& |date=2014-08-08 }} 中国江苏网 2012-11-27</ref>。

== 历史 ==
[[File:巡道工出品_photo_by_Xundaogong——康宁桥下动车_-_panoramio.jpg|thumb]]
因[[中国铁路第六次大提速|中国铁路第六次大提速]]在2007年4月18日开始实行，为缩短北京至哈尔滨间的列车运行时间，开始开行D25/26次、D27/28次共2对[[普通动车组列车|动车组列车]]<ref>[http://news.sohu.com/20070417/n249505705.shtml 哈铁局公布哈至京动车组时刻表与票价] [[搜狐网|搜狐网]] 2007年04月17日</ref>，全线运行于[[京哈铁路|京哈铁路]]。为增加运能，在2011年1月11日又开行了D101/102次列车<ref>[http://www.sjzonline.com/news/jiaotong/2010-12-22/26656.html 2011年北京铁路局调图新旧交替过渡方案出台] 石家庄热线 2010-12-22</ref>。2012年12月1日，因[[哈大客运专线|哈大客运专线]]通车，列车在沈阳北至哈尔滨区间改为在[[哈大客运专线|哈大客运专线]]行驶，同时终到站由[[哈尔滨站|哈尔滨站]]改为[[哈尔滨西站|哈尔滨西站]]，并加开D29/30次列车<ref>[http://hlj.sina.com.cn/news/ljyw/2012-11-26/070123342_2.html 哈尔滨西站始发列车共23个车次] [[新浪网|新浪网]] 2012年11月26日</ref>。

2015年9月1日，因[[哈齐客运专线|哈齐客运专线]]开通，原D25/D26次、D29/D30次列车延长至[[齐齐哈尔南站|齐齐哈尔南站]]始发终到<ref>{{cite news|url=http://news.163.com/15/0831/15/B2BUFL1500014JB5.html|title=北京站开行至齐齐哈尔南、珲春方向高速动车组列车|publisher=中国广播网(北京)|date=2015-08-31 |accessdate=2016-12-13}}</ref>。在每年11月15日至翌年3月15日期间，D25次列车临时缩短至哈尔滨西站终到；每年11月16日至翌年3月16日期间，D26次列车临时缩短至哈尔滨西站始发。2019年1月5日起，D25/D26次调整为[[佳木斯站|佳木斯站]]始发终到<ref>{{cite news|url=http://liaoning.nen.com.cn/system/2018/12/10/020736047.shtml|title=明年1月5日沈铁实施新运行图|publisher=辽沈晚报|date=2018-12-10 |accessdate=2019-01-13}}</ref>。

== 事故 ==
*2007年5月12日，D26次列车在行驶在[[四平站|四平站]]至[[毛家店站|毛家店站]]之间时，出现了驾驶室微机故障情况，导致列车停车数小时，最终列车晚点5小时左右于5月13日4时30分到达[[北京站|北京站]]<ref>[http://news.163.com/07/0513/17/3ECVRO2A000120GU.html 动车组D26次途中抛锚 晚点近五小时险成闷罐车] [[网易|网易]] 2007-05-13</ref>。
*2013年11月22日，D28次列车在行驶在[[台安站|台安站]]至[[盘锦北站|盘锦北站]]之间时，发生了[[2013年京哈铁路列车撞人事故|撞人事故]]，导致施工工人4死1伤<ref>[http://news.ifeng.com/mainland/detail_2013_11/27/31585435_0.shtml 铁总：D28次动车晚点撞死4人因工人违章] [[凤凰网|凤凰网]] 2013-11-27</ref>。
