== 登場角色 ==
=== 主要人物 ===
; {{nihongo|真城葵|真城 葵（ましろ あおい）|聲：[[富田美憂|富田美憂]]<ref>[https://animeanime.jp/article/2018/07/09/38623.html 「京都寺町三条のホームズ」富田美憂が好きでコレクションしているモノは？ 意外な素顔に迫ったインタビュー]</ref>}}
: 因為和前男友與朋友的感情問題，想要籌錢搭新幹線回埼玉，把家中的兩幅古董畫帶到了「藏」，卻輕易的被清貴看出其用意。現今正在藏打工。喜歡清貴。

; {{nihongo|家頭清貴|家頭 清貴（やがしら きよたか）|聲：[[石川界人|石川界人]]}}
: 骨董店「藏」的年輕鑑定師，[[京都府立大學|京都府大]]畢業，[[京都大學|京都大學]]的準研究生。目前幫忙爺爺顧店。因為「家頭」這個姓的發音很像「福爾摩斯」，所以被人稱作福爾摩斯先生，事實上也真如福爾摩斯一般，除了高超的鑑定技術外擁有過人的觀察及推理能力。

=== 次要人物 ===
; {{nihongo|圓生|円生（えんしょう）|聲：[[遊佐浩二|遊佐浩二]]}}
: [[南禪寺|南禪寺]]剛遁入空門的僧人，其實是偽造骨董犯。偽造的骨董與真品維妙維肖，沒有專精的鑑定技術無法成功辨偽。原本打算收手出家，卻受到清貴的刺激，暗地再度出山，偽造的骨董甚至能騙進美術館展示，偽造的書法被清貴識破後逃離南禪寺。
; {{nihongo|梶原秋人|梶原 秋人（かじわら あきひと）|聲：[[木村良平|木村良平]]}}
: 知名藝人，出身京都世家。委託結束後經常來骨董店作客。
; {{nihongo|家頭誠司|家頭 誠司（やがしら せいじ）|聲：[[小山力也|小山力也]]}}
: 清貴的祖父，骨董店「藏」的老闆，卻經常不在店裡。
; {{nihongo|家頭武史|家頭 武史（やがしら たけし）|聲：[[上田燿司|上田燿司]]}}
: 清貴的父親。小說家。早年喪妻。
; {{nihongo|瀧山利休|滝山 利休（たきやま りきゅう）|聲：[[小林沙苗|小林沙苗]]}}
: 腹黑美少年，較葵小一歲。自小崇拜清貴，雖然彼此待如兄弟，利休卻如侍從般環繞清貴身旁。曾在藏打工，對古美術品鑒定亦頗有見地，後來曾一度往法國留學學習柔道。對平凡的葵略有意見，經常促狹葵。
; {{nihongo|瀧山好江|滝山 好江（たきやま よしえ）|聲：[[大原沙耶香|大原沙耶香]]}}
: 外表30多歲，實際40多歲的美女。利休的母親，清貴祖父誠司的情人。利休是前段婚姻的孩子。
; {{nihongo|宮下香織|宮下 香織（みやした かおり）|聲：[[木下鈴奈|木下鈴奈]]}}
: 葵現在的高中同學，京都知名和服店千金。有個被選上[[齋王|齋王]]的姐姐，兩人皆會[[華道|華道]]。但不同於姐姐，心思細膩，時常為大局著想。委託結束後變成葵的好朋友。
