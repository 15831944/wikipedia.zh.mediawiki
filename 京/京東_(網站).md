'''京東商城'''（{{NASDAQ|JD}}）是[[中國大陆|中國大陆]]一家主要為[[B2C|B2C]]模式的購物網站，前稱'''360buy'''，由[[劉強東|刘强东]]創辦。2014年，京东集团在美国[[纳斯达克|纳斯达克]]证券交易所上市。

== 历史 ==
京東商城在2004年由現任[[行政總裁|行政總裁]][[劉強東|劉強東]]創辦，名称源自刘强东和前女友龚晓京的名字。<ref>{{Cite web|url=http://tech.qq.com/a/20101208/000269.htm|title=刘强东欲效仿李国庆：京东上市将邀初恋庆功_科技_腾讯网|accessdate=2018-05-27|work=tech.qq.com}}</ref>。初期稱為“京东多媒体网”，最早銷售電腦產品，後來出現各種商品。

2011年京東在成都成立了[[雲計算|雲計算]]中心。<ref>{{Cite web|url=http://www.cnetnews.com.cn/2011/0826/2054645.shtml|title=刘强东：京东拥抱云计算 成都数据中心明年启用 - CNET科技资讯网|accessdate=2018-05-27|work=www.cnetnews.com.cn}}</ref>

2013年3月初，京东网站部分域名更换，以JD.COM作为主域名<ref>{{cite web|title=京东启用JD.com为主域名：存技术难题与品牌变数|url=http://www.chinaz.com/news/2013/0305/294508.shtml|work=站长之家|accessdate=2013-03-09}}</ref>。同年3月30日，京东正式更换域名为JD.COM，并对首页进行改版，推出新的[[Logo|Logo]]与吉祥物形象“Joy”（一只[[卡通|卡通]][[金属|金属]][[狗|狗]]）<ref>{{cite web|title=京东正式更换域名为JD.com 吉祥物joy是条狗|url=http://news.xinhuanet.com/info/2013-03/31/c_132273891.htm|publisher=新华网|accessdate=2013-03-31}}</ref>。

== 爭議事件 ==
=== 涉嫌做空子公司 ===
2014年4月1日，京东因涉嫌[[做空|做空]]子公司，侵犯子公司和小股东权益被提起诉讼。<ref>{{Cite web|url=http://tech.163.com/14/0418/02/9Q32RU4C000915BF.html|title=京东20元转让合作方80%股权 被指掏空后贱卖_网易科技|accessdate=2018-05-27|last=网易|work=tech.163.com}}</ref>
=== 販售假貨手機事件 ===
2015年5月份[[央視|央視]]以專題報導型式發表京東商城販賣大量來路不明蘋果手機事件<ref>[http://news.cntv.cn/2015/05/24/VIDE1432444858437151.shtml 京東網購出貨大量翻新機]</ref>，許多消費者在京東直營自銷網頁上相信京東聲譽而購買的蘋果手機，短則一兩月長則數月就發生大量當機，前往蘋果維修點結果被告知是舊廢品翻新機也可能含有假貨零件，因此被原廠拒絕維修，蘋果公司由於採取一機多條碼稽核制度，可能因此才揭露此次京東事件，任何蘋果機的主機板、外殼、卡托皆有一組條碼，原裝機三碼必須一致。

=== 用户数据泄露 ===
2016年12月10日，有媒体报道<ref>{{cite web|title = 疑似京东12个G数据遭泄露，身份证、密码、电话等敏感信息无一逃脱|url = https://www.huxiu.com/article/174140.html|publisher = 虎嗅网|accessdate = 2016-12-14}}</ref>大小为12G的疑似京东用户数据文件泄漏，其中包括身份证、电话、密码等敏感信息。12月11日，京东旗下公众号“京东黑板报”回应称泄漏的数据来源是2013年Struts 2的安全漏洞问题<ref>{{cite web|title = 关于有媒体报道京东数据安全问题的声明 |url = https://mp.weixin.qq.com/s/Ke1I15J384p3NYgO6iejnQ|publisher = 京东黑板报|accessdate = 2016-12-14}}</ref>。

=== 作家[[六六|六六]]質疑京東銷售變質和假冒商品 ===
2017年4月，作家[[六六|六六]]在微博上发出一条消息称，有消费者从京东平台买到了腐烂变质的水果，而京东却不接受退货。她表示，消费者通过一次次的失望，已经否定京东的冷链能力。

2018年3月，作家[[六六|六六]]又在微信公众号发布文章提到她友人在京东上買到假冒商品，京东客服卻辯稱是发错货。京东随后进行道歉，表示对其朋友遇到的消费问题负责到底。刘强东也发布内部信表示，六六事件后，全集团的高管开展了深刻的反思，为此，京東专门推出了全流程更高标准的客户满意度准则，并在集团层面成立了客户卓越体验部<ref>[https://www.lieyunwang.com/archives/447058 百度向作家六六道歉，将下线与“上海美国领事”相关的代办签证等广告].獵雲網.</ref>。

=== 售卖违法违规商品被查 ===
2018年4月2日，北京市网信办、[[北京市公安局|北京市公安局]]、北京市通信管理局、北京市工商局、北京市文化市场行政执法总队就京东涉嫌售卖违法违规商品、出版物及其他印刷品的行为，依法约谈网站相关负责人，并责令整改<ref>{{cite web|title =京东被北京市多部门依法联合约谈 |url = http://www.xinhuanet.com/fortune/2018-04/02/c_1122628156.htm|publisher = 新华网 |accessdate = 2018-04-03}}</ref>。
不久《圣经》被京东、淘宝、当当、亚马逊等网站下架。淘宝、当当和亚马逊只剩下圣经故事、圣经精读等改编版本，京东则完全屏蔽搜索结果。此外，京东还屏蔽了《古兰经》的搜索结果。<ref>{{cite web|title=Solidot {{!}} 网店从上个月开始被要求下架《圣经》|url=https://www.solidot.org/story?sid=56056|website=www.solidot.org}}</ref>

=== 京东自营渠道买到假茅台 ===
2018年5月10日，有多位山东客户发现购买的“43度飞天茅台500ml”存在无nfc、无mt、标签印刷出错等问题，怀疑京东销售”假茅台“。京东随后发表声明，称该系列酒的供应商为北京糖业烟酒集团京酒销售有限公司，为正规经销商。京东初步判定运输途中可能存在调包情况。京东称将会对受到影响的客户进行赔付。<ref>{{cite news|author1=IT之家 远洋|title=京东回应客户买到假茅台：运输过程中被调包|url=https://www.ithome.com/html/it/359324.htm|accessdate=2018-05-13}}</ref><ref>{{cite news|author1=网易财经|title=京东回应平台出售假茅台：运输路途被调包 已报警|url=http://money.163.com/18/0511/07/DHGQS5OU002580SS.html|accessdate=2018-05-14}}</ref><ref>{{cite news|author1=和讯名家|title=京东：自营渠道假茅台因被调包，客户：这次不会被公关|url=https://finance.qq.com/a/20180511/033903.htm|accessdate=2018-05-14}}</ref>
