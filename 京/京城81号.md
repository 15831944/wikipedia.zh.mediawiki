《'''京城81号'''》（{{lang-en|'''The House That Never Dies'''}}），曾用名“朝内81号”<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2014-05-13/21424140966.shtml |title=《京城81号》定名曝预告 林心如化蝶 |date=2014-05-13 |publisher=新浪娱乐 |work= |language=zh-hans}}</ref>，导演[[叶伟民|叶伟民]]，主演吴镇宇、林心如、莫小棋、杨佑宁、秦海璐、李菁。<ref>{{cite web |url=http://www.sd.xinhuanet.com/news/2014-02/26/c_119513489.htm |title=吴镇宇林心如演《朝内81号》 比肩鬼吹灯和盗墓笔记 |date=2014-02-26 |publisher=新华网 |work= |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20140322145657/http://www.sd.xinhuanet.com/news/2014-02/26/c_119513489.htm |archivedate=2014-03-22 }}</ref><ref>{{cite web |url=http://gb.cri.cn/27564/2014/02/13/3465s4422609.htm |title=林心如、吴镇宇邂逅《朝内81号》 上演乱世纠葛 |date= 2014-02-13|publisher=国际在线 |work= |language=zh-hans}}</ref>该片是[[3D|3D]]惊悚影片，取材于[[北京|北京]]四大凶宅之首“朝内81号”。<ref>{{cite web |url=http://ent.163.com/14/0311/08/9N1RTF1D000300B1.html |title=《朝内81号》曝民俗异事 打造民国版“聊斋” |date=2014-03-11 |publisher=网易 |work= |language=zh-hans}}</ref>

2014年5月13日，影片正式更名为《京城81号》。对于此次“更名”之举，《京城81号》项目人表示是出于对影片故事本身的考虑，改名后格局更大，更加符合影片中包含的丰富的剧情元素，同时还否定此前谣传的“改档”之说，《京城81号》确定在2014年7月18日以3D形式上映。<ref>{{cite web |url=http://ent.163.com/14/0527/17/9T93CTHF000300B1.html |title =《京城81号》曝新版海报 吴镇宇林心如成阴阳脸 |date=2014-05-27 |publisher=网易 |work= |language=zh-hans}}</ref>

==剧情==
该片讲述“朝内81号霍氏一族”一百年来的命运。<ref>{{cite web |url=http://ent.163.com/14/0219/11/9LEO777N000300B1.html |title=《朝内81号》霍家三少混搭亮相 联手"闹凶宅" |date=2014-02-19 |publisher=网易 |work= |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20140322144353/http://ent.163.com/14/0219/11/9LEO777N000300B1.html |archivedate=2014-03-22 }}</ref>民国初年，位于京城八大胡同内的青楼歌舞升平，尤以头牌陆蝶玉艳惊四座，引得万人空巷。但她早已倾心京城大户霍家三少爷，几经波折，陆蝶玉嫁入霍家。怎料新婚之夜新郎却变成刚刚过世的霍家二少爷，冥婚后不久陆蝶玉离奇怀孕，一时鬼胎传言四起，家族上下不安，不久便举家惨遭灭门，留下阴宅保留至今……

一百年后，81号古宅后人许若卿重返故居，百年前发生在这里的惨案历历在目。同时，她的出现也意外唤起古宅内尘封已久的孤魂孽怨……

==制作==
* [[文隽|文隽]]坦言，“好的故事应该从民间汲取养分。被封为经典的《[[聊齋志異|聊齋志異]]》，作者蒲松龄先生花了几十年的时间收集大量的民间志怪故事。在准备剧本前，片方花了近三年时间走访取材，收集到3000多页的素材，让影片剧本丰富精彩。<ref>{{cite web |url=http://ent.163.com/14/0311/08/9N1RTF1D000300B1.html |title=《朝内81号》曝民俗异事 打造民国版“聊斋” |date=2014-03-11 |publisher=网易娱乐 |work= |language=zh-hans}}</ref>”
* 《朝内81号》在[[3D|3D]]方面的投资就有几千万, 建造一座真的朝内81号，然后用3D技术拍摄，单单建这一栋楼就投资1千万，这在业内目前是唯一的。<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2014-01-23/14334086938.shtml |title=《朝内81号》幕后团队：做业界良心3D大片 |date=2014-01-23 |publisher=新浪娱乐 |work= |language=zh-hans}}</ref>
* 监制[[文隽|文隽]]介绍，《朝内81号》这个项目和此前的惊悚片有很大不同，“国内惊悚片做到现在，都有点瞧不起自己了。为保证所建凶宅的艺术质量，剧组请到《[[色·戒|色·戒]]》美术指导刘世运，金马、金像双奖造型师张世杰把关，百余人团队历时一个月完工。<ref>{{cite web |url=http://news.mtime.com/2013/06/23/1513726.html |title=3D惊悚片"朝内81号"探班 林心如吴镇宇入主凶宅 |date=2013-06-23 |publisher=Mtim光网 |work= |language=zh-hans}}</ref>
* 7片中使用多达12900个道具，如雕花的门框、奢华的木棺、古朴的摆设，这些道具很难搞定，里面有些是古董，需要到处去借用，而且要注意不能碰坏，很贵。
* 《京城81号》在亿级规模的保驾护航下，1：1还原了“京城81号”老宅和京城八大胡同繁华盛景，动用大批古董当做道具，将片中霍家的奢华内景全数呈现出来。<ref>{{cite web |url=http://sn.ifeng.com/caijing/newswire/detail_2014_07/14/2590555_0.shtml |title=《京城81号》点映开启“惊悚大片”时代 |date=2014-07-15|publisher=凤凰网 |work= |language=zh-hans}}</ref>

==宣传发行==
监制陈辉表示，宣发和话题营销功不可没，“上映前4个月，观众就能在影院看到电影的预告片，"鬼宅"朝内81号话题的衍生，也在各社交网络大力推广，公交站牌、地铁通道、移动电视的宣传也无处不在。专注于东方惊悚，是该片票房成功的原因，毕竟人人都有本土元素归属感，人们会忍不住一探究竟。” 另外，在该片上映时，该片还做了很多落地活动。长长的营销周期，精准的落地发行，再配合好的档期，是《京城81号》票房的有力保证。<ref>{{cite web |url=http://news.hexun.com/2014-09-08/168268698.html|title=“短板”阻碍国产惊悚片挖潜|date=2013-09-08 |publisher=  齐鲁晚报  |work= |language=zh-hans}}</ref>

==票房==
《京城81号》破惊悚片首日五大纪录。

数据显示，该片凌晨点映平均上座率高达74%，堪比好莱坞大片电影规模。而18日首日上座率突破预期，平均高达84.05%，高出排名第二电影约40个百分点，在黄金时间段更是达到96%上下，呈现出一票难求的观影盛况，在人潮带动下，周六排片更是比首日高出10个百分点，令人咋舌。在超高上座率推动下，《京城81号》迎来首日开门红，拿下约4500万票房，首日一举打破国内惊悚片零点场纪录、排片纪录、上座率纪录、票房纪录以及空间上升最快纪录。<ref>{{cite web |url=http://ent.qq.com/a/20140719/021026.htm |title=《京城81号》破五大纪录 影市逆转被调侃鬼上身 |date=2014-07-19|publisher=腾讯娱乐 |work= |language=zh-hans}}</ref>

在上座率方面，惊悚片也无太大进展，一部几百万小规模惊悚片的上座率大致维持在5%—10%之间，而《京城81号》的出现却接连打破惊悚片排片和上座率的常态，创下国产惊悚片最高首日排片和最高上座率双项纪录。<ref>{{cite web |url=http://et.21cn.com/gundong/etscroll/a/2014/0719/14/27846693.shtml |title=林心如《京城81号》破惊悚片首日五大纪录 |date=2014-07-19 |publisher=21CN |work= |language=zh-hans |access-date=2014-07-19 |archive-url=https://web.archive.org/web/20140810002538/http://et.21cn.com/gundong/etscroll/a/2014/0719/14/27846693.shtml |archive-date=2014-08-10 |dead-url=yes }}</ref>从上映首日15%的排片上升到第二天25%的排片。<ref>{{cite web |url=http://news.xinhuanet.com/ent/2014-07/28/c_126802847.htm |title=《京城81号》票房过三亿 打破惊悚片五十年沉默 |date=2014-07-28|publisher=新华娱乐 |work= |language=zh-hans}}</ref>

这部不久前打破华语惊悚片五项纪录，成为暑期档最猛黑马的影片，并凭借两天破1亿、五天破2亿的大好票房， 又于7月26日晚正式迈过3亿票房大关， 打破2011年上映的《[[孤岛惊魂_(2011年电影)|孤岛惊魂]]》，将惊悚片的票房纪录再创新高。<ref>{{cite web |url=http://et.21cn.com/movie/xinwen/huayu/a/2014/0728/09/27904094.shtml |title=《京城81号》票房过三亿 打破惊悚片50年沉默 |date=2014-07-28 |publisher=21CN |work= |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20140811072510/http://et.21cn.com/movie/xinwen/huayu/a/2014/0728/09/27904094.shtml |archivedate=2014-08-11 }}</ref>。至2014年8月10日，《京城81号》票房公映24天共收4.06亿<ref >{{cite web|url=http://news.mtime.com/2014/08/12/1530146-all.html | language= | title=白发魔女传》内地连冠《秦时明月》《绣春刀》首周遇冷 |publisher=Mtime时光网  |date=2014年8月12日 |accessdate=2014年8月19日 }}</ref>。

数据显示，《京城81号》截止目前的票房已经超过《[[归来|归来]]》，暂列2014年华语电影票房第九位，总票房是内地影史上前七名惊悚电影的票房总和。<ref>{{cite web |url=http://yule.sohu.com/20140728/n402829358.shtml |title=《京城81号》9天票房过三亿 惊悚片市场引关注 |date=2014-07-28|publisher=搜狐娱乐 |work= |language=zh-hans}}</ref>至2014年8月8日，京城81号的内地票房已正式突破4亿，成为华语影史首部票房过4亿的惊悚片。
本片最终票房達到4.07億元人民幣。

==影响==

电影《京城81号》的热映，使[[朝阳门_(北京)|朝阳门 (北京)]]内大街81号院(简称朝内81号)的“鬼宅”之名再成热点，每天都有“胆大”者慕名探险。<ref>{{cite web |url=http://news.dichan.sina.com.cn/2014/07/28/1170149.html |title==朝内81号“鬼宅”因产权纠纷闲置14年 年底启动修缮 | date=2014-07-28|publisher=北京日报 |work= |language=zh-hans}}</ref> 许多游客是看过电影《京城81号》后专门赶来的狂热“粉丝”，就是为了“图刺激”。<ref>{{cite web |url=http://news.xinhuanet.com/travel/2014-07/24/c_126791575.htm |title==揭秘"著名鬼宅"朝内81号 地下室最受游客欢迎 | date=2014-07-24|publisher=人民网|work= |language=zh-hans}}</ref>电影上映以来，每天有近500位影迷和“探险迷”来此探访。然而，由于院内的两栋民国老楼年久失修，已成危楼，朝内81号院管理方近日颁布禁令，为保安全将严格限流。北京天主教基建办也再次表示，希望人们能理性看待这座宅院。<ref>{{cite web |url=http://finance.ifeng.com/a/20140725/12797212_0.shtml |title=《京城81号》带火“朝内81号”|date=2014-07-25|publisher=新民晚报 |work= |language=zh-hans}}</ref>而作为电影外景地的[[无锡市|无锡市]][[惠山古镇|惠山古镇]]也因此片引来不少游客。<ref>{{cite web|url=http://roll.eastday.com/c1/2014/0725/865849409.html|title=《京城81号》古宅镜头均在无锡拍摄 惠山古镇走红|date=2014-07-25|publisher=现代快报|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20140811164833/http://roll.eastday.com/c1/2014/0725/865849409.html|archivedate=2014-08-11}}</ref>
