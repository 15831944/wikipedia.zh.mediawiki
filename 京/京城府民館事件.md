'''京城府民館事件'''（{{lang-ja|京城府民館事件}}）是1945年（[[昭和|昭和]]20年）7月24日[[朝鮮日治時期|日属朝鮮]][[京畿道_(日本統治時代)|京畿道]][[京城府|京城府]]发生的爆炸事件。是日本統治下的朝鮮最後的独立運動事件。

== 事件开端 ==
[[File:Keijo_Public_Hall.JPG|thumb]]
朝鮮人唯一的[[众议院_(日本)|衆議院議員]][[朴春琴|朴春琴]]，1945年（昭和20年）6月24日与[[李光洙|李光洙]]结成大義党。准备7月24日在[[京城府民館|京城府民館]]召开反对欧美列強支配[[殖民地|殖民地]]的亚细亚民族愤激大会。

== 事件概要 ==
1945年7月24日京城府民館召开亚细亚民族愤激大会。[[满洲国|满洲国]]和[[中华民国|中华民国]][[汪精卫政权|汪精卫政权]]代表出席会议、商讨亚细亚民族解放。

下午6時许有人投入[[炸弹|炸弹]]，造成1人死亡、数十人受伤。会場陷入混乱、会议提前结束。嫌疑人趙文紀被逮捕。20天後8月15日[[二战|二战]]日本投降，[[朝鲜半岛|朝鲜半岛]]独立，趙文紀被释放。

== 后续 ==
趙文紀后来担任[[民族問題研究所|民族問題研究所]]理事長，追查[[亲日派|亲日派]]。2008年2月5日去世。
