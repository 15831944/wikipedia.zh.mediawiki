'''请勿追踪'''（{{lang-en|'''Do Not Track'''}}，简称DNT，又译'''请勿跟踪'''、'''不要追蹤我'''<ref>[https://mozilla.com.tw/firefox/dnt/ Firefox 瀏覽器 － 不要追蹤我 (Do Not Track) － Mozilla Taiwan]</ref>）是一项被提名的HTTP头。
当用户提出启用“请勿追踪”功能后，具有“请勿追踪”功能的浏览器会在http数据传输中添加一个“头信息”（headers），这个头信息向商业网站的服务器表明用户不希望被追踪。这样，遵守该规则的网站就不会追踪用户的个人信息来用于更精准的在线广告。

較知名的请勿追踪軟體有{{link-en|DoNotTrackMe|DoNotTrackMe}}、{{link-en|Disconnect|Disconnect Mobile|Disconnect}}、[[Ghostery|Ghostery]]、[[隐私獾|隐私獾]]等，其中Ghostery原本只是純偵測軟體，不過在後來也加入了阻擋功能。

== 外部連結 ==
*[https://www.abine.com/donottrackme DoNotTrackMe]，由Abine公司開發的请勿追踪軟體，可適用於大部分主流瀏覽器
*[https://disconnect.me/disconnect Disconnect]，由2011年成立的Disconnect公司開發的请勿追踪軟體，可適用於大部分主流瀏覽器，但不支援IE

*[https://www.eff.org/privacybadger Privacy Badger]，由[[電子前哨基金會|電子前哨基金會]]開發的請勿追蹤軟體，適用於Chrome和Firefox瀏覽器，但不支援Safari和IE
*[https://www.requestpolicy.com/ Request Policy]，Justin Samuel的請勿追蹤軟體，適用於Firefox瀏覽器
*[https://github.com/dillbyrne/random-agent-spoofer#readme Random Agent Spoofer]，dillbyrne/dbyrne的請勿追蹤軟體，適用於Firefox瀏覽器
