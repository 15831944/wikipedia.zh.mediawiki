《'''请茶歌'''》是20世纪50年代由[[文莽彦|文莽彦]]作词、[[解策励|解策励]]作曲的一首[[红色歌曲|红色歌曲]]。这首歌曲富有[[江西|江西]]本土特色，风格清新隽永，是中国流传最广的红色革命歌曲之一<ref name ="党史文苑">{{Cite journal|author=姚远|url=|title=红色革命歌曲《请茶歌》|journal=党史文苑|year=2017|volume=|issue=15|pages=43-44}}</ref>。

在[[2019年中央广播电视总台春节联欢晚会|2019年中央广播电视总台春节联欢晚会]]上，[[刘涛|刘涛]]和[[黄晓明|黄晓明]]在[[江西|江西]][[吉安|吉安]][[井冈山|井冈山]]分会场演唱了《请茶歌》<ref name="春晚">{{cite web|url=https://3g.china.com/act/ent/11015422/20190206/35168120.html|last=|first=|title=刘涛黄晓明春晚井冈山同台献唱 《请茶歌》堪称视听盛宴|publisher=中国娱乐网|date=2019-02-06|access-date=2019-02-12}}</ref>。

== 创作背景与影响 ==

《请茶歌》的原词，是文莽彦1957年5月在[[吉安|吉安]]体验生活时创作的系列组诗中的一首，不久便收入他本人的诗集《井冈山诗抄》，排为首篇<ref name ="创作评谭">{{Cite journal|author=张厚德|url=|title=应该恢复《请茶歌》的本来面貌|journal=创作评谭|year=1998|volume=|issue=5|pages=61-62}}</ref>。同年，曲作者解策励随中央交通部文工团来[[江西省歌舞团|江西省歌舞团]]工作，来到江西后的第一件事就是背起背包，沿着[[中国工农红军|红军]]当年走过的路，从[[永新县|永新县]]出发，徒步直奔革命圣地[[井冈山|井冈山]]，沿途寻找当年老红军进行慰问和采访，创作激情油然而生。恰巧此时，在[[井冈山革命博物馆|井冈山革命博物馆]]，解策励见到诗人文莽彦的-{zh-hans:著作; zh-hant:着作;}-《井冈山诗抄》，翻开第一页就是这首《请茶》，其诗情画意与她所切身体验的生活感受不谋而合。随后解策励为这首诗谱曲，采用[[江西民歌|江西民歌]]中的[[兴国山歌|兴国山歌]]和[[宜春评话|宜春评话]]，并将其抒情性、叙说性与民间风格揉为一体，在1958年创作了纯朴而富有民族特色的《请茶歌》<ref name ="井冈山报">{{cite news|author=樵夫|url=|title=《请茶歌》，风靡至今的红歌|publisher=井冈山报|pages=7|date=2017-07-10|accessdate=2019-02-12}}</ref>。

《请茶歌》曾作为[[赣沪列车|赣沪列车]]发车的播放曲<ref name ="创作评谭"/>。

== 歌曲特点 ==

《请茶歌》的音乐部分具有浓郁的江西[[民间音乐|民间音乐]]色彩。曲作者解策励运用了兴国山歌的“哎呀嘞”作歌头，演化成歌曲的第一句“同志哥”，在歌曲开头就引入了站在高山热情呼喊的艺术意境。两句歌词之后，歌曲由兴国山歌风韵变为叙述性较强而又跳动奔放的宜春评话风格。全曲山歌中有[[评话|评话]]，评话中有山歌，二者的融合天衣无缝<ref name ="农业考古">{{Cite journal|author1=孙效祖|author2=李林浩|url=|title=《请茶歌》—茶歌的奇葩|journal=农业考古|year=1991|volume=|issue=2|pages=133-135}}</ref>。

== 歌词篡改 ==

[[文化大革命|文化大革命]]期间，为迎合中国当时的政治形势，[[中华人民共和国文化部|文化部]]要求各地修改一批有影响的歌曲上报[[北京|北京]]。《请茶歌》歌词在这种背景下被修改了，原词作者的名字被抹去，改为“集体作词”。当时文莽彦受迫害下放劳动，没有任何人向他征询修改歌词的意见。被修改后的《请茶歌》基本上以标语口号式的句子，取代原先源于生活的艺术审美形象。1980年，文莽彦本人在江西省第四届文代会上就《请茶歌》被篡改的事提出反对意见。在之后，江西省文联征得有关方面同意，恢复了歌词的本来面貌，并明确署名“文莽彦词”<ref name ="井冈山报"/>。1989年，《请茶歌》荣获[[中央人民广播电台|中央人民广播电台]]“建国四十周年广播金曲奖”，然而这次被使用的仍然是文革期间的改词。文莽彦的遗孀王冬青在写信申诉无果后，拒不领受获奖证书和奖杯<ref name ="创作评谭"/>。

《请茶歌》后来也出现过其他多种篡改、冒牌版本，音乐是解策励作的曲子，但歌词有的被改成“情郎哥”之类<ref name ="创作评谭"/>。
