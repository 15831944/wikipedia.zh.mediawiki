'''干城車站'''，舊稱'''南華車站'''，是一座位於[[臺灣|臺灣]][[花蓮縣|花蓮縣]][[吉安鄉|吉安鄉]]的[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]上的[[鐵路車站|鐵路車站]]，已於1994年廢站。車站房舍目前提供社區協會使用。[[台灣日治時期|日治時期]]設站初名'''初音停車場'''，後改稱'''初英停車場'''。容易和位於臺中市的[[干城站|干城站]]混淆。

== 車站構造 ==
*島式月臺一座。

== 利用狀況 ==
*已廢站，現為花蓮工務段南華道班道班房。

== 車站周邊 ==
*[[木瓜溪|木瓜溪]]
*[[鯉魚潭_(花蓮)|鯉魚潭]]
*花蓮縣政府警察局吉安分局南華派出所

== 歷史 ==
*1916年11月1日：設「初音停車場」。
*1947年10月6日：改稱「初英停車場」。
*1960年7月1日：配合當地地名，改稱「南華車站」。
*1989年3月1日：因行政區屬[[吉安鄉|吉安鄉]]干城村，站名配合村名改為「干城車站」。
*1994年4月1日：廢站。
*2011年1月20日:「花現東風 情定縱谷-小站之旅」 郵輪專列 廢站以來首度列車停靠。
