'''干敏'''（{{bd|1961年|12月||}}），籍贯不详，[[中国人民解放军少将|中国人民解放军少将]]。

==生平==
大学学士，[[中国共产党|中国共产党]]党员。干敏长期在[[中国人民解放军第二炮兵部队|中国人民解放军第二炮兵部队]]服役，曾任[[中国人民解放军第二炮兵装备部|中国人民解放军第二炮兵装备部]]订货部部长等职<ref name=jin/>。在第二炮兵装备部订货部部长任内，2008年5月12日[[汶川大地震|汶川大地震]]时，干敏和二炮装备部部长[[张启华|张启华]]正在[[四川|四川]]出差，随后干敏参与2008年[[汶川大地震|汶川大地震]]救援，担任[[成都军区|成都军区]]抗震救灾联合指挥部二炮前指联络员<ref>{{Cite book|title=遍地英雄：第二炮兵部队抗震救灾实录|last=徐剑著|first=|publisher=北京：中国青年出版社 ,|year=2008年10月|isbn=978-7-5006-8465-7|location=|pages=27-28}}</ref><ref>{{Cite web |url =http://news.sohu.com/20080603/n257259835.shtml  |title =大校干敏的三种角色  |author =  |date =2017-11-02  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。2012年7月8日之前升任中国人民解放军第二炮兵装备部副部长。2016年，北京市人大常委会网站更新了北京市第十四届人大代表信息，其中干敏为北京市第十四届人大代表（驻京部队），任[[金轮工程|金轮工程]]指挥部部长<ref name=jin>{{Cite web |url = http://www.defence.org.cn/article-2-148236.html |title = 原二炮装备部副部长干敏担任金轮工程指挥部部长 |author =  |date = 2016-04-19  |publisher =  全球防务|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://210.75.193.155/rdzw/rddb/Rddb.do?method=showRddbRytz&id=20133573  |title =干敏  |author =  |date =  |publisher = 北京市人大常委会 |language =  |accessdate = 2017-11-02 |quote =  }}</ref>。
