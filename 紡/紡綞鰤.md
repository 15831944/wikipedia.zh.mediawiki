'''紡綞鰤'''，又稱'''雙帶鰺'''，俗名為海草、拉侖，為[[輻鰭魚綱|輻鰭魚綱]][[鱸形目|鱸形目]][[鱸亞目|鱸亞目]][[鰺科|鰺科]]的其中一個[[種|種]]。

== 分布 ==
本魚廣泛分布於全球各大洋的暖水域。

== 深度 ==
水深0至30公尺。

== 特徵 ==
本魚體成紡錘形狀；體背藍黑色，腹面銀白帶淺藍色調，體側面有2條平行的淺藍色縱帶。胸鰭與腹鰭約略等長；尾鰭分叉極深；臀鰭及第二背鰭後方均有1枚帶著2枚軟條的離鰭。第一背鰭有硬棘6枚；第二背鰭有硬棘1枚、軟條24至27枚；臀鰭有硬棘1枚、軟條15至16枚。體長可達110公分。

== 生態 ==
本魚屬於外洋性[[魚類|魚類]]，絕少到沿岸水域，以小魚及無脊椎動物為食。

== 經濟利用 ==
肉硬可食，無腥味，適合各種料理。英文稱作Rain bow runner是因為當上鉤時，會躍出水面，水花映出彩虹般的光芒，故名，具有[[雪卡魚毒|雪卡魚毒]]的紀錄。
