'''谢尔盖·列奥尼多维奇·齐赫文斯基'''（{{lang-ru|Сергей Леонидович Тихвинский}}，{{bd|1918年|9月1日|2018年|2月24日}}）中文名'''齐赫文'''，生于[[俄罗斯|俄罗斯]][[彼得格勒|彼得格勒]]（今[[圣彼得堡|圣彼得堡]]），[[俄罗斯科学院|俄罗斯科学院]]院士、[[汉学家|汉学家]]<ref name=xuezhe>{{Cite web |url = http://sputniknews.cn/society/201803031024832578/ |title =  齐赫文斯基：传奇学者与外交官的不平凡人生|author =  |date =2018-03-03  |publisher =  俄罗斯卫星通讯社|language =  |accessdate =  |quote =  }}</ref><ref>曹苏玲，齐赫文斯基与曹靖华——祝贺齐赫文斯基诞辰九十周年，国际汉学2009年01期</ref><ref>李明滨，齐赫文,中国人民的朋友——恭贺齐赫文院士九十华诞，国际汉学2009年01期</ref>。
==生平==
齐赫文斯基生于圣彼得堡一个医生家庭。1938年，还是[[列宁格勒大学|列宁格勒大学]]语言系五年级大学生的齐赫文斯基进入[[苏联|苏联]]外交部门工作。当时苏联尚无培养职业外交官的高校，外交岗位干部大多是具备高学历的[[苏共|苏共]]党员，且并不全是人文学科人才，例如苏联驻[[加拿大|加拿大]]大使是学工程毕业，驻中国大使[[潘友新|潘友新]]年轻时曾负责边境哨岗，副外长[[安德烈·葛罗米柯|葛罗米柯]]曾是经济师。1937年，[[斯大林|斯大林]][[大清洗|大清洗]]重创外交部门干部。此后才开始将高年级大学生引入外交岗位。其中就包括齐赫文斯基。在外交人民委员会，齐赫文斯基任[[中文|中文]]翻译，借助词典翻译外交照会<ref name=xuezhe/>。

半年后，他被叫到[[外交人民委员|外交人民委员]][[维亚切斯拉夫·米哈伊洛维奇·莫洛托夫|莫洛托夫]]处，那里正需要一名中文翻译。他被镶有防弹玻璃的吉尔牌轿车送到[[克里姆林宫|克里姆林宫]]，进入办公室，有位中国人孙福参与会谈。孙福请斯大林再次提供借款，以购买飞机、大炮等苏联武器抵抗[[侵华日军|侵华日军]]，齐赫文斯基翻译了此次对话。后来，齐赫文斯基在苏联驻[[乌鲁木齐|乌鲁木齐]]领事馆实习，任内他见证了苏联武器运进中国<ref name=xuezhe/>。

1943年12月，他担任苏联驻[[重庆|重庆]]大使馆二等秘书。1944年出任苏联驻[[北平|北平]]总领事<ref name=jiuwu>{{Cite web |url =http://news.163.com/13/0903/17/97S6PUCE00014JB6_2.html  |title = 中俄友人庆祝俄著名汉学家齐赫文斯基95岁寿辰（二） |author =  |date = 2013-09-03 |publisher =网易  |language =  |accessdate =  |quote =  }}</ref>。1945年，[[郭沫若|郭沫若]]到莫斯科参加[[俄罗斯科学院|俄罗斯科学院]]220周年庆典活动，与齐赫文斯基结识。郭沫若在其1946年出版的《苏联纪行》一书写到这位年轻的苏联外交官对中国历史和文化的了解<ref name=xuezhe/>。

1949年10月1日，[[中华人民共和国开国大典|中华人民共和国开国大典]]举行，苏联驻北京总领事齐赫文斯基应邀观礼，根据[[莫斯科|莫斯科]]方面的指示，他低调站在木制看台下排。[[北京饭店|当晚举行政府宴会]]，宴会活动开始前半小时，给他送来了刚刚被任命为[[中央人民政府政务院|中央人民政府政务院]]总理的[[周恩来|周恩来]]的信函，信中称希望苏联承认[[中华人民共和国|中华人民共和国]]。齐赫文斯基迅速将其译成[[俄文|俄文]]，以“紧急”字样密电莫斯科。次日早晨收到回复电报称，苏联承认中华人民共和国成立的历史事实，并且准备与其建立外交关系。苏联由此成为首个承认中华人民共和国的国家<ref name=xuezhe/>。1949年，齐赫文斯基被任命为苏联驻中华人民共和国大使馆临时代办，后任参赞<ref name=jiuwu/>。1949年10月16日，[[毛泽东|毛泽东]]接受苏联大使[[尼古拉·瓦西里耶维奇·罗申|罗申]]递交国书，参赞齐赫文斯基参加，这是中华人民共和国首次接受国书<ref name=xuezhe/><ref>{{Cite web |url =http://dangshi.people.com.cn/GB/16803567.html|title = 新中国首次接受国书始末 |author =孟红|date =2012-01-05|publisher =中国共产党新闻网 |language =  |accessdate =  |quote =  }}</ref>。齐赫文斯基见证过斯大林与毛泽东的会谈，并多次参加中华人民共和国国庆纪念活动<ref name=pp>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_2012382 |title = 俄著名汉学家齐赫文斯基去世，曾亲历斯大林毛泽东会谈 |author =  |date =2018-02-28  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

1950年代到1960年代，他曾在中国、[[英国|英国]]、[[日本|日本]]工作，参加过[[联合国大会|联合国大会]]，后在[[联合国教科文组织|联合国教科文组织]]代表苏联。1980年到1986年，任外交学院院长，同时负责[[苏联科学院|苏联科学院]]历史研究所的工作<ref name=xuezhe/>。

齐赫文斯基是历史学博士、教授。1981年当选苏联科学院（现俄罗斯科学院）院士。他曾师承俄罗斯汉学研究彼得堡流派的奠基人В.М.阿列克谢耶夫，并形成了独具特色的中国学研究流派。他撰写了10多部专著、500多篇论文。由他主编的《中国的改革与革命》系列丛书在2000年获俄罗斯联邦国家奖，[[俄罗斯联邦总统|俄罗斯联邦总统]][[弗拉基米尔·弗拉基米罗维奇·普京|普京]]亲自为他颁奖<ref name=jiuwu/>。自1990年代起，齐赫文斯基将全部精力投入学术工作，2017年秋季完成了十卷本《中国历史：从古代至21世纪初》的出版，同年11月10日在莫斯科举行了第十卷的首发式<ref name=xuezhe/><ref name=pp/>。2009年10月获[[中华人民共和国主席|中国国家主席]][[胡锦涛|胡锦涛]]授予“中俄关系杰出贡献”奖章，2013年被授予“世界中国学贡献奖”<ref name=bu>{{Cite web |url = http://www.fmprc.gov.cn/web/zwbd_673032/gzhd_673042/t1538480.shtml |title =驻俄罗斯大使李辉出席齐赫文斯基院士遗体告别仪式  |author =  |date = 2018-03-01  |publisher =中华人民共和国外交部  |language =  |accessdate =  |quote =  }}</ref>。

2018年2月24日，齐赫文斯基在莫斯科病逝，享年99岁<ref name=jiuwu/>。2月28日，俄罗斯科学院院士、俄中友协名誉主席齐赫文斯基遗体告别仪式在俄罗斯科学院举行，[[俄罗斯联邦外交部|俄罗斯联邦外交部]]副部长{{link-ru|伊戈尔·莫尔古洛夫|Моргулов, Игорь Владимирович|莫尔古洛夫}}、俄罗斯科学院副院长马卡罗夫、俄罗斯联邦外交部一亚局局长库利克、俄罗斯科学院远东所所长卢佳宁、俄中友协第一副主席库利科娃、中国驻俄罗斯大使[[李辉|李辉]]以及俄罗斯科学院、[[莫斯科国际关系学院|莫斯科国际关系学院]]等学术界代表及中国驻俄罗斯大使馆部分外交官共两百余人出席<ref name=bu/>。
