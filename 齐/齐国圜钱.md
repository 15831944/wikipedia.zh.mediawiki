'''齐国圜钱'''，或称'''賹化钱'''，中国古代[[战国时期|战国时期]]位于今[[山东省|山东省]]一带的诸侯国[[齐国|齐国]]所铸造流通的一类[[圜钱|圜钱]]，[[圆形方孔钱|圆形方孔]]，有郭{{efn|郭，钱币学术语，指凸起的轮廓}}，钱文有“賹化”、“賹四化”和“賹六化”三种，钱文释读仍有争议。齐国是较晚使用圜钱的国家之一，大约在战国晚期开始铸造，仍使用此前[[齐国刀币|齐国刀币]]的“化”为货币单位<ref name="gov">{{cite web |title=圜钱 |url=http://www.gov.cn/test/2006-05/09/content_276200.htm |website=中央政府门户网站}}</ref>。

== 铸行 ==
齐国早、中期铸行刀币，晚期兼行圜钱。賹化钱的铸行应开始于[[齐襄王|齐襄王]]复国之后<ref name="gov" />。除在齐国流通外，还流通到[[燕国|燕]]、[[赵国|赵]]等国<ref name="省情网-賹化钱">{{cite web |title=省情资料库>金融库>第一卷 货币>第二类 铜铸币>第二辑 賹化钱 |url=http://lib.sdsqw.cn/bin/mse.exe?seachword=&K=a&A=54&rec=9&run=13 |website=山东省情资料库 |accessdate=2018-09-07}}</ref>。朱活认为齐国圜钱的发行是受到了[[秦国|秦国]]势力东扩的影响<ref name="古钱新探">{{cite book |author1=朱活 |title=古钱新探 |date=1984 |publisher=齐鲁书社}}</ref>{{rp|106-107}}。

对于賹化圜钱的铸行地区在今山东一带，[[钱币学|钱币学界]]和[[考古学|考古学界]]意见基本一致；而其铸行时期则分歧较大。[[丁福保|丁福保]]认为是[[汉武帝|汉武帝]]时期菑川王之子刘胡所铸<ref>{{cite book |author1=丁福保 |title=历代古钱图说 |date=1985 |publisher=山海书店 |page=50}}</ref>，[[王献唐|王献唐]]认为是[[西周|西周]]初期<ref>{{cite book |author1=王献唐 |title=中国古代货币通考 |date=1979 |publisher=齐鲁书社}}</ref>，[[彭信威|彭信威]]认为是[[秦国|秦国]]灭齐之前<ref>{{cite book |author1=彭信威 |title=中国货币史 |date=1965 |publisher=上海人民出版社 |page=81}}</ref>。[[朱活|朱活]]认为是战国晚期，其给出的理由为在西周、春秋的遗址、窖藏中未曾发现賹化圜钱，仅在战国晚期墓葬中发现过<ref>{{cite journal |author1=朱活 |title=论齐圜钱钱范兼谈六字刀 |journal=中国钱币 |date=1988 |issue=1}}</ref>。赵匡华等人对齐国圜钱和刀币的合金配比进行比较后，认为圜钱大约与铸行于战国晚期的“[[齐法化|齐法化]]”刀币同时并行<ref name="赵匡华1993" />。

朱活认为齐国圜钱与齐法化刀币为同时流通的货币，但它们之间并不存在主币与[[辅币|辅币]]的关系，“一枚賹化圜钱的法定币值为一枚齐法化”，而齐法化刀币的重量是賹化圜钱的三十余倍，因而推断賹化圜钱是战国晚期齐国国力严重衰退时铸造发行<ref name="古钱新探" />{{rp|130-132}}。赵匡华等则对朱活的这一观点进行了质疑，认为賹化圜钱是刀币的辅币<ref name="赵匡华1993" />。按照重量测定和钱文分析，賹六化、賹四化和賹化三种圜钱的币值比率为6:4:1，赵匡华等认为齐法化刀币应合賹化36枚<ref name="赵匡华1993" />。

== 钱文释读 ==
齐国圜钱钱文的最后一个字“[[File:刀-bronze_from_knife_money.svg|20px]]”自清朝末年以来多释为“化”，解释为货币，还有“刀”等释读<ref name=zc>{{cite book|author=张驰 |title=钱币文化丛书 中国刀币汇考 |publisher=河北人民出版社 |location=石家庄 |isbn=7202022052 |date=1905-06-19}}</ref>{{rp|9}}。

第一个字“[[File:賹-bronze._coin.svg|20px]]”早期曾被误释为“宝”、“燕”、“贝”、“朋贝”等，后多释读为“賹”（“[[File:賹-bronze._coin.svg|20px]]”上部“𠍳”即古“嗌”字，用作声符）。[[丁福保|丁福保]]《古钱大辞典》中认为“賹即益，为汉武帝元朔二年所封[[菑川國|菑川]]懿王子刘胡之侯国，地望在山东[[寿光县|寿光县]]西……盖今之益都，在汉为益国也”，后来考古发掘证明賹化圜钱为战国晚期铸币，这一观点被推翻。朱活认为賹是当时贵金属的重量单位，也就是原本秦国所用的“[[镒|镒]]”<ref name="朱活1965" />。

== 材质 ==
齐国圜钱的材质全部为[[青铜|青铜]]合金，主要含有[[铜|铜]]、[[铅|铅]]和[[锡|锡]]三种金属。赵匡华等人曾取4枚賹四化和14枚賹六化进行理化分析，并通过与齐国刀币的合金配比参数比较，齐国圜钱与战国晚期的齐法化刀币的配比几乎一致<ref name="赵匡华1993">{{cite journal |author1=赵匡华 |author2=陈荣 |author3=孙成甫 |title=“賹四化”与“賹六化”的理化研究 |journal=中国钱币 |date=1993 |issue=2 |pages=22-28}}</ref>。
