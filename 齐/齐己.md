''' 齐己 '''（863年—937年）俗名'''胡得生 '''，晚年自号'''衡岳沙门'''，[[湖南|湖南]][[长沙|长沙]][[宁乡县|宁乡县]][[沩山乡|塔祖乡]]人，{{noteTag|1=[[宋朝|宋朝]]《[[高僧传|高僧传]]》卷三十载：“齐己乃[[益阳|益阳]]人也。”。[[宋朝|宋朝]][[陶岳|陶岳]]著《[[五代史补|五代史补]]》卷三《僧齐己》载：“僧齐己，[[长沙|长沙]]人。长沙有大沩同庆寺，僧多而地广，佃户仅千余家，齐己则佃户胡氏之子也。”。《宁乡县志》载：“在同庆寺南禅衣峰西北有齐己故居，后毁，乡人志之。”。[[刘国强|刘国强]]著《[[湖南佛教寺院志|湖南佛教寺院志]]》载：“齐己，晚唐[[潭州|潭州]][[益阳|益阳]]人，即今[[宁乡县|宁乡县]][[祖塔乡|祖塔乡]]人……”。[[胡则丘|胡则丘]]著《[[历代诗联咏桃江|历代诗联咏桃江]]》一书中，所记齐己则为[[桃江|桃江]][[石牛江|石牛江]]人，而且说有族谱为证。《五代史补》卷三《僧齐己》载的“长沙有大沩同庆寺”。《湖南佛教寺院志》载：“同庆寺，位于宁乡县沩山，距离密印寺7公里。唐宣宗大中七年（853）九月建，为沩山灵佑塔院，又称祖塔院……其地唐代属潭州益阳县境。”又载：同庆寺为唐代大诗僧齐己少年出家之地。所以说：齐己是晚唐潭州益阳县人，今长沙宁乡县塔祖乡人。}}[[唐朝|唐朝]]晚期著名[[诗僧|诗僧]]。<ref name=" 齐己籍贯考">{{cite web|url=http://www.taojiang.gov.cn/News/NewsDetail_16537.html|title=《齐己籍贯考》|accessdate=2011-11-16|work=龙玉牛|publisher=中国桃江公众信息网|language=zh-hans}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 简介 ==
863年，齐己出生于[[湖南|湖南]][[长沙|长沙]][[宁乡|宁乡]][[沩山乡|大沩山]][[同度寺|同度寺]]的一个佃户家庭，家境贫寒。6岁多就和其他佃户家庭的孩子一起为寺庙放牛，一边放牛时一边学习、作诗，常常用竹枝在牛背上写诗，而且诗句语出天然，同度寺的和尚们为寺庙声誉，便劝说齐己出家为僧，拜[[荆南|荆南]]宗教领袖[[仰山慧寂|仰山慧寂]]为师傅。<ref name=" 齐己籍贯考"/><ref name=" 牧童出身的诗僧齐己 ">{{cite web |url=http://www.ndcnc.gov.cn/datalib/2003/Character/DL/DL-20031211134850/|title=《牧童出身的诗僧齐己 》|accessdate=|work=|publisher=国家数字文化网|language=zh-hans}}</ref>

齐己出家后，更加热爱写诗。成年后，齐己出外游学，云游期间曾自稱「[[衡岳|衡岳]]沙弥」。<ref name=" 齐己籍贯考"/> 登[[岳阳|岳阳]]，望[[洞庭|洞庭]]，又过[[长安|长安]]，遍览[[终南山|终南山]]、[[华山|华山]]等风景名胜，还到过[[江西|江西]]等地。这段游学生活丰富了他的写作素材。且他的不少名作佳作是在外游历时写的。<ref name=" 牧童出身的诗僧齐己 "/>

齐己云游天下的时候，曾拿他的诗作《早梅》請教诗人[[郑谷|郑谷]]。诗句是：「万木冻欲折，孤根暖独回。前村深雪里，昨夜数枝开。风递幽香出，禽窥素艳来。明年犹应律，先发映春台」。郑谷阅读后，笑着说：「数枝」非早，不如「一枝」更佳。齐己听后，对郑谷肃然起敬，顶地膜拜。此后，人们便称郑谷为齐己的“一字之师”。<ref name=" 牧童出身的诗僧齐己 "/>

齐己游历天下回到长沙时，他的名声已经显赫天下，湖南[[节帅|节帅]]幕府中的诗人[[徐东野|徐东野]]曾评价他说：“我辈所作，皆拘于一途，非所谓通方之士。若齐己，才高思远，无所不通，殆难及矣”。<ref name=" 牧童出身的诗僧齐己 "/>

921年，齐己在去[[四川|四川]]途中路过[[荆州|荆州]]，被[[节帅|节帅]][[高季兴|高季兴]]挽留，安置在[[龙兴寺|龙兴寺]]，并任命为[[僧正|僧正]]。齐己在荆州，虽然月俸丰厚，但是他并不喜好钱财，于是写作了《渚宫莫问篇》十五章，以表明他的高洁志向。齐己在荆州期间写了许多诗，76岁的齐己圆寂于[[江陵|江陵]]。<ref name=" 齐己籍贯考"/> 死后以《白莲集》传于世。 <ref name=" 牧童出身的诗僧齐己 "/>

== 风格 ==
齐己为人秉性高洁，其貌不扬，不喜欢攀龙附凤，拜谒[[王侯|王侯]][[卿相|卿相]]。他的颈部长了一个[[瘤赘|瘤赘]]，时人戏称之为“诗囊”。他的性格豪放，平日破衲拥身，枲麻缠膝，怡然自得。齐己的诗作题材以山水风景为多，但是并不特意宣扬[[佛教|佛教]]遁世归隐、轮回报应的教理，与[[贯休|贯休]]非常相似。齐己的诗作特色為讽刺纨绔子弟骄奢淫逸的生活，对劳动人民深表同情。齐己的诗作数量丰富，讲究以禅入诗，传世诗歌有852首之多，《[[全唐诗|全唐诗]]》收录他的诗歌数量名列[[李白|李白]]、[[杜甫|杜甫]]、[[白居易|白居易]]、[[元稹|元稹]]之后，名列第五位。<ref>{{cite web|url=http://hunan.voc.com.cn/gb/content/2003-04/18/content_1939353.htm|title=《中华诗词宁乡花满枝》|accessdate=2003-04-18|work=|publisher=湖南日报|language=zh-hans}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 齐己对诗歌理论有深入研究，著有《风骚旨格》1卷，阐述诗的格式、类别、题材与写作方法，对唐朝以后历朝诗歌有影响。齐己的书法也非常优秀，尤其擅长[[行书|行书]]，笔迹洒落，并且与他的诗作一起传布四方。<ref name=" 牧童出身的诗僧齐己 "/>

== 评价 ==
唐朝[[徐东野|徐东野]]评价齐己说：“我辈所作，皆拘于一途，非所谓通方之士。若齐己，才高思远，无所不通，殆难及矣”。<ref name=" 牧童出身的诗僧齐己 "/>

[[唐朝|唐朝]][[尚颜|尚颜]]：诗为儒者禅，此格的惟仙。古雅如周颂，清和甚舜弦。{{noteTag|name= qiji1|1=[[唐朝|唐朝]] [[尚颜|尚颜]]《读齐己上人集》}}

[[元朝|元朝]][[辛文房|辛文房]]：性放逸，不滞土木形骸，颇任琴樽之好。尝撰《玄机分别要览》一卷，摭古人诗联，以类分次，仍别风、赋、比、兴、雅、颂。又撰《诗格》一卷。又与郑谷、黄损等共定用韵，为葫芦、辘轳、进退等格，并其《白莲集》十卷，今传。{{noteTag|name=qiji2 |1=[[元朝|元朝]] [[辛文房|辛文房]]《[[唐才子传|唐才子传]]》卷九 }}

[[明朝|明朝]][[胡震亨|胡震亨]]：齐己诗清润平淡，亦复高远冷峭，一经都官点化，《白莲》一集，驾出《云台》之上，可谓智过其师。{{noteTag|name= qiji3|1= [[明朝|明朝]] [[胡震亨|胡震亨]]《[[唐音癸签|唐音癸签]]》卷八}}

[[清朝|清朝]][[吴乔|吴乔]]：齐己《剑客》诗，杰作也。“夜来何处火，烧出古人冢”，非晚唐人无此诗思。{{noteTag|name=qiji4 |1=[[清朝|清朝]] [[吴乔|吴乔]]《[[围炉诗话|围炉诗话]]》卷二 }}

[[清朝|清朝]][[薛雪|薛雪]]：唐释齐己作《风骚旨格》，六诗、六义、十体、十势、二十式、四十门、六断、三格，皆系以诗，不减司空表圣。独是“十势”立名最恶，宛然少年棍谱，暇日当为易去乃妙。{{noteTag|name= qiji5|1= [[清朝|清朝]] [[薛雪|薛雪]]《[[一瓢诗话|一瓢诗话]]》}}

[[清朝|清朝]][[纪昀|纪昀]]：唐诗僧以齐己为第一。又：唐代缁流能诗者众。其有集传於今者，惟[[皎然|皎然]]、[[贯休|贯休]]及齐己。皎然清而弱。贯休豪而粗。齐己[[七言律诗|七言律诗]]不出当时之习。及[[七言古诗|七言古诗]]以[[卢仝|卢仝]]、[[马异|马异]]之体缩为短章，诘屈聱牙，尤不足取。惟[[五言律诗|五言律诗]]居全集十分之六。虽颇沿武功一派，而风格独遒。如《剑客》、《听琴》、《祝融峰》诸篇，犹有大历以还遗意。其[[绝句|绝句]]中《庚午年十五夜对月》诗曰：“海澄空碧正团圞，吟想元宗此夜寒。玉兔有情应记得，西边不见旧长安。”惓惓故君，尤非他释子所及。宜其与[[司空图|司空图]]相契矣。{{noteTag|name=qiji6 |1=[[清朝|清朝]] [[纪昀|纪昀]]《[[四库总目提要|四库总目提要]]》卷一五一}}
