'''齐宗华'''（{{bd|1930年|||}}）<ref>{{Cite web |url =  http://www.tac-online.org.cn/index.php?m=content&c=index&a=show&catid=490&id=1891|title =  中国翻译协会表彰的资深翻译家|author =  |date =2015-12-29  |publisher = 中国翻译协会 |language =  |accessdate =  |quote = 齐宗华，女，1930年出生，法语，北京国际学院教授。 }}</ref>女，祖籍[[山东|山东]]，生于[[英国|英国]][[伯明翰|伯明翰]]，[[中华人民共和国外交部|中华人民共和国外交部]]官员<ref name=shouxi>宗道一，齐宗华:中国外交界的首席法语译员，国际人才交流1994(7):12-17</ref><ref name=zhuming>宗道一，齐宗华——中国外交界著名翻译家，中国翻译1995(01):42-44</ref>。

==生平==
齐宗华祖籍山东，先祖“[[闯关东|闯关东]]”迁居[[吉林|吉林]][[伊通|伊通]]，后来全家又回到[[山海关|山海关]]内的[[天津|天津]]定居。齐宗华的祖父[[齐耀珹|齐耀珹]]曾在[[南开大学|南开大学]]攻读经济学，[[中华民国|中华民国]]初年曾任[[天津县|天津县]]知事，后任农商银行行长。齐宗华的父母齐祖辑、刘月如夫妇曾旅居英国伯明翰，并生下独生女齐宗华，后来他们携4岁的齐宗华迁居[[法国|法国]][[巴黎|巴黎]]，他们夫妇进入[[巴黎大学|巴黎大学]]学习。齐宗华在巴黎长大，在十年制[[天主教|天主教]]学校布风（Buffon）中学学习<ref name=shouxi/><ref name=zhuming/>。

[[抗日战争|抗日战争]]中期，齐祖辑夫妇携10岁的齐宗华返回中国，当时齐宗华已是熟练掌握[[法语|法语]]、[[英语|英语]]的布风中学八年级学生。回国后，齐宗华曾在[[天津|天津]][[天津原圣约瑟女校旧址|圣约瑟女子中学]]短暂念初中，后来到[[云南|云南]][[昆明|昆明]]的[[西南联合大学|西南联合大学]]附属中学学习。抗日战争胜利后不久，齐宗华随父母离开了生活三年的昆明到[[北平|北平]]。齐宗华在北平[[慕贞女中|慕贞女中]]毕业，考入[[中法大学|中法大学]]，学习法语读完大一后，齐宗华又考入[[燕京大学|燕京大学]]西方语言文学系学习英语。1952年，齐宗华自燕京大学毕业，成为该校最后一届毕业生<ref name=shouxi/>。

1952年毕业后，齐宗华被分配到[[中国人民保卫世界和平委员会|中国人民保卫世界和平委员会]]担任法语、英语译员，任职四年多。1952年秋，齐宗华第一次充当翻译，为在[[北京|北京]]举行的[[亚洲及太平洋区域和平会议|亚洲及太平洋区域和平会议]]的英、法、中[[同声传译|同声传译]]。此后，齐宗华多次作为中国代表团译员，出席各类政府、政党、民间国际会议，例如1950年代初期每两年一届的[[世界和平理事会|世界和平理事会]]会议，1957年11月[[社会主义国家共产党和工人党代表会议|社会主义国家共产党和工人党代表会议]]，1960年6月[[布加勒斯特会议|布加勒斯特会议]]，1960年[[莫斯科|莫斯科]][[各国共产党和工人党代表会议|各国共产党和工人党代表会议]]，1962年关于[[老挝|老挝]]问题的[[日内瓦会议_(1961年)|日内瓦会议]]，亚非经济会议，[[塔什干|塔什干]]亚非作家会议，以及第28届、第29届[[联合国大会|联合国大会]]会议等。她还是1963年底周恩来出访亚、非、欧14国的译员之一。齐宗华为[[毛泽东|毛泽东]]、[[刘少奇|刘少奇]]、[[周恩来|周恩来]]、[[朱德|朱德]]、[[邓小平|邓小平]]、[[彭真|彭真]]、[[陈毅|陈毅]]等中共领导人，以及[[宋庆龄|宋庆龄]]、[[郭沫若|郭沫若]]、[[廖承志|廖承志]]、[[刘宁一|刘宁一]]、[[程潜|程潜]]、[[罗隆基|罗隆基]]、[[蔡廷锴|蔡廷锴]]、[[侯德榜|侯德榜]]等中国其他领导人及[[民主党派|民主党派]]负责人担任过法语翻译工作，并由此结识了[[约里奥·居里|约里奥·居里]]（[[居里夫人|居里夫人]]的女婿）、法国作家[[罗曼·罗兰|罗曼·罗兰]]、住在法国[[马赛|马赛]]的[[西班牙|西班牙]]画家[[毕加索|毕加索]]等人<ref name=shouxi/><ref name=zhuming/>。

齐宗华首次在国外公开亮相是在[[匈牙利|匈牙利]][[布达佩斯|布达佩斯]]，为出席世界和平理事会会议的中国代表郭沫若任译员。一年后，齐宗华先后在[[捷克斯洛伐克|捷克斯洛伐克]][[布拉格|布拉格]]、[[奥地利|奥地利]][[维也纳|维也纳]]担任常驻世界和平理事会的前后三任中国书记[[李一氓|李一氓]]、[[郑森禹|郑森禹]]、[[朱子奇|朱子奇]]的译员<ref name=shouxi/><ref name=zhuming/>。

1956年，齐宗华自布拉格回国，调入[[中共中央对外联络部|中共中央对外联络部]]继续任译员。后来升任中共中央对外联络部[[欧洲|欧洲]]处副处长，并继续担任译员工作。1957年在莫斯科共产党和工人党代表会议期间，毛泽东同[[法国共产党|法国共产党]]总书记[[多列士|多列士]]会谈，齐宗华首次为毛泽东当译员。1956年9月[[中共八大|中共八大]]期间，[[中共中央副主席|中共中央副主席]]周恩来会见率团参加中共八大的[[阿尔巴尼亚劳动党|阿尔巴尼亚劳动党]]第一书记[[恩维尔·霍查|恩维尔·霍查]]，由中共中央对外联络部阿尔巴尼亚接待组成员齐宗华当译员，这是她首次为周恩来当译员。1969年5月初，[[法国总统|法国总统]][[戴高乐|戴高乐]]授意法国驻华大使[[艾蒂安·马纳克|艾蒂安·马纳克]]（是布风中学的校友）向中国领导人传递新任[[美国总统|美国总统]][[理查德·尼克松|尼克松]]对华新政策的口信，马纳克受到[[中华人民共和国主席|中华人民共和国代主席]][[董必武|董必武]]、[[中华人民共和国国务院总理|国务院总理]][[周恩来|周恩来]]等领导人接见，齐宗华担任法语译员<ref name=shouxi/><ref name=zhuming/>。

1973年，经周恩来直接安排，齐宗华调入[[中华人民共和国外交部|中华人民共和国外交部]]，历任外交部西欧司副处长、处长，以及主管法国事务的副司长。调入外交部后，齐宗华几乎参加了毛泽东、周恩来同法语国家来宾的每次会见<ref name=shouxi/><ref name=zhuming/>。当时[[王海容|王海容]]、[[唐闻生|唐闻生]]、齐宗华、[[罗旭_(外交部)|罗旭]]、[[章含之|章含之]]被称为中国外交界的“五朵金花”<ref name=zgqnw>{{Cite web|url = http://agzy.youth.cn/qsnag/sdxw0/201208/t20120822_2376011.htm|title = 中国外交界的“五朵金花”|author = |date = 2012-08-22|publisher = 中国青年网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170909233145/http://agzy.youth.cn/qsnag/sdxw0/201208/t20120822_2376011.htm|archive-date = 2017-09-09|dead-url = yes}}</ref>。从1950年代到1970年代，齐宗华的足迹几乎遍及整个[[欧洲|欧洲]]和[[非洲|非洲]]，除了[[西班牙|西班牙]]、[[葡萄牙|葡萄牙]]以及[[南部非洲|南部非洲]]<ref name=shouxi/><ref name=zhuming/>。

1980年，齐宗华离开外交部，进入北京[[国际关系学院|国际关系学院]]教法语。齐宗华教授在本职工作外，还在1983年之后在[[天津|天津]]、[[湖南|湖南]]、[[陕西|陕西]]等地的出版社出版了许多译作<ref name=shouxi/><ref name=zhuming/>。

==著作==
齐宗华的译作主要有：
*[[亨利·特鲁瓦|亨利·特鲁瓦]]《[[彼得大帝|彼得大帝]]》
*[[纪德|纪德]]《背德者》
*[[吕多维克·阿莱维|吕多维克·阿莱维]]《隆格瓦古堡》
*蒙西·克拉克·维尔逊《华盛顿夫人》<ref name=shouxi/><ref name=zhuming/>
