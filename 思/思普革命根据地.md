'''思普革命根据地'''是[[第二次国共战争|第二次国共战争]]时期，[[中国共产党|中国共产党]]云南[[地下党|地下党]]组织、[[中国人民解放军滇桂黔边纵队|中国人民解放军滇桂黔边纵队]]在云南省西南部建立的一块[[革命根据地|革命根据地]]<ref>{{cite web |url=http://dangshi.people.com.cn/GB/151935/164962/9786346.html |title=云南老区：红土高原在呼唤 |last= |first= |date=2009-08-04 |website=dangshi.people.com.cn |publisher=中国老区建设促进会 |access-date=2017-12-17 |quote=}}</ref>，[[中华人民共和国|中华人民共和国]]成立后改置为[[普洱专区|普洱专区]]，范围包括今[[普洱市|普洱市]]、[[临沧市|临沧市]]和[[西双版纳傣族自治州|西双版纳傣族自治州]]部分地区。

==历史==
1946年6月，中共思普特支成立。1948年12月，自卫军江越支队和[[墨江哈尼族自治县|墨江]]自卫军成立。1949年2月，自卫军普光部队在[[六顺镇|六顺]]组建，迤南边区人民自卫军第一支队在[[澜沧拉祜族自治县|澜沧]]组建。4月，澜沧行政专员公署建立，在[[澜沧拉祜族自治县|澜沧]]、[[上允县|上允]]、[[东朗县|东朗]]、[[宁江县|溯涛]]、[[孟连傣族拉祜族佤族自治县|孟连]]5个县设立了县人民政府。同时，迤南边区人民自卫军第一支队攻下[[双江拉祜族佤族布朗族傣族自治县|双江]]、[[临翔区|临沧]]，在临沧成立了[[缅宁专区|缅宁行政专员公署]]及[[双江拉祜族佤族布朗族傣族自治县|双江县]]、[[缅宁县|缅宁县]]、[[沧源佤族自治县|沧源县]]3个县级政权。8月，思普临时人民行政委员会在[[宁洱哈尼族彝族自治县|宁洱]]成立，下辖[[宁洱哈尼族彝族自治县|宁洱]]、[[墨江哈尼族自治县|墨江]]、[[思茅区|思茅]]、[[六顺县|六顺]]、[[景谷傣族彝族自治县|景谷]]、[[镇沅彝族哈尼族拉祜族自治县|镇沅]]、[[勐遮县|南峤]]、[[勐海县|佛海]]、[[景洪市|车里]]、[[江城哈尼族彝族自治县|江城]]、[[勐腊县|勐腊]]、[[景东彝族自治县|景东]]、[[澜沧拉祜族自治县|澜沧]]、[[宁江县|宁江]]、[[沧源佤族自治县|沧源]]等县级人民政权。9月，思普地区人民武装被整编为中国人民解放军滇桂黔边纵队第九支队。<ref>{{cite web |url=http://www.chinanews.com/cul/2011/08-05/3237187.shtml |title=云南的10块革命根据地 |author=应细飞 |date=2011-08-05 |website=www.chinanews.com |publisher=云南日报 |access-date=2017-12-17 |quote=}}</ref>

1950年3月，[[中共云南省委|中共云南省委]]批准撤销中共思普地委，成立中共宁洱地委，辖[[宁洱哈尼族彝族自治县|宁洱]]、[[思茅区|思茅]]、[[六顺县|六顺]]、[[墨江哈尼族自治县|墨江]]、[[景东彝族自治县|景东]]、[[景谷傣族彝族自治县|景谷]]、[[镇沅彝族哈尼族拉祜族自治县|镇沅]]、[[江城哈尼族彝族自治县|江城]]、[[澜沧拉祜族自治县|澜沧]]、[[宁江县|宁江]]、[[景洪市|车里]]、[[勐海县|佛海]]、[[勐遮县|南峤]]、[[易武县|镇越]]、[[沧源佤族自治县|沧源]]、[[双江拉祜族佤族布朗族傣族自治县|双江]]、[[临翔区|缅宁]]17个县委。5月，成立云南军区宁洱边防区和宁洱专区。1951年改名[[普洱专区|普洱专区]]。<ref>{{cite web |url=http://www.peds.gov.cn/zx_nr.asp?id=4814 |title=思普革命史梗概 |last= |first= |date=2017-01-03 |website=www.peds.gov.cn |publisher=普洱党史 |access-date=2017-12-17 |quote=}}</ref>
