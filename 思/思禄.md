== 早年 ==

== 统治 ==
1476年，思陆法成为木掸的国王，统一了之前的分裂的孟拱、孟养。1478年，思陆法开始反抗缅甸，占据了格达城，切断了缅甸与中国之间的要到，并多次派兵袭击缅甸北部的重镇美都。

== 去世 ==
1495年，思陆法去世，其子[[思凯法|思凯法]]继位。1498年，思凯法去世，其子[[色隆法|色隆法]]继位。木掸王国的国力在思陆法时期不断上升，统治了整个[[伊洛瓦底江|伊洛瓦底江]]以西，那加山脉以中，北至坎迪，西南至钦山，南至格达、太公的广大地区。
