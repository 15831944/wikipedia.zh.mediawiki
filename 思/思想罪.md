'''思想犯罪'''、'''思想罪'''或'''思罪'''（{{lang-en|'''Thoughtcrime'''}}），是英國作家[[乔治·奥威尔|乔治·奥威尔]]作品《[[一九八四|一九八四]]》中的[[新語_(一九八四)|新語]]名詞，是大洋國政府的一個罪名，若有犯罪思想則犯下思想罪，而不必在言論或行動上定罪。跟此詞相反的是好思想，即被黨稱讚的思想；基於新話簡單二分法的原則，好思想和犯罪思想中間沒有灰色地帶。

{{cquote|思想罪不會導致死亡，思想罪本身就是死亡。}}

故事主角史密斯亦稱思想罪是唯一可怕的罪惡。

== 思想警察 ==
{{main|思想警察}}
思想警察是小說中無所不在的秘密警察，專門監察及懲罰犯罪思想。

== 監控技術 ==
無所不在的電幕可以雙向接收和傳播聲畫，加上閉路電視，強大的[[電腦視覺|電腦視覺]]使思想警察可以注意每一個證明思想犯罪的舉手投足、表情和眼神。[[測謊機|測謊機]]、{{tsl|en|Penile plethysmograph|陰莖體積變化測試儀}}更可以偵測同性戀或戀童癖，以判定任何性犯罪。尤有甚者，更使用[[核磁共振成像|核磁共振成像]]來偵任何人腦內犯罪思想的大腦活動。

== 犯罪停止 ==
{{not|犯罪中止}}
犯罪停止是英國作家[[喬治·歐威爾|喬治·歐威爾]]作品《[[一九八四|一九八四]]》中新話的名詞，指在犯罪思想開始之前直覺且自發的停止進一步的思考。

這個詞類似社會學中的[[内化|内化]]（Internalization）。
