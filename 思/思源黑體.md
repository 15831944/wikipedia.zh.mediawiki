'''思源黑體'''（{{lang-en|Source Han Sans}}，「思源」一詞來自於成語「飲水思源」<ref>{{cite web|url=https://typekit.com/fonts/source-han-serif-traditional-chinese|title =Source Han Serif Traditional Chinese|website=Adobe Typekit|publisher=Adobe|access-date=2017-04-05|quote=範例文字：這句話後來演變成「飲水思源」這個成語，意為喝水的時候想一想流水的源頭，比喻不忘本。}}</ref><ref name="ryoko-adobe">{{cite web|author1=西塚涼子‏|title=Tweet #849270655040143360|url=https://twitter.com/ryon106/status/849270655040143360|website=Twitter|language=ja}}</ref>）是[[Adobe|Adobe]]與[[Google|Google]]所領導開發的[[開源|開源]][[字型|字型]]家族，1.001及更早版本以[[Apache_2.0|Apache 2.0]]许可证授權，而1.002及更新版本則使用[[SIL开源字体授权|SIL开源字体授权]]，屬於[[無襯線|無襯線]][[黑体_(字体)|黑體]]。思源黑體於2014年7月16日首次發布，支援[[繁體中文|繁體中文]]、[[簡體中文|簡體中文]]、[[日文|日文]]及[[韓文|韓文]]，並且各有7種字體粗細。公開之時為當時涵蓋[[字元|字元]]數量最多的字型，44,666个字元分屬於65,535個字形中，此為[[OpenType|OpenType]]字型技術的極限。<ref name="appledaily-20140716-434481"/><ref name="ltn-20140716">{{cite web|url = http://3c.ltn.com.tw/3c/news.php?no=13324&type=5|title = 免費字體 +1: Adobe與Google推出思源黑體|date = 2014-07-16|publisher = [[自由時報|自由時報]]|deadurl = yes|archiveurl = https://web.archive.org/web/20140727043625/http://3c.ltn.com.tw/3c/news.php?no=13324&type=5|archivedate = 2014-07-27}}</ref><ref name="techbang-20140716">{{cite web|url = http://www.techbang.com/posts/19067-google-and-adobe-announced-the-launch-of-a-new-digital-fonts-siyuan-bold |title = Adobe與Google聯手推出新數位字體：思源黑體，中日韓多國語系、七種字體 免費下載  | date= 2014-07-16 | publisher = [[T客邦|T客邦]] |author = janus }}</ref>

思源黑體的開發以[[Adobe|Adobe]]的[[西塚涼子|西塚涼子]]為中心，西文及數字採自[[Source_Sans_Pro|Source Sans Pro]]字型，[[漢字|漢字]]部分則分別以各語種為基礎，由不同設計公司製作——[[日文|日文]]為[[Iwata|Iwata]]、中文為[[常州华文|常州华文]]、韓文為[[Sandoll|Sandoll]]等，其字形皆依循各國國家標準。共包含7種[[字重|字重]]（ExtraLight 250、Light 300、Normal 350、Regular 400、Medium 500、Bold 700、Heavy 900）。<ref name="appledaily-20140716-434481">{{cite news|url =  http://www.appledaily.com.tw/realtimenews/article/new/20140716/434481/ |title = Google、Adobe推「思源黑體」新字體 | date= 2014-07-16 | publisher = [[蘋果日報|蘋果日報]] | author = 朱正庭 | location = 臺北}}</ref><ref name="techbang-20140716"/>

字型的名称，除了英文名{{lang|en|Source Han Sans}}，日文名是「{{lang|ja|源ノ角ゴシック}}」、简体中文为「{{lang|zh-cn|思源黑体}}」、繁体中文为「{{lang|zh-tw|思源黑體}}」、韩文为「{{lang|ko|본고딕}}」。字型文档的格式上，Adobe发布了三种版本供下载：多语言[[OpenType|OTF]]版、各个地区子集版、多语言[[OTC|OTC]]版。而Adobe的{{link-en|TypeKit|TypeKit}}服务也可以使用此字型，但出于同步功能，TypeKit版不能和下载版共存。2015年Adobe在思源黑体的基础上发布Source Han Code JP（{{lang|ja|源ノ角ゴシック Code JP}}）字体，其中西文、数字字符宽度均为2/3em（667单位），并且只有日文版。<ref>[https://blogs.adobe.com/CCJKType/2015/06/source-han-code-jp.html Introducing Source Han Code JP]</ref>

Google公司将此字型另起名为[[Noto_Sans_CJK|Noto Sans CJK]]发布，屬於Noto字體家族，但字形本身（包括西文、數字）和Adobe的版本完全一样，仅是字重的称呼不一样而已。<ref name="appledaily-20140716-434481"/><ref name="ltn-20140716"/><ref name="techbang-20140716"/>{{efn| name = noto |1= 電腦無法顯示的字型，會出現空白的[[虚缺号|方格]]，這些方格又稱為「豆腐」。而Noto字型的目的就是要消滅這些豆腐（意即盡可能地補上缺少的字元），所以「Noto」一詞其實源自於「no more tofu」（不再有豆腐）。<ref name="ithome-20140730"/>}}

2018年11月19日，2.000版本發佈。此版本加入香港漢字寫法，將臺灣及香港的繁體中文字體分開，並在辵部「辶」的寫法稍做微調。此外也加入了約1,750個新字形（部分為用戶請求），包括將在[[中日韓統一表意文字擴充區G|中日韓統一表意文字擴充區G]]加入Unicode的[[Biángbiáng面|Biáng]]字。<ref>{{Cite web|url=https://raw.githubusercontent.com/adobe-fonts/source-han-sans/release/SourceHanSansReadMe.pdf|title=Source Han Sans Version 2.000 ReadMe|accessdate=2018-11-19|author=|date=|publisher=}}</ref>

== 爭議 ==
{{unreferenced section}}
思源黑體在2.000版前僅支援[[繁體中文|繁體中文]]、[[簡體中文|簡體中文]]、[[日文|日文]]及[[韓文|韓文]]的漢字寫法，而其中[[繁體中文|繁體中文]]初始版本，命名為「TWHK」即「臺灣香港」，可是這版本的漢字寫法只據[[國字標準字體|國字標準字體]]為準，不支援[[常用字字形表|香港教育標準]]，也不支援[[舊字形|傳統印刷體字形]]。後兩者的使用者向計劃方留言表達不滿，爭取支援，卻引起了國字標準字體擁護者的罵戰。2015年6月更新版本，宣佈會進行地區分流。2018年11月的新版本增加香港繁體，但仍未能滿足傳承字形用戶的需要。
