在[[数学|数学]]领域，[[集合|集合]]的'''元素'''（{{lang-en|element}}）指构成该集合的任意{{Tsl|en|Mathematical object|数学对象|对象}}，也可以称作'''成员'''（{{lang-en|member}}）。

== 集合 ==

<math>A = \{1, 2, 3, 4\}</math>表示集合<math>A</math>中有四个元素，分别是数字1、2、3、4。由集合<math>A</math>中元素组成的集合是<math>A</math>的[[子集|子集]]，例如 <math>\{1, 2\}</math>。

集合本身也可以是元素。例如 <math>B = \{1, 2, \{3, 4\}\}</math>，那么集合<math>B</math>中的元素不是 1、2、3、4 四个数，而是1、2、集合 <math>\{3, 4\}</math>，共三个元素。

== 符号和术语 ==

符号「∈」表示「是<math>X</math>中的元素」的[[二元关系|关系]]，这种关系也称'''集合隶属关系'''（{{lang-en|set membership}}）。可以用
:<math>x \in A</math>

表示「<math>x</math>是<math>A</math>中的元素」，也可以表达为「<math>x</math>是<math>A</math>的成员」「<math>x</math>在<math>A</math>中」「<math>x</math>属于<math>A</math>」。

有时也用「<math>A</math>包含<math>x</math>」表达集合隶属关系，但因为这样的说法也可以用来表达「<math>x</math>是<math>A</math>的[[子集|子集]]」，应该谨慎使用，避免歧义。<ref name="schech">{{cite book |author = Eric Schechter |title= Handbook of Analysis and Its Foundations |publisher=Academic Press |year= 1997|isbn= 0-12-622760-8 }} p. 12</ref><ref name="boolos">{{cite speech |title=24.243 Classical Set Theory (lecture) |author=George Boolos |date=February 4, 1992 |location=[[麻省理工学院|麻省理工学院]] }}</ref>不过使用符号时没有歧义，可以用

: <math>A \ni x</math>

来表达「<math>A</math>包含<math>x</math>」。

不隶属的关系可以用符号「<math>\not\in</math>」表示，记作

:<math>x \notin A</math>

意思是「<math>x</math>不是<math>A</math>的元素」。

<math>\in</math>源自[[希臘字母|希臘字母]][[ε|ε]]，虽然已经极少，但在一些文献中还有在使用，例如John Milnor的"Topology from the Differentiable Viewpoint" (1965,1997)。

== 電腦編碼 ==
在資訊科技上：
:<math>\in</math> 的 [[Unicode|Unicode]] 字元值是 U+2208 (∈)，在 [[TeX|TeX]] 中可以用代號 <code>\in</code> 打出。
:<math>\subset</math> 的 Unicode 字元值是 U+2282 (⊂)，TeX 代號為 <code>\subset</code>。
:<math>\subseteq</math> 的 Unicode 字元值是 U+2286 (⊆)，TeX 代號為 <code>\subseteq</code>。
