'''元宝塘'''(工程名为艮山西路站)是建设中的[[杭州地铁6号线|杭州地铁6号线]]的一个车站，位于杭州市[[江干区|江干区]][[彭埠街道|彭埠街道]]元宝塘社区艮山西路东宁路，沿东宁路南北向布置。<ref>{{cite web |title=最新！杭州地铁6号线25个站点名和具体位置公布了 |url=http://zj.qq.com/a/20180711/038831.htm |website=腾讯大浙网 |language=zh}}</ref>

==结构==
元宝塘站为地下岛式站台，为地下两层双柱三跨钢筋混凝土箱形结构。车站总长195.2米，标准段基坑宽21.7米，深16.1米，端头井基坑宽25.6米，南端头深17.74米，北端头深18.38米。<ref>{{cite web |title=杭州6号线14标艮山西路站一期交改完工 |url=http://www.crfeb.com.cn/cg/1510/1903628/index.html |website=中铁一局}}</ref>
