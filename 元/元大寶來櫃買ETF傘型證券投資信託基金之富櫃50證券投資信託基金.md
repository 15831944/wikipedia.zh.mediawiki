
'''元大寶來櫃買ETF傘型證券投資信託基金之富櫃50證券投資信託基金'''（簡稱'''富櫃50'''、'''寶富櫃'''，{{otc|006201}}）是[[臺灣|臺灣]][[寶來投信|寶來投信]]（現已併入[[元大寶來投信|元大寶來投信]]）於2011年1月12日成立的[[指數股票型基金|指數股票型基金]]（[[ETF|ETF]]），成立日發行價格為[[新台幣|新台幣]]15元，以一千個[[受益權|受益權]]為交易單位，也是臺灣首檔店頭市場ETF<ref>{{cite web|url=https://www.chinatimes.com/money|title=財經頻道|first=|last=中時電子報|website=中時電子報}}</ref>。富櫃50採取追蹤'''富櫃五十指數'''的被動式管理，成分股包含臺灣[[上櫃|上櫃]]股票市值前五十大的個股。至2011年6月30日止，規模已達13.54億元。權值大的成分股可能因變更為[[臺灣證券交易所|上市]]，而使富櫃50產生大比重的變更。

== 富櫃五十指數 ==
'''櫃買「富櫃五十指數」'''（GreTai 50 Index）是財團法人中華民國證券櫃檯買賣中心所編製，櫃買「富櫃五十指數」是以市值排序，並考量公眾流通量及流動性等條件，選出五十支上櫃股票以流通市值加權計算之股價指數，用以表彰櫃檯買賣股票市場之績效。指數在交易時間內即時計算，每十五秒發布一次。

== 課稅 ==
*證券交易稅：千分之一。
*交易手續費：費率為'''0.1425％'''。

== 收益來源  ==
*指數成分股的現金股利率。
*開放借券的獲利金額。
*指數成分股及權重調整時，在被動式基金交易中的買賣價差金額。

== 收益分配 ==
*收益分配為每壹仟個[[受益權|受益權]]單位所分配之現金收益，單位為[[新台幣|新台幣]]元。實際金額請查看各基金收益分配公告。
*收益評價日為每年10月31日。
