'''元宝区'''是[[中华人民共和国|中国]][[辽宁省|辽宁省]][[丹东市|丹东市]]下辖的一个[[市辖区|市辖区]]。面积90.27平方千米，人口19.12万。邮政编码118000。区人民政府驻新柳街8号。

== 行政区划 ==
下辖6个[[街道办事处|街道办事处]]、1个[[镇_(中华人民共和国)|镇]]<ref>{{cite web |url=http://dm.dandong.gov.cn/qhsc.htm |title=存档副本 |accessdate=2009-11-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20091127085859/http://dm.dandong.gov.cn/qhsc.htm |archivedate=2009-11-27 }} 2009年丹东市行政区划统计表</ref>：
*街道办事处：六道口街道、七道街道、八道街道、九道街道、广济街道、兴东街道。
*镇：[[金山镇_(丹东市)|金山镇]]。
