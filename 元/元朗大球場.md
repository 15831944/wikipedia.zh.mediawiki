'''元朗大球場'''（{{lang|en|'''Yuen Long Stadium'''}}）是一個位於[[香港|香港]][[新界|新界]][[元朗|元朗]]的[[足球場|足球場]]，於1968年落成啟用，位於[[新界|新界]][[元朗區|元朗區]][[元朗體育路|體育路]]6號。由[[元朗區體育會|元朗區體育會]]興建，近年轉交至[[香港政府部門|香港政府部門]][[康樂及文化事務署|康樂及文化事務署]]管理，球場容量4,932人。老一輩元朗人通常稱球場為「'''大陂頭'''」。

==歷史==
元朗大球場前身是[[水邊村|水邊村]]的部分土地，及後[[元朗區體育會|元朗區體育會]]決定興建元朗大球場，並得到地區人士支持。1965年招標承建，由元朗昌記建設築公司以19萬8,800港元投得<ref>《大公報》，1965年9月15日，第2張第8版，「元朗興建運動場 首期工程將展開」</ref>。

元朗大球場於1968年12月啟用，首場比賽由[[元朗區體育會|元朗]]對[[消防足球隊|消防]]；同日舉行由[[香港業餘田徑總會|香港業餘田徑總會]]主辦，《[[天天日報|天天日報]]》[[贊助|贊助]]，在香港首次舉行的[[馬拉松|馬拉松賽跑]]，元朗大球場作為比賽的起點及終點<ref>[[韋基舜|韋基舜]]，《吾土吾情II》，第260頁，「春光明媚跑馬拉松」，成報出版社，ISBN 988984576-8</ref>。

1983年獲[[英皇御准香港賽馬會|英皇御准香港賽馬會]]捐助300萬港元擴建球場，工程包括一個可容2,000觀眾有蓋看台、一條8線全天候跑道及重鋪球場草坪，全部費用共達1,200萬元<ref>《大公報》，1983年5月22日，第2張第8版，「元朗擴建完成 可供足球賽使用」</ref>。

===重建計劃===
2013年10月，康樂及文化事務署落實重建有45年歷史的元朗大球場，使大球場能符合國際標準，日後可舉辦國際頂級賽事。當局建議將鄰近停車場納入球場範圍，並增座位數目至最少5,000個及加設上蓋，有關工程最快在2015年展開。<ref>{{cite newspaper|url=https://hk.news.yahoo.com/%E5%85%83%E6%9C%97%E5%A4%A7%E7%90%83%E5%A0%B4%E9%87%8D%E5%BB%BA%E8%AE%8A%E8%BA%AB%E5%9C%8B%E9%9A%9B%E7%B4%9A-214748174.html|title=元朗大球場重建變身國際級|newspaper=[[星島日報|星島日報]]|date=2013年10月26日}}</ref>

2017年3月發表的《[[香港施政報告|施政報告]]》提及重建元朗大球場已獲納入「體育及康樂設施五年計劃」，康樂及文化事務署向元朗區議會提交文件交代擬建設施詳情，重建後的大球場將提供包括各種符合舉辦國際性足球賽事的附屬設施。重建工程包括重置一個11人天然草地足球場，設[[泛光燈|泛光燈]]，重置一條全天候8線400米跑道及田項運動設施。為容納更多觀看國際性足球賽事的觀眾，康文署擬重置西看台，由現時2046個有蓋座位增至4000個有蓋座位，包括250個貴賓席，以及將現有不設上蓋的東看台全部2886個座位及附屬設施進行翻新，東西兩面看台將提供合共約6900個座位，比原擬建設施的總座位數量多約1900個，大球場也將設有多用途會議室、傳媒中心、記者工作區、新聞發布中心、評述員室、電視轉播室等附屬設施。<ref name="地委會文件">{{Cite web |url=https://www.districtcouncils.gov.hk/yl/doc/2016_2019/en/committee_meetings_doc/dfmc/14037/dfmc_2018_004.pdf |title=重建元朗大球場 最新的擬建設施 |accessdate=2018-07-08 |date=2018-01-05 |publisher=地委會文件2018／第4號　康樂及文化事務署}}</ref><ref>{{cite newspaper|url=http://hk.on.cc/hk/bkn/cnt/news/20171229/bkn-20171229131031447-1229_00822_001.html|title=元朗大球場重建有望　將可舉辦國際足球賽|date=2017年12月29日|newspaper=[[東方日報_(香港)|東方日報 (香港)]]}}</ref>

當局的重建方案中並不打算在東看台加建上蓋，只翻新其2886個座位及附屬設施<ref name="地委會文件"/>，在元朗區議會地區設施管理委員會上多名議員不滿方案，一度反對將前期工程諮詢提交立法會民政事務委員會。康文署稱在東看台加建上蓋有技術困難<ref>{{Cite web |url=https://www.inmediahk.net/node/1054436 |title=元朗大球場重建 區議會拒支持 不滿東看台無上蓋 |date=2018-01-10 |publisher=[[香港獨立媒體|香港獨立媒體]]}}</ref>。民政事務局副局長[[陳積志|陳積志]]稱現時部分球場的看台未設上蓋，認為現有方案切實可行<ref>{{Cite web |url=http://orientaldaily.on.cc/cnt/news/20180707/mobile/odn-20180707-0707_00176_046.html |title=元朗大球場重建 被批半桶水 |date=2018-07-07 |publisher=[[東方日報_(香港)|東方日報 (香港)]]}}</ref>。

==活動==
;足球
1970年代至1980年代，[[元朗區體育會|元朗]]足球隊曾以元朗大球場為主場，舉行[[香港甲組足球聯賽|香港甲組足球聯賽]]。現時元朗地區足球隊參與[[香港乙組足球聯賽|香港乙組足球聯賽]]及元朗區足球賽，部份賽事亦會安排在元朗大球場舉行<ref>[http://www.hkfa.com/zh-hk/match_venue.php 香港足總：比賽場地] {{webarchive|url=https://web.archive.org/web/20080930065418/http://www.hkfa.com/zh-hk/match_venue.php |date=2008-09-30 }}</ref>。

2008年7月22日，[[天水圍飛馬足球會|天水圍飛馬]]向[[香港足球總會|香港足球總會]]申請，於[[2008年至2009年香港甲組足球聯賽|2008-09球季]]，以元朗大球場作主場<ref>{{Cite news|url=http://www.takungpao.com/news/08/07/22/WF-935714.htm|title=接納四海「空降」/ 今季甲組增至13隊|publisher=大公網|date=2008年7月22日|accessdate=2008年7月22日|deadurl=yes|archiveurl=https://web.archive.org/web/20080915152917/http://www.takungpao.com/news/08/07/22/WF-935714.htm|archivedate=2008年9月15日}}</ref>。

2008年9月1日，足總主席梁孔德在其個人網誌上表示，2008年10月5日天水圍飛馬對[[南華足球隊|南華]]之賽事，已獲客隊南華同意安排在元朗大球場舉行；同日另一場甲組賽事（[[淦源足球會|淦源]]對[[公民足球隊|公民]]）亦雙方同意於元朗大球場進行比賽<ref>[http://hk.myblog.yahoo.com/chairman-blog/article?mid=1939 大披頭安排甲組賽事]，Chairman Blog，2008年9月1日</ref>。然而，10月5日當天卻因天雨關係而延期，足總只安排了天水圍飛馬對南華於3天後於元朗大球場進行補賽，事隔二十載後再次有甲組賽事於元朗大球場舉行<ref>{{Cite news|url=http://www.hkfa.com/includes_files/board_details.php?news_id=6154|title=快意空調甲組聯賽延期舉行|publisher=香港足球總會網站|date=2008年10月8日|accessdate=2008年10月5日}}</ref>；但淦源對公民則改於旺角大球場作賽<ref>[http://www.hkfa.com/zh-hk/match_schedule.php?match_month=10&match_year=2008 2008年10月賽程表]，香港足球總會</ref>。

[[2009年－10年香港甲組足球聯賽|2009-2010年度球季]]，[[香港足球總會|香港足球總會]]全面試行主客制，[[天水圍飛馬|天水圍飛馬]]獲批准繼續使用元朗大球場作為主場球場<ref>{{cite news |language=zh-hk |url=http://hk.apple.nextmedia.com/template/apple/art_main.php?iss_id=20090715&sec_id=25391&art_id=12989512 |title=11甲組隊落實主場編配 |publisher=《[[蘋果日報|蘋果日報]]》 |date=2009-07-15 |accessdate=2009-07-16}}</ref>。

;會景巡遊
[[元朗區|元朗區]]慶祝[[天后寶誕|天后寶誕]]之[[會景巡遊|會景巡遊]]，由1963年開始每年於農曆3月23日上午起舉行。巡遊於[[雞地|雞地]]集合出發，途經教育路到元朗大球場匯演，最後到[[大樹下天后廟|大樹下天后廟]]酬謝神恩<ref>[http://www.yl.hk/tinhau/index.htm 元朗天后寶誕會景巡遊]</ref>。

;萬人[[盆菜|盆菜]]
[[新界鄉議局|新界鄉議局]]在元朗大球場的「萬人[[盆菜|盆菜]]宴」，將盆菜熱潮推向高峰<ref>[http://www.chineseculture.org.hk/history/report/13prize_high4.htm 盆菜的傳承─由新界民俗到香港文化]</ref>。

==設施==
元朗大球場設有一個草地球場、一條400米長跑道、兩邊看台及計時大鐘。
