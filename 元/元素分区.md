根据[[元素|元素]]外层[[价电子|价电子]][[电子排布|构型]]的不同，周期表可以分成几个'''区'''；同一区里的元素，其填在最高能级上的电子的亚层轨道类型是相同的。分区的名称就是根据轨道的名称定的：

== s区元素 ==
{{main|s区元素}}
主要包括元素週期表中IA族元素（1族元素）和IIA族元素（2族元素），IA族元素包括[[氫|氫]]、[[鋰|鋰]]、[[鈉|鈉]]、[[鉀|鉀]]、[[銣|銣]]、[[銫|銫]]、[[鈁|鈁]]、[[Uue|Uue]]、Uhe九種元素，由於鈉和鉀等金屬的[[氫氧化物|氫氧化物]]是典型的[[鹼|鹼]]，因此除氫外的這八種元素又稱[[鹼金屬|鹼金屬]]，IIA族元素包括[[鈹|鈹]]、[[鎂|鎂]]、[[鈣|鈣]]、[[鍶|鍶]]、[[鋇|鋇]]、[[鐳|鐳]]、[[Ubn|Ubn]]、Usn八種元素，由於鈣，鍶，鋇的[[氧化物|氧化物]]之性質介於鹼金屬與[[稀土元素|稀土元素]]之間，因此又稱[[鹼土金屬|鹼土金屬]]。

由於[[氦|氦]]的[[電子排布|電子排布]]為1s<sup>2</sup>，故被分為s區元素。

鈁和鐳都是[[放射性|放射性]]元素。鋰最重要的[[礦石|礦石]]是[[鋰輝石|鋰輝石]]（LiAlSi<sub>2</sub>O<sub>6</sub>）。鈉主要以[[氯化鈉|氯化鈉]]溶液的形式存在於海洋，鹽湖及岩石中。鉀的主要礦物是[[鉀石鹽|鉀石鹽]]（2KCl·MgCl<sub>2</sub>·6H<sub>2</sub>O）。鈹的主要礦物是[[綠柱石|綠柱石]]（3BeO·Al<sub>2</sub>O<sub>3</sub>·6SiO<sub>2</sub>）。鎂的主要礦石是[[菱鎂礦|菱鎂礦]]（MgCO<sub>3</sub>）及[[白雲石|白雲石]]。另外，鈣，鍶，鋇則主要以[[碳酸鹽|碳酸鹽]]及[[硫酸鹽|硫酸鹽]]的形式存在，如[[方解石|方解石]]（碳酸鈣），[[石膏|石膏]]（二水合硫酸鈣），[[天青石|天青石]]（硫酸鍶），[[重晶石|重晶石]]（硫酸鋇）。

在本區元素中同一主族從上到下、同一週期從左至右性質的變化都呈現明顯的規律性。

== p区元素 ==
{{main|p区元素}}
p區元素包括元素週期表中IIIA族[[元素|元素]]~VIIIA族元素。

IIIA族元素（13族元素）又稱為[[硼族元素|硼族元素]]，包括[[硼|硼]]、[[鋁|鋁]]、[[鎵|鎵]]、[[銦|銦]]、[[鉈|鉈]]、[[鉨|鉨]]、[[Uht|Uht]]等元素；

IVA族元素（14族元素）又稱作[[碳族元素|碳族元素]]，包括[[碳|碳]]、[[硅|硅]]、[[鍺|鍺]]、[[錫|錫]]、[[鉛|鉛]]、[[鈇|鈇]]、[[Uhq|Uhq]]等元素；

VA族元素（15族元素）又稱作[[氮族元素|氮族元素]]，包括[[氮|氮]]、[[磷|磷]]、[[砷|砷]]、[[銻|銻]]、[[鉍|鉍]]、[[鏌|鏌]]、[[Uhp|Uhp]]等元素；

VIA族元素（16族元素）又稱為[[氧族元素|氧族元素]]，包括[[氧|氧]]、[[硫|硫]]、[[硒|硒]]、[[碲|碲]]、[[釙|釙]]、[[鉝|鉝]]、[[Uhh|Uhh]]等元素；

VIIA族元素（17族元素）又稱[[鹵素|鹵素]]，包括[[氟|氟]]、[[氯|氯]]、[[溴|溴]]、[[碘|碘]]、[[砹|砹]]、[[鿬|鿬]]、[[Uhs|Uhs]]等元素；

VIIIA族元素（18族元素）或O族元素，又稱為[[稀有氣體|稀有氣體]]或惰性氣體，包括[[氖|氖]]、[[氬|氬]]、[[氪|氪]]、[[氙|氙]]、[[氡|氡]]、[[鿫|鿫]]、[[Uho|Uho]]等元素。（氦為s區元素）

== d区元素 ==
{{main|d区元素}}
d區元素是元素週期表中的副族元素，即[[3族元素|第3族元素]]中的[[鈧|鈧]]、[[釔|釔]]、[[鎦|鎦]]、[[鐒|鐒]]、[[Upt|Upt]]，以及[[4族元素|第4]]至[[12族元素|第12族元素]]。這些元素中具有最高能量的電子是填在[[d軌域|d軌域]]上的。這些元素有時也被稱作[[過渡金屬|過渡金屬]]。D區的D來自英語的「diffuse」。

== f区元素 ==
{{main|f区元素}}
f區元素亦可稱為是'''內過渡元素'''，指的是元素週期表中的[[鑭系元素|鑭系元素]]和[[錒系元素|錒系元素]]，以及原子序139到152的[[超錒系元素|超錒系元素]]。大多數元素具有最高能量的電子是排布在[[原子軌道|f軌道]]上的。這一區中同週期的元素之間的性質差別很小，這一點在鑭系各元素中表現得很明顯。

== g区元素 ==
{{main|g区元素}}
g區元素包括[[擴展元素週期表|擴展元素週期表]]中[[第八週期元素|第八週期]]的第121號[[元素|元素]]-138號元素，其中所有元素均尚未被發現，包括：<br>[[Ubu|Ubu]]、[[Ubb|Ubb]]、[[Ubt|Ubt]]、[[Ubq|Ubq]]、[[Ubp|Ubp]]、[[Ubh|Ubh]]、[[Ubs|Ubs]]、[[Ubo|Ubo]]、[[Ube|Ube]]、[[Utn|Utn]]、[[Utu|Utu]]、[[Utb|Utb]]、[[Utt|Utt]]、[[Utq|Utq]]、[[Utp|Utp]]、[[Uth|Uth]]、[[Uts|Uts]]、[[Uto|Uto]]，以及[[第九週期元素|第九週期]]的171-188號元素，包括[[Usu|Usu]]、[[Usb|Usb]]、[[Ust|Ust]]、[[Usq|Usq]]、[[Usp|Usp]]、[[Ush|Ush]]、[[Uss|Uss]]、[[Uso|Uso]]、[[Use|Use]]、[[Uon|Uon]]、[[Uou|Uou]]、[[Uob|Uob]]、[[Uot|Uot]]、[[Uoq|Uoq]]、[[Uop|Uop]]、[[Uoh|Uoh]]、[[Uos|Uos]]、[[Uoo|Uoo]]。但是不具電荷的原子的原子序最大可達到173，而其後的元素必須以離子的形態存在，所以根據電子組態分區將變得毫無意義，故第9週期的173號元素以後的g區元素可能只在理論上成立。

== h区元素 ==
h区元素指的是該[[元素|元素]]具有最高能量的电子是排布在[[原子軌道|h轨道]]上。

照元素周期表的排列，若有發現或合成[[原子序|原子序]]大於或等於221（Bbu）以上的元素，它將會屬於h區元素，但是不具電荷的原子的原子序最大可達到173<ref name="Greiner">Walter Greiner and Stefan Schramm, ''Am. J. Phys.'' 76, 509 (2008), and references therein.</ref>，而其後的元素必須以離子的形態存在，所以根據[[電子組態|電子組態]]分區將變得毫無意義，故h区元素可能只在理論上成立，故更不可能有[[i区元素|i区元素]](具有最高能量的电子是排布在'''i轨道'''上)。

照元素周期表的排列，'''h区元素'''有40個（假設元素週期律與[[馬德隆規則|馬德隆規則]]永遠成立），目前沒有任何一個被發現，都分佈在[[第10周期元素|第10周期]]（含）以後，位於s區元素和g區元素之間，第10週期的h區元素有：Bbu、Bbb、Bbt、Bbq、Bbp、Bbh、Bbs、Bbo、Bbe、Btn、Btu、Btb、Btt、Btq、Btp、Bth、Bts、Bto、Bte、Bqn、Beq、Bep、Beh、Bes、Beo、Bee、Tnn、Tnu、Tnb、Tnt、Tnq、Tnp、Tnh、Tns、Tno、Tne、Tun、Tuu、Tub、Tut，但估计'''h区元素'''应该都不存在。

== ds区元素 ==
ds區元素是指元素週期表中的[[銅族元素|IB]]、[[鋅族元素|IIB]]兩族元素，包括[[銅|銅]]、[[銀|銀]]、[[金|金]]、[[鋅|鋅]]、[[鎘|鎘]]、[[汞|汞]]6種自然形成的[[金屬元素|金屬元素]]和[[錀|錀]]、[[鎶|鎶]]2種[[人工合成元素|人工合成元素]]以及理論上的Uhu、Uhb兩種尚未被發現的元素。ds區的名稱是因為它們的電子構型都是d<sup>10</sup>s<sup>1</sup>（ⅠB）或d<sup>10</sup>s<sup>2</sup>（ⅡB）。ds區是d區元素的一部分，ds區元素都是[[過渡金屬|過渡金屬]]。但由於它們的d層是滿的，所以體現的性質與其他過渡金屬有所不同（比如說最高的氧化態只能達到+3）。
