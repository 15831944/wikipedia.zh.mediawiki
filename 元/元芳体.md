'''元芳体'''，原是[[中國中央電視台電視劇頻道|中國中央電視台電視劇頻道]]撥出的电视剧《[[神探狄仁杰|神探狄仁杰]]》中[[狄仁杰|狄仁杰]]{{efn|1=狄仁杰，历史真实人物，[[唐朝|唐朝]][[武则天|武则天]]时期著名大臣。电视剧中扮演者是[[梁冠华|梁冠华]]。 }}的一句台词“元芳，此事你怎么看”，[[李元芳|李元芳]]{{efn|1=李元芳，《神探狄仁杰》中狄仁杰的卫队长，武功卓越，有勇有谋。扮演者[[张子健|张子健]]。 }}每次均回答，“大人，我觉得此事有蹊跷。”或“此事背后一定有一个天大的秘密。”在2012年，经过网络传播和网友改动成为“元芳，你怎么看”，此句子一般用于新闻事件的评论帖子。<ref name="fish " >{{cite web |url= http://news.qq.com/a/20121016/001349.htm|title=网友恶搞“元芳体”：你对幸福怎么看？我姓李 |accessdate=2012年10月16日 |publisher=腾讯网 |work= |language =zh-cn}}</ref><ref name=" bull" >{{cite web |url=http://ent.sina.com.cn/s/m/2012-10-18/12593766632.shtml |title=“元芳体”爆红网络：元方 你怎么看？ |accessdate= 2012年10月18日|publisher= 新浪网|work= |language =zh-cn}}</ref>

== 来源 ==
元芳体起源于2004年[[电视剧|电视剧]]《神探狄仁杰》中狄仁杰和助手李元芳的对话，每次狄仁杰发现案件之后，问李元芳，“元芳，此事你怎么看”，而后，李元芳按例回答“大人，我觉得此事有蹊跷。”或“此事背后一定有一个天大的秘密。”这个对话语句经常反复出现在电视剧中。<ref name="fish " /><ref name=" bull" />

== 传播 ==
李元芳和狄仁杰的这个句子被网络传播开来之后，改成“元芳，你怎么看”，呼为“元芳体”。<ref name="fish " /><ref name=" bull" />

== 当事人澄清 ==
“元芳体”在网络上走红以后，当事人[[张子健|张子健]]{{efn|1=张子健，中国演员，1968年出生，毕业于北京电影学院。电视剧《神探狄仁杰》中李元芳的扮演者。 }}曾接受多家媒体采访。据他透露，电视剧《神探狄仁杰》中，狄仁杰在断案时确实经常征求李元芳的意见。然而经《神探狄仁杰》剧组内部人员考证后发现，剧中狄仁杰从没有说过“元芳，此事你怎么看”这句台词，而李元芳的对答也并非“元芳体”那样千篇一律。因此，网络流传截图上的字幕“元芳，你怎么看？”均係网友恶搞；而“元芳体”本身也只是网友对于电视剧《神探狄仁杰》中台词的一种总结和发挥罢了。<ref>{{cite web |url=http://ent.sina.com.cn/v/m/2012-10-26/13483772999.shtml |title= 元芳扮演者张子健澄清大热台词：剧中无此句|accessdate=2012年10月26日 |publisher=新浪网 |work= |language =zh-cn}}</ref>
