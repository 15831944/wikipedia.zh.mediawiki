'''元明宗和世㻋'''（{{mong|m=ᠬᠦᠰᠡᠯᠡᠨ <br />ᠬᠠᠭᠠᠨ|Ppp=küsele qaγan|k=Хүслэн хаан}}；{{bo|z=ཀུ་ཤ་ལ།|w=ku sha la}}<ref>见《[[红史|红史]]》</ref>；{{bd|1300年|12月22日|1329年|8月30日|catIdx=Y}}），是[[元朝|元朝]]第九位[[皇帝|皇帝]]，[[蒙古帝国|蒙古帝国]]第十三位[[蒙古大汗|大汗]]，1329年2月27日至1329年8月30日在位，在位185天。[[元武宗|元武宗]]長子。清代乾隆晚期乾隆帝命改譯遼、金、元三史中的音譯專名，改譯'''和實拉'''{{NoteTag|清刊《[[元史|元史]]》、清修《[[續資治通鑑|續資治通鑑]]》所用譯名。}}，今日學界已無人使用。

他去世后，[[谥号|谥号]]'''翼獻景孝皇帝'''，[[庙号|庙号]]'''明宗'''，蒙古语称'''忽都篤皇帝'''（{{mong|m=ᠬᠤᠲᠤᠭᠲᠤ <br />ᠬᠠᠭᠠᠨ|k=Хутагт хаан|Ppp=qutuγtu qaγan}}）{{NoteTag|「忽都篤」在蒙古語中意思是「有福祿的」。}}。

1340年10月25日，[[元惠宗|元惠宗]]為'''元明宗'''上汉语[[尊号|尊号]]'''順天立道睿文智武大聖孝皇帝'''。

== 生平 ==
[[File:摛澡堂四庫全書薈要_（明）宋濂_元史·卷三十一.151.tif|thumb]]卷三十一：天厯二年正月乙丑，文宗復遣中書左丞伊爾特穆爾來迎。乙酉，薩題等至，入見帝於行幄，以文宗命勸進。丙戌，帝即位于和寧之北，扈行諸王、大臣咸入賀。乃命薩題遣人還報京師。</ref>]]
根据《元史》，天曆二年正月丙戌<ref>[[元史|元史]]卷三十一：天厯二年正月乙丑，文宗復遣中書左丞伊爾特穆爾來迎。乙酉，薩題等至，入見帝於行幄，以文宗命勸進。丙戌，帝即位于和寧之北，扈行諸王、大臣咸入賀。乃命薩題遣人還報京師。</ref>（[[儒略曆|儒略曆]]1329年2月27日），和世琜在漠北草原的[[哈拉和林|和宁]]之北即位，继续使用[[年号|年号]]“[[天历_(元朝)|天曆]]”，是为元明宗，1329年4月3日，[[元文宗|元文宗]]图帖睦尔派人将皇帝宝玺献给明宗，正式[[禅让制|禪讓]][[帝位|帝位]]<ref>[http://www.guoxue.com/shibu/24shi/yuanshi/yuas_033.htm 《元史·文宗本紀二》]</ref>，5月15日，元明宗正式立图帖睦尔为皇太子，8月16日，图帖睦尔受皇太子宝，8月25日，元明宗抵达[[元武宗|元武宗]]时建为中都的[[王忽察都|王忽察都]]，8月26日，皇太子图帖睦尔入见，两兄弟会面，元明宗宴请皇太子及诸王、大臣于行殿。

天曆二年八月六日（1329年8月30日），元明宗和世㻋被[[燕帖木儿|燕帖木儿]]毒死，明宗去世时享年30岁。

1329年9月8日，燕帖木儿重新拥戴元文宗[[復辟|復辟]]，因为1329年的[[年号|年号]]是'''天曆'''，[[历史|史]]称'''天曆之变'''。

== 尊谥庙号 ==

[[天历_(元朝)|天曆]]二年十月十三日（1329年11月4日），元文宗为兄長'''和世㻋'''上[[谥号|谥号]]'''翼獻景孝皇帝'''，[[庙号|庙号]]'''明宗'''，蒙古文称'''忽都篤皇帝'''。<ref>《元史》卷三十三《文宗本纪二》记载：天历二年十月，“丙申，中书省臣言：“臣等谨集枢密院、御史台、翰林、集贤院、奎章阁、太常礼仪院、礼部诸臣僚，议上大行皇帝尊谥曰翼献景孝皇帝，庙号明宗，国言谥号曰护都笃皇帝。”是日，奉玉册、玉宝于太庙，如常仪。”[http://www.guoxue.com/shibu/24shi/yuanshi/yuas_033.htm 《元史·文宗本紀二》]</ref>

《明宗皇帝谥册文》<ref> 《明宗皇帝谥册文》作者为虞集，册文内容节选自：苏天爵《国朝文类》卷十《册文》。</ref>，内容如下：
{{quote|臣闻统必有宗，生尝得以致其让；庙必有主，殁思所以尊其名。稽古考文，宜天锡诔。钦惟先皇帝夙秉勇智，惟时元良，体传次之成言，避讴歌而逖出。虽身居绝域，多历于岁年；而义动远人，乐为之先后。德威孔著，未堪大业之艰贞；事变匪常，犹悯生灵而慎动。庶来苏于徯戴，爰勘定以奉迎。已谨清宫，俄虚黄屋，臣民寡祐，永遗惠泽之敷施；天日有临，尚想神明之如在。礼严升祔。诚备显扬。谨遣摄太尉某官某，奉玉册玉宝，上尊谥曰翼献景孝皇帝，庙号明宗。

伏冀睿慈，俯回歆假，克绥丕祚，垂裕无疆。
}}

== 尊号 ==

元明宗的两个儿子[[元宁宗|元宁宗]]懿璘质班和[[元順帝|元惠宗]]妥懽帖睦尔在1332年9月2日[[元文宗|元文宗]]去世后相继登基称帝。

[[至元_(元惠宗)|至元]]六年十月四日（1340年10月25日），[[元順帝|元惠宗]]给元明宗上[[尊号|尊号]]'''順天立道睿文智武大聖孝皇帝'''。<ref>[[元順帝|元惠宗]]又称[[元顺帝|元顺帝]]，《元史》卷四十《顺帝本纪三》记载，至元六年，“冬十月甲申，奉玉册、玉宝尊皇考为顺天立道睿文智武大圣孝皇帝，亲祼太室。”[http://www.guoxue.com/shibu/24shi/yuanshi/yuas_040.htm 《元史·顺帝本紀三》]</ref>

== 評價 ==

* [[清朝|清朝]]史学家[[邵遠平|邵远平]]《[[元史类编|元史类编]]》的評價是：“册曰：艰艰备尝，人望所属；何嫌何疑，推肝置腹；人心不同，天命反覆；论定千秋，此直彼曲。”<ref>《元史类编》卷八《明宗》</ref>

* [[清朝|清朝]]史学家[[曾廉|曾廉]]《[[元书|元书]]》的評價是：“论曰：昔曹子臧、吴季札，贤者也。其君国子民也宜哉！然而义不受者，非独远情，亦知负飞及光之不厌，其欲将无以善其后也，闇哉明宗！焉有人披衮执玉，穆穆然位乎天位而肯北面俯首为人臣者乎？呜呼！此唐明皇不敢以望肃宗，父子且然，况兄弟哉!文宗盖惧北陲，复有海都、笃哇之流，托名拥戴，其言也顺而为患也。深抑亦私心，窃望周王之效法晋邸也。己则非夷，而以齐期人。不亦难乎？悠悠南行，甘咽其饵，悲夫！”<ref>《元书》卷十三《明宗本纪》</ref>

* [[清朝|清]]末[[民国|民]]初史学家[[屠寄|屠寄]]《[[蒙兀儿史记|蒙兀儿史记]]》的評價是：“和世㻋汗年未弱冠，远逊金山，耕牧十有三年。所谓旧劳于外，知民情伪者也。观其论台纲，谕百司，斤斤于先世成宪，是殆有心救弊者乎？然以此言论风采，自曝于风尘道路之间，致令傲弟权相闻而生心，遂有旺兀察都之变。《易》曰：‘君不密，则失臣。’此之谓矣。怀抱盛意，未见设施，惜哉！”<ref>《蒙兀儿史记》卷十四《和世㻋汗本纪》</ref>

* [[民国|民国]]官修[[正史|正史]]《[[新元史|新元史]]》[[柯劭忞|柯劭忞]]的評價是：“燕铁木儿立文宗，文宗固让于兄，犹仁宗之奉武宗也。明宗之弑，盖出于燕铁木儿，非文宗之本意。然与闻乎弑，是亦文宗弑之而已。”<ref>
[http://www.guoxue123.com/shibu/0101/00xys/019.htm 《新元史》卷二十《明宗本紀》]</ref>
