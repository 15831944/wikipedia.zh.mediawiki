'''元清閣'''（[[英文|英文]]：Yuen Ching Kok）是[[香港|香港]]一所供奉[[黃大仙|黃大仙]]的[[道教|道教]][[廟宇|廟宇]]，位於[[九龍|九龍]][[青山道|青山道]]的新圍村，地址為九龍[[蘇屋|蘇屋]]一段的[[呈祥道|呈祥道]]3738號地段，佔地面積約2公頃，分成高低兩組建築群。南鄰供奉[[觀世音菩薩|觀世音菩薩]]及[[孫悟空|孫悟空]]的[[紫陽洞|紫陽洞]]。

== 歷史 ==
在[[香港日治時期|香港日治時期]]，居住在九龍西部的市民前往[[嗇色園|嗇色園]]參神時屢受到日軍凌辱。那時，[[潮州|潮州]]商人周亮星先生得知居民苦況，便專程在1942年4月15日 （星期二）到嗇色園禮請黃大仙傳聖[[土瓜灣|土瓜灣]][[北帝街|北帝街]]23號二樓，道壇初名為『駐憇亭』。法壇及後於1942年8月22日（星期六）遷往[[九龍|九龍]][[聯合道|聯合道]]38號三樓，1942年9月25日（星期五）更名「黃大仙元清閣」，並定[[農曆|農曆]]十月初十日為成立日。爾後，黃大仙屢降乩示，指示當屆理事朝青山道覓地置閣，後蒙潮商陳創穆先生捐地，遂在九龍[[青山道|青山道]]的新圍村現址建設了香港另一座供奉黃大仙的[[道觀|道觀]]。現時的[[建築物|建築物]]在1955年春季落成，1955年4月24日（農曆閏三月初三日，星期日)舉行開光大典。

== 近況 ==
在2018年10月舉行會員大會，並於10月14日董事會經投票選出新任主席何炳堃，副主席周修忠、何鑑周、周全安、彭洪輝。  

在2005年9月，該道觀曾試圖進行籌募[[經費|經費]]計劃，重建成一幢現代化的三層道觀，附設[[升降機|升降機]]。
* 頂層：供奉黃大仙像、「[[王章|王章]]」像、「[[張禹|張禹]]」像。
* 中層：供善信供奉祖先靈位；
* 底層：素菜食堂、廚房及[[盥洗室|盥洗室]]；

* 設有24個位置的無蓋[[停車場|停車場]]。
* 然而直至2018年仍然未能籌得足夠經費展開該重建計劃。

== 現時建築 ==
* 大門
** 橫匾：「黃大仙祠」
** 對聯：「金生麗水乾卜留聖跡；華光普照坤造別洞天。」
* 主殿供奉道教的黃大仙像、「王章」像、「張禹」像；
* 戶外有蓋食堂：在農曆初一日、十五日及農曆8月23日的「黃大仙寶誕」，設有免費[[素食|素食]]招待前來觀光的人士。
* 龜池：像[[新加坡|新加坡]]的[[龜嶼|龜嶼]]之「大伯公廟」般的池塘，中央亦有一隻石龜，牠的背上有一條[[眼鏡蛇|眼鏡蛇]]。石碑：善因褔果。
* 辦事處、義診部、[[扶乩|扶乩]]房、善信休憩處、盥洗室。

== 服務 ==
* 它是目前中唯一提供扶乩服務給民眾的「黃大仙道觀」，客人只需把需要詢問的事情以紅色信箋寫好，在主殿的檀前燒化給黃大仙的聖像，坐在側堂的乩童會受感應寫出四句[[七言律詩|七言律詩]]回答。然而並不是所有信件都必定回覆。

* 中醫義診暫停

== 交通 ==
* 乘搭前往[[蘇屋邨|蘇屋邨]]的[[巴士|巴士]]，然後沿[[明愛醫院|明愛醫院]]方向前往，再穿過行人天橋便是。此外亦可乘搭[[九龍巴士|九巴]][[九龍巴士72線|72]]及[[九龍巴士81線|81]]在[[爾登華庭|爾登華庭]]站下車，然後沿[[大埔道|大埔道]]步行前往。
* 在農曆初一、十五及農曆8月23日的「黃大仙寶誕」，上午10時30分及11時正，在[[美孚新邨|美孚新邨]][[百老匯街|百老匯街]]設有兩班免費的客車接載[[觀光客|觀光客]]前往參觀。回程時間在下午1時45分及2時30分。
