'''元规则'''是指中国学者[[吴思|吴思]]提出的一条历史规律：暴力最强者说了算。他以此“原始的规则”来描述何为“决定规则的规则”。

== 解释 ==
这条规律认为历史上的明文法规和人们私下的认可的规矩（如[[行规|行规]]、[[家规|家规]]、[[江湖规矩|江湖规矩]]等等，吴思将其命名为“[[潜规则|潜规则]]”和“[[横规矩|横规矩]]”）都是由利害格局中具有最强伤害能力的一方制订的。例如平民面对[[土匪|土匪]]、[[官吏|官吏]]等等有枪有权（权力事实上还是从枪杆子里来的）势力的时候，所要遵循和认可的行为标准是由对方决定的。

== 实例 ==
吴思在《[[血酬定律：中国历史中的生存游戏|血酬定律：中国历史中的生存游戏]]》一书中的自序对暴力最强者立法定规的历史现象进行过列举，引用如下：

{{quote|
我留心看过《大明律》的制订过程，也留意过明朝初年更高级别的法规《大诰》如何制订，如何实施，如何失而复行，又如何架空撤消。制订《大明律》的时候，几个大儒参照唐朝的法律，一条一条地修订，一条一条地草拟，朱元璋又一条一条地品评，修改，改了又改，最后立为天下法。但是皇帝本人并不遵行，另外编撰了一套个人色彩浓重的严刑苛法《大诰》。朱元璋死后，他的孙子即位，放弃了《大诰》，随后被自己的叔叔打败，夺了帝位。新皇帝上台，又恢复了《大诰》。在这些来回折腾中，决定和选择法规的规则变得十分清晰，那就是：暴力最强者说了算。在晚清的频繁变法中，这条元规则再次清晰地显露出来：暴力竞争的胜利者说了算，无论胜利者是洋人还是女人。 
}}

现代历史上，[[毛泽东|毛泽东]]的政治口号「[[枪杆子里面出政权|枪杆子里面出政权]]」也表达了相同的含义。

== 拓展 ==
吴思在其后的文章<ref>[http://blog.china.com.cn/wusi/art/3642.html 岳飞之死与暴力均衡]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>中，把“'''元规则'''”所描述的内容加以拓展：当利害格局中某几方伤害能力彼此相差不多乃至相互制衡之时，他们之间就会出现“[[暴力均衡|暴力均衡]]”这种状态，这种状态下的各方可以平等的相互交往，一旦这种均衡被打破，他们之间就会出现以伤害能力（即暴力）为依凭的称霸与称臣、奴役与进贡现象，在他之后的数次演讲中{{NoteTag|大多数是关于“[[官家主义|官家主义]]”的演讲。}}更把“暴力均衡”称作为人类从事生产活动的前提。

== 批评与质疑 ==
有些人认为在如今的民主宪政国家下，再讲“暴力最强者说了算”不合时宜，吴思自己认为<ref>[http://blog.china.com.cn/wusi/art/537375.html 血酬史观——中国历史的一种分析框架]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>：
{{quote|
作为生产阶级的首领、生产各个要素的组织者、市场需求的发现者，生产集团的首领当家作主了，立法立规了，他们成了暴力最强者，他们控制了军队、控制了国家机器，这时候生产集团控制了暴力集团，他们可以根据自己的心愿立法；这个时候[[经济基础|经济基础]]确实可以决定[[上层建筑|上层建筑]]了。
}}
