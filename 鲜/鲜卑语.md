'''鲜卑语'''在中国史书中称为'''夷言'''、'''国语'''、'''北语'''、'''胡语'''或者'''胡言'''，為[[中国历史|中国]]与[[蒙古历史|蒙古历史]]上[[鲜卑|鲜卑族]]使用的一种[[语言|语言]]，较为保守的使用时期为2、3世纪交替至7世纪中叶，在[[东晋|东晋]][[十六国|十六国]]至[[南北朝|北朝]]时期被广泛地入主[[中原|中原]]的鲜卑族作为本民族语言使用。作为当时统治者所使用的语言，鲜卑语一度在[[中国北方|中国北方]]成为仅次于[[汉语|汉语]]的[[声望语言|声望语言]]。至到[[北魏|北魏]]时期，[[北魏孝文帝|孝文帝]]和[[文成文明皇后|冯太后]]下令进行[[孝文帝改革|汉化改革]]，迁都[[洛阳市|洛阳]]，改用汉语代替鲜卑语，更改鲜卑语姓名为汉字姓名。禁止入住中原的鲜卑人使用鲜卑语，无疑使到鲜卑语的生存空间大幅度地缩小。[[六镇之乱|六镇之乱]]后，汉化比较浅的六镇部将再度推广鲜卑语的使用，中国北部掀起了鲜卑化热潮。直至[[隋朝|隋朝]]末期，鲜卑语失传，鲜卑人作为一个独立于[[汉族|汉族]]以外的民族的历史從此结束。

然而，在北魏的改革过程中，為现代[[语言学家|语言学家]]留下了解读鲜卑语的线索。北魏更改胡名为汉名，基本采用了语音相近而转写和语义相近而直译的两种方法{{参|1={{Cite journal|title=“東胡民族考”|author=[[白鳥庫吉|白鳥庫吉]]|journal=《史學雜誌》|year=1911—1912年|volume=第21—24期|location=日本东京|language=ja}}}}。前者提供了一些鲜卑语的语音信息，后者相当于一套鲜卑语—汉语对义词汇表。例如更改[[拓跋姓|拓跋]]为[[元姓|元]]、[[獨孤姓|獨孤]]为[[刘姓|刘]]、[[贺赖姓|贺赖]]为[[贺姓|贺]]、[[步六孤姓|步六孤]]为[[陆姓|陆]]、[[吐奚姓|吐奚]]为[[古姓|古]]{{参|1={{Cite book|title=《北朝胡姓考》|author=[[姚薇元|姚薇元]]|publisher=科学出版社|year=1958年|location=中国北京|pages=第60页|language=zh-hant}}}}，更改[[可汗|可汗]]为[[皇帝|皇帝]]、[[可孙|可孙]]为[[皇后|皇后]]、莫贺为父、麽敦为母及阿干为兄等等。[[白鸟库吉|白鸟库吉]]与[[方壮猷|方壮猷]]等现代学者首先正确地认识到这些词语原名与汉名为鲜卑语语音语义对应汉语语音语义的关系，并且进行了对比研究{{参|1={{Cite journal|title=“鲜卑语言解读述论”|author=聂鸿音|journal=《民族研究》|year=2001年|volume=第1期|location=中国北京|pages=第63—70页|language=zh-cn}}}}。例如將“阿干”译为“兄”，可以对照[[蒙古语族|蒙古]]、[[满—通古斯语族|满—通古斯]]和[[突厥语族|突厥语族]]的多种语言之语音，/*aka(n)~*aga(n)~*axa(n)~*aki(n)/。{{参|1={{Cite journal|title=“鲜卑语言考”|author=[[方壮猷|方壮猷]]|journal=《燕京学报》|volume=第8期|location=中国北平|pages=第1429—1468页|language=zh-hant|date=1930年8月}}}}

根据《[[后汉书|后汉书]]》、《[[晋书|晋书]]》、《[[北史|北史]]》、《[[魏书|魏书]]》、《[[周书|周书]]》、《[[北齐书|北齐书]]》、《[[南史|南史]]》、《[[宋书|宋书]]》、《[[南齐书|南齐书]]》、《[[梁书|梁书]]》、《[[陈书|陈书]]》及《[[隋书|隋书]]》等史书的记载，至少在官方，曾经有许多使用鲜卑语编写的书籍，惟無一页存留至今，至今亦無发现使用鲜卑语刻写的石碑、铜线及铭文等[[文物|文物]]的痕迹，甚至不清楚鲜卑语到底為使用何種文字来书写。目前，唯一研究鲜卑语的途径就是参考中国史书中使用汉字注音的鲜卑族人名和地名。

== 归属 ==
鲜卑语一般被归入[[蒙古语族|蒙古语族]]{{参|1={{Cite journal|title=“《南齐书》中部分拓跋鲜卑语名词的复原考释”|author=乌其拉图|journal=《内蒙古社会科学》|volume=第23卷第6期|location=中国内蒙古呼和浩特|pages=第103页|language=zh-cn|date=2002年11月}}}}。史书记载鲜卑族为[[东胡|东胡]]遗民，又前载东胡语近似[[乌桓|乌桓]]语、乌桓语近似鲜卑语，后载[[库莫奚|库莫奚]]语近似[[吐谷浑|吐谷浑]]语、吐谷浑语近似[[契丹语|契丹语]]、契丹语近似[[蒙古语|蒙古语]]。

== 划分 ==
出于目前对鲜卑语的了解过少，语言学家多借用历史学家对鲜卑族内部的部落划分，直接用于指称鲜卑语内部的各种“语言”或“方言”。有柔然鲜卑语（柔然语）、乌桓鲜卑语（乌桓语）、{{link-en|拓跋鲜卑语|Tuoba language}}、慕容鲜卑语、吐谷浑鲜卑语（[[吐谷浑语|吐谷浑语]]）、室韦鲜卑语（[[室韦语|室韦语]]）、[[契丹语|契丹语]]、[[原蒙古语|蒙兀语]]（[[原蒙古语|原蒙古语]]）等。

== 文字 ==
