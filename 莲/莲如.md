'''莲如'''（{{lang-ja|'''蓮如'''}}，{{bd|1415年|4月4日|1499年|5月5日|catIdx= L莲}}）古日本[[室町时代|室町时代]][[净土真宗|净土真宗]]僧人，[[本愿寺|本愿寺]]中兴之祖，幼名'''布袋丸、幸亭丸'''讳称'''兼寿'''，尊称'''莲如上人'''，谥号'''慧灯大师'''，呼号'''本愿寺莲如'''。

== 生平 ==
[[File:Tyoufukuj3.jpg|thumb]]

1415年4月13日（[[应永|应永]]22年2月25日）<ref name="guregorioreki">[[公历|公历]]换算。本願寺派では、グレゴリオ暦に換算した生没年を用いる。</ref>，莲如出生于京都[[東山_(京都府)|东山]]，他是第七世存如的长子，母亲身份不明，幼名“布袋丸”。<ref>{{cite book |language = ja|author=高山秀夫 |authorlink = |title=《江戸から東京へ 物語でつづる部落の歴史》|year=1977年 |publisher=文理閣 |location= |pages=6页 |isbn= }}</ref>

1431年（[[永享|永享]]3年），17岁的莲如在青莲院剃度，名“中纳言兼寿”。

1442年（[[嘉吉|嘉吉]]2年），莲如开始在寺院中宣扬[[净土真宗|净土真宗]]的教义，1499年（[[明应|明应]]8年）2月20日，莲如圆寂。

== 祭祀 ==
[[淨土真宗|淨土真宗]][[西本願寺派|西本願寺派]][[佛壇|佛壇]]中央供奉 '''[[阿彌陀佛|阿彌陀如來]]'''、右側「[[親鸞|親鸞聖人]]」、左側為「[[莲如|蓮如聖人]]」。[[淨土真宗|淨土真宗]][[大谷派|大谷派]]則為中央 阿彌陀如來，右側十字名號:「歸命盡十方無礙光如來 (帰命尽十方無碍光如来)」，左側九字名號「南無不可思議光如來」。[[真宗高田派|真宗高田派]]中央 阿彌陀如來，右側懸掛十字名號與九字名號，左側供俸親鸞聖人。
