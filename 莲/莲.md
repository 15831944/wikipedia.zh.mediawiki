'''莲'''（[[学名|学名]]：''{{lang|la|Nelumbo nucifera}}''）是[[莲科|莲科]][[莲属|莲属]][[多年生|多年生]][[草本|草本]][[水生植物|出水植物]]，又称'''荷花'''、'''蓮花'''、'''荷'''，古称'''[[芙蓉|芙蓉]]'''、'''菡萏'''、'''芙蕖'''。荷花原产于[[中国|中国]]，在水中生活，通常被种植于[[池塘|池塘]]之中。花一般盛开于[[夏季|夏季]]，荷花也因而成为夏日之风物。

荷花的根茎种植在池塘或河流底部的淤泥上，而荷叶挺出水面。故說蓮花是出淤泥而不染。在伸出水面几厘米的花茎上长着花朵單朵，少數花莖上長著雙朵或更多。荷花一般长到150厘米高，横向扩展到3米。荷叶最大可达直径60厘米。引人人注目的莲花最大直径可达20厘米。

荷花有超过800个[[栽培品种|栽培品种]]<ref name=biods.2015157>{{cite journal|last1=Xue|first1=Jianhua|last2=Jiang|first2=Li|last3=Ma|first3=Xiaolin|last4=Bing|first4=Yanhong|last5=Zhao|first5=Sichen|last6=Ma|first6=Keping|title=Identification of lotus cultivars using DNA fingerprinting|journal=Biodiversity Science|date=2016|volume=24|issue=1|pages=3–11|doi=10.17520/biods.2015157}}</ref>，花色从雪白、黄色到淡红色及深黃色和深紅色，其外還有分灑錦等等的花色。

莲花可以用种子或根茎繁殖。特别的是，其种子莲子可以存活上千年。有科学家培育了一些有千年历史的莲子，繁殖出来的荷花依然生机盎然

</ref>，有超过3000年历史，但是过于珍贵，未进行栽培试验。<ref name="荷藕文化的中心—莲子">[http://blog.sina.com.cn/s/blog_5f1529bc0100dsod.html 荷藕文化的中心—莲子]，在河南省郑州市北部大河村发掘的“仰韶文化”房基遗址，发现室内台面上有炭化粮食和两粒莲子，经测定，距今有五千年的历史。</ref>

==特徵==
水生。根長在河底或池溏的泥土中，水平伸展約三公尺。葉和花挺出水面，高約 150 公分。葉圓形，直徑約 60 公分。花通常比葉略高，直徑可達20公分，花瓣多數。


=== 和睡蓮的區別 ===
睡蓮和莲二者的外观虽然相似，但亲缘关系甚远，两者可从其葉子的生長位置和葉子的形狀区分开来。
睡蓮不是蓮花的一个亚种，很多人因为误读，而把睡蓮叫做荷花，兩者不是同一種植物。<ref>[http://kids.yam.com/edu9/natural/general/material/20030716163534_1.php 九年一貫百寶箱]</ref>1981年的[[克朗奎斯特分類法|克朗奎斯特分類法]]單獨分出一個[[蓮科|蓮科]]，根據其外觀放入睡蓮目，1998年根據基因親緣關係分類的[[APG_分類法|APG 分類法]]認為本科應放入[[山龍眼目|山龍眼目]]。

蓮花喜好高熱高濕的環境，在溫帶分布的品種於冬天時的泥土上常綠部位會枯萎，藕莖會膨大。
而熱帶的品種多為常綠部位不枯萎，藕莖膨大不明顯。

== 種植 ==
荷花依賴走莖繁殖，如果走莖生長受到阻礙，開花性將受到影響，植株也會長的相對嬌小。
種植荷花的盆器依種類有別，迷你品種15厘米寬、10厘米深的盆即可種植，小型品種則是25厘米寬、15厘米深的盆，中型品種需40厘米寬、30厘米深的盆，大型品種需要田植，需60厘米以上宽的盆，90厘米以上的深度方可達到最佳生長，但有些品種可隨著盆子改變大小，有些品種在過小的盆子會導致花瓣數減少，葉子縮小且無波浪的邊緣，或者是不開花，結實量減少。

種植時間是以春季最佳，夏季雖可移植，但須注意失水，移植時最好以袋子套住葉子，並種植在散光處，等葉子數量增加後，再移到全日照處，此外必須注意水溫，夏季移植的常常使水溫高於30℃，據說這會導致開花量減少{{fact}}，等到葉子數量足以遮簷水面時，氣候已轉涼，日照減少。

== 用途 ==
===食用===
[[File:JaRenkonLotus11R.jpg|thumb]]
[[File:Nelumbo_nucifera_ハスの実を売る少年_DSCF2273.jpg|thumb]]]]
{{see also|莲子}}
莲的地下茎是莲[[藕|藕]]，叶是荷叶，果实是莲蓬，种子为[[莲子|莲子]]。莲的花、种子、嫩叶和[[根茎|根茎]]都可以食用；花瓣、荷叶和藕可以生吃。莲的不同部分也用在传统东亚的草药中。莲子、莲子芯（[[胚芽|胚芽]]）、莲叶、莲房、雄蕊和蓮梗都可以入药。

在亚洲，莲的花被用于[[插花|插花]]点缀，或将花瓣与水同煮或泡水作为[[凉茶|凉茶]]饮用。在[[济南|济南]]，莲花的花瓣被当地人油炸后食用。[[老舍|老舍]]的作品中即描述过此种风俗。

莲的叶子同样可以煮水喝，大的莲叶还可用于包装食物。蓮葉中含有[[黃酮|黃酮]]類物質，能夠促進胰島細胞功能的恢復，間接達到調節[[血糖|血糖]]的作用<ref>{{cite book |author = 向紅丁 |editor = 林巧玲 |others = 莊馨云,文字校對.|title = 不挨餓 不囤脂 糖尿病救命飲食運動法 | edition = 第二版 |date = 2015年9月 |publisher = 繪虹企業股份 |location = 新北市 |language = 中文 |isbn = 978-986-87392-1-5 |pages = 第125頁 }}</ref>。蓮藕是爆炒或煲湯的食材，是蓮的部位中最常為人吃的部份。

莲花的[[雄蕊|雄蕊]]可以被晒干制作成[[草本茶|草本茶]]。

莲花的种子[[莲子|莲子]]有非常广泛的用途，可以生吃，又常常用于烹饪，莲子汤、莲子[[糖水|糖水]]具有清热的效用。莲子煮软或压至极烂后可以作成[[莲蓉|莲蓉]]，被作为[[月饼|月饼]]、[[年糕|年糕]]和[[大福|大福饼]]等的馅料，并有蓮蓉包、蓮蓉糕等美食。莲子的芯，也就是其胚芽，是绿色的，味苦，食用莲子前要将其取出；同时，莲子芯也可以泡水饮用。

== 文化意义 ==
[[File:Lotus_jdp.jpg|thumb]]
[[File:內湖郭氏古宅蓮花池.jpg|thumb]]蓮花池。]]
=== 在中华文化 ===
中国古代的文学家认为荷花圣洁高雅，所以在古代诗词歌赋，经常会有歌颂荷花的篇章。比如[[曹植|曹植]]在《芙蓉赋》中说“览百卉之英茂，无斯华之独灵”，《[[群芳谱|群芳谱]]》中说，“凡物先华而后实，独此华实齐生。百节疏通，万窍玲珑，亭亭物华，出于淤泥而不染，花中之君子也。”[[周敦颐|周敦颐]]的《[[爱莲说|爱莲说]]》写到“予独爱莲之出淤泥而不染，濯清涟而不妖，中通外直，不蔓不枝；香远益清，亭亭净植；可远观而不可亵玩焉”，是对荷花最出名的赞誉。[[許顗|許顗]]《彥周詩話》稱：「世間花卉，無踰蓮花者，蓋諸花皆薰風暖日，獨蓮花得意於水月清渟逸香，雖荷葉無花時亦自香也。」出于对荷花的喜爱，文人给荷花取了很多不同的名字。依照外形，起名荷花，取“莲茎上负荷叶，叶上负荷花”（《[[本草纲目|本草纲目]]》）之意；另外还有，“未发为菡萏，已发为芙蓉”（《[[说文解字|说文解字]]》）。李商隱 詠荷詩:世間花葉不相倫,花入金盆時作塵。惟有綠荷紅菡萏,卷舒開合任天真。此花此葉長相映,翠減紅衰愁煞人。
依照亲水的生长习性，取名水芝、水花、水芸、水旦、水目、泽芝等。依照圣洁高雅的气质，取名君子花、凌波仙子、水宫仙子、玉环等。荷花的茎一般称为藕，但也有玉节、玉玲珑、玉臂龙等名称。荷花的果实莲子又称玉蛹、湖目。

2008年3月14日，[[香港郵政|香港郵政]]推出主題為「香港花卉」的[[郵票|特別郵票]]，其中3元的郵票圖案為荷花<ref>

===印度宗教===
蓮花在[[印度教|印度教]]中是重要的宗教符號，稱為Padma，意涵包括美麗、端莊、潔淨。兩腳盤起的[[跏趺坐|跏趺坐]]又稱為蓮花坐。許多印度教的神祀會被表現成坐在一朵蓮花上面。這些宗教符號也傳到[[佛教|佛教]]和[[耆那教|耆那教]]。

印度教中常和蓮花一起出現的神包括[[毗濕奴|毗濕奴]]、[[梵天|梵天]]、[[俱毗羅|俱毗羅]]、[[拉克什米|拉克什米]]、以及[[薩拉斯瓦蒂|薩拉斯瓦蒂]]。

佛教認為蓮花露在淤泥之上、葉子不沾水，表現的是佛教追求的不受俗世的慾望沾染。傳說中釋迦牟尼出生時走了七步，每一步都生出一朵蓮花。《妙法蓮華經》以蓮花象徵佛法的純淨。蓮花是[[佛門八寶|佛門八寶]]之一。相傳[[藏傳佛教|藏傳佛教]]的[[蓮花生|蓮花生]]在出生時是坐在一朵浮在水上的蓮花。

[[耆那教|耆那教]]的創教祖師們常被刻畫為坐在蓮花上。巴特摩巴羅波（Padmaprabha 蓮華光）的代表符號是紅色的蓮花。

==代表物==
荷花是[[花蓮縣|花蓮縣]]的縣花、[[济南市|济南市]]的市花、又為[[澳門市花|澳門的區花]]；荷花还是[[印度|印度]]和[[越南|越南]]的[[国花|国花]]。

==莲花效应==
[[File:Lotus3.jpg|thumb]]
{{Main|莲花效应}}

“蓮花效應”是指蓮葉表面具有[[疏水性#超疏水性|超疏水性]]以及[[自潔|自潔]]的特性。在東方文化裡，蓮花是純淨的象徵。雖然，蓮花喜歡生長在泥濘的溼地，但其葉子和花仍保持乾淨，這就是自潔的效果。[[植物學家|植物學家]]研究蓮葉表面發現它們有一個自然潔淨的機制。

蓮葉的微觀結構和表面化學意味著不會被水弄濕；水滴在葉片表面就如[[水银|水银]]一般，並且可以帶走污泥、小昆蟲及污染物。而且，水滴在[[芋头|芋头]]葉子亦有相似的現象。
