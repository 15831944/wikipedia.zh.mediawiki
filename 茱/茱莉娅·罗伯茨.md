'''茱莉亞·費歐娜·羅勃茲'''（{{lang-en|Julia Fiona Roberts}}，{{bd|1967年|10月28日|catIdx=Roberts}}），生于[[美國|美國]][[喬治亞州|喬治亞州]][[士麥那_(佐治亚州)|士麦那]]的女[[演员|演员]]。2001年以《[[永不妥協|永不妥協]]》獲得[[第73屆奧斯卡金像獎|第73届奧斯卡金像獎最佳女主角奖]]。

== 早年生活與家庭 ==
茱莉亞·羅勃茲出生於一個擁有[[爱尔兰|爱尔兰]]血统的演藝家庭，父母沃尔特·佳迪·罗伯茨與贝蒂·卢·贝蒂努斯都是演員，因此三個子女也相繼踏上演藝之路。茱莉亞小時候跟哥哥[[艾力克·羅勃茲|艾力克·羅勃茲]]一樣曾患有[[口吃|口吃]]。四歲時，父母離異。高中畢業後便搬到[[紐約|紐約]]和姐姐麗莎·羅伯茨一同實踐追星夢，後來因為他哥哥的幫助使她得到一個演出機會，但兩人關係並不親密。

茱莉亞羅勃茲畢業於[[亞特蘭大大學|亞特蘭大大學]][[新聞學|新聞學]]系。

== 影视事业 ==
1988年，茱莉亞·羅勃茲在《[[現代灰姑娘|現代灰姑娘]]》（{{lang|en|''Mystic Pizza''}}）中的演出吸引了觀眾的目光。<br />
1989年她在《[[鋼木蘭|鋼木蘭]]》（{{lang|en|''Steel Magnolias''}}）中扮演一位與[[糖尿病|糖尿病]]抗爭的新娘，出色的演技為她帶來了第一個[[奧斯卡金像獎|奥斯卡]]提名（[[奥斯卡最佳女配角|最佳女配角]]）。<br />
1990年《[[麻雀變鳳凰|麻雀變鳳凰]]》（{{lang|en|''Pretty Woman''}}）一片更是使她闻名全世界，也带来了第二个奥斯卡提名（最佳女主角）。<br />
接下来的《[[与敌共眠|与敌共眠]]》（{{lang|en|''Sleeping with the Enemy''}}）为她带来了巨大的票房成功，在該片中她扮演一位逃离丈夫重新开始生活的女子。她在1991年出演[[斯皮尔伯格|斯皮尔伯格]]导演的《[[虎克船長|虎克船長]]》（{{lang|en|''Hook}}）后，有两年的时间几乎没在-{zh-hans:屏幕; zh-hant:螢幕;}-上露面。<br />
1993年初，《[[人物|人物]]》杂志封面故事就提问到：“茱莉亞·羅勃茲到底出什么事了？”

[[File:Julia_Roberts_2011_Shankbone_3.JPG|thumb]]《{{lang|en|Jesus Henry Christ}}》的首映會。]]
同年晚些时候，她与[[丹泽尔·华盛顿|丹泽尔·华盛顿]]联袂出演《[[絕對機密|絕對機密]]》（{{lang|en|''The Pelican Brief''}}），取得了很大成功。但接下来几年内，她参与的一系列电影都既不叫好又不叫座。1997年她终于时来运转，出演了《[[新娘不是我|新娘不是我]]》（{{lang|en|''My Best Friend's Wedding''}}）。这部[[浪漫|浪漫]][[喜剧|喜剧]]带来了极高的票房收入，奠定了罗勃茲票房灵药的名誉。

2001年她在《[[永不妥协|永不妥协]]》（{{lang|en|''Erin Brockovich''}}）中精湛的演出受到评论家广泛好评，终于为她带来了[[奥斯卡最佳女主角奖|奥斯卡最佳女主角奖]]。在这部真人真事改编的电影中，她扮演了一位没有受过正式[[法律|法律]]教育但坚-{}-强果敢的女性，帮助小镇居民起诉[[能源|能源]]巨头“太平洋煤气电力公司”的[[环境污染|环境污染]]罪行。

== 私生活 ==
茱莉亞·羅勃茲的私生活一直为娱乐媒体穷追不舍，后来这种现象便反映在她1999年的电影《[[新娘百分百|新娘百分百]]》（{{lang|en|''Notting Hill''}}）中：她扮演一位[[好莱坞|好莱坞]]著名女演员{{lang|en|Anna Scott}}，爱上了一个在[[伦敦|伦敦]][[诺丁山|诺丁山]]開书店的普通人{{lang|en|William Thacker}}（[[休·葛蘭|休·葛蘭]]扮演）。据说她的这个角色就是茱莉亞的真实写照，以至于拍摄时有一场戏问起女主角片酬多少，茱莉亞脱口而出“1500万[[美元|美元]]”，而这个数字正好就是她出演这部电影的酬劳。

1991年茱莉亞羅勃茲与演员[[基夫·修打蘭|基夫·修打蘭]]订婚，但婚礼前几天取消婚约。后来她与[[鄉村音樂|乡村]]歌手{{Link-en|Lyle Lovett|Lyle Lovett}}私奔，两人1995年离婚。

2000年茱莉亞羅勃茲拍摄《[[危險情人|危險情人]]》（{{lang|en|''The Mexican''}}）时，在片场认识了[[摄影师|摄影师]]丹尼·莫達（{{lang|en|Danny Moder}}）。2002年7月4日，两人在茱莉亞·羅勃茲於[[新墨西哥州|新墨西哥州]]{{lang|en|Taos}}的庄园结婚。

2004年拍《[[瞞天過海2：長驅直入|瞞天過海2：長驅直入]]》（{{lang|en|''Ocean's Twelve''}}）時，她發現她懷上了一對雙胞胎，所以導演和編劇也很巧妙的把這個事件編入故事之中，讓角色變得更有趣。<br />
2004年11月18日茱莉亞产下[[龙凤胎|龙凤胎]]，女兒夏素·柏特西亞（{{lang|en|Hazel Patricia}}），兒子賓尼斯·華特（{{lang|en|Phinnaeus Walter}}）。<br />
2007年6月18日产下另一個儿子，亨利·丹尼爾（{{lang|en|Henry Daniel}}）。

== 評選 ==
在{{引书号转换|人物}}雜誌公布2005年「五十名全世界最漂亮人物」排行榜中，茱莉亞羅勃茲獲選為封面人物，創下三度榮登這項排行榜封面的最高紀錄。{{引书号转换|人物}}雜誌主編丹妮兒·杜賓今天上午接受媒體訪問時表示，茱莉亞羅勃茲今年獲選為「五十名全世界最漂亮人物」特輯封面人物的最主要原因是，她產下龍鳳胎雙胞胎升格當媽媽之後，「臉龐散發出一股自然的母性魅力」，使得原本就漂亮的這位女星更加美麗動人。這是三十七歲的茱莉亞羅勃茲第九度入選「五十名全世界最漂亮人物」排行榜，與今年同樣也榜上有名的哈莉·貝瑞打成平手。羅勃茲這次也是第三度登上這份{{引书号转换|人物}}雜誌排行榜的封面，創下無人能比的紀錄。

2010年最新一期{{引书号转换|人物}}雜誌公布「世界最漂亮人物」排行榜，茱莉亞羅勃茲再次獲選為封面人物。打破了自己早前的記錄，第四次登上了該排行榜的首位（這已經是她入圍第12次），亦曾被评为[[好莱坞|好莱坞]]最有权势的人物；在《[[福布斯|福布斯]]》杂志的“百人权势榜”中处于第12位，是好莱坞片酬最高的女星，接拍电影《[[蒙娜麗莎的微笑|蒙娜麗莎的微笑]]》（{{lang|en|''Mona Lisa Smile''}}）的片酬高达2500万美元。
