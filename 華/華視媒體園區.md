[[File:CTS_Media_Park_light_box_near_of_CTS_TV_Production_Building_20100607.jpg|thumb]]
'''華視媒體園區'''（CTS Media Park）是[[中華電視公司|中華電視公司]]的總部所在地，位於[[台北市|台北市]][[大安區|大安區]][[光復南路|光復南路]]，包括了多棟[[大樓|大樓]]。2009年4月6日，華視園區正式定名'''華視媒體園區'''並舉行揭牌典禮，佔地3萬坪，有15個[[攝影棚|攝影棚]]。<ref>黃慧敏 台北2009年4月6日電，[http://www.cna.com.tw/SearchNews/doDetail.aspx?id=200904060138 〈華視媒體園區揭牌 14家影音產業進駐〉] {{webarchive|url=https://web.archive.org/web/20140512215949/http://www.cna.com.tw/SearchNews/doDetail.aspx?id=200904060138 |date=2014-05-12 }}，[[中央通訊社|中央通訊社]]，2009年4月6日</ref>當時已進駐華視媒體園區者，除了華視關係企業之外，尚有[[公共電視文化事業基金會|公共電視文化事業基金會]]、[[原住民族電視台|原住民族電視台]]、[[客家電視台|客家電視台]]、[[台灣宏觀電視|台灣宏觀電視]]、[[民間全民電視公司|民間全民電視公司]]、[[鳳凰衛視|鳳凰衛視]]、[[星空傳媒|星空傳媒]]、[[FOX體育台|FOX體育台]]、[[國家地理頻道|國家地理頻道]]、[[風尚數位科技|風尚數位科技]]（Fashion Guide）、張乃仁音樂工作室等<ref>[http://www.cts.com.tw/Default.aspx?ch=even&cp=Detail&sno=2256 〈「華視媒體園區」正式揭牌營運 近50位媒體先進到場見證台灣影音製播新頁〉] {{webarchive|url=https://web.archive.org/web/20090410151326/http://www.cts.com.tw/Default.aspx?ch=even&cp=Detail&sno=2256 |date=2009-04-10 }}，《華視全球資訊網》，2009年4月</ref>。

2010年4月30日，華視舉行華視媒體園區1週年[[記者會|記者會]]，[[華視新聞|華視新聞]]主播[[蘇逸洪|蘇逸洪]]與藝人[[劉真_(藝人)|劉真]]主持，現場展示華視砸下約[[新台幣|新台幣]]2千萬元打造的「虛實合一」[[3D|3D]]播報系統；華視[[總經理|總經理]][[陳正然|陳正然]]表示，華視媒體園區成立1年，第1階段完成華視資源讓20家媒體夥伴互通共享，第2階段已邀請[[宏碁|宏碁]]電子化資訊管理中心協助建置「雲端運算與數位資料庫」<ref>{{cite news|author=張哲鳴|title=華視砸2千萬搞3D播報 劉真糗吃螺絲|url=http://ent.appledaily.com.tw/enews/article/entertainment/20100501/32478841|publisher=[[台灣蘋果日報|台灣蘋果日報]]|date=2010-05-01|accessdate=2016-06-04|language=zh-tw}}</ref>。

2017年，[[新北市|新北市]][[林口區|林口區]]「民視林口數位媒體總部」落成，民視撤租華視媒體園區攝影棚。

== 攝影大樓 ==
[[File:CTS_Studio_Building_face_20100607.jpg|thumb]]
「攝影大樓」（Studio Building）<ref>《華視二十年》，第33頁。</ref>在1971年2月10日開工，同年啟用，是華視第一代總部，最富有[[中國傳統建築|中國傳統建築]]特色。使用[[鋼筋混凝土|鋼筋混凝土]]結構，共有地下一層與地上四層，高度16.6公尺，[[總樓地板面積|總樓地板面積]]13556[[平方公尺|平方公尺]]。建築承包單位是[[行政院國軍退除役官兵輔導委員會|行政院國軍退除役官兵輔導委員會]][[榮民工程事業管理處|榮民工程事業管理處]]，建築時間未滿半年<ref>[http://nrch.cca.gov.tw/ccahome/photo/photo_meta.jsp?xml_id=0005830442 1970年代的攝影大樓]{{dead link|date=十月 2017 |bot=InternetArchiveBot }}</ref>。

「電視製作大樓」啟用之後，攝影大樓閒置多年，2000年代初期曾被[[東森電視|東森電視]]租用，東森電視退租後又荒廢，2010年時華視已收回自用。

== 文化大樓 ==
「文化大樓」（Cultural Building）<ref name="《華視二十年》，第35頁。">《華視二十年》，第35頁。</ref>，建築風格與攝影大樓相同，供[[華視訓練中心|華視訓練中心]]使用，1991年1月11日被拆除。

== 華視光復大樓 ==
[[File:CTS_Kuang-Fu_Building_20061223.jpg|thumb]]
原名「華視大樓」，在1983年9月1日啟用，是華視第二代總部，正門面臨光復南路。共有地上十四層與地下二層，總面積達一萬一百餘坪。外觀方正，早期以棕色為主要顏色，後改以白色為主要顏色。華視總部搬至「電視製作大樓」之後，華視大樓改名為「華視光復大樓」，現今作為商業大樓並租給外界使用。

== 電視製作大樓 ==
[[File:CTS_TV_Production_Building_20180106.jpg|thumb]]
「電視製作大樓」（TV Production Building）<ref>《華視二十年》，第34頁。</ref>是華視第三代總部，1989年2月華視邀請三位建築師-{[[沈祖海|沈祖海]]}-、[[高而潘|高而潘]]、[[丁達民|丁達民]]參觀訪問[[日本放送協會|日本放送協會]]、[[東京放送|東京放送]]、[[韓國放送公社|韓國放送公社]]總部後設計，1989年4月公開辦理比圖，經邀請有關專家學者評選後委託丁達民設計監造，[[中興工程顧問社|中興工程顧問社]]擔任工程顧問。<ref>《華視二十年》，第24頁。</ref>1990年7月，細部設計規劃完成。<ref>《華視二十年》，第22頁。</ref>1990年11月13日，[[毅成建設|毅成建設]]取得土木建築工程之承包權。1990年11月21日上午，華視副總經理[[陳祖耀|陳祖耀]]與毅成建設王總經理在華視大樓三樓貴賓室簽約，華視法律顧問[[林憲同|林憲同]][[律師|律師]]擔任見證人。<ref>《華視二十年》，第28與289頁。</ref>1990年11月29日，攝影大樓旁停車場四周正式圍欄，為電視製作大樓工程施工預作準備。<ref>《華視二十年》，第289頁。</ref>1991年1月11日10時拆除文化大樓後動土，動土典禮主持者為華視董事長[[易勁秋|易勁秋]]、華視總經理[[武士嵩|武士嵩]]與華視監察人[[王永在|王永在]]。<ref>廣播電視年鑑編纂委員會，《中華民國廣播電視年鑑：79～84》，中華民國廣播電視事業協會發行，《廣播與電視》雜誌社1996年5月20日-{出}-版，第208頁。</ref>1995年啟用，斥資新台幣30多億元，使用[[鋼骨結構|鋼骨結構]]（Steel Construction）、[[金字塔|金字塔]]造型與[[玻璃帷幕|玻璃帷幕]]，共有地上十四層與地下五層，總面積達12917坪，總樓地板面積42672平方公尺，地下層開挖深度達23.15公尺。內部設施包括全自動化UHF/VHF頻道播映系統與專業[[攝影棚|攝影棚]]9座，大樓頂端設有一個[[橢圓形|橢圓形]]的[[直昇機|直昇機]][[停機坪|停機坪]]。<ref>[http://nrch.cca.gov.tw/ccahome/photo/photo_meta.jsp?xml_id=0006335281 電視製作大樓照片集]{{dead link|date=十月 2017 |bot=InternetArchiveBot }}</ref>2005年12月8日，華視啟用電視製作大樓中庭[[噴水池|噴水池]]。<ref>中華民國電視學會無線電視年鑑編纂委員 編纂，《中華民國無線電視年鑑第十四輯：民國九十三年至九十四年》，中華民國電視學會2006年6月30日-{出}-版，第159頁。</ref>

== 文教大樓 ==
[[File:CTS_UHF_Studio_&_Administrative_Building_-_front_&_right.jpg|thumb]]
[[File:CTS_Culture_&_Education_Building_entrance_20190113.jpg|thumb]]
原名為「UHF攝影棚及行政大樓」（UHF Studio & Administrative Building）<ref name="《華視二十年》，第35頁。"/>：是華視各類教學節目的重鎮，也是華視文教基金會與華視訓練中心所在地，部分空間於2000年代某年成為[[星空傳媒|星空傳媒]]台灣總部所在地，2011年星空傳媒台灣總部遷至台北市[[內湖區|內湖區]][[堤頂大道|堤頂大道]]。2013年時，現在名稱改為「文教大樓」。

== 改建計畫 ==
2015年，為改善財務狀況，華視決定：拆除攝影大樓，修建高2至4層的「中央綠廣場」；拆除文教大樓，修建高70米、地下3層、地上12層的「新華視大樓」，2019年啟用；華視光復大樓出租給物業公司，整修為「e-Tower」，2016年7月啟用；電視製作大樓改建為「New Media Hotel」，2020年啟用。這一計畫在2015年6月25日在華視董事會通過<ref>[http://news.ltn.com.tw/news/life/paper/892477 華視大樓變飯店 董事會突襲成功]</ref>，同年7月16日也在[[台灣公共廣播電視集團|台灣公共廣播電視集團]]董事會通過<ref>[http://www.chinatimes.com/newspapers/20150717000141-260204 華視園區資產活化 打造小型文創基地]</ref>。
