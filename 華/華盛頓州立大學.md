'''華盛頓州立大學'''（[[英文|英文]]：{{lang|en|Washington State University}}、簡稱{{lang|en|WSU}}）是[[美国|美国]][[华盛顿州|华盛顿州]]资助办理的一所研究型大学，也是該州的[[贈地大學|贈地大學]]。它的主校位於華盛頓州東部的[[普爾曼_(華盛頓州)|普爾曼]]，而在[[斯波坎_(華盛頓州)|斯波坎]]、[[三城_(華盛頓州)|三城]]和[[溫哥華_(華盛頓州)|温哥华]]也設有分校，此外在全华盛顿州有它的学习中心，學生總人數23,428人<ref name="wsu_fact" />。華盛頓州立大學提供超過200樣不同範疇的教學及150種[[學科|專業學位]]<ref name="wsu_fact" />。

== 歷史 ==
華盛頓州立大學的前身是华盛顿州农业高校和科学学校，於1890年3月28日建立並在1892年1月13日正式開始授課。作為贈地大學，美國[[聯邦政府|聯邦政府]]及華盛頓州州政府於美國西北部撥出769[[平方公里|平方公里]]（19萬[[公頃|公頃]]）的土地給大學作為[[農業|農業]]及[[科學|科學]]的研究。1893年，在校長拜仁<ref>{{en icon}} Enoch A. Bryan，於1893年至1916年為校長。</ref>的推動下，大學開始提供文科的課程。翌年，大學在華盛頓州西部的[[皮阿拉普|皮阿拉普]]開始了其第一個的農業實驗，此後大學在華盛頓州的地位開始被確立，大學的辦公室和研究所也開始遍佈整個州。在1905年，該校改名為华盛顿州立學院及至1959年正式改名成為華盛頓州立大學。1989年華盛頓州立大學在在[[斯波坎_(華盛頓州)|斯波坎]]、[[三城|三城]]和[[溫哥華_(華盛頓州)|温哥华]]開設分校<ref>{{en icon}} [http://www.wsu.edu/WSU-history/index.html 華盛頓州立大學歷史] {{webarchive|url=https://web.archive.org/web/20071129191123/http://www.wsu.edu/WSU-history/index.html |date=2007-11-29 }} 華盛頓州立大學網站 於2007年11月11日造訪</ref>。

現時的竞争对手是鄰近的[[爱达荷大学|爱达荷大学]]和[[西雅圖华盛顿大学|西雅圖华盛顿大学]]。

== 校園 ==
[[File:Bryan_Hall.jpg|拜仁樓]]華盛頓州立大學位於[[普爾曼_(華盛頓州)|普爾曼]]的主校園佔地2.5平方公里（620公頃），依山而建，距離[[愛達荷州|愛達荷州]]的[[莫斯科_(愛達荷州)|莫斯科]]只有7[[英哩|英哩]]。校園的中心地帶為格蘭·泰利爾友誼林蔭大道，學生一般稱之為「大道」<ref>{{en icon}} Glenn Terrell Friendship Mall</ref>，這條大道是以於1967年至1985年在位的校長[[格蘭·泰利爾|格蘭·泰利爾]]命名，大學的[[圖書館|圖書館]]、[[學生會|學生會]]大樓及幾座教學大樓都位於大道的兩旁，而學校的活動也會在這裡舉行。

校園內的主要建築物都是要紅色的磚為外牆，這是因為在校園內的第一座建築物為紅磚的建築，自此以後所有的建築的仿傚類似的風格<ref>{{en icon}} [http://www.historylink.org/essays/output.cfm?file_id=5701 華盛頓州立大學歷史小資料(一)] HistoryLink.org 2004年5月22日 於2007年11月11日造訪</ref>。在19世紀末，校園建築物風格是當年流行的實用風格，充分反映建議作為教室及實驗室的用途，及至20世紀，建築物開始加入不同[[復古主義|復古主義]]的元素，例如大量的古典裝飾。到了現在，建築物的設計已經十分現代化，但仍然保留著紅磚外牆的傳統<ref>{{en icon}} [http://www.wsulibs.wsu.edu/holland/masc/masctour/wsu_buildings/exhibit1.htm 華盛頓州立大學的建築物] 華盛頓州立大學圖書館網站 2007年1月8日 於2007年11月11日造訪</ref>。

== 體育運動 ==
[[File:Martinstadium3.jpg|thumb]]
[[File:2008-1018-003-ButchTCougar.jpg|right]]: (Butch T. Cougar)]]
華盛頓州立大學是[[太平洋十大學聯會|太平洋十大學聯會]]的成員之一<ref>{{en icon}} [http://www.pac-10.org/genrel/washington-state-info.html 華盛頓州美洲獅] {{webarchive|url=https://web.archive.org/web/20071028090601/http://www.pac-10.org/genrel/washington-state-info.html |date=2007-10-28 }} 太平洋十大學聯會網站 於2007年11月11日造訪</ref>，體育運動的吉祥物是[[美洲獅|美洲獅]]，故華盛頓州立大學的學生一般被稱為「華盛頓州美洲獅」<ref>{{en icon}} {{link-en|华盛顿州美洲狮|Washington State Cougars|Washington State Cougars}}</ref>，而代表顏色是[[緋紅色|深紅色]]和[[灰色|灰色]]<ref>{{en icon}} [http://www.identity.wsu.edu/web/guidelines/colors.aspx 華盛頓州立大學顏色]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }} 華盛頓州立大學網站 於2007年11月11日造訪</ref>，參與的運動包括[[美式足球|美式足球]]、[[棒球|棒球]]、[[籃球|籃球]]、[[高爾夫球|高爾夫球]]、[[田徑|田徑]]和女子[[排球|排球]]等<ref>{{en icon}} [http://wsucougars.cstv.com/ 華盛頓州立大學體育運動網站] 於2007年11月11日造訪</ref>，其中以[[美式足球|美式足球]]最受注目。華盛頓州立大學在[[美式足球|美式足球]]上的最大對手是[[西雅圖華盛頓大學|西雅圖華盛頓大學]]的「華盛頓愛斯基摩犬」<ref>{{en icon}} {{link-en|华盛顿哈士奇|Washington Huskies|Washington Huskies}}</ref>，每年在[[感恩節|感恩節]]前的星期六就會舉行「[[蘋果盃|蘋果盃]]」一決高下<ref>{{en icon}} [http://www.coug.com/WSU%20UW%20Apple%20Cup%20History.htm 蘋果盃賽果] 華盛頓州立大學體育運動網站 於2007年11月11日造訪</ref>。
