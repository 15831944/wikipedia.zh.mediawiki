'''華金·「傑克」·加西亞'''（'''Joaquín "Jack" García'''，1952年出生），又名Jock和Jocko，是一位已退休的[[聯邦調查局|聯邦調查局]]（FBI）探員，他最出名是滲透[[紐約市|紐約市]]的{{link-en|甘比諾犯罪家族|Gambino crime family}}進行臥底工作。加西亞被認為是FBI的歷史上最成功和最資深的探員之一。<ref>{{Cite web |url = https://www.newstimes.com/news/article/Mob-made-retired-FBI-agent-shares-story-218881.php |title = Mob-made: retired FBI agent shares story |author = Melissa Bruen |date = 2009-05-24 |publisher = newstimes |language = en}}</ref>

== 背景 ==
加西亞在1952年的[[古巴|古巴]][[夏灣拿|夏灣拿]]出生。他的家人在他9歲時因為要逃離卡斯特羅的政權而逃往美國。他在紐約的[[布朗克斯|布朗克斯]]長大，在{{link-en|聖米高山學院|Mount Saint Michael Academy}}上學。他曾獲得欖球獎學金並在{{link-en|西德克薩斯州立大學|West Texas A&M University}}、{{link-en|威徹斯特社區學院|Westchester Community College}}和[[里士满大学|里士满大学]]打球，隨後他於1975年5月在里士满大学畢業。他的大尺寸身形（6尺4寸，390磅）有利於他的欖球和臥底生涯。加西亞畢業後很快就申請加入FBI，並最終於1980年5月宣誓就任特別探員。
