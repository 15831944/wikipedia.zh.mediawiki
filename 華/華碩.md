'''華碩電腦股份有限公司'''({{lang-en|'''ASUSTeK Computer Inc.'''}}，縮寫：'''ASUS'''，簡稱：'''華碩電腦'''、'''華碩'''；{{twse|2357}}、{{lse|ASKD}})是一家總部設於台灣[[臺北市|臺北市]][[北投區|北投區]]的跨國[[硬件|硬體]]及[[電子元件|電子元件]]公司。華碩其產品包括家用與商務用[[桌上型電腦|桌上型電腦]]、[[筆記型電腦|筆記型電腦]]、[[行動電話|行動電話]]、網絡設備、[[显示器|顯示器]]、[[數位投影機|投影機]]、[[主板|主機板]]、[[显示卡|顯示卡]]、[[光儲存|光儲存]]、[[多媒体|多媒體]]產品、[[外部设备|外圍設備]]、[[可穿戴設備|可穿戴設備]]、[[服务器|服務器]]、[[工作站|工作站]]和[[平板電腦|平板電腦]]。該公司也是一家[[代工生产|OEM]]製造商。2018年台灣品牌價值16.19億美元，在台灣名列第一。

華碩於臺灣證券交易所({{twse|2357}})與倫敦證券交易所({{lse|ASKD}})上市。

==沿革==
* 1989年，「弘碩電腦」成立。
* 1994年，弘碩電腦正式更名為「華碩電腦」。
* 1996年，華碩[[股票|股票]]於[[台灣證券交易所|台灣證券交易所]]正式掛牌上市。
* 2006年，根據[[iSuppli|iSuppli]]統計，華碩在[[ODM|ODM]]名列全球第1<ref>[http://www.appledaily.com.tw/appledaily/article/finance/20080612/30645364/ 華碩Eee PC年出貨 跨越500萬台高標]</ref>。出售轉投資網路IC公司[[鈺碩科技|鈺碩科技]](Attansic)給美國晶片大廠[[Atheros|Atheros]]，包含現金、股票、核心團隊和產品在內，總出售金額約新臺幣23.74億元<ref>[http://www.appledaily.com.tw/appledaily/article/finance/20061025/2979342/ 鈺碩嫁Atheros 華碩本季逾6億入袋]</ref>。
* 2007年，華碩宣佈在2008年1月1日將公司切割為「品牌」和「代工」兩個集團。「2007年十大臺灣國際品牌調查」，華碩獲選台灣十大國際品牌冠軍，品牌價值11.96億美元。
* 2008年，華碩切割為「品牌」和「代工」兩個集團，[[和碩聯合科技|和碩聯合科技]]負責電腦產品相關代工事業體；[[永碩聯合國際|永碩聯合國際]]負責[[機殼|機殼]]、[[寬頻|寬頻]]等非電腦產品相關代工事業體。成立「華碩文教基金會」。連續11年獲[[BusinessWeek|BusinessWeek]]評選為全球前100大科技公司，名列全球第9。
* 2010年，華碩召開股東臨時會，決議通過營業分割案，將代工業務分割讓與[[和碩聯合科技|和碩聯合科技]]，各自專注品牌及代工事業。
* 2011年，加入由[[Facebook|Facebook]]主導的[[開放運算計畫|開放運算計畫]](Open Compute Project)。<ref>{{Cite web |url=http://www.opencompute.org/blog/making-it-real-next-steps-for-the-open-compute-project/ |title=Making It Real: Next Steps for the Open Compute Project |access-date=2014-03-06 |archive-url=https://web.archive.org/web/20141018054943/http://www.opencompute.org/blog/making-it-real-next-steps-for-the-open-compute-project/ |archive-date=2014-10-18 |dead-url=yes }}</ref><ref>[http://3c.ltn.com.tw/3c/news.php?no=3688&type=9 華碩支持 Open Compute Project (OCP) 專案帶動開放雲端平台] {{webarchive|url=https://web.archive.org/web/20140606220636/http://3c.ltn.com.tw/3c/news.php?no=3688&type=9 |date=2014-06-06 }}</ref>
* 2012年，華碩雲端以「個人教育雲解決方案」為[[國立臺灣大學|國立臺灣大學]]建立「NTU Space」個人雲端服務<ref>[http://www.sogi.com.tw/mobile/articles/6192376-%E8%8F%AF%E7%A2%A9%E3%80%8C%E5%80%8B%E4%BA%BA%E6%95%99%E8%82%B2%E9%9B%B2%E8%A7%A3%E6%B1%BA%E6%96%B9%E6%A1%88%E3%80%8D%E5%85%A5%E4%B8%BB%E8%87%BA%E7%81%A3%E5%A4%A7%E5%AD%B8 華碩「個人教育雲解決方案」入主臺灣大學]</ref>。加入[[PlayStation_Mobile|PlayStation Mobile]]認證夥伴<ref>[http://www.cool3c.com/article/61786 華碩加入 PlayStation Mobile 認證夥伴]</ref>。
* 2013年，[[工研院|工研院]]與華碩及另外7家業者發起成立[[臺灣開放運算高峰會|臺灣開放運算高峰會]](Open Compute Project Taiwan)，為開放運算計畫在美國境外的第二個分會<ref>[http://www.ithome.com.tw/node/80535 臺灣雲協與Facebook合作成立OCP計劃臺灣分會]</ref>。
* 2017年，華碩執行長[[沈振來|沈振來]]宣佈華碩組織改革，將集團劃分為三大產品事業群，除原本的「電腦事業群」和「行動運算產品事業群」外，新設立「電競電腦事業群」。<ref>[https://udn.com/news/story/7240/2487586 華碩搶攻電競商機 新設「電競電腦事業群」]</ref>

* 2017年6月，華碩執行長沈振來對一級主管發信，預告當年6月底將展開改革與相關方向。


* 2018年4月，華碩發內部信件，接班計劃暫緩，董事會決議施崇棠續任董事長，被視為施崇棠接班人的策略長徐世昌，升任副董事長。


* 2018年8月，法說會預告公司將進行「轉型和傳承計劃」，訂出2018年第四季力拼手機業務由虧轉盈的目標記畫。


* 2018年9月，華碩因應組織調整，精簡電競部門，進行裁員。手機營運持續改善，目標第四季獲利。


* 2018年12月，執行長沈振來卸下原職自行創業，投入AIoT產業。華碩以雙執行長制，由現任副總裁許先越及胡書賓接任共同執行長。

== 事件==
===[[黄静诉华硕案|华硕门事件]]===
2006年2月9日，北京的黄静购买华硕笔记本电脑。在2006年2月15日，黄静向华硕提出索赔要求。在2006年4月14日，黄静因涉嫌敲诈勒索被海淀检察院批准逮捕，同年12月26日，检察院向黄静提出取保候审通知。次年11月9日检察院决定向黄静做出不起诉决定。在2008年6月5日黄静代理律师提出了国家赔偿，后检察院做出刑事赔偿申请通知书，2008年10月26日，黄起诉华硕，后又提出撤诉。<ref>{{Cite web|url=http://news.sohu.com/s2008/huashuosuopei/|title=女大学生遭遇“华硕门”-搜狐新闻|accessdate=2018-05-20|work=news.sohu.com}}</ref>
