'''過田子'''，或稱'''過田仔'''，是[[台灣|台灣]][[高雄市|高雄市]][[苓雅區|苓雅區]]內的一個區塊，也是組成該區的四個舊部落之一。

== 地名 ==
[[File:Leng-nga-liau_and_Koe-chhan-a_(1904_Taiwan_Houzu).png|thumb]]中的苓雅寮與過田子。]]
「過田子」一名，源自於該地區早期為[[苓雅寮|苓雅寮]]以東田野地帶<ref name="lingya">{{cite web|url=http://lingya.kcg.gov.tw/history.php|title=歷史沿革|publisher=苓雅區公所|accessdate=2011-01-09|deadurl=yes|archiveurl=https://web.archive.org/web/20110116002604/http://lingya.kcg.gov.tw/history.php|archivedate=2011-01-16}}</ref>。

== 地理位置 ==
過田子位在同屬苓雅區的苓雅寮及[[林德官|林德官]]之間，即[[成功路_(高雄市)|成功路]]與[[民權路_(高雄市)|民權路]]之間區塊<ref name="yanyun">{{cite book|title=苓雅寮煙雲|author=杜劍鋒|year=2005|publisher=高雄市文獻委員會|isbn=9860034699}}</ref>。其中包含15里：田西里、鼓中里、城北里、城西里、意誠里、人和里、城東里、仁政里、廣澤里、美田里、華堂里、和煦里、日中里、晴朗里、普照里，相鄰地區有[[大港埔|大港埔]]、[[前金區|前金]]、[[戲獅甲|戲獅甲]]等。

== 行政區劃沿革 ==
[[File:Lingya_trad_div.svg|thumb]]
過田子在[[臺灣清朝時期|清朝統治後期]]屬於[[大竹里|大竹里]]所轄；[[台灣日治時期|日本統治]]後，於1897年將過田子歸屬[[鳳山縣_(臺灣)|鳳山縣]]打狗辦務署，並且在1898年再改隸於台南縣鳳山辦務署大竹里。1909年，過田子劃為[[台南廳|台南廳]][[打狗支廳|打狗支廳]]過田子庄<ref>{{cite book|title=管轄要覽|year=1910|author=臺灣日日新報社}}</ref>。

1920年，過田子設為[[高雄州|高雄州]][[高雄街|高雄街]]下的的一個[[大字|大字]]<ref>{{cite book|title= 新舊對照管轄便覽|author=臺灣日日新報社|year=1921}}</ref>；而高雄街也在1924年改為「高雄市」。

== 歷史 ==
日治初期，[[打狗港|打狗港]]的築港工程使臨靠海岸的苓雅寮腹地漸失，人口因此逐漸向附近的過田子擴散。築港的同時，日本人也在過田子設立了移民村<ref name="yanyun"/>。

1933年時，過田子有人口291人<ref>{{cite book|title=昭和八年十二月三十一日 臺灣現住人口統計|year=1934|publisher=臺灣總督官房調查課}}</ref>。
