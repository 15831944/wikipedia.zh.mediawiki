'''過海隧道巴士608線'''，來往[[九龍城|九龍城]]（[[盛德街|盛德街]]）及[[西灣河|西灣河]]（[[嘉亨灣|嘉亨灣]]），途經[[東區海底隧道|東區海底隧道]]，由[[城巴|城巴]]營辦。本線的出現，成為城巴首條及現時唯一一條途經東區海底隧道的全日專營過海來往九龍城區的巴士路線。<ref>{{cite book|author=龔家豪 |coauthors= |title=香港巴士回顧2018 |page= 106-115頁|publisher=新尚線出版 |location=香港 |isbn=9789887862246 |date=2019-01 |accessdate= }}</ref>

==歷史==
為配合啟德發展區未來的人口增長，[[運輸署|運輸署]]在「2017-2018年度九龍城區巴士路線計劃」中，建議在平日上午繁忙時間，開辦一條新巴士服務往九龍城（盛德街）單向往西灣河（嘉亨灣），加強啟德發展區及港島的公共交通連繫，並決定以[[招標|招標]]形式挑選合適的巴士公司營辦本線<ref>{{Cite web |url=http://www.td.gov.hk/filemanager/en/util_uarticle_cp/2017-2018%20kc%20rpp.pdf |title=2017 - 2018年度九龍城區巴士路線計劃 |access-date=2018-04-27 |archive-url=https://web.archive.org/web/20180414234139/http://www.td.gov.hk/filemanager/en/util_uarticle_cp/2017-2018%20kc%20rpp.pdf |archive-date=2018-04-14 |dead-url=no }}</ref>。

2018年4月，運輸署決定向城巴批出本線的專營權<ref>{{Cite web |url=https://www.hk01.com/%E7%A4%BE%E6%9C%83%E6%96%B0%E8%81%9E/176567/%E5%9F%8E%E5%B7%B4%E4%B9%9D%E9%BE%8D%E5%8D%80-%E6%8F%92%E6%97%97-%E7%8D%A8%E7%87%9F%E5%95%9F%E5%BE%B7%E5%8D%803%E6%A2%9D%E6%96%B0%E7%B7%9A-%E6%96%99%E6%9C%88%E5%BA%95%E8%B5%B7%E6%8A%95%E5%85%A5%E6%9C%8D%E5%8B%99 |title=城巴九龍區「插旗」 獨營啟德區3條新線 料月底起投入服務 |access-date=2018-04-27 |archive-url=https://web.archive.org/web/20180417023310/https://www.hk01.com/%E7%A4%BE%E6%9C%83%E6%96%B0%E8%81%9E/176567/%E5%9F%8E%E5%B7%B4%E4%B9%9D%E9%BE%8D%E5%8D%80-%E6%8F%92%E6%97%97-%E7%8D%A8%E7%87%9F%E5%95%9F%E5%BE%B7%E5%8D%803%E6%A2%9D%E6%96%B0%E7%B7%9A-%E6%96%99%E6%9C%88%E5%BA%95%E8%B5%B7%E6%8A%95%E5%85%A5%E6%9C%8D%E5%8B%99 |archive-date=2018-04-17 |dead-url=no }}</ref>，城巴將此線編號命名為608號線<ref>{{Cite web |url=http://www.districtcouncils.gov.hk/kc/doc/2016_2019/tc/committee_meetings_doc/TTC/14249/Pages_from_TTC_15_11.pdf |title=城巴：有關要求盡快落實城巴20、22、608行駛時間 |access-date=2018-05-04 |archive-url=https://web.archive.org/web/20180430182609/http://www.districtcouncils.gov.hk/kc/doc/2016_2019/tc/committee_meetings_doc/TTC/14249/Pages_from_TTC_15_11.pdf |archive-date=2018-04-30 |dead-url=no }}</ref>。

2018年5月，城巴內部一度透露本線將提供雙向全日服務，但最後決定安排開辦早上繁忙時間由九龍單向港島的服務。當局已經安排連接啟德至馬頭角的[[承啟道|承啟道]]於2018年7月25日通車，並預計於同年8月投入服務，但只會提供平日早上單向前往港島的服務。<ref>[https://www.hk01.com/%E7%A4%BE%E6%9C%83%E6%96%B0%E8%81%9E/227068/%E6%96%B0%E5%B7%B4%E5%A3%AB%E7%B7%9A608%E5%B9%B3%E6%97%A5%E5%BE%97%E5%85%A9%E7%8F%AD%E5%85%BC%E5%96%AE%E7%A8%8B-%E6%9D%B1%E5%8D%80%E5%B1%85%E6%B0%91%E9%AC%A7%E7%88%86%E7%84%A1%E8%BB%8A%E6%90%AD 新巴士線608平日得兩班兼單程　東區居民鬧爆無車搭] {{Wayback|url=https://www.hk01.com/%E7%A4%BE%E6%9C%83%E6%96%B0%E8%81%9E/227068/%E6%96%B0%E5%B7%B4%E5%A3%AB%E7%B7%9A608%E5%B9%B3%E6%97%A5%E5%BE%97%E5%85%A9%E7%8F%AD%E5%85%BC%E5%96%AE%E7%A8%8B-%E6%9D%B1%E5%8D%80%E5%B1%85%E6%B0%91%E9%AC%A7%E7%88%86%E7%84%A1%E8%BB%8A%E6%90%AD |date=20180831035401 }}，HK01，2018年8月28日</ref>

2018年8月27日：本線開辦，但只在早上繁忙時間，提供九龍城單向往西灣河的服務。<ref>{{Cite web |url=https://www.td.gov.hk/filemanager/tc/content_13/ta%20608%20chi.pdf |title=開辦隧巴第 608 號線 (九龍城(盛德街) 往 西灣河(嘉亨灣))  |access-date=2018-08-22 |archive-url=https://web.archive.org/web/20180822113824/https://www.td.gov.hk/filemanager/tc/content_13/ta%20608%20chi.pdf |archive-date=2018-08-22 |dead-url=no }}</ref><ref>[http://www.nwstbus.com.hk/tc/uploadedPressRelease/10869_23082018_608_chi.pdf 城巴開辦九龍市區過海隧道服務608號線]</ref>

本線的原定回程路線是[[馬頭角道|馬頭角道]]左轉[[炮仗街|炮仗街]]，但由於相關路面狹窄，12.8米長巴士未能通行，使回程服務一度未能開辦。直至2018年9月，運輸署及城巴建議回程改經[[土瓜灣|土瓜灣]]下路、[[新柳街|新柳街]]、上路返回盛德街總站。結果，這建議獲區議會通過<ref>[https://www.facebook.com/buxoffice/photos/pcb.1278338558972547/1278338468972556/?type=3&theater 畢東尼區議員辦事處 facebook]</ref><ref>[https://www.hk01.com/社會新聞/242344/608巴士研全日雙向行車-消失的班次有望重現 608巴士研全日雙向行車　消失的班次有望重現]，HK01，2018年10月3日</ref>。

2018年12月2日起，本路線擴展至提供來回方向全日服務。<ref name = "608FULL">香港特別行政區政府運輸署，〈[https://scontent-hkg3-2.xx.fbcdn.net/v/t1.0-9/46362726_2183628935193714_3827969497541640192_n.jpg?_nc_cat=106&_nc_ht=scontent-hkg3-2.xx&oh=12254a583e0916be547737ee02b15329&oe=5C7B4B2C 調整隧巴第608號線服務]〉，2018年11月14日。</ref><ref name = "ctbf">城巴有限公司，〈[http://mobile.nwstbus.com.hk/pdf/E201802430a03.pdf 市區過海隧道路線 608 提升至全日服務往返西灣河至九龍城]〉〔乘客通告〕，2018年11月26日。</ref><ref>[https://www.hk01.com/18%E5%8D%80%E6%96%B0%E8%81%9E/264932/%E5%95%9F%E5%BE%B7%E5%B1%85%E6%B0%91%E6%B3%A8%E6%84%8F-%E8%A5%BF%E7%81%A3%E6%B2%B3%E4%BE%86%E5%9B%9E%E4%B9%9D%E9%BE%8D%E5%9F%8E-%E5%9F%8E%E5%B7%B4608%E4%BB%8A%E5%91%A8%E6%97%A5%E8%B5%B7%E5%85%A8%E6%97%A5%E6%9C%89%E5%BE%97%E6%90%AD 【啟德居民注意】西灣河來回九龍城　城巴608今周日起全日有得搭] {{Wayback|url=https://www.hk01.com/18%E5%8D%80%E6%96%B0%E8%81%9E/264932/%E5%95%9F%E5%BE%B7%E5%B1%85%E6%B0%91%E6%B3%A8%E6%84%8F-%E8%A5%BF%E7%81%A3%E6%B2%B3%E4%BE%86%E5%9B%9E%E4%B9%9D%E9%BE%8D%E5%9F%8E-%E5%9F%8E%E5%B7%B4608%E4%BB%8A%E5%91%A8%E6%97%A5%E8%B5%B7%E5%85%A8%E6%97%A5%E6%9C%89%E5%BE%97%E6%90%AD |date=20181205060736 }}，HK01，2018年11月29日</ref>

==使用狀況==
本路線行經[[東區海底隧道|東區海底隧道]]及[[觀塘繞道|觀塘繞道]]，特快來往[[土瓜灣|土瓜灣]]、[[啟德|啟德]]、[[九龍灣|九龍灣]]至[[東區_(香港)|港島東區]]。相對走線迂迴的[[過海隧道巴士106線|106線]]及[[過海隧道巴士606線|606線]]，本路線明顯更為直接及快捷，節省超過一半時間，車費亦較便宜，加上本線同時駛經[[九龍灣商貿區|九龍灣商貿區]]及[[鰂魚涌|鰂魚涌]][[太古坊|太古坊]]（分別吸引東區區民到九龍灣上班及[[土瓜灣|土瓜灣]]、[[啟德|啟德]]一帶居民到太古坊上班），繁忙時間兩邊方向都可望出現滿載乘客的情況，但其餘時間則客量稍為疏落。

本路線與多條[[新巴|新巴]]及[[城巴|城巴]]路線設有八達通轉乘優惠，變相擴大其服務範圍。不過，現時608線只在往港島方向提供與[[新巴694線|694線]]的$5折扣轉乘優惠。因此，有東區區議員要求城巴及新巴提供608及694的免費轉乘優惠，以加強[[小西灣|小西灣]]至九龍東的特快巴士服務。<ref>[https://www.districtcouncils.gov.hk/east/doc/2016_2019/tc/working_groups_doc/WGPTRS/16511/e_ttc_wgptrs_2018_031_tc.pdf 加強608與694的轉乘優惠]</ref>
