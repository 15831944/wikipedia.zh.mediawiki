'''{{zy|掸|shan4}}邦'''（{{lang-my|ရှမ်းပြည်နယ်}}，[[掸语|掸语]]：[[File:Mongtai.png|50px]] {{IPA2|mœ́ŋ tɑ́ɪ}}），是[[缅甸联邦|缅甸联邦]]的一个邦，155,801平方公里，人口约800多万人（2005年），是[[缅甸联邦|缅甸联邦]]裡面积最大和人口最多的一个邦。主體居民[[傣族|傣族]]（又称[[掸族|掸族]]，英文中“掸族”一词特指傣耶支系）。首府[[东枝|东枝]]。

== 地理 ==

掸邦东面与中国大陆[[云南省|云南省]]、[[老挝|老挝]]和[[泰国|泰国]]相接壤，南面[[克耶邦|克耶邦]]、[[克伦邦|克伦邦]]相連，西面與[[缅邦|缅邦]]相连，北面与[[克钦邦|克钦邦]]相連。

掸邦屬於[[高原|高原]]地區，著名的山岭间有江河谷地有[[黑河盆地|黑河盆地]]、[[锡波盆地|锡波盆地]]、[[登尼盆地|登尼盆地]]和[[景栋盆地|景栋盆地]]等。
主要河流为[[萨尔温江|萨尔温江]]，该江发源于中国大陆，流经掸邦，南流去。掸邦和老挝之间有'''[[湄公河|湄公河]]界河'''。

== 历史 ==
掸邦在唐宋时代梵语称为“[[妙香国|妙香国]]”，当时掸邦跟云南南部同属一个地区。 公元937年，[[后晋|后晋]][[天福_(年号)|天福]]二年，通海节度段思平灭南诏建国，定都[[羊苴咩城|羊苴咩城]]，国号[[大理|大理]]，疆域大致是现在的中国云南省、贵州省、四川省西南部，缅甸北部地区，以及老挝与越南的少数地区。公元1095年[[高升泰|高升泰]]改国号为[[大中国|大中国]]，1096年高升泰在死后归政于段正淳，史称[[后理国|后理国]]。公元1253年，[[元世祖|元世祖]][[忽必烈|忽必烈]]“革囊渡江”征云南，[[元朝|元朝]]灭[[大理国|大理国]]，后建[[云南等处行中书省|云南等处行中书省]]，原大理国王段氏被任为大理世袭总管。

1886年，'''[[英国|英国]]'''佔領缅甸本部（俗稱下緬甸），1890年'''英國'''征服撣聯邦，置众掸邦、北部克钦邦、西北部钦邦等众边区，为英帝国的保护边区。當時掸聯邦仍然由掸邦各族[[土司|土司]]王统治，而不隸属缅甸本部。南掸邦系[[孟艮|孟艮]]、整欠、猛勇、整卖、景线、六本、景海、猛龙、补哈、猛撤诸地，北掸邦系[[木邦|木邦]]、大山、猛育诸地。那裡的居民在[[漢代|漢代]]出現于文献，稱「乘象蠻」（版圖至[[老撾|老撾]]北部），國號越裳，歸[[永昌郡|永昌郡]]管。國王叫雍田，[[萬象|萬象]]一名由它而來。

撣邦南部曾屬於[[蘭納|蘭納]]王國。1776年暹羅贏得[[泰緬戰爭|泰緬戰爭]]，奪得含蘭納在内的幾乎所有今日撣邦土地。1893年，暹羅又把這些土地割讓給[[英屬緬甸|英屬緬甸]]。

1922年時，撣聯邦由33邦聯合組成，分別為：21個[[撣族|撣族]]邦、7個{{link-en|德努族|Danu people}}邦、2個[[巴奧人|巴奧族]]邦、1個[[德昂族|巴朗族]]邦、1個[[長頸族|巴當族]]邦、1個[[果敢族|果敢族]]邦。撣聯邦由'''英國'''保護時期（1890年－1948年），'''英國'''當局將[[佤邦|佤邦]]劃入撣聯邦，共計34邦。

第二次世界大戰[[緬甸戰役|緬甸戰役]]中，英軍被日泰聯軍重創，泰國在[[日本帝國|日本]]同意下，割撣邦南部给泰國，設立[[源泰州|源泰州]]。1945年6月，盟軍擊退當地泰國軍隊，撣邦被解放。

1947年2月11日，撣聯邦委員會制訂撣國歌與三色白圈[[掸邦旗帜|撣邦國旗]]，並定2月7日為撣國慶日。

1947年2月12日，撣邦、[[克欽邦|克欽邦]]、[[欽邦|欽邦]]及緬甸本部於撣邦境內{{link-en|彬龍|Panglong, Southern Shan State}}小鎮上，簽署{{link-en|彬龍協議|Panglong Agreement}}，協議之目的為：聯合撣邦、緬甸、克欽邦、欽邦及'''英國'''統治下的各邦，聯合向'''英國'''爭取聯合獨立。

1947年2月15日,撣聯邦委員會改稱為眾撣邦委員會。

1947年9月24日，倉促完成[[緬甸聯邦憲法|緬甸聯邦憲法]]。

1948年1月4日，聯合獨立成功，[[緬甸聯邦_(1948–1962)|緬甸聯邦]]誕生。國家依《緬甸聯邦憲法》運作。是年，為配合'''緬甸聯邦'''的國體，撣邦政府把2月7日「撣國慶日」改名為「撣邦節」，撣邦之名一直沿用至今。1948年，緬甸聯邦第一任聯邦總統由撣邦撣族土司王[[蘇瑞泰|蘇瑞泰]]擔任。任期1948年至1952年。

1952年，[[吳努|吳努]]為首之[[反法西斯人民自由同盟|反法西斯人民自由同盟]]成為[[緬甸聯邦執政黨|緬甸聯邦執政黨]]。同年，緬甸藉驅逐[[中國國民黨|中國國民黨]]殘軍為由，進軍撣邦。

1958年，[[奈溫|奈溫]]將軍強迫吳努讓其成立「[[看守政府|看守政府]]」。同年，《緬甸聯邦憲法》賦予撣邦的去、留權力，被看守政府擱置，由撣邦人民組織而成之軍隊，與看守政府之國防軍發生歷史性之第一次交戰。《緬甸聯邦憲法》明定憲法運作十年後，撣邦、克倫尼邦有權決定自己是否還願意成為[[緬甸聯邦_(1948–1962)|緬甸聯邦]]一份子。

1962年，吳努的[[聯邦黨|聯邦黨]]（由反法西斯人民自由同盟廉潔派改名）競選獲勝而組織政府。3月2日，緬甸軍事強人奈溫將軍政變奪權，同年，作廢《聯邦憲法》，成立[[革命委員會|革命委員會]]進行軍人獨裁統治。聯邦憲法之作廢等於宣告緬甸聯邦解體，從此，各邦紛紛為原有之自主權而與緬甸本部交戰，遍及緬甸全國之內戰於焉展開，並一發不可收拾。

1964年，撣邦人民組織正式之[[掸邦军|撣邦軍]]，正式為收復撣邦國土而努力。1968年，[[緬甸共產黨|緬甸共產黨]]進入撣邦東部，並於當地成立[[撣邦人民軍|撣邦人民軍]]，人民軍主要由當地[[佤族|佤族]]組成。1988年10月26日，{{link-en|撣邦眾民族民主聯盟|Shan Nationalities League for Democracy}}成立。

1989年3月12日，[[彭家聲|彭家聲]]的[[缅甸民族民主同盟军|果敢部隊]]於[[果敢縣|果敢縣]]宣佈脫離緬甸共產黨。並建置[[緬甸民族民主同盟軍|緬甸民族民主同盟軍]]。是年，與緬甸軍政府逹成停戰協議。4月，[[林明賢|林明賢]]之815軍區宣佈脫離緬甸共產黨，並另立門戶組織[[撣邦東部同盟軍|撣邦東部同盟軍]]。同年，與緬甸軍政府逹成停戰協議。4月17日，[[鮑有祥|鮑有祥]]之佤部隊於佤邦宣佈脫離緬甸共產黨，並成立緬甸國家團結黨，隨後更名為[[佤邦聯合黨|佤邦聯合黨]]，武裝部隊命名為[[佤邦聯軍|佤邦聯軍]]。5月18日，佤邦與緬甸軍政府逹成停戰協議。

1989年撣邦軍分裂，北撣邦軍與緬甸軍政府達成停戰協議。其餘撣邦軍繼續為收復國土而努力。

1990年緬甸舉行全國普選，[[昆吞屋|昆吞屋]]（Khun Toon Oo）領導之撣邦眾民族民主聯盟在撣邦地區取得壓倒性勝利。同時，[[翁山蘇姬|翁山蘇姬]]女士領導之[[全國民主聯盟|全國民主聯盟]]在緬甸本部地區也取得壓倒性勝利。遺憾的是，選舉結果被緬甸軍政府否定。

1993年12月12日，被外界稱為毒品大王之[[孟泰軍|孟泰軍]]領導[[坤沙|坤沙]]，宣佈撣邦脫離緬甸聯邦獨立建國，建立[[撣邦共和國|撣邦共和國]]。1996年1月24日，坤沙領導之孟泰軍在佤軍、緬軍、泰國政府之壓力下，撣邦共和國正式向[[緬甸軍政府|緬甸軍政府]]無條件投降。

== 法律 ==
2006年5月24日，掸邦宪法起草委员会公布第一版掸邦宪法，该部宪法将是掸邦所有法令的法源依据。第一版宪法中，最为突出的三项分别为：一、法定[[英语|英语]]为掸邦的唯一通用官方语言，各族的母语限使用于各族的社群里；二、选举制度采用比例代表制，有利于小政党的声音也能进入国会中；三、国会采用二院制，上议院或称民族院比下议院拥有更大的权力，比如，总统与副总统由上议院议员投票决定，各行政部门官员亦是。

== 主要镇区 ==

*[[東枝|東枝]]市：掸邦的首府，人口30万多人（1994年），通铁路和公路。[[黑河_(緬甸)|黑河]]的[[黑河機場_(緬甸)|機場]]。
*[[景棟|景棟]]市：位于[[萨尔温江|萨尔温江]]以东，是一个重要城镇。
*[[老街市_(果敢)|老街市]]：[[撣邦第一特區|撣邦第一特區]]（果敢）之政治、經濟、文化中心。
*[[邦康|邦康]]市：[[佤邦|撣邦第二特區]]（佤邦）之政治、經濟、文化中心。
*[[勐拉|勐拉]]市：[[撣邦东部第四特區|撣邦东部第四特區]]（[[勐拉鄉_(緬甸)|勐拉鄉]]）之政治、經濟、文化中心。
*[[大其力|大其力]]市：大其力市是一个靠近泰国边界的边境城市，在边境贸易中发挥着重要的作用。
*[[腊戍|腊戍]]市：掸邦北部的一个重要的铁路运输枢纽，航空线路可通往各地。
*[[南渡_(撣邦)|南渡]]市：南渡市因有冶炼厂而出名。
*[[木姐|木姐]]市：緬中邊境城市，為緬甸對中國大陆貿易門戶。
