《'''言葉之庭'''》（{{lang-ja|言の葉の庭}}）是2013年上映的[[日本电影|日本]][[动画电影|动画电影]]，由[[新海誠|新海誠]]编剧和执导、动画公司[[CoMix_Wave_Films|CoMix Wave Films]]制片、[[东宝|东宝]]发行，並由[[入野自由|入野自由]]和[[花澤香菜|花澤香菜]]擔綱主角配音。音乐方面，新海诚早期许多作品的音乐都由[[天门_(作曲家)|天门]]创作，而这次则由[[柏大辅|柏大辅]]负责。本片以源于日本古代对爱情的语言表述方法展開，诞生了这個由揭示主题的雨、《[[万叶集|万叶集]]》诗句以及日本的庭园等元素组成的的“孤悲”的《言叶之庭》之故事。此外，該片是目前新海誠執導電影中唯一無英文副標題的作品。

== 劇情 ==
作品以愛情來臨前的孤獨——「孤悲」來呈現「戀愛」的主題。梅雨時節，以成為鞋匠為目標的高中生秋月孝雄與神秘的女性雪野百香里，在雨中的[[新宿御苑|新宿御苑]]中相遇。故事以女性留給少年的一段《[[萬葉集|萬葉集]]》詩句為契機，就此開展……

故事以15岁高中生，秋月孝雄翘课来到[[新宿御苑|新宿御苑]]内为开端，此时的东京随着入梅而开始进入[[梅雨季|梅雨季]]。喜欢雨季的孝雄为自己规定仅在雨天的上午翘课，他以成為制鞋师為目標，到庭院内来也是思考设计鞋子并画出草图。这一次孝雄在亭子里与神秘的女性雪野百香里邂逅，27岁的她翘班来到庭院内饮酒并配以许多种类的巧克力，有时她也会看书。在百香里注意到孝雄的校服后转身离去，留下了一首让孝雄不解的、出自《万叶集》的日本[[短歌|短歌]]。自此以后，雨季早晨的新宿御苑内的那座亭子便成为他们二人短暂交谈的地点，但他们也并不很了解彼此。

在某一天，孝雄告诉了百香里成为制鞋师的梦想，这是孝雄第一次向外人倾诉自己的想法。百香里因味觉麻痹只能品出酒与巧克力的味道，并在后来发现她能吃出孝雄所做的早饭的味道。百香里为答谢孝雄为她提供早饭而为孝雄买了一本昂贵的制鞋书作为回礼。正打算做一双鞋的孝雄为此决意制作一双与百香里的脚相同码的鞋，以表明這段時日他對百香里所產生的好感。。

随着雨季的结束，孝雄开始专心于打工賺钱和做鞋。暑假結束後孝雄回到了校园，在學校走廊碰搶遇見百里香後，他的朋友告诉了孝雄，雪野百香里是他们学校的古文老师，并得知百香里遭受了以相泽为首的不良同学骚扰。为避免产生更多的麻烦和出于自己的隐忍，百香里选择翘班并来到了新宿御苑那座亭子内独处和休息。在得知百里香是自己學校的老師後，孝雄決定在下次涼亭碰面後表明自己的心意。究竟孝雄這份在大雨中所產生的愛慕之意能不能傳達給百里香得知?

== 角色 ==
;{{nihongo|秋月孝雄|秋月 孝雄（あきづき たかお）／タカオ}}
:15歲高中一年生，以成為鞋匠為目標。因為家庭的因素，使他具有與不符年齡的成熟氣息。某一下雨天的早上，孝雄翹課前往庭園思考鞋子的設計時，遇到了百香里，開始了一段與她的故事。
:由新海誠執筆的小說版中對於動畫的開放性結局有所補充；4年後在義大利當製鞋學徒的秋月和雪野再約雨庭相會。這個結局被新海誠視為正式地補完整個言葉之庭的故事。
;{{nihongo|雪野百香里|雪野 百香里（ゆきの ゆかり）／ユキノ}}
: 27歲，孝雄在雨天偶然遇到的神秘女性，相遇時一邊吃著巧克力，一邊喝著啤酒。個性溫柔善良，因為學校的流言沒有勇氣上班而逃避，與孝雄相遇後才逐漸釋懷。但在劇情中是秋月孝雄就读的高中的古文老师。她与孝雄第一次相遇是在入梅时节的新宿御苑，同时也发现了孝雄是自己所执教的高中的学生一事，于是用高中古文课中所教的一首万叶集短歌作为暗示。
:由于在学校中三年級的某位女同學的男朋友单恋她，而致使那位女同學聯合三年級的女生统一口径造谣污蔑她的事情，对她造成巨大的心理压力，致使她產生身心問題不敢到学校上课，并最终导致她的辞职，回到故鄉[[四國|四國]]從事教職。漫畫版其名改為「由香里」。
:在新海誠執導的次作《[[你的名字。|你的名字。]]》中也有登場，在[[糸守町|糸守町]]的高中擔任古典文學老师，但片中沒有交代離開東京到糸守町的緣由，也非劇情中的重要角色，只是作为彩蛋而存在。
;孝雄之母／秋月-{怜}-美
:47歲女性，已和孝雄的父親離婚。雖然熱愛家庭，但是比起家庭而言，認為戀愛更為重要，因在外有男友而時常不在家。小說版名為{{nihongo|'''秋月-{怜}-美'''|秋月 怜美}}。
;孝雄之兄／秋月翔太
:個性大方隨合，對於母親複雜的感情關係感到無奈，而決定離開家和女朋友同居。小說版名為{{nihongo|'''秋月翔太'''|秋月 翔太}}。
;孝雄之兄其女友／寺本梨花
:24歲的關西人，性格開朗，與孝雄相處融洽。小說版名為{{nihongo|'''寺本梨花'''|寺本 梨花}}。
;松本
:孝雄的同學、朋友。一年級上學期和佐藤戀愛。
;佐藤
:松本的女友，高中二年生。擁有很多朋友，很清楚校內的事情。
;相澤／相澤翔子
:高中三年生。自尊心很強，总是被人所包围着。聯合三年級女生集體欺負學校老師，並且散播不當流言，使老師被迫離職。小說版名為{{nihongo|'''相澤翔子'''|相沢 翔子}}。
;伊藤宗一郎
:為孝雄高中時的班導師，是雪野老師的前男友。
