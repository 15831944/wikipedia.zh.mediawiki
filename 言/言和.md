'''言和'''是[[上海禾念|上海禾念]]推出的一款[[语音合成|语音合成]]资料库及对应的虚拟角色，采用[[Vocaloid|Vocaloid]]技术。<ref name=gamedog1>{{Cite news |url=http://news.gamedog.cn/a/20161011/1904415.html |title=VOCALOID言和cosplay 天青色等烟雨而我在等你 |publisher=游戏狗 |accessdate=2018-05-06 |language=zh-hans |date=2016-10-11 |deadurl=no|archiveurl=https://web.archive.org/web/20161205090300/http://news.gamedog.cn/a/20161011/1904415.html |archivedate=2016-12-05 }}</ref>歌声样本取自中国籍日本声优[[刘婧荦|刘婧荦]]，画师MQ在募集活动设中计出该角色，后{{nihongo|Mercury|マーキュリー}}以该角色推出「言和Project」。<ref name=tshn>{{Cite web|url=http://thstars.com/vsingers/#yanhe|title=虚拟歌手 - 言和|accessdate=2018-05-06|publisher=天矢禾念娱乐集团|language=zh-hans|deadurl=no|archiveurl=https://web.archive.org/web/20180329135817/http://www.thstars.com/vsingers/#yanhe|archivedate=2018-03-29}}</ref><ref>{{Cite news |url=http://www.mnw.cn/yinyue/huayu/1531494-4.html |title=洛天依本人照片洛天依本人是谁 言和洛天依是什么关系？ |publisher=闽南网 |accessdate=2018-05-06 |pages=4 |language=zh-hans |date=2017-01-04 |deadurl=no |archiveurl=https://web.archive.org/web/20180506140440/http://www.mnw.cn/yinyue/huayu/1531494-4.html |archivedate=2018-05-06 }}</ref>上海禾念在2013年7月11日发布其首个产品。<ref>{{Cite web|url = http://ascii.jp/elem/000/000/860/860755/index-2.html|title = 中国ボーカロイドの推進企業に中国ボカロの未来を聞いた！|author = 山谷剛史|date = 2014-01-30|publisher = ASCII.jp|language = ja|deadurl = no|archiveurl = https://web.archive.org/web/20161229235503/http://ascii.jp/elem/000/000/860/860755/index-2.html|archivedate = 2016-12-29}}</ref><ref name=southcn>{{cite web|language=zh-hans| title=V家中国虚拟歌姬《言和》首张单曲CD登场|url=http://cartoon.southcn.com/a/2013-06/28/content_72044277.htm|website=南方动漫网|author=178动漫频道|publisher=南方新闻网|accessdate=2018-05-05|date=2013-06-28|deadurl=no|archiveurl=https://web.archive.org/web/20180506140438/http://cartoon.southcn.com/a/2013-06/28/content_72044277.htm|archivedate=2018-05-06}}</ref>是继[[洛天依|洛天依]]后第二款以中文为演唱语言的Vocaloid资料库，同时也是Vocaloid™ China中第一个以公开募集的形式诞生形象的虚拟歌手。

== 历史 ==
2013年3月23日，Vocaloid™ China参加上海ComicF04，公布新声库及新形象募集活动的相关事宜<ref>{{Cite web |url = http://weibo.com/2473274704/zoLBDb2QI|title = 新声源是地道的女孩子啦！不在现场的同学们... 来自VOCALOID-CHINA - 微博|author = VOCALOID-CHINA|date = 2013-03-23|publisher = [[新浪微博|微博]]|language = zh-hans}}</ref>。两天之后，即3月25日，Mercury企划、Vocaloid™ China与178.com动漫频道协办第二届“Vocaloid™ China”形象募集活动，参赛者可根据提供的一段试听音频（茉莉花）来设计角色，活动截至4月30日结束。5月14日，Vocaloid™ China公布了15件提名作品<ref>{{Cite web |url = http://weibo.com/2473274704/zwFIjb5Hr |title = #VOCALOID CHINA新形象#大家好~已经公布的1... 来自VOCALOID-CHINA - 微博|author = VOCALOID-CHINA |date = 2013-05-14|publisher = [[新浪微博|微博]]|language = zh-hans}}</ref><ref>{{Cite web|url = https://rocketnews24.com/2013/05/16/329331/|title = 中国ボーカロイド第2弾の候補イラスト15作が発表！ 意外とカワイイと話題に|author = 沢井メグ|date = 2013-05-16|publisher = ロケットニュース24|language = ja|deadurl = no|archiveurl = https://web.archive.org/web/20180403173438/https://rocketnews24.com/2013/05/16/329331/|archivedate = 2018-04-03}}</ref>。后于6月6日公布了5件入围作品（包括1件最终评选作品），并公布了与五位画师的访谈<ref>{{Cite web |url = http://weibo.com/2473274704/zA8UMpwhc |title = 大家~早上好，MERCURY SHANGHAI公司的新声... 来自VOCALOID-CHINA - 微博|author = VOCALOID-CHINA |date = 2013-06-06|publisher = [[新浪微博|微博]]|language = zh-hans}}</ref>。这标志着言和成为Vocaloid™ China中第一个以公开募集的形式诞生形象的虚拟歌手<ref name=tshn />。

2013年7月11日，《VOCALOID™3 Library 言和》在第九届中国国际动漫游戏博览会上正式发布<ref>{{Cite web |url = http://weibo.com/3500223314/zFaA9Bqk5|title = #言和产品发布会#时间是2013年7月11日 CCG ... 来自言和PROJECT - 微博|author = 言和PROJECT|date = 2013-07-09|publisher = [[新浪微博|微博]]|language = zh-hans}}</ref><ref>{{Cite web|url = http://getnews.jp/archives/371112|title = 中国語ボーカロイド第2弾『言和』が発売決定！|author = 日刊VOCALOIDガイド|date = 2013-07-01|publisher = {{tsl|ja|ガジェット通信}} / 東京產業新聞社|language = ja|deadurl = no|archiveurl = https://web.archive.org/web/20180403173510/http://getnews.jp/archives/371112|archivedate = 2018-04-03}}</ref>。言和是继[[洛天依|洛天依]]后第二款以中文为演唱语言的Vocaloid™3中文声库<ref name=gamedog1 />。

== 角色 ==
根据官方描述，她外表帅气，内心温柔。特征为其白色的短发和其修长的腿。象征物为[[北极兔|北极兔]]<ref name=tshn />。言和的设计画师MQ称，言和外表有些冷漠，但内心很温柔。通过配合参数的调整来改变形象，使得言和穿男装可以很帅气，穿女装可以很温柔，即采用了中性的设定，“是个带一点男孩子气的女生”<ref>{{Cite web |url=http://acg.178.com/zt/vocaloid_china_2nd_project/interview1.html |title=新声库形象募集评选官方画师特约访谈 |publisher=178动漫 |language=zh-hans |accessdate=2018-05-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20170910190854/http://acg.178.com/zt/vocaloid_china_2nd_project/interview1.html# |archivedate=2017-09-10 }}</ref><ref>{{Cite web |url=http://acg.178.com/zt/vocaloid_china_2nd_project/ |title=VOCALOID™ CHINA新声库形象募集评选结果发布 |publisher=178动漫频道 |language=zh-hans |accessdate=2018-05-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20130609133826/http://acg.178.com/zt/vocaloid_china_2nd_project/ |archivedate=2013-06-09 }}</ref>。
