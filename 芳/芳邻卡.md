== 资格 ==
计划申请芳邻卡的人必须是美国、加拿大两国的[[公民|公民]]或者[[永久居留權|永久居民]]。之前曾要求申请者必须在两国中的任何一国居住3年，这一规定目前已经被取消，以便让居住第三国的美、加公民返回本国之后能申请芳邻卡。但是两国的永久居民仍然需要居住三年的时间方可提交申请。申请者需要证明持有有效旅行证件以往返这两个国家，并且在过往的旅行中遵守两国的移民和海关法律。申请者需要经过犯罪记录的调查。除此之外，如果申请者年龄不足18岁，其父母必须提交书面文件认可申请。<ref>{{cite web|website=[[加拿大边境服务局|加拿大边境服务局]]|url=http://www.cbsa-asfc.gc.ca/prog/nexus/elig-admis-eng.html|title=Eligibility|accessdate=2017-07-22|archive-url=https://web.archive.org/web/20161225111135/http://www.cbsa-asfc.gc.ca/prog/nexus/elig-admis-eng.html|archive-date=2016-12-25|dead-url=yes}}</ref><ref>{{cite web |title= NEXUS Eligibility |website=[[美国海关及边境保卫局|美国海关及边境保卫局]]|url= https://www.cbp.gov/travel/trusted-traveler-programs/nexus/nexus-eligibility| accessdate = 2017-07-22 }}</ref>

== 申请 ==
[[File:NEXUSCard.png|thumb]]
申请者可以通过[[美国海关及边境保卫局|美国海关及边境保卫局]]的网站提交申请，或者向[[加拿大边境服务局|加拿大边境服务局]]提交纸质材料申请。申请费用为50美元/加元。18岁以下的申请者可以豁免申请费。

收到申请材料的机构会审查申请者的国籍和移民身份状态，并请求[[联邦调查局|联邦调查局]]（FBI）、[[加拿大国家安全情报局|加拿大国家安全情报局]]（CSIS）、[[皇家加拿大騎警|皇家加拿大騎警]]（RCMP）等机构协助对申请者进行背景调查，检查申请者是否有犯罪记录或恐怖主义组织有关。通过初步筛选的申请者会收到面谈邀请，前往指定的机构接收美国海关及边境保卫局和加拿大边境服务局官员的面试。申请得到批准后，申请者会在不久之后收到邮寄的芳邻卡卡片。该卡自申请者收到卡片之后的那个生日开始，有效期为5年。换新卡通常需要在旧卡失效前6个月时进行，申请者可能需要再次进行类似的面试，官员会决定持卡人是否还符合芳邻卡的条件。<ref>{{cite web|url=https://www.cbp.gov/travel/trusted-traveler-programs/nexus/how-apply|publisher=[[美国海关及边境保卫局|美国海关及边境保卫局]]|title=How to Apply| accessdate = 2017-07-22 }}</ref><ref>{{cite web|url=http://www.cbsa-asfc.gc.ca/prog/nexus/application-demande-eng.html|publisher=[[加拿大边境服务局|加拿大边境服务局]]|title=Join NEXUS| accessdate = 2017-07-22 }}</ref>

== 使用 ==
[[File:NexusLane.JPG|thumb]]（位于密歇根州-安大略省之间）口岸的芳邻卡车道]]
芳邻卡是符合[[西半球旅行计划|西半球旅行计划]]的旅行文件，<ref name = "cbsa-faq">{{cite web|title=Frequently Asked Questions about the NEXUS Program|url=http://www.cbsa-asfc.gc.ca/prog/nexus/faq-eng.html|publisher=[[加拿大边境服务局|加拿大边境服务局]]| accessdate = 2017-07-22 }}</ref>可以被旅行者用作替代护照的证件通过已安装相关设备的美国和加拿大机场、边境口岸往返美国和加拿大。<ref>{{cite web|title=WHTI Program Background|url=http://www.cbp.gov/travel/us-citizens/whti-program-background|publisher=[[美国海关及边境保卫局|美国海关及边境保卫局]]| accessdate = 2017-07-22 }}</ref>

芳邻卡持有者通常可以比一般旅客在通过边境时花费更少的时间。不过芳邻卡持有者仍然有可能接收常规的移民和海关检查，甚至被要求接收更仔细的二次检查（secondary screening）。大多数参与芳邻卡项目的边境口岸一般都有一条转为芳邻卡持有者保留的快速通道，部分口岸甚至预留了第二条快速通道。在陆路口岸，如果车辆上所有乘客（包括儿童）均持有芳邻卡，并且车辆上未装载需要特殊报关手续的货物，则这辆车可以使用专用的芳邻卡车道。

抵达美国的芳邻卡持有者可以在美国机场扫描卡片，或在设有[[美國境外預先清關|美國境外預先清關]]设施的海外机场（如[[温哥华国际机场|温哥华国际机场]]）提前扫描卡片入境美国。美国机场将比较持卡人的指纹与数据库中的数据。如果持卡人使用Global Entry自助设备，则无需单独填写海关申报单。芳邻卡持有者在搭乘美国国内航班时，可以在部分硬件条件具备的美国境内机场享受联邦[[运输安全管理局|运输安全管理局]]提供的TSA Pre快速安检待遇，旅客可以使用专门的快速安检通道，并无需脱鞋或取出笔记本电脑等。<ref>{{Cite web | url =https://www.cbp.gov/travel/trusted-traveler-programs/nexus/tsa-pre✓® | title =TSA Pre✓® | publisher = [[美国海关及边境保卫局|美国海关及边境保卫局]] | accessdate = 2017-07-22}}</ref>

== 法律效力 ==
在美国，芳邻卡作为可信旅行者项目的证件，被视为是可以合法入境美国的证件。虽然该卡包含持卡人的国籍信息，但是无论美国还是加拿大的法律都未将芳邻卡作为公民国籍的证据。大多数时候美国、加拿大持芳邻卡的公民在不携带护照的情况下都能自由往返，但是机场、口岸的官员仍然在需要二次检查的情况下保留检查护照的权力。<ref name = "cbsa-faq"/>不具有美国或加拿大国籍的芳邻卡（如[[美国永久居民|美国永久居民]]和[[加拿大永久居民|加拿大永久居民]]）持有人在旅行时需要携带他们的护照以及永久居民证件（[[美国永久居民|绿卡]]或[[加拿大永久居民卡|枫叶卡]]）。

除了在出入美国、加拿大国境时使用，一般芳邻卡在其他场合（例如银行开户），或者在其他国家不被接受作为身份证件。
