'''芳香性'''是一種[[化學|化學]]性質，有芳香性的分子中，由[[不饱和|不饱和]]键、[[孤对电子|孤对电子]]和空轨道组成的[[共軛系統|共軛系統]]具有特別的、仅考虑共轭时无法解释的稳定作用。可以将芳香性看作是环状[[离域|离域]]和环[[共振论|共振]]的体现。一般认为在这些体系中的电子，可以自由在由原子组成的环形结构上运动（离域），这些环形结构含有单键和双键相间，离域的结果是环键的键级趋于均化，给予体系稳定作用。

芳香性的理論最初由[[凱庫勒|凱庫勒]]發展出來，是以六元的[[苯|苯]]分子为原型建立起来。理論中的[[苯|苯]]有兩個共振形態，並有單键和双键相间，但理论上的两种苯（环己三烯）衍生物的这类异构体在实际上无法检测或分离出来，苯事实上是这两个异构体的“杂化体”，并且具有不考虑电子离域时无法解释的稳定性。

== 理論 ==
[[File:Benz4.png|right]]

右上圖中的雙箭頭於化學中代表共振的意思，但不代表該分子會變成那兩個形態，反而是指該分子處於該兩個形態的混合狀態。事實上，那兩個形態都不能代表分子實際的形態，但指出了電子大概的分佈。由此可見，分子真實的形態就是那兩個形態的融合，而它所發出的電磁波波長就是該兩個形態所發出的平均長度。

用組成[[π鍵|π鍵]]的電子密度分佈的位置，來更準確地解釋就是電子分佈在分子的環的上、下。

== 歷史 ==
凱庫勒最初在苯之中發現芳香性這特性，其後於1931年才有人以[[量子力學|量子力學]]來解釋此現象。當年亦證實芳香性之中的π電子的數量必定是4n+2的定律，並肯定芳香族分子的環必定是平面的。

== 芳香性化合物的特徵 ==
被分類為芳香性的化合物通常有以下的條件：

# 有一些[[離域電子|離域電子]]組成一些[[π鍵|π鍵]]，並且令整個環系統可以當成單與雙鍵的組合；
# 给出離域電子形成π鍵的原子需處於同一個平面；
# 原子需組成一個環；
# 組成π鍵的電子總數需为 4n+2，即<u>不是4的倍數</u>的雙數（[[休克爾規則|休克爾規則]]）；
# 可進行[[親電芳香取代反應|親電芳香取代反應]]和[[親核芳香取代反應|親核芳香取代反應]]。

苯就是一個较好的例子，它適合以上所有條件，並且有6个離域電子（即n=1）。有4n+2个π電子的化合物通常都是芳香性的。[[環丁二烯|環丁二烯]]只有4个離域電子，所以不屬於芳香性化合物。這些只有4n个π電子而又具近似平面结构的環狀化合物称为[[反芳香性|反芳香性]]化合物。

非芳香族的有機物就叫做脂肪族，脂肪族没有芳香族具有的特殊的共振稳定作用。在受到与平面垂直的外磁场作用时，離域的π電子环电流產生的感应磁场可将整个空间划分为屏蔽区与去屏蔽区，可在[[核磁共振|核磁共振]]结果上显示出来。

在商業中最重要的芳香化合物就是[[苯|苯]]和[[甲苯|甲苯]]，每年產量極高。從[[石油|石油]]中得到的苯和甲苯可用來做其他極有用的日用品材料，包括[[苯乙烯|苯乙烯]]、[[苯酚|苯酚]]、[[苯胺|苯胺]]及[[尼龍|尼龍]]。

== 芳香性化合物的分類 ==
芳香性化合物大致可分為：簡單的芳香化合物、[[多环芳香化合物|多环芳香化合物]]和[[雜環化合物|雜環化合物]]。

=== 雜環化合物 ===
[[雜環化合物|雜環化合物]]之中組成環的[[原子|原子]]不僅包括[[碳|碳]]，還包括[[氮|氮]]、[[氧|氧]]或[[硫|硫]]等原子，例如五元的[[吡咯|吡咯]]、[[呋喃|呋喃]]和[[噻吩|噻吩]]、六元的[[吡啶|吡啶]]、多环的[[吲哚|吲哚]]、[[喹啉|喹啉]]等。

=== 多环芳香化合物 ===
多环芳香化合物中有一类[[多环芳香烃|多环芳香烃]]，其分子由超過一個不包含雜環或取代基的芳香環融合在一起並同時分享兩個碳原子所組成，當中有一些是[[致癌物質|致癌物質]]。例子有[[萘|萘]]、[[蒽|蒽]]、[[菲|菲]]。

=== 簡單的芳香化合物 ===
[[芳香烴|芳香烴]]有大量的[[有机化合物|有机化合物]]的結構當中都包含簡單的芳香環，例如[[DNA|DNA]]（包含[[嘌呤|嘌呤]]、[[嘧啶|嘧啶]]等）、[[三硝基甲苯|三硝基甲苯]]（含苯環）、[[阿司匹林|乙酰水杨酸]]（含苯環）和[[对乙酰氨基酚|对乙酰氨基酚]]（含苯環）等。
