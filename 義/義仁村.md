'''義仁村'''是中華民國臺灣嘉義縣竹崎鄉轄下的行政區。位於嘉義縣中間位置，北鄰竹崎鄉獅埜村，西鄰竹崎鄉義和村，南鄰竹崎鄉灣橋村，東接竹崎鄉鹿滿村。
面積約4.7945平方公里，行政區包含13個鄰、395戶、人口數1172人。聚落內坪頂三水宮、慈保宮和清華山德源禪寺為該村景點。<ref>{{cite web |title=資料 |url=https://chuchi-hro.cyhg.gov.tw/News_Content.aspx?n=1F92FE0C4CBB3D34&sms=52269178FC281255&s=8E8F1ACCEC24DFB1 |website=嘉義縣竹崎戶政事務所 |accessdate=2018-10-21}}</ref>

==人文地理==
義仁村地理位置北接獅子頭溪，南接牛稠溪。

義仁村由多個聚落組成，其中兩個村落的命名或形成跟吳鳳有關，其中一個名為番子潭(另名番仔潭)，其命名與[[吳鳳|吳鳳]]有關，另一個為下寮仔，其聚落的居民多為吳鳳之後裔。<ref name="locate">{{cite web |title=資料1 |url=http://gn.moi.gov.tw/geonames/index.aspx |website=地名資訊服務網 |accessdate=2018-10-21}}</ref>

==教育==
本區域內有設立國小但並未設立國中，區域內的國小為義仁國小，義仁國小創立於1948年6月28日，

當時是嘉義縣竹崎鄉鹿滿國校的義仁分校，後於1951年獨立改名義仁國校，1968年改名為現在名稱，國中學區則為竹崎高中國中部。<ref>{{cite web |title=資料2 |url=http://www.yrps.cyc.edu.tw/school/yiren/brief.htm |website=義仁國小官網 |accessdate=2018-10-21}}</ref><ref>{{cite web |title=資料3 |url=http://edu.cyhg.gov.tw/News4.aspx?n=8B04CBD4471A6D45&sms=745127AC0E5C0036 |website=嘉義縣教育處─國小學區劃分表、國中學區劃分表 |accessdate=2018-10-21}}</ref>

==文化資源與景點==
義仁村有多個文化古蹟，有頂坪三水宮、慈保宮和清華山德源禪寺，它們不只是重要的文化資產，同時也是當地的景點。

頂坪三水宮的主神為臨水夫人，建立於1976年

慈保宮的主神為保生大帝。建立於1737年(清朝乾隆二年)

清華山德源禪寺的主祀神明為釋迦牟尼佛，建立於1740年(清朝乾隆五年)<ref>{{cite web |title=資料4 |url=http://crgis.rchss.sinica.edu.tw/ |website=文化資源地理資料庫 |accessdate=2018-10-21}}</ref>
