'''魯平'''（{{bd|1927年|9月27日|2015年|5月3日|catIdx=L魯}}），[[中華人民共和國|中國]]前政治人物，祖籍[[四川|四川]][[閬中|閬中]]，曾經擔任[[國務院港澳事務辦公室|國務院港澳事務辦公室]]主任及[[香港基本法起草委員會|香港基本法起草委員會]]祕書，上海市[[宋庆龄基金会|宋庆龄基金会]]前任主席。

== 早年生活 ==
魯平於[[上海租界|上海租界]]出生及成長，中學時於[[英國人|英國人]]開辦的[[雷士德工學院|雷士德工學院]]就讀，[[中国抗日战争|日本侵華]]期間，雷士德工學院被日本人接管，魯平遂轉讀聖約翰大學附中，考進聖約翰大學後修讀了農科，期間參加了[[中國共產黨|中國共產黨]]的地下組織。

== 港澳辦 ==
{{see also|殖民地時期香港政治制度改革#彭定康政改方案}}
1970年代末，鲁平开始接管[[香港|香港]]、[[澳门|澳门]]移交主權的工作。

1980年代初的[[中英谈判|中英谈判]]，鲁平作为中方的谈判人，是[[中英谈判|中英谈判]]的关键人物之一。1984年，[[中英联合声明|中英联合声明]]签署，香港面临重大的信心危机问题，鲁平作为中方的首席官员第一次面对探讨[[一国两制|一国两制]]下的政制方案。

[[六四事件|六四事件]]後，不少人對香港前途感到憂慮。當時港督[[衛奕信|衛奕信]]在施政報告宣佈一系列措施, 以圖挽回香港人及投資者的信心, 包括宣佈推行耗資龐大的「港口及機場發展計劃」（簡稱「新機場計劃」）。中方認為英方此舉會把香港的豐厚儲備花光，遂高調表達不滿，爭取過問新機場計劃。當時的港澳辦副主任魯平曾對著鏡頭一連質問了三次「<small>你說怎麼辦？</small>」，而被香港傳媒送上「魯怎辦」的外號。魯平退休出任[[浙江大學|浙江大學]]教授時，被香港傳媒問起這個外號，他表示他知道，不過並沒有放在心上。

1992年，末任[[香港總督|香港總督]][[彭定康|彭定康]]發表其首份[[施政報告|施政報告]]，推出「[[新九組|新九組]]」[[香港政治制度改革_(殖民地時期)|九五政改方案]]，大幅度增加[[香港立法局|香港立法局]][[直接選舉|直選]]議席，惹來[[中華人民共和國政府|中國政府]]不滿。時任[[國務院港澳事務辦公室|國務院港澳事務辦公室]]主任的魯平，代表當局叱責彭定康为「千古罪人」<ref>{{cite web|url=http://news.ifeng.com/special/decode/200706/0619_1305_137745.shtml|title=鲁平：我为什么怒斥彭定康为千古罪人|work=羊城晚报|publisher=凤凰网|date=2007-06-19}}</ref>，這話引起轩然大波，被指代表中國政府強硬的[[意識形態|意識形態]]及對香港問題毫不妥協的立場。[[香港回歸|香港回歸]]後，卸任港澳辦主任一職，在他擔任港澳辦主任的年代，港澳辦對港的影響力較[[新華社香港分社|新華社香港分社]]為大。

2007年5月，鲁平在接受[[香港有线电视新聞|香港有线电视新聞]]专访时首次透露，当年为挽救香港大资本财团的信心，[[中华人民共和国中央人民政府|中央]]曾打算在第三国设立保险基金，让财团可以直接把资产转移到第三国家，确保投资信心。

== 去世 ==
2015年5月3日19时20分，魯平因癌病在[[北京醫院|北京醫院]]去世，終年87歲。<ref>{{cite web|url=http://www.hmo.gov.cn/Contents/Channel_341/2015/0504/32056/content_32056.htm|title=国务院港澳事务办公室讣告|work=国务院港澳办|date=2015-05-04|deadurl=yes|archiveurl=https://web.archive.org/web/20150505003515/http://www.hmo.gov.cn/Contents/Channel_341/2015/0504/32056/content_32056.htm|archivedate=2015-05-05}}</ref><ref>{{cite web|url=http://hk.apple.nextmedia.com/realtime/china/20150504/53701871|title=國務院前港澳辦主任魯平病逝|work=[[香港蘋果日報|香港蘋果日報]]|date=2015-05-04}}</ref>

== 年表 ==
* 1980年代，任職[[國務院港澳事務辦公室|國務院港澳事務辦公室]]綜合組組長，其後升至秘書長
* 1985年，任職[[香港基本法起草委員會|香港基本法起草委員會]]副秘書長
* 1987年8月，升任至國務院港澳事務辦公室副主任
* 1990年11月，升任至國務院港澳事務辦公室主任
* 1992年10月，在[[中國共產黨第十四次全國代表大會|中國共產黨第十四次全國代表大會]]上當選為中央委員
* 1993年7月，任[[香港特別行政區籌備委員會預備工作委員會|香港特別行政區籌備委員會預備工作委員會]]副主任委員、秘書長
* 1996年1月，任[[香港特別行政區籌備委員會|香港特別行政區籌備委員會]]副主任委員、秘書長
* 1997年7月，卸任[[國務院港澳事務辦公室|國務院港澳事務辦公室]]主任
* 1997年9月，在[[中國共產黨第十五次全國代表大會|中國共產黨第十五次全國代表大會]]上卸任中央委員席位

== 家庭 ==
魯平夫人奚亮，他們育有一子一女魯恭和魯昭。儿媳婦是著名的[[小提琴|小提琴]]演奏家[[姚珏|姚珏]]。
