'''魯文·拉馬第'''（'''Reuven Ramaty'''，{{bd|1937年|2月25日|2001年|4月8日|catIdx=R}}）是一位[[美國|美國]]籍[[匈牙利|匈牙利]]天文學家，生於[[羅馬尼亞|羅馬尼亞]]。拉馬第是[[太陽物理學|太陽物理學]]、[[伽馬射線天文學|伽馬射線天文學]]、[[核天文物理學|核天文物理學]]和[[宇宙線|宇宙線]]研究的先驅者。

== 生平 ==
拉馬第於1937年生於[[羅馬尼亞|羅馬尼亞]][[蒂米什瓦拉|蒂米什瓦拉]]一個匈牙利人家庭中，[[第二次世界大戰|第二次世界大戰]]以後和家人於1948年遷往[[以色列|以色列]]。1961年拉馬第取得[[特拉維夫大學|特拉維夫大學]]物理學士學位，畢業後曾在以色列的中學任教三年。之後前往美國，於1966年取得[[洛杉磯加利福尼亞大學|洛杉磯加利福尼亞大學]]行星與太空科學博士學位。獲得博士學位以後進入 [[NASA|NASA]] [[戈达德太空飞行中心|戈达德太空飞行中心]]任職，並曾在[[馬里蘭大學|馬里蘭大學]]擔任物理學兼任教授。2001年在美國馬里蘭州[[银泉_(马里兰州)|银泉]]逝世。

== 研究 ==
拉馬第是高能太陽光譜成像探測器（HESSI）的研究成員之一，並且是研究團隊的創始成員。他的積極參與和熱心支持是 HESSI 被納入 NASA 的[[小探測工程|小探測工程]]（SMEX）第六個計畫的關鍵。2002年2月5日 HESSI 被發射後立刻改名為[[拉馬第高能太陽光譜成像探測器|拉馬第高能太陽光譜成像探測器]]（RHESSI）向拉馬第致敬。這是第一個以 NASA 科學家命名的太空探測器。

== 家庭 ==
拉馬第的妻子是 Vera，育有2女 Daphne 和 Deborah，另有孫輩5人。

== 其他 ==
拉馬第除了母語[[匈牙利語|匈牙利語]]和[[羅馬尼亞語|羅馬尼亞語]]以外，能說流利的[[希伯來語|希伯來語]]、[[英語|英語]]和[[法語|法語]]。
