《'''魯邦三世 PART5'''》（日語：ルパン三世 PART5），是由漫畫家加藤一彥原作動畫《[[魯邦三世|魯邦三世]]》的系列的第5部電視動畫版動畫，於2018年4月4日至9月19日期間共播出24集。

== 故事內容 ==
故事從魯邦三世與次元大介在一間法國鄉村小鎮公寓裡的房間開始。網路上存在一個黑暗的網站「馬可波羅」，在那裡你可以買到任何非法的東西，如毒品和槍支。為了竊取數位貨幣，魯邦入侵了一個嚴密的伺服器設施，並且遇到一位天才駭客少女艾咪。接著魯邦在「馬可波羅」上被懸賞高額的獎金，而魯邦跟艾咪要共同面對來自各方的殺手......

== 登场人物 ==

=== 主要人物 ===
; {{nihongo|鲁邦三世|ルパン三世（ルパンさんせい）|声：[[栗田贯一|栗田贯一]]（日本）；[[孫誠|孫誠]]（台灣）}}
: 本作主角。世界傳奇大盜，義賊，透過各種奇妙的想法和技術來獲得目標，具有高超的偷盜技術，沒有他偷不到的東西。

; {{nihongo|次元大介|次元 大介（じげん だいすけ）|声：[[小林清志|小林清志]]（日本）；[[張騰|張騰]]（台灣）}}
: 超級神槍手，拔槍只需0.3秒，嗜酒，是老菸槍。即使一人遇到麻煩也能掌握局面並打破困境，是魯邦的忠實搭檔。

; {{nihongo|石川五右卫门|石川 五ェ門（いしかわ ごえもん）|声：[[浪川大輔|浪川大輔]]（日本）；[[喬資淯|喬資淯]]（台灣）}}
: 日本武士，劍法快如閃電，沒有他砍不斷的東西，為石川家族的十三代世孫後裔，是魯邦的忠實搭檔。

; {{nihongo|峰不二子|峰 不二子（みね ふじこ）|声：[[澤城美雪|澤城美雪]]（日本）；[[汪世瑋|汪世瑋]]（台灣）}}
: 身世如謎的天才女子，擁有出色的外表以及完美的比例，以自我慾望做為行動的第一原則。

; {{nihongo|銭形警部|銭形警部（ぜにがたけいぶ）|声：[[山寺宏一|山寺宏一]]（日本）；[[陳幼文|陳幼文]]（台灣）}}
: 國際刑警組織ICPO的魯邦三世搜查官，一心一意只想將魯邦逮捕到案。
