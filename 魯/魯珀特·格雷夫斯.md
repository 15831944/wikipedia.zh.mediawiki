'''魯珀特·格雷夫斯'''（{{lang-en|Rupert Graves}}，{{bd|1963年|6月30日}}）是一位英國电影、电视和舞台男演员。他在早期的《[[看得见风景的房间_(电影)|翡冷翠之戀]]》和《[[莫里斯的情人|情難禁]]》等作品中成名，近期的作品《[[神探夏洛克|新福爾摩斯]]》中扮演的[[雷斯垂德探長|雷斯垂德探長]]把他的知名度進一步推高。

== 早年生活 ==

格雷夫斯出生于[[英格兰|英格兰]][[森麻實郡|-{zh-cn:萨默塞特; zh-hk:森麻實; zh-tw:索美塞特;}-郡]][[:en:Weston-super-Mare|Weston-super-Mare]]，父亲是音乐老师和音乐家Richard Harding Graves，母亲是旅行社协调员Mary Lousilla Graves（婚前姓Roberts）<ref>{{Cite web |url = http://www.imdb.com/name/nm0001291/bio |title = Rupert Graves Biography |accessdate = 2015-04-29 }}</ref>，還有一個兄弟是貨車司機<ref>{{Cite web |url = http://www.theguardian.com/film/2002/apr/22/artsfeatures |title = Rupert bared |accessdate = 2015-04-29 }}</ref>。他的右腿天生有點問題<ref>{{Cite web |url = http://www.theguardian.com/lifeandstyle/2015/jan/03/rupert-graves-q-and-a-sherlock |title = Q&A: Rupert Graves |accessdate = 2015-04-29 }}</ref>。<br><br>

格雷夫斯早年的教育並不理想，小學的時候老師在全班面前取笑他的名字<ref>{{Cite web |url = http://www.theguardian.com/lifeandstyle/2015/jan/03/rupert-graves-q-and-a-sherlock |title = Q&A: Rupert Graves |accessdate = 2015-04-29 }}</ref>，讓他非常難堪，長大後他一度想把名字改成Michael<ref>{{Cite web |url = http://www.theguardian.com/film/2002/apr/22/artsfeatures |title = Rupert bared |accessdate = 2015-04-29 }}</ref>。他十五歲就離開學校，十六歲時參加政府的青少年培訓計劃並成為一個馬戲團小丑<ref>{{Cite web |url = http://www.telegraph.co.uk/culture/tvandradio/11307980/rupert-graves-interview-last-tango-in-halifax.html |title = Rupert Graves: 'If I need cash I'll do anything, I don't really care' |accessdate = 2015-04-29 }}</ref>。<br><br>

== 事業 ==

格雷夫斯由在家鄉參演音樂劇開始他的演員生涯。十八歲搬到倫敦定居，參演了小眾電視劇Return Of The Saint，很快受到賞識並得到改編自五小冒險故事系列中的Five Go Down To The Sea劇集中戲份較重的角色，由此得到不少電視劇及電影的參演機會。年輕的他本來帶著很重的西部口音，在剛開始參與[[翡冷翠之戀|翡冷翠之戀]]的時候常常因為口音問題而要重拍<ref>{{Cite web |url = http://www.theguardian.com/film/2002/apr/22/artsfeatures |title = Rupert bared |accessdate = 2015-04-29 }}</ref>。但後來他克服了問題，近年常為BBC的紀錄片做旁白工作。<br><br>

格雷夫斯活躍於舞台劇，電影及電視劇。他拍過不少歐洲古裝劇，包括[[翡冷翠之戀|翡冷翠之戀]]，[[莫里斯的情人|情難禁]]，查理二世，[[神經大帝|神經大帝]]等電影及電視劇，所以他的古裝扮相非常深入民心。另一方面，他也喜歡嘗試不同的角色，例如在電視劇[[:en:Open_Fire_(British_film)|Open Fire]]中飾演易服癖的反派，電影[[莫里斯的情人|情難禁]]和舞台劇[[:en:Torch_Song_Trilogy|Torch Song Trilogy]]等作品中飾演同性戀角色，在電視劇[[:en:Law_&_Order:_UK|Law & Order: UK]]中飾演精神病人等。由於[[莫里斯的情人|情難禁]]在國際中大受歡迎，有不少為了與眾不同的性向而痛苦的同性戀者寫信給他，認為這套電影安撫了他們，而其中也有不少日本少女寫信告訴他很喜歡這套電影<ref>{{Cite web |url = https://www.youtube.com/watch?v=MtPNyKHvO9s |title = Rupert Graves talks about Scudder in Maurice |accessdate = 2015-04-29 }}</ref>。他在電影及電視劇演出成就非凡，在1996年主演的兩套電影－－分別是[[:en:Intimate_Relations_(1996_film)|Intimate Relations]]及[[:en:Different_for_Girls|Different for Girls]]，在同一年的[[蒙特利爾世界電影節|蒙特利爾世界電影節]]分別得到最佳電影及最佳男演員的殊榮。<br><br>

他主演過不少在倫敦[[西区剧院|西區劇院]]及[[百老匯劇院|百老匯劇院]]上演的舞台劇，其中多部得到評論界的高度讚譽和提名獎項，並廣受觀眾歡迎。單是他作為男主角Dan在[[百老匯劇院|百老匯劇院]]上演的誘心人（[[:en:Closer_(play)|Closer]]），一年內上演次數高達172次，他本人得到[[:en:Outer_Critics_Circle_Award|Outer Critics Circle Award]]的特別獎，此劇亦獲提名[[:en:Outer_Critics_Circle_Award|Outer Critics Circle Award]]最佳舞台劇，最佳導演，以及[[:en:Tony_Award_for_Best_Play|Tony Award for Best Play]]最佳舞台劇。可惜的是，在最近的訪問中，他表示因為家庭因素，最近沒有參演舞台劇的打算。<br><br>

== 個人生活 ==

1987年，格雷夫斯在家鄉[[:en:Weston-super-Mare|Weston-super-Mare]]一家咖啡廳邂逅彩繪玻璃藝術家（後來成為園藝家）Yvonne<ref>{{Cite web |url = http://www.standard.co.uk/news/the-secret-life-of-rupert-graves-6305043.html |title = The secret life of Rupert Graves |accessdate = 2015-04-29 }}</ref>。他們後來在倫敦同居，格雷夫斯跟Yvonne一起撫養Yvonne兩名分別十歲和十四歲的女兒，這段關係維持了十三年。<br><br>

2000年，格雷夫斯在他主演的舞台劇The Caretaker的首演派對上邂逅澳洲出生的製作統籌Suzanne Lewis（暱稱Susie）。婚後，他們擁有五個孩子<ref>{{Cite web |url = http://www.dailymail.co.uk/femail/article-2059763/Why-I-said-fame-After-starring-A-Room-With-A-View-Rupert-Graves-tipped-Hollywood-greatness--having-it.html |title = 
Why I said no to fame: After starring in A Room With A View Rupert Graves was tipped for Hollywood greatness – but he was having none of it |accessdate = 2015-04-29 }}</ref>：Joseph, Ella, Noah, Isaac和Zoe。<br><br>

由於早年在膾灸人口的[[莫里斯的情人|情難禁]]中飾演同性戀的角色，很多人經常詢問格雷夫斯的性向。他認為年輕時他的樣子漂亮，所以有些比他年長的演員追求他<ref>{{Cite web |url = http://www.dailymail.co.uk/femail/article-2059763/Why-I-said-fame-After-starring-A-Room-With-A-View-Rupert-Graves-tipped-Hollywood-greatness--having-it.html |title = 
Why I said no to fame: After starring in A Room With A View Rupert Graves was tipped for Hollywood greatness – but he was having none of it |accessdate = 2015-04-29 }}</ref>。他年輕時曾嘗試過不同性向，也承認有過一段同性關係，但他因此確認自己喜歡的是女性<ref>{{Cite web |url = http://www.standard.co.uk/news/the-secret-life-of-rupert-graves-6305043.html |title = The secret life of Rupert Graves |accessdate = 2015-04-29 }}</ref>。<br><br>

格雷夫斯喜歡足球，他習慣每星期跟其他演員，製作人和編劇踢足球<ref>{{Cite web |url = http://www.telegraph.co.uk/news/features/3637854/My-perfect-weekend-Rupert-Graves.html |title = My perfect weekend: Rupert Graves |accessdate = 2015-04-29 }}</ref>，他喜歡[[阿仙奴足球會|阿仙奴]]球隊<ref>{{Cite web |url = http://www.rupert-graves.com/media/mb032.html |title = 2001 - Take A Lad Like Him. |accessdate = 2015-04-29 |deadurl = yes |archiveurl = https://web.archive.org/web/20160304133406/http://www.rupert-graves.com/media/mb032.html |archivedate = 2016-03-04 }}</ref>。
