'''邵式平'''（{{bd|1900年|1月|1965年|3月24日|catIdx=S}}）<ref>出生年份亦有作1899年，当属阴阳历换算的问题，见{{cite web |url=http://www.jx.xinhuanet.com/renwu/2005-04/26/content_4132746.htm |title=存档副本 |accessdate=2007-10-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20070427160536/http://www.jx.xinhuanet.com/renwu/2005-04/26/content_4132746.htm |archivedate=2007-04-27 }}</ref>，字守一，[[江西|江西]][[弋阳|弋阳]]曹溪镇邵家畈村人。[[中華人民共和國|中華人民共和國]]成立后[[江西省|江西省]]首任[[省长|省长]]。

== 革命岁月 ==

1916年就读弋阳县立高等小学，与同学[[方志敏|方志敏]]结为挚友。

1922年考入[[北京师范大学|北京师范大学]]史地系。

1925年在北师大加入[[中国社会主义青年团|中国社会主义青年团]]，同年转为[[中国共产党|中国共产党]]党员。

1928年1月与方志敏、[[黄道|黄道]]等领导[[弋横暴动|弋横暴动]]，创建[[赣东北革命根据地|赣东北革命根据地]]。

1930年7月，任[[红十军|红十军]]政委。

1931年11月后任[[中央革命军事委员会|中央革命军事委员会]]委员、闽赣省苏维埃主席、闽赣省委书记、闽赣军区代政委等职。

1931年、1934年两度被选为中华苏维埃共和国中央执行委员。

在[[长征|长征]]中曾担任纵队参谋长、政治部主任。

到达陕北后，任[[陕北公学|陕北公学]]教育长、[[抗日军政大学|抗日军政大学]]第二分校副校长、[[晋察冀边区|晋察冀边区]]政府粮食局局长等职。

[[抗日战争|抗日战争]]胜利后到东北，历任中共辽吉省委副书记兼军区副政委，嫩江省委副书记兼军区副政委，西满财经委员会副主任，东北财经委员会、计划委员会副主任。

== 主政江西 ==
[[中華人民共和國|中華人民共和國]]成立后，历任中共华东局委员，中共江西省委第二书记，[[江西省人民政府|江西省人民政府]]主席、江西省省长。在[[中共八大|中共八大]]上，当选为中央候补委员。   

邵式平的成绩首先体现在[[南昌|南昌]]的城市建设上。担任省长期间，兼任南昌市城市建设委员会首席主任委员。1951年，南昌市规划城市建设，他在城东废城墙基础上，开辟[[八一大道|八一大道]]。邵式平本想把八一大道建成81米宽、[[井冈山大道|井冈山大道]]建成60米宽的“快速公路”，战时可作临时飞机场，但当时有人认为没有必要修那样宽的大道，甚至一些省级领导也不同意，最终双方只能妥协，八一大道只有60多米宽，井冈山大道只有40多米宽。即使这样，[[八一大道|八一大道]]建成时也仅次于北京的[[长安街|长安街]]。而[[八一广场|八一广场]]建成时仅次于[[天安门广场|天安门广场]]。现在八一大道依然担当着南昌城市主干道的重要角色，当时在其两侧建成的[[江西宾馆|江西宾馆]]、[[江西饭店|江西饭店]]、[[南昌百货大楼|南昌百货大楼]]、[[江西艺术剧院|江西艺术剧院]]、[[江西革命烈士纪念堂|江西革命烈士纪念堂]]至今仍是南昌市标志性建筑。

邵式平还指出：“南昌的发展要建一江两岸，学[[武汉三镇|武汉三镇]]。南昌市如果向江北发展，空间非常大，也少占良田，一定要搞[[昌北科学城|昌北科学城]]……1954年[[八一大桥|八一大桥]]木头桥改建水泥桥时应是四车道，人车分行，或上下立体，而苏联专家设计的两车道人车混行是错误的……南昌原水面只能扩大不能缩小，南湖、北湖、东湖、[[贤士湖|贤士湖]]、[[青山湖|青山湖]]，要湖湖相通，路边、湖周边多栽樟树。南昌湖水相连、绿树成阴了，[[四大火炉|四大火炉]]的帽子也就摘掉了。”<ref>[http://jiangxi.jxnews.com.cn/system/2006/06/29/002285483.shtml 邵式平鲜为人知的故事]</ref>这些设想在当时无疑是超前的。现在，老八一大桥已于1990年代拆除重建，南昌也在大力发展[[赣江|赣江]]西岸[[红谷滩|红谷滩]]，而城内各湖，虽面积稍有缩小，但大体还是遵循当时设想。

邵式平另一举措是大力开发山区。1950年代，江西人口快速增长，邵式平考虑到本省特点山多田少，动员全省五万干部上山下乡，号召大家向荒山野岭、闲置湖泊开战、创业，并建立多个垦殖场。接着，他又创办“[[江西共产主义劳动大学|江西共产主义劳动大学]]”，分校遍布全省各地，这一创举后来蔓延到全国。这些举措保证当时江西粮食供应，在1950年代末，江西不但没有饥荒，而且还是粮食输出大省。当时周边数省有大量人口流入江西，大多由垦殖场和共大安置、接纳。<ref>[http://www.oldfriendm.com/topic.cfm?id=2669 八一大道的联想----江西省首任省长邵式平二、三事] {{webarchive|url=https://web.archive.org/web/20071017192415/http://oldfriendm.com/topic.cfm?id=2669 |date=2007-10-17 }}</ref>但这些举措也同时破坏环境，直接后果是水土加重流失，[[鄱阳湖|鄱阳湖]]水面日益减少，1990年代末的[[退耕还林|退耕还林]]、[[退田还湖|退田还湖]]正是对其的纠正。

1965年3月24日，因[[心脏病|心脏病]]在南昌逝世，終年65歲。
