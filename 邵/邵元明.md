'''邵元明'''（{{bd|1956年|12月||}}），[[山东|山东]][[德州市|德州]]人。[[中国人民解放军中将|中国人民解放军中将]]<ref name=cxw/>。

曾任[[中国人民解放军第二炮兵部队|中国人民解放军第二炮兵部队]]第53基地（驻地[[云南|云南]][[昆明|昆明]]）参谋长、司令员。在[[深化国防和军队改革|深化国防和军队改革]]中，2016年任[[中国人民解放军火箭军|中国人民解放军火箭军]]副参谋长。2017年1月，任[[中央军委联合参谋部|中央军委联合参谋部]]副参谋长<ref name=cxw>{{Cite web |url = http://china.caixin.com/2017-02-08/101051100.html |title =  火箭军副参谋长邵元明任军委联合参谋部副参谋长|author =  |date = 2017-02-08 |publisher =财新网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.qq.com/a/20170208/017158.htm |title = 军委联合参谋部再添一将：邵元明履新副参谋长 |author =  |date = 2017-02-08 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。2017年4月，兼任[[国家反恐领导小组|国家反恐领导小组]]副组长<ref>{{cite news|title=邵元明出任国家反恐领导小组副组长，刘跃进任国家反恐办主任|url=http://www.thepaper.cn/newsDetail_forward_1662381|accessdate=2017-04-16|newspaper=澎湃新闻}}</ref>。2018年7月，晋升中将军衔<ref>{{cite web |title=军委总部再添将星：军委联合参谋部副参谋长邵元明晋升中将 |url=https://www.thepaper.cn/newsDetail_forward_2321159 |website=澎湃新闻 |accessdate=2018-08-05}}</ref>。
