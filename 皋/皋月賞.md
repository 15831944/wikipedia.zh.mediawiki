'''皐月賞'''（全長2000米）是日本三冠第一關，別稱日本二千堅尼，通常於四月的第三週於[[中山競馬場|中山競馬場]]舉行，路程為二千米，首屆冠軍馬是ロツクパーク，騎師是稲葉幸夫。二零零七年皐月賞冠軍馬是victory，騎師是[[田中勝春|田中勝春]]。

==歷屆勝出馬匹==
1939年 : ロツクパーク

1940年 : ウアルドマイン

1941年 : Stlite

1942年 : Kuri Hikari

1943年 : ダイヱレク

1944年 : クリヤマト

1945年 : 因二戰而停止。

1946年 : 因二戰而停止。

1947年 : Tokitsukaze

1948年 : Hide Hikar

1949年 : Tosamidori

1950年 : Kumonohana
