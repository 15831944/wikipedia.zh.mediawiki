'''奉新县'''在[[江西省|江西省]]西北部，为[[宜春市|宜春市]]下辖的一个县。

== 历史沿革 ==
奉新县[[春秋|春秋]]时属[[吴|吴]]，[[战国|战国]]时属[[楚|楚]]，[[秦|秦]]属[[九江郡|九江郡]]，[[汉|汉]]初属[[豫章郡|豫章郡]]。

[[汉景帝|汉景帝]]三年（公元前154年）置海昏县。

[[和帝|和帝]][[永元|永元]]十六年（公元104年）分海昏置[[建昌县|建昌县]]。

[[灵帝|灵帝]][[中平|中平]]二年（公元185年）分[[海昏|海昏]]、[[建昌|建昌]]两地，设置新吴县，县治在今[[会埠|会埠]]故县，属[[豫章|豫章]]。

[[唐中宗|唐中宗]][[神龙|神龙]]二年（公元706年）县衙由故县迁至冯川。

[[南唐|南唐]][[保太|保太]]元年（公元973年）[[吴|吴]]把帝位让于唐，唐嫌“吴”字，遂改新吴为奉新。

公元[[1949|1949]]年7月8日，中共占领[[解放|解放]]。

公元1958年前属[[南昌专区|南昌专区]]管辖。

公元1959年1月始划归[[宜春地区|宜春地区]]管辖。

公元2000年8月[[宜春|宜春]]撤地设市，仍辖奉新县。
　　

== 地理 ==
位于[[修水|修水]]支流潦水上游；境内以[[山地|山地]]为主。县境三面环山，地势西高东低，西部中低山地、中部多丘陵、东部低丘河谷平原，属典型的丘陵山区地形地貌。
最高峰[[五梅山|五梅山]]（海拔1516.3米），最低点[[宋埠中堡|宋埠中堡]]（海拔27米），均匀海拔300米。[[潦河|潦河]]水流贯境内西东。
属中亚热带潮湿天气，年均匀气温为17.3℃，年均匀降雨量为1612毫米，年相对湿度均匀为79%，无霜期年均匀为260天左右，年日照时数达1803小时。
国家商品粮、毛竹林基地，省猕猴桃开发基地。矿藏有花岗石、瓷土、萤石、钾长石等。举铜、干大、寺棠、万修等公途经境。

== 交通 ==
226、232、313、314省道，[[昌铜高速公路|昌铜高速公路]]。

== 经济 ==
以农业为主，有一个江西省十大开发区之一的冯田经济开发区，目前正在建设另一个黄溪开发区，努力使奉新经济朝省内二十强努力，纺织，化工，制药原料，碳素材料产值占大头。

== 文化教育 ==
高中学校只有一所奉新一中和奉新冶城职业学校。
中学 奉新二中 华林中学 阳光中学
