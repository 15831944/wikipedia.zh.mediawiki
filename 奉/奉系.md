'''奉系'''是民国军阀派系，属于北洋系，先后以[[张作霖|张作霖]]和其子[[张学良|张学良]]為首，并一度成为中国境内最大的[[军政府|军政府]]。

20世紀初，奉系張作霖接受[[招安|招安]]。<ref name="奉張父子">[[唐德剛|唐德剛]]：〈從北京政變到[[皇姑屯事件|皇姑屯]]期間的奉張父子〉，刊唐德剛著、張學良口述：《張學良口述歷史》，[[台北|台北]]：[[遠流出版|遠流出版]]，2009年3月1日初版</ref>{{rp|363}}奉系受到[[日本|日本]]支持。1918年，张作霖統一滿洲。后联合皖系与[[孙中山|孙中山]]的广东政权组成反直三角同盟。

== 源流 ==
奉系创始人张作霖在其领内所建立的政权首府位于[[奉天|奉天]]（今[[瀋陽|瀋陽]]），奉系因此得名。张作霖先用計趕走了[[皖系|皖系]][[督軍|督軍]][[段芝貴|段芝貴]]，在[[奉天省|奉天省]]獨攬大權。

张作霖于1918年任[[东三省巡阅使、|东三省巡阅使]]。1919年8月5日，张作霖挤走[[吉林督军|吉林督军]][[孟恩远|孟恩远]]；[[鲍贵卿|鲍贵卿]]接任吉林督军，吉林地方军队纳入奉军系统。

1921年，张作霖任[[蒙疆经略使|蒙疆经略使]]，扩军。

1922年10月，奉系在[[第一次直奉战争|第一次直奉战争]]失败。奉军开始从一支土匪风气部队改为实行现代军事制度的军队。1922年7月24日成立'''东三省陆军整理处'''，[[孙烈臣|孙烈臣]]为统监，[[张作相|张作相]]、[[姜登选|姜登选]]为副监，[[张学良|张学良]]为参谋长。陆军整理处下设两个处：军学处，处长为陈瑛；工务处，处长为张宣。成立秘书、副官、军务、军需、军医、军法稽查等处，分管相关事宜。东三省统一编制，统一番号，均冠以“东三省陆军”称谓，不再使用各省省称番号。裁汰老、弱、病、残、劣的士兵6.9万人，整编为三个师，二十七个混成旅和五个骑兵旅共17万人。[[韩麟春|韩麟春]]为东三省军工厂总办，大办兵工厂。张学良为东三省航空处总办。设立东北海军总司令部，张作霖兼总司令，[[沈鸿烈|沈鸿烈]]任海军上将副总司令，代行总司令职。扩建“[[东北讲武堂|陆军东北讲武堂]]”，堂长由张作霖自兼；增设监督，由张学良兼任；萧其煊任教育长，增设步、骑、炮、工、辎各科，学习时间明确规定一年。

1924年秋，[[第二次直奉戰爭|第二次直奉戰爭]]中，奉系以二百萬[[日元|日元]]重賂，爭取馮玉祥對直系「倒戈」，發動「[[北京政變|北京政變]]」（10月23日），幽禁[[曹錕|曹錕]]，摧毀其「賂選政府」，由[[國務總理|國務總理]][[黃郛|黃郛]]「[[中華民國大總統|攝行大總統職]]」過渡。<ref name="奉張父子"/>{{rp|351}}同時奉軍在少帥[[張學良|張學良]]指揮下，突破直軍防線，在山海關、秦皇島一帶包圍吳佩孚主力，繳械納降3萬餘人。<ref name="奉張父子"/>{{rp|351}}吳率殘卒數千，浮海遁走，使奉系於「第二次直奉戰爭」中全勝。<ref name="奉張父子"/>{{rp|351}}为向南方进军，奉系扩编军队，改旅为师，并冠以“东北陆军”字样。

至此，奉系大舉入關。<ref name="奉張父子"/>{{rp|352}}編制是自「東北陸軍」中編出6個「[[軍|軍]]」，由[[姜登選|姜登選]]、[[李景林|李景林]]、張學良、[[張作相|張作相]]、[[吳俊陞|吳俊陞]]、[[許蘭洲|許蘭洲]]分任第一至第六軍[[軍長|軍長]]。<ref name="奉張父子"/>{{rp|358}}每軍下轄3至個「[[旅|旅]]」及若干「獨立團」，如[[砲兵|砲兵]]、[[工兵|工兵]]、[[後勤學|輜重兵]]等。<ref name="奉張父子"/>{{rp|358}}另加[[空軍|空軍]]、[[海軍|海軍]]由少帥張學良直接指揮。<ref name="奉張父子"/>{{rp|358}}6個軍總共20餘萬人。<ref name="奉張父子"/>{{rp|358}}第一軍、第三軍合組「一三聯軍司令部」，張學良為司令，第三軍副軍長兼第六混成旅[[旅長|旅長]][[郭松齡|郭松齡]]為副司令。<ref name="奉張父子"/>{{rp|358}}第二次直奉戰爭後，奉軍再次擴編，改旅為師，改軍為[[軍團|軍團]]。<ref name="奉張父子"/>{{rp|358}}1925年春，「一三聯軍司令部」改為「京榆駐軍司令部」，駐天津。<ref name="奉張父子"/>{{rp|358}}直轄[[步兵|步兵]]6師12旅，[[騎兵|騎兵]]1師2旅，砲兵2旅，工兵1團。<ref name="奉張父子"/>{{rp|358}}張學良任第三軍團軍團長兼司令，郭松齡任副司令。<ref name="奉張父子"/>{{rp|358}}共有步騎砲工輜各兵種7萬5千人。<ref name="奉張父子"/>{{rp|358}}

11月初，奉系主帥、東三省巡閱使[[張作霖|張作霖]]抵達[[天津|天津]]，會晤馮玉祥商討善後，決定公推閒居天津之前「[[皖系|皖系]]」[[軍閥|軍閥]][[段祺瑞|段祺瑞]]暫時擔任「[[中華民國臨時政府_(段祺瑞)|中華民國臨時執政]]」，以後另作安排。<ref name="奉張父子"/>{{rp|352}}11月24日，段在北京就職。<ref name="奉張父子"/>{{rp|352}}張作霖率千人入北京觀禮。<ref name="奉張父子"/>{{rp|352}}就職後，段要「廢督裁兵」，解除各省兵權。<ref name="奉張父子"/>{{rp|352}}張作霖乃自請撤銷「鎮威將軍府」，並解除「東三省巡閱使」名位，改任「東北邊防屯墾督辦」，仍「節制東三省軍務」。<ref name="奉張父子"/>{{rp|353}}

馮和張作霖主張電請[[孫中山|孫中山]]北上。<ref name="奉張父子"/>{{rp|354}}段乃附和，聯名電請孫文[[大元帥|大元帥]]北上，為和平統一，共商國是。<ref name="奉張父子"/>{{rp|354}}張作霖、孫中山原為老友。<ref name="奉張父子"/>{{rp|354}}二人兒子張學良、[[孫科|孫科]]乃「[[民國四公子|民國四大公子]]」，交情不惡。<ref name="奉張父子"/>{{rp|354}}

第二次直奉戰爭後，奉軍第五軍副軍長[[闞朝璽|闞朝璽]]奉命率2個奉軍混成旅及若干地方部隊，出任[[熱河特別區|熱河特別區]][[都統|都統]]，駐節[[承德|承德]]。<ref name="奉張父子"/>{{rp|359}}李景林奉命出任直隸督辦，駐節天津，轄奉軍及改編地方部隊6萬餘人。<ref name="奉張父子"/>{{rp|359}}

1928年被[[國民革命軍|北伐军队]]击败，张作霖在退出关外时于[[奉天|奉天]]郊区的皇姑屯三洞桥（今沈阳市皇姑区与和平区交界处）被炸死，引发[[皇姑屯事件|皇姑屯事件]]。其子[[张学良|张学良]]掌握兵权后，同年宣布[[东北易帜|东北易帜]]，奉系改编为东北边防军，进入新奉系时期。1936年西安事变后不久，张学良遭蒋中正扣押，随后东北军被国民政府逐步裁撤、收编，于1937年彻底消亡。

== 主要将领 ==
[[File:Zhang_Zuo-lin.png|200px]]]]

舊派：是隨張作霖招安而來之「元老派」，包括[[張作相|張作相]]、[[張景惠|張景惠]]、[[湯玉麟|湯玉麟]]、[[張海鵬|張海鵬]]、[[張宗昌|張宗昌]]、[[馬占山|馬占山]]、[[許蘭州|許蘭州]]等。元老派原是一群[[綠林|綠林]]豪傑。大部分人和张作霖是一樣出身。

新派：又有士官派和大學派之別。

士官（[[日本|日本]][[陸軍士官學校_(日本)|陸軍士官學校]]）派：元老派覺得需要充實一批留日歸國「士官派」。在1920年代，以[[楊宇霆|楊宇霆]]為首領。[[姜登選|姜登選]]為中堅，包括[[吳俊升_(民國軍政人物)|吴俊升]]、[[孙烈臣|孙烈臣]]、[[于學忠|于學忠]]、[[何柱國|何柱國]]、[[張驤伍|張驤伍]]等。

大學（[[北京|北京]][[陸軍大學|陸軍大學]]）派：奉軍中下級骨幹，由「陸軍大學」和「東北講武堂」訓練出來。以[[張學良|張學良]]、[[郭松齡|郭松齡]]為首領。[[李景林|李景林]]。曾任張學良老師的郭松齡尤為張學良所倚重，不過後來大學派等人反叛奉系。

一些人物如[[孙烈臣|孙烈臣]]、[[于學忠|于學忠]]、[[郭松齡|郭松齡]]、[[李景林|李景林]]等雖為新派，但同時為[[张作霖|张作霖]]效力己久，是新舊派過渡人物。
