'''奉聖夫人'''（{{bd|？||1681年||catIdx=F}}），官书作[[朴姓|朴氏]]，夫家之《辉发萨克达氏家谱》则作布母布哩氏。[[正黄旗|正黄旗]]内务府满洲第三佐领下人<ref>《辉发萨克达氏家谱》，载《北京图书馆藏家谱丛刊·民族卷》第38册，国家图书馆出版社，2002年版</ref>。[[清朝|清朝]]入关后，第一位被正式册封{{efn|[[顺治帝|顺治帝]]曾在顺治十七年（1660年），要求礼部追封乳母[[李氏_(顺治帝)|李氏]]。是否正式追封不详。}}的[[皇帝|皇帝]]乳母。清朝册封朴氏为“[[奉圣夫人|奉圣夫人]]”，完全是因袭[[明朝|明朝]]制度。

==生平 ==

關於朴氏的出身背景 ，在清朝正史中語焉不詳。布母布哩氏未被《八旗滿洲氏族通譜》收載 ，只能說明布母布哩氏的娘家身世卑微 ，名不見經傳。朴氏初嫁正黄旗[[包衣人|包衣人]]巴萨哩为妻子，巴萨哩早亡而没有赶上从龙入关的风光。因此，夫家辉发[[萨克达氏|萨克达氏]]是由朴氏的带领下从龙入关的。故而萨克达一支视朴氏为始迁祖母，她的[[衣冠塚|衣冠塚]]還在滿井祖塋裡佔據了主穴位置<ref>据徐广源《清东陵史话》，紫禁城出版社1997年版，第169—170页。朴氏墓在孝陵东马兰峪河东村东南处，遗址尚存。</ref>。

[[順治帝|順治帝]]出生不久，朴氏就以“'''溫惠之資'''”選入紫禁城，當上了順治帝的乳媼。朴氏克敬克慎，忠於職守，受到了皇宮上下的一致稱讚。[[順治|順治]]十一年（1654年）三月，順治帝的第三子[[玄燁|玄燁]]出生。看護玄燁的重擔落在了朴氏的頭上。朴氏勤慎盡職，對乳子女的盡心餵養、撫育和侍奉，使得身處不同階級的主僕群體之間產生了深厚的親情。而這種餵養和撫育也使乳保與乳子女之間除主僕關係外，還存在著倫理關係。

[[康熙帝|康熙帝]]后来回忆道：“'''朕躬幼時，殫心調護，夙夜殷勤。撫視周旋，身不離左右，提攜備至，時罔怠於寒暄'''。”這一方面反映出朴氏在撫育順康兩代幼主時所起到的重大作用<ref>{{Cite web |url=http://jds.cssn.cn/webpic/web/jdsww/UploadFiles/ztsjk/2010/12/201012020920159360.pdf |title=满洲皇帝与保母|accessdate=2019-01-04 |author=作者：刘小萌 |date=2010-11-25|work= |publisher=中国人民大学清史研究所网，來源：《北京社會科學》|language=Zh-hans }}</ref>，同時，也表明康熙帝是重感情的，能夠知恩必報。朴氏的继任丈夫哈喇，從各方面支持、配合其當好保姆，因此，後來被授為二等[[輕車都尉|阿達哈哈番]]世職。順治十一年（1659年）七月，哈喇去世。順治帝念他“保護朕躬有年，忠勤素著”，贈給他“忠襄”的諡號，還特為他建墳立碑。

[[顺治|顺治]]十六年（1659年），[[郑成功|郑成功]]率兵围攻[[南京|南京]]，而清军屡吃败仗。[[顺治帝|顺治帝]]既惊恐，又暴躁，便举剑砍断御案，声扬要御驾亲征。[[孝莊文皇后|皇太后]][[博爾濟吉特氏|博爾濟吉特氏]]只好转身退去，另遣福临之乳母朴氏出面劝阻<ref>魏特著《汤若望传》，北京商务印书馆出版，1949年版</ref>。处于暴怒、急躁状态下的顺治帝失去了理智。对于朴氏的苦劝，不仅不听，还想要用剑砍她。

[[康熙|康熙]]十六年（1677年），朴氏被封為奉聖夫人，頂帽服色照[[公爵|公]]夫人品級。[[康熙|康熙]]二十年（1681年）六月，朴氏因病溘然長逝。康熙帝回憶往昔，悲從中來，指示有關衙門，朴氏“應得恩卹，宜從優厚”。她的喪禮不僅按照[[公爵|公]]夫人之禮辦理<ref>《清聖祖實錄》，卷六十八，東北師範大學出版社出版，1985年版</ref>，還特地在[[清孝陵|孝陵]]附近賞給她一塊兆域，為她建墳立碑，並加祭一次。朴氏入葬後，据清实录和起居注记载，康熙帝先後四次親臨園墓給朴氏奠酒<ref>{{Cite web |url=http://www.cssn.cn/zgs/zgs_zggds/201702/t20170222_3425502_2.shtml |title=再论清代皇室之乳保|accessdate=2019-01-04 |author=作者：沈欣 |date=2017-02-22|work= |publisher=中国社会科学网，来源：《北京社会科学》|language=Zh-hans }}</ref>。朴氏两世保母，所哺又是二个幼年登基的皇帝，故所得封赠最隆，而其他保母和亲族未必能享受如此优渥的厚遇。
