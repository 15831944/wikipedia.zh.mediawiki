'''奉节县'''位于[[重庆市|重庆市]]东部，与[[云阳|云阳]]、[[巫山县|巫山]]、[[巫溪|巫溪]]、[[湖北|湖北]][[利川|利川]]、[[恩施|恩施]]相邻。县城驻于[[永安镇|永安镇]]，位于长江以北。面积4098.37平方千米，居重庆市第二位，2007年末户籍人口1,041,465，居重庆市第八位。奉节是[[三峡库区|三峡库区]]县中移民工作量最大、搬迁最难的县。

== 历史沿革 ==
* 前314年，[[秦国|秦国]]于[[巴国|巴国]]地置[[巴郡|巴郡]]，鱼复县随巴郡同置。
* 222年，[[刘备|刘备]]在[[夷陵之战|夷陵战败]]后退守[[白帝城|白帝城]]，改鱼复为永安县。
* 280年（[[西晋|西晋]][[太康_(西晉)|太康]]元年），恢复鱼复县名。
* 554年（[[北朝|北朝]][[西魏|西魏]][[废帝|废帝]]三年），改鱼复为人复县。
* 649年（[[唐朝|唐]][[貞觀_(唐朝)|贞观]]二十三年），为尊崇诸葛亮奉刘备“托孤寄命，临大节而不可夺”，改为奉节县。
* 1667年（[[清朝|清]][[康熙|康熙]]六年），裁[[巫溪县|大宁县]]（今[[巫溪县|巫溪县]]）并入奉节县；
* 1729年（清[[雍正|雍正]]七年），复设置大宁县。
* 1911年（清[[宣统|宣统]]三年十月初六），奉节响应[[辛亥革命|辛亥革命]]，宣布独立。
* 1935年，川政统一，[[四川|四川]]实行行政督察区制，原定第九行政督察区设在奉节，后因其匪乱改在[[万县|万县]]。
* 1949年12月3日，奉节和平解放。12月19日成立奉节县人民政府。1950年至1997年2月属四川万县地区。
* 1997年3月，[[重庆|重庆]]直辖后隶属重庆市。

== 地理 ==
奉节属四川盆地东部山地地貌，长江横贯中部，山峦起伏，沟壑纵横。境内山地面积占总面积的88.3%，中山（海拔1000米以上）占总面积80.01%，最高海拔吐祥猫儿梁为2123米，三峡工程蓄水前最低海拔瞿塘峡口为86米。奉节北部为大巴山南麓的一部分，东部和南部为巫山和七曜山的一部分，长江横切七曜山形成著名的瞿塘峡。地貌总体为东南、东北高而中部偏西稍平缓，南北约为对称分布，以长江为对称轴，离长江越远海拔越高，有少量平缓河谷平坝。境内长江干流41.5公里，有梅溪河、大溪河、石笋河、草堂河、朱衣河等主要河流。

=== 气候 ===
奉节县境属[[亚热带季风气候|亚热带季风气候]]，春早、夏热、秋凉、冬暖，四季分明，无霜期长，雨量充沛，日照时间长。境内山高谷深，海拔高度变化很大，受地形地貌影响，气候垂直变化较为明显，形成典型的立体气候。年均气温在海拔低于600米的地区高于16.4℃，600-1000米的地区为16.4℃-13.7℃，1000-1400米的地区为13.7℃-10.8℃，高于1400米的地区低于10.8℃。无霜期约287天，年均降水量1150毫米左右，年均日照时数1639小时。

== 行政区划 ==
* 1952年6月26日，奉節縣的龍門、楠楊、柏楊3鄉劃歸湖北省[[利川縣|利川縣]]<ref>1952年6月26日內務部【內民字第252號】批覆</ref>。

辖16个镇，10个乡，4个民族乡，共368个村、16个居委会：
* 镇：永安镇、羊市镇、白帝镇、草堂镇、汾河镇、康乐镇、大树镇、竹园镇、公平镇、朱衣镇、甲高镇、吐祥镇、青龙镇、兴隆镇、新民镇、永乐镇。
* 乡：岩湾乡、平安乡、新政乡、红土乡、石岗乡、康坪乡、安坪乡、太和乡、五马乡、鹤峰乡、冯坪乡。
* 民族乡：云雾土家族乡、龙桥土家族乡、长安土家族乡。

== 经济 ==
* 2008年，全县[[国内生产总值|国内生产总值]]达到71.5亿元，财政收入达到3.85亿元。

== 事件 ==
陈茂国（被称为“鸟人”）是奉节县[[朱衣镇|朱衣镇]]帽峰村二组村民，2008年12月，他家1200多平方米因修建渝宜高速公路连接道被强拆，在补偿金问题上与当地政府未能达成一致。2009年8月3日，陈茂国爬上自家15米高的[[桉树|桉树]]“安营扎寨”。同年11月18日，政府答应了陈茂国80万元的赔偿要求，陈茂国下树，当天下午陈茂国被当地警方带走并被刑拘<ref>{{cite news|author=《云南信息报》|title=重庆“鸟人”陈茂国受审(组图)|url=http://news.163.com/10/0629/06/6AATVLDG00014AED.html|accessdate=2012-09-23|newspaper=河南商报|date=2010-06-29|publisher=网易|language=zh-hans}}</ref>。2010年9月3日，奉节县人民法院以聚众扰乱交通秩序罪判处陈茂国有期徒刑3年<ref>{{cite news|author=万州站记者黄金华 通讯员赵华、孙晓林|title=重庆“鸟人”陈茂国犯聚众扰乱交通秩序罪获刑３年|url=http://cq.cqnews.net/shxw/shwx/201009/t20100903_4576656.htm|accessdate=2012-09-23|newspaper=华龙网|date=2010年9月3日|publisher=重庆日报|location=重庆|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20101129212737/http://cq.cqnews.net/shxw/shwx/201009/t20100903_4576656.htm|archivedate=2010年11月29日}}</ref>。
