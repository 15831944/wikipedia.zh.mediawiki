'''伊莎贝尔·卢卡斯'''（{{lang-en|'''Isabel Lucas'''}}，{{bd|1985年|1月29日|||catIdx=L}}）是澳大利亚女演员，因出演《聚散离合》而知名。

== 生平 ==
1985年，卢卡斯出生在[[澳大利亚|澳大利亚]][[墨尔本|墨尔本]]。<ref name="wwd">{{cite news| title = Coming to America: Isabel Lucas| author = Marcy Medina| date = 18 June 2009| url = http://www.wwd.com/lifestyle-news/eye/coming-to-america-isabel-lucas-2174619?gnewsid=78e79bbedf59ad8c394d92dc019ebe59| work = WWD (Women's Wear Daily) Issue 06/18/2009| accessdate = 24 January 2010}}</ref>

<ref name="seven">{{cite web| title = Meet the Cast Isabel Lucas| url = http://www.seven.com.au/homeandaway/meetthecast_isabel-lucas | work = Home and Away – Yahoo!7|deadurl = yes| archiveurl = https://web.archive.org/web/20051128020424/http://seven.com.au/homeandaway/meetthecast_isabel-lucas| archivedate = 2005年11月28日}}</ref>
母亲阿特丽斯，是一名教师；父亲安德鲁，是一名飞行员。<ref>{{cite web| url =http://isabellucasinspired.org/press/news.php?newsid=28|title = isabellucasinspired| deadurl = yes| archiveurl =http://archive.is/6kNsz| archivedate =2013年4月15日}}</ref> 她的父亲是澳大利亚人，母亲是瑞士人，她会说[[德语|德语]]、[[法语|法语]]和[[澳大利亚英语|澳大利亚英语]]。<ref name="seven"/>
孩提时期，卢卡斯住在[[昆士兰|昆士兰]]凯恩斯。也曾住在瑞士和卡卡杜。<ref name="style.com">{{cite web| url = http://men.style.com/details/features/landing?id=content_9357| title = Transformers Star Isabel Lucas| publisher = Men.Style.com| page = 2| accessdate = 24 January 2010| deadurl = yes| archiveurl = https://web.archive.org/web/20090831235503/http://men.style.com/details/features/landing?id=content_9357| archivedate = 2009年8月31日}}</ref>
卢卡斯曾在澳大利亚昆士兰州凯恩斯的圣莫尼卡学院读书。
