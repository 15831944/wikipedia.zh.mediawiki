'''伊奧利亞沼'''（'''Aeolis Palus'''）是位於[[火星|火星]][[蓋爾撞擊坑|蓋爾撞擊坑]]西北方坑壁和[[夏普山_(火星)|夏普山]]西北方山麓之間的平原區域，中心座標{{coord|4.47|S|137.42|E|globe:Mars}}<ref name="IAU-20120516" />。

美國NASA的[[火星科學實驗室|火星科學實驗室]]任務於2012年8月將[[好奇號|好奇號]][[火星車|火星車]]送到該區域進行[[行星科學|行星科學]]探測任務，並在登陸後不久開始進行。

==太空探測==
2012年8月5日10:32 p.m. [[太平洋时区|PDT]]（任務時間，2012年8月6日5:32 UTC），[[喷气推进实验室|喷气推进实验室]]的任務控制中心接收到了火星科學實驗室發出代表成功降落在伊奧利亞沼內"Yellowknife" ''Quad 51''區的訊號<ref name="NASA-20120810">{{cite web |authors=[[NASA|NASA Staff]] |title=Curiosity's Quad - IMAGE |url=http://mars.jpl.nasa.gov/msl/multimedia/images/?ImageID=4408 |date=August 10, 2012 |publisher=[[NASA|NASA]] |accessdate=August 11, 2012 }}</ref><ref name="NASA-20120809">{{cite web |last1=Agle |first1=DC |last2=Webster |first2=Guy |last3=Brown |first3=Dwayne |title=NASA's Curiosity Beams Back a Color 360 of Gale Crate |url=http://www.nasa.gov/mission_pages/msl/news/msl20120809.html |date=August 9, 2012 |publisher=[[NASA|NASA]] |accessdate=August 11, 2012 }}</ref><ref name="BBC-20120809">{{cite news |last=Amos |first=Jonathan |title=Mars rover makes first colour panorama |url=http://www.bbc.co.uk/news/science-environment-19201742 |date=August 9, 2012|newspaper=[[BBC_News|BBC News]] |accessdate=August 9, 2012 }}</ref><ref name="USA-20120809">{{cite news |last=Halvorson |first=Todd |title=Quad 51: Name of Mars base evokes rich parallels on Earth |url=http://www.usatoday.com/tech/science/space/story/2012-08-09/mars-panorama-curiosity-quad-51/56922978/1 |date=August 9, 2012 |newspaper=[[USA_Today|USA Today]] |accessdate=August 12, 2012 }}</ref><ref name="NASA-20120806">{{cite web |author=NASA Staff|title=NASA Lands Car-Size Rover Beside Martian Mountain|url=http://www.jpl.nasa.gov/news/news.cfm?release=2012-230|date=6AUG2012 |publisher=[[NASA|NASA]] |accessdate=6AUG2012-1}}</ref>。好奇號火星車的任務將特別集中在蓋爾撞擊坑內伊奧利亞沼的登陸地點附近，接著就是前往夏普山山腳附近探測地質和地層<ref name="NASA-20120327" /><ref name="NASA-20120328" /><ref name="Space-20120329" /><ref name="NASA-20120806-2">{{cite web |author=NASA Staff|title=Mars Science Laboratory Landing Press Kit|url=http://www.jpl.nasa.gov/news/press_kits/MSLLanding.pdf |date=JUL2012 |publisher=[[NASA|NASA]] |accessdate=6AUG2012}}</ref>。
