'''伊恩·迈克尔·沃克'''（{{lang-en|'''Ian Michael Walker'''}}，{{bd|1971年|10月31日|catIdx=Walker, Ian Michael}}），是一名已退役的[[英格兰|英格兰]]足球运动员，他的职业生涯大部分时间都是在[[托特纳姆热刺足球俱乐部|托特纳姆热刺]]、[[莱斯特城足球俱乐部|莱斯特城]]和[[博尔顿足球俱乐部|博尔顿]]度过。他在退役后成为一名教练，现在担任[[中国足球超级联赛|中国足球超级联赛]]球队[[上海东亚足球俱乐部|上海上港]]的守门员教练。

== 俱乐部生涯 ==
1989年，沃克在英甲球队（现[[英超|英超]]）[[托特纳姆热刺足球俱乐部|托特纳姆热刺]]开始职业生涯，1990年曾被短暂租借到英乙球队（现英冠）[[牛津联足球俱乐部|牛津联]]和[[伊普斯维奇城足球俱乐部|伊普斯维奇]]锻炼。他为热刺效力了11个赛季，总出场次数达312次，期间也数次入选[[英格兰足球代表队|英格兰代表队]]。2000/01赛季，沃克在竞争中输给[[尼尔·沙利文|-{zh-hans:尼尔·沙利文;zh-hk:尼爾·蘇利雲;zh-tw:尼爾·沙利文;}-]]，失去主力守门员的位置，他在2000年9月正式提出转会要求。<ref name="Spurs accept Walker transfer request">{{cite news|title = Spurs accept Walker transfer request|url = http://news.bbc.co.uk/sport2/hi/football/teams/t/tottenham_hotspur/926711.stm|publisher = BBC Sport|date = 2000-09-15|accessdate = 2010-02-15}}</ref>

2001年7月，沃克和英超球队[[莱斯特城足球俱乐部|莱斯特城]]签约四年，转会金额为250万[[英镑|英镑]]，外加50万英镑的附加条款。<ref name="Leicester sign Walker">{{cite news|title = Leicester sign Walker|url = http://news.bbc.co.uk/sport2/hi/football/teams/l/leicester_city/1430128.stm|publisher = BBC Sport|date = 2001-07-09|accessdate = 2010-02-15}}</ref>但他在2001/02赛季没有帮助球队保级成功，<ref name="Man Utd relegate Leicester">{{cite news|title = Man Utd relegate Leicester|url = http://news.bbc.co.uk/sport2/hi/football/eng_prem/1911281.stm|publisher = BBC Sport|date = 2002-04-06|accessdate = 2010-02-15}}</ref>也导致他失去被选入英格兰代表队参加[[2002年世界杯|2002年世界杯]]的希望。<ref name="Relegation ruins Walker's hopes">{{cite news|title = Relegation ruins Walker's hopes|url = http://news.bbc.co.uk/sport2/hi/football/teams/l/leicester_city/1957997.stm|publisher = BBC Sport|date = 2002-04-29|accessdate = 2010-02-15}}</ref>沃克帮助莱斯特城在第一个赛季成功升级回英超联赛，并在2003年5月时隔六年后再次赢得英格兰代表队的召唤。<ref name="Ian Walker">{{cite news|title = Ian Walker|url = http://news.bbc.co.uk/sport2/hi/football/internationals/england/3020581.stm|publisher = BBC Sport|date = 2004-05-17|accessdate = 2010-02-15}}</ref>

但重返英超赛场的沃克却处境艰难，特别是在2004年1月31日和[[阿斯顿维拉足球俱乐部|阿斯顿维拉]]的一场英超联赛中，他在18分钟内连丢5球。<ref name="Leicester 0-5 Aston Villa">{{cite news|title = Leicester 0-5 Aston Villa|url = http://news.bbc.co.uk/sport2/hi/football/eng_prem/3417881.stm|publisher = BBC Sport|date = 2004-01-31|accessdate = 2010-02-15}}</ref>比赛中有莱斯特城球迷冲进场内，告诉沃克说他不配当一名英超球员。<ref name="Walker escapes FA action">{{cite news|title = Walker escapes FA action|url = http://news.bbc.co.uk/sport2/hi/football/eng_prem/3448795.stm|publisher = BBC Sport|date = 2004-02-02|accessdate = 2010-02-15}}</ref> 2004年2月10日，沃克在和[[博尔顿足球俱乐部|博尔顿]]的比赛中打进一个非常离奇的乌龙球，导致莱斯特城被对手1比1逼平。<ref name="Leicester 1-1 Bolton">{{cite news|title = Leicester 1-1 Bolton|url = http://news.bbc.co.uk/sport2/hi/football/eng_prem/3456283.stm|publisher = BBC Sport|date = 2004-02-10|accessdate = 2010-02-15}}</ref>但在2月28日，他在莱斯特城主场0比0战平[[伍尔弗汉普顿流浪足球俱乐部|狼队]]表现出色，<ref name="Leicester 0-0 Wolves">{{cite news|title = Leicester 0-0 Wolves|url = http://news.bbc.co.uk/sport2/hi/football/eng_prem/3507615.stm|publisher = BBC Sport|date = 2004-02-28|accessdate = 2010-02-15}}</ref>以一个“世界级”的扑救扑出[[肯尼·米勒|-{zh-hans:肯尼·米勒;zh-hk:堅尼·米拿;zh-tw:肯尼·米勒;}-]]的射门。<ref name="Cotterill praises Walker">{{cite news|title = Cotterill praises Walker|url = http://news.bbc.co.uk/sport2/hi/football/teams/l/leicester_city/3509713.stm|publisher = BBC Sport|date = 2004-02-28|accessdate = 2010-02-15}}</ref>在莱斯特城降级到[[英冠|英冠]]后，沃克选择留在球队。<ref name="Walker makes Foxes pledge">{{cite news|title = Walker makes Foxes pledge|url = http://news.bbc.co.uk/sport2/hi/football/teams/l/leicester_city/3633323.stm|publisher = BBC Sport|date = 2004-04-16|accessdate = 2010-02-15}}</ref>但一个赛季后，由于莱斯特城无力支付他的新合同，2005年5月6日，沃克离开莱斯特城。<ref name="Levein rues losing star players">{{cite news|title = Levein rues losing star players|url = http://news.bbc.co.uk/sport2/hi/football/teams/l/leicester_city/4547531.stm|publisher = BBC Sport|date = 2005-05-14|accessdate = 2010-02-15}}</ref>

成为自由球员的沃克很快和博尔顿签约一年。他在博尔顿俱乐部历史上首次參加欧洲比赛的次回合——客场对阵保加利亚球队[[普罗夫迪夫火车头足球俱乐部|普罗夫迪夫火车头]]的比赛中首发出场。虽然他在2005/06赛季中只是担任[[尤西·積亞基連倫|-{zh-hans:尤西·耶斯凯莱伊宁;zh-hk:尤西·積亞基連倫;zh-tw:尤西·耶斯凱萊伊寧;}-]]的副車，联赛中没有得到一分钟的出场时间，但依然在2006年5月得到一份为期两年的合同。2008年5月，一直没有得到联赛出场机会的沃克再次得到续约，但在2008年12月11日，沃克在和博尔顿协商提前终止合同。

== 国家队生涯 ==
沃克曾经为[[英格兰足球代表队|英格兰代表队]]出场4次。1996年，他在英格兰和[[匈牙利国家足球队|匈牙利国家队]]的友谊赛中首次为国家队上阵。他被公认最蹩脚的比赛是[[1998年世界杯|1998年世界杯]]外围赛英格兰在[[温布利球场|温布利球场]]0比1不敌[[意大利国家足球队|意大利]]，很多人认为他必须为[[吉安弗兰科·佐拉|佐拉]]的进球负责。沃克曾入选英格兰代表队参加[[1996年欧洲足球锦标赛|1996年欧洲足球锦标赛]]和[[2004年欧洲足球锦标赛|2004年欧洲足球锦标赛]]，但都只是后备球员，没有得到上场机会。2004年6月5日，沃克时隔7年后再次代表国家队出场，他在61分钟替补[[保羅·羅賓遜_(1979年出生)|保羅·羅賓遜]]出场，帮助英格兰队6比1击败[[冰岛国家足球队|冰岛]]，这也是他的最后一场国家队比赛。<ref name="Clockwatch: England 6-1 Iceland">{{cite news|title = Clockwatch: England 6–1 Iceland|url = http://news.bbc.co.uk/sport2/hi/football/euro_2004/england/3779019.stm|publisher = BBC Sport|date = 2004-06-05|accessdate = 2010-02-16}}</ref>

== 教练生涯 ==
2011年3月14日，沃克被任命为[[英格蘭足球議會北部聯賽|英格蘭足球議會北部聯賽]]球队[[彼谢普斯托福足球俱乐部|彼谢普斯托福]]的主教练。但2011年12月11日，在彼谢普斯托福0比5惨败给[[英格兰足球依斯米安联赛|依斯米安联赛]]球队[[卡尔夏登竞技足球俱乐部|卡尔夏登]]的[[英格兰足总锦标|英格兰足总锦标]]后，沃克被辞退。2012年4月，他接受博尔顿前队友[[尼古拉·阿内尔卡|-{zh-hans:尼古拉·阿内尔卡;zh-hk:尼古拉·安歷卡;zh-tw:尼古拉·安耐卡;}-]]的邀请，成为[[中国足球超级联赛|中国足球超级联赛]]球队[[上海申花|上海申花]]的守门员教练。<ref>{{cite news|title = 申花正式宣布与蒂加纳解约官方宣布新教练组成员 |url = http://sports.sina.com.cn/j/2012-04-26/18566039397.shtml|publisher = sina.com |date = 2012-04-26|accessdate = 2012-05-03}}</ref>在他的指导下，上海申花主力门将[[王大雷|王大雷]]取得很大的进步，并在时隔6年后再次入选[[中国国家足球队|中国国家足球队]]。2014年1月2日，沃克成为上海另一支中超球队[[上海东亚足球俱乐部|上海东亚]]的守门员教练。<ref>{{cite news|title = 官方公告：东亚俱乐部与前英格兰国门伊恩沃克签约|url = http://ssfl.online.sh.cn/sports/gb/content/2014-01/02/content_6612297.htm|publisher = 上海东亚足球俱乐部|date = 2014-01-02|accessdate = 2014-01-02|deadurl = yes|archiveurl = https://web.archive.org/web/20140102195814/http://ssfl.online.sh.cn/sports/gb/content/2014-01/02/content_6612297.htm|archivedate = 2014-01-02}}</ref>上海东亚的主力门将[[颜骏凌|颜骏凌]]在他的调教下取得很大进步，为了避免沃克被挖角，上海东亚在2014年5月15日即和他续约至2017年底。<ref>{{cite news|title=官方公告：东亚俱乐部与伊恩沃克续约至2017赛季! |url=http://eafc.online.sh.cn/sports/gb/content/2014-05/15/content_6870650.htm |publisher=上海东亚足球俱乐部 |date=2014-05-15 |accessdate=2014-06-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20140517131214/http://eafc.online.sh.cn/sports/gb/content/2014-05/15/content_6870650.htm |archivedate=2014-05-17 }}</ref>

== 个人 ==
伊恩·沃克的父亲[[麦克·沃克|麦克·沃克]]是一名[[威尔士|威尔士]]球员，同样司职守门员，他在[[沃特福德足球俱乐部|沃特福德]]效力期间生下伊恩。麦克·沃克退役后曾经担任[[诺里奇城足球俱乐部|诺里奇城]]和[[埃弗顿足球俱乐部|埃弗顿]]等球队主教练。

伊恩·沃克的妻子是前《[[太阳报_(英国)|太阳报]]》[[三版女郎|三版女郎]]苏琪（Suzi），居住在[[萨里郡|萨里郡]]科巴姆的豪宅。<ref>{{cite news| url=http://www.telegraph.co.uk/property/main.jhtml?xml=/property/2006/10/17/ppast17.xml | work=The Daily Telegraph | location=London | title=Living in the past (with all mod cons) | first=Nicola | last=Venning | date=2006-10-17 | accessdate=2010-05-04}}</ref> 1998年，沃克的女儿索菲（Sophie）诞生，但苏琪却因此发生[[分娩|产后子痫]]，<ref>[http://www.iconocast.com/News_Files/HNews23X23_X_05/Health7.htm Info on Health, Medical News, Healthy Living, Healthy Eating, Health News, Diet, Cancer, Cholesterol, Depression, Kidney, Migraine, Smoking<!-- Bot generated title -->] {{webarchive|url=https://web.archive.org/web/20070927120927/http://www.iconocast.com/News_Files/HNews23X23_X_05/Health7.htm |date=2007-09-27 }}</ref>之后又发展为[[慢性疲劳症候群|慢性疲劳症候群]]。<ref>[http://www.celebagents.co.uk/Celebs/43/Suzi_Walker.html Suzi Walker – Celebrity Agents, International Celebrity Agents, Personal Management & PR To TV/Radio and Sports Stars<!-- Bot generated title -->] {{webarchive|url=https://web.archive.org/web/20061011221633/http://www.celebagents.co.uk/Celebs/43/Suzi_Walker.html |date=2006-10-11 }}</ref> 2006年11月，苏琪承认沃克和33岁的美国舞者萨曼塔·瑟曼（Samantha Thurman）发生婚外情。<ref>{{cite news | url=http://www.thesun.co.uk/article/0,,2-2006510148,00.html | location=London | work=The Sun | title=Bolton keepers blonde catch | date=2007-05-17 | deadurl=yes | archiveurl=https://web.archive.org/web/20070129095855/http://www.thesun.co.uk/article/0%2C%2C2-2006510148%2C00.html | archivedate=2007-01-29 }}</ref> 2007年9月，沃克和瑟曼生下一个儿子贾克逊（Jaxson），他们一度生活在南佛罗里达，随着沃克工作的变动，现在一起生活在上海。
