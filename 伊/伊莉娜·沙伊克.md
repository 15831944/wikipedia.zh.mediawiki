'''伊蓮娜·瓦萊里耶夫娜·沙伊克里斯拉莫娃'''（{{lang-en|'''Irina Valeryevna Shaykhlislamova'''}}，{{lang-ru|'''Ири́на Вале́рьевна Шайхлисла́мова'''}}，{{bd|1986年|1月6日|||catIdx= S沙伊克}}）[[俄羅斯|俄羅斯]][[模特兒|模特兒]]，以化名'''伊莉娜·沙伊克'''（Irina Shayk）聞名，1986年出生于[[叶曼热林斯克|叶曼热林斯克]]市，在[[美国|美国]]发展后进入[[时尚|时尚]]圈，现为知名的内衣模特兒。

2007年，伊蓮娜·沙伊克为《[[体育画报|体育画报]]》拍摄[[泳衣|泳衣]]写真后一举成名。<ref>{{cite web |url=http://ent.ifeng.com/zz/detail_2011_08/05/8198168_0.shtml|title=《文化·星态》|accessdate=2011年8月5日|work=|publisher=凤凰网|language= zh-hans}}</ref>

== 职业生涯 ==
从担任模特后，拍摄过[[时尚_(杂志)|Vogue]] Mexico、[[时尚_(杂志)|Vogue]] Espana、[[时尚_(杂志)|Vogue]] Brasil封面，其他拍摄过封面的杂志还有Vs. Magazine、[[柯夢波丹|Cosmopolitan]]、[[Maxim_(magazine)|Maxim]]、[https://en.wikipedia.org/wiki/Glamour_(magazine) Glamour]、[[美麗佳人|Marie Claire]]、[[时尚芭莎|Harper's Bazaar]]、[[GQ|GQ]]、[https://en.wikipedia.org/wiki/Esquire_(magazine) Esquire]、[[Elle|Elle]]、[[Cosmopolitan|Cosmopolitan]]、[https://en.wikipedia.org/wiki/Allure_(magazine) Allure] 、7 Hollywood Magazine。

走过Kanye West、[https://en.wikipedia.org/wiki/Jeremy_Scott Jeremy Scott]、[https://en.wikipedia.org/wiki/Givenchy Givenchy]、[https://en.wikipedia.org/wiki/Philipp_Plein Philipp Plein]时装秀。

在2007至2010年成为意大利内衣品牌[https://en.wikipedia.org/wiki/Intimissimi Intimissimi]的广告代言人，2008年及2009年成为[https://en.wikipedia.org/wiki/Guess_(clothing) Guess]品牌的全新年度代言人。 
从2007年起至今，每年皆出现在[[体育画报|体育画报]][[運動畫刊泳裝特輯|泳装特刊]]里，2011年3月更成为当年度封面模特，也是第一位登上体育画报泳装特刊的俄罗斯模特。
2010年成为[[亞曼尼|Armani Exchange]]春夏代言人，拍摄[[肯伊·威斯特|Kanye West]] - Power音乐录影带。
2011年，成为西班牙鞋履品牌Xti代言人。
2012年再度代言意大利内衣品牌[https://en.wikipedia.org/wiki/Intimissimi Intimissimi]，并在同年拿下[[雅芳|AVON]]全球彩妆并代言至2015年。
2013年拍摄由[[Penelope_Cruz|Penelope Cruz]]导演的“大内密探” ([https://en.wikipedia.org/wiki/Agent_Provocateur_(lingerie) Agent Provocateur]，简称AP) 女性内衣副牌微电影广告，广告中[[Penelope_Cruz|Penelope Cruz]]的丈夫[[Javier_Bardem|Javier Bardem]]及妹妹[https://en.wikipedia.org/wiki/M%C3%B3nica_Cruz Mónica Cruz]也有客串出演。
2014年擔任俄羅斯冬奧開幕式舉牌人員，同年七月拍攝的電影"[[宙斯之子：赫拉克勒斯|Hercules]]"上映。
2015年担任太阳眼镜品牌[http://int.lindafarrow.com/ Linda Farrow]春夏代言人。同年十月成为[https://en.wikipedia.org/wiki/L%27Or%C3%A9al L'Oréal Paris]国际代言人。

== 个人生活 ==
她的前男朋友為足坛巨星[[C·罗纳尔多|C·罗纳尔多]]。在2010年[[南非世界杯|南非世界杯]]开幕前，C羅带着伊蓮娜去[[意大利|意大利]]海边度假三天，而两人戏水的照片也被八卦媒体曝光。2015年C罗获得金球奖，伊蓮娜并未陪伴，引出分手传言。2015年1月18日，双方证实分手传言。为这段5年感情画上了并不完美的句号。2015年，開始和演員[[布萊德利·庫柏|布萊德利·庫柏]]交往。<ref>{{cite web |url=http://www.felady.com/ent/0704/00005051.html|title=《C罗准娇妻 俄罗斯名模伊蓮娜性感写真》|accessdate=2010-07-04|work=|publisher=菲风网|language= zh-hans}}</ref>

== 成就荣誉 ==
伊蓮娜·沙伊克是俄罗斯顶级模特，为世界著名的女性内衣品牌[[Christian_Dior|Christian Dior]]的[[Lingerie|Lingerie]]代言，从而成为世界内衣模特的“一姐”。
