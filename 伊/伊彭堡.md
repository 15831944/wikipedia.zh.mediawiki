'''伊彭堡'''（{{lang-nl|Ypenburg}}）又譯為'''勇堡'''<ref group="註">「勇堡」譯名來源可參見：{{cite |title = 中央政府各機關派赴國外各地區出差人員生活費日支數額表 |ref = |version = 99年9月16日生效版本 |url = http://www.rootlaw.com.tw/LawContent.aspx?LawID=A040140021001800-1041228 }}</ref>，是[[海牙|海牙]]市內的一個社區，1936年隨著{{link-nl|伊彭堡機場|Vliegveld Ypenburg}}的建設而發展起來，1992年機場關閉後，1997年荷蘭政府將伊彭堡劃設為{{link-nl|第四次國土空間規劃備忘錄增訂|Vinex|Vinex}}新住宅區，因而持續茁壯，成為約有三萬居民人口的社區。2002年，海牙市政府進行行政區劃調整，伊彭堡兼併了部分[[雷斯威克|雷斯威克]]、{{link-nl|派奈克|Pijnacker}}和{{link-nl|諾特多普|Nootdorp}}的土地，並與{{link-nl|洪恩維克|Hoornwijk (wijk in Den Haag)}}、{{link-nl|弗爾帕克|Forepark (wijk in Den Haag)}}和{{link-nl|萊德斯亨芬|Leidschenveen}}共同組成{{link-nl|萊德斯亨芬－伊彭堡|Leidschenveen-Ypenburg}}，為海牙市的八個行政區之一<ref name="wijkprogramma">{{cite web |url=https://www.denhaag.nl/web/file?uuid=ea82b4e5-eef4-4a9d-b84c-b67371126505&owner=8ab21477-3e7e-4f5b-91b1-4b92b1fb54f7 |title = Wijkprogramma Ypenburg 2016-2019 |trans-title = 伊彭堡社區計畫 2016-2019 |publisher = 海牙市政府 |website = denhaag.nl |language = nl |date = 2015年11月24日 |access-date = 2018-08-13 }}</ref>。

== 歷史 ==
=== 史前至中古時代 ===
[[File:Drie_Ypenburgers.jpg|thumb]]

伊彭堡得名於歷史悠久的同名農場，直到1997年在興建伊彭堡購物中心時，考古學家發掘了一座非常古老的人類定居和墓葬遺址，才逐漸拼湊出此地的史前樣貌<ref name="YpenburgVertelt">{{cite web |url = http://www.haagseherinneringen.nl/pagina/331/historische_inleiding |title = Ypenburg vertelt - Historische inleiding |trans-title = 訴說伊彭堡 - 歷史介紹 |website = Haagse Herinneringen |language = nl |access-date = 2018-08-13 }}</ref>。在西元前約一萬年以前，現今的[[北海|北海]]是一片可直達[[英格蘭|英格蘭]]的陸地。直到最後一個[[冰河時代|冰河時代]]結束，海平面隨著融冰而再次上升，海岸線才不斷東移。早在西元前3800年，[[新石器時代|新石器時代]]的[[哈茲東克文化|哈茲東克]]人（Hazendonk）便來到海牙的沿海地區<ref name="archeologieopdekaart">{{cite web |url = https://www.archeologieopdekaart.nl/de-eerste-boeren/ypenburg-locatie-4/pointofinterest/detail |title = Ypenburg - Een grafveld bij een oud vliegveld |trans-title = 伊彭堡－一座老機場上的古墳墓 |website = Archeologie op de Kaart |language = nl |access-date = 2018-08-13 }}</ref><ref name="denhaag.nl">{{cite web |url = https://www.denhaag.nl/nl/in-de-stad/vrije-tijd-en-recreatie/kunst-en-cultuur/archeologie/archeologische-opgravingen-ypenburg.htm |title = Archeologische opgravingen Ypenburg |publisher = Gemeente Den Haag |website = denhaag.nl |date = 2010年3月23日 |language = nl |access-date = 2018-08-13 }}</ref>；地勢較低的地區易被洪水淹沒，或較為潮濕、不適合居住，人類於是選擇在一些地勢高突的沙丘上定居<ref name="denhaag.nl" />。伊彭堡便是其中最大的沙丘，高度約有100至750公尺<ref name="denhaag.nl" />。人類在沙丘上建造了1.8公尺高的房屋<ref name="denhaag.nl" />，主要以狩獵和採集維生，同時也耕種穀物和畜牧<ref name="YpenburgVertelt" />。有時他們也住在其他地方，但仍會到沙丘地區狩獵和採集果子與堅果<ref name="denhaag.nl" />。考古學家在伊彭堡遺址發現了[[火爐|火爐]]和水坑的遺留，以及墓葬遺跡和[[煤精|煤精石]]、[[琥珀|琥珀]]吊墜等貴重的陪葬物，代表當時人類已有飲用水的習慣和墓葬文化<ref name="ypje">{{cite web |url = http://www.historischypenburg.nl/themas/steentijd/ypje/ |title = Ypje van Rijswijk |website = Historische Vereniging Buitenplaats Ypenburg |language = en |access-date = 2018-08-13}}</ref><ref name="archeologieopdekaart" /><ref name="denhaag.nl" />。伊彭堡墓葬遺跡也是荷蘭西部最古老的墓地<ref name="YpenburgVertelt" />。直到西元前4000年至3500年，北海地區成為一片汪洋，海岸線向東推到了伊彭堡沙丘<ref name="denhaag.nl" /><ref name="anemaa">{{cite web |url = https://anemaa.home.xs4all.nl/ges/geschiedenis_den_haag_01_prehistorie.htm |title = Den Haag en de prehistorie |trans-title = 海牙與史前時代 |website = Geschiedenis van Den Haag |language = nl |access-date = 2018-08-13 }}</ref>，沙丘的周圍環境逐漸變得潮濕、渾濁。西元前約3400年，人類便離開了伊彭堡的定居地<ref name="denhaag.nl" />。史前時代後期，海岸線逐漸向西後退，在今海牙市留下了沙洲或沙丘地形<ref name="YpenburgVertelt" />。

在[[羅馬統治荷蘭時代|羅馬帝國時期]]，海岸線退到了比現今的海岸線更西的位置，海牙地區也被羅馬人攻佔，成為[[羅馬帝國|羅馬帝國]]的領土<ref name="YpenburgVertelt" />。到了[[勃艮第統治荷蘭時代|中世紀]]，受到[[西風帶|西風]]的影響，海岸線向東移動了數英里，伊彭堡成為了一片[[泥炭|泥炭]][[沼澤|沼澤]]<ref name="YpenburgVertelt" />。相較於伊彭堡，人類傾向定居在[[雷斯威克|雷斯威克]]和{{link-en|福爾堡|Voorburg}}等地勢較高的地區，也因此後人對中古時代的伊彭堡所知甚少<ref name="YpenburgVertelt" />。伊彭堡最初偶被用於農耕，後來數世紀成為泥炭的盛產地，遍布挖掘泥炭所留下的水坑<ref name="YpenburgVertelt" />。一直到近代[[填海|填海]]工程實施後，伊彭堡才有了漁業以外的發展<ref name="YpenburgVertelt" />。

=== 伊彭堡機場 ===
{{參見|伊彭堡機場}}
伊彭堡近海牙一側因為較遠離洪氾區，形成了第一個建成區：沿著{{link-nl|弗利特運河|Vliet (Zuid-Holland)}}，坐落著許多別緻的鄉村莊園，當地被名為伊彭堡農場<ref name="YpenburgVertelt" />。十九至二十世紀，伊彭堡開通了前往[[烏特勒支|烏特勒支]]和[[鹿特丹|鹿特丹]]的鐵路，二十世紀時更進一步闢建高速公路，以及沿之而建的雷斯威克單車道<ref name="YpenburgVertelt" />。最後，第一座「海牙」機場於1937年啟用<ref name="YpenburgVertelt" />。

海牙機場的建造藍圖可追溯至1919年<ref name="YpenburgVertelt" />。1926年，當時的海牙市長派坦先生（Mr. J.A.N. Patijn）主張在海牙附近新建一座更勝過[[阿姆斯特丹史基浦機場|史基浦]]的機場<ref name="Vliegveld">{{cite web |url = http://home.deds.nl/~vliegveldypenburg/ |title = Vliegveld Ypenburg - Van sportvliegveld tot vliegbasis |trans-title = 伊彭堡機場－從飛行運動機場到民用機場 |date = 2007年12月10日 |language = nl |access-date = 2018-08-14 }}</ref>。然而，由於1930年代經濟成長緩慢，建造機場成本過高，計畫持續延宕<ref name="YpenburgVertelt" /><ref name="Vliegveld" />。與此同時，滑翔飛行運動正在荷蘭蓬勃發展，伊彭堡農場附近地區的上空是熱門的運動地點<ref name="Vliegveld" />。1935年，由海牙飛行俱樂部（Haagsche Aero Club）的主席瑞因德斯先生（D.H. Reinders）領銜，在荷蘭國家航空學校和鹿特丹航空俱樂部的支持，以及海爾德靈與皮耶生銀行（Heldring & Pierson）等資本家的資金協助之下，這群飛行運動愛好者向雷斯威克政府購買土地，伊彭堡機場破土動工，由雷斯威克市長在建地上鋪上第一塊石磚<ref name="Vliegveld" /><ref name="HIP">{{cite web |url = http://www.historischdenhaag.nl/pagina/51/geschiedenis_vliegbasis |title = Geschiedenis Vliegbasis |website = Historisch Informatiepunt Den Haag |language = nl |access-date = 2018-08-14 }}</ref>。1936年7月11日，第一架飛機──[[德哈維蘭|德哈維蘭]]F.K.C豹蛾單翼機──降落在伊彭堡機場<ref name="Vliegveld" />。伊彭堡機場最初是運動競技機場，直到1937年8月18日獲得國家批准轉為民用機場。第一架降落在伊彭堡機場的航班是[[荷蘭皇家航空|荷蘭皇家航空]]的{{link-nl|柯爾霍芬|Koolhoven}}F.K.43飛機<ref name="HIP" />。

{{double image|right|Denhaag monument ypenburg1.jpg|200|Denhaag monument ypenburg2.jpg|120|伊彭堡紀念碑，由兩組青銅雕塑品組成：第一組（左圖）由六個肩並肩站立的抽象人形，放置在比利時青石基座上；另一組（右圖）則是兩個抽象人形，碑上刻有兩篇紀念伊彭堡戰役的詩作。<ref>{{cite web |url = http://vanderkrogt.net/standbeelden/object.php?record=ZH15de |title = Monument Ypenburg |website = Mens & Dier in Steen & Brons |language = nl |access-date = 2018-08-14 }}</ref>}}
在[[阿姆斯特丹史基浦機場|阿姆斯特丹史基浦機場]]和{{link-nl|鹿特丹瓦哈芬機場|Vliegveld Waalhaven}}之間，伊彭堡機場的經營狀況一直很勉強<ref name="Vliegveld" /><ref name="HIP" />。到了[[第二次世界大戰|第二次世界大戰]]爆發前，伊彭堡機場被轉作軍用機場<ref name="HIP" />。1940年5月10日，德軍襲擊了中立的荷蘭。德國傘兵和空降部隊意圖降落在伊彭堡機場，計畫後續從伊彭堡向海牙進攻，進而俘虜荷蘭皇室、政府和軍隊首腦<ref name="Vliegveld" />。荷蘭面臨了歷史上第一次大型空降作戰的威脅。德軍戰鬥機的激烈轟炸和大規模空降砲擊不斷，但荷軍頑強抵抗，最終成功守成伊彭堡機場。這次行動也是德軍在西歐「[[閃電戰|閃電戰]]」之中所遭遇到唯一的重大挫敗<ref name="Vliegveld" />。為紀念伊彭堡戰役，今日的伊爾西公園（ILSY-plantsoen）矗立了「伊彭堡紀念碑」。許多街道也以當時在伊彭堡作戰的荷蘭將士命名<ref name="Vliegveld" />。

戰爭結束後，伊彭堡機場繼續被用作軍用機場，一直到[[冷戰|冷戰]]結束後，伊彭堡機場於1992年9月20日關閉<ref name="Vliegveld" /><ref name="HIP" />。

== 人口 ==
{{as of|2018}}，伊彭堡共有27,165位社區居民，居民的平均年齡為34.6歲，低於全海牙市的市民平均年齡<ref name="demografie" />。其中，青壯年人口（20-64歲）佔61.0%，青少年與幼年人口（20歲以下）佔31.9%，老年人口（65歲以上）則佔有7.0%<ref name="demografie" />。自2002年以來，伊彭堡的老年人口日益增加，青少年與青壯年人口則皆逐年減少<ref name="demografie" />。伊彭堡的青壯年人口所佔比例（61.0%）也低於全海牙市（62.7%）<ref name="demografie" />。

伊彭堡的荷蘭本地人口佔58.8%，大幅高於全海牙市的比例（46.2%）<ref name="demografie" />。而在2015年伊彭堡的移民人口中，非西方族裔佔總社區人口的25.3%，西方族裔則佔11.8%<ref name="wijkprogramma" />。

根據海牙市政府估計，{{as of|2018|1|1}}，伊彭堡共有10,191個家戶，其中23.0%為單身家戶，12.2%為單親家庭，有無子女的雙親家庭分別佔了43.2%和21.6%<ref name="demografie" />。

伊彭堡社區居民的社經地位和教育程度普遍較高<ref name="wijkprogramma" />。
