'''伊尔蒂斯兵营'''（{{lang-de|Iltis-Kaserne}}），旧址位于[[中国|中国]][[山东省|山东省]][[青岛市|青岛市]][[市南区|市南区]][[香港西路|香港西路]]2号{{注|一说香港西路5-13号，曾用门牌号为湛山大路1号、湛山大路2号}}，是驻青[[德意志帝国海军|德军]]于1899年在[[青岛市|青岛市]][[太平山_(青島)|太平山]]南侧建设的一所[[兵营|兵营]]，也是德军在青岛修建的第一所永久性兵营。<ref>{{Cite book|title=青岛：老房子的记忆|last=李明|first=|publisher=山东画报出版社|year=2004|isbn=7-80603-828-0|location=济南|pages=38}}</ref><ref>{{Cite news|url=http://arc.dailyqd.com/2016-04/08/content_325017.htm|title=老照片：伊尔蒂斯兵营 总督署医院|author=|date=2016-04-08|work=青岛日报|accessdate=2017-05-08}}</ref>。
现为[[全国重点文物保护单位|全国文物保护单位]][[青岛德国建筑群|青岛德国建筑群]]的一部分。

==历史沿革==
===德租时期===
[[File:青岛伊尔蒂斯兵营.jpg|thumb]]
[[File:Bundesarchiv_Bild_134-A303,_Tsingtau,_Feldartillerie-Kasernement_bei_Tai_tung_tschen.jpg|thumb]]
1897年11月德军占领青岛后，共700多名海军士兵暂时驻扎在原清军的六座营房和[[总兵衙门_(青岛)|总兵衙门]]内。1898年1月26日，又一批卫戍部队随{{link-de|达姆施塔特号蒸汽船|Darmstadt (1890)|达姆施塔特号蒸汽船}}到达青岛，并于同年6月13日组成海军第三营（III. Seebataillon）<ref>{{cite web |url=http://www.marine-infanterie.de/html/3_8.html|title=Aufstellung des III. Seebataillons |accessdate=2017-06-07|work=|publisher=Geschichte der Marine-Infanterie (1675-1919)}}</ref>。为改善居住环境，德国当局对这些营房进行了临时性的修缮，包括建设排水工程、安装路灯、建设新的临时营房等。然而修缮旧营房花费较大，长远来看不合算，且旧营房不符合卫生标准。新营房的建设由此提上日程。1899年夏，[[斑疹伤寒|斑疹伤寒]]和[[回归热|回归热]]疫情在青岛爆发，导致29名德军士兵死亡{{注|这一数字在当时占德军卫戍部队的2%}}，这更使建设新营房的计划迫在眉睫<ref Name=chen>{{Cite news|url=http://wanbao.qingdaonews.com/html/2008-09/21/content_1028490.htm|title=伊尔蒂斯兵营：沉默的见证者|author=|date=2008-09-21|work=青岛晚报|accessdate=2017-05-08}}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref Name=ming>{{cite book|author=王栋|title=《青岛影像 1898-1928：明信片中的城市记忆》|year=2017|publisher=中国海洋大学出版社|location=青岛|isbn=978-7-5670-1113-7}}</ref>。

1899年10月，当局开始在伊尔蒂斯山（Iltis-Berg，今[[太平山_(青岛)|太平山]]）{{注|该名称是为了纪念1896年在[[镆铘岛|镆铘岛]]外海遭遇台风沉没的{{link-de|伊尔蒂斯号巡逻舰|SMS Iltis (1878)}}}}南坡、邻近靶场（Schießstand）及[[第一海水浴场|海水浴场]]（Badestrand）的位置建设新营房，第一期工程于1901年4月完工。兵营的自来水与排水设施也于随后开始施工，排水设施与奥古斯特-维多利亚海湾（Auguste-Viktoria-Bucht，今[[汇泉湾|汇泉湾]]）周边的排水系统相连接，下水陶管达3212米。在兵营的东南侧又建设了一座约9000平方米的[[汇泉广场|练兵场]]（Exerzier Platz），四周环绕跑马道。整个兵营的设计者是海军第三营工兵上尉兼要塞工程局（Fortifikation）局长穆勒上尉（Hauptmann Müller），政府建筑师拉菲尔特（Raffelt）负责工程监管，工程耗资95万[[德国金马克|金马克]]<ref Name=chen/><ref Name=ming/>。

兵营建成后，用于安置海军第三营的指挥部及二连、四连两个连队。1906年，[[俾斯麦兵营|俾斯麦兵营]]（Bismarck-Kaserne）初步建成后，第三营指挥部及部分部队移至俾斯麦兵营驻扎，胶澳海军炮兵分队（Matrosenartillerie Abteilung Kiautschou）进驻伊尔蒂斯兵营。1908-1909年度的《胶澳租借地发展备忘录》记载道，兵营附近自1909年9月起建设了一座枪械制造厂<ref>{{cite web |url=http://digital.staatsbibliothek-berlin.de/werkansicht?PPN=PPN77082420X&PHYSID=PHYS_0059&DMDID=DMDLOG_0001|title=胶澳租借地发展备忘录 1908-1909年度|work=德意志帝国海军部|publisher=柏林国家图书馆}}</ref>。

===第一次日治时期===
[[File:青岛旭兵营日本中学校.jpg|thumb]]
1914年11月，[[日军|日军]]占领青岛，伊尔蒂斯山改名为旭山，兵营也被改名为旭兵营。根据日军占领后不久出版的《日独战役青岛名所写真帖》记载，驻扎在这里的是日军重炮兵第二大队和独立工兵队<ref>{{Cite|author=三船写真馆|title=日独战役青岛名所写真帖|date=1914}}</ref>。1917年2月21日青岛军政署发布军政告示第11号，决定在在旭兵营设立[[青岛日本中学校|青岛中学校]]。3月末，在II号营房举行入学考试，4月4日开学，暂借兵营西半部分（即II号营房及周边几座附属建筑）作为校舍。7月5日，日本中学校成为日本[[文部省|文部省]]在外指定学校，时有学生110人，日籍教职员工28人。1921年6月21日，[[青岛日本中学校|日本中学新校舍]]建成，学校于6月末至7月迁走。<ref Name=ming/>

===青岛回归后，作为校舍使用===
1922年12月，[[北洋政府|北洋政府]]正式收回青岛主权。日军撤出兵营。该年9月，[[胶澳商埠|胶澳商埠督办公署]]在兵营筹办[[胶澳商埠公立职业学校|胶澳商埠公立职业学校]]，[[王在密|王在密]]出任校长。这是青岛接收后第一个官办学校，也是本地第一所由中国人创办的职业商工学校。学校开办费1.05万元，常年经费2.3万元，设农工商3科，实际仅招收了商科学生4个班，其中有初级2个班，补习生1个班，计150余人。翌年，又招收了初级生39人。商埠职业学校皆为男生，有教员和职员16人。1923年秋，在青岛进行秘密活动的[[邓恩铭|邓恩铭]]曾经在这里召集了8名学生开会，酝酿建立青年组织<ref Name=chen/><ref Name=hua>{{Cite book|title=画说青岛老建筑|last=窦世强，李明|first=|publisher=青岛出版社|year=2010|isbn=978-7-5436-3134-2|location=青岛|pages=30}}</ref>。

[[山东|山东]]各地[[士绅|士绅]]于1924年2月筹资开办的私立胶澳中学于该年10月自登州路旧校舍（原德军[[毛奇兵营|毛奇兵营]]）迁至此处。1926年，学校因经费困难改为公办，改名胶澳商埠公立中学校。作家[[王统照|王统照]]于1927年迁居青岛后曾在此任国文教师。1929年，学校又改称[[青岛特别市|青岛特别市]]市立中学校<ref Name=chen/>。

===1938年后，恢复军事用途===
1938年1月，日军再次占领青岛。日军占用市立中学校舍作为兵营，学校于4月被迫迁往[[西镇|西镇]]，几经辗转演变，最终成为今日的[[青岛一中|青岛一中]]。1945年[[日本投降|日本投降]]后，{{link-en|美国海军陆战队第六师|6th Marine Division (United States)}}进驻青岛，原伊尔蒂斯兵营先后成为美海军陆战队第六师第六坦克营（No.6 Tank Battalion）与[[中华民国国军|国军]][[青年军|青年军]]的兵营<ref Name=chen/><ref Name=hua/>。

1949年6月，[[解放军|解放军]]进驻青岛后，该兵营延续了军事用途。1950年8月24日组建的解放军[[中国人民解放军海军航空大学#海军航空工程学院|海军炮兵学校]]曾设于此处，直至1952年9月迁往[[烟台|烟台]]。1952年9月，[[中国人民解放军海军大连舰艇学院#中国人民解放军海军政治学院|海军联合学校]]校部及第一分校自[[南京|南京]]迁至此处，后先后改名为第二海军学校、第五海军学校{{注|亦有资料称1952年7月成立的海军政治干部学校设于此处，该校1958年并入海军高专}}。1957年9月27日，解放军海军在第五海军学校与第三海军学校{{注|驻莱阳路8号}}的基础上成立海军高级专科学校，设于伊尔蒂斯兵营旧址，1960年改名海军高级学校，1963年改为海军政治学校，1969年2月撤销。此后，伊尔蒂斯兵营旧址改为海军[[北海舰队|北海舰队]]司令部的驻地，直至今日。

==建筑特色==
初建时的伊尔蒂斯兵营由两座“工”字形大营房及大约10栋附属建筑组成。两座主营房设计相同，大致坐北朝南，正立面面朝东南，共两层，屋顶为坡屋顶[[阁楼|阁楼]]，屋顶最初为瓦楞铁皮屋顶，后改铺矩形[[红瓦|红瓦]]，同时设有地下室，[[花岗岩|花岗岩]]碎石砌墙基，正立面同时出现了砖石砌成和木材支撑的[[敞廊|敞廊]]，墙面以花岗岩石块和[[红砖|红砖]]交替装饰。主入口位于正立面中轴线处，其上方的檐口起[[山花|山花]]，北立面东西两端各有一尖顶[[塔式建筑|塔楼]]自屋顶突出，坡屋顶较为平缓，大大伸出墙体。其他附属建筑多为平房，建筑风格与大营房相一致。德国学者托尔斯腾·华纳（Torsten Warner）认为兵营建筑具有典型的[[南欧|南欧]]建筑风格<ref Name=chen/><ref Name=de>{{cite book |author=托尔斯腾·华纳 |title=《德国建筑艺术在中国：建筑文化移植》|year=1994 |publisher=Ernst & Sohn |location=柏林 |isbn=3433024294}}</ref>。

兵营的每座营房可容纳一名尉官，若干士官，以及一个[[连|连]]的士兵，每间营房60平方米，可容纳10名士兵居住，这要比[[德意志帝国|德国]]国内的标准大出二分之一。房间都设在设有明廊的南面，北面则被长长的走廊隔开。除了营房，还设有两座附属建筑，内设食堂、士兵厨房、军官厨房以及其他必要的后勤保障房屋<ref Name=chen/><ref Name=ming/>。

营地内的护坡、人行道和通道，全用石块及[[混凝土|混凝土]]砌造，主要通道可行[[汽车|汽车]]。兵营的下水系统和整个奥古斯特-维多利亚海湾一带的排水设施是连接的，陆续安装的下水陶管长达3212米。仿效[[胶澳督署医院|督署医院]]的经验，兵营在后面的高地上建造了一个调节蓄水器。蓄水器是由混凝土制成的水箱，大部分嵌在山岩中<ref Name=chen/>。

根据1924年7月考入私立胶澳中学的刘君雅的回忆，迁至原伊尔蒂斯兵营的胶澳中学，西邻第一公园（今[[青岛中山公园|青岛中山公园]]），靠近[[八大关|八大关]]，面临海滨，风景优美，空气颇好。校园极大，分为两院，一院为教学及办公大楼，二院为学生宿舍、礼堂、图书馆、健身社和教职员工大楼。还附设有学生实习工厂，有小型的制皂、制革、油漆、化学工艺、木工等工厂；还有物理、化学、生物、史地、音乐、美术等实验室和仪器标本室等。同时还设有400米跑道的大操场，有一个足球场、两个网球场、四个篮球场、两个排球场、一个器械场<ref Name=chen/>。

兵营营房的大部分建筑均保存至2000年，2001年后，许多原附属建筑陆续被拆除。其中， II号营房南侧和北侧的几栋平房于2001年至2003年间拆除，I号营房北侧的附属建筑于2004年拆除，II号营房北侧的附属建筑也经历过扩建。现仅剩两座大营房和II号营房北侧、西南侧、西侧（靠近兵营西门）的三栋附属建筑。现存所有建筑原有的红瓦曾于2005年7月后换成金黄色瓦片，这也使得该建筑群在俯瞰图和卫星地图上极为显眼。

==督署洗衣房==
[[File:青岛督署洗衣房.png|thumb]]
督署洗衣房（Gouvernements-Waschanstalt）又称卫戍区洗衣房（Garnison-Waschanstalt），是伊尔蒂斯兵营东侧的一座建筑群，位于今[[佛涛路|佛涛路]][[鹊山路|鹊山路]]路口西侧，其地址与伊尔蒂斯兵营旧址如今同属一个大院。

有关该建筑群的记载较少，大部分为政府公文。其中，1901-1902年度的《胶澳租借地发展备忘录》提到这座安装蒸汽洗涤设施的洗衣房，作为租借地卫生设施的一部分，将提上日程<ref>{{cite web |url=http://digital.staatsbibliothek-berlin.de/werkansicht?PPN=PPN770823653&PHYSID=PHYS_0041&DMDID=DMDLOG_0001|title=胶澳租借地发展备忘录 1901-1902年度|work=德意志帝国海军部|publisher=柏林国家图书馆}}</ref>。下一年度的备忘录则记载该工程的招标已于1903年9月开始，各项工程已开工建设<ref>{{cite web |url=http://digital.staatsbibliothek-berlin.de/werkansicht?PPN=PPN770823688&PHYSID=PHYS_0048&DMDID=DMDLOG_0001|title=胶澳租借地发展备忘录 1902-1903年度|work=德意志帝国海军部|publisher=柏林国家图书馆}}</ref>。1903-1904年度的《胶澳租借地发展备忘录》提到该建筑群已主体完工，该年冬季便可开始安装机械设备，至1905年3月至6月有望投入使用<ref>{{cite web |url=http://digital.staatsbibliothek-berlin.de/werkansicht?PPN=PPN770823696&PHYSID=PHYS_0043&DMDID=DMDLOG_0001|title=胶澳租借地发展备忘录 1903-1904年度|work=德意志帝国海军部|publisher=柏林国家图书馆}}</ref>。该年度备忘录附录中也提供了建筑群的照片。

1930年代时洗衣房建筑群曾为日本东洋洗衣公司（{{lang|ja|東洋ランドリー社}}）所使用。该建筑群现处于军事管理区内。
