'''伊万·弗拉基米罗维奇·科瓦廖夫'''（{{lang-ru|Иван Владимирович Ковалёв}} {{bd|1901年|6月15日|1993年|5月28日}}）[[苏联|苏联]]政治家、[[最高苏维埃|最高苏维埃]]代表、{{link-ru|苏联交通人民委员会|Министерство путей сообщения СССР}}委员、[[中长铁路|中长铁路]]苏方负责人。曾担任苏联顾问和专家代表团团长。1919年3月，科瓦廖夫加入[[苏联红军|红军]]。1948年-1950年为[[中国共产党中央委员会|中国共产党中央委员会]]首席顾问、[[联共_(布)|联共 (布)]]驻中共中央代表，领导苏共派往中共的军事专家。1949年7月，科瓦廖夫陪同[[刘少奇|刘少奇]]秘密访问苏联<ref>{{cite web |author1=何明 |title=1949年刘少奇访苏：获3亿美元贷款 明确苏联领导地位 |url=http://news.ifeng.com/history/special/chubingjiazhi/detail_2010_06/21/1648659_0.shtml |website=凤凰网历史综合 |publisher=人民出版社 |accessdate=2019-01-04 |date=2010年06月21日}}</ref>。同年，科瓦廖夫曾给斯大林通过电报发送了一份绝密的密码报告，他在报告中对当时的中国共产党进行了“诬蔑”，指出了当时中共内部潜在的派别。同年时任中共领导人[[毛泽东|毛泽东]]前往苏联会见[[斯大林|斯大林]]时，该报告被斯大林交给了毛泽东，使得有倾苏意向的[[高岗|高岗]]被暴露，为后来高岗倒台埋下伏笔。1949年底，毛泽东前往苏联，科瓦廖夫陪同回到苏联后，苏联顾问和专家代表团团长转由[[伊万·瓦西里耶维奇·阿尔希波夫|伊万·瓦西里耶维奇·阿尔希波夫]]担任。<ref>{{cite web |author1=刘志青 |title=新中国首席俄语翻译李越然 |url=http://dangshi.people.com.cn/GB/85038/10842003.html |website=人民网-中国共产党新闻网 |publisher=《党史博览》 |accessdate=2019-01-05 |archivedate=2010年01月26日08:12}}</ref>
