'''伊林娛樂'''（{{lang-en|'''EeLin Entertainment'''}}）是[[臺灣|臺灣]]的藝能娛樂公司，以[[模特兒|模特兒]]經紀起家，旗下除[[歌手|歌手]]、[[藝人|藝人]]、模特兒外，亦有多位傑出[[演員|演員]]。

==歷史==
[[File:EeLin_Entertainment_headquarters_entry_20170624.jpg|thumb]]
1997年，「西恩模特兒經紀公司」與「宏星輝模特兒經紀公司」合併為「伊林模特兒經紀公司」。2012年，伊林模特兒經紀公司加入[[旺旺中時媒體集團|旺旺中時媒體集團]]<ref>{{cite web |title=“陳婉若專訪－如果可以當個勇於追求夢想的女人，為什麼要選擇做個安於現狀的女孩” |url=http://www.wazaiii.com/articles?id=124 |publisher=Wazaiii |accessdate=2018-10-09}}</ref>，正式更名「伊林娛樂」，在[[上海|上海]]設分公司。

2016年12月，伊林娛樂於[[醒吾科技大學|醒吾科技大學]]開設「伊林學院」，伊林娛樂董事總經理[[陳婉若|陳婉若]]兼任伊林學院院長<ref>{{cite web |title=醒吾伊林學院正式成立 |url=http://www.hwu.edu.tw/files/14-1000-39272,r115-1.php?Lang=zh-tw |website=2018-10-09 |publisher=醒吾科技大學}}</ref>。

==組織==

*伊林娛樂有八大部門：
**創意活動部
**廣告平面部
**演藝經紀部
**人力經紀造型部
**專案節目部
**音樂事業部
**媒體公關部
**商品事業部
