'''伊朗伊斯兰共和国广播电视台'''（{{lang-fa|'''سازمان صدا و سیمای جمهوری اسلامی ایران'''}}）是[[伊朗|伊朗]]第一家[[广播电台|广播电台]]，成立于1938年。是伊朗唯一覆盖全国的广播电视媒体，在伊朗占据绝对强势地位。

== 历史沿革 ==
* 1938年，[[伊朗国家广播电视台|伊朗国家广播电视台]]成立。

== 概况 ==
[[File:IRIB_east_gate_2.JPG|right]]
[[File:IRIB_TV.jpg|right]]

IRIB的广播共有18个频率，同时承担对内、对外广播功能，其中的11个频率使用波斯语对内广播，另外7个对外频率以30种语言对全球广播。IRIB网站使用波斯语、[[英语|英语]]两种文字，除电台介绍、滚动新闻、服务信息之外，所有[[广播|广播]]频率都可实现网上点播。此外，IRIB还拥有自己的[[报纸|报纸]]和[[杂志|杂志]]。

作为[[政府|政府]]喉舌，目前[[伊朗|伊朗]]全部[[广播|广播]]资源均由国家掌控，采取两级办广播的方式。除[[国家电台|国家电台]]之外，28个[[省|省]]各设有一家省级台，[[市|市]]、[[县|县]]一级不设电台（与[[越南|越南]]类似）。全国共有[[广播频率|广播频率]]80多个。

多年来，伊朗广播不断发展，收听率和社会影响力与日俱增。

== 华语广播 ==
该台开设有中文广播，通过短波每天播出两小时。对[[德黑兰|德黑兰]]使用调频播出1小时。华语节目呼号为“'''伊朗伊斯兰共和国对外广播电台'''”。

在华语广播开台时，先播放[[伊朗国歌|伊朗国歌]]，然后播送《[[古兰经|古兰经]]》及中文翻译，之后播出新闻及其它专题节目。

短波广播已于2018年9月23日停播。
