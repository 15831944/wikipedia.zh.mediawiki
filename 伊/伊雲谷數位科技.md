'''伊雲谷數位科技股份有限公司'''（{{Lang|en|eCloudvalley Digital Technology Co., Ltd.}}），簡稱為'''伊雲谷'''，成立於2013年，主要服務為[[亞馬遜網路服務系統|亞馬遜網路服務系統]]，是[[亞馬遜網路服務系統|亞馬遜網路服務系統]]認可的亞太區雲端代理商之一，總部位於[[台灣|台灣]][[台北|台北]]。 

近年分別在[[香港|香港]]、[[中國大陸|中國大陸]]、[[菲律賓|菲律賓]]設有分部。<ref>https://www.ecloudvalley.com/zh-hant/contact_us/</ref>   

屬於雲端託管服務供應商（MSP, {{lang|en|Managed Service Provider}}），亦對[[B2B|B2B]]企業提供[[亞馬遜網路服務系統|亞馬遜網路服務系統]]技術諮詢服務。

2018年7月17日，伊雲谷(6689)成功登錄興櫃交易，成為亞洲第一檔「雲端數位管理服務公司」。<ref>{{Cite news|url=https://money.udn.com/money/story/5636/3255928|title=亞洲首檔雲公司  伊雲谷今登興櫃 {{!}} 財富管理 {{!}} 商情 {{!}} 經濟日報|last=經濟日報|newspaper=經濟日報|accessdate=2018-07-25}}</ref>

2018年「Deloitte亞太區高科技、高成長500強」（Deloitte Technology Fast 500 Asia Pacific, Fast 500）評選，伊雲谷搭上雲端服務、大數據分析趨勢，營收高成長3,007% 居台灣之冠，排名亞太總榜第33名。<ref>{{Cite web|url=https://www2.deloitte.com/tw/tc/pages/technology-media-and-telecommunications/articles/2018-fast500.html|title=《2018高科技高成長500強》評選結果|accessdate=2018-12-06|work=勤業眾信|language=zh-Hant}}</ref>

==  AWS 相關認證 ==
2016年12月，取得AWS受管服務合作夥伴 (AWS Managed Service Partner)；<p>
2017年3月，伊雲谷取得 AWS 原廠授權培訓夥伴 (AWS Authorized Training Partner)，提供完整且正規的 AWS 教育訓練；<p>
2017年4月，伊雲谷成為大中華區第一間高級培訓夥伴( AWS Premier Consulting Partner)；<ref>{{Cite web|url=https://aws.amazon.com/tw/partners/find/partnerdetails/?n=Ecloudvalley&id=001E000000jD1zaIAC&t=psf-overview|title=APN Partner Profile|accessdate=2018-05-17|work=Amazon Web Services, Inc.|language=en-US}}</ref><p>
2017年9月，通過 ISO 27001 與 IS0 27017 專業驗證，是經過 AWS 與第三方稽核與認可的雲端專家；2017年，通過 AWS Mobile Competency以及AWS Marketing and Commerce Competency 認可。<p>
2018年8月，伊雲谷中國大陸戰略夥伴，伊克羅德信息科技有限公司成為AWS大中華區 IoT Competency能力認證夥伴<ref>{{Cite news|url=https://www.ithome.com.tw/pr/125428|title=伊克羅德成為大中華區通過 AWS IoT 能力認證之諮詢合作夥伴|newspaper=iThome|accessdate=2018-08-24|language=zh-hant}}</ref><p>
2018年11月，伊雲谷獲得AWS全球首波通過 Well-Architected Partner 計劃的其中一員。<ref>{{Cite news|url=https://tw.stock.yahoo.com/news/%E4%BC%8A%E9%9B%B2%E8%B0%B7%E6%A6%AE%E7%99%BBaws-well-architected-partners-231500465.html|title=伊雲谷榮登AWS Well-Architected Partners - Yahoo!奇摩股市|newspaper=Yahoo!奇摩股市|accessdate=2018-12-03|language=zh-tw}}</ref><p>
