'''伊扎克·拉宾'''（{{lang-he|'''יצחק רבין'''|Yitzhak Rabin }}；{{bd|1922年|3月1日|1995年|11月4日|catIdx=Rabin, Yitzhak}}），[[以色列|以色列]]政治家、军事家。1974年至1977年任[[以色列总理|以色列总理]]；1992年起再次出任总理，直至1995年{{link-en|拉宾遇刺案|Assassination of Yitzhak Rabin|被刺身亡}}。他是首位出生於以色列本土的总理，首位被刺杀和第二位在任期间辭世的总理。以色列首位任內離世的总理是[[列维·艾希科尔|列维·艾希科尔]]。

== 青年时代与帕拉玛赫 ==
拉宾出生在[[巴勒斯坦|巴勒斯坦]]的[[耶路撒冷|耶路撒冷]]，当时巴勒斯坦还处于[[英国|英国]]的委任统治之下。他一岁时随家庭迁至[[特拉维夫|特拉维夫]]，并在那里长大。1940年拉宾以优异的成绩从当地的[[嘉道理农业高中|嘉道理农业高中]]毕业，当时他的愿望是成为一位灌溉工程师。

[[第二次世界大战|第二次世界大战]]爆发后，拉宾放弃了原来的志向，于1941年加入[[哈加拿|哈迦纳]]的[[帕拉玛赫|帕拉玛赫]]部队，并于1947年10月升任作战参谋长。在[[以色列独立战争|以色列独立战争]]中，他指挥了耶路撒冷防御战，並於[[内盖夫地区|内盖夫地区]]与埃及人作战。

伊扎克·拉宾参加的帕拉玛赫部队是一支精英突击队，属于秘密军事组织[[哈加拿|哈迦纳]]（犹太自卫军，即日后的[[以色列國防軍|以色列国防军]]）。他曾在巴勒斯坦种族屠杀行动中担任副指挥。

== 军队任职 ==
1962年，拉宾担任[[以色列國防軍|以色列国防军]]参谋长。他指挥以色列国防军在“[[六日战争|六日战争]]”中以压倒性优势战胜了[[埃及|埃及]]、[[叙利亚|叙利亚]]和[[约旦|约旦]]。以色列国防军占领耶路撒冷古城之后，拉宾當時為第一批以國人来到古城，然後在[[希伯来大学|希伯来大学]]的[[斯科普斯山|斯科普斯山]]山顶发表了演讲。

== 大使与劳工部长 ==
从以色列国防军退休之后，拉宾从军界转入政坛成为外交官，1968年开始担任驻[[美国|美国]]大使。1973年，作为工党成员进入以色列议会，并被任命为劳工部长。

== 第一次担任总理 ==
1974年，他当选为工党领袖并继[[果尔达·梅厄|戈尔达·梅厄]]之后成为以色列总理。

在此期间最著名的事件是“[[恩德培行动|恩德培行动]]”。一架以色列的飞机被劫持并飞离国境，在拉宾的指挥下，以色列国防军成功救出了飞机上的乘客。之后，他因两起事件而辞职：1976年12月的[[安息日|安息日]]，他组织部分军官和内阁成员迎接4架美国[[F-15|F-15]]战机的到来，触怒了以色列的宗教政党；他的夫人，[[利亚·拉宾|利亚·拉宾]]的存在银行的基金違反了以色列法律被曝光<ref>{{cite book |last=Avner |first=Yehuda |title = The Prime Ministers: An Intimate Narrative of Israeli Leadership |year=2010 |publisher = The Toby Press |ISBN = 978-1-59264-278-6 |language = en }}</ref>。拉宾为夫人付出了代价。

== 反对派成员和国防部长 ==
拉宾被迫辞职后，[[希蒙·佩雷斯|希蒙·佩雷斯]]代理总理一职，两个月后，[[利库德集团|利库德集团]]的[[梅纳赫姆·贝京|梅纳赫姆·贝京]]当选，接替了他的职务，這結束工黨三十年執政，利庫德集團從此在往後十五年的聯合政府佔有主導地位。拉賓此次辞职广受好评，人们认为拉宾此举体现了正直和责任感。

1984年至1990年間，工黨和利庫德集團共組聯合政府，拉宾作为联合政府的国防部长期間，对1987年[[第一次巴勒斯坦大起义|第一次巴勒斯坦大起义]]采取强硬措施，拉宾又命令军队残忍镇压巴勒斯坦示威者。他曾有一句话如是说：“我们应敲断他们（巴勒斯坦示威者）的手足”。

== 第二次出任总理 ==
[[File:Bill_Clinton,_Yitzhak_Rabin,_Yasser_Arafat_at_the_White_House_1993-09-13.jpg|thumb]][[比爾·克林顿|比爾·克林顿]]和巴解主席[[阿拉法特|阿拉法特]]1993年9月13日达成[[奥斯陆协议|奥斯陆协议]]的一刻]]

1992年，他成功地第二次当选总理，帶領工黨再次獲聯合政府的主導權。他在[[奧斯陸協議|奥斯陆和平协议]]的签署过程中发挥了重要作用。奥斯陆协议授予[[巴勒斯坦自治政府|巴勒斯坦自治政府]]的权利，并承认巴勒斯坦人对[[加沙地带|加沙地带]]和[[约旦河西岸地区|约旦河西岸地区]]的部分控制，但以色列仍然有效維持對該地區的控制。

由于在奥斯陆协议中的重要影响，拉宾与[[雅瑟尔·阿拉法特|雅瑟尔·阿拉法特]]、[[希蒙·佩雷斯|希蒙·佩雷斯]]共同获得了1994年的[[诺贝尔和平奖|诺贝尔和平奖]]<ref>{{cite web |url = http://nobelprize.org/nobel_prizes/peace/laureates/1994/rabin-bio.html |title = Yitzhak Rabin - Biographical |publisher = Nobelprize.org |accessdate = 2011-08-31 |language = en }}</ref><ref>[http://nobelprize.org/nobel_prizes/peace/laureates/1994/ Nobel Prize.org] 1994 Nobel Prize Laureates{{En icon}}</ref>，1995年拉賓再獲得[[世界和平獎|世界和平獎]]最高榮譽獎的肯定<ref>[http://www.wppac.net/popup/popInfo.asp?idx=3 World Peace Prize Top Honer Prize－Itzhack Rabin]WPPAC. {{En icon}}</ref>。奥斯陆协议使他在以色列国内的形象两极分化，一部分人将他视为带来和平的英雄，一部分人则视他为出卖以色列国土的叛徒，強烈反對他與被視為恐怖分子的阿拉法特進行任何政治談判<ref>{{cite book |last=Karsh |first=Efraim |title=Islamic Imperialism: A History |url=https://books.google.com/?id=LmZP3mixescC&pg=PA181&dq=many+Palestinians+viewed+the+peace+treaty+with+Israel+as+a+temporary+measure+only#v=onepage&q=many%20Palestinians%20viewed%20the%20peace%20treaty%20with%20Israel%20as%20a%20temporary%20measure%20only&f=false |publisher=Yale University Press |year=2006 |page=181 |isbn=0300106033}}{{En icon}}</ref>。

== 遇刺身亡 ==
1995年11月4日，[[奧斯陸協議|奧斯陸和平協議]]不久，在[[特拉维夫|特拉维夫]]的[[以色列国王广场|以色列国王广场]]上参加和平集会之后，一名极右翼激进主义分子[[伊蓋爾·阿米爾|伊蓋爾·阿米爾]]（Yigal Amir）用一把装有自制[[達姆彈|達姆彈]]的九公釐口徑[[左輪手槍|左輪手槍]]袭击了他，他對着拉宾的胸口開了兩槍，再向他的背部上补了兩槍，後拉宾在特拉维夫伊契洛夫医院的手术台上不治身亡，享年73歲<ref>[http://news.bbc.co.uk/onthisday/hi/dates/stories/november/4/newsid_2514000/2514437.stm 1995：Israeli PM shot dead BBC] On This Day{{En icon}}</ref>。拉賓去世后，以巴和平無以為繼。
