'''伊卜里'''（{{lang-ar|عبري}}）是[[阿曼|阿曼]][[扎希拉省|扎希拉省]]最大的城市，位于扎希拉省北部，是北扎希拉地区的首府，距离阿曼第一大城市[[马斯喀特|马斯喀特]]约300千米。伊卜里坐落在[[海拔|海拔]]359米以上<ref>{{Cite web|url=http://www.worldatlas.com/as/om/za/where-is-ibri.html|language=英语 |title=Where Is `Ibri, Oman? |publisher=WorldAtlas |accessdate=2017-11-12}}</ref>，其[[绿洲|绿洲]]闻名于它的油田比其他几乎任何地方都要多<ref name=ib>{{Cite web|url=http://www.omantripper.com/ibri/ |title=Ibri |publisher=OmanTripper |accessdate=2017-11-11 |language=英语}}</ref>。伊卜里拥有着独特的考古遗址，譬如伊卜里城堡<ref name=l1>{{Cite web|url=http://www.omantripper.com/ibri-castle/ |title=Ibri Castle |publisher=OmanTripper |accessdate=2017-11-11 |language=en}}</ref>、蝙蝠墓地（{{lang|en|Bat Necropolis}}）<ref name=l2>{{Cite web|url=http://www.omantripper.com/bat-necropolis/ |title=Bat Necropolis |publisher=OmanTripper |accessdate=2017-11-11 |language=en}}</ref>等。

== 天气 ==
[[File:Oman_(134).jpg|thumbnail]]
伊卜里的气候偏向炎热、干燥。当地的[[夏天|夏天]]温度变化通常在32℃～52℃内。从12月到次年2月（即[[冬天|冬天]]），天气较为凉爽，温度会降低至10℃至20℃左右。

== 经济 ==
在历史上，伊卜里以其贸易市场和盛产的水果而闻名<ref name=Prothero104>{{cite book|last=Prothero|first=G.W.|title=Arabia|year=1920|publisher=H.M. Stationery Office|language=英语|location=London|page=104|url=http://www.wdl.org/en/item/11767/view/1/104/}}</ref>
。

== 教育 ==
伊卜里拥有两个学院，分别为伊卜里科技学院和伊卜里应用科学学院。

== 交通 ==
有一条主要的高速公路（21号高速公路）途经伊卜里，连接到北部的边境城镇布赖米<ref name=lp>{{Cite web|language=英语|accessdate=2017-11-11 |url=https://www.lonelyplanet.com/oman/ibri |title=OMAN Ibri |publisher=[[Lonely_Planet|Lonely Planet]]}}</ref>。

== 旅游 ==
伊卜里拥有许多景点，包括苏拉衣夫城堡、阿斯瓦德城堡<ref name=ib/>、伊卜里城堡<ref name=l1/>和蝙蝠墓地<ref name=l2/>。虽然景点数目不算少，但伊卜里可供游客休息的地方却很少，因此对于游客而言，选择在距离伊卜里约一小时车程的[[布赖米|布赖米]]或[[尼兹瓦|尼兹瓦]]休息更好<ref name=ib/>。
