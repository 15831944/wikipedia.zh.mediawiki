'''伊朗门事件'''或稱'''軍售伊朗醜聞'''（Iran-Contra Affair 或 Irangate）是发生在[[美国|美国]]80年代中期的[[政治丑闻|政治丑闻]]。是指[[美国|美国]][[罗纳德·雷根|雷根]][[政府|政府]]向[[伊朗|伊朗]]秘密出售武器一事被揭露后而造成严重[[政治危机|政治危机]]的事件。因国际新闻界普遍将其与[[理查德·尼克松|-{zh-hans:尼克松;zh-hk:尼克遜''';zh-tw:尼克森;}-]][[水门事件|水门事件]]相比，故因此得名。

== 事情经过 ==
1985年以来，[[黎巴嫩真主党|黎巴嫩真主党]]绑架了多名美国人质。时任[[中央情报局|中央情报局]]局长的[[威廉·J·卡西|威廉·卡西]]从一个[[以色列人|以色列人]]那里获悉，正处于[[两伊战争|两伊战争]]的伊朗急需武器装备，特别是美式飞机的零部件和反[[坦克|坦克]]、地对空导弹。而伊朗同真主党关系密切，如果能够向伊朗提供军火，让其从中发挥影响，则有可能促使真主党释放美国人质。同时，还可以改善不断恶化的美伊关系，减缓[[伊拉克|伊拉克]]在[[海湾地区|海湾地区]]的扩张。这一计划得到里根的[[總統國家安全事務助理|国家安全事务助理]]麦克法兰支持并被命名为麦克法兰计划，而美国总统里根很可能知道这一计划并予以同意<ref>[http://home.snu.edu/~dwilliam/s98/usarab/icscandal.htm Iran Contra Affair] {{webarchive|url=https://web.archive.org/web/20100421104939/http://home.snu.edu/~dwilliam/s98/usarab/icscandal.htm |date=2010-04-21 }}</ref>。由于伊朗为美国的敌人，[[美国国会|美国国会]]禁止向其出售武器，所以所有的武器出售都是秘密进行地通过第三方进行的。1985年9月，由[[美国国家安全委员会|国家安全委员会]]政治军事处副处长[[奧利弗·諾斯|奧利弗·諾斯]]中校执行麦克法兰计划，经由以色列牵线搭桥，装载着美式装备的飞机停入了伊朗机场。9月15日，被绑架的美国[[律师|律师]]本杰明·韦尔获得释放。<ref>[http://gb.cri.cn/3821/2004/06/09/151@189897_2.htm “伊朗门”事件(3) ]</ref>计划成功之后，几个负责人决定继续交易，促使更多的美国人质被释放。同时，奧利弗·諾斯还负责给[[尼加拉瓜|尼加拉瓜]]的反政府武装[[康特拉|康特拉]]提供援助。由于国会签署了博兰修正案，中情局无法获得支援尼加拉瓜反政府武装的款项。参与军火买卖的军火商建议，将卖[[导弹|导弹]]的钱报低一些，每次由他转移一些“回扣”给中情局，再由中情局用钱款支援颠覆尼加拉瓜政权的活动。这一决定得到了新任国家安全事务助理波因德克斯特的同意。<ref name=a>[http://www.xschina.org/show.php?id=12677 告诉你一个真实的“伊朗门”]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>1986年6月，麦克法兰还曾亲自前往伊朗，希望同伊朗高层获得直接接触，但未得到正面回应。<ref>[http://gb.cri.cn/3821/2004/06/09/151@189897_1.htm “伊朗门”事件(2)]</ref>1986年7月26日，又一名美国人质被释放。11月2日，第三名美国人质被释放。截至伊朗门事件爆发，美国共向伊朗进行6次军火销售，3名人质因此释放。

== 暴露与调查 ==
1986年11月2日，一家[[黎巴嫩|黎巴嫩]][[杂志|杂志]]《帆船》揭露了6月间麦克法兰前往伊朗活动的事件。4日，伊朗议长证实了这一报道，伊朗门事件就此暴露<ref>[http://www.todayonhistory.com/11/2/MeiGuoYiLangMenShiJianBeiPiLu.html 美国伊朗门事件被披露]</ref>。1987年开始，[[美国参议院|美国参议院]]、众议院的两个特别调查委员成立，经过接近一年的调查，两个委员会在11月18日结束使命。多名参与人员援引[[美國憲法第五修正案|美國憲法第五修正案]]，拒绝回答问题；许多重要文件实际上已经被销毁；而事发时担任中情局长的卡西在调查期间去世也影响了调查工作的进行<ref>[http://www.u-s-history.com/pages/h1889.html Iran-Contra Affair]</ref>。諾斯、波因德克斯特等人声称，将军售收入用于支援尼加拉瓜反政府武装一事，总统里根并不知情并将责任揽至自己身上。调查认为里根虽然没有直接参與军售，但对其部下的违法行为仍负有一定责任。之后，多名当事人被送至法院接受起诉<ref>{{Cite web |url=http://xmujpkc.xmu.edu.cn/zckx/doc/8-2.doc |title=“伊朗门”事件 |access-date=2010-01-20 |archive-url=https://web.archive.org/web/20051222075642/http://xmujpkc.xmu.edu.cn/zckx/doc/8-2.doc |archive-date=2005-12-22 |dead-url=yes }}</ref>1989年，諾斯、波因德克斯特均被同一名独立[[检察官|检察官]]起诉。诺斯被判有罪，但其不服，并向联邦上诉法院上诉。1990年，[[联邦上诉法院|联邦上诉法院]]判其无罪。而波因德克斯特则推翻之前的供词，表明曾向里根做出过请示。但里根此时已经离任，并表现出[[老年痴呆症|老年痴呆症]]的迹象，无法印证其新的说法。法庭原本判其6个月有期徒刑，但上诉法庭同样将其无罪释放。<ref name=a/>
