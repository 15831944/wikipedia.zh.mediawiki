'''伊斯許理度賣命'''（{{lang|ja|いしこりどめ の みこと}}）為《[[古事記|古事記]]》記述之名（另作'''櫛石窗神'''或'''豐石窗神'''），《[[日本書紀|日本書紀]]》寫作'''石凝姥命'''，祂是[[日本神話|日本神話]]裡出現的[[神祇|神祇]]，也是作鏡連（{{lang|ja|かがみづくり の むらじ}}）的祖先，別名為'''石凝杼賣命'''、'''石凝戶邊神'''（{{lang|ja|いしこりとべ の かみ}}）等。此外，根據《[[先代舊事本紀|先代舊事本紀]]》的記載，石凝姥命乃天糠戶命（{{lang|ja|あまのぬかと の みこと}}，《日本書紀》之第三種說法記作「天拔戶」）之子。

== 概要 ==
=== 日本書紀之記載 ===
按《[[日本書紀|日本書紀]]》卷第一之記載，第一種說法描述因[[素戔嗚尊|素戔嗚尊]]胡作非為，[[天照大神|天照大神]]躲入[[天磐戶|天磐戶]]而閉坐其中，[[高天原|高天原]]及[[葦原中國|葦原中國]]陷入一片黑暗。故八百萬神命[[思兼神|思兼神]]想辦法，後者命'''石凝姥命'''為冶工，採集天香山之金製作日矛鏡、剝真名鹿之皮以作天羽皮吹；故此神即位在[[紀伊國|紀伊國]]的「日前大神」。另外第三種說法則提到[[天兒屋命|天兒屋命]]挖掘天香山之真阪木，上枝懸掛鏡作遠祖、天拔戶之子'''石凝戶邊'''所作的八咫鏡；中枝懸吊玉作遠祖、[[伊奘諾尊|伊奘諾尊]]之子[[玉祖命|天明玉]]所作[[八尺瓊勾玉|八阪瓊之曲玉]]；下枝懸以粟國[[忌部氏|忌部]]遠祖、[[天日鷲神|天日鷲]]所作之木棉，然後命忌部首遠祖、[[布刀玉命|大玉命]]執取這串真阪木，口誦祝詞以祈禱。

後來[[瓊瓊杵尊|天津彥彥火瓊瓊杵尊]]降臨葦原中國，天照大神賜給祂八坂瓊曲玉、八咫鏡、[[天叢雲劍|草薙劍]]等三種寶物，並命令{{link-ja|中臣氏|中臣氏|中臣}}祖先[[天兒屋命|天兒屋命]]、[[忌部氏|忌部]]祖先[[布刀玉命|太玉命]]、{{link-ja|猿女君|猿女君|猨女}}祖先[[天鈿女命|天鈿女命]]、鏡作祖先'''石凝姥命'''、玉作祖先[[玉祖命|玉屋命]]等五位神明陪伴。

=== 先代舊事本紀之記載 ===
《先代舊事本紀》卷第二神祇本紀提到，'''石凝姥命'''被[[思兼神|思兼神]]命為冶工，採天八湍河之川上天堅石，全剝真名鹿皮製作天之羽備；又採天香山之銅鑄造「[[日像鏡與日矛鏡|日矛鏡]]」，此鏡乃紀伊國所坐之國懸神（即現在[[日前神宮與國懸神宮|國懸神宮]]之國懸大神）。思兼神又命石凝姥命之子天糠戶神採天香山之銅造出「[[日像鏡與日矛鏡|日像鏡]]」（即現在[[日前神宮與國懸神宮|日前神宮]]之日前大神），最後才製作出八咫鏡。

=== 古事記之記載 ===
《[[古事記|古事記]]》上卷的記載和《日本書紀》的[[神話|神話]]故事近似，天照大御神遁入[[天磐戶|天石屋戶]]後，[[思兼神|思金神]]選天安河之天堅石及採天金山之鐵而求鍛人[[天津麻羅|天津麻羅]]；命'''伊斯許理度賣命'''造出[[八咫鏡|八咫鏡]]；命科御祖命作五百個[[八尺瓊勾玉|八尺瓊勾玉]]；召[[天兒屋命|天兒屋命]]、[[布刀玉命|布刀玉命]]拔天香山雄鹿肩骨，以天香山所生朱櫻燒之為卜。一切準備好之後，將八咫鏡、八尺瓊勾玉懸掛在枝繁楊桐樹上，讓布刀玉命捧著，並由天兒屋命頌禱祝詞。

後來[[天孫降臨|天孫降臨]]時，天照大神之孫[[瓊瓊杵尊|日子番能邇邇藝命]]降臨葦原中國之際，天照大神命天兒屋命、布刀玉命、[[天鈿女命|天宇姬命]]、'''伊斯許理度賣命'''、[[玉祖命|玉祖命]]等五位神明陪伴在側。

== 解說 ==
神名中的「石凝（{{lang|ja|いしこり}}）」意謂利用石頭作為鑄型，倒入熔化之鐵以等待凝固，令人聯想到製造鏡子的過程。因此，日本民間將此神明視為鍛冶、[[金屬|金屬]]加工之神。
