'''伊多拉斯'''（英語：Edoras）是[[托爾金|托爾金]]奇幻文學[[中土大陸|中土大陸]]裡的一座城鎮，[[第三紀元|第三紀元]]時期[[洛汗_(地名)|洛汗]]國（Rohan）的[[首都|首都]]，建造者為洛汗第一任國王[[伊歐|伊歐]]及其子[[布理哥|布理哥]]<ref>{{Cite web|url=http://www.glyphweb.com/arda/e/edoras.html|title=Edoras|accessdate=2018-02-21|publisher=阿爾達百科全書|language=en-GB}}</ref><ref name=":1">{{Cite book|title=魔戒武器聖戰|last=克理斯·史密斯|publisher=奇幻基地|year=2003年12月|isbn=986-7576-12-8|location=臺北}}</ref>。

== 地理 ==
《[[雙城奇謀|雙城奇謀]]》中描述，伊多拉斯坐落於[[白色山脈|白色山脈]]最大山峰的山腳下，平原中一座翠綠的山丘上，[[雪界河|雪界河]]流經附近。伊多拉斯邊界之處皆以壕溝及帶刺的圍籬圍住，大門前駐有守衛。在伊多拉斯大門外附近的山丘，有著歷任[[洛汗國王|洛汗國王]]的[[墳墓|墳墓]]，該處生長了許多[[永誌花|永誌花]]<ref name=":0">{{Cite book|title=魔戒二部曲：雙城奇謀|last=托爾金|publisher=聯經|year=2012年12月|isbn=978-957-08-4101-5|location=臺北}}</ref>。

=== 梅杜西金殿 ===
梅杜西位於山丘頂端、伊多拉斯的最高處，為洛汗國王的宮殿，其屋頂和大門都是以[[黃金|黃金]]打造的。在大殿內部，國王的寶座於三級台階上，面向北方；殿內四壁還掛著許多描述洛汗歷史的[[掛毯|掛毯]]<ref name=":0" />。

== 改編 ==

在[[彼得·傑克森|彼得·傑克森]]執導的《[[魔戒電影三部曲|魔戒電影三部曲]]》中的設定，伊多拉斯大約有二百戶居民<ref name=":1" />。其[[取景|取景]]地位於[[紐西蘭|紐西蘭]][[南島_(新西蘭)|南島]]坎特伯雷的周日山（Mount Sunday）上<ref>{{Cite web|url=https://www.newzealand.com/cn/feature/the-lord-of-the-rings-trilogy-filming-locations/|title=《指环王》三部曲取景地|accessdate=2017-12-29}}</ref>，電影拍攝人員花了六個月時間搭建伊多拉斯城牆、[[馬廄|馬廄]]、正門及梅杜西金殿等相關布景，並於三週內進行拍攝<ref>{{Cite book|title=魔戒電影誕生密史|last=布萊恩·希伯理|publisher=奇幻基地|year=2002年12月|isbn=957-28136-6-8|location=臺北}}</ref><ref>{{Cite book|title=艾倫·李的《魔戒》素描集|last=艾倫·李|publisher=上海人民出版社|year=2015年8月|isbn=978-7-208-13055-5|location=上海}}</ref>。而伊多拉斯的內部場景則是在[[威靈頓|威靈頓]]拍攝的<ref>{{Cite web|url=https://www.methvenmthutt.co.nz/visit-methven-tourism-guide-information/outdoor-activities-attractions/mt-sunday-edoras-set-for-lord-of-the-rings/|title=Mt Sunday - Edoras set for LOTR|accessdate=2017-12-29|publisher=Methven Mt Hutt NZ|language=en-US}}</ref>。
