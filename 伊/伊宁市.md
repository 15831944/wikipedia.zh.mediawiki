'''伊宁市'''（{{lang-ug|'''غۇلجا'''}}, [[拉丁维文|拉丁维文]]：Ghulja），位于[[新疆|新疆]]西北部，是[[伊犁哈萨克自治州|伊犁哈萨克自治州]]首府、[[伊犁河|伊犁河]]谷内重要的物资集散地和工业中心。[[清代|清代]]为[[伊犁将军|伊犁将军]]驻地，现为[[新疆生产建设兵团|新疆生产建设兵团]][[新疆生产建设兵团第四师|第四师]]和[[武警内卫部队|武警内卫部队]][[武警第7师|第七师]]师部驻地。

== 歷史 ==
[[汉代|汉代]]，这里位于北[[丝绸之路|丝绸之路]]上，为[[乌孙国|乌孙国]]领土，归[[西域都护府|西域都护府]]指挥。后属[[西突厥|西突厥]]，并臣从于[[唐朝|唐朝]][[北庭都护府|北庭都护府]]。[[南宋|南宋]]时属[[西辽|西辽]]，[[元代|元代]]属[[察合台汗国|察合台汗国]]，[[清|清]]初属[[准噶尔|准噶尔]]汗国。[[乾隆|乾隆]]年间清军征服准噶尔，在此设[[伊犁将军|伊犁将军]]统辖整个新疆。1871年，当[[阿古柏|阿古柏]]在[[喀什|喀什]]宣布独立时，[[俄国|俄国]]占领了固勒扎（Ghulja，宁远城）和整个伊犁河谷。1881年，清朝通过谈判收回伊犁。[[光绪|光绪]]年间[[新疆建省|新疆建省]]，在此设宁远县，1913年更名为伊宁县。1944年爆发“[[三区革命|三区革命]]”时，伊宁成为[[东突厥斯坦第二共和国|东突厥斯坦第二共和国]]首都，但经过谈判和调停后，最终复归中国。1952年由[[伊宁县|伊宁县]]析置伊宁市。1954年成为伊犁哈萨克自治州首府。

==伊宁事变==
{{Main|伊宁事变}}
{{See also|新疆居民外逃事件|伊宁二五事件}}

1944年，在[[苏联|苏联]]的扶持下，伊犁和附近的塔城、阿山（今称阿勒泰）三个地区相继爆发了骚乱和武装暴动，宣布脱离[[中华民国|中华民国]]，成立东突厥斯坦共和国并定都于伊宁。在骚乱和武装暴动过程中，既有与当时统治新疆的国民党军队之间的作战，亦不乏大批屠杀汉族平民的行为。此后三区在苏联的调停下取消独立并重新归附中国（[[中华人民共和国|中华人民共和国]]政权），这一事件也被[[中国共产党|中国共产党]]定性为“为民族解放及人民民主革命事业”而进行的“革命”，因而被称为“[[三区革命|三区革命]]”。1957年，[[新疆维吾尔自治区|新疆维吾尔自治区]]政府在伊宁市西公园内修建“三区革命烈士陵园”，将三区革命领导人遗骨移葬于此处，并树立由[[毛泽东|毛泽东]]题词的纪念碑；后又在西公园内修建“新疆三区革命历史纪念馆”。

[[中华人民共和国|中华人民共和国]]成立之后，这里仍一度受到苏联的较深影响；在1960年代中苏交恶时，伊犁、塔城等地区曾有数万哈薩克农牧民及数十名军官[[伊塔事件|逃往苏联]]（今哈萨克斯坦等国）。

20世纪90年代以来，伊宁又成为[[东突厥斯坦独立运动|东突厥斯坦独立运动]]组织活动频繁的地点之一，多次有[[伊斯兰原教旨主义|伊斯兰原教旨主义]]者和[[泛突厥主义|泛突厥主义]]者在此进行[[恐怖主义|恐怖主义]]暴力活动。其中最严重的一次是1997年2月5日（当年春节之前两天）起发生的“[[伊宁二五、二六事件|伊宁二五、二六事件]]”，在骚乱中极端分子以“驱逐汉人”、“建立維吾爾伊斯兰国家”等为口号进行打砸抢烧杀，导致多名漢族平民死亡，近200人受伤。

==地理==
===气候===
伊宁市属中温带大陆性半干旱气候。四季分明，各月温差大。降水较少，但多于新疆同纬度大部分地区，且分布较为均匀。冬季长而寒冷，夏季相对较短而温暖。1月平均气温-8.8℃，极端最低气温-40.4℃（1969年1月29日）。7月平均气温23.1℃，极端最高气温39.2℃（1997年7月19日）。年日照时数2852小时。年均降水量268.8毫米。

== 人口 ==
伊宁目前所属的[[伊犁哈萨克自治州|伊犁哈萨克自治州]]管辖了新疆西北部的伊犁、[[塔城地区|塔城地区]]、[[阿勒泰地区|阿勒泰地区]]等三地区，是中国唯一的[[副省级自治州|副省级自治州]]。伊宁是其直属的县市之一，并为首府所在地。伊宁歷史上一直是多民族混居的地方。現在全市人口中[[维吾尔族|维吾尔族]]、[[汉族|汉族]]、[[回族|回族]]、[[哈萨克族|哈萨克族]]较多，另有[[蒙古族|蒙古族]]、[[锡伯族|锡伯族]]、[[乌兹别克族|乌兹别克族]]、[[俄罗斯族|俄罗斯族]]等，民族风情浓郁，其中维吾尔、回、哈萨克等民族的人口大多信奉[[伊斯兰教|伊斯兰教]]。生活在伊寧的锡伯族，系清乾隆年间从中国东北西迁而来。

== 產業 ==
[[File:1964-03_1964年_伊宁毛纺厂.jpg|thumb]]
[[File:Yining-trainstation.jpg|缩略图]]]]

伊宁地处[[伊犁河|伊犁河]]谷中部的绿洲内，全境地势平缓，年平均气温8.4℃，降水量262毫米。城区内绿树成荫，绿化率达40%以上，有“塞外江南”之称。温和湿润的气候和充裕的水资源使这里很适合发展农业。这里主要种植[[小麦|小麦]]、[[亚麻|亚麻]]、[[甜菜|甜菜]]、[[烟草|烟草]]等作物，以及[[苹果|苹果]]、[[西瓜|西瓜]]、[[甜瓜|甜瓜]]等各类瓜果。境内[[煤炭|煤炭]]资源非常丰富，工业有采矿、纺织、食品、皮革、建材等。自二十世纪九十年代起，通过临近的[[霍尔果斯口岸|霍尔果斯口岸]]，伊宁与[[哈萨克斯坦|哈萨克斯坦]]等[[中亚|中亚]]各国的贸易增长较快。

==文化和旅游==
清朝伊犁将军最初驻伊宁，并从这里统治整个新疆地区。后筑[[伊犁九城|伊犁九城]]，其中惠远城（位于今[[霍城|霍城]]）为将军驻地，而宁远城即今伊宁市为商民聚集之所。城内外古迹较多，有[[伊犁将军府|伊犁将军府]]、陕西大寺（回族大寺）、拜都拉大寺、火龙洞等许多名胜和宗教建筑。
