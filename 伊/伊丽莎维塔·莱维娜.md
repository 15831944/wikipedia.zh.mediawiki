'''伊丽莎维塔（丽莎）莱维娜'''（{{lang-en|Elizaveta (Liza) Levina}}）是俄裔美国的 [[数理统计学|数学统计学家]]{{r|isi}}，[[密歇根大学|密歇根大学]]统计系Vijay Nair讲席教授{{r|profile}} 。她在高维统计（协方差矩阵估计（covariance matrix estimation）、[[圖模式|图模型]] （graphical models）），统计网络分析，及 [[非参数统计|非参数统计]] 等多个统计学领域做出了杰出贡献{{r|promo}}。

== 学术生涯 ==
莱维娜1994年在 [[圣彼得堡国立大学|圣彼得堡国立大学]] 取得数学学士学位，1997年在 [[犹他大学|犹他大学]] 取得数学硕士毕业，之后她前往 [[加利福尼亚大学伯克利分校|加利福尼亚大学伯克利分校]] 统计系深造，师从于当代最著名的统计学家之一[[彼得·毕克尔|彼得·毕克尔]]， 2002年取得统计学哲学博士学位{{r|promo}}。 她的博士论文题为《纹理分析中的统计学问题》{{r|mgp}}。

同年，莱维娜成为[[密歇根大学|密歇根大学]]统计系助理教授{{r|profile}}，2009年晋升为副教授{{r|promo}}，2014年晋升为正教授{{r|promo2}}。她从2012年起担任系博士生项目指导教授，2014年起担任系副主任。她同时也担任密歇根数据科学研究所（Michigan Institute for Data Science, MIDAS）和复杂系统研究中心（Center for the Study of Complex Systems）的兼职教授{{r|faculty_feature}}。

== 荣誉 ==
莱维娜2010年获得 [[美国统计协会|美国统计协会]] 颁发的诺瑟年轻学者奖（Gottfried E. Noether Award），以表彰她在非参数统计领域的突出贡献{{r|noether}}。该奖金每年授予一位青年统计学家。

她2011年当选为 [[国际统计学会|国际统计学会]]（ISI） 会员{{r|isi}}，2016年当选为 [[美国统计协会|美国统计协会]] (ASA) 会员{{r|fasa}}和[[国际数理统计学会|国际数理统计学会]] （IMS）会员{{r|imsf}}。

从2017年起，莱维娜成为密歇根大学统计系Vijay Nair讲席教授{{r|nair}}。

莱维娜应邀在2018年[[国际数学家大会|国际数学家大会]]上做45分钟报告{{r|icm}}。 2019年被选为[[国际数理统计学会|国际数理统计学会]] 奖章讲座（Medallion Lecture）演讲人之一{{r|medallion}}。

莱维娜教授是2016年汤普森ISI“高被引用作者”之一{{r|isi}}。

== 家庭 ==
莱维娜的丈夫，爱德华·伊欧尼迪斯（Edward Ionides）也在密歇根大学统计系任正教授{{r|family_0}}{{r|family_01}}。 他们有三个孩子{{r|family}}。
