'''伊奘冉尊'''（{{lang|ja|いさなみ の みこと/いざなみ の みこと}} ，《[[古事記|古事記]]》作伊邪那美命，又作伊耶那美）是[[日本神話|日本神話]]中開天闢地的[[神祇|神祇]]，為兄長[[伊奘諾尊|伊奘諾尊]]（{{lang|ja|いさなき の みこと}}）的妹妹同時也為妻子。又稱作黃泉津大神、道敷大神。

== 神話故事 ==
詳細可參考[[神誕生|神誕生]]或[[国诞生|國誕生]]。

與祂的兄長[[伊奘諾尊|伊奘諾尊]]（{{lang|ja|いさなき の みこと}}）作為日本神話中開天闢地的第七代兄妹神祇出場。日本列島以及許多島嶼等（如：淡路島、隱歧島、包括日本本島在內）皆由牠們所創造，其後兄妹產生眾島和諸神。

'''伊奘冉尊'''在生產火神[[軻遇突智|軻遇突智]]（日語：かぐつち，《[[古事記|古事記]]》作[[迦具土|迦具土神]]、火之迦具土神等，《[[日本書紀|日本書紀]]》作[[軻遇突智|軻遇突智]]、火產靈）時，因其火而燒傷[[生殖器|陰部]]而臥病在床最後死亡，在死前的嘔吐物、尿等也同時誕生出其他神祇。之後[[伊奘諾尊|伊奘諾尊]]憤而殺死[[軻遇突智|軻遇突智]]。

'''伊奘冉尊'''死後，[[伊奘諾尊|伊奘諾尊]]思念'''伊奘冉尊'''而到[[黃泉|黃泉之國]]並且打算帶'''伊奘冉尊'''回到地上[[葦原中國|葦原中津國]]。當時約定在回到地上之前絕對不能回頭看'''伊奘冉尊'''，這樣'''伊奘冉尊'''才能得以回到地上。之後因為在路途中回頭看到'''伊奘冉尊'''從腐敗的屍體逐漸回復成人的過程而讓'''伊奘冉尊'''無法復活完全，並且嚇到逃跑。'''伊奘冉尊'''對於無法完全復活而感到憤怒便開始追[[伊奘諾尊|伊奘諾尊]]並且派出鬼追。最後[[伊奘諾尊|伊奘諾尊]]在連繫黃泉路跟地上[[葦原中國|葦原中津國]]之間的[[黃泉比良坂|黃泉比良坂]]（日語：よもつひらさか）用一顆大岩石阻擋道路。

之後'''伊奘冉尊'''再也回不到地上[[葦原中國|葦原中津國]]並且成為[[黃泉|黃泉]]的主宰神，因此也被稱作黃泉津大神、道敷大神。
