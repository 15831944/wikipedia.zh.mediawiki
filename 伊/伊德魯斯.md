'''伊德魯斯'''（{{lang-id|Idrus}}，{{bd|1921年|9月21日|1979年|5月18日|CatIdx=Idrus}}），[[印度尼西亞|印度尼西亞]]（印尼）作家，以寫實小說聞名，在[[印度尼西亞文學|印尼文學]]史當中稱為「45年派散文家的代表人物」。

== 生平 ==
伊德魯斯在1921年9月21日生於[[巴東_(印尼)|巴東]]（今屬[[西蘇門答臘省|西蘇門答臘省]]）{{sfn|Badan Bahasa, Idrus}}。1942年[[荷屬東印度日佔時期|日本佔領東印度群島]]之前，他只是在荷蘭殖民政府開辦的學校讀書{{sfn|Pemprov DKI Jakarta, Idrus}}，閱讀西方的文學作品，學習編寫短篇小說{{sfn|Badan Bahasa, Idrus}}。1943年畢業後，伊德魯斯加入東印度群島的官辦出版社——[[圖書編譯局|圖書編譯局]]，擔任編輯{{sfn|Pemprov DKI Jakarta, Idrus}}。他在1943年至1944年期間創作了6篇短篇小說，之後把這些作品輯錄成書，取名為《地下隨筆》（''Corat-Coret di Bawah Tanah''）{{sfn|Balfas|1976|p=88}}。

1945年印尼的革命家在[[日本投降|日本投降]]之後[[印度尼西亞獨立宣言|宣布獨立]]；後來[[荷蘭|荷蘭]]人重返印尼，掀起了[[印度尼西亞獨立革命|印尼獨立革命]]的序幕。這時候伊德魯斯一直待在[[東爪哇省|東爪哇省]][[泗水_(印尼)|泗水]]市，他還在這裏目睹[[奧貝爾丁·瓦爾特·薩森·馬拉貝|奧貝爾丁·瓦爾特·薩森·馬拉貝]]率領的英軍和[[穆斯多波|穆斯多波]]率領的革命軍因為溝通失誤而展開的一場戰鬥{{sfn|Parrott|1975|p=85}}，這場戰鬥史稱[[泗水戰役|泗水戰役]]{{sfn|Balfas|1976|p=88}}。為此他創作了一篇短篇小說《[[泗水_(小說)|泗水]]》，剖析1945年10月至1946年5月戰鬥前後軍人和平民面臨的問題。印尼作家、文學評論家[[穆罕默德·巴爾法斯|穆罕默德·巴爾法斯]]說「可能這就是唯一一部諷刺印尼革命的作品了。{{sfn|Balfas|1976|p=89}}」

戰爭期間伊德魯斯開始變得愛好沉思，這時候他創作的作品包括短篇小說《通往羅馬的另一條路》（''Jalan Lain ke Roma''），中篇小說《婦女與民族》（''Perempuan dan Kebangsaan''，1949年首刊於《印度尼西亞》雜誌）和長篇小說《阿吉》（''Aki''，1950年）。後來由於受到[[印度尼西亞共產黨|共產黨]]支持的文化組織[[人民文化協會|人民文化協會]]的逼迫，伊德魯斯選擇逃往馬來西亞。他在馬來西亞發表了兩部作品《眼睛睜開》（''Dengan Mata Terbuka''，1961年）和《人的良心》（''Hati Nurani Manusia''，1963年），不過這兩部作品的評價比之前的作品遜色{{sfn|Balfas|1976|p=90}}{{sfn|Teeuw|1980|p=218}}。

他在1965年前往[[澳洲|澳洲]][[墨爾本|墨爾本]]，擔任[[蒙纳士大学|蒙纳士大学]]的講師。1974年，他在同一所大學考獲文學碩士學位；本來他還想在這所大學一邊擔任講師，一邊修讀博士課程，不過還沒畢業，他就在1979年5月18日於巴東甘榜丹那（Kampung Tanah）逝世{{sfn|Pemprov DKI Jakarta, Idrus}}。

== 風格 ==
[[File:Idrus,_Dari_Ave_Maria_ke_Djalan_Lain_ke_Roma,_1948_(obverse).jpg|thumb]]
伊德魯斯的寫作風格注重於現實殘酷的一面。他的文句短小精幹，而且還放棄了圖書編譯局派和《[[新作家|新作家]]》派作家的審美觀{{sfn|Teeuw|1980|p=217}}。巴爾法斯認為他的文筆滿是對世事的冷嘲熱諷和諷刺{{sfn|Balfas|1976|p=88}}，不過在獨立革命接近尾聲之際，他也變得越來越愛好沉思{{sfn|Balfas|1976|p=90}}。研究印尼文學的荷蘭學者[[A·德歐|A·德歐]]認為伊德魯斯可能是受到[[欧内斯特·海明威|欧内斯特·海明威]]、[[約翰·史坦貝克|約翰·史坦貝克]]、[[欧斯金·考德威尔|欧斯金·考德威尔]]和[[威廉·艾斯霍特|威廉·艾斯霍特]]的影響{{sfn|Teeuw|1980|p=218}}。

伊德魯斯筆下的角色大都是平常人，和[[阿米爾·哈姆扎|阿米爾·哈姆扎]]這些較早期的作家不一樣——他們筆下的角色大都是王室貴族。造成這個現象的原因之一是1942年初至1945年中期間日本佔領地政府的政治動機——他們認為促進群眾的權益是必要的，這樣自身的統治才能夠維持下去{{sfn|Balfas|1976|p=88}}。

== 評價 ==
德歐指出，經常有人認為伊德魯斯就是散文界的[[凱里爾·安哇爾|凱里爾·安哇爾]]，也就是藉着自己的作品為相關的體裁引入嶄新的寫作風格{{sfn|Teeuw|1980|p=217}}。也有說法認為在[[印度尼西亞文學|印度尼西亞文學]]史裏面，伊德魯斯是45年派散文家的代表人物{{sfn|Pemprov DKI Jakarta, Idrus}}。德歐不同意這個觀點，他認為安哇爾的貢獻重要得多{{sfn|Teeuw|1980|p=217}}。

== 個人生活 ==
伊德魯斯在1946年迎娶勒娜·蘇里（Ratna Suri），兩人育有四子兩女。伊德魯斯精通多國語言，除了印尼語，他還會說[[德語|德語]]、[[英語|英語]]和[[荷蘭語|荷蘭語]]{{sfn|Badan Bahasa, Idrus}}。
