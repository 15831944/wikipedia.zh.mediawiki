'''伊州区'''（{{lang-ug|ئىۋىرغول رايونى}}），是[[中华人民共和国|中国]][[新疆维吾尔自治区|新疆维吾尔自治区]][[哈密市|哈密市]]的[[市辖区|市辖区]]，為市政府所在地。總面积81,794平方公里，总人口56.78万人，市区人口28.6万人，有[[汉族|汉族]]、[[回族|回族]]、[[维吾尔族|维吾尔族]]、[[哈萨克族|哈萨克族]]等21个民族。伊州區是[[丝绸之路|丝绸之路]]重镇，也是聞名遐邇的[[哈密瓜|哈密瓜]]产地。

== 历史 ==
哈密古称「[[昆莫|昆莫]]」，[[塞人|塞人]]之地。

公元前2世纪初，稱伊吾盧，屬[[匈奴|匈奴]]，為[[呼衍王|呼衍王]]王庭。[[東漢|東漢]][[永平_(东汉)|永平]]十六年（73年），[[汉明帝|明帝]]征匈奴取得伊吾盧，置[[宜禾都尉|宜禾都尉]]屯田。[[汉章帝|章帝]]時罷屯田，又歸匈奴。[[三國|三國]]屬[[鮮卑|鮮卑]]西部。[[晋朝|晉]]屬[[敦煌郡|敦煌郡]]，置[[伊吾都尉|伊吾都尉]]。後[[前涼|前涼]]取之，仍置伊吾都尉。[[北魏|北魏]]置[[伊吾郡|伊吾郡]]。後為[[西突厥|西突厥]]佔據。[[隋|隋]][[大业_(年号)|大業]]六年（610年），[[隋炀帝|煬帝]]派[[薛世雄|薛世雄]]攻取，建築新城，置伊吾郡，並號稱是新伊吾。隋末復為西突厥乘亂奪據。[[唐|唐]][[贞观_(唐朝)|貞觀]]四年（630年），內附，置[[西伊州|西伊州]]。貞觀六年（632年），置都督府，並改名伊州，下設伊吾、納職和柔遠三縣，其中今日的哈密當時即屬[[柔遠縣|柔遠縣]]管轄。[[天宝_(唐朝)|天寶]]初年，改為伊吾郡，不久又恢復原樣。[[广德_(唐朝)|廣德]]以後，失陷於[[吐蕃|吐蕃]]。[[五代|五代]]為[[小月氏|小月氏]]佔據，稱為[[胡盧磧|胡盧磧]]。[[北宋|北宋]][[雍熙|雍熙]]以後，屬[[回鶻|回鶻]]。[[耶律大石|耶律大石]]西遷後歸[[西遼|西遼]]。[[元朝|元]]屬[[畏吾兒|畏吾兒]]，後在西域設[[别失八里等處行中書省|别失八里等處行中書省]]，哈密屬之，並封[[察合台|察合台]]的後裔[[兀納失里|兀納失里]]為肅王，哈密成為他的封地。當時稱伊州為哈密力，從此哈密之名便沿用至今。

[[明朝|明]][[洪武|洪武]]十三年（1380年），兀納失里開始向明朝納貢，封哈密國王。[[永乐_(明朝)|永樂]]二年（1404年）六月，兀納失里弟[[安克帖木兒|安克帖木兒]]遣使朝貢，封忠順王。永樂三年（1405年）春，安克帖木兒被[[鬼力赤|鬼力赤]]毒死，兄子[[脱脱_(哈密国)|脫脫]]襲位，明朝並於次年三月置[[哈密衛|哈密衛]]，哈密國成為設有明朝羈縻衛所的王國。[[正德_(明朝)|正德]]年間，哈密國被[[吐魯番汗國|吐魯番汗國]]吞併。

[[清|清]][[順治|順治]]四年（1647年），入貢。[[康熙|康熙]]十七年(1678年)，为蒙古[[准噶尔部|准噶尔部]]占领。康熙三十六年（1697年），哈密酋長[[額貝都拉|額貝都拉]]降清，擒獻準噶爾汗噶爾丹之子[[色布騰巴勒珠爾|色布騰巴勒珠爾]]，清朝授額貝都拉為札薩克，視同蒙古。額貝都拉子孫世襲爵位，管理哈密回民，史稱[[哈密回王|哈密回王]]。[[乾隆|乾隆]]二十四年（1759年），裁撤甘肅省[[靖逆廳|靖逆廳]]，置[[哈密廳|哈密廳]]，設辦事大臣、協辦大臣、撫民通判、巡檢，隸屬於[[烏魯木齊都統|烏魯木齊都統]]。乾隆三十七年（1772年），改屬甘肅省[[鎮西府|鎮西府]]，[[安西道|安西道]]道員移駐巴里坤。[[光緒|光緒]]七年（1881年），辟為商埠。光緒十年（1884年），升為[[哈密直隸廳|哈密直隸廳]]。光緒十二年（1886年），改屬新疆省，隸[[鎮迪道|鎮迪道]]，副將駐防，且考語為「衝、繁」。民國二年（1913年）改為[[哈密縣|哈密縣]]。但哈密回王的统治形式上到民國十九年（1930年）才结束。

2016年2月18日，哈密地區撤地區設地級市，原縣級哈密市改為市轄伊州區。

== 地理 ==
=== 气候 ===
伊州區属中温带大陆性干旱气候。四季分明，气温变化较为极端。1月平均气温-10.4℃，极端最低气温-32.0℃（1952年12月2日）。7月平均气温26.5℃，极端最高气温43.9℃（1952年7月15日）。年平均气温10.0℃。日照充足，年均日照时数达3285.1小时。年降水量仅39毫米。

== 旅游 ==
五堡的魔鬼城、白石头的草原风光、[[鸣沙山|鸣沙山]]的万民嘶鸣、回王坟、盖斯墓、五堡古墓群、天山庙、沁城岩画、庙尔沟、[[西路军进疆纪念园|西路军进疆纪念园]]、烈士陵园。

市南有古蹟[[哈密王墓|哈密王墓]]。

== 交通 ==
伊州區是[[中国内地|中国内地]]与[[西域|西域]]各地的[[交通枢纽|交通枢纽]]之一，[[兰新铁路|兰新铁路]]和312国道东西横穿哈密市境，另有在建的[[哈罗铁路|哈罗铁路]]连接哈密和[[罗布泊|罗布泊]]。
