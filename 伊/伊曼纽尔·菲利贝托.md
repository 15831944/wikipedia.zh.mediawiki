'''伊曼紐爾·菲利貝托'''（{{lang-it|'''Emanuele Filiberto'''}},{{bd|1528年|7月8日|1580年|8月30日|catIdx=A}}），綽號「'''鐵頭'''」（Testa di ferro）的[[萨伏依公爵|萨伏依公爵]]（1553年~1580年），因為他是精明強幹的優秀名將而得此綽號。在他年幼時，家族領地[[薩伏依|薩伏依]]自1536年長期被法國佔領[[意大利戰爭|以抗衡西班牙在義大利的霸權]]，後來因1557年西敗法的[[聖昆廷戰役|聖昆廷戰役]]而得回失去的國土，重建[[薩伏依王朝|薩伏依王朝]]。他遷都[[都靈|都靈]]並讓[[義大利語|義大利語]]成為薩國國語，大大促進了薩伏依的義大利化，但他提升薩伏依國際地位的努力卻沒有太大的成就，要到其子[[卡洛·埃曼努埃莱一世|卡洛·埃曼努埃莱一世]]才達到顯著的成就。

== 早年 ==
[[File:Bor-Nederlantsche-Oorloghen_9141.tif|thumb]]
他出生於[[香貝里|香貝里]]（當時[[薩伏依公國|薩伏依公國]]的首都，今法國東部一地），是[[薩伏依公爵|薩伏依公爵]][[查理三世_(萨伏伊)|卡洛三世]]與葡萄牙公主貝特里絲唯一長大成人的孩子。因為他8歲時父親失去了公國，他長大後就在西班牙[[哈布斯堡王朝|哈布斯堡王朝]]的手下領軍和法王[[法蘭西斯一世|法蘭西斯一世]]打仗，希望奪回祖先領地。1553年7月他佔領黑斯丁證明自己的軍事才幹；8月時他父親去世，他繼承薩伏依公爵的空殼頭銜，於1555年至1559年為他的堂兄──西王[[菲利普二世|菲利普二世]]擔任[[尼德蘭總督|尼德蘭總督]]{{sfn|Kamen|1997|p=64}}

== 光復國土 ==
1550年代伊曼紐爾率領西班牙軍入侵法國北部，並於1557年8月10日在聖昆廷締造了輝煌的勝利，{{sfn|Kamen|1997|p=67}} 使他的軍事名聲達到巔峰。他也是英王[[亨利八世|亨利八世]]女兒[[伊麗莎白一世|伊麗莎白一世]]的追求者。隨著法國和西班牙於1559年簽署有好的[[卡托-康布雷齐和约|卡托-康布雷齐和约]]（結束六十多年的[[意大利戰爭|意大利戰爭]]），伊曼紐爾的公爵領土被正式歸還，他還在法王[[亨利二世|亨利二世]]的牽線下，娶了他的表姊──法國公主瑪格麗特、貝瑞女公爵夫人，是法王亨利二世的妹妹。{{sfn|Kamen|1997|p=73-74}}兩人婚後唯一的孩子是薩沃伊的[[卡洛·埃曼努埃莱一世|卡洛·埃曼努埃莱一世]]。

雖然1559年的[[卡托-康布雷齐和约|卡托-康布雷齐和约]]將[[皮埃蒙特|皮埃蒙特]]和薩伏依的大部分還給伊曼紐爾，但是[[都靈|都靈]]、[[基耶里|基耶里]]、[[皮內羅洛|皮內羅洛]]、[[基瓦索|基瓦索]]、[[維拉諾瓦達|維拉諾瓦達]]暫時都還在法國手裡，因為法國王室認為這些土地是否直接屬於薩伏依公爵還有爭議。幸好伊曼紐爾的夫人瑪格麗特對夫家的權益據理力爭，靠著她的聰慧和品德，以及公爵的戰略技術，在談判桌上讓法國陸續歸還上述領地，直到1562年底只剩皮內羅洛、薩維利亞諾和佩羅薩未還。後來因為法國爆發從1562-1598年的[[法國宗教戰爭|宗教內戰]]，公伊曼紐爾終在1574年讓法王[[亨利三世|亨利三世]]歸還那三塊土地，同時讓制衡法軍的西班牙軍隊從薩伏依的阿斯蒂和桑梯亞撤出，完整光復了薩伏依公國的主權。他又在1563年將首都從香貝里遷到[[都靈|都靈]]（人口2.3萬），並建立大規模的堡壘工事。

== 軍事改革 ==
趁著法國陷入30多年的新、舊教內戰，他一心一意擴展領土。他特別注重軍事力量以保衛國家，像是以有限的方式建立徵兵制（民兵），對每個教堂的教區徵召一支軍隊，由在職軍人進行正規訓練，又建立地方騎兵隊，但讓中產以上者志願參加而不強迫。徵兵制不包含貴族，因為這種地方民兵只在防守戰時使用，不算[[常備軍|常備軍]]。他建立的常備軍一部分是在和平時期作防衛戍守，一部分在需要時由長期設置的隊長招募，因此常備軍主要是由公國人民組成的募兵制。除了陸軍，他也想到建立一支小型的海軍艦隊，但好不容易籌建的幾艘主力艦卻在1571年的[[勒班陀海戰|勒班陀海戰]]遭到重創，使他的海軍實力與計畫逐漸衰弱而無實際作用。<ref name="路易吉">路易吉·薩瓦爾托雷利 著，祝本雄等譯，《義大利簡史──從史前到當代》，頁381-384</ref>

== 內政集權 ==
鑒於16世紀上半國家的災禍，公爵對薩伏依的行政管理採取統一和中央集權的政策，同時削弱貴族力量。首先他不再召集國家議會，實際上等於解散能夠抗衡君權的（貴族）議會，這是因法國佔領期（1536-1559年）議會完全退化衰落，才讓伊曼紐爾有機可趁。其次他簡化、統一了民法跟刑法的立法跟司法管理，在地方法官之上增設省法官（即省督）；最高司法授權於法國在[[都靈|都靈]]和[[香貝里|香貝里]]建立的兩個議會，只是改名為元老院。第三他對薩伏依的法規、制度做了全面的進步改革，規定法條和規章都要用義大利文和法文兩樣並寫，於是義大利文取代了[[拉丁文|拉丁文]]成為官方語言。<ref name="路易吉"/>

== 財經發展 ==
財政上，伊曼紐爾為了擴大稅收以強化軍事，他大幅提升領地的直接、間接稅（包含鹽稅、葡萄酒稅和消費稅），使稅收極為沉重，但他小心謹慎地管理稅入以償付大多數債務並建立儲備基金，讓他收支平衡。在前法國佔領的經濟殘破之後，他全力發展國家的經濟，如注重農業、大力興修水利灌溉；以特權或保護政策建立多樣的手工業，如紡織業在鼓勵外國優秀工匠的移居而高度發展，又在1561年廢除[[農奴制|農奴制]]，並建立優質的郵政服務、改善貨幣流通和金融穩定。他還參與瓦爾達奧斯塔的開礦事業、改善交通；奪取鄧達伯爵封地（確保了與[[尼斯|尼斯]]的交通貿易）和奧內利亞（讓[[皮埃蒙特|皮埃蒙特]]有了出口海港），讓領土擴大也極大地有利於經濟發展。公爵甚至覬覦[[薩沃納|薩沃納]]（由[[熱那亞共和國|熱那亞共和國]]統治），但他的擴張計畫未能實現。<ref name="路易吉"/>

== 外交事業 ==
1560年代伊曼紐爾原本希望建立天主教聯盟，未能成功後轉而在西、法之間採取中立政策。在義大利積極結盟威尼斯並訂下親善協議，但因為威尼斯採取消極的中間政策（在西班牙、土耳其兩大強權中立），只對薩伏依交換友誼而不給予同盟協議的軍事義務。他為了保障國界、收復領土和戰時招募軍隊，竭盡全力跟瑞士人達成了安全協定，1560年和瑞士的天主教各省簽約，1567年又和新教各省簽約，因此收復了日內瓦湖南岸和熱克斯鎮。公爵在國內則因為本人是虔誠的天主教徒，所以初期大舉迫害新教徒，直到1561年被他傾向寬容的夫人說服而給予有限的信仰自由。但他仍大力支持1563年成功的[[特利騰大公會議|特利騰大公會議]]，回報是教宗讓薩伏依公爵獲得領地內主教、聖職的同意權；他也限制了[[宗教裁判所|宗教裁判所]]的過激的行為（和[[托斯卡納大公國|托斯卡納大公國]]一樣），如同阻止教會法院的肆無忌憚那樣。<ref name="路易吉"/>

1580年1月31日他的舅舅、葡萄牙国王[[恩里克一世_(葡萄牙)|恩里克一世]]去世，葡萄牙王國缺少男系血親的繼承人。當時伊曼紐爾努力争取葡萄牙王位，但是他很快就發現，在西班牙王腓力二世壓倒性的優勢下，自已毫无機會。於是他放棄此念，讓腓力继承了葡萄牙王位。

1580年8月他出兵攻打薩盧茲侯國（Marquisate of Saluzzo）時，死於飲酒過度的肝硬化。{{sfn|Leathes|Prothero|Ward|1964|p=399-400}}其子[[卡洛·埃曼努埃莱一世|卡洛·埃曼努埃莱一世]]將他葬在[[都靈主教座堂|都靈主教座堂]]。
