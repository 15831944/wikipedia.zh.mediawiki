'''伊一'''（{{lang-en|'''Eleven'''}}，{{bd|1989年|7月25日|||catIdx=Y伊}}），本名'''彭奕裔'''，[[湖南人|湖南人]]，2006年加盟浙江卫视，主持《中国星跳跃》、《大牌遇见好声音》。2014年因出演《奔跑吧兄弟》表现突出，被观众熟知。<ref>{{cite web |url=http://www.hunantv.com/c/20140218/1112314478.html |title=华少代班主持《谢天谢地》 三无身材遭伊一嫌弃 |accessdate=2014 |publisher=金鹰网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.163.com/14/0129/13/9JOR12MQ00031GVS.html |title=伊一马年央视春晚后台大拜年 揭秘零点跨年 |accessdate=2014 |publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web|url=http://www.hn.xinhuanet.com/2014-01/30/c_119191939.htm|title=新丝路冠军成"四小花旦" 湘妹伊一主持春晚|accessdate=2014|publisher=新华网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20140221214444/http://www.hn.xinhuanet.com/2014-01/30/c_119191939.htm|archivedate=2014-02-21}}</ref><ref>{{cite web |url=http://www.bj.xinhuanet.com/bjyw/2014-01/29/c_119184124.htm |title=疑春晚“四小花旦”主持阵容曝光 24岁伊一创纪录(图) |accessdate=2014 |publisher=新华网 |work= |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20160503164851/http://www.bj.xinhuanet.com/bjyw/2014-01/29/c_119184124.htm |archivedate=2016-05-03 }}</ref>

== 生平 ==
2001年开始學[[拉丁舞|拉丁舞]]。曾在[[浙江传媒学院|浙江传媒学院]]读大学。

2006年被[[浙江卫视|浙江卫视]]邀请加入，2010年8月正式加盟浙江卫视，最早的节目是《[[我爱记歌词|我爱记歌词]]》。

2014年主持《这就是生活》，以嘉宾身份参与《[[奔跑吧兄弟|奔跑吧兄弟]]》第一季，因吃西瓜环节表现突出，被网友称为中国版“宋智孝”、“真正的{{lang|en|Running Man}}”，12月参与电影拍摄，担任《“Give Me Five,奔跑吧,2015!”跨年晚会》主持。

2015年主持《[[奔跑吧兄弟|奔跑吧兄弟]]》2015魅力盛典、[[我看你有戏|我看你有戏]]。因出演电影处女作《奔跑吧兄弟》人气直线上升，以真实的情感流露制造了最大泪点，被制片人李亚平赞为“票房神器”。
