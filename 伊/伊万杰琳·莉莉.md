'''-{zh-tw:妮可;zh-hk:妮歌;zh-cn:妮可;}-·伊万杰琳·莉莉'''（{{lang-en|'''Nicole Evangeline Lilly'''}}，{{bd|1979年|8月3日}}），出生于[[加拿大|加拿大]][[阿尔伯塔省|阿尔伯塔省]]的[[萨斯喀彻温堡|萨斯喀彻温堡]]，是加拿大著名的模特兒和女演员。<ref>{{cite web|url=http://ent.pipi.cn/star/detail/006/5163/detail.html|title=《明星资料库：伊万杰琳·莉莉》|publisher=皮皮网|deadurl=yes|archiveurl=https://web.archive.org/web/20120325201702/http://ent.pipi.cn/star/detail/006/5163/detail.html|archivedate=2012-03-25}}</ref>

== 人物简介 ==

伊万杰琳·莉莉，出生于加拿大，在家中排行老二，后考入[[温哥华|温哥华]][[不列颠哥伦比亚省大学|不列颠哥伦比亚省大学]]，大学期间曾与[[福特|福特]]公司签约当临时演员，<ref>{{cite web|url=http://ent.pipi.cn/star/detail/006/5163/starinfo.html|title=《伊万杰琳·莉莉》|publisher=皮皮网|deadurl=yes|archiveurl=https://web.archive.org/web/20140918043024/http://ent.pipi.cn/star/detail/006/5163/starinfo.html|archivedate=2014-09-18}}</ref>毕业后在《[[迷失|迷失]]》中出演凯特而一夜成名。<ref>{{cite web |url=http://finance.ifeng.com/money/wealth/star/20090918/1256579.shtml|title=《盘点一剧成名身价百倍的外国明星》|accessdate=2009年9月18日|publisher=凤凰网}}</ref>伊万杰琳·莉莉和《迷失》中查理的扮演者[[多米尼克·莫納漢|多米尼克·莫納漢]]曾有过一段感情，后与《迷失》的制片助理[[诺曼·卡利|诺曼·卡利]]结婚，并于2011年怀孕。<ref>{{cite web |url=http://www.hinews.cn/news/system/2011/04/19/012346631.shtml|title=《迷失》女主演伊万杰琳怀孕，将迎来第一个孩子|accessdate=2011-04-19|publisher=南海网}}</ref>

== 演艺生涯 ==

* 2004年，在《[[迷失|迷失]]》中出演凯特·奥斯汀。
* 2010年，为[[欧莱雅|欧莱雅]]拍护发、护肤宣传片。<ref>{{cite web |url=http://culture.022china.com/2010/07-12/300334_0.html|title=名模伊万杰琳·莉莉2010欧莱雅广告大片|accessdate= 2010-07-12|publisher=中国网}}</ref>
