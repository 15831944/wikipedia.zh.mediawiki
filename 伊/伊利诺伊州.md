'''伊利諾州'''（{{lang-en|State of Illinois}}），又譯為-{zh-tw:'''伊利諾伊州''';zh-cn:'''伊利诺伊州''';}-，是一個位于[[美国|美国]]中西部的州，州名源自曾在此居住的伊利尼維克（{{lang|en|Illiniwek}}）[[印第安人|印第安人]][[部落|部落]]。「Illinois」這個名字就是[[法國|法國]]殖民者根據此部落名稱變形得來。伊利諾州的州府是位于该州南部的[[斯普林菲尔德_(伊利诺伊州)|斯普林菲尔德]]（Springfield）。伊利諾州的[[美國郵政署|美国邮政]]縮寫代碼为「'''IL'''」。

== 历史 ==
以印第安人伊利诺伊部落之名（Iliniwek）命名，意思是戰士。

17-18世纪法国探险家沿着[[密西西比河|密西西比河]]溯流而上，进入伊利诺伊中西部开展贸易，并开辟了密西西比河经伊利诺伊河进入[[五大湖|五大湖区]]的贸易通道，期间建立了[[皮奥里亚_(伊利诺伊州)|皮奥里亚]]等定居点。此后法国将伊利诺伊纳入[[新法蘭西|新法兰西殖民地]]。

1756-1763年的[[七年战争|七年战争]]中法国战败，将五大湖区和[[加拿大|加拿大]]割让给英国，但英国颁布禁止垦殖的禁令，不允许北美东岸[[十三个殖民地|十三个殖民地]]的居民迁入。

1783年[[巴黎條約_(1783年)|巴黎条约]]将密西西比河以东的英国殖民地割让与美国，其中包括伊利诺伊。

1809年2月3日，[[伊利诺伊领地|伊利诺伊领地]]成立。

1818年12月3日，伊利诺伊成为美国的第21个州。

1832年4月發生[[黑鷹戰爭|黑鷹戰爭]]，為美國政府與美洲原住民之間的戰爭。[[苏族|蘇族]]領導者黑鷹，率領[[密西西比河|密西西比河]]兩岸的數個蘇族，在伊利诺伊州與[[艾奥瓦州|愛荷華州]]一帶，與[[美國聯邦政府|美國政府]]作戰。原住民部落联军最終遭擊敗。

1834年[[林肯|林肯]]当选为州议员开始其[[政治|政治]]生涯，因此该州也被称为“林肯之地”（Land of Lincoln）。

1880-1920年间伊利诺伊州和芝加哥经历了经济发展的黄金时代，芝加哥人口迅速增长，与此同时伊利诺伊州多个城市成为了工业城，包括[[喬利埃特_(伊利諾州)|乔利埃特]]和[[罗克福德_(伊利诺伊州)|罗克福德]]就是在这一时期成为工业区。

1957年，[[芝加哥|芝加哥]]附近的[[阿貢國家實驗室|阿贡国家实验室]]啟動了美國的第一個實驗性核發電系統，至今有11家[[核电站|核电站]]正在運營，伊利諾伊州的核電發電量為全美第一。

1960年，[[Ray_Kroc|Ray Kroc]]在Des Plaines開設了第一家[[麦当劳|麥當勞]]專賣店（它仍然是一家[[博物馆|博物館]]，在街對面有麥當勞）。

1970年代开始，伊利诺伊州经历了去工业化和[[郊区化|逆城市化]]带来的衰退，多个城市因为税收锐减和市中心空心化出现了高犯罪率和高空置率的问题。其中一些城市因犯罪臭名昭著，如[[东圣路易斯_(伊利诺伊州)|东圣路易斯]]和[[斯普林菲尔德_(伊利诺伊州)|斯普林菲尔德]]。

== 地理與行政區劃 ==
该州位于[[美國中西部|美國中西部]]，毗邻[[密西根湖|密西根湖]]。与其接壤的州包括：北部的[[威斯康辛州|威斯康辛州]]，西部的[[愛荷華州|愛荷華州]]和[[密苏里州|密苏里州]]，南部的[[肯塔基州|肯塔基州]]和东部的[[印第安纳州|印第安纳州]]。

伊利諾州共轄有102個[[縣|縣]]，其中本州最大城市[[芝加哥|芝加哥]]，隸屬於[[庫克縣_(伊利諾伊州)|庫克縣]]。本州州政府所在地[[春田市|春田市]]（英語：Springfield），則屬[[桑加蒙縣_(伊利諾伊州)|桑加蒙郡]]管轄。請詳見：[[伊利诺伊州行政区划|伊利诺伊州行政区划]]。

== 经济 ==
2010年，伊利诺伊州[[國民生產總值|國民生產總值]]为6,442亿美元，在全美各州中名列第5位。人均國民生產總值为50,328美元，在各州中名列第15位。至2011年8月，伊利诺伊州的[[失业率|失业率]]为9.9%。
伊利诺伊州是一个工农业并举的州，在70年代以前工业占据主导地位，作为该州最大都市的芝加哥至今都有数量和规模都十分庞大的工业产业，并贡献了大量工作岗位。二战后工业转移到其他国家，伊利诺伊州的服务业开始取代工业成为经济发展的重要因素。
由于地形平坦以及土地肥沃，伊利诺伊州也是美国最重要的农业州之一，这支撑了该州农机产业的蓬勃发展。同时芝加哥设有全美乃至世界最大的谷物期货交易所，州内也有很多铁路设施专门用于粮食运输。

== 人口 ==
2010年，伊利諾州人口总数为12,830,632人，居全美第五位。美国2012年人口估算显示，该州人口为1288万。全美第三大城市[[芝加哥|芝加哥]]位于该州北部紧邻密西根湖。芝加哥是全美一个重要的工业和交通运输中心。全州超过半数的居民住在芝加哥市及其附近。

目前伊利諾州的種族比例可分為：
* 67.8% [[白人|白人]]
* 15.1% [[非裔美國人|黑人]]
* 12.3% [[拉美裔人|拉美裔人]]
* 3.4%  [[亞裔美國人|亞裔美國人]]
* 0.2%  [[印第安人|印地安原住民]]
* 1.9 % 其他混血民眾

伊利諾州人民的前五大[[籍貫|籍貫]]為：
* [[德國|德國]]人（19.6%）
* [[非裔美人|非裔美人]] (15.1%)
* [[愛爾蘭|愛爾蘭]]人（12.2%）
* [[墨西哥|墨西哥]]人（9.2%）
* [[波蘭|波蘭]]人（7.5%）

此外，五歲以下的居民占有7.1%，18歲以下則有26.1%。至於65歲以上則是12.1%。伊利諾州的[[女性|女性]]略多於[[男性|男性]]，百分比為51%。

=== 水运航道===
伊利诺伊州的水运系统有两个中心，一个是西北部的芝加哥所在的密歇根湖，另一个是西侧的密西西比河。两个航运区由伊利诺伊河和芝加哥运河相连，曾是美国中西部最重要的航运通道。随着六十年代以后美国的去工业化进程和船舶高效化(包括淘汰小型和微型船舶以及集装箱运输标准化)，加上公路运输体系的竞争，水运开始大幅度衰退，昔日繁忙的芝加哥运河被闲置，靠近市中心的河段变成观光通道，同时伊利诺伊河两岸诸多港城衰落，道两侧建立了大量自然保护区。昔日辉煌的东圣路易斯也彻底衰落。
