'''伊丽莎白·阿马利亚·欧根妮'''（{{lang-de|Elisabeth Amalie Eugenie}}，{{bd|1837年|12月24日|1898年|9月10日|catIdx=Sisi}}），通称「奧匈帝國的伊麗莎白」（{{lang|de|Elisabeth von Österreich-Ungarn}}）或“奥地利的伊丽莎白”，是[[弗朗茨·約瑟夫一世|弗朗茨·約瑟夫一世]]之妻子<ref>[http://www.worldcat.org/title/ao-di-li-yi-li-sha-bai-huang-hou/oclc/299063140 奥地利伊丽莎白皇后 ]</ref>，即[[奥地利帝国|奥地利帝国]]皇后兼[[匈牙利|匈牙利]]-{王后}-以及波希米亚及克罗地亚王后，她的美貌和魅力征服了整個[[歐洲|歐洲]]，被世人稱為「世界上最美麗的皇后」。

伊丽莎白出生在[[巴伐利亞|巴伐利亞]]的一个贵族家庭，通常被家人與朋友暱称為'''茜茜'''（{{lang|de|Sisi}}）<ref>茜茜这个德语名字经常在小说、[[茜茜公主_(電影)|电影]]和動畫系列裡写为{{lang-fr|Sissi}}。</ref>。虽然伊丽莎白身为女公爵，在她16岁遇见弗朗茨之前，却享受着自然的教育生活模式。婚姻将她推入了正式的[[哈布斯堡王朝|哈布斯堡王朝]]宫廷生活中，让她始料不及且感到志趣相异。婆婆[[苏菲公主_(巴伐利亚)|苏菲公主]]在伊丽莎白女儿出生后即接管了她们，甚至在伊丽莎白不知晓的情况下为她们[[洗礼|洗礼]]。因此在婚姻初期，她因女儿的抚养权与婆婆不和。雖然伊莉莎白在[[奧匈帝國|奧匈帝國]]的政治影響上有限，但她已經儼然成為一個[[文化偶像|文化偶像]]。伊麗莎白如同其在電影和戲劇製作一樣，被認為是一位在自由精神和傳統宮廷規矩徘徊的悲劇人物。

== 巴伐利亞女公爵 ==
[[File:Hélène_(à_gauche)_et_sa_soeur_Sissi.jpg|thumb]]
1837年12月24日伊莉莎白出生于[[德国|德国]][[慕尼黑|慕尼黑]]。伊丽莎白是[[巴伐利亚|巴伐利亚]]的[[马科西米利安·约瑟夫|马科西米利安·约瑟夫]][[公爵|公爵]]（[[维特尔斯巴赫家族|维特尔斯巴赫家族]]一个旁支）与同宗的[[露多维卡公主|露多维卡公主]]（[[巴伐利亚国王|巴伐利亚国王]][[马克西米利安一世_(巴伐利亚国王)|马克西米利安一世]]的女儿）的次女。她的童年大多在[[施塔恩贝格湖|施塔恩贝格湖]]畔的[[帕萨霍森|帕萨霍森]]城堡长大。父亲马克思公爵在宫廷中没有任何职务，他喜好旅游和马戏，不拘小节，喜欢自由自在，被人称为“疯子麦斯”，因此茜茜的童年没有受到太多拘束，并且马克思的性格也深深影响了茜茜。茜茜父母的婚姻纯粹是[[政治联姻|政治产物]]，并不幸福，并且马克思也有若干私生子女。童年的茜茜就表露出了忧郁，多愁善感的特质，这种性格伴随她一生。

1853年伊丽莎白随她母亲与18歲的姐姐海伦赴[[上奧地利|上奧地利]]的[[度假村|度假村]][[巴德伊舍|巴德伊舍]]，原定计划是海伦应当在那里引起其23歲的[[表親|表親]]、[[奥地利皇帝|奥地利皇帝]][[弗兰茨·约瑟夫一世|弗兰茨·约瑟夫一世]]的注意，并与之订婚。但出乎意外的是弗兰茨·约瑟夫一世竟然爱上了15岁的伊丽莎白。弗兰茨违拗了他母亲[[苏菲公主_(巴伐利亚)|索菲]]的意见，两人于1年後的1854年4月24日在[[维也纳|维也纳]]的[[維也納奧古斯丁教堂|奧古斯丁教堂]]结婚。弗兰茨·约瑟夫将伊舍的行宫作为结婚礼物送给了伊丽莎白，此后这座行宫改建成了一个E字形。

== 皇后時期 ==

从一开始16歲的伊丽莎白就很难接受[[哈布斯堡-洛林皇朝|哈布斯堡王朝]]宫廷内所使用的严格的宫廷规矩，因此她在皇宫里非常孤立。她本人喜欢骑马、读书和艺术，而这些又是维也纳宫廷无法理解的。婚后她生下兩個女兒、一個兒子：[[索菲|索菲]]（1855年—1857年）、[[奥地利的吉赛拉·路易斯·玛丽|吉赛拉]]（1856年—1932年）和[[皇储|皇储]][[鲁道夫_(奥匈帝国皇太子)|鲁道夫]]（1858年—1889年）。但被婆婆兼姨妈——[[苏菲公主_(巴伐利亚)|苏菲]]剥夺子女們的抚养权，她与丈夫弗兰茨·约瑟夫之间的关系开始恶化，遗传下来的精神不稳定也越来越明显。1857年悲劇發生，伊麗莎白不顧醫生的反對意見，帶她的兩位女兒离开了奥地利前往[[匈牙利|匈牙利]]度假。但兩位女孩患上[[腹瀉|腹瀉]]，儘管吉赛拉恢復得很快，但她2歲的姊姊索菲卻因此而死亡。長女的死亡導致伊麗莎白她的生活和休息造成影響，並和她的丈夫產生永久性的裂痕，他們的婚姻漸漸崩潰。1860年，伊麗莎白離開維也納後，患上了[[呼吸系統疾病|呼吸系統疾病]]，後來有人認為是[[心身症|心身症]]。她在[[馬德拉|馬德拉]]度過了冬天，在她仍堅參觀完[[伊奧尼亞群島|伊奧尼亞群島]]後再回到了維也納。不久之後抵達[[科孚島|科孚島]]時，她再次病倒。

[[File:Ferenc_József_koronázása_Budán.jpg|250px]]和伊丽莎白於[[布达佩斯|布达佩斯]]分別加冕为匈牙利國王和皇后]]

1867年，哈布斯堡王朝對動亂中的匈牙利提出[[1867年奧匈折衷方案|奥地利-匈牙利折衷方案]]。伊丽莎白始终对匈牙利民族持同情心，方案达成后她与她疏遠的的丈夫團聚一起，於[[布达佩斯|布达佩斯]][[加冕|加冕]]为匈牙利王后。隨著兩人的復合，不久她生下一個女兒，也是第四個孩子，[[奥地利的玛丽·瓦莱丽|玛丽·瓦莱丽]]（1868年—1924年），这次她坚持要按自己的方式抚养这个女兒。这是伊丽莎白与她丈夫弗兰茨·约瑟夫短暂和好的日子，她是伊丽莎白的爱女，也是父母之间的传话筒。但伊麗莎白仍然不能對她年齡較大孩子的養育有任何重大影響，反而主要是由其婆婆[[苏菲公主_(巴伐利亚)|蘇菲]]來撫養，她經常稱伊麗莎白為一位「愚蠢的年輕母親」。

[[File:Winterhalter_Elisabeth.jpg|thumb]]所繪的伊丽莎白畫像]]

但此后不久伊丽莎白就又开始了她漫无目的的旅行生活，這段期間她都沒有見到其孩子。她訪問了[[馬德拉|馬德拉]]、匈牙利、英格蘭和科孚島。1890年，她在[[希腊|希腊]]的[[科孚島|科孚島]]為她兒子的死，在島上委託建設一座宫殿。她称这座宫殿为[[阿喀琉斯宫|阿喀琉斯宫]]，并在宫前树立了一座[[阿喀琉斯|阿喀琉斯]]的像。伊丽莎白非常喜欢[[荷馬|荷馬]]著作《[[伊里亞德|伊里亞德]]》的英雄阿喀琉斯，因为两人同样的倔强。但后来她对这座宫殿又失去了兴趣。她死後於1907年，这座宫殿被卖给了德国皇帝[[威廉二世_(德国)|威廉二世]]；後來又被[[希臘|希臘]]收購了該宮殿，並改建為博物館。

伊麗莎白後來不僅只因為她的美麗而有名。報紙針對她的時尚感、飲食和運動處方、[[馬術|馬術]]運動的熱愛等發表數篇文章。她極端重視其外貌，以致花了很多時間來保持她的美麗；此外亦遵循著嚴格且苛刻的飲食和運動療法，以維持她約50公分（20英寸）的腰圍，這讓她達到近乎消瘦的程度。如果是在好幾年後，這種症狀可被列為[[神經性厭食症|神經性厭食症]]的典型案件。

伊麗莎白亦會寫詩，如〈Nordseelieder〉和〈Winterlieder〉等，兩者的靈感都來自她最喜愛的德國詩人[[海因里希·海涅|海因里希·海涅]]，她曾提到自己是[[莎士比亞|莎士比亞]]所指的童話女王[[Titania|Titania]]。她的詩大多涉及到她的行程、古典希臘和其他浪漫主題，以及對[[哈布斯堡王朝|哈布斯堡王朝]]具有諷刺意味的評論。在她沉迷住詩的這幾年，伊麗莎白深入研究古代及現代的[[希臘語|希臘語]]，並沉浸在[[荷馬|荷馬]]的《[[伊利亞特|伊利亞特]]》和《[[奧德賽|奧德賽]]》中。許多研究希臘的講師皆曾陪同皇后散步長達一小時，進而彼此討論對希臘的心得。據19世紀的當代學者指出，伊麗莎白比任何出生於巴伐利亞的希臘王后更了解希臘。

== 挫折 ==
[[File:Empress_Elisabeth_of_Austria,_1864.jpg|thumb]]所繪]]

1889年一个打斷伊麗莎白平靜的日子發生了，[[梅耶林惨案|她的儿子逝世]]。30岁的奥地利皇储[[鲁道夫_(奥匈帝国皇太子)|鲁道夫]]与女友的遺體一起在他於[[下奧地利|下奧地利]]的[[行宫|行宫]]里發現，經過事後調查認定並非謀殺而是自杀，伊丽莎白从此陷入忧鬱症，再也没有恢复过来。伊丽莎白毕生始终在别人身上找错，因此这次她也将她唯一的儿子的死归罪于他人。

魯道夫的死亡，導致伊麗莎白產生亦著名的個人標誌。从此以后她只穿黑色衣服，打着一把皮制[[傘|阳伞]]，用一把棕色扇子遮住面孔。只有極少數是由攝影師有幸趁她不知道時，趕緊拍出的其他服裝照片。同時在她的周围她收集了许多已故之人的东西：阿喀琉斯、[[海因里希·海涅|海因里希·海涅]]和她的表侄[[路德维希二世_(巴伐利亚)|路德维希二世]]。有时她梦到她自己的死，有时她会在海上希望会起大风暴，这样她可以与她的船一起沉入大海。

伊丽莎白很少见她於奧地利首都維也納的丈夫，但是在這最後幾年他們倆反而增加了通信頻率，他们的关系漸漸变为柏拉图式的關係。伊丽莎白乘麗華輪船Miramar在[[地中海|地中海]]航行，她喜爱的地点有[[蔚蓝海岸|蔚蓝海岸]]的[[Cap_Martin|Cap Martin]]，那里的旅游业在19世纪下半叶刚刚开始；[[瑞士|瑞士]]的[[日内瓦湖|日内瓦湖]]；[[奥地利|奥地利]]的[[Bad_Ischl|Bad Ischl]]，她在那里渡过夏天；[[科孚島|科孚島]]。皇后还访问了当时北方皇室没有去过的国家：葡萄牙、西班牙、摩洛哥、阿尔及利亚、马耳他、希腊、土耳其和埃及。旅行不仅变成了她生活的意义，而且帮助她逃避自己的痛苦。

== 逝世 ==
[[File:Luigi_Lucheni_02.jpg|thumb]]
[[File:Assassinato_luigi.jpg|right]]

1898年9月10日在[[瑞士|瑞士]][[日内瓦|日内瓦]]，60歲的伊丽莎白和Sztaray伯爵夫人沿着[[日内瓦湖|日内瓦湖]]边的勃朗峰滨湖路步行，准备登上日内瓦号轮船前往[[蒙特勒|蒙特勒]]，被[[意大利|意大利]]年轻的[[无政府主义|无政府主义]]者[[盧伊季·盧切尼|盧伊季·盧切尼]]用一把磨尖的[[锉刀|锉刀]]刺傷心脏。她遭到袭击后，不知道她的受傷严重程度，还继续登船。直到除去紧身衣，周围人才意识到问题的严重。终因流血过多身亡，时年60岁。她的最后一句话是：“出了什么事？”<ref name="test">Le Comte, Edward S. Dictionary of Last Words. New York: Philosophical Library, 1955, p75.</ref><ref>De Burgh, Edward Morgan Alborough, Elizabeth, empress of Austria: a memoir, J.B. Lippencott Co., 1899, p.317</ref>

据报道，盧切尼本来想[[暗杀|暗杀]][[奥尔良公爵|奥尔良公爵]]作為[[行動宣傳|行動宣傳]]，但奥尔良公爵临时将他的行程改变了。盧切尼事后说：“我只想要杀死一名皇室成员，不在意是什么人”。而伊丽莎白当时虽然匿名在日内瓦逗留，但报纸上还是报道了她的行踪，因此盧切尼决定刺杀伊丽莎白。

她被葬在维也纳的[[皇家墓穴_(維也納)|皇家墓室]]里，幾個世紀以來它一直是哈布斯堡王朝家族成員的主要安葬地。據說在得知了伊丽莎白的死亡，弗朗茨·約瑟夫一世曾悄悄地自言自語說：“她永遠不會知道我是多麼愛她。”

== 後世影響 ==
[[File:Kaiser_Franz_Joseph_tomb_-_Vienna.jpg|thumb]]

遊客常前往伊麗莎白旅行過的著名地方，無論是在[[奧地利|奧地利]]還是在國外。除了一般的如T恤衫和咖啡杯等紀念品，遊客大都希望能看到伊麗莎白在不同的地點的各式各樣的休息處，包括在[[維也納|維也納]]的[[霍夫堡皇宮|霍夫堡皇宮]]和[[美泉宮|美泉宮]]、在[[匈牙利|匈牙利]][[格德勒|格德勒]]的格德勒宮等處。

伊麗莎白熱愛[[匈牙利|匈牙利]]遠遠超過自己和[[奧地利|奧地利]]，她特別親近[[Marie_Festetics|Marie Festetics]]和[[Ida_Ferenczy|Ida Ferenczy]]。她也堅持認為她的服務人員應該要跟她一樣能流利地講匈牙利語。由於伊麗莎白對匈牙利依戀，匈牙利受惠奧地利，因為她支持有利匈牙利人的政策，但在同一時間也疏遠了維也納和捷克的波希米亞等相關政務。目前匈牙利有些地方以[[伊莉莎白橋|伊莉莎白橋]]這類方式來紀念她。隨著伊麗莎白鐵路命名之後，她也被選一個錢幣為主要的主題，發行了[[伊麗莎白西部鐵路紀念幣|伊麗莎白西部鐵路紀念幣]]。

1998年，[[Gerald_Blanchard|Gerald Blanchard]]自在奧地利[[維也納|維也納]]的美泉宮偷走了伊麗莎白曾配戴的鑽石和珍珠首飾。

在[[安德魯·洛伊·韋伯|安德魯·洛伊·韋伯]]2004年的電影版《[[歌劇魅影_(2004年電影)|歌劇魅影]]》，主角之一Christine所穿禮服的靈感是來自於[[弗朗兹·克萨韦尔·温德尔哈尔特|弗朗兹·克萨韦尔·温德尔哈尔特]]所繪的伊麗莎白畫像。（原本音樂劇版此段為劇中劇漢尼拔，所以Christine穿的並非伊麗莎白的服裝）

== 神化 ==
[[File:Hungary_szeged_elisabeth_2.jpg|thumb]]的伊麗莎白雕像]]
在奥地利与匈牙利进行折衷方案的谈判中伊丽莎白使用非官方途径起了很大的作用。她的作用在奥地利的官方历史中只简短地提到，但在匈牙利她直到今天依然被尊为一位国家的圣人。

在20世纪伊丽莎白成为了一个代表性人物，她被描述为一个喜爱自由的，被束缚在陈腐的宫廷仪式中的人。许多作家、电影编剧、戏剧作家都从她的生平中吸取题材。
[[File:Wappen_Kaiserin_Elisabeth.png|thumb]]]]
[[File:Elisabeth.jpg|thumb]]
1955年，因[[罗密·施奈德|罗密·施奈德]]与卡尔海因茨伯姆合演电影《[[茜茜公主_(電影)|茜茜公主]]》而使她的名字廣為人知，值得注意的是，電影與原著小說的原文片名「Sissi」與公主實際上的小名「Sisi」在拼字上有點出入。

伊丽莎白总是觉得她与1856年在巴黎逝世的海涅同心相契，将她自己看作海涅的学生，说海涅亲自告诉她怎样写诗。她决定她在1880年代写的诗在1950年發表。实际上这些诗一直到1980年代才發表。布莉吉特·哈曼（Brigitte Hamann）女士出版了伊丽莎白的诗歌和绘画，在进行解析后，撰写了一本传记：《伊丽莎白：不情愿的皇后》（Elisabeth, Kaiserin Wider Willen），首次向人们完整展示了和过往影视演绎截然不同的，伊丽莎白真实的一生。

1992年9月3日，取材于这部传记，一部由[[麥可孔策|麥可孔策]]编剧，[[席維斯·李維|席維斯·李維]]作曲的[[音乐剧|音乐剧]]《[[伊丽莎白|伊丽莎白]]》，首演于维也纳大剧院並创下了当地票房的历史纪录，“重振[[维也纳|维也纳]]戏剧艺术的辉煌”，自此成为迄今最成功的德语音乐剧。尽管维也纳原版停演于1998年4月25日（正巧是伊丽莎白逝世一百周年），它依然极受欢迎，各个不同版本在其他欧洲國家纷纷上演：匈牙利、荷兰、瑞典、德国、意大利、芬兰。日本[[寶塚歌劇團|寶塚歌劇團]]从1996年开始上演日本版，并在剧中添加了几首新的歌曲。2003年，《伊丽莎白》再次回到了维也纳大剧院，并在2005年12月4日落幕。从2003年开始，已有30万人观看了这部音乐剧的维也纳版现场，2012年、2013年及2015年6月13日，[[南韓|南韓]]音樂劇《Das Musical ELISABETH 伊莉莎白》，由[[玉珠鉉|玉珠鉉]]等人在南韓演出。

在维也纳的[[霍夫堡皇宫|霍夫堡皇宫]]有展示她私生活的伊丽莎白博物馆。
