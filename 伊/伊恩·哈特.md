'''伊恩·哈特'''（{{lang-en|'''Ian Hart'''}}，本名'''伊恩·戴維斯'''（'''Ian Davies'''）；{{bd|1964年|10月8日}}）是一名英國舞台劇、電視劇和電影演員，最知名於2001年[[奇幻電影|奇幻電影]]《[[哈利波特：神秘的魔法石_(電影)|哈利波特：神秘的魔法石]]》中飾演奎里納斯·奎若（Quirinus Quirrell）。<ref>{{Cite web |url = http://www.dailymail.co.uk/tvshowbiz/article-1230517/Harry-Potter-villain-attacks-audience-member-talking-West-End-play.html |title = Stage rage! Harry Potter star 'attacks a man in the audience' |date = 2009-11-25 |publisher = MailOnline |language = en}}</ref>

== 個人生活 ==
哈特的妻子名叫Lynne。這對夫婦有兩名女兒，他們居住在[[北倫敦|北倫敦]]的{{link-en|克朗奇區|Crouch End}}。
