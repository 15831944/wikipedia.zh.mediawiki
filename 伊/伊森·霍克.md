'''伊芬·格林·鶴健''' （{{lang-en|Ethan Green Hawke}}，{{bd|1970年|11月6日||}}）<ref name=FamilySearch-TBI-1970>{{cite web|title=Ethan Green Hawke - Texas, Birth Index|url=https://familysearch.org/ark:/61903/1:1:V88G-63Q|website=[[FamilySearch|FamilySearch]]|accessdate=2014-12-13}}</ref> 是一名美国电影男[[演员|演员]]、[[電影導演|导演]]及[[編劇|編劇]]。出演過著名的作品有《[[愛在黎明破曉時|愛在黎明破曉時]]》（1995年）、《[[震撼教育|震撼教育]]》（2001年）、《[[愛在日落巴黎時|愛在日落巴黎時]]》（2004年）、《[[愛在午夜希臘時|愛在午夜希臘時]]》（2013年）和《[[年少時代|年少時代]]》（2014年）。

== 早年 ==
霍克1970年11月6日生于美国[[德克萨斯州|德克萨斯州]][[奥斯汀|奥斯汀]]。他的父母当时都是[[德州大学奥斯汀分校|德州大学奥斯汀分校]]的学生，三年后两人分手。幼年时，霍克随他的母亲移居到[[新泽西州|新泽西州]][[普林斯顿|普林斯顿]]并就读于匈奴普林斯顿中学。上中学期间，霍克开始在当地剧院学习表演，并在学校的演出中登台。

== 职业生涯 ==
14岁时，霍克得到了他的第一个电影角色，参演电影《探寻者》（''[[:en:Explorers_(film)|Explorers]]''）。此后，霍克先后在英国和[[卡内基梅隆大学|卡内基梅隆大学]]学习表演，以及在[[纽约大学|纽约大学]]学习文学。在纽约期间，他与其他人一起创建了Malaparte剧团，并任艺术指导。

1988年，霍克参演电影《[[死亡诗社|死亡诗社]]》，由于电影的成功，霍克的演员事业也由此取得突破。此后，他先后接演了《[[:en:A_Midnight_Clear|A Midnight Clear]]》、《[[:en:Alive_(1993_film)|Alive]]》、《[[四个毕业生|四个毕业生]]》、《[[情留半天|情留半天]]》、《[[千鈞一髮_(電影)|千鈞一髮]]》、《[[愛在冰雪紛飛時|愛在冰雪紛飛時]]》（[[:en:Snow_Falling_on_Cedars_(film)|''Snow Falling on Cedars'']]）、《[[烈愛風雲|远大前程]]》、《[[训练日|训练日]]》等影片。2001年，他因在《训练日》中的表演得到[[奥斯卡最佳男配角奖|奥斯卡最佳男配角奖]]提名。

除了表演事業，霍克還導演過電影《[[:en:Chelsea_Walls|Chelsea Walls]]》，並出版了兩部小說。2005年，霍克因参與《[[愛在日落巴黎時|愛在日落巴黎時]]》（《情留半天》的續集）的劇本創作而獲得[[奥斯卡最佳原創劇本獎|奥斯卡最佳原創劇本獎]]的提名。

== 个人生活 ==
霍克1998年与演员[[乌玛·瑟曼|乌玛·瑟曼]]结婚，婚后育有两个小孩。2004年两人分居，2005年离婚。2008年6月，霍克跟曾照顧他孩子的保姆，麗恩·沙休恩（Ryan Shawhughes）結婚並於同年的7月18日，沙休恩誕下女兒，克萊門汀·珍·霍克（Clementine Jane Hawke）。
