'''伊凡娜·巴吉羅·馬西亞斯'''（{{lang-es|Ivana Baquero Macías}}，{{bd|1994年|6月11日}}），西班牙演員，11歲時受到導演[[葛雷摩·戴托羅|葛雷摩·戴托羅]]青睞、演出《[[羊男的迷宮|羊男的迷宮]]》，演技備受好評，因而成為[[哥雅獎最佳新人獎|哥雅獎最佳新人獎]]史上最年輕的得主。

== 早年生活 ==
伊凡娜·巴吉羅出生於[[西班牙|西班牙]][[巴塞隆納|巴塞隆納]]，並在家鄉長大、求學。她隨後進入巴塞隆納美國學校就讀，因而在母語[[西班牙語|西班牙語]]和[[加泰隆尼亞語|加泰隆尼亞語]]之外，也會說流利的[[英語|英語]]<ref>{{cite web |url=http://www.imdb.com/name/nm1419440 |title=Ivana Baquero |work=[[網際網路電影數據庫|網際網路電影數據庫]]（IMDb） |accessdate=2012年8月21日 |language=en}}</ref>。

== 演藝生涯 ==
在因為《羊男的迷宮》走紅之前，巴吉羅已經在數部電影中參與演出。其中最著名的是2005年電影《[[鬼撕人|鬼撕人]]》，和同劇演員[[卡莉絲塔·芙洛克哈特|卡莉絲塔·芙洛克哈特]]和[[理查·羅森堡|理查·羅森堡]]等合作。此外，她也在一些西班牙電視劇中露面<ref>[http://ivana-baquero.net/ Ivana Baquero Fansource] {{webarchive|url=https://web.archive.org/web/20100507205746/http://ivana-baquero.net/ |date=2010-05-07 }}</ref>。

2006年，當時11歲的巴吉羅從約一千名年輕演員中脫穎而出，獲得導演葛雷摩·戴托羅欽點而在電影《羊男的迷宮》中飾演女主角、8歲的女孩奧菲莉亞。
