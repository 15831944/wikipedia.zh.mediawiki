《'''伊藤潤二驚選集'''》（{{lang-ja|伊藤潤二『コレクション』}}）是以[[日本漫畫家|日本漫畫家]][[伊藤潤二|伊藤潤二]]的[[恐怖|恐怖]]漫畫作品《伊藤潤二傑作集》及《魔之碎片》改編而成的[[電視動畫|電視動畫]]，由[[STUDIO_DEEN|STUDIO DEEN]]製作，自2018年1月開播。

== 製作緣起 ==
2017年6月，[[朝日新聞|朝日新聞]]宣布將漫畫家伊藤潤二的作品改編為動畫<ref>{{cite web |url=https://www.animenewsnetwork.com/news/2017-06-30/work-by-horror-manga-artist-junji-ito-gets-anime-adaptation/.118229 |title=Work by Horror Manga Artist Junji Ito Gets Anime Adaptation |date=June 30, 2017 |website=[[Anime_News_Network|Anime News Network]] |accessdate=January 5, 2018 |language=en }}</ref>，同年8月宣布改編電視動畫由STUDIO DEEN製作<ref name="studio">{{cite web |url=https://www.animenewsnetwork.com/news/2017-08-10/junji-ito-anime-website-reveals-staff-visual-winter-2018-tv-premiere/.119950 |title=Junji Ito Anime's Website Reveals Staff, Visual, Winter 2018 TV Premiere |date=August 10, 2017 |website=[[Anime_News_Network|Anime News Network]] |accessdate=January 5, 2018 |language=en }}</ref>，由田頭忍擔任導演和角色設計，他表示動畫會改編自伊藤潤二的兩個短篇集《伊藤潤二傑作集》及《魔之碎片》，但為了給觀眾驚喜，他並沒有表示會改編哪些短篇<ref name="staff">{{cite web |url=https://www.animenewsnetwork.com/news/2017-10-12/junji-ito-collection-anime-reveals-visual-staff-january-7-premiere/.122619 |title=Junji Ito 'Collection' Anime Reveals Visual, Staff, January 7 Premiere |date=October 12, 2017 |website=[[Anime_News_Network|Anime News Network]] |accessdate=January 5, 2018 |language=en }}</ref>。動畫的標題在2017年10月12日宣布，同時宣布由澤田薫擔任系列構成，[[林友樹|林友樹]]負責音樂，並表示作品會有兩集[[OVA|OVA]]<ref name="OVAs">{{cite web |url=https://www.animenewsnetwork.com/news/2017-12-25/junji-ito-collection-anime-listed-with-12-tv-episodes-2-ova-episodes/.125735 |title=Junji Ito 'Collection' Anime Listed With 12 TV Episodes, 2 OVA Episodes |date=December 25, 2017 |website=[[Anime_News_Network|Anime News Network]] |accessdate=January 5, 2018 |language=en }}</ref>。
