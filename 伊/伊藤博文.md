'''伊藤博文'''（{{bd|1841年|10月16日|1909年|10月26日|catIdx=Itou, Hirobumi}}），[[日本|日本]]近代[[政治家|政治家]]，首任[[日本内阁总理大臣|日本内阁总理大臣]]，[[明治维新|明治维新]][[元老_(日本)|元老]]，[[中日甲午战争|中日甲午战争]]策划者，日本首任[[大韓帝國|朝鲜]][[统监府|统监府]]统监。1909年10月26日，在[[哈尔滨火车站|哈尔滨火车站]]被[[安重根|安重根]]刺殺。

== 幼年 ==
幼名利助，字俊辅，号春亩、滄浪閣主人，本姓[[林姓_(日本)|林氏]]。林氏是太祖出于日本第7代君主[[孝靈天皇|孝靈天皇]]之苗裔，{{tsl|ja|越智氏|越智宿禰}}姓{{tsl|ja|河野氏}}之支流。

[[天保_(日本)|天保]]12年10月16日（1841年），伊藤博文出生于[[江戶幕府|江戶幕府]][[周防國|周防國]][[熊毛郡_(山口縣)|熊毛郡]]束荷村字野尻（今日本[[山口县|山口县]]光市束荷字野尻），為山村贫农林十藏的嫡子，母親是秋山長左衛門的長女・琴子。他的父親被[[長州藩|長州藩]]{{tsl|ja|武家奉公人|藏元付中間}}水井武兵衞收为养子，改名為「水井十蔵」，而水井武兵衞又成[[長州藩|長州藩]][[足輕|足輕]]伊藤弥右衛門养子，改名為「伊藤直右衛門」，因此「水井十藏」亦改姓「伊藤」，繼承此家。

1853年，寄宿于寺院，后又在藩士福原等家作侍童，平时做杂役，闲时习文识字。

1856年，往相州服役，守卫边防。

1857年，返回长州，他先是投奔{{tsl|ja|来原良蔵|來原良藏}}，后入[[松下村塾|松下村塾]]，受教于改革派下级[[武士_(日本)|武士]]的领导人[[吉田松阴|吉田松阴]]门下，深受吉田器重。

1858年10月，伊藤去[[长崎|长崎]]，入幕府办的炮术传习所，学习军事。

1859年6月，回长州认识了[[桂小五郎|桂小五郎]]。10月，随从[[桂小五郎|桂小五郎]]一起去[[江户|江户]]。

1862年12月12日夜，伊藤与[[久坂玄瑞|久坂玄瑞]]、[[高杉晉作|高杉晉作]]等十几名志士，潜入品川御殿山新建的[[大英帝国|英国]][[公使馆|公使馆]]周围。伊藤在前面开路，手持木锯，锯断使馆四周的木栅栏，其他人随即鱼贯而入，扔出自制的[[燃烧弹|燃烧弹]]。随后，伊藤等人返回附近住处，彻夜痛饮，沉醉于[[攘夷|攘夷]]的狂热之中。

== 留學英國 ==
1863年，伊藤同[[井上馨|井上馨]]等5人，受长州藩派遣，秘密经[[中國|中國]]前往英國，進入[[倫敦大學學院|倫敦大學學院]]学习，导师[[亚历山大·威廉·威廉姆逊|亚历山大·威廉·威廉姆逊]]。他亲眼见到西方[[资本主义|资本主义]]国家的物质文明，思想发生变化，认识到仅仅凭着一股保卫民族独立的热情去盲目攘夷，絕非上策，只有开国，向西方学习，才能成就[[倒幕|倒幕]]大业。（参见「[[長州五傑|長州五傑]]」）在英期间，得知英國战舰炮轰[[萨摩藩|萨摩藩]]。

== 尊皇攘夷 ==
1864年（日本元治元年）6月，伊藤同井上回国，发现長州藩也被炮轰了，即加入長州藩军队。此时正值英法等四国联合舰队攻打[[下关市|下关]]前夕，伊藤即力劝藩主[[毛利敬親|毛利敬親]]不可攘夷。指出，与其空费国力，不如将此力用于恢复王政，「开國进取」以完成国内统一。后因藩主执意攘夷，终使发生四国舰队[[下關戰爭|炮击下關事件]]。

不久，幕府第一次讨伐长州，藩内保守派即向幕府表示「恭顺」。1865年（日本[[庆应|庆应]]元年）初，[[高杉晋作|高杉晋作]]等改革派下级武士[[功山寺舉兵|起兵]]，夺得藩政，伊藤坚决支持。伊藤是从攘夷到「开国进取」转变的先行者和促进者。
 
1866年1月，萨、长两藩下级武士结成倒幕联盟。6月，幕府第二次讨伐长州。伊藤前往石州，准备参战，抗击幕府。后受长州藩之命，去长崎向外商购买轮船，以对付幕府的海军，但未成功，又转往[[上海|上海]]。8月底买回两艘轮船。此时，征长战事以幕府失败告终。

1867年10月14日，朝廷向倒幕派发出讨幕密诏。同一天，幕府将军[[德川庆喜|德川庆喜]]提出「[[大政奉還|大政奉還]]」。12月9日（阴历），天皇发表「[[王政复古|王政复古]]」大诏令，宣布废除幕府制，建立新政府。伊藤得知讨幕消息后，即从长崎来[[兵库县|兵库]]，欲与长州藩诸队一起战斗，后因故回长州。

[[明治维新|明治维新]]成功后，在[[木户孝允|木户孝允]]、[[大久保利通|大久保利通]]和细田道一的领导下负责处理外交事务。历任外国事务局交涉员、判事，兵库县知事，会计官权判事，大藏少辅兼民部少辅，工部大辅等职。

== 担任相职 ==
[[File:ZhangShuWan_Tunnel_banner_by_Ito_Hirobumi_20120804.jpg|thumb]]
1885年12月根据他的建议废除[[太政官|太政官]]制，实行[[内阁制|内阁制]]，出任首届[[内阁总理大臣|内阁总理大臣]]兼宫内大臣，並开始起草宪法的任务。被誉为“[[明治宪法|明治宪法]]之父”。

1894年，伊藤博文参与策劃了日本对[[朝鲜王朝|朝鲜王朝]]的戰爭和中日[[甲午戰爭|甲午戰爭]]，戰后代表日本與[[清朝|清朝]]签订《[[马关条约|马关条约]]》，將[[台灣|台灣]]和[[澎湖|澎湖]]納入日本版圖。

1898年，清國[[戊戌變法|戊戌變法]]时前往[[北京|北京]]訪問，面见[[光绪皇帝|光绪皇帝]]和[[康有为|康有为]]，提供改革方针。[[慈禧太后|慈禧太后]]發動政变，大肆逮捕維新黨，博文遂參與救援被捕的[[黄遵憲|黄遵憲]]，並協助康有为和[[梁启超|梁启超]]逃往日本。

1900年，领导日本加入[[八国联军侵华战争|八国联军侵华战争]]，假藉中國的[[義和團之亂|義和團之亂]]伺機侵華。创立[[立宪政友会|立宪政友会]]，自任总裁，开日本[[两党制|两党政治]]的先河。

== 韩国统监與遇刺 ==
[[File:Prince_Ito_and_Crown_Prince_of_Korea.jpg|thumb]]|150px]]
[[File:Hirobumi_Ito.jpg|thumb]] (右) 一同坐上馬車準備前往[[統監府|統監府]]]]
[[File:Ito_before_death.jpg|thumb]]
[[File:Ito's_death.jpg|thumb]]
1905年，日本在[[日俄戰爭|日俄戰爭]]中的[[日本海海战|日本海海战]]（[[对马海峡海战|对马海峡海战]]）获胜后，伊藤博文被任命为第一任[[统监府|韩国统监]]，并于1907年迫使[[大韓帝國|大韓帝國]]签订第三次[[日韩协约|日韩协约]]，将大韓帝國变为日本的[[保護國|保護國]]。但是伊藤博文本人反對立刻合併大韩帝国，這與許多參與日本對韓政策參與者的想法並不同。

1909年，日本政府利用日俄戰爭中獲勝的有利形勢，派伊藤博文到[[哈爾濱|哈爾濱]]與俄國財政大臣[[弗拉基米尔·尼古拉耶维奇·科科夫佐夫|科科夫佐夫]]商談有關[[朝鮮半島|朝鮮半島]]，以及劃分日俄在[[中國東北|中國東北]]的勢力範圍等事宜。

1909年10月26日上午9時整，伊藤博文乘坐的專列抵達[[哈尔滨站|哈爾濱火車站]]，科科夫佐夫上車迎接。約20分鐘後，伊藤博文在科科夫佐夫陪同下，開始檢閱歡迎隊伍。9時30分，[[朝鲜|朝鲜]][[愛國主義|愛國主義者]][[安重根|安重根]]站在俄國儀仗隊後面，當伊藤博文走到俄國儀仗隊前，距安重根10步左右時，安重根穿過俄國軍人空隙，衝過儀仗隊，相距伊藤博文五步之遙，拔出[[手槍|手槍]]對準伊藤博文連發三槍。三發[[子彈|子彈]]命中伊藤博文的[[胸部|胸部]]、[[腹部|腹部]]，伊藤博文撲倒在地。隨行醫生迅速把伊藤博文抬到車上搶救，20分鐘後，伊藤博文宣告不治而亡。<ref>[http://news.xinhuanet.com/world/2014-01/19/c_119033844.htm]</ref><ref>{{cite news|title=安重根：擊斃伊藤博文 槍響震驚世界|url=http://news.xinhuanet.com/world/2014-01/19/c_119033844.htm|newspaper=新华网|date=2014-01-19}}</ref>

安重根列举了15条击毙伊藤博文的原因{{r|金崔|华文贵|李大武|徐明勋|page1=109-110|page2=46-47| page3=22-23| page4=36-37}}：
#杀死[[明成皇后|明成皇后]]
#廢黜[[朝鮮高宗|高宗皇帝]]
#强迫朝鲜签订《[[乙巳条约|乙巳条约]]》和《[[丁未条约|丁未条约]]》
#屠殺無辜的韓國人
#以武力篡奪韓國政府權力
#掠奪朝鮮鐵路、礦山、森林和河流資源
#強制使用日本紙幣
#解散韓國軍隊
#阻礙韓國教育
#禁止韓國人留學國外
#沒收和燒毀韓國教科書
#向世界各地傳播韓國希望日本保護的謠言
#欺騙日本天皇，說韓國和日本之間的關係是和平的，實際上充滿敵意和衝突
#破壞亞洲和平
#暗殺[[孝明天皇|孝明天皇]]

博文去世後，在[[日比谷公園|日比谷公園]]舉行[[國葬|國葬]]。此後主張立刻[[日韓合併|日韓合併]]的一派成為日本對韓政策的主導者。

2014年1月19日，位於哈爾濱火車站的[[安重根義士紀念館|安重根義士紀念館]]舉辦開館儀式，紀念此一歷史事件。
