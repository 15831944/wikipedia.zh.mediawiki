'''伊諾瓦科技'''（Enova Technology Corporation）於2000年4月成立，總公司位於新竹科學工業園區，是一家專注於硬體加解密解決方案應用與研發的IC設計公司，其產品提供軍事單位、政府機關、財務金融機構、研發部門、企業單位、保險與健保組織及個人資料安全保護<ref>{{Cite web|url = http://www.proware.com.tw/news/09192008_news_sp-403-aa.html|title = Proware releases the world’s most secured hardware RAID subsystem|accessdate = 2016-01-22|author = |date = |publisher = |archive-url = https://web.archive.org/web/20160128142425/http://www.proware.com.tw/news/09192008_news_sp-403-aa.html|archive-date = 2016-01-28|dead-url = yes}}</ref>。其加解密應用技術也通過美國聯邦資訊標準([[國家標準技術研究所|NIST]])及{{link-en|加拿大通信安全機構|Communications Security Establishment|加拿大通信安全機構(CSE)}}與 {{link-en|FIPS 140-2|FIPS 140-2}}<ref>{{Cite web|url = http://csrc.nist.gov/groups/STM/cmvp/documents/140-1/140crt/140crt1472.pdf|title = X-Wall MX-256C Validation Certificate|accessdate = 2016-01-21|author = |date = |publisher = }}</ref><ref>{{Cite web|url = http://csrc.nist.gov/groups/STM/cmvp/documents/140-1/140crt/140crt1471.pdf|title = X-Wall MX-256 Validation Certificate|accessdate = 2016-01-21|author = |date = |publisher = }}</ref>的認證。
== 公司發展 ==
=== 2000-2011 ===
伊諾瓦科技成立於2000年4月。總部位於[[臺灣|台灣]][[新竹市|新竹市]]。該公司於2002年推出全球第一個硬體[[加密芯片|加密]]邏輯晶片X-Wall<sup>®</sup>SE，其加密技術係做在ASIC的[[實體層|實體層]]，非一般的唯讀記憶體或軟體嵌入式(MCU)加密技術。X-Wall<sup>®</sup> SE也是開創先例搭配攜帶式電子鑰匙(Security Key)，而這個電子鑰匙與[[硬盘|硬碟]]的對應關係是一對一的，也是唯一的，如果沒有搭配正確的Security Key，則無法Access加密過的硬碟。<ref>{{Cite web|url = http://www.ctimes.com.tw/news/PrintNews.asp?O=HJM9JBKCCDWSA-0PEE|title = Enova X-Wall SE family加解密晶片|accessdate = 2016-01-21|author = |date = |publisher = }}</ref>

2003年至2010年，伊諾瓦科技陸續推出X-Wall<sup>®</sup>升級版加解密產品X-Wall<sup>®</sup> SE-A、X-Wall<sup>®</sup> LX、X-Wall<sup>®</sup> XO、X-Wall<sup>®</sup> CO & CO+、X-Wall<sup>®</sup> MX、X-Wall<sup>®</sup> FX，以及X-Wall<sup>®</sup> DX等；其中於2007年推出的X-Wall<sup>®</sup> MX，是一顆SATA轉SATA的實時加密處理器，可針對任何SATA硬碟或固態硬碟以每秒超過120MB的速度進行實時加密。此外，X-Wall<sup>®</sup> MX介於主機端SATA及用戶端SATA間，以[[高级加密标准|AES]] ECB/CBC 256-bit之強度加密整個SATA硬碟。X-Wall<sup>®</sup> MX通過FIPS 140-2 Level 2之物理安全性認證，適用於政府機關或企業的資料加密防護。而伊諾瓦科技推出SecureNAS系列，亦在網路硬碟伺服器加解密應用方面提供解決方案。

2010年，伊諾瓦科技推出的X-Wall<sup>®</sup> DX，是一顆USB轉USB的硬體加解密晶片，支援[[高级加密标准|AES]] 256-bit先進加密標準規格，其加解密引擎(encryption/decryption engine)已獲美國標準暨技術委員會([[國家標準技術研究所|NIST]]/National Institute of Standards and Technology)認證<ref>{{Cite news|url = http://www.hope.com.tw/DispProduct/tw/040310211229.shtml|title = 伊諾瓦推出首創與CPU同步的加密晶片－X-Wall|last = |first = |date = 2004-03-10|work = |accessdate = }}</ref>。

=== 2012年至今 ===
2012年至2013年，伊諾瓦科技陸續推出Enigma產品系列：Enigma 1、Enigma 2 (HCM: Hardware Real-time Crypto Module硬體加密模組)，兩者皆是連接USB埠的外接裝置，並以通過[[高级加密标准|AES]] (Advanced Encryption Standard) 256-bit的硬體加密引擎來執行加解密運算。<ref>{{Cite news|url = http://www.digitimes.com.tw/tw/dt/n/shwnws.asp?Cnlid=13&id=0000380500_09M66SFNLJFKE21UMLSWV&ct=1|title = 伊諾瓦於台北國際電腦展 推出硬體加密產品|last = |first = |date = |work = |accessdate = }}</ref>

近年來消費者開始對資安風險有較高危機意識，根據2012年在Ponemon研究機構的報告中也指出，有近半數接受調查的企業與機關，曾遺失USB隨身碟內的敏感或機密資料，而成為公司資料外洩的主要原因之一。iThome<ref>{{Cite web|url = http://www.ithome.com.tw/tech/86736|title = 加密型隨身碟 協助企業管控USB裝置|accessdate = 2016-01-21|author = |date = |publisher = }}</ref>隨即於2013年針對全景軟體、伊諾瓦科技（Enova）、Imation、金士頓、和澤電子（Leadconn）、SanDisk、Super Talent、泰騰恩科技（Tech Titan）與創見9款加密隨身碟產品進行評測。

2015年，為提升上一代MX晶片的加密安全強度及運算速度，伊諾瓦科技發表X-Wall<sup>®</sup> MX+<ref>{{Cite news|url = http://www.digitimes.com.tw/tw/dt/n/shwnws.asp?CnlID=13&Cat=10&id=416012|title = 伊諾瓦科技發表新一代實時加解密晶片|last = |first = |date = |work = |accessdate = }}</ref>，以SATA Gen3 6G的速度對文檔或整個SATA硬碟包含開機磁區、暫存區及作業系統進行實時加解密運算。此外，更包含數種授權與認證協議(如RSA，HMAC，CMAC，HASH，TCG OPAL 2.0等)，以及提供[[微软|Microsoft]] [[BitLocker|Encrypted Hard Drive]] (eDrive)解決方案。X-Wall<sup>®</sup> MX+並可加強在設備關係認證的可靠性。
