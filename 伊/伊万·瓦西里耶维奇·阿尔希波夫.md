'''伊万·瓦西里耶维奇·阿尔希波夫'''（{{lang-ru|'''Иван Васильевич Архипов}}'''，{{bd|1907年|4月18日（5月1日）|1998年|2月28日|catIdx=Ivan Vasilyevich Arkhipov}}）中苏友好时期，担任中国政务院经济总顾问、苏联顾问和专家代表团团长即苏联援华专家组总负责人，苏联部长会议第一副主席、俄中友好协会名誉主席，主张两国应建立睦邻友好的关系，极大推动了[[中苏关系|中苏关系]]正常化。<ref>[http://www.warheroes.ru/hero/hero.asp?Hero_id=15191 Герой Социалистического Труда Архипов Иван Васильевич]</ref>

== 生平  ==
1907年，出生于[[俄罗斯|俄罗斯]][[卡卢加|卡卢加]]。

1921年，在卡卢加特纳铁路机车车辆工厂工作。

1937年，任克里沃罗格冶金厂的总工程师。

1938年，任克里沃罗格市委书记，

1939年，任有色冶金部人事主任，冶金工业部副部长 。

1950年，任苏联驻华使馆参赞，接替[[伊万·弗拉基米罗维奇·科瓦廖夫|伊万·弗拉基米罗维奇·科瓦廖夫]]，担任苏联援华专家组总负责人。<ref>{{cite web |author1=刘志青 |title=新中国首席俄语翻译李越然 |url=http://dangshi.people.com.cn/GB/85038/10842003.html |website=人民网-中国共产党新闻网 |publisher=《党史博览》 |accessdate=2019-01-05 |archivedate=2010年01月26日08:12}}</ref>

1958年，苏联的对外经济关系部第一副主席。

1974年，任苏联部长会议第一副主席。

1986年，退休。

1998年，去世。

== 荣誉 ==
2009年11月7日在中国深圳举行的2009中国国际人才交流大会上，被评为“新中国60年最有影响的海外专家”，排名第一位。入选理由：“对新中国恢复国民经济和实施[[第一个五年计划_(中国)|第一个五年计划]]作出巨大贡献，为维护中苏两国人民友谊和改善中苏关系作出了不懈努力”。<ref>[http://news.sina.com.cn/c/2009-11-07/164918997744.shtml 新中国60年最有影响海外专家揭晓]</ref>
