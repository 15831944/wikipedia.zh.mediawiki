{{Engname|《端午金曲捞》|''The Golden Melody''}}，是[[中国大陆|中国大陆]][[江苏卫视|江苏卫视]]《原来是金曲》（正式播出时被更名为《[[金曲捞|金曲捞]]》）节目组于2016年[[端午节|端午节]]期间推出的一档特别节目。节目旨在关注华语乐坛本应走红却因种种原因被埋没的优秀创作，通过唤醒师的改编演唱使其华丽逆袭，重返观众视野，唤醒观众的音乐记忆<ref>{{cite web|title=《端午金曲捞》要来一场音乐的魔法|url=http://news.jstv.com/a/20160531/1464660687757.shtml|website=荔枝网|accessdate=2017-07-08|date=2016-05-31}}</ref>。《端午金曲捞》由制作过《2015－2016江苏卫视跨年演唱会》的制作人王希及其团队打造<ref>{{cite web|title=《金曲捞》总制片人王希：回归音乐唱好歌|url=http://ent.sina.com.cn/music/zy/2016-06-06/doc-ifxsvenv6615597.shtml|website=新浪娱乐|accessdate=2017-07-08|date=2016-06-06}}</ref>，由[[李好|李好]]担纲主持。节目于2016年5月21日在[[南京国际博览中心|南京国际博览中心]]录制<ref>{{cite web|title=沙宝亮参加《端午金曲捞》 录制“玩”出新花样|url=http://ent.ifeng.com/a/20160522/42624197_0.shtml|website=凤凰娱乐|accessdate=2017-07-08|date=2016-05-22}}</ref>，2016年6月9日21:20播出。及后于《金曲捞》开播前夕被更名为《金曲捞前传》，于2017年4月1日22:00重播。歌曲版权由[[QQ音乐|QQ音乐]]、[[酷狗音乐|酷狗音乐]]、酷我音乐、[[网易云音乐|网易云音乐]]、多米音乐与考拉FM联合取得。

本节目的台湾播映权由[[中天电视|中天电视]]购得，[[中天综合台|中天综合台]]于2017年12月23日20:00以《金曲捞前传》的名义重播。海外方面，[[新加坡|新加坡]]的[[HUB都会台|HUB都会台]]与[[马来西亚|马来西亚]]的[[NTV7|NTV7]]分别于2017年7月22日22:45及7月22日23:00以《金曲捞前传》的名义重播。

== 赛制 ==
《端午金曲捞》共有四首华语乐坛的遗珠之作等待被打捞。原唱委托唤醒师演绎歌曲改编版，-{zh-cn:现场200名80、90后大众听审行使投票权;zh-tw:現場200名七、八年級生作為大眾聽審，可在演唱過程中行使投票權;zh-hk:現場200名80、90後大眾聽審行使投票權;}-。歌曲演唱开始即投票开始。演唱结束，投票结束。投票数超过120票，即为唤醒成功，原唱出场同唤醒师合唱原版歌曲。反之，则为唤醒失败。

竞演现场会设置一个由5人组成的叨叨团，负责根据现场提示以及同原唱的问答互动与观众共同竞猜原唱身份<ref>{{cite web|title=《端午金曲捞》要为失意的金曲穿上时代的新衣|url=http://ent.163.com/16/0522/09/BNLKBV8G0003501E.html|website=网易娱乐|accessdate=2017-07-08|date=2016-05-22}}</ref>。

== 开场曲 ==
《端午金曲捞》的开场曲为胡瑶的《为了遇见你》，由[[薛之谦|薛之谦]]与[[李好|李好]]共同演唱。此曲之后亦被用作《[[金曲捞|金曲捞]]》首场竞演的开场曲。

== 收视率 ==
根据[[央视索福瑞|央视索福瑞]]于2016年6月9日晚统计的资料显示，《端午金曲捞》取得了首播52城收视率0.938，市场份额3.454%的成绩，为当晚省级卫视收视率最高的节目。
