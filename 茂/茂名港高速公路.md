'''茂名港高速公路'''属于[[汕湛高速公路云湛段|汕湛高速公路云湛段]]的支线高速公路，粤高速编号：'''S61'''<ref>{{Cite web |url=http://bbs.rednet.cn/forum.php?mod=viewthread&tid=46697857&page=1 |title=广东省交通运输厅关于印发广东省高速公路网路线命名和编号规则(2017年修订版)的通知 |accessdate=2018-02-08 |author=广东省交通运输厅 |date=2017-02-27}}</ref>，起点位于[[茂名市|茂名市]][[电白区|电白区]][[黄岭镇|黄岭镇]]，线路逞东南走向，终点位于[[茂名市|茂名市]][[电白区|电白区]][[马踏镇|马踏镇]]，接茂名港大道。全长33.36公里，设计时速为100公里/小时，双向4车道。项目于2015年12月28日动工，已于2017年12月28日通车<ref>{{Cite web |url=http://gd.qq.com/a/20171229/009262.htm |title=汕湛高速茂名段正式通车！途经18个镇街，13个是首次通高速 |accessdate=2018-02-08 |date=2017-12-29 |publisher=茂名发布}}</ref>。
