'''戰爭機器'''（英語：War Machine），原名'''詹姆斯·鲁珀特·「罗迪」·罗德斯'''（James Rupert "Rhodey" Rhodes），是[[漫威漫畫|漫威漫畫]]旗下的超級英雄角色之一。

2012年，IGN將戰爭機器列為「最出色的五十位復仇者」中的第31名<ref>{{cite web |last= |first= |date=April 30, 2012 |title=The Top 50 Avengers |url=http://www.ign.com/top/avengers/31 |publisher=[[IGN|IGN]] |accessdate=July 28, 2015}}</ref>。

==人物==
詹姆斯·罗德斯在越南執勤時碰到了做好第一套鋼鐵裝而在逃亡過程中被追殺的[[鋼鐵人|東尼·史塔克]]，兩人聯手打退了追兵，之後東尼隱瞞了自己鋼鐵人的身分，並為了報恩而邀罗德斯成為自己的私人飛行員和工程師。

在史塔克工業被[[鐵霸王|奧比戴亞·史丹]]搞垮後，東尼開始酗酒，結果在某次遇敵時不敵對方，這時罗德斯才發現自己的老闆竟是鋼鐵人的事實，他代替東尼穿上鋼鐵裝擊退對手，事後東尼便請他代替自己在必要時擔任鋼鐵人。

在某次事件中，東尼被迫假死，於是他將一套新開發的鋼鐵裝當成遺物託付給羅德，這時罗德斯才正式以戰爭機器的身分登場。

==能力與裝備==
罗德斯接受過美國海軍陸戰隊的訓練，這使他成為一個熟練的飛行員和得到完整的航空工程訓練，他瞭解飛行運行、也熟悉維修技術。

除此之外他也擅長使用各種常規武器，並能熟練地使用軍隊傳授的格鬥技巧。

罗德斯的動力裝甲擁有鋼鐵人裝甲的大部分能力，像是飛行（速度接近三馬赫）、超級力量（能舉起一百噸左右的重物）與耐力。除此之外也外加了許多武裝，像是集束炮、聲波、閃光彈、燃燒彈、榴彈、 微型導彈、導彈、迷你槍、格林機槍、特製大炮、磁場、電磁脈衝、激光和伸縮鋼刀等眾多武器。

比起依靠電弧反應爐發動能量衝擊波等能量攻擊的鋼鐵人，戰爭機器的武裝就比較偏向實彈攻擊為主。

==其他媒體==
=== 電影 ===

在'''[[漫威電影宇宙|漫威電影宇宙]]'''，戰爭機器於多部電影中出現。《[[鋼鐵人_(電影)|鋼鐵人]]》中由[[泰倫斯·霍華德|泰倫斯·霍華德]]演出（似乎不滿片酬而辭演）。《[[鋼鐵人2|鋼鐵人2]]》、《[[鋼鐵人3|鋼鐵人3]]》、《[[復仇者聯盟2：奧創紀元|復仇者聯盟2：奧創紀元]]》、《[[美國隊長3：英雄內戰|美國隊長3：英雄內戰]]》、《[[復仇者聯盟3：無限之戰|復仇者聯盟3：無限之戰]]》、《[[驚奇隊長|驚奇隊長]]》和《[[復仇者聯盟：終局之戰|復仇者聯盟：終局之戰]]》中的戰爭機器則換角為[[唐·奇鐸|唐·奇鐸]]演出。

《鋼鐵人2》中，[[鋼鐵人動力服#馬克二號_Mark_2|鋼鐵俠馬克二號]]（Mark II）被詹姆斯·羅德斯中校帶走，並交付給軍方，由[[賈斯汀·漢默|賈斯汀·漢默]]進行改裝。賈斯汀·漢默在右肩上加裝機槍，在左肩則裝上榴彈發射器，此裝甲仍保留了掌心的飛行穩定器兼集束炮及胸口以動力爐核心直接輸出的強化衝擊砲，為[[鋼鐵人動力服#戰爭機器I_War_Machine_Mark_I|戰爭機器馬克一號]]（War Machine Mark 1）。

《鋼鐵人3》中，為詹姆斯·羅德斯上校所量身訂製的盔甲，在左肩安裝有衝擊波發射器，此裝甲仍保留了掌心的飛行穩定器兼集束炮及胸口以動力爐核心直接輸出的強化衝擊砲，黑白兩色構成，為戰爭機器馬克二號（War Machine Mark 2）。此外，同款盔甲也以紅、白、藍的顏色在《鋼鐵人3》登場，加上胸口的星形圖案，外表與[[美國隊長|美國隊長]]相似，名為「'''[[鋼鐵愛國者|鋼鐵愛國者]]'''」（Iron Patriot）。

羅德上校也以黑白盔甲戰爭機器馬克二號於《復仇者聯盟2：奧創紀元》中登場，協助復仇者於蘇科維亞一役中擊敗[[奧創|奧創]]軍隊。於電影結尾與[[幻視_(漫畫)|幻視]]，[[緋紅女巫|緋紅女巫]]，[[獵鷹_(漫威)|獵鷹]]一同接受訓練，成為[[復仇者|復仇者]]。
 
羅德上校以戰爭機器馬克三號盔甲於《美國隊長3：英雄內戰》中登場，成為支持蘇科維亞協議陣營的一員。在追踪美国队长和冬日战士乘坐的飞机时，被試圖击落猎鹰的幻视意外击中反应炉并从高空坠入，幾乎瘫痪。

=== 遊戲 === 
*《[[漫威：未來之戰|漫威：未來之戰]]》，手機遊戲。戰爭機器是玩家可使用角色之一。
*《[[漫威英雄_Online|漫威英雄 Online]]》：玩家創角時可選擇戰爭機器為遊戲角色，或另付費購買。
*《美國超人VS卡普空群雄》，2對2形式[[格鬥遊戲|對戰格鬥遊戲]]。玩家可使用的角色之一。
*《美國超人VS卡普空群雄2》，3對3形式對戰格鬥遊戲。玩家可使用的角色之一，可以和[[鋼鐵人|鋼鐵人]]組隊使出超必殺合體技。
