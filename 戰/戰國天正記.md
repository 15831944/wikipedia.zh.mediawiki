《'''戰國天正記'''》是[[宮下英樹|宮下英樹]]所著的[[日本歷史|日本歷史]]系列漫畫。其漫畫標題有兩個解釋，第一是[[日本戰國時代|戰國時代]]，第二是主角的姓氏[[仙石秀久|仙石]]。此為宮下戰國系列第二部，全15冊。前作為[[戰國_(漫畫)|戰國]]，續作為[[戰國一統記|戰國一統記]]，另有外傳[[戰國外傳_桶狹間戰記|桶狹間戰記]]。

==故事簡介==
接續前作的劇情，消滅了[[淺井氏|淺井]]、[[朝倉氏|朝倉]]、[[足利氏|足利]]及[[比叡山|比叡山]]之後，'''織田信長'''的目標便放在[[武田氏|武田]]及[[本願寺|本願寺]]上，然而[[武田氏|武田]]與[[本願寺|本願寺]]之間卻有私底下的盟約，打算包圍[[織田氏|織田]]並殲滅之，被'''[[本願寺光佐|本願寺光佐]]'''稱為'''[[信長包圍網|天正信長包圍網]]-河童卷之計'''的戰事就此開始。

==重要人物==
===仙石家===
*[[仙石秀久|仙石秀久]]
: 通稱'''權兵衛'''，綽號是'''饅頭鼻'''，力大無窮、且擅長間諜行動，為'''羽柴秀吉'''的寄騎。在前作的[[小谷城|小谷城]]攻略戰表現優異，從信長處獲得1000石領地成了『千石武者』。
*[[本陽院|阿藤]]
: 秀久的夫人，是野野村正成之弟[[野野村幸成|野野村幸成]]之女。
*川仔
: 川伯的孫子，秀久的部下，負責管理仙石家中的內務。
*堀田正惟
: 通稱'''右馬助'''，秀久最初屬下四武士之一，個性有點粗暴。在[[伊勢長島殲滅戰|伊勢長島殲滅戰]]之後逃亡，逃亡一陣子後加入了本願寺的一向眾。
*萩原國秀
: 通稱'''孫太郎'''，秀久最初屬下四武士之一，外表俊秀如女。
*仙石盛政
: 通稱'''治左衛門'''，秀久最初屬下四武士之一，個性相當沉穩。在[[長篠之戰|長篠之戰]]中英勇陣亡。
*[[津田妙算|津田妙算]]
: 通稱'''杉坊'''，秀久最初屬下四武士之一，做盜賊期間化名'''粗葉粕太郎'''，為鐵砲傭兵團紀州[[根來眾|根來眾]][[津田氏|津田氏]]一族，是個鐵砲好手。其技術卻師承於另一傭兵團[[雜賀眾|雜賀眾]]領袖'''雜賀重秀'''之手，將他的才能視為危險的存在。
*仙石治盛
: 通稱'''右衛門'''，舊姓小川，後成為盛政家的養子。
*[[後藤基次|後藤基次]]
: 通稱'''又兵衛'''，原是小寺孝高的家臣，在孝高前去說服荒木村重失去音訊後，便暫時歸於秀久麾下。

: 通稱'''右馬頭'''，元就的長孫，毛利家的現任當主。受兩位叔父輔佐來治理毛利家。
*[[井上元盛|井上元盛]]
: 通稱'''中務丞'''，在元就幼少時於合議流放元就，專橫掌權。造成40年後使元就肅清井上一族。

: 通稱'''和泉守'''，經家的父親。
*[[吉川經家|吉川經家]]
: 通稱'''式部少輔'''，個性豪放磊落，被元春任命為[[鳥取城|鳥取城]]城主。外型是參考[[Richard_Hugh_McCaw|Richard Hugh McCaw]]而來。
*[[山縣長茂|山縣長茂]]
: 通稱'''源右衛門尉'''，經家的家臣，比誰都還信賴經家的能力。
*[[森下道譽|森下道譽]]
: 通稱'''出羽守'''，[[山名豐國|山名豐國]]前家臣，原本懷疑新上任的經家能力，共同戰鬥後漸漸認可經家。
*[[中村春續|中村春續]]
: 通稱'''對馬守'''，[[山名豐國|山名豐國]]前家臣，原本懷疑新上任的經家能力，共同戰鬥後漸漸認可經家。

===次要勢力===
====畠山家====
*[[畠山義綱|畠山義綱]]
: 通稱'''修理大夫'''，被家臣遊佐續光放逐，後請求上杉謙信幫助奪回七尾城。
*[[畠山義春|畠山義春]]
: 通稱'''民部大輔'''，義綱之弟。與兄長共同尋求謙信的協助，後謙信欲將他擁立為七尾城主。
*[[畠山義慶|畠山義慶]]
: 通稱'''修理大夫'''，義綱之子。被遊佐續光擁立為畠山家繼承人，但實權卻在續光手上。
*[[畠山春王丸|畠山春王丸]]
: 義慶之子，自稱「春」。義慶死後被遊佐續光擁立為繼承人。
*[[遊佐續光|遊佐續光]]
: 通稱'''美作守'''，畠山七人眾之一。放逐主君畠山義綱擁立其子其孫作為當主。
*[[長續連|長續連]]
: 通稱'''對馬守'''，畠山七人眾之一，深受義慶的信賴。
*[[長綱連|長綱連]]
: 長續連的長子，支持父親親織田的行動。
*[[長連龍|長連龍]]
: 通稱'''九郎左衛門'''，長續連的三子。

====別所家====
*[[別所長治|別所長治]]
: 通稱'''侍從'''，播磨國[[三木城|三木城]]主。曾經歸順於織田軍，後經賀相說服反叛織田軍。
*[[別所賀相|別所賀相]]
: 通稱'''山城守'''，別所家的首席家老，說服長治背叛織田軍。
*[[別所重棟|別所重棟]]
: 通稱'''主水正'''，賀相之弟。
*[[井上祐之|井上祐之]]
: 通稱'''一蓮坊'''，原武家階級的僧侶，因亂世重歸戰場，並負責保護主君長治的兒子。

====尼子家====
*[[尼子勝久|尼子勝久]]
: 以復興尼子家的榮耀為責，歸順於織田家，成為[[上月城|上月城]]主。
*[[山中幸盛|山中幸盛]]
: 通稱'''鹿之介'''，對於尼子家最為忠誠者。其外型參考[[席維斯·史特龍|席維斯·史特龍]]在《[[第一滴血|第一滴血]]》中的藍波造型。

===其他人物===
*[[國友藤太郎|國友藤太郎]]
: 鐵砲鍛造師，原是要暗殺羽柴秀吉的狙擊兵，後因秀吉的誠懇態度而願意幫織田家鍛造鐵砲。
*志賀源次郎
: 津田妙算盜賊時期的同夥，後出賣妙算，原欲當秀久的武士但遭拒。
*[[津田照算|津田照算]]
: 通稱'''杉坊'''，根來眾傭兵，津田妙算的堂哥。
*[[荒木村重|荒木村重]]
: 通稱'''信濃守'''，在織田軍時擔任羽柴秀吉攻略[[毛利氏|毛利氏]]的援軍，後背叛信長。
*[[松永久秀|松永久秀]]
: 通稱'''彈正少弼'''，以反覆無常及狡詐出名。
*[[小寺政職|小寺政職]]
: 通稱'''藤兵衛'''，播磨國的豪族。

*VOL.114~VOL.119－[[石山合戰|本願寺燒塵]]篇
*VOL.120~VOL.126－中國攻略篇--毛利草創之章
*VOL.127~VOL.138－中國攻略篇--渴殺鳥取之章
*VOL.139~VOL.148－武田征伐瓦解[[甲斐國|甲州]]篇
*VOL.149－結局
