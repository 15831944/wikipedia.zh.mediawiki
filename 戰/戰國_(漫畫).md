《'''戰國'''》是[[宮下英樹|宮下英樹]]所著的[[日本歷史|日本歷史]]系列漫畫。其漫畫標題有兩個解釋，第一是[[日本戰國時代|戰國時代]]，第二是同主角的姓氏[[仙石氏|仙石]]之日文讀音。此為宮下戰國系列第一部，全15冊。續作為[[戰國天正記|戰國天正記]]，另有外傳[[戰國外傳_桶狹間戰記|桶狹間戰記]]。

==作品概要==
作者不以知名度高的戰國武將作為主角，而是以在[[戶次川之戰|戶次川之戰]]大敗、知名度偏低的'''仙石秀久'''作為主角，且用'''史上最失敗卻又成功挽回的男人'''來形容他。從一介小兵到當上大名（諸侯）的心境描寫都非常細心。

本作另一個最大的特色就是作者會實地考察古戰場跟查閱歷史資料參考，推翻一些不合史實的理論。例如戰國騎兵其實都不是在馬上作戰，而是騎到敵人附近下馬開打。又如被歷史評價偏低的如'''齋藤龍興'''、'''朝倉義景'''等人，以及知名度較低的'''山崎新平'''、'''鳥居景近'''，在此作中也是由作者參考文獻畫出不同於一般人觀點的姿態。

==故事簡介==
仙石秀久是齋藤氏的家臣，作品一開始時織田軍已將攻陷[[岐阜城|稻葉山城]]，秀久在戰亂中與青梅竹馬阿蝶見面，兩人互相發誓要生存下去。其後秀久單人匹馬衝向織田軍陣地，為織田軍製造小混亂，秀久最終寡不敵眾成為戰俘……

==重要人物==
===木下家===
*[[仙石秀久|仙石秀久]]
: 通稱'''權兵衛'''，綽號是'''饅頭鼻'''，力大無窮，作事幾乎都不經大腦，乃本作的主角。
*[[木下秀吉|木下秀吉]]
: 通稱'''藤吉郎'''，後改姓為羽柴。[[織田五大將|織田五大將]]之一，秀久的上司。
*[[寧寧|寧寧]]
: 秀吉的正室，對秀吉的家臣都很照顧。
*[[羽柴秀長|羽柴秀長]]
: 通稱'''小一郎'''，秀吉的異父弟。
*[[蜂須賀正勝|蜂須賀正勝]]
: 通稱'''小六'''，秀吉的部下。
*[[竹中重治|竹中重治]]
: 通稱'''半兵衛'''，秀吉的軍師。

===織田家===
*[[織田信長|織田信長]]
: 通稱'''彈正忠'''，為人冷酷，但他的人格魅力卻能讓部下們忠心服侍。
*[[堀秀政|堀秀政]]
: 通稱'''久太郎'''，信長最寵愛的貼身護衛，長相相當秀麗。曾經對戰俘身分的仙石秀久處刑，但是穿著全身裝備的他被秀久擊敗。
*[[明智光秀|明智光秀]]
: 通稱'''十兵衛'''。[[織田五大將|織田五大將]]之一，信長都叫他「金柑」。個性狡猾，但做事深謀遠慮。曾在足利幕府麾下抵禦三好家攻擊。進入織田家後多數戰役表現活躍。外型是參考[[休·葛蘭|休·葛蘭]]而來。
*[[柴田勝家|柴田勝家]]
: 通稱'''修理亮'''。[[織田五大將|織田五大將]]之一，信長都叫他「戽斗」。被織田家中的人私底下稱為「閻羅王」。
*[[丹羽長秀|丹羽長秀]]
: 通稱'''五郎左衛門'''。[[織田五大將|織田五大將]]之一，信長都叫他「卷毛」。外型是參考[[森姆·積遜|森姆·積遜]]（Samuel L. Jackson）而來。
*[[瀧川一益|瀧川一益]]
: 通稱'''左近將監'''。[[織田五大將|織田五大將]]之一，綽號「進退瀧川」。
*[[佐久間信盛|佐久間信盛]]
: 通稱'''右衛門尉'''。由於擅長於制定行軍撤退路線，故綽號「撤退佐久間」。
*[[佐久間盛政|佐久間盛政]]
: 通稱'''玄蕃'''，信盛的姪子，仙石秀久於佐久間隊麾下時的好戰友。
*[[平手汎秀|平手汎秀]]
: 通稱'''甚左衛門'''，與佐久間信盛於三方原之戰時共同擔任協助德川軍的織田援軍。
*[[可兒吉長|可兒吉長]]
: 通稱'''才藏'''，擅於用槍，仙石秀久的戰友之一。
*[[坂井尚恒|坂井尚恒]]
: 通稱'''久藏'''，仙石秀久的戰友之一，都喚秀久為「權兄」。

===德川家===
*[[德川家康|德川家康]]
: 通稱'''次郎三郎'''，原本只是個勇猛的熱血漢子，後來開始懂得冷靜與忍耐。
*[[本多忠勝|本多忠勝]]
: 通稱'''平八郎'''，[[德川四天王|德川四天王]]之一，稱家康為大哥。
*[[榊原康政|榊原康政]]
: 通稱'''小平太'''，[[德川四天王|德川四天王]]之一，個性冷靜。
*[[酒井忠次|酒井忠次]]
: 通稱'''左衛門尉'''，[[德川四天王|德川四天王]]之首，戰略眼光高明。
*[[石川數正|石川數正]]
: 通稱'''伯耆守'''，與酒井忠次同為德川家重要的家老。

===武田家===
*[[武田信玄|武田信玄]]
: 通稱'''法性院'''，武田家當主。外型是參考[[馬龍·白蘭度|馬龍·白蘭度]]而來。
*[[諏訪勝賴|諏訪勝賴]]
: 通稱'''四郎'''，信玄的四子。
*[[山縣昌景|山縣昌景]]
: 通稱'''三郎兵衛'''，[[武田四名臣|武田四名臣]]之一，對於戰爭追求完美。外型是參考[[克拉克·蓋博|克拉克·蓋博]]而來。
*[[馬場信春|馬場信春]]
: 通稱'''美濃守'''，[[武田四名臣|武田四名臣]]之一，綽號「不死身的鬼美濃」。外型是參考[[保羅·紐曼|保羅·紐曼]]而來。
*[[高阪昌信|高阪昌信]]
: 通稱'''彈正'''，[[武田四名臣|武田四名臣]]之一，信玄極為寵愛的武田家臣。
*[[內藤昌豐|內藤昌豐]]
: 通稱'''修理亮'''。[[武田四名臣|武田四名臣]]之一。
*[[秋山信友|秋山信友]]
: 通稱'''伯耆守'''，被形容成形似猛牛、可佈可畏的男子。
*[[小山田信茂|小山田信茂]]
: 通稱'''兵衛尉'''，被稱為「燻銀」的武田家優秀將領，率領的部隊所使用的「投石」相當有名。

===本願寺===
*[[本願寺光佐|本願寺光佐]]
: 通稱'''顯如'''，石山本願寺宗主。外型是參考[[米克·賈格爾|米克·賈格爾]]而來。
*[[下間賴廉|下間賴廉]]
: 通稱'''刑部卿法印'''，被喻為「本願寺最強戰鬥坊官」，本願寺軍總司令官。

===齋藤家===
*[[齋藤龍興|齋藤龍興]]
: 通稱'''右兵衛大輔'''，稻葉山城城主時期只是個沈溺於女色的國君，亡國被流放之後開始暗中協助信長包圍網，最後投靠朝倉家。
*[[富田勢源|富田勢源]]
: 齋藤家的兵法指南役，即使齋藤家滅亡後仍繼續跟隨龍興。視力不好，但是名劍術的高手。
*阿蝶
: 齋藤家的侍女，仙石秀久的青梅竹馬。後陷於與仙石秀久及鳥居景近間的三角戀情而徬徨。
*阿鹿
: 仕於龍興的女忍者，對仙石秀久有好感。
*阿豬
: 齋藤家的侍女首領，對龍興相當忠心。

===淺井家===
*[[淺井長政|淺井長政]]
: 通稱'''備前守'''，原本只是個身軀壯碩但內心膽小的傢伙，在金崎撤退戰中，在與內心的掙扎中，決定放過信長，後來在經歷過與織田信長的戰鬥後極速成長，成為真正的戰國大名。
*[[赤尾清綱|赤尾清綱]]
: 通稱'''美作守'''，仕於淺井家三代的筆頭家老，相當了解長政。
*[[山崎新平|山崎俊秀]]
: 通稱'''新平'''，在協助足利氏防禦三好氏攻擊中表現活躍，因鼻子有傷總是帶著面具。後來與仙石秀久馬上單挑，陣亡。諱名'''俊秀'''是由作者自創。

===朝倉家===
*[[朝倉義景|朝倉義景]]
: 通稱'''左衛門督'''，生活過的太過平穩的大名，事情常只看到表面。
*[[高德院(朝倉氏)|高德院]]
: 義景的母親，對於女侍們而言是如同母親般的存在。常常罵義景是敗家子，但心理依舊很關心他。
*[[朝倉景健|朝倉景健]]
: 通稱'''孫三郎'''，常代替義景處理朝倉家中的事務。原是朝倉家中的守舊派，但了解到織田家的崛起後也願意成為革新派的一員。
*[[鳥居景近|鳥居景近]]
: 通稱'''兵庫助'''，家中革新派的中心人物。跟仙石秀久爭奪阿蝶，最後擄獲其芳心。長相帥氣，尊崇南蠻的'''騎士道'''。
*[[高橋景業|高橋景業]]
: 通稱'''新助'''，鳥居景近最好的朋友，也同樣尊崇南蠻的'''騎士道'''。
*[[山崎吉家|山崎吉家]]
: 通稱'''長門守'''，朝倉家的武鬥派重臣。

===其他人物===
*川爺
: 仙石秀久在齋藤家時對秀久很好的老爺爺，並教導了秀久在戰爭中多數人都是死於弓箭之下的道理。
*[[織田市|阿市]]
: 織田信長之妹，淺井長政之妻，被喻為絕世的美女。起初相當看不起淺井長政，後長政極速成長後不自覺的愛上了他。
*[[織田豔|阿豔]]
: 織田信長的叔母，但比信長還年輕。是[[岩村城|岩村城]]的女城主，後被武田軍的秋山信友使用懷柔而背叛至武田軍。
*[[阿閉貞征|阿閉貞征]]
: 通稱'''淡路守'''，姉川之戰時叫做阿閉貞秀，有著不錯的戰略眼光。原是淺井軍的武將，後背叛到織田軍。
*[[藤堂高虎|藤堂高虎]]
: 通稱'''與右衛門'''，以再興沒落的家名為目標。原是淺井軍的武將，後和阿閉貞征一同背叛到織田軍。在小谷城攻略戰中與仙石秀久及可兒吉長是互相競爭的好戰友。
*[[磯野員昌|磯野員昌]]
: 通稱'''丹波守'''，淺井軍的猛將。後居城被織田軍與淺井領地分斷，遭丹羽長秀長期包圍而降伏織田。
*[[朝倉景鏡|朝倉景鏡]]
: 通稱'''式部大輔'''，後受信長偏諱改名'''土橋信鏡'''。義景的表弟，對於義景相當反感。

==劇情章節==
本作劇情從1567年的[[稻葉山城之戰|織田信長攻陷稻葉山城]]開始，至1574年仙石秀久獲得[[近江國|近江國]]野洲郡的千石土地結束。故事內容有將故事分篇，以下為各篇篇名：
*VOL.1~VOL.22－仙石權兵衛登場篇
*VOL.23~VOL.31－[[金崎之戰|金崎撤退戰]]篇
*VOL.32~VOL.50－[[姉川之戰|姉川合戰]]篇
*VOL.51~VOL.76－[[比叡山|比叡山]]燒殺篇
*VOL.77~VOL.99－[[三方原之戰|三方原合戰]]篇
*VOL.100~VOL.124－[[一乘谷之戰|火燒一乘谷]]篇
*VOL.125~VOL.146－[[小谷城之戰|小谷城虎口攻略]]篇
*VOL.147~VOL.149－結局
