== 故事大綱 ==
在《[[幻影血脈|幻影血脈]]》的故事結束後50年，1938年的紐約，喬納森·喬斯達的孫子喬瑟夫·喬斯達（JoJo）開始了新的故事。

當年共同擊倒吸血鬼迪奧的羅伯特·E·O·史比特瓦根在[[墨西哥|墨西哥]]的考古團隊發現了沉睡在石柱上的古代生物「'''柱之男'''」和數十個石鬼面，便連同成了波紋門派的掌門人的史特雷一起前往墨西哥。史特雷在見到石鬼面後鬼迷心竅，将史比特瓦根打成重伤並戴上面具，成為吸血鬼。史特雷擔憂具有波紋素質的喬瑟夫將會成為自己的威脅，便前往美國企圖殺他。而[[納粹德國|納粹德國]]將領'''魯道夫·馮·修特羅海姆'''的部隊在墨西哥發現了昏厥的史比特瓦根，便把他救活、連同柱之男沉睡的石柱一同帶到了納粹基地之中。打倒了史特雷的喬瑟夫聽到史比特瓦根被納粹德國救治的消息，隻身前往墨西哥。修特羅海姆喚醒柱之男並將之命名為桑塔納，再透過實驗發現他有高度的智慧和戰鬥力，趕到基地的喬瑟夫將桑塔納打敗，但修特羅海姆也在戰鬥中自爆，並透露羅馬有另外三名柱之男的消息。

喬瑟夫前往羅馬並和威廉·A·齊貝林的孫子、同樣是波紋戰士的西薩·齊貝林會合，但兩人的實力遠不如三名柱之男：瓦姆烏、艾西迪西和卡茲。落敗的兩人向西薩的波紋師父莉莎莉莎求教，並通過了一個月的嚴苛試煉，且得知莉莎莉莎手上的艾哲紅寶石便是柱之男進化所需要的祕寶。喬瑟夫擊敗了前來奪取紅寶石的艾西迪西後，為了奪還被艾西迪西寄往[[瑞士|瑞士]][[聖莫里茨|聖莫里茨]]的紅寶石，和西薩、莉莎莉莎一同前往瑞士，他們在那裡與用納粹科技重生成[[改造人|改造人]]的修特羅海姆會合。西薩為了打倒柱之男，獨自一人潛入他們的根據地，但卻命喪瓦姆烏手中。莉莎莉莎對柱之男提出了賭上艾哲紅寶石的單挑對決，在經歷了[[馬戰車|馬戰車]]對決後，喬瑟夫擊敗了瓦姆烏，但莉莎莉莎卻敗給耍詐的卡茲。其後反過來利用對方照射的紫外線，進化為夢寐以求的究極生物。

進化成究級生物的卡茲把喬瑟夫打成重傷，在卡茲對喬瑟夫使出最後一擊時，意外擊中紅寶石並促使[[火山|火山]]噴發，火山把卡茲推到了大氣層之外，沒有辦法再回到地球的卡茲在[[宇宙|宇宙]]中生不如死地漂浮；另一方面、被火山爆發波及的喬瑟夫卻幸運的存活下來，最後與莉莎莉莎的僕人絲吉Q結婚。

=== 柱之男 ===
柱之男（{{lang|ja|柱の男}}）是故事中的主要反派，實為遠在人類出現以前就已存在的古代生物「闇之一族」，由於作品中出現的四個成員皆沉睡在石柱中、而且都是男性，故被其它劇中角色稱為柱之男，但這個族群中也有女性。柱之男畏懼陽光，只能生活在黑暗中。與人類相比、他們有更強的身體能力、更長的壽命和更高的智慧。只要接觸到不含有波紋的生物就能以溶化的方式將其吸收入自己的體內作為養分。柱之男有著自己獨特的戰鬥方式「流法」（{{lang|ja|モード}}）。柱之男的首領是卡茲，他企圖取得恩嘉紅寶石（{{lang|ja|エイジャの赤石}}）並搭配自己製作的石鬼面來讓自己再度進化。

: 第二部的最終反派。柱之男的領導者，智商最高，具有為達目的不擇手段的個性，渴望著克服[[太陽|太陽]]並再度進化，因此製造出了石鬼面。

: 在《[[幻影血脈|幻影血脈]]》後，史比特瓦根在美國發現石油，並搖身一變成為鉅富，成立了「史比特瓦根財團」（SPW財團），推動美國的醫療和經濟發展。1952年因[[心臟病|心臟病]]發而過世，享年89歲，終身[[單身|單身]]。去世後，他一手創立的財團仍持續地幫助喬斯達一家，直至系列作第六部《[[石之海|石之海]]》。

: 姓名由來是美國歌手[[史摩基·羅賓森|史摩基·羅賓森]]。

; {{nihongo|史特雷|ストレイツォ|Straits}}

: 在漫畫連載時期，有位讀者在史特雷死了之後寫了一張自己的感想寄給出版社，標題為「啊～！史特雷大人死了！」（{{lang-ja|あーん！スト様が死んだ！}}），該明信片被附在單行本的後面，由於文筆生動有趣，不但成為JoJo迷中的趣談、也被官方的電子遊戲所引用。

: 姓名由來是蘇格蘭歌手[[唐納文|唐納文]]。

; {{nihongo|羅吉茲|ロギンズ|Loggins}}、{{nihongo|梅西奈|メッシーナ|Messina}}

; {{nihongo|絲吉Q|スージーQ|Susie Q}}
: 聲優：[[小島幸子|小島幸子]]（日本）；[[黃瑩瑩|黃瑩瑩]]（香港）；[[張乃文_(配音員)|張乃文]]（台灣）

: 姓名由來有兩說：一是來自美國搖滾歌手{{link-en|Suzi Quatro|Suzi Quatro}}，二是由美國搖滾吉他手Dale Hawkins所寫的歌曲{{link-en|Susie Q|Susie Q (song)}}。

; {{nihongo|馬克|マルク|Mark}}
: 聲優：[[逢坂良太|逢坂良太]]（日本）；[[胡家豪|胡家豪]]（香港）；[[喬資淯|喬資淯]]（台灣）

; {{nihongo|喬治·喬斯達二世|ジョージ・ジョースターII世|George Joestar II}}
