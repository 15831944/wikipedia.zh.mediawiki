'''景怡'''（英語：King Yee）是香港[[東區區議會|東區區議會]]下轄的[[選區|選區]]，代號'''C12'''，成立於2004年，現任區議員為[[香港工會聯合會|工聯會]]成員[[梁國鴻|梁國鴻]]。

==範圍==
選區範圍為[[富景花園|富景花園]]和[[富怡花園|富怡花園]]，與其相鄰的選區有[[欣藍|欣藍]]、[[小西灣_(選區)|小西灣]]、[[環翠_(選區)|環翠]]和[[佳曉|佳曉]]。

==沿革==
景怡選區首次出現於[[2003年香港區議會選舉|2003年香港區議會選舉]]，因為[[藍灣半島|藍灣半島]]入伙，所以在[[欣怡|欣怡]]選區中分拆出[[富欣花園|富欣花園]]，跟藍灣半島合併成[[欣藍|欣藍]]，而[[富怡花園|富怡花園]]部份就與[[富景_(選區)|富景]]合併，改稱'''景怡'''。

[[2003年區議會選舉|2003年區議會選舉]]，當時人口估算約17,683人<ref>{{cite web |url=http://www.elections.gov.hk/elections/dc2003/tc_chi/summary/summary_e.html |title=存档副本 |accessdate=2012-04-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20111108185626/http://www.elections.gov.hk/elections/dc2003/tc_chi/summary/summary_e.html |archivedate=2011-11-08 }}</ref>，其中有10,196位是選民<ref>{{cite web |url=http://www.elections.gov.hk/elections/dc2003/tc_chi/eb/eb_num_e.html |title=存档副本 |accessdate=2012-04-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20110115012307/http://www.elections.gov.hk/elections/dc2003/tc_chi/eb/eb_num_e.html |archivedate=2011-01-15 }}</ref>。今次由原[[富景_(選區)|富景]]區區議員[[民主黨_(香港)|民主黨]][[陳耀德|陳耀德]]與[[民主建港協進聯盟|民建聯]][[唐偉源|唐偉源]]爭奪，最後陳以2,901票擊敗取得1,231票的唐偉源而勝出。

[[2007年區議會選舉|2007年區議會選舉]]，當時人口估算約16,905人<ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/summaryc.html |title=存档副本 |accessdate=2011-11-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20080501203936/http://www.elections.gov.hk/dc2007/chi/summaryc.html |archivedate=2008-05-01 }}</ref>，其中有10,432位是選民<ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/num_electors_eastern.html |title=存档副本 |accessdate=2015-12-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20150621034533/http://www.elections.gov.hk/dc2007/chi/num_electors_eastern.html |archivedate=2015-06-21 }}</ref>，陳耀德今次與同是居於[[富景花園|富景花園]]的[[民主建港協進聯盟|民建聯]][[梁國鴻|梁國鴻]]爭奪，不過陳耀德連任失敗，最後梁國鴻以2,074票險勝陳耀德28票。

[[2011年區議會選舉|2011年區議會選舉]]，當時人口估算約17,207人<ref>[http://www.elections.gov.hk/dc2011/chi/summaryc.html]</ref>，其中有11,062位是選民<ref>[http://www.elections.gov.hk/dc2011/chi/ebriefsc.html]</ref>。今次梁國鴻又與陳耀德交手。陳於選前數月成立業主立案法團取代由梁領導的業主委員會，再加上前[[民主建港協進聯盟|民建聯]]成員[[楊漢成|楊漢成]]爭奪，梁仍取得2,140票勝出。
