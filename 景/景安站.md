'''景安站'''位於[[台灣|台灣]][[新北市|新北市]][[中和區|中和區]]，為[[台北捷運|台北捷運]][[中和新蘆線|中和新蘆線]]（[[台北捷運中和線|中和線]]）與興建中的[[環狀線_(台北捷運)|環狀線中環段]]交會，其中環狀線將於2019年第四季通車。<ref>{{Cite web|url=http://www.dorts.gov.taipei/ct.asp?xItem=269400159&CtNode=20275&mp=115001|title=環狀線第一階段路線說明(精簡版)|accessdate=2017-11-18|date=2017-02-21|last=傅彥霖|work=www.dorts.gov.taipei|language=zh-TW}}</ref>

== 車站概要 ==
橘線車站位於景安路下方，環狀線車站位於景平路上的[[台64線|台64線]]快速高架道路上方；車站代碼為'''O02／Y10'''，在橘線尚未使用2016車站代碼前則採用工程代碼'''O18'''。

因本站英文站名「景安」之間未標註隔音符號（Jing' an），常常會讓外國人將本站誤念成金甘（Jin' gan）站。<ref>{{cite news|publisher=TVBS |title=「景安」變「金幹」北捷英譯站名惹議 |url=http://www.tvbs.com.tw/NEWS/NEWS_LIST.asp?no=miffy011820121223142631|date=2012-12-23}}</ref>

== 車站構造 ==
未來將成為高架與地下交會車站，高架站體完工後亦將與地下站體共用此出口。

另外，車站主體上方的共構住宅大樓已經興建完成，未來大樓下半部將成為中和新蘆線與環狀線的轉乘電扶梯通道，地面大廳層的付費區內部屆時亦將進行改建，以成為兩線的轉乘區。

*{{中和新蘆線}} ：地下六層車站，月台型式採[[側式疊式月台|側式疊式月台]]，本站開挖深度約30公尺，兩個[[側式月台|側式月台]]分別位於地下第四層與第六層兩月台層間上下平行重疊，本站設有[[半高式月台門|半高式月台門]]。
*{{環狀線}}：地上五層車站，兩個[[側式月台|側式月台]]，本站設有[[全高式月台門|全高式月台門]]。

== 利用狀況 ==
由於距離[[福爾摩沙高速公路|國道三號]]（福爾摩沙[[高速公路|高速公路]]）[[中和交流道|中和交流道]]的路程約1.7公里，因此有許多自臺北市區前往其他地區的國道客運路線，行經車站出口旁的景平路，並設站停靠，可轉乘前往[[中壢區|中壢]]、[[龍潭區_(桃園市)|龍潭]]、[[臺中市|臺中]]、[[桃園國際機場|桃園國際機場]]等地。另外也有可轉乘前往[[三峽區|三峽]]的快速公車路線。本處除捷運站外，景安路及景平路亦設有「捷運景安站」公車站牌，有多條公車路線行經。另外，台北捷運中採[[側式疊式月台|側式疊式月台]]的地下車站除本站外尚有3座，包括[[台北橋站|台北橋站]]、[[永安市場站|永安市場站]]及[[府中站|府中站]]。除此之外，環狀線有高架的側式月台並不是側式疊式月台。

根據{{TRTS ridership-date}}資料，景安站每日旅運量約為{{TRTS ridership|station=景安站}}人次，在台北捷運各站中排行第{{TRTS ridership-rank|station=景安站}}名。此外也為了防止人員蓄意跳軌等因素，本站在2016年增設[[半高式月台門|半高式月台門]]來保護民眾與行車的安全。
