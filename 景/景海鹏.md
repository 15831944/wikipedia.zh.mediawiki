'''景海鹏'''（{{bd|1966年|10月24日|catIdx=J景}}），男，汉族。[[中华人民共和国|中国]][[山西省|山西]][[运城市|运城市]]东杨家卓村人。[[中国人民解放军少将|中国人民解放军少将]]，[[中国航天员大队|中国航天员大队]]特级[[航天员|航天员]]，[[副军职|副军职]]{{NoteTag|2016年10月16日出席新闻发布会左胸前的[[级别资历章|级别资历章]]为黄色上面一颗金星。}}。

== 生平 ==
1984年，[[中国人民解放军空军|中国人民解放军空军]]在山西运城招考飞行员，第一次报名的景海鹏由于身体的原因而落选了<ref>{{cite book|editor=刘芳|title=走近太空之子|date=2012年3月|publisher=合肥：安徽文艺出版社|isbn=978-7-5396-3961-1 |page = 150 |accessdate=2016-10-17}}</ref>。复读一年后，从解州中学成功考上河北保定航校<ref>{{cite book |editor=《飞向太空丛书》编委会编|title=太空之子 世界宇航员轶事|date=2009年6月|publisher=世界图书广东出版公司|isbn=978-7-5100-0583-1|pages=189-190|accessdate=2016-10-17}}</ref>。1985年6月加入[[中国人民解放军空军|中国人民解放军空军]]，1987年9月加入[[中国共产党|中国共产党]]。当上飞行员之后，景海鹏先后在保定、运城、无锡、苏州<ref name=sz>{{cite web |url = http://news.sohu.com/20120616/n345800658.shtml|title = 苏州战友回忆往事 景海鹏曾是营中“篮球王子” |publisher =中国新闻网 |date = 2012年6月16日 }}</ref>工作，并驾驶过歼六等机型，曾任空军某师某团司令部领航主任，安全飞行1200小时，被评为空军一级飞行员。1998年1月，成为中国首批航天员，并改隶[[总装|总装]]改为陆军。2005年6月，入选[[神舟六号|神舟六号]]飞行乘组梯队，但未能搭乘飞船升空。2008年9月25日，搭乘[[神舟七号|神舟七号]]飞船升空<ref>{{cite book |author1=《中国人物年鉴》编委会 |editor=李维民 |title=中国人物年鉴 2012 |date=2013 |publisher=中国人物年鉴社 |ISSN = 10081771 |pages=111-112 |accessdate=2016-10-17}}</ref>。

2007年3月至2010年1月在[[清华大学航天航空学院|清华大学航天航空学院]]攻读工程硕士。<ref>{{Cite web|url=http://www.tsinghua.edu.cn/publish/thunews/9649/2017/20170730205752609737240/20170730205752609737240_.html|title=校友马伟明、景海鹏荣获中央军委颁授的“八一勋章”|accessdate=2017-08-04|work=www.tsinghua.edu.cn}}</ref>

2012年6月16日，与航天员[[刘旺|刘旺]]、[[刘洋_(航天员)|刘洋]]共同搭乘[[神舟九号|神舟九号]]飞船升空，执行[[天宫一号|天宫一号]]与神舟九号载人交会对接任务，任指令长<ref>{{cite web |url = http://epaper.bjnews.com.cn/html/2012-06/16/content_348221.htm?div=-1 |title = 景海鹏 曾乘神七漫步 如今再度问天 |publisher = 新京报 |date = 2012年6月16日 |language = zh-hans |author = 张太凌}}</ref>，成为中国第一个两次执行载人航天任务的航天员<ref>{{cite book |editor=唐国东 |title=翱翔太空 中国载人航天之路 |date=2012年9月|publisher=上海：上海交通大学出版社 |isbn=978-7-313-08984-7 |page = 171 |accessdate=2016-10-17}}</ref>。

2013年，晋升为少将军衔<ref>{{cite news|url=http://www.thepaper.cn/newsDetail_forward_1544363|title=13年间中国有10位航天员“飞天”，至少6位都成了少将|newspaper=澎湃新闻|accessdate=2016-10-16}}</ref>。

2016年6月，入选[[天宫二号|天宫二号]]与[[神舟十一号|神舟十一号]]载人飞行任务航天员乘组，担任指令长。2016年10月17日7时30分，神舟十一号飞船发射，飞行乘组由景海鹏和[[陈冬_(1978年)|陈冬]]组成，景海鹏也成为中国第一位三次执行航天任务的航天员<ref>{{cite web |url = http://www.thepaper.cn/newsDetail_forward_1544116 |title = 神舟十一号明日7时30分发射，飞行乘组由景海鹏和陈冬组成 |publisher=澎湃新闻|date = 2016年10月16日 }}</ref>。2016年11月18日，神舟十一号载人飞船返回舱在[[内蒙古自治区|内蒙古]]中部预定区域成功着陆，执行飞行任务的航天员景海鹏、陈冬身体状态良好<ref>{{Cite web|url=http://slide.news.sina.com.cn/c/slide_1_2841_105305.html#p=1|title=神舟十一号飞船返回地球|date=2016年11月18日|work=slide.news.sina.com.cn}}</ref>。

2017年7月，中央军委主席[[习近平|习近平]]签署命令，授予景海鹏等10位同志“[[八一勋章_(2017年)|八一勋章]]”<ref>{{Cite web |url = http://news.ifeng.com/a/20170728/51522244_0.shtml |title = 景海鹏、程开甲、王刚等10人被授予“八一勋章” |author =  |date = 2017-07-28 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web|url=http://news.xinhuanet.com/politics/2017-07/28/c_1121395265.htm|title=中央军委主席习近平签署命令 授予麦贤得等同志“八一勋章”|date=2017-07-28|publisher=新华社}}</ref>。2017年7月28日，中央军委颁授“八一勋章”和授予荣誉称号仪式在北京[[八一大楼|八一大楼]]举行，中共中央总书记、国家主席、中央军委主席习近平向景海鹏等10位“八一勋章”获得者颁授勋章和证书。这是“八一勋章”首次颁授。景海鹏同时获授“矢志报国、逐梦太空的英雄航天员”荣誉称号<ref>{{Cite web |url = http://news.ifeng.com/a/20170728/51524452_0.shtml |title = 习近平向“八一勋章”获得者颁授勋章和证书 |author =  |date =2017-07-28   |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
