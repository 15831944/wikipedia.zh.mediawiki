'''景盛苑'''（英語：'''King Shing Court'''）是[[香港|香港]]的一個[[居者有其屋|居者有其屋]]屋苑，位於[[新界|新界]][[北區_(香港)|北區]]的[[粉嶺|粉嶺]]，於1995年11月開始入伙，由[[香港房屋委員會|香港房屋委員會]]興建，其後於2001年景盛苑業主立案法團成立後，房委會正式將屋苑管理權交還予景盛苑業主立案法團管理，首次推出時售價為576,800 - 1,133,000港元。

== 屋苑資料 ==
=== 樓宇 ===
景盛苑為採用和諧一型的單位，可以靈活地運用空間。而大部分的業主通常都會間格為一房兩廳、兩房兩廳或三房兩廳。全屋苑合共有四幢樓宇，分別為：歡景閣、俊景閣、賢景閣及欣景閣。

== 屋苑設施 ==
景盛苑設有多項的設施，包括：幼稚園、日間托兒所、羽毛球場、籃球場、排球場及兒童遊樂場。另外，屋苑附近的商場林立，提供多元化的服務。除此之外，景盛苑內設有「自動化垃圾收集系統」，方便清潔、衛生和快捷的方式處理所收集之垃圾。

== 地理環境 ==
景盛苑的地理環境是處於私人樓宇及其他屋苑的中間位置。而且亦鄰近[[粉嶺火車站|粉嶺火車站]]，附近的屋苑包括[[碧湖花園|碧湖花園]]、[[牽晴間|牽晴間]]、[[花都廣場|花都廣場]]、[[欣盛苑|欣盛苑]]、[[昌盛苑|昌盛苑]]、[[華明邨|華明邨]]及[[華心邨|華心邨]]等等，所以附近亦都人來人往。
