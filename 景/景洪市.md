'''景洪市'''（[[傣仂语|傣仂语]]：[[File:Jinghong.svg|50px]]；{{lang-th|เชียงรุ่ง}}），是中国[[云南省|云南省]][[西双版纳傣族自治州|西双版纳傣族自治州]]的县级市，是州政府所在地。面积6959平方公里，人口37.12万人。原名允景洪（{{lang-khb|ᦞᦵᧂᦋᦵᧂᦣᦳᧂᧈ}} {{IPA|[veŋ˥˩ʦeŋ˥˩huŋ˧]}}），[[傣语|傣语]]意为黎明之城。

== 历史 ==
這裡早在[[西汉|西汉]]时期，屬[[哀牢|哀牢]]国。

[[東汉|東汉]]时期，哀牢王遣使内属[[中国|中国]]，于是東汉政府当地设[[永昌郡|永昌郡]]，以哀牢王为部族君长。

[[南北朝|南北朝]]，有土著反抗南朝政府，到[[隋朝|隋朝]]就由[[茫蠻|茫蠻]]部落控制此地。 

[[唐朝|唐朝]]，[[南诏|南诏]]統一[[云南|云南]]，于是控制此地。

[[五代|五代]]後，[[大理国|大理国]]在此招安傣族人，以族人為银生节度使，管轄此地。

[[南宋|南宋]]时，傣族人叭真（帕雅真）脫離大理獨立，建立景陇金殿国，以此地為首都，同样稱景陇（或勐泐）。

[[元朝|元朝]]时，元軍攻打[[景陇金殿国|景陇金殿国]]，占领景陇，这里改為车里。

[[国共內战|国共內战]]後，1950年2月，这里被解放军解放，仍称车里县。1953年1月23日，西双版纳傣族自治州成立，复置版纳建制。

1958年6月建立景洪县。1993年12月，撤县设市，名景洪市。

== 地理 ==
位于云南省最南部，西双版纳州中部，东邻[[勐腊|勐腊]]县和[[江城|江城]]县，西接[[勐海|勐海]]县，北连[[思茅|思茅]]市，南与[[缅甸|缅甸]]接壤，国境线长112.39公里。景洪市在横断山系纵谷区南端，地处澜沧江大断裂带两侧，具山原地形，北高南低，两侧高，中部低，山峦叠嶂，沟壑纵横。最高海拔2196.8米，最低海拔485米。市府所在地海拔552.7米。总面积7003.1平方公里。[[澜沧江|澜沧江]]（湄公河）穿过景洪全境。
=== 气候 ===
景洪市地跨东经100°25′－101°31′，北纬21°27′－22°36′之间，屬於[[热带季風氣候|热带季風氣候]]，因此长夏无冬，白天高溫最高的月份與泰國，緬甸等中南半島城市相同為4月份。3至5月份的夏季緊接而來為6至9月的雨季。冬季白天雖溫暖，夜晚的低溫有時可能低至5-10度不等，且因夜晚溫度偏低，日夜溫差大，年日均溫也大約只有22.5度，日均溫而言與香港，兩廣沿海各縣市相似。干湿季分明；静风少寒，全年基本无霜。

== 民族 ==
市内世居民族有[[傣族|傣族]]、[[汉族|汉族]]、[[哈尼族|哈尼族]]、[[基诺族|基诺族]]、[[拉祜族|拉祜族]]、[[彝族|彝族]]、[[布朗族|布朗族]]、[[瑶族|瑶族]]、[[佤族|佤族]]等13个民族。

2002年的少数民族人口为249721人，占总人口的67.27%。其中，傣族占全市人口35%，汉族121511人，占总人口的32.73%。

== 经济 ==
2004年全市生产总值38.77亿元，比上年增长12.7%。特别是[[旅游业|旅游业]]持续升温。全年接待海外游客3.14万人次、国内旅客231.66万人次，分别比上年增长38.9%、7.3%；旅游总收入21.65亿元，旅游创汇收入798.81万美元，分别增长21.5%、39.7%。

==旅游资源==
景洪市境內有173.55万亩自然保护区，分布有热带雨林，各种珍奇的热带特有动物生活在原始森林中，形成自然生态好风光。

澜沧江（湄公河）从市区境内穿过，在江畔可以参与进行漂流。

景洪市境内还建有西双版纳傣族园、热带花卉园、民族风情园、曼景兰旅游村、小街温泉、曼听公园、曼飞龙佛塔等著名景点。
