'''景美車站'''位於[[台灣|台灣]][[花蓮縣|花蓮縣]][[秀林鄉|秀林鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[北迴線|北迴線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[島式月台|島式月台]]一座。
*設有多卡通刷卡機。

== 利用狀況 ==
*目前為乙種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]，但無派員售票。
*僅有[[臺鐵區間車|區間車]]停靠。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1975年7月26日：設站。
*1975年9月：開始營運。
*1998年6月11日：[[新城（太魯閣）車站|新城（太魯閣）站]]至[[花蓮車站|花蓮站]]間雙軌化工程完工通車，已失去交會列車之需要，景美站因業務清淡，故於同日起由三等站直接降為招呼站，並指定由[[新城（太魯閣）車站|新城（太魯閣）站]]管理。
*2013年10月3日：因應新城～北埔間貨物班次增加，為路線容量需要，本站向兩側擴建待避線，升等為乙種簡易站，仍由新城站管理。
*2016年3月1日：啟用多卡通刷卡機。
