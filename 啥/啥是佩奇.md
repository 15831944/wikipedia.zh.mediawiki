《'''啥是佩奇'''》是[[动画电影|动画电影]]《[[小猪佩奇过大年|小猪佩奇过大年]]》于2019年1月17日发布的宣传片，由电影摄制组拍摄制作，时长5分40秒，剧情与电影无关，讲述了农村老人李玉宝为孙子全村寻找“佩奇”的故事。

《[[小猪佩奇过大年|小猪佩奇过大年]]》是由中英两国合拍的动画电影，由[[阿里巴巴影业集团|阿里巴巴影业]]、{{le|Entertainment One|Entertainment One}}出品。宣传片《啥是佩奇》的导演[[张大鹏_(导演)|张大鹏]]是电影的导演之一，拍摄广告片出身，据张大鹏称，短片《啥是佩奇》的参演人员都不是职业演员，在河北省张家口市怀来县取景拍摄，拍摄历时两天<ref name="对话">{{cite web |author1=新京报 |title=对话《啥是佩奇》导演张大鹏：这是个正经的中国故事 |url=https://baijiahao.baidu.com/s?id=1622970961769785806 |website=百家号 |accessdate=2019-01-18 |date=2019-01-18 |archive-url=https://web.archive.org/web/20190120012628/https://baijiahao.baidu.com/s?id=1622970961769785806 |archive-date=2019-01-20 |dead-url=no }}</ref>。

== 剧情 ==
临近年关，老人李玉宝打电话问眼看就要回村过节的小孙子想要什么礼物，结果却对孩子想要的[[小猪佩奇|佩奇]]是什么一头雾水。他借村里的喇叭问了一圈，得到的答案令人啼笑皆非，有人说是[[直播|直播]]网站性感女主播，有人拿出同名[[洗潔精|洗洁精]]，还有人说是[[棋牌游戏|棋牌]]的一种。最后一个曾经在城里当过[[保姆|保姆]]的村民告诉他是[[动画卡通|动画片]]里的一个红色的猪，最后懵懂的李玉宝按照她的描述用鼓风机自制了一个“佩奇”送给孙子。

== 网络爆红 ==
2019年1月16日，电影官方微博“电影小猪佩奇过大年”发布多个短片预告《啥是佩奇？明日揭晓》，1月17日17时到22时间，多个微博营销账号发布《啥是佩奇》正式短片，23时后陆续有王思聪等知名人物转发，使该话题成为微博热点<ref name="专家答疑">{{cite web |title=《啥是佩奇》 为啥刷屏，导演和专家答疑 |url=http://www.bjnews.com.cn/ent/2019/01/19/541067.html |website=新京报 |accessdate=2019-01-20 |date=2019-01-19}}</ref>。《啥是佩奇》短片一经发布，在[[微博|微博]]、[[微信|微信]]等中国大陆主流网络社区引起了广泛的关注，[[王思聪|王思聪]]、[[任素汐|任素汐]]等很多明星参与了转发和讨论<ref name="环球网娱乐">{{cite web |author1=环球网娱乐 |title=《啥是佩奇》一夜爆红，明星转发，导演懵圈，网友泪奔 |url=https://baijiahao.baidu.com/s?id=1623156993472021700&wfr=spider&for=pc |website=百家号 |accessdate=2019-01-20 |date=2019-01-20}}</ref>。而片中的同款鼓风机也在网上热销，但“小猪佩奇”的中国代理商则表示此行为涉嫌侵权<ref>{{cite web |title=网售“鼓风机佩奇”涉嫌侵权 |url=http://finance.people.com.cn/n1/2019/0120/c1004-30578526.html |website=北京青年报 |accessdate=2019-01-20 |date=2019-01-20}}</ref>。

由于话题爆红，一些官方机构的微博账号也参与相关的互动，如[[北京市公安局|北京市公安局]]官微“平安北京”提到“有人在携带鼓风机坐火车的时候，鼓风机内有[[汽油|汽油]]500毫升”，而[[应急管理部消防救援局|应急管理部消防救援局]]官微“中国消防”则反驳称片中李玉宝大爷的鼓风机是手摇式的，不是用汽油的，调侃平安北京乱蹭热点，平安北京则回覆称中国消防乱怼。著名演员[[刘佩琦|刘佩琦]]因为名字跟佩奇同音，也被网友拿来调侃<ref>{{cite web |title=佩奇被玩坏了！中国消防平安北京互怼，刘佩琦称自己才是本琦…… |url=http://m.gmw.cn/2019-01/18/content_1300141076.htm |website=光明网-重庆晨报 |accessdate=2019-01-20 |date=2019-01-18}}</ref>。

== 评价 ==
短片中植入了比较多的[[中国移动|中国移动]]广告，但导演张大鹏认为将该短片视为广告片不太准确，其主要还是一个电影的宣发视频，植入中国移动是因为它和电影是合作关系<ref name="对话" />。短片故事通讯并不是很发达，被质疑与农村现实并不符合，短片制片人鲁岩认为短片故事以强调情感为主，而且中国各地农村的情况并非相同，故事所呈现的在现实中仍然能够找到影子<ref>{{cite web |author1=肖扬 |title=《啥是佩奇》制片人：拍公益故事，不想道德绑架 |url=https://www.thepaper.cn/newsDetail_forward_2871257 |website=澎湃新闻-北青报 |accessdate=2019-01-18 |date=2019-01-18 |archive-url=https://web.archive.org/web/20190120012633/https://www.thepaper.cn/newsDetail_forward_2871257 |archive-date=2019-01-20 |dead-url=no }}</ref>。

对于短片网络爆红的原因，导演认为是佩奇本身就存在热度，自带流量，再就是作品的拍摄质量和风格也可能是爆红的原因<ref name="对话" />。1月18日下午，[[中国政法大学|中国政法大学]]流传法中心副主任朱巍接受新京报记者采访时表示是《啥是佩奇》的爆红是个[[营销|营销]]事件，不是原发性爆款，短片导演本意就是想戳中观众泪点，营销各人回家过年团聚的心理，把过年回家和小猪佩奇结合，对小猪佩奇[[智慧财产权|IP]]举办营销“是一种感情贸易化的操纵”。朱巍认为，“佩奇”在此次现象级传播中仅是一个文化符号，真正的核心点是作为“留守在乡村和在外工作的人之间的纽带”的[[手机|手机]]和[[互联网|互联网]]<ref name="专家答疑" />。
