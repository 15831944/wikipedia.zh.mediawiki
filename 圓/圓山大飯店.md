'''圓山大飯店'''是位於[[中華民國|中華民國]][[臺北市|臺北市]][[中山區_(臺北市)|中山區]][[劍潭山|劍潭山]]的一座地標性質的[[中國風|中國風]][[旅館|飯店]]，成立於[[第二次世界大戰|第二次世界大戰]]後，早年為臺灣首屈一指的大型國際性飯店。目前所見的宮殿風格建築於1973年落成，是臺北市的著名地標之一。除了位於台北市的主館外，圓山大飯店在[[高雄市|高雄市]]設有一處[[高雄澄清湖圓山大飯店|分館]]。台北圓山大飯店已晉升為[[交通部觀光局|交通部觀光局]]評鑑之[[五星級飯店|五星級飯店]]<ref name="旅宿網">{{cite web | title=圓山大飯店(交觀業字第1448號) | website=交通部觀光局臺灣旅宿網 | date=2018-08-10 | url=https://taiwanstay.net.tw/Home/Detail/a4debdc8a1ce4ae7b630d0effcf46e7c | language=zh-tw | access-date=2018-08-09}}</ref>。

== 沿革 ==
===成立初期===
圓山大飯店的位置原為[[台灣日治時期|台灣日治時期]]的[[臺灣神宮|臺灣神宮]]所在，[[中華民國國民政府|中華民國國民政府]]於[[二戰|二戰]]後接管台灣時，苦於臺灣沒有豪華的國家迎賓設施可以接待外國貴賓，遂拆除當時台灣神宮，並原地改為'''台灣大飯店'''；1952年，改由[[蔣宋美齡|蔣宋美齡]]等政要為首組成的「財團法人台灣敦睦聯誼會」（簡稱「敦睦聯誼會」；[[台灣省政府功能業務與組織調整|精省]]前稱為「台灣省敦睦聯誼會」）接手經營，並改為圓山大飯店現名，原先用途為招待訪華外國元首或政府首腦及[[蔣中正|蔣中正]]、[[蔣宋美齡|蔣宋美齡]]夫婦之訪客。當時圓山大飯店的建築規模甚小，在1963年時才將飯店的基礎設施全部建設完畢。1967年，圓山大飯店獲得[[美國|美國]][[财富_(杂志)|財星雜誌]]評為世界十大飯店之一。1973年10月10日，由建築師[[楊卓成|楊卓成]]設計新建的14層中國宮殿式大廈落成，圓山大飯店遂成為當時台北市的新地標。

1975年時，蔣宋美齡成立「臺北圓山聯誼會」接手原本敦睦聯誼會與私人會員服務相關的業務，是一個會員制的[[俱樂部|俱樂部]]<ref>{{cite web|url=http://www.grand-hotel.org/club/zh-TW/?Psn=5140 |language=zh-tw |title=關於聯誼會 |publisher=圓山大飯店 | date= |accessdate=2015-06-04}}</ref>。該聯誼會的活動設施位於圓山大飯店主建築後方，由一群與飯店建築風格相同的低矮建物組成。

===近年發展與現況===
經營圓山大飯店的臺灣敦睦聯誼會，雖然是以私人名義捐款成立的[[財團法人|財團法人]]，但實際上由[[中華民國交通部|中華民國交通部]]管轄<ref>[http://www.libertytimes.com.tw/2002/new/sep/22/today-e6.htm 圓山大飯店由交通部所屬的台灣省敦睦聯誼會管轄] {{webarchive|url=https://web.archive.org/web/20060527092104/http://www.libertytimes.com.tw/2002/new/sep/22/today-e6.htm |date=2006-05-27 }}</ref>，而當時的所有捐款人均為當時的中華民國政府要員。因為與政府的特殊關係，圓山大飯店長期成為台灣、乃至於中華民國對外的國際招待場所，其半官方機構的神秘色彩，使得一般民眾難以入住其內。但隨著1990年代後，大型民間飯店業者的崛起與挑戰，曾經一度使當時的圓山大飯店難以競爭。1995年，因屋頂會議廳施工不慎引發火災，迫使圓山大飯店進行全面性的整修工程，1998年完工之後才重新開幕。

為了增強競爭力，圓山大飯店近年來開始活化經營架構，除了提供多角化與平價化的服務之外，並先後引進民間的專業經理人（如[[嚴長壽|嚴長壽]]等）負責經營整頓，而得以維持其在台灣旅館業的[[旗艦|旗艦]]地位。

== 建築特色 ==
圓山大飯店的原址在日治時期曾是日本人建的[[臺灣神宮|臺灣神宮]]，由於地處的[[劍潭山|劍潭山]]山頭，突出於[[台北盆地|台北盆地]]之上，在[[風水學|風水學]]上地理位置極佳。饭店屋顶使用[[歇山式|歇山式]]。飯店的建築上採用相當多的[[龍|龍]]形雕刻，故有人稱此飯店為「龍宮」；除採用龍形之外，亦有[[石狮子|石獅]]、[[梅花|梅花]]等中國建築常用的圖案。

===逃生密道===
2006年台北圓山飯店密道被媒體曝光後，飯店人員當時也詳加介紹，因[[蔣中正|蔣中正]]、[[蔣經國|蔣經國]]時代有很多國內外政務官要，都來此地享受國宴，因保護元首而在建造過程中增加此設計。

圓山飯店總經理池漢乾指出，這兩條密道位于地下二樓，除了考量蔣中正的安全外，主要是當時圓山飯店出入的都是達官顯要，萬一發生空襲，貴賓可就近至地下二樓防空避難室躲避；若一旦發生威力更大的核彈空襲，即可迅速進入防空避難通道。 

這兩條密道往東可達[[北安公園|北安公園]]，全長78米，在沿線裝有電燈照明設備，共有45個燈座，86個階梯；往西可抵圓山聯誼會劍潭公園，全長80米，75個階梯，內部蜿蜒曲折，一直到出口處可以見到兩扇長期深鎖的金屬門，打開後還會見到兩座高約5米的石牆遮蔽，外界根本看不出這是坑道出口。 

兩條密道皆為2.3米寬，2.1米高，造型有如螺旋式走道，一路向下盤旋，若不小心可能就會摔個大跤，寬度可容納二至三人同時並行。最特殊的是，西側的密道旁，還設計成約一人寬，以磨石子為底，類似溜滑梯的滑行道。飯店說法是，這種設計便于讓行動不便的人員使用。由于滑道兩端距離約20米，又是一路往下盤旋，看過的飯店人員戲稱，這大概是世界最長的“水泥溜滑梯”。

== 備考 ==
1995年圓山大飯店因[[火災|火災]]受損後，曾有[[立法委員|立法委員]]提議拆除圓山大飯店，將其用地改築「國會山莊」以作為[[立法院|立法院]]新院址，此提案隨著圓山大飯店的修復工程完成無疾而終。

2014年[[太陽花學運|太陽花學運]]期間，又重新出現立院搬遷至圓山大飯店的提案，但未受重視<ref>{{cite news|title=立院沒人尊重 藍委建議搬到圓山飯店|url=http://news.ltn.com.tw/news/politics/breakingnews/987575|accessdate=2014-06-24|date=2014-04-13}}</ref>。

另[[桃園國際機場|桃園國際機場]]內，原有餐飲服務亦由圓山大飯店提供；但由於長期虧損，於[[嚴長壽|嚴長壽]]掌理時期終止經營。

臺北圓山大飯店入口左側，有[[于右任|于右任]]題字的[[草書|草書]]碑文「劍潭勝跡」。

===開車路線===
A、高速公路（國道一號）：
下松江路在民族東路右轉，到中山北路再右轉，過台北市立美術館再上中山橋，靠左車道前行一分鐘後會在前方山壁旁看到一個斜坡，從斜坡直接上圓山飯店。

B、新生高架快速道：
下橋往北安路走左側車道至美僑協會迴轉往市區方向，並沿著山壁的最左側車道繞一圈後，直接上圓山飯店。

C、中山北路往南（從士林往台北）：
沿著中山北路往南經劍潭青年活動中心直行，走左側車道並沿著山壁的最左側車道繞一圈後直接上圓山飯店。
==爭議==
