'''圓環緝菸事件'''，俗稱'''緝菸血案'''，是1947年2月27日[[臺灣菸酒公司|臺灣省專賣局]]臺北分局緝私組人員與憲警單位因查緝私菸而造成一死一傷的事件，該事件為[[二二八事件|二二八事件]]爆發的導火線。

== 經過 ==
=== 獲報查緝 ===

ISBN 978-9866480485 {{zh-tw}}.</ref><ref name="楊逸洲 71-75">楊逸舟; 張良澤（譯）. 《二·二八民變：台灣與蔣介石》. 臺灣臺北: [[前衛出版社|前衛出版社]]. 1997年10月: 第71頁至第75頁. ISBN 978-9579512213 {{zh-tw}}.</ref>。但當他們抵達淡水時，僅查獲5箱私煙，未有成果<ref name="責任歸屬研究報告 43-51">張炎憲、李筱峰、何義麟、黃秀政、陳儀深、陳翠蓮、陳志龍和黃茂榮. 《[[二二八事件責任歸屬研究報告|二二八事件責任歸屬研究報告]]》. 臺灣臺北: [[二二八事件紀念基金會|二二八事件紀念基金會]]. 2006年1月1日: 第43頁至第51頁. === 包圍警局 ===
晚上9時左右，專賣局業務委員會常務委員[[李炯支|李炯支]]與業務會第四組組長[[楊子才|楊子才]]得知消息後，趕往現場處理。見到2人卡車抵達後，民眾試圖向前毆打，迫使李炯支、楊子才轉至台北市警察總局，不過群眾也跟隨前往警察總局。之後600名至700名群眾包圍警察總局，要求交出並槍斃肇事兇手<ref name = "責任歸屬研究報告 43-51"/>。警察局警官出面說明<ref name = "責任歸屬研究報告 43-51"/>，李炯支和楊子才也表示將會懲辦查緝員，然而群眾仍持續要求交出肇事人。兩人不得已會同台北市警察局局長[[陳松堅|陳松堅]]，將6名查緝員送往憲兵隊看管。由於過去專賣局的查緝已造成數起死傷，而這次事件又有過去緝菸傷人的[[累犯|累犯]]（即盛鐵夫），不滿的民眾要求立即將6名查緝員逮捕，李炯支、楊子才兩人則以「刑罪罰惡，律有明文，未予擅便答覆」為由解釋，但民眾不服並認為警方試圖保護傅學通等人<ref name = "王建生 97-103"/><ref name = "二二八事件研究 267-274"/>。不滿的群眾於是搜索警察局，但找不到傅學通等人。

=== 憲民對峙 ===
[[File:228_Incident_h.jpg|right]]的抗議群眾，時間為[[1947年|1947年]][[2月28日|2月28日]]。]]
群眾得知查緝員已送至《[[台灣新生報|台灣新生報]]》報社對面的憲兵隊第4團團部後，於晚上9時包圍憲兵團並要求搜查犯人。團長[[張慕陶|張慕陶]]拒絕民眾要求，下令一排憲兵上前擺出射擊姿勢，群眾因而躲進《台灣新生報》報社騎樓。之後《[[中外日報|中外日報]]》記者[[周傳枝|周傳枝]]向《[[台灣新生報|台灣新生報]]》代理主編[[吳金鍊|吳金鍊]]借銅鑼以助長聲勢<ref name = "二二八事件研究 267-274"/><ref name = "責任歸屬研究報告 43-51"/>，雨停之後群眾又再度包圍憲兵隊。亦有青年組成示威隊伍開始於街上通告晚上所發生的事情<ref name = "責任歸屬研究報告 43-51"/><ref name = "解讀二二八 114-117"/><ref name = "再現台灣 20-23"/><ref name="二二八事件研究 274-289">褚靜濤. 《二二八事件研究》上卷. 臺灣臺北: 海峽學術出版社. 2011年8月10日: 第274頁至第289頁. ISBN 978-9866480485 {{zh-tw}}.</ref>。雖然張慕陶多次威脅和規勸<ref name = "責任歸屬研究報告 43-51"/>，由於群眾在憲兵隊及警察局兩邊的交涉一直得不到結果，群眾於是在細雨中徹夜包圍憲兵隊及警察局，直至天亮仍不肯離開。

=== 通告大眾 ===
部分民眾還要求《台灣新生報》報社刊登此事<ref name = "楊逸洲 71-75"/>，但由於報社得到台灣省行政長官公署宣傳委員會不得刊登事件消息的命令，對此吳金鍊加以拒絕且說明報社立場<ref name = "王建生 97-103"/><ref name = "責任歸屬研究報告 43-51"/>。在民眾威脅以汽油燒毀報社的情況下，吳金鍊不得已轉請報社社長[[李萬居|李萬居]]出面，後者同意刊登後民眾才離開報社<ref name = "責任歸屬研究報告 43-51"/>。隔天《台灣新生報》以5號字發表數百字的消息報導<ref name = "王建生 97-103"/><ref name = "楊逸洲 71-75"/>。2月28日，由於包圍警察局和憲兵隊，要求處理肇事者未果，憤怒的群眾在臺北市各處集結遊行、街頭演說、呼籲罷市<ref name = "陳柏言"/><ref name = "責任歸屬研究報告 43-51"/><ref name = "二二八事件研究 274-289"/><ref name="王建生 104-119">王建生、陳婉真和陳湧泉. 《1947 台灣二二八革命》. 臺灣臺北: [[前衛出版社|前衛出版社]]. 2002年4月: 第104頁至第119頁. === 圍攻專賣局 ===
[[File:228_incident_of_Taiwan.jpg|缩略图]]並焚燒物件（2月28日上午10時）。]]
28日，臺北市許多商店陸續關店，部分地區不滿的民眾、學生、退伍士兵紛紛罷工、罷課和罷市<ref name="責任歸屬研究報告 43-51" /><ref name="再現台灣 20-23" /><ref name="王建生 104-119" />，並在街頭[[示威|示威]][[遊行|遊行]]，要求懲罰兇手<ref name="二二八事件研究 274-289" /><ref name="楊逸洲 75-78" />。大小商店即刻響應相繼關門，市民紛紛湧上街頭觀看，市區擠滿人群，尤其[[大稻埕|大稻埕]]人山人海，民心憤慨，群情激昂。上午十時許，憤怒的市民衝入肇事查緝員所任職的專賣局臺北分局，痛毆專賣局職員，當場打死兩人、毆傷四人，搗毀辦公器具及玻璃，並將菸、酒、火柴、汽車等物搬出戶外焚毀<ref>行政院二二八事件研究報告，51-52；責任歸屬研究報告，48-49；解讀二二八，116-117；1947台灣二二八革命，104-105；憤怒的台灣，120-121；</ref>。十二時許，民眾前往南門專賣總局，因門戶緊閉僅打破玻璃。民眾又衝入前後任專賣局長的[[官邸|公館]]搗毀器物，專賣局南門工廠亦被搗毀<ref>行政院二二八事件研究報告，52；責任歸屬研究報告，49；1947台灣二二八革命，105-108</ref> {{noteTag|專賣局嚴重影響民眾生計、不肖專賣局人員勾結商人走私、專賣局縱容官商卻嚴罰小民、部分專賣局查緝員緝私後私吞、專賣局帶槍執勤最惹民怨，而且衝突不斷<ref>行政院二二八事件研究報告，51</ref>。}}。

== 後續 ==
攻擊專賣局後，群眾集結於[[臺灣省行政長官公署|台灣省行政長官公署]]（即現在行政院所在館舍）門口抗議，公署衛兵[[臺灣省行政長官公署衛兵開槍事件|無差別開槍射殺請願民眾]]，造成多人死傷，使得民眾的情緒更加憤怒。之後抗議民眾轉進公署附近「臺北新公園」（今二二八和平紀念公園）繼續示威集結，並佔據位於公園內之「臺灣廣播電台」。長期對政府的不滿情緒，加上公署此事件處理不當，3月1日起，事件演化成全島性的反抗政府事件，即影響台灣社會至深的[[二二八事件|二二八事件]]。

== 審判 ==
1947年3月，傅學通、葉得根、盛鐵夫、鐘延洲、趙子健、劉超群等六人均被逮捕，期間雖以「遭獄內押犯毆打」為由保外就醫<ref>[https://aa.archives.gov.tw/archivesData.aspx?SystemID=0000380551]</ref>，但仍於3月31日以殺人及傷害案件，經[[台北地方法院|台北地方法院]]檢察官偵查終結，向傅學通、葉得根等人提起公訴。4月3日下午，地方法院刑庭庭長宣判，誤殺台北市民陳文溪的傅學通判處[[死刑|死刑]]<ref>褚靜濤. 《二二八事件研究》上卷. 臺灣臺北: 海峽學術出版社. 2011年8月10日: 第472頁至第480頁. ISBN 978-9866480485 {{zh-tw}}.</ref>，[[褫奪公權|褫奪公權]]終身。以[[槍托|槍托]]打傷林江邁的[[葉得根|葉得根]]判處4年6個月[[有期徒刑|有期徒刑]]，褫奪公權3年。其餘盛鐵夫、鐘延洲、趙子健及劉超群無罪開釋，同年5月17日傅學通上訴[[最高法院|最高法院]]，獲得改判十年定讞，而葉得根則維持原判。
