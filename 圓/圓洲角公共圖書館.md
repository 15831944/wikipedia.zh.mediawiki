'''圓洲角公共圖書館'''（{{lang-en|Yuen Chow Kok Public Library}}）是[[香港|香港]][[沙田區|沙田區]]的一座[[圖書館|圖書館]]，屬[[康樂及文化事務署|康樂及文化事務署]][[香港公共圖書館|香港公共圖書館]]轄下的[[分區圖書館|分區圖書館]]，該圖書館[[總樓面面積|樓面面積]]2,900多平方米，位於[[香港|香港]][[新界|新界]][[沙田|沙田]][[圓洲角|圓洲角]][[銀城街|銀城街]]35號（毗鄰[[愉翠苑|愉翠苑]]），於2017年3月30日啟用，主要為[[圓洲角|圓洲角]]、[[廣源|廣源]]及[[沙田圍|沙田圍]]的居民服務，為沙田區第2個分區公共圖書館。<ref>[http://www.info.gov.hk/gia/general/201703/29/P2017032900335.htm 圓洲角公共圖書館明日啟用（附圖） 2017-03-29]</ref>

== 設計 ==
圖書館風格以簡約設計為主，以白色和灰色作主調，並採用落地玻璃。兒童圖書館的設計以大自然為主題，並設大量座位及活動空間，中央設玻璃天幕。而書架的指示牌設計為動物圖案，感覺具趣味性。

== 設施 ==
圖書館樓高4層，佔地超過2,900多平方米，2樓為學生自修室，3樓設成人圖書館、多媒體圖書館、電腦資訊中心及參考圖書館，4樓為主入口，設報刊閱覽室及咖啡閣，有通道往[[圓洲角體育館|圓洲角體育館]]。5樓為兒童圖書館和推廣活動室。學生自修室已於2016年12月啟用。

== 保安 ==
雖然圖書館設計創新，唯該圖書館規則與香港所有圖書館一樣，禁止公眾進行拍攝及錄影，{{or|而此館與新建的公共圖書館一樣，對禁止攝影的管理較一般圖書館更嚴。}}館內告示貼上多張「勿在館內拍攝」告示及{{or|派遣多名保安作為監視，以提防有人拍攝及錄影。圖書館亦不容許普通市民進行攝影申請，可見管理者的作風十分保守。}}
