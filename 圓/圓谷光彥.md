'''圆谷光彦'''（日語：{{lang|ja|円谷 光彦}}/{{lang|ja|つぶらや みつひこ}}，Tsuburaya Mitsuhiko），[[少年侦探团|少年侦探团]]的一员，是日本[[漫画|漫画]]和[[动画|动画]]作品《[[名侦探柯南|名侦探柯南]]》中的虚构人物。

命名來自[[日本|日本]]小說家[[内田康夫|内田康夫]]的推理小說「'''[[淺見光彥|淺見光彥]]'''系列」的主角。

== 人物 ==
7岁，体重15公斤，就讀於[[东京都|东京都]]米花市帝丹小学一年B班；双亲都是教师，家中有一姊姊[[名偵探柯南角色列表|圆谷朝美]]<ref group="參">漫畫第35卷，TV 289《光彥森林迷失記（前篇）》</ref>。

在一次的事件中，大膽推理犯人的犯案手法與《[[紅髮會|紅髮會]]》相似，因而遭到柯南的調侃；但結果如光彥所料，令柯南只能無奈接受<ref group="參">TV 225《生意興隆的秘密》，動畫原創</ref>。

== 形象 ==
喜欢藉由閱讀來增加科普知識，和柯南同樣堅信科學，不相信鬼怪。時常糾正[[小島元太|小島元太]]錯誤的發音或不正當的舉止，在同年龄的孩童中显得博学多聞、出類拔萃；雖远比不上[[江户川柯南|江户川柯南]]和[[灰原哀|灰原哀]]，但仍然相當欽佩他們兩人的學識。在柯南的熏陶下，推理水準有很大的提升，甚至可以自己解决简单的案子，忌妒心極重，有點自以為是，成天想和柯南比試。

此外，喜愛'''假面超人'''。

== 情感 ==
曾喜歡過[[吉田步美|吉田步美]]，但后来喜欢灰原<ref group="參">M5《[[往天國的倒數計時|往天國的倒數計時]]》</ref>。
