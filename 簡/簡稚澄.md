'''簡稚澄'''，前桃園市新屋動物保護教育園區（動物收容所）園長<ref name="oncc">{{Cite web |url=https://hk.on.cc/tw/bkn/cnt/news/20170203/bkntw-20170203023621648-0203_04011_001.html |title=外媒分析台女獸醫自殺案　揭台動物保育困境 |publisher=[[東方日報_(香港)|東方日報]]}}</ref>。畢業於北一女中、台大獸醫系。雖以第一名的成績考獲獸醫資格，但最後選擇放棄獸醫高薪的工作，前往位於桃園市新屋區為流浪動物服務<ref name="oncc"></ref><ref>{{Cite web |url=https://news.ltn.com.tw/news/life/paper/1075352 |title=美女獸醫死諫太震撼 同事拒絕再提起 |publisher=[[自由時報|自由時報]]}}</ref>。

擔任園長期間致力於流浪動物的醫療與照護，然個人能力有限，加上彼時台灣動保法令之不完備，在收容所期間不得已共執行了700多只流浪狗的安樂死。簡稚澄於台灣媒體中天新聞接受採訪時，描述了動物的撲殺過程，節目播出後因此遭受網路上不少動物保護人士以「女屠夫」、「女劊子手」等言詞攻擊。<ref name="oncc"></ref><ref>{{Cite web |url=https://skypost.ulifestyle.com.hk/article/2230479/%E7%BE%8E%E5%A5%B3%E7%8D%B8%E9%86%AB%E4%B8%8D%E6%95%B5%E3%80%8C%E5%B1%A0%E5%A4%AB%E3%80%8D%E6%83%A1%E5%90%8D%20%E5%85%A7%E5%BF%83%E6%8E%99%E6%89%8E%E8%B5%B0%E4%B8%8A%E7%B5%95%E8%B7%AF |title=美女獸醫不敵「屠夫」惡名 內心掙扎走上絕路 |publisher=[[晴報|晴報]]}}</ref>

2016年5月5日，簡稚澄服用用來替流浪狗安樂死之藥物輕生<ref name="bbc"></ref>，送院經多日搶救，12日宣告不治，20日在中壢殯儀館舉辦告別式，年僅32歲<ref>{{Cite web |url=https://www.ettoday.net/news/20160520/702054.htm |title=32歲桃美女動保園長簡稚澄　吞「狗安樂死藥」自殺身亡 |publisher=[[ETtoday新聞雲|ETtoday新聞雲]]}}</ref>。其留下之遺書提到：「希望這件事能被世人看到，選擇用狗安樂死的方法，是要凸顯現有台灣動保結構的問題，末端的資源、人力不足，源頭管制工作無法做好，流浪狗到了最下游的收容所都是苦難。」<ref name="bbc">{{Cite web |url=https://www.bbc.com/zhongwen/trad/chinese-news-38871970 |title=台灣動保法零安樂死新法上路：回顧「安樂死自己」的女獸醫 |publisher=[[BBC|BBC]]}}</ref>
