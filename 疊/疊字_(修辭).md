{{Distinguish|疊字}}

'''疊字'''，又稱'''重言'''，為[[複詞|複詞]]的一種，是利用[[類疊|類疊]]的修辭技巧。疊字由兩個或以上，字形和字義都相同的漢字，重疊在一起使用所組成的詞語。大多數的[[象聲詞|象聲詞]]都是由疊字組成。在[[書面文|書面文]]或口語都可能出現。

中國早在[[詩經|詩經]]時代就有運用疊字的例子，如「伐木丁丁，鳥鳴嚶嚶」（小雅．伐木）。

== 分類 ==
* 名詞疊字
*: 例：雪花片片
* 動詞疊字
*: 例：寫寫書法、畫畫圖
* 形容詞疊字
*: 例：濕漉漉、紅彤彤、白雪雪
*:* 一些方言如[[閩南語|閩南語]]有三字疊字，且三字[[音調|音調]]皆不同，有最高的強調性，如「紅紅紅」（[[白話字|白話字]]：áng-āng-ăng）。
* 副詞疊字
*: 例：鬱鬱寡歡

== 著名疊字作品 ==
* [[古詩十九首|古詩十九首]]：青青河畔草，鬱鬱園中柳。盈盈樓上女，皎皎當窗牖。娥娥紅粉妝，纖纖擢素手。
* 聲聲慢（[[李清照|李清照]]）：尋尋覓覓， 冷冷清清， 淒淒慘慘戚戚。

== 其他 ==
* [[南島語系|南島語系]]的構詞中疊詞為其一部份，比如[[印尼語|印尼語]]常以名詞之重言表示[[複數|複數]]。有些語言，例如在[[平格拉普環礁|平格拉普島]]使用的[[平格拉普語|平格拉普語]](Pingelapese language)，還會有三重疊詞，以表示很遠時期發生的事件。
