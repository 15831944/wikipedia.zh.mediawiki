'''笠鱗鮫'''（[[學名|學名]]''Echinorhinus cookei''），又名'''吻棘鲨'''，'''笠鱗棘鯊'''或'''庫氏棘鯊'''，是一種生活在[[太平洋|太平洋]]海底的大型[[鯊魚|鯊魚]]<ref name=ITIS>{{ITIS|ID=160716|taxon=Echinorhinus cookei|year=2006|date=30 January}}</ref>。

== 特徵 ==
笠鱗鮫沒有[[臀鰭|臀鰭]]，兩條細小的[[背鰭|背鰭]]在背部較後位置。牠們最長的達4米，外表非常像[[棘鯊|棘鯊]]，但身體沒有那些[[小齒|小齒]]。

== 分佈 ==
笠鱗鮫出沒於[[太平洋|太平洋]]，包括[[台灣|台灣]]、[[新西蘭|新西蘭]]、[[夏威夷|夏威夷]]、[[美國|美國]][[加利福尼亞州|加利福尼亞州]]及[[智利|智利]]等海域。

== 習性及棲息地 ==
笠鱗鮫沿水深11-425米的[[大陸棚|大陸棚]]生活，經常出沒於海底峽谷。牠們以其他[[鯊魚|鯊魚]]及[[硬骨魚|硬骨魚]]為主要食物，亦會吃[[八爪魚|八爪魚]]、[[魷魚|魷魚]]及[[貓鯊科|貓鯊科]]的卵。

牠們都是[[卵胎生|卵胎生]]的，每胎約有114頭幼鯊。

==經濟利用==
可做為食用魚。
