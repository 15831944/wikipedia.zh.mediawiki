'''崔圭夏'''（{{kor|최규하|崔圭夏|Choi Kyu-Hah}}，{{bd|1919年|7月16日|2006年|10月22日|Choi, Kyu-hah}}），字'''瑞玉'''（{{lang|ko|서옥}}），號'''玄石'''（{{lang|ko|현석}}），[[韓國|韓國]][[外交官|外交官]]及政治人物，曾為該國第12任[[韓國總理|總理]]、第10任[[韓國總統|總統]]。任職總統後隨即在[[全斗煥|全斗煥]]發動的[[肅軍政變|肅軍政變]]及[[5·17緊急戒嚴|5·17緊急戒嚴]]後被迫下台，故其總統任期僅約滿8個月，從未實際掌權。

== 生平 ==
崔圭夏曾在[[日本|日本]]留學，並有日本姓氏'''梅原'''。他早年於{{tsl|ja|東京高等師範学校||東京高等師範学校}}畢業後擔任教師，之後前往[[滿洲國|滿洲國]]就讀[[大同學院_(滿州國)|大同學院]]。

1948年[[大韓民國|大韓民國]]獨立後，他曾出任農林部糧政課長、外交部通商局長、外務次官及外務部長，並在1971年就任外交担当特別補佐官、1975年接替[[金鍾泌|金鍾泌]]出任國務總理。

== 代總統 ==
1979年10月26日，[[朴正熙|朴正熙]]被暗殺，他以總理身份代行總統權限，並於12月6日經統一主體國民勝出後正式接任總統一職。

崔圭夏是韓國當年[[維新體制|維新體制]]的代表人物。他在正式當選後的早期實行憲制民主化改革，但有關計劃在[[全斗煥|全斗煥]]發動的[[雙十二政變|雙十二政變]]之後停止推行。1980年，[[全斗煥|全斗煥]]發動軍事政變，並根據《[[戒嚴|戒嚴]]法》接掌總統權力，指派[[朴忠勳|朴忠勳]]充當[[代總統|代總統]]，並籌備選舉團準備新一屆總統選舉；而崔圭夏也因此黯然下台，結束為期不足一年的總統任職。

== 去世 ==
2006年10月22日（韓國時間）上午6時，崔圭夏本人被發現昏迷於其在[[首爾|首爾]]市[[麻浦區|麻浦區]][[西橋洞|西橋洞]]的家中，後被送至{{link-ko|首尔大学校病院|서울대학교병원}}，延至7时37分逝世，享壽87歲。崔圭夏的葬禮以{{link-ko|國民葬|국민장}}規格舉行，遺體葬於[[大田廣域市|大田]][[國立顯忠院|國立顯忠院]]。
