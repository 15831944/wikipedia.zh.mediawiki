'''崔協'''{{BD|864年||929年|4月9日|catIdx=C}}<ref name=ZZTJ276>《[[資治通鑑|資治通鑑]]·[[s:資治通鑑/卷276|卷276]]》</ref><ref name=AS>[http://sinocal.sinica.edu.tw [[中央研究院|中央研究院]] 两千年中西历转换]</ref>，字'''司化'''，清河郡东武城县人（今山东省德州市武城县），[[五代十国|五代十国]][[后梁|后梁]]、[[后唐|后唐]]大臣，在[[唐明宗|唐明宗]]李嗣源时代担任宰相（[[中书侍郎|中书侍郎]]、[[同中书门下平章事|同中书门下平章事]]）。

== 背景 ==
崔協出自[[清河崔氏|清河崔氏]]的清河小房，祖父崔瓘担任吏部尚書，他的父亲崔彥融担任[[楚州|楚州]]（今[[江苏省|江苏省]][[淮安市|淮安市]]）[[刺史|刺史]]，崔彥融担任万年县（今陕西省[[西安市|西安市]]）令的时候和崔蕘关系很好，崔蕘到县上拜望他，崔彥融不在，崔蕘发现他贿赂权贵的信件，于是开始鄙视他。后来崔彥融升任[[司勋郎中|司勋郎中]]，时任[[尚书左丞|尚书左丞]]的崔蕘不愿意见他，宰相知道了其中的过程，于是改任崔彥融为楚州刺史。崔彥融告诫崔協，崔蕘是他的仇人，于是称他为崔仇。崔協从小孝行，他考中[[进士|进士]]，担任度支巡官、渭南（今[[陕西省|陕西省]][[渭南市|渭南市]]）县尉、直史馆，历三署。<ref name=HFD58>《[[舊五代史|舊五代史]]·[[s:舊五代史/卷58|卷58]]》</ref>

== 后梁时代 ==
唐朝灭亡后，在中原兴起的是[[后梁|后梁]]。崔协担任[[尚書省|尚書省]]左司郎中、萬年县令、[[給事中|給事中]]。官至[[兵部侍郎|兵部侍郎]]。923年，梁末帝[[朱友贞|朱友贞]]派遣兵部侍郎崔协等册命吴越王[[钱镠|钱镠]]为[[吴越国|吴越国]]王。他一次遇到崔蕘的儿子崔居儉，时任[[中書舍人|中書舍人]]。他厉声对崔居儉说：“崔蕘的儿子，你怎么敢见我？”崔居儉上奏梁末帝。梁末帝于是贬崔协为[[太子詹事|太子詹事]]（完全名誉职位，后梁没有太子）。不久之后，他再担任[[吏部侍郎|吏部侍郎]]<ref name=HFD58/>。

== 后唐时代 ==
后梁被[[后唐|后唐]]取代后，崔协被[[唐庄宗|唐庄宗]]李存勖任命为御史中丞。但是宪司举奏他的奏章，文字用词常常错误，于是多次受到责罚。崔协外表气宇轩昂，说话高谈阔论、不近情理，当时人认为他徒有其表<ref name=HFD58/>。

唐明宗[[李嗣源|李嗣源]]即位初年，崔协被任命为禮部尚書、太常卿<ref name=HFD58/>。927年，李嗣源罢免了[[豆卢革|豆卢革]]、[[韦说|韦说]]两个[[宰相|宰相]]，枢密使[[孔循|孔循]]不愿意河北、河东人担任宰相，于是推荐崔协为宰相。宰相[[任圜|任圜]]推荐[[李琪_(五代)|李琪]]，但是另一个宰相[[郑珏|郑珏]]讨厌李琪，孔循对[[安重诲|安重诲]]说：“李琪不是没有才能，但是他不清廉，朝议中没有超过崔协的。”任圜上奏明宗：<ref name=ZZTJ275>《[[資治通鑑|資治通鑑]]·[[s:資治通鑑/卷275|卷275]]》</ref>

{{quote|安重诲被人欺卖，如崔协者，少识文字，时人谓之‘没字碑’。臣比不知书，无才而进，已为天下笑，何容中书之内，更有笑端！}}

李嗣源想让[[冯道|冯道]]担任宰相，孔循没给李嗣源行礼，一甩衣服就走了，说：“天下的事情一也是任圜，二也是任圜，除非崔协突然死去，如果死不了必须让他当宰相。”因此称病不朝，李嗣源派安重诲劝他，孔循才上朝。安重诲私下对任圜说：“崔协暂作备员，可以吗？”任圜说：“您不用李琪而使崔协为宰相，这就好像抛弃了苏合香丸，选取屎壳螂推的粪球。”孔循和安重诲在一起处理政事，每天都说李琪的坏话，说崔协的好话。正月十一日，终于任命端明殿学士冯道和崔协一起担任中书侍郎、同中書門下平章事，李琪没有被任命<ref name=ZZTJ275/>。

崔协不能处理许多重要问题，重要文件由他人代写。李嗣源让他判祭酒事。他提出每年录取二百名监生，被议论认为并不适当<ref name=HFD58/>。明宗问宰相冯道：“卢质近日吃酒否？”冯道回答：“卢质曾到臣居，亦饮数爵，臣劝不令过度，事亦如酒，过则患生。”崔协说：“臣闻‘食医心镜’，酒极好，不加药饵，足以安心神。”

天成四年，李嗣源从[[开封府|开封府]]回[[洛阳|洛阳]]，门下侍郎、同平章事崔协随行，经过长途跋涉，由于中风，二月廿七日（929年4月9日）去世，虚岁六十六<ref name=ZZTJ276/>，[[谥号|谥号]]'''恭靖'''<ref name=HFD58/>。
