'''崔富日'''，（[[朝鮮語|朝鮮語]]：'''최부일'''，{{bd|1945年|7月17日|catIdx=Choi}}），[[朝鮮民主主義人民共和国|朝鮮民主主義人民共和国]]軍人、政治人物。朝鲜民主主义人民共和国国防委员会委员、[[朝鮮民主主義人民共和國國務委員會|國務委員會]]委員、人民保安部部长及[[最高人民會議|最高人民會議]]法制委員會委員長，大将衔。朝鲜劳动党中央政治局候补委员兼中央军事委员会委员。

== 履历 ==
1992年4月，被授予人民军少将军衔，1995年10月晋升为中将，担任[[朝鲜人民军|朝鲜人民军]]副总参谋长一职<ref>{{cite web|url=http://unibook.unikorea.go.kr/?sub_num=54&state=view&idx=435&sty=T&ste=%C1%A4%B8%ED%B5%B5|title=统一部资料中心北韩主要人物信息|publisher=统一部资料中心|date=2011年12月31日被确认}}</ref>。1998年9月当选第10届最高人民会议代议员，2006年4月和2010年9月，接连晋升为人民军上将、大将。

2010年9月，当选朝鲜劳动党中央委员会委员兼中央军事委员会委员。

2005年10月[[延亨默|延亨默]]、2010年11月[[赵明禄|赵明禄]]、2011年12月[[金正日|金正日]]去世时，担任国家治丧委员会委员。

== 金正恩时代 ==
崔富日原隶属朝鲜人民军一军团的篮球运动员，经军队干部推荐进入[[金日成军事综合大学|金日成军事综合大学]]，毕业后成为军官。20世纪90年代中期，他凭借体育特长当上了人民军体育指导委员长，开始掌管4.25体育团。曾自荐成为[[金正哲|金正哲]]和[[金正恩|金正恩]]的篮球教练，并在平壤中心区域的国家队体育馆成立了“王室篮球队”。朝鲜国家篮球队全员被借调成为金正哲和金正恩的陪练，曾传出进军美国NBA身高2米36的[[李明勳|李明勳]]也在其中<ref>{{cite web|url=http://unibook.unikorea.go.kr/?sub_num=54&state=view&idx=435&sty=T&ste=%C1%A4%B8%ED%B5%B5|title=北韓閱兵指揮崔富日“火箭陞官”引關注|publisher=朝鲜日报中文网|date=2010-10-14}}</ref>。

2014年3月，崔富日在[[2014年朝鮮民主主義人民共和國最高人民會議選舉|當屆的最高人民會議選舉]]中當選為代議員<ref>[http://cnnews.chosun.com/big5/site/data/html_dir/2014/03/12/20140312000006.html 朝代議員名單公佈 天安艦主謀再當選] {{webarchive|url=https://web.archive.org/web/20140312213752/http://cnnews.chosun.com/big5/site/data/html_dir/2014/03/12/20140312000006.html |date=2014-03-12 }} 《朝鮮日報》中文版 2014 3 12</ref>。2019年4月，獲委任為國務委員會委員及最高人民會議法制委員會委員長<ref>{{Cite web|url=http://www.rodong.rep.kp/cn/index.php?strPageID=SF01_02_01&newsID=2019-04-12-0002|title=
朝鲜第十四届最高人民会议第一次会议举行第一天会议|date=2019-04-12|accessdate=2019-04-12|work=勞動新聞|language=zh|archive-url=https://web.archive.org/web/20180908092903/http://www.rodong.rep.kp/cn/index.php?strPageID=SF01_02_01&newsID=2019-04-12-0002|archive-date=2019-04-12|dead-url=no}}</ref>。
