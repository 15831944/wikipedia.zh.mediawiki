'''崔知温'''（{{bd|627年||683年|4月27日|catIdx=C}}），字'''礼仁'''，[[唐朝|唐朝]]官员，[[唐高宗|唐高宗]]年间任[[宰相|宰相]]。

== 家世 ==
崔知温生于[[唐太宗|唐太宗]][[贞观_(唐朝)|贞观]]二年（627年），出自[[清河崔氏|清河崔氏]]-{定著}-六房之一的[[许州|许州]]鄢陵房，祖父[[崔世枢|崔世枢]]官爵终唐朝司农卿、使持节[[陕州|陕州]]诸军事、陕州刺史、散骑常待、上大将军、清河郡开国公；父[[崔义直|崔义直]]官爵终长安县令、纪越二王府长史、使持节陕州诸军事、陕州刺史、清河郡开国公。<ref>《唐故正议大夫太子右赞善大夫判太子率更令上柱国清河崔府君（孝昌）墓志铭并序》</ref>他本人则曾任左千牛。

== 任职地方 ==
唐高宗[[麟德|麟德]]（664年-665年）年间，崔知温任[[灵州|灵州]]都督府司马。州边界上有浑、斛薛两个游牧部落，虽然降顺唐朝，却数次侵掠居民。百姓无暇务农，被迫学习骑射以防卫他们。崔知温建议把浑、斛薛迁到[[黄河|黄河]]以北，浑、斛薛不同意，胡人将军[[契苾何力|契苾何力]]也为他们说话，高宗起初拒绝了崔知温的建议。崔知温15次重新上奏，终于获准。当浑、斛薛人到了新的定居处后，发现牧地肥沃，喜欢上了那里。一次斛薛首领入朝[[长安|长安]]觐见高宗时，路过灵州，感谢崔知温的建议为他们造福，拜伏而去。

崔知温四迁[[兰州|兰州]]刺史。<ref>[[郁贤皓|郁贤皓]]《[[唐刺史考全编|唐刺史考全编]]》认为其任期在[[调露|调露]]年间（679年六月—680年八月）之前。</ref>一次，[[党项|党项]]羌军三万攻兰州，州治内几乎没有唐军，众人大为恐慌。崔知温开城门作无事状，党项军以为城内有伏兵，绕过兰州而行。后来将军[[权善才|权善才]]率救兵至，败党项。多数党项军投降，权善才起初想屠戮他们，但崔知温指出全数屠戮不妥，可能引发暴变，阻止了杀戮。权善才感激崔知温的建议，想送他500降人，崔知温拒绝了，说自己的建议是为了让国家得益，而非为己谋私利。后，党项余部因此投降崔知温。

== 任职中央 ==
后，崔知温被召回长安任尚书左丞。[[永隆_(唐朝)|永隆]]元年（680年）四月，因官秩不够高，拜[[黄门侍郎|黄门侍郎]]，特诏[[同中书门下三品|同中书门下三品]]，为实质宰相，与[[侍中|侍中]][[郝处俊|郝处俊]]、左庶子[[高智周|高智周]]、[[给事中|给事中]][[刘景先|刘景先]]兼修国史。<ref>[[:s:旧唐书/卷5|《旧唐书·高宗纪下》]]作[[仪凤|仪凤]]四年（679年）八月事。</ref>[[开耀|开耀]]元年（681年）七月，迁[[中书令|中书令]]。永淳元年（682年）四月，高宗想任用黄门侍郎[[郭待举|郭待举]]、兵部侍郎[[岑长倩|岑长倩]]、秘书员外少监、检校[[中书侍郎|中书侍郎]][[郭正一|郭正一]]、吏部侍郎[[魏玄同|魏玄同]]，对崔知温说：“待举等资任尚浅，且令预闻政事，未可与卿等同名。”于是授郭待举四人[[同中书门下平章事|同中书门下平章事]]，从此外司四品以下知政事者皆用此衔。[[弘道|弘道]]元年（683年）三月，崔知温卒，<ref>[[:s:新唐书/卷3|《新唐书》卷三]]</ref>以礼葬，赠荆州大都督<ref>此据《旧唐书》及《唐故中书令赠荆州大都督清河崔府君妻齐国太夫人杜氏墓志铭》。《新唐书》作[[幽州|幽州]]大都督，误。</ref>，谥'''忠'''。

[[唐德宗|唐德宗]]年间，绘图[[凌烟阁|凌烟阁]]。<ref>[[:s:新唐书/卷191|《新唐书》卷一百九十一]]</ref>
