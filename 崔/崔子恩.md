'''崔子恩'''（{{bd|1958年}}），[[黑龙江|黑龙江]]人，为[[中华人民共和国|中华人民共和国]][[电影|电影]][[导演|导演]]、[[编剧|编剧]]、[[小说家|小说家]]和[[教授|教授]]。崔子恩毕业于[[中国社会科学院|中国社会科学院]]研究生院，并且有[[文学|文学]][[硕士|硕士]]学位，他在[[北京电影学院|北京电影学院]]擔任電影研究所副研究員，其作品已经由[[中国大陆|中国大陆]]和[[香港|香港]]出版发行。<ref>{{cite web |url=http://news.163.com/06/0907/16/2QEC4N6U00011SM9.html |title=崔子恩 我从未改变态度 |accessdate=2006-09-07 |publisher=网易 |work= |language =zh-cn}}</ref>上世紀90年代始，在《花城》雜志和香港發表小說，迄今已經出版長篇小說5部，中短篇小說集3部。另外出版論說著作6部，導演DV長片12部，短片4部。曾多次在國際獲獎。<ref>{{cite web |url=http://www.books.com.tw/products/CN10072422 |title=博客來-光影記憶 |publisher=[[博客來網路書店|博客來網路書店]] |work= |language =zh-tw}}</ref>

== 生平 ==
1991年，崔子恩在北影电影文学系课堂上当着学生公布了自己是[[同性恋|同性恋]]的身份。<ref name="fish " >{{cite web |url= http://news.sina.com.cn/s/2006-04-19/18279664755.shtml|title=崔子恩：只有开始没有结束 |accessdate= 2006年4月19日|publisher=新浪网 |work= |language =zh-cn}}</ref>

2002年，{{tsl|en|International Gay and Lesbian Human Rights Commission|国际同性恋人权委员会}}（IGLHRC）给崔子恩颁奖，鼓励其在同性恋问题上的努力和[[维权|维权]]，虽然他的文字、讲座、电影、书籍一度被[[中华人民共和国政府|中华人民共和国政府]]禁止传播，但是在非正式渠道，仍然流通于社会。

2005年，崔子恩在[[北京|北京]]组织“大陆首届同性恋文化节”，遭到中华人民共和国政府的强力镇压，活动被迫终止，虽然遭遇到维权困难，但是崔子恩表露“这只是开始，没有结束”。<ref name="fish " />
