'''崔真理'''（{{kor|k='''최진리'''|rm='''Choi Jin-Ri<ref name="SMTOWN Live '10 World Tour in 上海站之「藝人演出確認函」（Choi Jin Ri）">{{Cite web |url = http://sh.piao.com.cn/view/images/shanghai/smt/querenhan.html |title = SMTOWN Live '10 World Tour in 上海站之「'''藝人演出確認函'''」（Choi Jin Ri） | publisher = [http://sh.piao.com.cn/ticket_19022.html sh.piao.com.cn（中國票務在綫）] |language = zh-cn |accessdate = 2015-01-08 }}</ref>'''}}；{{bd|1994年|3月29日|catIdx=Jin真}}），[[藝名|藝名]]：'''Sulli'''（{{kor|k='''설리'''}}；{{lang-zh|'''雪莉'''}}），是一名韓國[[歌手|歌手]]、[[演員|演員]]及[[主持人|主持人]]，出生於[[大韓民國|韓國]][[釜山廣域市|釜山直轄市]]，未正式出道就已經以童星的身份拍攝過多部電視劇和電影，2009年正式以[[f(x)_(組合)|f(x)]]身份出道，曾為韓國女子團體[[f(x)_(組合)|f(x)]]成員之一，在隊內擔任副唱及門面擔當。2015年8月7日，正式退出組合，改以演員身份繼續進行演藝活動。

== 早年生活與教育程度 ==
崔真理出生於[[大韓民國|韓國]][[釜山廣域市|釜山直轄市]]，有兩位哥哥和一位弟弟<ref>{{Cite web |url =http://kpopn.com/2013/08/05/187103/ |title =f(x) 和 Sulli 弟弟的合照 |author =LINDA |date =2013-08-05 |publisher =kpopn.com |language = zh-tw |accessdate = |quote = }}</ref>。老大崔炯輝（音譯）是海鮮餐廳廚師<ref>{{Cite web|url=https://www.vlive.tv/video/99107/playlist/95079|title=[V LIVE] [진리상점│Jinri Market]EP07. SULLI FAMILY|accessdate=2018-12-22|work=www.vlive.tv|language=en}}</ref>。真理未正式出道前就已經以童星的身份拍攝多部電視劇和電影，在片場客串的時候，被SM公司星探發掘，並受邀參加選拔賽。2005年，通過青少年組表演組大賞，進入所屬公司培訓，經歴為期四年的練習生時間。崔真理在當練習生期間曾被選為[[SM娛樂|SM娛樂]]公司練習生代表，參加SM十周年慶典，與社長李秀滿、東方神起允浩、安七炫等人，共同切生日蛋糕，並代表所有工作人員及藝人，領唱生日快樂歌。在小學五年級，曾與[[少女時代|少女時代]]成員[[金泰耎|太妍]]和[[黃美英|蒂芬妮]]同住同一個宿舍，直到她們以[[少女時代|少女時代]]成員的身份出道<ref>{{Cite web |url = http://www.allkpop.com/article/2011/06/fxs-sulli-tears-up-while-talking-about-snsds-taeyeon-tiffany#axzz2t5mMvzVc |title = f(x)'s Sulli tears up while talking about SNSD's Taeyeon, Tiffany | author= | date=2011-06-28 | publisher=allkpop.com | language=en }}</ref>。崔真理曾就讀於{{link-ko|中部初等學校（梁山市）|중부초등학교}}、{{link-ko|首爾淸潭初等學校|서울청담초등학교}}、{{link-ko|首爾淸潭中學校|청담중학교 (서울)}}和[[首爾公演藝術高等學校|首爾公演藝術高等學校]]。2012年，崔真理放棄2013年大學入學考試，把重心放在演藝事業上<ref>{{Cite web |url = http://www.koreastardaily.com/tc/news/21854 |title = f(x) Sulli放棄高考 Krystal努力籌備 | author= | date=2012-10-22 | publisher=韓星網 - TVdaily中文版 | language=zh }}</ref>。

== 影視事業 ==
=== 2012年：《致美麗的你》===
2012年，崔真理出演由[[SM娛樂|SM娛樂]]改編自日本知名漫畫《[[偷偷愛著你|偷偷愛著你]]》製作的電視劇《[[致美麗的你|致美麗的你]]》並首次擔當女主角。崔真理在劇中扮演一名陽光且正義的熱血少女，剪掉頭髮扮成男生到了GENIE體育高中就讀<ref>{{Cite web |url =http://kpopn.com/2012/07/16/133562/ |title =Sulli 為戲剪掉60公分長髮 |author =太咪 |date =2012-07-16 |publisher =kpopn.com |language =zh-tw |accessdate = |quote = }}</ref>。於第19屆[[SBS演技大賞|SBS演技大賞]]憑著該劇獲得「新星賞」。

=== 2014年：《海盜：奔向大海的山賊》、《時尚王》===
2013年8月13日，[[SM娛樂|SM娛樂]]證實崔真理出演電影《海盜：奔向大海的山賊》，並飾演Black Cat（黑貓）的角色<ref>{{Cite web |url =http://kpopn.com/2013/08/13/188255/ |title =Sulli 將出演電影《海盜》 |author =CHI |date =2013-08-13 |publisher =kpopn.com |language = zh-tw |accessdate = |quote = }}</ref>。電影斥資100億韓圜，由李錫勳執導，被媒體譽為韓國版的「加勒比海盗」。崔真理在劇中扮演一名有著活潑個性、擔任氣氛製造者的角色。劇情以朝鮮時代為背景，講述女海盜與男山賊為了抓捕一條鯨魚的過程發生的種種故事。在電影中崔真理亦是首次與韓國著名女演員[[孫藝珍|孫藝珍]]在大銀幕上合作，[[孫藝珍|孫藝珍]]在出席制作發布會還稱讚崔真理可愛，拍攝時氣氛非常愉快<ref>{{Cite web |url =http://www.koreastardaily.com/tc/news/43998 |title =孫藝珍大讚f(X)Sulli可愛 |author =Cynthia |date =2014-07-02 |publisher =韓星網 - TVdaily中文版 |language = zh |accessdate = |quote = }}</ref>。[[f(x)_(組合)|f(x)]]隊長[[宋茜|Victoria]]與[[SM娛樂|SM娛樂]]旗下歌手一起親身出席於7月29日在首爾紫陽洞樂天影院舉行的VIP試映會作支持<ref>{{Cite web |url =http://www.koreastardaily.com/tc/photo/45272 |title =SM娛樂旗下歌手助陣《海盜》VIP試映會 |author = |date =2014-07-30 |publisher =韓星網 - TVdaily中文版 |language =zh |accessdate = |quote = }}</ref>。電影於2014年8月6日在韓國上映短短三天，已經突破百萬觀看人次，亮眼成績獲得各界的關注<ref>{{Cite web |url =http://kpopn.com/2014/08/10/235343/ |title =《海盜》觀影人次破百萬 |author =CHI |date =2014-08-10 |publisher =kpopn.com |language =zh-tw |accessdate = |quote = }}</ref>。

2013年12月27日，電影《時尚王》的製作商證實崔真理將與[[周元_(韓國)|周元]]攜手合作擔任男女主角<ref>{{Cite web |url =http://kpopn.com/2013/12/27/205031/ |title = 周元,Sulli合作電影-時尚王 |author =CHI |date =2013-12-27 |publisher =kpopn.com |language =zh-tw |accessdate = |quote = }}</ref>。該劇根據同名漫畫《時尚王》而改編，講述故事是為了配得上同班最漂亮的女孩而決定了要成為帥氣男人的一位少年的故事，而崔真理飾演電影中令這位少年改變的「臉讚」少女。

== 暫停演藝活動事件 ==
2014年7月25日，[[SM娛樂|SM娛樂]]於f(x)官方網站公佈崔真理因網路上持續性的惡意評論和不實謠言而身心疲憊，所以向SM娛樂表明想暫停演藝活動，休息一段時間的想法。經SM娛樂慎重考慮過後，決定尊重崔真理的意思，同時基於保護藝人的立場，把活動行程減量到最少，讓她得到充分的休息<ref>{{Cite web |url = http://fx.smtown.com/Link/Board/6730951 |title = 안내 말씀 드립니다. | author= | date=2014-07-25 | publisher=SM娛樂 | language=ko }}</ref>。10月31日，出席在首爾往十里CGV的《時尚王》媒體試映會和VIP試映會<ref>{{cite news |language = zh |author = 梁夢竹 |url = http://www.epochtimes.com/b5/14/11/3/n4287195.htm |title = f(X) Sulli停工90天 首度露面宣傳新片 |publisher = 大紀元 |date = 2014-11-03 |accessdate = 2015-01-22 }}</ref>。

2015年1月13日，首次與團員[[朴善英|Luna]]公開出席SM娛樂舉辦的活動<ref>{{cite news |language = zh |url = http://www.koreastardaily.com/tc/news/54624m |title = f(X) Sulli、Luna減肥前後對比效果驚人 |publisher = 韓星網 |date = 2015-01-14 |accessdate = 2015-01-23 }}</ref>。

2016年4月24日，影音網站更出現了名為「去Club的雪莉」的影片，拍下了暫停演藝活動的雪莉在Club內狂歡跳舞的樣子，吸引韓國網民。<ref>[{{cite web |url=http://www.vlovekpop.com/20160427-sulli/ |title=存档副本 |accessdate=2016年4月27日 |deadurl=yes |archiveurl=https://web.archive.org/web/20160430052640/http://www.vlovekpop.com/20160427-sulli/ |archivedate=2016年4月30日 |df= }} 雪莉再度吸引FANS！「 去Club的雪莉」點撃率超過20萬！現崔真理已與崔子分手</ref>。

== 退出f(x)組合 ==
2015年6月25日，據韓國媒體Herald經濟報導爆料，崔真理將會退出f(x)，專心朝演員的方向前進。但與SM娛樂的合約尚未到期，在這一年之中會做為演員獨自發展。SM娛樂對此事件做出官方回應否認，但並未詳細說明，僅表示「目前內部並未確定崔真理未來動向，會慎重處理。」<ref>{{cite news |language = zh |author = 鍾志均 |url = http://www.chinatimes.com/newspapers/20150626000836-260112 |title = f(x)雪莉爆退團演戲去 |publisher = 中國時報 |date = 2015-06-26 |accessdate = 2015-08-08}}</ref>8月7日，SM娛樂通過韓國社交媒體表示，尊重崔真理集中個人演技活動的意願，同意崔真理退出f(x)組合，改以演員身份繼續進行演藝活動<ref>{{cite news |language = zh |author = JiaJia |url = http://www.koreastardaily.com/tc/news/66398 |title = 雪莉確定退出組合f(x)，SM娛樂：尊重個人意願，今後集中演技活動 |publisher = Korea Star Daily |date = 2015-08-07 |accessdate = 2015-08-08}}</ref>。
