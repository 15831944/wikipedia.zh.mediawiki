'''崔晴晴'''{{BD|1993年|11月1日|||catIdx= C崔}}又叫'''崔安娜'''，出生于[[美国|美国]][[加州|加州]][[洛杉矶|洛杉矶]]，就读于圣玛利诺学区杭廷顿中学，参加[[美国|美国]][[南加州小姐|南加州小姐]]比赛荣获8个单项大奖，一举走红。<ref name="zhguojy " /><ref name="cctv1 " >{{cite web |url=http://discovery.cctv.com/20080111/101016.shtml |title=容貌与才华获美主流认同 12岁选美小皇后的故事 |accessdate= 2008年|work= |publisher= CCTV|language =zh-cn}}</ref>

== 简介 ==
崔晴晴从小痴迷于[[动画片|动画片]]，达到了废寝忘食的地步，不过她的母亲对她要求严格，对她的功课非常重视，经过一次学习的募捐活动后，她从母亲的传统教育方式下解放，走向西方先进教育方式，开始重视孩子的自我意识和独立自主能力，并且带着她游走于[[欧洲|欧洲]]、[[北美|北美]]、[[中国大陆|中国大陆]][[北京|北京]]，她母亲说这是“读万卷书行万里路”，以让她获得更加宽广的知识面。<ref name="zhguojy ">{{cite web|url=http://www.jyb.cn/xwzx/gjjy/gjgc/t20080107_135799.htm|title=多才多艺小才女　华裔选美小皇后成长故事|accessdate=2008年|work=|publisher=中国教育网|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20160305014343/http://www.jyb.cn/xwzx/gjjy/gjgc/t20080107_135799.htm|archivedate=2016年3月5日|df=}}</ref><ref name="cctv1 " />

== 奖项 ==
2006年，崔晴晴荣获“[[南加州小姐|南加州小姐]]”（Miss Southern California），落砂机市长大卫·加迪瑞斯（David Gutierrez）亲自为其颁奖。<ref name=" ops" >{{cite web |url=http://www.chinanews.com/hr/tpbd/news/2006/08-17/775351.shtml |title=图:华裔首摘“南加州小姐”后冠 |accessdate= 2008年|work= |publisher=中国新闻网 |language =zh-cn}}</ref>

2008年，崔晴晴荣获“[[全美妙龄小姐|全美妙龄小姐]]”才艺冠亚军双奖。<ref>{{cite web |url=http://www.ce.cn/xwzx/gnsz/gdxw/200912/30/t20091230_20719029_2.shtml |title=2009年华裔选美八宗最：东方美征服全世界 |accessdate= 2008年|work= |publisher= 中国经济网|language =zh-cn}}</ref>
