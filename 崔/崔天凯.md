'''崔天凯'''{{BD|1952年|10月|catIdx=Cui崔}}，[[浙江省|浙江省]][[宁波市|宁波市]]人，生于[[上海|上海]]。[[中华人民共和国|中华人民共和国]][[大使|大使]]衔职业[[外交官|外交官]]，现任[[中华人民共和国驻美国大使|驻美国特命全权大使]]<ref>[http://www.apdnews.com/news/26530.html 崔天凯大使接受美《外交事务》杂誌专访摘要] {{webarchive|url=https://web.archive.org/web/20131127065458/http://www.apdnews.com/news/26530.html |date=2013-11-27 }}，《[[亚太日报|亚太日报]]》，2013年5月22日</ref><ref name=":0">{{Cite web |url = http://www.china-embassy.org/chn/sgzhichuang/dsjianli/ |title = 崔天凯大使简历 |accessdate = 4-10-2015 |author = 中華人民共和國駐美利堅合眾國大使館 |date = 2013-04-08 |publisher = }}</ref>。

== 简历 ==
1974年至1977年，崔天凯就读于上海师范大学（今[[华东师范大学|华东师范大学]]）{{NoteTag|1972年5月至1980年7月，[[华东师范大学|华东师范大学]]改名“上海师范大学”。|name="师大"}}[[华东师范大学外语学院|外语系]]英语专业，毕业後留校任教一年<ref name=":1">[http://cpc.people.com.cn/gbzl/html/121001415.html 中国党政干部资料库]</ref>。1978年至1979年，他再次進入上海师范大学，成為一外语系研究生<ref name=":0" />。1979年，他進入了[[北京外国语学院|北京外国语学院]]的联合国译员培训班，以研究生身分深造兩年<ref name=":0" />

1981年畢業後，崔天凯服務於纽约[[联合国秘书处|联合国秘书处]]会议服务司中文处，擔任翻譯員<ref name=":1">[http://cpc.people.com.cn/gbzl/html/121001415.html 中国党政干部资料库]</ref>。1984年，崔天凯加入[[中华人民共和国外交部|中华人民共和国外交部]]，歷任[[随员|随员]]、[[三等秘书|三等秘书]]。

1986年，崔天凯赴美國[[華盛頓|華盛頓]]的[[约翰·霍普金斯大学|约翰·霍普金斯大学]]{{Link-en|约翰·霍普金斯大学高级国际关系学院|Paul H. Nitze School of Advanced International Studies}}學習，並在翌年获得国际公共政策硕士学位。

1987年至1996年，崔天凯先后担任[[中华人民共和国外交部国际司|外交部国际司]]副处长、处长、参赞。1996年至1997年，他調至[[中华人民共和国外交部新闻司|外交部新闻司，擔任]]副司长及[[中华人民共和国外交部发言人|外交部发言人]]。

1997年，崔天凯獲委任為[[中华人民共和国常驻联合国代表团|中华人民共和国常驻联合国代表团]]{{Link-de|公使銜參贊|Gesandter-Botschaftsrat}}，再次在聯合國服務。

1999年，崔天凯被調回國，出任[[外交部政策研究室|外交部政策研究室]]常务副主任，2001年升為主任。2003年至2006年，他担任[[中华人民共和国外交部亚洲司|外交部亚洲司]]司长。

2006年1月，崔天凯出任外交部部长助理。

2007年9月接替回國出任外交部黨委書記、副部長的[[王毅|王毅]]<ref>[http://world.people.com.cn/GB/8212/14450/46162/6313169.html 王毅卸任驻日大使回国 出任外交部党委书记、副部长]</ref>为[[中国驻日本大使|中国驻日本大使]]<ref>[http://202.120.85.99/NAVManager/ShowNAVDetails.aspx?NAVId=902&Type=News 华东师范大学校友会 崔天凯校友返校]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |url = http://www.gov.cn/ldhd/2007-09/30/content_766360.htm |title = 胡锦涛任命崔天凯为驻日大使 免王毅驻日大使职务 |publisher = 新華社 |date=2007年9月30日 }}</ref>，並於11月2日向日本天皇明仁呈上國書<ref>{{Cite web |url = http://www.mofa.go.jp/mofaj/press/release/h19/11/1176044_816.html |title=新任駐日中華人民共和国大使の信任状捧呈について|date=2007-11-01 |accessdate=2010-09-26 |publisher=[[日本外務省|日本外務省]]}}</ref>。

2009年1月任滿回國，出任外交部副部长。

2013年4月接替回國復任外交部副部长的[[張業遂|張業遂]]<ref>[http://realtime.zaobao.com/2013/02/jz130228_026.shtml 张业遂任中国外交部副部长]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，擔任第十任[[中华人民共和国驻美国大使|驻美国特命全权大使]]<ref>{{cite web |url = http://news.ifeng.com/mainland/detail_2013_04/03/23819526_0.shtml |title = 崔天凯抵华盛顿履新 成为第十任中国驻美国大使 |publisher=凤凰网 |date=2013年4月3日 }}</ref>。

崔天凯亦为[[华东师范大学|华东师范大学]]校友会理事会顾问，华东师范大学日本校友会名誉会长<ref>[http://news.ecnu.edu.cn/s/192/t/342/e8/7c/info59516.htm 华东师范大学新闻中心 崔天凯任中华人民共和国第十任驻美国大使]</ref>。

据美国前国务卿[[希拉里·克林顿|希拉里·克林顿]]自传，崔天凯为2012年[[陈光诚|陈光诚]]事件中的中方主要谈判代表。在希拉里抵达中国的前夕，崔天凯代表中方与美方达成初步协议。在美方向中方提议将陈光诚送到中国大学学习法律后，崔天凯看到美方的建议名单包括其母校华东师范大学而感到不满。在陈光诚改变主意决定离开中国之后，希拉里与时任[[中央外事工作领导小组办公室|中央外事工作领导小组办公室]]主任[[戴秉国|戴秉国]]达成新的协议允许陈光诚赴美國留学。其后崔天凯负责与美方代表时任助理国务卿坎贝尔共同制定方案细节<ref>[http://www.voachinese.com/content/hiliary-book-chenguangcheng-20140611/1934389.html 希拉里新书：营救陈光诚 如同间谍小说] [[美国之音|美国之音]]，2014年6月11日</ref>。
