'''象山湾疏港高速公路'''是浙江省规划中的一条高速公路，又名'''宁波舟山港梅山港区沈海高速连接线'''。公路起点为[[宁波市|宁波市]][[北仑区|北仑区]][[春晓街道|春晓街道]]昆亭村南侧的[[六横疏港高速公路|六横疏港高速公路]]昆亭枢纽，终点为宁波市[[鄞州区|鄞州区]][[塘溪镇|塘溪镇]]鹳山村西北侧的[[甬莞高速公路|甬莞高速公路]]（[[甬台温复线高速公路|甬台温复线高速公路段]]）塘溪枢纽。高速规划总长度24.15km公里，主线采用双向六车道高速公路标准，设计速度100km/h，整体式路基宽度33.5m 。全线设昆亭枢纽、春晓互通、瞻岐互通、塘溪枢纽互通式立交4处，春晓互通、瞻岐互通收费站2处，临时接线收费站1处。<ref name="plan">{{Cite web |url=http://www.nbplan.gov.cn/art/2018/12/20/art_19063_1936.html |title= 宁波舟山港梅山港区沈海高速连接线（象山湾疏港高速）昆亭至塘溪段工程选址公示（批前公示） |accessdate=2018-12-22 |author=宁波市高等级公路建设指挥部 |date=2018-12-20 |publisher=宁波市规划局}}</ref><ref name="news-plan">{{Cite web |url=http://news.cnnb.com.cn/system/2018/12/22/030013078.shtml |title=连接甬台温高速复线 象山湾疏港高速这段工程或这样走 |accessdate=2018-12-22 |author=张颖 陈捷 |date=2018-12-22 |publisher=中国宁波网}}</ref>
