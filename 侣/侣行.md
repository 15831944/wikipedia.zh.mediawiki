《'''侣行'''》（{{lang-en|ON THE ROAD}}）是中国视频网站[[优酷|优酷]]联合张昕宇（[[绰号|绰号]]“270”<ref>读作“二百七”，源自曾经的体重。[http://cul.qq.com/a/20140219/024062.htm 《侣行》：行到极端 将世界作为生命的背景]</ref>）、梁红打造的一档网络自制户外[[真人秀|真人秀]]节目，共制作3季。节目通过将张昕宇、梁红等人的[[旅行|旅行]]经历拍摄成视频制作而成。《侣行》第一季于2013年6月13日在优酷正式开播，三个月即突破1亿次播放量<ref>{{Cite web |url = http://ent.163.com/14/0221/14/9LK767VE00031GVS.html |title = 中国人第一场南极婚礼 邱启明主持《侣行》直播|author = |date = 2014-02-21 |publisher =  网易娱乐|language = zh-hans |accessdate = 2015-12-03 |quote = }}</ref>。第二季于2013年年底开播，总播放量超过8000万。第三季于2015年6月25日在优酷开播<ref>{{Cite web |url = http://twent.chinayes.com/content/20150626/KKJMICMSN4W14.shtml |title = 《侣行》第三季上线首日播放破1100万 |author =  |date = 2015-06-26 |publisher = Yes娱乐新闻中心 |language = zh-hans |accessdate = 2015-12-03 |quote =  }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。2014年3月24日至31日，《侣行》剧场版登陆[[央视一套|央视一套]]<ref>{{Cite web |url = http://news.ifeng.com/gundong/detail_2014_03/24/35085409_0.shtml |title = 《侣行》剧场版央视今播|author = |date = 2014-03-24 |publisher = 凤凰资讯 |language = zh-hans |accessdate = 2015-12-03 |quote = }}</ref>。2017年4月29日，与[[腾讯视频|腾讯视频]]合作、[[小米手机|小米手机]]独家冠名的《我们的侣行》首播{{R|jm4}}<ref>{{cite news |author1=红网综合 |title=腾讯视频《我们的侣行》首播 活用“腾讯系力量”布局 |url=http://ent.chinadaily.com.cn/2017-05/02/content_29165586.htm |accessdate=2018-07-04 |publisher=中国日报中文网 |date=2017-05-02}}</ref>。2018年6月12日，《我们的侣行》第二季首播。

*2018-12-30 第一集 驶进北极圈
*2019-01-08 第二集 穿越极地之旅*
*2019-01-15 第三集 世界最孤独的小镇
2019-01-22 第四集 极地飞行师

==暂停播出==
