'''支城'''，是為了保護本城而部署的輔助用[[城堡|出城]]、[[要塞|砦]]（寨）、[[陣屋|陣屋]]等建築設施。亦稱做'''枝城'''等。在戰國時代時，形成了各式各樣的'''支城網'''－本城、支城間的連結網絡。

== 語源 ==
若是使用由自漢語的發音「本城（ほんじょう）」時，與其相對應會使用支城（しじょう）」。若是使用大和語「根城（ねじろ）」時，則會用「枝城（えだじろ）」與其對應。漢字結構上，「枝城」的「枝」以及「本城」的「本」分別是標示樹木的分支和樹木的根本，都是由樹木轉化而來且原本就幾乎是同義字。不管使用「支城」或是「枝城」，皆是由樹木的結構[[比喻|比喻]]城與城之間的關係。

== 種類 ==
支城依其功能而種類多樣、大小規模不一。<ref>図説・戦う城の科学 古代山城から近世城郭まで軍事要塞たる城の構造と攻防のすべて（SBクリエイティブ
・2015年4月16日）</ref>
; 防禦據點
* 詰城
: 作為防守本城用的出城。位在本城附近，設置在山地等適合防禦的要害處。
* 直轄支城
; 攻城戰用
* 向城、附城、對城
: 攻城戰時為與其相對峙而築的城塞（有時也指稱本城附屬之城）。

; 間接式支城
* 境城、境目城
: 設於領地邊境的城。為防禦[[織田信長|織田信長]]的[[中國征伐|中國征伐]]，[[毛利氏|毛利氏]]以[[備中國|備中國]]邊境上的「境目七城（[[冠山城|冠山城]]、[[高松城|高松城]]、[[宮地山城|宮地山城]]、[[鴨庄城|鴨庄城]]、[[日幡城|日幡城]]、[[松島城|松島城]]、[[撫川城|撫川城]]）」固守。
* 番城、番手城
: 沒有城主而由本城直轄的支城（[[江户时代|江戶時代]]的[[大阪城|大阪城]]或[[二條城|二条城]]等亦為[[公儀|公儀]]的番城）。
* 繋城、傳城
: 負擔有：以狼煙作通信聯絡、街道的[[關所|關所]]、兵力的集結、移動、[[屯駐地|駐紮基地]]等作用的城。

; 其他
* 陣城
: 為了[[野外戰|野戰]]等目的而臨時構築的簡易城塞。

== 支城網 ==
'''支城網'''主要發達於[[日本|日本]][[战国时代_(日本)|戰國時代]]，各地區的據點或作為根據地的本城周圍，築有輔助作用而稱作支城、枝城、端城等的城、砦、陣屋，其相互連結成的[[防衛戰|防衛網]]或是[[統治|統治]][[社会网络|網絡]]。
; 支城網的例子
* 治理過關東地方的[[後北條氏|後北條氏]]，以居城[[小田原城|小田原城]]（現[[神奈川縣|神奈川縣]][[小田原市|小田原市]]）為中心，建立了120個城寨構成的支城網。
* [[上野國|上野國]]的[[箕輪城|箕輪城]]（現[[群馬縣|群馬縣]][[高崎市|高崎市]]），曾有多達大小302處支城、堡壘所構建的支城網。
* 支配[[出雲國|出雲國]]的[[尼子氏|尼子氏]]，作為居城[[月山富田城|月山富田城]]（現[[島根縣|島根縣]][[安來市|安來市]]）的防衛線，在出雲各地構築了稱為[[尼子十旗|尼子十旗]]、[[尼子十砦|尼子十砦]]的10個主要支城和10的城寨。

== 終焉 ==
支城作為支撐本城防守的據點同時，也有風險成為對抗勢力的攻擊基地。因此為防止敵人反攻也曾有發生過[[破城|破城]]。

以[[中央集权|中央集權]]體制的天下統一為目標的[[織田信長|織田信長]]和[[豐臣秀吉|豐臣秀吉]]，整理其統治地區的支城予以破壞撤銷，最終到了[[江戶幕府|江戶幕府]]頒布[[一國一城令|一國一城令]]，使得支城（1615年）全部[[廃城|廢城]]處理。<ref>図解 戦国合戦がよくわかる本: 武具・組織・戦術から論功行賞まで（2013年6月21日 [//ja.wikipedia.org/wiki/PHP研究所 PHP研究所]）</ref>
