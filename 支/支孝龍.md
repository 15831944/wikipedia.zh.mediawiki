'''支孝龍'''，[[西晉|西晉]][[淮陽|淮陽]]人，以風姿見重，長於《小品般若》<ref>{{cite book |author1=釋慧皎 |title=梁[[高僧傳|高僧傳]] 卷四|quote= 少以風姿見重,加復神彩卓犖,高論適時。常披味《小品》,以為心要。}}</ref>。他具有深厚的玄學素養，對於[[佛教|佛教]]在整個魏晉玄學思潮中得以推擴，具有相對重要的地位<ref>{{cite journal |author1=吳中銘 |title=西晉「八達」之游所見名僧與名士之交往 |journal=社会科学研究 |date=2010-07-01 |issue=4 |page=152}}</ref>。

== 經歷 ==
支孝龍與當時名士[[阮瞻|阮瞻]]、[[庾凱|庾凱]]交往, 被稱為西晉「八達」之一<ref>{{cite book |author1=釋慧皎 |title=梁高僧傳 卷四|quote=陳留阮瞻頴川庾凱。並結知音之交世人呼為八達。}}</ref>。在魏晉玄學發展時期，能夠通釋、道思想，並且展開佛教義理的論述是相當不簡單的一件事。當時[[竺叔蘭|竺叔蘭]]初譯《[[放光般若經|放光般若經]]》，支孝龍便前往其住處加以抄錄、校訂，得到譯本研閱十幾日後，對眾開講，得到時人的推崇<ref>{{cite book |author1=釋慧皎 |title=梁高僧傳 卷四|quote=時竺叔蘭初譯放光經。龍既素樂無相。得即披閱旬有餘日。便就開講。}}</ref><ref>{{cite book |title=釋氏通鑑 卷二|quote=沙門支孝龍有機辨。是年講放光般若。人皆宗仰。阮瞻庚凱尤敬之。}}</ref>。

他精研[[般若|般若]]之學，但也深受當時[[玄學|玄學]]影響，他曾說：「抱一以逍遙。唯寂以致誠。<ref>{{cite book |author1=釋慧皎 |title=梁高僧傳 卷四|quote=時或嘲之曰。大晉龍興天下為家。沙門何不全髮膚去袈裟釋胡服被綾羅。龍曰。抱一以逍遙。唯寂以致誠。剪髮毀容改服變形。彼謂我辱我棄彼榮。故無心於貴而愈貴。無心於足而愈足矣。其機辯適時皆此類也。}}</ref>」，其用語「抱一」、「逍遙」出於[[道家|道家]]用語，「致誠」則為[[儒家|儒家]]用語，事實上其言論是較接近於[[清談|清談]]<ref>{{cite book |author1=文心工作室 |title=中文經典100句：四朝高僧傳 |date=2011/08/07 |publisher=商周出版社}}</ref>，這一方面顯示當時佛教傳入中國，仍受儒、道影響的情形，一方面也反映了支孝龍的玄學素養，和與名士阮瞻等人交好的情況，[[湯用彤|湯用彤]]便曾從支孝龍等人的事蹟，探討研究佛教玄學混合在晉代的發展情況<ref>{{cite book |author1=湯用彤 |title=漢魏兩晉南北朝佛教史 |date=1991 |publisher=上海書店 |pages=172-173||quote=吾人觀西晉竺叔蘭、支孝龍之風度，東晉‧康僧淵、帛高坐等之事迹，則老、莊清談佛教玄學之結合，想必甚早。王、何、嵇、阮之時，佛法或已為學士所眷顧。及名士避世江東，亦遂與沙門往還。}}</ref>。

== 評價 ==
