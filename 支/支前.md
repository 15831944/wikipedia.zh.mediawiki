'''支前'''即'''支援前线'''<ref>{{Cite web |url=http://www.crt.com.cn/news2007/News/jryw/2018/11/181111538935FI1DHKD07H11D4H4HD.html |title=特稿：全民动员 参军支前（图）|accessdate=2019-04-18 |author=莱阳市委宣传部供稿 |date=2018-11-01 |work= |publisher=中红网|language=Zh-hans }}</ref>、或称'''支前运动'''<ref name="a1"/>，是[[中国共产党|中国共产党]]领导的各级政权在[[第二次国共内战|第二次国共内战]]期间，动员、组织[[解放区|解放区]]民众为[[中国人民解放军|中国人民解放军]]提供[[后勤|后勤]]保障。

第二次国共内战期间，仅[[山东解放区|山东解放区]]就有59万青年参军，700万民工随军征战。就不完全统计而言，[[三大战役|三大战役]]之一的[[辽沈战役|辽沈战役]]中，东北诸省和[[熱河省_(中華民國)|热河省]]动员支前民工183万人。[[淮海战役|淮海战役]]中，[[华东解放区|华东]]、[[中原解放区|中原]]、[[冀鲁豫解放区|冀鲁豫]]、[[华中解放区|华中]]四个解放区动员民工543万人。加上[[平津战役|平津战役]]，三大战役总共动员民工880万余人次。有认为支前运动为解放军取得第二次国共内战胜利，立下汗马功劳。[[陈毅|陈毅]]曾评价：[[淮海战役|淮海战役]]的胜利，是人民群众用小推车推出来的<ref name="a1">{{Cite web |url=http://dangshi.people.com.cn/GB/165617/166495/168111/10105333.html |title=支前运动|accessdate=2019-04-18 |author= |date= |work= |publisher=人民网|language=Zh-hans }}</ref>。

中华人民共和国建立后，县以上各政府逐步撤销支前办公事室。目前，仅福建省因处[[臺海現狀|台海对峙前线]]，军事上长期高压，成为全国唯一保留省、市、县三级支前机构的省份。2016年时，福建省三级支前机构在福建全省已建立了69个支前物资供应站<ref>{{Cite web |url=http://www.81.cn/gfbmap/content/2016-02/01/content_770.htm |title=追踪最后的省级支前办|accessdate=2019-04-18 |author=罗剑，记者：徐文涛 |date=2016-02-01 |work= |publisher=中国军网，来源：解放军报|language=Zh-hans }}</ref>，主要为解放军进行应急物资的常态化、规范化保障。[[深化国防和军队改革|深化国防和军队改革]]后，保障范围由陆军扩大至其他兵种，并出台了《福建省支前物资供应站建设情况检查验收评分细则》<ref>{{Cite web |url=http://www.81.cn/jfjbmap/content/2016-07/29/content_152036.htm |title=物资供应站，从“支前”到“支联”|accessdate=2019-04-18 |author=李亮平，记者：朱达 |date=2016-07-29 |work= |publisher=中国军网，来源：解放军报|language=Zh-hans }}</ref>。
