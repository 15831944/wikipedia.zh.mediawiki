'''支墩坝'''是一种坝型，迎水面坝体被背水面的支墩所支撑。面板直接承受上游水压力和泥沙压力等荷载，通过支墩将荷载传给地基。<ref>{{cite web
|url=http://www.dur.ac.uk/~des0www4/cal/dams/gloss/fg.htm
|title= Glossary
|publisher=
|accessdate=2007-02-05
}}</ref>  <ref>{{cite web
|url=https://www.pbs.org/wgbh/buildingbig/dam/buttress_forces.html
|title=Buttress dam forces
|publisher=
|accessdate=2007-02-05
}}</ref>

支墩坝适合于宽谷、基岩暴露少的地方。<ref>{{cite web
|url=http://www.dur.ac.uk/~des0www4/cal/dams/conc/concf19.htm
|title=Introduction to Buttress Dams

}}</ref>

大头坝(massive head buttress dam)面板由支墩上游部分扩宽形成，称为头部。相邻支墩的头部用伸缩缝分开，为大体积混凝土结构。

1968年加拿大修建的[[马尼克五级连拱坝|马尼克五级连拱坝]]，坝高214m，是当前世界上最高的支墩坝。1975年巴西和巴拉圭修建的[[伊泰普水电站|伊泰普水电站]]大头坝，坝高196m，是当前世界上最高的大头坝。1948年阿根廷建造了艾斯卡巴坝,坝高83m，是当前世界上最高的倾斜盖面平板坝。苏联修建了一些土基上的溢流平板坝。

中国建造的支墩坝有：
1956年建成的[[梅山水库|梅山连拱坝]],坝高88.24m。1958年建成的[[金江水库|金江平板坝]],坝高54m。1960年建成的[[新丰江水库|新丰江大头坝]]，坝高105m。1980年建成的[[湖南镇水库|湖南镇梯形坝]]，坝高129m，是中国最高的支墩坝。
