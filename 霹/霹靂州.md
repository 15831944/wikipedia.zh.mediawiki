'''霹靂'''（旧称'''吡叻'''，[[马来语|马来语]]：{{lang|ms|Perak}}）是[[馬來西亞|馬來西亞]]的十三个州属之一，也是[[馬來半島|馬來半島]]11個州當中的第2大州，面积约21000平方公里，北部连接[[馬來西亞|馬來西亞]]东北部的馬、[[泰國|泰]]邊境以及[[吉打州|吉打州]]、西北部接壤[[槟城|槟城]]、东边有[[吉兰丹州|吉兰丹州]]和[[彭亨州|彭亨州]]、南边[[雪兰莪州|雪兰莪州]]。[[马六甲海峡|马六甲海峡]]则处于霹雳州西方。

“{{lang|ms|Perak}}”在[[马来语|马来语]]即为[[銀|銀]]，或许是來自霹靂州以前主要產品[[錫|錫]]的銀色顏色。此外也传说霹雳州地名其实源自河里鱼儿在阳光下闪烁似银而来。

霹雳州首府是[[怡保|怡保]]（{{lang|ms|Ipoh}}），英殖民时期以市内繁荣的锡矿开采业著名，锡价下滑使其经济受到了影响。霹雳州苏丹皇宫则位于皇城[[江沙|江沙]]（{{lang|ms|Kuala Kangsar}}）。

== 历史 ==
传说中霹雳州东北部曾是一个[[兴都教|兴都教]]王国，名为[[刚迦王国|刚迦王国]]（''{{lang|ms|Gangga Negara}}''）。考古发现也证明了霹雳州在史前已有人类居住。

霹雳州的现代史在[[马六甲王朝|马六甲王朝]]的灭亡后才开始发展。马六甲最后一个[[苏丹|苏丹]]，[[苏丹马末·沙|苏丹马末·沙]]（''{{lang|ms|Sultan Mahmud Shah}}''）的长子目札法沙（{{lang|ms|Raja Muzaffar Shah}}），于1511年[[葡萄牙|葡萄牙]]进攻[[马六甲|马六甲]]时逃离王国。于1528年，目札法沙（{{lang|ms|Raja Muzaffar Shah}}）在霹雳河畔成立了新的王朝。霹雳州境内蕴藏着丰饶的锡矿使霹雳王朝被欧洲殖民主义国家虎视眈眈。

=== 荷兰殖民时期 ===
荷兰人于17世纪欲独揽锡矿产业的意图失败，同时于霹雳河口和[[邦咯岛|邦咯岛]]上建筑堡垒。

荷兰人早期登陆霹雳州的历史始于1641年。当时荷兰成功控制了马六甲海峡，并独揽了锡矿和香料产业。荷兰人尝试说服目札法沙（{{lang|ms|Raja Muzaffar Shah}}），以一并独揽霹雳州的锡矿产业，但以失败告终。通过[[亚齐|亚齐]]王国的''{{lang|ms|Sultanah Tajul Alam Safiatuddin}}''，荷兰最后成功强迫霹雳州[[苏丹|苏丹]]签署条约，于1650年8月15日，允许荷兰于[[瓜拉霹雳|瓜拉霹雳]]建造工厂，但同时却在霹雳王室之中引起不满。

1651年，[[天猛公|天猛公]]带领群众破坏工厂，促使荷兰离开霹雳州。

1655年，荷兰派使者到霹雳州重新签署条约，并为工厂被破坏一事寻求赔偿。霹雳王朝由于没有遵守条约规定而被荷兰军队包围。霹雳人民联合[[亚齐|亚齐]]和''{{lang|ms|Ujung Salang}}''突袭荷兰军队。

1670年，荷兰人回到霹雳州，并在[[邦咯岛|邦咯岛]]上建造了''{{lang|ms|Kota Kayu}}''，今荷兰村（''{{lang|ms|Kota Belanda}}''）。由于听闻[[暹罗|暹罗]]王朝将进攻自己，霹雳王朝准许了建筑工程。虽然如此，霹雳于1685年又把荷兰人驱离[[邦咯岛|邦咯岛]]。荷兰人尝试重定条约，但始终失败。

=== 英国殖民时期 ===
18世纪时，[[武吉斯人|武吉斯人]]、[[亚齐|亚齐]]和[[暹罗|暹罗]]都尝试进攻霹雳王朝。1820年英国的插手使[[暹罗|暹罗]]王朝没能并吞霹雳王朝进入自己的版图。英国原本并没有在[[马来亚|马来亚]]殖民的意图，但锡矿投资的增加所带来大量的中国苦力移民各自为主，以自己的籍贯创立不同的私会党，联合当地马来首领和私会党以暴力争取地盘以及锡矿。霹雳王朝由于长期陷于继位问题的缘故而无法维持当地的治安。

旅人''{{lang|en|Isabella Lucy Bird}}''在自己的书内谈及当时还没继位的[[苏丹阿都拉|苏丹阿都拉]]''{{lang|ms|Raja Muda Abdullah}}''如何向身在[[新加坡|新加坡]]的朋友，[[陈笃生|陈笃生]]之长子[[陈金钟|陈金钟]]求助。[[陈金钟|陈金钟]]和新加坡一名英国商人写了一封有[[苏丹阿都拉|苏丹阿都拉]]亲笔签名的信函予克拉克爵士（''{{lang|en|Sir Andrew Clarke}}''），表达[[苏丹阿都拉|阿都拉]]希望将霹雳王国置于英国保护的意愿，以便“有能之士可以向他展示良好的行政体制”。1874年，身为[[海峡殖民地|海峡殖民地]][[总督|总督]]的克拉克在[[邦咯岛|邦咯岛]]召开会议。[[苏丹阿都拉|苏丹阿都拉]]在会议上成功继承王位。[[邦咯条约|邦咯条约]]也使[[苏丹阿都拉|苏丹阿都拉]]把除了宗教和文化事务外的所有政治要事交予一名{{le|参政司|Resident (title)|英国参政司}}。1875年，当地多名酋长刺杀了第一任参政司{{le|毕治|James W.W. Birch}}，引发了1876年短暂的[[霹雳战争|霹雳战争]]。[[苏丹阿都拉|苏丹阿都拉]]被放逐到[[塞舌尔|塞舌尔]]。新任参政司[[休·洛|休·洛]]熟悉马来文化传统，也是一名比较有才能的行政官。他也把[[橡胶树|橡胶树]]引进了[[马来亚|马来亚]]。

1896年，霹雳连同[[雪兰莪|雪兰莪]]、[[森美兰|森美兰]]和[[彭亨|彭亨]]成立了[[马来联邦|马来联邦]]，但英国领事系统却一直沿用至1948年[[马来亚联合邦|马来亚联合邦]]的成立为止。

1957年8月31日，霹雳州和[[马来亚联合邦|马来亚联合邦]]其他10个州属从英国政府争取了独立。英属[[沙巴|沙巴]]、[[砂拉越|砂拉越]]及[[新加坡|新加坡]]在1963年9月16日加盟[[马来亚联合邦|马来亚联合邦]]成立了[[马来西亚|马来西亚]]。1965年[[新加坡|新加坡]]从马来西亚独立。

== 行政机关 ==
根据霹雳州宪法，霹雳州实行君主立宪制度，以苏丹为州元首。现任苏丹为苏丹纳茲林慕依祖丁沙（{{lang|ms|Sultan Nazrin Muizuddin Shah ibni Almarhum Sultan Azlan Muhibuddin Shah Al-Magfullah}}）。

=== 2008年霹雳变天事件 ===
在2008年马来西亚全国大选之中，在野党联盟——[[人民联盟|人民联盟]]赢得了霹雳州的执政权。虽然[[人民联盟|人民联盟]]的成员党之一[[民主行动党|民主行动党]]获得了最多议席，霹雳州州务大臣却由前[[泛马回教党|伊斯兰党]]议员[[尼查|尼查]]（{{lang|ms|Mohammad Nizar Jamaluddin}}）出任。这是因为霹雳州宪法规定除非苏丹豁免，否则州务大臣必须是一名馬來人或回教徒。由于以华裔为主的[[民主行动党|民主行动党]]并没有任何信奉伊斯兰教的议员，州务大臣则得从[[人民联盟|人民联盟]]的其他成员党——[[人民公正党|人民公正党]]和[[泛马回教党|泛马回教党]]之中选出。民选的[[人民联盟|人民联盟]]州政府已经在2009年2月4日由時任首相[[纳吉阿都拉萨|纳吉阿都拉萨]]領導的國陣接受4名民联州议员退党跳槽转为支持国阵后，使国阵以28比27議席再次執政霹靂。

== 人口统计 ==
霹雳州现有人口大约2百万。原为全马人口最多的州属，锡产业下滑所导致的经济问题仍然存在。经济问题引致了人口流失到附近其他经济增长率较高州属，其中包括[[槟城|槟城]]、[[雪兰莪州|雪兰莪州]]和[[吉隆坡|吉隆坡联邦直辖区]]。2001年估计种族分布为：巫裔占53.68%（1,101,105人）、华裔占31.35%（643,129人）、印裔占12.78%（262,121人）、其他占0.32%（6,536人）以及非公民占1.87%（38,345人），其中華裔人口比例是馬來西亞一級地方行政區的第二名。

== 地理 ==
霹雳州涵盖了21,006平方公里的范围，占全马土地面积的6.4%。霹雳州也是马来半岛第二大的州属，为全马第四。

霹雳州白天时阳光普照，夜晚全年凉快，傍晚偶尔下雨。州内气温纪录最低摄氏23度，最高摄氏33度。全年雨量为3,218毫米。

== 经济 ==
霹雳州经济在于[[太平_(霹雳)|太平]]拉律发现锡产后便开始蓬勃增长。

在全球经济衰退之前，霹雳州乃马来西亚最富有的州属之一。1990年锡产业倒闭后，霹雳经济也严重地受到影响。锡价的下滑使锡矿公司一一倒闭。

这促使了霹雳州政府把州内的经济蓝图转向更高价值的生产业。20世纪80年代中叶，许多台湾的电子公司都被吸引到[[斯里并|斯里并]]（''{{lang|ms|Silibin}}''）和[[九洞|九洞]]（''{{lang|ms|Jelapang}}''）工业区。但到了90年代，大部分工厂都迁到了人均收入较低的中国。马来西亚汽车产商[[普腾|普腾]]也于丹戎马林设立了汽车工业城（''{{lang|en|Proton City}}''），成为了全马汽车产量最高的工厂。

农业也是霹雳州其中一个重要的工业，尤其是橡胶、椰子和油棕。人们对于霹雳州的天然景观和文化特色的发现也使旅游业增长快速。

州内完好的基本设施也令霹雳州成为创业的首选之地。

== 交通 ==
=== 公路 ===
怡保距离北部的槟城约165公里，距离南部的吉隆坡则约205公里（或两小时车程）。霹雳州内及通往外州的道路四通八达，南北高速大道则贯穿着州内各大小城镇，南北大道從怡保南部收費站到怡保北部收費站這段14.7公里路段为免付費路段。州内也有一大道，称怡保-红土坎大道，始于怡保九洞，终于曼绒县的红土坎。

除此，霹雳州也拥有往返吉隆坡国际机场的川行巴士服务，埠内巴士服务在州内也相当流行。州内目前有数家巴士公司提供长途巴士往返吉隆坡和国内各州。

=== 铁路 ===
[[马来亚铁道|马来亚铁道]]（KTMB）为霹雳州的铁路经营者，并提供数趟火车客运服务通往霹雳州及外州各地。州内车站包括丹绒马林、美冷、仕林河、宋溪、打巴路、金宝、华都牙也、怡保、他昔、和丰、江沙、硝山、太平、甘文丁、峇眼色海和巴里文打，其中[[怡保站|怡保火车站]]是州内最大的车站，位于怡保市旧街场的邦里马武吉甘当华合路（Jalan Panglima Bukit Gantang Wahab），临近有美丹杰埠内巴士总站、怡保大草场和马来西亚邮政公司怡保总行。怡保的火车站也被称为“怡保的泰姬陵”，建于1935年，是根据摩尔人的建筑风格而建，并于2007年双轨铁道工程展开期间进行维修、翻新和升级。

怡保至森州芙蓉的主干线为复线铁道，并可以容许电动火车行驶。而于2013年完工的怡保－巴东勿刹复线电气化铁道工程使得现今州内火车站提供平均时速为140公里的[[KTM电动列车服务|KTM电动列车服务]]。

===航空===
州內的机場为[[怡保國際機場|怡保國際機場]]及[[邦咯機場|邦咯機場]]。

== 旅游业 ==
{{Main|马来西亚商场列表#霹雳}}
霹雳州内有数个名胜地点，其中包括其首府怡保，以当地的[[沙河粉|沙河粉]]和[[海南鸡饭|海南鸡饭]]为其中简单却又具特色的佳肴。怡保市郊[[打扪|打扪]]（''{{lang|ms|Tambun}}''）盛产的柚子也吸引了本地其他州属的旅客慕名而来。

位于华都牙也的[[凯利古堡|凯利古堡]]（''{{lang|en|Kellie's Castle}}''）在1915年动工后由于主人史密夫（''{{lang|en|William Kellie Smith}}''）返回英国后在当地去世而没有竣工。许多人相信古堡里闹鬼，拥有密室和密道。今天，凯利古堡式当地一个著名的旅游胜地。

从红土坎乘船到[[邦咯岛|邦咯岛]]后，旅客可以尽览当地渔村和白色沙滩。游泳和潜水都是旅客到邦咯岛少不了的休闲活动。

== 食物 ==
''{{lang|ms|Lemang}}''是开斋节和哈芝节少不了的传统马来食物。它是把糯米在竹筒里慢火熬成，配合马来咖喱''{{lang|ms|Rendang}}''食用。据说这食物来源于当地土著以竹筒做饭而来。

''{{lang|ms|Tempoyak}}''是另一种著名马来食物。它是把榴莲肉在陶器里保存起来而成。

霹雳州的食物被公认为本区最好的，通常外地的旅客会特地到本州的餐馆或美食中心，大快朵颐一番。在怡保市不妨尝尝当地美食，如沙河粉、咚咚粉、香港猪肠粉、忠记兵如港大树头炸料粉、河嘻（潮州魚丸粉）、奇峰豆腐花、香港点心、炒粿条、偏担饭、盐锔鸡、怡保高温街芽菜鸡和古月添記客家擂茶(山城怡保擁有擂茶的故鄉的美名)等。此外，还有万里望花生、美罗鸡仔饼、实兆远的福州光饼和红酒面线、邦略岛的沙爹鱼、富有马来风味的叻沙以及安顺的香饼等。
