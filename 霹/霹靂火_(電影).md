{{engname|《霹靂火》|''Thunderbolt''}}是1995年由[[陳嘉上|陳嘉上]]執導的香港電影，為[[成龍|成龍]]主演的賽車電影。

== 劇情 ==
曾在日本[[三菱汽車|三菱車場]]受訓的陳火韜，在香港經營一家車房，車房不時協助警方處理反飛車行動。在一次警隊行動裡面，一名外籍飆車手駕駛跑車衝過路障，撞死一名交通警察。國際刑警表示，該飆車手名叫Warner Krugman，又稱為Cougar，他不但是一個賽車手，亦是犯罪組織的頭目。幾天後陳火韜遇上Cougar，決設計將他擒獲。陳火韜指證Cougar是撞死交通警察的飆車手後，Cougar被關進羈留室，可是Cougar的手下將他由羈留室中救出，並捉走陳火韜的兩名胞妹，要求跟他到日本賽車......

== 幕后花絮 ==
该片投入了2亿元港币的成本，創當時香港電影新高，電影于[[日本|日本]][[仙台|仙台]]、[[东京|东京]]，[[美国|美国]][[洛杉矶|洛杉矶]]、[[纽约|纽约]]取景。
== 轶事 ==
剧中副导演原是林超贤，主演成龙和林超贤都是拍动作片出身，在一些动作上都很有自己的想法，经常和导演陈嘉上出现分歧，之后两人的动作导演被换掉，由成龙和洪金宝顶替。

电影总算拍摄完毕后，作为女主角的袁咏仪需要补拍台词，后因袁咏仪沒到现场让成龙白等一整天，放话要封杀她。之后，成龙又到香港导演会投诉，其他的演员也加入了抵制的行列，使袁咏仪的处境非常尴尬。后来曾志伟做中间人让两人碰面，袁咏仪当面向成龙道歉，终于冰释前嫌。[http://www.chinapress.com.my/20190128/成龙封杀袁咏仪原因曝光/ 成龙封杀袁咏仪原因曝光]
