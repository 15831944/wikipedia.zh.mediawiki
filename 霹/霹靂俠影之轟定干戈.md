《'''霹靂俠影之轟定干戈'''》（{{lang-nan-poj|Phek-le̍k Kiap-iáⁿ chi Hōng-tēng-kan-ko}}）是[[台灣|台灣]][[大霹靂文創整合行銷股份有限公司|大霹靂文創整合行銷股份有限公司]]發行[[霹靂布袋戲|霹靂布袋戲]]於2013年6月7日發行，霹靂國際多媒體於2013年的開春鉅獻的新劇集。該劇集延續上一部《[[霹靂俠影之轟動武林|霹靂俠影之轟動武林]]》之劇情。

== 主要登場角色 ==

; 丹華抱一·[[鷇音子|鷇音子]]
: 身揹天榜，手持時計，居於羅浮山的神秘奇人，擅煉玄丹，偏向道家色彩。真實身分為素還真的化身之一，在無夢生前往相借非關時間的考驗中，途中歷經殊離山魔考一時失魂出神之下，使素還真第三魂體化為飛鷹離去，剛好在聖魔元史正在尋找宿主之時意外發現此飛鷹，而追上並點中宿主，並向時間樹緹命借時而出現的第三段時間化身。
: 未現身前以未來的形影解救了地獄變，一登場便影響了無夢生，雖表明立場中立，但卻以烽火天榜揭開未來武林最具影響力的風雲人物十二人，目的是為了攪亂武林。而三餘無夢生(或者說是素還真)是最大的阻礙，所以聖魔元史的目標才會瞄準了素還真，打算以素制素。
: 因是意外而締下契約的化身，不能與同為契約下的無夢生主動相見，否則就會像王見王一樣·死棋。而無夢生亦是如此。但就在引無夢生來相見時，被無夢生以搏命的方式，巧妙的利用逆時計化掉了此障礙。因此王見王，再也不會有死棋相剋之相。

; 百世經綸·[[一頁書|一頁書]]
: 與素還真並列為霹靂的兩大支柱，亦名『梵天』。高深的禪修大般若與武學修為，法相莊嚴、胸懷無私、正氣凜然，出道以來屢次幫助素還真攜手共同弭平武林狼煙、扭轉乾坤化解危機。身為佛門高僧，悲憫如一頁天書渡化迷航眾生，但行事作風果決明快、妒惡如仇，鏟奸除惡絕不手軟，為正道的精神指標及領導者之一，與眾不同的百世經綸，半闔眼眸看盡世俗庸人，微揚嘴角笑盡天下英雄。

; [[四智武童|四智武童]]
: 在三餘無夢生受到逆時計影響消失後出現的神祕孩童，踩著孔明車，唸著喻事的童謠，天真無邪，外表看似約七八歲的孩童，眼神卻流露著成熟與睿智，真實身份不言可喻。以佛家四智心品為名，延續三餘與鷇音子的鬥智競爭，一心杜絕聖魔元史之禍，不讓其影響天道的自然運行。 

; [[一字鑄骨|一字鑄骨]]
: 由鷇音子點名，秦假仙前往找尋的命中貴人，溫文且神秘，因一樁過去而失落了自己全身骨頭，故窮畢生歲月都在找尋自己的骨頭，卻苦無所獲；若有朝一日，當他將自己的骨頭收集完全時，又會發生什麼事？一字鑄骨的神祕，就連鷇音子也難以掌握。一字鑄骨最後被[[暴雨心奴|暴雨心奴]]所殺，死後所穿的鞋子離開一字鑄骨自行離去。

; 春鎖紅顏·[[步香塵|步香塵]]

; [[佛劍分說|佛劍分說]]

; [[疏樓龍宿|疏樓龍宿]]

; 快雪時晴·霽無瑕
: 風姿颯爽、直接俐落的豪氣女俠客，行走武道仗義不平，天生神力無窮，手持極重的泰若山劍，曾一劍擊退巨魔神，罕逢匹敵對手。真實身份為魔佛波旬中的女琊，因遭烽火關鍵衝撞，導致波旬三體分離，神智受創錯亂，方以俠女之姿行走江湖。

; 北狗·[[最光陰|最光陰]]
: 名列五大傳奇之一「北狗」，又稱老狗，臉帶狗頭面具，創立蝕劍刀法的冷酷刀客。
: 因長時與犬獒為伴，其行為模式與常人不同，因自覺長相與狗不同，而戴著狗頭面具遮掩自己外貌，曾參與第過一次凋亡禁決，殺盡無始暗界上千妖族，僅存活三名殘存者：酆都三千里、葬雲霄、釋閻摩，但凋亡禁決結束多年來始終下落不明。
: 其真實出身來自時間城日晷集日精而成的無魄之人，受魄冠大禮，掌掠時任務，後在離開時間城修行時遇到刀神九千勝，與之交手未落敗，故而結交莫逆，九千勝並贈與一隻雪獒，埋下日後在金獅帝國收養小蜜桃之契機。後九千勝帶其參加琅華宴，向召集人文熙載推薦其為「元字第座」，埋下與烈劍宗少主暴雨心奴不滿，於是凌辱殺死文熙載之女，陷害最光陰，再以地獄十八陣殺死九千勝。最光陰為替好友九千勝求得活命之機回到時間城，捨棄魄冠自願降級，將其心轉給九千勝，依靠著逆時計而活(綺羅生有雙心由來)，並與時間城訂下契約，顧守時間樹。依逆時計作用，每到他二十九歲的生命終數時就會回朔十年，而這回朔的十年間一切記憶都會消失，就在漫長的時間消磨下，讓其遺忘了初衷目的，也逐漸產生厭倦之心。於是他決定去做一些不一樣的事而離開時間城，自己丟棄逆時計並且開始行走江湖，因接受金獅窟受害者的請託，才因此參加了凋亡禁決。
: 常年伴陪身邊的小蜜桃於一次戰役中走失，老狗就此銷聲匿跡尋找其蹤，後因熟悉之感而找上綺羅生。

; 聖嬰主·[[焱無上|焱無上]]
: 怪樂地的真正領導者，自稱本爺，天生擁有意識燄能，其性狂傲好戰，能運使古傳六訣，被傳為妖界創者「鬥神血泣」之轉世。昔日巨魔神一役後，又意外因「逆．時計」的影響而逐漸退化成嬰兒之身，怪樂地一夕失去焱無上、妖天師兩大高手，所幸怪樂地依靠妖界的平衡而未被併吞，由妖剎剎接續領導，並詐稱嬰兒為妖剎剎之子。
: 後怪樂地被滅於鬼荒地獄變後，化為嬰兒的焱無上流轉至步香塵、娛笑顛之手，後為擺脫嬰兒之身，答應無夢生條件，以意識之刃殺除天之厲意識，讓意琦行擺脫天之厲的影響，不料焱無上運功將「逆．時計」回歸到無夢生身上，趁機欲一併除去意琦行意識(為綺羅生所救)。接續聯合封世末、獄天玄皇殲滅欲界第三天勢力，並在妖界三雄爭霸中取得勝利，其個性看起來散漫不拘小節，卻十分清楚目前妖界狀況，唯有解放妖界深層增加妖界實力，故在交戰當下詢問末世、獄天玄皇誰願意犧牲自我，就在獄天玄皇自願時，轉而將封世末殺死，其妖能皆回歸於獄天玄皇之身而回復完整的妖能，並將封末世身上所取出的初末雙心賜給釋閻摩提升妖能，其手段突顯聖嬰主對封世末所想，認為妖界當以團結為首要。

; 亡夜·[[釋閻摩|釋閻摩]]
: 外號亡靈之夜，無始暗界封印一族的鬼才，曾協助將封印荒初禁赦的異空間分割為三(魍道、紅蓮口、無盡空淵)，與葬雲霄、酆都三千里為參加凋亡禁決共患難的戰友，據痕千古所言釋閻摩享受被追逐的快意，卻插手救了同族。在當初的凋亡禁決結束後，穿梭於魍道、紅蓮口、無盡空淵，在不同的妖界空間流浪，自稱妖界空間穿行者，亦以妖界浮萍自居。後得到聖嬰主的賞賜獲得初末雙心（封末世之心與荒初禁赦之心）進而提升妖能，並負責打通妖界三處地脈任務。

;鬼荒·[[地獄變|地獄變]]
: 不夜羽在死後體內的彪族血統完全甦醒，為鬼王之死復仇而重生。個性冷豔詭異，性情兩極，觸怒動武時，如變身魔鬼夜叉般，狂野狠絕。地獄變以父之名承襲鬼王之力，乃惡鬼三凶原主，關乎未來武林局勢影響。
: 在三凶匯集後實力大幅提升，與武道七修兩人對戰時輕取一留衣之命，之後更擊殺戰雲界[[御宇天驕|御宇天驕]]，成為黑獄最強大的戰力之一，但因故與妖皇墮神闕產生芥蒂，不久選擇了聯合六首雲蛟叛變奪權。後接到三才神器持有者的挑戰，一人獨對意琦行、綺羅生與風雪一路禪等三位高手，最後戰敗落海，離體的鬼手與鬼瞳被旁觀的佛鄉與無夢生奪走。
: 受鷇音子所救而免於一死，在經歷生死之後開重新審思自己的生命意義，接受了自己做為不夜羽的過去，重回妖界。在獄天玄皇與無夢生的協助下重聚三凶，誓為妖界的未來而戰。

; 獨孤毒
: 荼山毒脈的領導者，在毒界地位崇高，每隔數年便會在荼山召開耽毒無極宴，個性陰沉狠辣，與西幽歐陽堇曾因毒宴而結為莫逆，故而欣賞其女欹月寒的膽魄，收其為義女、助其蛻變為傅月影，展開復仇，成立毒盟，以報復西疆覆滅主要目的。
:與金獅帝國護衛姜回為夫妻，姜回送一枚金獅幣為定情物，不料金獅帝國為大王儲步武東皇(烈顏不破)疑心父王偏心，派人覆滅金獅帝國，姜回也不幸死在痕千古之手，臨死之前委託北狗最光陰一封家書與金獅幣，北狗卻延遲送交家書五十年，讓獨孤毒做了多年期盼相聚的美夢，並從北狗之口得知「唇間留一痕」為殺人兇手及姜回遺願為銷毀金獅幣。最後在暮成雪請託下，提供「一眼留魂」此項寶物，此舉不僅除去欹月寒仇人暮成雪，也迫使步武東皇為保住性命犧牲痕千古，報了殺夫之恨。
:後協助老狗最光陰取金獅幣，保護女飛賊廉莊不受欲界逼殺，但毒盟卻在仇恨與江湖無情下，人才逐漸凋亡，使獨孤毒看破一切，將毒盟交付傅月影，許其影后之位，便選擇退隱。

; 說悄悄·[[傅月影|傅月影]]
: 獨孤毒義女，對於毒后以外的人皆展現柔媚小女人之姿，看似溫婉可人，卻善用自身艷傾天下之容來行使婦人之毒，無情毒辣，對於葬雲霄與幽夢樓一切人事物深惡痛絕。其真實身分是飽受打擊而吃下毒婦之心，產生劇烈變化後的欹月寒。
:在滅族仇人暮成雪、情人葬雲霄接連死去，兼之獨孤毒看破一切，將毒盟交付傅月影，許其影后，傅月影將目標放在其父顛不亂，在其身上得到些許父愛，並暗中推力讓其父顛不亂自此退隱月影軒，傅月影了卻此事後，便把目標放在步香塵，與之周旋。

; 年華不染·[[裳瓔珞|裳瓔珞]]

; 品禪妙足·[[譴彌勒|譴彌勒]]

; 闋聲雲舵
: 釋閻摩於找尋妖脈途中結識的佛鄉修行者，嘴上有一道受戒疤痕，於佛鄉藏脈深處現身，擅長聆聽心靈投射於萬物所呈現的音律。在欲界以人質要脅招降佛鄉時，甘願承擔罪業，盡斷人質生機，好讓裳瓔珞無後顧之憂與欲界全面一戰。

; 魔佛·[[波旬|波旬]]

; 塵外孤標·[[意琦行|意琦行]]
: 人稱絕代劍宿，乃武道七修第一人，其真實身分乃戰雲界三大驕傲之一「絕代天驕」，其性情冷傲脫俗，超然獨處，思想行為不同凡俗的絕代劍宿，在七修中有著「武始通修」之美譽，亦狂亦俠亦超塵。近日與同修好友太羽驚鴻·[[一留衣|一留衣]]， 好友. 白衣沽酒·[[綺羅生|綺羅生]] 挑戰鬼荒·[[地獄變|地獄變]]，最後慘遭敗戰，好友一留衣遭重傷死亡，意琦行決定回歸戰雲界，重拾「絕代天驕」身份。

; 冰王·玄冥氏
: 冰樓之主，專長打造神兵，尤擅神弓，與朝天驕、古陵逝煙、杜舞雩並列四大奇觀領導者，儒雅明理，作風穩健，與檯面上的霸主有所區隔。自陰元回歸後，便積極調查當年質子回歸的血案真相，並與戰雲界鳳座結盟，協助調查巨魔神被盜一案。

; 霜旒玥珂
: 冰樓公主，昔時冰樓遭封，自己也陷入沉眠狀態，因身為王脈一員，致使冰封過深，無法自行解封，後因疏樓龍宿之助而重見天日。習有冰樓絕技「雕霜沐雪奪形訣」，能為佛劍分說塑造重生之軀。性情嬌縱，興趣是收集鑑賞天下奇珍，也因此為龍宿的華麗風采所傾慕。 

; 鏤冰氏
: 冰樓副樓主，疏樓龍宿破解冰封後，進入冰樓所見的第一人，負責協助總理冰樓內務，謹慎細微，在得知龍宿目的後，領其破除被冰封的公主霜旒玥珂，得以重塑佛劍分說生機。

'''一劍風徽.[[杜舞雩|杜舞雩]]'''
:四奇觀之中的風，遺世獨立於馭風島，雖鮮少涉世，卻被鷇音子列名烽火天榜中。器宇軒昂，眉宇間難掩孤傲凌雲之氣，劍術超卓，是煙都大宗師最為忌憚之人，但面對四奇觀之間的紛爭，一劍風徽卻遲遲未出手介入。直到暴雨心奴將霜旒玥珂虐殺並把其屍身寄來，至冰樓將其屍安葬後，允許佛劍分說讓受重傷的焱無上安置在此，並決心不再退讓，殺至煙都向放走暴雨心奴的古陵逝煙興師問罪。

; 大宗師·古陵逝煙
: 煙都眾人崇仰的大宗師，高深莫測，武學造詣有「一式留神」之美稱。性格冷言疏淡，看似退居幕後，卻凡事盡在其眼中，運籌帷幄掌控煙都行事方針，心機深沉難以臆測，干係著冰樓血案與巨魔神失落之謎團。 

; 宮無后 
: 古陵逝煙特意栽培、賜宮字的用劍奇才，天生右眼尾有一滴血淚為誌，由於在五歲受宮刑，導致性格扭曲病態而殘酷，常以散髮披臉、紅羅大袍為裝扮，妖孽冶艷，雌雄莫辨，有如東洋豔鬼般。出任務時，才會戴上大紅宦官帽，殺人無情，百里冰泓、朝天驕在其劍下走不過一招便亡，足見其劍法之高。與煙都大宗師之間，呈現微妙而對立的師徒關係。

; 西宮弔影
: 當前煙都主事，亦為大宗師古陵逝煙座下大弟子。在朝天驕與焱無上陷入酣戰膠著時，奉命出手馳援。其性冷沉且具城府，言談中偶露銳氣，令人難以親近。有嚴重潔癖，慣以隨身手帕擦拭雙手。

; 澹台無竹
: 自稱一式留神的創招者，昔日古陵逝煙以武會友，以地劍祕笈與其交換。在葉小釵打敗宮無后後，攔路一較短長，甚至提點暴雨心奴關於杜舞雩的來歷。

; [[暴雨心奴|暴雨心奴]]
: 當初設計十八地獄陣一舉圍殺九千勝與最光陰的邪惡書生，個性陰狠狡詐，因為得不到，所以想一舉毀滅，殺人手段十分殘虐。
: 習有祆撒教特殊邪法，自許為祆撒大護法，認為自己是天使的眼淚所化，每當下雨時，他就會認為是上天在為他哭泣，哭的是自己還做得不夠惡！


=== 天葬十三刀 ===
: 組織內成員非為刀客之流，與葬刀會專門埋葬刀者的組織有所不同，其存在的目的為找出名曰「天葬十三刀」地名，後因聚集有志一同的人，故轉變成一個組織的稱呼。如果成員有意退出組織，需由組織內成員共同表達同意，才可以離開，一旦內部成員有人死亡，則由召集人黃羽客尋找適合人選遞補。另外天葬十三刀與葬刀會創辦者步武東皇有相當關係，步武東皇故能調遣痕千古、別黃昏兩人。

: 天葬十三刀由黃羽客出面召集各方能人異士、武功高手，目前所知奇技者為鬼盜(盜賊之最，盜術臻智能竊人靈魂)、喫神(佳餚能始食用者如臨仙境)、畫聖上官圓缺，武功代表者有痕千古、別黃昏。

; 塔鈴獨語·[[別黃昏|別黃昏]]
: 天葬十三刀之一，過去東皇為其子賦兒奔走治病，雖未能見效，別黃昏經歷喪子之痛後，為報答戚太祖恩情而聽命於他，並結識痕千古。曾參與屠殺金獅帝國的鷹戰行動，將百里冰泓打入懸崖。
: 個性冷沈孤僻，不太與人接近，少見情緒波動。習慣佇立於黃沙荒塔之下，專注仰望塔尖上的銅鈴，聆聽隨風迴響。殺人時，會先出現鈴聲迴盪，然後把對方拉入荒塔剪影幻境中擊殺。
: 直到偶然救了受傷的[[宮無后|宮無后]]，發現與其子賦兒有著相同罕見的血淚之眼後，對其身份產生了懷疑，進而調查宮無后的身世之謎。

; [[黃羽客|黃羽客]]
: 神秘的黃裳客，為天葬十三刀的召集人，專門在武林上尋找超強武者，充實天葬十三刀的陣容，因痕千古之死而找上北狗，以三招試探北狗實力，極為讚賞，積極邀其加入天葬十三刀。為烈劍宗門人。

; 鬼盜·[[渾千手|渾千手]]
: 為連莊之父[[廉知讓|廉知讓]]，因魔師太手上布料遭北狗看穿，所以和他有了交情。後奉命偷取傅月影之魂，遭元始天宰發現，遣迷達偷取魂魄。

; 六筆丹青·[[上官圓缺|上官圓缺]]
: 行走江湖的神秘畫家，背後揹著一個如扇貝的畫盤，插著六口「傳神六筆」，其中夢筆能畫出人心中所想之念，曾留畫贈予別黃昏。
: 在拜訪煙都之時，正逢煙都為大宗師[[古陵逝煙|古陵逝煙]]之死弔喪，為悼念大宗師而使用夢筆為大宗師作畫，畫出的卻不是[[古陵逝煙|古陵逝煙]]之相貌，為大宗師之死添上了疑雲。
