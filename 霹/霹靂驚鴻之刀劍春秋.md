《'''霹靂驚鴻之刀劍春秋'''》（{{lang-nan-poj|Phek-le̍k Keng-hông chi To-kiàm Chhun-chhiu}}）是[[台灣|台灣]][[大霹靂國際整合行銷股份有限公司|大霹靂國際整合行銷股份有限公司]]發行[[霹靂布袋戲|霹靂布袋戲]]於2012年10月12日發行！霹靂國際多媒體股份有限公司二○一二年新產品。該劇集延續上一部[[霹靂戰元史之動機風雲|霹靂戰元史之動機風雲]]之劇情。

== 詩序 ==

刀鋒橫空斬迷霧，劍影縹緲絕塵出。春來冬去又幾數，秋畫紅艷褪殘污。

== 劇集概述 ==
止戰之印蓋下，聖魔佛厲的種種，遭到全盤改寫，鴻濛隔斷苦境與中陰界的連結，[[鬼覺神知|鬼覺神知]]蜕變而成的玄玄[[血傀師|血傀師]]，掌握缺漏的記憶，開始對五劍、八厲等人進行反撲，在層層佈計的過程中，五劍之氣先後落入血傀師掌握，而在鴻濛解除後，懺罪之牆因[[天之佛|天之佛]]告懺而崩毀，紅潮正式為禍苦境，天之佛更因昔日抉擇與造業，以天罰自裁石封為蓮，至此，血傀師的陰謀再開新章。
武林四驚鴻喧囂未止，惡鬼三凶相繼現世，明月不夜羽亦化身地獄變再臨；策夢侯處心積慮設謀，八品神通於焉在握；叫喚淵藪昔戰延續，內發、外聚七修紛擾再起，促使意琦行正式踏入紅塵；綺羅生背負江山快手諸般情仇，與無故事的人、雲滄海的糾葛，再掀往昔波濤。天之厲強勢再出，蟄伏已久的魔皇，亦開始有所動作，在聯合背負中陰界之仇的宙王等人請戰天佛原鄉當口，修補佛元完成的一頁書再現塵寰，亂世板蕩，面對群雄詭謀環伺，一頁書要如何應對？握有三機讖的素還真，又會以何種姿態介入武林？烽火鑑兵臺召開名器觀論，血傀師以五始之氣合成名劍春秋，與黑月之淚、忘巧雲戟並列會上，吸引各方豪傑，神秘的新面孔亦相繼現世，此時葬刀會亦挾雄姿現蹤武林，一場由武道、兵器所交織而成的江湖恩怨，在有心人的操弄之下，將使局勢再次暗潮洶湧。

==主要登場角色 ==
;[[一頁書|一頁書]]
:中原武林巨擘，佛法修為精深，行事作風果決明快、妒惡如仇，鏟奸除惡絕不手軟，為正道的精神指標。曾受邪術影響而呈入魔狂態，後歷經百般試煉，終入天佛原鄉修補佛元完成，掃蕩魔氛、再渡紅塵。

;[[意琦行|意琦行]]
:武道七修第一人，以絕代剑宿之姿現身通天道，性情冷傲脫俗，超然獨処，不喜與世接物，思想特立獨行，不隨波逐流，在七修中有著「武始通修」之美譽，亦狂亦俠亦超塵。與一留衣曾在叫喚淵藪之淵頂一戰外聚七修，該役所使之紅爐点雪劍招，馀勁歷數百年至今未歇，乃意琦行在淵頂所留之警招。

;[[綺羅生|綺羅生]]
:內發七修中習練刀法的武者，在刀藝習成之後即離開叫喚淵藪，挑戰天下，名號「江山快手」，在因緣之下習得綺羅獸花術，名列奇花八部之一。綺羅獸花術乃以琉璃長針穿心血為線，刺牡丹以豔身。

;[[策夢侯|策夢侯]]
:二十四夢花境之主，奇花八部的幕後推手，談吐溫文儒雅，慵懶華貴，善書風花雪月之事，頗有魏晉文人雅士遺風。心性謹慎細膩，行事看似小心，不為流言所動，實則城府深沉，有其不為人知的一面。

;[[天之厲|天之厲]]
:厲族王者，沉穩霸氣的不世梟雄，能謀善斷，身負毀天滅地的闇能，以殺伐為創世之途，唯有五劍能誅之。在聖魔止戰之後，藏身於蜉蝣谷，直到日後諸厲元魄回歸，以「帝禍邪九世」之姿，再掀苦境風暴。

;[[魔皇|魔皇]]
:受佛元與厲元催化而出的純魔之體，創建天閻魔城闡提一脈，在初代聖魔大戰時身亡，後於緞君衡策劃下，聚佛厲魔三元而復活。隨著緞君衡的來訪鬼闕，一代魔皇，終於開啟了再涉世局的全新契機。

;[[天踦爵|天踦爵]]
:齊煙九點天踦爵，手持玉晶杖，懷揣流金時計，一派西洋爵士作風的神秘人物，其身份來歷成謎，個性幽默風趣，對瞬息萬變的武林事，有相當的掌握程度。對血傀師背後的力量，似乎也有幾分了解。

;[[北海無冰|北海無冰]]
:頹廢沉鬱、孤寂寡言，深藏許多故事的浪人劍客，以無故事的人自稱，實乃為遺忘、否認自己的過往，避居斷碑林中，與鷹女、篝火為伴。為醫治明月不夜羽之怪症，遍尋天下奇珍異寶。

;[[矩業烽曇|矩業烽曇]]
:天佛原鄉審座，享譽佛界的憤怒尊，握有審核審判等實權，個性剛烈果斷、執著堅定，對佛法修為愈高者，會採取愈嚴苛的審視準則，執法極少留情。代表意義為斬惡武神，有「烽火曇華」之稱。

== 登場角色 ==

;[[素還真|素還真]]
:目前暫時消失！握有三機讖，又會以何種姿態介入武林？

;[[血傀師|血傀師]]
:當鬼覺神知蓋下止戰之鑰，時空靜止之際所出現的異人，身揹奇特的乾尸，通曉武林許多不為人知的秘辛。一步黑洞無盡期，身份弔詭，撲朔迷離，利用失落的十二時辰，意圖重建武林新秩序，主宰佛厲大戰後的掌控權。

;天官賜福
:頭冠文武盔，手持玉卷旨，腳踏金舄履，宛若天官降世的神秘人物，出現於屈世途夢中示警。紛紛擾擾何時停。暗中關注著血傀師的舉動，更於公開亭揭破聖魔之僕的身分以及陰謀詭計。

;明月不夜羽
:纖弱文質，生性仁慈善良，但因體質特殊而感到些許自卑，一心渴望醫治怪症當個平凡人。真正身份乃擊風族公主，母親鷹后懷胎時莫名中箭重傷而誕生，因此流落江湖為無故事的人所救。聖魔失憶戲迷嘆，除了鷹族血統之外，體內似乎尚有一詭異的血緣。死後身軀產生變化，以「鬼荒地獄變」之姿態重現武林，欲奪回惡鬼三凶。

;惡骨
:幼時遭到拋棄的不良少女，自組惡童黨，帶著一群小弟四處打劫維生，體驗社會現實與人情冷暖，造就強烈的求生意志，以及惡根深植、自私狠毒的性格，卻也因此受血傀師的賞識收其為徒，更要其爬上叫喚淵藪取鬼手，脫胎換骨。現已取得鬼手，由其父親[[掌懸命|掌懸命]]將鬼手接上，承接巨大的力量。變相漲價誰家贏。後受到[[綺羅生|綺羅生]]開導，逐漸控制鬼手之力，卻也因[[策夢侯|策夢侯]]的出現令惡骨十分悲傷，逃離畫舫。

;[[劍布衣|劍布衣]]
:十擘雲集中最神秘低調的一人，憂患深口中身具扭轉乾坤之能的奇人。性格詼諧悠然，平時隱居於秋鳴山居，雖不問世事，但交友廣闊。人如其劍，在溫潤如玉的外表下，藏有極為銳利極端的一面。習有地水火風四大劍訣。其真實身分為未來八厲之徒，因為來聖王病危，劍布衣被派至過去，改變一切歷史。

;[[冰無漪|冰無漪]]
:元種八厲中的「水之厲」，不喜權勢、鬥爭，只愛美人不愛江山，帶有反派英雄特質的詼諧高手，乃厲族中的異數。與雷之厲較為友好，對地之厲行蹤之謎則心有芥蒂。其獨特的水銀功體能穿越無盡天峰。 曾被[[帝禍邪九世|帝禍邪九世]]吸收厲元，因體內神秘佛力保護，使冰無漪並未被完全吸收，但也成為活死人。爾後[[劍布衣|劍布衣]]被擒回佛鄉，卻也盜取九九轉輪器，冰無漪終得恢復。

;[[月藏鋒|月藏鋒]]
:中陰界三大劍客，瀟灑重諾的文儒劍客，因文才顯赫而受宙王器重，但多年前卻因一道私人花約觸怒聖顏，被貶至藏頭塚，自設藏頭詩謎困陷其中，直至素還真與歡奭解出十年一覺藏頭詩，方重獲自由。無奈花期已過，徒留憾事…。來到苦境後認識了[[劍布衣|劍布衣]]與[[冰無漪|冰無漪]]，三人成為好友，從此互相幫助、共患難。

;[[緝天涯|緝天涯]]
:胡同蜃市鬼掌，處事機敏有度，個性略偏冷傲。因鬼師緝氏祖規，年過二十必婚配以人，故緝天涯屆齡二十後，便設以紅鑾繡球招親。但其父鄙其先祖陋習，不願女兒受成規束縛，便假外人道，蜃市之主醜陋無比，其性兇惡如夜叉。故緝天涯招親歷年有餘，無人敢接繡球以結好。在父親為解鴻濛犧牲後，來到苦境尋找母親，爾後與母親[[九代師|九代師]]相認時，被'''葬刀會'''人士偷襲，幸虧有[[天踦爵|天踦爵]]的相助，使兩人擺脫追殺。後與母親欲往中陰界，再度遭受'''葬刀會'''逼殺。

;[[雲滄海|雲滄海]]
:一頁書於天佛原鄉結識的佛修者，受「佛鍊鑄天術」之手法重塑武骨，佛緣深植，為人孤僻清寡，為調查懺罪之牆血案而離開初禪修界，正式涉足武林。其身披「雲裟」，乃雲滄海專屬的披風狀法器。
