'''欣藍'''（英語：Yan Lam）是香港[[東區區議會|東區區議會]]下轄的[[選區|選區]]，代號'''C10'''，成立於2004年，現任區議員為[[香港島各界聯合會|香港島各界聯合會]]成員[[黃健興|黃健興]]。

==範圍==
選區範圍為[[富欣花園|富欣花園]]、[[藍灣半島|藍灣半島]]，與其相鄰的選區有[[翠灣_(選區)|翠灣]]、[[小西灣_(選區)|小西灣]]和[[景怡|景怡]]。

==沿革==
景怡選區首次出現於[[2003年香港區議會選舉|2003年香港區議會選舉]]，因為[[藍灣半島|藍灣半島]]入伙，所以在[[欣怡|欣怡]]選區中分拆出[[富欣花園|富欣花園]]，跟藍灣半島合併成'''欣藍'''，而[[富怡花園|富怡花園]]部份就與[[富景_(選區)|富景]]合併，改稱[[景怡|景怡]]。

[[2003年區議會選舉|2003年區議會選舉]]，由前[[欣怡|欣怡]]區議員[[梁雪冰|梁雪冰]]男友[[曾航生|曾航生]]、[[無黨派|無黨派]]的社工麥偉智（即後來的阿仙奴足球學校總監）與[[民主黨_(香港)|民主黨]][[朱偉祖|朱偉祖]]爭奪，最後朱以1,509票擊敗取得996票的麥偉智和448票的曾航生而勝出。

[[2007年區議會選舉|2007年區議會選舉]]，當時人口估算約15,561人<ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/summaryc.html |title=存档副本 |accessdate=2011-11-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20080501203936/http://www.elections.gov.hk/dc2007/chi/summaryc.html |archivedate=2008-05-01 }}</ref>，其中有7,976位是選民<ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/num_electors_eastern.html |title=存档副本 |accessdate=2015-12-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20150621034533/http://www.elections.gov.hk/dc2007/chi/num_electors_eastern.html |archivedate=2015-06-21 }}</ref>，朱偉祖今次與[[范徐麗泰|范徐麗泰]]（當時仍為立法會議員）的東區義工團團長[[黃健興|黃健興]]爭奪，黃借此吸收[[藍灣半島|藍灣半島]]的中產票源，最後黃健興以2,489票勝出，朱偉祖和鄰區的陳耀德皆連任失敗，令民主黨於柴灣區的區議員只剩下[[黎志強|黎志強]]。

[[2011年區議會選舉|2011年區議會選舉]]，當時人口估算約16,853人<ref>[http://www.elections.gov.hk/dc2011/chi/summaryc.html]</ref>，其中有8,674位是選民<ref>[http://www.elections.gov.hk/dc2011/chi/ebriefsc.html]</ref>。今次黃健興與[[民主黨_(香港)|民主黨]]梁俊健爭奪，最後黃以3,620票大勝。

[[2015年區議會選舉|2015年區議會選舉]]，當時人口估算約16,981人<ref>[http://www.elections.gov.hk/dc2015/chi/summaryc.html]</ref>，其中有9,269位是選民<ref>[http://www.elections.gov.hk/dc2015/chi/ebriefsc.html]</ref>。今次黃健興自動當選。
