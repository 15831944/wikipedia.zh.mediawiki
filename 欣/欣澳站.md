'''欣澳站'''（{{lang-en|'''Sunny Bay Station'''}}）是[[港鐵|港鐵]][[東涌綫|東涌綫]]及[[迪士尼綫|迪士尼綫]]共用的[[換乘站|轉車站]]，位於[[香港|香港]][[新界|新界]][[大嶼山|大嶼山]]東北部的[[陰澳|陰澳]][[打水灣|打水灣]]填海區，佔地9,500平方米，於2005年6月1日啟用，比較[[香港迪士尼樂園度假區|香港迪士尼樂園度假區]]提早近兩個月啟用。雖然欣澳站座落大嶼山，但在[[香港十八區|香港十八區]]區域劃分上欣澳站所在地為[[荃灣區|荃灣區]]，而不是[[離島區|離島區]]<ref name="district">{{Cite web |url=http://www.eac.gov.hk/pdf/legco/2016lc/2016boundaries/2016lcentw.pdf |title=2016年立法會選舉地方選區分界圖-新界西}}</ref>。

欣澳站自接駁至[[愉景灣|愉景灣]]專線巴士開通後，成為連接愉景灣的主要轉接點，取代[[東涌站_(東涌綫)|東涌站]]為轉接點之地位，縮減愉景灣居民轉乘港鐵前往香港各地的時間，而且不需要再繞道前往東涌站<ref>[[愉景灣隧道|愉景灣隧道]]的西入口位於[[小蠔灣|小蠔灣]]，[[港鐵車廠#小蠔灣車廠|小蠔灣車廠]]對面，在這兩站之間</ref>，除非翔東路封閉。

===不設獨立大堂===
{{ multiple image | direction = vertical | align = right | width = 220
| image1 = MTR SUN (30).JPG | caption1 = 出入閘機及客務中心
}}
由於欣澳站只定位為一個[[轉車站|轉乘車站]]，基本上本站不設獨立大堂，只在2號月台劃出面積細小的非付費區域，並設置一個客務中心，一系列票務設施如售票機、增值機、八達通查閱機以及數部出入閘機；而站內也不設有任何商店，只是在1號月台與3號月台間設有的[[自動櫃員機|自動櫃員機]]，以及在月台的兩端設置[[廁所|洗手間]]，以方便前往[[香港迪士尼樂園度假區|香港迪士尼樂園度假區]]的乘客使用。該站亦是前地鐵系統首個設有閘內洗手間的車站。

===月台===
{{ multiple image | direction = vertical | align = right | width = 220
| image1 = Sunny Bay Station 2017 12 part6.jpg | caption1 = 1號月台（往東涌方向）
| image2 = Sunny Bay Station 2017 12 part2.jpg | caption2 = 2號月台（往香港方向）
| image3 = Sunny Bay Station 2017 12 part8.jpg | caption3 = 3號月台（往迪士尼方向）
| image4 = Sunny Bay Station 2017 12 part3.jpg | caption4 = 1號及2號月台中間為機場快綫路軌及緊急月台
}}
欣澳站是繼[[寶琳站|寶琳站]]後，出入口、出入閘機和月台均設在同一層的車站。

欣澳站設有一個[[島式月台|島式月台]]及一個[[側式月台|側式月台]]，2號月台為側式月台，供東涌綫往[[香港站|香港]]方向的列車使用。本站唯一出口就設在此月台旁。

1號及3號月台則共用同一個島式月台，1號月台為東涌綫往[[東涌站_(東涌綫)|東涌]]方向的列車使用，3號月台為[[迪士尼綫|迪士尼綫]]往[[迪士尼站_(香港)|迪士尼]]方向的列車使用，該月台主要為東涌綫乘客轉乘迪士尼綫列車前往[[香港迪士尼樂園度假區|香港迪士尼樂園度假區]]之用。

而位於欣澳站中央，設有2個後備島式月台，該月台名為欣澳應急月台（前稱陰澳應急月台），只作東涌綫及機場快綫在緊急事故時疏散乘客之用。

機場快綫的路軌與東涌綫的路軌分離，這個設計的原意是讓東涌綫列車在欣澳站停站時，容許機場快綫的列車超越。但是因為運行圖設定的問題，機場快綫列車時常會早到，此時會在接近欣澳站時減速，甚至在欣澳站內的路軌短暫停留，以等候東涌綫列車駛離。

由於欣澳站中央為緊急月台、東涌綫及機場快綫的路軌，因此設有[[行人天橋|行人天橋]]連各月台，方便乘客由迪士尼綫轉乘東涌綫返回市區。 

欣澳站屬開放式設計，月台並不設[[空氣調節|空氣調節]]，月台上只有數部[[电风扇|涼風機]]降溫，因此夏天時非常炎熱，而下雨時更會有雨水從天幕與地面間的空隙隨風飄入。月台上的帳篷屋頂是以半曲輕鋼結構配合[[聚四氟乙烯|聚四氟乙烯]]物料纖維布料，下雨天時，雨水即可自動清洗帳篷。同时在興建時采用高度較低的[[月台閘門|月台閘門]]，使欣澳站成為全港首個採用月台閘門的鐵路車站，這種月台閘門的高度大約是其他港鐵站幕門的一半（1.5米）。而為了遷就迪士尼綫的列車設計，迪士尼綫的閘門較東涌綫的低，大約只有1.2米。

===出入口===

==利用狀況==
{{ multiple image | direction = vertical | align = right | width = 220
| image1 = Sunny Bay Public Transport Interchange.jpg | caption1 = 位於A出入口外的欣澳公共運輸交匯處
| image2 = MTR SUN (1).JPG | caption2 = 行人天橋

由於欣澳站附近沒有任何民居或旅遊設施<ref>根據[http://www.mtr.com.hk/jplanner/images/maps/sun.pdf 港鐵街道圖]的資料，欣澳站A出口只有欣澳站公共運輸交匯處及欣澳道，並無其他大型設施。</ref>，在香港迪士尼樂園開幕及愉景灣接駁巴士通車之前，該站曾經幾乎無居民或遊客使用。而由於[[北大嶼山公路|北大嶼山公路]]旁邊的一段[[機場鐵路|機場鐵路]]為開放式路段，是觀賞來往列車的好地方，加上風景優美，亦可沿大嶼山北岸步行到[[鹿頸灣|鹿頸灣]]、[[深水角|深水角]]，所以吸引了不少人利用欣澳站前往附近去拍攝來往的列車和日落景色，而車站主要以前往迪士尼及愉景灣之乘客為主。<ref>{{cite web |title=4個最靚日落泊車位 |url=https://www.uber.com/zh-HK/blog/%EF%BC%94%E5%80%8B%E6%9C%80%E9%9D%9A%E6%97%A5%E8%90%BD%E6%B3%8A%E8%BB%8A%E4%BD%8D/ |website=Uber |accessdate=2018-09-08}}</ref>

本站以東為[[港珠澳大橋|港珠澳大橋]]其中一個工程地盤，因此在[[港珠澳大橋|港珠澳大橋]]工程期間亦有一些工人使用該站上下班。自港珠澳大橋開通後，來往港珠澳大橋香港口岸的[[城巴B5線|B5線]]在此站外的欣澳公共運輸交匯處設站，因此亦吸引了來往香港和珠海、澳門的乘客使用本站。

此外，該站亦是東涌綫及迪士尼綫合共9個車站之中，出入閘人數最少的車站。

==接駁交通==
[[愉景灣|愉景灣]]的管理組織香港興業開辦往來愉景灣、經愉景灣隧道、翔東路及欣澳道，前往站外的[[欣澳站公共運輸交匯處|欣澳站公共運輸交匯處]]的專線巴士服務，編號為DB03R，於2005年9月17日投入服務，服務時間為每日上午6時45分至凌晨1時15分，欣澳站跟東涌站一同成為往來愉景灣的中轉車站，DB03R線亦曾是欣澳站唯一的接駁交通工具，但若翔東路封閉便會暫停服務。

2018年10月24日，[[城巴|城巴]]開辦往來欣澳站與[[港珠澳大橋香港口岸|港珠澳大橋香港口岸]]的[[城巴B5線|B5線]]<ref>{{cite web|url=https://www.nwstbus.com.hk/en/uploadedFiles/HZM/full.pdf|title=城巴機場快線連接港珠澳大橋香港口岸|work=[[城巴|城巴]]|date=2018-10-19|accessdate=2018-10-21}}</ref>，以便旅客前往香港口岸出境轉乘口岸穿梭巴士前往珠海和澳門，此乃欣澳站首條全日服務及接駁邊境口岸的專利巴士服務。

==歷史==
===西部外走廊===
{{see also|香港10號幹線#歷史|西部外走廊}}
[[香港殖民地時期#香港政府|香港政府]]落實興建[[香港迪士尼樂園度假區|香港迪士尼樂園度假區]]及[[迪士尼綫|迪士尼綫]]之前，早於1989年《機場及海港發展計劃》曾倡議在陰澳及[[竹篙灣|竹篙灣]]興建新貨櫃碼頭、新港澳碼頭及新鐵路西部外走廊，以配合[[香港國際機場|香港國際機場]]遷往[[赤鱲角|赤鱲角]]及發展大嶼山。

當時政府建議[[西部外走廊|西部外走廊]]在陰澳設站，以便與[[香港機場核心計劃#機場鐵路|機場鐵路]]作交匯，形成一個完善鐵路網絡；因此在機場快綫興建初期，已預定在陰澳興建車站。後因政府把西部走廊（現為[[西鐵綫|西鐵綫]]）的營運權交由[[九廣鐵路|九廣鐵路]]，加上[[青洲_(中西區)|青洲]]填海計劃遭否決及[[葵青貨櫃碼頭|新貨櫃碼頭決定在青衣島興建]]，使有關計劃一直擱置至今。

===命名===
規劃初期，位於大嶼山東北部的新車站取當地「[[陰澳|陰澳]]」為車站名稱，即'''陰澳站'''（英文名稱為'''Yam O Station'''），由於[[華特迪士尼公司|迪士尼公司]]及[[荃灣區議會|荃灣區議會]]認為陰澳的「陰」字未能配合迪士尼的歡樂形象，因此迪士尼公司及旅遊事務專員向香港政府[[地政總署|地政總署]]及[[香港鐵路有限公司|地鐵公司]]要求把陰澳打水灣地區及其車站名稱更改為「[[欣澳|欣澳]]」，以配合[[迪士尼樂園|迪士尼樂園]]開心歡欣的形象。

地政專員經過區議會諮詢後，決定批准是次變更，更於2003年刊登在[[香港憲報|憲報]]內。因此地鐵公司把車站正式命名為'''欣澳站'''（英文名稱改為'''Sunny Bay Station'''），寓意一個充滿陽光與歡欣的地方。後來，地鐵公司又於2006年1月1日將原有的「陰澳應急月台」易名為「欣澳應急月台」。

這次改名並引起學者及市民批評，他們認為是次名字變更為欣澳是錯誤理解陰澳的原意，更以外國文化取替了本地文化。
