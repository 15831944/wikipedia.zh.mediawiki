'''欣安邨'''（英語：'''Yan On Estate'''）是[[香港房屋委員會|香港房屋委員會]]轄下的一個[[香港公共屋邨|公共屋邨]]，位於[[新界|新界]][[沙田區|沙田區]][[馬鞍山_(香港市鎮)|馬鞍山]]的南部，項目名稱為「馬鞍山第86B區公共租住房屋發展計劃」，但已列入[[香港市區|擴展市區]]配屋區內，是[[馬鞍山_(香港市鎮)|馬鞍山]]最南端的公共屋邨，亦是馬鞍山第五個建成公共屋邨，[[宜居物業管理有限公司|宜居物業管理有限公司]]負責物業管理。

欣安邨現時共有3幢[[樓宇|樓宇]]，已於2011年5月30日落成並開始入伙。九巴[[九龍巴士43X線|43X]]及[[九龍巴士89C線|89C]]亦於2011年6月1日起繞經該邨，為新入伙居民提供對外交通服務<ref>[http://www.kmb.hk/tc/news/press/archives/news201105261316.html 九巴43X線及89C線更改路線]，九巴新聞稿，2011-05-26</ref>。

== 歷史 ==
欣安邨前身為[[城安臨時房屋區|城安臨時房屋區]]，共有33座臨時房屋，並於1999年清拆。

== 屋邨資料 ==
欣安邨由[[房屋署|房屋署]]發展及工程處屬下之總建築師(4)設計、工程師(134)負責工程計算及工料測量師(C7)負責工料測量，並由[[有利建築|有利建築]]作為總承建商，合約總值7億9,895萬港元<ref>{{cite web |language=en |url=http://www.yaulee.com/eng/businessDomain/yauleeConstru/curProject/curPro20080006.htm |title=Construction of Ma On Shan Area 86B |work=有利建築 |date= |accessdate=2010-05-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20110427031510/http://www.yaulee.com/eng/businessDomain/yauleeConstru/curProject/curPro20080006.htm |archivedate=2011-04-27 }}</ref><ref>{{cite web|language=zh|url=http://www.yaulee.com/tchi/businessDomain/yauleeConstru/curProject/curPro20080006.htm|title=馬鞍山八十六B區發展項目|work=有利建築|date=|accessdate=2011-01-16|deadurl=yes|archiveurl=https://web.archive.org/web/20110425234621/http://www.yaulee.com/tchi/businessDomain/yauleeConstru/curProject/curPro20080006.htm|archivedate=2011-04-25}}</ref>。屋邨建有3座40層高大廈，提供2,587個住宅單位及3個互助委員會辦事處，同時興建2幢1層作商店及服務行業（零售用途）的建築物，總樓面面積有410平方米。邨內設有社區園圃、羽毛球場、兩個兒童遊樂場、籃球場及五人足球場。

== 屋邨設計 ==
欣安邨設計別具心思，屋邨外牆以啡黃色作主調，飾以粉紅及粉綠色與周邊環境及建築物協調。地下入口大堂噴上了啡色為主調的石頭漆，信箱頂部配上環保木與石膏假天花及色彩繽紛的指示牌互相連接 ，使大堂設計舒適雅觀。而樓上走廊則以淡黃色為主體，4條走廊各配上一隻主題色，分別為粉紅、鮮黃、淺綠及淡藍色；而每層走廊的指示牌及各戶之門鐘牌亦以該走廊色調配上花、葉、實物及水的圖片一張，與周邊環境協調之餘，也彰顯其獨特主題及個性。根據調查顯示居民也很喜歡這類以指示牌為主之裝飾手段的廉價而實而不華的設計。

[[香港房屋委員會|香港房屋委員會]]於2010年12月6日發表新聞稿，宣布6個新屋邨租金，將以所屬地區現行最高租金水平釐定，欣安邨租金為每月每平方米51.3元<ref>[http://www.housingauthority.gov.hk/b5/aboutus/news/pressreleases/0,,2-0-21658--0,00.html 房委會釐定六個新屋邨租金] </ref>。

因該馬鞍山路新居屋地盤僅得80平方米作零售設施，房署在2013年8月下旬計劃增加鄰近欣安邨的零售配套設施，建議將恆泰路改道，以騰空3.26公頃土地興建欣安邨第2期，興建3幢42至44層大廈，提供1600個單位。欣安邨2期計劃於2019年動工，2024年完工，但第二期的零售設施會提前於2016年動工，2020年完工，以配合鄰近的新居屋單位入伙<ref>[http://hk.news.yahoo.com/%E9%A6%AC%E9%9E%8D%E5%B1%B1%E5%BB%BA-43%E5%B1%A4%E6%9C%80%E9%AB%98%E5%B1%85%E5%B1%8B-225019547.html 馬鞍山建 43層最高居屋]</ref><ref>[http://www.districtcouncils.gov.hk/st/doc/2012_2015/common/committee_meetings_doc/dhc/3910/st_dhc_2013_052_tcannex.pdf 欣安邨第2期 公共房屋發展項目方案]</ref>。

== 問題 ==
*鉛水事件後，每層大堂獲安裝臨時供水系統，但有居民發現水呈奶白色，擔心有問題不敢飲用。房署回應指經了解後證實為氣泡。<ref>{{cite news|title=欣安邨食水呈奶白色　房署稱屬氣泡無礙水質|url=http://hk.on.cc/hk/bkn/cnt/news/20151215/bkn-20151215193557978-1215_00822_001.html|accessdate=2017-12-27|publisher=東方日報|date=2015-12-15}}</ref>

*2016年1月26日，媒體報道欣悅樓多個樓層的走廊出現地磚拱起情況，管理處遂將受損位置圍封。房屋署指會盡快進行復修工作。<ref>{{cite news|title=極寒後見走廊爆磚　欣安邨欣悅樓居民好驚訝|url=http://hk.on.cc/hk/bkn/cnt/news/20160126/bkn-20160126190120083-0126_00822_001.html|accessdate=2017-12-27|publisher=東方日報|date=2016-01-26}}</ref>
