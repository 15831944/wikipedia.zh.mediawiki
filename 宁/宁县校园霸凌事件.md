'''宁县校园霸凌事件'''是指2018年12月14日，[[甘肃省|甘肃省]][[庆阳市|庆阳市]][[宁县|宁县]][[和盛镇_(宁县)|和盛镇]]杨庄村一位8岁女童遭遇校园霸凌的事件。<ref>{{cite web |url=https://www.thepaper.cn/newsDetail_forward_2851754 |title=甘肃庆阳一8岁女孩疑在校被打下体出血，当地警方介入调查 |author=李涛 张月朦 |date=2019-01-14 |website=[[北京青年报|北京青年报]] |access-date=2019-01-17 }}</ref>

== 事情经过 ==
=== 家属反馈 ===
2018年12月14日，杨庄小学一位肖姓老师的口红丢失怀疑是受害女童偷走，当日下午肖姓老师与受害女童的奶奶沟通要求赔偿，后放学时发现女童受伤“裤子前后都是血，还有血顺着腿往鞋子里流”。据受害女童的姑母表示，肖姓老师曾单独与受害女童沟通是否存在偷窃行为，后来“老师要求同学们把孩子的裤子脱光了打，用教鞭伸进孩子的下体打。一个班一共7个娃娃，5个娃娃都打了。”当晚受害女童的家属报案。<ref name="xdkb">{{cite web |url=http://news.xdkb.net/2019-01/14/content_1136133.htm |title=甘肃8岁女孩被教鞭打致下体出血！警方正调查，涉事教师还在上课 |author=于露 |date=2019-01-14 |website=[[现代快报|现代快报]] |access-date=2019-01-17 }}</ref>

2018年12月15日，家属将受害女童送往宁县和盛医院、庆阳市人民医院进行治疗，后转[[西安市儿童医院|西安市儿童医院]]治疗<ref name="xinhuanet" />，3天转回镇医院接受治疗。<ref name="xdkb" />

2019年1月13日，因受害女童反馈疼痛不止，再次住院治疗。<ref>{{cite web |url=http://news.youth.cn/sh/201901/t20190114_11843508.htm |title=甘肃一家长称8岁女孩在学校被打下体出血 家属控诉幼童曾遭校园霸凌 |author=李永鹏 李雪婷 岳橦 |date=2019-01-14 |website=[[中国青年网|中国青年网]] |access-date=2019-01-17 }}</ref>

=== 官方通报 ===
2019年1月15日，宁县公安局、宁县教育体育局就此事件进行通报，通报称杨庄小学两名一年级男童因怀疑受害女童偷拿其橡皮，并借钱一元未还，因此将受害女童裤子脱掉，用笤帚对受害女童下体进行乱打乱戳，致其下体受伤。<ref>{{cite web |url=http://epaper.legaldaily.com.cn/fzrb/content/20190116/Articel06009GN.htm |title=甘肃宁县通报“八岁女孩被打下体受伤”事件 |author=赵志锋 |date=2019-01-16 |website=[[法制日报|法制日报]] |access-date=2019-01-17 }}</ref>公安部门依据《[[中华人民共和国治安管理处罚法|中华人民共和国治安管理处罚法]]》规定，因涉事两男童未满14周岁不予处罚，责令两男童监护人严加管教，并依法承担民事责任，杨庄小学校长、副校长等人予以免职处分<ref name="xinhuanet">{{cite web |url=http://www.gs.xinhuanet.com/jizhe/2019-01/15/c_1123992380.htm |title=甘肃宁县通报8岁女孩受伤事件：公安机关介入调查 校长被免职 |author=梁军 |date=2019-01-15 |website=[[新华社|新华社]] |access-date=2019-01-17 }}</ref>。

2019年1月16日，据报道杨庄小学肖姓老师再次在当地派出所接受询问，受害女童仍在宁县医院接受治疗。<ref>{{cite web |url=https://app.bjtitle.com/8816/newshow.php?newsid=5223137 |title=8岁女孩受伤事件：丢口红老师接受警方询问 |author=付垚 张月朦 |date=2019-01-17 |website=[[北京青年报|北京青年报]] |access-date=2019-01-17 }}</ref>有志愿者生成受害女童伤情加剧被隔离在医院且接触女童需有教育局及派出所人员配同。<ref>{{cite web |url=https://app.bjtitle.com/8816/newshow.php?newsid=5223137 |title=志愿者称甘肃8岁受伤女孩转院受阻， 警方：以官方回应为准 |author=沈轶 陈彦霏 |date=2019-01-17 |website=[[澎湃新闻|澎湃新闻]] |access-date=2019-01-21 }}</ref>医院称女童子宫卵巢没受损应不影响生育。<ref>{{cite web |url=http://www.gs.chinanews.com/news/2019/01-17/312634.shtml |title=1月17日【一句话速读陇原】 |author=杨娜 |date=2019-01-17 |website=[[中国新闻网|中国新闻网]] |access-date=2019-01-21 }}</ref>

== 争议 ==
官方通报事件过程与家属反馈并不一致，受害女童的表哥告诉记者“事发导火索是老师丢口红，并非偷橡皮。”<ref>{{cite web |url=http://www.bjnews.com.cn/news/2019/01/15/539824.html |title=甘肃八岁女童遭校园欺凌：事发原因成焦点 |author=王瑞文 |date=2019-01-15 |website=[[新京报|新京报]] |access-date=2019-01-17 }}</ref>

澎湃评论表示，官方结论与家长说法大相径庭，一些疑问官方并没有明确说明，这些使此案成为[[羅生門_(小說)|罗生门]]引出更多质疑，官方结论难以服众。<ref>{{cite web |url=https://www.thepaper.cn/newsDetail_forward_2856768 |title=马上评｜关键信息语焉不详，舆情就会陷入“猜疑链” |author=于平 |date=2019-01-15 |website=[[澎湃新闻|澎湃新闻]] |access-date=2019-01-17 }}</ref><ref>{{cite web |url=https://www.thepaper.cn/newsDetail_forward_2856625 |title=马上评｜女童下体重伤，官方调查不能绕开疑点 |author=沈彬 |date=2019-01-15 |website=[[澎湃新闻|澎湃新闻]] |access-date=2019-01-17 }}</ref>

光明网评论表示，官方通报滞后，且与网络披露信息有许多不同，需要官方披露更多信息，从女童所处的环境，以及遭受的虐待来看，农村贫困家庭儿童仍是容易受到侵害的群体，需要社会更多关注。<ref>{{cite web |url=http://guancha.gmw.cn/2019-01/16/content_32361667.htm |title=8岁女童被施暴事件：通报尚不完整，女童需要救助  |author=光明网评论员 |date=2019-01-16 |website=[[光明网|光明网]] |access-date=2019-01-17 }}</ref>

== 后续 ==
2019年1月21日，庆阳市人民政府通报：1、肖姓老师在事发时不在教室，事件与口红无直接关系；2、肖姓老师严重失职失责，已决定解聘；3、肖姓老师无复杂的“社会背景”。<ref>{{cite web |url=http://www.ningxian.gov.cn/publicity/rsxx/11079 |title=关于宁县小学生被伤事件的情况通报 |author=政府办 |date=2019-01-21 |website=庆阳市人民政府 |access-date=2019-01-21 }}</ref>
