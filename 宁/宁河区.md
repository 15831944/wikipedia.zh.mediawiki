'''宁河区'''是[[中华人民共和国|中国]][[天津市|天津市]]下辖的一个[[市辖区|市辖区]]。区人民政府驻[[芦台镇|芦台镇]]。<ref>[http://www.ninghe.gov.cn/ 天津市宁河县政务网]</ref>

== 历史 ==
[[清|清]]雍正九年（1731年）建始宁河县，隶属顺天府东路厅；1973年8月由河北省划归天津市管辖。2015年7月撤县设区。<ref>{{cite web |url=http://202.108.98.30/description?dcpid=2015|title=中华人民共和国二〇一五年县级以上行政区划变更情况}}</ref>

== 行政区划 ==
宁河区现下辖14个[[镇|镇]]：
[[芦台镇|芦台镇]]、[[丰台镇_(天津市)|丰台镇]]、[[岳龙镇|岳龙镇]]、[[板桥镇_(宁河县)|板桥镇]]、[[苗庄镇_(宁河县)|苗庄镇]]、[[宁河镇|宁河镇]]、[[东棘坨镇|东棘坨镇]]、[[大北涧沽镇|大北涧沽镇]]、[[潘庄镇_(宁河县)|潘庄镇]]、[[七里海镇|七里海镇]]、[[造甲城镇|造甲城镇]]、[[廉庄镇|廉庄镇]]、[[俵口镇|俵口镇]]、[[北淮淀镇|北淮淀镇]]。
