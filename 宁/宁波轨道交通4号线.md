'''宁波轨道交通4号线'''是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]]建设中的一条轨道交通线路，全长36.11千米，标识色为'''{{color|#{{NingboRT color|4}}|森林绿}}'''。线路自西北向东南，连接[[慈城古建筑群|慈城古建筑群]]和[[东钱湖|东钱湖]]两大旅游区和[[庄桥站_(宁波市)|庄桥站]]、[[宁波站|宁波站]]、[[宁波东站|宁波东站]]三座铁路客运站。线路经过[[江北区_(宁波市)|江北]]、[[海曙区|海曙]]和[[鄞州区|鄞州]]三区，计划设25座车站<ref name="线网批复" />。宁波轨道交通4号线于2015年11月30日开工建设<ref name="begin-construction">{{cite web|url=http://news.cnnb.com.cn/system/2015/11/30/008436394.shtml|title=又一条地铁来了！宁波轨交4号线今开工 工期5年|publisher=甬派客户端|date=2015-11-30|accessdate=2015-11-30|archive-url=https://web.archive.org/web/20151208074829/http://news.cnnb.com.cn/system/2015/11/30/008436394.shtml|archive-date=2015-12-08|dead-url=no}}</ref>，计划于2019年7月全线洞通<ref>{{Cite web |url=http://www.nbmetro.com/news_dt.php?info/1407 |title=4号线建设迎新进展 庄丽盾构区间双线首通 东钱湖车辆段开工建设 |accessdate=2018-04-27 |author=宁波轨道交通综合部 |date=2018-04-25 |publisher=宁波轨道交通 |archive-url=https://web.archive.org/web/20180427120556/http://www.nbmetro.com/news_dt.php?info%2F1407 |archive-date=2018-04-27 |dead-url=no }}</ref>， 2020年底通车试运营<ref name="env_report">宁波市环境保护科学研究设计院, 宁波市轨道交通4号线工程环境影响报告书（简本）, 宁波, 2014年11月</ref>。

== 线路走向 ==
宁波轨道交通4号线从慈城站起高架向东南方向跨过[[萧甬铁路|萧甬铁路]]和[[杭甬客运专线|杭甬客运专线]]，经过慈城新城后到达[[北外环路_(宁波)|北外环路]]后进入北外环路中央分隔带二层，到达江北大道立交后入地，转至铁路庄桥站。此后再次穿越[[萧甬铁路|萧甬铁路]]并向南转向并跨过姚江。此后线路沿双东路、翠柏路、苍松路直至铁路宁波站。此后线路向东，进入长春路后穿过奉化江，沿兴宁路向东，到达沧海路南延伸段向南，经过宁波东站后沿沧海路、宁横路向南，经过鄞县大道，至首南路再度转向东面，经过下应抵达东钱湖镇，最后沿钱湖大道抵达东钱湖核心区<ref>{{cite web|url=http://news.cnnb.com.cn/system/2014/11/13/008203666.shtml|title=宁波轨道交通3号线一期工程拿到“准生证”|publisher=中国宁波网|date=2014-11-13|accessdate=2014-11-16|archive-url=https://web.archive.org/web/20141129020357/http://news.cnnb.com.cn/system/2014/11/13/008203666.shtml|archive-date=2014-11-29|dead-url=no}}</ref>。

== 历史 ==

=== 规划 ===
宁波轨道交通4号线最早出现在2003年8月编制完成的《宁波市城市快速轨道交通线网规划》中，该规划将4号线定为宁波轨道交通的3条辅助线之一，但当时并未列入2020年前须建成的线路之一。当时的轨道交通4号线西起慈城，东至鄞州东钱湖镇，全长40.6公里<ref name="renda-vote">{{cite web|url=http://news.cnnb.com.cn/system/2006/11/29/005214833.shtml|title=宁波市“十”字轨道交通将在2015年建成|publisher=中国宁波网|date=2006-11-29|accessdate=2013-11-09|archive-url=https://web.archive.org/web/20131109121844/http://news.cnnb.com.cn/system/2006/11/29/005214833.shtml|archive-date=2013-11-09|dead-url=no}}</ref>。2011年，4号线被列入《宁波市城市轨道交通近期建设规划（2013-2020年）》中，规划于2020年前建设，而起点和终点分别被缩短到[[慈城站_(地铁)|慈城站]]和[[云龙站_(地铁)|云龙站]]<ref name="四线一环">{{cite web|url=http://zjnews.zjol.com.cn/05zjnews/system/2011/05/26/017551895.shtml|title=2020年宁波将形成"四线一环"轨道交通网络格局|publisher=浙江在线|date=2011-05-26|accessdate=2013-11-09|archive-url=https://web.archive.org/web/20131109121633/http://zjnews.zjol.com.cn/05zjnews/system/2011/05/26/017551895.shtml|archive-date=2013-11-09|dead-url=no}}</ref>。2013年11月，该规划获得国务院批复<ref name="线网批复">{{cite web|url=http://www.nbdpc.gov.cn/cat/cat122/con_122_32409.html|title=《宁波市城市轨道交通近期建设规划（2013-2020年）》获国家批复|publisher=宁波市发展和改革委员|date=2013-11-08|accessdate=2013-11-09|deadurl=yes|archiveurl=https://web.archive.org/web/20131109124347/http://www.nbdpc.gov.cn/cat/cat122/con_122_32409.html|archivedate=2013-11-09}}</ref>。

=== 建设 ===
[[File:Route_change_of_L4,_NBRT.svg|thumb]]
在4号线正式开始建设之前，[[大卿桥站|大卿桥站]]、[[宁波火车站_(地铁)|宁波火车站]]结构已由1号线、2号线分别代为建设。2011年开始建设的[[北外环路_(宁波)|北外环路]]高架二层已同时动工建设4号线[[金山路站|金山路站]]至[[洪大路站|洪大路站]]的车站和区间，使之成为整条4号线最早动工建设的部分<ref name="北外环">{{cite web|url=http://news.cnnb.com.cn/system/2012/10/03/007479943.shtml|title=宁波北外环快速路高架上开建轨道交通|publisher=中国宁波网|date=2012-10-03|accessdate=2013-11-15|archive-url=https://web.archive.org/web/20141129030001/http://news.cnnb.com.cn/system/2012/10/03/007479943.shtml|archive-date=2014-11-29|dead-url=no}}</ref>。4号线试验站[[翠柏里站|翠柏里站]]也开始设计，但并未立即投入建设<ref name="cuibaili-env">{{cite article|author=中铁第四勘察设计院集团有限公司|title=宁波市轨道交通4号线试验站工程环境影响报告书（简本）|date=2011-08-08|accessdate=2011-08-13}}</ref>。2014年，根据轨道交通线网优化的成果，为了解决新增7号线与4号线换乘不利的问题，4号线原终点站云龙站改属7号线，而4号线终点改为钱湖大道尽头的东钱湖站，使得两线成十字交叉。同时，为了扩大轨道交通的服务范围，4号线下应段改为沿首南路敷设。这一改线方案由国家发改委办公厅批复同意<ref name="env_report" />。4号线工程可行性研究报告于2015年6月2日获得宁波市发改委批复<ref>{{cite web|url=http://news.cnnb.com.cn/system/2015/06/08/008334149.shtml|title=宁波轨道交通4号线工程可行性研究报告获批|publisher=中国宁波网|date=2015-06-09|accessdate=2015-06-10|archive-url=https://web.archive.org/web/20150614003310/http://news.cnnb.com.cn/system/2015/06/08/008334149.shtml|archive-date=2015-06-14|dead-url=no}}</ref>。

宁波轨道交通4号线于2015年11月30日正式开工建设<ref name="begin-construction" />。2017年4月26日，翠柏里站主体结构封顶，成为全线新建车站中首座封顶的车站<ref>{{cite web|url=http://chinacem.com.cn/qydt/2017-05/219491.html|title=城轨公司宁波4001标见缝巧插针 翠柏里站主体结构封顶|publisher=中铁一局集团有限公司|date=2017-05-05|accessdate=2017-05-26|archive-url=https://web.archive.org/web/20180424202504/http://chinacem.com.cn/qydt/2017-05/219491.html|archive-date=2018-04-24|dead-url=no}}</ref>。5月27日，矩形盾构“阳明号”从该站南端始发，翠柏里–大卿桥区间成为4号线首个进行盾构施工的区间，也是中国大陆首条穿越核心城区的类矩形盾构隧道<ref>{{cite web|url=http://news.cnnb.com.cn/system/2017/05/27/008639988.shtml|title=轨道交通4号线建设的独门武器"阳明号"始发 系宁波首创|publisher=甬派|date=2017-05-27|accessdate=2017-05-31|archive-url=https://web.archive.org/web/20170614103009/http://news.cnnb.com.cn/system/2017/05/27/008639988.shtml|archive-date=2017-06-14|dead-url=no}}</ref>。2018年4月23日，庄桥火车站至丽江路区间成为全线首个双向贯通的区间<ref>{{cite web|url=http://news.cnnb.com.cn/system/2018/04/23/008745485.shtml|title=4号线攻克全线最大风险点 庄丽盾构区间双线首通|publisher=中国宁波网|date=2018-04-23|accessdate=2018-04-24|archive-url=https://web.archive.org/web/20180424202540/http://news.cnnb.com.cn/system/2018/04/23/008745485.shtml|archive-date=2018-04-24|dead-url=no}}</ref>。

== 车站 ==
宁波轨道交通4号线规划设车站25座。其中慈城站、官山河站、长兴路、奥体中心站、洪塘中路站、洪大路站为高架站，其余为地下站<ref name="env_report" />。高架站中，金山路站、奥体中心站、洪塘中路站和洪大路站采用了类似[[上海轨道交通1号线|上海轨道交通1号线]]北延段的设计，位于北外环路高架二层。这一设计可显著减少土地利用和工程造价<ref name="北外环" />。

== 车辆 ==
宁波轨道交通4号线采用B型地铁车辆，编组为6辆编组，4动2拖，最高速度80公里/小时<ref name="bidding-1">{{cite web|url=http://www.nbdpc.gov.cn/cat/cat85/con_85_33132.html|title=宁波市轨道交通4号线工程设计总体总包（SJ4001标段）（勘察设计）招标公告|date=2014-01-20|accessdate=2014-01-22|deadurl=yes|archiveurl=https://web.archive.org/web/20140201201042/http://www.nbdpc.gov.cn/cat/cat85/con_85_33132.html|archivedate=2014-02-01}}</ref>，初期配车36列<ref>{{cite web|url=http://www.bidding.gov.cn/gcjszbgg/117788.htm |title=宁波市轨道交通2号线二期、4号线工程车辆监造等服务招标公告 |publisher=宁波市公共资源交易中心 |date=2017-04-25 |accessdate=2017-06-12 |archiveurl=https://archive.is/20170611165029/http://www.bidding.gov.cn/gcjszbgg/117788.htm |archivedate=2017-06-11 |deadurl=yes }}</ref>。

== 行车组织 ==
宁波轨道交通4号线计划初期平峰间隔10分钟，远期平峰间隔5分钟，高峰期在奥体中心站和南高教园区站间开行区间车<ref name="env_report" />。

== 附属设施 ==
=== 车辆基地 ===
宁波轨道交通4号线设置东钱湖车辆段和慈城停车场。东钱湖车辆段位于鄞州区[[东钱湖镇|东钱湖镇]]境内，钱湖大道以南，玉泉南路以西，小洋泾支流以东，接轨于钱湖大道站，承担4号线车辆的定修、临修、旋轮作业和列车检测、停放任务。慈城停车场位于江北区[[慈城镇|慈城镇]]境内，北邻萧甬铁路，南靠夹田大河，西临慈城连接线，接轨于慈城新城站，承担4号线车辆的乘务、停放、整备等任务。车辆的大修、架修任务由5号线经堂庵跟车辆段承担<ref name="env_report" />。

=== 主变电所 ===
宁波轨道交通4号线设有谢家和甬兴两座主变电所，分别位于洪塘中路站和学府路站附近。4号线同时与2号线共享夏禹主变电所，位于2号线丽园南路站附近<ref name="bidding-1" />。

=== 信号系统 ===
宁波轨道交通4号线采用由[[交控科技|交控科技]]提供的信号系统<ref>{{Cite web |title=宁波市轨道交通4号线工程信号系统采购项目（材料设备）中标公示 |url=http://www.bidding.gov.cn/jsgcclsbzbgg/129686.htm |accessdate=2019-02-23 |publisher=宁波市公共资源交易中心 |date=2017-10-30 |archive-url=https://web.archive.org/web/20190224115803/http://www.bidding.gov.cn/jsgcclsbzbgg/129686.htm |archive-date=2019-02-24 |dead-url=no }}</ref>。
