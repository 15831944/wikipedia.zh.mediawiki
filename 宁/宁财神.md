'''宁财神'''（{{bd|1975年|2月23日|catIdx=N宁}}），[[上海人|上海人]]，本名'''陈万宁'''。他最初给自己取藝名为“宁采臣”{{noteTag|1=“宁采臣”是[[清朝|清朝]][[小說|小說]]《[[聊齋志異|聊齋志異]]·[[聂小倩|聂小倩]]》中的男主角，作者[[蒲松齡|蒲松齡]]。}}，後改為其諧音“宁[[财神|财神]]”<ref>{{cite newspaper| url=http://gzdaily.dayoo.com/html/2009-02/07/content_459661.htm| title=宁财神做电视不看电视做舞台喜欢乱搞| publisher=[[广州日报|广州日报]]| date=2009-02-07| deadurl=yes| archiveurl=https://web.archive.org/web/20160304075126/http://gzdaily.dayoo.com/html/2009-02/07/content_459661.htm| archivedate=2016-03-04}}</ref>。[[情景喜剧|情景喜剧]]《[[武林外传|武林外传]]》的[[编剧|编剧]]。

代表作有剧本《[[武林外传|武林外传]]》(2006)、《[[防火墙5788|防火墙5788]]》、《[[大笑江湖|大笑江湖]]》、《[[人在囧途|人在囧途]]》、《[[龙门镖局|龙门镖局]]》(2013)，[[小说|小说]]《缘分的天空》、《假装纯情》、《无数次亲密接触》。

== 生平 ==
他15岁进入大学，毕业于[[华东理工大学|华东理工大学]][[金融|金融]]专业，毕业后随朋友到[[北京|北京]]，从事与自己专业相关的工作——[[期货交易|期货交易]]。他的业余爱好是听歌、看碟。他喜欢[[张学友|张学友]]的歌和[[周星驰|周星驰]]的电影<ref>{{cite web|url=http://datalib.ent.qq.com/star/3003/starinfo.shtml | language= | title=宁财神基本资料 |publisher=腾讯娱乐 }}</ref>。妻子程娇娥。

2014年6月24日傍晚，他因涉嫌吸食[[甲基苯丙胺|冰毒]]，在北京市[[朝陽區_(北京市)|朝陽區 (北京市)]]工體北路一公寓內被当地公安查获并[[行政拘留|行政拘留]]15天，及後承認過去半年曾數次吸食，他对自己的吸毒行为表示“深切的歉意和懊悔”<ref>[http://ww1.sinaimg.cn/large/4cd3493fjw1ehrmquhvejj20de1i5qbo.jpg 北京市公安局微博通報]，2014年6月26日</ref><ref >{{cite web|url=http://society.people.com.cn/n/2014/0626/c1008-25205769.html | language= | title=著名编剧宁财神因吸毒被北京警方控制 |publisher=人民网 |date=2014年6月26日 |accessdate= }}</ref>，知情人曝宁财神吸毒被拘与之前因吸毒被抓的电影导演[[张元_(导演)|张元]]有关。<ref>{{cite web |url=http://epaper.jinghua.cn/html/2014-06/27/content_101069.htm |title=知情人曝宁财神吸毒被拘与张元有关 |publisher=京华时报 |date=2014-06-27 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20140701215420/http://epaper.jinghua.cn/html/2014-06/27/content_101069.htm |archivedate=2014-07-01 }}</ref>从拘留所释放后，他在个人微博发表了一篇《道歉信》<ref >{{cite web|url=http://news.163.com/14/0706/13/A0FLORVS00011229.html?360lm | language= | title=宁财神发公开道歉信:1.2克冰毒给生活带来剧变 |publisher= 财经网 |date=2014-07-06 |accessdate= }}</ref>，并表示此生再不涉毒。他于7月6日被释放<ref>[http://ent.ifeng.com/a/20140707/40163881_0.shtml 宁财神发道歉信 好友姚晨继续力挺：说到要做到]</ref>，他吸毒后的首次公开亮相是出席[[张靓颖|张靓颖]]新专辑《[[第七感|第七感]]》[[发布会|发布会]]。<ref>[http://news.hexun.com/2014-07-23/166912322.html 获释后助阵张靓颖 宁财神称“还活着”]</ref>

== 其他 ==
宁财神自2013年7月14日（按播出时间计）起担任“[[非诚勿扰_(节目)|非诚勿扰]]”节目的现场评论员。<ref>{{cite news | url=http://news.163.com/13/0710/02/93CV5PJU00014AED.html | title=曾子航走了宁财神来了(图) | work=网易 | date=2013-07-10 | accessdate=2013-07-14}}</ref>从2014年6月28日（按播出时间计）起，宁财神不再出现在《非诚勿扰》节目中。<ref>{{cite web|url=http://ent.ifeng.com/a/20140629/40147774_0.shtml|title=宁财神《非诚勿扰》被紧急处理：画面和声音都剪掉|publisher=人民网|author=蒋波|date=2014年6月29日|accessdate=2014年6月29日}}</ref>

2014年6月24日宁财神因吸食毒品被公安机关查获逮捕。在警方审问时，宁财神交代自己在7个月内共购买了三次[[冰毒|冰毒]]，总共价值三千[[人民币|元]]。<ref>{{cite web |title=宁财神共购3000元冰毒 或将拘留15日罚款2000元 |url = http://www.huanzhongnet.com/index.php?m=content&c=index&a=show&catid=29&id=6540 |publisher=环中网}}</ref>。10月8日，宁财神己正式入[[广电总局|广电总局]]的“封杀劣迹艺人”列单中，其国内参与制作的电影、电视剧、电视节目、广告节目、网络剧、微电影等，都列入暂停播出范围。<ref>[http://ent.qq.com/a/20141009/007422.htm   广电总局下发通知 影视网全面叫停“劣迹艺人”] {{webarchive|url=https://web.archive.org/web/20141015131616/http://ent.qq.com/a/20141009/007422.htm |date=2014-10-15 }}</ref>
