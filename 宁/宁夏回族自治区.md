'''宁夏回族自治区'''，简称'''宁'''，是[[中国|中国]][[中国西北地区|西北]]的[[中华人民共和国行政区划|少数民族自治区]]，位于[[黄河|黄河]]中游、[[蒙古高原|蒙古高原]]南部。大部分地区因黄河缓慢穿流而为[[河套|河套]]的一部分。宁夏下辖五个地级市，首府[[银川市|银川]]。

== 历史 ==
{{Main|宁夏历史}}

早在三万年前的[[旧石器时代|旧石器时代]]晚期，人类就在宁夏这块土地上繁衍生息。春秋战国为[[羌戎|羌戎]]、[[匈奴族|匈奴族]]散居地，秦属[[北地郡|北地郡]]，汉、魏分属郡州。唐、宋分归两道（路）。元初在[[西夏|西夏]]故地置[[宁夏行省|宁夏行省]]（取「[[西夏|夏]]地安宁」意得省名，始有“宁夏”地名），以西夏故都为[[中兴府|中兴府]]，改[[宁夏路|宁夏路]]。至元二十四年（1287年）将宁夏行省并入[[甘肃行省|甘肃行省]]，于宁夏设[[宁夏府路|宁夏府路]]。<ref>{{cite web |url = http://www.tcm100.com/ShuJuKu/GuJinDiMingDuiZhao/zzDiMing825.htm |title=宁夏 |website = Tcm100.com |date= |accessdate=2017-08-14 }}</ref>

明设[[宁夏卫|宁夏卫]]，属[[陕西布政使司|陕西布政使司]]，清属[[甘肃省_(清)|甘肃省]]。1928年从[[甘肃省_(中华民国)|甘肃省]]析置[[宁夏省_(中华民国)|宁夏省]]，是[[中华民国大陆时期|中华民国大陆时期]][[塞北四省|塞北四省]]（[[热河省|热河省]]、[[察哈尔省|察哈尔省]]、[[绥远省|绥远省]]、[[宁夏省_(中华民国)|宁夏省]]）之一。1954年[[宁夏省_(中华人民共和国)|宁夏省]]撤销，北部的「阿拉善和碩特特別旗」與「額濟納舊土爾扈特特別旗」（即現在的[[阿拉善盟|阿拉善盟]]）歸入[[內蒙古自治區|內蒙古自治區]]，其余部分并入[[甘肃省|甘肃省]]，宁夏属县属[[甘肃省|甘肃省]][[银川专区|银川专区]]<ref>{{cite web |url = http://www.chinataiwan.org/zt/szzt/zzqcjz/xj/xzgk/200909/t20090903_990172.htm |title = 宁夏回族自治区的历史 |website = Chinataiwan.org |date=2009-09-03 |accessdate=2017-08-14 }}</ref>。1958年10月25日'''宁夏回族自治区'''成立，[[甘肃省|甘肃省]][[银川专区|银川专区]]撤销，宁夏各县直属自治区。宁夏为中国的五大省级[[中华人民共和国行政区划|少数民族自治区]]之一。少数民族回族信仰伊斯兰教。
相比1949年中华人民共和国成立前寧夏省的版圖，宁夏回族自治区要小80%。

== 地理 ==
{{main|宁夏地理}}

宁夏位于东經104°17′至107°40′，北纬35°14′至39°22′，面积6.6万平方公里，山地高原占60%多，最高峰苏峪口北海拔3556米，年均气温5-9度，与[[陕西省|陕西省]]、[[甘肃省|甘肃省]]和[[内蒙古自治区|内蒙古自治区]]接壤。

宁夏地处中国东部季风区、西北干旱区和青藏高原区三大自然区域的交汇地带，黄河干流自南而北穿行于宁夏中北部。这种特殊的地理位置，导致了宁夏自然环境具有显著的过渡性、复杂性和不均衡性的独特面貌。大体上说，宁夏由南向北地势呈阶梯状下降，气温和蒸发量递增，降水量递减。

== 人口 ==
{{参见|宁夏回族自治区民族构成列表}}

[[File:Yinchuan_Highway.jpg|thumb]]
[[File:Tongxin_mosque.JPG|thumb]]]]

主体民族是[[汉族|汉族]]，约占宁夏总人口的2/3。

首要少数民族是[[回族|回族]]：约占宁夏总人口的1/3，同时占全国回族人口的1/5。回族人口主要分布在[[固原|固原]]和[[吴忠市|吴忠市]]，占宁夏回族人口的83.2％。寧夏回族据说是[[元朝|元朝]]跟随[[蒙古|蒙古]]大军東征西伐的三、四十万[[色目人|色目人]]的后裔，也融入了很多漢族成分，其定居宁夏的历史约有700多年。

[[第五次全國人口普查|第五次全國人口普查]]结果：2000年11月1日，总人口548.64万人，性别比（以女性为100）105.26。其中，汉族359.06万人，占总人口比重为65.45%；回族186.24万人，占总人口比总为33.95%。（源自《宁夏统计年鉴-2008》）截止到2017年末，宁夏回族自治区的总人口为682万人。

由于中國官方對[[漢族|漢族]]的[[計劃生育政策|計劃生育政策]]，占人口大多數的[[漢族|漢族]]的生育率小幅下降。但因为中國少数民族在生育等政策上的優惠，中国的[[回族|回族]]人口的增长速度几乎等同于全國平均增长速度，致使其在总人口中的比重少许增大。宁夏回族自治区1997年与1949年比较，[[回族|回族]]人口增长3.9倍，同期[[汉族|汉族]]人口增长为3.7倍。回族人口在总人口中的比重由约32％上升到约34％；其他少数民族人口增长25倍，在总人口中的比重由0.08％上升到0.49％<ref>[http://www.nx.xinhuanet.com/special/no1/2006-03/03/content_6377154.htm 宁夏人口情况] {{webarchive|url=https://web.archive.org/web/20100628202606/http://www.nx.xinhuanet.com/special/no1/2006-03/03/content_6377154.htm |date=2010-06-28 }}</ref>。

== 经济 ==
{{main|宁夏经济}}

根据[[中国国内生产总值核算#数据类型|初步核算]]的结果，2010年，[[宁夏回族自治区地区生产总值|宁夏GDP总量]]达到1,643.41亿元，位居[[中华人民共和国省级行政区地区生产总值列表|全国第廿九位]]，比上年增长13.5％。分产业看，第一产业完成增加值160.28亿元，增长7.0％；第二产业完成增加值833.16亿元，增长16.0％，第三产业完成增加值649.97亿元，增长11.6％。按年平均人口计算，人均GDP达到26860元，居全国第十七位<ref>宁夏回族自治区2010年GDP初步核算数：来源于[http://www.nxtj.gov.cn/tjgb/201103240013.htm 2010年宁夏回族自治区国民经济和社会发展统计公报] {{webarchive|url=https://web.archive.org/web/20120118195111/http://www.nxtj.gov.cn/tjgb/201103240013.htm |date=2012-01-18 }}</ref>。由於天然條件惡劣，宁夏擁有著名貧困區[[西海固|西海固]]。

与其他[[中国一级行政区|省市区]]的[[中华人民共和国经济|经济发展]]相比，宁夏属于中等发展速度的省级行政区；二产业所占比重较大；人均GDP属于中等偏后。1979-2007年29年，按绝对数比较，2007年[[宁夏回族自治区地区生产总值|宁夏GDP总量]]为1978年的64.17倍；按不变价格推算（以同期全国平均物价水平为基准），GDP年平均增长率9.5％，低于同期全国9.8％的平均水平，年均增长率次于[[山西省地区生产总值|山西]]、[[安徽省地区生产总值|安徽]]等15个省份居16位。1995年，宁夏GDP总量正式超过青海稳居[[中国省级行政区历年地区生产总值列表|第29位]]。GDP总量的发展，1988年突破50亿元，1993年突破100亿元，2004年突破500亿元，2008年突破1000亿元。人均GDP的发展，1988年首次突破1千元（全国1987年突破1千元），2000年突破5千元（全国1995年突破5千元），2005年突破1万元（全国2003年突破1万元）。

2007年，[[宁夏回族自治区地区生产总值|宁夏GDP总量]]（[[中国国内生产总值核算#数据类型|最终核实数]]）达到834.16亿元，与上年相比增幅达到13.9％，GDP占全国总量的0.31％；其中[[第一产业|第一产业]]97.9亿元，占GDP比重11.74％；[[第二产业|第二产业]]420.28亿元，占50.38％；[[第三产业|第三产业]]315.98亿元，占37.88％；人均GDP为13,743元，次于[[陕西省地区生产总值|陕西]]、[[青海省地区生产总值|青海]]等22[[中国一级行政区|省市区]]居第23位<ref>2007年度[[中国一级行政区|各省级行政区]]GDP数据来源于[http://219.235.129.54/cx/table/table.jsp 国家统计数据库，《2007年地区生产总值（摘要版）》] {{webarchive|url=https://archive.is/20080510120443/http://219.235.129.54/cx/table/table.jsp |date=2008-05-10 }}</ref>。经济结构中，二、三产业占有较大比重。以2007年为例，二产业占GDP比重次于[[江西省地区生产总值|江西]]、[[内蒙古自治区地区生产总值|内蒙]]等14[[中国一级行政区|省区]]居全国第15位，三产业占GDP比重次于[[云南省地区生产总值|云南]]、[[甘肃省地区生产总值|甘肃]]等17省居第18位。

== 行政区划 ==
截至2017年7月，宁夏回族自治区共有5个[[地级市|地级市]]；以下分为22个县级行政区，包括9个[[市辖区|市辖区]]、2个[[县级市|县级市]]、11个[[县|县]]；以下再分为232个乡级行政区，包括41个[[街道_(行政区划)|街道]]、98个[[镇|镇]]、93个[[乡|乡]]。
