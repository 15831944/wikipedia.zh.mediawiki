'''宁波至余姚城际铁路'''是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]]的一条[[市郊铁路|市郊铁路]]，是[[发改委|发改委]]市域（郊）铁路第一批示范项目之一<ref>{{Cite web |url=http://www.ndrc.gov.cn/zcfb/zcfbtz/201706/t20170628_852798.html |title=关于促进市域（郊）铁路发展的指导意见（发改基础〔2017〕1173号） |accessdate=2018-03-16 |author=国家发展改革委 |date=2017-06-20}}</ref>。线路规划起自[[宁波东站|宁波东站]]，初近期宁波-余姚段利用既有[[萧甬铁路|萧甬铁路]]抵达[[余姚站|余姚站]]，运营长度48.7千米<ref>{{Cite web |url=http://zjnews.zjol.com.cn/zjnews/nbnews/201703/t20170317_3327141.shtml |title=好消息！宁波至余姚城际铁路即将开通运营 |accessdate=2018-09-08 |author=宁波发布  |date=2017-03-17}}</ref>，全程耗时35分钟。二期规划从[[蜀山站|蜀山站]]西端新建36.9公里线路向北通过高铁[[余姚北站|余姚北站]]，终于[[慈溪市|慈溪市]]境内的[[杭州湾站|杭州湾站]]<ref>发改基础[2014]2865号 [http://www.sdpc.gov.cn/gzdt/201412/t20141222_657423.html 国家发展改革委关于浙江省都市圈城际铁路规划的批复]. 国家发展改革委. 2014年12月16日</ref>。自宁波站至余姚站的服务于2017年6月10日开通<ref>{{cite web|url=http://www.zj.gov.cn/art/2016/9/9/art_5526_2180696.html|title=利用即有萧甬铁路开行宁波至余姚城际工程可行性研究及勘察设计项目招标公告|publisher=浙江省交通运输厅|date=2016-09-09|accessdate=2016-11-13}}</ref><ref name="open-announce">{{cite web|url=http://news.cnnb.com.cn/system/2017/06/08/008643807.shtml|title=宁波至余姚城际铁路将于6月10日10时33分起开通试运营|publisher=中国宁波网|date=2017-06-08|accessdate=2017-06-08}}</ref><ref name="p1-open">{{cite web|url=http://news.cnnb.com.cn/system/2017/06/10/008644624.shtml|title=宁波至余姚市域列车今始发 下午起至16日市民可免费试乘|publisher=中国宁波网|date=2017-06-10|accessdate=2017-06-11}}</ref>。

== 历史 ==
宁波至余姚、慈溪的城际铁路出现在2009年确定的《宁波市余慈地区轨道交通线网规划》中。当时，宁波至余慈城际铁路规划为市域2号线，走向为宁波机场路至甬余夫公路，此后转入余姚城东路，抵达高铁[[余姚北站|余姚北站]]。此后线路进入梁周线，最后抵达[[杭州湾新区|杭州湾新区]]<ref>{{cite web|url=http://fgj.cixi.gov.cn/art/2009/11/24/art_11865_359660.html|title=《宁波市余慈地区轨道交通线网规划》通过技术会审|publisher=慈溪市发展和改革局|date=2009-11-24|accessdate=2016-11-13|archive-url=https://web.archive.org/web/20161114071020/http://fgj.cixi.gov.cn/art/2009/11/24/art_11865_359660.html|archive-date=2016-11-14|dead-url=yes}}</ref>。2012年，余姚市提出利用[[杭甬客运专线|杭甬客运专线]]开通后既有线[[萧甬铁路|萧甬铁路]]富余运力开行城际列车的想法<ref>{{cite web|url=http://news.cnnb.com.cn/system/2015/12/30/008450090.shtml|title=宁波到余姚要开城际列车了 计划将在明年开行|publisher=甬派客户端|date=2015-12-30|accessdate=2016-11-13}}</ref>。2013年公布的《浙江省都市圈城际铁路近期建设规划》中，线路改为一期利用萧甬铁路富余运力，政府购买服务，由[[上海铁路局|上海铁路局]]运营，而二期将采用与[[萧甬铁路|萧甬铁路]]相同的等级和制式新建从蜀山至杭州湾的线路<ref>{{cite web|url=http://www.zjdpc.gov.cn/art/2013/10/11/art_8_586735.html|title=浙江省都市圈城际铁路近期建设规划环境影响报告书（简本）|publisher=中铁第四勘察设计院|date=2013-09}}</ref>。2017年4月底，余姚城际动车维保基地完工，余姚站和宁波站完成改建<ref>{{cite web|url=http://www.cnnb.com.cn/dwhzlm/system/2017/04/26/008628440.shtml|title=宁波至余姚城际铁路有望5月开通 试运行票价不超12元|publisher=中国宁波网|date=2017-04-26|accessdate=2017-06-08}}</ref>，两组CRH6F动车组列车于5月28日运抵余姚<ref>{{cite web|url=http://news.cnnb.com.cn/system/2017/05/28/008640219.shtml|title=宁波至余姚城际铁路动车组"到岗" 或下月运行|publisher=甬派|date=2017-05-28|accessdate=2017-06-08}}</ref>。2017年6月10日上午10:32，余姚站开行S1205次列车，城际铁路服务开通<ref name="p1-open" />。

== 车辆和班次 ==

宁波至余姚城际铁路采用[[CRH6F|CRH6F]]城际动车组列车运营，为CRH6F列车首次在既有线开行城际列车。列车共2列，为8节编组，定员1476人，从宁波至余姚全程35分钟，使用车次为“S1XX”。车组以出生于余姚的明朝思想家[[王阳明|王阳明]]之名命名为“阳明号”<ref>{{cite web|url=http://www.chnrailway.com/html/20170613/1674436.shtml|title=“阳明号”城际列车首发|publisher=中华铁道网|date=2016-09-09|accessdate=2017-06-13}}</ref>。开通初期，受限于萧甬铁路宁波至庄桥区间的运力限制，每日开行8对列车<ref name="eastday">{{cite web|url=http://news.eastday.com/eastday/13news/auto/news/csj/20170316/u7ai6602777.html|title=好消息！宁波至余姚城际铁路即将开通运营|publisher=宁波发布|date=2017-03-16|accessdate=2017-03-16}}</ref>，上午6:04至10:05间开行4对列车，晚间16:01至20:05间开行4对列车<ref name="open-announce" />。

== 车票 ==
宁波至余姚城际铁路运营初期全程票价10元，票价低于高铁和普通旅客列车。车票不通过[[中国铁路客户服务中心|中国铁路客户服务中心]]网站出售，可在车站使用自动售票机和人工售票购票，也可使用[[宁波市民卡|宁波市民卡]]、[[甬城通|甬城通]]、余姚公交卡等智能卡刷卡进站。普通卡可享受9.5折优惠，学生卡、高龄老人卡可享受5折优惠，高龄老人也可凭有效证件进站购买打折票。离休干部、残疾军人、因公致残人民警察、重度残疾人可持卡或凭证件免费乘车。检票时间与中国铁路相同，开车前15/20分钟开始检票，开车前3分钟停止检票<ref name="open-announce" />。

== 停靠车站 ==
宁波至余姚城际铁路一期共设五座车站，除丈亭站为废站再建外均为既有站。开通时只运营宁波至余姚直达列车<ref name="eastday" />。
