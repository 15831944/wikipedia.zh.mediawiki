'''宁波至奉化城际铁路'''，又名'''宁波都市圈城际铁路S3线'''，是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]]建设中的一条[[市郊铁路|市郊铁路]]。奉化线起自[[宁波轨道交通3号线|宁波轨道交通3号线]][[高塘桥站|高塘桥站]]，终于[[奉化区|奉化区]]境内的[[金海路站_(宁波市)|金海路站]]，全长21.808公里<ref name="route">{{cite web|url=http://www.chinanews.com/sh/2014/07-09/6368195.shtml|title=宁波至奉化城际铁路走向基本确定 力争明年开工|publisher=宁波日报|date=2014-07-09|accessdate=2014-09-14|archive-url=https://web.archive.org/web/20140710031428/http://www.chinanews.com/sh/2014/07-09/6368195.shtml|archive-date=2014-07-10|dead-url=no}}</ref>。线路由[[宁波轨道交通集团|宁波轨道交通集团]]负责建设和运营，于2015年12月30日开工建设<ref name="begin-construction">{{cite web|url=http://news.cnnb.com.cn/system/2015/12/30/008450047.shtml|title=宁波—奉化城际铁路开工 计划2019年6月建成|publisher=甬派客户端|date=2015-12-30|accessdate=2015-12-30|archive-url=https://web.archive.org/web/20160401013554/http://news.cnnb.com.cn/system/2015/12/30/008450047.shtml|archive-date=2016-04-01|dead-url=no}}</ref>，计划与宁波轨道交通3号线同步贯通运营<ref>{{cite web|url=http://news.ifeng.com/a/20140829/41776835_0.shtml|title=宁波至余姚 慈溪 奉化 3条城际铁路获国家环保部批复|publisher=都市快报|date=2014-08-29|accessdate=2014-09-14|archive-url=https://web.archive.org/web/20141218144810/http://news.ifeng.com/a/20140829/41776835_0.shtml|archive-date=2014-12-18|dead-url=no}}</ref>。同时，线路预留了拆分独立运营的条件<ref name="env-final">中铁第四勘察设计院集团有限公司. 宁波至奉化城际铁路工程环境影响报告书（全文公示版）, 2015年7月</ref>。

== 线路走向 ==
宁波至奉化城际铁路起点为宁波轨道交通3号线[[高塘桥站|高塘桥站]]，以地下线形式向南，下穿[[宁波绕城高速公路|宁波绕城高速公路]]后转为高架线，进入姜山镇环镇北路，此后折向东转入雁湖路，经过鄞州工业园和甬江村，至方桥后转入东环路西侧向南，在跨越葭浦江后转向路中设置，最终到达金海路。线路全长21.808公里，其中地下线0.784公里，高架线20.624公里，过渡段长0.402公里<ref name="route" /><ref name="SJFH01">{{cite web|url=http://www.bidding.gov.cn//gcjs1/57271.htm|title=宁波至奉化城际铁路工程设计总体总包（SJFH01标段）（设计）招标公告|publisher=宁波市公共资源交易中心|date=2015-02-10|accessdate=2015-02-10|deadurl=yes|archiveurl=https://web.archive.org/web/20150224202053/http://www.bidding.gov.cn//gcjs1/57271.htm|archivedate=2015-02-24}}</ref>。方桥至大成四站及前后区间与[[机场路_(宁波)|宁波机场快速路]]南延段共构<ref name="fh-sec-start">{{cite web|url=http://nb.zjol.com.cn/system/2017/06/13/021534287.shtml|title=二层跑城铁三层跑汽车宁波至奉化快速路开建|publisher=浙江在线|date=2017-06-13|accessdate=2017-06-15|archive-url=https://web.archive.org/web/20170905030812/http://nb.zjol.com.cn/system/2017/06/13/021534287.shtml|archive-date=2017-09-05|dead-url=no}}</ref>。

远期奉化线将从方桥站拆分。方桥站以北的线路将被纳入3号线，维持80公里最高时速不变。而方桥站以南线路将从方桥站北延，经过[[宁波轨道交通2号线|2号线]][[栎社站|栎社站]]，终于[[宁波轨道交通5号线|5号线]][[金房站|金房站]]。南端也将从金海路站继续南延，经过[[奉化站|奉化火车站]]后向南抵达宁海境内。南段时速届时将提高到120公里<ref name="env-final" />。另外，[[蒋家站|蒋家站]]规划增设[[溪口|溪口]]的支线<ref name="溪口支线">{{cite web|title=《奉化市公共交通规划》规划草案公示|url=http://www.fh.gov.cn/zwgk/ghbg/ghjd/201412/t20141212_140234.html|accessdate=2017-08-18|archive-url=https://web.archive.org/web/20180706152657/http://www.fh.gov.cn/zwgk/ghbg/ghjd/201412/t20141212_140234.html|archive-date=2018-07-06|dead-url=yes}}</ref>。

== 历史 ==
宁波至奉化城际铁路最初是作为宁波轨道交通线网的一部分进行规划的。2005年公布的《奉化市市域总体规划》中包含了轨道交通规划，该规划接驳3号线，包含前往城区和溪口镇的一主一支两条线路<ref>{{cite web|url=http://www.fhplan.gov.cn/contents/178/1526.html|title=奉化市城市总体规划(奉化市市域总体规划)公开征询意见|publisher=奉化市人民政府|date=2005-01|accessdate=2015-02-25|deadurl=yes|archiveurl=https://web.archive.org/web/20150224194435/http://www.fhplan.gov.cn/contents/178/1526.html|archivedate=2015-02-24}}</ref>。在2006年公布的《宁波轨道交通线网控制性详细规划》线路总平面图中，也标示了通往奉化，接驳3号线的城际轨道线路<ref>{{cite web|url=http://hbj.ningbo.gov.cn/UploadFiles/lan2/200692616433621.htm|title=宁波市城市快速轨道交通建设规划环境影响评价公示|publisher=宁波市环境保护科学研究设计院|date=2006-09-20|accessdate=2015-02-25|deadurl=yes|archiveurl=https://web.archive.org/web/20140314205724/http://hbj.ningbo.gov.cn/UploadFiles/lan2/200692616433621.htm|archivedate=2014年3月14日|df=}}</ref>。2013年，宁波至奉化城区的城际铁路被纳入《浙江省都市圈城际铁路近期建设规划》，上报国家发改委审批，并于2014年12月获得批复<ref>国家发展改革委, 国家发展改革委关于浙江省都市圈城际铁路规划的批复, 发改基础[2014]2865号.</ref>。在此期间，奉化线的走向和建设时序得到了进一步明确。线路确定起点为宁波轨道交通3号线高塘桥站，终点为金海路站，全长21.8公里，并预留进一步延伸的条件<ref>{{cite web|url=http://fh.cnnb.com.cn/system/2015/02/12/011143963.shtml|title=宁波至奉化城际铁路4月底完成总体设计|publisher=奉化日报|date=2015-02-12|accessdate=2015-02-25|deadurl=yes|archiveurl=https://web.archive.org/web/20150224193555/http://fh.cnnb.com.cn/system/2015/02/12/011143963.shtml|archivedate=2015-02-24}}</ref>。奉化线工程可行性研究报告于2015年11月获得浙江省发改委批复，为浙江省都市圈城际铁路规划中首条获批的线路<ref>{{cite web|url=http://www.zjdpc.gov.cn/art/2015/11/11/art_802_1565818.html|title=我省首条都市圈城际铁路项目工可报告获得批复|publisher=浙江省发展和改革委员会|date=2015-11-11|accessdate=2015-11-12|archive-url=https://web.archive.org/web/20190126110637/http://www.zjdpc.gov.cn/art/2015/11/11/art_802_1565818.html|archive-date=2019-01-26|dead-url=no}}}</ref>。

2015年12月30日，宁波至奉化城际铁路先行节点，位于奉化区境内的金海路站开工建设<ref name="begin-construction" />。2017年1月，位于鄞州区境内的甬江村站开工<ref>{{cite web|url=http://yz.cnnb.com.cn/system/2017/01/05/011508831.shtml|title=宁波至奉化城际铁路,TJFH04标段开工|publisher=鄞州新闻网|date=2017-01-05|accessdate=2017-06-15|archive-url=https://web.archive.org/web/20180522111811/http://yz.cnnb.com.cn/system/2017/01/05/011508831.shtml|archive-date=2018-05-22|dead-url=no}}</ref>。2017年6月13日，位于奉化区境内的机场快速路共线段开工建设<ref name="fh-sec-start"/>。

== 车站 ==
宁波至奉化城际铁路初期规划设10座车站，其中高塘桥站为地下站，其余车站均为高架站。

== 车辆 ==
{{main|宁波轨道交通3号线#车辆}}
宁波至奉化城际铁路采用6辆编组B型地铁车辆。线路初期目标时速为80公里，远期拆分后方桥至金海路时速为120公里<ref name="env-final" />。初期列车由[[宁波中车产业基地|宁波中车产业基地]]生产，采用[[中车时代电气|中车时代电气]]牵引，共配车21列（不含3号线车辆）<ref name="application-table">{{cite web|url=http://www.zjinvest.com.cn/newshow.asp?id=1424&mnid=1434 |title=杭州中车电气成功中标宁波地铁牵引系统项目 |publisher=浙江省经济建设投资有限公司 |date=2016-09-13 |accessdate=2017-06-11 |archiveurl=https://archive.is/20170611163022/http://www.zjinvest.com.cn/newshow.asp?id=1424&mnid=1434 |archivedate=2017-06-11 |deadurl=yes }}</ref>。

== 附属设施 ==
=== 车辆基地 ===
宁波至奉化城际铁路与[[宁波轨道交通3号线|宁波轨道交通3号线]]共享首南车辆段，位于[[鄞州区|鄞州区]]首南街道境内，宁波绕城高速公路、高阳路和明光路合围的地块内，承担列车定期和临时修理以及部分列车的停放和运用任务。线路南端设有奉化停车场，位于[[奉化区|奉化区]]金海路站后东南地块内，承担列车停放和运用任务<ref name="env-final" />。列车的大修和架修任务则由1号线[[天童庄车辆段|天童庄车辆段]]承担。远期拆分后，奉化停车场将扩建为具备定修能力的车辆段，线路北端将增设车何村停车场，而列车大修、架修任务则改由5号线经堂庵跟车辆段承担<ref name="env-simplified">中铁第四勘察设计院集团有限公司. 宁波至奉化城际铁路工程环境影响报告书（简本）, 2015年4月</ref>。

=== 主变电所 ===
宁波至奉化城际铁路在南渡站附近设置一座主变电所，连接奉化220千伏广济变电所，设置两台110/35千伏主变压器<ref name="env-final" />。

=== 控制中心 ===
宁波至奉化城际铁路与宁波轨道交通3号线一同纳入宁波市轨道交通控制中心进行控制<ref name="SJFH01" />。
