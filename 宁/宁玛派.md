'''寧瑪派'''（{{bo|z=རྙིང་མ་པ་|t=Nyingma|w=rnying ma pa}}），是[[藏傳佛教|藏傳佛教]]流派之一，由於該派的僧人均戴[[紅色|紅色]][[僧帽|僧帽]]，俗稱為「紅帽派」或「紅教」。

寧瑪藏語音譯，是古、舊的意思；在各派中歷史最久，教理從公元8世紀時傳下來，形成於公元11世紀。尊奉[[蓮花生大士|蓮花生大士]]為其始祖，創派人為[[釋迦迥乃|釋迦迥乃]]。其傳承最高修法為「[[大圓滿法|大圓滿法]]」。

== 教法 ==
它所傳承的教法主要是前弘期傳譯的密法，故稱為「舊派」。寧瑪派的教法傳承，主要有：
*一、遠傳佛語傳承
*二、近傳伏藏傳承


寧瑪派的教法全部包括在九乘三部裏，九乘包括
#顯三乘〈一聲聞，二縁覺，三菩薩〉，显、密都必须学习的共三乘；
#[[外密三乘|外密三乘]]（四事部、五行部、六瑜伽部）；
#[[內密三乘|內密三乘]](七瑪哈瑜伽、八阿努瑜伽，九阿底瑜伽)。
在以上諸傳承教法中，寧瑪派最注重的教法是[[大圓滿|大圓滿]]，特指[[內密三乘|無上阿底瑜伽口訣部]]。

== 儀式與宗教活動 ==
紅教修士有二种，红僧与白僧。前者多数是[[出家人|出家人]]，后者多数是[[居士|俗家]]的[[陀羅尼|密咒士]]（雅巴）与[[瑜伽士|瑜伽士]]。

== 寺廟 ==
自[[朗達瑪|朗達瑪]]滅佛後至十七世紀前，寧瑪派的僧人多散居於山中，沒有較有規模的寺廟。十七世紀時建立了六大主寺，再由主寺分出較小的子寺。在歷史上，寧瑪派並沒有政教合一的領袖。以各主寺的住持為意見領袖。二十世紀時，流亡的藏人在印度重建了六大祖寺。

*[[多吉扎寺|多吉扎寺]]，位于[[上藏|上藏]][[山南地區|山南地區]][[貢嘎縣|貢嘎縣]]的雅鲁藏布江江北岸
*[[敏珠林|敏珠林]]，Mindroling Monastery，位于上藏山南地區的[[扎囊縣|扎囊縣]]，建立於西元1676年。
*[[佐钦寺|佐钦寺]]，Dzogchen Monastery，位于[[中藏|中藏]][[康區|康區]][[德格|德格]]东北，又被称为竹慶寺、卓千寺，第一世佐欽法王貝瑪仁增(1625–1697)於西元 1685 在四川省甘孜藏族自治州德格縣創建。
*[[雪謙寺|雪謙寺]]，Shechen Monastery，位于中藏康區德格，又被称为協慶寺，建立於西元1695年。
*[[噶陀寺|噶陀寺]]，Katok Monastery，位于[[下藏|下藏]]康區[[白玉縣|白玉縣]]，西元1656 重建。
*[[白玉寺|白玉寺]]，Palyul Monastery，位于下藏康區白玉縣，西元1665年白玉的開山祖師仁真更絨謝瑞創建了白玉寺，名為「白玉尊勝菩提園寺」。

除了六大祖寺外，西藏山南地區的扎囊縣的[[桑耶寺|桑耶寺]]為寧瑪派的代表性寺院。由於宁玛派没有领导全派的中心寺院和政治勢力，[[桑耶寺|桑耶寺]]曾經被[[薩迦派|薩迦派]]統治。統治時期，對該寺進行過修葺，並派遣僧人住寺，此後就形成了寧瑪、薩迦兩派共處一寺的局面。其他寧瑪派重要的寺廟還有持有[[龙钦宁提|龙钦宁提]]的[[多智钦寺|多智钦寺]]。

== 祖師 ==
大圓滿傳承歷史中，大圓滿的傳承是：
*法身佛[[普賢王如來|普賢王如來]]、報身佛[[金剛薩埵|金剛薩埵]]、[[五方佛|五方佛]]等。
*[[噶拉多傑|噶拉多傑]] (Garab Dorje－義為極喜金剛)，將大圓滿法由佛界帶到俗世，是人間的第一個上師。
*[[文殊友|文殊友]] (Manjushrimitra－又翻譯為妙吉祥友、藏音蔣巴些寧)。
*[[師利星哈|師利星哈]] (Sri Singha－義為吉祥獅子)。
*[[蓮華生大士|蓮華生大士]]與[[無垢友|無垢友]] (Vimalamitra)，將大圓滿法帶入西藏，流傳至今。

*[[堪布阿旺巴桑|堪布阿旺巴桑]] (1879－1941)。
*[[蔣揚欽哲確吉羅卓仁波切|蔣揚欽哲確吉羅卓仁波切]] (Jamyang Khyentse Chökyi Lodrö，1893－1959)。
*[[夏札桑吉多傑仁波切|夏札桑吉多傑仁波切]] (Chatral Sangye Dorje Rinpoche，1913－2015)。
*[[頂果欽哲|頂果欽哲]] (1910－1991)。
