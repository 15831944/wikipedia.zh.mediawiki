'''宁波大学站'''是[[浙江省|浙江省]][[宁波市|宁波市]]一座高架轨道交通车站，位于[[宁波大学|宁波大学]]主校门西侧，服务于宁波大学及周边地区。车站于2015年9月26日启用<ref name="main">{{cite web|url=http://news.cnnb.com.cn/system/2010/12/23/006790463.shtml|title=2号线车站设计人性化而有宁波味 效果图抢先看|author=陈锦源|publisher=中国宁波网|date=2010-12-23|accessdate=2011-01-21}}</ref>。

== 位置 ==
宁波大学站位于[[江北区_(宁波市)|江北区]]宁镇公路中央，[[宁波大学|宁波大学]]主校门西侧<ref name="main" />。

== 车站形制 ==
宁波大学站外包长134.1米，宽28米，高24.08米，设2个出口，分别位于[[宁镇公路|宁镇公路]]两侧。地面为出口，地上二层为站厅，地上三层为[[侧式车站|直线侧式车站]]，线距4.2米，站台宽度在13.2米到21.0米之间。附属站房位于车站南侧<ref name="main" />。

== 车站装饰 ==
宁波大学站为2号线文化站之一。车站文化墙主题为“情系桑梓”。文化墙共两幅，均采用了重叠手法，分别表现了[[王宽诚|王宽诚]]、[[包玉刚|包玉刚]]、[[赵安中|赵安中]]等宁波帮人士帮助宁波发展以及[[邓小平|邓小平]]与包玉刚关心宁波大学创建两个主题<ref>{{cite web|url=http://www.nbmetro.com/news_dt.php?info/961|title=2号线一期6大车站文化墙亮相|publisher=宁波轨道交通集团|date=2015-09-23|accessdate=2015-09-27}}</ref>。
