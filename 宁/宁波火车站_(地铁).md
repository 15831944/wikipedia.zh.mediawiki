'''宁波火车站'''，建设时曾命名为'''铁路南站站'''，是[[浙江省|浙江省]][[宁波市|宁波市]]一座地下轨道交通车站，服务于铁路[[宁波站|宁波站]]，同时为[[宁波轨道交通2号线|宁波轨道交通2号线]]和[[宁波轨道交通4号线|4号线]]的换乘枢纽<ref name="main">{{cite web|url=http://news.cnnb.com.cn/system/2010/12/23/006790463.shtml|title=2号线车站设计人性化而有宁波味 效果图抢先看|author=陈锦源|publisher=中国宁波网|date=2010-12-23|accessdate=2011-01-21}}</ref>。2号线车站于2015年9月26日启用，4号线车站计划于2020年开通。

== 车站形制 ==
宁波火车站站位于[[海曙区|海曙区]]，铁路[[宁波站|宁波站]]地下。站房地下三层为轨道交通2号线站台，位于站房垂直正中位置，长330米，宽21.1米，埋深22.6米，建筑面积1.87万平方米。北广场地下二层为轨道交通4号线站台，位于北广场地下车库下方，长340米，宽21.1米，埋深16.6米，建筑面积1.37万平方米。两站台均为宽岛式站台，形成“T”形换乘<ref>关于铁路宁波站改建工程配套项目轨道交通2、4号线铁路南站站项目建议书的复函, 甬发改交通函〔2009〕124号.</ref>。

== 车站装饰 ==
[[File:Ningbo_Railway_Station_metro_platform,_2015-07-25_03.JPG|thumb]]
宁波火车站为2号线装修重点站之一。车站契合铁路宁波站“天一生水”的主题，装修主题为海洋，使用蓝色作为主色调，同时采用水滴作为吊顶装饰<ref name="decoration-1">{{cite web|url=http://auto.zjol.com.cn/system/2014/12/17/020415591.shtml|title=宁波地铁2号线栎社机场站亮相 装修玩起中国风|publisher=钱江晚报|date=2014-12-17|accessdate=2014-12-17}}</ref>。车站主题雕塑名为“八方辐辏”。雕塑材质为不锈钢，通过交错的圆弧表现四通八达的交通网络，同时地面装饰有宁波市花山茶花，寓意宁波火车站为宁波的交通枢纽<ref>{{cite web|url=http://www.nbmetro.com/news_dt.php?info/961|title=2号线一期6大车站文化墙亮相|publisher=宁波轨道交通集团|date=2015-09-23|accessdate=2015-09-27}}</ref>。

== 交通接驳 ==
宁波火车站接驳铁路宁波站，同时接驳宁波汽车南站、公交宁波火车站和公交宁波火车站北广场站<ref name="query" />。
