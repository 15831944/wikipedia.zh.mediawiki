==历史==
===民国创立===
[[光绪|清光绪]]十年（1884年），[[洋务运动|洋务运动]]领导者之一、[[宁绍兵备道|宁绍兵备道]][[薛福成|薛福成]]在在宁波城西衙署内以独秀山基础扩建了后乐园（现址为[[中山广场_(宁波)|宁波中山公园]]），后乐园的修建旨在为教育民众、启迪思想提供场所，其中设有揽秀堂藏书楼<ref name="history1">{{Cite news |url=http://daily.cnnb.com.cn/nbwb/html/2012-03/18/content_443515.htm |title=宁波中山公园的百年沧桑 |author=孙武军 |date=2012-03-18 |work=宁波晚报 |page=A07 |accessdate=2018-07-18 |archiveurl=https://web.archive.org/web/20180719001947/http://daily.cnnb.com.cn/nbwb/html/2012-03/18/content_443515.htm |archivedate=2018-07-19 |deadurl=no }}</ref>。清光绪二十年（1894年），[[吴引荪|吴引荪]]继任宁绍兵备道道员，将揽秀堂藏书楼改称为崇实书院藏书楼，并通过购书和捐赠的方式增加了藏书楼的藏书量<ref name="history2">{{Cite book |title=宁波市海曙区志 |date=2014-07 |publisher=浙江人民出版社 |isbn=978-7-213-06170-7 |chapter=第二十七编 文化}}</ref>{{rp|1535}}。1905年，崇实书院改为宁波教育会。1913年，[[宁波府|宁波府]]所属六县的公会在后乐园建造了三座西式楼房，将薛福成、吴引荪以及后来各时期添置的全部书籍藏于其中，并将三座藏书楼命名为“薛楼”。在此期间，随着人们求知需求的增加，社会上自发形成了四明学会图书馆、四明少年图书馆、宁波少年图书馆、青年会藏书室和通俗图书馆等具有平等、开放性质的图书馆<ref name="history3">{{Cite journal |title=从传统藏书楼向现代图书馆的转变——宁波公共图书馆创建和发展探微 |author=华东杰 |journal=黑龙江史志 |issue=13 |year=2014}}</ref>。

1926年，四明学会图书馆、宁波少年图书馆、通俗书报社三个团体在后乐园召开联席会议，共同商议创建宁波公共图书馆的事宜。1927年，[[國民革命軍|國民革命軍]][[國民革命軍北伐|北伐]]胜利后宁波设市，5月，宁波市长[[罗惠侨|罗惠侨]]决定将薛楼改为宁波市市立图书馆。宁波市市立图书馆于9月正式开馆，由张汝钊任首任馆长，该馆成为宁波历史上第一个真正意义上的公共图书馆，也成为了宁波市图书馆的前身<ref name="history3"/>。1931年宁波市建制被撤销，宁波市立图书馆与鄞县县立图书馆合并，图书馆更名为鄞县县立图书馆<ref>{{Cite book |title=宁波市地名志|author=宁波市地名委员会|publisher=中华书局|location=宁波|chapter=第一卷 概述}}</ref>{{rp|22}}<ref name="history4">{{Cite web |url=https://www.nblib.cn/art/2017/6/1/art_2061_97118.html |title=宁波市图书馆建馆九十周年征文启事 |accessdate=2018-07-19 |author=宁波市图书馆 |date=2017-06-01 |publisher=宁波市文化信息中心 |archiveurl=https://web.archive.org/web/20180719001946/https://www.nblib.cn/art/2017/6/1/art_2061_97118.html |archivedate=2018-07-19 |deadurl=no }}</ref>。

===当代发展===
1949年5月25日[[中国人民解放军|中国人民解放军]]攻克宁波后，宁波市军事管制委员会成立。军管会文教部于6月接管了鄞县县立图书馆。接管时馆内共有馆藏图书1.5万余册，约有一半为古籍，藏书经宁波市文教局组织整理，其中的可用图书移交至宁波市人民文化馆阅览室，古籍则移藏于[[天一阁|天一阁]]古物陈列所<ref name="history5">{{Cite web |url=http://www.zjwh.gov.cn/fwxx/tswh/1nbs/30046.htm |title=宁波市图书馆 |accessdate=2018-07-19 |author=浙江省文化厅 |date=2015-07-08 |publisher=浙江省文化信息中心 |archiveurl=https://web.archive.org/web/20150718193558/http://www.zjwh.gov.cn/fwxx/tswh/1nbs/30046.htm |archivedate=2015-07-18 |deadurl=no }}</ref>。宁波市人民文化馆书报阅览室在这一时期继承了宁波公共图书馆的职能<ref name="history4"/>。

1953年，宁波市图书馆在[[中山广场_(宁波)|中山公园]]旧馆址重新成立，馆舍面积660平方米，藏书8946册。藏书中大部分为人民文化馆阅览室移交回的前鄞县县立图书馆藏书。1962年9月9日，图书馆迁址至[[海曙区|海曙区]]药行街桑园巷的前仁慈堂旧址，迁址后馆舍面积扩增为1400平方米，设有采编室、外借处、古籍部、少年儿童阅览室、流通辅导组。截至[[文化大革命|文化大革命]]前，图书馆共藏书159249册，其中古籍约73000册<ref name="history3"/>。文革开始后，1969年10月，馆舍被占为市文化系统“[[五七干校|斗、批、改干校]]”，图书馆随之关闭。1970年2月21日，宁波市图书馆建制撤销并与宁波市文管会合并组成宁波市图书文物馆。文革结束后，宁波市图书馆又于1978年恢复了建制<ref name="history5"/>。

1988年11月宁波市图书馆迁入由旅港宁波籍人士、世界船王[[包玉剛|包玉剛]]捐资修建的永丰路新址，新址面临[[姚江|姚江]]，馆舍面积8187平方米。永丰路新址于1989年5月1日正式对外开放<ref name="history6">{{Cite book |title=宁波图书馆志 |publisher=宁波出版社 |year=2002 |isbn=978-7-80602-703-5 |location=宁波 |chapter=第二卷 公共图书馆事业发展状况}}</ref>{{rp|59}}。2001年宁波市政府投资对图书馆进行扩建装修，市馆舍总面积达到了目前的12000平方米<ref name="history5"/>。发展至今，宁波市图书馆现有图书外借室、少儿借阅室、报刊阅览室、古籍地方文献阅览室、天一音乐馆、外文阅览室等十余个服务窗口及采编部、辅导部、技术部、办公室等部门<ref name="Intro"/>。2013年，宁波市政府在专题会议纪要中提出在[[宁波东部新城|宁波东部新城]]为宁波市图书馆新建馆舍，新馆于2018年11月竣工验收实现部分开放，并于12月28日开馆<ref name="newlib-open"/>。新馆舍投用后承担着宁波市公共图书馆服务体系的中心馆的职能<ref name="new1">{{Cite web |url=https://www.nblib.cn/col/col2541/index.html |title=宁波市图书馆新馆建设项目概况 |accessdate=2018-07-19 |author=宁波市图书馆 |archiveurl=https://web.archive.org/web/20160305003044/http://www.nblib.cn/col/col2541/index.html |archivedate=2016-03-05 |deadurl=no }}</ref><ref>{{Cite web |url=http://zjnews.zjol.com.cn/zjnews/nbnews/201809/t20180907_8212311.shtml |title=18个项目年内基本建成，宁波市图书馆新馆项目12月试开馆 |accessdate=2018-10-25 |date=2018-09-07 |publisher=宁波晚报}}</ref>。

==馆舍==
宁波市图书馆现运营有两个主要馆舍，一为由包玉刚捐资建造于1989年投用的永丰路馆舍，一为位于[[宁波东部新城|宁波东部新城]]于2018年投用的新建馆舍。
=== 永丰馆 ===
宁波市图书馆永丰馆位于海曙区永丰路135号，为纪念捐赠500万港元建设该馆舍的著名旅港宁波籍人士、世界船王包玉刚而被命名为'''宁波市包玉刚图书馆'''<ref name="history6"/>{{rp|59}}{{noteTag|1=目前该名称已不再见于官方报道，但也从未明确停止使用。}}。馆舍于1988年兴建，1989年5月1日开馆投入使用，建成投用时建筑面积8187平方米，由五层的书库、四层的阅览楼、三层的多功能厅、二层的门厅组成，其中书库楼1508平方米，阅览楼1600平方米，多功厅楼1100平方米，门厅楼516平方米。建成后，馆内的计算机网络等设备和消防、电气等设施先后进行过升级或添置<ref name="history6"/>{{rp|62-63}}。

2001年宁波市政府投资2000多万元再次对图书馆进行了较大规模的扩建<ref name="history3"/>，目前馆舍建筑总面积达到15368平方米。其中读者借阅室7067平方米，书库4140平方米，报告厅及会议室825平方米<ref name="2016Annual">{{Cite web |url=https://www.nblib.cn/module/download/downfile.jsp?classid=0&filename=170814155613846.pdf |title=2016宁波市图书馆年报 |accessdate=2018-07-19 |author=宁波市图书馆 |date=2017-02-27 |publisher=宁波市文化信息中心 |archiveurl=https://web.archive.org/web/20180719001946/https://www.nblib.cn/module/download/downfile.jsp?classid=0&filename=170814155613846.pdf |archivedate=2018-07-19 |deadurl=no }}</ref>。

=== 新馆 ===
[[File:Ningbo_Library_East_New_Town_20180907-2.jpg|right]]
宁波市图书馆新馆位于宁波东部新城，毗邻宁波市政府、宁波行政服务中心<ref name="new1"/>，新馆建设项目于2015年8月17日开工建设<ref name="new2">{{Cite web |url=https://www.nblib.cn/art/2015/6/26/art_2542_70319.html |title=宁波市图书馆新馆项目正式开工建设 |accessdate=2018-07-19 |author=计文元 |date=2015-06-26 |publisher=宁波市图书馆 |archiveurl=https://web.archive.org/web/20160305070128/http://www.nblib.cn/art/2015/6/26/art_2542_70319.html |archivedate=2016-03-05 |deadurl=no }}</ref>，2016年底主体结顶<ref>{{Cite web |url=https://www.nblib.cn/art/2016/11/1/art_2542_89410.html |title=市图书馆新馆主体建筑正式结顶 |accessdate=2018-07-19 |author=计文元 |date=2016-11-01 |publisher=宁波市图书馆 |archiveurl=https://web.archive.org/web/20180719001946/https://www.nblib.cn/art/2016/11/1/art_2542_89410.html |archivedate=2018-07-19 |deadurl=no }}</ref>，2018年11月30日正式竣工验收<ref name="new6">{{Cite web |url=http://news.cnnb.com.cn/system/2018/12/01/030007077.shtml |title=宁波市图书馆新馆正式竣工！一大波美图来了 |accessdate=2018-12-02 |author=记者 林旻 陈青 通讯员 周爱武 |date=2018-12-02 |publisher=中国宁波网 |archive-url=https://web.archive.org/web/20181202155113/http://news.cnnb.com.cn/system/2018/12/01/030007077.shtml |archive-date=2018-12-02 |dead-url=no }}</ref>，并于2018年12月28日开馆<ref name="newlib-open">{{Cite web |url=https://www.nblib.cn/art/2018/12/28/art_1953_115630.html |title=宁波图书馆新馆开馆 |accessdate=2019-01-01 |author=刘燕 |date=2018-12-28 |publisher=宁波市图书馆 |archive-url=https://web.archive.org/web/20190101194002/https://www.nblib.cn/art/2018/12/28/art_1953_115630.html |archive-date=2019-01-01 |dead-url=no }}</ref>。

2013年，宁波市政府专题会议纪要中提出在宁波东部新城为宁波市图书馆新建馆舍<ref name="new1"/>。在当年举行的宁波市图书馆新馆建筑概念设计方案国际邀请竞赛中，{{link-en|SHL建筑师事务所|Schmidt Hammer Lassen Architects|丹麦SHL建筑师事务所}}上海办事处的方案在4家中国公司和2家国外公司的方案中拔得头筹，SHL建筑师事务所上海办事处同时也是[[上海图书馆东馆|上海图书馆东馆]]设计方案的中标事务所<ref name="new3">{{Cite web |url=https://www.nblib.cn/art/2014/1/29/art_2543_49622.html |title=宁波市图书馆新馆方案设计交流会举行 |accessdate=2018-07-19 |author=周爱武 |date=2014-01-29 |publisher=宁波市图书馆 |archiveurl=https://web.archive.org/web/20160305065955/http://www.nblib.cn/art/2014/1/29/art_2543_49622.html |archivedate=2016-03-05 |deadurl=no }}</ref><ref name="new4">{{Cite web |url=http://www.shl.dk/cn/shanghai-east-library/ |title=上海图书馆东馆 |accessdate=2018-07-19 |author=SHL建筑师事务所 |archiveurl=https://web.archive.org/web/20180307071011/http://www.shl.dk/cn/shanghai-east-library/ |archivedate=2018-03-07 |deadurl=no }}</ref>。

新馆建设投资约2.49亿元，馆舍呈盒式结构共五层，其中地下一层，地上四层，建筑面积达到31405平方米，建筑高度约24米。建筑以中庭为中心，往上通过中部回廊向上来到三个巨大的开放式布置空间，连接东、西、南三幢建筑。新馆广泛采用了[[绿色建筑|绿色建筑]]的技术，包括通过利用自然光线帮助改善室内照明、利用中庭构造实现优良的自然通风、利用[[清水混凝土材质|清水混凝土材质]]在夏季和冬季帮助保持建筑恒温等措施和技术实现了[[被动式节能屋|被动式节能]]<ref name="new5">{{Cite web |url=http://www.shl.dk/cn/ningbo-new-library/ |title=宁波市图书馆新馆 |accessdate=2018-07-19 |author=SHL建筑师事务所 |archiveurl=https://web.archive.org/web/20170821080855/http://www.shl.dk/cn/ningbo-new-library/ |archivedate=2017-08-21 |deadurl=no }}</ref>。

新馆建筑内有8000平方米的开敞文化市集。新馆日期望接待量为7000至8000人，馆内将设有3000个固定自习桌。新馆一层附带夹层，除大厅、借阅和阅览区（室）外，还包括采编加工室、配送中心、自习室、咖啡厅等功能空间。二层有古籍阅览室、专业古籍修复区、恒温恒湿密集书库、展厅，并设置了东西两个休闲露台。三楼为特色馆藏区及音乐主题阅览区，其中音乐主题阅览区则包含有纸质书籍阅览室、馆藏黑胶唱片欣赏区、Hifi欣赏室、音乐沙龙及专业录音棚。四楼为乔石书房、创客空间、培训室及沙龙室。建筑地下一层与市民广场联通，并设有读者餐厅<ref name="new5"/><ref name="new6"/>。

新馆的功能定位为宁波市公共图书馆服务体系的中心馆，宁波地区文献保障中心、公共图书馆数字资源及服务中心、地方文献数字化建设中心、纸质图书采编配送中心、公共图书馆业务培训教育中心、公共图书馆服务网络发展中心<ref name="Intro"/>。

==馆藏==
1997年，宁波市图书馆建立了地方文献书库。地方文献书库中的资料内容丰富，涉及宁波的政治、经济、文化等各个领域，收藏了甬籍文人、[[宁波籍院士列表|院士]]等著名人士的著述以及宁波地方报刊、地方志等。地方文献书库是宁波市图书馆的特色馆藏之一<ref name="history6"/>{{rp|71}}。此外，图书馆还提供艺术藏本、盲人读物等特色馆藏<ref>{{Cite web |url=https://www.nblib.cn/col/col1996/index.html |title=特色馆藏 |accessdate=2018-07-19 |author=宁波市图书馆 |archiveurl=https://web.archive.org/web/20170706234616/http://www.nblib.cn/col/col1996/index.html |archivedate=2017-07-06 |deadurl=no }}</ref>。

截止2018年底，纸质藏书255.53万册（其中古籍近8万册）、电子图书350余万种、电子期刊1.2万种，电子数据库68TB<ref name="Intro"/>。馆藏数量目前仍在不断攀升，2018年新入藏图书数量达到33.5万册<ref name="2018Annual">{{Cite web |url=https://www.nblib.cn/art/2019/2/3/art_1999_116094.html|title=宁波图书馆2018年度阅读报告 |date=2019-02-03 |accessdate=2019-02-26 |author=宁波市图书馆}}</ref>。

===馆务概况===
宁波市图书馆2018年全年进馆人数1,383,572人次，有持证读者28.7万人；共外借图书118.9万册、360余万册次，其中76585册次通过手机约书邮送方式外借；举办阅读活动770场，共4,942,967人次参加<ref name="2018Annual"/>。截至2016年底宁波市图书馆共有馆员116人，2016当年财政经费共7121.14万元<ref name="2016Annual"/>。

===服务窗口===
永丰馆和新馆两馆均各设有为读者提供借书证办理、咨询等基础服务项目的读者服务台，以及供开展音乐和影视资料欣赏与交流的天一音乐馆<ref name="service">{{Cite web |url=https://www.nblib.cn/col/col1992/index.html |title=宁波市图书馆服务窗口 |accessdate=2018-07-19 |author=宁波市图书馆 |publisher=宁波市文化信息中心 |archiveurl=https://web.archive.org/web/20190226153953/https://www.nblib.cn/col/col1992/index.html |archivedate=2019-02-26 |deadurl=no }}</ref>。

永丰馆设有三个借阅室和四个阅览室，三个借阅室分别为综合借阅室、社科借阅室和少儿借阅室，分别提供了基础藏书、中文社科图书馆和少儿图书的开架借阅；四个阅览室分别为报刊阅览室、电子阅览室、地方文献·参考阅览室、古籍·艺术阅览室，提供[[中华人民共和国|中华人民共和国]]建国以来的过刊过报、馆藏数字资源、史料志书年鉴及宁波风俗民情自然资源文献、古籍及艺术藏本的阅览服务。此外，专设有供读者自学的125座读者自修室、供活动使用的250座报告厅、提供饮食、文化和图书馆业务用品等的读者休闲吧<ref name="service"/>。

新馆设有三种借阅区、五个阅览区、两个特色书房和两个特色空间。三种阅览区分别为图书借阅区、幼儿借阅区、少儿借阅区。五个阅览区分别为报刊阅览区、古籍阅览区、电子阅览区、地方文献阅览区、外文阅览区。两种特色书房分别是：收藏来自宁波友好城市赠予的图书、画册等相关文献资料的友城书房，以及收藏原[[全国人民代表大会常务委员会委员长|全国人大常委会委员长]][[乔石|乔石]]私藏图书、手札、书画、照片及遗物的乔石书房。两个特色空间分别为：用于举办文化艺术展览活动的艺术空间，以及用于举办创意交流与实践活动的创客空间。此外还设有300座报告厅、200座多功能厅各一个。与永丰馆不同的是，新馆设有提供自助借阅服务的自助图书馆，提供盲文图书阅读服务的视障阅览室以及供预约使用的大小研习室共4间<ref name="service"/>。

===馆外服务===
宁波市图书馆除设有[[宁波大榭开发区|大榭开发区]]分馆、[[宁波高新技术产业开发区|高新区]][[梅墟街道|梅墟]]分馆、[[宁波保税区|保税区]]分馆、[[东钱湖镇|东钱湖]]分馆、[[宁波市医疗中心李惠利东部医院|李惠利东部医院]]分馆外，还利用巴士或当地空间在设立乡镇、校园、政府机关等设立流动图书馆。并免费提供图书和期刊，免费提供不定期更换图书、“你点书我买单”等一系列服务<ref>{{Cite web |url=https://www.nblib.cn/col/col1972/index.html |title=流动图书馆 |accessdate=2019-02-23 |publisher=宁波市图书馆 |archive-url=https://web.archive.org/web/20171008034422/http://www.nblib.cn/col/col1972/index.html |archive-date=2017-10-08 |dead-url=no }}</ref>。此外，图书馆还在海曙区、鄞州区、[[江北区_(宁波市)|江北区]]、[[镇海区_(宁波市)|镇海区]]、[[北仑区|北仑区]]、[[象山县|象山县]]和部分地铁站设置了一定数量的24小时自助图书馆<ref>{{Cite web |url=https://www.nblib.cn/col/col1973/index.html |title=24小时城市街区自助图书馆 |accessdate=2019-02-23 |publisher=宁波市图书馆 |archive-url=https://web.archive.org/web/20170521020524/http://www.nblib.cn/col/col1973/index.html |archive-date=2017-05-21 |dead-url=no }}</ref>，其中[[宁波轨道交通|宁波轨道交通]][[鼓楼站_(宁波市)|鼓楼站]]的月借阅量在全国200多台信用借阅柜中月借还量位居第一<ref>{{Cite web |url=http://www.nbmetro.com/news_lead.php?info/314 |title=宁波地铁鼓楼站“移动图书馆”，月借还量全国第一 |accessdate=2019-03-07 |author=宁波轨道交通资产管理部 |date=2019-02-26 |publisher=宁波轨道交通}}</ref>。
