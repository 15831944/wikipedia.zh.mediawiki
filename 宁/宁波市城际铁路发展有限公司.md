'''宁波市城际铁路发展有限公司'''是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]]一家[[市郊铁路|城际（市域）轨道交通]]建设和运营企业。宁波城际铁路发展有限公司成立于2017年3月16日，为宁波交通投资控股有限公司、余姚交通投资有限公司出资成立的国有企业，承担宁波都市圈城际铁路的投资、建设和运营职能。<ref name="official intro">{{Cite web |url=http://www.nbcjtl.com/html/Text.aspx?cid=138 |title=宁波市城际铁路发展有限公司简介 |accessdate=2018-03-16 |author=宁波市城际铁路发展有限公司}}</ref><ref>{{Cite web |url=http://zjnews.zjol.com.cn/zjnews/201706/t20170610_4199667.shtml |title=揭秘 宁波-余姚城际列车是如何做到全国首创的？ |accessdate=2018-03-16 |author=王健 |date=2017-06-10 |publisher=浙江在线}}</ref><ref>{{Cite web |url=http://www.nbcredit.net/zx/gjcx/gjcxinit.shtml?dispatch=getQydjJbxx&nbxh=MzMwMjAwMDAwMDA3OTQ1MA==&uniscid=91330200MA28Y4D90T&zch=330200000095042&qymc=宁波市城际铁路发展有限公司&zt=续存 |title=宁波市城际铁路发展有限公司基本信息 |accessdate=2018-03-16 |author=宁波市市场监督管理局 |date=2017-09-07 |publisher=宁波市企业信用网}}</ref>
