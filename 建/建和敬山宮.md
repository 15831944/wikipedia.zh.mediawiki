'''建和敬山宮'''位於[[臺灣|臺灣]][[臺東縣|臺東縣]][[臺東市|臺東市]][[建和里|建和里]]，為一間主祀[[太陽星君|太陽星君]]的[[道教|道教]][[廟宇|廟宇]]<ref name="客家委員會"/>。另同祀有[[玄天大帝|玄天大帝]]、[[觀世音菩薩|觀世音菩薩]]、[[中壇元帥|中壇元帥]]以及[[太陰娘娘|太陰娘娘]]等<ref name="客家委員會"/><ref name="教育大市集"/><ref name="農田水利入口網"/>。是[[射馬干部落|建和社區]]內居民主要的信仰中心<ref name="更生日報"/>。

== 沿革 ==
1946年，當時居住在建和社區內的一位[[原住民|原住民]][[卑南族|卑南族]]婦女身染重病，請來了卑南族傳統巫師治療仍沒有效果，因此後來同社區的[[漢人|漢人]][[許家禮|許家禮]]將身染重病的婦人帶到臺東市海濱公園一旁的[[太皇殿|太皇殿]]祭拜殿內主祀之太陽星君以求治癒重病。<ref name="教育大市集"/><ref name="農田水利入口網"/>

經過幾次祭拜後該名婦女的重病康復，因此許家禮便將太皇殿祭拜的太陽星君分靈後，帶回建和社區的家中祭拜。而社區內的居民口耳相傳之下，如罹患[[疾病|疾病]]或有事相求時，都會來到許家禮的家中祭拜分靈後的太陽星君。隨著信徒的增加，三年後信徒們集資打造了一尊太陽星君的神像，並請示神像之下，輪流在村民許家禮、[[林水師|林水師]]等人家中供奉。<ref name="教育大市集"/><ref name="農田水利入口網"/>

1960年建和社區的漢人與卑南族人再次共同合資近千萬來組建敬山宮，於該年農曆3月16日落成。爾後在1995年到1996年之間敬山宮分別進行擴建，於主殿上方增加了二樓與三樓，並以[[金箔|金箔]]裝潢在主殿中使得今日敬山宮之規模。<ref name="社區文化通"/>

== 建築特色 ==

敬山宮草創之初原為一座硬山式建築，經過1995年以及1996年之擴建後，總佔地面積約有200坪。正殿祭祀太陽星君，同祀的神明有玄天大帝、觀世音菩薩、中壇元帥以及太陰娘娘。1995年曾建的二樓大殿祭祀著[[中華聖|中華聖]]、[[五穀大帝|五穀大帝]]、[[關聖帝君|關聖帝君]]、[[太乙真人|太乙真人]]。1996年再次曾建的三樓祭祀[[東華帝君|東華帝君]]，同祀神明有[[王母娘娘|王母娘娘]]與[[神農大帝|神農大帝]]等。<ref name="教育大市集"/>

敬山宮的廟前廣場設有一座八卦源臺，最初是舉辦法會時奉神旨所興建，高三層，最上層為天台爐，中層為地母爐，下層為五虎爐，周圍塑有九龍八鳳的環繞裝飾<ref name="教育大市集"/><ref name="社區文化通"/>。主殿的右側有座[[土地公廟|土地公廟]]，其建廟的歷史比起敬山宮更早，建和社區的居民常都稱這座土地公廟為大伯公廟，是[[臺灣戰後時期|臺灣戰後時期]]移入此地的[[客家人|客家人]]所建<ref name="客家委員會"/><ref name="教育大市集"/><ref name="農田水利入口網"/>。

== 重要祭典 ==
敬山宮一年之中最盛大之祭典為農曆3月19日的太陽星君聖誕，其他主要祭典有農曆正月15日的上元求平安、7月15日的[[中元普渡|中元普渡]]，以及10月15日的下元謝神。並且敬山宮也會配合[[知本|知本]]地區的廟宇聯合遶境活動迎駕太陽星君出巡。敬山宮設有建和鑼鼓陣及誦經團，鑼鼓陣是由社區男性約10人組成，平時利用農耕後的閒暇時間練習。誦經團則是男、女性均有，人數約20人，每逢初三、六、九的夜間誦經團會在廟中誦經禮神。每年敬山宮都會舉辦進香活動，好招信徒前往全臺各地主祀太陽星君的13間廟宇巡迴聯誼。<ref name="教育大市集"/><ref name="農田水利入口網"/>
