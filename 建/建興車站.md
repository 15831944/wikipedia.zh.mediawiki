'''建興車站'''位於[[屏東縣|屏東縣]][[枋寮鄉|枋寮鄉]]，曾為[[臺灣鐵路管理局|臺灣鐵路管理局]][[屏東線|屏東線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*側式月台一座。
*木造站房。

== 利用狀況 ==
*廢站後月台、站房即拆除。
*廢站前為招呼站，由[[枋寮車站|枋寮站]]管理。

== 車站週邊 ==
本站以鄰近聚落水底寮而得名。但水底寮市區距此站超過1公里，與大站─枋寮車站相差無幾，因此客源有限。
*建興路
*[[水底寮_(屏東縣)|水底寮]]

== 歷史 ==
*1953年1月16日：'''水底寮站'''設站。設站時為簡易站，並指定由[[枋寮車站|枋寮站]]管理。
*1966年1月1日：因行政區域屬[[枋寮鄉|枋寮鄉]]建興村，站名配合地名變更為建興站。
*1979年1月1日：由簡易站降為招呼站。
*1991年12月1日：廢站。
