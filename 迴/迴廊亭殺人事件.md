《'''迴廊亭殺人事件'''》是[[日本|日本]]作家[[東野圭吾|東野圭吾]]的[[長篇小說|長篇]][[推理小說|推理小說]]。[[光文社|光文社]]於1991年發行了[[單行本|單行本]]名為{{nihongo|《'''迴廊亭的殺人'''》|回廊亭の殺人|かいろうていのさつじん}}，1994年發行了光文社[[文庫版|文庫版]]，發行文庫版的時候書名改為現在的標題。

全書以第一人稱描寫，「我」再度來到迴廊亭是為了還原當年火災的真相，「我」將不惜一切代價踏上復仇之路。電視劇於2011年6月24日播出，由[[常盤貴子|常盤貴子]]主演。

== 故事簡介 ==
一代企業家一原高顯死後，正當妻子要繼承其龐大遺產之時，遺書將要在他所經營的日式旅館回廊亭中，向一眾家族的人公開。除家人以外，還有一名叫菊代的老太太也應邀出席。然而，菊代的真正目的是為了探尋半年前發生在回廊亭的事件真相……

== 登場人物 ==
; {{nihongo|一原 高顯|{{ruby|一原|いちがはら}} {{ruby|高顕|たかあき}}|Ichigahara Takaaki}}
: 已故。一代企業家，逝世後留下龐大的遺產。迴廊亭的擁有者。

; {{nihongo|桐生 枝梨子|{{ruby|桐生|きりゅう}} {{ruby|枝梨子|えりこ}}|Kiryuu Eriko}}
: 三十多歲相貌不佳的女子。一原高顯的秘書，高顯曾向枝梨子表示有意取她為妻，重視的是她的才能。

; {{nihongo|里中 二郎|{{ruby|里中|さとなか}} {{ruby|二郎|じろう }}|Satonaka Zirou}}
: 桐生枝梨子的戀人。在迴廊亭曾經發生的火災中找到焦黑的遺體。

; {{nihongo|本間 重太郎|{{ruby|本間|ほんま}} {{ruby|重太郎|しげたろう}}|Honma Shigetarou}}
: 已故。企業家一原高顯的恩人。

; {{nihongo|本間 菊代|{{ruby|本間|ほんま}} {{ruby|菊代|きくよ}}|Honma Kikuyo}}
: 本間重太郎的遺孀，即將七十歲的老太婆。在丈夫死後仍受一原家照顧，中間連繫人是桐生枝梨子。手中握有桐生枝梨子自殺的遺書，在一原高顯遺書宣布的前一天拆開讓一原家族讀。

; {{nihongo|一原 蒼介|{{ruby|一原|いちがはら}} {{ruby|蒼介|そうすけ}}|Ichigahara Sousuke}}
; {{nihongo|一原 健彥|{{ruby|一原|いちがはら}} {{ruby|健彦|たけひこ}}|Ichigahara Takehiko}}
: 蒼介是高顯的弟弟。大學教授。雖然年紀應不小，但頭髮濃黑不像中老年人。
: 健彥是蒼介的兒子。二十七歲，從事戲劇工作，實際上仍然倚靠父親蒼介供養。

; {{nihongo|一原 直之|{{ruby|一原|いちがはら}} {{ruby|直之|なおゆき}}|Ichigahara Naoyuki}}
: 高顯同父異母的弟弟。單身。與高顯年紀相差二十幾歲，後來繼承了高顯的企業。

; {{nihongo|藤森 曜子|{{ruby|藤森|ふじもり}} {{ruby|曜子|ようこ}}|Huzimori Youko}}
; {{nihongo|藤森 加奈江|{{ruby|藤森|ふじもり}} {{ruby|加奈江|かなえ}}|Huzimori Kanae}}
: 曜子是高顯同父異母的妹妹。約四十多歲，藤森是丈夫的姓氏。
: 加奈江是曜子的女兒。一副大小姐的氣勢。

; {{nihongo|一原 紀代美|{{ruby|一原|いちがはら}} {{ruby|紀代美|きよみ}}|Ichigahara Kiyomi}}
; {{nihongo|一原 由香|{{ruby|一原|いちがはら}} {{ruby|紀代美|きよみ}}|Ichigahara Kiyomi}}
: 紀代美是高顯無血緣關係的義妹。
: 紀代美的女兒由香和加奈江一樣，都是不重禮節的年輕人。

; {{nihongo|小林 真穗|{{ruby|小林|こばやし}} {{ruby|真穂|まほ}}|Kobayashi Maho}}
: 迴廊亭的女主人。五十多歲卻依然嬌豔。

; {{nihongo|古木|{{ruby|古木|こぼく}}|Koboku}}
: 一原高顯的總法律顧問。

; {{nihongo|驂澤 弘美|{{ruby|鰺沢|あじさわ}} {{ruby|弘美|ひろみ}}|Azisawa Hiromi}}
: 古木律師的助理。年紀輕、五官端正，相當俊美。

; {{nihongo|矢崎|{{ruby|矢崎|やざき}}|Yazaki}}
: 職稱警部。此案件的搜查官。
