'''邦加檳港'''又称[[槟港|槟港]]、[[邦加檳榔|邦加檳榔]]（[[印尼語|印尼語]]：Pangkal Pinang） 是[[印尼|印尼]][[邦加島|邦加島]]的最大市鎮，也是[[邦加－勿里洞|邦加－勿里洞]]省的首府。

== 地理位置 ==

位於[[南緯|南緯]]2度10分、[[東經|東經]]106度8分，在邦加島東岸中部。

== 人口 ==

1990年時，人口有108,411；2005年，增至134,082。 
居民主要是自[[中國|中國]][[廣東省|廣東省]]遷移过来的[[客家人|客家人]]，當地人稱之為'Peranakan' (即「僑生兒女」)。是故當地人主要講[[客家話|客家話]]。

== 經濟 ==

'''邦加檳港'''是[[爪哇海|爪哇海]]的重要[[港口|港口]]，主要[[出口|出口]][[貨品|貨品]]有[[錫|錫]]、[[胡椒|胡椒]]、[[魚|魚]]、[[椰乾|椰乾]](copra)等。 港內亦有[[造船業|造船業]]和到[[爪哇海|爪哇海]]捕魚的[[深海漁業|深海漁業]]等。

該市內有一個機場，亦有闢路到島上的其他市鎮。

家庭工業也不少，計有[[紡織|紡織]]、編製[[蓆子|蓆子]]、[[金工|金工]]、[[雕刻|雕刻]]、[[製籃子|製籃子]]等。
