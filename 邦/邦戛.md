== 名稱 ==
根據古代的故事，“'''Pemangkat'''”是取自第一個居住邦戛的人名“'''Mangkat'''”先生。邦戛的名稱是由印尼語翻譯，當地華人把“Pemangkat”稱為「'''邦戛'''」。

== 歷史 ==
在[[三發蘇丹國|三發蘇丹國]]時代，邦戛為來自[[中國|中國]][[廣東|廣東]]的採金礦工的村莊。在荷蘭殖民時代，邦戛是戰略的堡壘，為日軍於1942年首先登陸的地方在西加里曼丹。

== 地理 ==
邦戛位於三發縣西南部，東接[[三巴落|三巴落區]]，西臨[[納土納海|納土納海]]，南與[[沙拉迪加|沙拉迪加區]]毗鄰，北以[[三發河|三發河]]入海口與[[南假獅|南假獅區]]為界。全區地勢平坦，本區中部有一座丘陵稱作象山因為形狀似坐著的大象，山腳下坐落[[邦戛坡中央大伯公廟|邦戛坡中央大伯公廟]]，大伯公廟附近有坐落於山坡上的[[邦戛天后宮|邦戛天后宮]]。

== 行政區劃 ==
邦戛現轄8個[[村|村]]：
# Desa Pemangkat Kota (邦戛埠頭村)
# Desa Harapan
# Desa Penjajap (本也葉村)
# Desa Jelutung
# Desa Perapakan Besi
# Desa Lonam (老𡎜角村)
# Desa Gunung Gajah Sejahtera
# Desa Sebatuan

== 人口 ==
全區人口約44.783人，邦戛人口以[[馬來族|馬來族]]為主，其他族群為[[華人|華人]]，[[達雅族|達雅族]]，[[布吉人|布吉人]]，[[爪哇族|爪哇族]]等。居邦戛的華人多數是[[客家人|客家人]]，也有少數的[[潮州人|潮州人]]和[[廣府人|廣府人]]。
