'''似古菊石'''（[[學名|學名]]：{{lang|la|'''''Anarcestes'''''}}）是生存在[[泥盆紀|泥盆紀]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]，是最原始的菊石類群之一，其外觀還保有菊石之祖先──[[桿石目|桿石]]的一些特徵。<ref name="Anarcestes">{{Cite web |url = http://liusir.pixnet.net/blog/post/248634002-anarcestes-%E4%BC%BC%E5%8F%A4%E8%8F%8A%E7%9F%B3-%28%E4%BF%9D%E5%AD%98%E8%83%8E%E6%AE%BC%29 |title = Anarcestes 似古菊石 (保存胎殼) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-14 }}</ref>其[[化石|化石]]分布於[[摩洛哥|摩洛哥]]、[[捷克|捷克]]和[[阿爾及利亞|阿爾及利亞]]等地。<ref name="Fossilworks" />

== 特徵 ==
似古菊石殼體的外觀就像是將桿石的外殼捲起來的模樣，且保有諸如簡單的縫合線、只有腹側有單一的葉等桿石的特徵。<ref name="Anarcestes" />
