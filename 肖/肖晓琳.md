'''肖晓琳'''（{{bd|1956年或1962年||2017年|6月28日}}），女，生于[[湖南|湖南]][[长沙|长沙]]，毕业于[[北京广播学院|北京广播学院]]，为[[中国中央电视台|中国中央电视台]]著名节目[[主持人|主持人]]、[[制片人|制片人]]，并曾任[[中央电视台社会与法频道|中央电视台社会与法频道]]副总监。<ref name="lion">{{cite web |url = http://www.zhuchijie.com/show_113_z7_t1.html |title = 肖晓琳 |website = 主持界 |accessdate=2010 |language =zh-cn }}</ref>

== 简介 ==
肖晓琳，1956年出生于湖南省长沙市。

1977年，[[文化大革命|文化大革命]]结束之后，[[恢复高考|恢复高考]]，肖晓琳考入[[北京广播学院|北京广播学院]]（今[[中国传媒大学|中国传媒大学]]）播音系。<ref name="lion " />

1985年，肖晓琳担任[[长沙市电视台|长沙市电视台]]（今[[湖南广播电视台|湖南广播电视台]]，不是现在的[[长沙电视台|长沙电视台]]）播音员。<ref name="lion" />

1987年，肖晓琳考入[[中国社会科学院研究生院|中国社会科学院研究生院]]新闻系。<ref name="lion" />

1988年，肖晓琳调入[[中国中央电视台|中国中央电视台]]担任节目主持人和制片人。<ref name="lion" />

2017年7月12日，央视员工向媒体证实，肖晓琳因[[直肠癌|直肠癌]]在[[美国|美国]]逝世。<ref>{{cite web |url = http://entertainments.dwnews.com/news/2017-07-12/59825067.html | language= |title = 央视主持人肖晓琳病逝 曾创办《今日说法》 |publisher=多维新闻 |date=2017-07-12 |accessdate= }}</ref>
