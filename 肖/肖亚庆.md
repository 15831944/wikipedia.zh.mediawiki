'''肖亚庆'''（{{bd|1959年|9月|catIdx=X肖}}），[[北京市|北京市]]人；原[[中南大学|中南矿冶学院]]材料系毕业，教授级高级工程师。1981年12月加入[[中国共产党|中国共产党]]，是[[中共|中共]]第十七届中央候补委员、第十九届中央委员。曾任[[中华人民共和国国务院副秘书长|中华人民共和国国务院副秘书长]]，[[中国铝业公司|中国铝业公司]][[总经理|总经理]]、党组书记，[[中国铝业股份有限公司|中国铝业股份有限公司]][[董事长|董事长]]、兼[[首席执行官|首席执行官]]。现任[[国务院国有资产监督管理委员会|国务院国有资产监督管理委员会]]主任、党委副书记。

== 早年经历 ==
1982年8月，大学毕业后的肖亚庆被分配到[[哈尔滨|哈尔滨]]“[[东北轻合金加工厂|东北轻合金加工厂]]”工作。此后十七年间，他历任技术处工程师，副科长、科长，副处长、处长，副总工程师、总工程师等职；1998年，出任东北轻合金有限责任公司总经理。

1999年，肖亚庆调任[[重庆|重庆]][[西南铝加工厂|西南铝加工厂]]厂长。于2000年4月，对这家连年亏损且负载累累的[[国有企业|国有企业]]实施债转股的改革，并其原有基础上组建了“西南铝业(集团)有限责任公司”，并出任西南铝业首任[[董事长|董事长]]、兼[[总经理|总经理]]。当年，该公司实现了扭亏为盈。一年后，西南铝业成为[[重庆市|重庆市]]工业企业50强，并名列中国最大500家企业集团。肖亚庆个人也被评为“重庆市十大杰出青年企业家”<ref>[http://ceo.icxo.com/htmlnews/2004/05/14/216730.htm 肖亚庆掌舵中铝] {{webarchive|url=https://web.archive.org/web/20060923163723/http://ceo.icxo.com/htmlnews/2004/05/14/216730.htm |date=2006-09-23 }} 世界经理人</ref>。

== 执掌中铝 ==
2004年4月，肖亚庆接替[[郭声琨|郭声琨]]出任[[中国铝业公司|中国铝业公司]][[总经理|总经理]]、党组书记，[[中国铝业股份有限公司|中国铝业股份有限公司]][[董事长|董事长]]的职务<ref>[http://www.cnceo.com/webcontent/cnceo/person/fangtan/20070218/853420070218231224.shtml 肖亚庆:咬定核心价值观不放松] 《全球CEO经典文摘》</ref>。时年45岁的肖亚庆，采取了一系列的措施，使得[[中国铝业|中国铝业]]不断的发展壮大；由最初的“七厂一院”变成了现在由50多家分、子公司和成员单位组成的大集团。

2007年，[[中国铝业|中国铝业]]更是全面开始向外扩张。1月初，中铝通过换股方式，对山东铝业和兰州铝业进行重组后；6月，又以8.6亿[[美元|美元]]收购了[[加拿大|加拿大]]秘鲁铜业公司；8月，通过增资扩股的方式，战略重组云铜集团。如果说这些都还不足以引起世界关注的话；那么，2008年1月31日，中铝和美国铝业公司联合收购矿业巨头——[[力拓|力拓]]公司在[[伦敦|伦敦]]上市的股票，则彰显了中国企业在世界金融资本市场的运作能力和日趋成熟的操作技巧。中铝和美铝共同斥资140.5亿美元，获得[[力拓|力拓]]9%的股份，成为这家跨国集团的最大单一股东。此前，另一矿业巨头[[必和必拓|必和必拓]]也曾试图收购[[力拓|力拓]]。由于[[中国铝业|中国铝业]]的“搅局”，为[[必和必拓|必和必拓]]收购[[力拓|力拓]]的计划增加了障碍<ref>[http://chinese.wsj.com/gb/20080206/ffe173546.asp?source=UpFeature2 中铝收购力拓股份尽展圆融手段] 《华尔街日报》中文网络版</ref>。

== 步入政界 ==
2009年，因[[中国铝业|中铝]]对[[力拓|力拓]]股权收购而名声大噪的肖亚庆，在刚刚50岁时被任命为[[中华人民共和国国务院副秘书长|中华人民共和国国务院副秘书长]]<ref>[http://news.sina.com.cn/c/2009-02-18/023415177492s.shtml 中铝总经理肖亚庆出任国务院副秘书长]</ref>。2016年2月，出任[[国务院国有资产监督管理委员会|国务院国有资产监督管理委员会]]主任<ref>[http://news.wenweipo.com/2016/02/01/IN1602010042.htm 肖亚庆出任国资委主任]</ref>。2017年6月起，兼任[[国务院国有资产监督管理委员会|国务院国有资产监督管理委员会]]党委副书记。
