《'''極速秒殺2'''》（{{lang-en|'''''Mechanic: Resurrection'''''}}）是一部2016年上映的[[美國|美國]][[驚悚片|驚悚]][[動作片|動作片]]，由[[丹尼斯·甘塞爾|丹尼斯·甘塞爾]]負責執導。本片為2011年電影《[[極速秒殺|机械师]]》的續集。電影主演包括[[傑森·史塔森|傑森·史塔森]]、[[潔西卡·艾芭|潔西卡·艾芭]]、[[湯米·李·瓊斯|湯米·李·瓊斯]]、[[楊紫瓊|楊紫瓊]]和山姆·哈茲爾丁。該片定於2016年8月26日在美國上映。

== 劇情 ==
有一種武藝高強、手法俐落，在謀殺後會故佈疑陣有如意外的高級[[職業殺手|職業殺手]]，被稱為「[[機械師|機械師]]」（Mechanic）。亞瑟·畢夏，就是「機械師」當中的傳奇。[[極速秒殺|上一集]]，亞瑟與自己的弟子史提夫·麥肯納反目成仇，對決之後，亞瑟勝出，成功脫身，並且金盆洗手。

亞瑟離開[[江湖|江湖]]五年以後，又被昔日的仇家盯上了，而且這不只是仇家，還是個少年時的好夥伴利雅·克雷恩。原來亞瑟少年時被一名軍火商捉走，訓練為[[殺手|殺手]]，亞瑟逃出了魔掌，而他的少年至交克雷恩卻被軍火商收為[[義子|義子]]，繼承了軍火事業，克雷恩深恨亞瑟拋棄了他，誓言找到亞瑟。

亞瑟隱居於[[泰國|泰國]]，在紅粉知己「梅」的介紹下，認識了[[柬埔寨|柬埔寨]]的[[人道救援|人道救援]]組織負責人吉娜，吉娜既美貌又充滿愛心，亞瑟很快就認定她是自己一生的挚爱，兩人火速墜入情網。克雷恩於是[[綁架|綁架]]了吉娜作為要脅，要亞瑟[[刺殺|刺殺]]三名自己畏懼的對手，完成三個高難度的任務。這三個對手，也是三位縱橫天下的黑色[[領導人|領導人]]，分別是在[[馬來西亞|馬來西亞]]服刑的[[非洲|非洲]][[軍閥|軍閥]]、[[雪梨|雪梨]]的[[人口販賣|人口販賣]]大亨，最後一位是僑居[[保加利亞|保加利亞]]的重型武器大王，[[美國人|美國人]]「麥斯·亞當斯」。

亞瑟由[[南洋|南洋]]、[[澳洲|澳洲]]繞[[地球|地球]]一圈，以救出心愛的人，當他完成前面兩個任務時，他轉念一想，就算任務都完成了，他和吉娜還是會被克雷恩滅口，為了愛人的性命，他還是去刺殺麥斯·亞當斯，不過亞瑟有了一個新的計畫.....

== 製作 ==

=== 拍攝 ===
2014年11月4日，電影在[[泰國|泰國]]的[[曼谷|曼谷]]正式拍攝<ref>{{cite news|last1=McNary|first1=Dave|title=Afm: Jason Statham’s ‘Mechanic: Resurrection’ Set for Jan. 22, 2016|url=http://variety.com/2014/film/news/jason-statham-mechanic-resurrection-2016-1201351021/|accessdate=November 8, 2014|publisher=variety.com|date=November 8, 2014}}</ref>。[[保加利亞|保加利亞]]、[[巴西|巴西]]、[[喬治市|喬治市]]和[[澳大利亞|澳大利亞]]的[[悉尼|悉尼]]也作為電影的取景地。
