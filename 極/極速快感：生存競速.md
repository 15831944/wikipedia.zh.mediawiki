'''《極速快感：生存競速》'''（{{lang-en|Need for Speed: Rivals}}） 是一款由Ghost Games（前EA 歌德堡）非線性（開放式世界觀）[[賽車遊戲|競速遊戲]]，也是EA長壽作之一的[[極速快感系列|極速快感系列]]第18代作品。《生存競速》於2013年11月19日在[[Microsoft_Windows|Microsoft Windows]]、[[PlayStation_3|PlayStation 3]]及[[Xbox_360|Xbox 360]]上發行，[[PlayStation_4|PlayStation 4]]和[[Xbox_One|Xbox One]]版本則會在2013年第四季販售。

== 遊戲概覽 ==
{{seealso|極速快感：超熱力追緝}}

《極速快感：生存競速》的遊戲模式會類似於2010年發行的[[極速快感：超熱力追緝|極速快感：超熱力追緝]]。<ref name="EG-reveal">{{cite web|last=Robinson |first=Martin |url=http://www.eurogamer.net/articles/2013-05-23-how-far-cry-3-and-uk-racing-veterans-are-helping-bring-need-for-speed-to-the-next-generation |title=How Far Cry 3 and UK racing veterans are helping bring Need for Speed to the next generation • News • Xbox One • |publisher=Eurogamer.net |date=2013-05-15 |accessdate=2013-05-23}}</ref>玩家可以選擇加入非法賽車手或是警察陣營，各自會有不同的挑戰、風險與獎勵。<ref name="revealCVG"/>《生存競速》會有11種不同追捕與逃脫科技工具可供升級及使用，例如[[電磁脈衝|電磁脈衝]]、[[震波|震波]]以及[[路障|路障]]等等<ref name="EG-reveal"/>。遊戲中的虛構城市紅景郡、地圖是屬於開放性的，警察和車手之間持續上演著街頭追逐驚險場面。<ref name="EG-reveal"/>而開放世界設定則與2012年推出的[[極速快感：新全民公敵|極速快感：新全民公敵]]類似，包含各種各樣的跳躍動作、速度檢測陷阱、車輛解鎖等。<ref name="Poly-Ghost">{{cite web|url=http://www.polygon.com/2013/5/23/4356942/need-for-speed-rivals-customization-alldrive-cars-interview-ghost-games |title=How Ghost Games hopes to put itself on the map with Need for Speed Rivals |publisher=Polygon |date=2012-11-14 |accessdate=2013-05-23}}</ref>

=== 新工作室參與 ===
隨著 EA 加拿大與[[EA_Black_Box|Black Box]]2012年的整併並定位專注於線上免費遊戲<ref>{{cite web|url=http://www.computerandvideogames.com/334628/ea-confirms-vancouver-studio-layoffs/|title=EA confirms Vancouver studio layoffs|last=Robinson|first=Andy|publisher=[[Computer_and_Video_Games|Computer and Video Games]]|date=2012-02-06|accessdate=2013-01-26}}</ref>，EA提前於2011年建立起新的工作室—— EA 歌德堡（現Ghost Games）。EA歌德堡位於[[瑞典|瑞典]][[哥德堡|哥德堡]]，專門負責開發[[寒霜引擎|寒霜引擎]]遊戲<ref>{{cite web |title=What We Do – EA Gothenburg |url=http://eagothenburg.ea.com/what-we-do/ |publisher=[[EA_Gothenburg|EA Gothenburg]] |archiveurl=https://www.webcitation.org/6C5fFAsdY?url=http://eagothenburg.ea.com/what-we-do/ |archivedate=2012-11-11 |accessdate=2012-11-11 |dead-url=yes }}</ref><ref>{{cite web|url=http://www.vg247.com/2012/03/05/ea-gothenburg-studio-to-focus-on-frostbite-development/ |title=EA Gothenburg studio to focus on Frostbite development |publisher=VG247 |date=2012-03-05 |accessdate=2012-11-17}}</ref><ref>{{cite web |title=EA Gothenburg hiring for new Need for Speed |url=http://www.computerandvideogames.com/372103/ea-gothenburg-hiring-for-new-need-for-speed |publisher=CVG|accessdate=2012-11-15 }}</ref>、當中包括極速快感系列的開發工作<ref name=NewNFSconfirmed>{{cite web|url=http://www.videogamer.com/xbox360/need_for_speed_most_wanted_2012/news/confirmed_ea_gothenburg_developing_unannounced_need_for_speed.html |title=Confirmed: EA Gothenburg developing unannounced Need For Speed |publisher=Videogamer.com |date= |accessdate=2012-11-17}}</ref>。根據工作室成員履歷、不少成員都有賽車遊戲開發的經歷，包括系列前作《[[極速快感：亡命天涯|亡命天涯]]》、Forza Horizon和Project Gotham Racing。

2013年5月23日，EA 確認極速快感系列續作《生存競速》，並附上宣傳片段。<ref name="revealCVG">{{cite web|url=http://www.computerandvideogames.com/408626/need-for-speed-rivals-announced-for-current-and-next-gen/ |title=News: Need for Speed Rivals announced for current and next-gen |publisher=ComputerAndVideoGames.com |date= |accessdate=2013-05-23}}</ref><ref>{{cite web|url=http://www.computerandvideogames.com/408289/new-need-for-speed-game-teased-with-screenshot/ |title=News: New Need for Speed game teased with screenshot |publisher=ComputerAndVideoGames.com |date= |accessdate=2013-05-23}}</ref><ref>{{cite web|url=http://www.computerandvideogames.com/408430/need-for-speed-reveal-teased-for-today/ |title=News: Need for Speed reveal teased for today |publisher=ComputerAndVideoGames.com |date= |accessdate=2013-05-23}}</ref>並同時確認《生存競速》由EA 旗下瑞典Ghost Games連同Criterion Games 合作開發。<ref name="revealVG247">{{cite web|url=http://www.vg247.com/2013/05/23/need-for-speed-rivals-announced-for-november-release-on-ps4-and-xbox-one/ |title=Need for Speed: Rivals lands on PS4, Xbox One in November |publisher=VG247 |date=2008-01-28 |accessdate=2013-05-23}}</ref>
