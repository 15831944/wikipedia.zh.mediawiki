'''極速民謠'''（{{lang-sr|турбо фолк}} 又名'''塞爾維亞浪潮'''）是起源於[[塞爾維亞|塞爾維亞]]的音樂風格。<ref name="Turbo-folk :: ">{{cite web |url=http://www.urbandictionary.com/define.php?term=turbo%20folk |title=Turbo-folk :: |author= |authorlink= |coauthors= |date= |format= |work= |publisher= |pages= |language= |archiveurl= |archivedate= |quote= |accessdate = 28. 8. 2010.}}</ref>其特點是融合了[[塞爾維亞民間音樂|塞爾維亞民間音樂]]，現代[[流行音樂|流行音樂]]和[[舞曲|舞曲]]等風格。而歌詞題材，通常描繪愛情或生活，也有以暗喻手法描繪保皇派和反共的抒情主題。極速民謠 與 [[保加利亞|保加利亞]]的[[查爾加|查爾加]]、 [[羅馬尼亞|羅馬尼亞]]的[[馬內爾|馬內爾]]、[[希臘|希臘]]的[[斯卡拉迪科|斯卡拉迪科]]、以及[[阿爾巴尼亞|阿爾巴尼亞]]的[[塔拉瓦|塔拉瓦]]風格相似，其後於2004年演變成為[[流行-民謠|流行-民謠]]。雖然極速-民謠與[[塞爾維亞人|塞爾維亞人]]關係密切 ，是[[塞爾維亞|塞爾維亞]]主流音樂，但該類型在[[克羅地亞|克羅地亞]]、[[斯洛文尼亞|斯洛文尼亞]]、[[波斯尼亞和黑塞哥維那|波斯尼亞和黑塞哥維那]]、 [[馬其頓共和國|馬其頓]]、[[保加利亞|保加利亞]]和[[黑山|黑山]]一帶也廣泛流行。

== 曲風起源 ==
極速-民謠這個術語是由黑山的塞爾維亞男唱作人[[安東尼耶·普西奇|安東尼耶·普西奇]]（Antonije Pušić），在八十年代末所創建，用以描述其音樂風格。而於九十年代初期，首位歌手以此曲風作主打歌為[[波斯尼亞和黑塞哥維那|波斯尼亞和黑塞哥維那]]的塞爾維亞女歌手[[莉帕·布雷那|莉帕·布雷那]]（Lepa Brena）。<ref name= :Turbo-folk2: ">{{cite web |url=http://www.serbia.com/turbo-folk-music/ |title=The first star of turbo-folk music was Lepa Brena :: |author= |authorlink= |coauthors= |date= |format= |work= |publisher= |pages= |language= |archiveurl= |archivedate= |quote= |accessdate = 5.6. 2015.}}</ref>

== 流行與衰退期 ==
雖然極速民謠於八十年代末形成，但是真正鼎盛是從九十年代初期開始。當時極速民謠藝術家受到黑社會和政府資助，從而讓他們可以不斷在電台宣傳這類型的歌曲，誘致大街小巷也在播放它們。而另一方面，前國家元首[[斯洛博丹·米洛舍維奇|斯洛博丹·米洛舍維奇]]，當時支持戰爭的[[政治宣傳|政治宣傳]] 訊息，可以進一步在主流媒體散播給大眾。因為他命令受資助的音樂人，把有關政治的東西一一投放在歌曲裏。<ref name= :Turbo-folk3: ">{{cite web |url=http://www.tol.org/client/article/9442-turbo-folk-politics.html|title=Turbo Folk Politics: |author= |authorlink= |coauthors= |date= |format= |work= |publisher= |pages= |language= |archiveurl= |archivedate= |quote= |accessdate = 7.5. 2003.}}</ref>
 
極速民謠於九十年代中期達到了頂峰，然而，隨著[[斯洛博丹·米洛舍維奇|斯洛博丹·米洛舍維奇]]卸任和離逝之後，它從九十年代的[[塞爾維亞|塞爾維亞]]音樂象徵轉變為平庸的音樂。
 
== 批評 ==
雖然極速民謠受歡迎，但一直受到爭議。因為一些人民認為此曲風與塞爾維亞民族主義，甚至是戰爭意識形態有關。而問題的癥結是在於它象徵著，1990年代[[南斯拉夫內戰|南斯拉夫內戰]]中，所誘致的道德和文化衰落。
