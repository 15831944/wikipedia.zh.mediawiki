《'''我miss係大佬'''》是[[日本|日本]][[漫畫家|漫畫家]][[森本梢子|森本梢子]]的原作[[漫畫|漫畫]]，及後是同名的[[電視劇|電視劇]]和[[動畫|動畫]]。

電視劇由[[日本電視台|日本電視台]]製作。[[中國大陸|中國大陸]]和台灣称'''-{zh-hans:极道鲜师;zh-hant:極道鮮師}-'''(分別於[[星空卫视|星空卫视]]、上海电视台[[艺术人文频道|艺术人文频道]]播出和[[緯來日本台|緯來日本台]]播出)，[[香港|香港]]稱'''-{我miss係大佬}-'''(在[[香港有線電視|有線電視]]及[[亞洲電視|亞洲電視]]播出，意思為「我的女老师是江湖大哥」)。

電影版於2009年宣佈開拍，日本在2009年7月11日上映。有關電影版請見[[我miss係大佬_(電影)|我miss係大佬 (電影)]]。

== 作品概要 ==
漫畫原載於[[集英社|集英社]]的《[[YOU_(雜誌)|YOU]]》，目前單行本發行到15集。電視版一共播映3部，動畫版於2004年1月至3月播放，共12話。電視版第1部於2002年4月17日至7月3日，每週三22:00至22:54播映，共12話，平均收視率17.6%。而第1部特別篇《再見3年D組…小美淚的畢業禮》於2003年3月26日，週三21:30至23:33播映，收視率18.1%。第2部於2005年1月15日至3月19日，每週六21:00至21:54播映，共10話，平均收視率27.8%。第3部於2008年4月19日至6月28日，每週六21:00至21:54播映，共11話，平均收視率22.6%。而第3部《畢業特別篇'09》於2009年3月28日，週六21:30至23:28播映，收視率18.1%。極道鮮師更決定拍攝電影版《極道鮮師THE MOVIE》，於2009年7月11日公映，為整個系列劃上句號，作為完結篇。

== 故事簡介 ==
主角山口久美子，自少雙親因為交通意外而身亡，被外祖父收養，而外祖父經營的是大江戶一家(漫畫原作為黑田一家)，雖然久美子是唯一繼承人，但她的志願是當教師，最後在白金學院高校的男子學校開始其教學生涯。但是她教導的3年D組(原作為3年4班)裡面，全部都是無心向學的不良少年。久美子立志改變自己的學生成為成功的人，為學生勞心勞力。起初她的學生毫不領情，但漸漸受她影響，成為像久美子般重情重義的人，而班中的高材生澤田慎亦由一開始覺得所有老師不可信，變得尊敬久美子，最後更愛上了她。<BR>

<big>漫畫原作與電視版差異：</big><BR>
⒈'''久美子任職的學校'''：電視劇中，久美子歷經白金、黑銀、赤銅三所學校，每次帶的都是該校的3年D組。然而原作中，久美子進入白金高校後，從高二開始就帶著4班的學生一路到畢業，總共帶了2年，隨著4班畢業連載也結束了。而後續的番外篇中，久美子仍在白金高校教書，遇到新的問題學生(1年4班)。(白金學院並未結束營運，久美子也一直都待在白金學院，並未到其他學校教書。)<BR>
⒉'''學生'''：電視劇中出場的學生角色，只有第1部的學生(白金學院3年D班)是原作角色，後面第2部與第3部學生(黑銀、赤銅)，皆為電視原創角色，但是部分劇情仍引用了原作的片段情節。例如，第2部的小田切龍與矢吹隼人，套用了許多原作中澤田慎的戲份，將原作澤田慎的人物設定一分為二讓這兩人承襲。<BR>
⒊'''久美子的戀情'''：漫畫原作中，久美子單戀篠原律師多年，篠原律師並非對久美子無意，但糾結於久美子的身分，從未給予回應，兩人之間始終沒有進展。根據二當家的觀察：「大小姐對律師，比較像是對鄰家哥哥的仰慕。」最後篠原律師決定回北海道老家，終於表態希望帶久美子一起走，然而久美子拒絕了。而澤田慎在畢業後考取東大法律系，放榜後向久美子告白，久美子因驚嚇過度落荒而逃。而藤山老師點出久美子的盲點：「在妳難過時、遇到麻煩時，總是依賴著誰？想哭的時候，又會想到誰？可不是那個篠原律師喔~~」讓久美子開始深思自己的感情。而作者在連載結束後的以篇番外篇形式，交代了慎與久美子在多年前的最初邂逅與戀情後續發展。最後於2009年以「五代目」為標題的四格漫話，描述阿鐵與阿實正在照顧慎與久美子的兩個孩子的情景，間接交代兩人的結局。電視劇中，每一部久美子都有單戀的對象，但最後都不了了之，直到三部+電影完結，久美子一直都沒有交往對象。而「澤田慎」雖然在第1部最後坦承對久美子有好感，但第1部特別篇中劇情安排其畢業後決定前往非洲，之後都未再出現，或許是這個因素，電視劇中的久美子始終沒有遇到真命天子。<BR>
⒋'''極道'''與'''高校'''：電視劇以描述久美子與學生之間的校園生活為主，對於久美子的極道背景輕描淡寫、未深入交代。而原作中，除了校園生活之外，對於黑田一家(大江戶一家)的歷史背景、勢力有更詳細的介紹，也交代了久美子的身世故事。

; 澤田慎〈18〉
: [[松本潤|松本潤]]([[嵐|嵐]])飾演。
: 電視劇：
: 在之前就讀的學校因為毆打老師而被退學，轉進白金高中。頭腦清晰、運動萬能，在學生之間十分顯眼，雖然看來酷酷的，但非常為朋友著想，也很有人望，在班上處於領導地位。 : 是班上唯一發現久美子並非等閒之輩的人，且暗戀久美子。父親是議員，因為無法認同父親想法與父親起衝突，搬離家中獨自在外居住，有一個妹妹--夏美。特別篇中，慎決定畢業後到非洲，之後並未再交代他的事情。
: 原作漫畫：
: 父親是警視廳刑事部中央局長，有一個哥哥。原本可以進入明星高中--青玉高校，卻因為國中畢業前毆打老師失去入學資格。與父親有心結，為了反抗父親，故意選擇就讀聲名狼藉的白金高校，並將頭髮染紅，離家在外獨居。成績優異，沉穩冷靜，個性看似冷淡，卻願意為朋友兩肋插刀，是班上的首腦級領導人物。
: 最初帶領全班反抗久美子，後來漸漸認同久美子，是班上少數知道久美子家庭背景的學生。因為奮力救出被綁架的久美子，被黑田組眾人視為恩人，受到黑田龍一郎與二當家的賞識，與黑田組關係良好。被二當家以「紅獅子少俠」為名號向江湖中人介紹，被盛傳是黑田組未來的「四代目」，名聲在道上漸漸傳開。
: 外型俊美，頗有異性緣，然而對女孩子總是淡漠以對。了解久美子之後，漸漸被吸引，久美子也常不自覺地依賴慎。對久美子暗戀的篠原律師懷有敵意，常常與之暗中較勁。原本對未來升學沒有方向，後來為了守護久美子，志願成為黑田組的律師。漫畫最終話中考上東大法律系，並向久美子告白。
: 連載結束後，作者在漫畫雜誌「YOU」中不定期以番外篇形式，交代久美子和慎之間的感情進展，最後於2009年以「五代目」為標題的四格漫畫，介紹了久美子和慎的兩個小孩。
: 雖然慎與久美子是在白金高校認識的，但兩人其實在多年前便曾經偶遇。作者以短篇番外描述兩人最初的邂逅，當時11歲的慎目睹搶劫案路見不平，與路過的高中生久美子共騎腳踏車追逐犯人。不過這段往事在兩人心中並未留下深刻的印象。
