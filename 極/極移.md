'''極移'''是[[地球|地球]]的[[地球自轉|自轉]]軸在地球表面橫越的運動，這是將地球視為在一個固定不變的參考座標系（所謂的''地球中心、地球固定''或[[地心地固坐标系|ECEF]]參考系）下所做的測量，這種變動只有幾米。

== 分析 ==
從一個常用的定義參考軸，也就是以1900年北極的平均位置為依據的[[國際習用原點|國際習用原點]]，分析地球自轉軸的瞬間位置，或是[[北極|北極]]，似乎包含兩個類似週期並逐漸向西漂移的主要運動。

這兩個週期或多或少的部分近似[[圓周運動|圓周運動]]，以大約一年的被迫擺動和和433天的自由擺動，稱為[[钱德勒摆动|钱德勒摆动]]的圓周運動，另外也有較不知名的緩慢的漂移。這些運動都由[[國際地球自轉服務|國際地球自轉服務]]（IERS）持續進行觀測和說明。
<ref>[http://www.iers.org/iers/earth/rotation/polmot/figure1.html Polar Motion Map BW] {{webarchive|url=https://web.archive.org/web/20080908001112/http://www.iers.org/iers/earth/rotation/polmot/figure1.html |date=2008-09-08 }}, [http://hpiers.obspm.fr/eoppc/eop/eopc01/plottemean.gif Color] {{webarchive|url=https://web.archive.org/web/20090919130039/http://hpiers.obspm.fr/eoppc/eop/eopc01/plottemean.gif |date=2009-09-19 }}</ref>.

平均位移遠超過擺動的數值，會導致觀測地球的太空船上使用的軟體產生誤差。因為在分析員所忽略的5米的圓周運動中，可能存在20米的偏移，从而使計算[[經度|經度]]和[[緯度|緯度]]的精確度受到影響。而後者是[[國際地球自轉服務|國際地球自轉服務]]依據極移測定的。

== 成因 ==
從1900年以來，极点漂移了大約20米，部分可以歸責於地核、地幔的运动，还有类似[[冰川|冰川]]融解所造成的水體的重分配，以及{{tsl|en|Post-glacial rebound|地殼反彈|地殼均衡的反彈}}，即過去承擔冰川或冰床的土地緩慢上升<ref name="munk">Munk, Walter (2002)</ref>。
