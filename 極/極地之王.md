《'''極地之王'''》（{{lang-en|''Alpha''}}）是一部[[2018年電影|2018年]]美國[[時代劇|歷史劇情]][[冒險片|冒險片]]，由[[休斯兄弟|亞伯·休斯]]執導，丹尼爾·賽巴斯汀·溫登豪普特（{{lang|en|Daniele Sebastian Wiedenhaupt}}）撰寫劇本，其主演包括[[寇帝·史密-麥菲|寇帝·史密-麥菲]]、{{le|蕾奧娜·華莉拉|Leonor Varela}}、{{le|延斯·赫爾騰|Jens Hultén}}和約翰尼斯·赫伊屈爾·約翰森（{{lang|en|Jóhannes Haukur Jóhannesson}}）。故事描述一名[[末次冰盛期|末次冰盛期]]的年輕獵人與一隻受傷的狼成為了朋友。該片由[[索尼影視娛樂|索尼影業發行]]定於2018年8月17日在美國上映。

== 劇情 ==
在距今兩萬年前的[[舊石器時代晚期|舊石器時代晚期]]，地球正值弱肉強食的[[末次冰盛期|冰河時期]]，一名少年獵人克達在与族人一起狩獵[[西伯利亞野牛|西伯利亞野牛]]時發生了意外而掉落到一处悬崖峭壁上，昏迷的他被族人误认为已经死亡，后来克達和一隻受傷的[[歐亞狼|狼]]成為朋友，將牠取名「阿爾法」（Alpha），這也是人與狼之間首次搭建的橋樑<ref name="Variety(2017-06-16)" /><ref name="Deadline(2015-09-15)" />。

== 製作 ==
該片於2015年9月15日首次對外公布，[[休斯兄弟|亞伯·休斯]]被選來擔任導演，且由製片公司{{lang|en|Studio 8}}負責製作，預計以[[IMAX|IMAX 3D]]的格式發行<ref name="Deadline(2015-09-15)">{{cite news|date=2015-09-15 |last=Fleming Jr |first=Mike |title=Jeff Robinov's First Studio 8-Hatched Film: Albert Hughes-Directed Ice Age Survival Tale 'Solutrean' |url=http://deadline.com/2015/09/the-solutrean-albert-hughes-jeff-robinov-studio-8-first-project-survival-tale-1201528665/ |work=[[Deadline_Hollywood|Deadline Hollywood]] |access-date=2016-05-15|archive-url=https://web.archive.org/web/20180620074331/http://deadline.com/2015/09/the-solutrean-albert-hughes-jeff-robinov-studio-8-first-project-survival-tale-1201528665/|archive-date=2018-06-20|dead-url=no}}</ref>。演員[[寇帝·史密-麥菲|寇帝·史密-麥菲]]於2015年11月12日加入演出<ref name="Smit-McPhee">{{cite news|date=2015-11-12 |last=White |first=James |title=Kodi Smit-McPhee starring in The Solutrean |url=http://www.empireonline.com/movies/news/kodi-smit-mcphee-starring-solutrean/ |work=[[Empire_(magazine)|Empire]] |access-date=2016-05-15|archive-url=https://web.archive.org/web/20180626045959/http://www.empireonline.com/movies/news/kodi-smit-mcphee-starring-solutrean/|archive-date=2018-06-26|dead-url=no}}</ref>。主要選角於2016年2月19日完成<ref>{{cite news|date=2016-02-19 |last=Busch |first=Anita |title=Albert Hughes' Ice Age Epic 'The Solutrean' Gets Rolling For Studio 8 |url=http://deadline.com/2016/02/the-solutrean-studio-8-epic-ice-age-movie-cast-start-date-1201705873/ |work=[[Deadline_Hollywood|Deadline Hollywood]] |access-date=2016-05-15|archive-url=https://web.archive.org/web/20180626030401/http://deadline.com/2016/02/the-solutrean-studio-8-epic-ice-age-movie-cast-start-date-1201705873/|archive-date=2018-06-26|dead-url=no}}</ref>。

[[主要拍攝|主要拍攝]]於2016年2月22日起在[[溫哥華|溫哥華]]開始進行，預計於5月20日結束，其他拍攝地也包括了{{le|德蘭赫勒|Drumheller}}<ref>{{cite web|date=2016-04-30 |last=Kolafa |first=Pat |title=Ice age epic filmed near Hoodoos |url=http://www.drumhellermail.com/news/28399-ice-age-epic-filmed-near-hoodoos |work=The Drumheller Mail |access-date=2016-05-15|archive-url=https://web.archive.org/web/20180814103103/http://www.drumhellermail.com/news/28399-ice-age-epic-filmed-near-hoodoos|archive-date=2018-08-14|dead-url=no}}</ref>。在過程中，劇組溫哥華東肯特大道附近的邊界搭景；此外，部份的拍攝也在[[本拿比|本拿比]]及[[冰島|冰島]]進行<ref>{{cite web |date=2016-02-22 |title=The Solutrean Starts Filming in Vancouver This Week |url=http://www.whatsfilming.ca/2016/02/22/the-solutrean-starts-filming-in-vancouver-this-week/ |website=What's Filming? |access-date=2016-05-15|archive-url=https://web.archive.org/web/20180814071544/http://www.whatsfilming.ca/2016/02/22/the-solutrean-starts-filming-in-vancouver-this-week/|archive-date=2018-08-14|dead-url=no}}</ref>。劇組也於2016年4月在加拿大[[艾伯塔省|艾伯塔省]]的[[省立恐龍公園|省立恐龍公園]]取景<ref>{{cite web |date=2016-02-03 |title=What's filming in Vancouver right now |url=http://www.onlocationvacations.com/2016/02/03/64258/ |website=On Location Vacations |access-date=2016-05-15|archive-url=https://web.archive.org/web/20180814071400/http://www.onlocationvacations.com/2016/02/03/64258/|archive-date=2018-08-14|dead-url=no}}</ref>。

據報導，在拍完該片後，有五隻艾伯塔省的[[美洲野牛|美洲野牛]]遭到殺害，這使劇組已接受當局的調查<ref>{{cite web |last=Rieger |first=Sarah |date=2016-06-27 |title='The Solutrean' Under Investigation After 5 Alberta Bison Were Allegedly Killed |url=http://www.huffingtonpost.ca/2016/06/27/the-solutrean_n_10702216.html |work=[[Huffington_Post|Huffington Post]] |accessdate=2016-07-31|archive-url=https://web.archive.org/web/20180724062647/http://www.huffingtonpost.ca/2016/06/27/the-solutrean_n_10702216.html|archive-date=2018-07-24|dead-url=no}}</ref>。

== 發行 ==
2017年6月，該片的片名從《{{lang|en|The Solutrean}}》改為《{{lang|en|Alpha}}》。其美國上映日最初定於2017年9月15日，之後延期至2018年3月2日<ref name="Variety(2017-06-16)">{{cite news|url=https://variety.com/2017/film/news/solutrean-alpha-white-boy-rick-matthew-mcconaughey-1202469254/ |title='Solutrean' Retitled 'Alpha,' Gets New Release Date (EXCLUSIVE) |last=Lang |first=Brent |work=Variety |date=2017-06-16 |access-date=2017-09-04|archive-url=https://web.archive.org/web/20180626165630/https://variety.com/2017/film/news/solutrean-alpha-white-boy-rick-matthew-mcconaughey-1202469254/|archive-date=2018-06-26|dead-url=no}}</ref>。2017年12月，該片又改至2018年9月14日<ref>{{cite news|last=D'Alessandro |first=Anthony |date=2017-12-28 |title=Sony Moves 'Alpha' To Fall, Bumps 'Goosebumps 2' To October |url=http://deadline.com/2017/12/jack-black-goosebumps-2-alpha-new-release-dates-1202233292/ |work=[[Deadline_Hollywood|Deadline Hollywood]] |accessdate=2017-12-29|archive-url=https://web.archive.org/web/20180728092932/http://deadline.com/2017/12/jack-black-goosebumps-2-alpha-new-release-dates-1202233292/|archive-date=2018-07-28|dead-url=no}}</ref>。2018年4月，該美國上映日最終定為2018年8月17日<ref>{{cite news|last=D'Alessandro |first=Anthony |title='Alpha' & 'White Boy Rick' Swap Release Dates On Sony Schedule |url=http://deadline.com/2018/04/alpha-white-boy-rick-swap-release-dates-on-sony-schedule-1202361842/ |work=Deadline Hollywood |date=2018-04-10 |accessdate=2018-04-10|archive-url=https://web.archive.org/web/20180615083218/http://deadline.com/2018/04/alpha-white-boy-rick-swap-release-dates-on-sony-schedule-1202361842/|archive-date=2018-06-15|dead-url=no}}</ref>。

=== 評價 ===
《極地之王》整體獲得了正面居多的評價。在[[匯總媒體|匯總媒體]]網站[[爛番茄|爛番茄]]上根據102篇評論，新鮮度達81%，平均得分6.5 / 10<ref>{{cite web|url=https://www.rottentomatoes.com/m/alpha_2018|title=Alpha (2018)|publisher=[[Rotten_Tomatoes|Rotten Tomatoes]]|accessdate=2018-08-14|archive-url=https://web.archive.org/web/20171205223907/https://www.rottentomatoes.com/m/alpha_2018|archive-date=2017-12-05|dead-url=no}}</ref>。而在[[Metacritic|Metacritic]]上則獲得61分<ref>{{cite web|url=http://www.metacritic.com/movie/alpha|title=Alpha reviews|publisher=[[Metacritic|Metacritic]]|accessdate=2018-08-15|archive-url=https://web.archive.org/web/20180809021536/http://www.metacritic.com/movie/alpha|archive-date=2018-08-09|dead-url=no}}</ref>。據[[CinemaScore|CinemaScore]]調查，觀眾的評價在A+至F間落於「B+」<ref name=opening/>。
