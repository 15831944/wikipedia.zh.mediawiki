《'''極地惡靈'''》（{{lang-en|'''''The Terror'''''}}）是一部美國[[恐怖電影|恐怖]]電視劇，於2018年3月25日在[[AMC_(電視頻道)|AMC]]上播放。本劇改編自[[丹·西蒙斯|丹·西蒙斯]]所寫的2007年暢銷同名小說《{{link-en|極地惡靈 (小說)|The Terror (novel)|極地惡靈}}》<ref>{{Cite web |url = https://deadline.com/2016/03/the-terror-anthology-series-scott-free-1201712908/ |title = AMC Orders ‘The Terror’ Anthology Drama Series From Scott Free |author = Nellie Andreeva |date = 2016-03-02 |publisher = Deadline |language = en}}</ref>，虛構化敘述1845年至1848年{{link-en|富蘭克林的失蹤遠征隊|Franklin's lost expedition|約翰·富蘭克林爵士的失蹤遠征隊}}前往北極的事件。2018年6月22日，宣布AMC已經續訂共有十集的第二季，於2019年播放。<ref>{{Cite web |url = https://tvline.com/2018/06/22/the-terror-renewed-season-2-amc-japanese-internment-camp-world-war-ii/ |title = AMC Renews The Terror for Season 2, Set in WWII Japanese Internment Camp |author = Dave Nemetz |date = 2018-06-22 |publisher = TVLine |language = en}}</ref>
