'''郯城麦坡地震活断层遗址'''，是一个位于[[中国|中国]][[山东省|山东省]][[临沂市|临沂市]][[郯城县|郯城县]][[高峰头镇|高峰头镇]]麦坡村的[[国家级典型地震遗址|国家级典型地震遗址]]。该遗址地处[[马陵山|马陵山]]西坡，断层两侧土壤分异，分别为中生代砖红壤和新生代深褐壤，有“一步亿年”之誉，具有地震科学研究价值<ref name="论文0"/>。

郯城麦坡地震活断层遗址是[[郯城大地震|郯城大地震]]的地质断层遗址，遗址由高低分异的山体岩石组成，属典型的地震错动痕迹，南北长约2600米，东西宽约160米<ref name="论文0"/><ref name="论文1"/>。遗址内同时有碑记等文化遗迹遗产<ref name="论文1"/>。

== 地质背景 ==
郯城麦坡地震活断层遗址所在的郯城县处于沂蒙山南麓，位于[[鲁中南低山丘陵|鲁中南低山丘陵]]南部，居于山东省南部，属[[临郯苍平原|临郯苍平原]]腹心地带，境内地势低平、北高南低，断层纵横<ref name="报道1"/>。县域海拔26-58米，年均降水量919.0毫米，年均气温13.9℃，年均日照时数为2023.7小时，无霜期207天<ref name="报道1"/>。

郯城地区所处的[[郯庐断裂带|郯庐断裂带]]是中国东部最大的巨型断裂带，该断裂带南起长江江北的[[湖北省|湖北省]][[黄梅县|黄梅县]]，经[[安徽省|安徽省]][[庐江县|庐江县]]、山东省郯城县，跨越北海，过[[沈阳|沈阳]]后分为西支的[[依兰—伊通断裂带|依兰—伊通断裂带]]和东支的[[密山—抚顺断裂带|密山—抚顺断裂带]]，总体呈北北东走向<ref name="论文2"/>。断裂带沿线出现了一系列的盆地地貌，该断裂带形成于[[中生代|中生代]]，经历了二至三个演化期<ref name="论文2"/>。郯庐断裂带具有较强的地质活动性，沿断裂带震害活动频繁，历史上主要有1517-1673年、1765年至今两个活跃期，其中前者由北向南迁移发震，后者则由南向北发震<ref name="论文3"/>。

== 遗址历史 ==
清康熙七年六月十七日戌时（公历1668年7月25日20时左右），山东省莒县-郯城县附近发生了震级8.5级的强烈地震，震中烈度达[[中国地震烈度表|中国地震烈度表]]XII(12)度<ref name="论文1"/>。据《康熙郯城县志》记载，“康熙七年六月十七日戌时地震，有声自西北来，一时楼房树木皆前俯后仰，以顶至地者连二三次，逐一颠即倾，城楼垛口、官舍、民房井村落、寺观一时俱倒塌如平地”“其时地裂、泉涌、上喷二三丈高，遍地流水，沟会皆盈，移时即化为乌有”“阁邑震塌房屋约数十万间，其地裂处或缝宽不可越，或深不敢视，其陷处皆如阶级，有层次，裂缝两岸皆有淤泥细沙，其所陷深浅阔狭形状难以备述，真为旷古奇灾” <ref name="论文1"/>。

地震造成震中周围五十多平方千米内的150多个州县受到不同程度破坏，死亡5万余人，有感范围至少达方圆800千米，中国[[山东|山东]]、[[江苏|江苏]]、[[安徽|安徽]]、[[河南|河南]]、[[浙江|浙江]]、[[江西|江西]]、[[湖北|湖北]]、[[直隶|直隶]]、[[陕西|陕西]]、[[山西|山西]]、[[福建|福建]]等省及[[朝鲜半岛|朝鲜半岛]]皆有震感并存有文献记载<ref name="论文1"/>。

2006年8月，[[中国地震局|中国地震局]]批准郯城麦坡地震活断层遗址为国家级典型地震遗址，[[中国科学院|中国科学院]]院士、中国地震局副局长[[丁国瑜|丁国瑜]]为遗址题写了标志碑<ref name="论文0"/><ref name="论文1"/><ref name="报道2"/><ref name="报道3"/>。

== 环境景观 ==
麦坡遗址断层出露部分南北红黑两色岩层交界分明。其中，红色岩层形成于约1万年前，黑色岩层距今1.5亿年，破裂带总长达140千米<ref name="论文1"/>。麦坡遗址是郯庐断裂带内出露条件最好，剖面最典型的一段，南北长约2600米，东西宽约160米<ref name="报道2"/>。有层叠错动、断头沟等地震错动痕迹<ref name="论文1"/>。遗址内同时有碑记等文化遗迹遗产<ref name="论文1"/>。

== 遗址保护 ==
郯城麦坡地震活断层遗址由于属地表地质遗址，因此受到自然风化和人为破坏的双重威胁。目前，该遗址已有部分区段出现了岩石、土层削蚀风化的情况，部分陡崖、沟渠亦受农业种植破坏<ref name="论文1"/><ref name="报道2"/>。有专家呼吁应对部分重点区域进行封闭式隔断保护<ref name="论文1"/>。[[临沂市地震局|临沂市地震局]]亦呼吁，应重视保护这一宝贵遗址，并计划下一步将遗址圈起来重点保护<ref name="报道2"/>。
