'''糯良乡'''（[[佤语|佤语]]：Glong Bung<ref>{{cite book |url=http://mall.cnki.net/reference/ref_search.aspx?bid=R200610031&cid=007 |title=《佤汉简明词典》 |author=颜其香等 |date=1981 |publisher=[[云南民族出版社|云南民族出版社]]}}</ref>），是{{PRC admin/fullname|53/09/27/203/000|3}}下辖的一个[[乡级行政区|乡]]。糯良一词由[[傣语|傣语]]音译，“糯”为水塘，“良”为祭祀，传说塘内有龙，人们常祭祀而得名。乡政府驻糯良村，陆距沧源县城24千米，现任乡长是卫明勇，党委书记是赵海燕。[[沧源佤山机场|沧源佤山机场]]位于糯良乡境内。<ref name="政区大典">{{cite book |author1=段丽元主编 |author2=[[中华人民共和国民政部|中华人民共和国民政部]] |title=《中华人民共和国政区大典 云南卷》 |url= |location=[[北京市|北京]] |publisher=[[中国社会出版社|中国社会出版社]] |page=1502-1503 |date=2016年05月 |isbn=978-7-5087-5306-5}}</ref><ref>{{cite web |url=http://www.xzqh.org/html/show/yn/19053.html |title=糯良乡 |author= |date=2010-06-03 |website=www.xzqh.org |publisher=行政区划网 |access-date=2018-01-06 |quote=}}</ref>

==历史沿革==

早期历史上，糯良由[[勐角董土司|勐角董土司]]管辖。1945年[[沧源设治局|沧源设治局]]废区划乡后，糯良隶属于乐良乡。1954年区划调整，隶属[[勐省镇|勐省区]]（政府驻糯良村）。1968年在[[人民公社化运动|人民公社化运动]]中勐省区改为勐省公社，次年更名九大公社，1971年成立'''糯良公社'''，同时勐省公社迁回勐省村，1973年[[勐来乡|勐来公社]]从糯良公社分出。1984年改置'''糯良区'''，1988年成立糯良乡。<ref name="县志">{{cite book |author1=李明富主编 |author2=沧源佤族自治县地方志编撰委员会 |date=1998年12月 |title=《沧源佤族自治县志》 |url= |location=[[昆明|昆明]] |publisher=云南民族出版社 |page=42 |isbn=7-5367-1498 |ignore-isbn-error=true }}</ref><ref name="民政志">{{cite book |author=沧源佤族自治县民政局、地方志办公室 |date=2006年 |title=《沧源佤族自治县民政志》 |url= |location=[[昆明|昆明]] |publisher=云南民族出版社 |page=33 |isbn=7-5367-3386-0 |author-link= }}</ref>

==地理==

糯良乡位于沧源县中部，自东北顺时针依次与[[勐省镇|勐省镇]]、[[单甲乡|单甲乡]]、[[勐董镇|勐董镇]]、[[勐角傣族彝族拉祜族乡|勐角傣族彝族拉祜族乡]]和[[勐来乡|勐来乡]]接壤。糯良乡总面积140.35平方千米，[[中华人民共和国第六次全国人口普查|2010年普查]]总人口14,665人，居民以[[佤族|佤族]]为主，其次是[[汉族|汉族]]、[[拉祜族|拉祜族]]，佤族占比超过总人口的98%。乡内主要山脉有[[大黑山山脉|大黑山]]、[[班考大山|班考大山]]等，地势南北高，东和东南部低，最高点为大黑山（2,469米），最低点为老温泉和坝尾（1,100米）。主要河流有[[勐董河|勐董河]]、怕秋河（[[拉勐河|拉勐河]]支流）等，属[[澜沧江|澜沧江]]水系，主要水库有[[南撒水库|南撒水库]]等。气候上属[[亚热带季风气候|亚热带季风气候]]，全年气候温和，立体气候显著，年均气温16℃，年均降水1499.5毫米。2011年全乡森林覆盖率达69.54%。矿产主要有[[铅|铅]]、[[锌|锌]]、[[铜|铜]]、[[铁|铁]]、[[石灰岩|石灰岩]]等，其中铜矿和铅锌矿开采量较大，但是当地并无商业冶炼能力，矿石销往中国内地。<ref name="政区大典"></ref><ref name="概况">{{cite book |author1=沧源佤族自治县概况编写组 |author2=沧源佤族自治县修订委员会 |date=2007年11月 |title=《沧源佤族自治县概况》 |url= |location=[[北京|北京]] |publisher=[[民族出版社|民族出版社]] |page=36 |isbn=978-7-105-08553-8 |ignore-isbn-error=true |author-link= }}</ref>

==经济==

糯良乡的经济以农业为主，2011年农民人均纯收入3,237元。2011年，勐角乡年产粮食5,962吨；[[甘蔗|甘蔗]]种植面积8,142亩，年产3.23万吨；茶场1.03万亩，年产鲜叶501.5吨，其他农作物还有{{le|烤烟|Flue-cured tobacco}}、[[核桃|核桃]]、[[油菜|油菜]]等。[[畜牧业|畜牧业]]方面，2011年末生猪和其他大牲畜共存栏16,022头。2011年政府财政收入28万元。<ref name="政区大典"></ref>

==基础设施==

糯良乡内有小学8所，初中1所，在校学生共1,726人。医疗方面有卫生院1所、街道门诊2个、村卫生室8个，共有病床12张，专业卫生人员26人。交通方面，S314[[中华人民共和国公路等级|省道]][[314省道_(云南)|小沧公路]]经过糯良，其他有乡村公路16条共64.25千米，其中砂石路面14条共60.9千米、水泥公路1条2.55千米。尚在规划中的[[瑞孟高速公路|瑞孟高速公路]]将在糯良乡境内设立沧源互通，并将建设连接沧源县城的一级路。[[沧源佤山机场|沧源佤山机场]]位于糯良乡境内，每天有两班往返[[昆明市|昆明]][[昆明长水国际机场|长水国际机场]]的航班，由[[祥鹏航空|祥鹏航空]]和[[中国东方航空|中国东方航空]]运营。<ref name="政区大典"></ref><ref>{{cite web |url=http://news.sina.com.cn/o/2017-09-06/doc-ifyktzim8384532.shtml |title=好消息！瑞丽至孟连将建高速 全长520公里 看看经过你家没？ |last= |first= |date=2017-09-06 |website=news.sina.com.cn |publisher=云南网 |access-date=2017-12-10 |quote=}}</ref><ref>{{cite web |url=http://www.chinanews.com/cj/2016/11-29/8078490.shtml |title=“云端上的机场”云南沧源佤山机场将于12月8日通航 |author=杜潇潇 |date=2016-11-29 |website=www.chinanews.com |publisher=中国新闻网 |access-date=2018-01-06 |quote=}}</ref>
