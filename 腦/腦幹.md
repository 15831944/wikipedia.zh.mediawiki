[[脑|脑]]部除了大脑，小脑，间脑以外的区域，合称'''腦幹'''，由[[中腦|中腦]]（midbrain）、[[橋腦|橋腦]]（pons）、[[延髓|延髓]]（medulla）三部分組成，上接[[間腦|間腦]]、下接[[脊髓|脊髓]]。腦幹位于大脑下方，小脑前方。它負責调节复杂的反射活动，包括调节呼吸作用、心跳、血压等，对维持机体生命有重要意义。



12对脑神经之中除了嗅神经和视神经外，脑干含有动眼神经、滑车神经、[[三叉神经|三叉神经]]、外旋（外展）神经、[[顏面神经|顏面神经]]、聽（前庭蝸）神经、舌咽神经、[[迷走神经|迷走神经]]、副神经及舌下神经这10对处理脑神经讯息的神经核。而[[植物人|植物人]]與[[腦死|腦死]]的差別在於:植物人是因為大腦功能嚴重受損，導致感覺及運動能力喪失，身體處於無意識狀態（只剩軀殼，沒有靈魂的概念）。腦死病人則是生命中樞（腦幹）的功能喪失，因此病人的[[瞳孔|瞳孔]]會放大。病人被醫生宣布腦死後，即使依賴[[人工呼吸系統|人工呼吸系統]]，身體其他器官功能最多也只能維持24小時。

== 结构 ==
# 中腦（midbrain）：位於[[腦幹|腦幹]]最上方。
# 橋腦（pons）：位於[[中腦|中腦]]和[[延髓|延髓]]中間。
# 延腦（medulla）：又稱[[延髓|延髓]]，連結[[橋腦|橋腦]]和[[脊髓|脊髓]]。

{{Link-en|網狀致活系統|reticular activating system}}，又稱'''RAS'''）亦分布在[[腦幹|腦幹]]之中，散佈於[[視丘|視丘]]及[[延髓|延髓]]，負責調節睡眠和清醒間的過渡期、控制意識，使精神集中<ref>成人內外科護理</ref>。

== 功能 ==
* 中腦（midbrain）：是第三及第四對腦神經之起源，負責聽覺、視覺的傳播。
* 橋腦（pons）：第5~8對腦神經之神經核居住於內，內含呼吸調節區，長吸區。
* 延髓（medulla）：第9~12對腦神經之起源，內含心臟中樞、血管運動中樞、呼吸節律中樞，打噴嚏、咳嗽、打嗝、嘔吐、吸吮及吞嚥之控制中樞亦位於此，因此有「生命中樞」之稱。
