《'''竞雄女侠·秋瑾'''》（{{lang-en|'''The Woman Knight of Mirror Lake'''}}）是一部2011年的華語電影，由[[邱禮濤|邱禮濤]]執導，[[黃奕|黃　奕]]、[[黃秋生|黃秋生]]、[[杜宇航|杜宇航]]、[[鄭嘉穎|鄭嘉穎]]、[[陳嘉桓|陳嘉桓]]、[[夏文汐|夏文汐]]、[[林雪|林雪]]、[[熊欣欣|熊欣欣]]、[[劉兆銘|劉兆銘]]、[[黃又南|黃又南]]、[[徐天佑|徐天佑]]、[[劉俊緯|劉俊緯]]、[[閰清|閰清]]主演。

== 花絮 ==
秋瑾外孙女[[美籍华裔|美籍华裔]][[王焱华|王焱华]]强烈质疑该片，9月1日，秋瑾亲侄孙、75岁的[[秋经武|秋经武]]在[[浙江|浙江]][[绍兴|绍兴]]收到了从美国王焱华处寄来的信，称其已向[[中华人民共和国国务院总理|中华人民共和国国务院总理]][[温家宝|温家宝]]和[[中华人民共和国外交部部长|中华人民共和国外交部部长]][[杨洁箎|杨洁箎]]反映了这一情况，他评价该片“庸俗、低俗、媚俗”。<ref>{{cite web |url= http://news.ifeng.com/mainland/detail_2011_09/01/8867624_0.shtml |title=秋瑾后人质疑《竞雄女侠·秋瑾》三俗 致信温家宝反映 |accessdate=2011 |publisher=凤凰网 |work= |language=zh-hans}}</ref>

这是一部商业化包装的历史传记片，由于有诸多不符合历史事实的东西，引发观众不满。<ref>{{cite web |url=http://ent.qq.com/a/20111018/000372.htm |title=《竞雄女侠·秋瑾》—女侠是怎样炼成的 |accessdate=2011 |publisher=腾讯网 |work= |language=zh-hans}}</ref>主创的意思是“七分实三分虚”。<ref>{{cite web |url=http://yule.sohu.com/20111014/n322213954.shtml |title=《竞雄女侠·秋瑾》迅猛但不让人晕血的风格 |accessdate=2011 |publisher=搜狐网 |work= |language=zh-hans}}</ref>

该片曾经迟迟未上映，是没有取得秋瑾后人的授权。<ref>{{cite web |url=http://www.chinanews.com/yl/2011/09-06/3310737.shtml |title=《竞雄女侠·秋瑾》欲十月公映 仍未获得公映许可 |accessdate=2011 |publisher=中新网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://gb.cri.cn/27564/2011/10/10/5753s3396740.htm |title=《竞雄女侠·秋瑾》曲折上映 黄奕低薪演秋瑾 |accessdate=2011 |publisher=国际在线 |work= |language=zh-hans}}</ref>

该片[[北京市|北京市]]首映。<ref>{{cite web |url=http://news.xinhuanet.com/photo/2011-10/08/c_122129840_2.htm |title=电影《竞雄女侠·秋瑾》北京首映 |accessdate=2011 |publisher=新华网 |work= |language=zh-hans}}</ref>
