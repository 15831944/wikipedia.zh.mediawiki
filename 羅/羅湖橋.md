'''羅湖橋'''（{{lang-en|'''Lo Wu Bridge'''}}）位於[[深圳河|深圳河]]之上，是連接[[香港|香港]]和[[中國内地|中國内地]]唯一一座鐵路橋，也是兩地複線的鐵路通道。羅湖鐵路橋前身是一條行人木橋，建於清朝末年<ref>https://www.discuss.com.hk/viewthread.php?tid=12333058&extra=&page=3</ref>。
<ref>{{cite web |title=风雨百年罗湖桥 |url=https://bbs.voc.com.cn/topic-6291436-1-1.html |accessdate=2018-12-11}}</ref>
<ref>{{cite web |title=1951年的罗湖铁路桥 |url=http://www.ilishi.net/html/200909/18789.html |accessdate=2018-12-11}}</ref>

== 历史 ==
=== 第一代 ===
第一代羅湖鐵路橋建於1906年，並於1909年建成，隨同[[九廣鐵路|九廣鐵路]]一同於1910年啟用<ref>{{cite web |url=http://www.kcrc.com/en/images/about/KCRC018_Booklet.pdf |title=存档副本 |accessdate=2010-12-01 |0=2011-07-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20110716043736/http://www.kcrc.com/en/images/about/KCRC018_Booklet.pdf |archivedate=2011-07-16 }}</ref>。
於二次大戰期間，英軍曾拆卸鐵路橋，日軍於佔領香港後曾重建原本鐵路橋，但在日軍撤退時被炸毀<ref>http://news.163.com/special/p/photostory041213.html</ref>。

=== 第二代 ===
英國重新接管香港後著手重建，並於1957年完工<ref>http://news.163.com/41213/5/17G82SBO0001124T.html</ref>。1980年代[[羅湖|羅湖]]設立出入境管制站以前，往返兩地都要徒步走過羅湖橋。

=== 第三代 ===
2003年，[[深圳河|深圳河]]因治理工程擴闊，需要換上較長的橋樑，即現時的羅湖橋，於2004年6月啟用。第二代羅湖橋則遷移至[[羅湖站_(香港)|羅湖車站]]旁保存<ref>{{cite web |url=http://www.dsd.gov.hk/FileManager/TC/publications_publicity/newsletter/news29/29-pg03.htm |title=存档副本 |accessdate=2008-12-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20090304142333/http://www.dsd.gov.hk/FileManager/TC/publications_publicity/newsletter/news29/29-pg03.htm |archivedate=2009-03-04 }}</ref>。
