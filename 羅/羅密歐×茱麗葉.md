《'''羅密歐×茱麗葉'''》（{{lang|ja|ロミオ×ジュリエット}}），改編自[[英國|英國]][[詩人|詩人]]及[[劇作家|劇作家]][[威廉·莎士比亞|威廉·莎士比亞]]劇著《[[羅密歐與茱麗葉|羅密歐與茱麗葉]]》的[[電視動畫|電視動畫]]作品。2007年4月4日至9月26日於[[中部日本放送|中部日本放送]]（CBC）制作播放。

== 故事簡介 ==
從前被遺忘的空中大陸「新·[[維洛那|維洛那]]」，神秘的巨大力量「愛斯卡勒斯」使其漂浮在空中，為大地帶來了生氣及人民的安定繁榮。《羅密歐與茱麗葉》所講述的是一對被命運捉弄的戀人，14年前蒙太古家族的血腥政變取代了凱普萊特家族的開明統治，從此進入蒙太古的暴政時代，而他們純真的愛情故事，14年後由此在無情的亂世中正式展開……

兩大家族=凱普萊特（另譯=卡帕萊特） 蒙太古（另譯=蒙特鳩）

== 登場人物 ==
=== 主要角色 ===
; {{nihongo|羅密歐·康多雷邦特·蒙太古|ロミオ|聲：[[水島大宙|水島大宙]]／臺︰[[賀宇傑|賀宇傑]]}} 
: 蒙太古家的兒子，對父王的暴政反感，在一次的機緣中邂逅了茱麗葉是兩人崎嶇情路的開始。為了茱麗葉，能夠犧牲自己的一切。
; {{nihongo|茱麗葉·赫爾瑪塔·阿斯洛·凱普萊特|ジュリエット|聲：[[水沢史絵|水沢史絵]]／臺︰[[謝佼娟|謝佼娟]]}}
: 凱普萊特家唯一倖存的族人，長年假扮為男孩子奧丁及化為赤色旋風行俠仗義，16歲生日時由撫養她的遺臣康拉德告知身世，從此走上愛情及繼承家業的坎坷道路。

聚集在鳶尾花的旗幟下，捲起自由之風。
; {{nihongo|裘里奧|キュリオ|聲：[[鳥海浩輔|鳥海浩輔]]／臺︰[[劉傑|劉傑]]}}

; {{nihongo|考狄利亞|コーディリア|聲：[[松來未祐|松來未祐]]／臺︰[[林美秀|林美秀]]}}

; {{nihongo|蒙太古|モンタギュー|聲：[[石井康嗣|石井康嗣]]／臺︰[[詹君陽|詹君陽]]}}

; {{nihongo|茂丘西奧|マキューシオ|聲：[[柿原徹也|柿原徹也]]／臺︰[[于正昌|于正昌]]}}

; {{nihongo|提圖斯|タイタス|聲：[[浦山迅|浦山迅]]／臺︰[[于正昌|于正昌]]}}

; {{nihongo|阿布拉姆|アブラム|聲：[[飯田浩志|飯田浩志]]}}
: 蒙太古家親衛隊長。

== 用語解釋 ==
; {{nihongo|新·維洛那|天空浮遊大陸ネオ・ヴェローナ}}
: 故事展開的舞臺的地方，天空浮遊的大陸。
; {{nihongo|愛斯卡勒斯|大樹エスカラス}}（另譯=艾斯加拉斯）
: 「新·維洛那」繁榮起源的雙子大樹。除了光、水和土地，憐愛也是使其生長的必要條件。可是，由於蒙太古的暴政，樹的心臟開始變得衰弱，葉子一片一片枯落。
; {{nihongo|龍馬|竜馬}}
: 羅密歐平日以名為'''希艾羅'''（另譯=雪洛）的龍馬代步，外表和馬匹無異，背部長有巨大的翅膀，擁有像龍一樣的尾巴，可上天下地的動物。裝備馬鞍和韁繩後，能像馬匹那樣策騎。不是普通百姓能擁有的東西，象徵著皇族及貴族的專利品。

2007年4月13日開始，每逢星期五於日本[[音泉|音泉]]網絡廣播劇廣播。由聲演[[茱麗葉|茱麗葉]]的配音員{{lang|ja|[[水沢史絵|水沢史絵]]}}，以及考狄莉亞的配音員{{lang|ja|[[松来未祐|松来未祐]]}}負責主持。
