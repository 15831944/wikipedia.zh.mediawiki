'''羅伯特·卡普榮'''（英語：Robert B. Capron Jr.，{{bd|1998年|7月9日}}）<ref>{{cite web |url=http://www.phillyburbs.com/entertainment/local_entertainment/beloved-wimp-back-for-more-sweet-natured-fun/article_01c7fc90-7cf9-5619-a0d3-8c069a2f8e9f.html |title=Beloved wimp back for more sweet-natured fun – phillyBurbs.com : Local Entertainment |accessdate=March 25, 2011 |deadurl=yes |archiveurl=https://web.archive.org/web/20120318070806/http://www.phillyburbs.com/entertainment/local_entertainment/beloved-wimp-back-for-more-sweet-natured-fun/article_01c7fc90-7cf9-5619-a0d3-8c069a2f8e9f.html |archivedate=March 18, 2012 |df= }}</ref>是一位美國演員，另外他也是[[配音員|配音員]]，[[電影製片人|電影製片人]]，[[喜劇演員|喜劇演員]]和[[電影導演|電影導演]]，他因為在《遜咖冒險王》系列電影的前三集中，飾演主角葛瑞·海夫里（Greg Heffley）最好的朋友羅利·傑佛森（RowleyJefferson）而打開知名度。

羅伯特第一次登上大螢幕是在電影《[[新娘大作戰|新娘大作戰]]》裡軋了一角，之後在電影《[[忠犬小八|忠犬小八]]》及迪士尼電影《[[魔法師的學徒_(2010年電影)|魔法師的學徒]]》中也都有部分的演出。而在《遜咖冒險王》系列電影中，除了他的演出之外，他與他的母親凱伊·卡普榮（Kaye Capron）不僅在現實生活是母子關係，
也片中也是飾演一對母子。卡普榮住在[[羅德島|羅德島]]的斯基尤特（Scituate）。他目前在[[布朗大学|布朗大學]]修學戲劇，電影和編劇。
