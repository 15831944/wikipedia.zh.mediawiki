'''羅蘭'''（{{Lang-en|Roland}}，{{Lang-ja|ローランド}}，{{bd|1992年|7月27日|||}}）是日本一名{{Link-ja|公關 (日本)|ホストクラブ|公關}}、[[模特兒|模特兒]]、[[日本電視藝人|藝人]]和[[商人|商人]]。他以日本公關史上收入最高的男公關而聞名，年薪在3億日元左右。<ref>{{cite web |title=不陪酒也不陪睡！日本第一牛郎靠這招年收8千萬 長相曝光 |url=https://news.ebc.net.tw/News/Article/124614 |publisher=[[東森新聞|東森新聞]] |accessdate=2018-12-30}}</ref><ref>{{cite web |author1=田暐瑋 |title=隨便逛就買下勞斯萊斯！日本第一牛郎「賣臉不賣身1年就撈8千萬」奢華生活曝光 |url=https://star.ettoday.net/news/1229772 |publisher=ETtoday星光雲 |accessdate=2018-12-30}}</ref><ref>{{cite web |title=日本第一牛郎 年赚360万 |url=https://www.zaobao.com.sg/znews/others/story20180808-881743 |website=[[聯合早報|早報網]] |publisher=[[新明日報|新明日報]] |accessdate=2018-12-30}}</ref> 目前其經紀合約隸屬於由他所開創的羅蘭事務所（{{lang|en|Roland Office}}）。羅蘭同時也是羅蘭戴爾股份公司（{{lang|ja|ローランデール株式会社}}）[[董事長|董事長]]，<ref>{{cite web |title=現代ホスト界の帝王Rolandが究極のヘアオイルを研究開発!上質なシルクのような髪に導く「THE GOLD」サラサラとした指通りを叶える「THE BLACK」を2019年2月6日に先行発売開始。 |url=https://www.nikkan.co.jp/releases/view/88610 |publisher=日刊工業新聞 |accessdate=2018-12-30}}</ref> 以及施瓦茨股份公司（{{lang|ja|株式会社シュヴァルツ}}）和茱莉亞集團（{{lang|en|JULIA GROUP}}）的代表董事。
