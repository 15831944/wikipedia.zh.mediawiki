== 文字 ==
=== 歷史 ===
儘管羅興亞語與其他的孟加拉語言關係緊密，但其書寫文字原本是用[[阿拉伯字母|阿拉伯字母]]寫的<ref name=scribd>{{cite book |url=http://www.scribd.com/doc/21180958/Rohingya-Language-Book-A-Z |title=Starting with Rohingyalish |author=Mohammed Siddique Basu |language=en |accessdate=2012-04-18}}</ref>，而不是採用[[孟加拉文|孟加拉文]]。而近年羅興亞人亦積極開發用其他文字系統來書寫羅興亞語，例如：[[拉丁字母|拉丁字母]]、[[缅文|缅文]]、又或採用[[烏爾都字母|烏爾都字母]]之類與傳統阿拉伯字母不同的地方變體。當中以拉丁字母為近年的標準。

最早期以阿拉伯字母書寫羅興亞語的紀錄有超過200年<ref name=proposal />。當[[阿拉干地區|阿拉干地區]]仍在英國殖民統治（1826年–1948年）時，羅興亞人主要以英語及烏爾都語作文字上的溝通。1948年緬甸從英國獨立之後，所有官方溝通均採用緬文。直到1960年代初，羅興亞人學者才意識到他們要有一套用來書寫自己語言的系統。

1975年，學者開發出一套用阿拉伯字母書寫羅興亞語的系統；另外有學者透過採納烏爾都字母的額外文字來彌補標準阿拉伯字母的不足。然而，兩者皆不滿意，因為大多數羅興亞人均覺得很難閱讀。

隨着電腦和互聯網的普及，令民眾意識到若要令文字普及化，就一定要容易在電腦上使用。於是就出現了今日的拉丁字母方案。

羅興亞語採用的拉丁字母除了使用原有A到Z26個字母以外，還包括兩個額外的[[衍生拉丁字母|衍生拉丁字母]]（上表綠底文字）：[[Ç|Ç]]用來表示[[捲舌音|捲舌]]R（[{{IPAlink| ɽ}}]）<ref name=proposal>{{cite web |first1=Lorna A. |last1=Priest |first2=Martin |last2=Hosken |author=[[SIL_International|SIL International]] |title=''Proposal to add Arabic script characters for African and Asian languages'' |year=2010 |date=2010-08-12 |url=http://www.dkuug.dk/jtc1/sc2/wg2/docs/n3882.pdf |pages=13-18, 34-37}}</ref>，而[[Ñ|Ñ]]用來表示跟前的元音[[鼻音化|鼻音化]]（例如：「五」''fañs'' {{IPA|/faⁿs/}}）。另外，字母[[C|C]]表示的是{{IPA|/ʃ/}}音。為更準確地表示羅興亞語的音韻，五個元音<aeiou>可與[[尖音符|尖音符]]結合成為<áéíóú>。羅興亞語的[[ISO_639-3|ISO 639-3]]代碼為“RHG”<ref>[http://www.sil.org/iso639-3/codes.asp?order=639_3&letter=r ISO 639 Code Tables - SIL International] {{webarchive|url=https://web.archive.org/web/20110618154156/http://www.sil.org/iso639-3/codes.asp?order=639_3&letter=r |date=2011-06-18 }}</ref>。基本上，所有字母均可以在標準[[键盘布局#美国国际|美国国际键盘布局]]輸入。

另一方面，為保存過往用阿拉伯字母的文本，[[Unicode|Unicode]]已通過在第11版撥出區段（U+10D00..10D39）加入羅興亞語的50個額外字母與標調字母，用以表示他們使用的[[哈乃斐羅興亞文字|哈乃斐羅興亞文字]]<ref name=proposal /><ref>{{cite web |url=http://www.unicode.org/alloc/Pipeline.html |archive-url=https://web.archive.org/web/20171203035836/http://www.unicode.org/alloc/Pipeline.html |title=Proposed New Characters: The Pipeline |date=2017-11-28 |archive-date=2017-12-03 |access-date=2018-01-21 |language=en }}</ref>。

==== 元音 ====
羅興亞語有六個單元音和六個雙元音<ref name=scribd />，當中以 <o>/<ó> 來表示「開口o」（[{{IPAlink|ɔ}}]），而 「閉口o」（[{{IPAlink|o}}]） 則以 <ou>/<óu> 來表示。不過根據拼寫法則， <ou> 視作兩個字母。四個較常見的雙元音為<ai>/<ái>、<ei>/<éi>、<oi>/<ói>及<ui>/<úi>，<au>/<áu>和<wa>/<wá>。

==== 聲調 ====
羅興亞語會在強調的元音上加上[[重音符|重音符]]，而重覆元音即延長其發音時間<ref name=website />。透過強調重覆元音的其中一個，可用來表達不同的聲調，例如：<aá>、<áa>及<áá>，連同<aa>合共四個聲調<ref name=website />。
