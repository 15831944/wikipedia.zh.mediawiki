'''羅詠詩'''，[[JP|JP]]（'''Lo Wing Sze Anthea'''，{{bd|1971年|7月22日|catIdx=Lo羅}}），[[香港|香港]]出生，籍貫[[廣東|廣東]][[汕尾|汕尾]][[海豐|海豐]]，為現任萬通集團[[董事總經理|董事總經理]]及M1酒店[[創辦人|創辦人]]兼[[董事|董事]] <ref>[http://newwmagazine.com/visit_detail.php?id=12 旅遊世家第二代 萬通集團掌舵人 羅詠詩 展現新一代女性典範]，財通雜誌，2016年3月4日</ref>，是[[香港旅遊業|香港旅遊業]]界人物。

羅詠詩同時擔任多項政府公職和社會服務組織的領導層，是[[嶺南大學|嶺南大學]][[校董會|校董會]]成員、輸入優秀人才及專才諮詢委員會委員<ref>[http://www.info.gov.hk/gia/general/201806/27/P2018062700392.htm 政府委任輸入優秀人才及專才諮詢委員會成員]，香港特別行政區政府新聞公報，2018年6月27日</ref>、[[勞工及福利局|勞工及福利局]][[社會福利|社會福利]]諮詢[[委員會|委員會]]委員<ref>[http://www.info.gov.hk/gia/general/201511/20/P201511200390.htm 政府公布社會福利諮詢委員會成員任命]，香港特別行政區政府新聞公報，2015年11月20日</ref>、[[康樂文化事務署|康樂文化事務署]][[博物館|博物館]]諮詢[[委員會|委員會]]委員<ref>[http://www.info.gov.hk/gia/general/201610/20/P2016102000764.htm 新的博物館諮詢委員會成立]，香港特別行政區政府新聞公報，2016年10月20日</ref>、[[華人廟宇委員會|華人廟宇委員會]]財務及管理小組委員、非本地高等及專業教育上訴委員會委員<ref>[http://www.info.gov.hk/cml/cbc/a149.htm 非本地高等及專業教育上訴委員會]，各公務委員會及其他名表</ref>。羅詠詩亦是百仁基金[[副主席|副主席]]兼[[秘書長|秘書長]] <ref>[http://www.charity-online.org/board-of-director/ 百仁基金 主席、副主席]，百仁基金</ref>、[[香港女童軍總會|香港女童軍總會]]名譽副會長<ref>[http://www.hkcd.com.hk/content/2015-11/06/content_3505405.htm 女童軍周年晚宴嘉賓雲集]，香港商報，2015年11月6日</ref><ref>[http://orientaldaily.on.cc/cnt/news/20160115/00176_120.html 女童軍注新血 玩轉百周年]，東方日報，2016年1月15日</ref>、國際小母牛香港分會籌款委員會委員<ref>[https://www.heiferhk.org/zh-hant/about-heifer/committees/ 籌款委員會]，國際小母牛香港分會</ref>及[[大坑火龍|大坑火龍]]文化館(第四期[[活化歷史建築夥伴計劃|活化歷史建築夥伴計劃]])理事。

== 簡介 ==
羅詠詩高中起負笈[[澳洲|澳洲]]，先後於[[悉尼大學|悉尼大學]]和[[新南威爾斯大學|新南威爾斯大學]]畢業，分別取得[[經濟|經濟]]系[[學士|學士]]和[[財經|財經]]系[[碩士|碩士]]學位，並同步取得澳洲註冊[[會計師|會計師]]專業資格。

1996年從[[澳洲|澳洲]]回港後，羅詠詩在香港[[匯豐銀行|匯豐銀行]]曾任[[管理|管理]][[見習生|見習生]]及在[[香港城市大學|香港城市大學]]曾任[[管理|管理]]系一級[[助教|助教]]。於1999年起投身[[旅遊|旅遊]][[服務|服務]]界，就任萬通集團[[董事總經理|董事總經理]]，並於2010年同時兼任M1酒店[[創辦人|創辦人]]兼[[董事|董事]]<ref>[http://www.31bm.com/da/836.html 罗咏诗- 名人堂]，生意场</ref>。

== 學歷 ==
羅詠詩在1994年於[[澳洲|澳洲]][[悉尼大學|悉尼大學]]取得[[經濟|經濟]]系[[學士學位|學士學位]]，主修[[經濟|經濟]]及[[會計|會計]]並副修[[商業|商業]][[法律|法律]]及[[稅務|稅務]]；在1996年畢業於[[澳洲|澳洲]][[新南威爾斯大學|新南威爾斯大學]]為[[財經|財經]]系[[碩士|碩士]]，主修[[銀行業|銀行業]]及[[財經|財經]]並副修[[市場學|市場學]]及[[資訊科技|資訊科技]]，同時取得澳洲註冊會計師資格。於2018年取得香港會計師資格。

== 家庭 ==
羅詠詩已婚，育有兩名兒子<ref>[http://orientaldaily.on.cc/cnt/news/20160930/00176_090.html 羅詠詩靚囝囝 英國會考做狀元]，東方日報，2016年9月30日</ref>。

== 事業發展 ==
[[File:M1酒店榮獲城市青年商會主辦的「第十七屆創意創業大賞」.jpg|thumb]]
羅詠詩於1999年就任萬通集團[[董事總經理|董事總經理]] <ref>[http://www.hsmrt.com/luoyongshi/2460.html 万通集团董事罗咏诗]，华商韬略—华商名人堂</ref>，協助集團發展[[一站式|一站式]][[旅遊|旅遊]][[服務|服務]]，擁有龐大的[[旅遊|旅遊]]車隊和[[主題公園|主題公園]]與[[旅遊|旅遊]]景點銷售網絡，集團每年皆獲頒[[香港迪士尼樂園|香港迪士尼樂園]]卓越銷售獎和[[海洋公園|海洋公園]]旅行社最高銷售獎；亦作橫向發展至開創[[物業|物業]][[投資|投資]]和[[信貸|信貸]]服務等業務。

為M1酒店[[創辦人|創辦人]]兼[[董事|董事]]的羅詠詩帶領集團於2010年創立旗下的精品酒店品牌「M1」，開展本身的精品酒店業務<ref>[http://ps.hket.com/article/486906 「舊樓改建酒店 備150房間最佳」萬通旅運總監：首選單邊坐向物業]，地產站Property Station，2014年12月1日</ref><ref>[http://www1.hkej.com/dailynews/culture/article/1101276/%E6%97%85%E9%81%8A%E4%B8%96%E5%AE%B6%E7%AC%AC%E4%BA%8C%E4%BB%A3%E7%BE%85%E8%A9%A0%E8%A9%A9++%E6%86%91%E6%95%B8%E6%93%9A%E9%80%B2%E6%94%BB%E7%B2%BE%E5%93%81%E9%85%92%E5%BA%97 旅遊世家第二代羅詠詩 憑數據進攻精品酒店]，信報，2015年7月21日</ref>。首間M1酒店座落於[[旺角|旺角]][[油麻地|油麻地]][[砵蘭街|砵蘭街]] <ref>[http://paper.wenweipo.com/2010/04/12/ME1004120005.htm 砵蘭街酒店1.65億易手]，文匯報，2010年4月12日</ref>，憑著長期高入住率及貼心獨特的服務及配套，酒店於2013年榮獲城市青年商會主辦的「第十七屆創意創業大賞」<ref>[https://www.youtube.com/watch?v=zB2yyraPHh0 第17屆稻香創意創業大賞得獎者 - M1 酒店: 羅詠詩 得獎訪問]，YouTube，2013年8月17日</ref>。

2015年中，第二間M1酒店於[[北角|北角]]開幕，環境及設計別具香港地區特色，深受外國旅客歡迎。

羅詠詩於2016年獲資本才俊月刊頒發年度 CEO 獎項<ref>[https://www.facebook.com/CapitalCEO.HK/photos/a.427271107343133.97722.420425681361009/1149501611786742/?type=3&theater 「CAPITAL CEO Nite 2016」頒獎典禮 得獎 CEO]，Facebook - CAPITAL CEO 資本才俊，2016年9月2日</ref>，其商業成就獲得市場肯定。

隨著首兩間酒店的成功，第三間位於[[旺角|旺角]]的M1酒店於2017年正式開幕。

== 社會服務 ==
[[File:M1酒店學生生涯規劃服務行業探索.jpg|thumb]]
[[File:百仁基金菁青相惜分享會.png|thumb]]
[[File:書館街十二號活化為大坑火龍文化館.png|thumb]]
羅詠詩自2006年起積極投身社會服務和公益慈善活動，推廣[[青少年|青少年]][[生涯規劃|生涯規劃]]服務、文化保育與[[社企|社企]]關愛三大範疇。

2009年加入香港青年聯會並擔任賢慧社任[[主席|主席]] <ref>[http://the-sun.on.cc/cnt/news/20110808/00410_010.html SUN世代：賢慧女強人做好行善呢份工]，太陽報，2011年8月8日</ref>，參與籌辦多項慈善服務。2012年連同賢慧社成員及義工團到清遠連山山區義教<ref>[http://orientaldaily.on.cc/cnt/news/20120813/00176_087.html 青聯賢慧社山區教英文]，東方日報，2012年8月13日</ref>。於2010年至2013年共籌辦四屆「青聯賢慧愛家日」<ref>[http://orientaldaily.on.cc/cnt/news/20120303/00176_079.html 賢慧三花籌備愛家日]，東方日報，2012年3月3日</ref>，帶領超過3,000位來自基層家庭的小童及家長免費遊覽海洋公園<ref>[http://orientaldaily.on.cc/cnt/news/20120411/00176_089.html 1500貧戶成員 暢遊海洋公園]，東方日報，2012年4月11日</ref><ref>[http://paper.wenweipo.com/2013/04/14/AY1304140002.htm 青聯賢慧愛家日 800童遊海洋公園]，文匯報，2013年4月14日</ref>。於2010年至2014年共舉辦八場「半邊天講座系列」<ref>[http://paper.wenweipo.com/2014/01/13/AY1401130005.htm 何超瓊與60才俊分享心得]，文匯報，2014年1月13日</ref>，邀請城中傑出女士及政商界名人與會員分享心得。

2010年加入[[香港交通安全隊|香港交通安全隊]]西九龍總區<ref>[http://orientaldaily.on.cc/cnt/news/20111204/00176_079.html 青聯羅詠詩入交通安全隊]，東方日報，2011年12月4日</ref>，直至2014年，歷任[[指揮官|指揮官]]與總隊監，宣揚交通安全工作。

2014年起擔任國際小母牛香港分會籌款委員會委員，支持各項籌款活動以及小母牛競步善行<ref>[http://orientaldaily.on.cc/cnt/news/20141103/00176_089.html 小母牛慈善賽 杜擴業愈跑愈有]，東方日報，2014年11月3日</ref>。

2014年起擔任百仁基金[[副主席|副主席]]兼[[秘書長|秘書長]]，每年推出「菁青相惜」分享會<ref>[http://www.charity-online.org/%E3%80%8C%E8%8F%81%E9%9D%92%E7%9B%B8%E6%83%9C%E5%88%86%E4%BA%AB%E6%9C%83%E3%80%8D-%E6%9D%B1%E8%8F%AF%E4%B8%89%E9%99%A2%E7%9B%A7%E5%B9%B9%E5%BA%AD%E7%B4%80%E5%BF%B5%E4%B8%AD%E5%AD%B8/ 「菁青相惜分享會」- 東華三院盧幹庭紀念中學 主講嘉賓羅詠詩]，百仁基金，2014年5月15日</ref>，由各界名人和青年企業家，前往中學分享本身在成長和面對挑戰時的見聞與感受。數年間有超過90間中學參與，累積受惠學生逾31,500人，並向參與學校提供了超過三百萬港元的生涯規劃項目資助<ref>[http://news.takungpao.com.hk/paper/q/2016/0118/3269220.html 百仁基金助港生生涯規劃]，大公網，2016年1月18日</ref>。百仁基金同步發展「百仁之友」計劃<ref>[http://www.charity-online.org/centum-youth/ 百仁之友]，百仁基金</ref>，亦為8,000多名中學生舉辦中港兩地行業探索、領袖訓練和考察實習。

2014年起統籌「百仁基金暑期實習計劃」，邀請數十間企業提供百多個多元化的中港實習職位，讓大學生親自在現實工作中，真正了解能力、興趣與志向的融匯。

2015年起擔任百仁基金與香港齊心基金會合辦的「百仁齊心多角度辯論賽」總統籌，以四角辯論模式，由參賽學校就不同議題展開辯論，刺激同學為社會議題作多角度思考<ref>[http://news.takungpao.com.hk/paper/q/2016/0506/3315052.html 百仁齊心校際辯論賽聚焦熱點]，大公網，2016年5月6日</ref>。

2015年以執行委員和主要聯繫人身分，協助齊惜福 (Food for Good)籌辦「齊惜福之友」，籌得逾八百萬港元的款項，支持液化廚餘機長機運作，推廣綠色飲食文化。

2015年向大坑坊眾福利會作出主要贊助，成功協助其獲選為第四期[[活化歷史建築伙伴計劃|活化歷史建築伙伴計劃]]，把位於[[大坑|大坑]]書館街的[[三級歷史建築|三級歷史建築]]前孔聖義學學堂，活化成「大坑火龍文化館」<ref>[https://topick.hket.com/article/616335/%E6%B2%B3%E6%9D%B1%E5%A4%AB%E4%BA%BA%E9%86%AB%E5%B1%80%E6%B4%BB%E5%8C%96%E6%88%90%E7%94%9F%E6%85%8B%E7%A0%94%E7%A9%B6%E4%B8%AD%E5%BF%83 書館街12號活化為火龍文化館]，TOPick新聞，2015年6月17日</ref><ref>[http://news.takungpao.com.hk/paper/q/2015/0618/3027738.html 書館街12號變火龍餐廳]，大公網，2015年6月18日</ref>。並親自擔任大坑火龍文化館(第四期活化歷史建築伙伴計劃) 主席，協助為期四年的籌辦，以及日後的推廣和營運<ref>[https://www.heritage.gov.hk/tc/rhbtp/Tai_Hang_Fire_Dragon_Heritage_Centre4.htm 第四期活化計劃 書館街12號 : 大坑火龍文化館]，香港歷史文物- 保育‧活化</ref>。[[大坑舞火龍|大坑舞火龍]]獲列入第三批[[中國國家級非物質文化遺產|中國國家級非物質文化遺產]]名錄<ref>[http://www.info.gov.hk/gia/general/201106/03/P201106030223.htm 香港四項傳統活動成功列入國家級非物質文化遺產名錄]，香港特別行政區政府新聞公報，2011年6月3日</ref>，是香港首個擁有展館的相關項目。
