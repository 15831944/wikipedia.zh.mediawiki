'''羅伯特·伊萊爾'''（{{lang-en|'''Robert Iler'''}}）是一名美國演員，他最出名是在[[HBO|HBO]]電視劇《[[人在江湖_(美國電視劇)|人在江湖]]》中飾演{{link-en|Anthony Soprano Jr.|Anthony Soprano Jr.}}。<ref>{{Cite web |url = https://www.esquire.com/entertainment/tv/news/a45677/sopranos-cast-where-are-they-now/ |title = The Cast of The Sopranos: Where Are They Now? |date = 2016-06-11 |publisher = esquire.com |language = en}}</ref>

== 早年 ==
伊萊爾出生於[[紐約市|紐約市]]，而且他有[[愛爾蘭人|愛爾蘭]]血統。<ref>{{Cite web |url = https://usatoday30.usatoday.com/life/television/news/2007-03-29-sopranos-cover_N.htm |title = Gandolfini, Iler sit down for father-son chat |publisher = USA Today |language = en}}</ref><ref>{{Cite web |url = http://transcripts.cnn.com/TRANSCRIPTS/0105/18/lkl.00.html |title = Cast of `Sopranos' Discusses Show's Success |date = 2001-05-18 |publisher = CNN |language = en}}</ref>
