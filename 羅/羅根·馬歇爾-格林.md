'''羅根·馬歇爾-格林'''（{{lang-en|'''Logan Marshall-Green'''}}，{{bd|1976年|11月1日||}}）是[[美國|美國]]的一位演員。他出演過《[[24_(電視劇)|24]]》、《[[橘郡風雲|橘郡風雲]]》、《[[亡命天涯_(美国电视剧)|亡命天涯]]》、《[[暗警|暗警]]》、《[[戰後啟示錄|戰後啟示錄]]》等電視劇，還有演出電影《[[普罗米修斯_(电影)|普羅米修斯]]》<ref>{{Cite web |url = https://www.vanityfair.com/hollywood/2012/06/logan-marshall-green-prometheus-ridley-scott-michael-fassbender-charlize-theron |title = Logan Marshall-Green on Prometheus, Childhood Fantasies, and Working with Ridley Scott |author = Julie Miller |date = 2012-06-08 |publisher = vanity fair |language = en}}</ref><ref>{{Cite web |url = https://metro.co.uk/2012/02/06/prometheus-photo-shows-noomi-rapace-with-logan-marshall-green-310050/ |title = Prometheus photo shows Noomi Rapace with Logan Marshall-Green |author = Ann Lee |date = 2012-02-06 |publisher = Metro.co.uk |language = en}}</ref>、《[[蜘蛛俠：強勢回歸|蜘蛛俠：強勢回歸]]》和《[[人類升級|人類升級]]》<ref>{{Cite web |url = http://collider.com/logan-marshall-green-interview-upgrade-quarry-season-2/#images |title = Logan Marshall-Green on ‘Upgrade’ and If He Knows What ‘Quarry’ Season 2 Would Have Been About |author = Steve 'Frosty' Weintraub |date = 2018-05-30 |publisher = collider |language = en}}</ref><ref>{{Cite web |url = https://www.hollywoodreporter.com/heat-vision/upgrade-premiere-logan-marshall-green-writer-director-leigh-whannell-jason-blum-talk-blumhouses-firs-1116192 |title = How 'Upgrade' Became Blumhouse's First Sci-Fi Film |author = Alex Cramer |date = 2018-05-31 |publisher = The Hollywood Reporter |language = en}}</ref>。

== 早年生活 ==
馬歇爾-格林生于[[南卡羅萊納州|南卡羅萊納州]]的[[查尔斯顿_(南卡罗来纳州)|查尔斯顿]]，父母都是教師<ref>{{Cite web |url = https://www.theatermania.com/new-york/news/rising-stars_6282.html |title = Rising Stars |author = Michael Portantiere |date = 2005-07-07 |publisher = TheaterMania |language = en}}</ref>。他有一個雙生兄弟名叫Taylor，他們兩人於1990年代早期在{{link-en|巴靈頓高中 (羅德島州)|Barrington High School (Rhode Island)|巴靈頓高中}}上學。他在[[田納西大學|田納西大學]]接受了本科教育，並在大學裡為學校報紙《''The Daily Beacon''》當一名娛樂作家，報道酒吧、音樂和戲劇界的事。

他後來参加了[[纽约大学|纽约大学]]的研究生表演计划，获艺术硕士学位。

== 生涯 ==
馬歇爾-格林曾演出2003年的《[[法律與秩序：特殊受害者|法律與秩序：特殊受害者]]》和2004年的《[[律政風雲|律政風雲]]》，之後在[[福斯廣播公司|福斯廣播公司]]的《[[橘郡風雲|橘郡風雲]]》和《[[24_(電視劇)|24]]》中飾演常設角色。

馬歇爾-格林因在2004年{{link-en|尼爾·拉布特|Neil LaBute}}的舞台劇《''The Distance from Here''》中的演出而獲得一項{{link-en|戲劇桌獎|Drama Desk Award}}。<ref>{{Cite web |url = http://www.dramadesk.org/awards/?y=2004&n=&a=ALL |title = Nominees and Recipients - Drama Desk - 2004 |publisher = Drama Desk |language = en}}</ref>

他在2012年演出[[列尼·史葛|列尼·史葛]]的電影《[[普羅米修斯_(電影)|普羅米修斯]]》，在片中飾演普羅米修斯號的船員Holloway。他也在{{link-en|凱琳·庫薩瑪|Karyn Kusama}}的2015年恐怖驚悚電影《[[詭宴|詭宴]]》中飾演Will。

2018年，馬歇爾-格林在[[雷·沃納爾|雷·沃納爾]]的科幻[[肉體恐怖|肉體恐怖]]電影《[[人類升級|人類升級]]》中飾演主角<ref>{{Cite web |url = https://variety.com/2016/film/news/prometheus-star-logan-marshall-green-stem-1201943796/ |title = ‘Prometheus’ Actor Logan Marshall-Green to Star in Sci-Fi Movie ‘Stem’ |author = Dave McNary |date = 2016-12-15 |publisher = Variety |language = en}}</ref>。他第一次執導電影《''{{link-en|Adopt a Highway (film)|Adopt a Highway (film)|Adopt a Highway}}''》，主演有[[伊芬·鶴健|伊芬·鶴健]]和{{link-en|貝蒂·加布里埃爾|Betty Gabriel}}，以及由[[傑森·布倫|傑森·布倫]]擔當監製。

2018年7月26日，馬歇爾-格林在youtube播客節目The Official Podcast上出現。<ref>{{Cite web |url = https://www.youtube.com/watch?v=if5_TiZNtC4 |title = The Official Podcast #86 With Logan Marshall-Green |publisher = youtube |language = en}}</ref>

== 個人生活 ==
馬歇爾-格林在2012年12月12日與女演員Diane Gaeta結婚<ref>{{Cite web |url = https://marriedwiki.com/wiki/diane-gaeta |title = Diane Gaeta - Biography |publisher = marriedwiki.com |language = en}}</ref>。他們生了一名兒子<ref>{{Cite web |url = https://www.eonline.com/ca/news/531567/logan-marshall-green-welcomes-baby-boy-tennessee-see-pics-of-his-newborn-son |title = Logan Marshall-Green Welcomes Baby Boy Tennessee—See Pics of His Newborn Son! |date = 2014-04-14 |publisher = eonline.com |language = en}}</ref>。馬歇爾-格林也是他妻子的女兒的繼父，而女兒的父親是演員{{link-en|強尼·劉易斯|Johnny Lewis}}<ref>{{Cite web |url = http://www.lamag.com/longform/the-secret-life-of-johnny-lewis/ |title = The Secret Life of Johnny Lewis |author = Bill Jensen |date = 2014-01-30 |publisher = Los Angeles |language = en}}</ref>。
