'''罗马尼亚'''（{{lang-ro|România}}），位於[[東歐|歐洲東部]]。羅馬尼亞國境西方分別為[[匈牙利|匈牙利]]與[[塞爾維亞|塞爾維亞]]接壤，[[保加利亞|保加利亞]]在南方，北方與东北方則是[[乌克兰|乌克兰]]與[[摩尔多瓦共和國|摩尔多瓦共和國]]。羅馬尼亞有一小段位於[[黑海|黑海]]邊的海岸線，[[多瑙河|多瑙河]]幾乎佔了南邊與[[保加利亞|保加利亞]]之間的國界大半，並且從該國东邊海岸流入黑海的西岸。羅馬尼亞的首都為[[布加勒斯特|布加勒斯特]]，位於該國南部[[多瑙河|多瑙河]]支流[[登博维察河|登博维察河]]所流經的平原地帶上，是羅馬尼亞第一大城與工商業城市。

== 歷史 ==
{{main|羅馬尼亞歷史}}
[[File:RomaniaBorderHistoryAnnimation_1859-2010.gif|thumb]]

[[舊石器時代|舊石器]]及[[新石器時代|新石器時代]]，在羅馬尼亞境內已有人類活動，但歷史上有記載的羅馬尼亞淵源最早可以回溯到公元101年—104年時。罗马尼亚的祖先为[[达契亚|达契亚]]人，是[[色雷斯人|色雷斯人]]的一支。

约公元前1世纪，布雷比斯塔建立了第一个中央集权的君主制国—[[达契亚|达契亚]]。公元101年—104年，[[羅馬帝國|羅馬帝國]]的[[图拉真|图拉真]][[大帝|大帝]]出动50个军团最终打敗了达契亚国，將此地納入羅馬帝國的一部分，直至因[[哥特人|哥特人]]入侵而被迫於公元271—275年撤離。在中世纪此地區有[[瓦拉几亚|瓦拉几亚]]、[[摩尔达维亚|摩尔达维亚]]與[[特兰西瓦尼亚|特兰西瓦尼亚]]三個獨立的君主制[[公國|公國]]，其中前兩者是[[奥斯曼帝國|奥斯曼帝國]]裡的自治[[附庸國|附庸國]]，後者則在稍後被併入[[奧匈帝國|奧匈帝國]]範圍內。今羅馬尼亞之名是由[[拉丁語|拉丁語]]romanus（意指“羅馬公民”）演變而來，最早見於16世紀[[意大利人|意大利人]]在此三地的遊記。

1854—1856年的[[克里米亞戰爭|克里米亞戰爭]]後，[[法兰西第二帝国|法國]]鼓吹兩個君主制公國[[瓦拉几亚公国|瓦拉几亚公国]]與[[摩達維亞公國|摩達維亞公國]]合併，阻止[[俄羅斯帝国|俄羅斯帝国]]南下。瓦拉齊亞公國與摩達維亞公國在1859年時合併，合併後称為[[罗马尼亚联合公国|罗马尼亚联合公国]]，附属于[[奥斯曼帝国|奥斯曼帝国]]，並在1877年5月9日宣布独立。

1878年[[南比萨拉比亚|南比萨拉比亚]]地区隶属俄罗斯帝国。1881年[[罗马尼亚联合公国|罗马尼亚联合公国]]改称[[罗马尼亚王国|罗马尼亚王国]]，1918年1月南比萨拉比亚地区宣布独立，称[[摩尔达维亚民主共和国|摩尔达维亚民主共和国]]，同年3月又与罗马尼亚合并。

在[[第一次世界大戰|第一次世界大戰]]之後奧匈帝國瓦解，使得[[羅馬尼亞王國|羅馬尼亞王國]]的領土進一步擴張到[[瓦拉几亚|瓦拉几亚]]、[[布科維納|布科維納]]與[[比薩拉比亞|比薩拉比亞]]等地區。1940年時，[[蘇聯|蘇聯]]佔領了羅馬尼亞的一部分領土，該地區成為今日的摩爾多瓦與[[烏克蘭|烏克蘭]]的[[加利西亚_(中欧)|加利西亚]]一小部分。[[第二次世界大战|第二次世界大战]]时，[[扬·安东内斯库|扬·安东内斯库]]政权加入德、意、日[[三国同盟|三国同盟]]，羅馬尼亞於1941年加入[[蘇德战争|蘇德战争]]，收復1940年被苏联拿走的领土並於[[敖德薩圍城战_(1941年)|敖德薩圍城戰]]打败苏軍並佔領该市，羅馬尼亞亦在[[克里米亞战役_(1941年–1942年)|克里米亞战役]]協助德軍，但是在[[斯大林格勒战役|斯大林格勒战役]]中，羅馬尼亞軍傷亡慘重。1944年德國敗退，[[苏联红军|苏联红军]]發動攻勢攻入罗马尼亚，並攻占[[布加勒斯特|布加勒斯特]]。同年8月23日，[[罗马尼亚共产党|罗马尼亚共产党]]领导发动反法西斯武装起义，推翻了安东內斯库政权，罗马尼亚转而加入了同盟國陣營。1945年3月6日成立联合政府。羅馬尼亞在蘇聯的強力影響之下，建立起蘇聯式的[[社會主義|社會主義]]政權。1947年12月30日[[羅馬尼亞社會主義共和國|羅馬尼亞人民共和國]]成立，1948年2月，罗马尼亚共产党与[[罗马尼亚社会民主党|社会民主党]]合并，成立[[羅馬尼亞共產黨|罗马尼亚工人党]]，成为罗馬尼亞唯一执政党。1965年通過新憲法，將國名更改為[[羅馬尼亞社會主義共和國|羅馬尼亞社會主義共和國]]，党的名字也改为[[罗马尼亚共产党|罗马尼亚共产党]]，[[尼古拉·齐奥塞斯库|尼古拉·齐奥塞斯库]]当选[[总书记|总书记]]，成为罗马尼亚的领袖，后来更成为[[独裁者|独裁者]]。齐奥塞斯库当政时期，罗马尼亚在政治和外交上刻意和苏联保持距离，但其自主权特别是经济上的自主权很有限。

1989年12月22日爆发了[[1989年羅馬尼亞革命|1989年羅馬尼亞革命]]，[[齐奥塞斯库|齐奥塞斯库]]政权被推翻，結束了其長達24年的[[獨裁|獨裁]]統治，然后[[齐奥塞斯库|齐奥塞斯库]]与妻子於1989年12月25日被槍決。隨後罗马尼亚救国阵线委员会接管国家一切权力，将国名改为罗马尼亚並推行[[民主化|民主化]]政策。羅馬尼亞新的民選政府逐漸修改路線，成為[[社會民主主義|社會民主主義]]國家。2004年3月29日，罗马尼亚加入[[北约|北约]]。<ref>李学军，严锋，[http://news.xinhuanet.com/world/2004-03/30/content_1390530.htm 东欧7国29日正式成为北约新成员] {{webarchive|url=https://web.archive.org/web/20080315114605/http://news.xinhuanet.com/world/2004-03/30/content_1390530.htm |date=2008-03-15 }}，新华网</ref>

2004年11月28日，罗马尼亚举行总统选举，[[阿德里安·讷斯塔塞|阿德里安·讷斯塔塞]]和[[特拉扬·伯塞斯库|特拉扬·伯塞斯库]]进入第二轮。12月12日，总统选举第二轮投票举行<ref>周荣子，[http://news.xinhuanet.com/world/2004-12/12/content_2324491.htm 罗马尼亚举行总统选举第二轮投票]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}，新华网</ref>，伯塞斯库当选。<ref>周荣子，[http://news.xinhuanet.com/world/2004-12/14/content_2330440.htm 布加勒斯特市长伯塞斯库当选罗马尼亚新总统] {{webarchive|url=https://web.archive.org/web/20050309115735/http://news.xinhuanet.com/world/2004-12/14/content_2330440.htm |date=2005-03-09 }}，新华网</ref>

2007年1月1日罗马尼亚正式加入[[欧盟|欧盟]]。<ref>林惠芬，陈进，[http://news.xinhuanet.com/world/2007-01/01/content_5556537.htm 罗马尼亚举国欢庆加入欧盟] {{webarchive|url=https://web.archive.org/web/20160215233045/http://news.xinhuanet.com/world/2007-01/01/content_5556537.htm |date=2016-02-15 }}，新华网</ref>2007年2月28日，在野社会民主党以伯塞斯库有违宪行为为由正式提出弹劾总统案，4月19日，罗议会以绝对多数通过了弹劾总统议案。弹劾案次日获得宪法法院确认后，总统[[特拉扬·伯塞斯库|伯塞斯库]]随即被停职一个月，参议院议长沃克罗尤出任临时总统。4月20日，宪法法院对[[尼古拉·沃克罗尤|尼古拉·沃克罗尤]]出任临时总统的资格予以确认，接替因议会通过弹劾议案而停职的[[特拉扬·伯塞斯库|伯塞斯库]]。<ref>林惠芬，[http://news.xinhuanet.com/world/2007-04/21/content_6006103.htm 罗马尼亚参议长沃克罗尤出任临时总统]，新华网</ref>5月21日，罗马尼亚就议会弹劾总统议案举行全民公决，这次全民公决的投票率为44.02%，其中赞成议会弹劾总统议案的选民为24.9%，反对者为74.32%。弹劾案未获通过，伯塞斯库复职。<ref>林惠芬，[http://news.xinhuanet.com/world/2007-05/21/content_6127748.htm 综述：罗全民公决否决议会弹劾总统案] {{webarchive|url=https://web.archive.org/web/20101020125348/http://news.xinhuanet.com/world/2007-05/21/content_6127748.htm |date=2010-10-20 }}，新华网</ref>

2009年12月9日，羅馬尼亞中央選舉辦公室定於公佈總統選舉第二輪投票最終結果，預計得到[[羅馬尼亞民主自由黨|羅馬尼亞民主自由黨]]支援的現總統伯塞斯庫以50.33%的微弱多數戰勝社會民主黨主席傑瓦訥，獲得連任。伯塞斯庫和傑瓦訥的得票率太接近，而社民黨又聲稱本次大選存在舞弊行為，這導致選舉結果被蒙上了“缺乏合法性”的陰影。

2012年7月6日，在野社会自由联盟在罗马尼亚议会提出总统弹劾议案，理由是总统伯塞斯库越权行使总理职权。议会通过了弹劾议案，按照宪法规定，伯塞斯库被停职，一个月内，全民公决将决定总统的职位去留。7月29日，全民公投由于投票率未过半，选举委员会宣布此次弹劾失败。

== 政治 ==
{{main|羅馬尼亞政治}}
罗马尼亚是一个[[民主|民主]]国家，议会实行[[两院制|两院制]]，上院为[[罗马尼亚参议院|参议院]]（罗：Senat），有140名议员，下院为[[罗马尼亚众议院|众议院]]（罗：Camera Deputaților），有346名议员。参众两院的议员在四年一次的选举中产生。

[[罗马尼亚总统|总统]]每五年选一次，[[罗马尼亚总理|总理]]由总统任命，内阁人选再由总理任命，由议会通过。

== 经济 ==
{{main|罗马尼亚经济}}
在第二次世界大戰之前，羅馬尼亞是歐洲第二大的食品生產商。在1937年透過高達720萬噸的石油產量，其經濟成長全歐第二，世界第七。<ref>[http://www.aneir-cpce.ro/chapter1/his1.htm] {{Wayback|url=http://www.aneir-cpce.ro/chapter1/his1.htm |date=20160303192153 }}，羅馬尼亞經濟。</ref><span data-segmentid="5" class="cx-segment">罗马尼亚探明'''[[天然气|天然气]]'''储量为7260亿立方米（25.94万亿立方英尺），在已知探明天然气储量的国家中排名第30位</span> 。<ref>{{Cite web|url=http://ec.europa.eu/energy/publications/doc/2008_moe_maquette.pdf|title=Europe's energy position past & future|accessdate=|author=|date=|publisher=ec.europa.eu}}</ref>

[[第二次世界大戰|第二次世界大戰]]結束後，羅馬尼亞成為[[東方集團|東方集團]]的一員，並實行[[社會主義|社會主義]]式的[[計劃經濟|計劃經濟]]。身為東方集團的一員，羅馬尼亞限制私有經濟的發展，在工業上不允許私人企業存在，商業和服務業均為國家或集體所有制。羅馬尼亞革命之後，採取一系列自由化政策實施改革，任何個人，包括國營單位或合作單位的員工、農民及退休者，均可申請經營各種工商業和服務業活動。

2007年1月羅馬尼亞進入[[歐盟|歐盟]]並促使其國際貿易自由化，其中羅馬尼亞和[[保加利亞|保加利亞]]的加入也使歐盟船隻得以通過[[黑海|黑海]]。

== 军事 ==
{{main|羅馬尼亞武装部队}}
[[File:Expomil_2005_01_TR-85M1_03--A.jpg|缩略图]]]]
[[File:Romanian_MIM-23_HAWK.jpg|thumb]]]]

1994年10月25日罗马尼亚军队建军，是目前罗马尼亚的主要武装力量。最高国防委员会是罗马尼亚最高军事决策机构，罗马尼亚国防部是罗马尼亚军队的领导机构。1994年3月起国防部长改由文职人员担任，现任国防部长为[[米尔恰·杜沙|米尔恰·杜沙]]。2003年10月，罗马尼亚修改宪法，取消[[义务兵役制|义务兵役制]]。罗军不断推进军队职业化进程，计划逐步实行军旅制。2010年罗马尼亚国防部预算共有18亿欧元军费，约占国内生产总值的1.31%。罗马尼亚目前现役军人数为9万人。

== 外交 ==
{{main|羅馬尼亞建交列表}}
截至2015年，罗马尼亚已同188个国家建立了外交关系，它重点发展同[[美国|美国]]、[[欧盟|欧盟]]和[[北约|北约]]的关系，同时注重与鄰近國家及[[亚太地区|亞太]]国家的关系，並积极发展和[[中华人民共和国|中华人民共和国]]的关系。罗马尼亚于於2004年3月29日加入[[北约|北约]]，並在2007年1月1日加入[[歐盟|歐盟]]。

== 地理 ==
[[File:Ro-map.png|thumb]]
[[File:Romania-relief.png|thumb]]]]
{{main|羅馬尼亞地理}}
罗马尼亚土地面積238,391平方公里，是[[東南歐|東南歐]]面積最大的國家，並在[[歐洲|歐洲]]排名第十二<ref name="statistical">{{cite web|url=http://www.insse.ro/cms/files/pdf/ro/cap1.pdf|title=Geography, Meteorology and Environment|year=2004|publisher=Romanian Statistical Yearbook|language=ro |accessdate=2009-09-07}}</ref>。羅馬尼亞和[[塞爾維亞|塞爾維亞]]、[[保加利亞|保加利亞]]之間主要是以[[多瑙河|多瑙河]]為界。而與[[摩爾多瓦|摩爾多瓦]]之間則是以多瑙河的另一條支流[[普魯特河|普魯特河]]為界。多瑙河與普魯特河在羅馬尼亞匯合後，在羅馬尼亞境內注入[[黑海|黑海]]，而入海口處所形成的[[多瑙河三角洲|多瑙河三角洲]]是歐洲最大的[[三角州|三角州]]，並因為其豐富的生態變化，而被[[聯合國教科文組織|聯合國教科文組織]]列入[[世界遺產|世界遺產]]名單內，並在此建立[[多瑙生態圈保護區|多瑙生態圈保護區]]（Danube Biosphere Reserve）。

===地形===
[[喀尔巴阡山脉|喀尔巴阡山脉]]以半环行盘踞中部，山脉以西为[[特兰西瓦尼亚高原|特兰西瓦尼亚高原]]，以东为[[摩尔多瓦高原|摩尔多瓦高原]]，以南为[[多瑙河平原|多瑙河平原]]和[[多布罗加丘陵|多布罗加丘陵]]。
===气候===
过渡性[[温带大陆性湿润气候|温带大陆性湿润气候]]，1月平均气温-5－1℃，7月平均气温20－23℃，年降水量400－1000毫米，东南部较干燥。

== 人口 ==
{{main|羅馬尼亞人口}}
根據2002年時的普查，羅馬尼亞擁有21,680,974的人口。在該國人口中，主要組成的[[羅馬尼亞人|羅馬尼亞人]]佔了全國人口的89.5%，而境內最大的少數族裔則分別為[[匈牙利人|匈牙利人]]（6.6%）與[[羅姆人|羅姆人]]（2.5%）。

== 文化 ==

[[File:Castle_in_Radnót.jpg|thumb]]
{{main|羅馬尼亞文化}}

===语言===
羅馬尼亞的[[官方語言|官方語言]]是[[羅馬尼亞語|羅馬尼亞語]]，淵源自[[拉丁語|拉丁語]]，語言結構与附近東歐國家所使用的[[斯拉夫語族|斯拉夫語族]]語言十分不同。羅馬尼亞文是一種與[[法文|法文]]、[[意大利语|意大利语]]、[[西班牙文|西班牙文]]與[[葡萄牙文|葡萄牙文]]較為接近的[[羅曼語族|羅曼語族]]語言。在羅馬尼亞使用羅馬尼亞文作為第一[[母語|母語]]的人口佔了91%，其次則分別是[[匈牙利文|匈牙利文]]與[[羅姆文|羅姆文]]。在[[二战|二战]]以前，[[德语|德语]]是罗马尼亚境内的少数民族[[德意志人|德意志人]]的母语，因此德语也曾是罗马尼亚使用人数最多的[[少数民族语言|少数民族语言]]之一，至今在[[特兰西瓦尼亚|特兰西瓦尼亚]]地区仍有较大影响。至于[[外语|外语]]方面，法语一直是羅馬尼亞境內最多人會使用的外語，該國也是[[法語圈國際組織|法語圈國際組織]]（la francophonie）的成员國之一，但近年來其比例已被[[英语|英语]]後來居上，後者是大部分羅馬尼亞年輕族群第一優先學習與使用的外語。根据Eurobarometer report（2006）的报告，罗马尼亚有29%的人口、即620万人口使用英语。

===宗教===
罗马尼亚是[[世俗国家|世俗国家]]，没有[[国教|国教]]。多数人信奉[[羅馬尼亞正教會|罗马尼亚东正教]]，少数人信奉[[罗马天主教|罗马天主教]]和[[新教|新教]]。羅馬尼亞有一部分[[韃靼人|韃靼人]]，是[[諾蓋人|諾蓋人]]與[[克里米亞韃靼人|克里米亞韃靼人]]（因[[那海|那海]]的封地在多瑙河下游）。

===体育===
羅馬尼亞的女子組體操曾經相當優秀並且亮眼，具備堅強的實力，擁有出眾的競技水平，與美國、俄羅斯、中國並駕齊驅，是目前全世界獲得過最多世界冠軍的國家，有23個奧運會冠軍、47個世界冠軍，總共70個世界冠軍。

===其他===
羅馬尼亞在歐洲及世界藝術文化方面，舉凡：繪畫、音樂、體操、服飾、電影，有顯著且突出耀眼的影響與表現。

著名的影視娛樂題材──[[吸血鬼|吸血鬼]]，即是以德古拉作為範本所創作的。
