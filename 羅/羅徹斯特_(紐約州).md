'''罗切斯特'''（{{lang-en|'''Rochester'''}}），位於[[美國|美國]][[紐約上州|紐約上州]]西部，[[安大略湖|安大略湖]]的南岸，屬於[[門羅縣_(紐約州)|門羅郡]]，也是郡政府所在。2004年市民人口為212,481人，是[[紐約州|紐約州]]第三大城市。以之為中心的[[都會區|都會區]]2000年人口為1,037,831。

== 自然地理 ==
羅徹斯特位在[[五大湖區|五大湖區]]，珍尼西河（Genesee River）由南向北流經市區匯入安大略湖，其中有多處因冰河地形造成的落差，有豐富的水力資源，提供羅徹斯特在早期工業發展的動力來源。[[伊利運河|伊利運河]]在改道前也流經市區中央，可溝通五大湖區與連結[[哈德遜河|哈德遜河]]到達[[紐約市|紐約市]]，更有利於貨品的輸送。羅徹斯特的東南方即為[[五指湖|五指湖]]（Finger Lakes）區域，有多個因為冰河退卻蝕刻形成的南北狹長型湖泊，鄰近區域以生產葡萄酒著名。

羅徹斯特位於溫帶，属于[[温带大陆性湿润气候|温带大陆性湿润气候]]，四季分明，大陆性强。春季百花盛開，有「花城」（Flower City）的美譽，夏季涼爽，六七月的平均最高溫為攝氏27度；秋季氣候宜人且落葉葉色繽紛，冬季則寒冷多雪，每年的十一月到三月為雪季，一月二月的最低溫平均為攝氏零下9度。羅徹斯特緯度偏高，又由於位在大湖南岸，每年冬季來自加拿大與極地的冷氣團從湖面帶來大量水氣，受到這種[[大湖效應|大湖效應]]的影響，冬季降雪甚多，累積年降雪量平均約200cm。

冬季寒冷，潮湿，日照少，日最高气温低于{{convert|0|C}}的平均日数为47天，日最低气温低于{{convert|-10|C}}的平均日数为32天，低于{{convert|-15|C}}的有11天；夏季相对潮湿，日最高气温超过{{convert|30|C}}的日数年均只有21天，{{convert|35|C}}以上的气温罕见。<ref name = "NWS Buffalo, NY"/>最冷月（1月）均温{{convert|24.7|F|1|disp=flip}}，极端最低气温{{convert|−22|F|0|disp=flip}}（1934年2月9日）。<ref name = "NWS Buffalo, NY"/>最热月（7月）均温{{convert|70.8|F|1|disp=flip}}，极端最高气温{{convert|102|F|0|disp=flip}}（1936年7月9日、7月10日）。<ref name = "NWS Buffalo, NY"/>无霜期平均为167天（4月30日至10月13日）；可测量降雪平均期为11月7日至4月12日。<ref name = "NWS Buffalo, NY"/>年均降水量约{{convert|40.5|in|sigfig=3|disp=flip}}，年极端最少降水量为{{convert|20.30|in|disp=flip}}（1887年），最多为{{convert|50.06|in|disp=flip}}（1873年）。<ref name = "NWS Buffalo, NY"/>年均降雪量为{{convert|99.5|in|cm|disp=flip}}；1932–33年的降雪量最少，积累降雪量只有{{convert|27.3|in|cm|disp=flip}}，1959–60年的降雪量最多，积累降雪量为{{convert|161.7|in|m|disp=flip}}。<ref name = "NWS Buffalo, NY"/>

== 經濟活動 ==
早年羅徹斯特利用充沛水力推動水車來磨麵粉，素有「麵粉城」（Flour City）的暱稱；水力也用以發展輕工業如製造鈕扣等。但是後來鐵道運輸興起，水運便捷的優勢不再，加上有其他更有效率的能源，羅徹斯特早年因水力而興盛的輕工業也隨之沒落。現今在市區中心的High Fall一帶仍保存有過去的水車與工業生產遺跡，供遊客與民眾追憶當年。

近代羅徹斯特成為光學工業的樞紐。[[喬治伊士曼|喬治伊士曼]]（George Eastman）在此地創立伊士曼[[柯達|柯達]]公司，是傳統攝影與底片的龍頭，全球營運總部即設在市區中的柯達大樓。影印複印技術的創始先驅[[全錄|全錄]]公司也是發軔於羅徹斯特，九十年代後雖將營運總部搬遷，不過羅徹斯特仍是重要的研發中心。羅徹斯特還是光學儀器與隱形眼鏡的跨國企業[[博士倫|博士倫]]的全球總部與創始地所在。由於三大光學相關跨國企業都以羅徹斯特為根據地，羅徹斯特吸引了其他眾多中小型公司從事光學影像等相關的商業活動，因此有「世界影像中心」的美稱。

羅徹斯特也是美國東岸知名超市聯鎖衛格門（Wegmans）的創始地與總部所在。

近來由於光學產業受到電子數位科技的挑戰，羅徹斯特的主要公司都在進行轉型重整與組織改造。自2005年起，柯達公司不再是當地最大的雇主而屈居第二。目前在羅徹斯特與周邊都會區裡最大的聘僱單位是[[羅徹斯特大學|羅徹斯特大學]]與其所轄的思壯紀念醫院系統（Strong Health System）。

== 教育機構 ==
羅徹斯特有數所大學院校，其中最著名的為[[羅徹斯特大學|羅徹斯特大學]]與[[羅徹斯特理工學院|羅徹斯特理工學院]]（Rochester Institute of Technology）。其他的院校包括Colgate Rochester Crozer Divinity School、Saint John Fisher College、Roberts Wesleyan College、Nazareth College和Monroe Community College。

羅徹斯特大學是私立綜合研究型大學，根據US News的綜合排名為全美前四十名，其下所屬的[[伊士曼音樂學院|伊士曼音樂學院]]（Eastman School of Music）為全美最佳的音樂學院，以學理與演奏藝術兼重著稱；賽門商業管理學院（Simon Graduate School of Business Administration）是全美前三十名的商學院；Laboratory for Laser Energetics擁有全球最大功率的雷射；而醫學院亦有甚佳的風評，尤其在臨床醫學的表現；所附設置思壯紀念醫院是紐約西部最大的醫學中心。除此之外，光學，經濟學，政治學，心理學等學科也都是著名的科系強項。羅徹斯特理工學院則以大學本科教育與專精職業訓練聞名，光學攝影與影像處理十分聞名，其工業設計在全世界名列前茅，全球前十五大設計學院。另設置了國家聾啞高等教育中心，是專供美國聾啞殘障人士就學的高等教育機構。

羅徹斯特與附屬衛星城鎮也有數所美國評比最佳的高中。
