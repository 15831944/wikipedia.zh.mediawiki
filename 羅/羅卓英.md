'''羅卓英'''（{{bd|1896年|3月19日|1961年|11月6日|catIdx=L羅}})，原名'''東潘'''，字'''尤青'''，別號'''慈威'''（又作'''慈衛'''），1896年3月出生於廣東大埔縣百侯鄉曲灘村宜洋坪，客家人。幼年過嗣叔父，居住同縣湖寮鄉嶺下村。民國12年（1923）畢業於保定軍校第八期炮科，與[[陳誠|陳誠]]同為砲兵科同期同學，並繼陳誠之後為[[國民革命軍第十八軍|第十八軍]]第2任軍長，與[[周至柔|周至柔]]（步科）則是同期同學，同是[[土木系_(國民革命軍)|土木系]]的重要將領。後升上將銜，第十九集團軍總司令、第九戰區副司令長官、西南遠征軍第一路司令長官等職。

== 早年經歷 ==
[[File:Luo_Zhuoying1.jpg|thumb]]
羅卓英早年參加東征與[[北伐|北伐]]等役，由下級軍官升至[[國軍|國軍]]第十一師師長及第十八軍軍長，並積極參與[[江西|江西]]剿共。1924年羅卓英辭去湖山中學校長職務，在廣州加入國民革命軍，任第一炮兵營上尉連長。

廣東革命政府接連兩次東征，討伐叛變的陳炯明，取得勝利。羅卓英參戰有功，被提爲少校副營長、營長。

== 中年經歷 ==
羅卓英率部先後再次參加北伐戰爭以及1930年的中原大戰，他因戰功累升團長、旅長、參謀長、副師長、師長等職。

[[抗日戰爭|抗日戰爭]]期間參加[[上海|上海]]、[[南京|南京]]、[[武漢|武漢]]諸役。任第十九集團軍總司令、第九戰區副司令長官等職。

1942年4月2日，蔣介石派羅卓英為中國遠征軍一路司令長官<ref name=民 />{{rp|6779}}。4月5日，蔣介石偕羅卓英及史迪威由重慶乘飛機經昆明抵臘戍<ref name=民 />{{rp|6782}}。羅卓英入緬作戰，後轉往[[印度|印度]]，主持蘭伽遠征軍訓練。

1943年3月19日，軍事委員會任命羅卓英為[[第三戰區|第三戰區]]副司令長官<ref name=民 />{{rp|7079}}。4月22日軍去委員會免去其第三戰區副司令長官職務<ref name=民 />{{rp|7106}}。後轉任軍令部次長，兼[[軍事委員會|軍事委員會]]訓練總監。

1945年當選為[[中國國民黨|中國國民黨]]第六屆[[中央執行委員|中央執行委員]]。

八年抗戰期間，羅卓英自淞滬會戰開始即首當其衝，繼而轉戰常熟廣德，拱衛首都南京，指揮保衛武漢，組織反攻南昌，獲取上高大捷，乃至三次長沙會戰，一度緬甸遠征，直至最後勝利，可謂南征北戰，幾至無役不與。

八年抗戰，論作戰次數之頻繁，戰區跨度之大，戰績之輝煌，所有國軍將領中，實難有人能超過羅卓英。

抗戰勝利後，曾任[[廣東省政府|廣東省政府]]主席。1947年6月中旬，[[越法戰爭|越法戰爭]]以來，政局混亂，[[華僑|華僑]]在[[越南民主共和國|越南]]生命財產均無保障，近紛紛向雲南、廣西兩省邊境避難，人數已達4000餘人，雲南省僑務處、社會處已派員前往救濟；6月19日，[[廣東省政府|廣東省政府]]主席羅卓英就越方限令華僑疏散，電請[[行政院|行政院]]派艦保護撤退<ref name=民 >{{cite book |editors=李新總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室編，韓信夫、姜克夫主編 |title=《中華民國史大事記》 |location=[[北京|北京]] |publisher=[[中華書局|中華書局]] |year=2011 }}</ref>{{rp|8373}}。7月19日，國民政府派羅卓英為廣東選舉事務所主任委員；國民政府同時派定其為[[國民大會代表|國民大會代表]]，[[立法院|立法院]][[立法委員|立法委員]]<ref name=民 />{{rp|8385-8386}}。

後在[[國共內戰|國共內戰]]東北戰場接任[[東北行轅|東北行轅]]副主任，協助行轅主任[[陳誠|陳誠]]。後事不可為，1949年秋在[[臺灣|臺灣]]加入陳誠的[[東南軍政長官公署|東南軍政長官公署]]，任公署副長官，戰事從大陸作戰轉為島嶼作戰，[[金門戰役|金門戰役]]初起，羅卓英正巧與[[胡璉|胡璉]]從臺北抵金門，原意以[[胡璉|胡璉]]取代[[湯恩伯|湯恩伯]]與[[李良榮|李良榮]]；惟正值戰事激烈進行，羅卓英以第十八軍老長官身分於旁督戰，但未參與指揮。

1949年其族人多隨之遷往臺灣，鄉居[[屏東縣|屏東縣]]。12月31日，東南軍政長官公署政務委員會委員羅卓英呈請辭職，羅卓英准予免職<ref>{{cite news |title=《總統府公報》第貳肆伍號 |editors=總統府第五局 |date=1949-12-31 }}</ref>。

== 晚年 ==
抵達臺灣後，羅卓英曾任中華民國[[總統府|總統府]]戰略顧問、[[光復大陸設計研究委員會|光復大陸設計研究委員會]]委員兼台南研究區主任、國民大會代表。

1961年在[[臺北|臺北]]因糖尿病併發症病逝，陳誠親撰祭文《哭尤兄》曰：“假令論及昔年報國，與其謂誠稍有貢獻，毋寧悉以屬之於兄。”情雖謙抑，譽非過實。

== 紀念 ==
1947年與1983年為紀念1946年接收該島時廣東省政府主席羅卓英（號慈威），取名「南威島」兼有接收艦隊威鎮南海之意。

==著作==
*《呼江吸海樓詩集》
*《正氣歌注》

羅卓英身經百戰，馬上英姿早已載入史冊。少為人知的，是他文採風流，精擅律絕。戎馬倥傯之際，幾乎無役不詩。抗戰勝利後，他出版有《呼江吸海樓詩集》，收入 183題共593首詩，皆創作於1937年7月至1946年11月，涵蓋整個抗戰時期，時羅卓英在四十二至五十一歲之間。其中多數篇章，如《蘆溝曲》、 《羅店》、《吊寶山殉城姚營官兵》、《上高會戰奏捷四首》、《遠征軍賀捷詩三首》等，經戰地記者轉載傳播，一時膾炙人口，乃至蜚聲國際。
