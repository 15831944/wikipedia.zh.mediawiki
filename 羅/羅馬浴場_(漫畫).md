{{日本作品|羅馬浴場|テルマエ・ロマエ|Thermae Romae}}是日本女性漫畫家[[山崎麻里|山崎麻里]]（{{lang|ja|ヤマザキマリ}}）的漫畫作品。原在《[[Comic_Beam|Comic Beam]]》（[[Enterbrain|Enterbrain]]）上進行不定期連載，自2010年4月開始改為定期連載。此后，公佈了預計将于2012年改編為電影的消息，主角由[[阿部寬|阿部寬]]、[[上戶彩|上戶彩]]飾演。

本作曾获得由書店店員選出的2010年度“[[漫畫大獎|漫畫大獎]]”<ref>{{Cite|url=http://trendy.nikkeibp.co.jp/article/news/20100317/1031258/|title=「マンガ大賞2010」にヤマザキマリの『テルマエ・ロマエ』 - トレンド - 日経トレンディネット|date=2010-03-17}}</ref>、“第14屆[[手塚治虫文化獎|手塚治虫文化獎]]短篇獎”<ref>{{Cite|url=http://www.asahi.com/shimbun/award/tezuka/10c.html|title=手塚治虫文化賞：朝日新聞社インフォメーション}}</ref>，2011年度“《[[這本漫畫真厲害！|這本漫畫真厲害！]]》少年漫畫第2名”，以及2011年度“[[全国书店店员选出的推荐漫画|全国书店店员选出的推荐漫画]]第3名”<ref>{{ja}}{{Cite web|publisher=Nikkei Business Publications, Inc.|url=http://trendy.nikkeibp.co.jp/article/pickup/20110309/1034873/|title=発表！ 今すぐ読みたいマンガBEST10　全国書店員1000人が選んだおすすめコミック2011（发表！现在就想看的漫画BEST10 全国书店店员1000人选出的推荐漫画2011）|accessdate=2012-02-26}}</ref><small>（[[日语|日语]]：{{lang|ja|[[:ja:全国書店員が選んだおすすめコミック|全国書店員が選んだおすすめコミック）]]}}</small>等荣誉。

== 故事簡介 ==
故事背景在[[哈德良|哈德良]]時代，從公元128年到公元138年的[[古羅馬|古羅馬]]。在工作上遇到挫折的浴場設計師'''路西斯'''因為不明的原因穿越到現代的日本，與家中經營溫泉的日本女性'''小達五月'''相遇，路西斯對日本的溫泉文化和現代的衛浴設施大為訝異，便將這樣的設計運用在古羅馬時代。

== 出場人物 ==
===古羅馬===
; {{nihongo|路西斯·慕德斯特|ルシウス・モデストゥス|[[日本配音員|聲]]：[[FROGMAN|FROGMAN]]、[[陳欣|陳欣]]（香港）；演：[[阿部寛|阿部寛]]}}
: 從[[雅典|雅典]]學習到最新建築技術，專門設計浴場的羅馬建築技師。樸實剛健、嚴謹正直，天生的[[工作狂|工作狂]]，因此和妻子離婚，之後更每天投入工作。為了隨時能響應帝國軍的徵招，除了鍛鍊出不下於古希臘雕刻中的健美身材外，在騎術、劍術、槍術和箭法都有不錯的水準。
:在苦惱如何創新出新的建築使其更受歡迎時穿越到現代的日本，因為對羅馬以外的地區都不怎麼熟悉，而誤把日本當成未開化的國家，把日本人叫成「扁臉族」、「奴隸」等等。但是受到日本浴場文化的衝擊而獲得啟發，於是在回到古羅馬後就把這些點子用在新浴場的建造上而日漸成為極有名氣的建築師，最後更受到哈德良皇帝的召見，並在完成其要求的新浴場後而備受其信任。
: 和哈德良皇帝一樣喜歡素雅的建築物。對哈德良皇帝忠心不二。
: 長相蠻受日本女學生歡迎。邂逅小達五月後，兩者發展出唯妙的情愫，最後和選擇穿越時空留在古羅馬的小達五月結婚，育有一子。
; {{nihongo|馬克斯|マルクス・ピエトラス|聲：[[東地宏樹|東地宏樹]]、[[葉振聲|葉振聲]]（香港）}}
: [[羅馬|羅馬]]的石工[[雕刻家|雕刻家]]。與年老的師父同住一起。
: 路西斯的老朋友，和他是能共同去浴場泡澡、時常一起飲酒的好交情。
: 相較於嚴謹的路西斯，個性開朗且十分好色，喜好嫖妓。
: 是路西斯在工作與家庭關係上會進行商談與吐露心聲的對象，並用自己的手藝讓路西斯得自日本的靈感在羅馬呈現出來。
: 在第一部電影版之中是路西斯妻子的外遇對象，在第二部時和解並恢復了交情。
; {{nihongo|[[哈德良|哈德良]]|ハドリアヌス|Hadrian|聲：[[大塚明夫|大塚明夫]]、[[林保全|林保全]]（香港）}}
: 當時的羅馬皇帝。因為[[安提諾烏斯|安提諾烏斯]]的死常常鬱鬱寡歡。
: 軍事與內政都有不錯成績，更是喜愛新建築物的建築家，但因為年過五十，常感覺自己體力大不如前。
: 聽聞了造成轟動的新浴場的建築師路西斯的的事蹟後，而常請他為自己打造新浴室，相當信任且重用路西斯。
; {{nihongo|雷比達|レピドゥス|聲：[[白熊寬嗣|白熊寬嗣]]}}
: 羅馬的地方[[執政官|執政官]]。
; {{nihongo|セウェルス|Sextus Julius Severus}}
: 羅馬軍隊的[[將軍|將軍]]。
; {{nihongo|[[大马士革的阿波罗多洛斯|大马士革的阿波罗多洛斯]]|アポロドロス|Apollodorus of Damascus}}
: 路西斯尊敬的知名建築師。早期受到皇帝[[哈德良|哈德良]]重用，並且有許多偉大實績。但由於建設構想跟不上哈德良的新意，後來被哈德良疏遠。
: 後來因為發明弩炮等攻城武器而再度獲得哈德良的信賴，對路西斯獲得皇帝信賴之事感到高興，認為自己後繼有人。
; {{nihongo|盧修斯·埃里烏斯|ルキウス・ケイオニウス・コンモドゥス|Lucius Aelius Caesar，出生時名Lucius Ceionius Commodus}}
:哈德良皇帝原本選擇的皇位繼承人，打算讓他在奧里略成年之前守護羅馬。但在旁略比亞行省總督任上染上重病，即位前去世，使哈德良大發雷霆。
:為人雖然聰慧，但非常好色。
; {{nihongo|[[馬爾庫斯·奧列里烏斯|馬爾庫斯·奧列里烏斯]]|マルクス・アウレリウス・アントニヌス|Marcus Aurelius}}
:即後來的奧里略皇帝。雖然書中僅僅描寫他的少年時代，卻已經顯露他的才華。

===現代日本===
; {{nihongo|小達五月|小達さつき（おだて・さつき）}}
:路西斯與哈德良共浴時穿越到日本伊藤溫泉後遇見的「扁臉族」女性，28歲，東京大學畢業的古羅馬研究者。
:非常的熱愛古羅馬文化，為研究而學會英語、義大利語、法語和拉丁語，是路西斯穿越到日本後第一個能溝通的對象。
:路西斯和她初次相遇時覺得她很像月神黛安娜，因此一直以黛安娜稱呼她，直到最終話才叫她的名字。
:在發覺自己愛上路西斯後，為了再和已經回到古羅馬時代的路西斯見面而四處奔走。聽了祖父鐵藏偶然間去了古羅馬和哈德良相遇並碰見路西斯的事情後就加入牛津大學對拜亞的古羅馬遺跡調查團隊，並在那發現路西斯留下的建築遺跡。
:之後在遺跡的泉水幫助下來到古羅馬，並在馬爾庫斯的相助下與路西斯重逢。之後兩人結婚，並育有一子。

; {{nihongo|小達鐵藏|小達鉄蔵（おだて・てつぞう）}}
:五月的祖父，也是她唯一的親人。在伊藤溫泉附近經營針灸、整骨，是名可以只看他人的氣色就能看出其健康狀況的名醫。
:交友廣闊，從大企業的會長到拳擊賽前冠軍、政界的高官都有所交往，因此孫女五月在拉贊助時受到他很大的幫忙。
:偶然間回到過去遇見哈德良並為其整治，但知道他已命不久矣的事實，後獲取黃金手環作為報酬，之後在泡溫泉時偶遇路西斯，但還未說一句話就回到現代。
:在孫女五月苦惱著要不要去找路西斯時要她不要擔心自己，而是該努力追求自己的幸福，促成五月和路西斯的重逢。

===剧情梗概===
大约公元130年，强盛繁华的罗马帝国正处在哈德良（市村正亲 饰）统治的时代，时下罗马人民沐浴之风盛行，公共浴池更是受到广大民众的欢迎。随着奢华之风抬头，浴场设计师路西斯·慕德斯特（阿部宽 饰）那古朴简约的设计风格渐渐受到冷遇。某天，他意外通过公共浴场穿越到21世纪的日本，这群被路西斯稱为“扁脸族”的人，其先进创新汤文化令路西斯大为羞愧和赞叹。在此之后，他多次穿越时空，将日本浴室中的创新理念带到古罗马，他由此咸鱼翻身，大受欢迎，甚至得到哈德良的赏识和重用。在此过程中，哈德良王朝落下帷幕，看似华而不实的凯奥尼乌斯（北村一辉 饰）成为新的皇帝。路西斯拒绝为凯奥尼乌斯设计浴室，他的命运来到了新的十字路口……
