'''羅樂林'''（{{lang-en|Law Lok Lam}}；{{bd|1948年|10月20日|catIdx=L羅}}），香港知名甘草演員，曾于1970年代先后加入[[佳藝電視|佳藝電視]]及[[亞洲電視|亞洲電視]]，[[無綫電視|無綫電視]]（TVB）資深藝員（現為基本藝人合約），曾為無綫電視監製[[梁材遠|梁材遠]]的常用演員。

== 背景 ==
羅樂林於[[廣東省|廣東省]][[汕頭市|汕頭市]]出生，在[[澳門|澳門]]與[[廣州|廣州]]成長與就學。他是家中獨子，在廣州讀小學時參與過廣東省體育運動學校的業餘乒乓球訓練{{noteTag|資料出自香港電台第二台廣播節目《守下留情》於2015年8月11日播出的羅樂林訪問。}}。而在澳門讀至中二年級便到香港生活。在學期間，羅樂林因親戚為[[泰國|泰國]]華僑的關係而接觸[[泰拳|泰拳]]運動。亦會學習[[書法|書法]]和彈奏[[小提琴|小提琴]]{{noteTag|資料出自香港電台第二台廣播節目《守下留情》於2015年8月11日播出的羅樂林訪問。}}。他及後隨父親定居父親友人位於[[香港島|香港島]][[北角|北角]]的家中。羅樂林來港發展時在父親的旅行社做旅遊領隊，又在親戚的塑膠廠做跟車工人。於1969年與朋友一同報考[[邵氏兄弟|邵氏兄弟]]南國實驗劇團第9期訓練班。當時與他同期的同學有只有9歲的香港女演員兼女歌手[[吳香倫|吳香倫]]。羅樂林就讀訓練班之時，曾以[[現代舞|現代舞]]舞蹈演員身份參演1970年由[[張徹|張徹]]導演的電影《小煞星》。同年因獲得張徹的賞識而被推薦到[[邵氏電影|邵氏電影]]的製片部工作。他在未轉往電視方面發展前參演了一些文藝片的拍攝如1972年電影《年輕人》「小羅」一角。{{noteTag|關於羅樂林早年生活、就學和報考南國實驗劇團第9期訓練班的資料出自香港電台第二台廣播節目《守下留情》於2015年8月10日播出的羅樂林訪問。}}

由於羅樂林希望演出武俠片，所以在1972年與公司提早解除為期5年的電影合約，並轉投富國影業電影公司。不久富國影業突然倒閉。羅樂林以特約演員身份參演一些粵語片如和電影劇集《少年十五二十時》（1976年無綫電視作品的其中一集）。{{noteTag|關於羅樂林在富國影業工作和任職特約演員的資料出自香港電台第二台廣播節目《守下留情》於2015年8月11日播出的羅樂林訪問。}}至1975年正式於電視界發展。其首部電視劇作品為1975年的《[[射雕英雄傳_(1976年電視劇)|射雕英雄傳]]》。當時他所飾演[[魯有腳|魯有腳]]一角的表現不錯。結果獲電視台的員工引薦給佳藝電視的高層如[[樂易玲|樂易玲]]、[[蕭笙|蕭笙]]、[[李惠民|李惠民]]等人認識，且得到《[[神鵰俠侶|神鵰俠侶]]》一劇的試鏡機會。雖然最終出演劇中的男主角，但羅樂林自覺有鄉音和感到力不從心。因此認為自己演得很差。他早於1976年在[[佳藝電視|佳藝電視]]演出電視劇《[[神鵰俠侶|神鵰俠侶]]》，為香港首位飾演男主角[[楊過|楊過]]的演員<ref>{{cite news | title=羅樂林點評楊過小龍女華仔陳玉蓮最經典 | url=http://www.chinapress.com.my/20160526/%E7%BE%85%E6%A8%82%E6%9E%97%E9%BB%9E%E8%A9%95%E6%A5%8A%E9%81%8E%E5%B0%8F%E9%BE%8D%E5%A5%B3-%E8%8F%AF%E4%BB%94%E9%99%B3%E7%8E%89%E8%93%AE%E6%9C%80%E7%B6%93%E5%85%B8/ | format= | publisher=中國報 | date=2016-5-26 | accessdate=2019-4-10 }}</ref>，[[李通明|李通明]]則飾演[[小龍女|小龍女]]。{{noteTag|關於羅樂林飾演「楊過」的始末的資料出自香港電台第二台廣播節目《守下留情》於2015年8月12日播出的羅樂林訪問。}}自從佳藝電視倒閉後，羅樂林得知無綫電視有足夠的人材，遂轉投演員數目不多、[[亞洲電視|亞洲電視]]的前身麗的電視{{noteTag|關於羅樂林轉投麗的電視原因的資料出自香港電台第二台廣播節目《守下留情》於2015年8月12日播出的羅樂林訪問。}}，演出過《[[天蠶變|天蠶變]]》、《[[大內群英|大內群英]]》、《[[太極張三豐_(電視劇)|太極張三豐]]》、《[[秦始皇|秦始皇]]》等。一直到了1990年，他獲[[馮美基|馮美基]]邀請轉投[[無綫電視|無綫電視]]<ref>{{cite web |url=https://www.mpweekly.com/entertainment/focus/local/20170615-62583 |title=【到底要死幾多次】羅樂林《蘭花劫》被刺殺 原來做過《神鵰俠侶》楊過？ |last= |first= |date=2017-6-15 |website= |publisher=明周 |access-date= 2019-4-10 |quote=}}</ref>{{noteTag|關於羅樂林轉投無綫電視的資料出自香港電台第二台廣播節目《守下留情》於2015年8月13日播出的羅樂林訪問。}}。羅樂林入行多年最為人熟悉的演出是在《[[大時代|大時代]]》中飾演炒股票致精神失常的「股神」葉天，曾把股票心得傳授予主角方展博（[[劉青雲|劉青雲]]），臨死前傳授方一首稀奇古怪的詩，稱為「股票必勝法」<ref>{{cite web |url=https://topick.hket.com/article/588779/%E3%80%8A%E5%A4%A7%E6%99%82%E4%BB%A3%E3%80%8B%E7%B6%93%E5%85%B8%E7%99%BE%E7%A7%91%20%20%20%E8%82%A1%E7%A5%A8%E5%BF%85%E5%8B%9D%E6%B3%95%E6%98%AF... |title=《大時代》經典百科 股票必勝法是... |last= |first= |date=2015-4-21 |website= |publisher=香港經濟日報 |access-date= 2019-4-10 |quote=}}</ref>。在2014年，羅樂林在《[[萬千星輝頒獎典禮2014|萬千星輝頒獎典禮2014]]》榮獲頒發「專業演員大獎」<ref>{{cite web |url=http://event.tvb.com/tvb2015/awardlist.php |title=台慶2014 - 頒獎禮得獎名單回顧 |last= |first= |date=2014 |website= |publisher=無綫電視 |access-date= 2019-4-10 |quote=}}</ref>；藉此肯定及嘉許他在演藝圈中的貢獻及付出。

== 家庭與婚姻 ==
* 羅樂林曾有兩段婚姻。其中第二位前妻為息影演員[[陳寶儀|陳寶儀]]（與[[陳寶珠|陳寶珠]]同為粵劇名伶[[宮粉紅|宮粉紅]]養女）。他於2011年5月31日接受[[查小欣|查小欣]]主持的[[商業電台|商業電台]]節目訪問中承認與妻子陳寶儀已離婚，但仍一起同住。羅樂林有一個與[[陳寶儀|陳寶儀]]生下的女兒{{Link-yue|羅苡之|羅苡之}}；2012年10月4日羅苡之與男演員[[陳志健|陳志健]]結婚。另外，羅樂林與第一任妻子尚育有兩名女兒<ref>{{cite web |url=https://eastweek.my-magazine.me/main/12170 |title=死出我天地 羅樂林 |last= |first= |date=2011-4-21 |website= |publisher=東周刊 |access-date= 2019-4-10 |quote=}}</ref>。
