'''快手'''是中国大陆的一款短视频[[手机应用|手机应用]]，其前身為2011年面世的一款製作、分享GIF圖片的應用「GIF快手」，該應用於2012年轉型為短視頻社區，並於2014年易名快手至今。截至2017年11月，其總使用者已達7億。該應用已獲多輪融資。

快手首席内容官曾光明曾表示，快手大部分用户“来自二线以下城市，最高学历低于高中”。<ref>{{cite web|title=今日头条与快手被查 中国强化互联网平台管制|url=http://www.bbc.com/zhongwen/simp/chinese-news-43748622|website=BBC中文网|accessdate=2018-04-13|date=2018-04-13}}</ref>該應用面世後因其推广假货视频、未成年人未婚生子视频等引發了爭議，並於2018年被中國大陸相關部門勒令整改。

== 歷史 ==
快手的前身，叫“GIF快手”，诞生于2011年3月，最初是一款用来制作、分享GIF图片的手机应用。

2012年，“GIF快手”转型为短视频社区，2014年改名为快手并开始迅速发展，于2015年6月用户突破1亿<ref>{{Cite web|url=https://www.kuaishou.com/about.html|title=关于快手|accessdate=2018-02-27|work=www.kuaishou.com}}</ref>2016年开通直播功能，2017年3月，总用户量达4亿，日活跃用户5000万，2017年11月，总用户超过7亿<ref name=":0">{{Cite web|url=http://www.igeek.com.cn/article-925596-0.html|title=1.5亿月活、估值30亿美元，传快手将于今年底在美上市_前沿科技_爱极客|accessdate=2018-02-27|work=www.igeek.com.cn|archive-url=https://web.archive.org/web/20180228041340/http://www.igeek.com.cn/article-925596-0.html|archive-date=2018-02-28|dead-url=yes}}</ref>，日活跃超1亿<ref name=":0" />。

快手先后获得几轮融资。2012年4月，快手获得晨兴资本A轮融资数百万美元。2015年1月，快手获得了数千万美元B轮融资，由红杉资本和晨兴资本联合投资。2016年3月，快手完成C轮融资，约为2.5亿元，由百度领投，红杉资本、晨星资本等跟投<ref>{{Cite web|url=http://tech.163.com/17/0323/15/CG7LU2EG00098GRC.html|title=被百度和腾讯投资，快手是家什么样的公司？_网易科技|accessdate=2018-04-13|last=网易|work=tech.163.com}}</ref>，当时估值为20亿美金；2017年3月，快手宣布完成新一轮3.5亿美元的融资，由腾讯领投。<ref>{{Cite web|url=http://tech.qq.com/a/20170323/025069.htm|title=快手宣布获得3.5亿美元战略投资 腾讯领投_科技_腾讯网|accessdate=2018-04-13|work=tech.qq.com|language=zh-CN}}</ref>

2018年3月16日，[[中华人民共和国国家新闻出版广电总局|中华人民共和国国家新闻出版广电总局]]下发特急文件《关于进一步规范网络视听节目传播秩序的通知》，规定禁止用户非法抓取、剪拼改编视听节目，并严格管理包括网民上传的类似重编节目，同时要求加强对各类节目接受冠名、赞助的管理<ref>{{cite web|title=广电总局下发特急文件：坚决禁止非法抓取、剪拼改编视听节目|url=https://www.thepaper.cn/newsDetail_forward_2037922|website=北京商报|accessdate=2018-03-22|date=2018-03-22}}</ref>。文件发出后，有部分人士认为快手会因该事件受到不同程度的影响<ref>{{cite web|title=广电总局发文禁止抓取改编视听节目 B站快手抖音怎么办？|url=https://www.jiemian.com/article/2007823.html|website=界面新闻|accessdate=2018-03-31|date=2018-03-22}}</ref>。

===违规开展业务被约谈===
2018年4月4日，[[国家广播电视总局|国家广播电视总局]]就快手播出“有违社会道德”节目等问题，会同属地管理部门约谈了快手主要负责人。经查，快手除上述问题外，长期无视法规训诫，在不具备《信息网络传播视听节目许可证》的情况下运营视听节目服务。依据《互联网视听节目服务管理规定》，广电总局要求快手全面清查库存节目、全面清查库存节目、追究播出违法违规有害视听节目的网站审核人员、主管人员责任，并确保未经审核的节目不得播出<ref>{{Cite web |url=http://media.people.com.cn/n1/2018/0404/c14677-29908261.html|title=国家广播电视总局严肃约谈今日头条、快手负责人|accessdate=2018-04-04 |date=2018-04-04 |publisher=人民网}}</ref>。为配合整改工作，快手已从小米、应用宝、oppo商店、豌豆荚等应用商店下架Android版本，然而在苹果[[App_Store_(iOS)|App Store]]里仍能搜到并下载，但目前显示无法更新。此外，打开iOS版本的快手，“发现”一栏的最上方写着“落实整改 砥砺前行”的标语<ref>{{Cite web |url=http://www.kaixian.tv/gd/2018/0405/549266.html|title=今日头条暂停新注册用户上传视频，快手下线安卓商店|accessdate=2018-04-06|date=2018-04-05 |publisher=蓝鲸TMT}}</ref>。

同日，[[国家网信办|国家网信办]]也依法约谈快手主要负责人。经查，快手“未能落实企业主体责任，出于博取眼球、获取流量目的，疏于账号管理，任由未成年人主播发布低俗不良信息，突破社会道德底线、违背社会主流价值观，污染网络空间，严重影响青少年健康成长”。依据《[[中华人民共和国网络安全法|中华人民共和国网络安全法]]》《互联网信息服务管理办法》《互联网直播服务管理规定》等法律法规，要求快手暂停推荐功能的有关算法，并将违规网络主播纳入跨平台禁播黑名单，禁止其再次注册直播账号。快手相关负责人表示将进行彻底整改，暂停更新有关频道5天，禁止未满18周岁的未成年人注册网络主播，已有账号一律关停，进一步完善审核管理机制，建立未成年人保护体系，用正确的价值观指导算法<ref>{{Cite web |url=http://www.cac.gov.cn/2018-04/06/c_1122644271.htm|title=国家网信办依法约谈“快手”“火山小视频”直播短视频平台 |accessdate=2018-04-07|date=2018-04-06 |publisher=中国网信网 }}</ref>。

4月6日，快手已经在多个招聘网站发布了关于“内容审核编辑”岗位的招聘信息，招聘信息中提到“快手公司面向全国招聘内容审核编辑3000人”。在职位描述中，则提到了“有良好的政治觉悟与素养优先，有较强的政治敏感度与鉴别能力优先，共青团员和党员优先”的任职要求<ref>{{Cite web |url=http://tech.ifeng.com/a/20180408/44941068_0.shtml|title=快手修正“价值观”：急招3000人审核团队，党员优先|accessdate=2018-04-08|date=2018-04-08 |publisher=PingWest  }}</ref>。

2018年8月14日，[[国家廣播電視總局|国家廣播電視總局]]发布消息称，要求北京市文化市場行政執法總隊根據《互聯網視聽節目服務管理規定》，對快手所屬的北京快手網路科技有限公司發布警告和罰款的行政處罰，同时要求该公司在已開展整改工作的基礎上，“不斷升級有效管用的管理辦法，無死角清除存量問題節目，堅決遏制突出問題節目死灰復燃，嚴密管控新增變種問題節目”<ref>{{cite web|title=大陸廣播電視總局清網 今日頭條、快手、抖音又被罰|url=https://www.chinatimes.com/realtimenews/20180814004255-260404|website=旺報 |accessdate=2018-08-14}}</ref>。

==爭議==
=== 假货推广视频问题===
有媒体报道称，快手平台存在大量有关假货推广的短视频。对此，快手平台回应称，将对利用平台制假售假的账号予以封禁处理<ref>{{cite news|title=快手、抖音存大量假货视频 官方回应|url=http://news.mydrivers.com/1/571/571133.htm|accessdate=2018-03-31|work=快科技|date=2018-03-26 }}</ref>。

=== 被曝传播未成年人未婚生子视频 ===
2017年10月31日，有网民发微博称，14岁少女在快手晒怀孕视频，还有自称00后女孩晒孕照，有网友发布微博称，在快手上看到有14岁女孩发的怀孕视频，并配有多张视频截图。消息一出，立刻引起网友热议。<ref>{{Cite web|url=http://finance.sina.com.cn/roll/2017-11-02/doc-ifynmnae1305686.shtml|title=14岁少女在快手晒怀孕视频 还有自称00后女孩晒孕照|accessdate=2018-04-13|work=finance.sina.com.cn}}</ref>对此，快手科技回应称，已经对平台上涉及“未婚妈妈”等类似视频，进行了清理。对于自称年龄低于14岁用户的案例，快手平台已固定证据与有关部门沟通。出于保护未成年人隐私等合法权益考虑，快手并没有将此情况对外披露。<ref>{{Cite web|url=http://www.xilu.com/news/shaonvzhibopingtaifahuaiyunshipin.html|title=少女直播平台发怀孕视频 保护未成年人需多方合力_西陆网|accessdate=2018-04-13|work=www.xilu.com}}</ref>

2018年3月31日，[[中国中央电视台新闻频道|中国中央电视台新闻频道]]曝光快手平台存在大量未成年人未婚生子的视频。这些未成年母亲大多生活在农村或小城镇，早早辍学生子，往往是奉子成婚。自快手平台兴起后，她们也掀起了争当“全网最小妈妈”、“全村最小妈妈”的风潮，且所上传的视频也被系统筛选为推荐视频。事发后的4月1日凌晨，快手在官方微博上做出回应，称平台已经进行了全站清查，查删封禁相关视频和账号，同时关闭推荐功能，在搜索入口进行拦截，增补了审核规则，升级人工智能识别系统，加强核查。目前快手平台已无法找到低龄孕妈妈的内容，甚至“00后宝妈”已无法进行搜索，报道中提及的相关账号也已经被封号<ref>{{cite news|title=央视曝早孕网红 进展：封杀未成年妈妈视频|url=http://www.ahwang.cn/china/20180401/1753981.shtml|accessdate=2018-04-02|work=中新网|date=2018-04-01}}</ref>。4月3日，快手CEO宿华在微信公众号中发表文章《接受批评，重整前行》，对于快手出现的问题进行道歉。文章同时指出快手将“用正确的价值观指导算法”、梳理完善社区运行规则、建立未成年人保护体系，并坚决打击和清除低俗色情内容<ref>{{cite news|title=短视频底线在哪？ 快手CEO道歉：将重整社区运行规则|url=http://ip.people.com.cn/n1/2018/0404/c179663-29907191.html|accessdate=2018-04-04|work=人民网|date=2018-04-04}}</ref>。4月13日，快手新推出“家长控制模式”，而该模式在iOS和Android均已上线<ref>{{cite news|title=快手上线“家长控制模式” 加强未成年人保护|url=http://industry.caijing.com.cn/20180413/4435700.shtml|accessdate=2018-04-14|work=新浪科技|date=2018-04-13}}</ref>。
