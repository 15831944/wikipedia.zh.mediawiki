《'''快把我哥帶走'''》（{{lang-ja|兄に付ける薬はない！}}）是中國漫畫家幽·灵創作的人氣連載漫畫作品，本漫畫描寫了學生兄妹「哥哥·時分」和「妹妹·時秒」為中心的日常生活故事。

目前已超過5億點擊閱讀<ref>{{Cite web |date= 2017-03-06|url= http://animeanime.jp/article/2017/03/06/32837.html|title= 中国発のコメディ「兄に付ける薬はない!」4月放送スタート 中村悠一、雨宮天ら出演|publisher= アニメ!アニメ!|accessdate=2017-03-07}}</ref>。在快看漫畫APP連載，長期位於快看漫畫排行榜前列，每話閱讀將近1000萬。

由五洲電影公司發行有限公司翻拍電影版，於2018年8月17日中國內地上映，哥哥時分由彭昱暢 飾，妹妹時秒由張子楓 飾演，導演-鄭芬芬執導溫馨家庭喜劇。

== 故事簡介 ==
真的有哥哥的人才不會是兄控！”每天坑妹一萬次的哥哥和暴走的妹妹相愛相殺，
還有眾多好基友加戲不斷，日常小片段卻會讓人捧腹大笑不止。這個兄妹番才不是什麼德國骨科......

== 登場人物 ==
* 聲優排列順序為 【中國大陸版／日本電視版】。

; {{nihongo|時分|時分（シーフン）||聲：邊江／[[中村悠一|中村悠一]]}}
: 男主角。時秒的哥哥。普通中學學生，為籃球社的社員。父母離異，兄妹兩人獨立生活，雖然总是做出愚蠢行为，还和妹妹經常吵架（然而都会被会武功的妹妹胖揍一顿），卻是在關鍵時刻靠得住（然而还是靠不住）的好哥哥。
; {{nihongo|時秒|時秒（シーミョ）||聲：[[小連殺|小連殺]]／[[雨宮天|雨宮天]]}}
: 女主角。時分的妹妹。與哥哥同校。經常哥哥吵架，興趣是揍哥哥時分。在食堂与开心见面后开始暗戀開心。
; {{nihongo|甄開心|開心（カイシン）||聲：[[皇貞季|皇貞季]]／[[小野賢章|小野賢章]]}}
: 時分的好友。皮膚白皙、個性傻氣。時秒的暗戀對象。籃球隊。有許多女生都愛慕他。
; {{nihongo|苗妙妙|妙妙（ミョウミョウ）||聲：[[新月冰冰|新月冰冰]]／[[森永千才|森永千才]]}}
: 時秒的好友。可愛，熱心性格。但又过于冲动。
; {{nihongo|萬歲|万歳||聲：阿杰／[[花江夏樹|花江夏樹]]}}
: 萬幸的哥哥。高富帥，原就讀於貴族學校，因受不了太多女生的追求而轉到普通中學。與時分和開心同班。暗戀時秒。
; {{nihongo|萬幸|||}}
: 萬歲的弟弟，12歲，天才跳級生，和哥哥一起轉到普通中學。是萬家集團的第二大股東。

== 電視動畫 ==
於2017年改編為動畫版，動畫由日本動畫公司Imagineer和Fanworks共同製作。

* 中國大陸版於2017年3月28日起每周二在騰訊視頻、[[嗶哩嗶哩|bilibili]]更新。
* 日本電視版於2017年4月7日起每周五20：55在[[東京都會電視台|TOKYO MX]]首播。
