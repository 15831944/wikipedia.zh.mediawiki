'''-{zh-cn:快餐; zh-tw:速食; zh-hk:快餐}-'''，亦稱'''-{zh-cn:速食; zh-tw:快餐; zh-hk:速食;}-'''，是可迅速准备和供應的[[食物|食物]]的總稱，通常是可以徒[[手|手]]拿取的食物，无需使用[[餐具|餐具]]進食，大部分具有包裝，可以外带或[[外賣|外賣]]。

由于可大量烹煮，成本低、製作時間短等因素，快餐从20世纪初期西方陸續進入工業化後开始流行，隨處可見的快餐店成为西方饮食文化的象征，然而在1950年以候冷凍櫃和[[微波爐|微波爐]]大量出現後，東方也出現了乾燥形式的包裝技術，使得速食開始擴散進入家庭，大多數食物都可以通過高效率加工手段速食化，只要簡單加熱就可以享用，很多人已經每日從超市購入這種產品食用，在21世紀後逐漸變為冷凍包裝即食品的總稱（現代快餐）。但本文主要以強調美式速食餐廳制度下生產的快食品為主（傳統快餐）。

== 歷史、起源與發展 ==
快餐在某種意義上被視為現代文化的一種標記，不同文化之間，快餐的內容也大同小异。在[[古羅馬|古羅馬]]城市中有販賣麵包與橄欖油的小攤子，而在[[東亞|東亞]]，比較容易見到得則是麵攤、饅頭店等。在[[中東|中東]]則可能是賣扁平麵包和[[沙拉|沙拉]]、[[三明治|三明治]]的攤子。

現代快餐的起源通常與[[汉堡包|漢堡]]畫上關聯，因為最早的快餐店是以漢堡作為主要產品。最早開設的快餐店是位于美国的[[白城堡|白城堡]]，在1921年就已出現，當時一個漢堡的價錢是五美分。後來白城堡在漢堡肉餅上弄了五個小洞，這樣可以增加受熱表面積，使肉熟得更快，所需的烹調時間也相對縮短。白城堡是快餐業的始祖，並且也衍生了眾多的競爭業者。

一般快餐大多以[[美國飲食|美式]]為主流，但[[中国菜|中式]]、[[港式快餐|港式]]、[[日本料理|日式]]、[[臺灣料理|臺式]]等[[東方世界|東方]]式快餐亦有一定發展。兩者的共同点是採用了工業化生产，利用流水線加工、預製冷凍餐點、烹煮配送現成食材、一次性餐具，佐料包等。同時將餐點菜色分開供客人自行選配。快餐的特点是能在櫃台直接帶走或者简单處理就可以食用，甚至通过[[微波炉|微波炉]]加熱。

== 東方式快餐與西方速食 ==
[[File:高雄的快餐.jpg|thumb]]
和西方快餐不同，中式和日式快餐有湯汁，通常使用[[筷子|筷子]]、湯匙進食，但現在也常見包裝好的飯糰可以直接吃。傳統上，東方式餐饮的制作是复杂的，待大廚準備食材烹調，顧客往往要在位子坐上许久，才會上菜。但快餐则不需要等待食物製作时间，具有效率。

在亚洲國家，隨著20世紀工商社會的成型，逐漸發展出現代快餐的飲食模式，如[[中式小吃|中式小吃]]、[[台灣|台灣]]的[[滷肉飯|滷肉飯]]、[[港式快餐|港式快餐]]、[[日本|日本]]的[[丼物|丼物]]、韓國外送服務等。90年代后，部份东方快餐业者致力于研究改良菜式、配料和烹調方法，務求使快餐食物和飲料更健康及多樣化，與[[美國|美國]]引入的美式快餐區分開來。

== 快餐業者實例 ==
快餐的涵蓋範圍遍布全球，尤其以美式快餐為主。

目前全世界最大的快餐連鎖店是麥當勞。其創始人[[雷·克罗克|雷·克罗克]]于1954年在美国创立麥當勞餐廳，以汉堡为主。與麥當勞齊名的[[肯德基|肯德基]]炸雞店由[[哈蘭德·桑德斯|哈蘭德·桑德斯]]上校於1939年在肯塔基州路易斯維爾創建，以[[炸雞|炸雞]]聞名世界，現被[[百勝|百勝餐飲集團]]收购。

至于其他美式快餐店方面，[[漢堡王|漢堡王]]於1954年於美國邁阿密-戴德縣開設首間分店，其汉堡以炭火[[燒烤|燒烤]][[汉堡排|漢堡扒]]和新鮮蔬菜配製、成為漢堡王的招牌食品。[[溫蒂漢堡|溫蒂漢堡]]則創立於1972年，其重要創舉是發明了顧客免下車的"[[得來速|得來速]]"点餐服務。Subway以新鲜、健康、即点即做，低[[卡路里|热量]]的[[潜艇三明治|潜艇三明治]]为主要餐点。

亞洲快餐方面，[[香港|香港]]的三大快餐集團[[大家樂|大家樂]]、[[大快活|大快活]]和[[美心MX|美心MX]]均是港式快餐的代表。[[台灣|台灣]]以遍布各地的[[美食街|美食街]]、[[夜市|夜市]]的店家販售的即食食品為主，如[[鬍鬚張魯肉飯|鬍鬚張魯肉飯]]、[[頂呱呱|頂呱呱]]以及[[丹丹漢堡|丹丹漢堡]]等連鎖業者。日本的[[摩斯漢堡|摩斯漢堡]]以包括「[[米漢堡|米漢堡]]」在內的[[和風|和風]]美式速食著稱。

== 缺点==
快餐雖然有低成本、速度快的優點，但以大量[[食用油|食用油]]、[[調味料|調味料]]和[[食品添加物|食品添加物]]來烹饪的方式，使得快餐被批評危害人體健康，尤其是加速[[心血管疾病|心血管疾病]]與其他[[內臟|內臟]]疾病。
==商業==
*快餐文化泛指任何在商業操作上尋求短期獲利快速回本的運轉形式，因為這類的商業投資帶有投機的心態尋求不尋常的短期獲利而忽略在外在條件上如商譽、名聲、客戶關係、長期經營的條件，往往通常帶有負面的意味。
