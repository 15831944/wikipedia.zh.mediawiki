{{Infobox software|name=Fast Duplicate File Finder|screenshot=|developer=MindGems Inc.|status=推出|operating system=[[Microsoft_Windows|Microsoft Windows]]|platform=[[IA-32|Intel x86 32-bit]] and [[x64|x64]]|language=英文|genre=[[文件管理器|文件管理器]]|license=[[免費軟件|免費軟件]]|website={{URL|http://www.mindgems.com/}}}}'''快速重复文件查找器''' (英文:Fast Duplicate File Finder) 是一个由[[Microsoft_Windows|Microsoft Windows]]旗下MindGems开发的工具，有[[免費軟體|免费]]版和商用版。用于扫描计算机中重复的文件，并显示清单，让用户删除不需要的重复副本，用来释放[[硬盘驱动器|硬盘]]的空间。 兼容所有[[Microsoft_Windows|Microsoft Windows]]的版本，包括最新的 [[Windows_8|Windows8]], [[Windows_7|Windows7]] 和所有[[Windows_Server|Windows Server]]操作系统及其相应的32和64位元版本。

产品收到积极的评价。[[PC_World|PCWorld.com]]于2011年3月2日指软件速度很快，<ref>{{Cite web|url=http://www.pcworld.com/article/231456/fast_duplicate_file_finder.html|title=Fast Duplicate File Finder|accessdate=14 February 2013|date=Mar 2, 2011|work=PCWorld Review}}</ref> Overclocker.com指软件有质量<ref>{{Cite web|url=http://www.overclockers.com/fast-duplicate-file-finder-freeware-utility/|title=Fast Duplicate File Finder - Freeware Utility|accessdate=14 February 2013|date=June 18, 2009|work=Overclockers Review}}</ref> ，而[[CNET|CNET]]觀察到免費版本的速度是有限的，比专业版更慢。<ref>{{Cite web|url=http://download.cnet.com/Fast-Duplicate-File-Finder/3000-2248_4-75183786.html|title=Fast Duplicate File Finder - CNET Download.com|accessdate=14 February 2013|date=January 18, 2011|work=CNET Review}}</ref>
