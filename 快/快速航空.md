'''快速航空'''（Xpress Air）是[[印度尼西亚|印度尼西亚]]的一家[[航空公司|航空公司]]，自身定位为主要提供飞往东部地区航线的国内航空公司，2014年开通了飞往[[马来西亚|马来西亚]]的国际航班<ref>{{cite web |url=http://www.thejakartapost.com/news/2014/06/11/express-air-link-pontianak-and-johor-bahru-weekend.html |title=Express Air to link Pontianak and Johor Bahru this weekend |date=2014-06-11}}</ref>。随着两架[[波音737|波音737]]客机投入使用，快速航空成为第一家开通雅加达直航飞往[[望加锡|望加锡]]、[[特尔纳特_(印尼)|特尔纳特]]、[[索龙_(印尼)|索龙]]、[[马诺夸里|马诺夸里]]、[[查亚普拉|查亚普拉]]等24座东部城市的私人航空公司。望加锡的[[苏丹哈桑丁国际机场|苏丹哈桑丁国际机场]]是从[[爪哇岛|爪哇岛]]飞往印尼东部最主要的航空枢纽，索龙则是[[西巴布亚|西巴布亚]]地区的第二大枢纽，有航线连接西巴布亚周围各地。

快速航空得到了{{link-nl|印尼鸽子航空|Aero Nusantara Indonesia|印尼鸽子航空股份有限公司}}的支持（值得信赖的飞机维修商），确保其乘客的安全和飞行质量。

快速航空被{{le|民航总局 (印度尼西亚)|Directorate General of Civil Aviation (Indonesia)|印尼民航总局}}列为第一类安全质量航空公司<ref>{{cite web|url=http://hubud.dephub.go.id/?en+news+detail+1464+8|title=Direktorat Jenderal Perhubungan Udara|last=|first=|date=|website=hubud.dephub.go.id|publisher=Kementerian Perhubungan|access-date=|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20120222144750/http://hubud.dephub.go.id/?en+news+detail+1464+8|archivedate=2012-02-22}}</ref>。

==历史==
快速航空从2003年6月23日起开始经营[[查亚普拉|查亚普拉]]到[[雅加达|雅加达]]之间的航线，从那时起，该航空公司已经发展成为印度尼西亚东部的主要航空公司之一。快速航空目前正在扩建机队，以服务巴布亚、苏拉威西岛、马鲁古和努沙登加拉地区的更多目的地。快速航空亦考虑发展通往西部地区的航线，连接[[日惹|日惹]]、[[泗水_(印尼)|泗水]]、[[坤甸|坤甸]]等地。

2012年，快速航空开始使用新的名称（将Express Air变为Xpress Air，用以表现其为更现代化及待客友善的航空公司），公司口号也变为“Terbanglah Indonesia”（翱翔吧，印度尼西亚）。

==事故==
* 2008年11月6日，一架仅在快速航空服役了六周的[[多尼尔328|多尼尔328]]在[[法克法克|法克法克]][[法克法克托利亚机场|托利亚机场]]重着陆，无人员死亡。
* 2009年8月24日，一架[[多尼尔328|多尼尔328]]在[[塔纳默拉机场|塔纳默拉机场]]滑出跑道冲入土堆，导致右侧发动机和螺旋桨发生实质性损害<ref>{{cite web |url=http://aviation-safety.net/database/record.php?id=20090614-0 |title=Accident description |last= |first= |date= |website=aviation-safety.net |publisher=Aviation Safety Network |access-date=2017-11-06 |quote=}}</ref>。
* 2013年5月13日，一架[[波音737#737-200|波音737-200]]的引擎遭遇技术问题，后来安全降落在[[查亚普拉|查亚普拉]][[仙谷国际机场|机场]]。
