《'''快進的日曆'''》（{{jpn|j=早送りカレンダー|hg=はやおくりカレンダー|rm=Hayaokuri karendā|f=y}}）是日本女子[[日本偶像|偶像]]組合[[HKT48|HKT48]]的第11張[[單曲|單曲]]，於2018年5月2日由[[環球音樂_(日本)|EMI RECORDS/UNIVERSAL MUSIC]]發行。同名[[A面與B面|標題曲]]的[[AKB48組合構成#中心位置|Center]]（中心成員）由[[矢吹奈子|矢吹奈子]]與[[田中美久_(HKT48)|田中美久]]擔任。

== 概要==
單曲分為Type-A、Type-B、Type-C和劇場版，《快進的日曆》和《不想當成是季節的錯》收錄於所有版本，《快進的日曆》的音樂錄影帶則收錄於劇場版以外的所有版本。劇場版收錄了《假想戀愛》；《Just a moment》及其音樂錄影帶、影片「HKT48的#速遊 Vol.1」收錄於Type-A；《想見你而變得厭惡》及其音樂錄影帶、影片「HKT48的#速遊 Vol.2」收錄於Type-B；《我的想念終有一日會化作彩虹為止》及其音樂錄影帶、影片「HKT48的#速遊 Vol.3」收錄於Type-C。劇場版以外所有版本均有全國[[握手會|握手會]]參加券和[[生寫真|生寫真]]，劇場版則是個別握手會的參加券<ref>{{cite web |url=http://www.hkt48.jp/discography/single/193 |language=ja |script-title=ja:早送りカレンダー |publisher=[[HKT48|HKT48]] |accessdate=2018-05-10 |deadurl=no |archiveurl=https://web.archive.org/web/20180411111443/http://www.hkt48.jp/discography/single/193 |archivedate=2018-04-11 }}</ref>。

2018年3月31日，HKT48在[[埼玉縣|埼玉縣]][[埼玉市|埼玉市]][[埼玉超級競技場|埼玉超級競技場]]舉行的演唱會「HKT48春季巡演2018～這就是博多的做法！～」（{{lang|ja|HKT48春のアリーナツアー2018～これが博多のやり方だ！～}}）中宣佈於同年5月2日推出單曲，並且首次由[[矢吹奈子|矢吹奈子]]和[[田中美久_(HKT48)|田中美久]]擔任雙中心成員，[[運上弘菜|運上弘菜]]則首次成為選拔成員<ref>{{cite web |url=https://thetv.jp/news/detail/142300/ |language=ja |script-title=ja:HKT48新曲で“なこみく”のWセンターが決定！ ドラフト3期研究生のお披露目も |publisher={{Link-ja|THE TELEVISION|ザテレビジョン}} |date=2018-04-01 |accessdate=2018-05-10 |deadurl=no |archiveurl=https://web.archive.org/web/20180403173934/https://thetv.jp/news/detail/142300/ |archivedate=2018-04-03 }}</ref>。4月10日，攝於[[福岡縣|福岡縣]][[福岡市|福岡市]]的單曲音樂錄影帶在[[Youtube|Youtube]]上公開，這是HKT48首輯完完全全只在根據地福岡完成拍攝的音樂錄影帶<ref>{{cite web |url=https://natalie.mu/music/news/277373 |language=ja |script-title=ja:HKT48、なこみくWセンター「早送りカレンダー」MVで福岡巡って“つまみんダンス” |publisher=[[natalie_(網站)|natalie]] |date=2018-04-10 |accessdate=2018-05-10 |deadurl=no |archiveurl=https://web.archive.org/web/20180411111453/https://natalie.mu/music/news/277373 |archivedate=2018-04-11 }}</ref>。4月30日，HKT48在[[沖繩縣|沖繩縣]][[宜野灣市|宜野灣市]]{{Link-ja|沖繩會議中心|沖縄コンベンションセンター}}舉行的巡迴演唱會中首次演唱單曲<ref>{{cite web |url=https://www.oricon.co.jp/news/2110565/full/ |language=ja |script-title=ja:HKT48、沖縄で“なこみく”センター新曲初披露 指原「泣きそう…」 |publisher=[[Oricon|Oricon]] |date=2018-04-30 |accessdate=2018-05-10 |deadurl=no |archiveurl=https://web.archive.org/web/20180510104259/https://www.oricon.co.jp/news/2110565/full/ |archivedate=2018-05-10 }}</ref>。

在《[[告示牌_(雜誌)|告示牌]]》Top Singles Sales榜中，單曲首週銷量為173,625張，取得週榜第1名，而在[[日本熱門100金曲榜|日本熱門100金曲榜]]亦同樣是第1名<ref>{{cite web |url=http://www.billboard-japan.com/d_news/detail/63061/2 |language=ja |script-title=ja:【ビルボード】HKT48「早送りカレンダー」17.3万枚を売り上げて総合首位に初登場　SingTuyo「KISS is my life.」ダウンロード1位＆ラジオ5位＆Twitter2位で総合3位に |date=2018-05-09 |publisher=[[告示牌_(雜誌)|告示牌]] |accessdate=2018-05-10 |deadurl=no |archiveurl=https://web.archive.org/web/20180510104259/http://www.billboard-japan.com/d_news/detail/63061/2 |archivedate=2018-05-10 }}</ref>。在[[Oricon公信榜|Oricon公信榜]]中，單曲首週銷量約16.5萬張，位列週榜第1名，並且更新了由自己保持的女藝人自出道作開始連續取得週榜冠軍的紀錄<ref>{{cite web |url=https://www.oricon.co.jp/news/2110984/full/ |language=ja |script-title=ja:【オリコン】HKT48、“なこみく”センター曲で11作連続首位 女性歴代1位記録更新 |date=2018-05-08 |publisher=Oricon |accessdate=2018-05-10 |deadurl=no |archiveurl=https://web.archive.org/web/20180508040133/https://www.oricon.co.jp/news/2110984/full/ |archivedate=2018-05-08 }}</ref>。
