'''塞波加公国'''（{{lang-it|Principato di Seborga}}）是一個位於[[意大利|意大利]]西北部[[利古利亞|利古利亞]]的小城鎮塞波加的半獨立的[[亲王國|亲王國]]。國家元首是[[塞波加君主|塞波加君主]]。鄰近的邊境是獨立自主的[[摩納哥|摩納哥]]。1963年27歲的[[嬉皮|嬉皮]]青年[[西格諾·卡波恩|西格諾·卡波恩]]在查阅村里老神父寫的村史后，發現村子起源於954年，在1079年－1729年之間曾是[[神聖羅馬帝國|神聖羅馬帝國]]下屬的亲王國。眾村民决定「復興亲王國」。

2009年，亲王喬治一世駕崩，但原來喬治一世一直也沒有結婚，也沒有子女。國民計劃在兩個月內從所有國民裡選出新的國王。

== 背景 ==
上述事實，並不意味塞波加是實在的親王國。民間風俗和[[意大利|意大利]]無異。意大利共和國視其為領土內的村莊（不同於同樣領土被包圍在半島的[[聖馬力諾|聖馬力諾]]或[[梵蒂岡|梵蒂岡]]）。早在[[神聖羅馬帝國|神聖羅馬帝國]]時代，塞波加就是它的附庸國。天主教[[熙篤會|熙篤會]]在1118年治理國家。

12世紀得到教宗[[聖伯爾納鐸|聖伯爾納鐸]]支援直到1729年1月20日，本國地位證明塞波加是個獨立親王國，此時親王國是隸屬於[[薩伏依王朝|薩伏依王朝]]的一個小親王國。即是[[維托里奧·阿梅迪奧二世|維托里奧·阿梅迪奧二世]]與薩丁尼亞親王。不管怎樣，他永遠需被記錄在[[薩丁尼亞王國|薩丁尼亞王國]]，寫塞波加臨時過度期區域。後來在1815年，[[拿破崙戰爭|拿破崙戰爭]]和[[維也納會議|維也納會議]]後，這裏的领土被重新分割。奧地利獲得義大利北部的[[倫巴第-威尼斯王國|倫巴第-威尼斯王國]]，許多其它義大利中北部也成為[[哈布斯堡|哈布斯堡]]的領地。教宗的教宗國得到恢復。[[皮德蒙特-薩丁尼亞王國|皮德蒙特-薩丁尼亞王國]]在[[義大利半島|義大利半島]]上的領地被恢復，此外還獲得[[熱那亞共和國|熱那亞共和國]]。1861年[[意大利王國|意大利王國]]統一，但在這裏沒有提到塞波加列在意大利薩伏伊王朝的資產清單上，因此1963年卡波恩宣稱：「它（塞波加）從来就不是現代意大利的一部分！」卡波恩提出這个驚人觀點後，立即得到塞波加村民一致擁護。眾村民决定“復興親王國”，並選舉他成為“塞波加親王國”親王——[[喬治一世親王|喬治一世親王]]。他在60年代在任的時候，就創立花卉合作社，幫助花農的生計的措施。

不過，塞波加親王國和意大利政府間並無緊張氣氛（執行自己的法律、公共衛生、電訊、教育以及其他公共服務）。它在意大利只是個省，塞波加正式交纳稅收，並享有在本地和國家（意大利）的行政权與投票權。例如2001年意大利[[參議院|參議院]]選舉，參議院投票人數高達84.21%。

== 近來事件 ==
1995年4月23日，有個非正式的公民投票（關於永遠承認屬於意大利和接受在意大利之下不合法的意大利法律），塞波加人民投票304人贊成，4人反對。為求獨立於意大利的憲法和普通規則大表贊成。1995年該國還通過《憲法》，“選舉”國王喬治一世為“終身亲王”，該國還另設一個24人議會，並组有8個部長的内閣，因此產生餐館老闆兼任外交部長、而雜貨店老闆娘任議長的狀況。

後來喬治一世亲王突然决定退位。他之所以這麼做是因為覺得自己已屆70高龄，而亲王位“需要更多新能量”。據悉新亲王將通過村民選舉產生，新國王必須在“塞波加村出生的公民”中選出，任何其他地區的人都休想成為該國统治者。

2006年6月，自稱為[[神聖羅馬帝國|神聖羅馬帝國]][[霍亨斯陶芬王朝|霍亨斯陶芬王朝]]的[[腓特烈二世_(神聖羅馬帝國)|腓特烈二世]]後人的女王[[霍亨斯陶芬王朝|霍亨斯憑陶芬安茹金雀花]]寫信給[[意大利總理|意大利總理]]，表示願意把親王國的主權交回意大利，但喬治一世表示Yasmine並非親王國的合法繼承人，所以她的意願對親王國無效<ref>{{cite news|url=http://www.telegraph.co.uk/news/main.jhtml?xml=/news/2006/06/13/wsebo13.xml|title=?|date=2006-06-13|accessdate=2009-12-01}}</ref>。

== 經濟，民間風俗和觀光事業 ==
[[File:Seborga-panorama.jpg|thumb]]
隨著觀光旅遊業的消費逐渐活跃，亲王國歷史悠久的市中心再度復甦，保護守衛由商業的在這裏之上發展。塞波加甚至還有自己的軍隊。據悉該國入口處有個醒目標誌寫有“塞波加亲王國”，並设有出境入境的“海關”。而守衛該國海關和邊境的“國家軍隊”.

塞波加亲王國的當地貨幣是「luigino」，于1994年到1996年期间印刷。國際銀行承認「luigino」的存在，但其並不具法律效力。1「luigino」相當於6美元，可在該村商店和酒吧中使用。

「luigino」是適用於城市以內（歐羅也是同樣合法的流通貨幣，以及以前的意大利里拉和法國法郎）；意大利政府率先表示不歡迎。儘管國内一夜間突然又“冒”出一個“國家”，但是意大利政府并不把他們當回事，只當成他們在搞一場鬧劇。但所有塞波加亲王国的公民都非常有“愛國意識”，他們對外稱自己是“塞波加人”。據悉塞波加亲王國如今擁有362名人口，並有自己的[[憲法|憲法]]、[[國旗|國旗]]、[[護照|護照]]、[[郵票|郵票]]和貨幣。

郵票只有集郵價值，多年來塞波加只在意大利設立唯一的[[郵政中心|郵政中心]]。遊客中心一般簽證是旅遊護照，在國際上是无效的。
