'''塞尔柱灰泥人像'''是[[塞尔柱帝国|塞尔柱帝国]]于11至13世纪盛行的[[纸灰粉刷|灰泥]]人像，是权力和权威的象征，所以这些人像通常会被装饰以明亮色彩并[[镀金|镀金]]。

== 塞尔柱的伊斯兰艺术 ==
[[塞尔柱帝国|塞尔柱帝国]]是[[中亚|中亚]]游牧民族的一个[[突厥|突厥]]王朝，是东方[[伊斯兰世界|伊斯兰世界]]的新统治者。在[[丹丹纳干战役|丹丹纳干战役]]中击败了[[伽色尼王国|伽色尼人]]、[[白益王朝|白益王朝]]后，塞尔柱人成为了[[阿拔斯王朝|阿拔斯王朝]]和[[逊尼派|逊尼派]][[伊斯兰教|伊斯兰教]]的新[[哈里发|哈里发]]。过了仅仅半个世纪，塞尔柱帝国便建立了一个庞大的帝国，其疆域包含现代[[伊朗|伊朗]]、[[伊拉克|伊拉克]]和大部分[[安纳托利亚|安纳托利亚]]。在塞尔柱帝国的统治下，伊朗的文化生活欣欣向荣。在这一时期，众多的建筑艺术得到了发展，并影响了该地区和周边的艺术发展{{sfn|Yalman|2001}}。

在[[陶瓷|陶瓷]]领域，精美的图案是在釉绘画、光泽装饰和多彩画中创造的。金属物体以镶嵌[[银|银]]和[[金|金]]的方式装饰。塞尔柱帝国发明了许多具有动物、男人和女人的形象化图案<ref name="dk">{{cite web |url=https://www.davidmus.dk/en/collections/islamic/dynasties/seljuks |title=The Seljuks and their Successors |author=<!--Not stated--> |date=2017 |website=THE DAVID COLLECTION  |publisher=THE DAVID COLLECTION  |access-date=2017-11-15 |language=英语 |quote=}}</ref>。在[[穆斯林|穆斯林]]文化中，拟人化的表现手法并不少见。但是标志性形象在圣地（如[[清真寺|清真寺]]）是被严格禁止的。而在世俗的地方，这种标志性形象是十分常见的<ref name="metwar"/>。

== 塞尔柱宫殿 ==
现存的所有塞尔柱宫殿都成为了遗址，通过对这些遗迹的挖掘可以证明，塞尔柱宫殿曾经以用[[瓷砖|瓷砖]]和纸灰粉刷浮雕装饰，并绘上[[人像|人像]]和[[几何|几何]]图案。例如，在拉什格里巴扎尔（[[英语|英语]]：{{lang|en|Lashgari Bazar}}）的一个前伽色尼时期宫殿，其中的大殿里发掘出的一个[[壁画|壁画]]上绘制了44名士兵。他们都有类似的圆脸和杏仁形状的眼睛，人种上来讲与来自[[中亚|中亚]]的[[突厥|突厥]]人有关{{sfn|Öney|1984|pp=133-5}}。

塞尔柱灰泥人像一般用来装饰类似于观众厅的皇家宫殿。例如，有些塞尔柱灰泥人像被发现装饰于塞尔柱苏丹的大宫殿，或当地附庸或继承人的较小皇室<ref name="metwar"/>。塞尔柱灰泥人像可能是更大的灰泥装饰的一部分，隐藏在它后面的基础墙。一个完整的灰泥人像的例子来自12世纪后期的[[雷伊_(伊朗)|雷伊]]。在[[阿富汗|阿富汗]]{{link-en|巴斯特 (阿富汗)|Bast, Afghanistan|巴斯特}}、[[撒马尔罕|撒马尔罕]]和[[乌兹别克斯坦|乌兹别克斯坦]]均能发现类似的例子<ref name="kianush">{{cite web |url=http://www.artarena.force9.co.uk/seljuks.htm |title=THE SELJUKS 11th - 13th Century |last=Kianush |first=K. |date=1999 |website=IRANSAGA |publisher=Art Arena |access-date=2017-11-15 |language=英语 |quote=}}</ref>。这些灰泥人像被涂成红色、蓝色或者其他明亮的颜色并镀金。如果宫殿里有比较黑暗的房间，意味着这些灰泥人像需要尽可能的突出<ref name="metwar"/>。

== 形式 ==
[[File:Persian_-_Standing_Warrior_Holding_a_Sword_-_Walters_241.jpg|thumb]]发现的一个战士的灰泥人像]]

灰泥是一种柔软的水泥状水基材料，无论在干燥环境中还是潮湿环境中都易于雕刻。除此之外，它的轻便性使其易于粘贴到墙上。许多创作于12世纪的塞尔柱灰泥人像被完好保存下来，因为它们的沙漠一直保持着干燥的环境。一般而言，塞尔柱灰泥人像通常会被涂成红色、蓝色或者其他明亮的颜色并镀金<ref name="metwar">{{cite web |url=https://www.metmuseum.org/art/collection/search/451909 |title=Standing Figure with Jeweled Headdress |date=2017 |website=The Metropolitan Museum of Art |publisher=The Metropolitan Museum of Art |access-date=2017-11-15 |quote=}}</ref>。

塞尔柱灰泥人像被认为是权力和权威的象征，例如皇家卫士、皇家大臣和朝臣等。战士的人像被描绘为握住一把剑，并穿着色彩丰富的长袍，长裤，和长长的靴子。皇室的人物被描绘成戴着皇冠。收藏于[[美国|美国]][[纽约市|纽约市]][[大都会艺术博物馆|大都会艺术博物馆]]的两个塞尔柱灰泥人像都戴着冠冕，其中一个人戴着翼冠。塞尔柱的所有灰泥人像都有圆形的面孔，典型的[[高颧骨|高颧骨]]和杏仁状的眼睛，反映了[[突厥人|突厥人]]和[[蒙古人|蒙古人]]的人种特征<ref name="dk"/>。
