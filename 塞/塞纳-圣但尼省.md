'''塞纳-圣但尼省'''（{{lang-fr|le département de la Seine-Saint-Denis}}）是[[法国|法国]]的一个[[省_(法国)|省]]。省会为[[博比尼|博比尼]]。

== 历史 ==
塞纳-圣但尼省成立于1968年1月1日，由原[[塞纳省|塞纳省]]的东北部（24个城镇）和原[[塞纳瓦兹省|塞纳瓦兹省]]的一小部分（16个城镇）组成。

== 地理 ==
塞纳-圣但尼省位于[[巴黎|巴黎]]的东北部，面积236平方公里，是法国最小的省之一。另两个为[[上塞纳省|上塞纳省]]和[[瓦勒德马恩省|瓦勒德马恩省]]，她们都是巴黎近郊省份。

== 气候 ==
与巴黎一样，秋季多雨，夏季炎热。

== 经济 ==
塞纳-圣但尼省有两个著名的经济区域：[[夏尔·戴高乐国际机场|夏尔·戴高乐国际机场]]机场区（{{lang|fr|Aéroport Roissy-Charles-de-Gaulle}}）和塞纳-圣德尼省城镇联合体（{{lang|fr|La Communauté d'agglomération Plaine Commune}}，指巴黎东北部城镇密集地区）。

每年有近700家企业创建于这个省份，但相对邻近省份较低的受教育水平，导致了本省较高的失业率。

== 人口 ==
塞纳-圣但尼省居民数约為170萬人。大量來自北非的移民移居於此。

塞納-聖但尼省估計有60萬[[穆斯林|穆斯林]]，是法國穆斯林最多的地區之一。全省25歲以下人口超過4成失業，毒品及暴力罪案頻發。省內設有40個[[市镇_(法国)|市鎮]]，其中36個被法國政府列為「敏感城區」，被外界形容為連法國警察和[[國家憲兵_(法國)|國家憲兵]]也不敢進入的「生人勿近」區域。<ref name=gatestone2015>{{Cite web |url=http://www.gatestoneinstitute.org/5128/france-no-go-zones |title=European 'No-Go' Zones: Fact or Fiction? Part 1: France |quote=A 120-page research paper entitled "No-Go Zones in the French Republic: Myth or Reality?" ... 36 of which are on the French government's official list of "sensitive urban zones" or ZUS. ... rejecting French values and instead are immersing themselves in radical Islam. ...}}</ref><ref name=mingpao2015>[http://www.mingpaocanada.com/tor/htm/News/20151119/ttaa4_r.htm 聖但尼省60萬穆斯林 輿論稱禁區 「外人勿近」]，明報加東版(多倫多)，2015-11-19。</ref>

法國智庫{{Link-en|蒙田研究所|Institut Montaigne}}在2011年發表報告，指出塞納-聖但尼省的穆斯林移民逐漸受[[伊斯蘭|伊斯蘭]]激進思想影響，而[[伊斯蘭教法|伊斯蘭教法]]在塞納-聖但尼省正逐漸取代法國法律，大有形成「伊斯蘭隔離社區」之勢。<ref name=gatestone2015/><ref name=mingpao2015/>

塞納-聖但尼省現在是全法國犯罪率最高的省。<ref>{{Cite news |url=http://www.bbc.com/zhongwen/simp/world/2016/08/160814_france_chinese_death |title=法国华人遭殴打致死 华社举行哀悼仪式 |work=BBC中文網 |date=2016-08-14}}</ref>

==交通==
5条[[法兰西岛大区快铁|法兰西岛大区快铁]]均经过塞纳-圣但尼省境内。此外，[[塞纳-圣但尼省城际客运|塞纳-圣但尼省城际客运]]下属20余条线路，覆盖了该省中部的大部分市镇。

== 教育 ==
塞纳-圣但尼省有兩所大学，[[圣但尼大学|圣但尼大学]]（即[[巴黎第8大学|巴黎第8大学]]）和[[北巴黎大学|北巴黎大学]]（Université Paris-Nord）多个技术学院（[[IUT|IUT]]以及[[大学校|大学校]]）。

== 文化 ==

== 传媒 ==
数年以前，塞纳-圣但尼[[省委员会|省委员会]]发行了月刊《塞纳-圣德尼省杂志》。

== 旅游 ==
* [[布日机场|布日机场]]（Le Bourget）
* 法国航空航天博物馆
* [[圣但尼市|圣但尼市]]（Saint-Denis）
* 于[[八世纪|八世纪]]建造的[[哥特|哥特]]风格的[[圣但尼|圣但尼]][[主教堂|主教堂]]（Basilique de Saint-Denis）
* [[法兰西体育场|法兰西体育场]]（Stade de France）

另由于此地历史悠久，且曾为法国国王领地，故该地区的建筑十分有特色。

== 政治 ==
塞纳-圣但尼省曾是[[法国共产党|法国共产党]]的两个在省议会中占据多数的省份之一（另一个为[[马恩河谷省|马恩河谷省]]），但近几年移民的衝突，此省也成为法國右翼民粹主義政黨[[民族阵线_(法國)|民族阵线]]（Front national）的主要根据地。

2005年11月在此省的许多城镇发生了[[2005年法国骚乱|动乱]]。
