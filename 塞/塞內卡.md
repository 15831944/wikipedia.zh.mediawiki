'''卢修斯·阿奈乌斯·塞內卡'''或'''辛尼加''' （[[拉丁语|拉丁语]]：{{lang|la|Lucius Annaeus Seneca}}，{{bd||约前4年|65年|catIdx=S}}），[[古羅馬|古羅馬]]時代著名的[[科爾多瓦|科爾多瓦]][[斯多亞學派|斯多亞學派]][[哲學家|哲學家]]、[[政治家|政治家]]、[[劇作家|劇作家]]。曾任[[尼祿|尼祿]]皇帝的導師及顧問，62年因躲避政治鬥爭而引退，但仍於65年被[[尼祿|尼祿]]逼迫，以切開血管的方式自殺。塞內卡主张：提高道德、智慧，保持精神上的安宁是人唯一的任务，鼓吹[[禁欲主义|禁欲主义]]，要求人们放弃现实生活和欲望，等待神的启示和精神上的解脱。由于塞内卡的这种思想对未来基督教影响较大，被德国哲学家[[恩格斯|恩格斯]]在其著作《布鲁诺·鲍威尔和早期基督教》中称为“[[基督教|基督教]]的叔父”。<ref name="bad">{{cite book |author1=周启迪 |title=世界上古史 |date=2016.6 |publisher=北京师范大学出版社 |location=中国北京 |page=300 |edition=第四版}}</ref><ref>{{cite web |author1=恩格斯 |title=布鲁诺·鲍威尔和早期基督教 |url=https://www.marxists.org/archive/marx/works/1882/05/bauer.htm |accessdate=2018-09-02}}</ref>

== 著作 ==
* 《對話錄》（Dialogues）12卷
* 《論憐憫》（De clementia）2卷
* 《論恩惠》（De beneficiis）7卷
* 《書信集》（Epistles）124篇
* 《[[天問_(塞內卡)|天問]]》（Naturales quaestions）7篇
另有悲劇8種

== 名言 ==
* 保護人民是一個國君最偉大的美德。
* 不是因為事情困難，而讓我們不敢做；是因為我們不敢做，事情才變得困難。
* 真理不会使你富有，但真理会让你自由。
* 一個人越卑鄙，他就越固執地想扮演高尚的角色，有些人還因此成功了
* 機會總是眷顧那些做好準備的人。
*其實不用擔心，你們中的很多人一輩子都不會遇見你夢想的真愛。只會因為害怕孤獨地死去而選擇隨便找個人，相互飼養。
