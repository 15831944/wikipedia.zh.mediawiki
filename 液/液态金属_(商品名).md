'''Liquidmetal'''（由'''液態'''與'''金屬'''兩字所複合）與 '''Vitreloy'''是一系列由[[加州理工學院|加州理工學院]]研究團隊所開發出來的[[非晶态金属|非晶态金属]][[合金|合金]]的商業名稱，目前由該團隊所組織的液態金屬科技公司（Liquidmetal Technologies Inc.<ref>[http://www.liquidmetal.com/ Liquidmetal Technologies Inc. 官方網站(英文)]</ref>）進行行銷，並是公司的產品名稱與商標名稱。

雖然名稱中包含液態，但在常溫下其實並不是液態，而是非結晶的固態（又稱[[无定形体|无定形体]]），類似[[玻璃|玻璃]]，這樣的物質也被稱為'''金屬玻璃'''<ref> {{Cite web 
	|url = http://highscope.ch.ntu.edu.tw/wordpress/?p=2962 |title = 金屬玻璃（Metallic Glass） 
	|author = 何鎮揚老師, 葉名倉教授|date = 2009-07-29 
	|language = 中文 
	 }}</ref>。

==特性==

「液態金屬」這種合金內含數種原子，這些原子的大小具有顯著差異，形成一種低自由空間的緊密混合物。 這一物質不像結晶物質那樣在明顯的熔點下由固態突然轉為液態，反而更像是玻璃：隨著溫度的升高，[[黏度|黏滯度]]會逐漸降低。 由於在較高溫時具有可塑性，因此在使用模具進行成型時，可以易於控制它的機構特性。 這一黏滯性也防止原子產生足夠的移動而構成有秩序的[[晶體結構|晶格]]<ref>(晶格的英文是 lattice</ref>，因此在加熱成型與冷卻之後，仍然能保持非結晶的狀態。

==用途==
Liquidmetal 結合了多種在一般金屬中所沒有的特性，因此有廣泛的應用。

第一個商業用途是製作[[高爾夫球|高爾夫球]]所用球杆的杆頭，

[[File:IPhone_SIM_Removal_Tool.jpg|thumb]] [[SIM卡|SIM卡]]取卡工具]]
一個知名的用途是[[蘋果公司|蘋果公司]]在所發售的[[iPhone|iPhone]]中所附的 SIM 卡取卡工具是使用 liquidmetal 所製，被認為是該公司對這一金屬材料的使用可行性上的操練。<ref>[http://www.appleinsider.com/articles/10/08/17/liquidmetal_created_sim_ejector_tool_for_apples_iphone_ipad.html Liquidmetal created SIM ejector tool for Apple's iPhone, iPad]</ref>
