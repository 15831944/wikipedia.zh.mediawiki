'''汗液'''，或'''汗'''，是由[[人|人]]等[[高等动物|高等动物]]透過[[汗腺|汗腺]]所分泌出的液體。汗的分泌受到[[植物性神经系统|植物性神经系统]]调节。

== 汗液的作用 ==
* 降温：在汗腺排出汗液的过程中，汗液的排出和蒸发帮助生物带走大量的热量，达到散热的目的；
* 保护：湿润[[皮肤|皮肤]]、软化角质，酸化并抑制[[细菌|细菌]]；
* [[排泄|排泄]]：排出體內廢物。

== 主要成分 ==
汗液的主要成份是[[水|水]]，约占总成分的98%到99%，其余物质为[[氯化钠|氯化钠]]，极少量的[[尿素|尿素]]、[[氨|氨]]和其他[[盐|盐]]类。

在人体生理中，小汗腺分泌的汗液，主要包括水、氯化钠、[[钾|钾]]、[[钙|钙]]、尿、 阿摩尼亞、[[乳酸|乳酸]]及[[氨基酸|氨基酸]]等；大汗腺分泌的汗液，除水外主要有[[铁|铁]]、脂质(中性脂肪、[[脂肪酸|脂肪酸]]、[[胆固醇|胆固醇]]及[[类脂|类脂]]质)、荧光物质、有臭物质。

== 中医观点 ==
在中医学上认为：
* 汗系人体内津液所化生，通过卫气的调节从汗孔而排泄。《[[灵枢|灵枢]]·决气》：“汗出溱溱，是谓津。”
* 发汗有散热和调节体温的作用。《[[素问|素问]]·生气通天论》：“体若燔炭，汗出而散。”
* 汗与血液有密切关系，由于“心主血”，汗是津液所化生，故称“汗为心之液”，并有“夺血者无汗，夺汗者无血”之说(《灵枢·营卫生会》)。
