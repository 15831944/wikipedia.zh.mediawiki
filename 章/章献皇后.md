'''章献明肃皇后'''（{{bd|968年||1033年|5月11日|catIdx=L刘}}），名不詳（一說名'''刘娥'''），[[益州|益州]]华阳（今[[四川省|四川省]][[成都市|成都市]][[华阳镇|华阳镇]]）人，是[[宋真宗|宋真宗]]赵恒的[[皇后|皇后]]，[[宋朝|宋朝]]第一位[[摄政|摄政]]的[[皇太后|皇太后]]，功绩赫赫，常与[[汉朝|汉]]之[[呂雉|吕太后]]、[[唐朝|唐]]之[[武则天|武则天]]并称，[[史书|史书]]称其“有吕武之才，无吕武之恶”。

== 早年 ==
刘娥生于[[宋太祖|宋太祖]]开宝元年（968年），[[祖籍|祖籍]][[太原|太原]]，祖父是[[后晋|后晋]]、[[后汉|后汉]]的右骁卫大将军刘延庆，父亲[[刘通_(北宋)|刘通]]是宋太祖时的虎捷都指挥使、领[[嘉州_(北周)|嘉州]]（今四川乐山）[[刺史|刺史]]。刘娥早年嫁給蜀地的一个银匠龚美。劉娥跟隨丈夫龔美一起來到京城[[開封|開封]]謀生。劉娥善击鼗（[[拨浪鼓|拨浪鼓]]），頗有名氣，後遇上王府选姬，龔美易名刘美，謊稱為刘娥之兄，並让刘娥入王府。刘娥入府後頗得寵，但赵恒的[[乳母|乳母]][[秦国夫人|秦国夫人]]认为刘娥出身寒微，劝赵恒不要亲近刘娥未果，秦國夫人只好告知[[宋太宗|宋太宗]]。太宗大怒，命逐刘娥出京，並為趙恆選婚，赐[[忠武军节度使|忠武军节度使]][[潘美|潘美]]的八女潘氏为莒国夫人。赵恒虽迫于皇命將刘娥送出王府，卻將刘娥偷偷藏在王宫指挥使张耆家中，不时与刘娥私会。

== 妃嫔时期 ==
[[至道|至道]]三年三月癸巳日，五十九岁的宋太宗病逝，遗诏传位于已立为太子两年的赵恒。赵恒登基為宋真宗後冊立郭氏为皇后，又追封早逝的莒國夫人潘氏为莊怀皇后（后来宋仁宗改为章怀皇后），未幾，又將刘娥迎入後宫。[[景德|景德]]元年（1004年）正月，封刘娥为[[美人_(位号)|美人]]。

景德初年，郭皇后年仅九岁的儿子[[赵祐|赵祐]][[夭折|夭折]]；半月后，另一名两月大的皇子也夭折了。郭皇后伤心过度，在景德四年（1007年）四月十六日病逝。真宗欲冊立劉美人為繼-{后}-，但劉美人無子嗣且出身低，群臣都不赞同，反而請奏册立十四岁的[[沈贵妃_(宋真宗)|沈才人]]为皇后。沈才人雖在真宗登基後於[[大中祥符|大中祥符]]元年（1008年）才入宫，但出身世家，為宰相[[沈伦|沈伦]]之孙。真宗不悦，未應允群臣請奏，不再谈論立-{后}-之事。

刘美人虽有宠，却無子嗣，劉美人的侍女[[章懿皇后_(宋真宗)|李氏]]，一日突夢仙人下降为子，後於大中祥符三年（1010年）四月十四日生下一子赵受益（即[[宋仁宗|宋仁宗]]赵祯）。刘美人取李氏之子撫養，因此皇子虽是李氏所生，却只認刘娥为母，而真宗早在皇子出生前三月便宣布刘美人怀孕，並進劉美人為[[修仪|修仪]]。與刘修儀交好的[[杨淑妃_(宋真宗)|杨才人]]则晋封為[[婕妤|婕妤]]。皇子虽是刘氏之子，實際上多是杨婕妤抚养。

劉修儀性情機敏，通書史，熟知朝政。宋真宗每每在退朝批閱奏章至深夜，多次參考劉修儀的意見。之後赵受益的生母李氏被封为崇阳县君。不久，李氏又因生下一女而晋封才人，但皇女不久夭亡。李氏自认命薄无福，终其一生都并未与儿子相认。大中祥符五年十二月丁亥，真宗終於如願冊立劉修儀為皇后。

== 掌權皇后 ==
[[天禧_(北宋)|天禧]]四年二月（1020年），真宗患病，難以處理政事，政务及群臣的奏章实际上都由劉皇后處理。後來真宗病篤，下诏由皇太子赵禎在资善堂听政，劉皇后从旁辅助。真宗對於劉皇后參政的認可引起群臣不安，並有史家認為此事與[[北宋帽妖案|北宋帽妖案]]有內在邏輯關聯。

== 掌政太后（1022-1033） ==
[[乾兴|乾兴]]元年（1022年）二月甲寅，宋真宗病逝于延庆殿，遗诏曰：太子赵禎即位，皇-{后}-刘氏为皇太-{后}-，杨淑妃为[[皇太妃|皇太妃]]，军国重事“权取”皇太-{后}-处分。而小皇帝赵祯此时只有十一岁，实际上就是由刘太后处理政务。宰相丁谓想独揽大权，後被劉太后所貶。丁謂罷相貶黜後，刘太后开始和仁宗一起听政决事，正式[[垂帘听政|垂帘听政]]。

刘太后自知出身卑微，宋朝以[[士大夫|士大夫]]为尊，因此大力抬高母家，一直追尊加封祖宗：曾祖父[[刘维岳|刘维岳]]成了天平军节度使兼侍中兼中书令兼尚书令，曾祖母宋氏最后封到安国太夫人；祖父[[刘延庆|刘延庆]]为彰化军节度使兼中书令兼许国公，祖母元氏封齐国太夫人；父亲[[刘通|刘通]]为开府仪同三司魏王，母亲庞氏封晋国太夫人。

刘太后号令严明，赏罚有度，虽然难免有些偏袒家人，但并不纵容他们插手朝政。在大是大非面前，她更尊重士大夫们的意见，[[王曾|王曾]]、[[张知白|张知白]]、[[吕夷简|吕夷简]]、[[鲁宗道|鲁宗道]]都得到了她的重用，刘氏姻族也没有做出为害国家的祸事。刘太后性格简朴，為皇后时服饰简朴，当了太-{后}-依然未改习性。宫中侍女见皇帝侍女服饰华丽，觉得自己身为太-{后}-侍女，怎么能被比下去呢？报与刘太后，刘太后不为所动，說：“那是皇帝嫔御才能享用的，你们哪有这样的资格？”

虽然刘太后掌权日久，不愿把权柄交给仁宗，但她却依然是个慈母。仁宗少时体弱多病，刘太后忙于政务，让[[杨淑妃_(宋真宗)|杨淑妃]]照顾。仁宗称刘太后为“大娘娘”，杨妃为“小娘娘”。而仁宗生母李氏，刘太后晉封她为顺容，迁往真宗[[永定陵|永定陵]]，成为守陵的先帝诸妃之一。而且早在真宗年间，便寻访到李氏家人封官。

[[明道_(宋仁宗)|明道]]元年（1032年）二月，李氏患了重病，刘太后连忙派太医前去诊治，并晋封她为'''[[宸妃|宸妃]]'''。然而李氏封妃当天便病逝，享年四十六岁。起初，太后只想以普通宫人的身份殓葬了事，然而听了[[宰相|宰相]][[吕夷简|吕夷简]]的劝说，以一品禮儀給李妃在[[皇仪殿|皇仪殿]]治丧，并给李妃穿上皇太后冠服下葬。李妃的父亲得到追封，兄弟[[李用和|李用和]]也再次晋升。

刘太后虽不愿还政于仁宗，却并未想过自立。[[程琳_(北宋)|程琳]]献图《武后临朝图》，刘太后亲掷于地，道：“我绝不会做这样的事！”太后表态后，群臣如释重负，仁宗也心怀感激，恭孝唯谨，更于[[天圣|天圣]]七年（1029年）九月颁布[[诏书|诏书]]，将太后生辰[[长宁节|长宁节]]的仪礼升级到与皇帝生辰[[乾元节|乾元节]]相同的程度。

明道二年（1033年）二月，举朝要行祭[[太庙|太庙]]大典，刘太后自觉天命已不久，想要在生前穿一次天子[[衮冕|衮冕]]，便提出自己要着衮冕祭祀太庙。群臣大哗，却只得将皇帝衮衣上的饰物稍减了几样，呈了上去。

明道二年二月乙巳这天，皇太-{后}-刘氏穿着天子衮衣、头戴仪天冠，在近侍引导下步入太庙行祭典初献之礼，並由楊太妃擔任亚献、终献为仁宗皇后[[郭皇后_(宋仁宗)|郭氏]]。仪式结束后，刘太后在太庙文德殿接受了群臣给自己上的尊号：'''应天齐圣显功崇德慈仁保寿皇太后'''。自此，刘太后彻底还政于仁宗。

明道二年三月，刘太后病重，仁宗[[大赦|大赦]]天下為劉太后祈福，並四处征召名医，却无法挽留刘太后的生命。甲午日刘太后病逝于[[宝慈殿|宝慈殿]]，享年六十五岁；第二日，仁宗在[[皇仪殿|皇仪殿]]召群臣，哭道：“太后临终前数度拉扯身上衣服，可有什么心愿未了？”参知政事[[薛奎|薛奎]]曰：“太后不愿先帝见她身穿天子服入葬。”仁宗恍然大悟，下令给刘太后换上皇后冠服。

仁宗正伤感，群臣却纷纷上议说刘太后并非生母，生母是李宸妃，而燕王更说李宸妃是劉太后所毒殺。仁宗震惊，派人召来[[李用和|李用和]]，让他亲自去查看李宸妃的棺木。後仁宗前往李妃暫厝的洪福院，见李宸妃葬品如一品夫人，身穿皇太-{后}-冠服。仁宗大悔，叹道：“人言岂可尽信。”並来到刘太后[[牌位|牌位]]前拜谢自责。同年九月仁宗下诏，刘太后和李妃同时迁葬[[永定陵|永定陵]]。灵柩起驾这天，仁宗先为刘太后发引，不但执孝子礼，还不顾宰相们的劝阻亲自执绋之礼，一直步行送出皇仪殿。随后他才再去往李宸妃停棺的洪福院为生母起灵，伏在棺木上痛哭道：“劬劳之恩，终身何所报乎！”

刘太后死后，刘氏家族受尊崇更胜昔日。刘太后曾有遗诏，命仁宗尊养母杨太妃为皇太后。仁宗遵其旨意，尊封杨氏为'''保庆皇太后'''，杨太后虽未垂帘听政，仁宗却克尽孝道奉养。

==野史==
从明朝流传下来“[[狸猫换太子|狸猫换太子]]”的故事，说宋真宗的德妃刘氏和宸妃李氏同时有孕，李宸妃先产下皇子，刘德妃妒忌，勾结李宸妃身边内官，把一只剥了皮的[[狸猫|狸猫]]换去[[皇子|皇子]]，真宗以为李宸妃产下怪胎，把李宸妃打入冷宫，将刘德妃生下的皇子立为储君，并册立刘德妃为皇后。

另一种说法则是刘德妃的皇子不幸夭折，于是刘德妃把李宸妃的儿子据为己有，宣称是自己的儿子，真宗照样立她为后。

两种说法的结局都是：刘氏逼李宸妃自尽，却有好心的宫人代李妃而死，而李妃流落民间，直到包拯横空出世，才得以揭开这桩宫闱迷案，使李妃与儿子相认。刘氏因为做了坏事不久便死去，包拯也因为替宋仁宗找回了亲生母亲而官升龙图阁大学士。

近世考據此說認為是源自明朝所出的民間故事，故事當中的劉德妃相關事蹟可能是影射[[明憲宗|明憲宗]]的寵妃[[恭肃端顺荣靖皇贵妃|萬貴妃]]的事蹟，宋仁宗應是隱喻[[明孝宗|明孝宗]]，時人有感明孝宗未對萬貴妃害母之行為做出嚴懲反而厚葬善待，故編出“[[狸猫換太子|狸猫換太子]]”的故事以償不平之鳴。
