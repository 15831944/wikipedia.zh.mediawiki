《'''宮心計'''》（{{lang-en|'''Beyond the Realm of Conscience'''}}）是[[香港|香港]][[電視廣播有限公司|電視廣播有限公司]]拍攝製作的古裝宮廷[[劇集|劇集]]，由[[佘詩曼|佘詩曼]]、[[楊怡|楊怡]]、[[陳豪|陳豪]]及[[鄭嘉穎|鄭嘉穎]]領銜主演，並由[[米雪|米雪]]、[[關菊英|關菊英]]、[[李施嬅|李施嬅]]、[[韓馬利|韓馬利]]、[[謝雪心|謝雪心]]及[[李國麟_(演員)|李國麟]]聯合主演。監製[[梅小青|梅小青]]，此劇為2009年節目巡禮劇集之一及2009無綫節目精選第二季劇集之一，亦是[[無綫電視|無綫電視]]'''42周年台慶劇'''。由於劇集本身涉及大量宮廷內鬥元素，故坊間經常以《[[大長今_(電視劇)|大長今]]》、《[[金枝慾孽|金枝慾孽]]》與之比較。

==劇情==
唐朝掌管後宮內務的最高權力中心——尚宮局，統領四房，分別為司珍房（首飾）、司制房（刺繡）、司膳房（膳食）及司設房（陳設）。劉三好（[[佘詩曼|佘詩曼]] 飾）與姚金鈴（[[楊怡|楊怡]] 飾）自幼入宮當學婢，二人聰穎機伶，尤以三好品性純良，得司珍阮翠雲（[[關菊英|關菊英]] 飾）及司制鍾雪霞（[[米雪|米雪]] 飾）喜愛，爭取收歸門下，三好遂成二人爭鬥禍心。 三好一直緊守亡母遺訓—「做好事、說好話、存好心」，以善行仁心遊走於宮中這個爾虞我詐的地方；反而情同姐妹的金鈴，利慾薰心，為求成妃，不擇手段。縱使三好鍾情棋博士高顯揚（[[鄭嘉穎|鄭嘉穎]] 飾），但金鈴知道皇帝李怡（[[陳豪|陳豪]] 飾）最愛是三好，妒意驟生，為保皇妃之位，竟誣告三好擊殺太后（[[韓馬利|韓馬利]] 飾），要置三好於死地。但計謀再深，總有算錯的時候；手段再強，亦有疏忽的地方；透過三好與金鈴迥異的人生，帶出只有憑「行善積德」，才是處世待人「至強無敵」之道。

==歷史考證==
# 唐朝之女官有六局，分別為[[尚宮局|尚宮局]]、[[尚儀局|尚儀局]]、[[尚服局|尚服局]]、[[尚食局|尚食局]]、[[尚寢局|尚寢局]]、[[尚功局|尚功局]]。
##劇中之司設隸屬[[尚寢局|尚寢局]]、司膳隸屬[[尚食局|尚食局]]。而司珍、司制則隸屬[[尚功局|尚功局]]並非尚宮局。<ref>《新唐書》<br>志第三十七　百官二</ref>
# 妃子之間的稱呼應是「妃」，妃子對皇后應稱呼「皇后」，不會以「姊妹」相稱。当时亦无“娘娘”之称。
# 劇中所有女性的穿著應是五代加宋初的混合體。
#唐初的時候，後宮制度是四夫人，而劇中出現的惠妃、麗妃、是唐朝中期嬪妃的封號，是三夫人之一 ，另一是華妃（沒有在劇中出現過）。但是在唐朝中後期已改回四夫人(貴淑德賢)，因此麗妃和賢妃/德妃不可能同時出現。
# 歷史上，[[馬元贄|馬元贄]]認為光王愚鈍所以在武宗駕崩後迎立光王，即位以後，宣宗顯出並非愚鈍的本性，而[[馬元贄|馬元贄]]的下場史書並沒有記載。
# 歷史上，宣宗非常討厭李德裕，並非敬重。
# 太皇太后郭氏於宣宗即位後，仍為「太皇太后」，並未降為「太后」，其暴卒於「興慶宮」並非「上陽宮」。<ref>《舊唐書》<br>列傳第二 后妃下</ref>
# 郑太后在宣宗死后仍在世，被尊为太皇太后。宣宗登基时已有数子。
# 唐朝的人還是以正坐為主，很少用椅子，古代后妃接見男性時除親屬之外都會垂簾，成為后妃後即使是父母也要向女兒行臣禮，不可以再直呼女兒名字，更不可能當眾教訓女兒。
# 三品將軍夫人當眾對一品妃子無禮而不可能不受到任何處分。
# 日本棋博士都是[[江戶時代|江戶時代]]武士打扮，當時日本是[[平安時代|平安時代]]，與平安時代的服飾、談吐、禮儀完全不吻合。況且，當時未有鬼塚、赤村、松蒲這些苗字。
