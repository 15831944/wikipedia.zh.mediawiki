'''宮之坂站'''（{{jpn|j=宮の坂駅|hg=みやのさかえき|rm=Miyanosaka eki|f=y}}）是一個位於[[東京都|東京都]][[世田谷區|世田谷區]]宮坂一丁目，屬於[[東京急行電鐵|東京急行電鐵]][[世田谷線|世田谷線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''SG07'''。
== 歷史 ==
* 1945年（[[昭和|昭和]]20年）7月15日 - 近[[上町站|上町]]的豪德寺前站與近山下的舊宮之坂站統合成為宮之坂站。
* 1966年（昭和41年）1月20日 - 站名的片假名「ノ」改為平假名「の」。

== 車站構造 ==
[[對向式月台|對向式月台]]2面2線的[[地面車站|地面車站]]。下行月台內一度有商店營業，但在[[昭和|昭和]]60年代發生火災後閉店。

== 利用狀況 ==
官方未公布本站的乘車人數。世田谷線全線的乘車人數請參照[[東急世田谷線#使用情況|東急世田谷線#使用情況]]。

== 車站周邊 ==
* [[下高井戶站|下高井戶]]方向月台旁是世田谷區的宮坂區民中心。車站有展示過去用於世田谷線、之後用於[[江之島電鐵|江之島電鐵]]的{{link-ja|江之島鎌倉觀光600形電車|江ノ島鎌倉観光600形電車|車輛（東急デハ87→江ノ電601）}}。
* 附近有{{link-ja|世田谷八幡宮|世田谷八幡宮}}。站名取自於神社旁的[[宮坂|宮之坂]]（{{link-ja|東京都道427號瀨田貫井線|東京都道427号瀬田貫井線|都道427號}}）。該坂在世田谷線開通時部分崩塌，因此長度較短。
* 本站是[[招財猫|招財猫]]發源地[[豪德寺|豪德寺]]的最近站。但要鄰站[[山下站_(東京都)|山下站]]距離[[小田急小田原線|小田急小田原線]][[豪德寺站|豪德寺站]]較近。
* 站名念作'''みやのさか'''，但地名念作'''みやさか'''。
