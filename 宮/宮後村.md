'''宮後村'''是[[中華民國|中華民國]][[嘉義縣|嘉義縣]][[新港鄉|新港鄉]]轄下的行政區，面積約0.73平方公里，戶政計24鄰、人口共1826人。<ref>{{Cite web|url=http://minsyong-hro.cyhg.gov.tw/News.aspx?n=B7A00EFC5D61DE2C&sms=984706722BDB3061|title=新港鄉村里鄰人口數|accessdate=2017-07-06|author=|date=|publisher=嘉義縣民雄戶政事務所}}</ref>

== 地理 ==
宮後村位於新港鄉中央地帶，村內約三分之二是農地。東臨[[大潭村|大潭村]]，南靠[[宮前村|宮前村]]，西接[[大興村|大興村]]，北面[[古民村|古民村]]。經縣道157號往東北方向可達[[溪口鄉_(台灣)|溪口鄉]]。

== 聚落歷史 ==
宮後村有數個舊地名，如：現今的大興路與宮後街之間、東興廟的東方，因昔日建有四間連棟店鋪，故稱「四崁仔」；新港市街的東北方稱「後角仔」；舊時多林投樹，又名「林投巷」；頂園仔（後壁園仔）是新港市街東北方的菜園；新港街北方的頂頭位置稱「北頭仔」，約在現今的水利工作站與中正路115巷之間。<ref>{{Cite book|title=嘉義縣志：地理志|last=|first=|publisher=嘉義縣政府|year=2010|isbn=9789860208016|location=嘉義縣太保市|pages=679-682}}</ref>

== 教育 ==
[[國立新港藝術高級中學|國立新港藝術高級中學]]於2006年（民國95年）8月1日正式成立。在2008年新校舍落成啟用前，暫借[[國立嘉義大學|國立嘉義大學]]林森校區作為臨時校區。王延煌先生為第一任校長，廖連喜先生為第二、三任校長。<ref>{{Cite web|url=http://www.sgash.cyc.edu.tw/editor_model/u_editor_v1.asp?id={442BC169-7EFE-48EF-84F9-DEB0C5D0E618}|title=國立新港藝術高中校史|accessdate=2017-07-06|author=|date=|publisher=}}</ref>

== 其他特色 ==
