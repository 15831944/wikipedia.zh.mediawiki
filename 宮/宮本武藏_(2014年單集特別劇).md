《[[宮本武藏|宮本武藏]]》（日語：宮本武蔵）為朝日電視台為慶祝慶祝開播55週年連續兩晚播出的兩集特別劇，撥出日期為2014年3月15日及16日，由木村拓哉主演，這是他繼1995年的電視劇《君は時のかなたへ》及2006年電影《武士的一分》以來的最新時代劇作品。宣傳標語為「'''斬った数だけ、強くなれた。斬った数だけ、涙を流した。'''」（殺的人越多，能使你更強。殺的人越多，流下悲情淚。）。第一夜收視率為14.2%，第二夜則為12.6%（關東地區）<ref>{{Cite web |url=http://npn.co.jp/article/detail/32170905/ |title=【週刊テレビ時評】どうした？ “平成の視聴率男”キムタク　テレ朝開局55周年記念ドラマ「宮本武蔵」も不振 |publisher=リアルライブ |date=2014-03-18 |accessdate=2014-04-06}}</ref>。改編自吉川英治的同名小説，本劇也是2003年播出的NHK大河劇《武藏 MUSASHI》的11年後的最新改編電視劇作品。

== 劇情概要 ==
無師自通的宮本武藏，年少輕狂，為成為天下第一而到處踢館。一路下來的經歷及磨練讓自己領悟到劍禪合一的精髓，巖流島之戰更讓他揚明千古，成為歷史上鼎鼎有名的劍客！關原之戰的戰場上，他遇見了武藝高強的佐佐木小次郎，令他敬佩不已。在走遍天下、四處踢館之後，最後與宿敵佐佐木小次郎重逢，兩人間的對決在巖流島正式展開…<ref>http://japan.videoland.com.tw/channel/20140903/introduction.html</ref>
