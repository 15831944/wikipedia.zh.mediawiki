'''爱丽丝伪娘团'''是中国[[湖北省|湖北省]][[武汉市|武汉市]]的一个[[伪娘|伪娘]][[cosplay|cosplay]]团体，成立于2009年10月1日。成立之时的成员是在武汉就读的大学生，后来团体规模逐渐扩大至约200人，大部分是[[中南民族大学|中南民族大学]]、[[武汉大学|武汉大学]]、[[武汉理工大学|武汉理工大学]]、[[武汉传媒学院|武汉传媒学院]]、[[湖北广播电视大学|湖北广播电视大学]]等[[中华人民共和国高等教育|高等学校]]的学生，也有少部分从事其他职业的成员。团体分为后勤、舞蹈、化妆、摄影、声援等七个部门，其中进行表演的是舞蹈部门，由约十位核心成员组成。由于现实生活和外界评价的缘故，有部分原舞蹈部成员退出爱丽丝伪娘团，使得团体成员偶尔有变动<ref>{{cite web |title=大学制造“伪娘”凶猛 称比女人魅力还强(图) |url=https://news.qq.com/a/20101015/001030.htm |website=腾讯新闻 |publisher=汉网-长江日报 |date=2010-10-15 |access-date=2019-02-13 |archive-url=https://web.archive.org/web/20101018171924/http://news.qq.com/a/20101015/001030.htm |archive-date=2010-10-18 |dead-url=no }}</ref><ref>{{cite web |title=伪娘胸猛 武大“爱丽丝伪娘团”风靡网络 |url=http://www.99danji.com/news/3803/ |website=99单机资讯 |date=2011-04-12 |access-date=2019-02-13 |archive-url=https://web.archive.org/web/20120522103546/http://www.99danji.com/news/3803/ |archive-date=2012-05-22 |dead-url=no }}</ref>。

== 发展历程 ==
爱丽丝伪娘团的前身是武汉一个名为「'''囧客帝国'''」的高等学校cosplay社团，社团中既有男性成员也有女性成员。在2009年10月1日的一次上台表演中，本应该上台表演的女性成员均因故缺席，而为了这次上台表演，之前社团的团员进行了很多次排练，故社团的男性成员认为不应该取消此次表演，便男扮女装以伪娘的形式上台表演。这次表演获得不错的反响，所以他们决定以此为契机成立一个由伪娘组成的cosplay表演团体，当时有9位成员<ref>{{cite web |author1=萤火虫动漫游戏嘉年华 |title=【萤火虫SH01】爱丽丝伪娘团官方采访 |url=https://www.bilibili.com/video/av6612482/ |website=Bilibili |date=2016-10-09 |access-date=2019-02-13 |archive-url=https://web.archive.org/web/20171120180749/http://www.bilibili.com/video/av6612482/ |archive-date=2017-11-20 |dead-url=no }}</ref>。

由于当时伪娘这一主题争议性较大，个别成员因为受到家庭的反对或者是社会的不公正评价影响而退出爱丽丝伪娘团，这使得上台表演的核心成员偶尔有变动。如有一位热爱舞蹈表演的武汉大学计算机系的成员在加入爱丽丝伪娘团后被父母得知，受到父母的反对，不得不退出爱丽丝伪娘团<ref name="伪娘团我们是正常男人是爱好但不拒商业赚钱" />。

爱丽丝伪娘团后来开始招募新成员，招募对象主要是高等学校的学生。团长豪歌表示招募要求是年满18岁，热爱[[日本动漫|日本动漫]]等[[ACG|ACG]]文化，能够进行舞蹈表演，且在外貌上要较接近于女性，如身材较苗条。鉴于担忧成员有[[心理变态|心理变态]]行为，豪哥表示团体不接受[[异装癖|异装癖]]和[[戀褲襪|恋丝袜者]]，但基于[[隐私权|隐私权]]的考虑不会对成员的[[性取向|性取向]]进行干预<ref name="伪娘团我们是正常男人是爱好但不拒商业赚钱">{{cite web |title=伪娘团：我们是正常男人 是爱好但不拒商业赚钱 |url=http://news.sohu.com/20120425/n341566168.shtml |website=搜狐新闻 |publisher=解放网-新闻晨报 |date=2012-04-25}}</ref>。经过成员招募后，爱丽丝伪娘团成员数量逐渐增加，现大约有200人<ref>{{cite web |title=让直男都把持不住的男人们!深扒国内第一伪娘团 |url=http://news.163.com/15/1215/12/BASK11PR000155IU.html |website=网易新闻 |publisher=哒哒良品|date=2015-12-15}}</ref>。

== 活动 ==
作为表演团体，爱丽丝伪娘团通过几种不同的方式进行表演活动，包括通过拍摄cosplay的照片后上传到社交网站上，有时以[[写真集|写真集]]的形式进行贩卖；拍摄表演视频上传到[[Bilibili|Bilibili]]等视频网站上；作为嘉宾或参演团队出席[[漫展|漫展]]<ref>{{cite web |title=伪娘团x少女皇帝 《萌王EX》性转风席卷上海萤火虫 |url=https://games.qq.com/a/20161010/026836.htm#p=1 |website=腾讯游戏 |date=2016-10-10}}</ref>；参加选秀节目<ref>{{cite web |title=《中国达人秀》武汉海选 爱丽丝伪娘团成功晋级 |url=http://ent.163.com/11/0405/05/70RRTEV500032DGD.html |website=网易娱乐 |publisher=荆楚网-楚天都市报(武汉) |date=2011-04-05 |access-date=2019-02-13 |archive-url=https://web.archive.org/web/20110408211036/http://ent.163.com/11/0405/05/70RRTEV500032DGD.html |archive-date=2011-04-08 |dead-url=no }}</ref>等。

== 评价 ==
=== 争议 ===
爱丽丝伪娘团所体现的伪娘文化经常饱受争议。有[[心理学家|心理学家]]指出，单纯为了表演和穿女装并没有问题，但是不排除部分可能会患有[[扮异性症|扮异性症]]这一心理疾病，可能会对以后的婚恋家庭造成不良影响。[[社会学家|社会学家]][[冯桂林|冯桂林]]表示，观看爱丽丝伪娘团表演的观众包含身心尚未发育健全的青少年，这部分群里由于人生经历较少，缺乏判断力和自制力，可能会进行盲目模仿，造成不良后果。也有人将伪娘文化定义为低俗文化<ref>{{cite web |author1=毕晓哲 |title=“伪娘团”商演不断是个“恶劣信号” |url=http://opinion.china.com.cn/opinion_95_39095.html |website=中国网 |date=2012-04-10 |access-date=2019-02-13 |archive-url=https://web.archive.org/web/20120624165438/http://opinion.china.com.cn/opinion_95_39095.html |archive-date=2012-06-24 |dead-url=no }}</ref><ref name="比女人还女人武汉高校男生组伪娘团引争议">{{cite web |title=比女人还女人 武汉高校男生组“伪娘团”引争议(图) |url=http://news.163.com/12/0409/13/7ULF6HIR00014AEE.html |website=网易新闻 |publisher=华声在线(长沙) |date=2012-04-09}}</ref>。 

而针对于商业演出，[[中国人事科学研究院|中国人事科学研究院]]院长吴江认为爱丽丝伪娘团存在的根本问题是伪娘文化迎合了畸形的审美市场需求，使得一些高等学校男生希望通过加入伪娘团来获得商业报酬<ref>{{cite web |author1=吴江 |title=究竟谁在消费高校“伪娘团”？ |url=http://opinion.china.com.cn/opinion_96_39196.html |website=中国网 |publisher=中国青年网 |date=2012-04-10 |access-date=2019-02-14 |archive-url=https://web.archive.org/web/20120624172409/http://opinion.china.com.cn/opinion_96_39196.html |archive-date=2012-06-24 |dead-url=no }}</ref>。

=== 正面评价 ===
虽然伪娘文化存在争议，但也有人表示这是当今社会创新意识的表现，是大学自由精神的体现。有人指出在[[京剧|京剧]]中，也有[[男旦|男旦]]这一男性扮演的女性角色，如戏曲表演艺术家[[梅兰芳|梅兰芳]]便是[[京剧四大名旦|京剧四大名旦]]之一<ref name="比女人还女人武汉高校男生组伪娘团引争议" />。也有人倡议不应该对伪娘文化有偏见，应该客观进行评价，不能对伪娘文化进行全局否定，因为爱丽丝伪娘团并不违反相关法律，伪娘文化本身也是艺术创新的体现<ref>{{cite web |title=王垚烽：摘下“道德”有色眼镜看待“伪娘团” |url=http://news.163.com/12/0410/07/7UND5FMS00012Q9L.html |website=网易新闻 |publisher=红网(长沙) |date=2012-04-10}}</ref><ref>{{cite web |title=郭文婧：不要一棍子打死“伪娘”文化 |url=http://news.163.com/12/0416/12/7V7A2A1E00012Q9L.html |website=网易新闻 |publisher=中工网(北京) |date=2012-04-16}}</ref>。

在经济上，《[[西安日报|西安日报]]》评论员李亮认为爱丽丝伪娘团进行商业化可以培养学生的社会实践经验，获得的报酬也可以减轻家庭负担，同时还能促进动漫这一新兴产业的发展<ref>{{cite web |title=众说纷纭“爱丽丝伪娘团” |url=http://news.163.com/12/0410/06/7UN6M7CU00014AED.html |website=网易新闻 |publisher=西安日报|date=2012-04-10}}</ref>。

而[[中国共产主义青年团|中国共产主义青年团]][[武汉理工大学|武汉理工大学]]委员会中管理学校社团的周老师表示，对于学生组办伪娘社团一事，由于高等学校社团的创立有一定的自主性，由学生自主管理而不是由学校直接管理，所以只要符合学校对于社团的要求即可。对于娱乐性社团，可以丰富大学生的课余生活，故会支持<ref>{{cite web |title=武汉高校伪娘团商演惹争议 回应：这是一场Cosplay |url=http://ent.163.com/12/0411/08/7UQ19OJK00032DGD.html |website=网易娱乐 |publisher=潇湘晨报 |date=2012-04-11}}</ref>。
