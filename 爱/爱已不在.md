《'''爱已不在'''》（{{lang-en|"Outside"}}）是苏格兰DJ兼唱片制作人[[卡尔文·哈里斯|卡尔文·哈里斯]]与英国歌手[[埃利·古尔丁|埃利·古尔丁]]合作的一首歌曲，于2014年10月20日作为哈里斯第四张录音室专辑《{{le|音浪狂潮|Motion (Calvin Harris album)}}》的第四首单曲发行。这是哈里斯和古尔丁自2013年单曲《{{le|我要你的爱|I Need Your Love (Calvin Harris song)}}》的再度合作。歌曲也收录于古尔丁第三张录音室专辑《{{le|疯狂快感|Delirium (Ellie Goulding album)}}》豪华版。

== 创作 ==
歌曲以[[D小调|D小调]]写就，每分钟128拍，和弦进程为B♭-Gm-Dm-C<ref>{{cite web|title=Calvin Harris "Outside" Sheet Music|url=http://www.musicnotes.com/sheetmusic/mtd.asp?ppn=MN0143549|website=Music Notes|accessdate=2017-08-04}}</ref>。

== 专业评价 ==
歌曲获音乐评论家非常正面的评价，被称作两人之前合作的杰出续作，是专辑的一大亮点。[[PopMatters|PopMatters]]的布伦特·福克纳（{{lang|en|Brent Faulkner}}）称赞歌曲是“《音浪狂潮》的精髓之一”，写道：“第一次听便能知道它是最强劲的流行乐曲”<ref>{{cite web |last=Faulkner |first=Brent |url=http://www.popmatters.com/review/187920-calvin-harris-motion/ |title=Calvin Harris: Motion |publisher=PopMatters |date=2014-11-18 |accessdate=2014-12-21}}</ref>。《[[娱乐周刊|娱乐周刊]]》的凯尔·安德森（{{lang|en|Kyle Anderson}}）评论称，哈里斯从歌曲中“找到了（古尔丁的）脆弱颤音和歪曲合成器铺垫的完美平衡点”<ref>{{cite web |last=Anderson |first=Kyle |url=http://www.ew.com/ew/article/0,,20868198,00.html |title=Motion Review |work=Entertainment Weekly |publisher=Time Inc. |date=2014-11-19 |accessdate=2014-12-21}}</ref>。{{le|Idolator|Idolator (website)}}的罗比·道（{{lang|en|Robbie Daw}}）认为歌曲“很悦耳，比卡尔文和埃利之前的合作作品，2012年的《我要你的爱》稍微出色点。能听到两位歌手一同掀翻录音棚还能产生如此大的魔力，让人觉得欣慰<ref>{{cite web |last=Daw |first=Robbie |url=http://www.idolator.com/7567094/calvin-harris-ellie-goulding-outside-listen-motion |title=Calvin Harris And Ellie Goulding's "Outside": Listen To Their 'Motion' Track |publisher=Idolator. Spin Media |date=2014-10-20 |accessdate=2014-12-22}}</ref>。”

{{lang|en|Music Times}}的卡罗琳·梅耶斯（{{lang|en|Carolyn Menyes}}）认为：“哈里斯给合成器节奏调弦的效果和古尔丁的低沉嗓音配合得完美。我们之前在《我要你的爱》中已经听到过了，这次它全力回归，虽然呈现出另一种魔幻般的合体<ref>{{cite web |last=Menyes |first=Carolyn |url=http://www.musictimes.com/articles/12849/20141020/calvin-harris-ellie-goulding-outside-review-listen.htm |title=Calvin Harris, Ellie Goulding "Outside" Review: "I Need Your Love" Duo Reunite for Spacey Breakup Anthem [LISTEN] |publisher=Music Times |date=2014-10-20 |accessdate=2014-12-22}}</ref>。”[[Allmusic|Allmusic]]的希瑟·费尔斯（{{lang|en|Heather Phares}}）认为曲子“没能很好地夺回（哈里斯和古尔丁）之前作品中的莫利，但确实充分运用了她这把很能骗人的强大女高音，这一次还是处在弱拍环境中<ref>{{cite web |last=Phares |first=Heather |url=http://www.allmusic.com/album/motion-mw0002759398 |title=Motion – Calvin Harris |publisher=AllMusic. All Media Network |accessdate=2014-12-21}}</ref>。”《[[偏锋杂志|偏锋杂志]]》的詹姆斯·雷尼斯（{{lang|en|James Rainis}}）认为《爱已不在》最显著的特征是，钩符的韵律与[[卡莉·蕾·杰普森|卡莉·蕾·杰普森]]和[[猫头鹰城市|猫头鹰城市]]的歌曲《[[美好时光_(猫头鹰城市与卡莉蕾歌曲)|美好时光]]》几乎如出一辙，但缺乏《美好时光》的娇媚魅力<ref>{{cite web |last=Rainis |first=James |url=http://www.slantmagazine.com/music/review/calvin-harris-motion |title=Calvin Harris: Motion |publisher=Slant Magazine |date=2014-11-04 |accessdate=2014-12-21}}</ref>。”{{le|AXS TV|AXS TV}}的蔡斯·亨特（{{lang|en|Chase Hunt}}）认为歌曲是“一次正经的搭配，但结果预想到”<ref>{{cite web |last=Hunt |first=Chase |url=http://www.axs.com/review-calvin-harris-motion-a-solid-set-but-played-safe-28549 |title=Review: Calvin Harris 'Motion' a solid set but played safe |publisher=AXS TV |date=2014-11-16 |accessdate=2014-12-22}}</ref>。

== 商业表现 ==
歌曲空降[[英国单曲排行榜|英国单曲排行榜]]第6位，首周售出28902份<ref>{{cite web |last=Jones |first=Alan |url=http://www.musicweek.com/businessanalysis/read/official-charts-analysis-ed-sheeran-pips-calvin-harris-to-no-1-on-albums/060086 |title=Official Charts analysis: Ed Sheeran pips Calvin Harris to No.1 on albums |work=Music Week |publisher=Intent Media |date=2014-11-10 |accessdate=2014-12-21}} {{subscription required}}</ref>，次周保持第6位，销量为35590份<ref>{{cite web |last=Jones |first=Alan |url=http://www.musicweek.com/businessanalysis/read/official-charts-analysis-wake-me-up-returns-to-singles-summit-as-children-in-need-charity-track/060127 |title=Official Charts analysis: Wake Me Up returns to singles summit as Children In Need charity track |work=Music Week |publisher=Intent Media |date=2014-11-17 |accessdate=2014-12-21}} {{subscription required}}</ref>。第三周，歌曲跌至第10位，销量为33208份<ref>{{cite web |last=Jones |first=Alan |url=http://www.musicweek.com/businessanalysis/read/official-charts-analysis-1d-land-fourth-consecutive-no-1-lp-as-four-sells-141-780/060190 |title=Official Charts analysis: 1D land fourth consecutive No.1 LP as Four sells 141,780 |work=Music Week |publisher=Intent Media |date=2014-11-24 |accessdate=2014-12-21}} {{subscription required}}</ref>。

在美国，截至2015年3月7日周末，歌曲成为哈里斯第八首《公告牌》{{le|告示牌舞曲/混音電台榜|Dance/Mix Show Airplay|舞曲/混音电台榜}}冠军单曲，追平[[大卫·库塔|大卫·库塔]]和[[麦当娜|麦当娜]]，同时这也是古尔丁在该榜的第三首冠军单曲<ref>{{cite web|url=http://www.billboard.com/articles/columns/chart-beat/6480337/chart-highlights-madonnas-living-for-love-debuts-on-pop-songs|title=Chart Highlights: Madonna's 'Living for Love' Debuts on Pop Songs, Leads Dance Club Songs|work=Billboard|publisher=Nielsen Business Media, Inc |date=2015-02-23|accessdate=2015-02-23}}</ref>。在[[公告牌百强单曲榜|公告牌百强单曲榜]]上，歌曲最高排名第29位，成为哈里斯第八首、埃尔丁第五首前四十强单曲。

Ellie Goulding rocks skintight pants and layered jumper as she arrives back home after filming new video with Calvin Harris |work=Daily Mail |publisher=DMG Media |date=2014-10-23 |accessdate=2014-12-21}}</ref>。视频于2014年11月12日首播<ref>{{cite web |last=Corner |first=Lewis |url=http://www.digitalspy.co.uk/music/news/a609517/calvin-harris-ellie-goulding-premiere-outside-music-video.html |title=Calvin Harris, Ellie Goulding premiere 'Outside' music video |publisher=Digital Spy |date=2014-11-12 |accessdate=2014-12-21}}</ref>。影片刻画了两对关系混乱、时常恶语相向的夫妇。哈里斯和古尔丁分别扮演了两对夫妇中的一方，似乎都遭受虐待。视频穿插了古尔丁在郊区房屋前面对镜头唱歌的镜头，超现实的画面扭曲了前景和背景，所有物体都被定格，古尔丁悬在半空中，浮空的矩形镜子最终破碎。视频结尾处，古尔丁和哈里斯出现在房子中，所有东西都定格了，让观众猜测哪个版本才是现实的<ref>{{cite web |last=Gottlieb |first=Steven |url=http://www.videostatic.com/watch-it/2014/11/12/calvin-harris-outside-emil-nava-dir |title=Calvin Harris f/ Ellie Goulding "Outside" (Emil Nava, dir.) |publisher=VideoStatic |date=2014-11-12 |accessdate=2014-12-21}}</ref>。
