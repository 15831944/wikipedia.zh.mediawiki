'''爱瑅币'''（{{lang-en|'''ATB Coin'''}}）是一种数字加密货币的电子支付系统，以隔离见证、闪电网络、智能合约等为技术基础，以[[区块链|区块链]]为技术核心，可用于防护核算、以爱瑅币为单位的物品价值的储存和转账，同时也可以用作对准备接受该种加密货币的商家的商品以及服务的电子支付工具<ref name="加密货币ATBCoin将开启中国区ICO计划">{{cite web | url=http://stock.sohu.com/20170711/n500732793.shtml | title=加密货币ATBCoin将开启中国区ICO计划 | publisher=搜狐证券 }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="AiTiBi Coin投资者突破1000人">{{cite web | url=http://finance.huanqiu.com/roll/2017-07/10945077.html?qq-pf-to=pcqq.c2c | title=AiTiBi Coin投资者突破1000人|publisher=环球网}}</ref><ref name="ATB Coin Cryptocurrency ICO Now Underway Across Globe">{{cite web | url=http://www.newsbtc.com/2017/06/14/atb-coin-cryptocurrency-ico-now-underway-across-globe/ | title=ATB Coin Cryptocurrency ICO Now Underway Across Globe|publisher=NewsBTC}}</ref><ref name="AiTiBiCoin Press Conference - Thursday, June 8">{{cite web | url=https://www.benzinga.com/pressreleases/17/06/p9571748/aitibicoin-press-conference-thursday-june-8 | title=AiTiBiCoin Press Conference - Thursday, June 8|publisher=Benzinga}}</ref>。2017年6月12日，爱瑅币进行了[[ICO|ICO]],最大发行量为3亿3千万个<ref name="ATB Coin Cryptocurrency ICO Now Underway Across Globe">{{cite web | url=http://www.newsbtc.com/2017/06/14/atb-coin-cryptocurrency-ico-now-underway-across-globe/ | title=ATB Coin Cryptocurrency ICO Now Underway Across Globe|publisher=NewsBTC}}</ref>。

==功能介绍==

以[[区块链|区块链]]为技术核心的爱瑅币平台由赫伯特·W·胡佛三世等人联合开发<ref name="AiTiBiCoin Press Conference - Thursday, June 8">{{cite web | url=https://www.benzinga.com/pressreleases/17/06/p9571748/aitibicoin-press-conference-thursday-june-8 | title=AiTiBiCoin Press Conference - Thursday, June 8|publisher=Benzinga}}</ref>。爱瑅币采用开放的区块链、避雷网、即时操作<ref name="ATB Coin">{{cite web | url=http://www.newsbtc.com/bitcoin_businesses/atb-coin/ | title=ATB Coin|publisher=Newsbtc}}</ref>等代替式科技数据，以用于加密货币自由市场中进行交易<ref name="AiTiBi Coin投资者突破1000人">{{cite web | url=http://finance.huanqiu.com/roll/2017-07/10945077.html?qq-pf-to=pcqq.c2c | title=AiTiBi Coin投资者突破1000人|publisher=环球网}}</ref>。

==操作原则==

爱瑅币采用PoS协议<ref name="ATB Coin">{{cite web | url=http://www.newsbtc.com/bitcoin_businesses/atb-coin/ | title=ATB Coin|publisher=Newsbtc}}</ref>，其初衷是为了寻找PoW协议的缺陷并对其进行完善。
