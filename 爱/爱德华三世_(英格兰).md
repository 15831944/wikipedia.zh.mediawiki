'''愛德華三世'''（{{lang-en|Edward III}}，{{bd|1312年|11月13日|1377年|6月21日|catIdx=E}}），[[英格蘭|英格蘭]]國王，1327年到1377年在位。

== 早年 ==

爱德华三世是被谋杀的[[爱德华二世|爱德华二世]]的儿子，生于[[伯克郡|伯克郡]][[温莎_(伯克郡)|温莎]]。其母[[伊莎贝拉_(法兰西)|法兰西的伊莎贝拉]]與[[馬奇伯爵|馬奇伯爵]][[羅傑·莫蒂默，第一代馬奇伯爵|罗杰·莫蒂默]]合谋废黜爱德华二世后成为英格蘭的实际统治者。马奇伯爵获得大量封地（包括著名的[[登比封地|登比封地]]）并在英格蘭飞扬跋扈。

父王遭废黜後，爱德华三世由[[坎特伯雷大主教|坎特伯雷大主教]][[沃尔特·雷诺兹|沃尔特·雷诺兹]]（此人是爱德华二世的宠臣之一）加冕为英格兰国王。实权被摄政的太后和其情夫马奇伯爵掌握。然而，爱德华三世很快聚集起实力惩罚杀死他父亲的人。1330年，爱德华三世亲政，立刻下令处死马奇伯爵并没收其财产。

== 愛德華三世与[[百年战争|百年战争]] ==

[[File:Traité_de_Bretigny.jpg|thumb]]
1336年，为报复[[法国|法国]]国王[[腓力六世|腓力六世]]侵犯在[[佛兰德|佛兰德]]的英国商人，爱德华三世曾下令禁止向法国出口羊毛。由于对腓力六世继承法蘭西王位不满（爱德华是法国[[卡佩王朝|卡佩王朝]]最后一代国王[[查理四世_(法兰西)|查理四世]]的外甥），爱德华三世对法国怀有越来越大的敌意。1337年，爱德华三世正式自称为法蘭西國王，而且此后直到1801年，每个英國君主也都自称是法蘭西國王。他旋即联合[[神圣罗马帝国|神圣罗马帝国]]皇帝[[路易四世_(神圣罗马帝国)|路易四世]]，拼凑一支反法力量。当1338年腓力六世宣布没收他在法国的一切领地时，爱德华的怒气爆发了。1339年、1340年他从法国北部发动了两次进攻但皆失败；1340年6月，英格蘭舰队在[[斯鲁伊斯海战|斯鲁伊斯海战]]中大获全胜。但军费开支过于巨大使爱德华三世无力持续作战。1344年他甚至宣布[[破产|破产]]，导致[[佛罗伦萨|佛罗伦萨]]的两家大[[银行|银行]]倒闭。

1346年，爱德华三世携长子[[黑太子爱德华|黑太子爱德华]]卷土重来，在[[克雷西战役|克雷西战役]]中大败法军。这次战斗是杰出的将领黑太子爱德华初次亮相，也是在此战中，[[欧洲|欧洲人]]第一次使用了[[火器|火器]]。到1347年，爱德华三世已攻占法国重镇[[加来|加来]]，并准备进军[[巴黎|巴黎]]加冕。1348年，他拒绝了竞选神圣罗马帝国皇帝的邀请。

1356年英军在[[普瓦捷战役|普瓦捷战役]]中再度取得大胜，并俘获了法蘭西國王[[约翰二世_(法兰西)|约翰二世]]。爱德华三世即草擬[[倫敦條約|倫敦條約]]逼約翰二世接受，但條文太過苛刻，受各方法蘭西贵族反對。

1359年爱德华三世再次入侵法国，围攻[[兰斯|兰斯]]，而后更推进到巴黎；在那里，他与法蘭西王國摄政[[查理五世_(法兰西)|王太子查理]]（后为国王'''查理五世'''）签署了[[布勒丁尼條约|布勒丁尼條约]]，控制了法国北方的大片领土，但承诺放弃法蘭西王位。1360年，他在法蘭西封黑太子爱德华为[[阿基坦|阿基坦]]公爵。

1364年查理五世正式即位后，拒绝承认加来条约，爱德华就又开始使用法蘭西国王的称号。但查理五世是一个难以对付的对手，他采取了不少有力度的改革措施。同时，在法蘭西出现了[[貝特朗·杜·蓋克蘭|迪·盖克兰]]这样的优秀将领。总之，爱德华三世的战争计划从此就开始失利。他的扩张活动到此为止了。

== 其他政绩 ==
1348年，爱德华三世统治之下的英国遭受了有史以来最大的灾难之一——[[黑死病|黑死病]]。他颁布了一些法令来强迫人们继续劳动。1353年，议会通过法案，禁止民众向教会法庭上诉及向[[教皇|教皇]]交纳[[什一税|什一税]]。大约在1348年，爱德华三世创立了[[嘉德勋章|嘉德骑士团]]。

爱德华三世在位时的另一个主要的敌对目标是[[苏格兰|苏格兰]]。早在1333年，愛德華三世击退过入侵英格兰的苏格兰勢力。1356年他又曾进入苏格兰南部。

晚年受到情妇[[佩雷斯夫人|佩雷斯夫人]]的控制。也是在这时，被认为是[[异端|异端]]的[[罗拉德派|罗拉德派]]宗教改革运动盛行起来。
