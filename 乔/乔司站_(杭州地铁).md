== 结构 ==
乔司站为高架三层侧式站台，站房总面积12740平方米，其中地下室3662平方米，主体建筑8188平方米（含商业部分1066平方米），天桥1190.5平方米，车站总长127米，标准段宽度14米。<ref>{{cite web |title=杭州地铁1号线乔司南站、乔司站站房工程简介 |url=http://zzhzbbs.zjol.com.cn/data/attachment/forum/201201/09/09163063b66bghee0hllp3.jpg}}</ref>

== 轶事 ==
*2013年夏季，乔司站等高架站存在站台玻璃大量碎裂的情况，其中穹顶玻璃碎裂较多，三个车站加起来共有32块，侧面挡风玻璃碎了14块，<ref>{{cite web |title=地铁乔司南站乔司站翁梅站 站台玻璃大面积碎裂-浙江新闻-浙江在线 |url=http://zjnews.zjol.com.cn/system/2013/10/30/019675003.shtml |website=浙江在线}}</ref>为解决此问题，地铁公司将穹顶玻璃全部更换为不透明的铝合金板<ref>{{cite web |title=浙江地铁1号线乔司高架站钢雨棚幕墙改造工程招标公告 |url=https://zj.bidcenter.com.cn/diqurili-15078291-1.html |website=浙江招标网}}</ref>，使得白天的站台变得较为昏暗。

*2013年夏季，由于乔司附近大规模停电，导致大量纳凉人群涌入车站。地铁方面采取了用铁马和警戒条隔出特定区域的方式，接纳了这批“纳凉族”。<ref>{{cite web |title=杭州地铁1号线，乔司站再次遭遇纳凉大军-浙江新闻-浙江在线 |url=http://zjnews.zjol.com.cn/05zjnews/system/2013/07/26/019492995.shtml |website=浙江在线}}</ref>

*2015年6月27日，由于宜家杭州商场开业后首个双休日，客流量激增导致乔司站瘫痪，当日13：25左右到14：30，乔司站双向列车均不停靠。当日乔司站单站单日进站的客流量11800多人次，出站客流量是14600多人次，为历史最高。<ref>{{cite web |title=想进杭州宜家要排队一小时 地铁乔司站一度过站不停-浙商网-浙江在线 |url=http://biz.zjol.com.cn/system/2015/06/28/020715011.shtml |website=浙江在线}}</ref>
