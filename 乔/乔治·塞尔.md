'''乔治·塞尔'''（'''György Széll'''，{{bd|1897年|6月7日|1970年|7月30日|catIdx=Szell, Gyorgy}}）是一位[[美国|美国]]籍的[[匈牙利|匈牙利]][[指挥家|指挥家]]。

1897年塞尔生于[[布达佩斯|布达佩斯]]。很早就开始了其音乐生涯，在1908年，11岁的他就以[[钢琴家|钢琴家]]的身份登台演出。后来他在[[维也纳|维也纳]]学习音乐作曲。

1970年塞尔在[[克利夫兰_(俄亥俄州)|克利夫兰]]去世。

== 职业生涯 ==
* 1930年 [[布拉格|布拉格]]德国剧院歌剧指挥。
* 1940年 指挥[[纽约爱乐|纽约爱乐]]。
* 1946年－1970年 [[克利夫兰管弦乐团|克利夫兰管弦乐团]]音乐指导。他把该乐团带进了[[美国五大管弦乐团|美国五大管弦乐团]]的行列。

== 风格 ==
塞尔的指挥雷厉，线条清晰。有人认为，他的录音是[[托斯卡尼尼|托斯卡尼尼]]的立体声版本。

== 代表录音 ==
* [[勃拉姆斯|勃拉姆斯]]第一钢琴协奏曲 [[柯曾|柯曾]]钢琴
* [[少年魔术号角|少年魔术号角]]
