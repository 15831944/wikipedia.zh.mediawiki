[[喬峰|喬峰]]，得知自己為契丹人後改名'''蕭峰'''，是[[金庸|金庸]]的[[武侠小说|武侠小说]]《[[天龙八部_(小说)|天龙八部]]》中三位主角之一（另兩位主角為[[段譽|段譽]]和[[虛竹|虛竹]]）。為金庸小說中武功最絕頂的高手之一。也是金庸作品中唯一一位甫一出場便名動江湖武藝頂天的主角。

蕭峰身材魁偉，濃眉大眼，高鼻闊口，一張四方的國字臉，頗有風霜之色，顧盼之際，極有威勢，為人正派、重義、豪邁、直爽。在江湖上素有「[[北乔峰，南慕容|北乔峰，南慕容]]」之稱，全書中蕭峰與人單挑對打未曾敗過。

== 生平 ==
喬峰生父是[[蕭遠山|蕭遠山]]，由[[喬三槐|喬三槐]]收养，曾有未婚妻子[[阿朱|阿朱]]，并有義弟[[虛竹|虛竹]]、[[段譽|段譽]]，義兄[[耶律洪基|耶律洪基]](後來一刀二斷)，完顏阿骨打。

萧姓为契丹-{后}-族（太后萧氏一族），萧氏父子是契丹贵族。喬峰九月[[重阳节|重阳节]]左右出生。然而，蕭峰生母是漢人，所以也承其血統。

喬峰自幼聰明伶利，武功極高；江湖上與[[姑蘇慕容|姑蘇慕容]]氏家主[[慕容復|慕容復]]齊名，「北喬峰，南慕容」只要是武林人士盡皆朗朗上口。

七岁时，喬峰在受委屈無處可申時，他选择了自力救济，狙杀了势利的[[大夫|大夫]]，以平息其心中的怒火。

師從[[少林寺|少林寺]][[玄苦|玄苦]]大師，後加入[[丐幫|丐幫]]，[[丐幫|丐幫]]前代幫主「劍髯」[[汪劍通|汪劍通]]因為深知喬峰是契丹後裔，所以十分提防，並給予喬峰「三大難題」、「七大功勞」的考驗。

後來喬峰立下很多汗馬功勞，仁義兼備，并于那一年[[泰山|泰山]]大会连创九名强敌，为丐帮解困，獲汪劍通推選為新幫主，期间掌管丐帮八年，一直率领丐帮抗击[[契丹|契丹]]，其間吒叱風雲，是領導武林羣雄的泰山北斗。

常年在[[中原|中原]][[北方|北方]]活動，以幫助[[北宋|北宋]]抗擊外敵為己任，表現出許多非凡的氣概和事迹，是極其高大的英雄形象。

在[[無錫|無錫]]城[[松鶴樓|松鶴樓]]中與[[大理|大理]][[皇子|皇子]][[段譽|段譽]]志趣相投，義結金蘭。

在[[杏子林|杏子林]]丐幫大會中得知自己身世是[[契丹|契丹]]後裔而黯然離去，也在途中結識[[姑蘇慕容|姑蘇慕容]]氏二婢之一「聽香水榭」主人[[阿朱|阿朱]]。

後來因養父養母和師父被「大惡人」殺害而被冤枉為兇手。

去[[少林寺|少林寺]]探望師父時，發現師父已死，躲在[[菩提院|菩提院]]佛像後時，被[[少林派|少林派]]掌門人[[玄慈|玄慈]]方丈、[[達摩院|達摩院]]首座[[玄難|玄難]]和[[戒律院|戒律院]]首座[[玄寂|玄寂]]發現，遭到三高僧圍攻，[[玄慈|玄慈]]使出[[少林七十二绝技|少林七十二绝技]]之一「[[大金剛掌|大金剛掌]]」撃來時，喬峰無意間將阿朱連累，受了重傷。

為了求「閻王敵」[[薛慕華|薛慕華]]神醫醫治阿朱，不顧[[聚賢莊|聚賢莊]]群豪正準備討伐他，毅然孤身拜莊，最後被迫於聚賢莊上以一敵百，喬峰雖曾立誓不殺漢人，但不得不大開殺戒，面對三百餘人，只得拼命相摶，自己亦為護阿朱而受傷，幸獲黑衣大漢「恩公」救回。

傷癒後上[[雁門關|雁門關]]觀看生父留下字跡，卻發現字跡已被抹去，而阿朱居然守候於此五日五夜！受阿朱情意深深感動的喬峰，與阿朱攜手下山，相互愛戀。

途中由原名「喬峰」恢復契丹名「蕭峰」，之後努力追查殺害養父母及師父兇手，卻因丐幫副幫主[[馬大元|馬大元]]夫人[[康敏|康敏]]的誤導，以為[[段正淳_(天龙八部)|段正淳]]是「帶頭大哥」，阿朱為救父不惜[[易容|易容]]改装， 喬峰因而錯手殺死阿朱。喬峰應允阿朱遺願--照顧阿朱之妹，自小受星宿老怪[[丁春秋|丁春秋]]教導而頑劣不堪的[[阿紫|阿紫]]。

自此流落大[[遼|遼]]，與[[契丹|契丹]]皇帝[[耶律洪基|耶律洪基]]結為兄弟。

後因消彌遼國政變，蕭峰以一敵百扺抗千兵萬馬，箭殺遼國南院大王「楚王」[[耶律涅魯古|耶律涅魯古]]，擒拿遼國天下兵馬大元帥[[耶律重元|耶律重元]]，及救回耶律洪基一命，蕭峰獲封為遼國南院大王，居於[[佑聖宮|佑聖宮]]，多個月以來忙於處理軍務。

蕭峰得知阿紫失蹤後，四處查探，查到阿紫在[[少林寺|少林寺]]後，便立刻率領「[[燕雲十八騎|燕雲十八騎]]」直闖少林寺武林大會，使出「[[降龍十八掌|降龍十八掌]]」中的「亢龍有悔」擊開[[星宿派|星宿派]]創派師祖「星宿老怪」[[丁春秋|丁春秋]]而救到[[阿紫|阿紫]]，並從[[玄慈|玄慈]]方丈口中得知他就是「帶頭大哥」，蕭峰遭受「南慕容」[[慕容復|慕容復]]、「星宿老怪」丁春秋和[[丐幫|丐幫]]下代幫主[[游坦之|莊聚賢]]圍攻，蕭峰無惧三高手，掌击丁春秋，斜劈慕容复，拳打莊聚賢，三招擊退三大絕頂高手，但自知久戰必定不敵，幸得兩義弟[[虛竹|虛竹]]和[[段譽|段譽]]引走[[丁春秋|丁春秋]]和[[慕容復|慕容復]]，蕭峰和[[游坦之|莊聚賢]]单打独斗，立时便大占上风，只是蕭峰的「[[降龍十八掌|降龍十八掌]]」和他的「[[冰蚕毒掌|冰蚕毒掌]]」硬拚数掌時，都不禁寒气袭体，蕭峰乘[[游坦之|莊聚賢]]举掌全力相迎之际，倏地横扫一腿，两支小腿胫骨同时折断，便即摔倒。

搗破[[全冠清|全冠清]]的陰謀，並查明自己身世，原來「大惡人」和「恩公」都是生父[[蕭遠山|蕭遠山]] (當年雁門關一役亂報謊言之人為[[慕容博|慕容博]]、帶頭大哥為[[玄慈|玄慈]])。

後來[[耶律洪基|耶律洪基]]欲迫喬峰攻打大宋，喬峰不從，且脅迫耶律洪基下令，終生不許遼軍一兵一卒越過宋遼疆界後，自盡於[[雁門關|雁門關]]前，死後屍體由阿紫抱起一同摔入当年父亲自尽时的關外深谷中。

== 性格 ==
喬峰（1062年-1095年）（根据新修版段誉的生日推断）。

而據史料記載，耶律洪基出生於1032年，遼道宗耶律洪基比喬峰年长13歲（修訂版）或8歲（新修版），所以喬峰应该是1045或1040年出生。

喬峰外表看似粗豪，但其實心思極為缜密，善於觀察細節，做事三思而後行。

為人重情重義，義薄雲天，恩怨分明，英氣逼人，豪氣勃發，亦心存慈悲仁心。

喜結交朋友，金蘭之交包括[[虛竹|虛竹]]、[[段譽|段譽]]、[[耶律洪基|耶律洪基]]、[[完顏阿骨打|完顏阿骨打]]等人。

喬峰亦嗜酒，千杯不醉，是以其酒量之高，與段譽之「[[六脈神劍|六脈神劍]]」鬥酒之處可見得到。

== 感情 ==
喬峰為一代豪傑，「兒女情短，英雄氣長」，在江湖游走之際雖然頗享盛名，但是卻無半位紅粉知己。

喬峰與姑蘇慕容婢女[[阿朱|阿朱]]首次於[[無錫|無錫]]杏子林中相見，而後在少林寺中，喬峰因無意造成阿朱中了[[玄慈|玄慈]]方丈的「[[大金剛掌|大金剛掌]]」，千方百計以內力醫治阿朱，甚至不視武林群豪敵視前往聚賢莊求醫。

在莊中大開殺戒後，阿朱被薛神醫救回，並且趕路至雁門關外等候喬峰五天五夜！兩人爾後相處逐漸對彼此傾心，約定在喬峰報仇後兩人雙赴關外不問世事、專牧牛羊、與世無爭。

不料喬峰誤將[[段正淳_(天龙八部)|段正淳]]視為大仇人，阿朱為救父，更為情郎(當慕容復在少室山上被段譽以六脈神劍輕易打得全無還手之力，喬峰突然醒悟阿朱所以殉身不止因為救父，更因為害怕若自己殺掉段氏人物，段氏高手以六脈神劍尋仇時喬峰萬難抵擋)，甘願假扮段正淳讓喬峰打死，悲嘆「塞上牛羊空許約」！

喬峰應阿朱臨死之願照顧阿紫，阿紫隨著相處時間越久亦暗戀喬峰，然而喬峰始終深愛阿朱，對於阿紫只有兄妹之情。

== 身世 ==
喬峰原為[[大遼|遼]]人[[蕭遠山|蕭遠山]]與其妻子的獨子，在喬峰出生[[滿月|滿月]]時，举家前往娘家探望，行經[[雁門關|雁門關]]外时遭到一群[[宋|大宋]][[武林高手|武林高手]]襲擊，喬峰之母遭殺害，父親憤怒傷心之際抱著妻子屍體跳入深谷中，而喬峰則被當時的漢人首領送至一戶農家[[喬三槐|喬三槐]]與其妻子撫養。

喬峰成年後成為[[丐幫|丐幫]]幫主，因為被副幫主夫人[[康敏|康敏]]陷害，得知了自己為契丹人的真相。而後傾全力查詢自己的真正身世，不料當年在場的人士一一遭到殺害，而後喬峰得知自己生父名為[[蕭遠山|蕭遠山]]，遂復姓改名為「蕭峰」。

在少室山上，喬峰重逢了生父[[蕭遠山|蕭遠山]]，不過父親最後在少林派的無名[[掃地僧|掃地僧]]的點化下，皈依佛門，從此父子倆便不再見面。

後來喬峰與大遼皇帝[[耶律洪基|耶律洪基]]結拜為兄弟，耶律洪基曾提及比他大13歲，所以其出生年應該在1045年。

== 武功 ==
小說中並無提及[[喬峰|喬峰]]懂得那幾種武功，但至少應是集[[少林|少林]]、[[丐幫|丐幫]]兩家武功之長；絕學為[[丐幫|丐幫]]代代相傳的「[[降龍十八掌|降龍十八掌]]」、「[[打狗棒法|打狗棒法]]」，还有[[少林|少林]]「龍爪手」、「降魔掌」。書中只有他在聚賢莊一戰因政治民族理由而使「[[太祖長拳|太祖長拳]]」才提及他的招式。根據[[吴霭仪|吴霭仪]]《金庸小說中的男子》一書，金庸说他故意不寫乔峰的招式，以令他與別不同<ref>[http://www.wenyi.com/literature/jinyong/way/nzj5.htm 金庸的男主角——[[喬峰|喬峰]]〔[[吴霭仪|吴霭仪]]《金庸小說中的男子》〕]{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }} </ref>。[[喬峰|喬峰]]與其他金庸小說的第一主角不同，是僅有三名於出場時即身負高強武藝的主角之一，故事發展亦沒有安排他再學會其他武功，擺脫了一般武俠小說的套路，令這個人物明顯與其他小說的主角有所分別。（另二人是《書劍恩仇錄》的陳家洛和《雪山飛狐》的胡一刀。然而陳家洛初登場時雖已身負頗高武功，卻仍有多人處於其上，及不上文泰來、張召重、黃維揚、陸菲青、袁士宵、無塵、趙半山等，並且於後來再習成其他武功，方才晉身最頂級高手之列。而《雪山飛狐》由於篇幅極短，並且胡一刀只在追述往事中登場，情況明顯與蕭峰有所不同。）

金庸先生本人在新三版的《倚天屠龙记》中曾评价乔峰的整体武功水平为“武功盖世”：「丐帮神功降龙十八掌，在北宋年间本为二十八掌，当时帮主喬峰武功盖世，却因[[契丹人|契丹人]]身份遭驱除出帮，他去繁就简，将二十八掌减了十掌，成为降龙十八掌，由义弟灵鹫宫[[虚竹|虚竹子]]代传，由此世代传承。」另外，从书中对乔峰多次對敵的描写，除聚賢莊上孤身力敵三百多名武林高手這樣的極端情勢外，幾乎從不落於下風並鮮有負傷。及至少室山大會時，場上數千中原豪傑中，大半均欲取其性命，卻遲遲未有人敢於上前挑戰，作者說是因：「眾人均知縱使最後定能將蕭峰誅殺，但頭上數十人卻也定要死在他掌下。」這裡所說的「眾人」幾乎已包括了當代武林中所有高手，其中卻無一人認為自己能力敵蕭峰，甚至都認定頭上數十人必死，反差實屬驚人。蕭峰唯一敗仗乃是於藏經閣對戰無名掃地僧，唯他雖不敵無名僧，卻是場上唯一能把對方打傷的人，以致無名僧亦出言讚賞蕭峰武功，而當世武功最高強的人中，當時只有虛竹一人不在場而已。縱觀以上，加上江湖高手口中的“武功第一”（少林五老）、“天下无敌”（萧远山）等等评价，可見蕭峰武功之高，實與同時代的人處於不同的層次上。

[[少林七十二絕技|少林七十二絕技]]之一，出掌輕柔，若有若沒。

; 龍爪手
[[少林七十二絕技|少林七十二絕技]]之一，手指坚硬愈钢，铁指开砖如泥，手如钢爪般抓树撕皮。

; 擒龍功
可以凌空取物，夺人兵刃。

; 排云双掌
'''乔峰'''的得意功夫 掌掌凌厉，霸气十足。

; 太祖長拳
傳說因「[[宋太祖|宋太祖]]」[[趙匡胤|趙匡胤]]而得名，講求實戰攻防，起如風，擊如電，前手領，後手追，兩手互換一氣摧，威力巨大。聚賢莊之戰喬峰曾以「太祖長拳」大敗少林玄難大師的天竺神功。

; 打狗棒法
[[丐幫|丐幫]]鎮幫之寶，是[[丐幫|丐幫]]主代代相傳的武功，輕動靈便，變化精微，招術奇妙，共有絆、劈、纏、戳、挑、引、封、轉八訣，棒影飛舞，神龍夭矯，如流星趕月。

数百年来，[[丐帮|丐帮]]逢到危难关头，帮主亲自出马，往往便仗这打狗棒法除奸杀敌，震慑群邪。

; 降龍廿八掌（降龍十八掌前身）
[[丐幫|丐幫]]鎮幫神功，對應天上[[二十八星宿|二十八星宿]]，講究剛柔並濟，當剛則剛，當柔則柔，轻重刚柔随心所欲，刚劲柔劲混而为一，劲力忽强忽弱，忽吞忽吐，从至刚之中生出至柔，天下阳刚第一，掌法之妙，天下无双，招招须用真力。

说是外门武学中的巅峰绝诣，動作雖似簡單無奇，但掌掌現神龍，招招威力無窮，招式简明而劲力精深的武功，精要之处，全在运劲发力，全凭劲强力猛取胜，当真是无坚不摧、无固不破，虽招数有限，但每出一掌均有龍吟虎嘯之勢、每出一招均具绝大的威力。

而後「降龍廿八掌」經[[喬峰|喬峰]]及其義弟[[虛竹|虛竹]]去繁存菁，成為「[[降龍十八掌|降龍十八掌]]」後，更顯精要且掌力更厚。

新三版《[[射鵰英雄傳|射鵰英雄傳]]》第十二回中，已較二版多寫「'''[[北宋|北宋]]年間，[[丐幫|丐幫]]幫主[[蕭峰|蕭峰]]以此（降龍十八掌）邀鬥天下英雄，極少有人能擋得他三招兩式，氣蓋當世，群豪束手'''」
