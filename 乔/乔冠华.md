'''乔冠华'''（{{bd|1913年|3月28日|1983年|9月22日|乔冠华}}），[[江苏|江苏]]省[[盐城|盐城]]市[[建湖县|建湖县]]庆丰镇东乔村人。曾用笔名'''乔木'''、'''-{于}-怀'''。[[中国|中国]]政治家、外交家，[[清华大学|清华大学]]哲学系毕业，[[德国|德国]][[图宾根大学|图宾根大学]][[哲学博士|哲学博士]]。1939年经[[廖承志|廖承志]]等人介绍，加入[[中国共产党|中国共产党]]，[[中国共产党第十届中央委员会委员列表|中共第十届中央委员]]。曾任[[中华人民共和国外交部|中华人民共和国外交部]]部长、[[中国人民对外友好协会|中国人民对外友好协会]]顾问等职务。

== 早年经历 ==
乔冠华出生在[[江苏|江苏]][[盐城|盐城]]的一个地主家庭，父亲是当地一位比较开明的工商业者。乔冠华幼年时就显示出天资聪颖，有“过目成诵”之誉<ref>[http://tv.people.com.cn/GB/39805/42848/177128/index.html 共和国十任外长] 人民网</ref>。早年，他曾先后在[[盐城|盐城]]第二高等小学、宋村亭湖中学、盐城淮关中学上学。由于学习成绩优异，他多次跳级插班，16岁时就考入[[清华大学|清华大学]]哲学系。

在大学学习期间，乔冠华开始接触[[马克思主义|马克思主义]]，并积极参加了[[中国共产党|中国共产党]]领导的一些活动。1933年乔冠华从[[清华|清华]]毕业后，即赴[[日本|日本]][[东京帝国大学|东京帝国大学]]继续攻读哲学。由于，参加了一些[[日本共产党|日本共产党]]所举行的政治活动，他受到[[日本|日本]]执政当局的驱逐。后辗转到[[德国|德国]]，并于1936年获[[德国|德国]][[图宾根大学|图宾根大学]][[哲学|哲学]][[博士|博士]]学位；时年23岁。

== 外交生涯的开端 ==

[[抗日战争|抗日战争]]爆发后，乔冠华回到中国参与抗战，并主要從事新聞工作。期间，他曾在[[香港|香港]]以“乔木”笔名为《时事晚报》、《世界知识》等刊物撰写国际述评，获得一定影响。1939年，加入[[中国共产党|中国共产党]]。1941年，他曾出任香港《华商报》和《大众生活》编委的职务。1942年秋，乔冠华奉命到达[[重慶|重慶]]，负责主编党刊《群众周刊》；同时在[[周恩来|周恩来]]的直接领导下，在《[[新華日報|新華日報]]》主持《國際專欄》，并开始参加一些有关的外事工作<ref>[http://www.yancheng.gov.cn/ycweb/TemplateA/Default/DefaultInfoDetail.aspx?CategoryID=5cf09e6d-6952-4e2f-9788-9bc7daa0f23f&InfoGuid=ee726bc5-93e5-4047-8a81-9e1c1ca7af59 乔冠华] {{webarchive|url=https://web.archive.org/web/20110518155258/http://www.yancheng.gov.cn/ycweb/TemplateA/Default/DefaultInfoDetail.aspx?CategoryID=5cf09e6d-6952-4e2f-9788-9bc7daa0f23f&InfoGuid=ee726bc5-93e5-4047-8a81-9e1c1ca7af59 |date=2011-05-18 }} [[盐城|盐城]]政府网</ref>。由此，乔冠华逐渐成为[[周恩来|周恩来]]在外交事务上的得力助手之一。

抗战胜利后，乔冠华随[[周恩来|周恩来]]赴[[上海|上海]]，参加中共代表团工作，并创办英文版《新华周刊》。1946年底，他再次回到香港，并出任[[新华社香港分社|新华社香港分社]]的社长。

== 建国后 ==
1949年10月，[[中华人民共和国|中华人民共和国]]成立后，乔冠华历任外交部国际新闻局局长、政策委员会副主任，[[中央人民政府|中央人民政府]]办公厅副主任，[[中国人民外交学会|中国人民外交学会]]副会长，[[外交部亚洲司|外交部亚洲司]]司长、部长助理等职务。1964年3月，任[[外交部|外交部]]副部长。1970年9月20日，夫人龚澎去世。1973年11月，与[[章含之|章含之]]结婚。1974年11月，乔冠华被任命为继[[周恩来|周恩来]]、[[陈毅|陈毅]]、[[姬鹏飞|姬鹏飞]]之后的中华人民共和国的第四任外交部长。

=== 重要外交活动和工作 ===
#1950年10月，陪同中华人民共和国特派代表[[伍修权|伍修权]]出席[[联合国安理会|联合国安理会]]，控诉[[美国|美国]]对[[台湾|台湾]]的武装侵略。
#1951年7月，担任中国代表团团长[[李克农|李克农]]的主要顾问，参加[[板门店|板门店]]朝鲜停战谈判。
#1954年4月，随同[[周恩来|周恩来]]总理出自在[[瑞士|瑞士]]举行的[[日內瓦會議_(1954年)|日内瓦会议]]。<ref>{{cite web |author1=李亚平 |title=周恩来在日内瓦会议上的外交艺术 |url=http://dangshi.people.com.cn/GB/85038/11921000.html |date=2010年06月21日|website=人民网-中国共产党新闻网 | accessdate=2019-01-11}}</ref>
#1961年10月至1962年8月，陪同[[国务院副总理|国务院副总理]]、外交部长[[陈毅|陈毅]]出席第二次日内瓦会议。
#1969年3月，中国和[[苏联|苏联]]发生[[珍宝岛事件|珍宝岛武装冲突事件]]，乔冠华被任命为中苏谈判中国代表团团长。
#1971年11月，「中国在[[联合国|联合国]]的合法席位恢复」后，第一次率中国代表团出席第26届联合国大会并在大会上发表讲话，全面阐述了中国的外交政策。自此至1976年，均以中国代表团团长身份出席历届[[联合国|联合国]]大会。
#1972年2月，美国总统[[尼克松|尼克松]]访华时，负责与[[基辛格|基辛格]]谈判，草拟《中美联合公报》。
#1972年9月，[[日本|日本]]首相[[田中角荣|田中角荣]]访华期间，乔冠华参加了中日谈判和两国联合声明的起草工作。
#1974年11月，出任外交部部长。
#1976年12月，离职接受隔离审查。
#1982年后，担任中国人民对外友好协会顾问。

=== 文革中 ===
在[[文化大革命|文化大革命]]初期，乔冠华受到冲击，与[[陈毅|陈毅]]、[[姬鹏飞|姬鹏飞]]、[[黄华|黄华]]等人一起，被列为[[外交部|外交部]]的“打倒”对象。1973年，经[[毛泽东|毛泽东]]点名，'''乔冠华'''恢复了工作。复出后的乔冠华和“[[四人帮|四人帮]]”走得很近，并且在此后的一系列讲话中，对[[周恩来|周恩来]]的外交路线予以批判。

乔冠华心里一直十分内疚自责，利用一次见外宾的机会，对1973年底发生的对周恩来不公正的批评，当面向周恩来表示当时自己的发言是错误的，对不起总理，请他原谅。周恩来非常宽容地说：“那怎么能怪你呢？那是总的形势，大家都讲了嘛，你在我身边工作几十年，又管美国这一摊，怎么能不讲呢？再说了，我也有失误，也不能说不能批评我，而且这不是你们能左右的事情。”虽然如此，但是仍然有传闻说，[[周恩来|周恩来]]妻子[[邓颖超|邓颖超]]晚年时都一直不能原谅乔冠华的行为<ref>[http://www.dwnews.com/gb/MainNews/Forums/BackStage/2008_1_26_18_13_26_893.html 乔冠华、章含之夫妇：倒向四人帮 向周恩来、邓小平夺权的失败经历] {{webarchive|url=https://archive.is/20130425180437/http://www.dwnews.com/gb/MainNews/Forums/BackStage/2008_1_26_18_13_26_893.html |date=2013-04-25 }} 多维新闻网</ref>。

1976年12月，“[[四人帮|四人帮]]”倒台后不久，乔冠华就被宣布隔离审查。1982年12月，在[[王震|王震]]的过问下，乔冠华重获人生自由，在[[中国人民对外友好协会|中国人民对外友好协会]]挂任「顾问」职衔。1983年9月22日上午10时40分，乔冠华因为[[癌症|癌症]]，在[[北京|北京]]医院逝世，享年70岁。1985年4月5日，乔冠华的骨灰被安葬于[[苏州|苏州]]太湖半岛东山藤湾湖沙村华侨公墓。

== 家庭 ==
1934年，乔冠华在日本东京[[帝国大学|帝国大学]]学习时，与[[日本|日本]]女子Akihito相恋，却因为日本执政当局对乔冠华的驱逐而分离，Akihito而后育有一子，孙女[[乔楚儿|乔楚儿]]。1943年，乔冠华与女外交家[[龚澎|龚澎]]结婚，两人育有一子[[乔宗淮|乔宗淮]]，曾任[[外交部|外交部]]副部长，一女[[乔松都|乔松都]]。1970年9月20日，龚澎病逝。后来在[[毛泽东|毛泽东]]的介绍下，于1973年11月与“北洋遗老”[[章士钊|章士钊]]的养女[[章含之|章含之]]结婚。
