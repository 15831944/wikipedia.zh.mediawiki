'''湛江'''（[[邮政式拼音|邮政式拼音]]：{{lang|la|'''Tsankiang'''}}、{{lang|la|'''Chankiang'''}}、{{lang|la|'''Tsamkong'''}}，[[香港語言學學會粵語拼音方案|粤语拼音]]：zaam<sup>3</sup>gong<sup>1</sup>)，简称'''湛'''，别称'''港城'''，旧称'''[[广州湾|广州湾]]'''（{{lang|la|Kwangchouwan}}），法占时期称'''白雅特城'''（Fort-Bayard），是[[中华人民共和国|中华人民共和国]][[广东省|广东省]]下辖的[[地级市|地级市]]，广东三个[[省域副中心城市|省域副中心城市]]之一（另外两个是[[茂名|茂名]]和[[汕头|汕头]]），是中国大陆除[[海南島|海南島]]外，最南端的土地。位于[[雷州半島|雷州半島]]，广东省西南部，东濒[[南海|南海]]，南隔[[琼州海峡|琼州海峡]]与大特区[[海南省|海南省]]相望，西临[[北部湾|北部湾]]，地处广东、[[广西|广西]]、海南三省区的交汇地带。

湛江历史悠久，文化底蕴深厚。[[徐闻|徐闻]]三墩古港是汉代“[[海上丝绸之路|海上丝绸之路]]”的重要始发港。[[雷州|雷州]]古城是[[国家级历史文化名城|国家级历史文化名城]]。19～20世纪广州湾曾被[[法国|法国]]租借纳入[[法蘭西第二殖民帝國|法兰西第二殖民帝国]]，爱国诗人[[闻一多|闻一多]]为此写下诗篇《[[七子之歌|七子之歌]]·广州湾》。

湛江历来以环境优美而著称，旅游资源得天独厚。早在1959年湛江就获得了“花园城市”的称号<ref>{{cite web |author1=1959年3月7日，全国造林园林化现场会议在湛江召开。湛江获得“花园城市”称号 |title=中国最适合养老20座城市来了，湛江竟排第一 |url=http://www.zhanjiang.gov.cn/fileserver/statichtml/2017-02/6aeba473-fca8-4043-ab6f-c66f5f2bfa6b.htm?cid=ead92330-50d2-48ec-b766-c6b8c59d40b6 |website=中国·湛江政府门户网站 |accessdate=2017-02-19}}</ref>，此后又获得过许多赞誉。例如[[邓小平|邓小平]]曾说“北有[[青岛|青岛]]，南有湛江”，湛江因而得名“南方的青岛”<ref>{{cite web |author1=张弸 |title=关于邓小平同志“北有青岛，南有湛江” |url=http://www.zjds.org/file/5/5/5138.shtml |website=湛江党史网 |accessdate=2013-11-13}}</ref>。中華人民共和國總理[[周恩来|周恩来]]称赞湛江“房子建在树林中，既幽静，又雅致”，像个“小巴黎”<ref>{{cite web |author1=苗体君 |title=周恩来一生在湛江仅有的15个小时 |url=http://dangshi.people.com.cn/GB/18162831.html |website=人民网-中国共产党新闻网 |accessdate=2012-06-13}}</ref>。[[中國人民解放軍|中國人民解放軍]]元帥[[陈毅|陈毅]]赞美湛江“冬犹暖，秋如夏，凉风动，炎氛化”<ref>{{cite web |author1=出自陈毅诗作《满江红·雷州半岛》 |title=雷州青年运河诗集 |url=http://www.zjst.cn/zh/productdetail.aspx?detailid=659 |website=湛江市少年儿童图书馆 |accessdate=2019-01-06}}</ref>，是“东方[[日内瓦|日内瓦]]”。作家[[冰心|冰心]]、[[艾青|艾青]]在湛江分别留下《湛江十日》、《湛江，夹竹桃》等散文佳作。世界第三大联盟邮轮公司——[[丽星邮轮|丽星邮轮]]曾多次抵达湛江。

湛江是中国东南沿海重要的港口城市，是[[北部湾|北部湾]]中心城市之一<ref>{{cite web |title=国务院办公厅关于批准湛江市城市总体规划的通知 |url=http://www.gov.cn/zhengce/content/2017-06/19/content_5203697.htm |website=中华人民共和国中央人民政府 |accessdate=2017-06-19}}</ref>和[[粤西|粤西]]城镇群中心城市<ref>{{cite web |author1=2007年广东省第十次党代会把湛江市确定为粤西城镇群的中心 |title=广东省第十次党代会开幕式 |url=http://newscenter.southcn.com/n/2007-05/21/content_71727157.htm |website=南方网 |accessdate=2019-01-08}}</ref>。湛江是大西南和华南地区的出海通道的物流中心，是中国大陆通往东南亚、中东、欧洲、非洲和大洋洲航程最短的港口城市，是1984年全国首批14个[[沿海开放城市|沿海开放城市]]之一，也是首批“[[一带一路|一带一路]]”海上合作支点城市。<ref>{{cite web |title=湛江概况 |url=http://www.zhanjiang.gov.cn/fileserver/statichtml/2017-03/acb5621c-ee48-4922-84d9-146949fee79d.htm |website=中国·湛江政府门户网站 |accessdate=2018-03-29}}</ref>湛江也是[[中国-东盟自由贸易区|中国-东盟自由贸易区]]产业合作先行先试区，在亚太经济圈中处于重要的地缘战略位置，是“[[十二五|十二五]]”定位的42个全国性综合交通枢纽城市之一<ref>{{cite web |title="十二五"我国需基本建成42个全国性综合交通枢纽 |url=http://www.gov.cn/gzdt/2013-04/01/content_2367171.htm |website=中央政府门户网站 |accessdate=2013-04-01}}</ref>。湛江还是正大军区级中国海军[[南海舰队|南海舰队]]司令部所在地。

2014年，湛江承办首届中国国际海洋经济博览会和中国国际水产博览会。2015年，湛江承办广东省第十四届运动会。<ref>{{cite web | url = http://gd.people.com.cn/GB/123943/123994/9216146.html|title =湛江成功申办2015年广东省第十四届省运会|work =人民网| publisher = | author = | accessdate = 2009-07-03 }}</ref> 湛江的体育事业以跳水著名。

== 历史 ==
4000年前（[[新石器时代|新石器时代]]晚期，约[[夏|夏]]、[[商朝|商]]之间），今湛江辖地就有人类居住。

前214年，岭南新设[[象郡|象郡]]，湛江辖地归属象郡。前111年，南部[[徐闻|徐闻]]、[[海康|海康]]、[[遂溪|遂溪]]三县属[[交州|交趾部]]徐闻县；北部[[吴川|吴川]]、[[廉江|廉江]]两县分属交趾部[[高凉县_(西汉)|高凉县]]和[[合浦县|合浦县]]。

634年，改名「雷州」，统管[[雷州半岛|雷州半岛]]徐闻、海康、遂溪三县。

[[元朝|元朝]]，徐闻、海康、遂溪三县逮属[[湖广行中书省|湖广行中书省]]之[[海北海南道|海北海南道]][[雷州路|雷州路]]；吴川县、石城县则隶属[[高州路|高州路]]。

[[明朝|明朝]]初，两广与江西、湖南分开，雷州逮属广西承宣布政使司。由于广西壮、黎、瑶三个民族反明起事猛烈，明朝皇帝朱元璋采用分而治之的办法，洪武二年（1369年）把黎族聚居的海南及广西门户雷州、钦州、廉州划归广东承宣布政使司，雷州府下辖徐闻、海康、遂溪三县，府治海康县。吴川县、石城县（今廉江）则隶属高州府。

[[清朝|清朝]]，沿袭明朝编制，徐闻、海康、遂溪三县隶属雷州府；吴川县、石城县（今廉江）则隶属高州府。

=== 19世纪末以来 ===
*参见[[广州湾|广州湾]]
1899年11月，[[法国|法国]]与清政府签订《[[广州湾租界条约|中法互订广州湾租界条约]]》，将雷州府遂溪县、高州府吴川县两县所属部分陆地、岛屿以及两县间的麻斜海湾（今[[湛江港|湛江港]]）划为法国[[租界|租界]]，统称“[[广州湾|广州湾]]”，划入[[法属印度支那|法属印度支那]]联邦范围，设广州湾行政总公使署，受法国[[印度支那|印度支那]]总督管辖。

1943年2月，被[[日本|日本]]侵略军所占，並建立親日政權。

1945年8月15日[[日本|日本]]宣布无条件投降，侵占广州湾的日军在寸金桥头挂起“投降”横额。8月26日南京国民政府电令粤桂南区总指挥[[邓龙光|邓龙光]]将军负责受降和接收，并委派[[李月恒|李月恒]]到广州湾筹建湛江市。国民政府军第46军175师徐即开进广州湾，其他各师分赴雷州半岛各地。9月21日上午9时，日军代表[[渡部市藏|渡部市藏]]向[[邓龙光|邓龙光]]将军递交投降书，广州湾光复。

1945年8月18日中法政府在重庆签订《中华民国政府与法国临时政府交接广州湾租借地条约》，宣布法国把1899年11月16日租借的广州湾归还中国。根据这一条约，10月19日中法交接广州湾仪式在原法国驻广州湾公使署举行，法国人统治47年的广州湾正式交回中国。

1946年1月广州湾正式建市，因历史上曾属[[椹川县|椹川县]]，境内东海岛曾设椹川巡检司，广州湾更名为湛江市。

1949年12月18日，解放军攻占湛江。此后，五县先后属广东省南路区行政督察专员公署、高雷区专员公署、粤西区行政公署、湛江专区专员公署、湛江地区行政专员公署。

1983年9月，地区与市合并为湛江市，为[[省辖市|省辖市]]。

== 地理 ==
=== 区位 ===
湛江位于[[中国大陆|中国大陆]]最南端的[[雷州半岛|雷州半岛]]，广东省西南部，东濒[[南海|南海]]，南隔[[琼州海峡|琼州海峡]]与[[海南省|海南省]]相望，西临[[北部湾|北部湾]]，西北与[[广西|广西]]壮族自治区的[[合浦|合浦]]、[[博白|博白]]、[[陆川县|陆川县]]毗邻，东北与广东省[[茂名|茂名]]市属茂南区和[[化州|化州]]、[[电白|电白]]县接壤。湛江纵贯北纬20°15′—21°55′，横跨东经109°40′—110°55′，是广东、[[广西|广西]]、海南三省区的交汇处，是中国西南地区主要出海通道之一<ref>{{cite web|url =http://www.seanet.com.cn/files/geography.htm|title =湛江概况-地理位置|work =湛江水产网|publisher =|author =|accessdate =2009-07-03}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

=== 水系 ===
湛江较大的河流有[[鉴江|鉴江]]、[[南渡河_(雷州半岛)|南渡河]]、[[九洲江|九洲江]]，建有[[鹤地水库|鹤地水库]]、[[长青水库|长青水库]]、[[大水桥水库|大水桥水库]]3座大型[[水库|水库]]及[[雷州青年运河|雷州青年运河]]。

湛江海岸线长达1556公里，约占[[广东省|广东省]]海岸线的2/5和全国的1/10，为广东省之最；拥有148.7万亩海洋[[滩涂|滩涂]]，占全省的48%，也是广东省之最。

=== 气候 ===
[[湛江|湛江]]大部属[[热带季风气候|热带季风气候]]，北部属于[[亚热带季风气候|亚热带季风气候]]，雨季主要集中於5到9月。氣候與[[臺北市|臺北市]]極相似，冬季平均高溫約19-20度，低溫約14度。夏季平均高溫約32-34度，低溫约26度。

=== 语言 ===
湛江是广东省内方言最为复杂的市，主要有'''[[雷州话|黎话]]'''（即[[雷州话|雷州话]]，属[[闽语|闽语]]）、'''湛江白话'''（属[[粤语|粤语]]）以及'''[[𠊎話|𠊎话]]'''（即[[哎話|哎話]]，属[[客家话|客家话]]）。其中说[[雷州话|雷州话]]的人数最多，集中在原雷州府属三县周边县城农村郊区；霞山赤坎两区則以湛江白话为多。近年來[[普通話|普通話]]在湛江也頗為流行。

[[雷州話|雷州話]]（属[[闽语|闽语]]）還细分為：雷城话、徐闻音、海康音、遂溪音、廉江音、郊区音，现在[[雷州話|雷州話]]以雷城话为代表。湛江白话（属[[粤语|粤语]]）也细分为：湛江市区白话、吴川白话、梅菉白话、廉江白话、遂溪白话、龙头塘蓬白话等<ref>{{cite web|url =http://www.edu.cn/20011115/3010037.shtml|title =复杂的湛江地方语言|work =中国教育和科研计算机网|publisher =|author =|accessdate =2009-07-03|archive-url =https://web.archive.org/web/20080907085329/http://www.edu.cn/20011115/3010037.shtml|archive-date =2008-09-07|dead-url =yes}}</ref>。

1984年5月4日，湛江被列为中国首批（共14个）[[对外开放|对外开放]][[中華人民共和國沿海開放城市|港口城市]]之一。同期（[[改革开放|改革开放]]时期），定位于发展“两水一牧”（水产、水果、畜牧业），导致相比于其他开放城市，经济水平一直落后。2005-2010年，工业主导地位增强。第二产业在三次产业中增幅最高、增速最快。经济总量连上台阶，产业结构不断优化。大力实施“工业立市、港口兴市、生态建市”发展战略，着力推动深水良港—重化工业—现代物流—海湾城市“四位一体”协同发展，超预期完成了“十一五”规划的主要指标，经济综合实力显著提升。

现有1个[[国家级经济技术开发区|国家级经济技术开发区]]——[[湛江经济技术开发区|湛江经济技术开发区]]，已被广东省政府认定为“省级高新技术产业开发区”。<ref>{{cite web| url =http://zjphoto.yinsha.com/file/201010/2010101618304937.htm| title =湛江经济技术开发区被省政府认定为省级高新技术产业开发区| work =碧海银沙| publisher =| author =| accessdate =2010-10-17| deadurl =yes| archiveurl =https://web.archive.org/web/20101018074038/http://zjphoto.yinsha.com/file/201010/2010101618304937.htm| archivedate =2010-10-18}}</ref>另外还有5个省级经济开发试验区。

=== 航空 ===
湛江机场位于广东省湛江市西郊，原称西厅机场，距湛江市中心5公里。1952年开始通航，为国家4D级机场，按4C 标准保障，可起降B737-800以下的机型。
湛江国际机场正在规划建设，飞行等级4E，飞行跑道3200米，为广东第三大干线机场，预计2018年全面动工建设，2020年建成。
至2018年2月，湛江机场通航城市共30个，其中国际（地区）通航城市共3个，分别是：柬埔寨金边、泰国曼谷、香港；国内通航城市共27个，包括北京、上海、广州、深圳、南京、长沙、武汉、揭阳（汕头）、成都、重庆、郑州、三亚、杭州、昆明、海口、南昌、厦门、西安、桂林、珠海、贵阳、福州、宁波、大连、沈阳、石家庄、天津等地；合作运营的航空公司包括东上航、南航、国航、春秋航、天津航、华夏航、奥凯航、深圳航、祥鹏航、河北航、江西航等11家。
近年来，湛江机场发展迅猛，自2014年起，湛江机场年旅客吞吐量相继突破100万人次、120万人次、150万人次，2017年突破200万人次，稳步跨进全国旅客吞吐量200万人次机场行列。

1994年，国家计委和民航总局批准了湛江机场的扩建项目，机场扩建工程在1995年正式破土动工，1997年10月验收并投入使用：扩建后的湛江机场的飞行区等级为4D级，跑道长2400米，能满足波音757以下飞机的起降要求；站坪面积5.8万平方米，能同时停放7架飞机（5大2小）；候机楼面积6600平方米，设计旅客吞吐量为130万人次；货运中心总面积1200平方米，设计货物吞吐能力为每年10万吨；另外，配套的通信网络工程，绿化美化工程也相继完成。

2005年2月3日上午，广东省机场管理集团公司湛江机场公司玉林办事处启用。湛江机场玉林办事处投入运营后，将在玉林市开展客货销售业务，负责组织协调玉林的客货源。该公司还计划在玉林市设立旅客离港系统，玉林旅客在购票时就可以拿到登机牌。另外，湛江机场还将在玉林开设直达机场的民航班车，使旅客享受到“一条龙”的优质服务。湛江与玉林之间只有两个多小时的车程，湛江机场驻玉林办事处的设立，通过湛江机场的航线资源，在玉林与广东和港澳地区之间架起了空中桥梁，为构筑两广九市旅游经济圈，推动玉林与湛江的两市的合作与交流必将起到有力的促进作用。

2006年，湛江—贵阳和湛江—揭阳（汕头）—上海航线相继开通，使得湛江市与西南、华东地区重点城市间的经贸、文化等方面的交流更加便捷。湛江—贵阳航线是继湛江—昆明航线开通后，粤西地区联系大西南的又一条重要空中通道，承运者华夏航空有限公司是湛江机场公司成功引进的第二家民营航空公司。湛江—贵阳航班编号为G52607/8，计划每天飞行1班。

2008年6月27日晚8时许，一架中国东方航空股份有限公司（China Eastern Airlines Corporation Limited，简称“东航”）CRJ客机雨后徐徐降落在湛江机场，这标志着停飞近两年的昆明至湛江、湛江至香港两条航线正式恢复飞行。湛江至香港、湛江至昆明航班计划每周各三班，逢周二、周五、周日执行。湛江至香港航班号为MU2193/4，湛江起飞时间为19:55，香港起飞时间为21:15；湛江至昆明航班号为MU2195/6,昆明起飞时间为18:00，湛江起飞时间为22:40。湛江开通至香港、昆明航班，对提升湛江城市形象、扩大对外开放、带动招商引资、满足区域社会经济发展等意义重大，架起了粤西通向大西南和香港的空中金桥，也为湛江走向世界架起了便捷的空中桥梁。

湛江机场与全国多个城市通航，高峰时期每周航班有100多班，旅客吞吐量最多年份超过60万人次，货邮行吞吐量为4000多吨，在全国机场排名中，名列第34位。

2014年湛江与茂名、广东省机场管理集团公司进行多次磋商，并取得阶段性成果。广东省发改委主任李春洪透露，最新的机场选址将靠近茂名，建成后两市可共享。据悉，湛江机场迁建项目规模远期按4E规划，近期按4D建设，投资约60亿元，2014年正在紧锣密鼓进行选址，预计2015年动工建设，2017年建设成为粤西地区共享的国际机场。

===出租车 ===
湛江市出租车起步价为7元/2公里，超出2公里后每公里2.3元，载客行驶达15公里后，计价器自动加收50%空驶费；每天晚上11时至次日凌晨5时30分期间，超出2公里后在公里租价基础上提高20%；等候费(包括交通拥挤)每分钟0.4元；行经收费桥梁、公路、渡口等费用由乘客支付(行经已纳入机动车辆路桥通行费年票的道路设施则不须付费)。车辆行驶满2公里后，按每218米跳表一次；车辆候车停泊时间每2分30秒跳表一次。

== 物产 ==
湛江[[海鲜|海鲜]]驰名省港地区。[[对虾|对虾]]、名贵鱼类、[[蚝|蚝]]、[[珍珠|珍珠]]、[[鲍鱼|鲍鱼]]等海产珍品的养殖规模较大，对虾、珍珠养殖产量连续多年为全国之冠，大批优质海产品畅销[[广州|广州]]、[[深圳|深圳]]、[[珠海|珠海]]等地，并出口香港、澳门、日本、韩国、美国、欧盟等国家和地区。湛江海产品之丰富闻名遐尔，其新鲜、质优、价廉，湛江市人们品尝海鲜、购买海产品的理想之地，近年来有“要吃海鲜到湛江”之说。2010年湛江喜获“中国海鲜美食之都”殊荣。<ref>{{cite web | url =http://gd.people.com.cn/GB/123943/123996/10929481.html|title =湛江喜获“中国海鲜美食之都”殊荣|work =广东频道--人民网| publisher = | author = | accessdate = 2009-07-03 }}</ref>
<br>海鲜：烤对虾、芷寮[[蟹|蟹]]湯、[[龙虾|龙虾]]三吃、[[海蜇|海蜇]]苦瓜、烤[[鳓鱼|鳓鱼]]排、清蒸[[黃花魚|黄花鱼]]、清蒸[[花鳝|花鳝]]、[[海参|海参]]什錦湯、[[东风螺|东风螺]]炒[[江瑤柱|江瑶柱]]<ref>{{cite web |url = http://www.southcn.com/travel/guangdong/zhanjiang/zhinan/200212030794.htm |title=湛江美食 |publisher=南方网旅游频道 |author= |accessdate=2009-07-03 }}</ref>
<br>名菜：[[湛江鸡|湛江鸡]]
<br>特色小吃：[[猪肠粉|猪肠粉]]，[[炒粉|炒粉]]，紫茎番薯，白切狗，[[牛杂|牛杂]]，烧蚝，番薯粥<ref>{{cite web|url=http://www.3608.com/gonglue/gl_4290.html|title=湛江饮食综述|publisher=全游网|author=|accessdate=2009-07-03|deadurl=yes|archiveurl=https://web.archive.org/web/20111028120510/http://3608.com/gonglue/gl_4290.html|archivedate=2011-10-28}}</ref>

*<big>'''湛江海产品：'''</big>湛江是全国唯一获得“中国海鲜美食之都”称号的城市，海产品享誉海内外。湛江海鲜驰名省港澳，新鲜、质优、价廉是它的特点，这使得湛江成为人们品尝海鲜、购买海产品的理想之地，有“要吃海鲜到湛江”之说，这里有南方规模较大的海产品交易市场---南方国际水产交易中心。同时，湛江也是名贵海鲜[[鲍鱼|鲍鱼]]、[[龙虾|龙虾]]的主要产地，产品远销港澳、日韩及东南亚。主要海鲜名产如下：
**'''官渡蚝：'''位于坡头区的官渡镇地理位置优越，是五里山港旁的天然海滩，生蚝吊养在无污染、无公害的原生态海域中，养殖出来的生蚝肉质鲜嫩肥美，口感清甜无渣、爽滑，可制作多种多样的生蚝食品；是目前原生态海鲜产品种类中，唯一无人工投料养殖的生态[[生蚝|生蚝]]。
**'''湛江虾：'''湛江海域宽广，港湾众多，沿海各地都产虾，不存在“物以稀为贵”。虾是湛江人常用的菜料，早晚菜市都有出售，也是饭桌上常见之菜式。湛江产的南美白对虾优质，年销售量占全国三分之二，其中大部分出口。有人测算中国市场每三条虾就有一条湛江虾，而美国人的餐桌每5条虾，就有一条来自湛江。特别是在2005年的国际虾战中，湛江虾胜出首获美国零关税，身份靡然。在虾的市场，湛江虾举足轻重，一直被商家看好，也使湛江虾市场这块蛋糕越做越大。除了南美白对虾，湛江还盛产花虾，比较名贵的品种是金须。
**'''虾蛄（或称皮皮虾）：'''湛江人所说的虾蛄即广州人说的濑尿虾，在湛江沿海昔日比比皆是，历史上曾拿来喂猪养鸭，是“下里巴人”。近年来，人们在烹饪上大造文章，使濑尿虾戴上光环，成为著名的水产类美食。昔日烂贱，今日高贵，大都市的最豪华宾馆酒楼，有濑尿虾的菜牌，高级的宴席有濑尿虾的菜谱，价格倍增。因其有虾蟹合一的美味，很受食客青睐，故在虾族中地位上升。湛江的濑尿虾异军突起，在虾的市场中也占有重要的一席。
**'''硇洲鲍鱼：'''位于湛江港湾的硇洲岛盛产味道鲜美、肉质嫩滑、营养丰富，素有“餐桌上的软黄金”之称的硇洲[[鲍鱼|鲍鱼]]，鲍鱼以加工形态有“干鲍鱼”和“鲜鲍鱼”之分；“干鲍鱼”又分“淡干鲍”和“咸干鲍”两种；品尝“干鲍鱼”以“淡干鲍”为好，是要求它品质优良，个头厚大肉质的丰腴与汁液的甘美清香；而品尝“鲜鲍鱼”则讲求鲍鱼的新鲜和美味的肉质，以每年的5月份最肥美，而在10、11月份肉较瘦。
**'''硇洲龙虾：'''硇洲除了盛产鲍鱼，还盛产龙虾。龙虾是海中最大型的爬行[[虾|虾]]类，美味珍奇稀罕，视为虾中“真龙天子”，身价百倍。湛江的龙虾，驰名远近，特别是硇州龙虾，过去在香港，被列为正宗龙虾。龙虾生活在深海，捕捉困难生长期漫长，幼虾到成虾，需5—7年，产量低，价格高，成了“阳春白雪”，使人可望而不可及。
**'''芷寮蟹：'''湛江东连南海，西枕北部湾岸线漫长海域广阔，滩涂众多，是[[蟹|蟹]]的良好栖身之地，自古以来盛产优质名蟹。湛江的海蟹与江南的淡水大闸蟹，平分秋色，各领风骚。湛江吴川的芷寮米蟹，在宋代已视为蟹中上品，产于蟹的养份充沛的吴川吴阳江海交汇的芷寮，以腿粗肉厚，膏满脂丰而著称，是中国四大名蟹之一。
**'''民安灶蟹：'''麻章区太平镇和东海岛民安镇的灶蟹也闻名遐迩，饮誉海外。湛江人称蟹洞为灶，这类蟹均在洞中捕捉，故称“灶蟹”。太平、民安的灶蟹，肥、香、鲜、脆，多为“双衣”“重壳”，味道别具一格，特别是具有“长命”的特点可活生生运到京城，历来被列为朝廷[[贡品|贡品]]。湛江流传不少贡蟹的故事，[[雷州府|雷州府]]白鸽寨武官劳守备误期押蟹到京；进士陈瑸违旨侍病母迟晋京授职，均借助赶蟹上路保其鲜活为名，巧脱欺君之罪，免于一死。故事除讴歌智慧外也反映了本地贡蟹品牌。
**'''江瑶柱：'''位于遂溪西部沿海的草潭、江洪是北部湾著名的渔港，盛产[[江瑤柱|江瑶柱]]。江瑶柱属[[蚌|蚌]]类，因其形如牛耳称牛耳螺，壳薄肉厚，肉质鲜嫩美味可口是海中珍品。其中又以草潭江瑶柱为上品，肉更鲜更嫩味道更胜一筹。以草潭江瑶柱制成的干品，味醇、无渣、爽口、压之易散开。
**'''鱿鱼丝：'''湛江产的[[鱿鱼|鱿鱼]]丝与其他沿海地区不同，相较于[[舟山|舟山]]鱿鱼丝，湛江鱿鱼丝偏干，口味偏咸。

*'''农产品：'''[[水稻|水稻]]、[[甘蔗|甘蔗]]、[[橡胶|橡胶]]、[[剑麻|剑麻]]、[[香茅|香茅]]、[[咖啡|咖啡]]、红江橙、[[菠萝|菠萝]]、[[西瓜|西瓜]]、[[香蕉|香蕉]]、[[龙眼|龙眼]]、[[荔枝|荔枝]]、[[芒果|芒果]]、[[火龙果|火龙果]]、[[大樹菠蘿|大樹菠蘿]]
*'''矿产：'''金、银矿、[[金红石|金红石]]、钛铁矿、锆英砂、石灰石、[[高岭土|高岭土]]、硅藻土、膨润土、花岗石、瓷土、石墨、[[天然气|天然气]]。
*'''湛江海水珍珠：'''湛江珍珠具有悠久的历史，品质优良，早在古代[[海上丝绸之路|海上丝绸之路]]时已是重要的[[贸易|贸易]]产品。湛江的海水珍珠又称南珠，是以马氏珠母贝产出的珍珠。由于湛江的北部湾得天独厚的地理环境，养育出来的南珠圆润、细腻、色彩瑰丽。湛江的南珠以粒大、圆润、光彩迷人，被誉为国之瑰宝驰名世界。而且国际珠宝业界历来就有“东珠不如西珠，西珠不如南珠”之说。珍珠还是名贵的药物，《[[本草纲目|本草纲目]]》中介绍：“珍珠粉涂面，令人润泽好色”。珍珠还有镇静、安神、化痰的功效。湛江的[[珍珠|珍珠]]生产主要分布在雷州、[[徐闻|徐闻]]、遂溪等县(市)沿海地区的10个乡镇，珍珠养殖面积5.38万亩年产珍珠20多万吨，有4.5万人从事珍珠产业总产值10多亿元。其中流沙珍珠2005年申报国家原产地保护标志成功，可以说湛江市具有养殖南珠的得天独厚自然条件，而且又有位于省内的南珠研究中心为依托，有近40年的人工养殖珍珠的实践经验，自90年代以来，湛江市海水珍珠贝苗年产量占全国的90%以上，海水珍珠（南珠）年产量约占全国70%，海水珍珠年加工、销售占全国70%以上。

== 著名人物 ==
*[[陈文玉|陈文玉]]（570 -638），[[雷州|雷州]]人，唐贞观五年（公元631年）为合州（今雷州）刺史。
*[[陳上川|陈上川]]（1626-1715），明朝高州府吴川县南三都田头村（今湛江市坡头区南三镇田头村）人，明末清初军事移民海外，为越南南部的开发立下基业的华侨先驱之一。
*[[陈瑸|陈瑸]]（1656-1718），雷州附城南田村人。曾任湖南巡抚、福建巡抚、闽浙总督等职。一生清正廉洁，勤政爱民，康熙皇帝称之为“清廉中之卓绝”。
*[[林召棠|林召棠]]（1786-1872），清朝高州府吴川县霞街村（今湛江吴川市霞街村）人，[[清朝|清朝]][[文学家|文学家]]。
