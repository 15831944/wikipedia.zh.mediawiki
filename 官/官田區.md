'''官田區'''（{{lang-nan-tw|Kuann-tiān khu}}）位於[[台灣|台灣]][[臺南市|臺南市]]中央地帶，北接[[六甲區|六甲區]]，南靠[[善化區|善化區]]，東南接[[大內區|大內區]]，西臨[[麻豆區|麻豆區]]、[[下營區|下營區]]。屬於[[新營區|大新營生活圈]]一員。是第10、11屆[[中華民國總統|中華民國總統]][[陳水扁|陳水扁]]與第14、15屆省轄[[臺南市長|臺南市長]][[許添財|許添財]]的故鄉。

== 歷史 ==
'''官田區'''舊稱「官佃」。[[台灣荷西殖民時期|荷治時期]]，此地屬於「王田」，即政府田地。[[鄭氏王朝|明鄭時期]]改王田為「官田」，將土地交由文武百官招募佃農屯墾，其中以[[陳永華_(明朝將領)|陳永華]]一族承佃最多，逐漸形成漢人聚落，故稱「官佃」。

[[臺灣清治時期|清治時期]]此地屬「-{官佃庄}-」，[[台灣日治時期|日治時代]]1920年改稱為「-{[[官田庄|官田庄]]}-」，屬[[臺南州|臺南州]][[曾文郡|曾文郡]]管轄，-{庄}-役場位於官田（後之官田村）。戰後改為[[臺南縣|臺南縣]]官田鄉，1953年將鄉治由官田村遷往隆田村（舊名番子田）。2010年12月25日改稱官田區。

官田區在三千年前即有人類活動的區域，卻是近一百年才於日治時代確立了今日官田區的疆域。官田區的區政雖年輕，人民與土地的歷史卻可溯源甚古。

== 地理 ==

[[File:官田、大崎_(_南111鄉道、台南藝術大學_).jpg|thumb]]、[[台南藝術大學|台南藝術大學]]）]]
[[File:福爾摩沙公路_(_官田區_).jpg|thumb]]官田路段]]

官田區位於臺南市中央偏北地點，[[曾文郡|曾文區]]之東；以[[曾文溪|曾文溪]]為界，東連[[中央山脈|中央山脈]]。中央山脈使得背風的[[嘉南平原|嘉南平原]]在入冬以後，難得雨水滋潤。為使農耕免於看天田的困境，官田區和鄰近行政區遂開鑿池埤蓄水備用，成為台地湖沼最密集的地方，除大大小小的池埤，最有名的是[[烏山頭水庫|烏山頭水庫]]、統領埤、蕃子田埤等。本區土地面積7,79公頃，占全縣總面積3.5%，其中農耕地面積有385.535公頃，山坡地面積有1,38公頃。主要產業為農業，農產有稻米、甘蔗、瓜果、蔬菜，特產為菱角、芒果。畜牧有乳牛、雞、豬。

== 特產 ==
*菱角：本區由於烏山頭水庫的設立，灌溉渠道密佈，[[菱角|菱角]]栽種面積達300公頃，年產量達400萬公斤，位居全國之冠，有「菱角之區」的美名，並於每年10月舉辦「官田菱角節」活動。
*天然氣：2004年6月，[[臺灣中油公司|臺灣中油公司]]在本區隆田-{}-里探勘到[[天然氣|天然氣]]，蘊藏量估計約4億立方公尺，現正進行開發。
