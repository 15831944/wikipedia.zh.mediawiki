'''官涌之戰'''是1839年11月发生的[[清朝|清帝國]]与[[大英帝國|大英帝國]]之间的武装冲突事件，是[[鸦片战争|鸦片战争]]的前哨战。清軍為了防範[[英軍|英軍]]的侵犯，於[[尖沙咀|尖沙咀]]以北[[官涌山|官涌山]]佈防。清[[道光|道光]]十九年九月廿九日至十月初八日 (1839年11月4－13日) ，[[義律|義律]]所率英舰向清軍進行海陸攻擊，於10天内攻击了6次，但是未能攻克官涌。

==背景==
道光十九年（1839年），[[钦差大臣|钦差大臣]][[林则徐|林则徐]]与[[两广总督|两广总督]][[邓廷桢|邓廷桢]]和水师提督[[关天培|关天培]]商议，为了控制外洋海面，在[[尖沙嘴|尖沙嘴]]和官涌山岗一带驻军。关天培调[[陈连升|陈连升]]防守。官涌营盘居高临下，可以俯击英船。英军不断派人到此偷袭。
1839年6月20日，[[九龍|九龍]][[尖沙咀|尖沙咀]]村發生[[林維喜案|林維喜案]]。[[英國|英國]]水兵在村內醉酒鬧事，打死村民林維喜，林則徐要求英國商務總監義律交出兇手，義律卻自己輕判了事。此事因而成為[[第一次鴉片戰爭|第一次鴉片戰爭]]的導火線，為[[香港|香港]]被割讓埋下伏筆。同年11月，中英隨即爆發穿鼻之戰，英方戰敗後逃回[[尖沙咀|尖沙咀]]，林則徐便下令軍隊駐在官涌山還擊。

== 第一次 ==
官涌营盘守军在陈连升和守备[[伍通标|伍通标]]率领下截击偷袭的英兵，打伤英军士兵并缴获了枪械，英军逃跑.

== 第二次 ==
九月廿九日（11月4日）夜，英兵舰数艘，对准官涌营盘一字排开，猛烈仰攻，但目标太高，不好命中。清军居高临下，向英兵舰进行俯击，英舰撤退。

== 第三次 ==
十月初三日（11月8日）英兵舰以大舰正面对官涌营盘开炮，同时派小船侧面包抄，乘海潮冲击靠近岸边，百余人乘隙抢上山岗，把总[[刘明辉|刘明辉]]率众截击，击伤英军数十名，死亡数人，其余滚下山岗。

== 第四次 ==
十月初四日（11月9日），英军在[[胡椒角|胡椒角]]开炮试探，营盘守军游击[[德连|德连]]用[[大炮|大炮]]、[[抬炮|抬炮]]一齐还击，英舰被击中，潜逃。

林则徐、邓廷桢和关天培，增调水师官兵二百人，添置大炮六座，并调派候补知府[[余保纯|余保纯]]、候补县丞[[张鹍|张鹍]]、[[新安县|新安县]][[知县|知县]][[梁星源|梁星源]]、驻守[[大鹏湾|大鹏湾]]的[[参将|参将]][[赖恩爵|赖恩爵]]、[[都司|都司]][[洪名香|洪名香]]、宋王台参将[[张斌|张斌]]就近带兵往官涌通力合作，准备夹攻来犯英军。

== 第五次 ==
十月初六日（11月11日）晚，清军按五个山梁分五路用大炮主动出击，英军逃跑。

== 第六次 ==
十月初八日（11月13日）下午，英舰来犯，清军仍以五个山梁，五路大炮叠轰，击退英舰。

== 影响 ==
林则徐、邓廷桢、关天培为防英军再犯，在尖沙嘴、官涌添置炮台，经[[余保纯|余保纯]]、[[赖恩爵|赖恩爵]]、[[梁星源|梁星源]]等实地勘察，于尖沙嘴山麓、官涌之南山上各建炮台一座，由内地调配大炮五十多门，还将大鹏湾营改为协，派副将驻守。
