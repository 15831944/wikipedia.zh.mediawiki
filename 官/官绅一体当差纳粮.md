'''官绅一体当差纳粮'''，或称'''士（仕）民一体当差纳粮'''、'''士绅一体当差、一体纳粮'''，是指[[清朝|清朝]][[雍正帝|雍正帝]]时期推行的一项改革政策，主要是取消民间[[缙绅|缙绅]]的某些特权，要求士绅和广大民众一样服徭役，纳粮捐税{{参|{{cite journal |author1=赵立波 |title=《雍正帝的三大宠臣》 |journal=文史天地 |date=2018-07-03 |issue=07期 |accessdate=2019-03-19}}}}。一体当差纳粮废除了他们享受免徭役赋税特权，彻底铲断了官绅的财路，不但国家财政丰润了起来，而且也解决封建王朝的平等缴税问题{{参|{{cite web |title=崇祯上吊自杀前，为何让人重新厚葬魏忠贤？|url=https://www.toutiao.com/a6551762515162825229/ |website=快资讯 |accessdate=2019-03-20}}}}。到了[[乾隆帝|乾隆帝]]执政期间免除了士绅的劳役，但是还是给交税纳粮。

== 背景 ==
官，指官员、官吏，绅，是指土绅，也就是地主。读书人一般通过[[科举|科举]]考试，考个功名就成了官绅。官绅一般不用服兵役、徭役，不用交粮食税，不负担国家财政。而官绅们的家人，甚至家奴也不用纳粮交税，挂靠在他家的土地也不用缴税{{参|{{cite web |title=什么是古代官绅？为啥说“一体当差一体纳粮”断了官绅财路？ |url=http://www.sohu.com/a/251559127_100059881 |website=搜狐网 |accessdate=2019-03-19}}}}。长此以往，土地兼并愈演愈烈，国家税收也越来越少。

还有就是，士绅在社会上具有话语权和发言权，在广大的江南地区一直是国家赋税的重要地方，士绅一体当差纳粮其实是皇权与地主士绅的较量，士绅一体当差纳粮和[[摊丁入亩|摊丁入亩]]双管齐下，在加上[[文字狱|文字狱]]，从经济和文化上对江南士绅集团造成了重创{{参|name=参考1}}。

== 过程 ==
雍正帝即位后，为了缓和社会矛盾，缩小贫富差距，推行官绅一体当差，一体纳粮。广大百姓对此十分支持，但是引起了士绅们的强烈不满。河南布政使[[田文镜|田文镜]]为了彻底贯彻政策，不但取消了士绅和读书人不用服役的特权，按照各地田亩的数量来统一征调劳力，还派遣兵勇押着这些人去干苦力，修提筑坝，稍有怨言，就侮骂殴打。遭到了以[[王逊|王逊]]、[[范瑚|范瑚]]为首的河南考生反对。他们威胁当地官员，将以罢考的方式反对这一政策{{参|{{cite journal |author1=刘鲁方 |title=《别人对你态度好,不是因为怕你》 |journal=《新一代》 |date=2016 |accessdate=2019-03-19}}}}{{参|{{cite journal |author1=冯啸天 |title=《清代雍正年间名臣田文镜评议》 |journal=《广西师范大学》 |date=2008年}}}}。雍正帝得知后给各省发布谕旨：“如再有抗顽生监，即行重处，毋得姑贷”{{参|《大清世宗宪皇帝实录》：谕直隶各省总督巡抚。凡百姓完纳钱粮。当令该户亲身投纳。不许里长甲首。巧立名目。希图侵蚀。不肖生员监生。本身田产无多。辄恃一衿。包揽同姓钱粮。自称儒户宦户。每当地丁漕米徵收之时。迟延拖欠。有误国课。通都大邑固多。而山僻小县尤甚。该督抚、著即严查晓谕。革除儒户宦户名目。如再有抗顽生监。即行重处。毋得姑贷。倘有瞻顾、不即革除此弊者。或科道参劾。或被旁人告发。治以重罪}}。雍正帝以铁腕手段维护这一政策，也使得这项政策免于形式{{参|{{cite web |title=雍正朝推行的官绅“一体当差，一体纳粮”政策，效果如何？ |url=http://www.qulishi.com/article/201803/276608.html |website=趣历史 |accessdate=2019-03-19}}}}。雍正帝让士绅们当差服劳役，颜面尽失，得罪了不少人。掌握舆论权的他们对雍正帝进行了无休止的抹黑和泼脏水。

== 后续 ==
乾隆帝即位后，对这项政策进一步改革，乾隆帝认为意识清朝的统治基础也需要士绅支持，如果地方的士绅不支持清朝统治，则大清的统治根基不稳，造成社会动荡不安，所以在一定程度上维护士绅们的利益{{参|{{cite web |title=雍正自信乾隆一定能完成新政，可乾隆为何一登基就废掉了新政？|url=https://new.qq.com/omn/20180825/20180825A0WQ5V.html |website=腾讯网 |accessdate=2019-03-20}}}}。在必须纳粮交税的前提下，规定从下向上的各级官绅逐步不用当差服役。这对于那些读书人来说，也是莫大的恩惠，无不对乾隆帝大加颂扬{{参|name=参考1|{{cite web |title=乾隆登基后雍正朝的新政继承下来了吗？他当真是败家子？看完秒懂 |url=http://dy.163.com/v2/article/detail/E2LS4VCL0523UNM6.html |website=网易网 |accessdate=2019-03-19}}}}。

== 争议 ==
而也有人指出，清朝时期实际上[[旗人|八旗子弟]]和皇族不用纳粮，而一体当差纳粮表面上看似“为民”，实际上是为了恩养旗人，压榨汉民罢了，同时通过打击官绅势力也可以对那些怀念故国明朝文人士大夫进行除了[[文字狱|文字狱]]之外的经济高压政策{{参|{{cite web |title=雍正官绅一体纳粮并非德政，收税是为了奉养一批人 |url=http://www.sohu.com/a/114604490_351483 |website=搜狐网 |accessdate=2019-03-19}}}}。
