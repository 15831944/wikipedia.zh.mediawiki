《'''實驗品家庭'''》（{{lang-ja|実験品家族 -クリーチャーズ・ファミリー・デイズ-}}），是香港漫畫家Yanai創作的漫畫，於[[日更COMICS|日更COMICS]]連載，並於日本[[新潮社|新潮社]]「{{tsl|ja|月刊コミック@バンチ|月刊comic＠bunch}}」展開日文版連載，為旗下首部登上日本雜誌連載的作品。

2017年宣布動畫化，動畫改編是由[[企鵝影視|企鵝影視]]、[[日更COMICS|日更計劃]]、[[大火鳥動畫|大火鳥動畫]]三方聯合出品。

== 關於 ==
丹尼斯的父母是世間無法容許的瘋狂科學家夫婦，他們與5名親生孩子在與世隔絕的小孤島的研究所生活，持續進行著秘密實驗，將除了丹尼斯之外的孩子全都改造成特異人種。一日，孤島的非法實驗被揭發，父母因而被警察拘捕、孩子們則在社工安排下到毗鄰的較大島嶼（以反轉的香港的南丫島為藍本）上居住。問題是，除了丹尼斯以外的四個兄姐都異於常人，且從小在孤島研究所長大而缺乏現代人的生活常識，唯一身為「普通人類」的天才末弟丹尼斯，肩負起外出採購、煮食等，照顧兄姐們起居生活，一直為他們能順利融入外面的世界、同時隱藏好各人的特異身分而努力及苦惱著……

== 登場人物 ==
; {{nihongo|丹尼斯|タニス||配音：[[杉山里穗|杉山里穗]]（日）；[[咩咩|咩咩]]（中）}}
: 年齡：11歲
: 家中五弟。天才兒童，學習什麽東西都非常快。11歲，被父母視為天才因而沒有被改造。除了擁有一般知識之外，在發明及醫術上亦特別有天份，然而是個路痴。

; {{nihongo|斯諾|スノー||配音：[[浪川大輔|浪川大輔]]（日）；[[曈音|曈音]]（中）}}
: 年齡：17歲
: 家中長兄。和白色的狗融合在一起的改造人，可保持人型、犬型或半獸型。一個身體卻有兩個意識掌控。

; {{nihongo|艾抒絲|アシス||配音：[[早見沙織|早見沙織]]（日）；[[月夜橋|月夜橋]]（中）}}
: 年齡：16歲
: 家中二姐。具有植物的基因，可以頗隨意地長出植物在身上。性格樂天隨性，甚少爭執，和雙胞胎妹妹的艾思莉的感情很好。

; {{nihongo|艾思莉|アイスリ||配音：[[森奈奈子|森奈奈子]]（日）；[[山新|山新]]（中）}}
: 年齡：16歲
: 家中三姐。具有蛛形綱（Arachnida）的基因，背上長著4條蜘蛛腿，愛吃，尤其是昆蟲類。性格衝動不愛受約束，因此也容易和丹尼斯吵架，相對較聽從雙胞胎姐姐的艾抒絲。

; {{nihongo|翠西|スイシ||配音：[[生田善子|生田善子]]（日）；[[沈念如|沈念如]]（中）}}
: 年齡：14歲
: 家中四姐。在改造下得到了讀心術，但也因此失去了表情。基本上不信任他人，家人也不太瞭解她在想什麽，時常看書。不喜歡人多的地方和思考太多的人。

; {{nihongo|堇|スミレ||配音：[[久保由利香|久保由利香]]（日）；[[閻麼麼|閻麼麼]]（中）}}
: 年齡：10歲
: 性格天真浪漫的女孩，甚至不知道世上有壞人這回事。在動畫版第二話中邂逅丹尼斯並請他吃果凍，在第三話再遇而認識，丹尼斯對她一見鍾情，但她對丹尼斯的哥哥斯諾懷有好感。

; {{nihongo|熊懐冬|ゆう かい とう||配音：[[堀江一真|堀江一真]]（日）；[[藤新|藤新]]（中）}}
: 年齡：32歲
: 接受了主角5人入住的房東、萬年研究生。心理學專業。對丹尼斯兄弟姐妹5人非常的友好，散發著家庭煮夫的關愛光輝。但是對於收留主角5人一事，似乎有著自己的目的。
