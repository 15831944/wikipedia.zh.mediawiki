《'''獻給世人'''》<ref>{{cite web|title=【#NGT48】獻給世人的最新單曲《獻給世人》，全世界同步數位發行！ |url=https://www.facebook.com/sonymusicjpop/photos/a.201274016336/10161271827771337/?type=3&theater|publisher=台灣索尼音樂JPOP官方臉書|language=zh-tw|date=2018-10-19|accessdate=2018-10-19}}</ref>（{{jpn|j=世界の人へ|hg=せかいのひとへ|rm=Sekai no hito e|f=y}}，{{lang-ru|Людям всего мира}}{{refnest|group=注|唱片封面的單曲名稱以俄文書寫。}}）是日本女子[[日本偶像|偶像]]組合[[NGT48|NGT48]]的第4張[[單曲|單曲]]，於2018年10月3日由{{Link-ja|Ariola Japan|アリオラジャパン}}發行。同名[[A面曲|標題曲]]由[[荻野由佳|荻野由佳]]擔任[[AKB48組合構成#中心位置|Center]]（中心位置成員）<ref name="n">{{cite news|url=https://natalie.mu/music/news/297494|title=NGT48の4thシングル「世界の人へ」詳細明らかに、ウラジオストクで撮影したMV公開|newspaper=[[Natalie_(網站)|音樂Natalie]]|publisher=Natasha Inc.|date=2018-08-30|accessdate=2018-08-31}}</ref>。

== 概要 ==
單曲分為Type-A、Type-B、Type-C和NGT48 CD盤，總共4個版本。其中，《獻給世人》、《Soft serve》和「《獻給世人》MV幕後花絮 in俄羅斯｜海參崴」收錄於所有版本，《獻給世人》的音樂錄影帶則收錄於NGT48 CD盤以外的所有版本。NGT48 CD盤收錄了《今天輸了也沒關係》；《心中的太陽》及其音樂錄影帶、「NGT48 研究生 短篇電影《短暫夏天的再見 -泉 篇-》」收錄於Type-A；《窗簾的圖案》及其音樂錄影帶、「NGT48 研究生 短篇電影《短暫夏天的再見 -園 篇-》」收錄於Type-B；《直到快要哭泣》及其音樂錄影帶、「NGT48 研究生 短篇電影《短暫夏天的再見 -文 篇-》」、影片「[[宮島亞彌|宮島亞彌]] [[畢業_(演藝界)|畢業]]紀念 最終訪談」收錄於Type-C。NGT48 CD盤以外所有版本均有全國[[握手會|握手會]]參加券和隨機[[生寫真|生寫真]]一張<ref>{{cite web |url=http://www.sonymusic.co.jp/artist/ngt48/discography/BVCL-907 |language=ja |script-title=ja:世界の人へ【Type-A 初回仕様（CD＋DVD）】 |publisher=[[NGT48|NGT48]] |accessdate=2018-11-01}}</ref><ref>{{cite web |url=http://www.sonymusic.co.jp/artist/ngt48/discography/BVCL-909 |language=ja |script-title=ja:世界の人へ【Type-B 初回仕様（CD＋DVD）】 |publisher=NGT48 |accessdate=2018-11-01}}</ref><ref>{{cite web |url=http://www.sonymusic.co.jp/artist/ngt48/discography/BVCL-911 |language=ja |script-title=ja:世界の人へ【Type-C 初回仕様（CD＋DVD）】 |publisher=NGT48 |accessdate=2018-11-01}}</ref><ref>{{cite web |url=http://www.sonymusic.co.jp/artist/ngt48/discography/BVCL-913 |language=ja |script-title=ja:世界の人へ【NGT48 CD盤（CD Only）】 |publisher=NGT48 |accessdate=2018-11-01}}</ref>。
