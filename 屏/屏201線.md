'''屏201線 琉球－海仔口'''，是位於[[台灣|台灣]][[屏東縣|屏東縣]]的一條鄉道，北起[[屏東縣|屏東縣]][[琉球鄉|琉球鄉]]琉球，[[屏東縣|屏東縣]][[琉球鄉|琉球鄉]]琉球，全長4.185（現場牌面）、3.811（縣府公告）公里{{r|thb}}。2015年9月24日交通部公告解編該公路<ref name="motc2015">{{cite web|url=http://odmdoc.motc.gov.tw/IFDEWebBBS_MOTC/ExternalBBS.aspx?ThirdDocId=104RD06046|publisher=[[中華民國交通部|中華民國交通部]]|accessdate=2017-03-08|date=2015-09-24|title=公告修正屏東縣鄉道公路路線表（含路線表及修正對照表）|deadurl=yes|archiveurl=https://web.archive.org/web/20160806154226/http://odmdoc.motc.gov.tw/IFDEWebBBS_MOTC/ExternalBBS.aspx?ThirdDocId=104RD06046|archivedate=2016-08-06}}</ref>。

== 路線說明 ==
[[File:屏206線與屏201線岔路.jpg|thumb]]
;全線均位於[[屏東縣|屏東縣]][[琉球鄉|琉球鄉]]內
*'''琉球'''（起點，{{jct|country=TWN|TRPT|202}}岔路）→ 中山路 → 上杉路路口（{{jct|country=TWN|TRPT|204}}岔路共線） → 和平路路口（{{jct|country=TWN|TRPT|204}}岔路分離） → 中正路 → '''南福'''（{{jct|country=TWN|TRPT|205}}岔路） → '''天福'''（{{jct|country=TWN|TRPT|206}}岔路） → '''海仔口'''（終點，{{jct|country=TWN|TRPT|202}}岔路）

== 歷史沿革 ==
本線於1966年編成{{r|1966-2}}並且維持相同路線。此公路已於2015年9月24日交通部公告解編<ref name="motc2015"/>。

*白沙尾漁港
*琉球國小
*琉球國中
*天福漁港
