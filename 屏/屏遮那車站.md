'''屏遮那車站'''位於[[台灣|台灣]][[嘉義縣|嘉義縣]][[阿里山鄉|阿里山鄉]]，為[[行政院農業委員會林務局|林務局]][[阿里山森林鐵路|阿里山森林鐵路]][[阿里山線|阿里山線]]之[[鐵路車站|鐵路車站]]。

== 車站構造 ==
無

== 利用狀況 ==
*阿里山線(主線)的停靠站。
*因受[[八八風災|八八風災]]影響，暫時停駛。

== 車站週邊 ==
== 歷史 ==
*1912年12月25日 - 設立「平遮那驛」。
*1965年6月1日 - 更名為屏遮那<ref>林務局(54) 5 28林產字第一九○一八號呈；收錄於臺灣省政府公報五十四年夏字第六十八期</ref>。
