'''屏山天水圍公共圖書館'''（{{lang-en|Ping Shan Tin Shui Wai Public Library}}）是[[香港|香港]][[元朗區|元朗區]]的一座[[圖書館|圖書館]]，屬[[康樂及文化事務署|康樂及文化事務署]]轄下[[香港公共圖書館|香港公共圖書館]]系統管理的[[主要圖書館|主要圖書館]]，該圖書館[[總樓面面積|樓面面積]]6,100[[平方米|平方米]]，是香港規模第二大的公共圖書館，僅次於總館[[香港中央圖書館|香港中央圖書館]]，位於[[新界|新界]][[元朗區|元朗區]][[屏山_(香港)|屏山]][[聚星路|聚星路]]1號[[屏山天水圍文化康樂大樓|屏山天水圍文化康樂大樓]]高座<ref name="location & open time"/>。於2013年2月28日啟用。是香港第一間設有戶外閱讀環境的公共圖書­館<ref>[http://hk.news.yahoo.com/屏山天水圍圖書館周四啟用-153556230.html 屏山天水圍圖書館周四啟用]</ref>，主要為元朗區，以至整個新界西北區居民服務。

== 命名 ==
規劃及興建時曾稱「新天水圍公共圖書館」<ref>{{cite news|title=立法會十二題：公共圖書館服務|url=http://www.info.gov.hk/gia/general/200111/21/1121157.htm|publisher=香港政府新聞公報|date=2011年11月21日}}</ref>，及後依照康文署參照附近設施與附近道路名稱的場地命名指引，正式命名為上述名稱。<ref>{{cite web|title=康樂及文化事務署 新天水圍主要圖書館命名 及 配合新天水圍主要圖書館啟用後元朗區流動圖書館服務的新安排|url=http://www.districtcouncils.gov.hk/yl/doc/common/committee_meetings_doc/dfmc/2012/dfmc(2012)_061.pdf|publisher=地委會文件|date=2012／第 61 號|deadurl=yes|archiveurl=https://web.archive.org/web/20130123191602/http://www.districtcouncils.gov.hk/yl/doc/common/committee_meetings_doc/dfmc/2012/dfmc(2012)_061.pdf|archivedate=2013-01-23}}</ref>
名稱包含[[屏山_(香港)|屏山]]及[[天水圍|天水圍]]兩個地方，與[[南丫島南段公共圖書館|南丫島南段公共圖書館]]及[[南丫島北段公共圖書館|南丫島北段公共圖書館]]同樣具有10個[[漢字|漢字]]，為香港公共圖書館系統中名稱最長的圖書館之一，而英文名稱則是系統中最長，具有31個[[羅馬字母|羅馬字母]]。為方便稱呼，使用者常稱該圖書館為「屏山圖書館」或「天水圍圖書館」，唯「天水圍圖書館」亦可指位於[[天澤邨|天澤邨]]的[[天水圍北公共圖書館|天水圍北公共圖書館]]。

== 歷史 ==
{{main|天水圍公共圖書館#歷史}}
該圖書館是為了取代此前位於[[嘉湖銀座|嘉湖銀座]]的[[天水圍公共圖書館|天水圍公共圖書館]]而興建，該館雖使用約十年，但一直屬臨時性質。

在[[臨時區域市政局|臨時區域市政局]]解散後，[[香港特區政府|特區政府]]計劃於[[天福路|天福路]]和[[屏廈路|屏廈路]]交界（近[[西鐵綫|西鐵綫]][[天水圍站_(西鐵綫)|天水圍站]]）興建新的圖書館和綜合大樓，樓面面積按新主要圖書館的規格6,200平方米而建<ref>{{cite web|title=立法會民政事務委員會 在地區層面設置公共圖書館|url=http://www.legco.gov.hk/yr09-10/chinese/panels/ha/papers/ha0409cb2-1203-1-c.pdf|publisher=香港特別行政區立法會|date=2010年4月9日}}</ref><ref>{{cite news|title=天水圍超級圖書館 明年啟用|url=http://orientaldaily.on.cc/cnt/news/20121103/00176_023.html|date=2012年11月3日|newspaper=東方日報}}</ref>。

初期香港政府新聞公報及[[香港報章|香港報章]]《[[AM730|AM730]]》聲稱該圖書館會於2012年年底啟用<ref>{{cite news|title=康文署今增兩個暖水游泳池迎冬季|url=http://www.info.gov.hk/gia/general/201111/30/P201111300164.htm|publisher=香港政府新聞公報|date=2012年6月23日}}</ref><ref name="am730">{{cite news|title=自然光透進低樓層省電三成 圖書館首設戶外閱讀|url=http://www.am730.com.hk/article_print.php?article=103122|newspaper=AM730|date=2012年6月23日}}</ref>。共228個座位的[[自修室|自修室]]在2012年11月19日率先投入服務<ref>{{cite web|title=《2013年圖書館指定 (修訂) 令》|url=http://www.gld.gov.hk/egazette/pdf/20131701/cs2201317011.pdf|publisher=香港特別行政區政府|date=2013年1月11日}}</ref>，而圖書館則在《[[香港政府憲報|香港特別行政區憲報]]》刊發由[[康樂及文化事務署|康樂及文化事務署]]署長[[馮程淑儀|馮程淑儀]]簽署的命令下，才正式宣布於2013年2月28日起投入服務，而位於嘉湖銀座（現稱[[置富嘉湖|置富嘉湖]]）的圖書館亦於當日停止運作<ref>{{cite web|title=《2013年圖書館指定 (修訂) 令》|url=http://www.gld.gov.hk/egazette/pdf/20131701/cs2201317011.pdf|publisher=香港特別行政區政府|date=2013年1月11日}}</ref><ref>[http://www.metrohk.com.hk/pda/pda_detail.php?id=185985&selectedDate=2012-05-10&categoryID=all 港綠建築奪獎]</ref><ref>[http://hk.news.yahoo.com/屏山天水圍公共圖書館啟用-094000467.html; 屏山天水圍公共圖書館啟用] 《星島日報》 2013年2月25日</ref>。

== 設施 ==
圖書館樓高7層，面積6,100平方米。除了成人圖書館、兒童圖書館及參考圖書館外，該館1樓設有擺放青少年讀物的青少年圖書館，是全港第二間設有同類設施的公共圖書館。而3樓報刊閱覽部旁的小說天地，是專門擺放不同類型小說的區域。

圖書館初期藏書約270,000冊，現已增至330,000冊<ref>{{cite news|title=屏山天水圍公共圖書館舉行「南亞裔親子故事時間」（附圖）|url=http://www.info.gov.hk/gia/general/201304/01/P201303280457.htm|publisher=香港政府新聞公報|date=2013年4月1日}}</ref>，逾15,000項錄­音資料及唯讀記憶[[光碟|光碟]]、參考館藏（包括香港特藏及地區特藏、一般參考圖書、字典及指南等）逾30,000冊。圖書館亦設有多媒體資料圖書館、電腦資訊中心及[[互聯網|互聯網]]／電子資源工作站，內有多部[[電腦|電腦]]供讀者使用。另外，圖書館各層均設有自助借書機及在地下設立特快歸還服務，簡化借還書手續及推行圖書館自助化。

== 使用狀況 ==
該館是全元朗區最多人使用的公共圖書館，每天平均到訪人數約3,000至4,000多人，平均借出的圖書館資料數量約3,000至4,000項。<ref>{{cite web|title=元朗區公共圖書館舉辦的推廣活動暨使用概況匯報|url=http://www.districtcouncils.gov.hk/yl/doc/2016_2019/tc/committee_meetings_doc/dfmc/9795/dfmc(2016)_026.pdf|publisher=地委會文件|date=2016／第 26 號}}</ref><ref>{{cite web|title=元朗區公共圖書館舉辦的推廣活動暨使用概況匯報|url=http://www.districtcouncils.gov.hk/yl/doc/2016_2019/sc/committee_meetings_doc/dfmc/9796/dfmc(2016)_038.pdf|publisher=地委會文件|date=2016／第 38 號}}</ref>另外，該館也是全港第一所在夜間舉辦活動的公共圖書館，約70名中學生在2013年10月11日通宵在圖書館內進行多項活動。<ref>{{cite web|title=七十多名中學生夜探屏山天水圍公共圖書館　體驗科學與創意之旅（附圖）|url=http://www.info.gov.hk/gia/general/201310/11/P201310110420.htm|publisher=香港政府新聞公報|date=2013年10月11日}}</ref>

== 爭議 ==
=== 禁止公眾拍攝 ===
雖然圖書館設計創新，並曾獲多個建築大獎，並被喻為「香港最美公共圖書館」。<ref>{{cite news|title=香港最美公共圖書館隱身天水圍：感受當藏書成為城市最迷人風景的氛圍|url=http://www.adaymag.com/2017/04/19/tin-shui-wai-public-library.html|accessdate=2017-04-21|publisher=A Day Machine|date=2017-04-19}}</ref>惟該圖書館規則與香港所有圖書館一樣，禁止公眾進行拍攝及錄影。而此館對禁止攝影的管理較一般圖書館更嚴。康文署為防有人拍攝及錄影，於場內各層多個位置，包括入口、服務台、升降機前、樓梯、兒童圖書館以至自修室等貼上多張「禁止拍攝」告示，以及各層派遣保安作為監視。有媒體在圖書館6樓的綠化平台僅拍攝街外風景亦被保安阻止。保安員指出圖書館的平台屬於康文署管理範圍，因此不論使用手機或相機都不可以攝影。元朗區議員鄺俊宇批評措施欠人性化。<ref>{{cite web|title=圖書館靚景「被禁拍」 綠化天台受歡迎 區議員倡放寬|url=http://www.msn.com/zh-hk/news/newsinstant/%E5%9C%96%E6%9B%B8%E9%A4%A8%E9%9D%9A%E6%99%AF%E3%80%8C%E8%A2%AB%E7%A6%81%E6%8B%8D%E3%80%8D-%E7%B6%A0%E5%8C%96%E5%A4%A9%E5%8F%B0%E5%8F%97%E6%AD%A1%E8%BF%8E-%E5%8D%80%E8%AD%B0%E5%93%A1%E5%80%A1%E6%94%BE%E5%AF%AC/ar-BBoczTf?li=AA570l|publisher=都市日報|date=2016年1月15日|deadurl=yes|archiveurl=https://web.archive.org/web/20161030202213/http://www.msn.com/zh-hk/news/newsinstant/%E5%9C%96%E6%9B%B8%E9%A4%A8%E9%9D%9A%E6%99%AF%E3%80%8C%E8%A2%AB%E7%A6%81%E6%8B%8D%E3%80%8D-%E7%B6%A0%E5%8C%96%E5%A4%A9%E5%8F%B0%E5%8F%97%E6%AD%A1%E8%BF%8E-%E5%8D%80%E8%AD%B0%E5%93%A1%E5%80%A1%E6%94%BE%E5%AF%AC/ar-BBoczTf?li=AA570l|archivedate=2016年10月30日}}</ref>

屏山原居民[[鄧達智|鄧達智]]認為，大樓設計別樹一幟，可以成為旅遊景點，不必完全禁止公眾拍照。<ref>{{cite web|title=【公共空間禁拍】戶外圖書館美景　有得睇冇得影|url=http://www.hk01.com/%E7%A4%BE%E5%8D%80/1620/-%E5%85%AC%E5%85%B1%E7%A9%BA%E9%96%93%E7%A6%81%E6%8B%8D-%E6%88%B6%E5%A4%96%E5%9C%96%E6%9B%B8%E9%A4%A8%E7%BE%8E%E6%99%AF-%E6%9C%89%E5%BE%97%E7%9D%87%E5%86%87%E5%BE%97%E5%BD%B1|publisher=香港01|date=2016年1月17日}}</ref>康文署表明不會對圖書館的攝影規限修改或放寬。<ref>{{cite news|author1=元朗區議會|title=元朗區議會地區設施管理委員會 二零一六年度第二次會議記錄|url=http://www.districtcouncils.gov.hk/yl/doc/2016_2019/tc/committee_meetings_minutes/dfmc/dfmc(2016)_2nd_minutes.pdf|accessdate=2017-04-21|date=2016-03-04}}</ref>
