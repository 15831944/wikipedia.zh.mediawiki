'''屏南县'''（[[平话字|平话字]]：''Bìng-nàng''）位于[[福建省|福建省]]东北部，歷史上曾是[[福州十邑|福州十邑]]之一，现在是[[宁德市|宁德市]]下辖的一个县。屏南县东南方向连[[蕉城区|蕉城区]]，东北接[[周宁县|周宁县]]，北连[[政和县|政和县]]，西北至西是[[建瓯市|建瓯市]]，西南至南为[[古田县|古田县]]。通行[[福州話|福州話]]的一個分支。

== 行政区划 ==
下辖4镇，7乡。
*镇：古锋镇、长桥镇、双溪镇、代溪镇。
*乡：屏城乡、甘棠乡、棠口乡、熙岭乡、路下乡、岭下乡、寿山乡 。

== 资源 ==
因出产夏香菇而有“中国夏香菇之乡”之称号。屏南县的水电蕴藏量达40万[[千瓦|千瓦]]，是全国100个农村电气化试点县之一。

== 文化 ==
屏南县的四平戏、屏南双溪铁枝、屏南平讲戏及木拱桥传统营造技艺被列入[[国家级非物质文化遗产名录|国家级非物质文化遗产名录]]。

== 旅游 ==
[[File:Pingnan_Confucian_Temple_2014.08.03_07-35-19.jpg|300px]]]]
* [[白水洋|白水洋]]-鸳鸯溪：[[宁德世界地质公园|宁德世界地质公园]]的重要组成部分，5A级旅游景区。
* [[万安桥_(屏南)|万安桥]]：俗称长桥、彩虹桥，位于长桥村东面的长桥溪上，始建于北宋，是中国现存最长的木拱廊桥。全国重点文物保护单位[[闽东北廊桥|闽东北廊桥]]之一。
