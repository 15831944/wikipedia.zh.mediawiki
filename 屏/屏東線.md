{{屏東線RDT}}
'''屏東線'''是指[[高雄車站|高雄]]至[[枋寮車站|枋寮]]間，由[[臺灣鐵路管理局|臺灣鐵路管理局]]經營的傳統鐵路幹線，早期沿線景觀多為農田與魚塭。目前配合[[高雄市區鐵路地下化計劃|高雄市區鐵路地下化計劃]]，將原高雄機檢段遷至[[潮州基地|潮州基地]]以及潮枋間電氣化，地貌已經有很大改變。

== 路線資料 ==
*經營管轄：[[臺灣鐵路管理局|臺灣鐵路管理局]]
*路線距離：高雄－枋寮間 61.3 公里
*[[軌距|軌距]]：1,067 毫米
*車站數：18（含起訖車站；2007年3月）
*開業時間：1941年12月15日
*雙線區間：高雄－潮州 36.1 公里
*[[電氣化鐵路|電化]]區間：高雄－潮州（[[交流電|交流]]25KV）36.1 公里
*隧道數目：1座
**新左營（不含）－後庄（不含）間（含[[縱貫線_(南段)|縱貫線]]）：[[高雄市區鐵路地下化計畫|高雄鐵路地下化]]
*六塊厝（不含）－潮州、鎮安（不含）－佳冬（不含）間高架化

== 運行形態 ==
*對號列車（本區間銜接[[縱貫線_(鐵路)|縱貫線]]與[[南迴線|南迴線]]，因此對號列車多為跨線列車，或是具回送性質短程列車。）
**[[西部幹線|西部幹線]]
***行駛區間為基隆、七堵到潮州為主。
***行駛車種為自強號與莒光號。
**[[南迴線|南迴線]]
***主要行駛區間為新左營到臺東，亦有延伸到臺南、臺中或是花蓮的列車。
***行駛車種為自強號與莒光號，假日也開行復興號。
*[[臺鐵區間車|區間車]]
**以雲林縣、嘉義縣市、臺南市、高雄市、屏東縣等鄰近縣市為運行範圍。
**列車運行區間主要為
***新竹、后里、嘉義到潮州的電聯車。
***新左營至潮州的電聯車。
***潮州至枋寮的柴聯車與復興號客車

=== 過去的運行型態 ===
過去由於以[[高雄港車站|高雄港車站]]（當時稱為「打狗停車場」）作為高雄端點站，即使是遷站至現今的[[高雄車站|高雄車站]]、[[鼓山車站|鼓山]]－高雄間新建[[縱貫線_(南段)|縱貫線]]路軌，仍有列車維持鼓山－[[壽山車站|壽山]]－高雄港－壽山－[[三塊厝車站|三塊厝]]－高雄的運行模式，直到高雄港停辦客運，壽山、三塊厝廢站及高雄港－高雄間廢線。

== 歷史 ==
* 原本被稱為'''鳳山支線'''的打狗（今[[高雄港車站|高雄港]]）－[[九曲堂車站|九曲堂]]段鐵路於1907年完工，次年正式通車。1913年，九曲堂—[[屏東車站|屏東]]段通車，之後路線持續延伸至[[潮州車站|潮州]]、溪州（今 [[南州車站|南州]]），最終於1941年延伸通車至[[枋寮車站|枋寮]]。路線名稱也曾經隨著通車路段終點站的多次變更，陸續改稱為「屏東線」、「潮州線」（1920年9月26日）、「溪州線」，全線通車後正式定名為屏東線。
* 1944年：林邊至枋寮間路線，因戰時鋼料缺乏，成為[[不要不急線|不要不急線]]而拆除。
* 1953年1月16日：林邊至枋寮間路線恢復通行。
* 1987年4月10日：高屏鐵路雙軌計畫關鍵工程新高屏溪橋完工<ref>交通部：中華民國77年《交通年鑑》，台北，1989年10月，頁329。</ref>。
* 1987年6月17日：高屏鐵路雙軌計畫新高屏溪橋切換通車<ref>交通部：中華民國77年《交通年鑑》，台北，1989年10月，頁329。</ref>，原日本時代單線[[下淡水溪鐵橋|下淡水溪鐵橋]]停用。
* 1988年7月1日：分三年實施屏東至枋寮間鐵路改善工程，依台鐵甲級路線標準辦理，37公斤鋼軌更換為50公斤重軌、電氣路牌行車制更新為單線自動閉塞號誌(計劃1990年6月底完工)<ref>台灣鐵路管理局：《臺灣鐵路百週年紀念》，台北，1987年6月，頁343～344。</ref>。
* 1989年7月1日：高屏鐵路雙軌計畫高雄至屏東間完成機械號誌先行雙線通車<ref>交通部：中華民國78年《交通年鑑》，台北，1990年10月，頁369。</ref>。
* 1990年6月30日：高屏鐵路雙軌計畫高雄至屏東間雙單線CTC行車號誌啟用<ref>交通部：中華民國79年《交通年鑑》，台北，1991年10月。</ref>。
* 1995年5月3日：[[高雄車站|高雄]]－[[鳳山車站_(臺鐵)|鳳山]]間電車線通電<ref name=TraElec>[http://www.railway.gov.tw/tw/CP.aspx?sn=3634&n=6842 交通部臺灣鐵路管理局-臺鐵簡介-台灣鐵路電訊]</ref>。
* 1996年7月10日：[[鳳山車站_(臺鐵)|鳳山]]－[[屏東車站|屏東]]間電車線通電<ref name=TraElec />。
* 2009年：因受到[[八八水災|八八水災]]影響，[[林邊車站|林邊]]－[[佳冬車站|佳冬]]間林邊溪橋沖毀，於水災後至12月30日間列車僅行駛至[[南州車站|南州車站]]，僅區間車和復興號行駛。[[林邊車站|林邊]]－[[佳冬車站|佳冬]]間林邊溪橋已經在12月30日修復完成。
* 2012年1月10日：[[鎮安車站|鎮安]]（不含）－[[佳冬車站|佳冬]]（不含）間高架化路線啟用，為屏東線鐵路第一段啟用的高架化區間。
* 2013年6月25日：[[高雄潮州鐵路捷運化計畫|高雄潮州鐵路捷運化計畫]]中的[[歸來車站|歸來]]到[[潮州車站|潮州]]間西正線高架化通車，列車在屏東到潮州間改行駛高架路線<ref>{{Cite web | url = http://news.ltn.com.tw/news/local/paper/678295 |title = 屏東至潮州鐵路高架化 6.25單線通車 | publisher = 自由時報 |language=zh-tw }}</ref><ref>{{Cite web | url = http://news.ltn.com.tw/news/local/paper/691582 |title = 屏東鐵路百年大事 屏潮線高架化 通車 | publisher = 自由時報 |language=zh-tw }}</ref><ref>{{Cite web | url = http://www.railway.gov.tw/tw/news_Detail.aspx?SN=5789 |title = 慶祝歸來＝潮州高架鐵路通車暨新站啟用潮州站發售精美紀念套票 | accessdate = 2013-06-25 | publisher = [[臺灣鐵路管理局|臺灣鐵路管理局]] }}</ref>，原平面路線拆除，供施作屏東至潮州高架、電氣、雙軌化用。
* 2015年8月23日：啟用[[屏東車站|屏東站]]新的高架站體，於8月22日夜間進行屏東站兩端切換，連接高架的屏東站場與屏潮計畫中的東正線。在8月23日～8月25日陸續測試路線與號誌，逐步啟用東正線。
* 2015年8月26日：延駛部份電氣化自強號與區間車，以[[潮州車站|潮州站]]作為起站與到站。且因[[屏東車站|屏東站]]的第二東主正線尚未鋪設，故加開數趟回送列車到西勢折返。
* 2015年10月15日：潮州基地正式啟用，高雄機務段、高雄檢車段遷移至此；西部幹線列車營運大部分改至[[潮州車站|潮州]]到開，少部份由[[屏東車站|屏東]]或[[高雄車站|高雄]]折返運用。原先由[[新左營車站|新左營]]屏東線區間車截短自潮州站到開。
*2018年10月14日：[[高雄市區鐵路地下化計畫|高雄鐵路地下化]]通車，除原有[[鳳山車站_(臺鐵)|鳳山]]外，另增設[[民族車站|民族]]、[[科工館車站|科工館]]、[[正義車站|正義]]3個通勤車站。

== 改建計劃 ==
*[[高雄潮州鐵路捷運化計畫|屏東至潮州高架雙軌電氣化（屏潮計畫）]]
**進行高架化工程，同時將原先屏東到潮州單線變更為雙線，並完成電氣化。
**新建潮州車輛基地，作為西部幹線列車整備之用。
**範圍自屏東北端到潮州南端。
**全線於2015年8月23日通車。<ref>{{cite news|author=鄭瑋奇|title=高雄潮州高架雙軌電氣化 下月通|url=http://www.tssdnews.com.tw/?FID=6&CID=279962|accessdate=2015-08-09|newspaper=[[台灣新生報|台灣新生報]]|date=2015-07-15}}</ref>
**屏東新站2015年8月23日完成啟用。<ref>{{cite news|author=毛麗貞|title=台鐵屏東高架新站 8月下旬啟用|url=http://www.tssdnews.com.tw/?FID=14&CID=275178|accessdate=2015-08-09|newspaper=[[台灣新生報|台灣新生報]]|date=2015-06-05}}</ref>
**全數工程於2017年11月3日完成。

*[[南迴鐵路電氣化|南迴電氣化計畫]]
**進行電氣化工程，同時將南州到林邊雙線化。
**範圍自潮州到臺東。
**預計2020年12月通車，2024年3月全部完工。<ref>[http://www.rrb.gov.tw/02110.aspx?lan=ch&id=442 「南迴鐵路電氣化工程」建設計畫，預計109年12月電氣化通車 ] 交通部鐵路改建工程局</ref>

== 著名景觀 ==
*[[高屏溪橋|高屏溪橋]]：為舊鐵橋經多次颱風重創之後往平地遷建，舊鐵橋所在被闢為公園。其出海口夕照被譽為南部二十大美景之一。
*[[下淡水溪鐵橋|下淡水溪鐵橋]]：舊線最長的鐵橋，亦是昔日台鐵最長的鐵路橋樑。位於[[九曲堂站|九曲堂站]]=[[六塊厝站|六塊厝站]]之間，現為公園。
*[[竹田驛站|竹田驛站]]、[[竹田博士莊園|竹田博士莊園]]：於[[竹田車站|竹田車站]]附近。
*[[台灣海峽|台灣海峽]]、[[小琉球|小琉球]]：[[林邊車站|林邊車站]]～[[枋寮車站|枋寮車站]]段。
