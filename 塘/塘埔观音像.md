'''塘埔观音像'''（又名：渔湖观音像）是一尊来自[[广东省|广东省]][[汕头市|汕头市]][[潮阳区|潮阳区]][[关埠镇|关埠镇]][[石井佛堂|石井佛堂]]的大型[[观音|观音]]像<ref name="wx1">{{cite web |url=http://www.112seo.com/cityarticle-3573187.html |title=【揭秘】揭阳渔湖为何突然出现观音像？真相竟然是...... |author=揭阳空港生活 |date=2017-02-03 |website= 微信公众号焦点|publisher= |access-date=2017-10-10 |quote=}}</ref><ref>{{cite web |url=http://www.gdfabu.com/shantou/4292972.html |title=后续 揭阳塘埔观音像疑来自“关埠镇”石井佛堂,18年前从台湾... |author=汕头新鲜事 |date=2017-02-04 |website=广东发布 |publisher= |access-date=2017-10-10 |quote= |archive-url=https://web.archive.org/web/20171009200048/http://www.gdfabu.com/shantou/4292972.html |archive-date=2017-10-09 |dead-url=yes }}</ref>，2017年2月2日于[[揭阳市|揭阳市]][[渔湖镇|渔湖镇]][[塘埔村|塘埔村]]发现<ref name="qq"/><ref name="wx2">{{cite web |url= http://www.112seo.com/cityarticle-3568038.html|title= 观音显灵？揭阳某地江边突现一尊观音像，围观群众表示很神奇！|author=揭阳吃喝玩乐 |date=2017-02-03 |website=微信公众号焦点 |publisher= |access-date= 2017-10-10|quote=}}</ref>，因此得名为塘埔观音。

== 来历 ==
1998年，[[潮阳区|潮阳区]][[关埠镇|关埠镇]]石井佛堂从[[台湾|台湾]]请来一尊大型[[观音|观音]]像，曾经是莲花池池边的露天[[佛像|佛像]]；2013年9月，因[[台风天兔_(2013年)|台风天兔]]吹倒的大[[椰树|椰树]]砸坏观音像的右手手指和脚，2017年1月26日（农历丙申年十二月廿九日）漂流至[[榕江_(广东)|榕江]]<ref name="wx1" />。

2017年2月2日（农历正月初六）于塘埔村榕江边被村民发现，当时观音像在水中呈站立姿态而未下沉或倾倒，除右手的手指和脚外全身均完好无缺；村民见此像纷纷围观和参拜，被认为是观音菩萨[[显灵|显灵]]。之后船只将观音像打捞上岸，暂时放置在附近的榕树下<ref name="qq">{{cite web |url=http://rufodao.qq.com/a/20170208/023505.htm |title= 广东揭阳江中现观音像 村民打捞参拜|author=中华网 |date=2017-02-08 |website=腾讯网 |publisher= |access-date= 2017-10-10|quote=}}</ref>。2017年2月3日，观音像请到了一处室内，像上披一块红布，摆上鲜[[花|花]]和[[水果|水果]]供奉。<ref>{{cite web|url= https://www.yangqiu.cn/jyxingkong/1602592.html|title= 今日微资讯【后续】渔湖塘埔观音像披上红色袈裟，鲜花水果供奉！|author= 揭阳星空网|date= 2017-02-03|website= |publisher= |access-date= 2017-10-10|quote= |archive-url= https://web.archive.org/web/20171010005547/https://www.yangqiu.cn/jyxingkong/1602592.html|archive-date= 2017-10-10|dead-url= yes}}</ref>2017年2月6日（农历正月初十）修复观音像的右手[[手指|手指]]，供奉在一座新建的小庙里。<ref>{{cite web|url=http://www.tuwenba.com/article/MDA2MTE3N9TE5.html|title=揭阳渔湖观音神像损坏的手已修，到场喜捐者送相片一张、大吉一双！|author=揭阳空港生活|date=2017-02-10|website=图文吧|publisher=|access-date=2017-10-10|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20171010005922/http://www.tuwenba.com/article/MDA2MTE3N9TE5.html|archivedate=2017年10月10日|df=}}</ref>2017年3月16日（农历二月十九）举行杨净开观道场法会。<ref>{{cite web|url=http://www.sohu.com/a/129106262_443924|title=今日！空港区渔湖塘埔村观音举行杨净开观道场法会！|author=揭阳空港生活|date=2017-03-16|website=搜狐网|publisher=|access-date=2017-10-10|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20171009194659/http://www.sohu.com/a/129106262_443924|archivedate=2017-10-09}}</ref>

== 描述 ==
[[File:Tangpu_Guanyin_guidepost.jpg|thumb]]
塘埔观音像通体[[白色|白色]]，材质似[[石|石]]似[[玉|玉]]。右手持宝珠，左手垂下。

目前塘埔观音像供奉在当时发现地点的一棵榕树旁供游客参拜，在观音像的附近地区立有5块“往塘埔观音菩萨”的指路牌，其中有两块路牌被拆除。
