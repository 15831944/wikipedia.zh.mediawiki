《'''薙刀社青春日記'''》（{{lang-ja|あさひなぐ}}）是[[小崎亞衣|小崎亞衣]]所著的[[日本|日本]][[校園漫畫|校園漫畫]]作品，以日本傳統武道女子[[薙刀|薙刀]]為題材，講述[[高級中學|高中生]]東島旭加入[[薙刀|薙刀]][[學生社團|社]]的故事。漫畫在《[[Big_Comic_Spirits|Big Comic Spirits]]》2011年8月號開始連載，並獲第60屆[[小學館漫畫獎|小學館漫畫獎]]的一般向部門賞<ref>[http://comic-soon.shogakukan.co.jp/blog/news/60th-shogakukan-manga-award-announce/ 【速報】第60回小学館漫画賞、受賞作決定!!]</ref>。

2017年宣佈改編成[[舞台劇|舞台劇]]及[[劇場版|真人電影]]，角色多由[[乃木坂46|乃木坂46]]成員演出，其中主角東島旭的飾演者，分別由[[齋藤飛鳥|齋藤飛鳥]]與[[西野七瀨|西野七瀨]]擔綱。

== 故事簡介 ==
故事講述二坂高校新生東島旭，標準的文化系女孩，初中時為美術社社員的她，進入高中後，突想嘗試運動性社團，此時學姊一句：「薙刀是高中社團的[[美國夢|美國夢]]！即使不擅長運動的人也能因此轟動全國！」的話語，讓她振奮不已，當即決定加入薙刀社。嚴密的基礎訓練雖令她叫苦連天，但靠着堅毅精神慢慢茁壯成隊中骨幹，與社員們一起朝著全國大賽拚命努力。

作者在連載前沒有任何薙刀相關經驗，主角形象即源自作者本人，作者高中時便隸屬美術社，為取材而花了一年學習薙刀。

=== 弦平高校薙刀社 ===

; {{nihongo|俵里緒|俵 里緒|たわら りお|Tawara Rio}}

; {{nihongo|中村真禮|中村 真礼|なかむら まれ|Nakamura Mare}}
: 弦平薙刀社社員，自認實力不足而推薦俵擔任主將。
; {{nihongo|糸川美保|糸川 美保|いとかわ みほ|Itokawa Miho}}

; {{nihongo|林朱莉|林 朱莉|はやし あかり|Hayashi Akari}}
: 弦平薙刀社社員。

: 出雲英豐薙刀社社員。
; {{nihongo|天羽靜子|天羽 静子|あまは しずこ|Amaha Shizuko}}
