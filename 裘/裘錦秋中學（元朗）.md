'''裘錦秋中學（元朗）'''（{{lang-en|Ju Ching Chu Secondary School (Yuen Long)}}，簡稱{{lang|en|JCCSS (YL)}}），是[[香港|香港]][[新界|新界]][[元朗區|元朗區]]內一所band 2全日制政府津貼男女[[中學|中學]]，於1975年由[[邱德根|邱德根]]先生創辦，前身為'''裘錦秋英文書院（元朗分校）'''，於1994年遷往[[天水圍|天水圍]][[賞湖居|賞湖居]]旁。

== 校訓 ==
裘錦秋中學的校訓是「自強不息」，語出《[[易經|易經]]·乾卦》，原文「天行健，君子以自強不息」，意思是「自然運行，穩健無間之賢人志士，要發憤自勵，不停不棄」。<ref>[http://yl.jccssyl.edu.hk/home/information/motto.htm 學校校訓]</ref>

== 校史<ref>[http://yl.jccssyl.edu.hk/home/information/history.htm 學校官方網站概覽]</ref> ==
* 1966年：本興學辦校、造福社會、作育英才之旨，於[[荃灣|荃灣]]創辦裘錦秋英文書院。
* 1975年及1977年：校董會分別在元朗、[[屯門|屯門]]再開辦兩所中學，即元朗裘錦秋英文書院及屯門裘錦秋英文書院。元朗分校位於安樂路元朗遠東發展大廈內，地下為入口及校務處，1和2樓為課室，美術室，3樓為教員室，圖書室，活動室及小食部。因應校舍細小限制，要分為上下午班上課。
* 1982年：三所中學均轉為政府資助津貼中學。
* 1990年：在[[上海|上海]]開辦上海裘錦秋實驗學校。
* 1994年9月：該校正式命名為裘錦秋中學（元朗）。

== 校舍<ref>[http://yl.jccssyl.edu.hk/home/information/facilities/facilities.htm 校史]</ref> ==
該校校舍座落於[[天水圍|天水圍]][[嘉湖山莊|嘉湖山莊]][[賞湖居|賞湖居]]旁，鄰近[[天水圍公園|天水圍公園]]及[[天水圍運動場|天水圍運動場]]。校舍設[[課室|課室]]30個、輔導教學室3個、科學特別室6個、電腦室4個；另有禮堂、圖書館、標準球場及有蓋雨天操場。

自1999年9月起，校方在禮堂、所有課室、圖書館及各特別室安裝[[冷氣|冷氣]]設備，此外，為改善教學環境，各課室內均裝有「六機一幕」，即[[電腦|電腦]]、[[投影機|投影機]]、[[電視機|電視機]]、[[高映機|高映機]]、[[錄影機|錄影機]]及[[屏幕|屏幕]]，為師生提供更舒適的學習環境。 

校舍於2003年9月開始進行改善工程，經歷探土、圍板、打樁、灌漿等工序後，校舍新翼在2005年5月正式啟用。新教學大樓樓高七層，處於舊翼左側並與舊翼相連，各樓層採用開放式樓梯；新教學大樓外牆以灰白色為主調，配以淺紫；走廊外牆鋪上磁磚。新教學大樓除新建了7個標準「千禧課室」外，更新建各項簇新而先進的設施，其中包括可容納200多人的大型綜合演藝廳——「錦蘭堂」、語言學習中心、[[多媒體|多媒體]]學習中心、天台球場及學生活動中心等。

== 社制 ==
設有四個社別。分別為：
*東（East）－代表顏色為綠色。
*南（South）－代表顏色為紅色。
*西（West）－代表顏色為藍色。
*北（North）－代表顏色為黃色。
