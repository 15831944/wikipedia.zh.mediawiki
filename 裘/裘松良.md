'''裘松良'''（{{bd|1957年|9月}}），[[浙江|浙江]][[富阳|富阳]]人。[[应用数学|应用数学]][[博士|博士]]、教授。曾任[[浙江理工大学|浙江理工大学]]校长、浙江省应用数学研究会理事长<ref>{{cite web |url=http://math.zstu.edu.cn/Content.asp?id=103 |language= |title=校长裘松良教授连任浙江省应用数学研究会理事长 |publisher=浙江理工大学 |date=2011-01-12 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20120730233507/http://math.zstu.edu.cn/Content.asp?id=103 |archivedate=2012年7月30日 |df= }}</ref>和[[浙江大学|浙江大学]]理学院兼职教授。

== 简介 ==
1984年毕业于[[杭州师范学院|杭州师范学院]]数学系。1989年取得[[湖南大学|湖南大学]][[应用数学|应用数学]][[理学博士|理学博士]]学位。1989年6月任教于[[杭州电子工业学院|杭州电子工业学院]]，后来任副校长。2004年11月至2016年6月担任浙江理工大学校长。<ref>{{cite web |url=http://culture.zstu.edu.cn/Art/Art_343/Art_343_664.html |language= |title=裘松良校长一行赴缙云考察 |publisher=浙江理工大学 |date=2010-06-12 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20160304185143/http://culture.zstu.edu.cn/Art/Art_343/Art_343_664.html |archivedate=2016年3月4日 |df= }}</ref><ref>{{cite web |url=http://news.zstu.edu.cn/Detail/?id=20241 |language= |title=校长裘松良率团访问法国、德国、丹麦5所高校 |publisher=浙江理工大学 |date=2012-10-26 |accessdate= }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref><ref >{{cite web|url=http://news.163.com/16/0624/14/BQB5SEQ600014JB5.html | language= | title=浙江任免季晓斌、徐敏俊、曹水萍、劳晓峰、王俭、陈文兴等职务 |publisher= 中国经济网 |date=2016-06-24 |accessdate= }}</ref>

他曾担任[[教育部|教育部]]评估专家组组长<ref>{{cite web |url=http://www1.tyust.edu.cn/yuanxi/yingke/ShowArticle.asp?ArticleID=201 |language= |title=教育部评估专家组组长裘松良教授走访应用科学学院 |publisher=太原科技大学 |date=2006年11月21日 |accessdate= }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

研究领域：拟共形映照及其应用、Ramanujan理论与特殊函数。1996年以来在《[[中国科学_(期刊)|中国科学]]》上发表了2篇论文，有16篇论文被[[SCI|SCI]]、[[EI|EI]]所收录，并被广泛引用。1997年和1999年两度获得[[浙江省|浙江省]]教委科技进步一等奖；并获得1999年浙江省自然科学优秀论文一等奖。
