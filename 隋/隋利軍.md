'''隋利軍'''{{BD|1952年|7月28日}}，[[中华人民共和国|中國]][[音樂家|音樂家]]、[[中國共產黨|中國共產黨]]黨員，[[漢族|漢族]]，[[黑龍江省|黑龍江省]][[雞西市|雞西市]]人，擅長演奏[[嗩吶|嗩吶]]及[[中國傳統音樂|國樂]]的[[作曲|作曲]]，是黑龍江國樂創作代表性人物、中華人民共和國國家一級作曲家<ref>{{Cite journal|title=著名作曲家：隋利军|author=|url=|journal=《北方音乐》|issue=2013年06期|doi=|others=|year=2013|volume=|page=|pmid=}}</ref>，在兩岸樂壇十分活躍。他也是[[滿族|滿族]]音樂理論研究者。<ref name=":0">{{Cite web|url=http://www.folkmusic.org.cn/index.php?s=/home/article/detail/id/429.html|title=隋利军|accessdate=2018-07-08|author=|date=|publisher=中國民族管弦樂學會|language=zh-cn|archiveurl=https://web.archive.org/web/20180708044320/http://www.folkmusic.org.cn/index.php?s=%2Fhome%2Farticle%2Fdetail%2Fid%2F429.html|archivedate=2018-07-08|dead-url=no}}</ref>

隋利軍現任[[中國民族管弦樂學會|中國民族管弦樂學會]]常務理事、中國作曲家專業委員會副主任、黑龍江省民族管弦樂學會會長，[[黑龍江省農業大學|黑龍江省農業大學]]、[[沈阳大学|瀋陽大學]]、[[國立臺南藝術大學|中華民國國立臺南藝術大學]]亦聘其為教授。<ref name=":1">{{Cite web|url=http://www.hljwht.gov.cn/index.php/home/whlj/detail/id/5544.html|title=隋利军|accessdate=2018-07-08|author=|date=2017-07-19|publisher=黑龍江省文化廳|language=zh-cn|archiveurl=https://web.archive.org/web/20180708043929/http://www.hljwht.gov.cn/index.php/home/whlj/detail/id/5544.html|archivedate=2018-07-08|dead-url=no}}</ref>

== 生平 ==
1952年7月28日出生在今[[黑龍江省|黑龍江省]][[雞西市|雞西市]]境內<ref name=":0" />。小學、中學分別畢業於雞西市穆稜煤礦小學、雞西市穆稜煤礦中學。1968年時參與[[上山下鄉運動|上山下鄉運動]]，到[[嫩江县|嫩江县]]山河農場當[[知識青年|知識青年]]。1974年考入[[上海音樂學院|上海音樂學院]]民族器樂系，修習[[嗩吶|嗩吶]]，師從[[任同祥|任同祥]]。1977年畢業後分配到黑龍江省歌舞劇院工作，曾任嗩吶聲部首席、吹管聲部首席。1987年，黑龍江省歌舞劇院公派[[中央音樂學院|中央音樂學院]]作曲系進修，次年隋利軍加入[[中國共產黨|中國共產黨]]。1989年自中央音樂學院畢業後，回到黑龍江省歌舞劇院負責[[作曲|作曲]]。1997年起任黑龍江省歌舞劇院民族樂團團長，2008年升任劇院副院長。2012年，隋利軍退休後又被回聘為專職作曲。<ref name=":1" />

== 榮譽 ==
2002年，隋利軍獲「扎根龍江做奉獻」優秀藝術家稱號。2011年獲第六屆黑龍江省政府十佳文藝工作者稱號。<ref name=":1" />

== 作品 ==
隋利軍自中央音樂學院畢業，回到黑龍江之後，第一首成功作品為[[三弦|三弦]]彈唱[[協奏曲|協奏曲]]《黑土歌》。1990年年初首演時由[[馮少先|馮少先]]擔任彈唱者。在[[臺灣地區|臺灣地區]]首演時，一些跟著[[蔣中正|蔣中正]]來臺的[[榮民|榮民]]還因而感動落淚，思鄉情起。<ref name=":1" />

隋利军主要代表作品有《黑土歌》，[[二胡|二胡]]獨奏曲《小白菜》，國樂合奏曲《關東序曲》、《東北風》，[[笙#鍵笙|三十六簧笙]]協奏曲《薩滿天神之舞》等。<ref name=":1" />
