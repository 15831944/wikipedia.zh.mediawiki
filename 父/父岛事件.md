'''父岛事件'''，又称'''小笠原事件'''，是[[第二次世界大战|第二次世界大战]]末期发生的一起[[日本军|日军]][[食人|食人]]事件。从1945年2月23日至3月25日期间，驻守[[小笠原群岛|小笠原群岛]][[父岛|父岛]]的[[大日本帝國陸軍|日本陆军]]和[[大日本帝國海軍|海軍]]先后杀害了八名关押于要塞中的[[美国海军|美国海军]]飞行员[[战俘|战俘]]，并肢解、烹食了其中五人的肉。食人的目的并非为了缓解[[饥荒|饥荒]]，而仅仅是以“鼓舞士气”为名义的泄愤{{Sfn|稲葉慎|2004|p=26}}。[[日本投降|日本投降]]后，[[美国海军陆战队|美国海军陆战队]]进驻父岛，日军食人真相败露。食人事件的相关军官被[[美国|美国]][[軍事法庭|军事法庭]]作为[[战犯|战犯]]惩处，其中五人处以[[死刑|死刑]]。由于大量事件细节在取证和审判过程中公之于众，父岛事件成为二战日军众多食人[[日本战争罪行|罪行]]中最骇人听闻的案例之一{{Sfn|堀江芳孝|1984|p=134}}。

== 背景 ==
父岛位于[[东京|东京]]东南约{{convert|980|km|mi|abbr=off}}处，是[[小笠原群岛|小笠原群岛]]的主岛，也是除[[硫磺岛|硫磺岛]]外其中最大的一个岛。由于其险要的地理位置和起伏多变的地形，早在1921年，[[日本陆军|日本陆军]]就开始在父岛建设[[炮台|炮台]]等军事设施。[[太平洋战争|太平洋战争]]爆发后，日本加快了父岛军事化的进度，将其打造成一座拱卫[[绝对国防圈|绝对国防圈]]的要塞{{Sfn|エルドリッヂ|2008|p=127}}。除了陆军外，[[大日本帝國海軍|海軍]]也加入到建设工程中，在父岛设置大量海岸炮、高射炮、碉堡，并备有机场和无线电通讯站等。在{{convert|23.45|km2|sqmi|abbr=off}}的岛上，驻扎了六个陆军连队和一个海军航空队，总计超过13,550人{{Sfn|エルドリッヂ|2008|p=131}}。其工事之完善，兵力之密集，可与硫磺岛媲美{{Sfn|Bradley|2008|p=10}}。

随着日军在[[馬里亞納群島及帛琉戰事|馬里亞納群島及帛琉戰事]]的节节败退，[[小笠原群岛|小笠原群岛]]日益成为[[美军|美军]]的攻击目标。由于父岛处于美军从[[马里亚纳群岛|马里亚纳群岛]]至[[日本本土|日本本土]]的空袭路线上，[[中太平洋|中太平洋]]群岛上的日军会将美军的行动通过父岛的无线电中继站转报[[东京|东京]]的大本营，父岛安设的[[高射炮|防空火力]]也对美军轰炸机构成了严重威胁{{Sfn|エルドリッヂ|2008|p=127}}。为此，[[美国海军|美国海军]]舰载机时常从航母上出动，攻击岛上的日军阵地和海上的货船。在交战中，一些美军战机被父岛上猛烈的高射炮火击落，从此音讯全无。其中的大部分飞行员被美军认定为[[阵亡|阵亡]]，但事实上，有少数飞行员坠落于父岛及其附近海域，沦为[[日本军|日军]]俘虏{{Sfn|エルドリッヂ|2008|p=137}}。

和[[新几内亚|新几内亚]]等東南亞日占岛屿粮草寸断、饿殍遍野的情况不同，父岛和日本本土之间的运输线一直处于畅通状态<ref name=":0">{{Cite web|url=http://14.xmbs.jp/ch.php?ID=ryuhpms56&c_num=159278|title=小笠原事件|accessdate=|author=|date=2008|publisher=}}</ref>，因為父島不在美軍跳島戰術路徑之上。所以，直到[[日本投降|日本投降]]，该岛守军都能源源不断地得到蔬菜、肉类甚至[[日本酒|清酒]]供应。但小笠原群岛驻军司令{{Internal link helper/ja|立花芳夫|立花芳夫}}少将对此仍不满足，总希望能有一些特别的“下酒菜”，便心生杀俘取肉的念头<ref name=":0" />。1945年2月，美军开始[[硫磺岛战役|进攻硫磺岛]]，身处{{convert|275|km|mi|abbr=off}}外的父岛守军人人自危，迫近的威胁促使日军将领最终下定决心，赶在美军对父岛有所行动、大难临头之前，将所有战俘全部解决{{Sfn|エルドリッヂ|2008|p=138}}。

== 经过 ==
[[File:Japaneese_War_Trials_A.pdf|缩略图]]
按照立花少将的命令，日军首先处决俘虏中的下级士官，以煽动将士对美军士兵的仇恨{{Sfn|Bradley|2008|p=219}}。1945年2月23日，五天前被俘的两名[[TBF復仇者式轟炸機|TBM復仇者式轟炸機]]飞行员，19岁的马弗·马尔尚（Marvie W. Mershon）和格雷迪·约克（Grady Alvan York, Jr.）少尉在父岛的一处墓地被处决{{Sfn|エルドリッヂ|2008|p=139}}。次日，日军挖出了马尔尚的[[肝脏|肝脏]]，并砍下了他的一条大腿，割出长{{convert|1|ft|cm|abbr=off}}、宽{{convert|4|in|cm|abbr=off}}、重约{{convert|6|pdr|kg|abbr=off}}的瘦肉，包好后送到立花的司令部，和鱼肉、蔬菜、面条等一道做成了丰盛的[[壽喜燒|壽喜锅]]{{Sfn|エルドリッヂ|2008|p=139}}。在场的所有军官都吃得非常尽兴，一醉方休{{Sfn|Bradley|2008|p=217-218}}，立花甚至还要求“再来一碗”{{Sfn|秦郁彦|2005|p=264}}。

这只是一系列食人事件的开始。宴席过后，在中国战场就曾杀人吃肉的[[大日本帝國陸軍|陆军]]第308连队长的场末勇少佐把这件事告诉了[[大日本帝國海軍|海軍]]的吉井静雄大佐{{Sfn|Bradley|2008|p=192}}，后者彼唱此和，决定对海军关押的俘虏也同样处置。这一次经过了事前计划，指定军医前往处决现场，负责解剖遗体，摘出肝脏。2月28日下午，[[新泽西州|新泽西]]出身的19岁美军无线电员詹姆斯·戴伊（James "Jim" W. Dye, Jr.）被带到吉井的司令部，随后被[[斬首|斬首]]。遗体被军医剖开，取出肝脏，经过烹调后由吉井大佐和的场少佐等人分而食之{{Sfn|Hearn|2003|p=139-140}}。其他部分的肉也被日军士兵割下来，煮成了肉汤。

1945年3月14日，[[美军|美军]]宣布[[硫磺岛战役|硫磺岛战役]]胜利结束。作为报复，父岛守军加快了处决美军战俘的进度{{Sfn|Bradley|2008|p=266}}。3月17日，吉井大佐派人以审讯为由，从看护美国战俘的堀江芳孝陆军少佐和[[日裔美國人|日裔美國人]]岩竹信明等人处带走了24岁的[[美国海军|美国海军]]中尉沃伦·沃恩（Warren Earl Vaughn），并当着150名海军将士的面将他斩首{{Sfn|エルドリッヂ|2008|p=137}}{{Sfn|Bradley|2008|p=267-268}}。行刑后，吉井大佐随即让军医松下兼久肢解了沃恩的遗体，肝脏被用来做当天的晚餐，其余部分再次被割肉煮汤喝了{{Sfn|Hearn|2003|p=153}}{{Sfn|エルドリッヂ|2008|p=141}}。

3月25日，24岁的美军飞行员弗洛伊德·霍尔（Floyd Hall）也遭处决。事实上，早在16天前的3月9日，他的命运就被日本人决定了。当天的场少佐发布了一道命令{{Sfn|Bradley|2008|p=269}}：

{{Quote|
一、大隊欲食用美国飞行员霍尔中尉之肉<br>
二、冠中尉應負責此肉之配給<br>
三、坂部軍醫應参加行刑，並摘除肝脏及[[胆囊|胆囊]]<br>
昭和20年3月9日 午前9時 大隊長 陸軍少佐的場末勇}}行刑当日，霍尔中尉被带到警卫室外的一个弹坑边，[[父岛|父岛]]上的卫生兵都奉命到场。的场少佐接连命令三名士兵[[死刑|行刑]]，却都被以“有损名誉”为由拒绝。最终在他的逼迫下，一名姓中村的伍长用[[日本刀|军刀]]砍下了霍尔的头颅{{Sfn|Bradley|2008|p=270-273}}。森国造海军少将随即命令士兵用[[刺刀|刺刀]]对遗体进行刺杀训练。接着，全体卫生兵走上前来，负责解剖的寺木军医切开了霍尔的胸腹，边解剖边向卫生兵们展示心、肺、胃和肠的位置。同时，他把霍尔的肝脏切下来，用[[玻璃纸|玻璃纸]]包好。随后，他开始切霍尔的左腿，砍到[[肌腱|肌腱]]处就换[[剪刀|剪刀]]剪。最后一共得到约{{convert|8|pdr|kg|abbr=off}}的大腿肉，送到的场少佐的指挥所。当天晚上，的场和其他[[大日本帝國陸軍|陆军]]军官设宴招待森少将，将霍尔的肝脏切块用竹签串起来，加上蔬菜和酱油，做成一顿[[日式烤鸡串|日式烤串]]。席间陆海军军官交杯换盏，大快朵颐{{Sfn|Bradley|2008|p=270-273}}。至于拒绝食用的军官，则被的场等人嗤之以鼻，赶出宴席{{Sfn|Hearn|2003|p=122}}。

根据历史学家{{Internal link helper/ja|田中利幸|田中利幸 (歴史学者)}}战后的统计，关押于[[父岛|父岛]]的[[盟军|盟军]][[战俘|战俘]]死亡率高达75%，远远高于日本平均值27.1%{{Sfn|エルドリッヂ|2008|p=153-154}}。只有极少数的飞行员在父岛附近坠落后能幸免于难，其中就包括日后的第41任[[美国总统|美国总统]]，[[乔治·H·W·布什|乔治·H·W·布什]]。1944年9月2日，当时还是中尉的布什驾驶[[TBF復仇者式轟炸機|TBM復仇者式轟炸機]]从[[聖哈辛托號航空母艦|聖哈辛托號航空母艦]]上起飞，执行对父岛夜明山上无线电通讯站的轰炸任务。行动中，他的座机被[[高射炮|高射炮]]击落，机组三人中只有他一人成功跳伞，落入海中{{Sfn|Wise Jr.|Baron|2014|p=11}}。当他打开[[充气船|充气救生筏]]时，看见日军已派出几艘小船正准备抓他。他立刻联想起[[澳大利亚|澳大利亞]]战俘{{Link-en|伦纳德·锡弗利特|Leonard Siffleet}}被日军斩首的那张[[:File:LeonardGSiffleet.jpg|照片]]，便拼命朝远离父岛的方向划。直到三小时后，精疲力尽的他被在附近游弋的美軍{{Link-en|长须鲸号潜艇 (SS-230)|USS Finback (SS-230)|长须鲸号潜艇}}救起，才最终虎口逃生{{Sfn|Galdorisi|Phillips|2009|p=84}}。

== 调查与审判 ==
[[File:G386390.jpg|缩略图]]签署投降书]]
父岛守军终日对[[美军|美军]]即将到来的进攻殚精竭虑，但进攻最终没有发生，因为根据[[跳島戰術|跳島戰術]]，父岛并不在美军的登陆目标内{{Sfn|Hearn|2003|p=23}}。1945年8月15日，日本宣布[[日本投降|无条件投降]]，父岛守军放下武器，悉数向美军投降。9月3日，{{Internal link helper/ja|立花芳夫|立花芳夫}}中将和森国造少将分别代表[[大日本帝國陸軍|日本陆]][[大日本帝國海軍|海軍]]签署投降文书。投降后，[[美国海军陆战队|美国海军陆战队]]上校普雷斯利·里西（Presley Rixey）向日军将领询问当时在父岛落难的美军[[飞行员|飞行员]]的下落。日军一方面谎称他们在美军的空袭中被炸死，另一方面加紧毁尸灭迹的工作{{Sfn|エルドリッヂ|2008|p=145}}。尽管日军将领统一了口徑，但其手下的士官关于战俘问题却闪烁其词，引起了里西上校的怀疑。通过大范围走访和搜索，美军逐渐掌握了日军杀害并分食美军飞行员的犯罪事实。1946年1月1日，在堀江芳孝陆军少佐的协助下，参与食人事件的相关军官被美军悉数[[逮捕|逮捕]]，并在[[关岛|关岛]]成立搜集证据的“调查委员会”和审判的“战犯委员会”，负责进一步检察工作{{Sfn|エルドリッヂ|2008|p=152}}。

1946年8月5日，由[[美国海军|美国海军]]主持的关岛战犯审判开庭，共有25名日军作为被告被起诉。由于没有考虑到现代战争的交战方会有如此野蛮的行径，[[日内瓦公约|日内瓦公约]]中竟没有关于如何惩罚[[食人|食人]]的条款，[[国际法|国际法]]中也没有将食人作为[[战争罪行|战争罪行]]的前例{{Sfn|エルドリッヂ|2008|p=153-154}}{{Sfn|Erickson|1965|p=108}}。于是法庭只能以“杀人”和“毁坏遗体并妨碍其有尊严地下葬”为罪名起诉。被告方先后以“东西道德有别”、“父岛粮食不足”等为由予以辩护，但都被逐一驳回。法庭还出示了[[美国海军陆战队|海军陆战队]]逮捕被告时的证据，证明当时{{Internal link helper/ja|立花芳夫|立花芳夫}}体态肥胖，且日军栖身的洞穴中仍储有大量食材和[[日本酒|清酒]]，粮食缺乏不能自圆其说。最终，关岛法庭宣布所有被告罪名成立{{Sfn|エルドリッヂ|2008|p=153-154}}。立花中将、吉井大佐、的场少佐等五名主犯被判处[[绞刑|绞刑]]。立花中将在判决时被扒光了衣服，只剩一条裤衩，以示羞辱{{Sfn|Bradley|2008|p=386}}。包括森少将在内的五人被判处[[無期徒刑|无期徒刑]]（森国造后来因[[荷蘭東印度群島戰役|荷蘭東印度群島戰役]]中的战争罪行另行审判，交由[[荷兰|荷兰]]绞决）。至于负责砍头和肢解的15名士兵，也全被美军作为[[战犯|战犯]]判处5年至20年不等的[[有期徒刑|有期徒刑]]{{Sfn|エルドリッヂ|2008|p=153-154}}。

==争议==
曾任第二鱼雷艇队[[海军少尉|少尉]]候补生的{{link-ja|土屋公献|土屋公献}}在2008年的一次报告会中表示，他在父岛和沃恩中尉有过数次交谈的机会，在行刑当天他也在场。本来应由他处死沃恩中尉，但最终改由参军学生中一名拥有[[剑道|剑道]]四段资格者执行，土屋则在沃恩附近目睹了行刑的全过程。在POW研究会（戰俘研究会）的采访中，土屋承认有极度饥饿以致“丧失理智”的士兵把沃恩的尸体挖走，但坚称“人肉宴”的说法并非事实。对此，该研究会的泽田猛认为，土屋的说法与现有资料出入较大，仍需结合关岛军事法庭的纪录进行比照研究。但同时他也认同，战后的军事法庭在公正性方面多少会受“{{link-en|成王败寇|Victor's justice}}”思想的影响。{{Sfn|澤田猛|2008}}
