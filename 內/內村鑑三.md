{{Japanese|japanese=内村鑑三|kana=うちむら かんぞう|romaji=Uchimura Kanzō}}
'''内村鑑三'''（1861年3月26日-1930年3月28日），[[日本|日本]][[作家|作家]]、[[基督徒|基督徒]]、[[傳福音|傳教士]]，以及[[明治|明治時代]]及[[大正|大正時代]]的[[無教會主義|無教會主義]]之創始人。

== 早年生活 ==
內村鑑三出生於[[江戶|江戶]]。他從年紀很小時開始顯示出他優秀的語文能力，所以他從11歲開始學習[[英語|英語]]。在1877年，他獲得了[[日本|日本]]第一所以[[英文|英文]]為主要授課語言的學校，[[札幌農學校|札幌農學校]]（即今[[北海道大學|北海道大學]]）之學生的喜訊。

在內村鑑三的到來前，[[威廉·史密斯·克拉克|威廉·史密斯·克拉克]]，一個畢業於[[安默斯特學院|安默斯特學院]]的[[御雇外國人|外籍講師]]，一直向[[日本|日本]]政府申請成立[[札幌農學校|札幌農學校]]。而他最初成立[[札幌農學校|札幌農學校]]的主要用途是傳授農業科技。克拉克是一個致力於通過[[主日學|聖經班]]傳受[[基督教|基督教]]領袖的[[基督教|基督教]][[傳教士|傳教士]]。他的所有學生都宣誓變成了「耶穌的公約的信徒」，並承諾自己繼續學習聖經，也盡最大努力去在他們的生活中履行。克拉克一年後回到[[美國|美國]]。但内村鑑三受到他的影響，決定留下跟他學習。在[[先輩|先輩]]的壓力下，内村鑑三在他一年級時簽了約並在1878年時，即他16歲那年接受了[[衛理宗|衛理宗]]的[[洗禮|洗禮]]。

由於他不滿意教會的使命及立場，首先內村和他的日本朋友合力在[[札幌|札幌]]建立一個獨立的奇特教會。本試驗變成了現在稱為[[無教會主義|無教會主義]]。通過克拉克對他的教誨，這一小群人都相信能夠獨立實踐基督教信念，沒有依賴於一個機構或[[神職人員|神職人員]]。

== 海外生涯 ==
內村鑑三啟程前往[[美國|美國]]不久後，在1884年有一次不愉快的婚姻。他的第一個朋友是[[貴格會|貴格會]]成員莫里斯先生和太太，因為他們在[[賓夕法尼亞州|賓夕法尼亞州]]幫助他找工作。內村鑑三受[[貴格會|貴格會]]的影響而開始認同[[和平主義|和平主義]]。

接下來為8個月在[[賓夕法尼亞州|賓夕法尼亞州]]艾恩一所[[精神病學|精神病醫院]]緊張的工作。但他厭倦了緊張的工作並辭職，在1885年9月穿越[[新英格蘭|新英格蘭]]到[[安默斯特學院|安默斯特學院]]求學。安默斯特學院的校長成了他的精神導師，並鼓勵他進入{{link-en|哈特福德神學院|Hartford Seminary}}就讀。在完成他的第二個本科學位後，他進入了哈特福德神學院，但在一個學期後，便對神學教育失望。他於1888年回到[[日本|日本]]。

== 日本基督教領袖 ==
內村返回日本之後，成為一名教師，但因和控制了學校的當局或外國傳教團體有不妥協的立場而被解僱或被迫辭職。他最有名的不妥協事件，是他在高校的一次正式儀式拒不低頭（即不[[偶像崇拜|偶像崇拜]]）[[明治天皇|明治天皇]]畫像和《[[教育敕語|教育敕語]]》。因為他的宗教信仰不符合日本的教學理念，他又轉向為大眾報紙《[[萬朝報|萬朝報]]》創作，成為資深[[專欄作家|專欄作家]]。內村的名氣使他成為受歡迎的專欄作家，並抗議現代日本的第一次工業污染案件{{link-ja|足尾礦毒事件|足尾鉱毒事件}}。
