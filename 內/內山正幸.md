'''内山正幸'''（{{jpn|j = 内山正幸|hg = うちやま まさゆき|rm = Uchiyama Masayuki|f = y}}，姓名有时会部分或全部写成[[假名|假名]]；{{bd|1952年|3月29日|2010年|8月31日}}），别名'''白鳥剣'''，是一名日本[[動畫師|動畫師]]，从屬於有限会社ラストハウス。内山出生於1952年3月29日，2010年8月31日因心臟衰竭而過世，享年58歲<ref name="anido">[http://www.anido.com/index-j.html アニドウ・ホームページ] {{Wayback|url=http://www.anido.com/index-j.html |date=20101225135800 }}（[http://megalodon.jp/2010-0913-1339-54/www.anido.com/index-j.html ウェブ魚拓]）</ref>。

==簡歴==
內山正幸的職業生涯始於1969年Neo Media Studios。在那裡，他接受東映動畫的製作人木村圭一郎的指導學習。在完成幾部專案作品後，內山離開Neo Media Studios成為一段時間的自由業動畫師，一直到和他在Neo Media Studios結識最要好的朋友之一[[金田伊功|金田伊功]]一起加入スタジオゼット（Studio Z）。內山後來成立了動畫工作室Last House，在那裡他很快成為了主要的動畫領導人。

在20世紀的1980年代和1990年代，他的工作室相當活耀、生產力達到高峰，主要參與了[[東映動畫|東映動畫]]（東映Animation）的作品。在[[宇宙戰艦大和號|宇宙戰艦大和號]]、[[人造人009|人造人009]]、[[銀河鐵道999|銀河鐵道999]]之後，也和漫畫家[[鳥山明|鳥山明]]的改編作品有密切關係。從“怪博士與機器娃娃”到“龍珠”系列的製作長期擔任作畫監督輪替，他71次擔任“龍珠Z”的作畫監督。

內山正幸致力於同時參加大量的動畫作品系列，但同時也影響他的動畫品質。在[[龍珠|龍珠]]製作時，內山享有監督的地位，主因是他的年紀相較於其他主管年長。內山以設計流暢的動畫、速度感的動態圖像而聞名。比起其他動畫師，他的線條特別尖銳硬直且稜角分明，所以也經常被粉絲譴責。

== 動畫小組 ==
內山正幸所指導的動畫團隊往往比內山本人更有效率和才華洋溢，所以他的許多動畫師經常被其他人士關注。內山也經常糾正他底下的團隊來減少工作量，並且使用大量重複的原畫。

在他的監督下看到的任何高於平均水平的動畫，幾乎歸功於在他底下工作的動畫師。和內山最為密切合作的動畫師包括：[[小原太一郎|小原太一郎]]、[[志田直俊|志田直俊]]。
