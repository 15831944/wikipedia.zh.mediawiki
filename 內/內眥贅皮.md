'''內{{zy|眥贅|-{zh-cn:zì zhuì;zh-tw:ㄗˋ ㄓㄨㄟˋ;zh-hk:zaai6 zeoi6;zh-sg:zì zhuì}-}}皮'''亦稱'''内眦皱襞'''，俗稱'''蒙古褶'''，是人類[[上眼瞼|上眼瞼]]眼框內側的皮膚皺摺，常見於[[東亞|東亞]]、[[東北亞|東北亞]]、[[中亚|中亚]]和[[美洲|美洲本土]]民族、部分[[斯堪的纳维亚|斯堪的纳维亚]]和[[波蘭|波蘭]]人，包括[[南非|南非]]的[[科伊桑人|科伊桑人]]和[[馬達加斯加|馬達加斯加]]原住民等，但各人種都有可能出現這個特徵。<ref>Montagu, A. (1989). Growing Young. Bergin & Garvey: CT.</ref><ref name=askd>{{cite web |title=Epicanthus |url = http://epicanthus.askdefine.com/ |accessdate=25 April 2014}}</ref><ref>{{cite web |title = epicanthic fold (anatomy) |url = http://www.britannica.com/EBchecked/topic/189688/epicanthic-fold |work=Encyclopaedia Britannica |accessdate=25 April 2014}}</ref>

內眦眥褶被一些人認為是上眼瞼褶皺的延續，並與鼻梁的高度有一定的關係。很明顯的內眦皺皮常常只在低鼻梁的人群中出現，但低鼻梁的人也可能沒有。

==年龄==
在世界上所有亚洲非亚洲人群中，3到6个月之前的胎儿都有内眦赘皮，还有在任何种族的儿童其鼻根没有发育之前也会存在。东亚人成年之后的内眦赘皮可能与强烈的[[幼态延续|幼态延续]]有关系。<ref>https://jamanetwork.com/journals/jamafacialplasticsurgery/fullarticle/479760

== 病理學 ==
大多數白人沒有此特徵，但在[[胎兒酒精綜合症|胎兒酒精綜合症]]、[[唐氏综合症|唐氏综合症]]、[[貓哭症|貓叫症候群]]、[[橡皮人症候群|橡皮人症候群]]患者身上可發現這個特徵。

== 整形手術 ==
矯正或移除內眥贅皮的手術稱為內眥贅皮整形術，現在這是很罕見的手術。該手術被整形外科醫師認為很有挑戰性，因為內眥贅皮覆蓋在[[淚小管|淚小管]]上<ref>Meronk, F. [http://www.drmeronk.com/asian/asian-eyelid-eplasty.html "Epicanthoplasty".] {{Wayback|url=http://www.drmeronk.com/asian/asian-eyelid-eplasty.html |date=20080513083414 }} Retrieved May 11, 2006.</ref>。
