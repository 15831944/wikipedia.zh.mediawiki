'''內新里'''，位於大里區中央偏東北。

== 里名由來 ==
因境內中較大的聚落「內新」而得名。

== 地理 ==
內新里劃分為33鄰。本里東隔[[大里溪|大里溪]]與[[立仁里_(大里區)|立仁里]]相接，西以[[中興路|中興路]]為界臨[[長榮里_(大里區)|長榮里]]，南為[[中新里_(大里區)|中新里]]，北側為[[日新里_(大里區)|日新里]]。

== 行政區劃沿革 ==
內新里於清代時隸屬藍興堡內新莊；明治35年（1902年）時，改隸臺中廳臺中區內新庄，至大正9年（1920年）則隨行政區調整，隸屬大屯郡大里庄內新大字。二次大戰結束後，國民政府接收臺灣，原先內新大字範圍被劃歸為'''內新村'''與'''東昇村'''，隸屬臺中縣大里鄉。民國59年（1970年），為因應眾多的外來人口，內新村被劃分為'''內新村'''、'''西榮村'''和'''日新村'''；民國82年（1993年）隨大里鄉升格為縣轄市'''大里市'''而改制為'''內新里'''。民國91年（2002年），行政區再次調整，析內新里南側置「'''中新里'''」，範圍就此底定至今。
