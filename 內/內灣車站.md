'''內灣車站'''位於[[台灣|台灣]][[新竹縣|新竹縣]][[橫山鄉_(臺灣)|橫山鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[內灣線|內灣線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[岸式月台|岸式月台]]兩座。

== 營運狀況 ==
*目前為乙種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]；原由[[合興車站|合興站]]管理，後改由[[九讚頭車站|九讚頭站]]管理，現由[[竹東車站|竹東站]]管理。
*行駛車輛為[[台鐵DR1000型柴油客車|DR1000]]型[[台鐵柴油客車|柴油客車]]，等級為[[臺鐵區間車|區間車]]。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。
*目前內灣站在靠山側新增一座岸式月台。

== 歷史 ==
[[File:Turntable_at_Neiwan_Station.jpg|thumb]]]]
*1951年9月11日：設站。
*1989年5月1日：降等為簡易站，由[[合興車站|合興站]]管理。
*2000年3月24日：改由[[九讚頭車站|九讚頭站]]管理。
*2001年9月1日：改由[[竹東車站|竹東站]]管理。
*2011年11月11日：變更為乙種簡易站，仍由[[竹東車站|竹東站]]管理。
*2013年9月30日：啟用多卡通刷卡機。
*2017年：增設轉車盤及觀賞步道。<ref>{{cite news|title=內灣轉車盤完工 「火車調頭」還可上演聲光秀|url=http://news.ltn.com.tw/news/life/breakingnews/2027010|publisher=自由時報|date=2017-04-05|accessdate=2017-10-20}}</ref>
