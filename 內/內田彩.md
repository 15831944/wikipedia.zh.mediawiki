'''内田彩'''（{{bd|1986年|7月23日}}）是[[日本|日本]]的女性[[聲優|聲優]]、歌手及演員。出生於日本[[群馬縣|群馬縣]][[太田市|太田市]]。於《[[仰望天空的少女瞳中的世界|仰望天空的少女瞳中的世界]]》的第八回廣播節目上出道。經紀公司為[[ACROSS_ENTERTAINMENT|ACROSS ENTERTAINMENT]]。

== 經歷 ==
*高中畢業後進入了代代木動畫學院，朝表演的道路上前進。JTB Entertainment第一期生。
*2008年，JTB Entertainment學院在籍中，作為聲優初次亮相於電視動畫《おでんくん》<ref name=qq20150927>{{Cite web |url=http://comic.qq.com/a/20150927/012982.htm |title=内田彩：前几年还一直在从事打工工作 |work=騰訊動漫 |date=2015-09-27}}</ref>。
*2009年，在電視動畫《[[仰望天空的少女瞳中的世界|仰望天空的少女瞳中的世界]]》首次正式演出。同年10月完成了《[[KIDDY_GiRL-AND|KIDDY GiRL-AND]]》電視動畫的主演。
*2012年，以「[[μ's|μ's]] from [[LoveLive!|LoveLive!]]」的名義，參與[[Animelo_Summer_Live|Animelo Summer Live]] 2012 -infinity∞的演出。
*2013年7月21日，舉辦了場名為「内田彩Birthday Party♪」的生日活動。門票在開賣後短時間內就銷售一空，因此後來還追加了額外的門票名額。
*2014年6月1日，在"JAM 2014「アニソンDay」"上，以個人名義，正式的首度發表新曲「アップルミント」（蘋果薄荷）和「-{Breezin'}-」。
*2014年11月10日，被聘為日本群馬縣的「群馬縣觀光代言人」。
*2014年11月12日，發售名為「アップルミント」（蘋果薄荷）的CD專輯，以此正式作為一位歌手出道。
*2015年，在第九屆声優アワード（[[聲優獎|聲優獎]]），作為「μ's」的成員之一獲頒了「歌唱賞」的殊榮。
*2016年4月，正式離開[[JTB_Entertainment|JTB Entertainment]]，移籍至目前所屬的新事務所[[Across_Entertainment|Across Entertainment]]。<ref>http://ameblo.jp/ichigoshiroppu/entry-12148232178.html ご報告</ref>
*2016年8月13日，在日本武道館舉辦《AYA UCHIDA Complete LIVE ～COLORS》演唱會。

== 人物 ==
*在現場直播、演唱會和各種活動中，很經常性的被鏡頭捕捉到喝水的動作。
*成為聲優的契機是小時候十分喜歡《美少女戰士》並想成為水手月亮。<ref name="50问题">{{Cite web |url=http://twi-ani.com/anitter/report/00160.html |title=声優・内田彩さんに50の質問！～声優の輪～ |publisher=ツイアニ |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20130609172149/http://twi-ani.com/anitter/report/00160.html |archivedate=2013-06-09 }}</ref>
*身高156cm。興趣是點心製作、觀賞演唱會、逛街。
*喜歡讀書，在中學時代因為同學遞給她看了「[[声優グランプリ|声優グランプリ]]」這本雜誌，因此知道了聲優這樣的職業。<ref>2014/11/27錄製的[http://www.bilibili.com/video/av1809301/ "内田彩的請給我更多的水 第8回"-10分40秒處]</ref>一直想著「這就像進入故事中一樣是一個美妙的工作」，在調查聲優中，想成為聲優這樣的心情變得更強烈了。<ref name="应援">[http://www.seigura.com/senior/message/message_detail/tabid/126/pdid/uchida01/Default.aspx 応援メッセージ] {{webarchive|url=https://web.archive.org/web/20130629065558/http://seigura.com/senior/message/message_detail/tabid/126/pdid/uchida01/Default.aspx |date=2013-06-29 }} [http://ja.wikipedia.org/wiki/%E5%A3%B0%E5%84%AA%E3%82%B0%E3%83%A9%E3%83%B3%E3%83%97%E3%83%AA 声グラWEB] [http://ja.wikipedia.org/wiki/%E4%B8%BB%E5%A9%A6%E3%81%AE%E5%8F%8B%E7%A4%BE 主婦の友社]</ref>
*受影響比較深的聲優，培訓學校時代開始關係友好的是[[緒方惠美|緒方惠美]]。憧憬的聲優是[[大谷育江|大谷育江]]。<ref name="50问题" />
*十分喜歡唱歌，現階段主要都是角色歌，總有一天會唱自己自身的歌曲，在2014年出了第一張個人的歌曲專輯《アップルミント》<ref name=qq20150927/>。
*喝酒很厲害的人，喜歡的酒是啤酒和紅葡萄酒。
*老家養著一隻名為“キキ”的貓。<ref name="blog1">{{lang|ja|[http://ameblo.jp/ichigoshiroppu/entry-10471864468.html †おきき†], 内田彩オフィシャルブログ「うちだのたまご。」}}</ref>
*喜歡製作點心，如蘋果派、巧克力蛋糕、裝飾蛋糕捲等，製作各式各樣的東西。<ref name="blog2">{{lang|ja|[http://ameblo.jp/ichigoshiroppu/entry-10736113080.html †あみあみ†]内田彩オフィシャルブログ「うちだのたまご。」}}</ref><ref name="blog3">{{lang|ja|[http://ameblo.jp/ichigoshiroppu/entry-11166236032.html もぎゅっと！]内田彩オフィシャルブログ「うちだのたまご。」}}</ref>
*母親曾是壘球選手。<ref>Love Live!μ's First LoveLive!演唱會 Printemps小組片段</ref>
