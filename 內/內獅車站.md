'''內獅車站'''位於[[臺灣|臺灣]][[屏東縣|屏東縣]][[枋山鄉|枋山鄉]]，位於[[加祿車站|加祿]]至[[枋山車站|枋山]]間，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[南迴線|南迴線]]的[[鐵路車站|鐵路車站]]。得名於附近部落內獅村，設立於1992年10月5日，最初為簡易站，並由[[加祿車站|加祿車站]]管理，後於1993年11月1日降為招呼站。內獅車站進出旅客甚少，以2017年全年為例，上車人數150人，下車人數241人，運量每日僅一人左右，目前為臺鐵客運量最低的車站<ref name=":0" />。未來本站將成為臺鐵[[恆春線|恆春線]]的轉運站，計畫運行直通[[新左營車站|新左營車站]]，站體也將改建。<ref>[http://news.ltn.com.tw/news/life/breakingnews/1905687 全台最小車站「轉運」啦！ 內獅不再「一天沒半人」]</ref>

== 車站構造 ==
*[[島式月台|島式月台]]一座，其中一側已無法使用，月台雨遮帶有中國宮廷式建築風格。
*於靠站房側興建新月台，以減少跨越軌道乘車之風險。
*降等為招呼站後，僅保留正線(一股道)，其餘側線及[[轉轍器|轉轍器]]均已拆除。
*本站無天橋及地下道，往來月台必須跨越軌道。
*車站站房無人看守，除機房設備外，無提供售票服務，另設有多卡通刷卡機。

== 營運概況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]。
*停靠狀態：
**[[臺鐵區間車|區間車]]：順行3514次，逆行3501次。
**[[普快車|普快車]]：順行3672次，逆行3671次。
*全台灣乘客數量最少的車站。<ref name=":0">{{cite news|title=最冷門火車站 內獅1天「沒半人」|url=http://www.appledaily.com.tw/appledaily/article/headline/20150507/36536763/|publisher=蘋果日報|date=2015-05-07|accessdate=2015-05-08}}</ref>
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1992年10月5日：[[南迴線|南迴線]]正式營業時設簡易站，並指定由[[加祿車站|加祿站]]管理。
*1993年11月1日：由簡易站降為招呼站。
*2004年10月至2006年3月：共發生三起[[臺鐵南迴線連續破壞事件|南迴線連續破壞事件]]。
*2016年6月28日：啟用多卡通刷卡機。
