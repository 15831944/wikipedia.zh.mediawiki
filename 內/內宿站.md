'''內宿站'''（{{jpn|j=内宿駅|hg=うちじゅくえき|rm=Uchijuku eki|f=y}}）是一個位於[[埼玉縣|埼玉縣]][[北足立郡|北足立郡]][[伊奈町|伊奈町]]內宿台三丁目，屬於[[埼玉新都市交通|埼玉新都市交通]][[伊奈線|伊奈線]]（New Shuttle）的[[鐵路車站|鐵路車站]]。該線的終點。

== 車站構造 ==
[[島式月台|島式月台]]1面2線的[[高架車站|高架車站]]。午間只使用2號線。

== 車站周邊 ==
{{RoughTranslation}}
*伊奈町町制記念公園
*埼玉縣縣民活動總合中心

车站周边的小针内宿规划调整进行的当口，车站为了在这里被建设的住宅区的象被做(制作)了一样的,不过，更地的那样的部分大量地驿前作为闲散。2004年车站周边的整备被做，轮转印刷机整备，公共厕所的重建，长椅等的设备就绪了。轮转印刷机有伊奈町循环巴士「いなまる（伊奈丸）」北循环的车站。

== 历史 ==
*1990年（[[平成|平成]]2年）8月22日 開業。
