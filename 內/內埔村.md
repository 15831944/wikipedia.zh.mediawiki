'''內埔村'''，是[[中華民國|中華民國]][[嘉義縣|嘉義縣]][[竹崎鄉|竹崎鄉]]轄下的行政區。東鄰白杞村，西鄰紫雲村，南鄰昇平村，北鄰義隆村。
面積約2.661平方公里，行政區包含17鄰、577戶、人口數1821人。聚落內的香光寺為該村景點。<ref name="村里鄰人口數">{{cite web |title=竹崎鄉村里鄰人口數 |url=https://chuchi-hro.cyhg.gov.tw/News.aspx?n=1F92FE0C4CBB3D34&sms=52269178FC281255 |website=嘉義縣竹崎戶政事務所官網 |publisher=嘉義縣竹崎戶政事務所}}</ref><ref name="村里介紹">{{cite web |title=內埔村 |url=https://chuchi.cyhg.gov.tw/cp.aspx?n=4329B06A20B5967F&s=A6899105950B1EE1 |website=嘉義縣竹崎鄉公所官網 |publisher=嘉義縣竹崎鄉公所}}</ref>

==地理==
對外交通道路為嘉159號公路，此區域同時是內埔五村十三庄經濟和物產得交易中心。主要的水系為溫水溪與[[濁水溪|濁水溪]]。<ref name="地名資料">{{cite web |title=內埔村地名資料 |url=http://gn.moi.gov.tw/GeoNames/GNMap/map_Admin/map/MapMain.aspx# |website=地名資訊服務網 |publisher=內政部}}</ref>

==聚落歷史==
此區域位居昔日阿里山八社之一「蘆麻產社」轄區和獵場的中央核心地帶，且俗以內為大，故稱為「內埔」。居民以分屬溪洲祖、嶺尾祖和學堂祖的林姓人家後裔居多。<ref name="地名資料" />

==教育==
本區域內目前有昇平國中及內埔國小。<ref name="村里介紹" />
===昇平國中===
昇平國中創立於1970年，設立目的是[[嘉義縣政府|嘉義縣政府]]為便利竹崎鄉內埔、昇平、桃源、白杞、塘興、紫雲、義隆等七個村落之學生就學。嘉義縣政府先由竹崎國中成立內埔分部，同時進行新校區的規劃籌建工作，1971年待新校舍完成後，師生便遷回現址上課。1972年獨立設校，由[[台灣省政府教育廳|台灣省政府教育廳]]定名為「嘉義縣立昇平國民中學」。<ref name="認識昇平">{{cite web |title=認識昇平 |url=http://www.spjh.cyc.edu.tw/ |website=嘉義縣昇平國中官網 |publisher=嘉義縣昇平國中}}</ref>

==文化資源==
===興洲宮===
創立於1702年。主祀神明為[[莆府千歲|莆府千歲]]，農曆九月十七日為莆府千歲誕辰祭典的祭祀活動。<ref>{{cite web |title=興洲宮 |url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/juchi/1014011-XZG |website=文化資源地理資訊系統 |publisher=中央研究院人社中心地理資訊科學研究專題中心}}</ref>

===玉山岩香光寺===
主祀神明為[[釋迦牟尼佛|釋迦牟尼佛]]，創立於1876年，最初名「玉山岩」為供奉[[觀音菩薩|觀音菩薩]]的木造殿堂，1906年更名為「金蘭寺」。1975年經信徒大會同意再更名為「香光寺」
<ref>{{cite web |title=玉山岩香光寺 |url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/juchi/1014011-XGS |website=文化資源地理資訊系統 |publisher=中央研究院人社中心地理資訊科學研究專題中心}}</ref>

===內埔青岩宮===
建立於1906年，因發生[[梅仔坑大地震|梅仔坑大地震]]，當時主要的玉山岩觀音佛祖廟宇倒塌，而被請入民間祭祀。西元1976年重修整建至今。<ref>{{cite web |title=內埔青岩宮 |url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/juchi/1014011-QYG |website=文化資源地理資訊系統 |publisher=中央研究院人社中心地理資訊科學研究專題中心}}</ref>
