'''皎勃東'''（'''Kyaukpadaung'''；{{lang-my|ကျောက်ပန်းတောင်း မြို့}}，{{IPA-my|tɕaʊʔ bə dáʊɴ mjo̰|}}），又譯'''乔克巴当'''<ref>{{cite web |url=http://cul.cn.yahoo.com/ypen/20120404/967677.html |title=中国远征军全面崩溃之谜：或为杜聿明一念之差 |date=2012-04-04 |publisher=[[环球网|环球网]] |accessdate=2012-05-18 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20120522203910/http://cul.cn.yahoo.com/ypen/20120404/967677.html |archivedate=2012年5月22日 |df= }}</ref>，是[[緬甸|緬甸]]中部[[曼德勒省|曼德勒省]]的一個城鎮，也是截至2012年5月時緬甸全國第三大的城鎮。

== 名稱 ==
[[Image:Making_Jaggery.JPG|thumb]]自古以來都是重要的石蜜生產地。]]
「皎勃東」這個名稱由三個[[緬語|緬語]]字組成，分別是：
* 「皎」（Kyauk，{{lang|my|ကျောက်}}）：意思就是石塊；
* 「勃」（Pann，{{lang|my|ပန်း}}）：意思就是花；
* 「東」（Taung，{{lang|my|တောင်း}}）：意思就是籃子；
此外，皎勃東還有「[[石蜜|石蜜]]之城」（{{lang|my|ထန်းညက်}}）的外號。[[石蜜|石蜜]]即[[:en:Jaggery|Jaggery]]，是一種不把[[糖蜜|糖蜜]]與結晶糖分離的[[原糖|原糖]]。位於皎勃東以東10公里的[[波帕山|波帕山]]，自古以來都是重要的石蜜生產地。當地居民會採集一種[[棕櫚科|棕櫚科]]植物的樹液，並以之來提煉石蜜。

==地理==
皎勃東位於[[死火山|死火山]][[波帕山|波帕山]]以西10公里，鄰近都是[[丘陵|丘陵]]地區。

== 交通 ==
皎勃東無論是鐵路或公路在[[第二次世界大戰|第二次世界大戰]]開始不斷完善，有定線車輛開往鄰近城鎮。皎勃東位處於前首都[[仰光|仰光]]至第二大城市[[曼德勒|曼德勒]]的高等级公路上<ref>{{cite web |url=http://mm.mofcom.gov.cn/aarticle/jmxw/200902/20090206033227.html |title=仰光至曼德勒高等级公路建设工程部分完成 |date=2009-02-09 |journal=《[[缅甸之声|缅甸之声]]》周刊 |accessdate=2012-05-18 |language=zh-hans}}</ref>，同時亦通往[[:en:Myingyan|Myingyan]]跟[[:en:Meiktila|Meiktila]]。皎勃東距曼德勒有200多公里，車程約4小时<ref>{{cite news |url=http://dzb.cnpc.com.cn/sygdb/html/2012-04/27/content_1351888.htm |title=“唐嫂”本是纯爷们儿 |date=2012-04-27 |accessdate=2012-05-19 |authors=林培军、周凤珊、罗延智 |language=zh-hans |publisher=[[石油管道报|石油管道报]] }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

== 通信 ==
In 1994, telephone system was upgraded, from Manual Switching Exchange to Auto Telephone System. Local Area Code is 061, it was same as [[Chauk|Chauk]].

== 氣候 ==
皎勃東位處於乾熱地帶，雨量稀少，全年總降雨量少於1000 mm。夏季的氣溫：最高36～39度，最低25～28度；冬季的氣溫：最高30～31度，最低19～23度。波帕地區是緬甸中部的一個綠洲。
