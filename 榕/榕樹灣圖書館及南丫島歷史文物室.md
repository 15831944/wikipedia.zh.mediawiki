'''榕樹灣圖書館及南丫島歷史文物室'''<ref name="指定背景建議（中）"/>（{{lang-en|Yung Shue Wan Library cum Heritage and Cultural Showroom, Lamma Island}}）<ref name="指定背景建議（英）"/>是[[香港|香港]][[離島區|離島區]]的一座建築物，是香港[[民政事務總署|民政事務總署]]社區重點項目計劃（離島區）的工程項目之一（工程計劃編號：61RG）<ref name="指定背景建議（中）"/>，樓高3層，[[總樓面面積|樓面面積]]472[[平方米|平方米]]<ref name="指定背景建議（中）"/><ref name="民政事務總署文件"/>，耗資4,480萬[[港元|港元]]<ref name="民政事務局跟進文件">{{cite web|url=https://www.legco.gov.hk/yr17-18/chinese/fc/pwsc/papers/pwsc20180530pwsc-242-1-ec.pdf|title=工務小組委員會　2018年5月30日的會議　與PWSC(2018–19)15文件相關須跟進的事項|work=[[民政事務局|民政事務局]]|date=2018-06-08|accessdate=2018-12-20|language=}}</ref>。

== 背景 ==
由於人口較少，南丫島在離島區內屬於社區設施較少的地區。原有的南丫島北段公共圖書館位於[[榕樹灣碼頭|榕樹灣碼頭]]附近一座一層高，樓面面積約70平方米的建築物。當時該圖書館每星期開放23小時，並在星期二、五及日休館。另一方面，根據島上的考古發現，南丫島為香港具歷史的景點之一，其歷史可追溯至[[新石器時代|新石器時代]]及[[青銅時代|青銅時代]]。然而，當時島上並無供展覽此歷史景點的場地。<ref name="民政事務總署文件">{{cite web|url=https://www.legco.gov.hk/yr14-15/chinese/panels/ha/papers/ha20150324cb2-1045-5-c.pdf|title=立法會CB(2)1045/14-15(05)號文件|work=[[民政事務總署|民政事務總署]]|date=2015-03-24|accessdate=2018-12-20|language=}}</ref>

為了加強南丫島的圖書館服務，以及提供一個讓大眾認識南丫島的歷史的地點，[[離島區議會|離島區議會]]建議拆卸原有的一層高的建築物，並原址重建一幢三層高的建築物，同時翻新在工程範圍內的戶外地方<ref name="民政事務總署文件"/>；此項目在2013年4月連同「改善梅窩銀礦灣沙灘及附屬設施」一同被通過落實<ref name="民政事務總署文件"/><ref name="星島地區報">{{cite news|url=http://203.80.0.224/pdf_district/20150911/HLD20150911.E02-4C.PDF|title=活化景點打造離島新地標　區議員冀帶動區內經濟（新界區　P.2）|date=2015-09-11|accessdate=2018-12-21|work=[[星島地區報|星島地區報]]}}</ref>，而工程項目的佈局及建築設計則在2014年12月獲得通過<ref name="民政事務總署文件"/>。

2016年12月28日，位於原址樓高一層的建築物的南丫島北段公共圖書館停用，以便進行工程。為維持相關地區的公共圖書館服務，[[康樂及文化事務署|康樂及文化事務署]]於近榕樹灣廣場的地段設立臨時圖書館服務站，以方便當區居民，服務時間則與停用前的南丫島北段公共圖書館相同。<ref name="HKPL">{{cite web|url=https://www.hkpl.gov.hk/tc/library-notices/notice_lmnpl_vkjd47v6qef7sokkkiiohikug5.html|title=南丫島北段公共圖書館暫停開放　服務站繼續提供服務|date=2016-12-28|accessdate=2018-12-21|work=[[康樂及文化事務署|康樂及文化事務署]]|deadurl=no|archiveurl=https://archive.today/20181221153038/https://www.hkpl.gov.hk/tc/library-notices/notice_lmnpl_vkjd47v6qef7sokkkiiohikug5.html|archivedate=2018-12-21}}</ref>

== 樓層及設施 ==
此建築物樓高三層<ref name="headline"/>，地下設有歷史文物室，而在一樓和二樓則設有圖書館<ref name="民政事務總署文件"/><ref name="星島地區報"/><ref name="東方"/>；至於三樓則為機電設施<ref name="東方">{{cite news|url=http://orientaldaily.on.cc/cnt/news/20141213/mobile/odn-20141213-1213_00176_024.html|title=南丫島臨海擬建圖書館|work=[[東方日報_(香港)|東方日報]]|author=陳素貞|date=2014-12-13|accessdate=2018-12-20}}</ref>。

'''南丫島歷史文物室'''<ref name="指定背景建議（中）"/>（{{lang-en|Heritage and Cultural Showroom, Lamma Island}}）<ref name="指定背景建議（英）"/>，是附設於該建築物之地下的歷史文物室，建築面積約71[[平方米|平方米]]<ref name="民政事務總署文件"/>。歷史文物室將展出南丫島的出土文物、以往的家居用具、農具、漁具<ref name="星島地區報"/><ref name="headline"/>、考古發現<ref name="民政事務總署文件"/><ref name="headline"/>及相片<ref name="民政事務總署文件"/>等。歷史文物室乃是由離島區議會、[[古物古蹟辦事處|古物古蹟辦事處]]、[[香港歷史博物館|香港歷史博物館]]和[[香港樹仁大學|香港樹仁大學]]共同籌備<ref name="民政事務總署文件"/>。

== 設計 ==
此建築以「一本關於南丫島的書」為設計概念，外形看似多條長形木條或[[竹簡|竹簡]]排列而成的書<ref name="headline"/><ref name="東方"/>，凸顯其用途之一：圖書館<ref name="東方"/>。此建築亦通過[[中国书法|中国书法]]，於混凝土外牆及側面木條上刻畫南丫島的歷史文化<ref name="東方"/>，而混凝土外牆上刻有宋朝思想家、詩人[[朱熹|朱熹]]所著的詩《觀書有感》<ref name="headline">{{cite web|url=http://hd.stheadline.com/news/columns/0/20190107/730711/|title=﻿Executive日記——南丫島文青新景點 新圖書館刻詩句散發書卷味|work=[[頭條日報|頭條日報]]|author=Kelly Chu|date=2019-01-07|accessdate=2019-01-07}}</ref>。

因應南丫島臨海建築物多以半開放式設計，下層歷史文物室的設計也採用了相同設計，使海成為展品的一部分，與周邊環境融合。<ref name="東方"/>

== 評價 ==
時任[[離島區議會|離島區議會]]副主席[[周轉香|周轉香]]在2015年接受報章訪問時表示，此工程項目重點發展旅遊，能成為推動南丫島當地經濟、帶動就業與改善民生的機會。<ref name="星島地區報"/>
