'''彼得·費希特爾'''<ref name="PDF report">[http://report.nat.gov.tw/ReportFront/report_download.jspx?sysId=C09901400&fileNo=001 出國報告：德國與波蘭相關人權文化設施考察] {{Wayback|url=http://report.nat.gov.tw/ReportFront/report_download.jspx?sysId=C09901400&fileNo=001 |date=20140819084851 }}（[https://webcache.googleusercontent.com/search?q=cache:1pwi2PctGWYJ:report.nat.gov.tw/ReportFront/report_download.jspx?sysId=C09901400&fileNo=001 Google存檔]）</ref>（{{lang-de|'''Peter Fechter'''}}，{{bd|1944年|1月16日|1962年|8月17日}}，一译'''彼得·菲查'''、'''彼得·菲赫特'''、'''彼得·费克特'''、'''彼得·凡西特'''、'''彼得·費查'''），是一位[[德国|德国]]瓦工。18岁时，他与朋友一起企图从[[東德|東德]]翻越[[柏林墙|柏林墙]]前往[[西柏林|西柏林]]。他的同伴成功翻过柏林墙，当他爬到墙的顶部时，被[[东德|东德]]的守卫开枪击中。當時，柏林墙東西兩邊的人民都看到他中槍，并有西方記者在場。西柏林边防军曾投掷急救包<ref name="bjwyw">{{cite web |language = zh-hans |url = http://www.artsbj.com/Html/paimai/shijian_2519_491.html |title = 德国柏林墙：自由与民主 |publisher=北京文艺网 |date=2009-11-13 |accessdate=2010-05-25}}</ref>，但东柏林一面却沒有人敢于施予援手。彼得最终因失血过多死亡。他成为柏林墙建成之后第一个被击毙的“[[叛逃共和国|共和国叛逃者]]”<ref>{{失效連結}}{{cite web|language=zh-cn|publisher=中国选举與政治|title=柏林墙倒塌之后|url=http://www.chinaelections.org/newsinfo.asp?newsid=208059|date=2011-06-05|accessdate=2011-06-17}}</ref>。事件在[[冷戰|冷戰]]時期轟動一時。

== 事件 ==
[[File:Fechter_Mauerkreuz.jpg|200px]]的彼得·費希特爾紀念地十字架]]
[[File:Peter_Fechter_Berlin_Wall_Memorial.jpg|200px]]
[[File:Body_of_Peter_Fechter_lying_next_to_Berlin_Wall.jpg|thumb]]

事件发生在建造柏林圍牆的翌年。当时是18岁的彼得·費希特爾是建造圍牆的工人<ref name="PDF report" />。{{Fact|他想和居住在西柏林的姐妹一同生活，}}于是决定利用在修築隔離牆時翻越柏林圍墙，企圖投奔到西柏林<ref name="PDF report" />。費希特爾和他的朋友Helmut Kulbeik計劃先藏在Zimmer街（{{lang-de|Zimmer Straße}}）附近的一個木匠工作室（離牆很近），在觀察邊境警卫后从窗户跳到主墙和另一条已经开建的平行围栏之间的「死亡地帶」（{{lang-en|death-strip}}）中。他們又穿過「死亡地帶」，翻過2米（6.5[[英呎|英呎]]）高並且上面有铁丝网的柏林圍牆，到达[[查理檢查哨|查理檢查哨]]附近的西柏林。但是两人被东德的警卫发现并被枪击，他的朋友顺利地翻越了2米高的圍牆成功逃到[[西德|西德]]柏林，但還在墙上的費希特爾在数百名目击者注視下被打中骨盆，但他沒有死亡，西柏林的目擊者（包括记者）都看到了他掉在靠近东德的「死亡地帶」。雖然他又爬回到靠近東柏林鐵網圍籬邊求救，卻沒有得到任何一方的醫療救助<ref name="PDF report" />。之后，西柏林民众有人说“做点什么吧！”，西德的警察便向費希特爾投放了急救包，但是没有帮助<ref>[http://www.taiwakobo.de/serie/mauer/litfin/litfin.htm ベルリン@対話工房 2005年11月13日掲載 - ベルリンの見えない壁 忘れられた最初の犠牲者] {{Wayback|url=http://www.taiwakobo.de/serie/mauer/litfin/litfin.htm |date=20160304110954 }}（2004年6月9日） 2011年2月6日閲覧。</ref>。东德警卫、在查理檢查哨工作的士兵，包括[[记者|记者]]在内的群众，只能看着他逐渐虚弱无法做任何事情，直至死亡。在一小时后費希特爾因出血过多死亡，有数百西柏林民衆自发性走到一起示威，齐声怒骂对面的东德是“殺人犯！”<ref name="cnn">[http://articles.cnn.com/1997-03-03/world/9703_03_briefs_wall.death_1_berlin-wall-manslaughter-charges-guards?_s=PM:WORLD Guards on rial for notorious Berlin Wall shooting]{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }} [[CNN|CNN]] World March 03, 1997 2011年2月6日閲覧。{{En icon}}</ref>。

== 紀念 ==
[[File:Fechter_Mahnmal_Nah-Vorderansicht.jpg|alt=彼得·費希特爾的纪念碑]]记载着「他只是想要自由而已。」]]
事件之后，西德的一方附近立起了十字架。{{Fact|[[耶鲁大学|耶鲁大学]]的男声合唱队耶鲁俄罗斯合唱队「{{tsl|en|Yale Russian Chorus|耶魯俄羅斯合唱團}}」受到当时西德柏林市长[[维利·勃兰特|维利·勃兰特]]的招待。事件的第二周，他们在现场附近歌唱了[[莫扎特|莫扎特]]作曲的圣歌《[[圣体颂|圣体颂]]》。一年后，勃兰特在现场献上花环。}}
