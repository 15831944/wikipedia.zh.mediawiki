'''盈中站'''是[[杭州地铁7号线|杭州地铁7号线]]的一个车站，位于[[杭州市|杭州市]][[萧山区|萧山区]]建设四路笑岩线路口东侧。<ref>{{cite web |title=地铁盈中站、坎山站开工围挡、萧山公交开辟暑期亲子线路…这些讯息你一定要了解下 |url=http://www.sohu.com/a/240503339_349176 |website=搜狐 |date=2018-07-11}}</ref>约2021年12月底开放。

== 车站结构 ==
盈中站为地下2层岛式车站，沿建设四路东西向布置。<ref>{{cite web |title=地铁7号线全部站点敲定！萧山占了一半多，经过你家了吗？ |url=http://www.sohu.com/a/210791449_99977203 |website=搜狐 |date=2017-12-15}}</ref>
