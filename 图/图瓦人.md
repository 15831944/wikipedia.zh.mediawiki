'''图瓦人'''（{{mong|m=ᠲᠤᠸᠠ|k=Тува}}；[[圖瓦語|圖瓦語]]：{{lang|ru|Тывалар}}），是一個使用[[突厥語|突厥語]]的民族，[[明朝|明朝]]时为[[兀良哈|兀良哈]]部落，[[清朝|清朝]]时被称作[[阿爾泰乌梁海|阿爾泰乌梁海]]或[[唐努乌梁海|唐努乌梁海]]，该地区现归属于[[俄罗斯联邦|俄罗斯联邦]]的[[图瓦共和国|图瓦共和国]]。现代图瓦民族的主要聚居地也在俄罗斯的图瓦共和国。亦有少量族群分布于[[中华人民共和国|中华人民共和国]][[新疆维吾尔自治区|新疆维吾尔自治区]]北部[[喀纳斯湖|喀纳斯湖]]区域与[[蒙古国|蒙古国]]。他们是半游牧民族（养山羊、绵羊、牛、牦牛、驯鹿、[[駱駝|駱駝]]），住桦树木屋（早期从事渔猎，风俗有滑雪以及乘木马驰冰上），多与同族结婚。

==族源與歷史==
圖瓦人事實上是蒙古[[勿日央亥|勿日央亥]]部落的人，后泛指林中百姓。[[匈奴|匈奴]]統治時期，該地已有少數住民，圖瓦人先世是[[铁勒|铁勒]]的都波（生活在東[[萨彥嶺|萨彥嶺]]），與他們最接近的还有[[科里亞克族|科里亞克族]]與[[楚科奇人|楚科奇人]]等的[[古亞洲人|古亞洲人]]，在[[突厥汗國|突厥汗國]]時代被征服，逐漸[[突厥化|突厥化]]，成為[[木馬突厥|木馬突厥]]之一；[[回紇|回紇]]汗國時代，有部分回紇人被派到這裡，因此圖瓦人也有回紇人的部落。另外二支是彌列哥（[[蔑兒乞|蔑兒乞]]）、餓支。后被[[柯爾克孜人|吉尔吉斯]]人统治。到[[蒙古汗国|蒙古汗國]]時代又被蒙古化（1207年与[[瓦剌|瓦剌]]部长[[忽都合别乞|忽都合别乞]]投降蒙古）。中国境内的图瓦人被官方登记为[[蒙古族|蒙古族]]。元朝有一禿馬部，是林木中百姓，可能也是圖瓦人前身。他们先是元代臣民，在明代他们是[[喀爾喀|喀爾喀]]和[[托輝特部|托輝特部]]臣民。他們生活在[[薩彥嶺|薩彥嶺]]至阿爾泰山。

圖瓦人在[[元朝|元朝]]叫'''圖巴人'''，在元代階級制度中屬於[[色目人|色目人]]，[[速不台|速不台]]與[[者勒蔑|者勒蔑]]也是出自[[兀良哈|兀良哈]]部。一些人認為圖瓦是一个由突厥過渡到蒙古的過渡民族。當時圖瓦人有三支，其中一支是人數極少的'''[[圖法拉爾人|圖法拉爾人]]'''，有数百人分布在[[伊尔库茨克州|伊尔库茨克州]]，他們的語言有[[古都波語|古都波語]]的特色。蒙古人曾统治过他们，他们曾在19世纪末举行过反抗蒙古封建王公的起义。他們在文化、服饰、生活习惯上非常接近蒙古。

有证据显示，现代图瓦人是[[蒙古人|蒙古人]]、[[萨摩耶德人|萨摩耶德人]]以及[[愒人|愒人]]的后裔。消亡了的南部萨摩耶德人，包括马托尔人、科依巴尔人、卡尔马立人等被东部图瓦人同化成为托德金、托法拉尔、索约茨、杜喀等部落。图瓦人伊尔吉特部也被认为是萨摩耶德人的后裔。另外，图瓦语中称呼叶尼塞河的词也被认为是源自[[乌拉尔语系|乌拉尔语系]]萨摩耶德语支。图马特、明伽、蒙古什、萨尔察克等部被认为是源自蒙古人。

== 地理分布 ==
現代圖瓦人分為兩大類：西圖瓦人和托金斯克圖瓦人（Тувинцы-тоджинцы）。 後者住在[[图瓦共和国|圖瓦共和國]]的[[托金斯克區|托金斯克區]]，約佔所有圖瓦人的5％。同時，在布里亞特共和國的也有少數和現代圖瓦人擁有相近語言的民族，他們被稱為奧卡圖瓦人(Oka Tuvans)。

== 語言 ==
图瓦人的母语为[[图瓦语|图瓦语]]，属于[[阿尔泰语系|阿尔泰语系]][[突厥语族|突厥语族]][[西伯利亚语支|西伯利亚语支]]，是保留了较多古突厥语成分的语言，與[[楚瓦什人|楚瓦什人]]有些相似。居住于俄罗斯的图瓦居民使用母语范围较广，形成了用[[斯拉夫字母|斯拉夫字母]]拼写的书面图瓦语，且多兼通[[俄语|俄语]]。居住于中国的图瓦居民使用母语范围相对较窄，仅用于口语，尚无文字，居民多兼通[[哈萨克语|哈萨克语]]、[[蒙古语|蒙古语]]。

== 宗教 ==
不同于其他[[突厥語族|突厥語族]]，图瓦人的宗教信仰多为[[藏传佛教|藏传佛教]]。也有信[[萨满教|萨满教]]。

== 文化 ==
[[File:Tuvans.jpg|thumb]]

图瓦人的文化特色较鲜明地表现在音乐上，他们拥有世界其他民族少见的“[[呼麦|呼麦]]唱法”，即利用[[喉头|喉头]]压力产生不同频率的[[泛音|泛音]]，使得一人能够同时演唱多个声部。图瓦的呼麦歌唱家中，在世界范围内享有盛名的有：[[恒哈图|恒哈图乐队]]、女歌唱家[[珊蔻·娜赤娅克|珊蔻·娜赤娅克]]。图瓦族分为两种类型，一种是山地、平原的[[畜牧业|畜牧业]]者（西部图瓦人）和高山森林中的狩鹿者（东部图瓦人）。他们按以前传统的方式生活：贩盐、[[游牧|游牧]]。
