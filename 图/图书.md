'''图书'''，通常在狭义上的理解是带有[[文字|文字]]和[[图像|图像]]的纸张的[[集合|集合]]。书通常由[[墨水|墨水]]、[[纸张|纸张]]、[[羊皮紙|羊皮紙]]或者其他材料固定在书脊上组成。组成书的一张纸称为一张，一张的一面称为一[[页|页]]。但随着[[科学|科学]][[技术|技术]]的发展，狭义图书的概念也在扩展，制作书的材料也在改变，如电子格式的[[电子书|电子书]]，阅读载体也出现了专门的[[电子书阅读器|电子书阅读器]]。

从广义理解的图书，则是一切传播[[訊息|訊息]]的[[传播媒体|媒介]]。书也指文学作品或者其中的一部分。在[[图书馆信息学|图书馆信息学]]中，书被称为{{link-en|专著|Monograph}}，以区别于[[杂志|杂志]]、[[学术期刊|学术期刊]]、[[报纸|报纸]]等[[连载期刊|连载期刊]]。<!--{{TransH}}The body of all written works including books is literature.{{TransF}}-->所有的书面作品（包括图书）的主体是[[文学|文学]]。在[[小说|小说]]和一些类型（如[[传记|传记]]）中，书可能还要分成[[卷|卷]]。对书特别喜爱的人被称为[[爱书者|爱书者]]或[[藏书家|藏书家]]，更随意的称呼是[[书虫|书虫]]或者[[书呆子|书呆子]]。

买书的地方叫[[书店|书店]]，[[图书馆|图书馆]]則是可以借閱書籍的地方。2010年，[[谷歌公司|谷歌公司]]估计，从[[印刷术|印刷术]]发明至今，大概出版了一亿三千万本不同書名的书籍。<ref>{{cite web |url = http://booksearch.blogspot.com/2010/08/books-of-world-stand-up-and-be-counted.html |date=2010-08-05 |title=Books of the world, stand up and be counted! All 129,864,880 of you.|accessdate=2010-08-15 |quote=After we exclude serials, we can finally count all the books in the world. There are 129,864,880 of them. == 現代的圖書定義 ==
[[聯合國教科文組織|聯合國教科文組織]]對圖書的定義是：凡由出版社（商）出版的不包括封面和封底在內49頁以上的印刷品，具有特定的書名和著者名，編有[[國際標準書號|國際標準書號]]，並有定價並取得版權保護的出版物稱為圖書。

圖書需要一個定義的原因是為了方便聯合國教科文組織進行國際間的圖書發行統計比較，1964年11月在法國巴黎舉行的會議上，提出並通過採用以下定義：圖書是不包括封面和封底在內，至少含49頁的非期刊類印刷品，並且於該國家公開予公眾<ref>{{cite web | url = http://portal.unesco.org/en/ev.php-URL_ID=13068&URL_DO=DO_PRINTPAGE&URL_SECTION=201.html | title = Recommendation concerning the International Standardization of Statistics Relating to Book Production and Periodicals | accessdate = 2011-01-16 | date = 1964 | publisher = UNESCO }}{{en}}</ref>。

==书的历史==


===古代的书===
在远古[[文明|文明]]创造了[[书写系统|书写系统]]之后，文字會刻在石头、[[粘土|粘土]]、树皮、金属片上。碑文或者铭文这类形式占据了历史的绝大部分，這類的研究稱為[[金石学|金石学]]。5千年前埃及開始使用[[字母的歷史|字母]]。古埃及人在[[莎草紙|莎草紙]]上书写，莎草这是一种生长在尼罗河边的植物，这种用[[莎草纸|莎草纸]]制成的书也是至今为止发现最早的书。最初词语并不分开，也没有[[标点|标点]]；书写顺序有时候从右到左，有时候从左到右，甚至轮流着来。这种轮流书写的方式被称为[[牛耕式轉行書寫法|牛耕式轉行書寫法]]，就像农夫用牛在耕地一样。

====平板书====
平板书是用坚固耐用的书写介质，适用于传播和书写，但是很重。

{{link-en|泥板|clay tablet}}，顾名思义，是平整、乾燥的泥片，易于携带，使用尖笔（可能是湿的）刻印。从[[青铜时代|青铜时代]]直到[[铁器时代|铁器时代]]，它都是书写的介质，上面會用[[楔形文字|楔形文字]]書寫。

{{link-en|蜡板|Wax tablet}}是在木板上涂有足够的腊，可以在上面用{{link-en|尖笔|stylus}}书写。它们在学校、会计、做笔记时都是常见的书写材料。蜡板有着可重复利用的优点，腊可以融解后重塑。把一些蜡板绑在一起可能就是现代书的原型之一。<ref>Leila Avrin. ''Scribes, Script and Books'', p. 173.</ref>。

==== 卷軸 ====

[[莎草紙|莎草紙]]是薄的類似[[紙|紙]]的材質，是由莎草的莖編織，再用類似錘子的工具重擊而成。一而成，莎草紙是[[古埃及|古埃及]]書寫的工具，可能在[[埃及第一王朝|埃及第一王朝]]時就已開始，目前發現最早的莎草紙書是[[埃及第五王朝|埃及第五王朝]]時Nefertiti Kakai法老時的會計簿，約西元前2400年<ref>{{Cite book

根據[[希羅多德|希羅多德]]的記載（《[[歷史_(希羅多德)|歷史]]》5:58），[[腓尼基|腓尼基]]人在西元前9至10世紀將莎草紙帶到埃及。希臘文中作為書寫材料的莎草紙（''biblion''）及書（''biblos''）兩個字都是來自腓尼基的港口[[朱拜勒|朱拜勒]]，原因是因為莎草紙是由朱拜勒進入希臘的<ref>Leila Avrin. ''Scribes, Script and Books'', pp. 144–145.</ref>。希臘文中有另外一個字tome ({{lang-el|τόμος}})，最早是指片或塊，後來也用來指「一卷莎草紙」，後來拉丁文也開始使用<!--with exactly the same meaning as ''volumen'' (see also below the explanation by Isidore of Seville).-->

不論是材質是莎草紙、[[羊皮紙|羊皮紙]]或是紙張，卷軸是古希臘、羅馬、中國、希伯來及馬其頓文化中主要的書籍形式。後來更現代的<!--{{link-en|抄本|codex}}-->codex形式書籍在[[古典时代晚期|古典时代晚期]]進入羅馬世界，但卷軸在亞洲的應用維持了更長的時間。

====後來的書====
到公元1世纪时，[[希腊|希腊]]和[[罗马|罗马]]用动物的皮来记录国家的法律、历史等重要内容，和[[中国|中国]][[商朝|商朝]][[甲骨文|甲骨文]]一样都是古代书籍的重要形式。在印刷术发明前，书的复制都是由手工完成，其成本与人工都相当高。在中世纪时期只有少数教会、大学、贵族和政府有着书籍的应用。直到10世纪[[毕昇|毕昇]][[印刷术|印刷术]]的发明，书籍才作为普通老百姓能負擔的物品，从而得以广泛的传播。进入1990年代，随着网络普及，书已摆脱了纸张的局限，[[电子书|电子书]]又以空间小、便于传播、便于保存等优势，成为未来书的发展趋向。

== 現代圖書的製作 ==
[[File:Urval_av_de_bocker_som_har_vunnit_Nordiska_radets_litteraturpris_under_de_50_ar_som_priset_funnits_(2).jpg|thumb]]中重要的一部份，特別是封面設計。當書被放在一疊，彧是放在書架上時，書脊是書唯一會露出，可以讓人看到書籍相關資料的部份，在書店裡，常常是書脊上的一些細節先吸引購書者的眼光]]
{{Main|出版}}

書籍[[印刷|印刷]]及{{link-en|装订|Bookbinding}}的方式自十五世紀到二十世紀初都沒什麼太大的改變。當[[生產機械化|生產機械化]]更加盛行時，1900年代的[[印書機|印書機]]（book printer）和[[約翰尼斯·古騰堡|約翰尼斯·古騰堡]]發明的機器越來相近。

古騰堡發明的[[印刷机|印刷机]]使用可以移動的金屬活字組成單字，在成行成頁後再透過[[凸版印刷|凸版印刷]]大量印刷。

現代書籍用紙會用一些專門為印書設計的紙。傳統上，圖書會用低白度的紙張印刷（容易閱讀），紙張會選用不透光的，減少從紙的這一頁可以看到這頁另一面內容的情形<!--and are (usually) made to tighter [[caliper|caliper]] or thickness specifications, particularly for case-bound books.-->。依圖書形式的不同，選用紙的品質也會有所不同，多半的書會用銅版紙、{{link-en|模造紙|woodfree uncoated paper}}、[[光面纸|光面纸]]<!--and [[special_fine_paper|special fine paper]]s-->等。

現代書籍主要是由[[平版印刷|平版印刷]]的方式印製，書籍在印刷時，書籍的數頁會印在同一張的大型紙張上，並且位置經過設計，在摺疊之後，各頁就會依正確的順序排列。現代的書傾向以一些標準的尺寸印刷，{{link-en|書籍尺寸|Book size}}也就是書籍各頁摺疊後裁切的尺寸。由標準書籍尺寸從200至300年前就開始流行，後來成為出版業的標準。除美國以外的英語國家在此領域會使用英制，歐洲的出版商又使用另外一套的標準。

===現在的製作流程===

有些印製數量較少的圖書會用單張紙膠印機印製，其他大部份的圖書會用捲筒紙印刷機印製，進料是用捲筒紙，可以在較短時間印製較多的複本。在生產完成後，一本書的書頁會疊成一疊，後續裝訂成書。

== 尺寸 ==
[[File:Codex_Gigas_facsimile.jpg|thumb]]的複刻本]]
[[File:Kyauksa.JPG|thumb]]
現代書的尺寸是依平版印刷機的印刷區域為準，書籍的幾頁會安排在同一個印刷機的大版面中，因此可以完整的利用版面，而位置及順序也經過調整，在摺疊及裁切後可以依正確的順序出現。

最常見的尺寸有
* 四開（quarto,4to）：印刷的大紙張會摺疊二次，會有四張紙共八頁，最長邊約11-13英吋（約30公分）。
* 八開（octavo,4to）：最常見硬皮書的尺寸，印刷的大紙張會摺疊三次，會有八張紙共十六頁，最長邊約9 ¾英吋（約23公分）。
* 十二開（duoDecimo,12mo）：介於四開及八開之間的尺寸，最長邊約7 ¾英吋（18公分）。
* 十六開（sextodecimo,16mo）：印刷的大紙張會摺疊四次，會有十六張紙共三十二頁，最長邊約6 ¾英吋（約15公分）。

比十六開更小的尺寸有：
* 二十四開：最長邊到5 ¾英吋（13公分）
* 三十二開：最長邊到5英吋（12cm）
* 四十八開：最長邊到4英吋（10cm）
* 六十四開：最長邊到3英吋（8cm）

世界上現存最大的中世紙手抄本是[[魔鬼聖經|魔鬼聖經]]，尺寸是92 × 50 × 22&公分。世界上最大的書是石製的，在[[缅甸|缅甸]]的Kuthodaw Pagoda。

== 圖書的識別及分類==
20世紀起，[[圖書館員|圖書館員]]開始關注如何追蹤每年出版的圖書，[[國際圖書館協會聯合會|國際圖書館協會聯合會]]發展了一系列相關的工具，包括{{link-en|国际标准书目著录|International Standard Bibliographic Description}}（ISBD）。

[[File:ISBN.JPG|thumb]]及[[條碼|條碼]]]]

每本書都有唯一的[[国际标准书号|国际标准书号]]（ISBN），[[出版社|出版社]]在每本書出版時都會申請，若是改版也會申請新的編號。国际标准书号是由<nowiki>ISBN</nowiki>協會管理，包括四部份，分別是國家碼、[[出版社|出版社]]碼、標題碼及[[校验码|校验码]]，校验码可以是0–9及X（10）。[[欧洲商品编码|EAN]]針對書籍的[[條碼|條碼]]由依国际标准书号，前面再加上978，再計算一個新的校验码。

工業化國家的商業出版社一般會針對出版書籍指定ISBN，因此讀者可以假定其ISBN是在國際系統中的一部份，沒有例外。不過許多工業化國家或開發中國家的政府出版品未完全參與ISBN系統，因此出版品不一定會有ISBN。

大型或公開的書籍收藏（如[[圖書館|圖書館]]）會需要{{link-en|图书目录|Library catalog}}，會針對每一本書安排索書號（或檢書），並根據索書號規劃書籍所放的書架。一般索書號會依[[圖書分類法|圖書分類法]]來規劃。索書號會貼在圖書館書籍的书脊下方，但會離底部有一小段距離，也會貼在書籍的內頁。

[[File:SanDiegoCityCollegeLearningResource_-_bookshelf.jpg|thumb]]支撐，在书脊上可看到索書號]]

像是[[美國國家標準協會|ANSI]]/NISO Z39.41 - 1997之類的國家標準，設定了在书脊列出書籍資訊（如{{link-en|書名|Title (publishing)}}或是作者）的正確方式，也設定了像[[DVD|DVD]]、[[錄影帶|錄影帶]]及[[軟體|軟體]]等類似書可以上架存放物品的資訊標示方式。

[[杜威十进制图书分类法|杜威十进制图书分类法]]是一個最早也最為人知的常用圖書分類法，另一個最用的系統是用英文字母識別的[[美國國會圖書館圖書分類法|美國國會圖書館圖書分類法]]。兩個系統都偏重當時分類法發展時，在美國書書館已有較多藏書的內容，因此在針對像電腦或是有關其他文化的主題時會比較麻煩{{Citation needed|date=2015-03}}。

書籍的[[元資料|元資料]]可以包括其[[国际标准书号|国际标准书号]]（ISBN）或其他分類號碼、貢獻者（[[作者|作者]]、[[編輯|編輯]]、譯者、插圖製作者）及出版商、書籍日期及尺寸、以及書籍的語言等。

== 用途 ==
書的主要用途是[[閱讀|閱讀]]，此外也可以有其他的用途：
* 書可以是藝術家的創作，本身就可以是一件藝術品，這稱為{{link-en|藝術家的書|artists' book}}。
* 書可以讓讀者或是專業作者來評論，評論的文字稱為[[書評|書評]]。
* 書可以讓一群人一起閱讀，之後進行業餘及學術的討論，稱之為{{link-en|讀書會|book discussion club}}。
* 書可以讓學生閱讀後分析，並撰寫有關其內容分析的文章，稱為{{link-en|讀書心得|book report}}
* 書有時可以用來作為家中的[[裝飾藝術|裝飾藝術]]，例如在[[書房|書房]]中。

== 图书的种类 ==
===按內容分===
[[杜威十進位圖書分類法|杜威十進位圖書分類法]]是由美國圖書館專家麥爾威·杜威發明，將書分為十大類，中華民國圖書館學學者[[賴永祥|賴永祥]]以之為本、修訂的[[中文圖書分類法|中文圖書分類法]]被中華民國、香港、澳門、新加坡等地大多數圖書館所採用。

通行於歐美許多大學圖書館的[[美國國會圖書館圖書分類法|美國國會圖書館圖書分類法]]以26個拉丁字母中的21個，將書區分為21種類；傳至中國後、改為[[中國圖書館分類法|中國圖書館分類法]]，以26個拉丁字母，將書區分為26種類。在中國古代使用的是[[四部分類法|四部分類法]]，將書分為經、史、子、集四類。

而[[通用十進位圖書分類法|通用十進位圖書分類法]]是19世紀末在杜威十進位圖書分類法的基礎上繼續研發的分類方法，此種分類方法由數字和特殊符號組成，在世界各地的圖書館中被廣泛應用。
