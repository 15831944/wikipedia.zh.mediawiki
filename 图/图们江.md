'''图们江'''（{{kor|名=北|諺=두만강|漢=豆滿江|羅=Tumangang|f=y}}，{{Lang-rus|Туманная река|Tumannaya reka}}，{{lang-mnc|ᡨᡠᠮᡝᠨ<br />ᡠᠯᠠ|z=tumen ula}}），发源于[[长白山|长白山]]东南部的[[长白山天池|长白山天池]]<ref name=times1006>{{Cite news|author= Onishi, Norimitsu |title=Tension, Desperation: The China-North Korean Border |date=2006-10-22 |newspaper=[[The_New_York_Times|The New York Times]]|url=http://www.nytimes.com/2006/10/22/weekinreview/22marsh.html |language=en}}</ref>，干流全长500多公里，注入[[日本海|日本海]]。上游是[[中华人民共和国|中国]]与[[朝鲜民主主义人民共和国|朝鲜]]的界河，下游15公里为朝鲜与[[俄罗斯|俄罗斯]]的界河。<ref name=hc>{{cite web|title=图们江简介|url=http://www.hcmarathon.com/news/view-id8.html|website=2016珲春图们江国际马拉松旅游节|accessdate=2016-11-11|deadurl=yes|archiveurl=https://web.archive.org/web/20161112021505/http://www.hcmarathon.com/news/view-id8.html|archivedate=2016-11-12}}</ref>

== 名稱 ==
图们江（{{朝|두만강|豆滿江|Duman-gang}}，{{Lang-ru|{{Mousetext|Туманная река|Tumannaya reka}}}}、图曼纳亚河，{{Lang-ja|{{Mousetext|豆満江|Tomankō}}}}、[[平假名|平假名]]：{{Lang|ja|{{Mousetext|とまんこう|Tomankō}}}}），在[[金朝|金朝]]、[[元朝|元朝]]称“爱也窟河”，其下游段称“统门河”；[[明朝|明朝]]称“啊也苦河”，其下游段称“徒门河”；在[[清朝|清朝]]時按[[滿語|滿語]]讀音譯作“土門江”。

“土门”满语原称“土门色禽”（{{lang-mnc|ᡨᡠᠮᡝᠨ<br />ᠰᡝᡴᡳᠶᡝᠨ|z=tumen sekiyen}}），土门意为“万”，色禽意思是“河源”意思就是「万水之源」。其實“土門”、“豆滿”、“圖們”都是一音之轉<ref>{{cite journal|author1=于逢春|title=图们、土门与豆满、豆漫之词源与译音考|journal=中国边疆史研究|date=2009年6月|volume=第19卷|issue=第2期|url=http://www.nssd.org/articles/article_read.aspx?id=30888203|accessdate=2016-11-14}}</ref>。

[[朝鮮王朝|朝鮮王朝]]《[[新增东国舆地胜览|新增东国舆地胜览]]》卷五十咸镜道庆源都护府豆满江（图们江）：“[[女真语|女真语]]谓万为豆满，以众水至此合流，故名”。後來中國方面改用與原來讀音相近，但沒有意義的“圖們”來作為這條河流的譯名；而朝鮮半島方面則使用“荳滿江”這個名稱<ref>{{cite book|author1=崔洪彬,|coauthors=全信子|title=图们江的诉说: 朝鲜族|date=2001|publisher=云南大学出版社|url=https://books.google.de/books?id=pUh5AAAAIAAJ&hl=zh-CN|accessdate=2016-11-12}}</ref>。這裡的民族有崇拜水精的習慣：[[金蛙王|金蛙王]]與[[努爾哈赤|努爾哈赤]]也與此河有關。<ref>崔羲秀.朝鲜民族传说中的努尔哈赤[Ｊ].艺术世界,1993(1).</ref>

== 干流 ==
图们江，辽代称驼门。金、元称爱也窟河，其下游段称统门河。明代称啊也苦河，其下游段称徒门河。清代称河源段为大浪河，图们市以下则叫统门水，也叫土门江。康熙年间称土门，清高宗（乾隆） 钦定《金史语解》中称图们。1962年中朝两国协定，“红土水”与溺流水以下称图们江。图们江在清朝时按满语读音译作“土门江”。

据《中国古今地名辞典》记载：“土门”其实并非汉语，也不是朝鲜语，“土门”满语原称“土门色禽”，土门意为“万”，色禽意思是“源”、“河源”，“图们色禽”亦即万水之源， 后来略去“色禽”，简称“土门”。可是，后来日本借“土门”这两个汉字词挑起朝鮮和清朝之间的领土争端，指清朝非法侵占朝鮮的土地。事件间接引致朝鮮独立，并使日本有机会进一步侵占朝鮮的利益。与此同时，中国方面改用与原来读音相近，但没有意义的“图们”来作为这条河流的译名。
朝鮮语称豆满江，实是图们江直译音。朝鲜李朝《新增东国舆地胜览》卷五咸镜道庆源都护府豆满江（图们江）：“女真语谓万为豆满，以众水至此合流，故名”。长白山：“女真语谓善颜阿林，以白色的山峰，故名，胡（女真）人奉之为神山。”《李朝肃宗实录》记载，十七年十一月丙寅：“长白山，胡人（满族人）或称白头山，以长白故也”。<ref>{{cite news|title=龙井县水利志}}</ref>

== 地理 ==
图们江发源于[[长白山|长白山]]东南部，全长505公里，其中490公里为中朝边界，最下游15公里为俄朝边界<ref name=nie>{{cite journal|author1=聂作平|title=图们江口走向大海的梦|journal=中国国家地理|date=2007年|issue=第05期|url=http://www.dili360.com/cng/article/p5350c3d690a7f52.htm|accessdate=2016-11-11}}</ref>。图们江流向东北又折向东南，其干流流经[[和龙|和龙]]、[[龙井市|龙井]]、[[图们|图们]]、[[珲春|珲春]]四市。图们江在珲春市[[敬信镇|敬信镇]][[防川|防川村]][[土字牌|土字牌]]（东经130度42分，北纬42度17分处）出中国境。<ref name=osgeo>{{cite web|author1=zhangxiang|title=中国图们江自然地理概况及气候特征|url=http://www.osgeo.cn/post/fe5b4#_2|website=osgeo中国|accessdate=2016-11-12|date=2016-10-24}}</ref>

在中朝邊境的一段，河的北岸是[[中国|中国]][[吉林省|吉林省]]的[[延邊朝鮮族自治州|延邊朝鮮族自治州]]，南岸是[[朝鮮|朝鮮]][[咸鏡北道|咸鏡北道]]、[[两江道|两江道]]。<ref name=osgeo/>

== 經濟开发 ==
[[File:Tumen_River_Bridge.jpg|缩略图]]]]
1992年，在[[中国|中国]]、朝鲜、[[韩国|韩国]]、[[俄国|俄国]]和[[蒙古国|蒙古国]]的共同建议和参与下，[[联合国开发计划署|联合国开发计划署]]制定了图们江地区发展规划，对圖們江沿岸進行積極開發。中国已在位于图们江下游的[[琿春|琿春]]市设立珲春边境经济合作区；朝鲜也将靠近图们江的[[罗津|罗津]]和[[先锋郡|先锋]]两市划为[[罗先直辖市|罗先直辖市]]，指定其为[[经济特区|经济特区]]，引进外资加以开发。图们江上有公路桥、铁路桥连接朝鲜和中国<ref>{{cite web|author1=四海漫游|title=一衣带水的中朝图们江大桥|url=http://travel.sohu.com/20150507/n412566224.shtml|website=搜狐旅游|accessdate=2016-11-12|date=2015-05-07}}</ref>。图们江河口附近有连接朝鲜铁路和俄罗斯[[西伯利亚铁路|西伯利亚铁路]]的「豆满江站」，由于朝俄两国间的铁路[[轨距|轨距]]不相同，旅客列车需在该站更换轉向架，货物列车则在该站进行货物换装。<ref>{{cite journal|author1=趙成儀|title=中國大陸參與東北亞區域經濟合作 的嘗試─圖們江流域開發計畫 的現況與前景|journal=展望与探索|date=2008年3月|volume=第6卷|issue=第3期|url=https://nccur.lib.nccu.edu.tw/bitstream/140.119/73628/1/PRCParticipates.pdf|accessdate=2016-11-12}}</ref><ref>{{Cite web |language=en |title=Tumen River Area Development Program |year=2009 |publisher=Network of East Asian Think-tanks (NEAT) |url=http://www.neat.org.ph/index.php?option=com_content&view=article&id=247:tumen-river-area-development-program&catid=76:subregional-cooperation&Itemid=130 |archiveurl=https://www.webcitation.org/6VbV9Yrxy?url=http://www.neat.org.ph/index.php?option=com_content |archivedate=2015-01-15 |deadurl=yes |access-date=2016-11-12 }}</ref><ref>{{Cite news|author=Kim, Myung-sung |script-title=ko:최고 싱크탱크 도 "두만강 지역 개발하자"|date=2015-01-14 |language=ko|newspaper=朝鲜日报|url=http://news.chosun.com/site/data/html_dir/2015/01/14/2015011400213.html |archiveurl=https://web.archive.org/web/20150115173444/http://news.chosun.com/site/data/html_dir/2015/01/14/2015011400213.html |archivedate=2015-01-15 |deadurl=no}}</ref>

2009年11月，[[中国国务院|中国国务院]]发布了《中国图们江区域合作开发规划纲要以长吉图为开发开放先导区》，这是国内首个沿边区域经济发展的国家战略规划<ref>{{cite web|title=中国图们江区域合作开发规划纲要(全文)|url=http://chinaneast.xinhuanet.com/2009-11/17/content_18251163.htm|website=新华网|publisher=中国吉林网|accessdate=2016-11-12|date=2009年11月17日|deadurl=yes|archiveurl=https://web.archive.org/web/20170108140511/http://chinaneast.xinhuanet.com/2009-11/17/content_18251163.htm|archivedate=2017年1月8日}}</ref>。

== 划界问题 ==
{{main|中朝边界|朝俄边界}}
[[File:国际客运列车在图们江铁路桥上行驶，离开朝鲜进入俄罗斯_-_panoramio.jpg|缩略图]]]]
1712年（[[康熙|康熙]]53年），中朝两国首次确定以图们江水为界<ref name=hc/>。1860年（[[咸丰_(年号)|咸丰]]十年）[[清政府|清政府]]在《[[中俄北京条约|中俄北京条约]]》中把图们江最后15公里的北岸划归[[俄羅斯帝国|俄羅斯帝国]]<ref name="Boundary">{{Cite journal|first={{lang|ru|В. И.}}|last={{lang|ru|Головнин}}|script-title=ru:Прошлое как оружие|journal={{lang|ru|Россия в глобальной политике}}|volume=6|number=35|year=2008 |url=http://www.globalaffairs.ru/numbers/35/10838.html |archiveurl=https://web.archive.org/web/20101206062610/http://www.globalaffairs.ru/number/n_12054 |archivedate=2010-12-06|deadurl=no|language=ru}}</ref>，俄罗斯至此与朝鲜成为陆上邻国，中国也失去了日本海的出海口<ref>[http://news.xinhuanet.com/fortune/2006-06/12/content_4683984.htm 踏访图们江出海口]</ref>。1886年，清政府御史[[吴大澂|吴大澂]]与俄签订《[[s:中俄珲春东界约|中俄珲春东界约]]》，成功争取到中国船只图们江出海权<ref>[http://www.ecdc.net.cn/newindex/chinese/page/tumen/tumen_jishi/010/2.htm 关于我国船只经图们江口通海航行权利的历史、现实和出路]</ref>。1909年9月4日，[[清朝|清朝]]和[[日本帝国|日本帝国]]签订《[[圖們江中韓界務條款|圖們江中韓界務條款]]》，确认圖們江是中、韓兩國國界<ref>清政府、大日本帝國.《[[s:圖們江中韓界務條款|圖們江中韓界務條款]]》，1909年9月4日，見《海关中外条约》，卷2，頁762-764。</ref>。1938年，日苏[[张鼓峰事件|张鼓峰事件]]后，中国图们江出海口被日本和苏联封锁。<ref>{{cite journal|author1=石磅|coauthors=赵春江|title=张鼓峰事件 历时仅11天的战役|journal=中国国家地理|date=2014年|issue=第09期|url=http://www.dili360.com/article/p541ff98a817f655.htm|accessdate=2016-11-12}}</ref>

1962年[[中华人民共和国|中华人民共和国]]和[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]签定边界条约，确认[[延边|延边]]属于中国，但是中国将[[长白山天池|长白山天池]]一部分劃歸朝鲜<ref>{{cite web|author1=大明号|title=用地图说话——1962年，中、朝是如何平分长白山天池的？|url=http://www.cn1n.com/China/territory/20161110/1622342075.htm|website=中国在线|accessdate=2016-11-11|date=2016-11-10}}</ref>。1990年，[[苏联|苏联]]与朝鲜签订边界协议将圖們江出海口附近的[[鹿屯島|鹿屯島]]划归苏联，韩国抗议，并要求苏联归还领土<ref name="noktundo">{{cite web|url=http://ru.apircenter.org/publications/the-problem-of-the-noktundo-island-in-the-media-in-south-korea/ |script-title=ru:Проблема острова Ноктундо в средствах массовой информации Южной Кореи|language=ru|publisher=ru.apircenter.org |accessdate=2015-06-30 |archiveurl=https://web.archive.org/web/20130910052035/http://ru.apircenter.org/publications/the-problem-of-the-noktundo-island-in-the-media-in-south-korea/ |archivedate=2013-09-10|deadurl=no}}</ref>。1991年5月16日《[[s:中华人民共和国和苏维埃社会主义共和国联盟关于中苏国界东段的协定|中华人民共和国和苏维埃社会主义共和国联盟关于中苏国界东段的协定]]》中的第九条，苏方同意中国船只（悬挂中国国旗）可沿协定有关界点以下的图们江（苏联地图为图曼纳亚河）通海往指定航行。同時苏方也附加限制，僅允許季節性捕撈的漁船，而不准商業運行的船隻出海。<ref name=weng>{{cite web|author1=翁寒松|title=中俄圖們江領土爭議的實況|url=http://www.worldjournal.com/3565376/article-%E4%B8%AD%E4%BF%84%E5%9C%96%E5%80%91%E6%B1%9F%E9%A0%98%E5%9C%9F%E7%88%AD%E8%AD%B0%E7%9A%84%E5%AF%A6%E6%B3%81/|website=世界新闻网|accessdate=2016-11-11|deadurl=yes|archiveurl=https://web.archive.org/web/20161112084547/http://www.worldjournal.com/3565376/article-%E4%B8%AD%E4%BF%84%E5%9C%96%E5%80%91%E6%B1%9F%E9%A0%98%E5%9C%9F%E7%88%AD%E8%AD%B0%E7%9A%84%E5%AF%A6%E6%B3%81/|archivedate=2016-11-12}}</ref>
