'''图巴拉尔人'''或'''土巴拉尔人'''（{{lang-ru|Тубалары}}，''Tubalar''），亦作'''图巴人'''（{{lang-ru|Туба}}，''Tuba''），是一个生活在[[俄罗斯|俄罗斯]][[阿尔泰共和国|阿尔泰共和国]]的[[突厥语|突厥语]]民族，主要集中于[[乔伊斯基区|乔伊斯基区]]和[[图罗恰克斯基区|图罗恰克斯基区]]，即[[阿尔泰淖尔|阿尔泰淖尔]]一带。图巴拉尔人过去一直过着传统的狩猎生活，直到俄罗斯人到来后才开始务农。<ref> {{Cite book | author = Суляндзига Р. В., Кудряшова Д. А., Суляндзига П. В. | title = Коренные малочисленные народы Севера, Сибири и Дальнего Востока Российской Федерации | date = 2003 | accessdate = 2019-01-04 | url = http://www.raipon.info/peoples/tubalars/tubalars.php | language = ru }} </ref>

图巴拉尔人在[[沙俄|沙俄]]时期被称作黑鞑靼，在[[苏联|苏联]]时期被划入[[阿尔泰人|阿尔泰人]]，<ref>{{Cite web |url = http://war.163.com/photoview/4T8E0001/72509.html |title = 苏联时期的西伯利亚阿尔泰人 |date = 2014-08-25 |publisher = 网易军事 |language = zh |accessdate = 2019-01-04 }}</ref>直到2000年才被政府承认为独立民族。图巴拉尔人在2010年约有2000人，其中只有200多人说图巴拉尔语。<ref>{{Cite web |url = https://bigenc.ru/ethnology/text/4218506 |title = ТУБАЛА́РЫ |publisher = 俄罗斯大百科全书 |language = ru |accessdate = 2019-01-04 }}</ref>图巴拉尔语亦被认为是[[阿尔泰语|阿尔泰语]]的一种北部方言。<ref>{{Cite web |url = https://bigenc.ru/linguistics/text/4218497 |title = ТУБАЛА́РСКИЙ ЯЗЫ́К |publisher = 俄罗斯大百科全书 |language = ru |accessdate = 2019-01-04 }}</ref>
