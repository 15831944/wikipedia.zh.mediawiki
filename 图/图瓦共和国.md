'''图瓦共和国'''（{{lang-rus|Республика Тыва|Respublika Tyva}}；[[圖瓦語|圖瓦語]]：'''{{lang|tyv|Тыва Республика}}'''）是[[俄罗斯联邦|俄罗斯联邦]]中的一个[[俄罗斯联邦主体|联邦主体]]<ref>{{cite web |url = http://www.constitution.ru/en/10003000-02.htm |title = Chapter 1. The Fundamentals of the Constitutional System | The Constitution of the Russian Federation |publisher=Constitution.ru |date= |accessdate=}}</ref>，是[[西伯利亚联邦管区|西伯利亚联邦管区]]的一个[[自治共和国|自治共和国]]，首府为[[克孜勒|克孜勒]]。图瓦共和国位于[[蒙古国|蒙古国]]西北、[[西伯利亚|西伯利亚]]以南。


== 历史 ==
{{Main article|图瓦历史}}
[[File:Chadan_ugol.jpg|thumb]]場]]

从1207年至17世纪末，历史区域[[唐努乌梁海|唐努乌梁海]]为[[林中百姓|林中百姓]]的一部[[乌梁海|乌梁海]]的所在地，一直归[[蒙古族|蒙古人]]（[[蒙古帝国|蒙古帝国]]、[[元朝|元朝]]、[[北元|北元]]等）管辖。后来在17世纪末清军击败[[准噶尔汗国|准噶尔汗国]]军队后归[[清朝|清朝]]管辖。

1911年[[辛亥革命|辛亥革命]]爆发后，[[外蒙古|外蒙古]]宣布独立，而[[俄罗斯帝国|俄罗斯帝国]]亦在圖瓦策劃獨立活動。1914年，俄国出兵占领了图瓦。

1917年爆发的[[1917年俄国革命|俄国革命]]推翻了俄罗斯帝国。图瓦的大部分地區于1918年7月5日至1919年7月15日，被[[亚历山大·高爾察克|亚历山大·高爾察克]]的[[白軍|白軍]]佔領。彼得·伊万诺維奇·圖爾恰尼諾夫为该地区的統治者。1918年秋天，图瓦版图的西南部分被中国北洋政府军队所占领，南方被[[哈丹巴特尔·马格萨加夫|哈丹巴特尔·马格萨加夫]]所领导的蒙古军队所占领。1919年7月至1920年2月，[[蘇聯红军|蘇聯红军]]佔領了图瓦。1920年2月19日至1921年6月，[[中國|中國]]軍隊佔領圖瓦，統治者為[[嚴世超|嚴世超]]。1921年8月14日，布爾什維克成立了[[唐努－圖瓦人民共和國|唐努－圖瓦人民共和國]]，通稱唐努图瓦。1926年，首都「別洛查爾斯克」（{{lang|ru|Белоца́рск}}）改名为「[[克孜勒|克孜勒]]」（{{lang|ru|Кызыл}}），意思是红色。在世界大战期间{{when}}，图瓦成为一个独立的国家，但是一直在苏联的控制下。图瓦虽然有自己的“呼拉尔”（议会），但是没有选举权。从1926年起称[[图瓦人民共和国|图瓦人民共和国]]。1944年10月，又被并入[[蘇聯|蘇聯]]的[[俄羅斯蘇維埃聯邦社會主義共和國|俄羅斯蘇維埃聯邦社會主義共和國]]，享有[[俄羅斯自治州|俄羅斯自治州]]的权利；而在中华民国36年（1947年）出版的一本地图册<ref>世界分国精图，1946年上海：大中国图书局</ref>认为：“外蒙古西北部的唐努乌梁海一区'''早已'''脱离外蒙古而独立，仍隶属于我国。”但目前就[[中華民國外交部|外交部]]的說明，因國際局勢與實際需求，為尊重國際社會的共識，因此認同其為俄國境內下屬之80多個「[[俄罗斯联邦主体|聯邦主體]](federal subject)，類似州或省」之一<ref>中华民国外交部回函：{{quote|關於台端所詢唐努烏梁海歸屬乙節，查唐努烏梁海係泛指介於薩彥嶺至唐努山之間狹長地帶，面積約17萬平方公里，原屬外蒙古一部份。另查蒙古於1961年10月27日加入聯合國，係一主權國家，與世界140餘國維持外交關係，其中包括中國大陸，我國作為國際社會之一員，自應尊重國際社會的共識。行政院於民國91年元旦修正通過「台灣地區與大陸地區人民關係條例施行細則」第3條及第56條修正條文，不再視蒙古為大陸地區，以利我與蒙古正常交往。我國於2002年在蒙古設立代表處，秉持理性及務實的態度，加強雙方實質關係，不涉及固有疆域之爭議，蒙古於2003年在台北設立代表處，推動台蒙雙邊交流，建立互惠互利合作關係。

蘇聯於1921年在唐努烏梁海設立[[圖瓦人民共和國|圖瓦人民共和國]]，惟未獲中華民國政府承認。1944年，圖瓦人民共和國加入蘇聯，改名為圖瓦蘇維埃社會主義自治共和國。1945年中華民國與蘇聯簽署《[[中蘇友好同盟條約|中蘇友好同盟條約]]》，維持外蒙古現狀。中華民國駐蘇聯大使傅秉常於1948年5月照會蘇聯外交部聲明唐努烏梁海係中華民國領土。1949年，中華民國宣布廢止《[[中蘇友好同盟條約|中蘇友好同盟條約]]》。蘇聯於1991年瓦解後，'''圖瓦共和國成為俄羅斯聯邦境內83個「聯邦主體(federal subject)，類似州或省」之一。 我作為國際社會的一份子，務實考量國際局勢與實際需求'''，為拓展與俄羅斯的實質關係，我於民國82年7月在俄國首府莫斯科設立代表處，俄羅斯也於民國85年12月在台北設立代表處，雙邊年度貿易金額已突破40億美元，今後亦將在現有基礎上繼續努力，推動雙邊關係。

== 政府 ==
國家管理機構：圖瓦為主席制共和國，共和國結構於1993年被承認。最高法律機關為上議院，五年選舉一次。執行機構為以主席為首的部委員會。

== 自然地理 ==
* 面积：170 500km²
* 气候：[[温带大陆型气候|温带大陆型气候]]，冬季寒冷，夏季温暖。一月份平均气温−32 °C ，七月份平均气温+18 °C
* 地形：四周被[[赛留格木山|赛留格木山]]、[[唐努山|唐努山]]、西[[萨彦岭|萨彦岭]]和东萨彦岭环抱，东部为[[上叶尼赛盆地|上叶尼赛盆地]]。境内主要河流为[[叶尼塞河|叶尼塞河]]。[[地貌|地貌]]以[[森林|森林]]、[[草甸|草甸]]和[[草原|草原]]为主。
* 自然资源：图瓦的主要矿藏包括金矿(主要以砂[[金|金]]形式存在)、[[铜|铜]]、[[铱|铱]]、[[铁|铁]]矿(分布在[[克木奇克河|克木奇克河]]流域和[[乌木奇克河|乌木奇克河]]流域)、煤、云母、岩盐、[[石棉|石棉]]、石墨、[[大理石|大理石]]、菱镁、[[硫磺|硫磺]]。境内矿泉众多。河流和湖泊内渔产丰富。
* 时区：图瓦位于东七区(UTC+7)

==人口==
人口：307,930人 (2007年)<ref name="2010Census">{{ru-pop-ref|2010Census}}</ref>，主要民族包括[[图瓦族|图瓦族]](77%)、[[俄罗斯族|俄罗斯族]](20.1%)，还有少量[[科米族|科米族]]和[[哈卡斯族|哈卡斯族]]。男女比例为1：1.108。

==經濟==
圖瓦共和國是東西伯利亞經濟區的組成部分。它是一個農業國家，同時又蘊含著豐富的礦藏。圖瓦共和國分為3個基本經濟區：中心區——人口最密集的主要工業區，包括[[克孜爾|克孜爾]]、[[阿克多武拉克|阿克多武拉克]]、[[柴旦|柴旦]]；南部畜牧業區是最重要的農業[[畜牧業|畜牧業]]區；東北區——木材加工、狩鹿及[[漁業區|漁業區]]。

==工業==
圖瓦共和國的主要工業部門有[[電力工業|電力工業]]、[[食品工業|食品工業]]、[[建材工業|建材工業]]、木材及[[木材加工工業|木材加工工業]]、采[[礦工業|礦工業]]。最重要的工業是[[礦山開採業|礦山開採業]]，即[[有色金屬|有色金屬]]、[[石棉|石棉]]、[[石煤|石煤]]、[[金子|金子]]及其它礦產的開採。

最大的企業：[[圖瓦“鈷”企業|圖瓦“鈷”企業]]（[[赫烏阿克西市|赫烏阿克西市]]）、[[圖瓦“石棉”企業|圖瓦“石棉”企業]]（[[阿克多武拉克市|阿克多武拉克市]]）、[[水銀開採開發企業|水銀開採開發企業]]（[[德爾裏克哈雅市|德爾裏克哈雅市]]）、[[金子開採企業|金子開採企業]]（[[巴伊休克|巴伊休克]]、[[奧伊娜阿|奧伊娜阿]]）。發展比較快的食品行業有肉和[[麵粉|麵粉]]廠、[[奶|奶]]和[[啤酒|啤酒]]釀造廠。

==農業== 
圖瓦共和國是一個專業生產畜產品的農業國，80%多的產品為畜產品。主要表現為[[養馬業|養馬業]]、[[養鹿業|養鹿業]]、[[養駝業|養駝業]]、[[養細毛綿羊業|養細毛綿羊業]]和養肉、[[奶牛業|奶牛業]]。農業經營用地約500萬公頃，占共和國領土的三分之二。其中耕地為50萬公頃，草場為10萬公頃，牧場為400多萬公頃。

==交通運輸== 
圖瓦共和國的主要交通方式是汽車。主要公路幹線有阿克多武拉克—克孜勒—[[阿巴坎|阿巴坎]]—[[克拉斯諾亞爾斯克|克拉斯諾亞爾斯克]]。
