'''图图嫩多'''（{{lang-es|Tutunendo}}）是位于[[哥伦比亚|哥伦比亚]][[乔科省|乔科省]][[基布多|基布多]]市的小镇。小镇坐落于{{Link-es|哥伦比亚60号国道|Ruta Nacional 60 (Colombia)}}上，距离[[基布多|基布多]]14公里，人口为4022人。该镇降雨量极大，是[[南美洲|南美洲]]雨级和世界三大雨极之一<ref name="1R">{{Cite web |url = https://traveltips.usatoday.com/climate-tutunendo-choco-colombia-104538.html |title = Climate of Tutunendo and Choco, Colombia |language = en |accessdate = 2019-02-24 }}</ref>，颇具生物多样性。其经济主要包括农业，渔业以及大量开采的矿产资源和木材资源，是基布多市周末旅游业收入的主要来源。

==地理==
[[File:Tutunendo.JPG|缩略图]]
图图嫩多位于哥伦比亚西部[[乔科省|乔科省]]的[[热带雨林|热带雨林]]中，因紧邻[[阿特拉托河|阿特拉托河]]支流[[伊里河|伊里河]]支流[[图图嫩多河|图图嫩多河]]而得名，海拔105米，距离乔科省省会[[基布多|基布多]]14公里。

===气候===
图图嫩多气候为[[热带雨林气候|热带雨林气候]]。极端温暖、高湿度、缺乏风和多降水是其气候类型的特征。年均降雨量11米，1974年的降雨量曾达到26米。<ref name="2R">{{Cite web |url = https://wk.baidu.com/view/9dc4da5cb207e87101f69e3143323968011cf4ae |title = 地球上最极端的15个地方 |date = 2017-06-14 |language = zh |accessdate = 2019-02-24 }}</ref>城市位于赤道以北大约五度，日夜时间几乎相等。持续不断的云层和雨水在图图嫩多气候中非常常见。云层通常覆盖着天空，平均每天只有三到四个小时的阳光透过云层。一年平均有280天降雨，超过三分之二的降雨（68％）为夜间降雨，年平均[[相对湿度|相对湿度]]达到90％。图图嫩多是世界上雨量最大的三个地方之一，其他两个地方为[[印度|印度]]的[[乞拉朋齐|乞拉朋齐]]和[[夏威夷|夏威夷]]的{{le|怀厄莱阿莱山|Mount Waialeale}}。秋天是一年中最潮湿的季节，降雨量最大的月份为8月和9月，7月、10月和11月也很潮湿。即便是一年中最干燥的几个月2月和3月，每月仍有近20天的降水。<ref name="1R"/>

==人口==
图图嫩多拥有4022名居民，大多数为14至25岁的年轻人，其中70%~90%是[[黑人|黑人]]，10%~30%是[[美洲土著|美洲土著]]，多为{{le|埃姆贝拉人|Emberá}}和[[库那族|库那人]]。<ref name="3R">{{Cite web |url = https://enamoratedelchoco.co/tutunendo/ |title = Tutunendo significa Río de Rosas o Fragancia |language = es |accessdate = 2019-02-24 }}</ref>

==经济==
因该镇位于热带雨林之中，地理位置又相对偏僻，该镇经济以[[自给农业|自给农业]]和[[伐木|伐木]]为主，包括了[[渔业|渔业]]以及大量开采的矿产资源和木材资源。<ref name="1R"/>该镇几乎没有工业。

==交通==
{{Link-es|哥伦比亚60号国道|Ruta Nacional 60 (Colombia)}}穿过小镇，距离省会城市[[基布多|基布多]]18公里，距哥伦比亚第二大城市[[麦德林|麦德林]]213公里。该镇及附近没有铁路和机场。

==旅游==
由于其[[热带雨林|热带雨林]]景观和[[生物多样性|生物多样性]]，图图嫩多是[[基布多|基布多]]市域内主要的旅游目的地。旅游景点均为自然景观，包括河流、瀑布，但因其地理位置偏僻较少有游客踏足。<ref name="1R"/><ref>{{Cite web |url = https://web.archive.org/web/20130615023420/http://www.tutunendo.com/sitios-tur%C3%ADsticos/ |title = Sitios Turísticos |language = es |accessdate = 2019-02-24 }}</ref>
