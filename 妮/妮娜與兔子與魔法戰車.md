《'''妮娜與兔子與魔法戰車'''》（{{jp|j='''ニーナとうさぎと魔法の戦車'''|hg=にいなとうさきとまほうのせんしゃ|rm=Nīna to Usagi to Mahō no Sensha|lead=yes}}）是[[日本|日本]][[輕小說|輕小說]]作家{{Tsl|ja|兎月竜之介|兔月龍之介}}著作、插畫家[[BUNBUN|BUNBUN]]負責插圖，2010年9月由[[集英社|集英社]][[Super_Dash文庫|Super Dash文庫]]發行，並獲得第9届[[SD小說新人獎|SD小說新人獎]]大賞<ref>{{cite web|url=http://prizesworld.com/prizes/lightnovel/spdh.htm|script-title=ja:スーパーダッシュ小説新人賞受賞作・候補作一覧1-13回 - 文学賞の世界|language=ja|publisher=文学賞の世界|accessdate=2018-11-29}}</ref>。該作[[漫畫化|漫畫化]]由{{link-ja|藪口黑子|薮口黒子}}負責，在《[[Super_Dash_&_Go!|Super Dash & Go!]]》（集英社）連載<ref>{{cite web|url=https://natalie.mu/comic/news/58599|script-title=ja:集英社の新雑誌スーパーダッシュ＆ゴー！本日創刊|language=ja|website=[[Natalie_(網站)|Natalie漫畫]]|date=2011-10-23|accessdate=2018-11-29}}</ref>。另外，繁體中文版由葉廷昭翻譯、[[青文出版社|青文菁英文庫]]代理出版，2013年12月25日初版首刷發行。兔月龍之介其後基於此作於網路小說平台KAKUYOMU上連載《米奇與兔子與機械迷宮》（{{lang|ja|ミキとうさぎと機械の迷宮}}）<ref>{{cite web|url=https://kakuyomu.jp/works/1177354054884254145|script-title=ja:ミキとうさぎと機械の迷宮（兎月竜之介） - カクヨム|language=ja|author=兔月龍之介|publisher=カクヨム|accessdate=2018-11-29}}</ref>。

== 故事大綱 ==
五年前投下的新型魔法炸彈貌似終止了國與國之間的戰爭。戰役雖然結束，但是戰鬥卻從未終止過。由於魔法炸彈而暴走的無人戰車「野戰車」，仍然威脅著人們的日常生活<ref name="shuesha">{{cite web|url=http://dash.shueisha.co.jp/_nina/index.html|script-title=ja:Super Dash文庫 > ライブラリー > ニーナとうさぎと魔法の戦車|language=ja|publisher=Super Dash文庫|accessdate=2014-08-13|archive-url=https://web.archive.org/web/20130309040022/http://dash.shueisha.co.jp/_nina/index.html|archive-date=2013-03-09|dead-url=yes}}</ref>。

妮娜則是受戰禍牽連而浪跡天涯的少女。某日妮娜因偷取婚禮會場的食物時被逮個正著，她本來已經有了伏法的覺悟，但這時來了一群搭乘魔動戰車的少女赦免了妮娜<ref name="shuesha"/>。這群人就是保護城鎮免受戰爭衍生的災禍「野戰車」侵襲的私立戰車隊，通稱「無頭群兔」的成員。隨後鎮上響起了野戰車進犯的警報，群兔正好欠缺一位砲手，而妮娜以前曾經有駕駛戰車的經驗，於是妮娜順利成章地成為了群兔新的砲手，為鎮民的平和生活而奮鬥<ref name="shuesha"/>。

; {{nihongo|妮娜|ニーナ|[[聲優|聲]]：[[悠木碧|悠木碧]]}}

; {{nihongo|桃樂絲|ドロシー·クローゼル}}
: 聲：[[川澄綾子|川澄綾子]]

; {{nihongo|奇奇|キキラヤ·ヴィライネン}}

: 「群兔」接續手。愛称為「奇奇」（{{lang|ja|キキ}}）。平日非出勤時在工廠兼職整備工人。因在大戰時右胸負傷，受傷勢影響變得無法釋放魔力。
; {{nihongo|愛爾紗·阿多拉巴魯特|エルザ·アドラバルト}}
: 聲：[[戸松遥|戸松遥]]
: 「群兔」操縱手。平日非出勤時在麵包店工作。
; {{nihongo|克兒|クリスティーナ·オーランド}}
: 聲：[[花澤香菜|花澤香菜]]
: 「群兔」動力手。愛称為「克兒」（{{lang|ja|クー}}）。平日非出勤時進行翻譯工作，受魔彈爆炸影響變得嗜睡。
; {{nihongo|小櫻|アメミヤ·サクラ}}
: 聲：[[田村由香里|田村由香里]]

; {{nihongo|愛麗絲|アリス·マドガルド}}
: 「群兔」戰隊所屬的少女。平日非出勤時與妮娜同樣在醫生的辦事處工作。
; {{nihongo|醫生|ドクター}}
: 本名為魯道夫·豪瑟（{{lang|ja|ルドルフ·ハウザー}}），妮娜等人所居住的安佛列格鎮的鎮長，原軍醫。
; {{nihongo|貝雅莉潔·卡克蘭德|ベアトリーチェ·カークランド}}
: 醫生的秘書。對醫生抱有好感，同時也是位優秀的魔女。

; {{nihongo|索尼婭|ソーニャ}}

: 佛克西歐共和國陸軍中將，猫派。認為人類並不會與持相同理念之人相爭，只要所有人秉持同一想法，便會天下太平，並以此唯一解發動戰爭。

== 作品特殊用語 ==
; 魔法板
: 表面刻有魔法符文的特殊金属板。咬在口中可將魔力注入到右胸部被稱為「第二心臟」的器官，並可根據法術觸發魔法。
; 魔動戰車
: 以魔法為驅動的戰車。
; 野戰車
: 新型魔力爆彈投下後開始暴走的流浪魔動戰車，思考力幾乎為零，透過名為「巢」的魔力母体獲得半永久性的魔法不停行駛。
; 「無頭群兔」
: 既是私立戰車隊「群兔」（ラビッツ）的尊称、同時也是蔑称。

== 作品世界觀 ==
=== 世界主要構成国家 ===
; 佛克西歐共和國（{{lang|ja|フィクシオ共和国}}）
: 作品的主要舞台，曾與阿克西亞帝國發生戰爭。
; 原阿克西亞帝國（{{lang|ja|アーケシア元帝国}}）
: 愛爾紗的故鄉。戰後被瓜分管治。
; 席巴雷爾社會主義聯邦（{{lang|ja|シヴァレル連邦}}）
: 控制並高壓統治索爾夏的社會主義國家，與佛克西歐接壤。
; 索爾夏共和國（{{lang|ja|ソルシャ共和国}}）
: 妮娜的故鄉。大陸邊陲的極北小国，現正發生大規模内乱。
; （{{lang|ja|フォースランド王国}}）
: 克兒的故鄉，擁有300年以上歷史的君主制政權國家。
; 秋津国（{{lang|ja|秋津国}}）
: 小櫻的故鄉，為大戰時阿克西亞的同盟国。魔法炸彈的引爆地、因此戰後近一半国土消失殆盡。
; 泰蘭合眾國（{{lang|ja|タイラン合衆国}}）
: 愛麗絲的祖国。
; 布里塔尼亞（{{lang|ja|ボリタニア}}）
: 桃樂絲的祖国，該國人主要特徵為紅髮和褐色肌膚。
