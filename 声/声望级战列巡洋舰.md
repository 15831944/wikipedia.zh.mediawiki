'''声望级战列巡洋舰'''（'''{{lang|en|Renown class battlecruiser}}'''）是[[英国|英国]]海军建造的一级[[战列巡洋舰|战列巡洋舰]]。同级舰有两艘，分别是：声望号（{{lang|en|HMS Renown}}）和[[反擊號巡洋戰艦|反擊號]]（HMS Repulse，一譯為「卻敵號」）。

== 建造 ==
该级舰原计划作为[[复仇级战列舰|复仇级战列舰]]经改良后的第三、四号舰：声望号和反击号。然而[[第一次世界大战|第一次世界大战]]爆发后，建造工作于1914年8月暂停，因为战争的爆发使得这两艘军舰无法按及时完工参战。1914年10月，第二度出任[[第一海务大臣|第一海务大臣]]的[[第一代费舍尔男爵约翰·费舍尔|约翰·费舍尔]]海军元帅获得批准将这两艘军舰从[[战列舰|战列舰]]改为[[战列巡洋舰|战列巡洋舰]]从而加快建造进度以赶在战争结束前完工参战，于是两舰的建造工作于1915年1月得以继续。时任海军造舰总监的[[丁尼生－戴恩科特爵士|丁尼生－戴恩科特爵士]]很快拿出了一个全新的设计方案以满足[[第一代费舍尔男爵约翰·费舍尔|约翰·费舍尔]]海军元帅的要求，而海军造船厂也同意在15个月内完工。尽管最后工期有所延长，但也只比[[日德兰海战|日德兰海战]]稍晚了几个月。完工之时，这两艘军舰是当时世界上吨位最大、航速最快的主力舰。

== 概貌 ==
声望级战列巡洋舰采用[[长艏楼|长艏楼]]船型，外飘型舰艏，适航性好。高航速被置于优先考虑的地位，动力装置采用多达42台燃油型锅炉。沿用复仇级战列舰的15英寸口径主炮，数量减少到6门。副炮安装在三联装炮塔中，全部位于[[露天甲板|露天甲板]]之上，提高了射界，减少了高速航行以及恶劣海况时海浪的影响。为了获得更高的航速动力装置占用重量导致防御装甲占用的重量被削减，在当时英国海军在役的战列巡洋舰中，声望级的装甲防护能力算是最差的。其设计主导思想就是针对敌方的[[巡洋舰|巡洋舰]]舰队，大口径火炮可以在敌舰射程外发起致命打击，高航速可以机动自如或逃避敌人的主力舰。这是英国海务大臣[[約翰·費舍爾，第一代費舍爾男爵|费舍尔]]的“速度就是最好的防御”观点的具体体现。由于舰体结构重量较轻，在全部主炮射击时，会发生剧烈震动，声望号的舰体还因此而发生过损坏。

== 服役 ==
[[日德兰海战|日德兰海战]]之后，根据海战的经验新服役的两舰接受强化装甲和结构的改装。[[第一次世界大战|第一次世界大战]]结束后两舰再次进行了强化装甲的改装，并增加航空设施。声望级两舰在服役生涯中曾多次运载英国皇室政要出访。1930年代再次现代化改装，改装的重点依然是防御装甲，声望号改建较大，舰桥全部拆除进行重建，动力系统换装新型的锅炉，副炮全部拆除安装新型高平两用炮，加装各型高射炮及新型雷达等。由于[[第二次世界大战|第二次世界大战]]爆发，反击号未能进行与声望号类似的大规模改装。
1941年11月反击号和威尔士亲王号战列舰组成[[Z舰队|Z舰队]]前往[[新加坡|新加坡]]威慑[[日本|日本]]，12月两舰在[[马来亚|马来亚]]东海岸海域被日本舰载攻击机用鱼雷击沉。随着[[胡德号战列巡洋舰|胡德号]]、反击号相继损失，声望号成为英国海军在役的最后一艘战列巡洋舰。声望号凭借其较高的航速，执行猎杀敌方袭击舰，护航[[航空母舰|航空母舰]]、运输船队等很多作战行动。1944年开赴[[远东|远东]][[太平洋|太平洋]]战场。
战争结束后声望号退役，1948年拆毁。
