'''兩岸漁業衝突'''，是指中華民國（臺灣）與中華人民共和國（中國大陸）雙方海域的漁業衝突，也是[[海峽兩岸關係|海峽兩岸關係]]的一部份。

== 背景 ==
* 2004年6月7日，[[中華民國國防部|中華民國國防部]]公告《限制、禁止水域圖》，分別對台澎地區、金門地區、馬祖地區、烏坵地區、東引地區、南沙地區及東碇島做繪製。<ref>{{cite web|title=限制、禁止水域圖|url=http://www.mac.gov.tw/ct.asp?xItem=61582&CtNode=5671&mp=1|website=行政院大陸委員會}}</ref>
* 2016年7月1日，[[行政院海岸巡防署|行政院海岸巡防署]]實施《護永專案》，推動海洋保育，並取締越界漁船。<ref>{{cite web|title=-專案介紹-護永專案|url=http://www.cga.gov.tw/GipOpen/wSite/ct?xItem=108900&ctNode=9628&mp=marine|website=行政院海岸巡防署}}</ref>

== 歷史事件 ==
* 1991年7月21日，[[閩獅漁事件|閩獅漁事件]]發生。
* 2014年1月17日，中國大陸籍貨船「新康輝號」，未經申請許可，進入金門西北方限制水域，經[[行政院海岸巡防署海洋巡防總局|行政院海岸巡防署海洋巡防總局]]第九（金門）海巡隊取締，貨船人員依法處罰新台幣100萬元以上。<ref>{{cite news|title=陸貨船侵入水域 依法罰百萬|url=https://udn.com/news/story/2/1919660|agency=中央社|publisher=中時電子報|date=2014-01-18}}</ref> 
* 2016年8月15日，中國大陸籍漁船「閩龍漁66640號」，在彭佳嶼東北方海域19.7浬處進行捕撈作業，經海巡人員取締，發現船上滿載400公斤漁獲。同日，中國大陸籍漁船「閩龍漁60364號」，在彭佳嶼東北方海域23浬處進行捕撈作業，經海巡人員取締，發現船上滿載500公斤漁獲。<ref>{{cite news|title=中國漁船偽裝台灣漁船越界捕魚 海巡扣回 2船|url=http://news.ltn.com.tw/news/society/breakingnews/1796543|agency=自由時報|date=2016-08-16}}</ref> 
* 2016年8月25日，中國大陸籍漁船「閩東漁69536號」，在[[花嶼|花嶼]]海域越界進行捕撈作業，經第八（澎湖）海巡隊行駛[[行政院海岸巡防署軍事設備列表|PP-10061巡防艇]]驅離無效後，將其開罰。<ref>{{cite news|title=澎湖海巡查扣越界陸船 查扣重罰|url=https://udn.com/news/story/2/1919660|agency=中央社|publisher=聯合新聞網|date=2016-08-25}}</ref> 
* 2016年11月8日，中國大陸籍漁船「閩東漁61292號」，在花嶼西北方海域11.6浬處越界進行捕撈作業，經澎湖海巡隊行駛[[行政院海岸巡防署軍事設備列表|PP-10038巡防艇]]取締，漁船蛇行逃逸拒檢，甚至2度意圖衝撞巡防艇，直到海巡隊發射震撼彈，才控制局面。<ref>{{cite news|title=越界漁船造浪頑抗　海巡特勤打震撼彈制伏|url=http://www.appledaily.com.tw/realtimenews/article/new/20161108/984614/|agency=蘋果日報|date=2016-11-08}}</ref>
* 2017年2月4日，中國大陸籍漁船「閩長漁24188號」，在彭佳嶼西北方海域約20浬處越界進行捕撈作業，經海巡人員取締，發現船上滿載2000公斤漁獲，且該船已有遭驅離5次的紀錄。<ref>{{cite news|title=曾遭驅離5次陸船又越界 還大撈2千公斤漁獲|url=https://udn.com/news/story/2/2265920|agency=聯合新聞網|date=2017-02-05}}</ref>同日，中國大陸籍漁船「閩獅漁07605號」，在台中港西北方海域22浬處越界進行捕撈作業，經台中海巡隊行駛[[行政院海岸巡防署軍事設備列表|PP-2037巡防艇]]取締，該船已有遭驅離2次的紀錄；最後由[[臺灣臺中地方法院檢察署|臺灣臺中地方法院檢察署]]裁處漁船人員每人新台幣9萬1千元罰款，共新台幣340多萬元。<ref>{{cite news|title=中國漁船越界捕魚又妨礙公務 重罰340餘萬驅離出境|url=http://news.ltn.com.tw/news/society/breakingnews/2041415|agency=自由時報|date=2017-04-19}}</ref>

=== 南澳31049號事件 ===
2017年5月6日，中國大陸籍漁船「南澳31049號」，在花嶼西北方海域20.4浬處（座標：{{Coord|23.533333|118.966667}}）越界進行捕撈作業，且為[[休漁期|休漁期]]期間內；上午5點20分，由澎湖海巡隊PP-10038巡防艇取締，並廣播數十次告知漁船受檢，但漁船仍蛇行逃逸拒檢，漁工甚至攀爬到船舷邊當人肉盾牌，最後海巡隊射擊5發橡膠子彈嚇阻；5點41分，海巡隊順利登檢漁船，並將2名受傷漁工送往[[三軍總醫院|三軍總醫院]]澎湖分院醫治，同時將漁船押返。<ref>{{cite news|title=陸船蛇行拒檢「還爬船邊組人肉盾牌」　海巡射橡膠彈2人送醫|url=http://www.ettoday.net/news/20170506/918649.htm|agency=ettoday|date=2017-05-06}}</ref><ref>{{cite news|title=陸船越界祭人肉盾牌　海巡開5槍打傷2人迫就範|url=http://www.appledaily.com.tw/realtimenews/article/local/20170506/1112824/|agency=蘋果日報|date=2017-05-06}}</ref>

== 各界反應 ==
* 2014年10月23日，時任[[行政院院長|行政院院長]][[江宜樺|江宜樺]]在聽取海巡署報告後表示，簽署《兩岸漁業協議》，有助穩定兩岸漁業及海域秩序。<ref>{{cite news|title=兩岸漁業協議 陸委會：將從資源養護納協商|url=http://www.chinatimes.com/realtimenews/20141023004389-260409|agency=中時電子報|date=2014-10-23}}</ref>
* 2015年1月23日，時任馬英九總統表示：「雖然中國大陸與我方不太願意簽署漁業協議，以免碰觸到疆界、海域這些敏感話題，但台灣在執法上卻沒有任何困難」。<ref>{{cite news|title=不願碰疆界議題 馬：中拒與我談漁業協議|url=http://www.storm.mg/article/40245/20150123/|agency=風傳媒|date=2014-10-23}}</ref><ref>{{cite news|title=兩岸未簽漁業協議　馬英九：但不影響執法|url=http://www.appledaily.com.tw/realtimenews/article/new/20150123/547475/|agency=蘋果日報|date=2014-10-23}}</ref>
