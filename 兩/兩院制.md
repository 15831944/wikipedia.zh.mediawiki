{{立法機構}}
[[File:Unibicameral_Map.svg|400px]]
'''兩院制'''是一種以兩個獨立運作的[[議會|議會]]（[[議院|議院]]）組成[[立法機構|立法機構]]的政治制度，亦被稱為'''二院制'''。與之相對的制度稱為[[單院制|單院制]]或[[一院制|一院制]]，即是只有單一議會作為立法機構。

==概要==
兩院制起源於[[羅馬共和國|羅馬共和國]]時期，當時設有全是[[貴族|貴族]]產生的[[元老院|元老院]]和羅馬男性[[公民|公民]]產生的[[羅馬人民大會|平民大會]]，以互相制衡。

[[歐洲|歐洲]][[封建時代|封建時代]]的等級會議中，當時的議會包括[[神職人員|神職人員]]、[[貴族|貴族]]、[[平民|平民]]等三至四個等級的成員，後來不同等級的成員各自開會，共商國事，遂乃形成多院制（包括[[三院制|三院制]]和[[四院制|四院制]]）。現時多院制已不存在，大部分國家不是兩院制，就是一院制。

[[英國|英國]]式兩院制是源於[[英國國會|英國國會]]建立初期，貴族和平民的利益無法平衡，於是設立由[[貴族|貴族]]組成的[[英國上議院|英國上議院]]和以平民組成的[[英國下議院|英國下議院]]以互相制衡。美國式兩院制是源於英國，美國是由[[美國州份|州]]組成的[[聯邦|聯邦]]，每個州均有自己的淵源和文化與[[族群|族群]]，為避免人口多的州份侵害人口少的州份的利益，於是設立[[美國參議院|參議院]]，各州不論人口均派有兩名代表；同时设有以人口比例分配的[[美國眾議院|眾議院]]；所有法律均需兩院通過才能正式通過（但兩院另有一些對方沒有的權力）。

兩院制當中兩個上下議院的分別，现行主要有两種模式：
* 在[[聯邦|聯邦]]國家中，通常上議院以成員邦為單位選出，另一院由全民[[直選|直選]]。
* 在非联邦的单一政体国家中，通常上議院由委任及[[間接選舉|間接選舉]]議員組成，另一院由全民直選。

过去，在[[君主立憲制|君主立憲制]]國家中，通常上議院由貴族組成，下議院由平民直接選舉，但近年由于民主化改革已很少有世袭贵族组成的上议院。即使在称为“贵族院”的[[英国上议院|英国上议院]]，现在绝大多数议员也是由政府委任的“终身贵族”，只有少数议员是由世袭贵族选举出的代表，以及英國國教會的部分主教和兩名大主教。

在兩院制的[[民主國家|民主國家]]中，如果其中一院不是由直選產生，則直選產生的另一院通常擁有較大權力。一些國家的上議院（如[[英國上議院|英國上議院]]）甚至只剩下近乎象徵式的權力，稱為「跛腳的兩院制」。

== 優點 ==
*实现内部制衡，防止国会的专横独裁、团体专制
*因为立法需两院通过（其中一院覆核立法內容），可防止草率立法，提高立法和议政品質
*平衡各方利益，過去上議院代表貴族利益，现今的上議院则多代表鄉鎮利益，制衡城市主導的下議院
*利于多元意见的表达

== 缺點 ==
*立法程序繁琐，效率较一院制低
*与一院制相比，需要更多人力和财力
*其中一院不能完全代表民意，特別是上议院
*两院若分別由不同党派掌控，容易出現政爭，较困難進行立法程序

==職權劃分==
两院的职权划分主要有三种情形<ref>{{cite book zh| author = 中国大百科全书出版社|title = 中国大百科全书·政治学 |publisher = 中国大百科全书出版社 |date = 1992 |location = 北京 |pages = 437 |url = |doi = |id = |isbn = }}</ref>：
* 类似于[[英国|英国]]，下議院占較大优勢。[[直接選舉|直接選舉]]產生的[[英国下議院|英国下議院]]对[[财政預算案|财政預算案]]有先议权，委任及[[終身貴族|終身貴族]]產生的上議院只有为期一个月的延期通过权，法案在下议院[[三讀|三讀]]通过，上議院反对无效，故上議院的權力只是象徵。[[英国首相|首相]]領導的[[英国内阁|内阁]]只对下議院负责。
* 类似于[[法国|法国]]，[[直接選舉|直接選舉]]產生的[[國民議會|國民議會]]主導政府運作及政權更替，但[[間接選舉|間接選舉]]產生的[[法國參議院|參議院]]仍有部分权力，這体现在地方分权上，因为參议员是经大区及省地方议会間選產生。
* 类似于[[美国|美国]]，[[美国国会|国会]]两院有平等權力，兩院議員皆是[[直接選舉|直接選舉]]產生。所有[[法案|法案]]皆須於兩院同時通過；兩院皆有部份對方沒有之權力：如[[美国眾議院|美国眾議院]]有[[宣戰|宣戰]]權，徵[[稅|稅]]及预算法案僅眾議院有權提出；[[美国參議院|美国參議院]]擁有總統對联邦政府的主要官員、联邦大法官的任命同意權、[[條約|條約]]締結同意權等。類似國家有[[意大利|意大利]]、[[澳大利亞|澳大利亞]]、[[日本|日本]]等，但不同的是，在美國，參議院有较大权力。

== 類似兩院制的制度 ==
部分國家在中央政府設置類似兩院制的議事機關，但實為一院制。

==={{ROC}}===
[[中華民國|中華民國]]曾於建國初期实行过两院制，將[[中华民国初年国会|國會]]分為[[中華民國參議院|参议院]]和[[中華民國眾議院|众议院]]。

1928年12月29日[[東北易幟|東北易幟]]後，[[國民政府|國民政府]]設置[[立法院|立法院]]，雖具有國會地位，但僅有制定法律的功能，[[立法委員|其成員]]也非[[民選|民選]]產生，由[[監察院|監察院]]行使彈劾權，但並非[[議會|議會]]。立法院僅僅是立法機構，亦是[[橡皮圖章|橡皮圖章]]。

1947年12月25日[[行憲|行憲]]後，中央政府設[[國民大會|國民大會]]與立法院，民意代表由選舉產生，但國民大會是國家最高权力機構，立法院則繼續維持單純的立法功能，等於兩個权力機構共同行使國會職權，但仍非兩院制的範疇。

第一屆國大代表和立法委員在遷臺43年後長期未進行改選，直至1991年及1992年才分別實行全面改選。此後，因為國大代表和立法委員皆為民選，故同被視為國會議員，而代表和委員數目至2000年5月19日止，共有逾五百人，國大具有修憲權力，立法院則有立法的權力。

2005年[[任務型國民大會|任務型國民大會]]通過修憲將國民大會停止運作後，立法院的職權才轉變為等同其他民主國家的一院制議會。

==={{PRC}}===
[[中華人民共和國|中華人民共和國]]的「[[兩會|兩會]]」體制中，[[全國人民代表大會|全國人民代表大會]]跟前[[蘇聯|蘇聯]]的[[蘇聯最高蘇維埃|蘇聯最高蘇維埃]]一樣，是最高議事機構；而[[中國人民政治協商會議|中國人民政治協商會議]]是[[中國共產黨|中國共產黨]]和[[民主黨派|民主黨派]]、其他社會團體共同協商國策的機構，但並非议会，無立法、創制和複決的權力。全國人大和全國政協雖統稱為「兩會」，但並非兩院制。只有全国人大才是最高立法及权力机关。
