'''兩國'''（{{jpn|j=両国|hg=りょうごく|rm=Ryōgoku|f=y}}）是日本[[東京都|東京都]][[墨田區|墨田區]]的一個町名。[[郵政編碼|郵政編碼]]是130-0026。相當于墨田區{{link-ja|本所區|本所区|本所地區}}。也可以指兩國車站附近的地區。日本最重要的相撲場館[[兩國國技館|兩國國技館]]就位於這裡。這裡在過去曾是武藏國和下總國的交界處。在中央區也有部份地區被稱為兩國，也就是日本橋兩國。

== 地理 ==
屬墨田區{{link-ja|本所區|本所区|本所地域}}。北與墨田區[[橫網|橫網]]之間以[[總武本線|總武線]]路廊為界，南與墨田區千歲之間以{{link-ja|豎川 (東京都)|竪川 (東京都)|豎川}}及[[首都高速7號小松川線|首都高速小松川線]]高架道路為界，西與[[中央區_(東京都)|中央區]][[東日本橋|東日本橋]]、[[台東區|台東區]][[柳橋_(台東區)|柳橋]]之間以[[隅田川|隅田川]]為界，東與墨田區綠之間以[[東京都道463號上野月島線|清澄通]]（地下是[[都營地下鐵大江戶線|都營地下鐵大江戶線]]）為界。町内有[[國道14號|國道14號]]（[[京葉道路|京葉道路]]）通過。此外，「兩國」也通稱[[兩國站|兩國站]]周邊地區。最早是指{{link-ja|兩國橋|両国橋}}周邊。

== 歷史 ==
[[江戶時代|江戶時代]]的1659年（[[萬治|萬治]]2年）12月13日，幕府在此架橋，稱為'''大橋'''，通稱「兩國橋」。

兩國橋連結了現在的兩國與[[神田_(千代田區)|神田]]、[[日本橋_(東京都中央區)|日本橋]]方向，使得江戶能擴展到隅田川東側。現在的兩國地區還包含了北側的[[本所_(墨田區)|本所]]、南側的[[深川_(江東區)|深川]]兩處江戶時代新興住宅區。

1686年（[[貞享|貞享3年]]）（一說是[[寛永|寛永年間]]（1622年-1643年）），{{link-ja|葛飾郡|葛飾郡|南葛飾郡}}編入[[武藏國|武藏國]]，隅田川（當時名為大川）是[[下總國|下總國]]與武藏國的國境，大橋（兩國橋）是跨越兩方的橋，因此稱為'''兩國'''橋。

元祿14年8月19日（1701年9月21日），[[江戶幕府|江戶幕府]]命令{{link-ja|吉良義央|吉良義央}}從吳服橋搬至兩國3丁目（當時稱為「本所」）的{{link-ja|松平信望|松平信望}}屋敷，後發生[[元祿赤穗事件|元祿赤穗事件]]。

總武線[[兩國站|兩國站]]在1904年開業（當時站名為「兩國橋站」）。這是為了讓本所站（現在的[[錦糸町站|錦糸町站]]）能駛入都心方向而作的延伸，隅田川架橋工成的困難造成往[[秋葉原站|秋葉原]]、[[御茶之水站|御茶之水]]方面的延伸線延期。1932年延伸後至1972年[[錦糸町站|錦糸町]]-[[東京站|東京]]間的地下線開業為止，往[[千葉站|千葉]]以東方向的列車多以兩國站為始發站。站前有[[東京都電車|路面電車]]可搭乘，此時的兩國站作為終點站而活躍。

兩國因相撲場館[[兩國國技館|兩國國技館]]而成為全國知名的地名。[[辰野金吾|辰野金吾]]與{{link-ja|葛西萬司|葛西萬司}}設計的舊兩國國技館位於兩國二丁目{{link-ja|回向院|回向院}}舊境內，之後移至{{link-ja|藏前國技館|蔵前国技館}}。1985年，新的兩國國技館完工，位於兩國站北側的墨田區橫網。因此，兩國地區有許多[[相撲部屋|相撲部屋]]與[[相撲火鍋|相撲火鍋]]店，宛如相撲之町。此外，舊國技館在1983年拆解後曾作為[[日本大學|日大]]講堂使用，現在是辦公大樓、住宅、[[劇場|劇場]]、餐廳等聚集的複合設施「兩國CITY CORE」（両国シティコア）。

以[[元祿赤穗事件|元祿赤穗事件]]聞名的吉良屋敷在本所松坂町（現在兩國二丁目至兩國三丁目）。現在區立{{link-ja|本所松坂町公園|本所松坂町公園}}內保留一部份的宅邸史蹟。區立兩國公園内有[[勝海舟|勝海舟]]誕生地的石碑。吉良邸跡南側、竪川北岸的鹽原太助住居有座鹽原橋。

=== 地名、町名由來 ===
地名的「兩國」是來自隅田川上的「兩國橋」。「兩國橋」因連結了西側的[[武藏國|武藏國]]與東側的[[下總國|下總國]]而得名<ref>{{Cite book|title = 地名で読む江戸の町|author = 大石学|isbn = 978-4569615486|language = ja|publisher = PHP研究所|date=2001年3月}}</ref>。原先，本地位於「兩國橋」東側而稱「東兩國」，後町名簡化成「兩國」。

=== 沿革 ===
*1929年（[[昭和|昭和]]4年）成立東兩國一～四丁目。
*1966年（昭和42年）去除「東」的冠稱，為現行的'''兩國一～四丁目'''。

== 中央區側的兩國（日本橋兩國） ==
本來兩國是指隅田川西側的武藏國側，元祿（1688）時代開發本所深川以後，現在兩國的舊下總國側（墨田區側）稱為'''東兩國'''。包含北方柳橋的兩國是江戶著名的歡樂街。兩國橋西詰的'''兩國廣小路'''[[畸形秀|見世物]]等小屋林立。明治11年東京市15區創設後，西側稱為日本橋兩國，東側稱為本所東兩國。本所、深川從過去的下町（芝、京橋、日本橋、神田）來看是位於「對岸」，因此形成西側是較優越的本家兩國，對岸是較低下的東兩國的看法。

隨著兩國站開業與兩國國技館開館，兩國逐漸縮減為東兩國地區。1971年實施{{link-ja|住居表示|住居表示}}，中央區日本橋兩國周邊的日本橋米澤町、日本橋藥研堀町改稱為[[東日本橋|東日本橋]]。墨田區側在1967年從東兩國改稱兩國。

東日本橋町内原來有條東日本橋兩國商店街，2004年左右更名為東日本橋藥研堀商店街。現在中央區内有保留兩國名稱的設施僅剩下兩國廣小路的石碑與兩國郵便局。

此外，藥研堀是[[七味唐辛子|七味唐辛子]]的發源地（七味也有人稱為「藥研堀」）。

平成25年4月，墨田區側的JR兩國站前進行「兩國站廣小路」的站前廣場整備工程。雖然是模仿當時「喧囂的兩國廣小路」，但因為與實際的兩國廣小路地點不同，因此在命名上作出區別避免混淆。

== 兩國花火 ==
1732年（[[享保|享保]]17年）起，全國爆發大饑荒（[[享保大饑荒|享保大饑荒]]），江戶也爆發霍亂，八代[[征夷大將軍|將軍]][[德川吉宗|德川吉宗]]為了弔念死者與驅趕病魔，在兩國舉行水神祭與[[施餓鬼|施餓鬼]]。此時會在河上施放煙火，就是後來兩國花火大會的起源。1733年（[[享保|享保]]18年）5月開始舉行，負責的花火師是兩國的篠原彌兵衛，有名的「鍵屋」六代目。後與「玉屋」一同負責花火大會。

兩國花火大會曾數度中斷，是日本歷史最悠久的花火大會。1978年（[[昭和|昭和]]53年）在中斷15年後重新舉行，後改稱為「'''隅田川花火大會'''」。

== 大江戶、兩國機關人偶祭 ==
2006年起於毎年10月舉行，祭典裡有各種[[機關人偶|機關人偶]]的展示、江戶手妻（てづま）的介紹等等，並有街頭藝人表演。

2011年改名為「大江戶兩國、傳統祭」，介紹江戶手妻、殺陣、琵琶、曲藝、大道藝等江戶時代的娛樂活動。

== 兩國地區的知名人物 ==
* [[勝海舟|勝海舟]]
** 著名的{{link-ja|幕末三舟|幕末の三舟}}之一。出生於本所龜澤町（現在的兩國4丁目）。區立兩國公園内有「勝海舟誕生地」石碑。
* [[芥川龍之介|芥川龍之介]]
** 1910年代-1920年代活躍的日本代表性短編[[小說家|作家]]。出生於{{link-ja|京橋區|京橋区}}[[入船_(東京都中央區)|入船町]]（現在的[[中央區_(東京都)|中央區]][[明石町_(東京都中央區)|明石町]]）、成長於本所小泉町（現在的兩國3丁目）。當地有「芥川龍之介成長地」石碑。
* {{link-ja|大橋巨泉|大橋巨泉}}
** 全方位藝人的先驅，1960年代-1980年代活躍的電視[[主持人|主持人]]。2001年-2002年擔任[[日本參議院|日本參議院]]議員。出生於兩國。
* {{link-ja|石橋省三|石橋省三}}
** 前[[NHK|NHK]][[播音員|播音員]]。長年擔任[[電視|電視]][[廣播|廣播]][[大相撲|大相撲]]實況轉播。出生、成長於兩國。
* {{link-ja|加藤晴久|加藤晴久}}
** 1935年生。在東京大學教養學部擔任教職長達27年。現在是東京大學與惠泉女學園大學名譽教授、法語教育振興協會副理事長、NHK電視廣播的法語講座講師。1996年參加有來自世界110國法語教師參加的國際會議。曾獲法國政府授與{教育功勞章、[[法國藝術及文學勳章|藝術及文學勲章]]。
