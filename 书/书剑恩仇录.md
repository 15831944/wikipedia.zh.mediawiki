《'''書劍恩仇錄'''》是[[金庸|金庸]]筆下第一本[[武侠小说|武侠小说]]，已经被多次改编成[[剧本|剧本]]拍成[[电影|电影]]和[[电视剧|电视剧]]。

時間是在[[清朝|清朝]][[乾隆帝|乾隆]]年間。出場的真實歷史人物包括[[乾隆帝|乾隆帝]]、[[陈世倌|陈世倌]]、[[兆惠|兆惠]]、[[福康安|福康安]]等。

==故事大綱==
故事講述[[清朝|清朝]][[乾隆|乾隆]]年間，反清[[幫會|幫會]]「[[紅花會|紅花會]]」一眾頭目的事蹟。

乾隆年間，江湖上勢力最雄厚的[[反清|反清]]幫會為紅花會，其時總舵主-{r|于}-萬亭身故，遺命立天池怪俠袁士霄之弟子，也是自己義子的陳家洛接任。紅花會內外香堂十三名頭目領命遠赴回疆迎接[[陳家洛_(書劍恩仇錄)|陳家洛]]，其間四當家奔雷手文泰來因掌有皇帝身世的秘密，而遭遇朝廷眾高手追捕，連番力戰下不敵被擒。陳家洛與其餘頭目合力營救，過程中認識一群為了奪[[回部|回部]]聖物而與朝廷對抗的回部英雄，彼此互助，結成生死之盟。此外又巧遇武當高手「綿裏針」陸菲青、鐵膽莊主周仲英等英雄，相互結交，終在連番波折後，成功救出文泰來，並得知皇帝身世之秘。

原來當時的皇帝[[愛新覺羅弘曆|愛新覺羅弘曆]]（即慣稱的「乾隆皇」）本為漢人，襁褓時被當時的四皇子[[胤禛|胤禛]]以女嬰換去，此後即被當作胤禛之子，並繼承皇位，而乾隆的真正身份竟是陳家洛的親哥哥，女嬰則是陳家洛所稱的大姊。前紅花會總舵主-{r|于}-萬亭正因得知此事，希望以此為契機，藉由乾隆改服漢人衣冠，一舉推翻滿清皇朝，故此遺命眾人擁立乾隆親弟陳家洛為紅花會之主，一則動之以親情，二則以此為脅，謀舉大事。紅花會眾頭目為成其事，遂傾全幫之力，連同陸菲青、周仲英等人，合力擒獲乾隆，並最終訂下反滿復漢之盟。正當眾人心慶大事可期之際，卻傳來朝廷與回部大舉開戰的消息，為全義氣，眾人遂火速赴援。回部一戰，讓陳家洛邂逅回部美女香香公主喀絲麗，相互許下終身之約，又得悟曠世武學，武功大進，並以此撃敗生平第一強敵火手判官張召重。

重回中原後，竟傳來消息謂皇帝欲納香香公主為妃。陳家洛得訊即與乾隆聯絡，欲相告與香香公主已訂婚盟之事，卻反遭乾隆以復興大事為脅，迫其幫助勸說香香公主入宮為妃，陳家洛終無奈答允。爾後卻突然傳來香香公主自殺而死的惡耗，眾人心痛之餘，又得悉乾隆背棄盟約，欲將眾人連同陳家洛一舉擒殺。其時正值紅花會群雄大會京師，遂與大內眾高手決一死戰，硬闖突圍。此後陳家洛心傷愛人慘死，又感復興無期，意興闌珊，遂與紅花會群雄豹隱回疆。

==主要角色==
{{Main|書劍恩仇錄角色列表}}
*[[陳家洛_(書劍恩仇錄)|陈家洛]] ( 男 ) —— 愛作士宦家貴公子打扮，漢人陳世官之次子，紅花會總舵主[[于萬亭|于萬亭]]之義子。師从天池怪侠袁士霄，武器為九條珠索（打穴）和奇型盾牌，後繼任于萬亭為總舵主。
*[[霍青桐|霍青桐]] ( 女 ) —— [[回部|回部]]女子，外號「翠羽黃衫」，天山派天山雙鷹之弟子。
*[[文泰來|文泰來]] ( 男 ) —— 紅花會的四當家，外號「奔雷手」，暱稱「四哥」，在紅花會中也是數一數二的高手，擅長使用剛猛的拳法，妻子為十一當家鴛鴦刀駱冰。
*[[駱冰|駱冰]] ( 女 ) —— 紅花會十一當家，綽號「鴛鴦刀」，暱稱「冰妹」，丈夫為四當家奔雷手文泰來。
*[[余魚同|余魚同]] ( 男 ) —— 紅花會十四當家。形象英俊瀟灑，一表人才。擅長笛，武器即是一根以金鑄造的笛子，曾中秀才，因此被譽為「金笛秀才」。起初暗戀四嫂駱冰，後走出不倫之戀，與李沅芷相戀。
*[[李沅芷|李沅芷]] ( 女 ) —— 小說中是浙江[[提督|提督]][[李可秀|李可秀]] ( 虛構人物 ) 的愛女。李沅芷清秀可人，會武功，師承[[陸菲青|陸菲青]]，喜歡女扮男裝，性格精靈刁蠻，擅出鬼點子，是個正義的小俠女，與余魚同相戀。
*[[喀絲麗|喀絲麗]] ( 女 ) —— 霍青桐的妹妺，回部人稱「香香公主」。
*[[乾隆皇|乾隆皇]] ( 男 ) —— 小說中為滿清的漢人皇帝，漢人陳仕官之長子，陳家洛之親兄，化名東方耳。

==出版情况==
1955年在[[香港|香港]]《[[新晚報|新晚報]]》发表，連載一年。曾以《書劍江山》為名出版（臺灣《[[聯合報|聯合報]]》連載時亦用此標題）。新修版於2002年推出，除部分內容有所修訂外，第二十回後更增補一章{{〈}}魂歸何處{{〉}}，約五千多字，「從陳家洛口中，引出金庸對於人生、情愛、民族的種種深刻思索」<ref>引自[http://www.ylib.com/hotsale/new_jin/allbook1.htm#01 遠流博識網]</ref>。

===譯本===
*英譯：{{lang|en|''[http://www.earnshaw.com/other-writings/book-sword The Book & The Sword]''}}，{{lang|en|[[Graham_Earnshaw|Graham Earnshaw]]}} 翻譯，[[閔福德|閔福德]]（{{lang|en|John Minford}}）和{{lang|en|[[Rachel_May|Rachel May]]}} 編輯，2005年由[[牛津大學出版社|牛津大學出版社]]出版。
*日本[[德間書店|德間書店]]於1996年10月至1997年1月間出版的金庸武俠小説日譯本第一波作品，由[[岡崎由美|岡崎由美]]翻譯，全4冊。2001年4月至2001年5月另發行文庫版全4冊。

== 注釋 ==
{{references}}
在1949年之前，“[[回人|回人]]”一稱是廣義的信仰伊斯蘭教的少數族群的稱呼，甚至狹義地更指為新疆的維吾爾族。比如孫中山所提出的[[五族|五族]]共和中的“回”即主要指新疆的少數民族。這一點可以從小說中對應的“回疆”這一歷史稱為看出。而今天的“[[回族|回族]]”，在此之前被廣泛稱為“回民”，並不認定為少數民族，而是宗教族群。今天[[維吾爾族|維吾爾族]]的官方名稱是二十世紀三十年代才提出，並且在1949年之後民族識別的時候才確定的。舊稱「回人」，即指維吾爾人或[[新疆|新疆]]的其他穆斯林民族無疑。而維吾爾人的可能性最大。作為少數民族的上層人士，霍青桐及其父與漢人有更多交往甚至熟識[[漢語|漢語]]都不奇怪。維吾爾語屬[[突厥語|突厥語]]族，甚至維吾爾族屬於泛[[突厥|突厥]]民族都是沒問題的。不過金庸在名字的創作上想必未必做了考證，也就不用細究。
