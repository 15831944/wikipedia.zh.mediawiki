'''泽漆'''（[[学名|学名]]：{{lang|la|''Euphorbia helioscopia''}}）为[[大戟科|大戟科]][[大戟属|大戟属]]的植物。

== 形态 ==
二年生[[草本植物|草本植物]]，含有乳汁。匙形或狭倒卵形叶子，边缘有细锯齿；春夏开花，具有大戟科特有的杯状花序。

== 分布 ==
分布于[[欧亚大陆|欧亚大陆]]、[[北非|北非]]以及[[中国大陆|中国大陆]]的[[广布全国|广布全国]]等地，生长于海拔50米至3,800米的地区，常生长在荒野、路旁、山沟以及山坡，目前已由人工引种栽培。

== 中医用途 ==
以茎、叶入药，性微寒，味苦辛，小毒，功能利水消肿、化痰散结、杀虫，主治水肿腹胀、瘰疬结核、肺热咳喘等症。

== 别名 ==
五朵云，五灯草，五风草（别名）
