'''廣式普通話'''，簡稱'''廣普'''，指夾雜[[粵語|粵語]]發音或文法的[[普通話|普通話]]，由於普通話和粵語語音差別很大，形成文化差異，「'''普通話'''」更被[[廣東人|廣東人]]譏稱為「'''煲冬瓜'''」、「'''刨冬瓜'''」（谐音），故有[[俗語|俗語]]「天不怕，地不怕，就怕老廣說北方話」。

== 形成 ==
发音方面，和粵語有很大差別：普通话无[[入聲|入聲]]、有[[翹舌音|翹舌音]]、有[[儿化韵|儿化韵]]等；所以在粵語同音的字，在普通話未必同音、同调；而且，不少人不習慣北方口音導致到有[[懒音|懒音]]的问题（特別是港式普通話），他们学习普通话时会遇上更大困难。例如：
* '''继续''' 读成 '''记住'''
* '''坏了''' 读成 '''歪了'''
* '''讓''' 读成 '''樣'''

在词汇上，粵語的[[口語|口語]]用词、外语专有名词[[譯名|譯名]]又和普通話不同。例如：
* '''Ronaldo'''普通話譯'''[[羅納爾多|羅納爾多]]'''（luona'erduo），粵語譯作“朗拿度”（long<sup>5</sup>na<sup>4</sup>dou<sup>6</sup>）
* 用'''老细'''稱呼'''老闆'''（实为「老世」，在[[粤语|粤语]]中同音）


另外还有一种现象，是用「归纳推理」出的「普通话发音」来读粤语特有词汇，或直接使用粤语读音。例如：
* 粵語 '''lun<sup>6</sup>zun<sup>6</sup>'''（「论尽」，意思为「笨手笨脚」）读成普通话拼音 '''lūnzūn'''

在中国大陸[[改革开放|改革开放]]之前，由于户籍原因，人口不能随意迁移，粤语人口甚少接触到普通话。广式普通话的出现主要是[[改革开放|改革开放]]之后，北方人迁移至[[广东|广东]]，广东的粤语人口为了方便沟通，便跟他们讲语音不同的普通话，久而久之便形成一种独特的口语，所以广式普通话仅于广东人与外地人交谈时出现。但于[[改革开放|改革开放]]后，中国加强[[推广普通话|推广普通话]]政策，所以在80年代后出生的人基本不会讲广式普通话，而是讲较标准的普通话。

1980年代[[香港|香港]]与[[中国內地|中国內地]]的商業上交往密切，与[[广东|广东]]的情况相类似，年輕一輩會主動自學，或主要從中小學學校的教授普通話科中學習基本普通话，但仍是很「粵語交集」的，並不懂普通話使用的專有詞彙。廣式普通話，又稱港式普通話（港式國語） 。

==特点==
和众多南方地区一样，外加粤语没有任何词汇是带有[[翘舌音|翘舌音]]，所以只有平音词汇。基本上不会发翘舌音，zh、ch、sh、er多数读成z、c、s、e，甚至有些词汇部分后鼻音例如ang、eng、ing多数读成前鼻音an、en、in。

如粵語存在“你走先”的句式。

== 例子 ==
廣式普通話經常鬧出笑話，以下是常见的例子：
# 原句︰坐在船頭看郊區。<br>變成︰坐在床頭看嬌妻。
# 原句︰我是搭[[飛機|飛機]]來的。<br>變成︰我是[[打飛機|打飛機]]來的。
# 原句︰[[水餃|水餃]]多少[[錢|錢]]一碗？<br>變成︰[[睡覺|睡覺]]多少錢一晚？
# 原句︰純得像[[白紙|白紙]]。<br>變成︰蠢得像[[白癡|白癡]]。
# 你們先滾吧！<br>意思是打[[火鍋|火鍋]]時想讓人先用，但誤用詞語，變成罵人說話。
# 要飯嗎？<br>意思是「要添點[[米飯|米飯]]嗎」，但誤用詞語，變成詢問對方是不是[[行乞|行乞]]。
