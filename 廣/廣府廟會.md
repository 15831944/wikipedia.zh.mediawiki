'''廣府廟會'''，是一個由[[广州市人民政府|廣州市政府]]在新春時節發起舉辦、融合本地與北方元素的一個的嘉年華（本土傳統是神誕墟市），是廣州市[[越秀區|越秀區]]政府借亞運修復[[广州都城隍庙|城隍廟]]後，於2011年[[元宵节|元宵節]]開始首屆「廟會」<ref>{{Cite web|url=http://www.gz.gov.cn/gzgov/msjq/201610/bb6d1c99c99b4a968ccc8b50588cc79e.shtml|title=民俗节庆 广府庙会|accessdate=|author=|date=2016-10-25|publisher=广州市人民政府}}</ref>。

== 内容 ==
會期是農曆正月十五到正月廿一，期間有木偶薈萃、中華絕活、武林大會和元宵燈會等10個主題活動，還包含了祈福、民俗、美食、商貿與休閒等不少附加活動。其中将城隍诞搞成此官方盛会的一部分，广州城隍出巡会在期间结合各色民俗文化向游客等展示<ref>{{Cite web|url=http://cul.china.com.cn/minsu/2012-02/12/content_4803495.htm|title=广州广府庙会大巡游 传统民俗中增添“洋味”|accessdate=|author=|date=2012-02-12|publisher=文化中国-中国网}}</ref>。當局是通過製造一個新廟會的方法，創建不同於地方傳統的大中華化廣府民俗文化特色活動，借助民俗文化巡游表演向市民和游客展示新式文化，加大影響力<ref>{{Cite web|url=http://www.gzlib.gov.cn/gzsd/50459.jhtml|title=城隍爷大阵仗出巡 洋小伙执幡数万人围观|accessdate=|author=广州日报|date=2012-02-12|publisher=广州图书馆}}</ref>，向遊客與市民宣傳展示官方打造的廣府文化。

== 緣起 ==
越秀區文廣新局局長王衛國表示，2010年12月剛上任不久，就收到任務籌建「廣府廟會」，就同陳曉丹、何願飛等人去[[五仙觀|五仙觀]]處開神仙會謀劃，由此開創活動。而越秀區常委、宣傳部長陳曉丹說，由於看見2009年越秀區搞過的廣府文化嘉年華活動，很受市民歡迎，就決定仿照北方廟會的形式，打造一個文商旅相融合的活動平臺。王說，活動的常態目標第一是廣大市民節日狂歡，第二是做成廣府文化的盛宴，第三是做成文化、商業、旅遊融合的新名片<ref>[http://news.artron.net/20150313/n720730.html 广府庙会五岁了　明年还要“搞搞新意思”] 2015-03-13  新快報  鄧毅富、朱海靖</ref>。

== 本土化問題 ==
廣州傳統是沒有辦廟會這回事，在廣州類似於廟會的民間活動是'''誕'''<ref name=":1">[http://www.gd.chinanews.com/2017/2017-12-07/2/391607.shtml 广府庙会“粤”精彩] 2017年12月07號 中新網廣東</ref>，广府传统'''诞会'''是'''神诞'''与'''赛会'''（或“出会”）的结合，故称诞会。从时间和构成看，传统诞会主要分'''有诞无会'''（如新桥天后诞）、'''诞期出会'''（如钟村康公出会）和'''非诞期出会'''（如十乡会）等三种类型<ref name=":0">{{Cite web|url=http://www.cssn.cn/zt/rwln/wh/tswh/201710/t20171025_3680708.shtml|title=弘扬广府特色诞会民俗文化|accessdate=|author=朱光文|date=2017-10-25|publisher=中国社会科学网}}</ref>。

廣府廟會並非廣東傳統的本土[[嶺南文化|廣府文化]]、民系、[[粵語|粵語]]的組成，而這個廣府與[[河北省|河北省]]的廣府城、廣府古城也是完全沒有關係。[[中山大学|中山大学]]中文系教授叶春生认为，尽管元宵逛庙会并非岭南传统，但越秀区举办广府庙会，可作为文化展示平台，吸引大量人气<ref>{{Cite web|url=http://fashion.ifeng.com/travel/around/gz/detail_2013_02/05/21963531_0.shtml|title=行广州最老花街 逛新式广府庙会|accessdate=|author=新快报|date=2013-02-05|publisher=}}</ref>。亦有意见指应培育回'''广府诞会'''，保障南粤民间信仰与迎神赛会的话语体系可进行基于传统的传承和建构<ref name=":0" />。
