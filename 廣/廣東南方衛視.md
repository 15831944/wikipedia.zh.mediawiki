'''广东南方衛視'''（TVS-2/2<sup>★</sup>，简称'''南方卫视'''，又称'''广东广播电视台粤语频道'''），是一个以[[粤語|粤語]]為主的[[衛星電視|衛星電視]]頻道，是中國大陸獲[[国家广电总局|国家广电总局]]批准上星的地方語言電視頻道。该台稱該頻道是中國內地第一個純粵語播出的衛星電視頻道。

== 历史 ==
南方衛視的前身为1994年6月2日启播的'''[[广东有线|广东有线广播电视台]]第1台都市频道，'''2001年7月1日，广东有线广播电视台被撤销，其体育频道划归广东电视台，余下的其他频道与[[广东经济电视台|广东经济电视台]]整合成“[[南方电视台|南方电视台]]”，'''南方電視台都市頻道'''正式啟播。

2004年7月28日，频道上星，分為在廣東省內外播出的兩個頻道，省內版繼續使用'''南方都市'''的呼號，衛星版則命名為'''南方電視台衛星頻道'''（簡稱'''南方衛視'''）。兩個頻道實行兩套節目表。

2010年，南方衛視和南方都市統一頻道呼號為'''南方衛視'''，省内版命名為南方衛視（地面版），省外版則命名為南方衛視（上星版）<ref>{{Cite web|url=http://epg.gdtv.cn/?c=15|title=TVS2广东南方卫视-上星版 - 节目表 广东广播电视台|accessdate=2017-10-13|author=|date=|publisher=}}</ref>。上星版的台標在地面版台標的基礎上以數字2的右上角加上小星分別。

2014年5月起，因應原南方廣播影視傳媒集團、[[广东人民广播电台|广东人民广播电台]]、廣東電視台、南方電視台、廣東廣播電視技術中心合併為[[广东广播电视台|广东广播电视台]]後的調整，原南方衛視更名為“'''廣東南方衛視'''”，但台标保持不變。

2015年7月起，廣東南方卫视更换节目包装，同时启用全新口号“心随梦想”<ref>{{Cite web |url=http://m.yinews.cn/article/771540.shtm |title=广东南方卫视全新上线，将领衔众多节目一路飘红！ |access-date=2016-07-31 |archive-url=https://web.archive.org/web/20160816175427/http://m.yinews.cn/article/771540.shtm |archive-date=2016-08-16 |dead-url=yes }}</ref>。

== 主要節目 ==
頻道的主要節目有《[[都市笑口組|都市笑口組]]》、《[[今日最新闻|今日最新聞]]》、《[[城事特搜|城事特搜]]》、《我愛返尋味》、《粤唱粤好戲》、《[[七十二家房客_(电视剧)|七十二家房客]]》、《嶺南風情畫》、《谁语争锋》等。

大部分節目使用[[粵語|粵語]]（[[廣州話|廣州話]]）播出，但也有個別節目（含新聞）中的主持人或記者使用普通話，也經常出現普通話的商業廣告。

== 各地播放 ==
南方衛視通過衛星覆蓋全球大多數地區。在中國大陸，南方衛視覆蓋[[廣東|廣東]]全境及[[廣西|廣西]]、[[海南省|海南]]部分地區。

=== 地面版（TVS-2，廣東省內） ===
南方衛視透過廣東省內各有線電視平台（[[廣東有線|廣東有線]]、[[廣州有線|廣州有線]]、[[深圳市|深圳]]天威視訊）及[[地面數碼電視|地面數字電視]]（數碼地面電視廣播）免費播出。

=== 上星版（TVS-2<sup>★</sup>，廣東省外） ===

==== 衛星參數 ====
南方衛視主要由[[中星6A|中星6A]]衛星傳輸，接收參數為：衛星軌道[[东经|東經]]125度，[[中心频率|中心頻率]]3845MHz，[[偏振|偏振]]方式水平，前向糾錯3/4，符號率17.778。另在[[亚太5号|亞太5號]][[长城平台|長城平台]]（東經138度Ku波段）也有轉播。

==== 廣西 ====
2012年12月28日，广西全区有线电视中的南方卫视停播，广西广电网络表示，因南方电视台和广西广电网络的落地协议到期而停播南方卫视，原频道位置变成[[中国中央电视台综合频道|中国中央电视台综合频道]]。至此，除梧州外，广西各个地方的有线电视网络已无任何一个粤语电视频道。梧州方面，原[[广东电视台珠江频道|珠江台]]的位置改用南方卫视替代，原南方卫视变为央视综合频道，珠江台被改以“测试频道”的名义播放。

==== 海南 ====
海南有线曾经传送过[[广东广播电视台珠江频道|珠江台]]节目，2013年替换为南方卫视至今。

==== 香港 ====
2004年8月，开始透過[[香港寬頻bbTV|香港寬頻bbTV]]播出。2017年1月1日，因香港寬頻bbTV終止服務，停止在该平台播出。

2005年6月，开始透過[[now寬頻電視|now寬頻電視]]播出。

2006年11月5日，开始透過[[無綫網絡電視|無綫網絡電視]]播出。2011年11月1日，因合约结束，停止在無綫網絡電視播出。[[File:TVS-2_Logo_by_aTV.png|thumb]]
香港亞洲電視于2009年10月1日早上7:00開始通过数字电视13频道轉播南方衛視。由於南方衛視以[[4:3|4:3]]播出的原因，亞洲電視轉播版本左右兩邊以[[黑邊|黑邊]]填補空白，同時左上角加入亞洲電視台标，台标顏色修改至接近南方卫视台标的[[橙黃色|橙黃色]]。

2011年1月1日凌晨零時起，亞視所有自製及轉播頻道的台标颜色均統一為橙色，及後左右兩邊用来填充画面空白的长条與[[本港台|本港台]]、[[亚洲电视国际台|國際台]]、[[亞洲電視亞洲台|亞洲台]]一樣的'''白色為主色的靜止圖板'''取代。

與香港收費電視轉播平台全天24小時轉播情況不同，亞視在凌晨1:00-6:00會播出过往[[本港台|本港台]]播放的電視劇與資訊節目，部分跟首播日期更有十多年差距。原因是亞視曾向[[廣管局|廣管局]]承諾會於香港地面数字電視廣播頻譜上開設[[亞視資訊娛樂台|亞視資訊娛樂台]]（即後來的[[歲月留聲|歲月留聲]]），並在新頻道的全天节目安排内最低提供百分之二十的本地製作節目，惟因亞視財困最終擱置開設此台，並引入南方衛視替代，而亞視原先承諾推出的兩成本地製作則以凌晨时段重播舊節目充數。

在2011年2月15日，亚视转播该频道的《[[今日最新闻|今日最新聞]]》時，突然在畫面顯示有關節目需要停播，却未明確交代原因。2月16日，亞視發出新聞稿表示節目內含有違反香港電視業務（節目及廣告）守則內規定的廣告宣傳內容，接獲廣管局裁決及指示後需要停播此節目<ref>[http://www.tudou.com/programs/view/qWTgHDvvf64/?spm=a2h0k.8191414.qWTgHDvvf64.A.qArwo7 南方衛視今日最新聞被香港廣管局指示於2月16日開始在亞洲電視數字頻道屏蔽]</ref><span style="font-size:smaller;">（[https://web.archive.org/web/20110222082213/http://app.hkatv.com/atvnews/main.php?id=3721 新聞稿原文]）。</span>11月3日中午起恢復播放今日最新聞，但用圖形覆蓋植入式廣告的部分。香港收費電視轉播平台沒有受任何影響。

[[亞洲電視|亞洲電視]]於2012年9月28日宣佈，由同年10月1日凌晨開始停止轉播南方衛視<ref>[https://www.youtube.com/watch?v=C8AyiynUunw TVS 停止放映]</ref><span style="font-size:smaller;">（[https://web.archive.org/web/20121008030700/http://app2.hkatv.com/v5/atvnews/atvnews.php?id=4515 新聞稿原文]）</span>。原有轉播中的[[深圳衛視|深圳衛視]]和[[中央電視台綜合頻道|中央電視台綜合頻道]]則維持不變，12月31日晚23:30，13頻道改為播出[[歲月留聲|歲月留聲]]，直至亞洲電視於2016年4月2日[[亞洲電視欠薪及不獲續牌事件|本地免費電視牌照屆滿而停播]]。

==== 澳門 ====
[[澳門公共天線公司|澳門各公共天線公司]]的網絡都采用模擬制式電視傳送南方衛視（信号为亞太5號長城平台的版本）。2013年8月8日起，遵守澳門中級法院對有線與公天的判決，原有公共天線公司的自行接收的信號改為使用澳門有線信號，原因为該頻道未有列入頻道表中而在澳門公天網絡中取消，现通过纯[[澳門有線電視|澳門有線電視]]平台亦可收看南方卫视节目。

==== 海外 ====
南方衛視透過中國海外[[长城平台|長城平台]]覆蓋全球大多數地區，于當地卫星、有线电视和IPTV平台播放。

===上星版（广东省外）===
除播出广东南方卫视节目外，省外版亦会转播原[[南方电视台|南方电视台]]其他频道的节目，如[[TVS1|TVS1]]的《一起旅游吧》和《真实故事》、[[TVS3|TVS3]]的《全民放轻松》、[[TVS5|TVS5]] 的《放学我当家》等。
