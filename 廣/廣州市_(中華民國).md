'''廣州市'''，簡稱「穗」，是[[中華民國|中華民國]]在[[兩岸分治|兩岸分治]]前所設置的12[[直轄市_(中華民國)|直轄市]]之一，亦是中華民國政府遷台前[[华南地区|华南地区]]唯一的直轄市；。是南部地區交通樞紐中心，亦是嶺南政治、經濟、文化中心。

=== 廣東軍政府时期 ===
广州作为中国近现代革命的策源地，19世纪末，[[辛亥革命|辛亥革命]]元老中国现代教育奠基人[[何子渊|何子渊]]、[[丘逢甲|丘逢甲]]等于此地积极创办和推广新式学堂，不仅培育一大批思想进步锐意创新的社会精英，而且还催生“折衷中西，融汇古今”的[[岭南画派|岭南画派]]，为辛亥革命积蓄巨大能量。而1911年爆发的[[黃花崗起義|“三·二九”起义]]，打响辛亥革命的第一枪，拉开辛亥革命的序幕。<ref>{{cite news |url = http://www.chinanews.com/cul/2011/03-28/2933787.shtml |title=“三·二九”广州起义：辛亥革命的第一枪| publisher=广州日报|date= 2011-03-28}}</ref>

1910年和1911年，[[倪映典|倪映典]]率千余名新军于[[燕塘起义|燕塘起义]]及[[黄兴|黄兴]]等人发起的[[黃花崗起義|黃花崗起義]]都以失败告终<ref>{{cite book |editor = 李禎蓀（主編），吳佳華、黃繼勝（副主編） |title=《名城廣州小百科》 |year=1992 |publisher=中山大學出版社 |location=廣州 |isbn=7-306-00553-7 |pages=87頁、89頁 }}</ref>。

1912年[[中華民國|中華民國]]成立後废[[广州府|广州府]]，成立广东军政府。1917年7月，孙中山抵达广州准备另立軍政府。8月，约百名原在北京的国会议员在广州召开“国会非常会议”，成立中华民国军政府，选举孙中山为大元帅，行使中华民国行政权，展开[[護法運動|護法運動]]。后因“非常国会”倒向[[桂系|桂系]]，迫使孙离开广州。同年设立广州市政公所。1918年10月，市政公所发出第一号布告，宣布将拆除[[廣州城牆|全部城墙]]，将旧城墙基开辟为马路。1921年2月15日，《广州市暂行条例》公布实施，广州市政厅成立，广州也因此成为中国第一个市<ref>{{cite news |url = http://www.gzminge.org.cn/infodetail.jsp?id=809 |title = 广州成为中国第一个市 |publisher = 中国国民党革命委员会广州市委员会 |accessdate = 2010-08-30 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1921年4月，非常国会声称成立中华民国政府，选孙中山为大总统，在广州就职<ref>{{cite book |editor = 李禎蓀（主編），吳佳華、黃繼勝（副主編） |title=《名城廣州小百科》 |year=1992 |publisher=中山大學出版社 |location=廣州 |isbn=7-306-00553-7 |pages=90頁 |chapter=孫中山在廣州三次建立政權 }}</ref>。[[中國國民黨|中國國民黨]]1924年在广州举行[[中國國民黨第一次全國代表大會|第一次全国代表大会]]<ref>{{cite book |author=廣州市文史研究館  |title=《廣州百年大事記》 |year=1984 |publisher=廣東人民出版社 |location=廣州 |chapter=一九二四年一月 }}</ref>。同年孙中山在[[长洲岛_(广州)|长洲岛]]创办[[中華民國陸軍軍官學校|中華民國陸軍軍官學校]]<ref>{{cite book |author=廣州市文史研究館  |title=《廣州百年大事記》 |year=1984 |publisher=廣東人民出版社 |location=廣州 |chapter=一九二四年五月 }}</ref>。1925年爆发历时16个月的[[省港大罷工|省港大罷工]]，期间示威人士被镇压，史称[[沙基慘案|沙基慘案]]<ref>{{cite book |author=廣州市文史研究館  |title=《廣州百年大事記》 |year=1984 |publisher=廣東人民出版社 |location=廣州 |chapter=一九二五年六月 }}</ref>。中华民国国民政府同年在广州正式成立<ref>{{cite book |author=廣州市文史研究館  |title=《廣州百年大事記》 |year=1984 |publisher=廣東人民出版社 |location=廣州 |chapter=一九二五年七月 }}</ref>。

=== 国民政府/中华民国政府时期 ===
1926年[[蔣中正|蔣介石]]在廣州东校场带领國民革命軍誓师[[北伐|北伐]]。同时，[[毛泽东|毛泽东]]、[[彭湃|彭湃]]等人在这里举办[[广州农民运动讲习所|农民运动讲习所]]，为[[中国共产党|中国共产党]]培养骨干力量<ref>{{cite book |editor=李禎蓀（主編）、吳佳華、黃繼勝（副主編） |title=《名城廣州小百科》 |year=1992 |publisher=中山大學出版社 |location=廣州 |isbn=7-306-00553-7|pages=96頁 |chapter=廣州農民運動講習所 }}</ref>。1927年“四·一二”事件之后，[[李濟深|李济深]]在广州戒严，持续逮捕处决两千多名共产党员，共产党组织受到严重破坏<ref>{{cite web |url = http://www.ce.cn/xwzx/gnsz/szyw/200705/17/t20070517_11395884.shtml|title=广州暴动之意义与教训 |date=2007-05-17 |accessdate=2010-06-02|work=新华网|publisher=中国经济网}}</ref>。12月，[[广州起义_(1927年)|广州起义]]爆发，[[张太雷|张太雷]]、[[叶挺|叶挺]]、[[叶剑英|叶剑英]]等在广州举行武装起义，建立了历时三天的广州苏维埃政府<ref>{{cite web |url = http://history.news.163.com/05/1209/12/24HFG3NK00011EBF.html |title = 1927年12月11日 广州起义爆发|date=2005-12-11|accessdate=2010-06-03|publisher=网易历史}}</ref>。

1929年至1936年为[[陈济棠|陈济棠]]主政广州时期，广州的经济、文化、交通和城市建设有显著发展，留下[[海珠桥|海珠桥]]、[[中山纪念堂|中山纪念堂]]、[[中山大学|中山大学]]五山新校舍、[[爱群大酒店|爱群大厦]]等著名建筑。<ref name="“南天王”与广州">{{cite news|url=http://gzdaily.dayoo.com/html/2007-08/04/content_20868.htm |title=陈济棠时期下的广州基建 |publisher=大洋网 |accessdate=2010-09-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20140821193411/http://gzdaily.dayoo.com/html/2007-08/04/content_20868.htm |archivedate=2014-08-21 }}</ref>1930年，广州曾一度改设特别市，同年改省辖市。陈1936年发动[[两广事变|两广事变]]，最後失败逃往香港。[[京广铁路|粤汉铁路]]也在同年修通。<ref name="新闻背景：粤汉铁路铁路百年建设史">{{citenews|url=http://news.qq.com/a/20091226/001460.htm| title=粤汉铁路的历史| publisher=腾讯网|accessdate= 2010-09-04}}</ref>

[[中国抗日战争|抗日战争]]爆发後，1938年12月广州正式沦陷，期间人口也由超过120万迅速减至30万<ref name=autogenerated1>广州市地方志·大事记·民国时期</ref>，绝大部分逃至[[香港|香港]]<ref>{{cite web|url=http://www.ycwb.com/gb/content/2005-06/09/content_917848.htm|title=广州沦陷期的畸形繁荣|date=2005-06-09|accessdate=2010-06-02|work=羊城晚报|publisher=金羊网|archive-url=https://web.archive.org/web/20101024225847/http://www.ycwb.com/gb/content/2005-06/09/content_917848.htm|archive-date=2010-10-24|dead-url=yes}}</ref>。1942年香港沦陷后发生粮荒，又有逾46万人回穗<ref name=autogenerated1 />。1945年8月15日[[中華民國國军|中華民國國军]]进入，广州光复。1947年復為[[行政院|行政院]]院轄市。

1949年2月，[[中国人民解放军|中国人民解放军]]逼近长江，[[中華民国政府|中華民国政府]]南迁广州<ref>{{Cite web |url = http://history.news.163.com/08/0303/08/463M6A1V00011246.html |title = 国民党“迁都”广州，司徒雷登落井下石 |accessdate=2012-06-29 |author= |date=2008-03-03 |work=网易新闻 |publisher=}}</ref>。10月14日，解放軍[[陈赓|陈赓]]部击败守卫广州的國军。此后广州为[[中华人民共和国|中华人民共和国]]中央直辖市、[[广东省人民政府|广东省人民政府]]驻地<ref>{{cite book |editor=李禎蓀（主編）、吳佳華、黃繼勝（副主編） |title=《名城廣州小百科》 |year=1992 |publisher=中山大學出版社 |location=廣州 |isbn=7-306-00553-7 |pages=102頁 |chapter = 廣州解放 }}</ref>。
