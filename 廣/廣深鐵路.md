'''廣深鐵路'''，又稱'''廣深線'''（{{lang-en|Guangshen Railway}}或{{lang|en|Guangzhou-Shenzhen Railway}}），[[中國鐵路總公司|中國鐵路總公司]]称'''广九线'''<ref>{{cite book|title=《中国铁路地图集》|author=中国铁道出版社组织编写|location=北京|publisher=中国铁道出版社|isbn=978-7-113-13840-0|date=2011年12月}}</ref>，在1911年至1949年间称为'''廣九鐵路（华段）'''（{{lang|en|Chinese Section of the Canton-Kowloon Railway}}），連接[[中国大陸|中国大陸]][[广东省|广东省]][[广州市|广州市]][[越秀区|越秀区]]的[[广州站|广州站]]及[[广东省|广东省]][[深圳市|深圳市]][[罗湖区|罗湖区]]的[[深圳站|深圳站]]，现由[[中国铁路广州局集团有限公司|中国铁路广州局集团]]的[[子公司|子公司]][[廣深鐵路股份有限公司|廣深鐵路股份有限公司]]營運的鐵路線。

廣深鐵路全長147公里，為I級鐵路幹線。[[廣州站|廣州]]至[[深圳站|深圳]]段均為[[多重複線|四线]]，其中一、二线分别为上行和下行的最高時速200公里级别高速线路（现最高运营时速为180km/h），目前仅供中國鐵路[[和谐号|和谐号]]动车组列車行走。而三、四线分别为上行和下行的最高時速140公里级别普速线路，供普通客運或貨運列車行走（同時包括城際直通車）。為減少貨物列車對日間旅客列車運行的干擾，大多數货物列车均安排在夜间运行。

廣深鐵路在常平站與[[京九铁路|京九铁路]]连接，在广州站与[[京广铁路|京广铁路]]和[[广茂铁路|广茂铁路]]连接，在平湖南站与[[平南铁路|平南铁路]]和[[平盐铁路|平盐铁路]]连接，在南端经[[罗湖桥|罗湖桥]]與[[香港鐵路有限公司|香港鐵路有限公司]]營運的[[東鐵綫|東鐵綫]]相接通到[[香港|香港]]的[[紅磡站|紅磡站]]。另外设有连接广州鱼珠码头、黄埔新港和石龙港的支线。

== 历史 ==
{{Seealso|九廣鐵路}}

=== 晚清时期 ===
==== 背景 ====
[[File:KCRC_Chinese_Bond.JPG|thumb]]

英国于1842年获得[[香港岛|香港岛]]后就宣布[[香港|香港]]为[[自由港|自由港]]，着手逐步将香港建立为东方的自由贸易枢纽。1850年至1851年期间由[[洪秀全|洪秀全]]等人发起的[[太平天国|太平天国反清运动]]，大量华南商人进入香港避乱，导致香港人口急速增长；与此同时，香港也开始成为亚洲的重要货物[[转口港|转口港]]。在这样的环境条件下，粤、港两地的人口、货物运输越趋频繁。

[[清朝|清]][[光緒|光緒]]十六年（1890年）末，時任广东候补知府[[易学灏|易学灏]]奏请[[两广总督|两广总督]][[李瀚章|李瀚章]]，拟以凑集商股方式，修建[[省城|省城]][[广州|广州]]至香港九龙半島的铁路，加强两地的交通联系<ref name=ycwb>{{Cite web |url=http://www.ycwb.com/ycwb//2007-08/26/content_1596472.htm |title=老城纪事：广九风雨 |accessdate=2011-05-10 |publisher=羊城晚报 |date=2007-08-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20090417174852/http://www.ycwb.com/ycwb/2007-08/26/content_1596472.htm |archivedate=2009-04-17 }}</ref>。易学灏建议，铁路从广州的[[黄埔区|黄埔]]出发，经[[东莞|东莞]][[虎门镇|太平镇]]至九龙半岛的[[深水埗|深水埔]]（深水埗）<ref>{{Cite web |url=http://paper.wenweipo.com/2006/04/07/WH0604070003.htm |title=香江史地：尖沙咀鐘樓 |accessdate=2011-05-10 |publisher=文汇报 |date=2006-04-07 }}</ref>。当时的深水埗正正位于边界以北，仍属新安县管辖，因北京条约的签订而成为一个边界小镇和渔村，其地位日益重要；而英国的野心仍然在膨胀，在签订北京条约后不久就擅自将深水埗纳入自己的势力范围，图谋占领整个九龙半岛<ref>{{cite book |author=孙庆、沙辰 |title=中英关系史话 |year=1999 |publisher=社会科学文献出版社 |location=北京 |isbn=7801493427 |page=}}</ref>。一直对铁路持开明态度、主张兴建铁路的李瀚章，对易学灏的建议亦表同意，并于1890年10月写就《在籍知府易學灝請由省河至深水埔試辦鐵路似可准行請核復飭遵由》上报朝廷请求批准<ref>{{Cite web |url=http://catalog.digitalarchives.tw/dacs5/System/Exhibition/Detail.jsp?OID=565098 |title=件名：在籍知府易學灝請由省河至深水埔試辦鐵路似可准行請核復飭遵由 |accessdate=2011-05-27 |publisher=數位典藏與數位學習國家型科技計畫 |date= }}</ref>，惟清廷因当时因正在筹建[[卢汉铁路|卢汉铁路]]和[[东三省铁路|东三省铁路]]（关东铁路），负担甚重，且[[粤汉铁路|粤汉铁路]]尚未形成，“省河至深水埔”的铁路计划未被批准。

[[中日甲午战争|中日甲午战争]]后，为了经济和政治利益的原因，[[列强|列强]]在中国夺取铁路权益的竞争更趋激烈，逼使清政府给予特许权，让他们的公司可以在中国融资、建造和营运铁路。光绪二十四年（1898年）6月，清政府与英国签订《[[展拓香港界址专条|展拓香港界址专条]]》，其中就有“将来中国建造铁路至九龙英国管辖之界，临时商办”的规定<ref name=dashiji>{{cite book |author=中国铁路史编辑研究中心 |title=中国铁路大事记（1876-1995） |year=1996 |publisher=中国铁道出版社 |location=北京 |isbn=7-113-02359-2 |page=}}</ref>。同年8月，英国成功迫使清政府给予承办包括[[广九铁路|广九铁路]]在内等五条铁路的建筑承办权。尽管清廷内部存在异议，但迫于英国的压力，清政府仍命铁路督办大臣[[盛宣怀|盛宣怀]]与代表[[中英银公司|中英银公司]]的英商[[怡和洋行|怡和洋行]]，于1899年3月28日签订了《广九铁路草合同》，并议定此路由广东省的省城广州至英国殖民地香港九龙，将来与粤汉铁路连接。此草合同规定，一切比照《[[沪宁铁路|沪宁铁路]]草合同》办理<ref name=gtjtz>{{cite book |author=广州铁路（集团）公司史志编委会 |title=广铁集团志 |year=2002 |publisher=中国铁道出版社 |location=北京 |isbn=7-113-03637-6 |page=}}</ref>。《草合同》签署后，中英银公司于1899年对铁路走线进行了初步勘测，但后来因英国需要应付南非的[[第二次布尔战争|第二次波耳戰爭]]，无暇顾及广九铁路，同时中英银公司也出现财政困难，而将此事暂时搁置<ref name=gtjtz />。

南非殖民战争结束后，英国多次派特使和中英银公司为代表，催请清廷就修筑广九铁路一事援约议订正式合同。至光绪三十三年（1907年），清廷外务部[[侍郎|侍郎]][[唐绍仪|唐绍仪]]与中英银公司正式在[[北京|北京]]签订《广九铁路借款合同》。合同共20条，订明英国[[贷款|贷款]]150万[[英镑|英镑]]给清政府修建华界之内的铁路（華段），由中英银公司代政府发售[[债券|债券]]支付，贷款折扣94%，年息5%，还款期限30年，以本路财产及未来营业收入为[[担保物权|担保]][[抵押|抵押]]；并且还规定“中国将来不另建一路，以夺本路利益”；同时，为酬谢中英银公司“造路时出力”，给予酬金3.5万英镑<ref name=dashiji />。

==== 建造 ====
[[File:Tai-sha-tau_Railway_Station.jpg|thumb]]
[[File:广九铁路俱乐部旧址.jpg|thumb]]建造外籍工程師俱樂部，現存一棟位於中山一路小東園內，為[[廣州市歷史建築|廣州市歷史建築]]]]
经中、英双方商定，广九铁路以[[罗湖桥|罗湖桥]]中孔第二节为界，分为华、英两段。华段即为现今之广深铁路，由广州东郊的[[大沙头火车站|大沙头火车站]]起，经仙村、石龙，至深圳止，当时全长142.77公里（约89英里）。根據《借款合同》，华段的建造由两广总督委派华人为总办来主持，任用英国人为总工程师、总会计师。而最终时任两广总督袁樹勛于宣统二年（1910年）委派中国军舰制造专家[[魏瀚|魏瀚]]任广九铁路总办，而清廷仍然自聘華人铁路建筑工程师[[詹天佑|詹天佑]]為广九铁路的顾问<ref>{{Cite web |url=http://hk.szhk.com/2007/12/26/282673613307459.html |title=（十一）英促建广九铁路 中迫借百万巨款 |accessdate=2011-05-10 |publisher=南方都市报 |date=2007-12-16 |archive-url=https://web.archive.org/web/20110607210605/http://hk.szhk.com/2007/12/26/282673613307459.html |archive-date=2011-06-07 |dead-url=yes }}</ref>。另一方面，英段全长35.78公里（约22英里），由罗湖经大埔、沙田，终到尖沙咀，由香港政府负责融资、兴建，并且已于1906年7月率先开工、1910年10月1日通车。

而华段于1907年上半年完成设计，同年8月11日分三段（大沙头至[[仙村|仙村]]、仙村至石龙、石龙至深圳）同时动工。广九铁路名义上华、英两段分别由中、英双方各自修建和管理，但实际上华段的勘测、设计、施工，均由英方总揽，华段总工程司、总管帐、运输主任等职，均由中英银公司派人担任。华段分三段施工、分段通车，其中大沙头至仙村一段于1910年11月28日竣工，同年12月6日通车；仙村至石龙一段于1911年3月13日竣工，同年4月24日通车；石龙至深圳一段于1911年9月28日竣工。1911年10月8日，广九铁路华段全线通车<ref name=gtjtz />。不久之后，[[辛亥革命|辛亥革命]]爆发，推翻了清朝帝制。

=== 民国时期 ===
[[File:Opening_of_Chinese_Section_of_the_Canton-Kowloon_Railway.jpg|thumb]]
1911年10月28日，华、英两段在[[罗湖桥|罗湖桥]]接轨，广州至九龙全线通车，并开办来往两地的[[广九直通车|直通列车服务]]。在1949年前，[[中華民國|中華民國]]政府均称此铁路为「廣九鐵路」（{{lang|en|Canton-Kowloon Railway (CKR)}}），而[[香港|香港]]則將它稱為「[[九廣鐵路|九廣鐵路]]」（{{lang|en|Kowloon-Canton Railway (KCR)}}）。鐵路綫在[[中國大陸|中國大陸]]的路段稱為「華段」，而在香港的路段則稱為「英段」。

广九铁路华段当时为[[标准轨距|标准轨距]][[单线铁路|单线铁路]]，由位于廣州大沙頭火车站向东南引出，沿[[珠江|珠江]]东行，在[[石龙镇_(东莞市)|石龙]]前越[[东江|东江]]及其支流，经[[樟木头镇|樟木头]]，到达新安縣（今[[深圳市|深圳市]]），经罗湖桥与英段线路连接。铁路最大[[坡度|坡度]]16.25‰，[[最小曲线半径|最小曲线半径]]约437米；钢轨为[[汉阳铁厂|汉阳铁厂]]造，每米重42.25公斤。华段共修建78座铁路桥，其中规模较大的石厦桥、东莞桥、石龙南桥都是相当复杂的工程，耗费资金、人力巨大。华段全线设车站17座及旅客乘降所13处。大沙头站设机车车辆修理厂及机务段，[[深圳站|深圳站]]设折返段；大沙头站、[[石龙站|石龙站]]及深圳站有转盘及煤台；[[南岗站|南岗站]]设有给水设备<ref name=gtjtz />。

[[File:Canton-Kowloon_Railway_Poster.jpg|thumb]]
[[File:廣九季刊二十五年第四期.jpg|thumb]]
民国元年（1912年），华、英两段订立联轨营业合同，所有广州至九龙直通客货车辆配比，华、英配备数量按线路长度比例为四比一，客货列车在深圳交接机车，车厢全线直通。广九铁路开通初期，每天上午9点各有一趟“慢车”分别从广州和九龙对开，下午4时许到达，全程需7个多小时，被老百姓戏称为“火龙船”；同时也偶有“特别快车”，约3小时就可到达。同年，[[中華民國臨時政府_(南京)|中华民国临时政府]][[中華民國交通部|交通部]]派商办广东粤汉铁路总公司总理詹天佑对华段进行验收，但因台风暴雨冲毁部分路基桥梁，未能完成验收程序。延至民国2年（1913年），另派交通部技正[[罗国瑞|罗国瑞]]验收<ref name=shengzhi>{{Cite web|url=http://www.gd-info.gov.cn/books/dtree/showbook.jsp?paths=415&stype=v&siteid=mastersite&sitename=%E5%B9%BF%E4%B8%9C%E7%9C%81%E6%83%85%E4%BF%A1%E6%81%AF%E5%BA%93|title=广东省情信息库：铁路志|accessdate=2011-05-10|publisher=广东省人民政府地方志办公室|date=|deadurl=yes|archiveurl=https://archive.is/20121203092008/http://www.gd-info.gov.cn/books/dtree/showbook.jsp?paths=415&stype=v&siteid=mastersite&sitename=%E5%B9%BF%E4%B8%9C%E7%9C%81%E6%83%85%E4%BF%A1%E6%81%AF%E5%BA%93|archivedate=2012-12-03}}</ref>。

自1912年以来，广九铁路华段一直由广九铁路管理局进行管理。至民國17年（1928年）12月3日，新任[[中华民国铁道部|中华民国铁道部]]部长[[孙科|孙科]]向国民政府提出报告，要求管理统一，各路车辆互调受铁道部直接掌管，不准任何方干涉。1929年1月24日，[[国民政府行政院|国民政府行政院]]同意该报告，准予实行。同月，铁道部将广东省政府管辖的广九、[[广三铁路|广三]]、[[粤汉铁路|粤汉]]三路局划归铁道部直接管辖<ref name=dashiji />。1930年，铁道部暂定各铁路管理局等级，广九铁路管理局定为三等局，与[[正太铁路|正太]]、[[南浔铁路|南浔]]铁路管理局相同。1934年，中、英双方开始就联运合同、收入分配问题进行谈判，华段和英段的长度对比是四比一，但按照旧联运合同的收入分配原则，英段竟占65%，而华段仅占35%。为此，双方在1934年9月15日在香港签订新的联运合同，收入分配修改为华段占72%、英段占28%，并规定客、货车辆分摊辆数<ref name=dashiji />。

民国25年（1936年）4月，粤汉铁路全线通车后，英国为进一步谋求将广九与粤汉两线接轨，遭到广州市商界人士的坚决反对。至民国26年（1937年），国民政府动工建设广州[[黄埔港|黄埔港]]，并于1937年1月开工建设一条由粤汉铁路西联站（今[[棠溪站|棠溪站]]），经[[沙河街道_(广州市)|沙河]]、[[石牌街道|石牌]]、[[魚珠街道|鱼珠]]至黄埔港的「黄埔港支线」；1937年7月7日，发生了[[卢沟桥事变|卢沟桥事变]]，[[中国抗日战争|中国抗日战争]]全面爆发。同年8月，国民政府为获得英美两国的援助，方便援华物资送入内地，答应连通广九、粤汉两路。当时黄埔港支线尚未铺轨，故利用已建成的黄埔港支线路基修建联络线。1937年8月，黄埔港支线铺轨至石牌与广九铁路接轨，设[[石牌站_(广州)|石牌站]]，称为「广北联络线」，全长14公里，自此粤汉铁路和广九铁路接轨。

==== 抗战时期 ====
[[File:Tai-sha-tau_Railway_Station_in_1938.jpg|thumb]]石像下拍照]]
[[File:Shilong_Railway_Bridge_1938.jpg|thumb]]
战争很快从华北蔓延到华南地区。1937年10月，日军步步进逼，开始[[空袭|空袭]]粤汉、广九等铁路，其中跨东江的石龙南、北铁路大桥更是重点目标之一，大桥钢梁多处中弹穿孔。1938年初，日军已经逐渐取得岭南地区的[[制空权|制空权]]，空袭更见频繁，为此中国军队在广九铁路沿线以及重要的桥梁附近均布置有一定数量的[[高射炮|地面防空火力]]，并多次击落来犯敌机<ref>{{Cite web|url=http://www.zc720.com/Article/ShowInfo.asp?ID=162|title=侵华日机迫降增城|accessdate=2011-05-20|publisher=增城全景游|date=}}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name=dongguankangri>{{Cite web |url=http://dgds.sun0769.com/historystuff/shownews.asp?id=159 |title=东莞抗日大事记 |accessdate=2011-05-20 |publisher=中共东莞市委党史研究室 |date= }}</ref>。1938年9月至10月17日，日军为切断这条交通运输线，多次轰炸石龙站和石龙南、北铁路大桥，造成石龙北桥断裂，南桥受损，不过很快都被修复。

1938年10月21日，广州沦陷，切断了广九和粤汉两路的聯繫。从1937年8月起到1938年10月，通过广九铁路和粤汉铁路联运，大批军事物资包括飞机及其零件、枪炮、弹药等，从香港源源不断输入到内地，对抗战作出了重要支援<ref>{{Cite web |url=http://www.ce.cn/new_hgjj/hongguanzl/200508/17/t20050817_4449978.shtml |title=香港:抗战物资中转站 |accessdate=2011-05-20 |publisher=中国经济网 |date=2005-08-15 }}</ref>。1938年10月底，为阻挡日军南侵东莞，中国驻军奉命炸毁石龙南铁路大桥两个桥墩，广九铁路遂告中断<ref>{{Cite web |url=http://dgds.sun0769.com/shownews.asp?id=943 |title=日本对东莞的经济侵略 |accessdate=2011-05-20 |publisher=中共东莞市委党史研究室 |date= }}</ref>。1938年11月，广九铁路华段全线被日军占领<ref>{{cite book |author=王致中 |title=中國鐵路外債研究（1887-1911） |year=2003 |publisher=經濟科學出版社 |location=北京 |isbn=7505837591 |page=275}}</ref>。

广州沦陷时，[[广州天河机场|天河机场]]（现址为[[天河体育中心|天河体育中心]]）也被占领，成为日军的空军基地。为了满足战争需要，日军于民国29年（1940年）开始大规模扩建天河机场<ref>{{Cite web|url=http://www.hpma.cn/article-img.aspx?menuid=1327&tab=tab_imgNews&tabid=93|title=广州龙口东路侵华日军飞机库考证|accessdate=2010-12-30|publisher=广东革命历史博物馆|date=}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。由于机场占地面积扩大，工程同时拆迁了广九铁路东山至石牌间3公里线路，由东山改绕永村、[[沙河街道_(广州市)|沙河]]至石牌，在与广北联络线接轨处设[[广州东站|天河站]]（今广州东站）。至此，广九铁路较原有旧线延长了3.235公里，广九铁路华段总长改为146公里。广北联络线也相应缩短为西联站至天河站的联络线，长为9.6公里<ref name=youdian>{{cite book |author=廣州交通郵電志編纂委員會 |title=《廣州交通郵電志》 |year=1993 |publisher= 廣東人民出版社|location=广州 |isbn=7-218-01141-1 |page=}}</ref>。同年，日军为打通海陆联运的交通线，着手修筑黄埔港支线，但并不采用国民政府于1937年设计的石牌站出岔方案，而是自[[吉山站|吉山站]]出岔另修黄埔港支线，接通黄埔港<ref name=gtjtz />。

抗日战争时期，广九铁路沿线也是重要的[[游击战|游击战场]]。国民党军队以及由[[中国共产党|中国共产党]]领导的[[东江纵队|东江纵队]]多次在广九铁路沿线伏击日军部队，并获得了不少战果。日军控制广九铁路后，力图连接粤汉、[[平汉铁路|平汉]]、[[北宁铁路|北宁铁路]]，构成“大陆交通线”，而游击战的目标就是破坏交通线，牵制日军行动。1940年10月下旬，国民党广东省保安第八团在广九铁路湖洞径路段伏击日軍、汪精衛政府军，俘敌30余人<ref name=dongguankangri />。1943年7月底，东江纵队第三大队袭击驻[[茶山镇_(东莞市)|茶山]]的日、汪军，全歼伪军一个中队。1944年1月，东红纵队第三大队及宝安大队袭击了广九铁路[[常平站|常平站]]，歼灭汪军一个连。1943年11月，为支援在太平洋战场上的战斗，日军发动打通广九铁路的战役，向铁路沿线大局进攻，实行“扫荡”、“清乡”，占领了广九铁路中段的常平、樟木头、塘厦、天堂围、平湖等地，东江纵队奔袭茶山、横沥、常平、樟木头等车站，破坏广九铁路交通和通讯设施<ref name=dongguankangri />。

1944年1月23日，美军飞机袭击广九铁路，炸断了石龙大桥。同年年初开始，日军为了尽快修复广九铁路，在[[清溪镇_(东莞市)|清溪]]、[[樟木头镇|樟木头]]一带砍伐大量的[[松树|松树]]用作[[枕木|枕木]]，与此同时，东江纵队也于1944年2月25日开始向广九铁路全线出击。日军在侵占砍树、修路的三个多月中，遭东江纵队连续打击，伤亡惨重，松树也无法砍伐，被迫于1944年6月7日撤回樟木头站<ref name=dongguankangri />。1944年7月，美国《[[美亚杂志|美亚杂志]]》在《东江纵队与盟国在太平洋的战略》一文中说：广九铁路在抗日游击队袭扰下，“日军虽已控制了铁路两头两年多，但他们还不能使一列火车畅行全线”<ref>{{Cite web |url=http://www.tieliu.com.cn/zhishi/2007/200709/2007-09-11/20070911132428_81069.html |title=抗战时，广九铁路活跃一支“飞鹰队” |accessdate=2011-05-20 |publisher=人民铁道报 |date=2007-09-11 |deadurl=yes |archiveurl=https://archive.is/20130424145503/http://www.tieliu.com.cn/zhishi/2007/200709/2007-09-11/20070911132428_81069.html |archivedate=2013-04-24 }}</ref>。1944年8月29日起，日军700人、汪军2300人对广九铁路沿线进行又一次“扫荡”。到9月3日，东江纵队与日（汪）军进行战斗10次，歼敌240余人。至1945年，为配合[[八路军|八路军]]、[[新四军|新四军]]春季攻势作战，东江纵队在广九铁路东西两侧主动进攻，1月份作战10余次，毙伤俘日、汪军200余人，并使日军两列火车相撞，死9人<ref name=dongguankangri />。

==== 戰後復員時期 ====
抗日战争期间，广九铁路作为重要的战略目标，饱受轰炸和损害，日军占领时期，日军对因战争破坏的桥梁和线路进行了临时性修复，勉强继续维持断续区段行车，但日军对铁路采取掠夺性使用，设备严重失修。战争结束后，修复铁路成为当务之急。1945年11月，国民政府交通部接受[[联合国善后救济总署|联合国善后救济总署在华分署]]从各国运输至中国的大批救济物资，其中包括大量铁路机车、车辆、钢轨、桥梁钢料、枕木、机械等，主要用于修复粤汉铁路、[[浙赣铁路|浙赣铁路]]两大干线，以及广九铁路、[[南浔铁路|南浔铁路]]两线。1946年1月，交通部开始组织实施的粤汉铁路复路工程，决定先修复广九铁路，以便从香港北运联合国善后救济总署提供的钢轨、枕木、桥梁、道岔等材料<ref name=dashiji />。

1946年3月1日起，国民政府交通部宣布全国铁路实行干线区域管理体制，广九铁路华段划归粤汉区铁路管理局<ref name=dashiji />。为理顺粤汉、广九两线的客货运输，国民政府于1947年建「永云联络线」，连接广九铁路永村线路所及广北联络线[[雲麓站|云麓线路所]]，使来自粤汉铁路方向客车能够直达[[大沙头火车站|大沙头火车站]]。自此大沙头火车站成为两线客运的终点站，并同时更名为广州东站，与[[广州南站_(货运)|广州南站]]（黄沙火车站）相对应<ref name=shengzhi />。

==== 意外事故 ====
[[File:Accident_on_Canton-Kowloon_Railway_1948.jpg|thumb]]
[[File:KCR_tragedy_1949.gif|thumb]]对此进行了报道]]
自开通运营以来，广九铁路一直是连接广州和香港的重要交通线，但当时社会秩序相对混乱，营运管理也不完善，故歹徒抢劫、列车出轨等各类意外事故时有发生。1923年2月22日，一趟慢车从大沙头开出，运行至增城沙村时，遭土匪拦路抢劫，百多名乘客被掳。同年7月5日，广九铁路列车又在南岗北红冈炮台出坳时被土匪抢劫，掳去数十名乘客，还将车上3名警察打死<ref name=ycwb />。

1937年1月16日深夜，一列广九铁路旅客列车运行至石沥滘站时发生火灾，导致旅客死亡200余人。两天之后的国民党《[[中央日报|中央日报]]》以《广九路惨案真相》为题，报道了这起惨绝人寰的火灾事故，而事故原因则是旅客携带的[[松香|松香]]玩具所致。当时[[路透社|路透社]]详细报道了事故的调查，“据初步调查，此次火灾，乃由乘客一人，偶尔不慎，将火触及松香质之玩具，顿时延及各种行李车。乘客见火，惊乱失措，纷纷逃往邻车，致因以挤倒而踏死者多人，火车驶时，风力本大，火乃借势，由窗中射出，而殃及邻车，直至延烧第三车”<ref>{{Cite web |url=http://dz.xdkb.net/html/2011-01/17/content_66157.htm |title=一只玩具引发的特大铁路惨案 |accessdate=2011-05-27 |publisher=现代快报 |date=2011-01-17 }}</ref>。事后一些死难者亲属更控告铁路当局业务过失杀人及将大部分尸体秘密掩埋灭迹<ref>{{Cite web |url=http://www.gzzxws.gov.cn/gzws/gzws/ml/33/200809/t20080912_7308.htm |title=广州百年大事记第三十三辑：一九三七年 |accessdate=2011-05-27 |publisher=广州文史 |date= }}</ref>。

1948年9月3日，当日天气十分恶劣，一趟开往香港的7次特别快车于下午1時55分由广州东站开出，下午5時30分当运行至[[李朗站|李朗站]]、[[布吉站|布吉站]]之间，在近布吉东面松原坡之斜坡上列车突然脱轨，颠覆客车7辆，包括头等车两辆、二等车两辆、三等车两辆及餐车一辆，机车沿山坡跌入水田，200多条枕木及道轨全部损坏。由于列车满载乘客，事故造成严重死伤，死者70余，伤者约百人。由于布吉接近深圳和香港，故伤员都被连夜送往[[九龙医院|九龙医院]]救治。事后成立“广九路覆车调查团”，调查结果认为，事故发生路段铁轨多处弯曲，轨道夾板、螺栓被拆，同时由于当日列车晚点，故超速运行，导致事故的发生<ref name=ycwb /><ref>{{cite book |author= |title=香港年鑑第二回 |year=1949 |publisher=香港華僑日報 |location=香港 |isbn= |page=}}</ref>。

1949年9月29日，广九铁路87次客货混合慢车在[[林村站|林村站]]附近脱轨颠覆，导致数十名乘客死伤。

=== 中华人民共和国时期 ===
1949年10月1日，[[中华人民共和国|中华人民共和国]]成立，但[[国共内战|国共内战]]的余波仍在持续。1949年10月14日，[[中国人民解放军|解放军]]接管广州市，广九直通列车中断运行。三天后，解放军先后占领塘下、天堂围、深圳等地，广州至石龙区间恢复通车。10月18日，中国人民解放军广州军事管制委员会铁道处成立。10月23日，广州军管会交通接管委员会铁道处，宣布接管广东省境内铁路。10月26日，广九铁路广州至深圳段恢复通车，但列车不能直接开往香港。同月，广九铁路华段改称广深铁路。1949年12月7日，[[衡阳铁路管理局|衡阳铁路管理局]]广州铁路分局成立，广深铁路划归广州铁路分局管辖<ref name=shengzhi />。

1950年，[[深圳站|深圳墟火车站]]遷到較接近[[羅湖口岸|羅湖口岸]]、[[罗湖桥|罗湖桥]]的現址，与香港的罗湖站仅一桥之隔，并在1953年更名深圳站。1951年，广州铁路分局对广州地区铁路及站场进行调整，[[大沙头火车站|广州东站]]更名为广州站，成为广州市内最主要的铁路客运站。1953年1月，铁道部决定对全国铁路运输体制机构进行调整，衡阳铁路管理局改组为[[广州铁路局|广州铁路管理局]]和[[柳州铁路局|柳州铁路管理局]]，原广州铁路分局改组为运输分局，广深铁路归广州运输分局管辖。

1959年7月下旬，广深铁路沿线[[暴雨|暴雨]]成灾，路基多处冲成缺口，中断行车15天。1966年6月下旬，广东东江发生严重[[水灾|水灾]]，广深铁路仙村至石龙段被洪水淹没，线路破坏较严重。为了彻底解决长期困扰广深铁路的水患问题，铁路部门于1966年至1967年对线路进行了大幅改造，包括抬高路基、加宽路基、修建排水设施等，筑成全面防护体系。1967年至1969年，广深铁路全线进行大修，这次大修的目的是提高线路运输能力，包括更换[[钢轨|钢轨]]、铺设[[钢筋混凝土|钢筋混凝土]][[轨枕|轨枕]]，并铺设无缝路轨多段，共102.2公里。广深铁路是中国最早试验无缝路轨的铁路之一，早在1957年开始在吉山至下元区间焊接短距离无缝路轨，1960年建成使用，1964年试验完毕后拆除路轨进行整修<ref name=youdian /><ref name=shenzhenjtz>{{cite book |author=閻志禎、李漢君 |title=深圳市交通運輸志 |year=2001 |publisher=方志出版社 |location=北京 |isbn=7-80122-699-2 |page=}}</ref>。

1974年4月10日，在原流花桥站改建的[[广州站|广州新客站]]建成使用，广深铁路起点由原广州站（大沙头站）改成广州新客站，广深铁路全长调整为147.261公里<ref name=gtjtz />。自此，广州站成为京广、广三、广深三条铁路的交汇点。而广深铁路改线之后，位于大沙头的原广州站恢复广州东站旧称，其所有客运业务逐步移至广州新客站，大沙头火车站暂时作为简易[[客技站|客车技术整备作业站]]保留。至1980年代，伴随广州的城市发展，昔日处于城市边缘的大沙头火车站已经被城市包围。1984年11月，广州东客技站至天河一段原广九铁路被拆除，云永联络线随之撤销，如今成为了广州市内环路的一部分；而大沙头火车站也于1985年初被清拆<ref name=gztlsn>{{Cite web|url=http://210.76.65.23/books/dtree/showbook.jsp?paths=415&stype=v&siteid=mastersite&sitename=%E5%B9%BF%E4%B8%9C%E7%9C%81%E6%83%85%E4%BF%A1%E6%81%AF%E5%BA%93|title=第七节 广州铁路枢纽及省内主要车站|accessdate=2010-12-30|publisher=广东省省情信息库|date=}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1976年10月16日，由深圳開往廣州的92次特快列車在广深线41公里处脱轨，18名旅客受伤，中断行车近25小时。<ref>{{Cite web |url=http://www.raildoor.com/news/anquan/200804/20080428231609_2880.html |title=中国从70年代至今重大铁路事故一览 |accessdate=2011-06-02 |publisher= |date=2008-04-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20080612211920/http://www.raildoor.com/news/anquan/200804/20080428231609_2880.html |archivedate=2008-06-12 }}</ref>

==== 复线建设 ====
1980年代起，由于中国进行[[改革开放|改革开放]]，[[深圳|深圳]]成立了[[经济特区|经济特区]]，[[珠江三角洲|珠三角地区]]经济迅速发展，进口货物和旅客大量增加，原有广深铁路单线难以满足运输需要。为此，与广东省政府商议后，广州铁路局于1981年6月10日，向铁道部报送《广深铁路改造设计任务书》，请求增建广深铁路复线。1983年6月25日，[[国务院|国务院]]批复了有关计划。惟当时国内尚有不少等待改造、新建的铁路项目，国家铁路建设资金紧张，故广州铁路局与国务院协商后，决定选择自筹资金、以路养路、以路建路的改革方式<ref name=liaowang>{{cite journal | author =戚休、王志纲  | title =“包”字在广深铁路开花结果  | journal =《瞭望周刊》  | publisher =新华通讯社  | location =北京  | issn =  |date=1987年4月}}</ref>。

1983年12月15日，被誉为中国铁路改革“第一号种子”的[[广深铁路公司|广深铁路公司]]在深圳成立，1984年1月1日正式开始营运，直接隶属于[[广州铁路局|广州铁路局]]<ref>{{Cite web |url=http://lw.xinhuanet.com/htm/content_3960.htm |title=广深“公交铁路”体验 |accessdate=2011-05-27 |publisher=新华社瞭望周刊社 |date=2008 }}</ref>；国家允许公司从高度集中的国家铁路管理体制中脱离出来，实行“自主经营，自负盈亏，自我改造，自我发展”的[[责任承包制|经济承包责任制]]，其实行的新型管理模式被称为“广深模式”。国家不给分文，公司自行承担广州至深圳间铁路客货运输和对外服务业务，实行客货运输价格上浮50%和上缴利润递增包干的特殊政策，并靠自身创造的利润来自行分阶段完成广深铁路复线电气化改造建设<ref name=liaowang />。

广深铁路公司有独立的设计施工审核权、客货运输经营权、行车调度指挥权、对外业务洽谈权和外汇留成使用权。公司领导体制的改革，加快了铁路工程进展速度，缩短、简化了铁路工程设计审批程序。广深铁路复线工程以分项招标方式进行施工单位招标，并于1984年2月27日开展施工。为了配合深圳市路网工程，自深圳北站至深圳站区间一段复线改建为高架铁路；而石龙南、北大桥一段，因大桥工程艰巨，且经核算该段运输能力尚能满足近期要求，故暂不进行复线改造。整个复线改造项目共投资7亿元人民币，于1987年1月26日通车，成为当时广东省第一条[[雙線鐵路|复线铁路]]。工程完成后，广深铁路上行线全长142公里，下行线全长147.1公里，最小曲线半径由原来的450米改善至800—1000米，列车最高运行速度为110公里/小时。复线建设对广深铁路的运输能力产生显著的提高，综合运输能力提高70%。在复线开通的1987年，广深铁路旅客发送量达到了1467.89万人次，是1983年的152%；年货物发送量为754.2万吨，年货物到达量1064.1万吨，分别是复线前的176%和165%<ref name=shenzhenjtz />。

在建设广深铁路复线的同时，广州东新客站的建设规划随之提上日程。广州市承办了1987年[[中华人民共和国第六届运动会|第六届全国运动会]]（六运会），并建设[[天河体育中心|天河体育中心]]作为六运会会场。由于从广州站到天河体育中心的路程较长，于是开始考虑在天河体育中心附近建设一个铁路客运站，作为六运会的配套设施，故决定将广深铁路[[广州东站|天河站]]作为广州第二大铁路客运站，专办广深、广九客车始发终到。改造后的天河站于1987年在六运会开幕之前投入使用。为提高车站的知名度，天河站于1988年4月1日更名为广州东站<ref name=gtjtz />。

==== 提速改造 ====
广深铁路是中国最早实现在既有线上提速的铁路试点。由20世纪90年代起，多次对线路进行提速改造，广深铁路的提速经验后来被陆续推广到三大干线（[[京广铁路|京广铁路]]、[[京沪铁路|京沪铁路]]、[[陇海铁路|陇海铁路]]），为后来的[[中国铁路大提速|中国铁路大提速]]打下了基础。

===== 準高速改造 =====
[[File:广深准高速列车水牌.JPG|thumb]]
1980年代末，中国开始将眼光投向[[高速铁路|高速铁路]]，建设[[京沪高速铁路|京沪高速铁路]]的构思也在酝酿之中。但当时中国高速铁路开发工作已经滞后世界水平几十年，技术差距较大，因此铁道部决定从国情实际出发，以时速160公里的准高速铁路作为突破口，为未来发展高速铁路进行探索和试验。选定试验区段对既有线路进行技术改造，可以用较少的投资、较短的时间将既有线路改造成能够开行时速160公里旅客列车的准高速铁路。而当时的广深铁路具有其它铁路线所没有的优越条件，包括不断增长的运输需求、地形条件较好、长度适中等，使其成为首要的试验目标<ref>{{cite journal | author = 龚深弟 | title = 铁道部通过广深线旅客列车最高时速160km的实施方案 | journal = 《中国铁路》| pages =  | publisher = 铁道部科学技术信息研究所 | location = 北京 | issn = 1001-683X |date=1990年9月}}</ref>。

[[中华人民共和国铁道部|铁道部]]于1989年，成立了由[[中国铁道科学研究院|中国铁道科学研究院]]和广州铁路局组成的联合专家组，对广深线旅客列车的最高速度提高到160km/h进行了前期可行性研究。1990年，铁道部发布《铁计【1990】1号文》，正式将“广深铁路实现旅客列车最高速度160km/h的技术方案研究”列入1990年铁道部科学技术发展项目。同年，铁道部以《铁科技函【1990】474号文》下达了《广深线准高速铁路科研攻关及试验计划的通知》，至此，广深铁路准高速机车车辆、线路工程、信号系统、速度分级控制及安全评估试验等15重点技术攻关研究计划开始全面执行，其中[[東風11型柴油機車|东风11型准高速内燃机车]]、[[韶山8型电力机车|韶山8型准高速电力机车]]、[[中国铁路25Z型客车|25Z型准高速双层客车]]、[[中国铁路25Z型客车|25Z型准高速客车]]、准高速旅客列车速度分级控制、旅客列车移动电话系统，准高速铁路接触网及受流技术等8项专题列入“八五”国家科技攻关计划<ref name=tdzs1995>{{cite journal | author = 葛闻安 | title = 广深准高速铁路——中国铁路高速化的起点 | journal = 《铁道知识》 |  pages = 8-9 | publisher = 中国铁道学会 | location =北京 | issn = 1000-0372|date=1995年3月}}</ref>。

1990年9月，铁道部向国家计委报送了《关于广深铁路技术改造项目建议书的报告》；同年12月，国家计委批准立项。1991年4月，铁道部和广东省政府联合成立了广深准高速铁路建设领导小组。广深准高速铁路技术改造工程于1991年12月28日在石龙特大桥正式动工，由广州铁路局成立的广深准高速铁路建设指挥部负责建设，总投资48亿元人民币。工程主要有线路平面改造、改造既有小曲线半径、换铺60公斤/米的的重型无缝路轨共197公里、[[正线|正线]][[道岔|道岔]]换成12号{{link-en|可动心道岔|Swingnose crossing}}、线路两侧设隔离封闭栅栏、所有[[平交道|平交道口]]改为[[交流道|立体交叉]]、安装带超速防护功能的信号系统等，以及其他复线建设时未完成的后续工程，包括石龙段双线、[[广州东站|广州东站]]改扩建等。对原有双线进行提速改造期问，为了使广深铁路适应[[广梅汕铁路|广梅汕铁路]]、[[京九铁路|京九铁路]]开通以后运量增长的需要，及减少工程对日常运营的影响，广深铁路总公司决定利用原有的复线和准高速改造时废弃的路基和施工便道，并新建了部分线路，在广州东站至[[平湖站|平湖站]]区段建成了第三线，长65.5公里<ref>{{cite journal | author = 王慎德、李福元 | title = 广深准高速既有线改造得失观 | journal = 《中国铁路》| pages =  | publisher = 铁道部科学技术信息研究所 | location = 北京 | issn = 1001-683X |date=1993年10月}}</ref>。

广深准高速铁路设计速度160公里/小时，下元至平湖的准高速区段长98.5公里，其中新塘至石龙之间设有时速200公里的高速试验段。铁路采用从[[法国|法国]]引进的U—T多信息[[闭塞_(铁路)|自动闭塞]]及分级速度监控设备，采用四显示自动闭塞，使[[铁路信号机|通过信号机]]具有四种显示，能预告列车前方多个闭塞分区占用状态；线路安装UM-71型[[轨道电路|轨道电路]]，机车安装TVM-300机车信号系统，列车追踪最小间隔时间为8分钟。1994年9月21日开始夜间综合试验；同年12月8日，完成了历时79天的提速试验，试验中准高速列车最高时速达到174公里<ref>{{cite book |author=广州铁路(集团)公司 |title=广深准高速铁路安全运营一周年文集 |year=1996 |publisher=中国铁道出版社 |location=北京 |isbn=7-113-02396-7 }}</ref>。广深准高速铁路从1994年12月22日正式开通，成为中国第一条准高速铁路。

1996年，为配合京九铁路的兴建，广深铁路新建了平湖至深圳区间20公里的第三线、[[平湖南站|平湖南站]]、深圳北客技站等配套，广深线形成了两条时速160公里（一、二线）、一条时速120公里（三线）的三线铁路<ref>{{Cite web |url=http://210.76.65.23/books/415/2907.html |title=1988年至1994年广东省境内新建铁路简介 |accessdate=2011-05-27 |publisher=广东省情信息库 |date= }}</ref>。

与此同时，中国铁路信号系统的革新也首先在广深铁路开始。1990年代初，中国引进了[[阿尔斯通公司|阿尔斯通公司]]的[[计算机联锁|计算机联锁系统]]。该系统于1991年11月19日率先在广深铁路[[红海站|红海站]]开通使用，成为中国铁路干线上第一个计算机联锁车站。计算机联锁对车站值班员的操作命令及现场情况进行逻辑运算，从而实现对信号机及道岔等进行集中控制<ref>{{cite journal | author =周文龙 | title = 我国铁路干线上第一个微机联锁 | journal = 铁道知识 | pages = 12-13 | publisher = 《铁道知识》杂志社 | location = 北京 | issn = 1000-0372 |date=1992年4月}}</ref>。

===== 电气化改造 =====
[[File:Xinshisu.jpg|thumb]]」[[X2000列車|X2000擺式高速动车组]]]]
广深准高速铁路原设计为[[電氣化鐵路|電氣化鐵路]]，但由于1990年代初进行准高速改造时资金不足，故推迟了电气化的实施，在过渡阶段使用内燃机车牵引。至1996年，广深铁路总公司改制为广深铁路股份有限公司，并分别于[[香港交易所|香港交易所]]、[[纽约证券交易所|纽约证券交易所]]挂牌上市，广深铁路运用发行[[H股|H股]]募集资金及自有资金持续投入建设准高速铁路、高速电气化及配套工程<ref>{{Cite web|url=http://www.gsrc.com/big5/article.php?article_id=102 |title=发展历程 |accessdate=2011-05-27 |publisher=广深铁路股份有限公司 |date= |deadurl=yes |archiveurl=https://web.archive.org/web/20110822105116/http://www.gsrc.com/big5/article.php?article_id=102 |archivedate=2011-08-22 }}</ref>。1996年6月，召开了广深高速电气化铁路实施方案论证会，确定广深线电气化建设以“先客运、后货运，先高速、后普速”为原则。1997年2月，总投资8亿元人民币的广深线高速电气化工程全面开工<ref name=shenzhenjtz />。本工程與香港[[九廣東鐵|九廣東鐵]]（[[港鐵|港鐵]][[東鐵綫|東鐵綫]]）採用一致電化標準（25000V[[交流電|交流電]]，50Hz；此等標準最初來自[[台鐵|台鐵]][[縱貫鐵路|縱貫鐵路]]電化工程），以便列車在廣州至香港間暢通直達。

工程的重点在于延长原有的时速200公里试验段，进一步提高广深铁路一、二线的列车运行速度。石牌至平湖段108.5公里为满足时速200公里运行条件的高速区段，其中下元至茶山段27.14公里设有时速250公里的试验段；在时速200公里或以上的高速区段之内，首次采用截面面积为127平方毫米的铜绞线承力索和截面面积120平方毫米的银铜合金接触线的简单链形悬挂[[接触网|接触网]]，并试验在三线并行区段使用硬横跨悬挂接触网。此外，广深电气化铁路还采用了德国[[西门子|西门子]]供电监控设备和英国的[[中性區|自动过分相]]装置<ref name=tdzsdqh>{{cite journal | author =冯金柱 | title = 广深准高速铁路电气化新技术 | journal = 《铁道知识》 | pages = 11 | publisher = 铁道知识杂志社 | location = 北京 | issn = 1000-0372 |date=2000年5月}}</ref>。

1998年5月28日，广深线高速电气化工程竣工，同年8月28日正式投入运营。广深铁路率先使用由[[瑞典|瑞典]]租賃、最高營運速度达200km/h、被命名为「[[新時速|新時速]]」的[[X2000列車|X2000擺式高速动车组]]运行广深城际列车<ref name=shenzhenjtz />。由於广深铁路全線採用了眾多达到九十年代國際先進水準的技術和設備，因此当时广深铁路被视為[[中國高速鐵路|中國高速鐵路]]的試驗基地和展示中國鐵路發展的“窗口”。广深准高速铁路有80多项科技成果被授予1999年度铁道部科学技术进步奖，其中时速200公里电气化新技术获铁道部科技进步一等奖<ref name=tdzsdqh />。

1999年6月8日，广深铁路三线（普速线）电气化工程动工，并于2000年9月16日竣工，廣深鐵路成為中國第一條三線並行、全線封閉、全线電氣化的鐵路干线<ref>{{Cite web |url=http://paper.wenweipo.com/2000/09/06/CH0009060013.htm |title=廣深第三線電氣化竣工 |accessdate=2011-05-27 |publisher=文汇报 |date=2000-09-06 }}</ref>。

===== 四线建设 =====
[[File:Guangshen_Railway_(Shenzhen_section).jpg|thumb]]
至2000年代初，由于客货运量的快速增长，尽管广深铁路已经建成了三条电气化正线，但运输能力仍然日渐紧张，部分区段的通过能力以及达到或超过饱和。广深铁路三线原则上的分工为两部分：一、二线为准高速双线，供时速200公里高速列车、时速160公里准高速列车运行；三线为时速120公里的普速单线，供普通旅客列车、货物列车运行。但实际上由于第三线为单线铁路，造成部分普速旅客列车、甚至货物列车均在准高速双线上混跑，穿插于准高速双线与第三线运行。各种高速、准高速、普速客车以及货物列车混跑，除影响运输能力、调度指挥不便外，更可能会造成安全隐患和维护困难<ref>{{cite journal | author = 刘英贤  | title = 广深铁路增建第四线扩能分流与对策 | journal = 《中国铁道学会粤京港沪第五届学术年会》 | publisher =  | location =  | issn =  |date=2001年4月}}</ref>。因此，广深铁路股份有限公司于2001年开始考虑新建一条与第三线平行的四线，以实施高速与普速列车分线运行的目标。而在过渡期期间，在普速线沿线新建了[[塘美站|塘美站]]、[[南社站|南社站]]等几个[[会让站|会让站]]，以提高运输能力<ref>{{Cite web |url=http://news.sina.com.cn/s/2004-05-25/19582625239s.shtml |title=广深线上的小站 |accessdate=2011-06-02 |publisher=深圳晚报 |date=2004-05-25 }}</ref>。

2003年11月，广东省政府和铁道部签署了《关于加快广东省铁路建设等问题的会谈纪要》，其中就包括了广深铁路广州东至深圳之间增建第四线的规划等九大项目。2004年10月1日，铁道部、广东省政府关于加快广东铁路建设有关问题的会谈纪要签字仪式在广州举行；同月，铁道部正式批准立项建设广深铁路第四线<ref>{{Cite web |url=http://www.gd.gov.cn/govpub/pic/0200701160030.htm |title=广东铁路建设规模空前 1390亿推进九大铁路项目 |accessdate=2011-06-02 |publisher=羊城晚报 |date=2007-01-16  }}</ref>。

2004年12月31日，广深铁路第四线局部开工建设。2005年8月26日，铁道部和广东省政府在石龙镇宣布广深四线建设全面开工，工程总投资52.6亿元人民币，主要由[[中铁二十五局集团|中铁二十五局集团]]承建。广深四线全长147公里，设计速度140公里/小时，需穿越35座桥梁，其中特大桥有3座（深圳铁路特大桥、下李朗三线铁路大桥、吉山特大桥），大中桥31座，并新建、改建站段21个<ref>{{Cite web |url=http://news.china.com/zh_cn/domestic/945/20050827/12605822.html |title=我国首条四线铁路广深四线开工 总投资达47亿元  |accessdate=2011-06-02 |publisher= 信息时报 |date=2005-08-27 }}</ref>。除了新建第四线外，工程同时对第三线进行了提速改造，设计速度由原来的120公里/小时提高到140公里/小时。

2007年4月18日，新建广深四线工程完工，在[[中国铁路第六次大提速|中国铁路第六次大提速]]当日投入运营，自此广深铁路成为中国第一条全程封闭、四线并行的铁路，实现高速旅客列车与普速客车、货物列车的分线运行，大大提升了综合运输能力，而广深城际列车的运行密度也因此得以大幅提高。广深四线开通之后，广州站至广州东站间[[运行图|平行运行图]]区间最大通过能力由每天188对增加到240对；高速线（一、二线）广州东站至深圳站间平行图最大能力由每天163对增加到190对；普速线（三、四线）广州东站至平湖站、及平湖站至深圳站的平行图最大能力由每天55对、60对，分别增加到188对、130对<ref>{{Cite web |url=http://www.china.com.cn/chinese/2005/Jan/744728.htm |title= 我国首条四线铁路动工 广深线将实现客货分流  |accessdate=2011-06-02 |publisher=深圳商报 |date=2005-01-01 }}</ref><ref>{{Cite web |url=http://www.ycwb.com/ycwb//2007-01/11/content_1346582.htm |title=广深铁路旅程缩短十五分钟 |accessdate=2011-06-02 |publisher=羊城晚报 |date=2007-01-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20140106040605/http://www.ycwb.com/ycwb//2007-01/11/content_1346582.htm |archivedate=2014-01-06 }}</ref>。

在广深四线建成通车的同时，于1911年3月建成、具有相当历史意义的石龙南桥停止使用<ref>{{Cite web |url=http://www.gdwht.gov.cn/shownews.php?BAS_ID=21186 |title=苏桂芬一行赴东莞石龙镇视察铁路桥保护情况 |accessdate=2011-06-02 |publisher=东莞文广新局 |date=2008-10-27 |deadurl=yes |archiveurl=https://archive.is/20120801054623/http://www.gdwht.gov.cn/shownews.php?BAS_ID=21186 |archivedate=2012-08-01 }}</ref><ref>{{Cite web |url=http://gcontent.nddaily.com/1/93/193002e668758ea9/Blog/d3a/391b93.html |title=广深线上60载不改守桥心 |accessdate=2011-06-02 |publisher=南方都市报 |date=2009-09-30 |deadurl=yes |archiveurl=https://archive.is/20120715131432/http://gcontent.nddaily.com/1/93/193002e668758ea9/Blog/d3a/391b93.html |archivedate=2012-07-15 }}</ref>。目前，广深铁路在东莞市石龙镇共有三座跨越东江南干流的铁路桥，分别为石龙南桥（1911年建成，单线，已停用）、石龙南特大桥（1994年建成，双线）、四线石龙南桥（2007年建成，双线）<ref>{{Cite web |url=http://www.gdwh.com.cn/whyc/2009/0721/article_1720.html |title=东莞广深铁路石龙铁路南桥旧址(2008) |accessdate=2011-06-02 |publisher=广东文化网 |date=2009-07-21  }}</ref>。

== 广深城际列车 ==
[[File:CRH6A(GSR_Custom)_at_Guangzhou-Shenzhen_Rail_Line_2019_04.jpg|thumb]]定制的CRH6A运行在广州市区（摄于2019年）]]
