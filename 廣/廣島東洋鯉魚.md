'''廣島東洋鯉魚'''（{{lang|ja|広島東洋カープ}}，Hiroshima Toyo Carp）是一支隸屬[[日本職棒|日本職棒]][[中央聯盟|中央聯盟]]的球隊。成立於1949年11月28日[[日本職棒|日本職棒]]兩聯盟分立前夕成立，當時隊名為廣島鯉魚隊，同年12月26日加入[[日本職棒|日本職棒]][[中央聯盟|中央聯盟]]。廣島隊當初是以廣島綜合球場為主場，到1957年七月才遷移至廣島市民棒球場，2008年球季結束後，市民球場正式功成身退，主場遷移至剛落成的[[馬自達Zoom-Zoom廣島球場|馬自達廣島球場]](新廣島市民球場。此外，廣島隊一直以來，除了與歐力士藍浪隊並存的1991年至2004年之外，都是日職唯一英文隊名中吉祥物採用單數表示的，十分特別。　　

廣島隊是由縣政府、市政廳、吳市、中國新聞社（日本的「中國地方」指的是本州最西部的岡山、廣島、山口、島根、鳥取五縣，與中華人民共和國、中華民國均無關）、日本專賣公社、廣島電鐵、東洋工業等廣島地區政治界與金融界籌資設立的，成立之初由於營運資金缺乏，曾經於1951年打算與當時的大洋鯨隊合併，但被廣島市民強烈反對，廣島市民並且發起募捐籌措廣島隊的營運資金，這樣的募捐活動，一直持續到1968年才停止。　　

1968年東洋工業（[[馬自達汽車|馬自達汽車]]前身）松田家族入主[[廣島|廣島]]，成為最大股東，廣島隊因而冠上了「東洋」的名稱，日後也獲得三次[[日本大賽|日本大賽]]冠軍。不過1980年代之後，松田家族已經淡出[[馬自達|馬自達]]公司的經營，以[[馬自達|馬自達]]公司為主的松田家族依舊是球團的最大股東，廣島隊同時也是12球團中唯一獨立經營運作的球團。　　
 
廣島隊於1993年~1996年間曾與[[中華職棒|中華職棒]][[時報鷹隊|時報鷹隊]]建立合作關係，這段時間派出多名旗下球員短期加入[[時報鷹隊|時報鷹隊]]參賽，累計實戰經驗，包括奇戈R.C、李維C.R、鈴木健、[[菊地原毅|菊地原毅]]、千代丸亮彥等選手，但因屬於「野球留學」性質，日後在這些球員[[日本職棒|日本職棒]]名鑑上的經歷介紹中，皆省略了加入[[時報鷹隊|時報鷹隊]]這一段。至於這些回歸[[日本職棒|日本職棒]]的球員中，則以奇戈R.C成績最佳。　　

值得一提的是，由於積極開發球隊周邊商品、與地方結合，以及控制薪資，廣島隊已經連續32年賺錢（黑字）。這對於一支位於[[廣島|廣島]]這個中型城市，連年戰績不甚理想的球隊，實在是相當難得。

== 概要 ==
廣島東洋鯉魚棒球隊的根據地為[[廣島縣|廣島縣]]，主場地在[[新廣島市民球場|MAZDA Zoom-Zoom廣島棒球場]]，二軍球隊（西部聯盟所屬）的主場則在[[山口縣|山口縣]][[岩國市|岩國市]]的廣島東洋鯉魚由宇練習場。

跟其他日本職棒球隊不同的是，[[馬自達|馬自達]]汽車公司雖然為主要大股東，卻不參與球團的經營，改由該公司創辦人的松田家族經營（球團現任老闆松田元為馬自達實質創辦人[[松田重次郎|松田重次郎]]的曾孫）。雖然該球隊的股票並不開放給廣島市民持有，但由於馬自達沒有實際參與經營，仍在日本普遍存有「市民球團」的印象。

關於媒體採用的簡稱，一般[[漢字|漢字]]用「廣」、[[拉丁字母|羅馬字母]]用「C」，也有職棒雜誌用「鯉魚」（原文作：コイ）略稱之。

2013年球季，廣島鯉魚拿下中央聯盟第三名，終止廿一世紀以來沒進入過A級隊伍的紀錄，並在央聯季後賽首輪打敗第二名的[[阪神虎|阪神虎]]，但於央聯冠軍賽中輸給[[讀賣巨人|讀賣巨人]]，無緣晉級日本一決賽。

2016年9月10日，與讀賣巨人天王山之戰，靠著[[黑田博樹|黑田博樹]]的好投加上打線發揮．最終以6:4獲勝，拿下暌違25年的央聯例行賽冠軍，並在CS以4：1擊退[[橫濱DeNA海灣之星|橫濱DeNA海灣之星]]，順利晉級[[日本大賽|日本大賽]]，但最終被[[北海道日本火腿鬥士|北海道日本火腿鬥士]]上演日本大賽史上第三次讓二追四逆轉戲碼，無緣隊史第四冠。

2017年9月18日，廣島鯉魚8局上靠著洋將巴提斯塔(Xavier Batista)在一、二壘有人時擊出的左外野安打，打下超前分，終場3:2獲勝，連2年登上中央聯盟冠軍寶座。然而決勝輪再度與橫濱DeNA碰頭，卻再度上演去年被讓二追四逆轉的戲碼，無緣晉級日本一。

2018年9月26日，靠著[[九里亜蓮|九里亜蓮]]的好投加上打線發揮．最終以10:0完封[[東京養樂多燕子|東京養樂多燕子]]，完成央聯3連霸，在CS決勝輪接受讀賣巨人的挑戰，最終3連勝(加上1場保送勝)，直落4橫掃讀賣巨人，3年內2度進軍日本一，挑戰衛冕軍[[福岡軟銀鷹|福岡軟銀鷹]]，最終1勝4敗1和持續無緣日本一。

== 球隊名沿革 ==
* 1950年：廣島鯉魚隊。此隊名由廣島縣出身的[[政治家|政治家]][[谷川昇|谷川昇]]命名，其由來為：
# 流經廣島市的太田川為[[鯉魚|鯉魚]]的產地。
# 位於廣島市的[[廣島城|廣島城]]別稱為'''鯉城'''。由於球隊成立之初，廣島歷經[[廣島市原子彈爆炸|原子彈轟炸]]，為了鼓勵廣島地區重新站起來，以鯉躍龍門之意象取了這個隊名。雖然候選隊名尚有彩虹、原子、黑熊、鳩、綠等，但谷川昇認為「文獻中記載，鯉魚比其他魚類還要長，外型可愛又能幻化成躍過龍門的動物，況且廣島市西區的己斐（音讀成koi，與[[日文|日文]]的『鯉魚』こい同音）也是從鯉魚轉化而來，也能與『戀愛』相通（因日文的「恋」與鯉魚同音）」，因此後來以英文Carp取名為「カープ」。
* 1968年：馬自達（舊稱東洋工業）成為最大股東，為了稅賦制度問題而更名為'''廣島東洋鯉魚隊'''。
