'''廣州海事博物館'''，是[[廣東|廣東]][[廣州|廣州]]籌建中的一個[[博物館|博物館]]。

== 位置 ==
2013年時，第二届[[廣州市城市規劃委員會|广州市城市规划委员会]]第八次会议上，海事博物馆项目確認選址黄埔[[南海神庙|南海神庙]]东侧興建<ref>[http://gz.house.163.com/13/0205/07/8MUD16T700873C6D.html 广州海事博物馆选址敲定] 2013-02-05 信息时报</ref>，具体在南海神庙南广场的西面，浴日亭以南位置。

== 定位 ==
博物馆定位为广州“十二五”重大文化设施项目之一，计划展现广州作为[[海上丝绸之路|海上丝绸之路]]发祥地之一及千年商都在中国大陆，以及世界贸易当中的地位和历史<ref>[http://www.sh-leafun.com/supernatural-fx-showreel/ 上海励丰中标广州海事博物馆] 上海励丰</ref>。

== 建築計劃 ==
项目由黄埔区政府负责，场馆面积预计超过1万平方米。在2014年10月规划部门通过的规划修编中，场馆会有4组建筑组成，最高4层。而博物馆用地中水域面积超过一半，博物馆的4层高主楼会以漂浮在水中央的大船形式建造<ref>[http://www.chinanews.com/cul/2014/10-31/6735691.shtml 广州海事博物馆将建 主楼将如大船漂浮在水中] 2014-10-31 南方日报</ref>。主要建设内容包括展陈区、藏品库房、综合服务区、业务管理用房、设备技术用房等，工程总投资达1.83亿元<ref>[http://www.gz.gov.cn/GZ62/2.2/201712/06d4cd4ba1c04b7da3d332bd2bc404d4.shtml 广州市黄埔区人民政府办公室关于市政协十三届一次会议第2035号委员提案的答复] 广州市人民政府 2017-12-05</ref>。
