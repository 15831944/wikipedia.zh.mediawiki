'''廣東道官立小學'''（{{lang-en|Canton Road Government Primary School}}）是位於[[香港|香港]][[九龍|九龍]][[尖沙咀|尖沙咀]][[廣東道|廣東道]]178號的一間[[官立小學|官立小學]]，於1959年創校。

== 辦學宗旨 ==
*以「德、智、體、群、美」訓勉學生。
*實行「訓輔合一，全校參與」的原則，讓全校老師參與輔導工作，關懷學生，締造和諧關愛的學習環境。
*實施小班教學，營造愉快有效的學習環境。
*通過「全人教育」，培養學生的多元智能，並激發學生潛能。

== 科目 ==
學校共開設9個學科，包括中文、英文、數學、常識、體育、音樂、電腦、普通話、視藝科<ref>{{cite web|url= https://crgps.edu.hk/SchoolChara.asp?top=3|title= 各學科特色}}</ref>。

== 班級編制 ==
班級總數為18班，每級約有3班。

== 學校設施 ==
學校設有18個課室，1個禮堂，1個操場。
