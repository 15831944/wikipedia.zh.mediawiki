'''廣瀨正志'''（{{bd|1947年|11月6日}}<ref> 『聲優名鑑』、602頁、[[成美堂出版|成美堂出版]]、1999年、ISBN 978-4415008783</ref>），日本男性配音員，出身於[[大阪府|大阪府]]。隸屬於[[Production_baobab|Production baobab]]。

== 經歷、人物 ==
2014年9月24日因為生病需要長期住院療養<ref>[http://pro-baobab.jp/ ぷろだくしょんバオバブ公式サイト]</ref><ref>[http://www.sponichi.co.jp/entertainment/news/2014/09/24/kiji/K20140924008985250.html 声優の広瀬正志 長期入院へ…「ガンダム」のランバ・ラル役] スポニチアネックス 2014年9月24日</ref>，令部分動畫角色的配音有所更改：
* 原本在《[[尋找失去的未來|尋找失去的未來]]》中配音的'''本城作之進'''一角改由[[大塚明夫|大塚明夫]]接替。<ref>{{Cite web |author= |date= |url=http://ushinawareta-mirai.com/news/|title=ファンの皆様へ|work=失われた未来を求めて |publisher= |accessdate=2014-09-28}}</ref>
* 《[[GUNDAM創戰者|GUNDAM創戰者TRY]]》中的'''拉爾先生'''一角改由[[寶龜克壽|寶龜克壽]]接替。<ref>[http://gundam-bf.net/news.php#8385 お知らせ] {{webarchive|url=https://web.archive.org/web/20140625054441/http://gundam-bf.net/news.php |date=2014-06-25 }}</ref>
* 《[[ONE_PIECE|ONE PIECE]]》中的'''利克·德爾多三世'''一角改由[[銀河萬丈|銀河萬丈]]接替。
