'''廣島岩國道路'''（{{Jpn|j=広島岩国道路|kn=ひろしまいわくにどうろ|rm=Hiroshima iwakuni dōro}}）是從[[日本|日本]][[廣島縣|廣島縣]][[廿日市市|廿日市市]]至[[大竹市|大竹市]]的[[国道2号_(日本)|國道2號]][[繞道|繞道]]。由[[西日本高速公路|西日本高速公路]]（NEXCO西日本）以[[收費道路|一般收費道路]]方式管理。

高速公路路線編號與[[山陽自動車道|山陽自動車道]]・{{link-ja|小郡道路|小郡道路}}・{{link-ja|山口宇部道路|山口宇部道路}}（{{link-ja|嘉川交流道|嘉川インターチェンジ}} - {{link-ja|宇部系統交流道|宇部ジャンクション}}間）共同被編為「'''E2'''」<ref>{{Cite web|url=http://www.mlit.go.jp/road/sign/numbering/list/index.html|title=高速道路ナンバリング一覧|accessdate=2017年2月26日|publisher=国土交通省}}</ref><ref>{{Cite web|url=http://www.mlit.go.jp/road/sign/numbering/map/index.html|title=高速道路ナンバリング路線図|accessdate=2017年2月27日|publisher=国土交通省}}</ref>。

== 概要 ==
廣島岩國道路是為了紓緩國道2號交通所建設的一般收費道路。

{{link-ja|廿日市系統交流道|廿日市ジャンクション}} - {{link-ja|大竹系統交流道|大竹ジャンクション}}區間與山陽道本線連通，可視為山陽道的一部分。根據法院裁定，山陽道需在此區間另外建設道路，但由於山陽道的建設計畫遭到地方居民反對，至今仍未有具體的建設規劃。本道路的規格屬第1種第3級，最高速度為80km/h。另外，本區間也是[[亞洲公路1號線|亞洲公路1號線]]「AH1」的一部分。

{{link-ja|廿日市交流道|廿日市インターチェンジ}} - 廿日市系統交流道區間則可視為連結山陽道與國道2號（{{link-ja|西廣島繞道|西広島バイパス}}）的支線。

[[日本道路公團|日本道路公團]]最初規劃的「廣島岩國道路」是從廣島縣廿日市市至[[山口縣|山口縣]][[岩國市|岩國市]]。1990年開通廿日市市至大竹市後，1994年新規畫大竹市至岩國市的[[地域高規格道路|地域高規格道路]]{{link-ja|岩国大竹道路|岩国大竹道路}}，2000年廣島岩國道路的大竹市以西區間正式廢除。原先廣島岩國道路的大竹市-岩國市區間是規劃在瀨戶內海側，規劃岩國大竹道路時則改走內陸，連通{{link-ja|國道188號|国道188号}}{{link-ja|岩國南繞道|岩国南バイパス}}<ref>{{Cite web|url=http://www.shugiin.go.jp/Internet/itdb_shitsumon.nsf/html/shitsumon/b169478.htm|title=衆議院議員平岡秀夫君提出「岩国・大竹道路」に関する再質問に対する答弁書|date=2008-06-13|accessdate=2014-02-07|publisher=内閣}}</ref>。

未來將與{{link-ja|廣島南道路|広島南道路}}連通，可前往廣島市中心。

== 收費 ==
廣島岩國道路的收費方式與[[高速自動車國道|高速自動車國道]]不同，將車種分為普通車、大型車與特大車三類。高速自動車國道的[[輕自動車|輕自動車]]等與中型車列為普通車。

普通車的費率為34日圓/km<ref name="mlit131220">{{Cite web|url=http://www.mlit.go.jp/common/001022810.pdf#page=6 |title=新たな高速道路料金に関する基本方針 |format=PDF |date=2013-12-20 |accessdate=2015-06-13 |publisher=国土交通省 |page=5 }}</ref>或43日圓/km<ref name="mlit110715">{{Cite web|url=http://www.mlit.go.jp/common/000160309.pdf |title=高速道路の割高区間等の料金割引について |format=PDF |date=2011-07-15 |accessdate=2015-06-13 |publisher=国土交通省 |pages=1-2 }}</ref><!-- ターミナルチャージの有無による違いとみられる -->，比高速自動車國道普通區間（24.6日圓/km）為高，但從2014年4月起，暫定十年內提供[[ETC|ETC]]車輛享有與普通區間同等的費率<ref name="nexconr140314">{{Cite web |url=http://corp.w-nexco.co.jp/corporate/release/hq/h26/0314/pdfs/01.pdf |title=「新たな高速道路料金」について  |accessdate=2015-06-13 |date=2014-03-14 |format=PDF |work=新たな高速道路料金について |publisher=西日本高速道路 }}</ref>。因此，非ETC車的全線普通車費用為770日圓，ETC車為430日圓。
若是從山陽自動車道通過廿日市系統交流道 - 大竹系統交流道區間，則先計算前後高速自動車國道的距離費用，再加算廣島岩國道路的費用。

=== ETC時間段折扣 ===

由於是一般收費道路，最初並未有時間段折扣，直至2006年4月1日起與山陽自動車道導入相同折扣（深夜折扣、通勤折扣）<ref>{{Cite web |url=http://corp.w-nexco.co.jp/corporate/covenant/debt/h18_0331/pdfs/etc_discount.pdf |title=一般有料道路へのＥＴＣ時間帯割引の導入について |accessdate=2015-06-14 |format=PDF |publisher=西日本高速道路 }}</ref>。另也享有緊急綜合對策、生活對策的時間段折扣（深夜折扣擴充、平日夜間折扣、假日白天折扣/假日特別折扣）。2014年4月起實施深夜折扣與假日折扣。現在也是通勤折扣後繼的平日早晚折扣實施對象。
普通車中屬於高速自動車國道（5區分）中型車的車輛不能享有休日白天折扣、假日特別折扣、假日折扣。
