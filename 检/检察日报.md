《'''检察日报'''》是[[中华人民共和国最高人民检察院|中华人民共和国最高人民检察院]]的[[机关报|机关报]]。

== 历史 ==
1991年1月11日，[[中国共产党|中国共产党]][[最高人民检察院党组|最高人民检察院党组]]决定成立“创建《中国检察报》筹备小组”。1991年2月2日，[[中华人民共和国新闻出版署|中华人民共和国新闻出版署]]批复，同意《中国检察报》创办，由最高人民检察院主管并主办。1991年4月12日，最高人民检察院举办“《中国检察报》筹备工作电话会议”，确定办报方针为“立足检察、面向社会”。1991年4月，[[邓小平|邓小平]]题写了《中国检察报》报名。1991年5月25日，《中国检察报》试刊号出版；同年7月4日正式创刊。<ref name="jcrb">[http://zjby.jcrb.com/bsjj/201306/t20130619_1138058.shtml 检察日报社概况]，检察日报官网，2013-06-19</ref>

1992年1月1日，《中国检察报》从周一刊改为周二刊，在每周的周一、周五出版；同月，《中国检察报》被[[中华全国法制新闻协会|中华全国法制新闻协会]]吸收成团体会员。1994年1月1日，《中国检察报》从周二刊改为周四刊，在每周的周二、周四、周六、周日出版。1996年1月1日，《中国检察报》从周四刊改为日报，并更名为《检察日报》。<ref name="jcrb" />

== 下属传媒 ==
检察日报社的下属期刊有<ref name="jcrb" />：
*《[[人民检察|人民检察]]》
*《[[方圆|方圆]]》
*《[[法治新闻传播|法治新闻传播]]》

检察日报社拥有中华人民共和国第一家由[[国务院新闻办公室|国务院新闻办公室]]直接管理的法治类[[网络媒体|网络媒体]]《[[正义网|正义网]]》。1999年1月《检察日报》电子版问世，2000年更名为《正义网》。2006年，检察日报社开始承办最高人民检察院网站。检察日报社还负责管理最高人民检察院影视中心以及电视法治栏目《[[法治中国|法治中国]]》<ref name="jcrb" /><ref>[http://zjby.jcrb.com/zjbs/ 中检报史]，检察日报，于2014-11-03查阅</ref>

===最高人民检察院影视中心===
'''最高人民检察院影视中心'''，位于[[北京市|北京市]][[石景山区|石景山区]]香山南路109号，是“由最高人民检察院成立、[[国家新闻出版广电总局|国家新闻出版广电总局]]批准、具有独立[[事业单位|事业单位]]法人资格和影视制作资质的专业影视机构。”

最高人民检察院影视中心的前身是1996年成立的'''中国检察影视中心'''，2003年更名'''最高人民检察院影视中心'''<ref name=jj/>。该中心由检察日报社代管<ref name=jj/><ref>{{Cite web |url =http://www.gjsy.gov.cn/cxzl/ |title = 最高人民检察院影视中心 |author =  |date =  |publisher = 事业单位在线 |language =  |accessdate =2018-08-12  |quote =  }}</ref>，主要负责策划、出品、拍摄、制作各类题材尤其是检察题材的电视剧、电影、微电影、专题片、电视栏目等影视作品，同时配合最高人民检察院的中心工作开展影视宣传及法治教育。该中心曾多次获“[[五个一工程奖|五个一工程奖]]”、“[[飞天奖|飞天奖]]”、“[[华表奖|华表奖]]”、“[[白玉兰奖|白玉兰奖]]”等奖项。该中心邀请作家[[莫言|莫言]]任艺术总监，作家[[周梅森|周梅森]]任顾问<ref name=jj>{{Cite web |url =  http://www.spptv.cn/news/jsxx/201704/t20170414_4126.html|title =影视中心简介  |author =  |date =  |publisher =  中国法制影视传媒|language =  |accessdate =  |quote =  }}</ref>。
