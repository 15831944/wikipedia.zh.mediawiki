'''钓鱼执法'''（entrapment），又称'''钓鱼式执法'''、'''倒钩（执法）'''或'''执法圈套'''，指的是[[行政|行政]][[执法|执法]]部门有意隐蔽身份，采取手段，候待甚至引诱被执法人做出[[违法|违法]]行为，而后将其抓捕的执法形式。因其为执法而引诱犯罪，纵容犯罪的出发点，有执法而违法的争议。

== 历史 ==
钓鱼执法在[[证据学|证据学]]上存在一定的[[历史|历史]]，但是随着[[法治|法治]]的日渐成熟，[[非法|非法]]取得的证据已不被承认被一些国家所抛弃<ref>{{cite book|last=Gaines|first=Larry|authorlink=|coauthors=Miller, LeRoy|title=Criminal Justice In Action: The Core |year=2006|publisher=Thomson/Wadsworth|location=Belmont, CA|isbn=0-495-00305-0}}</ref>；例如[[證據排除法則|證據排除法則]]和[[毒樹果實理論|毒樹果實理論]]。

===中國大陸===
====名词====
*“'''钓头'''”：管理“钩子”的人，负责组织钩子工作，分配利润，听从执法部门调遣。根据新华社记者的报道：“一名自称做过“钓头”的女性对记者透露，一般来说，一个区域大致有一两个“钓头”，“钓头”手下有20个左右“钓钩”。开展集中执法行动之前，交通营运执法部门的负责人会与“钓头”接头。一星期开展一次集中行动，每次都能抓200辆左右黑车。执法部门与“钓头”谈好，抓一辆黑车给500元，200元“钓头”自留，200元给“钓钩”，另外100元是给执法人员的回扣。“钓头”和“钓钩”之间一个月结一次“工资”。”<ref name="钓头和钩子">{{cite web |url=http://news.xinhuanet.com/focus/2009-10/25/content_12325506.htm |title=“钓头”和“钓钩”者追踪 |deadurl=yes |archiveurl=https://web.archive.org/web/20091030093310/http://news.xinhuanet.com/focus/2009-10/25/content_12325506.htm |archivedate=2009年10月30日 |df= }}</ref>
*“'''钩子'''”/“'''钓钩'''”：听从钓头安排，负责钓鱼，即诱骗鱼（民）上当的人员，成功上车后，当车驶近执法人员时，要一手拉手刹，一手拔车钥匙，配合执法人员查扣车辆。每钓到一条鱼（民），可获利约400元人民币。<ref>{{cite web |url=http://news.xinhuanet.com/mrdx/2009-10/26/content_12330562.htm |title=“孙中界事件”：搭车的陈某某确系“钩子” |deadurl=yes |archiveurl=https://web.archive.org/web/20091030094509/http://news.xinhuanet.com/mrdx/2009-10/26/content_12330562.htm |archivedate=2009年10月30日 |df= }}</ref>钩子因遭鱼（民）怨恨，曾有多起被杀死的极端情况发生<ref>{{cite web |url=http://www.he.xinhuanet.com/news/2004-09/24/content_2930214.htm |title=河北秦皇岛市打击“黑车”竟然出现“职业钓鱼族” |deadurl=yes |archiveurl=https://web.archive.org/web/20091031114435/http://www.he.xinhuanet.com/news/2004-09/24/content_2930214.htm |archivedate=2009年10月31日 |df= }}</ref>。
*“'''鱼（民）'''”：机动车驾驶员，除了非法营运的私人出租车主外，还有私家车主等有可能会因贪图小利<ref name="广州钓鱼">{{cite web |url=http://gd.people.com.cn/GB/123937/123963/10257356.html|title=“钓鱼”执法惊现广州? 运管局出面澄清}}</ref>，或出于好心<ref name="百余车主">{{cite web |url=http://news.sina.com.cn/c/2009-10-28/040118921721.shtml |title=百余名遭上海钓鱼执法车主要求退还罚款}}</ref>而上当受骗，被钩子钩到，每次被钓损失5千-5万元人民币不等<ref name="钓头和钩子"/>。
*“'''黄牛'''”：因和扣车部门有关系，可以帮助鱼（民）以更少的支出，将被扣车辆取出的人，大大降低了鱼（民）的损失<ref name="百余车主" />。

===香港===
在香港，此类执法现象称为“放蛇”，指[[行政|行政]]、[[执法|执法]]部门隐蔽身份（如假扮嫖客、妓女、賭客或其他顧客），待当事人从事[[违法|违法]]行为時揭露身份，将其抓捕。<ref>[http://hk.news.yahoo.com/article/100303/4/gu14.html 警搗元朗3地下麻將館]</ref><ref>[http://hk.news.yahoo.com/article/100128/4/gcj9.html 電腦店售盜版 海關放蛇搗破]</ref>放蛇行动包括警察假扮妓女，将嫖客逮捕，<ref>[http://hm.people.com.cn/GB/10078431.html 港警花扮妓女"放蛇" 80岁老翁"上钩"被捕]</ref>以及假扮嫖客，在答應性交易後将妓女逮捕的行为。<ref>[http://www.ziteng.org.hk/events/ma05_c.html 投訴遭拒鳳姐啞忍　警放蛇食霸王雞] {{webarchive|url=https://web.archive.org/web/20090603192332/http://www.ziteng.org.hk/events/ma05_c.html |date=2009-06-03 }}</ref>（但若警察的確與妓女發生性行為，則根據香港法例，不能對該妓女予以起訴）。此外也有警察假意购买毒品，然后抓捕毒贩。<ref>[http://www.dsaj.gov.mo/EventForm/ContentFileGen.aspx?Rec_Id=4805 從香港警方放蛇緝毒說起]</ref>

{{fact|然而，香港俚語中的放蛇應對應英語中的{{link-en|string operation|string operation|sting operation}}，即犯罪者本身犯下罪行再於招攬顧客時遇上假扮顧客的執法者（如毒犯被假扮顧客的執法者拘捕前，他早已犯下藏毒罪），與執法者故意設下圈套，讓尚未犯罪的目標被當場拘捕的entrapment不同。}}

===美国===
美国也有这一类的执法现象<ref name="ny">[http://news.xinhuanet.com/world/2005-02/04/content_2545627.htm 美国式扫黄：警方“放蛇” 假戏真做] {{webarchive|url=https://web.archive.org/web/20090602062319/http://news.xinhuanet.com/world/2005-02/04/content_2545627.htm |date=2009-06-02 }}</ref><ref>[http://news.hunantv.com/x/g/20091029/304710.html 纽约警方“放蛇”华商中招 卖酒给少年两华人被捕]</ref>。如在网上装作未满14岁少女吸引恋童癖者，等待后者要求发生性关系<ref name="entrap">{{cite web | title = What is entrapment? | url = http://www.slate.com/articles/news_and_politics/explainer/1999/09/what_is_entrapment.html| publisher = http://www.slate.com | accessdate = 2012-02-14}}</ref>，假扮不法交易商引出不法交易者<ref name="goo">{{cite web | title = Con Artist Starred in Sting That Cost Google Millions | url = http://online.wsj.com/article/SB10001424052970204624204577176964003660658.html |  publisher = http://online.wsj.com Online Wallstreet journal| accessdate = 2012-02-14}}</ref>。美国法律规定，被“钓鱼”中套的一方只有在证明了以下三点后才能说明自身无过错：
#犯罪的想法来源于执法方而不是其自发的；
#执法方诱使其犯下了罪行；其中的“诱使”需要强有力的证明，仅仅证明执法方营造了有利犯罪的环境或条件不能说明“诱使”，必须证明执法方有强制性或故意劝服的行为<ref name="entrap"/>。
#在执法方诱使之前，其并未准备好以及未有自发意愿犯下指定罪行。如果一个执法方的“卧底”向一个携带了一公斤海洛因的人购买海洛因后将其抓捕，后者将不能自证无过错，因为他贩卖毒品的意图明显<ref name="entrap"/>。
