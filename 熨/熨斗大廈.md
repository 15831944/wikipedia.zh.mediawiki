'''熨斗大廈'''（'''Flatiron Building'''），建造時稱為'''福勒大廈'''（Fuller Building），在1902年完工，當時為[[美國|美國]][[紐約市|紐約市]]最高的大樓之一，同时也是早期摩天大楼的代表作、钢架建筑的先驱。地址為[[曼哈頓|曼哈頓]]島[[第五大道|第五大道]]175號，坐落在[[23街|23街]]、[[百老匯大道|百老匯大道]]和[[第五大道|第五大道]]交叉的一個三角形的街區上，尖頭指向[[麥迪遜廣場|麥迪遜廣場]]的南邊。大樓周邊的社區稱為[[熨斗區|熨斗區]]。

== 建築結構 ==
熨斗大樓是由[[芝加哥|芝加哥]]的建築師[[丹尼爾·伯恩罕|丹尼爾·伯恩罕]]設計，採用[[布雜藝術|布雜藝術]]式建築風格。如同古典希臘[[柱式|柱式]]一樣，其石灰岩和{{link-en|上釉陶瓦|Glazed architectural terra-cotta}}牆面垂直方向地被分為三部分。此大樓為使用鋼骨結構建築的始祖之一，因此建築高度能達到285呎（87公尺），對於當時使用其他建法的建築來說是非常困難的。

建築尖端的部分大約只有6.5呎（2公尺）寬，但高達22層樓，285呎（87公尺）。雖然[[公園街大樓|公園街大樓]]（1899年完工）有同樣的年齡和同樣的高度，但熨斗大樓仍被認為是[[曼哈頓|曼哈頓]]最老的[[摩天大樓|摩天大樓]]。

熨斗大樓的特殊形狀並不是首例，事實上它是世界上第三棟熨斗形狀的大樓，第一棟為1892年在[[多倫多|多倫多]]建造的[[古德汉大厦|古德汉大厦]]，而第二棟為1897年在[[亞特蘭大|亞特蘭大]]建造的某棟大樓，以上兩棟大樓的建築大小不及熨斗大樓那樣的雄偉。

== 影響 ==
{{quote|
我發現我目瞪口呆地望著某一摩天大樓──熨斗大廈的船頭在午夜時分時破浪前進，駛過百老匯大道和第五大道。──[[赫伯特·喬治·威爾斯|赫伯特·喬治·威爾斯]]（1906年）
}}

熨斗大樓的名稱來自於其三角型的街區<ref>"New Building on the Flatiron" ''New York Times'' (3 March 3 1901), page 8: "the famous 'flatiron' block"</ref>，而他看起來就像的一個[[熨斗|熨斗]]一樣，在提供資金建設此大樓的公司創立者喬治·A·福勒在逝世兩年後，被命名為福勒大樓。<ref>"Flatiron Structure to be Called the Fuller Building", ''New York Times'' (9 August 1902) page 3.</ref>當時當地居民對此大樓相當不看好，甚至下賭注看大樓被風吹倒時的瓦礫能飛多遠。

== 今日的熨斗大廈 ==
今日，熨斗大廈經常在電視廣告和記錄片裡出現，為紐約市早期著名的地標{{NoteTag|使用熨斗大樓當作紐約市象徵的情形在[[911恐怖攻擊事件|911恐怖攻擊事件]]後，因[[世界貿易中心|世界貿易中心]]的倒塌，變得越來越顯著。}}。此大廈出現在《[[晚間秀|晚間秀]]》的節目開場影片，以及《[[蜘蛛人|蜘蛛人]]》系列電影裡的[[Daily_Bugle|Daily Bugle]]大廈，是個熱門的旅遊景點，1979年被列入國家史蹟名錄，1989年成為美國[[國家歷史性地標|國家歷史性地標]]之一<ref name="nhlsum">{{cite web |url = http://tps.cr.nps.gov/nhl/detail.cfm?ResourceId=1809&ResourceType=Building
|title=熨斗大廈 |date=2007-09-12 |work=國家歷史性地標概要列表 |publisher=國家公園服務處}}</ref><ref name="nrhpinv">{{cite web |url = http://pdfhost.focus.nps.gov/docs/NHLS/Text/79001603.pdf" |title=熨斗大廈|author=Carolyn Pitts|work=National Register of Historic Places Registration|date=1989-02-09 |publisher=國家公園服務處}}</ref><ref name="nrhpphotos">{{cite web |url = http://pdfhost.focus.nps.gov/docs/NHLS/Photos/79001603.pdf |title=熨斗大樓自1979年以來的照片和外牆影像|work=National Register of Historic Places Inventory |date=1989-02-09 |publisher=國家公園服務處}}</ref>。然而，此大廈也是一個功能性強的辦公大樓，目前即將被[[德國|德國]]的一家大型出版社接手設為總部<ref name="macmillan">[http://us.macmillan.com/splash/about/index.html Macmillan: About]</ref>。

2005年，熨斗大廈整建時，牆上貼著一幅高達15層樓的違法廣告，引來紐約市居民的反對，引起紐約市建築部的注意並介入此反對事件，最後要求大廈持有人拆除廣告<ref name="nytimes">{{cite web |url = http://www.nytimes.com/2005/04/08/nyregion/08flatiron.html?_r=1&oref=slogin |title=紐約時報|date=2005-04-08 |publisher=[[紐約時報|紐約時報]]}}</ref>。
