'''環聯資訊有限公司'''（英語：[[TransUnion|TransUnion]]）是[[香港|香港]]消費者[[信貸|信貸]]資料服務機構，提供[[信貸|信貸]]資料管理服務，蒐集個人[[信貸|信貸]]綜合紀錄，於[[香港|香港]]載有520萬位消費者之個人[[信貸|信貸]]紀錄<ref>[https://www.transunion.hk/zh/about-us/about-transunion 關於我們 環聯]</ref><ref name="AE">[https://www.americanexpress.com/hk/ch/content/credit-know-how/credit-report-basics.html 信貸報告小知識 美國運通]</ref>。

== 公司歷史 ==
環聯資訊有限公司於1982年成立，前身為「香港資信有限公司」，由12間提供[[汽車|汽車]]及設備[[信貸|信貸]]融資的機構共同成立。於1985年，開始收集各類負面[[信貸|信貸]]資料，建立綜合[[信貸|信貸]]資料庫，於80年代後期，機構股東數目增至16個<ref name="TU History">[https://www.transunion.hk/zh/about-us/company-history 公司發展歷程 環聯]</ref>，其後於1999年被[[美國|美國]][[TransUnion|TransUnion]] International正式收購<ref name="TU History"></ref>。於2003年，隨著香港個人資料私隱專員公署修訂個人信貸資料實務守則，容許共用較大範圍的正面個人信貸資料<ref>[http://www.hkma.gov.hk/media/gb_chi/publications-and-research/quarterly-bulletin/qb200603/fa1.pdf 共用正面個人信貸資料的好處 香港金融管理局季報]</ref>，截至2015年底，環聯資料庫的信貸報告儲存量增至8720萬份<ref name="TU History"></ref>。

== 產品及服務 ==
*信貸報告：整合個人過往所有信貸活動紀錄，包括[[信貸|信貸]]紀錄、[[信貸|信貸]]賬戶貸款額及結餘、還款紀錄、公眾紀錄及過往查詢紀錄信貸資料<ref name="AE"></ref>。 
*[https://www.transunion.hk/zh/education/do-you-know-your-credit-score 信貸評分]：分為A至J級反映個人信貸狀況，以A級為最高<ref>[https://finance.ctgoodjobs.hk/article/show_article.aspx/1173-15686-%E5%80%8B%E4%BA%BA%E4%BF%A1%E8%B2%B8%E7%B4%80%E9%8C%84-%E4%BD%A0%E4%BF%82a%E5%AE%9Aj 個人信貸紀錄 你係A定J？ CTFinance]</ref> 如何計算信貸評分?<ref>[https://www.transunion.hk/zh/education/do-you-know-your-credit-score 環聯如何計算信貸評分?]</ref>
*信貸提示：當個人的信貸資料出現變更，以[[短訊|短訊]]及/或[[電郵|電郵]]向客戶發出通知<ref>[https://www.transunion.hk/zh/# 環聯信貸報告 環聯]</ref>。
*債務及財務狀況分析：讓消費者快速預覽其債務紀錄、每月收入、支出和債務負債比率(DTI)<ref>[https://www.transunion.hk/zh/education/debt-analysis 債務及財務狀況分析 環聯]</ref>。
