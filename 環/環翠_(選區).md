'''環翠'''（英語：Wan Tsui）是香港[[東區區議會|東區區議會]]下轄的[[選區|選區]]，代號'''C13'''，成立於1994年，現任區議員為[[民主建港協進聯盟|民建聯]]成員[[龔栢祥|龔栢祥]]。

==範圍==
選區範圍為[[環翠邨|環翠邨]]、[[茵翠苑|茵翠苑]]、[[悅翠苑|悅翠苑]]、[[仁樂大廈|仁樂大廈]]和[[建業大樓|建業大樓]]，與其相鄰的選區有[[景怡|景怡]]、[[翡翠_(選區)|翡翠]]、[[興民|興民]]、[[翠德|翠德]]、[[漁灣|漁灣]]和[[佳曉|佳曉]]，另外還有南區的[[赤柱和石澳|赤柱和石澳]]。

==沿革==
環翠選區首次出現於[[1994年香港區議會選舉|1994年香港區議會選舉]]，由[[民主建港協進聯盟|民建聯]][[龔栢祥|龔栢祥]]、[[自由黨_(香港)|自由黨]]張漢廷和原[[柴灣南|柴灣南]]區議員[[覃定邦|覃定邦]]爭奪，最後[[龔栢祥|龔栢祥]]勝出。

[[1999年香港區議會選舉|1999年香港區議會選舉]]，當時人口估算約17,108人<ref>{{cite web |url=http://www.elections.gov.hk/elections/dcelect99/cg12.htm |title=存档副本 |accessdate=2012-04-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20111229025617/http://www.elections.gov.hk/elections/dcelect99/cg12.htm |archivedate=2011-12-29 }}</ref>，其中有7,482位是選民<ref>{{cite web |url=http://www.elections.gov.hk/elections/dcelect99/cg12.htm |title=存档副本 |accessdate=2012-04-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20111229025617/http://www.elections.gov.hk/elections/dcelect99/cg12.htm |archivedate=2011-12-29 }}</ref>。今次由[[民主建港協進聯盟|民建聯]][[龔栢祥|龔栢祥]]與[[民主黨_(香港)|民主黨]][[楊健德|楊健德]]爭奪，最後龔以2,782票大比數擊敗取得864票的楊健德勝出。

[[2003年香港區議會選舉|2003年香港區議會選舉]]，當時人口估算約17,022人<ref>{{cite web |url=http://www.elections.gov.hk/elections/dc2003/tc_chi/summary/summary_e.html |title=存档副本 |accessdate=2012-04-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20111108185626/http://www.elections.gov.hk/elections/dc2003/tc_chi/summary/summary_e.html |archivedate=2011-11-08 }}</ref>，其中有9,325位是選民<ref>{{cite web |url=http://www.elections.gov.hk/elections/dc2003/tc_chi/eb/eb_num_e.html |title=存档副本 |accessdate=2012-04-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20110115012307/http://www.elections.gov.hk/elections/dc2003/tc_chi/eb/eb_num_e.html |archivedate=2011-01-15 }}</ref>。今次[[龔栢祥|龔栢祥]]自動當選。

[[2007年香港區議會選舉|2007年香港區議會選舉]]，當時人口估算約15,840人<ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/summaryc.html |title=存档副本 |accessdate=2015-12-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20080501203936/http://www.elections.gov.hk/dc2007/chi/summaryc.html |archivedate=2008-05-01 }}</ref>，其中有9,128位是選民<ref>{{cite web |url=http://www.elections.gov.hk/dc2007/chi/num_electors_eastern.html |title=存档副本 |accessdate=2015-12-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20150621034533/http://www.elections.gov.hk/dc2007/chi/num_electors_eastern.html |archivedate=2015-06-21 }}</ref>，龔栢祥與[[泛民區選聯盟|泛民區選聯盟]][[黃靄詩|黃靄詩]]爭奪，龔栢祥以2,335票勝出。

[[2011年香港區議會選舉|2011年香港區議會選舉]]，當時人口估算約15,135人<ref>[http://www.elections.gov.hk/dc2011/chi/summaryc.html]</ref>，龔栢祥自動當選。

[[2015年香港區議會選舉|2015年香港區議會選舉]]，龔栢祥爭取連任，受到住在同區的公民黨成員[[吳彥強|吳彥強]]挑戰，最終龔栢祥以約六成得票勝出。
