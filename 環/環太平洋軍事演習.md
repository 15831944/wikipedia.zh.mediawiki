== 背景 ==
{{see|太平洋戰爭|珍珠港事件}}

== 三個階段 ==
[[File:120727-N-VD564-015_-_Maritime_Forces_RIMPAC_2012.jpg|缩略图]]
環太平洋軍事演習分為三個階段<ref name="The Department of National Defence and the Canadian Armed Forces" />
#[[港口|港口]]（Harbour）：目的是建立參與[[國家|國家]]的[[專業|專業]]與[[個人|個人]]之間，彼此的關聯性。人員才有機會面對面了解簡要概況、訓練以及詳細地[[規劃|規劃]]。
#部隊整合（Force Integration Training）：它可以鍛鍊每個國家在一個強大的跨國指揮和控制環境的執行力。
#自由發揮（Free Play Phases）：自由發揮階段測試這些[[技能|技能]]，[[指揮官|指揮官]]和其下屬單位可能面臨[[環太平洋地區|環太平洋地區]]的[[真實|真實]]情況，涵蓋來自空中、水面、水下與陸地的威脅等，具有挑戰性的全面作戰。

== 2012年 ==

2012年，[[俄羅斯太平洋艦隊|俄羅斯太平洋艦隊]]首次參加環太平洋演習。<ref name="Russia 2012" />

[[網易|網易]]〈163軍情觀察室〉寫到，由美国主导的“环太平洋”军演本是[[冷战|冷战]]时期的产物，参演国基本都是美国的盟国，如今美国仍以各种名目组织这样大规模的演习，除提高各国的海上协同作战能力，确保美国及盟国的海上通道畅通无阻外，还意在展示武力，对整个亚太地区的军事施压，“防备亚太地区崛起的新兴军事力量”。在新增加的参演国名单中，[[俄罗斯|俄罗斯]]格外惹人注意，因为从1971年开始发起的该系列演习原来就是针对[[苏联|苏联]][[太平洋艦隊_(俄羅斯)|太平洋舰队]]。苏联解体后，演习改为两年举行一次，俄曾多次派遣军事观察员观摩演习。“环太平洋2012”联合军演的主题是“能力、适应、伙伴”。 在近年美国亚太兵力重新部署等重大军事战略调整中，“[[伙伴|伙伴]]”一词出现的[[频率|频率]]特别高。囊中羞涩的美国希望从容易遭受攻击的一线基地中撤出，让盟国军队更多地分担任务，减轻美军压力，同时这项举措也将有助于美国节省军事开支。<ref name="網易軍事">{{Cite web |url = http://war.163.com/13/0702/13/92PGTPCF00014J0G.html |title = 中国参加环太军演：政治意义大于军事意义 |author = 李小建 |date =  |publisher = 网易 |language = zh-cn |accessdate = 2016-06-10 |quote = 近年来，“环太平洋”演习的规模呈逐渐扩大的势头。从2006年的8个国家参与扩展到2012年的22个国家。其中，俄罗斯、印度、墨西哥、菲律宾、新西兰、挪威和汤加等国是首次参加2012年的演习。而孟加拉、文莱、巴西、越南、柬埔寨、巴布亚新几内亚和斯里兰卡等国也在2012年作为观察员参加演习，其余国家则派遣了部队实质性参与演习。 }}</ref>

:同上評論，李小建認為，以前的历次演习中，基本都突出美国的“[[领导|领导]]”色彩，基本是美国指挥和主导了演习中的各部分项目，而到了2012年的时候，演习除美海军第三舰队司令比曼中将担任联合特遣队指挥官外，其他的重要职位大部分由非美军将领担任。其中，日本海上自卫队海將補（少將）担任联合特遣队副指挥官；海军分遣队指挥官由[[澳大利亚皇家海军|澳大利亚皇家海军]]准将担任；空中分遣队指挥官由[[加拿大皇家空军|加拿大皇家空军]]准将担任；陆上分遣队司令则由[[美國海军陆战队|美國海军陆战队]]准将担任。至于美国在2012年邀请俄罗斯参演，其目的是缓和美俄在反导上的矛盾，向俄罗斯表明，美国没有将俄罗斯当做敌人，以增加双方的互信。俄罗斯则也想通过与美国和[[太平洋地区|太平洋地区]]多国之间的互动，加强[[军事|军事]]、[[政治|政治]]、[[经济|经济]]等联系，为俄罗斯未来的发展提供一个更为有利的地缘政治环境，从而保证俄罗斯的利益不受到侵蚀和损害。<ref name="網易軍事" />

== 2014年 ==
[[File:Multinational_task_force_in_the_24th_biennial_Rim_of_the_Pacific.jpg|thumb]]
財團法人國家政策研究基金會《國改研究報告》提到，相較於歷屆的環太平洋多國聯合軍演，這次演習設置科目內容更加廣泛，對於應付[[恐怖攻擊|恐怖攻擊]]、[[自然災害|自然災害]]，以及維護局部區域穩定等，共同威脅領域的比重有所增加；同時，這次演習增列更多[[人道主義|人道主義]]救助內容；此外，2014年環太平洋多國聯合海上軍演，是不針對任何第三方的海上聯合軍演，也是環太平洋國家海軍深入合作的一次嘗試，其目標在於提高參演各國海軍聯合行動，以及共同應對海上威脅的能力。<ref>{{Cite web |url = http://www.npf.org.tw/2/13969 |title = 中共首度參與2014年環太平洋軍演研析 |author = 曾復生 |date = 2014-08-12 |publisher = 財團法人國家政策研究基金會 |language = zh-tw |accessdate = 2016-06-10 |quote = 另美國與菲律賓則是選擇在環太平洋聯合軍演期間，開始在南中國海進行「聯合海上戰備與訓練」。菲國目前與中共在此海域陷入領海主權爭端，當美菲簽署新軍事協議後，菲國將配合美國「睡蓮計劃」，建立可信的海上防禦能量。 }}</ref>

[[BBC中文網|BBC中文網]]報導，[[中国人民解放军海军|中国人民解放军海军]]共派出中華神盾的 052C 型 171 [[海口號飛彈驅逐艦|海口號飛彈驅逐艦]]、054A 型 575 [[岳陽號飛彈護衛艦|岳陽號飛彈護衛艦]]、886 [[903型綜合補給艦|千島湖號綜合補給艦]]，和[[920型醫院船|和平方舟號醫院船]]，同時參演的還有隨艦搭載的兩架直升機，1個潛水分隊、1個特戰分隊、1個醫療分隊，共計1100餘名官兵參演，是僅次於美國規模排列第二的隊伍。有[[中國|中國]]學者認為，中國軍方首次參加「環太平洋」軍演不僅對缺乏現代作戰[[經驗|經驗]]的[[中國海軍|中國海軍]]來說是一次難得的觀摩、學習和鍛煉機會，也是中美兩軍相互展示軍事力量透明善意姿態，有利緩解近來比較緊張的中美兩國和兩軍關係<ref name="BBC Chinese" />。

日本共同社報導，中國與包括[[日本|日本]]、[[韓國|韓國]]和[[印度|印度]]在內的23個國家一同參加演習，但中國海軍和[[日本海上自衛隊|日本海上自衛隊]]沒有一起[[訓練|訓練]]的計劃。海上自衛隊幕僚長（參謀長）{{Link-ja|河野克俊|河野克俊}}表示，雖然日本與[[中國|中國]]的雙邊關係沒有進展，但非常歡迎中國加入多邊框架。據報導，日本將派出[[伊勢號護衛艦|伊勢號護衛艦]]、[[金剛級護衛艦|宙斯盾艦霧島號]]，以及三架[[P-3獵戶座海上巡邏機|P-3獵戶座海上巡邏機]]，參予演集的[[海上自衛隊|海上自衛隊]]隊員約740人。日本在1980年首次參加該軍演<ref name="BBC Chinese" />。

== 2016年 ==
[[File:RIMPAC_Participant_Nations.png|thumb]]
在2015年12月，美國國防部長[[卡特|卡特]]和印度國防部長{{Link-en|馬諾哈爾·帕里卡爾|Manohar Parrikar}}宣布，印度將參加環太2016年军演<ref>[http://indianexpress.com/article/india/india-news-india/india-to-participate-in-exercises-rimpac-and-red-flag-in-us-next-year/ India to participate in world’s largest maritime warfare exercise in US next year]</ref>。儘管中美两国在中國南海局勢緊張，中國仍将参加環太2016年军演<ref>[https://news.usni.org/2016/04/18/secdef-carter-china-still-invited-to-rimpac-2016-despite-south-china-sea-tension SECDEF Carter: China Still Invited to RIMPAC 2016 Despite South China Sea Tension]</ref>。

據《[[法新社|法新社]]》報導，中國首次參加環太軍演以降低不信任感後，中國卻積極經營[[南海|南海]]，再度觸動中美緊張關係，引發包括美國在內國際的譴責。中國被批評積極在[[南海|南海]]填海造島和興建軍事設施，導致[[中美關係|中美關係]]陷入緊張。美國軍方透露，現在正重新評估是否邀請中國參加今年太平洋地區大規模的海上聯合演習。[[美國國防部長|美國國防部長]]卡特說，中國獲得參加今年環太平洋軍演邀請。卡特沒有表示已經就此作出決定，只說美國將繼續就此展開評估。卡特指出美國的亞太戰略不是排除任何人，而是維持當地的安全架構，讓所有人參與進來。然而中國卻自我孤立，這也是為何所有夥伴均向我們靠攏。<ref>{{cite news |language = zh-tw |author1 = [http://trad.cn.rfi.fr/auteur/%E5%B0%8F%E5%B1%B1/ 小山] |author2 = 法廣RFI中文部 |url = http://www.appledaily.com.tw/realtimenews/article/new/20160323/823104/ |title = 美猶豫邀解放軍入環太平洋演習 |publisher = 蘋果日報 |pages =  |date = 2016-03-23 |accessdate = 2016-06-10 }}</ref>

《[[美國之音|美國之音]]》報導，2016年環太平洋軍演由美軍[[太平洋艦隊|太平洋艦隊]]主辦，[[美國海軍|美國海軍]]第三艦隊[[司令|司令]]{{Link-en|諾拉·W·泰森|Nora W. Tyson}}中將擔任聯合特遣隊指揮官。其他主要領導職位由[[加拿大|加拿大]]、[[日本|日本]]、[[澳大利亞|澳大利亞]]和[[新西蘭|新西蘭]]等國的[[軍官|軍官]]擔任。第25屆環太平洋軍演預計將於2016年6月30日至8月4日在[[夏威夷群島|夏威夷群島]]和[[加利福尼亞州|加利福尼亞州]]及其附近地區舉行。參加演習的各國軍隊將演練並展示軍隊的海上能力和靈活性，這些能力包括救災、海上安全合作、制海和復雜作戰等。實戰演練項目包括[[兩棲作戰|兩棲作戰]]、[[射擊|射擊]]、[[導彈|導彈]]、反潛和防空[[演習|演習]]等。<ref>{{Cite web |url = http://www.voacantonese.com/a/rimpac-20160531/3356770.html |title = 27國參加2016環太平洋軍事演習 |author =  |date = 2016-06-01 |publisher = 美國之音 |language = zh |accessdate = 2016-06-09 |quote = 參加今年演習的軍隊來自澳大利亞、巴西、汶萊、加拿大、智利、哥倫比亞、丹麥、法國、德國、印度、印度尼西亞、意大利、日本、馬來西亞、墨西哥、荷蘭、新西蘭、挪威、中國、秘魯、南韓、菲律賓、新加坡、泰國、湯加、英國和美國。 }}</ref>

2016年7月1日中國解放軍五艘軍艦從舟山啟航抵達[[夏威夷|夏威夷]]參演<ref>[http://udn.com/news/story/7331/1799690-%E7%92%B0%E5%A4%AA%E5%B9%B3%E6%B4%8B%E8%BB%8D%E6%BC%94-%E6%B5%B7%E8%BB%8D5%E8%89%A6%E6%8A%B5%E7%8F%8D%E7%8F%A0%E6%B8%AF 聯合報-2016環太平洋軍演 ] {{webarchive|url=https://web.archive.org/web/20160717033816/http://udn.com/news/story/7331/1799690-%E7%92%B0%E5%A4%AA%E5%B9%B3%E6%B4%8B%E8%BB%8D%E6%BC%94-%E6%B5%B7%E8%BB%8D5%E8%89%A6%E6%8A%B5%E7%8F%8D%E7%8F%A0%E6%B8%AF |date=2016-07-17 }}</ref>，規模僅次美國和[[加拿大|加拿大]]位居第三，戰艦為驅逐艦西安號和導彈護衛艦衡水號，另有和平方舟醫院船、綜合補給艦高郵湖、援潛救生船長島號，前往夏威夷途中與美國兩艘驅逐艦會合並聯合演習空中威脅應對，具備防空能力的中美4艘驅護艦，分區域擔負對空防禦任務。值得注意的是此次小型演習由西安號擔任[[旗艦|旗艦]]，指揮美國軍艦。環太軍演正式開始後中方不再參與高強度戰爭內容，只參與反海盜和人道救援科目，此次特點在於美國製造了一個美軍潛艦艙段的模型放於水下，由中方援潛救生船長島號放下小潛器在水下演練對接，確保有救援美軍遇難潛艇的能力。<ref>[http://tv.cctv.com/2016/07/01/VIDEOPXXGMlFrFcDiwQfQ1KE160701.shtml 央視-我海军参演“环太”]</ref>美國部分媒體指出此次軍演讓人看不懂，美國同時在南海領土爭端強硬表態一年多，最後卻又策畫這種演習無疑向亞太盟友發出混亂訊號；美國原本致力於傳達維持地區穩定和安全的戰略訊息正受到破壞，而中國卻得以通過參與環太軍演而贏得國際聲望。<ref>[http://www.chinatimes.com/newspapers/20160701000797-260301 中時- 美釋雙重訊號 允陸赴環太軍演]</ref>

== 2018年 ==
[[2018年|2018年]][[1月|1月]]美國仍有對中國發出環太平洋軍事演習邀請，但[[5月23日|5月23日]][[美国国防部|美国国防部]]正式聲明今年演習會將除去中國參與，理由是中國在[[南海爭議|南海爭議]]上的軍事擴張造成區域局勢不穩<ref>[https://www.politico.com/story/2018/05/23/us-china-naval-military-exercise-566721 U.S. kicks China out of military exercise]</ref>。對此中國表示此舉無助中美關係，並仍堅持認為南海為中國領土他們的作為美國無權說三道四<ref>{{Cite news |url=http://www.storm.mg/article/441401 |title=美國把中國踢出「環太平洋聯合軍演」中國回嗆：少拿南海軍事化說嘴 |last=風傳媒 |accessdate=2018-05-24}}</ref>。

2018年5月30日，[[中華民國|中華民國]][[國防部|國防部]]部長[[嚴德發|嚴德發]]在立法院備詢指出，一直都有向美方爭取參加環太平洋軍演，嚴德發表示現在是大好機會，但還是要看美方態度。<ref>[http://www.cna.com.tw/news/firstnews/201805300105-1.aspx 爭取參加環太平洋軍演 嚴德發：現在是大好機會 發稿時間：2018/05/30 12:02最新更新：2018/05/30 12:28 中央社記者劉麗榮台北30日電]</ref>

2018年6月27日，2018年度環太平洋軍事演習受邀參加的國家新增[[以色列|以色列]]、[[巴西|巴西]]、[[斯里蘭卡|斯里蘭卡]]與[[越南|越南]]等國，第一次參加該演習。<ref>[https://news.cts.com.tw/cts/international/201806/201806281929728.html 環太平洋軍演登場 越南首次參加 2018/06/28 19:50 王皓 編譯 華視新聞]</ref>
