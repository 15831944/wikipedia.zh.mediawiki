'''環狀線'''是[[臺北捷運|臺北捷運]]興建中的路線，又可細分為「南環段」、「第一階段」、「北環段」和「[[環狀線東環段|東環段]]<ref>{{Cite web |url=https://www.dorts.gov.taipei/cp.aspx?n=2353836DD875EA15 |title=捷運規劃中路網-環狀線東環段 |accessdate=2019-04-21 |work= |publisher=臺北市政府捷運工程局 |date=2019-03-05}}</ref>」四個路段。本路線採用[[中運量系統|中運量系統]]，其中[[十四張站|十四張站]]至[[新北產業園區站|新北產業園區站]]為高架路線，其他部分（十四張站東側、新北產業園區站北側第二階段路線）則為地下路線。目前僅新北產業園區—板橋—中和—大坪林的第一階段已動工，2019第四季通車後搭乘捷運往返新店、中和、板橋、新莊，不必再經過臺北市，縮短乘車時間並分散運量，減少轉乘站的擁擠。

==簡介==
環狀線最早可追溯至1980年代末，[[臺北市政府捷運工程局|臺北市政府捷運工程局]]規劃從新店至三重之'''捷運黃線'''構想。原係由臺北縣政府（今[[新北市政府|新北市政府]]）主辦，並以[[BOT模式|BOT]]方式辦理。惟經臺北縣政府辦理公告招商，因僅有一家團隊參與投件，經資格審查結果認定為不合格。後考量改由政府自建之工程品質與安全較執行BOT方式為佳，於2006年4月20日宣佈環狀線改採用政府自建，並交回臺北市政府捷運局辦理。工程分為兩階段，第一階段為[[新北產業園區站|新北產業園區站]] — [[大坪林站|大坪林站]]，於2009年興建，預定於2018年12月完工，隔年經交通部會勘後通車。第二階段即為南環段與北環段，計畫於第一階段通車後再展開發包與興建作業。而東側南北向捷運於2018年3月15日宣布仍與環狀線同屬一路線，但由原本的在[[中和區|中和區]]相交的Q字形路線，改為銜接回文山區動物園站的O字形路線。舊有[[南北線_(台北捷運)|南北線]]廢止，目前東側南北向捷運仍於可行性研究階段。

環狀線雖使用中運量系統，但未來並不會與現有[[文湖線|文湖線]]的中運量系統有軌道上的銜接（兩線交會處皆非位於同一水平面，且機電系統、路軌、列車皆不同）。而第一階段中與板南線交會的兩站皆為站外轉乘，東環段的永春站將會是與板南線唯一站內轉乘的車站。

==路線==
===南環段===
路線自[[文山區|文山區]][[動物園站_(臺北)|動物園站]]起以地下路線方式經[[國立政治大學|國立政治大學]]校門前三角地宿舍區、指南路後續沿國立政治大學校內四維道，再從木新路、秀明路、[[木柵路|木柵路]]，接著穿越[[景美溪|景美溪]]至[[新店區|新店區]][[大坪林站|大坪林站]]後即轉為高架路線，約5.73公里。

===第一階段===
又稱為「中環段<ref>{{Cite web|url=https://www.gov.taipei/News_Content.aspx?n=F0DDAF49B89E9413&sms=72544237BBE4C5F6&s=D42EB689D7F46C50|title=臺北市政府全球資訊網-市府新聞稿-出席「2018臺北3C電器空調影音大展」 柯文哲：未來電器發展應朝向智慧方便化及節能減碳 促進循環經濟減少垃圾量|accessdate=2019-02-20|author=|date=|publisher=}}</ref>」或「西環段<ref>{{Cite web|url=https://www.motc.gov.tw/uploaddowndoc?file=yearbook%2F106%2F106YB13/106YB130202.pdf&filedisplay=106YB130202.pdf&flag=doc|title=第十三篇都市交通 - 中華民國交通部|accessdate=2019-02-20|author=|date=|publisher=}}</ref>」。興建中，預計2019年年中完工，第四季通車營運<ref>{{Cite web|url=https://www.mirrormedia.mg/story/20190212edi009|title=北捷環狀線第一期　今年第4季通車 |accessdate=2019-02-15 |work= |publisher=鏡週刊 |date=2019-02-12}}</ref>。<br>路線自大坪林站穿越[[新店溪|新店溪]]，經[[中和區|中和區]]景平路、中山路及[[板橋區_(新北市)|板橋區]][[板橋車站_(臺灣)|板橋車站]]，並越過[[大漢溪|大漢溪]]至[[新莊區|新莊區]]思源路和[[新北產業園區|新北產業園區]]，共15.4公里。<ref>{{Cite web |url=http://www.dorts.gov.taipei/cp.aspx?n=EBC5FB619CFBEA9A |title=環狀線第一階段路線說明 |accessdate=2018-02-01 |work= |publisher=臺北市政府捷運工程局 |date=2017-09-29}}</ref><ref>{{Cite web |url=http://www.dorts.gov.taipei/cp.aspx?n=82392DF40EB914B8 |title=環狀線北環段及南環段路線說明 |accessdate=2018-02-01 |work= |publisher=臺北市政府捷運工程局 |date=2017-09-29}}</ref>[https://www.google.com/maps/d/embed?mid=1ys6pfVqMzEc2SjcLP0ryfZaf0zs&ll=25.02170779635746,121.50089319999995&z=12 各站]依序為：

*Y12 中和站：位於新北市中和區景平路、中山路二段路口，為環狀線與後續捷運萬大線線交會轉乘站。

===北環段===
路線自新北產業園區起轉為地下後經[[五股區|五股區]][[更寮_(新北市)|更寮地區]]、[[蘆洲區|蘆洲區]][[徐匯中學站|徐匯中學站]]、[[三重區|三重區]]集賢路並穿過[[淡水河|淡水河]]，經過[[士林區|士林區]]中正路，至[[國立故宮博物院|國立故宮博物院]]附近轉向，穿越雞南山[[自強隧道_(台北市)|自強隧道]]東側，抵達[[劍南路站|劍南路站]]，約14.93公里。

===東環段===
{{main|環狀線東環段}}
原為[[南北線_(台北捷運)|南北線]]計畫，該計畫取消後另案規劃，用來補足臺北市東側缺乏南北向捷運系統的缺口，並分別銜接環狀線南環段及北環段。根據[[臺北市政府捷運工程局|臺北市政府捷運工程局]]所公布的規劃路線，目前共規劃有兩方案，細部路線規劃如下：
*瑞光路案：自劍南路站往南轉至敬業三路、東轉至樂群二路、瑞光路進入[[內湖科技園區|內湖科技園區]]。再沿著陽光街、舊宗路過成美橋到達[[松山車站_(台灣)|松山站]]，並往南行經[[永春站_(台北市)|永春站]]、[[象山站|象山站]]，最後穿越象山回到[[動物園站_(臺北市)|動物園站]]，約13.12公里。
*堤頂大道案：自劍南路站往南轉至敬業三路、東轉至樂群二路、堤頂大道進入[[內湖科技園區|內湖科技園區]]東側。再沿著舊宗路過成美橋到達[[松山車站_(台灣)|松山站]]，並往南行經[[永春站_(台北市)|永春站]]、[[象山站|象山站]]，最後穿越象山回到[[動物園站_(臺北市)|動物園站]]，約12.23公里。

*1987年2月：臺北縣政府委託中華民國都市計畫學會辦理「臺北縣綜合發展計畫」，提出連結六縣轄市之捷運路線。後將研究結果送交臺北市政府捷運工程局作為參考。<ref name=Circle_Line_BOT>{{Cite book|title=民間參與台北捷運系統環狀線之可行性研究|author=邱穀工程顧問有限公司|date=2004年12月}}</ref>
*1988年5月：臺北市政府捷運工程局在評估遠期路網時，將臺北縣政府的結論納入其中。<ref name=Circle_Line_BOT/>
*1991年3月13日：可行性研究呈交中央審查，但未獲核定。<ref name=Circle_Line_BOT/>
;台北縣政府主導
*1991年6月1日：台北縣政府委託[[國立交通大學|國立交通大學]]進行「臺北縣中運量捷運系統發展計畫之研究」。<ref name=Circle_Line_BOT/>
*其後經過「[[台灣省政府|省]][[台北市政府|市]]協調會報」及交通部多次協商，裁示「台北市政府為主辦單位，台北縣政府配合參與規劃」。<ref name=Circle_Line_BOT/>
*1992年12月3日：台北縣政府召開「研商本縣中運量環狀線整合暨進入板橋新站路線共構案」會議，會議決議板橋段改道，與[[板橋車站|板橋新站]]共構。<ref name=Circle_Line_BOT/>
*1993年1月：經過台北市捷運局修訂，縣府提出《捷運系統環狀線規劃報告書》。<ref name=Circle_Line_BOT/>
*1999年5月29日：台北縣政府希望環狀線後續作業其為主導機關，交通部於9月2日同意由臺北縣政府主辦後續相關業務。<ref name=Circle_Line_BOT/>
*2003年2月17日：[[行政院環境保護署|環保署]]通過全線[[环境影响评价|環評]]審查。
*2004年3月26日：行政院核定第一階段路線。同年12月20日，行政院核定財務計畫並採用BOT方式辦理。
*2005年3月2日：交通部同意[[臺北縣政府|臺北縣政府]]擔任後續BOT招商及建設主管機關。
*2005年8月15日：公告招商，截止收件後，計有一申請人參與投標，惟經臺北縣政府資格審後，不合格。
*2006年4月20日：改採用政府自建方式辦理。
;台北市政府捷運工程局興建
*2006年7月17日：臺北縣政府與[[臺北市政府|臺北市政府]]達成協議，主管機關改為[[臺北市政府捷運工程局|臺北市政府捷運工程局]]，並報請[[中華民國交通部|交通部]]同意。
*2006年11月8日：交通部同意由臺北市政府擔任後續捷運建設及土地開發地方主管機關。
*2006年11月24日：臺北縣、市政府簽訂行政契約。由臺北市政府捷運工程局承接環狀線工程，完工後交由[[臺北大眾捷運股份有限公司|臺北大眾捷運股份有限公司]]負責營運。
*2007年11月28日：開始進行前置作業。
*2008年10月2日：第一階段14站名正式公佈。
*2009年3月2日：[[義大利|義大利]]商安薩爾多百瑞達（AnsaldoBreda）得標，由屬於同集團的兩家公司組成安薩爾多團隊獲得機電系統標案，採用鋼輪[[鋼軌|鋼軌]]系統，日後這兩家安薩爾多公司被收購，成為電機製造商[[日立製作所|日立公司]]旗下企業，2015年安薩爾多百瑞達亦隨收購而更名為「[[日立軌道義大利|日立軌道義大利]]」公司。
*2009年4月7日：中環段車站細部設計開始。
*2011年12月19日：[[台北捷運環狀線先導公車|臺北捷運環狀線先導公車]]開始營運。行駛中環段路線。
*2015年11月8日：南北環段環評通過。<ref>{{Cite web|url=http://www.traffic.ntpc.gov.tw/_file/1559/SG/22766/D.html|title=環狀線第一階段施工現況|accessdate=2013-10-24|date=2012-11-02|work=|publisher=新北市政府交通局|deadurl=yes|archiveurl=https://web.archive.org/web/20131022210231/http://www.traffic.ntpc.gov.tw/_file/1559/SG/22766/D.html|archivedate=2013-10-22}}</ref>
*2018年3月15日：宣布東側南北向捷運仍與環狀線同屬一路線，從此環狀線成為名副其實的環狀路線。

== 電聯車 ==
[[File:TRTC_Circular_Line_EMU_train_by_ntpcdorts.jpg|thumb]]
{{main|臺北捷運環狀線電聯車}}

'''臺北捷運環狀線電聯車'''是一款屬於[[臺北捷運|臺北捷運]]的[[中型鐵路系統|中運量]][[動力分散式|動力分散式電聯車]]，由義大利商（[[日立軌道義大利|日立軌道義大利]]）與[[台灣車輛|台灣車輛]]承造，生產17列列車，是台灣繼[[台鐵EMU300型電聯車|台鐵EMU300型電聯車]]之後再度購入由[[義大利|義大利]]製造的列車。

==爭議==
位於環狀線南機廠的劉氏宗祠、利記公厝、悠記古厝、厚記古厝、陳氏古厝等有歷史的建築物（均無法定文資身份）都被拆除，引起文化保存與都市開發的爭議。
