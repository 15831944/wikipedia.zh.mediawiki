'''環河路'''是[[台灣|台灣]][[新北市|新北市]][[新店區|新店區]]沿[[新店溪|新店溪]]右岸興建的一條道路。與其共線的新店環河快速道路是新店地區主要的快速道路之一，大致呈南北向，北接[[水源快速道路|水源快速道路]]，全線皆為平面道路，大部分路段禁行機車。在[[秀朗橋|秀朗橋]]與[[安坑交流道|安坑交流道]]段與[[北100線|北100線]]舊線平行。

== 行經行政區域 ==
全線都在新北市新店區境內。

== 歷史 ==

== 快速道路 ==
新店環河快速道路全長4.5公里，北與[[水源快速道路|水源快速道路]]直接，南邊抵達新店碧潭大橋。
