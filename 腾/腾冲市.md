'''腾冲市'''是位于中国[[云南省|云南省]]西部中缅边境的一个[[县级市|县级市]]，腾冲市由云南省直辖、[[保山市|保山市]]代管，治[[腾越镇|腾越镇]]。以中缅[[边境贸易|边境贸易]]、著名侨乡，以及[[二战|二战]][[中缅印战区|中缅印战区]]的主战场著称。腾冲被列为云南省[[历史文化名城|历史文化名城]]。

== 历史 ==
*[[諸葛亮|諸葛亮]]治平定孟獲等部落首領。
*南北朝時期由南朝閹縻管理。
*大理國段氏脫離唐朝自立。
*元朝征服大理，納入大理世襲總管管轄。
*[[明朝|明朝]]在此筑城屯兵。
*明末[[徐霞客|徐霞客]]一生中最后一次游历至此。
*19世纪末，腾冲成为清朝雲南省与时缅甸、印度（英屬東印度公司）贸易的重要口岸。
*1911年，[[同盟会|同盟会]]成员在[[腾越厅|腾越厅]]发动[[腾越起义|腾越起义]]，成为[[辛亥革命|辛亥革命]]在云南的先声。
*1913年，废腾越厅，置[[腾冲县|腾冲县]]，属[[雲南省_(中華民國)|云南省]]，其后为[[迤西道|迤西道]]、[[滇西道|滇西道]]、[[腾越道|腾越道]][[治所|治所]]。
*1942年，腾冲县成为滇西抗日抗战的主战场。
*1942年，属第六行政督察区，后为第十二、第十一行政督察区，腾冲县皆为督察区[[驻地|驻地]]。
*1944年9月14日，[[中国远征军|中国远征军]]经过127天的血战，全歼日军。腾冲县成为全国沦陷区中，[[国民革命军|国民革命军]]第一个彻底光复的[[县城|县城]]{{noteTag|1=抗日战争期间，一些县城被中国军队收复后，再度沦陷。1944年8月30日、9月2日，[[八路军|八路军]][[胶东军区|胶东军区]]东海军分区部队收复[[文登县|文登县]]、[[荣成县|荣成县]]两县县城，未再度沦陷，故是中国军队最早彻底收复的县城。}}。
*1949年底，[[第二次国共内战|第二次国共内战]]战事进入尾声。12月，[[卢汉云南起义|卢汉云南起义]]。随云南全省被[[中华人民共和国政府|中华人民共和国政府]]掌控。1950年后，腾冲县历属[[保山专区|保山专区]]、[[德宏傣族景颇族自治州|德宏傣族景颇族自治州]]、保山专区、[[保山地区|保山地区]]、[[保山市|保山市]]。
*1980年代以后，腾冲再度成为云南的第二大贸易口岸，特别以[[翡翠|翡翠]]贸易为大宗。
*2010年，在腾冲县清水乡驼峰村重建的[[驼峰机场|驼峰机场]]，就是为了纪念[[二战|二战]]时期中国和[[盟军|盟军]]主要的空中通道“[[驼峰航线|驼峰航线]]”。
*2015年，经[[中华人民共和国民政部|民政部]]批准，撤销腾冲县设立腾冲市。<ref>{{cite news|title=民政部批准云南省撤销腾冲县设立腾冲市|url=http://www.tengchong.gov.cn/info/1012/7135.htm|newspaper=县民政局|date=2015年8月5日}}</ref>

== 地理 ==
腾冲的国境线长达148.7公里，从腾冲到缅甸[[克钦邦|克钦邦]]首府[[密支那|密支那]]217公里。腾冲县处于横断山西南端。腾冲位于[[高黎贡山|高黎贡山]]西麓，气候与植物垂直分布明显，气候类型包括从[[热带季风气候|热带季风气候]]到[[高山气候|高山气候]]的多种类型。因此不仅森林覆盖率高，而且物种极为丰富。

=== 气候 ===
腾冲属高原型[[亚热带季风气候|亚热带季风气候]]，一年分为雨季（5－10月）和旱季（11－4月），冬无严寒、夏无酷暑。最冷月（1月）平均气温7.8℃，最热月（8月）平均气温19.8℃。冬季干燥，日照时间长，夏季多雨潮湿。

== 人口民族 ==
2010年，全县普查时点常住人口为644765人。全县共有家庭户151771户，家庭户人口为619430人，平均每个家庭户的人口为4.08人，比2000年第五次全国人口普查的4.26人减少了0.18人。常住人口中，居住在城镇的人口为135318人，占20.99%；居住在乡村的人口为509447人，占79.01%。与2000年第五次全国人口普查相比，城镇人口增加了85725人，乡村人口减少了34885人，城镇人口比重上升12.64个百分点。

2014年，总人口（常住人口）65.99万人，其中：城镇人口26.07万人，占总人口的39.5%，乡村人口39.92万人，占60.5%。人口出生率为11.59%，死亡率5.83%，自然增长率5.76%。

==文化==
===小吃===
腾冲有下列著名的小吃。

; 腾冲饵丝
腾冲饵丝是用腾冲特产[[硬米|硬米]]制作的，它有丝状和卷成块的 ，丝状的一般是汤了再吃，把水煮开后，加上特制好的汤（汤可以是用[[骨头|骨头]]熬的，带点淡淡的甜味：也可以用[[干腌菜|干腌菜]]煮汤，干腌菜做的汤清谈带点酸味，不油腻），再加上一些肉末、[[葱|葱]]花、[[蒜|蒜]]油，制作简单不费时。卷成块的一般用来炒着吃，也就是'''大救驾'''。不过炒也可以，汤也可。

; 大救驾炒饵块
大救驾就是把饵块切成小块加上[[番茄|番茄]]、葱、鸡蛋、少量的蒜和一些常用的香料一起炒，不过炒饵块要乘热吃，冷了会变硬，口感会不好。

; 青龙过海汤
青龙过海汤是用干腌菜、胡辣子、葱、[[薑|薑]]、蒜加开水冲制而成：如果用干腌菜煮汤还可以加上白[[豆腐|豆腐]]、[[棕苞米|棕苞米]]，味道会更好。

; 撒撇
撒撇为傣族特色小吃之一，其主料是一种特制的细米线，他与[[凉粉|凉粉]]有点相似， 但味道截然不同，主要是在蘸水上。撒撇有酸撒和苦撒。酸撒是用[[柠檬|柠檬]]水加[[韭菜|韭菜]]、[[芝麻油|芝麻油]]、[[辣子油|辣子油]]还有其它香料调制而成，苦撒与酸撒的差别就是用[[牛苦水|牛苦水]]代替柠檬，苦撒可以去热。一般米线上加一些牛干巴、牛肺。因其解暑去热故而在云南十分流行，但是并非腾冲所独有。
