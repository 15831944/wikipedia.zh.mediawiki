'''雕塑'''是一種[[造型艺术|造型艺术]]，以物质材料和手段制作的[[三维|三维]]空间形象的[[视觉艺术|视觉艺术]]。雕塑最早主要是使用[[雕|雕]]（通过减除材料来造型）及[[塑|塑]]（通过叠加材料来造型）的方式，在[[石|石]]、[[金属|金属]]、[[木|木]]、[[陶瓷|陶瓷]]等材料來創作，不過[[現代主義|現代主義]]的雕塑，在材料及創造手法上都有很高的自由度，可以利用雕、[[焊接|焊接]]、[[模塑|模塑]]或[[鑄造|鑄造]]的方式，在各種不同的材質上進行創作。

{{link-en|石雕|Stone sculpture}}比其他材質易腐壞的雕塑更可以在長時間後保存下來，古文化所留下的雕塑也多半以石雕及[[陶藝|陶藝]]為主。同時期也有很多傳統的木雕，但很多都已經腐朽無法保存。古文化的雕塑許多曾都有亮麗的顏色，但現在多半已經褪色<ref name="artmuseums.harvard.edu">{{Cite web 
 |url= http://www.colourlovers.com/blog/2008/06/16/gods-in-color-painted-sculpture-of-classical-antiquity
 |title= Gods in Color: Painted Sculpture of Classical Antiquity
 |accessdate=2013-07-10 
 |publisher=The Arthur M. Sackler Museum
}}</ref>。

在很多文化中，雕塑是宗教崇拜的中心，一直到最近幾世紀之前，大型雕塑都還非常昂貴，非一般個人有能力可能負擔，因此大型雕塑多半是宗教或政治的產物。像古地中海、印度、中國、南美及非洲的古文化都留下許多的雕塑。

[[西方文化|西方文化]]中的雕塑起源於[[古希臘|古希臘]]，許多希臘的雕塑家在[[古典希臘時期|古典希臘時期]]創作了許多傑出的雕塑品。在中古世紀，[[哥特式艺术|哥特式]]雕塑表現了基督教中的痛苦及激情，文藝復興時期崇尚古典作品，像[[米开朗基罗|米开朗基罗]]的[[大衛像|大衛像]]就是當時著名的作品<ref>{{Cite web 
 |url=http://collection.sina.com.cn/jczs/20130221/1553104125.shtml 
 |title=米开朗基罗的雕塑《大卫》 
 |accessdate=2013-07-11 
 |work= 艺术中国网  
 |publisher=新浪收藏 
 |date=2013-02-21
}}</ref>。[[現代主義|現代主義]]雕塑較不著重傳統的工藝及人體的描繪，改用{{link-en|集合藝術|Assemblage (art)}}和{{link-en|拾得物|found objects}}作為其作品的呈現。

== 類型 ==
雕塑的簡單分類可以分為圆雕、[[浮雕|浮雕]]及透雕。圆雕類似塑像，指不附着在任何背景上、可以从各个角度欣赏的立体的雕塑，也因為欣賞者可能從任何一個角度欣赏作品，每一面都需要詳加規劃<REF>{{Cite web 
 |url=http://vr.theatre.ntu.edu.tw/fineart/chap12/chap12-02.htm
 |title=浮雕與圆雕
 |accessdate=2013-07-11 
 |publisher=視覺素養學習網
}}</ref>。浮雕至少有一部份是附着在背景上，可以依投射方式分類為淺浮雕及高浮雕<REF>{{Cite web 
 |url=http://www.mudiao5000.com/fu-diao/zs-fudiao.htm
 |title=什么是浮雕
 |accessdate=2013-07-11 
 |publisher=上海胜佰杰木雕厂
}}</ref>。當作品中有許多的事物要敘述時，用圆雕的方式不易呈現，一般會使用浮雕。在{{link-en|建築雕塑|Architectural sculpture}}及用小型雕塑裝飾其他物品（像[[陶藝|陶藝]]、金屬製品或[[珠寶|珠寶]]）時，也常會選擇浮雕。透雕又称为镂空雕，是界于圆雕和浮雕之间的一种雕塑。在浮雕的基础上，镂空其背景，有分為单面透雕和双面透雕。

另一種分類則是依其工法來分類，雕是將材料從作品上移除，塑則是利用將材料塑形的方式來完成作品。像[[鑄造|鑄造]]、[[沖壓|沖壓]]及[[模塑|模塑]]等方式都可以雕塑，而且可以一次製作多個相同的成品。

==材料和技術==
由於雕塑的[[材料|材料]]相當多種，有[[粘土|粘土]]、[[石|石]]、[[木|木]]、[[金属|金属]]、[[石膏|石膏]]、[[树脂|树脂]]、[[象牙|象牙]]、{{Link-en|英国复合土|Milliput}}等。遂發展出幾種類別：

* [[木雕|木雕]]：木雕的歷史相當悠久，早期的主題多半以宗教為核心，除雕塑[[宗教|宗教]]偶像之外，也用於祭壇、宗教場所的裝飾。在木造建築興盛的國家，木雕是隨著建築一同發展的。現代的木雕則有相當多元化的風貌，藝術家喜用抽象形來表現。此外，木雕也是印刷術發明的重要功臣，詳細請看[[木刻|木刻版畫]]
* {{link-en|石雕|Stone sculpture}}：石雕發展的地區跟適合雕刻的石頭產地是很有關係性的，產石發達的地區，石雕的歷史也相當悠久。過去石雕的作品常被用來當做花園擺飾、門口標誌、或是[[陵寢|陵寢]]建築裝飾等，基於石頭厚重（不易被偷竊）、雕刻細節不易因風吹雨打而破壞，石雕作品多以戶外環境為主。常見的石雕材料是[[大理石|大理石]]。
* [[泥塑|泥塑]]：材料會使用天然粘土或者現代的人工合成雕塑泥，可以作為獨立的藝術形式存在，也可以輔助其它的雕塑形式。
* [[冰雕|冰雕]]：是一種以冰為主要材料來雕刻的藝術形式，源自於法國，因為材料的可變性及其容易熔化，冰雕表現出了許多難點。
* [[沙雕|沙雕]]：起源於美國，利用沙灘上的沙進行雕塑，最常見的是小朋友在堆沙堡，但也有藝術家以沙雕為其創作的題材。
* [[金属雕塑|金属雕塑]]：大型的雕塑多半會以[[青銅|青銅]]或其他銅合金為其材質．利用鑄造的方式製作。有些金属雕塑需要用到金屬的[[延展性|延展性]]進行加工，其中又以金的延展性最好，有時會用在首飾上。
* [[陶艺|陶艺]]：是最早的雕塑之一，以[[粘土|粘土]]為其原料。

==維護==
[[File:Pollution_-_Damaged_by_acid_rain.jpg|thumb]]
有些雕塑對於環境的條件相當敏感，像[[溫度|溫度]]、[[濕度|濕度]]、暴露在光線及[[紫外線|紫外線]]下等。[[酸雨|酸雨]]也可能破壞特定的建築材料和石雕，其原因是因為雨中的[[硫酸|硫酸]]和石頭（石灰石、砂岩、大理石和花岗岩）中的鈣質反應，形成[[石膏|石膏]]而剥落。

許多當代的雕塑品會在公開場合展示，若雕塑品可以立刻被認出來，一般也不太容易被偷走。不過在二十世紀初期，金屬的價值高到一個程度，因此大型的青銅雕塑品可能會因為其金屬的價值因引來窃賊的注意，價值數百萬的雕塑品被偷走，融化成金屬後再賣出。賣出金屬的售價可能只佔原雕塑品價值中很小的比例<ref>[http://www.bbc.co.uk/news/uk-england-london-16266378 BBC: Barbara Hepworth sculpture stolen from Dulwich Park, 20 December 2011]. Example of theft of large bronze sculpture for the value of the metal.</ref>。
