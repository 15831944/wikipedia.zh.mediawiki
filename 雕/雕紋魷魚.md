'''雕紋魷魚'''（[[學名|學名]]：{{lang|la|'''''Glyphiteuthis'''''}}）是一[[屬|屬]]已[[滅絕|滅絕]]的[[頭足類|頭足類]]。生存在[[白堊紀|白堊紀]]的[[海洋|海洋]]中。<ref name="Fossilworks" />其學名是由「glyphi」（雕刻的）和「teuthis」（[[魷魚|魷魚]]）兩單字組成。由於經常發現外皮保存良好的雕紋魷魚[[化石|化石]]，且其外皮有著斜紋雕刻痕跡，因而得名。<ref name="libanotica">{{Cite web |url = http://blog.xuite.net/leeshaoki/fossil/325148676-%E9%BB%8E%E5%B7%B4%E5%AB%A9%E9%9B%95%E7%B4%8B%E9%AD%B7%E9%AD%9A%28Glyphiteuthis+libanotica%29 |title = 黎巴嫩雕紋魷魚(Glyphiteuthis libanotica) @ 小毛的化石收藏部落格 :: 隨意窩 Xuite日誌 |language = zh-hant |accessdate = 2018-06-15 }}</ref>
