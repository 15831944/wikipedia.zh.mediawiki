'''烂面烧饼'''（[[汉语拼音|拼音]]：'''lànmiàn shāobǐng'''；[[江淮官话|江淮官话]]镇江[[大港话|大港话]]：{{IPA|/lɛ̃ miɪ̃ sɔ piŋ/}}<ref name="丹徒县志">{{cite book|title=丹徒县志|url=https://books.google.com/books?id=SngoAAAAMAAJ|year=1993|publisher=江苏科学技术出版社|pages=849–850}}</ref>是起源于[[镇江|镇江]]东乡[[大港街道_(镇江市)|大港]]、[[大路镇_(镇江市)|大路]]、[[姚桥镇|姚桥]]一带，流行于[[镇江|镇江]]、[[扬州|扬州]]、[[常州|常州]]的面饼<ref name="烂面烧饼名满天下">{{cite web | url=http://www.yznews.com.cn/zjyz/2011-01/15/content_3716024.htm| title=烂面烧饼名满天下 | publisher=扬州晚报| date= 2011-01-15| accessdate= 2017-07-15}}</ref>，因为需要用和得比较烂的面制作，故名烂面烧饼。在常州又称作烂菜饼<ref name="唐鲁孙2013">{{cite book|author=唐鲁孙|title=故园情|url=https://books.google.com/books?id=4Et4oAEACAAJ|year=2013|publisher=广西师范大学出版社|isbn=978-7-5495-2705-2}}</ref>。

==制作方法==

先在容器中和面，水分比其他面食略多，使面更烂一点。完成后醒一段时间待用。

馅心通常使用绿色蔬菜，如[[菠菜|菠菜]]、[[苋菜|苋菜]]、[[小白菜|青菜]]、[[韭菜|韭菜]]、[[荠菜|荠菜]]等，也可以使用[[萝卜|萝卜]]丝{{noteTag|1=扬州话里的烂面烧饼通常指萝卜丝肉馅的烧饼<ref name="扬州烧饼">{{cite web | url=http://www.yznews.com.cn/zjyz/2011-05/07/content_3717931.htm| title=扬州烧饼 | publisher=扬州晚报| date= 2011-05-07| accessdate= 2017-07-15}}</ref>，别的馅的则称作馅名加烧饼，如苋菜烧饼<ref name="夏天搭粥的几种饼">{{cite web | url=http://www.yznews.com.cn/yzwb/html/2011-07/30/content_294482.htm| title=夏天搭粥的几种饼 | publisher=扬州晚报| date= 2011-07-30| accessdate= 2017-07-15}}</ref>。}}。除了绿色蔬菜以外也可以加少许猪肉末。但无论如何都要将馅切碎，煸熟后挤干多余的水。

包馅时，馅料要足，面团要压扁，直至从外面可以看见菜色。随后再放入锅内炕熟，可以放少许油，但切忌过量。

==习俗==

镇江<ref name="烂面烧饼名满天下"></ref>、扬州<ref name="扬州烧饼"></ref>、常州<ref name="常州月亮饼">{{cite web | url=http://www.cztv.tv/folder337/folder338/2015-09-27/139326.html?_hgOutLink=news/newsDetail&id=139326| title=中秋君“驾到” 老味道的常州月亮饼还记得吗？ | publisher=常州晚报| date= 2011-05-07| accessdate= 2017-07-15}}</ref>三地中秋均会制作此饼。而在起源地镇江东乡，小孩出生100天后，外婆家必须要送烂面烧饼来祝贺，俗称「拍屁股」<ref name="烂面烧饼名满天下"></ref>。
