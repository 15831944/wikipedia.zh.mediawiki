'''喀喇昆仑山脉'''（[[突厥语|突厥语]]意为“'''黑色岩山'''”/ 英文Karakoram）位于[[中华人民共和国|中华人民共和国]]、[[塔吉克斯坦|塔吉克斯坦]]、[[阿富汗|阿富汗]]、[[巴基斯坦|巴基斯坦]]和[[印度|印度]]等国[[边境|边境]]。宽度约为240公里，长度为800公里。平均海拔超过5500米。

== 地理 ==
喀喇昆侖山脈擁有8000米以上高峰4座，7500米以上高峰15座。喀喇昆侖山脈主山脊稱“[[大喀喇昆侖山|大喀喇昆侖山]]”，各段分別以大冰川命名，分為7個“[[慕士塔格山|慕士塔格山]]”（意為“冰山”），主山脊兩側的山地稱“[[小喀喇昆侖山|小喀喇昆侖山]]”。世界中、低緯度山地[[冰川|冰川]]長度超過50公里的共有8條，其中喀喇昆侖山佔6條，山脈的冰川總面積達1.86萬平方公里。

== 地质形成 ==
喀喇昆仑山属[[燕山褶皱系|燕山褶皱系]]。从[[白垩纪|白垩纪]]到[[第三纪|第三纪]]，[[印度次大陆|印度次大陆]]向北位移并与[[欧亚大陆|欧亚大陆]]碰撞抬升生成。岩石以[[花岗岩|花岗岩]]、[[片麻岩|片麻岩]]、[[结晶板岩|结晶板岩]]及[[千枚岩|千枚岩]]为主。喀喇昆仑山地震活动频繁。

== 氣候 ==
喀喇昆侖山[[垂直氣候|垂直氣候]]差異明顯。印度河上游一些海拔3000米以下的谷地，屬乾旱荒漠，而大冰積累區的降水卻十分豐富。大部分山區終年低溫，空氣較為稀薄，太陽輻射強烈，溫度變化較大，並常有強風。

== 动植物 ==
谷地中分布较多[[中亚植物|中亚植物]]，海拔较高处分布较多[[欧洲植被|欧洲植被]]。喀喇昆仑山的动物主要包括：[[牦牛|牦牛]]、[[雪豹|雪豹]]、[[西藏羚羊|西藏羚羊]]、[[土拨鼠|土拨鼠]]、[[野驴|野驴]]、[[短耳兔|短耳兔]]、[[鹧鸪|鹧鸪]]、[[岩雷鸟|岩雷鸟]]、[[雪鸡|西藏雪鸡]]、[[原鸽|原鸽]]等。
