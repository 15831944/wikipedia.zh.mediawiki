'''喀什市'''（{{lang-ug|'''قەشقەر'''}}，[[拉丁维文|拉丁维文]]：{{lang|en|Qeshqer}}，汉语音译'''喀什噶爾'''），是中国最西端的城市，为[[国家历史文化名城|国家历史文化名城]]之一。位于[[新疆维吾尔自治区|新疆维吾尔自治区]]西南部[[喀什地區|喀什地區]]，[[帕米尔高原|帕米尔高原]]和[[塔里木盆地|塔里木盆地]]的交接处。

喀什数个世纪以来一直是新疆南部的第一大城，是[[天山山脈|天山]]以南地区的政治、经济、文化、交通和军事中心。在[[中亚|中亚]]地区，喀什也是一座地位非常重要的城市。它是古代[[丝绸之路|丝绸之路]]北、中、南线的西端总交汇处，历来就是中西交通枢纽和商品集散地，贸易非常发达，有“[[巴扎|巴扎]]（集市）王国”之称，当地有俗语“五口通八国，一路连欧亚”。

== 历史 ==
喀什的经济和地理位置非常重要，历来均是各方势力争夺的对象，其归属曾发生过多次变化。早在6000年前的[[新石器时期|新石器时期]]，喀什便有人类生活。根据《[[穆天子传|穆天子传]]》的传说，[[周穆王|周穆王]]西征之时，曾经路过喀什附近，与当地部落进行了大规模的贸易。至迟在公元前2世纪，喀什地区出现[[疏勒|疏勒]]国，为西域城郭诸国之一。

前177年左右，疏勒受[[大月氏|大月氏]]压迫，被迫投靠[[匈奴|匈奴]]，为其属国。前128年，[[西汉|西汉]][[张骞|张骞]]出使[[西域|西域]]之后，经过反复争夺，前60年，疏勒正式被纳入了汉朝势力范围，受[[西域都护府|西域都护府]]节制。西汉末年，汉朝内乱，疏勒再次成为匈奴、[[大月氏|大月氏]]、[[莎车|莎车]]等势力争夺的战场。

公元74年开始，[[东汉|东汉]][[班超|班超]]又长期以这里作为根据地，与[[北匈奴|北匈奴]]争夺对西域的控制权。东汉势力进入[[三国|三国]]时代后，[[疏勒|疏勒]]在116年左右成为[[大月氏|大月氏]][[贵霜帝国|贵霜帝国]]和东汉的共同属国，定[[佛教|佛教]]为国教。国力强盛，称霸西域南部。

3世纪末年，疏勒内乱不断，323年，被[[龟兹|龟兹]]征服，成为附庸，后又先后从属于[[柔然|柔然]]、[[嚈哒|嚈哒]]和[[突厥|突厥]]。突厥人因当地[[玉|玉]]石贸易发达，将当地命名为Khasa，即“玉”之意。

675年，[[唐朝|唐朝]]在击溃[[西突厥|西突厥]]后，在此设疏勒都督府，为[[安西都护府|安西都护府]]的[[安西四镇|四镇]]之一。之后唐朝又与[[吐蕃|吐蕃]]经过多年争夺，最终控制了疏勒。754年，唐朝爆发[[安史之乱|安史之乱]]，安西都护府全军回迁，疏勒成为吐蕃和[[阿拉伯帝国|阿拉伯帝国]]争夺的对象。

840年左右，[[回鹘|回鹘]]西迁，成为天山以南的主体民族，疏勒也归附于其建立的[[黑汗王朝|黑汗王朝]]，并改用突厥语名称'''喀什'''。893年成为回鹘都城。

915年，其汗[[萨图克·博格拉汗|薩图克·布格拉汗]]皈依[[伊斯兰教|伊斯兰教]]，喀什地区逐渐伊斯兰化。960年，伊斯兰教成为[[喀喇汗王朝|喀喇汗王朝]]国教。之后，喀喇汗与信仰佛教的[[于阗|于阗]]进行了长期的宗教战争，一度惨败。但是在[[圣战|圣战]]的号召下，终于在1006年，喀喇汗国依靠来自于伊斯兰地区的支援军队征服了于阗。

1017年，[[契丹|契丹]][[辽朝|辽朝]]军队西征，引起喀喇汗国内乱，1041年，分裂成东西两部，喀什成为[[东喀喇汗国|东喀喇汗国]]首都。1134年，东喀喇汗国被[[西辽|西辽]]征服，成为其附庸。1215年，[[屈出律|屈出律]]篡夺西辽皇位，迁都喀什。1218年，被[[成吉思汗|成吉思汗]]灭亡。

1225年，为[[察合台汗国|察合台汗国]]的陪都，在[[阿里不哥|阿里不哥]]和[[忽必烈|忽必烈]]争夺汗位的战争中遭到毁灭性的打击。1306年，察合台汗国分裂，喀什成为东部汗国首都，1387年，东部汗国首都北迁[[伊犁|伊犁]]。

1392年，喀什被[[帖木尔帝国|帖木尔帝国]]占领，1432年被收复。1466年后陷入长期战乱，成為[[杜格拉特|杜格拉特]]氏族属地。1514年后属[[叶尔羌汗国|叶尔羌汗国]]。

[[File:Kashgar-apakh-hoja-d04.jpg|thumb]]]]
1680年，喀什的伊斯兰教[[白山派|白山派]]领袖[[阿帕克和卓|阿帕克和卓]]引导蒙古准噶尔汗[[噶尔丹|噶尔丹]]进军叶尔羌，叶尔羌汗国成为附属于准噶尔的傀儡政权。[[策妄阿拉布坦|策妄阿拉布坦]]继任准噶尔汗后，将天山以南置于[[准噶尔汗国|准噶尔汗国]]的直接统治之下。

1757年，[[清朝|清朝]]平定[[阿睦尔撒纳|阿睦尔撒纳]]，灭准噶尔汗国。定北将军[[班第|班第]]释放了被蒙古人囚禁在[[伊犁|伊犁]]的阿帕克和卓曾孙[[波羅尼都|波羅尼都]]和[[霍集占|霍集占]]。不久二人发动叛亂，史称[[大小和卓之亂|大小和卓之亂]]。1759年，清军进驻喀什噶尔，并在当地派驻了[[總理回疆事務參贊大臣|總理回疆事務參贊大臣]]，管理新疆[[回部|回部]]的军政要务

清朝灭亡后，1913年[[中华民国|中华民国]]设疏勒县。1933年，[[沙比提大毛拉|沙比提大毛拉]]、[[穆罕默德·伊敏|穆罕默德·伊敏]]在此成立[[東土耳其斯坦伊斯蘭共和國|東土耳其斯坦伊斯蘭共和國]]。

1952年[[中华人民共和国|中华人民共和国]]由疏勒县析置喀什市，1986年喀什被定为中国[[国家历史文化名城|国家历史文化名城]]。目前为[[喀什地区|喀什地区]]行政公署驻地。2010年5月，中央新疆工作会议上中央正式批准喀什设立经济特区，成為中國第五個经济特区，享受产业、税收、金融、土地、外贸等特殊扶持政策<ref>[http://news.sina.com.cn/o/2010-05-22/002117546649s.shtml 新疆喀什获批成立经济特区] {{webarchive|url=https://web.archive.org/web/20150823030757/http://news.sina.com.cn/o/2010-05-22/002117546649s.shtml |date=2015-08-23 }}，新浪网，2010年5月22日，于2010年5月21日查阅。</ref>。

喀什是[[东突厥斯坦|东突厥斯坦]]独立运动的策源地之一，也是各相关组织在中国境内的主要活动地点。至今，这里及附近地区仍时常发生[[恐怖主义|恐怖主义]]暴力事件。2008年8月4日，针对边防支队的[[2008年喀什袭击事件|恐怖袭击]]，造成至少16死16伤。

== 地理 ==
喀什以南为[[昆仑山|昆仑山]]脉，以北为天山南脉。喀什噶尔河穿流而过。全市年平均气温11.8 ℃，降水量60毫米，属典型的[[温带沙漠气候|温带沙漠气候]]。境内地势平坦，土地肥沃，绿洲农业发达，物产丰富。盛产各种瓜果，有红[[桃|桃]]、[[葡萄|葡萄]]、黄[[杏|杏]]、[[苹果|苹果]]、[[梨|梨]]、[[甜瓜|甜瓜]]、[[西瓜|西瓜]]、[[石榴|石榴]]、[[樱桃|樱桃]]、酸[[梅|梅]]、[[巴旦木|巴旦木]]、[[阿月浑子|阿月浑子]]等诸多品种，并衍生出了规模较大的食品和特色农副产品加工业。

== 文化和旅游 ==
[[File:Kashgar.jpg|thumb]]

喀什在古代曾是一个不同文化的交汇之所，出现过许多著名诗人、学者。目前[[维吾尔族|维吾尔族]]约占全市人口总数的八成左右，整座城市洋溢着浓郁的维吾尔族的风情。这里文艺兴盛，手工业发达。无论男女老幼，皆能歌善舞；当地流行的木卡姆舞曲，传遍了天山南北。特产[[陶器|陶器]]和[[铜器|铜器]]，历史悠久，頗負盛名。

喀什历史悠久，古迹众多。[[阿巴和加麻札|阿巴和加麻札（墓）]]是一处家族墓地，绘饰精美，曾被误传为[[香妃|香妃]]墓。而兴建于公元15世纪的[[艾提尕尔清真寺|艾提尕尔清真寺]]则是中国规模最大的[[清真寺|清真寺]]，是全疆的[[伊斯兰教|伊斯兰教]]活动中心和喀什的标志性建筑。此外，还有经教学院、佛教洞窟、古代遗址和多处古代墓葬。

== 经济 ==
[[File:Sonntagsmarkt_Kashgar.jpg|thumb]]
[[File:Kashgar_Scene.jpg|thumb]]
[[File:Kashgar-mezquita-id-kah-d01.jpg|thumb]]]]

喀什位于中国进入中亚、西亚等地的国际大通道上，向西开放的桥头堡和前沿，具有“五口通八国，一路连欧亚”的独特区位优势，2008年进出口总额16亿美元，以外经贸带动的转口、加工型经济正在形成。

经济以农业（主要作物为棉花，小麦）和畜牧业为主，辅有林业，加工业和旅游业。喀什林果总产占全疆三分之一，肉类总产占全疆四分之一，棉花产量居全疆第二位，有着重要的经济地位。冬小麦、大麦、玉米等粮食播种面积460.6万亩，产量158.29万吨。2003年棉花播种面积236.6万亩，产量22.5万吨，2008年收购价在4.0~5.2元/千克。

中国[[国家发改委|国家发改委]]2010年初发布《关于2009年西部大开发进展情况和2010年工作安排的通知》，表示2010年将推动广西[[东兴市|东兴市]]、云南[[瑞丽|瑞丽]]、新疆[[喀什|喀什]]、内蒙古[[满洲里|满洲里]]等为沿边[[经济特区|经济特区]]。

== 交通 ==
* 铁路：从[[乌鲁木齐|乌鲁木齐]]乘K9786/7556次列车由[[南疆铁路|南疆铁路]]往喀什。[[喀和铁路|喀和铁路]]亦於2010年底通车。[[中吉乌铁路|中吉乌铁路]]和[[中巴铁路|中巴铁路]]已列入国家铁道发展计划，即将开工建设。

* 公路：[[喀喇昆仑公路|喀喇昆仑公路]]北起喀什，经[[红其拉甫|红其拉甫]]连接[[巴基斯坦|巴基斯坦]]。

* 国道：[[314国道|314国道]]、[[315国道|315国道]]

* 航空：[[喀什机场|喀什机场]]（KHG），距离市区9公里，是全疆第二大機場，目前已开通喀什至北京、上海和广州经停乌鲁木齐的直航航班。同时也是国内较早开通面向中西亚、南亚的国际直航航线的机场。
