'''锯嘴蜂鸟'''在[[生物分类学|生物分类学]]上是[[蜂鸟科|蜂鸟科]][[隐蜂鸟亚科|隐蜂鸟亚科]]中的一个种。是[[巴西|巴西]]特有的鸟。因为鸟嘴有细小的锯齿而得名。

== 生态环境 ==
锯嘴蜂鸟生活在低洼地、[[大西洋|大西洋]]沿岸森林中。

== 分布地域 ==
主要分布于[[巴西|巴西]]东南部沿海的森林。

== 特征 ==
在蜂鸟的种类中，锯嘴蜂鸟的体型较大，而且[[体重|体重]]也较重，约有10[[克|克]]。它有一条淡黄色的[[眉毛|眉毛]]；下[[颚|颚]]和[[咽喉|咽喉]]为[[橙色|橙色]]，里面有一深色条纹从下颚一直延伸到[[胸部|胸部]]；[[尾羽|尾羽]]略圆，边缘略带紫色，尖部为淡黄色。

== 保护 ==
历史上，为进行采矿和园林种植，[[森林|森林]]大量被砍伐，威胁到锯嘴蜂鸟赖以生存的的低地森林。如今的主要威胁是城市化、工业化、农业扩大、殖民地化和相关道路的建设。
