== 历史 ==
机场曾用名包括Angeles Mesa Drive 機場（1928–1930）、聯合機場（1930–1934）、聯盟航空站（1934–1940）、洛克希德航空站（1940–1967），好莱坞-伯班克機場（1967–1978），伯班克-格倫代爾-帕薩迪納機場（1978–2003），鲍勃·霍普机场（2003-2016）<ref name=dailynews>{{Cite web |url=https://www.dailynews.com/2017/02/19/why-the-bob-hope-airport-name-is-changing/ |title=Why the Bob Hope Airport name is changing |date=2012-02-17 |publisher=Los Angeles Daily News}}{{en}}</ref>。2016年，机场管理方宣布将机场的[[商业名称|商业名称]]改回好莱坞-伯班克機場，但机场在法律文件等正式文书上的名称仍为'''鲍勃·霍普机场'''（'''Bob Hope Airport'''）。<ref name=dailynews/><ref>{{Cite web |url=http://www.epochtimes.com/gb/16/5/5/n7803492.htm |title=好莱坞伯班克更响亮？鲍勃霍普机场改名 |date=2016-05-05 |publisher=大纪元}}></ref>

== 對外交通 ==

=== 巴士 ===
洛杉矶[[洛杉矶县都会运输局|MTA公交]]94路与22路在机场附近的Hollywood Way与Thornton Ave交界处停靠。[[美鐵|美鐵]]也提供机场火車站去往[[贝克斯菲尔德_(加利福尼亚州)|贝克斯菲尔德]]的长途巴士。

=== 鐵路 ===
[[美鐵|美鐵]]与[[Metrolink|Metrolink]]伯班克机场南站連接本机场与加州各大城市。
