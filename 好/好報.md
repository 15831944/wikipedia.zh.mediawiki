《'''好報'''》（英語：Good News），[[香港|香港]]已停刊的[[免費報章|免費報章]]，為[[小型報|小型報]]及[[雜誌|雜誌]]形式的[[週報|週報]]，於2012年11月30日創刊，由[[東方報業集團|東方報業集團]]（{{sehk2|0018}}）創辦<ref name="《太陽》免費送《好報》">{{cite news|url=http://orientaldaily.on.cc/cnt/news/20121128/00190_001.html|title=讀者好人有好報 《太陽》免費送《好報》|date=2012年11月28日|newspaper=東方日報|accessdate=2011-11-30}}</ref><ref name="好人有《好報》">{{cite news|url=http://the-sun.on.cc/cnt/news/20121128/00416_001.html|title=東方太陽照 好人有《好報》|date=2012年11月28日|newspaper=《[[太陽報_(香港)|太陽報]]》|accessdate=2011-11-30}}</ref>。與當時香港市面上由Metro International的《[[都市日報|都市日報]]》、[[星島新聞集團|星島新聞集團]]的《[[頭條日報|頭條日報]]》、[[中原地產|中原地產]][[施永青|施永青]]的《[[am730|am730]]》、[[香港經濟日報集團|香港經濟日報集團]]的《[[晴報|晴報]]》及[[壹傳媒|壹傳媒]]的《[[爽報_(香港)|爽報]]》競爭<ref name="主場新聞">{{cite news|url=http://thehousenews.com/media/東方集團出版新免費報/|title=東方集團出版新免費報|publisher=主場新聞|date=2012-11-28|accessdate=2011-11-30|deadurl=yes|archiveurl=https://web.archive.org/web/20121201011056/http://thehousenews.com/media/%E6%9D%B1%E6%96%B9%E9%9B%86%E5%9C%98%E5%87%BA%E7%89%88%E6%96%B0%E5%85%8D%E8%B2%BB%E5%A0%B1/|archivedate=2012-12-01}}</ref>，及後《爽報》於2013年停刊，而《[[新晚報|新晚報]]》亦於2014年停刊。

由於受到政治前景不明朗，該報章於2016年3月11日停刊，連同「東方日報慈善基金」，以及「太陽報愛心基金」也一併終止<ref>[http://the-sun.on.cc/cnt/news/20160307/00407_006.html 免費《好報》 無奈停刊 東方日報慈善基金 太陽報愛心基金 展開結束程序]太陽報 2016年3月7日</ref>。

== 內容 ==
《好報》定位類似香港坊間的[[娛樂雜誌|娛樂雜誌]]，內容以藝人和名人揭秘等報道為主<ref>{{cite news|url=http://paper.wenweipo.com/2012/12/31/FI1212310012.htm|title=滄海遺珠：港進4G時代 免費報混戰|date=2012年12月31日|newspaper=文匯報|accessdate=2014-03-29}}</ref>，並且加入了[[潮流|潮流]]、[[消閒|消閒]]、[[美容|美容]]及奇聞等內容<ref name="《太陽》免費送《好報》"/><ref name="好人有《好報》"/><ref name="東方報業半年純利6704萬">{{cite news|url=http://orientaldaily.on.cc/cnt/finance/20121124/00202_015.html|title=東方報業半年純利6704萬|date=2012年11月24日|newspaper=《[[東方日報_(香港)|東方日報]]》|accessdate=2011-11-30}}</ref>。

== 派發策略 ==
《好報》的派發方式於首階段逢星期五隨《[[太陽報_(香港)|太陽報]]》附送，在香港各區免費派發；次階段則會每星期出版兩次，甚至會分拆為兩疊，未來可能獨立發行<ref name="《太陽》免費送《好報》"/><ref name="東方報業半年純利6704萬"/><ref name="主場新聞"/><ref>{{cite news|url=http://orientaldaily.on.cc/cnt/news/20121129/00176_097.html|title=政情：收料站：政圈熱切期待《好報》誕生|date=2012年11月29日|newspaper=東方日報|accessdate=2011-11-30}}</ref>。

== 辦報組織 ==
《好報》為東方報業集團旗下首份免費報刊<ref name="東方報業半年純利6704萬"/>，其他旗下報刊還包括《[[東方日報_(香港)|東方日報]]》、《[[太陽報_(香港)|太陽報]]》及《[[太陽馬經|太陽馬經]]》，並且擁有on.cc東方互動（http://on.cc ）及Money 18（http://money18.com.hk ）等網站<ref name="東方報業半年純利6704萬"/>。
