'''好妹妹乐队'''是[[中华人民共和国|中国]]的一支独立二人男声乐团。乐队名字来源于台湾歌手[[孟庭苇|孟庭苇]]的歌曲〈你究竟有几个好妹妹〉<ref>{{Cite web |url=http://et.21cn.com/gundong/etscroll/2012/07/16/12380133.shtml |title=“好妹妹乐队” ：每一首歌都是一个故事 |access-date=2013-11-06 |archive-url=https://web.archive.org/web/20141009023622/http://et.21cn.com/gundong/etscroll/2012/07/16/12380133.shtml |archive-date=2014-10-09 |dead-url=yes }}</ref>。2013年，乐队成立[[音乐工作室|音乐工作室]]「[[春生工作室|春生工作室]]」<ref>{{cite web|url=http://www.biosmonthly.com/interview_topic/9532|title=當代漂泊風景裡，唱一首溫暖誠懇的歌——專訪好妹妹樂隊﻿|author=BIOS monthly|}}</ref>。

== 乐队成员 ==
好妹妹乐队的队长[[张小厚|张小厚]]，毕业于[[浙江理工大学|浙江理工大学]]04级[[建筑环境|建筑环境]]与[[设备工程|设备工程]]系。主唱[[秦昊_(歌手)|秦昊]]毕业于[[吉林动画学院|吉林动画学院]]05级[[卡通漫画造型|卡通漫画造型]]专业。两人都不是音乐专业毕业，因共同爱好弹琴唱歌而结识，决定组成乐队，并制作发行独立专辑。

== 成名曲目 ==
乐队早期以翻唱[[华语流行|华语流行]]乐坛的经典抒情老歌为主，获得较大反响的作品为《[[新白娘子传奇|新白娘子传奇]]》的插曲〈青城山下白素贞〉。

乐队也有多首原创作品广受好评。如首张专辑《春生》中的〈相思赋予谁〉、〈冬〉、〈你飞到城市的另一边〉，第二张专辑《南北》中的〈风从海面吹过来〉、〈一个人的北京〉、〈晚风〉等<ref>[http://i.xiami.com/haomeimei/profile 好妹妹乐队 虾米音乐人简介]</ref>。另外值得一提的是，特别为[[单身|单身]]者所作的玩笑单曲《祝天下所有的情侣都是失散多年的兄妹》也在网络上掀起热评。<ref>[http://music.weibo.com/t/i/100052516.html 好妹妹乐队-浪客秦昊的音乐主站>祝天下所有的情侣都是失散多年的兄妹]</ref>

== 宣传 ==
2012年至2013年，乐队展开多场全国巡演，并在中国的多个[[音乐节|音乐节]]中参与演出。其中，好妹妹乐队也与台湾歌手孟庭苇、[[南方二重唱|南方二重唱]]等密切合作，在2013年9月的南方二重唱上海巡演中，乐队担当表演嘉宾，合唱原创单曲〈流浪春天的侧记〉。<ref>[http://ent.163.com/13/1003/08/9A8FFV3R00031H0O.html 南方二重唱上海开唱 孟庭苇好妹妹乐队助阵]</ref>
