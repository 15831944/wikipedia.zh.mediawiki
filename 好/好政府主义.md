'''好政府主义'''是1922年5月[[胡适|胡适]]、[[蔡元培|蔡元培]]等人发表《[[我们的政治主张|我们的政治主张]]》的宣言中所主张的政治倾向，<ref>{{Cite web |url=http://www.cnki.com.cn/Article/CJFDTotal-SHWZ200202021.htm|title=我们的政治主张——民国史上一次知识界的“参政”经历|date=|publisher=CNKI}}</ref>他们认为“[[好政府|好政府]]”的基本标准是[[宪政的政府|宪政的政府]]、[[公开的政府|公开的政府]]和[[有计划的政府|有计划的政府]]。<ref>{{Cite web |url=https://books.google.com/books?id=5CbkAAAAMAAJ&q=%E5%AE%AA%E6%94%BF%E7%9A%84%E6%94%BF%E5%BA%9C%E3%80%81%E5%85%AC%E5%BC%80%E7%9A%84%E6%94%BF%E5%BA%9C%E5%92%8C%E6%9C%89%E8%AE%A1%E5%88%92%E7%9A%84%E6%94%BF%E5%BA%9C&dq=%E5%AE%AA%E6%94%BF%E7%9A%84%E6%94%BF%E5%BA%9C%E3%80%81%E5%85%AC%E5%BC%80%E7%9A%84%E6%94%BF%E5%BA%9C%E5%92%8C%E6%9C%89%E8%AE%A1%E5%88%92%E7%9A%84%E6%94%BF%E5%BA%9C&hl=zh-CN&sa=X&ved=0ahUKEwiB-Y7j4tnfAhVGdXAKHalRAIEQ6AEIMDAC|title=上海出版志|date=|publisher=Google图书}}</ref>他们呼吁[[南北议和|南北议和]]，召集1917年被解散的国会，完成制宪。同时主张裁兵、财政公开、改良选举等。

==政治实践==
1922年9月19日，在[[吴佩孚|吴佩孚]]的支持下，以[[王宠惠|王宠惠]]为[[国务总理|国务总理]]的“好人政府”成立，但该届政府内阁中的“好人”徒有名分却无实权，成为军阀之间斗法的工具，只存在72天便夭折了。<ref>{{Cite web |url=http://www.cnki.com.cn/Article/CJFDTotal-WSCQ201506004.htm|title=“好政府主义”从倡导到夭折的前前后后|date=|publisher=CNKI}}</ref>

==相关评价==
同时代的中国革命派以及[[马克思主义者|马克思主义者]]认为，以胡适为代表的“好政府主义”在中国政治思想史上并无太多的新意，“好政府”主张表明了这样一个事实：建立一个理想的[[共和国|共和国]]在不少人的心目中仍有相当大的吸引力，在民国时期他们总想扮演共和国的精神守护神的角色。譬如，[[颜吾芟|颜吾芟]]、[[李震（历史学家）|李震]]认为，“好人政府”是近代中国[[自由派|自由派]][[知识分子|知识分子]]“精英”的一次幼稚且失败的执政尝试，对其嗤之以鼻。<ref>{{Cite web |url=http://www.npc.gov.cn/npc/zgrdzz/2014-02/11/content_1826098.htm|title=昙花一现的民国“好人政府”|date=|publisher=中国人大网}}</ref>[[陈独秀|陈独秀]]给[[李大钊|李大钊]]写信说，“好政府主义”简直就是“跪着造反”。[[周恩来|周恩来]]也认为，“好政府主义”无疑是给破楼将倾的房子打补丁，不但徒劳无功，更是阻止革命的发展。

中国当代一些自由派知识分子对胡适等人鼓吹的“好政府主义”则有不同看法，譬如，《[[三联生活周刊|三联生活周刊]]》[[李伟（《三联生活周刊》副主编）|李伟]]在《胡适：孤立的人最强大》一书中认为，对于当时中国之前途与命运引发一场热烈的讨论再所难免，胡适等人更多还是关注在具体问题上面，即要建立一个怎样的社会管理机构（政府）和社会运行机制。只不过，现实政治却让胡适大失所望，“好人政府”最终沦为军阀之间斗法的工具，昙花一现。<ref>{{Cite web |url=https://books.google.com/books?id=1vTenQEACAAJ&dq=%E8%83%A1%E9%80%82%EF%BC%9A%E5%AD%A4%E7%AB%8B%E7%9A%84%E4%BA%BA%E6%9C%80%E5%BC%BA%E5%A4%A7&hl=en&sa=X&ved=0ahUKEwiY9q3IpvTfAhXQZt4KHYKsCH4Q6AEIJzAA|title=中国华侨出版社|date=|publisher=Google图书}}</ref>
