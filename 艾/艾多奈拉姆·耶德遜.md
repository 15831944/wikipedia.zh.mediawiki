'''艾多奈拉姆·耶德遜'''（'''Adoniram Judson'''，{{bd|1788年|8月9日|1850年|4月12日}}）是一位[[美國|美國]][[傳教士|傳教士]]。也是一位終身奉獻於[[緬甸|緬甸]]的[[宣教士|宣教士]]（1813年）。

== 生平 ==
耶德遜出生於1788年8月9日，在[[美國|美國]][[馬薩諸塞州|馬薩諸塞州]]{{link-en|馬爾頓鎮|Malden, Massachusetts}}的一位牧師家庭中。1807年畢業於普維敦斯學院（Providence College，現今的[[布朗大學|布朗大學]]），領受學士學位。之後的一段時間，以教書和編寫教科書為主。在1808年進入{{link-en|安多福神學院|Andover Theological Seminary}}攻讀神學。<ref>潘恩，《傳教偉人耶德遜》，（香港：基督教輔僑出版社，1963），1-2。</ref>

== 前往亞洲傳教 ==
{{Fact|在神學院中，耶德遜偶然讀到了一篇[[布坎南|布坎南]]博士的「東方之皇」說教辭。之後又讀到一本有關[[沈美士|沈美士]]（Symes）出使[[緬甸|緬甸]]的紀錄書，使耶德遜對[[亞洲|亞洲]]傳教工作深受感動。從此，耶德遜決心獻身去亞洲傳教。[[美國|美國]]「傳道部」在1812年2月6日，在一公開儀式中第一次按立四位青年包括耶德遜，封立為牧師，任職傳教士到國外做宣教事工|time=2007-12-20}}。2月19日與{{link-en|安·哈信天|Ann Hasseltine Judson}}結婚。夫婦同行這一段艱辛的路程。

== 進入緬甸傳教 ==
在這遠程航行中，耶德遜在船上從事翻譯[[新約聖經|新約聖經]]，由[[希臘文|希臘文]]翻譯為[[英文|英文]]。由此可以讓他更熟悉[[聖經|聖經]]的名辭，以至於影響之後到[[緬甸|緬甸]]時翻譯[[緬甸語|緬甸文]][[聖經|聖經]]的[[事工|事工]]。回顧當年耶德遜往[[亞洲|亞洲]]傳教的行程有多少風險，據其他方面的情報，在[[緬甸|緬甸]]傳教前途甚不樂觀。於是認為到[[印度|印度]]比較沒有危險性，但情況並不是像想像中那麼妙，遭遇各種不如意的事情，甚至同伴之妻因患[[肺癆|肺癆]]去世。最後耶德遜已窮途絕境，並無他路可走，只好趁一艘駛往[[緬甸|緬甸]]的[[葡萄牙|葡萄牙]]船前往[[仰光|仰光]]。

1813年6月22日起程，這時候是風季，航行於[[印度洋|印度洋]][[孟加拉灣|孟加拉灣]]，最為險惡。在行程中，耶夫人產下了一已死胎中的嬰孩，夫人亦患重病。1813年7月13日當天船直入[[仰光大江|仰光大江]]，耶德遜終於到達[[緬甸|緬甸]]－[[仰光|仰光]]，開始了他的福音事工。<ref>潘恩，《傳教偉人耶德遜》，（香港：基督教輔僑出版社，1963），6-10。</ref>

== 對緬甸的貢獻 ==
耶德遜在緬甸初期的工作，就是學習[[緬甸語|緬甸語文]]。耶德遜後來甚至把[[聖經新約|聖經新約]]，[[舊約|舊約]]翻譯成[[緬甸語|緬文]][[聖經|聖經]]。1835年，耶德遜的緬文聖經譯本印刷出版。<ref>潘恩，《傳教偉人耶德遜》，（香港：基督教輔僑出版社，1963），11-32。</ref>

==緬甸的宗教==
緬甸是一個虔誠的佛教國家。如今雖仍然是信奉佛教，但根據耶德遜時代相比，已有不少基督徒。當耶德遜進入緬甸時，沒有一個基督徒。1850年他離世時（62歲）緬甸已有七千多基督徒。耶德遜前在[[仰光|仰光]]建立的第一間教會，就是現在的吳諾（緬甸第一位基督徒）紀念堂。今天的[[仰光大學|仰光大學]]裡能看到一所耶德遜紀念教堂（Judson Memorial Baptist Church）。
