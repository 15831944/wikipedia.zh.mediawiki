'''艾薇薇'''（{{bd|1989年|3月30日|2011年|5月4日|catIdx=Ai艾薇薇}}）[[中国|中国]][[江苏|江苏]]人，知名[[模特儿|模特儿]]和[[世界杯|世界杯]]宝贝。<ref>{{cite web |url=http://www.jiaodong.net/ent/system/2011/05/20/011279364.shtml |title=22岁世界杯宝贝艾薇薇过劳死 经纪人微博留素描缅怀 |accessdate=2011-05-20 |work=|publisher=胶东在线 |language= zh-hans}}</ref><ref name="multiple111 " >{{cite web |url=http://news.cntv.cn/society/20110520/105458.shtml |title=福州22岁女模特过劳死 因长期通宵积劳成疾 |accessdate= 2011年5月20日|work=|publisher= 中国网络电视台|language=zh-hans}}</ref>

== 简介 ==
艾薇薇16岁就因为家庭贫寒而离开家庭，一边打工挣钱一边上学，大专毕业后在[[福州|福州]]工作，随后去[[上海|上海]]发展，签约[[聚龙网络影视传媒|聚龙网络影视传媒]]，成为其旗下终身模特儿，并且一直来往于[[北京|北京]]、[[上海|上海]]、[[福州|福州]]三地，是聚龙网络影视传媒的一姐。因为长期通宵达旦地劳累工作，她患上了[[急性混合细胞白血病|急性混合细胞白血病]]，2011年5月14日凌晨，艾薇薇在福建一家医院因治疗无效而去世，终年22岁，去世前父母、男友及好友均在身边。<ref name="multiple111 " />

== 评价 ==
艾薇薇的经纪人[[朱兴|朱兴]]说：“她为了工作，有时候可以三天三夜不眠不休。”，“艾薇微是个倔强的小姑娘，做事很认真，甚至可以说是‘工作狂’，通宵工作是常有的事。”<ref name="multiple111 " />
