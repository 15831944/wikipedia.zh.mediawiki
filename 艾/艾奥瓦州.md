'''愛荷華州'''{{notetag|又譯「衣-{}-阿-{}-華」、「愛-{}-阿-{}-華」、「愛-{}-我-{}-華」，中国大陸正式译名「艾-{}-奥-{}-瓦」。中華民國外交部、國立編譯館與國家教育研究院使用的漢語標準譯名為「愛-{}-荷-{}-華」。}}（{{lang-en|State of Iowa}}）是位於[[美國中西部|美國中西部]]大平原的一個[[美国州份|州份]]；20世纪中后期，愛荷華州由农业经济為主过渡到多元化经济，如高级制造业，金融保险，生物技术和绿色能源产品等；愛荷華州被列为美国治安最好的州之一，也是基础教育最好的州之一。[[首府|首府]]為[[德梅因|德梅因]]。

== 歷史 ==

艾奥瓦名稱源自印第安部落（Ioway）。

文件記載歐洲人首次到達艾奥瓦州是在1673年 Jacques Marquette和Louis Jolliet 旅行到密西西比河。

美國南北戰後，艾奥瓦的人口和農業快速發展，而在大蕭條和二戰時期逐漸發展大型農業與都市化。

== 人口 ==
2003年人口普查顯示，艾奥瓦共有2,944,062人。種族比例分為：
* 92.6%是無拉美血統的白人；
* 2.1%是非裔美國人；
* 2.8%是[[拉美裔人|拉美裔人]]；
* 1.3%是亞裔美國人；
* 0.3%是[[印第安人|印地安原住民]]；
* 1.1%是混血。

艾奥瓦的前五大居民祖先為：
# [[德國|德國]]人（35.7%）
# [[愛爾蘭|愛爾蘭]]人（13.5%）
# [[英國|英國]]人（9.5%）
# 本土化血統者（6.6%）
# [[挪威|挪威]]人（8.4%）。

艾奥瓦州有6.4%的居民在5歲以下，25.1%的居民在18歲以下，14.9%的居民在65歲以上，女性比男性多，約有50.9%。

=== 宗教 ===
在2001年時一個由[[紐約市立大學|紐約市立大學]]所主導的調查之結果中，顯示約有52%的艾奥瓦居民為[[新教徒|新教徒]]或其衍生教派，23%為[[羅馬天主教|羅馬天主教]]信徒，6%為其他信仰，與13%無宗教信仰者<ref>[http://www.gc.cuny.edu/faculty/research_studies/aris.pdf American Religious Identification Survey]</ref>。

== 地理 ==
80,392 平方英里。行政區位上，艾奥瓦州北接[[明尼蘇達州|明尼蘇達州]]，西邊與[[內布拉斯加州|內布拉斯加州]]與[[南達科他州|南達科他州]]相鄰，南接[[密蘇里州|密蘇里州]]，東鄰[[威斯康辛州|威斯康辛州]]與[[伊利諾州|伊利諾州]]。

== 政治 ==
=== 美國總統大選黨團會議 ===
英文稱之為「'''Iowa presidential caucus'''」。愛荷華州每逢4年一度的[[美國總統大選|美國總統大選]]期間，一定會成為全國各地矚目的焦點。美國總統大選通常在該年的1月份開始進行初選的投票，初選的類型可分為「[[黨團會議|黨團會議]]（caucus）」和「[[初選|初選]]（primary）」兩種。最早舉行黨團會議的州就是'''愛荷華州'''，而最早舉行初選的州則是'''[[新罕布什尔州|新罕布什尔州]]'''。新罕布什尔州的初選投票會比愛荷華州的黨團會議投票還要晚一個禮拜的時間。由於領先全國其他州開出初選結果，所以讓這兩州的初選投票有「美國總統大選前哨戰」之稱。
