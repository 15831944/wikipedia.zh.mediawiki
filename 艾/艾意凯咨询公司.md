'''艾意凯咨询公司'''是一家国际一流的[[管理咨询|管理咨询机构]]，于1983年由[[贝恩策略顾问|贝恩策略顾问公司]]的三位合伙人：理查德·库奇、伊恩·埃文斯和詹姆斯·劳伦斯成立。该公司总部设于[[英国|英国]][[伦敦|伦敦]]，在全球21个办事处拥有超过1500名[[顾问|顾问]]。<ref>{{cite web|url=http://www.lek.com/about|title=About L.E.K.|publisher=艾意凯咨询公司|access-date=2017-05-25}}</ref>作为一家[[合伙企业|合伙]]企业，艾意凯咨询公司采用人资事业服务模式（Generalist Model），为客户企业提供[[策略管理|管理策略]]、行业经验、[[并购|合并收购]]等，以供应泛亚乃至全球的项目需求。该公司专门开发大型领域之业务，如[[私募股权基金|私募股权基金]]、[[国防|国防]]、[[航空|航空]]与[[旅游|旅游]]、[[生命科学|生命科学]]、[[医疗保健|医疗保健]]、[[能源|能源]]（[[煤炭|煤炭]]、[[电力|电力]]、[[可再生能源|可再生能源]]）、[[娱乐|娱乐]]、[[运输|运输]]、[[建材|建材及工业品]]、[[零售|零售]]、[[消费品|消费品]]以及[[金融服务|金融服务]]<ref>{{cite web|url=http://www.lek.com/industries|title=L.E.K. Industries|publisher=艾意凯咨询公司|access-date=2017-05-25}}</ref>。而艾意凯咨询公司的咨询占据了[[富时100指数|富时100指数]]的三分之一，是唯一上市的英国策略管理咨询公司。<ref>{{cite web|url=https://managementconsulted.com/consulting-interviews/lek-consulting/|title=L.E.K. Strategy Consulting Interviews and Culture|publisher=Management Consulted|access-date=2017-05-25}}</ref>

==服务==
艾意凯咨询公司旨在于提供[[市场|市场]]洞见及[[策略|策略]]工具，帮助客户在商业环境中运用[[资源|资源]]，以提升客户销量和企业[[盈利|盈利]]。该公司客户涵盖[[跨国公司|跨国公司]]、[[国有企业|国有企业]]、[[民营企业|民营企业]]、[[政府机构|政府机构]]和[[组织|行业组织]]。其服务范围包括：      
*为现有[[市场|市场]]或新市场提供[[策略|策略]]市场见解<ref>{{cite web | url =http://www.lek.com/cn/case-studies/%E4%B8%BA%E5%BB%BA%E6%9D%90%E4%BC%81%E4%B8%9A%E5%88%B6%E5%AE%9A%E6%96%B0%E4%BA%A7%E5%93%81%E7%9A%84%E5%B8%82%E5%9C%BA%E6%88%98%E7%95%A5%EF%BC%8C%E9%A9%B1%E5%8A%A8%E7%9B%88%E5%88%A9%E5%A2%9E%E9%95%BF| title = 为建材企业制定新产品的市场战略，驱动盈利增长| publisher = 艾意凯咨询公司| access-date =2017年5月25日}}</ref>
*制定市场经济增长策略<ref>{{cite web | url =http://www.lek.com/cn/case-studies/%E5%88%B6%E5%AE%9A%E8%87%B4%E8%83%9C%E7%9A%84%E5%B8%82%E5%9C%BA%E8%90%A5%E9%94%80%E6%88%98%E7%95%A5| title = 制定致胜的市场营销战略| publisher = 艾意凯咨询公司| access-date =2017年5月25日}}</ref>
*甄别及评估[[投资|投资]]机会与定价策略，并提供调查服务<ref>{{cite web | url =http://www.lek.com/cn/case-studies/%E6%8A%98%E6%89%A3%E9%9B%B6%E5%94%AE%E5%95%86%E9%87%8D%E6%96%B0%E8%AF%84%E4%BC%B0%E5%AE%9A%E4%BB%B7%E7%AD%96%E7%95%A5%EF%BC%8C%E5%AE%9E%E7%8E%B0%E6%95%B0%E5%8D%83%E4%B8%87%E7%BE%8E%E5%85%83%E7%9A%84%E6%96%B0%E5%88%A9%E6%B6%A6%E7%A9%BA%E9%97%B4| title =折扣零售商重新评估定价策略，实现数千万美元的新利润空间| publisher = 艾意凯咨询公司| access-date =2017年5月25日}}</ref>
*提升业务组合业绩
*优化业务流程及[[供应链|供应链]]<ref>{{cite web | url =http://www.lek.com/cn/case-studies/%E7%90%86%E9%A1%BA%E4%B8%9A%E5%8A%A1%E6%B5%81%E7%A8%8B%EF%BC%8C%E4%B8%BA%E8%88%AA%E7%A9%BA%E5%85%AC%E5%8F%B8%E6%AF%8F%E5%B9%B4%E8%8A%82%E7%9C%811.9%E4%BA%BF%E7%BE%8E%E5%85%83| title = 理顺业务流程，为航空公司每年节省1.9亿美元| publisher = 艾意凯咨询公司| access-date =2017年5月25日}}</ref>

==交换计划==
艾意凯咨询公司每年会进行为期六个月的交换计划（SWAP Program）。其公司员工将与其他海外办事处的同行互换工作地点，在不同的环境下学习专业能力，扩大资源网络。<ref>{{cite web|url=http://www.lek.com/join-lek/the-lek-career/international-opportunities|title=International Opportunities|publisher=艾意凯咨询公司|access-date=2017年5月25日}}</ref>

==中国分支机构==
艾意凯咨询于1998年入驻[[中国|中国]]开展业务，在首都[[北京|北京]]市和[[上海|上海]]市设立分支机构，而在华业务总部是上海的分公司，其团队由常驻[[中国|中国]]内陆的合伙人和双语专业人士组成。现任中国分支机构董事总经理为陈玮。<ref>{{cite web|url=http://www.lek.com/cn|title=艾意凯咨询在中国|publisher=艾意凯咨询公司中文网|access-date=2017年5月25日}}</ref>
