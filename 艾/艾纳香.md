'''艾纳香'''（[[学名|学名]]：{{lang|la|''Blumea balsamifera''}}）是[[菊科|菊科]][[艾纳香属|艾纳香属]]的植物。别名“大艾”、“大风艾”（广西）。

== 形态 ==
多年生[[草本|草本]]或半灌木状，多分枝，幼嫩部密被绢状毛；长椭圆形至披针形叶子互生，有锯齿，上面发皱，密生黄褐色短硬毛，下面有黄褐色密绢状绵毛；春末开黄色小花，头状[[花序|花序]]排列成伞形，两性花花冠管状，雄花丝状。

== 分布 ==
分布于[[巴基斯坦|巴基斯坦]]、[[印度|印度]]、[[缅甸|缅甸]]、[[泰国|泰国]]、[[马来西亚|马来西亚]]、[[中南半岛|中南半岛]]、[[台湾岛|台湾岛]]以及[[中国大陆|中国大陆]]的[[福建|福建]]、[[广东|广东]]、[[广西|广西]]、[[贵州|贵州]]、[[云南|云南]]等地，生长于海拔600米至1,000米的地区，多生于林缘、林下、河床谷地以及草地上。

== 用途 ==
叶子为制造香料的原料，用[[蒸馏|蒸馏]]所得的白色艾粉，复制成艾片，用作芳香开窍药，也用作杀菌、防腐。
