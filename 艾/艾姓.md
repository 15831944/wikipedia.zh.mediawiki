'''艾姓'''，《[[百家姓|百家姓]]》收，分布很广，以[[華北|華北]]為多。[[中國歷史|中國歷史]]上[[宋朝|宋朝]]有[[艾仲儒|艾仲儒]]，[[明朝|明朝]]有[[艾南英|艾南英]]，[[清朝|清朝]]有[[艾元征|艾元征]]。

== 源流 ==
*[[夏朝|夏朝]][[少康|少康]]大臣汝艾的后代。
*[[商朝|商朝]]时期[[江西|江西]]的封国-[[艾国|艾国]]。
*[[春秋|春秋]][[宋国|宋国]][[宗室|宗室]]后代改姓艾岁，而[[艾歲|艾歲]]爾後又改為'''艾姓'''、[[歲姓|歲姓]]。
*[[春秋|春秋]]时[[齐国|齐国]][[大夫|大夫]]艾孔的后代。
*[[北魏|北魏]][[孝文漢化|孝文漢化]]时，[[胡人|胡人]][[去斤|去斤]]氏改姓艾。
*[[北宋|北宋]]时期来到中原并定居的[[开封犹太人|开封犹太人]]改姓。
*雲南蒙古族後裔改姓。
*中華人民共和國建國後，部分蔣姓的文化界人士的改姓。

== 歷史名人 ==
*[[艾晟|艾晟]]，真州（今江苏省仪征）人，宋徽宗崇寧年間，曾任通仕郎行杭州仁和县尉管句学事、秘书省校书郎。
*[[艾宣|艾宣]]，钟陵（今江西进贤县）人，宋朝画家，爱画工笔花鸟和飞禽走兽，细至入徽，栩栩如生。
*[[艾可久|艾可久]]，上海人，明世宗嘉靖41年（1562年）进士，历官太常寺傅、御史、衡州知州、山东副使、江西和陕西参政、按察使、南京通政使等。
*[[艾自新|艾自新]]，邓川中所人，[[艾自修|艾自修]]胞兄，明朝學者。讲学邓川、大理、太和等地。與其胞弟[[艾自修|艾自修]]齊名，人稱“二艾”。
*[[艾自修|艾自修]]，邓川中所人，明朝官員、專研宋儒理學的學者。明神宗萬曆28年(1600年)進士，以县令升湖南辰州知州，所至均有惠政。
*[[艾南英|艾南英]]，（1583～1646）明末散文家、文学评论家。临川段溪艾家村（今江西省东乡县岗上积乡艾家村）人。天启举人，策文讥刺权宦魏忠贤，罚停考三科。清军南下，江西郡县尽失。入闽见南明唐王朱聿键，任监察御史。次年病卒。今存《禹贡图注》。
*[[艾元征|艾元征]]（1624—1676），山东济阳孙耿人，清臣。济阳名士张稷诺的学生，历任翰林院学士、户部侍郎、左都御史、刑部尚书等职。顺治三年(1646)考中进士，改翰林庶吉士。明年授弘文院检讨。十年(1653)迁左春坊左庶子、兼秘书院侍读。历侍讲学士、秘书院学士，擢礼部右侍郎兼东阁学士。十八年(1661)授国史院学士。康熙二年(1663)擢户部右侍郎，四年转左侍郎，九年调吏部，迁都察院左都御史。十一年晋刑部尚书。
*[[艾思奇|艾思奇]]（1910－1966），中国左派哲学家。云南腾冲人。
*[[艾大炎|艾大炎]]（1910－1996），河北武清人，原名艾秀峰。早年留学日本，归国后曾任《大公报》编辑，首任[[河北省人民政府|河北省人民政府]]秘书长，高教部留学生管理司司长，国务院科技干部局副局长。

== 現代名人 ==
*[[艾中信|艾中信]]：中國油畫大師，中央美術學院教授、原副院長。
*[[艾兴|艾兴]]，1924年8月出生，江西东乡人。现任山东大学机械学院教授、博士生导师，中国工程院院士
*[[艾伟|艾伟]]，男，漢族，台湾艺人，出演过多部电视剧，代表作為花系列作品。亦主持美食节目
*[[艾敬|艾敬]]，女，漢族，中華人民共和國籍，創作型歌手
*[[艾知生|艾知生]]，原广播电影电视部部长
*[[艾怡良|艾怡良]]，台灣創作女歌手，為華語樂壇靈魂樂唱腔的代表。
