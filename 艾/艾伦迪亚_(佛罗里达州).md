'''艾伦迪亚'''（{{lang-en|Islandia}}）曾是[[美国|美国]][[佛罗里达州|佛罗里达州]][[邁阿密-戴德縣|邁阿密-戴德縣]]轄下的城市，2012年3月16日法令生效後廢除<ref name="ordinance">[http://www.miamidade.gov/govaction/matter.asp?matter=112413&file=true&yearFolder=Y2011 Miami-Dade Legislative Item File Number: 112413]</ref><ref name="boundaries">[http://gisweb.miamidade.gov/GISSelfServices/Data/HTML/Municipality_poly.htm]</ref>，現為[[非建制地區|非建制地區]]。建立于1960年。面积约171.9平方公里（约合66.4平方英里）。根据[[2010年美国人口普查|2010年美国人口普查]]，该地區有人口18人。论人口在本州排行第407。<ref name=census>{{cite web |url=http://www.census.gov/prod/www/abs/decennial/ |title = Census  of Population and Housing |publisher = U.S. Census Bureau |accessdate = November 18, 2011}}</ref>
