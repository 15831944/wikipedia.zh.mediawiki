'''艾兰·奧拉夫·沃克'''（英語：'''Alan Olav Walker'''，1997年8月24日－），是一位生於[[英格兰|英格蘭]][[北安普顿|北安普頓]]的挪威籍[[DJ|DJ]]和[[音樂製作人|音樂製作人]]。他因[[电子舞曲|電子舞曲]]單曲《[[人間迷走|人間迷走]]》（Faded）一炮而紅 ，該曲的官方音樂影片在[[YouTube|YouTube]]上的觀賞人次已超過23億2888萬人次，訂閲人數已達24,511,657人(截至2019年4月19日上午10時56分)。

== 事業 ==

[[File:Alan_Walker_(001216).jpg|thumb]][[拉爾維克|拉爾維克]]表演|276x276像素]]
=== 早期發展 ===
沃克生於[[英格兰|英格蘭]][[北安普顿|北安普頓]]，兩歲時與[[挪威|挪威]]籍母親和[[英国|英國]]籍父親搬到挪威[[卑爾根|卑爾根]]。在數位時代中長大，沃克開始對[[電子計算機|電腦]]感興趣，後來被[[程序设计|編程]]和[[平面设计|平面設計]]吸引。他沒有任何音樂背景，只依靠觀看[[YouTube|YouTube]]影片自學。從2011年便開始學習及嘗試製作音樂，他用[[筆記型電腦|筆記型電腦]]做了30至40首歌。

2012年，沃克找到了[[意大利|義大利]]的唱片技師David Whistle（以前稱為DJ Ness）去學習製作音樂，並使用了為初學者而設的編曲軟體[[FL_Studio|FL Studio]]。同年，他開始將自己所創作的音樂上傳到YouTube頻道和[[SoundCloud|SoundCloud]]頁面，在網上的評價幫助下，他的音樂受到了音樂品牌的注目。

=== 2014年 ===

==== Fade ====
沃克在2014年11月19日經[[YouTube|YouTube]]知名音樂品牌[[NoCopyrightSounds|NoCopyrightSounds]]（NCS）發佈了他的[[電子音樂|電子音樂]]曲目「Fade」。此曲目發佈後引發了網路上廣泛的迴響，成為了NCS頻道最熱門的曲目之餘，亦成為了[[YouTube|YouTube]]上最常用的影片[[背景音乐|背景音樂]]之一。該曲目在YouTube上的觀看次數突破3.7億，<ref>{{Cite web|url=https://www.youtube.com/watch?v=bM7SZ5SBzyY|title=Alan Walker - Fade|accessdate=|author=Alan Walker|date=|publisher=YouTube}}</ref>在[[Spotify|Spotify]]和[[SoundCloud|SoundCloud]]上的播放次數更分別達到了8992萬和1700萬次。<ref>{{Cite web|url=https://play.spotify.com/track/30ykgxNKHLr95PkHHqmrjl|title=Fade - Alan Walker|accessdate=|author=Alan Walker|date=|publisher=Spotify}}</ref><ref>{{Cite web|url=https://soundcloud.com/nocopyrightsounds/alan-walker-fade-ncs-release|title=Alan Walker - Fade [NCS Release]|accessdate=|author=Alan Walker|date=|publisher=SoundCloud}}</ref>「Fade」其後收錄在NCS[[音樂專輯|專輯]]「NCS: Uplifting」。

=== 2015年 ===

====  Spectre、Force ====
沃克在2015年1月6日經NCS發佈了他的[[電子音樂|電子音樂]]曲目「 Spectre」，成為該頻道內第二熱門的曲目，僅次於「Fade」。「 Spectre」其後收錄在NCS的[[音樂專輯|專輯]]「NCS: The Best of 2015」。「Force」隨後亦於2015年4月2日經NCS發佈。「Force」其後收錄在NCS的專輯「NCS: Infinity」。

==== Faded ( [[人間迷走|人間迷走]] ) ====
沃克與[[瑞典|瑞典]][[索尼音樂|索尼音樂]]簽約，在2015年12月4日推出他的第一首[[单曲|單曲]]《人間迷走》，以他早前的電子音樂曲目「Fade」作為基礎，由[[挪威|挪威]]籍女歌手[[艾瑟琳·索爾海姆|艾瑟琳·索爾海姆]]獻唱。這首歌推出後引發環球的極大迴響，高居[[挪威|挪威]]VG-LISTA單曲榜和[[瑞典|瑞典]]Sverigetopplistan排行榜榜首，先後登上32國[[iTunes|iTunes]]冠軍以及21國流行單曲榜頭十名，同時打入[[Spotify|Spotify]]全球榜頭10名，沃克成為全球樂評及樂迷相當關注的樂壇新秀。《人間迷走》在YouTube的觀看次數已經突破23億，同時擁有1557萬個讚，<ref>{{Cite web|url=https://www.youtube.com/watch?v=60ItHLz5WEA|title=Alan Walker - Faded|accessdate=|author=Alan Walker|date=|publisher=YouTube}}</ref>登上了[[最受喜歡的YouTube影片列表|最受喜歡的YouTube影片列表]]頭10名。國際首屈一指的DJ[[提雅斯多|提雅斯多]]，著名DJ [[達許柏林|達許柏林]]及[[硬威爾|硬威爾]]亦有為《人間迷走》製作[[重混|重混]]。

=== 2016年   The Walker Tour ===

==== Sing Me To Sleep ( [[唱首歌伴我入眠|唱首歌伴我入眠]] ) ====
沃克在2016年6月3日經[[瑞典|瑞典]][[索尼音樂娛樂|索尼音樂]]推出[[单曲|單曲]]《唱首歌伴我入眠》，繼續由女歌手艾瑟琳·索爾海姆獻唱。此曲目推出後登上7國iTunes冠軍、超過2.1億次Spotify串流。<ref>{{Cite web|url=https://play.spotify.com/track/6UcTHiP25meOD9ox8Rspgx|title=Sing Me To Sleep - Alan Walker|accessdate=|author=Alan Walker|date=|publisher=Spotify}}</ref>《唱首歌伴我入眠》在YouTube的觀看次數已突破5億。著名DJ [[Marshmello_(音樂製作人)|Marshmello]] 亦為此單曲製作重混。<ref>{{Cite web|url=https://www.youtube.com/watch?v=2i2khp_npdE|title=Alan Walker - Sing Me To Sleep|accessdate=|author=Alan Walker|date=|publisher=YouTube}}</ref>

==== Alone ( [[不孤單_(艾倫·沃克的歌曲)|不孤單]] ) ==== 
沃克在2016年12月2日推出單曲《不孤單》，由挪威籍女歌手諾米·鮑獻唱。《不孤單》在YouTube的觀看次數已突破8.3億，<ref>{{Cite web|url=https://www.youtube.com/watch?v=1-xGerv5FOk|title=Alan Walker - Alone|accessdate=|author=Alan Walker|date=|publisher=YouTube}}</ref>並在Spotify達到了3億次串流。<ref>{{Cite web|url=https://play.spotify.com/track/3ujORpm8Hf7QWEtfqSAZL8|title=Alone - Alan Walker|accessdate=|author=Alan Walker|date=|publisher=Spotify}}</ref>

==== Routine ====
沃克與David Whistle在2016年12月23日推出單曲「Routine」，「Routine」在YouTube的觀看次數已突破4889萬。

=== 2017年 ===

==== Ignite ( 點燃 )====
沃克與K-391在2017年4月8日推出單曲「Ignite」，是為了替[[索尼|Sony]]公司宣傳手機[[Sony_Xperia_XZs|Xperia XZs]]而作。

==== Tired ( [[Tired|心已累]] ) ====
沃克在2017年5月18日推出單曲「Tired」，由Gavin James獻唱。為《World of Walker》三部曲的序章。「Tired」在YouTube的觀看次數已突破1.2億 <ref>Alan Walker-Tired-YouTube</ref>，在Spotify的播放次數已突破1.6億。

==== Sky ====
Alan Walker與Alex Skrindo在2017年6月9日推出單曲「Sky」，「Sky」在YouTube的觀看次數已突破5411萬。

==== The Spectre ( [[幽靈_(歌曲)|幽靈]] ) ====
沃克在2017年9月15日推出單曲「The Spectre」，為沃克於2015年發表的作品「Spectre」加上人聲的混音版。「The Spectre」在YouTube的觀看次數已突破5.4億 ，在Spotify的播放次數已突破1.2億。

==== All Falls Down ( [[全面崩壞|全面崩壞]] ) ====
沃克與{{le|Noah Cyrus|Noah Cyrus}}和{{le|Digital Farm Animals|Digital Farm Animals}}在2017年10月26日推出單曲「All Falls Down」。為《World of Walker》三部曲的首部曲。「All Falls Down」在YouTube的觀看次數已突破1.6億 ，在Spotify的播放次數已突破2.7億。

=== 2018年   World Of Walker Tour: Part I ===

==== Ignite ( [[點燃_(歌曲)|點燃]] ) ====
K-391與沃克在2018年5月12日推出單曲「Ignite」，由挪威歌手{{le|Julie Bergan|Julie Bergan}}與[[大韓民國|韓國]]歌手[[勝利_(歌手)|勝利]]獻唱。為沃克及K-391於2017年發表的作品「Ignite」加上人聲的混音版。「Ignite」在YouTube的觀看次數已突破1.7億 ，在Spotify的播放次數已突破9080萬。

==== Darkside ( [[黑暗面|黑暗面]] ) ====
沃克在2018年7月27日推出單曲「Darkside」，由[[西班牙|西班牙]]歌手[[Au/Ra|歐拉]]以及挪威歌手Tomine Harket獻唱。為《World of Walker》三部曲的二部曲。「Darkside」在YouTube的觀看次數已突破3億 ，在Spotify的播放次數已突破1.4億。

==== Diamond Heart ( 鑽石心 ) ====
沃克在2018年9月28日推出單曲「Diamond Heart」，由Sophia Somajo獻唱。為《World of Walker》三部曲的最終章。「Diamond Heart」在YouTube的觀看次數已突破1.2億 ，在Spotify的播放次數已突破7556萬。

=== 2018年末 - 2019年 Different World Tour ===

==== Different World ( 理想世界 ) ====
沃克在2018年11月30日推出單曲「Different World」，與Sofia Carson ，K-391和CORSAK合作推出 。「Different World」在YouTube的觀看次數已突破1500萬  ，在Spotify的播放次數已突破2000萬。

==== Do It All For You ( 為你做一切 ) ====
沃克在2018年推出單曲「Do It All For You」，與Trevor Guthrie合作推出 。「Do It All For You」在Spotify的播放次數已突破409萬。

==== Lonely ( 孤獨 ) ====
沃克在2018年推出單曲「Lonely」，與Steve Aoki，ISÁK和Omar Noir合作推出 。「Lonely」在Spotify的播放次數已突破523萬。

==== Lily ( 麗莉 ) ====
沃克在2018年推出單曲「Lily」，與K-391和Emelie Hollow合作推出 。「Lily」在Spotify的播放次數已突破895萬。

==== I Don't Wanna Go ( 我不想離開 )====
沃克在2018年推出單曲「I Don't Wanna Go」，與{{le|Julie Bergan|Julie Bergan}}合作推出 。「I Don't Wanna Go」在Spotify的播放次數已突破628萬。

==== Lost Control ( 失去控制 ) ====
沃克在2018年推出單曲「Lost Control」，與Sorana合作推出 。「Lost Control」在Spotify的播放次數已突破2274萬。

==== Intro ====
沃克在2018年推出單曲「Intro」。「Intro」在Spotify的播放次數已突破344萬。

==== Interlude ====
沃克在2018年推出單曲「Interlude」 。「Interlude」在Spotify的播放次數已突破270萬。

==== Faded (Interlude) ====
沃克在2018年推出單曲「Faded (Interlude)」 。「Faded (Interlude)」在Spotify的播放次數已突破200萬。

==== Are You Lonely ( 你孤獨嗎？ ) ====
沃克在2019年2月推出單曲「Are You Lonely」，與Steve Aoki和ISÁK合作推出 ，和「Lonely」為同曲不同詞 ，於2月22日在Youtube發行。

==== On My Way  ( 我的道路 ) ====
沃克在2019年3月21日推出單曲「On My Way」，與 [[PUBG_mobile|PUBG mobile]]、[[莎賓娜·卡本特|莎賓娜·卡本特]]、[[Farruko|Farruko]]合作 ，於3月21日在Youtube發行，此單曲[[音樂錄影帶|音樂錄影帶]]已經累積7278萬觀看次數。

== 公眾形象 ==
沃克最初被稱為“DJ Walkzz”。他最終用自己的名字艾倫·沃克（Alan Walker）作為他的藝名。他的標誌包含其名字的首字母「A」和「W」，而walker是步行者的意思，艾倫•沃克運用walker來當作探索大自然的領導者，而提醒大家地球正在崩塌，要好好關愛地球這顆美麗之星
