艾蒂安-路易·布雷 （1728年2月12日 – 1799年2月4日）是一位具有远见的[[法国|法国]][[新古典主義|新古典主义]][[建筑师|建筑师]]。他的作品对现代建筑师影响颇深。

== 生平 ==
[[File:Hotel_de_Brunoy.jpg|右]]
布雷生于巴黎，师从[[雅克-弗朗索瓦·布隆代爾|雅克-弗朗索瓦·布隆代尔]]、[[杰曼·博夫兰|杰曼·博夫兰]]和{{Tsl|en|Jean-Laurent Le Geay|让-劳伦特·勒吉}}，学习17、18世纪主流的法式[[古典主義建築|古典主义建筑]]和中世纪结束后发展起来的[[新古典主义|新古典主义]]。1762年，他被选入{{Tsl|en|Académie Royale d'Architecture|法国皇家建筑学会}}，成为[[腓特烈二世_(普鲁士)|腓特烈二世]]的首席建筑师——很大程度上这是一个荣誉头衔。1762年到1778年间，他设计了许多私人住宅，尽管其中大部分已不复存在；有幸保存至现代的著名建筑包括布吕努瓦市政厅（已于1930年拆除）和亚历山大酒店，两处都在巴黎。他为弗朗索瓦·拉辛·德·蒙维尔（François Racine de Monville）设计的建筑显然也没能保留下来，但布雷对蒙维尔的影响在“{{Tsl|en|Désert de Retz|瑞兹沙漠}}”这座花园中保留了下来。与[[克劳德·尼古拉斯·勒杜|克劳德·尼古拉斯·勒杜]]一样，布雷是对法国[[新古典主义建筑|新古典主义建筑]]最有影响力的大师之一。

== 几何风格 ==
1778年至1788年间，布雷在[[国立桥路学校|国立桥路学校]]任教，研究建筑理论。这是他最有影响力的一段时期。受到古典形式的启发，他发展出了独特的抽象几何风格。他作品的特点是去除了所有不必要的装饰，将基本的几何形状放大，以及重复元素的运用，如各式各样的柱子。

对于布雷而言，规律性、对称性和多样性是建筑的黄金法则。

== 艾萨克·牛顿爵士纪念堂 ==
[[File:Newton_memorial_boullee.jpg|右]]
布雷提倡建筑表达其目的的理念，他的批评者称之为“{{Tsl|en|Architecture parlante|建筑说话}}”。这一理念是19世纪后期[[布雜藝術|布杂艺术]]学派的一个重要组成部分。布雷的风格在他为英国科学家艾萨克·牛顿设计的纪念堂中得到了最著名的体现。牛顿在去世50年后成为启蒙思想的象征。<ref>{{Cite web|url=http://www.archdaily.com/544946/ad-classics-cenotaph-for-newton-etienne-louis-boullee|title=AD Classics: Cenotaph for Newton / Etienne-Louis Boullée|accessdate=2016-10-04|date=2014-09-10|language=en-US}}</ref><ref>{{Cite web|url=http://www.archdaily.com/544946/ad-classics-cenotaph-for-newton-etienne-louis-boullee|title=AD Classics: Cenotaph for Newton / Etienne-Louis Boullée|accessdate=2016-10-04|date=2014-09-10|language=en-US}}</ref> 这座纪念堂本身是一个150米高的球体，比[[胡夫金字塔|胡夫金字塔]]还要高，周围环绕着两圈巨大的屏障和数百株柏树。<ref>{{Cite web|url=http://www.archdaily.com/544946/ad-classics-cenotaph-for-newton-etienne-louis-boullee|title=AD Classics: Cenotaph for Newton / Etienne-Louis Boullée|accessdate=2016-10-04|date=2014-09-10|language=en-US}}</ref> 虽然这座建筑从未建成，但1784年，许多布雷所作的钢笔画被刻印出来，在专业圈子里得到了广泛传阅。<ref>{{Cite web|url=http://www.archdaily.com/544946/ad-classics-cenotaph-for-newton-etienne-louis-boullee|title=AD Classics: Cenotaph for Newton / Etienne-Louis Boullée|accessdate=2016-10-04|date=2014-09-10|language=en-US}}</ref><ref>{{Cite web|url=http://www.archdaily.com/544946/ad-classics-cenotaph-for-newton-etienne-louis-boullee|title=AD Classics: Cenotaph for Newton / Etienne-Louis Boullée|accessdate=2016-10-04|date=2014-09-10|language=en-US}}</ref> 这种形式的纪念碑、纪念堂并不是墓葬，而是用于纪念实际葬在别处的人的。代表牛顿的空石棺放在球体的最低点处。纪念堂的设计创造了白日与黑夜的效果：弧形天顶上有许多小洞，当石棺被穿过这些小洞的阳光照亮的时候，人仿佛看见了夜空中的星星，这就产生了黑夜的效果。而白日的效果则来自于悬挂在球体正中、散发出神秘光芒的[[環形球儀|浑仪]]。因此，在建筑设计中使用光线导致了建筑内部的外观发生变化。<ref>{{Cite web|url=http://www.bbc.com/future/story/20160318-the-memorial-to-newton-that-would-have-eclipsed-the-pyramids|title=The memorial to Newton ghat would have eclipsed the pyramids|accessdate=March 21, 2016|author=Adam Proctor|date=2016-03-16}}<code style="color:inherit; == 图罗尔酒店大厅 ==
1761年1月21日的《预言报》提到了酒店大厅中的护壁板，因此它一定是在1758至1759年间装修的。[[玛黑区|玛黑区]]改建而成的克劳德-查尔斯-多米尼克·图罗尔（Claude-Charles-Dominique Tourolle）酒店保存了下来（当时的奥尔兰街现在已经改名为夏尔洛街），但19世纪中叶时大厅中的护壁板和壁炉台被拆除，搬到了[[圣奥诺雷市郊路|圣奥诺雷市郊路]]上的一栋房子里，这栋建筑现在归[[因特拉利中心|因特拉利中心]]（Cercle Interallié）所有。壁炉台上方、长墙中心的凹处摆着边缘带凹槽的爱奥尼亚式壁柱，上面放着圆拱形的镜子。白色大理石的女像柱支撑着壁炉台的台面。齿状壁带下方有一圈完整的楣梁。这套白金配色的装潢在1790年还是很时髦的。

== 亚历山大酒店 ==
[[File:Immeuble,_16_rue_de_la_Ville-l'Évêque,_Paris_2013.jpg|右]]
亚历山大酒店，又称苏特酒店，位于巴黎城市主教街（1763年-1766年）。它是布雷在巴黎唯一保存下来的住宅设计，是为金融家安德鲁-克劳德-尼古拉·亚历山大建造的。<ref>The house passed to the marquis de Collonge, then to the [//en.wikipedia.org/wiki/Nicolas_Jean_de_Dieu_Soult maréchal Soult], from 1802 to 1818, whose name it now also sometimes bears.</ref> 酒店的[[Cour_d'honneur|荣誉庭院]]（cour d'honneur）中，四根嵌在墙壁平面凹处的爱奥尼亚柱创造了一个入口（现在已经装上了玻璃门）。庭院角落的侧门上方镶嵌了独立的框架，而椭圆形的牛眼窗上方则覆盖着布幔状的外饰，这后来成为了新古典主义风格建筑一个常见的特征。花园前竖立着一排高大的柱子，支撑着高高的天花板。

== 文化遗产 ==
布雷的想法对他同时代的建筑师产生了重大影响，不仅仅是因为他曾为一些重要的建筑师授课，例如{{Tsl|en|Jean Chalgrin|让·查尔格林}}、{{Tsl|en|Alexandre-Théodore Brongniart|亚历山大-西奥多尔·布龙尼亚特}}和{{Tsl|en|Jean-Nicolas-Louis Durand|让-尼古拉斯-路易·杜兰德}}。他的一些作品直到20世纪才为人所知；他的《论建筑艺术》（Architecture，essai sur l'Art）一书感情充沛地支持新古典主义，但直到1953年才出版。这本书收录了他从1778年到1788年的作品，大部分都是规模过于宏大、以至于完全不切实际的公共建筑设计。

布雷喜爱宏大的设计，这使得他看起来既像一个妄自尊大的人，又像一个空想家。他对对立（抵消相反的设计元素)的关注和对光影的使用十分创新，直到今天还对建筑师们有着深刻影响。他在20世纪被“重新发现”，并影响了[[阿尔多·罗西|阿尔多·罗西]]等建筑师。

{{Tsl|en|Peter Greenaway|彼得·格里纳韦}}的电影《{{Tsl|en|The Belly of an Architect|建筑师的肚子}}》（1987）讲述了一个虚构的建筑师为布雷的作品举办展览的故事。这部电影运用了许多具有布雷特点的视觉元素。
