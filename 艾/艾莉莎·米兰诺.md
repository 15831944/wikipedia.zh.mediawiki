'''艾莉莎·米兰诺'''（{{lang-en|Alyssa Milano}}；{{bd|1972年|12月19日|||catIdx= M米}}）出生于[[美国|美国]][[纽约州|纽约州]][[纽约市|纽约市]][[布鲁克林区|布鲁克林区]]，为美国女演员、歌手和制作人。

== 生平 ==
1978年，米兰诺出生于[[美国|美国]][[纽约州|纽约州]][[纽约市|纽约市]][[布鲁克林区|布鲁克林区]]，艾丽莎米兰的母亲叫“林”，一个时装设计师和人才经理，父亲托马斯·M·米兰诺，是个电影音乐编辑人。<ref name="BIO">{{cite web|url=http://www.filmreference.com/film/43/Alyssa-Milano.html|title=Alyssa Milano Biography (1974–)|work=Filmreference.com|accessdate=2003-01-23}}</ref>
米兰诺成长于纽约市[[史泰登岛|史泰登岛]]。
她有一个弟弟叫科里（出生于1982年），同时也是一个演员。
在2003年的一次采访中，米兰回忆起童年时光时说，她成长于一个“温馨的、真实的家庭。”<ref name="intimate">{{Cite episode| title = Intimate Portrait: Alyssa Milano| url = http://www.imdb.com/title/tt0367888/| series = [[Intimate_Portrait_(TV_series)|Intimate Portrait]]| credits = Suju Vijayan, Julie Frankel, Alyssa Milano| network = [[Lifetime_(TV_network)|Lifetime]]| airdate = 2003}}</ref>
米兰诺的堂兄是[[艾米丽·安·劳埃德|艾米丽·安·劳埃德]]和[[埃里克·劳埃德|埃里克·劳埃德]]，两人也是演员。
