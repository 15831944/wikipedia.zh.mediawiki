== 简介 ==
艾斯黛拉·沃伦出生在[[加拿大|加拿大]][[安大略省|安大略省]]彼得堡，她的母亲叫伊斯特，是一所小学的[[校长|校长]]，父亲叫东·沃伦，是一个二手车经销商，她是家中三个孩子中最年轻的女儿。<ref>{{cite web |url=http://www.filmreference.com/film/54/Estella-Warren.html |title=Estella Warren Biography (1978-) |accessdate=7 January 2008}}</ref>
