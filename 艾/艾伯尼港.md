'''艾伯尼港'''（{{lang-en|Port Alberni}}），當地華人舊譯'''砵亞板汝'''，是[[加拿大|加拿大]][[不列颠哥伦比亚省|不列颠哥伦比亚省]]西南部的一座城市，坐落[[溫哥華島|溫哥華島]]中部艾伯尼内灣（{{lang|en|Alberni Inlet}}）的盡頭，離省府[[維多利亞_(卑詩)|維多利亞]]西北約195公里。據2011年加拿大人口普查所示，艾伯尼港市內人口為17743人，人口密度每平方公里897.9人<ref name="2011census" />；都會區人口則達25465人<ref name="2011censusCA">{{Cite web |url=http://www12.statcan.gc.ca/census-recensement/2011/dp-pd/prof/details/page.cfm?Lang=E&Geo1=CMA&Code1=940&Geo2=PR&Code2=59&Data=Count&SearchText=port%20alberni&SearchType=Begins&SearchPR=59&B1=All&Custom=&TABID=1 |title=Census Profile - Census Agglomeration |accessdate=2014-02-25 |work=[[加拿大統計局|加拿大統計局]] |date=2014-01-08 }}</ref>。該市是[[艾伯尼-格里夸地区|艾伯尼-格里夸地区]]的行政中心。

== 歷史 ==
艾伯尼港的名稱來自艾伯尼内灣，而内灣則以[[西班牙|西班牙]]軍隊的艾伯尼隊長（{{lang|es|Pedro de Alberni}}）命名<ref name="GeoBCPA"/>；艾伯尼於1790年至1792年間駐守西班牙設於溫哥華島西岸努特卡灣的聖米格爾堡（{{lang|es|Fuerte San Miguel}}）。[[哈德遜灣公司|哈德遜灣公司]]的皮毛交易員於1850年代與該帶的原住民展開交易，而一間英國船運公司則於1860年在内灣盡頭設立一座鋸木廠<ref name="CE">{{Cite web |url=http://www.thecanadianencyclopedia.com/en/article/port-alberni/ |title=Port Alberni |accessdate=2014-02-25 |work=''The Canadian Encyclopedia'' }}</ref>。歐裔殖民於1880年代初相繼遷至此處，開墾這裏的土地並展開務農<ref name="PAhist">{{Cite web |url=http://www.portalberni.ca/centennial_history.html |title=History |accessdate=2014-02-25 |work=艾伯尼港市政府 |deadurl=yes |archiveurl=https://web.archive.org/web/20140302140338/http://www.portalberni.ca/centennial_history.html |archivedate=2014-03-02 }}</ref>，但當中不少人後來轉投伐木業<ref name="CE"/>。

艾伯尼郵局和賽沃德－艾伯尼郵局（{{lang|en|Sayward Alberni Post Office}}）分別於1885年和1886年在現艾伯尼港市北部開業；前者於1891年關閉後，後者於同年改稱艾伯尼郵局。離該處以南一英里的新艾伯尼郵局於1900年開業，再於1910年改稱艾伯尼港郵局<ref name="GeoBCPA"/>。艾伯尼港郵局一帶的聚落於1912年3月12日正式建制為艾伯尼港市<ref name="GeoBCPA">{{Cite web |url=http://apps.gov.bc.ca/pub/bcgnws/names/19255.html |title=BC Geographical Names: Port Alberni (city) |accessdate=2014-02-25 |work=GeoBC }}</ref>，而艾伯尼郵局一帶的聚落則於1945年8月13日設為艾伯尼市<ref name="GeoBCAlb">{{Cite web |url=http://apps.gov.bc.ca/pub/bcgnws/names/541.html |title=BC Geographical Names: Alberni |accessdate=2014-02-25 |work=GeoBC }}</ref>。

1964年3月27日，[[阿拉斯加|阿拉斯加]]發生地震（史稱[[耶穌受難日地震|耶穌受難日地震]]），所引發的[[海嘯|海嘯]]兩度衝擊艾伯尼港一帶。該帶375間房屋受損，當中55間房屋被沖走，但該帶沒有居民傷亡<ref name="CE"/>。

1967年10月，艾伯尼港市和艾伯尼市合併成新的艾伯尼港市<ref name="GeoBCPA"/>。

== 交通 ==
艾伯尼港市內的主要幹道為卑詩省4號公路，向西通往[[吐芬奴|吐芬奴]]（經支路伸延至[[尤克盧利特|尤克盧利特]]），向東伸延至溫哥華島東岸並駁上其他公路通往[[帕克斯維爾|帕克斯維爾]]和[[乃磨|乃磨]]等地。航空交通方面，艾伯尼港（艾伯尼谷地區）机场（{{lang|en|Port Alberni (Alberni Valley Regional) Airport}}，IATA代碼：YPB）位於艾伯尼港西北11公里，提供航班前往[[溫哥華|溫哥華]]。

== 地理 ==
艾伯尼港位于艾伯尼内灣盡頭；該處也是蘇馬斯河（{{lang|en|Soomas River}}）匯入艾伯尼内灣所在。

=== 气候 ===
艾伯尼港屬[[地中海式氣候|地中海式氣候]]（[[柯本氣候分類法|柯本氣候分類法]]Csb）<ref>{{Cite web |url=http://en.climate-data.org/location/10668/ |title=Climate: Port Alberni |accessdate=2014-02-25 |work=climate-data.org }}</ref>，冬季多雨，夏季乾燥炎熱。
