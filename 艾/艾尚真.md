'''艾尚真'''（{{lang-en|'''Charmaine Xu'''}}，{{bd|1986年|9月9日|||catIdx= X徐}}）原名'''徐萌'''，[[重庆人|重庆人]]，毕业于[[武汉大学|武汉大学]]，2008年[[國際中華小姐競選|國際中華小姐競選]]季軍，獲稱“中国第一黄金比例”。<ref>{{cite web |url=http://www.chinadaily.com.cn/hqgj/jryw/2012-02-16/content_5168950.html |title=中国第一黄金身材 重庆姑娘艾尚真 |accessdate= 2012年|work= |publisher=中国日报网 |language =zh-cn}}</ref><ref>{{cite web |url= http://roll.sohu.com/20120221/n335371570.shtml|title= 艾尚真网络爆红客串车模 黄金比例身材最全写真|accessdate= 2012年|work= |publisher=搜狐网 |language =zh-cn}}</ref>

== 简介 ==
2010年代，艾尚真返回[[中国大陆|中国大陆]]发展事业，易名為艾尚真，表示“艾尚真的含义是：爱心，时尚，高尚，真诚。我想在我的工作道路上坚持这个原则。”<ref name="wyi">{{cite web |url=http://news.163.com/12/0217/07/7QER9NLB00014AED.html |title=艾尚真：我不愿意当花瓶 |accessdate= 2012年|work= |publisher=网易 |language =zh-cn}}</ref>
