'''郎世寧'''，原名'''朱塞佩·伽斯底里奧內'''（{{lang-it|'''Giuseppe Castiglione'''}}，{{bd|1688年|7月19日|1766年|7月17日}}），[[義大利人|義大利人]]，[[天主教|天主教]][[耶穌會|耶穌會]][[傳教士|傳教士]]及[[中國|中國]][[清朝宫廷画家列表|宮廷畫家]]，寫實畫風紀錄了[[清朝|清朝]]初期的中國宮廷人物與景色。

== 生平 ==
[[郎世寧|郎世寧]]本名為朱塞佩·伽斯底里奧內（朱塞佩·卡斯提永尼）（Giuseppe Castiglione），出生於義大利[[米蘭公國|米蘭公國]]的[[聖馬爾切利諾|聖馬爾切利諾]]，自幼受到良好的藝術文化氛圍的薰陶，青年時期隨[[卡洛科納拉|卡洛科納拉]]學習繪畫與建築。郎世寧年輕時進入一個米蘭著名畫家的畫室。郎世寧的父母了解他對於繪畫的喜愛，所以他們幫助他進入畫室。文獻是說當時五六歲的郎世寧既進入著名畫家的畫室，但文獻卻不說畫師是誰。我們猜測是[[Filippo_Abbiati|Filippo Abbiati]]或是[[卡洛科納拉|卡洛科納拉]]，當時最有名的[[畫家|畫家]]之一，那位畫師收到來自[[米蘭|米蘭]]的[[聖堂|聖堂]]以及[[貴族宮殿|貴族宮殿]]的最重要委託。六歲的郎世寧在畫室開始打工，磨顏色，摹仿老師的畫，還複製在米蘭收集成的畫。在1707年，19歲時左右加入了[[熱那亞|熱那亞]][[耶穌會|耶穌會]]。剛開始只為義大利的[[教堂|教堂]]畫[[壁畫|壁畫]]，1714年居往在[[葡萄牙|葡萄牙]][[里斯本|里斯本]]及[[科英布拉|科英布拉]]。逐漸對中國產生了相當大的興趣，1715年，前往中國。期間曾於[[澳門|澳門]]學習中文、並以「[[郎世寧|郎世寧]]」作為漢名<ref name="童超－49.1" /><ref name="高木森－6.1.2"/>。

康熙五十四年（1715年）到中國時，郎世寧被[[康熙帝|康熙帝]]以[[藝術家|藝術家]]的身份召進宮中，他成為宮廷畫家後，每天都在畫室作畫。此外，他繼續學習漢文和滿文。康熙六十一年（1722年），康熙帝駕崩。[[雍正帝|雍正帝]]繼位後，施行禁教政策，下令驅逐在中國境內的傳教士，不過只有在宮廷服務的傳教士例外，因此他不僅沒有受到驅逐，反而受到更高的禮遇，因為他擁有生動現實的繪畫風格，受到雍正帝的青睞。這一段時期，他創作了《[[聚瑞圖|聚瑞圖]]》、《[[嵩獻英芝圖|嵩獻英芝圖]]》、《[[八駿圖|八駿圖]]》等作品，展示歐洲明暗畫法的風格，使歐洲油畫技藝在宮廷內盛極一時。雍正帝下令擴建[[圓明園|圓明園]]，他參與圓明園的規劃設計，負責繪製園林建設的藍圖，也畫出不少裝飾殿堂的作品<ref name="童超－49.1" /><ref name=" 郎世寧宮廷畫"/>。但來自康熙的委任工作大部分都丟了，只有很少數的還在，我們不太知道當時郎世寧畫了什麼。雍正時，郎世寧完成一幅最重要畫作之一。這幅畫是郎世寧在北京開始的。那幅委任畫作就是一幅“[[百駿圖|百駿圖]]”的手卷，擺在桌上的手卷裡，可以看見風景和百匹馬，那些全是皇帝的馬。在百駿圖中，第一幅見證畫，郎世寧馬上讓大家看自己的獨創性。他的獨創是聯合義式方法與中國的美學。郎世寧從劇場畫法的透視感，呈現出百駿圖的風光。這是一個具有不同重點的透視感，而不是一個線性的透視。郎世寧的方法，是用交點透視(linear perspective)，或稱直線透視法，讓透視視覺跟中華畫空間的方式結合。所以，一幅畫裡也有很多案例，有的部份遵照透視的規定，也有別的部份遵照等角投影(axonometric isometric projection)，中國畫家用的，或在別處遵照空氣透視(atmospheric perspective)，用水墨，液體的顏料。例如用在畫像和顏色的對照。

雍正十三年（1735年），[[雍正|雍正]]帝駕崩。[[乾隆帝|乾隆帝]]繼位後，由於乾隆帝雅好藝術，郎世寧更加受到重用。他曾多次陪同乾隆帝出巡，出席各種重大場合，創作一批極具歷史價值的藝術珍品，代表作如《[[乾隆大阅图|乾隆大阅图]]》。他繼續擔任[[圓明園|圓明園]]西洋樓的建築設計，還曾一度擔任掌管皇家園林工作的[[奉宸苑|奉宸苑]]苑卿一職，官居[[正三品|正三品]]<ref name="童超－49.1" />。

乾隆二十三年（1758年），[[乾隆|乾隆]]帝曾為[[郎世寧|郎世寧]]舉辦七十歲大壽，證明他在宮中頗受禮遇及恩寵。晚年亦為[[乾隆|乾隆]]帝及其妃留下不少的肖像。乾隆三十一年（1766年），郎世寧去世，享年78歲，再追封[[侍郎|侍郎]]銜。葬於[[滕公柵欄|滕公柵欄]]<ref name="童超－49.3" />。

在他之後的西洋傳教士畫家有[[王致誠|王致誠]]、[[艾啟蒙|艾啟蒙]]（Ignaz Sichelbarth）、[[賀清泰|賀清泰]]、[[潘廷章|潘廷章]]等。

今日[[郎世寧|郎世寧]]的畫作在中國畫拍賣市場中是相當高價的作品，2000年他的《[[蘋野秋鳴|蘋野秋鳴]]》賣出價是1765.5萬港幣，是當時第二高價賣出的中國畫，2005年5月，他的《[[秋林群鹿|秋林群鹿]]》更是拍賣出的價格達到2149.68萬人民幣<ref name=" 郎世寧身價"/>。

== 傳教 ==
郎世寧雖然最初以傳教士的身份來中國，但在[[康熙|康熙]]、[[雍正|雍正]]、[[乾隆|乾隆]]三代對[[基督宗教|基督宗教]]的壓制中，他並沒有因此被逐出[[紫禁城|紫禁城]]，反而極受禮遇，有很大一部分的原因是，他在[[清朝|清朝]]宮中擁有宮廷畫師的職位，儘管他本人始終不曾忘記傳教的想法，然而在禁教政策之下，凡是有任何企圖傳教的舉止，都會被禁止，使得他終生未能實現傳教的願望，因此在中國，多稱他為[[宮廷畫家|宮廷畫家]]，而少稱他為傳教士<ref name="童超－49.2" /><ref name="高木森－6.1.2"/><ref name=" 說說郎世寧"/>。

== 評價 ==
郎世寧在一生大半待在中國，歷代[[康熙|康熙]]、[[雍正|雍正]]、[[乾隆|乾隆]]三朝，計約有50餘年。他不僅為後世留下大量珍貴的畫作，而且對中西文化交流有一定程度的貢獻。他的宮廷畫作品融合中西繪畫藝術，不但表現出西方繪畫中的寫實、立體、注重線條和結構等特點，也表現出中國傳統繪畫的傳神與韻味，在數十年的探索中，致力於融合中西方繪畫技法於一體，創作出獨特的畫風，儘管擁有一定的藝術價值，然而他的繪畫作品較為重視技巧而非藝術的探索，因此不論在中國和西方均未被視為偉大的作品。另外他參與[[圓明園|圓明園]]的規劃設計，其內部的建築、環境和式樣採用了西方藝術巴洛克式風格，而裝飾細節上則有具有東方中國的藝術神韻。他引進西方[[文藝復興|文藝復興]]時期開創的明暗寫實畫法，並改用膠狀顏料在[[宣紙|宣紙]]上作畫，也就是今日的[[膠彩畫|膠彩畫]]作法。他曾試圖要求康熙帝開辦學習用西方[[透視|透視]]原理來繪畫的繪畫學校，但不被採用，後來與中國學者[[年希堯|年希堯]]一起出版了一本《[[視學|視學]]》，是中國第一部西洋透視學專著。<ref name="童超－49.2" /><ref name="高木森－6.1.2"/><ref name=" 說說郎世寧"/>。

== 代表作品 ==
郎世寧的代表作品有《[[八駿圖|八駿圖]]》、《[[聚瑞圖|聚瑞圖]]》、《[[嵩獻英芝圖|嵩獻英芝圖]]》、《[[乾隆大阅图|乾隆大阅图]]》{{备注|1=此畫描繪乾隆四年（1739年）乾隆帝在京效南苑閱兵的情況<ref name="童超－49.2" />。}}、《[[百駿圖|百駿圖]]》{{备注|1=此畫描繪駿馬遊牧嬉戲的場面<ref name="童超－49.2" />。}}、《[[弘曆及后妃像|弘曆及后妃像]]》、《[[平定西域戰圖|平定西域戰圖]]》、《[[十駿犬圖|十駿犬圖]]》等。後世藝評家認為他畫[[馬|馬]]最為傳神<ref name=" 說說郎世寧"/><ref name=" 郎世寧身價"/>。
