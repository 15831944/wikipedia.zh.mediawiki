== 生平 ==
1971年7月23日，郎永淳出生于[[江苏省|江苏省]][[徐州市|徐州市]][[睢宁县|睢宁县]]。1989年起，郎永淳在[[南京中医学院|南京中医学院]][[针灸|针灸]]专业学习五年。1994年毕业后，郎永淳考入[[北京广播学院|北京广播学院]]（[[中国传媒大学|中国传媒大学]]的前身）播音系。1995年，郎永淳在[[中国中央电视台|中国中央电视台]]主持《[[新闻30分|新闻30分]]》节目<ref name="cat" />。2011年8月，郎永淳和[[欧阳夏丹|欧阳夏丹]]成为《[[新闻联播|新闻联播]]》主播，2011年9月25日两人首次亮相《新闻联播》<ref>{{cite web|url=http://news.qq.com/a/20110809/001426.htm|title=《新闻联播》新主播郎永淳欧阳夏丹下周上岗|publisher=腾讯|date=2011-08-09|work= |language=zh-cn}}</ref><ref>{{Cite web |url = http://www.chinanews.com/cul/2014/11-18/6788755.shtml |title = 郎永淳：第一次播《新闻联播》吓出一身冷汗 |author =  |date = 2014-11-18 |publisher = 中国新闻网 |language =  |accessdate =  |quote =  }}</ref>。他还先后读了[[湖南大学|湖南大学]]工商管理学院管理科学与工程专业，[[中国人民大学|中国人民大学]][[EMBA|EMBA]]，[[北京大学光华管理学院|北京大学光华管理学院]][[EBA|EBA]]项目<ref name=mengxiang>{{Cite web |url = http://news.youth.cn/jsxw/201703/t20170326_9354061.htm |title = "同学,让我们来聊聊梦想" |author =  |date =2017-03-26  |publisher = 中国青年网 |language =  |accessdate =  |quote =  }}</ref>。

2015年9月2日，有知情人士爆料，郎永淳已经从中国中央电视台辞职，离开传媒界，当天他最后一次主持了《新闻联播》<ref>{{Cite web |url = http://news.ifeng.com/a/20150902/44576036_0.shtml |title = 知情人士：央视主持人郎永淳已辞职 |author =  |date = 2015-09-02 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。郎永淳的一位好友在接受媒体采访时称，郎永淳把家庭看得很重，他辞职是为了照顾患[[乳腺癌|乳腺癌]]的妻子和无人照料的孩子，是经过深思熟虑的决定<ref>{{Cite web |url =http://lady.people.com.cn/n/2015/0907/c1014-27550617-6.html  |title =郎永淳辞职央视后首亮将赴美陪妻儿 揭秘离开央视背后真正原因  |author =  |date =2015-09-07  |publisher =人民网  |language =  |accessdate =  |quote =  }}</ref>。辞职后，郎永淳投身钢铁行业<ref name=mengxiang/>。2016年1月15日，找钢网在上海举行新一轮战略融资发布会，郎永淳以找钢网高级副总裁兼首席战略官的身份亮相<ref>{{Cite web |url = http://money.163.com/16/0121/09/BDRFT9BV00253G87.html |title = 郎永淳：我为什么在钢铁的寒冬加入找钢网 |author =  |date = 2016-01-21 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

2017年10月5日，郎永淳[[酒后驾驶|酒后驾驶]]小型越野客车，在[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]][[西大望路|西大望路]][[八王坟|八王坟]]路口发生交通事故，10月6日因涉嫌危险驾驶罪被[[北京市公安局|北京市公安局]]朝阳分局依法[[刑事拘留|刑事拘留]]，10月16日被移送检察机关，郎永淳在检察环节认罪认罚，随后被起诉至[[北京市朝阳区人民法院|北京市朝阳区人民法院]]<ref>{{Cite web |url =http://tech.163.com/17/1027/14/D1OR4DIA00097U7R.html   |title =曝前央视名嘴郎永淳酒驾被拘 2年前离职去创业圈  |author =  |date = 2017-10-27 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.guancha.cn/society/2017_10_27_432507.shtml  |title =央视前名嘴郎永淳醉驾被北京警方拘留 曾主持《新闻联播》  |author =  |date = 2017-10-27 |publisher =观察者网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.ifeng.com/a/20171027/52824329_0.shtml |title = 郎永淳醉驾案新进展：认罪认罚 已被检察机关诉至法院 |author =  |date = 2017-10-27 |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。11月3日，北京市朝阳人民法院对郎永淳危险驾驶案公开开庭审理并当庭判决，以被告人郎永淳犯危险驾驶罪判处其拘役三个月，并处罚金四千元<ref>{{Cite web |url =  http://ent.ifeng.com/a/20171103/42999595_0.shtml|title =  郎永淳危险驾驶案宣判 拘役三个月罚金四千元|author =  |date =2017-11-03  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref>。
