'''芒市宾馆'''（[[英文|英文]]：Mangshi Hotel）是[[云南省|云南省]][[德宏傣族景颇族自治州|德宏傣族景颇族自治州]]的一家[[五星级酒店|五星级酒店]]，位于[[芒市|芒市]]城东部菩提街1号<ref>{{cite web |url=http://news.ifeng.com/gundong/detail_2014_02/12/33716965_0.shtml |title=三节合一 芒市宾馆不涨价 |last= |first= |date=2014-02-12 |website=凤凰网资讯 |publisher=云南信息报 |access-date=2017-10-21 |quote=}}</ref>。芒市宾馆共有309间客房和3个餐厅<ref group="注">中餐厅、咖啡厅和大堂吧</ref><ref>{{cite web |url=http://www.mshotel.cn/bggk.htm |title=宾馆概况 |last= |first= |date= |website= |publisher=芒市宾馆 |access-date=2017-10-21 |quote=}}</ref><ref>{{cite web |url=http://www.mshotel.cn/Dining.htm |title=餐饮 |last= |first= |date= |website= |publisher=芒市宾馆 |access-date=2017-10-21 |quote=}}</ref>，另有4个会议室和1个会见厅<ref>{{cite web |url=http://www.mshotel.cn/meet.htm |title=会议室 |last= |first= |date= |website= |publisher=芒市宾馆 |access-date=2017-10-21 |quote=}}</ref>。

==历史==

1956年，芒市宾馆建成开业。2003年，德宏汇合投资集团有限公司在原址投资重建宾馆，现占地60余亩，总建筑面积6万余平方米<ref name="芒市宾馆成为德宏首家五星级酒店">{{cite web |url=http://fashion.ifeng.com/travel/china/detail_2011_07/19/7789782_0.shtml |title=芒市宾馆成为德宏首家五星级酒店 |last= |first= |date=2011-07-19 |website=凤凰网资讯 |publisher=云南网 |access-date=2017-10-21 |quote=}}</ref>。2011年11月被全国旅游星级饭店评定委员会批复授予为五星级酒店，系德宏州第一家五星级酒店<ref name="芒市宾馆成为德宏首家五星级酒店"></ref>。2013年12月21日，芒市宾馆获得“德宏州旅游饭店业突出贡献奖”，并成为“云南省旅游饭店业协会副会长单位”，为德宏第一家获得该荣誉的宾馆<ref>{{cite web |url=http://news.ifeng.com/gundong/detail_2013_12/25/32444232_0.shtml |title=芒市宾馆斩获两殊荣创德宏60年历史 |last= |first= |date=2013-12-25 |website=凤凰网资讯 |publisher=云南信息报 |access-date=2017-10-21 |quote=}}</ref>。2014年8月，由于芒市突降暴雨使得芒市宾馆旁的[[板过河|板过河]]水位猛涨，挡墙倒塌，导致芒市宾馆建筑的部分地基下陷，事故共造成了近百万元的经济损失<ref>{{cite web |url=http://news.ifeng.com/a/20140831/41800014_0.shtml |title=板过河挡墙倒塌堵塞的河道现已疏通且险情已妥善处置 |last= |first= |date=2014-08-31 |website=凤凰网资讯 |publisher=云南信息报 |access-date=2017-10-21 |quote=}}</ref>。

==中缅友谊树==

'''中缅友谊长青树'''又称'''中缅友谊树'''，位于芒市宾馆内。1956年12月，[[中华人民共和国总理|中华人民共和国前总理]][[周恩来|周恩来]]和[[陈毅|陈毅]]、[[贺龙|贺龙]]到缅甸访问；1956年12月14日，周恩来与[[缅甸总理|缅甸前总理]]{{le|巴瑞|Ba Swe|吴巴瑞}}从[[畹町镇|畹町]]进入德宏<ref group="注">这是周恩来唯一一次从陆地口岸步行回国</ref>，到芒市举行中缅边民联欢大会。12月15日，两国总理在下榻的芒市宾馆亲手种植了象征[[中缅关系|中缅友谊]]长存的[[白兰|缅桂花树]]。后由德宏州人民政府自行修建“缅桂亭”和周围草坪雕像等设施。1996年，云南省政府把缅桂亭和中缅友谊长青树确定为云南省首批爱国主义教育基地<ref group="注">现已不再是爱国主义教育基地</ref>。<ref>{{cite book |last=张 |first=方元 |date=2000年5月 |title=新编德宏风物志 |url=http://f.wanfangdata.com.cn/view/%E4%B8%AD%E7%BC%85%E5%8F%8B%E8%B0%8A%E9%95%BF%E9%9D%92%E6%A0%91.aspx?ID=LocalChronicleItem_5921433 |location=[[昆明市|昆明]] |publisher=云南人民出版社 |page= P61-P62 |isbn=7-222-02968-0 |author-link= }}</ref>

2006年12月18日，原[[云南行政长官列表|云南省代省长]][[秦光荣|秦光荣]]和原{{le|商务部 (缅甸)|Ministry of Commerce (Myanmar)|缅甸联邦商务部}}部长{{le|丁乃登|Tin Naing Thein}}为芒市宾馆内的中缅友谊树揭碑<ref>{{cite web|url=http://www.dh.gov.cn/Web/_F0_9_061218120002B642E75E.htm|title=秦光荣、丁乃登为中缅友谊树揭碑|last=李|first=武周|date=2006-12-18|website=德宏州人民政府|publisher=德宏团结报|access-date=2017-10-21|quote=}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。
