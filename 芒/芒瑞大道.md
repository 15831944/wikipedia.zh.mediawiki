'''芒瑞大道'''是[[德宏傣族景颇族自治州|德宏傣族景颇族自治州]][[芒市|芒市]]至[[瑞丽市|瑞丽市]]的一条道路，定位为城市II级主干道和城市快速路。道路全长75.591公里，沿[[芒市河|芒市河]]、[[瑞丽江|龙江]]沿岸分布，也是一条景观道路，总投资概算90.273亿元，道路分两期建设，由瑞丽[[景成集团|景成集团]]承建。道路建成有助于推动[[芒瑞一体化|芒瑞一体化]]的发展。<ref name="德宏团结报"></ref><ref>{{cite web |url=http://dehong.ectpa.org/article/wlzx/201204/20120400018051.shtml |title=芒瑞大道即将投入建设 项目签约 |last= |first= |date=2012-04-12 |website=dehong.ectpa.org |publisher=孔雀之乡网 |access-date=2018-01-04 |quote=}}</ref><ref>{{cite web |url=http://www.mangshi.gov.cn/jrms/msnews/content-7-442-1.html |title=芒瑞大道要建成景观大道 |author=佟晓媛 |date=2012-05-18 |website=www.mangshi.gov.cn |publisher=新芒市 |access-date=2018-01-04 |quote=}}</ref><ref>{{cite web |url=http://www.dh.gov.cn/Web/_F0_0_28D0791P4837H9IETF9WD5YL6K.htm |title=芒瑞大道一期工程将于12月底通车 |last= |first= |date=2017-12-19 |website=www.dh.gov.cn |publisher=德宏州人民政府 |access-date=2018-01-04 |quote=}}</ref>

一期工程于2012年4月29日开工，起点为芒市[[风平镇|风平]][[风平佛塔|佛塔]]，终点为[[遮放镇|遮放]]贡米厂，坝区路基宽50米，山区路基宽28米，路上不设收费站，总投资34.93亿元，长38.66公里，工程规划设计桥梁58座、隧道3座、涵洞111道，分5个合同段实施。2017年12月30日，芒瑞大道一期工程建成通车。<ref name="德宏团结报">{{cite web |url=http://www.cnepaper.com/dhtjb/html/2018-01/03/content_1_2.htm |title=芒瑞大道一期工程正式通车 |author=段秋娟 |date=2018-01-03 |website=www.cnepaper.com |publisher=德宏团结报 |access-date=2018-01-04 |quote=}}</ref><ref>{{cite web |url=http://yn.yunnan.cn/html/2017-12/31/content_5033049.htm |title=芒瑞大道一期工程建成通车 全长38.66公里 |author=刘祥元 |date=2017-12-31 |website=yn.yunnan.cn |publisher=云南日报 |access-date=2018-01-04 |quote= }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
