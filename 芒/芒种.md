'''芒种'''，是[[节气|二十四节气]]之一，落在6月6日前后（5日－7日）。在該日，[[太阳|太阳]]到达[[黄经|黄经]]75度<ref name="羅東鎮農會">{{cite web|url=http://www.24solar.tw/24solar-9.htm|title=24節氣-芒種|publisher= 羅東鎮農會|accessdate=2016-06-03|language=zh-tw|archiveurl=https://web.archive.org/web/20150425042756/http://www.24solar.tw/24solar-9.htm|archivedate=2015-04-25|deadurl=no}}</ref>。芒種一詞之意，是[[黃河|黃河]]流域地區在此時稻子已結實成「種」，而結實的稻子榖粒上長出了細芒<ref name="羅東鎮農會"/><ref name="mdnkids">{{cite web|url=http://www.mdnkids.com.tw/24seasons/spring6.htm|title=☆ 節氣大搜尋－芒種|publisher=財團法人國語日報社|accessdate=2016-06-03|language=zh-tw|archiveurl=https://web.archive.org/web/20150826155737/http://www.mdnkids.com.tw/24seasons/spring6.htm|archivedate=2015-08-26|deadurl=no}}</ref><ref name="CWB"/><ref name="toutiao"/>。芒種也是農作物種植時間的分界點。到了此時，[[華南|華南]]地區的[[梅雨|梅雨]]季節即將結束，轉變為午後雷陣雨。天氣也會逐漸轉為乾熱，使得不適合夏日栽種的農作物的生長率、存活率相對降低<ref name="mdnkids"/>。而[[长江中下游|长江中下游]]以及[[日本|日本]]南部地区則在此時进入梅雨季节<ref name="大方廣"/><ref name="農曆網"/><ref name="北京一中"/><ref name="天氣網">{{cite web|url=http://www.weather.com.cn/zt/tqzt/09mzzt/06/62420.shtml|title=芒种期间天气特点|publisher=中国天气网|date=2011-05-31|accessdate=2016-06-03|language=zh-cn|archiveurl=https://web.archive.org/web/20150615041043/http://www.weather.com.cn/zt/tqzt/09mzzt/06/62420.shtml|archivedate=2015-06-15|deadurl=no}}</ref>。而太陽也在芒種當天直射[[澳門|澳門]]地區<ref>{{cite news|title=今日芒種太陽直射澳門|url=http://news.qoos.com/今日芒種太陽直射澳門-2059667.html|publisher=[[澳門日報|澳門日報]]|author=楊永權|date=2016-06-05|accessdate=2016-06-06|language=zh-mo|archiveurl=https://web.archive.org/web/20160606034459/http://news.qoos.com/%E4%BB%8A%E6%97%A5%E8%8A%92%E7%A8%AE%E5%A4%AA%E9%99%BD%E7%9B%B4%E5%B0%84%E6%BE%B3%E9%96%80-2059667.html|archivedate=2016-06-06|deadurl=no}}</ref>。

== 物候==
芒种在大中華地區分為三[[七十二候|候]]，即：
*螳螂生：[[螳螂|螳螂]]破卵而出。
*鶰始鸣：[[伯勞鳥|伯勞鳥]]開始在枝頭出現，並且開始鳴叫。
*反舌无声：[[反舌鳥|反舌鳥]]停止了鳴叫。<ref name="大方廣"/><ref name="漢典">{{cite web|url=http://hl.zdic.net/msjr/jieqi/9.html|title=芒種 二十四節氣|publisher=漢典|accessdate=2016-06-03|language=zh-hant|archiveurl=https://web.archive.org/web/20150511175001/http://hl.zdic.net/msjr/jieqi/9.html|archivedate=2015-05-11|deadurl=no}}</ref><ref name="農曆網"/><ref name="中华贾姓"/>
至於日本，則在[[江戶時代|江戶時代]]由曆學家[[涉川春海|涉川春海]]改訂了七十二候，使之符合日本之氣候風土。因此，芒種在日本分為以下三候：
*螳螂生：[[螳螂|螳螂]]破卵而出。
*腐草為螢：[[螢火蟲|螢火蟲]]開始出現。
*梅子黃：[[梅子|梅子]]果實黃熟。<ref>{{cite book|script-title=ja:年中行事事典|pages=360|date=1958-05-23|author=西角井正慶|publisher=東京堂|language=ja}}</ref><ref>{{cite web|url=https://www.bioweather.net/column/aroma/011.html|script-title=ja:『蟷螂生・腐草為螢・梅子黄』な季節|publisher=BioWeather Service|date=2013-06-10|accessdate=2016-06-03|language=ja|archiveurl=https://web.archive.org/web/20160411085618/http://bioweather.net/column/aroma/011.html|archivedate=2016-04-11|deadurl=no}}</ref>

==農漁業==
芒種前後，[[臺灣|臺灣]][[基隆|基隆]]、[[澎湖|澎湖]]與[[臺南|臺南]]外海會有大量[[小卷|小卷]]漁獲、[[高雄|高雄]]外海和[[巴士海峽|巴士海峽]]則會出現[[鰹魚|鰹魚]]群、而[[小琉球|小琉球]]與[[蘇澳|蘇澳]]外海則可捕獲[[飛魚|飛魚]]。至於當令水果則有[[西瓜|西瓜]]、[[荔枝|荔枝]]、[[芒果|芒果]]、[[鳳梨|鳳梨]]等<ref name="mdnkids"/><ref name="農委會">{{cite web|url=http://www.coa.gov.tw/view.php?catid=1612|title=芒種（國曆6月5或6或7日）|publisher=[[行政院農業委員會|行政院農業委員會]]|accessdate=2016-06-03|language=zh-tw|archiveurl=https://web.archive.org/web/20120416054649/http://www.coa.gov.tw/view.php?catid=1612|archivedate=2012-04-16|deadurl=no}}</ref>。[[水稻|水稻]]則進入穀粒充實期<ref name="CWB"/>。而[[中國大陸|中國大陸]]的[[四川盆地|四川盆地]]則在此時結束了麥收季節，[[中稻|中稻]]與[[紅苕|紅苕]]的移栽也接近尾聲<ref name="漢典"/><ref name="農曆網"/>。[[华北|华北]]地區的一般麥田也在此時開始收割，中國東北地區也插完了稻秧<ref name="中华贾姓"/>。

==氣候==
芒種時節，[[長江|長江]]沿岸多雨，而[[黄淮平原|黄淮平原]]也開始進入雨季，芒種前後如果有持續多日的陰雨天氣以及大風、[[冰雹|冰雹]]的話，常使小麥無法及時收割，導致農業損失<ref name="中华贾姓"/>。芒種之後，长江中下游地区進入梅雨季节，日照時數減少，雨量增加，偶有低溫情形，至於[[華南|華南]]地區則進入一年降水最多的时节。而中國西南地區也在芒種時節前後進入了多雨時節，該地區西部的高原冰雹現象也逐漸增多<ref name="中华贾姓"/>。中國東北的[[嫩江|嫩江]]流域在1971年至2000年的30年间最热的一天，也出現在芒种期间，高達37.1°C<ref name="中华贾姓"/>。1981年至2010年的氣象觀測資料顯示，[[臺灣|臺灣]]在芒種當日的平均氣溫為26.8°C、平均最高氣溫為30.4°C、而平均最低氣溫則為24.0°C<ref name="CWB">{{cite book|title=臺灣24節氣與氣候---1981～2010資料統計|publisher=[[交通部中央氣象局|交通部中央氣象局]]|date=2011年8月|ISBN=978-986-028-376-1|url=http://www.cwb.gov.tw/V7/service/notice/download/publish_20110923100320.pdf|accessdate=2016-06-03|language=zh-tw|archiveurl=https://web.archive.org/web/20151106014449/http://www.cwb.gov.tw/V7/service/notice/download/publish_20110923100320.pdf|archivedate=2015-11-06|deadurl=no}}</ref>。至於[[香港|香港]]在同樣時間範圍的平均氣溫則為27.3°C、平均最高氣溫為29.6°C、而平均最低氣溫則為25.6°C<ref name="HKO">{{cite web|url=http://www.hko.gov.hk/cis/statistic/ext_st_corn_on_ear_c.htm|title=由香港天文台錄得的芒種氣候資料|publisher=[[香港天文台|香港天文台]]|accessdate=2016-06-03|language=zh-hk|archiveurl=https://web.archive.org/web/20151231070059/http://www.hko.gov.hk/cis/statistic/ext_st_corn_on_ear_c.htm|archivedate=2015-12-31|deadurl=no}}</ref>。[[日本|日本]]南部地区也在此時进入了梅雨季节<ref name="天氣網"/>。[[沖繩|沖繩]]地區也在[[小滿|小滿]]至芒種這段期間進入梅雨季，也因此梅雨被當地居民稱做「小滿芒種」<ref>{{cite book|script-title=ja:風と波を知る101のコツ: 海辺の気象学入門|chapter=-{夏を呼ぶ「カーチベー」と秋の訪れ告げる「ミーニシ」}-|pages=76|publisher=枻出版社|author=森朗|date=1998-04-20|accessdate=2016-06-03|language=ja|ISBN=978-487-099-164-4}}</ref>。

==習俗==
中國古代在芒種日當天，民間會有祭祀花神的儀式，以餞送花神歸位，並表達對花神的感激之意。《[[紅樓夢|紅樓夢]]》中也有描述此習俗之景況<ref name="漢典"/><ref name="toutiao"/><ref name="北京一中"/>。而[[皖南|皖南]]地區則在芒種時節有安苗的習俗。該習俗始于明初，是為了在種完水稻後祈求秋季有好收成，以新麥麵蒸發包，將麵捏成蔬果、穀物與家畜的形狀，並用蔬菜汁將其染色作為供品<ref name="漢典"/><ref name="toutiao"/>。至於[[贵州省|贵州省]][[黔東南苗族侗族自治州|黔東南苗族侗族自治州]][[黎平县|黎平县]]的[[侗族|侗族]]則在芒種日有打泥巴仗的習俗，當日新婚夫妇一同至田间插秧，並在插完秧苗後抓起田裡的泥巴互相投掷<ref name="toutiao"/>。[[河北省|河北省]][[盐山县|盐山县]]则在这天有“嫁树”的习俗，用刀在[[枣树|枣树]]上划几下，以求多结果实<ref name="中华贾姓"/>。[[日本|日本]]各地在芒種時也有各種祈求豐收的農業祭祀活動<ref>{{cite web|url=http://www.tenki.jp/suppl/saijiki_shuuka/2015/06/06/3851.html|script-title=ja:二十四節気「芒種（ぼうしゅ）」、田植えにまつわる祭事を紐解くと…|date=2015-06-06|publisher=tenki.jp|accessdate=2016-06-03|language=ja|archiveurl=https://web.archive.org/web/20150607080332/http://www.tenki.jp/suppl/saijiki_shuuka/2015/06/06/3851.html|archivedate=2015-06-07|deadurl=no}}</ref>。

==俗諺==
===臺灣===
#「芒種夏至，檨仔（芒果）落蒂」：臺灣南部的芒果會在芒種前後採收並上市<ref name="羅東鎮農會"/><ref name="mdnkids"/><ref name="農委會"/>。
#「芒種蝶仔討無食」：芒種時，許多花種的花期已過，導致蝴蝶沒有大量花粉可以採食<ref name="羅東鎮農會"/><ref name="mdnkids"/><ref name="農委會"/>。
# 「四月芒種雨，五月無乾土，六月火燒埔」：芒種日若降雨，就會持續下到農曆五月，導致沒有任何土地是乾燥的。到了農曆六月則會轉為炎熱的天氣<ref name="羅東鎮農會"/><ref name="農委會"/>。
# 「芒種瘋鯊」：臺灣[[鹿港|鹿港]]一帶海域在芒種前後會有[[鯊魚|鯊魚]]出現<ref name="羅東鎮農會"/><ref name="農委會"/>。
# 「芒種逢雷美亦然，端陽有雨是豐年」：芒種時，若有大雷雨降臨，即為豐收的預兆<ref name="mdnkids"/><ref name="農委會"/>。

===中國大陸===
# 「芒種夏至天，走路要人牽；牽的要人拉，拉的要人推」：[[江蘇|江蘇]]、[[安徽|安徽]]、[[四川|四川]]、[[湖北|湖北]]、[[貴州|貴州]]、[[江西|江西]]等地俗諺。有指夏季炎熱的天氣使人們變得懶散者<ref name="大方廣"/>，亦有指因阴雨天多容易滑倒者<ref name="天氣網"/>。
# 「芒種忙忙栽」：四川俗諺，指該地中稻與紅苕移栽在芒種時接近尾聲，若栽種過遲將導致產量降低<ref name="漢典"/>。
# 「芒种夏至是水节，如若无雨是旱天」：[[廣東|廣東]]俗諺，指這段期間若無雨則將乾旱<ref name="天氣網"/>。
# 「芒种夏至，水浸禾田」：廣東俗諺，指這段期間雨多<ref name="天氣網"/>。
# 「芒种落雨，端午涨水」：[[湖南|湖南]]俗諺，指這段期間雨多且容易淹水<ref name="天氣網"/>。
# 「芒种夏至常雨，台风迟来；芒种夏至少雨，台风早来」：[[福建|福建]]俗諺，說明芒種雨量與颱風之關係<ref name="天氣網"/>。

== 逸事 ==
