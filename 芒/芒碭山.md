'''芒碭山'''，位於[[河南省|河南省]][[永城市|永城市]]北部的黃淮大平原上，屬皖北丘陵的西延部分，有「仙女峰」之稱。<ref>{{web reference|URL=http://news.takungpao.com.hk/mainland/topnews/2017-09/3498818.html|title=秦漢靈韻藏一山|work=大公報|year=2017年|date=9月29日}}</ref>芒碭山海拔156.8米高，由10多座小山頭如保安山、僖山、夫子山、鐵角山、黃土山、周山等組成。以[[漢梁王墓群|漢梁王墓群]]和[[漢高祖|漢高祖]][[劉邦|劉邦]]斬白蛇一事而聞名。<ref name="mang">{{Cite web |url=http://www.big5.henan.gov.cn/hngk/system/2006/08/11/010001866.shtml |title=漢墓累累芒碭山 |access-date=2012-07-08 |archive-url=https://web.archive.org/web/20130405231117/http://www.big5.henan.gov.cn/hngk/system/2006/08/11/010001866.shtml |archive-date=2013-04-05 |dead-url=yes }}</ref>而芒碭山群的總面積超過10平方公里。[[中國|中國]][[中央電視台|中央電視台]]曾於2000年播放過一部合共6集的電視連續劇《追蹤309》，正是以永城芒碭山為背景。永城芒砀山旅游区（包括：汉梁王陵景区、刘邦斩蛇处景区、大汉雄风景区、芒砀山地质公园）在2017年，它成为河南省第13个[[国家5A级景区|国家5A级景区]]。<ref >{{cite web|url=http://www.jinbw.com.cn/shehuinews/2017-02-19/n_148748158349457.html | language= | title=好消息！永城芒砀山旅游区将晋升为国家5A级景区 |publisher=猛犸新闻客户端 |date=2017-02-19 |accessdate= }}</ref>

== 歷史 ==
在春秋時期，芒碭山屬[[宋國|宋國]]領土。據史書記載，[[孔子|孔子]]曾經從曹國到宋國期間，恰好經過芒碭山避雨。<ref>{{web reference|URL=http://www.chinadaily.com.cn/hqgj/jryw/2014-04-01/content_11512354.html|title=上古時期河南芒碭山為著名採石場 黄帝曾巡視|work=大河報|year=2014年|date=4月1日}}</ref>[[秦代|秦代]]初[[秦始皇|秦始皇]]置[[碭縣|碭縣]]，又置[[碭郡|碭郡]]。{{noteTag|見後魏酈道元《水經注》卷23〈陰溝水　汳水　獲水〉：「獲水又東逕碭縣故城北，應劭曰：縣有碭山，山在東，出文石，秦立碭郡，蓋取山之名也。」。}}這是秦朝實施的郡縣制度三十六郡的其中一郡。後來到了秦朝末年，[[漢高祖|漢高祖]][[劉邦|劉邦]]斬白蛇起義。而[[陳勝|陳勝]]起義之後卻被車夫庄賈殺害，被埋葬於芒碭山主峰南側山腳下。現在該處存有[[陳勝|陳勝]]的墓塚跟[[郭沫若|郭沫若]]手書的石碑。{{noteTag|見李開元《秦崩：從秦始皇到劉邦》（2010年由聯經出版公司出版）第216頁〈走進歷史現場 陳勝墓〉。}}但碭縣在[[西漢|西漢]]時期被分為芒、碭二縣，分屬於梁國和沛郡。[[梁孝王|梁孝王]][[劉武|劉武]]曾在芒碭山左右建設包括落猿岩、棲龍岫、鶴州、雁池、鳧島等風景的東苑。{{noteTag|資料出自方硯《古都遺韻》（2014年由現代出版社出版）。}}之後在[[東漢|東漢]]時，芒縣又改稱[[臨睢縣|臨睢縣]]。

在[[東漢|東漢]]末年，[[曹操|曹操]]曾為了補貼軍費開支而設立「[[摸金校尉|摸金校尉]]」以及「發丘中郎將」，專門盜取和發掘[[永城市|永城]]芒碭山山上的[[漢梁王墓群|漢梁王墓群]]72船寶物。{{noteTag|東漢末年[[陳琳|陳琳]]著有《為袁紹檄豫州》。原文記述了討伐曹操的其中一個原因為盜墓：「又梁孝王，先帝母昆，墳陵尊顯；桑梓松柏，猶宜肅恭，而操帥將吏士，親臨發掘，破棺裸屍，掠取金寶，至令全朝流涕，士民傷悲。操又特置發丘中郎將，"摸金校尉"，所過隳突，無骸不露。」。}}羅貫中在《[[三國演義|三國演義]]》第二十八回創作過名為「斬蔡陽兄弟釋疑 會古城主臣聚義」的故事，其中描述了[[劉備|劉備]]、[[張飛|張飛]]等人曾與[[曹操|曹操]]爭奪[[徐州|徐州]]。[[張飛|張飛]]敗走後據守芒碭山，並築寨稱王。不過當中的真偽未有定論。
<ref name="發現真實三國，讓你驚呼連連">[http://museum.ymculture.org.tw/archive/archive03.asp?kno=22&tno=62&dno=117 財團法人陽明海運文化基金會〈發現真實三國，讓你驚呼連連〉]</ref><ref name="千里走單騎">[https://hk.chiculture.net/21602/html/c11/21602c11.html 中國文化研究院〈千里走單騎〉]</ref>

此後，芒碭山被歷代的一些將領用作興建宮室或據點。例如[[唐代|唐代]]詩人李白的《丁都護歌》寫道：「雲陽上征去，兩岸饒商賈。 吳牛喘月時，拖船一何苦。 水濁不可飲，壺漿半成土。 一唱督護歌，心摧淚如雨。 萬人系盤石，無由達江滸。 君看石芒碭，掩淚悲千古。」所記述的是唐朝皇室徵召百姓到芒碭山上建宮室。另外，[[明代|明代]]末年流寇[[李自成|李自成]]亦佔據過芒碭山抗衡明軍。<ref>{{web reference|URL=http://travel.hebnews.cn/2013-12/02/content_3643108.htm|title=八代九王 葬於芒碭|work=大河村網站|year=2013年|date=12月2日}}</ref>
在1935年至1945年，[[抗日戰爭|抗日戰爭]]爆發。不少人民以芒碭山為防空洞避難。結果，多人死亡，連抗日運動烈士鲁雨亭亦在此殉難。<ref name="鲁雨亭">[http://www.itta.org.cn/ds/nr/xl/xl122.htm 上海市信息化培訓協會〈鲁雨亭〉]</ref>於1980年代，由於採石爆破工程不斷進行，芒碭山群的大部分山體已被嚴重破壞。至1995年春天，芒碭山發生了一宗盜墓案，盜墓賊炸開了漢梁王墓，盜走玉璧等309件寶物。同年9月7日，[[永城市|永城市]]公安局副政委王學習在[[永城市|永城市]]薛湖鎮一個窯場內與民警當場拘捕5人。1999年8月9日，王學習再與專案組人員進駐芒碭山，並拘捕了23個盜墓賊。<ref>{{web reference|URL=http://www.henan.gov.cn/hngk/system/2006/08/11/010001866.shtml|title=漢墓累累芒碭山|work=河南省人民政府門户網站|year=2006年|date=8月11日}}</ref>

== 漢王墓群 ==
芒碭山群中的保安山山下共有21座[[漢代|漢代]]梁國王陵和地下宮殿，其中已被發掘的有7座。據[[河南省|河南省]]文物局南水北調文物保護辦公室主任及研究所前研究員張志清指出李王後墓是目前在芒碭山發現的漢代梁國王室陵墓中最大的一個，也是[[中國|中國]]目前發現規模最大、結構最復雜的石室墓，其規模遠超[[河北|河北]][[滿城|滿城]][[劉勝|劉勝]]墓、[[山東|山東]][[曲阜|曲阜]]魯王墓、[[徐州|徐州]]楚王墓和芒碭山梁孝王的墓。李王後墓有東面及西面兩個墓道。露天開鑿的東墓道總長37.7米，位於芒碭山山峰的東坡。前庭被3000多塊巨大的塞石密封，塞石上刻有序號、尺寸、工匠名字、所處方位、施工次序等有助研究[[漢代|漢代]]書法、文字、歷法和尺度等。<ref>{{web reference|URL=http://www.henan.gov.cn/hngk/system/2006/08/11/010001866.shtml|title=漢墓累累芒碭山|work=河南省人民政府門户網站|year=2006年|date=8月11日}}</ref>

經現代考古學家清理[[梁孝王|梁孝王]]墓群後，掘出的墓道塞石有近3000塊，每塊重1000公斤至2000公斤。該3000塊石頭是為防止後人盜墓而設置的。但是專家推算[[梁孝王|梁孝王]]墓群在未被盜掘時的墓道塞石數量應有5700塊至6200塊。此外，埋葬在山上的為[[西漢|西漢]][[梁國|梁國]]八代九個王。考古學家至今在芒碭山柿園的漢墓中發掘出《[[四神雲氣圖|四神雲氣圖]]》，現存放於[[河南省|河南省]][[鄭州市|鄭州市]][[河南博物院|河南博物院]]內。畫中繪畫了墓室主人欲於死後乘龍昇天、追求生世般快樂的理想和願望。另一方面，考古學家也於僖山出土了金縷玉衣，濾煙宮燈、罕見玉璧等珍貴文物。1991年，柿園漢墓的發掘工作被評為「[[全國十大考古新發現|全國十大考古新發現]]」。而在1994年，[[西漢|西漢]][[梁孝王|梁孝王]]陵寢園的發掘工作也被評為「[[全國十大考古新發現|全國十大考古新發現]]」。<ref>{{web reference|URL=http://www.henan.gov.cn/hngk/system/2006/08/11/010001866.shtml|title=漢墓累累芒碭山|work=河南省人民政府門户網站|year=2006年|date=8月11日}}</ref>

== 遺跡 ==
芒碭山的南側山腳下布有一座高祖廟用以紀念[[漢高祖|漢高祖]][[劉邦|劉邦]]「[[斬白蛇起義|斬白蛇起義]]」一事<ref>{{web reference|URL=http://www.epochtimes.com/b5/7/6/4/n1732798.htm|title=河南永城耗資逾一兆元建劉邦銅像引發爭議|work=大紀元|year=2007年|date=6月4日}}</ref>。除此之外，山上建設了[[魯雨亭|魯雨亭]]紀念館保留抗日時期的歷史<ref>{{web reference|URL=http://www.ycszxw.gov.cn/ReadNews.asp?NewsID=128|title=市政協機關到芒山鲁雨亭紀念館敬獻花圈緬懷先烈|work=永城市政協辦公室|year=2017年|date=4月2日}}</ref>。[[孔子|孔子]]避雨處、[[孔子|孔子]]石像以及為紀念孔子而建的文廟均設於芒碭山。<ref>{{web reference|URL=http://chinakongmiao.com.cn/templates/T_detail/index.aspx?nodeid=869&page=ContentPage&contentid=3816|title=未列入國家祀典的禮制廟宇：紀念孔子廟和孔氏家廟|work=中國孔廟保護協會第12屆年會論文集|year=2011年|date=8月3日}}</ref>
