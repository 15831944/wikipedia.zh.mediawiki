'''芒市公交'''是[[云南省|云南省]][[德宏傣族景颇族自治州|德宏傣族景颇族自治州]][[芒市|芒市]]的[[公共汽车|公共汽车]]系统，由芒市益民公共汽车有限公司运营<ref name="云南信息报"></ref>，芒市明森公共交通运输有限公司经营其中的13路师专专线公交<ref name="师专"></ref>。截止2013年已投入运营的公交车辆共66辆，合计62.3标台<ref name="公示">{{cite web |url=http://fj.yn.gov.cn/docfile/gongshi/file/3077/%E8%8A%92%E5%B8%82%E4%BA%BA%E6%B0%91%E6%94%BF%E5%BA%9C%E5%85%B3%E4%BA%8E%E8%8A%92%E5%B8%82%E5%85%AC%E5%85%B1%E4%BA%A4%E9%80%9A%E4%B8%93%E9%A1%B9%E8%A7%84%E5%88%92%EF%BC%88%E8%8D%89%E6%A1%88%EF%BC%89%E7%9A%84%E5%85%AC%E7%A4%BA378768.doc |title=芒市人民政府关于芒市公共交通专项规划（草案）的公示 |last= |first= |date=2013-11-11 |website=fj.yn.gov.cn |publisher=芒市政府办公室 |access-date=2017-08-27 |quote=}}</ref>。现共有运营线路7条，截止2016年总运营里程51.4公里<ref name="公示"></ref><ref name="云南信息报"></ref><ref group="注">与现在相比不含最新开通的13路师专专线公交，5路车未改道</ref>，可以使用[[IC卡|IC卡]]<ref name="德宏网"></ref>。公交公司总站的用地未来将设在环东路上，面积约20亩<ref name="网易新闻"></ref>。

==历史==

2010年时，芒市仅有3条公交线路，即法帕专线、帕底专线和城区线，分别是现在的1路、2路和3路<ref group="注">原城区线只到奥环水泥厂，改为3路车后线路延伸到木康村</ref>，后又开通弘映山庄专线（即今5路车），均为招手即停<ref>{{cite web |url=http://www.dhtravel.net/a/yk/jt/2013/0516/168.html |title=潞西市(芒市)公交车线路 |last= |first= |date=2010-06-02 |website=www.dhtravel.net |publisher=德宏旅游信息网 |access-date=2017-09-16 |quote= |accessdate=2017-09-16 |archiveurl=https://web.archive.org/web/20170917034226/http://www.dhtravel.net/a/yk/jt/2013/0516/168.html |archivedate=2017-09-17 |deadurl=yes }}</ref>。2011年更换市内公交站牌后开始严格执行到站停车的规定，不再招手即停<ref>{{cite web|url=http://special.yunnan.cn/2008page/dehong/html/2011-09/16/content_1827039.htm|title=芒市公交站牌站台换新装|last=刘|first=莹|date=2011-09-16|website=special.yunnan.cn|publisher=云南网|access-date=2017-09-16|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20170917033258/http://special.yunnan.cn/2008page/dehong/html/2011-09/16/content_1827039.htm|archivedate=2017-09-17}}</ref>。2012年4月开通第一条市内公交，即5路公交；2014年3月24日开通第二条市内公交，即10路公交<ref name="德宏团结报">{{cite web |url=http://www.cnepaper.com/dhtjb/html/2014-03/09/content_1_4.htm |title=芒市新开第10路公交车 |author=陈子鸥 |date=2014-03-09 |website=www.cnepaper.com |publisher=德宏团结报 |access-date=2017-08-27 |quote=}}</ref><ref name="云南信息报">{{cite web |url=http://news.ifeng.com/gundong/detail_2014_02/26/34186506_0.shtml |title=芒市新开10路公交车 途经大学城 |last= |first= |date=2014-02-26 |website=凤凰资讯 |publisher=云南信息报 |access-date=2017-08-27 |quote=}}</ref>。

==问题==

芒市公交出行分担率仅4%，线网密度仅1.89<ref group="注">国家标准要求为3-4</ref><ref name="公示"></ref>，路线覆盖面少，东密西疏，部分道路公交重复系数较高，部分线路非直线系数过高<ref name="公示"></ref>，但很多城区却处于公交服务的空白区<ref name="德宏团结报"></ref>，市内公交发展仅仅只是横向发展，没有纵向发展，南北方向的线路规划较完善，东西方向的线路规划不足，公交线路主要是单线，环线设计不足，总密度不足，且覆盖的学校很少，不利于公交主要服务对象学生的出行<ref name="云南信息报"></ref>。

公交站点数量不足，公交站点300米及500米半径覆盖率仅41.3%和63%<ref group="注">国家标准要求为60%和90%</ref><ref name="公示"></ref>，且站台设施较差<ref name="网易新闻"></ref>。

公交车辆不足，万人拥有率仅2.36台/万人<ref group="注">国家标准要求为6.7-8.3台/万人</ref>，这也导致发车频率不足、运量不够、乘客等待时间过长<ref name="公示"></ref>，公交保养设施不足，车况较差，末班车时间过早<ref name="网易新闻">{{cite web |url=http://news.163.com/14/1102/06/AA1AGGSV00014Q4P.html |title=明年芒市拟新开3条公交线路 |last= |first= |date=2014-11-02 |website=网易新闻 |publisher=云南信息报 |access-date=2017-08-27 |quote=}}</ref>。

至今暂无公共交通固定专用停保场地<ref name="公示"></ref>，公交场站尚属临时租用，场内保养设施不足。首末站为临时占用道路或租用其他机构的场地，沿途车站数量少、不规范、分布不合理，缺少乘客候车休息设施和线路走向、站点标识、运营时间等<ref name="优先发展公共交通">{{cite web |url=http://www.dh.gov.cn/Web/_19788.htm |title=德宏州人民政府关于城市优先发展公共交通工作的实施意见（德政发〔2016〕6号） |last= |first= |date=2016-06-24 |website=www.dh.gov.cn |publisher=德宏州人民政府 |access-date=2017-08-27 |quote=}}</ref>。
