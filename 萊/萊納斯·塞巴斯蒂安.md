'''萊納斯·加布里埃爾·塞巴斯蒂安'''（{{Lang-en|Linus Gabriel Sebastian}} {{Bd|1986年|8月20日}}，生於加拿大不列顛哥倫比亞省）是一個加拿大[[YouTuber|Youtuber]]，主持人，製作人和{{Internal link helper/en|Linus Media Group|}}的創始人。

他最出名的是創建并託管了三個Youtube科技頻道 [https://www.youtube.com/user/LinusTechTips '''''Linus Tech Tips''''']、[https://www.youtube.com/user/Techquickie '''''Techquickie''''']和 [https://www.youtube.com/channel/UCeeFfhMcJa1kjtfZAGskOCA '''''TechLinked''''']以及幕後頻道[https://www.youtube.com/user/ChannelSuperFun '''''Channel Super Fun''''']和一個不為人知的不屬於{{Link-en|Linus Media Group|}}的私人頻道[https://www.youtube.com/user/LinusCatTips '''''LinusCatTips''''']。這其中的三個頻道：''[https://space.bilibili.com/12434430 Linus Tech Tips]''、''[https://space.bilibili.com/12564758/ Techquickie]''和''[https://space.bilibili.com/25282560/ Channel Super Fun]''在[[中国大陆|中國大陸]]由輝光字幕組（[http://www.nixiesubs.com/ 官網]）在[[Bilibili|嗶哩嗶哩]]運營。這些（[[YouTube|YouTube]]）頻道的訂閱總數超過一千一百三十萬。從2007年到2015年，他還是現已經解散的加拿大計算機零售商NCIX的技術視頻主持人。2015年雜誌{{Link-en|Inc.|Inc. (magazine)}}將萊納斯·塞巴斯蒂安評為"科技圈內最有實力的30位玩家"的第四名.<ref>{{Cite news|url=https://www.inc.com/john-rampton/30-power-players-in-tech-you-need-to-know.html|title=30 Power Players in Tech You Need to Know|author=|date=2015-07-21|work=|newspaper=Inc.com|accessdate=}}</ref>他也是Floatplane Media的CEO。

截止至2018年3月，他創建的頻道''Linus Tech Tips''在YouTube上市排名第七位最受關注的科技頻道<ref>{{Cite web|url=http://vidstatsx.com/youtube-top-100-most-viewed-science-tech|title=YouTube Top 100 Most Viewed Science & Tech Video Producers|accessdate=|author=|date=2016年1月4日|publisher=VidstatsX}}</ref> 2014年，Tubefilter將該頻道評選為在YouTube科技類頻道中 "谷歌首選廣告頻道的前1%"

== 職業生涯 ==

=== NCIX 和 Linus Tech Tips ===
萊納斯·塞巴斯蒂安曾擔任現已解散的加拿大計算機零售商NCIX的產品經理。該公司要求他成為其科技頻道的主持人，該頻道旨在幫助展示他們的產品。萊納斯·塞巴斯蒂安得到一位身份不明的攝影師和剪輯師的幫助，他使用有限的資源，用從公司總裁的兒子那借來的相機拍攝視頻。<ref name=":0">{{Citation|last=Linus Sebastian|title=How I Became: LinusTechTips|date=2014-03-24|url=https://www.youtube.com/watch?v=XJbLHLnTrd4|accessdate=}}</ref>他的第一個視頻是Sunbeam CPU散熱器的介紹。<ref>{{Citation|last=NCIX Tech Tips|title=Sunbeam Tuniq Tower (NCIX Tech Tips #1)|date=2007-07-24|url=https://www.youtube.com/watch?v=OhA0X6_z3lY|accessdate=2018-11-23}}</ref>

由於頻道早期的高成本但低的收視率，萊納斯·塞巴斯蒂安被指示創建了''Linus Tech Tips''頻道，以作為NCIX頻道更便宜的分支，以允許更低的產品價值而不影響NCIX品牌的聲譽。他將{{Internal link helper/en|TigerDirect|}}和{{Link-en|Newegg|}}描述為競爭對手。<ref name=":0" />''Linus Tech Tips''創建於2008年11月24''日''

萊納斯·塞巴斯蒂安沒有在NCIX全職製作視頻，在公司工作期間，他曾擔任全職銷售，高端PC設計師和產品經理。<ref>{{Cite web|url=http://www.linusmediagroup.com/history/|title=Our Team|accessdate=2018-11-23|work=Linus Media Group|language=en-US}}</ref><ref>{{Cite news|url=http://lifehacker.com/im-linus-sebastian-of-linustechtips-and-this-is-how-i-1719745217|title=I'm Linus Sebastian of LinusTechTips, and This Is How I Work|last=Orin|first=Andy|newspaper=Lifehacker|accessdate=2018-11-23|language=en-US}}</ref>

=== Linus Media Group ===
萊納斯·塞巴斯蒂安于2013年1月和Luke Lafreniere、Edzel Yago和Brandon Lee在一個車庫里創立了Linus Media Group。該公司開發了''Linus Tech Tips''作為一個獨立企業。莱纳斯·塞巴斯蒂安也開始在''Techquickie''上投入精力。

2013年9月，萊納斯·塞巴斯蒂安接受了克里斯·皮里洛（{{Lang-en|Chris Pirillo}}）的采访。莱纳斯·塞巴斯蒂安讨论了他的频道的未来，并回答了通过社交媒体发送给Pirillo的问题。

在2014年的一次采访中，萊納斯·塞巴斯蒂安透露，在他開始他的YouTube职业生涯的早期阶段，在没有创业资金的情况下，在找出如何养家糊口以及如何雇用员工的压力下，他度过了一个不眠之夜。

2015年，该公司首次搬入商业办公區域，而之前是在不列颠哥伦比亚省萨里的一个住宅里工作。移动过程记录在一系列VLOG中，成为Linus Tech Tips频道历史上最受好评的系列视频之一。从那以后，莱纳斯·塞巴斯蒂安在搬迁后還偶尔發佈了了办公室VLOG。

截至2018年9月16日，''Linus Tech Tips''，''Techquickie''和''Channel Super Fun''三個频道分别發佈了4284个视频，566个视频和158个视频，包括了产品介紹，建築日志，VLOG和原创网络系列節目。 ''Linus Tech Tips''每天都会发布一個新的视频。

在2018年5月，萊納斯·塞巴斯蒂安宣布他的目标是要收購YouTube频道''NCIX Tech Tips''，但沒能成功。相反，{{Link-en|Linus Media Group|}}决定创建一个新的YouTube頻道''TechLinked''，并挖來了前''NCIX Tech Tips''主持人Riley Murdock。

=== 較大的項目 ===

==== The WAN Show（2012年至今） ====
The WAN Show 是由萊納斯·塞巴斯蒂安和Luke Lafreniere主持的日常直播。節目中，他們討論了科技界的新聞報道，提出了他們的觀點和問題。截至2018年3月5日已經播放了226期。其它{{Link-en|Linus Media Group|}}的員工偶爾會代替萊納斯·塞巴斯蒂安或Luke Lafreniere來主持。

==== 全屋水冷（{{Lang-en|Whole Room Water Cooling}}）（2014年-2015年） ====
“全屋水冷”記錄了萊納斯·塞巴斯蒂安試圖通過將工作站連接到房屋的水冷循環並把所有熱量從建築物排出以降低工作站及房間的溫度。在7個月的7集后，萊納斯·塞巴斯蒂安宣佈該項目失敗。塞巴斯蒂安提到這主要是由於管道腐蝕，導致冷卻部件和PC部件無法使用。

==== 廢品大戰（{{Lang-en|Scrapyard Wars}}）（2015年至今） ====
廢品大戰是一個跟隨萊納斯·塞巴斯蒂安和Luke Lafreniere這兩個有時會成為競爭對手人的視角，讓他們在有限的預算、時間和主題內組建出一台性能最佳或性价比最高的PC。

===== 第一季 =====
在第一季中，萊納斯·塞巴斯蒂安和Luke Lafreniere各自拿著300加元（約1500人民幣）建造一台遊戲PC。他在兩天中的三集后結束。

===== 第二季 =====
在第二季中，萊納斯·塞巴斯蒂安和Luke Lafreniere需要以500加元建造一台水冷PC而不使用“現成的”水冷部件，這一季共四集。預算分為300加元給組裝PC，剩餘的200加元給DIY水冷。

===== 第三季 =====
在第三季中，萊納斯·塞巴斯蒂安、Luke Lafreniere和嘉賓{{Link-en|Austin Evans|}}（美國科技YouTuber）面臨挑戰，以無限制的預算組建出一台具有最優性價比的遊戲PC。第三季共有七集。

===== 第四季 =====
在第四季中，萊納斯·塞巴斯蒂安、Luke Lafreniere以及來自定制計算機BS Mods的嘉賓Bob Stewart和Rob Rosenberg得到了900加元來組建一台有著定制配件機箱的遊戲PC。第四季共有四集，萊納斯·塞巴斯蒂安和Bob Stewart獲得了勝利
