'''萊蕨'''（[[學名|學名]]：{{lang|la|'''''Leptochilus decurrens'''''}}）為[[水龍骨科|水龍骨科]][[萊蕨屬|萊蕨屬]]下的一個種。
[[File:Leptochilus_decurrens_2.jpg|替代=萊蕨營養葉明顯比繁殖葉寬，繁殖葉呈現長線形]]

== 命名 ==
萊蕨屬之拉丁學名"''Leptos"意思為''纖細、細小的；"''chilos''"為唇的意思。種小名"''decurrens"''意思為葉邊緣逐漸向莖漸縮，到達葉基呈現翼狀延伸。

== 形態特徵 ==
根莖長匍匐狀，被覆黑褐色、常披針形之窗格狀鱗片，葉柄間距約1~3公分。

葉兩型；營養葉餅短或不顯著，葉片匙形，長約30公分，寬8公分，末端漸尖，基部下延；葉脈網狀，網眼內有游離小脈，側脈明顯，平行，近葉緣分叉；孢子葉柄長25~35公分，葉片窄線狀披針形至線形，長20~30公分，寬5~10釐米，孢子囊全面著生於葉背。<ref>{{Cite book|title=《蕨類圖鑑》|last=郭|first=城孟|publisher=遠流臺灣館|year=2001|isbn=|location=|pages=}}</ref>

孢子單細胞，腎形，黃色，具有向外凸出的刺。

== 地理分佈 ==
* 東南亞為分佈中心，北以印度東北、中國南部及台灣一線為界，台灣產於中、低海拔的成熟闊葉林內。

* 萊蕨在印度之族群，已列入國際自然保護聯盟瀕危物種紅色名錄IUCN，屬於無為等級。

== 民俗用途 ==
印度的某些原住民，會利用萊蕨作為要藥用途，用來治療皮膚問題。他們也相信萊蕨具有抗菌之作用。

== 染色體數目 ==
自然環境中，兩倍體n=36最為常見。極少數機會會產生四倍體n=72以及八倍體n=144
