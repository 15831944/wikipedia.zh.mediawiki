'''萊斯特·萊昂內爾·沃爾夫'''（英語：Lester Lionel Wolff，{{bd|1919年|1月4日}}）是一名來自[[美國|美國]][[紐約|紐約]]的政治人物，他是[[美國|美國]][[民主黨|民主黨]]前[[美國眾議院|眾議院]]議員，現任美國貿易發展署總裁。2014年沃爾夫獲頒美國最高市民獎[[國會金質獎章|國會金質獎章]]。

沃爾夫是一位相當知名的亞洲事務處理專家，他是[[杜魯學院|杜魯學院]]太平洋社區研究所的主席，及許多外交政策書籍的作者和[[公共廣播電台|公共廣播電台]]（PBS）「問國會」節目的主持人。

== 早年生活和教育 ==
沃爾夫出生在[[纽约|紐約市]]，他的家族自18世紀以來就定居於此。他在[[曼哈頓|曼哈頓]]的[[猶太教改革派|猶太教改革派]]教堂裡完成他的猶太成年禮。沃爾夫大學時就讀於[[纽约市立学院|紐約市立學院]]，之後畢業於[[紐約大學|紐約大學]]並取得市場行銷學位。

== 早期職業生涯 ==
從1939到1941年間他在[[紐約大學|紐約大學]]裡任教，之後他升任[[紐約市立大學|紐約市立大學]]系主任。在[[第二次世界大战|第二次世界大戰]]期間，沃爾夫曾經擔任過民用空中巡邏隊的隊員，中隊長和[[獵潛艦|獵潛艦]]船員。

在私人企業裡，沃爾夫被公認為是市場研究和媒體傳播的權威。他是《長島新聞》和《布朗克斯家庭新聞》的執行長。沃爾夫還創立了自己的公司，專攻食品工業，並被選為紐約零售雜貨商會議常務董事。另外他也是當地電視節目《The Lines》的製作人兼主持人，及溫蒂·巴瑞（1912年4月18日－1978年2月2日）主演的名人綜藝節目製作人。

沃爾夫一直積極投身於於慈善事業，並領導猶太人聯合呼籲會的一個分會。

== 美國聯邦眾議員時期 ==
1957年，沃爾夫曾被[[美國眾議員|美國眾議員]]選為消費者研究小組委員會的顧問委員會主席。

他於1964年當選為眾議員，自1965年1月3日上任，連任至1981年1月3日。他最初代表第三選區參選，後來因區域重新分配變成第六選區。沃爾夫之後擔任美國國會亞太事務委員會主席及麻醉品濫用和控制特別委員會主席。他在領導民用空中巡邏隊的國會中隊時，官階升至上校。

1978年，[[美國|美國]]國會代表團訪問[[中國|中國]]期間，沃爾夫會見了現代中國之父[[鄧小平|鄧小平]]（1904年8月22日－1997年2月19日）。此次鄧沃會談對於 [[中華人民共和國|中華人民共和國]]與[[美國|美國]]間建立正式的外交關係產生至關重要的影響。隨後[[美國總統|美國總統]][[吉米·卡特|吉米·卡特]]（James Earl Jimmy Carter, Jr.，1924年10月1日－）在未經國會同意下無預警的宣布與[[中華民國|中華民國]]斷交，並正式承認[[中華人民共和國|中華人民共和國]]，此一舉動也終止[[美國|美國]]與[[中華民國|中華民國]]簽訂的25年共同防衛條約。因為此次事件，美國國會於1979年4月10日宣布通過訂立《[[臺灣關係法|台灣關係法]]》來維持與[[中華民國|中華民國]]的非正式外交關係及維護其國家安全和雙方的商業利益，沃爾夫即是該法的制定者之一。

1969年，沃爾夫向[[白宮|白宮]]提出《對外援助法》的修訂版本（該法第一版於1961年簽訂），以恢復[[以色列|以色列]]和[[阿拉伯|阿拉伯]]國家之間直接和平對談的倡議。此外1978年他也參與了[[大衛營協議|大衛營協議]]（[[埃及|埃及]]與[[以色列|以色列]]和平條約）的簽訂。

== 國會議員卸任後的工作 ==
他目前是美國貿易發展署的總裁，也是[[杜魯學院|杜魯學院]]太平洋社區研究所的主席。他出版了許多有關外交政策的書籍。自20世紀80年代中期以來，他每週主持一次[[公共廣播電台|公共廣播電台]]（PBS）的「問國會」節目。由於他對於亞洲文化與各國關係的經驗和知識，使他成了一位相當受歡迎的亞洲顧問。從1987年到2007年間，他擔任格里芬公司的董事。在2010年，沃爾夫獲頒世界和平獎的最高榮譽獎，於2014年獲得美國最高平民榮譽[[國會金質獎章|國會金質獎章]]<ref>[http://www.wppac.net/popup/popInfo.asp?idx=23 World Peace Prize Top Honer Prize－Hon. Lester Wolff] WPPAC. </ref>。

隨著2017年10月30日詹姆斯 D. 馬丁（James D. Martin，1918年9月1日－2017年10月30日）的逝世，沃爾夫成了美國目前年齡最大的前國會議員。

== 個人生活 ==
沃爾夫於1940年與布蘭奇·希爾弗斯結婚；而她於1997年逝世<ref name=voice>{{cite web |url=https://www.villagevoice.com/2017/05/30/long-islands-98-year-old-former-congressman-eats-dumplings-hates-trump-makes-tweets/ |title=Long Island’s 98-Year-Old Former Congressman Eats Dumplings, Hates Trump, Makes Tweets |author=Barkan, Ross |date=2017-05-30 |accessdate=2017-06-20 |work=[[Village_Voice|Village Voice]]}}</ref>。目前沃爾夫居住於[[紐約州|紐約州]]的羊肉鎮。
