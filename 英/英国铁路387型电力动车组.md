'''英铁387型电力动车组''「电力之星」'''''（{{lang-en|British Rail Class 387 ''Electrostar''}}）是一款由[[Adtranz|Adtranz]]（今属[[庞巴迪运输|庞巴迪运输]]）在英国[[德比|德比]]制造的列車。它主要服务于[[泰晤士联线|泰晤士联线]]、[[盖特威克机场快线|盖特威克机场快线]]及{{link-en|c2c|c2c}}等线路。它是隶属于「電力之星」系列的列车，同系列的其他型号列車包括[[英鐵357型電力動車組|357]]、{{link-en|英鐵375型電力動車組|British Rail Class 375|375}}、{{link-en|英鐵376型電力動車組|British Rail Class 376|376}}、{{link-en|英鐵377型電力動車組|British Rail Class 377|377}}、[[英鐵378型電力動車組|378]]及{{link-en|英鐵379型電力動車組|British Rail Class 379|379}}型。
