'''英國駐武漢總領事館'''（{{lang-en|'''British Consulate-General Wuhan'''}}）為[[英國|英國]]在[[武漢|武漢]]派設的[[官方|官方]][[外交|外交]][[英國駐外機構列表|代表機構]]，館址位於[[江岸區|江岸區]][[武漢天地|武漢天地]]企業中心5號樓（现已更名为'''平安金融中心'''）33层，[[領事轄區|領区範圍]]含[[湖北省|湖北省]]、[[河南省|河南省]]。該館是[[中華人民共和國|中華人民共和國]]成立後英國在華設立的第五个[[總領事館|總領事館]]，也是自1949年10月以來、外國於武漢設立的第四個[[領事館|領事館]]，於2015年1月10日正式開館。<ref name="英国国务大臣为武汉新领事馆揭幕">{{cite web |url = https://www.gov.uk/government/news/british-minister-launches-new-consulate-in-wuhan.zh |title = 英国国务大臣为武汉新领事馆揭幕 |accessdate=2015-02-15}}</ref>

== 历史 ==
=== 开馆 ===
1861年3月7日，英国外交官威利司等人来到[[汉口|汉口]]，在汉租赁栈房。随后，划定[[汉口英租界|英租界址]]，南起现[[江汉路|江汉路]]，北至现合作路，东南沿长江滨，西北至现[[鄱阳街|鄱阳街]]，后又延伸至[[中山大道_(武汉)|中山大道]]方向，两次共划地795亩。英国驻汉领事馆官邸位于汉口天津路与沿江大道的交汇处，是一栋二层红瓦黄墙西洋建筑，它是当年英租界的第一栋建筑，也是近代武汉建筑史上的第一栋西洋建筑，有办公楼三栋，共有房间23间，领事馆舍有房间13间，历经战争和时间的洗礼，目前只有1栋保存完好。当汉口市民还在点油灯照明时，英国人点亮了武汉的第一盏点电灯。他们在汉口兴办电灯公司，火力发电，专供英租界使用。1903年，当汉口街头还奔跑着人力车的时候，英租界出现了汉口第一部小汽车，是英国驻汉领事馆官邸的专用车。从英国驻汉领事馆官邸开始，西洋建筑风潮抵汉，武汉三镇有了钢筋建筑，近代建筑也开始萌芽。<ref name="英国驻汉领事馆的前世今生 旧址已成优秀历史建筑">{{cite web |url = http://www.hb.xinhuanet.com/2015-03/28/c_1114791542.htm |title = 英国驻汉领事馆的前世今生 旧址已成优秀历史建筑 |accessdate = 2017-02-10 |deadurl = yes |archiveurl = https://web.archive.org/web/20170211155723/http://www.hb.xinhuanet.com/2015-03/28/c_1114791542.htm |archivedate = 2017-02-11 }}</ref>

=== 被毁与重建 ===
1938年10月25日，[[武汉会战|武汉会战]]结束，日军进入市区，武汉进入日占时期。1942年12月，美军对被日军占领的汉口进行报复性轰炸，从一元路到黄埔路的[[汉口日租界|日租界]]几乎被夷为平地。英国驻汉领事馆被炸毁一栋，炸损一栋。1946年，[[二战|二战]]结束，英国驻汉口总领事重新上任，对领事馆进行了重修。

1949年10月1日，随着中华人民共和国的成立，武汉的外国领事馆相继关闭，英国驻汉领事馆也于1950年4月宣布关闭。2012年12月4日，武汉市将英国领事馆旧址列入“优秀历史建筑”名单，挂牌进行保护。

== 现状 ==
2015年1月10日，英国正式於[[江岸區|江岸區]][[武漢天地|武漢天地]]企業中心设立驻武汉总领事馆。驻武汉总领馆是英国[[中华人民共和国|在华]]最新设立的[[领事馆|领事馆]]，也是所有领馆中规模最小的一个。驻武汉总领馆致力于在政务、贸易投资、科技创新和教育文化等方面为中英两国建立互利关系，包括协助双向的合法人员流动，并为领区内的英国公民提供帮助。领事馆可以提供领区内例如医院和律师等的基本信息，但是没有配备领事官员。在得到允许的情况下，驻武汉总领馆可协助联络[[英国驻华大使馆|英国驻华大使馆]]北京领事处并提供帮助。由於[[英國駐華大使館|英國駐華大使館]]及[[領事館|領事館]]並不處理普通居民的[[簽證|簽證]]申請，因此武漢領事館目前僅為轄區內的[[中華人民共和國政府|中國政府]][[公務員|公務員]]提供簽證服務。一般市民若需辦理簽證，需要前往各地签证中心。
