'''英国驻华大使馆'''（{{lang-en|'''British Embassy, Beijing'''}}）是[[大不列颠及北爱尔兰联合王国|大不列颠及北爱尔兰联合王国]]在[[中华人民共和国|中华人民共和国]]设立的[[英國駐外機構列表|外交代表機構]]，位于北京市朝阳区[[建国门|建国门]]外光华路11号<ref>{{cite web |author=中华人民共和国外交部 |title=(United Kingdom)大不列颠及北爱尔兰联合王国大使馆 |url = http://www.fmprc.gov.cn/chn/pds/gjhdq/gj/oz/1206_45/1206x1/ggzwwjjgxx/t527355.htm |accessdate=2012-07-18}}</ref>，是英国规模最大的驻外使馆之一<ref name="gaishu">{{cite web |author=英国驻华大使馆 |title = 英国驻华大使馆 |url = http://ukinchina.fco.gov.uk/zh/about-us/our-embassy/|accessdate=2012-07-18}}</ref> 。其主要负责的工作包括扩大两国间贸易与投资往来，加强两国在政治、经济、文化、国防等领域的联系，提供签证服务，为英国公民提供外交保护，同时在中国开展扶贫项目等<ref name="gaishu" />。目前[[領事轄區|領事轄區]]范围包括[[北京|北京]]、[[天津|天津]]、[[河北|河北]]、[[山东|山东]]、[[甘肃|甘肃]]、[[宁夏|宁夏]]、[[吉林|吉林]]、[[黑龙江|黑龙江]]、[[内蒙古|内蒙古]]、[[新疆|新疆]]、[[山西|山西]]、[[陕西|陕西]]、[[西藏|西藏]]、[[青海|青海]]和[[辽宁|辽宁]]在内的省自治区和直辖市<ref>{{cite web|author=英国驻华大使馆 |title = 使领馆服务分区 |url = http://ukinchina.fco.gov.uk/zh/about-us/our-embassy/how-we-can-help/areas-we-cover |accessdate=2012-07-18}}</ref> 。

== 历史 ==
=== 旧英国公使馆 ===
旧的英国公使馆成立于1861年3月26日，馆舍原为[[奉恩镇国公|奉恩镇国公]][[奕樑|奕樑]]的府邸。现在这个位置位于北京正义路的西侧，东交民巷和东长安街之间。“1959年1月，[[中华人民共和国外交部|外交部]]西欧司副司长通知英国，之前的北京中心规划要重建，英国公使馆占据的地方要由司法机构的一个大型新建筑占用。因此，要求公使馆的工作人员在1959年5月31日前搬离这里。”旧公使馆的北边部分目前由[[中华人民共和国国家安全部|中国国家安全部]]和[[中华人民共和国公安部|公安部]]使用，余下部分則為其他政府部门占用（[[中华人民共和国最高人民法院|最高人民法院]]则在南边[[俄罗斯|俄罗斯]]公使馆旧址上兴建）。使馆工作人员獲知，他们所放弃使用的一栋建筑现作為公安部退休官员的活动中心。据称这些退休官员非常喜欢这里，以至于公安部在盖新的办公大楼时，并没有拆除这个老建筑，而是搬进了这里。<ref name="history">{{cite web|author=英国驻华大使馆|title=我们的使馆和里程碑|url=http://ukinchina.fco.gov.uk/zh/about-us/our-embassy/history|accessdate=2012-07-18|deadurl=yes|archiveurl=https://web.archive.org/web/20120708170614/http://ukinchina.fco.gov.uk/zh/about-us/our-embassy/history|archivedate=2012-07-08}}</ref>

=== 现在的使馆 ===
英国自1959年9月开始在北京使用现在的使馆和大使官邸，这是两座位于围墙花园内的两层建筑，由中国当局在第一使馆区建设，用以交换之前因應当局要求搬出的原使馆。<ref name="history" />

=== 使馆的钟 ===
使馆的钟现在位于使馆主会议中心外，网球场对面，是1897年为了纪念[[维多利亚女王|维多利亚女王]]登基60周年而铸造的，最初悬挂在旧公使馆内小教堂外的塔内。使馆现在的休闲设施大约是1983年置办的，钟也是在那时候安装的，并作为新酒吧的名字<ref name="history" />。“在使馆教堂外……有一个石亭，里面挂着一口钟，是为庆祝维多利亚女王登基60周年而建造。钟塔位于公使馆中间四条路的相交处。如Allen解释说：‘钟塔位于树影摇曳的几条路的相交处，当然它也掩映在其中，从每个方位看都很漂亮，枪弹在上面造成许多印迹。它只受过一次袭击，但是并没有人受伤。’由于钟塔在公使馆建筑中心的特点，钟塔自然而然成为了最佳的生活中心，人们曾经一群一群地聚集在那里，讨论最新的消息和传闻。当发生重要攻击时，会猛烈敲钟以发出警报；而当鸣钟时则表示有火灾。石亭四周用于张贴各种通告——向被包围的人進行指导、以及电报、訊息、布告或是传闻。这里有时还会张贴官方统计的公使馆内居住人员數目。”<ref>{{cite book |author=J. L. CRANMER-BYNG |title=THE OLD BRITISH LEGATION AT PEKING |language=en }}{{PDFlink|[http://ukinchina.fco.gov.uk/resources/en/pdf/21778300/The_Old_British_Legation_at_Peking ''THE OLD BRITISH LEGATION AT PEKING'']|935 KB}}</ref>

== 使馆部门 ==
英国驻华大使馆下设领事处、签证处、贸易与投资处和新闻处<ref>{{cite web |author = 英国驻华大使馆 |title=我们提供的服务|url=http://ukinchina.fco.gov.uk/zh/about-us/our-embassy/how-we-can-help/ |accessdate=2012-07-18}}</ref> 。
* 领事处：为英国公民提供全范围的领事服务包括护照签发，文件证明，婚姻注册等等。
* 签证处：主要负责处理和发放英国入境签证。
* 贸易与投资处（UKTI）：负责贸易发展。
* 新闻处：为公众和媒体提供关于英国及其政府政策方面的訊息。
