== 顶级职业联赛 ==
[[File:LOL_Top-level_professional_Leagues.svg|缩略图]]
截至2019年，英雄联盟在全球共有十三个顶级职业联赛。

粗体赛事名称表示为英雄联盟职业联赛发展情况比较优秀地区的顶级联赛，这些联赛可以聯同[[英雄聯盟季中邀請賽|英雄聯盟季中邀請賽]]巴西、獨聯體、日本、拉丁美洲、大洋洲、東南亞、土耳其和越南賽區之中表現最好的區域(2017年是越南、但為東南亞[[GPL超級聯賽|GPL超級聯賽]]、因為當時越南的VCS属于GPL)各自分配到[[英雄聯盟世界大賽|英雄聯盟世界大賽]]直入小組賽的参赛资格。其余各區赛事的冠軍隊伍只能得到[[英雄聯盟世界大賽|英雄聯盟世界大賽]]入圍賽資格，需再比拼出四隊最優秀隊伍進入小組賽。<ref>{{Cite web |url = https://lol.garena.tw/news/3714 |title = 2017 世界大賽總覽|date = 2017年9月18日 |publisher = Garena |language = zh-tw }}</ref>
