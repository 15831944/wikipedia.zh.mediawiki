'''皇家空軍第28陸軍協調中隊'''（No. 28 (Army Co-operation) Squadron）的指揮部設在牛津郡的[[英國皇家空軍|英國皇家空軍]]{{Link-en|本臣皇家空軍基地|RAF Benson|本臣空軍基地}}，現時採用的機種是亞古士打韋斯特蘭的EH101灰背隼（HC3/HC3A）直升機。 <ref name="raf">[http://www.raf.mod.uk/organisation/28squadron.cfm 第28中隊] {{Wayback|url=http://www.raf.mod.uk/organisation/28squadron.cfm |date=20150911233813 }} - 英國皇家空軍</ref><ref name="rafweb">[http://www.rafweb.org/Sqn026-30.htm 第26至30中隊歷史] {{webarchive|url=https://web.archive.org/web/20120318040734/http://www.rafweb.org/Sqn026-30.htm |date=2012-03-18 }} - 私人英國皇家空軍資料網絡</ref><ref name="ehattons">[http://www.ehattons.com/StockDetail.aspx?SID=20508 吸血鬼戰鬥機] - E哈頓模型</ref><ref name="geo">[https://web.archive.org/web/20010422095743/http://www.geocities.com/cwlam2000hk/insignia_of_hong_kong.htm 香港徽章] - CW Lam</ref>

== 歷史 ==
[[File:Hurricane_recon_plane_28_Sqn_RAF_being_armed_c1943.jpg|thumb]]正在裝填彈藥。]]

第28中隊於1915年11月7日組成。[[第一次世界大戰|第一次世界大戰]]後，被派遺到[[印度|印度]]支援英國陸軍。[[第二次世界大戰|第二次世界大戰]]期間，第28中隊採用韋斯特蘭呂山德聯絡機，1942年12月後期,採用颶風戰鬥轟炸機。1943至1945年，第28中隊在緬甸境內運作。1945年起改用[[噴火戰鬥機|噴火戰鬥機]]。1949年，第二次世界大戰完畢後，其總部遷往[[英屬香港|英屬香港]]，並駐守皇家空軍[[前皇家空軍基地|啓德空軍基地]]，肩負起英國遠東區空軍的戰鬥和偵察任務。<ref name="raf"/><ref name="rafweb"/><ref name="ehattons"/><ref name="geo"/>

== 曾採用的機種 ==
[[File:WessexHC2_28Sqn_HongKong1983.jpeg|thumb]]（Wessex HC2） 正從香港[[啟德機場|啟德機場]]起飛。]]

* 1951年 - [[德哈佛蘭|德哈佛蘭]] [[吸血鬼戰鬥機|吸血鬼戰鬥機]]（de Havilland DH.100 Vampire）
* 1956年 - 德哈佛蘭 [[毒液戰鬥轟炸機|毒液戰鬥轟炸機]]（de Havilland DH.112 Venom）
* 1962年 - [[霍克獵手戰鬥機|霍克獵手戰鬥機]]（Hawker Hunter）
* 1968年 - 韋斯特蘭 旋風直升機（Westland Whirlwind）

第28中隊從1972年起採用[[威塞克斯式直昇機|韋斯特蘭韋塞克斯直升機]]。1978至1996年駐守[[英屬香港|英屬香港]][[石崗機場|石崗皇家空軍基地]]期間，就是採用這一款直升機來代表[[駐港英軍|駐港英軍]]執行搜救、警務、撲滅山火和緊急醫療運送等任務。1996至1997年6月英軍撤離期間，部隊調回皇家空軍啓德基地，並且是最後一隊撤離香港的英國皇家空軍部隊。<ref name="raf"/><ref name="rafweb"/><ref name="ehattons"/><ref name="geo"/>

==現時角色==
1995年3月，皇家空軍訂購了22架灰背隼HC3直升機，首架在2001年3月7日由當時的GKN韋斯特蘭付運。第28中隊在2001年7月17日正式在[[本臣空軍基地|本臣空軍基地]]重組，是該部隊長期以來，第一次駐守英國本土。<ref name="raf"/><ref name="rafweb"/><ref name="ehattons"/><ref name="geo"/>

2003年4月1日，[[巴爾幹半島內戰|巴爾幹半島內戰]]期間，第28中隊首次以[[北約|北約]]維持和平部隊之名，被派駐[[波士尼亞|波士尼亞]]的巴尼亞盧卡城。同日，灰背隼直升機取得它的作戰能力批核，亦是[[中隊|中隊]]首次正式使用[[奧古斯塔偉士蘭AW101灰背隼直升機|灰背隼直升機]]執行任務。2004年3月31日, 28中隊終止[[波士尼亞|波士尼亞]]的任務，並開始為在[[伊拉克|伊拉克]]執行任務作準備。 <ref name="raf"/><ref name="rafweb"/><ref name="ehattons"/><ref name="geo"/>

2005年3月1日開始到現在，第28中隊展開[[英軍|英軍]]在[[伊拉克|伊拉克]]境內執行的TELIC作戰計劃，並駐守[[巴士拉|巴士拉港]]。<ref name="raf"/><ref name="rafweb"/><ref name="ehattons"/><ref name="geo"/>

由於經[[丹麥|丹麥]]增購了6架灰背隼HC3A直升機，為免令第28中隊在管理上變得難於操作，2007年12月3日第28中隊被分為3小隊：分別為1─第28中隊；2─第78中隊和3─[[奧古斯塔偉士蘭AW101灰背隼直升機|灰背隼直升機]]工程中隊。<ref name="raf"/><ref name="rafweb"/><ref name="ehattons"/><ref name="geo"/>

第28中隊現在分為1個總部飛行隊，2個任務飛行隊和[[奧古斯塔偉士蘭AW101灰背隼直升機|灰背隼直升機]]任務改装飛行隊。現時的總指揮是李治·陸克先生。<ref name="raf"/><ref name="rafweb"/><ref name="ehattons"/><ref name="geo"/>
