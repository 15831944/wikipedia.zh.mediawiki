'''英聯邦王國'''（{{lang-en|Commonwealth realm}}）是[[英聯邦|國協]]之中共戴[[英國君主|同一位君主]]為[[君主立憲制|君主立憲制]][[國家元首|國家元首]]的[[獨立國家|獨立國家]]<ref name="Buck1">{{cite web |url = http://www.royal.gov.uk/MonarchAndCommonwealth/QueenandCommonwealth/WhatisaCommonwealthRealm.aspx |title = What is a Commonwealth Realm? |accessdate = 2013-07-23 |deadurl = yes |archiveurl = https://web.archive.org/web/20101202194643/http://www.royal.gov.uk/MonarchAndCommonwealth/QueenandCommonwealth/WhatisaCommonwealthRealm.aspx |archivedate = 2010-12-02 }}</ref>。目前共有16個英聯邦王國，而這些國家的國家元首同為[[伊莉莎白二世|伊莉莎白二世]]女王。英聯邦王國互不隸屬，但一起構成迄今为止世界上现存最大的一个[[共主邦联|共主邦联]]，而每個英聯邦王國也分別是[[大英國協|國協]]范围内的[[主權國家|主權國家]]。2010年，這16国[[領土|領土]]面积总合約1870万平方公里，人口数共约1.34億。女王在各個英聯邦王國之中，也有相應不同的稱號，例如在[[英國|英國]]稱為「'''蒙上帝恩典，大不列顛及北愛爾蘭聯合王國與其屬土及領地之女王，[[英聯邦元首|國協元首]]，信仰的保護者'''」，而在[[巴巴多斯|巴巴多斯]]的稱號則為「'''蒙上帝恩典，巴巴多斯女王，國協元首'''」。

對於[[英國|英國]]以外的英聯邦王國，身為國家元首的女王會依據該國[[總理|總理]]的提名，任命[[總督|總督]]作為自己的全權代表。而[[澳大利亞|澳大利亞]]各州的總督（州督）和[[加拿大|加拿大]]各省的總督（省督兼聯邦副總督），也是女王不在該國直接行使元首職權時在當地的全權代表。這些官員可以執行的權利，不但與君主立憲制度之下的君主相若，而且還可以代表女王，按照該國憲法行使一些王家特權。

在16個英聯邦王國中，14個曾經是英國的自治殖民地。這些國家多在1931年西敏法令頒布和1961年[[西印度群島聯邦|西印度群島聯邦]]解體後成為英聯邦王國的。餘下2個例外分別是[[巴布亞新幾內亞|巴布亞新幾內亞]]和[[英國|英國]]（各殖民地的原宗主國）。巴布亞新幾內亞一部分原為英國殖民地，1905年交與澳大利亞管轄；另一部分原為德國殖民地，[[一戰|一戰]]中由澳大利亞佔領，後割讓予澳大利亞作爲[[國際聯盟|國際聯盟]]托管領土。其后，兩部分同時獨立並合併，加入大英國協。

而在[[大英國協|國協]]內，英國和其他國家是平等的。值得一提的是，[[斐濟|斐濟]]雖然以英女王為'''最高酋長'''，但這個並非國家元首的稱號，因此斐濟不算為一個英聯邦王國。

==起源==
直到20世纪初，英国政府对[[大英帝国|大英帝国]]在全世界的殖民地和属地仍旧保持不同程度的控制。虽然加拿大、澳大利亚等殖民地都已先后「建国」并拥有正常运作的自治政府，但英国政府的殖民地部仍对这些殖民地保留监督和干预的权力，英国国王按照英国政府的建议在各自治殖民地任命总督，各地总督不仅是当地的首脑，也负责当地政府和伦敦的帝国政府之间的联络。而在其他一些殖民地，例如印度，英国政府则仍旧维持实质性的控制权。

鑒於在[[第一次世界大戰|第一次世界大戰]]之后在加拿大、澳大利亚、新西兰等主要殖民地内的完全自治呼声日渐增强，并且由于各殖民地自身政治日渐成熟以及英国本身的国力相对衰退，[[英國|英國]]在戰後陸續允許一些[[英國殖民地|殖民地]]更大的自治权，並改稱英屬[[自治領|自治領]]。英国政府对自治领内部事务的参与逐渐减少，在帝国内逐渐衍生了新的惯例，即君主对于各殖民地内部事务依照殖民地政府的建议行事，而不是英国政府。1926年，英國政府發表《1926年貝爾福宣言書》，宣佈各自治領应成為[[政治|政治]]和[[外交|外交]]上完全獨立，以英王為[[國家元首|國家元首]]的主權國家。也即是各自治領的[[總督|總督]]不再是英國政府[[殖民地部|殖民地部]]的代表，而是英王指派的全權代表。

貝爾富宣言的後續效應陸續有來，首先是《[[s:1927年王家和議會頭銜法令|s:1927年王家和議會頭銜法令]]》，承認[[愛爾蘭自由邦|愛爾蘭自由邦]]的獨立地位。於1931年12月11日，英國國會通過《[[1931年西敏法令|1931年西敏法令]]》，給予各英屬自治領完全自主權，國際場合上是與英國完全平等的[[主權國家|主權國家]]。加拿大首先於同年確認西敏法令，成為第一個英聯邦王國。其他自治領和一些前英殖民地在[[第二次世界大戰|第二次世界大戰]]後通過西敏法令，獲得獨立的立法權，成為主權國家，不過依然奉英王為國家元首，但君主在各王国以不同身份担任该国元首，因此这些国家與[[英國|英國]]組成[[共主邦聯|共主邦聯]]。<ref>

== 王權的更改 ==
由於《1931年西敏法令》並無保障英王在各英聯邦王國內的元首地位，因此，各國可以通過公投、立法、修憲或頒布新憲以廢除與英國共主邦聯的關係。因此在各英聯邦王國國內，都出現了要求廢除[[英国君主|英国君主]]元首地位的聲音。這些聲音導致各英聯邦王國內都出現'''共和派'''和'''保王派'''的分野。近年來，各國的共和派的勢力漸漸抬頭，導致多個英聯邦王國都舉辦了公投，決定英王在該國的地位。20世紀后半葉英聯邦王國的組成趨於穩定，自1992年到目前為止沒有英聯邦王國選擇擺脫與英國的共主邦聯的關係。

2012年各英聯邦王國在澳大利亞[[珀斯|珀斯]]舉行的國協政府首腦會議框架下達成共識，將修改有關各境域共戴君主的繼承權問題的法律以減少對天主教徒的歧視以及確立男女繼承權平等的原則，此共識成爲“珀斯協定”。這是自從各境域“互不隸屬、共戴一王”的現代關係形成后第一次嘗試改變王位繼承規則，因此在各境域衍生出一系列的法律和憲法層面的問題。例如：2013年初，澳大利亞[[昆士蘭州|昆士蘭州]]曾宣佈不將修法權交予澳大利亞聯邦，而將自行立法修改王位繼承法律。澳大利亞和英國政府以及一些憲法專家当时擔心此擧可能會造成昆士蘭州在法律上脫離澳大利亞的王權，形成與包括澳大利亞在内的其它境域分開的“昆士蘭女王”。此问题最后由澳大利亚联邦采取强硬立场以宪政危机威胁昆士兰，导致昆士兰屈服而解决。此外，在一部分加勒比海境域還出現爭議，辯論當地憲法是否允許王位繼承權不經自主立法而隨英國法律的變更而變更。总的来说，由于各国修法进度不同，在一定时期内各王国的王位继承法出现不一致，直到所有王国完成修法并使新规则从同一日期起生效为止。2015年3月26日，[[英國副首相|英國副首相]]宣佈，所有英聯邦王國都依據各自法律完成對王室權利法律的修正；同日，新的法律在所有英聯邦王國境内生效<ref name="inforce">[http://www.parliament.uk/business/publications/written-questions-answers-statements/written-statement/Commons/2015-03-26/HCWS490/ Statement by Nick Clegg MP, UK parliament website,] 26 March 2015 (retrieved on same date).</ref>。
