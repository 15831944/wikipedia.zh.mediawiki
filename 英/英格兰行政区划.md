[[英國|英國]][[英格蘭|英格蘭]]的[[行政區劃|行政區劃]]共劃分爲四個層級，而實際上只有「郡級」和「區級」兩個層級設有政府機構，其它則為一般民間[[法人|法人]]機構代為運作的'''虛級化'''層級。

== 最高行政區劃 - 地域層級 ==
[[File:EnglandSubdivisions.png|thumb]]
{{Main|英格蘭的區域}}

英格蘭的最高行政區劃爲'''[[英格蘭的區域|區域]]'''（Regions），一共有9個區域，其中包含可以被視為[[英格蘭的都市郡和非都市郡|郡級]]的[[大倫敦|大倫敦]]地區。每一個區域下包含數個實體的郡級地區，1994年經由立法程序正式設立，並從1999年起成為[[歐洲議會|歐洲議會]]英格蘭選區界線。

== 一級行政区划 - 郡層級 ==
{{Main|英格蘭的都會郡和非都會郡}}
[[File:EnglandUnitaries.png|thumb]]地圖；粉紅色為非都會郡（shire counties)；紫色為都會郡（metropolitan borough）及大倫敦區；紅色為單一管理區。]]

英格蘭依國會在1997年通過的《[[1997年郡尉職權法案|1997年郡尉職權法案]]》（Lieutenancies Act）委任[[郡尉_(英國)|郡尉]]（Lord-Lieutenants - 英格蘭郡的君主個人代表）代表某地域，這些地域也順理成章為[[名譽郡|名譽郡]]。因各個名譽郡的面積過大，其地名只適用於當地居民身份及英國國會選區界線，但並不作為各地方政府的行政区划看待。

現行制度將英格蘭分為以下四類郡級行政區劃：
* [[都市郡|都市郡]]（Metropolitan county） - 1974年成立，分別為[[大曼徹斯特郡|大曼徹斯特郡]]、[[默西賽德郡|默西賽德郡]]、[[南約克郡|南約克郡]]、[[泰恩-威爾郡|泰恩-威爾郡]]、[[西約克郡|西約克郡]]及[[西密德蘭郡|西密德蘭郡]]等6郡。其下管轄「都市區」（metropolitan district），其中甚至有大部分被進一步賦與自治資格而成為[[都會自治市|都會自治市]]（Metropolitan borough）。1986年，都市郡政府被撤消，權利下放給都市自治市，自治市因而成為事實上的單一管理區。
* [[非都市郡|非都市郡]]（non-metropolitan counties） - 1974年成立；其下管轄[[非都市郡區|非都市郡區]]（{{lang|en|non-metropolitan district}}）。
* [[單一管理區|單一管理區]]（Unitary authority） - 1995年起成立。單一管理區合併郡和區兩級政府的功能，管理區內不再分區，如[[懷特島郡|懷特島郡]]。[[伯克郡|伯克郡]]為特例：該郡在名義上存在，但郡政府已被撤消，下轄各區成為事實上的單一管理區。
* [[大倫敦|大倫敦]]（Greater London） - 1965成立；分為[[倫敦市|倫敦市]]及32個[[倫敦自治市|倫敦自治市]]。由1999年實施的國家[[權力下放|權力下放]]，將大倫敦交予由定期民選產生的[[倫敦市長_(大伦敦)|倫敦市長]]及市議員組成的[[大倫敦政府|大倫敦政府]]管理。

== 二級行政區劃 - 區層級（自治市鎮）==
{{Main|英格蘭的區}}

英格兰的二級行政區有以下四類[[自治市鎮|自治市鎮]]：

* [[都市自治市|都市自治市]]（Metropolitan district/borough） - 又稱'''都市郡區'''，從屬[[都市郡|都市郡]]。由於都市郡政府都已撤消，都市區接管郡級權力，事實上為[[單一管理區|單一管理區]]。
* [[非都市郡區|非都市郡區]]（Non-metropolitan district） - 從[[非都市郡|非都市郡]]分劃，和當地郡議局分享權力。
* [[倫敦自治市|倫敦自治市]]（London Borough） - 32個倫敦自治市議會享有[[單一管理區|單一管理區]]的地位，但[[大倫敦政府|大倫敦政府]]仍管理各自治鎮內的公共運輸、治安、消防、經濟發展等範籌。
* [[單一管理區|單一管理區]]（Unitary authorities）- 也可以算是二級行政區劃（[[英格蘭的區|區層次]]），它是合併、且跨越「郡」和「區」兩級政府的功能和權限。

== 民政區 ==
[[民政教區|民政教區]]（Parish level）是英格蘭地方最低層級行政區。 除了大倫敦受法律上准許外，英格蘭開始盛行民政區的成立。

== 2004年区划修訂草案 ==
2004年11月4日，東北英格蘭的成立區域議會[[公民投票|公民投票]]被否決。居住東北英格蘭的選民兼要選擇一旦成立時的單一管理區運作模式。

但東北英格蘭公民投票的否決，令副首相[[彭仕國|彭仕國]]宣佈，西北英格蘭和[[約克郡及亨伯|約克郡及亨伯]]同類的公民投票將無期壓後<ref>[http://www.odpm.gov.uk/stellent/groups/odpm_about/documents/page/odpm_about_032697.hcsp Statement by Deputy Prime Minister] {{webarchive|url=https://web.archive.org/web/20061211182837/http://www.odpm.gov.uk/stellent/groups/odpm_about/documents/page/odpm_about_032697.hcsp |date=2006-12-11 }}</ref>。
