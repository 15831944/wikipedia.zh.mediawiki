{{vgname2|英雄联盟2016赛季全球总决赛|en=''League of Legends 2016'' World Championship|也被称为英雄联盟S6全球总决赛}}是由[[Riot_Games|Riot Games]]公司举行的第六届[[英雄联盟全球总决赛|英雄联盟全球总决赛]]。本年度的世界冠军为[[SK_Telecom_T1|SK Telecom T1]]，<ref>{{Cite web |url = https://gnn.gamer.com.tw/7/139247.html |title = 《英雄聯盟》2016 世界大賽隊伍 SKT 蟬聯冠軍 以 3:2 成績擊敗 SSG 獲勝 |author = Jessica |date = 2016年10月30日 |publisher = [[巴哈姆特電玩資訊站|巴哈姆特電玩資訊站]] |language = zh-tw }}</ref>亚军为Samsung Galaxy，而H2k-Gaming跟ROX Tigers并列第3。该届主题曲为《[[燃（歌曲）|燃]]》（Ignite）。<ref>{{Cite web |url = http://www.gamedog.cn/news/20160927/1892889.html |title = S6全球总决赛主题曲上线 电音风格强烈骚气十足 |author = We're |date = 2016年9月27日 |publisher = 游戏狗 |language = zh-cn }}</ref>

据统计，该届比赛总计独立观众数达到了3.9亿，其中在SK Telecom T1对阵Samsung Galaxy的冠军战中，更是出现了同时1470万在线观看的高峰，而该场比赛的独立观看人数也达到了4300万人，成为英雄联盟全球总决赛中观看人数最多的冠军战。<ref>{{Cite web |url = http://games.qq.com/a/20161207/018216.htm |title = 2016《英雄联盟》全球总决赛观赛数据公布 |date = 2016年12月7日 |publisher = 腾讯游戏 |language = zh-cn }}</ref>

==参赛队伍==
本次比赛共有16支队伍参加，其中北美，欧洲，中国，韩国各有3支队伍参赛，台港澳区有2支队伍参赛，另有一支巴西队伍和一支独联体队伍持外卡参赛。<ref>{{Cite web |url = http://games.qq.com/a/20160906/023964.htm |title = LOL S6总决赛16支参赛队名单全部出炉 9月11日抽签 |date = 2016年9月6日 |publisher = 腾讯游戏 |language = zh-cn }}</ref>

2016年10月7日，[[I_May|I May]]辅助选手Road与{{link-en|Froggen|Froggen}}在一局游戏中互相辱罵。在该局游戏中，由于前期双方多人在中路进行大乱斗，导致Froggen死亡7次，之后Road和Jinjiao集結攻擊对面外防御塔，但随后Jinjiao和Road均被对手擊殺，于是Road发起投降，但因其队友不同意，使得Road发了一长条G，并對Froggen人身攻擊，Froggen则解释说对面布里茨玩的太好。随后两人开始争吵。由于此时Froggen正在直播，使得Road在两人争吵期间所有言论均被粉丝记录下来。事后，关于此事的帖子被欧洲网友顶上热门，随后很多人请求Riot Games惩罚Road。<ref>{{Cite web |url = http://games.qq.com/a/20161007/022379.htm |title = IM膨胀起来在搞事？辅助Road辱骂Froggen遭禁赛 |date = 2016年10月8日 |publisher = 腾讯网 |language = zh-cn }}</ref>北京时间2016年10月8日，Riot Games判决Road禁赛一场，并对其罚款2000美元。<ref>{{Cite web |url = http://lol.tgbus.com/news/ssxw/420001.shtml |title = Road违规已被禁赛！IM战队现已面临重大危机 |author = Sup |date = 2016年10月8日 |publisher = 巴士LOL |language = zh-cn }}</ref>
