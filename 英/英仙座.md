'''英仙座'''跨越了[[秋季|秋季]]的[[银河|银河]]，所以对于[[天文爱好者|天文爱好者]]来说，不管是使用[[双筒望远镜|双筒望远镜]]还是其他望远镜，这裡都是搜寻天体的好地方。这个[[星座|星座]]有很多耀眼的亮星。即使是在市郊也能看见该星座轮廓。银河在该星系不像在旁边的[[天鹅座|天鹅座]]那样明亮，这使得人们能够看见很多亮[[星团|星团]]，[[气体星云|气体星云]]和[[行星状星云|行星状星云]]。这个星座有两个[[梅西耶天体|梅西耶天体]]：[[M34|M34]]与[[M76|M76]]（见下）。

== 历史与神话 ==
在有关北天的[[希腊神话|希腊神话]]中，英仙[[珀尔修斯|珀尔修斯]]是一个中心人物。他杀死了[[戈耳工|戈耳工]][[美杜莎|美杜莎]]，用美杜莎的[[头|头]]把海中的[[鲸|鲸]]怪变为石头。由此他解救了缚在鲸怪经过[[岩石|岩石]]上的[[安德洛墨达|安德洛墨达]]。本星座代表的就是珀尔修斯。

== 有趣的星 ==
* [[英仙座β|英仙座β]]星（[[大陵五|大陵五]]）:这是最早被人类所认知的[[變星#外因變星|食变星]]。由一颗主星和一颗光线微弱的伴星组成。伴星有规律的挡住主星的光。大陵五在2.867[[日|日]]的光变周期中以[[星等|星等]]在+2.1到+3.3间变化。

== [[深空天体|深空天体]] ==
* [[雙星團|h+χ雙星團]]：這兩個[[疏散星團|疏散星團]]（分別是NGC 869和NGC 884）是以雙筒鏡和小望遠鏡觀賞夜空時最美麗的目標之一。這一對著名的雙星團可能早在西元前130年的[[喜帕恰斯|喜帕恰斯]]星表上就已經存在了，兩個星團都屬於英仙座OB 1星協，兩者相距僅數百光年''χ''被認為是NGC 884，距離地球7,600光年，''h''是NGC 869，距離地球6,800光年。
* [[疏散星團|疏散星團]][[M34|M34]]：這個疏散星團總體的視星等為5.5等，距離大約1,400光年，有100餘顆恆星散佈在比滿月還大的範圍內，星團實際的大小約14光年。M34也是雙筒望遠鏡觀賞的好目標，使用望遠鏡看時則以低倍率的效果最佳。
* [[M76|M76]]：這是一個[[行星狀星雲|行星狀星雲]]，也稱為[[小啞鈴星雲|小啞鈴星雲]]。它位于[[NGC_869|NGC 869]]和[[NGC_884|NGC 884]]双星团西南8度地方，是[[梅西耶天体列表|梅西耶天体列表]]中仅有的4个[[行星状星云|行星状星云]]之一。他的視直徑大約65角秒，視星等10.1等，必须使用3英寸（8厘米）的望远镜，放大120倍才能看清它。
* [[NGC_1499|NGC 1499]]：也稱為[[NGC_1499|加州星雲]]，這是一個[[發射星雲|發射星雲]]，在1884-85年間被美國天文學家[[愛德華·愛默生·巴納德|巴納德]]發現，是天文攝影的一個好目標。由於表面的亮度很低，所以不適合以裸眼進行目視觀測。
* [[NGC_1333|NGC 1333]]：是一個恆星誕生場所的反射星雲。
* [[NGC_1023|NGC 1023]]：这是一个透镜状[[椭圆星系|椭圆星系]]。使用4英寸（10厘米）或更大的望远镜可见。
* [[IC_348|IC 348]]：疏散星團，恆星形成區域。

== 流星群 ==
* [[英仙座流星雨|英仙座流星雨]]：这个是每年出现的流星群之一。高峰在8月11日或8月12日。每小时可见50 - 60颗[[流星|流星]]。它过去的一个名字是'''圣劳伦斯的眼泪''' - 这是因为每年这个时候都会举办纪念[[圣劳伦斯|圣劳伦斯]]的活动。
