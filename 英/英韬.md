'''英韬'''（原名'''杨筠生'''，{{bd|1925年||2012年|3月25日|catIdx=Ying英韬}}）出生于[[天津|天津]]，毕业于[[华北联合大学|华北联合大学]]，为近现代著名[[漫画家|漫画家]]。<ref name="fhws ">{{cite web |url=http://culture.ifeng.com/whrd/detail_2012_03/28/13494564_0.shtml |title=漫画家英韬逝世享年87岁 曾创办《讽刺与幽默》 |publisher= 凤凰网|accessdate= 2012年|work= |language =zh-cn}}</ref><ref name="zhsw ">{{cite web |url=http://www.yangtse.com/system/2012/03/28/013032198.shtml |title=漫画家英韬逝世享年87岁 曾创办《讽刺与幽默》 |publisher=扬子晚报 |accessdate=2012年 |work= |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20160305042133/http://www.yangtse.com/system/2012/03/28/013032198.shtml |archivedate=2016-03-05 }}</ref><ref name="mas ">{{cite web |url=http://www.chinadaily.com.cn/micro-reading/dzh/2012-03-27/content_5531746.html |title=著名漫画家英韬去世 |publisher= 中国日报网|accessdate= 2012年|work= |language =zh-cn}}</ref>

== 生平 ==
1925年，英韬出生于[[天津|天津]]，取名“杨筠生”。<ref name="fhws "/><ref name="zhsw "/><ref name="mas "/>

1949年，英韬毕业于[[华北联合大学|华北联合大学]]艺术学院美术系。<ref name="fhws "/><ref name="zhsw "/><ref name="mas "/>
大学毕业后，英韬先后出任《[[人民铁道报|人民铁道报]]》编辑、记者，《[[人民日报|人民日报]]》美术组编辑，《[[人民日报|人民日报]]》文学艺术部副主任，《[[讽刺与幽默|讽刺与幽默]]》主编。<ref name="fhws "/><ref name="zhsw "/><ref name="mas "/>

2012年3月25日，英韬因为[[膀胱癌|膀胱癌]]病逝。<ref name="fhws "/><ref name="zhsw "/><ref name="mas "/>

== 评价 ==
漫画家[[徐鹏飞|徐鹏飞]]：“英韬先生的逝世是《讽刺与幽默》的巨大损失，也是中国漫画界的巨大损失。 ”<ref name="fhws "/><ref name="zhsw "/>
