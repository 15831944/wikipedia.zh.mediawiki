{{vgname2|英雄联盟季中邀请赛|en=Mid-Season Invitational|aka=英雄联盟季中冠军赛|简称MSI}}是由[[Riot_Games|Riot Games]]公司举行的《[[英雄联盟|英雄联盟]]》[[电子竞技|电子竞技]]大赛，该赛事成立于2015年，每年举办一届，每届比赛都会有来自全球14大赛区（即[[英雄联盟职业联赛|LPL赛区]]、[[英雄联盟韩国冠军联赛|LCK赛区]]、[[LMS職業聯賽|LMS赛区]]、{{link-en|英雄聯盟冠軍聯賽|League of Legends Championship Series|NA LCS赛区}}、{{link-en|英雄聯盟冠軍聯賽|League of Legends Championship Series|EU LCS赛区}}、[[GPL超級聯賽|GPL赛区]]、[[英雄联盟日本联赛|LJL赛区]]、{{link-ru|英雄联盟大洋洲职业联赛|Oceanic Pro League|OPL赛区}}、{{Link-pt|英雄联盟巴西联赛|Campeonato Brasileiro de League of Legends|CBLoL赛区}}、{{link-ru|Liga Latinoamérica Norte de League of Legends|Liga Latinoamérica Norte de League of Legends|LLN赛区}}、{{link-ru|Copa Latinoamérica Sur de League of Legends|Copa Latinoamérica Sur de League of Legends|CLS赛区}}、[[英雄联盟欧陆联赛|LCL赛区]]、[[英雄联盟土耳其冠军联赛|TCL赛区]]、VCS赛区）的14支队伍参赛，参赛队伍为对应赛区的春季赛季后赛的冠军。而在2017年以前，外卡赛区的参赛队伍则为该年的{{link-ru|国际外卡邀请赛|IWCI}}（IWCI）的冠军。<ref>{{Cite web |url = http://games.sina.com.cn/o/z/lol/2015-03-05/1032606243.shtml |title = LOL季中赛公告：全球各赛区冠军队一决高低 |date = 2015年3月5日 |publisher = 新浪游戏 |language = zh-cn }}</ref>目前该比赛已举办4届。

== 2015季中邀请赛 ==
{{main|英雄联盟2015季中邀请赛}}
2015季中邀请赛于2015年5月7日-10日在[[美国|美国]][[佛罗里达州|佛罗里达州]][[塔拉哈西|塔拉哈西]]的[[佛罗里达州立大学|佛罗里达州立大学]]的{{link-en|Donald L. Tucker Civic Center|Donald L. Tucker Civic Center}}举行，总奖金达200,000美金。<ref>{{Cite web |url = http://lol.duowan.com/1504/293104231862.html |title = LOL季中赛赛事简介：6支战队争夺20W美元
| date = 2015年4月16日 |publisher = 多玩 |language = zh-cn }}</ref>在该届比赛中，中国大陆战队[[EDward_Gaming|EDward Gaming]]夺得冠军，而韩国战队[[SK_Telecom_T1|SK Telecom T1]]则取得亚军。<ref>[http://lol.qq.com/webplat/info/news_version3/152/4579/4580/3106/201505/341334.shtml 登顶世界之巅 EDG战队勇夺季中邀请赛冠军]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 2016季中邀請賽 ==
{{main|2016英雄联盟季中冠军赛}}
2016季中邀請賽於2016年5月4日-14日在位于[[中國|中國]][[上海|上海]]的[[上海东方体育中心|东方体育中心]]舉行，總獎金達450,000美金。<ref>{{Cite web |url = http://games.qq.com/a/20160217/018649.htm |title = 2016英雄联盟季中邀请赛（MSI）来到上海 |date = 2016年2月17日 |publisher = 腾讯网 |language = zh-cn |deadurl = yes |archiveurl = https://web.archive.org/web/20161222170437/http://games.qq.com/a/20160217/018649.htm |archivedate = 2016年12月22日 }}</ref>在該屆比賽的冠軍戰上，來自韓國的[[SK_Telecom_T1|SK Telecom T1]]以3：0擊敗來自北美的{{link-en|Counter Logic Gaming|Counter Logic Gaming}}，并奪得冠軍。<ref>{{Cite web |url = http://gnn.gamer.com.tw/6/131846.html |title = 《英雄聯盟》季中邀請賽韓國 SKT 贏得冠軍 拿下 LOL 電競全球首度大滿貫成就 |author = RU |date = 2016年5月15日 |publisher = [[巴哈姆特電玩資訊站|巴哈姆特電玩資訊站]] |language = zh-tw }}</ref>同時，SK Telecom T1也成為英雄聯盟史上第一支大滿貫隊伍。<ref>{{Cite web |url = http://www.4gamers.com.tw/news/post/22920 |title = 2016MSI／SKT直落三擊敗CLG，史上首次電競大滿貫成就 |author = Panda |date = 2016年5月15日 |publisher = [[4gamers|4gamers]] |language = zh-tw |deadurl = yes |archiveurl = https://web.archive.org/web/20160719022032/http://www.4gamers.com.tw/news/post/22920 |archivedate = 2016年7月19日 }}</ref>

== 2017季中邀請賽 ==
{{main|2017英雄联盟季中冠军赛}}

2017季中邀請賽於2017年4月8日-5月21日在[[巴西|巴西]]舉行，總獎金達1,690,000美金。其中入圍賽於4月28日-5月6日在[[聖保羅_(巴西)|圣保罗]]的[[CBLOL_Studio|CBLOL Studio]]舉行，而小組賽和淘汰賽則於5月10日-21日在[[里約熱內盧|里約熱內盧]]的[[里約奧林匹克體育館|里約奧林匹克體育館]]舉行。該屆比賽的冠軍為[[SK_Telecom_T1|SK Telecom T1]]，亞軍為[[G2_Esports|G2 Esports]]，而[[Team_WE|Team WE]]與[[Flash_Wolves|Flash Wolves]]併列第3。

== 2018季中邀请赛 ==
{{Main|2018英雄联盟季中邀请赛}}2018英雄联盟季中邀请赛移师欧洲，入围赛及小组赛于2018年5月3日至15日在德国[[柏林|柏林]]的EU LCS Studio举行，而之后的淘汰赛在2018年5月18日至20日来到法国[[巴黎|巴黎]]的维特雷-天顶体育馆。由RNG（LPL賽區/中）、FW（LMS賽區/台港澳）、和KZ（LCK賽區/韓）和FNC（ECS EU賽區/歐）四隊晉級進行淘汰赛，最終中國的RNG於決賽以3-1戰勝韓國的KZ，為LPL繼2015年拿下的第二次冠軍。
