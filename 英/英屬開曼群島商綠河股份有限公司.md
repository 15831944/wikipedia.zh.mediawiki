'''英屬開曼群島商綠河股份有限公司'''（{{Lang-en|Green River Holding Co. Ltd.}}，簡稱'''綠河股份有限公司'''，西元2000年1月由[[台灣|台灣]][[企業家|企業家]]謝榮輝以天然[[橡膠|橡膠]]實木板材加工起家，主要設廠於[[泰國|泰國]][[南部|南部]][[宋卡府|宋卡府]]邊境區域<ref>{{Cite web |url = http://www.chinatimes.com/newspapers/20180304000237-260202 |title =綠河董事長 謝榮輝 創造泰國台商奇蹟|author =彭媁琳|date = 2018-03-04 |publisher =工商時報| accessdate = 2018-07-02 }}</ref>。西元2011年1月14日公司更名為「'''英屬開曼群島商綠河股份有限公司'''」，發展塑合板及[[橡膠|橡膠]]實木板材<ref>{{Cite web |url = https://www.youtube.com/watch?v=RERBvRl_N6o |title = 【MoneyDJ財經新聞】站穩泰國放眼亞洲 F-綠河搶攻綠色商機 |author = 葉家瑋 |date = 2016-04-25期 |publisher = MoneyDJ財經新聞  |accessdate = 2018-07-02 }}</ref>，其股票於西元2013年7月於櫃買中心登錄興櫃，並於2015年10月正式上櫃掛牌<ref>{{cite news|title=【消失的國界】泰國最大橡膠產地　台商點「皮」成金 |url=https://www.youtube.com/watch?v=Z8zY0EKy3M8&t=4s |language=zh-hant |author=李天怡主 |publisher=三立新聞台 |date=2015-08-24 |accessdate=2018-07-21 }}</ref>。

==歷史沿革==
英屬開曼群島商綠河股份有限公司創辦人是謝榮輝，其父為[[台灣|台灣]]底層之[[礦工|礦工]]。1990年謝榮輝懷抱創業夢想，與國小同學黃登士到[[泰國|泰國]]賣售[[主機板|主機板]]，因泰國經濟不景氣，1993年經商失敗返台。與其妻子在[[台灣|台灣]]重新出發，創業生產棧板，5年賺進千萬元。因生意所需，謝榮輝數次前往馬來西亞、印尼等地找木材原料，意外發現在泰國生產塑合板的機會，於是1997年再次附[[泰國|泰國]][[南部|南部]]宋卡府邊境區，與其國小同學黃登士(現任綠河總經理)、大學好友李木文(現任綠河銷售長)共同開設實木加工廠。2003年綠河股份有限公司已成為泰國最大橡膠實木出口商。2008年遇上全球金融風暴，公司產線一度停擺，面臨倒閉危機，幸而2009年原料市場需求短缺，導致原料降價，轉而獲利。2017年公司營收超過34億，已成為[[亞洲|亞洲]]最大的[[橡膠|橡膠]]木塑合板製造商<ref>{{cite news|title=《財訊雙週刊》488期-賺不停台灣好公司 |url=https://www.pubu.com.tw/magazine/58343 |language=zh-hant |publisher=財訊雙週刊/財信雜誌社股份有限公司  |date=2015-10-21 |accessdate=2018-07-21 }}</ref>。

== 公司主要產品 ==
公司主要產品是塑合板, 貼皮板及實木板材。

=== 實木板材 ===
「實木板材」是採用[[橡膠|橡膠]]樹木直接加工成板材，與拼接實木不同，可應用於[[家具|家具]]、[[地板|地板]]、複合式[[建材|建材]]及其他裝飾[[材料|材料]]等。其[[原料|原料]]之基本要求是質地堅密、含水率低、纖微均勻、木節少。優點耐久、不易變形。<ref name="h2 ">{{cite web|url= https://www.ettoday.net/news/20180210/1111725.htm |language=zh-hant |title=泰南拓荒廢材變黃金　綠河董事長謝榮輝|publisher=ETtoday新聞雲 | date=2018-02-10 |accessdate=2018-07-16 }}</ref>。

=== 塑合板 ===
「塑合板」是高級均質的刨花板。將[[橡膠|橡膠]]木枝椏材和小徑木，加工成木片，經處理、篩選後，以膠合劑[[高溫|高溫]]、[[高壓|高壓]]製成大塊素板。

=== 貼皮板 ===
塑合板經二次加工，外層壓合一層薄薄的[[美耐皿|美耐皿]](可防水、防火、耐磨)。貼皮板可應用於廚具、[[家具|家具]]、[[綠色建築|綠色建築]]及[[環保|環保]]建材等<ref name="h2 "/>。
