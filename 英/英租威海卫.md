'''英租威海衛'''，為[[英國|英國]]於1898年至1930年在[[大清|大清]][[山東省_(清)|山東省]]建立的[[租借地|租借地]]。英租威海衛的發展模式不同於[[英屬香港|英屬香港]]，佔據此地的主要目的是制衡[[俄羅斯|俄羅斯]]佔領[[旅順|旅順]]的影響；此地主要作用[[英國皇家海軍|英國皇家海軍]][[訓練|訓練]]基地及休養地。1924年6月，中華民國與大不列顛及北愛爾蘭聯合王國两国订立《[[收回威海衛條約|收回威海衛條約]]》；1930年10月1日，國民政府收回威海衛大部，唯[[劉公島|劉公島]]為大英帝國海軍續租十年，直至1940年撤退。

== 历史 ==
1898年（大清國[[光緒|光緒]]二十四年）3月28日，[[大不列颠及爱尔兰联合王国|大不列顛及愛爾蘭聯合王國政府]]向[[清朝|大清國政府]]要求租借威海衛。1898年7月1日，大清與[[英國|英國]]雙方共同簽署《[[订租威海卫专条|威海卫租界專條]]》，租期與俄答大旅相同，為期25年。1930年10月，国民政府收复除[[刘公岛|刘公岛]]以外的所有英租威海卫领土，英国续租刘公岛十年，到1940年刘公岛被日军占领。而中国政府再次控制刘公岛则要到1945年[[二战|二战]]结束后才得以实现。

== 行政 ==
1898年，英国在威海卫成立临时行政署，隶属英国海军部。1899年，行政署划归陆军部管辖。1901年起，行政署归殖民部管辖。

1901年，英国成立威海卫行政公署，为租借地最高管理机关。行政公署專員其下设正华务司、副华务司和医官长。英国当局初期对农村地区基本无为而治。从1905年起，在农村推行总董制，以加强管制。

== 经济 ==
此时的威海卫为[[自由港|自由港]]，货物进出口免关税，因而工商业得到发展。在这一时期，旅游业得到一定的发展。旅游项目有狩猎、温泉等。

== 军警 ==
1899年，英国在威海卫创建[[威海卫华勇营|中国军团]]，1906年解散。

军团解散后，部分士兵转业成为警察，到1910年发展成由3名来自欧洲的警督及55名华人警员组成的警察队伍。

1922年香港爆发[[香港海員大罷工|海员罢工]]事件后，香港警察派人前往威海卫招募[[香港威海卫警察|警员]]。

== 教育 ==
皇仁学堂为唯一一所由殖民当局所设的学校。
