'''英國獨立黨'''（{{lang-en|United Kingdom Independence Party}}，[[缩写|缩写]]為{{lang|en|UKIP}}）是[[英國|英國]]的一個[[右翼民粹主义|右翼民粹主义]]政黨。該黨主要的政治理念是推動[[英國脫離歐盟|英國脫離歐盟]]。

== 歷史 ==
該黨成立之初便以使[[英國脫離歐盟|英國脫離歐盟]]為目標，支持透過舉行公投退出歐盟。

不少英國獨立黨支持者是來自原[[保守黨_(英國)|保守黨]]的疑歐派強硬支持者，要求退出歐盟，並認為保守黨對歐盟的立場並不堅定。

[[2009年歐洲議會選舉|2009年歐洲議會選舉]]，獨立黨一舉取得11個議席，與三大主流政黨[[保守黨_(英國)|保守黨]]、[[工黨_(英國)|工黨]]和[[自由民主黨_(英國)|自由民主黨]]並駕齊驅。其後的[[2010年英國大選|2010年英國大選]]，獨立黨獲得九十多萬票，儘管未能取得任何議席。其後在多場地方議會的選舉，獨立黨勝出多個地方議席，使獨立黨逐漸引起英國社會的關注。

[[2013年英国地方选举|2013年英國地方议会選举]]，獨立黨取得大量地方政府议席，正式成為地方議會中的第四大黨。

[[2014年歐洲議會選舉_(英國)|2014年歐洲議會選舉]]，獨立黨成為英國73個席次中第一大黨，總得票數比工黨和保守黨為多。不少是原保守黨的支持者。

2014年9月，兩名保守黨下議院議員辭職並退出保守黨，以英國獨立黨名義參加補選，這已是保守黨兩個月內第二度有國會議員投靠反歐盟的英國獨立黨，另一名議員卡斯威爾（Douglas Carswell）上月才以保守黨無意認真推動歐盟改革為由率先倒戈。<ref>[http://news.ltn.com.tw/news/world/paper/817224 同黨議員變節、閣員傳不雅照 英相難堪] 2014-09-29 自由時報</ref>

[[2015年英国大选|2015年英国大选]]中，英国独立党获得总票数仅次于[[保守党_(英国)|保守党]]和[[工党_(英国)|工党]]，位居第三位，但仅获得[[英国下议院|下议院]]的一个席位<ref>{{cite web |url = http://www.bbc.com/news/election/2015/results |title=Election 2015 - Results |work=BBC |date=2015-05-08}}</ref>。党魁[[奈杰尔·法拉奇|奈杰尔·法拉奇]]也在自己的选区中败选<ref>{{cite web |url = http://www.cnn.com/videos/world/2015/05/08/uk-election-nigel-farage-loses-bid-for-thanet-south.cnn |title=UKIP leader Nigel Farage loses seat |work=CNN |date=2015-05-08}}</ref>。他在大选后提出要辞去党魁一职，但未被党内接受<ref>{{cite web |url=http://news.sky.com/story/1481942/ukip-rejects-nigel-farages-resignation |title=UKIP Rejects Nigel Farage's Resignation|work=Sky News|date=2015-05-11 |accessdate=2015-05-11}}</ref>。論得票率，獨立黨的得票從3.1%增長至12.6%，是英國第三大黨。

[[2016年英國脫離歐盟公投|2016年英國脫離歐盟公投]]後，法拉奇在2016年7月4日宣佈已達成個人從政目的，辭去英國獨立黨黨魁一職<ref>{{Cite news |url = http://www.theguardian.com/politics/2016/jul/04/nigel-farage-resigns-as-ukip-leader |title=Nigel Farage resigns as Ukip leader after 'achieving political ambition' of Brexit |first1=Rowena |last1=Mason |first2=Peter |last2=Walker |publisher=[[衛報|衛報]]|date=2016-07-04 |accessdate=2016-07-04 |language=en-gb}}</ref>。

2017年5月英国地方选举中独立党仅获得[[兰开夏郡|兰开夏郡]]的一个席位——丧失了145个席位<ref>{{cite web | url = http://www.bbc.co.uk/news/election-2017-39795422 | title = Local elections 2017: The results mapped |work=BBC | date= 2017-05-05}}</ref>。
