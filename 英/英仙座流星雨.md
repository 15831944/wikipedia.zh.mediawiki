'''英仙座流星雨'''（Perseids）是以[[英仙座|英仙座]]γ星附近为[[輻射點|輻射點]]出现的[[流星雨|流星雨]]，也称'''英仙座γ流星雨'''。每年在7月20日至8月20日前后出现，于8月13日达到高潮。与[[象限仪座流星雨|象限仪座流星雨]]、[[双子座流星雨|双子座流星雨]]并称为'''年度三大流星雨'''。

== 母天体 ==
[[母天体|母天体]]是周期为120[[年|年]]的[[斯威夫特·塔特爾彗星|斯威夫特·塔特爾彗星]]。<ref>[http://books.google.com.tw/books?id=kKNY1zxp0M8C&pg=PT40&dq=%E8%8B%B1%E4%BB%99%E5%BA%A7%E6%B5%81%E6%98%9F%E9%9B%A8&hl=zh-TW&sa=X&ei=gZuFVLjUDsK1mwX2n4DwAg&ved=0CB0Q6AEwAA#v=onepage&q=%E8%8B%B1%E4%BB%99%E5%BA%A7%E6%B5%81%E6%98%9F%E9%9B%A8&f=false 最好看的地理百科]</ref>此[[彗星|彗星]]于1862年被[[路易斯·斯威夫特|路易斯·斯威夫特]]和塔特爾发现，之后[[意大利|意大利]][[天文学家|天文学家]][[乔万尼·施亚帕雷利|乔万尼·施亚帕雷利]]（Giovanni Schiaparelli）提出它可能是英仙座流星雨的母天体。这也是彗星首次被认定为流星雨的母天体。当时预测1982年左右斯威夫特·塔特爾彗星被周期回归，但却没有被观测到。1991年和1992年英仙座流星雨以往年2倍以上的规模爆发，因此推测母彗星也将回归，当年9月27日[[日本|日本]]彗星专家[[木内鶴彦|木内鶴彦]]再次发现了斯威夫特·塔特爾彗星。

== 活动 ==
在高潮时平均一小时可以出现30{{连接号}}60个[[流星|流星]]，是一年中最大的活跃期。高潮的前后数日，每小时也可以出现10个以上。由于正值暑假，夜间气温也比较高，被称为最为容易观测的流星雨之一。

英仙座流星雨的流星，速度快，而且可能在流逝中途越發明亮。而且明亮的流星和[[火球|火球]]也較多，容易形成[[流星痕|流星痕]]。这是因为其對地[[速度|速度]]為59 [[km|km]]/[[s|s]]，在流星雨中属于速度比较快的。因为相對明亮，所以在少雲的时候也可以被觀测到。

== 历史 ==
英仙座流星雨已被觀測了近2000年，最早的观测记录出現在公元36年[[中国|中国]]的史籍，當時記錄每小时观测到100个以上的流星，8至11世纪在[[日本|日本]]与[[韩国|韩国]]也分别有观测记录，但12至19世纪则只有零星记录。在早期中世紀的歐洲，英仙座流星雨被稱為聖勞倫斯的淚水，因為其出現時間與3世紀基督聖徒[[羅馬的聖老楞佐|羅馬的聖老楞佐]]（Lawrence of Rome）殉道的日子8月10日相近。直至1835年英仙座流星雨才被[[比利時|比利時]]天文學家[[朗伯·阿道夫·雅克·凱特勒|阿道夫·凱特勒]]確認为每年度出現的流星雨<ref>{{cite web|url=http://www.nasa.gov/pdf/677215main_Perseids2012.pdf |title=NASA Chat: Stay ‘Up All Night’ to Watch the Perseids! |publisher=NASA |author= Dr. Bill Cooke, Danielle Moser and Rhiannon Blaauw| pages=55|date=2012-08-11|accessdate=2013-08-16}}</ref><ref name="Kronk"/>，仅次于[[狮子座流星雨|狮子座流星雨]]为第二个被命名的流星雨。截止19世纪至少有20次观测记录，特别是斯威夫特·塔特爾彗星被发现的1862年，在[[中国|中国]]和[[日本|日本]]均被观测到。在中国有记录显示，[[满月|满月]]时曾连续四个夜晚出现，高潮时曾出现每小时4800个的“流星暴”。之后的1921年也有记录显示增多，最近几十年出现比较稳定。母彗星回归前后的1991年到1994年间可以观测到往年2倍以上的流星。

== 觀測 ==
這個流星雨在每年的7月中旬出現，最活躍的時間在8月8日至14日之間，高峰出現在8月12/13日。在高峰期，每小時的流星數目高達60顆以上，橫越整個天空，但受限於[[斯威夫特·塔特爾彗星|斯威夫特·塔特爾彗星]]的路徑，這個流星雨只能在[[北半球|北半球]]看見。

=== 1972年 ===
在記錄上是歷年來最活躍的一次，尤其是在1972年的8月12日從超過12000尺高的科羅拉多州所做的觀測。

=== 2007年 ===
流星雨的尖峰初現在8月12-13日，周日至周一的夜晚，適逢新月，因此觀測到的[[天頂每時出現率|ZHR<sub>max</sub>]]為93顆<ref>{{cite web |url=http://www.imo.net/live/perseids2007/ |title=存档副本 |accessdate=2011-09-27 |0=2011-09-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20110927041829/http://www.imo.net/live/perseids2007/ |archivedate=2011-09-27 }}</ref>。

=== 2008年 ===
英仙座流星雨出現於7月17日至8月24日，在8月13日觀測到的[[天頂每時出現率|ZHR<sub>max</sub>]]為116顆（在20:10 UT）<ref>{{cite web |url=http://www.imo.net/live/perseids2008/ |title=存档副本 |accessdate=2017-08-11 |0=2016-10-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20161024035510/http://www.imo.net/live/perseids2008/ |archivedate=2016-10-24 }}</ref>

=== 2009年 ===
英仙座流星雨出現於7月14日至8月24日，在8月13日觀測到的[[天頂每時出現率|ZHR<sub>max</sub>]]為173顆<ref>{{cite web |url=http://www.imo.net/live/perseids2009/ |title=存档副本 |accessdate=2017-08-11 |0=2016-10-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20161016083404/http://imo.net/live/perseids2009/ |archivedate=2016-10-16 }}</ref>
