'''英華小學'''（'''Ying Wa Primary School'''）是[[香港|香港]]一所男子小學，位於[[香港|香港]][[九龍|九龍]][[深水埗區|深水埗區]][[英華街|英華街]]三號。

== 歷史 ==
舊有的英華書院小學部於1931年創校。於1963年，[[英華書院|英華書院]]校舍由[[旺角|旺角]][[弼街|弼街]]搬到[[牛津道|牛津道]]，而牛津道的校舍未能容納小學部，小學部因此而停辦，舊有的小五及小六學生需到當時新開設的[[中華基督教會基華小學|中華基督教會基華小學]]繼續上課，部分英華書院小學部的教師亦過渡到基華小學。<ref>{{cite book|title=古樹英華|page=93–94|publisher=英華書院校友會有限公司|author=劉紹麟|isbn=962-86125-1-4}}</ref><ref>{{cite web|url=https://www.keiwa.edu.hk/52main.htm|title=學校歷史|publisher=中華基督教會基華小學|accessdate=2018-01-14}}</ref>

在2003年，[[中華基督教會基全小學|中華基督教會基全小學上午校]]以基全小學的名義轉為全日制學校，而下午校則合併到新註冊成立的「英華小學」，並遷入[[深水埗|深水埗]][[英華街|英華街]]的校舍，稱為小學「復校」<ref name="school report">{{cite web|url=http://www.yingwaps.edu.hk/CustomPage/8/school-report11-12.pdf|title=英華小學校務報告 2011–12年度|publisher=英華小學}}</ref>。新的英華小學為獨立於書院的學校，首任小學校長由基全小學下午校校長簡燕玲擔任，舊有的下午校師生及校長亦轉到新的英華小學繼續上課。<ref name="school report"/><ref>{{cite book|title=皕載英華|isbn=978-962-04-4416-6|pages=485–494|author=《皕載英華》出版委員會|publisher=三聯書店（香港）有限公司|date=2018年11月}}</ref><ref>{{cite news|url=http://orientaldaily.on.cc/archive/20030210/new/new_a01cnt.html|title=英華結龍涉歧視新移民|date=2003-02-10|accessdate=2018-01-14|publisher=香港東方日報}}</ref>

2007年起由津貼小學轉為直資，與英華書院結龍，畢業學生能以直升方法升讀相關中學——[[英華書院|英華書院]]。

在2017香港最具教育競爭力小學50強龍虎榜中排名第19位<ref>[http://www.hkpep.com/index.php?main_page=page_2&curkey=xiaoxue#.WulN83--nIU   2017香港最具教育競爭力小學50強龍虎榜-香港專業教育出版社]</ref>。

== 辦學願景 ==
英華小學與[[英華書院|英華書院]]具同一願景：「以基督精神，辦全人教育」。兩校的共同使命是：使學生能掌握兩文三語、認識資訊科技、具思考能力及創造力；培養學生多智能、多才藝；教導學生關懷別人、樂於服務；使學生自律自信、愛國愛人。本校致力提供一個自由、民主、公義、團結的優良學習環境，提升學生達致「樂善勇敢」的精神，活出校訓「篤信善行」的生命。<ref>[http://www.yingwaps.edu.hk/CustomPage/paragraphGroup.aspx?ct=customPage&webPageId=2&pageId=2&nnnid=4 英華小學辦學願景及使命]</ref>

== 教師資料 ==
英華小學共有教職員95名，包括4位外籍老師。學校另有1位駐校社工，1位創意及出版總監，1位音樂總監，1位綠化項目經理。英華小學100%老師擁有學士學位，其中逾32.3%老師持有碩士學位，3.2%老師更持有博士學位。全部老師均已接受教師專業訓練。
