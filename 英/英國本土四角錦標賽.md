'''英國本土四角錦標賽'''（'''British Home Championship'''）是[[英國|聯合王國]]內四支代表，[[英格蘭足球代表隊|英格蘭]]、[[蘇格蘭足球代表隊|蘇格蘭]]、[[北愛爾蘭足球代表隊|北愛爾蘭]]（1921-22年前為[[愛爾蘭|愛爾蘭]]）及[[威爾斯足球代表隊|威爾斯]]，之間每年一度的[[足球|足球]]錦標競賽，由1883-84年至1983-84年期間舉行。2007年部分成員探討再度舉行本土錦標賽的可能，但英格蘭態度冷淡<ref>[http://news.bbc.co.uk/sport2/hi/football/internationals/6329523.stm 本土成員國計劃新聯賽] </ref>。蘇格蘭、威爾斯、北愛爾蘭及愛爾蘭在2009年開始舉辦一頂名為「'''[[些路迪盃|-{zh-hans:凯尔特人杯; zh-hant:些路迪盃;}-]]'''」（Celtic Cup），每兩年一屆的錦標賽 <ref>[http://news.bbc.co.uk/sport1/hi/football/internationals/7114148.stm 'Celtic Cup' set to start in 2009]</ref>。

==比賽概要==
早於1880年代足球發展初期，英國本土的代表隊已經常進行友誼性比賽，但礙於各地的足例稍有不同，比賽時會按主場的地區球例進行，此安排按理可行但執行不易。為解決問題，本土四個足球總會在1882年12月6日出席在[[曼徹斯特|曼徹斯特]]舉行的會議，成立[[國際足球協會理事會|國際足球協會理事會]]並訂定一套統一的國際通行的足球條例。同時決定舉辦歷史上首項國際足球錦標競賽—英國本土四角錦標賽。

四角賽在1883-84年首度舉行，由[[蘇格蘭足球代表隊|蘇格蘭]]奪得冠軍。當時比賽沒有特定時間舉行，但一般會選擇球季後舉行。最初四角賽的冠軍會被稱為世界最佳球隊，隨著世界各地足球逢勃發展而被摘除。而更具規模的[[世界盃|世界盃]]及[[欧洲足球锦标赛|-{zh-hans:欧洲足球锦标赛; zh-hant:歐洲國家盃;}-]]相繼出現而使四角賽的重要性隨年月而消退。但新的國際賽亦使四角賽在特定年份有重要意義，1949-50年及1953-54年的四角賽亦是[[1950年世界杯足球赛|1950年]]及[[1954年世界杯足球赛|1954年]]世界杯的-{zh-hans:資格賽; zh-hant:外圍賽;}-，而1966-67年及1967-68年的四角賽則決定那支球隊可晉身[[1968欧洲足球锦标赛|1968年]][[欧洲足球锦标赛|-{zh-hans:欧洲足球锦标赛; zh-hant:歐洲國家盃;}-]]第二輪的-{zh-hans:資格賽; zh-hant:外圍賽;}-。

由於[[世界盃|世界盃]]及[[欧洲足球锦标赛|-{zh-hans:欧洲足球锦标赛; zh-hant:歐洲國家盃;}-]]等大型國際賽導致四角賽除英格蘭對蘇格蘭外的觀眾急跌、賽期擠迫、「足球流氓」流行、[[北愛爾蘭|北愛爾蘭]]局勢動盪不安（導致1981-82年比賽得取消）及[[英格蘭|英格蘭]]與[[蘇格蘭|蘇格蘭]]希望與更強的對手比賽等因素，四角賽在1983-84年後宣告停辦。但被視為“較弱”的兩支球隊，北愛爾蘭及威爾斯，卻包辦最後一屆四角賽的冠亞軍，而四角賽的獎杯仍然保留在[[貝爾法斯特|貝爾法斯特]]北愛爾蘭足球總會的總部內。

==比賽形式==
比賽以單循環形式進行，每隊之間互相對賽一次，即每隊三場比賽，每年共有六場比賽。球隊之間每年輪流作主場或作客比賽，如英格蘭今年主場對蘇格蘭，明年則作客，如此類推。

勝方球隊可獲2分，平手各得1分，負方無分，績分構成績分榜，每年比賽完畢得分最高的球隊為冠軍，如兩支或以上的球隊同分則並列冠軍。1978-79年以後引入[[得失球差|得失球差]]在同分時高下，如[[得失球差|得失球差]]相同，以入球較高一隊獲勝。
