{{vgname2|英雄联盟日本联赛|en=League of Legends Japan League|简称'''LJL'''}}是日本《[[英雄联盟|英雄联盟]]》的顶级联赛，该比赛于2014年成立，<ref>{{Cite web |url = http://news.candou.com/684469.shtml |title = LOL外卡赛区日本LJL战队简介 LJL来历 |date = 2016年6月24日 |publisher = 蚕豆网 |language = zh-cn }}</ref>举办地则为日本[[东京|东京]]。<ref>{{Cite web |url = http://lol.17173.com/news/01302016/122408492.shtml |title = 看看备受世界冷落的联赛 日本LJL比赛分析 |author = Shelly |date = 2016年1月30日 |publisher = 17173 |language = zh-cn }}</ref>每年都有春、夏两季的常规性季赛，每个赛季都有6支队伍参赛。该比赛的常规赛采用双循环BO3赛制，并持续10周，而常规赛前三的队伍将进入季后赛，该比赛的季后赛则采用阶梯制，其中常规赛排名第2、3的队伍将会进入半决赛，而排名第1的队伍和在半决赛中取得胜利的队伍则将会进入决赛。<ref>{{Cite web |url = http://www.343u.com/article/28460.html |title = 2017年LJL赛区资料介绍 日本ljl联赛水平怎么样 |date = 2017年3月9日 |publisher = 343游戏网 |language = zh-cn }}</ref>
