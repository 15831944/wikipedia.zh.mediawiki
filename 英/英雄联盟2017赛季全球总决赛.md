{{vgname2|英雄聯盟2017賽季全球总决赛|en=League of Legends 2017 World Championship|也被称为英雄聯盟S7全球总决赛}}是由[[Riot_Games|Riot Games]]公司舉行的第7屆[[英雄聯盟全球总决赛|英雄聯盟全球总决赛]]。<ref>{{cite|url=http://lol.esportsmatrix.com/zh-TW/News/Detail?id=5179|title=S7各戰隊的隊員名單}}</ref>该比赛于2017年9月23日-11月4日在[[中国大陆|中国大陆]]举行，其中入围赛和小组赛分别于9月23日-28日和10月5日-15日在[[武汉|武汉]]的[[武漢體育中心|武漢體育中心]]举行，八強赛于10月19日-22日在[[广州|广州]]的[[广州体育馆|广州体育馆]]举行，四強赛事于10月28日-29日在[[上海|上海]]的[[上海东方体育中心|上海东方体育中心]]举行，而决赛则于11月4日在[[北京|北京]]的[[国家体育场_(北京)|国家体育场]]举行。<ref>{{Cite web |url = http://sports.le.com/news/993224019.html |title = 2017《英雄联盟》全球总决赛落地中国 |date = 2017年2月8日 |publisher = 乐视体育 |language = zh-cn }}</ref>该届主题曲为《{{link-en|傳奇永不消逝|Legends Never Die (League of Legends song)}}》（Legends Never Die）。<ref>{{Cite web |url = http://www.yxdown.com/news/201709/368486.html |title = LOLS7总决赛主题曲《传奇永不熄》发布 你被燃到了吗 |author = 提莫队长 |date = 2017年9月15日 |publisher = 游迅网 |language = zh-cn }}</ref>
