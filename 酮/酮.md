'''酮'''是一类[[有机化合物|有机化合物]]，通式RC(=O)R'，其中R和R'可以是相同或不同的[[原子|原子]]或[[官能团|官能团]]。酮的结构特征是具有一个与两个碳原子相连接的[[羰基|羰基]]（C=O）。<ref>IUPAC Gold Book [http://goldbook.iupac.org/K03386.html ''ketones'']</ref>
最简单的酮是[[丙酮|丙酮]]。

酮是唯一一类羰基碳直接与两个碳原子相连的有机物。同为[[羰基化合物|羰基化合物]]的[[醛|醛]]、[[羧酸|羧酸]]、[[酯|酯]]、[[酰卤|酰卤]]和[[酰胺|酰胺]]中，羰基碳原子的一侧则分别与氢、羟基（氧）、氧、卤素与氮原子相连，不同於酮。

有α-氢的酮可发生[[酮-烯醇互变异构|酮-烯醇互变异构]]。酮与[[强碱|强碱]]反应产生相应的[[烯醇负离子|烯醇负离子]]，反应一般通过烯醇[[去质子化|去质子化]]进行。

“酮”(拼音：tóng，注音：ㄊㄨㄥˊ)在古汉语中有两义：“用[[馬乳|馬奶]]製成的酪”与“醋”。近代被用作欧洲语言中此类化合物名的音译。对比：英语''ketone''（“酮”，< ''acetone''，“丙酮”），以及德语中语义相同的''Keton''、''Aceton''。

== 命名 ==
简单的酮常用[[IUPAC命名法#醛|普通命名法]]。

*[[芳香酮|芳香酮]]中芳基作为取代基来命名。
*多元酮命名时，应选取含羰基尽可能多的碳链作主链，并标明羰基的位置和羰基的数目。
*不饱和酮的命名除羰基的编号应尽可能小以外，还要表示出[[不饱和键|不饱和键]]所在的位置。
*[[脂环酮|脂环酮]]命名时，编号从羰基碳原子开始，仅在名称前多加一个“环”字。

很多天然酮都有俗名，例如，天然[[麝香|麝香]]的主要成分[[麝香酮|麝香酮]]（muscone）为十五元环酮等。

== 结构和性质 ==
[[File:KetoneVarietyPack.png|thumb]]，一种常规溶剂；[[草酰乙酸|草酰乙酸]]，[[糖代谢|糖代谢]]中间体；[[乙酰丙酮|乙酰丙酮]]的单烯醇式（烯醇部分蓝色标注）；[[环己酮|环己酮]]，尼龙的前体；[[麝香酮|麝香酮]]，一种动物香料；[[四环素|四环素]]，一种抗生素。]]

酮中的羰基碳原子为sp<sup>2</sup>[[杂化|杂化]]，平面三角构型，其C-C-O和C-C-C键角约为120°。

羰基中氧原子的[[电负性|电负性]]高于碳原子的电负性，故羰基有[[极性|极性]]，氧原子具[[亲核性|亲核性]]，碳原子具[[亲电性|亲电性]]。由于羰基能够与水形成[[氢键|氢键]]，酮分子比羰基置换成亚甲基的分子来说更易溶于水。酮一般只作氢键受体而不作为氢键供体，无法形成自身分子间的氢键，因而比具有类似[[分子量|分子量]]的[[醇|醇]]和[[羧酸|羧酸]]更具挥发性。酮类是常用的香水和溶剂组分。

=== 酮的分类 ===
酮可根据其所含[[取代基|取代基]]分类，也可根据羰基两侧取代基是否相同而分类为对称和不对称的酮。丙酮和[[苯酮|苯酮]]属于对称酮；[[苯乙酮|苯乙酮]]（C<sub>6</sub>H<sub>5</sub>C(O)CH<sub>3</sub>）则属于不对称酮。立体化学中，不对称酮被视为[[前手性|前手性]]分子。

按照烃基的不同，酮还可分为[[脂肪酮|脂肪酮]]，[[脂环酮|脂环酮]]和[[芳香酮|芳香酮]]。[[芳香酮|芳香酮]]的羰基直接连在芳香环上。

按照羰基的数目，酮可以分为一元酮和多元酮。 

最简单的[[二酮|二酮]]是[[丁二酮|丁二酮]]（CH<sub>3</sub>C(O)C(O)CH<sub>3</sub>），它曾被用作爆米花中的奶油增味剂。分子对称的[[乙酰丙酮|乙酰丙酮]]（2,4-戊二酮）虽称为二酮，但它主要以单酮烯醇式存在，此烯醇式是[[配位化学|配位化学]]中的常见配体。 

不饱和酮多指含有[[烯烃|烯烃]]和[[炔烃|炔烃]]结构的酮类。例子有[[甲基乙烯基酮|甲基乙烯基酮]]（CH<sub>3</sub>C(O)CH=CH<sub>2</sub>），[[Robinson成环反应|Robinson环化]]中的重要中间体。酮官能团自身也是[[不饱和|不饱和]]的，多数情况下可通过[[氢化|氢化]]还原。

无杂原子取代的饱和单环环酮母核的分子式为(CH<sub>2</sub>)<sub>n</sub>CO，其中n=3（[[环丙酮|环丙酮]]）-10。环己酮是生产[[尼龙|尼龙]]的前体。[[异佛尔酮|异佛尔酮]]为不饱和的非对称酮，用于聚合物的制备。[[麝香酮|麝香酮]]是一种动物信息素。

=== 酮－烯醇异构 ===
{{main|烯醇}}
[[File:Keto_enol_tautomerism.svg|thumb]]

至少有一个[[α-氢|α-氢]]的酮可发生[[酮-烯醇互变异构|酮－烯醇异构化]]，产生对应的[[烯醇|烯醇]]。异构化可以被[[酸|酸]]或者[[碱|碱]]催化进行。通常酮式比烯醇式更稳定，因此可以利用平衡通过[[炔烃|炔烃]]的水解反应来制备酮。

=== 酮的酸性 ===
酮的α-氢的酸性（[[酸度係數|pKa]] ≈ 20）比一般的烷烃氢的酸性（pKa ≈ 50）强的多。这是因为酮被去质子化后形成的[[烯醇离子|烯醇离子]]具有一定的稳定性。等当量碱夺取α-氢后产生的烯醇离子，为[[两位负离子|两位负离子]]，可在亲核性较强的碳原子一端也可在碱性较强的氧原子一端参与[[亲核取代反应|亲核取代反应]]，在合成上十分有用。

== 表徵 ==
=== 光谱 ===
在[[红外光谱|红外光谱]]，酮和醛於1700[[波数|cm<sup>−1</sup>]]附近有强烈吸收。具体吸收位置取决于分子中的其他官能团。

酮在[[核磁共振氢谱|核磁共振氢谱]]中难以确认，在[[核磁共振碳谱|核磁共振碳谱]]中羰基碳则位於低场约200ppm的位置。因无[[核极化效应|核极化效应]]，故信号峰一般较弱。醛和酮在核磁共振中具有相似的化学位移，需通过多重共振实验进行辨别。

=== 有机定性实验 ===
酮的[[Brady试验|Brady试验]]结果呈阳性，可与[[2,4-二硝基苯肼|2,4-二硝基苯肼]]缩合为相应的[[腙|腙]]。[[Tollens试剂|Tollens试验]]中酮呈阴性，可借此性质而将酮与醛区分开。甲基酮可通过[[卤仿反应|卤仿反应]]鉴别。

== 合成 ==
酮可通过多种方法合成。工业中，酮多通过[[烃|烃]]类被氧化制备，氧化剂一般是空气（中的氧气）。例如每年产量十亿公斤的[[环己酮|环己酮]]即是通过[[环己烷|环己烷]]被空气氧化而制备的。而丙酮则是通过[[异丙苯法|异丙苯氧化法]]来制备的。

实验室中，酮一般通过[[醇的氧化|二级醇的氧化]]制备。 
:R<sub>2</sub>CH(OH)  +  O   →   R<sub>2</sub>C=O  +  H<sub>2</sub>O

[[高锰酸钾|高锰酸钾]]与[[六价铬|六价铬]]试剂（如[[重铬酸钾|重铬酸钾]]）是典型的氧化剂。此外还有比较温和的条件如：使用[[Dess-Martin氧化|Dess-Martin氧化]]或[[Swern氧化|Moffatt-Swern氧化]]。

许多发展出来的其他常用合成方法包括：
* 通过[[偕卤素水解|偕卤素水解]]制备。
* 通过[[炔烃|炔烃]]的[[水解|水解]]制备。反应经[[烯醇|烯醇]]中间体，需[[硫酸汞|硫酸汞]]催化。得到的烯醇异构化为酮。这个反应通常是利用末端的炔烃来合成酮。 
* 通过计量的有机金属试剂与[[Weinreb酮合成|Weinreb胺]]反应。
* 芳基酮可以通过[[F-C反应|Friedel-Crafts酰化]]及相关的[[Houben-Hoesch反应|Houben-Hoesch反应]]或[[Fries重排反应|Fries重排]]制备。
* 烯烃[[臭氧化|臭氧化]]或二羟化-氧化（如[[Sharpless不对称双羟基化反应|Sharpless不对称双羟基化反应]]），可使烯烃断裂为醛或酮。产物取决于烯烃的双键取代类型。
* [[Kornblum–DeLaMare重排反应|Kornblum–DeLaMare重排]]中，酮通过过氧化物与碱反应制备。
* [[Ruzicka环化反应|Ruzicka环化]]中，环酮通过二羧酸制备。
* [[Nef反应|Nef反应]]中，酮通过二级硝基化合物的盐的水解制备。
* [[Fukuyama偶联反应|Fukuyama偶联]]中，酮通过硫酯和有机锌化合物制备。
* 通过[[酰氯|酰氯]]与[[有机镉化合物|有机镉化合物]]或[[有机铜化合物|有机铜化合物]]反应制备酮。
* [[Dakin-West反应|Dakin-West反应]]提供了一种有效的从羧酸来制备酮的方法。
* 酮亦可通过[[格氏试剂|格氏试剂]]与[[腈|腈]]反应然后水解来制备。
* 通过[[酸酐|酸酐]]脱羧来制备。
* 酮可通过[[卤代酮的还原脱卤|卤代酮的还原脱卤]]反应制备。

== 反应 ==
酮能够参与许多[[有机反应|有机反应]]。其中最主要的一类就是羰基的[[亲核加成反应|亲核加成反应]]以及烯醇对于亲电底物的加成反应。亲核加成反应包括：
* 与水反应（水解）得到[[偕二醇|偕二醇]]。水解平衡在大多数情况下倒向酮一边，不利于偕二醇的生成。
* 与[[乙炔化物|乙炔化物]]反应得到α-[[羟基炔|羟基炔]]。
* 与[[氨|氨]]或[[伯胺|伯胺]]反应得到[[亚胺|亚胺]]。
* 与[[二级胺|二级胺]]反应得到[[烯胺|烯胺]]。
* 与[[格氏试剂|格氏试剂]]或[[有机锂试剂|有机锂试剂]]反应，然后用水处理后得到[[三级醇|三级醇]]。
* 与[[醇|醇]]或[[醇盐|醇盐]]反应得到[[半缩酮|半缩酮]]或其共轭碱。此反应常用来保护酮羰基。
* 与[[氨基钠|氨基钠]]反应，C-C键断裂，得到酰胺RCONH<sub>2</sub>以及烷烃R'H，即[[Haller-Bauer反应|Haller-Bauer反应]]。<ref>[http://nagoyaren.homeip.net/chem/reactions/123.htm Haller-Bauer Reaction] {{webarchive|url=https://web.archive.org/web/20070910060716/http://nagoyaren.homeip.net/chem/reactions/123.htm |date=2007-09-10 }}</ref>
* [[亲电加成|亲电加成]]，与亲电试剂反应得到共振稳定的正离子。
* 与[[磷叶立德|磷叶立德]]反应得到[[烯烃|烯烃]]，即著名的[[Wittig反应|Wittig反应]]。
* 与[[硫醇|硫醇]]反应得到[[硫缩酮|硫缩酮]]。
* 与[[肼|肼]]或者1-位取代的肼[[衍生物|衍生物]]反应得到[[腙|腙]]。
* 与金属[[氢化物|氢化物]]反应得到金属烷氧盐，水解后得到[[醇|醇]]。即[[酮的还原|酮的还原]]。
* 与[[卤素|卤素]]反应得到α-[[卤代酮|卤代酮]]，反应经[[烯醇式|烯醇式]]中间体。参见[[卤仿反应|卤仿反应]]。
* 与[[重水|重水]]反应得到α-[[氘代|氘代]]酮。
* 经[[Norrish反应|Norrish反应]]光解碎裂化。
* 1,4-氨基二酮经[[Robinson-Gabriel合成|Robinson-Gabriel合成]]，脱水得到[[噁唑|噁唑]]。 
* [[Willgerodt重排反应|Willgerodt反应]]中，芳基烷基酮、硫与胺反应，得到酰胺。
* 与[[羟胺|羟胺]]反应制备[[肟|肟]]。

== 应用 ==
酮在工业中大量用于溶剂、聚合物及医药的生产，其中应用最多的是[[丙酮|丙酮]]、[[丁酮|丁酮]]及[[环己酮|环己酮]]。生物化学中酮类亦是常见的物质。烃类在氧气不足时燃烧，可得酮类及其他各类化合物。

== 毒性 ==
低级酮毒性大多较低，有些是常用的有机溶剂。某些不饱和酮的毒性较高，如[[甲基乙烯基酮|甲基乙烯基酮]]（{{LD50}}7 mg/kg（口服））。
