《'''造夢同人誌'''》是{{lang|ja|[[ヒロユキ|ヒロユキ]]}}所畫並連載於[[芳文社|芳文社]]雜誌《[[Manga_Time_Kirara_Carat|Manga Time Kirara Carat]]》2004年12月號至2008年10月號的[[四格漫畫|四格漫畫]]作品，[[單行本|單行本]]全6冊。[[電視動畫|電視動畫]]於2007年7月至9月播放。

== 登場人物 ==
; {{nihongo|長菜夏美|長菜なじみ（おさな なじみ）||CV：[[浅野真澄|浅野真澄]]}}
: 因受到露理影響而開始畫同人誌的少女。全名的日文發音剛好和「青梅竹馬(幼馴染)」完全一樣。
; {{nihongo|露理|露理（つゆり）||CV：[[斋藤桃子|斋藤桃子]]}}
: 菜美的同学，经常戴着[[贝雷帽|贝雷帽]]。
; {{nihongo|賈斯汀|ジャスティス||CV：[[安元洋貴|安元洋貴]]}}
: 和菜美从小就相识。虽然和她同一个学校，不过班级不同。
; {{nihongo|北野青空|北野ソーラ（きたの ソーラ）||CV：[[古山貴實子|古山貴實子]]}}
: 经常坐在賈斯汀手臂上的娇小的女孩子。
; {{nihongo|星純一郎|星純一郎（ほし じゅんいちろう）||CV：[[波多野和俊|波多野和俊]]}}
: 在菜美第一次参加的[[同人志即卖会|同人志即卖会]]，唯一购买了她的同人志的人。
; {{nihongo|二道|二道かねる（にどう かねる）||CV：[[阪田伊都|阪田伊都]]}}
: 一边在公司上班、一边画同人志的女性。
; {{nihongo|星龍一郎|星龍一郎（ほし りゅういちろう）||CV：[[岸尾大辅|岸尾大辅]]}}
: 星純一郎的哥哥。
