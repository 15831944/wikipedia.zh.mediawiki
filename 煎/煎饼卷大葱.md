== 简介 ==
'''[[煎饼|煎饼]]卷[[大葱|大葱]]'''由煎饼，大葱，[[甜面酱|甜面酱]]制作而成，它一方面代表的是山东的饮食，另一方面又代表了'''山东省地方文化。'''知名度相当高，是中国[[山东省|山东省]]的一个特色名吃。

== 来源 ==
在山东，有大约三分之一的人是拿煎饼做主食的，有[[日照|日照]]、[[临沂|临沂]]、[[济南|济南]]、[[泰安|泰安]]、[[枣庄|枣庄]]、[[济宁|济宁]]、[[曲阜|曲阜]]等，<ref>百度百科，煎饼种类</ref>其中泰安，曲阜的煎饼，最可口；甜面酱在山东是很受欢迎的，其味道又咸又甜，会提升人的食欲；不同于南方的小葱，山东产的大葱香辣可口味道鲜美，之于[[鲁菜|鲁菜]]更是不可缺少的一个调味料，一根约20-30厘米长，大多食用葱白部分。

在很久以前，像是闯关东等走南闯北的需要，煎饼的留存时间久，味道香，山东人就经常带着它当主食，再卷上大葱和甜面酱，制作简单，食之可口，食材又便宜。随着时代的变迁，现在已经不是大家经常采用的吃法了。作为一个特色，在鲁菜饭店里倒是有它的身影，但是现如今，已增添了很多吃法，比如酱料的增加。<ref>百度百科，煎饼是山东特产，为何只有临沂、泰安等部分地区把它作为主食？</ref>

== 传统制作方法 ==
'''准备好：煎饼，大葱，甜面酱。'''

# 洗净大葱去叶留葱白大段。
# 将甜面酱单面均匀涂抹于煎饼上。
# 将葱白放在涂好甜面酱面的煎饼上。
# 无酱面在外，酱葱在里将煎饼卷起。<ref>百度百科，煎饼卷大葱，食用方法</ref>

== 相关传说 ==
相传古时候有位姑娘将大葱削制成毛笔，将大酱做成墨水，将煎饼做成写字用的纸张，然后将酱墨和葱笔卷入煎饼纸内，送给自己的心上人作为充饥的食物，便有了现在的煎饼卷大葱。<ref>百度百科，煎饼卷大葱，相关民间故事</ref>
