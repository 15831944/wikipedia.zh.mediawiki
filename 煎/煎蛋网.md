'''煎蛋'''是一个中文团队[[博客|博客]]，采用[[Wordpress|Wordpress]]程序，于2006年10月12日上线<ref name="more">{{cite web|title=About|url=http://jandan.net/more|publisher=煎蛋|accessdate=2011-02-13}}</ref> 。 其主要内容翻译自其他语言的[[网站|网站]]<ref>{{cite news|author=王晓光|title=煎蛋、网不易和臭豆|url=http://www.bundpic.com/link.php?linkid=768|accessdate=2011-02-13|newspaper=外滩画报|date=2007-08-09}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> <ref>{{cite web|author=oioi|title=投稿|url=http://jandan.net/contribute|publisher=煎蛋|accessdate=2011-02-13|deadurl=yes|archiveurl=https://web.archive.org/web/20110218050153/http://jandan.net/contribute|archivedate=2011年2月18日|df=}}</ref>，目的是为了让中文网友了解其他国家的信息<ref name="more"/><ref>{{cite news|author=鲁朵雅|title=看过煎蛋后，地球上没有新鲜事|url=http://ccwb.yunnan.cn/html/2009-05/02/content_41062.htm|accessdate=2011-02-13|newspaper=春城晚报|date=2009-05-02}}</ref> 。煎蛋的主要栏目包括：小游戏、小学堂、[[FMyLife|发霉啦]]、无聊图、妹子图、走近科学、Geek、设计快读等。

（在2018年11月由于审查原因，部分栏目已被关闭。）

== 版权 ==
===版权协议变更===
煎蛋的文字原本采用[[创作共用|创作共用]]署名-非商业-保持一致协议授权，因其站点文字屡次被其他网站在没有遵守该协议的情况下抄袭，在2008年11月1日煎蛋发文宣布放弃此协议。许多网友纷纷在文章留言讨论煎蛋放弃创作共用协议和申明版权的是否合理。对此煎蛋随后数次更新该文章，表示会遵守创作共用协议，只申明文字版权，其他内容版权归原作者。<ref name="copyright">{{cite web | url=http://jandan.net/2008/11/01/no-cc.html | title=煎蛋放弃 CC 协议 | accessdate=2011-02-13}}</ref>

===2014年网易订阅的纠纷===
2014年6月21日，煎蛋网指责[[网易|网易]]订阅频道抓取了其的内容但是没有标注煎蛋的版权信息及反向链接。<ref name="jdnetease">{{cite web | url= http://jandan.net/2014/06/21/dy-163.html | title= 请网易公司尊重版权，关闭网易订阅频道 [更新网易公告] | publisher = 煎蛋 | date=2014-06-21 | accessdate=2014-07-09}}</ref>24日，网易订阅频道发布公告致歉并安排于PC浏览时添加原文地址。<ref name="neteasejd">{{cite web | url= http://dy.163.com/article/T1374543275708/9VGLOMQL00964K1K.html | title= 致关爱煎蛋的各位朋友：误会消除啦 煎蛋回来啦！| publisher = 网易 | date=2014-06-24 | accessdate=2014-07-09}}</ref>

===2017年视觉中国版权纠纷===
2017年4月20日，煎蛋在自己网站与微信公众号公开文章中称<ref>{{cite web|url=http://jandan.net/2017/04/20/getty.html |title=被视觉中国索赔图片版权，很难受 |publisher=煎蛋 |date=2017-04-20 |accessdate=2017-04-20 |archiveurl=https://archive.is/20170420092455/http://jandan.net/2017/04/20/getty.html |archivedate=2017-04-20 |deadurl=yes }}</ref><ref>{{cite web | url= http://mp.weixin.qq.com/s/5Ap8xbW2f8nMOEkXci-bhQ | title= 被视觉中国索赔图片版权，很难受| publisher = 煎蛋，微信公众号 | date=2017-04-20 | accessdate=2017-04-20}}</ref>，其于4月14日收到[[视觉中国|视觉中国]]的版权确认邮件，由于煎蛋长期翻译、“搬运”外媒文章，其中有涉及来自[[盖帝图像|盖帝图像]]，涉嫌侵权。自称“煎蛋网的唯一员工”sein迅速回复，表示一切文章均按照[[創用CC|創用CC]]模式进行，并希望达成谅解。然而在随后邮件交涉中，双方无法达成统一。

sein认为视觉中国的赔偿要求会让网站陷入停摆，选择将邮件公示，力图争取关注与支持。

4月26日，煎蛋网与视觉中国宣布达成和解，准备长期合作<ref>{{Cite web|url=http://www.ithome.com/html/it/306418.htm |title=煎蛋网与视觉中国发布联合声明：已达成长期合作 |accessdate=2017-04-27 |work=IT之家 |archiveurl=https://archive.is/20170427070011/http://www.ithome.com/html/it/306418.htm |archivedate=2017-04-27 |deadurl=yes }}</ref>。由于保密等原因，公告中没有公布金额等细节，此前发出的与视觉中国的初步交涉过程和往来邮件等也被煎蛋网自行删除<ref>{{Cite web|url=http://jandan.net/2017/04/26/thanks.html |title=关于图片版权事件，一些想说的话 |accessdate=2017-04-27 |last=煎蛋网 |work=煎蛋网 |archiveurl=https://archive.is/20170426172851/http://jandan.net/2017/04/26/thanks.html |archivedate=2017-04-26 |deadurl=yes }}</ref>。

== 栏目 ==
{{fansite|段落}}
; 发霉啦
FML即 Fuck My Life，煎蛋将其中文化翻译为发霉啦系列。<ref>{{cite web|author=煎蛋网|url=http://jandan.net/tag/%E5%8F%91%E9%9C%89%E5%95%A6|title=发霉啦|accessdate=2013-07-01|language=zh-hans}}</ref>

栏目内容以翻译自美国网站[[FMyLife|FMyLife]]用户发表的短故事为主要内容，{{Fact|偶尔翻译VIE DE MERDE（FMyLife的法国姊妹站）内容|time=2011-02-14T13:30:10+00:00}}。大部分都是记录各人在生活中遇到的不可思议或者相当无奈的事情。也提供了[https://web.archive.org/web/20101218013140/http://jandan.net/fml 单独页面]给蛋友们吐槽自己的亲历事件。

; 小学堂
小学堂的内容都是关于科学有趣的小视频的（[[Youtube|多數來自Youtube]]），視頻中的内容由煎蛋网的编者们翻译成中文。

; 小电影
网民可以在这个栏目贴出自己认为好玩的视频。<ref>{{cite web|author=煎蛋网|url=http://jandan.net/v|title=小电影|accessdate=2013-07-01|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20130817192754/http://jandan.net/v|archivedate=2013年8月17日|df=}}</ref>

; 无聊图
网友们在此處分享自己喜欢的图片，内容大多为带有吐槽属性的图片和搞笑图片。<ref>[jandan.net/guanyu]</ref>

; 妹子图
网友在此栏目分享自己认为好看的女性图片。{{Fact|虽然煎蛋网声称对图片尺度有限制，要求符合[[中华人民共和国|中国]]法律法规，但分享过于暴露的黄色图片的现象屡禁不止。|time=2014-07-08T17:28:08+00:00}}<ref>{{cite web|author=煎蛋网|url=http://jandan.net/ooxx|title=妹子图|accessdate=2013-07-01|language=zh-hans}}</ref>

; 走进科学
此栏目主要翻译一些科普性质的文章。<ref>{{cite web|author=煎蛋网|url=http://jandan.net/tag/%E8%B5%B0%E8%BF%9B%E7%A7%91%E5%AD%A6|title=走进科学|accessdate=2013-07-01|language=zh-hans}}</ref>

==吉祥物==
煎蛋网的[[吉祥物|吉祥物]]为一只名为“超载鸡”的黄色小鸡<ref>{{cite news|title=火爆吉祥物“超载鸡”将空降欢乐谷|url=http://wb.sznews.com/html/2017-01/13/content_3708818.htm|accessdate=2017-01-18|agency=深圳晚报|date=2017年1月13日}}</ref>，主要在网站[[标志|标志]]及网站过载时出现，或作为网站图片中的暗藏[[水印|水印]]。
