'''达旺县'''是[[阿鲁纳恰尔邦|阿鲁纳恰尔邦]]下辖的17县之一，位于[[麦克马洪线|麦克马洪线]]印方侧，目前由[[印度|印度]]实际控制。
== 地理 ==
达旺县位于[[阿鲁纳恰尔邦|阿鲁纳恰尔邦]]的西北部，东南面与[[东卡门县|东卡门县]]和[[西卡门县|西卡门县]]相邻，西南与[[不丹|不丹]]的[[塔希冈宗|塔希冈宗]]和[[塔希央奇宗|塔希央奇宗]]相接，北与[[中华人民共和国|中国]][[西藏自治区|西藏自治区]][[山南市|山南市]][[错那县|错那县]]的實際控制地區接壤。

中國宣稱對該地區擁有主權，在行政區劃上將其歸入西藏自治區山南市的错那县。

== 历史 ==

1984年10月6日，从[[西卡门县|西卡门县]]析出，建立达旺县。
== 行政区划 ==
达旺县下辖10个行政单位，分属于3个分区：
* 达旺分区：达旺，Kitpi；
* Lumla分区：Bongkhar，Dudunghar，Lumla ，Zemithang；
* Jang分区：Jang， Mukto,，Thingbu，Lhou。
