'''达赖喇嘛'''（{{bo|z=ཏཱ་ལའི་བླ་མ་|w=Taa-la'i Bla-ma|p=Dalä Lama}}）是[[藏传佛教|藏传佛教]]中[[格鲁派|格鲁派]]（黄教）[[轉世傳承|轉世傳承]]的领袖。除[[第一世达赖喇嘛·根敦朱巴|第一世達賴喇嘛]]之外，歷代達賴喇嘛的名字都是“嘉措”（{{bo|z=རྒྱ་མཚོ་|p=Gyaco|w=rgya mtsho}}），藏文意为“海洋”，譯為蒙古語後，即為“达赖”（{{mong|m=ᠳᠠᠯᠠᠶ|k=далай}}）；“喇嘛”是藏语对上师的称呼，也隱含著「智慧」之意。“达赖喇嘛”通常指“智慧深似海”，即是[[蒙古|蒙古]][[土默特部|土默特部]]首领[[俺答汗|俺答汗]]给予第三世达赖喇嘛[[索南嘉措|索南嘉措]]的尊号。达赖喇嘛与同属西藏的[[班禅额尔德尼|班禅额尔德尼]]、外蒙古的[[哲布尊丹巴呼图克图|哲布尊丹巴呼图克图]]和内蒙古的[[章嘉呼图克图|章嘉呼图克图]]并称为藏传佛教格鲁派的四大活佛。[[根敦朱巴|根敦朱巴]]为格鲁派创始人[[宗喀巴|宗喀巴]]的弟子，被追認為第一世达赖喇嘛。第四世達賴喇嘛[[云丹嘉措|云丹嘉措]]是[[俺答汗|俺答汗]]的孙子，也是唯一一名[[蒙古族|蒙古人]]出任達賴喇嘛。目前的达赖喇嘛是第十四世[[丹增嘉措|丹增嘉措]]。

== 歷世达赖喇嘛 ==
達賴喇嘛轉世傳承始於[[根敦朱巴|根敦朱巴]]，目前已历十四世。現今第十四世達賴喇嘛[[丹增嘉措|丹增嘉措]]曾多次公開聲稱將在他這一世中止轉世，他可能是最後一任達賴喇嘛<ref>{{cite news |title = 為排除政治干擾 達賴喇嘛擬結束轉世 |url = http://news.ltn.com.tw/news/world/breakingnews/1101259 |newspaper = 自由時報 |date = 2014-09-09 |access-date = 2014-10-09 |archive-url = https://web.archive.org/web/20141015165636/http://news.ltn.com.tw/news/world/breakingnews/1101259 |archive-date = 2014-10-15 |dead-url = no }}</ref>。而[[中华人民共和国政府|中华人民共和国政府]]对此回应称，历任达赖喇嘛的产生必须符合历史定制和宗教仪轨，必须经过灵童寻访和坐床等一系列环节，这些环节都须经过中央政府批准，才能获得合法地位，丹增嘉措无决定权<ref>{{cite web |url = http://mil.huanqiu.com/china/2014-12/5261023.html |title = 达赖狂言将终结转世制度 中方：达赖说了不算 |access-date = 2016-08-09 |archive-url = https://web.archive.org/web/20160825233151/http://mil.huanqiu.com/china/2014-12/5261023.html |archive-date = 2016-08-25 |dead-url = no }}</ref>。
