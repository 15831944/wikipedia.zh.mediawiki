'''达姆施塔特行政区'''（Regierungsbezirk Darmstadt）是[[德国|德国]][[黑森州|黑森州]]的3个[[行政区_(德国)|行政区]]之一，位于该州的南部，首府[[达姆施塔特|达姆施塔特]]。黑森州超过一半的人口居住在达姆施塔特行政区内。

== 行政区划 ==

达姆施塔特行政区下设10个县：
# [[贝格施特拉瑟县|贝格施特拉瑟县]]（Bergstraße），首府[[赫蓬海姆|赫蓬海姆]]（Heppenheim）
# [[达姆施塔特-迪堡县|达姆施塔特-迪堡县]]（Darmstadt-Dieburg），首府[[达姆施塔特|达姆施塔特]]（Darmstadt）
# [[格罗斯-格劳县|格罗斯-格劳县]]（Groß-Gerau），首府[[格罗斯-格劳|格罗斯-格劳]]（Groß-Gerau）
# [[霍赫陶努斯县|霍赫陶努斯县]]（Hochtaunuskreis），首府[[巴特霍姆堡|巴特霍姆堡]]（Bad Homburg vor der Höhe）
# [[美因-金齐希县|美因-金齐希县]]（Main-Kinzig-Kreis），首府[[根浩森|根浩森]]（Gelnhausen）
# [[美因-陶努斯县|美因-陶努斯县]]（Main-Taunus-Kreis），首府[[霍夫海姆|霍夫海姆]]（Hofheim am Taunus）
# [[奥登瓦尔德县|奥登瓦尔德县]]（Odenwaldkreis），首府[[爱尔巴赫|爱尔巴赫]]（Erbach）
# [[奥芬巴赫县|奥芬巴赫县]]（Offenbach），首府[[迪岑巴赫|迪岑巴赫]]（Dietzenbach）
# [[莱茵高-陶努斯县|莱茵高-陶努斯县]]（Rheingau-Taunus-Kreis），首府[[巴特施瓦尔巴赫|巴特施瓦尔巴赫]]（Bad Schwalbach）
# [[维特劳县|维特劳县]]（Wetteraukreis），首府[[弗里德贝格|弗里德贝格]]（Friedberg）
