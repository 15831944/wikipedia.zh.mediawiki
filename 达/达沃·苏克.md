'''达沃·苏克'''（'''Davor Šuker'''，{{bd|1968年|1月1日|catIdx=S苏}}），是已退役的[[克罗地亚|克罗地亚]][[足球|足球]]运动员，出生于[[南斯拉夫社会主义联邦共和国|前南斯拉夫]][[奥西耶克|奥西耶克]]，在球场上司职前锋，曾获得过[[1998年世界杯|1998年世界杯]][[金靴奖|金靴奖]]，克羅地亞足球傳奇巨星之一。苏克现在是克罗地亚足协主席。

== 职业生涯 ==
苏克在1984年参加家乡球队[[奥西耶克足球俱乐部|奥西耶克]]，开始了他的球员生涯。1989年，他转会到[[萨格勒布迪纳摩足球俱乐部|萨格勒布迪纳摩]]。在[[南斯拉夫|南斯拉夫]]足球联赛崭露头角之后，他在1990年与[[罗马尼亚国家足球队|罗马尼亚国家队]]的比赛中一举成名。1991年，[[南斯拉夫内战|南斯拉夫内战]]爆发前，苏克代表[[南斯拉夫国家足球队|南斯拉夫国家队]]参加了两场比赛，在其中他射入了首个国际比赛入球。

同年苏克转会到[[西班牙|西班牙]][[塞維利亞足球俱樂部|塞維利亞]]。1992年克罗地亚独立之后，苏克成为了[[克罗地亚国家足球队|克罗地亚国家队]]成员。在获得[[西甲联赛|西甲]]最佳射手的称号之后，苏克在1996年[[欧洲足球锦标赛|欧洲足球锦标赛]]上，为克罗地亚队贡献了三個入球。其中对[[丹麦国家足球队|丹麦队]]一役中，不但获得点球和在禁区内挑射入球，更曾经在半场之外吊射世界著名门将[[彼得·舒梅切尔|舒梅切尔]]把守的大门，而舒梅切尔也当场竖起大拇指称赞其精准脚法。

1996年苏克加盟[[皇家马德里|皇家马德里]]，帮助该队赢得1997年西甲联赛冠军以及1998年的[[欧洲冠军杯|欧洲冠军杯]]。苏克在1998年[[世界杯|世界杯]]以6個入球勇夺金靴奖，協助克罗地亚取得了前所未有的季军成绩。1999年苏克来到[[英超|英超]]的[[阿森纳足球俱乐部|-{zh-hans:阿森纳; zh-hant:阿仙奴;}-]]，又接连转会到[[西汉姆联足球俱乐部|-{zh-hans:西汉姆联; zh-hant:韋斯咸;}-]]以及[[1860慕尼黑|1860慕尼黑]]。

== 成就 ==
苏克直到退役为止，一共在国际比赛上射入46个进球，其中45个是属于克罗地亚队的。他也因此成为克罗地亚国家队历史上至今为止进球最多的球员，而他在国际比赛进球率则达到0.65，在欧洲乃至世界足坛都很少见。苏克的进球左脚居多，也因此在1996年欧锦赛时，媒体称赞他“左脚灵活得可以拉小提琴”。2003年[[欧洲足联|欧洲足联]]50周年庆典之际，他被克罗地亚足协提名为[[欧洲足联五十周年纪念奖|黄金球员]]，即50年内该国最出色的球员。2004年3月，[[国际足联|国际足联]]委托球王[[贝利|贝利]]选定的[[FIFA_100|国际足联125名伟大球员]]名单中，他是唯一一名克罗地亚球星。

== 退役 ==
退役后，苏克在克罗地亚[[萨格勒布市|萨格勒布市]]开办了一家“达沃·苏克足球学校”。
