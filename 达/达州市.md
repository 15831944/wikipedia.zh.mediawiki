'''达州市'''是[[中华人民共和国|中华人民共和国]][[四川省|四川省]]下辖的[[地级市|地级市]]，位于四川省东北部。市境东接[[重庆市|重庆市]]，南连[[广安市|广安市]]，西邻[[南充市|南充市]]、[[巴中市|巴中市]]，北界[[陕西省|陕西省]][[汉中市|汉中市]]、[[安康市|安康市]]。地处[[大巴山|大巴山]]南麓，[[川|川]][[渝|渝]][[陕|陕]]结合部，[[嘉陵江|嘉陵江]]支流[[渠江|渠江]]流域。全市总面积1.66万平方公里，常住人口559.77万，是四川省的人口大市、农业大市、资源富市、工业重镇和交通枢纽，是著名的革命老区。达州因其丰富的[[天然气|天然气]]资源而有“中国气都、巴人故里”之称，是国家“川气东送”工程的起点站。2011年达州籍画家陈清泉国画《气壮山河》走进人民大会堂。<ref>{{Cite web|title=达州籍画家陈清泉国画《气壮山河》走进人民大会堂-文明联盟四川站|url=http://sc.wenming.cn/zbsc/dz/201103/t20110310_111818.html|work=sc.wenming.cn|accessdate=2019-04-03}}</ref>

== 历史沿革 ==
今市境古时为[[巴国|巴国]]辖地。[[西汉|西汉]]属[[巴郡|巴郡]][[宕渠县|宕渠县]]。[[东汉|东汉]][[永元|永元]]八年（96年）分置[[宣汉县|宣汉县]]，治今达州市区，隶巴郡。东汉末，巴郡改为[[巴西郡|巴西郡]]，[[建安_(东汉)|建安]]二十三年（218年）分置[[宕渠郡|宕渠郡]]。[[南朝梁|南朝梁]][[大同_(南梁)|大同]]二年（536年）改置[[万州_(石城)|万州]]，治宣汉县。[[西魏|西魏]]废帝二年（553年）改万州为[[通州_(西魏)|通州]]，同时宣汉县改为[[石城县_(西魏)|石城县]]。[[隋朝|隋]][[开皇|开皇]]十八年（598年）改石城县为[[通川县|通川县]]。[[大业|大业]]初为[[通川郡|通川郡]]治。[[唐朝|唐]][[武德|武德]]元年（618年）通川郡改称通州。[[天宝|天宝]]元年（742年）再改为通川郡。[[乾元|乾元]]元年（758年）再改为通州。通川县皆为治所。

[[北宋|北宋]][[乾德|乾德]]三年（965年）将通州改为达州，因此地“联络金房，翼带汉沔，西出渠阆，东下夔巫，地居四达之路”，故名。[[明朝|明]][[洪武|洪武]]九年（1376年）改达州为[[达县|达县]]。[[正德_(明朝)|正德]]九年（1514年）复改县为州，属[[夔州府|夔州府]]。[[清朝|清]][[雍正|雍正]]六年（1728年）升为[[达州直隶州|达州直隶州]]，属[[四川省|四川省]]。[[嘉庆|嘉庆]]七年（1802年）改为[[绥定府|绥定府]]。[[民国|民国]]二年（1913年）撤府，县属[[川东道|川东道]]。民国二十四年（1935年），置四川省第十五行政督察区。1933年，[[中国共产党|中国共产党]]建立[[川陕省|川陕省]]，达县为核心地区。

[[中华人民共和国|中华人民共和国]]成立后，1950年，设[[达县专区|达县专区]]，属[[川北行署区|川北行署区]]；设[[大竹专区|大竹专区]]，属[[川东行署区|川东行署区]]。1952年，各专区直属四川省。1953年，撤销大竹专区，将大竹、渠县、邻水3县划归达县专区。达县专区辖[[达县|达县]]、[[万源|万源]]、[[宣汉|宣汉]]、[[开江|开江]]、[[邻水|邻水]]、[[大竹|大竹]]、[[渠县|渠县]]、[[南江|南江]]、[[巴中|巴中]]、[[平昌|平昌]]、[[通江|通江]]11县。1970年，达县专区改称[[达县地区|达县地区]]。1976年由达县析置[[达县市|达县市]]。1979年由万源县析置[[白沙工农区|白沙工农区]]。1993年7月，达县地区、达县市分別更名为[[达川地区|达川地区]]、[[达川市|达川市]]，并将巴中、通江、南江、平昌4县划归新设立的[[巴中地区|巴中地区]]。1999年6月，撤销达川地区，设立地级达州市，原县级达川市改设[[通川区|通川区]]。2013年7月，撤销达县，改设[[达川区|达川区]]。

== 自然资源 ==
达州是国家商品粮生产基地、中国[[苎麻|苎麻]]之乡、中国油[[橄榄|橄榄]]之都、中国[[黄花|黄花]]之乡、中国富[[硒|硒]]农产品之都。矿产储量十分丰富，已探明可开发利用矿产资源28种，其中，[[天然气|天然气]]资源总量3.8万亿立方米，探明储量7000亿立方米，是继[[新疆|新疆]][[塔里木|塔里木]]、[[内蒙古|内蒙古]][[鄂尔多斯|鄂尔多斯]]之后全国最具开发潜力的大气田，是国家“川气东送”工程的起点，到2015年，天然气年生产能力将达到200亿立方米、天然气净化附产[[硫磺|硫磺]]达到400万吨，达州将成为亚洲最大的硫磺生产基地；[[煤炭|煤炭]]储量7.3亿吨，火电装机容量200多万千瓦；[[石灰石|石灰石]]储量5亿吨；苎麻种植面积51万亩，占全国的30%。

== 地理 ==
跨东经106°39′45″—108°32′11″，北纬30°19′40″—32°20′15″，东西绵延177.5公里，南北长223.8公里。地貌山、丘、谷、坝俱全，以中低山，丘陵地貌为主，占辖区面积的98.8%，地势东北高西南低，地形复杂高差悬殊大。全市平均气温14.7℃—17.6℃，多年平均降水量1100mm左右，无霜期258—300天<ref>{{cite web |title=达州概况 |url=http://www.dazhou.gov.cn/articview_20161201161404138.html |publisher=达州市人民政府 |date=2018-05-11}}</ref>。

== 交通 ==
达州处于成都、重庆、西安、武汉四大名城交汇辐射的中心地带，历为川渝鄂陕结合部物资集散地和商贸中心，是国家公路运输179个主枢纽和四川规划的12个次级综合交通枢纽之一。“铁公水空”形成立体交通网络，高速路到成都四个半小时、重庆两个小时、西安四个小时、武汉六个小时。达州火车站是西南第四大火车站，达万高速可直通万州港，达州机场直航北京、上海、广州、深圳、昆明等城市，渠江航运经重庆可直达上海。
 
== 产业发展 ==
为贯彻省委“三大发展战略”，全市上下将全面实施城市“642”、工业园区“15221”和企业“百十一”三大总体战略工程。目前已形成能源、化工、冶金、建材、机电、医药、纺织、食品加工、商贸物流等为主体的产业体系；全市“1＋7”园区发展格局初步构建，4个园区进入省产业园区“1525”工程。其中，达州经济开发区已成为川东北地区建成面积最大、设施最完善的工业园区，已入驻项目73个，正准备申报国家级开发区，宣汉普光、大竹、渠县工业园区已列入《四川省“十二五”经济开发区发展规划》。达州农产品加工集中区获得国家农业部“全国农产品加工基地”称号。

== 山水景观 ==
明代著名地理学家[[徐霞客|徐霞客]]赞达州为“西南奇胜”。历代能工巧匠、文人墨客，留下了大量文物古迹。全市现拥有国家4A级旅游景区2个、3A级旅游景区4个、全国重点文物保护单位7个，国家级自然保护区1个，国家级森林公园2个，国家级地质公园1个。
