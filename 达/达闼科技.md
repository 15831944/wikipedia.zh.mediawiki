'''达闼科技'''是一家云端[[智能机器人|智能机器人]]运营商，于2015年成立。该公司主要研发云端智能机器人运营级别的[[云计算|云计算]]网络安全、大型混合人工智能机器学习平台和安全智能终端和机器人控制器技术。

== 历史 ==
达闼科技在2015年成立，该公司建立了以云端大脑和[[网络安全|网络安全]]为基础之云端智能机器人架构，其研发领域围绕着云端融合智能、高速安全网络和安全智能[[终端|终端]]和机器人控制技术等。

该公司提出并建立了复合人工辅助和[[机器学习|机器学习]]的云端认知计算平台，并将其应用于导盲等领域。现时，该公司正构建全球覆盖的高速安全骨干网络，研发了“DATA”系列产品（一款为云端机器人设计的云端智能控制终端）。
 
该公司的移动内联网云服务使用上述之核心技术，具有云端机器人远程操控的信息安全的功能，该技术已在多个行业得到用户的认同。
<ref>{{cite news|title=达闼科技开启智能领域新征程-科技新闻|url=http://www.stdaily.com/index/kejixinwen/2017-02/24/content_518536.shtml|work=中国科技网}}</ref>

== 产品 ==
=== 硬件 ===
==== 云端智能机器人 ====
云端智能机器人融合了人工智能与人类智慧，并结合了云网端，能够构造云端机器人大脑、机器人神经网络和服务型机器人本体。<ref>{{cite web|title=马斯克担心机器人控制世界 《终结者》会成真么？|url=http://www.sohu.com/a/126458412_117107|website=www.sohu.com}}</ref>
==== DATA云端智能控制终端 ====
DATA云端智能控制终端是机器人实体和云端大脑之间的控制器，其应用实体包括启动云端导盲机器人。“服务机器人本体+专属网络+云端大脑”三者合一，使服务型机器人本体受控于云端大脑，实现“一个大脑、一张网络、百万台机器人”。
<ref>{{cite news|title=达闼科技打造全球云端智能机器人平台-中新网|url=http://www.chinanews.com/it/2017/05-26/8235108.shtml|work=www.chinanews.com}}</ref>

达闼科技在DATA上首先实现了以[[高通|高通]]平台为基础之[[硬件虚拟化|硬件虚拟化]]技术，其部分性能指标能达到商用水平。DATA上实现了以公司的超级硬件接口为基础的“三明治”式硬件扩展架构，将来可以凭此实现云端智能机器人的连接。[[人工智能|人工智能]]利用“三明治”架构支持双Modem，可实现单机双LTE。<ref>{{cite news|title=全球首款虹膜识别云端控制终端DATA闪耀2017MWC|url=http://news.163.com/17/0303/09/CEJHFDKU00018AOP.html|publisher=网易新闻}}</ref>
==== 导盲头盔 ====
META是该公司的第一款云端智能[[可穿戴式電腦|穿戴设备]]，其形态类似头盔，使用了基于HARI的混合智能服务，具有[[人脸识别|人脸识别]]、物体识别、路径规划和避障等功能，能协助视力障碍人群。<ref>{{cite news|title=2017北京国际科技产业博览会开幕 云端智能机器人成亮点_新闻中心|url=http://news.china.com.cn/2017-06/08/content_40989392.htm|publisher=中国网}}</ref><ref>{{cite web|title=新征程！达闼科技云端智能机器人平台发布_网易手机|url=http://mobile.163.com/17/0223/20/CE03I22L0011802A.html|website=mobile.163.com}}</ref>
=== 软件 ===
==== 移动内联网MCS ====
移动内联网云服务MCS（Mobile-intranet Cloud Service），采用了“云网端”一体化的技术。
<ref>{{cite web|title=在乌镇展出导盲机器人 软银领投种子轮的达闼这样看AI趋势|url=http://tech.sina.com.cn/i/2016-11-21/doc-ifxxwrwk1534400.shtml|website=tech.sina.com.cn}}</ref>
其构成分为XaaS云、VBN网络和终端三部分。该技术在终端设计上，使用了”一片双芯”、虚拟化双操作系统等技术，分割了安全环境与互联环境；设计方面，天网拥有网络隐藏的独立信道，透过软件定义网络，软件定义边界，以及区块链技术，提供设备和机器人网络服务；该公司还提供云端智能服务，该服务采用虚拟私有云的技术，能提供设备管控，图像识别，语音识别，大数据分析等服务。
<ref>{{cite news|title=达闼科技携MCS亮相北京科博会|url=http://news.sina.com.cn/o/2016-05-23/doc-ifxsktkz9736423.shtml|work=news.sina.com.cn}}</ref>
==== XaaS Cloud ====
该平台以公司的HARI（Human-Augmented-Robotic-Intelligence）架构为基础，结合了[[人工智能|人工智能]]与人类智能等，具有图像识别，企业协作，机器人运营等功能。
<ref>{{cite news|title=“双创”的风，在往哪个方向吹？|url=http://digitalpaper.stdaily.com/http_www.kjrb.com/kjrb/html/2016-10/10/content_350940.htm?div=-1|publisher=中国科技网}}</ref>
==== VBN ====
“VBN”（VPN Backbone Network）技术透过SDN/NFV构建私有IP骨干网，其通道在网络中不可见，不同于传统网络接入集中式的认证方式，该技术基于[[区块链|区块链]]的去中心化认证，能消除被攻击的风险，构建软件定义边界（SDP）的隐身网络。
<ref>{{cite news|title=海外办公构建新生态 达闼科技VBN安全专网解决方案跨海落地|url=http://news.sina.com.cn/o/2016-11-03/doc-ifxxmyuk5745444.shtml|work=news.sina.com.cn}}</ref>
