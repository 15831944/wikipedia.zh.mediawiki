'''达尔文'''（[[英語|英語]]：'''{{lang|en|Darwin}}'''），又譯'''達爾溫'''，是位于[[澳大利亚|澳大利亚]]西北海岸的主要[[城市|城市]]，[[北領地|北領地]]的[[首府|首府]]，全國第16大城市，[[人口|人口]]約15万人（2016年），属于[[热带气候|热带气候]]，经常受到[[雷暴|雷暴]]和[[龙卷风|龙卷风]]的侵袭，最早有记录的熱帶氣旋发生在1867年，1974年12月25日的[[強烈熱帶氣旋特雷西|強烈熱帶氣旋特雷西]]几乎毁灭了整个城市。达尔文也是唯一经歷過現代[[战争|战争]]的[[澳大利亚|澳大利亚]]城市。1942年2月19日，[[日本帝國|日本帝國]]空軍对达尔文进行了两次[[達爾文空襲|轰炸]]，此后在[[第二次世界大战|第二次世界大战]]中，达尔文遭受過63次轰炸。

达尔文是[[澳大利亚原住民|澳大利亚原住民]]居民最集中的城市，还有很大一部分居民是从[[东南亚|东南亚]]和[[东亚|东亚]]移居的移民，所以达尔文被称为是“澳大利亚多元文化的首府”，由于它距离[[亚洲|亚洲]]最近，所以是重要的出口[[港口|港口]]，主要出口活牲畜（[[牛|牛]]、[[羊|羊]]）和[[矿物|矿物]]。达尔文也是澳大利亚重要的[[军事|军事]]基地和北部海岸巡逻艇的基地。

== 历史 ==
[[File:Darwin_Palmerstone_Town_Hall_DSC03578.jpg|thumb]]

达尔文是于1839年被[[英国|英国]][[生物学|生物学]]家[[查尔斯·达尔文|查尔斯·达尔文]]考察南[[太平洋|太平洋]]时掌舵的船长以他的名字命名的。1869年只有135人定居在此，属于[[南澳大利亚州|南澳大利亚州]]，以当时[[英国首相|英国首相]]命名为[[帕尔摩斯顿|帕尔摩斯顿]]，1870年设立了第一条和南部连接的电报线，1880年[[金|金]]矿的发现使达尔文迅速繁荣起来，1911年设立北領地，正式定名为达尔文市。

在第二次世界大戰[[太平洋戰爭|太平洋戰爭]]中，1942年242架[[日本帝国|日本帝国]]轟炸機对达尔文进行了两轮轰炸，极大地破坏了城市，导致243人炸死，以后又受到了一系列的轰炸。

1974年12月25日的[[強烈熱帶氣旋特雷西|強烈熱帶氣旋特雷西]]，导致50人死亡，70%的城市建筑被毁，有3万人被空运撤离。以后城市几乎全部重建，1980年代在南部20千米又建立了一个卫星城。

1989年8月，當時在[[香港|香港]]的一個組織「居留權代表團」提出租借達爾文作為建設「[[新香港|新香港]]」的建議，惟建議最終不了了之。

2003年9月17日，貫穿[[阿德萊德|阿德萊德]]至達爾文的[[汗號列車|鐵路]]完工通車，打通了澳洲南北的交通，使國民除了飛機以外多了一項廉價的選擇。
這裏有船到達[[東帝汶|東帝汶]][[帝力|帝力]]。

== 地理和气候 ==
[[File:Darwin_skyline.jpg|thumb]]
[[File:Darwin_1824.jpg|thumb]]
达尔文與三个外國的首都距離還比距離自己的首都近。达尔文距澳洲首都[[堪培拉|堪培拉]]3144千米，距离[[巴布亚新几内亚|巴布亚新几内亚]]首都[[莫尔兹比港|莫尔兹比港]]1818公里，距离[[东帝汶|东帝汶]]首都[[帝力|帝力]]只有656千米，距离[[印度尼西亚|印度尼西亚]]首都[[雅加达|雅加达]]2735千米。即使距离[[菲律宾|菲律宾]]首都马尼拉也只有3206千米，距离[[新加坡|新加坡]]3360千米。

达尔文屬[[热带草原氣候|热带草原氣候]]（柯本氣候分類法：Aw）<ref name="DarwinAirportAWS">{{cite web | title = Darwin Airport | publisher=Bureau of Meteorology |url=http://www.bom.gov.au/climate/averages/tables/cw_014015_All.shtml | accessdate =4 November 2014}}</ref>，一年之中只分为[[雨季|雨季]]和[[旱季|旱季]]，旱季为5月至9月，6月和7月是最凉的月份，气温为15 到35℃，雨季时常有[[热带风暴|热带风暴]]，降水最多的是12月至3月期间，这段期间[[湿度|湿度]]超过70%，常发生[[雷暴|雷暴]]。

* 1月份平均最高温度为32℃。
* 1月份平均最低温度为24℃。
* 7月份平均最高温度为30℃。
* 7月份平均最低温度为20℃。
* 全年平均降[[水|水]]1669mm。
* 降水量最高的1月份，平均降水为422 mm。
* 全年降[[雨|雨]]日110天。

== 政府 ==
[[File:Northern_Territory_Legislative_Assembly.jpg|170px]]

北領地的议会就是设在达尔文，达尔文的行政职能自1957年后即由市议会执行，市议会由市长和12名市议员组成，四个选区，每个选区选举3名市议员。

== 经济 ==
[[File:Darwin_CBD-2005.jpg|left]]

达尔文的[[经济|经济]]支柱主要是采矿业和[[旅游|旅游]]业，采矿业每年产值为28亿[[美元|美元]]，主要矿产为黄[[金|金]]、[[铝|铝]]土、[[锰|锰]]，海中的油气田和海底[[铀|铀]]矿。8%的达尔文居民从事旅游业，旅游业还有很好的扩展前景。驻军和军事设施也雇佣许多达尔文居民并促使经济增长，为了维护东帝汶治安，澳大利亚在达尔文的驻军在2001年已经达到11000人，纵贯南北的铁路也增加了和[[亚洲|亚洲]]之间的贸易，使得达尔文的港口地位日益重要。

2005年有两个重要项目动工，一个是威克曼天然气工厂，预计2006年投产；另一个是扩建斯托山码头，刚开始动工；还有一个[[唐人街|中国城]]项目已经完工建成，2007年已有部分商户和政府办公机构入住。

== 教育 ==
根据2001年的统计资料，全市有35所小学12所中学，小学在学生为6441人，中学在学生为3943人（其中公立学校2825人，私立学校1118人），全澳大利亚境内都承认北領地颁发的中学毕业证（NTCE）。

主要的大学是[[查爾斯達爾文大學|查爾斯達爾文大學]]，在北領地各地有许多分校，中心校区在达尔文市区，有19000学生。

== 文化 ==
[[File:DarwinStreet.jpeg|thumb]]

达尔文是“通往亚洲的大门”，是多元文化的典型，全市人口包括有75个民族，米切尔路是夜总会和餐馆集中的街，有许多露天餐馆。

每年举办“达尔文节”，包括[[戏剧|戏剧]]、[[音乐|音乐]]、[[电影|电影]]等；每年8月举办“啤酒罐船比赛”，用啤酒罐做的船划船比赛，还有达尔文杯赛马也在8月举行。

有一个大型水族馆的“[[印度洋|印度洋]]和[[澳大利亚|澳大利亚]]珍珠业展览馆”也座落在达尔文。
