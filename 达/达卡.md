'''达卡'''（{{lang-bn|ঢাকা}}），是[[孟加拉国|孟加拉国]][[首都|首都]]，[[達卡專區|达卡专区]]首府。达卡坐落于[[恒河三角洲|恒河三角洲]][[布里甘加河|布里甘加河]]北岸，<ref name="xinhua">{{cite web |url=http://news.xinhuanet.com/ziliao/2003-11/22/content_1192813.htm |title=清真寺之城——孟加拉国首都达卡 |publisher=[[新华网|新华网]] |date= |accessdate=2008年12月11日 |language=zh |deadurl=yes |archiveurl=https://web.archive.org/web/20081228195234/http://news.xinhuanet.com/ziliao/2003-11/22/content_1192813.htm |archivedate=2008年12月28日 }}</ref>是孟加拉国最大的城市，也是[[南亚|南亚]]主要城市之一，其[[都会区|都会区]]有1200多万人口。<ref name="BangladeshStatPock2008"/>达卡有着丰富的历史，被称为“[[清真寺|清真寺]]之城”，其颇具民族风情的[[穆斯林|穆斯林]][[纺织品|纺织品]]非常闻名。<ref name="xinhua"/><ref name="sta">{{cite web

达卡旧称贾罕吉尔纳加尔（[[孟加拉语|孟加拉语]]: জাহাঙ্গীর নগর），1608年由[[莫卧儿帝国|莫卧儿帝国]]孟加拉地区总督苏贝达·伊斯兰汗建立，<ref name="xinhua"/>在17世纪是世界性的[[穆斯林|穆斯林]]贸易中心。然而，达卡的现代城市建设主要始于19世纪[[英属印度|英国殖民时期]]，并迅速成为仅次于[[加尔各答|加尔各答]]的[[孟加拉|孟加拉地区]]第二大城市。1947年[[印巴分治|印巴分治]]后，达卡成为[[东巴基斯坦|东巴基斯坦]]的行政首都，并在1972年成为独立的孟加拉国的首都。此间的20多年，达卡经历了各种混乱，包括强制推行[[戒严|戒严法]]、[[孟加拉人民共和国临时政府|孟加拉国]]宣布独立、军事镇压、战争破坏，以及自然灾害等。

达卡是孟加拉国的[[政治|政治]]、[[文化|文化]]和[[经济|经济]]中心。达卡在孟加拉国的城市之中有着最高的识字率（62.3%）<ref name="Banglapedia">{{cite web|url=http://banglapedia.search.com.bd/HT/D_0156.htm |title=Dhaka District |publisher=Banglapedia |date= |accessdate=2008年12月11日 |language=en }}</ref>和最多样的经济结构。虽然其城市基础设施是全国最发达的，但仍面临着[[环境污染|环境污染]]、[[交通堵塞|交通堵塞]]、供应短缺等多方面的挑战。最近几十年来，达卡实现了运输、通信和公共建设的现代化，吸引了可观的国外投资，商业和贸易稳步发展，因此也增加了许多外来人口。主要城市是[[中国|中国]][[广州市|广州市]]和[[巴基斯坦|巴基斯坦]][[卡拉奇|卡拉奇]]。

== 历史 ==
[[File:National_Assembly_of_Bangladesh_(06).jpg|thumb]]
{{main|达卡历史}}
达卡所在的区域最早成为有人定居的城市可以追溯到公元7世纪。这一地区曾被[[佛教|佛教]]王国[[迦摩缕波|迦摩缕波]]和[[波罗王朝|帕拉帝国]]统治，直到公元9世纪时被[[印度|印度]][[森纳王朝|森纳王朝]]所控制。<ref name="PMH">{{cite web
| date = 2006-09-05
| url = http://www.dhakacity.org/his_pre_mughal.html
| title = History
| publisher=Dhaka City Corporation
| format = PHP
| accessdate = 2006-09-05
| deadurl = yes
| archiveurl = https://web.archive.org/web/20080410184215/http://www.dhakacity.org/his_pre_mughal.html
| archivedate = 2008-04-10
}}</ref>许多人认为该城得名于巴拉尔·森纳12世纪时在此修建的达嘎湿瓦力神庙。达卡及周边地区在那一时期被划界为“孟加拉地区”。该城由Lakshmi Bazar、Shankhari Bazar、Tanti Bazar、Patuatuli、Kumartuli、Bania Nagar和Goal Nagar等贸易中心组成。森纳王朝之后，达卡相继被[[德里苏丹国|德里苏丹国]]的[[突厥|突厥]]和[[普什图族|阿富汗]]统治者统治，直到1608年[[莫卧儿帝国|莫卧儿人]]的到来。
