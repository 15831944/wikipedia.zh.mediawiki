'''达延汗'''（{{mong|m=ᠳᠠᠶᠠᠨ<br>ᠬᠠᠭᠠᠨ|k=Даян Хаан|Ppp=Dayan qaγan}}；{{bd|1473年||1517年}}），是“大元汗”的另一种音译，意为“大元大可汗”，因多次遣使入貢，[[明朝|明]]人常称其为'''小王子'''，本名'''巴图蒙克'''（{{mong|m=ᠪᠠᠲᠤᠮᠥ᠋ᠩᠬᠡ|k=Батмөнх|Ppp=Batumöngke}}；又译把秃蒙克、把秃猛可），是[[成吉思汗|成吉思汗]]第十五世孙，[[巴彦蒙克|巴彦蒙克]]（[[阿噶多尔济|阿噶多尔济]]之孙）之子，[[蒙古大汗|蒙古大汗]]。

[[满都鲁|满都鲁]]死后，他的第二位妻子[[满都海哈屯|满都海夫人]]拥立其姪孫巴图蒙克为大汗，就是达延汗，33歲的满都海夫人也改嫁给了7歲巴图蒙克。巴图蒙克1480年（一说1479年）继位，是为达延汗。

达延汗年幼时，满都海被授予“彻辰”称号，[[摄政|摄政]]。1487年，16岁的达延汗开始亲政。经过与[[卫拉特|卫拉特]]部的数度征战，将其击溃。1506年，达延汗征服[[亦思马因|亦思马因]]、[[火筛|火筛]]、[[亦卜剌|亦卜剌]]，统一漠南蒙古。后人称其为蒙古的“中兴之主”。正德十二年（1517年），入侵應州，死於征途。

# [[乌鲁斯博罗特|乌鲁斯博罗特]]（[[济农|济农]]，死后无嗣，满都海哈屯所生）
# [[巴尔斯博罗特|巴尔斯博罗特]]（[[济农部|济农部]]、[[土默特部|土默特部]]的始祖，满都海哈屯所生）
# [[阿尔苏博罗特|阿尔苏博罗特]]（[[多伦土默特部|多伦土默特部]]的始祖，满都海哈屯所生）
# [[阿尔楚博罗特|阿尔楚博罗特]]（[[扎鲁特|扎鲁特]]、[[巴林部|巴林部]]的始祖，满都海哈屯所生）
# [[斡齐尔博罗特|斡齐尔博罗特]]（[[克什克腾|克什克腾]]的始祖，满都海哈屯所生）
# [[阿尔博罗特|阿尔博罗特]]（掌[[浩齐特部|浩齐特部]]，后部归图鲁博罗特之孙库登汗，满都海哈屯所生）
# [[格哷博罗特|格哷博罗特]]（掌[[敖汉部|敖汉部]]、[[奈曼部|奈曼部]]，后部归图鲁博罗特曾孙岱青杜勒、额森伟徵诺颜，苏密尔哈屯所生）
