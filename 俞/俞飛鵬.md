'''俞飛鵬'''{{BD|1884年|12月29日|1966年|12月19日}}，[[浙江|浙江]][[奉化|奉化]]人。[[中華民國|中華民國]]軍事將領，[[中華民國陸軍|中華民國陸軍]][[二級上將|二級上將]]，是[[蔣中正|蔣中正]]的同乡和嫡系亲信，后勤总管。

== 早年經歷 ==
俞飞鹏出生于[[浙江|浙江]][[奉化|奉化]]，家庭贫寒，1906年考入[[宁波|宁波]]府师范学堂，1908年毕业回[[奉化|奉化]]担任小学体育教师。

== 中年經歷 ==
1911年[[辛亥革命|辛亥革命]]爆发，俞飞鹏赶往[[上海|上海]]，加入上海学生军。[[蒋介石|蒋介石]]担任沪军第五团团长后，即任命俞为军需官。

1913年，经蒋力荐，俞被保送入[[北京|北京]][[军需学校|军需学校]]，系统地学习了军事运输和后勤知识。

1924年蒋被[[孙中山|孙中山]]任命为[[黄埔军校|黄埔军校]]筹备委员会委员长，他任命俞为筹备委员兼军需部临时主任。蒋当上黄埔军校校长后，即任命俞为军需部副主任。

1926年11月20日，俞父親俞德桂病故原籍，俞電蔣介石請假奔喪，蔣在南昌覆電：「南昌俞總監鑒：呈悉，驚聞有失怙之戚，不勝愴悼。惟軍事方殷，該總監轉餉輸粟，責任殊鉅，古有墨縗從軍之禮，況在革命尚未成功之日，所望勉抑哀思，在軍成服，所請辭職，礙難照准。中正。號。」<ref>安淑萍、王長生著，《蔣介石誄辭說屑》，台北：傳記文學出版社，2009年5月15日，第8頁</ref>

1928年，被任命为[[国民革命军|国民革命军]]兵站总监，同年调任行政院军政部军需署署长。

1930年，任[[中華民國交通部|國民政府交通部]]次长直至部长，其任职期间，建成9省长途[[电话|电话]]网。开通中英、中美国际长途无线电话，开通[[上海|上海]]、[[广州|广州]]、[[汉口|汉口]]之间无线电话。

1937年3月3日，中國國民黨中政會決議，[[顧孟餘|顧孟餘]]辭去交通部長，俞飛鵬任交通部長<ref name="民" >{{cite book |editors=[[李新_(1918年)|李新]]總主編，中國社會科學院近代史研究所中華民國史研究室，韓信夫、姜克夫主編 |title=《中華民國史大事記》 |location=北京 |publisher=[[中華書局|中華書局]] |year=2011 |isbn=9787101079982}}</ref>{{rp|5377}}。1938年，兼任中央军事委员会后勤部部长。1941年兼任中缅运输总局局长，在[[仰光|仰光]]沦陷前将当地物资抢运回国。

1947年，被授予[[中華民國陸軍|陆军]][[二级上将|二级上将]]军衔。7月19日，國民政府特任俞飛鵬為行政院政務委員兼糧食部部長<ref name=民 />{{rp|8386}}。7月30日，糧食部長俞飛鵬在糧政會議上稱：中國現已經由美國轉請聯合國糧食委員會撥付糧食萬噸，借以救濟上海、南京、北平、天津、廣州、廈門、青島及福州等17城市<ref name=民 />{{rp|8391}}。

== 晚年經歷 ==
1949年，到[[台湾|台湾]]后，辞去一切军政职务，只担任[[中華民國總統府|总统府]][[中華民國總統府國策顧問|国策顾问]]。其间先后担任[[招商局|招商局]]（今[[陽明海運|陽明海運]]）董事长，[[中華民國中央銀行|中央银行]]副总裁等。1966年因[[肺炎|肺炎]]壽終於[[台北市|台北市]]榮民總醫院，年八十四歲。12月22日，蔣介石輓以「愴懷勳碩」，12月24日公祭。<ref>台北：《[[中央日報|中央日報]]》，1966年12月24日，第3版，刊安淑萍、王長生著，《蔣介石誄辭說屑》，台北：傳記文學出版社，2009年5月15日，第8頁，ISBN 978-957-8506-69-5</ref>

==身後==
據說俞飛鵬所獲頒的中華民國最高榮譽[[青天白日勳章|青天白日勳章]]，遭到其後人變賣，由收藏家以八百萬台幣取得。<ref>[http://tw.nextmedia.com/applenews/article/art_id/33584991/IssueID/20110809 「免死金牌」名將勛章800萬賣掉 僅頒發208枚 俞飛鵬子孫流出]</ref>
