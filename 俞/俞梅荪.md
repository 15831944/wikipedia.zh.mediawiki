'''俞梅荪'''，男，1953年生于[[中华人民共和国|中国]][[北京|北京]]<ref name="俞梅荪：从国务院秘书到访民，感怀胡耀邦“简法护民”立法观">{{cite web | url=http://www.chinainperspective.com/ArtShow.aspx?AID=77404 | title=俞梅荪：从国务院秘书到访民，感怀胡耀邦“简法护民”立法观 | accessdate=2015-12-16 |publisher=纵览中国|}}</ref>，[[俞颂华|俞颂华]]之孙<ref name="中南海众生相—专访前国务院办公厅秘书俞梅荪（一）">{{cite web | url=http://www.rfa.org/mandarin/zhuanlan/butongdeshengyin/jkdv-11302012152623.html | title=中南海众生相—专访前国务院办公厅秘书俞梅荪（一） | accessdate=2012-11-30 |publisher=自由亚洲电台|}}</ref>，1984年毕业于[[北京大学|北京大学]]法律系，[[公共知识分子|公共知识分子]]、法律学者，前国务院办公厅秘书<ref name="前国务院秘书俞梅蓀上告无门">{{cite web | url=http://cn.rfi.fr/%E4%B8%AD%E5%9B%BD/20121201-%E5%89%8D%E5%9B%BD%E5%8A%A1%E9%99%A2%E7%A7%98%E4%B9%A6%E4%BF%9E%E6%A2%85%E8%93%80%E4%B8%8A%E5%91%8A%E6%97%A0%E9%97%A8 | title=前国务院秘书俞梅蓀上告无门  | accessdate=2012-12-01 |publisher=法国国际广播电台|}}</ref><ref name="百年胡耀邦， 中共之殇或国家之光">{{cite web | url=https://www.voachinese.com/a/huyaobang-china-20151118/3064046.html | title=百年胡耀邦， 中共之殇或国家之光  | accessdate=2015-11-18 |publisher=美国之音|}}</ref>。1984至1994年从事立法工作，曾任国务院经济法规研究中心综合秘书组组长、中国经济法研究会研究部主任、国务院办公厅秘书，任[[顾明|顾明]]秘书。参与《“七五”立法规划》和一些重要法律法规的起草，后因“泄露国家机密罪”入狱三年。出狱后开启漫长的维权之路<ref name="俞梅荪：从国务院秘书到访民，感怀胡耀邦“简法护民”立法观">{{cite web | url=http://www.chinainperspective.com/ArtShow.aspx?AID=77404 | title=俞梅荪：从国务院秘书到访民，感怀胡耀邦“简法护民”立法观 | accessdate=2015-12-16 |publisher=纵览中国|}}</ref>。

==人物生平==

*1953年生于[[中华人民共和国|中国]][[北京|北京]]；
*1984年毕业于[[北京大学|北京大学]]法律学系；
*1984年起在[[中南海|中南海]]国务院经济法规研究中心、国务院法制局从事立法工作，历任综合秘书组副组长、组长，参与[[中华人民共和国|中华人民共和国]]“七五立法”<ref name="胡耀邦簡法護民的立法觀">{{cite web | url=http://www.open.com.hk/old_version/0905p72.html | title=胡耀邦簡法護民的立法觀
 | accessdate=2009-04-12 |publisher=开放杂志|}}</ref>规划和一些重要法律法规的起草；
*1987年起，任中国经济法研究会研究部副主任、主任，《经济法制》杂志编辑部主任；
*1986年任《经济法治系统工程研究》课题组长；
*1989年起担任国务院办公厅秘书，担任全国人大法律委员会副主任顾明的秘书；
*1991年被评为经济法学副研究员；
*1994年1月疑遭人陷害，被北京市国家安全局以涉嫌危害国家安全的泄密罪逮捕，同年八月被判处有期徒刑三年；时任第七届全国人大法律委员会副主任、[[中国政法大学|中国政法大学]]校长[[江平|江平]]為俞梅蓀做无罪辩护<ref name="前国务院秘书俞梅蓀上告无门">{{cite web | url=http://cn.rfi.fr/%E4%B8%AD%E5%9B%BD/20121201-%E5%89%8D%E5%9B%BD%E5%8A%A1%E9%99%A2%E7%A7%98%E4%B9%A6%E4%BF%9E%E6%A2%85%E8%93%80%E4%B8%8A%E5%91%8A%E6%97%A0%E9%97%A8 | title=前国务院秘书俞梅蓀上告无门  | accessdate=2012-12-01 |publisher=法国国际广播电台|}}</ref>。
*出狱后，俞梅荪开始了长达十余年的维权上访历程。
