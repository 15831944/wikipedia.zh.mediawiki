[[File:俞鴻鈞.jpg|thumb]]
'''俞鴻鈞'''（{{bd|1898年|1月4日|1960年|6月1日|catIdx=Y俞}}），前[[中華民國|中華民國]]行政院長，財經專家。與[[政學系|政學系]]關係密切。

== 早年經歷 ==
俞鴻鈞生於今[[廣東省|廣東省]][[江門市|江門市]][[蓬江區|蓬江區]][[荷塘镇|荷塘镇]]塘坦村，[[上海聖約翰大學|上海聖約翰大學]]西洋文學系畢業。[[國民革命軍北伐|國民革命軍北伐]]時，出任[[上海市|上海市]]財政局代理局長而邁入政壇，以具備傑出[[英語|英語]]能力和財政專長而知名。

== 中年經歷 ==
{{main|中國通貨膨脹 (1938年-1950年)|中華民國國庫黃金運送台灣行動|俞鴻鈞內閣}}
俞鴻鈞曾擔任[[上海市政府|上海市政府]][[秘書長|秘書長]]。1937年3月23日，行政院決議俞鴻鈞代理上海市市長<ref name="民" >{{cite book |editors=[[李新_(1918年)|李新]]總主編，中國社會科學院近代史研究所中華民國史研究室，韓信夫、姜克夫主編 |title=《中華民國史大事記》 |location=北京 |publisher=[[中華書局|中華書局]] |year=2011 |isbn=9787101079982}}</ref>{{rp|5389}}。5月10日，上海市市長俞鴻鈞就上海越界築路問題發表談話，指責日方「枝節橫生」，「漠視我主權」，並稱「市府方針決不變更」<ref name=民 />{{rp|5422}}。1937年7月27日，國民政府任命俞鴻鈞為上海市市長<ref>國民政府公報第2714號，1937年7月28日</ref>，俞鴻鈞發表告上海市民書稱，抱鎮定沉着精神，應付一切事變<ref name=民 />{{rp|5523}}。[[國民政府|國民政府]]撤往[[重慶|重慶]]後，調任[[外交部|外交部]]政務次長、[[中央信託局|中央信託局]]局長、[[財政部|財政部]]長。

[[File:1947年國防部財政部布告(傑諧字第七十二號).jpg|thumb]]俞鴻鈞發布布告(諧字第七十二號)。]]
1947年3月13日，財政部長俞鴻鈞繼任四行聯合總處副主席<ref name=民 />{{rp|8311}}。1947年4月23日，國民政府准免財政部部長俞鴻鈞本職；特任俞鴻鈞為行政院政務委員，兼財政部部長<ref name=民 />{{rp|8340}}。5月23日，財政部長俞鴻鈞向國民參政會報告財政，謂去年預算赤字2.7億元，原因與軍費及復員救濟支出過分龐大；在答覆參政員質詢時，稱孚中、揚子公司調查報告不能公布，黃金潮案尚在審理中<ref name=民 />{{rp|8360}}。

1948年1月3日，財政部部長俞鴻鈞在[[國民參政會|國民參政會]]駐委例會上報告財政設施情況<ref name=民 />{{rp|8482}}。5月21日，[[中華民國中央銀行|中央銀行]]總裁[[張嘉璈|張嘉璈]]辭職，俞鴻鈞被任命為中央銀行[[總裁|總裁]]<ref name=民 />{{rp|8607}}。7月26日，[[蔣介石|蔣介石]]偕夫人赴[[莫干山|莫干山]]，接見俞鴻鈞等商討財政問題，經上海返回南京<ref name=表 />{{rp|55}}。

[[動員戡亂|動員戡亂]]時，俞鴻鈞從國庫裡撥出一筆經費，以箱子銀元，提供蔣在[[溪口鎮_(宁波市)|溪口]]指揮全國軍政事務之所有辦公費用<ref>翁元：《我在蔣介石父子身邊的日子》，台北：圓神出版社，2001年11月新版，第17頁</ref>。1949年1月10日，蔣介石派[[蔣經國|蔣經國]]「赴上海訪俞鴻鈞先生，希其將中央銀行[[現金|現金]]移存台灣，以策安全。」<ref name="靜" >蔣經國：〈危急存亡之秋〉，刊《風雨中的寧靜》，台北：[[正中書局|正中書局]]，1988年</ref>{{rp|131}}1月16日，蔣「約見俞鴻鈞席德懋二先生指示中央中國兩銀行[[外滙|外滙]]處理要旨」<ref name=靜 />{{rp|134}}。3月13日，赴[[溪口鎮_(宁波市)|溪口]]<ref name=靜 />{{rp|168}}。10月1日，蔣經國奉命自[[廣州|廣州]]飛往[[香港|香港]]訪俞鴻鈞，10月2日蔣經國即返廣州<ref name=靜 />{{rp|250}}。

[[中華民國政府|中華民國政府]]遷台後，俞鴻鈞兼任中央銀行總裁與[[交通銀行|交通銀行]]、[[中國農民銀行|中國農民銀行]]、[[台灣銀行|台灣銀行]][[董事長|董事長]]。1953年4月，蔣介石以俞鴻鈞繼[[吳國楨|吳國楨]]為[[台灣省政府|台灣省政府]]主席<ref name="表" >[[陳布雷|陳-{布}-雷]]等編著：《蔣介石先生年表》，台北：[[傳記文學|傳記文學]]出版社，1978年6月1日</ref>{{rp|74}}。俞並兼任[[台灣警備總司令部|臺灣省保安司令部]]司令。8月，颱風成災，蔣介石特囑咐台灣省政府主席俞鴻鈞辦理緊急救濟<ref name=表 />{{rp|74}}。1954年至1958年，任[[行政院長|行政院長]]。1955年9月，俞鴻鈞發表嚴正聲明，藉「自治」之名出賣[[新疆|新疆]]，係蘇俄侵略陰謀，一切均屬無效<ref name=表 />{{rp|82}}。1956年7月7日，俞鴻鈞主持[[中部橫貫公路|中部橫貫公路]]的開工典禮。

== 晚年 ==
1957年4月，發生[[劉自然事件|劉自然事件]]，俞鴻鈞內閣請辭以示負責，獲慰留。

1958年，因拒絕[[監察院|監察院]]約詢案遭到彈劾。辭職後復任中央銀行總裁。

俞鴻鈞曾告訴蔣經國：「外行的生意決不可以做，就是內行的生意，倘使沒有實權，亦不可以做。」1949年5月，蔣經國從蔣介石游[[普陀山|普陀山]]，見某寺有堂名曰「雲水」，遂以自況父子相依、海上漂泊，「人生就似雲水」，想起前幾年從事組訓工作，念及俞鴻鈞之言。<ref>「深感這幾句話，太深刻而有意義了。其實不但做生意如此，即做事亦何嘗不是如此」見蔣經國：〈五月反省錄〉，1949年5月，《蔣經國家書》第四冊，台北：「國史館」藏</ref>
