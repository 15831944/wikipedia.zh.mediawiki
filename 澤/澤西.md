'''澤西'''（Jersey），[[英国|英國]][[王权属地|王冠属地]]，位於[[诺曼底大区|諾曼地半島]]外海20公里處的海面上，是[[英吉利海峡|英吉利海峽]]靠近[[法国|法國]]海岸線的[[海峡群岛|海峽群島]]裡，面積與人口數都最大的一座。澤西島與周邊兩座無人島群——曼逵爾（Minquiers）與艾逵胡（Ecréhous）共同組成'''澤西行政區'''（Bailiwick of Jersey），是[[英国|英國]]的皇家屬地而非英國本土的一部分。

== 簡介 ==

澤西行政區的名義元首為英國女王[[伊丽莎白二世|伊麗莎白二世]]，而首長則稱為[[行政长官|行政官]]（Bailiff），雖然[[軍事|國防]]與[[外交|外交]]上的事務完全由英國负责，但是高度自治的澤西行政區擁有自己的[[稅務|稅務]]與[[立法|立法]]系統，有自己的[[眾議院|眾議院]]，甚至發行自己的[[澤西鎊|澤西島鎊]]（其幣值與英國[[英镑|英鎊]]相當，並且可在英國境內使用，但不在國際貨幣市場通行）。目前其相鄰的[[根西|根息行政區]]已經進行改制，將首長換成[[总理制|总理制]]（稱為首席部長），澤西也將會在不久的將來跟進。

== 歷史 ==

澤西行政區的歷史最早可以回溯到933年時，海峽群島被[[诺曼底公爵|諾曼第公爵]][[長劍威廉|長劍威廉]]（William Longsword）併吞成為[[諾曼第公國|諾曼第公國]]的一部分，其后裔[[征服者威廉|征服者威廉]]成為[[英格蘭|英格蘭]]國王，海峽群島也成為[[英格兰王国|英格兰王國]]的一部分。雖然在1204年時法國人收復[[诺曼底大区|諾曼第]]地區，但並沒有在同時將海峽群島收回，使得這些島嶼成為這段史蹟的見證。[[第二次世界大战|第二次世界大戰]]時澤西島與根西島曾遭德軍佔領過，佔領期自1940年5月1日到1945年5月9日止，是二戰期間唯一被[[納粹德國|德國]]掌控過的英國領土。

==堂區==
[[File:Jersey_Map.svg|right]]
澤西行政區的首府為[[圣赫利尔|聖赫利爾]]。島上居民除了以[[英语|英語]]作為[[官方语言|官方語言]]外，也有許多人口是以[[法语|法語]]為[[母語|母語]]，因此法文也是該行政區的官方語言之一。行政上，澤西島分為12個堂區。所有堂區均有其海岸，並以不同的基督教聖人取名，以紀念區教堂的主保聖人。這12個堂區分別如下：
*[[格魯維爾|格魯維爾]]（Grouville，紀念[[格魯維爾的聖馬田|格魯維爾的聖馬田]]，Saint Martin de Grouville）
*[[聖布雷拉德|聖布雷拉德]]（Saint Brélade）
*[[聖紀文|聖紀文]]（Saint Clément）
*[[圣赫利尔|聖赫利爾]]（Saint Helier）
*[[聖約翰_(澤西)|聖約翰]]（Saint John）
*[[聖勞倫斯|聖勞倫斯]]（Saint Lawrence）
*[[聖馬丁|聖馬丁]]（紀念[[老聖馬田|老聖馬田]]，Saint Martin le Vieux）
*[[聖瑪麗|聖瑪麗]]（Saint Mary）
*[[聖旺|聖旺]]（Saint Ouen）
*[[聖彼得|聖彼得]]（Saint Peter）
*[[聖救主|聖救主]]（Saint Saviour）
*[[聖三一|聖三一]]（Trinity）

== 氣候與其他 ==
位於英國以南的澤西島屬於[[温带海洋性气候|温带海洋性气候]]，[[天气|天氣]]較溫和，因此成為英國人非常喜歡的度假觀光勝地之一。觀光業再加上獨立的低稅率環境，使得[[金融服務業|服務金融業]]逐漸在最近幾年來抬頭變成財政主力。除此之外，澤西島的畜牧業也頗負盛名，島上的[[澤西牛|澤西牛]]與花卉栽種是非常重要的輸出產品。
