'''大安森林公園站'''位於[[台灣|台灣]][[台北市|台北市]][[大安區_(臺北市)|大安區]]，為[[台北捷運|台北捷運]][[淡水信義線|淡水信義線]]（[[信義線|信義線]]）的[[捷運車站|捷運車站]]。本站總開挖面積達14,196平方公尺，為台北捷運之最。{{r|柯妙貞|page=353}}

==車站概要==
車站位於[[信義路_(台北市)|信義路三段]]龍門里與民炤里交界處，[[大安森林公園|大安森林公園]]北側，接近[[建國高架道路|建國南路]]口；車站代碼為'''R06'''。因緊鄰大安森林公園而得名。

本站站名長達六字，與台北捷運[[內湖線|內湖線]][[南港軟體園區站|南港軟體園區站]]、[[高雄捷運紅線|高雄捷運紅線]][[高雄國際機場站|高雄國際機場站]]及[[桃園機場捷運|桃園機場捷運]][[新北產業園區站|新北產業園區站]]、[[機場第一航廈站|機場第一航廈站]]、[[機場第二航廈站|機場第二航廈站]]、[[桃園體育園區站|桃園體育園區站]]並列為全台灣最長鐵道車站名。

==車站構造==
地下二層車站<ref>{{cite web |url=http://www.dorts.gov.tw/news/newsletter/ns207/rp207_02.htm |title=存档副本 |accessdate=2008-05-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20080513230855/http://www.dorts.gov.tw/news/newsletter/ns207/rp207_02.htm |archivedate=2008-05-13 }} 捷運報導第207期：大安森林公園站－臺北市新地標，為本市打造出一座有特色、具特殊風貌的捷運車站。陽光大廳及下凹庭園特殊空間的塑造，讓陽光、綠意引進地下車站；遊客可於 地下空間欣賞大自然，並在戶外的山谷中(下凹庭園)欣見桃花源地。下凹庭園的戶外半圓形飲食區與水池、環形迴廊與水瀑、自然山林造景的外環坡地，提供一個 與大自然互動、交心，進行捷運“心靈”之旅的環境。</ref><ref>http://www.xianqiao.net:8080/gb/6/10/5/n1477537.htm{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }} 大紀元報道：台北捷运 大安森林公园站愿景图揭幕</ref>，一個[[島式月台|島式月台]]，六個出入口。

==利用狀況==
本處除捷運站外，信義路上設有「捷運大安森林公園站」公車站牌，有多條公車路線行經。

本站因緊鄰大安森林公園，出口2～5均在公園內，其中出口3、4為光塔，為本站最為顯著的地標。自穿堂層至外的下凹式庭園都是本站最為吸引人潮的地方，幾乎一遇上週休二日（含國定假日、連續假日）都會聚集許多觀光客前往。

唯一美中不足的是，雖然本站是所有台北捷運車站中第一個擁有庭園的車站，因為採用大量的玻璃帷幕搭建車站（下凹式庭園及出口3、4），加上玻璃也會因太陽輻射下加熱，在通車前曾一度爆出在夏天時很容易變成另類的「大溫室」，會致使站內溫度將比站外溫度高。捷運局於事後表示夏天將會開啟遮陽罩。<ref>{{cite web |url=http://www.libertytimes.com.tw/2013/new/aug/1/today-taipei7.htm |title=存档副本 |accessdate=2014-01-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20131226115545/http://www.libertytimes.com.tw/2013/new/aug/1/today-taipei7.htm |archivedate=2013-12-26 }} 玻璃帷幕設計錯誤？捷運大安森林公園站像溫室 熱爆</ref>
