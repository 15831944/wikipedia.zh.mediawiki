'''大卫·勒格诺'''（{{lang-en|'''Dave Legeno'''}}，1963年10月12日–2014年7月）是一名英国演员和[[混合格鬥|混合格鬥家]]。<ref>{{cite web |url=http://people.mtime.com/914779/ |title=大卫·勒格诺 |accessdate= |publisher=时光网 |work= |language =zh-cn}}</ref>

== 生涯 ==
1963年，勒格诺出生于[[英国|英国]][[伦敦市|伦敦市]]的[[馬里波恩|馬里波恩]]<ref>{{Cite web |url = https://www.latinpost.com/articles/17006/20140711/harry-potter-fenrir-greyback-actor-dave-legeno-dies-hiking-in-death-valley.htm |title = 'Harry Potter' Fenrir Greyback Actor Dave Legeno Dies Hiking in Death Valley |author = Yara Simon |date = 2014-07-11 |publisher = Latin Post |language = en}}</ref>，他在学校学校的是表演艺术，并且对[[威廉·莎士比亚|威廉·莎士比亚]]的[[戏剧|戏剧]]研究颇深，之后远赴美国出演电影和电视剧。

勒格诺演出的第一個角色是在[[佳·烈治|佳·烈治]]的《[[邊個夠我薑|邊個夠我薑]]》。他曾在《[[蝙蝠俠：開戰時刻|蝙蝠俠：開戰時刻]]》、《[[傳奇女王伊利沙伯：黃金盛世|傳奇女王伊利沙伯：黃金盛世]]》、《[[百勝雄師|百勝雄師]]》和《[[第七軍團：最後戰役|第七軍團：最後戰役]]》中演出。勒格诺最出名是在《[[哈利波特：混血王子的背叛_(電影)|哈利波特：混血王子的背叛]]》、《[[哈利波特：死神的聖物1|哈利波特：死神的聖物1]]》和《[[哈利波特：死神的聖物2|死神的聖物2]]》中飾演狼人-{zh-cn:芬里尔·格雷伯克;zh-tw:焚銳·灰背;}-（Fenrir Greyback）。

勒格诺為電子遊戲《''{{link-en|The Getaway: Black Monday|The Getaway: Black Monday}}''》配音，聲演主角Eddie O'Connor<ref>{{Cite web |url = https://www.imdb.com/title/tt0418705/ |title = The Getaway: Black Monday (2004) |publisher = IMDb |language = en}}</ref>。

== 逝世 ==
2014年7月6日，他的屍體在美國[[加州|加州]]的[[死亡谷國家公園|死亡谷]]被一對遠足者發現。驗屍報告指他死於中暑，在被人發現屍體前，已經死了3至4天。無任何謀殺懷疑。<ref>{{Cite web |url = https://www.hollywoodreporter.com/news/dave-legeno-harry-potter-dies-717977 |title = 'Harry Potter' Actor Dave Legeno Dies at 50 |author = Debbie Emery |date = 2014-07-11 |publisher = The Hollywood Reporter |language = en}}</ref><ref>{{Cite web |url = https://www.bbc.com/news/entertainment-arts-28274488 |title = Harry Potter actor Dave Legeno dead in Death Valley |date = 2014-07-11 |publisher = BBC News |language = en}}</ref><ref>{{Cite web |url = https://web.archive.org/web/20141006083532/http://www.suntimes.com/news/obituaries/28610916-418/harry-potter-actor-dave-legeno-dies-of-heatstroke-in-death-valley.html#.W0XXWPlKjIU |title = ‘Harry Potter’ actor Dave Legeno dies of heatstroke in Death Valley |date = 2014-08-13 |publisher = Chicago Sun-Times |language = en}}</ref>

電影《[[第七軍團：最後戰役|第七軍團：最後戰役]]》中有紀念他。
