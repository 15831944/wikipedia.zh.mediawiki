'''大衛·安東尼·盧埃林·歐文，歐文男爵'''，[[名譽勳位|CH]]，[[英國樞密院|PC]]，[[倫敦皇家內科醫學院|FRCP]]（英文：'''David Anthony Llewellyn Owen, Baron Owen'''，{{bd|1938年|7月2日|||Owen, David}}）為[[英國|英國]]政治家，1977年至1979年擔任英國外交大臣，是近代最年輕的[[外交及國協事務大臣|外交及國協事務大臣]]。1981年，歐文離開[[工黨|工黨]]而創立[[社會民主黨|社會民主黨]]（Social Democratic Party，由溫和的[[工黨|工黨]]成員組成）。1992年因為其政治生涯的貢獻受封為終身貴族。為上議院[[中立議員|中立議員]]（crossbencher）

== 生平 ==
歐文1938年生於[[威爾斯|威爾斯]]。1962年當他在[[倫敦|倫敦]]的[[聖托馬斯醫院|聖托馬斯醫院]]實習時，就獲得工黨授意參加議員選舉。1965年他當選[[普利茅斯蘇吞區|普利茅斯蘇吞區]]（Plymouth Sutton）的議員。

== 成立社會民主黨 ==
1980年代，歐文與[[詹金斯|詹金斯]]（Roy Jenkins）等人組成[[社會民主黨|社會民主黨]]（Social Democratic Party, SDP），試圖要英國人民擺脫兩黨選擇第三黨，結果共有28名兩大黨的國會議員脫黨加入SDP，也在那時的國會補選中獲得大勝。

== 狂妄症候群 ==
2008年，歐文在英國的醫學雜誌《[[腦|腦]]》（Brain）發表文章，討論政治領導人物的精神疾病，他創造「狂妄症候群」（Hubris Syndrome）一詞，來說明政治領導人物因為權力過大而造成與現實脫節。
