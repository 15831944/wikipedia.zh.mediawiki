《'''大長今'''》（{{kor|k=대장금|rm=Dae Janggeum|hanja=大長今}}）是[[大韩民国|韓國]][[文化廣播公司|MBC]]於2003年9月15日至2004年3月30日播出的[[MBC月火連續劇|創社42週年月火連續劇]]，[[李丙勳|李丙勳]]執導，[[金榮昡|金榮昡]]編劇，女主角長今由[[李英愛|李英愛]]飾演。在[[亚洲|亞洲]]各地播放时掀起「長今熱」，随带引发[[韓劇|韓劇]]观剧熱潮，全球超過90個國家播出此劇，在世界各地傳播韓國傳統文化並掀起了[[韓流|韓流]]。製作費約70億韓元，韓國國內廣告和海外出口獲得了380億韓元的收益，電視劇生產誘導效果高達1119億韓元，拍攝地及[[大長今公園|大長今公園]]也成为[[東亞|東亞]]旅客遊韓熱點。<ref>[http://cn.chosun.com/big5/site/data/html_dir/2013/09/14/20130914000000.html “大長今熱”今猶在 田園生活也愜意] 朝鮮日報中文網</ref><ref>[http://chinese.yonhapnews.co.kr/allheadlines/2013/09/22/0200000000ACK20130922001000881.HTML# MBC電視台打造《大長今》特輯慶祝播放十周年] 韓聯社</ref>

== 故事背景 ==
{{main|大長今}}
朝鮮王朝第11代國王[[朝鮮中宗|中宗]]時期，確實有過一位名為[[長今|長今]]的醫女，《大長今》即是以這名醫女為藍本所創作的虛構故事電視劇。

全劇橫跨朝鮮第9代國王[[朝鮮成宗|成宗]]至13代國王[[朝鮮明宗|明宗]]時代，以長今的故事為主軸，結合了當時所發生的歷史事件，但政治方面著墨不多以賜死[[废妃尹氏|廢妃尹氏]]及[[燕山君|燕山君]]發起[[甲子士禍|甲子士禍]]作為故事開端，童年長今在不知情的情況下參與[[中宗反正|中宗反正]]的聯絡工作，後期不難看出儒臣們勢力大到能使國王屈服，亦能興起黨爭。中宗的[[文定王后|文定王后]]非常著急其子（即[[朝鮮明宗|明宗]]）的病情，可顯示文定王后所面對的宮廷鬥爭非常厲害。

值得一提的，為了在劇中推廣朝鮮宮廷御膳文化，歷代朝鮮宮廷尚膳也經劇組改編後，亦巧妙地一同出現在劇中；如[[蕩平菜|蕩平菜]]、神仙爐、炆軟春雞、駝酪粥等。

== 收視率 ==
=== {{flagicon|KOR}} 韓國收視奇蹟 ===
*2003年9月15日起，在韓國播放期間收視率一直保持在50%左右，並以46.2%的平均收視率獲得2004年度收視之冠，尤其是2004年3月23日大結局時的收視率達到57.8%<ref>[http://cn.chosun.com/big5/site/data/html_dir/2004/03/24/20040324000034.html 《大長今》23日謝幕 平均收視率爲46.2%] 朝鮮日報</ref>，播放達七個月之久，直接收益100億韓元。更誇張的是，《大長今》播完後相當一段時間電視觀眾急劇減少，有人稱這種現象為「大長今後遺症」，因為《大長今》之後沒有可以和它媲美的電視劇出現，所以造成了收視率嚴重滑坡。 

=== {{flagicon|ROC}} 台灣(中華民國) ===
*2004年5月起，在台灣[[八大戲劇台|GTV戲劇台]]熱映3個月，最高收視率達6.22%，收視人口為143萬，不僅創下歷年來韓劇最高，躍居全台第一，甚至連[[三立台灣台|三立台灣台]]本土劇《[[台灣龍捲風|台灣龍捲風]]》也甘拜下風。2011年，《大長今》在台灣已重播多次，平均每年重溫不只1次，收視率仍打敗其它播出中的新韓劇。<ref>[http://ent.appledaily.com.tw/enews/article/entertainment/20111022/33757539/ 《大長今》台客看11次不嫌膩 重播再登收視王] 蘋果日報</ref>

=== {{flagicon|JPN}} 日本 ===
*2004年10月8日起，日本由[[NHK衛星第2頻道|NHK衛星第2頻道]]播放，前半部分的收視率就已經達到了《[[冬季戀歌|冬季戀歌]]》的二點五倍，打破了韓劇在日本的收視紀錄。 

=== {{flagicon|HKG}} 香港 ===
*2005年1月24日起，在香港[[電視廣播有限公司|無綫電視]]播出，全剧平均收視36点，大結局平均收視47點，最高收視50點，收看觀眾人數多達321萬，差不多占全香港人數的一半，為無線自1991年設立個人收視紀錄儀以來最高收視節目，同時也躋身於香港25年電視劇收視紀錄排行榜三甲之首，名次分別為《大長今》（韓劇）第一、《[[上海灘_(1980年電視劇)|上海灘]]》（港劇）第二、《[[101次求婚|101次求婚]]》（日劇）第三。 大長今的五十點收視亦成為香港電視劇受歡迎程度的一個指標。《大長今》也是香港電視史上首次播映的古裝韓劇。

== 評價 ==
[[File:2005TIBE_KoreaPavilion_Corridor_DaeJangGeumPoster.JPG|thumb]]韓國館懸掛《大長今》的巨型海報]]
*一般[[舆论|舆论]]认为《大长今》内容积极向上。[[湖南卫视|湖南卫视]]在宣传推介《大长今》时，使用了“青春励-{志}-”来形容《大长今》。
*也有个别人士认为《大长今》讴歌[[封建社会|封建]][[朝代|王朝]]制度（源自2005年10月中旬的网上文章 [https://web.archive.org/web/20060225062251/http://ent.qianlong.com/4543/2005/10/19/2040@2850384.htm 《我看《大长今》：做奴才居然做出快感来！》]），将[[现实|现实]]理想化，同时也看不出长今除了一心为母亲报仇之外还有多少志向和抱负，“励志”无从谈起。
*出于艺术加工的需要，一些剧情对中医及针灸的作用进行了夸张处理，例如神化了针灸对神经系统的功效，使用针灸可以对人体进行麻醉，并且还可以应用到鱼类；由消化系统处理吸收的中药汤剂可以对抗天花病毒等等。
*但是各种批评中也存在着颠倒黑白、混淆事实的情况：2005年7月，《[[天涯虚拟社区|天涯社区]]》的一名用户刊文 [https://web.archive.org/web/20080501072742/http://www8.tianyaclub.com/publicforum/Content/filmTV/1/109590.shtml 《由韩剧大长今看韩国人无耻的历史文化剽窃》] 中谈及“（《大长今》）声称韩国人发明了[[针灸|针灸]]”（实际上是中国古人最先开始使用针灸治疗疾病的），并在同年9月初《大长今》在湖南卫视開展中国大陆首映时，由反韩流人士在网络大面积传播开来。在中国影视界人士[[张国立|张国立]]在“炮轰[[韩流|韩流]]”的言论中引用后，迅速进入[[平面媒体|平面媒体]]，引起了网络论坛的激烈论战。然而到目前为止的结论是：尽管《大长今》的各种剪辑版本的片源都非常容易获得，没有人能够给出《大长今》中“声称针灸是朝鲜人发明的”的确切、具体的证据；因此，这种批評是没有根据的。《大长今》導演李炳勳接受採訪時，亦加以澄清：“在《大长今》裡，没有说针灸是韩国人发明的。这是误解。中医是从中国传来的，这一点是肯定的；像《[[本草纲目|本草纲目]]》、《[[黄帝内经|黄帝内经]]》等书，一直流传到今天。”
*《[[凤凰卫视|凤凰网]]》评论文章 [https://web.archive.org/web/20060130052906/http://news.phoenixtv.com/phoenixtv/83885040617914368/20050927/648057.shtml 《《大长今》的平民视角》] 认为，《大长今》特点在于“平民视角”。类似的观点还有：《大长今》的主题是人性、宽容与现代[[女性主義|女权主义]]（不是男性化的女权主义），并非传媒所称的“青春励-{志}-”。

== 劇情紕漏和錯誤 ==
《大长今》制作上存在一些歷史性纰漏和錯誤，較為明顯的幾處有：
# 劇中長今曾被韓尚宮派去照顧[[文定王后|文定王后]]的褓姆尚宮。但褓姆尚宮是照顧國王子女的尚宮，不可能去照顧身為大臣子女的文定王后。劇中將照料文定王后的乳母稱為褓姆尚宮為一大錯誤。
# 劇情在[[朝鮮中宗|中宗]]的时代背景下（即[[明武宗|明武宗]]正德、[[明世宗|明世宗]]嘉靖年間），出现了「[[滿漢全席|满汉全席]]」。具体的剧情是：明朝使臣為冊封世子一事至太平館，因使臣患有[[糖尿病|消渴症]]（即糖尿病），韓尚宮為使臣呈上一桌清淡的菜餚，而惹使臣大怒；之後，崔尚宮便說要為使臣呈上[[滿漢全席|滿漢全席]]，討使臣歡心以利冊封世子一事，並表示自己的烹飪技巧了得，滿漢全席對她來說不是難事，并在后来给[[明朝|明朝]][[使节|使节]]呈上了满汉全席。而事实上，满汉全席在[[清朝|清朝]][[乾隆|-{乾}-隆]]年间的《[[扬州画舫录|扬州画舫录]]》中才首次提到，是[[清朝|清朝]][[康熙帝|康熙帝]]六十六歲時，為滿、漢兩族特設三天六宴，提供三百多款佳餚；而中宗在位時正值[[明武宗|明武宗]]、[[明世宗|明世宗]]在位，因此絕不可能有滿漢全席。这里明显是制作人员搜集資料時缺乏相关知识造成的，而且《大长今》剧中所表现出的满汉全席规模也远远不及现实中的。
# 劇中淑媛李氏（李連生）在中宗過世後，由文定王后下旨留在宮中並晉升為正四品昭媛。實際上，淑媛李氏在中宗十五年（1520）就已經逝世，比中宗更早。史實上亦未晉升她的品階。該角色雖以中宗的淑媛李氏為範本，但原型應為一百多年後的孝宗時期的[[安嬪李氏_(朝鮮孝宗)|安嬪李氏]]。
# 劇中文定王后為慶源大君患痘瘡一事於殿外哭鬧，被貞顯王后（慈順大妃）目睹而訓斥。但史實上貞顯王后於中宗二十五年（1530）就已經病逝，而慶源大君則是於中宗二十九年（1534）才出生。因此貞顯王后在慶源大君出生前已經離世，不可能因此事出面訓斥文定王后。
# 劇中濟州醫女張德的原型：長德早於成宗十九年（1488）去世，與長今並非同一時代的人，而史籍上長德的徒弟為貴今。
# 劇中的所有衣服樣式實際為朝鮮中後期的韓服樣式。
# 劇中尚宮的唐衣分深綠和淺綠，乃導演為了區分職位高低而設計。實際上尚宮唐衣應全部為深綠。

== 惡搞 ==
由於《大長今》在多個地方大受歡迎，這套劇亦成為了被[[惡搞|惡搞]]的對象。茲詳列如下：
* 香港的[[手提電話|手提電話]]服務商曾推出TVB與[[英皇娛樂|英皇娛樂]]製作的一個下載節目《飛越大長針》，由[[Twins|Twins]]、[[梁洛施|梁洛施]]及[[關智斌|關智斌]]主演；
* [[胡恩威|胡恩威]]的[[舞台劇|舞台劇]]《[[東宮西宮_(香港舞台劇)|東宮西宮]]》的第4集《西九龍皇帝》加插了一個環節《大匙羮》，利用各種菜式來諷刺當時的[[香港政治|香港政治]]；
* 台灣[[中天電視|中天電視]]的[[政治戲仿節目|政治戲仿節目]]《[[全民大悶鍋|全民大悶鍋]]》有一個名為《大長斤》的環節。
* 在中国大陆地区，也有将《大长今》的韩语主题歌《呼唤》（{{lang|ko|오나라}}）的谐音恶搞成为“打猪歌（[[武大郎|武大郎]]挨豬打）”等[[空耳|空耳]]的内容，还有人创作了《大长今》的恶搞[[漫画|漫画]]。
* 《大长今》播出后，中国大陆地区的许多娱乐节目中有主持人、演员装扮成徐长今和闵政浩的样子来娱乐大众。
* 2006年12月，日本[[18禁|18禁]][[電視頻道|電視頻道]]「JamCity」計畫與韓國[[色情片|色情片]]製作公司「QFilm」共同製作一部與《大長今》極為相似的[[色情片|色情片]]《色女長今》，日本名稱為《官能宮女長今的花園》（{{lang|ja|官能女官ヂャングムの花園}}），由{{lang|ja|イ・ギョンオ}}導演，由一位知名度不高的韓國[[AV女優|AV女優]]{{lang|ja|ダナ}}扮演徐長今，分為第一篇《禁忌料理》（{{lang|ja|禁断の料理編}}）、第二篇《熱情醫術》（{{lang|ja|悦楽の医術編}}）、第三篇與第四篇（當時皆尚未開拍），預計2007年3月24日在日本[[情趣用品|情趣用品]]店與韓國[[色情網站|色情網站]]上市。MBC批評該劇侵犯著作權且損害韓國形象，揚言將先與《大長今》日本版權所有者討論再採取法律行動。
* 在菲律賓，《大長今》被當地的著名搞笑節目《{{tsl|en|Bubble Gang|Bubble Gang}}》惡搞成為[[同性戀|同性戀]]版本。
