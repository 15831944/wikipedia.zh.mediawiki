'''大陳島撤退'''是發生在1955年（[[民國紀元|民國四十四年]]）2月8日-24日，[[中華民國政府|中華民國政府]]在[[美國第七艦隊|美國第七艦隊]]護航下，將目前屬於[[浙江省|浙江省]][[台州列島|台州列島]]的[[大陳島|大陳島]]上的2.8万余名居民（居民包含上下大陳島、[[漁山列島|漁山列島]]與[[南麂列島|南麂列島]]等）全部撤退到[[台灣|台灣]]的作戰計畫，這批「義民」登陸後，因「義無反顧」、「摧家毀舍」，選擇追隨[[中華民國國軍|中華民國國軍]]遷台，故被稱為「[[臺灣大陳人|大陳義胞]]」。

== 背景 ==
1949年，[[第二次国共内战|第二次国共内战]]情勢逐漸明確，[[中国人民解放军|中国人民解放军]]节节胜利，[[中華民國國軍|中華民國國軍]]退守到西南山地與東南沿海的島嶼一帶，繼續進行作戰。1950年，解放军逐步進軍沿海島嶼。春、夏之際，攻占[[海南島|海南島]]，國軍也自[[舟山撤退|舟山撤退]]，加上西南山區的部隊逐步退到泰緬邊境，國軍僅有固守[[台灣|台灣]]及[[金門|金門]][[馬祖|馬祖]]與浙東諸島，並發動沿海的游擊突襲戰術，來牽制干擾解放军。

1950年[[朝鲜战争|朝鲜战争]]爆發，解放军將駐防在東南沿海的兵力移往[[朝鮮半島|朝鮮半島]]，使得大陳島等東南沿海島嶼所受威脅大減，獲得稍微喘息的機會。

1953年韓戰停戰，解放军開始將重心從朝鮮半島移到[[台灣問題|台灣問題]]上，東南沿海島嶼是解放军首先要處理的地區。韓戰停戰後，解放军迅速強化東南沿海基礎建設，從1950年起上海修建可供噴射戰鬥機起降的機場試圖奪取被國軍掌控之制空權。

[[File:旗津大陳社區彩繪牆.jpg|thumb]]
1954年國共多次於東南沿海進行空戰，位於台灣的噴射戰鬥機航程不足無法支援大陳地區，只能依靠[[P-47戰鬥機|P-47戰鬥機]]維持戰力；在技術優勢下[[中国人民解放军空军|中国人民解放军空军]]取得大陳週邊的制空權。9月3日，解放军對[[金門|金門]]發動大規模的[[九三砲戰|九三砲戰]]，引發第一次[[台海危機|台海危機]]。11月14日，於大陳駐守的[[艾瓦茲級護航驅逐艦|太平號驅逐艦]]遭解放軍[[魚雷快艇|魚雷快艇]]擊沉，國軍於大陳地區戰力迅速萎縮，只能固守各島失去主動權。11月，解放军[[戰機|戰機]]先後濫炸砲擊大陳及一江山<ref name="年表">[[陳布雷|陳-{布}-雷]]等編著：《蔣介石先生年表》，台北：[[傳記文學|傳記文學]]出版社，1978年6月1日</ref>{{rp|79}}。

取得客觀條件優勢後，1955年1月18日，解放军发起[[一江山島戰役|一江山島戰役]]，激戰一日過後攻下[[一江山島|一江山島]]。由於一江山島是大陳島的屏障，失去一江山後，大陳島直接暴露在解放军火炮射程內，防守也相對困難很多；加上以距離而論，台灣到大陳島較中國大陸到大陳島的距離遠，在物資補給與戰力維持上相對不易，解放军的[[米格-15戰鬥機|米格-15戰鬥機]]在當地擁有空中优势。另外，[[中華民國|中華民國]]方面也與[[美國|美國]]簽署《[[中美共同防禦條約|中美共同防禦條約]]》，而金門、馬祖及大陳島都不在共同防禦的範圍之內（防禦範圍僅限於台灣與[[澎湖|澎湖]]），經過多方評估後，最後中華民國政府決定主動撤離大陳島，將大陳島上的居民與軍隊全數遷移至台灣，加強鞏固對台灣本島的防守<ref>{{Cite news|url=http://www.chinatimes.com/newspapers/20140618000578-260115|title=照片驚見3歲的他...大陳義胞激動|last=中時電子報|newspaper=中時電子報|accessdate=2016-12-29|language=zh-TW}}</ref>。

== 過程 ==
中華民國政府決定大陳撤退後，[[中華民國國防部|國防部]]將撤退行動命名為[[金剛計劃|金剛計劃]]，由[[美國海軍|美國海軍]][[第七艦隊|第七艦隊]]和國軍合力進行大陳島上軍民的撤退行動。

過程中，[[美軍|美軍]]主要負責大陳島上居民的撤離，並協助航行路線的防衛，在整个撤退过程中，中国人民解放军没有对此一撤退行动进行干扰，也间接保证了此一撤离能顺利完成。其原因是在1月30日[[美国政府|美国政府]]决定帮助中華民國國軍从大陈岛撤退时，已透過[[苏联政府|苏联政府]]向中華人民共和国政府转达了信息：希望在美军帮助國軍撤离大陈岛时，[[中国人民解放军|解放军]]不要采取行动。中華人民共和国政府接到[[苏联|苏联]]转达的这一信息后，指示人民解放军不要攻击从大陈岛撤退的的美軍與中華民國國军<ref>{{cite web |url = http://www.people.com.cn/GB/junshi/42964/3126070.html |title = 杨贵华：一江山岛登陆作战及其胜利的意义和影响}}</ref>。1955年2月2日，毛澤東在[[中國人民解放軍海軍|解放軍海軍]]司令部1955年2月1日關於[[中國人民解放軍海軍航空工程學院|海岸炮兵]]使用的建議給[[中國共產黨中央軍事委員會|中共中央軍委]]之請示[[電報|電報]]上寫下批語予[[彭德懷|彭德懷]]：“在[[蒋军|蒋军]]撤退時，無論有無美（艦）均不向港口及靠近港口一帶射擊，即是說，讓敵人安全撤走，不要貪這點小便宜。”<ref>[[中共中央文獻研究室|中共中央文獻編輯委員會]]編：《建國以來毛澤東文稿》第五冊，北京：中央文獻出版社，1991年2月</ref>{{rp|23}}由於中共方面的间接默许以及美國的協助，使得撤離行動得以和平順利的進行與落幕。

最後一批居民2000人、官兵4000人于2月24日的“[[飛龍計劃|飛龍計劃]]”中全部撤離南麂島<ref>{{cite web |url = http://news.ifeng.com/history/video/detail_2013_03/29/23687232_0.shtml |title=台湾借“飞龙计划”带走最后一批南麂岛军民}}</ref>，大陳軍民轉移工作順利完成，撤運抵台義胞共16,487人<ref name="年表" />{{rp|80}}。大陳島四天共撤離全數大約28,000名居民與軍人，其中居民前往[[台灣|台灣]]安置，軍隊則移防至金門、馬祖等地。

[[中华人民共和国政府|中华人民共和国政府]]则称，國軍在撤退的过程中，对岛上设施进行了极大的破坏，不仅埋设了大量[[地雷|地雷]]，还将大陈岛变成了一片火海。等到解放军登陆大陈岛，只见到满目疮痍，只找到一位老人和一条狗。直到后来中华人民共和国政府动员人民上岛垦荒，才让大陈岛恢复了生机<ref>[http://cpc.people.com.cn/GB/85037/85038/7243033.html 大陳島墾荒隊]</ref><ref>[http://www.taizhou.com.cn/a/20070213/content_11867.html "2月13日：中國人民解放軍解放了大陳島"] {{webarchive|url=https://web.archive.org/web/20111219154450/http://www.taizhou.com.cn/a/20070213/content_11867.html |date=2011-12-19 }}</ref>。

== 抵台後生活 ==
大陳島撤退後，中華民國在大陸地區只有金門、馬祖兩個據點。撤離的大陳居民先是都在[[基隆港|基隆港]]上岸，而後在基隆停留大約一至兩個星期後，政府按漁、農、工、商等志願名額協助義胞登記職別，再陸續安排分發安置到全台灣各地，包括[[臺北都會區|臺北都會區]]、[[桃園縣|桃園縣]]、[[宜蘭縣|宜蘭縣]]、[[花蓮縣|花蓮縣]]、[[臺東縣|臺東縣]]、[[臺南市|臺南市]]、[[高雄市|高雄市]]、[[屏東縣|屏東縣]]等縣市，都有大陳島的居民被安置，而他們的後代更是散佈在全台各地。今天在台灣仍可以見到「[[大陳新村|大陳新村]]」、「[[一江新村|一江新村]]」等安置大陳居民的地方所在<ref name="意识经验">張敦智：〈從大陳島到五和新村的地方意識與移民經驗〉，雲林科技大學</ref>。

為表揚他們對中華民國政府與[[反共|反共]]立場的支持，這批撤離的居民在當時多被稱為「[[臺灣大陳人|大陳義胞]]」。由於他們的命運曾與中華民國政府緊緊結合，抵台後的大陳居民大多對當時執政的[[中國國民黨|中國國民黨]]及蔣家有著很強烈的情感認同，日後投效軍旅、公務者不在少數。而在大陳居民的居住區更是有人修建「蔣公廟」，廟內祭拜[[蔣中正|蔣中正]]，以表達對其感恩之意，其中以[[高雄市|高雄市]][[旗津區|旗津區]]大陳新村的[[蔣公感恩堂|蔣公感恩堂]]最為人所知<ref name="意识经验"/><ref>[https://tw.appledaily.com/new/realtime/20181031/1444170/ 「蔣公是我們的終身父母」　他被當神建廟祭祀]</ref>。

多數大陳人講[[台州話|台州話]]，為[[吳語|吳語]]的一支，抵臺後難以與臺灣的主流族群閩南人溝通，因而陷入孤立狀態<ref>[http://km.moc.gov.tw/myphoto/show.asp?categoryid=34 大陳撤退] {{webarchive|url=https://web.archive.org/web/20150112002737/http://km.moc.gov.tw/myphoto/show.asp?categoryid=34 |date=2015-01-12 }}，國家文化資料庫</ref>。但也有少部分如[[漁山列島|漁山島]]居民先祖來自福建閩南地區，以講閩南語為主，抵臺後與使用臺灣閩南語的[[臺灣閩南人|臺灣閩南人]]幾乎沒有隔閡。漁山島居民在台東[[富岡漁港|富岡漁港]]形成聚落[[富岡新村|富岡新村]]，藝人[[柯受良|柯受良]]出身在此。

一些大陳島居民遷往台灣後，生活相當困苦，其中有人自言居住環境比當初在大陳島上要差很多<ref name="意识经验"/><ref>[http://hk.crntt.com/doc/1013/5/1/3/101351319.html?coluid=9&kindid=6030&docid=101351319&mdate=0612093912大时代的故事：来自浙江大陈岛的基隆居民(组图) 大時代的故事：基隆的浙江大陳居民]</ref>。
