'''大巽他鼠屬'''（'''''Sundamys'''''），[[哺乳綱|哺乳綱]][[囓齒目|囓齒目]][[鼠科|鼠科]]的一屬<ref name = MSW3>{{MSW3 Muroidea |id = 13001821 |pages = 1503-1504| heading = Genus ''Sundamys''}}</ref>，而與大巽他鼠屬同科的動物尚有[[西里伯斯鼠屬|西里伯斯鼠屬]]、扁顱鼠屬（刺扁顱鼠）、矢鼠屬（長尾矢鼠）、狹鼠屬（神女狹鼠）等之數種[[哺乳動物|哺乳動物]]<ref>中國科學院，《中國科學院動物研究所的世界動物名稱資料庫》，{{cite web |url=http://vzd.brim.ac.cn/division/fauna/index.asp |title=存档副本 |accessdate=2016-11-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20050718232917/http://vzd.brim.ac.cn/division/fauna/index.asp |archivedate=2005-07-18 }}</ref>。
