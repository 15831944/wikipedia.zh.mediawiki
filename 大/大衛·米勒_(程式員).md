'''大衛·史提芬·米勒'''（{{lang-en|David Stephen Miller}}，{{bd|1974年|11月26日|||catIdx=Miller, David}}），網路暱稱為 DaveM，生於美國[[新澤西州|新澤西州]][[新布朗斯維克|新布朗斯維克]]，著名程式員與[[駭客|駭客]]，負責[[Linux核心|Linux核心]]網路功能以及[[SPARC|SPARC]]平台的實作。他也參與其他開源軟體的開發，是[[GCC|GCC]]督導委員會的成員之一。
==生平==
根據2013年8月的統計，米勒是Linux核心原始碼第二大的貢獻者，自2005年開始，已經提交過4989個patch<ref>{{cite web| url=http://www.ohloh.net/accounts/davem |title=Ohloh Page for user 'davem' kernel contributions | work = Ohloh, a free public directory of open source software and people | publisher= [[Ohloh|Ohloh]] |accessdate=2013-06-21|website = }}</ref>。
