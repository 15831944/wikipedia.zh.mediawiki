'''大溪車站'''位於[[台灣|台灣]][[宜蘭縣|宜蘭縣]][[頭城鎮|頭城鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[宜蘭線|宜蘭線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[岸式月台|岸式月台]]兩座。
**順行（南下）為第一月台，逆行（北上）為第二月台，兩者以站內天橋連接。

== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]，由[[頭城車站|頭城車站]]管理。
*僅[[臺鐵區間車|區間車]]停靠。
*本站已取消「鐵路之旅-小站巡禮紀念章」之戳章。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。
*降等前為台鐵少數未進行電腦化的車站，亦曾發售[[名片式車票|名片式車票]]。2017年9月10日於[[頭城車站|頭城車站]]進行最後特賣，之後不再販售。<ref>[http://www.railway.gov.tw/tw/news_Detail.aspx?SN=13683 牡丹站及大溪站名片式車票特賣會]</ref>

== 歷史 ==
*1920年12月10日：設站。
*1995年3月21日：由[[三等站|三等站]]降為[[簡易站|簡易站]]，並指定由[[大里車站|大里站]]管理。
*2003年1月1日：管理站由[[大里車站|大里站]]改為[[頭城車站|頭城站]]。
*2015年6月30日：啟用多卡通刷卡機。
*2017年9月1日：本站與[[牡丹車站|牡丹站]]從[[簡易站|簡易站]]降為[[招呼站|招呼站]]。

*內大溪路（大溪產業道路[http://www.urmap.com/?link=QSaXgV3e6Xyv2BHrYXH0Yu5r2i5IYXbr2XHvTXRrYi5IgiRvgo3v2os02VbvTuRI2]，為宜一線部分路段），可經雙泰產業道路至[[新北市|新北市]][[雙溪區|雙溪區]]。
