'''女性家族部'''，[[大韩民国|大韩民国]]国家行政机关之一，部门首长称女性家族部部長，同时被任命为国务委员。

== 沿革 ==
* 1998年2月28日 - 总统直属女性特别委员会设置。
* 2001年1月29日 - 总统直属女性特别委员会扩大为女性部。
* 2005年6月23日 - 部门改组为女性家族部。
* 2008年2月29日 - 部分职责划入[[大韩民国保健福祉家族部|保健福祉家族部]]，部门改组为女性部。
* 2010年3月19日 - 保健福祉家族部有关家族、青少年等方面职责划入，部门改组为女性家族部。

== 职责 ==
女性政策的制定与管理、[[家庭暴力|家庭暴力]]与[[性虐待|性虐待]]的防止及受害者的保护、卖春的防止及受害者的保护、推进女性权益地位的提高等相关事务、妇女儿童的福利与家庭政策。设立世界大韩民族女性组织，策划与海外的韩侨的合作。
