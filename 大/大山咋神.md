'''大山咋神'''（{{lang|ja|おほやまくひ の かみ}}）為《[[古事記|古事記]]》表記之名，祂是[[日本神話|日本神話]]裡出現的[[山神|山神]]，別名有'''山末之大主神'''（{{lang|ja|ヤマスエノオオヌシノカミ}}）、'''鳴鏑神'''（{{Lang|ja|ナリカブラノカミ}}）、'''日吉山王權現'''（{{Lang|ja|ヒエサンノウゴンゲン}}）等。

== 概要 ==
=== 古事記之記載 ===
根據《古事記》上卷的記載，[[大年神|大年神]]與天知迦流美豆比賣（{{Lang|ja|アメノチカルミヅヒメ}}）生下'''大山咋神'''，別名'''山末之大主神'''，坐[[近江國|近江國]]日枝山（{{Lang|ja|ひえのやま}}，即後來的[[比叡山|比叡山]]），又坐葛野之松尾，乃鳴鏑之神。

== 解說 ==
神名裡的「咋」（{{Lang|ja|クイ}}）作「杙」解，也就是建造房屋時作記號用的木樁。將木樁打入大山，所以具有「擁有大山的神明」之意。《古事記》提到的近江國日枝山即現在的[[比叡山|比叡山]]，自從[[天台宗|天台宗]][[延曆寺|延曆寺]]在此建立後，便作為其守護神。該寺在[[平安時代|平安時代]]末期衍生出一支[[山王神道|山王神道]]的流派，山王（{{Lang|ja|さんのう}}）意味比叡山之王。此流派後來經[[天海|天海]]更名為「山王一實神道」（{{Lang|ja|さんのういちじつしんとう}}），故大山咋神出現「'''日吉山王權現'''」（{{Lang|ja|ヒエサンノウゴンゲン}}）的別名。

[[室町時代|室町時代]]武將[[太田道灌|太田道灌]]特地由川越日吉神社的大山咋神分香出來，創建日枝神社以作為[[江戶城|江戶城]]的守護神。後來[[江戶時代|江戶時代]]成為[[德川氏|德川氏]]的[[氏神|氏神]]，[[明治|明治]]以降被認為是[[皇居|皇居]]的鎮守。位於比叡山山麓的[[日吉大社|日吉大社]]乃全日本日枝神社的總本社，後來日枝神社供奉分香而來的[[大物主神|大物主神]]，大物主神被稱為「大比叡」、大山咋神被稱作「小比叡」，二神合稱「山王」。

== 信仰 ==
在日本主祭大山咋神的[[神社|神社]]除了各地的松尾神社、日吉神社、日枝神社、山王神社外，另計有下述：
* [[日吉大社|日吉大社]]（[[滋賀縣|滋賀縣]][[大津市|大津市]]）
* [[松尾大社|松尾大社]]（[[京都府|京都府]][[京都市|京都市]][[西京區|西京區]]）
* [[岡山神社|岡山神社]]（[[岡山縣|岡山縣]][[岡山市|岡山市]][[北區_(岡山市)|北區]]）
* [[日光二荒山神社#境內外社|日枝神社]]（[[栃木縣|栃木縣]][[日光市|日光市]]）
