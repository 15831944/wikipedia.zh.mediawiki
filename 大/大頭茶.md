'''大头茶'''（[[学名|学名]]：{{lang|la|'''''Gordonia axillaris'''''}}）为[[山茶科|山茶科]][[大头茶属|大头茶属]]下的一个种。<ref name="賞樹日記，香港自然網">{{cite news|author =|url =http://www.hknature.net/treediary/diary_33.html|title =大頭茶 Gordonia axillaris|work =賞樹日記，香港自然網|page =|date =}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="樹木谷">{{cite news|author =|url =http://www.hktree.com/tree/gordonia.htm|title =大頭茶|work =樹木谷|page =|date =|deadurl =yes|archiveurl =https://web.archive.org/web/20070610021138/http://www.hktree.com/tree/gordonia.htm|archivedate =2007-06-10}}</ref>。
== 型態 ==

大頭茶特徵樹皮光滑，木質蒴果，帶翅種子。<ref name = "賞樹日記，香港自然網"/><ref name = "樹木谷"/>

樹皮淡褐色；嫩枝與芽密被毛。葉革質，螺旋狀互生於枝端，長橢圓形或長橢圓狀倒披針形，先端鈍或圓，常微凹頭，上半部疏具細圓齒緣，側脈及細脈不明顯。花腋生或近頂生，單一或成對，近無柄；花瓣皺摺狀，先端凹，白色；雄蕊多數，不等長，基部癒合。蒴果長橢圓形，被長毛。種子扁平具翅。<ref name = "南澳原生植物標本園解說手冊">{{cite news  |title = 南澳原生植物標本園解說手冊 |publisher=行政院農業委員會林務局羅東林區管理處出版|isbn=957-01-3081-4|page = p.289 |date = }}</ref>

== 形狀及習性 ==

常綠性喬木，老葉掉落前轉紅。喜溫暖多霧的生育地，常見於低中海拔近山脊處或較向陽的坡面，多與紅楠、杜英或長尾尖葉櫧等較喜陽光的種類混生。<ref name = "南澳原生植物標本園解說手冊"/>

== 物候 ==

花期為每年十月至翌年一月。<ref name = "賞樹日記，香港自然網"/><ref name = "樹木谷"/>
10-2月開花，7-10月果熟。<ref name = "南澳原生植物標本園解說手冊"/>

== 分佈 ==

原產地為香港。生長於灌叢和早期次生林中。主要分佈在[[華南|華南]]、[[臺灣|臺灣]]及中南半島等地<ref name = "賞樹日記，香港自然網"/><ref name = "樹木谷"/>
中國及中南半島。低地至海拔2300公尺山區的闊葉林中。<ref name = "南澳原生植物標本園解說手冊"/>

== 用途 ==

庭園綠化樹種，花大型耀眼。木材緻密，紅色，供建築及器具用材。<ref name = "南澳原生植物標本園解說手冊"/>
樹幹彈性好的大頭茶，是獵人們青睞的陷阱材料，捕捉動物的效率也很高。果實成熟時，常吸引猴群來取食，是獵人常持槍等待的地點。另外，因大頭茶的木材不易吸水，是良好的薪材，亦可做工寮用的柱子，或做成刀柄、杵等日常用具。<ref>{{cite news  |title = 綠色葛蕾扇─南澳泰雅的民族植物|publisher=行政院農業委員會出版|page = p.132、133 |date = }}</ref>

== 特殊 ==

大頭茶於[[鋁離子|鋁離子]]濃度高的污染地區亦能生存，並且能將[[鋁|鋁]]吸收及儲存<ref>《綠色香港─生態欣賞與認識》，花千樹出版。162頁。ISBN 962-8884-44-1。</ref>。
