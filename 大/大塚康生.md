'''大塚康生'''{{BD|1931年|7月11日|catIdx=Ō}}是日本[[島根縣|島根縣]]出身的動畫師，在1960至80年代期間與動畫導演[[高畑勳|高畑勳]]及[[宮崎駿|宮崎駿]]等人合力製作多部作品。

== 經歷 ==
大塚康生於1931年在島根縣[[津和野町|津和野町]]出生，8歲那年全家搬移至[[山口市|山口市]]定居。小時因正逢[[二次大戰|二次大戰]]，當時載運士兵的火車引起大塚康生的興趣，而開始嘗試塗鴉火車或一些機器車輛的圖畫。

二戰結束後，大塚康生曾在[[山口縣|山口縣]]縣署裡的統計課就職。之後在1952年前往東京到[[厚生省|厚生省]]取締麻藥毒品的事務所下工作<ref>{{Cite book |script-title=ja:作画汗まみれ 増補改訂版 |location=日本 |publisher=[[德間書店|德間書店]] |author=大塚康生 |date=2006-05 |pages=18頁  |ISBN=978-419-861-361-7 |accessdate=2012-09-27 }}</ref>。此期間大塚康生因觀賞了《[[長著駝峰的小馬|長著駝峰的小馬]]》、《斜眼暴君》（《[[國王與小鳥|國王與小鳥]]》的雛形作品） 等令他印象深刻的動畫作品，而產生踏入動畫界工作的念頭。

1957年大塚康生進入[[東映動畫|東映動畫]]，成為該公司草創期的初屆職員<ref>{{Cite book |script-title=ja:BRUTUS特別編集 スタジオジブリ |location=日本 |publisher={{lang|ja|マガジンハウス}} |pages=44頁 |ISBN=978-4-83-878658-9 |accessdate=2012-09-06 }}</ref>。首次參與的作品則是在1958年公開的《[[白蛇傳_(動畫)|白蛇傳]]》中擔任動畫師，其為日本第一部彩色動畫。在[[高畑勳|高畑勳]]及[[宮崎駿|宮崎駿]]加入了東映動畫後，大塚康生與他們兩人在東映時映共同推出的電影動畫為1968年的《[[太陽王子_霍爾斯的大冒險|太陽王子 霍爾斯的大冒險]]》<ref>{{Cite book |title= 出發點1979～1996 |location=台灣 |publisher=[[台灣東販|台灣東販]] |author=[[宮崎駿|宮崎駿]] |date=2006-02-01 |pages=541頁  |ISBN=978-957-473-843-4 |accessdate=2012-09-27 }}</ref>。之後大塚康生與高畑勳及宮崎駿陸續轉往到A Production（後來的[[SHIN-EI動畫|SHIN-EI動畫]]）、日本Anime等動畫室，此期間製作了《[[熊貓家族|熊貓家族]]》、《[[未來少年柯南|未來少年柯南]]》、《[[魯邦三世卡里奧斯特羅之城|魯邦三世卡里奧斯特羅之城]]》等代表作品<ref>{{Cite book |script-title=ja:BRUTUS特別編集 スタジオジブリ |location=日本 |publisher={{lang|ja|マガジンハウス}} |pages=45頁 |ISBN=978-4-83-878658-9 |accessdate=2012-09-06 }}</ref>。

除了高畑勳與宮崎駿外，包括[[芝山努|芝山努]]及[[近藤喜文|近藤喜文]]、[[貞本義行|貞本義行]]等多數日本動畫業界人士都曾受過大塚康生的對於動畫技法方面的指導。而在1991年代代木Anime學院便曾聘請大塚康生作為校內動畫科的講師，Telecom Animation工作室也邀請過他作為公司顧問。現今退出業界一線的大塚康生仍從事著教導動畫界後輩的事務。

== 其它 ==
* 平時的打扮是頭上戴著一頂[[貝雷帽|貝雷帽]]。
* 個人擅長的外語是[[中文|中文]]跟[[英文|英文]]<ref>{{Cite web |url=http://www.1101.com/ghibli/2004-07-22.html |script-title=ja:ジブリの仕事のやりかた。宮崎駿・高畑勲・大塚康生の好奇心。 |publisher={{lang|jaほぼ日刊イトイ新聞}} |date=2004-07-22 |accessdate=2012-09-27 }}</ref>。
* 大塚康生私下著迷軍用車輛，其中特別喜好[[吉普車|吉普車]]，並曾為[[田宮公司|田宮公司]]販售的一些吉普車模型提供造型設計。而大塚康生也曾有放棄動畫師、轉職[[MAX模型|MAX模型]]公司工作的念頭<ref>{{Cite book |script-title=ja:大塚康生インタビュー アニメーション縦横無尽  |location=日本 |publisher={{lang|ja|実業之日本社}} |author=大塚康生、森遊機 |date=2006-01-31 |pages=154頁  |ISBN=978-440-861-255-3 |accessdate=2012-09-27 }}</ref>。
* 在東映動畫工作期間，大塚康生曾發生過喝醉酒時，把車子[[菲亞特500|菲亞特500]]開進田園旁施工工地裡後、車子陷入工地泥巴裡開不出去的事情，而被當時的宮崎駿和其他同事們私下當成惡搞的題材<ref>{{Cite book |title= 出發點1979～1996 |location=台灣 |publisher=台灣東販 |author=宮崎駿 |date=2006-02-01 |pages=196～200頁  |ISBN=978-957-473-843-4 |accessdate=2012-09-27 }}</ref>。
