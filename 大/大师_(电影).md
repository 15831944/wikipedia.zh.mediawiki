== 剧情 ==
讲述了一个类似[[山達基|山達基]]（一般稱為'''科學教'''）的宗教组织的建立过程。[[二战|二战]]之后，一个富有感召力的知识分子 Lancaster Dodd（[[菲利浦·西摩·霍夫曼|菲利浦·西摩·霍夫曼]] 饰演）发起了一个宗教组织「The Cause」。一个流浪汉 Freddie Quell（[[華堅·馮力士|華堅·馮力士]] 饰演）加入其中，并成为 Lancaster 的左膀右臂。正当这个宗教组织吸引了大批信徒、如日中天之时，Freddie 却陷入对这个宗教信仰体系和精神导师的深深怀疑之中。

== 评价 ==
影片在[[好莱坞|好莱坞]]试映后，得到了广泛好评：“观看《大师》是我经历过的最棒的看片体验”、“《大师》证明了菲利普·塞默·霍夫曼是活着的最出色的演员之一”；并被认为是[[金狮奖|金狮奖]]与奥斯卡的有力争夺者<ref>{{cite web|url=http://ent.163.com/12/0809/09/88F2S17Q000300B1.html | language= | title=《大师》挺进威尼斯主竞赛单元 好评多有望擒狮 |publisher=Mtime时光网 |date=2012-08-09 |accessdate= }}</ref>。

== 類似山達基教 ==
在劇本公開後，很快便形成了「The Cause」與山達基教（科學教）之間的比較。<ref name="NYMag3">{{Cite web |url = http://www.vulture.com/2009/12/so_this_new_paul_thomas_anders.html |title = So This New Paul Thomas Anderson Movie Is Definitely About Scientology, Right? |author = Lane Brown |date = 2009-12-03 |publisher = NYMag.com |language = en}}</ref><ref>{{Cite web |url = http://www.vulture.com/2010/03/why_does_paul_thomas_andersons.html |title = Universal Passes on Paul Thomas Anderson’s Scientology Movie |author = Lane Brown |date = 2010-03-17 |publisher = NYMag.com |language = en}}</ref><ref>{{Cite web |url = https://www.theguardian.com/world/2011/apr/26/scientology-hollywood-film-studio |title = Church of Scientology snaps up Hollywood film studio |author = Ed Pilkington |publisher = The Guardian |language = en}}</ref>新聞媒體注意到霍夫曼與山達基教創始人[[L·羅恩·賀伯特|L·羅恩·賀伯特]]（1911年-1986年）的體型相似，他在第二次世界大戰中服役於美國海軍，並從醫院出院後在1950年創立了信仰系統，如同劇本中的宗教 。本片故事在英國結束，大致上在同一時間，{{link-en|聖山莊園|Saint Hill Manor}}成為了賀伯特的住所和第一個山達基教「組織」。<ref name="NYMag3" /><ref name="Scientology">{{Cite web |url = http://www.mtv.com/news/2764073/will-scientologists-declare-war-on-paul-thomas-andersons-the-master/#fbid=Sh0pkd5XnLJ |title = WILL SCIENTOLOGISTS DECLARE WAR ON PAUL THOMAS ANDERSON'S THE MASTER? |author = Jen Yamato |date = 2010-06-10 |publisher = MTV |language = en}}</ref>此外，本片提到的一個「身體健壯的水手」（abled bodied seaman）的需要和提到Freddie的心理失常都是山達基教成員在行政詞典中所使用的術語。<ref name="screendaily">{{Cite web |url = http://www.screendaily.com/news/paul-thomas-anderson-on-making-the-master-scientology-was-the-least-of-our-problems/5046000.article?blocktitle=Latest-news&contentID=1846 |title = Paul Thomas Anderson on making The Master: 'Scientology was the least of our problems' |author = Andreas Wiseman |date = 2012-09-03 |publisher = ScreenDaily |language = en}}</ref>

製作公司官方否認本片是鬆散地基於賀伯特，監製JoAnne Sellar也否認與賀伯特有任何聯繫，表示「這是第二次世界-{}-大戰的劇情，這是關於一個在二戰後的流浪漢。」<ref name="Filming">{{Cite web |url = http://www.mercurynews.com/2011/06/07/controversial-hollywood-movie-to-begin-filming-in-vallejo/ |title = Controversial Hollywood movie to begin filming in Vallejo |author = Rachel Raskin-Zrihen |date = 2011-06-07 |publisher = The Mercury News |language = en}}</ref>[[哈維·溫斯坦|哈維·溫斯坦]]否認本片是關於山達基教的：「保羅對我說，電影是關於二戰後一名[[士兵|士兵]]的旅程...這個士兵發生的其中一件事情就是他去了一個異教。」<ref>{{Cite web |url = http://www.huffingtonpost.com/2012/07/26/harvey-weinstein-obama-romney-guns_n_1706394.html |title = Harvey Weinstein On Obama, Gun Control, 'The Master' And The Record-Breaking 'The Intouchables' |author = Mike Hogan |date = 2012-07-26 |publisher = The Huffington Post |language = en}}</ref>安德遜表示他「總是認為賀伯特是一個偉大的人物，如此有趣和誇張，而不可能忽略」<ref name=VillageVoice>{{Cite web |url = http://www.villagevoice.com/film/paul-thomas-anderson-the-masters-master-6436166 |title = Paul Thomas Anderson, The Master's Master |author =  Scott Foundas |date = 2012-09-05 |publisher = VillageVoice.com |language = en}}</ref>，他承認Lancaster Dodd的靈感來自於羅恩·賀伯特，而且他應該知道的是什麼人會理解，表示「我不想把它變成一部傳記作品，這不是L·羅恩·賀伯特的故事。」

幾個網站建議「重要的荷里活山達基成員」反對該項目，因為他們擔心它可能揭露了太多的信仰，而其他人甚至推測[[山達基教會|山達基教會]]有足夠的力量阻止環球影業准許製作這部電影。<ref name="Filming" /><ref name="Scientology" />然而，山達基教的代表並沒有聯繫任何製作團隊。<ref name="NYTimes">{{Cite web |url = http://www.nytimes.com/2012/04/22/movies/paul-thomas-anderson-film-may-be-about-scientology.html?_r=2&pagewanted=all |title = Filmmaker’s Newest Work Is About ... Something |author = Michael Cieply |date = 2012-04-18 |publisher = The New York Times |language = en}}</ref>當山達基教會的發言人{{link-en|Karin Pouw|Karin Pouw}}被問到如果教會對電影有什麼顧慮時，她說：「我們沒有看過這部電影，所以無論如何都不能談」，教會只是從報道中讀到而知道關於本片。<ref name="NYTimes" />根據安德遜說，教會從來沒有對這個項目做任何直接或間接的詢問，或試圖阻礙項目的進展，<ref name=VillageVoice />而當他們在製作電影時，山達基教是他們的問題中最少的一個。<ref name="screendaily" />

在2012年5月，據報導安德遜為他的朋友[[湯·告魯斯|湯·告魯斯]]放映本片，告魯斯是一名演員和直言不諱的山達基成員，而他對電影的多部分有一些問題。<ref>{{Cite web |url = http://www.thewrap.com/master-paul-thomas-anderson-reaches-out-scientologist-tom-cruise-exclusive-41111/ |title = 'The Master': Paul Thomas Anderson Reaches Out to Scientologist Tom Cruise (Exclusive) |author = Sharon Waxman |date = 2012-05-22 |publisher = The Wrap |language = en}}</ref>告魯斯曾經在安德遜的1999年電影《[[人生交叉剔|人生交叉剔]]》中出演過。<ref>{{Cite web |url = http://www.oscars.org/oscars/ceremonies/2000 |title = THE 72ND ACADEMY AWARDS | 2000 |publisher = Academy of Motion Picture Arts and Sciences |language = en}}</ref>山達基教會的官員據聽說，當他們得知一個場景暗示信仰系統是領導人幻想出生的產物時，告魯斯便「勃然大怒」。告魯斯遇到問題的場景，就是Dodd的兒子告訴Quell，Dodd只是隨口編造信仰出來。他們不僅為這個陳述爭論，並且大概與[[隆納·德沃爾夫|小L·羅恩·賀伯特]]與他自己父親的衝突相似。雖然教會成員據說有反對其他的鏡頭，安德遜沒有從電影中把鏡頭刪除。他說告魯斯「看過電影。我們之間是有點事，不過一切都很好。」
