'''大信镇'''是[[中华人民共和国|中国]][[青岛市|青岛市]][[即墨区|即墨区]]下辖的一个镇。
== 行政区划 ==
大信镇下辖乔家村、大信村、官路村、宫家村、新胜村、李家韩洼村、韩家韩洼村、陈家村、仲戈庄村、丁家村、小金家村、大金家村、郝家庄村、碾子头村、小信村、新付庄村、碓臼泊村、石固埠村、高家洼村、大范家村、青钢农场村、小范家村、南王庄村、司家疃一村、司家疃二村、司家疃三村、花园村、赵家疃村、李家疃村、南翟家疃村、北翟家疃村、北王庄村。<ref>{{Cite web|url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2015/37/02/82/370282119.html|title=统计用区划和城乡划分代码/山东省/青岛市/即墨市/大信镇|accessdate=2017-04-23|author=国家统计局|date=2015-09-30|publisher=}}</ref>
== 人口 ==

2010年[[中国第六次人口普查|中国第六次人口普查]]时，大信镇共有人口30043人。共有家庭9036户，平均每户2.90人。14岁以下的少年儿童共4183人，占总人口13.92%；15-64岁人口共22645人，占总人口75.37%；65岁及以上的老年人共3215人，占总人口的10.70%。男性共计15775人，占总人口52.50%；女性共计14268，占总人口47.49%。本地居住的人口中，拥有本地户籍的人口为24296人，占80.87%。<ref name="人口普查2010">{{cite web| url =http://www.yearbookchina.com/navipage-n2013030152000017.html| title =《中国2010年人口普查分乡、镇、街道资料》| author =中国国务院人口普查办公室、中国国家统计局人口就业统计司等| year =2010| website =统计年鉴分享平台| page =表15 山东省乡、镇、街道人口| format =收费| access-date =2018-07-19| archive-url =https://web.archive.org/web/20180719084356/http://www.yearbookchina.com/navipage-n2013030152000017.html| archive-date =2018-07-19| dead-url =no}}</ref>
