《'''大决战之辽沈战役'''》（{{lang-en|'''Decisive Engagement: The Liaoxi Shenyang Campaign'''}}）1991年上映于[[中国大陆|中国大陆]]的战争片，导演李俊、杨光远、韦廉、景慕逵、翟俊杰，主演古月、苏林、赵恒多、吴志远、孙海英、姚居德。

该片是《[[大决战|大决战]]》系列电影之一，由[[八一电影制片厂|八一电影制片厂]]制作<ref>{{cite web |url=http://news.qq.com/a/20110520/000862.htm |title= 电影《大决战——辽沈战役》|accessdate=2011年5月20日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>，被誉为战争史诗片。<ref>{{cite web |url=http://ent.sina.com.cn/m/2008-12-09/ba2287599.shtml |title=三十年经典电影：《大决战》系列(1991～1992) |accessdate= 2008年12月9日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

==剧情==
1948年春天（民国三十七年），[[中国人民解放军|中国人民解放军]]首脑毛泽东为争夺政权，筹划和[[中华民国总统|中华民国总统]]蒋中正举行决战，9月12日，中国人民解放军发动[[辽沈战役|辽沈战役]]，11月1日，[[林彪|林彪]]领导[[东北野战军|东北野战军]]在[[东北三省|东北三省]]取得胜利。<ref>{{cite web |url=http://news.china.com.cn/rollnews/2012-05/31/content_14461901.htm |title=解放战争时期：《大决战--辽沈战役》 |accessdate=2012-05-31 |publisher=中国网 |work= |language =zh-cn}}</ref>
