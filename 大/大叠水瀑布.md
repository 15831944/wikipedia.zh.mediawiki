'''大叠水瀑布'''也叫'''飞龙瀑'''<ref group="注">按照石林风景名胜区的规划定名为飞龙瀑，现一般仍称大叠水瀑布</ref>，古称'''叠水蒸云'''，是[[云南省|云南省]][[昆明市|昆明市]][[石林彝族自治县|石林彝族自治县]]的一个瀑布，位于[[巴江_(南盘江)|巴江]]下游流出石林县境处，距离县城18千米，距离[[路南石林|石林风景区]]27千米<ref>{{cite web |url=http://www.chinastoneforest.com/showArticle.aspx?cid=25&aid=391 |title=大叠水（暂停游览） |author= |date= |website=www.chinastoneforest.com |publisher=石林旅游网 |access-date=2018-02-28 |quote=}}</ref>，是云南省最大的[[瀑布|瀑布]]<ref>{{cite web |url=http://yn.people.com.cn/news/yunnan/n2/2016/0311/c375169-27916196.html |title=云南省最大瀑布：石林大叠水 |author= |date=2016-03-11 |website=yn.people.com.cn |publisher=[[人民网|人民网]]云南频道 |access-date=2018-02-28 |quote=}}</ref>，号称“珠江第一瀑”<ref name="县志">{{cite book |author=昆明市路南彝族自治县志编纂委员会 |date=1996年11月 |title=《路南彞族自治县志》 |url= |location=[[昆明市|昆明]] |publisher=[[云南民族出版社|云南民族出版社]] |page= |isbn=7-5367-1294-4 |author-link= }}</ref>{{rp|455}}。大叠水瀑布也是[[成龙|成龙]]在电影《[[神话_(电影)|神话]]》中飞身跃入瀑布的取景地<ref>{{cite web |url=http://www.amoytraveler.com/huanqiu/4466.html |title=电影神话瀑布拍摄场地 云南最大瀑布大叠水瀑布 |author= |date=2016-10-26 |website=www.amoytraveler.com |publisher=厦门旅行者网 |access-date=2018-02-28 |quote=}}</ref>。

==水体==

大叠水瀑布的水源由巴江、马料河、[[大可河|大可河]]汇集供给，丰水期流量每秒140立方米，枯水期流量每秒3.2立方米<ref name="县志"></ref>{{rp|455-456}}。[[石林彝族自治县#2010年石林旱灾|2010年石林县的旱灾]]中，上游河道水量骤减，大叠水瀑布接近断流<ref>{{cite web |url=http://news.sina.com.cn/c/p/2010-03-18/164019892280.shtml |title=组图：云南旱情致石林县大叠水瀑布几近断流 |author=李昌莉 |date=2010-03-18 |website=news.sina.com.cn |publisher=[[云南网|云南网]] |access-date=2018-02-28 |quote=}}</ref>。在丰水期，由于上游河水夹带泥沙，大叠水瀑布整体呈土黄色，每年的七八月是观瀑最佳的“回水天”<ref>{{cite web |url=http://travel.yunnan.cn/html/2014-08/22/content_3338421.htm |title=石林大叠水瀑布暴涨 “回水天”震撼登场 |author=张勇 |date=2014-08-22 |website=travel.yunnan.cn |publisher=[[春城晚报|春城晚报]] |access-date=2018-02-28 |quote= |archive-url=https://web.archive.org/web/20180228162638/http://travel.yunnan.cn/html/2014-08/22/content_3338421.htm |archive-date=2018-02-28 |dead-url=yes }}</ref>。

==景区==

大叠水风景区是[[路南石林|石林风景名胜区]]内的瀑布观景区，包括大叠水瀑布、小叠水瀑布、仙人洞、珍珠泉等景点<ref>{{cite book |author1=《石林彝族自治县概况》编写组 |author2=《石林彝族自治县概况》修订本编写组 |title=《石林彝族自治县概况》 |url= |location=[[北京市|北京]] |publisher=[[民族出版社|民族出版社]] |date=2007年 |isbn=7-1050-8540-1 | ref =概况}}</ref>{{rp|176-177}}，景区面积2.87平方千米<ref name="县志"></ref>{{rp|455}}。截止2014年，大叠水瀑布景点的门票为成人每人18元，儿童每人9元<ref>{{cite web |url=http://travel.yunnan.cn/html/2014-08/15/content_3329441.htm |title=石林大叠水瀑布暴涨 游客需谨慎靠近 |author=赵倩 |date=2014-08-15 |website=travel.yunnan.cn |publisher=[[云南网|云南网]] |access-date=2018-02-28 |quote= |archive-url=https://web.archive.org/web/20180228162541/http://travel.yunnan.cn/html/2014-08/15/content_3329441.htm |archive-date=2018-02-28 |dead-url=yes }}</ref>。
