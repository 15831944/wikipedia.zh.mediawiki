'''大狩部站'''（{{jpn|j=大狩部駅|hg=おおかりべえき|rm=Ōkaribe eki|f=y}}）位於[[北海道|北海道]][[新冠郡|新冠郡]][[新冠町|新冠町]]字大狩部，是[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[日高本線|日高本線]]上的站，電報略號是'''オリ'''。

名稱源自[[阿伊努語|阿伊努語]]的「o-punkar-us-pe」，意思為下游處藤蔓叢生的河流<ref>{{Cite web|url=http://www.pref.hokkaido.lg.jp/ks/ass/grp/2130P.pdf|title=アイヌ語地名リスト|accessdate=2017-08-16|author=北海道庁|date=|publisher=北海道庁}}</ref>。

== 車站構造 ==
[[側式月台|側式月台]]1面1線的地面站。沒有站房，只有一座小型的候車室。是[[祕境車站|祕境車站]]。

== 車站周邊 ==
非常地靠近海，路基除了有侵蝕的危險，附近也常有落石發生。
* [[國道235號|國道235號]]
* 海馬岩（{{lang|ja|とどいわ}}）
** 以前是[[北海獅|北海獅]]聚居的岩地，但1955年時[[自衛隊|自衛隊]]進行打靶後，北海獅自此絕跡。
* 御野立所行啓記念碑
* 道南巴士「大節婦」站牌

== 歷史 ==
* 1958年（[[昭和|昭和]]33年）7月15日 - 設立。僅辦理旅客業務。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後成為北海道旅客鐵道（JR北海道）轄下的車站。
