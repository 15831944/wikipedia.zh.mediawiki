'''大韓民國氣象廳'''（{{lang-ko|대한민국 기상청}}、{{lang-en|Korea Meteorological Administration, KMA}}），[[大韓民國|大韓民國]][[大韓民國環境部|環境部]]下属的一個[[行政機關|行政機關]]，負責[[氣象|氣象]]及[[地震|地震]]、[[火山|火山]]有關的觀測、預報和研究等事務。在1990年12月27日，從中央氣象台升格為氣象廳。地址為[[首爾|首爾]][[銅雀區|銅雀區]][[汝矣大方路|汝矣大方路]]16巷61。

== 沿革 ==
=== 前史 ===
==== 近代的氣象觀測 ====
*[[1884年|1884年]][[6月|6月]] - [[大日本帝國|日本]]内務省地理局（中央氣象台前身）委託[[釜山|釜山]]的電信局員進行氣象觀測<ref name="chuo-159">[https://books.google.co.jp/books?id=_RPFL2q90Z4C&pg=frontcover 中央気象台一覧] P.159-160 中央気象台 1932年</ref>。
*[[1886年|1886年]] - [[仁川廣域市|仁川廣域市]]、[[釜山|釜山]]、[[元山|元山]]的海關開始進行氣象觀測<ref name="chuo-159"/>。
*[[1887年|1887年]] - 俄羅斯公使[[韋貝|韋貝]]在[[首爾|京城]]設置觀測儀器<ref name="chuo-159"/>。
*[[1897年|1897年]] - 俄羅斯政府在仁川港小丘上設置觀測所，開始提供氣象信號<ref name="chuo-159"/>。

==== 開展氣象觀測業務 ====
*[[1904年|1904年]][[3月|3月]] - [[日俄戰爭|日俄戰爭]]期間、在[[日本軍|日本軍]]的要求之下、日本[[文部省|文部省]]在[[釜山|釜山]]、[[木浦|木浦]]、[[仁川|仁川]]、[[龍巖浦|龍巖浦]]、[[元山|元山]]等地設置臨時觀測所、日本中央氣象台設立臨時觀測所的建制（明治37年勅令第60號）<ref name="sisatsu"/><ref name="sisei-35th"/><ref name="keijo-2"/><ref name="chuo-159"/>。
*[[1905年|1905年]][[1月|1月]] - 文部省在[[金策市|城津]]設置臨時觀測所<ref name="chuo-159"/>。
*[[1907年|1907年]][[2月|2月]] - [[大韓帝国|大韓帝国]]政府在農商工部建立觀測所的官制，並在[[京城|京城]]、[[平壤|平壤]]、[[大邱|大邱]]等地設置觀測所<ref name="keijo-2"/><ref name="chuo-159"/>。業務的監管實則在[[統監府|統監府]]觀測所<ref name="chuo-159"/>。
*1907年[[3月|3月]] -廢止文部省中央氣象台臨時觀測所的建制<ref name="chuo-159"/>、統監府成立觀測所建制<ref name="keijo-2">[https://books.google.co.jp/books?id=IjBo6tTdfS4C&pg=frontcover 京城府史 第二巻] P.69-70 京城府 1936年</ref>、氣象觀測業務移交統監府<ref name="sisei-35th">[https://books.google.co.jp/books?id=lDyAUdA7leUC&pg=frontcover 施政三十年史] P.82 朝鮮総督府 1940年</ref><ref name="keijo-2"/>。仁川臨時觀測所改為觀測所、其餘臨時觀測所改為分所<ref name="keijo-2"/>。
*[[1908年|1908年]][[4月|4月]] - 統監府觀測所的建制遭廢除、大韓帝國農商工部借用釜山、木浦、仁川、龍岩浦、元山、城津等六個統監府觀測所<ref name="niji-nenpo">[https://books.google.co.jp/books?id=DNG_Ry4lE0EC&pg=frontcover 第二次韓国施政年報] P.110-111 統監府 1908年</ref><ref name="sisatsu">[https://books.google.co.jp/books?id=B-p2N0SlYEwC&pg=frontcover 朝鮮森林視察復命書] P.32  農商務省山林局 1913年</ref><ref name="keijo-2"/>。
*[[1910年|1910年]][[10月1日|10月1日]] - 由於[[日韓合併|日韓合併]]、氣象觀測業務移交朝鮮總督府[[朝鮮總督府|朝鮮總督府]][[遞信局|遞信局]]<ref>[https://books.google.co.jp/books?id=tk0TJh--VK0C&pg=frontcover 朝鮮総督府施政年報: 明治四十三年] P.236 朝鮮総督府 1912年</ref>。
*[[1911年|1911年]][[10月|10月]] - 設置[[江陵市|江陵]]測候所<ref name="chuo-159"/>。
*[[1912年|1912年]][[4月|4月]] - 氣象觀測業務從通信局移交[[朝鮮總督府内務局|内務部]]的[[朝鮮總督府學務局|學務局]]<ref>[https://books.google.co.jp/books?id=HLccjaDR65sC&pg=frontcover 朝鮮総督府施政年報: 明治四十五年・大正元年] P.242-243 朝鮮総督府 1914年</ref>。
*[[1914年|1914年]][[5月|5月]] - 設置[[羅先特別市|雄基]]與[[中江郡|中江鎮]]測候所<ref name="chuo-159"/>。
*[[1916年|1916年]]前後 - 完成全朝鮮的氣象觀測網<ref name="sisei-35th"/>。
*[[1924年|1924年]][[10月|10月]] - 於觀測所設置無線收發裝置<ref name="sisei-25th">[https://books.google.co.jp/books?id=oYjSOIlWgq0C&pg=frontcover 施政二十五年史] P.908 朝鮮総督府 1935年</ref>。與日本内地的[[氣象廳_(日本)|中央氣象台]]以及[[海洋氣象台|海洋氣象台]]、[[沖縄|沖縄]]、[[大連|大連]]、[[樺太|樺太]]的[[大泊町|大泊]]等台連線交換氣象資訊，應用於天氣預報<ref name="sisei-25th"/>。
*[[1931年|1931年]][[1月|1月]] - [[龍巖浦|龍巖浦]]測候所移轉至[[新義州市|新義州]]<ref name="chuo-159"/>。
*[[1933年|1933年]][[8月|8月]] - 觀測所設置短波收發器<ref name="sisei-25th"/>。取得[[中國|中國]]、[[南洋群島|南洋群島]]、[[西伯利亞|西伯利亞]]等地的觀測資料、應用於天氣預報<ref name="sisei-25th"/>。
=== 獨立後 ===
*[[1949年|1949年]][[8月|8月]] - 成立國立中央觀象台。
*[[1963年|1963年]][[2月|2月]] - 國立中央觀象台改名中央觀象台。
*[[1967年|1967年]][[4月|4月]] - 從交通部移轉至[[大韓民國未來創造科學部|科學技術處]]。
*[[1978年|1978年]] - 開始[[地震|地震]]的觀測與記錄。
*[[1982年|1982年]][[1月|1月]] - 中央觀象台改稱中央氣象台，地方觀象台改稱地方氣象台。
*[[1990年|1990年]][[12月|12月]] - 中央氣象台升格為氣象廳。
*[[1992年|1992年]][[3月|3月]] - 地方氣象台改稱地方氣象廳、測候所改稱氣象台。
*[[2008年|2008年]][[2月29日|2月29日]] - 從科學技術部移管至[[大韩民国环境部|環境部]]。
*[[2015年|2015年]][[1月12日|1月12日]] - 設置首都圏氣象廳。
*[[2015年|2015年]][[6月|6月]] - 新設氣象支廳。

== 國立氣象科學院 ==
負責氣象研究業務的氣象廳下屬機關。

== 地方氣象廳 ==
地方氣象廳設在[[水原市|水原]]・[[江陵市|江陵]]・[[大田廣域市|大田]]・[[光州廣域市|光州]]・[[濟州市|濟州]]・[[釜山廣域市|釜山]]等六城市。

地方氣象廳以有人自動氣象觀測設備（ASOS : Automated Synoptic Observing System）記錄[[氣壓|氣壓]]、[[氣溫|氣溫]]、[[風向|風向]]、[[風速|風速]]、[[濕度|濕度]]、[[降水量|降水量]]、[[日射量|日射量]]、[[日照時間|日照時間]]、地面溫度、[[初霜|初霜]]溫度等要素，輔以人工記錄、能見度、雲量、雲形、蒸發量等資訊。積雪量則以自動積雪觀測裝置（[[積雪板|積雪板]]、[[雪尺|雪尺]]）觀測。地方氣象廳提供管轄地區的廣域預報與週間預報。

=== 氣象台 ===
氣象台設有七處、屬其地方氣象廳管轄。氣象觀測所以有人自動氣象觀測設備（ASOS : Automated Synoptic Observing System）記錄氣壓、氣溫、風向、風速、濕度、降水量、日射量、日照時間、地面溫度、初霜溫度等要素，輔以人工記錄、能見度、雲量、雲形、蒸發量等資訊。積雪量則以自動積雪觀測裝置（積雪板、雪尺）觀測。氣象台提供其管轄地區的天氣預報。

== 國家氣象衛星中心 ==
2009年設立、為營運韓國首枚靜止軌道衛星[[千里眼_(衛星)|千里眼]]的氣象廳下屬機關。

== 航空氣象廳 ==
負責航空氣象業務的氣象廳下屬機關。本廳位於[[仁川國際機場|仁川國際機場]]。航空氣象觀測業務多屬自動化、韓國各機場皆設有航空氣象觀測裝置（AMOS），以記錄風向、風速、氣溫、氣壓、降水量等資訊。

== 自動氣象觀測裝置（AWS） ==
自動氣象觀測裝置（Automatic Weather System、AWS）是設置在大韓民國各市・郡・區，共約500個、由氣象廳管理營運的設備。AWS觀測項目包含降水有無・降水量、氣溫、風向・風速等。在氣象機關之間的地帶則多記錄氣壓（2007年開始）、濕度、積雪量計、雲高・雲量計、能見度計（以上2010年開始）。觀測資料以有線或無線網路以分鐘為單位送到氣象廳並應用。AWS的數據可以在網頁上直接查詢<ref>http://www.kma.go.kr/weather/observation/aws_table_popup.jsp</ref>數據產生之技術圖形每五分鐘更新一次<ref>http://www.kma.go.kr/weather/observation/aws_distribution_popup.jsp</ref>。

==天氣ON==
氣象廳為對國民傳達快速正確的氣象資訊所設立的網路氣象新聞播送服務。2007年9月29日開始試營運、2008年7月正式上路。

== 以北五道氣象服務 ==
雖然至今仍然未能實際統治[[三八线|三八线]]以北的[[以北五道|以北五道]]，但大韓民國氣象廳仍然堅持在官網上每天提供[[咸镜北道_(韩国)|咸镜北道]]、[[咸镜南道_(韩国)|咸镜南道]]、[[黄海道_(韩国)|黄海道]]、[[平安北道_(韩国)|平安北道]]和[[平安南道_(韩国)|平安南道]]的氣象信息<ref>http://www.weather.go.kr/weather/forecast/northkorea01.jsp</ref>，以此表明不忘[[朝鮮半島統一問題|國家統一]]。
