'''大寮區'''是[[中華民國|中華民國]][[高雄市|高雄市]]西南方的一個[[市轄區|市轄區]]，北毗[[鳥松區|鳥松區]]、[[大樹區|大樹區]]，西鄰[[鳳山區|鳳山區]]、西南隔[[鳳山丘陵|鳳山丘陵]]接[[小港區|小港區]]，南接[[林園區|林園區]]，東隔[[高屏溪|高屏溪]]鄰[[臺灣省|臺灣省]][[屏東縣|屏東縣]][[屏東市|屏東市]]、[[萬丹鄉|萬丹鄉]]、[[新園鄉_(台灣)|新園鄉]]。目前人口約有11.2萬人，曾是台灣人口最多的鄉，但因商業較不發達、市區分佈較分散，生活機能一般多依賴緊鄰之[[鳳山區|鳳山區]]、[[小港區|小港區]]提供。

2008年9月通車之[[高雄捷運|高雄都會捷運系統]][[高雄捷運橘線|橘線]]，東側終點站[[大寮站|大寮站]]就設置於本區西側，加上本區北端尚有[[台鐵|台鐵]][[後庄車站|後庄車站]]，對外交通堪稱便利。另本區與[[左營區|左營]]、鳳山、岡山並列高雄市[[眷村|眷村]]的集中地。

== 名稱由來 ==
早期[[下淡水溪|下淡水溪]]（今[[高屏溪|高屏溪]]）邊土質肥沃，利於耕種，當地住民遂建置草寮屯墾，分為四處：溪埔寮（今溪寮里）、潮州寮（今潮寮里）、頂大寮（今上寮里）、下大寮（今[[大寮里|大寮里]]），後四寮合一，統名「大寮」，此為地名由來。

[[清代|清代]]屬於[[鳳山縣_(台灣)|鳳山縣]]小竹里。明清年間，大量[[泉州|泉]][[漳州|漳]]移民進入台灣，當時由於[[鳳山丘陵|鳳山丘陵]]位於[[鳳山縣_(臺灣)|鳳山]]南方，形似「蛋卵」，而將其名取為鳳彈（蛋）山，因而得名「鳳彈」，而鳳彈最早的發展見於康熙廿四年《[[台灣府志|台灣府志（蔣志）]]》一書中記載：「一汛鳳彈，把總一員，兵五十名」；而在康熙五十九年[[李丕煜|李丕煜]]所編纂的《[[鳳山縣志|鳳山縣志]]》和《鳳山縣輿圖》中即可看到名為「鳳彈」（忠義里一帶）、「打鹿潭」（山頂里一帶）、「小竹橋」（義和里一帶）的[[汛|汛塘]]（軍備據點），後來人口逐漸發展，便裁撤汛溏，規劃為普通聚落。至[[光緒|光緒]]二十年（1894年），[[廩生|廩生]][[盧德嘉|盧德嘉]]編撰《[[鳳山縣采訪冊|鳳山縣采訪冊]]》時，已出現如山仔頂、打鐵店、考潭寮、翁公園等二十餘個聚落；[[台灣日治時期|日治]]初期屬台南縣鳳山支廳，1920年設[[大寮庄|大寮-{庄}-]]，屬[[高雄州|高雄州]][[鳳山郡|鳳山郡]]；[[民國|民國]]三十五年（1946年）建制[[高雄縣|高雄縣]]大寮鄉。2010年12月25日隨[[五都改制|五都改制]]，改為[[高雄市|高雄市]]'''大寮區'''。

== 雙姓「張簡」 ==
大寮區為[[台灣|台灣]]「[[張簡姓|張簡]]」[[雙姓|雙姓]]的發源地。[[明朝|明朝]][[洪武|洪武]]時，[[南靖|南靖]][[文人|文人]]'''簡德潤'''在[[漳州|漳州]][[南靖|南靖]]的[[書院|書院]]教書，[[入贅|入贅]]於當地[[門閥|門閥]]'''張進興'''家，被'''張進興'''視為[[義子|義子]]，娶其媳婦劉氏為妻（張進興兒子早亡），生了三子。後又娶[[側室|側室]]盧氏，生了五子，遂以其中四名[[過繼|過繼]][[張姓|張姓]]，四名留於[[简姓|簡家]]，後來才出現了'''張簡'''[[雙姓|雙姓]]。[[清朝|清]][[康熙帝|康]][[雍正帝|雍]]年間，簡家第十二代簡成禎來台，在[[台灣南部|台灣南部]]定居，其族脈皆分佈於[[鳳山丘陵|鳳山丘陵]]一帶，[[咸豐|咸豐]]年間在今拷潭、內坑一帶建立了「追遠堂」，為張簡家族之[[宗祠|宗祠]]。

== 農產 ==
大寮區的[[農業|農業]]人口約一萬兩千餘人，[[紅豆|紅豆]]產量為高雄市之冠，佔全市百分之九十；冷凍[[毛豆|毛豆]]全國之冠，此外[[大豆|大豆]]、[[甘藍|甘藍]]亦為大宗。
