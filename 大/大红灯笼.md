[[File:红红的灯笼高高挂_-_panoramio.jpg|thumb]]
'''大红灯笼'''或'''红灯笼'''，是當代中国及[[華人|華人]]在[[节庆|节庆]]期间装饰的[[灯籠|灯籠]]，亦是中国的[[象征|象征]]，在[[中國大陸|中國大陸]]甚至带有某种特定的政治含义<ref>{{Cite web |url=http://www.wenming.cn/syjj/dfcz/hn/201602/t20160205_3138113.shtml |title=信阳：大红灯笼传播核心价值观
|accessdate=2016-03-30 |author=责任编辑：梁艳红|date=2016-02-05 |work= |publisher=中国文明网，来源：信阳文明网  |language=Zh-hans }}</ref>。其外型為扁圓形，有上下輪，下輪垂穗，但其設計造型并不是中国传统固有的灯饰造型<ref name="a1"/>。

在[[香港|香港]]、[[澳門|澳門]]、[[臺灣|臺灣]]等地以及海外華人聚居地，大紅燈籠通常只被視為一般喜慶或襯托[[中式建築|中式建築]]的裝飾，並沒有特定政治含義，常見於[[春節|春節]]。同樣以紅色為喜慶象徵的[[越南|越南]]在新春期間也會懸掛。

==起源==

[[File:Fotothek_df_roe-neg_0006714_006_Mehrere_Personen,_darunter_eine_Fotografierende,_unter_den_Kolonnaden_des_chines.jpg|thumb]]的[[萊比錫|萊比錫]]在前[[東德|東德]]時期的1954年接見到訪的中華人民共和國官員時掛上大紅燈籠]]
大紅燈籠起源于北京[[天安门城楼|天安门城楼]]在1949年[[中華人民共和國|中華人民共和國]][[开国大典|开国大典]]及以后所悬挂的八盏大红灯笼<ref>{{Cite web |url=http://www.zjwh.gov.cn/dtxx/2014-10-01/170862.htm |title=天安门城楼凌晨挂红灯 灯笼图样曾由周总理敲定 |accessdate=2016-03-30 |author=作者：郭谦、武文娟|date=2014-10-01 |work= |publisher=浙江省文化厅网站来源：法制晚报 |language=Zh-hans }}</ref>。灯笼设计者亦有不同说法，一说由日本籍左翼画家[[小野泽亘|小野泽亘]]（中文名肖野）和投奔中共的青年[[森茂|森茂]]设计制作，造型来自于日本的[[丸提灯|丸提灯]]<ref name="a1">{{Cite web |url=http://daxianggonghui.baijia.baidu.com/article/32952 |title=日本灯笼中国挂 ||accessdate=2016-03-30 |author=|date=2015-10-20 |work= |publisher=大象公会|language=Zh-hans }}</ref>。

另一种说法，则是由张仃和钟灵二人设计<ref>{{Cite web |url=http://www.chinawriter.com.cn/bk/2006-11-03/7831.html |title=开国灯笼”是日本人设计的吗？|accessdate=2016-03-30 |author=程世刚|date=2006-11-03 |work= |publisher=中国作家网|language=Zh-hans }}</ref>。

類似結構的燈籠在明代已經出現，而結構及形狀皆類似大紅燈籠的扁圓形燈籠在[[中華民國大陸時期|中華民國大陸時期]]的北京已經出現，只是下輪密封和沒有垂穗。這種燈籠和日本的丸提燈同樣屬於一類稱為[[氣死風燈|氣死風燈]]的傳統燈籠，因此大紅燈籠有可能是傳統中式氣死風燈結合日式丸提燈的部份特徵再加上新設計的產物<ref>{{Cite web |url=http://www.newsancai.com/b5/traditional/4-folk/21749-.html |title=老北京過年瑣憶（圖）|accessdate=2016-04-01 |author=李方|date=2010-02-01 |work= |publisher=新三才|language=Zh-hant }}</ref>。
