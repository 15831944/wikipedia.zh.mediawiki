'''大山車站'''位於[[台灣|台灣]][[苗栗縣|苗栗縣]][[後龍鎮_(台灣)|後龍鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[海岸線_(臺鐵)|海岸線]]的[[鐵路車站|鐵路車站]]，目前為苗栗縣定歷史建築。

== 車站構造 ==
*木造車站。
*跨線[[天橋|天橋]]。
*[[公共廁所|公共廁所]]於站體外。
*[[島式月台|島式月台]]、[[岸式月台|岸式月台]]各一座。

== 利用狀況 ==
*目前為乙種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]，僅有[[臺鐵區間車|區間車]]停靠。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*此站已停止販售[[埃德蒙森式鐵路車票|名片式車票]](庫存已售罄)。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

==  公車資訊 ==
== 歷史 ==
*1922年10月11日：開業，初名'''大山腳'''。
*1965年4月1日：改稱今名。<ref>{{cite book |auhor = 戴震宇 |authorlink =  |itle = 《台灣的老火車站》 |pbisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>唯居民迄今仍沿習舊稱。
*1991年4月1日：由三等站降為甲種簡易站，並指定由[[後龍車站|後龍站]]管理。
*2015年6月30日：啟用多卡通刷卡機。
*2017年9月16日：調整為乙種簡易站。
