'''大碶疏港高速公路'''是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]]一条高速公路，浙江省高速公路网编号S1。全线位于[[北仑区|北仑区]]，连接[[北仑港|北仑港]]二、三期集装箱码头和[[甬台温高速公路|甬台温高速公路]]北仑出口，全长5.042公里，是宁波市第一条全线高架的高速公路。2008年12月16日建成通车。<ref name="main">[http://www.nftz.gov.cn/zfzx/info.aspx?Id=5924 宁波大碶疏港高速公路本月28日通车]</ref>

== 建设目的 ==
大碶疏港高速公路连接北仑港与[[甬台温高速公路|甬台温高速公路]]。在高速公路建设之前，由于港口吞吐量日渐增大，北仑城区的交通压力日渐增大。而大碶疏港高速公路通车之后，使得货车通过北仑城区的时间得到缩短，同时也减轻了城区交通的压力。<ref name="main" />

== 收费 ==
大碶疏港高速公路通行费用目前由甬台温高速公路北仑主线收费站代收，小车收费10元，大车收费20元。<ref>{{Cite web |url=http://www.haishu.gov.cn/zwdt/NewsInfo2.aspx?Id=173451 |title=７月１日起大碶疏港公路单向收费 |access-date=2017-12-14 |archive-url=https://web.archive.org/web/20171214073226/http://www.haishu.gov.cn/zwdt/NewsInfo2.aspx?Id=173451 |archive-date=2017-12-14 |dead-url=yes }}</ref>

==互通枢纽和服务设施列表==
大碶疏港高速公路共设4个互通出入口，1个主线收费站。<ref>[http://zjnews.zjol.com.cn/05zjnews/system/2008/12/12/015072594.shtml 大碶疏港公路月底通车]</ref>
