'''大溪社'''，俗稱大溪神社，為[[臺灣日治時期|臺灣日治時期]]在[[新竹州|新竹州]][[大溪郡|大溪郡]][[大溪街|大溪街]]（今[[桃園市|桃園市]][[大溪區|大溪區]]）大溪公園內的[[神社|神社]]。

== 歷史 ==
大溪社選址於[[新竹州|新竹州]][[大溪郡|大溪郡]][[大溪街|大溪街]]字草店尾193番地（今大溪公園內）興建，於昭和7年（1932年）10月30日鎮座。祭祀[[開拓三神|開拓三神]]（[[大國魂命|大國魂命]]、[[大己貴命|大己貴命]]、[[少彥名命|少彥名命]]）、[[北白川宮能久親王|北白川宮能久親王]]。例祭日10月30日。1939年當時由時任[[近代社格制度|縣社]][[新竹神社|新竹神社]]社司宮島由多加管理<ref>{{Cite web |url =http://hyerm.ntl.edu.tw:2135/cgi-bin/gs32/gsweb.cgi/ccd=57jrKS/brwfull?filename=ENE%60L%5BPITO%87%93%8F%8EHX%60P%5D%95%9EK%9E%86%92uW%60%5Dpc%7EO%60%60PMW%5DMU%5DEX%95I%A1%95%A3&dbid=%88%98%84%85M%5DEEX%9E%A1F%A1%8B%8Fp%5CK%60k%5EyRW%5DU |title = 臺灣圖書館，日治時期圖書影像系統，昭和十四年度之"臺灣ニ於ケル神社及宗敎"中關於大溪神社的記載|author =臺灣總督府文教局社會課|publisher =臺灣總督府文教局社會課|language =ja }}</ref>。

戰後神社廢止，並陸續拆除原有建物。民國54年（1965年）在拜殿原址上建立涼亭-超然亭<ref>{{Cite web |url = http://blog.xuite.net/apex.cheng/wretch/117809554-%E7%A5%9E%E7%A4%BE%E9%81%BA%E8%B7%A1(%E5%85%AB)++%E5%A4%A7%E6%BA%AA%E7%A5%9E%E7%A4%BE |title = 時空旅人，神社遺跡（八）大溪神社|date =2010-02-16 |language=zh-TW }}</ref>。後整修原參道並新設5對石燈籠，也將原有石燈籠構件集中安置於參道旁至今。

== [[神社建築|神社建築]] ==
大溪社原配置有參道、社號標、下乘札、[[鳥居|神明鳥居]]2基（其中一之鳥居和石狛犬皆由大溪菸草小賣人組合長加藤仁作出資奉納<ref>{{Cite book | author = 金子展也 | title = 台湾旧神社故地への旅案内―台湾を護った神々 | location = 東京 | publisher = 神社新報社 | date = 2015-09-01 | pages =55頁第3段、68頁注20 | ISBN = 978-4-908128-02-8 | language = ja }}</ref>）、[[切妻造|切妻造]]手水舍、石燈籠4對、石狛犬1對、[[神明造|神明造]]拜殿、玉垣與駁坎、[[神明造|神明造]]本殿<ref>{{Cite web |url =http://hyerm.ntl.edu.tw:2135/cgi-bin/gs32/gsweb.cgi/ccd=ngKNJx/brwfull?filename=EFR%5BbJMRIFcDRI%89%99%85%84R%5EGSJ%8C%87P%97%8D%89%7ENYNEp%5DNH%7FLaXDY%94%8BYcZ%97G%A3%97%A1&dbid=%88%98%84%85M%5DDRK%8F%90Q%90%8C%88%7FMZOFqZOK%80M%5EYS |title =臺灣圖書館，日治時期圖書影像系統，"[大溪街]街勢一覽"所藏大溪社舊照 |author = 大溪街役場|publisher =大溪街役場|language =ja }}</ref>。現存神社主體部分基座、石狛犬和燈籠構件若干。
