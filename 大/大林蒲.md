'''大林蒲'''是位於[[臺灣|臺灣]][[高雄市|高雄市]][[小港區|小港區]]的一個濱海聚落，西南濱[[台灣海峽|台灣海峽]]，東邊則是緊鄰[[臨海工業區|臨海工業區]]，北邊是目前已經遷村的[[紅毛港_(高雄市)|紅毛港]]，南邊是[[邦坑|邦坑]]、[[鳳鼻頭|鳳鼻頭]]。大林蒲境內或鄰近的大型工廠有[[中油大林煉油廠|中油大林煉油廠]]、[[台電大林發電廠|台電大林發電廠]]及[[中鋼公司|中鋼公司]]等<ref name="大紀元 www.epochtimes.com 2014 王淑芬"/>。

== 歷史沿革 ==
大林蒲的開基者，於永曆十五年（1661年）隨[[延平郡王|延平郡王]][[鄭成功|鄭成功]]驅逐荷蘭人，自[[福建|福建]][[漳州府|漳州府]]遷移來台。由西海岸茄萣邊登陸，屯墾定居。因此地野林草埔茂盛，故取名為「大樹林」、「大林埔」。當移民砍林闢地墾成良田時，林地存於農地之間，由於林中樹木種類繁多，乃統稱「林」為地名；蒲當「埔」用，也即草木叢生的野地，最後終於成為其地名<ref name="diming">{{Cite web |url= http://placesearch.moi.gov.tw/search/place_list.php?id=22465 |title= 地名詳細資料 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}
</ref><ref>{{Cite web |url= http://163.32.161.16:81/lifetype/index.php?op=ViewArticle&articleId=347&blogId=207 |title= 地名探源--大林蒲 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

道光十七年(1837年)，鳳山知縣曹謹為了要開[[曹公圳|曹公圳]]，於[[鳳鼻頭|鳳鼻頭]]山麓，鑿建大型抽水機<ref>{{Cite web|url=http://cgcanal.cca.gov.tw/chinese/html/o&n%20map/a/o&n%20map-1.html|title=古今地圖看曹公圳|publisher=文建會|deadurl=yes|archiveurl=https://web.archive.org/web/20101127164515/http://cgcanal.cca.gov.tw/chinese/html/o%26n%20map/a/o%26n%20map-1.html|archivedate=2010-11-27}}</ref>，中林仔及追港仔一帶居民被迷信所祟，認定龍脈損害，朝夕不保，乃相繼遷入大樹林定居，新居住地就以具有驅邪的香草－[[菖蒲|菖蒲]]的「蒲」字，取代原來的「埔」字，從此易名為大林蒲<ref>謝孟宓. 高雄市小港區大林蒲閩南話研究.高雄師範大學台灣文化與語言研究所碩士論文，2007年.</ref>。

大林蒲於清朝時以「頂沖下蒲」聞名。「沖」指右沖，即今[[右昌|右昌]]；「蒲」指大林蒲，此兩地都以「飛脊厝」齊名。屋脊一如廟宇左右齊飛，是科舉功名的標幟。在「萬般皆下品、唯有讀書高」的傳統時代，據傳大林蒲在[[嘉慶|嘉慶]]年間，共出[[吳鳳鳴|吳鳳鳴]]與[[陳媽得|陳媽得]]等兩位進士，以及一位秀才[[邱文|邱文]]<ref>高雄市里政資訊網：大林蒲，{{cite web |url=http://vc.ikh.tw/khk37/?pn=tview&ID=0vdwnou34355 |title=存档副本 |accessdate=2014-07-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20140714151305/http://vc.ikh.tw/khk37/?pn=tview&ID=0vdwnou34355 |archivedate=2014-07-14 }}</ref>。然而，經查閱清代台灣進士名錄之後，嘉慶年間台灣並無進士且兩人並未名列於進士榜中<ref>清代台灣進士表，http://distance.shu.edu.tw/taiwan/ch11/CH11_SEC04.HTM</ref>。因此，所謂的「進士」之說，其實僅是「歲貢」，由於民間對「科名」不甚了解，故誤以為歲貢就是進士<ref>黃福鎮. 2000,《大高雄風土誌》。高雄：復文圖書出版社。楊倉亭. 2006 〈「小港區的由來」演講活動紀實〉，《高市文獻》，第19卷，第4期，頁159―164。</ref>。

1933年時，大林蒲人口有2454人<ref>{{cite book|title=昭和八年十二月三十一日 臺灣現住人口統計|year=1934|publisher=臺灣總督官房調查課}}</ref>。

== 地方廟宇宗教文化 ==
[[File:大林蒲鳳林宮指示牌.jpg|200px]]
如同台灣各地的廟宇宗教文化，[[大林蒲鳳林宮|大林蒲鳳林宮]]是大林蒲最大的廟宇，也是大林蒲的文化活動中心。

鳳林宮的廟史可追溯自康熙三十六年（1697年），主要奉祀溫府、朱府和池府千歲。民國六十三年（1974年）時，政府開發[[臨海工業區|臨海工業區]]，鳳林宮舊址的部分土地被徵收，且信徒鑒於廟宇建築陳舊不堪，遂發起募捐重建，於民國六十七年（1978年）建成至今<ref>{{Cite web |url= http://tw.myblog.yahoo.com/fonglin2925/ |title= 鳳林宮部落格 }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

鳳林宮位於二樓，一樓為廣場，多數當地居民經常在此休憩聊天。廟口廣場有一個小戲台，且有許多攤販在此擺攤。除了每天的早市以外，每周四於廟口廣場也有夜市。可說是大林蒲經濟活動最為熱絡的地方。

== 遷村議題 ==
{{see also|臺灣空氣污染}}
{{cquote|市區1條丙烯管線就被炸成這樣，「大林蒲地下有百支管線」，民眾的驚恐難以言喻。|鳳林里長 許再生  2014/08/17}}

繼鄰近的[[紅毛港_(高雄市)|紅毛港]]遷村完成後，高雄市政府於2011年與2016年辦理遷村意願的民意調查，有七成以上居民同意遷村<ref>{{Cite web|url=http://dalinpu.kcg.gov.tw/home02.aspx?ID=$1001&IDK=2&EXEC=D&DATA=2066|title=大林蒲遷村計劃緣由|accessdate=2017-05-02|author=|date=2017-02-12|publisher=|archive-url=https://web.archive.org/web/20171205042035/http://dalinpu.kcg.gov.tw/home02.aspx?ID=$1001&IDK=2&EXEC=D&DATA=2066|archive-date=2017-12-05|dead-url=yes}}</ref>。

2015年及其過去數年來因為[[南星計畫|南星計畫]]新的遊艇產業園區的環保議題<ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/1284916|title=「高市南星計畫遊艇產業園區」 說明會遭數百地方民眾杯葛！ - 生活 - 自由時報電子報|accessdate=2017-05-02}}</ref>，再加上大林蒲通往高雄市區的交通要道中林路的坍塌事件<ref>{{Cite news|url=http://www.chinatimes.com/realtimenews/20151027004572-260405|title=中林路崩塌需繞道 民眾仍不安心|last=中時電子報|newspaper=中時電子報|accessdate=2017-05-02|language=zh-TW}}</ref><ref>{{Cite news|url=http://www.epochtimes.com/b5/16/9/19/n8314280.htm|title=高雄中林路崩塌不斷 立委要求經濟部清查 - 大紀元|date=2016-09-19|newspaper=大紀元 www.epochtimes.com|accessdate=2017-05-02|language=zh-TW}}</ref>及[[2014年臺灣高雄氣爆事故|高雄氣爆]]，使大林蒲遷村議題持續發酵。

[[馬英九政府|馬英九政府]]估計遷村費用共需要700億元並要求高雄市政府盡快辦理<ref name=" 唐佩君 大紀元 www.epochtimes.com 2014">{{cite web | title=楊秋興支持石化專區 解決遷村 - 大紀元 | website=大紀元 www.epochtimes.com | date=2014-08-20 | url=http://www.epochtimes.com/b5/14/8/20/n4229217.htm | language=zh | access-date=2017-11-14| author=唐佩君| quote=[[行政院|行政院]][[政務委員|政務委員]][[楊秋興|楊秋興]]表示，高雄氣爆後管線不回填，但台灣不能沒有石化產業，中央正規劃高雄石化專區，但同時高雄市政府應解決專區周遭居民遷村事宜。楊秋興同時也是[[國民黨|國民黨]]高雄市長參選人，今天接受[[飛碟電台|飛碟電台]]主持人[[唐湘龍|唐湘龍]]訪問，針對高雄氣爆後如何兼顧安全及石化產業發展，提出上述看法。}}</ref>，然而高雄市政府無力獨自負擔高額的遷村相關費用，在遷村計畫前途未卜的情形下，遲遲無法開展初步的遷村普查<ref name="大紀元 www.epochtimes.com 2014 王淑芬">{{cite web | author=王淑芬 | title=設石化專區 大林蒲居民促遷村 - 大紀元 | website=大紀元 www.epochtimes.com | date=2014-08-17 | url=http://www.epochtimes.com/b5/14/8/17/n4226963.htm | language=zh | access-date=2017-11-14}}</ref>。



2016年5月，[[2016年中華民國總統選舉|政黨輪替]]後，該計畫重新獲得中央政府的支持。11月19日時，行政院長林全率領高雄市長、中油、中鋼、台電董事長以及台灣港務公司總經理前來大林蒲，和當地居民座談，也為當地多年承受的污染，向他們鞠躬致歉，從大林蒲討論遷村議題起，林全是第一位到大林蒲關注的行政院長<ref name="自由時報電子報 2016 王榮祥、洪臣宏">{{cite web | author=王榮祥、洪臣宏| title=大林蒲污染數十年 林全傾聽…道歉 - 焦點 | website=自由時報電子報 | date=2016-11-20 | url=http://news.ltn.com.tw/news/focus/paper/1053726 | language=zh | access-date=2017-11-13}}</ref>。

獲得中央力挺的高雄市政府於2017年<ref name="聯合新聞網 2017 謝梅芬">{{cite web | author=謝梅芬 | title=大林蒲遷村普查一坪換一坪 獲林全院長支持 - 高屏離島 - 地方 | website=聯合新聞網 | date=2017-08-04 | url=https://udn.com/news/story/7327/2623257 | language=zh | access-date=2017-11-13}}</ref>，在大林蒲和[[鳳鼻頭|鳳鼻頭]]舉辦多場遷村普查說明會<ref>{{Cite news|url=https://udn.com/news/story/3/2331062|title=看嘸大林蒲遷村普查內容 高市辦6說明會|author=|last=聯合新聞網|date=|work=|newspaper=即時|accessdate=2017-05-02}}</ref>，並開始進行遷村普查。
