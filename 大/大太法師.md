'''大太法師'''（{{lang-ja|ダイダラボッチ}}）為[[日本|日本]]（特別是[[關東地方|關東地方]]和[[中部地方|中部地方]]）傳說中的[[巨人|巨人]]，也有人說他是踏鞴製鐵<ref>踏鞴就是日本冶鐵法裡腳踩的大型風箱，關於該製鐵法詳情請參看{{ja}}[http://www2.kct.ne.jp/~monohito/tatara.html もののけ姫と「たたら」] {{Wayback|url=http://www2.kct.ne.jp/~monohito/tatara.html |date=20090613000144 }}內文。</ref>的鍛冶神，別名有'''大太郎法師'''、'''踏鞴法師'''等。

== 概要 ==
民俗學家[[柳田國男|柳田國男]]到日本各地考察、收集關於大太法師的傳說，並於1927年由[[中央公論新社|中央公論社]]出版《ダイダラ坊の足跡》。該書提到「大太」具有「[[巨人|巨人]]」的意義，再加上「[[法師|法師]]」，乃是與[[一寸法師|一寸法師]]相對立的人物。各地對於大太法師的發音也不盡相同，計有「でいだらぼっち」、「だいらんぼう」、「だいだらぼう」、「でいらんぼう」、「だいらぼう」、「デエダラボッチ」、「デイラボッチ」、「デイラボッチャ」、「デーラボッチャ」、「デエラボッチ」、「デーラボッチ」等。

== 文獻記載 ==
=== 常陸國風土記之記載 ===
《常陸國風土記》記錄了關於{{link-ja|大串貝塚|大串貝塚}}的傳說，此[[貝塚|貝塚]]位於今[[茨城縣|茨城縣]][[水戶市|水戶市]]鹽崎町，自[[繩文時代|繩文時代]]前期形成的遺跡。
{{Quotation|平津驛家西一二里有岡，名曰大櫛。上古有人，體極長大，身居丘壟之上，手摎海濱之蜃<small>大蛤也</small>。其所食貝，積聚成岡，時人取大朽之義，今謂大櫛之岡，其踐跡<small>長卌余步，廣廿余步，尿穴徑可廿余步計。|《常陸國風土記》{{link-ja|鹿島郡 (茨城縣)|鹿島郡 (茨城県)|香島郡}}／[[那珂郡|那賀郡]]</small><ref>參看[http://miko.org/~uraki/kuon/furu/text/fuudo/hitachi/hitachi03.htm 衣袖漬常陸國風土記 香島郡／那賀郡]。</ref>}}

=== 播磨國風土記之記載 ===
《播磨國風土記》託賀郡（今[[兵庫縣|兵庫縣]][[多可郡|多可郡]]）條記錄了與天同高、足跡成[[沼澤|沼澤]]之巨人，跟前一段《常陸國風土記》的記載近似。

{{Quotation|一、託賀略記：託賀郡右<small>此當為上。</small>，所以名託賀者。昔在大人，常勾行也，自南海到北海，自東巡行之時，到來此土云。他土卑者，常勾伏而行之。此土高者，申而行之，高哉！故曰：「託賀郡」。託賀<small>此云たし，高也；申，伸也。</small>，其踰跡處，數數成沼。|《播磨國風土記》[[多可郡|託賀郡]]<ref>請見[http://miko.org/~uraki/kuon/furu/text/fuudo/harima/harima04.htm 衣袖漬常陸國風土記 託賀郡／賀毛郡／美囊郡]。</ref>}}

== 各地流傳之傳說 ==
=== 填土造山 ===
# 傳說大太法師挖掘[[甲斐國|甲州]]之土造出[[富士山|富士山]]，是故現在的甲州變成了[[盆地|盆地]]<ref>見《日本の謎と不思議大全 東日本編》，安部晃司他著，人文社編集部編，2006年，ISBN 978-4-7959-1986-0，頁122。</ref>。
# [[上野國|上州]]當地流傳榛名富士山是大太法師造山而成，至於被他挖掘之地則成了{{link-ja|榛名湖|榛名湖}}（位在[[群馬縣|群馬縣]][[高崎市|高崎市]][[榛名山|榛名山]]山頂）。因為榛名富士山比富士山還低，本來大太法師還打算再補上一些砂土，因遇上夜晚而作罷。
# [[淺間山|淺間山]]嫉妒富士山身為[[妹妹|妹妹]]，個頭卻比她高，向後者要求分一些土給她。富士山聽了之後照做，但淺間山卻因土量不足而發怒，噴發的泥土形成前掛山（淺間山之[[火山口|火山口]]）。
# 關東地方的名山素有「西富士、東筑波」之美稱，假若將兩座山放在[[天平|天平]]上秤重，[[筑波山|筑波山]]比富士山還要輕。所以據說筑波山摔落地面的結果，跌成西男體山（[[標高|標高]]871[[公尺|公尺]]）、東女體山（標高877公尺）兩座山峰。

=== 手、腳造成的殘跡 ===
# 傳說[[上野國|上州]]（今[[群馬縣|群馬縣]]）的[[赤城山|赤城山]]山麓的水窪乃是巨人所造成。
# [[長野縣|長野縣]][[大町市|大町市]]北部的{{link-ja|青木湖|青木湖}}、{{link-ja|中綱湖|中綱湖}}、{{link-ja|木崎湖|木崎湖}}等（合稱{{link-ja|仁科三湖|仁科三湖}}）傳說亦由大太法師踩踏而成。
# [[茨城縣|茨城縣]][[水戶市|水戶市]]中央有一座相當大的{{link-ja|千波湖|千波湖}}，據說是大太法師的足跡所造成。
# 傳說[[遠江國|遠州]]深山裡住著巨人，某天他吃飯時咬到一顆小石子，吐出來後便形成今日[[濱名湖|濱名湖]]裡的礫島（{{ja}}つぶてじま）<ref>請參照{{ja}}[http://www.hamanakotabi.net/perfectguide/pg007.html 観光情報浜名湖パーフェクトガイド浜名湖について / 浜名湖たびNET]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}，「浜名湖にまつわる伝説」之段落。</ref>。
# [[東京都|東京都]][[世田谷區|世田谷區]]裡的「代田」（{{ja}}だいた）以及[[埼玉縣|埼玉縣]][[埼玉市|埼玉市]]裡的「太田窪」（{{ja}}だいたくぼ），這兩個地名皆與巨人的足跡有所淵源<ref>「大太」的日文發音同「代田」和「太田」。</ref>。
# [[靜岡縣|靜岡縣]]境內有座名為「ダイラボウ山」，山頂有一全長150[[公尺|公尺]]之窪地，傳說是巨人從[[琵琶湖|琵琶湖]]運送泥土至富士山時，左腳踩踏所造成。
# [[神奈川縣|神奈川縣]][[相模原市|相模原市]]也傳說巨人舉起富士山打算運至其他地方，因疲累暫時放下，並坐在富士山上歇息，想不到這座山已生根固定在地面上。
# [[香川縣|香川縣]]中部的[[飯野山|飯野山]]傳說有巨人[[尿|小便]]的遺跡，其尿液匯成了今日的{{link-ja|大束川|大束川}}。
# [[愛知縣|愛知縣]][[東海市_(日本)|東海市]]加木屋町裡有個名為「陀陀法師（{{ja}}だだほうし）」的小村落，據說也跟大太法師有關。不過2000年時因[[名古屋鐵道|名古屋鐵道]]{{link-ja|八幡新田站|八幡新田駅}}西側拓寬工程的關係，該遺跡已經埋沒消失。

=== 休息、洗澡、吃飯 ===
# 傳說巨人曾在流經[[榛名山|榛名山]]山麓的[[利根川|利根川]]洗腳。
# [[人|人類]]尚未出現的太古時代，傳說巨人曾在今[[栃木縣|栃木縣]]羽黑山山腰的[[鬼怒川|鬼怒川]]河床洗腳。
# 據說[[長野縣|長野縣]][[鹽尻市|鹽尻市]]的[[八岳中信高原國定公園|高法師高原]]是大太法師曾坐下來稍作休憩的地方。
# 如同前文所敘《常陸國風土記》的記錄，位於今[[茨城縣|茨城縣]][[水戶市|水戶市]]鹽崎町的{{link-ja|大串貝塚|大串貝塚}}傳說是巨人吃完東西後，[[貝殼|貝殼]]聚集成山丘所形成。

=== 幫助人類 ===
據說[[秋田縣|秋田縣]]的{{link-ja|橫手盆地|横手盆地}}進行洩湖水開墾之際，大太法師從水中現身，並幫忙掏清淤泥等工事，後來化身成座落在[[秋田市|秋田市]]的太平山三吉神社。位於秋田縣太平山山麓的太平地區現在雖然讀作「{{ja}}たいへい」，但[[明治|明治]]時期以前讀成「{{ja}}おいだら」，據說即是跟大太法師有關。
