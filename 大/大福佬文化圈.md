'''大福佬文化圈'''（{{lang-nan|Tōa Hok-ló Bûn-hòa-khoan}}），由[[台灣|台灣]]的歷史學者[[翁佳音|翁佳音]]所提出，主張將全世界使用[[泉漳片|泉漳語]]或[[潮州語|潮州語]]的族群視為一體並成為獨立的研究範疇，即[[閩南文化|閩南文化]]概念之延伸與拓展。地理上涵蓋了[[閩南|閩南]]、[[粤東|粤東]]（包含[[潮汕|潮汕]]）、[[臺灣|臺灣]]、[[東南亞|東南亞]]等，國籍與族群上則包含[[閩南人|閩南人]]（泉漳人、潮州人）、[[閩南裔臺灣人|閩南裔臺灣人]]以及[[海外華人|東南亞華人]]等。自16世紀開始，閩南漢族大量向海外[[移民|移民]]，但遷徙各地的閩南族裔在[[歷史|歷史]]、[[語言|語言]]與[[文化|文化]]等方面仍保持緊密的交流與聯繫。這項概念的提出，受到20世紀的[[荷蘭|荷蘭]]學者{{le|彼得·蓋|Pieter Geyl}}的核心史觀「[[大荷蘭|大荷蘭文化圈]]」的影響，但不同的是，大福佬並沒有如大荷蘭般的[[民族統一主義|民族統一主義]]等相關論述，僅作為[[文化圈|文化圈]]的論述。<ref>{{cite web |url=http://www.ith.sinica.edu.tw/quarterly_download.php?name=fulltext_13_2_1.pdf&filename=126717022712.pdf |title=「牽手khan-chhiu」來看臺灣世界史──從臺灣歷史慣用語論大福佬文化圈概念 |accessdate= |last= |first= |author=翁佳音 |date=2006年12月 |work= |publisher=臺灣史研究 }}</ref><ref>{{cite web |url=http://140.112.142.79/publish/pdfs/49/49_06.pdf |title=評介洪麗完著，《熟番社會網絡與集體意識——臺灣中部平埔族群歷史變遷(1700-1900)》 |accessdate= |last= |first= |author=林文正 |date=2012年 |work= |publisher=臺大歷史學報 }}</ref><ref>{{cite web |url= |title=《廈荷詞典》、《荷華文語類參》所見ê十九世紀福佬語行業文化語詞 |accessdate= |last= |first= |author=黃龍泉 |date=2013年 |work= |publisher=台語研究 }}</ref>
