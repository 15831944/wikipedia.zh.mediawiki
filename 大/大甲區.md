== 地理 ==
===位置與範圍===
本區形狀類似丁字型，東西最大跨距約10公里；南北最大跨距8公里，總面積約58平方公里。位置在行政區劃上位於[[台中市|台中市]]西北端，北邊與[[苗栗縣|苗栗縣]][[苑裡鎮|苑裡鎮]]以南房裡溪與新復溝為界；南邊與[[清水區_(台灣)|清水區]]相隔[[大甲溪|大甲溪]]，西側與[[大安區_(臺中市)|大安區]]及[[臺灣海峽|臺灣海峽]]相鄰，東面則與[[外埔區|外埔區]]相接。在地形上位於[[苗栗丘陵|苗栗丘陵]]南端；[[清水平原|清水平原]]北面；[[后里台地|后里台地]]的西側<ref name="大甲地理位置"/><ref name="臺中市大甲區行政區域圖">{{Cite web|url =http://taiwanarmap.moi.gov.tw/moi/GetImage.aspx?city=%25E8%2587%25BA%25E4%25B8%25AD%25E5%25B8%2582&town=%25E5%25A4%25A7%25E7%2594%25B2%25E5%258D%2580&cg=new|title =《臺中市大甲區行政區域圖》|author =[[中華民國內政部|內政部]][[內政部地政司|地政司]]|date =2014|publisher =[[中華民國內政部|內政部]][[內政部地政司|地政司]]|language =zh-tw|accessdate =2016-10-28|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20160304075858/http://taiwanarmap.moi.gov.tw/moi/GetImage.aspx?city=%25E8%2587%25BA%25E4%25B8%25AD%25E5%25B8%2582&town=%25E5%25A4%25A7%25E7%2594%25B2%25E5%258D%2580&cg=new|archivedate =2016-03-04}}</ref>。
===地形與水文===
本區地形主要以[[臺灣地理|大甲扇狀平原]]以及[[后里台地|后里台地]]所構成。大甲扇狀平原為[[大甲溪|大甲溪]]與[[大安溪|大安溪]]因[[堆積作用|堆積作用]]而形成之[[沖積扇平原|沖積扇平原]]。后里台地則是介於大甲溪與大安溪下游間一長方形的緩傾台地。台地東西跨距長約12公里長，靠近大甲的台地西側地形陡降，但西北與西南角突出升高形成[[丘陵|丘陵]]地形，即為大安溪南岸標高236公尺的[[鐵砧山|鐵砧山]]；及在大甲溪北岸標高179公尺的水尾山。本區水文除[[大甲溪|大甲溪]]與[[大安溪|大安溪]]，區內尚有瓦瑤溪、[[房裡溪|房裡溪]]、溫寮溪等溪河以及頂店圳、虎眼圳、日南圳、九張犁圳等水圳，由東向西注入台灣海峽<ref name="大甲地形地貌地質"/>。
===地質與土壤===
本區地質主要含有兩種[[地層|地層]]，大部分為地質年代[[全新世|全新世]]之[[沖積層|沖積層]]，其土壤為沖積土，由礫石、沙、粉沙及黏土等成分組成；其次為后里台地西側，地質年代[[更新世|更新世]]之{{le|階地堆積層|Terrace (geology)}}，其土壤為紅棕壤，內含紅土、礫石、沙及粉沙等成分<ref>{{Cite book| author =陳明終| title =《大甲風貌》| location =臺中大甲| publisher =大甲風貌編輯委員會| date =1981-08| pages =| ISBN =| accessdate =2016-11-02| url =http://old.dajia.taichung.gov.tw/book01/book01.html| language =zh-tw| quote =| deadurl =yes| archiveurl =https://web.archive.org/web/20161104014842/http://old.dajia.taichung.gov.tw/book01/book01.html| archivedate =2016-11-04}}</ref><ref>{{Cite web|url =http://gis.moeacgs.gov.tw/gwh/gsb97-1/sys8/index.cfm|title =地質資料整合查詢|author =[[經濟部中央地質調查所|經濟部中央地質調查所]]|date =|publisher =[[經濟部中央地質調查所|經濟部中央地質調查所]]|language =zh-tw|accessdate =2016-11-02|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20161004101145/http://gis.moeacgs.gov.tw/gwh/gsb97-1/sys8/index.cfm|archivedate =2016-10-04}}</ref>。
===氣候===
本區之氣候為[[亞熱帶季風氣候|亞熱帶季風氣候]]，夏季濕熱多雨，冬季乾旱多風。全年平均氣溫約攝氏24度，年降雨量約1500毫米，雨季落在5~8月間的[[梅雨|梅雨]]、[[颱風|颱風]]與夏季[[西南季風|西南季風]]盛行期間<ref name="大甲氣候"/>。

=== 行政區劃 ===
本區所轄之行政區主要可分為[[大安溪|大安溪]]以北之『溪北區域』與[[大安溪|大安溪]]以南之『溪南區域』。溪北區域為為台中市唯一在[[大安溪|大安溪]]以北的行政區。溪南區域為本區政經中心，不僅為本區主要人口活動區域，亦是相鄰之[[大安區_(臺中市)|大安區]]、[[外埔區|外埔區]]等鄰近之鄉鎮之生活中心<ref name="大甲地理位置"/>。本區部分下轄行政區與道路名稱，早期之戶政人員以[[封神榜|封神榜]]為藍本進行命名，如以[[周王朝|周王朝]]發祥地[[西岐|西岐]]命名之西岐里、代表[[太白金星|太白星]]、[[文曲星|文曲星]]、[[武曲星|武曲星]]之太白里、文曲里、武曲里等<ref>{{cite news |language = zh-tw |author = 張軒哲 |coauthors =  |url = http://news.ltn.com.tw/news/local/paper/540721|title = 〈中部〉大甲里名路名 戲說封神榜 |publisher = 自由電子報  |pages =  |date = 2011-11-21 |accessdate = 2016-10-29}}</ref><ref>{{cite news |language = zh-tw |author = 陳榮昌 |coauthors = |url = http://history.n.yam.com/taiwanhot/place/20150318/20150318922974.html|title = 大甲多里及路有封神名　區公所規畫封神大甲爭觀光 |publisher = 台灣好新聞 |pages =  |date = 2015-03-18 |accessdate = 2016-10-29}}</ref>。

===警政消防===
*警政：本區為[[臺中市政府警察局|臺中市政府警察局]]大甲分局之駐在地，區內設有大甲派出所、日南派出所、西岐派出所與大甲交通分隊。
*消防：本區為[[臺中市政府消防局|臺中市政府消防局]]第五大隊之駐在地，區內設有大甲分隊與幼獅分隊。

====省道與縣市道====
{{Jct|country=TWN|PH|1}}（[[縱貫公路|縱貫公路]]）：縱貫本區南北向並連結溪南區域與大甲溪北區域之主要道路，北由[[苑裡鎮|苑裡鎮]]新復溝起進入本區，向南於大甲溪橋進入[[清水區_(臺灣)|清水區]]，沿途經過大甲幼獅工業區、大安溪橋、大甲體育場、[[光田綜合醫院|光田綜合醫院]]大甲院區等。

{{Jct|country=TWN|PH|61}}（[[西部濱海快速公路|西部濱海快速公路]]）：縱貫本區西北角，由南房裡溪自[[苑裡鎮|苑裡鎮]]進入本區，途經銅安里、西岐里、建興里後，進入[[大安區_(臺中市)|大安區]]。

{{Jct|country=TWN|CH|121}}：溪北區域重要道路，日南地區與[[福爾摩沙高速公路|國道三號]][[苑裡交流道|苑裡交流道]]之主要聯絡道路，北可進入[[苑裡鎮|苑裡鎮]]與[[通霄鎮|通霄鎮]]。

{{Jct|country=TWN|CH|132}}：溪南區域重要道路，自西由[[外埔區|外埔區]]橫貫大甲市區進入[[大安區_(臺中市)|大安區]]，沿途經過[[福爾摩沙高速公路|國道三號]][[大甲交流道|大甲交流道]]、[[大甲高中|大甲高中]]、[[大甲車站|大甲車站]]、臺中市政府警察局大甲分局、[[大甲文昌祠|大甲文昌祠]]、大甲體育場與順天國中。

{{Jct|country=TWN|CH|140}}：該道路全線未入本區，但其為溪北聯絡[[福爾摩沙高速公路|國道三號]][[苑裡交流道|苑裡交流道]]之重要道路。

=== 史前文化與遺址<ref>{{Cite web |url = http://digimuse.nmns.edu.tw/da/collections/ha/a0/intro/ |title = 臺灣史前文化 臺灣中部考古學概述 |author = [[國立自然科學博物館|國立自然科學博物館]]、數位典藏國家型科技計畫 |date =   |publisher = [[國立自然科學博物館|國立自然科學博物館]]、數位典藏國家型科技計畫  |language = zh-tw |accessdate = 2016-10-02 |quote =  }}</ref><ref>{{Cite book| author =廖瑞銘| title =《大甲鎮志》第三篇 開墾篇| location =臺中大甲| publisher =臺中縣大甲鎮公所| date =2009-01| pages =| ISBN =| accessdate =2016-10-30| url =http://old.dajia.taichung.gov.tw/book14/the-history/0277-0460.pdf| language =zh-tw| quote =| deadurl =yes| archiveurl =https://web.archive.org/web/20160402160311/http://old.dajia.taichung.gov.tw/book14/the-history/0277-0460.pdf| archivedate =2016-04-02}}</ref> ===
*[[番仔園文化|番仔園文化]]與番仔園遺址：番仔園文化為距今2000~400年前鐵器時代的史前文化，其相關遺址遍布[[大肚台地|大肚台地]]與[[苗栗丘陵|苗栗丘陵]]南部，以本區之番仔園遺址作為[[命名遺址|命名遺址]]，番仔園遺址位於[[鐵砧山|鐵砧山]]西側山麓紫竹寺與[[大甲高工|大甲高工]]附近。
*鐵砧山平頂遺址：距今4000~3000年前，屬於新石器時代[[牛罵頭文化|牛罵頭文化]]的遺址，位於[[鐵砧山|鐵砧山]]忠烈祠後約100公尺。
*水源地遺址：距今4000~3000年前，屬於新石器時代[[牛罵頭文化|牛罵頭文化]]的遺址，位於本區中山里水尾溪北側斷崖上。
*麻路頭遺址：距今3500~2000年前，屬於新石器時代後期的[[營埔文化|營埔文化]]，本區東南方，水尾溪南岸，距溪水不到100公尺。
