Japan的數據資料]</ref>。

==故事內容概要==

由「拉格那母星」生存下來的人與其他漂流在[[宇宙|宇宙]]中的人組成「[[銀河|銀河]]漂流艦隊」，為了找尋適合生存的[[星球|星球]]，派遣出「宇宙少年騎士團」四處尋找。李昂等四人被派到[[太陽系|太陽系]]而發現了[[地球|地球]]，無意間將正在「星際大奇航」[[遊戲|遊戲]]中的[[胖虎|胖虎]]與[[小夫|小夫]]帶進所搭乘的[[太空船|太空船]]中離開了[[地球|地球]]。[[哆啦A夢_(漫畫角色)|哆啦A夢]]與[[野比大雄|大雄]]、[[源靜香|靜香]]為了救出他們，乘坐「[[宇宙救命小艇|宇宙救護船]]」追了下去。但是在途中因為空間跳躍時遭到強力[[電磁波|電磁波]]擊中，導致[[太空船|太空船]]損毀。幸而李昂的[[太空船|太空船]]亦在相同地方也是因為跳躍裝置故障而停滯，[[哆啦A夢_(漫畫角色)|哆啦A夢]]等人因此而進入了李昂的[[太空船|太空船]]，恰巧與[[胖虎|胖虎]]和[[小夫|小夫]]相遇。

當李昂一行人回到了「銀河漂流艦隊」前，發現與原先艦隊中多出了以李昂的[[父親|父親]]李貝爾特為首的「[[獨立|獨立]]軍」，這個獨立軍的目的就是要以[[武力|武力]]侵略適合生存的[[星球|星球]]，他們經由派出去的[[間諜|間諜]]佛蕾亞得知了[[地球|地球]]的情報，並且計畫要進攻[[地球|地球]]。

李昂在艦隊之神「優格特樹」的指引下，見到了自己的[[母親|母親]]，並且經由其指示，要瓦解「獨立軍」必須先擊倒控制著他[[父親|父親]]李貝爾特司令官的安葛爾摩亞。在哆啦A夢以及大雄等人的協助下，成功解救了危機<ref name="tetujin5"/>。

== 概要 ==
哆啦A夢長篇作品系列中，這是第三部和[[宇宙|宇宙]]名稱有關的作品，之前有《[[大雄的宇宙開拓史|大雄的宇宙開拓史]]》、《[[大雄的宇宙小戰爭|大雄的宇宙小戰爭]]》。在這作品公開之前，另外還有《[[大雄與鐵人兵團|大雄與鐵人兵團]]》、《[[大雄與白金迷宮|大雄與白金迷宮]]》和《[[大雄與銀河超特急|大雄與銀河超特急]]》三部是和宇宙相關的作品。整體來說，這是大長篇系列第六部的宇宙冒險故事。<ref name="tetujin3">參考{{ja}}[http://homepage3.nifty.com/ulc/movie/doraemon_utyuu.htm STONEMAN MOVIE IMPRESSION]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }} 的解說內容</ref>

這部作品的內容以[[環境|環境]]為主述，由於[[環境污染|環境污染]]等公害而造成原本居住的[[星球|星球]]無法生存，這是在[[科幻小說|SF]][[電影|電影]]中常使用的題材。<ref name="tetujin3"/> 負責[[音樂|音樂]]的是和前一作品《[[大雄的南海大冒險|大雄的南海大冒險]]》一樣的大江千里，主題曲為知名演唱團體[[SPEED_(日本音樂團體)|SPEED]]所主唱。<ref name="tetujin4">參考{{ja}}[http://www.dora-movie.com/film_history/history_20.html 官方網站介紹]</ref>

== 背景 ==

''' 背景 '''

故事的場景是在距離地球約20[[光年|光年]]的「銀河漂流艦隊」，那是由「拉格那母星」生存下來的人以及在宇宙中其他星球飄流出來的人所組成的艦隊，目的是為了要找尋充滿綠色[[植物|植物]]的理想星球。<ref name="tetujin5">參考{{zh}}《大雄的宇宙漂流記》，原著：[[藤子·F·不二雄|藤子·F·不二雄]]，青文出版社2000年4月14日出版，ISBN 957-595-504-8。</ref>

== 評價 ==
20周年的大作品，背景舞台為廣大壯觀的[[宇宙|宇宙]]，仿佛就像是[[星際大戰|星際大戰]]之類的太空劇一般。哆啦A夢在這個故事中所使用的[[哆啦A夢道具列表|神秘道具]]，有許多都是長久未出現過讓人懷念的道具。例如「時光回溯照相機」、「材質變換機」等。同時這個作品還包含有「[[諾斯特拉達姆士|諾斯特拉達姆士]]預言」的要素，反派角色的安葛爾摩亞就是其預言中「世紀末會把地球毀滅掉的魔王」而命名<ref>參考{{zh}}《大雄的宇宙漂流記》，第117頁內容。</ref>。而在電影開頭尋找胖虎和小夫直至大雄跟里安等人會面的情節，亦充滿了宇宙的神秘感，到中段極其巨大的母船並在船內擁有城市及完善的制度更令人眼前一亮。

主題曲除了哆啦A夢之歌以外，另一首由[[SPEED_(日本音樂團體)|SPEED]]所主唱的《當季節逝去時》（{{lang|ja|季節がいく時}}）<ref>參考{{zh}}《大雄的宇宙漂流記》，第144、145頁內容。</ref> 雖然也與劇情並無太大關連，但是卻迎合了20周年時光的流逝這個涵義。<ref>參考{{ja}}[http://www17.ocn.ne.jp/~tksem/20thmovie/hyouryuu.htm 愛好者對於故事內容的評語] {{webarchive|url=https://web.archive.org/web/20070125190316/http://www17.ocn.ne.jp/~tksem/20thmovie/hyouryuu.htm |date=2007-01-25 }}</ref>

不過另一方面，也有些人認為做為一個宇宙戰爭的場景，卻缺少了戰爭那種緊張感，好似故事開頭大家在玩遊戲一般。艦隊的對戰時，也缺少有迫力的感覺，而且安葛爾摩亞居然那麼不堪一擊，也是讓大家認為不合常理的地方。同時，馬茲拉帶領大家到艦隊的聖地參拜艦隊的神「優格特樹」，結果只得到一個啟示就是攻擊安葛爾摩亞，其實這根本不需要神的指示就應該要去做的事情。故事情節中還有靜香全裸入浴的場景，也並不適合兒童而被批評。<ref>參考{{ja}}[http://www.accessup.org/anime/comment.asp?file=Doraemon%3A%20Nobita%27s%20Space%20drift%20reccord&wanted=m {{lang|ja|映画ドラえもん のび太の宇宙漂流記}}] 對於宇宙漂流記電影的評論資料。</ref>
