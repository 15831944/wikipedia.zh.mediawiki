'''大同運動中心'''是[[台灣|台灣]][[台北市|台北市]]第9座啟用市民運動中心，其樓地板面積11,400平方公尺<ref>[http://tsh.tms.gov.tw/web/topicdemo2.php?AS_ID=13 台北市運動地圖資訊網] {{webarchive|url=https://web.archive.org/web/20100312092523/http://tsh.tms.gov.tw/web/topicdemo2.php?AS_ID=13 |date=2010-03-12 }}</ref>，地下3層，地上6層。2008年8月完工，但因招商多次流標<ref>洪敏隆.[http://www.libertytimes.com.tw/2009/new/jun/17/today-taipei10.htm 招商不順 大同區運動中心淪養蚊館].自由時報.2009-06-17 {{webarchive|url=https://web.archive.org/web/20090621170941/http://www.libertytimes.com.tw/2009/new/jun/17/today-taipei10.htm |date=2009-06-21 }}</ref>，直至2009年12月12日正式啟用<Ref>{{Cite web |url=http://tcgwww.taipei.gov.tw/fp.asp?fpage=cp&xItem=1062640&ctNode=7816&mp=105011 |title=臺北市政府全球資訊網 |access-date=2010-02-18 |archive-url=https://web.archive.org/web/20170731160413/http://tcgwww.taipei.gov.tw/fp.asp?fpage=cp&xItem=1062640&ctNode=7816&mp=105011 |archive-date=2017-07-31 |dead-url=yes }}</ref>，承包經營單位是[[遠東鐵櫃鋼鐵廠股份有限公司|遠東鐵櫃鋼鐵廠股份有限公司]]，特色設施為溜冰場。
[[遠東鐵櫃鋼鐵廠股份有限公司|遠東鐵櫃鋼鐵廠股份有限公司]]經營至2018年8月22日營運期滿，重新招標<ref>[https://ppp.mof.gov.tw/ppp.website/case/ShowGovplanAnnounce.aspx?AnnounceNo=tms1070821&LogID=%20 財政部推動促參司]</ref> 點交予新廠商[[舞動陽光有限公司|舞動陽光有限公司]] 進行重新裝修後，於2019年1月5日營運。新增設施包括銀髮促進教室及運動恢復室。

== 地理位置 ==
地址位在[[台北市|台北市]][[大同區_(臺灣)|大同區]]大龍街51號。

== 設備 ==
大同運動中心：5樓為羽球場及溜冰場，4樓為舞蹈教室，3樓為體適能中心，2樓為綜合球場、國際會議廳及銀髮促進教室，1樓為社區教室、飛輪教室、兒童遊戲室及飛輪教室，地下2樓為50公尺游泳池、男女蒸氣室及烤箱、SPA池及兒童戲水池，地下3樓為24小時開放停車場。
