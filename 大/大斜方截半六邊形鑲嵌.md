在[[幾何學|幾何學]]中，'''大斜方截半六邊形鑲嵌'''是[[歐幾里德平面|歐幾里德平面]]上[[六邊形鑲嵌|六邊形鑲嵌]]的一種變形，是種[[平面|平面]][[鑲嵌|鑲嵌]]，屬於[[半正鑲嵌圖|半正鑲嵌圖]]的一種，它的每個[[頂點|頂點]]上皆有[[一|一]]個[[正方形|正方形]]、[[一|一]]个[[六邊形|六邊形]]和[[一|一]]個[[十二邊形|十二邊形]]。在[[施萊夫利符號|施萊夫利符號]]中用t<sub>0,1,2</sub>{6,3}來表示。
== 其他名稱 ==
* 大斜方三角六邊形鑲嵌 （Great rhombitrihexagonal tiling）
* 康威稱大斜方截半六邊形鑲嵌為''' truncated hexadeltille'''，因為'''大斜方截半六邊形鑲嵌'''可由[[六邊形鑲嵌|六邊形鑲嵌]]透過[[大斜方|大斜方]][[康威多面體表示法|變換]]而構造出來。
== 均勻表面塗色 ==
大斜方截半六邊形鑲嵌指有一種[[表面塗色|表面塗色]]，多邊形的邊上作面著色。

[[File:Uniform_polyhedron-63-t012.png|100px]]

另一種表面塗色則允許六邊形的顏色交替。
== 相關多面體及密鋪 ==
有8種均勻鑲嵌圖可以基於六邊形鑲嵌或雙三角形鑲嵌來構造。在8種形式中，截半三角形鑲嵌的拓撲結構與六邊形鑲嵌相同。
