'''大和族'''（{{jpn|j=大和民族|hg=やまとみんぞく|rm=Yamato Minzoku|f=h}}），又称'''日本民族'''、'''和人'''、'''大和人'''，是构成[[日本人|日本人]]的主体[[民族|民族]]，以[[日本語|日本語]]為母語，约占當前[[日本人口|日本人口]]总数的99%（其余为[[阿伊努族|阿伊努族]]與[[琉球族|琉球族]]），主要分布於整個[[日本列島|日本列島]]。最早起源於[[阿伊努人|阿伊努人]]對[[日本|日本]]另一地區居民的稱呼，後來「大和」一詞有時成為日本的代稱，例如[[大和魂|大和魂]]即指日本精神。在[[大日本帝国|大日本帝国]]时代，大和人（[[内地人|内地人]]）是指不同于[[阿伊努族|阿伊努族]]、[[琉球族|琉球族]]、[[朝鲜族|朝鲜族]]、[[漢族|漢族]]、[[台灣原住民|蕃人]]、[[鄂罗克人|樺太土人]]、[[尼夫赫人|尼夫赫人]]的民族。

== 神話起源 ==
*鄭張尚芳引星野恒《日本国号考》谓在筑後国山门郡，王辑五《中国日本交通史》说在今九州肥後菊池郡山门乡。他们另外有一名是[[日向|日向]]族，是阿伊努人对他们称呼。阿伊努神話說他們來到日本前，日本是阿伊努人的地方。
*神話起源說：根据上古传说，大和族是[[天照大神|天照大神]]的子孙。大和這個稱呼應該是從大和时代開始的，這個名字來自存在於公元4世紀，之后日本人自稱為大和族。從[[古墳時代|古墳時代]]開始，大和政權統一了割據的小國。此時日本定都於奈良大和地區，國王雖屢次遷都，但因遷都範圍仍局限於大和地區，所以日本被稱為'''大和'''（やまと，Yamato）<ref>参考[[大和時代|大和時代]]</ref>。以飛鳥地區為中心的大和政體，經過幾代氏族的發展（通過天皇授予封號或是家主繼承），逐漸推廣到日本本州和九州。而大和這個名字也引申為當時中央政體使用的以抑制在整個日本分散的其他地方氏族和諸侯國的政策<ref>参考[[飛鳥時代|飛鳥時代]]</ref>。大和族還有另兩名字，叫'''[[天孫民族|天孫民族]]'''與'''天降民族'''，意思是他們是從[[高天原|高天原]]掉下來與神造（[[天照大神|天照大神]]），皇族是神統，任何地方也找不到；隨[[神武天皇|神武天皇]]東征大和，建立王權，大和族也因此得名。战前日本教育说日本人与天皇是一家，是神的后裔<ref>關於大和族源，另有從朝鮮半島移居而來『朝鮮半島直接渡來說』。約3500年前東亞地區氣溫逐漸下降，即出現寒冷化現象，引發人口遷徙的『北亞人種間接渡來說』，與16世纪末，赴日传教的[[葡萄牙|葡萄牙]]人罗德里格斯在《日本教会史》中写道：“日本最早的移民来自[[浙江|浙江]]，日本国王即其苗裔。”『江南农民直接渡来说』。參考：[http://www.zj-jtest.com/chrbwhtsh/rbwh3-2.htm 倭人的起源] {{webarchive|url=https://web.archive.org/web/20081229060838/http://www.zj-jtest.com/chrbwhtsh/rbwh3-2.htm |date=2008-12-29 }}</ref>。

== 分佈==
大和族的人口約1億2511万7千人（2016年7月），主要分布於整個[[日本列島|日本列島]]，全世界大部份國家亦有少量分佈。[[二戰|二戰]]前，在[[朝鮮半島|朝鮮半島]]、[[台灣|台灣]]、[[庫頁島|庫頁島]]、[[北方四島|北方四島]]、[[中國東北|中國東北]]、[[夏威夷|夏威夷]]、[[關島|關島]]及[[德國|德國]]在[[太平洋|太平洋]]島嶼的[[殖民地|殖民地]]也有所分佈。

二戰後，蘇聯趁日本局勢混亂時趁機強佔[[庫頁島|庫頁島]]和[[北方四島|北方四島]]，將島上的大和族和一些日本原住民全趕到北海道，而中國東北則由於日本軍隊的倉皇撤退，導致將近有千人以上的大和族被遺棄在此地並和當地漢人同化，1949年成立的中華人民共和國並不承認這些遺留東北的日本人是少數民族。

== 語言 ==
绝大多数的大和族人使用[[日语|日语]]。

== 宗教 ==
多数信仰本土宗教[[神道|神道]]、及7世紀前後傳至日本的[[佛教|佛教]]，[[儒學|儒學]]亦有影響，少數人信仰[[基督宗教|基督宗教]]。大部分人同时信仰神道与佛教，甚至出現[[神佛習合|神佛習合]]的混合信仰。

==文化==
最早的大和族在風俗上與[[南島民族|南島民族]]有相似之處，如斷髮紋身，扎頭巾。日語语法與[[高句丽語|高句丽語]]相似，但古早文化上與[[南島民族|南島民族]]更相近，例如穿[[横幅|横幅]]与贯头衣、[[男女混浴|男女混浴]]、[[椎髻|椎髻]]、種[[粳稻|粳稻]]、吃[[刺身|刺身]]（[[生魚片|生魚片]]）、黑齒、跣足。而其大部分文化与[[六朝|六朝]]相近。其文化称為[[照叶树林文化|照叶树林文化]]，与[[云南|云南]]与江南地方文化相似。[[魏志倭人傳|魏志倭人傳]]記載[[倭人|倭人]]有[[紋面|紋面]][[紋身|紋身]]習慣，與百越相似；而在[[履中天皇|履中天皇]]与[[安康天皇|安康天皇]]時期，仍有[[部落民|卑姓者]]黥面。

大和人有嚴格的[[等級制度|等級制度]]，對內分為[[天皇|天皇]]、[[公卿|公卿]]、[[大名|大名]]、[[武士|武士]]、[[農民|農民]]、[[技術工人|工匠]]、[[町人|町人]]與[[部落民|部落民]]。[[明治维新|明治维新]]前，除了早期[[渡来人|渡来人]]、大名与武士以外（有姓氏在古代日本是有头有臉的象徵），庶民並無姓氏（也没资格有），天皇為[[日本神祇|神祇]]也無姓氏（有名無姓是皇室特權）。

==起源研究==
一般認為大和族是由[[通古斯人|通古斯人]]、[[彌生人|彌生人]]、[[朝鮮族|朝鮮族]]、[[汉族|汉族]]、[[南島民族|南島民族]]、中南半島人等民族構成。從人類遺傳學角度分析日本、東亞和西伯利亞人口，發現了日本主要人口為繩文人、彌生人和中國漢族的混合人群<ref>[https://www.ettoday.net/news/20140103/312781.htm?feature=%E9%81%A0%E9%9B%84%E5%BC%8A%E6%A1%88&tab_id=1078 日本人口大起底：中國漢族是重要組成部分？]</ref>。


绳文人膚黑，鬚髮濃密，頭骨比現代日本人大，轮廓明显。弥生人身高比绳文人高，皮肤白皙，五官细致，脸型瘦长，鼻子窄高，和现今日本人较相似（弥生人特徵是单眼皮，高眉毛，乾耳垢，齿根较长较高，脸瘦长，身体表面积少，体毛少，不容易因体毛上的汗冻结而冻伤，耐寒性强，男性身高在1.65米左右）。这两种人不断混血形成日本民族。明治时代认为弥生人是外来民族取代了土著（日本人[[金关丈夫|金关丈夫]]认为在[[弥生时代|弥生时代]]早期有一支朝鲜半岛与中国东面沿海种族登陆北九州赶走大量绳文人）。

彌生族属“新蒙古人種”系统，最初居住于[[本州|本州]]西端和[[北九州|北九州]]，以后逐渐向东扩散，经[[近畿|近畿]]地区扩散至[[关东|关东]]地区。在扩散过程中，他们不断与绳纹人混血，形成現在的日本人。但東北地區的大和族，仍保留了較多繩紋人基因，只是接受新移民的文化，而北海道、西北九州、[[四國|四國]]离岛和[[鹿儿島|鹿儿島]]大和族则较少或几乎未受移民混血的影响，仍以绳纹人子孙为主要居民。于是在弥生时代，形成了日本人二重构造，即移民系统弥生人（主要分布于[[西日本|西日本]]，來自[[滿洲|滿洲]]、[[朝鮮半島|朝鮮]]、[[江南|江南]]）和绳纹系日本人（主要分布于东日本、北海道、[[琉球群岛|南西諸島]]）。移民系弥生人其特点是身材高（平均约162公分左右）、脸型长而扁平等，更接近于现代日本人與韓國人（朝鮮族）。这就是[[植原和郎|植原和郎]]说的日本人二重结构。弥生人也有不同来源，有韓人（出云系）、汉人（日向系）。来自汉地的弥生人多数由长江流域至山东一帶到日本。

[[大和王权|大和王权]]成立后，向外扩张，最远至[[北陸|北陸]]与[[出羽|出羽]]，向南打败[[熊袭|熊袭]]与入侵[[新罗|新罗]]與[[伽耶|伽耶]]，[[白冮口海战|白冮口海战]]后停止向朝鲜扩张，大和族也正式定型（也是日本人说纯血主义开始时期，不再存在大規模移民，之后汉族的迁入多是零星迁入，对日本的种族并没有太大的改变）。后[[蝦夷|蝦夷]]到十三世纪被同化，再向北扩展至北海道[[渡島半島|渡島半島]]，在明治时代前不再扩展（因没法种水稻与阿伊努人抵抗）。
