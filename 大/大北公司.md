'''大北公司'''（{{Lang-da|'''GN Store Nord A / S'''}}）是一家[[丹麥|丹麥]]的聽力儀器和聽力學診斷設備（{{Link-en|GN ReSound|ReSound}}和GN Otometrics）及耳機（{{Link-en|Jabra |Jabra (headset)}}（GN Netcom））的製造商。

==上市==
大北公司在[[哥本哈根证券交易所|NASDAQ OMX Copenhagen]]上市 (ISIN code DK001027263-2).
