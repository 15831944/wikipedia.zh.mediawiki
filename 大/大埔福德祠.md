'''大埔福德祠'''位於[[臺灣|臺灣]][[臺南市|臺南市]][[中西區_(台灣)|中西區]]的大埔街與開山路交叉口，舊稱'''大埔尾土地祠'''{{NoteTag|《重修臺灣縣志‧卷六》：「又文武各衙門左，俱有土地祠。朔望日本衙門行香祭，則本官主之。其居民所祀土王曰福德祠。在東安坊者六：嶺頂、番薯崎、觀音亭邊、諸羅倉邊、嶽帝廟右、龍川井。在西定坊者四：南巷口、新街尾、海防廳邊、[[佛頭港景福祠|佛頭港]]。在寧南坊者五：打石街、安海街、磚仔橋、大南門邊、大埔尾。在鎮北坊者六：禾寮港街、[[總祿境下土地廟|總爺街]]、熟皮寮、赤嵌樓左、粗糠崎、林投井。在永康里者二：彌陀寺右、燒{{僻字|𥕟|左「石」右「屚」}}嵌。他如大南門外仙草寮、小北門外柴頭港及安平鎮新豐里、長興里、土庫、歸仁北里、舊社街，在在有祠。每歲二月二日、八月十五日沿戶鳩資演劇，張燈慶讚，亦春祈秋報之意。」}}{{r|范}}，是一間供奉[[福德正神|福德正神]]的[[土地廟|土地廟]]，此外廟中還陪祀有註生娘娘及[[延平郡王祠|延平郡王祠]]的五營{{r|傅}}{{r|福德}}。

大埔福德祠的土地公與六合境另外兩間土地公廟的土地公據說為親兄弟，其中大埔福德祠是排行老三的「三伯公」，大伯公是[[仁厚境福德祠|仁厚境福德祠]]的土地公，而二伯公是[[油行尾福德爺廟|油行尾福德爺廟]]的土地公{{r|謝}}。

== 沿革 ==
[[File:大埔福德祠福德正神.JPG|250px]]
大埔福德祠的具體創建年代不明，一說是在清康熙年間或更早，曾在道光二十二年（1842年）重建{{r|范}}；另一說是在道光二十二年（1842年）由信眾集資興建{{r|傅}}{{r|謝}}{{r|福德}}。而在最初廟址是位在今延平郡王祠文物館附近，之後於咸豐九年（1859年）四月由管理人曾君騰等10人發起重修，同治五年（1866年）再次重修{{r|傅}}{{r|范}}{{r|謝}}{{r|福德}}。而在進入[[臺灣日治時期|日治時期]]後，管理人暨大埔街保正葉永聲在大正元年（1912年）12月倡修，次年（1913年）1月完工{{r|范}}。

而後一說是因為大正三年（1914年）[[開山神社|開山神社]]改建{{r|傅}}{{r|謝}}，另一說是因為昭和十六年（1941年）開闢大埔尾街通往臺南高等女學校（今[[臺南女中|臺南女中]]）的道路{{r|范}}，導致廟宇遭到拆除。又有一說是大正年間因改建開山神社，舊廟在改建範圍內而被拆，之後信眾將土地公神像安置在民宅（大埔街52號）{{r|福德}}。昭和年間開闢道路又將該民宅拆除，信眾只好將神像移到大埔街62號的豆腐店{{r|福德}}。

而在舊廟被拆後，神像暫時寄放在[[台南臨水夫人媽廟|臨水夫人媽廟]]，直到二次大戰後的民國37年（1948年）由蔡萬生等人發起重建，才在今址重新建廟，並在多次整修後成為今貌{{r|傅}}{{r|范}}{{r|謝}}。另外又有說法是二次大戰後，大埔街拓寬，安置土地公神像的豆腐店部分土地被徵收，神像先是移到裕興菸酒專賣店（開山路170號），後有地理師透露打鐵舖附近有五水匯流，是風水寶地{{r|福德}}。信眾們於是將神像寄放在臨水夫人廟，在該處集資建廟，於民國37年（1948年）完工{{r|福德}}。

== 建築 ==
福德祠由拜亭與本殿組成，而拜亭的造型為馬背，本殿為燕尾。其室內有彩磁壁飾，為曾流行一時的裝飾手法，今已不多見{{r|傅}}。

== 其他 ==
大埔福德祠正龕有三尊土地公神像，據說是立廟之初已有{{r|福德}}。三尊神像眼神有差異，居中神像平視，龍邊神像近觀，虎邊神像遠眺，似乎代表該廟土地攻能近觀、遠眺，明察秋毫境內大小事{{r|福德}}
