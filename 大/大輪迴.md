'''《大輪迴》'''是[[民視無線台|民視無線台]]自2001年至2002年間於每週五晚間播出的[[類戲劇|類戲劇]]。是[[盛竹如|盛竹如]]來民視第一部主持的類戲劇節目。

== 概要 ==
主要介紹有生死就有[[輪迴|輪迴]]的故事。有陰陽就有生死，有生死就有輪迴。該劇在[[緯來電視網|緯來電視網]]重播時，連同後續的《人間走馬燈》系列單元，一律改以《[[藍色水玲瓏|藍色水玲瓏]]》之名播出。
== 旁白 ==
* [[盛竹如|盛竹如]]
== 開場白 ==
在我們老祖宗的觀念裡，宇宙是由陰和陽所造成的，兩者相互推動，生命的變化盡在其中，有陰陽就有男女，有男女就有糾纏，有糾纏就有生死，有生死就有輪迴，人與人之間無可避免的紛爭，導致悲劇不斷的產生，今天我們將走出沉重的因果循環，用最自然的態度，看待人間的愛恨糾纏。

==每集結語==
大輪迴提醒您，千萬要記住歷史的教訓，因為只要我們忘了教訓，歷史肯定會再重演。

== 主要演員 ==
詳見[[玫瑰瞳鈴眼|玫瑰瞳鈴眼]]、[[藍色水玲瓏|藍色水玲瓏]]演員表。
