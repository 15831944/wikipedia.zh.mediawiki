'''大山脚站'''（{{lang-ms|Stesen Kereta Api Bukit Mertajam}}）位于[[马来西亚|马来西亚]][[槟城州|槟城州]][[大山脚|大山脚]]鱼池路（Jalan Muthu Palaniappan），属于[[KTM城际铁路|KTM城际铁路]]和[[KTM通勤铁路北马区|KTM通勤铁路北马区]]。[[马来亚铁道公司|马来亚铁道公司]]是该车站的主要经营者。

==重建==
[[File:Bukit_Mertajam_Railway_Station_platform.jpg|thumb]]
[[File:Bukit_Mertajam_Railway_Station_carpark.jpg|thumb]]
大山脚原有的火车站位于火车头街，而火车站也在2013年搬迁至鱼池路峇冬拉哇国民中学斜对面的新火车站。而根据马矿业─金务大透露，之所以将旧车站搬迁至鱼池路是因为需要重新规划铁道。而旧车站搬迁后，大山脚旧火车站建筑物也被拆除，正式走入历史，现成为[[马来亚铁道|马来亚铁道]]主要的办事处。<ref>{{cite news|url=http://www.sinchew.com.my/node/940549|title=大山脚．技术问题展延．大山脚火车站24日迁新站|date=2013-09-23|publisher=[[星洲网|星洲网]]}}</ref>

新车站的售票处位于一楼，停车场设有逾120辆停车位。
新车站拥有两个入口，一个位于鱼池路而另一个在巴东拉浪路宰猪场旁。<ref>{{cite news|url=http://www.kwongwah.com.my/?p=208285|title=大山脚新火车站 空间较宽敞 设计现代化|publisher=[[光华日报|光华日报]]}}</ref>
