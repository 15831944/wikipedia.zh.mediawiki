'''大谷 直人'''（おおたに なおと；{{lang-en|Naoto Ōtani}}、{{bd|1952年|6月23日|||Ohtani Naoto}}）為[[日本|日本]]的[[法官|法官]]。

== 概要 ==
[[北海道|北海道]][[赤平市|赤平市]]出身。就讀過{{link-ja|東京都立富士高等學校・附屬中學校|東京都立富士高等学校・附属中学校|東京都立富士高等學校}}、{{link-ja|東京大學大學院法學政治學研究科・法學部|東京大学大学院法学政治学研究科・法学部|東京大學法學部}}畢業。{{link-ja|司法修習|司法修習}}29期，擔任法官、東京地方裁判所判事補。之後擔任最高裁判所調査官、司法研修所教官、[[東京地方裁判所|東京地方裁判所]]部總括判事、[[最高裁判所|最高裁判所]]刑事局長・人事局長、{{link-ja|静岡地方裁判所|静岡地方裁判所}}所長、{{link-ja|最高裁判所事務總長|最高裁判所事務総長}}、[[大阪高等裁判所|大阪高等裁判所]]長官等職務。

2015年起擔任[[最高裁判所裁判官|最高裁判所判事]]，<ref>{{Cite web |date=2015-02-18 |url=https://www.nikkei.com/article/DGXLZO83336170X10C15A2CR8000/ |title=大谷最高裁判事「心傾け誠実に」 就任会見 |publisher=日本経済新聞社 |accessdate=2018-01-09}}</ref>[[2018年|2018年]][[1月9日|1月9日]]就任第19任[[最高裁判所長官|最高裁判所長官]]<ref>{{Cite web |date=2018-01-09 |url=https://www.jiji.com/jc/article?k=2018010901131&g=soc |title=「信頼される裁判所に」＝大谷新長官が就任会見－最高裁 |work=時事ドットコム |publisher=時事通信社 |accessdate=2018-01-09}}</ref>。在裁判所内被稱為「裁判員制度先生」。
