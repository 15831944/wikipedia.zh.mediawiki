'''大前茜'''（{{lang-ja|大前 茜}}，{{bd|1982年|8月21日<ref name="ken-pro">{{cite web|url=http://www.kenproduction.co.jp/member.php?mem=w11|title=賢Production所屬期間的聲優簡介|publisher=[[賢Production|賢Production]]|accessdate=2018-08-30|language=ja|archiveurl=https://web.archive.org/web/20100121144344/http://www.kenproduction.co.jp/member.php?mem=w11|archivedate=2010-01-21|deadurl=yes|df=}}</ref><ref name="animate times_pro">{{Cite web |script-title=ja:【Profile】Teoriaの鄙うた |work=大前茜的個人部落格 |author= |publisher={{link-ja|Animate Times|アニメイトタイムズ}}|date= |url=http://blog.animate.tv/akaneken/index.php?itemid=8999 |accessdate=2018-05-10|language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20101106004341/http://blog.animate.tv/akaneken/index.php?itemid=8999 |archivedate=2016-03-04 }} ※2016年4月18日網站已關閉。</ref>||}}），[[日本|日本]]女性前[[配音員|配音員]]。出身於[[埼玉縣|埼玉縣]]<ref name="ken-pro" /><ref name="animate times_pro" />。[[身高|身高]]146cm。O型[[血型|血]]。

== 來歷、人物 ==
本名相同。原本志向是當[[警察|警察]]而開始學[[柔道|柔道]]，但身高只有146公分，無法符合標準而放棄。

1999年，大前在[[富士電視台|富士電視台]]音樂綜藝節目、《{{link-ja|快進擊TV歌唱衛門|快進撃TVうたえモン|快進擊TV！歌唱衛門}}》「聲優之星誕生」單元中得獎，並帶來自己畫的插圖上節目獲得讚揚與好評<ref name="animate times_pro" />。同年，她以電視動畫《[[櫻桃小丸子|櫻桃小丸子]]》（聲演洋子）出道，並在另一齣電視動畫《[[ONE_PIECE_(動畫)|ONE PIECE]]》演過配角。2001年以後全面投入配音界以來，主要參加海外電影、電視影集及動畫的日語配音。

擁有對[[烏鴉|烏鴉]]生態詳盡皆知的專業知識。興趣和特長是插圖<ref name="ken-pro" />，因此她的畫工非常熟練，同時還會把自己的畫上傳至[[部落格|部落格]]公開。2006年播出的電視動畫《[[魔法老師_(動畫)|魔法老師!?]]》除了幫角色配音之外，並兼任片尾插圖、各話標題的命名，讓她的畫畫才能有了開花的成果。

2010年11月1日，大前在自己的網誌發表已在10月31日，離開出道以來從屬的[[賢Production|賢Production]]，正式從配音界退休<ref name="animate times">{{Cite web |url=http://blog.animate.tv/akaneken/index.php?itemid=8999 |script-title=ja:【11年間の奇跡】Teoriaの鄙うた |work=大前茜的個人部落格 |author= |publisher=Animate Times |date= |accessdate=2018-05-10 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20101106004341/http://blog.animate.tv/akaneken/index.php?itemid=8999 |archivedate=2016-03-04 }} ※2016年4月18日網站已關閉。</ref>。退休的原因是她想轉行從事以前有工作經驗的其它工作<ref name="animate times" />。

== 最終神話戰爭伊特奧佩拉 ==
