《'''大风歌'''》是[[汉朝|汉朝]]开国皇帝高帝[[刘邦|刘邦]]，在公元前195年初平定[[英布|英布]]之乱后，回到故里[[沛县|沛县]]，在宴请乡邻的酒宴上，酒酣耳热之余，击筑而歌，《大风歌》由此问世<ref>{{cite web |url=http://book.sina.com.cn/cul/history/2013-01-10/1410399708.shtml|title=汉高祖与大风歌|accessdate=2013年1月10日|work=|publisher=新浪读书|language=zh-hans}}</ref>。後以其开头两字大风命名，大风歌气魄恢弘，流传千古。

《[[昭明文选|昭明文选]]》称之为歌一首（并序·七言），列为杂歌。《[[乐府诗集|乐府诗集]]》称之为《'''大风起'''》。[[明代|明代]]文艺批评家[[胡应麟|胡应麟]]称《大风歌》是“千秋气概之祖”<ref>{{cite web |url=https://www.douban.com/group/topic/112702962/|title=你为什么喜欢汉高祖刘邦？ |accessdate= 2018年2月14日|work=|publisher=豆瓣网|language=zh-hans}}</ref>。

==后世影响==
[[唐太宗|唐太宗]][[李世民|李世民]]《重幸武功》云：“共乐还乡宴，欢比大风诗”，《大风歌》成为开国帝王吟诗作赋的表率。晚唐诗人[[林宽|林宽]]《歌风台》云：“蒿棘空存百尺基，酒酣曾唱大风词。莫言马上得天下，自古英雄尽解诗”<ref>{{cite web |url=http://ah.people.com.cn/n/2015/1223/c358327-27382045-4.html|title=汉高祖刘邦与大风歌的历史渊源|accessdate=2015年12月23日|work=|publisher=人民网|language=zh-hans}}</ref>。

==百家讲坛==
2011年2月24日，[[中国中央电视台|中国中央电视台]]《[[百家讲坛|百家讲坛]]》栏目以“大风歌”为名，讲述了汉高帝刘邦一生。由[[河南大学|河南大学]]教授[[王立群|王立群]]主讲，对刘邦进行反思和总结。<ref>{{cite web |url=http://tv.cntv.cn/videoset/C24850/|title=百家讲坛《大风歌》|accessdate=2011年2月24日|work=|publisher=CNTV|language=zh-hans}}</ref>。
