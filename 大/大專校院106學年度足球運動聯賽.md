'''大專校院106學年度足球運動聯賽(2016-2017 University Football Association)'''為第13屆[[大專校院足球運動聯賽|大專校院足球運動聯賽]]。男子組分為公開組男生第一級、公開組男生第二級和一般男生組，女子組分為公開組女生第一級。本年度公開組男生第一級增列資格賽，並將隊伍數量增加至16隊，本年度公開組男生第一級第十五名、第十六名將降級到來年公開組男生第二級。公開組男生第二級冠軍、亞軍將升級到來年公開組男生第一級。一般男生組前四名將來年公開組男生第二級。<ref>{{Cite web |url = http://web2.ctusf.org.tw/index.php?option=com_content&view=article&id=981 |title = 更改「中華民國大專校院106學年度足球運動聯賽 競賽規程及資格賽競賽規程 |author =  |date = 2017-08-29 |publisher = 中華民國大專院校體育總會 |deadurl = yes |archiveurl = https://web.archive.org/web/20171204171228/http://web2.ctusf.org.tw/index.php?option=com_content&view=article&id=981 |archivedate = 2017-12-04 }}</ref>
