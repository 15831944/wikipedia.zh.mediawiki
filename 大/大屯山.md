'''大屯山'''位於[[台灣|台灣]]西北部的[[陽明山國家公園|陽明山國家公園]]內，轄區在[[台北市|台北市]]的[[北投區|北投區]]，海拔標高是 1,092 公尺，為一[[錐狀火山|錐狀活火山]]，地理上屬於[[大屯火山彙|大屯火山群]]的'''大屯山'''亞群，目前是台灣境內唯一的活火山<ref name="0404_CNA"></ref><ref name="0604_NP"></ref>，由大屯火山觀測站（TVO）負責監測與研究事務<ref name="0727_Ymsnp"></ref>。

== 地理 ==
===大屯山噴發活動===
大屯山的噴發活動約始於250萬年前，在今日的大磺嘴附近有了噴發活動，形成原始的大屯山。之後沉寂了約一百多萬年，直到約70萬年前，另一次火山噴發將原始大屯山分離成現今的大屯主峰、大屯西峰和大屯南峰，因此在這些峰頂都沒有火山口，古火山口反而位於三峰與[[中正山|中正山]]（十八份山）之間。
而最近的一次噴發則約在 6,000～5,000 年前<ref name="0727_Ymsnp"></ref><ref name="0627_Smile"></ref>。

=== 概要 ===
大屯山主峰現在的海拔1092公尺，設有二等[[三角點|三角點]]、聯勤內補006號三角點。山頂為民航局助航站設施，一般遊客無法進入。峰頂附近的觀景台，以夕陽、雲海與秋芒景觀聞名<ref name="陽明山國家公園">{{cite web |url=http://www.ymsnp.gov.tw/index.php?Itemid=131 |title=大屯主峰  |author=|publisher=陽明山國家公園| accessdate=2016-02-03}}</ref>。主峰與其周遭四個小山頭，圍繞中央低窪濕地，即大屯坪。大屯山主峰北方有[[寄生火山|寄生火山]][[菜公坑山|菜公坑山]]；南方為大屯南峰，亦是一錐形火山，峰群與西側鐘形的[[面天山|面天山]]、[[向天山|向天山]]構成一系列馬鞍形山坳；向天山頂部西側有一完整漏斗形噴火口地形，積水成池，即向天池；除此之外其他峰群頂部無明顯噴火口地形。大屯山區的登山步道群包括[[軍艦岩|軍艦岩]]、[[丹鳳岩|丹鳳岩]]、中正山、竹子湖、大屯群峰地區、貴子坑環山步道、忠義山步道等等。

因位處台灣島北端，海拔高又迎東北季風，冬季時偶有降雪。


== 歷史 ==
*1764年（乾隆29年），「'''屯山積雪'''」曾收列於「淡水廳四景」，就是指這個地方。
*1871 年（同治10年），《淡水廳志》林逢原「淡北八景」，亦指於此。
*同治年間陳維英「淡北八景」之一<ref name="觀光志">{{cite web |url=http://www.tamsui.ntpc.gov.tw/archive/file/1020709histou.pdf |title=觀光志  |author=林玫君|publisher=新北市政府| accessdate=2016-02-03}}</ref>
*1932年出版之地圖，大屯山主峰海拔標記為 1081公尺。
*民國以後，新北市政府則以「'''屯山春色'''」收列為淡水新八景之一<ref name="淡水新八景">{{cite web |url=http://www.tamsui.ntpc.gov.tw/content/?parent_id=10248 |title=淡水新八景|author=|publisher=新北市政府| accessdate=2016-02-03}}</ref>。

[[File:大屯山20160124.jpg|right]]時的「屯山積雪」之景。]]
*2016年1月的[[2016年1月北半球寒流|2016年1月北半球寒流]]，再現「屯山積雪」之景。
*2017年，[[中央研究院|中央研究院]]研究員[[林正洪|林正洪]]團隊，首度證實大屯火山存在有[[岩漿庫|岩漿庫]]<ref name="0202_Pansci"></ref>。
*2018年2月再度降雪。<ref>{{Cite news|url=https://www.ttv.com.tw/news/view/10702040015200N/579|title=北部也下雪了! 陽明山雪霰雙降白茫茫銀白世界! 陽明山降雪 民眾揪團衝上山追雪太刺激! 民眾轉往冷水坑泡腳取暖想追雪趕快衝! 估週一水氣減 下雪機率減|author=|date=2018-02-04|work=台視新聞|accessdate=2018-02-04}}{{繁體中文}}</ref>
