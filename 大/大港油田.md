'''大港油田'''是位于[[中华人民共和国|中国]][[华北|华北]]地区的[[油田|油田]]，始建于1964年1月<ref>{{cite web |title=大港油田 |url=http://www.petrochina.com.cn/hdipad/yqt/201404/6b5bdc78f70a48b8832e9732a303d567.shtml |website=中国石油天然气集团有限公司 |accessdate=2019-01-19}}</ref>。[[中国石油天然气股份有限公司|中国石油天然气股份有限公司]]大港油田分公司负责其勘探、生产、加工、运输等工作，其总部位于天津市[[滨海新区|滨海新区]][[海滨街道_(天津市滨海新区)|海滨街道]]。大港油田大港探区的勘探面积达到18,629平方公里，由15个油气田、24个开发区组成<ref name="YQCY">{{cite web |title=揭秘大港油田的发展史 |url=http://www.youqichuyun.com/article-1042-1.html |website=油气储运网 |accessdate=2019-01-19}}</ref>。

1963年12月，[[河北省|河北省]][[黄骅市|黄骅县]]发现工业油流，中华人民共和国[[石油部|石油部]]决定组织华北石油会战<ref>{{cite web |title=激情燃烧的岁月：华北石油会战 |url=http://uzone.univs.cn/news2_2008_729496.html |website=石油之光 |accessdate=2019-01-19}}</ref>，1964年12月20日，港5井发生井喷，宣告大港油田诞生。继港5井后，港7井和港3井也开始产油，日产油均上百吨，随后港东、港中、港西3个油田相继发现<ref name="G5J">{{cite web |title=港5井：托起渤海湾的希望 |url=http://news.cnpc.com.cn/system/2014/06/25/001493760.shtml |website=中国石油新闻中心 |accessdate=2019-01-19}}</ref>。1999年，成立中国石油天然气股份有限公司大港油田分公司。

1975年，大港油田的原油年产量达到468.24万吨，位居全国第三，仅次于[[大庆油田|大庆油田]]、[[胜利油田|胜利油田]]<ref name="G5J"/>。截至2005年底，大港油田累计探明的石油地质储量有9.36亿吨、天然气储量734.77亿立方米，累计生产原油1.32亿吨、天然气164.5亿立方米；至2015年，探明石油地质储量12.4亿吨、天然气储量740亿立方米；累计生产原油1.81亿吨、天然气235亿立方米<ref name="ZQXW">{{cite web |title=“变老”的大港油田路在何方 |url=http://nengyuan.qlwb.com.cn/2016/0421/603660.shtml |website=齐鲁晚报网 |accessdate=2019-01-19}}</ref>。

大港油田以工人驻地和华北石油勘探指挥部驻地为基础，逐渐发展为社区，社区区域横跨[[天津市|天津市]][[滨海新区|滨海新区]]和河北省[[沧州市|沧州市]]。大港油田机关驻地“二号院”和“三号院”发展为社区中心，建有车站、酒店、医院、学校。2013年12月19日，天津市民政局批复同意将海滨街道、[[港西街道|港西街道]]两街道办事处所辖区域合并，设立新的海滨街道，涵盖了大港油田社区的核心地带<ref>[http://gk.tj.gov.cn/gkml/000125735/201312/t20131231_39244.shtml 天津市民政局关于滨海新区调整部分街镇行政区划的批复（津民复〔2013〕19号）]</ref>。[[天津工程职业技术学校|天津工程职业技术学校]]、[[大港油田第一中学|大港油田第一中学]]、[[大港油田实验中学|大港油田实验中学]]等学校坐落于此。2017年起，又陆续建成港西新城。
