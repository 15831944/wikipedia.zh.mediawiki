'''大庆铁人王进喜纪念馆'''，位于[[黑龙江省|黑龙江省]][[大庆市|大庆市]][[让胡路区|让胡路区]]，是为开发[[大庆油田|大庆油田]]的英雄、“铁人”[[王进喜|王进喜]]建立的纪念馆。

==历史==
大庆铁人王进喜纪念馆的前身是“铁人王进喜同志英雄事迹陈列室”，始建于1971年。1989年，在位于大庆市解放二街8号的“铁人王进喜同志英雄事迹陈列室”旧址上新建了“铁人王进喜纪念馆”。该馆总占地面积5.4万平方米，其中绿地面积3万平方米，主馆建筑面积1240平方米<ref name=dqx>{{Cite web |url = http://www.sohu.com/a/50580091_125321 |title = 王进喜纪念馆和他的铁人精神 |author =  |date =2015-12-25 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref name=jj>{{Cite web |url =http://www.hljtv.com/zhuanji/redhome/rw/wjx_main.shtml  |title =铁人王进喜同志纪念馆简介  |author =  |date =  |publisher = 黑龙江网络广播电视台 |language =  |accessdate = 2018-03-01 |quote =  }}</ref>。

2003年2月，由大庆地区石油石化企业、中共大庆市委、大庆市人民政府协商，[[中共黑龙江省委|中共黑龙江省委]]、[[黑龙江省人民政府|黑龙江省人民政府]]支持，[[中国石油天然气集团公司|中国石油天然气集团公司]]批准，决定迁建铁人王进喜纪念馆。2003年10月8日王进喜诞辰80周年之际，新馆奠基。2006年8月10日，[[温家宝|温家宝]]题写馆名。2006年9月26日大庆油田发现47周年纪念日，新馆开馆<ref name=dqx/><ref name=beifang>{{Cite web |url =http://news.enorth.com.cn/system/2007/06/05/001699288.shtml  |title =铁人王进喜纪念馆简介  |author =  |date =2007-06-05   |publisher = 北方网 |language =  |accessdate =  |quote =  }}</ref><ref name=hexun>{{Cite web |url = http://tech.hexun.com/2011-06-23/130837350.html |title =  网络媒体龙江行采访团参观大庆铁人王进喜纪念馆|author =  |date = 2011-06-23 |publisher = 和讯 |language =  |accessdate =  |quote =  }}</ref>。新馆位于大庆市让胡路区铁人大道与世纪大道交汇处，馆区占地面积11.6公顷，纪念馆主体建筑面积2.15万平方米<ref name=beifang/>。

==展览==
[[File:大庆铁人馆序厅浮雕.jpg|thumb]]
纪念馆主体建筑前方的广场平台上，两片草坪之间立有一尊铁人王进喜手持刹把的大型[[花岗岩|花岗岩]]雕塑《铁人》，高6.5米，重60余吨，是[[中国东北|中国东北]]地区最大的人物石雕。纪念馆主体建筑外形是“工人”二字的组合，平面呈“工”字形，侧看呈“人”字形。主体建筑高47米，正门台阶47级，象征王进喜47年的人生历程。建筑顶部造型为[[钻头|钻头]]，象征大庆油田奋发向上的精神<ref name=dqx/>。

铁人王进喜纪念馆馆藏文物5324件，其中定级文物152件<ref name=beifang/>。纪念馆序厅内，一进门是一组铸铜雕像《石油魂》，表现了以王进喜为代表的英雄群体。纪念馆的展厅分布在二层及三层，共陈列200多幅照片，300多件实物，展示了王进喜的个人经历及在大庆石油会战中的业绩，以及大庆人学习铁人精神的情况。展览以王进喜生平为主线，以大庆石油发展史为副线，采用照片、文字、电动图表、硅胶像、沙盘、场景复原、多媒体等展示手段，展现了“爱国、创业、求实、奉献——石油魂”的主题。各展厅之间通道处，挂有巨幅国画《大庆工人无冬天》及会战诗抄墙、战报墙、宣传铁人和石油会战的美术作品<ref name=dqx/>。

纪念馆馆区内有大型主题雕塑《崛起》、《奋进》、《民族脊梁》及装饰性雕塑《石油交响曲》，还有《六个传家宝》、《回收队》、《干打垒》、《五把铁锹闹革命》、《人拉肩扛》、《修旧利废》、《缝补厂》雕塑。还有端水湖、标杆山及王进喜率领1205钻井队使用过的同型贝乌—40型钻机（БУ-40型钻机，[[苏联|苏联]]制）<ref name=dqx/><ref name=beifang/>。
