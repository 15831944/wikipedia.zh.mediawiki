== 車站構造 ==
*站體為鋼筋混凝土建築
*[[島式月台|島式月台]]、[[岸式月台|岸式月台]]各一座。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#二等站|二等站]]，各級列車每班都有停靠。
**此站為2612/2618/2632次[[成追線|成追線]][[臺鐵區間車|區間車]]的終點。
*設有自動售票機2部，提供販售西部幹線[[新竹車站|新竹]]＝'''大甲'''＝[[員林車站|員林]]及山線[[成功車站|成功]]＝[[臺中車站|臺中]]沿線各車站之車票。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]、[[icash|icash 2.0]]及[[HappyCash|HappyCash]]付費。

== 歷史 ==
*1922年10月11日：設「大甲驛」。
*1966年2月：改建為鋼筋混凝土建築物。
*1985年10月：通勤電車運轉行駛。
*2014年7月16日：為加強疏運需要，開始停靠[[太魯閣列車|太魯閣號]]。
*2015年6月30日：啟用多卡通刷卡機。
*2015年10月15日：原有的278、285次[[太魯閣列車|太魯閣號]]改由[[普悠瑪列車|普悠瑪號]]行駛。
*2016年10月20日：原[[普悠瑪列車|普悠瑪號]]行駛之278、285次改由[[太魯閣列車|太魯閣號]]行駛。
*2018年5月4日：跨越前後站的人行路橋動工，預計次年2月完工。<ref>https://udn.com/news/story/11322/3124487</ref> <ref>http://news.ltn.com.tw/news/life/breakingnews/2415582</ref>
