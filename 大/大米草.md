'''大米草'''（[[学名|学名]]：''{{lang|la|Sporobolus anglicus}}''），[[禾本科|禾本科]][[鼠尾粟属|鼠尾粟属]]，多年生草本[[植物|植物]]。被[[國際自然保護聯盟|國際自然保護聯盟]][[物種存續委員會|物種存續委員會]]的入侵物種專家小組（ISSG）列入[[世界百大外來入侵種|世界百大外來入侵種]]。

== 形态 ==
大米草具有根状[[茎|茎]]，秆高达1.2-1.5米，[[叶片|叶片]]狭披针形，上面具有致密的纵肋，[[叶舌|叶舌]]为一圈密生的纤毛，总状[[花序|花序]]3-6枚呈总状排列，穗轴顶端延伸至刺芒状，小穗狭披针形，含一小花。

本种可以在海水能到达的海滩生长，冬季枯死，春季萌生，如同普通野草，但根系广阔。

== 物种起源 ==
大米草已确定起源自杂交种[[汤氏米草|汤氏米草]]（{{lang|la|''S.'' × ''townsendii''}}）的[[异源多倍体|异源四倍体]]，其原始亲本是原生于欧洲与西非沿海的[[海岸米草|海岸米草]]（''{{lang|la|S. maritimus}}''）及原生于北美洲东岸的[[互花米草|互花米草]]（''{{lang|la|S. alterniflorus}}''）。互花米草约于1870年代随着船只从北美洲被引入英国[[南安普敦|南安普敦]]水域，与当地的海岸米草发生杂交形成汤氏米草，原本不育的汤氏米草偶然出现了具有生殖能力的双二倍体，最后形成大米草这一新物种{{r|jncc}}。

因此，大米草最初在[[大西洋|大西洋]]沿岸繁衍开来，包括[[美国|美国]]东海岸，[[英国|英国]]、[[法国|法国]]沿海和[[非洲|非洲]]西北部沿海。

==分类==
本种最初归类于[[米草屬|米草屬]]（''{{lang|la|Spartina}}''）{{r|theplantlist}}，后来随着整个米草属于2014年并入鼠尾粟属{{r|kew}}。

== 用途 ==
大米草秆、叶可作[[饲料|饲料]]或绿肥，也可作造纸原料；在[[欧洲|欧洲]]可以作为[[牛|牛]]的一种饲料。

==入侵物种==
由于大米草比其原始亲本具有更高的耐盐性及更强的生存力，因而在西欧，大米草反而将海岸米草排挤出原生地。一开始时，大米草被视为可以保护海滩免受海潮侵蚀的植物，因而被引种到各地，但却也因为其生命力而成为了危害各地原有生态的入侵物种{{r|jncc}}。

1963年[[南京大学|南京大学]]教授仲崇信出于保护沿海滩涂的考虑，将其引种到[[江苏|江苏]]沿海，1964年引种到[[浙江|浙江]]，1980年引种到[[福建|福建]]。但没有想到的是，在新区域，由于没有天敌，气候适宜，大米草迅速生长，破坏当地原有的[[生态系统|生态系统]]，使泥滩变为草地，危害滩涂的生态[[环境|环境]]，导致贝类、鲍鱼、水藻等大量死亡，影响[[海带|海带]]、[[紫菜|紫菜]]的种植和[[牡蛎|牡蛎]]养殖。由于草质粗糙，[[中国|中国]]牛不喜欢这种饲料，因此无法消耗，甚至导致水质下降，诱发[[赤潮|赤潮]]，以至堵塞航道。尤其严重的是在福建侵入[[红树林|红树林]]，导致红树林大批死亡，红树林区域日益萎缩，用机械方法除草非常困难，现在尚没有好的方法彻底根除大米草，是一种错误引种导致的生物入侵。

{{fact|无论如何，一些地区的大米草开始出现原因不明的大片自然枯萎，逆转了扩张趋势，这些地区也不再需要采取人工控制来抑制其生长，只需交由自然界去维持平衡。}}
