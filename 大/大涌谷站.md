'''大涌谷站'''（{{jpn|j=大涌谷駅|hg=おおわくだにえき|Ōwakudani eki}}）是一座位於[[神奈川縣|神奈川縣]][[箱根町|箱根町]]{{link-ja|仙石原}}，屬於[[箱根空中纜車|箱根空中纜車]]的[[鐵路車站|鐵路車站]]，車站編號為'''OH 63'''<ref>{{cite press release|url=http://www.odakyu.jp/program/info/data.info/8052_1284200_.pdf|script-title=ja:小田急線・箱根登山線・箱根ロープウェイ・箱根海賊觀光船にて２０１４年１月から駅ナンバリングを順次導入します！|publisher=小田急電鐵||language=ja|date=2013-12-24|accessdate=2018-11-22}}</ref>。大涌谷站分別連結了早雲山和桃源台兩地，旅客可在此站轉乘往返兩地。該站旁即箱根火山中最大的噴氣地帶{{link-ja|大涌谷}}，是該觀光地區的着名景點。

== 簡史 ==
* [[1959年|1959年]][[12月5日|12月5日]] - 箱根空中纜車開業<ref name="kaisetsu">{{Cite web |url=http://www.hakoneropeway.co.jp/about/|script-title=ja:箱根ロープウェイをもっと知ろう！|language=ja|publisher=箱根空中纜車|date=2015-08-27|accessdate=2015-10-18}}</ref>，當時為終點站<ref name="kaisetsu" />。
* [[1960年|1960年]][[9月7日|9月7日]] - 桃源台間纜車路線開通，成為中途站。
* [[2000年|2000年]][[1月|1月]]至[[2002年|2002年]][[6月|6月]]期間 - 往返早雲山站間路段進行更新車廂工程。
* [[2006年|2006年]][[6月1日|6月1日]]至[[2007年|2007年]][[5月31日|5月31日]]期間 - 往返桃源台站間路段進行更新車廂工程，2007年6月全線使用新型車廂營運<ref>[http://www.jplaza.co.jp/shopinfo/474.html 「箱根ロープウェイ　大涌谷～桃源台間新型架け替え工事終了に伴う運転再開のお知らせ」（『JTBジェイプラザ・オンライン』2007年5月30日）]</ref>。
* [[2012年|2012年]][[4月17日|4月17日]] - 由於進行車站重建（總建設成本約10億日元），臨時站舍啓用<ref>[http://orikomix.com/shop/news.php?id=867 「大涌谷駅舎建替え工事のお知らせ」（「箱根ロープウェイ最新情報」（『オリコミクス』）、2012年3月2日）]</ref>。
* [[2013年|2013年]][[4月24日|4月24日]] - 新站舍開始營業<ref name="new-stn-pr"/>。
* [[2015年|2015年]][[5月6日|5月6日]] - 因箱根山的火山性地震頻發<ref>{{Cite web |url=http://www.onken.odawara.kanagawa.jp/modules/mysection1/item.php?itemid=44|script-title=ja:箱根のやや活発な地震活動（4月26日〜）|publisher=神奈川縣温泉地学研究所|language=ja|accessdate=2015-10-18}}</ref>、氣象廳針對箱根山地區發出[[火山噴發預警級別|火山噴發預警第二級警報]]<ref>{{Cite web|url=http://www.jma.go.jp/jp/volcano/forecast_03_20150506060003.html|script-title=ja:火山名 箱根山 噴火警報（火口周辺）|publisher=[[気象庁|気象庁]]地震火山部|language=ja|date=2015-05-06|accessdate=2015-10-18}}</ref>，箱根空中纜車全線停駛<ref>{{Cite web|url=http://www.kanaloco.jp/article/101152|script-title=ja:折り紙でおもてなし　箱根・桃源台窓口の女性奮闘：カナロコ|publisher=[[神奈川新聞|神奈川新聞]]|language=ja|date=2015-06-08|accessdate=2015-10-18}}</ref>。
* [[2016年|2016年]]
** [[4月23日|4月23日]] - 來往姥子間的纜車服務再開，但因為火山氣體噴發影響僅開放部份站舍，也不開放出站<ref>{{Cite press|url=http://www.hakoneropeway.co.jp/info_img/223_pdf.pdf|script-title=ja:箱根ロープウェイ 営業運転区間の延長のお知らせ|publisher=箱根空中纜車|language=ja|date=2016-04-18|accessdate=2018-11-29}}</ref>。
** [[7月26日|7月26日]] - 全線再開<ref>{{Cite news|url=http://www3.nhk.or.jp/news/html/20160722/k10010605011000.html|agency=NHK|date=2016-07-22|accessdate=2016-07-22|language=ja|script-title=ja:箱根ロープウェイ ２６日から全線での運行再開|archive-url=https://web.archive.org/web/20160723163313/http://www3.nhk.or.jp/news/html/20160722/k10010605011000.html|archive-date=2016-07-23|dead-url=yes}}</ref>。
* [[2017年|2017年]][[1月10日|1月10日]]至[[4月14日|4月14日]]期間，往返早雲山站的路段進行系統改良工程暫停營運，改以替代運輸巴士服務<ref>{{Cite press|url=https://www.hakone.or.jp/files/w002_2016112113110225159.pdf|script-title=ja:設備改良工事実施に伴う一部区間の営業運転休止について|publisher=箱根空中纜車|language=ja|date=2016-11-14|accessdate=2018-11-29}}</ref>。

== 車站 ==
=== 構造 ===
大涌谷站目前的站體是於2013年4月22日開始營業<ref name="new-stn-pr">{{Cite press|url=http://www.odakyu.jp/program/info/data.info/7917_6548208_.pdf|script-title=ja:箱根ロープウェイ 大涌谷駅舎建替え工事　２０１３年４月２４日（水）完成　雄大な景観に調和したデザインと自然環境保護へ配慮した新駅舎|publisher=小田急|language=ja|date=2013-02-14|accessdate=2018-11-29}}</ref><ref name="new-stn-twnnw">{{Cite news|url=https://www.townnews.co.jp/0609/2013/05/03/186467.html|agency=TOWNNEWS|date=2013-05-03|accessdate=2018-11-29|language=ja|script-title=ja:まるで建物全体が展望台　大涌谷駅舎オープン}}</ref>，站體地下一層、地上兩層共三層<ref name="new-stn-pr"/><ref name="new-stn-twnnw"/>，採用了LED照明及風力發電等環保設計<ref name="new-stn-pr"/>。一樓主要為乘車處和售票等纜車營業設施<ref name="stn">{{Cite web|url=https://www.hakoneropeway.co.jp/station/ohwakudani/|script-title=ja:ロープウェイの乗車案内 - 箱根ロープウェイ|publisher=箱根空中纜車|language=ja|accessdate=2018-11-29}}</ref>，二樓為餐廳「大涌谷車站食堂」<ref name="stn"/>，各樓層均以電梯連接<ref name="new-stn-pr"/>。該站也安裝了往返早雲山站路段的絞盤機等動力設備<ref name="kaisetsu" />。
