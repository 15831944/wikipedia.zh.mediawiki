'''大急流城'''（[[英語|英語]]：'''{{lang|en|Grand Rapids}}'''），又译{{cn|'''格兰德拉匹兹'''}}、'''大湍城'''、'''大急流市'''，是[[美國|美國]][[密歇根州|密歇根州]]的第二大城市，[[肯特郡_(密歇根州)|肯特郡]]郡治所在地。

== 地理与气候 ==
大急流城属[[溫帶大陸性濕潤氣候|溫帶大陸性濕潤氣候]]，四季分明。冬季寒冷而漫长，潮湿，日照少，日最高气温低于{{convert|0|C}}的平均日数为53天，日最低气温低于{{convert|-15|C}}的平均日数为12天，{{convert|-20|C}}以下的有2.6天；夏季相对炎热潮湿，日最高气温超过{{convert|30|C}}的日数年均有25天，{{convert|35|C}}以上的气温只出现在40%以下的夏季。<ref name = "NWS Grand Rapids, MI"/>最冷月（1月）均温{{convert|24.4|F|1|disp=flip}}，极端最低气温{{convert|−24|F|0|disp=flip}}（1899年2月13日与14日）。<ref name = "NWS Grand Rapids, MI"/>最热月（7月）均温{{convert|72.5|F|1|disp=flip}}，极端最高气温{{convert|108|F|0|disp=flip}}（1936年7月13日）。<ref name = "NWS Grand Rapids, MI"/>无霜期平均为161天（5月2日至10月9日）；可测量降雪平均期为11月5日至4月9日。<ref name = "NWS Grand Rapids, MI"/>年均降水量约{{convert|38.3|in|sigfig=3|disp=flip}}，年极端最少降水量为{{convert|20.92|in|disp=flip}}（1930年），最多为{{convert|48.80|in|disp=flip}}（2008年）。<ref name = "NWS Grand Rapids, MI"/>年均降雪量为{{convert|74.9|in|cm|disp=flip}}；1905–06年的降雪量最少，积累降雪量只有{{convert|20.0|in|cm|disp=flip}}，1951–52年的降雪量最多，积累降雪量为{{convert|132.0|in|m|disp=flip}}。<ref name = "NWS Grand Rapids, MI"/>
