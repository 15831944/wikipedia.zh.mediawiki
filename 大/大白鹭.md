'''大白鹭'''（学名：''Ardea alba''），别名'''白鹭鸶'''、'''鹭鸶'''、'''风漂公子'''、'''白漂鸟'''、'''冬庄'''。[[鹭科|鹭科]][[蒼鷺屬|蒼鷺屬]]的一种。

== 生态环境 ==
稻田、河岸、沙滩、泥滩及沿海小溪流，成散群出現。

== 分布地域 ==
非洲、欧洲、亚洲及大洋洲。

== 特征 ==
[[File:Ardea_alba;_3_chicks,_Morro_Bay_Heron_Rookery_2_-_by_Mike_Baird.jpg|thumb]]
体长90cm，[[白鷺屬|白鷺屬]]中最大，白色，繁殖期間背部有飾羽;黑脚，黑腿，腿上部份帶綠或紅;细长黃[[喙|喙]]，繁殖期間全部或部份會變成黑色;叫声-于繁殖巢群中发出呱呱叫声。

== 食物 ==
[[鱼|鱼]]、甲壳动物和[[昆虫|昆虫]]，偶爾也會捕食[[鼠類|鼠類]]。

== 繁殖 ==
與其他水鳥一起以樹枝於樹上築巢，繁殖期為3-9月。

== 文化 ==
大白鹭的圖像描畫在[[巴西|巴西]]5[[雷亞爾|雷亞爾]][[鈔票|鈔票]]背面上, [[新西蘭|新西蘭]]$2[[硬幣|硬幣]]上亦有大白鹭的圖像.
