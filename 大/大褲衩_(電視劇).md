《'''大裤衩'''》（{{lang-en|Big Shorts TV}}）是由加拿大新境界影视公司（New Realm Studios）和新唐人电视台联合制作的制作的系列[[讽刺|讽刺]][[喜剧|喜剧]]<ref>{{cite web |title=讽刺喜剧《大裤衩》第二季杀青 |url=http://www.epochtimes.com/gb/15/11/5/n4566208.htm |website=大纪元 EPOCH TIMES}}</ref>。该剧以[[中国中央电视台|中国中央电视台]]时政栏目《焦点追踪》节目组为故事背景（暗指[[焦点访谈|焦点访谈]]），以[[中国|中国]]近两年重大时政新闻发展为线索，演绎出一幕幕爆笑而写实的黑色幽默剧。
该剧集共分为两季：第一季共16集，于2015年4月4日首播，每周播出一集。第二季共11集，于2016年2月14日在网络和多家中文电视台同步播出, 每周六播出两集。<ref>[http://www.epochtimes.com/gb/15/11/5/n4566208.htm 讽刺喜剧《大裤衩》第二季杀青 | 大纪元]</ref>

== 评价与争议 ==
* 该剧剧名使用了网友对[[中央电视台总部大楼|中央電視台總部大樓]]的戏称“大裤衩”<ref>{{cite news|title=聯邦前朝撥出逾1900萬元 供民間製作批中國電視劇 劇集多在新唐人電視台播出|url=https://www.mingpaocanada.com/van/htm/News/20161012/vaba_r.htm|accessdate=2018-05-19|work=明報|publisher=明報加西版(溫哥華)|date=2016.10.12}}</ref><ref>{{cite web|title=华社质疑加拿大传媒基金拨款 要求社会资源平等共享|url=http://www.ccinfo.ca/wap/bencandy.php?fid=3&id=6181|website=枫桥网|accessdate=2018-05-19|date=2016-10-13}}</ref>。

* 《大裤衩》第二季延续第一季的讽刺喜剧风格。在幽默搞笑的同时，也令人深刻思索。

* 这是一部集娱乐与实事为一身的讽刺喜剧，每一集都取材于中国大陆近年来发生的真实事件，在搞笑幽默的同时又不失深刻的思索。

* 该剧播出后于网络上获得了不错的反响，有网友将其比喻为美剧《[[生活大爆炸|生活大爆炸]]》和《[[纸牌屋|纸牌屋]]》的合体。<ref>{{cite web |title=讽刺喜剧《大裤衩》第二季杀青 |url=http://www.epochtimes.com/gb/15/11/5/n4566208.htm |website=大纪元 EPOCH TIMES}}</ref>

* 该剧受到持反[[中共|中共]]立场的美国[[新唐人電視台|新唐人電視台]]资助。

*中国大陆人民观看此视频后觉得内容体现中国现实。中国中央电视台始终未对此节目公开表示不满。
