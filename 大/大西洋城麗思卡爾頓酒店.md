'''大西洋城麗思卡爾頓酒店'''（{{lang-en|'''Ritz-Carlton Atlantic City'''}}）起初是一間在[[新澤西州|新澤西州]][[大西洋城|大西洋城]]海濱大道上的[[酒店|酒店]]，建於[[咆哮的二十年代|咆哮的二十年代]]初期，以豪華的裝飾和著名的客人而出名的。它在1969年開始被用作為公寓式酒店，然後在1978年被購買來想要發展成為一間酒店/賭場。該建築物於1982年改建為'''麗思公寓'''。

== 歷史 ==
在1937年的大蕭條期間，業主拖欠抵押，麗思卡爾頓酒店在破產的情況下被整頓。這間酒店是城市內其中一間在第二次世界大戰期間被當作軍人訓練的[[營房|軍營]]和休養的地方。在戰爭之後，它在1940年代被賣到Schine酒店，然後在1959年被賣到[[喜來登酒店|喜來登酒店]]，變成喜來登麗思卡爾頓酒店。<ref name = timeline /><ref name = HBO> {{Cite web |url = http://www.nytimes.com/2010/09/05/arts/television/05empire.html?sq=Ritz%20Carlton%20Atlantic%20City&st=cse&adxnnl=1&scp=4&adxnnlx=1308960212-GRVaC0ake34Rd/1CRfPZkg |title = On the Boardwalk, HBO Hangs Out With a New Mob |author = Charles McGrath |date = 2010-09-03 |publisher = The New York Times |language = en}}</ref>麗思酒店於1969年6月改建為[[公寓式酒店|公寓式酒店]]。<ref name = ACexperience>{{Cite web |url = http://www.atlanticcityexperience.org/index.php/the-prohibition-era/7-buildings-and-locations/6-ritz-carlton-atlantic-city |title = Living in the Ritz |publisher = Atlantic City Experience |language = en}}</ref>1978年，一個投資集團購買了這棟建築物，將其改建為酒店和賭場。然而，與{{link-en|公職人員受賄醜聞|Abscam}}調查相關的不利宣傳結束了這個計劃。<ref>{{Cite web |url = http://www.nytimes.com/1981/05/31/nyregion/tale-of-atlantic-city-hotel-that-tried-to-be-a-casino.html |title = TALE OF ATLANTIC CITY HOTEL THAT TRIED TO BE A CASINO |author = Donald Janson |date = 1981-05-31 |publisher = New York Times |language = en}}</ref>參議員{{link-en|哈里森·A·威廉斯|Harrison A. Williams}}告訴一位臥底聯邦調查局探員，他經由讓他們翻新現有的房地產來可以幫助投資者節省3000萬美元，而不是建立一個新的房地產。威廉斯的妻子是該項目的主要投資者Hardwicke公司的受僱顧問和股東，而威廉斯預計將獲得100萬美元的賞金來幫助安排融資該項目。威廉斯後來因為不相關的指控而被判罪。1982年，大約花費了2500萬美元，將其改建為322間住所和6間商務套房，其中一些是正式住所，而另外一些是度假屋。<ref>{{Cite web |url = http://www.nytimes.com/1983/08/28/realestate/atlantic-city-condominiums-bought-for-fun-and-profit.html?scp=20&sq=Ritz%20Carlton%20Atlantic%20City&st=cse&pagewanted=2 |title = ATLANTIC CITY CONDOMINIUMS BOUGHT FOR FUN AND PROFIT |author = Donald janson |date = 1983-08-28 |publisher = The New York Times |language = en}}</ref>與此同時，新成立的[[麗思卡爾頓酒店管理集團|麗思卡爾頓酒店管理集團]]付錢給該大樓的業主放棄使用麗思卡爾頓的名字，以避免與他們的酒店混淆。<ref>http://www.ritzac.com/Ritz%20History%20Time%20Line.pdf</ref>從那時起，這座大樓就成為了麗思公寓。

=== 努基·湯普森與《酒私風雲》 ===
在[[HBO|HBO]]原創電視劇《[[酒私風雲|酒私風雲]]》中，[[大西洋縣|大西洋縣]]的財務主管[[努基·湯普森|努基·湯普森]]佔用了基於[[馬爾博羅-布倫海姆酒店|馬爾博羅-布倫海姆酒店]]的虛構版本的麗思卡爾頓酒店第8樓全層，這也是位於海濱大道上。基於這個角色的真正人物[[努基·強森|努基·強森]]，在[[禁酒時期|禁酒令]]和[[大蕭條|大蕭條]]期間控制著大西洋城<ref name = HBO />，佔用了套房來進行他的日常業務<ref name = ACexperience />，直到在1941年因[[逃稅|逃稅]]罪名被逮捕。他於1929年在麗思酒店和大使酒店主持了歷史性的[[黑社會|有組織犯罪]]首腦會議，「[[大西洋城會議|大西洋城會議]]」。為了響應電視劇的受歡迎程度，組織了參觀該大樓的活動。酒店的前服務生[[詹姆斯·H·博伊德_(政治人物)|詹姆斯·H·博伊德]]自1920年代以來就是強森的最高執行者，也是《酒私風雲》角色[[吉米·達摩狄|吉米·達摩狄]]的靈感來源。

=== 歷史地位 ===
2011年2月，新澤西州的州份歷史保護處（State Historic Preservation Office）授予該建築物一張「合資格證明書」，獲取資格列入{{link-en|新澤西州史蹟名錄|New Jersey Register of Historic Places|州份}}和[[國家史蹟名錄|聯邦史蹟名錄]]。

== 著名事件、客人和表演者 ==
1922年1月，總統[[沃倫·G·哈定|沃倫·G·哈定]]使用早期的無線電話向客人說話。<ref>{{Cite web |url = http://query.nytimes.com/mem/archive-free/pdf?res=9B06E2DC1130EE3ABC4A51DFB7668389639EDE |title = Hotel To Install Radio |date = 1922-01-22 |publisher = The New York Times |language = en}}</ref>同一年，[[麗思卡爾頓酒店管理集團|麗思卡爾頓公司]]在大西洋城和[[紐約市|紐約市]]的酒店之間提供機票。<ref>{{Cite web |url = http://query.nytimes.com/mem/archive-free/pdf?res=9B02E1DA1639EF3ABC4950DFB5668389639EDE |title = Plan Air Service to Atlantic City Ritz-Carlton to Give Passengers Choice of Land or Water Routes Regular daily air service between the Ritz-Carlton Hotels in New York and Atlantic City will be inaugurated in a few days |date = 1922-03-31 |publisher = The New York Times |language = en}}</ref>

著名客人包括表演者[[埃迪·康托爾|埃迪·康托爾]]、{{link-en|蘇菲·塔克|Sophie Tucker}}和{{link-en|勞倫斯·蒂貝特|Lawrence Tibbett}}；作家{{link-en|布魯斯·弗萊查爾德·巴頓|Bruce Fairchild Barton|布魯斯·巴頓}}；美國總統[[卡爾文·柯立芝|卡爾文·柯立芝]]、[[沃倫·G·哈定|沃倫·G·哈定]]和[[赫伯特·胡佛|赫伯特·胡佛]]；1920年代至30年代[[紐約市市長|紐約市市長]]{{link-en|吉米·沃克 (政治人物)|Jimmy Walker|吉米·沃克}}；黑幫[[艾爾·卡彭|艾爾·卡彭]]和[[查理·盧西安諾|查理·盧西安諾]]。<ref name = timeline />

在全盛時期，酒店的表演名人中有[[保羅·懷特曼|保羅·懷特曼]]、[[冰·哥羅士比|冰·哥羅士比]]、{{link-en|瑞德·尼可斯|Red Nichols}}和{{link-en|米爾頓·伯利|Milton Berle}}。
