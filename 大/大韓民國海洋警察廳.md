'''大韓民國海洋警察廳'''（{{朝|'''대한민국 해양경찰청'''|大韓民國海洋警察廳|Daehan-Minguk Haeyang-gyeongchal-cheong|}}，{{lang-en|'''Korea Coast Guard'''}}），是[[大韓民國|大韓民國]]為維持海上安全及治安而設置的機構，非戰爭時期曾屬[[海洋水產部|海洋水產部]]管轄，現為直屬韓國政府的獨立機關。韓國海洋警察廳在2010年起應付中國船艇非法跨越水域事故，決定加強黃海的韓國領海巡邏。海洋警察廳總部設於[[仁川市|仁川市]][[松島|松島]]東，共有7696名職員，進駐大韓民國沿海各海洋警察站。2014年5月19日，韩国总统[[朴槿惠|朴槿惠]]在就[[世越號沉沒事故|世越號沉沒事故]]向韩国国民发表的電視讲话中，宣布解散韩国海洋警察厅，韓國國會並於2014年11月7日修正通過「政府組織法」，將海洋警察廳的搜查和情報職能移交給警察廳，將海洋搜救、海洋警備職能移交給國民安全處（已取消並將職能移交給改組後的[[大韓民國行政安全部|行政安全部]]）所屬海洋警備安全本部，而大韓民國海洋警察廳亦於2014年11月19日0時起正式解散<ref name="dismiss">[http://news.xinhuanet.com/world/2014-05/19/c_1110744737.htm 新华网：朴槿惠为沉船事故再次道歉 决定解散海洋警察厅 2014-05-19]</ref><ref>[http://chinese.yonhapnews.co.kr/domestic/2014/05/19/0401000000ACK20140519001000881.HTML 韓聯網：朴槿惠表示沉船事故最终责任在于自己 2014-05-19]</ref><ref name="dismiss1">[http://news.tvb.com/world/545cdc946db28c013f000008 南韓國會通過解散海洋警察廳]</ref><ref name="dismiss2">[http://news.china.com.cn/2014-11/19/content_34094113.htm 南韓海洋警察廳正式解體]</ref><ref name="dismiss3">[http://chinese.korea.net/NewsFocus/Policies/view?articleId=123062  韩国设立国民安全处与人事革新处]</ref>。2017年7月20日，韓國國會通過文在寅政府所提案修正的政府組織法，將海警部門改組為政府直屬[http://gosi.kcg.go.kr/ 海洋警察廳]<ref>[http://chinese.joins.com/big5/article.aspx?art_id=169465&category=002002 韓國會通過「政府組織法」修訂案]</ref>。

== 歷史 ==
海洋警察廳的前身，海洋警察局成立於1953年12月23日的[[釜山|釜山]]，與此同時[[韓國警察廳|韓國警察廳]]成立一支水警隊。 1962年10月，海洋警察廳在[[仁川|仁川]]、[[麗水|麗水]]、[[浦項|浦項]]、[[群山市|群山市]]成立新行動基地; 其中航空隊由1963年2月至1980年代期間暫停運作。 從1980年代起海洋警察廳船隊編制擴張，及韓國警察廳水警隊於1991年8月改名為韓國國家水警局，直至2007年合併入海洋警察廳止。船隊編制包括數艘3000公噸位以上的艦隻。2008年5月，海洋警察局經過機構內部改組，及新成立「搜救保養課」。 21世紀初期，由於中韓雙方在[[黃海|黃海]]發生[[專屬經濟區|專屬經濟區]]爭議問題，韓國海警曾多次扣押與逮捕被指控為「越界」、「濫捕」的中國籍漁船與船員，因而發生多次外交論戰。未來目標包括正策劃在2015年前引進多艘5000公噸位艦隻，及擴大其[[非對稱戰術|非對稱戰術]]隊伍，鼓勵其他執法部門的參與。

2014年5月19日，韩国总统[[朴槿惠|朴槿惠]]在就[[世越號沉沒事故|世越號沉沒事故]]向韩国国民发表的電視讲话中，宣布解散韩国海洋警察厅，韓國國會並於2014年11月7日修正通過「政府組織法」，將海洋警察廳的搜查和情報職能移交給警察廳，將海洋搜救、海洋警備職能移交給國民安全處（已取消並將職能移交給改組後的[http://www.mois.go.kr 行政安全部]）所屬海洋警備安全本部，而大韓民國海洋警察廳亦於2014年11月19日0時起正式解散。

2017年7月20日，韓國國會通過[[文在寅|文在寅]]政府所提案修正的政府組織法，將海警部門改組為政府直屬[http://gosi.kcg.go.kr/ 海洋警察廳]。

== 目標 ==
* 發展和維持行動能力，確保大韓民國領海主權。 
* 獲認同為大韓民國海出保安的守護者。 
* 成為[[東北亞|東北亞]]最前衛的海上保安機構。 
* 發展和遍佈負責區內海上搜救反應能力。 
* 發展保存和保護海洋環境。 
* 全面迎合公眾請求。 
* 繼續改善海洋警察廳營運。 

== 主要職責 ==
* 航空及海事[[搜索及拯救|搜索及拯救]]
* 海上保安
* 海上環境保護──防止海上污染及防止污染海底生態。 
* 國際事務──快速阻止海上跨界罪行，包括視乎跨境罪行趨勢、非法入境者阻截。 
* 航運安全管理──提供海上旅遊和康樂空間一個無限保安。 
* 海上污染緊急應變

== 組織 ==
韓國海洋警察廳由一名廳長和一名副廳長率領。 並分為六個局和23個分區。 現時有13所警署、74間派出所、245間警崗。 另有一所[[韓國海洋警察學院|韓國海洋警察學院]]、科研中心、維修局。

==設備==
* 巡邏船
* 罪案應變艇
* 防污應變艇
* 6架定翼機、16架[[自轉旋翼機|自轉旋翼機]]

海洋警察廳使用四款1000[[公頓|公頓]]以上級別的巡邏船隻，三款250公頓以上級別的巡邏輪，和三款公頓級別[[馬達快艇|快速艇]]。 海洋警察廳亦有數艘特殊用途船隻，例如滅火輪、[[躉船|躉船]]、高速巡邏艇、[[氣墊船|氣墊船]]。目前最大的巡邏艇是5000噸級的[[三峰號|三峰號]]，負責守備和日本有主權問題的[[獨島|獨島]]。
