'''大岩桐'''（[[學名|學名]]：{{lang|la|''Sinningia speciosa''}}），園藝學上大岩桐園藝貿易的俗稱，是[[苦苣苔科|苦苣苔科]][[岩桐屬|岩桐屬]]的[[多年生植物|多年生]][[球根|球根]]花卉家族成員<ref name=xinnong>{{cite web |url=http://www.xinnong.com/jishu/zz/miaomu/1200468750.html |title=大岩桐栽培技术 |date=2008-01-16 |author=江苏农业信息网 |accessdate=2013-03-01 |language=zh-hans }}</ref>，

長大而鮮艷的花朵，所以常用作[[园艺植物列表|园艺植物]]。生長於[[南美洲|南美洲]][[巴西|巴西]]的热带草原<ref name=xinnong />。大岩桐通常在春、夏季開花，它的抗寒區為10-12<ref name=homeguides>{{cite web |url=http://homeguides.sfgate.com/propagate-gloxinia-21674.html |title=SFGate: How to Propagate Gloxinia |author=by Kristi Waterworth, Demand Media |language=en |accessdate=2013-03-01 }}</ref>。

== 分類 ==
在1817年於[[巴西|巴西]]發現大岩桐，並引進培殖，本來沿用相同的名稱，被歸類為[[大岩桐屬|大岩桐屬]]的植物，花商的大岩桐有時會用於區分根莖類的品種，但後來透過跟其他同屬植物比較，才知道當年的分類錯了，所以現時被歸類為岩桐屬植物。

== 種植 ==
大岩桐可由種子種植，但從發芽到開花，需時七個月，因此，一般種植都會從葉片或枝幹杄插來種植新株<ref name=homeguides />。它可生長出體積大、天鵝絨般的質感，並且顏色鮮豔的花朵，是一種受歡迎的室內植物。栽種環境的要求相似於[[非洲|非洲]][[紫羅蘭|紫羅蘭]]，不同的是，它通常更需要陽光及經常有休眠狀態，塊莖應保持乾爽，直至它重新發芽。
