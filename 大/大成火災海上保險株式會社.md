'''大成火災海上保險株式會社'''（{{jpn|j=大成火災海上保険|hg=たいせいかさいかいじょうほけん|rm=Taisei Kasai kaijō hoken|f=y}}）是第一家以[[臺灣|臺灣]]為基地的[[產物保險|產物保險]]公司，1920年由[[李延禧|李延禧]]、[[益子逞輔|益子逞輔]]發起成立，登記設立資本額為500萬日圓，[[實收資本額|實收資本額]]為125萬日圓。大成火災海上保險株式會社總部設於[[臺北市_(州轄市)|臺北市]][[表町_(臺北市)|表町]]2丁目11番地{{備註|即現今[[臺北站前商圈|臺北站前商圈]]的「站前大廈」所在地，地址是臺北市中正區館前路6號。}}，成立目的是在日本[[南進政策|南進政策]]下，以臺灣為根據地開拓[[中國南方|中國南方]]及東南亞保險市場。二戰結束後，該公司在臺資產由[[臺灣產物保險|臺灣產物保險]]接收，日本分公司則使用原名繼續營運，直到2002年併入{{link-ja|日本損害保險|損害保険ジャパン}}為止。{{r|連克|page=44}}<ref name="曾耀鋒">{{Cite journal|author=曾耀鋒|title=〈{{lang|ja|日本統治時代の台湾における大成火災の事業展開}}〉|journal=《{{lang|ja|日本台湾学会会報}}》|issue=15|pages=69-82|location=[[千葉市|千葉市]]|publisher=日本臺灣學會|year=2013|language=ja}}</ref>

== 設立 ==
日本在[[第一次世界大戰|第一次世界大戰]]期間，透過戰爭帶動國內產業，使經濟蓬勃發展；為了替國內產業開拓市場，日本政府將目標投向「南洋」地區（今日的[[東南亞|東南亞]]）。於是，在臺灣總督[[安東貞美|安東貞美]]、[[明石元二郎|明石元二郎]]推動下，臺灣進入「大正南進期」。總督府及[[臺灣銀行|臺灣銀行]]鼓勵以「臺資為主，日人指導」的集資模式，邀集日本、臺灣資本家共組企業開拓南洋地區與中國南方市場。大成火災海上保險株式會社就是在這個背景下成立的公司。<ref>{{Cite journal|author=中村孝志|others=李玉珍、卞鳳奎譯|title=〈大正南進期與臺灣〉|journal=《臺北文獻》|issue=132|pages=195-263|year=2000|language=zh-tw}}</ref>

1920年，大成火災海上保險株式會社在日本金融家[[益子逞輔|益子逞輔]]奔走下成立，受到[[李春生|李春生]]家族第三代[[李延禧|李延禧]]大力支持。李延禧是美國[[哥倫比亞大學|哥倫比亞大學]]商學院畢業的社會菁英，除了提供益子逞輔創業基金，還說服他父親[[李景盛|李景盛]]出任大成火保首任董事長。在李氏家族鼓勵下，眾多臺灣資本家也紛紛加入投資，[[板橋林家|板橋林家]]、[[霧峰林家|霧峰林家]]、[[高雄陳家|高雄陳家]]、[[鹿港辜家|鹿港辜家]]和[[基隆顏家|基隆顏家]]等臺灣望族更推派子弟出任大成火保董事、監察人等職務。{{r|連克|page=51-52}}

== 經營 ==
[[File:臺灣文化與民主運動領袖林獻堂_Lin_Hsien-tang,_Leader_of_Taiwanese_Culture_and_Democracy_Movement.jpg|250px]]曾是大成火保的重要股東之一，圖為霧峰林家當時的族長[[林獻堂|林獻堂]]。]]
1920年4月1日，臺灣總督府商工課准許大成火保銷售[[火災保險|火災保險]]契約，大成火保正式開業。開業初期，大成火保迅速掌握臺灣公部門建築物的火災保險業務，業績因此大幅成長。1921年結算時，大成火保業績已超越所有同業，包括那些在臺灣設有分公司的日本火災保險企業；在1926年至1929年間，大成火保的火災保單更有著20%的市佔率，可說是當時臺灣產物保險領域的龍頭公司。{{r|連克|page=52}}

大成火保的業績能如此迅速地發展，主要歸功於臺灣股東的人脈網絡。大成火保在全臺各地都有保險經紀公司（代理店）協助銷售保單；這些保經公司多半是由地方望族經營，擁有龐大細密的地方人際網絡，藉由中介地方人士購買大成火保的保單收取仲介費用營利。{{r|連克|page=52-53}}例如霧峰林家的[[林獻堂|林獻堂]]擔任「[[大東信託株式會社|大東信託株式會社]]」董事長時，便曾受大成火保請託，媒介霧峰-{庄}-長[[林夔龍|林夔龍]]與該公司談生意，使雙方談成霧峰-{庄}-役場與吳厝國民學校{{備註|今[[臺中市霧峰區四德國民小學|臺中市霧峰區四德國民小學]]。}}校舍的火災保險契約。<ref>{{Cite book|author=[[林獻堂|林獻堂]]|editor=[[許雪姬|許雪姬]]|title=《灌園先生日記（十三）一九四一年》|pages=135|year=2007|location=臺北市|publisher=中央研究院臺灣史研究所|language=zh-tw}}</ref><ref>{{Cite book|author=林獻堂|editor=許雪姬|title=《灌園先生日記（十五）一九四三年》|pages=75、121|year=2008|location=臺北市|publisher=中央研究院臺灣史研究所|language=zh-tw}}</ref>

在海外市場方面，大成火保雖是在南進政策下成立的公司；但比起南洋地區，股東卻更渴望打進廣大的日本火災保險市場。透過常務董事益子逞輔積極斡旋，{{Link-ja|日本農商務省|農商務省 (日本)}}於1922年2月17日核准大成火保在東京設立分公司，同年5月1日正式開業。有別於臺灣總公司剛開業便迅速成長，日本分公司要等到1923年[[關東大地震|關東大地震]]後，業績才顯著提升。{{r|連克|page=57}}

關東大地震雖導致許多火災事故，但日本火災保險公司依法毋須理賠受災保戶的損失，因為相關損失金額足足是當時所有日本火災保險公司總資產的10倍。而大成火保卻率先以保險金額的1成作為慰問金，發放給受災保戶，一舉打響企業知名度。到了1924年8月，大成火保日本分公司的有效契約金額已超過1億日圓；除了社會知名度帶來的收益，大成火保因為沒有加入[[同業公會|同業公會]]，得以採用較實際的[[風險評估|風險評估]]方法，使其損失遠低於其它同業，業績收益也就顯著提升。{{r|連克|page=57-60}}

== 終戰之後 ==
[[第二次世界大戰|第二次世界大戰]]結束後，[[臺灣省行政長官公署|臺灣省行政長官公署]]委託大成火災海上保險株式會社接收在臺日資火災保險公司資產，以此為基礎成立「[[臺灣產物保險|臺灣產物保險]]」。大成火保臺灣總公司與其它日資保險公司在臺業務正式結業<ref>{{Cite book|author=吳若予|title=《戰後臺灣公營事業之政經分析》|pages=225|year=1992|publisher=業強出版社|language=zh-tw}}</ref>。至於大成火保在日本的公司法人，則於1950年以原名重新辦理公司登記，由{{link-ja|第一銀行 (日本)|第一銀行|日本第一銀行}}與{{link-ja|朝日生命保險|朝日生命保険}}共同出資成立，獨立營運至2002年併入{{link-ja|日本損害保險|損害保険ジャパン}}為止<ref name="曾耀鋒"/>，為今日[[損害保險日本興亞|損害保險日本興亞]]的前身之一。
