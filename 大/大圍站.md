'''大圍站'''（{{lang-en|'''Tai Wai Station'''}}）位於[[香港|香港]][[沙田區|沙田區]][[大圍|大圍]]，是[[港鐵|港鐵]][[東鐵綫|東鐵綫]]與[[馬鞍山綫|馬鞍山綫]]的[[换乘站|轉車站]]。此車站的擴建工程由建築師[[Aedas|Aedas]]負責設計，[[前田建設|前田建設]]為總承建商，扩建的车站大堂已于2003年12月启用，面积增至原来的三倍。

==結構==
大圍站是一個架空車站，大堂設於地面，並連接A－D出入口，而1及2號月台南端亦設有E－G出入口往返路面。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Tai Wai Station 2017 08 part12.jpg | caption1 = 月台層轉車通道
}}
大圍站設有4個[[側式月台|側式月台]]，位於大堂之上。其中3號月台和2號月台間設有4條斜道連接，使馬鞍山綫的南行乘客可在轉乘東鐵綫往市區時無須轉換樓層，斜道設計更方便傷殘人士使用。然而，由馬鞍山綫轉乘東鐵綫往[[羅湖站_(香港)|羅湖站]]／[[落馬洲站|落馬洲站]]或由東鐵綫轉乘馬鞍山綫時，必須經大堂方可轉乘列車。為方便乘客由馬鞍山綫轉乘東鐵綫（往紅磡）列車，因此馬鞍山綫採用'''右上左落'''的行駛方向，與港鐵其他路綫不同。

東鐵綫1號月台南面設有E、G出入口，而2號月台南面則設有F出入口。兩個月台出入口處均有客務中心，而2號月台亦設有失物/學生乘車優惠辦事處。1號月台北面有一部自動櫃員機。

==使用狀況==
由於大圍站目前是東鐵綫及馬鞍山綫唯一的轉車站，加上附近也有多個大型屋苑及大量出租[[單車|單車]]的店舖，因此不論平日還是假日，均有大量人流，人流密集導致大圍站在繁忙時段十分擠擁。預計在[[沙中綫|沙中綫]]大圍至紅磡段通車後，才能有效分流乘客。<ref>{{cite web |author1=立法會財務委員會 |title=財務委員會 工務小組委員會參考文件 補充資料 51TR－ 沙田至中環線 － 設計及地盤勘測 |url=http://www.legco.gov.hk/yr07-08/chinese/fc/pwsc/papers/pi08-05c.pdf |accessdate=2018-09-09}}</ref><ref>{{cite web |author1=香港旅遊發展局 |title=沙田至大美督 |url=http://www.discoverhongkong.com/tc/see-do/great-outdoors/cycling/sha-tin-to-tai-mei-tuk.jsp |website=香港旅遊發展局 |accessdate=2018-09-09}}</ref><ref>{{cite news |title=立法會二十一題：東鐵線 |url=http://www.info.gov.hk/gia/general/201711/15/P2017111500474.htm |accessdate=2018-09-09 |agency=政府新聞網 |date=2017-11-15}}</ref>

由於大圍站距離[[沙田車公廟|車公廟]]較[[車公廟站_(香港)|車公廟站]]近，因此在[[農曆新年|農曆新年]]期間，市民會選擇由大圍站步行前往車公廟參拜，警方會因應情況在大圍站實施人潮管制。<ref>{{cite news |title=車公廟超旺 採人潮管制 避免混亂 |url=http://www.orangenews.hk/news/system/2015/02/22/010008074.shtml |accessdate=2018-09-09 |publisher=橙新聞 |date=2015-02-22}}</ref>

每當上下班繁忙時間之後，部份馬鞍山綫的港鐵列車在本站3號月台清客後，便會暫停服務，返回大圍車廠。

==歷史==
*1983年8月15日，大圍臨時站開設，位於現時大圍工業區對開隔音屏位置的[[九廣鐵路|九廣鐵路]]（英段）臨時車站，車站主體是用木材搭建（俗稱「木頭站」）。<ref>{{cite web |title=1983年大圍臨時火車站 (木頭站) |url=https://www.discuss.com.hk/viewthread.php?tid=14814648 |accessdate=2018-09-09}}</ref>
*1986年4月23日，位於臨時站以南現時富嘉花園對開位置的大圍站啟用，當時車站只有兩個相對的出入口（現時的D出入口及客務中心所在位置）。
*1993年12月，位於2號月台近單車公園（今港鐵大圍車廠）的大堂（現時的F出入口）啟用。
*1995年，車站進行改善工程，新建兩組扶手電梯。更改入閘區域，使升降機毋須職員協助使用。
*2001年，馬鞍山綫動工，大圍站週邊開始進行擴建工程。

*2003年12月24日，大圍站擴建部份局部性啟用，當中只有非馬鞍山綫主體建築部份啟用，今天的A、B、C出入口啟用。原有的大堂同日封閉進行改建工程，並將車站主題顏色由[[綠色|綠色]]變成[[深藍色|深藍色]]。隨著馬鞍山綫通車，餘下的大堂部份及月台轉乘通道啟用。
*2005年4月，新建的大圍站公共運輸交匯處啟用，B出入口重開。同年，D出入口週邊翻新完成重開。位於1號月台近海福花園的大堂（現時的E、G出入口）啟用。
*2009年，東鐵綫各月台進行較低成本的翻新工程，以遮掩東鐵綫車站月台殘舊的觀感。工程包括重新為各車站配主題色，並加入花的代表，為月台橫樑及柱子翻上月台配色的油漆，貼上印有車站主題花的圖案、車站附近/主題建築物/街道、車站名書法字的牆紙，加設/更新直立站牌、路線圖、指示牌、告示等等。此站配色為深藍色，主題花為鬱金香，街景為沙田車公廟。
*2011年5月，大圍站擴建部份進行首次翻新工程，工程包括在非收費區增建商店，包括[[Subway|Subway]]、[[奇華餅家|奇華餅家]]及日式[[便當|便當]]外賣店Bento Express by JUSCO。工程在2012年6月完成。

===事故===
*2014年2月25日上午6時許，車站側近美田路[[大快活|大快活]]快餐店的廚房，疑因油煙槽油污滴落爐頭引起搶火，職員撲救無效慌忙報警。其間大量濃煙湧出，令樓上月台、路軌、快餐店內外，以至美田路都煙霧瀰漫。店內食客慌忙捧着餐盤走出店外，在對開空地趕吃未完的早餐；月台的乘客一度誤以為又有列車事故，紛紛掩鼻走避；列車駛至時，車上乘客亦大吃一驚，但列車服務未受影響。消防員接報到場，食店天花的灑水系統已開啟，消防員開喉迅速將火撲熄，其間10多名員工及40名食客自行疏散，幸無人受傷。由於大堂水浸，港鐵一度需封閉大圍站C出入口，至早上9時35分始重開。<ref>[http://hk.apple.nextmedia.com/news/art/20140226/18638158 快餐店起火　煙攻月台 蘋果日報 (2014年2月26日) ]</ref>
*2014年3月30日，受到[[黑色暴雨警告信號|黑色暴雨警告信號]]發出的影響，此站與[[黃大仙站|黃大仙]]、[[荃灣西站|荃灣西]]、[[九龍塘站_(香港)|九龍塘站]]一樣出現水浸，雨水湧進露天路軌。傍晚9時47分，D出入口暫時關閉，乘客需要使用C出入口。當時東鐵綫要維持有限度服務，紅磡至大圍為12分鐘一班，大圍至羅湖則為8分鐘一班<ref>[https://www.facebook.com/mtrupdate?fref=ts MTR Service Update-2225 東鐵綫水浸服務安排]</ref>。

==爭議事件==
[[File:Anti_Tai_Wai_Station_Property_Development_Protest_20121216.JPG|200px]]規劃]]
有港鐵前員工指出，大圍站向來屬執法嚴謹的車站，因負責執行港鐵附例的查票組，其辦公室正是位處該站，查票組無論出動或上下班也會經過該站，故市民在大圍站被他們截查的的機會率也最高。<ref>{{cite news|title=【古箏學生妹】大圍站執法嚴 前港鐵員工揭露原因|url=http://hk.apple.nextmedia.com/realtime/news/20150917/54217267|newspaper=蘋果日報|date=2015-09-17}}></ref>

===市民在站內外反對大圍站上蓋規劃 被要求離開===
2012年12月16日中午，有市民在大圍站大堂進行「無聲抗議」，反對[[大圍站上蓋發展項目|大圍站上蓋發展項目]]規劃，自稱站長的男士表示市民阻礙著大堂通道，不可在港鐵範圍內展示標語。其後在站外落地玻璃幕牆抗議，亦被7至8名港鐵職員包圍，港鐵職員指該地面範圍都屬港鐵範圍。<ref>[http://www.vjmedia.com.hk/articles/2012/12/18/26795 寫在大圍站反屏風樓「無聲抗議」之後 反大圍站上蓋建屏風樓關注組 2012年12月18日]</ref>

===女學生攜帶古箏事件===
[[File:Tai_Wai_Station_Students_Warned_by_MTR_staff_20150915.jpg|200px]]
2015年9月16日，facebook「大圍聯盟」群組於15日晚上載一張照片，一名身穿校服的女學生，背著一個與她身高相約的古箏，站在港鐵大圍站。當時，她正被3名港鐵職員包圍，疑因攜帶體積大的深啡色古箏盒，而遭截停查問。有網友指，事件發生於傍晚6時許，該名女生被指古箏太大，不獲進入月台。該相片在網上瘋傳，網民紛紛「洗版式」抨擊港鐵處理乘客攜帶大型物件的條例時，「雙重標準」及「執法不公」。他們以水貨客事件作比較，指港鐵對水貨客寬容，卻嚴禁乘客攜帶大型樂器上車。當中更有不少音樂愛好者坦言「身同感受」，促港鐵檢討現行條例。有網民亦隨即在[[facebook|facebook]]上載多張攝於港鐵車廂的相片，相中可見乘客帶著關刀、長槍及洗衣機等上車，但未見有職員阻止，有網民質疑港鐵雙重標準，欺善怕惡。<ref>{{cite news|title=女生揹古箏就截洗衣機唔截　港鐵被轟大細超|url=http://hk.on.cc/hk/bkn/cnt/news/20150916/bkn-20150916101210161-0916_00822_001.html|newspaper=on.cc 東網|date=2015-09-16}}></ref>

===用軟尺度男子手杖長度===
2015年9月29日，有網民發現大圍站月台見到有懷疑[[視障人士|視障人士]]拿著一支杖求助，高呼「都冇人嚟幫我嘅」，其後先有一名女職員問是否要乘車。惟女職員入了月台室後，一名男職員表示該杖需短於130厘米為最好，之後就拿出軟尺量度。港鐵職員的做法引來網民強烈反彈，認為事件荒謬。<ref>{{cite news|title=大圍站港鐵職員走火入魔　竟用軟尺度男子手杖長度|url=http://www.passiontimes.hk/article/09-29-2015/25973|newspaper=熱血時報|date=2015-09-29}}></ref>

===過百名市民於大圍站大堂示威 抗議禁帶樂器===
{{main|號召各界攜帶大型樂器搭東鐵}}
[[File:Tai_Wai_Station_Protest_Overview_20151003.jpg|200px]]
2015年10月3日傍晚6時許，超過一百名市民於大圍站大堂示威，抗議港鐵禁止攜帶大型樂器及縱容[[水貨客|水貨客]]，當中數十名為音樂人，分別㩦帶大型樂器如[[大提琴|大提琴]]、[[結他|結他]]、[[手風琴|手風琴]]到場，成功入閘乘車。抗議亦吸引大批市民到場，對港鐵執法不一表示不滿，認為港鐵放生水貨客，部分更指罵港鐵職員，有人更在大堂地下寫上港鐵「擾亂民生 打壓藝術」紅色大字，亦有人揮動[[港英旗|港英旗]]，稱寧要「港英年代的地鐵」。到接近7時，有樂手各自在大堂不同地方拿起樂器演奏，吹奏出《[[龍的傳人|龍的傳人]]》和《鳳凰花鼓》，贏到不少掌聲及喝采，成為全場焦點。到8時許人群散去。<ref>{{cite news|title=抗議港鐵截樂器 逾百人大圍站示威|url=http://news.mingpao.com/pns/dailynews/web_tc/article/20151004/s00001/1443894509194|newspaper=明報|date=2015-10-03}}></ref>
