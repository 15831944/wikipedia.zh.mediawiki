'''大峰山'''（おおみねさん）是位在[[奈良縣|奈良縣]]南部的[[山|山]]。現在廣義指[[大峰山脈|大峰山脈]]、狹義指'''山上岳'''（さんじょうがたけ）。歷史上的大峰山指大峰山脈的山上岳南邊的小篠到[[熊野|熊野]]的山峰。相對的，小篠包含山上岳沿稜線到[[紀之川|吉野川]]河岸稱作[[金峰山|金峰山]]。

== 概要 ==
吉野到熊野的[[大峯奧駈道|大峯奧駈道]]自古就是來自藤原、平城的僧侶（[[山伏|修驗者]]）為修行所開闢的道路。熊野修驗擴張勢力之際，[[長久_(年號)|長久年間]]（1040年 - 1044年），修驗者（義叡、長圓）為熊野到吉野的大峯奧駈道設立體系。熊野到吉野的參詣稱作順峯。吉野到熊野的參詣稱作逆峯。

這一帶在1936年（[[昭和|昭和]]11年），指定為[[吉野熊野國立公園|吉野熊野國立公園]]，2004年（[[平成|平成]]16年）7月，史跡「[[大峯山寺|大峯山寺]]」、史跡「[[大峯奧駈道|大峯奧駈道]]」等作為「[[紀伊山地的靈場和參拜道|紀伊山地的靈場和參拜道]]」之一部登錄為[[聯合國教科文組織|聯合國教科文組織]]的[[世界遺産|世界遺産]]。

[[深田久彌|深田久彌]]的[[隨筆|隨筆]]『[[日本百名山|日本百名山]]』和各種一覽表中，大峰山（1,915m）是採用廣義的大峰山[[最高峰|最高峰]]「[[八經岳|八經岳]]」（八劍山）的標高。

== 山上岳 ==
[[標高|標高]]1,719m。位在奈良縣[[吉野郡|吉野郡]][[天川村|天川村]]。

這一帶自古就是作為[[修驗道|修驗道]]之山的山伏修行場。作為道場的大峯山意指[[吉野山|吉野山]]延續到熊野的山脈全體。之中有山上岳（舊名：金峯山）山頂付近的修驗道之根本道場[[大峯山寺|大峯山寺]]山上藏王堂，山全體被視為聖域，現在也維持[[女人禁制|女人禁制]]。通往山上岳的登山道因宗教上的理由，有傳達女人禁制要旨的大門（女人結界門），並設置協力維護1300年傳統的依賴看板。

== 稻村岳（女人大峯） ==
標高1,726m。位在奈良縣[[吉野郡|吉野郡]][[天川村|天川村]]。

包括本峰和大日山之2峰。山上岳維持女人禁制，但是相隣的稻村岳作為女性信者的修行場在1959年（昭和34年）開放，所以也被稱作「女人大峯」。

== 八經岳 ==
標高1,915m。位在奈良縣[[吉野郡|吉野郡]][[天川村|天川村]]和[[上北山村|上北山村]]的境界。

八經岳（八劍山或佛經岳）是近畿地方的最高峰。
