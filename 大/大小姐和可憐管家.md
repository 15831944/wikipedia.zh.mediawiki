{{日本作品|大小姐和可憐管家|お嬢様と憐れな（こ）執事}}是由SkyFish公司旗下品牌Carol Works製作於2017年5月26日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]<ref>{{cite web|script-title=お嬢様と憐れな（こ）執事 特集|url=http://www.game-style.jp/special/201704/06/00awa_cw.php|website=Game-Style|language=ja|date=2017年4月6日|accessdate=2017年10月4日}}</ref><ref>{{cite news|script-title=【発売直前！】Carol Worksの新作はお嬢様たちのお世話をする美少年を描いたラブコメADV！|url=http://www.tgsmart.jp/article.aspx?a=19609|agency=TG Smart|date=2017年5月16日|accessdate=2017年10月4日|language=ja}}</ref>。

==故事==
轟六道因為父親轟天道的關係導致兩人被迫流落街頭，但是在叔父的幫忙下住進叔父的家並且成為管家，但是不久之後因為大小姐轟火凛的任性要求六道被迫裝扮女裝轉學到聖絢女學院。後來因為三位大小姐們都希望六道成為自己的專屬管家，使得六道只好做出抉擇。

===主要角色===
; {{nihongo|轟六道|轟 六道（とどろき りくどう）||聲優：美咲桃子}}
:本作的男主角，因為父親的關係而擔任管家，後來由於火凛的任性要求而被迫裝扮女裝轉學到聖絢女學院成為火凛的同學。
; {{nihongo|轟火凜|轟 火凛（とどろき かりん）||聲優：{{lang|ja|香山いちご}}}}
:血型：A型，身高：156cm，三圍：B91/W57/H88<ref>{{cite web|script-title=お嬢様と憐れな（こ）執事|url=http://www.getchu.com/soft.phtml?id=940973|website=Getchu.com|language=ja|accessdate=2017年10月4日}}</ref>
:轟三姊妹的長女，擔任學生會長。平時在外對六道態度冷淡，實際上很喜歡他，在兩人獨處就會對他展開熱情追求。為了增加兩人相處時間而向父親提出將六道轉學到聖絢女學院的要求。
; {{nihongo|轟水萌|轟 水萌（とどろき みなも）||聲優：{{lang|ja|歩サラ}}}}
:血型：AB型，身高：160cm，三圍：B94/W59/H90
:轟三姊妹的次女，夢想能成為保育士而隱瞞家人在越前保育園打工。喜歡六道並且有拿他衣物去聞的癖好。
; {{nihongo|轟土筆|轟 土筆（とどろき つくし）||聲優：{{lang|ja|琉花あか}}}}
:血型：B型，身高：151cm，三圍：B88/W54/H83
:轟三姊妹的三女。
; {{nihongo|音無光子|||聲優：{{lang|ja|美空なつひ}}}}
:血型：A型，身高：162cm，三圍：B92/W60/H89
:服侍轟家的女僕。

===其他角色===
; {{nihongo|轟天道|轟　天道（とどろき てんどう）||聲優：桂艶丸}}
:六道的父親，過去原本是大企業的社長，由於亂花錢的壞習慣而被罷職流落街頭，後來被雷道收留後擔任男僕工作。
; {{nihongo|轟雷道|轟　雷道（とどろき らいどう）||聲優：桂艶丸}}
:天道的雙胞胎弟弟，對於女兒們處在叛逆期而很少對話感到煩惱，因此委託六道幫忙改善父女之間關係。
; {{nihongo|桐生寺茉莉|桐生寺 茉莉(きりゅうじ まつり)||聲優：shizuku}}
:六道的同學，擔任學生副會長。十分溺愛弟弟悠的弟控，常常妄想奪走弟弟的貞操。
; {{nihongo|小山內虎子|小山内 虎子（おさない とらこ）||聲優：鹿瀬紫卯}}
:六道的同學，火凛的崇拜者。
; {{nihongo|越前和華|越前 和華（えちぜん わか）||聲優：薫子}}
