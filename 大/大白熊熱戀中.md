《'''大白熊熱戀中'''》（{{lang-ja|恋するシロクマ}}）由[[日本漫畫家|日本漫畫家]]{{lang|ja|[[ころも|ころも]]}}所作的[[日本漫畫|日本漫畫]]作品。於《[[月刊Comic_Gene|月刊Comic Gene]]》2015年7月號雜誌上持續連載至今。

本作是以狩獵者和獵物之間所展開的[[北極熊|北極熊]]與[[海豹|海豹]]男同性戀愛主題作品<ref>{{Cite web |date= 2017-02-26|url= http://eiga.com/news/20170226/8/|title=ショートアニメ「恋するシロクマ」3月4日から全国の劇場で予告編前に上映スタート|publisher={{lang|ja|{{link-ja|映画.com|映画.com}}}}|accessdate=2017-03-19|language=ja}}</ref>。

== 登場角色 ==
; {{nihongo|海豹君|アザラシ君}}
: 聲－[[花江夏樹|花江夏樹]]
: 對於白熊先生的求愛發言，時常感到害怕而顫抖不已。
; {{nihongo|白熊先生|シロクマさん}}
: 聲－[[梅原裕一郎|梅原裕一郎]]
: 雖然性別相同但是種族與海豹君不同，但卻對牠一見鐘情。
; {{nihongo|凱西|キャシー}}
; {{nihongo|虎鯨|シャチ}}
: 聲－[[大谷亮平|大谷亮平]]
: 被凱西喜歡著。位於食物鏈的頂點，個性沉默。

== 劇場動畫 ==
本作於2017年3月4日在全國大約50家電影院播出上映之前的預告片<ref>{{Cite web |date= 2017-02-15|url= http://www.animatetimes.com/news/details.php?id=1487117234|title=花江夏樹さん·梅原裕一郎さんW主演『恋するシロクマ』、劇場にてぷちアニメ化決定！アフレココメントも到着|publisher={{link-ja|animate Times|アニメイトタイムズ}}|accessdate=2017-03-19|language=ja}}</ref>。每部短片時間大約1分鐘。3月1日曾於[[YouTube|YouTube]]上先行公開動畫短片。
