'''大月JCT'''（平假名：おおつきジャンクション）是位於[[山梨縣|山梨縣]][[大月市|大月市]]的[[中央自動車道|中央自動車道]]本線與富士吉田線之[[系統交流道|系統交流道]]。

== 連接道路 ==
*[[中央自動車道|中央自動車道]]本線
*中央自動車道富士吉田線

== 結構 ==
[[File:Otsuki.svg|File:Otsuki.svg]]

== 歷史 ==
* 1969年3月17日 - [[中央自動車道|中央自動車道]][[相模湖IC|相模湖IC]]至[[河口湖IC|河口湖IC]]開通。
* 1977年12月20日 - 中央自動車道[[勝沼IC|勝沼IC]]至大月JCT開通，此系統交流道也同時啟用。
* 1982年9月20日 - 勝沼至河口湖方向之匝道開通。
* 2003年 - 改善工程完成，此系統交流道與[[大月IC|大月IC]]一體化。
