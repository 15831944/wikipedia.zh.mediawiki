'''大臣補佐官'''（{{jpn|j=大臣補佐官|hg=だいじんほさかん}}）是2014年（[[平成|平成]]26年）5月30日《國家公務員法等部分改正法律》（平成26年法律第22號）施行時，於[[內閣府|內閣府]]、[[復興廳|復興廳]]、各[[日本行政機關|省]]新設的非必要{{link-ja|特別職|特別職}}[[官職|官職]]。其職務由《{{link-ja|內閣府設置法|内閣府設置法}}》第14條之2、《{{link-ja|復興廳設置法|復興庁設置法}}》第10條之2、《{{link-ja|國家行政組織法|国家行政組織法}}》第17條之2規定，受各省[[國務大臣|大臣]]命令，輔佐各省首長大臣進行特定政策相關企劃、立案及政務（內閣府設置法與復興廳設置法的「該省首長大臣」以「[[內閣官房長官|內閣官房長官]]或[[內閣府特命擔當大臣|特命擔當大臣]]」、「[[復興大臣|復興大臣]]」取代） <ref>{{Cite web |url=http://law.e-gov.go.jp/htmldata/H11/H11HO089.html |title=内閣府設置法 |access-date=2018-09-16 |archive-url=https://web.archive.org/web/20170803124449/http://law.e-gov.go.jp/htmldata/H11/H11HO089.html |archive-date=2017-08-03 |dead-url=yes }}</ref><ref>[http://law.e-gov.go.jp/htmldata/H23/H23HO125.html 復興庁設置法]</ref><ref>{{Cite web |url=http://law.e-gov.go.jp/htmldata/S23/S23HO120.html |title=国家行政組織法 |access-date=2018-09-16 |archive-url=https://www.webcitation.org/67a4jQCxc?url=http://law.e-gov.go.jp/htmldata/S23/S23HO120.html |archive-date=2012-05-11 |dead-url=yes }}</ref>。《{{link-ja|內閣法|内閣法}}》中設於[[內閣官房|內閣官房]]的大臣補佐官稱為{{link-ja|內閣總理大臣補佐官|内閣総理大臣補佐官}}。

== 概要 ==
由規定特別職的《{{link-ja|國家公務員法|国家公務員法}}》第2條之3規定，大臣補佐官地位在[[大臣政務官|大臣政務官]]之下、《{{link-ja|人事院規則}}》所指定之{{link-ja|內閣總理大臣秘書官|内閣総理大臣秘書官}}及{{link-ja|秘書官|秘書官|國務大臣秘書官}}等機關首長秘書官（特別職）<ref>{{link-ja|人事院總裁|人事院総裁}}秘書官、{{link-ja|會計檢查院長|会計検査院長}}秘書官、{{link-ja|內閣法制局長官|内閣法制局長官}}秘書官、{{link-ja|宮內廳長官|宮内庁長官}}秘書官</ref>之上<ref>{{Cite web |url=http://law.e-gov.go.jp/htmldata/S22/S22HO120.html |title=国家公務員法 |access-date=2018-09-16 |archive-url=https://web.archive.org/web/20060906012138/http://law.e-gov.go.jp/htmldata/S22/S22HO120.html |archive-date=2006-09-06 |dead-url=yes }}</ref>。

大臣補佐官的任免，內閣府與復興廳由[[內閣總理大臣|內閣總理大臣]]、各省由各大臣提出，[[內閣_(日本)|內閣]]決行。內閣府的大臣補佐官員額在6人以內，復興廳與各省大臣補佐官員額為1人以內。可由國會議員兼任，也可為專職或[[兼職|兼職]]。專職的大臣補佐官月薪與{{link-ja|內閣官房副長官補|内閣官房副長官補}}、{{link-ja|內閣廣報官|内閣広報官}}及{{link-ja|內閣情報官|内閣情報官}}、專職的{{link-ja|內閣總理大臣補佐官|内閣総理大臣補佐官}}同等<ref>{{Cite web |url=http://law.e-gov.go.jp/htmldata/S24/S24HO252.html |title=特別職の職員の給与に関する法律 |access-date=2018-09-16 |archive-url=https://web.archive.org/web/20171002223455/http://law.e-gov.go.jp/htmldata/S24/S24HO252.html |archive-date=2017-10-02 |dead-url=yes }}</ref>。

大臣補佐官制度在2014年（平成26年）9月正式導入於[[第2次安倍內閣_(改造)|第2次安倍內閣 (改造)]]<ref>{{Cite web|url=http://megalodon.jp/2016-0706-0016-49/gendai.ismedia.jp/articles/-/40381|title=新設ポストの大臣補佐官に期待はできるか|date=2014-09-14|accessdate=2016-07-06|publisher=現代ビジネス}}</ref>。
