'''大连市公安局巡警支队女子骑警大队'''，为[[中华人民共和国|中华人民共和国]]辽宁省[[大连市|大连市]]的一支骑警队，1994年12月设立，其任务有治安、礼仪等，曾被誉为“华夏警花第一骑”，<ref>{{Cite book|title=辽海重地辽宁 1|last=周力主编|first=|publisher=北京：[[中国旅游出版社|中国旅游出版社]]|year=2015.04|isbn=978-7-5032-5189-4|location=|pages=158-159}}</ref>但却曾因开销过大被指责为“[[花瓶#引申义|花瓶]]”。

== 历史 ==
1994年12月，时任[[大连市|大连市]]市长的[[薄熙来|薄熙来]]设立了大连市骑警队伍，后该警队发展为中国的第一支女子骑警队，[[韩玉春|韩玉春]]负责完成了骑警队的形象标志设计。<ref>{{Cite book|title=纪念《通化日报》创刊60周年丛书之二 天南地北通化人|last=张恩喜主编|first=|publisher=[[通化日报社|通化日报社]]|year=2005.09|isbn=|location=|pages=239}}</ref>自成立以来，该骑警队便被中国大陆媒体评价为“滨城靓丽的风景线”、“大连市的名片”、“华夏警花第一骑”，<ref>{{Cite book|title=辽宁|last=周力主编|first=|publisher=北京：中国旅游出版社|year=2013.01|isbn=978-7-5032-4587-9|location=|pages=154}}</ref>也引来了其他城市的效仿，截止2013年共13座城市设置了自己的骑警队。<ref name=bbc2013>{{cite web |title=薄熙来创办大连女骑警被指劳民伤财 |url=https://www.bbc.com/zhongwen/simp/china/2013/05/130527_china_dalian_mounted_police |website=[[BBC|BBC News 中文]] |accessdate=2019-01-25 |archiveurl=https://web.archive.org/web/20190125210022/https://www.bbc.com/zhongwen/simp/china/2013/05/130527_china_dalian_mounted_police |archivedate=2019-01-25 |language=zh-Hans |date=2013-05-27| deadurl=no}}</ref><ref name="sohu-dianji1152">{{cite web |title=点击今日第1152期：大连女骑警抓了多少贼？|url=http://news.sohu.com/s2013/dianji-1152/ |website=news.sohu.com |accessdate=2019-01-25 |archiveurl=https://web.archive.org/web/20190111132830/http://news.sohu.com/s2013/dianji-1152/ |archivedate=2019-01-11|language=zh-cn| deadurl=no}}</ref><ref name=wuhan2004>{{cite web |author1=易笑 |author2=刘水清 |author3=潘峰 |author4=夏国清 |author5=李江林 |author6=胡汉生 |title=骑警会亮相江城吗？(图) |url=http://news.sohu.com/2004/05/24/55/news220235549.shtml |website=news.sohu.com |publisher=[[武汉晚报|武汉晚报]] |accessdate=2019-01-25 |archiveurl=https://web.archive.org/web/20190112095814/http://news.sohu.com/2004/05/24/55/news220235549.shtml |archivedate=2019-01-12 |language=zh-cn |date=2004-05-24| deadurl=no}}</ref><ref>{{cite web |author1=张苗凤 |title=盘点全国8支女骑警队 昆明女子骑巡队已被低调撤销 |url=https://www.guancha.cn/society/2013_05_29_147838.shtml |website=www.guancha.cn |publisher=[[新京报|新京报]] |accessdate=2019-01-25 |archiveurl=https://web.archive.org/web/20190125230215/https://www.guancha.cn/society/2013_05_29_147838.shtml |archivedate=2019-01-25 |date=2013-05-29 |deadurl=yes }}</ref>

1994年成立之初，骑警大队为副处级建制，隶属大连市公安局巡警支队。当时配备警马11匹，警员18名（均为男性）。1996年，巡警支队从女警大队抽调了8名女警担任骑警。1998年又招收了15名18岁到20岁的女性。1999年，骑警大队与女警大队合并为大连市公安局巡警支队女子骑警大队。<ref name="chinaorg">{{cite web |author1=宋广辉 |title=大连骑警每匹警马日花费80元 被指长官意志产物|url=http://china.org.cn/chinese/2013-05/28/content_28953145_4.htm |website=china.org.cn |publisher=[[中国青年报|中国青年报]] |accessdate=2019-01-25 |archiveurl=https://web.archive.org/web/20130528214638/http://china.org.cn/chinese/2013-05/28/content_28953145_4.htm |archivedate=2013-05-28 |date=2013-05-28| deadurl=no}}</ref>2004年时，大连市女骑警队拥有78匹警马，其中10余匹[[蒙古马|蒙古马]]、前苏联种马，系骑警建队之初购买，剩下的马匹均为[[香港赛马会|香港赛马会]]所赠的英国纯种马。在47名民警中有39名女性民警（其中女骑警30人），此外也从国外聘请了骑马教练。<ref name=wuhan2004 />截至2013年，女子骑警大队共招收了7代女警，有骑警65人，警马100匹（均来自香港赛马会所赠）。<ref name="chinaorg" />

骑警队还设有对外开放营运的大连女子骑警大队训练基地，总占地面积3.6万平方米，其中有周长415米的跑马场、2500平方米的办公楼和6000平方米的表演场，1999年开始兴建，2000年投入使用，2001年对外开放。<ref name="chinaorg" /><ref name=chinanews>{{cite web |author1=姜思羽 |author2=庄惠琳 |title=女子骑警队：大连人眼中“有用的花瓶” |url=http://www.chinanews.com/sh/2013/05-28/4866714.shtml |website=www.chinanews.com |publisher=[[中国新闻网|中国新闻网]] |accessdate=2019-01-25 |archiveurl=https://web.archive.org/web/20161112092352/http://www.chinanews.com/sh/2013/05-28/4866714.shtml |archivedate=2016-11-12 |language=zh-cn |date=2013-05-28| deadurl=no}}</ref><ref name="yicai">{{cite web |title=大连女子骑警基地涉嫌违法经营 三千万收入流向存疑 |url=https://www.yicai.com/news/2742179.html |website=www.yicai.com |publisher=[[第一财经|第一财经日报]] |accessdate=2019-01-25 |archiveurl=https://web.archive.org/web/20190125223251/https://www.yicai.com/news/2742179.html |archivedate=2019-01-25|date=2013-05-30|language=zh-cn| deadurl=no}}</ref>

骑警队曾先后获得“巾帼文明示范岗”、“青年文明号”、“全国百佳班组”等荣誉。<ref>{{cite web |author1=林基波 |author2=周贺 |title=苦、累之后是骄傲 大连女骑警讲述“靓丽”故事 |url=http://www.people.com.cn/GB/shehui/43/20020227/675010.html |website=www.people.com.cn |publisher=[[辽沈晚报|辽沈晚报]] |accessdate=2019-01-26 |archiveurl=https://web.archive.org/web/20020306024752/http://www.people.com.cn/GB/shehui/43/20020227/675010.html |archivedate=2002-03-06 |language=zh-cn |date=2002-02-27| deadurl=no}}</ref>

== 争议 ==
大连曾在2004年时被《[[武汉晚报|武汉晚报]]》指出其经费不足。大连市政府每年为骑警大队拨专项经费140万元，但警队仍有100万元的赤字，每年投放在每匹马的费用也有近3万元。<ref name=wuhan2004 />

2013年5月27日，大连市退休警察赵明在网上公开申请大连市女骑警的信息公开，当中包括警队编制、成本、出警记录、社会效益等信息。赵明认为，警察低收入、超负荷工作，但骑警队伍大额投入、回报严重失调，其设置只是“长官意志的产物”，不合当下国情，是在浪费[[纳税人|纳税人]]的钱财。此外，骑警队伍还承担了表演的任务，是[[形象工程|形象工程]]，而非警察的职责所在。<ref>{{cite web |author1=宋广辉 |title=大连女骑警被指为形象工程 市民申请公开成本-搜狐滚动 |url=http://roll.sohu.com/20130527/n377187604.shtml |website=roll.sohu.com |publisher=[[中国青年报|中国青年报]] |accessdate=2019-01-25 |archiveurl=https://web.archive.org/web/20190125211103/http://roll.sohu.com/20130527/n377187604.shtml |archivedate=2019-01-25 |language=zh-cn |date=2013-05-26| deadurl=no}}</ref>搜狐网亦评论称该巡警队类似“[[花瓶#引申义|花瓶]]”。<ref name="sohu-dianji1152" />

对此，大连市公安局回应称，女子骑警队每匹马的养护成本约为每月2500元，自成立以来[[巡逻|巡逻]]7000余次，完成受检任务1000余次。大连市公安局巡警支队副支队长于福田称，大连因治安比较稳定，所以骑警的主要任务不在治安方向，而是更侧重于礼仪与表演，曾承担过大型活动的礼仪执勤和巡逻，出席过领导人的检阅视察，甚至在马术比赛中担任裁判。于福田认为女骑警“为大连城市形象和旅游市场所作出的贡献，是无法量化计算的”、“决不能简单地以‘花瓶’视之”。<ref>{{cite web |author1=宋广辉 |title=大连警方：女骑警不是“花瓶”-中国青年报 |url=http://zqb.cyol.com/html/2013-05/28/nw.D110000zgqnb_20130528_1-05.htm |website=zqb.cyol.com |publisher=中国青年报 |accessdate=2019-01-25 |archiveurl=https://web.archive.org/web/20161112091810/http://zqb.cyol.com/html/2013-05/28/nw.D110000zgqnb_20130528_1-05.htm |archivedate=2016-11-12 |language=zh-cn |date=2013-05-28| deadurl=no}}</ref><ref>{{cite web |author1=杨大伟 |title=现代五项女骑警“客串”裁判 成为赛场亮丽风景 |url=http://sports.sohu.com/20130525/n377049662.shtml |website=sports.sohu.com |publisher=大连新闻网 |accessdate=2019-01-25 |archiveurl=https://web.archive.org/web/20190125225542/http://sports.sohu.com/20130525/n377049662.shtml |archivedate=2019-01-25 |language=zh-cn |date=2013-05-25| deadurl=no}}</ref>在[[中国新闻社|中国新闻社]]对骑警基地游客的采访中，多数大连市民对女骑警队充满自豪感。<ref name=chinanews />

女子骑警训练基地也被指出涉嫌违法经营──训练基地被大连市旅游局列为国家3A级旅游景区，并在每年的5月至10月间向游客开放，游客凭票入场后可观看马术表演、与骑警合影等。然而，《[[人民警察法|人民警察法]]》第22条规定“人民警察不得从事营利性的经营活动或者受雇于任何个人或者组织。”此外，训练基地的门票上也没有加印税务等级章，估测可能在2001年至2013年之间产生了3000万元的门票收入。<ref name="yicai" />
