'''大理国银鎏金镶珠金翅鸟'''，[[大理国|大理国]]时期（[[宋代|宋代]]）文物，1978年发现于[[云南省|云南省]][[大理市|大理市]][[崇圣寺_(大理)|崇圣寺]]主塔千寻塔的塔顶。现藏于[[云南省博物馆|云南省博物馆]]。

== 出土 ==
云南省大理市点翠山应乐峰下的[[崇圣寺_(大理)|崇圣寺]]兴建于南诏[[保和_(年號)|保和]]十三年（[[唐文宗|唐文宗]][[开成|开成]]元年，公元836年），毁于清穆宗[[同治|同治]]十一年（1872年），仅三座古塔留存。1978年，中国政府拨专款对年久失修的三塔进行维修加固，文物工作者在维修三塔时，在主塔千寻塔塔顶发现南诏、大理国时期文物文物680多件<ref name="杜瑜丽2008" />，其中尤以大理国时期文物居多，大理国银鎏金镶珠金翅鸟就是其中一件<ref name="王永胜2006">{{cite journal |author1=王永胜 |title=大理国鎏金镶珠银质金翅鸟 |journal=今日民族 |date=2006 |issue=2 |pages=47-48}}</ref>，同时出土的还有[[银背光金阿嵯耶观音像|银背光金阿嵯耶观音立像]]等<ref name="雅昌网">{{cite web |title=银鎏金镶珠金翅鸟 |url=https://news.artron.net/20131129/n540344.html |website=雅昌网-新华日报 |accessdate=2019-01-07 |date=2013-11-29 |archive-url=https://web.archive.org/web/20190107233345/https://news.artron.net/20131129/n540344.html |archive-date=2019-01-07 |dead-url=no }}</ref>。

== 文化背景 ==
[[大鹏金翅鸟|大鹏金翅鸟]]以[[龙|龙]]、[[蛇|蛇]]为食的传说可追溯到印度史诗《[[摩诃婆罗多|摩诃婆罗多]]》，传说生主{{le|达刹|Daksha}}的两个女儿{{le|迦德卢|Kadru}}与{{le|毗娜达|Vinata}}嫁给仙人[[迦叶波|迦叶波]]，姐姐迦德卢生下了一千枚蛋，妹妹毗娜达生下了两枚蛋，前者孵化出一千条龙蛇（[[那伽|那伽]]），后者死掉一枚，另一枚孵化出大鹏金翅鸟（[[迦楼罗|迦楼罗]]）。由于母亲毗娜达受迦德卢欺负，沦为迦德卢的女奴，迦楼罗为拯救母亲，不断吞噬蛇类，从此金翅鸟与蛇类世代为敌。迦楼罗在印度神话中是一种性情猛烈，类似鹫鹰的巨鸟，是[[印度教|印度教]]三大主神之一[[毗湿奴|毗湿奴]]的坐骑，佛教[[天龙八部|天龙八部]]之一，通常与人面鸟身、鸟面人身或全鸟身的形象出现<ref name="杜瑜丽2008">{{cite journal |author1=杜瑜丽 |title=大鹏金翅鸟与白族的金鸡崇拜 |journal=云南农业大学学报(社会科学) |date=2008 |issue=2 |pages=115-117}}</ref>。

迦楼罗的形象随佛教传入[[东亚|东亚]]，与[[白族|白族]]等族原本的金鸡崇拜相结合，在中国西南地区往往以金鸡形象出现。古代的大理频发水患，百姓认为是邪龙作祟导致的，佛教故事中龙畏惧大鹏金翅鸟和佛塔，因此大理人修建佛塔，并将大鹏金翅鸟塑在塔顶或藏于塔内，以期望达到镇压水患的目的<ref name="杜瑜丽2008" />。

== 外观描述 ==
这座金翅鸟的鸟头饰羽冠，脖颈细长，翅膀向内卷作欲飞状，两爪锋利有力，立于莲座之上。尾、身之间插有镂空火焰形背光，背光上饰有五粒[[水晶|水晶]]珠。头、翼、身、尾为分别锤刻制作，通过[[焊接|焊接]]连为整体。这座金翅鸟为整器鎏金，体态雄健圆浑，工艺细腻，造型精美<ref name="张蕊">{{cite web |author1=张蕊 |title=云南省三件国宝入选《国家宝藏》第二季 来看看它们稀奇在哪儿 |url=http://yn.yunnan.cn/system/2018/12/03/030131532.shtml |website=云南网 |accessdate=2019-01-07 |date=2018-12-03}}</ref><ref name="珍品鉴赏">{{cite web |title=宋大理国银鎏金镶珠金翅鸟 |url=http://www.ynmuseum.org/detail/181.html |website=云南省博物馆 |accessdate=2019-01-07}}</ref>。
