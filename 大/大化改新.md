'''大化改新'''（{{lang-ja|大化の改新}}）是西元645年時[[日本|日本]]的一連串社会政治改革。其主要内容是废除當時[[豪族|豪族]]專政的制度，並效法[[中国|中国]][[唐朝|唐朝]]体-{制}-成立[[中央集权|中央集权]]国家，對日後日本历史發展影響深遠。

== 背景 ==
大化改新以前，[[苏我氏|苏我氏]]等豪族壟政，[[天皇|天皇]]家族实权甚少。公元645年6月，皇室[[中大兄皇子|中大兄皇子]]（后来的[[天智天皇|天智天皇]]）联合贵族[[中臣镰足|中臣镰足]]发动[[政变|政变]]，刺杀当时掌政的权臣[[苏我入鹿|苏我入鹿]]，並使其父[[苏我虾夷|苏我虾夷]]自杀。最終，皇室得以重掌大權，中大兄皇子等人便擁立[[孝德天皇|孝德天皇]]繼位。

[[孝德天皇|孝德天皇]]（645年─654年在位）即位后，迁都[[难波京|难波京]]（今[[大阪市|大阪市]]），建元[[大化_(孝德天皇)|大化]]。此年號意为「偉大的變化」，也是[[日本歷史|日本歷史]]上首個年號。

646年元旦，孝德天皇颁布《[[改新之詔|改新之詔]]》，推行政治改革，史稱“'''大化改新'''”。此后日本逐渐步入[[封建社会|封建社会]]，邁向以天皇为首的中央集权国家。

== 內容 ==
# 效法[[隋朝|隋]][[唐朝|唐]]之[[均田制|均田制]]，行[[班田制|班田收授法]]：廢除豪族对土地與部民的私有權，並將其收為國有。政府计口授田，凡六歲以上之國民，每六年按人口班给田地（稱「口分田」）。所班田地不得買賣，死後歸還政府。
# 效法唐朝[[租庸调制|租庸调-{制}-]]：受田農民须向國家上繳穀物（此為「租」），服勞役或纳布代役（此為「庸」），並上貢地方土產（此為「调」）。
# 中央設'''二官'''：[[神祇官|神祇官]]、[[太政官|太政官]]和'''八省'''（中務、式部、治部、民部、兵部、刑部、大藏、宮內）、'''一台'''（彈正台）；一般地方設國、郡、里。
# 兵制：首都置五衛府，地方設軍團，國民需服兵役。
