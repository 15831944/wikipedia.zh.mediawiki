'''大甲城'''是清朝時由民間仕紳於今天[[臺灣|臺灣]][[臺中市|臺中市]][[大甲區|大甲區]]所興建的城池，城內市街的中心點即是[[大甲鎮瀾宮|大甲鎮瀾宮]]{{r|張}}。該城建於清道光年間，於[[臺灣日治時期|臺灣日治時期]]的[[明治|明治]]、[[大正|大正]]年間逐步被拆除{{r|張}}。

== 沿革 ==
大甲地區自雍正九年（1731年）[[大安溪|大安溪]]口開設貿易港後，逐漸發展成[[大甲堡|大甲堡]]各庄的經濟中心，而在嘉慶二十一年（1816年）鹿港巡檢移駐大甲後，大甲街的地位變得更加重要{{r|張}}。

道光七年（1827年），在仕紳林聰、林甲成等人提議下，開始集資興建大甲城{{r|張}}。興建時就近取材，城牆用的是[[大甲溪|大甲溪]]河床的溪石，外面再用石灰、黑糖與米餅混合成漿塗刷，周長五百一十丈，高二丈{{r|張}}。

到了日治時期，大甲城因為市區改正計畫的緣故逐漸被拆毀，首先在明治三十八年（1905年）拆掉城牆，兩年後拆掉了東門，明治四十一年（1908年）拆掉了南門與北門，至於西門則到了大正年間（1920年代）才被拆毀{{r|張}}。

== 城門與市街 ==
大甲城共有四座城門，分別是東門「拱晨門」、西門「觀海門」、南門「薰風門」、北門「湛恩門」，其位置分別位在今天大甲鎮政路的土地銀行附近（東）、孔雀路路嬌姑廟旁（西）、順天路[[林氏貞孝坊|林氏貞孝坊]]北邊（南）、順天路258號北門土地公廟附近（北）{{r|張}}。

城內主要市街呈現十字型，其南北大道即是今天的順天路，當時稱為「大街」，寬六尺；城中心為鎮瀾宮，城的西北邊有[[大甲文昌祠|文昌廟]]，東北邊有舊大甲城隍廟址在今天的民生路與鎮瀾街的交叉口一帶，而官府衙門則在大甲第一市場對面，其中巡檢署即順天路電信局、郵局一帶，守備署與兵營則在今天的武營路一帶{{r|張}}。至於集市則有北門街的米市與三角街的魚市等{{r|張}}。
