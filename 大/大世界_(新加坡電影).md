《大世界》（[[漢語拼音|漢語拼音]]：dà shì jiè / [[閩南語|閩南語]]：Tua Seh Kai） 是[[唐永健|唐永健]]執導的2011年新加坡賀歲喜劇電影，由[[向雲|向雲]]、[[林湘萍|林湘萍]]等主演，該片於2011年1月27日在新加坡上映。<ref name="insing">{{cite web | url=http://movies.insing.com/movie/a-great-great-world/id-24020000 |title=inSing movies |accessdate=22 January 2011 |work= SingTel Digital Media Pte Ltd. | date=}}</ref>該片以新加坡家喻戶曉的{{link-en|大世界遊樂園|Great World Amusement Park}}為背景，故事從1940年跨越至21世紀，講述大世界遊樂園中小人物的溫馨故事。這部電影的一個重要特點是大量使用各種[[漢語|漢語]]，許多演員能夠以不孰悉的漢語來演出。<ref>{{cite news|title=《大世界》方言大考验　周初明6小时化老妆|url=http://entertainment.xin.msn.com/zh/celebrity/features/asia/article.aspx?cp-documentid=4625621&page=3|publisher=xin.msn.com|date=7 February 2011}}</ref><ref>{{cite news|title=向云露肉拍大世界 方言台词考倒众演员|url=http://nanyang.com/node/374066|publisher=[[Nanyang_Siang_Pau|Nanyang Siang Pau]]|date=6 August 2011|deadurl=yes|archiveurl=https://web.archive.org/web/20140113173627/http://www.nanyang.com/node/374066|archivedate=2014年1月13日}}</ref>

== 劇情 ==
本劇描述位於[[新加坡|新加坡]]的名為大世界的遊樂園，當地人也以閩南話稱呼“Tua Seh Kai”而聞名。從1940年代到21世紀，這部電影呈現了四個以大世界遊樂園為中心的故事。

阿敏（[[王儷婷|王儷婷]]飾）是一個23歲的大學生，在外婆的舊像館“明珠像館”時發現發現以往顧客忘了來取的相片。隨著好奇心，她展開了尋找這些老顧客故事的旅程。透過年邁已高的阿明叔（{{link-en|周初明|Chew Chor Meng}} 飾），揭露出當年四段發生在大世界遊樂場的喜怒哀樂。

*大世界遊樂園的兒童劇場。小丑阿卜（{{link-en|程旭輝|Henry Thia}}飾）是如何卯足全力，勢必要跟1958年來到青天戲院出席《環遊世界80天》電影首映的好萊塢影星[[伊莉莎白·泰勒|伊莉莎白·泰勒]]合照

*大世界的鬼車，雖然幾分詭異，但也成了男生在約會時藉機耍勇的地方。“地甘”妹美娟（[[白薇秀|白薇秀]]飾）和粗俗的馬來亞流氓阿良，在1965年新加坡和馬來西亞分家時，營造了純純的愛情故事。

*除了小丑和這對情侶外，還有一朵在紅鶴夜總會即將凋零的台柱——玫瑰（[[向雲|向雲]]飾）。他如何與新一代的歌星爭艷奪力，如何應付那刁鑽的鄭經理呢？又如何通過舊情人的出現，在1975年的紅鶴夜總會裡再次開花。

*1941年永春園，畫下經歷的一刻。在年輕阿明（{{link-en|周初明|Chew Chor Meng}} 飾）的婚宴中，餐館裡的員工從鬧哄哄的廚房裡烹煮出一道道垂涎美食慶祝喜宴。怎知，喜宴背後竟暗藏危機。當晚，[[新加坡戰役|新加坡被日軍淪陷]]了。

深受這四個故事的感動，阿敏對過往日的種種格外珍惜，並決定繼續讓明珠像館的歷史延續下去。

== 歷史鏡頭 ==
*1965年8月9日新加坡與馬來亞分離後，電視轉播[[李光耀|李光耀]]的演說
*最後一個故事描述[[日軍|日軍]]軍機在[[轟炸新加坡|新加坡的轟炸]]（1941年12月8日），隔年（1942年）[[新加坡日佔時期|日軍便佔領新加坡]]。
*在紅鶴夜總會歌星玫瑰所演唱的歌曲“ 玫瑰玫瑰我愛你 ”，是由[[陳歌辛|陳歌辛]]於[[抗戰|抗戰]]期間所創作的歌曲。

== 票房收入 ==
截至2011年2月21日，這部電影已在新加坡票房收入超過200萬[[新加坡幣|新加坡幣]]。

== 配音 ==
這部電影於2011年底在美亞電影台播放。2013年，在第8頻道播出；影片中了露西阿姨的上海話和其他角色原本的漢語普通話（現代標準漢語）發音台詞以外，其他的漢語的發音全部被配音成[[現代標準漢語|現代標準漢語]]。
