'''K12通用機槍'''（定型前稱：'''XK12'''）是一挺由[[大韩民国|南韓]][[S&T重工業|S&T Motiv]][[大宇集团|大宇集团]]（前稱：大宇精密工業）所設計和生產的[[通用機槍|通用機槍]]，用以取代[[大韓民國國軍|大韓民國國軍]]以內裝備的舊型[[M60通用機槍|M60通用機槍]]，<ref>[http://www.asiae.co.kr/news/view.htm?idxno=2011081807265620151 국산 저격용소총 첫 개발됐다 - 아시아경제]</ref>發射[[7.62×51mm_NATO|7.62×51毫米]][[北大西洋公约组织|北约]][[口徑|口徑]][[步枪|步枪]][[子彈|子彈]]。

2009年，K12的原型（XK12）在首爾ADEX國際航空航天與防務展期間，首度向公眾展示。<ref>{{cite web

== 歷史 ==
[[越南战争|越南战争]]期間，南韓曾經[[韓國參戰越南戰爭|派遣國軍]]內數量相當的精銳部隊（比如[[大韓民國第2海軍陸戰師|青龍部隊]]）到越南，支援[[美軍|美軍]]以換取美國金援。當時美國向南韓國軍提供[[M60通用機槍|M60通用機槍]]作為軍事援助，這型機槍很快就由大宇精密工業（現在為[[S&T重工業|S&T Motiv]]）得到特許生產。從1970年到1990年，M60一直是南韓制式通用機槍，由[[步兵|步兵]]手持，或架設在[[载具|車輛]]和[[直升機|直升機]]以上使用。由於南韓國軍深受美國軍事主義的影響，1980年代[[美国陆军|美国陆军]]採用[[M249班用自動武器|M249班用自動武器]]時，[[大韓民國陸軍|南韓陸軍]]和[[大韓民國海軍陸戰隊|海軍陸戰隊]]跟隨其腳步，推出了[[大宇K3輕機槍|大宇K3輕機槍]]，一種可謂[[FN_Minimi輕機槍|FN Minimi輕機槍]]的“韓國化”版本輕機槍，供步兵[[班|班]]用作{{Link-en|班用自動武器|Squad automatic weapon|自動武器}}。當2000年代，老化的M60首先需要被替換下來時，[[排|排]]級機槍的地位由K3取代，後者將步兵排當中的[[中型機槍|中型機槍]]排除掉，但這背離了使用中型機槍進行排級火力支援的美國主義。M60仍然在車輛和航空器以上使用，但即使是一些南韓生產的[[装甲车|装甲车]]也只架設K3。<ref name="sadj12sep14">[http://www.sadefensejournal.com/wp/?p=2751 K12 South Korean Light Machine Gun] - SAdefensejournal.com, 12 September 2014</ref>

由於K3至今仍在生產中，而M60的生產線早已關閉，南韓軍隊似乎很明顯地會以後者用於替代M60。然而，由於[[5.56×45mm_NATO|5.56×45毫米]][[北大西洋公约组织|北約]][[口徑|口徑]][[子彈|子彈]]的射程較短，而且火力有限，南韓[[坦克|坦克]]、裝甲車和直升機上仍然保留著發射7.62×51毫米北約口徑彈藥的M60衍生型。大多數M60的運作仍然相當良好，軍方不想花錢尋找用以替換的通用機槍，選擇使用他們的預算專注於更昂貴和高價值的資產。直到2006年，南韓陸軍開始研製南韓國產中型[[通用直升機|通用直升機]]，即“[[Surion直升机|韓國通用直升機]]”之時，終於開始尋找新型中型機槍。新型直升機創造了他們對新型機槍的需求，如同從越南戰爭中撤出至今的[[UH-1直升機|UH-1H「休伊」]]和它們架設的M60依然存在。陸軍將領最初試圖從外國供應商那裡購入現貨，特別是[[赫爾斯塔爾國營工廠|赫爾斯塔爾]][[M240通用機槍|M240H]]，但陸軍和S&T Motiv的其他人則利用這個機會，研製一款南韓國產的7.62毫米口徑機槍。<ref name="sadj12sep14"/>

2010年7月開始，S&T Motiv以XK12的名義進行研發。在開始時原型槍架設於KUH-1「Surion」直升機原型機以上並且進行測試時，他們已經發射了300,000發子彈並無出現任何嚴重問題。2012年，XK12獲採用後被命名為'''K12輕機槍'''，並且用作南韓通用直升機的制式裝備。目前，K12僅用於計劃中的300架直升機以上，但亦有可能擴大其使用範圍。南韓陸軍裝備了2,300輛坦克，而每輛坦克通常架設兩挻M60通用機槍，其他類似的車輛則架設一挻M60。此外，南韓在[[伊拉克战争|伊拉克]]和[[阿富汗戰爭_(2001年)|阿富汗]]執行[[联合国|联合国]][[联合国维持和平部队|維和任務]]的經歷，讓兵士們都希望中型機槍能夠重返步兵裝備列當中。一旦韓國大量軍購，稍後還可能會開拓其出口市場。<ref name="sadj12sep14"/>

== 設計細節 ==
K12基於K3（FN Minimi的“韓國本土”型）的設計、佈局以至使用[[氣動式_(槍械)|長行程導氣活塞]]和[[滾轉式槍機|滾轉式槍機]]的功能這一切，在意義上與[[FN_Minimi_7.62通用機槍|FN Minimi 7.62]]、[[Mk_48輕量化機槍|Mk 48 Mod 0/1]]、[[HK_MG5通用機槍|HK MG5]]與[[IWI內蓋夫NG7通用機槍|IWI內蓋夫NG7]]這些從[[5.56×45mm_NATO|5.56×45毫米]][[北大西洋公约组织|北約]][[口徑|口徑]]的[[轻机枪|轻机枪]]放大口徑為[[7.62×51mm_NATO|7.62×51毫米]]北約口徑的[[通用機槍|通用機槍]]相似。

它藉由{{Link-en|北約標準化協議|Standardization Agreement|STANAG}} [[M13彈鏈|M13]]金屬製[[彈鏈|可散式彈鏈]]鏈鈎供彈，不可使用[[彈匣|彈匣]]等供彈具。横閂式保險與[[大宇K3輕機槍|大宇K3]]／[[FN_Minimi輕機槍|FN Minimi]]一樣，[[机匣|机匣]]由[[钢|钢]]壓製成而供彈機盤則是[[鋁合金|鋁合金]]製成。儘管在設計上相似，但機匣和其他重要部件都被放大以容納更大的[[彈藥|彈藥]]。K12對於12公斤（26.46磅）的槍械類別而言實在有點沉重，主要原因是它作為一挻將{{Link-en|手槍握把|Pistol grip}}、[[鏟子|鐵鏟]]型握把和與FN Minimi傘兵型的設計相似的內置鋼制管狀滑動式[[槍托|槍托]]全部同時裝上的機槍，設計目的是為了由將其拆卸的操作手將它迅速轉換成為在地面上射擊的機槍；鐵鏟型握把可以藉由取出兩顆固定[[定縫銷釘|銷釘]]後，與扳機聯動桿組件一併加以移除，然後就可展開其內置槍托，成為地面上使用的輕機槍。標準機槍功能包括折疊式（K3樣式）[[兩腳架|兩腳架]]、快速更換式[[槍管|槍管]]、氣體調節器，和（M240H樣式）{{Link-en|槍口消焰器|Flash Suppressor|消焰器}}。它還裝有一個可折疊式網狀防空瞄準具，可用以在直升機上射擊；同時亦裝上了可折疊的立框式標尺型瞄準具，以進行更精確的瞄準射擊。瞄準具都裝設在[[机匣|機匣蓋]]頂部的[[皮卡汀尼導軌|MIL-STD-1913戰術導軌]]上，而導軌亦安裝在供彈機蓋和[[護木|護木]]的兩側。目前K12還沒有可選用的[[雷射瞄準器|雷射瞄準器]]或是光學[[瞄準鏡|瞄準鏡]]，但官方預計在不久的將來會配備某型光電附件。<ref name="sadj12sep14"/>

== 衍生型 ==
2015年，S&T Motiv推出了K12的[[同軸武器|同軸機槍]]型號'''XK-12C1'''，配備了更重型的[[槍管|槍管]]和電子控制擊發系統。它的設計是用以填補目前安裝在許多車輛以上的M60C同軸電子控制擊發機槍的用途，因為它們已正在接近其運營壽命上限，並將很快需要被取代下來。<ref>[http://www.sadefensejournal.com/wp/?p=3685 ADEX 2015] - SAdefensejournal.com, 5 August 2016</ref>
