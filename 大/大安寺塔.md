'''大安寺塔'''于[[中国|中国]][[浙江省|浙江省]][[金华市|金华]][[义乌市|义乌市]][[稠城街道|稠城街道]]绣湖社区县前街绣湖湖畔，原属大安教寺（原在绣湖湖心岛柳洲，今已不存），按县志所载始建于北宋[[大观_(年号)|大观]]四年（1110）<ref>清·诸自谷等修、程瑜等纂，《嘉庆义乌县志·卷十八·寺观》：大安教寺，县西一百五十步绣湖心之柳洲，旧为尼寺，名普安，在县东北百五十步，唐咸通八年建，宋治平赐额，大观三年徐秉哲改为僧寺，徙置今所，有塔五级。……按寺塔旧志未详何代何人兴建，今于塔中得废砖三，一云“庚寅岁建”，一云“孝妻王氏建追荐故夫”，一云“省干吴忠翊速生天界”，考大观三年己丑移寺，当是四年庚寅建塔，存此以备参考。</ref>，由邑人吴圭（1068-1121，字彦成）出资为供奉[[傅大士|双林大士]]而建<ref>吴圭出资造塔的记载见于吴氏宗谱、吴圭墓志铭等，如宋·梅执礼，《吴彦成墓志铭》（载《浦阳大辂吴氏宗谱》卷一）：县作浮图绣川湖中，以奉双林大士，又作神霄、玉清、万寿宫，栋宇像设，所费皆不赀，翁一一乐施，不以累众。</ref><ref>{{cite web|url=http://www.yw.gov.cn/zjyw/dfzj/ywfz/2008jk1_3/scjk/200912/t20091215_11119.shtml|title=《首次修缮大安寺塔时间考》|accessdate=2017-08-17|work=张金龙|publisher=义乌市政府|archive-url=https://web.archive.org/web/20170817082933/http://www.yw.gov.cn/zjyw/dfzj/ywfz/2008jk1_3/scjk/200912/t20091215_11119.shtml|archive-date=2017-08-17|dead-url=yes}}</ref>（亦有建塔镇妖之说），另一说则认为其建于南宋[[乾道_(南宋)|乾道]]六年（1170），为吴若古（1112-1168，吴圭侄子，官至忠翊郎）之妻王氏为追念亡夫而建<ref>{{cite web |url=http://www.jhnews.com.cn/zzxb/2010-04/01/content_975817.htm|title=《他对义乌地标的身世提出异议》|accessdate=2017-08-17|work=王志坚|publisher=浙中新报}}</ref><ref>{{cite web|url=http://epaper.jhnews.com.cn/site1/jhrb/html/2010-09/12/content_1185208.htm|title=《义乌地标大安寺塔身世之争》|accessdate=2017-08-16|work=胡国洪|publisher=金华日报|archive-url=https://web.archive.org/web/20170817083002/http://epaper.jhnews.com.cn/site1/jhrb/html/2010-09/12/content_1185208.htm|archive-date=2017-08-17|dead-url=yes}}</ref>。明永乐年间顶层倒塌，永乐十四年（1416）重修，四层和五层重新修复，1996年重建塔顶，为义乌境内现存最早的砖木结构建筑。

今塔为六角五层的砖木结构[[楼阁式塔|楼阁式塔]]，现存砖砌塔身及少量木结构，副阶、腰檐、平座栏杆和塔刹已毁，但平座、倚柱、阑额、斗拱等砖质仿木结构保存尚好<ref>{{cite web |url=http://www.jhnews.com.cn/2014/0704/388191.shtml?emfhe4|title=《义乌城千年的湖光塔影 寻访八婺宝塔之义乌大安寺塔》|accessdate=2017-08-17|work=方跃镇|publisher=金华新闻网}}</ref>。塔身残高23.42米，底层内径2.9米，塔壁厚1.35米，转角处设倚柱，每层六面间隔辟门窗，其位置逐层相错，塔砖上有“庚寅岁建”、“孝妻王氏建追荐故夫”、“省干吴忠翊速生天界”、“吴知县宅”等多处铭文<ref>{{cite web |url=http://www.zjww.gov.cn/wszt/exhibit_hall_detail.jsp?eid=492|title=《大安寺塔》|accessdate=2017-08-16|work=|publisher=浙江省文物局}}</ref>。
