'''大華車站'''位於[[臺灣|臺灣]][[新北市|新北市]][[平溪區|平溪區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[平溪線|平溪線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[岸式月台|岸式月台]]一座。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]。
*行駛車輛為DR1000型[[柴油|柴油]]客車，等級為[[臺鐵區間車|區間車]]。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*本站因應「大華煤礦」側線的設立和附近民眾的需求而設置。
*1956年10月12日：設站，並指定由[[三貂嶺車站|三貂嶺站]]管理。
*1994年4月1日：由簡易站降為招呼站。
*2001年7月16日：管理站由[[三貂嶺車站|三貂嶺站]]改為[[瑞芳車站|瑞芳站]]；[[平溪線|平溪線]]沿線各站（'''大華'''、[[望古車站|望古]]、[[嶺腳車站|嶺腳]]、[[平溪車站|平溪]]、[[菁桐車站|菁桐]]）同日皆改由[[瑞芳車站|瑞芳站]]管理。
*2013年9月30日：啟用多卡通刷卡機。
