'''大觀音亭興濟宮'''是位在[[臺灣|臺灣]][[臺南市|臺南市]][[北區_(台南市)|北區]]的著名廟宇，於民國七十四年（1985年）11月27日公告為[[三級古蹟|三級古蹟]]{{r|文}}{{r|文2}}。該寺廟是由主奉[[觀音菩薩|觀音菩薩]]的「大觀音亭」與主奉[[保生大帝|保生大帝]]的「興濟宮」所組成，雖分屬[[佛教|佛教]]、[[道教|道教]]，但自創建以來即兩廟相連，關係密切，同屬一組織管理。現稱「財團法人大觀音亭興濟宮」。

== 沿革 ==
[[File:大觀音亭正殿.JPG|250px]]
[[File:臺南興濟宮正殿.jpg|200px]]
[[File:興濟宮保愛生民匾.JPG|200px]]
[[File:大觀音亭興濟宮官廳.jpg|250px]]
[[File:大觀音亭興濟宮官廳_(二).JPG|250px]]
[[明鄭|明鄭]][[永曆|永曆]]三十二年（1678年），軍民在[[臺灣府城|臺灣府城]]的尖山南坡建「觀音宮」，供奉[[觀音菩薩|觀音菩薩]]{{r|石}}{{r|王}}。康熙時重修擴建之後，因拜亭採用了[[抱廈|抱廈]]的形式而改名「觀音亭」，後來又因要與小東門的觀音亭區分，而習稱「大觀音亭」{{r|王}}。隔年（1679年），來自[[泉州|泉州]][[同安|同安]]鄭軍將士又在「大觀音亭」旁建今興濟宮奉祀保生大帝（大道公），當時稱為「大道公廟」，後因為與[[良皇宮|良皇宮]]分別，稱「頂大道公廟」，良皇宮則稱「下大道公廟」{{r|傅}}。兩者均為明鄭時期規模最大的觀音廟與大道公廟{{r|石}}。

而在[[施琅|施琅]]克臺之後，由於總鎮衙署與所轄三營分別建在尖山北坡與覆鼎金（今[[衛生福利部臺南醫院|衛生部臺南醫院]]）之間，都在兩廟附近，所以有不少官兵前來祈求平安{{r|石}}。大觀音亭方面，主要在康熙三十二年（1693年）{{NoteTag|1=由里民發起，該次重修中重建了後堂。}}、乾隆五十六年（1791年）{{NoteTag|1=里人陳漳山等人重修並獻一間店鋪作為香火資金。}}、嘉慶二十年（1815年）{{NoteTag|1=水師提督[[王得祿|王得祿]]率眾捐錢整修。}}、道光十年（1830年）{{NoteTag|1=信徒蕭元錕捐錢整修。}}與同治十年（1871年）{{NoteTag|1=由里眾發起重修。}}整修過，至於興濟宮，則主要在嘉慶二年（1797年）{{NoteTag|1=由總兵[[哈當阿|哈當阿]]與巡道[[劉大懿|劉大懿]]重修。}}與道光十一年（1831年）{{NoteTag|1=由王得祿及鎮道[[劉廷斌|劉廷斌]]、平慶等人重修。}}與同治十年（1871年）{{NoteTag|1=由里眾發起重修。}}整修過{{r|石}}。而在同治十三年（1874年）之後，因所謂的[[開山撫番|開山撫番]]導致官兵多染上瘴癘，所以兩廟的香火日益興盛，成為府城裡同類廟宇中香火最鼎盛的廟{{r|石}}。這些參與的官兵因曾向興濟宮大道公祈求保佑，後來果然靈驗，因而上報朝廷，乃下令將興濟宮列入祀典，成為臺灣唯一有列入官方祀典春秋二祭的保生大帝廟{{r|大道公|page=118}}。

而進入[[臺灣日治時期|臺灣日治時期]]之後，兩廟曾於[[昭和|昭和]]二年（1927年）一起重修{{r|團}}。另外日治末期時，興濟宮春秋二祭因故中斷{{r|大道公|page=116}}。

[[太平洋戰爭|太平洋戰爭]]中大觀音亭興濟宮受創，遂於戰後的民國三十八年（1949年）重修。民國時期兩廟在民國五十、六十年代（1960年代、1970年代）{{NoteTag|1=1964年大觀音亭重修三寶殿，1968年大觀音亭與興濟宮一起重修。}}重修過幾次，而最近一次則是在民國一百零六年（2017年）{{r|團}}。

2007年，啟建七朝清醮。2011年參考《欽定大清會典事例》，結合興濟宮三獻禮，恢復由地方官員親臨致祭的春祭大典{{r|大道公|page=116}}。2012年5月興濟宮春祭時，曾舉辦「醫神大道公藥籤藥材展」並製作「祀典興濟宮保生大帝藥籤解」，介紹藥籤的藥方內容{{r|大道公|page=95}}。

== 建築特色 ==
大觀音亭與興濟宮分別在右側與左側，中間為據說是供前來參拜[[官吏|官吏]]更衣與休憩之用的官廳，三者以八角門互通{{r|王}}，兩廟皆深三進，位在一基座之上，由三川殿（門）、拜亭、龍虎井、正殿、過水廊與後殿組成{{r|傅}}。廟前正對的觀亭街古稱「觀音亭街」，是臺南少見的參拜道，但現在已經拓寬{{r|傅}}。

廟中所藏古物甚多，如[[清朝|清朝]][[太子太保|太子太保]][[王得祿|王得祿]]所題楹聯與所獻匾額，歷代重修碑記等，另外廟中之彩繪乃出自臺南名師[[陳壽彝|陳壽彝]]之手筆{{r|傅}}。

== 祀神 ==
[[File:祀典興濟宮虎爺.JPG|thumb]]
大觀音亭以佛教為主，供奉有：
*三川殿：[[哼哈二將|哼哈二將]]
*正殿：[[觀音菩薩|觀音菩薩]]（挾侍[[善財|善財]]、[[龍女|龍女]]）、[[準提菩薩|準提菩薩]]、[[齊天大聖|齊天大聖]]、[[達摩祖師|達摩祖師]]、[[十八羅漢|十八羅漢]]、[[註生娘娘|註生娘娘]]、[[月下老人|月下老人]]、花公花婆。
*後殿：[[橫三世佛|三寶佛]]、觀音菩薩（挾侍善財、龍女）、[[大勢至菩薩|大勢至菩薩]]、[[彌勒佛|彌勒佛]]、[[韋馱菩薩|韋馱菩薩]]、[[伽藍菩薩|伽藍菩薩]]、[[地藏菩薩|地藏菩薩]]（挾侍[[地藏菩薩#化身九華|道明尊者]]、[[地藏菩薩#化身九華|閔公尊者]]）、[[觀世音菩薩#千手千眼觀音|妙莊王]]（傳說中觀音菩薩之父）、[[緊那羅#漢傳佛教的“灶神”傳說|鑑齋菩薩]]。

興濟宮以道教為主，供奉有：
*正殿：[[保生大帝|保生大帝]]（侍神：江仙官、[[飛天大聖|張聖者]]、採藥童子、製藥童子、[[下壇將軍|下壇將軍]]）、姑爺王舍人、敏明姑娘、[[中壇元帥|中壇元帥]]、[[五營神將|五營神將]]、[[福德正神|福德正神]]
*後殿：聖父母（保生大帝之父母協成元君、玉華大仙）、[[三官大帝|三官大帝]]、[[南斗星君|南斗星君]]、[[北斗星君|北斗星君]]、[[關聖帝君|關聖帝君]]、[[關平|關平將軍]]、[[周倉|周倉將軍]]、[[文昌帝君|文昌帝君]]、[[太歲星君|太歲星君]]{{r|團}}。

祀典興濟宮的下壇將軍（虎爺）據說喜歡炸雞，供奉的炸雞要等3、4個小時才可撤，其中又以[[肯德基|肯德基]]的炸雞為最{{r|大道公|page=120}}。

== 藥籤 ==
[[File:祀典興濟宮籤筒.jpg|thumb]]
[[File:祀典興濟宮藥籤櫃.jpg|thumb]]
興濟宮內設有藥籤，信徒要求籤時要先點三柱香向保生大帝稟明症狀，而後靜坐在椅子上，將三柱香輕貼在手腕脈門上，請神明「把脈」{{r|大道公|page=93、94}}。待三柱香的香灰各掉一次後，才到對應的科目籤筒抽一支籤，然後將之放在案桌上，[[擲筊|擲筊]]確認，若無三聖杯則要重抽{{r|大道公|page=94}}。確認後才到藥籤櫃拿對應的籤紙，到藥房抓藥{{r|大道公|page=94}}。回家煎藥時得在正廳面朝外煎藥，並焚香祈求保生大帝到來，之後將香插在煎藥的烘爐上{{r|大道公|page=94}}。
