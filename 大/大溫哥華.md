'''大溫哥華'''（{{Lang-en|'''Greater Vancouver'''}}或'''Metro Vancouver'''；亦稱'''大溫哥華地區'''）是[[加拿大|加拿大]][[卑诗省|卑诗省]]西南部海岸地區，包含[[溫哥華|溫哥華]]和周邊城鎮的都會區。由[[大溫哥華區域局|大溫哥華區域局]]管理。

在當地英文媒體里，「Greater Vancouver」同「Metro Vancouver」經常混用。但嚴格來说，「Metro Vancouver」指的是1966年建立的「[[大溫哥華區域局|大溫哥華區域局]]」（英文：'''Greater Vancouver Regional District'''），是一個有法律地位的機構。另一方面，一些不屬於大溫哥華區域局的城市，例如東邊的[[米逊_(卑诗省)|米逊]]、[[阿伯茨福德|阿伯茨福德]]、[[奇利瓦克|奇利瓦克]]和北邊的[[惠斯勒|惠斯勒]]、[[斯阔米什|斯阔米什]]等市鎮，亦被当作大溫哥華地區的一部份。

== 人口 ==
	
根据[[加拿大统计局|加拿大统计局]]2016年人口普查，[[大温哥华地区|大温哥华地区]]的人口为2,463,431人，比2011年2,313,328人增加了6.5％。私人住宅1,027,613户，其中有人居住的私人住宅的960,894户，空置率6.5%。 土地面积为2,883平方公里（1,113平方英里），2016年人口密度为855人 /平方公里（2,213人/平方英里），使其成为[[卑诗省|卑诗省]]人口和人口密度最大的地区。

另外，[[大温|大温]]不设地方行政架构的地区（如[[大学保留地|大学保留地]]、北温和西温以北的高地以及[[菲沙河|菲沙河]]上的[[:en:Barnston_Island,_British_Columbia|班斯敦岛]]）则统称[[大温A选区|大温A选区]]。 2016年[[大温A选区|大温A选区]]的人口为16,133。

[[大温|大温]]还有17个原住民保留地，不受任何市镇管辖。 2016年保留地的人口共计7,900。<ref>{{Cite web |url=http://www.metrovancouver.org/services/first-nation-relations/AboriginalPublications/ProfileOfFirstNations.pdf |title=大温第一民族概况 |accessdate=2017-09-10 |work=Metro Vancouver}}</ref>

其他位于[[低陆平原|低陆平原]]东部的市镇（如[[阿伯茨福德|阿伯茨福德]]、[[奇利瓦克|奇利瓦克]]和[[米逊_(卑诗省)|米逊]]）虽然隶属[[菲沙河谷区域局|菲沙河谷区域局]]，但有时也被当成[[大温地区|大温地区]]的一部分。<ref name="AboutMV">{{Cite web |url=http://www.metrovancouver.org/about/Pages/default.aspx |title=Who is Metro Vancouver |accessdate=2012-04-08 |work=Metro Vancouver}}</ref>
