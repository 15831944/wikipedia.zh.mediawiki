'''大地测量学'''，根据[[德国|德国]]大地测量学家[[:en:Friedrich_Robert_Helmert|F·R·Helmert]]的经典定义，它是一门量测和描绘地球表面的学科，也包括确定地球重力场和海底地形。

== 历史 ==
{{main|大地測量學史}}
大地测量学起源於土地的划分與地产的界定，其历史可以追溯到[[古埃及|古埃及]]时代。

公元前240年，亚历山大学者[[埃拉托斯特尼|埃拉托斯特尼]]进行了[[亚历山大城|亚历山大城]]和[[赛尼城|赛尼城]]（Syene，今阿斯旺）间的大地测量工作。当日光直射赛尼城井底时，在亚历山大城日光南偏7度12分，若假设日光彼此平行，则可估计地球周长为252.000古埃及尺。11世纪[[阿拉伯帝国|阿拉伯帝国]]和[[纽伦堡|纽伦堡]]的测量仪器的发展对大地测量学的发展具有十分重要的意义，同样重要的还有角度[[函数|函数]]的发现及荷兰科学家[[威理博·斯涅尔|威理博·斯涅尔]]首创的[[三角测量|三角测量]]法。

从10年开始，通过精确的计算方法人们提高了地图的精度。在1740年，由法国科学家进行通过对北欧城市[[拉普兰|拉普兰]]和科鲁的大地测量，确定了[[地球椭球|地球椭球]]的半径，由此开始了大范围的[[大地测量|大地测量]]。为了使不同的投影与大地测量的结果能够更好的结合，高斯发展了[[平差计算|平差计算]]方法。它同样能够使局部地球参考系和全球参考系很好的统一。

大地测量学在19和20世纪的发展：
* 米制单位和格林威治起始子午圈的导入，由石英表和无线电技术发展而来的全球[[时间系统|时间系统]]。

* [[大地水准面|大地水准面]]测量，重力测量与[[物理大地测量学|物理大地测量学]]的横向联结。

* 角度测量，距离测量等仪器精度百倍的提高。

* 由1960年开始的、使全天候不间断测量成为可能的[[卫星大地测量|卫星大地测量]]：[[全球定位系统|全球定位系统]]等。

* [[甚长基线干涉测量|甚长基线干涉测量]]（VLBI）在大地动态测量及大地曲率中的应用。

== 範圍 ==
* 确定[[地球|地球]]形状及其外部[[重力场|重力场]]及其随时间的变化，建立统一的[[大地测量坐标系|大地测量坐标系]]，研究地壳形变（包括地壳垂直升降及水平位移），测定[[极移|极移]]以及海洋水面地形及其变化等。

* 研究[[月球|月球]]及[[太阳系|太阳系]]行星的形状及其重力场。

* 建立和维持具有高科技水平的国家和全球的天文大地[[水平控制网|水平控制网]]和精密[[水准网|水准网]]以及[[海洋大地控制网|海洋大地控制网]]，以满足国民经济和国防建设的需要。

* 研究为获得高精度测量成果的仪器和方法等。

* 研究地球表面向[[椭球|椭球]]面或平面的投影数学变换及有关的大地测量计算。

* 研究大规模、高精度和多类别的地面网、空间网及其联合网的数学处理的理论和方法，测量[[数据库|数据库]]建立及应用等。

* 大地测量学的一个特殊领域是海洋测量和河流的水文断面的研究。

== 学科内分支 ==
* [[几何大地测量学|几何大地测量学]]（天文大地测量学）：它的基本任务是确定地球的形状和大小及确定地面点的几何位置。

* [[椭球面大地测量学|椭球面大地测量学]]（数学大地测量学，椭球几何学）：主要研究椭球上曲线的分析与计算，椭球曲面投影变换的数学理论。

* [[物理大地测量学|物理大地测量学]]（理论大地测量学）：它的基本任务是用物理方法（重力测量）确定地球形状及其外部[[重力场|重力场]]。

* [[空间大地测量学|空间大地测量学]]：主要研究人造地球卫星及其他空间探测器为代表的空间大地测量的理论，技术与方法。

* [[工程测量学|工程测量学]]：工程建设和资源开发的勘测设计、施工、竣工、变形观测和运营管理各阶段中进行的各种测量工作的总称。
