'''大學站'''（{{lang-en|'''University Station'''}}）是[[港鐵|港鐵]][[東鐵綫|東鐵綫]]的車站，位於[[香港|香港]][[新界|新界]][[沙田區|沙田區]][[馬料水|馬料水]]，舊稱'''馬料水站'''（{{lang-en|'''Ma Liu Shui Station'''}}），因當時在車站旁邊的[[香港中文大學|香港中文大學]]而得名，而現時車站已被大學的建築包圍。車站並且分別於1983年、2001年及2012年擴建成現時規模。

[[蘇芮|蘇芮]]及[[夏韶聲|夏韶聲]]於歌曲《車站》MV以大學站一號月台取景及舊[[九廣鐵路|九廣鐵路]]箭嘴標誌為背景。

==結構==
大學站是一個高架車站，主要大堂是位於2號月台側的下一層。車站設有兩個[[側式月台|側式月台]]，只有中央部份設有上蓋，兩端均為露天。

===大堂===
東面大堂、西面大堂及兩個月台由兩條路軌下的通道連接（一條屬收費區內，一條則在非收費區）。雖然北面大堂設有閘機方便出入D出口，但卻沒有客務中心，乘客需返回東面大堂的B出口或者西面大堂的C出口才可找到客務中心。

[[File:University_Station_2017_08_part1.jpg|thumb]]

====商店及自助服務====
大學站內設有不同類型的商店，例如有[[便利商店|便利店]]、[[麵包糕餅店|麵包糕餅店]]、[[快餐店|快餐店]]等<ref name="shops"/>。由於港鐵的收費區是不能進食，因此大部份售賣[[食物|食物]]的商店均不會設在收費區內。

此外，大學站也有不少自助服務設施供乘客使用，包括[[自動櫃員機|自動櫃員機]]及[[自動販賣機|自動售賣機]]<ref name="shops"/>。收費區內亦設有[[廁所|洗手間]]。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = University Station 2019 03 part1.jpg | caption1 = 1號月台﹙往羅湖／落馬洲方向﹚

該站為三個設有「[[空隙黑點|空隙黑點]]」提示的車站之一（另外兩個是[[九龍塘站_(香港)|九龍塘站]]及[[旺角東站|旺角東站]]），兩個月台均在急彎之上（大學站內列車的限速為每小時55公里），空隙為港鐵中最闊的，可達34厘米，因此乘客較易在此站發生墜軌意外。2008年，一名男子在北行的月台登上列車時從空隙墜軌受傷，他其後入稟[[香港區域法院|區域法院]]控告港鐵[[疏忽|疏忽]]，但被判敗訴。<ref>[http://legalref.judiciary.gov.hk/doc/judg/word/vetted/other/en/2009/DCPI000764_2009.doc CHAN CHUNG KUEN v MTR CORPORATION LIMITED DCPI 764/2009]</ref>另外，由於大學站的急彎會使駛經的高鐵動車組剷毀月台邊緣，因此高鐵動車組不能在東鐵綫上行駛，所以當年在[[廣九直通車|廣九直通車]]引入[[CRH1|CRH1]]及[[CRH5|CRH5]]高鐵列車的計劃因而終止，並以另建[[廣深港高速鐵路香港段|廣深港高速鐵路香港段]]代替上述計劃。

另外，1號月台可直達西面大堂及2012年新建的北面大堂，使用C或D出入口通往香港中文大學。

==利用狀況==
大學站大堂外的[[公共交通交匯處|公共交通交匯處]]設有[[香港巴士|巴士]]和[[香港小巴|小巴]]駛經連接[[9號幹線|9號幹線]][[吐露港公路|吐露港公路]]馬料水段及[[2號幹線|2號幹線]][[大老山公路|大老山公路]]到正對面的馬鞍山路，橫跨[[沙田海|沙田海]]的架空天橋前往[[馬鞍山_(香港市鎮)|馬鞍山]]住宅區，以前大學站乃馬鞍山居民出入市區的重要車站，即使[[馬鞍山綫|馬鞍山綫]]通車後，由馬鞍山前往[[羅湖站_(香港)|羅湖]]／[[落馬洲站|落馬洲]]的乘客還是在該站上車比較方便，加上本站正位於中文大學附近，學生和教職員均需依賴鐵路往來中大及港九新界各地，車站附近設有接駁巴士往返中大校園各區，以及部分往[[香港科學園|香港科學園]]上班的人士會在本站轉乘[[九龍巴士272A線|九龍巴士272A線]]或[[九龍巴士272K線|272K線]]，因而該站有一定的人流支持。每逢平日早晚繁忙時間人流較多。另外，每逢假日亦因馬料水碼頭有大量前往吐露港附近郊遊地區（如[[塔門|塔門]]）的渡輪航綫而有特別多的遊人使用本站。<ref>{{cite web |title=塔門 |url=http://www.discoverhongkong.com/tc/see-do/great-outdoors/hikes/tap-mun.jsp |publisher=香港旅遊發展局 |accessdate=2018-12-15}}</ref>

另外，每逢[[馬場站_(東鐵綫)|馬場站]]開放，大學站會為一個臨時轉綫站，往[[紅磡站|紅磡]]方向列車的乘客若要前往馬場站，必須於大學站等候途經馬場站的列車。而前往火炭站的乘客若乘坐途經馬場站的列車，亦須於大學站等候途經火炭站的列車。而在2008年[[2008年夏季奧林匹克运动会|北京奧運]]及[[2008年夏季残奥会|殘奧會]]期間，前往[[香港奧運馬術比賽場地|沙田主賽場]]觀看[[2008年夏季奧林匹克運動會馬術比賽|馬術比賽]]盛裝舞步賽事和場地障礙賽的觀眾，均須在此站外面換乘大會的穿梭巴士[[九龍巴士HK1線|HK1線]]。<ref>{{cite news |title=香港奧運馬術項目門票額外增添萬四張（附短片） |url=https://www.info.gov.hk/gia/general/200807/27/P200807270170.htm |accessdate=2018-12-15 |publisher=政府新聞網 |date=2008-07-27}}</ref>

==歷史==
===易名===
大學站前稱為'''馬料水站'''，於1956年9月24日正式啟用。啟用初期每日只有南北行各三班列車停靠。隨著[[香港中文大學|香港中文大學]]即將動工，於1966年12月11日宣佈改名為'''大學站'''。2016年12月11日，適逢大學站易名50週年，當天所有憑學生八達通乘坐港鐵乘客，均可享特別優惠。

===翻新及擴建===
為應付使用大學站的乘客數量大幅增加，九廣鐵路於1998年為車站進行擴建及改建工程，包括設有商店、餐廳外，亦增設[[升降機|升降機]]連接月台，方便有需要的人士。現時南行方向的B出口為新建大堂。工程於2000年落成，並於同年10月5日舉行竣工禮。

===月台翻新===
2007年[[兩鐵合併|兩鐵合併]]後，港鐵為東鐵綫各車站月台進行較低成本的翻新工程。工程包括重新為各東鐵綫車站配上主題色，並加入代表花種，為月台橫樑及柱子翻上月台配色的油漆，貼上印有車站主題花的圖案、車站主題建築物、車站名書法字的牆紙，加設和更新直立港鐵式站牌、路線圖、指示牌、告示等等。本站被配色為淺藍色，主題花為牽牛花，街景為中大科學館。

===加設出入口===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = University Station 2017 07 part2.jpg | caption1 = D出入口大堂
}}
2012年9月28日，港鐵為應付[[三三四學制|三三四新學制]]下增加的學生人數，並且方便前往新教學大樓（康本國際學術園、伍何曼原樓及建築學院）及校巴站的人士，而增設D出入口。該出入口距離新落成的中文大學新綜合教育大樓只有3分鐘路程。工程歷時1年3個月完成。新出入口共設有4部出入閘機、1部闊閘機、1部售票機及1部增值機，可以疏導大學站其它3個出入口每日共逾3萬人次乘客流量。

新出入口採用[[環境保護|環境保護]]設計，大量採用自然光，由[[混凝土|混凝土]]陣、隔熱玻璃及反光板所組織而成，陽光從指縫滲進車站，預計每年可以節省1/3[[電力|電力]]<ref>[http://hk.news.yahoo.com/%E5%A4%A7%E5%AD%B8%E7%AB%99%E6%96%B0%E5%87%BA%E5%8F%A3%E6%8E%A1%E7%92%B0%E4%BF%9D%E8%A8%AD%E8%A8%88-221242465.html 大學站新出口採環保設計] 《星島日報》 2012年9月20日</ref>；又採用特別設計，方便收集雨水〈儲水量容達5,500升〉，用以灌溉附近的[[植物|植物]]<ref>[http://www.takungpao.com.hk/sy/2012-09/21/content_1140531.htm 港鐵大學站方便疏導客流量 將設新出口 ]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }} 《大公報》 2012年9月21日</ref>；出入口的外牆留有空間，讓風吹進車站大堂，同樣讓熱空氣從頂部透氣位升走；鐵路[[枕木|枕木]]亦改為安裝成為木櫈。工程獲[[美國綠建築協會|美國綠建築協會]]頒發「領先能源與環境設計」銀獎。<ref>[http://orientaldaily.on.cc/cnt/news/20120920/00176_035.html 大學站新出口奪環保獎]《東方日報》2012年09月20日</ref>

===重舖地台===
2015年3月起，港鐵為東鐵線車站原有的混凝土石米地台改為地磚。

==站名爭議==
2007年12月2日[[兩鐵合併|兩鐵合併]]，東鐵綫現有的'''大學站'''，與當時規劃中的港鐵[[港島綫西延|西港島綫]]「大學站」站名重複，港鐵當其時未就此事有所定案。類似情況的[[旺角東站|旺角站]]，九廣東鐵旺角站則被易名為東鐵綫[[旺角東站|旺角東站]]。有輿論認為依傳統地名，東鐵綫原有的大學站應恢復原名'''馬料水站'''，新建之大學站則命名[[石塘咀|石塘咀]]站，但港鐵發言人指西港島綫的大學站仍未正式命名，並重申名稱將會反映該區的地區特色<ref>《大學站鬧雙胞》，《明報》，2007年10月23日</ref>。2009年，西港島綫動工興建後，港鐵將西港島綫的大學站暫時易名為'''[[香港大學站|香港大學站]]'''<ref>[http://www.mtr.com.hk/chi/projects/hongkong_wil_design.html 港鐵網站-西港島綫資料]</ref>，直到2014年3月22日，根據蘋果動新聞的報道及西環變幻時群組的相片，車站地盤月台焗漆板上的毛筆字，顯示香港大學站已成為正式名稱。

其後，根據2012年1月12日一份港鐵文件，中文名字有機會更改為'''中文大學站'''，英文名字改為'''Chinese University Station'''。<ref>[http://www.mtr-southislandline.hk/pdf/consultation-meeting/legco/2nd-round/powerpoint/ppt-13-jan-12.pdf 新一代鐵路車站的設計] {{webarchive|url=https://web.archive.org/web/20120131032344/http://www.mtr-southislandline.hk/pdf/consultation-meeting/legco/2nd-round/powerpoint/ppt-13-jan-12.pdf |date=2012-01-31 }} - 港鐵公司</ref>有關消息在該文件以外，未有被其他港鐵網站確認。1月26日，港鐵公司指有關站名為「手民之誤」，尚未落實。<ref>{{cite news|title=「大學站」改名「中文大學站」 港鐵：手民之誤|url=http://hk.news.yahoo.com/%E5%A4%A7%E5%AD%B8%E7%AB%99-%E6%94%B9%E5%90%8D-%E4%B8%AD%E5%A4%A7%E7%AB%99-%E6%B8%AF%E9%90%B5-%E6%89%8B%E6%B0%91%E4%B9%8B%E8%AA%A4-211451691.html|accessdate=2012-01-26|newspaper=明報|date=2012-01-26|deadurl=yes|archiveurl=https://web.archive.org/web/20120128173843/http://hk.news.yahoo.com/%E5%A4%A7%E5%AD%B8%E7%AB%99-%E6%94%B9%E5%90%8D-%E4%B8%AD%E5%A4%A7%E7%AB%99-%E6%B8%AF%E9%90%B5-%E6%89%8B%E6%B0%91%E4%B9%8B%E8%AA%A4-211451691.html|archivedate=2012-01-28}}</ref>

==事故==
===大學站列車出軌事故===
1988年6月4日下午2時許，一列九廣鐵路（英段）{{efn|name=merger_kcr_name|在兩鐵合併前，東鐵綫的前稱為「九廣鐵路（英段）」、「東鐵」或「九廣東鐵」，並由[[九廣鐵路公司|九廣鐵路公司]]（九鐵）營運。}}南行貨運列車在大學站以北（現[[白石角|白石角]]）出軌，直到第二日早上10時40分恢復服務。期間東鐵{{efn|name=merger_kcr_name}}服務中斷一整日。[[九廣鐵路公司|九鐵]]{{efn|name=merger_kcr_name}}方面派出接駁巴士疏導乘客，但場面混亂。例如在[[沙田站|沙田站]]，晚上大批乘客爭先恐後上接駁巴士，有人甚至爬窗進入。出事地點位於白石角，該段鐵路本來是一急彎，後來於1995年把這段拉直。現在是[[香港教育大學|香港教育大學]]運動場。

=== 大學站月台跳軌事件 ===
大學站尚未設月台幕門。於2013年3月16日，一名香港中文大學女學生跳軌遭列車輾過，傷重不治。家人表示她有精神病紀錄，有定期到精神科醫生何定邦覆診治病。<ref>[http://hk.apple.nextmedia.com/news/art/20130317/18198154 中大女生大學站跳軌遭輾斃] 蘋果日報</ref>
