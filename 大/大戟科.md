'''大戟科'''（[[学名|学名]]：{{lang|la|Euphorbiaceae}}）属[[真双子叶植物|真双子叶植物]][[金虎尾目|金虎尾目]]中的一个大[[科|科]]，有240[[属|属]]大约6,000[[种|种]]。

== 形态 ==
*大部分为单[[叶|叶]]，稀为复叶的，大部分互生，少数对生；有托叶，叶子的基部或叶柄上有时具有腺体。
*[[花|花]]单性，雌雄同株或异株，总状、圆锥花序，或为杯状聚伞花序，花被两轮，各轮3～5枚，大多数不具花瓣，或全无花被；雄蕊从1个到10个以上不等，分离或合生；子房上位，通常3室；花盘发达，环状、杯状或分裂成腺体状，无花盘的稀少。
*果实多为[[蒴果|蒴果]]，成熟时分裂成3瓣，也有不开裂的[[浆果|浆果]]或[[核果|核果]]；[[种子|种子]]有丰富的胚乳。

== 分类 ==
大戟科原來包含非常廣泛的植物，但在重新分類後，很多原屬大戟科的植物被重新分配到[[葉下珠科|葉下珠科]]及其他科裡。

[[大戟亚科|大戟亚科]]的种类有杯状的聚伞花序，外表看像一朵花，实际是由一朵雌花和外围的多朵雄花组成的，单个花没有[[花瓣|花瓣]]，外边包围着杯状的苞片。

== 分布 ==
广泛分布在全世界，主要生长在[[热带|热带]]地区，尤其是[[中南半岛|中南半岛]]和热带[[美洲|美洲]]种类最多，在热带[[非洲|非洲]]也有许多种。大部分为[[草本|草本]]，但生长在[[热带|热带]]地区的种类有许多是[[灌木|灌木]]或[[乔木|乔木]]，也有类似[[仙人掌|仙人掌]]或[[石楠|石楠]]的[[多肉植物|肉质型]]的种类。[[中国|中国]]有61属大约360种，主要生长在[[长江|长江]]以南地区。

== 用途 ==
[[巴豆亚科|巴豆亚科]]和[[大戟亚科|大戟亚科]]的种类都含有乳汁，大戟亚科的乳汁有毒，巴豆亚科的无毒，有许多种类有毒，可以制[[农药|农药]]。重要的经济种包括[[橡胶树|橡胶树]]、[[油桐|油桐]]、[[蓖麻|蓖麻]]、[[木薯|木薯]]、[[巴豆|巴豆]]以及作为观赏[[花卉|花卉]]的[[一品红|一品红]]等。近年來，[[桐油樹|桐油樹]]的油亦被用以製造[[生物柴油|生物柴油]]。
