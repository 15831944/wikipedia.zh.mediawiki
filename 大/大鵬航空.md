'''大鵬航空'''全名為大鵬航空股份有限公司（[[英文|英文]]：Roc Aviation Company、RAC），成立於1992年，屬於普通航空業，是以航空攝影測量為主的專業飛航公司。

==機隊==
大鵬航空原有機隊為兩架Britten-Norman BN-2 Islander型雙螺旋槳雙引擎[[固定翼飛機|固定翼飛機]]<ref>{{Cite web |url=http://en.wikipedia.org/wiki/BN-2 |title=英文版維基百科BN-2 |accessdate= |work= |publisher=英文版維基百科 |date= }}</ref>，編號分別為B-68801及B-68802。依照[[交通部民用航空局|民航局]]的[[航空器|航空器]]記錄，編號B-68801是BN-2B-26型，出廠日期為1992年6月22日，此機塗裝是白底紅藍線條。編號B-68802是BN-2B-20型，出廠日為1991年5月24日，此機塗裝是深藍底<ref>{{Cite web |url=http://www.caa.gov.tw/big5/files/10107機齡.xls |title=2012年7月國籍航空器型別及出廠製造日期(機齡)一覽表 |accessdate= |work= |publisher=交通部民用航空局 |date= }}</ref><ref>{{Cite web |url=http://cwhung.blogspot.tw/2012/05/blog-post_29.html |title=大鵬航空參訪 |accessdate= |work= |publisher=飛行場の測候所 |date= }}</ref>。其中編號B-68801的BN-2飛機已於2012年8月墜毀於中央山脈東側。2014年9月，大鵬航空另引進3槳片的BN-2T，編號為B-68808。

==事故==
{{Main|2012年大鵬航空BN-2墜機事件}}
*2012年8月30日，編號B-68801的空照機，上午7時25分從[[臺北松山機場|臺北松山機場]]起飛，執行[[宜蘭縣|宜蘭]]-[[花蓮縣|花蓮]]空拍作業中，9時32分在花東交界區附近發出緊急訊號後失蹤。9月1日上午9時55分，由協助支援搜救之[[群鷹翔國土資源航空|群鷹翔航空公司]]，回報目視疑似失聯飛機地點，經緯度為北緯23度20分23.28秒東經121度01分44.11秒（{{coord|23|20|23.28|N|121|1|44.11|E}}）<ref>{{Cite web |url=http://www.caa.gov.tw/big5/news/index01.asp?sno=595 |title=大鵬航空失聯B-68801號飛機尋獲 |accessdate=2012-09-01 |work= |publisher=交通部民用航空局 |date=2012-09-01 }}</ref>
