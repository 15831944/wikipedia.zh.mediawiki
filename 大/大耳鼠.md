'''大耳鼠'''（'''チンプイ'''）（或譯作'''超能大耳獸'''、'''-{大耳獸之比}-'''、'''大耳獸芝比'''、'''-{大耳獸普普}-'''、'''金普依'''、'''外星小叮叮'''）是[[藤子不二雄|藤子·F·不二雄]]較晚期所著的一部[[漫畫|漫畫]]及[[動畫|動畫]]。

== 概要 ==
'''大耳鼠'''的漫畫於1985年－1988年連載於《[[藤子不二雄|藤子不二雄]]園地》([[藤子不二雄|藤子不二雄]]ランド)，一共58話，原本預定要畫的59話及60話，因為[[藤子不二雄|藤子·F·不二雄]]的去世而沒有完成，也因此沒有最後的結局。動畫最早於1989年11月－1991年4月於[[朝日電視台|朝日電視台]]放映，共112話。

== 故事介紹 ==
描寫一個平凡的地球少女'''春日恵理'''，被馬爾星人選為呂諾夫太子的妃子，由於她的拒絕，便讓馬爾星的大耳鼠之比留在地球陪伴惠理生活，希望能逐漸使她答應婚事，而之比也成了惠理的好朋友。由於馬爾星擁有比地球更高的神奇科技，馬爾星人也會使用「科法」（倚藉科學方法進行的，而非「魔法」），使得在惠理的生活中發生了很多趣事。

== 主要登場人物 ==
=== 地球人 ===
* '''[[春日惠理|春日惠理]]'''（{{lang|ja|春日 エリ}}）（CV：{{flagicon|Japan}}[[林原惠|林原惠]]；{{flagicon|Taiwan}}[[龍顯蕙|龍顯蕙]]；{{flagicon|Hong Kong}}[[鄭麗麗|鄭麗麗]]）
: 漫畫的主角，小學六年級女生，個性活潑開朗大方，但容易性急生氣，喜好偷懶，功課不太好。角色設定類似《[[多啦A夢|多啦A夢]]》中的[[野比大雄|大雄]]的女性化和[[源靜香|靜香]]。
* '''春日百合'''（{{lang|ja|春日 百合}}）（CV：{{flagicon|Japan}}[[鈴木弘子|鈴木弘子]]；{{flagicon|Taiwan}}[[許雲雲|許雲雲]]）
: 恵理的媽媽。個性和恵理一樣大而化之，但又性急易生氣，特色是喜歡使喚人做事。
* '''春日和'''（{{lang|ja|春日 のどか}}）（CV：{{flagicon|Japan}}[[津村鷹志|津村鷹志]]；{{flagicon|Taiwan}}[[陳幼文|陳幼文]]）
: 恵理的爸爸。
* '''內木 翔'''（{{lang|ja|內木 翔}}）（CV：{{flagicon|Japan}}[[佐佐木望|佐佐木望]]；{{flagicon|Taiwan}}[[許雲雲|許雲雲]]；{{flagicon|Hong Kong}}[[盧素娟|盧素娟]]）
: 恵理喜歡的男同學，個性和善頭腦好成績不錯，但體育方面（如[[棒球|棒球]]）稍差。此角色在漫畫版的姓名為'''內木一郎'''，電視版為'''內木翔'''。角色設定類似《[[多啦A夢|多啦A夢]]》中的[[出木杉英才|出木杉英才]]。
* '''小金山詩奈美'''（{{lang|ja|小金山スネ美}}）（CV：{{flagicon|Japan}}[[梨羽侑里|梨羽侑里]]；{{flagicon|Taiwan}}[[雷碧文|雷碧文]]）
:恵理的同學，家裡有錢，又是[[貴族|貴族]]的後代，很喜歡誇躍自己，也很喜歡內木。角色設定類似《[[多啦A夢|多啦A夢]]》中的[[小夫|小夫]]的[[女性化|女性化]]。
* '''大江山政男'''（{{lang|ja|大江山 政男}}）（CV：{{flagicon|Japan}}[[中村大樹|中村大樹]]；{{flagicon|Taiwan}}[[陳幼文|陳幼文]]）
: 強壯的男生，是同學之中的孩子王。角色設定類似《[[多啦A夢|多啦A夢]]》中的[[胖虎|胖虎]]。

=== 馬爾星人 ===
* '''[[之比|之比]]'''（{{lang|ja|チンプイ}}）（CV：{{flagicon|Japan}}[[堀絢子|堀絢子]]；{{flagicon|Taiwan}}[[雷碧文|雷碧文]]；{{flagicon|Hong Kong}}[[林丹鳳|林丹鳳]]）
: 馬爾星球來的外星鼠，常運用馬爾星球的科技來幫助恵理，個性和善，特別喜歡吃[[拉麵|拉麵]]。
* '''汪達'''（{{lang|ja|ワンダユウ}}）（CV：{{flagicon|Japan}}[[八奈見乘兒|八奈見乘兒]]；{{flagicon|Taiwan}}[[黃天佑|黃天佑]]；{{flagicon|Hong Kong}}[[陳永信|陳永信]]）
: 馬爾星的狗型外星人，非常急切希望恵理嫁到馬爾星去，為此常想出種種奇怪的方式。
* '''呂諾夫殿下'''（{{lang|ja|ルルロフ殿下}}）（CV：{{flagicon|Japan}}[[菊池正美|菊池正美]]；{{flagicon|Hong Kong}}[[黃鳳英|黃鳳英]]）
: 馬爾星球的王子，在漫畫中從未露面，根據旺德鼬和普普的描述，是一個無論頭腦、體育或其他方面都很強的人。
* '''老爺爺'''（{{lang|ja|ご隠居}}）（CV：{{flagicon|Japan}}[[永井一郎|永井一郎]]）
: 呂諾夫殿下的爺爺。對汪達他們很嚴格，不過也是個和藹可親的老爺爺。
* '''佳拉喜'''
: 貓型馬爾星人，喜歡普普，一開始普普很怕佳拉喜，很有可能是老鼠怕貓的特性，但佳拉喜其實溫柔體貼，後來普普對佳拉喜產生好感。
