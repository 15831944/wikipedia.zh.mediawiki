'''大理'''（[[白语|白语]]：Dablit Guaif；937年－1094年，1096年－1253年）是一個位於[[雲南|雲南]]及周邊地區由[[白蠻|白蠻]]首领[[段思平|段思平]]建立的政權。其政治中心在[[洱海|洱海]]一帶，疆域大概是現在的[[中华人民共和国|中国]][[雲南省|雲南省]]、[[貴州省|貴州省]]、[[四川省|四川省]]西南部、[[缅甸|缅甸]]北部地區及[[印度|印度]]、[[老撾|老撾]]與[[越南|越南]]的少部分地區。

[[唐玄宗|唐玄宗]][[開元|開元]]年間 (738年)，[[西洱河|西洱河]]地區的諸部落在唐的扶植下建國[[南诏|南诏]]。唐末時期 (902年) [[鄭買嗣|鄭買嗣]]篡南詔建[[大長和|大長和]]。經歷了幾十年戰亂之後，大理國於五代[[後晉|後晉]]初年 (937年) 由[[段思平|段思平]]建國。大理都城[[羊苴咩城|羊苴咩城]]（今云南省大理市），國號'''大理'''，以其尊崇[[佛教|佛教]]，又稱'''妙香大國'''<ref>{{cite book |language = zh-cn |author = 徐俊 |title = 中国古代王朝和政权名号探源 |publisher = 华中师范大学出版社 |date = 2000年11月 |location = 湖北武昌 |pages = 252-253 |ISBN = 7-5622-2277-0 }}</ref>。羊苴咩城也就改称大理城。宋哲宗年間 (1095年)，[[高昇泰|高昇泰]]篡位，改国号为“[[大中_(国家)|大中]]”，1096年高昇泰死后，归政于[[段正淳|段正淳]]，史称'''后理'''{{efn|“后理”意为“后大理”，但也有说法认为“后”是尊称，“后理”即“大理”。见 杨延福：《辨后理国之“后”》，《云南文物》，1983（14）}}。1253年，[[蒙古帝国|蒙古帝国]]的[[忽必烈|忽必烈]]（即后来的元世祖）奉[[蒙哥|蒙哥]]大汗之命“革囊渡江”征[[云南|云南]]，灭大理国，后建[[云南等处行中书省|云南等处行中书省]]，原大理国王段氏被任为世袭[[大理总管|大理总管]]。

== 历史 ==
{{雲南历史}}

=== 建国 ===
[[段思平|段思平]]先祖為[[武威郡|武威郡]]人，始祖[[段儉魏|段儉魏]]，佐南詔大蒙國蒙氏為清平官，為[[東汉|東汉]][[武威太守|武威太守]][[段熲|段熲]]的第17代子孙，六傳至[[段思平|段思平]]。思平為[[幕覽|幕覽]](小府副將)，後因戰功，成為[[通海節度使|通海節度使]]。公元929年，[[楊干貞|楊干貞]]奪[[趙善政|趙善政]]之位，建[[大義寧|大義寧]]，改元[[兴圣|兴圣]]， 其弟弟[[楊詔|楊詔]]指[[段思平|段思平]]有帝王之相。在疑懼下，[[楊干貞|楊干貞]]急欲追殺[[段思平|段思平]]。

在被[[楊干貞|楊干貞]]追殺期間，[[段思平|段思平]]曾到[[秀山|秀山]]神祠占卜得到啟示，藏匿在自己舅父的部族中。會善政臣守[[高方|高方]]和[[段思平|段思平]]關係密切，派[[段思平|段思平]]之弟[[段思良|段思良]]和軍師[[董迦羅|董迦羅]]前來加以保護。

930年，[[楊干貞|楊干貞]]的位置被其弟杨诏所篡，杨诏得位后，改元[[大明_(杨干真)|大明]]<ref>《[[大理國史|大理國史]]》，[[段玉明|段玉明]]</ref>。

[[段思平|段思平]]向東方的黑爨三十七蠻部借兵，會於[[石城|石城]]，以[[董迦羅|董迦羅]]為軍師，討伐[[楊干貞|楊干貞]]，所向皆克，遂進攻[[大理|大理]]。

當時[[楊詔|楊詔]]等陳兵據橋，段軍不能通過。當晚[[段思平|段思平]]在睡夢中得到三個夢境：「人無首；玉瓶無耳；鏡破」，感到大惑不解，其軍師[[董迦羅|董迦羅]]指[[段思平|段思平]]的夢境乃天大吉兆，因為「君乃丈夫，去首為天；玉瓶去耳為王；鏡破則無對者」，因而軍心大振。是日，段軍找到一名浣紗婦女，指引段軍適合大軍渡河的地點，曰：「人從我江尾，馬從三沙矣，爾國名大理。」

[[段思平|段思平]]大軍按照浣紗婦女的指引，渡河成功，楊詔兵敗自殺，楊干貞知道兵敗消息后棄城而逃，為段思平軍所擒，大義寧滅亡，段思平即位，建號大理國，建元文德，仍都大理。

=== 與宋朝的關係 ===
目前能查到大理國與同一時期的[[宋朝|宋朝]]相關史料並不是很多，內容主要也以經濟關係為主，有時甚至會中断貿易。

整體來說，大理與宋朝關係並不密切。[[宋神宗|宋神宗]][[熙寧|熙寧]]九年 (1076年)，大理國遣使貢金裝碧玕山、氈罽、刀劍、犀皮甲鞍轡。[[宋徽宗|宋徽宗]][[政和|政和]]五年 (1115年)，[[段和譽|段和譽]]曾派使請求入貢。七年 (1117年)，使者至東京，貢馬三百八十匹及麝香、牛黃、細氈、碧玕山諸物，宋廷封段氏為金紫光祿大夫、檢校司空、雲南節度使、上柱國、大理國王。不过，大理三百年来實際上一直是宋朝的西南屏障，使宋朝得以專注应对北方政權的威脅。直至[[宋理宗|宋理宗]]年間大理被[[蒙古|蒙古]]攻滅後，[[忽必烈|忽必烈]]從西南方打開缺口對南宋進攻，對南宋始構成直接威脅。

=== 高氏掌權 ===
{{See also|大中國 (國家)|中国公}}
1080年，[[楊義貞|楊義貞]]弒大理皇帝[[段廉義|段廉義]]，自立為帝，因[[高昇泰|高昇泰]]受其父[[高智昇|高智昇]]之命滅楊義貞，擁立[[段壽輝|段壽輝]]為大理皇帝，並因平亂有功昇為鄯闡，隔年高智昇與高昇泰父子以天變為不祥理由，逼迫上明帝段壽輝退位出家，並擁立段正明繼位。由於段正明在位時，掌大權仍是[[高昇泰|高昇泰]]，甚至在1094年廢了段正明，並自立為帝。1096年，高昇泰基於雲南諸部反對的理由，在臨終囑咐其子[[高泰明|高泰明]]還政段氏，於是[[高泰明|高泰明]]擁立[[段正明|段正明]]的弟弟[[段正淳|段正淳]]為大理皇帝，但在後大理國時期仍占有重要地位直到後大理滅國。

=== 大理蒙古战争 ===
{{Main|大理蒙古戰爭}}
孝義皇帝[[段祥興|段祥興]]在位期間，正值蒙古大舉南侵之時，戰爭造成大理國大敗，大理派出的大將[[高禾|高禾]]因此戰死。雖說當時蒙古軍後因[[窩闊台|窩闊台]]死亡而暫時退兵。

=== 灭亡 ===
{{Main|大理總管府}}
1251年段祥興駕崩，傳位給他的兒子[[段興智|段興智]]。隔年蒙古再次攻打大理，最終相國[[高泰祥|高泰祥]]戰敗被殺，段興智逃至[[善闡府|鄯闡]]，蒙古大將[[兀良哈台|兀良哈台]]攻下鄯闡，並生擒段興智，滅了大理國，[[蒙哥|蒙哥]]為表示自己的仁慈及安撫遺民，封段興智為世襲總管。

大理國雖然滅亡，但直到1257年[[忽必烈|忽必烈]]成立[[雲南等處行中書省|雲南等處行中書省]]為止，段氏仍然是大理故地的統治者，到[[段俊|段俊]]時段氏更壟斷雲南行省平章政事一職。

在[[元朝|元]]末時，段氏大理總管與[[元梁王|元梁王]]衝突升高，甚至在1343年發生戰爭。後在[[明朝|明]]興之時謀求獨立，但遭到[[藍玉|藍玉]]、[[沐英|沐英]]的進攻而化為泡影。

== 政治制度 ==
大理國政治制度，多因襲[[南詔|南詔]]舊制，加強文官統治。國君稱「驃信」，宰相稱「[[清平官|清平官]]」，有坦綽、布燮、久贊、彥贊。其中彥贊為大理所增設。清平官下有「九爽」。

== 文化 ==
=== 佛教 ===
由於在南詔後期時，佛教開始興盛，到大理國時期達到極盛，並崇尚佛教[[密宗|密宗]][[阿利僧派|阿利僧派]]，所開筵席全是素食，甚至大理國國王22代，就有7代國王禪位為僧。在1976年清理重要寺院[[崇圣寺_(大理)|崇聖寺]]的主塔時，甚至在塔頂發現和唐代八世紀造像風格相同的觀音像，這說明了大理受到佛教的影響非常深遠。

== 行政區劃 ==
大理國後期廢除了[[節度|節度]]和[[都督|都督]]的軍事轄區，對其他建制略作了調整，設置八[[府|府]]、四[[郡|郡]]、四[[鎮|鎮]]。
* '''八府'''：除大理首府以外的善闡府（今[[昆明|昆明]]）、威楚府（今[[楚雄|楚雄]]）、統矢府（即弄棟府，今[[姚安|姚安]]）、會川府（今[[會理|會理]]）、建昌府（今[[西昌|西昌]]）、騰越府（今[[騰沖|騰沖]]）、謀統府（今[[鶴慶|鶴慶]]）、永昌府（今[[保山|保山]]）；
* '''四郡'''：東川郡（今[[會澤|會澤]]）、石城郡（今[[曲靖|曲靖]]）、河陽郡（今[[澄江|澄江]]）、秀山郡（今[[通海|通海]]）；
* '''四鎮'''：西北的成紀鎮（今[[永勝|永勝]]）、西南的蒙合鎮(今[[巍山|巍山]]）、西部的鎮西鎮(今[[盈江|盈江]]）、東部的最寧鎮(今[[開遠|開遠]]）。
*'''三十七部'''：普摩部(今[[曲靖|曲靖]])、磨弥部(今[[沾益|沾益]])、纳垢部(今[[马龙|马龙]])、罗鸠部(今[[罗平|罗平]])、夜苴部(今[[富源|富源]])、磨弥殿部(今[[宣威|宣威]])、落温部(今[[陆良|陆良]])、落蒙部(今[[路南|路南]])、师宗部(今[[师宗|师宗]])、仁德部(今[[寻甸|寻甸]])、閟畔部(今[[东川|东川]])、嵩盟部(今[[嵩明|嵩明]])、际鹿部(今[[泸西|泸西]])、维摩部(今[[丘北|丘北]])、弥勒部(今[[弥勒|弥勒]])、阳城堡部(今[[晋宁|晋宁]])、强宗部(今[[阳宗海|阳宗海]])、步雄部(今[[江川|江川]])、罗加部(今[[澄江|澄江]])、宁部(今[[华宁|华宁]])、休腊部(今[[河西|河西]])、因远部(今[[元江|元江]]）、罗婺部（今[[武定|武定]]）、华竹部（今[[元谋|元谋]]）、罗部（今[[罗次|罗次]]）、屈中司部（今[[开远|开远]]）、纳楼部（今[[蒙自|蒙自]]）、教合部（今[[文山壯族苗族自治州|文山]]）、矣尼迦部（今[[马关|马关]]）、王弄山部（今[[河口|河口]]）、乌蒙部（今[[昭通|昭通]]）、乃娘部（今[[彝良|彝良]]）、芒布部（今[[镇雄|镇雄]]）、乌撒部（今[[威宁|威宁]]）、于矢部（今[[普安|普安]]）、休制部（今[[玉溪|玉溪]]）、嶍峨部（今[[峨山|峨山]]）。
* '''賧'''（甸）等則隸屬大理首府、府、郡、鎮。
