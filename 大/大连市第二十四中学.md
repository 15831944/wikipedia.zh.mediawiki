'''大连市第二十四中学'''位于[[中华人民共和国|中国]][[辽宁省|辽宁省]][[大连市|大连市]][[中山区_(大连市)|中山区]]，是一所辽宁省著名的重点中学，在大连简称为二十四中。二十四中在大连享有较高声誉。学校被誉为东北小清华，是辽宁省最好的高中之一。<ref>[http://www.dlhs24.com.cn/xxgk/ShowArticle.asp?ArticleID=65 大连市第二十四中学简介] {{webarchive|url=https://web.archive.org/web/20110614064125/http://www.dlhs24.com.cn/xxgk/ShowArticle.asp?ArticleID=65 |date=2011-06-14 }}</ref>如今拥有各类社团40余个，并拥有学生自办刊物《蓝帆》。

== 位置 ==
大连市第二十四中学地址是大连市中山区解放路217号，位于中山区[[劳动公园|劳动公园]]南侧的绿山旁。中学校舍的一侧原是劳动公园小学，由于校园较小，二十四中曾与劳动小学共用一个操场。但现在劳动小学原有校舍和操场已被二十四中学合并。

== 校史 ==
大连市第二十四中学正式建校为1958年9月，其历史可以追溯到1920年日本占领时期由日本人设立的'''大连第五寻常小学校'''。前总理[[周恩来|周恩来]]曾为学校题词。
以下为学校的主要历史沿革：
* 1920年：设立'''大连第五寻常小学校'''；
* 1925年：改为'''大连春日小学校'''；
* 1941年：改为'''大连春日国民学校'''；
* 1946年：日本投降，撤出大连，改为'''大连日侨初等学校第七校'''；
* 1949年：改名为'''育才学校'''；时任校长为[[毛达恂|毛达恂]]（原[[旅大市|旅大市]]第一任地委书记，[[杨开慧|杨开慧]]的堂妹[[杨开英|杨开英]]任第一届教育主任）<ref>[http://www.ln.gov.cn/zfxx/qsgd/dls/rwdl/200912/t20091207_455113.html 大连二十四中庆祝建校60周年 ]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}辽宁省人民政府，2009年12月07日</ref>。
* 1951年：'''大连市第四初中'''成立；
* 1956年秋：育才学校与大连市第四初中合并，校名使用第四初中； 
* 1958年起：改称'''大连第二十四中学'''。
* 1962年：被确定为辽宁省重点中学。
* 2004年：被评为辽宁省示范性高中。

== 教育资源==
学校占地面积29440平方米，建筑面积28675平方米，绿化面积8960平方米，设有体育馆、科技馆、报告厅、图书馆以及各种特色特长教室等。学校现有教职员工123人、教学班36个，学生1500余名。

== 办学实力==
大连市第二十四中学在素质教育方面一直走在东北地区的前列，学校坚持育人为本，全面贯彻素质教育的办学方针。让学生得到全面发展。

先后培养出20名特级教师，200余名高级教师，市级骨干教师和学科带头人33人。
