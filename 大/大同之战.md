'''大同之战'''是指中国[[明朝|明朝]][[明英宗|明英宗]]正统十四年（1449年），明朝同[[瓦刺|瓦刺]]蒙古部落在山西大同的一场军事斗争。

== 背景 ==
[[元朝|元朝]]灭亡后，蒙古分成[[瓦剌|瓦剌]]、[[鞑靼|鞑靼]]和[[兀良哈|兀良哈]]三大部。为了防止蒙古南下，明朝在北方设立[[九边|九边]]，同时还驻守着四十万军队。而[[大同|大同]]就是九边重镇之一。由于大同地理位的重要性，明太祖[[朱元璋|朱元璋]]封他的第十三子[[朱桂|朱桂]]为代王，坐镇大同，又在大同设置[[山西行都指挥司|山西行都指挥司]]，管理山西一切军政大事。

[[明英宗|明英宗]]时，[[瓦剌|瓦剌]]部落逐渐强盛起来。瓦剌头领[[马哈木|马哈木]]死后，他的儿子[[脱欢|脱欢]]吞并了鞑靼和兀良哈，统一了蒙古各部族。脱欢死后，其子[[也先|也先]]继承王位，控制了东起[[辽河|辽河]]，西至关山东麓的大片土地。为了和平相处，明朝与瓦剌达成协议，明朝在[[大同|大同]]、[[宣府|宣府]]等地设立[[马市|马市]]，双方互相贸易，交流物资，而瓦剌以马匹来换取中原的布匹、粮食、瓷器、药材、武器、铁制品等。正统十一年（1446年）冬季，也先因为粮食短缺，派使臣来大同地区借粮，还要求与大同的守备太监[[郭敬|郭敬]]商谈事宜。明英宗得知此事后，命郭敬不要见瓦剌的使臣，拒绝了瓦剌的要求，引起瓦剌的不满。之后，明朝政府又单方面撕毁贸易协议，随意降低了马价，使瓦剌受到损失。从此，明朝与瓦剌关系破裂，局势紧张，双方都在调集军队，加紧备战。

== 过程 ==
正统十四年（1449年）秋七月，[[也先|也先]]召集諸部兵分四路大舉進攻明朝邊境。[[脫脫不花|脫脫不花]]以兀良哈攻遼東，阿剌知院攻宣府（今[[河北|河北]][[宣化区|宣化]]），圍[[赤城县|赤城]]，又遣別騎攻[[甘州|甘州]]。七月十一日，也先攻[[大同市|大同]]，明朝大同右参将[[吴浩|吴浩]]于猫儿庄（今[[山西|山西]]阳高县北一带）迎战瓦剌，迅即兵败战死{{参|《明史》（卷328）：「十四年七月，遂誘脅諸番，分道大舉入寇。脫脫不花以兀良哈寇遼東，阿剌知院寇宣府，圍赤城，又遣別騎寇甘州，也先自寇大同。參將吳浩戰死貓兒莊，羽書踵至。」}}。七月十五日，大同总督[[宋瑛|宋瑛]]、驸马都尉[[井源|井源]]、总兵官[[朱冕|朱冕]]、左参将都督[[石亨|石亨]]四员将领各率兵一万赴阳和（今山西阳高县西北）地区防御。紧接着，西寧侯[[宋瑛|宋瑛]]、武进伯朱冕、左参将石亨等率明军與瓦剌戰於陽和。由于太监[[郭敬|郭敬]]在其中作梗，致使明军大败，全军覆灭。宋瑛、朱冕战死，石亨單騎奔还逃回大同城内，郭敬躲藏在草丛中才躲过一劫{{参|《明史》（卷10）：「秋七月己丑，瓦剌也先寇大同，參將吳浩戰死，下詔親征。吏部尚書王直帥隣臣諫，不聽。癸巳，命郕王居守。是日，西寧侯宋瑛、武進伯朱冕與瓦剌戰於陽和，敗沒。」}}{{参|《明史》（卷173）：其秋，也先大舉寇大同，亨及西寧侯宋瑛、武進伯朱冕等戰陽和口。瑛、冕戰沒，亨單騎奔還。}}{{参|{{cite web|author1=翟禹|title=论“土木之变”|url=http://www.nmgass.com.cn/zycg/zycg/2017-03-21/164514.html|publisher=内蒙古社会科学院历史研究所|accessdate=2013-12-26}}}}。也先军队锐不可挡，大同明军交战失利。塞外城堡，接连失陷。明军几经接战，前线败报频传{{参|《明史紀事本末》（卷32）：“秋七月，也先图犯边，其势甚张。侍讲徐呈语其友刘溥曰：“祸不远矣！”亟命妻子南归，皆重迁，有难色。呈怒曰：“尔不急去，不欲作中国妇耶！”乃行。八日，也先大举入寇，兵锋锐甚。大同兵失利，塞外城堡，所至陷没。边报日至，乃遣驸马都尉井源等四将，各率兵万人出御之。源等既行，太监王振劝上亲征。”}}。

== 后续 ==
太監王振此前曾有主导远征[[云南|云南]][[麋川土司|麋川土司]]获胜的功绩，此时希望借与蒙古开战的机会继续增大自己的权利，便蛊惑明英宗力主御驾親征。[[吏部尚書|吏部尚書]][[王直_(明朝官员)|王直]]等人认为“边鄙之事，自古有之”，只要“将士用命，必可图胜”。朝廷应以守为主，劝谏英宗“不必亲御六师，以临塞下”。英宗则全然不顾，在不知瓦剌军队主力方向的情况下，诏令迅速集结军队，两日内出兵讨伐也先{{参|{{cite web|author1=孟修|title=土木之变与庚戌之变比较研究——朝贡体制框架下的明蒙关系新探|url=http://www.doc88.com/p-8156141396242.html|publisher=《中国长城博物馆》|accessdate=2011年第2期}}}}。

正统十四年(1449年)七月十七日，明英宗不顧吏部尚書王直等群臣反對偕同王振率军五十餘萬御驾亲征。命皇弟[[郕王|郕王]][[朱祁鈺|朱祁鈺]]留守京师{{参|《明史紀事本末》（卷32）：「遂偕王振並官軍五十餘萬人。」但李贤《天顺日录》载：“二十余万人中伤居半，死者三之一，骡马亦二十余万，衣甲兵器尽为胡人所得，满载而还。”}}。后明英宗兵败被俘，称[[土木堡之变|土木堡之变]]。朱祁镇被俘后，也先曾两次带着朱祁镇以索取金币为名，攻打大同。大同都督[[郭登|郭登]]识破了也先的计谋，曾两次设计，想夺回明英宗，袭击瓦刺军，但都没有成功。
