'''大报恩寺'''位于[[江苏省|江苏省]][[南京市|南京市]][[秦淮区|秦淮区]][[中华门_(南京)|中华门]]外雨花路东侧[[秦淮河|秦淮河]]畔长干里，原址为南朝时期著名的[[长干寺|长干寺]]。大报恩寺中著名的琉璃塔修建于明朝[[永乐_(明朝)|永乐]]至[[宣德|宣德]]年间。

== 历史 ==

明朝修建大报恩寺的原因有不同的传说，一说是[[明朝|明朝]][[明成祖|永乐帝（朱棣]]）为纪念其惨死的疑似为其生母的[[碽妃|碽妃]]而興建的；另一说是为了纪念'''建文帝'''[[朱允炆|朱允炆]]。在1412年到1431年期间兴建的一组规模庞大，有如宫殿般金碧辉煌的建筑群。大报恩寺施工极其考究，完全按照皇宫的标准来营建。地基上先钉入粗大木桩，然后纵火焚烧，使之变成[[木炭|木炭]]，再用铁轮滚石碾压夯实，木炭上加铺一层朱砂，以防潮、杀虫。寺内有殿阁20多座，画廊118处，经房38间。历时19年，耗银250万两，徵调工役10多万人。

位于大殿后的大报恩寺[[琉璃|琉璃]]塔建造于永乐十年（1412年），[[宣德|宣德]]三年（1428年）竣工，九层八面，高达78.2米，甚至数十里外长江上也可望见。塔身白瓷贴面，拱门琉璃门券。門框飾有狮子、白象、飞羊等佛教題材的五色琉璃磚。刹顶镶嵌金银珠宝。角梁下悬挂風鈴152个，日夜作響，聲聞數里。自建成之日起就点燃长明塔灯140盞，每天耗油64斤，金碧辉煌，昼夜通明。塔内壁布满佛龛。该塔是[[金陵四十八景|金陵四十八景]]之一。明清时代，一些欧州商人、游客和传教士来到南京，称之为「南京瓷塔」，将它与[[罗马鬥獸场|罗马鬥獸场]]、[[亚历山大地下陵墓|亚历山大地下陵墓]]、[[比萨斜塔|比萨斜塔]]相媲美，称之为[[中古世界七大奇观|中古世界七大奇观]]之一，也是南京的象征。

1856年，[[太平天国|太平天国]]发生内乱，由于担心[[石达开|石达开]]部队占据制高点向城内发炮，大报恩寺被北王[[韦昌辉|韦昌辉]]下令炸毁，仅存一青铜色塔刹（1930年代后失踪）和8米高的石碑。相传建造该塔时曾一式烧製三份琉璃构件，一份用来建塔，两份埋入地下用于替换。1958年在[[眼香庙|眼香庙]]、[[芙蓉山|芙蓉山]]、窑岗村一带出土的大批琉璃构件上多带有墨书的字号标记。构件现分藏于[[中国国家博物馆|中国国家博物馆]]、[[南京博物院|南京博物院]]和[[南京市博物馆|南京市博物馆]]。2004年后南京市政府开始筹划复建事宜，2007年，大报恩寺遗址公园正式启动建设。2008年8月7日，在南京大报恩寺遗址出土的铁函中发现了[[七宝阿育王塔|七宝阿育王塔]]，内藏“佛顶真骨”。被评为2010年度[[全国十大考古新发现|全国十大考古新发现]]。

大报恩寺重建工程是南京市2010年城市建设“十六个重大项目”之一，[[万达集团|万达集团]]董事长[[王健林|王健林]]向该工程捐赠10亿元人民币。2015年12月16日，大报恩寺遗址公园开园并于次日正式开放。

== 现存遗迹 ==
*大报恩寺碑：包括南面的御制大报恩寺左碑和北面的御制大报恩寺右碑两座，分别刻制于明[[永乐_(明朝)|永乐]]二十二年（1424年）和[[宣德|宣德]]三年（1428年），两碑原对称布置于寺内香河桥南北两侧，相距约100米，原碑亭今均已不存。现状左碑碑身、碑额已毁，仅存[[赑屃|龟趺座]]''（图）''，右碑尚基本完好，但龟趺座已残缺，碑文模糊不清<ref>{{cite web |url=http://news.sina.com.cn/o/2009-10-10/022616411436s.shtml|title=《大报恩寺“露出”香河桥》|accessdate=2011-12-10|work=蔡震|publisher=扬子晚报}}</ref>。《金陵梵刹志》一书中录有两碑碑文，左碑约计444字，右碑约计429字<ref>{{cite web|url=http://www.nju.gov.cn/web_xx/html/s/?target=16-2625.html|title=《金陵九大古碑——为母报恩大报恩寺碑》|accessdate=2011-12-10|work=微风|publisher=南京日报|archive-url=https://web.archive.org/web/20101220230603/http://nju.gov.cn/web%5Fxx/html/s/?target=16-2625.html|archive-date=2010-12-20|dead-url=yes}}</ref>。
