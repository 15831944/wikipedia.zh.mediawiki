'''大茂山'''，又名'''神仙山'''，原为中国[[五岳|五岳]]之一的北岳'''恒山'''，<ref>{{Cite web |url=http://www.tangxian.gov.cn/web/web/newsInfo.asp?id=CA2D3124-E287-42A4-B2A6-9086552A4452&menuType= |title=古北岳恒山大茂山生态旅游区 |access-date=2012-10-24 |archive-url=https://web.archive.org/web/20160304134456/http://www.tangxian.gov.cn/web/web/newsInfo.asp?id=CA2D3124-E287-42A4-B2A6-9086552A4452&menuType= |archive-date=2016-03-04 |dead-url=yes }}</ref>位于中国[[河北省|河北省]][[阜平县|阜平县]]（1193年析置自[[曲阳县|曲阳县]]）、[[唐县|唐县]]、[[涞源县|涞源县]]交界处，主峰位于唐县[[石门乡_(唐县)|石门乡]]，为古[[五岳|北岳]]所在地。大茂山距离唐县县城西北75公里，距离[[保定|保定]]市区120公里。河北古北岳国家森林公园即位于大茂山地区。<ref name="gm">[http://www.hebei.gov.cn/article/20100203/1382824.htm 大茂山国家森林公园更名]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 历史 ==
大茂山古称恒山，其历史可上溯到传说中的[[三皇五帝|三皇五帝]]时期，据《史记·封禅书》记载，[[舜|舜]]曾经到恒山祭祀。<ref>“(舜)十一月巡狩至北岳,北岳恒山也,皆如岱宗之礼。”西汉·司马迁《史记》</ref>[[汉文帝|汉文帝]]时，因[[避讳|避讳]]缘故，恒山曾改名为常山。[[汉武帝|汉武帝]]时，在河北曲阳设立北岳祠祭祀。[[汉宣帝|汉宣帝]]神爵元年（前61年），诏封五岳，其中以恒山为北岳。<ref>“其三月，幸河东，祠后土，有神爵(雀)集，改元为神爵。制诏太常：“夫江海，百川之大者也，今阙焉无祠。其令祠官以礼为岁事，以四时祠江海洛水，祈为天下丰年焉。”自是五岳、四渎皆有常礼。东岳泰山于博，中岳泰室于嵩高，南岳潜山于潜，西岳华山于华阴，北岳常山于上曲阳，河于临晋，江于江都，淮于平氏，济于临邑界中，皆使者持节侍祠。”参见《汉书》郊祀志第五下</ref>[[北魏宣武帝|北魏宣武帝]]时，始在大茂山之南的曲阳县城建北岳庙，之后历代多在北岳庙遥祭北岳。[[北宋|北宋]]时期，大茂山一带曾为[[宋|宋]][[辽|辽]]边境。<ref>“北岳常山，今谓之大茂山者是也，半属契丹，以大茂山分脊为界，岳祠旧在山下。”北宋·沈括《梦溪笔谈》卷二十四 杂志一</ref>1193年，析曲阳县北部以及其他县地置阜平县，自此大茂山不再隶属曲阳县，但北岳庙仍位于曲阳县城。明代中后期，出现北岳在河北曲阳还是山西[[浑源|浑源]]的争论，但是当时仍将恒山定在河北曲阳。直到清顺治十七年（1660年）七月，北岳改为山西浑源境内的[[恒山|恒山]]。<ref name="baodingribao">[http://news.bdall.com/epaper/bdrb/html/2009-08/10/content_91316.htm　古北岳大茂山《保定日报》]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 自然景观 ==
大茂山包含太乙峰、爷峰、奶奶峰、媳妇峰、母子峰、贞女峰等20余座山峰，其中主峰太乙峰海拔1898.4米。大茂山为[[唐河|唐河]]支流恒水河（通天河）的发源地。动植物资源丰富，天然次生林333.33公顷，森林覆盖率达66.5%。1997年，建为省级森林公园；2005年，建为大茂山国家森林公园；2010年1月，大茂山国家森林公园更名为河北古北岳国家森林公园，并将倒马关省级森林公园（大石峪、秀水峪景区）纳入其中。<ref name="gm"></ref><ref name="baodingribao"></ref>大茂山有天然洞穴（黄龙洞、黑龙洞等）、飞瀑等景观，恒山积雪为唐县古八景之一。<ref>[http://news.sina.com.cn/c/2003-09-02/16221662653.shtml 图文：河北保定观光景点--大茂山原始森林]</ref><ref>[http://www.heb.chinanews.com/todaybd/news/msgj/2006-04-13/113.html 大茂山]</ref>

== 人文景观 ==
大茂山曾为佛教、道教圣地，山顶建有北岳恒山神庙、奶奶庙等建筑。[[北岳庙|北岳庙]]原位于大茂山，自北魏宣武帝时始迁至曲阳县城，离大茂山主峰有70公里的距离。<ref>[http://www.he.xinhuanet.com/zfwq/2009-08/14/content_17400513.htm 古北岳“申遗”再铸大茂山辉煌] {{webarchive|url=https://web.archive.org/web/20160304101846/http://www.he.xinhuanet.com/zfwq/2009-08/14/content_17400513.htm |date=2016-03-04 }}</ref>大茂山地区遗存有多处碑刻，展现了大茂山的历史文化。<ref>[http://news.sina.com.cn/c/cul/2007-07-25/100112268436s.shtml 曲阳县 50余通碑刻详尽记录古北岳历史文化]</ref>
