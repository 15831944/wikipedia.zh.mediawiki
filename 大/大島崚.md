== 人物簡介 ==
2008年參加[[SPACE_CRAFT|SPACE CRAFT]]事務所集團主辦的『型男聲優團體』結果獲選，同時跟大島選上的有{{link-ja|安倍龍太郎 (藝人)|安倍龍太郎 (タレント)|安倍龍太郎}}<ref>{{cite web |url=http://www.spacecraft.co.jp/ikemen.html|title=Audition|work=Space Craft Group|accessdate=2018-02-14|language=ja}}</ref><ref>{{cite web |url=http://spacecraft.co.jp/oshima_ryo/|title=スペースクラフトによる公式プロフィール |accessdate=2018-02-14 |archiveurl= https://web.archive.org/web/20120407095407/http://spacecraft.co.jp/oshima_ryo/|archivedate=2012-04-07|work=Space Craft Group }}</ref>。翌年2009年，大島以電視動畫《[[戀愛班長|戀愛班長]]》角色東條潮成為在聲優業界出道<ref name="pashplus" /><ref name="animatetimes" />。2011年的舞台劇版《戀愛班長》開始，與舞台演員的工作並肩齊行<ref name="pashplus" />。

2012年，大島從SPACE CRAFT離所<ref>{{cite web |url= https://ameblo.jp/ooshima-ryo0918/entry-11341321286.html|title=【初めまして。最初から本音でいきます】ACTING LIFE|work=大島崚的官方個人部落格|date=2012-08-30|accessdate=2018-02-14|language=ja}}</ref>。有一段時間是[[自由職業|自由身]]，2013年4月起加入現在所屬的{{link-ja|A-Light|A-Light}}<ref>{{cite web|url= https://ameblo.jp/ooshima-ryo0918/entry-11739799473.html|title=【今年もありがとうございました。】ACTING LIFE|work=大島崚的官方個人部落格|date=2013-12-30|accessdate=2018-02-14|language=ja}}</ref>。

興趣有[[料理|料理]]、唱[[卡拉OK|卡拉OK]]、造酒<ref name="a-light-pro" />。特長是彈[[電貝斯|貝斯]]、唱卡拉OK、讀書<ref name="a-light-pro" />。
