'''大肚車站'''位於[[台灣|台灣]][[臺中市|臺中市]][[大肚區|大肚區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[海岸線_(臺鐵)|海岸線]]的[[鐵路車站|鐵路車站]]。{{Fact|是[[成追線|成追線]]常誤點的路段，會誤點3-4分鐘。|time=2019-02-06}}

== 車站構造 ==
*[[島式月台|島式月台]]一座。
*大肚站有一連通前後站區的行人立體穿越設施(日光穿越道)，可與月臺連通。因涉及票務，目前暫時封閉。因此欲進出[[月台|月台]]仍需在站務人員的指揮下直接跨越鐵軌。
*大肚站北邊有一[[大肚調車場|大肚調車場]]，於1979年6月15日啟用，目的是鐵路電氣化工程之後用來支援[[彰化車站|彰化車站]]的[[貨物列車|貨物列車]]編組，後來因台鐵的貨運業務萎縮，1990年8月15日起貨物列車編組移回彰化車站，站場改用來暫放已報廢待拆解的各種鐵路車輛。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]，停靠車種以[[臺鐵區間車|區間車]]為主，亦有停靠少數[[莒光號列車|莒光號]]（南下(逆行)的507、561次及北上(順行)的522次)。
*[[區間快車|區間快車]]停靠2033次（週日停駛）及2085次（週日行駛）。
*[[莒光號列車|莒光號]]部份班次不停靠。
*[[自強號列車|自強號]]僅停靠109次(週日停駛)。
*對號列車停靠率：16.7%；所有車種停靠率：76.5%。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]、[[icash|icash 2.0]]及[[HappyCash|HappyCash]]付費。

== 歷史 ==
*1920年12月15日：因[[海岸線_(臺鐵)|海線]][[大甲溪|大甲溪]]南段通車新設'''大肚驛'''，即今本站。原[[臺中線|山線]]大肚驛廢除並新設立王田驛，即今[[成功車站|成功車站]]。<ref>[http://blog.xuite.net/ticket0610/notebook/55296850 臺中線成功車站沿革考證]，2011-12-15</ref>。
*2015年6月30日：啟用多卡通刷卡機。

== 其他 ==
*由大肚車站到[[臺中線|臺中線]]（山線）南端[[成功車站|成功車站]]之「[[吉祥語車票|大肚－成功]]」車票，在意義上有祝人[[懷孕|懷孕]]成功順利得子的吉祥感覺，因此成為頗受歡迎的紀念品。
