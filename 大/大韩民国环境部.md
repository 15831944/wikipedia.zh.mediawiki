'''环境部（환경부）'''，[[大韩民国|大韩民国]]国家[[行政机关|行政机关]]之一，部门首长称环境部部長，同时被任命为国务委员。

== 沿革 ==
*1967年2月17日 - [[大韩民国保健福祉部|保健社会部]]保健局环境卫生课设立。
*1980年1月5日 - 设立环境厅，为保健社会部的外厅。
*1990年1月3日 - 设立环境处。
*1994年12月24日 - 改组为环境部。
*2008年2月29日 - 政府机构改革，[[大韩民国气象厅|气象厅]]由[[大韩民国科学技术部|科学技术部]]移交环境部。

== 职责 ==
保护[[自然环境|自然环境]]及生活环境、防止[[环境污染|环境污染]]等相关事务。
