'''大熊座'''是一个[[星座|星座]]，在[[北半球|北半球]]的大部分常年可见。

== 神话 ==
这个[[星座|星座]]拥有全天最显著的星象，即[[北斗七星|北斗七星]]。许多古代文明都认为这个星座的星象是一只[[熊|熊]]（但[[古代中国|古代中国]]认为是一个[[斗|斗]]，[[埃及古代|埃及古代]]稱為公牛的左腳）。在[[希腊神话|希腊神话]]中，[[宙斯|宙斯]]为了使美丽的[[仙女|仙女]][[卡利斯托|卡利斯托]]逃脱天后[[赫拉|赫拉]]的忌妒，把她变成了一只熊。但赫拉仍然穷追不舍，命令狩猎女神[[阿耳忒弥斯|阿耳忒弥斯]]射杀这只熊，宙斯不得已把大熊提升为天上的星座。

== 概述 ==
这个星座所属的天区里有7个[[梅西耶天体|梅西耶天体]]，还有许多[[双星|双星]]和[[变星|变星]]。

== 著名的星 ==
* [[大熊座ζ|大熊座ζ]]星（古代中国称为[[开阳|开阳]]，北斗六）：位于大熊座中间的ζ星与它附近的[[辅星|辅星]]构成了一对裸[[眼|眼]]可见的双星。大熊座ζ星本身又是一个[[双星系统|双星系统]]，它是1650年发现的，是[[望远镜|望远镜]]时代最早发现的双星。它的两[[子星|子星]]的[[星等|星等]]分别是+2.27和+3.94，相距14.4[[弧秒|弧秒]]。

* [[大熊座ξ|大熊座ξ]]星：这是一个[[黄色|黄色]]的双星，互相环绕运行的[[轨道周期|轨道周期]]是60[[年|年]]。两子星星等是+4.3和+4.8，相距1.8弧秒。

== [[深空天体|深空天体]] ==
* [[M81|M81]]和[[雪茄星系|M82]]：M81是一个经典的[[漩涡星系|Sb型漩涡星系]]，在小型望远镜里，它是一个有明亮中心的大[[椭圆|椭圆]]光球。其星等是+6.8。M82与M81相距0.5[[度|度]]，是一个边缘朝向我们的[[不规则星系|不规则星系]]，用6英寸（15厘米）或更大的望远镜能够看到，有一个奇特的垂直尘埃带切过它的核。它的星等是+8.4。M81 距离[[地球|地球]]1200[[万|万]][[光年|光年]]。

* [[M97|M97]] 和 [[M108|M108]]：这个所谓的[[奇耦合|奇耦合]]由一个大而苍白的[[行星状星云|行星状星云]]（M97，星等：+11.2）和一个测向地球的漩涡星系(M108，星等：+10)构成. M97也叫[[枭状星云|枭状星云]]，因为它的盘上有两个暗区, 用12英寸(30厘米)望远镜可见。
