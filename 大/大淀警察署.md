'''大淀警察署'''（{{lang-ja|大淀警察署（おおよどけいさつしょ）}}）為日本[[大阪府警察|大阪府警察]]所管轄的[[警察局|警察署]]之一。

== 管轄区域 ==
大阪市北區[[大淀區|舊・大淀區域]]（除[[曾根崎警察署|曾根崎警察署]]・[[天滿警察署|天滿警察署]]管轄區域之外的北區域）

== 所在地 ==
* [[大阪市|大阪市]][[北區_(大阪市)|北區]][[中津_(大阪市)|中津]]一丁目5番25號 
最近的地鐵站為[[大阪市營地鐵|大阪市營地鐵]]·[[御堂筋線|御堂筋線]]·[[中津站_(大阪市營地下鐵)|中津站]]

== 沿革 ==
* [[1919年|1919年]]（[[大正|大正]]8年）：中津警察署開署。
* [[1943年|1943年]]（[[昭和|昭和]]18年）：改稱大淀警察署。

==交番==
大淀警察署下轄4個[[派出所|交番]]（[[派出所|派出所]]）。

* 大淀南交番（大阪市北區大淀南三丁目3番57號）
* 小波派交番（大阪市北區長柄東二丁目3番17號）
* 天神橋七丁目交番（大阪市北區天神橋七丁目1番19號）
* 本庄交番（大阪市北區本庄西二丁目11番19號）
