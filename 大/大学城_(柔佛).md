'''大学城'''（{{lang-ms|Taman Universiti}}，即“士姑来大学城”）是[[马来西亚|马来西亚]][[柔佛|柔佛]][[新山县|新山县]][[依斯干达公主城|依斯干达公主城]][[士姑来|士姑来]]的一个住宅区，由新山发展商康地威（Country View Sdn Bhd）建设，现有人口约55,100人，占地{{convert|1,082.76|acre|m2}}<ref>https://onemap.mbip.gov.my/portal/apps/webappviewer/index.html?id=71f5859e4d904e0eb4325682bb1d4fcc</ref>。广义上的大学城则包括周围的[[龙寰镇|龙寰镇]]、[[碧华园|碧华园]]、[[士姑来丽雅园|士姑来丽雅园]]、[[德沙士姑来花园|德沙士姑来花园]]，属于[[依斯干达城州议席|依斯干达城州议席]]。

大学城坐落于士姑来与埔来之间，于1984年开始动工，1986年开始售卖首批房屋<ref>{{Cite web|url=http://www.countryview.com.my/historical-background-3.aspx|title=Historical Background {{!}} Premier Property Developer In  Johor {{!}} Country View Berhad|accessdate=2018-06-22|work=www.countryview.com.my}}</ref>。[[马来西亚工艺大学|马来西亚工艺大学]]（UTM）建立后，这里便被命名为“大学城”，而工大距离大学城只有约5分钟的车程，居民主要由公务员及大学生组成。

== 区域和道路命名 ==
大学城被分为10个Zon（区）：Pertanian（农业）、Penyiaran（广播）、Perubatan（医药）、Perdagangan（贸易）、Kebangsaan（国民）、Kejayaan（成功）、Kebudayaan（文化）、Kemajuan（进步）、Kemuliaan（荣誉）以及Pendidikan（教育）。路名是由Jalan（路）+区域名组成的。如：Jalan Kebangsaan 23。P开头的小区在教育路的西边，而K开头的小区则在教育路的东边。教育路和文化路是大学城的主要道路。

大学城的核心是在国民区，也就是[[永旺|永旺]]大学城购物中心和大多数商店的所在地。

== 商店 ==
该处早期的商店都是两层楼的。2002年7月31日，大学城[[佳世客|佳世客]]（JUSCO，现称[[永旺|永旺]]商场，AEON Mall）正式开张。这是马来西亚的第九间永旺商场，同时也是柔佛州首间永旺商场。<ref>{{cite web|url=http://www.aeonretail.com.my/promotion/store-locations/store-and-shopping-centre/location_detailTU.php|title=AEON CO. (M) BHD.|website=AEON CO. (M) BHD.}}</ref>

== 教育 ==
大学城区内目前没有华文小学，最临近的华文小学位于{{tsl|ms|Mutiara Rini|丽宁镇}}新开放的道文小学和士姑来10英里的辅士小学。在道文小学尚未由[[居銮|居銮]]搬迁至丽宁镇之前，所有报读华文小学的大学城居民都会被安排至[[振林山|振林山]][[五间店|五间店]]的[[平民小学|平民小学]]就读。
