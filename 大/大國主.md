'''大己貴神'''（{{jpn|j = 大己貴神|hg = おおなむち<small>（或おおあなむち）</small> の かみ|rm = Oponamuti no kami|kyu = |hgt = おほなむち<small>（或おほあなむち）</small> の かみ}}，大穴牟遲神），一般稱作'''大國主神'''（{{jpn|j = 大国主神|hg = おおくにぬし の かみ|rm = Opokuninutsi no kami|kyu = 大國主神|hgt = おほくにぬし の かみ}}）、'''[[大物主神|大物主神]]'''（{{jpn|j = 大物主神|hg = おおものぬし の かみ|rm = Opomononutsi no kami|kyu = |hgt = おほものぬし の かみ}}），是[[日本|日本]]傳説中出現的人物，也是{{link-ja|三輪氏}}等祖。鎭座于'''[[出雲大社|杵築大社]]'''（[[島根縣|島根縣]][[出雲市|出雲市]]）。

== 事績 ==
在素戔嗚尊之後，和[[少彥名命|少彥名命]]（鎭座于[[筑陽神社|筑陽神社]]（築陽神社/[[島根縣|島根縣]][[八束郡|八束郡]][[東出雲町|東出雲町]]））協力經營天下，教導{{link-ja|禁厭}}、[[醫藥|醫藥]]等、完成[[葦原中國|葦原中國]]之造國。將國土譲予天孫[[天津彥彥火瓊瓊杵尊|天津彥彥火瓊瓊杵尊]]，隱退杵築之地，後成為[[出雲大社|出雲大社]]之祭神。

《日本書紀》、《古事記》中記載造國、讓國等神話、《古事記》中記載因幡之白兔、前往[[根之國|根之國]]、[[沼河比賣|沼河比賣]]等神話，也在《出雲國風土記》的許多故事中出現。

== 信仰 ==
「大國」也可讀「{{lang|ja|ダイコク}}」，所以和同音的[[大黑天|大黑天]]（大黑様）習合，在民間信仰廣為流傳。其子之事代主神和[[惠比壽|惠比壽]]習合，所以大黑様和惠比壽也被視為親子。

== 主要神社 ==
以[[出雲大社|出雲大社]]為代表，全國的出雲神社。[[北海道|北海道]]內以[[北海道神宮|北海道神宮]]為主的神社，和戰前[[台灣|台灣]]、[[樺太|樺太]]的神社，作為「[[開拓三神|開拓三神]]」和[[少彥名神|少彥名神]]共同祭祀。

* '''[[出雲大社|杵築大社]]'''（[[島根縣|島根縣]][[出雲市|出雲市]]） - 祭祀大己貴神。
* '''[[大神神社|大神大物主神社]]'''（[[奈良縣|奈良縣]][[櫻井市|櫻井市]]） - 祭祀[[大物主神|大物主神]]（大己貴神之{{link-ja|和魂}}、{{link-ja|幸魂}}、{{link-ja|奇魂}}）<ref group="注" name="三輪" />。[[神體|神體]][[三輪山|御諸山]]。
* '''[[大和神社|大倭坐大國魂神社]]'''（[[奈良縣|奈良縣]][[天理市|天理市]]） - 祭祀[[大國魂神|大國魂神]]（大己貴神之{{link-ja|荒魂}}）<ref group="注" name="大倭" />。神體八[[尺|尺]]瓊（八坂[[曲玉|曲玉]]/或云八尺[[鏡|鏡]]）。此神體[[永久_(年號)|永久]]六年（1118年）燒失，今以玉一顆爲神體。
* '''[[大歲神社|大歲神社]]'''（[[奈良縣|奈良縣]][[橿原市|橿原市]]） - 祭祀大歲神（與大己貴神同人）。
