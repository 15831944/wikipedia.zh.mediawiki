'''大壮'''（{{bd|1987年|10月31日|}}），本名'''王軒'''，中國大陆[[流行音乐|流行音乐]]男歌手，中國大陆-{zh-cn:视频;zh-tw:視訊}-平台[[陌陌|陌陌]]的[[主播|主播]]。<ref name="甲"/>

==演艺经历==
2016年，大壮进驻[[陌陌|陌陌直播]]平台，以表演唱歌为主。<ref>{{cite web |title=同样是网络主播 为啥陌陌大壮《我们不一样》能够爆红？ |url=http://ent.chinadaily.com.cn/2017-11/06/content_34191523.htm |website=中国日报 |accessdate=2018-10-21}}</ref>

2017年1月，参加《陌陌直播17惊喜夜》<ref>{{cite news |title=不要再次错过！ 陌陌直播推出17惊喜夜精彩回顾_fina |url=http://news.sina.com.cn/o/2017-01-21/doc-ifxzunxf1632396.shtml |work=新浪新闻|accessdate=2018-10-28 |date=2017-01-20}}</ref>；6月，因演唱歌曲《[[我們不一樣|我們不一樣]]》而走红<ref name="甲">{{cite web |title=陌陌主播大壮和《我们不一样》的故事 |url=http://yue.ifeng.com/a/20171107/39842690_0.shtml |website=[[凤凰网|凤凰音乐]] |accessdate=2018-09-15|date=2017-11-07}}</ref>；11月，发行歌曲《差一步》。<ref name="fenghuang">{{cite web |title=从直播间走向大世界，大壮、张鑫磊、点点等陌陌主播踏上跨界之路 |url=http://ent.ifeng.com/a/20180803/43083117_0.shtml |website=[[凤凰网|凤凰网娱乐]] |accessdate=2018-09-15|date=2018-08-03}}</ref><ref name="dongfang">{{cite web |author1=直播圈扒姐 |title=陌陌直播捧红大壮王轩，下一个走上“中国好声音”的主播会是谁？ |url=http://fashion.eastday.com/a/180828225756099.html |website=[[东方网|东方网]] |accessdate=2018-09-15|date=2018-08-28}}</ref><ref name="guangming">{{cite web |title=陌陌主播大壮登台《中国好声音》 李健转身观众疯狂欢呼 |url=http://e.gmw.cn/2018-08/19/content_30632339.htm |website=[[光明网_(中国)|光明网]] |accessdate=2018-09-15|date=2018-08-19}}</ref>

2018年初，参加《安徽卫视春节联欢晚会》和《山东卫视春节联欢晚会》；7月，参加浙江卫视《[[2018中国好声音|2018]][[中国好声音|中国好声音]]》节目，并成为李健战队六强学员。<ref name="fenghuang"/><ref name="dongfang"/><ref name="guangming"/>9月底，参加湖南卫视《幻乐之城》节目录制，与[[郁可唯|郁可唯]]合作演出。<ref>{{cite web |title=周迅在MOMO《幻乐之城》展现高情商 大壮隔空演绎母子情 |url=http://ent.rednet.cn/c/2018/10/08/4734305.htm |website=红网 |accessdate=2018-10-19}}</ref>年底，参加《浙江卫视领跑2019演唱会》。

2019年2月，他参演的首部影视剧作品《失控》在优酷视频播出。

== 争议 ==

2016年，大壮因欠好友李某巨款而遭到对方起诉，其出行、住宿也受到限制。2017年4月，成为网络红人的大壮主动申请还钱，并在规定时间内还清所有欠款，案件得以了结。2019年3月26日，大壮的这一事件被CCTV-12《庭审现场》报道长达三十分钟。<ref>{{cite news |title=《我们不一样》原唱大壮被央视点名曾是“老赖” |url=http://www.bjnews.com.cn/news/2019/03/28/561434.html |accessdate=2019-03-28 |work=新京报 |date=2019-03-28}}</ref>
