'''大林車站'''位於[[台灣|台灣]][[嘉義縣|嘉義縣]][[大林鎮_(台灣)|大林鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]及[[大林糖廠|大林糖廠]][[小梅線|小梅線]]（廢止）的[[鐵路車站|鐵路車站]]，是[[臺灣鐵路管理局|臺鐵]]第一座[[綠建築|綠建築]]車站。

== 車站構造 ==
*[[島式月台|島式月台]]一座、[[岸式月台|岸式月台]]兩座。
*配合[[臺鐵捷運化|臺鐵捷運化]]下「臺鐵都會區捷運化暨區域鐵路先期建設計畫」案改建為[[跨站式站房_(台鐵)|跨站式站房]]。新站房為[[臺灣鐵路管理局|臺鐵]]首座以[[綠建築|綠建築]]概念所設計的車站。
*本站設有[[無障礙|無障礙]]電梯5座。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]。
**[[自強號列車|自強號]]：僅停靠順行170次。
**[[莒光號列車|莒光號]]：除順行506、562、516、754次；逆行507、751次與環島[[觀光列車|觀光列車]]外均有停靠。
**[[臺鐵區間快車|區間快車]]：皆停靠，3029次為起點站。
**[[臺鐵區間車|區間車]]：皆停靠。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。
*站內設施
[[File:Dalin_Station,_ticket_hall_(Taiwan).jpg|thumb]]
**列車資訊自動播音服務系統。
**電腦售票窗口：設有兩個。
**哺乳室
**公共電話機：設有三部，一般電話兩部，殘障使用電話一部。
**[[自動售票機|自動售票機]]：設有兩台。
**商店服務：已停止營業，改為二代選物販賣機（夾娃娃機）
**自動閘門：尚未設置。
**列車資訊LED顯示系統：剪票口兩面（目前因系統問題暫時未啟用）、各側月台一面
**列車資訊液晶顯示器：於剪票口一台

== 歷史 ==
*1903年12月15日：設'''大莆林驛'''。
*1920年：改稱大林。
*2007年4月2日：配合臺鐵捷運化增建月台及跨站式站房動工。
*2008年12月5日：新月台及跨站式站房啟用。
*2013年9月30日：啟用多卡通刷卡機。

== 臺糖大林車站 ==
*1913年12月29日：設立'''大莆林驛'''，屬於新高製糖株式會社。
*由於路線可達北港，便於[[北港朝天宮|北港朝天宮]]進香，當時亦以「進香鐵道」之名招攬旅客。
*1920年：改稱'''新高大林'''。
*1934年：改稱'''大林'''。
*目前已無遺跡，僅殘留車站北邊路段的三軌線。
