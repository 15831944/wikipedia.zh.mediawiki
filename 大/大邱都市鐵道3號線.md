'''大邱都市鐵道3號線'''（{{kor|諺=대구 도시철도 3호선|漢=大邱 都市鐵道 3號線|羅=Daegu Doshicheoldo 3 hoseon|f=y}}）是[[韓國|韓國]][[大邱廣域市|大邱廣域市]]的第三條[[地鐵|地鐵]]，為[[單軌鐵路|單軌鐵路]]系統，路線從[[北區_(大邱)|北區]][[漆谷慶大醫院站|漆谷慶大醫院站]]至[[壽城區|壽城區]][[龍池站_(大邱)|龍池站]]，全長23.95公里，屬於[[大邱都市鐵道|大邱都市鐵道]]的一部分，由[[大邱都市鐵道公社|大邱都市鐵道公社]]經營。2009年7月24日動工，於2015年4月23日開通運營。

== 路線資料 ==
* 總長：23.95km
* 軌間距離：850mm
* 站數：30
* 單線區間：全線
* 電化區間：全線（直流1500V接觸網觸電）
* 閉塞方式：ATO、ATP、TTC
* 最高時速：70km/h

== 歷史 ==
* 2006年10月26日 - 完成最終總體規劃。
* 2009年7月24日 - 開始動工。
* 2013年10月2日 - 決定路線名稱。
* 2014年6月19日 - 確定延期至2015年4月正式開始營運。
* 2015年2月9日 - 開始試營運。
* 2015年4月23日 - 正式開始營運。
