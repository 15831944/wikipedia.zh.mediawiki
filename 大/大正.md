'''大正'''（{{jpn|j=大正|hg=たいしょう|rm=Taishō|f=y}}）是[[日本|日本]][[大正天皇|大正天皇]]在位期間使用的[[日本年號|年號]]，使用時間從1912年7月30日至1926年12月25日止。承續年號為[[明治|明治]]，接續年號為[[昭和|昭和]]。

== 改元 ==
*[[1912年|1912年]]（明治45年）7月30日[[明治天皇|明治天皇]]去世，[[大正天皇|大正天皇]]即位，改年號為[[大正|大正]]。
*[[1926年|1926年]]（大正15年）12月25日（[[聖誕節|聖誕節]]）[[大正天皇|大正天皇]]去世，[[昭和天皇|昭和天皇]]即位，改年號為[[昭和|昭和]]。

== 出處 ==
「大正」一詞出自《[[易經|易經]]》第十九卦中的「臨，剛浸而長。說而順，剛中而應，'''大'''亨以'''正'''，天之道也。」大正這一年號過去曾四次被選為候補，於[[大正天皇|大正天皇]]即位改元時被採用。按《大正天皇實錄》，同時尚有「天興」、「興化」為候補年號，而[[天皇|天皇]]採「大正」為用。

== 大事年表 ==
*大正2年（1913年）：[[大正政變|大正政變]]。
*大正3年（1914年）：[[櫻島|櫻島]]火山爆發；[[西門子事件|西門子事件]]；[[第一次世界大戰|第一次世界大戰]]爆發。
*大正5年（1916年）：[[夏目漱石|夏目漱石]]逝世。
*大正7年（1918年）：[[西伯利亞出兵|西伯利亞出兵]]；[[1918年米骚动|米骚动]]；第一次世界大戰結束。
*大正8年（1919年）：[[巴黎和会_(1919年)|巴黎和会]]；《選舉法》改正。
*大正9年（1920年）：[[國際聯盟|國際聯盟]]設立；[[尼港事件|尼港事件]]；第一回[[國勢調查_(日本)|國勢調查]]。
*大正10年（1921年）：首相[[原敬|原敬]]遭到暗殺；[[昭和天皇|裕仁親王]]就任[[攝政|攝政]]。
*大正11年（1922年）：[[南洋廳|南洋廳]]設置；[[华盛顿会议|华盛顿会议]]举行；[[山縣有朋|山縣有朋]]、[[森鷗外|森鷗外]]逝世。
*大正12年（1923年）：[[關東大地震|關東大地震]]；《國民精神作興詔書》發布；[[虎之門事件|虎之門事件]]。
*大正13年（1924年）：[[皇太子|皇太子]][[裕仁|裕仁親王]]（日後的昭和天皇）與[[香淳皇后|良子]]結婚；[[甲子園球場|甲子園球場]]落成啟用。
*大正14年（1925年）：[[宇垣裁军|宇垣裁军]]；[[治安維持法|治安維持法]]；[[普通選舉法|普通選舉法]]制定；[[電台廣播|電台廣播]]開始。
*大正15年（1926年）：[[大正天皇|大正天皇]]駕崩；改元[[昭和|昭和]]。
