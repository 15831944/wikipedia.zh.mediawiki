'''大理白族自治州'''（[[白语|白语]]：Darl•lit Baif•cuf zirl•zirl•zox），简称'''大理州'''，是[[中华人民共和国|中华人民共和国]][[云南省|云南省]]下辖的[[自治州|自治州]]，位于云南省中部偏西。州境东邻[[楚雄州|楚雄州]]，南接[[普洱市|普洱市]]、[[临沧市|临沧市]]，西临[[保山市|保山市]]、[[怒江州|怒江州]]，北界[[丽江市|丽江市]]。地处[[横断山脉|横断山脉]]与[[云贵高原|云贵高原]]的过渡带，[[云岭|云岭]]南麓，地势西北高，东南低，平均海拔2090米。西部有[[澜沧江|澜沧江]]，东北有[[金沙江|金沙江]]，[[洱海|洱海]]位于州境中部。总面积2.83万平方公里，人口354.4万，[[汉族|汉族]]人口比例约51%，[[白族|白族]]人口比例约32%，自治州首府驻[[大理市|大理市]]。大理是首批[[国家历史文化名城|国家历史文化名城]]，拥有深厚的历史文化和丰富的旅游资源，[[苍山_(大理)|苍山]]、[[洱海|洱海]]、[[大理古城|大理古城]]等都是著名风景区。

== 历史 ==
[[西汉|西汉]][[元封|元封]]二年（前109年），[[汉武帝|汉武帝]]开[[西南夷|西南夷]]，出兵攻打[[滇国|滇国]]，滇王降。汉王朝在大理地区设立[[叶榆县|叶榆]]、[[云南县|云南]]、[[邪龙县|邪龙]]、[[比苏县|比苏]]等县，属[[益州郡|益州郡]]管辖。[[东汉|东汉]][[永平|永平]]十二年（69年）[[哀牢国|哀牢国]]内附，遂分益州郡西部置[[永昌郡|永昌郡]]，初治[[巂唐县|巂唐县]]（今[[云龙县|云龙县]]境），后迁[[不韦县|不韦县]]（今[[保山市|保山市]]）。[[三国|三国]][[蜀汉|蜀汉]][[建兴|建兴]]三年（225年），丞相[[诸葛亮|诸葛亮]]平定南中诸郡，分[[建宁郡|建宁郡]]、[[越巂郡|越巂郡]]、[[永昌郡|永昌郡]]三郡置[[云南郡|云南郡]]，治[[云南县|云南县]]（今[[祥云县|祥云县]][[云南驿镇|云南驿镇]]），今大理州东部属云南郡，西部属永昌郡，隶[[庲降都督|庲降都督]]。[[西晋|西晋]][[泰始|泰始]]七年（271年）改庲降都督地为[[宁州|宁州]]。[[永嘉|永嘉]]五年（313年）析永昌、云南二郡置[[河阳郡|河阳郡]]，治[[河阳县|河阳县]]（今[[大理市|大理市]][[凤仪镇|凤仪镇]]）。[[咸康|咸康]]八年（342年）析置[[西河阳郡|西河阳郡]]，河阳郡改为[[东河阳郡|东河阳郡]]。[[南朝齐|南朝齐]]今州境分属东河阳、西河阳、西河、云南四郡。[[隋代|隋代]]属[[南宁州|南宁州]]总管府。

[[唐朝|唐朝]]初年，[[洱海|洱海]]地区有六个较大的部落居住，称为“[[六诏|六诏]]”，其中以[[蒙舍诏|蒙舍诏]]的势力最大，因居于最南故又称[[南诏|南诏]]。[[开元|开元]]二十六年（738年），蒙舍诏首领[[皮罗阁|皮罗阁]]在唐朝支持下，合六诏为一，统一了洱海地区，建立了南诏国，都城建在[[太和城|太和城]]（今大理市太和村）。唐[[大历|大历]]十四年（779年）南诏国迁都[[羊苴咩城|羊苴咩城]]（今[[大理古城|大理古城]]）。南诏国时期，洱海地区周边设立“{{僻字|𧸘|左“贝”右“佥”}}”或“睑”，北部属[[剑川节度|剑川节度]][[谋统郡|谋统郡]]（今[[鹤庆县|鹤庆县]]），西部属[[永昌节度|永昌节度]][[胜乡郡|胜乡郡]]（今[[永平县|永平县]]）。[[唐昭宗|唐昭宗]][[天复|天复]]二年（902年），南诏权臣[[郑买嗣|郑买嗣]]发动宫廷政变，建立了[[大长和国|大长和国]]，南诏亡。[[后唐|后唐]][[天成|天成]]三年（928年），[[赵善政|赵善政]]篡位，建立了[[大天兴国|大天兴国]]，次年[[杨干贞|杨干贞]]废赵善政自立，建[[大义宁国|大义宁国]]。[[后晋|后晋]]天福二年（937年），通海节度[[段思平|段思平]]联合滇东三十七部，进军大理，推翻了大义宁国，建立了'''[[大理国|大理国]]'''。大理国后期废除了节度和都督的军事辖区，改设府、郡。除大理首府外，今州境尚有[[谋统府|谋统府]]（今鹤庆县）、义督赕（今剑川县，属谋统府）、[[胜乡郡|胜乡郡]]（今永平县，隶[[永昌府|永昌府]]）、[[南涧县|南涧县]]（今[[南涧彝族自治县|南涧彝族自治县]]，属[[威楚府|威楚府]]）。[[南宋|南宋]][[宝祐|宝祐]]元年（1253年），[[元世祖|元世祖]][[忽必烈|忽必烈]]率大军灭大理国，建立[[云南行省|云南行省]]。南诏、大理国历唐、宋两朝，达五百余年，使云南形成了一个稳定的政治统一体。

[[元朝|元朝]]，云南政治中心东移[[昆明|昆明]]，大理地区设立大理上、大理下两个[[万户府|万户府]]，又设[[云龙甸军民府|云龙甸军民府]]（治今[[云龙县|云龙县]]）。[[元宪宗|元宪宗]]三年（1253年）以谋统府改置[[鹤州|鹤州]]（治今[[鹤庆县|鹤庆县]]）。元宪宗四年（1254年）置[[义督千户|义督千户]]（治今[[剑川县|剑川县]]剑川湖西岸）。宪宗七年（1257年）设[[品甸千户所|品甸千户所]]（治今[[祥云县|祥云县]]）。[[至元|至元]]七年（1270年）改万户府置'''[[大理路|大理路]]'''（治今大理市）。至元十一年（1274年）置[[蒙化府|蒙化府]]（治今[[巍山县|巍山县]]）；改胜乡郡置[[永平县|永平县]]，属大理路；改品甸千户所为[[云南州|云南州]]，属大理路；改义督千户为[[剑川县|剑川县]]，属鹤州；于白崖赕置[[建宁县|建宁县]]（治今[[弥渡县|弥渡县]][[红岩镇_(弥渡县)|红岩镇]]），属[[赵州|赵州]]；以德原城置邓川州（治今[[洱源县|洱源县]][[邓川镇|邓川镇]]德源城）；改浪穹千户为[[浪穹县|浪穹县]]（治今洱源县），属[[邓川州|邓川州]]。至元十四年（1277年）改蒙化府为[[蒙化路|蒙化路]]。至元二十年（1283年）降蒙化路为[[蒙化州|蒙化州]]，属大理路。至元二十三年（1286年）升鹤州为[[鹤庆路|鹤庆路]]。至元二十四年（1287年）改南涧县置[[定边县|定边县]]，属[[威楚路|威楚路]][[镇南州|镇南州]]，后废。至元二十五年（1288年）宁罢建宁县，并入大理路，至元二十六年（1289年）置[[太和县|太和县]]（治今大理市）。[[泰定|泰定]]前改云龙甸军民府为[[云龙州|云龙州]]（治今[[云龙县|云龙县]][[旧州镇_(云龙县)|旧州镇]]），属大理府。

[[明朝|明]][[洪武|洪武]]十五年（1382年）改大理路为'''[[大理府|大理府]]'''；改鹤庆路为[[鹤庆府|鹤庆府]]；降云南州为[[云南县|云南县]]，隶大理府；置[[凤羽县|凤羽县]]（治今洱源县[[凤羽镇|凤羽镇]]），属邓川州，后省入浪穹县。洪武十六年（1383年）设[[迷渡市巡检司|迷渡市巡检司]]，隶赵州；复置定边县，属[[楚雄府|楚雄府]]。洪武十七年（1384年）云南县改属赵州；升剑川县为[[剑川州|剑川州]]（治今剑川县[[金华镇|金华镇]]），属[[鹤庆府|鹤庆府]]。[[正统|正统]]十三年（1448年）复置蒙化府。[[弘治|弘治]]六年（1493年）割太和县海东九里，云南县二里，赵州一里置[[宾川州|宾川州]]（驻今[[宾川县|宾川县]][[州城镇_(宾川县)|州城镇]]），属大理府。[[万历|万历]]二十八年（1600年）移邓川州治至今洱源县邓川镇旧州。[[崇祯|崇祯]]二年（1629年）移云龙州治至雒马井（今云龙县[[宝丰乡|宝丰乡]]）。崇祯十三年（1640年）移邓川州治至今邓川镇新州。

[[清朝|清]][[雍正|雍正]]七年（1729年）撤定边县，并入蒙化府。雍正八年（1730年）移大理督捕通判驻弥渡城。[[乾隆|乾隆]]三十五年（1770年）改蒙化府为[[蒙化直隶厅|蒙化直隶厅]]，有巡检司，驻南涧，即故定边城；剑川州改属[[丽江府|丽江府]]；降鹤庆府为[[鹤庆州|鹤庆州]]，属丽江府。

[[民国|民国]]元年（1912年）析蒙化厅漾濞江巡检司和云龙州、永平县、浪穹县插花地置[[漾濞县|漾濞县]]（治今[[苍山西镇|苍山西镇]]下街）。民国二年（1913年）以赵州、祥云、宾川、蒙化四县插花地析置[[弥渡县|弥渡县]]（治今[[弥城镇|弥城镇]]）；浪穹县更名[[洱源县|洱源县]]，取“洱海之源”之意；同年废府、州、厅存县，计有[[大理县|大理]]、云南、洱源、[[赵县|赵县]]、[[邓川县|邓川]]、宾川、云龙、弥渡、[[鹤庆县|鹤庆]]、剑川、永平、[[蒙化县|蒙化]]、漾濞等县，各县属[[滇西道|滇西道]]（次年改[[腾越道|腾越道]]）。1914年改赵县为[[凤仪县|凤仪县]]。1915年漾濞县治迁今址。民国十六年（1927年）废道制，各县直属云南省；云南县因与云南省重名，改称[[祥云县|祥云县]]，取旧名“祥州”、“云南”各一字得名。民国三十一年（1942年）蒙化县改属云南省第五行政督察区，鹤庆、剑川县改属第七行政督察区。民国三十五年（1946年）鹤庆、洱源、宾川、剑川属第十区，大理、永平、凤仪、邓川、云龙、弥渡、蒙化、漾濞属第十一区

[[中华人民共和国|中华人民共和国]]成立后，1950年置[[大理专区|大理专区]]，专员公署驻下关（今大理市[[下关镇|下关镇]]），辖大理、凤仪、漾濞、永平、蒙化、弥渡、云龙、祥云、宾川、[[顺宁县|顺宁]]、[[缅宁县|缅宁]]、[[云县|云县]]、洱源、邓川14县。1951年析大理县玉洱乡、凤仪县珠海乡设置[[下关市|下关市]]。1952年缅宁县划归[[缅宁专区|缅宁专区]]。1954年蒙化县更名[[巍山县|巍山县]]，顺宁县更名[[凤庆县|凤庆县]]。1956年11月16日批准设立[[大理白族自治州|大理白族自治州]]，11月22日正式成立；原[[丽江专区|丽江专区]]的鹤庆、剑川二县划入大理州；凤庆、云县划归[[临沧专区|临沧专区]]；将巍山县分为[[巍山彝族自治县|巍山彝族自治县]]和[[永建回族自治县|永建回族自治县]]（驻今巍山县[[永建镇|永建镇]]）。1958年宾川县迁驻[[金牛镇|金牛镇]]。1960年撤销下关市及大理、凤仪、漾濞三县，合并为[[大理市|大理市]]；洱源、邓川二县并入剑川县；宾川、弥渡二县并入祥云县；巍山、永建二县合并为[[巍山彝族回族自治县|巍山彝族回族自治县]]。1962年撤销大理市，恢复下关市、大理县、漾濞县；恢复洱源、弥渡、宾川三县。1963年析巍山县设立[[南涧彝族自治县|南涧彝族自治县]]。1983年撤销大理县和下关市，合并设立[[大理市|大理市]]。1985年6月，漾濞县改置[[漾濞彝族自治县|漾濞彝族自治县]]。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全州[[常住人口|常住人口]]为345.6万人<ref>{{cite book |author=大理白族自治州统计局|title=《大理白族自治州2010年第六次全国人口普查主要数据公报》}}</ref>。同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加了15.9万人，增长4.82%。平均每年增加近1.6万人，年平均增长率为0.47%。其中，男性为175.2万人，占总人口的50.69%；女性为170.4万人，占总人口的49.31%。总人口性别比（以女性为100）为102.82。0－14岁的人口为67.2万人，占总人口的19.44%；15－59岁的人口为237.1万人，占总人口的68.61%；60岁及以上人口为41.3万人，占总人口的11.95%，其中65岁及以上的人口为28.5万人，占总人口的8.25%。居住在城镇的人口112.1万人，占总人口的32.44%；居住在乡村的人口为233.5万人，占总人口的67.56%。

=== 民族 ===
常住人口中，[[汉族|汉族]]人口为175.2万人，占总人口的50.69%；各[[少数民族|少数民族]]人口为170.4万人，占总人口的49.31%。大理本地的少数民族居民日常使用[[白语|白语]]，全州通用[[汉语|汉语]]。

== 经济 ==
2017年全州[[地区生产总值|地区生产总值]]1066.5亿元，比上年增长9.7%。人均生产总值29846元。城镇居民人均可支配收入31779元，农村居民人均可支配收入10525元<ref>{{cite web |author1=大理白族自治州统计局 |title=大理白族自治州2017年国民经济和社会发展统计公报 |url=http://www.dalidaily.com/zhuanti/120180201/1105802.html |website=大理日报}}</ref>。

2017年大理州共接待国内游客4120万人次，同比增长9.4%；海外游客102万人次，同比增长9.2%；实现旅游业总收入647亿元，同比增长21%<ref>{{cite web |title=大理州去年旅游收入增两成 |url=http://www.dali.gov.cn/dlzwz/5116382746297499648/20180117/320288.html |website=大理白族自治州人民政府}}</ref>。
