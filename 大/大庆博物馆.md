'''大庆博物馆'''，位于[[黑龙江省|黑龙江省]][[大庆市|大庆市]][[大庆高新技术产业开发区|大庆高新技术产业开发区]]文苑街2号，是大庆市属综合性博物馆，国家一级博物馆。隶属大庆市文化广电新闻出版局<ref>{{Cite web|url = http://www.dqdaily.com/ztxw/2012/2012-05/08/content_1178891.htm|title = 大庆博物馆焕然一新迎接公众|author = |date = 2012-05-08|publisher = 大庆网|language = |accessdate = |quote = }}{{Dead link|date=2019年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

==简介==
大庆博物馆的前身是1964年组建的大庆展览馆。馆址多次变迁。1975年，位于大庆市儿童公园北侧的大庆展览馆落成。1991年，该馆陈列的铁人[[王进喜|王进喜]]事迹展览整体迁到铁人纪念馆。1998年，在事业单位机构改革中，大庆展览馆改名为大庆博物馆。2001年，大庆市文物管理站和大庆博物馆[[合署办公|合署办公]]。2005年7月，大庆博物馆新馆开工建设。2008年，大庆博物馆迁入位于大庆市开发区火炬新街教育文化中心的新馆，建筑面积18700平方米，和旁边的[[大庆歌剧院|大庆歌剧院]]、[[大庆书苑|大庆书苑]]组成大庆五湖新区独具特色的建筑群。2008年6月18日，大庆博物馆新馆自然部分对社会免费开放<ref>大庆博物馆，北方文物2009年第4期</ref><ref name=dqb>{{Cite web |url =  http://www.dqsbwg.com/about.asp?fileid=1|title = 关于庆博 |author =  |date =  |publisher =  大庆博物馆|language =  |accessdate =2017-05-30  |quote =  }}</ref>。2009年7月，被[[国家文物局|国家文物局]]公布为国家二级博物馆。2009年12月，获评为全国科普教育基地。2011年11月22日，二期布展工程结束，对社会全面开放<ref name=dqb/>。2017年，大庆博物馆被[[中国博物馆协会|中国博物馆协会]]评为第三批国家一级博物馆<ref>{{Cite web |url = http://www.chinamuseum.org.cn/a/xiehuigonggao/20170120/9351.html|title = 关于发布第三批国家一级博物馆名单的通知 |author =  |date =2017-01-20  |publisher = 中国博物馆协会 |language =  |accessdate =  |quote =  }}</ref>。
    
大庆博物馆是中国第一家以[[中国东北|中国东北]][[第四纪|第四纪]]古环境、古动物、古人类为主题的综合性博物馆。馆藏标本、化石、文物超过20万件，是中国乃至世界专业性收藏[[猛犸象|猛犸象]]—[[披毛犀|披毛犀]]动物群化石数量最多、种属最全的博物馆之一。该馆设有《东北第四纪自然环境》、《东北第四纪哺乳动物》、《大庆地区古代人类文明》三个陈列<ref name=dqb/>。在馆藏的文物中，有一级文物48件、二级文物386件、三级文物1167件。<ref>[http://kydww.wwpc.net.cn/cms/wwj/area.do?cmd=areaDetail&c_unit_code=23060221800001 全国可移动文物登录网：大庆博物馆] {{Wayback|url=http://kydww.wwpc.net.cn/cms/wwj/area.do?cmd=areaDetail&c_unit_code=23060221800001 |date=20170822053641 }}</ref>
