'''大村車站'''位於[[台灣|台灣]][[彰化縣|彰化縣]][[大村鄉_(台灣)|大村鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
大村站是一個設有[[岸式月台|岸式月台]]兩座、兩條路線的車站。身為臺鐵最早啟用的[[跨站式站房_(台鐵)|跨站式車站]]之一，大村的站房本身就設在月台間的[[人行天橋|跨線天橋]]上，其中購票及服務設施設在三樓大廳，與通往前後站的樓梯均有連通。車站設置初期原本只有連往前站（福進路方向）的出口，後站（可透過新闢的道路連往中正西路之出入口）則是在2011年10月28日正式開通。

在大村車站前站方向道路旁的角地設有一個小型的民眾公園，在公園中有一座[[風車|風車]]造型的建築物，是當地較為顯眼的地標。前站面向臺鐵縱貫線以東，乘客可於台一線往福進路進入大村車站。後站則面向大村市區，可由新闢道路至過溝三巷前往市區。

==  公車轉乘 ==
由於大村車站並沒有緊鄰主要幹線道路，因此距離本站最近的公車站牌位於步行約300公尺外的中山路（[[台1線|台1線]]）上之「蓮花池」站，包括[[國光客運|國光客運]]、[[彰化客運|彰化客運]]與[[員林客運|員林客運]]皆有班車停靠。

== 歷史 ==
*1934年11月1日：「大村臨時停車場」開業，僅停汽油車。
*1942年左右：因[[第二次世界大戰|第二次世界大戰]]時期缺乏汽油，汽油車停駛，本站不再有車停靠，戰後亦未復站。
*1976年4月21日，一列南下的[[台鐵觀光號|觀光號]]列車在通過本站舊址南方的過溝平交道時，與闖越平交道之[[彰化客運|彰化客運]][[大村國中|大村國中]]學生專車衝撞，造成共41人死亡、40人重傷的嚴重事件。事後大村鄉建立[[淑女祠|淑女祠]]紀念其中16名女學生{{R|NTW_140421}}。
*2006年3月31日：配合[[臺鐵捷運化|臺鐵捷運化]]政策重新設站，並指定由[[員林車站|員林站]]管理。
*2006年4月4日：新站正式啟用，位置在[[台灣日治時期|日治時期]]站址北方100公尺。
*2011年10月28日：後站正式啟用。
*2015年6月30日：啟用多卡通刷卡機。
