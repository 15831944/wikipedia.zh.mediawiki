'''大鸨'''（[[拼音|拼音]]：{{lang|la|dà bǎo}}；[[注音|注音]]：ㄉㄚˋ　ㄅㄠˇ），学名{{lang|la|'''''Otis tarda'''''}}，属于[[鹤形目|鹤形目]][[鸨科|鸨科]]。

== 生态环境 ==
栖息于草原和半荒漠环境。在农耕地越冬。

== 分布地域 ==
[[欧洲|欧洲]]；[[西北亚|西北亚]]至[[中东|中东]]，[[中亚|中亚]]及[[中国|中国]]北方地区。

== 特征 ==
体形硕大的鸨，上体具有宽大的棕色和黑色横纹，下体及尾下覆羽白色，繁殖期雄鸟颈前具有状如胡须的白色丝状羽。

== 食物 ==

== 繁殖与保护 ==
全球易危

== 文化 ==
在中国因为古人对该物种认识的不足，无法准确区分雌雄两性，而主观认为该物种“与万鸟配” ，「[[老鴇|老鴇]]」因而成为了后来[[妓院|妓院]]老板的別稱。
