'''大廟'''，意思是設施雄偉或富於香火的宗教建築，亦可指「陽廟」(與「陰廟」相對者)，其可以指：

== 臺灣 ==
* [[桃園景福宮|桃園景福宮]] ：在[[臺灣|臺灣]][[桃園市|桃園市]][[桃園區|桃園區]]，居民習稱為「大廟」，為當地重要的信仰中心。
* [[保西代天府|保西代天府]] ：位於臺灣[[臺南市|臺南市]][[歸仁區|歸仁區]]，當地信仰中心，[[南瀛三大代天府|南瀛三大代天府]]中最早創立者。
*[[番社同安廟|番社同安廟]]：位於台灣[[宜蘭縣|宜蘭縣]][[壯圍鄉|壯圍鄉]]，為當地知名的信仰中心，座落於濱海公路(蛇)坐西朝龜山島(龜)，是[[壯圍鄉|壯圍鄉]]規模最大的[[玄天上帝|玄天上帝]]廟。
* [[佳里金唐殿|佳里金唐殿]]：位於臺灣臺南市[[佳里區|佳里區]]，逢[[子_(地支)|子]]、[[卯|卯]]、[[午|午]]、[[酉|酉]]年陰曆正月主辦[[蕭壠香|蕭壠香]]，當地信仰中心，南瀛三大代天府之一。
* [[南鯤鯓代天府|南鯤鯓代天府]]：位於臺灣臺南市[[北門區|北門區]]，是台灣相當知名的一座廟宇，素有「台灣王爺總廟」之稱，南瀛三大代天府之一。
* [[西港慶安宮|西港慶安宮]]：位於臺灣臺南市[[西港區|西港區]]，逢-{[[丑|丑]]}-、[[辰|辰]]、[[未|未]]、[[戌|戌]]年[[農曆|農曆]]四月主辦[[西港香|西港香]]，當地信仰中心。
* [[學甲慈濟宮|學甲慈濟宮]]：在臺灣臺南市[[學甲區|學甲區]]，每年都會舉辦[[上白礁謁祖祭典|上白礁謁祖祭典]]。
* [[漚汪文衡殿|漚汪文衡殿]]：在臺灣臺南市[[將軍區|將軍區]]，是當地人信仰中心，廟宇宏偉壯觀，香火鼎盛。
* [[五甲龍成宮|五甲龍成宮]]，在臺灣[[高雄市|高雄市]][[鳳山區|鳳山區]]，是[[五甲|五甲]]地區最雄偉的廟宇之一，俗稱「大廟」、「五甲大廟」。
* [[內惟鎮安宮|內惟鎮安宮]]：在臺灣高雄市[[鼓山區|鼓山區]][[內惟|內惟]]地區，俗稱「內惟大廟」，主祀[[池府千歲|池府千歲]]。
* [[高雄關帝廟|高雄關帝廟]]：位於臺灣高雄市[[苓雅區|苓雅區]]，是高雄知名廟宇之一，終年香火鼎盛。
* [[文武聖殿|文武聖殿]]：在臺灣高雄市[[鹽埕區|鹽埕區]]，是當地的信仰中心，終日香火不斷。
* [[左營元帝廟|左營元帝廟]]：在臺灣高雄市[[左營區|左營區]]，俗稱左營大廟，當地信仰中心，主祀[[北極玄天上帝|北極玄天上帝]]，是[[左營_(左營區)|左營]]地區的知名廟宇。
* [[高雄三鳳宮|高雄三鳳宮]]：在臺灣高雄市[[三民區|三民區]]，是高雄的著名廟宇，香火旺盛。
* [[覆鼎金保安宮|覆鼎金保安宮]]：在臺灣高雄市三民區，位於[[金獅湖|金獅湖]]畔。
* [[東港東隆宮|東港東隆宮]]：在臺灣[[屏東縣|屏東縣]][[東港鎮_(臺灣)|東港鎮]]，每三年都會舉辦[[東港迎王平安祭典|東港迎王平安祭典]]。
* [[車城福安宮|車城福安宮]]：在臺灣屏東縣[[車城鄉_(臺灣)|車城鄉]]，是全臺灣最大間的[[土地公|土地公]]廟。
* [[花蓮勝安宮|花蓮勝安宮]]：在臺灣[[花蓮縣|花蓮縣]][[吉安鄉|吉安鄉]]，是臺灣頗具盛名的廟宇。
* [[樹林濟安宮|樹林濟安宮]]：在臺灣 [[新北市|新北市]][[樹林區|樹林區]]，在臺已有3百餘年的歷史。
*[[樹林鎮南宮|樹林鎮南宮]]：在臺灣[[新北市|新北市]][[樹林區|樹林區]]，為馬鳴山鎮安宮分靈。
*[[新營太子宮|新營太子宮]]：在臺灣[[臺南市|臺南市]][[新營區|新營區]]，為中壇元帥信仰具有代表性的廟宇。
*[https://web.archive.org/web/20180226211826/http://120.108.10.131/SR/US/ 西屯悟修道德宮]，在臺灣[[台中市|台中市]][[西屯區|西屯區]],主奉祀神尊太上道祖'玄天上帝'濟公師父（第三師父)為在地具代表性信仰宮廟。

== 中國大陸 ==
* [[七曲山大庙|七曲山大庙]]：为[[文昌帝君|文昌帝君]]祖庙，现存建于元明清三代的建筑群，为[[全国重点文物保护单位|全国重点文物保护单位]]。
* [[大庙飞来殿|峨眉山大庙]]：其飞来殿及香殿皆为元代原构，是中国南方罕有的元代建筑群之一，为全国重点文物保护单位。

== 香港 ==
* [[佛堂門天后古廟|佛堂門天后古廟]]：是[[香港|香港]]最古老的[[天后廟|天后廟]]，也是[[中國|中國]][[廣東|廣東]]沿岸現存最古老的天后廟。

== 澳門 ==
* [[聖母誕辰主教座堂|聖母誕辰主教座堂]] ：是[[天主教澳門教區|天主教澳門教區]]的主教座堂，由於昔日當地華人稱教堂為廟，故曾名大廟。
