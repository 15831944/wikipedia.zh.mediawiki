'''大町站'''（{{jpn|j=大町駅|hg=おおまちえき|rm=Ōmachi eki|f=y}}）是一個位於[[千葉縣|千葉縣]][[市川市|市川市]]{{link-ja|大町 (市川市)|大町 (市川市)|大町}}，屬於[[北總鐵道|北總鐵道]][[北總線|北總線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''HS07'''<ref>京成時刻表Vol.25 2010年7月9日</ref>。鄰近近{{link-ja|市川市動植物園|市川市動植物園}}所在地，因而設有「市川市動植物園」作為副站名。

== 年表 ==
* 1991年（[[平成|平成]]3年）3月31日 - 開業。
* 2010年（平成22年）7月17日 - 設有車站編號（HS07）。

== 車站結構 ==
[[對向式月台|對向式月台]]2面2線的[[高架車站|高架車站]]。月台牆壁繪有周邊學校的學童描繪的動物。

== 使用情況 ==
2015年度的1日平均上下車人次為'''1,636人'''，是北總線內上下車人次最少的，也是市川市內鐵路車站當中上下車人次最低的<ref>{{PDFlink|[https://www.keisei-agency.co.jp/transit_media/ebook/pdf/17.pdf 京成線・北総線基本データ - 京成エージェンシー]}}</ref>。
