'''大山羡代'''（{{lang-ja|大山 のぶ代}}，本名：{{lang|ja|'''山下 羨代'''（やました のぶよ）}}，{{bd|1933年|10月16日|||Ooyama, Nobuyo}}<ref>[http://www.nikkansports.com/entertainment/news/1476957.html 夫砂川が新告白…大山のぶ代が年齢詐称、実は81歳] - [[ニッカンスポーツ|日刊スポーツ]] 2015年5月15日配信</ref><ref>[http://www.sanspo.com/geino/news/20150515/geo15051517570037-n1.html 大山のぶ代、3歳サバ読んでいた！夫・砂川啓介が明かす] - [[サンケイスポーツ|サンケイスポーツ]] 2015年5月15日配信</ref><ref>{{Cite news |title = 認知症の大山のぶ代さん、夫が会見「いとおしさ感じる」 |newspaper = [[朝日新聞|朝日新聞]] |date = 2015-05-15 |url = http://www.asahi.com/articles/ASH5H66BGH5HUTFL008.html |accessdate = 2015-05-24}}</ref>），為[[日本|日本]]女性[[演员|演员]]、老牌[[聲優|聲優]]，以配音代表作《[[哆啦A夢_(角色)|哆啦A夢]]》聞名遐邇。籍贯[[東京都|東京都]]。2015年5月，大山羨代的丈夫砂川啟介在電台節目透露，她數年前患[[失智症|失智症]]（老人癡呆症），病症時好時壞，但若身體狀況許可仍會從事配音<ref>[http://www.cna.com.tw/news/firstnews/201505130384-1.aspx 為哆啦A夢配音26年 女星大山羨代失智]，中央社，2015年5月13日</ref>。同年10月，砂川啟介在電視節目透露她已經忘了哆啦A夢<ref>[http://hk.apple.nextmedia.com/enews/realtime/20151024/54349214 【叮噹：你就是我的聲音，記得嗎？】大山羨代腦退化 已忘「叮噹」]</ref>。2016年6月，砂川啟介罹患[[尿管癌|尿管癌]]幾度進出醫院，安排大山羨代住進老人院。

== 漢字名 ==
由於藝名將本名之羨字改寫為[[平假名|平假名]]，故時有誤譯'''大山千代'''、'''大山信代'''、'''大山忍代'''、'''大山延代'''、'''大山春代'''等。

==人物==
* 从20世纪70年代起就活跃于声优界的大山羡代，曾演出过《[[无敌超人桑波特3|无敌超人桑波特3]]》中的[[神胜平|神胜平]]、《[[哈里斯的旋风|哈里斯的旋风]]》中的[[石田國松|石田國松]]等早期动画的众多角色。
* 其健康爽朗的独特声質，使她经常为小男孩类型的角色献声。而就在其代表作《[[哆啦A夢_(角色)|哆啦A夢]]》而言，尽管并不是第一任负责饰演哆啦A梦的声优，但其出色精湛的演技，使他的形象栩栩如生，之后该作的一系列剧场版也由她负责饰演同一角色。大山羡代的配音与角色间天衣无缝的契合。使人们几乎遗忘了哆啦A梦前兩任的声优[[富田耕生|富田耕生]]與[[野澤雅子|野澤雅子]]。或许是哆啦A梦的形象过于深入人心，大山羡代投身这份工作後專注於詮釋哆啦A梦此一角色，之後几乎没有再擔任其他動畫的配音工作。但實際上她的本職是演員，配音則是並行工作的一環。
* 由于在《[[哆啦A夢|哆啦A夢]]》中26年堪称完美的表现，近几年特别是藤本弘先生（[[藤子·F·不二雄|藤子·F·不二雄]]）去世后，大山羡代就一直出席在日本所有相关活动，换句话说在日本，如果這個哆啦A梦活动没有大山羡代出席，就不能被視為算正式活动。此外不仅是动画与CD，包括BANDAI公司等游戏商制作有关多啦A梦的游戏及玩具时，其声音均由大山羡代录制，为录制声控玩偶的多啦A梦，她花了三天时间。
* 大山羡代曾表示：
: 「先生常开玩笑地对我说，他老是分不清自己到底是娶了声优老婆，还是哆啦A梦？」
: 「在外面买东西时我会特别注意，所以还好；不过在家裏和我老公吵架时，倒是常骂到一半，他就会念我：『-{喂}-～妳幹嘛又用哆啦A梦的声音啦！』（笑）。」<!--摘自台湾版Dorachannel网站-->
* 曾担任[[石原裕次郎|石原裕次郎]]主演的著名警匪劇《[[向太陽怒吼_(日本電視劇)|向太阳怒吼]]》的其中五集编剧。<!--《大雄的魔界大冒险》的对白中有提及本剧名称，从吉林美术出版社版翻译-->
* 她也是个有名的料理研究者，著有《有趣酒菜》一书。 <!--译自日文维基-->
* 配音生涯最有名的代表作品為《哆啦A梦》同名主角「'''哆啦A梦'''」的配音。在《哆啦A梦！全部的梦集合特别版》（已于2005年3月18日广播）之后退休，从1979年4月以来从业26年。
* 在「超级机器人大战」系列中，出現過她所配過音的《无敌超人桑波特3》主角神勝平，原本製作單位想請回大山羨代參與配音，但她表示「在出演哆啦A夢期間盡量不接其他角色配音」，以維持哆啦A夢的印象而推拒。但儘管如此，她仍然曾參與有聲書版《[[封神演義_(漫畫)|封神演義]]》的演出。
* 在水田山葵接棒配音以前，几乎全日本人都知道“哆啦A梦=大山羡代”。
* 在将多啦A梦的配音工作移交给水田山葵后，于2006年5月25日出版自传《我曾經是哆啦A梦》（{{lang|ja|ぼく、ドラえもんでした。 / 涙と笑いの26年うちあけ話}}）。
* 水田山葵接棒后第一回《室内钓鱼池》便是大山羡代第一次为多啦A梦配音的剧目。
* 由於大山羡代所有子女年幼夭折，為補足一生失去孩子的缺憾，她对[[哆啦A夢|哆啦A夢]]視如親生孩子，倾注了所有心血。声优一般會将工作和生活刻意分开，但是她却反其道而行。家里所有东西大到玩偶、小到水龙头装饰都是[[哆啦A夢|哆啦A夢]]。
