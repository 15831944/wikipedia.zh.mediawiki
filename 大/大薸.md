'''大薸'''（[[学名|学名]]：{{lang|la|''Pistia stratiotes''}}），又名'''大萍'''、'''大蕊萍'''、'''大莲'''、'''水芙蓉'''，为[[天南星科|天南星科]][[大薸属|大薸属]]的唯一物種。

== 形态 ==
多年生浮生[[草本|草本]]，根须垂悬水中，是[[漂浮性水生植物|漂浮性水生植物]]。楔形叶片呈莲座状簇生在短缩茎上，叶肉疏松，浮生于水面。腋芽能生出匍匐茎，顶端长出新株。白色花，雌雄同株，[[肉穗花序|肉穗花序]]。[[浆果|浆果]]内有绿黄色的种子。

== 分布 ==
分布在全球[[热带|热带]]、[[亚热带|亚热带]]以及[[中国|中国]]的[[浙江|浙江]]、[[安徽|安徽]]、[[广东|广东]]、[[广西|广西]]、[[山东|山东]]、[[云南|云南]]、[[福建|福建]]、[[四川|四川]]、[[湖北|湖北]]、[[江苏|江苏]]、[[湖南|湖南]]等地，生长于海拔200米至1,900米的地区，常生于沟渠中、沟渠边、静水中、湖边、稻田中、流水中、稻田边和溪边，目前尚未由人工引种栽培。

== 别名 ==
{{Citation needed|水浮莲、|time=2012-10-07}}天浮萍、大萍叶（江苏） 水荷莲（浙江、江西石城、云南元阳） “卡过”（傣族语） 肥猪草（安徽滁州）

== 水浮蓮之中醫藥療用 ==
其葉可入藥，味[[苦|苦]]、[[性涼|性涼]]，能[[袪風|袪風]][[發汗|發汗]]、[[利尿|利尿]][[解毒|解毒]]。
