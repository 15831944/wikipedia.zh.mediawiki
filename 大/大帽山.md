'''大帽山'''（{{lang-en|Tai Mo Shan}}），俗稱「'''大霧山'''」、「'''白果山'''」，是[[香港|香港]]最高的[[各国最高点列表|山峰]]，海拔957米，比第二高的山峰[[鳳凰山_(香港)|鳳凰山]]高23米。大帽山昂然屹立新界中部，幅員廣闊，傲視同群，坡度相對平緩，其附近範圍被劃入[[大帽山郊野公園|大帽山郊野公園]]和[[城門郊野公園|城門郊野公園]]之內。附近則有[[大欖郊野公園|大欖郊野公園]]及[[林村郊野公園|林村郊野公園]]。大帽山位於[[荃灣區|荃灣區]]和[[元朗區|元朗區]]交界，並有小部分屬於[[大埔區|大埔區]]。大帽山上多條山脊均有俗稱，加上山體偏高，平均倚附在山脊俱在700米以上。

==地理資料==
大帽山主要由[[火山岩|火山岩]]組成，山谷中亦有不少[[沉积岩|沉積岩]]。山坡的上半部露出山脊，亦有一些石澗由高地流到深淵山谷。東南部主要可分為三條主流，大曹石澗下[[曹公潭|曹公潭]]，大圓石澗下[[三疊潭|三疊潭]]及大城石澗下[[城門水塘|城門水塘]]；東北部與支脈[[觀音山_(元朗區)|觀音山]]相互由[[梧桐寨|梧桐寨]]瀑布開始，下經[[林村谷|林村谷]]匯合[[大埔河|大埔河]]後在[[太和邨|太和邨]]附近流出[[吐露港|吐露港]]。西南部由[[甲龍|甲龍]]至[[河背水塘|河背水塘]]引水道經蓮花山及田夫仔下[[大欖涌水塘|大欖涌水塘]]。
大帽山處香港中央，亦是香港最高峰。諸澗發源大帽，群脊伸延八方，前白略有梧桐水系、大雷水系、大石水系、大曹水系、大圓水系及大城水系，後者有觀音山延脈、鹿山延脈、草山及針山延脈等，可通之途、可組之線多姿多采，各色其色，山頂飽覽香港絕大部份地方，天色好時更可遠望广东深圳，是一座令人難忘的山體。

==氣候與景觀==
大帽山為全香港平均溫度最低的地方之一<ref>[http://www.hko.gov.hk/cis/regionc.htm 香港分區氣候]</ref>，也是香港少數被歸類為[[溫帶季風氣候|溫帶季風氣候]]的地方，全年總平均溫度只有攝氏17.2度左右，比香港市區平均的攝氏23.3度低6.1度，當隆冬下市區溫度在徘徊10度左右時，清晨山頭溫度多已接近冰點（0度）或更嚴寒，並多會出現強烈的[[風寒效應|風寒效應]]，此時登山的遊客往往會感到比實際溫度低數度至十度左右的[[體感溫度|體感溫度]]，若做不足保暖措施便可能增加[[凍傷|凍傷]]的風險。因此冬季時有結[[冰|冰]]或[[霜|霜]]凍現象，甚至會間中出現[[凍雨|凍雨]]、[[霰|霰]]、[[雨夾雪|雨夾雪]]及輕微降[[雪|雪]]的冬季特殊天氣現象。全年亦有103天處於寒冷天氣（最低氣溫低於12.0度）狀態，乃全港之冠。另全年平均[[相對濕度|相對濕度]]亦高達百分之89，尤其每年三、四月[[霧|霧]]氣最濃。

在每年9月底至11月期間的秋高氣爽的日子，萬里晴空山頭經常清晰可見，身處其上，能俯瞰香港陸地版圖超過八成半地域，東面的[[獅子山_(香港)|獅子山]]、[[飛鵝山|飛鵝山]]；南面港島的[[太平山|太平山]]及背後的[[南丫島|南丫島]]，遠處嶼南的[[大東山|大東山]]、[[鳳凰山_(香港)|鳳凰山]]；西面[[九徑山|九徑山]]及[[青山_(香港)|青山]]；西北面的[[雞公嶺|雞公嶺]]、[[大刀屻|大刀屻]]及[[觀音山_(元朗區)|觀音山]]；東北面的[[馬鞍山_(香港山峰)|馬鞍山]]及[[八仙嶺|八仙嶺]]及[[吐露港|吐露港]]。[[元朗平原|元朗平原]]以至可遠眺[[深圳|深圳]][[福田|福田]]等均盡入眼簾。

由於位於高地，氣候獨特，接近高空大氣，加上山頂全無障礙物妨礙信號收發，[[香港天文台|香港天文台]]、[[民航處|民航處]]及{{fact|[[中國人民解放軍戰略支援部隊網絡系統部|中國人民解放軍戰略支援部隊網絡系統部]]|2017-10-25}}於大帽山設置兩個雷達站及一個監聽站，提供氣象及航空[[雷達|雷達]]影像和數據，以及監察香港異見人士動向。據聞[[香港殖民地時期|殖民地時期]]更是重要軍事瞭望台。

大帽山雖然是香港第一高峰，由於在930米以上屬於天文台禁區，在回歸前一般市民可望而不可及。而訪遊大帽山，遊人多會沿著大帽山林道（車路）登上山頂部分，而大帽山林道的最高點就在頂部禁區閘口之前，該處之海拔高度已近930米，但此地並非大帽山上可以登遊的最高位置。

=== 2016年1月強烈寒潮 ===
一股來自[[俄羅斯|俄羅斯]][[西伯利亞|西伯利亞]]非常強烈的[[寒潮|寒潮]]於1月22日中午前後抵達[[香港|香港]]，大帽山的[[氣溫|氣溫]]由中午約攝氏16度暴跌至23日清晨的零下2至3度，並首先出現結[[霜|霜]]和結[[冰|冰]]現象，當天出現稍後更因北風強烈及[[潮濕|潮濕]]低溫而形成[[霧凇|霧凇]]、頻密[[雨夾雪|雨夾雪]]、[[霰|霰]]、[[凍雨|凍雨]]及陣[[雪|雪]]天氣。24日大帽山曾經錄得零下6.0度的氣溫，全日最高溫度只有零下3度，翌日也錄得零下5.9度低溫。而25日清晨4時大帽山錄得最低草溫達零下6.7度，是大帽山以至有記錄以來最嚴寒的一天<ref>{{cite news|url=http://hk.apple.nextmedia.com/realtime/breaking/20160124/54682360|title=【真‧寒流】【大帽山】草溫低見-5.6°C 天光仲凍|accessdate=2016-01-24|newspaper=蘋果日報|date=2016-01-24}}</ref>。

== 疑似軍事用地 ==
大帽山山頂旁加建了一個[[球體|球]]形建築物，用途不明，但在2014年有媒體稱是一個監聽香港無線電訊號的監聽站，估計由[[解放軍駐港部隊|解放軍駐港部隊]]興建，更認為最有可能是由空軍三局負責監聽情報，更指用於監察如市民、政界人士和外國政要等的電話及無線電通訊<ref>{{Cite web |url=http://orientaldaily.on.cc/cnt/news/20141118/00176_059.html |title=加雜誌指大帽山「白波」屬駐軍監聽站 |date=2014-11-18 |publisher=東方日報 (香港)}}</ref><ref>{{Cite web |url=https://hk.news.appledaily.com/local/daily/article/20141118/18939001 |title=外國軍事雜誌視察大帽山頂：解放軍「巨波」監聽港電話  |date=2014-11-18 |publisher=蘋果日報 (香港)}}</ref>，而內裏軍人平日使用民用車牌車輛出入，行為低調神祕。由於是軍事重地，遊人不宜久留。禁區部分四周圍有鐵絲網，更嚴禁攝影。

《[[漢和防務評論|漢和防務評論]]》雜誌稱，該建築為[[總參三部|總參三部]]建設的信號情報監聽站，比在[[新疆|新疆]]、[[西藏|西藏]]的監聽站還大；從士兵的服裝判斷，是中共解放軍空軍所有；該雜誌又稱，監聽站是中國版《[[稜鏡計畫|稜鏡計畫]]》的一部分，解放軍已經在全中國各地與[[台灣海峽|台灣海峽]]建立了35座同類的監聽站，但香港的規模最大。<ref>{{cite news|author=管淑平(譯)|title=中國版稜鏡計畫 監聽全香港|url=http://news.ltn.com.tw/news/world/paper/831085|accessdate=2014-11-19|newspaper=《[[自由時報|自由時報]]》|date=2014-11-18}}</ref><ref>{{cite news|author=張謙|title=陸版稜鏡 共軍在香港設站監聽|url=http://www.cna.com.tw/news/acn/201411170277-1.aspx|accessdate=2014-11-19|newspaper=[[中央社|中央社]]|date=2014-11-17}}</ref><ref>{{cite news|author=余浚安|title=香港大帽山神秘設施　被疑監聽全港通訊|url=http://www.appledaily.com.tw/realtimenews/article/new/20141117/507997/%E9%A6%99%E6%B8%AF%E5%A4%A7%E5%B8%BD%E5%B1%B1%E7%A5%9E%E7%A7%98%E8%A8%AD%E6%96%BD%E3%80%80%E8%A2%AB%E7%96%91%E7%9B%A3%E8%81%BD%E5%85%A8%E6%B8%AF%E9%80%9A%E8%A8%8A|accessdate=2014-11-19|newspaper=[[台灣蘋果日報|台灣蘋果日報]]|date=2014-11-17}}</ref><ref>{{cite news|author=劉屏|title=共軍截聽無線通訊 全台淪陷|url=https://tw.news.yahoo.com/%E5%85%B1%E8%BB%8D%E6%88%AA%E8%81%BD%E7%84%A1%E7%B7%9A%E9%80%9A%E8%A8%8A-%E5%85%A8%E5%8F%B0%E6%B7%AA%E9%99%B7-215037037--finance.html|accessdate=2014-11-19|newspaper=[[旺報|旺報]]|date=2014-11-18}}</ref>

對於以上的懷疑，香港特區政府在回覆[[立法會議員|立法會議員]]質詢時指大帽山頂有關範圍並不屬全港19個軍事用地之一，稱該處一直以來是特區政府的土地，用於設置不同的通訊裝置，並補充說香港19處軍事用地均是根據中英兩國政府於1994年就[[香港軍事|香港軍事]]用地未來用途的安排所互換照會而設立的<ref>{{Cite web |url=http://www.info.gov.hk/gia/general/201410/29/P201410290489.htm |title=立法會九題：軍事用地 以下為今日（十月二十九日）在立法會會議上陳家洛議員的提問和保安局局長黎棟國的書面答覆︰|accessdate=2018-07-12 |date=2014-10-29 |publisher=香港政府新聞公報}}</ref>。

==動植物==
大帽山山峰東南面有[[森林|森林]]，但受[[氣候|氣候]]限制，[[喬木|喬木]]只能伸展至海拔550米為止，再高的地方[[植物|植物]]則以[[灌木|灌木]]及[[草|草]]為主。有多種稀有[[香港原生植物|香港原生植物]]生長於大帽山，包括[[葛量洪茶|葛量洪茶]]、[[香港金線蘭|香港金線蘭]]、[[穗花杉|穗花杉]]等等。從前，大帽山曾盛產一種[[綠茶|綠茶]]。現時一些山坡上仍可看到[[梯田|梯田]]的痕跡。

[[動物|動物]]方面，大帽山有各種[[鳥|鳥]]類、[[蛇|蛇]]及[[大头龟|大头龟]]、[[蝴蝶|蝴蝶]]，亦有不少[[螢火蟲|螢火蟲]]。
