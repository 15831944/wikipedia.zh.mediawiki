'''大蕉松鼠'''（[[學名|學名]]：''{{lang|la|Callosciurus notatus}}''），又名'''芭蕉松鼠'''、'''東方松鼠'''及'''三色松鼠'''，為[[松鼠科|松鼠科]][[麗松鼠屬|麗松鼠屬]]的動物。

[[File:Callosciurus_notatus_in_Singapore_Zoo.jpg|thumb]]

== 特征 ==
身體連的尾巴一般長 20-30 公分左右（約 7.9 - 12 寸），並擁有灰色或棕色板栗的腹部和腹旁有黑白色相間的條紋。

== 分佈 ==
分佈於[[印尼|印尼]]、[[馬來西亞|馬來西亞]]、[[新加坡|新加坡]]和[[泰國|泰國]]；多棲息於[[森林|森林]]、[[紅樹林|紅樹林]]、[[公園|公園]]、[[花園|花園]]及[[農業|農業]]範圍。多被果農認為有害。

== 習性 ==
主要食用由樹葉和果實，但也有時也會食用昆蟲和鳥蛋。大蕉松鼠有時會破開樹枝並抓食當中所含有的螞蟻幼蟲。牠們亦可進食體積比牠們身體大得多的[[水果|水果]]，如[[芒果|芒果]]、[[菠蘿蜜|菠蘿蜜]]、[[椰子|椰子]]等。牠們非常快速和敏捷，能在樹與樹之間跳上幾米，一般很少在地面上遊蕩。
