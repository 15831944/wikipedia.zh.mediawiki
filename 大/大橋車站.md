'''大橋車站'''位於[[台灣|台灣]][[臺南市|臺南市]][[永康區|永康區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]的[[鐵路車站|鐵路車站]]。規劃中的[[臺南捷運藍線|臺南捷運藍線]]，亦預計在此設B10站。

== 車站構造 ==
*[[岸式月台|岸式月台]]兩座。
*地上車站。
*有別於一般車站單調樣式，大橋車站刻意經過配色彩繪，站體顯得十分活潑。

== 營運狀況 ==
*本站為[[臺灣鐵路管理局車站等級#簡易站|簡易站]]，由[[臺南車站|臺南站]]管理。
*本站僅停靠區間車。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]、[[icash|icash2.0]]以及[[有錢卡|HappyCash]]進出站。

站名取自所在地古地名「大橋」，而這名稱又可溯自17世紀[[荷蘭|荷蘭人]]在此興建的磚橋。詳見[[大橋_(臺南市)|大橋]]。

此處原是臺南東郊的農業區域，而1980年完成[[中華東路_(臺南市)|中華東路]]及永康中華路（後來編為[[台1線|台1線]]）後，才開始快速發展。車站即位於台1線永康陸橋下方，附近商業繁盛、高樓林立，且有大專院校與軍營。因客源眾多，雖僅停靠區間車，運量仍居臺南市境內第三大站（僅次於臺南、新營），業績優良。

2010年全年上車人數844,450人、下車人數818,151人<ref>[http://www.railway.gov.tw/Upload/intro/file/99Year/pdf/t11.pdf 各站客貨起訖量] {{webarchive|url=https://web.archive.org/web/20120616133631/http://www.railway.gov.tw/Upload/intro/file/99Year/pdf/t11.pdf |date=2012-06-16 }}，臺灣鐵路管理局</ref>，是臺鐵捷運化新設車站中，業績第二名（僅次於[[汐科車站|汐科車站]]）。

== 歷史 ==
*選址時期站名曾暫稱'''奇美'''、'''六甲頂'''等。
*2001年：臺鐵在進行[[台鐵捷運化|捷運化]]評估時，認為大橋附近商業繁榮、人口集中，為12處新站選址中條件最佳者<ref>台鐵鐵路捷運化 永康大橋站開工．聯合報，2001年6月6日，18版/台南縣新聞</ref>，因此優先興建。
*2001年6月5日：舉行開工典禮。
*2002年10月4日：正式啟用。
