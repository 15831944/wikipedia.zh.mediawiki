'''大森車站'''（{{jpn|j=大森駅|hg=おおもりえき|rm=Ōmori eki|f=y}}）是一個位於[[東京都|東京都]][[大田區|大田區]][[大森北|大森北]]一丁目，屬於[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''JK 18'''。

此站停靠的路線在軌道名稱上是[[東海道本線|東海道本線]]（詳情參見路線條目與{{link-ja|鐵道路線的名稱|鉄道路線の名称}}），但此站只有行走[[電車線與列車線|電車線]]的[[京濱東北線|京濱東北線]]列車停靠，旅客導覽不顯示「東海道（本）線」。依據[[特定都區市內|特定都區市內]]，本站屬於「[[特定都區市內#設定區域一覧|東京都區內]]」。

== 車站構造 ==
[[島式月台|島式月台]]1面2線的[[地面車站|地面車站]]，設有[[跨站式站房|跨站式站房]]。設有[[綠窗口|綠窗口]]、自動閘機、指定席售票機。

== 使用情況 ==
2016年（平成28年）度的1日平均'''上車'''人次為'''95,189人'''。JR東日本車站第45位。此站是不與其他路線接續的JR車站當中上下車人次最多的。

== 事故 ==
1885年10月1日，一輛由[[新橋站_(日本)|新橋]]前往本站的臨時列車（[[池上本門寺|池上本門寺]]的[[參拜|參拜]]列車，14節客車編組）在自下行路軌切換到上行路軌的過程中，在[[轉轍器|轉轍器]]上發生客車[[脫軌|脫軌]]翻車的意外，造成乘客一死一傷，是日本鐵路史上第一起牽涉到旅客死亡的事故。
