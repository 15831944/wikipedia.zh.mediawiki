'''大湖車站'''位於[[高雄市|高雄市]][[路竹區|路竹區]]，鄰近[[台灣省道|省道]][[台1線|台1線]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]的[[鐵路車站|鐵路車站]]。也是[[湖內區|湖內區]]的主要門戶。

== 車站構造 ==
*[[島式月台|島式月台]]、[[側式月台|側式月台]]各一座。
*[[電梯|電梯]]一座。

== 營運狀況 ==
*本站雖位於路竹區，但乘客主要來源地為車站前方的[[湖內區|湖內區]]大湖地區，為高雄市境內最北的台鐵車站。
*目前為三等站，主要停靠車種以[[臺鐵區間車|區間車]]為主，[[臺鐵區間快車|區間快車]]3029次，[[莒光號列車|莒光號]]僅停靠513次。
**[[莒光號列車|莒光號]]：僅停靠南下(逆行)513次。
**[[自強號列車|自強號]]：不停靠。
**觀光列車不停靠。
*配合台鐵新推行的鐵路捷運化政策，在西側增建[[側式月台|側式月台]]1座。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1900年11月29日：設立「大湖街乘降場」
*1920年：全臺地名大變革，站名配合更改為「大湖」。
*2006年8月1日：因業務量下降，降為三等站，但臺鐵運務段及車站專區的網頁標示還是二等站（於2016年6月標示已更為三等站）。
*2013年9月30日：啟用多卡通刷卡機。
