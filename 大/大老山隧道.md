'''大老山隧道'''（{{lang-en|'''Tate's Cairn Tunnel'''}}），是[[香港|香港]]最長的行車隧道，簡稱'''大隧、老隧'''，全長達3.95公里，車速限制每小時70公里，是連接[[九龍|九龍]]與[[沙田|沙田]]的4條隧道或道路中，通往九龍東的道路，另外3條為[[獅子山隧道|獅子山隧道]]、[[青沙公路|青沙公路]]及[[大埔公路|大埔公路]]。隧道穿過[[大老山|大老山]]，連接新界東沙田[[小瀝源|小瀝源]]和九龍東黃大仙[[鑽石山|鑽石山]]，為一條雙程雙線分隔雙管隧道，[[隧道交通督導員|隧道人員]]獲授權執行香港法例368章《行車隧道（政府）條例》。

雖然隧道九龍的實際入口位於鑽石山[[南蓮園池|南蓮園池]]旁，但連接隧道鑽石山入口以南和[[九龍灣|九龍灣]][[麗晶花園|麗晶花園]]（[[彩虹交匯處|彩虹交匯處]]）之間，穿越介乎[[彩虹邨|彩虹邨]]和[[采頤花園|采頤花園]]中間的一段約一公里長高架引道，屬隧道區域一部分，與[[觀塘繞道|觀塘繞道]]重疊，故實際起點是以麗晶花園計算，故此大老山隧道連同引道全長5.85公里。

==歷史==
[[File:Tates_Cairn_Tunnel_Sha_Tin_portal.JPG|thumb]]
大老山隧道在1988年5月1日動工興建，承建商是[[金門建築|金門]][[西松建設|西松]]聯營公司，並在1991年6月竣工<ref name=tctc>{{Cite web |url=http://www.tctc.com.hk/chi/tunnel/facts.html |title=大老山隧道有限公司 |accessdate=2009-12-15 |work= |publisher= |date= |archive-url=https://web.archive.org/web/20100123122106/http://www.tctc.com.hk/chi/tunnel/facts.html |archive-date=2010-01-23 |dead-url=yes }}</ref>，耗資19.6億港元，於1991年6月26日通車<ref name=tctc/>及於7月1日開幕。收費廣場位於[[沙田區|沙田區]][[小瀝源|小瀝源]]，共有14個收費亭，其中9個為人手收費亭，餘下5個為[[快易通|快易通]]客戶專用的自動繳費通道。

==營運機構==
[[大老山隧道有限公司|大老山隧道有限公司]]在1988年以「[[民間興建營運後轉移模式|建造、營運、移交]]」方式獲發大老山隧道30年的專營權，全部資金由私營公司注入，隧道公司負責隧道及有關項目的建造工程，亦負責專營權有效期內的管理及營運，透過收取隧道費獲得合理回報（13%），當專營權屆滿時，隧道設施將歸還[[香港政府|香港政府]]。公司股東包括[[港通控股有限公司|港通控股有限公司]]39.5%股權、[[新創建集團有限公司|新創建集團有限公司]]、[[西松建設|西松建設]]株式會社和[[中建企業有限公司|中建企業有限公司]]。

隧道公司已經於2007年轉虧為盈，但累積回報率至今仍只有6.61%，與原本預計回報率13.02%仍有距離<ref>{{cite web |url=http://news.sina.com.hk/news/20121207/-2-2841929/1.html |title=存档副本 |accessdate=2013-05-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20130414071240/http://news.sina.com.hk/news/20121207/-2-2841929/1.html |archivedate=2013-04-14 }} 明報:大老山隧道申加價11%</ref>，因此隧道公司過去數年多次申請加價。香港政府基於「小政府大市場」原則，不擬回購大老山隧道公司專營權<ref>[http://archive.news.gov.hk/isd/ebulletin/tc/category/infrastructureandlogistics/050615/html/050615tc06001.htm 政府不擬回購大老山隧道專營權]</ref>，加費申請通常均獲通過。最近一次加費申請發生在2012年3月，其後政府在2013年5月14日宣佈，[[行政長官|行政長官]]會同[[行政會議|行政會議]]批准大老山隧道有限公司經修訂的加費申請，電單車隧道費由12元加至13元，其餘各類別車輛和額外車軸的隧道費增加兩元，當中私家車及的士的隧道費由15元加至17元，經修訂的加費申請的增幅由原本建議的17％至24％，下調至現時的6％至13％，平均增幅由原來的19.6％降至11.1％。新收費已於2013年8月1日開始實施。<ref>http://www.news.gov.hk/tc/categories/infrastructure/html/2013/05/20130514_174505.shtml 香港政府新聞網:大老山隧道8月起加價</ref>

2018年7月11日凌晨大老山隧道專營權屆滿，港府正式接管大老山隧道，香港法例368章《行車隧道（政府）條例》及其附屬法例適用範圍擴展至大老山隧道，同時393章《大老山隧道條例》及其附屬法例被廢除。

== 使用情況 ==
大老山隧道興建目的之一是紓緩[[獅子山隧道|獅子山隧道]]的車流，2012年大老山隧道是沙田區4條聯外隧道中每日平均汽車流量第二高的隧道，僅次於獅子山隧道，2012年平均每日有55,767架車輛使用<ref>[http://www.td.gov.hk/filemanager/en/content_4609/table31s.pdf 運輸署:隧道汽車流量、青嶼幹線汽車流量及汽車渡輪服務]</ref>，因收費較獅子山隧道昂貴，本隧道使用量一直不及獅子山隧道，但有效分流前往九龍東和港島東的車輛，於平日繁忙時間同樣擠塞。然而[[大欖隧道|大欖隧道]]通車分流新界西繞道前往港島的車輛，使本隧道使用率減少。

== 空氣質素問題 ==
1996年大老山隧道空氣質素曾遭受非議，當非空調巴士（俗稱「[[熱狗_(巴士)|熱狗]]」）駛經隧道時，乘客需要關上車窗或以手帕掩著口鼻，亦曾有區議員發起戴口罩過大老山隧道運動<ref>[https://web.archive.org/web/20010404060419/http://susis.ust.hk/~bus/ 1996年6月沙田區議會交通及運輸委員會主席彭長緯議員發起戴口罩過大隧，抗議大老山隧道空氣質素日趨轉壞]，298特區: 車隊冷氣化。</ref><ref>1996年無綫電視新聞透視預告: 「大老山隧道空氣質素差，隧道公司否認。法例要求乜嘢標準?」</ref>，九巴則為相關車長提供面罩式防毒面具，並優先安排有關路線全空調化解決。

===備註===
*上述轉車收費由2018年3月1日生效
*12歲以下小童（用小童或個人八達通）次程轉車時為上述轉車收費的一半
*65歲或以上長者（用長者或個人八達通）及12歲以下合資格殘疾兒童（用「殘疾人士身分」個人八達通）次程轉車時為HK$2或上述轉車收費的一半（以較低者為準）；而由75X往大埔方向轉乘277X往粉嶺方向，次程轉車時為免費
*12-64歲合資格殘疾人士（用「殘疾人士身分」個人八達通）次程轉車時為HK$2或上述轉車收費（以較低者為準）；而由75X往大埔方向轉乘277X往粉嶺方向，次程轉車時為免費
*合資格學生(用「學生身分」個人八達通)乘搭上述同一條或同一組別的路線回程，全程或分段收費為HK$12或以上，為上述轉車收費的一半
*有關轉車費必須使用同一張八達通轉乘另一條相同方向的路線，方可享此優惠，若以現金轉車的乘客或乘搭其他路線（如[[過海隧道巴士680線|過海隧道巴士680線]]、[[過海隧道巴士681線|過海隧道巴士681線]]、[[過海隧道巴士682線|過海隧道巴士682線]]、[[新巴798線|新巴798線]]等）必須於轉車時繳付車上所列的收費（特別註明除外）；而轉車費不可同時享用其他八達通轉乘優惠（如有）
*轉乘同一條或同一組別路線（包括74B/74C/74X、82P/82X、85S/85X、277E/277P/277X）並不能享有轉乘優惠。
*使用[[九巴月票|九巴月票]]的乘客，如在大老山隧道轉車站轉乘，會被扣除額外1程合資格車程。

==大老山隧道有限公司==
'''大老山隧道有限公司'''({{lang-en|Tate's Cairn Tunnel Company Limited}})擁有由[[香港政府|香港政府]]批出專營權（建造、營運、轉移）管理營運大老山隧道。公司創辦於1988年，當時由[[華潤集團|華潤集團]]及[[新創建集團|新創建集團]]持有。但現在主要股東為[[港通控股|港通控股]]、[[新創建集團|新創建集團]]、[[西松建設|西松建設]]以及[[許愛周|許愛周]]家族持有的[[中建企業|中建企業]]。

大老山隧道設有[[職工會|職工會]]，分別為[[香港隧道及公路幹線從業員總會|香港隧道及公路幹線從業員總會]]大老山隧道分會（[[工聯會|工聯會]]屬會），理事長為[[葉國偉|葉國偉]]先生；另一工會為香港收費道路從業員協會（[[職工盟|職工盟]]屬會，登記編號：926）。

有關[[專營權|專營權]]已於2018年7月11日屆滿，移交予[[香港政府|香港政府]]。
