'''大禹車站'''位於[[台灣|台灣]][[花蓮縣|花蓮縣]]-{[[玉里鎮|玉里鎮]]}-，曾為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]，於1995年3月1日裁撤。

== 車站構造 ==
*島式月臺一座。(已拆除)

== 利用狀況 ==
*廢止前為三等站。
*目前本站在當地社區民眾的整理之下，維持一定程度的整潔，狀況比台鐵[[臺東線|臺東線]]許多無人招呼站還要好。

== 車站周邊 ==
;教育單位
*花蓮縣大禹國民小學

;政府機關
*花蓮縣警察局玉里分局大禹派出所

== 歷史 ==
*1917年11月1日：設「'''末廣'''」驛，「末広」日文念作すえひろ（suehiro）。
*1946年：為紀念古代「[[大禹治水|大禹治水]]」的精神，行政區域改為[[玉里鎮|玉里鎮]][[大禹里|大禹里]]，故站名配合地名改為「大禹」。
*1995年3月1日：廢站。
