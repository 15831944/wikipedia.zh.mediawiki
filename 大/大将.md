'''大將'''，也稱'''一級上將'''，是部分國家設有的軍銜，一般位於[[元帅|元帅]]和[[二級上將|二級上將]]之間。通常以四顆星為代表。
== 中華人民共和國 ==
[[File:PLA_dà_jiàng_Type_55.svg|缩略图]]
[[File:Yi_Ji_Shang_Jiang(PLAGF_OF-11_Type_88).gif|thumb]]
{{Main|中國人民解放軍大將}}
大將是[[中國人民解放軍|中國人民解放軍]]1955年軍銜制的一級軍銜，位於[[元帥|元帥]]之下，[[上將|上將]]之上，1965年随着解放軍廢除軍銜制而取消。而'''[[中国人民解放军一级上将|一级上將]]'''是中國人民解放軍1988年恢復軍銜制後設立的最高軍銜，因時任[[中央軍委主席|中央軍委主席]][[邓小平|邓小平]]拒絕受銜，一直設而未授，於1994年取消。
;相关历史
[[中華人民共和國|中華人民共和國]]建立後，[[中國人民解放軍|中國人民解放軍]]於1955年給在[[土地革命|土地革命]]、[[抗日戰爭|抗日戰爭]]、[[解放戰爭|解放戰爭]]中作出巨大貢獻的十位軍事領導人授予大將官軍銜。

1988年7月1日，[[第七届全国人民代表大会常务委员会|七届全国人大常委会]]第二次会议审议通过了中央军委提出的《中国人民解放军军官军衔条例》。<ref>{{Cite web |url = http://military.people.com.cn/n/2014/0712/c1011-25272225.html |title = 中国人民解放军军衔制 |author =  |date = 2014-07-12 |publisher = 人民网 |language = zh-cn |accessdate = 2017-10-06 }}</ref>1988年9月23日，[[中華人民共和國國務院|中華人民共和國國務院]]、[[中央軍委|中央軍委]]頒布了《中國人民解放軍現役士兵服役條例》。由此形成了中國人民解放軍完整的軍銜體系。此軍銜制度于1988年10月1日正式實施。在實行新軍銜制的同時，1988年7月1日，第七屆全國人大常委會第二次會議審議通過了中央軍事委員會《關於確認1955年至1965年期間授予的軍官軍銜》的議案，對1955年至1965年期間被授予軍官軍銜的人員，其軍銜予以確認，表明“55式軍銜”作為個人榮譽至今仍然有效。<ref>徐平 著 《新中國實行軍銜制纪实》 金城出版社 2010年出版 ISBN 978-7-80251-378-5 第七章 P225</ref>

當時公布的軍銜制度不設[[元帥|元帥]]、大將和[[大尉|大尉]]，而以一級上將為最高軍銜。軍官軍銜設3等11級，即[[一級上將|一級上將]]、[[二級上將|上將]]、[[中將|中將]]、[[少將|少將]]；[[大校|大校]]、[[上校|上校]]、[[中校|中校]]、[[少校|少校]]；[[上尉|上尉]]、[[中尉|中尉]]、[[少尉|少尉]]。海軍、空軍軍官在軍銜前分别冠以“海軍”、“空軍”。專業技術軍官，在軍銜前冠以“專業技術”。由於當時的中央軍委高層[[鄧小平|鄧小平]]、[[楊尚昆|楊尚昆]]主動提出自己不受軍銜，因此，一級上將空缺。

1994年7月1日，中華人民共和國第八屆全國人民代表大會常務委員會第七次會議對1988年制定的《中國人民解放軍軍官軍銜條例》作了修改，取消了一級上將軍銜，軍官軍銜由原來的3等11級改為3等10級。

== 中華民國 ==
{{See Also|中華民國上將列表#一級上將}}
一級上將為戰時將領之最高軍階，於[[二級上將|二級上將]]之上，服現役年齡無上限，並為終身職，是目前[[中華民國國軍|中華民國國軍]]唯一沒有「服現役限制」及「除役年齡」的軍階。一級上將擔任[[中華民國國防部參謀本部|參謀總長]]（2013年1月15日以前）或[[中華民國總統府戰略顧問|總統府戰略顧問]]。

一級上將依[[:s:陸海空軍軍官士官服役條例|陸海空軍軍官士官服役條例]]第5條規定終身現役，除非自請退伍，迄今退伍原因可分以下三種：
*出任[[行政院院長|行政院院長]]：陸軍一級上將[[郝柏村|郝柏村]]，為避免「軍人干政」疑慮，於出任行政院院長前，主動辦理退伍。
*赴[[中國大陸|中國大陸]]探親：陸軍一級上將[[羅本立|羅本立]]，因法律規定赴[[中國大陸|中國大陸]]探親，需「非現役軍人」才准放行，故主動請辭辦理退伍。
*出任[[中華民國國防部部長|國防部部長]]：空軍一級上將[[唐飛|唐飛]]、陸軍一級上將[[湯曜明|湯曜明]]、海軍一級上將[[李傑_(臺灣)|李傑]]、空軍一級上將[[李天羽|李天羽]]等連續四任參謀總長都因出任國防部部長而辦理退伍。

;歷史法源
依《陸海空軍一級上將在台期間繼續保留現役及其待遇辦法》規定：國軍在臺期間，現役一級上將，屆滿現役限齡或服役限齡，經免職而未派新職者，得繼續保留為[[現役|現役]]，不辦[[退伍|退伍]]。保留現役之一級上將，應由[[中華民國國防部|國防部]]設置[[辦公室|辦公室]]，以便從事[[國防|國防]]及[[軍事|軍事]]問題之[[研究|研究]]工作，備供[[總統|總統]]諮詢。保留現役之一級上將，其月支薪給、[[車輛|車輛]]及[[眷舍|眷舍]]等，均以維持原職所有[[待遇|待遇]]為原則；並得於辦公室內設置[[秘書|秘書]]或[[參謀|參謀]]一員、[[衛士|衛士]]一名，以辦理其有關[[行政|行政]]事宜。<ref>{{Cite web |url = http://law.mnd.gov.tw/scp/Query4A.asp?FullDoc=all&Fcode=A004000013 |title = 陸海空軍一級上將在台期間繼續保留現役及其待遇辦法 |author =  |date = 中華民國70年12月8日國防部淦準字第5364號令修正發布全文6條 |publisher = 中華民國國防部 |language = zh-tw |accessdate = 2014-10-19 |quote = 保留現役一級上將於再任實際軍職（或其他軍職）時，其待遇即另依新任軍職之規定辦理。如調任外職，應予外職停役，並停止其第三條及第四條之待遇。免除外職時，辦理回役，恢復其待遇。競選停役者亦同。}}</ref>

目前《陸海空軍一級上將在台期間繼續保留現役及其待遇辦法》已於1999年9月1日廢止。<ref>[http://law.mnd.gov.tw/scp/Newsdetail.asp?NO=1A004000013 廢止時間]</ref>最初由1965年「五十四年十二月十日國防部（54）約法字第0316號令」訂定發布，再於1981年「七十年十二月八日國防部（70）淦湜字第5364號令」修正發布，後於1999年「八十八年九月一日國防部（88）鐸錮字第12418號令」發布廢止。<ref>[http://law.mnd.gov.tw/scp/Query4.asp?B2=%AAk%B3W%AAu%AD%B2&FNAME=A004000013 法規沿革]</ref>

;階級識別
在[[國民政府|國民政府]]時期，依據當時陸軍服制條例附表，一級同二級上將領章都是金面三顆三稜角星，而造成軍常服一、二級上將官階不分的情形。大禮服靠袖章辨別，一級為加綴平列金辮雙環。政府[[行憲|行憲]]後，國軍參考美軍服制修改，陆、空军一級上將肩、領章為四枚五角星排列为一行，海军及海军陆战队一级上将为排列为菱形的四枚五角星。
