== 目的和任务 ==
URA成立于1965年，由美国总统顾问委员会和国家科学院的管理。 其目标是在国家利益下建立和运作物理和生物科学研究设施，以扩大知识前沿，促进创新，并促进未来科学家的培养。<ref name="URA">http://www.ura-hq.org/about/index.html {{Webarchive|url=https://web.archive.org/web/20131012100535/http://www.ura-hq.org/about/index.html|date=2013-10-12}} </ref>

在1965年至2007年期间，URA是[[美國能源部|美国能源部]] (以及其前身组织)的主要负责的机构，负责在伊利诺伊州巴达维亚附近建立和运营[[费米国立加速器实验室|费米国家加速实验室]] 。截至2007年1月1日，[http://www.fnal.gov/ 费米实验室] 由费米研究联盟有限责任公司运营，该公司是一家有限责任公司，由两部分组成，大学研究协会（URA）和芝加哥大学。[[费米国立加速器实验室|费米实验室]]是[[兆電子伏特加速器|兆電子伏特加速器]]的所在地，在欧洲核研究组织(CERN)的[[大型強子對撞機|大型强子对撞机(LHC)]] 启动之前，一直是世界上最重要的用于基本粒子物理学研究的能量最高的加速器。

费米实验室未来的战略愿景是通过在伊利诺伊州巴达维亚的费米实验室的深地中微子实验（DUNE），将自身建成世界上中微子研究的中心。该实验设施[https://lbnf.fnal.gov/ 长基线中微子设施(LBNF)]，位于费米实验室和南达科他州里德的[https://www.sanfordlab.org/ 桑福德地下研究设施(SURF)]，是美国第一个主要的此类国际设施。

费米实验室仍然在加速器研究和开发的首要设施，同时也是粒子天体物理学和相关天文科学研究的主要中心，在Sloan数字的天空勘测， [https://www.auger.org/ 皮埃尔·奥格宇宙射线天文台项目]，在明尼苏达州Soudan地下实验室低温暗物质的搜索，以及最近在智利进行的 [https://www.darkenergysurvey.org/ 暗能量调查]中发挥领导作用。

== 组织和管理 ==
该组织在其理事机构，其成员大学校长理事会的授权下行事。组织事务管理委托给[https://www.ura-hq.org/about-ura/governance/ 董事会和主席团成员公司]。华盛顿特区总部办公室负责协调理事会的活动，负责监督和管理市建局的企业以及与联邦政府，行业和学术界的公司关系。

== URA访问学者计划 ==
URA [https://www.ura-hq.org/awards/visiting-scholars-program/ 访问学者计划(计划)] 建立于2007年，旨在支持大学研究协会，公司机构的教师和学生在费米实验室进行长达一年的工作。在URA成员大学的支持下，URA委员会最新的计划延长到2017-2021。
