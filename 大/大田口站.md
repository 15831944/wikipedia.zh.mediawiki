'''大田口站'''（{{jpn|j=大田口駅|hg=おおたぐちえき|rm=Ōtaguchi eki|f=y}}）是一位於[[日本|日本]][[高知縣|高知縣]][[長岡郡|長岡郡]]大豐町黑石、隸屬於[[四國旅客鐵道|四國旅客鐵道]]（JR四國）的[[鐵路車站|鐵路車站]]。車站編號為'''D30'''。本站只停靠普通列車。

==車站結構==
設有單層站舍一座。原本是全座木製結構建築物，<ref>[http://df50shikoku.web.fc2.com/ekisha/dosan/otashike.htm] 原本站舍的面貌。攝於1981年。</ref>但翻新時，將面向馬路的半邊站舍外牆拆掉，然後改以[[混凝土|混凝土]]再重新建成另一半的新外牆，最後變成一座非常獨特的建築物。<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/dosan/ST-OTAGUCHI.jpg |title=存档副本 |accessdate=2012-08-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304092750/http://www.shikoku.org.uk/JR/LINE/dosan/ST-OTAGUCHI.jpg |archivedate=2016-03-04 }} 翻新後的站舍，攝於2008年。</ref>站舍的間格則沒有大的變動，候車室及儲物室仍然保留，但車站入口及玄關位則加闊及加上電話亭。<ref>[http://frederic1no1tabi.web.fc2.com/eki/ootaguchi_sta/ootaguchi_sta.html] 見第2組第10幅圖。</ref>

車站現時為簡易委託站，由車站附近的「吉田商店」負責發售車票。<ref>同上，第3組第10、11幅圖及注腳。</ref>

現時設[[島式月台|島式月台]]一個，提供1線2面的月台配置，有效長度為3卡列車。月台末端向[[土佐穴內站|土佐穴內站]]方向為斜道，並接連[[平交道|平交道]]往站舍。有蓋候車亭亦設於斜道旁，較特別地方為候車亭兩旁加設了混凝土擋風牆，<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/dosan/ST-OTAGUCHI_h-201010b.jpg |title=存档副本 |accessdate=2012-08-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304064044/http://www.shikoku.org.uk/JR/LINE/dosan/ST-OTAGUCHI_h-201010b.jpg |archivedate=2016-03-04 }} 加裝混凝土擋風牆的候車室。</ref>是一個頗貼心的設計。

在月台和站舍之間，除了1號月台路軌外，亦保留舊貨物軌道的路軌，但是站舍對出的側式月台（舊貨物／客運月台）則已經被拆去。相比其他[[土讚線|土讚線]]內的很多本來設有2面3線的月台，往往把離站舍最遠或位於中間的月台棄用大有不同。

現時每天只提供下行8班、上行7班的普通列車服務，上午除首班次前往土佐山田外，其他皆以高知為總站，下行則全以阿波池田為總站。

== 歷史 == 
* 1934年10月28日：開業。
* 1970年10月1日：簡易委託化。
* 1987年4月1日：日本國鐵分割民營化，車站的營運由JR四國繼承。
