'''大同區'''位於[[中華民國|中華民國]][[臺北市|臺北市]]西邊，是臺北市最早發展的區域，[[臺北孔子廟|臺北市孔廟]]、[[大龍峒保安宮|大龍峒保安宮]]均位於此區，涵蓋[[大稻埕|大稻埕]]與[[大龍峒|大龍峒]]兩大區域；[[大稻埕|大稻埕]]曾繁華一時，有許多早期洋房，也是傳統食材（南北貨）、中藥、紡織品及工業用品的集散地，2010年臺北市舉辦[[臺北國際花卉博覽會|臺北國際花卉博覽會]]而制定區花，大同區為[[茶花|茶花]]，[[臺北市政府民政局|臺北市政府民政局]]也特別製作區花版門牌。

== 詞源 ==
本地原名'''[[大浪泵社|巴浪泵社]]'''，為[[臺灣|臺灣]][[平埔族|平埔]][[凱達格蘭族|凱達格蘭族]]的一支系部落。乾隆28年〈西元1763年〉余文儀續修臺灣府志時，載有「-{大浪泵庄}-」。道光24年〈西元1844年〉之契字述及「大隆同四十四坎」，即稱「大隆同」。

[[清|清]][[同治|同治]]年間，該區北部一帶，屬[[淡水廳|淡水廳]][[大加臘堡|大加臘堡]]「[[大隆同莊|大隆同莊]]」，後改稱「大龍峒」。

1909年，臺北廳直轄地方分為三大區，即艋舺、大稻埕、大龍峒三區，大龍峒轄區甚廣，包含今大同區、中山區。

1920年，施行市制，台灣總督府始置大龍峒町<ref>[http://www.dtdo.gov.taipei/ct.asp?xItem=1605454&CtNode=41198&mp=124031 參考政府的官方用詞]</ref>。

1925年，大龍峒地區興建孔廟。

1946年1月，[[臺北市|臺北市]]被中華民國政府接管而改制[[省轄市|省轄市]]時，而臺北市被規劃為十大區，合併日治時代的大龍峒町、蓬萊町、太平町、大橋町、河合町為區，但因轄區內有孔廟及大同路（現大同街），為了紀念孔子天下大同之精神、及取國父世界大同之崇高理念，配合舊地名而命名為「大同區」<ref>曾迺碩、黃得時，《臺北市志：卷一沿革志封域篇》，臺北市文獻委員會，1988年6月，第135頁</ref>。

== 地理位置 ==
*東：以[[台北捷運淡水線|臺北捷運淡水線]]用地東側與[[中山區_(臺北市)|中山區]]為界。
*西：以[[淡水河|淡水河]]中心線與[[新北市|新北市]][[三重區|三重區]]為鄰。
*南：以[[市民大道|市民大道]]、[[延平北路|延平北路]]一段、[[忠孝西路_(臺北市)|忠孝西路]]二段與[[中正區_(臺北市)|中正區]]、[[萬華區|萬華區]]為界。
*北：以[[中山高速公路|中山高速公路]]用地北側及[[基隆河|基隆河]]中心線與[[士林區|士林區]]為界。

== 行政區域 ==
[[File:TaipeiDaTongSubDistrictMap.png|thumb]]
全區共分為25里，各里名稱為：
*蘭州次分區：老師里、國慶里、隆和里、[[國順里|國順里]]、景星里、鄰江里，共6里。
*大龍次分區：至聖里、重慶里、揚雅里、保安里、斯文里、蓬萊里，共6里。（[[大龍峒|大龍峒]]地區）
*延平次分區：大有里、永樂里、南芳里、玉泉里、延平里、[[朝陽里_(臺北市)|朝陽里]]，共6里。（[[大稻埕|大稻埕]]地區）
*建成次分區：民權里、建功里、建泰里、雙連里、光能里、建明-{里}-、星明-{里}-，共7里。

原有文昌里，因戶數過少，在2002年9月1日區里行政區域調整中併入老師里。
