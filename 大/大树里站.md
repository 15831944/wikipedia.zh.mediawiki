'''大树里站'''位于[[甘肃省|甘肃省]][[酒泉市|酒泉市]][[金塔县|金塔县]]航天路西侧<ref>{{cite web |author=赵杨 李秀婷 魏香镜 辛均庆 闫昆仑 |title=运载火箭今与燃料加注设备对接 |url=http://www.people.com.cn/h/2011/0924/c25408-3611141466.html |publisher=南方日报 |accessdate=2019-03-29}}</ref>，是[[清绿铁路|清绿铁路]]一个中间车站，西临[[若水|若水]]，与大树里生活区相隔S315省道，因附近有生命力顽强的胡杨树而得名<ref>{{cite web |author1=马京生 |title=大树里轶闻 |url=http://www.laoziliao.net/rmrb/1990-04-08-6 |accessdate=2019-03-28}}</ref>。大树里站为雷达测量站站部及雷达主站驻地，<ref>{{cite web |title=曾经工作奋斗的酒泉卫星中心回东风的路 |url=http://www.lanzhoushizx.com/lzfz/65768.html |accessdate=2019-03-28}}</ref>部署有大型光电望远镜。<ref>{{cite web |author1=李华泽   李筱梅 |title=测控通信总设计师于志坚：天地有通途 |url=http://scitech.people.com.cn/GB/25509/55912/71407/71437/4869714.html |accessdate=2019-03-28}}</ref>大树里站设置有通往[[东风航天城|东风航天城]]观光的检查站。<ref>{{cite web |author1=新民周刊 |title=神七发射期间酒泉通行证千元难求 |url=http://news.sina.com.cn/c/2008-10-08/104616416357.shtml |accessdate=2019-03-28}}</ref>
