'''大规模监控'''（{{lang-en|Mass surveillance}}，也称为'''大规模监视'''或'''大规模监听'''）是一种针对全体人口或大多数人口的复杂[[监控|监控]]，其目的是掌握民众的行为。<ref>{{cite web |title=Mass Surveillance Technologies |url=https://www.eff.org/issues/mass-surveillance-technologies |publisher=Electronic Frontier Foundation |accessdate=15 September 2013}}</ref>大规模监控通常是由政府或是政府所属的情报机构进行的，此时可强调为“大规模政府监控”；但除了政府，大型企业集团也可能自主进行，或是受到政府支持而进行这样的监控。因为各国的法律和司法系统不同，大规模监控的合法性与需要的法律授权也有很大的不同。

大规模监控常以打击[[恐怖主义|恐怖主义]]、避免社会动乱、保护[[国家安全|国家安全]]、打击[[儿童色情|儿童色情]]以及保护儿童等理由被提出；提出者认为这是达到上述目的必要手段，并使用“[[无所隐瞒论|无所隐瞒论]]”对其进行辩护。相反的，大规模监视也经常因为侵犯隐私权、限制公民的政治权力和自由、违反法律或宪法而被批评。目前的担忧认为，大规模监控可能会带领国家走向“监控国家”和“电子警察国家”，不但公民自由受到侵犯，而且[[政治异见人士|政治异见人士]]可能会被[[政府监听项目列表|棱镜之类的计划]]暗算，这种国家也可称之为[[极权国家|极权国家]]。

== 国家 ==
据{{link-en|隐私国际|Privacy International}}在2007年进行的一项涵盖47个国家的调研显示，监控行为较上一年增加，而隐私保护则有所减弱。综合各因素，有八个国家和地区被评为“普遍存在监控的社会”。这八个国家和地区中，排名从低到高为中国、马来西亚、俄罗斯、新加坡、英国、台湾、泰国和美国。希腊则被认为具有“充分的安全防卫措施防止滥用”监控的社会，是整个调查中得分最高的国家。<ref>{{cite web |url=https://www.privacyinternational.org/reports/surveillance-monitor-2007-international-country-rankings |title=Surveillance Monitor 2007 - International country rankings |publisher=[[Privacy_International|Privacy International]] |date=28 December 2007}}</ref>

全世界许多国家设置了成百上千的摄像头，分布在城市、郊区甚至乡村地区<ref>{{cite web |url= http://blogs.zdnet.com/BTL/?p=9662 |title=Police Surveillance: Go Snoop, Yourself |author=Tom Steinert-Threlkeld |publisher=ZDNet |date=13 August 2008}}</ref><ref name="yougov1">{{cite web |url=http://www.yougov.co.uk/extranets/ygarchives/content/pdf/TEL060101024_3.pdf |title=YouGov / Daily Telegraph Survey Results |year=2006 |publisher=YouGov |accessdate=15 September 2013}}</ref>。2007年9月，[[美国公民自由联盟|美国公民自由联盟]]（ACLU）认为，人们“正处于陷入监视型社会的真正危险中，这完全背离了美国的价值观”，并表示“在将来可能出现的黑暗社会，人们迈出的每一步、做的每一笔交易、进行的每一此通讯都会被记录、汇编并储存起来，并随时准备被当局用来对付我们。”<ref>{{cite web |url=https://www.aclu.org/technology-and-liberty/why-surveillance-society-clock |publisher=American Civil Liberties Union |title= Why a Surveillance Society Clock? |date=4 September 2007 |accessdate=15 September 2013}}</ref>

2013年3月12日，[[无国界记者|无国界记者]]发表了《大规模网络监控的特别报告》。报告中列出了“互联网敌人”清单<ref name=RSFSRS-March2013>[http://surveillance.rsf.org/en/ ''The Enemies of the Internet Special Edition : Surveillance''] {{webarchive|url=https://web.archive.org/web/20130831072750/http://surveillance.rsf.org/en/ |date=2013-08-31 }}, Reporters Without Borders, 12 March 2013</ref>。无国界记者认为，清单中的国家积极对新闻提供商进行强制性监控，严重违反信息自由和人权。在最初的清单中，这些国家分别为[[巴林|巴林]]、[[中国|中国]]、[[伊朗|伊朗]]、[[叙利亚|叙利亚]]和[[越南|越南]]{{NoteTag|需要注意的是，清单发布时棱镜计划还没有曝光，这份清单并没有着重于互联网的大规模监控。随后的清单中则包括了更多国家，包括[[美国|美国]]在内。}}<ref name=RSFSRS-March2013>[http://surveillance.rsf.org/en/ ''The Enemies of the Internet Special Edition : Surveillance''] {{webarchive|url=https://web.archive.org/web/20130831072750/http://surveillance.rsf.org/en/ |date=2013-08-31 }}, Reporters Without Borders, 12 March 2013</ref>。

巴林被无国界记者列入了2013年3月的“互联网敌人”清单。无国界记者认为，清单中的国家积极对新闻提供商进行强制性监控，严重违反信息自由和人权。巴林是过滤、监控互联网程度最高的国家之一。巴林皇室管理整个互联网，以非常先进的工具来监控其国民。异见人士和新闻提供商的网络活动被密切关注，而监控水平依然在持续上升。<ref name=RSFSRS-March2013/>

==={{PRC}}===
{{main|{{le|中国的大规模监控|Mass surveillance in China}}}}
{{See also|1=国内安全保卫|2=社会信用系统|3=金盾工程|4=防火长城|5=中华人民共和国网络审查|6=天网 (中国)}}

无国界记者将中国列入了“互联网敌人”名单。互联网审查和大规模监控在中国普遍存在，而且還動用[[人工智慧|人工智慧]]、[[人臉辨識|人臉辨識]]等技術，而且不少中國本土的科技企業（例如[[海康威視|海康威視]]、[[商湯科技|商湯科技]]等）也為本國的大規模監控計劃充當耳目。<ref>{{cite web |title=人臉識別 + 社會信用系統，一場重塑人類行為的社會實驗？ |url=https://theinitium.com/article/20180525-wsj-introduction/ |website=theinitium.com |language=zh-Hant}}</ref>[[新疆|新疆]]等地更被指是中國政府大規模監控技術的測試場。<ref>{{cite web |title=人民受大數據監控 專家：新疆成露天監獄|url=https://udn.com/news/story/6809/3004249 |website=聯合新聞網}}</ref>
<br>
目前在中国大陆，“社会信用体系”用于失信被执行人方面，向社会公示，在[[最高人民法院|最高人民法院]]相关网站即可查询。<br />
其中，根据《最高人民法院关于限制被执行人高消费的若干规定》、《最高人民法院关于公布失信被执行人名单信息的若干规定》<ref name= rggd>{{cite web |author1=最高人民法院 |title=最高人民法院关于限制被执行人高消费的若干规定 |url=http://www.court.gov.cn/shenpan-xiangqing-1650.html |accessdate=2018-11-25}}</ref><ref name=rggd1>{{cite web |author1=最高人民法院 |title=最高人民法院关于公布失信被执行人名单信息的若干规定 |url=http://zxgk.court.gov.cn/thirdPage.html?href=shouye/html/sfjs/fy_201804210245.html&system=sfjs |accessdate=2018-11-25}}</ref>中规定，“执行人未履行生效法律文书确定的义务，并具有下列情形之一的，人民法院应当将其纳入失信被执行人名单，依法对其进行信用惩戒：（一）有履行能力而拒不履行生效法律文书确定义务的；（二）以伪造证据、暴力、威胁等方法妨碍、抗拒执行的；（三）以虚假诉讼、虚假仲裁或者以隐匿、转移财产等方法规避执行的；（四）违反财产报告制度的；（五）违反限制消费令的；（六）无正当理由拒不履行执行和解协议的”，且具有部分豁免条款，期限一般为三年。<ref name= rggd1/><br />
被纳入失信被执行人名单的，以下行为被禁止：“（一）乘坐交通工具时，选择飞机、列车软卧、轮船二等以上舱位；{{注释|实际执行中包括[[高速动车组列车|高速动车组列车]]全部席位（G字头）、[[动车组列车|动车组列车]]一等座及以上席位（D字头），C字头列车则根据最高设计运行时速区段限制（即以全程中设计时速最高的部分为准），250km/h及以下的归为[[动车组列车|动车组列车]]，250km/h以上的归为[[高速动车组列车|高速动车组列车]]。}}（二）在星级以上宾馆、酒店、夜总会、高尔夫球场等场所进行高消费；（三）购买不动产或者新建、扩建、高档装修房屋；（四）租赁高档写字楼、宾馆、公寓等场所办公；（五）购买非经营必需车辆；（六）旅游、度假；（七）子女就读高收费私立学校；（八）支付高额保费购买保险理财产品；（九）其他非生活和工作必需的高消费行为。”<ref name= rggd/>同时，在“在政府采购、招标投标、行政审批、政府扶持、融资信贷、市场准入、资质认定等方面，对失信被执行人予以信用惩戒。”<ref name= rggd1/><br />
而针对前文中提到的信用评等等，至今在中国大陆仍无任何法律依据。<ref>{{cite book |author1=法律出版社法规中心 |title=新编中华人民共和国法律法规全书 |location=法律出版社  |pages=所有 |edition=第十一版（2018版） |accessdate=2018-11-25}}</ref>

==={{USA}}===
{{main|美国的大规模监控}}
{{See also|1=NSA|2=五眼联盟|3=棱镜计划|4=爱德华·斯诺登|5=美國國家安全局無證監聽爭議}}

美国的大规模监控可以追溯到第一次世界大战的战时监控与审查制度，在第一次和第二次世界大战结束后，美国又继续进行大规模监控。而随着联邦执法机构和情报机构的建立和扩张，如联邦调查局、中央情报局和国家安全局，国家体制化的监控开始用于打压持不同政见者。而在1946年签订的[[英美协定|英美协定]]，则在1955年演化成了如今的[[梯队系统|梯队系统]]。<ref>The Guardian, 02DEC2013 https://www.theguardian.com/world/2013/dec/02/history-of-5-eyes-explainer</ref>梯队系统由五个英语国家组成，也就是所谓的“[[五眼联盟|五眼联盟]]”，主要的工作是截取电子通讯，而针对本国人的监控能力则越来越强。<ref>Global Society Information Watch, 2014 Report https://giswatch.org/en/communications-surveillance/unmasking-five-eyes-global-surveillance-practices</ref>

在2001年[[九一一事件|九一一事件]]发生后，美国针对国内和国际的大规模监控则超越的《[[美国宪法|美国宪法]]》和《[[美国权利法案|权利法案]]》允许的程度。在袭击发生之后的2001年9月14日，美国总统[[乔治·布什|乔治·布什]]宣布进入[[国家紧急状态|国家紧急状态]]，此后，[[奥巴马|奥巴马]]担任总统时，也会每年以行政命令持续维持紧急状态，<ref>Ned Resnikoff, http://www.msnbc.com/all/obama-quietly-extends-post-911-state</ref>时美国处于持续的紧急状态中。此外，美国政府还提出了数个国家安全法案，包括《[[爱国者法案|爱国者法案]]》、《精确法案》和《国外情报监控法案》的修正案中的[[棱镜计划|棱镜计划]]。不同政见者和政治批评家认为，这些法案和行政命令造就了一张巨大的情报监视数据库网络，1950年代美国暗杀异见人士和国家领导人的非法“反谍计划”（COINTELPRO），如今却成为了国家体制，使美国逐渐变成一个[[警察国家|警察国家]]。<ref>Meredith Clark http://www.rollingstone.com/politics/lists/the-new-political-prisoners-leakers-hackers-and-activists-20130301</ref><ref>http://sputniknews.com/us/20160706/1042493824/political-prisoners-suffer-in-america.html</ref><ref>Amy Goodman http://www.democracynow.org/2014/2/25/exclusive_inside_the_army_spy_ring</ref><ref>Amy Goodman http://www.democracynow.org/2012/12/27/the_fbi_vs_occupy_secret_docs</ref>

2013年，[[爱德华·斯诺登|爱德华·斯诺登]]曝光了[[棱镜计划|棱镜计划]]，指出国家安全局运作了庞大的监控网络，该网络由无数个谍报项目组成。国家安全局能够拦截来自数十个国家的数十亿用户的互联网通讯和电话，对互联网骨干网络上的几乎一切数据进行大规模无差别的监听。无国界记者随后因此将美国列入了“互联网敌人”名单。
