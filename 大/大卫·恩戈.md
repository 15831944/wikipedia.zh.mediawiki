'''大卫·恩戈'''（{{lang-fr|'''David N'Gog'''}}，{{bd|1989年|4月1日|catidx=Ngog, David}}）是一名[[喀麥隆|喀麥隆]]裔[[法國|法國]]職業足球員，擔任[[前鋒_(足球)|前鋒]]。他的表兄是法國國腳[[让-阿兰·布姆松|保桑]]<ref name="LFCProfile">{{cite web|title=David N'Gog profile |publisher=利物浦官方網站 |url=http://www.liverpoolfc.tv/team/squad/ngog/ |accessdate=2008-08-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20080806014752/http://www.liverpoolfc.tv/team/squad/ngog/ |archivedate=2008-08-06 }}</ref>。2018年1月加盟[[蘇超|蘇超]]球會[[羅斯郡|羅斯郡]]。

==生平==
===球會===
;早年
尼哥的足球事業始於法國首都球隊[[巴黎聖日耳門|巴黎聖日耳門]]，在2006年6月該隊與尼哥簽下合約，期間尼哥多次入選[[法國國家足球隊|法國]]多個年齡組別的國家足球隊。早在2005年時，尼哥就第一次代表巴黎聖日耳門亮相。不過在那個賽季，他並沒有獲得聯賽上場機會。2006/07年賽季，尼哥也僅8次進入聯賽大名單，4次替補出場，一共也只踢了63分鐘，除了一腳射門外再沒貢獻其他的技術統計。

在2007/08年賽季剛開始時，尼哥得到了大量的上場機會。從2007年9月23日主場對[[波爾多足球俱樂部|波爾多]]至2008年1月13日主場對[[朗斯足球俱乐部|朗斯]]，他連續11場出場，其中8場正選，但尼哥只取得一個入球，助攻一次。也許正是因為這樣，尼哥在後面的比賽中僅僅替補出場3次。整個賽季共出場14次，共656分鐘，一個入球，助攻一次，傳出7次關鍵球。而正是由於在後半個賽季幾乎沒上場機會，使尼哥自己覺得不受重用，於是在合約還剩一年到期時選擇不與球隊續約，巴黎聖日耳門也只能在2008年夏天將他賣出。但在巴黎，尼哥也第一次體驗到了成功的滋味，他隨隊奪得了2008年的[[法國聯賽盃|法國聯賽盃]]冠軍。

;利物浦
2008年7月24日，英格蘭的[[利物浦足球俱樂部|利物浦足球俱樂部]]向巴黎聖日耳門以150萬[[英鎊|英鎊]]收購尼哥，與他簽下4年合約<ref>{{cite web |url=http://www.liverpoolfc.tv/news/drilldown/N160632080724-1149.htm|title=Liverpool Sign French Striker N'Gog |accessdate=2008-07-24 |date=2008-07-24 }}</ref><ref>{{cite web |url=http://msn.foxsports.com/other/story/8371236/Reds-target-French-youngster |title=Reds target French youngster |accessdate=2008-07-23 |date=2008-07-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20081207061005/http://msn.foxsports.com/other/story/8371236/Reds-target-French-youngster |archivedate=2008-12-07 }}</ref>。尼哥首次為利物浦上陣是一場[[友誼賽|友誼賽]]對陣[[維拉利爾足球會|維拉利爾]]，2008年7月30日他在另一場友誼賽對陣[[格拉斯哥流浪|格拉斯哥流浪]]的賽事中取得他加盟後第一個入球。然後，他在利物浦陸續取得入球，他在另一場賽季前的友誼賽，協助球隊以4-1擊敗[[挪威足球超级联赛|挪威]]球隊[[華拉倫加足球會|華拉倫加]]。在[[英超|英超]]中，尼哥首次上陣是在聯賽對陣[[阿士東維拉|阿士東維拉]]的賽事，後備入替[[費蘭度·托利斯|費蘭度·托利斯]]。尼哥在[[歐聯|歐聯]]小組賽對[[PSV燕豪芬|PSV燕豪芬]]的賽事中，射入代表紅軍正式比賽的首個入球，助球隊3-1獲勝<ref>[http://www.skysports.com/story/0,19528,11860_4616613,00.html Sky Sports report]</ref>。2009年2月7日，尼哥首度在[[英超|英超]]聯賽正選上陣對[[樸茨茅夫足球會|樸茨茅夫]]，在2009年3月3日對[[新特蘭足球會|新特蘭]]的聯賽中射入一球，助紅軍2-0獲勝<ref>http://news.bbc.co.uk/sport2/hi/football/eng_prem/7908457.stm BBC Match Report</ref>。

尼哥在2009/10年賽季取得的首個入球是在主場以4-0擊敗[[史篤城足球會|史篤城]]的賽事。這是他在利物浦正式比賽的第4個入球，其中3個是在[[英超聯賽|英超聯賽]]。在隨後的一個月，他接著取得入球，在[[英格蘭聯賽盃|英格蘭聯賽盃]]第三輪賽事，作客1-0擊敗[[列斯聯足球會|列斯聯]]。10月25日，在主場對陣[[曼聯|曼聯]]的關鍵賽事，他後備入替球隊射手[[費蘭度·托利斯|費蘭度·托利斯]]，在第96分鐘射入一球鎖定勝局，協助球隊以2-0擊敗曼聯。

在新領隊[[罗伊·霍奇森|鶴臣]]上任後的首場正式比賽，尼哥獲得正選機會，並且梅開二度協助球隊在[[2010/11賽季歐洲聯賽|歐霸盃外圍賽]]第三圈首回合以2-0擊敗[[拉博特尼基足球俱樂部|拉布尼基]]。<ref>{{cite web

;保頓
2011年8月31日，利物浦官方网站宣布，尼哥正式转会加盟[[博尔顿足球俱乐部|博尔顿]]<ref>[http://www.liverpoolfc.tv/news/latest-news/ngog-completes-bolton-move/ Ngog completes Bolton move] {{webarchive|url=https://web.archive.org/web/20120119072917/http://www.liverpoolfc.tv/news/latest-news/ngog-completes-bolton-move |date=2012-01-19 }}</ref>，他和保頓签订了一份为期三年的合同。

=== 國家隊 ===
尼哥曾入選多個年齡組別的[[法國國家足球隊|法國國家足球隊]]，而且表現出色。現時是[[法國U21|法國U21]]的重要一員，但他現在仍然可選擇為[[喀麥隆國家足球隊|喀麥隆國家足球隊]]效力。<ref>[http://www.thesun.co.uk/sol/homepage/sport/football/2736274/Cam-n-join-us-call-to-Ngog.html]</ref>
