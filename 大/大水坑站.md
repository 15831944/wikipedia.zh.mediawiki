'''大水坑站'''（{{lang-en|Tai Shui Hang Station}}）位於[[香港|香港]][[沙田區|沙田區]][[馬鞍山_(香港市鎮)|馬鞍山]]的[[港鐵|港鐵]][[馬鞍山綫|馬鞍山綫]]車站，已於2004年12月21日啟用。車站取名自鄰近的大水坑村。

==結構==
大水坑站位於馬鞍山路中央地面，佔地約3,760平方米<ref>{{cite web|url=http://www.mos.hk/osdata/ma_rail/page09.htm|title=大水坑站|work=鞍山網絡|publisher=馬鞍山民康促進會|accessdate=2014年4月30日|language=zh-hant|deadurl=yes|archiveurl=https://web.archive.org/web/20090922195930/http://www.mos.hk/osdata/ma_rail/page09.htm|archivedate=2009年9月22日}}</ref>，共分為3層。大堂位於地面，月台設於大堂之上，而[[行人隧道|行人隧道]]則位於大堂之下。大堂與月台層之間及設有[[升降機|升降機]]和[[電動扶梯|扶手電梯]]，方便乘客上落。

===大堂===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Tai Shui Hang Station 2017 08 part6.jpg | caption1 = 大堂
}}
大水坑站大堂內設有7間商店，包括[[便利商店|便利店]]、[[微波爐便當|微波爐便當]]售賣店及物業顧問等，以及[[自動櫃員機|自動櫃員機]]<ref name="shop"/>。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Tai Shui Hang Station 2017 11 part1.jpg | caption1 = 1號月台
| image2 = Tai Shui Hang Station 2017 11 part2.jpg | caption2 = 2號月台
| image3 = Tai Shui Hang Station 2018 06 part2.jpg | caption3 = 月台佈置
}}
車站設有兩個月台，採用[[島式月台|島式月台]]方式排列，月台北面設有預留空間以便馬鞍山綫列車增至8卡行走，預留空間會於沙中線興建時於預留結構上擴建，現時擴建工程已經完成。此站是馬鞍山綫往烏溪沙方向唯一一個月台是彎曲的車站。而此站北面設有[[渡线|渡线]]，方便列車調動或信號故障時才使用。

===出入口===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Tai Shui Hang Station Enterance 201208.jpg | caption1 = A及B出入口位於行人隧道內
| image2 = MTR TSH (8).JPG | caption2 = 行人隧道出入口
}} 
大水坑站的東面是[[富安花園|富安花園]]及大水坑村，西面為[[錦泰苑|錦泰苑]]和[[曉峯灣畔|曉峯灣畔]]。車站只有南面設有[[驗票閘門|閘機]]，通往連接行人隧道的兩個出入口。由於車站位處[[馬鞍山路|馬鞍山路]]的中央分隔帶，因此乘客必須利用行人隧道進出車站。此行人隧道24小時開放予行人使用橫過馬鞍山路。

嚴格來說，大水坑站只得一個出入口，而所謂的A、B出入口只是行人隧道的公眾部分（東、西兩邊的出入口）。

==鄰接車站==
大水坑站是馬鞍山綫的中途站，乘客可在此乘坐往烏溪沙或大圍列車，往返[[九龍|九龍]]及[[新界|新界]]等地。

==利用情況==
{{original research|章節}}
大水坑站位於沙田新市鎮(馬鞍山部分)的南端，進出站內的多為[[錦泰苑|錦泰苑]]、富安花園、大水坑村、曉峯灣畔的住客，也有附近學校的學生使用車站乘搭馬鞍山綫。乘客主要利用馬鞍山綫到[[馬鞍山市中心|馬鞍山市中心]]，或經[[大圍站|大圍站]]轉乘[[東鐵綫|東鐵綫]]到市區。

==歷史==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = KCR TaiShuiHang Platform.jpg | caption1 = 兩鐵合併前的月台（2006年）
}}
===命名及啟用===
大水坑站於2004年12月21日隨馬鞍山鐵路（現今馬鞍山綫）通車啟用。由於車站位於[[富安花園|富安花園]]的對面，因此原來建議的站名為'''富安站'''（{{lang-en|'''Chevalier Garden Station'''}}）<ref>{{cite web|url=http://www.districtcouncils.gov.hk/archive/st_d/chinese/doc/tt_minutes_5_00.doc|title=交通及運輸委員會第五次會議|publisher=沙田區議會|date=2000年9月12日|accessdate=2014年4月30日|language=zh-hant}}</ref><ref>{{cite web|url=http://www.districtcouncils.gov.hk/archive/st_d/chinese/doc/tt_minutes_7_00.doc|title=交通及運輸委員會第六次會議|publisher=沙田區議會|date=2000年11月14日|accessdate=2014年4月30日|language=zh-hant}}</ref>。
