'''大港站'''原是一座[[胶济线|胶济线]]上的铁路车站，位于[[山东省|山东省]][[青岛市|青岛市]][[市北区|市北区]]商河路2号，建于1910-1911年<ref Name=qing>{{cite book |author=袁宾久 |title=《青岛德式建筑》|year=2009 |publisher=中国建筑工业出版社 |location=青岛 |isbn=9787112109050}}</ref>，后于20世纪末停用，目前用作铁路办公用房。使用时为[[四等站_(中国大陆)|四等站]]，邮政编码为266012。

==历史==
关于大港火车站的建造时间，相当多的媒体、资料称其建于1899年，并冠以“[[青岛|青岛]]最早[[火车站|火车站]]”的名号<ref>{{cite web |url=http://news.qingdaonews.com/qingdao/2013-12/25/content_10194299_all.htm|title=探访青岛最早的火车站 曾是出青岛最早起点|accessdate=2013-12-25|work=|publisher=青岛新闻网}}</ref>，实际上，大港火车站现址直至1900年代初尚在海岸线之外。

[[青岛大港|青岛大港]]周边的第一个铁路车站建筑是一座建于1904年的临时站房，其原址在今大港站站房东北方，夹在铁路与[[渤海路_(青岛)|渤海路]]之间<ref>{{cite web |url=http://digital.staatsbibliothek-berlin.de/werkansicht?PPN=PPN770823696&PHYSID=PHYS_0087&DMDID=DMDLOG_0001|title=胶澳租借地发展备忘录 1903-1904年度|work=德意志帝国海军部|publisher=柏林国家图书馆}}</ref>。这座站房仅仅是一栋规模极小的平房，德租时期的资料一般称之为“港口火车站（Hafenbahnhof）”或“铁路停靠站（Eisenbahnhaltestelle）”<ref>{{cite web |url=http://digital.staatsbibliothek-berlin.de/werkansicht?PPN=PPN770823769&PHYSID=PHYS_0072&DMDID=DMDLOG_0001&view=overview-toc|title=胶澳租借地发展备忘录 1904-1905年度|work=德意志帝国海军部|publisher=柏林国家图书馆}}</ref>。

现存的大港火车站（{{lang-de|Bahnhof Großer Hafen}}）建于1910-1911年<ref Name=qing/><ref>{{Cite|author=Schantung-Eisenbahn-Gesellschaft|title=Geschäftsbericht der Schantung-Eisenbahn-Gesellschaft für das Jahr 1911|journal=|date=1911}}</ref>。1914年，胶澳总督府曾计划在大港附近、今[[普集路|普集路]]一带建设一座新的、较[[青岛火车站|青岛火车站]]规模更大的车站，但因[[一战|一战]]与[[青岛战役|青岛战役]]的爆发而告吹。后来的市政当局一直在这一带为新站房预留土地，[[青岛市政府|青岛市政府]]下辖的青岛工务局于1935年颁布的新城市规划也曾计划在此建设新的车站，但最终因[[抗战|抗战]]爆发而未付诸实施。该地块直至1950年代仍为空地。

大港火车站于20世纪末停用。2011年，旧站房被翻修，翻修过程中包括[[瓦片|瓦片]]等原装构件遭到遗弃，亦出现不符合规范的翻修方式。站房临站台一侧曾有一个德制铸花双面表盘，这种表盘原本被安装在[[胶济铁路|胶济铁路]]沿线每一座站房上，如今仅剩大港火车站一副表盘存世<ref Name=qing/>。该表盘于2014年被送往[[济南|济南]][[胶济铁路济南车站旧址|胶济铁路博物馆]]。

自[[女姑口站|女姑口火车站]]于2010年代初拆除后，大港火车站成为青岛市内现存最早的铁路车站站房。
