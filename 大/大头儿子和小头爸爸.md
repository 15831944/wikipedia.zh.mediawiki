《'''大头儿子和小头爸爸'''》是1995年[[中国中央电视台|中国中央电视台]]和[[上海东方电视台|上海东方电视台]]联合制作的一部[[亲子教育|亲子教育]]题材的[[动画片|动画片]]，总集数为156集，依據[[郑春华|郑春华]]所著的同名[[儿童文学|儿童文学]]作品改编。故事通过展现[[中国大陆|中国大陆]][[改革开放|改革开放]]后一个典型三口之家的家庭生活，教育小朋友如何懂事。这部动画以及其主题曲很受欢迎，成为许多中国大陆孩子心中的经典。

《大头儿子于小头爸爸》在[[中国中央电视台第一套节目|中国中央电视台第一套节目]]儿童节目《[[动画城|动画城]]》中首播，也在其他许多电视台播映过。后来有发行[[VCD|VCD]]。这部动画片曾获得第十六届[[大众电视金鹰奖|大众电视金鹰奖]]。动画的新版《新大头儿子和小头爸爸》于2013年11月28日在[[中国中央电视台少儿频道|中国中央电视台少儿频道]]《银河剧场》首播。

== 主要人物 ==
*大头儿子，头特别的大。
*小头爸爸，头特别的小，富有童心。
*围裙妈妈，身体又圆又胖，唠唠叨叨，厨艺高强。

==版权争议==
杭州大头儿子公司[[起诉|起诉]][[央视动画|央视动画]]，提出三个动画形象连同其他费用，一共索赔[[人民幣|人民幣]]156万元。杭州大头儿子公司称，央视动画未经许可且未支付报酬，在2013年将《大头儿子和小头爸爸》人物形象改编为新人物形象，推出《新大头儿子和小头爸爸》，并对新人物形象进行展览、宣传，侵害了大头儿子公司对美术作品享有的[[著作权|著作权]]。参与创造的美术师[[刘泽岱|刘泽岱]]否认授权央视动画。<ref>{{cite web |url= http://edu.zjol.com.cn/system/2015/02/10/020506098.shtml |title=动画人物形象版权归属起纷争 “大头儿子”闹上法庭 |author= |website=zjol.com.cn |language=zh-cn |date=}}</ref>一审判决，央视动画在拍《新大头儿子和小头爸爸》时未向原作者许可，构成[[侵权|侵权]]；但法院并没有判决央视动画停止侵权，法院的解释为《新大头儿子和小头爸爸》获得广泛认知度和很好的社会效果，如果判决停播，会造成社会资源的巨大浪费，因此以提高赔偿额作为停止侵权行为的替代方式，判决央视动画赔偿杭州大头儿子公司人民币126.612万元。一审判决后，央视动画、杭州大头儿子公司均对判决不服并提出了上诉。最终，法院认定双方上诉主张都不成立，驳回上诉，维持原判。<ref>{{cite web |url= http://jiangsu.china.com.cn/html/law/political/4343135_2.html |title=翻拍“大头儿子”引官司 央视动画二审被判侵权 |author= |website=china.com |language=zh-cn |date= |archive-url=https://web.archive.org/web/20160304095841/http://jiangsu.china.com.cn/html/law/political/4343135_2.html |archive-date=2016-03-04}}</ref>
