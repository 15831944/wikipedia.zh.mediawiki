'''大疆创新'''，全名为'''大疆创新科技有限公司'''（英文：Dà-Jiāng Innovations，简称'''DJI'''），是中華人民共和國一家以生产、研发无人飞行器及飞行影像系统为主的公司，企業总部位于[[廣東省|廣東省]][[深圳市|深圳市]]，创始人汪滔。公司致力于为无人机工业、行业用户以及专业航拍应用提供智能飞控产品和解决方案，通过持续的创新，大疆致力于为无人机工业、行业用户以及专业航拍应用提供性能强、体验佳的革命性智能飞控产品和解决方案。目前市场占有率为70%<ref>{{cite news|author1=Fan Feifei|title=DJI, Qianxun partnership to promote the BeiDou system|url=http://www.chinadaily.com.cn/business/tech/2017-04/25/content_29068931.htm|agency=中国日报|date=2017-04-25}}</ref>

== 概况 ==
大疆創新是世界上最大的消費级無人機製造商，其傳奇般的創業故事源自[[香港|香港]]。出生於[[杭州|杭州]]的[[汪滔_(工程師)|汪滔]]，於2003年從[[華東師範大學|華東師範大學]]退學，到[[香港科技大學|香港科技大學]]讀電子與計算機工程學系。汪滔於2006年畢業後，便創立了大疆創新。大疆創新創辦人兼行政總裁汪滔說：「假如我沒有去香港，便不會取得今天的成就。假如我留在內地或去了[[美國|美國]]，也不會創立大疆創新。」

大疆的研發總部位於深圳，在美國、[[德國|德國]]、[[荷蘭|荷蘭]]、[[日本|日本]]、[[北京|北京]]和香港設有辦公室。

為了配合國際業務的急速增長，大疆創新於2015年在[[香港科技園|香港科技園]]成立了研發團隊。汪滔說：「香港具備頂尖的研究型大學、集合大量國際人才、加上世界級的科學園提供完善的科研設備、配合開放的經濟環境，以及鄰近深圳總部和製造基地，是我們建立科研團隊的當然之選。」<ref>{{cite news |language = zh-hans |url = http://www1.investhk.gov.hk/zh-hk/success-stories/seeing-the-unseen/ |title = 本地大學畢業生汪滔在香港實現飛行夢想，並轉化為市值過億的無人機業務 |publisher = investHK |date = 2016-01-02 |accessdate = 2016-04-03 |deadurl = yes |archiveurl = https://web.archive.org/web/20160426090530/http://www1.investhk.gov.hk/zh-hk/success-stories/seeing-the-unseen/ |archivedate = 2016年4月26日 |df =  }}</ref>。隨著影響力逐漸提升，大疆還贊助鼓勵年輕人參與機器人比賽、無人機競速、航拍比賽等，並在無人機本業上也逐漸踏入專業領域。

== 發展歷史 ==
2006年，汪滔在香港科技大學攻讀研究生課程，與此同時，他和一起做畢業課題的兩位同學正式創立大疆，汪滔將他在大學獲得的獎學金的剩餘部分全部拿出來研發生產[[直升機|直升機]]飛行控制系統。公司最初只有五六個人，在深圳的民房辦公。因為辦公環境簡陋，他們無法招到特別優秀的人，就連一起創業的兩位同學在歷經兩年的艱難期後也相繼離開。但汪滔沒有放棄，終於在2008年他研發的第一款較為成熟的直升機飛行控制系統XP3.1面市。

2010年，大疆每月的銷售額能有幾十萬元人民币了，也正是在這一年，香港科技大學方面向汪滔團隊投資了200萬元。原來汪滔只希望能養活一個10到20人的團隊就行了，漸漸地他發現這個行業市場前景很寬闊。

當時[[多軸飛行器|多軸飛行器]]已經興起，大疆在新西蘭的一位代理商告訴汪滔，他每個月售出200多個[[雲台|雲台]]，90%的購買者會將雲台懸掛到多旋翼飛行器上。比較而言，這位代理商每月只能售出幾十個直升機飛行控制系統，說明多旋翼飛行器市場比直升機市場大得多。他很快把在直升機上積累的技術運用到多旋翼飛行器上，並迅速打響口碑，一年後市場占有率達50%以上。

2014年，大疆售出了大約40萬架無人機——許多是其主力機型“大疆精靈”（Phantom）系列——今年的銷售收入有望突破10億美元，相比2014年的5億美元增長一倍。知情人士透露，大疆的利潤已經達到1.2億美元。在2009年和2014年間，大疆的銷售額以每年兩到三倍的速度增長。<ref>{{cite news |language = zh-hans |url = http://newseed.pedaily.cn/201509/201509011321370.shtml |title = 大疆汪滔：9年身價從100元到287億，發生了什麼？ |publisher = 新芽 |date = 2015-09-01 |accessdate = 2016-04-03}}</ref>

2015年12月，深圳市政府直接向大疆批出南山區地鐵5號線[[留仙洞站|留仙洞站]]南面的T501-0078號地段30年地上權；及後，再於2016年12月投得鄰近的T501-0081地段，並合併發展為現時正在興建的新總部-[[大疆天空之城大廈|大疆天空之城大廈]]，由英國建築師[[霍朗明|霍朗明]]設計。

2016年，大疆於租用[[香港|香港]][[銅鑼灣|銅鑼灣]][[TOWER_535|TOWER 535]]面積約10,000平方呎的3層地舖作為香港首家旗艦店，於2016年第3季開業。<ref>[https://www.dronesplayer.com/23465/dji-%E5%93%81%E7%89%8C%E6%97%97%E8%89%A6%E5%BA%97%E5%B0%87%E9%80%B2%E9%A7%90%E9%A6%99%E6%B8%AF%E9%8A%85%E9%91%BC%E7%81%A3-%E6%93%AC%E4%BB%8A%E5%B9%B4%E7%AC%AC-3-%E5%AD%A3%E9%96%8B%E6%A5%AD/ DJI 品牌旗艦店將進駐香港銅鑼灣　擬今年第3季開業 Dronesplayer 2016年5月11日]</ref>

由於存在內建後門等資訊安全疑慮，美國軍方於2017年8月宣布立即停止並棄用所有大疆產品<ref>{{cite web|url=https://www.cnbc.com/2017/08/04/us-army-bans-chinese-made-drone-citing-cyber-vulnerabilities.html|title=US Army bans Chinese-made drone, citing 'cyber vulnerabilities'|first=Jeff|last=Daniels|date=2017-08-04|website=www.cnbc.com}}</ref>，大疆則以升級隱私模式回應，但強調與飛行安全相關的重要數據也會停止傳送。

數據顯示，大疆主導整個無人機市場，在[[美國|美國]]和[[加拿大|加拿大]]擁有近三分之二的份額。除業餘愛好者外，大疆目標客戶還有[[公用事业|公用事业]]公司、執法部門和[[地產商|地產商]]<ref name=":0">{{Cite web|url=https://news.mingpao.com/pns/dailynews/web_tc/article/20171202/s00013/1512151267944|title=大疆否認蒐美數據向華泄密|accessdate=|author=|date=2017-12-02|publisher=明報}}</ref>。

2018年，大疆進行內部整頓改革時發現多個部門有貪腐行為，以致採購零件價格高於一般行情，造成公司損失至少10億元人民幣。該公司已將16名員工移交司法單位調查偵辦，並開除了另外29人。大疆已在公司內成立專業反腐小組，繼續進行後續反貪腐調查<ref name=dronesplayer>{{cite news|url=https://dronesplayer.com/drone-market/dji-%e5%85%a7%e9%83%a8%e7%88%86%e8%b2%aa%e8%85%90%e9%a2%a8%e6%b3%a2%e8%87%b4-10-%e5%84%84%e5%85%83%e6%90%8d%e5%a4%b1-16-%e4%ba%ba%e7%a7%bb%e9%80%81%e5%8f%b8%e6%b3%95%e9%83%a8%e9%96%80/|title=DJI 內部爆貪腐風波損失 10 億元　16 人移送司法部門|publisher= Drones Player|author=陶子亭|date=2019-01-19}}</ref>。

===關係事業===
====機器人大賽====
為培育獎勵社會人才儲備，2015年起大疆擔任主投資方，主辦[[全國大學生機器人大賽|全國大學生機器人大賽]]中的RoboMaster機甲大師聯賽，為一種非破壞性機器人[[遙控|遙控]]對戰比賽，雙方分成步兵機、英雄機、空飛無人機、工程機、哨兵機共五種兵種對戰，並長期與[[鬥魚TV|鬥魚TV]]等直播平台合作在其數碼科技板塊轉播比賽實況。<ref>[https://www.robomaster.com/zh-CN 機甲大師聯賽]</ref>雙方機器人只對外觀長寬高尺寸限制，其餘在比賽規則內自由發揮製作，其五兵種分配涵蓋了目前[[機器人|機器人]]學中的所有學門，尤其哨兵機必須是無人的AI透過影像辨識自主操作，增加了學生們研發的深度。
2018年起另舉辦面向社會大眾和企業團體的RoboMaster人工智能挑戰賽，獲得IEEE机器人和自动化国际会议協辦，每隊參賽的機器人為自由發揮的一或兩台，在場地上進行突破障礙賽和少量非破壞比分戰鬥，但最大規則是機器人必須全智能自主不可有人為介入。<ref>[https://www.robomaster.com/zh-CN/resource/pages/729 ICRA 2018 DJI RoboMaster 人工智能挑战赛]</ref>

====動畫製作====
2017年投資了動畫[[機甲大師|機甲大師]]以比賽中的青年為題材，進行中日合製動畫，將比賽訊息推廣至日本，2016年起也有日本代表隊跨海來參賽。

====瑞典哈苏====
自2015年起，入股[[瑞典|瑞典]][[哈苏|哈苏]]相機，2017年取得多數股權，成為大疆旗下子公司，並於2018年的新產品Mavic 2 Pro使用哈蘇的L1D-20C相機。

===行业应用[[無人機|無人機]]===
[[File:DJI_Mavic_Pro_for_police_use_2018-07-21_163207.jpg|thumb]]
