《'''大南迁'''》，前名為《'''葛藤凹'''》，2012年拍摄的[[唐朝|唐朝]]背景电视剧，总导演[[阎清秀|阎清秀]]，主演[[邹兆龙|邹兆龙]]、[[俞飞鸿|俞飞鸿]]、[[贺刚|贺刚]]、[[符涛|符涛]]、[[李若嘉|李若嘉]]、[[汤镇业|汤镇业]]、[[姚居德|姚居德]]。该剧名字“葛藤凹”原意“[[葛藤|葛藤]]”是一种驱邪保平安的吉祥物，“凹”是山凹，是[[客家文化|客家文化]]。该片是世界客属第25届恳亲大会献礼之作。<ref name="cat " />该剧改编自[[刘善群|刘善群]]小说《葛藤凹》。<ref name="cat " />2014年8月8日[[东南卫视|东南卫视]]及[[陕西卫视|陕西卫视]]首播。

==剧情==
唐朝末年，民间[[黄巢起义|黄巢起义]]，朝中[[监察御史|监察御史]]利中汉因为刚正不阿惩办贪官而被贪官们暗地里整死，落得个满门抄斩的凄凉下场，部分家眷逃过一劫，开始南迁到[[闽|闽]][[粤|粤]]地区的“葛藤凹”，[[宦官|宦官]][[田令孜|田令孜]]得知消息后，打算斩草除根，设计假手他人，一个叫“过猛”的贪官子弟去追杀利中汉的家眷，结果反而被利中汉的妻子救了一命，利中汉的家眷搬迁至葛藤凹后，土著山人[[畲族|畲族]]对他们的到来并不欢迎，几经磨合，才慢慢接受这些陌生的汉人，后成为不分彼此的和谐族群，唐军攻打葛藤凹时，被他们合力击退。

==花絮==
该剧是以客家祖地“[[石壁|石壁]]”为背景的电视剧。<ref name="cat ">{{cite web|url=http://www.fujian.gov.cn/ztzl/lyfj/lyzx/201211/t20121130_549307.htm|title=客家历史剧《葛藤凹》电影版与电视剧版同时上映|accessdate=2012-11-30|publisher=福建省官方网|work=|language=zh-cn}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

该剧原著作者是[[宁化县|宁化县]]客研会名誉会长刘善群。<ref name="cat " /><ref>{{cite web |url=http://news.163.com/12/1206/07/8I1CGEBC00014AED.html |title=宁化客家文化：葛藤绽新枝 更待繁花开 |accessdate= 2012-12-06|publisher=网易 |work= |language =zh-cn}}</ref>

该剧曾在[[浙江|浙江]][[横店影视城|横店影视城]]、三明[[月亮湾|月亮湾]]影视拍摄基地、清流[[七星岩|七星岩]]、宁化[[隆陂水库|隆陂水库]]等地取景。<ref name="cat " /><ref>{{cite web |url=http://et.21cn.com/gundong/etscroll/2012/06/06/12070456.shtml |title=《葛藤凹》剧组到泰宁取景 |accessdate=2012-06-06 |publisher=21CN |work= |language=zh-cn |archive-url=https://web.archive.org/web/20160305015916/http://et.21cn.com/gundong/etscroll/2012/06/06/12070456.shtml |archive-date=2016-03-05 |dead-url=yes }}</ref>

2012年6月17日，该剧举行了关机仪式。<ref>{{cite web |url=http://fj.sina.com.cn/city/sanming/information/s/2012-06-19/0914142.html |title=《葛藤凹》关机仪式暨新闻发布会举行 |accessdate=2012年6月19日 |publisher=新浪网 |work= |language =zh-cn}}</ref>

2012年11月23日，该剧在宁化世界客属文化交流中心客家会堂首映。<ref>{{cite web |url=http://news.163.com/12/1123/08/8GVTS30J00014AED.html |title=世客会献礼电影《葛藤凹》在宁化首映 |accessdate=2012-11-23  |publisher=网易 |work= |language =zh-cn}}</ref>

该剧有100多名三明工贸学校的学生担任群众演员。<ref>{{cite web |url= http://fj.sina.com.cn/city/sanming/information/s/2012-06-08/093170.html|title=三明工贸学校100多名学生《葛藤凹》里扮角色 |accessdate=  2012年6月8日 |publisher=新浪网 |work= |language =zh-cn}}</ref>
