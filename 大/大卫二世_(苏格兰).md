'''大卫二世''' （David II of Scotland）{{BD|1324年|3月5日|1371年|2月22日}}，是14世纪中叶[[苏格兰王国|苏格兰]]国王，也是[[苏格兰君主列表#布鲁斯王朝|布鲁斯王朝]]的第二代和末代君王。他幼年登基，一生波折，最后却由于没有子女，而让王权传到了外甥的[[斯图亚特家族|斯图亚特家族]]。

== 出生 ==
大卫的父亲[[罗伯特·布鲁斯|罗伯特·布鲁斯]]领导苏格兰赢得了[[苏格兰历史#第一次独立战争|第一次独立战争]]的胜利，最终登王位称[[罗伯特·布鲁斯|罗伯特一世]]。但是布鲁斯始终无子，于是苏格兰议会通过法令，如果最后布鲁斯无子而终，王位将由他已故长女的独子[[罗伯特二世_(苏格兰)|罗伯特·斯图亚特]]继承。

1324年，50岁的[[罗伯特·布鲁斯|罗伯特·布鲁斯]]终于有了儿子大卫，大卫也成了王位的第一继承人。1328年，根据[[苏格兰王国|苏格兰王国]]与[[英格兰王国|英格兰王国]]的北安普顿条约，4岁的大卫与英格兰新国王[[爱德华三世_(英格兰)|爱德华三世]]7岁的亲妹妹[[英格兰|英格兰]]的琼公主(Joan of England)成婚，以示两国友好。

== 幼年登基 ==
1329年6月7日，[[罗伯特·布鲁斯|罗伯特·布鲁斯]]去世，5岁的大卫成为苏格兰国王，称“大卫二世”。1331年11月，大卫二世在[[斯昆|斯昆]]行[[加冕|加冕]]礼。

但是与此同时，曾称苏格兰国王的[[约翰·巴里奥|约翰·巴里奥]]之子[[爱德华·巴里奥|爱德华·巴里奥]]获得了英格兰国王[[爱德华三世_(英格兰)|爱德华三世]]的暗中支持，带兵进入苏格兰，自称自己家族才是苏格兰真正的国王，[[苏格兰历史#第二次独立战争|第二次独立战争]]爆发。[[爱德华·巴里奥|爱德华·巴里奥]]在1332年后称苏格兰国王，但是基本不获苏格兰人支持，是英格兰的傀儡政权。苏格兰支持布鲁斯家族的贵族与民众与之展开战斗，但是在[[英格兰王国|英格兰]]军队的协助下，巴里奥方取得绝对军事优势。

1334年5月，年幼的大卫与王后逃到[[法国|法国]]避难，得到法兰西国王[[腓力六世_(法兰西)|腓力六世]]的保护。

== 返回苏格兰 ==
1337年，英法[[百年战争|百年战争]]爆发，英格兰军队不得不转移对于苏格兰战事的注意力。1341年，独立战争的天平已经倒向苏格兰方，17岁的大卫返回苏格兰执政，直接领导对英战争，希望能不负他父亲的威名。他坚守与自己同盟[[腓力六世_(法兰西)|腓力六世]]的约定，连续率军进攻[[英格兰|英格兰]]北方的目标，迫使[[爱德华三世_(英格兰)|爱德华三世]]分兵救援。

1346年，为了解救英军对法国[[加莱|加莱]]的围城，大卫亲率{{Fact|12000人|}}跨越国境远征英格兰。两军在英格兰境内杜汉（Durham）附近的[[内维尔十字之战|内维尔十字]]发生激战，结果苏格兰军队几乎全军覆没。大卫本人面部中两箭，被英格兰军队俘虏。

== 独立战争后 ==
大卫二世在英格兰[[温莎城堡|温莎城堡]]等地被关押了十一年，直到1357年两国达成协议，第二次独立战争结束。大卫二世被放回苏格兰，但是[[苏格兰王国|苏格兰王国]]必须缴纳高额赎金。

大卫二世回国后，发现贫穷的王国根本担负不起这笔债务。由于他本人无子女，他曾试图与[[爱德华三世_(英格兰)|爱德华三世]]协商，用指定英格兰的某王子为苏格兰王位继承人为交换，试图免去支付赎金。但是这项计划遭到苏格兰政府内外的一致反对，最终没能实行，而大卫二世与各方的矛盾也日益加剧。

== 布鲁斯王朝的终结 ==
1369年，大卫与第二任妻子离婚，并准备迎娶第三任妻子。但是1371年，大卫二世便在[[南艾尔郡|南艾尔郡]]登多纳德城堡去世，最终没有留下任何子女。

继承大卫二世的是他父亲[[罗伯特·布鲁斯|罗伯特·布鲁斯]]的长女马乔里公主的独子，即大卫的外甥[[罗伯特二世_(苏格兰)|罗伯特·斯图亚特]]。有趣的是早在大卫出生前，罗伯特就被指定为王位继承人，大卫的出生延迟了他登基的时间达42年之久。

由于大卫是[[罗伯特·布鲁斯|罗伯特·布鲁斯]]的独子，他的去世也象征着[[苏格兰君主列表#布鲁斯王朝|布鲁斯王朝]]的终结。斯图亚特家族由于娶了罗伯特·布鲁斯的女儿，成了此后数百年间苏格兰和后来英格兰的统治者。一百多年后的国王[[詹姆斯五世_(苏格兰)|詹姆斯五世]]临终前曾感叹自己家族的王权是“由一个女人而来”，指的就是大卫二世的同父异母姐姐马乔里公主。
