{{条目消歧义|大武}}
'''大武'''是[[周公|周公]]所創的樂舞，內容描述[[周武王|周武王]]伐[[紂|紂]]{{reftag|name=zhougong}}。其是六代舞之一{{notetag|六代舞共有六種，另外五種為雲門、大章、大韶、大夏和大濩。{{reftag|name=zhougong|{{cite web |url=http://www.epochtimes.com/b5/15/10/3/n4541883.htm |title=周朝的貴族樂舞教育——《六代舞》 |author=皇甫容 |date=2015-10-04 |accessdate=2017-05-03 }}}}}}，用以「享先祖」{{reftag|name=zhouli|{{cite wikisource |title=周禮－春官宗伯 |wslink=周禮/春官宗伯 |wslanguage=zh|quote=奏無射，歌夾鐘，舞《大武》，以享先祖。}}}}。

==結構==
大武的舞蹈結構較為複雜、完整，為藝術水準較高的六段體{{reftag|name=liji}}組合舞。

《禮記》記載大武六段（「六成」）如下：{{reftag|name=liji|{{cite wikisource |title=禮記－樂記 |wslink=禮記/樂記 |wslanguage=zh|quote=且夫《武》，始而北出，再成而滅商。三成而南，四成而南國是疆，五成而分周公左、召公右，六成復綴，以崇天子。}}}}{{reftag|name=jiemi|{{cite web |url=http://liyi.91ddcc.com/c_13218.html |title=揭秘周武王伐纣乐舞：《大武》 |date=2016-09-27 |accessdate=2017-05-03 }}}}

#滅商
#向南方進軍
#把南方收入疆土
#分為左、右兩隊，由輔佐[[周武王|武王]]的[[周公|周公]]、[[召公|召公]]分別率領
#隊伍合併，齊聲讚揚武王的功德
