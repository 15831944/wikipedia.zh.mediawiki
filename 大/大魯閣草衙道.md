'''大魯閣草衙道'''（'''Taroko Park'''）是臺灣[[高雄市|高雄市]][[前鎮區|前鎮區]]的大型購物中心，於2016年5月9日開幕，為[[新光三越|新光三越]]與[[大魯閣集團|大魯閣集團]]所有。占地2.6萬坪(8.6公頃)，建築面積達4.2萬坪(13.9公頃)，全區包括購物商場、[[國賓影城|國賓影城]]、[[健身中心|健身中心]]、[[賽車場|賽車場]]與[[飯店|飯店]]等<ref>{{cite news|author=林孟汝|title=大魯閣草衙道將登場 內部設施搶先看|url=http://www.cna.com.tw/news/afe/201605060199-1.aspx|accessdate=2016-05-09|newspaper=[[中央通訊社|中央通訊社]]|date=2016-05-06}}</ref>。

==歷史==
草衙道為[[大魯閣集團|大魯閣集團]]承租[[高雄捷運|高雄捷運]][[南機廠_(高雄)|南機廠]]附屬商業設施用地，投資興建的綜合型開發案。

全區分為兩期的開發案
*第一期為賽車主題遊樂園、取得[[鈴鹿賽道|鈴鹿賽道]]海外授權的迷你鈴鹿卡丁車賽道和2萬坪的購物商場，於2014年6月15日動土，2016年5月9日正式開幕。
*第二期將建置城市型休閒飯店。預計2018年動工，2025年完工。<ref name=>[http://tw.sports.yahoo.com/news/%E6%94%9C%E6%89%8B%E6%97%A5%E5%95%86-%E5%A4%A7%E9%AD%AF%E9%96%A3-%E9%AB%98%E9%9B%84%E8%93%8B%E6%B1%BD%E8%BB%8A%E4%B8%BB%E9%A1%8C%E6%A8%82%E5%9C%92-213000151.html 《攜手日商 大魯閣 高雄蓋汽車主題樂園》，中時電子報]</ref><ref>{{Cite web |url= http://mtbu.kcg.gov.tw/cht/news_content.php?sn=2012102501 |title= 捷運南機廠開發案再現高雄多元投資亮點 |date= 2012-10-25 |language= zh-tw |deadurl= yes |archiveurl= https://web.archive.org/web/20140314085630/http://mtbu.kcg.gov.tw/cht/news_content.php?sn=2012102501 |archivedate= 2014-03-14 }}</ref><ref>{{Cite web |url= http://mtbu.kcg.gov.tw/cht/news_content.php?sn=2013030201 |title= 「大魯閣草衙道開發案日本發表會」再現高雄多元投資亮點 |date= 2013-03-02 |language= zh-tw |deadurl= yes |archiveurl= https://web.archive.org/web/20130803041326/http://mtbu.kcg.gov.tw/cht/news_content.php?sn=2013030201 |archivedate= 2013-08-03 }}</ref><ref>{{Cite web |url= http://www.trk.com.tw/partners/index/module/www |title= 大魯閣草衙道 |language= zh-tw }}</ref><ref>{{Cite web |url= http://money.udn.com/money/story/5649/1405273-%E5%A4%A7%E9%AD%AF%E9%96%A3%E8%B6%85%E9%85%B7...%E8%B3%BD%E8%BB%8A%E9%81%93%E6%90%AC%E9%80%B2%E5%95%86%E5%A0%B4 |title= 大魯閣超酷...賽車道搬進商場 |language= zh-tw |deadurl= yes |archiveurl= https://web.archive.org/web/20160513203929/http://money.udn.com/money/story/5649/1405273-%E5%A4%A7%E9%AD%AF%E9%96%A3%E8%B6%85%E9%85%B7...%E8%B3%BD%E8%BB%8A%E9%81%93%E6%90%AC%E9%80%B2%E5%95%86%E5%A0%B4 |archivedate= 2016-05-13 }}</ref><ref>{{Cite web |url= http://udn.com/news/story/7241/1564584-%E5%A4%A7%E9%AD%AF%E9%96%A3%E8%8D%89%E8%A1%99%E9%81%93-5%E6%9C%889%E6%97%A5%E9%96%8B%E5%B9%95 |title= 大魯閣草衙道 5月9日開幕 |language= zh-tw }}</ref>
