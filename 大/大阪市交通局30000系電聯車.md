'''30000系'''是供[[大阪市營地下鐵|大阪市營地下鐵]]使用的[[通勤型電車|通勤型電車]]。分別於2009年3月18日及2011年12月10日，開始在[[谷町線_(大阪市營地下鐵)|谷町線]]及[[御堂筋線|御堂筋線]]投入服務。<ref>{{cite web |url=http://railf.jp/news/2009/03/22/195000.html |title=大阪市交通局30000系，営業運転を開始 |accessdate=2009年4月28日 |work= |publisher=Japan Railfan Magazine |date=2009年3月18日 }} {{ja icon}}</ref>

== 概要 ==
這款城軌車由[[近畿車輛|近畿車輛]]負責製造，在2009年，為了令谷町線加快空調化及取締老化的[[大阪市交通局30系電聯車|30系]]及[[大阪市交通局20系電聯車|22系]]電車，[[大阪市交通局|大阪市交通局]]開始投入到谷町線営業。而在2011年12月10日，12列10節30000系車組現已在御堂筋線開始投入服務，取代了原有的[[大阪市交通局10系電聯車|10系]]電車。<ref name="dj327">{{cite journal| title =大阪市交通局　30000系（地下鉄御堂筋線）| journal = Tetsudō Daiya Jōhō Magazine | volume = 40| issue = 327| page = p.72| publisher = 交通新聞| location = 日本| date = 2011年7月| url = | doi = | id = | accessdate = }}</ref><ref name="osaka20111206">{{cite web|url=http://www.kotsu.city.osaka.jp/news/news/h23/30000kei-debut.html|title=12月10日、御堂筋線に新型車両「30000系」がデビューします|date=2011年12月|work=新聞稿|publisher=大阪市交通局|location=日本|language=ja|archiveurl=|archivedate=|accessdate=2011年12月6日}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

另外，列車內的前方到站廣播可以播放4種語言，依順序播放[[日語|日語]]、[[英語|英語]]、中國[[國語|國語]]及[[韓語|韓語]]等等。
