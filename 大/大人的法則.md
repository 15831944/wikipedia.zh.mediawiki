「'''成人法則'''」（{{lang-ja|おとなの掟}}）是[[日本|日本]]的Doughnuts Hole（{{lang|ja|ドーナツ・ホール}}）的樂曲。2017年2月7日由{{lang|en|Ariola Japan}}以[[數位音樂下載|數位音樂下載]]發售。

== 概要 ==
這首歌曲是[[TBS電視台|TBS電視台]]連續劇《[[四重奏_(日本電視劇)|四重奏]]》的主題歌，由[[松隆子|松隆子]]、[[滿島光|滿島光]]、[[高橋一生|高橋一生]]、[[松田龍平|松田龍平]]四人組成的Doughnuts Hole負責演唱<ref name="dounutshole">{{Cite web |date=2017-01-14 |url=http://www.tbs.co.jp/quartet2017/news/ |title=お知らせ |work=火曜ドラマ『カルテット』 |publisher=TBSテレビ |accessdate=2017-02-23}}</ref>。此曲由[[椎名林檎|椎名林檎]]作詞及作曲，編曲則由椎名及{{link-ja|齋藤貓|斎藤ネコ}}共同完成，演奏成員為{{link-ja|ヒイズミマサユ機}}（[[钢琴|钢琴]]）、"{{lang|ja|斎藤ネコカルテット}}"、田村優弥（[[大号|大号]]）等人<ref name="dounutshole"/>。

這首歌曲登上了[[日本熱門100金曲榜|日本熱門100金曲榜]]單曲排行榜第4位，並於[[iTunes|iTunes]]每周歌曲排行榜（2017年2月6日〜2月12日）獲得第1位。另外，這首歌曲也在[[RecoChoku|RecoChoku]]、[[mora|mora]]、[[music.jp|music.jp]]、[[Google_Play_Music|Google Play Music]]、[[dwango|dwango]]、[[mu-mo|mu-mo]]、[[oricon|oricon]]、[[LINE_(應用程式))|LINE 音樂]]、[[亚马逊日本|Amazon J-POP]]、[[蘋果公司|蘋果音樂 J-POP]]等各主要數位音樂下載網站獲得第一位，于開始提供下載后一星期下載次數超過10万<ref name="musicman">{{Cite web |date=2017-02-15|url=http://www.musicman-net.com/artist/65445.html|title=松たか子らによるDoughnuts Hole「おとなの掟」がiTunes週間ランキング他で1位を獲得 |work= |publisher=MUSICMAN-NET|accessdate=2017-02-23}}</ref>。在其後的幾個星期，這首歌曲仍然停留在各排行榜首十名以内。3月21日連續劇大結局放映後，這歌再次登上第一位，並成爲當時2017年度最多下載次數的單曲<ref>{{Cite web |date=2017-03-29|url=http://top.tsite.jp/entertainment/j-pop/i/34829590/index|title=ドラマ「カルテット」限定ユニットDoughnuts Hole（松たか子、満島ひかり、高橋一生、松田龍平）「おとなの掟」、大ヒットの理由 |work=T-SITE NEWS|publisher=[[TSUTAYA|TSUTAYA]] |accessdate=2017-04-02}}</ref>。

==背景及發佈==
[[File:Shiina_Ringo_2016.jpg|180px]]]]
[[TBS電視台|TBS]]電視劇《[[四重奏_(日本電視劇)|四重奏]]》の制作にあたり、連續劇的製作者を務める佐野亜裕美は音楽家の[[椎名林檎|椎名林檎]]に主題歌の制作を依頼した。佐野は当初からドラマ出演者である[[松たか子|松たか子]]、[[満島ひかり|満島ひかり]]、[[高橋一生|高橋一生]]、[[松田龍平|松田龍平]]の役者4人で主題歌を歌う構想を持っていたため、椎名にも役者4人で歌う曲として楽曲制作のオファーをしたという。更に佐野はオファーにあたり、椎名にドラマの台本を渡した上で、「ドラマの世界の続きではあるけれど、ある種、それまで履いていた靴を脱ぎ捨て、別の世界にいっても構いません。素敵な曲であれば、ジャンルは問いません」と述べ、自由に制作してほしい旨を伝えた<ref name="quartet-producer-interview">{{Cite web |url=https://www.buzzfeed.com/tatsunoritokushige/quartettalk?utm_term=.ya25xDbm4e#.mr8rMd3YWJ|title=椎名林檎手がけた「カルテット」主題歌に込められた思い|pages=|publisher=BuzzFeed|author=徳重辰典 |work= |date=2017-02-12 |accessdate=2017-02-15}}</ref>。一方で、佐野は2017年現在の日本のドラマ界全体の風潮として、白黒はっきりとしたわかりやすさ、また、こう見せたいと制作者側が決めるドラマばかりではつまらないとも感じており、『カルテット』ではドラマの多様性を伝えたいと考えていた<ref name="quartet-producer-interview" />。佐野は椎名へはそういった自身の気持ちは直接は伝えていなかったものの、完成された楽曲の歌詞には"ああ／白黒つけるのは恐ろしい／切実に生きればこそ／そう人生は長い／世界は広い／自由を手にした僕らはグレー"との一節が含まれており、多様性が肯定された詩になっていたという<ref name="quartet-producer-interview" />。また椎名にとっても、かねてよりことあるごとに楽曲提供したい相手として名前を挙げてきた中の一人である松への、初めての提供曲となった<ref name="nataliepp5">{{Cite interview|subject= 椎名林檎 |date= 2017-05-29 |interviewer= 内田正樹 |http://natalie.mu/music/pp/sheenaringo04/page/5|title= 「椎名林檎と彼奴等がゆく 百鬼夜行2015」インタビュー (5/5) |program= [[ナタリー_(ニュースサイト)|音楽ナタリー]] |accessdate= 2017-05-31 }}</ref>。

這首樂曲由4人組Doughnuts Holeによって歌われている。彼らはTBS系列のテレビドラマ《四重奏》のためにドラマの出演者である松たか子、満島ひかり、高橋一生、松田龍平の4人によって期間限定で結成されたグループであり<ref name="dounutshole" />、ドラマ本編で4人が組む[[弦楽四重奏|弦楽四重奏]]カルテットのグループ名と同名である<ref>{{Cite web |url=http://www.tbs.co.jp/quartet2017/chart/ |title=人物相関図 |work=火曜ドラマ『カルテット』 |publisher=TBSテレビ |accessdate=2017-02-23}}</ref>。4人はこれまで何らかの形で音楽に携わった経験があり、松は女優業以外に2014年の「[[レット・イット・ゴー_(ディズニーの曲)|レット・イット・ゴー]]」、加えてシングル21枚、アルバム9枚をリリースしている歌手であり、満島は過去[[Folder_(音楽グループ)|Folder]]、[[Folder5|Folder5]]として音楽活動を行っていた。更に高橋は過去『[[スタジオパークからこんにちは|スタジオパークからこんにちは]]』出演時に[[THE_BOOM|THE BOOM]]の「[[JAPANESKA|からたち野道]]」を披露した経験があり、松田も[[RIP_SLYME|RIP SLYME]]のメンバー[[PES|PES]]のソロアルバムにボーカルとして参加している<ref name="excite" />。楽曲は2017年2月7日に[[アリオラジャパン|アリオラジャパン]]より[[デジタル・ダウンロード|デジタル・ダウンロード]]でシングル・リリースされた<ref name="japan-release-date" />。

ドラマ終了後に[[銀座|銀座]]の複合施設「[[GINZA_SIX|GINZA SIX]]」のCM音楽制作の依頼を受けた椎名は、CMのキャスティングにカルテットの主演4人を推薦した。キャスティングはならなかったものの、[[テーマソング|テーマソング]]となった「[[目抜き通り_(椎名林檎とトータス松本の曲)|目抜き通り]]」の歌詞は“カルテット〜その後〜”という体で書いている<ref name="nataliepp5"/>。

== 制作及錄音、樂曲結構 ==
[[音樂製作人|音樂製作人]]の[[佐橋佳幸|佐橋佳幸]]からの請求を、「[[アップル・レコード|アップル・レコード]]時代の[[ポール・マッカートニー|ポール・マッカートニー]]的な楽曲が求められている」と解釈した椎名は、それに沿って曲を書いた<ref name="nataliepp5"/>。作曲の際には、主演の4人の声をしっかり認識してもらうために楽器の音は最小限まで減らすなど、まず構造的なアプローチから入った<ref name="nataliepp5"/>。歌詞については、第一話の台本はもらったもののそれは読まず、公式サイトにあった「大人の恋はやっかいだ」という言葉と[[配役|配役]]を目にした印象のみで仕上げた<ref name="thetvsong">{{cite web|url= https://thetv.jp/feature/drama-academy/archives/song/|title= 【特集】第92回2017年1‐3月クール ドラマアカデミー賞 結果発表 ドラマソング賞 おとなの掟／Doughnuts Hole|date= 2017-05-10|accessdate= 2017-05-31|publisher= [[ザテレビジョン|ザテレビジョン]], [[KADOKAWA|KADOKAWA]]|deadurl= yes|archiveurl= https://web.archive.org/web/20170811105051/https://thetv.jp/feature/drama-academy/archives/song/|archivedate= 2017年8月11日|df= }}</ref>。レコーディングにはソングライターの椎名に加え、[[ピアノ|ピアノ]]演奏で[[ヒイズミマサユ機|ヒイズミマサユ機]]、斎藤ネコカルテットより第1ヴァイオリンで[[斎藤ネコ|斎藤ネコ]]、第2ヴァイオリンでグレート栄田、[[ヴィオラ|ヴィオラ]]で山田雄司、 [[チェロ|チェロ]]で[[モルゴーア・クァルテット#メンバー|藤森亮一]]、[[チューバ|チューバ]]で田村優弥、[[MIDI|Midi]]で[[井上雨迩|井上雨迩]]が参加している<ref name="quartet-producer-interview" /><ref name="credits" />。また「おとなの掟」の楽曲構成について、[[スポーツ報知|スポーツ報知]]は「松、満島が切ないメロディーを歌いはじめ、しだいに高橋、松田の声が重なっていくような構成」とコメントしている<ref name="hochi" />。

当初椎名はグループの4人が歌唱に加えて演奏も行うものとの早とちりして[[弦楽四重奏|4人分の弦楽]]パートも用意したと冗談めかしてコメントしていたが<ref name="credits" />、実際は[[ミュージックビデオ|ミュージックビデオ]]で4人が[[リップシンク|リップシンク]]しながら[[弦楽器|弦楽器]]もそれぞれ当て振りすることを想定して書いたものだった<ref name="thetvsong"/>。

== 評價 ==
リアルサウンドの若田悠希は、「おとなの掟」における椎名の仕事について「知識や経験をいくら積んだとしても〈正解不正解〉を正しく選べるようにはなれない--そんな大人の姿がユーモラスかつシリアスに描かれる『カルテット』の主題を、見事に掬いとって「おとなの掟」を書き上げた」とドラマ自体のテーマを掬い取っている点を評し、更に「様々な設定や背景が複雑に絡み合う『カルテット』を、さらに深く掘り下げて楽しむためのヒントを与えてくれる主題歌である」とコメントしている<ref name="realsound-review" />。

「おとなの掟」は、2017年2月20日付の[[日本熱門100金曲榜|日本熱門100金曲榜]]総合シングルチャートに第4位でチャート・デビューした<ref name="bbjhot100-0220" />。また配信開始1週間で10万ユニット以上のデジタル・セールスをあげており<ref name="ro69-digital-sales" />、Japan Hot 100を構成する各部門ではダウンロード1位、ラジオ35位、Twitter76位、ストリーミング11位となっている。[[Billboard_JAPAN|ビルボードジャパン]]はチャート成績について「まだまだ余力を残しており、しばらくチャートを賑わせそうだ」と評している<ref name="bbjhot100-0220" />。

這首歌曲獲得[[第92回日劇學院賞|第92回日劇學院賞]]的主題曲及連續劇歌曲奬<ref>[https://thetv.jp/feature/drama-academy/archives/ 【特集】第92回ドラマアカデミー賞 結果発表 | ザテレビジョン]、[[ザテレビジョン|ザテレビジョン]]、[[KADOKAWA|KADOKAWA]]、2017年5月10日閲覧。</ref>。

== 音樂影片（MV） ==
《大人的法則》に付随する楽曲の[[ミュージック・ビデオ|ミュージック・ビデオ]]は2017年1月18日に[[YouTube|YouTube]]上でショート・バージョンが公開され、楽曲を歌唱している松、満島、高橋、松田の4人が出演している。ビデオは、松と満島の2人が[[ジャズバー|ジャズバー]]で歌うシンガーのような衣装を纏い、更に高橋と松田は"シンガー"の2人を撮影するカメラマンと監督役を演じ、「ミュージック・ビデオの中でミュージック・ビデオを撮影している」という内容となっている。ねとらぼエンタは、このビデオの内容について「ユニークな構造」と指摘し、更に「曲調に合わせたムーディーな仕上がりの動画」とコメントしている<ref>{{Cite web |url=http://nlab.itmedia.co.jp/nl/articles/1701/18/news131.html|title=松たか子＆満島ひかりが色っぽくハモる ドラマ「カルテット」、椎名林檎による主題歌「おとなの掟」のMVが公開|pages=|publisher=アイティメディア|author=深戸進路 |work=ねとらぼ |date=2017-01-18 |accessdate=2017-02-15}}</ref>。
