'''大象露西'''（{{lang-en|'''Lucy the Elephant'''}}）是一座六層高的[[大象|大象]]外形的新穎建築，用木材和錫板構成，由[[詹姆斯·V·拉夫爾提|詹姆斯·V·拉夫爾提]]在1881年於新澤西州[[大西洋城|大西洋城]]以南兩英里（3.2公里）的{{link-en|馬蓋特城 (新澤西州)|Margate City, New Jersey|馬蓋特城}}上建成的。露西最初被命名為「大象雜貨店」（Elephant Bazaar），其宗旨是促進房地產銷售並吸引遊客。今天，露西是美國現存最古老的{{link-en|路邊景點|Roadside attraction|路邊旅遊景點}}。<ref>{{Cite web |url = https://www.wnyc.org/story/elephants-never-forget-and-at-6-stories-tall-this-ones-unforgettable/ |title = Elephants Never Forget — And At 6 Stories Tall, This One's Unforgettable |author = Emma Jacobs |date = 2015-07-12 |publisher = wnyc.org |language = en}}</ref>

== 歷史：1881年-1899年 ==
[[File:Lucy-USpatent268503_1882.jpg|thumb]]
1881年，美國專利局授予[[詹姆斯·V·拉夫爾提|詹姆斯·V·拉夫爾提]]一個專利權，給他在17年期間製造、使用或出售動物外形的建築物。拉夫爾提在現稱為{{link-en|馬蓋特城 (新澤西州)|Margate City, New Jersey|馬蓋特城}}的南大西洋城中資助了的第一座大象外形建築的設計和建造。他聘請費城建築師William Free進行設計。露西是以[[玲玲馬戲團|拿姆貝理馬戲團]]的一隻著名大象「[[大象金寶|大象金寶]]」而塑造出來的，建造費用為25,000至38,000美元。<ref name="Jersey Icons">{{Cite web |url = https://www.northjersey.com/story/entertainment/2017/08/13/jersey-icons-lucy-elephant/525958001/ |title = Jersey Icons: Lucy the Elephant |author = John C. Ensslin |date = 2017-08-13 |publisher = North Jersey |language = en}}</ref>

最初名為「大象雜貨店」（Elephant Bazaar），建築高65英尺（19.7米）、長60英尺（18.3米）、寬18英尺（5.5米）和重約90噸。它被列為美國{{link-en|美國最高雕像列表|List of the tallest statues in the United States|第11高的雕像}}。用了近100萬件木材，以及需要200小桶的釘子、4噸的螺栓和鐵條來建造露西；12,000平方英尺的錫覆蓋著外部。整個結構有22個窗戶。

=== 早期使用和出售 ===
最初，拉夫爾提帶著潛在的房地產顧客從露西的{{link-en|象轎|Howdah}}上觀看土地。象轎提供了{{link-en|馬蓋特城 (新澤西州)|Margate City, New Jersey|馬蓋特城}}、[[大西洋城|大西洋城]]天際線、海灘和[[大西洋|大西洋]]的獨特景緻，並為現代遊客在參觀期間提供作觀景台。

該建築於1887年賣給了費城的Anton Gertzen，保留在他的家族中直到1970年為止。據報導，Anton的媳婦Sophia Gertzen於1902年給該建築稱為「大象露西」。露西的頭部外形是有[[亞洲象|亞洲象]]和雄性大象有長牙的特徵。最初，這頭大像被認為是雄性，但最後一般都認為是雌性。

== 歷史：1900年-1999年 ==
[[File:NORTH_ELEVATION_-_Margate_Elephant,_Atlantic_Avenue_and_Decatur_Street,_Margate_City,_Atlantic_County,_NJ_HABS_NJ,1-MARGCI,1-3.tif|183x183px]]
到20世紀上半葉，露西被當作餐廳、商務辦公室、小屋和酒館（最後因為[[禁酒令|禁酒令]]而關閉）。這座建築物在許多紀念品明信片上有被描繪，通常被稱為「大西洋城的大象酒店」。（實際的酒店位於附近的建築物，而不是在大象裡面。）

== 歷史：2000年-現在 ==
[[File:Lucy_the_Margate_Elephant_HABS_NJ,1-MARGCI,1-7.jpg|[[Historic American Buildings Survey]] image|179x179px|right|frameless]]
每年的7月20日，這座建築的生日都會舉辦兒童遊戲和用好多喇叭吹奏聲來慶祝。

巴黎大象雅爾丹（Jardin the Paris Elephant）是一座真實大小的大象建築，靈感來自「{{link-en|大象巨人|Elephantine Colossus}}」（露西的巨大版本，由拉夫爾提於1885年在[[康尼島|康尼島]]建成的），它出現在2001年的電影《[[情陷紅磨坊|情陷紅磨坊]]》中當作[[妮歌·潔曼|妮歌·潔曼]]角色的閨房。
