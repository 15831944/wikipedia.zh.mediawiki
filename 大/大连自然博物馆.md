'''大连自然博物馆'''是位于[[辽宁省|辽宁省]][[大连市|大连市]]的一座集[[地质|地质]]、[[古生物|古生物]]、[[动物|动物]]、[[植物|植物]][[标本|标本]]收藏、研究、展示于一体的综合性[[自然科学|自然科学]]博物馆，其前身始建于1907年。新馆为现代欧式建筑，位于[[黑石礁|黑石礁]]海滨，建筑面积1.5万平方米，展览面积1万平方米，是国内唯一拥有27万平方米海域的博物馆。

==馆史==
大连自然博物馆始建于1907年，前身是[[日本|日本]]占领[[中国东北|中国东北]]以后，由日本[[南滿洲鐵道株式會社|南滿洲鐵道株式會社]]创办的地质调查所。1945年抗日战争结束后，8月23日由中国长春铁路公司接管，易名为「东北地方志博物馆」；中长铁路科研所委托苏联地质专家叶果洛夫担任馆长，并对原有的陈列进行修整。1959年，在[[抗日战争|抗战]]胜利14周年[[第二次世界大戰對日戰爭勝利紀念日|纪念日]]（9月3日），正式定名为「大连自然博物馆」。1995年为实施科教兴市战略，保护自然文化遗产，大连市委、市政府决定移址[[黑石礁|黑石礁]]海滨建新馆。1998年10月，新馆建成并对外开放。<ref>{{Cite web |url= http://www.dlnm.org/organization.html |title= 馆史回顾 |author= 大连自然博物馆 }}</ref>

== 展品 ==
大连自然博物馆有各种标本近20万件，珍贵标本6千余件。馆藏特点是海洋生物标本和「热河生物群」[[化石|化石]]标本，其中海兽标本20余种，其种类和数量在国内自然史博物馆中是最多的，其中重达66.7吨的[[黑露脊鲸|黑露脊鲸]]标本，在亚洲属罕见。「热河生物群」化石标本在国内是种类最多最有特点的，其中一窝[[鹦鹉嘴龙|鹦鹉嘴龙]]化石是世界上迄今为止惟一的、数量最多、保存最完整的的国宝级化石标本。<ref>{{Cite web |url= http://www.dlnm.org/indxe1.html |title= 馆内介绍 |author= 大连自然博物馆 }}</ref>

== 门票 ==
80人以下以下的小型团体，无需预约，只需携带身份证件，便可直接参观
80人以上大型团体需提前1-10天进行预约

每周周一闭馆，周二至周日正常开馆，节假日不休息。
