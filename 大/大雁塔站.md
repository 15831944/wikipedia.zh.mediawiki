'''大雁塔站'''位于[[西安市|西安市]][[雁塔区|雁塔区]]，是[[西安地铁|西安地铁]][[西安地铁3号线|3号线]]的一个车站，因位于[[大雁塔|大雁塔]]附近而得名。同时也为[[西安地铁4号线|4号线]]的换乘站。

== 位置 ==
大雁塔站位于[[西影路|西影路]]、[[小寨东路|小寨东路]]与[[雁塔北路|雁塔北路]]交叉处，[[大雁塔北广场|大雁塔北广场]]北侧。其中3号线车站位于地下2层，沿小寨东路和西影路地下成东西走向；4号线车站位于地下3层，沿雁塔北路地下成南北走向<ref>{{cite news|url=http://www.sxdaily.com.cn/n/2012/1215/c46-5035290-1.html|title=西安地铁三号线大雁塔站交通导改示意图|publisher=陕西传媒网|date=2012-12-15|accessdate=2017-10-14|archive-url=https://web.archive.org/web/20171014183448/http://www.sxdaily.com.cn/n/2012/1215/c46-5035290-1.html|archive-date=2017-10-14|dead-url=yes}}</ref>。

===站厅===
[[File:Concourse_of_Dayanta_Station_(20171002120517).jpg|thumb]]
大雁塔站的站厅位于小寨东路下方的地下一层，付费区在站厅北面。

===站台===
大雁塔站3号线站台位于小寨东路地底，属于双柱三跨岛式站台，全部站台均设有屏蔽门。

== 出口 ==
大雁塔站一共有4个出口，其中B、C两出口通往大雁塔北广场。

== 事件 ==
===文化墙相关争议===
[[File:Concourse_mural_at_Dayanta_Station_(20171002101430).jpg|thumb]]
该站C出口处的文化墙上的[[壁画|壁画]]名为《[[天竺|天竺]]佛光》，其中同时出现[[玄奘|玄奘]]、[[大雁塔|大雁塔]]与[[印度|印度]][[泰姬陵|泰姬陵]]，表达[[中印关系|中印两国的古今交流]]。2017年2月12日，有网民提出质疑，认为玄奘西天取经的时期与泰姬陵的建设时期相差千年，可能会引发民众误解“玄奘去一千年后的[[莫卧儿帝国|莫卧儿帝国]]取[[古兰经|古兰经]]”，而且这种状况有损西安的文化历史名城形象。作为回应，地铁车站文化墙设计团队成员之一的[[西安美术学院|西安美术学院]][[教授|教授]]表示该壁画体现的是现代装饰壁画的视觉艺术特征，强调时空观的交错，并非传统壁画在某一时间点讲述某一故事，而且附近的[[陕西历史博物馆|陕西历史博物馆]]已经将玄奘取经讲述的很清楚，在站内无需重复解释。西安地铁工作人员则称，地铁三号线每面文化墙左下方都有文化墙名称及内容解释，该文化墙的设计和最终展示虽未征求市民意见，但都已经过专家论证<ref>{{cite news|url=http://ehsb.hsw.cn/shtml/hsb/20170214/631550.shtml?xyz|title=玄奘去过千年后的泰姬陵？|publisher=[[华商报_(陕西)|华商报]]|date=2017-02-14|accessdate=2017-10-14}}</ref>。
