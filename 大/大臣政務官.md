'''大臣政務官'''（{{jpn|j=大臣政務官|hg=だいじんせいむかん}}、{{lang|en|Parliamentary Secretary}}, Vice-Minister）是[[日本|日本]][[內閣府|內閣府]]、[[復興廳|復興廳]]與各省設置的[[官職|官職]]。通常省略「大臣」簡稱'''政務官'''。

== 概要 ==
大臣政務官是依《{{link-ja|國會審議活性化及政治主導政策決定制度確立相關法律|国会審議の活性化及び政治主導の政策決定システムの確立に関する法律}}》，廢除原有的政務次官，與[[副大臣|副大臣]]一同新設的職務。

大臣政務官協助各府省首長[[大臣|大臣]]（內閣府為[[內閣官房長官|內閣官房長官]]或[[內閣府特命擔當大臣|特命擔當大臣]]）參與特定政策及企劃與處理政務。其地位由《{{link-ja|內閣府設置法|内閣府設置法}}》第14條、《{{link-ja|國家行政組織法|国家行政組織法}}》第17條等規定。各大臣政務官的職務範圍由各府省首長大臣決定。大臣政務官的任免由各府省首長大臣提出，[[內閣|內閣]]決行（與身為{{link-ja|認證官|認証官}}的[[副大臣|副大臣]]不同，不須受[[天皇|天皇]]認證）。

大臣政務官在內閣總辭時，與[[內閣總理大臣|內閣總理大臣]]等其他[[國務大臣|國務大臣]]一樣同時失去其職位。是《{{link-ja|國家公務員法|国家公務員法}}》上的{{link-ja|特別職|特別職}}，慣例上由國會議員出任<ref>與國務大臣一樣可由國會議員以外人士出任，但目前還未有此情況產生。</ref>。

大臣政務官的地位在副大臣之下、{{link-ja|大臣補佐官|大臣補佐官}}與[[事務次官|事務次官]]之上<ref>《國家公務員法》第二條之3、《{{link-ja|特別職職員薪俸相關法律|特別職の職員の給与に関する法律}}》別表第一、《{{link-ja|一般職職員薪俸相關法律|一般職の職員の給与に関する法律}}》別表第十一、《{{link-ja|人事院規則|人事院規則}}》九―四二別表。與檢査官（會計檢査院長除外）、人事官（人事院總裁除外）、內閣危機管理監、侍從長等同等。</ref>。官職名為內閣府大臣政務官（非內閣總理大臣政務官）、復興大臣政務官（非復興廳大臣政務官）、總務大臣政務官（非總務省大臣政務官）等。

大臣政務官與副大臣的不同處是，副大臣是針對該府省的政策進行全面性的輔佐（但內閣府副大臣不處理{{link-ja|大臣委員會|大臣委員会}}相關事務），大臣政務官是針對特定政策進行協助，副大臣（内閣府副大臣除外）在大臣不在時代行各省大臣職務（但不代行國務大臣職務），大臣政務官則無此權力。

=== 官職名實施前的變更 ===
《{{link-ja|內閣府設置法|内閣府設置法}}》、《國家行政組織法部分修正法》（平成11年法律第90號）、《中央省廳等改革國家行政組織相關法律整備等相關法律》（平成11年法律第102號）最早制定的官名未冠上「大臣」，只稱為「政務官」（內閣府政務官、防衛<!--庁の字は入らない-->政務官、總務政務官等），之後同月30日公布、施行的《國會審議活性化及政治主導政策決定制度確立相關法律》（附則第7條至附則第9條），將前述三條法律提及的「政務官」改為「大臣政務官」與「長官政務官」，2001年1月6日正式施行。

== 英文名稱 ==
最初大臣政務官的英文名稱是仿效英國譯為「Parliamentary Secretary」。

2006年（平成18年）2月下旬，{{link-ja|經濟產業大臣政務官|経済産業大臣政務官}}{{link-ja|片山五月|片山さつき}}的名片將其職稱譯為「{{lang|en|'''Vice-Minister'''}}」。現在部分省將大臣政務官譯為「Vice-Minister」。副大臣則譯為「Senior Vice-Minister」。

== 長官政務官 ==
法律上由國務大臣出任首長的各廳（{{link-ja|大臣廳|大臣庁}}）設有同様職務'''長官政務官'''。
