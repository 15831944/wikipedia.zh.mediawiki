[[File:大三巴牌坊.jpg|thumb]]
'''大三巴牌坊'''，其正式名稱為'''聖保祿大教堂遺址'''（{{lang-pt|Ruínas da Antiga Catedral de São Paulo}}），一般稱為'''大三巴'''或'''牌坊'''<ref>[[聖若瑟修院大樓及聖堂|聖若瑟修院]]一區被稱為小三巴，部份澳門市民直接稱為'''牌坊'''</ref>，是澳門天主之母教堂（聖保祿教堂）正面前壁的遺址。大三巴牌坊是[[澳門|澳門]]的標誌性[[建築物|建築物]]之一<ref name="paulo_name">[http://paper.wenweipo.com/2007/06/15/zt0706150049.htm 遊遺跡：火燒旺地 澳門地標大三巴] - [[文匯報|文匯報]] 2007年6月15日 於2007年11月1日造訪</ref>，同時也為「[[澳門八景|澳門八景]]」之一<ref>[http://www.macautouristguide.com/cn/8scene.html 澳門八景] {{webarchive|url=https://archive.is/20061104161908/http://www.macautouristguide.com/cn/8scene.html |date=2006-11-04 }} - 澳門旅遊通勝 於2007-11-03造訪</ref>。2005年與[[澳門歷史城區|澳門歷史城區]]的其他文物成為[[聯合國|聯合國]][[世界文化遺產|世界文化遺產]]<ref name="macauheritage">[http://www.macauheritage.net/mherit/mhfileC.asp 澳門歷史城區小檔案] {{webarchive|url=https://web.archive.org/web/20070718055527/http://www.macauheritage.net/mherit/mhfileC.asp |date=2007-07-18 }} - 澳門世界遺產 於2007-08-03造訪</ref>。

== 名稱由來 ==
[[File:Jesuit_Convent,_Macao.jpg|200px]]
[[File:SPaulo_foundation.jpg|200px]]
大三巴牌坊是位於[[聖保祿山|聖保祿山]]<ref>因聖保祿山上多[[柿|柿]]樹，故民間又稱作為柿山</ref>的'''天主之母教堂'''（{{lang|pt|A lgreja da Madre de Deus}}）前壁。與[[大炮台|大炮台]]及兩者之間的前[[聖保祿學院|聖保祿學院]]為一整體。而「三巴」這個名字是來自於「[[聖保祿|聖保祿]]」的葡萄牙文（{{lang|pt|São Paulo}}），而「大」是指最大的教堂，故「大三巴」是指「最大的教堂」<ref name="paulo_name" />。

== 歷史 ==
[[File:Macao,_China._Photograph_by_John_Thomson,_1870_Wellcome_L0055554.jpg|left]]拍摄]]
聖保祿教堂最早創建於1583年，在1595年和1601年先後發生三次大火而燒毀。[[教堂|教堂]]於1602年重建，由[[意大利|意大利]]籍[[耶穌會|耶穌會]]會士卡洛斯•斯皮諾拉[[神父|神父]]設計，並保留當初1602年重建的奠基石，由於，重建後的教堂是奉獻給聖母，又被稱為聖母堂，教堂於1603年建成，至於前壁及石階於1640年才完成，是當時遠東最大的天主教石建教堂<ref>[www.cnr.cn/2007zt/2007aomen/ssam/200709/t20070929_504583837.html 新澳門八景] - 中央人民廣播電臺 2007年10月8日 於2007年11月1日造訪</ref>。

1762年，[[葡萄牙帝国|葡萄牙帝国]]政府下令沒收耶穌會財產，聖母教堂先後被改作軍官[[宿舍|宿舍]]、兵營。<ref>濠鏡拾遺，[[澳門電視台|澳門電視台]]，於2010年8月3日在澳門電視台衛星頻道播放。</ref>

1835年1月26日黃昏，教堂的廚房起火，火勢過大，最後把教堂燒剩前壁及68級台階，。由於教堂前壁與中國古代的牌坊非常相似，故本地人便稱之為「大三巴牌坊」<ref name="macauheritage" />。

1990年代初，政府曾對教堂遺址進行[[考古學|考古]]調查及修復工作，並發現了當時教堂背後的建築遺址及埋葬教士的墓地。及後在1996年，政府在該遺址建成了[[天主教藝術博物館與墓室|天主教藝術博物館與墓室]]，展出不少澳門教會的珍貴文物<ref>[http://www.macauheritage.net/Info/mwhC.asp?id=204 澳門歷史城區-大三巴牌坊] {{webarchive|url=https://web.archive.org/web/20071109175941/http://www.macauheritage.net/info/mwhC.asp?id=204 |date=2007-11-09 }} - 澳門文物網 於2007年11月1日造訪</ref>；同時，也在大三巴牌坊的背部安裝了鐵架樓梯，以供遊人參觀，可是樓梯的鐵架插了在牌坊的結構之內，此舉引來一些文物專家的不滿。

在回歸前的[[澳葡政府|澳葡政府]]已欲將牌坊申報為[[世界文化遺產|世界文化遺產]]，但由於[[政治|政治]]及技術因素而未能成功<ref>[http://gd.xinhuanet.com/newscenter/ztbd/2007-06/25/content_10390315.htm 澳門申遺　8年籌備 9分鐘敲定] {{webarchive|url=https://web.archive.org/web/20140728044153/http://gd.xinhuanet.com/newscenter/ztbd/2007-06/25/content_10390315.htm |date=2014-07-28 }} - 新華網 2007年6月25日 於2007年11月1日造訪</ref>。

2005年，[[中華人民共和國政府|中華人民共和國政府]]成功向[[聯合國教科文組織|聯合國教科文組織]]申報世界文化遺產，大三巴牌坊成為澳門歷史城區的一部份<ref name="macauheritage" />。

== 建築特色 ==
[[File:MATER_DEI.JPG|thumb]]
[[File:Macau_Cathedral_of_Saint_Paul_back_square.JPG|thumb]]
{{maplink|frame=yes|frame-width=200|zoom=13|type=shape|text=地圖}}
大三巴高27[[公尺|公尺]]、闊23.5公尺、石壁厚2.7公尺田<ref>[http://www.cnheritage.com/special/yctsfx/2005/8/content_13.shtml 艺术和历史的纪念碑 ——大三巴] {{webarchive|url=https://web.archive.org/web/20071202110458/http://www.cnheritage.com/special/yctsfx/2005/8/content_13.shtml |date=2007-12-02 }} - 中华遗产杂志社 2005年8月5日 於2007年11月3日造訪</ref>，
。前壁右側有一石旗桿夾（左邊與之成對的一副已不存），下有68層階級，與教堂位置並不一致，因其興建時朝教堂正面前壁右側已建成聖保祿修院，所以必須歪曲遷就。

* '''第一層'''：最上層為一個三角形山花，上有銅[[鴿|鴿]]象徵[[聖神|聖神]]，左右有日月星辰。頂部有一[[十字架|十字架]]。

* '''第二層'''：正中為大[[耶穌|耶穌]]，圍以釘死其之刑具，往外由4枝混合式壁柱組成，柱間以[[天使|天使]]淺[[浮雕|浮雕]]點綴，柱兩邊有弧形[[山牆|山牆]]。

* '''第三層'''：中央為[[聖母|聖母]]蒙召升天，天使奏樂歡慶，周圍飾以[[牡丹|牡丹]]和[[莫邪菊|莫邪菊]]，象徵來自[[中國|中國]]和[[葡國|葡國]]。此一層中央共有6枝混合式壁柱，兩側以方尖柱代替壁柱。各柱間均以淺浮雕裝飾，左邊是智慧之樹及一隻七頭龍，其上有一聖母，側有中文「聖母踏龍頭」字樣，而右邊對稱位置上則是生命之泉及一隻西式[[帆船|帆船]]，上有海星聖母。在柱組外邊是一頂呈弧狀之扶壁，右邊是一骷髏及中文字「念死者無為罪」；左邊則是一[[魔鬼|魔鬼]]浮雕，中文則是「鬼是誘人為惡」，此層最外側兩塊牆身是由兩條帶有圓頂之方尖柱，左面牆身再現鴿子，下有一開啟的門；而右面為被箭所穿之王冠，下方門閉鎖，意即[[信仰|信仰]]而非權勢才是天國之道。而牆側則設有中國[[舞獅|舞獅]]造型的開大口之獅子；作滴水之用。

* '''第四層'''：由10支科林斯[[柱式|柱式]]及3個窗口組成，窗楣上有均有七朵玫瑰花浮雕裝飾，中間窗口側兩柱間以棕櫚樹裝飾，側窗洞兩邊柱間對稱設有壁龕，供奉4位銅製[[耶穌會|耶穌會]][[聖人|聖人]]，這些銅像是澳門早年製炮工廠的出品<ref>[http://www.macautourism.gov.mo/cn/discovering/sightseeing_detail.php?catid=38 澳門遊蹤 - 大三巴牌坊] {{webarchive|url=https://web.archive.org/web/20071030080707/http://www.macautourism.gov.mo/cn/discovering/sightseeing_detail.php?catid=38 |date=2007-10-30 }} - 澳門旅遊局 於2007年11月1日造訪</ref>。

* '''第五層'''：牌坊最下層有3個入口並有10枝愛奧尼柱式支撐及裝飾牆面，正門兩側各3枝，而側門一邊各2枝；正門上寫有[[拉丁語|拉丁語]]「MATER DEI」（意思為天主之母）。側門門楣上有[[耶穌會|耶穌會]]會徽「IHS」<ref>IHS是「耶穌」的拉丁文 IESUS HOMO (HOMINUM) SALVATOR的 縮寫</ref>的[[浮雕|浮雕]]圖案。

== 其他 ==
由於大三巴牌坊是澳門的地標性建築之一，故政府會定期對牌坊進行檢查和修復。最近一次修復工作時，為確保遊客看到大三巴牌坊，當局仿傚[[歐洲|歐洲]]的做法並設計了一幅大型帆布，懸掛於工程排柵的最底層，帆布上印有大三巴牌坊底層的真實圖案，方便遊客拍照<ref>[http://www.macauheritage.net/Trends/NewsC.asp?nid=5600 大三巴牌坊五月十日起維修　清潔及加固工程恢復原景觀] {{webarchive|url=https://web.archive.org/web/20071202122455/http://www.macauheritage.net/Trends/NewsC.asp?nid=5600 |date=2007-12-02 }} - 澳門文物網 2004年4月30日 於2007年11月1日造訪</ref>。此外，由於澳門的氣候較為[[潮濕|潮濕]]，隨風飄揚的植物[[種子|種子]]很容易便會停留在牌坊的石壁縫隙中生長成雜草，雜草的根會對牌坊結構構成損害，有鑑及此，當局也會定期派人除去雜草以回復牌坊的原貌<ref>[http://www.macaoart.net/News/ContentC.asp?region=L&id=12463 種子隨風飄至無法根除 文化局定期為牌坊除草] {{webarchive|url=https://web.archive.org/web/20071203052953/http://www.macaoart.net/News/ContentC.asp?region=L&id=12463 |date=2007-12-03 }} - 澳門藝術網 2006年6月26日 於2007年11月1日造訪</ref>。

此外，大三巴牌坊作為旅遊熱點，很多不同的活動如[[澳門國際音樂節|澳門國際音樂節]]的露天音樂會<ref>[http://www.macauyso.org.mo/communicate.php 舉辦活動] - 澳門青年交響樂團協會 於2007年11月2日造訪</ref>、[[藝穗|藝穗]]巡遊<ref>[http://www.iacm.gov.mo/show_news.asp?id=633287524161 藝穗○七藝術大巡遊接受報名] {{webarchive|url=https://web.archive.org/web/20071027152727/http://www.iacm.gov.mo/show_news.asp?id=633287524161 |date=2007-10-27 }} - 民政總署 2007年10月24日 於2007年11月2日造訪</ref>和[[澳門國際青年舞蹈節|澳門國際青年舞蹈節]]<ref>[http://news.xinhuanet.com/tai_gang_ao/2006-07/09/content_4809714.htm 第十屆國際青年舞蹈節澳門開幕 巡遊表演引關注] - 新華網 2006年7月09日 於2007年11月2日造訪</ref>也會在這裡及鄰近的[[耶穌會紀念廣場|耶穌會紀念廣場]]舉行。而每逢[[哪吒誕|哪吒誕]]，在大三巴牌坊的旁邊更會豎立一座傳統的中式牌坊，與大三巴牌坊本身構成一個強烈的中西對比。

[[澳門元|澳門元]]5元硬幣上有大三巴牌坊的圖案。「三巴」一名亦自2008年起用於西北太平洋熱帶氣旋用名，以取代「珍珠」。<ref>大公網，[http://www.takungpao.com.hk/news/08/05/29/YM-912367.htm 2008熱帶氣旋加入五個新名字] {{webarchive|url=https://web.archive.org/web/20080604090047/http://www.takungpao.com.hk/news/08/05/29/YM-912367.htm |date=2008-06-04 }}，2008年5月29日。</ref>
