'''大觀社區迫遷事件'''是指從2016年8月至2019年3月期間，在[[中華民國|中華民國]]（[[臺灣|臺灣]]）[[新北市|新北市]][[板橋區_(台灣)|板橋區]]大觀社區發生的迫遷事件。最早在2010年，[[中華民國國軍退除役官兵輔導委員會|中華民國國軍退除役官兵輔導委員會]]向大觀社區居民提起民事訴訟，最後居民全部敗訴。2016年，大觀社區居民成立「[[大觀事件自救會|大觀事件自救會]]」，開始進行組織抗爭、與國軍退除役官兵輔導委員會協商，並提出「原地安置」等訴求。

2017年4月，國軍退除役官兵輔導委員會展開兩波強制拆遷行動。大觀事件自救會先是在4月26日突襲佔領國軍退除役官兵輔導委員會，並在5月4日阻擋[[行政院院長|行政院院長]][[林全|林全]]的座車。同年9月，國軍退除役官兵輔導委員會提出「榮譽國民之家」、「[[社會住宅|社會住宅]]」、「包租代管」等安置方案。隨後大觀事件自救會呼籲[[中華民國總統|中華民國總統]][[蔡英文|蔡英文]]出面解決爭議。

到了2018年4月7日，蔡英文表示期待有最好的處理。但在5月4日，大觀事件自救會與國軍退除役官兵輔導委員會的協調會破局。2019年3月4日，大觀社區居民收到將在3月18日起全面拆遷的通知。3月15日，大觀社區居民與國軍退除役官兵輔導委員會達成共識，同意全部搬遷。

== 歷史背景 ==
=== 非列管眷村 ===
[[File:Chinese_Florence_Nightingale.jpg|thumb]]成立的[[中華民國婦女聯合會|中華民國婦女聯合會]]興建「[[婦聯一村|婦聯一村]]」，其福利中心便是大觀社區的前身。]]

大觀社區的土地原本屬於[[板橋區_(台灣)|板橋]][[林本源家族|林本源家族]]<ref name = "林靖豪 01">{{Cite web |url = http://www.eventsinfocus.org/news/1105 |title = 枋橋西╱何地棲》大觀路社區的故事：聚合 |author = 林靖豪 |date = 2016年10月12日 |publisher = [[焦點事件|焦點事件]] |language = Zh-hant |accessdate = 2019年3月14日 |quote =  }}</ref>。早在1956年，由[[宋美齡|宋美齡]]成立的[[中華民國婦女聯合會|中華民國婦女聯合會]]為了安置軍眷、[[榮譽國民|榮譽國民]]，在板橋鎮[[浮洲|浮洲]]興建起臺灣第一個[[眷村|眷村]]「[[婦聯一村|婦聯一村]]」<ref name = "大觀事件自救會 02">{{Cite web |url = https://www.thenewslens.com/article/95965 |title = 【圖輯】板橋大觀社區的歷史＆迫遷事件始末 |author = 大觀事件自救會 |date = 2018年5月20日 |publisher = [[關鍵評論網|關鍵評論網]] |language = Zh-hant |accessdate = 2019年3月14日 |quote =  }}</ref><ref name = "黃怡菁 01">{{Cite web |url = https://news.pts.org.tw/article/383635 |title = 大觀社區起源 60多年前安置榮民建村 |author = 黃怡菁和郭俊麟 |date = 2018年1月24日 |publisher = [[公視新聞|公視新聞]] |language = Zh-hant |accessdate = 2019年3月14日 |quote =  }}</ref><ref name = "李冠杰 01">{{Cite web |url = https://news.pts.org.tw/article/421557 |title = 從眷村福利中心轉變 大觀社區走過60餘年 |author = 李冠杰、黃郁期和洪靖淳 |date = 2019年2月1日 |publisher = [[公視新聞|公視新聞]] |language = Zh-hant |accessdate = 2019年3月14日 |quote =  }}</ref>，這也是往後浮洲發展的基礎<ref name = "林靖豪 01"/>。而在該眷村中，透過招商投資所設置的福利中心，便是大觀社區的前身<ref name = "大觀事件自救會 02"/><ref name = "林靖豪 01"/><ref name = "李冠杰 01"/>。1963年，[[台风姬罗莉亚_(1963年)|超級強烈颱風葛樂禮]]侵襲[[臺灣|臺灣]]，造成的水患嚴重摧毀「婦聯一村」、「[[婦聯二村|婦聯二村]]」，婦女聯合會將「婦聯一村」住戶撤遷到[[內湖區|內湖區]]、[[中和區|中和區]]等地的其他眷村，但並沒有遷走福利中心的居民<ref name = "大觀事件自救會 02"/>，該聚落居民便留在原地重建家園<ref name = "林靖豪 01"/><ref name = "許詩愷 03">{{Cite web |url = https://www.civilmedia.tw/archives/67899 |title = 大觀專題：面對政府執行拆遷 連土地公也無能為力 |author = 許詩愷 |date = 2017年9月9日 |publisher = [[公民行動影音紀錄資料庫|公民行動影音紀錄資料庫]] |language = Zh-hant |accessdate = 2019年3月14日 |quote =  }}</ref>。

1966年，在居民不知情的情況下，大觀社區的土地被登記為「國有土地」<ref name = "大觀事件自救會 02"/>，並交給[[國防部陸軍司令部|陸軍總司令部]]管理<ref name = "大觀事件自救會 01">{{Cite web |url = https://www.thenewslens.com/article/114986 |title = 【圖輯】「被國有地」之後：板橋大觀社區迫遷事件簡介＆爭議釐清 |author = 大觀事件自救會 |date = 2019年3月8日 |publisher = [[關鍵評論網|關鍵評論網]] |language = Zh-hant |accessdate = 2019年3月14日 |quote =  }}</ref>。1968年，[[臺北市政府|臺北市政府]]在大觀路二段設立[[國軍退除役官兵輔導委員會板橋榮譽國民之家|板橋榮譽國民之家]]<ref name = "大觀事件自救會 02"/><ref name = "林靖豪 01"/><ref name = "陳品存 01">{{Cite web |url = http://www.eventsinfocus.org/news/1119 |title = 枋橋西╱何地棲》大觀路社區的故事：離散 |author = 陳品存 |date = 2016年10月12日 |publisher = [[焦點事件|焦點事件]] |language = Zh-hant |accessdate = 2019年3月14日 |quote =  }}</ref>，並在1982年改由[[中華民國國軍退除役官兵輔導委員會|中華民國國軍退除役官兵輔導委員會]]管理<ref name = "許詩愷 03"/>。1974年，榮譽國民之家正式接管土地後，並未對居民有任何處置<ref name = "陳品存 01"/>。在1985年大觀路拓寬時，在經過榮譽國民之家給予許可後，居民退後2公尺重建房屋，同時獲得[[臺北縣政府|臺北縣政府]]的建物徵收補償<ref name = "大觀事件自救會 02"/><ref name = "許詩愷 03"/><ref name = "陳品存 01"/><ref name = "許詩愷 04">{{Cite web |url = https://www.civilmedia.tw/archives/67900 |title = 大觀專題：法律與人權相抵觸 迫遷烽火不斷的關鍵 |author = 許詩愷 |date = 2017年9月9日 |publisher = [[公民行動影音紀錄資料庫|公民行動影音紀錄資料庫]] |language = Zh-hant |accessdate = 2019年3月14日 |quote =  }}</ref>。1993年，居民開始提出承購土地的申請<ref name = "大觀事件自救會 02"/><ref name = "許詩愷 03"/><ref name = "陳品存 01"/>。

國軍退除役官兵輔導委員會榮譽國民之家對此召開「處理私人使用國有公用土地協調會」，認為居民可以提出1974年以前的水電、居住證明<ref name = "大觀事件自救會 02"/><ref name = "許詩愷 04"/>，理論上申購並沒有問題<ref name = "許詩愷 03"/><ref name = "陳品存 01"/>。隨後板橋榮譽國民之家要求居民提供水電證明，作為後續向[[財政部國有財產署|財政部國有財產署]]辦理現狀移交承租、承購的依據<ref name = "大觀事件自救會 02"/><ref name = "大觀事件自救會 01"/>。但居民積極配合提交證明文件後，在榮譽國民之家的疏失下遭到遺漏，國軍退除役官兵輔導委員會因而沒有繼續辦理<ref name = "許詩愷 03"/><ref name = "陳品存 01"/><ref name = "大觀事件自救會 01"/>。

=== 訴訟定讞 ===
[[File:FUZHOU_bridge(NewTaipei)2018.jpg|thumb]]通過[[板橋區_(台灣)|板橋鎮]][[浮洲|浮洲]]地區都市計畫。]]

早在2000年，[[中華民國財政部|中華民國財政部]]訂定頒布《各機關經管國有公用被占用不動產處理原則》，其中賦予政府得以動用民事訴訟排除居民的法源依據，這導致[[華光社區|華光社區]]、大觀社區和[[紹興社區|紹興社區]]皆被控告<ref name = "大觀事件自救會 02"/><ref name = "大觀事件自救會 01"/>。2002年，臺北縣政府通過板橋市浮洲地區都市計畫，將大觀社區所在的土地劃為社會福利措施用地<ref name = "大觀事件自救會 02"/><ref name = "陳品存 01"/><ref name = "許詩愷 04"/>。而由於國有土地政策的轉變，大觀社區當地居民便成為「惡意侵佔戶」<ref name = "李冠杰 01"/><ref>{{Cite web |url = https://pnn.pts.org.tw/project/inpage/2035/35/38 |title = 法無明文 大觀社區居民爭不到的居住權？ |author = 王柔婷 |date = 2019年3月18日 |publisher = PNN公視新聞議題中心 |language = Zh-hant |accessdate = 2019年4月4日 |quote =  }}</ref>。

2010年開始，國軍退除役官兵輔導委員會板橋榮譽國民之家將社區居民分為六批，開始提起民事訴訟，要求大觀社區拆除房屋、歸還土地、返還5年不當所得<ref name = "大觀事件自救會 02"/><ref name = "許詩愷 03"/><ref name = "陳品存 01"/><ref name = "林靜梅 01">{{Cite web |url = https://news.pts.org.tw/article/343511 |title = 退輔會促拆屋還地 大觀社區居民抗議 |author = 林靜梅和陳立峰 |date = 2016年12月12日 |publisher = [[公視新聞|公視新聞]] |language = Zh-hant |accessdate = 2019年3月14日 |quote =  }}</ref><ref>{{Cite web |url = https://news.pts.org.tw/article/385739 |title = 大觀社區被判拆屋還地 居民無心過年 |author = 黃怡菁和邱福財|date = 2018年2月15日 |publisher = [[公視新聞|公視新聞]] |language = Zh-hant |accessdate = 2019年3月14日 |quote =  }}</ref>。2014年，法院訴訟定讞，裁定大觀社區的居民全部敗訴<ref name = "陳品存 01"/><ref name = "陳佳鑫 01">{{Cite web |url = https://news.pts.org.tw/article/425531 |title = 大觀社區居民敗訴 退輔會設條件緩拆 |author = 陳佳鑫和彭耀祖 |date = 2019年3月14日 |publisher = [[公視新聞|公視新聞]] |language = Zh-hant |accessdate = 2019年3月14日 |quote =  }}</ref>。2016年5月，榮譽國民之家強制執行收取不當得利罰鍰，平均每戶須賠40萬元<ref name = "陳品存 01"/><ref name = "許詩愷 02">{{Cite web |url = https://www.civilmedia.tw/archives/66207 |title = 無安置有罰鍰 大觀迫遷戶：看不到未來 |author = 許詩愷 |date = 2017年7月17日 |publisher = [[公民行動影音紀錄資料庫|公民行動影音紀錄資料庫]] |language = Zh-hant |accessdate = 2019年3月14日 |quote =  }}</ref>。同年9月，榮譽國民之家陸續寄出公告，要求居民在10月7日之前自行拆屋還地<ref name = "陳品存 01"/>。

最初國軍退除役官兵輔導委員會並無提出安置方案<ref name = "許詩愷 02"/>，但在遭遇組織抗爭後，先後提出入住榮譽國民之家、派人發放青年住宅傳單等<ref name = "大觀事件自救會 01"/>。而在2017年9月26日，為了安置大觀社區拆遷戶，國軍退除役官兵輔導委員會提供「榮譽國民之家」、「[[社會住宅|社會住宅]]」、「包租代管」等數種安置拆遷戶的方案<ref name = "大觀事件自救會 01"/>，但仍有疑慮的大觀社區居民則是持保留態度<ref>{{Cite web |url = https://news.pts.org.tw/article/372058 |title = 安置大觀拆遷戶 "包租代管"案引發抗議 |author = 黃怡菁和郭俊麟 |date = 2017年9月26日 |publisher = [[公視新聞|公視新聞]] |language = Zh-hant |accessdate = 2019年3月14日 |quote =  }}</ref><ref>{{Cite web |url = https://pnn.pts.org.tw/project/inpage/89/35/38 |title = 包租代管安置拆遷戶？ 大觀居民仍有疑慮 |author = 何宇軒 |date = 2017年9月27日 |publisher = PNN公視新聞議題中心 |language = Zh-hant |accessdate = 2019年3月14日 |quote =  }}</ref><ref name = "張智琦 01">{{Cite web |url = https://www.coolloud.org.tw/node/89266 |title = 【大觀迫遷】退輔會拋包租代管安置方案 居民態度保留 |author = 張智琦 |date = 2017年9月26日 |publisher = [[苦勞網|苦勞網]] |language = Zh-hant |accessdate = 2019年3月14日 |quote =  }}</ref>。

== 長期抗爭 ==
=== 早期協商 ===
