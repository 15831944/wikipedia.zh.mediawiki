《'''論語'''》是一本以記錄[[春秋時期|春秋時期]][[思想家|思想家]][[孔子|孔子]]言行為主的言論匯編，在古書中又別以'''論'''、'''語'''、'''傳'''、'''記'''等字單稱<ref>{{cite book|author1=周予同|title=群經概論|date=1997|publisher=臺灣商務印書館|isbn=9789570513547|pages=124-126|url=https://books.google.com.tw/books?id=NZB8XRYrd2UC&lpg=PA125&ots=rRcojGP_I-&hl=zh-TW&pg=PA125#v=onepage&q&f=false}}</ref>，為[[儒家|儒家]]重要經典之一，在[[四庫全書|四庫全書]]中為經部。“論”通行的读音为“ㄌㄨㄣˊ（lún）”<ref>{{cite web | url=http://www.yywzb.com.cn/more.asp?infoid=5338 | title=《论语》的“论”为什么读lún?（第759期） | publisher=[[語言文字報|語言文字報]] | date=2014-01-22 | accessdate=2014-02-27 }}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，為論纂、編纂的意思，「語」為談說的意思，如《[[國語_(書)|國語]]》之類，合起來指言论的汇编。東漢[[班固|班固]]的《[[漢書|漢書]]·藝文志》稱：「《論語》者，孔子應答弟子、時人，及弟子相與言而接聞於夫子之語也。當時弟子各有所記，夫子既卒，門人相與輯而論纂，故謂之《論語》。」

《論語》涉及多方面內容，自[[汉武帝|汉武帝]]“罢黜百家，独尊儒术”之後，它被尊为“五经之輨轄，六艺之喉衿”<ref>[[趙岐|趙岐]]《孟子题辞》</ref>，是研究孔子及[[儒家思想|儒家思想]]尤其是先秦儒家思想的一手資料。[[南宋|南宋]]時[[朱熹|朱熹]]將《[[大學_(經傳)|大學]]》、《論語》、《[[孟子_(四書)|孟子]]》、《[[中庸|中庸]]》合為「[[四書|四書]]」，使之在儒家經典中的地位日益提高。[[元代|元代]][[延佑|延佑]]年间，[[科举|科举]]开始以“四书”开科取士。此后一直到清朝末年推行[[洋务运动|洋务运动]]，废除科举之前，《论语》一直是学子士人推施奉行的金科玉律。

== 流传 ==
《論語》自[[战国|战国]]前期成书问世，《魯論》和《齊論》則分別為魯人和齊人所研習的版本。因[[秦始皇|秦始皇]]的“[[焚书|焚书]]”政策，几乎惨遭灭顶之灾。到[[西漢|西漢]]初年，朝廷明令“大收篇籍，广开献书之路”，有些冒着生命危险收藏书籍的人，纷纷向朝廷“献书”，因口口相传再手抄笔录辗转反复，字句往往有所差异。魯人口頭傳授的《魯論》20篇；齊人口頭傳授的《齊論》22篇，比魯論多了“問王”、“知道”兩篇；[[魯共王|魯共王]][[劉餘|劉餘]]拆毀[[孔子之宅|孔子之宅]]發現[[孔子家族|孔子家族]]所藏[[古文|古文]][[竹簡|竹簡]]，即《古論》21篇，其堯曰篇中的「子張問何如可以從政」獨立出第二個“子張”篇。西漢末年，[[漢成帝|漢成帝]]帝師[[張禹_(西漢)|張禹]]以《魯論》為主，根據《魯論》刪除了《齊論》的“問王”、“知道”兩篇，有21篇，稱為《張侯論》；西元[[二世纪|二世纪]]中期[[東漢|東漢]]末年，[[鄭玄|鄭玄]]又以《張侯論》為底本，定下《論語》今本，並參考《齊論》、《古論》，作《論語注》，《齊論》、《古論》不久亡佚。不到一百年後，[[三國|三國]]時期[[何晏|何晏]]把鄭玄本及其反對派的觀點编成了《論語集解》。何晏之後，[[南北朝|南北朝]]时代，[[皇侃|皇侃]]受[[佛教|佛教]]的影响，编成《论语义疏》。西元八至九世纪，[[唐朝|唐朝]]的[[韓愈|韓愈]]、[[柳宗元|柳宗元]]對郑玄、何晏所編之本都持懷疑態度，要回到原典。但由於鄭玄在世時已與孔子相差600餘年，《論語》原典並不可考<ref>[http://book.sina.com.cn/news/c/2007-03-31/1323212751.shtml 歷史學家朱維錚批評于丹根本不知《論語》為何物]</ref>。今本據阮元校勘《十三经注疏》本統計，約1萬2千字。<ref>[[魏崧|魏崧]]《壹是纪始》作“論語一萬二千七百字”。[[黄侃|黄侃]]《手批白文十三經提要》说：“鄭耕老作一萬三（一作二）千七百字，歐陽公作一萬一千七百五字。阮元校勘的《十三经注疏》本統計是一萬二千七百字，此和鄭耕老所說相同。”[[錢穆|錢穆]]在《孔子與論語》一字說“其總字數，不出一萬六千四百多字。123”</ref>

2017年1月，中國考古學家宣佈於[[江西省|江西省]][[南昌市|南昌市]][[西漢|西漢]][[海昏侯墓|海昏侯墓]]內出土文物中發現《齊論》竹簡，其中包括以「智道」（即「知道」）爲題的篇章。<ref>[http://www.appledaily.com.tw/realtimenews/article/international/20170528/1127935/1/2%E5%8D%83%E5%B9%B4%E3%80%8A%E8%AB%96%E8%AA%9E%E3%80%8B%E5%A4%9A%E4%BA%862%E7%AF%87%E3%80%80%E3%80%8A%E9%BD%8A%E8%AB%96%E8%AA%9E%E3%80%8B%E5%B0%87%E9%87%8B%E8%AE%80 【珍寶片】2千年《論語》多了2篇　《齊論語》將釋讀]</ref>

== 結構 ==
[[File:Analects_LunYu_Rongo2.JPG|thumb]]藏]]
現存的《論語》共20篇，492章。每篇篇名取自正文開頭，或“子曰”、“子謂”后首句的前二、三字，與篇目中的內容無關。按照習慣，通常把前10篇稱為“上論”，后10篇稱為“下論”。《論語》以〈學而〉為首。
[[File:Analects.JPG|260px]]
[[File:Rongo_Analects_02.jpg|260px]]

# 學而第一（論語各篇的標題與談述的內容無關。本篇的章句主要讲“务本”的道理，引导初学者进入“道德之门”）<ref>括号内的简介来自于：徐志刚译注《论语通译》（2005年4月第5次印刷）</ref>
# 為政第二（主要讲治理国家的道理和方法）
# 八佾第三（主要记录孔子谈论礼乐）
# 里仁第四（主要讲仁德的道理）
# 公冶長第五（主要讲评价古今人物及其得失）
# 雍也第六（记录孔子和弟子们的言行）
# 述而第七（主要记录孔子的容貌和言行）
# 泰伯第八（主要记孔子和曾子的言论及其对古人的评论）
# 子罕第九（主要记孔子言论，重点为孔子的行事风格，提倡和不提倡做的事）
# 鄉黨第十（主要记录孔子言谈举止，衣食住行和生活习惯）
# 先進第十一（主要记录孔子教育言论和对其弟子的评论）
# 顏淵第十二（主要讲孔子教育弟子如何实行仁德，如何为政和处世）
# 子路第十三（主要记录孔子论述为人和为政的道理）
# 憲問第十四（主要记录孔子和其弟子论修身为人之道，以及对古人的评价）
# 衛靈公第十五（主要记录孔子及其弟子在周游列国时的关于仁德治国方面的言论）
# 季氏第十六（主要记孔子论君子修身，以及如何用礼法治国）
# 陽貨第十七（主要记录孔子论述仁德，阐发礼乐治国之道）
# 微子第十八（主要记录古代圣贤事迹，以及孔子众人周游列国中的言行，也记录了周游途中世人对于乱世的看法）
# 子張第十九（主要记录孔子和弟子们探讨求学为道的言论，弟子们对于孔子的敬仰赞颂）
# 堯曰第二十（本篇较为短小，疑有错漏。主要记录古代圣贤的言论和孔子对于为政的论述）

==評價==
[[北宋|北宋]]政治家[[趙普|趙普]]称：临死时对[[宋太宗|宋太宗]]道，臣有《论语》一册，一半可用来助君打天下，一半可用来助君治天下。

[[格奥尔格·威廉·弗里德里希·黑格尔|黑格尔]]：“我们看到孔子和他的弟子们的谈话（按即“论语”——译者贺麟注），里面所讲的是一种常识道德，这种常识道德我们在哪里都找得到，在哪一个民族里都找得到，可能还要好些，这是毫无出色之处的东西。 孔子只是一个实际的世间智者，在他那里思辨的哲学是一点也没有的——至于一些善良的、老练的、道德的教训，从里面我们不能获得什么特殊的东西。 西塞罗留下给我们的《政治义务论》便是一本道德教训的书，比孔子的书内容丰富，而且更好。我们根据他的原著可以断言：为了保持孔子的名声，假如他的书从来未曾有过翻译，那倒是更好的事。”<ref>黑格尔《哲学史讲演录》第一卷，商务印书馆1959年版，第119-120页</ref>

[[王品集團|王品集團]]前董事長[[戴勝益|戴勝益]]曾宣稱其可以用半部《論語》創王品。

[[北京师范大学|北京师范大学]]教授[[于丹|于丹]]：於2006年11月1日出版[[于丹论语心得|于丹论语心得]]一书，引言“道不远人，让我们在圣贤的光芒下学习成长。”
