'''烈空坐'''（{{lang-ja|'''レックウザ'''}}；{{lang-en|'''Rayquaza'''}}），是[[精靈寶可夢|精靈寶可夢]]系列登場的{{NUMBEROFPOKEMON}}種虛構[[角色|角色]]（[[怪物|怪獸]]）中的一種，烈空坐為龍屬性與飛行屬性的天空寶可夢。主要顏色為綠色，身上環繞著黃色的花紋，有著[[龍|龍]]的姿態，與[[蓋歐卡|蓋歐卡]]、[[固拉多|固拉多]]為豐緣地區中傳說的超古代寶可夢。

主要棲息於臭氧層中。並以空氣中的水與灰塵為食物。
 
身長為7米，在目前所知的寶可夢中次於[[吼鯨王|吼鯨王]]、[[大鋼蛇|大鋼蛇]]、[[大岩蛇|大岩蛇]]。

== 遊戲中的烈空坐 ==
*在[[神奇寶貝_紅寶石·藍寶石|紅寶石、藍寶石、綠寶石版]]中首度出現。豐緣聯盟編號為200，全國圖鑑編號為384。 
* [[神奇寶貝_紅寶石·藍寶石|紅寶石、藍寶石版]]中，烈空坐在打敗四天王與冠軍，進入殿堂之後，可在「空中之柱」最頂層捕捉，等級為70。
*而[[神奇寶貝_紅寶石·藍寶石|綠寶石版]]中，前往空中之柱喚醒烈空坐並平息蓋歐卡與固拉多的紛爭後，便能在「空中之柱」最頂層捕捉烈空坐，不用進入殿堂，等級同樣為70。
*[[神奇寶貝_心金、魂銀|心金、魂銀版]]中，捕捉到[[蓋歐卡|蓋歐卡]]與[[固拉多|固拉多]]後，會出現烈空坐，等級為50。
*[[神奇寶貝_鑽石·珍珠|鑽石、珍珠、白金版]]中，可以學會新招式「龍之波動」。
*「攻擊」與「特攻」上的數值皆高達150，種族值為680，其餘能力也有不錯的數值。
*特性為「氣閘」，效果是全場的寶可夢皆不受天氣的影響。

== 動畫中的烈空坐 ==
*《[[神奇寶貝鑽石&珍珠|神奇寶貝鑽石&珍珠]]》43集中，在夢妖魔的夢境中登場，並為神奧聯盟冠軍[[竹蘭|竹蘭]]使用的寶可夢，當小智一行人想逃離夢境時，曾與[[夢妖魔|夢妖魔]]合體為[[紫色的烈空坐|紫色的烈空坐]]，最後被小智等人打敗。

*《[[特別篇　最強超級進化～第二章～|特別篇　最強超級進化～第二章～]]》中，[[烈空坐|烈空坐]]被傳說的巨石召喚並進行了超進化( 超級進化 )，因人類對巨石的實驗而惱怒並用龍之波動進行破壞，[[艾嵐|艾嵐]]和[[大吾|大吾]]讓[[噴火龍|噴火龍]]和[[巨金怪|巨金怪]]超級進化以對其進行阻止。烈空坐靈活地避開了[[超級噴火龍|超級噴火龍]]X的[[噴射火焰|噴射火焰]]，並用畫龍點睛輕易地擊落噴火龍。最後它使用流星群大肆破壞，致使巨石進一步產生反應而消失，自己也在巨石消失後揚長而去。網路上的[[風行網|風行網]]卡通動畫是收錄在《[[神奇寶貝XY|神奇寶貝XY]]》43集。 

*《[[特別篇　最強超級進化～第三章～|特別篇　最強超級進化～第三章～]]》中，[[烈空坐|烈空坐]]受傳說的巨石能量的吸引並在巨石實體化後出現並進行了超進化，與原始回歸的[[蓋歐卡|蓋歐卡]]與[[固拉多|固拉多]]爭奪傳說的巨石。在躲過固拉多的日光束與蓋歐卡的根源波動後，用[[流星群|流星群]]對雙方進行攻擊，隨後又被固拉多的噴火和蓋歐卡的根源波動擊中，它對蓋歐卡發出破壞光線將其擊飛很長一段距離並撞上固拉多。固拉多以斷崖之劍反擊[[超級烈空坐|超級烈空坐]]，但烈空坐承受住了這一擊，並在隨後以兇猛而快速的畫龍點睛擊中雙方，迫使蓋歐卡與固拉多解除原始回歸狀態。正當它準備獨占傳說的巨石時，被[[弗拉達利|弗拉達利]]派來的潛艇發出的閃光彈暫時致盲。在巨石的能量被潛艇收入潛艇後，烈空坐無法感知出巨石的所在，於是解除了超級進化並離開。網路上的[[風行網|風行網]]卡通動畫是收錄在《[[神奇寶貝XY|神奇寶貝XY]]》44集。 

*《[[光環的超魔神　胡帕|光環的超魔神　胡帕]]》中，一隻[[異色烈空坐|異色烈空坐]]被[[胡帕的真身|胡帕的真身]]召喚出來幫助[[小智|小智]]一行人對抗[[胡帕的分身|胡帕的分身]]，其間進行了超級進化，並在小智的指揮下成功拖延了時間。在阿爾宙斯解除危機後，與其他傳說中的寶可夢一樣各自離開。 

*《[[夢妖魔！從惡夢中逃脫!!|夢妖魔！從惡夢中逃脫!!]]》中，由[[夢妖魔|夢妖魔]]幻化的[[竹蘭|竹蘭]]有一隻[[烈空坐|烈空坐]]並且戰勝了[[固拉多|固拉多]]。

== 電影中的烈空坐 ==
在《[[神奇寶貝劇場版：裂空的訪問者_代歐奇希斯|裂空的訪問者 代歐奇希斯]]》中登場，認為代歐奇希斯是侵略者，因此對代歐奇希斯詹開激烈的殊死戰，數次因代歐奇希斯的必殺技—精神提升而居於下風，激戰到了最後，雖然被代歐奇希斯打到掉入冰冷的海水，但還是成功以「破壞死光」重創了代歐奇希斯。

而四年後，當時被重創的代歐奇希斯從隕石中復活，因此想將代歐奇希斯再次消滅，但後來拉魯斯市的防衛系統因烈空坐的破壞發生問題，而自己也受到了波及，但後來受到代歐奇希斯的幫助，由於感謝代歐奇希斯的協助，不再對代歐奇希斯存有敵意，回到了自己居住的地方。
