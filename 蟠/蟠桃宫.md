'''蟠桃宫'''，全名“护国太平蟠桃宫”，是位于[[北京|北京]][[东便门|东便门]]内护城河南岸的一座[[道教|道教]]宫观。现已无存。<ref name=sdr>[http://www.shouduren.com/html/2009/08/030202297045.htm 旧京城南的庙会，首都人，2009-8-3]</ref>

==历史==
[[File:蟠桃宫道士.jpg|thumb]]
蟠桃宫始建于[[元朝|元朝]]，一说始建于[[明朝|明朝]]。后来衰落。[[清朝|清朝]][[康熙|康熙]]元年（1662年）[[工部|工部]][[尚书|尚书]][[吴达礼|吴达礼]]重建王母殿后三间殿<ref name=sdr/>。据说吴达礼重修蟠桃宫时，在蟠桃宫东侧又建了观音庵，当时庵名“白云深处”，相传是由前朝宦官的旧宅改建<ref name=xhw/>。蟠桃宫香火兴盛，不少青年男女都来烧香、拴娃娃，祈求子嗣<ref name=sdr/>。

蟠桃宫庙会十分有名，但在[[中华人民共和国|中华人民共和国]]成立后不久即停办。1985年，因展宽马路，将蟠桃宫的山门、鼓楼及部分东配殿拆除，后来在1987年因修[[东便门桥|东便门立交桥]]，将蟠桃宫的建筑全部拆除，部分建筑构件由文物部门保存<ref name=sdr/>。在东便门立交桥西南路边的绿地中，尚存一块《护国太平宫碑》，周围环以铁护栏<ref name=xhw>[http://www.bj.xinhuanet.com/bjpd_sdwm/2006-06/19/content_6543163.htm 蟠桃宫庙会，新华网，2006-06-19] {{Wayback|url=http://www.bj.xinhuanet.com/bjpd_sdwm/2006-06/19/content_6543163.htm |date=20120506230947 }}</ref>。2010年代初，为该碑加建了一座碑亭。该碑阳面碑额为“护国太平宫碑”，阴面碑额为“万世流芳”。阳面碑文为：

==建筑==
蟠桃宫坐南朝北，和其西北方向的[[北京城东南角楼|北京城东南角楼]]遥对。蟠桃宫共有两层殿宇，其主要建筑有：<ref name=sdr/><ref name=xhw/>

*山门：坐南朝北。门前有石狮和旗杆各一对。门上石匾内书“护国太平蟠桃宫”，左右墙上嵌有“蟠桃盛会”四个绿边琉璃大字。
*钟楼、鼓楼：位于山门内
*前殿：祀[[西王母|西王母]]娘娘，并筑有一座鳌山，山上塑有众仙腾云前来为王母娘娘祝寿。
*后殿：祀[[斗母元君|斗母元君]]
**配殿：在前殿、后殿两侧，分别祀财神[[赵公明|赵公明]]、[[三官|三官]]、送生娘娘等诸神。<ref name=sdr/><ref name=xhw/>

==庙会==
[[File:蟠桃宫和东南角楼.jpg|thumb]]]]
相传[[农历|农历]]三月初三是[[西王母|西王母]]娘娘的寿诞，会举行蟠桃圣会。故蟠桃宫每年农历三月初一至初三举办庙会。<ref name=sdr/>

从[[崇文门|崇文门]]外沿护城河向东行三里，便到蟠桃宫。路上车水马龙，人头攒动。时逢春末夏初，游人大多穿上夏装。在游客中，女性占了很大比重。《都门赘语·蟠桃宫诗》曰：“暮春天气最清和，如蚁游人夹道行。多少仙姬争艳冶，不知谁是[[许飞琼|许飞琼]]。”<ref name=sdr/>

蟠桃宫前面及东面很空旷，大鼓书、[[单弦|单弦]]、[[相声|相声]]、变[[戏法|戏法]]等的艺人在庙外献艺。民间花会[[高跷|高跷]]、[[十不闲|十不闲]]、[[舞狮|舞狮]]、[[旱船|旱船]]等也来此朝拜。做生意的小商贩所摆的摊位鳞次栉比，连绵数里。此时正值[[春耕|春耕]]开始，买卖[[农具|农具]]的人也很多。<ref name=sdr/>
[[File:蟠桃宫庙会.jpg|thumb]]
蟠桃宫庙会上有一种赶脚用的毛驴，俗称“对槽驴”。游客可以从崇文门外桥头乘此种毛驴来到蟠桃宫门前。<ref name=sdr/>
 
蟠桃宫对面的护城河畔长堤很平坦，是庙会期间赛马的好地方。豪门贵族的纨绔子弟多好此道。[[李鸿章|李鸿章]]的孙子[[李国杰|李国杰]]为了多玩两天，曾经命当时的主管衙门下令，将蟠桃宫庙会由三天变为五天。<ref name=sdr/>

约在1960年代，蟠桃宫庙会停办。2007年，蟠桃宫庙会恢复，改在[[北京明城墙遗址公园|北京明城墙遗址公园]]内举办<ref>{{Cite web |url =http://news.163.com/11/0408/01/71363SJQ00014AED.html  |title =京味吆喝唤回“蟠桃宫”记忆  |author =  |date = 2011-04-08 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://china.huanqiu.com/hot/2017-03/10404547.html |title =蟠桃宫庙会重现 汇聚各色中华老字号美食  |author =  |date =2017-03-30  |publisher = 环球网 |language =  |accessdate =  |quote =  }}</ref>
