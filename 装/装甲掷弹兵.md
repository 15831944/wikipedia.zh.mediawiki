'''裝甲擲彈兵'''（德语：Panzergrenadier）是[[第二次世界大戰|第二次世界大戰]]期間[[德國|德國]][[陸军|陸军]]所創設的一个[[兵科|兵科]]，是[[现代|现代]][[陸軍|陸軍]]中[[装甲步兵|装甲步兵]]的先驅。[[奧地利|奧地利]]、[[瑞士|瑞士]]、[[智利|智利]]的裝甲步兵也称为裝甲擲彈兵。

== 裝甲擲彈兵名称的由来 ==
[[擲彈兵|擲彈兵]]是17世纪中叶[[欧洲|欧洲]][[陸軍|陸軍]]的一个兵科，最早是指军队中能投擲[[手榴弹|手榴弹]]的[[步兵|步兵]]。由于當時的手榴彈體積重量類似小型砲彈，因此須要在步兵當中挑選臂力過人的士兵才有辦法投擲，再者當時手榴彈十分不穩定，往往尚未扔出去即爆炸。這些士兵在[[战斗|战斗]]中需要在己方[[战线|战线]]的最前面向敌方投掷手榴弹，因此在欧洲军队中将擲彈兵的稱號作为[[表彰|表彰]][[军人|军人]]英勇战斗表现的荣誉称号，基本上這些擲彈兵是步兵當中的精銳及佼佼者。

*1942年7月5日，为了鼓舞德军[[步兵|步兵]]的士气並延續[[普鲁士|普鲁士]]军队的传统，德國陸軍下令將裝甲師內的[[步槍團|步槍團]]（Schützen-Regiment）改稱[[裝甲擲彈兵團|裝甲擲彈兵團]]（Panzergrenadier-Regiment）。

*1942年10月15日，德國陸軍下令將步兵師（Infanterie-Division）內的步兵團（Infanterie-Regiment）改稱為擲彈兵團（Grenadier-Regiment），同時[[摩托化步兵師|摩托化步兵師]]（Infanterie-Division (mot.)）內的[[摩托化步兵團|摩托化步兵團]]（Infanterie-Regiment (mot.)）也改稱[[摩托化擲彈兵團|摩托化擲彈兵團]]（Grenadier-Regiment (mot.)）。

*1943年6月23日，德國陸軍下令將摩托化步兵師（Infanterie-Division (mot.)）改稱為[[裝甲擲彈兵師|裝甲擲彈兵師]]（Panzergrenadier-Division）。

*1944年12月1日，德國陸軍下令將[[裝甲擲彈兵師|裝甲擲彈兵師]]（Panzergrenadier-Division）內的[[摩托化擲彈兵團|摩托化擲彈兵團]]（Grenadier-Regiment (mot.)）改稱為[[裝甲擲彈兵團|裝甲擲彈兵團]]（Panzergrenadier-Regiment）。

== 裝甲擲彈兵的種類 ==
廣義的裝甲擲彈兵汎指[[第二次世界大戰|第二次世界大戰]]期間[[德國|德國]]成立的所有摩托化和裝甲化步兵部隊，包括：

隸屬於裝甲兵科：

*[[裝甲師|裝甲師]]的摩托化步槍旅（Schützen-Brigade）/[[摩托化步槍團|摩托化步槍團]]（Schützen-Regiment (mot.)）/機車步槍營（Kradschützen-Bataillon）/[[裝甲擲彈兵旅|裝甲擲彈兵旅]]/摩托化[[裝甲擲彈兵團|裝甲擲彈兵團]]（Panzergrenadier-Regiment (mot.)）/裝甲化[[裝甲擲彈兵團|裝甲擲彈兵團]]（Panzergrenadier-Regiment (gp.)）

*輕裝師的摩托化騎兵步鎗團（Kavallerie-Schützen Regiment (mot.)）/機車步槍營（Kradschützen-Bataillon）

*摩托化步兵師的機車步槍營（Kradschützen-Bataillon）

*[[裝甲擲彈兵師|裝甲擲彈兵師]]/摩拖化擲彈兵團（Grenadier-Regiment (mot.)）/摩托化[[裝甲擲彈兵團|裝甲擲彈兵團]]（Panzergrenadier-Regiment (mot.)）

*[[44年裝甲旅|44年裝甲旅]]的裝甲化裝甲擲彈兵營

*[[裝甲擲彈兵旅|裝甲擲彈兵旅]]

隸屬於步兵科：

*摩托化步兵師（Infanterie-Division (mot.)）/摩托化步兵團（Infanterie-Regiment (mot.)）/摩托化擲彈兵團

*摩托化擲彈兵旅（Grenadier-Brigade (mot.)）

== 裝甲擲彈兵的作戰方式 ==
装甲擲彈兵的主要被设定作用于伴随[[装甲部队|装甲部队]]执行军事作战任务，在[[战斗|战斗]]中为[[坦克|坦克]]或其他易受直接火力打击的摩托化单位提供[[步兵|步兵]]战术支援。掷弹兵的活动范围通常在装甲部队行进道路之前，负责搜索以及暴露敌人的主要防御力量之所在，并尽可能地摧毁威胁到本方装甲单位的敌方火力点。防御作战时掷弹兵被认为是重要的机动防御者，通常被用于堵塞敌人装甲矛头突破己方阵地所产生的缺口，或在己方增援赶到之前阻止敌方扩大突破口。反击作战中装甲掷弹兵亦多被用于辅助攻击行动，作用是清除或爆破阻碍己方装甲突击力量前进的各种设施。主要装备[[机枪|机枪]]、[[步枪|步枪]]、[[冲锋枪|冲锋枪]]、[[火炮|轻型火炮]]、[[掷弹筒|掷弹筒]]、[[装甲车|装甲车]]以及其它装备等。
