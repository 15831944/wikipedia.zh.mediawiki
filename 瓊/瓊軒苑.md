'''瓊軒苑'''（'''King Hin Court'''）是[[香港房屋委員會|香港房屋委員會]]的[[居屋|居屋]]屋苑之一，位於[[九龍|九龍]][[黃大仙區|黃大仙區]][[斧山|斧山]]瓊東街1號，與[[富山邨|富山邨]]、[[瓊山苑|瓊山苑]]、[[瓊麗苑|瓊麗苑]]、[[新麗花園|新麗花園]]、[[宏景花園|宏景花園]]及[[嘉峰臺|嘉峰臺]]毗鄰，早於2002年3月26日落成，屬「出售剩餘居屋單位」的第四期屋苑<ref>[http://www.info.gov.hk/gia/general/200809/09/P200809090210.htm 出售剩餘居屋單位第四期明起接受申請]</ref>。根據2011年[[人口普查|人口普查]]的結果及透過[[瓊富選區|瓊富選區]]<ref>[http://www.census2011.gov.hk/tc/district-profiles/ca/wong-tai-sin/h20.html 2011年人口普查 - 黃大仙區議會瓊富選區 (H20) 統計便覽]</ref>及相鄰屋苑（例如[[富山邨|富山邨]]、[[宏景花園|宏景花園]]、[[瓊山苑|瓊山苑]]等）的人口數字，估計約有1,000名居民。

2005年因應[[中國大陸|中國大陸]]放寬[[自由行|自由行]]，房屋署曾考慮將瓊軒苑連同另外兩個屋苑改裝為[[旅館|旅館]]，但因酒店業界反對而告吹<ref>[http://news.yonghua.net.cn/htmldata/2005_03/13/article_205476_1.html 香港"琼轩苑" 真假"李逵"之辩]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，最後恢復以居屋形式出售。

== 樓宇 ==
瓊軒苑為單幢式屋苑，樓高41層，[[新十字型|新十字型]]設計，每翼都有不同最高樓層（較風車型相似），也是[[新十字型|新十字型]]設計大廈中比較特別之一（景和樓呈T字型），09室設有特大窗戶（右邊單位），每層10個單位，提供28個一房單位、106個兩房單位及210個三房單位，合共344個單位，高層部分單位可享[[維多利亞港|維多利亞港]]海景，但望山單位會望到[[鑽石山火葬場|鑽石山火葬場]]景觀<ref>[http://www.singtao.com/archive/fullstory.asp?andor=or&year1=2002&month1=06&day1=06&year2=2002&month2=06&day2=06&category=all&id=20020606a02&keyword1=%20&keyword2=%20 藍田鯉安苑成「居屋王」]</ref>。大廈設有4部[[日立製作所|日立]]製[[升降機|升降機]]，每層均有兩部直達。此外，於2002年落成的瓊軒苑，亦是香港最後一個採用1987年版設計新十字型大廈的屋苑。值得一提，本屋苑爲全港唯一提供原裝一房單位的新十字型大廈，更爲全港唯一設有附屬平台單位的居屋屋苑（位於29樓09室）。

== 設施 ==
瓊軒苑內只設兩層[[停車場|停車場]]，提供43個有蓋或露天停車位，其餘設施需依靠鄰近的屋邨及區內的公共設施。區內設有[[蒲崗村道|蒲崗村道]][[學校村|學校村]]、[[東九龍分科診所|東九龍分科診所]]、[[斧山道運動場|斧山道運動場]]、[[斧山道游泳池|斧山道游泳池]]、[[斧山公園|斧山公園]]／[[南蓮園池|南蓮園池]]、[[牛池灣文娛中心|牛池灣文娛中心]]及大型購物商場[[荷里活廣場|荷里活廣場]]等。

== 沙士宿舍 ==
在2003年香港流行[[嚴重急性呼吸系統綜合症|-{zh-hans:非典; zh-hk:沙士;}-]]期間，瓊軒苑在[[房屋署|房屋署]]和前[[政府物料供應處|政府物料供應處]]（現稱[[政府物流服務署|政府物流服務署]]）協助下，曾被徵用作為[[醫管局|醫管局]]的前線醫護人員臨時宿舍，期間最少有329名醫護人員曾經遷入。當時單位曾加裝[[電視機|電視機]]、睡床及衣櫃等設施<ref>[http://www.legco.gov.hk/yr03-04/chinese/panels/hs/papers/hs0719cb2-3105-1c.pdf 天恩邨和瓊軒苑醫護人員臨時宿舍內電器和家具的處置事宜]</ref>，在屋苑推售前已回復原貌，臨時配置的傢俬與爐具，已於2004年拍賣，地板、窗簾等已經丟棄，而隨樓附送的[[冷氣機|冷氣機]]亦經過徹底[[消毒|消毒]]<ref>[http://paper.wenweipo.com/2008/07/26/HK0807260053.htm 瓊軒苑曾為「沙士宿舍」]</ref>。

== 物業管理 ==
現時的物業管理公司為[http://www.synergis.com.hk/html/tc/ 新昌管理集團]（2008年 - ）。
