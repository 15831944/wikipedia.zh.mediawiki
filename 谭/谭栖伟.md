'''谭栖伟'''（{{bd|1954年|8月|}}），[[重庆市|重庆市]][[石柱土家族自治县|石柱土家族自治县]]人。男，[[土家族|土家族]]。1973年3月参加工作，1973年11月加入[[中国共产党|中国共产党]]。[[中共重庆市委党校|中共重庆市委党校]][[区域经济学|区域经济学]]专业毕业。

1985年7月，任中共[[石柱土家族自治县|石柱土家族自治县]]委副书记、县长。1989年9月，任中共[[石柱土家族自治县|石柱土家族自治县]]委书记。1991年11月，任四川省[[黔江地区|黔江地区]]行署副专员。1993年9月，任中共黔江地委委员、行署副专员。1997年3月，任中共重庆市[[南岸区|南岸区]]委副书记、区长。2001年12月，任中共南岸区委书记。2006年3月，任[[重庆市人民政府|重庆市人民政府]]副市长。2013年1月，任[[重庆市人大常委会|重庆市人大常委会]]副主任<ref>{{cite news|title=重庆市人大常委会副主任谭栖伟接受调查|url=http://district.ce.cn/newarea/sddy/201405/03/t20140503_2751780.shtml|accessdate=2016-07-03|newspaper=中国经济网}}</ref>。

2013年，当选[[第十二届全国人民代表大会|第十二届全国人民代表大会]]重庆地区代表<ref>{{cite news|title =全国人大代表信息-谭栖伟|url =http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=122099|accessdate =2013-04-05|newspaper =全国人大网|deadurl =yes|archiveurl =https://web.archive.org/web/20140503185035/http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=122099|archivedate =2014-05-03}}</ref>。因涉嫌严重违纪违法，2014年5月15日，重庆市第四届人大常委会第九次会议决定罢免其[[第十二届全国人民代表大会代表|第十二届全国人民代表大会代表]]职务<ref>{{cite news|title=第十二届全国人民代表大会常务委员会代表资格审查委员会关于个别代表的代表资格的报告（2014年6月27日第十二届全国人民代表大会常务委员会第九次会议通过）|url=http://www.npc.gov.cn/wxzl/gongbao/2014-08/22/content_1879698.htm|accessdate=2016-07-03|newspaper=中国人大网}}</ref>。

2014年5月3日，涉嫌严重违纪违法正接受组织调查<ref>{{cite web |url = http://news.163.com/14/0503/11/9RAN3UA100014JB5.html |language=zh-cn |title = 重庆市人大副主任谭栖伟涉严重违纪违法被调查 |publisher= 中央纪委监察部网站 |date=2014-05-03 |accessdate= }}</ref>。据称谭落马是因在中央巡视组巡视期间受到举报<ref>{{cite web |url = http://news.163.com/14/0504/09/9RD1EGP20001124J.html |language= zh-cn |title=重庆人大副主任谭栖伟被查 或涉赵红霞案 |publisher= 华商网 |date=2014-05-04 |accessdate= }}</ref>。2014年9月，谭栖伟因利用职务上的便利为他人谋取利益，本人或通过其妻收受巨额贿赂，上述行为已构成严重违纪违法，并涉嫌犯罪，[[中纪委|中纪委]]决定给予谭栖伟开除党籍、开除公职处分，将其涉嫌犯罪问题及线索移送司法机关依法处理<ref>[http://www.ccdi.gov.cn/ajcc/201409/t20140902_27073.html 重庆市人大常委会原副主任谭栖伟被"双开"] {{webarchive|url=https://web.archive.org/web/20141214154430/http://www.ccdi.gov.cn/ajcc/201409/t20140902_27073.html |date=2014-12-14 }}</ref>。

2015年4月10日，经最高检指定，谭栖伟受贿案由[[河北省人民检察院|河北省人民检察院]]侦查终结后移送[[河北省|河北省]][[衡水市|衡水市]]人民检察院审查起诉，[[河北省|河北省]][[衡水市|衡水市]]人民检察院已向衡水市中级人民法院提起公诉<ref>{{cite web |url = http://news.163.com/15/0411/08/AMTH4SP300014AED.html |language=zh-cn |title = 河北检察机关对谭栖伟涉嫌受贿案提起公诉 |publisher= 正义网-检察日报 |date=2015-04-11 |accessdate= }}</ref>。

据河北省衡水市中级人民法院官方微博消息，河北省衡水市中级人民法院今日公开宣判重庆市人大常委会原副主任谭栖伟受贿案，认定被告人谭栖伟犯受贿罪，判处有期徒刑十二年，并处没收个人财产人民币一百万元。法院审理查明：谭栖伟直接或间接多次非法收受他人所送的财物，共计折合人民币1143万余元。案发后，赃款及孳息全部追缴<ref>{{cite web |url = http://www.chinacourt.org/article/detail/2016/01/id/1783870.shtml |title = 重庆市人大原副主任谭栖伟受贿1143万获刑12年 |accessdate = 2016年1月7日 |publisher = 中国法院网}}</ref>。
