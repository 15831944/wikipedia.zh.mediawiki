'''谭斌'''（{{bd|1942年|8月||}}）原名'''谭力夫'''，生于[[陕西|陕西]][[延安|延安]]，原籍[[湖南|湖南]][[资兴|资兴]]，[[中华人民共和国|中华人民共和国]]政府官员，[[中华人民共和国文化部|文化部]]办公厅主任，[[故宫博物院|故宫博物院]]党委书记兼副院长。[[文革|文革]]期间因创作出鼓吹“[[血统论|血统论]]”的对联而名声大噪。

==生平==
[[文革|文革]]初期，1966年6月谭力夫加入[[中国共产党|中国共产党]]。1966年7月29日，谭力夫贴出署名“鬼见愁”的著名对联：上联“老子英雄儿好汉”，下联“老子反动儿混蛋”，横批“基本如此”（后来对联的横批从“基本如此”演变为“绝对如此”）。该对联鼓吹“[[血统论|血统论]]”，意思是父亲是高级干部，儿子必是好汉；父亲是“[[牛鬼蛇神|牛鬼蛇神]]”，儿子必天生反动。该对联一贴出便在全中国迅速传播，引起轰动。对联贴出仅两天，8月2日[[中央文革小组|中央文革小组]]组长[[陈伯达|陈伯达]]在接见[[红卫兵|红卫兵]]时便指出这是“血统论”，8月6日[[江青|江青]]建议将该对联改成“老子英雄儿接班，老子反动儿背叛。”10月9日，[[周恩来|周恩来]]表示“谭力夫讲话是典型的形'左’实右。”10月24日[[毛泽东|毛泽东]]说：“学生有些出身不大好的，难道我们都出身好吗？”除了对联外，8月12日，谭力夫又和他人共同写出[[大字报|大字报]]《从对联谈起》，提出要将该对联的内容作为[[中国共产党|中国共产党]]的阶级路线推行，“提升为政策，上升到本本条条”。8月20日，谭力夫在其所在的[[北京工业大学|北京工业大学]]的学生辩论会上，就如何对待[[文化大革命工作组|工作组]]发表长篇发言，该发言事后作为《谭力夫讲话》迅速流传，引起全国震动<ref name=shoucang/>。

1966年10月16日，陈伯达在中央工作会议上不点名批判谭力夫说：“中央通过了《[[中国共产党中央委员会关于无产阶级文化大革命的决定|十六条]]》不过12天光景，有一个大学的什么文革筹委会委员，就跳出来发表讲话，对抗《十六条》。这个讲话传得很广。有人对毛主席亲自主持制定的中央决定没有一点兴趣，但对那篇讲话却视若至宝，为之翻印，为之广播。”后来，[[康生|康生]]、江青、[[张春桥|张春桥]]、[[姚文元|姚文元]]等中央文革小组成员也多次在各类场合不点名批判谭力夫<ref name=shoucang/>。

1966年12月18日，谭力夫被逮捕关进监狱。为遏制“血统论”在全国的发展，中央文革小组准备在北京开万人批斗大会，彻底批判“血统论”，批斗谭力夫。后此事被周恩来出面干预，制止了对谭力夫的公开批判。经周恩来与中央文革小组多次交涉，1967年5月29日谭力夫获释出狱。此后，在整个文化大革命期间，谭力夫再未参加任何群众组织，变成了“逍遥派”<ref name=shoucang/>。

1968年7月自[[北京工业大学|北京工业大学]]无线电系毕业后，1969年谭力夫被分配到某军垦农场接受再教育，1970年应征入伍。他先后在[[内蒙古|内蒙古]][[集宁|集宁]]、[[山西|山西]][[大同|大同]]、[[北京|北京]]等地[[中国人民解放军|中国人民解放军]]部队服役。后以中国人民解放军[[大校|大校]]军衔奉调转业到国务院直属单位从事经济、人事管理工作。1989年7月自[[北京大学|北京大学]]法律系函授毕业。1996年2月，被国务院定为享受[[政府特殊津贴|政府特殊津贴]]的有突出贡献的文化行政管理专家。他是高级工程师。1994年4月，任[[中国国家图书馆|北京图书馆]]党委书记兼常务副馆长。1996年7月，任[[中华人民共和国文化部|文化部]]办公厅主任<ref name=shoucang/>。1997年10月到2003年9月，任[[故宫博物院|故宫博物院]]党委书记兼副院长<ref name=shoucang>{{Cite web |url = http://zlk.sssc.cn/30/24309/ |title = 谭力夫 |author =  |date =  |publisher = 盛世收藏网 |language =  |accessdate = 2018-08-15 |quote =  }}</ref><ref name=tuandui>{{Cite web |url = http://www.dpm.org.cn/about/history_leader.html |title =  历任团队|author =  |date =  |publisher =故宫博物院  |language =  |accessdate = 2018-08-15 |quote =  }}</ref><ref>{{Cite web |url =http://collection.sina.com.cn/cjrw/20131031/1444131905.shtml  |title =  执掌故宫博物馆的人：历任的五位院长|author =  |date =2013-10-31  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。
