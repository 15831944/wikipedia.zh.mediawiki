'''谭志桂'''{{BD|1962年|11月}}，[[浙江|浙江]][[上虞|上虞]]人，1982年8月参加工作，1984年9月加入[[中国共产党|中国共产党]]。浙江省委党校研究生学历。曾任嵊州市委书记，绍兴市委副书记，现任绍兴市人大常委会主任。<ref name="Bio1">{{Cite web |url=http://www.sx.gov.cn/art/2017/4/14/art_53023_1114674.html |title=绍兴市人大常委会主任、副主任简历 |accessdate=2018-01-20 |author=绍兴市人大常委会 |date=2017-04-14 }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 简历 ==
曾任共青团绍兴市委工农青年部副部长；共青团绍兴市委常委、工农青年部副部长、部长；共青团绍兴市委副书记；嵊县临城区委副书记（挂职）；椒江市市长助理（挂职）；绍兴市水利电力局副局长、党组成员；绍兴市水利水电局局长、党组副书记、书记；嵊州市委副书记、代市长、市长；嵊州市委书记、人大常委会主任；绍兴市委常委、宣传部部长；绍兴市委常委、诸暨市委书记；绍兴市委副书记、政法委书记；绍兴市人大常委会党组书记等职。<ref name="Bio2">{{Cite web |url=http://district.ce.cn/newarea/sddy/201704/14/t20170414_21959952.shtml |title=绍兴市八届人大常委会主任、副主任简历(主任谭志桂) |accessdate=2018-01-20 |date=2017-04-14 |publisher=中国经济网地方党政领导人物库}}</ref>

现任绍兴市第八届人民代表大会常务委员会主任。
