'''谭其蓁'''（{{bd|1888年||1959年|10月27日|catIdx=Tan譚其蓁}}）， 字砺陶，[[四川省|四川省]][[滎經縣|滎經縣]]人。[[中华民国|中华民国]]政治家，曾任[[第一屆立法委員|第一屆立法委員]]

== 生平<ref>我的父親譚勵陶(其蓁)，譚儁昌，中國人民政治協商會議四川省滎經縣委員會文史資料工作委員會.《滎經文史資料選輯》第3輯.，1989年12月第1版.第52-63頁</ref> ==
* 少時就讀於民立第一小學，後入華陽學堂(華陽縣中的前身)，與[[張群|張群]]、文澄等同班。
* 光緒三十四年（1908年）考入四川省城高等學堂本科
* 宣統三年（1911年）五月選派幼年生<ref>清華大學校史研究室.清華大學史料選稿第四卷解放戰爭時期的清華大學（1946--1948）.清華大學出版社，1994年04月第1版.第637頁</ref>，第三批庚子赔款留美生，入北京游美預備學堂，同年赴美国[[密西根大学|密西根大学]](Ann Arbor， MI)留学<ref name=boxer> 陈学恂、田正平. 《第三次庚子赔款留美学生名单（部分）》（原载《清华学校一览》1917年，转录自刘真主编《留学教育》第1册第194-196页）. 《中国近代教育史资料汇编 留学教育》. 上海教育出版社. 1991:  200-201.</ref> 。于1915年获密西根大学政治学士<ref name=boxer3> 1909年至1911年间凭借庚子赔款赴美国留学的三批学生名单 http://zh.wikipedia.org/wiki/%E7%AC%AC%E4%B8%80%E6%AC%A1%E5%BA%9A%E5%AD%90%E8%B5%94%E6%AC%BE%E7%95%99%E7%BE%8E%E5%AD%A6%E7%94%9F%E5%88%97%E8%A1%A8#1911.E5.B9.B4.E7.AC.AC.E4.B8.89.E6.89.B9</ref>； 于1916年获密西根大学经济硕士 <ref name=boxer3/>。
*作为公派留学生，心理倾向反满革命， 支持辛亥革命 <ref> 一张中华民国金币券承载的历史信息 (“辛亥革命前所购 纪念品 谭其蓁”) {{cite web |url=http://www.hnswjst.com/content_view.asp?viewcode=24229~52428045 |title=存档副本 |accessdate=2012-08-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304200701/http://www.hnswjst.com/content_view.asp?viewcode=24229~52428045 |archivedate=2016-03-04 }} </ref>
* 返國後先後任四川省財政廳擔任秘書及成都東門統捐局長
* 又在四川當時的高等學校、諸如四川法政、商業、工業等專門學校講授「銀行論」「英文」等。
* 派任雅屬地區任省議員選舉的監督
* 民國十一年（1922年）全國各省開展聯省自治運動，任四川省憲起草委員會委員。
* 民國十四年（1925年），川軍第三軍軍長[[劉成勛|劉成勛]](禹九)在雅安成立西康屯懇使署(兼管民政事宜)，擔任屯署財務處長、顧問等職。劉易幟就任國民革命軍第二十三軍軍長：委為政治部主任。
* 民國十八年（1929年），任公立[[四川大學|四川大學]]外國文學院院長<ref>四川大學校史編寫組編.《四川大學史稿》.四川大學出版社，1985年10月第1版.第378頁</ref>
* [[西康省|西康省]]参议会副参议长<ref> 一九四六年雅属事变与行政检讨会 http://www.hoplite.cn/templates/hpzc2nz0083.html</ref>
* 民國三十六年（1947年）当选为第一屆中华民国[[立法委员|立法委员]]，并任中华民国[[立法院|立法院]]第一届第一会期边政委员会召集人<ref>  中華民國立法委員名鑑資料庫  http://npl.ly.gov.tw/do/www/committeeList?appdateId=0101</ref>。
* 1955年，任四川省文史研究館研究員 <ref> 四川省人民政府参事室 四川省文史研究馆 >> 参事馆员之窗 >> 馆员 >> 历任馆员 谭其蓁馆员 http://www.sccw.gov.cn/?thread-16282-1.html </ref>

其兄谭其茳 (1880.3.26—1941.7.23)， 字芗陶，隐名创去，四川荣县人。1902年补诸生，不久考入四川大学堂。1905年赴[[日本|日本]]攻读法政等， 加入[[同盟会|同盟会]]。1909年回国后，任西安优级师范学堂教员，兰州师范学校教员，创办芗石书社， 出售梁启超编辑的《新民丛报》、《庸言》等刊物， 吸引学堂学生、青年士人阅读。1911年 率众起义于甘肃任秦川都督， 1912年任甘肃临时政府交通司次长。1913年任国民党四川支部长。因介绍成都军官学校学生入国民党，被捕入狱。出狱后从事讨袁活动。后任潼关护国军政府政务司司长，四川督军陈宦政治顾问。1918年任四川靖国军总司令熊克武秘书。1919年当选为四川省议员。1922年任川东边防军训练处总办。1923年任西川道尹等。抗战爆发后，历任川康绥靖公署顾问，西康省临时参议会议长，国民党西康省执行委员会代主委等<ref> 中国留学生大辞典.南京：南京大学出版社，1999.
