'''谭力'''（{{bd|1955年|10月||catIdx=T谭}}），[[重庆|重庆]]人。[[重庆师范大学|重庆师范大学]]政史系政史专业毕业，大学学历。1972年2月参加工作，1987年6月加入[[中国共产党|中国共产党]]，先后在[[四川省|四川省]]和[[海南省|海南省]]任职。譚力在四川任職时间超過三十年，因獲[[周永康|周永康]]提拔而青雲直上<ref name="zykqx">{{cite web|url=http://www.hket.com/eti/article/2edc923d-9db6-436b-8d0b-1cd7e6ea70b4-169737?inews=172473|title=周永康親信 海南副省長譚力落馬|publisher=《[[香港经济日报|香港经济日报]]》|date=2014年7月8日|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20150924030727/http://www.hket.com/eti/article/2edc923d-9db6-436b-8d0b-1cd7e6ea70b4-169737?inews=172473|archivedate=2015年9月24日}}</ref>。2014年7月譚力被免职调查，他是繼海南省前副省長[[冀文林|冀文林]]之後，海南省2014年落马的第二位副省級官員，也是又一名具有四川背景的被调查官員<ref name="zykqx"/>。

== 官场生涯 ==
谭力1955年10月出生于重庆市，16岁时以知青身份到四川省长宁县桃坪公社兴龙一队插队。一年半后，他进入四川省高县师范学校读书，毕业后在小学教书，后来被调入县文教局、县委宣传部工作。1973年中国[[普通高等学校招生全国统一考试|恢复高考]]后，谭力考入重庆师范学院，毕业后历任[[中共四川省委|中共四川省委]]第二党校行政管理学教研室教师、副主任、主任，中共[[郫县|郫县]]县委书记、县长，成都市市长助理。1998年6月至2001年3月担任市委宣传部长、1998年10月任市委常委。2001年3月在邓小平诞辰100周年筹备会上，出任中共[[广安|广安]]市委书记、市人大常委会主任，提出了“致富思源,共建广安”的口号,秉承邓小平“一定要建设好广安”的夙愿,开始大刀阔斧地施展计划,并成功举办了小平百年诞辰的盛典。2004年出任中共绵阳市委书记等职。

在[[邓小平|邓小平]]家乡广安任职期间，谭力发起“我为小平家乡种植一棵树”计划，邀请很多中央领导到广安植树。虽然邓小平生前亲自要求家乡“按政策办事，不搞特殊照顾”，但谭力2013年毫不讳言的称要把广安的政治优势转化为发展优势<ref>{{cite web|url=http://cpc.people.com.cn/n/2013/1021/c69113-23274710.html|title=谭力：把政治优势转化为发展优势 把广安建设得更美好|publisher=广安日报|date=2013年10月21日|language=zh-hans}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。谭力于2008年[[汶川地震|汶川地震]]期间，被拍下一张喜笑颜开的照片，被网友斥为“'''谭笑笑'''”<ref name="zbt">{{cite web|url=http://www.zaobao.com.sg/realtime/china/story20140708-363652|title=海南副省长谭力被查 传涉周永康案|publisher=联合早报|date=2014年7月8日 | language = zh-hans }}</ref>。2009年3月调往海南，任省委常委、宣传部部长<ref>{{cite web |accessdate=2012-04-11 |url = http://news.qq.com/a/20090331/000285.htm |title = 原绵阳市委书记谭力任海南省委常委、宣传部长|publisher=[[腾讯网|腾讯网]]}}</ref>，2010年1月任[[海南省人民政府|海南省人民政府]]副省长<ref>{{cite web |accessdate=2012-04-11 |url = http://news.sina.com.cn/c/2010-01-21/113619516023.shtml|title=谭力被任命为海南省人民政府副省长|publisher=[[新浪网|新浪网]]}}</ref>。2011年2月起兼任[[海南国际旅游岛先行试验区|海南国际旅游岛先行试验区]]工委书记。<ref>{{cite web|url=http://news.qq.com/a/20140708/066212.htm|title=海南常委副省长谭力被查 为今年第17只老虎|publisher=腾讯网|date=2014-07-08| language =zh-hans }}</ref>

2014年7月8日，海南省委常委、副省长的谭力因涉嫌严重违纪违法，接受组织调查。2015年11月24日，谭力涉嫌受贿一案，由最高人民检察院指定广东省人民检察院侦查终结后，移送广东省广州市人民检察院审查起诉。2015年11月，广州市人民检察院已向广州市中级人民法院提起公诉。

== 腐败被查 ==
2014年2月下旬，[[法国国际广播电台|法国国际广播电台]]获取消息<ref>{{cite web|url=http://www.chinese.rfi.fr/%E4%B8%AD%E5%9B%BD/20140219-%E4%BC%A0%E6%B6%89%E5%86%80%E6%96%87%E6%9E%97%E6%A1%88-%E6%B5%B7%E5%8D%97%E7%9C%81%E5%A7%94%E5%AE%A3%E4%BC%A0%E9%83%A8%E9%95%BF%E8%B0%AD%E5%8A%9B%E8%A2%AB%E9%99%90%E5%88%B6%E5%87%BA%E5%A2%83|title=传涉冀文林案 海南省委宣传部长谭力被限制出境|publisher=法国国际广播电台|date=2014年2月19日| language =zh-hans }}</ref>称谭力因涉及[[冀文林|冀文林]]以及[[周永康案|周永康案]]被限制出境。<ref name="zbt"/>《[[香港经济日报|香港经济日报]]》称譚力與因涉黑被判死的四川前富豪[[刘汉_(四川商人)|刘汉]]關係密切<ref name="zykqx"/>。

2014年4月，知情人向[[明鏡新聞網|明鏡新聞網]]透露称譚力在担任綿陽市委書記期间以綿陽科學城產業基金的名義将资金转給[[劉雲山|劉雲山]]之子劉樂飛，劉樂飛又將这笔钱投給[[江澤民|江澤民]]孫子[[江志成|江志成]]。這些關係虽然在一定程度上保護了譚力，但隨著周永康更多問題被揭露，譚力涉嫌的犯罪问题也被不斷舉報。<ref>{{cite book| author = 高楠楠| title = 《老虎》| publisher = 財大出版社| year = 2014| isbn = 1630322814| pages = 第28页| url = http://www.mirrorbooks.com/MIB/news/news.aspx?ID=N000022445| access-date = 2014-07-08| archive-url = https://web.archive.org/web/20140714225626/http://www.mirrorbooks.com/MIB/news/news.aspx?ID=N000022445| archive-date = 2014-07-14| dead-url = yes}}</ref>

2014年7月8日，中央纪委监察部发布海南省委常委、副省长谭力因涉嫌严重违纪违法被组织调查的消息。[[BBC中文网|BBC中文网]]查证谭力与前政治局常委、前四川省委书记[[周永康|周永康]]多有交集<ref>{{cite web|url=http://www.bbc.co.uk/zhongwen/simp/china/2014/07/140708_official_investigated_tan_li.shtml|title=海南省副省长谭力涉违法违纪遭调查|publisher=BBC中文网|date=2014年7月8日| language =zh-hans }}</ref>。

2014年7月30日，海南省第五届人民代表大会常务委员会第九次会议30日表决通过了有关人事任免事项，免去了谭力海南省人民政府副省长职务。<ref>[http://news.ifeng.com/a/20140730/41369563_0.shtml 谭力被免去海南省副省长职务]</ref>9月30日，中纪委通报谭力因受贿被双开<ref>{{cite web |url = http://www.ccdi.gov.cn/xwtt/201409/t20140930_31671.html |title = 海南省原省委常委、副省长谭力严重违纪违法被开除党籍|accessdate = 2014年9月30日 |publisher = 中央纪委监察部网站}}</ref>，2015年11月24日，谭力涉嫌受贿案由[[广东省|广东省]][[广州市|广州市]][[人民检察院|人民检察院]]审查起诉<ref>{{cite web |url = http://news.xinhuanet.com/legal/2015-11/24/c_128462525.htm |title = 海南原常务副省长谭力涉嫌收受巨额贿赂被公诉|accessdate = 2015年11月24日 |publisher = 新华网}}</ref>。该案于2016年1月27日在广州市中级人民法院开庭审理，检方指控，2001年至2014年，谭力利用职务之便，为他人谋取利益，非法收受财物共计折合人民币8625万余元<ref>{{cite web |url = http://news.ifeng.com/a/20160128/47264004_0.shtml|title =海南省原副省长谭力受审 |accessdate = 2016年1月28日 |publisher = 京华时报 }}</ref>。2016年10月13日，广州市中级人民法院宣判，谭力犯受贿罪，判处无期徒刑，剥夺政治权利终身，并处没收个人全部财产；对谭力受贿所得财物予以追缴，上缴国库<ref>{{cite web |url = http://news.sina.com.cn/c/nd/2016-10-13/doc-ifxwvpaq1174111.shtml|title =海南原副省长谭力受贿8625万 一审被判无期 |accessdate = 2016年10月13日|publisher = 央视 }}</ref>。

2016年3月3日，成都路桥工程股份有限公司涉嫌单位行贿罪一案在广州市中级人民法院开庭审理。检方指控称：成都路桥向谭力共计行贿折合人民币超过6000万元，这些钱分别以工程费、劳务费等诸多名义从该公司提取，并进入谭力个人腰包；部分款项是以房产、佛像、[[象牙|象牙]]等形式行贿<ref>{{cite web |url = http://companies.caixin.com/2016-03-03/100915604.html|title = 成都路桥涉嫌行贿原海南副省长逾六千万元 |accessdate = 2016年3月3日|publisher = 财新网 }}</ref>。
