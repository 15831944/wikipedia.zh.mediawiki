《'''极品飞车：无间风云'''》（Need for Speed: Undercover）是著名的[[竞速游戏|竞速游戏]][[极品飞车系列|极品飞车系列]]的第12代产品，由[[EA_Black_Box|EA Black Box]] 开发，[[美国艺电|美国艺电]]（EA）发售于2008年11月<ref name="Riccitiello1">{{cite web |url=http://www.eurogamer.net/article.php?article_id=153998|title=New Need for Speed game unveiled|date=2008-06-18 |author=Ellie Gibson|publisher=[[Eurogamer|Eurogamer]]|accessdate=2008-06-18}}</ref><ref name="Riccitiello2">{{cite web |url=http://www.gamesindustry.biz/articles/riccitiello-we-were-torturing-vancouver-studio|title=Riccitiello: We were torturing Vancouver studio|date=2008-06-18|author=Mark Androvich|publisher=Gamesindustry.biz|accessdate=2008-06-18}}</ref><ref name="tech2">{{cite web |url=http://www.tech2.com/india/news/games/new-nfs-and-skate-announcements-soon/30961/0|title=New NFS and Skate Announcements Soon?|date=2008-03-08|author=Avinash Bali|publisher=Tech2.com India|accessdate=2008-03-09}}</ref>，有[[PlayStation_2|PlayStation 2]]、[[PlayStation_3|PlayStation 3]]、[[Xbox_360|Xbox 360]]、[[Wii|Wii]]、[[Microsoft_Windows|Microsoft Windows]]、[[PSP|PSP]]、[[任天堂DS|任天堂DS]]<ref name="NGS2008">{{cite web |url=http://www.nokia.com/A4136001?newsid=1264321|title=The faces of innovation gather at annual Nokia Games Summit|publisher=Nokia|accessdate=2008-10-29}}</ref>和[[手机|手机]]平台的版本；随后于2009年4月27日发行了[[iPhone_OS|iPhone OS]]版。EA称该游戏在八个平台上共销售520万份<ref>http://www.digitalspy.co.uk/gaming/a145624/ea-reconfirms-massive-global-layoffs.html</ref>。《极品飞车：无间风云》也是在第六代游戏主机（[[PlayStation_2|PlayStation 2]]）上发行的最后一款极品飞车系列游戏。

==背景設定==
《极品飞车：无间风云》的特色是与[[午夜俱乐部|午夜俱乐部]]系列相似的一幅新的[[开放世界|开放世界]]地图，包括共计109英里长的道路和庞大的道路系统，成为目前为止EA创造的最大的极品飞车的“世界”。游戏环境中包括四个区：棕榈港、新月港、黄金海岸山脉、日落溪谷。部分区和《極速快感：全民公敵》相同，只是更换了位置。这四个区组成了一座坐落于湾岸的城市三城湾。道路系统包括四座跨海大桥，按顺时针顺序分别是：主盖伊通道（通过人工岛连接日落溪谷和新月港的大洋高速公路）。

==遊戲特色==
警察系统和《[[极品飞车：无间追踪|极品飞车：无间追踪]]》和《[[极品飞车：生死卡本谷|极品飞车：生死卡本谷]]》相似。屏幕下方有类似的表示追捕情况的条形，右半为蓝色（PS2和Wii版为绿色）表示“逃脱”，左半为红色，表示“被捕”，因玩家速度和警察接近程度而不断变动。同样保留了逃脱追捕后的“降低威脅时间”、威脅度级别、瞬間反應和追擊中斷物件。同样，警车范围从基本的城市巡逻车到联邦跟踪车+警用重型SUV、直升机。但与前代游戏不同的是，在这一代游戏中，低威脅度同样可能出现高级警车，反之亦如此，警车种类并不完全取决于威脅度，而是更大程度上取决于wheelman级别：即使威脅度很低，但wheelman等级很高的情况下，联邦警力仍然会立即参与追捕<ref name="1UP rv">{{cite web |title=Need for Speed: Undercover Review from 1UP|url=http://www.1up.com/do/reviewPage?cId=3171367&p=4|author=John Davison|publisher=1UP.com|date=November 18, 2008|accessdate=2008-11-19}}</ref>。路障、钉刺带、[[PIT_maneuver|PIT maneuver]]等警察常用的策略得以发扬光大，但部分策略只有联邦警察才使用。关于玩家驾驶警车的内容，生涯模式中有一关要求玩家偷走一部警车，除此之外玩家只能在一个称为“警与匪”（“Cops and Robbers”）的多人游戏中驾驶警车

车辆损毁系统回到《[[极品飞车：生死卡本谷|极品飞车：生死卡本谷]]》中，但与第十一代游戏《[[极品飞车：街头狂飙|极品飞车：街头狂飙]]》不同，毁损效果只有视觉效果，不会影响车辆性能。不过，在新增的“高速路对战”模式和生涯模式的部分任务中，车仍然可以彻底损毁（“Totaled”）。在这些任务中，会出现车辆受损条，指示玩家车辆的受损情况。这些任务的首要目标是将指定车辆在未彻底损坏的情况下送到指定地点。其他情况下，毁损都会在每场比赛或追捕之后自动免费修复，而不需要像《街头狂飙》那样要由玩家付出金钱或“修理标志”亲自维修<ref name="Gamespot rv">{{cite web |title=Gamespot: Need for Speed Undercover Review|url=http://www.gamespot.com/xbox360/driving/needforspeed/review.html|author=|publisher=Gamespot|date=|accessdate=2009-01-27}}</ref>。

车辆定制与《极品飞车：街头狂飙》相似，提高了图像级别和细节，改进了新调色盘和“matte”漆。作为奖励，EA在游戏中添加了与《极品飞车：无间追踪》中的BMW M3 GTR相似的标贴

玩家通过参与任务、赢得比赛或警察追逐中展示优良驾驶技术推动游戏剧情发展，获得wheelman点数（声望值）；而这会解开后续任务，为游戏加入类似RPG的元素。

游戏中也有宣传T-Mobile Sidekick的[[置入性行销|置入性行销]]。

==剧情==

这款游戏发行之前几天，EA公开了剧情的部分细节。一段发布的视频展示玩家正在逃离三城湾警察局（Tri-City Bay Police Department，缩写为TCBPD）抓捕的场面<ref name="IGN: Info" />，另一段则展示玩家得到Chase Linh侦探和Lt. Jack Keller的帮助<ref name="IGN: Info" />，还有一段则是玩家与一名由歌手、女演员[[Christina_Milian|Christina Milian]]扮演的Carmen Mendez交友。

另外还公开了六名案犯角色：

* Chau Wu，由Daniel D. Lee（uncredited）扮演
* Gregory "G-Mac" MacDonald（前三城湾警局卧底警察，who's gone native），由电视连续剧[[The_Shield|The Shield]]中的演员[[David_Rees_Snell|David Rees Snell]]扮演
* Rose Largo，由演员Heather Fox扮演
* Hector Maio，由《[[越狱|越狱]]》中的演员[[Kurt_Caceres|Kurt Caceres]]扮演
* Freddy "Nickel" Rogers，由演员Lawrence B. Adisa扮演
* Zack Maio由女演员[[Jessica_Alba|Jessica Alba]]的兄弟[[Joshua_Alba|Joshua Alba]]扮演

故事中玩家被设定为游戏中虚构的金融城市“三城湾”的一名打入当地地下犯罪团伙的警员，玩家需要完成危险的“工作”并参与赛车，以此渗入并最终端掉一个包括非法街头赛车者和偷车贼的国际性犯罪走私团伙。玩家与警方的唯一联系是[[Maggie_Q|Maggie Q]]扮演的Chase Linh，后期是[[Paul_Pape|Paul Pape]]扮演的Lt. Jack M. Keller。经过一段时间后，玩家开始证明自己是优秀的赛车手和wheelman。随着卧底工作的深入，玩家不得不解决掉自己结交的罪犯“朋友”，却无意中陷害了一个犯罪团伙的头目Chau Wu。Chau Wu于是电告玩家只有一件事可以挽回：有一台提供为他免刑证据的车被偷走，Chau Wu认为窃贼是G-MAC（实际上是Hector和Zack Maio在不知道车主是谁的情况下偷走的），希望要回这台车。卡门害怕Chau Wu追赶他，要玩家带走[[宝马M6|宝马M6]]（PS2和Wii版中是[[保时捷911_GT2|保时捷911 GT2]]），然而，Chau Wu发现他的车正在玩家手里。他给玩家打电话，要把Chase Linh扣为人质，直到玩家送回那台车。结果，玩家将车送回Chau Wu，希望他能释放Chase Linh，却发现她早已在Chau Wu手下工作。Chase Linh随后用一把从Chau Wu处找到的无声手枪杀死了Chau Wu和随从，并携带手头上Chau Wu的财物逃走，意图栽赃玩家为凶手。警察起初中计，追捕玩家，但Lt. Jack Keller要求警察停止对玩家的追捕，而是抓捕Chase Linh，并最终将其抓获。随后，Later, Lt. Keller告诉玩家，卡门被证实没有参与任何违法行为，抓捕Chase Linh的证据就是Chau Wu的存有关于码头车辆和其他犯罪活动的PDA。最后一幕中，卡门要求玩家开车将她送回大学，因为她正在大学就学。
