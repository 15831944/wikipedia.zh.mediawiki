《'''限制級戰警'''》（{{lang-en|'''''xXx'''''}}，或記作{{lang|en|'''''Triple X'''''}}）是一部2002年美國[[動作片|動作片]]，為[[羅伯·柯漢|羅伯·柯漢]]執導。由[[馮·迪索|馮·迪索]]、[[艾莎·阿基多|艾莎·阿基多]]、[[山繆·傑克森|山繆·傑克森]]和[[馬頓·索柯斯|馬頓·索柯斯]]主演。

電影於2002年8月9日在美國上映。儘管評價褒貶不一<ref name="review">{{cite web |title = ''xXx'' (2002) |url = http://www.rottentomatoes.com/m/xxx/ |work = [[Rotten_Tomatoes|Rotten Tomatoes]] |accessdate = January 10, 2010 }}</ref>，但在全球獲得了高達2.77億美元的票房收入<ref name="boxoffice" />。續集《[[限制級戰警2：極限公國|限制級戰警2：極限公國]]》於2005年上映。 2016年推出第二續集《[[限制級戰警：重返極限|限制級戰警：重返極限]]》(接續首集)

== 剧情 ==
桑德·凯吉（[[范·迪塞尔|范·迪塞尔]]）是一个爱好极限运动的男人，他被奥古斯都·吉本斯（[[塞缪尔·L·杰克逊|塞缪尔·L·杰克逊]]）发掘后，推荐到了[[美国国家安全局|美国国家安全局]]做了一个特工。经过刻苦的训练之后，凯吉被派往捷克首都布拉格刺杀犯罪分子尤里（[[马顿·索柯斯|马顿·索柯斯]]）。<ref>{{cite web |url= http://movie.douban.com/subject/1306737/|title= 极限特工|publisher= 豆瓣网|accessdate= 2002年|work= |language =zh-cn}}</ref>
