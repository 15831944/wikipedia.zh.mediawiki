《'''极限挑战'''》是由上海[[东方卫视|东方卫视]]所制作的一档大型励志综艺节目，其节目内容跟社会热点息息相关。每一集成員都需要通过不同的考验来闯关，闯关过程中会收到不同的指令及提示，帮助参与者获得提示。根据游戏设置，节目会加入不同的嘉宾，並最由韩国团队提供技术支持。第1至第4季的固定成員為[[孫紅雷|孫紅雷]]、[[黃磊|黃磊]]、[[黃渤|黃渤]]、[[王迅_(演员)|王迅]]、[[羅志祥|羅志祥]]、[[張藝興|張藝興]]；第5季起，孫紅雷、黃渤退出節目，[[迪麗熱巴|迪麗熱巴]]、[[雷佳音|雷佳音]]及[[岳雲鵬|岳雲鵬]]以固定成員身分加入<ref name="第5季轉變">[https://m.weibo.cn/detail/4350211897546936 《极限挑战5》2固定班底退出了]东方卫视极限挑战官方[[新浪微博|新浪微博]] 2019-03-15[2019-03-18]</ref>。

第一季节目共计12期，于2015年6月14日起每周日晚21点在东方卫视播出，其中8月16日，8月23日，8月30日节目停播三周。第二季节目共计13期，于2016年4月17日起每周日晚21点在东方卫视播出。第三季节目共计12期，于2017年7月9日起每周日晚22点在东方卫视播出，其中9月10日至10月29日节目停播八周，并于11月3日起改至周五22:00播出余下三期。第四季节目共计12期，于2018年4月29日起每周日晚20点30分在东方卫视播出。第五季节目預計于2019年5月12日起在东方卫视播出<ref name="S5">[https://www.bilibili.com/read/cv2362306/ 《极限挑战》第五季正式定档，将成为第二季度最晚开播节目！][[Bilibili|Bilibili]]專欄 2019-03-31[2019-04-07]</ref>。

== 节目争议 ==

=== 抄袭无限挑战节目内容问题 ===
[[韓國|韓國]][[MBC|MBC]]電視台已於2015年3月19日与[[中國|中國]][[中央電視台|中央電視台]]签下了输出節目《無限挑戰》版权的合同<ref>{{Cite web|url = http://chinese.joins.com/big5/article.do?method=detail&art_id=137330|title = MBC央視合作中國版《無限挑戰》 10月開播|accessdate = 2015-07-01|author = |date = |publisher = }}</ref>，而沒購買版權，於2015年6月首播的[[東方衛視|東方衛視]]節目《極限挑戰》卻與《無限挑戰》的遊戲環節極為相似，因而引發版權爭議。MBC在同年7月中发布了消息：「今年6月14日开始，中国东方卫视开始播出的综艺《极限挑战》与我台播放的《无限挑战》类似，疑似抄袭我台节目，这并不是中国版的《[[无限挑战|无限挑战]]》，只是个山寨版本」，并附上了MBC原版与《极限挑战》的截图<ref>{{Cite web|url = http://webptt.com/m.aspx?n=bbs/CN_Entertain/M.1437153107.A.791.html|title = 韓國MBC電視臺:《極限挑戰》抄襲|author = |date = 2015年7月18日|publisher = }}</ref><ref>{{Cite web|url = http://www.koreastardaily.com/tc/news/65296|title = MBC TV斥東方衛視《極限挑戰》抄襲：原樣複製《無限挑戰》|accessdate = 2015-07-17|author = |date = |publisher = }}</ref>，指《极限挑战》无论是在镜头切换还是字幕位置都跟《无限挑战》相似；同时指出：「从目前播出的节目来看，明顯包含了《无限挑战》的大量特辑节目内容，如第169-170集的『追著尾巴跑』、第110-112集的『拿著錢箱跑吧』、第406-407集的『極限打工』、第158-159集的『越獄吧青春痘』等环节」，還對节目有韩方人员参与表示震惊。不少觀眾看到《極限挑戰》之後也懷疑是抄襲了《無限挑戰》。

另外亦有消息指部份环节與[[韓國|韓國]]《[[Running_Man|Running Man]]》等節目相似，第五季第一期节目环节中明显抄袭了原节目第346期之内容。

而东方卫视的相關人员則指出：「不管是节目还是影视作品，不怕产品的创意雷同，这是非常常见的现象。但是就算同一个创意，不同的地域、不同的国籍、不同的明星、不同的导演、不同的社会环境、不同的职业分类、不同的规则设计，呈现出来的效果是不一样的。更多的节目其实丰富了观众选择的空间」。{{NoteTag|1=出自腾讯娱乐对东方卫视代表的采访。}}

至現在播出的第二季中，仍然有明顯抄襲《[[无限挑战|无限挑战]]》環節或主題的集數，在《极限挑战》依然受中國觀眾歡迎下，抄襲情況繼續引起觀眾熱議。

=== 郑则仕参与录制镜头被剪 ===
在第三季第三集中肥猫[[鄭則仕|郑则仕]]亦有参演并作为游戏关卡的NPC，然而在节目播出时，节目组却将郑则仕的镜头全部删除<ref>{{Cite news|url=http://ent.ifeng.com/a/20170725/42958859_0.shtml|title=郑则仕参加《极限挑战》录制镜头全被剪！|author=|date=|work=|accessdate=}}</ref>。有网友拍到录期间郑则仕与张艺兴互动领取任务卡。同时很少发微博的郑则仕，也在节目播出前罕见性的发了一条微博替极限挑战宣传，并晒出照片。

而在随后更新的[[优酷网|优酷]]网络完整揭秘版中，郑则仕仍无任何镜头。

=== 捆綁「跑男」宣傳 ===
由本節目第一季播出開始，由於與浙江衛視的王牌節目《[[奔跑吧兄弟|奔跑吧兄弟]]》（後更名為《[[奔跑吧|奔跑吧]]》）播出時間相近，兩檔節目經常被比較，且言論大多數以黑《奔跑吧》為主，每次宣傳都引來跑男粉的不滿，職責《極限挑戰》「捆綁《跑男》宣傳」、「倒貼《跑男》」，加上東方衛視粉絲與浙江衛視粉絲因收視率問題經常在百度貼吧罵戰，相關話題也成為了兩家電視台粉絲之間爭吵的導火索。
