'''江泽慧'''（{{bd|1938年|2月|||catIdx=江}}）女，生于[[江苏省|江苏省]][[江都县|江都县]][[仙女镇_(扬州市)|仙女庙镇]]（今[[扬州市|扬州市]][[江都区|江都区]]），祖籍[[安徽省|安徽省]][[宣城市|宣城市]][[旌德县|旌德县]][[江村|江村]]，[[中华人民共和国|中华人民共和国]]木材科学家。[[江泽民|江泽民]]之妹。

==生平==
江泽慧的父亲是[[江上青|江上青]]，[[江泽民|江泽民]]是江上青的[[嗣子|嗣子]]<ref>[http://news.sina.com.cn/c/2009-09-23/044418705938.shtml 江泽慧人民日报撰文纪念江上青牺牲七十周年，新浪，2009年09月23日]</ref>。1956年，江泽慧中学毕业报考大学，但是三个候选[[北京大学|北京大学]]、[[南京大学|南京大学]]、[[上海交通大学|上海交通大学]]均未录取。当时政策规定是由政府调配，江泽慧被调配入学[[安徽农学院|安徽农学院]]林业学，为此她倍感沮丧；但是三哥江泽民写信鼓励，并每月寄十元资助她<ref>{{cite book | language = zh-hans | author = [[罗伯特·劳伦斯·库恩|罗伯特·劳伦斯·库恩]]著，谈峥、于海江譯 | title = 他改变了中国——江泽民传 | url =  | date =2005年1月 | location = 上海 | publisher = 上海译文出版社 | id = | isbn = 7-5327-3655-5 |page=62}}</ref>。1960年6月，自安徽农学院林学系毕业。1962年加入[[中国共产党|中国共产党]]。历任安徽农学院林学系助教、讲师、副教授、教授，林学系党总支副书记、书记。1991年3月任安徽农学院副院长。1992年享受[[中华人民共和国国务院|国务院]][[政府特殊津贴|政府特殊津贴]]<ref name=jianli>{{Cite web |url = http://finance.sina.com.cn/roll/20040628/1622837548.shtml |title =中国林科院院长江泽慧简历 |author =  |date =2004-06-28  |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。

1993年2月起，任[[安徽省人大常委会|安徽省人大常委会]]副主任<ref name=jianli/>。1994年5月至1998年5月任[[安徽农业大学|安徽农业大学]]校长。1996年1月任[[中华人民共和国林业部|林业部]]党组成员（副部长级），后任[[国家林业局|国家林业局]]党组成员（副部长级）。1996年2月至2006年12月，任[[中国林业科学研究院|中国林业科学研究院]]院长、院分党组书记。后任中国林业科学研究院首席科学家、教授、[[博士生导师|博士生导师]]、[[博士后|博士后]]指导小组成员<ref name=lky>{{Cite web |url =http://www.caf.ac.cn/news/yqjj/200909/2009-09-24-10-40.html  |title =江泽慧（1996年2月至2006年12月任院长、院分党组书记）  |author =  |date =2009-09-24  |publisher =中国林业科学研究院  |language =  |accessdate =  |quote =  }}</ref>。
 
2009年时，她还正在担任[[全国政协人口资源环境委员会|全国政协人口资源环境委员会]]副主任，中国林业科学研究院图书馆馆长，国家林业局科技委副主任，[[中国花卉协会|中国花卉协会]]会长，[[中国林学会|中国林学会]]第一副理事长，[[国际木材科学院|国际木材科学院]]院士、中国木材学会常务副理事长，[[国际竹藤组织|国际竹藤组织]]董事会联合主席，国际木材解剖学会会员，[[中国科学技术协会|中国科学技术协会]]常务委员会委员，[[国务院学位委员会|国务院学位委员会]]林业工程学科评审组成员、《全国农业科技发展纲要》领导小组成员兼专家顾问组副组长、《林业科学研究》主编、中国现代科学全书《林业工程卷》主编<ref name=lky/>。

江泽慧长期从事森林利用学、木材学、木材解剖学的教学、科研、管理工作，曾主持或参加国家重大科技攻关项目、国家自然基金项目、攀登计划项目、省市科研项目21项（其中主持13项）。曾获国家、省部级科技进步奖12项（其中国家科技进步二等奖2项、三等奖1项，省部级科技进步一等奖3项、二等奖3项、三等奖3项）<ref name=lky/>。
