'''江东二路站'''是[[杭州地铁7号线|杭州地铁7号线]]的起讫站，位于[[杭州市|杭州市]][[萧山区|萧山区]]大江东产业集聚区青六北路江东二路。<ref>{{cite web |title=【城事】江东二路站、桥头堡站……大江东地铁站沿线这样布局！杭州轨道交通三期站点周边具体用地规划方案出炉！-大江东华数 |url=http://www.cclycs.com/m251479.html |website=大江东华数}}</ref>预计2020年12月31日完工<ref>{{cite web |title=关于杭州地铁7号线江东二路站施工期间 调整交通组织措施的通告 |url=http://www.hzpolice.gov.cn/Html/201808/07/ad2965bc-90ec-41e0-b115-86188f81b435.html |website=杭州公安局 |date=2018-08-29}}</ref>约2021年12月底随7号线一起启用。

== 车站结构 ==
江东二路站为地下二层岛式车站，车站总长463.425m，标准段宽21.9m，主体采用双柱三跨钢筋混凝土箱型框架结构，主体围护结构采用800mm厚地下连续墙。<ref>{{cite web |title=08-20杭州地铁七号线江东二路站地连墙顺利开工 |url=http://www.huahaieng.com/index.php/News/article/id/2291 |website=中铁上海工程局集团华海工程有限公司}}</ref>站后设有出入场线连接江东三路停车场。

== 历史 ==
2018年7月30日，开工建设。
