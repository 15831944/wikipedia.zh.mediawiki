'''江川路'''，是[[上海市|上海市]][[闵行区|闵行区]]南部的一条道路。东起[[沪闵路|沪闵路]]，西至西河泾东岸，以[[云南|云南]][[江川|江川]]为名<ref>{{Cite book|title=上海地名志|author1=陈征琳|author2=邹逸麟|author3=刘君德|author4=郭维武|author5=陈亦明|author6=赵永复|publisher=上海社会科学院出版社|year=1998年|isbn=7-80618-527-5|location=上海|pages=345|url=http://www.wanfangdata.com.cn/local/toFullText.do?item_id=2829921&gazetteers_id=FZ016021&item_num=014002003127}}</ref>。1958年修建，名'''一号路'''，全长7千米，1980年更名为江川路<ref>{{Cite book|title=闵行区地名志|author1=汪祖超|author2=沈金荣|author3=包妙祥|author4=何正渝|author5=俞建源|publisher=上海市社会科学院出版社|year=2000|isbn=7-80618-780-4|location=上海|pages=285}}</ref>。沿途有[[闵行经济技术开发区|闵行经济技术开发区]]。

== 历史 ==

1957年12月，[[上海市一届二次党代会|上海市一届二次党代会]]正式作出了“在上海周围建立卫星城镇，分散一部分工业企业、减少市区人口过分集中”的决定。此时，上海主要规划和建设了[[闵行区|闵行]]、[[嘉定区|嘉定]]、[[安亭镇|安亭]]、[[松江区|松江]]、[[吴泾镇|吴泾]]五个卫星城<ref>{{cite web |author1=李如瓛 |title=上海卫星城：从认识到实践 |url=http://money.163.com/16/0112/10/BD4EE96N00253B0H.html |publisher=东方早报 |accessdate=2019-03-08 |ref=上海卫星城}}</ref>，其中“闵行”即老闵行地区，现为[[江川路街道|江川路街道]]。1958年底，闵行卫星城内有上海电机厂、上海重型机器厂、上海汽轮机厂和上海锅炉厂等，这些工厂的员工以及他们的家属共8万人，拥挤在的闵行老镇。为缓解居民居住压力，政府规划建设新的建筑区域<ref>{{cite web |author1=李佳敏 |title=闵行一条街：以特殊形式呈现的上海建筑 |url=http://sh.eastday.com/m/20130801/u1a7561900_1.html |publisher=东方网 |accessdate=2019-03-08}}</ref>。

新区域的规划由上海民用建筑设计院院长陈植负责，并遵循“先成街后成坊”的原则，采用了一条街的形式。1959年4月3日，“闵行一条街”（即今江川路的[[沪闵路|沪闵路]]至兰坪路段）建造工程正式开工。为了迎接建国十周年，宽44米的道路，连同路两边37幢四五层高的楼房，共7.29万平方米，仅用六个月即竣工，尤其是沿街的11栋建筑才耗时78天即告落成<ref>{{cite web |title=上海“社会主义新型大街”----“闵行一条街” |url=http://www.archives.sh.cn/shjy/scbq/201304/t20130424_38401.html |publisher=上海档案信息网 |accessdate=2019-03-08}}</ref>。当时的“闵行一条街”生活设施齐全，有闵行鞋帽商店、迎宾茶叶店、大庆食品店、[[光明|光明]]食品店、聚商[[老正兴|老正兴]]餐馆，被[[刘少奇|刘少奇]]称为“闵行的[[南京路_(上海)|南京路]]”<ref>{{cite web |author1=梅陇小邻通 |title=曾经名气响当当的“闵行一条街”究竟在哪儿？ |url=http://www.sohu.com/a/194536762_739590 |accessdate=2019-03-08}}</ref>。

1960年初，兰坪路东西两侧建成16幢住宅和商业楼。1961年7月，拓展至[[瑞丽路|瑞丽路]]。

江川路初建成时，路两边种植白杨树。由于冬天落叶，夏天生虫，[[刘少奇|刘少奇]]建议改种植常青树，而后于1961年起种植香樟树。
