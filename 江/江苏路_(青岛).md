'''江苏路'''是[[中国|中国]][[青岛市|青岛市]][[市南区|市南区]]的一条南北向临海道路，也是全市最早建成的一批道路之一。道路南起[[太平路_(青岛)|太平路]]，与[[广西路_(青岛)|广西路]]、[[湖南路_(青岛)|湖南路]]、[[沂水路|沂水路]]、[[龙山路|龙山路]]、齐东路、伏龙山路、莱芜一路、苏州路、观象一路交汇，北抵[[胶州路|胶州路]]及[[热河路|热河路]]路口接[[上海路_(青岛)|上海路]]。全长1361米，车行道宽15米；全路段有两处大坡，坡度较大。

[[德国|德国]]租借时期道路始建，定名为'''俾斯麦街''' ({{lang-de|Bismarckstraße}})，[[日本|日本]]统治时期改名为'''万年町'''（{{jpn|j=万年町|kyu=萬年町|hg=まんねん まち}}），1923年改现名。

江苏路的六交路口以南段多为德租时期建筑，以北段多为民国时期建筑。由其路中段15号位于一个小丘上同时带有复古风格和[[新艺术运动|德国青年派风格]]的[[江苏路基督教堂_(青岛)|江苏路基督教堂]]，最具代表性。

== 周边历史建筑介绍 ==
===太平路-广西路段===
[[File:青岛山东矿业公司旧址_Ehemalige_Schantung-Bergbau-Gesellschaft.jpg|thumb]]
江苏路南端是江苏路[[太平路_(青岛)|太平路]]路口，即是青岛老城区的前海海岸线。路口西侧是太平路25号，建于1901年的[[山东路矿公司|山东矿业公司旧址]]<ref>{{cite book |author=袁宾久 |title=《青岛德式建筑》|year=2009 |publisher=中国建筑工业出版社 |location=青岛 |isbn=9787112109050}}</ref>。

其北面不远处为一建于30年代的日本建筑，该建筑墙外便是江苏路广西路路口。上述两栋建筑在江苏路上没有门，也没有相应门牌号。

===广西路-湖南路段===
====阿伦斯建筑公司住宅旧址====
从江苏路广西路路口东北角至江苏路湖南路路口东南角，即该路段东侧，由南向北分别是江苏路1号、3号和湖南路4号，都是建于1914年的老住宅，初建时都是德国人H.阿伦斯（H.Ahrens）所开办的建筑公司的地产。1914年日德[[青岛战役|青岛战役]]时，这三栋建筑遭到日军轰炸，均受到不同程度损毁。第一次日据时期它们（由北向南）分别成为[[香港上海汇丰银行|汇丰银行]]、[[太古洋行|太古洋行]]、[[怡和洋行|怡和洋行]]的地产。1923年太古洋行曾对江苏路3号进行过扩建。2000年，江苏路1号与湖南路4号列入[[青岛市历史优秀建筑|青岛市第一批历史优秀建筑]]名单。2013年江苏路3号被违规翻修，此后无缘无故地成了军事管理区。2016年11月，江苏路1号也经历了翻修<ref>{{cite web |url=http://www.tsingtau.org/kataster-karten-tsingtau-1914/|title=Katasterkarten Tsingtau}}</ref>。

====安治泰主教公寓旧址====
江苏路1号、3号和湖南路4号的对面，江苏路湖南路路口西南角，是安治泰主教公寓旧址，最初是[[天主教|天主教]][[圣言会|圣言会]]地产，可能由总督府建设局建筑师[[贝尔纳茨|贝尔纳茨]]（P.Bernatz）设计（此人是来自[[慕尼黑|慕尼黑]]的天主教徒，曾为天主教圣言会设计过多座建筑），门口保存至今的1899石牌证明它是青岛现存最早的德租时期建筑之一。[[安治泰|安治泰]]主教（Bischof Johann Baptist von Anzer）为圣言会在天主教[[兖州教区|鲁南代牧区]]的宗座代牧，一些德租时期的资料称该建筑为“Das Palais des Bischofs v.Anzer”（安治泰主教宫），“安治泰主教公寓”的习惯性中文称呼便出自此处，但实际上安治泰并未在此居住过，该建筑只是圣言会为增加财政收入而建设的出租公寓之一。该建筑现为民居，且有严重的财产纠纷。2000年列入第一批[[青岛市历史优秀建筑|青岛市历史优秀建筑]]名单。

该建筑地上两层带阁楼，地下一层，南北立面对称，东西端各有一塔楼。塔楼曾于1949年后被拆除，约2009年在大修中被修复。整个墙面使用中式[[青砖|青砖]]，饰以[[花岗岩|花岗岩]]装饰，屋顶最初铺设[[仰合瓦|中式灰瓦]]，1910年代改铺矩形红瓦。南立面大部分为木制[[敞廊|敞廊]]，饰以中式元素。像这种大量使用中式建筑材料与装饰元素的建筑在建筑材料和专业建筑师匮乏的德租时期初期很常见。

====皇家大法官魏尔克住宅====
该路段西侧南端的江苏路2号-4号电业局招待所的位置，是[[胶州帝国法院|胶澳皇家法院]]大法官格奥尔格·魏尔克（Dr.Georg Wilke）住宅的原址，始建于1900年，地产属于总督府，魏尔克在此居住至1902年，拆除时间不详。该建筑与对面的安治泰主教公寓一样，使用了中式青砖灰瓦及木制[[敞廊|敞廊]]。

====俄国驻青岛领事馆旧址====
江苏路4号的北侧，江苏路6号的位置最早是建于1900-1901年的[[兰山路#顺和洋行青岛分行|顺和洋行]]业务经理赫尔曼·罗伊特尔（Hermann Reuter）的住宅。罗伊特尔在青岛生活的时间可能并不长。约1909年，该地产被转手给原[[津浦铁路|津浦铁路]]总办[[李德顺|李德顺]]{{注|由于1912年以前总督府禁止华人在欧人区购买地块，李德顺购买的地块均被记在其德籍夫人李柯氏（Frau Margarete Li, geb. Krüger）的名下}}。1907年[[汉堡－美洲公司|汉堡-美洲航运公司]]（Hamburg-Amerika Linie）驻青岛代表汉斯·克罗帕切克（Hans Kropatscheck）租下了这栋房子。次年，兼任[[俄罗斯帝国|俄国]]驻青岛名誉副领事的克氏在此开办俄国驻青岛领事馆。大约1910年该房被改建，1914年遭日军轰炸损毁。现存建筑大约建于第一次日据时期。

2009年3月至4月，该建筑被私自改造，拆除屋顶与山墙加盖三楼<ref>{{cite web|url=http://wanbao.qingdaonews.com:8080/epaper/qdwb/html/2009-04/17/content_2072388.htm|title=江苏路老建筑被破坏之惨状}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。由于该建筑属于城市风貌保护区范围内{{注|《青岛市城市风貌保护管理暂行办法》附件二，1990年5月21日}}，这种行为当属违法。[[青岛市人民代表大会|青岛市人大]]刚刚于当年2月通过“七号议案”，即《关于加强青岛历史文化名城保护和利用议案的实施方案》，[[青岛市市长|青岛市市长]][[夏耕|夏耕]]又于当年3月、4月多次提及利用老城风貌发展旅游业的事项<ref>{{cite web |url=http://news.qq.com/a/20090309/001979.htm|title=青岛市长夏耕：正申报中国最佳旅游城市|accessdate=2009-03-09|work=吕卫红 晓虹|publisher=正义网（原载）、腾讯新闻}}</ref><ref>{{cite web |url=http://www.qingdaonews.com/gb/content/2009-04/14/content_8030154.htm|title=大众日报：青岛市长夏耕谈青岛旅游业重任|accessdate=2009-04-14|work=|publisher=大众日报（原载）、青岛新闻网}}</ref>，因此对老建筑的破坏行为属于顶风作案，本地媒体、市民舆论反应强烈{{注|可参见2009年4月15日《青岛早报》第17版和4月17日《青岛晚报》第28版}}。青岛[[城管|城管]]曾多次进行调查，但由于违法业主在幕后动用[[关系_(社会学)|关系]]、人脉“解决”此事，最终不了了之。

====贝恩住宅旧址====
江苏路8号最初是[[兰山路_(青岛)#顺和洋行青岛分行|顺和洋行]]经理罗兰德·贝恩（Roland Behn）的住宅，始建于1900年，由[[广包公司|广包公司]]（Baufirma F.H.Schmidt）承建，最初只建成了东半部份，西半部份于大约1903年建成。几年后该房被卖给李德顺。现为市南区质监局的办公楼<ref>{{cite book |author=袁宾久 |title=《青岛德式建筑》|year=2009 |publisher=中国建筑工业出版社 |location=青岛 |isbn=9787112109050}}</ref>。2000年成为[[青岛市历史优秀建筑|青岛市第一批历史优秀建筑]]。

====俾斯麦街美国长老会住宅旧址====
江苏路10号是大约建于1901年的[[美北长老会|美国长老会]]住宅旧址，最初登记在长老会牧师{{le|保罗·伯尔根|Paul D. Bergen}}名下。美国驻青岛领事馆迁入[[美国驻青岛总领事馆|沂水路1号]]前，这里曾是其办公地点<ref>{{cite book |author=袁宾久 |title=《青岛德式建筑》|year=2009 |publisher=中国建筑工业出版社 |location=青岛 |isbn=9787112109050}}</ref>。该楼至1941年[[太平洋战争|太平洋战争]]爆发前一直为美国长老会所有。现为民居。2000年列入第一批[[青岛市历史优秀建筑|青岛市历史优秀建筑]]名单。2018年7月初，该楼外立面被违规翻修，大部分的绿色釉质立柱被拆除。<ref>{{cite web|url=http://news.qingdaonews.com/qingdao/2018-07/09/content_20175007.htm|title=网传江苏路10号历史建筑遭破坏？来看记者现场探访|date=2018-07-09|publisher=青岛新闻网}}</ref>

====江苏路12号====
江苏路12号位于基督教堂正西面，其所在地块最初有一栋单层木制临时住宅楼，约建于1899年左右，住宅主人为当时在租借地任[[驻外武官|驻外武官]]的[[埃里希·冯·法金汉|埃里希·冯·法金汉]]（Erich von Falkenhayn）。法金汉于1900年离开青岛。江苏路12号址现存的建筑大约建于1908年，其最初业主不明。由于该楼有“二提督楼”的俗称，该建筑也曾被猜测为总督副官的私邸。该建筑后来也成为李德顺的地产，曾长期为李德顺的私宅。该建筑1945年后曾为青岛市警察局局长高芳先的私人住宅，1949年后曾长期作为[[青岛市人民检察院|青岛市人民检察院]]办公楼。现在是市公安局交通稽查大队驻地。2000年列入[[青岛市历史优秀建筑|青岛市第一批历史优秀建筑]]名单。

===湖南路-沂水路段东侧===
====俾斯麦街施蒂洛住宅旧址====
该路段东侧南端是江苏路5号，建于1914年，是德籍船舶机械师奥托·施蒂洛（Otto Stielow）的房产。现在[[安徽路_(青岛)|安徽路]]的[[安徽路_(青岛)#嘉木美术馆|嘉木美术馆]]当年也是此人的房产。

其北侧是两栋建于20、30年代的老住宅，其中靠北（紧靠江苏路小学正门）一栋为江苏路7号。

====督署学校旧址和督署小教堂====
{{Main|青岛督署学校|青岛督署小教堂}}
江苏路5号东侧位置曾是建于1899年的督署小教堂（Gouvernementskapelle），江苏路5号建成前它也曾经是江苏路街边的建筑。1910年江苏路基督教堂建成前，这里是总督府和[[德意志帝国海军|德国海军]]使用的教堂。1910年后改为督署小学校的体育馆。1999年[[青岛江苏路小学|江苏路小学]]改造时被拆除。这两栋建筑的北面便是江苏路小学，其历史可追溯到1901年建成的[[青岛督署学校|督署学校]]。督署学校的老楼于1999年江苏路小学改造时落架重建。

====德国信义会教堂旧址====
{{Main|江苏路基督教堂 (青岛)}}
江苏路小学北面的高地（曾被称为“青石山”）上便是建于1908年至1910年的[[江苏路基督教堂_(青岛)|江苏路基督教堂]]，青岛的地标之一。

====督署医院旧址====
{{Main|青岛督署医院|青岛大学附属医院}}
基督教堂西北角，即平原路江苏路路口以北，是青岛规模比较大的一家医院，最早是始建于1899年督署医院（Gouvernement Lazarett），建成时是一座规模庞大而建筑风格别致的医院建筑群。日据时期改为陆军医院，1916年后改为青岛病院。1921年3月，青岛病院在原督署医院III号大病房南侧的花园扩建门诊大楼，该楼于1998年拆除，建造新门诊楼。1927年医院改为同仁会医院。抗战胜利后的1946年，[[山东大学|山东大学]]在青岛复校，设医学院，这里作为附属医院，青岛人叫它“山大医院”，1958年山大已迁走，老青岛还叫它“山大医院”，现在的名称是[[青岛大学附属医院|青岛大学附属医院]]，医院中仅存有规模最大的III号大病房旧址及4栋附属建筑。2016年院方开始对大病房旧址进行破坏性整修<ref>{{cite book |author=袁宾久 |title=《青岛德式建筑》|year=2009 |publisher=中国建筑工业出版社 |location=青岛 |isbn=9787112109050}}</ref>。

===沂水路-六路口段东侧===
基督教堂北面，隔街相望的是江苏路17号，有说法称其民国时期曾是青岛保安总队的队部。

====督署医院办公楼====
{{Main|青岛督署医院|青岛大学附属医院}}
医院东面的江苏路19号曾有一两层老建筑，最早是始建于1901年的督署医院办公楼（Lazarettverwaltung），后来曾是青岛医学专科学校，1946年合并于[[山东大学医学院|山东大学医学院]]，后又曾作为护士学校，旧楼已拆除。

江苏路该段东侧老建筑现存比较稀疏。

====菲利普住宅旧址====
江苏路21号是一栋大约建于1913-1914年间的老住宅，1913年地籍图显示它属于一个叫菲利普（Filipp）的人。有资料称该建筑曾为[[美孚石油|美孚石油公司]]青岛分公司经理住宅。现为青岛市中公证处<ref>{{cite web |url=http://www.tsingtau.org/kataster-karten-tsingtau-1914/|title=Katasterkarten Tsingtau}}</ref>。2006年列入第二批[[青岛市历史优秀建筑|青岛市历史优秀建筑]]名单。

====皇家大法官克鲁森住宅旧址====
上去长长的大上坡，坡顶东侧的江苏路27号最初是建于1909年、由广包公司承建的皇家大法官格奥尔格·克鲁森博士（Dr.Georg Crusen）住宅，1923年成为日本驻青岛领事馆总领事官邸，日本军方、[[兴亚院|兴亚院]]与领事馆三方首脑多次在这里开秘密会议。1949年后曾长期为市公安局五处使用，现为市公安局经济犯罪侦查支队。2006年列入[[青岛市历史优秀建筑|青岛市第二批历史优秀建筑]]名单。

===六路口-胶州路段===
向北侧再走过去是青岛有名的“六路口”，也被很多住在附近的人称做“三角地”，江苏路在这里与[[观象一路|观象一路]]、[[苏州路|苏州路]]、[[伏龙路|伏龙路]]、[[齐东路|齐东路]]、[[龙山路|龙山路]]相会，上下起伏，左右跌宕，成为一处景观。江苏路观象一路路口的观象一路1号是[[萧红、萧军、舒群故居|萧红、萧军、舒群故居]]。1934年夏，三位作家迁居至此。齐东路与伏龙路的夹角上是民国实业家[[从良弼|从良弼]]的住宅，现为一家酒店。向伏龙路望去，还能看到高踞于路旁的[[中华基督教会同道堂|中华基督教会同道堂旧址]]。

从“六路口”向北，街道两旁有很多造型各异的建于20-40年代的老建筑。惟有江苏路42与44号建于德租时期的1914年<ref>{{cite web |url=http://www.tsingtau.org/kataster-karten-tsingtau-1914/|title=Katasterkarten Tsingtau}}</ref>。再向北就是江苏路胶州路路口，即江苏路的北端。西侧的红砖教堂是建于1940-1941年的[[青岛圣保罗堂|圣保罗教堂]]。
