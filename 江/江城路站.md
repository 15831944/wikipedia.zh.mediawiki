'''江城路站'''位于[[杭州市|杭州市]][[上城区|上城区]]江城路河坊街交叉路口，为杭州地铁[[杭州地铁5号线|5号线]]及[[杭州地铁7号线|7号线]]<ref>{{cite web |title=近期的杭州地铁建设都有哪些新动向？紫金港站征地，江城路站...... |url=http://wemedia.ifeng.com/67615061/wemedia.shtml |website=凤凰网}}</ref>的地下换乘站，5号线车站预计2019年开放，7号线车站将在5号线开通后开工建设，预计2021年开放。<ref>{{cite web |title=杭州地铁7号线江城路站开工了？- 杭州网 - 杭州新闻中心 |url=http://hznews.hangzhou.com.cn/chengshi/content/2018-06/13/content_7018968.htm |website=杭州网}}</ref>
[[File:20180706拆卸中的建国南苑28幢.jpg|thumb]]

==车站结构==
江城路为5/7号线T形换乘站，有B1、B2、C、D、E1、E2，6个出入口。

5号线车站主体位于江城路地下，呈南北走向，车站长194.6米，宽22.7米。<ref>{{cite web |title=杭州地铁5号线江城路站主体结构开工 |url=http://www.zjsgzw.gov.cn/art/2017/5/24/art_17_14163.html |website=浙江省国资委}}</ref>7号线车站主体位于河坊街地下，呈东西走向。
车站总建筑面积约1.7万㎡，共设风道1组，风亭3个。<ref>{{cite web |title=越来越方便了，杭州地铁7号线江城路站即将开建 |url=http://www.sohu.com/a/235388357_99964967}}</ref>
