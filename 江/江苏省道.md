'''江苏省级干线公路'''，简称'''江苏省道'''，是指由[[江苏省|江苏省]][[江苏省交通运输厅|交通运输厅]]负责修建、养护和管理的省级干线公路。与江苏省境内的[[中华人民共和国国道|国道]]共同组成江苏省的干线公路网。

== 发展历程 ==
1949年末，[[苏北行政公署|苏北行政公署]]公路局划定区内18条重要公路。1950年初，[[苏南行政公署|苏南行政公署]]交通管理局划定区内9条重要公路。[[江苏省|江苏省]]建省后，省交通厅於1961年确定了全省27条公路为干线公路。<ref name="jsdfz">{{cite book|author1=江苏省地方志编纂委员会|editor=王建平|title=江苏省志 交通志.公路篇|publisher=江苏古籍出版社|location=南京|isbn=7-80643-608-1|edition=2001年12月第1版|url=http://www.jssdfz.com/book/glp/JTZML.HTM|accessdate=2018-01-28}}</ref>

1979年4月，省交通局划定全省107条县际以上公路为干线公路，并统一进行编号。1981年，国家首次划定国家干线公路网，江苏多条干线公路纳入[[中华人民共和国国道|国家干线公路]]范围内。<ref name="jsdfz"/>

1987年7月，省交通厅根据国道布局，结合[[市管县|市管县]]的新行政区划，重新划定“5射40纵25横”共70条省级干线公路（简称省道）。<ref name="jsdfz"/>

2003年，《江苏省省道网调整方案》出台<ref name="jssd2003">{{cite web|author1=江苏省人民政府办公厅|title=省政府办公厅关于转发省交通厅江苏省省道网调整方案的通知|url=http://www.jiangsu.gov.cn/art/2003/1/3/art_46144_2545578.html|website=江苏省人民政府网|accessdate=2018-01-28|date=2003-01-03}}</ref>，首次将[[国道主干线|国道主干线]]之外的高速公路列入省道，并扩容普通省道规模。江苏省道调整为“9射38纵26横5环”共78条线路9138公里，其中高速公路省道“5射4纵3横5环”共17条线路2346公里，普通省道“4射34纵23横”共61条线路6792公里。至2010年底规划省道网基本建成。<ref>{{cite web|author1=江苏省交通运输厅|title=《江苏省省道公路网规划（2011－2020年）》政策解读|url=http://temp.jscd.gov.cn/art/2014/7/7/art_5551_867402.html|website=江苏省交通运输厅网|accessdate=2018-01-28|date=2014-07-07}}</ref>

2012年，《江苏省省道公路网规划（2011-2020 年）》正式出台<ref name="jssd2012">{{cite web|author1=江苏省人民政府办公厅|title=省政府办公厅关于印发江苏省省道公路网规划（2011－2020年）的通知|url=http://www.jiangsu.gov.cn/art/2012/11/5/art_46144_2545048.html|website=江苏省人民政府网|accessdate=2018-01-28|date=2012-11-05}}</ref>，新的规划中高速公路省道扩容为“13纵9横14支”共36条线路，普通省道扩容为“4射48纵44横5环84连”共185条线路（其中包括14条规划国道的过渡线路）。随后新的[[s:国家公路网规划（2013年－2030年）|国家公路网规划]]於2013年发布<ref>[http://zfxxgk.ndrc.gov.cn/PublicItemView.aspx?ItemID={93c7d13b-aa0d-4beb-955e-268adade8a8f} 国家公路网规划(2013年-2030年)] {{webarchive|url=https://web.archive.org/web/20161016051120/http://zfxxgk.ndrc.gov.cn/PublicItemView.aspx?ItemID=%7B93c7d13b-aa0d-4beb-955e-268adade8a8f%7D |date=2016-10-16 }}</ref>，国道网大规模扩容，江苏14条规划省道列入国道。
== 编号规则 ==
现行江苏普通省道的路线编号由省道[[标识符|标识符]]“S”和3位数字编号组成。3位数字中，首位数字用“1、2、3、4、5、6、0”分别标识[[省会|省会]]放射线、北南纵线、东西横线、高速公路互通连接线、港区或机场连接线、经济类节点或景区连接线，以及环线，后两位为顺序码。

1987年划定的省道编号由江苏省份简称“[[苏|苏]]”和3位数字编号组成。3位数字中，首位数字用“1、2、3”分别标识省会放射线、北南纵线、东西横线，后两位为顺序码，分别按由正北方向顺时针、由东向西、由北向南的排序规则自“01”起递增。<ref name="jsdfz"/>

2003年调整的省道编号由省道标识符“S”和3位数字编号组成。3位数字的编号规则同前，并新增首位数字“0”标识环线，同时将各首位数字的“01～20”顺序码赋予高速公路省道，普通省道由“21”起递增重新编号。<ref name="jssd2003"/>

2008年《江苏省高速公路网命名与编号规则》出台后，省级高速公路编号改由省道标识符“S”和“1～99”区间的数字编号组成，其中“1～70”区间的[[奇数|奇数]]、[[偶数|偶数]]编号分别标示北南纵线和东西横线，“71～99”区间编号标示支线高速公路。

2012年调整的普通省道编号结构与编号规则同前，新增编号在原有编号之后按相应排序规则递增，并新增首位数字“4、5、6”分别标识高速公路互通连接线、港区或机场连接线、经济类节点或景区连接线，同时利用空出的首位数字“2、3”的“01～20”编号作为规划国道未正式出台前的过渡编号。<ref name="jssd2012"/>

== 分类 ==
江苏省道分为高速公路省道和普通省道。作为江苏国省干线公路的重要组成部分，按功能层次还可分为高速干线、快速干线和一般干线3类，其中高速公路省道属高速干线，普通省道属快速干线和一般干线。

高速公路省道由36条路线组成，按布局类型分为南北向线路、东西向线路和高速公路支线3类。

普通公路由185条路线组成，按布局类型分为放射线路、南北向线路、东西向线路、环线线路、高速公路互通连接线和重要连接线路6类。
