《'''江河水'''》，[[中国东北地区|東北]][[民間音樂|民間樂曲]]，最初版本為[[笙|笙管]]演奏，后改編成[[管子_(乐器)|双管]]独奏曲<ref>http://book.dajianet.com/print/1088442/198841/ 中國音樂名作快讀 第28章《江河水》</ref>。該樂曲[[二胡|二胡]]版本的流傳度較大。

== 樂曲歷史 ==
《江河水》來自于「[[遼寧鼓樂|遼寧鼓樂]]」，系[[聲樂|聲樂]][[曲牌|曲牌名]]，原名《江兒水》，在[[崑曲|崑曲]]、[[京劇|京劇]]中常用在慶功飲宴或發兵征戰時歌唱<ref> http://www.hkco.org/ActivityDetail.aspx?ainfo=178&lang=C{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }} 香港中樂團 第二講   9月21日  二胡（管）名曲《江河水》—同聲一哭抒困憂 </ref>。
*[[王石路|王石路]]、[[朱廣慶|朱廣慶]]、[[朱長慶|朱長慶]]和[[谷新善|谷新善]]將其改編為[[管子_(乐器)|雙管]]獨奏曲，1962年在[[廣州|廣州]]首屆[[羊城花會|羊城花會]]上首演。
*[[黃海懷|黃海懷]]又將其移植為二胡曲，于1963年5月“第四屆[[上海之春音樂節|上海之春]]全國首屆二胡獨奏比賽”中演奏，是二胡演奏者的必修曲目之一。
*1964年被選作為音樂舞蹈史詩《[[东方红_(音乐舞蹈史诗)|東方紅]]》中第一場《東方的曙光》的配樂。

== 二胡曲 ==
二胡曲由[[黃海懷|黃海懷]]改編，作品採用了單三部曲式結構，由引子和3個段落組成。
# 引子是由主題旋律中提煉出來的一些特徵性音構成的散板部份，節奏較為自由，強弱對比非常強烈，富有戲劇性，後以無聲的抽泣結束全曲。
# 第1段：富有東北民間特點的慢板樂段，情緒悲痛，有強烈的戲劇性。
# 第2段：轉入較平靜的情緒，運弓較輕，力度較弱，並較少用揉弦技法。
# 第3段：非常激憤，到了歇斯底裡的程度，速度也稍快，語氣更為急促，一氣呵成。
# 52小節后該曲的情緒趨於平靜，結尾漸慢、減弱，以悠遠的餘音結束全曲。

1977年[[小澤征爾|小澤征爾]]帶領[[波士頓交響樂團|波士頓交響樂團]]訪華，曾被[[閔惠芬|閔惠芬]]演奏的《江河水》感動得流淚：「拉出了人間悲切」<ref>大紀元 http://www.epochtimes.com/b5/3/11/22/n415955p.htm 北美新聞 【專訪】上海民族樂團二胡演奏家閔惠芬 </ref>。
