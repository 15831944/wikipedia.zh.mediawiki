'''{{PAGENAME}}有限公司'''是位于[[江苏省|江苏省]][[句容市|句容市]]石山头27号的一个[[國營農場|国有茶场]]，地处[[宁镇山脉|宁镇山脉]]南麓。现隶属于[[江苏省司法厅|江苏省司法厅]]戒毒管理局旗下<ref>{{Cite web |url=http://sft.jiangsu.gov.cn/col/col56997/index.html |title=单位职能 |accessdate=2018-11-06 |author=|date=2017-08-07 |work= |publisher=江苏省司法厅网站|language=Zh-hans }}</ref>的[[江苏恒实集团|江苏恒实集团]]<ref name="a1">{{Cite web |url=http://www.jdztbw.com/about/?19.html|title=关于我们 |accessdate=2018-11-06 |author=|date= |work= |publisher=江苏句东农场有限公司招投标网|language=Zh-hans }}</ref>。网站公告显示，茶场与[[江苏省句东强制隔离戒毒所|江苏省句东强制隔离戒毒所]]相关<ref>{{Cite web |url=http://www.jdztbw.com/list/?3_1.html|title=招标公告 |accessdate=2018-11-06 |author=|date= |work= |publisher=江苏句东农场有限公司招投标网|language=Zh-hans }}</ref>。

茶场创建于1951年，历称[[句容县|句容县]]普渡桥茶场、暨南农场、句东农场、江苏省[[五七干部学校|五七干部学校]]、石山头茶场。茶场占地约1.91平方公里。其中茶园1385亩、耕地439亩<ref name="a1"/>。2012年及之前，在[[国家统计局|国家统计局]]网站统计中<ref>{{Cite web |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2012/32/11/83/321183428.html |title=句东农场 |accessdate=2018-11-06 |author=|date= |work= |publisher=国家统计局网站|language=Zh-hans }}</ref>，所在区域以'''句东农场'''之名列为句容市的一个[[类似乡级单位|类似乡级单位]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|32/11/83/428/000}}}}</ref>。
