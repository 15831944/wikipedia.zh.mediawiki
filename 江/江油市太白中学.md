== 历史 ==
* 1970年，冶金工业部长城钢厂第四子弟校创立，设有小学部和中学部，为太白中学前身。
* 1985年，学校开始筹建高中部。
* 1993年，小学部从学校分离，中学部更名为“长城特殊钢公司第四中学校”。小学部更名为“长城特殊钢公司第四小学校”（现太白小学）。
* 2001年，学校正式从长钢集团公司剥离，划归江油市政府管理。
* 2002年，学校经江油市政府批准更名为“四川省江油市太白中学”。<ref>{{Cite news|url=http://www.mianyang.net.cn/zhuanti/zmy/news/2010_01/taibaizhongxue.html|title=江油市太白中学|author=贾城|date=2010-01|work=绵阳广播电视报|accessdate=}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 硬件设施 ==
学校占地53亩，建筑面积29300平方米。太白中学建成了多媒体交互式校园教学网系统，同时每个教室配置电视与标准广播系统。该校拥有网络教室，多媒体教室，教学实验室，语音室，演播室，电子备课室等功能室。江油中学的图书馆藏书3万余册，另有电子图书库图书资料3万余册。

== 办学情况 ==
太白中学是江油市三所公办普通高中之一。在升学率上，太白中学不及江油一中与江油中学，亦有更低的录取分数。<ref>{{Cite web|url=http://www.sc518.com/forum.php?mod=viewthread&tid=529855&highlight=%CC%AB%B0%D7%D6%D0%D1%A7|title=江油市太白中学高考喜报|accessdate=2017-09-03|author=太白中学|date=2016-06-23|publisher=}}</ref>同时，太白中学也会进行补习班招生。<ref>{{Cite web|url=http://www.sc518.com/forum.php?mod=viewthread&tid=537921&highlight=%CC%AB%B0%D7%D6%D0%D1%A7|title=江油市太白中学高四年级高考喜报及招生公告|accessdate=2017-09-03|author=太白中学|date=|publisher=}}</ref>

== 评价 ==

=== 积极 ===
有观点认为，太白中学近年进步显著，亦在管理方面有所加强，为江油市成绩中下的中学生提供了一个较好的高中学习平台。<ref>{{Cite web|url=http://www.sc518.com/forum.php?mod=viewthread&tid=653009&page=2|title=江油论坛帖子“太白中学怎么样?”|accessdate=2017-09-03|author=|date=|publisher=}}</ref>

=== 消极 ===
太白中学日益严重的补课问题引起了多方的批评。<ref>{{Cite web|url=http://www.sc518.com/forum.php?mod=viewthread&tid=481873&highlight=%CC%AB%B0%D7%D6%D0%D1%A7|title=江油论坛帖子“关于太白中学的猖獗补课，请有关部门介入”|accessdate=2017-09-03|author=|date=|publisher=}}</ref>太白中学也存在恶性争夺生源的行为。
