《'''江华条约'''》又称《'''江华岛条约'''》、《'''日朝修好條規'''》、《'''日鮮修好條規'''》、《'''丙子修好条约'''》，是[[大日本帝国|大日本帝国]]与[[李氏朝鲜|朝鲜王朝]]于1876年2月26日在[[江华岛|江华岛]]签订的[[不平等条约|不平等条约]]，从此朝鲜逐步沦落为大日本帝国的半殖民地。

== 背景 ==
{{seealso|江華島事件}}
1875年9月，日本派出军舰[[云扬号|云扬号]]航行到朝鲜西海岸测量海图，遭到朝鮮[[江华岛|江华岛]]砲台守軍砲擊，雙方發生交火，最後日本方攻陷炮台，史称“[[江华岛事件|江华岛事件]]”（日本稱「{{lang|ja|江華島事件}}」，韓國稱「{{lang|ko|雲揚號事件}}」）。

== 内容 ==
《江华条约》共计12款，是朝鲜和外国签订的第一个不平等条约。日本取得了自由勘测朝鲜海口、[[领事裁判权|领事裁判权]]、贸易等权利。自此，朝鲜对外开放，外国商品進入朝鲜。条约的主要内容包括承认朝鲜为独立国及日本享有的领事裁判权等。

== 影响 ==
《江华条约》是日本「[[大陆政策|大陆政策]]」实践的开始。此政策以征服中国为中心，但征服中国的第一步是征服朝鲜，簽訂江華條約为以后日本吞并朝鲜做了铺垫。
