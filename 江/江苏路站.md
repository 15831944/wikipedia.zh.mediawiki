'''江苏路站'''位于[[上海|上海]][[长宁区|长宁区]][[江苏路_(上海)|江苏路]][[愚园路|愚园路]]，为[[上海轨道交通2号线|上海轨道交通2号线]]和[[上海轨道交通11号线|11号线]]的地下[[島式站臺|島式車站]]及换乘站，两线通过站厅通道换乘。

== 車站歷史 ==
*1999年9月20日：江苏路站啟用。
*2009年12月31日：11号线开通，作为11号线终点站。本站成为2号线与11号线的换乘站<ref>{{cite news|title=上海市轨交11号线主线线路12月底开通|url=http://www.gov.cn/fwxx/ly/2009-12/11/content_1484905.htm|publisher=中华人民共和国中央人民政府|date=2009-12-11}}</ref>。
*2013年8月31日：从本站至罗山路站的11号线二期通车，该站不再作为终点站折返列车<ref>{{cite news|title=11号线江苏路站调整上车方式|url=http://www.jiading.gov.cn/mspd/shgj/content_18835|publisher=上海嘉定|date=2013-06-04}}</ref>。

===站厅===
江苏路站2号线站厅位于地下一层，沿愚园路布置。站厅北侧为付费区，内设4组楼梯及自动扶梯以及一组直梯通往站台。11号线站厅位于地下二层，沿江苏路布置，呈南北向。站厅付费区位于站厅东侧，设有4组楼梯/自动扶梯和1座直梯通往站台。11号线站厅付费区南侧设有楼梯和自动扶梯通往位于地下一层的换乘大厅。

江苏路站换乘大厅位于地下一层、11号线站厅上方，沿江苏路布置，长约80米，宽约12米，以舒缓2号线、11号线两条大客流线路的换乘人流<ref name="change"/>。2号线站厅和换乘大厅设有通道连接，通道中间设有栏杆以隔离不同方向的换乘客流以防止客流对冲，所有换乘乘客应在换乘通道内靠左行走<ref>{{cite news|title=上海地铁江苏路站新增隔离护栏 防客流对冲|url=http://2010.qq.com/a/20100816/000164.htm|publisher=腾讯网|date=2010-08-16}}</ref>。

== 車站出口 ==
江苏路站2号线和11号线目前统一编号编号，共有7个出口
; 二号线
* 2號出口：愚園路
* 3號出口：愚園路
* 4號出口：愚園路、江蘇路
;十一号线
* 5号（11号线1号）出口：江苏路、市三女中
* 6号（11号线2號）出口：[[江苏路|江苏路]]、[[愚园路|愚园路]]（此出口原为2号线1号口）
* 7号（11号线3号）出口：江苏路、愚园路（忠和坊）、[[安定坊|傅雷旧居 (江苏路)]]
* 8号（11号线4号）出口：江苏路、愚园路（安定坊）
