'''江北站'''（{{jpn|j=江北駅|hg=こうほくえき|rm=Kōhoku eki|f=y}}）位於[[日本|日本]][[東京都|東京都]][[足立區|足立區]]{{link-ja|江北 (足立區)|江北 (足立区)|江北}}四丁目，是[[東京都交通局|東京都交通局]][[日暮里-舍人線|日暮里-舍人線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''NT 08'''。

== 車站結構 ==
島式1面2線[[高架車站|高架車站]]。設有[[月台閘門|月台閘門]]。

== 歷史 ==
* 2008年（[[平成|平成]]20年）3月30日 - 開業。

== 使用情況 ==
2016年度的1日平均上下車人次為'''9,512人'''（上車人次：4,781人、下車人次：4,731人）<ref>[http://www.kotsu.metro.tokyo.jp/nippori_toneri/kanren/passengers.html 東京都交通局 各駅乗車人員一覧]</ref>。日暮里-舍人線車站當中次於[[日暮里站|日暮里站]]、[[西日暮里站|西日暮里站]]、[[見沼代親水公園站|見沼代親水公園站]]、[[西新井大師西站|西新井大師西站]]排行第5位。
