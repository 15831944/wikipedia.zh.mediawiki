'''{{FULLPAGENAME}}'''是[[江苏省人民政府|江苏省人民政府]]司法厅监狱管理局的直辖单位，位于[[句容市|句容市]]东门外。主要收押、改造江苏省被判刑的未成年犯<ref name="a1">{{Cite web |url=http://www.jsjy.gov.cn/jsjyw/zsdw/2016-06/29/content_a7ff4834f4034d6aba7f5b6d2d4428bb.shtml |title=未成年犯管教所（镇江女子监狱） |accessdate=2018-11-03 |author=|date= |work= |publisher=江苏省监狱管理局网站|language=Zh-hans }}</ref>。与[[镇江女子监狱|镇江女子监狱]][[合署办公|合署办公]]<ref>{{Cite web |url=http://js.chinaso.com/tt/detail/20180411/1000200033118751523447025970760283_1.html |title=江苏女性犯罪增加，镇江女子监狱将搬迁至江宁 |accessdate=2018-11-03 |author=|date=2018-04-11 |work= |publisher=中国搜索|language=Zh-hans }}</ref>。

管教所的历史可追溯至1954年在安徽省[[郎溪县|郎溪县]]成立的[[少管所|少管所]]，是中华人民共和国最早建成的9个少管所之一。1960年，迁至今址。2004年，改称现名。占地1.5平方公里<ref name="a1"/>。2012年及之前，在[[国家统计局|国家统计局]]网站统计中<ref>{{Cite web |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2012/32/11/83/321183417.html |title=江苏省第一少年管教所 |accessdate=2018-11-03 |author=|date= |work= |publisher=国家统计局网站|language=Zh-hans }}</ref>，所在区域以'''江苏省第一少年管教所'''之名列为句容市的一个[[类似乡级单位|类似乡级单位]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|32/11/83/417/000}}}}</ref>。
