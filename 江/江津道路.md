'''江津道路'''（{{lang|ja|江津道路／ごうつどうろ}}）是連接[[島根縣|島根縣]]的[[江津市|江津市]]與[[濱田市|濱田市]]，總長17.6[[公里|公里]]的[[高規格幹線道路|高規格幹線道路]]。

{{link-ja|江津交流道|江津インターチェンジ}}以東由[[国土交通省|国土交通省]]以[[国道9号|国道9号]]'''江津繞道'''為道路名管理，現為[[山陰自動車道|山陰自動車道]]的現道活用區間。

江津交流道以西由[[西日本高速公路|西日本高速公路]]以'''山陰道'''為道路名管理，並以山陰自動車道[[高速自動車國道並行一般国道快速道路|並行]]國道9號的[[快速道路|快速道路]]名義整備。

== 概要 ==
[[File:Gotsu_road.JPG|thumb]]
* 起点：島根縣江津市渡津町
* 終点：島根縣濱田市高佐町
* 全長：17.6公里
* {{link-ja|規格|道路構造令}}：第1種第3級
* 設計速度：80[[km/h|km/h]]
* 道路幅員：20.5[[公尺|公尺]]
* [[車道|車道]]幅員 : 3.5公尺
* 車道數 : [[2車道高速公路|暫定2車道]]

== 歷史 ==
* [[1990年|1990年]]度：開徵道路用地
* [[1992年|1992年]]度：[[都市計畫|都市計畫]]決定
* [[1993年|1993年]][[3月|3月]]：江津市渡津町 - 嘉久志町間2.8公里啟用
* [[2003年|2003年]][[9月21日|9月21日]]：江津市嘉久志町（江津交流道）- 濱田市高佐町（濱田系統交流道）間14.8公里啟用，'''全線通車'''

== 交通量 ==
平日24小時交通量（平成17年度道路交通調查）
* 江津交流道 - 江津西交流道：1,539
* 江津西交流道 - 濱田東交流道：1,744
* 濱田東交流道 - 濱田系統交流道：1,795
從[[2010年|2010年]][[6月28日|6月28日]]開始實施的通行費免費化實驗後的一個禮拜，交通量在江津交流道 - 江津西交流道間從約 2100台／日增加至約 6500台／日，約3.1倍的成長。
