== 花絮 ==
该片有林心如打斗的动作场面。<ref>{{cite web |url=http://ent.enorth.com.cn/system/2006/05/10/001299652.shtml |title=林心如演《江湖龙女》 “弱女”变“打女”(图) |accessdate=2006 |publisher=北方网 |work= |language=zh-hans}}</ref>也是林心如继《[[一石二鸟|一石二鸟]]》之后的又一动作喜剧。<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2006-04-20/20341056875.html |title=林心如TAE新片演绎现代版“杨过与小龙女”(图) |accessdate=2006 |publisher=新浪网 |work= |language=zh-hans}}</ref>
