'''江油市第三涪江大桥'''又称'''江油市涪江三桥'''，是[[四川省|四川省]][[绵阳市|绵阳市]][[江油市|江油市]]在[[汶川大地震|汶川大地震]]后建设的一座跨越[[涪江|涪江]]连接江油南部市区两岸的大型公路桥。大桥在2009年10月开通，全长340.05米，为双向四车道加两条非机动车道和绿化带设计，桥两端为立交式引桥。桥梁现在为江油市工业园区车辆前往绵阳市区的主要通道之一。

== 建设历史 ==
涪江三桥建设始于2007年1月22日，项目由江油市规划和建设局批复同意选址实施。在2008年5月12日汶川大地震后，大桥在建部分受到了严重损坏，2#桥墩2-b柱工程出现大量裂缝，而3#、4#、5#墩在建工程受破坏相对较小。2008年8月25日，河南省交通厅与江油市交通局就大桥恢复建设工作达成一致意见。2009年9月28日，大桥重新开工。<ref name=":0">{{Cite web|url=https://www.jiangyou.gov.cn/4371953/4528852.html|title=汶川特大地震江油抗震救灾志 第五篇 灾后重建|accessdate=2018-01-31|author=江油市政府方志办|date=2016-11-17|publisher=江油市人民政府}}</ref>2009年9月底，涪江三桥在河南省对口援建下竣工通车，共投资9800余万元。大桥建成后，被冠以“中原爱心桥”的名称以纪念河南省对江油市的震后对口支援。<ref>{{Cite news|url=http://news.163.com/09/1017/05/5LQ72FCC000120GR.html|title=河南人均一块钱建起中原江油连心桥|author=张高峰|date=2009-10-17|work=河南商报|accessdate=}}</ref>

== 结构 ==
江油市第三涪江大桥为钢筋混凝土[[梁桥|梁桥]]，全长340.05米，宽29米，其中机动车道宽15米，为双向四车道布局；两侧有绿化带、非机动车道与人行道。桥梁设计[[結構荷重|荷载]]城市-A级。人群荷载3.5千牛每平方米。<ref name=":0" />

== 周边 ==
大桥西端连接江油新开发区，东端连接江油河南工业园。桥梁西端两旁为江油市第三个公园——体育公园。大鹏路跨该桥梁而过，连接{{中国省道名|205|P=四川}}（江油市区称李白大道）和江彰大道。
