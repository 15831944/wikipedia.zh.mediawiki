'''江苏省新海中学'''始建于1930年，因位于'''新海连市'''（现[[江苏|江苏]][[连云港|连云港]]）而得名。是江苏省最早的18所省属重点中学之一，首批国家级示范普通中学。

== 学校简介 ==
新中校址位于苍梧路58号。校园占地近300亩，建筑面积达8万平方米。<ref name=intro>新海高级中学简介，{{cite web |url=http://www.xhgz.com/bencandy.php?fid=34&id=256 |title=存档副本 |accessdate=2011-03-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20141105134945/http://www.xhgz.com/bencandy.php?fid=34&id=256 |archivedate=2014-11-05 }}</ref>

== 学校历史 ==
江苏省新海高级中学成立于1924年，初名“新浦普爱平民义务学校”，1948年学校更名为“山东省立新海中学”，1953年1月学校更名为“江苏省新海中学”，2001年初、高中分离，学校由江苏省新海中学高中部和原连云港市新浦中学的高中部合并成立“江苏省新海高级中学”至今，初中部则成立连云港市新海实验中学。
