'''江夏区'''，是[[中国|中国]][[湖北省|湖北省]][[武汉市|武汉市]]所辖的一个[[市辖区|市辖区]]。面积为2009[[平方公里|平方公里]]，2014年人口为59万人。<ref>{{Cite web|url = http://www.jiangxia.gov.cn/qqjjrkms/7445.jhtml |title = 江夏区人口状况 |date =2007-05-28 |accessdate =2017-05-17 |publisher = 武汉市江夏区政府网站}}</ref>

==历史沿革==
周代属[[楚|楚]]，[[秦|秦]]属南郡。西汉高祖六年（前201年）设置[[江夏郡|江夏郡]]，辖境相当今湖北安陆、钟祥、潜江、沔阳、嘉鱼、蒲圻、崇阳以东，以及河南光山、[[新县|新县]]以西、[[信阳|信阳]]以东、淮河以南地。今武汉之地属[[沙羡县|沙羡县]]。

378年改汝南县。581年改江夏县，589年改治郢城（现在的[[武汉市|武汉市]][[武昌区|武昌区]]）。[[辛亥革命|辛亥革命]]后，1912年置武昌县，隶属于江汉道，1930年改隶湖北省第一专署。1949年中华人民共和国成立后，在位于武汉市武昌区的傅家坡设立武昌县人民政府，隶属于[[大冶市|大冶专区]]。1952年改由[[孝感市|孝感专区]]管辖。1959年11月，武昌县划归武汉市，1960年县政府迁往纸坊镇。1961年4月，又将其划归孝感专区，1965年划归[[咸宁市|咸宁专区]]。1975年11月，武昌县再次划归武汉市，成为武汉市的郊县之一。1995年撤县建区，成立武汉市江夏区。<ref>{{Cite web |url = http://www.jiangxia.gov.cn/News/ViewNews.aspx?id=810 |title = 江夏区历史概况 |date = 2007-05-28 |accessdate = 2008-12-11 |publisher = 武汉市江夏区政府网站 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==地理==
===位置===
江夏区坐落于[[长江|长江]]中游南岸，地处东经114º01’～ 114º35’、北纬29º58’～ 30º32’。正北与[[武汉市|武汉市]][[洪山区|洪山区]]毗连，东北与[[鄂州市|鄂州市]]接壤，东南与[[大冶市|大冶市]]相邻，正南与[[咸宁市|咸宁市]]交界，西南与[[嘉鱼县|嘉鱼县]]相靠，西北与[[汉阳区|汉阳区]]和汉南区隔长江相望。区境南北长83.2公里，东西宽54.17公里。区治纸坊街道办事处。<ref name="武汉方志网">{{Cite web|url = http://www.whfz.gov.cn/shownews.asp?id=15493 |title = 武昌县 |accessdate =2009-10-26 |publisher = 武汉方志网}}</ref>

===地形地貌===
江夏区在地层区划上属扬子地层区下扬子分区大冶小区。地质特点是以原古生界为基底，自震旦纪以来，长期接受沉积，地层发育极为良好。地质结构主要是印支运动和燕山运动的产物，褶皱和断裂发育，岩浆活动有一定表现。地貌为鄂南幕阜丘陵向江汉冲积平原延伸的过渡地带。地势南北狭长微倾；东西北依湖滨江，以垅岗平原为主要地貌类型。

区境各山南面承接鄂南幕阜山系向北蜿蜒，以郑店与乌龙泉连界的个字山为主，向三方呈个字形伸延。个字山北为八分山，其西北为神山、金冈山、天光山等；其东南为马鞍山、仙人山、天子山、石山、浮山等；其东迤北为青龙山、狮子山、锦绣山、凤凰山、二龙山、丁姑山、黄龙山等。其中以八分山为最高，海拔272．5米，总面积O．8平方公里；其次为二龙山之龙泉山，海拔232．5米；丁姑山海拔200米以上。

区境西北濒临长江，东、北、西南三面环湖，有金水贯串其间，形成江湖环抱之势；又因全县地势南北狭长而东西倾斜，水系呈离心扇形。按自然出江口划分：金水出金口入江，梁子湖水出樊口，汤逊湖水出武泰闸。这三个水系包括许多港汊、塘堰和湖泊。整个水域面积达106．9万亩，占全县总面积的35．5%。<ref name="武汉方志网"/>

===气候===
江夏区属亚热带大陆性季风气候，光照充足，气候温和，雨量充沛，四季分明，酷暑严冬时间短，无霜期长。

==行政区划==
截至2017年1月10日，江夏区辖[[湖泗街道办事处|湖泗]]、纸坊、安山、金口、郑店、乌龙泉、五里界、法泗、山坡、舒安10个[[街道办事处|街道办事处]]，金港新区、庙山、藏龙岛，大桥4个办事处和梁子湖风景区。<ref>{{Cite web|url = http://www.xzqh.org/html/list/2961.html |title = 武汉市江夏区 |accessdate =2008-12-11 |publisher = 行政区划网站 |format = htm}}</ref>

==經濟發展==
2010年，江夏區全區經濟總值（GDP）共計227億元，是2005年的2.6倍，年均增長15.5%；全區財政收入42.74億元，是2005年的5.3倍，年均增長39.6%。在湖北省80個縣（市區）經濟發展綜合評比中，通過在经济总量、人均、速度、后劲等6大类共25个小项指标中的量化考核，以百分制總分78.4分排名第一。這也是江夏區連續三年（2008年～2010年)在湖北省縣域經濟綜合測評中排名第一<ref>{{Cite web |url =http://www.plcsq.com/article-1757.html |title =2010年全省县域经济10强揭晓 黄陂排名第4 |date =2011-03-28 |accessdate =2011-09-10 |publisher =盤龍城社區 |deadurl =yes |archiveurl =https://web.archive.org/web/20110607123542/http://www.plcsq.com/article-1757.html |archivedate =2011-06-07 }}</ref>。[[上海通用汽车有限公司|上海通用汽车有限公司]]继烟台东岳、沈阳北盛之后，将第三个生产基地设置于武汉江夏区。

==历史名人==
* [[熊廷弼|熊廷弼]]（1569-1625）：明末辽东三杰。
* [[譚鑫培|譚鑫培]]（1847-1917）：著名的京劇表演藝術家，京劇生行的譚派的創始人。
* 魏宸组 (1885-1942)：中華民國外交家，曾任中華民國駐荷蘭，比利時，德國，波蘭公使，中華民國外交部次長。
* [[熊秉坤|熊秉坤]]（1885-1969）：曾參加辛亥革命武昌起義，追隨孫中山參加護法運動，曾擔任南京国民政府军事参议院中将参议的革命功臣。
* [[项英|项英]]（1898-1941）：中國共產黨和中國工農紅軍的早期領導人之一，新四軍的創建人和主要領導人之一，曾任新四軍副軍長。
* [[彭孟缉|彭孟缉]] （1908-1997）：中華民國陸軍一級上將。
