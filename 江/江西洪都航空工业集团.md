'''江西洪都航空工业集团有限责任公司'''（Jiangxi Hongdu Aviation Industry Group，简称'''中航工业洪都'''）创建于1951年，是[[中国航空工业集团|中国航空工业集团]]所属的特大型企业，原名为南昌飞机制造公司，前身为国营洪都机械厂。<ref>{{Cite web |url = http://www.hongdu.com.cn/news/zjhd_s.asp?id=184 |title = 中航工业洪都-公司简介 |author = 中航工业洪都 |language = zh-hans |access-date = 2012-10-08 |archive-url = https://web.archive.org/web/20120822232030/http://www.hongdu.com.cn/news/zjhd_s.asp?id=184 |archive-date = 2012-08-22 |dead-url = yes }}</ref>

1954年至1958年，制造了379架[[初教五|初教五]]。此后生产[[初教六|初教六]]。1957年至1968年，生产了727架[[运-5|运-5]]，此后运五由[[石家庄飞机制造厂|石家庄飞机制造厂]]生产。1968年至2012年，生产[[强五|强五]]。1990年代与[[巴基斯坦|巴基斯坦]]合作研制[[K8型教练机|K8型教练机]]，出口到多个国家。

南昌飞机制造厂还生产[[农-5|农-5]]农林飞机、[[L-15|L-15]]猎鹰高级[[教练机|教练机]]等飞机。

2005年12月27日，[[美国国务院|美国国务院]]指责该公司向[[伊朗|伊朗]]提供武器和技术，并对该公司进行[[经济制裁|经济制裁]]。
