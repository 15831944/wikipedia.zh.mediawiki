'''江之島電鐵線'''（{{jpn|j=江ノ島電鉄線|hg=えのしまでんてつせん|rm=Enoshima dentetsusen|f=y}}）是一條由[[神奈川縣|神奈川縣]][[藤澤市|藤澤市]]南藤澤[[藤澤站|藤澤站]]起途經同市{{link-ja|片瀨海岸|片瀬海岸}}一丁目[[江之島站|江之島站]]前往[[鎌倉市|鎌倉市]]{{link-ja|御成町 (鎌倉市)|御成町 (鎌倉市)|御成町}}[[鎌倉站|鎌倉站]]，屬於[[江之島電鐵|江之島電鐵]]的鐵路線。1902年（[[明治|明治]]35年）開通藤澤－片瀨（現時江之島）段，1910年（明治43年）全線開通至小町（現時鎌倉），時常簡稱為'''江之電'''（{{jpn|j=江ノ電|hg=えのでん|rm=Enoden|f=y}}）。車站編號使用的路線記號為'''EN'''。

全線可以使用IC卡乘車券[[PASMO|PASMO]]、[[Suica|Suica]]。[[ICOCA|ICOCA]]、[[TOICA|TOICA]]等其他公司、局的卡類不可用於閘機，但{{link-ja|增值機|チャージ機}}等服務只限於有人站。<!-- 2、3例で充分。キリがないので追加しない（特に使用不可のカード） -->

== 路線概要 ==
路線的長度為10公里，軌距為1067毫米的窄軌。沿線共有15個車站，全路段都採單線配置，但其中有4個車站和1個[[號誌站|號誌站]]擁有雙軌設施可作為列車會車用。使用的電壓為600[[伏特|伏特]][[直流電|直流電]]，由架空電纜供電。由鎌倉站至腰越站屬[[鎌倉市|鎌倉市]]，而[[江之島站|江之島站]]至[[藤澤站|藤澤站]]則屬[[藤澤市|藤澤市]]。

雖然江之島電鐵線所使用的是小型的普通鐵路用列車，但是在[[腰越站|腰越]]至[[江之島車站|江之島]]兩站之間有一小段路線，列車是行駛在一般的道路而非專屬軌道區域內，因此常被認為是[[路面電車|路面電車]]的一類。不過由於江之島電鐵是根據日本的《[[鐵道事業法|鐵道事業法]]》而不是一般管理路面電車之類[[輕軌|輕軌]]運輸的《[[軌道法|軌道法]]》獲得經營特許，因此在法律上仍屬一般鐵路的範疇之內。
== 路線資料 ==
* 路線距離（{{link-ja|營業距離|営業キロ}}）：10.0公里
* [[軌距|軌距]]：1067毫米
* 站數：15個（包括起終點站。另有1個信號所）
* 複線路段：沒有（全線單線）
* 電氣化路段：全線（[[直流電|直流電]]600V）
* [[閉塞_(鐵路)|閉塞方式]]：自動閉塞式（ARC）（代用閉塞方式為指導通信式）
* 可進行[[列車交會|列車交會]]車站、信號場：5個（長谷、稻村崎、峰原（信）、江之島、鵠沼）
* 保安設備：江之電型點制御速度照查式[[自動列車停止裝置|eATS]]
* 最高容許速度：60公里/小時
* 2008年度輸送人員實績：15,709,499人（一日平均43,039人。其他的資料參見[[#使用情況|使用情況]]）

== 巴士 ==
江之島電鐵在該區也有提供巴士服務。

== 其他 ==
*日本電影《[[太陽之歌|太陽之歌]]》即在本線的七里滨站取景。
*江之島電鐵線與[[台灣鐵路管理局|台灣鐵路管理局]][[平溪線|平溪線]]合作行銷，自2013年5月起，遊客可以免費兌換另一方「[[一日週遊券|一日週遊券]]」<ref>{{Cite web |url = http://www.enoden.co.jp/info/heikeisen.html |script-title = ja:台湾・平渓線との乗車券交流|publisher = 江ノ島電鉄株式会社 |accessdate = 2014-04-24 }}</ref>。至2014年3月31日因活動反應良好，持續延長一年<ref>{{Cite web |url = http://www.railway.gov.tw/tw/news_Detail.aspx?SN=7086&NewsType=0&n=6807 |script-title =  zh-tw:臺鐵局平溪線與日本江之島電鐵無償兌換一日券好評延長 |date = 2014-04-02 |publisher = 交通部臺灣鐵路管理局 |accessdate = 2014-04-24 }}</ref>。2015年再延續一年，至2016年3月31日為止。2016年5月1日推出平溪線X江之島第二波乘車券兌換，並推出台日鐵道觀光護照兌換活動，至2017年3月31日為止<ref>{{Cite web |url = https://www.enoden.co.jp/page/heikeisen/c/ |script-title = ja:台湾へ平渓線に乗りに行こう|publisher = 江ノ島電鉄株式会社 |accessdate = 2016-12-28 }}</ref>。

*2016年夏天，江之島電鐵線與[[高雄捷運|高雄捷運]]推出一系列活動，包括憑展覽套票可至江之電兌換「旅遊passport」等優惠。 並特別授權，允許於[[高雄捷運|高雄捷運]] [[高雄國際機場站|高雄國際機場站]]使用[[藤澤站|藤澤站]]版本的進站音樂《好きです江ノ電》（喜歡江之電），做為該站的發車音樂使用。
