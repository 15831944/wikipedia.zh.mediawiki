== 基督新教 ==
[[File:南京江苏路基督教堂仰望.jpg|thumb]]
可统计信徒约120万人，大部分集中分布于北部[[宿迁|宿迁]]、[[淮安市|淮安]]、[[盐城|盐城]]、[[徐州|徐州]]、[[连云港|连云港]]5市。在江苏北部，仅原[[真耶稣教会|真耶稣教会]]的信徒就有20万人。

南京的[[金陵协和神学院|金陵协和神学院]]是[[中国基督教新教|中国基督教新教]]唯一的全国性学府<ref>{{cite web |url=http://www.njuts.cn/showarticle.asp?ID=1|title=金陵协和神学院简介|accessdate=2011-10-11|work=金陵协和神学院网}}</ref>。

在一些城市里保留了一部分过去传教士所建的老教堂，如南京的[[南京圣保罗堂|圣保罗堂]]、[[基督教莫愁路堂|汉中堂]]和苏州的[[苏州圣约翰堂|圣约翰堂]]、[[宫巷堂|乐群社会堂]]、[[使徒堂|思杜堂]]、大西路福音堂（原属卫理公会）。近年来，江苏南北各地又新建了一批大型教堂。

== 天主教 ==
江苏省可统计的[[天主教|天主教]]信徒21万人，分属[[天主教南京教区|天主教南京教区]]、[[苏州教区|苏州教区]]、[[海门教区|海门教区]]、[[徐州教区|徐州教区]]4个教区。其中人数最多的[[南京教区|南京教区]]信徒共有12万人，以[[无锡市|无锡市]]最为集中。辖区包括苏州、南通、徐州以外的10个市。
