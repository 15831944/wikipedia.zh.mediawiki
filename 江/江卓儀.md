'''江卓儀''' （{{lang-en|'''Sonia Kong'''}}, {{bd|1987年|1月12日||}}），[[香港|香港]]女子[[沙灘排球|沙灘排球]]代表，兼任[[now寬頻電視|now寬頻電視]]沙灘排球評述員及[[足球|足球]]節目主持、平面[[模特兒|模特兒]]，正職為[[醫院|醫院]][[護士|護士]]。現於東華三院吳祥川紀念中學當教練

==簡歷==
江卓儀小學五年級暑假曾參加[[麥當勞|麥當勞]]青少年足球訓練，同年開始參與[[排球|排球]]運動，中三加入青年軍。16歲時在[[謝穎紅|謝穎紅]]<ref>{{cite news |language = zh-hant |url = http://hk.apple.nextmedia.com/sports/art/20091205/13491890 |title = 特稿 謝穎紅掛住江卓儀 |publisher = 蘋果日報 |date = 2009-12-05 |accessdate = 2016-07-30}}</ref>的邀請下從室內排球轉打沙灘排球<ref>{{cite news |language = zh-hant |url = http://hk.apple.nextmedia.com/sports/art/20110317/15083461 |title = 江卓儀： T-back打底唔舒服 |publisher = 蘋果日報 |date = 2011-03-17 |accessdate = 2016-07-30}}</ref> ，並與謝穎紅拍檔參賽，其教練為[[香港培正中學|香港培正中學]]體育部主任[[鄭景亮|鄭景亮]]<ref>{{Cite web |language = zh-hant | publisher = 《體路》（Sportsroad） | url = http://www.tianjinwe.com/rollnews/201011/t20101120_2551592.html |title = 李致和任慈善活力行大使籲多做運動 |date = 2014-10-25 |accessdate = 2016-07-30 |quote= ……以及香港沙灘排球代表隊總教練鄭景亮出席…… }}</ref>。二人組合其後曾兩度代表香港參加[[亞運會|亞運會]]、2008年[[世界錦標賽|世界錦標賽]]、又曾於亞洲巡迴賽取得第七名，並於2010年6月[[土耳其|土耳其]]舉行的[[世界大學生沙排錦標賽|世界大學生沙排錦標賽]]中取得第九名<ref>{{cite news |language = zh-hant |url = http://orientaldaily.on.cc/cnt/sport/20100726/index.html |title = 沙排美女二人組！Go！ |publisher = 東方日報 |date = 2010-07-26 |accessdate = 2016-07-30}}</ref>，是參賽之亞洲隊伍中的最佳成績。

江卓儀小學就讀於[[中華傳道會呂明才小學|中華傳道會呂明才小學]]，中學就讀於[[保良局羅傑承（一九八三）中學|保良局羅傑承（一九八三）中學]](原名：保良局八三年總理中學)<ref> 黎子健、林泳珊. [http://plk83.edu.hk/home/information/ty/ty0708.pdf 青春火花 香港沙灘排球代表隊隊員江卓儀] . 《青意》<sub>（保良局八三年總理中學學生報）</sub>. 2008，（第24期）: 2</ref>，於[[香港理工大學|香港理工大學]]護理系畢業<ref>  [http://www.sportsoho.com/pg/magcontent/757008/- 七年之癢·謝穎紅/江卓儀] . 《運動版圖》（Sportsoho）. 2010-08，（第19期）</ref>。曾於[[屯門醫院|屯門醫院]]任全職護士，2010年為備戰[[廣州亞運會|廣州亞運會]]轉為兼職<ref>{{Cite web |language = zh-hk | publisher = sina全球新聞 | url = http://dailynews.sina.com/bg/news/spt/sptothers/mingpao/20101024/18371934905.html |title = 沙排twins為亞運棄正職星級大師姐——謝穎紅、江卓儀 |date = 2010-10-24 |accessdate = 2016-07-30 |quote= ……為一圓亞運和奧運夢，畢業於理大護理系的Sonia毅然放棄屯門醫院的全職護士工作…… }}</ref>。2007年加入now寬頻電視，兼任沙灘排球評述員及[[英格蘭超級聯賽|英格蘭超級聯賽]]主持，其後任[[香港甲組足球聯賽|香港甲組足球聯賽]]節目主持。
