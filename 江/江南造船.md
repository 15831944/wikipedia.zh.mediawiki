'''江南造船（集团）有限责任公司'''成立于1996年，是[[中国船舶工业集团|中国船舶工业集团]]所属的大型[[造船|造船]]企业，可追溯到成立于1865年的'''[[江南机器制造总局|江南机器制造总局]]'''，后更名为'''江南船坞'''、'''江南造船所'''、'''江南造船厂'''，直至现今的名称。江南造船集团是中国规模最大、设施最先进、生产品种最为广泛的现代化造船基地。<ref name="概况"/>

==历史==
[[File:Shanghai_location_Changxingdao_island.svg|thumb]][[長興島_(上海)|長興島]]]]
[[Image:YuanWang2c.JPG|thumb]]停泊在[[新西蘭|新西蘭]][[奧克蘭港|奧克蘭港]]，2005年10月]]
江南造船是中国历史最悠久的军工造船企业，中国的第一炉[[钢铁|钢铁]]、第一门钢[[炮|炮]]、第一艘铁甲[[兵轮|兵轮]]、第一台万吨[[水压机|水压机]]、第一艘[[潜艇|潜艇]]、第一艘[[护卫舰|护卫舰]]、第一艘自行设计的远洋[[货轮|货轮]]、第一艘海洋考察通信船、第一艘[[油轮#液化气油轮|石油液化气船]]、第一艘跨海[[火车渡轮|火车渡轮]]等都是出自该厂<ref name="概况">{{Cite web |url = http://jnshipyard.cssc.net.cn/compay_mod_file/mode_3.php?cart=1 |title = 企业概况 |work = |publisher = 江南造船（集团）有限责任公司 |date = |accessdate = 2012-10-17 }}</ref>。2008年，江南造船集团整体搬迁至[[长兴岛_(上海)|长兴岛]]的中船[[上海长兴岛船舶制造基地|长兴岛船舶制造基地]]，同时拆分出[[上海江南造船厂|上海江南造船厂]]直接隶属于[[中船集团|中船集团]]，成为一家主营民用工程船舶的公司。<ref>{{Cite web | url =http://shjnshipyard.cssc.net.cn/component_general_situation/ |title = 企业概况 |work = |publisher = 上海江南造船厂 |date = |accessdate = 2012-10-17}}</ref>中國第3艘航母[[003型航空母舰|003型航空母舰]]在江南造船建造中。

==设施==
2008年，江南造船總部搬入长兴岛，占地面积约183.4万平方米，建筑面积46.8万平方米，拥有岸线1364米。主要设施包括：<ref>{{Cite web|url =http://www.shchangxing.gov.cn/website/list.asp?sd=cyfz&psd=kfjs&ssd=zdqyqk|title =重点企业|work =上海长兴岛|publisher =|date =|accessdate =2012-10-17}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
*室内[[船台|船台]]4座
*400米×30米室外船台1座
*240米×40米下水[[船坞|船坞]]1座
*港池1座
*450吨[[门式起重机|门式起重机]]1台
*542米的[[舾装码头|舾装码头]]泊位2个

远期规划为：<ref name="概况"/>
*占地面积580万平方米
*岸线长度3800米
*四座大型[[船坞|船坞]]
*17座[[舾装码头|舾装码头]]
*民用船舶年造船能力450万吨
