'''江南卷柏'''（[[學名|學名]]：''Selaginella moellendorffii'' Hieron.）或名'''異葉卷柏'''，為卷柏科[[卷柏|卷柏屬]]下的一個種，生長在低海拔[[亞熱帶|亞熱帶]][[阔叶林|闊葉林]]中，為常見的地生小型[[蕨類植物|蕨類植物]]（廣義）。

== 特徵 ==
主莖直立，高約25cm，主軸上的小葉貼伏莖上，小葉卵形，長4-5mm，寬1-3mm，全緣，有一不明顯中脈，螺旋狀排列，主軸上小葉排列較側枝疏，植株下半部不分支且莖無毛，上半部2-3回分支。小枝連葉寬3-4mm，小葉為兩形葉，分為側葉及中葉，側葉橫長形，基部歪斜，具緣毛，長1-2mm，寬約0.5mm，中葉卵圓形，具長緣毛，中脈彎曲，末端尾尖，基部圓，長約1mm，寬約0.3mm；孢子葉同形，孢子囊穗四角柱形，長5-10mm，偶見根支體與小枝末端生長褐色不定芽。<ref name=":0">{{Cite web|url=http://140.112.113.3/taiwanferns/observ4.aspx?id=19|title=異葉卷柏|accessdate=2019-03-23|author=郭城孟|date=2010-07-01|publisher=智慧藏學習科技股份有限公司}}</ref><ref name=":1">{{Cite book|title=Flora of Taiwan Second Edition Volume One|last=HUANG(Editor-in-Chief)|first=|publisher=Editorial Committee Flora of Taiwan, Second Edition|year=1994|isbn=957-9019-52-5|location=TAIPEI, TAIWAN, ROC|pages=53}}</ref>

== 習性 ==
江南卷柏為地生小型蕨類，生長在低海拔半遮蔭或陰暗山溝山谷，也可見於山徑道路之陰暗邊坡。<ref name=":0" /><ref name=":1" />

== 分布 ==
江南卷柏常見於中國大陸南部、中南半島、琉球群島與台灣低海拔山區。<ref name=":0" /><ref name=":1" />

== 重要性 ==
異葉卷柏是一屬於[[石松門|石松門]]，且相當古老的[[維管束植物|維管束植物]]，發現在大約4億年前的[[化石|化石]]中，其缺乏[[真叶植物|真葉]]的特徵是植物演化上的重要節點，使其成為植物[[演化|演化]]研究與基因學中，相當重要的[[模式生物|模式生物]]。

== 相似種 ==
[[全緣卷柏|全緣卷柏]]與江南卷柏相似，皆常見於台灣低海拔[[亞熱帶|亞熱帶]][[阔叶林|闊葉林]]中，但其為二回羽狀分枝，側枝末端不分支且外形上為魚骨狀，與異葉卷柏之三回羽狀分枝不同，此為兩種之重要鑑別特徵。
