'''江东新城'''（江东工业园区），是位于[[杭州|杭州]][[萧山区|萧山区]]的一个工业园区和新城区。2009年江东新城与[[萧山经济技术开发区|萧山经济技术开发区]]党工委、管委会实行“一套班子、两块牌子”管理。<ref name="开发区">{{cite web |url=http://www.hangzhou.gov.cn/main/zjhz/hzlj/2010/gjjkfq/T350546.shtml |language= |title=萧山经济技术开发区 |publisher=中国杭州 |date=2011-04-14 |accessdate= |deadurl=yes |archiveurl=https://web.archive.org/web/20130609075728/http://www.hangzhou.gov.cn/main/zjhz/hzlj/2010/gjjkfq/T350546.shtml |archivedate=2013-06-09 }}</ref>江东新城向西通过[[江东大桥_(杭州)|江东大桥]]可到达[[下沙|下沙]]，再通过杭州[[德胜路|德胜快速通道]]到达市区。

== 简介 ==
江东工业园区于2001年开始筹建，2006年批准成为省级开发区。它位于[[钱塘江|钱塘江]]以东，萧山北部。目前涵盖[[义蓬街道|义蓬]]、[[河庄街道|河庄]]和[[新湾街道|新湾]]三个街道，总规划面积为148平方公里。<ref name="简介">{{cite web |url=http://www.xetz.gov.cn/gb/kfqgk/kfqjj.htm |language= |title=开发区简介 |publisher=萧山经济技术开发区 |date= |accessdate=2012年9月 |deadurl=yes |archiveurl=https://web.archive.org/web/20120815154842/http://www.xetz.gov.cn/gb/kfqgk/kfqjj.htm |archivedate=2012-08-15 }}</ref>

新城目标建设为以先进制造业为主体，高新技术产业和现代服务业为先导，集科研、金融、高端商务、现代物流、人居休闲等功能于一体的现代化工业新城。目前基本建成首期11.29平方公里，已引进71个项目，投产企业29家，总投资近300亿元。2011年工业总产值为126.7亿元。<ref name= "简介"/>

== 交通 ==
规划中的[[杭州地铁8号线|杭州地铁8号线]]西起[[下沙|下沙]]二号大街，与[[杭州地铁1号线|1号线]]形成单向同站台转乘，过江后东至江东新城。
