'''{{FULLPAGENAME}}'''是[[江苏省人民政府|江苏省人民政府]]司法厅监狱管理局的直辖单位，位于[[句容市|句容市]][[边城镇_(句容市)|边城镇]]小茅山<ref>{{Cite web |url=http://www.jsjy.gov.cn/jsjyw/zsdw/2016-06/29/content_a43426ce104a438da884c96020b1e7ca.shtml |title=边城监狱 |accessdate=2018-11-03 |author=|date= |work= |publisher=江苏省监狱管理局网站|language=Zh-hans }}</ref>。

1985年时，由原江苏省古城少年犯管教所、国营小茅山陶瓷厂组建，历称江苏省第二少年犯管教所、江苏省第二少年管教所<ref name="a1"/><ref name="a2"/>、江苏省第二未成年犯管教所<ref>{{Cite web |url=http://lxyz.ujs.edu.cn/show.asp?id=377 |title=访韦岗，缅怀先烈，继承遗志复兴民族;观监狱，拒腐防变，争先创优永葆先进-理学院大学物理支部开展党员教育活动 |accessdate=2018-11-03 |author=|date=2016-11-16  |work= |publisher=江苏大学网站|language=Zh-hans }}</ref>。2008年11月，改称现名。2011年时，曾与[[江苏省镇江中学|镇江中学]]签署共建德育教育基地和对未成年犯实施帮教的协议<ref name="a1">{{Cite web |url=http://jszjzx.zje.net.cn/Front/InfoDetail/?InfoID=0c8adec2-16ae-43b5-aa79-e550fe933b2b&CategoryNum=013001 |title=省镇中与省第二少管所共建德育基地  |accessdate=2018-11-03 |author=|date=2011-02-22 |work= |publisher=江苏省镇江中学门户网站|language=Zh-hans }}</ref>，双方有合作关系<ref name="a2">{{Cite web |url=http://jszjzx.zje.net.cn/Front/InfoDetail/?InfoID=789150a2-198e-469c-bb1f-ed6a30866b53&CategoryNum=014005 |title=为学生守护一片纯净的天空——江苏省镇江中学法制工作总结  |accessdate=2018-11-03 |author=|date=2011-06-01  |work= |publisher=江苏省镇江中学门户网站|language=Zh-hans }}</ref>。至2018年8月，边城监狱已连续29年实现无脱逃、无狱内重特大案件、无重大疫情、无工伤死亡责任事故的“四无”目标，保持江苏省监狱系统“四无”时间最长纪录<ref>{{Cite web |url=http://gx.legaldaily.com.cn/content/2018-09/11/content_7642740.htm?node=77168 |title=边城监狱连续第29年实现“四无”目标 |accessdate=2018-11-03 |author=通讯员：陈正愿|date=2018-09-11 |work=李纪平 |publisher=法制网|language=Zh-hans }}</ref>。

2012年及之前，在[[国家统计局|国家统计局]]网站统计中<ref>{{Cite web |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2012/32/11/83/321183418.html |title=江苏省第二少年管教所 |accessdate=2018-11-03 |author=|date= |work= |publisher=国家统计局网站|language=Zh-hans }}</ref>，所在区域以'''江苏省第二少年管教所'''之名列为句容市的一个[[类似乡级单位|类似乡级单位]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|32/11/83/418/000}}}}</ref>。
