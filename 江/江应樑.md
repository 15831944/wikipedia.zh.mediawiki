'''江应樑'''（{{bd|1909年|2月|1988年|}}），祖籍[[广西壮族自治区|广西]][[贺县|贺县]]，[[客家人|客家人]]<ref>{{cite web |url=http://bianjiang.cssn.cn/sykz/sy_bjxj/201303/t20130315_1810946.shtml |title=江应樑：探秘滇西 |author=许晓明 |date=2013-03-15 |website=bianjiang.cssn.cn |publisher=《[[中国民族报|中国民族报]]》 |access-date=2018-12-19 |quote=}}</ref>，[[中国民主同盟|民盟]]盟员，中国历史学家、人类学家、少数民族史专家。

==生平==

江应樑1909年出生于[[云南省|云南]][[昆明市|昆明]]，1932年从[[暨南大学|国立暨南大学]]历史社会学系毕业，1936年入[[中山大学|中山大学]]攻读人类学硕士，师从历史学家[[朱谦之|朱谦之]]和[[杨成志|杨成志]]，1937年至滇西傣区考察[[傣族|傣族]]。同年获得毕业，次年执教中山大学历史系，后辗转于多所高校任教。1948年起任[[云南大学|云南大学]]教授。曾任中国民族学会、中国人类学会、中国社会学会顾问、西南边疆民族历史研究所所长、中国人类学学会第一届主席团成员。<ref>{{cite web |url=http://www.cuaes.org/scholardetail-11.htm |title=专家学者——江应樑 |website=www.cuaes.org |publisher=中国人类学民族学研究会 |access-date=2018-12-19 |quote=}}</ref><ref>{{cite web |url=https://www.kunming.cn/news/c/2017-04-14/4586079.shtml |title=八十年的坚持 两代人的传承——记德宏师范高等专科学校“江应樑傣族博物馆” |author=雷强 |date=2017-04-14 |website=www.kunming.cn |publisher=昆明信息港 |access-date=2018-12-19 |quote=}}</ref>著有《傣族史》、《摆夷的经济生活》、《摆夷的生活文化》、《明代雲南境內的土官與土司》、《百夷傳校注》、《西南邊疆民族志》、《西南边疆民族论丛》等，主编有《中国民族史》。

2017年，[[德宏师范高等专科学校|德宏师范高等专科学校]]建立“江应樑傣族博物馆”，展出[[傣族|傣族]]铜器、历史典籍以及江应樑傣学研究的成果和他搜集、使用过的物品、在德宏考察拍摄的照片，博物馆藏有的数千件傣族文物均是江应樑及其子江晓林多年搜集的藏品。该博物馆也是中国唯一一所真正意义上的傣族博物馆。<ref>{{cite web |url=http://yn.chinadaily.com.cn/a/201704/13/WS5bdbe120a3101a87ca90a02a.html |title=全国唯一一所真正意义上的傣族博物馆在云南德宏师专开馆 |date=2017-04-13 |website=yn.chinadaily.com.cn |publisher=中国日报网 |access-date=2018-12-19 |quote=}}</ref><ref>{{cite web |url=http://zqb.cyol.com/html/2017-04/17/nw.D110000zgqnb_20170417_4-06.htm |title=父子两代用80年建成一个傣族博物馆 |author=张文凌 |date=2017-04-17 |website=zqb.cyol.com |publisher=《[[中国青年报|中国青年报]]》 |access-date=2018-12-19 |quote=}}</ref>
