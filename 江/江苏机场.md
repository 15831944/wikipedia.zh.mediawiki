[[江苏省|江苏省]]境内现有九座[[民航机场|民航机场]]、五座[[军用机场|军用机场]]，四座[[直升机场|直升机场]]。另有[[通用机场|通用机场]]分布于十三个[[地级市|地级市]]。

1912年在南京诞生的[[小营机场|小营机场]]，仅比中国最早的机场——[[南苑机场|南苑机场]]晚两年。此后，[[国民政府|国民政府]]在江苏省修建了南京小营、明故营、大校场、徐州九里山、大郭庄、海州杨圩、常州洪庄等25个中小型[[军用机场|军用机场]]（部分为军民合用机场）。这些小型机场在中华人民共和国建立后逐渐废弃{{noteTag|name=noteA}}。

[[中华人民共和国|中华人民共和国]]成立后，[[华东军区|华东军区]]重建了九里山机场。[[南京军区|南京军区]]则新建了常州奔牛机场、连云港白塔埠等军用机场。[[大跃进|大跃进]]前，[[中华人民共和国民用航空业|民用航空]]接受地方政府领导。1950年代末，[[江苏省运输厅|江苏省运输厅]]投资新建了淮阴、南通两地的小型民航机场（另有[[新海连|新海连]]机场，未通航，是否省厅投资不详），开辟[[地方航线|地方航线]]，但未能维持运营。至1962年4月，地方航线全部停航，部分国内航线停航{{noteTag|name=noteD}}。

1974年，民航江苏局客运人数突破万人{{noteTag|name=noteE}}。[[改革开放|改革开放]]后，[[江苏经济|江苏经济]]得到长足发展。1985年常州奔牛机场、连云港白塔埠机场转为军合用机场，同年，江苏航空客运人数突破十万人{{noteTag|name=noteE}}。此后又重建了[[南通兴东机场|南通兴东机场]]。

与[[公共航空运输|公共航空运输]]相比，江苏省的[[通用航空|通用航空]]规模不大。2010年，仅有12部通用航空飞行器，全年飞行时长2550小时。2015年时，江苏共有8个通用机场，6座运营中。从业人员130人，其中飞行员36人<ref name="jiangsu1"/>，全年飞行时长6558小时。2016年，有[[通用航空企业|通用航空企业]]12家，占全国4.1%，通用航空飞行器增至31部，占全国1.5%<ref>{{Cite web |url=http://www.jiangsu.gov.cn/jsyw/201603/t20160331_426899.html|title=我省筹建10个通用航空机场|accessdate=2017-08-15 |author= |date=2016-03-31 |work= |publisher=江苏省人民政府网站，来源：新华日报|language=Zh-hans }}</ref>。在[[江苏省人民政府|江苏省人民政府]]的计划中，[[十三五规划|十三五规划]]中布局16座。十三五规划结束时，即2020年，布局35座。远期达到70座<ref name="jiangsu1">{{Cite web |url=http://www.sohu.com/a/152456297_99912517 |title=江苏省规划布局10个国家级、60个区域级通用机场|accessdate=2017-08-14 |author= |date=2017-06-27 |work= |publisher=搜狐网|language=Zh-hans }}</ref>。但地级市的计划则可能超过省内计划（参见：泰州市）。
