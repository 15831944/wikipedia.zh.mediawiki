'''江陵之战'''是[[东汉|东汉]]末年[[孙权|孙权]]和[[刘备|刘备]]联军对抗[[曹操|曹操]]的战事。此战是赤壁军事行动的一部分，在208年的[[夷陵之战_(208年)|夷陵之战]]后的[[乌林|乌林]]陆地遭遇战和曹操军在赤壁之战的水战中溃败后立刻开战。注意乌林的战事是赤壁之战的副产物，不可混同。

江陵城與當陽北方周围激战正酣时，[[荆南|荆南]]却没有太多激烈的战斗。由于不能把江陵从支援它的城市（[[益州|益州]]诸城除外，详见[[夷陵之戰_(208年)|夷陵之战]]）中孤立出来，战事演变成消耗战，最终因曹操军大量伤亡告终。约一年后，曹操无力应对人力物力的持续损耗，命曹仁撤离江陵。<ref>陈寿《三国志·吳主傳》：“瑜、仁相守歲余，所殺傷甚眾。仁委城走。”</ref>

== 背景 ==
赤壁大胜后，孙刘联军实施下一步战略，试图将撤退的曹操军向江陵（在今荆江，非今湖北[[江陵县|江陵县]]）驱赶，夺取[[南郡|南郡]]。

== 战役 ==
===绝曹仁粮道===
周瑜担心曹操分散在各处战略要地并未受损的各队精兵共计十万，刘备建议周瑜断绝江陵的补给，以赶走曹仁。故周瑜劝刘备派[[关羽|关羽]]通过渗透战斗阻断曹仁的粮道，让关羽越过江陵的战略要点攻击敌后，孤立江陵，以便协同进攻。刘备命[[张飞|张飞]]、关羽统兵。关羽受命，率军北上阻拦敌援军，阻断主要道路。<ref>陈寿《三国志·魏志十八·李通傳》：“刘备与周瑜围曹仁于江陵，别遣关羽绝北道。”</ref>关羽和[[苏非|苏非]]（未知是否即吴将[[苏飞|苏飞]]）率水军和二千人的偏师沿[[汉水|汉水]]而上<ref>陈寿《三國志·吳志九·周瑜傳》裴注引《吳錄》：備謂瑜云：「仁守江陵城，城中糧多，足為疾害。使張益德將千人隨卿，卿分二千人追我，相為從夏水人截仁後，仁聞吾入必走。」</ref>，关羽的二千人北上深入曹軍防線二百里，攻擊江陵北方的[[汉津|汉津]]等地。把守[[襄阳|襄阳]]的[[乐进|乐进]]發兵驱逐。<ref>陈寿《三国志·魏志十七·樂進傳》：“留屯襄阳，击关羽、苏非等，皆走之。”</ref>曹仁遣派江陵的徐晃與[[当阳|当阳]]的[[滿寵|滿寵]]聯軍企圖夾擊奪回关羽控制的漢津口，但是沒有成功。<ref>陈寿《三国志·吳主傳》：“备、瑜等复追至南郡，曹公遂北还，留曹仁、徐晃於江陵，使乐进守襄阳。”</ref><ref>《三国志·魏志十七·徐晃傳》:「又与满宠讨关羽於汉津。」</ref><ref>陈寿《三国志·魏志二十六·滿寵傳》：「建安十三年，从太祖征荆州。大军还，留宠行奋威将军，屯当阳。」</ref>

关羽數千人又東上繞過駐軍當陽的满宠，佔據[[荆城|荆城]]（今湖北荊門市钟祥市西南）並攻擊[[寻口|寻口]]（鍾祥市漢水西岸），屯駐[[夏口|夏口]]的[[文聘|文聘]]與乐進趕往救援。<ref>陈寿《三国志·魏志十八·文聘傳》：“与乐进讨关羽於寻口，有功，进封延寿亭侯，加讨逆将军。”</ref>文聘水軍又去搶奪关羽在汉津的辎重物資，並在[[荆城|荆城]]燒毀关羽的战船<ref>陈寿《三国志·魏志十八·文聘傳》：“又攻羽辎重於汉津，烧其（关羽）船於荆城。”</ref>，成功打退關羽軍的進攻。但北方的乐進、满宠、文聘等人，也始終沒有南下江陵救援曹仁，也無法攻擊周瑜軍。北方數萬曹軍被跟關羽連續對峙與戰鬥牽制，刘备與周瑜断绝江陵外援的作戰獲得一定的成果。

===刘备與刘琦收復荆南之戰===
荆州在长江以南还有四郡在曹操管下，当时北上驅逐關羽未果的[[徐晃|徐晃]]南歸，跟曹仁會師攻擊周瑜<ref>陈寿《三国志·魏志十七·徐晃傳》：「与曹仁击周瑜於江陵。」</ref>。正是曹仁與周瑜大戰方酣之際。刘备以[[刘琦|刘琦]]为荆州刺史帶部南下進兵荆南。四郡[[武陵|武陵]][[金旋|金旋]]<ref>《三辅决录注》则载金旋反抗刘备失败被杀。</ref>、[[长沙|长沙]][[韩玄|韩玄]]、[[桂阳|桂阳]][[赵範_(三国)|赵範]]、[[零陵|零陵]][[刘度|刘度]]皆望風歸降。之后刘备以[[诸葛亮|诸葛亮]]为[[军师中郎将|军师中郎将]]，总督长沙、桂阳、零陵政务。

同時[[庐江|庐江]]的[[雷绪|雷绪]]军主动来投，使他的军力大增。<ref>陈寿《三国志·先主传》：“廬江雷緒率部曲數万口稽顙。”</ref>雷绪在得知曹操乌林败绩后，就在[[庐江|庐江]]起事。曹操先前将旧部分散在江北以防叛乱的部署起了作用：他很快派[[夏侯渊|夏侯渊]]召集军队平乱，雷绪虽战败且失去地盘，但率部数万口投靠刘备，增强了刘备的力量。

=== 僵持 ===
孙刘联军虽然也受到了损失，但和曹操军相比就显小了。几个月前，曹仁在一次夺回夷陵的尝试中损失了三千多精骑兵，他和来援的[[徐晃|徐晃]]也无力压制独力守御周瑜大营的凌统。因此江陵守军士气低落，曹仁也知道自己需要采取措施改变战局。为防士气继续低落，他募得士兵三百人，<ref>陈寿《三国志》卷九诸夏侯曹传第九：“仁登城望之，乃募得三百人。”</ref>组成由将军[[牛金|牛金]]率领的突击队，希望他们在战场上取得小胜或表现出勇敢来振奋士气。当敌军前部来到城外时，这股小部队被围。曹仁令帐下数十壮士准备相救，[[长史|长史]][[陈矫|陈矫]]劝止：“敌军士气正盛，就算我们损失几百人又如何。”<ref>陈寿《三国志》卷九诸夏侯曹传第九：“贼众盛，不可当也。假使弃数百人何苦，而将军以身赴之！”</ref>曹仁无视陈矫恳求，出城冲入敌军。陈矫看不到曹仁，以为他阵亡了。但出人意料的是，曹仁第一次尝试就救出了牛金，又回头去救其余的人。曹仁率部安全退回城中，他和牛金部的战死者合计也只有数人。陈矫吃惊之余叹道：“将军真天人也。”曹操很快得知，因曹仁在此战中的英勇表现赐封他为[[安平亭侯|安平亭侯]]。受到嘉奖的曹仁在城外扎营。周瑜亲自率军突袭曹仁营，一次右肋中箭，受了重伤。<ref>陈寿《三国志·吳志九·周瑜傳》：“瑜親跨馬擽陳，會流矢中右脅，瘡甚。”</ref>

=== 曹仁撤退 ===
战爭過了一年，曹操雖持续增派援军與北方諸将意圖突破聯軍對江陵的封鎖線，豫州的太守[[李通_(三国)|李通]]企圖迎回曹仁從江陵北上突圍，但戰到中途李通即突然病死<ref>陈寿《三国志·魏志十八·李通傳》：“通率众击之，下马拔鹿角入围，且战且前，以迎仁軍，勇冠諸將，通道得病薨”</ref>。最終，曹仁軍傷亡者眾多，堅持不下而退出江陵。<ref>陈寿《三國志·吳主傳》</ref>此戰後，孙权军控制長江中下游，以及與西蜀的[[劉璋|劉璋]]接壤，实现了攻取长江上游要地南郡的目标。另一方面，劉備則在荊南四郡培值力量，以趙雲領桂陽太守，让[[向朗|向朗]]总督[[秭歸|秭歸]]、[[夷道|夷道]]、[[巫山|巫山]]、[[夷陵|夷陵]]四縣。<ref>陈寿《三国志·蜀志十一·向朗傳》：“先主定江南，使朗督秭歸、夷道、巫（山）、夷陵四縣軍民事。”</ref>就此，孙权則以黃蓋領武陵太守，全柔領桂陽太守。而周瑜就分[[公安|公安]]給劉備屯駐。

== 戰後影響 ==
孫權跟劉備於戰後，就荊州分配權出現矛盾，劉備為此更主動找孫權要求土地。而孙权就基於政治考量，害怕他反叛，故安排妹妹嫁给刘备。<ref>陈寿《三国志》卷三十二《蜀书·先主传》：“权稍畏之，进妹固好。”</ref>直至周瑜死後，親劉的魯肅上台，建議讓劉備督導荊州，才延遲雙方因荊州問題而決裂。刘备自此成為完全獨立的軍閥，也具备了入侵[[益州|益州]]的条件，并将于211年进军益州。

因为随着双方对抗的停止和战斗转为围城，曹操于209年3月退回位于[[谯|谯]]的前沿基地，孙权也放弃了东线对合肥的进攻，传统认为江陵之战是赤壁军事行动的完结，江陵围城战的余下战事也不再被大多史学家认作赤壁军事行动的一部分了。孙权攻陷江陵通常被认为是赤壁军事行动的战果。

== 演绎 ==
为了戏剧效果，在很多文学作品中，刘备征服江南四郡被写成分离的战事，并有关羽和[[黄忠|黄忠]]的对决，成为[[京剧|京剧]]等其他文艺的创作资源。事实上，这些都并非事实。

与[[罗贯中|罗贯中]]在历史小说《[[三国演义|三国演义]]》所述相反的是，韩玄并未被[[魏延|魏延]]所杀，史书也没记载魏延在此时加入刘备军乃至参与此战。
