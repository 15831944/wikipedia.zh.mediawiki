'''江苏省演艺集团有限公司'''，是[[江苏省|江苏省]]唯一一家省属演艺类文化企业，也是[[中华人民共和国|中华人民共和国]]规模最大的综合性文艺团体之一。

==历史==
江苏省演艺集团有限公司是中华人民共和国最早的文化体制改革试点单位，也是最早尝试文艺院团[[转企改制|转企改制]]的演艺团体之一。2001年9月，江苏省演艺集团成立，成为中华人民共和国第一个演艺集团。当时，江苏省将全部省属艺术院团——江苏省歌剧舞剧院、江苏省京剧院、[[江苏省昆剧院|江苏省昆剧院]]、江苏省锡剧团、江苏省扬剧团和江苏人民艺术剧院、江苏省人民剧场、江苏省演出公司、江苏文化音像出版社等单位整合，成立江苏省演艺集团，实现了文化行政管理部门同文艺表演团体政事分开、管办分离<ref name=lu>{{Cite web|url =http://cul2.jschina.com.cn/system/2015/02/17/023775833.shtml|title =江苏省演艺集团改革创新繁荣发展之路|author =|date =2015-02-17|publisher =中国江苏网|language =|accessdate =|quote =}}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。

江苏省演艺集团成立后，将原来各单位的人事、财务、行政、后勤管理权以及演出营销权等统归总部调度。江苏省演艺集团设艺术表演、创作、管理三大类12种工作岗位，择优上岗，实行5%末位淘汰制，工资实行“岗位工资+绩效工资”。将艺术生产体制从以获奖为参照改为以市场和获奖有机结合为参照<ref name=lu/>。

2004年8月，[[中共江苏省委|中共江苏省委]]、[[江苏省人民政府|江苏省人民政府]]联合下发《江苏省演艺集团深化改革的实施意见》，江苏省演艺集团全员转企改制，从[[事业单位|事业单位]]转为企业。转企改制后，政府从花钱养人为主改为通过政府采购、补贴等经济手段支持艺术创作为主<ref name=lu/>。

2006年3月8日，由江苏省演艺集团相对控股的江苏演艺文化产业股份有限公司成立<ref>{{Cite web |url =  http://ent.sina.com.cn/x/2006-03-09/08421010257.html|title =
江苏省演艺集团启动股份制改造  |author =  |date =2006-03-09  |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref>。

2009年，江苏省演艺集团经营收入在中华人民共和国国有文艺院团中率先“破亿”。2009年后，演出场次通常保持在每年超过5000场<ref name=lu/>。

江苏省演艺集团成立后，先后排演了许多剧目和节目，例如[[昆剧|昆剧]]《1699·桃花扇》、《南柯梦》，[[京剧|京剧]]《飘逸的红纱巾》、《镜海魂》，江苏民间歌舞《好一朵茉莉花》，[[扬剧|扬剧]]《王昭君》，[[锡剧|锡剧]]《珍珠塔》，[[话剧|话剧]]《世纪彩虹》、《枫树林》，[[评弹|评弹]]《唐宋古韵忆江南》、[[歌剧|歌剧]]《运之河》、[[音乐剧|音乐剧]]《锦绣过云楼》等。其中，昆剧《1699·桃花扇》获[[中共中央宣传部|中共中央宣传部]]“[[五个一工程|五个一工程]]”奖、文华大奖特别奖；京剧《飘逸的红纱巾》获中共中央宣传部“五个一工程”奖、第五届中国京剧节现代戏金奖第一名、文华优秀剧目奖；歌剧《运之河》获第二届中国歌剧节优秀剧目奖；话剧《枫树林》获第十四届文华奖文华大奖；爱之旅合唱团获全国合唱比赛银奖、6项国际性合唱艺术节金奖<ref name=lu/>。

[[中共江苏省委|中共江苏省委]]宣传部负责对江苏省演艺集团的工作实施方针政策的指导。中共江苏省委宣传部受中共江苏省委委托，与中共江苏省委组织部共同负责提名、考察与管理江苏省演艺集团的领导干部<ref>{{Cite web|url = http://www.jswmw.com/9598/201602/t2680537.shtml|title = 中共江苏省委宣传部2016年度部门预算|author = |date = 2016-02-24|publisher = 江苏文明网|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170223163905/http://www.jswmw.com/9598/201602/t2680537.shtml|archive-date = 2017-02-23|dead-url = yes}}</ref>。

==直属院团==
江苏省演艺集团有限公司下辖直属院团11个<ref name=jj>{{Cite web |url = http://www.jsyanyi.cn/jianjie/view.asp?id=341 |title = 集团简介 |author =  |date =  |publisher = 江苏省演艺集团有限公司 |language =  |accessdate = 2017-04-30 |quote =  }}</ref>：

*江苏省演艺集团歌舞剧院：1956年12月成立江苏省歌舞团。1959年12月在该团基础上成立江苏省歌舞话剧院。1962年9月撤销江苏省歌舞话剧院，恢复江苏省歌舞团。1987年1月成立江苏省歌舞剧院<ref>{{Cite web|url = http://app.nanjing.gov.cn/njnj/sz8_12/10/html/10Noname006.html|title = 第三节 舞蹈|author = |date = |publisher = 中国南京|language = |accessdate = 2017-04-30|quote = }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
*江苏省演艺集团京剧院：1953年，由苏南大众京剧团、苏北实验京剧团合并成立江苏省大众京剧团。1955年更名为江苏省京剧团。1960年5月，以江苏省京剧团为基础，合并南京市青年京剧团、南京市京剧团、扬州专区京剧团部分演职员，在南京成立江苏省京剧院<ref name=xiju/><ref>{{Cite web |url = http://www.taizhou.gov.cn/art/2016/3/3/art_29206_586465.html |title = 江苏省演艺集团京剧院 |author =  |date = 2016-03-03 |publisher =中国泰州  |language =  |accessdate =  |quote =  }}</ref>。
*江苏省演艺集团昆剧院：1960年，上级抽调省苏昆剧团的张继青等13位“继”字辈演职员来南京建立江苏省昆剧团一团，留苏州的改为江苏省昆剧团二团。1972年一、二团合并，驻苏州。1977年一团调回南京，1977年11月更名为江苏省昆剧院，驻南京<ref name=xiju/><ref>{{Cite web |url = http://www.china.com.cn/chinese/zhuanti/153379.htm |title =  江苏省昆剧院|author =  |date =  |publisher =  中国网|language =  |accessdate =2017-04-30  |quote =  }}</ref>。
*江苏省演艺集团话剧院：1953年成立江苏省话剧团。1989年8月更名为江苏人民艺术剧院<ref>{{Cite web |url =http://news2.jschina.com.cn/system/2013/11/27/019441615.shtml  |title = 三台好剧连轴转 江苏省话剧院12月上旬在宁演出 |author =  |date = 2013-11-27 |publisher = 中国江苏网 |language =  |accessdate =  |quote =  }}</ref><ref name=xiju>{{Cite web|url = http://app.nanjing.gov.cn/njnj/sz8_12/10/html/10Noname004.html|title = 第一节 戏剧|author = |date = |publisher = 中国南京|language = |accessdate = 2017-04-30|quote = }}{{Dead link|date=2018年10月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
*江苏省演艺集团锡剧团：1953年4月在南京成立江苏省锡剧团。以苏南文联锡剧实验剧团作为基础，吸收苏南文工团部分团员组成<ref name=xiju/>。
*江苏省演艺集团扬剧团：1953年4月在南京成立江苏省扬剧团。以苏北实验扬剧团作为基础，吸收苏南、苏北文工团部分团员组成。后来又有南京实验剧团、江苏省青年扬剧团部分演职员参加，并自[[上海|上海]]、[[扬州|扬州]]调入部分主要演员<ref name=xiju/>。
*江苏省演艺集团交响乐团：1979年组建江苏省歌舞团管弦乐队。1987年1月成立江苏省歌舞剧院管弦乐团。1995年10月与江苏省歌剧团乐队合并组建江苏省歌舞剧院交响乐团<ref>{{Cite web |url = http://www.musiceol.com/news/html/2010-9/201091513391854558846.html |title = 江苏交响乐团 |author =  |date = 2010-09-15 |publisher =  中音在线|language =  |accessdate =  |quote =  }}</ref><ref name=yinyue/>。

*江苏省演艺集团飞扬乐团：2007年1月成立江苏省演艺集团飞扬乐团。
