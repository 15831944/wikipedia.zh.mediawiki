'''江陵县'''位于[[湖北省|湖北省]]中南部，[[江汉平原|江汉平原]]腹地、[[荆江|荆江]]河段北岸。全县版图呈[[马鞍形|马鞍形]]，东西距53.5千米，南北距36.2千米。面积1032平方千米。2010年末总人口40.5万人。

== 历史沿革 ==
[[江陵|江陵]]，因“地临江”、“近州无高山，所有皆[[陵阜|陵阜]]”而得名，自[[楚庄王|楚庄王]]元年（前689年）至[[秦昭襄王|秦昭襄王]]二十九年（前278年，秦将[[白起|白起]]拨[[郢|郢]]，在郢始置江陵县），20 代[[楚王|楚王]]在江陵建都，延续411年，成就了作为[[春秋五霸|春秋五霸]][[战国七雄|战国七雄]]的宏图大业。其后至[[宋太祖|宋太祖]][[乾德_(北宋)|乾德]]元年（963年）[[荆南|荆南]]（南平）国亡，先后又有14代帝王在此建都。江陵作为中国的[[历史名城|历史名城]]之一，除建都外，还一直是郡、府、州、路治所，中华人民共和国成立后一直为[[荆州地区|荆州地区]][[行政公署|行政公署]]所在地。1994年，经[[国务院|国务院]]批准撤销荆州地区、[[沙市市|沙市市]]及'''江陵县'''，设立[[荆沙市|荆沙市]]（1996年12月更名为荆州市）及荆州、沙市、江陵三区，原江陵县一分为三，江陵区人民政府驻[[郝穴镇|郝穴镇]]。1998年7月，江陵撤区置县，恢复县治。

== 经济 ==
2010年全县完成地区生产总值34.35亿元，全县人均地区生产总值10364.1元，产业结构为44:23:33。

== 行政区划 ==
江陵县辖[[郝穴镇|郝穴]]、[[资市镇|资市]]、[[滩桥|滩桥]]、[[熊河|熊河]]、[[白马寺|白马寺]]、[[沙岗|沙岗]]、[[普济|普济]]7个镇和[[马家寨|马家寨]]、[[秦市|秦市]]2个乡及[[三湖|三湖]]、[[六合垸|六合垸]]2个管理区，共11个乡镇管理区，共有197个行政村和19个社区居委会和7个生产大队。[[江北监狱|江北监狱]]亦在辖区之内。
2010年全县年末总户数11.94万户，总人口40.5万人，其中常住人口33.1万人，乡村人口27.9万人（含[[丹江口|丹江口]][[移民|移民]]2749人），[[城镇人口|城镇人口]]12.6万人，新出生人口3766人，死亡人口1823人。

== 自然地理 ==
位于东經112 °12′52″至112°44′22″，北纬29°54′36″至30°16′45″。
* 属北[[亚热带|亚热带]][[季风|季风]]湿润气候区；
* 年平均气温16℃～16.4℃，极端值最热39.2℃，最冷-19℃；
* 无霜期246天～262天；年平均降雨量900～1100毫米。

== 历史遗迹 ==
* [[镇安寺铁牛|镇安寺铁牛]]建于清咸丰九年(公元1859年)，是[[中华第一堤|中华第一堤]]的标志，是人民群众战胜洪水的象征。建国后，[[周恩来|周恩来]]、[[李鹏|李鹏]]、[[江泽民|江泽民]]、[[朱镕基|朱镕基]]、[[李瑞环|李瑞环]]、[[温家宝|温家宝]]等党和国家领导人都曾到此视察，指导抗洪。
* [[沙岗红军街|沙岗红军街]]原名[[沙岗老街|沙岗老街]]，全长0.6公里,1928年至1932年的五年间，先后有中共[[湘鄂西|湘鄂西]]省委、省[[苏维埃|苏维埃]]、江陵县苏维埃及10多处红军工作机构设在红军街。[[贺龙|贺龙]]、[[段德昌|段德昌]]、[[周逸群|周逸群]]、[[柳直荀|柳直荀]]等共产党人曾在此战斗。“[[沙岗年关暴动|沙岗年关暴动]]”是湘鄂西暴动的开始，创建了[[白鹭湖|白鹭湖]][[游击大队|游击大队]]和[[中国工农红军第六军|中国工农红军第六军]]。[[File:江陵铁牛.JPG|355px]]
