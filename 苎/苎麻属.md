'''苎麻属'''（[[拼音|拼音]]：zhù，[[注音|注音]]：ㄓㄨˋ，[[学名|学名]]：''{{lang|la|Boehmeria}}''）属[[荨麻科|荨麻科]]植物。其中苎麻（[[学名|学名]]：''{{lang|la|Boehmeria nivea}}''）分成白葉種及綠葉種。

== 形态 ==
苎麻属为[[多年生草本|多年生草本]][[宿根植物|宿根植物]]。地下部分由根和地下茎形成麻兜，可活数十年，茎丛生，背有茸毛，[[叶|叶]]广卵形或近圆形[[叶|叶]]，背面密生白茸毛為白葉種，而無白茸毛稱綠葉種，单性[[花|花]]，雌雄同株，复穗状[[花序|花序]]，雌花在花序的上部，黄绿色瘦果，极小。並且單纖維長度是麻類中最長的(60-250毫米)。《[[本草綱目|本草綱目]]》云：“苧，家苧也；又有山苧，野苧也；有紫苧，葉面紫；白苧，葉面青；其背皆白。”

==习性==
喜光和温暖湿润气候，耐旱性弱，常用种子或分地下莖(吸枝法)、分株、扦插、压条等法繁殖，一年可收三次。收获苎麻多由中老年婦女帶頭,下田割苎麻。苎麻割下後,先摘葉,再去骨(葉可用來湿肥,骨可曬乾當柴燒)，剩下的苎麻皮便带回家中,準備取纖維了。提取苎麻纖維的工具很特别,它由一把呈長弧形的鐵刀片(刀片的兩邊開鈍口)。织麻布首先要「[[绩|绩]]麻」，将一段一段的麻丝捻成纱线。至今仍说的成绩、功绩之绩即从此出。绩麻是个非常精细烦难的工作。

== 用途 ==
[[File:Meister_nach_Chang_Hsüan_001.jpg|thumb]]
茎部韧皮纤维有光泽，耐霉、易染色，为重要的纺织作物，优质纸原料。且与华夏文化传统连接深厚。<ref>[http://ctc.webtex.cn/info/2010-8-13@429968.htm 苎麻产业]</ref>古代的五种麻之一。<ref>据史料记载：“古者先布以苎始，棉花至元始入中国，古者无是也。所为布，皆是苎，上自端冠，下讫草服。”《文献通考》“麻之品五：曰青麻、白麻、黄麻、冬、苎麻。”</ref>苎麻所织的布称为[[夏布|夏布]]。<ref>《闽书·南产志》: "闽中名苧麻织之曰缭，以成夏布，其用行天下。"</ref><ref>嘉庆《南平县志》卷一: " (苧布)各乡多有,唯细密精致,几类纱罗,曰锎板,出峡阳者佳,远市四方。"</ref>夏布也就是夏天所用的布。<ref>《周书》：“葛，小人得其叶，以为羹；君子得其材，以为絺绤，以为君子朝廷夏服”。</ref><ref>《宋史》“且先王之服，冬裘夏葛以适寒暑，未有能易之者也。”</ref><ref>[[文震亨|文震亨]]《[[长物志|长物志]]》“衣冠制度,必与时宜。吾侪既不能披鹑带索，又不当欲缀垂珠，要须夏葛冬裘，被服娴雅，居城市有儒者之风，入山林有隐逸之象。”</ref>从种麻开始，每年春、秋季各摘採一次，经过打麻、挽麻团、挽麻芋子、牵线、穿扣、刷浆、织布、漂洗、整形、印染等工序纯手工纺织成布。新夏布如很硬，必须放在水锅里适量放入碱水，进行煮练，取出用衣槌捣练，使它柔软。幅宽二尺二寸。每匹大约45-60厘米宽。苎麻成布之后，需加灰锻濯漂白，<ref>郑玄注《礼记·深衣》:"深衣者，用十五升布，锻濯灰治，绳之以采。"</ref>制成白纻、白紵細布。<ref>鄭玄注“白而細疏曰紵。”</ref><ref>《[[本草纲目|本草纲目]]》十五“時珍曰︰苧麻作紵，可以績紵，故謂之紵。凡麻絲之細者，為絟；粗者為紵。 陶弘景雲︰苧即今績苧麻是也。麻字從廣，從林（音派），象屋下林麻之形。”</ref>《[[礼记|礼记]]》中的[[深衣|深衣]]即由苎麻所制。<ref>诗经曰“麻衣如雪”，麻衣即深衣.</ref>《乐府白紵歌》曰：“质如轻云色如银，制以为袍余作巾。”中唐时“沤纻为缊袍”、“春衫裁白纻，朝帽挂乌纱。”乃是一种时尚，<ref>常建，《渔浦》</ref><ref>《酬乐天得稹所寄纻丝布白轻庸制成衣服以诗报之》</ref>以至于唐人到了“裁衫催白纻”的痴狂程度。<ref>刘复,唐,《春游曲》</ref>。白纻細布染色后被称为纻丝，多用于制造官袍。<ref>。《明史》“其制，盘领右衽袍，用纻丝或纱罗绢”。</ref>

苎麻根可作為澱粉釀酒原料，更可以提鍊出凝酸銨。其葉可供食用、飲料，可從葉分離出氯原酸，加熱後可產生咖啡酸及奎寧酸。

中国为苎麻主要出产国。四川省达州市大竹县被称作中国苎麻之乡。苎麻量产为中国第一。优质苎麻纺织原料种植基地分布在湖南、江西、四川。<ref>[http://ctc.webtex.cn/info/2013-7-10@566577.htm 麻原料]</ref>蜀布为上古四川特产的苎麻布，蜀推行到巴，由巴推行到湘、糠、皖、浙。<ref>《本草纲目注》“苎麻旧不著所出州土，今闽、蜀、江、浙多有之。”</ref>到东汉初，江浙的苎麻布才开始行销到中原，称为越布。

苎麻布也用于制做漆器。在中国古代漆器中，其胎(即质)大多为木胎或夹苎胎(即麻布胎,古今多以夏布为之。亦称做脱胎) 。夹苎胎就是采用苎麻纺成的织物为主要原料之一制成的一种漆器胎体。木胎、竹胎都比较厚重，只有夹苎胎既轻巧又结实。夹苎胎又称脱胎,方法是在一个器形模具上一层层地缠绕麻布或丝绸,缠一层涂一层漆,层层粘合,干后去掉模子。古代叫"脱空"或者"脱沙"。夹苎胎漆器以精美的妆奁用器和小件酒器为多。

目前苎麻纺织手工技艺快濒临灭绝，现已列为[[中国国家级非物质文化遗产|中国国家级非物质文化遗产]]。苎麻也是日本的古传纺织作物。<ref>《三國志·魏書·倭人》：“婦人披髮屈紒,作衣如單被,穿其中央,貫頭衣之.種禾稻,紵麻,蠶桑,緝績,出細紵,縑綿。”</ref>[[琉球王国|琉球王国]]宫古岛的供品[[宫古上布|宫古上布]]、八重山上布以及[[新潟县|新潟县]][[越後上布|越後上布]]用苎麻纺织，<ref>[http://www.mineya.com/home/jofu/j_about.html 八重山上布]</ref>又称紺上布。1921年，宫古上布获得平和世界博覧会金奖。1978年成为[[日本文化财产|日本文化财产]]。<ref>[http://www.miyakojima.cc/page2.htm 宫古上布]</ref>越後上布中的[[小千谷縮|小千谷縮]]也就是[[川端康成|川端康成]]在《[[雪国|雪国]]》里，提到的一种在银装素裹的雪天里纺织的苎麻绉纱——白縮，专门用于制作夏季[[和服|和服]]礼服。<ref>[http://www.hn.xinhuanet.com/life/2010-08/24/content_20701072.htm 苎麻和兰草] {{webarchive|url=https://web.archive.org/web/20150924031243/http://www.hn.xinhuanet.com/life/2010-08/24/content_20701072.htm |date=2015-09-24 }}</ref>在韩国，忠清南道舒川郡韩山面生产的苎麻叫做韩山紵（한산저），一向享有盛名。韩山苎麻编织也是[[韩国无形文化遗产|韩国无形文化遗产]]。每年举办韩山苎麻文化节。
