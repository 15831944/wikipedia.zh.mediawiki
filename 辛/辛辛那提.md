'''辛辛那堤'''（{{lang-en|'''Cincinnati'''}}，{{IPAc-en|ˌ|s|ɪ|n|s|ᵻ|ˈ|n|æ|t|i}}）辛辛那提市是美国[[俄亥俄州|俄亥俄州]][[漢密爾頓縣_(俄亥俄州)|漢密爾頓縣]]的县府。辛辛那提建立于1788年，位于俄亥俄州与[[肯塔基州|肯塔基州]]边界的[[俄亥俄河|俄亥俄河]]北岸，靠近[[印第安纳州|印第安纳州]]。在2010年的人口普查中，该市的城市居民人口数为296,954人，是俄亥俄州第三大的城市。据[[2010年美国人口普查|2010年美国人口普查]]，辛辛那提城市圈人口数已达2,172,191人，是全美排名第28的都会区，也是俄亥俄州最大的都会区。

19世纪初期，辛辛那提是中部地区第一个能与沿海大城市在面积和财力上一较高下的新兴城市。作为美国首个内陆大都市，辛辛那提被看作是第一个纯粹的美国城。早期的辛辛那提市没有多少欧洲移民居住，也没有像其他东部城市一样受到欧洲移民太多的影响。然而，19世纪末期，随着交通方式由蒸汽船运输向铁路运输转变，辛辛那提市的发展速度急剧放缓，该市的人口数量和州内影响力也被另一个内陆城市-[[芝加哥|芝加哥]]超越。

辛辛那提有两支职业球队，MLB[[辛辛那提红人|辛辛那提红人]]和NFL[[辛辛那提孟加拉虎|辛辛那提孟加拉虎]]（過去還有NBA[[辛辛那提皇家|辛辛那提皇家]]，但今日已搬遷至[[萨克拉门托|萨克拉门托]]，並更名為國王）。除此之外，辛辛那提还是重大的网球巡回赛“辛辛那提大师赛”的举办地。该市还举办了其他许多大型赛事及活动，如“飞猪马拉松大赛”、“俄亥俄山谷音乐节”、“感恩节赛跑”等等。辛辛那提大学的前身是始建于1819年的俄亥俄医学院。

辛辛那提因拥有种类繁多的古建筑而闻名。[[越莱茵河区|越莱茵河区]]位于市中心以北，此处有全美国数量最多的[[意大利|意大利]]式建筑，可以与[[纽约|纽约]]、[[维也纳|维也纳]]和[[慕尼黑|慕尼黑]]的历史街区在面积和布局上相媲美。19世纪末期，辛辛那提因为其宏伟壮观的建筑群被看作是美国的[[巴黎|巴黎]]，这些著名古建筑包括音乐厅、辛辛那提酒店，希利托百货等等。越莱茵河区主要于1850年至1900年建成，多年来一直是德国移民的生活中心，是被美国《国家史迹名录》收录的最大历史胜地之一。

== 历史 ==
[[File:Cincinnati_I.jpg|thumb]]

1788年辛辛那堤建城。一开始叫Losantiville。

1790年改名为辛辛那堤。这个名字是对[[乔治·华盛顿|乔治·华盛顿]]的一个尊称，它来自[[古罗马|古罗马]]时期的一位将军[[路西斯·奎蒂斯·辛辛納圖斯|路西斯·奎蒂斯·辛辛納圖斯]]，这位将军在拯救罗马后退位。今天在俄亥俄，尤其在辛辛那堤依然有许多[[美國革命|美國革命]]时期的战士的后代，这些战士当时在这里分得土地而留下。

1802年辛辛那堤还被称为是一个[[村|村]]，1819年被提升为一个[[城市|城市]]。1811年俄亥俄河上开始使用[[蒸汽机船|蒸汽机船]]，加上从迈阿密和[[伊利運河|伊利運河]]的完工使得城市居民数于1850年达到11.5万。1835年左右辛辛那堤获得了“猪都”之称，因为当时它是美国的猪贩中心，市内街上有许多猪群。[[亨利·华兹华斯·朗法罗|亨利·华兹华斯·朗法罗]]称辛辛那堤为“西方皇后”，它成为[[地下铁道|地下铁道]]的重要终点站，这条地下铁道帮助许多奴隶从南方逃出。虽然辛辛那堤并没有山，但它还是有“七山城”的称呼，这可能是与[[罗马市|罗马市]]和罗马时代的辛辛那堤的应称。

与[[匹兹堡|匹兹堡]]和[[纳什维尔|纳什维尔]]一样它是一个美国向西扩展时期的边界城市。而作为一个内陆港口和运河城市它可以与[[路易斯维尔_(肯塔基州)|路易斯维尔]]、[[圣路易斯_(密苏里州)|圣路易斯]]和[[新奥尔良|新奥尔良]]相比。作为一个移民和工业城市它可以与[[布鲁克林区|布鲁克林区]]、[[费城|费城]]、[[克里夫兰|克里夫兰]]、[[芝加哥|芝加哥]]和[[底特律|底特律]]相比。

由于它有滨河景致以及许多公园，许多评论者突出了它的美丽。[[温斯顿·邱吉尔|温斯顿·邱吉尔]]称辛辛那堤为美国内陆最美的城市。当地的肥皂公司宝洁公司所赞助的众多[[肥皂剧|肥皂剧]]中的一部就是使用辛辛那堤的市容拍的。

从许多角度上来看辛辛那堤是一个前锋城市。1850年这里建立了美国第一座犹太医院。同年这里出产了第一张贺卡。1853年这里建立了美国第一个县立消防队。1867年成立的辛辛那提红人于1869年成为世界上第一个职业棒球队。1880年辛辛那堤是世界上第一个建造和拥有一条大铁路的城市，这条铁路连接美国东西。1902年市内建造了世界上第一座加强水泥摩天大楼。[[美国男童子军|美国男童子军]]是1905年从辛辛那堤起家的。由于市内有许多德国移民，在禁酒期以前辛辛那堤是美国啤酒工业的先驱。到1939年为止辛辛那堤的电台是美国第一家以50万瓦功率发射的电台。1954年这里建立了第一家公共电视台。

1979年12月3日在辛辛那堤美国最大的摇滚乐音乐会灾难，11人死亡，数百人受伤。

== 政治 ==
近年来辛辛那堤与其它中西部城市一样主要选[[民主党_(美国)|民主党]]，但是其都市地区以及汉密尔顿县主要选[[共和党_(美国)|共和党]]。

市政府由一个九人议会组成，其成员是市民选举产生的。1924年以前市政府是通过选区选举出来的，但当时这个机构受共和党的腐败，导致共和党从1880年代到1920年代几乎独霸市政府。1923年开始的改革运动在1924年终于通过全民公投的方式取消了选区制度。从1924年到1957年市政府通过[[比例代表制|比例代表制]]产生，从1957年开始得票最多的九位候选人入选。市长由议会选举产生。从1987年开始得票最多的选手自动成为市长。从1999年开始市长单独选举。

== 经济 ==
辛辛那提是许多重要的大型公司的总部所在地，如[[宝洁公司|宝洁公司]]、克罗格公司、美国联合百货公司（[[梅西百货公司|梅西百货公司]]和布鲁明黛百货公司的母公司）、Benziger兄弟出版社、美国金融集团、康威吉斯公司，欧慕尼凯医药科技咨询公司、[[五三银行|五三银行]]、西南金融集团、斯克里普斯报业集团、兴康软件公司、辛辛那提贝尔公司以及[[花王株式会社|花王株式会社]]美国分部等等。辛辛那提还是许多中小型公司的所在地，它们也是辛辛那提经济中不可或缺的一部分。

=== 其它 ===
[[美国内战|美国内战]]爆发前辛辛那堤是[[地下鐵路_(秘密結社)|地下铁道]]的一个重要港口。它正好位于肯塔基与反奴隶制度的州的边界上，因此许多人通过辛辛那堤逃离奴隶制。

20世紀初，為了解決辛辛那提的[[交通擁塞|交通擁塞]]問題，開始興建[[辛辛那提地铁|辛辛那提地铁]]，但之後由於[[第一次世界大戰|第一次世界大戰]]和[[第二次世界大戰|第二次世界大戰]]等因素，該計畫時而進展、時而停頓，至戰後，工程完全停擺，未完成的[[地下車站|地下車站]]及[[隧道|隧道]]廢棄至今日。

[[马丁·路德·金|马丁·路德·金]]被枪杀后辛辛那堤爆发动乱，动乱中两人死亡。

2001年辛辛那堤再次爆发种族动乱。动乱的起因是有人控告辛辛那堤市政府，因为证明在所有被警察枪杀的人中年轻黑人的比例特别高。就在开庭审判前一个19岁的无武装的黑人在被警察追逐过程中被枪杀，导致动乱。

== 地理和气候 ==
=== 地理 ===
[[File:Cincinnati-skyline-from-kentucky-shore-night.jpg|thumb]]

辛辛那堤的地理位置是北纬39°8'10"，西经84°30'11"。

据[[美国人口调查局|美国人口调查局]]数据其总面积为206.1平方公里，其中陆地面积201.9平方公里，水面面积4.1平方公里，水面占所有面积的2.01%。

辛辛那堤都市地区的总人口为2,130,151，是美国第27大的都市地区，除辛辛那堤市外它还包括俄亥俄的五个县、肯塔基的七个县和印第安纳的三个县。

=== 市区 ===
辛辛那提主城区包括俄亥俄州南部和肯塔基州北部。根据美国人口调查局的数据，辛辛那提市总面积为79.6平方英里（210km²），其中78.0平方英里（200 km²）为陆地，1.6平方英里（4.1 km²）为水域。城市地形复杂，建立在大片山丘、断崖之上。此外，在布鲁格拉斯区，还有大片的低矮山脊坐落在俄亥俄河两岸。辛辛那提的地理位置处于美国中西部，上南方北边缘，三分之二的美国人开车一天以内就能到达。

2003年的人口普查结果是辛辛那堤居民数为317,361人，分166,012户，共72,566个家庭，人口密度为1,640.5/km²。其中52.97%是白人，42.92%是黑人，0.21%是美国原著民，1.55%是亚裔，0.04%是太平洋原著民，0.63%是其他人，1.68%是混血儿。1.28%是西班牙或拉丁美洲后裔。

25.1%的户中有18岁以下的儿童，26.6%是[[结婚|结婚]]同居的家庭，18.6%是只有妇女，没有丈夫的，51.0%不是家庭。42.8%的户是单身汉，11.1%有65岁以上的人居住。平均每户有2.15人，每个家庭有3.02人。

24.5%的人在18岁以下，12.9%的人在18岁到24岁之间，31.6%的人在25到44岁之间，18.7%的人在45到64岁之间，12.3%的人老于65岁。平均年龄为32岁，性别比为女：男100：89.4，18岁以上的人的性别比为100：85.6（女：男）。

2003年辛辛那堤是美国最穷的城市之一。每户的平均年收入为29,493美元，每个家庭的平均年收入为37,543美元。男子平均年收入为33,063美元，女子为26,946，平均每人年收入为19,962美元。21.9%的人和18.2%的家庭生活在[[贫困线标准|贫困线标准]]以下。32.0%18岁以下和14.8%65岁以上的人生活在贫困线标准以下。

== 教育 ==
[[File:050819_036_Univ_Cincy.jpg|thumb]]的McMicken Hall]]
辛辛那提是[[辛辛那提大学|辛辛那提大学]]和[[澤維爾大學|澤維爾大學]]的所在地。辛辛那提大学（简称UC），是全美以工程、音乐、建筑、经典考古学和心理学为主要领域的重要研究生研究机构之一。辛辛那提大学的医学研究中心和音乐学院都享有极高赞誉，辛辛那提大学音乐学院的杰出校友有凯瑟琳·芭特尔，艾尔·希尔特和费斯·普林斯。泽维亚大学是一所教会大学，曾是辛辛那提教区俄亥俄雅典娜神学院的附属学校。

大辛辛那提地区还有[[迈阿密大学|迈阿密大学]]（最初的公立常春藤盟校之一）和[[北肯塔基州大学|北肯塔基州大学]]（NKU）。北肯塔基州大学位于肯塔基高地，距离市区东南偏南8英里（13km）。辐射州际公路系统丹尼尔卡特胡须大桥与471号州际公路将北肯塔基州大学与辛辛那提市中心相连，方便辛辛那提市的市民前往这所最新的肯塔基州大学。[[安东内利学院|安东内利学院]]是一所职业技术学院，主校区设在辛辛那提市，另在俄亥俄州、密西西比州设有分校。[[辛辛那提州立学院|辛辛那提州立学院]]是一所社区大学，下设中西部厨艺学校，该校是全美最优秀的厨艺学校之一。

许多来自大辛辛那提区的高中生还会前往[[俄亥俄州立大学|俄亥俄州立大学]]求学，该大学是俄亥俄州西北领地的第一所大学。
