'''瑪麗·辛·楊'''（{{lang-en|'''Mary Sean Young'''}}，{{bd|1959年|11月20日||}}）是一名美國女演員。她最知名的演出是在電影《[[银翼杀手|-{zh-hk:2020;zh-tw:銀翼殺手;zh-cn:银翼杀手;}-]]》（1982年）、《[[星際奇兵_(1984年電影)|星際奇兵]]》（1984年）、《{{link-en|諜海軍魂|No Way Out (1987 film)}}》（1987年）、《[[華爾街_(1987年電影)|華爾街]]》（1987年）、《{{link-en|親親表妹|Cousins (1989 film)}}》（1989年）和《[[神探飛機頭|神探飛機頭]]》（1994年）。

== 早年 ==
她出生在[[肯塔基州|肯塔基州]]的[[路易維爾_(肯塔基州)|路易維爾]]，是電視製片人和記者Donald Young, Jr.與編劇、公關專員和記者Lee Guthrie（本名Mary Lee Kane）的女兒。<ref>{{Cite web |url = http://www.filmreference.com/film/27/Sean-Young.html |title = Sean Young Biography (1959-) |publisher = Filmreference.com |language = en}}</ref><ref>{{Cite web |url = http://variety.com/1995/scene/people-news/donald-young-99128286/ |title = Donald Young |date = 1995-03-05 |publisher = Variety |language = en}}</ref>楊畢業於[[俄亥俄州|俄亥俄州]][[克利夫蘭高地|克利夫蘭高地]]的{{link-en|克利夫蘭高地高中|Cleveland Heights High School}}，接著是[[密芝根州|密芝根州]]{{link-en|英特洛肯 (密芝根州)|Interlochen, Michigan|英特洛肯}}的{{link-en|英特洛肯藝術中心|Interlochen Center for the Arts#Interlochen Arts Academy|英特洛肯藝術學院}}。她還上過紐約市的{{link-en|美國芭蕾學校|School of American Ballet}}。在成為女演員之前，楊曾擔任[[模特兒|模特兒]]和[[舞蹈|舞蹈演員]]。

== 電影生涯 ==
=== 1980年代 ===
[[File:Sean_Young_on_the_red_carpet_at_the_60th_Annual_Academy_Awards.jpg|alt=]]]]
楊於1980年的電影《''{{link-en|Jane Austen in Manhattan|Jane Austen in Manhattan}}''》中開始了她的電影生涯，隨後在1981年電影《''{{link-en|Stripes (film)|Stripes (film)|Stripes}}''》中扮演角色。然後，她在經典[[科幻|科幻]]電影《[[银翼杀手|-{zh-hk:2020;zh-tw:銀翼殺手;zh-cn:银翼杀手;}-]]》（1982年）中扮演女主角Rachael，與[[夏里遜·福|夏里遜·福]]共同演出。1986年的電視劇《''{{link-en|American Playhouse|American Playhouse}}''》中，楊在根據[[法蘭西斯·史考特·費茲傑羅|法蘭西斯·史考特·費茲傑羅]]的故事《''Myra Meets His Family''》改編的集數「Under the Biltmore Clock」中與{{link-en|林尼·馮·多倫|Lenny Von Dohlen}}主演。第二年，她在電影《[[華爾街_(1987年電影)|華爾街]]》中飾演[[米高·德格拉斯|米高·德格拉斯]]的角色{{link-en|Gordon Gekko|Gordon Gekko}}的妻子小角色。她的角色原本有很大戲份，但因為與[[奧利華·史東|奧利華·史東]]的衝突而被明顯降低了。<ref name=Karen>{{Cite web |url = http://ew.com/article/2008/01/29/sean-young-ew-profile/ |title = Sean Young: The EW profile |author = Karen Valby |date = 2008-01-29 |publisher = EW.com |language = en}}</ref>

她其中最著名的角色之一就是在1987年的《{{link-en|諜海軍魂|No Way Out (1987 film)}}》中飾演一名無情的[[華盛頓哥倫比亞特區|華盛頓]]政客（[[真·赫曼|真·赫曼]] 飾演）的情人。其他著名的演出包括《[[星際奇兵_(1984年電影)|星際奇兵]]》、《''{{link-en|Fatal Instinct|Fatal Instinct}}''》和《''{{link-en|Baby: Secret of the Lost Legend|Baby: Secret of the Lost Legend}}''》。

1988年，楊與[[占士·活士|占士·活士]]在《''{{link-en|The Boost|The Boost}}''》中演出。活士後來控告她騷擾他和他當時的未婚妻，聲稱除了其他破壞行為外，楊在他家門階留下一個毀容的洋娃娃。楊否認這些指控，並聲稱活士是惡意提起訴訟。楊說：「這是一件迷戀被拒絕的事情，就是這樣....所以控告我！而他[活士]告我了。」 這件訴訟於1989年在法庭外[[庭外和解|和解]]。<ref name=Karen />楊獲得了227,000美元，用於支付她的法律費用。<ref>{{Cite web |url = https://www.theguardian.com/film/2015/mar/26/blade-runner-sean-young-interview |title = Blade Runner's Sean Young: ‘If I were a man I'd have been treated better’ |author = Danny Leigh |date = 2015-03-26 |publisher = The Guardian |language = en}}</ref>

楊在[[添·布頓|添·布頓]]的1989年電影《[[蝙蝠俠_(1989年電影)|蝙蝠俠]]》中飾演[[維琪·維爾|維琪·維爾]]，但在排練期間，她墮馬後折斷了手臂，而她被[[金·碧辛嘉|金·碧辛嘉]]取代了。在續集《[[蝙蝠俠再戰風雲|蝙蝠俠再戰風雲]]》中，為了贏得[[貓女|貓女]]的角色（最初給了[[安納·貝寧|安納·貝寧]]，但在貝寧懷孕後最終給了[[米雪·菲花|米雪·菲花]]），楊製作了一件自製的貓女服裝，並在電影製作期間試圖接觸布頓和演員[[米高·基頓|米高·基頓]]。<ref name=Karen />她甚至裝扮貓女上了《{{link-en|瓊·瑞佛斯秀|The Joan Rivers Show}}》，繼續爭取角色並向添·布頓懇求。

=== 1990年代 ===
楊曾被揀選在1990年電影《[[至尊神探|至尊神探]]》中扮演Tess Trueheart。不過，她被解僱並由[[格倫妮·海德利|格倫妮·海德利]]飾演該角色。楊後來聲稱她被解僱是因為她拒絕[[華倫·比提|華倫·比提]]的求愛而受懲罰，比提否認這一說法<ref name=Karen />。1991年，她因在《[[死前之吻_(1991年電影)|死前之吻]]》的角色而得到金草莓獎[[金草莓獎最差女主角|最差女主角]]和最差配角女演員。她在1994年的喜劇電影《[[神探飛機頭|神探飛機頭]]》中飾演配角。她也在1995年喜劇電影《''{{link-en|Dr. Jekyll and Ms. Hyde|Dr. Jekyll and Ms. Hyde}}''》中飾演Helen Hyde。1990年代的大部分期間，她住在[[亞利桑那州|亞利桑那州]]的[[塞多納_(亞利桑那州)|塞多納]]，而且她的職業生涯冷卻下來<ref name=Karen />。1997年，楊在1997年電子遊戲《{{link-en|2020 (1997年遊戲)|Blade Runner (1997 video game)|-{zh-hk:2020;zh-tw:銀翼殺手;zh-cn:银翼杀手;}-}}》中重新飾演Rachael的角色。她的臉被三維掃描和重現，這是在遊戲中第一次使用這種技術。

=== 2000年代 ===
自2001年，楊在各種獨立電影中演出，並在電視上露面，其中包括《''{{link-en|Poor White Trash|Poor White Trash}}''》、《''{{link-en|Mockingbird Don't Sing|Mockingbird Don't Sing}}''》、《''{{link-en|Sugar & Spice|Sugar & Spice}}''》、《[[高校風雲_(電視劇)|高校風雲]]》和《''{{link-en|Reno 911!|Reno 911!}}''》。2005年，她在[[俄羅斯|俄羅斯]]花了四個月時間拍攝迷你劇《''{{link-en|Yesenin (TV series)|Yesenin (TV series)|Yesenin}}''》，其中她飾演舞者[[伊莎朵拉·鄧肯|伊莎朵拉·鄧肯]]。她也在2007年的《[[籃球兄弟|籃球兄弟]]》中飾演一個小角色。

2007年末，楊拍完兩部電影《''The Man Who Came Back''》和《''Haunted Echoes''》。

2008年，楊在電視節目《''{{link-en|Gone Country (TV series)|Gone Country (TV series)|Gone Country 2}}''》參加比賽，其中包括在[[田納西州|田納西州]]{{link-en|巴黎 (田納西州)|Paris, Tennessee|巴黎}}的亨利縣露天遊樂場（Henry County Fairground）舉辦的一場名人{{link-en|撞車大賽|Demolition derby}}的比賽。楊繼續贏得名人大賽的預賽，然後繼續與21名職業撞車大賽車手競爭。楊獲得了第四名。

=== 2010年代 ===
楊在2010年6月的《''{{link-en|The Young and the Restless|The Young and the Restless}}''》中飾演加拿大酒吧女侍Meggie McClain，與好朋友{{link-en|艾瑞克·布瑞登|Eric Braeden}}一同演出。<ref>{{Cite web |url = http://www.theyoungandtherestless.com/specials/interviews_detail_5464.html |title = Sean Young - Interviews |publisher = Wayback Machine |language = en |deadurl = yes |archiveurl = https://web.archive.org/web/20120401091511/http://www.theyoungandtherestless.com/specials/interviews_detail_5464.html |archivedate = 2012-04-01 }}</ref><ref>{{Cite web |url = http://michaelfairmansoaps.com/news/sean-young-on-her-upcoming-stint-as-sultry-maggie-on-yr/2010/05/11/ |title = Sean Young on her upcoming stint as “Sultry” Maggie on Y&R! |date = 2010-05-11 |publisher = Michaelfairmansoaps.com |language = en}}</ref>她在7月14日回歸該電視劇，並一直持續到2011年2月。<ref>{{Cite web |url = http://www.tvguide.com/news/sean-young-restless-1020162/ |title = Sean Young Returning for More Young and the Restless |author = Kate Stanhope |date = 2010-06-02 |publisher = TVGuide.com |language = en}}</ref>

2010年，她參加了ABC節目《''{{link-en|Skating with the Stars|Skating with the Stars}}''》第一季，擔任與專業溜冰選手{{link-en|丹尼斯·佩圖霍夫|Denis Petukhov}}一起溜冰的名人參賽者<ref>{{Cite web |url = http://www.thefutoncritic.com/news/2010/11/11/pro-skater-pairings-host-and-color-commentator-for-abcs-skating-with-the-stars-announced-11114/20101111abc03 |title = Pro Skater Pairings, Host and Color Commentator for ABC's "Skating With the Stars" Announced |date = 2010-11-11 |publisher = TheFutonCritic.com |language = en}}</ref>，但她是第一個被淘汰的名人。

2011年10月，楊在《[[大衛牙擦騷|大衛牙擦騷]]》上現身<ref>{{Cite web |url = http://gawker.com/5846755/sean-young-begs-for-work-in-bizarre-late-show-appearance |title = Sean Young Begs for Work in Bizarre Late Show Appearance |publisher = Gawker.com |language = en |deadurl = yes |archiveurl = https://web.archive.org/web/20111005232312/http://gawker.com/5846755/sean-young-begs-for-work-in-bizarre-late-show-appearance |archivedate = 2011-10-05 }}</ref>。在採訪中，她描述了她在撫養兩個兒子後如何尋找電影工作，並製作了一段短片促進她找工作，在《大衛牙擦騷》上播放。後來她被揀選在一部關於[[尼古拉·特斯拉|尼古拉·特斯拉]]的電影演出，預計於2013年發行，名為《''Fragments From Olympus: The Vision of Nikola Tesla''》（截至2017年9月尚未上映）。

2013年10月，楊在《''{{link-en|Star Trek: Renegades|Star Trek: Renegades}}''》中飾演Dr. Lucien，這是為新《[[星空奇遇記|星空奇遇記]]》系列的首集而做的一個{{link-en|粉絲電影|Fan film|粉絲項目}}（2015年8月播放），其中有幾位前《星空奇遇記》的演員演出，包括{{link-en|提姆·羅斯 (1956年出生)|Tim Russ|提姆·羅斯}}（由他執導首集），還有{{link-en|沃爾特·凱尼格|Walter Koenig}}、[[王以瞻|王以瞻]]和{{link-en|J. G. Hertzler|J. G. Hertzler}}。

2013年6月，楊在{{link-en|紐約溜冰劇場|Ice Theatre of New York}}舉辦的一場慈善溜冰活動中表演，參加名人溜冰比賽，對手是YouTube名人{{link-en|米高·巴克利|Michael Buckley (Internet celebrity)}}和美國奧運擊劍銀牌得主{{link-en|提姆·馬荷斯|Tim Morehouse}}。

在2017年，她為《[[銀翼殺手2049|銀翼殺手2049]]》再次回歸1982年Rachael的角色，飾演原版（使用第一部電影中的鏡頭）和新型[[克隆|複製]]版角色。這是通過使用CGI面部去老化來與她在1982年的演出一致，還有用了另一名女演員作為替身來完成的。辛·楊在新電影中也有被掛名。<ref>{{Cite web |url = https://www.gamespot.com/articles/how-blade-runner-2049-resurrected-that-character-f/1100-6453912/ |title = How Blade Runner 2049 Resurrected That Character From The Original |author = Michael Rougeau |date = 2017-10-09 |publisher = GameSpot |language = en}}</ref>

== 個人生活 ==
1990年，楊嫁給了Robert Lujan，與他生下兩個兒子。這對夫婦於2002年離婚，並於2011年再婚。

2008年1月，楊因為酗酒問題而要進行康復。後來於2011年逗留在康復診所，在真人騷《{{link-en|名人復健室|Celebrity Rehab with Dr. Drew}}》上有敘述。

=== 被捕 ===
2012年2月26日，在[[第84屆奧斯卡金像獎|第84屆奧斯卡金像獎]]的後派對（after party）期間，楊被捕。據稱，她因為沒持有入場券而被要求離開。接著發生了爭吵，結果楊摑了保安人員的耳光。楊說她沒有喝醉，而且表現良好，並說保安人員是挑釁者。<ref>{{Cite web |url = https://www.hollywoodreporter.com/news/sean-young-arrest-oscars-295669 |title = Sean Young Responds to Post-Oscars Arrest: 'I Was Sober' |author = Erin Carlson |date = 2012-02-28 |publisher = The Hollywood Reporter |language = en}}</ref>

2018年8月，楊和一名男伴被監視錄像發現她們盜竊一間[[皇后區|皇后區]]的商店。被盜的物品總計約12,000美元。楊被要求問話。<ref>{{Cite web |url = https://www.usatoday.com/story/life/people/2018/08/10/sean-young-blade-runner-actress-alienist-burglary/955370002/ |title = Sean Young, 'Blade Runner' actress, wanted for questioning in $12,000 burglary |date = 2018-08-10 |publisher = USA Today |language = en}}</ref>
