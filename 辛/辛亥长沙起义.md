'''辛亥长沙起义''' 1911年10月22日在[[长沙|长沙]]爆发的[[武昌起义|武昌起义]]后第一次响应性起义。焦达峰、陈作新领导。<ref name="zhanged2">张创新. [2005] (2005). 中国政治制度史. Edition 2. 清华大学出版社 publishing. ISBN 7-302-10146-9, ISBN 978-7-302-10146-8. pg 377.</ref>

== 背景 ==
1904年，[[黄兴|黄兴]]等人在长沙建立[[华兴会|华兴会]]，策划武装起义，1905年8月，[[同盟会|同盟会]]成立后，[[禹之谟|禹之谟]]、[[陈家鼎|陈家鼎]]等人在黄兴指示下在长沙成立同盟会湖南分会，1906年[[刘道一|刘道一]]等人领导[[萍浏醴起义|萍浏醴起义]]，1911年4月[[广州|广州]][[黄花岗起义|黄花岗起义]]失败后，湘籍党人[[谭人凤|谭人凤]]、[[宋教仁|宋教仁]]等将革命重心放在[[长江|长江]]流域，成立同盟会中部总会，制定[[湖南|湖南]]、[[湖北|湖北]]互相响应的方针。

10月13日武昌起义的消息传到湖南，同盟会湖南分会举行会议，决定10月22日在长沙响应，以[[焦达峰|焦达峰]]、[[陈作新|陈作新]]为首的同盟会战时统筹部，负责起义。清朝湖南当局在19日开始加强防御，民间传言政府将对新军营房实行轰击。焦达峰等决定23日再起义。

10月21日晨，湖南[[巡抚|巡抚]][[余诚格|余诚格]]获得起义情报，命于次日将长沙新军全部调往[[株洲|株洲]]，全城搜捕革命党，焦达峰、陈作新因此决定提前起义。
== 过程 ==
10月22日清晨，新军士兵到协操坪集合，将四十九标二营军装库打开，取出枪械弹药。每人发白布臂章一块、子弹10发，此时[[安定超|安定超]]号召革命，宣读焦达峰、陈作新关于起义的命令。然后，放信号枪3响，新军哗变。

[[彭友胜|彭友胜]]率北路军攻北门，守城[[巡防营|巡防营]]士兵未抵抗，开城门投降，新军占领荷花池军装局。安定超率东路军进入东城，也未遇抵抗。两军会师，中午时分从东、西辕门攻击抚台衙门。焦达峰、陈作新等也到巡抚衙门外，炮兵营李金山在又一村抓获防营统领黄忠浩，带至小吴门城楼斩首。起义军冲入抚署，湖南巡抚余诚格命人高悬“大汉”白旗投降；本人则逃往上海。清朝官员逃离。

湖南成立焦达峰、陈作新为正、副[[都督|都督]]的“中华民国军政府湖南都督府”，立宪派成立了参议院，举[[谭延闿|谭延闿]]为院长；都督府设立民政、军政两部，分由谭延闿、[[黄鸾鸣|黄鸾鸣]]为部长，传檄全省，号召各地革命，各道、府、州、县闻风响应，到11月5日，湖南除[[常德|常德]]以西地区外，全部光复。军政府镇压残余清军，并派湘军援鄂。
== 事后 ==
10月31日新军第25混成协第50协第2营管带梅馨从邵阳赶到长沙，杀害焦达峰、陈作新。随后梅馨迎立谭延闿任湖南都督。1912年[[中华民国临时大总统|中华民国临时大总统]][[孙中山|孙中山]]在[[南京|南京]]追授焦达峰[[大将|大将]]军衔。
