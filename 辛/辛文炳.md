'''辛文炳'''{{BD|1912年|2月1日|1999年|3月20日|catIdx=Xin辛}}，生於[[日治台灣|日治台灣]][[台南州|台南州]]台南市（今[[台南市|台南市]]）人。[[中華民國|中華民國]][[政治人物|政治人物]]，曾任台南市議員、台南市議會議長，1960年至1964年出任[[台南市長|台南市長]]。卸任後曾任[[立法委員|立法委員]]，並參與創辦[[南台工專|南台工專]]。

== 早年生平 ==
辛文炳於[[明治|明治]]四十五年（1912年）2月出生於舊府城東安坊馬公廟街。{{r|府|page=55}}[[大正|大正]]十四年（1919年），辛文炳入台南市第三公學校(今進學國小)，大正十四年（1925年）畢業後考入[[台南州立第二中學校|台南州立第二中學校]] (今國立臺南一中)，[[昭和|昭和]]八年（1933年）入[[日本|日本]][[明治大學|明治大學]]法學部，昭和十一年（1936年）畢業。

昭和十七年（1942年）「興南乘合自動車株式會社」成立後，辛文炳出任常任取締役（社長辛西淮）。{{r|府|page=92}}

戰後出任興南公共汽車股份有限公司（戰前為興南自動車會社）副董事長。{{r|府|page=105}}

== 政途 ==
辛文炳於民國三十九年（1950年）參選[[台南市議會|台南市議會]]第一屆市議員選舉並當選，1950年當選副議長，隨後被要求加入[[中國國民黨|中國國民黨]]。{{r|府|page=241}}1954年5月14日原議長[[楊請|楊請]]因當選市長請辭議長，辛文炳在5月17日補選中當選議長，此後連任第二、三、四屆議長。1958年發生台南市議會向市政府提出不信任案，被視為辛文炳與時任市長[[葉廷珪_(台南)|葉廷珪]]不和的開端。

== 臺南市長任內 ==
1960年辛文炳出馬競選[[1960年中華民國縣市長選舉#臺南市|第四屆臺南市長選舉]]，以64,143票擊敗[[葉廷珪_(台南)|葉廷珪]]當選臺南市市長。然而此次選舉發生作票爭議，亦引起《[[自由中國|自由中國]]》雜誌發表社論要求選舉無效。<ref>〈違法舞弊的臺南市長選舉應宣告無效〉，《自由中國》社論（二），《自由中國》第二十二卷第十期，頁5-6。1960年5月16日。</ref>檢舉事件先後開了五次庭前後半年訴訟，最後於10月駁回原告。{{r|府|page=147}}

辛文炳任內完成整修中山公園（今[[臺南公園|臺南公園]]）、興建國宅、永樂市場及拓寬[[民權路|民權路]]等政績，並整建[[延平郡王祠|延平郡王祠]]。然而任內辛文炳也曾產生對官僚系統無法適應，在就職兩年後即曾因受不了國民黨的官場作風而一度北上[[中興新村|中興新村]]打算向省主席[[周至柔|周至柔]]請辭，最後在省府秘書長親自接見勸退打消辭意。{{r|府|page=151}}1964年辛文炳任滿，在面對周遭勸進聲下仍然放棄追求連任。

== 卸任後 ==
卸任臺南市長後，辛文炳出任省政府顧問。之後擔任[[南台工專|南台工專]]校長，1972年代表[[中國國民黨|中國國民黨]]在[[台灣省|台灣省]]第四選區（雲嘉南南）參選為第一屆[[立法委員|立法委員]][[1972年中華民國立法委員選舉|增額選舉]]當選，並且連任。

1996年臺灣舉行[[1996年中華民國總統選舉|第一屆民選總統選舉]]，辛文炳因認同[[李登輝|李登輝]]「台灣利益優先」論點，接任李登輝陣營臺南市競選總部主任委員。之後受總統李登輝敦聘為總統府國策顧問。<ref>[http://secr.stust.edu.tw/tc/node/2ndch 第二任董事長辛文炳]，南台科技大學網站</ref>

辛文炳服膺於[[臺灣地位未定論|臺灣地位未定論]]，認為國民黨統治台灣屬於「外來政權的非法統治」，早在1964年赴美國訪問時，即在接受[[波多黎各|波多黎各]]廣播訪談時坦承國民黨是外來政權。{{r|府|page=246}}晚年辛文炳熱心於推動台灣加入[[聯合國|聯合國]]。{{r|府|page=249}}1998年12月，辛文炳表示不能認同台灣屬於中華民國的看法，認為臺灣前途應由臺灣人民共同決定，遂主動辭去[[總統府國策顧問|總統府國策顧問]]。{{r|府|page=247-248}}七個月後，李登輝總統發表兩岸是「特殊國與國關係」。

1999年3月，辛文炳因心臟病發病逝。[[中華民國總統府|總統府]]明令褒揚令。<ref>[http://www.president.gov.tw/Default.aspx?tabid=131&itemid=6092&rmid=514 褒揚令－辛文炳]，中華民國總統府網站</ref>

== 家族 ==
*其父為臺南著名士紳[[辛西淮|辛西淮]]，曾擔任西港庄庄長，獲頒紳章。戰後一度遭國民黨政府拘禁五個月。{{r|府|page=111}}辛文炳為家中長男。[[興南客運|興南客運]]為其家族事業。而辛文炳長女辛美惠後來嫁給前國民黨主席[[吳伯雄|吳伯雄]]大伯[[吳鴻森|吳鴻森]]的四子吳運東。
*其妻翁梅，為台南士紳[[翁木|翁木]]的長女。
*辛文炳之弟辛文恭曾一度以無黨籍身分登記參選[[1968年中華民國縣市長選舉|第六屆臺南市長選舉]]，然而在登記後辛家先是有「某治安機關」調查人員以有人檢舉臺南貨運公司逃漏稅為由，搜索查扣帳冊。並在永樂市場官司案中，市政府以120萬元為擔保，聲請裁定對辛文炳財產執行130萬假扣押，這一切被認為是國民黨當局要迫使「辛派」退出選舉的手段。{{r|府|page=189}}最後辛文恭撤銷登記。<ref>〈台南辛文恭讓賢 林錫山穩操左券〉。《自立晚報》五版，1968年4月3日</ref>在[[林錫山_(台南)|林錫山]]當選後，國民黨市黨部即對永華市場案透過黨政協調，使林錫山撤回上訴案辦理補正手續。{{r|府|page=189}}
*辛文炳一姐辛永秋，嫁給[[吳新榮|吳新榮]]妻弟毛昭葵。有一同父異母妹[[辛永清|辛永清]]，為中華料理師<ref name="林欣誼">{{Cite news|url=https://tw.news.yahoo.com/%E5%AE%B6%E9%84%89%E8%8F%9C%E6%8F%9A%E5%90%8D%E6%97%A5%E6%9C%AC-%E8%BE%9B%E6%B0%B8%E6%B8%85%E6%B4%BB%E5%87%BA%E8%87%AA%E6%88%91-213000454.html|title=家鄉菜揚名日本 辛永清活出自我|language=zh-tw|publisher=《中國時報》|author=林欣誼|date=2012-12-02|accessdate=2015-06-30}}</ref>。
