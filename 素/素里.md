'''素里市'''（{{lang-en|Surrey}}，又称'''苏里市'''）是一座位於[[加拿大|加拿大]][[卑詩省|卑詩省]][[大溫哥華地區|大溫哥華地區]]的城市。根据2016年人口普查，素里市的人口有517,887人<ref name=2016censusBCdivisions>{{cite web | url=http://www12.statcan.gc.ca/census-recensement/2016/dp-pd/hlt-fst/pd-pl/Table.cfm?Lang=Eng&T=701&SR=1&S=3&O=D&RPP=9999&PR=59 | title=加拿大各省和地区人口和住宅数量，2011-2016年人口普查 - 100％数据（卑诗省） | publisher=[[加拿大统计局|加拿大统计局]] | date=2017年2月8日 | accessdate=2017年2月19日}}</ref>，是[[卑詩省|卑詩省]]繼[[溫哥華|溫哥華]]市而後人口第二多的城市。

素里位於[[菲沙河|菲沙河]]南岸，西鄰[[三角洲_(卑詩省)|三角洲]]，北面隔河與[[新西敏市|二埠]]、[[高貴林|高貴林]]、[[高貴林港|高貴林港]]、[[匹特草原|匹特草原]]及[[:en:Barnston_Island_(British_Columbia)|班士敦島]]對望，東臨[[蘭里市|蘭里市]]和[[蘭里區|蘭里區]]，南接[[美加邊境|美加邊境]]，並從三方包圍[[白石_(卑詩省)|白石市]]。素里市内共有六個鎮中心，包括：[[:en:Fleetwood_Town_Centre|費列特活]]、[[:en:Whalley,_British_Columbia|惠利]]、[[:en:Guildford_Town_Centre|喬德福]]、[[:en:Newton_Town_Centre|紐頓]]、[[:en:Cloverdale,_British_Columbia|克樂佛代爾]]和[[:en:South_Surrey|南素里]]。<ref name="CityofSurrey"/>

== 歷史 ==
[[File:Holland_Park,_Surrey_BC.jpg|thumb]]
素里於1879年成立<ref name="CityofSurrey"/>，所佔的土地原是一群說[[哈爾魁梅林語|哈爾魁梅林語]]的原住民棲身之地。來自[[英格蘭|英格蘭]][[薩里郡|薩里郡]]（{{lang|en|Surrey}}）的移民布魯瓦（{{lang|en|H.J. Brewer}}）從[[新西敏市|二埠]]隔著[[菲沙河|菲沙河]]眺望這片土地時，認為與其家鄉極為貌似，於是將之命名為「{{lang|en|Surrey}}」。其他移民往後亦陸續定居於此。當時素里一帶主要是樹林，林中植物包括道格拉斯杉、杉樹、紅崖柏、毒堇、黑莓和小紅莓。現時惠利（{{lang|en|Whalley}}）所在之處的一部分當年是被昆特蘭原住民用作墓地。早期移民率先定居於克樂佛代爾和南素里，主要以務農、捕魚、捕獲牡蠣、或經營小商店為生。

連接素里和二埠的[[帕圖洛橋|帕圖洛橋]]於1937年開通，展開了素里發展史的新一頁。[[二戰|二戰]]後素里陸續發展成溫哥華和[[本那比|本那比]]的衛星城鎮，尤以北素里區為甚。隨著來自國内外其它地區的移民陸續湧入，素里於1980和1990年代也經歷了前所未見的增長，並於1993年正式升格為市。<ref name="CityofSurrey"/>

== 政治 ==
素里市議會是由8名市議員組成。該市現任市長是戴安·瓦特茲（[[:en:Dianne_Watts|Dianne Watts]]），對上一次市選於2011年11月舉行。

素里市於[[卑詩省議會|卑詩省議會]]中佔8席；[[卑詩新民主黨|卑詩新民主黨]]和[[卑詩自由黨|卑詩自由黨]]於2013年省選中分別贏得3席和5席。在聯邦政壇方面，素里市於[[加拿大國會|加拿大國會]][[加拿大國會下議院|下議院]]佔4席：[[加拿大保守黨|加拿大保守黨]]和[[加拿大新民主黨|加拿大新民主黨]]於[[2011年加拿大國會下議院選舉|2011年大選]]中各得2席。

== 教育 ==
[[File:Sfusurrey.jpg|thumb]]的素里分校位於素里中央城大樓]]

素里市的公共教育服務由36號校區提供。該校區在市内營運100間公立小學和21間公立中學，是全省規模最大的公立教育局。此外，市内亦有數間私立中小學。

專上教育方面，[[西門菲沙大學|西門菲沙大學]]、[[昆特侖理工大學|昆特侖理工大學]]和[[道格拉斯學院|道格拉斯學院]]等院校皆於素里市設有校園。

== 交通 ==
[[File:Expolinebridge.jpg|left]]
素里剛成立時的交通主要以馬車為主。隨著連接素里和[[新西敏市|二埠]]的鐵路橋於1904年開通，卑詩電力鐵路公司也於1910年增設了來往素里和二埠、溫哥華和菲沙河谷的市際電車服務，但該電車綫卻於1950年代初停駛。[[溫哥華架空列車|溫哥華架空列車]]系統於1990年沿[[天空之橋_(溫哥華)|天空之橋]]從二埠伸展至素里[[史葛路站|史葛路站]]，並於往後數年間在素里市内增設3個車站，標誌著鐵路服務重返素里。現時架空列車服務和素里市内的巴士綫皆由[[大溫哥華地區|大溫哥華地區]]的公共交通機構[[運輸聯線|運輸聯線]]營運。

素里市内的主要[[高速公路|高速公路]]包括卑詩省1號和[[卑詩省99號公路|99號省道]]：1號省道（[[橫加公路|橫加公路]]）以東西向橫跨素里市北部，並透過[[曼港橋|曼港橋]]跨越菲沙河通往高貴林；99號省道則跨越素里市南部，向南直達[[美加邊境|美加邊境]]，並於[[和平門|和平門]]口岸連接[[5號州際公路|5號州際公路]]。南北向的15號省道則連接1號省道以及另一個美加口岸。

== 姊妹及友好城市 ==
