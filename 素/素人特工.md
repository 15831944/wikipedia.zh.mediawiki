{{engname|《素人特工》|''The Rookies''}} 是一部由[[袁錦麟|袁錦麟]]執導的中國動作冒險喜劇片，主要演員有[[王大陸|王大陸]]、[[張榕容|張榕容]]、[[蜜拉·喬娃維琪|蜜拉·喬娃維琪]]、[[許魏洲|許魏洲]]和[[劉美彤|劉美彤]]等。電影將於2018年12月21日在中國大陸正式上映。

== 劇情 ==
[[跑酷|跑酷]]愛好者趙風，意外地捲入一場國際犯罪交易。後來，他遇到了刑警淼淼、發明家丁山和醫生LV，四個人組成菜鳥特工隊，在國際特工Bruce的帶領下，一起對抗犯罪組織，並展開一場華麗大冒險。

== 製作 ==
《素人特工》於2016年向廣電總局備案<ref >{{cite web|url=http://dy.chinasarft.gov.cn/shanty.deploy/blueprint.nsp?id=015a8287fa224128402881a659a4b9fa&templateId=012a2e051030004740284c812a2d62df | language= | title=《素人特工》電影拍攝製作備案公示表 |publisher=[[中華人民共和國國家新聞出版廣電總局|國家新聞出版廣電總局]] |date= |accessdate= 2018-04-24}}</ref>，於2017年2月上旬廣電總局劇本立項同意拍攝<ref>{{cite web |url=http://dy.chinasarft.gov.cn/shanty.deploy/blueprint.nsp?id=015a8282a1493fc8402881a659a4b9fa&templateId=0129f8148f650065402881cd29f7df33# |title=《素人特工》劇本立項同意拍攝 |date=2017-02-28 |publisher=[[中華人民共和國國家新聞出版廣電總局|國家新聞出版廣電總局]] }}</ref>，於2017年9月開拍，耗時四個多月時間拍攝，於2018年1月23日殺青<ref >{{cite web|url=http://news.mtime.com/2018/01/24/1577625.html#p6 | language= | title=《素人特工》殺青 |publisher=時光網 |date=2018-01-24 |accessdate= }}</ref>。導演[[袁錦麟|袁錦麟]]表示，他想把動作電影變得更年輕，更有趣，更有活力，更國際化，所以他打造了《素人特工》<ref >{{cite web|url=http://ent.cctv.com/2018/04/26/PHOAR8tFLUa9h5rdHyTiHpG6180426.shtml#zZmX5NDBOcj7eV180426 | language= | title=國院線國產影片推介會 |publisher=[[央視網|央視網]] |date=2018-04-26 |accessdate= }}</ref>。故事背景發生在[[匈牙利|匈牙利]]的首都[[布達佩斯|布達佩斯]]，拍攝的景點有匈牙利的[[匈牙利國家歌劇院|國家歌劇院]]，[[自由橋|自由橋]]，[[蓋勒特山|蓋勒特山]]，[[瓦茨街|瓦茨街]]和[[自由廣場_(布達佩斯)|自由廣場]]等<ref>{{cite web |url=http://epaper.yzwb.net/html_t/2018-08/09/content_485953.htm?div=-1 |title=《素人特工》拍攝的匈牙利景點 |author=孔小平 |date=2018-08-09 |publisher=[[揚子晚報|揚子晚報]] }}</ref>。
