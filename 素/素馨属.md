'''茉莉'''，为[[木樨科|木樨科]]'''素馨属'''又名'''茉莉屬'''、'''茉莉花屬'''（''Jasminum''）常绿灌木或藤本植物的统称。茉莉花色洁白，香气浓郁，是最常见的芳香性盆栽花木。最著名的一种是[[双瓣茉莉|双瓣茉莉]]（''Jasminum sambac''），也就是人们平常俗称的茉莉花。中国福建省的福州市是著名的茉莉花产地。

== 形态 ==
茉莉一般株高0.5－3m，其枝细长。单[[叶|叶]]对生，椭圆形或卵圆形，全缘，叶色翠绿，质薄有光泽。[[花|花]]多为白色，具有浓郁的清香，花期5－10月，7－8月开得最盛，有香气。

[[File:Jasminum_grandiflorum_HRM1.jpg|thumb]]是[[突尼西亚|突尼西亚]]的[[国花|国花]]]]

== 分布 ==
茉莉喜温暖湿润和阳光充足环境，原产于[[印度|印度]]、[[巴基斯坦|巴基斯坦]]，[[中国|中国]]早已引种，在廣泛地栽種
。

== 用途 ==
{{medical}}
茉莉的花、叶、根可以入药，亦可泡茶，著名的[[茉莉花茶|茉莉花茶]]主要原料就是其花瓣。茉莉花可治外感发热，胃气不和，腹胀腹泻。

做為精油的芳香療法可用於抗憂鬱、殺菌、鎮定劑。對焦慮、情緒不穩等問題有幫助。對皮膚及頭皮很好。

== 养护 ==
# 温度。茉莉花喜欢高温，通常在20℃左右开始萌芽，25℃以上孕蕾，28℃至33℃左右形成花蕾，其生长适温为22℃至35℃，温度过高，植株处于半休眠状态，开花少，开花不均匀，温度过低植株生长缓慢，且不耐低温，长时间在5℃以下，部分枝条就会冻死。
# 光照。茉莉花喜阳光充足的环境，光照充足，叶色浓绿，枝干粗壮，花蕾形成多，着色好，香气浓。否则，造成植株徒长，叶片淡绿变簿，枝干也纤细，节间较长而嫩，花蕾少，开花少，花朵小香气淡。
# 水分。茉莉花喜土壤湿润，可以每周一次，约250毫升。但怕盆内积水，若盆土长期处于潮湿状态，根系就会因盆土内缺氧而引起根系腐烂，叶片逐渐变黄而脱落，甚至整株死亡。
# 施肥。茉莉花喜肥，但施肥时要按照其生长规律进行，冬季和夏季超过35℃以上温度不施肥生长期应保持足够的合理的肥分，注意[[氮|氮]]、[[磷|磷]]、[[钾|钾]]肥比例均匀，孕蕾期以磷钾肥为主，以促使更多花芽形成，切忌施肥过浓或施用未腐熟的有机肥，以免造成根系烧伤，引起叶片变黄，整个生长期每隔1周至2周喷施一次沤制好的[[矾|矾]]肥水或[[硫酸亚铁|硫酸亚铁]]溶液。
# 病虫害防治。茉莉花易受[[红蜘蛛|红蜘蛛]]等虫害的危害，因此应加强通风，平时注意观察，一旦发现有虫体危害，及时用[[三氯杀螨蜗醇|三氯杀螨蜗醇]]进行防治。茉莉叶斑病为茉莉的一种常见病害，该病一般5月至6月开始发生，7月至8月为发病盛期。发病初期，染病叶片上产生淡褐色斑点，后向四周扩展，病部变薄呈褐色透明状，圆形或不规则形，病部变薄，半透明，褐色，病斑边缘明显。最后病斑上生黑色颗粒状物。应及时清除病叶，集中销毁。发病期，喷施65%[[代森锌|代森锌]]可湿性粉剂600倍至800倍液，或75%[[百菌清|百菌清]]可湿性粉剂800倍液等，连喷3次至4次，间隔10天至14天。
