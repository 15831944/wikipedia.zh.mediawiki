'''盘羊'''（学名''{{lang|la|Ovis ammon}}''），又叫'''大头羊'''、'''大角羊'''，是一种生活在[[中亚|中亚]]高原上的野生[[羊属|羊]]，也是体型最大的一种绵羊。根据分布地区的不同，可分为9个亚种。

== 分布 ==
盘羊生活在中亚的众多山脉中，其分布地区从[[阿尔泰山脉|阿尔泰山脉]]、南[[西伯利亚|西伯利亚]]、[[蒙古国|蒙古国]]、[[西藏|西藏]]、[[天山山脉|天山山脉]]一直到[[尼泊尔|尼泊尔]]和[[帕米爾高原|帕米爾高原]]，它们可以在7000米的高处生存。

由于狩猎以及家畜的竞争盘羊几乎到处都变少了，在一些地区甚至完全消失了。在中国东北、蒙古部分地区、南西伯利亚、[[哈萨克斯坦|哈萨克斯坦]]和[[乌兹别克斯坦|乌兹别克斯坦]]大部分地区盘羊已经绝种。在[[喜马拉雅山脉|喜马拉雅山脉]]、内蒙古、西藏大部分地区以及[[新疆|新疆]]它们今天已经很少了。在[[俄罗斯|俄罗斯]]的阿尔泰山脉仅有少数生存。

所有亚种的数量均减少，在整个[[亚洲|亚洲]]盘羊的总数估计少于八万头。目前它们最主要的分布地区在[[塔吉克斯坦|塔吉克斯坦]]、[[吉尔吉斯斯坦|吉尔吉斯斯坦]]和蒙古部分地区。

== 特征 ==
分布地区东北区域的雄兽肩高可达120厘米，体重可达200千克。西南区域的动物一般小些。

盘羊毛的颜色从淡棕色至白灰色，胸、腹部的颜色浅一些。大多数亚种的熊兽的脖子是白色的。它们没有类似[[赤羊|赤羊]]的鬃毛。

雄性的弯角粗大，长达1米以上，向下扭曲呈螺旋状，外侧有环棱；雌兽的角非常短，而且弯度不大。

== 习性 ==
盘羊是群生动物，其习性与其它野生绵羊一样。在发情期外公羊和母羊各自形成约五至十头羊组成的群。发情期在冬季，这样幼羊可以在春季出生。

盘羊以草和树叶为生。

盘羊的腿比较长，身材比较瘦，与其它野绵羊相比其爬山技巧比较差，因此在逃跑时一般避免逃向太陡峭的山坡。

== 亚种 ==
盘羊有以下亚种：
* 指名亚种（{{lang|la|Ovis ammon ammon}}）：南西伯利亚和西蒙古，阿尔泰山脉和[[薩彥嶺|薩彥嶺]]，体型大，角重，颈和体灰棕色，受威胁
* 天山亚种（{{lang|la|Ovis ammon karelini}}）：天山山脉，[[那灵河|那灵河]]以北
* 马可波罗亚种（{{lang|la|Ovis ammon polii}}）：帕米爾高原，体型中等，角极大而且形成一个完整的圆形，其尖端向外，受威胁
* 西藏亚种（{{lang|la|Ovis ammon hodgsoni}}）：[[西藏高原|西藏高原]]，喜马拉雅山脉，体型大，角不形成完整的圆形，受威胁
* {{lang|la|Ovis ammon collium}}：生活在哈萨克斯坦东北，其生活地区与其它亚种隔绝，受威胁
* {{lang|la|Ovis ammon nigrimonatna}}：哈萨克斯坦中部和南部，基本上已经灭绝
* 戈壁亚种（{{lang|la|Ovis ammon darwini}}）：[[戈壁沙漠|戈壁沙漠]]，蒙古和中国，受威胁
* 华北亚种（{{lang|la|Ovis ammon jubata}}）：[[内蒙古|内蒙古]]，受威胁严重
* 阿尔金亚种（{{lang|la|Ovis ammon dalailamae}}）：[[阿尔金山脉|阿尔金山脉]]，西藏北部，受威胁
* {{lang|la|Ovis ammon severtzovi}}：乌兹别克斯坦东北

{{lang|la|Ovis ammon severtzovi}}长期被看作是盘羊和赤羊之间的一个过渡种，但是最新的[[染色体|染色体]]分析证明它们的确属于盘羊。比如它们与所有盘羊亚种一样有56个染色体。

阿尔金亚种常被看作是西藏亚种的一个孤立群而被分入该亚种。
