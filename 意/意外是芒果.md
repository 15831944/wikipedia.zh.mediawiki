《'''意外的芒果'''》（{{jpn|j=意外にマンゴー|rm=Igai ni Mango}}）是日本[[日本偶像|偶像]]組合[[SKE48|SKE48]]的第21張[[單曲|單曲]]，2017年7月19日由[[avex_trax|avex trax]]發行。A面曲由小畑優奈担当[[AKB48組合構成#中心位置|中心成員]]。

== 概要 ==
本作為前作《金的愛，銀的愛》發行11個月後的2017年首張單曲。單曲分為初回限定盤Type A、B、C、D，通常盤Type A、B、C、D，以及劇場盤9種版本。

歌曲的選拔成員在2017年6月5日的研究生公演結束後發表。選拔成員相比前作減少了2名（16名），其中Team KII的北野瑠華和研究生矢作有紀奈都是初次成為選拔成員。這是小畑優奈初次成為選拔成員及初次擔任中心成員的單曲作品<ref>{{Cite news|url=http://realsound.jp/2017/06/post-81205.html|script-title=ja:SKE48、21thシングル『意外にマンゴー』発売　センターは初選抜の小畑優奈|publisher=Real Sound|date=2017-06-07||language=ja}}</ref>。

歌曲在2017年6月22日的「总选举感謝公演〜首先是感谢！其他的话之后再说〜」上正式發表<ref>{{Cite news|url=http://www.musicman-net.com/artist/68580.html|script-title=ja:SKE48、総選挙感謝公演で新曲「意外にマンゴー」初披露|publisher=Musicman-net|date=2017-06-23||language=ja}}</ref>。

Type D的B面曲收錄大矢真那的個人畢業曲「永遠的Legacy」<ref>{{Cite news|url=https://www.rbbtoday.com/article/2017/06/27/152680.html|script-title=ja:SKE48の大矢真那が卒業を発表|publisher=Real Sound|date=2017-06-27||language=ja}}</ref>。

== 封面及封底照片 ==
這次的單曲封面照片是在[[台灣|台灣]]被稱為水上樂園的旅遊勝地[[澎湖群岛|澎湖群岛]]拍攝<ref>{{Cite news|url=https://realsound.jp/2017/06/post-85051.html|script-title=ja:SKE48、シングル『意外にマンゴー』アートワーク公開　台湾リゾート地で水着ショット披露|publisher=Real Sound|date=2017-06-21||language=ja}}</ref>。拍攝地點事先由成員提案，經過開會決定，最終於澎湖縣馬公市中央老街、白沙鄉通樑保安宮、西嶼鄉二崁古厝、白沙鄉吉貝沙尾拍攝，拍攝團隊並將今年4月20日至6月22日舉辦的「2017澎湖國際海上花火節」之煙火場景如實收錄。
