'''意大利联合交战陆军'''（{{lang-it|Esercito Cobelligerante Italiano}}），或'''南方军'''（{{lang-it|Esercito del Sud}}），'''意大利的解放军'''（{{lang-it|Corpo Italiano di Liberazione}}）是意大利皇家陆军在第二次世界大战中1943年九月加入盟军时的名字。同时期加入盟军的 [[義大利皇家海軍|意大利皇家海军]]和意大利皇家空军则被称为意大利联合交战海军和意大利联合交战空军，而支持法西斯意大利的陆军部队加入意大利社会共和国，成为意大利社会共和国陆军。

意大利联合交战陆军因为1943年9月8日盟军与意大利停战而生。由于盟军进攻南意大利，意大利国王维托里奥·埃马努埃莱三世于1943年七月解除了墨索里尼的首相职务，国王指定负责和盟军谈判的意大利元帅[[佩特羅·巴多格裡奧|佩特羅·巴多格裡奧]]为首相。

意大利联合交战陆军有大约二十到三十万的部队在意大利活动，其中有两万名（后来增加到五万）战斗部队与约二十万的辅助支援部队。 大致上，意大利联合交战陆军组成了盟军第15军团八分之一的战斗部队。

== 组成 ==
1943年9月28日，意大利联合交战陆军队在莱切（Lecce）附近组建，有些士兵才刚刚逃过德国的追捕。遵照皇家陆军第70V命令，意大利第一摩托化战斗群组建完成。此战斗群包括2个前意大利皇家陆军的师：第18和第58步兵师。本战斗群有合计295名军官，5387名士兵。

第一摩托化战斗群的首次行动在卡西诺，米尼亞諾蒙泰倫戈(Cassino,Monte Lungo）。这次行动大幅降低了盟军对意大利士兵是否愿意在盟军作战的怀疑——尽管第一摩托化群承受了重大伤亡，其表现仍然优秀。在支援美军第五军后第一摩托化战斗群重组，转移到英军第8军最左翼的波兰第二兵团一边。

== 意大利的解放军 ==
1944年4月17日，这支22000人的军队更名为意大利的解放军（Corpo Italiano di Liberazione）。志愿者的加入使其有能力组建第二支师。意大利的解放军分为Nembo师和Utili师。Nembo是旧意大利陆军伞兵师的名字，Utili的名字来自于该部队的指挥官，Umberto Utili中将。该师人员主要是第一摩托化战斗群的旧部。1944年，5000名意大利军人在古斯塔夫防线表现良好，不过也使意大利人再一次受到重大损失 

== 1944年下半年至1945年的意大利联合交战陆军 ==
在1944年7月的菲洛特拉诺（Filottrano）的战斗后，意大利人被送往后方休整并训练。同时他们获得了标准的全新英国装备。

1945年开始意大利的解放军分成6个战斗群，Cremona、Legnano、Friuli、Mantova、 Piceno、Folgore。每个战斗群都相等于未满编的师，全部使用英国武器和军服。每一个战斗群有432名军官和8578名其他职位。每个战斗群有116门野战炮，170门迫击炮，502台轻机枪，1277辆载具。战斗群的名字全部来自旧意大利皇家陆军。战斗群跟随美军和英军在歌特防线作战。

参谋长为梅塞元帅，最高指挥官为保罗费莱蒂中将。

=== 战斗群 ===
每个步兵团有三个步兵营，一个迫击炮连（装备ML3英寸迫击炮）和一个反坦克连（装备6磅炮）。炮兵团有4个炮群（装备25磅炮），一个反坦克连（装备17磅炮）和一个防空团（装备英式博福斯40毫米高炮）。

== 意大利陆军 ==
1946年，意大利王国成为意大利共和国，意大利联合交战陆军改编为意大利陆军。
