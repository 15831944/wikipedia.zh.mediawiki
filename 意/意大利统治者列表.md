==东哥特王國國王==
476年，日耳曼人雇佣军长官[[奥多亚塞|奥多亚塞]]（Odoacer）废黜最后一个[[羅馬皇帝列表|西罗马帝国皇帝]][[罗慕路斯·奥古斯都|罗慕洛·奥古斯都]]，被东罗马皇帝[[拜占廷人芝诺|芝诺]]封为'''意大利的统治者'''（''dux Italiae'' ），不久自称'''意大利国王'''（''rex Italiae''），但他仍然是[[东罗马帝国|东罗马帝国]]在[[罗马|罗马]]的行政官。

483年，[[东哥特人|东哥特人]]领袖[[狄奧多里克大帝|狄奧多里克]]殺死統治者[[奥多亚塞|奥多亚塞]]，并在[[意大利|意大利]]创建了[[东哥特王国|东哥特王国]]。

== [[加洛林王朝|加洛林王朝]] ==
[[意大利国王|意大利国王]]是在[[加洛林王朝|加洛林王朝]]时期开始使用的一个头衔。后来拥有这个头衔的人只是名义上声称对整个意大利拥有统治权的人，因为在19世纪之前意大利从未统一过。在大多数情况下，意大利国王的称号由[[神圣罗马帝国皇帝|神圣罗马帝国皇帝]]兼领。后来[[德国|德国]]统治者承认了意大利的分裂，这一称号逐渐消失。意大利国王的王冠是著名的[[伦巴第铁王冠|伦巴第铁王冠]]，一般均由[[教皇|教皇]]在[[帕维亚|帕维亚]]为之加冕。

[[查理曼|查理曼]]的第三个儿子[[丕平_(意大利)|丕平]]，在781年由[[哈德里安一世|哈德里安一世]]教皇加冕为意大利国王。他统治在意大利部，意大利中央成为[[教宗国|教宗国]]。

* [[丕平_(意大利)|丕平]]：781年-810年
* [[伯纳德_(意大利)|伯纳德]]：810年-818年
* [[洛泰尔一世|洛泰尔一世]]：818年-839年
* [[路易二世_(法兰克)|路易二世]]：839年-875年
* [[查理_(秃头)|查理（秃头）]]：875年-877年
* [[卡洛曼_(巴伐利亚)|卡洛曼]]：877年-879年
* [[查理_(胖子)|查理（胖子）]]：879年-888年

== 神圣罗马帝国皇帝 ==
{{参见|神圣罗马帝国皇帝列表}}
* [[奥托一世_(神圣罗马帝国)|奥托一世]]：962年-973年
* [[奥杜因_(阿尔勒)|奥杜因]]：1002年-1014年

从962年到1806年，意大利王国被纳入[[神圣罗马帝国|神圣罗马帝国]]，皇帝并兼意大利国王。奥杜因期间，他派一个严肃的独立行政官在意大利。然而，在[[腓特烈一世_(神圣罗马帝国)|腓特烈·巴巴罗萨]]在位后（1152年-1190年）, 皇家当局在意大利各城邦的势力愈来愈小，皇帝要威势他的统治必须与这些城邦合作。

== 拿破仑统治 ==
在[[拿破仑战争|拿破仑战争]]时期，拿破仑在意大利北部建立了一个由法国控制的意大利王国。

* [[拿破仑一世|拿破仑一世]]：1805年 - 1814年

拿破仑的儿子[[拿破仑二世|拿破仑二世]]被授予“罗马人国王”的头衔，而实际上并不存在一个“罗马王国”。这个头衔是用于表明[[拿破仑一世|拿破仑一世]]想做[[神圣罗马帝国皇帝列表|神圣罗马帝国皇帝]]的继承人。
