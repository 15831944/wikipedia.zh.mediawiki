'''義大利殖民帝國'''（{{lang-it|Impero coloniale italiano}}）是由近代[[義大利王國_(1861年-1946年)|義大利]]所建立的[[殖民地|殖民地]][[帝國|帝國]]。而這個[[殖民帝國|殖民帝國]]在[[第二次世界大戰|第二次世界大戰]]中除了擁有意大利外，更支配著[[地中海|地中海]]和[[東非|東非]]的其他屬土諸如[[厄立特里亞|厄立特里亞]]、[[衣索匹亞|衣索匹亞]]等東非殖民地。帝國勢力在1940年達到頂峰，而后随着[[意大利投降|意大利投降]]以及[[第二次世界大战|第二次世界大战]]结束，意大利殖民帝国也逐渐成为历史。

就如[[德意志帝國|德意志帝國]]一樣，義大利比起其他歐洲強國較遲達成統一。所以，其殖民帝國也比歐洲列強的殖民地帝國來得要遲。亦因如此，它並不能和其他歐洲列強競逐[[亞洲|亞]][[美洲|美]]兩洲的勢力（[[中國|中國]]的[[天津租界|天津租界]]例外），儘管在[[新大陸|新大陸]]有很多意大利裔移民。

==[[義大利王國|義大利王國]]和[[法西斯主義|法西斯主義]]==
1929年[[經濟大衰退|經濟大衰退]]之後，[[帝國主義|帝國主義]]成為義大利[[法西斯主義|法西斯主義]]者[[墨索里尼|墨索里尼]]演講的主題。他認為殖民地的建立對義大利人口和經濟的振興有很大的作用。而帝國主義也是墨索里尼為了讓義大利取代[[英國|英]][[法國|法]]兩國，成為[[地中海|地中海]]主要勢力不可或缺的一部份。

==義屬東非==
意属东非（{{lang|it|Africa Orientale Italiana}}）是意大利王国在东非的一个殖民地。于1936年到1941年期间存在。由阿比西尼亚、意属索马里兰和厄立特里亚三个区域组成。1936年，总理墨索里尼下令侵略阿比西尼亚。意属东非宣告建立。下设6个州。1940年8月，意大利又侵占了英属索马里兰。但进入1941年，意大利战局趋于被动。1941年，阿比西尼亚恢复独立，意属东非解体。

===義屬索馬里蘭（今[[索馬里|索馬里]]）（1889～1960年）===
在1889至1890年間，[[義大利|義大利]]在[[東非|東非]]的[[索馬里蘭|索馬里蘭]]建立了殖民地。1925年，因[[肯亞|肯亞]]得到[[朱巴蘭|朱巴蘭]]，其殖民地得以擴張。

===義屬厄利垂亞（1889～1941年）===
[[File:ST-Massowa.jpg|thumb]]]]

1889年，義大利在[[厄利垂亞|厄利垂亞]]建立殖民地。這個殖民地被法西斯義大利視作為一個驕傲。而今時今日在厄利垂亞的首都[[阿斯馬拉|阿斯馬拉]]，許多被保留下來的[[古蹟|古蹟]]和[[紀念碑|紀念碑]]都可追溯到百多年前由義大利所統治的時期。由於為數眾多的建築遺產，阿斯馬拉也許能成為一個[[世界遺產|世界遺產]]。

===[[阿比西尼亞|阿比西尼亞]]（今[[衣索匹亞|衣索匹亞]]）（1935～1941年）===
[[File:Soldatietiopia.jpg|thumb]]中士兵撤离。]]

儘管義大利在1895至1896年間的[[第一次義大利－衣索匹亞戰爭|第一次義大利－衣索匹亞戰爭]]中被[[阿比西尼亞|阿比西尼亞]]打敗，但當墨索里尼執政後，義大利便在1935至1936年間發動[[第二次義大利－衣索匹亞戰爭|第二次義大利－衣索匹亞戰爭]]，成功擊敗阿比西尼亞。1936年5月9日，義大利國王[[維托里奧·埃馬努埃萊三世|維克托·伊曼紐爾三世]]宣佈成為[[衣索匹亞|衣索匹亞]]皇帝。

而墨索里尼欲把數以百萬計的移民者送到東非的殖民地中落地生根，以取得在東非的豐厚經濟利潤。但是，義大利入侵衣索匹亞一事已經受到國際間的敵視。因此，義大利對這個新殖民地的統治權大約只有五年。

===利比亞（1912～1943年）===
[[File:Visita_del_RE_a_Bengasi.JPG|thumb]]

義大利在1911年與[[奧斯曼土耳其帝國|奧斯曼土耳其帝國]]發生戰爭，其後土耳其戰敗，雙方簽訂了《[[洛桑條約|洛桑條約]]》。條約使得義大利得到了[[北非|北非]]的[[的黎波里|的黎波里]]及[[昔蘭尼加|昔蘭尼加]]的領土，並與[[法國|法國]]協議，共同分割北非的海岸。義大利對此舉感到十分驕傲，並稱之為「義大利第四個海岸」。事實上，義大利在20年代中花了整整近十年的時間來「平定」這個殖民地。

1934年，的黎波里及昔蘭尼加殖民地合併為「[[利比亞|利比亞]]」殖民地。墨索里尼採用了這個一千八百多年前[[羅馬帝國|羅馬帝國]]君主[[戴克里先|戴克里先]]沿用的名字，就是欲揚威海外，建立非洲帝國，以圖重振羅馬帝國時代的雄風。

1943年，[[纳粹德國|纳粹德國]][[非洲軍團|非洲軍團]]及義大利部隊被[[英國|英]][[美國|美]]盟軍趕出北非，義大利對利比亞的統治權也隨之而逝。

== 亚洲 ==
===中國[[天津意租界|天津意租界]]（1902～1943年）===
[[File:马可波罗广场近景.jpg|thumb]][[马可波罗广场|马可波罗广场]]]]
{{main|天津意租界}}

1900年，[[清朝|清朝]]爆發[[八國聯軍之役|八國聯軍之役]]，義大利也參與其中。事後，中國與列強求和，義大利因此分得中國的[[天津港|天津港]]，以作為一個貿易集散地。而這個只得46公頃的城市可算是歐洲殖民者中得到的最小一個殖民地。但天津意租界却是意大利在境外的唯一一处租界，也是意大利在亚洲的唯一一块领地.

1943年，英美盟軍擊敗義大利，墨索里尼遭國王軟禁，墨索里尼便臨時建立起[[意大利社会共和国|意大利社会共和国]]，並與中國[[汪精衛政權|汪精衛政權]]成立的政權達成協議，取消義大利在天津的統治權。

== 歐洲 ==
===阿爾巴尼亞===
在[[第一次世界大戰|第一次世界大戰]]結束後一段時間，義大利佔領了[[阿爾巴尼亞|阿爾巴尼亞]]。其後，兩國在1920年9月2日簽署協定，義大利部隊可以撤出阿爾巴尼亞，條件就是該國把[[薩贊島|薩贊島]]割讓給義大利。

1939年，阿爾巴尼亞完全成為了義大利殖民帝國的勢力範圍。意大利獨裁者對此感到莫大成功，並將此成功與德國吞併[[德奧合併|奧地利]]及[[捷克斯洛伐克|捷克斯洛伐克]]相提並論。1939年4月7日，義大利正式入侵阿爾巴尼亞，五天後阿軍完全喪失抵抗能力，阿爾巴尼亞國王[[索格一世_(阿尔巴尼亚)|索格一世]]逃亡至倫敦。而義大利國王[[維托里奧·埃馬努埃萊三世|維托里奧·埃馬努埃萊三世]]則被加冕為阿爾巴尼亞國王，墨索里尼更架設一個阿爾巴尼亞的法西斯政府，由阿爾巴尼亞前首相擔任政府首腦。此後，阿爾巴尼亞軍隊直屬於義大利麾下，直至墨索里尼下台。

1941年，德義兩國發動[[巴爾幹戰役|巴爾幹戰役]]，[[南斯拉夫|南斯拉夫]]陷落，境內的[[科索沃|科索沃]]及[[馬其頓|馬其頓]]一部份土地與阿爾巴尼亞合併，成為義大利的領地。而在1942至1943年間，阿爾巴尼亞也不斷發生一些抵抗義大利統治的運動。1943年夏，反抗勢力已經完全控制阿爾巴尼亞。1943年9月，隨著墨索里尼政權的倒台，義大利完全地退出了阿爾巴尼亞。

===希臘===
[[多德卡尼斯群島|多德卡尼斯群島]]同是義大利在1911～12年間的[[義土戰爭|義土戰爭]]中，打敗衰弱的奧斯曼土耳其帝國而奪回來的土地。而當時的義大利正欲奪得[[罗得岛|罗得岛]]及其週邊的島嶼，以挑戰英國在[[地中海|地中海]]的統治地位。

而作為第一次世界大戰戰勝國的義大利，有權可以獲得戰敗國土耳其的割地。在1919年簽訂的《[[色佛尔条约|色佛尔条约]]》中，羅得島及多德卡尼斯群島劃入義大利。1923年的洛桑條約更確立了國際間的承認。

1943年9月，義大利把多德卡尼斯群島的控制權交予德國。

===蒙特內哥羅===
在1941年，[[蒙特內哥羅|蒙特內哥羅]]重建起[[君主立憲制|君主立憲制]]，隨後更宣稱成為義大利的保護國。1943年9月，蒙特內哥羅被德军控制。

== 新羅馬帝國的構思 ==
[[File:Meharisti.jpg|thumb]]

'''[[新羅馬帝國|新羅馬帝國]]'''（[[義大利語|義大利語]]：'''{{lang|it|Nuovo Impero Romano}}'''；[[拉丁語|拉丁語]]：'''{{lang|la|Novum Imperium Romanum}}'''）是墨索里尼創造出來，以形容意大利殖民地帝國，特別是指義大利佔領下的衣索匹亞。這個構想建立於[[第二次世界大戰|第二次世界大戰]]前義大利[[民族主義|民族主義]]的高峰時期，並與羅馬帝國有很大關係：
* [[亞得里亞海|亞得里亞海]]改稱為「'''{{lang|it|Mare Nostro}}'''」（義大利語，意為「我們的海洋」）。在義大利入侵阿爾巴尼亞後，亞得里亞海落入了義大利手上。而這個名字在羅馬帝國時代是指整個地中海，因為羅馬帝國的勢力曾經遍佈整個地中海，墨索里尼打算重新建立羅馬帝國的輝煌時代。
* 義大利的政權名曰「[[法西斯|法西斯]]」，取材自羅馬帝國時代執政官手握的束棒（{{lang|it|Fasces}}），代表至高無上的權力。
* 首都為[[羅馬|羅馬]]，與羅馬帝國時代一樣。
* [[伊曼紐爾三世|伊曼紐爾三世]]加冕為皇帝。

== 帝國的野心 ==
義大利軍隊曾經短暫地佔領[[英屬索馬利蘭|英屬索馬利蘭]]。1941年，義大利佔領了今[[克罗地亚|克罗地亚]]境內的[[達爾馬提亞|達爾馬提亞]]及[[科托爾灣|科托爾灣]]和[[斯洛文尼亚|斯洛文尼亚]]西南部地区。墨索里尼更欲進一步佔領[[馬耳他|馬耳他]]、[[突尼西亞|突尼西亞]]、[[法屬索馬里蘭|法屬索馬里蘭]]及[[科西嘉|科西嘉]]等地。法國陷落後，墨索里尼更希望當戰勝英國後，北非軍隊可以一舉佔領[[阿爾及利亞|阿爾及利亞]]、[[埃及|埃及]]和[[苏丹共和国|蘇丹]]等地，但遭德國的冷淡回應。

== 帝國的终结 ==
當英美聯軍從東、北非進攻義大利時，義大利殖民帝國終於迎向完結。1943年12月，墨索里尼的政府被推翻，義軍撤出了阿爾巴尼亞及多德卡尼斯群島。其後在1947年簽訂的《[[義大利和平條約|義大利和平條約]]》中，義大利正式宣告喪失所有殖民地，儘管在1949年時，[[聯合國|聯合國]]決定把[[義屬索馬利蘭|義屬索馬利蘭]]繼續置於義大利托管之下。直至1960年7月1日，[[英屬索馬利蘭|英屬索馬利蘭]]與[[義屬索馬利蘭|義屬索馬利蘭]]合併，並形成今天的獨立國家——[[索馬里|索馬里]]。
