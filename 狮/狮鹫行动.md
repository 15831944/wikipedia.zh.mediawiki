'''狮鹫行动'''（{{lang-de|Unternehmen Greif}}，{{IPA-de|ɡʀaɪ̯f|pron}}，意同[[狮鹫|格里芬]]）是[[第二次世界大战|第二次世界大战]][[突出部之役|突出部战役]]期间由[[阿道夫·希特勒|阿道夫·希特勒]]亲自提出、[[納粹德國|納粹德國]][[武装党卫军|武装党卫军]]中校[[奥托·斯科尔兹尼|奥托·斯科尔兹尼]]策划实施的一次渗透行动，也被称为“麒麟作战”。在行动中，会说[[英语|英语]]的德军士兵身着[[盟军|盟军]]制服，使用缴获的盟军车辆装备混入盟军内部以刺探情报并扰乱秩序{{Sfn|Pallud|1987|p=3}}，配合装甲旅发起奇袭。行动的最终目的是夺取至少一座横跨[[默兹河|默兹河]]的桥梁。由于缺乏足够的制服、装备、车辆和人员，这一目标未能实现{{Sfn|Zabecki|2015|p=496}}。但无处不在的流言曾一度使盟军上下陷入混乱和恐慌。当奇袭失败后，斯科尔兹尼率其装甲旅进行了无果的强攻。在强攻之前，他们换回了德军的制服，以普通的[[战斗员|战斗员]]身份参战。这也使他们在战后免于[[战争法|战争法]]的制裁。

== 背景 ==
[[File:Bundesarchiv_Bild_101I-567-1503C-15,_Gran_Sasso,_Mussolini_vor_Hotel.jpg|缩略图]]，1943年9月12日]]
{{Main|奥托·斯科尔兹尼}}
[[武装党卫军|武装党卫军]]中校[[奥托·斯科尔兹尼|奥托·斯科尔兹尼]]至1944年已成功策划实施了一系列大胆的[[特种行动|特种行动]]。尤其在[[橡树行动|橡树行动]]中，斯科尔兹尼带领空降[[突击队|突击队]]突袭[[大薩索山|大薩索山]]，救出了监禁中的意大利[[Duce|法西斯领袖]][[贝尼托·墨索里尼|贝尼托·墨索里尼]]而名噪一时，被称为“欧洲最危险的男人”{{Sfn|Zabecki|2015|p=496}}。1944年10月15日，斯科尔兹尼又带人[[綁票|綁架]]了[[匈牙利王国_(1920–1946)|匈牙利]]独裁者[[霍爾蒂·米克洛什|霍爾蒂·米克洛什]]的儿子，阻止后者向[[苏联|苏联]]媾和{{Sfn|O'Reilly|Dugard|p=38|2014}}。完成这项任务后，他回到德国，并受到希特勒的接见。10月22日，在[[東普魯士|東普魯士]]的[[狼穴|狼穴]]，希特勒告知斯科尔兹尼，他计划对深入西欧的[[盟军|盟军]]发动反击，重新夺回[[安特卫普|安特卫普]]，并交给斯科尔兹尼一项重要任务{{Sfn|O'Reilly|Dugard|p=39|2014}}{{Sfn|Clearwater|1999|p=60}}。

按照[[阿道夫·希特勒|希特勒]]的安排，斯科尔兹尼将组建一支[[特種部隊|特種部隊]]。主要任务是趁盟军有所防备之前，夺取[[默兹河|默兹河]]位于[[列日|列日]]至[[那慕尔|那慕尔]]间的桥梁，接应前来的德军装甲师。鉴于之前美军突击队曾成功运用缴获的坦克渗透德军防线，希特勒认为，斯科尔兹尼的部队也应换上盟军制服行动，不仅有利于顺利完成任务，还能通过下达错误命令、干扰通讯联络以及误导部队等手段扰乱盟军军心{{Sfn|Delaforce|2006|p=56}}。在会见的最后，希特勒对这位他最倚重的突击队指挥官寄予厚望：“我知道给你的时间非常仓促，但我相信你一定能克服万难完成任务......不过，这次我不允许你亲临前线，你可千万不能被俘！”{{Sfn|Skorzeny|2016|p=153}}

[[奥托·斯科尔兹尼|斯科尔兹尼]]起初对这一任务相当惊讶，但他深知[[元首_(納粹德國)|元首]]的命令不可违抗，最终还是同意了这一方案，并立刻着手准备。考虑到行动内容有违[[海牙公约|海牙公约]]，如果有士兵穿着盟军制服被俘，很可能会被当成[[间谍|间谍]]就地处决。为此，他和总参谋长[[海因茨·古德里安|海因茨·古德里安]]上将以及[[阿爾弗雷德·約德爾|阿爾弗雷德·約德爾]]大将等将领进行了多轮商讨，并确定具体的目标{{Sfn|Mitcham|2006|p=30}}。行动代号定为“[[狮鹫|狮鹫]]”，一种[[希腊神话|希腊神话]]中的神兽{{Sfn|O'Reilly|Dugard|p=38|2014}}。这并不是第一个以“狮鹫”为名的军事行动。早在1944年8月，[[納粹德國陸軍|德軍]]在[[苏联|苏联]][[奥尔沙|奥尔沙]]和[[维捷布斯克|维捷布斯克]]境内进行的大规模反游击扫荡就曾用过这个代号。

== 准备 ==

=== 人员 ===
从[[阿道夫·希特勒|希特勒]]委派任务，到[[突出部战役|守望莱茵行动]]发动，留给[[奥托·斯科尔兹尼|斯科尔兹尼]]的时间只有五周，他必须争分夺秒征集到足够的人手和装备。仅费时四天，斯科尔兹尼就向[[國防軍最高統帥部_(德國)|國防軍最高統帥部]]作战部长[[阿爾弗雷德·約德爾|約德爾]]大将提出了组建一支由3,300人组成的特种装甲旅——第150装甲旅的构想，立刻得到后者全力支持，准许他便宜行事{{Sfn|Pallud|1987|p=3}}。10月25日，德国最高统帅部发布命令，从[[德国国防军|全军]]征集熟悉英语的士兵向斯科尔兹尼报道（能说[[美国英语|美国口音]]者更佳）。斯科尔兹尼本希望能在11月10日前募集到足够的人手，但结果令他大失所望：来自各兵种的2,500人中，真正能纯熟运用英语并对[[美国文化|美国文化]]了若指掌的竟只有10人；有30－40人英语流利但对[[俚语|俚语]]不甚了了；100－150人的英语水平勉强达标；200人在学校接受过英语教育；其余的“基本只会说'Yes'”{{Sfn|Pallud|1987|p=6}}。这些人被安排在位于[[巴伐利亚|巴伐利亚]][[格拉芬韦赫尔|格拉芬韦赫尔的]]训练营中，为行动做准备。

[[奥托·斯科尔兹尼|斯科尔兹尼]]从训练营中挑选出150名英语最佳的士兵，组成名为“施蒂劳部队（{{Lang-de|Einheit Stielau}}）”的突击队{{Sfn|Pallud|1987|p=10}}。他们在训练营中接受了美国文化突击培训：看美国电影、吃美国口粮、抽美国香烟、学美国敬礼等。有些人甚至到[[奥得河畔科斯琴|科斯琴]]和[[林堡|林堡]]的[[戰俘營|戰俘營]]中，跟美军战俘聊天以練習英语。除此之外，他们还进行了必要的敌后作战训练：包括使用敌方武器、识别敌方所属、安置炸弹，以及面对盘问时如何应对{{Sfn|Beevor|2015|p=93}}。这批突击队员按其任务分为三种{{Sfn|Pallud|1987|p=10}}：
* 由5~6人组成的爆破小组负责炸毁敌方[[桥梁|桥梁]]、[[弹药库|弹药库]]和油料库。
* 由3~4人组成的侦察小组负责侦察[[默兹河|默兹河]]两岸盟军动向，沿途颠倒路标、混淆真假[[地雷陣|雷区]]、并向遇到的美军部队传递虚假命令。
* 先头小组将和德军进攻部队密切配合，剪断[[電話線|電話線]]并摧毁[[无线电|无线电基站]]以扰乱盟军通信，同时发布假命令。

=== 装备 ===
[[File:PantherM10.jpg|缩略图]]的[[黑豹坦克|黑豹坦克]]]]
为了达成任务，[[奥托·斯科尔兹尼|斯科尔兹尼]]还要求准备一系列[[盟军|盟军]]的武器装备，包括2,400套美军制服（其中将军制服10套、军官制服70套）、15辆[[谢尔曼坦克|谢尔曼坦克]]、20辆[[自行火炮|自行火炮]]、30辆装甲侦察车、40辆摩托车、100辆[[吉普|吉普]]、120辆卡车，以及足够的枪支弹药等{{Sfn|Pallud|1987|p=4}}。由于种种原因，没有一项满足要求。从各地的[[戰俘營|戰俘營]]搜刮而来[[盟军|盟军]]军服中只有少数是美军战斗时穿的[[夾克|夾克]]，其余多为毫无用处的军大衣{{Sfn|Infield|1981|p=80}}。送到斯科尔兹尼手里的车辆只有两辆[[谢尔曼坦克|谢尔曼坦克]]（其中一辆报废）、四辆装甲侦察车、30辆吉普和15辆卡车{{Sfn|Pallud|1987|p=8}}。为弥补数量上的不足，只能用德军装备涂上美军涂装后充数：装甲兵总监为第150装甲旅提供了12辆[[黑豹坦克|黑豹坦克]]，拆除车长指挥塔并在车体和炮塔围上薄钢板，伪装成[[M10坦克歼击车|M10坦克歼击车]]的样子{{Sfn|Perrett|1981|p=39}}；[[三号突击炮|三号突击炮]]也经过类似改造，使其轮廓类似[[M7牧師式自走炮|M7牧師式自走炮]]{{Sfn|Pallud|1987|p=8}}；[[82式水桶車|82式水桶車]]则被用来冒充美军[[威利吉普車|威利吉普車]]。斯科尔兹尼本人对这种临时拼凑不抱多大希望：“（这种伪装）充其量只能蒙住最年轻的美军士兵，而且还得是在夜间从非常远的距离观察。”{{Sfn|Skorzeny|2016|p=157}}为防止{{Internal link helper/en|友军误击|Friendly fire}}，斯科尔兹尼的手下不配[[鋼盔|鋼盔]]，戴蓝色或粉色[[围巾|围巾]]；坦克则将炮口指向九点钟方向，所有车辆后部都涂有一个黄色三角形标志{{Sfn|Pallud|1987|p=9}}。

虽然[[阿道夫·希特勒|希特勒]]要求包括狮鹫行动在内所有[[突出部之役|守望莱茵行动]]的内容必须严格保密，但大规模征集会说英语的士兵以及盟军装备还是在德军内部产生了诸多流言和猜测。[[奥托·斯科尔兹尼|斯科尔兹尼]]没有出面阻止传播，而是予以默许甚至鼓励。他知道盟军情报部门必定有所行动，流言越多、越能使他们疲于奔命，对实现真正的作战目标越有利{{Sfn|Pallud|1987|p=9}}。就连斯科尔兹尼的手下也被蒙在鼓里——受流言影响，很多士兵直到行动前一周都还以为他们的目的是潜入[[法国|法国]]以解{{Internal link helper/en|敦刻尔克之围|Siege of Dunkirk (1944–45)}}，甚至是前往法国首都[[巴黎|巴黎]]绑架[[德怀特·艾森豪威尔|艾森豪威尔]]{{Sfn|Beevor|2015|p=93}}{{Sfn|Pallud|1987|p=9}}。

== 行动 ==
1944年12月14日夜，第150装甲旅于德国小城[[巴特明斯特艾费尔|明斯特艾费尔]]附近集合。他们分为X、Y和Z三个{{Internal link helper/en|战斗群|Kampfgruppe}}，跟随[[親衛隊第1師|武装党卫军第1装甲師]]、[[親衛隊第12師|第12装甲師]]以及{{Internal link helper/en|第12国民掷弹兵师|12th Infantry Division (Wehrmacht)}}先锋，于12月16日下午开始向目标{{Internal link helper/en|费恩斯高地|High Fens}}挺进{{Sfn|Pallud|1987|p=10}}。和大部分参战德军一样，第150装甲旅也因[[滨湖洛斯海姆|洛斯海姆]]路况恶劣而堵在了路上。两天后，第1装甲师仍未能抵达行动规划的出发点，加上作战文件被盟军缴获，奇袭的效果已不复存在。严格来说，狮鹫行动此时就已宣告失败{{Sfn|Pallud|1987|p=10}}。

12月17日夜，[[奥托·斯科尔兹尼|斯科尔兹尼]]参加了{{Internal link helper/en|第6装甲集团军|6th Panzer Army}}的参谋会议。在会议上他提出，既然奇袭无望，就应将第150装甲旅转作普通的[[装甲掷弹兵|装甲掷弹兵]]参与进攻。他的提议获得批准，三个战斗群分别往[[马尔默迪|马尔默迪]]以南集结，向位于{{Internal link helper/en|利尼厄维尔|Bellevaux-Ligneuville}}的第1装甲师师部报道。12月21日凌晨，第150装甲旅在Z战斗群未到位的情况下分兵两路，趁夜突袭[[马尔梅迪|马尔梅迪]]。防守马尔梅迪的美军虽然兵力悬殊但抵抗异常顽强，斯科尔兹尼的数次进攻最终都以失败告终。随着天气转好，[[盟军|盟军]]飞机得以重新开展[[對地攻擊|對地攻擊]]，夺取马尔梅迪愈发困难。12月28日，第150装甲旅由{{Internal link helper/en|第18国民掷弹兵师|18th Volksgrenadier Division (Wehrmacht)}}接替，回到本土，并于1945年1月23日解散。据斯科尔兹尼估计，约有15%的士兵在数日的战斗中[[陣亡|陣亡]]、负伤、或[[任務中失蹤|下落不明]]{{Sfn|Pallud|1987|p=11-15}}。

=== 渗透行动 ===
[[奥托·斯科尔兹尼|斯科尔兹尼]]在战后美军的审问中供称，他在[[突出部之役|突出部战役]]之初共派出八支突击队，其中六支成功打入盟军内部。1944年12月16日清晨，趁[[德意志國防軍|德軍]]大规模炮击造成的通信混乱，四支侦察小组和两支爆破小组伪装成掉队的士兵混入掩护{{Internal link helper/en|美国第1军|First United States Army}}侧翼的装甲纵队中，至16日夜已抵达[[列日|列日]]至[[那慕尔|那慕尔]]的[[默兹河|默兹河]]沿岸地区，随即按计划着手侦察和破坏行动{{Sfn|Infield|1981|p=88}}。

一支德军侦察小组在17日光明正大地开进了美军控制的小城[[于伊|于伊]]，他们花好几个小时观察美军的兵力部署和防御阵地，沿途和碰见的美军敬礼、聊天，顺便给问路的车队指引了错误的方向，没有引起任何怀疑{{Sfn|Infield|1981|p=86}}。另一支小组伪装成美军[[宪兵|宪兵]]，误导了滞留在从[[默兹河|默兹河]]至[[巴斯托涅|巴斯托涅]]道路上的大量盟军。美国{{Internal link helper/en|第84步兵师|84th Division (United States)}}某部受其影响，在森林里迷路了整整三天{{Sfn|Infield|1981|p=89}}。还有一支小组报告称已抵达[[马尔梅迪|马尔梅迪]]，沿途摧毁一座油料库，并扫清若干[[地雷陣|雷区]]。他们无意间进入了马尔梅迪，详细记录了美军的部署并将报告发给[[親衛隊第1師|第1装甲師]]的进攻矛头：[[约亨·派佩尔|派普]]战斗群。[[突出部之役|突出部战役]]伊始，渗透行动进展十分顺利。斯科尔兹尼得意地宣布：“美国人正忙着找自己的装甲师！”{{Sfn|Skorzeny|2016|p=172}}美军方面也报告称许多补给车队误入歧途、失联多日{{Sfn|Butcher|1946|p=710}}。
[[File:84TH_DIVISION_MPs_CHECKING_VEHICLES_AT_TRAFFIC_CIRCLE_near_Marche.jpg|缩略图]]附近盘查过往车辆]]困惑之余，[[盟军|盟军]]很快意识到有人故意在后方实施破坏活动。12月23日，[[盟軍遠征部隊最高司令部|盟军远征部队最高司令部]]发布消息，提醒各部队警惕“携带伪造证件、身穿英军制服”的德国间谍{{Sfn|Butcher|1946|p=710}}。盟军迅速在桥梁和城镇等要地建立起大量检查站和岗哨以盘查沿途士兵，通过[[搜身|搜身]]、检查证件、以及询问那些只有[[美国人|美国人]]才知道答案的问题（如美国[[美国州份|各州]]的首府、[[棒球|棒球]]比赛的结果、以及[[米老鼠|米老鼠]]女朋友的名字等）等手段，试图揪出潜伏在部队中的间谍。此举正中[[奥托·斯科尔兹尼|斯科尔兹尼]]下怀：狐疑和恐惧在盟军士兵中迅速蔓延，频繁的检查也让原本就已堵塞的交通更加迟缓。很多真的[[美军|美军]]士兵因各种原因被错抓甚至误杀：有人是因为穿着缴获的[[德国国防军|德军]]军靴；有人是因为答不上[[宪兵|宪兵]]出的刁钻问题；有人甚至仅仅是因为名字听起来像德国人{{Sfn|Butcher|1946|p=714}}。美国陆军参谋长[[乔治·卡特莱特·马歇尔|乔治·马歇尔]][[五星上将|上将]]不禁感慨：“这五十万的美国大兵每次见面就互相玩猫捉老鼠的游戏。”{{Sfn|Orfalea|1999|p=200-203}}

盟军高级将领也不能幸免。{{Internal link helper/en|美国第8军|VIII Corps (United States)}}军长{{Internal link helper/en|布鲁斯·克拉克|Bruce C. Clarke}}准将在巡视[[圣维特|圣维特]]时被一处检查站的宪兵拦下，他对上了口令，但误将[[芝加哥小熊|芝加哥小熊队]]说成是[[美國聯盟|美國聯盟]]的一员，而被逮捕审讯长达五个小时{{Sfn|McNeese|2003|p=85}}{{noteTag|芝加哥小熊队属于[[国家联盟|国家联盟]]。}}；“享受”同等待遇的还有{{Internal link helper/en|第12集团军群|Twelfth United States Army Group}}司令[[奥马尔·布拉德利|奥马尔·布拉德利]]中将。他在驱车返回总部的路上遇宪兵反复盘查，其中一个问他[[伊利诺伊州|伊利诺伊州]]首府的所在地。布拉德利给出了正确答案[[斯普林菲尔德_(伊利诺伊州)|斯普林菲尔德]]，结果却被拘留，因为宪兵一口咬定伊利诺伊州的首府是[[芝加哥|芝加哥]]{{Sfn|Caddick-Adams|2015|p=362}}{{noteTag|芝加哥是伊利诺伊州最大的城市，很多不明就里的美国人以为芝加哥是首府。}}。

另一方面，由于盟军警戒提高，以及自身对[[美国文化|美国文化]]的陌生，德军突击队也开始陆续落网。一支德军突击队在补给站加油时随口说出了[[英式英语|英式英语]]的“Petrol”而非[[美式英语|美式英语]]的“Gas”而引起怀疑，最终被捕；一名伪装成美军军官的德军士兵因伪造证件上画蛇添足地修改了真件的一处[[Typo|印刷错误]]而露馅{{noteTag|当时由美国陆军军务局配发的身份证最上方存在一处印刷错误：“Not a Pass. For '''Indentification''' Purposes Only”。却被伪造证件的德国人一丝不苟地改成了拼写正确的“'''Identification'''”。}}；还有一队德军四个人坐一辆[[威利吉普車|吉普車]]，不符合美军一般只乘两人的习惯而暴露{{Sfn|Caddick-Adams|2015|p=363}}。在[[奥托·斯科尔兹尼|斯科尔兹尼]]派出去的44人中，共有36人成功返回。值得注意的是由于当时德军习惯搜刮美军的衣物和装备，并不是每个被逮捕的德军士兵都来自斯科尔兹尼的突击队{{Sfn|Pallud|1987|p=14}}。

=== “刺杀艾森豪威尔” ===
在当时盛行于盟军的各种流言中，引发最大恐慌的莫过于所谓“刺杀艾森豪威尔”计划。1944年12月17日，由曼弗雷德·佩纳斯（{{Lang-de|Manfred Pernass}}）[[中士|中士]]、君特·比林（{{Lang-de|Günther Billing}}）军士长和威尔海姆·施密特（{{Lang-de|Wilhelm Schmidt}}）二等兵组成的德军突击队在[[艾瓦耶|艾瓦耶]]附近因对不上口令而被捕。在审讯中，施密特声称[[奥托·斯科尔兹尼|斯科尔兹尼]]已派突击队前往[[巴黎|巴黎]]，准备将[[盟軍遠征部隊最高司令部|盟军远征部队最高司令部]]司令[[德怀特·艾森豪威尔|艾森豪威尔]]上将及其参谋一网打尽{{Sfn|Pallud|1987|p=15}}。由于狮鹫行动的内容已通过被缴获的文件大白于天下，加之斯科尔兹尼早因[[橡树行动|解救墨索里尼]]和推翻[[霍爾蒂·米克洛什|米克洛什]]等一系列特种作战而恶名昭彰，美军立刻听信了这一流言并加强了艾森豪威尔的护卫。艾森豪威尔在[[凡尔赛_(伊夫林省)|凡尔赛]]的总部特里亚农旅馆被[[机枪|机枪]]和[[坦克|坦克]]围得水泄不通，他本人必须24小时待在旅馆内，不得外出，同时另派容貌与他相似的[[替身演員|替身]]在旅馆附近散步，以期引出德国杀手{{Sfn|Butcher|1946|p=725}}。艾森豪威尔就在严密的监视中孤单地度过了1944年的[[圣诞节|圣诞节]]。12月28日，他再也无法忍受这种近乎禁闭的安保措施，怒气冲冲地走出办公室，扬言哪怕有人取他性命他也不在乎，同时在全欧洲发布[[通緝令|通緝令]]，悬赏捉拿斯科尔兹尼这个“杀手、间谍和破坏分子”{{Sfn|Infield|1981|p=165}}。

甚至连[[英国陆军元帅列表|英国陆军元帅]][[第一代阿拉曼的蒙哥馬利子爵伯納德·蒙哥馬利|伯納德·蒙哥馬利]]也在这场混乱中遭了殃。当蒙哥马利听说艾森豪威尔被限制自由后，他便前往[[马尔梅迪|马尔梅迪]]，想借机在前线美军中提升一下威望。他不知道的是，阿登地区正盛行一个流言，称斯科尔兹尼已派一名和蒙哥马利惟妙惟肖的间谍出没于各检查点套取情报，所以美军对任何长得像蒙哥马利的人都格外警惕{{Sfn|Weingartner|1991|p=207-224}}。当他的座驾行驶到第一个检查点时，美军看守示意停车检查。蒙哥马利对此不屑一顾，下令继续前进。愤怒的看守开枪打穿了蒙哥马利座驾的轮胎，将他从车子里拖出来关进了旁边的一座谷仓。美军士兵表示不认识蒙哥马利，坚持要求他出示证件。蒙哥马利觉得这是在侮辱他，火冒三丈，威胁要把所有士兵都交[[軍事法庭|军事法庭]]审判。直到数小时后，由一名美军认识的英军上尉出面，这位陆军元帅最终才重获自由。得知消息的[[德怀特·艾森豪威尔|艾森豪威尔]]幸灾乐祸地表示：“斯科尔兹尼总算干了回正事。”{{Sfn|Infield|1981|p=88}}

== 后续 ==
[[File:Spies_executed..jpg|缩略图]]
行动中被捕的18名德军士兵被盟军[[軍事法庭|军事法庭]]以间谍罪起诉，所有人罪名成立，全部判为[[死刑|死刑]]。1944年12月23日，佩纳斯、比林和施密特三人被美军[[枪决|枪决]]{{noteTag|据称比林在行刑前的遗言是“元首希特勒万岁！”}}，另有三人于同日定罪，三天后在{{Internal link helper/en|亨利沙佩勒|Henri-Chapelle}}被处决；12月26日，七名德军间谍被定罪，30日被枪毙；31日又有三人定罪，1月13日在[[于伊|于伊]]伏法。[[德怀特·艾森豪威尔|艾森豪威尔]]指示[[奥马尔·布拉德利|布拉德利]]处置间谍问题，后者委派{{Internal link helper/en|美国第1军|First United States Army}}军长[[考特尼·霍奇斯|考特尼·霍奇斯]]中将组织军事法庭和行刑队{{Sfn|Pallud|1987|p=15}}。至于行动初日就被识破的前线指挥官君特·舒茨（{{Lang-de|Günther Schulz}}）中尉，他不知为何一直拖到1945年5月方才审判定罪，并于6月14日在德国城市[[不伦瑞克|布伦瑞克]]被{{Internal link helper/en|美国第9军|Ninth_United_States_Army}}处决{{Sfn|Sekiguchi|Stedman|p=103|2005}}。

[[奥托·斯科尔兹尼|奥托·斯科尔兹尼]]在战争的最后几个月内参与指挥了[[东普鲁士|东普鲁士]]、[[波美拉尼亚|波美拉尼亚]]和[[雷马根|雷马根]]等地的战事。1945年5月19日，他在[[萨尔斯堡|萨尔斯堡]]向{{Internal link helper/en|美国第30步兵团|30th Infantry Regiment (United States)}}[[投降|投降]]{{Sfn|Zabecki|2015|p=496}}。艾森豪威尔闻讯亲自派人前去审讯，调查他在{{link-en|马尔梅迪屠杀|Malmedy massacre}}中扮演的角色。1946年9月，斯科尔兹尼被押往[[纽伦堡|纽伦堡]]，以证人身份出席[[纽伦堡审判|纽伦堡审判]]{{Sfn|Infield|1981|p=136}}。1947年8月18日，斯科尔兹尼和其他九名参与狮鹫行动的军官作为被告出席了审判德军战争罪行的{{Internal link helper/en|达豪审判|Dachau trials}}。法庭指控斯科尔兹尼非法夺取美军战俘的衣物、使用有毒子弹、并违反[[海牙公约|海牙公约]]，“利用美军制服混入战斗，{{Internal link helper/en|背信弃义|Perfidy}}地开枪杀害美军士兵”{{Sfn|Skorzeny Trial|1949|p=91}}。斯科尔兹尼对此全部予以否认，坚称他们在渗透行动中从未对美军有敌对行为，也没有任何证据证明他们曾穿着美军制服向美军开枪，因此属于{{Internal link helper/en|合法战争诈术|Ruse de guerre}}{{Sfn|Skorzeny Trial|1949|p=92}}。英国前{{Internal link helper/en|特种作战行动处|Special Operations Executive}}特工{{Internal link helper/en|弗列斯特·托马斯|F. F. E. Yeo-Thomas}}出庭为斯科尔兹尼作证，以他的亲身经历证明二战期间美国、英国、苏联、波兰、法国等国的士兵都曾身穿德军制服进行过类似的渗透和欺诈行动而未受法律制裁，斯科尔兹尼不应为此承担罪责{{Sfn|Infield|1981|p=145}}。法庭最终宣布所有罪名不成立，将斯科尔兹尼当庭释放。尽管如此，审判长亚布拉罕·罗森堡（{{Lang-en|Abraham Rosenberg}}）对斯科尔兹尼的厌恶之情仍溢于言表：“我坚持认为他仍是欧洲最危险的男人。”{{Sfn|Infield|1981|p=143}}

== 流行文化 ==
* 1949年的美国电影《{{Internal link helper/en|战场 (电影)|Battleground (film)|战场}}》从[[第101空降师|第101空降师]]的视角讲述了狮鹫行动。不过现实中德军间谍并没有出没于[[巴斯托涅|巴斯通涅]]，而是集中于[[马尔梅迪|马尔梅迪]]。
* 1959年的美国电影《{{Internal link helper/en|最后的闪击战|The Last Blitzkrieg}}》，讲述一名参与狮鹫行动的德军突击队员如何训练、渗透、以及进行破坏活动的故事。有趣的是，扮演德军的主演[[范·強森|范·強森]]在10年前的《{{Internal link helper/en|战场 (电影)|Battleground (film)|战场}}》中却饰演识破德军间谍的真美军士兵。
* 1964年的系列电视剧《{{Internal link helper/en|卡夫悬案剧场|Kraft Suspense Theatre}}》第二季第二集的主题为狮鹫行动。
* 1965年的经典战争片《[[坦克大決戰_(電影)|坦克大決戰]]》中描写了德军的渗透行动。
* 2002年的美国电影《{{Internal link helper/en|哈特的战争|Hart's War}}》以[[突出部之役|突出部战役]]为背景，其中出现了伪装成美军的德军士兵。
