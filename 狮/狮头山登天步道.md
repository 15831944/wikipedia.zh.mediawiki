'''狮头山登天步道'''，建成于2009年（民国98年）5月，为[[台湾|台湾]][[南投县|南投县]][[集集镇|集集镇]]东南方的[[登山步道|登山步道]]，位于[[集集拦河堰|集集拦河堰]]之东侧。步道下为国家一级古迹[[开辟鸿荒_(石碣)|开辟鸿荒]]与原浊水桥吊桥的旧桥礅，吊桥右侧的废弃国军营舍是狮头山登天步道的起点<ref>{{cite web | url = https://www.medicaltravel.org.tw/Scenery-Content.aspx?l=3&idstr=S1_376480000A_000337&position=0&regcode=M&p=4&name= | title = 狮头山登天步道 | work = 台湾国际医疗全球资讯网 | accessdate = 2018-06-01}}</ref>。

== 景点简介 ==
狮头山登天步道始建于2006年（民国95年），[[集集镇|集集鎮公所]]斥资2000多万台币打造，于2009年5月完工。登天步道标高342公尺，坡度接近90度<ref name='ltn'>{{cite web| url=http://news.ltn.com.tw/news/society/breakingnews/1458748| title=集集登天步道及水雕峡谷美景 潜藏危机| work=自由时报| author=謝介裕| location=南投| date=2015-09-28| accessdate=2018-06-03| deadurl=no| archiveurl=https://web.archive.org/web/20160211153430/http://news.ltn.com.tw/news/society/breakingnews/1458748| archivedate=2016-02-11| df=}}</ref>。步道分为东西两侧，两侧都是木制阶梯，东侧有668阶，西侧有753阶，约半小时可以到达山顶。山顶地势较为平缓，设有一个小公园与观景台。在观景台中可以看到[[浊水溪|浊水溪]]两岸、[[集集拦河堰|集集拦河堰]]、集鹿大桥等景点<ref>{{cite web| url=http://tw.hiking.biji.co/index.php?q=trail&act=detail&id=546&title=狮头山登天步道| title=狮头山登天步道| work=健行笔记| accessdate=2018-06-03| deadurl=no| archiveurl=https://web.archive.org/web/20170607162153/http://tw.hiking.biji.co/index.php?q=trail| archivedate=2017-06-07| df=}}</ref><ref name='taiwanhot'>{{cite web | url= http://www.taiwanhot.net/?p=441320 | title= 強化觀光風貌 地方盼「獅頭山登天步道」周邊設施今年完成改善 | work= TaiwanHot | author= 吳素珍 | location= 南投 | date= 2017-04-12 | accessdate= 2018-06-03 | deadurl= no | archiveurl= https://web.archive.org/web/20180604051415/http://www.taiwanhot.net/?p=441320 | archivedate= 2018-06-04 | df=  }}</ref>，以及集集拦河堰下游长约2公里，由河水自然冲刷而成的「水雕峡谷」景观<ref name = 'ltn'/>。

== 步道整修 ==
登天步道所在地地质脆弱，每逢大雨后都有落石崩塌，导致步道损坏，同时木制步道自然腐化也造成损坏<ref name = 'ltn'/>。[[集集镇|集集镇公所]]曾多次对步道进行维修，例如公元2016年（民国105年）公所对步道进行为期半年的整修，于同年12月完成。但由于修缮工程浩大，集集镇公所多次维修的效果仍然有限<ref name = 'taiwanhot'/>。2017年4月12日[[行政院中部聯合服務中心|行政院中部聯合服務中心]]邀请集集镇公所、[[交通部观光局|交通部观光局]]、[[日月潭國家風景區#日月潭國家風景區管理處|日月潭国家风景区管理处]]等单位共同召开对「狮头山登天步道」的修缮工程会议，协商改善措施，并于4月28日到现场勘查<ref>{{cite web | url = https://www.eycc.ey.gov.tw/News_Content2.aspx?n=F4F47170BF08AEF4&sms=28AB1D16ECF7E63C&s=CF62305C42147635 | title =汤副执行长至集集镇狮头山登天步道修缮工程会勘| work = 行政院中部联合服务中心| date = 2017-04-28 | accessdate = 2018-06-03}}</ref>。日月潭國家風景區管理处处长洪維新已表示将对狮头山登天步道的受损程度进行整体评估，而后更换材质、强化结构<ref name = 'taiwanhot'/>。
