'''猴硐車站'''（硐音ㄉㄨㄥˋ<ref>[http://dict2.variants.moe.edu.tw/variants/rbt/word_attribute.rbt?quote_code=QzA4NzMx 硐通「洞」] {{webarchive|url=https://web.archive.org/web/20170831184529/http://dict2.variants.moe.edu.tw/variants/rbt/word_attribute.rbt?quote_code=QzA4NzMx |date=2017-08-31 }}</ref>）位於[[台灣|台灣]][[新北市|新北市]][[瑞芳區|瑞芳區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[宜蘭線|宜蘭線]]的[[鐵路車站|鐵路車站]]，是臺鐵少數幾座仍保有質樸鄉土風光的車站，週邊已廢棄的[[瑞三礦業|瑞三礦業]]廠區增加了車站的歷史感。車站雖名為「[[猴硐|猴硐]]」，實際位置卻在[[基隆河|基隆河]]的對岸，即西鄰的[[三爪子|三爪子]]境內。台視八點檔連續劇[[流氓教授_(電視劇)|流氓教授]]的劇內火車場景部分，都在此地拍攝完成。

== 車站構造 ==
*[[島式月台|島式月台]]兩座。

== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]。
*[[太魯閣列車|太魯閣號]]、[[普悠瑪列車|普悠瑪號]]、[[復興號列車|復興號]]、[[自強號列車|自強號]]皆不停靠。
*[[莒光號列車|莒光號]][[觀光列車|觀光團體列車]]不停靠，[[莒光號列車|莒光號]]南下（順行）停靠：642次、612次（逢週六停駛）；北上（逆行）停靠：561次、653次。
*[[臺鐵區間快車|區間快車]]（2016年10月後）
**逆行1551次、4021次、4027次及順行班次4026次、4030次不停靠本站，其餘班次（4028次、4052次、4023次）均停靠本站。
*[[臺鐵區間車|區間車]]：皆有停靠。
*宜蘭線4174次[[臺鐵區間車|區間車]]為終點站。
*本站的[[台語|台語]]站名廣播依猴硐的古名「猴洞仔」發音。
*與[[瑞芳車站|瑞芳站]]同為[[平溪線|平溪線]]主要始發車站。
*於站名更改為猴硐前，本站為台鐵少數仍有販售[[埃德蒙森式鐵路車票|名片式車票]]的車站，現已停售。
*因站名更改之故，名片式車票也將停用，本站於2014年3月17日至3月22日舉行最後特賣活動，活動結束後將不再販售<ref>[http://news.ltn.com.tw/news/life/paper/762388 侯硐改猴硐 台鐵絕版車票明開賣]</ref>。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費<ref>[http://www.railway.gov.tw/tw/CP.aspx?sn=1312&n=1333 臺鐵多卡通電子票證乘車營運規則]</ref>。

== 交通方式 ==
自[[瑞芳區|瑞芳]]市區沿[[明燈路|明燈路]]轉[[鄉道北37線|鄉道北37線]]行至[[猴硐|猴硐]]聚落即可。
== 歷史 ==
[[File:TRA_Houtong_Station_in_March_2014.JPG|恢復舊站名並進行過正立面整修工程的猴硐車站，攝於2014年3月]]
*1920年1月27日：設立，初名'''猴硐'''。
*1962年4月1日：與[[龍泉車站|龍泉]]、[[源泉車站|源泉]]、[[大山車站|大山]]、[[山佳車站|山佳]]、[[鎮安車站|鎮安]]、[[平和車站|平和]]等車站同時改稱'''侯硐'''。有一段時間車站門口使用'''侯（猴）硐'''標示。
*2013年12月10日：改回舊名'''猴硐'''<ref>{{cite news|url=http://news.ltn.com.tw/news/local/paper/759776 |title=「侯硐」火車硬票停售 舊票大清倉 |author=李雅雯 |work=自由時報 |date=2014-03-06 |accessdate=2015-08-03}}</ref>。
*2014年3月22日：因站名更改之故，名片式車票自本日16時起停止販售。
