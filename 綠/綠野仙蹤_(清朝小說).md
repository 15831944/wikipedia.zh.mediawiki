《'''绿野仙踪'''》，[[清朝|清朝]][[乾隆帝|乾隆帝]]时期[[李百川|李百川]]的[[长篇小说|长篇小说]]<ref name="multiplebk210 " />。創作於清乾隆十八年（1753年）至乾隆二十七年（1762年），是清中葉一部優秀的長篇小說。陶家鶴在他為《綠野仙蹤》寫的〈序〉中將其與《[[水滸傳|水滸傳]]》、《[[金瓶梅|金瓶梅]]》並舉，鄭振鐸也將《綠野仙蹤》與《[[紅樓夢|紅樓夢]]》、《[[儒林外史|儒林外史]]》並列為清中葉三大小說。

== 内容 ==
全书融合了[[神魔小说|神魔小说]]、[[世情小说|世情小说]]、[[历史小说|历史小说]]于一炉。<ref name="multiplebk210 " >{{cite book |language =zh-hans|author=李百川|title=《绿野仙踪》|year=1997年|publisher=中州古籍出版社|location=河南|pages=导言|isbn=9787534814945}}</ref> 讲述了[[明朝|明朝]][[嘉靖帝|嘉靖帝]]时期[[落第|落第]]士子冷于冰不肯攀附权贵[[严嵩|严嵩]]，又亲眼目睹朝廷忠良被害、授业恩师病故，看破官场腐败、社会黑暗、醒悟人生短促，于是决心出家，遁入玄门。冷于冰在[[杭州|杭州]]遇上了化身[[叫花子|叫花子]]的[[道士|道士]][[火龙真人|火龙真人]]，历经火龙真人的考验，他最终拜入火龙真人门下，学成呼风唤雨、召神遣将的本事，开始周游天下，扶危济困。冷于冰后来度脱灵[[猿|猿]]猿不邪、大盗[[连城璧|连城璧]]、[[狐女|狐女]]、农夫金不换、浪子温如玉等成仙，并协助忠良铲除奸相[[严嵩|严嵩]]，后被皇帝敕封普惠真人，位列[[金仙|金仙]]。<ref name="multiplebk1 " >{{cite book |language =zh-hans|author=李百川|title=《绿野仙踪》|year=2008年|publisher=珠海出版社|location=广东|pages=导言|isbn=9787806899687}}</ref>

其中温如玉倾家荡产一节最精彩，方言俚语很多，表现了清代普通人的生活状态，是古代世情小说的经典章节<ref>{{Cite book|title=《绿野仙踪》|last=李百川|first=|publisher=|year=|isbn=9787806899687|location=|pages=}}</ref>

== 思想 ==
全书揭露了官场黑暗，描绘了社会世态，具有强烈的批判意识，作者[[李百川|李百川]]寄希望于出现一个理想化的人物来拯救乾坤，重整纲常。<ref name="multiplebk1 " />

== 回目 ==
全书一共一百回。<ref name="multiplebk210 " /><ref name="multiplebk1 " />

== 风格 ==
全书以[[清朝|清朝]][[乾隆帝|乾隆帝]]中期的社会状态刻画[[明朝|明朝]][[严嵩|严嵩]]时期的社会生活，描写世情时，刻画细致入微，再现了[[清朝中叶|清朝中叶]]的现实生活。<ref name="multiplebk1 " />

== 禁毁 ==
[[同治|同治]]七年，[[江苏巡抚|江苏巡抚]][[丁日昌|丁日昌]]查毁《绿野仙踪》。<ref name="multiplebk210 " /><ref name="multiplebk1 " />
