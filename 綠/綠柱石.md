'''绿柱石'''（Beryl），又称为“'''绿宝石'''”，化学式为Be<sub>3</sub>Al<sub>2</sub>(<nowiki>SiO</nowiki><sub>3</sub>)<sub>6</sub>，其中含有[[氧化铍|氧化铍]]（BeO）14.1%，[[氧化铝|氧化铝]]（Al<sub>2</sub>O<sub>3</sub>）19%，[[氧化硅|氧化硅]]（SiO<sub>2</sub>）66.9%。六方晶系，[[晶体|晶体]]呈六方柱形，柱面有纵纹，晶体可能非常小，但也可能长达几米。[[硬度|硬度]]为7.5-8，[[比重|比重]]为2.63-2.80。纯净的绿柱石是无色的，甚至可以是透明的。但大部分为[[绿色|绿色]]，也有浅[[蓝色|蓝色]]、[[黄色|黄色]]、[[白色|白色]]和玫瑰色的，有玻璃光泽。[[英语|英语]]绿柱石（Beryl）一词来源于[[希腊语|希腊语]]“海水般的蓝绿色”（beryllos）。

== 种类 ==
自从史前时代，有许多种类的绿柱石就被看作是珍贵的[[宝石|宝石]]，纯[[绿色|绿色]]的被称为[[祖母绿|祖母绿]]（Emerald）。[[蓝色|蓝色]]的叫[[海藍寶石|海藍寶石]]（Aquamarine）。[[紅色|紅色]]的叫[[紅綠柱石|紅綠柱石]]（Red beryl）。[[粉色|粉色]]的为摩根石（Morganite），為了紀念美國銀行家約翰·皮爾龐特·摩根（John Pierpont Morgan）而被命名。透明無色的為透綠柱石（Goshenite），名字源於首次發現它的產地歌珊（Goshen）。亮[[黄色|黄色]]和黄绿色为金绿柱石（Heliodor），名字來自希臘文「太陽」（hēlios）與「禮物」（dōron），蜜黄色的比较常见。<br>

值得注意的是，綠柱石在古代被稱作「綠玉」，但「金綠玉」指的並非金绿柱石，而是金綠寶石，「變色綠玉」才是金绿柱石。由於[[銫綠柱石|銫綠柱石]]特性和綠柱石有一些不同，故在2003年被國際礦物學協會（IMA）認定為另一種礦物。<ref>見維基百科[[銫柱石|銫柱石]]條目</ref>

== 矿床 ==
绿柱石主要产于[[花岗岩|花岗岩]]伟晶岩中，但是也见于[[砂岩|砂岩]]、[[云母|云母]][[片岩|片岩]]中，经常和[[锡|锡]]、[[钨|钨]]共生，主要矿产在[[欧洲|欧洲]]的[[奥地利|奥地利]]、[[德国|德国]]、[[爱尔兰|爱尔兰]]；[[非洲|非洲]]的[[马达加斯加|马达加斯加]]，[[亚洲|亚洲]]的[[乌拉尔山|乌拉尔山]]和[[中国|中国]]的西北。

最著名的[[祖母绿|祖母绿]]产地为[[南美洲|南美洲]]的[[哥伦比亚|哥伦比亚]]，是存在于[[石灰岩|石灰岩]]基中。此外[[巴西|巴西]]，[[南非|南非]]，乌拉尔山，[[美国|美国]]也出产。在美国的新英格兰曾发现一个5.5[[米|米]]×1.2米，将近18[[吨|吨]]重的巨大绿柱石晶体。

== 应用 ==
绿柱石作为[[矿物|矿物]]主要应用于提取[[金属|金属]][[铍|铍]]，在欧洲古代用透明的绿柱石作成球来占卜，后来才改成[[水晶|水晶]]做的球。品相好的绿柱石是珍贵的宝石，用做饰物。
