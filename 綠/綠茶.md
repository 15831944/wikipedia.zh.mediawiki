'''绿茶'''是[[茶叶|茶叶]]的一种。经過[[杀青|杀青]]、[[揉捻|揉捻]]、[[干燥|干燥]]，大部份[[白毫_(茶葉)|白毫]]脱落，熱水浸泡后依種類呈[[綠色|淡綠色]]至黃綠色的茶湯。绿茶通过杀青工艺的不同可分为四种：炒青茶、蒸青茶、烘青茶、晒青茶。中国大陆與台灣在绿茶制作上以炒青為主；日本製作綠茶的方式則以蒸青為主。

炒青即将茶叶鲜叶在[[摄氏|摄氏]]120度左右的铁锅中翻炒，使其水分散失，使其变得绵软，以便做形。传统工艺为徒手翻炒，來到现代为机械翻炒。綠茶沒有焙火工序，所以可以稱為[[生茶|生茶]]。<ref>[http://blog.xuite.net/stephen_cyk/stephen/40531896-%E7%94%9F%E8%8C%B6.%E7%86%9F%E8%8C%B6%E6%90%9E%E4%B8%8D%E6%B8%85%E6%A5%9A%3F!+%E5%86%8D%E5%8A%A0%E4%B8%8A%E8%8C%B6%E7%9A%84%E7%99%BC%E9%85%B5%E5%BA%A6%E5%88%86%E9%A1%9E,+%E9%82%A3%E5%B0%B1%E6%9B%B4%E8%AA%AA%E4%B8%8D%E6%B8%85%E6%A5%9A%E4%BA%86! 生茶.熟茶搞不清楚! 再加上茶的發酵度分類,那就更說不清楚了!]</ref>

中国大陆大部分名茶为绿茶，如 产于杭州西湖[[龙井茶|龙井]]、苏州洞庭山[[碧螺春|碧螺春]]、秦岭山脉北麓的[http://baike.baidu.com/view/86384.htm 紫阳富硒毛尖]、[http://baike.baidu.com/view/2807409.htm 汉中仙毫]等。在[[清明|清明]]之前，茶树刚发芽时采摘的，为“明前茶”也称为“莲心”或“银针”；在[[谷雨|谷雨]]前采摘的，为“雨前茶”，茶叶已展开，也称为“旗枪”；在[[立夏|立夏]]前采摘的为“三春茶”，已经有两片叶展开，也称为“雀舌”；立夏后再采摘，叶片已变厚，为“四春茶”，也可称做“梗片”。

以[[青茶|青茶]]為主流的台灣，在[[新北市|新北市]][[三峽區|三峽區]]亦生產龍井茶及碧螺春等綠茶。

== 中國綠茶 ==
* [[西湖龍井|西湖龍井]]：中國綠茶的代表，[[浙江省|浙江省]][[杭州|杭州]][[西湖|西湖]]周邊產，古代作為貢茶。特徵是扁平狀的炒青綠茶，帶綠的葉片， 帶有翡翠色的茶色，散發著草或豆的清香，甘甜。在清明節前後（4月5日左右）以一芯一葉方式摘取的，被視為最高級品。其中又以[[獅子峰|獅子峰]]的獅峰龍井被視為特級品。
* [[黃山毛峰|黃山毛峰]]：[[安徽|安徽]]綠茶的代表，產於世界遺產[[黃山|黃山]]。特徵是帶黃白色毛的烘青綠茶，帶有透明黃金色的茶色，散發著栗子的氣味。以一芯二葉方式摘取的特級品被稱為「雀舌」。1955年被認定為中國十大名茶。
* [[碧螺春|碧螺春]]：產於[[江蘇省|江蘇省]][[苏州市|苏州市]]的東山及西山，尤以產於[[東山_(蘇州)|東山]]柑橘果園旁的被視為高級品。和龍井茶同為炒青綠茶，但其形狀是如捲貝般的螺旋狀，古代作為貢茶。春分到穀雨時期，以一芯一葉方式摘取的。
* [[黃山綠牡丹|黃山綠牡丹]]：產於[[安徽省|安徽省]]，[[清明節|清明節]]到[[穀雨|穀雨]]期，用一芯一葉到一芯三葉的方式摘取葉片，再用線綁成牡丹花的形狀。加入熱水後，會如牡丹花般的優雅盛開，澀味少，持久的甘甜及清香<ref>日本文藝社 日本茶紅茶中國茶健康茶</ref>。
* [[山心毛尖茶|山心毛尖茶]]与[[葵峰绿茶|葵峰绿茶]]：產於[[广西|广西]][[玉林市|玉林市]][[兴业县|兴业县]][[山心镇|山心镇]]，清明前后生产者为最佳，帶白毛的烘青綠茶，帶有透明黃金色的茶色，入口芬芳，弥漫着淡淡的茶香，沁人心脾。

== 注意事项 ==
饮用浓茶会导致胃肠动力异常。<ref>{{cite web|author1=蒋易容|title=胃不好怎么办？|url=http://www.dxy.com/column/3684|publisher=丁香医生|accessdate=2016-09-06}}</ref>綠茶裡含有單寧酸，因此茶喝太多都很傷腸胃，可能會導致胃痛、噁心等症狀。

綠茶會減慢身體吸收鐵質的速度，因此原本就缺鐵、貧血，或是生理期來的女性，都不應該大量攝取綠茶。
