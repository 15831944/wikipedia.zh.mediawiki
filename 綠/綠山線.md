'''綠山線'''是[[淡海輕軌|淡海輕軌]]的一條路線，也是淡海輕軌第一條興建的路線，2018年12月23日通車。<ref>{{Cite web |url = https://www.dorts.ntpc.gov.tw/home.jsp?id=34&parentpath=0,3,127,11 |title = 路線說明 |work=新北市政府捷運工程局|date=2018-09-12 |accessdate = }}</ref><ref>{{Cite web |url = https://tw.appledaily.com/new/realtime/20181129/1475409/ |title = 淡海輕軌「開箱」試乘　綠山線下月通車 |work=蘋果日報|date=2018-11-29 |accessdate = }}</ref><ref>[http://www.ntmetro.com.tw/news/?mode=data&id=794&parent_id=10436&type_id=10449 交通部准了!! 淡海輕軌 12月23日通車典禮 24日起免費搭乘1個月]</ref>

== 簡介 ==
綠山線屬於[[輕軌運輸系統|輕軌運輸系統]]，[[紅樹林站|紅樹林站]]至[[淡水行政中心站|淡水行政中心站]]為高架車站，[[濱海義山站|濱海義山站]]至[[崁頂站_(新北市)|崁頂站]]為平面車站。

綠山線起始於[[淡水信義線|淡水信義線]][[紅樹林站|紅樹林站]]旁，以高架型式沿中正東路北行，轉至淡金路（[[臺2線|臺2線]]）續往北，在[[淡海新市鎮|淡海新市鎮]]之濱海路轉往西，在濱海路一段306巷前轉為地面形式，接著至沙崙路轉往北至淡海新市鎮第一期第一開發區之北緣止。<ref>{{Cite web |url = https://www.dorts.ntpc.gov.tw/home.jsp?id=34&parentpath=0,3,127,11 |work=新北市政府捷運工程局|title = 路線說明 |date=2018-09-12 |accessdate = }}</ref>

綠山線的[[公共藝術|公共藝術]]由國際知名藝術家[[幾米|幾米]]打造，每個車站都有幾米創作的藝術供候車的乘客欣賞。<ref>林麒瑋.[https://udn.com/news/story/7323/2848153 淡海輕軌綠山線合龍 明年底完工通車].聯合報.2017-11-30</ref>
