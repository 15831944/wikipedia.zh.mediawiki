'''瓜地馬拉共和國'''（{{lang-es|República de Guatemala}}；{{IPA-es|gwateˈmala|lang}}），通稱'''瓜地馬拉'''，位於[[中美洲|中美洲]]，西濒[[太平洋|太平洋]]，东临[[加勒比海|加勒比海]]，北与[[墨西哥|墨西哥]]接壤，东北邻[[伯利兹|伯利兹]]，东南邻[[洪都拉斯|洪都拉斯]]和[[萨尔瓦多|萨尔瓦多]]。

== 名词来源 ==
瓜地馬拉“Guatemala”来自于[[纳瓦特尔语|纳瓦特尔语]]词汇库瓦赫特马尔兰（“Cuauhtēmallān”），或“有很多树的地方”派生于{{link-en|喀切人|K'iche' people|喀切玛雅}}词汇“很多树”{{sfn|Campbell|1997|p=}}{{sfn|Troika study abroad programs|2006|p=}}或有可能专指 Cuate/Cuatli 树（{{link-en|Eysenhardtia}}）。这个名称是跟随[[佩德罗·德·阿尔瓦拉多|佩德罗·德·阿尔瓦拉多]]的[[特拉斯卡拉州|特拉斯卡拉]]士兵在[[西班牙美洲殖民地|西班牙征服]]这块土地时所起的。{{sfn|Rain Forest Wordpress|2013|p=}}

== 历史 ==
{{main|危地马拉历史}}
从4世纪到10世纪瓜地馬拉的[[佩滕|佩滕]]低地地区是[[玛雅|玛雅]]文化的中心，10世纪末低地地区的玛雅文化被消灭后在中央高地地区仍然存在。1524年[[西班牙人|西班牙人]]来到瓜地馬拉，摧毁了当地的玛雅文化并开始殖民瓜地馬拉。由于西班牙殖民者的消灭政策，几乎所有此前的玛雅书籍都遭销毁殆盡，僅少數殘存至今。

1821年9月15日瓜地馬拉等中美洲國家宣布脱离西班牙統治。1824年到1839年瓜地馬拉為[[中美洲联邦|中美洲联邦]]的一员，直到1840年瓜地馬拉才完全独立。

1841年到1871年[[拉斐爾·卡雷拉|拉斐爾·卡雷拉]]及其之後的保守派人士组织瓜地馬拉政府。他们為保護自身利益，试图保留殖民时期留下来的等级制度，並不重視国家经济现代化。

1871年[[胡斯托·魯菲諾·巴里奧斯·奧延|胡斯托·魯菲諾·巴里奧斯·奧延]]的自由黨政權開始執政到1944年国家才開始致力於经济现代化。同时美國主導的[[联合果品公司|联合果品公司]]也開始收購瓜地馬拉大面积的咖啡和香蕉庄园並介入瓜地馬拉的政治。

1944年十月革命后前独裁者{{tsl|en|Jorge Ubico|豪爾赫·烏維科}}被推翻，瓜地馬拉實行民主與言論自由，瓜地馬拉史上第一部保护工人利益的法律生效，1950年到1954年执政的瑞士裔[[哈科沃·阿本斯·古斯曼|哈科沃·阿本斯]]总统施行了土地改革。当时僅全国人口2%的大地主卻佔有全国约70%的可耕地，而阿本斯開出土地改革的支票，承諾買下大地主持有的土地並重新分配給小農。阿本斯以此贏得選舉，土地改革計畫也立刻實行。針對擁有土地大於223英畝的地主，只要有未用於生產的土地便要徵收。最後從1059個農場徵收了平均4300英畝的土地，約有十萬名農民得到這些土地的所有權。

[[联合果品公司|联合果品公司]]在瓜地馬拉拥有其最大的庄园，而且还拥有[[中美洲国际铁路公司|中美洲国际铁路公司]]和瓜地馬拉仅有的一个海港。在联合果品公司的驱使下，美国外交部开始对瓜地馬拉展开宣传战，以反共的名義打擊阿本斯政權。[[中央情报局|中央情报局]]与瓜地馬拉军隊中的反对派合作，展开了所謂Operation PBSUCCESS的行动。1954年阿本斯政權被推翻，阿本斯總統流亡海外，{{tsl|en|Carlos Castillo Armas|卡洛斯·卡斯蒂略·阿马斯|卡斯蒂略·阿马斯}}成为新的独裁者<ref>John Vandermeer, Ivette Perfecto（2009）。《生物多樣性的早餐》，頁70-73。臺北市：綠色陣線協會，ISBN 978-986-84095-4-5</ref>。阿马斯新政府立刻廢除前阿本斯政府所作的改革。1957年阿马斯被刺，諷刺的是他的继承人是一个在40年代独裁政府中就已经以其血腥出名的军人{{tsl|en|Miguel Ydígoras Fuentes|米格爾·伊迪戈拉斯·富恩特斯}}。

不满阿本斯政權被推翻的农民组织了一支游击队，開始與政府軍展開長達36年的內戰，直到1996年交戰雙方才签署一份和平条约，正式结束內戰。这场內战中約數十万人丧生，上百万人流离失所。瓜地馬拉军队对原住民[[玛雅人|玛雅人]]的迫害近似[[种族灭绝|种族灭绝]]，仅1982年9月[[里奧斯·蒙特|里奧斯·蒙特]]當政時期就有9000位玛雅人慘遭杀害。1983年起瓜地馬拉政府逐步减少迫害原住民，国家重啟民主化歷程，1985年重新舉行大选。但全国人民贫富不均的现象依然没法解决，今天僅佔1%的少數人卻拥有瓜地馬拉60%以上的可耕地和财富。

== 地理 ==
[[File:Guatemala_Topography.png|thumb]]
{{main|瓜地馬拉地理|瓜地馬拉火山列表|瓜地馬拉湖泊列表}}
危地马拉位于中美洲，其北部的低地平原佩滕是[[热带雨林|热带雨林]]，中部的高地上的火山可达4200米，太平洋畔狭窄富饶的平地屬於热带气候。

中部的高原也是危地马拉的文化中心，高原地區海拔約1300到1800公尺，全年气温温和，白天温度約摄氏18到28度之间，在更高处一月和二月往往比较冷。危地马拉的首都[[危地马拉市|危地马拉市]]也位於此。該市有250万居民，是危地马拉最大的城市。旅游中心有[[阿蒂特兰湖|阿蒂特兰湖]]、老首都[[旧危地马拉|旧危地马拉]]、古老的玛雅城市[[蒂卡尔|蒂卡尔]]和一些其它著名的城市如[[克萨尔特南戈|克萨尔特南戈]]和[[奇奇卡斯德南哥|奇奇卡斯德南哥]]。

瓜地馬拉有37座[[火山|火山]]，其中四個是活躍的：[[帕卡亞火山|帕卡亞火山]]、[[聖地亞古多火山|聖地亞古多火山]]、[[富埃戈火山|富埃戈火山]]及[[塔卡那火山|塔卡那火山]]。[[富埃戈火山|富埃戈火山]]和[[帕卡亞火山|帕卡亞火山]]曾在2010年噴發。2018年[[富埃戈火山|富埃戈火山]]再次喷发，导致数百人
伤亡<ref>[http://sputniknews.cn/society/201806061025577075/ 危地马拉政府：火山爆发致约200人失踪].俄罗斯卫星通讯社.</ref>。

瓜地馬拉境內有數個[[湖泊|湖泊]]，其中最大湖泊為[[伊薩瓦爾省|伊薩瓦爾省]]境內的[[伊薩瓦爾湖|伊薩瓦爾湖]]、最深湖泊為[[索洛拉省|索洛拉省]]境內的[[阿蒂特蘭湖|阿蒂特蘭湖]]。

== 政治 ==
危地马拉是一个多党制的，独立的民主共和国。

1986年1月15日颁布的宪法规定其為三權分立政體，即[[行政|行政]]（[[總統|總統]]和[[副總統|副總統]]）、[[立法|立法]]（[[危地马拉国会|危地马拉国会]]）、和[[司法|司法]]（最高法院）的分离。

1993年修改憲法，又再增加了最高法院大法官數量從9人到13人、正副總統與國會代表任期由五年縮減為四年、大法官任期由六年縮減為五年、市長與市議員任期由兩年半增加至四年。總統由法定成年人直選，僅限一任，不得連任（但可隔屆參選）。

国会和总统的选举每四年一次，所有18岁以上的公民拥有选举权，军人没有选举权。国会指命最高法院成员，其任期为四年。

2003年11月9日的大选中全国大联盟的候选人[[奥斯卡·贝尔赫|奥斯卡·贝尔赫]]当选为总统，同时全国大联盟是议会中最强大的党派。参选率为54.5%。

如同其它拉丁美洲国家一樣，危地马拉没有悠久的民主传统，党派非常不稳定，许多党派在一个执政期后就解散。一些党派外的组织如军队和企业在政治界拥有很大的影响。

2007年9月9日，舉行总统和国会选举。<ref>冷彤，[http://news.xinhuanet.com/newscenter/2007-09/10/content_6697857.htm 危地马拉新一届总统选举平稳举行]，新华网</ref>[[奥托·佩雷斯·莫利纳|奥托·佩雷斯·莫利纳]]和[[阿尔瓦罗·科洛姆|阿尔瓦罗·科洛姆]]进入总统选举第二轮投票。<ref>冷彤，[http://news.xinhuanet.com/newscenter/2007-09/10/content_6698448.htm 危地马拉选举初步结果揭晓 无一获得过半数选票]，新华网</ref>最後由[[阿尔瓦罗·科洛姆|阿尔瓦罗·科洛姆]]贏得選舉，他成為危地马拉首位左派總統。

2011年选举，[[奥托·佩雷斯·莫利纳|奥托·佩雷斯·莫利纳]]代表右翼的愛國黨再度重返總統選舉，順利在第二輪投票以53.74%的百分比，比過代表中間偏右民主自由黨的Manuel Baldizón，當選瓜地馬拉總統。

有人称，该国人权状况不佳，[[原住民|原住民]]语言广播一直处于非法状态，直至最近{{when}}才有所缓解。<ref>http://www.culturalsurvival.org/news/indigenous-march-guatemala-ends-congressional-resolution</ref>

== 外交 ==

因長期內戰關係，瓜地馬拉的外交飽受孤立，從1980年代民主化和與叛軍簽署和平條約後，瓜地馬拉逐步重返國際舞台。

===與{{BLZ}}的邊界糾紛===
[[英國|英國]]與瓜地馬拉在1859年簽訂了條約，訂立出當時的[[英屬宏都拉斯|英屬宏都拉斯]]邊界，此界線即為今日[[貝里斯|貝里斯]]之國境範圍。惟瓜地馬拉認為該條約不符法律規範，因在十八世紀時英國與西班牙即簽訂合約，根據該條約，從西奔河（Sibun River）以下的土地及外海的島嶼等現今貝里斯南部4627平方英哩的國土，均為當年英國強行非法佔領。瓜地馬拉境內發行的許多地圖，均將貝里斯劃入國土範圍內。現兩國之間亦設有海關，可正常通行與運輸貨品。

== 经济 ==
[[File:Chichicastenango_Market.jpg|thumb]]
[[File:Coffee_beans_being_sorted_and_pulped.jpg|thumb]]
=== 農業 ===
农业占国民经济总值的四分之一，占出口的三分之二。半数的工作力是农业工作人员。[[咖啡|咖啡]]、[[糖|糖]]、[[香蕉|香蕉]]與[[荳蔻|荳蔻]]是主要产品，其中又以[[咖啡|咖啡]]為大宗，為全世界第六大咖啡生產國，主要產地有[[安地瓜|安地瓜]]、[[阿蒂特蘭湖|阿蒂特蘭湖]]、[[科萬|科萬]]、[[韋韋特南戈|韋韋特南戈]]、[[弗賴哈內斯|弗賴哈內斯]]、[[聖馬科斯省|聖馬科斯省]]等地。瓜地馬拉有一咖啡協會（ANACAFE）負責行銷瓜地馬拉咖啡。

瓜地馬拉的甘蔗種植面積廣大，為全國第二大農產品。香蕉則居第三位。另外瓜地馬拉也是全世界最主要的荳蔻生產國及出口國，主要出口至中東各國。

=== 工業 ===
工业和建筑仅占国民经济总值的五分之一。

1996年签署的和平条约结束了长达36年的战争，为外商投资消除了主要障碍。提高政府收入、与国际投资企业谈判和加强政府和私人金融机构的效率和公开性是未来的重要任务。假如国际农产品的价格不暴跌的话，在未来数年中危地马拉的经济增长应该与2000年差不多。

1998年的[[飓风|飓风]]在危地马拉较其邻国造成的危害比较小。

=== 旅遊 ===
[[File:Tikal_mayan_ruins_2009.jpg|thumb]]國家公園]]
瓜地馬拉旅遊產業發展蓬勃，根據瓜地馬拉觀光局（INGUAT）的分類<ref>[http://www.visitguatemala.com/ INGUAT官方網站]</ref>，可將全國分為數個區域：
* [[瓜地馬拉省|瓜地馬拉省]]周邊（Guatemala）：以現代化與殖民風格的建築為主，主要景點有[[安地瓜|安地瓜]]（UNESCO世界文化遺產）、[[瓜地馬拉市|瓜地馬拉市]]、[[阿馬蒂特蘭湖|阿馬蒂特蘭湖]]等。
* 高地（Highlands）：以[[馬雅|馬雅]]文化為主，主要景點有[[阿蒂特蘭湖|阿蒂特蘭湖]]、[[奇奇卡斯特南戈|奇奇卡斯特南戈]]、[[基切省聖克魯斯|基切省聖克魯斯]]、[[托托尼卡潘|托托尼卡潘]]、[[克薩爾特南戈|克薩爾特南戈]]等。
* [[佩滕省|佩滕省]]（Petén）：以[[馬雅|馬雅]]文明的探險為主，主要景點有[[蒂卡爾|蒂卡爾]]（UNESCO世界自然與文化遺產）、[[佩滕伊察湖|佩滕伊察湖]]、[[弗洛雷斯_(貝登省)|弗洛雷斯]]及多座馬雅遺跡等。
* [[伊薩瓦爾省|伊薩瓦爾省]]（Izabal）：以加勒比海風情為主，主要景點有[[利文斯頓|利文斯頓]]、[[伊薩瓦爾湖|伊薩瓦爾湖]]、[[馬雅|馬雅]]遺跡Quiriguá(UNESCO世界文化遺產)等。
* [[維拉帕斯|維拉帕斯]]（The Verapeces）：以豐富的自然景觀為主，主要景點有Semuc Champey、Grutas de Lanquín等。
* [[太平洋|太平洋]]海岸：以海上活動為主，主要景點有[[帕卡亞火山|帕卡亞火山]]、Monterrico、[[聖何塞港|聖何塞港]]等。
* 東部（The East）：以宗教與自然景觀為主，主要景點有[[圭哈湖|圭哈湖]]、[[埃斯基普拉斯|埃斯基普拉斯]]等。

== 交通 ==
[[File:BICITAXIS.JPG|right]]計程車]]
[[File:Nissan_Platina_as_taxi_in_Guatemala_City.jpg|right]]
瓜地馬拉有兩大主要[[國際機場|國際機場]]，分別為[[瓜地馬拉市|瓜地馬拉市]]的[[拉奧羅拉國際機場|拉奧羅拉國際機場]]與[[北碇省|北碇省]]Santa Elena的[[馬雅世界國際機場|馬雅世界國際機場]]。

瓜地馬拉陸上交通以[[公路|公路]]為主，CA1號公路連接[[墨西哥|墨西哥]]、[[薩爾瓦多|薩爾瓦多]]與[[瓜地馬拉市|瓜地馬拉市]]，CA9號公路連接瓜國兩大商港，CA2號公路為[[太平洋|太平洋]]沿岸聯絡道路，CA13則為[[貝里斯|貝里斯]]之聯絡道路。公路以汽車與公車為主，公車主要據點為[[瓜地馬拉市|瓜地馬拉市]]。瓜地馬拉曾經有鐵路，現已荒廢，於[[瓜地馬拉市|瓜地馬拉市]]第一區有一鐵道博物館，可以懷念過去瓜國的鐵道時光。

瓜地馬拉有兩大主要商港，一個為位於[[太平洋|太平洋]]沿岸城市[[聖何塞港|聖何塞港]]的港口奎特札爾港（Puerto Quetzal），另一個為[[加勒比海|加勒比海]]沿岸的[[巴里奧斯港|巴里奧斯港]]。

== 人口 ==
危地马拉43%的人口是当地玛雅人的后裔。在加勒比海岸也有非洲人的后裔。大多数危地马拉人生活在农村，但城市化不断加快。最主要的宗教是[[天主教|天主教]]，不过许多当地的传统也流入了天主教的崇拜和祈祷。约40%的人信[[新教|新教]]，1%的人信奉传统的玛雅宗教。

虽然西班牙语是官方语言，但一些当地人依然说不同的[[玛雅语言|玛雅语言]]。尤其在农村有时会有人不懂西班牙语。

1996年的和平条约中规定将一些官方文件和选举文件翻译为当地的语言。

== 教育 ==
在基礎教育方面，政府有設立一些公立小學和初級中學。這些學校都是免學費的，但校服、書籍、文具用品、交通等費用仍需自理，因此較貧困的兒童仍然無法上學。許多中產階級和富人的孩子就讀私立學校。在高等教育方面，全國僅有一所公立大學（[[聖卡洛斯大學|聖卡洛斯大學]]，縮寫 USAC）和十餘所私立大學（參見[[瓜地馬拉大學列表|瓜地馬拉大學列表]]）。聖卡洛斯大學設立於1676年，是瓜地馬拉規模最大且最古老的大學，也是美洲第4古老的大學。

在15歲及以上的人口當中，識字率只有74.5％，於中美洲各國名列最末。雖然識字率低，瓜地馬拉政府仍希望在未來20年內有所改善。<ref>[http://stats.uis.unesco.org/unesco/TableViewer/document.aspx?ReportId=121&IF_Language=eng&BR_Country=3200&BR_Region=40520 Education (all levels) profile – Guatemala]. UNESCO Institute for Statistics. Retrieved on 2012-01-02.</ref>像{{link-en|關懷兒童|Child Aid}}等一些組織，協助中部高地的村莊訓練教師，以努力提高兒童教育成效。缺乏農村教師培訓計畫，是農村識字率較低的一個關鍵因素。

在外國居留子弟的教育方面，[[瓜地馬拉市|瓜地馬拉市]]設有美國學校、馬雅國際學校及法國學校等。

== 文化 ==
=== 宗教 ===
瓜地馬拉主要信仰[[天主教|天主教]]，約占總人口之51%。其餘宗教包括[[新教|新教]]約佔36.7%，馬雅信仰約佔0.4%，無宗教信仰者占9.1%。

=== 語言 ===
{{main|瓜地馬拉語言}}
瓜地馬拉的官方語言為[[西班牙語|西班牙語]]。除西班牙語之外，瓜地馬拉大部分地區亦通用多種馬雅方言，加勒比海地區另有Garífuna方言。

=== 飲食 ===
在危地马拉，玛雅和西班牙殖民者的文化影响还很深。许多衣饰和食品依然是用传统的玛雅方式制作的，还有许多玛雅遗址存在。在加勒比海岸，非洲文化的影响也在舞蹈中可见、在宗教歌曲中可听、在食品中可尝。
