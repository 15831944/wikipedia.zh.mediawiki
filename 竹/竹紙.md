'''竹纸'''是一种用[[竹子|竹子]]为原材料制作的纸张，流行於[[東亞|東亞]]及[[東南亞|東南亞]]。[[中国|中国]]竹纸制造历史悠久，其工艺流程较为复杂，在古代被广泛使用。<ref>{{cite web |title=竹纸 |url=http://www.dpm.org.cn/collection/studie/230895.html?hl=%E7%AB%B9%E7%BA%B8 |website=故宫博物院 |accessdate=2018-10-05}}</ref>[[四川|四川]][[夹江|夹江]]和[[浙江|浙江]][[富阳|富阳]]的竹纸制造工艺被列为国务院第一批[[国家级非物质文化遗产代表性项目名录|国家级非物质文化遗产名录]]。<ref name="feiyi">{{cite web |title=国务院关于公布第一批国家级非物质文化遗产名录的通知 |url=http://www.gov.cn/zwgk/2006-06/02/content_297946.htm |website=中华人民共和国中央人民政府 |accessdate=2018-10-05 |archive-url=https://web.archive.org/web/20071222112306/http://www.gov.cn/zwgk/2006-06/02/content_297946.htm |archive-date=2007-12-22 |dead-url=no }}</ref>
==特点==
竹纸制造有分生料法和熟料法，生料法产出纸张色黄，纸质脆硬；熟料法产出纸张色白，纤维束少，竹纤维直而挺硬。与其他纸品相比，竹纸的抗老化性更差；但竹纸的制作材料易得，导致制作成本更低。<ref name="yanjiu">{{cite book |author1=邱敏 |title=古书竹纸研究 |date=2015年 |publisher=南京艺术学院}}</ref>

==历史==
有关竹纸的起源目前没有定论，中國关于竹纸的文献记载可以追溯到[[唐朝|唐朝]]。唐[[李肇_(唐朝)|李肇]]《唐国史补》言：“纸则有韶之竹笺。”竹纸在[[宋朝|宋朝]]兴起，目前存世宋版书已有许多使用竹纸。[[明朝|明朝]][[万历|万历]]以前的刻本大多使用[[棉纸|棉纸]]，至万历后竹纸使用量迅速增加，而绵纸几乎不被采用。例如[[天顺_(明朝)|天顺]]五年初版的《大明统一志》使用白绵纸，但[[隆庆_(明朝)|隆庆]]年后全部使用竹纸。[[清朝|清朝]]时竹纸使用达到鼎盛，仅闽浙地区的竹纸品种就有约五十种，制造工艺也越来越成熟。<ref name="yanjiu" />清末西洋机制纸涌入中国，导致竹纸逐渐衰落。但是到了[[抗日战争|抗日战争]]时期，进口物资被封锁，手工造纸又出现生产高峰。国民政府迁都重庆，带动了四川夹江竹纸的繁荣与变革，生产出优良书画用纸仿画纸。<ref name="dili" >{{cite journal |author1=萧春雷 |title=竹纸：一千年的繁华梦影 |journal=中国国家地理 |date=2009年 |issue=12}}</ref>但这只是竹纸衰落中的回光返照，如今的竹纸缺乏市场竞争力，已经面临严重危机。<ref>{{cite web |author1=三江都市报 |title=夹江竹纸面临衰落危机 |url=https://sichuan.scol.com.cn/lsxw/20090610/200961092635.htm |website=四川在线 |accessdate=2018-10-05 |archive-url=https://web.archive.org/web/20110423023221/http://sichuan.scol.com.cn/lsxw/20090610/200961092635.htm |archive-date=2011-04-23 |dead-url=no }}</ref>

[[日本|日本]]自[[平安時代|平安時代]]從中國傳入竹紙製作技術，成為[[和紙|和紙]]的一種，工藝傳承至現代。<ref name="小林" >{{cite news|author1=小林亜里|title=竹紙 十人十色の面白さ◇ギャラリー開き自ら手漉き 変化に富む風合い◇ |newspaper=[[日本経済新聞|日本経済新聞]]朝刊|date=2017-09-19}}</ref>

==工艺==
[[File:Process_of_Bamboo_paper_-_Insolation.jpg|thumb]]
===中國===
明代《[[天工開物|天工開物]]》詳細記錄了竹紙的製作工藝，分為五大步驟，斩竹漂塘、煮篁足火、荡料入帘、覆帘压纸、透火焙干；<ref>{{cite book |author1=宋應星 |title=《天工開物·殺青·造竹紙》}}</ref>清代《新詩造紙書畫譜》中提到的二十二道造竹紙工序為，“看竹體裁、伐竹作料、浸於池水、去皮存質、搥搗如絲、太陽以曝、灰水漿之、役工堆料、始入煌鍋、池水擺洗、二次擺洗、三次擺洗、泡浸灰水、灰水泡料、不堆再漂、納於窖中、用碓舂細、木槽盛之、需以簾床、上榨則實、焙而后乾、告厥成功”，<ref name="yanjiu" />都屬於熟料法。而常見的竹紙往往使用生料法。生料法省略蒸煮工藝，不深究漂白，所產紙色黃，俗稱毛邊紙。抗日戰爭時期，遷居[[重慶|重慶]]的[[張大千|張大千]]無法使用[[宣紙|宣紙]]而被迫使用夾江竹紙。張不滿於竹紙的韌性和色澤，親自來到夾江改良工藝，通過在竹料紙漿里添摻麻料纖維，使用進口[[漂白粉|漂白粉]]，使新產出的夾江竹紙的韌性加強，色澤更白。<ref name="dili" />

===日本===
現代日本的竹紙以嫩竹為原材料，浸泡半年至三年不等，再把竹纖維加入[[碳酸鈉|碳酸鈉]]蒸煮，瀝乾纖維後鎚打成餅狀，然後把纖維餅浸泡到水裡，用架子漉起紙漿，然後曬乾。<ref name="漉">{{cite web |title=竹紙の作り方 |url=http://itteki.jp/event/4439.html |website=若州一滴文庫 |accessdate=2018-10-09 }}</ref>
