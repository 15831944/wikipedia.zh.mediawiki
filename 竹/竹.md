'''竹'''，又称'''竹子'''，是'''竹亞科'''（[[學名|學名]]：{{Lang|la|Bambusoideae}}）植物的通称，属于[[禾本目|禾本目]][[禾本科|禾本科]]，可分為3族、约123属{{r|Soreng2015|Soreng2010|Gratani et al.}}，是禾本科中唯一具有[[乔木|乔木]]形态的类群，如热带型的[[簕竹族|簕竹族]]与温带型的[[青篱竹族|青篱竹族]]，但也包括低矮的[[草本|草本]]竹类[[黍竺族|黍竺族]]（{{Lang|la|Olyreae}}）。

== 分布 ==
竹子主要分布在[[地球|地球]]的北纬46度至南纬47度之间的[[热带|热带]]、[[亚热带|亚热带]]和暖[[温带|温带]]地区。世界上除了南极洲与欧洲大陆以外，其他各大洲均可发现第四次[[冰川|冰川]]以后的乡土竹种。亚洲分布最为丰富，其次为非洲和拉丁美洲，北美和大洋洲很少<ref>{{cite book|author1=[[康喜信|康喜信]]、[[胡永红|胡永红]]等|title=上海竹种图志|date=2011-07|publisher=上海交通大学出版社|pages=1|url=http://book.chaoxing.com/ebook/detail_81303024868cd84b6ee46775a03086eb6f1c2f975.html|ISBN=9787313071415|accessdate=2016-01-19}}</ref>。

中国廣東、浙江、江西、福建、湖南、云南和四川等都有大量不同种类的竹子分布。

由于高山和海洋的阻隔，世界范围内的竹类资源大致可以分为三大竹区。

===亚太竹区===
亚太竹区为世界最大竹区。南至南纬42度的[[新西兰|新西兰]]，北至北纬51度的[[库页岛|库页岛]]中部，东至太平洋诸岛，西至印度西南部。本区竹子约50多属，900多种。既有丛生竹，又有散生竹，前者约占3/5，后者约占2/5，其中有经济价值的约有100多种。

===美洲竹区===
美洲竹区至南纬47度的[[阿根廷|阿根廷]]南部，北至北纬40度的[[美国|美国]]东部，共有18个属，270多种。
===非洲竹区===
非洲竹区竹子分布范围较小，南起南纬22度[[莫桑比克|莫桑比克]]南部，北至北纬16度[[苏丹|苏丹]]东部。

== 特征 ==
竹为[[多年生植物|多年生植物]]，但不具備年輪，因而與一般人所認知的不同，竹子是一種巨大的「草」，大都具有地下[[根状茎|根状茎]]。通常通过地下匍匐的[[根茎|根茎]]成片生长。也可以通过开花结籽繁衍。节明显，各节生芽，地下茎各节的芽可萌发成地下横走的[[竹鞭|竹鞭]]或地上的[[竹竿|竹竿]]。竿节上的芽常形成各节的分枝。分枝上的叶为营养叶，批针形，具短柄。有一些种类的[[竹笋|竹笋]]可以食用。

[[File:Bamboo_Rhizome.jpg|thumb]]

在某些地区，也有丛生的竹子，其地下茎形成多节的假鞭，节上无芽无根，由顶芽出土成秆，整个植株丛状生长分布，典型的如凤尾竹、慈竹、麻竹和孝顺竹。一般生长于水热条件比较充足的区域。

竹子也是世界上生长速度最快的植物，有些竹地上部分的空心茎每天可长40厘米，完全成长后的高度可达35─40米{{cn}}。竹生长快速的原因是其枝干分节，故当其他植物只有顶端的分生组织在生长时，竹子却每节都在同时生长。但是，随着竹的不断长大，竹节外-{面包}-裹的鞘就会脱落，竹的高度就停止生长了，但其内部的组织生长充实依然在不断进行中。

== 栽培与繁殖 ==
多採用[[分株|分株]]、[[埋枝|埋枝]]、[[移鞭|移鞭]]、[[播种|播种]]法繁殖。[[竹鞭|竹鞭]]萌芽力与丛生性都很强，早春发芽前剪断竹鞭，连同竹竿分别栽种，当年夏季就可长出新竹。而竹子开花后的种子也可以播种。

栽种竹子宜选背风向阳、潮湿的环境。竹子生长快，生长量大，对水肥要求高，要求有充足的水湿，但亦要排水良好。要求土质深厚肥沃，富含有机质和[[矿物|矿物]][[元素|元素]]的偏酸性土壤。常见的庭院栽种有毛竹、凤尾竹、淡竹、旱园竹、刚竹等。

=== 内部分类===

== 竹樹開花 ==
{{main|竹樹開花}}
[[File:PICT0079_bambus-i-blomst_13apr2007_1200-81.jpg|right]]
[[File:Phyllostachys_Glauca_'Yunzhu'_in_flower.jpg|thumb]]
[[File:Panda_eating_Bamboo.jpg|thumb]]的主要食物，占其食物总量的99%]]
不論是[[中國|中國]]或[[印度|印度]]，都有竹子開花就會帶來災禍的傳說。中國過往經常說「竹樹開花，必有大災」，而印度亦有「竹樹開花，饑荒發生」的傳說。

竹一旦開花，鄰近的竹亦會相繼開花，造成大片竹林死亡，對一些以竹為主食的動物，勢必造成毀滅性的損害。

另一方面，[[竹米|竹米]]（竹的種子）亦是[[齧齒類|齧齒類動物]]的優良食物。科學家曾計算過，在[[印度東北部|印度東北部]]50年一度的竹樹開花周期發生時，老鼠的數量會激增4倍。突然增加的鼠量使村莊所儲存的糧食急速的被消耗，導致饑荒的發生<ref>Discovery Channel，(2006年)，《竹子開花的惡兆》，[[動物星球頻道|動物星球頻道]]。</ref>。所以，亦有說法指饑荒的發生只是因為齧齒類動物的突然增生，而非因為天災。

至於那一種說法為真，現時還未有定論。

竹子開花的週期頗長，通常為數十年到上百年不等，大部分竹類開的花皆為枯草般的色澤，形如稻穗狀。

== 竹子的文化意义 ==
[[File:Hsü_Wei_001.jpg|thumb]]
[[孔子|孔子]]的學生[[子路|子路]]将自己比作南山的竹子，削尖後能穿透犀牛的厚皮，意思是资质很好。《孔子家语》曰：“山南之竹，不搏自直，斩而为箭，射而达”。《记岭南行》：“岭南人，当有愧于竹。食者竹笋，庇者竹瓦，载者竹筏，爨者竹薪，衣者竹皮，书者竹纸，履者竹鞋，真可谓一日不可无此君也耶！”[[王徽之|王徽之]]居空屋中，便令种竹，曰：“何可一日无此君？”<ref>《晋书·王徽之传》</ref>。《旧唐书·李密传》：“罄南山之竹，书罪未穷；决东海之波，流恶难尽。”[[王勃|王勃]]《慈竹赋》：“如母子之钩带，似闺门之悌友；恐独秀而成危，每群居而自守。”[[蘇東坡|蘇東坡]]稱“可使食无肉，不可居无竹。无肉令人瘦，无竹令人俗。”<ref>苏轼：《于潜僧绿筠轩》</ref>，又《墨君堂记》赞竹云：“风雪凌厉，以观其操；崖石荦确，以致其节。得志，遂茂而不骄；不得志，瘁瘦而不辱。群居不倚，独立不惧。”。[[王丹桂|王丹桂]]的《秦楼月·咏竹》：“性贞洁，柔枝嫩叶堪图写。堪图写，四时常伴，草堂风月。孤高劲节天然别，虚心永永无凋谢。无凋谢，绿阴摇曳，瑞音清绝。”

在[[中國畫|中國畫]]的[[花鳥畫|花鳥畫]]裏，[[梅|梅]]、[[蘭|蘭]]、竹、[[菊|菊]][[四君子|四君子]]是重要題材，各代表一些[[美德|美德]]。北宋画家[[文同|文同]]画的竹子远近闻名。

中國成語裏多有提到竹子的：[[青梅竹馬|青梅竹馬]]、[[勢如破竹|勢如破竹]]、[[胸有成竹|胸有成竹]]。竹、[[松|松]]和[[梅|梅]]一起被稱為“歲寒三友”。竹子的空心，被中國文人引申為“虛心”，[[白居易|白居易]]《养竹记》说：“竹心空，空以体道，君子见其心，则思应用虚受者”；竹子的竹節，被引申為“氣節”；竹子的耐寒長青，被視為“不屈”<ref>[[郑板桥|郑板桥]]《郑板桥集·补遗》赞扬竹子：“盖竹之体，瘦劲孤高，枝枝傲雪，节节干霄，有似乎士君子豪气凌云，不为俗屈。”</ref>；竹子的高挺，被視為“昂然”；竹子的清秀俊逸，被引申為“君子”。

傳說在[[鞭炮|鞭炮]]發明之前，人們以燃燒竹子发出的爆裂响声來驅逐邪祟，所以有[[竹報平安|竹報平安]]（竹爆平安）這句成語，表示趨吉避凶。

中國民間傳說竹子的[[生日|生日]]為農曆[[五月十三日|五月十三日]]，又稱[[竹醉日|竹醉日]]。

== 用途 ==
[[File:Usage_of_Bamboo.JPG|thumb]]
竹子四季长青，用途广泛。在庭院中，是不可缺少的点缀假山水榭的植物。[[桂林|桂林]][[漓江|漓江]]旁广植[[凤尾竹|凤尾竹]]，成为一道独特的风景。[[安吉大竹海|安吉大竹海]]、[[蜀南竹海|蜀南竹海]]与[[赣南竹海|赣南竹海]]是中国有名的[[竹海|竹海]]景观。

由于竹子生长快，近几年为[[环保|环保]]考量，有大量[[家具|家具]]与[[纸|纸]]改用竹子制造。竹子也可制作工艺品、[[樂器|樂器]]等。

[[竹青|竹青]]是去火清凉的[[中药|中药]]材。将新鲜竹片放在火上烘烤得到的汁液被称为[[竹沥|竹沥]]，用于治疗[[咳嗽|咳嗽]]和[[化痰|化痰]]。[[竹笋|竹笋]]是常见的美食。{{fact|竹子的种子被称为[[竹米|竹米]]，因为不易得，所以是珍贵的美食。|time=2015-01-03}}

近年，将竹材用工程化方法，经物理和化学作用制成的竹纤维，被用作纺织品，做成毛巾和衣物等。

将竹材通过烘培，制成竹炭，被用在许多场合，包括去除环境气味，以及特殊风味食品。竹炭经过粉碎和活化制成的活性炭，有很好的吸附和净化作用，被用在汽车和家居，以及污水处理等。

竹也常用於建造[[棚架|棚架]]（[[搭棚|搭棚]]）、做為[[建材|建材]]及[[掃帚|掃帚]]、[[桌|桌]]、[[椅|椅]]等[[日用品|日用品]]。

在日本及韓國，有人會用之配以[[海鹽|海鹽]]、黃土等，以類似煉鋼的方法，加工成青竹鹽（或稱竹鹽），作為調味料、甚至加進個人衛生用品之中，{{Fact|竹鹽牙膏就是當中較知名的產物。據稱有消炎抗菌、防敏抗蛀、保持牙齦[[膠原|膠原]]、以預防[[牙齦|牙齦]]退化及[[牙周病|牙周病]]的效果。|time=2017-07-08T10:24:31+00:00}}

著名观赏性竹种有[[金錢竹|金錢竹]]、[[刚竹|刚竹]]、[[淡竹|淡竹]]、[[斑竹|斑竹]]、[[箭竹|箭竹]]、 [[方竹|方竹]]、[[孟宗竹|孟宗竹]]、[[綠竹|綠竹]]、[[刺竹|刺竹]]、[[長枝竹|長枝竹]]、[[麻竹|麻竹]]、[[桂竹|桂竹]]、[[紫竹|紫竹]]、[[凤尾竹|凤尾竹]]等等。
