'''竹幕'''（{{lang-en|Bamboo curtain}}）指[[冷战|冷战]]期间[[西方国家|西方国家]]对[[亚洲|亚洲]][[社会主义|社会主义]]阵营與[[资本主义|资本主义]]阵营邊界的称呼，相對于歐洲的[[鐵幕|鐵幕]]，亦是[[铁幕|铁幕]]在[[亚洲|亚洲]]的扩张，该词出现于1950年，最先由香港新聞學者[[張國興|張國興]]的《竹幕八月記》（''Eight Months Behind the Bamboo Curtain''）提出。

== 使用 ==
'''竹幕'''指在[[中华人民共和国|中华人民共和国]]和[[中华民国|中华民国]]（台湾）之间的[[台湾海峡|台湾海峡]]、中华人民共和国與[[香港|香港]][[澳門|澳門]]的邊界、[[大韩民国|南韓]]和[[朝鲜民主主义人民共和国|北韓]]之间的[[三八线|三八线]]、[[越南民主共和国|北越]]和[[越南共和国|南越]]之间的[[一七线|一七线]]以及[[东南亚|东南亚]]西北部模糊的社会主义国家边界。

== 历史 ==
由于当时社会主义国家和西方[[资本主义|资本主义]]之间交流很少，被认为是处于一种隔绝的隐秘状态。而東亞社会主义国家沒有像[[東歐|東歐]]大部份社会主义国家一樣是[[蘇聯|蘇聯]]的[[卫星国|卫星国]]（除了[[蒙古|蒙古]]、[[阿富汗|阿富汗]]以及蘇聯控制下的[[中亞|中亞]]各國例外）。

[[中华人民共和国|中华人民共和国]]在1960年便[[中蘇分裂|與蘇聯决裂]]，从此社会主义阵营开始分裂，并于1962年与[[印度|印度]]爆发[[中印战争|中印战争]]，并于1966年发动了[[文化大革命|文化大革命]]，1972年和1979年分别与[[日本|日本]]和[[美国|美国]]建交。而[[越南|越南]]、[[蒙古人民共和国|蒙古]]、[[老撾|老撾]]、[[阿富汗民主共和国|阿富汗]]、[[柬埔寨人民共和国|柬埔寨人民共和国]]和[[南也门|南也门]]则繼續随从苏联。[[红色高棉|红色高棉]]（1979年前）則支持中国。余下的[[朝鲜民主主义人民共和国|北韓]]当局沒偏向中或蘇的一方。

1989年發生的[[東歐劇變|東歐劇變]]和[[六四事件|中国六四事件]]影響東亞大部分社会主义国家的程度似乎不大，大多數國家仍舊繼續維持[[一黨專政|一黨專政]]和[[社会主义|社会主义]]的局面。然而[[蒙古國|蒙古國]]、[[柬埔寨|柬埔寨]]和[[阿富汗|阿富汗]]社会主义政權垮台以及[[南葉門|南葉門]]與[[北葉門|北葉門]]統一後放棄社会主义。最終[[中华人民共和国|中华人民共和国]]、[[越南|越南]]、[[老挝|老挝]]選擇經濟上的[[改革開放|改革開放]]來鞏固政權及嘗試改善國民的生活，只有[[朝鲜民主主义人民共和国|北韓]]仍未开放。

== 现状 ==
随着社会主义国家的陆续转变及开放和冷战的结束，资本主义国家与[[铁幕|铁幕]]或竹幕背后国家的差异逐渐消失，这个词也成为了历史名词，今天已经很少使用了。
