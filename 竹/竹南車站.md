'''竹南車站'''位於[[臺灣|臺灣]][[苗栗縣|苗栗縣]][[竹南鎮|竹南鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(北段)|縱貫線]]、[[臺中線|臺中線]]、[[海岸線_(臺鐵)|海岸線]]的[[鐵路車站|鐵路車站]]，為苗栗縣內重要交通轉運車站之一，也是進入中部後第一個大站，更是竹南鎮、頭份市、三灣鄉和造橋鄉對外的重要門戶。 

本站為山海線北端交會點，亦有「台鐵山海關」之稱<ref>[http://www.railway.gov.tw/tw/news_Detailnw.aspx?SN=25399 《漫遊鐵三角 活動正式開始》，交通部臺灣鐵路管理局官網]</ref>，西站前為[[竹南鎮|竹南鎮]]中心，東站緊鄰[[頭份市|頭份市]]，每年國定假日有大量返鄉人口回來。

== 營運狀況 ==
*目前為[[臺灣鐵路管理局車站等級#一等站|一等站]]。
*[[普悠瑪列車|普悠瑪號]]：皆不停靠。
*[[太魯閣列車|太魯閣號]]：皆停靠。
*[[莒光號列車|莒光號]]：除[[觀光列車|環島觀光列車]]外其餘皆停靠。
*[[自強號列車|自強號]]：除了北上（順行）106次(週五行駛)、128次(週一至週六行駛)、138次、182次(週六行駛)、6178次(週日行駛)，南下（逆行）113次、129次、143次(週五行駛)、181次之外，其餘皆停靠。
*[[台鐵區間快車|區間快車]]、[[臺鐵區間車|區間車]]：皆停靠，部分以本站為出發站或終點站。
*長編組[[臺鐵區間車|區間車]]為終點站。
*[[海岸線_(臺鐵)|海線]][[臺鐵區間車|區間車]]為起點、終點站。
*[[臺中線|山線]][[苗栗車站|苗栗車站]]－竹南車站[[對號列車|對號列車]]北上有18班次，南下有19班次。
*2013年度旅客人次排名為全國第20名，旅客成長率高達8.96%，更是前20名車站中成長率第1名。<ref>{{cite news|url=http://www.railway.gov.tw/Upload/intro/file/102Year/t11.pdf |title=102年台鐵車站營收旅次 |publisher=台灣鐵路局 |date=2014-03-31 |deadurl=yes |archiveurl=https://web.archive.org/web/20150924084601/http://www.railway.gov.tw/Upload/intro/file/102Year/t11.pdf |archivedate=2015-09-24 }}</ref>
*由於[[縱貫線_(北段)|縱貫線北段]]與[[臺中線|山線]]、[[海岸線_(臺鐵)|海線]]之東、西正線並非直線相通，因此通過竹南站的各級列車，會先減速經過道岔後再加速離站。
*[[縱貫線_(北段)|縱貫線]]經[[臺中線|山線]]南下列車，可於竹南站北邊的道岔，以限速60公里的速度通過，但若有其他列車需要待避，則不會作此安排。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
[[File:TRATRACK118.JPG|right]]
*1902年8月10日：車站落成，位於三角店-{庄}-，但站名取自1公里外的中港街（今竹南鎮中港里）。站房為木造車站。
*1903年5月25日：中港－[[苗栗車站|苗栗]]段通車。
*1920年10月1日：中港站改稱為'''竹南站'''。
*因'''山海線'''地位升格一等站。
*1929年：因為路線坡度過陡，拆除原本經過隧道的舊路線
-舊路線:原香山車站-(經台13線)-竹南車站，後因路線坡度過陡，路線改由香山車站-(經崎頂車站)-竹南車站
*1931年3月1日：臺北－竹南間雙軌完成通車。
*1935年11月6日：因同年4月24日台中大地震受損，車站改建為鋼筋水泥站房啟用。
*1943年4月：新建南號誌樓。
*1947年9月：新建北號誌樓。
*1995年5月22日：配合山線雙軌工程施工，舊站房拆除。
*1997年1月6日：新建站房完工啟用。
*2001年：南號誌樓拆除，移至[[車埕車站|車埕車站]]重組。
*2007年11月：北半段跨站站房完工。
*2009年1月：南半段及站房主體工程完工。
*2009年8月15日：跨站式車站完工，正式啟用。
*2011年2月26日：配合2011年[[台灣燈會|台灣燈會]]在苗栗活動疏運，創下單日旅客人次16萬紀錄<ref>[http://cnnews.rti.org.tw/index_MailFW.aspx?nid=283214 苗栗燈會擠爆 鹿港明年接手將重交通動線規劃] {{webarchive|url=https://web.archive.org/web/20140102191152/http://cnnews.rti.org.tw/index_MailFW.aspx?nid=283214 |date=2014-01-02 }}</ref>。
*2013年5月30日：啟用多卡通刷卡機。
*2014年7月16日：為加強疏運需要，開始停靠[[太魯閣列車|太魯閣號]]。
*2015年10月15日：改點後[[普悠瑪列車|普悠瑪號]]取代原來[[太魯閣列車|太魯閣號]],也開始停靠[[普悠瑪列車|普悠瑪號]]以始用278車次、285車次經海線每日來回。
*2016年10月20日：台鐵年度改點中將278、285車次改回[[太魯閣列車|太魯閣號]]行駛，本站再度停靠太魯閣自強號，取消停靠普悠瑪自強號。

== 改善計劃 ==
