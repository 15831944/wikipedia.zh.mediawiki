'''竹林里'''位於[[臺南市|臺南市]][[西港區|西港區]]西北部，原本里內只有「大竹林」一個聚落{{r|縣|page=588}}{{r|西港常民文化|page=35}}{{r|地名}}。但2018年1月29日-{里}-界調整後，原屬慶安里的「中洲仔」被改劃入竹林里的範圍{{r|里鄰調整}}。

== 沿革 ==
該里在日治時期屬於[[西港庄|西港庄]][[大字|大字]]劉厝，二次大戰後「大竹林」設置為竹林村{{r|縣|page=588}}{{r|西港常民文化|page=35}}{{r|地名}}。2010年12月25日，因臺南縣市合併升格，改為西港里。2018年1月29日，調整了與慶安里之間的里界{{r|里鄰調整}}。

== 聚落與地名 ==
竹林里的主要聚落是大竹林，地名來據說是當初開發時該地長滿刺竹的關係{{r|縣|page=588}}{{r|西港常民文化|page=35}}{{r|地名}}。該聚落是來自「福建省漳州府龍溪縣廿九都石美堡寮裡社（錦湖社）」郭姓移民{{NoteTag|分別是「開基九世祖」郭開高、郭闢高、郭旋與石是郭夢嚴等人，據汾陽殿碑文的說法他們是隨[[鄭成功|鄭成功]]先後入臺{{r|縣|page=588}}{{r|西港常民文化|page=35、36}}{{r|地名}}。}}所開墾，傳有八房，之後部分族人在南邊建立了大塭寮（永樂里）{{r|縣|page=588}}{{r|西港常民文化|page=35、36}}{{r|地名}}。也有說法是澎湖岐頭的郭姓來此移墾{{r|地名}}。大竹林與大塭寮關係密切，有著「頂下寮」聯庄關係{{r|西港常民文化|page=36}}{{r|地名}}。另外大竹林以庄廟汾陽殿為中心，分成東北角、中寮角、中央角、頂頭角四個角頭{{r|縣|page=589}}{{r|西港常民文化|page=36}}。

中洲仔在大竹林與西港仔街之間，原屬慶安里。康熙年間因為位在含西港與西港仔之間，繁榮一時{{r|縣|page=597}}{{r|西港常民文化|page=115}}。該地多為周姓居民，有當地中洲仔之名是從「中周仔」轉變而成的說法{{r|西港常民文化|page=116}}{{r|地名}}。據說中周子與北邊的檳榔林、下中洲過去曾合祀池府千歲，並參與[[八份姑媽宮|姑媽宮]]請水活動（[[西港香|西港香]]前身）{{r|縣|page=597}}{{r|地名}}。另外根據[[臺灣堡圖|臺灣堡圖]]等資料，過去的中洲仔稱為「下中洲」，而下中洲過去原稱「頂中洲」{{r|縣|page=597}}{{r|西港常民文化|page=116}}。

== 教育 ==
在大竹林西邊設置有成功國小，大致位在竹林、劉厝、永樂三里的中間位置{{r|地名}}。該校創於1947年，原是西港國民學校（今西港國小）分離教室，1953年獨立設為「成功國民學校」{{r|地名}}{{r|縣}}。據說是因為當初獨立設校時三個村（里）沒有共同「符號」，所以基於期許「學業成功」而將校名取為成功{{r|地名}}。該校有臺南縣第一座蝴蝶生態園{{r|西港常民文化|page=39}}。

== 宗教信仰 ==
大竹林的庄廟是[[大竹林汾陽殿|汾陽殿]]，主祀天上聖母{{r|縣|page=589}}。該廟主神據說是大竹林郭姓祖先當初來臺時所帶來供奉，但直到1976年才建廟，之前是供奉在祖厝（郭姓宗祠）{{r|西港常民文化|page=35}}{{r|地名}}。
