'''竹村'''（{{jpn|j='''竹むら'''|hg=たけむら|rm=Take Mura}}）是[[东京都|东京都]][[千代田区|千代田区]]一家创建于1930年（昭和5年）的[[和果子|日式糕点]]店。<ref name ="all">[https://allabout.co.jp/gm/gc/220041/ 歴史的建造物で味わう「竹むら」神田須田町]</ref>该店是[[东京都选定历史的建造物|东京都选定历史的建造物]]，<ref name = "tokyo"/>并以在多个动漫作品中出场而闻名。该店特色是日式炸馒头（揚げ饅頭）。<ref>[http://cn-tw.intheluggage.com/japan-tokyo-akihabara-restaurant-cafe_sweets-cafe--竹村%E3%80%80神田.html 竹村　神田]</ref>
