'''竹輪'''（在[[香港|港]][[澳門|澳]]稱為'''獅子狗'''、'''獅子狗魚蛋'''或'''獅子卷'''）是日本傳統食品，做法是把魚肉泥、麵粉、蛋白、調味料混合，裹在[[竹籤|竹籤]]或細木枝上並以火[[烤|烤]]或[[蒸|蒸]]熟。現在多已改用機器製造，不再使用傳統的做法。

== 歷史 ==
最早出現在文獻記載上是在[[類聚雑要抄|類聚雑要抄]]，書中繪製了稱為[[蒲鉾|蒲鉾]]的魚漿料理，就是今日竹輪的前身。

== 作法 ==
人們將[[狭鳕|狭鳕]]、[[狗母魚|狗母魚]]、[[鯊魚|鯊魚]]、[[飛魚|飛魚]]、[[多线鱼|多线鱼]]等的魚肉裡加上[[鹽|鹽]]、[[糖|糖]]、[[澱粉|澱粉]]、[[蛋白|蛋白]]等塞入竹圈，以竹製或金屬製的粗棒狀扦子厚厚夾住，然後火[[烤|烤]]或[[蒸|蒸]]熟。

竹輪是空心[[圓柱體|圓柱狀]]，多數[[直徑|直徑]]2厘米，長4至6厘米，頭尾淺[[黃色|黃色]]，中間[[棕色|棕色]]，有[[皺紋|皺紋]]狀表面，像被[[燒烤|燒烤]]過一樣。而根據不同地域，使用的[[魚|魚]]和[[製作|製作]]形式及[[味道|味道]]，竹輪有各自各的特徵。

== 不同地區的竹輪 ==
*在[[青森縣|青森縣]]有一種作法獨特的竹輪，稱為'''牡丹燒'''（{{lang|ja|牡丹焼き}}）或'''牡丹竹輪'''（{{lang|ja|牡丹ちくわ}}），原料主要是[[狹鱈|狹鱈]]。烤完之後，烤焦的部分就像是[[牡丹花|牡丹花]]花瓣的形狀一樣，因此得名。
*[[島根縣|島根縣]]的野燒竹輪（{{lang|ja|野焼きちくわ}}）、[[廣島縣|廣島縣]]（{{lang|ja|竹付き鯛ちくわ}}）、[[岡山縣|岡山縣]]的豆竹輪（{{lang|ja|豆ちくわ}}）、山陰的手握竹輪（{{lang|ja|手握りちくわ}}）、和飛魚竹輪（{{lang|ja|あご竹輪}}）、[[熊本縣|熊本縣]]的日奈久竹輪（{{lang|ja|日奈久ちくわ}}）、[[愛知縣|愛知縣]]的豐橋竹輪（{{lang|ja|豊橋竹輪}}）、[[德岛縣|德岛縣]]的竹竹輪（{{lang|ja|竹ちくわ}}）等，日本各地皆有不同的做法。
*在[[鳥取縣|鳥取縣]]東部和[[長崎縣|長崎縣]]某些地區，有在魚漿中加入豆腐的'''豆腐竹輪'''（{{lang|ja|とうふ竹輪}}）。<ref name="日経">{{cite web|url=http://www.nikkei.com/article/DGXKZO94295010R21C15A1NZ1P01/|title=起源は江戸時代　鳥取「とうふちくわ」 主役に脇役…変幻自在|publisher=[[日本經濟新聞|日本經濟新聞]]|date=2015-11-24|accessdate=2018-01-06}}</ref>
*在[[台灣|台灣]][[基隆市|基隆市]]，保留[[台灣日治時代|日治時代]]傳統製法的竹輪被稱為'''吉古拉'''，源自日語發音Chikuwa（{{lang|ja|ちくわ}}）。

== 獅子狗魚蛋名稱由來 ==

在1980年代初，香港[[TVB|TVB]]兒童節目時間播放一套[[日本動畫|日本配音卡通]]《[[忍者小靈精|忍者小靈精]]》，當中角色[[獅子狗|獅子狗]]最喜愛吃竹輪，當時絕大部份[[香港人|香港人]]從未見過這種食品，只知是類似[[魚蛋|魚蛋]]的小食，逐直接稱之為「獅子狗魚蛋」或「獅子狗」<ref>{{cite news|url=https://www.hk01.com/飲食/98077/秋葵獅子狗卷食譜-簡易日式便當-星星竹輪佐飯清新不膩|title=【秋葵獅子狗卷食譜】簡易日式便當　星星竹輪佐飯清新不膩|publisher=香港01|date=2017年6月15日|author=韓詠儀}}</ref><ref>{{cite news|url=https://hk.lifestyle.appledaily.com/etw/magazine/article/20170212/3_56273566/鳥取名物-獅子狗-魚肉加豆腐只因當年窮-|title=鳥取名物「獅子狗」 魚肉加豆腐只因當年窮？|publisher=蘋果日報|date=2017年2月12日}}</ref>。
