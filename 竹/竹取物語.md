《'''竹取物語'''》或名為《'''竹取翁物語'''》，是[[日本|日本]]最早的[[物語|物語]]作品。[[作者|作者]]與創作[[年代|年代]]不詳、乃是以[[女文字|女文字]]（[[假名|假名]]）所寫成的[[物語文學|物語文學]]。

《竹取物語》開闢了新型文人文學創作的先河，它敘述'''赫奕姬'''<ref>{{Cite web|url = http://www.zdic.net/c/b/18/38821.htm|title = 汉语词汇“赫奕”|accessdate = }}</ref>（又译作辉夜姬{{noteTag|かぐや姬，漢譯有不同寫法，如“煇夜姬”、“赫映姬”、“竹取公主”等等。但据日本語言學傢山田孝雄考證，此詞源出漢語“赫奕”，意謂光輝炫耀貌、显赫貌、美盛貌}}）被'''竹取翁'''自發光的[[竹|竹]]中取出，由其夫婦所養育。一般認為該書在[[平安時代|平安時代]]前期寫成、不過實際的成立年代依舊未詳。《[[萬葉集|萬葉集]]》卷十六第三七九一歌中、有一首〈竹取翁〉詠嘆天女的長歌、與本故事當有關聯。

== 作者 ==
作者未詳。不過就當時國民[[識字率|識字率]]來想，應當是上流階級、住在能得到[[貴族|貴族]]生活資訊的[[平安京|平安京]]近隣。又因其內容諭含[[反體制|反體制]]的思想，所以應該不是[[藤原氏|藤原氏]]的人。精通[[漢學|漢學]]、[[佛教|佛教]]、民間傳承、能寫假名文字，性別應該是男性。藉著以上推測、作者推定有[[源順|源順]]、[[源融|源融]]、[[遍昭|遍昭]]、[[紀貫之|紀貫之]]等說。

== 大綱 ==
故事寫了一位伐竹翁在竹林遇到一小女嬰，起名為赫奕姬。赫奕姬僅經過三個月就長成亭亭玉立的少女。五名[[貴族|貴族]]子弟向她[[求婚|求婚]]，她答應嫁給能尋得她喜愛的寶物的人。結果求婚者均遭失敗。皇帝想憑藉權勢強娶，也遭到她的拒絕。最後，赫奕姬在這群茫然失措的凡夫俗子面前突然[[升天|升天]]。而辛苦將她扶養長大的老爺爺和老奶奶，只好目送她的離開。

== 登場人物與時代 ==
* 雖然多半人物為架空創作、但是以部份史實人物擔任重角出演亦是本作的特徵。
* [[赫奕姬|赫奕姬]]（かぐやひめ）的原型人物是《[[古事記|古事記]]》裡，[[垂仁天皇|垂仁天皇]]的妃子，「迦具夜比賣命（かぐやひめ）」，她也是[[大筒木垂根王|大筒木垂根王]]的女兒。現在有些童話故事，會把其與住在月上廣寒宮的[[嫦娥|嫦娥]]描寫爲同一位女性。
* 竹取翁的名諱，讚岐造麿：來自[[開化天皇|開化天皇]]朝的讚岐垂根王。
* 石作皇子：被比定為[[丹比真人|丹比真人]]，姓氏錄有-{云}-:「丹比‧石作,同族也.」
* 阿部御主人：[[安倍御主人|安倍御主人]]。
* 車持皇子：被比定為[[藤原不比等|藤原不比等]]。（不比等母氏姓「車持」。）
* 石上麻呂足：[[石上麻呂|石上麻呂]]。
* 以上多半為[[天武天皇|天武天皇]]·[[持統天皇|持統天皇]]御世時活躍的人物，並參與[[壬申之亂|壬申之亂]]。是以故事時代、人物取材自[[奈良時代|奈良時代]]初期的可能性頗高。其後則演變為今日[[平安時代|平安時代]]的印象。
