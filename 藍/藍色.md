'''蓝色'''是一种[[颜色|颜色]]，它是'''红绿蓝'''光的[[三原色|三原色]]中的其中一元，在这三种原色中它的[[波长|波长]]最短（约470-440[[纳米|纳米]]）。

由于空气中灰尘对日光的[[瑞利散射|瑞利散射]]，晴天的天空是蓝色的。由于[[水|水]]分子中的[[氢|氢]]-[[氧|氧]]键对约750纳米的光的吸收，大量的水集中在一起呈蓝色，由于[[氘|氘]]-氧键吸收波长比较长的光（约950纳米），因此[[重水|重水]]是无色的。

蓝色的[[互补色|互补色]]是[[橘色|橘色]]。

== 音乐 ==
在音樂中藍色是一個經常被使用的名稱。許多樂隊和歌名以藍色命名。此外有两種由美國起源的音樂，一個稱為[[藍調|-{zh-cn:布鲁斯; zh-tw:藍調; zh-hk:藍調; zh-sg:蓝调;}-]](''Blues'')，另一個被稱作[[节奏布鲁斯|-{zh-cn:节奏布鲁斯; zh-tw:節奏藍調; zh-tw:節奏藍調; zh-sg:节奏蓝调;}-]](''R&B'')。

== 政治 ==
* 在[[美国|美国]]，藍色一般是[[民主党_(美国)|民主党]]的颜色。
* 在[[英国|英国]]和[[加拿大|加拿大]]，蓝色是[[保守党|保守党]]的颜色。其他的[[保守主義|保守主義]]黨派不少也以藍色作為象征，如[[歐洲人民黨|歐洲人民黨]]，法國[[人民運動聯盟|人民運動聯盟]]等。
* 在[[中華民國|中華民國]][[臺灣|臺灣]]，藍色是[[中國國民黨|中國國民黨]]（國民黨）的象徵顏色。現時臺灣政治上分成兩大陣營：泛藍與泛綠。
* 在[[香港|香港]]，藍色泛指[[反佔中|反佔中]]，親[[香港政府|政府]]、親[[香港警察|警察]]、親[[中華人民共和國|中]]。
* 在[[大韓民國|大韓民國]]，藍色是[[新政治民主聯合|新政治民主聯合]]的象徵顏色。過去[[民主黨_(大韓民國2011年)|民主黨]]以綠色為象征，2013年後改為藍色。藍色過去為[[新國家黨|新國家黨]]的象征，2012年改為紅色。
* 在[[聯合國|聯合國]]旗幟上的領域是淺藍色，選擇代表和平和希望。它引起了術語“ 藍化”。
* 在[[玻利維亞|玻利維亞]]，是[[左翼|左翼]][[社會主義|社會主義]]運動的顏色。
* 在[[印度|印度]]，藍色與[[賤民|賤民]]運動有關。
* 在[[猶太教|猶太教]]中是最重要象徵陰影的顏色。在[[以色列國旗|以色列國旗]]有兩個藍色橫條紋和藍色[[大衛之星|大衛之星]]，是[[猶太復國主義|猶太復國主義]]的象徵。藍色也象徵以色列政黨中右翼的[[利庫德集團|利庫德集團]]。

== 用途和象征意义 ==
* 一些自然界的景象都是藍色的，如[[海水|海水]]和[[天空|天空]]，給人柔和舒服的感覺。
* 是天空的顏色，象徵虛空、無窮以及神聖的。
* 由於是白天天空的顏色，因此會讓人有精神（讓大腦誤以為是在白天、應該打起精神來），應避免夜間使用，許多家電使用藍光LED做指示，其實是違反人體工學的。
* 在许多国家[[警察|警察]]的制服是蓝色的。警察和救护车的灯一般是蓝色的，容易使人冷靜。因此蓝色有时有急救的意思。
* 藍色在標記色彩標準中被用作提示、指示。
* 藍色筆為書寫用筆。
* 许多[[空军|空军]]和[[海军|海军]]的军装是蓝色的。一般空军为[[天蓝色|天蓝色]]，海军为[[藏青色|藏青色]]。
* [[蓝领|蓝领]]一般指工厂中从事体力劳动的工人。这个称呼来自于工人工作时穿的蓝色工作服。与此相对的是[[白领|白领]]，一般指办公室中的职员，他们一般穿白[[衬衫|衬衫]]。
* 在[[伊斯兰教|伊斯兰教]]中，蓝色是一种纯洁的颜色。
* 欧洲的贵族往往被称为有“蓝色的血”。有人认为这个称呼来自于他们苍白的皮肤。
* 在[[性別角色|性別角色]]裡，古代西方[[紅色|紅色]]或[[粉紅色|粉紅色]]一般是[[男性|男性]]的颜色，[[女性|女性]]則是藍色或[[粉藍色|粉藍色]]，東方[[紅色|紅色]]或[[白色|白色]]一般是[[男性|男性]]的颜色，[[女性|女性]]則是[[綠色|綠色]]或[[黃色|黃色]]，現代則一般多以藍色作為[[男性|男性]]的颜色，[[女性|女性]]則是[[紅色|紅色]]或[[粉紅色|粉紅色]]，这个习惯是从[[第一次世界大战|第一次世界大战]]后才开始的[http://answers.google.com/answers/threadview?id=238733]。有些情況下會以[[黑色|黑色]]、[[綠色|綠色]]或[[棕色|棕色]]來代表男性。至於[[淺藍色|淺藍色]]則常和男嬰聯繫在一起，因在傳統上西方父母往往會為男嬰穿上淺藍色的衣服。最主要原因是以冷色調的藍色中和男性的[[紅色|紅色]]體質，增加其穩重和克制；以暖色調的[[紅色|紅色]]中和女性的藍色體質，增加其勇氣與積極性。而在某些國家則以直觀方式分類，譬如[[日本|日本]]，男性顏色包括[[紅色|紅色]]，[[天藍色|天藍色]]，[[靛藍色|靛藍色]]；女性顏色包括，[[粉紅色|粉紅色]]，[[銀灰色|銀灰色]]，[[桃紅色|桃紅色]]；中性顏色兩者兼用的是：[[白色|白色]]，[[黑色|黑色]]，[[黃色|黃色]]，[[綠色|綠色]]，[[紫色|紫色]]，[[橙色|橙色]]等等。
* [[藍皮書|藍皮書]]一般是一部年書或類似的參考書。
* 從1860年代開始以最高速度横渡[[大西洋|大西洋]]的船只被授予藍色佩戴的稱號。
* 藍帶一般用來標誌質量非常高的事物。
* [[牛津大学|牛津大学]]的颜色是深蓝色，[[剑桥大学|剑桥大学]]的颜色是浅蓝色。
* 在西方受內傷時皮下的淤血往往被稱為藍色塊。華文稱這樣的淤血為烏青塊。
* 在醫學的示意圖中[[靜脈|靜脈]]一般被畫成藍色，實際上靜脈血是紫紅色的。
* [[微软|微软]]的[[Windows|Windows]]系统在系统发生无法矫正的错误的情况下显示[[藍白當機畫面|藍白當機畫面]]。
* 在[[美國南北戰爭|美國南北戰爭]]中，北方的軍裝是藍色的，因此藍色往往被用來代表北方，南方的軍裝是灰色的，因此灰色往往被用來代表南方。
* [[國際商用機器公司|國際商用機器公司]]的暱稱“大藍色”(''Big Blue'')。該公司製造的第一台戰勝[[西洋棋|西洋棋]]世界冠軍[[蓋利·卡斯帕羅夫|蓋利·卡斯帕羅夫]]的計算機被稱為[[深藍|深藍]]。
* 在英语国家中，假如在一个月中有两次满月的话，这个现象被称为“蓝月”。
* 蓝色被看作是一种比较寒冷的颜色。
* 在英文中，憂鬱與藍色共用同一詞彙「blue」。
* 歐美國家中藍色可能也有「色情物」的意涵，例如blue joke和blue film。
* [[鹼性|鹼性]][[物質|物質]]用多種[[酸鹼指示劑|酸鹼指示劑]]檢驗後，變色為藍色。
* [[pH值|pH值]]<3的物質用[[剛果紅|剛果紅]]檢驗後，變色為藍色。
* 在決鬥時兩方往往會分藍色和[[紅色|紅色]]，例如[[拳擊|拳擊]]、[[泰拳|泰拳]]和[[跆拳道|跆拳道]]。
* 大部分软件的[[用户界面|用户界面]]默认颜色是蓝色。
* 在MediaWiki軟體裡，[[紅色|紅色]]代表尚未撰寫的條目，藍色代表已經被創建及編輯過可瀏覽內容的條目。
*“藍色控”是指人特別鍾愛藍色，且對於看到任何關於藍色的人事物會感到興奮，想要擁有所有關於藍色的事物，稱為“藍色控”。
* 藍色在傳統西方神秘學中代表豪邁偉岸或冷靜硬朗的星座兩極，分別包括[[射手座|射手座]]或[[摩羯座|摩羯座]]；[[木星|木星]]或[[土星|土星]]；數字[[7|7]]與[[8|8]]，代表藍色的廣大與浩瀚。
