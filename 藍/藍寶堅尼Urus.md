'''藍寶堅尼Urus'''（{{lang-it|Lamborghini Urus}}）是義大利汽車製造商[[藍寶堅尼|藍寶堅尼]]所生產的一輛[[运动型多用途车|运动型多用途车]]（SUV）。這輛車於2017年12月4日面世，並將於2018年正式上市。<ref>{{cite news|url=https://www.motor1.com/news/180777/lamborghini-urus-everything-we-know/ |title=2019 Lamborghini Urus: Everything We Know |work=motor1.com |first=Jeff |last=Perez |date=2017-09-18 |accessdate=2018-05-03 }}</ref>Urus的名稱之意即[[原牛|原牛]]，指的是現代家養牛的祖先。<ref>{{cite journal|title=Lamborghini Urus|journal=Pebble Beach Concours d'Elegance program|date=2012-08|page=134}}</ref>

== 概念車 ==
[[File:P1080941-Lamborghini_Urus_Concept.jpg|thumb]]
[[File:Lamborghini_Urus_07.jpg|thumb]]
2012年4月23日，藍寶堅尼Urus概念車在2012年[[北京國際汽車展覽會|2012年北京國際汽車展覽會]]上首度亮相。<ref name="DRIVE">{{cite web|url=https://www.drive.com.au/motor-news/revealed-lamborghini-urus-suv-20120421-1xddp.html|title=Drive News - Lamborghini Urus SUV|date=2012-04-22}}</ref>此外，這輛SUV也曾登上2012年[[圓石灘車展|圓石灘車展]]。Urus概念車搭載著[[兰博基尼盖拉多|兰博基尼盖拉多]]的5.2升{{tsl|en|V10 engine|V10引擎}}，最大動力輸出達{{cvt|600|PS|hp kW|0|abbr=off}}，並配有全輪驅動系統。<ref>{{cite web |url=http://www.nydailynews.com/autos/street-smarts/lamborghini-urus-12-reasons-world-wild-article-1.2238604 |title= 12 things that will make Lamborghini’s SUV ridiculously fast and incredibly important |date= 2015-06-01 |first=Nick |last=Kurczewski |work=NYDailyNews.com |publisher= New York Daily News |accessdate= 2018-02-05 }}</ref>據報導，Urus擁有與其競爭對手（例如：[[保时捷卡宴|保时捷卡宴]]、[[賓利Bentayga|賓利Bentayga]]及[[奥迪Q7|奥迪Q7]]）相比最低的碳排放量，與旗下其他車型相比，Urus的定位更趨於日常駕駛，其外觀則承襲了該公司V12旗艦車型[[蘭博基尼Aventador|Aventador]]的鋒利線條設計。<ref>{{cite web |url=https://carbuzz.com/news/lamborghini-urus-suv-concept-revealed |title= Lamborghini Urus concept revealed |date= 2012-04-22|first=Jay |last=Traugott |publisher= Carbuzz |accessdate= 2018-02-05 }}</ref>藍寶堅尼在2017年發布的量產版Urus將搭載不同的動力單元，外觀也進行了修改。

== 動力系統 ==
[[File:Lamborghini_Urus_08.jpg|thumb]]
藍寶堅尼Urus並未搭載該廠著名的自然進氣{{tsl|en|V10 engine|V10引擎|V10}}（原概念車所使用的）及[[V12发动机|V12]]引擎，而是採用全新開發的4.0升{{tsl|en|twin-turbo|雙渦輪}}[[V8发动机|V8发动机]]，最大馬力達{{cvt|641|hp|kW|0|abbr=off}}。<ref>{{cite web |url=https://www.carmagazine.co.uk/car-news/first-official-pictures/lamborghini/lamborghini-urus-suv-2018-pictures-specs-and-full-details/ |title=Lamborghini Urus (2018) SUV: everything you need to know |date= 2018-01-11 |first=Tim |last=Pollard |work=Car Magazine |publisher=Bauer Consumer Media |accessdate=2018-05-03 }}</ref>藍寶堅尼稱這具引擎僅供藍寶堅尼獨家使用，並非是曾使用於{{tsl|en|Audi S and RS models|奧迪S及RS系列}}與[[奥迪A8|A8]]，且與{{tsl|en|Bentley Continental|賓利Continental|賓利Continental V8}}及{{tsl|en|Bentley Continental Flying Spur|賓利Continental Flying Spur|Flying Spur V8}}共享的[[奥迪|奥迪]]4.0升V8引擎之重新調校版本。<ref>{{cite web |url=https://www.whichcar.com.au/news/2018-lamborghini-urus-to-pack-485kw-twin-turbo-v8-punch |title=2018 Lamborghini Urus to pack 485kW twin-turbo V8 punch |date= 2017-05-16 |first=Barry |last=Park |work=WhichCar.com |publisher=Bauer Media |accessdate=2018-05-03 }}</ref>

== 生產製造 ==
2017年12月4日，這款車於藍寶堅尼[[聖阿加塔-博洛涅塞|聖阿加塔-博洛涅塞]]總部正式發表，成為該品牌繼{{tsl|en|Lamborghini LM002|藍寶堅尼LM002|LM002}}後的首款SUV。<ref>{{cite web |url=http://www.topgear.com.ph/news/car-news/report-lamborghini-boss-confirms-urus-suv-will-go-into-production |title=Report: Lamborghini Boss Confirms Urus SUV Will Go Into Production |publisher=Top Gear Philippines |date=2013-09-26 |accessdate=2013-09-26}}</ref><ref>{{cite web |url=https://www.motorauthority.com/news/1084159_lamborghini-urus-suv-to-debut-december-4 |title=Lamborghini Urus SUV to debut December 4 |first=Viknesh |last=Vijayenthiran |publisher=''Motor Authority'' |date=2017-09-13 |accessdate=2019-03-07}}</ref>藍寶堅尼計畫於2018年2月開始量產，預估首年產量達1000台，而至2019年將每年產3500台。<ref>{{Cite news|url=https://blog.caranddriver.com/2019-lamborghini-urus-suv-horsepower-revealed-by-ceo/|title=2019 Lamborghini Urus SUV Horsepower Revealed by CEO|access-date=2017-11-07|language=en}}</ref>為了生產此全新車型，藍寶堅尼同步擴建其位在聖阿加塔-博洛涅塞的生產基地。<ref>{{cite web |url=https://autoweek.com/article/supercars/lamborghini-urus-suv-factory-will-double-supercar-makers-output |title=Enormous Urus Factory Will Double Lamborghini's Output |first=Jay |last=Ramey |publisher=Autoweek |date=2017-10-17|accessdate=2019-03-07}}</ref>
