'''藍皮書計劃'''（{{lang-en|'''''Project Blue Book'''''}}）是一部美國[[時代劇|歷史時代]]電視劇，於2019年1月8日在[[歷史頻道|歷史頻道]]上首播<ref>{{Cite web |url = https://deadline.com/2018/10/project-blue-book-premiere-date-trailer-history-channel-ufo-drama-series-1202477171/ |title = ‘Project Blue Book’: History Reveals Premiere Date & Trailer For Robert Zemeckis' UFO Drama Series |author = Erik Pedersen |date = 2018-10-05 |publisher = Deadline Hollywood |language = en}}</ref>。[[埃丹·吉倫|埃丹·吉倫]]飾演主角[[約瑟夫·艾倫·海尼克|約瑟夫·艾倫·海尼克]]，本劇共有10集<ref>{{Cite web |url = https://www.denofgeek.com/us/tv/project-blue-book/268549/project-blue-book-release-date-trailer-cast |title = Project Blue Book Release Date, Trailer and Cast |author = Joseph Baxter |date = 2018-12-18 |publisher = Den of Geek! |language = en}}</ref><ref>{{Cite web |url = https://www.usatoday.com/story/life/tv/2017/05/25/blue-book-series-history-channel-robert-zemeckis/102147536/ |title = 'Blue Book' value: History picks up UFO drama from Robert Zemeckis |author = Jayme Deerwester |date = 2017-05-25 |publisher = USA Today |language = en}}</ref>。本劇改編自真實事件的[[藍皮書計劃|藍皮書計劃]]，該計劃調查研究[[不明飛行物|不明飛行物]]（UFO）。2019年2月10日，歷史頻道續訂有10集的第二季<ref>{{Cite web |url = https://tvline.com/2019/02/10/project-blue-book-renewed-knightfall-season-2-premiere-date-history/ |title = History Renews Project Blue Book, Sets Knightfall Season 2 Premiere Date |author = Ryan Schwartz |date = 2019-02-10 |publisher = TVLine |language = en}}</ref>。
