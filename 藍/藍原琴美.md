'''藍原琴美'''（{{bd|1990年|5月2日|catIdx=Aihara Kotomi}}）<ref>{{cite web|url=http://tower.jp/artist/2440233/%E8%97%8D%E5%8E%9F%E3%81%93%E3%81%A8%E3%81%BF|title=藍原琴美 - TOWER RECORD ON LINE|publisher=TOWER RECORD ON LINE|date=2017-06-26|accessdate=2017-12-20|author=TOWER RECORD}}</ref><ref name="profile">{{Cite web|url=http://www.atomicmonkey.jp/jp/archives/talent/aiharakotomi|title=藍原琴美的個人資料|publisher=Atomic Monkey|date=2016-10-16|accessdate=2018-05-31}}</ref>是[[日本|日本]]的女性[[聲優|聲優]]，[[青森縣|青森縣]]出身<ref name="profile"/>。[[Atomic_Monkey|Atomic Monkey]]所屬<ref name="profile"/>。

== 人物 ==
幼稚園時看《[[美少女戰士|美少女戰士]]》時，喜歡有男子氣概的[[天王遙|水手天王星]]，並會購買水手天王星的各種商品<ref name ="zenpen">{{Cite web|url=https://nizista.com/views/article2?id=84e27140a28d11e6a516534a54ca5a0c|title=藍原ことみインタビュー（前編） 「小さい頃から2次元の男の子になりたくて、それなら“声優”だと思いました」|publisher=Nizista|date=2016-11-19|accessdate=2016-11-21}}</ref>。小學五年級時看了《[[庫洛魔法使|庫洛魔法使]]》和《[[HUNTER×HUNTER|HUNTER×HUNTER]]》後，開始想變成「二次元的男孩子」，並萌生當聲優的想法<ref name="zenpen"/><ref name="fresh2015">{{Cite web|url=http://www.at-x.com/at-x_contents/summerfes2015/fresh.html|title=フレッシュ夏フェス隊！ ウチの夏フェス2015!|publisher=AT-X|accessdate=2017-06-03}}</ref>。

憧憬的聲優是[[緒方惠美|緒方惠美]]<ref name="zenpen" />，並且也仰慕在《[[美少女戰士Sailor_Stars|美少女戰士Sailor Stars]]》中飾演星野光的[[新山志保|新山志保]]<ref name ="kouhen">{{Cite web|url=https://nizista.com/views/article2?id=b96b2eb0a29211e6a1e7f19d3441c335|title=藍原ことみインタビュー（後編） 「演技も歌も、技術以上に気持ちが大切だと思っています」|publisher=Nizista|date=2016-11-20|accessdate=2016-11-21}}</ref>。

大學畢業後在母親建議下參加甄選，後來在網站Koebu獲得特別賞。之後於[[聲優獎|聲優獎]]的新人發掘甄選的決勝戰中獲得了就讀Atomic Monkey聲優·演技研究所的特別名額，並在養成所中接受[[長澤美樹|長澤美樹]]的指導<ref name="zenpen"/>。
