'''藍田站'''是[[港鐵|港鐵]][[觀塘綫|觀塘綫]]一個車站，位於[[九龍|九龍]][[觀塘區|觀塘區]][[鯉魚門道|鯉魚門道]]以南-{[[晒草灣|晒草灣]]}-的山坡。藍田站於1986年6月7日動工，1989年8月6日未有隨[[東區海底隧道|東區海底隧道]]鐵路部分通車，直至1989年10月1日正式啟用。

==車站設計==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Lam Tin Station 2014 04.JPG | caption1 = 進入此站的架空路軌（圖中右方）
}}
基於地勢，藍田站建於山坡之上，靠近觀塘一方路軌為架空，靠近油塘一方則建於地面，以月台所處樓層而論，本站屬於[[架空車站|架空車站]]，但由於車站設計成無窗密封環境才使人誤會自己身處地底而已，不過可以在鯉魚門道觀察，看到車站是興建在山坡上，西北面為架空部分，東南面為地底部分，位於山丘之下。由於車站上蓋是[[匯景花園|匯景花園]]的9至16座住宅大樓，故此車站月台和大堂的支柱比其它車站為粗，以支撐車站上方的物業。

由於設計計算錯誤，路軌旁的廣告燈箱比其他車站較高，與月台乘客的視角並非處於同一水平，而燈箱之間的距離也較大。此外，路軌與廣告燈箱之間亦留有較闊的空間作為逃生或供維修時使用，此站是第一個使用此設計的前地鐵車站，隨後前地鐵路線新建的車站和行車隧道均採用此設計。

===大堂===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Lam Tin Station 2017 09 part3.jpg | caption1 = 大堂（收費區內）
| image2 = Lam Tin Station 2019 02 part4.jpg | caption2 = 大堂（收費區外）
}}
藍田站內由1989年至2011年只設[[恒生銀行|恒生銀行]]，到2011年初才開設少量商店方便乘客購物，包括[[便利商店|便利店]]及[[麵包糕餅店|麵包糕餅店]]<ref name="shop"/>。此外，藍田站也有不少自助服務設施供乘客使用，包括[[自動櫃員機|自動櫃員機]]及[[自動販賣機|自動售賣機]]<ref name="shop"/>，以及「[[會員服務站|會員服務站]]」機供乘客登記為港鐵友禮會會員、查閱獎分及換領獎賞<ref>{{cite web|url=http://www.mtr.com.hk/ch/customer/services/csp.html|title=鐵路四處遊－「會員服務站」|publisher=港鐵公司|work=港鐵網頁|accessdate=2016年9月26日|language=zh-hant}}</ref>。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Lam Tin Station 2018 07 part2.jpg | caption1 = 2號月台（往黃埔方向）
| image2 = Lam Tin Station 2018 06 part1.jpg | caption2 = 連接月台及大堂的扶手電梯
}}
藍田站站設有2個月台，並以[[島式月台|島式月台]]排列，並設有[[月台幕門|月台幕門]]。由於此站與過海段同期興建，故此月台的焗漆板亦設有書法字，成為首個擁有書法字裝飾的觀塘綫車站（[[油塘站|油塘站]]及[[調景嶺站|調景嶺站]]為[[將軍澳綫|將軍澳綫]]同期興建，故一般視為[[將軍澳綫|將軍澳綫]]車站）。

在興建將軍澳綫之前，原本在觀塘綫的列車由藍田站開往[[鰂魚涌站|鰂魚涌站]]方向時，會有一小段的露天路段才進入東區海底隧道，但是後來該露天路段因為將軍澳綫通車而改為備用路段，並命名為[[藍田站後備行車隧道|藍田站後備行車隧道]]，可供工程車或車務調動使用。另外，當將軍澳綫未能提供過海服務時，觀塘綫列車可利用此隧道提供有限度的過海服務（如2013年12月16日，將軍澳綫電力故障導致全線暫停服務，觀塘綫部分班次通過此隧道以北角為終點站）。

[[File:Lam_Tin_Station_2018_07_part3.jpg|center]]
[[File:Lam_Tin_Station_2018_07_part4.jpg|center]]

另外在[[將軍澳綫|將軍澳綫]]通車後，由於觀塘綫由油塘站開往藍田站方向的路段是跨越備用行車隧道上方，加上要遷就觀塘綫和將軍澳綫在油塘站的走綫，導致設計為一急彎兼下坡路段。故列車由油塘站開往藍田站時當駛至路段一半時，會減速行駛以防止乘客跌倒，惟列車減速下坡前甚少有廣播提醒乘客緊握扶手，仍然偶有乘客因此而跌倒。

===出入口===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Sceneway Plaza Level 3 Access.jpg | caption1 = A出入口實際連接[[匯景廣場|匯景廣場]]中庭，乘客需要穿越商場才能抵達大堂。
| image2 = Lam Tin Station Exit A 20151122.jpg | caption2 = A出入口（啟田道）

藍田站設有5個出入口，主要連接[[鯉魚門道|鯉魚門道]]及滙景花園一帶。由於藍田站位於鯉魚門道以南晒草灣山坡，與藍田北的住宅區有一定距離，所以當時地鐵公司決定與發展商興建行人通道連接啟田道路面，該行人通道剛剛穿過[[匯景廣場|匯景廣場]]中庭，再沿藍田站範圍的扶手電梯才可到達啟田道，此種需要取道商場通道方可進出車站設計為全港首創。在各個出入口，均設有指示牌顯示出入口周邊的建築物及設施列表。

==利用狀況==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = Lam Tin Station 2018 06 part2.jpg | caption1 = 連接月台至大堂的升降機
}}
藍田站啟用初期，由於上蓋物業[[滙景花園|滙景花園]]仍未入伙，加上距離山上的屋邨和屋苑，以及山下的[[麗港城|麗港城]]較遠，故人流偏低。後來在通車一年後，隨著麗港城和滙景花園入伙，人流逐漸上升。而藍田站在將軍澳綫通車前亦曾經是[[將軍澳|將軍澳]]居民接駁地鐵過海的主要車站。同時大型日資[[百貨公司|百貨公司]][[八佰伴|八佰伴]]亦在該處開業，人流隨即急增。儘管八佰伴已結束營業多時，而將軍澳居民亦因為將軍澳綫通車而不再取道藍田站過海，但匯景花園商場的人流仍然十分高，加上山上的屋邨及屋苑均設有商場，因此現時藍田站的使用量仍然不俗。

===命名及選址===
在香港地鐵規劃「早期系統」時，離開觀塘站後下一站為'''觀塘村站'''，服務現時蓝田站周邊地區，車站選址鄰近鯉魚門道的空置校舍，早在1976年已被徵用。但隨着[[香港地鐵|前地鐵]]早期系統規模縮減，上行總站只至觀塘。

直到1984年有日本財團建議興建行車鐵路兩用的[[東區海底隧道|東區海底隧道]]，觀塘綫得以伸延至港島的[[鰂魚涌站|鰂魚涌]]，走綫之間新設本站以服務蓝田一带居民，最終本站於1989年10月1日啓用。

===車站擁有權===
在1989年至2008年，由於地下鐵路公司（2000年改組為地鐵有限公司，即今[[香港鐵路有限公司|香港鐵路有限公司]]）接受日本財團提出以[[民間興建營運後轉移模式|民間興建營運後轉移模式]]（BOT，建造-營運-移交）形式建[[東區海底隧道|東區海底隧道]]至觀塘路段，故通車後藍田站連上蓋項目[[匯景花園|匯景花園]]擁有權屬於[[新香港隧道有限公司|新香港隧道有限公司]]。根據當時的安排，地下鐵路公司雖然並未負擔東隧和此站的建築費用，但需每半年繳交一次隧道費和本站租金給東隧公司，至2008年鐵路隧道部分和本站專營權屆滿，移交政府為止，之後繼續由政府徵收此項租金。但因2000年地鐵有限公司上市，政府遂與東隧公司簽訂補充協議，東隧的鐵道隧道和本站擁有權於專營權屆滿後，以一千港元的象徵式代價，經由地政總署移交給地鐵公司（今稱[[港鐵公司|港鐵公司]]），而非移交政府。<ref>{{cite news|publisher = [[東方日報_(香港)|東方日報 (香港)]]|date=2010-05-24|url=http://orientaldaily.on.cc/cnt/news/20100524/00176_007.html|title=值15億東隧鐵路贈港鐵}}</ref>

===啟用=== 
藍田站於1989年8月9日啟用，當日英國[[戴安娜_(威爾斯王妃)|威爾斯王妃戴安娜]]在此站舉行開幕儀式，揭開車站啟用紀念牌扁。同日下午，車站月台路軌附近爆水喉，令兩邊路軌浸水，列車駛進站時都要慢駛，對很多乘客做成不便。結果在翌日（8月10日）地鐵方面再一次派單程票給乘客，前後總共派了三天，以作補償乘客的損失及再安排新的通車日。

直至同年10月1日，藍田站正式開放給予乘客出入，當日亦為觀塘綫首段（即[[修正早期系統|修正早期系統]]北段）[[石硤尾站|石硤尾]]至[[觀塘站|觀塘]]通車10週年。

===加建商鋪===
藍田站是港鐵，亦是前地鐵自啟用超過20年以來，除了[[恆生銀行|恆生銀行]]外而未有其他類別商店的車站。隨港鐵獲得車站擁有權後，於2010年7月進行加建商鋪工程。工程把近A出入口一邊的[[廣告|廣告]]燈箱及車站啟用紀念牌扁搬至近D出入口出閘機一邊，並且把近A出入口一邊的車費表，向東移前至另一條柱，而[[港鐵單程票|單程車票]][[售票機|售票機]]、[[八達通|八達通]]查閱機及八達通增值機，向東稍為移前至近入閘機斜對面。另外，工程期間消防喉轆則臨時搬至售票機舊有的位置，完工後整合至加建商鋪範圍所預留的位置。所有設施於2010年10月中完成改動。

===使用後備行車隧道===
2013年12月16日中午12時40分，將軍澳綫調景嶺站架空電纜故障，油塘站至[[寶琳站|寶琳站]]及[[康城站_(香港)|康城站]]服務癱瘓。港鐵抽調觀塘綫列車，以梅花間竹的方式提供有限度將軍澳綫過海服務。當每4分鐘、以[[觀塘站|觀塘站]]作終點站的列車調頭往觀塘重新接載乘客時；每8分鐘一班的列車便會繼續駛往藍田站，並在接載乘客後使用此後備行車隧道直接駛往鰂魚涌站，以[[北角站|北角站]]作終點站。
