[[File:Evolutionary_track_5m.svg|thumb]]的藍迴圈。]]

藍迴圈是恆星生命中的一個階段。在[[恆星演化|恆星演化]]領域中，在這個階段，它會從一顆溫度較低的恆星演化成較熱的恆星，然後再冷卻。這個名稱來自在[[赫羅圖|赫羅圖]]上[[主序星#演化軌跡|演化軌跡]]的形狀，它形成一個朝向圖的藍色（即更熱）一側的迴圈。

藍迴圈可以發生在[[紅巨星|紅巨星]]、[[紅巨星分支|紅巨星分支]]星或[[漸近巨星分支|漸近巨星分支]]星。有些恆星可能會經歷一個以上的藍迴圈。許多[[變星#脈動變星|脈動變星]]，例如[[造父變星|造父變星]]就是藍迴圈上的恆星。在[[水平分支|水平分支]]上的恆星，即使它們暫時比紅巨星或漸近巨星分支更熱，通常不會被稱為藍迴圈星。對單顆恆星而言，因為迴圈發展得很慢是無法觀測到的。因此，它是從理論和赫羅圖中的恆星性質和分布推斷出來的。

== 紅巨星 ==
{{main|紅巨星分支}}
[[File:CoRoT_HR.jpg|thumb]]在恆星演化軌跡上的藍色迴圈。]]

紅巨星分支（RGB）上的大多數恆星都有一個無活性的氦核，並停留在紅巨星分支上，直到發生[[氦閃|氦閃]]將它們移動到水平分支。然而，質量比{{solar mass|2.3}}更大的恆星沒有惰性的核心。它們在到達[[紅巨星分支技術|紅巨星分支]]之前，就會順利點燃氦，並在核心燃燒氦的同時變得更熱。在這個階段，質量越大的恆星會變得越熱，大約{{solar mass|5}}的恆星通常都會經歷一次持續大約一百萬年的一次藍迴圈。這種類型的藍迴圈在恆星的一生中只會經歷一次<ref name=bonn/><ref name=xu/><ref name=halabi/>。

== 漸近巨星分支 ==
{{main|漸近巨星分支}}

在漸近巨星分支（AGB）上的恆星多數都有碳和氧的惰性核心，並在核心周圍交替的同心殼層中進行氫和氦的核融合反應。氦殼的開始燃燒會導致[[漸近巨星分支#AGB階段|熱脈衝]]，在某些情況下，這將導致恆星暫時增加其[[有效溫度|有效溫度]]並執行一個藍迴圈。當外殼交替開關時，可能會出現許多次熱脈衝，因而同一顆恆星可能會出現多次藍迴圈<ref name=groenewegen/>。

== 紅超巨星 ==
{{main|紅超巨星}}
紅超巨星是已經離開[[主序星|主序帶]]，並膨脹得很巨大且表面低溫的恆星。它們的高亮度和低[[表面重力|表面重力]]，意味著它們正在迅速地失去質量。越明亮的紅超巨星失去質量的速度越快，會使它們越來越熱，也越來越小。質量最大的恆星，這可能導致它從紅超巨星永遠演變成藍超巨星；但在某些情況下，恆星會執行藍迴圈，並恢復成為紅超巨星<ref name=meynet/><ref name=saio/>。

== 不穩定帶 ==
{{main|不穩定帶}}
執行藍迴圈的恆星穿過HR圖主序帶上方的黃色區域，因此它們中的許多恆星都會穿越稱為不穩定帶的區域，而在這個區域的恆星，外層通常是不穩定和脈動的。[[室女座W型變星|室女座W型變星]]被認為是在漸近巨星分支上，穿越不穩定帶的藍迴圈星。[[造父變星|造父變星]]被認為是在紅巨星分支中穿越不穩定帶，而質量更大的藍迴圈星。這兩種類型的恆星，在其生命的這個階段都有發光不穩定的[[光球|光球]]，而儘管大多數恆星的質量都還不足以融合碳或到達[[超新星|超新星]]，但往往都具有[[超巨星|超巨星]]的[[光譜|光譜]] <ref name=groenewegen/><ref name=turner/><ref name=duerbeck/>。
