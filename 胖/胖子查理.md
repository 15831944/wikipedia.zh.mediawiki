'''查理三世'''（{{lang-de|Karl III der Dicke}}）或称胖子查理（{{lang-fr|Charles le Gros}}；{{bd|839年|6月13日|888年|1月13日|catIdx=C}}）是[[加洛林王朝|加洛林王朝]]的[[德国君主列表|东法兰克国王]]（876年起），[[法国君主列表|西法兰克国王]]（884年起）和[[神圣罗马皇帝|神圣罗马皇帝]]（称'''查理三世'''，881年起）。

== 生平 ==

查理三世是[[东法兰克王国|东法兰克王国]]的缔造者[[日耳曼人路易|日耳曼人路易]]之子。由于先后获得了东法兰克和西法兰克的王位，查理三世在位时短暂地重新统一了[[查理大帝|查理大帝]]统治过的全部疆土（包括[[意大利|意大利]]，查理三世于879年成为意大利国王），并于881年由[[教皇|教皇]][[若望八世|若望八世]]加冕为“神圣罗马皇帝”。但他的统治非常虚弱，帝国的统一很大程度上是空有其表。887年，查理三世被他的侄子[[克恩滕|克恩滕]]公爵[[阿努尔夫_(东法兰克)|阿努尔夫]]废黜并取代。

胖子查理的垮台标志着统一的法兰克国家的历史的结束。帝国彻底分裂：东法兰克和[[洛林|洛林]]归属阿努尔夫；西法兰克归[[巴黎|巴黎]]伯爵[[厄德|厄德]]；[[上勃艮第|上勃艮第]]由[[鲁道夫一世_(勃艮第)|鲁道夫一世]]继承；下勃艮第（[[普罗旺斯|普罗旺斯]]）归[[瞎子路易|瞎子路易]]。

== 死亡 ==

查理三世最后只剩下在西法兰克政权。888年1月13日，卡尔死于[[加洛林帝国|加洛林帝国]][[多瑙艾辛根|多瑙艾辛根]]。死后被葬在[[博登湖|博登湖]][[赖兴瑙岛|赖兴瑙岛]]。

== 配偶与子女 ==

查理三世与[[施瓦本的理查蒂丝|施瓦本的理查蒂丝]]（845年—920年）结婚。两人生有两子，名为'''卡洛曼'''<ref>死于876年。</ref>和'''伯纳德'''<ref>生于881年，死于891年。只活了10岁。</ref>。
