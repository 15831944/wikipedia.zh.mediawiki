《'''邊緣人 ～教你製作情婦的方法～'''》 （{{lang-ja|'''フリンジマン～～愛人の作り方教えます'''}}，{{lang-en|'''Fringe Man'''}}）為[[東京電視台|東京電視台]]2017年播出的電視劇，由板尾創路、[[大東駿介|大東駿介]]、淵上泰史、森田甘路、東幹久、村松利史等人主演，於2017年10月7日首播<ref>{{cite web|url=https://thepage.jp/detail/20171011-00000005-wordleaf|title=愚かな男性たちの行動に笑いあり『フリンジマン～愛人の作り方教えます～』|publisher=thepage|accessdate=2018-01-13|date=2017-10-11|language =ja}}</ref><ref>{{cite web|url=http://tower.jp/article/feature_item/2017/09/24/0705|title=PassCode、“bite the bullet”がドラマ「フリンジマン～愛人の作り方教えます～」の主題歌に決定|publisher=tower|accessdate=2018-01-13|date=2017-09-23|language =ja}}</ref>。

== 情婦聯盟規章 ==
# 四人之間彼此不能有私人隱藏的秘密。
# 四人要為彼此找情婦的行動上全力支援。
# 四人的情報絕對不能向其他人洩漏。
# 禁止愛上情婦或被情婦所愛上。
