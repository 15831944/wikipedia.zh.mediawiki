'''故宫改建计划'''，是指在[[清朝|清朝]]灭亡後，[[中國|中國]]國內部分人士要求将现在[[北京故宫|北京故宫]]内的建筑拆除用作它用的计划。

== 缘起 ==
1928年，[[国民政府|国民政府]][[国民政府委员会|委员]][[经亨颐|经亨颐]]提出“废除故宫博物院，分别拍卖或移置故宫一切物品”的议案，称故宫博物院“研究宫内应如何设备，[[皇帝|皇帝]]所用的事物”，“岂不是预备哪个将来要做皇帝，预先设立大典筹备处吗？”“皇宫不过是天字第一号逆产，逆产应当拍卖。”国民政府将经亨颐的提案函请中央政治会议复议，结果引发激烈争论，终被否决。（參見[[國民政府中央逆產處理委員會|國民政府中央逆產處理委員會]]。）

1948年，[[中共|中共]]刚发动[[平津战役|平津战役]]之时，曾因采纳[[梁思成|梁思成]]、[[张奚若|张奚若]]等人的建议，使[[北京城|北京古城]]得以幸存，这也让中共一度颇得建筑界的支持。然而，中共执政后北京的城墙、城楼、牌楼等古建筑却开始被陆续拆除。这也让以梁思成为代表的保留古建筑派深感失望与气愤。

1952年10月，[[北京市政府|北京市政府]]召开会议讨论工程项目，梁思成在笔记本上记录了一位发言者的意见：“不同意[[天安门|天安门]]内做[[中央政府|中央政府]]。”可见当时已有拆除并改建故宫之声浪。 

1955年，梁思成遭批判。当时在[[中宣部|中宣部]]任职的[[何祚庥|何祚庥]]在《学习》杂志发表批判文章称：“旧北京城的都市建设亦何至于连一点缺点也没有呢？譬如说，北京市的城墙就相当地阻碍了[[北京市|北京市]]城郊和城内的交通，以致我们不得不在城墙上打通许许多多的缺口；又如北京市当中放上一个大故宫，以致行人都要绕道而行，交通十分不便。”
 
1958年1月，[[毛泽东|毛泽东]]在[[南宁会议|南宁会议]]和最高国务会议上讲话：“北京、[[开封|开封]]的房子，我看了就不舒服”，“[[南京|南京]]、[[济南|济南]]、[[长沙|长沙]]的城墙拆了很好，北京、[[开封|开封]]的旧房子最好全部变成新房子。”“古董不可不好，也不可太好。北京拆[[牌楼|牌楼]]，城门打洞，也哭鼻子。这是[[政治问题|政治问题]]。” 同年4月14日，[[周恩来|周恩来]]致信[[中共中央|中共中央]]并传达[[国务院常务会议|国务院常务会议]]精神，称“根据[[毛主席|毛主席]]的指示，今后几年内应当彻底改变北京市的都市面貌”。此后，北京市迅速制定了一个十年左右完成旧城改建的计划，故宫改建计划随即提出。

== 大跃进 ==
[[大跃进|大跃进]]时期，由于全国大炼钢铁，占用大量土地。为获得土地与炼钢材料，一些古建筑纷纷被拆除。由于故宫在政治上的特殊地位，其改建也被正式提上日程。

1958年9月，《北京市总体规划说明（草稿）》有这样的文字：“故宫要着手改建”“把[[天安门广场|天安门广场]]、故宫、[[中山公园|中山公园]]、文化宫、[[景山|景山]]、[[北海公园|北海]]、[[什刹海|什刹海]]、[[积水潭|积水潭]]、前三门护城河等地组织起来，拆除部分房屋，扩大绿地面积，使成为市中心的一个大花园，在节日作为百万群众尽情欢乐的地方。”同时，北京市城市建设委员会提认为，可保护“天安门以及故宫里的一些建筑物”，“故宫要改建成一个群众性的文体、休憩场所”。 

不久之后的1958年12月15日，故宫博物院向北京市文化局递交了《清除糟粕建筑物计划和1959年第一批应拆除建筑物的报告》，其中称对故宫内各处残破坍塌及妨碍交通的道路、妨碍下水道之小房及门座等建筑，需即行拆除。北京市文化局迅即同意此报告，并提出要求：“(1)能暂时利用者，可不拆除；(2)对过去宫廷仆役(太监、宫女等)所住的房屋及值班房等，选择几处有典型性的加以保留，并标出文字说明，以便和帝王奢侈生活进行对比，向观众进行阶级教育；(3)拆除室内的墙时，应注意建筑物的安全；(4)能用材料，拆除时应注意保护，拆除后应妥为保存和利用；(5)拆除的建筑物应照相留影。”结果，绛雪轩罩棚、养性斋罩棚、集卉亭、鹿囿、建福门、惠风亭等一批“糟粕”建筑，于一年之内被拆除。 

1964年，时任[[国务院|国务院]]副总理的[[李富春|李富春]]提出《关于北京城市建设工作的报告》，在报告中将故宫改造计划写为“二十世纪六十年代初的改造方案，是“在故宫内部建设一条东西向的马路，并将[[文华殿|文华殿]]、[[武英殿|武英殿]]改造成娱乐场所”。而知情者介绍，在故宫内开辟东西向道路，也仅为当时众故宫改造方案之一。为了给故宫改造计划宣传造势，时任北京副市长的[[明史|明史]]专家[[吴晗|吴晗]]写道：“今天的北京城并不是历史上各个王朝北京城的原样，不但位置不同，规模、设计、建置也不相同。同样，作为政治中心的中心，统治者在那里发号施令的宫殿，也是如此，不但[[辽代|辽]]、[[金代|金时代]]的[[宫殿|宫殿]]没有了，就是[[元朝|元朝]]的也被拆除了。”“现在保留下来的清朝的宫殿，不但不是[[明朝|明朝]]的原来建筑，而且也不完全是[[清朝|清朝]]的建筑。当然，作为一个古代建筑艺术品，应否保留以及如何保留，是一个可以研究的问题。”“北京城的历史发展告诉我们，无论是城市建置、政治中心、街道布局、房屋高低等等，都不是不可改变的。相反的，结论是必须改变。”

== [[文化大革命|文化大革命]] ==

“文革”时期的“故宫整改方案”则是“在[[太和殿|太和殿]]前竖立两座大标语牌，一东一西，高度超过古十八米高的太和殿，用它压倒‘王气’；太和殿宝座要搬倒，加封条；在宝座台上塑持枪农民的像，枪口对准被推翻的皇帝。把过去供皇帝到太和殿主持大典之前临时休憩之处的中和殿，改建为‘人民休息室’，把一切代表封建意识的宫殿、门额，全部拆掉”等。然而这些方案中有的项目竟实现了（如“人民休息室”计划），但也有的因无暇顾及才得以幸免。 

1966年，“文化大革命”爆发，故宫改建计划被列为中共北京市委要给[[刘少奇|刘少奇]]盖宫殿的“罪证”。 故宫改建计划也因此无疾而终。然而，紫禁城因此并未获得安宁。文化大革命对中国所有的古建筑与文物均带来浩劫，故宫也未能幸免。
 
1966年5月23日，泥塑“收租院”展览在[[神武门|神武门]]城楼开幕，后移至故宫[[奉先殿|奉先殿]]继续展出。为此，1966年6月至7月，奉先殿工字型大殿被改为长方型。毛泽东像被挂于[[奉先殿|奉先殿]]内，殿前清代祭祖所用焚帛炉，被认为与展览内容不符而予拆除。 

1966年8月3日，故宫[[城隍庙|城隍庙]]内藏泥塑神像十一座和泥塑马一对被毁；8月16日，除“收租院”展览外，故宫其余各处停止开放，实行闭馆。 

故宫博物院大理石门匾，被纸盖住，墨笔大书“血泪宫”三字；神武门外砖墙上，“火烧紫禁城！！！”“砸烂故宫！”的大字报被贴出。 

而新的故宫“整改方案”随即出台。顺贞门、天一门、文华殿、乾隆花园内的门额被摘，中和殿宝座被拆。所幸的是，“整改方案”的其他项目未实施行，“[[批判资产阶级反动路线|批判资产阶级反动路线]]”大潮在1967年5月间入侵故宫博物院。带领全院职工批判“黑线”、贯彻“红线”的军宣队，转眼间成了被批判揪斗的对象，10月不得不撤离故宫，“整改方案”寿终正寝。 

1967年5月26日，[[周恩来|周恩来]]派[[北京卫戍区|北京卫戍区]]一营进驻故宫博物院，实行军事保护，使故宫宫殿及文物免遭直接破坏。 

1970年，故宫[[钦安殿|钦安殿]]前抱厦五间被拆除（2005年修复）。 

1971年7月，故宫博物院恢复开放。

1973年3月27日，受[[洛阳|洛阳]][[白马寺|白马寺]]接待[[西哈努克亲王|西哈努克亲王]]的需要，故宫[[慈宁宫|慈宁宫]]大佛堂全部文物被调拨至洛阳白马寺，至今未归还。搬运文物时，慈宁宫大佛堂建筑彩画遭到破坏。

== 高楼对故宫的包围 ==
1972年，设计高度逾百米的[[北京饭店|北京饭店]]东楼在故宫东侧兴建，施工中发现其构成对[[中南海|中南海]]的窥视，东楼高度被减至87.6米；1974年2月10日，经[[国务院|国务院]]批示，五座遮挡性楼房在故宫[[西华门|西华门]]内南北两侧开工，次年11月22日竣工。而在此前，有人建议将故宫[[午门|午门]]楼提高以起遮挡作用，被周恩来否定。 

尽管周恩来提出，北京旧城的建筑高度不能超过45米。然而，随着北京城市现代化的进行，故宫周边的景象正在发生巨大的改变。[[东方广场|东方广场]]的建设饱受争议；[[中国国家大剧院|国家大剧院]]破坏了长安街上的“帽子风景线”，由于其与周围建筑的风格不能融合，而被一些人戏称为“水母”、“大水泡”、“坟头”。 

而近年来在北京旧城内一系列的拆除[[胡同|胡同]]、大建高楼等行为一方面将北京仅存的古都风貌严重破坏，也威胁到故宫、[[北海公园|北海]]等古迹的生存空间。从空中或[[景山|景山]]来看北京，故宫等建筑已经被高楼大厦所包围，故宫附近的缓冲地带也日益被蚕食。

不过为保护[[世界文化遗产|世界文化遗产]][[故宫|故宫]]的周边环境，以达到[[联合国教科文组织|联合国教科文组织]]保护世界文化和自然遗产政府间委员会对世界遗产周边缓冲区的要求，2004年[[北京市规划委员会|北京市规划委员会]]、[[北京市文物局|北京市文物局]]对[[故宫|故宫]]保护缓冲区作出[[规划|规划]]草案，并经过市民评议确定了以1984年市政府批准（1987年修正）的旧[[皇城|皇城]]及北城区（含故宫）保护范围建设控制地带为基础，并结合[[北京历史文化名城保护规划|北京历史文化名城保护规划]]、[[北京皇城保护规划|北京皇城保护规划]]要求的故宫保护缓冲区。缓冲区的范围北至北二环中路，南至[[前三门大街|前三门大街]]东西至东西[[皇城根|皇城根]]，占地面积780[[公顷|公顷]]；建设控制区占地面积597[[公顷|公顷]]；故宫占地面积86[[公顷|公顷]]；共计1463[[公顷|公顷]]。

该缓冲区方案将于2005年2月经批准报[[世界遗产中心|世界遗产中心]]备案后正式生效。
