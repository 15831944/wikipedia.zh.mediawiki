'''故乡的风'''（{{Lang-ja|ふるさとの風，ふるさとのかぜ}}；{{Lang-en|Furusato No Kaze}}）是[[日本|日本]]的一個[[廣播電台|廣播電台]]，由日本政府的绑架问题对策本部向[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]进行[[短波|短波]]广播。

== 概要 ==
2007年7月开始向-{zh-hans:朝鲜;zh-hant:北韓}-广播，广播语言有[[日语|日语]]和[[朝鲜语|-{zh-hans:朝鲜;zh-hant:韓}-语]]<ref>[http://www.zaobao.com/special/newspapers/2007/07/bbc070710.html 日本向朝鲜绑架的日人进行短波广播 ] 联合早报网 2007-07-10</ref>。

== 广播内容 ==
主要广播有关失踪的日本人及失踪者家属的信息。

== 历史 ==
* 2007年7月9日 - 开始广播。
* 2007年7月16日 - -{zh-hans:朝鲜;zh-hant:北韓}-开始干扰该电台。
* 2008年3月12日 - 增加频率，分别是日语节目在23:30-00:00所用的11995kHz、-{zh-hans:朝鲜;zh-hant:韓}-语节目在00:00-00:30所用的13725kHz。
* 2008年3月30日 - 日语节目改用11775kHz、朝鲜语节目改用11690kHz。
* 2008年7月14日 - 23:33分开始隔夜替换式使用9585kHz。
* 2008年11月 - 23:30-00:00、01:00-01:30播出的日语节目增加使用9880kHz和9780kHz，00:00-00:30、02:00-02:30播出的-{zh-hans:朝鲜;zh-hant:韓}-语节目增加使用9780kHz和9820kHz。
* 2008年11月17日 - 9585kHz停止作为隔夜替换式使用频率，日语节目在22:33-22:57开始改为9965kHz常用，00:30-01:00播出的-{zh-hans:朝鲜;zh-hant:韓}-语节目开始使用9965kHz。

== 发射台 ==
故乡的风在[[台湾|台湾]][[新北|新北]][[淡水|淡水]]使用100千瓦和250千瓦的短波发射机以及在[[帕劳|帕劳]]使用100千瓦的短波发射机对朝鲜进行广播。由于目前朝鲜经常对这个电台进行干扰，所以在朝鲜周边国家很难清晰收到该电台信号。
