'''故宫出版社'''，位于[[北京市|北京市]][[东城区_(北京市)|东城区]][[景山前街|景山前街]]4号，是[[故宫博物院|故宫博物院]]直属独立企业法人单位。由[[中华人民共和国文化部|中华人民共和国文化部]]主管，故宫博物院主办<ref name=whb/>。

==简介==
故宫出版社成立于1983年，当时名为“'''紫禁城出版社'''”。2010年完成转企改制，成为全民所有制企业。2011年9月，经[[中华人民共和国文化部|中华人民共和国文化部]]、[[中华人民共和国国家工商行政管理总局|中华人民共和国国家工商行政管理总局]]、[[中华人民共和国国家新闻出版广电总局|中华人民共和国国家新闻出版广电总局]]批复，更名为“'''故宫出版社'''”。该出版社在故宫博物院内，是故宫博物院下属的专业出版社，主要出版与故宫博物院业务相关的文物、历史、古建筑、博物馆、艺术等方面的出版物<ref name=whb>{{Cite web|url = http://www.mcprc.gov.cn/gywhb/jgsz/zgxwcbdw/zgxwcbdw_cbs/201509/t20150925_458105.html|title = 故宫出版社|author = |date = 2015-09-25|publisher = 中华人民共和国文化部|language = |accessdate = |quote = |archive-url = https://web.archive.org/web/20170212192348/http://www.mcprc.gov.cn/gywhb/jgsz/zgxwcbdw/zgxwcbdw_cbs/201509/t20150925_458105.html|archive-date = 2017-02-12|dead-url = yes}}</ref><ref name=jj>{{Cite web|url =http://www.culturefc.cn/index/static.jhtml?act=about&type=2|title =故宫出版社简介|author =|date =|publisher =故宫文化网|language =|accessdate =2017-05-16|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170616052531/http://www.culturefc.cn/index/static.jhtml?act=about&type=2|archivedate =2017-06-16}}</ref><ref>{{Cite web |url =http://money.163.com/15/0624/02/ASRF07OS00253B0H.html  |title =  工行900万元贷款助力故宫出版社改革转型|author =  |date = 2015-06-24 |publisher =  网易|language =  |accessdate =  |quote =  }}</ref>。

故宫出版社定期出版《故宫博物院院刊》、《紫禁城》两种刊物<ref name=jj/>。故宫出版社还设有故宫书店、北京故宫文化传播有限公司，以及故宫文化网<ref>{{Cite web|url =http://www.culturefc.cn/index/static.jhtml?act=about&type=7|title =联系方式|author =|date =|publisher =故宫文化网|language =|accessdate =2017-05-16|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170616051318/http://www.culturefc.cn/index/static.jhtml?act=about&type=7|archivedate =2017-06-16}}</ref>。
