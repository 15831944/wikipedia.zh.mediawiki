'''倭寇'''，是[[东亚|东亚]]地區对东亚日本人的的贬称。一般是指13至16世紀期間活躍於[[朝鮮半島|朝鮮半島]]及[[中國|中國]]沿岸的海盜，这些海盗一般為[[日本人|日本人]]，故此又叫日本海盗。歷史上倭寇来源众多，成分复杂。元朝早期以日本人為主，明朝中晚期則以沿海的南方漢人爲主體，《[[明史|明史]]日本傳》有云：「大抵真倭十之三，从倭者十之七」。在倭寇最強盛之時，他們的活動範圍遠至東亞各地、甚至是內陸地區。

前期的倭寇，以日本人为主体，被称为「真倭」；[[戚繼光|戚繼光]]《[[紀效新書|紀效新書]]》倭寇的盔甲、武器、劍術戰法描述的非常清楚。後期的倭寇，出現了以[[中国人|中国人]]为主体、其他東亞族裔為輔，被称为「假倭」，明史《[https://zh.wikisource.org/zh-hant/籌海圖編_(四庫全書本) 籌海圖編]》中提道：「今之海寇，動計數萬，皆託言倭奴。而其實出於日本不過數千，其餘則皆中國赤子無賴者入而附之耳。」朝鮮《[[朝鮮王朝實錄|世宗實錄]]》（二十八年十月壬戌）亦記載「然其間倭人不過一二而本國民假著倭服成黨作亂」。<ref>[http://data.book.hexun.com.tw/chapter-5005-1-3.shtml 海盜簡史]，作者：李子遲，出版社：重慶出版社，和訊讀書</ref>

== 引申意义 ==
倭寇一詞的由來，是從[[高句麗|高句麗]][[廣開土王|廣開土王]]碑（西元414年）碑文上所記載「倭，寇○○（地名）」而來，在此「寇」當動詞，即「[[倭人|倭人]]侵略（某地）」之意，而之後「倭寇」二字作為名詞獨立，用以稱呼來自[[日本|日本]]的侵略者。另一方面，在16世纪[[壬辰倭亂|壬辰倭亂]]以及20世纪[[第二次中日战争|第二次中日战争]]中，兩國的人民都稱呼日本侵略軍為“倭寇”或“日寇”。而在往後的排日風潮中，倭寇和日寇更成為敵視日本人的用語。

== 前後两期倭寇 ==
[[File:Wokou.jpg|thumb]]

雖然同被稱為倭寇，但兩者的成份分別很大。根據年代大致可以分為「前期倭寇」及「後期倭寇」。後期倭寇基本上是中日混编队，起因为[[明朝|明朝]]的[[闭关锁国|闭关锁国]]（海禁）政策。

=== 前期倭寇（真倭） ===
前期倭寇是以日本海盜為主，目的是为了报复蒙朝联军的侵略和女真海盗的掳掠。在[[元日战争|元日战争]]之前，高丽史已經有“倭寇金州”的记载。

元日戰爭時，倭寇主要攻擊範圍為當時臣服於[[元朝|元朝]]的[[高麗_(918年－1392年)|高麗]]。由於先前臣服於元軍的高麗軍，在[[忽必烈|忽必烈]][[元日战争|兩次進攻日本的軍事行動]]中，侵略日本，因此前期倭寇對於朝鮮沿海進行含有報復式的海盜行為。[[朝鲜历史|朝鮮史]]取[[对马岛|对马岛]]、[[壹岐島|壹岐島]]、[[平戶島|平戶島]]三地之名，稱為「三島倭寇」（以[[平戶藩|平戶藩]][[松浦家|松浦家]]為核心）。[[元史|元史]]和[[新元史|新元史]]中都有倭寇骚扰元朝[[杭州|杭州]]的记载。

後來日本進入[[南北朝时代_(日本)|南北朝]]的分裂狀態，倭寇的活動也由於政府管制力減弱而加劇。由於前期倭寇對於日本與明朝、高麗貿易造成破壞，初成立的明朝政府對日本南朝發出討伐倭寇的要求，隨後敕封討寇有功的南朝征西將軍宮[[懷良親王|懷良親王]]「日本國王」（[[後醍醐天皇|後醍醐天皇]]之子）的稱號。之後北朝（[[室町幕府|室町幕府]]擁立）統一南朝後，幕府將軍[[足利義滿|足利義滿]]再度進行討伐，隨後受封為新的「日本國王」。

[[李氏朝鲜|朝鮮王朝]]的開國君主[[李成桂|李成桂]]因為討伐倭寇有功得到很大的声望和勢力，後來進行政變取代[[高麗_(918年－1392年)|高麗王朝]]建立朝鲜王朝。之後隨著明朝與日本間[[明日貿易|勘合貿易]]的盛行，以及對馬島與朝鮮之間的貿易開放，前期倭寇也逐漸式微。

=== 後期倭寇（假倭） ===
後期倭寇基本上是中日混编队，因明朝太祖皇帝[[朱元璋|朱元璋]]的[[海禁|海禁政策，]]從而沿海地區之民衆不得從事海上經商，故而大部分的晚期倭寇都是些走私行商不成，最後被朝廷逼成倭寇的南方漢族商人。故此，明晚後期的倭寇亦可分成兩方面：

* 一方面以日本人為主體的倭寇組織，是爲真倭，戚繼光在《[[紀效新書|紀效新書]]》將倭寇的裝備、武器、劍術戰法描述的非常清楚，和官府對決的戰鬥確實以精湛的日本劍術，這些日本武術的海盜甚至擁有一攻五的好武藝。由於嘉靖二年發生在[[浙江|浙江]]因争貢事件的[[寧波之亂|寧波之亂]]和日本因應仁之亂後發生的一連串動盪，使得海盜的活動再次活躍起來，為此日本頒佈倭寇取締令作為限制。
* 另一方面以中國東南沿海發展起來的海盜，是為假倭。自[[明成祖|明成祖]]發動[[靖難之變|內戰]]，遷都[[北京|北京]]，以及[[嘉靖|嘉靖]]後的圍堵的政策，形成一股在中國東南沿海進行報復性侵擾勢力。由於這些南方人體型上也比北方人矮小，比北方人更難與日本人區別，所以沿用「倭寇」這名詞來稱呼由日本人與南方中國人所組成的海盜集團。後期倭寇集團的海員以混编組成，有不少是南方[[中國人|中國人]]、[[朝鮮人|朝鲜人]]以及其他東亞民族，其中更不乏有鎮守海疆的軍官參與<ref>[http://mingching.sinica.edu.tw/Academic_Detail/254 宋怡明教授演講「明代東南沿海地區的衛所和軍戶：國家制度與日常政治」紀要]</ref><ref>[http://mingching.sinica.edu.tw/Academic_Detail/415 宋怡明教授演講「國家怎麼玩：以明代東南沿海的軍戶與衛所為例」紀要]</ref>，並產生日本人只佔少數的海盜集團{{NoteTag|見本頁「[[#引申意义|引申意義]]」一段。}}。

==== 高丽贱民参与说====
[[東京大学|東京大学]]教授田中健夫提出1370年到1390年初倭寇襲擊的大幅增加是因为新加入的高麗[[贱民|贱民]]群体<ref name=tanaka>田中 (1982)</ref>。田中还提出还有纯由朝鲜人组成的倭寇集团<ref name=tana>田中 (1987)</ref>。另外，他还提出李氏朝鲜当时借着倭寇的外患为由整顿内部，确立了国家体制<ref name=ta>田中 (1997)　田中说，当时日本有南北朝内乱，朝鲜半岛也处于相同的混乱中，所以日本人和高麗・朝鮮人賎民的国家意识稀薄，才会出现这种状况。</ref>。

从[[明太祖|明太祖]][[洪武|洪武]]年间开始，日本倭寇即不斷寇邊，明朝即执行[[海禁|海禁政策]]，禁止民间对海外通商，但是官方的朝贡贸易（也叫[[明日貿易|勘合貿易]]）依旧合法。明成祖派[[鄭和下西洋|鄭和下西洋]]亦刺激了这种朝贡贸易的发展。明政府是期望用朝贡贸易满足外国人的通商需求，从而替代民间的私人贸易，以达到海禁之目的。然而，中国东南沿海一带的民众长期依靠海外贸易谋生，海外商人亦对中国产品有巨大需求，故私人贸易不可能禁绝，而且转为走私。明代中叶，随着中国东南沿海地区商品经济快速发展，对外贸易更加频繁。但後來到了[[嘉靖|嘉靖]]二年（西元1523年）爆發了[[寧波之亂|寧波之亂]]，加上稍早時，[[葡萄牙|葡萄牙]]人入侵發生了[[屯門海戰|屯門之戰]]與[[西草湾之战|西草湾之战]]，此後沿海治安多次陷入危機，明朝遂宣佈中斷一切貿易，期望以圍堵的政策來減輕倭寇的威脅。

後來的發展上，明人[[汪直_(海商)|汪直]]扮演了很重要的角色，汪直曾上書朝廷請求開海禁，被拒絕後，把基地設在日本的[[平戶藩|平戶藩]]，以反對明朝海禁政策的南方中國人為基礎，與日本人、[[葡萄牙|佛朗機]]人合作進行走私，在後期倭寇中頗為知名。[[胡宗宪|胡宗宪]]、[[戚继光|戚继光]]與[[俞大猷|俞大猷]]等人都曾參與討伐倭寇的軍事行動。

15世紀半至16世紀，其活動舞臺為中國沿岸、東南亞方面，成員以華人及朝鮮人為主，被稱為「假倭」。此時期倭寇發生的主要原因是明代中國的生產力增加，但卻施行海禁政策。史書所見的最後倭寇，在1624年7月侵犯[[福建|福建]]沿海。「推其禍始，乃由[[閩|閩]]、[[浙|浙]]沿海奸民與倭為市；而閩浙大姓沒其利，陰為主持，牽連以成俗。當時撫臣[[朱纨|朱纨]]欲絕禍本，嚴海禁；大家不利，連為蜚語中之，而紈驚死矣。紈死而海禁益弛，於是[[宋素卿|宋素卿]]、[[汪直_(海商)|王直]]、[[陳東_(明朝)|陳東]]、[[徐海_(明朝)|徐海]]、曾一本、許恩之流爭挾倭為難。自淮揚以南至廣海萬餘里，無地不被其殘滅，而閩禍始慘矣」。更有甚者，為部分沿海明民苦於生計鋌而走險，出海走私[[南洋|南洋]]一帶，地方官府為掩飾社會問題，捕獲後一概以「倭寇」之名處刑。

== 倭寇的影響 ==
[[明朝|明朝]]與[[高麗|高麗]]、[[朝鮮王朝|朝鮮王朝]]都曾經對[[室町幕府|室町幕府]]要求取締海盜行為。明朝甚至於與日本發展了[[勘合貿易|勘合貿易]]，日本[[足利幕府|足利幕府]]對明朝稱臣受封為[[日本國王|日本國王]]。而朝鮮王朝以倭寇根據地名義[[应永外寇|進攻對馬島]]也是受到倭寇的影響。

== 八幡船 ==
日本[[室町時代|室町時代]]到[[江戶時代|江戶時代]]的海盜船通稱為「八幡船」，這是由於倭寇多半信奉[[武神|武神]][[八幡大菩薩|八幡大菩薩]]，並繪製旗幟，上書「南無八幡大菩薩」七字，代表[[神明|神明]]庇佑的[[香火|香火]]與[[船舶|船舶]]的號記。

== 活動地區 ==
倭寇的根據地包括日本的[[對馬島|對馬島]]、[[壹岐|壹岐]]、[[平戶島|平戶島]]、[[五島列島|五島列島]]，以及中國、朝鮮半島的沿海島嶼。[[荷蘭人|荷蘭人]]未到[[臺灣|臺灣]]之前，臺灣也是倭寇的活動據點。
