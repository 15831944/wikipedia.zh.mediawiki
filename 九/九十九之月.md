{{日本作品|九十九之月|ツクモノツキ}}是[[Sugar_Pot|Sugar Pot]]于2012年2月24日發售的[[戀愛冒險|戀愛冒險]][[日本成人遊戲|成人遊戲]]。<ref>{{cite web|author=ErogameScape -エロゲー批評空間-|script-title=ja:ツクモノツキ|url=http://erogamescape.ddo.jp/~ap2/ero/toukei_kaiseki/game.php?game=16016|accessdate=2016年9月18日|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20130902221150/http://erogamescape.ddo.jp/~ap2/ero/toukei_kaiseki/game.php?game=16016|archivedate=2013年9月2日}}</ref>

== 概要 ==
本作品是以日本[[九州|九州]][[長崎站|长崎车站]]为舞台的恋爱故事。预约初回版的玩家可以得到一张收录了主题曲的CD[[单曲|单曲]]。

== 登場人物 ==
; {{nihongo|大羽慶太|大羽慶太（おおば けいた）||声：无}}
:故事中的男主角，為玩家在游戏中扮演的角色。幼时父母双亡，在各个亲戚之间奔波流离。因为一场意外事故而失去了小时候的记忆。<ref name="keita">{{cite web|author=Sugar Pot|script-title=ja:キャラ紹介|url=http://www.sugarpot-hp.com/soft/tukumonotuki/t02_5.html|publisher=Sugar Pot|accessdate=2013-05-04|language=ja}}</ref>现在和妹妹摩耶租住一间房子，除了亲戚的资助以外，在家庭餐厅CuCu打工维持生计。
; {{nihongo|大羽摩耶|大羽摩耶（おおば まや）||声：{{lang|ja|[[宮沢ゆあな|宮沢ゆあな]]}}}}
:庆太的妹妹，擅长家务和料理，性格有点胆小怕生，经常会觉得有点自卑，让庆太很头疼。<ref>{{cite web|author=Sugar Pot|script-title=ja:キャラ紹介|url=http://www.sugarpot-hp.com/soft/tukumonotuki/t02_1.html|publisher=Sugar Pot|accessdate=2013-05-04|language=ja}}</ref>总是惯着津云。
; {{nihongo|津云|ツクモ（つくも）||声：[[青叶林檎|青叶林檎]]}}
:庆太在神社后山上遇到的神秘少女，称呼庆太为“哥哥”，后来跟随庆太和摩耶一同生活。虽然年龄只比庆太小一岁，但内心却还是个小孩子。<ref>{{cite web|author=Sugar Pot|script-title=ja:キャラ紹介|url=http://www.sugarpot-hp.com/soft/tukumonotuki/t02_2.html|publisher=Sugar Pot|accessdate=2013-05-04|language=ja}}</ref>和雏子的关系很好。名字“津云”在日语中与“九十九”同音。
; {{nihongo|北条茜|北条茜（ほうじょう あかね）||声：[[上田朱音|上田朱音]]}}
:庆太的青梅竹马，是学校内剑道部主将，总是挥舞着竹刀到处走动。<ref>{{cite web|author=Sugar Pot|script-title=ja:キャラ紹介|url=http://www.sugarpot-hp.com/soft/tukumonotuki/t02_3.html|publisher=Sugar Pot|accessdate=2013-05-04|language=ja}}</ref>性格开朗活泼到根本不像是一个女孩子，经常和庆太拌嘴，同时也是庆太的同班同学。
; {{nihongo|鷲見雪菜|鷲見雪菜（わしみ ゆきな）||声：{{lang|ja|[[藤森ゆき奈|藤森ゆき奈]]}}}}
:庆太和摩耶在神社遇见的巫女。行为举止端庄典雅，总是一身巫女的装扮，也引得不少人的注目。似乎拥有着某种神奇的能力。<ref>{{cite web|author=Sugar Pot|script-title=ja:キャラ紹介|url=http://www.sugarpot-hp.com/soft/tukumonotuki/t02_4.html|publisher=Sugar Pot|accessdate=2013-05-04|language=ja}}</ref>
; {{nihongo|塚本麻香|塚本麻香（つかもと あさか）||声：{{lang|ja|[[早瀬やよい|早瀬やよい]]}}}}
:庆太的表姐，同时也是庆太打工快餐店CuCu的店长。<ref name="keita"/>拥有着如同春风般的微笑，不过有时候会突发性的天然呆。深知庆太和摩耶的经历，很是关心他们两人。
; {{nihongo|房东小姐|大家さん（おおやさん）||声：[[鈴音華月|鈴音華月]]}}
:姓氏是赤木。是庆太和摩耶租住的公寓——赤木庄的房东，所以都称呼为房东小姐。<ref name="keita"/>有个女儿叫雏子，对她很是关爱。
; {{nihongo|雛子|雛子（ひなこ）||声：{{lang|ja|[[雪村とあ|雪村とあ]]}}}}
:房东小姐的女儿，性格比较内向，总是躲在房东小姐的身后，似乎有点害怕庆太。和津云的关系很好。<ref name="keita"/>
