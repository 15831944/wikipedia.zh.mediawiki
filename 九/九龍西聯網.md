'''九龍西聯網'''（全稱'''九龍西醫院聯網'''，{{lang-en|'''Kowloon West Cluster'''}}）是[[香港|香港]][[醫院管理局|醫院管理局]]的地區性醫院聯網，負責服務[[深水埗區|深水埗區]]、[[葵青區|葵青區]]、[[荃灣區|荃灣區]]及[[離島區|離島區]]（只包括[[北大嶼山|北大嶼山]]）約127萬人口<ref>統計自2011年香港人口普查數據</ref>；現任總監為謝文華醫生。<ref>{{Cite news|url=http://www.info.gov.hk/gia/general/201701/19/P2017011900595.htm|title=醫管局公布高層管理人員任命（附圖）|date=2017-01-19|accessdate=2017-01-20|work=[[香港特別行政區政府|香港特別行政區政府]]新聞公報}}</ref>

== 醫院 ==
九龍西聯網轄下由5間醫院組成，全部設有急症服務。

*[[瑪嘉烈醫院|瑪嘉烈醫院]]：大型區域性急症全科醫院，設有聯網的創傷和腫瘤科中心、醫院管理局傳染病中心及毒理化驗室。
*[[明愛醫院|明愛醫院]]：地區性全科醫院，提供急症、延續及寧養紓緩服務以及兒童發展障礙服務。
*[[葵涌醫院|葵涌醫院]]：精神科醫院，提供精神科急症及外展服務。
*[[仁濟醫院|仁濟醫院]]：全科醫院，提供急症及復康服務，設有中醫診所。
*[[北大嶼山醫院|北大嶼山醫院]]：提供急症室、社區健康中心、專科門診及社區外展等服務。

== 歷史 ==
針對人手短缺及求過於供等問題，九龍西聯網於2012年計劃於未來增加醫護人員及後勤的[[人力資源|人力資源]]，包括於[[明愛醫院護士學校|明愛醫院護士學校]]增加100個學額、分別在明愛醫院及瑪嘉烈醫院增加2,060個及1,250個磁力共振及電腦斷層掃描名額、在仁濟醫院設立能夠提供3,000個門診及50個手術名額的耳鼻喉中心及於廣華醫院增加設備5張初生嬰兒深切治療病牀等等<ref>[http://hk.news.yahoo.com/九西聯網醫護流失改善-212155333.html 九西聯網醫護流失改善] {{webarchive|url=https://archive.is/20130105150926/http://hk.news.yahoo.com/%E4%B9%9D%E8%A5%BF%E8%81%AF%E7%B6%B2%E9%86%AB%E8%AD%B7%E6%B5%81%E5%A4%B1%E6%94%B9%E5%96%84-212155333.html |date=2013-01-05 }} 《明報》 2012年11月1日</ref>。

於2013年至2014年年度，九龍西聯網獲分配約103億港元財政資源，佔[[香港政府|香港政府]]配給醫院管理局的撥款近22%。鑑於一直有不少意見反映[[醫院管理局|醫院管理局]]的[[人事管理|人事管理]]及資源運用方面分配不均，及被質疑資源有否按照地區人口或者人口年齡組別而作出分配等等。醫院管理局檢討督導委員會於2014年就此作出討論，預計於2015年3月月底舉辦會議作出最後審議，及於同年年中完成檢討報告。檢討當中包括[[九龍中聯網|九龍中聯網]]、[[九龍東聯網|九龍東聯網]]及九龍西聯網之間存在的一些問題，冀望借助[[啟德醫院|啟德醫院]]首期於2021年啟用的契機，將其劃入九龍中聯網，並且為九龍中聯網、九龍東聯網及九龍西聯網重新劃線，預計屆時被納入九龍中聯網的醫院和醫療機構將會由6間增加至10間，組成一個大型聯網，提供前所未有的大規模服務，以取代九龍西聯網當前的領導地位<ref>[http://orientaldaily.on.cc/cnt/news/20150201/00176_040.html 伊院服務擬遷往啟德] 《東方日報》 2015年2月1日</ref><ref>[http://orientaldaily.on.cc/cnt/news/20150201/00176_041.html 涵10機構 九龍中勢冠七大聯網] 《東方日報》 2015年2月1日</ref>。此舉，亦能夠理順服務及令到資源分配更為準確<ref>[http://orientaldaily.on.cc/cnt/news/20150202/00176_047.html 九龍三聯網資源將重新分配] 《東方日報》 2015年2月2日</ref><ref>[http://hk.apple.nextmedia.com/news/art/20150131/19023535 聯網擬重劃　倡廣華入九龍中] 《蘋果日報》 2015年1月31日</ref>。

2016年12月1日，原屬於九龍西聯網的黃大仙及[[旺角|旺角]]區正式劃入九龍中聯網，包括廣華醫院、東華三院黃大仙醫院和聖母醫院，以及相關地區的服務單位。聯網行政重組的正式實行預期能夠加強服務整合，以減少病人跨網求診的情況。而病人臨床醫療服務的重組，則於2017年4月1日後，在確保相關系統及支援服務已完全就緒後才會分階段進行，以減低重組可能對病人造成的任何不便。<ref>{{cite news|title=九龍中、西兩聯網重組行政安排|url=http://www.info.gov.hk/gia/general/201611/30/P2016113000696.htm|agency=香港政府新聞公報|date=2016-11-30}}</ref>
