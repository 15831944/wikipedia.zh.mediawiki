'''九龍巴士269C線'''是[[香港|香港]][[九龍巴士|九龍巴士]]營運的一條巴士路線，來往[[天水圍市中心|天水圍市中心]]及[[觀塘碼頭|觀塘碼頭]]，為[[元朗區|元朗區]]的[[天水圍新市鎮|天水圍新市鎮]]和[[九龍東|九龍東]]的[[黃大仙區|黃大仙區]]及[[觀塘區|觀塘區]]居民提供服務。

'''九龍巴士69C線'''是269C的繁忙時間特別路線，來往[[天恩邨|天恩邨]]及觀塘碼頭，為天水圍北居民提供往來[[黃大仙區|黃大仙區]]及[[觀塘區|觀塘區]]服務<ref>{{cite news |language = 中文 |url = http://www.kmb.hk/tc/news/press/archives/news201509082275.html |title = 九巴69C線方便往返天水圍(北)至觀塘碼頭 |date = 2015年9月8日 |access-date = 2016年10月23日 |archive-url = https://web.archive.org/web/20161023205512/http://www.kmb.hk/tc/news/press/archives/news201509082275.html |archive-date = 2016年10月23日 |dead-url = no }}</ref>。

'''九龍巴士269S線'''是269C的繁忙時間特快路線，來往天水圍市中心及觀塘碼頭，並途經九龍灣商貿區。<ref>{{cite news |language = 中文 |url = http://www.kmb.hk/tc/news/press/archives/news201610132408.html |title = 九巴開辦269S全新早晨專線 |date = 2016年10月13日 |access-date = 2016年10月23日 |archive-url = https://web.archive.org/web/20161016235710/http://www.kmb.hk/tc/news/press/archives/news201610132408.html |archive-date = 2016年10月16日 |dead-url = no }}</ref>

==歷史==
本路線於1994年4月26日投入服務，祇在平日星期一至六早上提供一個單向班次，由[[元朗區|元朗區]]的[[天瑞邨|天瑞邨]]前往[[觀塘區|觀塘區]]的[[藍田站|藍田地鐵站]]，途經[[元朗大馬路|元朗大馬路]]、[[新界環迴公路|新界環迴公路]]（今[[新田公路|新田公路]]和[[粉嶺公路|粉嶺公路]]）、[[吐露港公路|吐露港公路]]及[[大老山隧道|大老山隧道]]，使用空調巴士行走，全程收費$15。同年10月17日，增設平日星期一至五下午回程服務，途經[[新蒲崗|新蒲崗]]及[[鑽石山站|鑽石山地鐵站]]。

1998年5月26日，269C線擴展至每天全日服務，來往[[天水圍市中心|天水圍市中心]]及[[觀塘碼頭|觀塘碼頭]]，取道[[三號幹線|三號幹線]]（今[[青朗公路|青朗公路]]，包括[[大欖隧道|大欖隧道]]和[[汀九橋|汀九橋]]、[[長青公路|長青公路]]、[[長青隧道|長青隧道]]、[[青葵公路|青葵公路]]）、[[呈祥道|呈祥道]]及[[龍翔道|龍翔道]]，而不再途經新田公路、粉嶺公路、吐露港公路及大老山隧道，行車時間大為縮短。

2000年7月17日，平日早上增設特別班次，由[[朗屏邨|朗屏邨]]開出，同時由天水圍市中心開出的班次不會途經[[元朗市|元朗市]]。

2004年2月24日，九巴重組269C線，分拆出3條常規路線，仍維持全線空調巴士服務。
*69C（第一代）（天水圍市中心↔觀塘碼頭）祇在每天非繁忙時間提供服務，維持重組前的269C線服務水平；
*268C（朗屏站↔觀塘碼頭）祇在每天繁忙時間提供服務，取代過往269C線早上由朗屏邨開出，及每天在元朗市的服務；
*269C（天水圍市中心↔觀塘碼頭）祇在每天繁忙時間提供服務，每天全日不再途經元朗市。

在2004年8月15日，268C及269C線最終成功把來往[[東九龍|東九龍]]至元朗市及[[天水圍新市鎮|天水圍新市鎮]]的乘客分流，兩線進一步提升至每天全日服務，正式取代69C線。

2006年起，269C線平日早上增設特別班次，由[[天水圍|天水圍]][[天恩邨|天恩邨]]單向前往觀塘碼頭，途經天水圍北。

2008年2月24日，269C線在天水圍市中心的總站，遷往嘉恩街[[栢慧豪園|栢慧豪園]]基座。

2009年5月20日，269C線平日早上增設特別班次，由天水圍[[天慈邨|天慈邨]]單向前往觀塘碼頭。同年12月28日，天恩邨往觀塘碼頭之特別班次加強服務。

2015年9月12日，天恩邨往觀塘碼頭之特別班次成為第二代69C線，服務時段不變，69C同時增設星期一至五下午繁忙時間由觀塘碼頭返回天恩邨的回程服務。

2016年10月17日，繁忙時間特快路線'''269S'''投入服務，來往[[天水圍市中心|天水圍市中心]]及[[觀塘碼頭|觀塘碼頭]]，並途經九龍灣商貿區。只限星期一至五上午繁忙時間服務。

2018年8月27日，'''269S線'''增設回程服務，只限星期一至五下午繁忙時間服務。<ref>{{Cite web |url=http://search.kmb.hk/KMBWebSite/AnnouncementPicture.ashx?url=1534816594_2203_0.pdf |title=存档副本 |access-date=2018-08-25 |archive-url=https://web.archive.org/web/20180821162951/http://search.kmb.hk/KMBWebSite/AnnouncementPicture.ashx?url=1534816594_2203_0.pdf |archive-date=2018-08-21 |dead-url=no }}</ref>

2019年2月25日，269C線天慈特別班次改經[[天水圍站|天水圍站]]，並在星期一至五加開07:25一班。<ref>{{Cite web |url=http://search.kmb.hk/KMBWebSite/AnnouncementPicture.ashx?url=1550558768_1510_0.pdf |title=269C特別車路線永久改道及修訂行車時間表 |access-date=2019-02-20 |archive-url=https://web.archive.org/web/20190219173052/http://search.kmb.hk/KMBWebSite/AnnouncementPicture.ashx?url=1550558768_1510_0.pdf |archive-date=2019-02-19 |dead-url=no }}</ref>

==服務狀況==
269C線開辦20多年以來，已經歷兩次成功的大型重組，背後與其所服務的兩個地區——元朗區和[[九龍東|九龍東]]出現騰飛的發展，以及營辦商密切注視路線形勢的態度不無關係。

1994年開辦時，旨在方便新界西部的[[元朗市|元朗市]]和當時新開發的[[天水圍新市鎮|天水圍新市鎮]]的通勤人士前往九龍東上班。由於該區居民習慣利用巴士到達[[荃灣|荃灣]]或[[葵涌|葵涌]]，再轉乘[[香港地鐵|地下鐵路]]前往[[九龍|九龍]]；而香港工業北移到內地，[[觀塘區|觀塘區]]的工廠大多人去留空，本線客量偏低。其行車里數達53公里之長，是當時香港十條最長途的巴士路線之一。

與此同時，[[屯門公路|屯門公路]]偶然會發生如大石／山泥傾瀉或涉及傷亡的交通意外等嚴重事故，阻礙來往荃灣和[[屯門|屯門]]及元朗的交通，後兩者的居民開始察覺到前往[[上水站|上水站]]轉乘[[九廣鐵路|九廣鐵路]]（今[[港鐵|港鐵]][[東鐵綫|東鐵綫]]）來往[[九龍|九龍]]的好處，連帶取道新界環迴公路的公共交通需求，亦有如雨後春筍般叢生。元朗區兩個[[新市鎮|新市鎮]]同時獲得不同程度的發展下，本線客量開始上升，惟未至於達到飽和狀態。

用作分流往返九龍和新界西北的[[香港3號幹線|三號幹線（郊野公園段）]]於1998年5月25日（星期一）下午正式通車，九巴為迎接新幹線開通，決意為旗下元朗區來往九龍和[[香港島|香港島]]的巴士路線進行一次大型重組，由幹線通車翌日起實施<ref>《蘋果日報》A8版 本港新聞（1998年5月26日）</ref>。本線服務水平獲得大幅度提升，由本來的星期一至六早上由[[天瑞邨|天瑞邨]]及傍晚由藍田地鐵站開出的各一班車，更改為來往天水圍市中心及觀塘碼頭，每天清晨至翌日凌晨提供雙向頻密班次服務，且有自己一套車隊,，以12輛[[丹尼士巨龍|丹尼士巨龍]]12米雙層空調巴士（3AD）為主力<ref>{{Cite web |url=https://vignette.wikia.nocookie.net/hongkongbus/images/b/b4/RDP_1998.pdf/revision/latest?cb=20090913135420&path-prefix=zh |title=存档副本 |access-date=2018-01-28 |archive-url=https://web.archive.org/web/20180129004337/https://vignette.wikia.nocookie.net/hongkongbus/images/b/b4/RDP_1998.pdf/revision/latest?cb=20090913135420&path-prefix=zh |archive-date=2018-01-29 |dead-url=no }}</ref>，取代以往從其他路線指派使用的[[利蘭奧林比安|利蘭]]／[[富豪奧林比安|富豪奧林比安]]／丹尼士巨龍11米雙層空調巴士（AL／AV／AD）。棄用繞路的大老山隧道、吐露港公路和新界環迴公路，選用較直接的呈祥道和三號幹線，能減少行車里程外，亦有助加快行車時間，全程收費得以維持不變。

可是，九巴的用心並未得到元朗區居民的支持。祇因實施路線重組當天，大量乘客未能在原定時間登上所需巴士路線而有所不滿。本線最嚴重的脫班情況發生在當天晚上，數十名乘客在觀塘碼頭等候近2小時才有車，期間不耐煩的乘客數度與站長理論<ref>《蘋果日報》本港新聞（1998年5月27日）</ref>。經此一役，九巴隨後因應乘客實際需求逐步調整巴士班次。

一款適宜行走長途路程的12米雙層空調巴士[[Neoplan_Centroliner|Neoplan Centroliner]]（九巴「超時代巴士」，AP），於1998年引入，並在投入服務前，被派往天水圍作宣傳。該款巴士為數祇有20輛，全由[[九龍灣|九龍灣]]車廠管理，其中數輛隨後成為269C線常用車款。另一方面，九巴積極推出八達通轉乘優惠計劃，乘客利用當時新興的電子貨幣「[[八達通|八達通]]」，連續繳付兩程指定車程的車資，次程可獲得優惠。九巴在1999年通過試驗後，認為可以拓展至其他路線<ref>《明報》A20版（1999年3月19日）</ref>。首個非試驗性質的轉乘計劃便隨之而生，以[[九龍巴士296A線|296A線]]為首，轉乘包括本線在內5條往返觀塘的新界路線，乘客使用計劃可享有車資優惠。

踏入2000年代，[[元朗新市鎮|元朗新市鎮]]和天水圍新市鎮得到重點發展，乘客通勤的需求有明顯增長，本線車隊數目不斷攀升，由1999年的17輛巴士<ref>{{Cite web |url=http://www.hkitalk.net/HKiTalk2/thread-1053316-1-1.html |title=存档副本 |access-date=2018-01-27 |archive-url=https://web.archive.org/web/20180127202408/http://www.hkitalk.net/HKiTalk2/thread-1053316-1-1.html |archive-date=2018-01-27 |dead-url=no }}</ref>增至最高峰時期共派出48輛巴士行走，超越現時74X及277X線成為全港巴士路線之冠。此外，還增設由新市鎮不同小區開出的特別班次，以應付早上前往東九龍上班的人潮。2000年7月，首次開辦早上特別班次，由朗屏邨開往觀塘碼頭，同時段由天水圍市中心開出的班次，則取道[[元朗公路|元朗公路]]避過元朗市，可讓天水圍新市鎮有更直接的車程，和元朗區居民較容易登車外，亦為本線其後進行的第二次大重組奠下基石。

與此同時，天水圍居民對往返九龍東的交通需求日益增加，而元朗居民難以登上前往九龍東的巴士情況漸趨惡化<ref>{{cite news |language = 中文  |url = http://hk.apple.nextmedia.com/news/art/20030129/3090149 |title = 乘客怨九巴269C班次不足 |publisher = 蘋果日報  |date = 2003年1月29日 |accessdate = 2015年5月3日}}</ref><ref>{{cite news |language = 中文  |url = http://hk.apple.nextmedia.com/news/art/20021010/2878521 |title = 269C巴士行車時間過長 |publisher = 蘋果日報  |date = 2002年10月10日 |accessdate = 2015年5月3日}}</ref>，引致不少「非法邨巴」應運而生<ref>[http://the-sun.on.cc/channels/news/20031127/20031127022641_0001.html 元朗往觀塘 繁忙時行駛 村巴疑無牌廉價拉客]，太陽報，2003-11-27</ref>。兩大前提下，分拆269C線成為兩條獨立路線的訴求，變得非常殷切；得到地區人士的爭取和巴士公司的周詳考量，願望終能在2004年逐步實現。

2004年2月實行的第一階段重組，主要分為2種服務時間和3個服務區域，即上、下午繁忙時段，各有一條往返觀塘碼頭和元朗及天水圍的路線，編號分別為[[九龍巴士268C線|268C]]和269C；以及在非繁忙時段維持重組前的269C線主線，編號更改為69C，以茲識別。第2階段在同年8月進行，主要是把69C線的服務時段併入268C線和269C線，令前者得以功成身退。該項重組模式亦在同期的[[九龍巴士269B線|269B線]]（天水圍市中心↔[[紅磡碼頭|紅磡碼頭]]）實行，連同269C線的重組，由兩線衍生出來的69B線和69C線可視為過渡期，讓區內居民適應，避免重蹈大欖隧道通車、巴士路線一次性重組而造成混亂情況的覆轍。

重組過後，來往九龍的元朗和天水圍居民，以及元朗前往天水圍的人士，也有足夠的替代交通服務，兩地居民可謂皆大歡喜。2006年起，269C線配合天水圍人口持續增長，針對前往日漸蓬勃的九龍東商業區上班一族，加開早上特別班次前往觀塘。時至今日，平日早上的繁忙時間，分別有由天慈邨和天恩邨開出的特別班次（後者已演變為第二代69C線，更增設了返回天恩邨的班次），與天水圍市中心開出的主線分擔客量，使整個天水圍新市鎮由北至南也有便捷的公共巴士服務。至於在分拆後，服務元朗新市鎮的268C線，無論派車或在班次上也得以提升水平。

269C線的競爭對手為[[港鐵|港鐵]]，但以現時巴士服務和鐵路網絡比較下，來往九龍東和元朗及天水圍的服務仍以巴士佔優，現時基本上所有天水圍區乘客來往九龍東已被本線吸納。

此外，本線與另一條編號相近、來往[[將軍澳新市鎮|將軍澳]][[尚德邨|尚德邨]]和長沙灣（深旺道）的[[九龍巴士296C線|296C線]]同樣駛經觀塘道，主要用車亦大致相同，由於兩線西行方向的目的地相距甚遠，在觀塘道的車站編排需盡量避免乘客混淆，只有牛頭角下邨站因地方太小而被編在同一位置。
