'''[[九龍巴士|九龍巴士]]61X線'''是[[香港|香港]]一條由[[九巴|九巴]]營辦的新界特快巴士路線<ref name="chunghwabook-1">[http://www.chunghwabook.com.hk/media/upload/9789888263059.pdf 九龍區巴士路線與社區發展 - 目錄] {{webarchive|url=https://web.archive.org/web/20141215235233/http://www.chunghwabook.com.hk/media/upload/9789888263059.pdf |date=2014-12-15 }}，書名：九龍區巴士路線與社區發展，ISBN 9789888263059，出版商：[[中華書局_(香港)|中華書局 (香港)]]，出版日期：2013年7月，第22頁</ref>，來往[[屯門市中心|屯門市中心]]及[[九龍城碼頭|九龍城碼頭]]，途經[[安定邨|安定邨]]／[[友愛邨|友愛邨]]、[[豐景園|豐景園]]、[[龍翔道|龍翔道]]、[[黃大仙|黃大仙]]、[[鑽石山|鑽石山]]、[[新蒲崗|新蒲崗]]、[[九龍城|九龍城]]及[[土瓜灣|土瓜灣]]，是屯門區以至整個新界西北來往九龍城、土瓜灣的唯一全日專利巴士線。

1986年初開辦，是九巴首批特快巴士路線（X線）之一，主要為來往[[屯門新市鎮|屯門新市鎮]]及[[黃大仙區|黃大仙區]]和[[九龍城區|九龍城區]]的市民提供服務，而隨着機場於1998年搬遷至[[赤鱲角|赤鱲角]]，[[啟德發展計劃|啟德發展計劃]]尚未全面竣工，本線客源亦略為減少。

== 歷史 ==
*1986年4月7日：開辦，來往[[屯門市中心|屯門市中心]]及[[九龍城碼頭|九龍城碼頭]]，是九巴首批特快路線之一（使用其時通車不久的[[荃灣路|荃灣路]]）。
*約1990年至1991年：配合沿線道路改善工程，先後作出多項永久改道措施：
**[[皇珠路|皇珠路]]近[[友愛邨|友愛邨]]一段道路改善工程完成後，本線來回程須改用[[屯門鄉事會路|屯門鄉事會路]]、[[豐安街|豐安街]]、[[海珠路_(香港)|海珠路]]、[[海皇路|海皇路]]及[[皇珠路|皇珠路]]，並於豐安街設置巴士站；當屯門鄉事會路與海珠路交界落成啟用，便不再途經豐安街，巴士站改為設於海珠路；
**為配合[[大老山隧道|大老山隧道]]及[[觀塘繞道|觀塘繞道]]工程，[[龍翔道|龍翔道]]東行不能直接右轉[[斧山道|斧山道]]南行，本線往九龍城碼頭方向須改經[[蒲崗村道交匯處|蒲崗村道交匯處]]直接前往[[彩虹道|彩虹道]]。
*1991年3月31日：九巴調整車費，全程收費增至$6.5。
*1993年4月5日：九巴調整車費，全程收費增至$7.2。
*1995年4月2日：九巴調整車費，全程收費增至$8.2。
*1995年4月25日：增設空調巴士服務，空調巴士收費為$12。
*1996年4月4日：九巴調整車費，唯空調巴士全程收費維持不變。
*約1996年至1997年：往九龍城碼頭方向繞經[[龍蟠街|龍蟠街]]及[[鳳德道|鳳德道]]並加設分站，方便鑽石山一帶的乘客。
*1997年12月1日：九巴調整車費，全程收費增至$9（非空調）／$12.8（空調）。
*1998年8月18日：取道新落成的[[青朗公路|青朗公路]]、[[青衣西北交匯處|青衣西北交匯處]]、[[長青公路|長青公路]]、[[長青隧道|長青隧道]]及[[青葵公路|青葵公路]]，[[荃灣路|荃灣路]]和[[葵涌道|葵涌道]]祇於[[汀九橋|汀九橋]]實施特別交通管制期間駛經。
*2002年10月19日：提供全線空調服務，全程收費為$11.7，分段收費維持不變。
*2004年2月8日：屯門區總站延長至位於[[河傍街|河傍街]]與[[仁政街|仁政街]]交界的[[屯門站公共運輸交匯處|屯門西鐵站]]，擴展服務範圍至[[屯門新墟|屯門新墟]]。
*2007年12月2日：[[九廣鐵路|九廣鐵路]]與[[地下鐵路|地下鐵路]]進行合併，九巴把各有關車站統稱為「鐵路站」。
*2008年6月8日：九巴調整車費，全程收費增至$12.3。
*2010年3月14日：配合位於[[杯渡路|杯渡路]]旁的[[屯門站公共運輸交匯處|屯門站公共運輸交匯處]]正式啟用而遷往該處。
*2010年5月7日：往[[九龍城碼頭|九龍城碼頭]]方向，於[[土瓜灣道|土瓜灣道]]近[[明倫街|明倫街]]增設分站。
*2011年5月15日：九巴調整車費，全程收費增至$12.7。
*2012年12月26日：配合[[屯門公路巴士轉乘站|屯門公路巴士轉乘站]]（南行）啟用，往九龍城碼頭方向加停該站，並提供[[九龍巴士59M線|59M]]、[[九龍巴士61M線|61M]]、[[九龍巴士67M線|67M]]、[[九龍巴士68A線|68A]]轉乘本路線的轉乘優惠。<ref name="SIULAM">[http://www.kmb.hk/tc/SiuLamInterchange.html 九巴屯門公路巴士轉乘計劃]，九巴官方網站</ref>
*2013年1月26日：配合[[屯門公路巴士轉乘站|屯門公路巴士轉乘站]]（南行）增加轉乘路線，提供[[九龍巴士58M線|58M]]、[[九龍巴士60M線|60M]]、[[九龍巴士66M線|66M]]轉乘本路線的轉乘優惠。<ref name="SIULAM">[http://www.kmb.hk/tc/SiuLamInterchange.html 九巴屯門公路巴士轉乘計劃]，九巴官方網站</ref>
*2013年3月17日：九巴調整車費，全程收費增至$13.3。
*2014年7月6日：九巴調整車費，全程收費增至$13.9。
*2014年10月11日：與[[九龍巴士60M線|九巴60M線]]對調總站，並遷回至屯門市中心（與此同時配合[[九龍巴士66M線|九巴66M線]]更改路線）。

=== 八達通巴士轉乘計劃 ===
乘客登上本線後150分鐘內以同一張八達通卡轉乘以下路線，或從以下路線登車後150分鐘內以同一張八達通卡轉乘本線，次程可獲$4.2車資折扣優惠：

*'''61X往九龍城碼頭''' → 2F任何方向或211往翠竹花園
*2F任何方向或211往[[黃大仙站|黃大仙站]] → '''61X往屯門市中心'''

===屯門公路轉車站轉乘優惠===
由往荃葵青方向的[[九龍巴士59M線|59M]]、[[九龍巴士61M線|61M]]、[[九龍巴士67M線|67M]]及[[九龍巴士68A線|68A]]（洪水橋或以後上車）於上車後2.5小時內於[[屯門公路巴士轉乘站|屯門公路巴士轉乘站]]（南行）轉乘本路線往九龍城碼頭方向，可獲回贈首程車費（即總車資為本路線全程收費）；於朗屏邨至大道村乘搭[[九龍巴士68A線|68A]]轉乘本路線往九龍城碼頭方向，總車費為本路線全程車費另補付$0.6。

== 使用車輛 ==
本線開辦至2016年8月15日由九巴屯門車廠（U）及九龍灣車廠（K）共同派車行走，初期分別以[[利蘭勝利二型|利蘭勝利二型]]（G）和[[丹尼士喝采型|丹尼士喝采型]]（N）提供服務。為提高載客量，九龍灣車廠隨即改派[[丹尼士統治者巴士|丹尼士統治者巴士]]（DM）取代舊有車款。在客量沒有明顯增長的情況下，派車一直維持至1994年中。當時，兩廠引入簇新的[[丹尼士巨龍|丹尼士巨龍]]11米（S3N）至本線，卻因[[九龍城碼頭巴士總站|九龍城碼頭巴士總站]]的泊位不足以讓這種長度的巴士通過，行駛了一段相當短的時間，祇剩2輛（S3N324/FZ7937、S3N342/FZ7574）獲保留，其餘用車為[[利蘭奧林比安|利蘭奧林比安]]9.5米（BL）、[[都城嘉慕威曼|都城嘉慕]]9.7米（M）和九龍灣車廠的丹尼士統治者巴士。當時，獲准留在本線的丹尼士巨龍11米巴士，到達[[九龍城碼頭|九龍城碼頭]]後祇能停泊在空間較大的[[九龍巴士6F線|6F線]]（現[[九龍巴士6C線|6C線]]）泊位，無論對車務運作或候車乘客均構成不便，最終需待總站改善工程完成後，即[[屯門公路|屯門公路]]實施[[巴士專線|巴士專線]]的同一年，亦適逢香港巴士空調化趨勢降臨，本線除再度使用同一批次的丹尼士巨龍11米及九龍灣車廠引進最新型號的[[富豪奧林比安|富豪奧林比安]]11米（S3V）等非空調巴士外，還加派了空調巴士投入服務行列，使得整體載客量、舒適度和行車速度也大為改善。

隨着空調巴士服務漸受乘客歡迎，本線在1990年代後期，進一步加強空調服務，換入高載客量的[[丹尼士巨龍|丹尼士巨龍]]11米（AD）、[[富豪奧林比安|富豪奧林比安]]11米（AV）等空調巴士。2002年10月起，本線提升為全線空調服務，非空調用車的空缺由上述空調用車補上。直至2004年，增設低地台巴士服務，採用了原服務於[[九龍巴士60X線|60X線]]的[[猛獅24.310|猛獅24.310]] 12米（AMN）空調巴士。數年間，個別用車先後以[[丹尼士三叉戟三型|丹尼士三叉戟]]12米（ATR）、[[富豪超級奧林比安|富豪超級奧林比安]]12米（3ASV）、猛獅24.310 12米、富豪奧林比安11米、丹尼士巨龍11米等替換。例如：2009年11月29日曾因應用車調動，換入1輛披上亞歷山大車身的丹尼士三叉戟三型（ATR101/NG6783），把1輛猛獅24.310（AMN8/JM3221）換到[[九龍巴士259D線|259D線]]；另外，同日因[[九龍巴士52X線|52X線]]延長至[[旺角|旺角]]（[[柏景灣|柏景灣]]），行駛於[[彌敦道|彌敦道]]的巴士需採用符合環保標準的引擎，故本線2輛配備歐盟II期引擎的富豪奧林比安11米（AV373/HN7542、AV436/HT1292）被派往52X線行走，換入了2輛配備歐盟I型引擎的同款巴士（AV148/GY2825、AV151/GY4088）。

在2004年6月，此路線首次加入低地台服務，屯門車廠（U）派出原本行走60X線的猛獅24.310 12米（AMN3、4、6、9、10及11），而九龍灣車廠（K）派出原本行走7號線的一輛富豪超級奧林比安12米（3ASV192）；同年11月，九龍灣車廠（K）改以丹尼士三叉戟12米（ATR17／HW3472）取代，並在2011年加入原本來自龍運巴士的同款巴士行走。 

2012年，此線曾兩度加入富豪B9TL12米（AVBW）以及斯堪尼亞K310UD 12米（ASU）行走，但後快便回復原有派車。同時，因應丹尼士巨龍年齡漸高，將所有非低地台用車統一為富豪奧林比安。2013年末，所有富豪奧林比安全部被取代，實現全線低地台化。 

因應此線其中一個字軌所柯打的52X線需行經低排放區，2015年末加入來自259D線的Enviro500 MMC 12米樣辦車（ATENU135／SH2747）行走，至2016年初改用配置SCR設備的丹尼士三叉戟（ATR333／KV7167）行走。2016年3月曾一度加入來自968線的Enviro500 MMC 12米（ATENU21／SB5197）行走。2016年4月重新加入來自269D線一部已配置SCR設備的丹尼士三叉戟（ATR269／JT647）行走，2016年8月再度加入來自60X線的Enviro500 MMC 12米（ATENU178／SJ6871）行走。 

2016年5月至6月間，部分來自258D線的丹尼士三叉戟換入本線，從而令猛獅24.310正式全面撤出往返九龍區的路線。雖然如此，但是部分猛獅24.310不時會以「特見」形式行走本線。

2016年8月15日，本路線改為屯門車廠派車行走。同年11月起加入猛獅A95（AMNE、AMNF）行走。 

現時本線使用2輛[[丹尼士三叉戟三型|丹尼士三叉戟]]12米（ATR）及11輛[[猛獅A95|猛獅A95]]（AMNE、AMNF），均屬[[九巴屯門車廠|屯門車廠]]。

== 營運狀況 ==
市區和新市鎮分別有其發展時序，就[[屯門新市鎮|屯門新市鎮]]而言，[[屯門市中心|屯門市中心]]是區內首先發展的社區之一，內有多個私人住宅和商業項目，加上位處道路骨幹[[屯門公路|屯門公路]]旁，正好成為[[屯門|屯門]]以至整個新界西北的交通樞紐。[[九巴|九巴]]為各個[[新市鎮|新市鎮]]打造多條特快巴士路線，目的為應付龐大客流量，快捷地把乘客帶到市區。在1980年代中開辦的新界巴士路線，需根據[[公共交通協調政策|公共交通協調政策]]的規定，把[[九龍區|九龍區]]起訖點設於適當地點，唯只適用於駛經[[彌敦道|彌敦道]]的新界路線。另外，[[荃灣路|荃灣路]]的開通亦為新界西的公共交通發展提供了良好的條件。

屯門市中心貴為新界西北新市鎮中，首個具規模的市中心發展，公共交通應能做到四通八達。對於屯門新市鎮來說，巴士可算是最當時得令的公共交通工具，擁有經營權的九巴便率先在屯門開辦兩條特快巴士路線，以減輕較早前開辦途經[[荃灣|荃灣]]及[[葵涌|葵涌]]往返市區的路線的客量壓力。配合人口分佈和遷徙等因素，屯門首2條往返九龍市區的特快巴士路線便分別以[[佐敦|佐敦]]及[[九龍城|九龍城]]為目的地，途經[[深水埗|深水埗]]、[[旺角|旺角]]、[[油麻地|油麻地]]及[[黃大仙|黃大仙]]、[[鑽石山|鑽石山]]、[[新蒲崗|新蒲崗]]、[[啟德機場|啟德機場]]，後者就是本路線。

本線主要服務屯門往返九龍北及[[東南九龍|東南九龍]]的乘客，在此以前並無類似路線開辦服務，故所有客源均來自其他交通工具及巴士路線，惠及乘客還遍及[[元朗|元朗]]和[[觀塘|觀塘]]，開辦之始已深受歡迎，經過一年半載的時間，九巴增辦一條往返屯門市中心及[[鯉魚門|鯉魚門]]的特快巴士路線(62X)，同樣途經黃大仙及新蒲崗，有效分擔客量。其後十年間，客量得以維持，但自[[香港國際機場|香港國際機場]]於1998年中搬遷至[[大嶼山|大嶼山]]以北的[[赤鱲角|赤鱲角]]，本線需求略為下降。隨着往返[[屯門區|屯門區]]及[[觀塘區|觀塘區]]的特快巴士路線相繼提升服務水平，以及[[啟德發展計劃|啟德發展計劃]]遲遲未落實，本線的通勤客量被逐漸蠶食；唯一不變的，就是每逢[[農曆新年|農曆新年]]也要加開特別班次往返屯門及鑽石山，為昔日遷徙到屯門的[[黃大仙區|黃大仙區]]居民，提供廉宜和直接的交通服務，與親友歡度佳節。

屯門公路轉車站啟用後，來往屯門至黃大仙、鑽石山一帶，若直接乘搭62X、258D或259D，車費是$16.6；而此路線的全程收費較為便宜，只是$13.9(不論是直接乘搭還是經屯門公路轉車站轉乘本線，車費一樣)，因此亦吸引部份「搭平車」的屯門居民乘搭。

當未來啟德發展計劃竣工後，[[土瓜灣|土瓜灣]]、九龍城及新蒲崗亦將有一番新景象，為該處的交通服務提供更佳的營運環境；另一方面，[[港鐵|港鐵]][[沙中綫|沙中綫]]落成後，現有的[[西鐵綫|西鐵綫]]會與[[馬鞍山綫|馬鞍山綫]]連接，貫通東南九龍、鑽石山等地，惟搭乘本路線來往屯門及九龍城的時間及走綫都較東西走廊快捷及直接（因後者需要先經過元朗、尖東及紅磡才抵達九龍城，走綫顯然較其迂迴）；此路線同時服務黃大仙及新蒲崗一帶，車費亦比港鐵、[[九龍巴士62X線|62X]]及[[九龍巴士259D線|259D]]線便宜。再者，沙中線未能服務到該區域，因此預料沙中綫通車對其客量影響不大。
