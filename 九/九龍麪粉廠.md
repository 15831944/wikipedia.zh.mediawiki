'''九龍麪粉廠'''（英文：'''Kowloon Flour Mills'''）是[[香港|香港]]市區僅存的[[麵粉|麵粉]]廠，位於[[九龍|九龍]][[觀塘工業區|觀塘工業區]][[海濱道|海濱道]]161號，於1966年2月1日起運作至今。

== 歷史 ==
[[File:九龍麪粉廠_Kowloon_Flour_Mills,_1966.jpg|缩略图]]
九龍麪粉廠是泰國華僑[[林國長|林國長]]於71歲來港創辦。

1960至1970年代觀塘仍未[[填海|填海]]及興建[[觀塘繞道|觀塘繞道]]，麪粉廠就剛好在海邊，當時是用輸送管道將由躉船運來的[[小麦|小麥]]吸入管道。

填海後公司改以貨櫃運送小麥，並向政府租下觀塘繞道下面的地皮放置貨櫃，再將小麥泵進管道入廠。

2011年10月，時任[[發展局|發展局]]局長[[林鄭月娥|林鄭月娥]]表示有意保育九龍麪粉廠，作為[[九龍東|九龍東]]的歷史地標，並點名指九龍麪粉廠可以保育活化。

== 建築 ==
[[File:觀塘_九龍麪粉廠_Kowloon_Flour_Mills_in_Kwun_Tong,_2018.jpg|缩略图]]
九龍麪粉廠為一座灰色的九層高建築。

其西面設有六條用作儲存[[小麥|小麥]]的大圓柱，向海的一條柱[[直書|直書]]「九龍麪粉廠」五個大字。

而二樓則有搬運小麥用的[[起重机|吊臂]]以及輸送管道橋樑，橫跨海濱道連接對出的碼頭。該管道當年能夠延伸至觀塘海徬，方便將躉船運來的小麥吸入管道。

== 運作 ==

# 從躉船或貨櫃運送的小麥原料會先送入2樓，然後傾倒入數層樓高的大圓柱倉庫中備用<ref>{{Cite web|url=http://lausoldier.blogspot.com/2016/06/blog-post_6.html|title=觀塘的九龍麪粉廠|accessdate=2018-08-16|author=Soldier Lau|date=2016-06-06|publisher=Soldier 的世界}}</ref>
# 包裝過程會首先把小麥泵入廠房5樓的機器打磨成粉，再「吹」上7樓篩出雜質
# 最後這些麵粉會跌落6樓再「吹」上7樓，然後包裝入袋

== 保育 ==
[[File:起動九龍東_九龍麪粉廠_電箱_"Energizing_Kowloon_East"_Kowloon_Flour_Mills_Switch_Box,_2018.jpg|缩略图]]」其中一個項目是利用電箱展示工業文化，此電箱位於九龍麪粉廠對面的行人路]]九龍東正轉型作香港第二個[[核心商業區|核心商業區]]，不少[[觀塘|觀塘]]和[[九龍灣|九龍灣]]昔日的工廠大廈會遭收購拆卸並重建作商業大廈。

林鄭月娥在2011年曾指出九龍麪粉廠「都幾標誌性，可反映五、六十年代」，並表示能作為一個[[工業區|工業區]]的工業文物。
