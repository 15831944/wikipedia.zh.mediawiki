'''九州大學'''（{{jpn|j=九州大学|hg=きゅうしゅうだいがく|rm=Kyūshū daigaku}}；[[英語|英語]]譯名：{{lang|en|Kyushu University}}），簡稱'''九大'''（きゅうだい），是[[日本|日本]]一所本部位於[[福岡縣|福岡縣]][[福岡市|福岡市]][[西區_(福岡市)|西區]]元岡的[[國立大學|國立]]研究型[[綜合大學|綜合大學]]。九大前身是舊制[[帝国大学|帝国大学]]－'''九州帝國大學'''（1911年），起源於1867年的[[福岡藩|福岡藩]]醫學校「贊生館」。

九大是全國首例導入「全學大學院重點化」制度的大學。

== 沿革 ==
1867年，當時統治[[福岡|福岡]]地區的[[黑田藩|黑田藩]]於天神設立了實行醫學教育的藩校「[[贊生館|贊生館]]」，1872年由於施行學制而廢校的同時，其附屬醫院在此期間還是繼續開設。而此醫院在1874年成為修猷館的附屬診療所，1879年改組為福岡縣立福岡醫學校的附屬醫院。而在福岡縣立福岡醫學校廢校後還是以福岡縣立福岡醫院繼續存在。

1886年帝國大學令公布後，在[[九州|九州]]設置一所帝國大學的機會相當高，終於在1900年1月29日的第14次帝國議會中通過「九州東北帝國大學設置建議案」，在建議案通過後尚有自古醫學就相當盛行的[[長崎縣|長崎縣]]和設有第五高等學校的[[熊本縣|熊本縣]]也在爭取帝國大學的立校，最後由起源於贊生館的[[福岡縣|福岡縣]]立福岡醫院作為[[京都帝國大學|京都帝國大學]]的分科大學而成為福岡醫科大學，也是後來的九州帝國大學的母體。

雖然之後由於資金籌措的困難一直遲遲無法設立九州帝國大學，但是古河財閥在1906年提出「福岡工科大學、仙台理工大學、札幌農科大學」建築建設的捐贈金，學校的設置才有了轉機，古河財閥大約100萬元的捐贈金中有六成作為九州帝國大學工科大學校舍建設的資金，再加上[[福岡縣|福岡縣]]的捐贈金終於在1911年1月1日九州帝國大學於是成立。古河財閥剩下的四成捐贈金則使用在升格為東北帝國大學農科大學的札幌農學校（1907年9月，之後成為[[北海道帝國大學|北海道帝國大學]]）以及新設立的[[東北帝國大學|東北帝國大學]]理科大學（1911年1月1日）中。

1947年由於帝國大學令改為國立總合大學令，故名稱改為九州大學。1949年，包含位於福岡地區的舊制九州大學，再加上九州大學附屬醫學專門部、福岡高等學校、久留米工業專門學校成為新制的九州大學。舊福岡高等學校的設施與教職員併入九州大學第一分校（1955年與第二分校整合為九州大學分校，1963年開始稱為教養部），舊久留米工業專門學校設為第二分校（1955年與第一分校合併），舊陸軍步兵第四十八連隊兵舍遺址則設為第三分校（1951年廢止）。

2003年根據法律第29號吸收九州藝術工科大學，2004年根據國立大學法人化，從原本由國家直接設置管理的國立大學轉為國立大學法人。

== 学术 ==
九州大學在完成「全學大學院重點化」後導入學府·研究院制度，將大學院原本作為研究與教育組織的「研究科」，分成教育組織「學府」與研究組織「研究院」，也就是將大學院的研究與教育單位分離。教職員屬於研究院，大學院生屬於學府，而大學部學生則屬於學部，基本上研究院長也可兼任學府長與學部長。自2006年3月至今，將這種制度導入至全校的只有九州大學，而部分實行的也只有幾所大學。

== 年表 ==
* 1867年 由'''黒田藩'''設立'''賛生館'''
* 1903年 福岡醫科大学設立（京都帝国大学之分科大学）
* 1910年 [[九州帝国大学|九州帝国大学]]設立
* 1911年 九州帝国大学工科大学官制施行 ，以分科大学設立醫科大学、工科大学
* 1919年 帝国大学令修改，各分科大学改制為醫学部與工学部，並設立農学部
* 1924年 設立法文学部，招收二名女学生
* 1939年 設立理学部
* 1947年 国立総合大学令生效，更名為九州大学
* 1949年 法文学部改組後成為文学部、法学部、経済学部。文学部並從教育学部分離。福岡高等学校、久留米工業専門学校、舊制九州大学統合後成為新制九州大学及各分校
* 1953年 新制大學院（研究所）設置
* 1963年 分校經改組 設立教養部
* 1964年 設立藥学部
* 1967年 設立齒学部
* 1968年 九州藝術工科大学創立（現九州大學藝術工學部）
* 1994年 廢除教養部
* 1997年 醫学系研究科與工学研究科開始"大学院重点化"
* 2000年 大学院重点化之完成同時引進学府、研究院制度，大学院（研究所）的所有研究科皆改組為研究院與学府
* 2003年 與九州藝術工科大学併校 藝術工学研究院・藝術工学府・藝術工学部
* 2004年 [[国立大学法人|国立大学法人]]化 正式廢除九州藝術工科大学、並設立法務学府（LAW SCHOOL）
* 2005年 開設伊都校区
