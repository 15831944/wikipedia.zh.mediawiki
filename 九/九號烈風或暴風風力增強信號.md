'''九號烈風或暴風風力增強信號'''（{{lang-en|'''Increasing Gale Or Storm Signal No. 9'''}}）為[[香港天文台|香港天文台]][[香港熱帶氣旋警告信號|熱帶氣旋警告信號]]中第二高的信號，僅次於[[十號颶風信號|十號颶風信號]]，俗稱'''九號風球'''，表示天文台預測風力將較[[八號烈風或暴風信號|八號烈風或暴風信號]]顯著加強。香港自1917年創立此信號起至今只曾發出50次。

==定義==
烈風或暴風的風力現正或預料會顯著加強。

根據香港天文台於1997年印製的漁民作業天氣資料卡，九號信號的意義詳述為「預料持續風力會在數小時內增至每小時88至117公里的時候，將會懸掛此信號」，意味着香港持續風力會顯著增強到最少達暴風程度。但天文台只會在預期香港可能有颶風的情況下，才考慮發出九號信號；即使香港持續吹暴風，若預期不會增強至颶風程度，天文台仍然不會發出九號信號<ref>[http://www.rthk.hk/radio/radio1/programme/hkfootpath/episode/480221 《舊日的足跡 - 天文台台長岑智明》 59分39秒至59分57秒；1小時24分05秒至1小時24分20秒]，[[香港電台|香港電台]]</ref>。

==歷史==
1917年，[[皇家香港天文台|皇家香港天文台]]首次設立相等於現時九號信號的「六號風球」，代表烈風將增強<ref>
[http://www.hko.gov.hk/publica/gen_pub/WeatheringTheStorm-4.pdf 風雲可測──第4節，城市的現代化（第42頁）]，香港天文台</ref>，為可能懸掛最高信號作準備。1931年，六號信號改為九號，自此九號信號一直沿用至今。1917年8月，天文台首次懸掛烈風增強信號。進入1920年代，颱風頻繁襲港，創下至今最多九號信號的年代，共達10次；其中1923年創下最多九號信號的年份，達4次。1929年的[[己巳風災|己巳年颱風]]首次由八號信號直接改掛十號信號，而省略九號信號，最後一次由八號直升至十號信號則為1957年9月[[颱風姬羅莉亞_(1957年)|颱風姬羅莉亞]]。歷來最長的九號信號為1922年7月[[壬戌風災_(1922年7月)|壬戌風災]]，生效長達24.5小時；1931年改制後最長則為[[颱風鸚鵡_(2008年)|2008年颱風鸚鵡]]，生效11小時；最短則為1979年[[颱風荷貝|颱風荷貝]]，僅懸掛25分鐘便改掛十號信號。

香港天文台曾把本警告稱為「九號烈風或暴風風力增強信號」<ref>{{cite web |url=http://info.gov.hk/gia/wr/199906/07/0607031.htm |title=天氣稿第25號－熱帶氣旋警告|accessdate=2008-08-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20080827130434/http://www.info.gov.hk/gia/wr/199906/07/0607031.htm |archivedate=2008-08-27 }}</ref><ref>{{cite web |url=http://info.gov.hk/gia/wr/200309/02/0902370.htm |title=天氣稿第199號－熱帶氣旋警告|accessdate=2008-08-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20081004085019/http://www.info.gov.hk/gia/wr/200309/02/0902370.htm |archivedate=2008-10-04 }}</ref><ref>http://www.info.gov.hk/gia/wr/201708/23/P2017082300343.htm</ref>、「九號烈風或暴風增強信號」<ref>{{cite web |url=http://info.gov.hk/gia/wr/200808/22/P200808220273.htm |title=天氣稿第151號－熱帶氣旋警告|accessdate=2008-08-23 |deadurl=yes |archiveurl=https://web.archive.org/web/20080915214719/http://www.info.gov.hk/gia/wr/200808/22/P200808220273.htm |archivedate=2008-09-15 }}</ref><ref>http://www.info.gov.hk/gia/wr/201207/23/P201207230919.htm</ref>及「九號暴風增強信號」<ref>http://www.hko.gov.hk/informtc/historical_tc/warning_bulletin_Hope_chi.pdf</ref><ref>http://www.info.gov.hk/gia/wr/199909/16/0916093.htm</ref>。而九號信號現今的正式名稱是「'''九號烈風或暴風風力增強信號'''」<ref>http://www.weather.gov.hk/textonly/explain/introc.htm</ref>，表示烈風或暴風的風力現正或預料會顯著加強，香港天文台在業務上使用這個名字時可能稍有出入。九號信號由設立至今，一直以上下相對的兩個三角形（[[File:Aspecto_do_sinal_nº_9_de_tempestade_tropical_de_Macau_na_dia.svg|15px]]）表示。2001年及以前，九號信號的晚間燈號為「綠綠綠」（[[File:Aspecto_do_sinal_nº_9_de_tempestade_tropical_de_Macau_em_noite.svg|4px]]）。

==九號烈風或暴風風力增強信號之最==
{{main|香港九號烈風或暴風風力增強信號列表}}
1917年設立數字熱帶氣旋警告信號系統至1930年期間，香港曾發出六號信號14次；自1931年改制至今，天文台發出九號烈風或暴風風力增強信號36次{{noteTag|1943年當局曾暫停懸掛風球一年，1943年10月20日至1945年8月底當局簡化風球制度，只設一或二號風球，二號風球包含現今八號、'''九號'''及十號信號之風力}}，當中有29次是在1946年或之後<ref>{{zh-hant}}{{cite web|url=http://www.hko.gov.hk/cgi-bin/hko/warndb_c1.pl?opt=1&sgnl=9&start_ym=194601&end_ym=201307&submit=%B7j%B4M |title=香港天文台警告及信號資料庫 |publisher=[[香港天文台|香港天文台]]|date= }}</ref>。九號信號及改制前之六號信號合計，一共發出50次。

===全年最早發出的九號信號===
*1961年[[颱風愛麗斯_(1961年)|颱風愛麗斯]]，由'''5月19日08:35至10:30'''，共維持1小時55分鐘，發出及取消時間均為夏令時間，亦是唯一一次需於5月發出九號烈風或暴風風力增強信號。

===全年最晚發出的九號信號===
*1954年[[超強颱風柏美娜_(1954年)|超強颱風柏美娜]]，由'''11月6日10:15至15:00'''，共維持4小時45分鐘，亦是唯一一次需於11月發出九號烈風或暴風風力增強信號。

===生效時間最長的九號信號===
*2008年[[颱風鸚鵡_(2008年)|颱風鸚鵡]]，由'''8月22日13:40至8月23日00:40'''，共維持11小時。

===生效時間最短的九號信號===
*1979年[[超強颱風荷貝_(1979年)|超強颱風荷貝]]，由'''8月2日12:35至13:00'''，共維持25分鐘，發出及取消時間均為夏令時間。

===發出最多九號信號的年份===
*'''1964年'''，共發出3次。

==影響==
*港內、港外線渡輪暫停
*往來[[澳門|澳門]]（包括[[氹仔|氹仔]]）渡輪及直昇機停航
*主題公園暫停開放
*[[辦公室|辦公室]]停工
*所有學校停課
*庇護工場暫停開放
*電車、纜車及絕大部份巴士路線停駛
*銀行暫停營業、交易所休市、貨櫃停止交收
*機場航班可能受影響，請留意航空公司通知
*[[港鐵|港鐵]]提供隧道段有限度服務，請留意[[港鐵公司|港鐵公司]]通知（可能在未知會情況下暫停服務）
*[[青嶼幹線|青嶼幹線]]只開放下層，青嶼幹線上層、[[汀九橋|汀九橋]]、[[昂船洲大橋|昂船洲大橋]]、[[深圳灣大橋|深圳灣大橋]]全線封閉；[[南灣隧道|南灣隧道]]東行封閉。

==應變措施及注意事項==
===應變措施===
*鎖緊門窗，把門栓拴好，窗板或大閘上牢。當風的大玻璃窗應加貼膠紙，減少玻璃破裂時所引致的損傷。
*不要站近當風的窗戶。把家具及貴重物件搬離風口位。萬一窗戶被吹破，確保仍有一個安全地方暫避，故應早點決定萬一當風的窗隻破裂時，哪一個房間可作棲身之用。
*光管招牌負責人須安排截斷招牌的電力供應。
*車輛應停泊在最不容易遭受破壞的地方。
*如情況許可，市民應儘早回家，避免逗留在街上。

===注意事項===
*香港不同地區的天氣情況不能夠單憑發出的信號推斷。只知道發出了什麼信號並不足夠，市民應該留意電台、電視台及天文台網頁及「打電話問天氣」系統所提供的熱帶氣旋最新消息及有關報告，然後就發出的信號決定採取適當的相應行動。
*市民應已採取所有預防措施。切勿外出，應遠離當風的門窗，以免被隨風吹來的碎片擊中。
*受地形或鄰近建築物影響，市民所在區域的風力與香港普遍風勢可能有顯著差異。離岸海域及高地風力通常較強，不當風的地區風力較弱。
*天文台透過多種途徑，特別是互聯網，向公眾提供各區風力及雨量的詳細資料。市民應該因應各自的具體情況和可接受的風險水平，就警告採取適當的預防措施。
*即使熱帶氣旋正逐步遠離香港，也不應放鬆戒備，因為烈風可能仍然肆虐一段時間，故應留在室內安全的地方，直至風勢緩和為止。

===防風措施報告===
防風措施報告一般在九號或十號信號生效期間於「熱帶氣旋警告」中顯示。
*各位現時切勿外出，如果已經在有遮蔽的地方躲避，就應該繼續在那裡停留，而且要遠離窗戶。屋內門戶必須關緊，小孩要安置在不當風的地方，千萬不要接觸被風吹倒的電線。
*窗門玻璃已遭受大風的壓力，如再受物件撞擊就會破裂，因此必須遠離當風的窗戶，並且預早選定一個安全的地方，以防玻璃萬一破裂時，可以躲避。除非毫無危險，現在不要修補破裂的窗戶及門戶。
*倘若「風眼」直接掠過香港，風勢會停息數分鐘至數小時不等，但具有破壞性的風力隨時會突然恢復從另一個方向吹來。不要離開有遮蔽的地方，同時留意風向轉變。請留意有關熱帶氣旋警告的廣播。
*因為海面［有非常大浪／巨浪／極巨浪／水流湍急且方向不定］，市民應該遠離岸邊及停止所有水上活動。
*外出人士若不能在短時間內回家，請即找尋安全地方躲避，直至危險過去為止。（如黑色暴雨警告信號生效，「若不能在短時間內回家」等字則不適用。）
*香港國際機場的航班可能受到天氣影響。旅客前往機場前，請先向航空公司查詢航班情況。

==風暴消息==
{{Main|風暴消息}}
當八號或以上信號生效期間，[[電視廣播有限公司|無綫電視]][[翡翠台|翡翠台]]及[[明珠台|明珠台]]、[[ViuTV|ViuTV]]及[[ViuTVsix|ViuTVsix]]和[[港台電視31、31A|港台電視31、31A]]會定時提供一節[[風暴消息|風暴消息]]（[[亞洲電視|亞洲電視]]在2016年停播前亦有同樣安排）（[[亞洲電視|亞洲電視]]在2016年停播前亦有同樣安排）。其中於2008年[[颱風鸚鵡_(2008年)|颱風鸚鵡]]襲港期間的風暴消息內，[[翡翠台|翡翠台]]同步直播[[北京奧運|北京奧運]]男子乒乓球項目。但若天文台在非30分或非00分更改[[香港熱帶氣旋警告信號|熱帶氣旋警告]]，電視台可以在廣告或節目中插播臨時的[[風暴消息|風暴消息]]，例如2012年的[[颱風韋森特_(2012年)|強颱風韋森特]]，天文台於凌晨12時45分發出[[十號颶風信號|十號颶風信號]]，[[翡翠台|翡翠台]]和[[本港台|本港台]]立即播出《[[風暴消息|風暴消息]]》。而[[風暴消息|風暴消息]]不會在新聞時段播出，相關風暴消息會在新聞時段內報道。另外所有電台亦會在每小時15分、30分、45分及58分廣播天文台最新熱帶氣旋警告，而相關風暴消息則會在正點及半點新聞內報道，若天文台更改熱帶氣旋警告，則由當時的節目主持或節目助理報告有關消息。

==爭議==
九號信號全名為「九號烈風或暴風風力增強信號」，表示當熱帶氣旋接近香港時，本港風力將「顯著增強」。然而，由於「顯著增強」一詞定義含糊。當天文台發出九號信號時，都對信號有不同定義，因此經常引起市民混淆。

===強烈熱帶風暴正面襲港爭議===
強烈熱帶風暴之中心最高持續風力達88至117公里。因此當一強烈熱帶風暴襲港，可能因暴風範圍太小未能令香港吹暴風，要到近距離橫過香港才有機會為香港帶來暴風。一強烈熱帶風暴能否為香港帶來暴風，除了暴風範圍大小，地形亦是一重要因素，所以強烈熱帶風暴路徑上細微的改變足以令香港的風勢有很大的變化。天文台在多次應對強烈熱帶風暴正面襲港之處理手法均引起討論是否應發出九號信號。

香港1973年受[[颱風黛蒂_(1973年)|颱風黛蒂]]正面吹襲，天文台懸掛九號信號。唯由於登陸前減弱為「強烈熱帶風暴」，當時天文台立即改掛八號信號。

直至1997年[[颱風維克托|颱風維克托]]正面襲港，天文台在維克托減弱為強烈熱帶風暴的情況下，仍然維持九號信號，直至風暴遠離香港才改掛八號信號。因而創下當時維持時間最長的九號信號（6小時50分，唯在2008年已被類似情況的鸚鵡的11小時所打破）。自此之後，每逢有颱風登陸前減弱為強烈熱帶風暴並橫過香港，天文台也維持九號信號，因此出現了[[颱風瑪姬|颱風瑪姬]]及[[颱風鸚鵡|颱風鸚鵡]]等的情況。唯1999年的[[颱風森姆|颱風森姆]]是例外，由於當時天文台預測失誤，在森姆移至香港以東80公里時，才改掛八號信號並正式升格為「颱風」，但其後又迅速減弱，因而沒有改掛九號信號。

不少市民會誤以為當強烈熱帶風暴正面襲港時，天文台便會發出九號烈風或暴風風力增強信號。從而出現了1993年[[強烈熱帶風暴貝姬_(1993年)|強烈熱帶風暴貝姬]]、1995年[[1995年太平洋颱風季#颱風海倫(Helen)|強烈熱帶風暴海倫]]及[[強烈熱帶風暴帕卡_(2017年)|強烈熱帶風暴帕卡]]等，中心風力達強烈熱帶風暴上限，有些甚至接近颱風下限，非常接近香港而帶來較大風力的熱帶氣旋。不過在2017年，天文台台長[[岑智明|岑智明]]回應記者查詢時表示，只有當颱風以上級數的熱帶氣旋正面襲港時，並有可能發出[[十號颶風信號|十號颶風信號]]時，才有機會發出九號信號，正式排除強烈熱帶風暴正面襲港發出九號信號的可能性。[[強烈熱帶風暴貝姬_(1993年)|強烈熱帶風暴貝姬]]明顯是強度預報失誤，在實測長洲持續錄得颶風的情況下，仍然維持強烈熱帶風暴的級別，即使香港普遍風勢相當大，仍然維持八號信號。至於1995年[[1995年太平洋颱風季#颱風海倫(Helen)|強烈熱帶風暴海倫]]則受地形因素影響，而2017年[[強烈熱帶風暴帕卡_(2017年)|強烈熱帶風暴帕卡]]則始終未增強至颱風級別，預測香港風力不會增強至颶風程度，因而沒有發出九號信號。

因此實際上，在以上例子中，只有1993年[[強烈熱帶風暴貝姬_(1993年)|強烈熱帶風暴貝姬]]襲港時，才達到天文台發出九號或以上信號的標準。
