'''九尾'''（日文：'''{{lang|ja|キュウコン}}'''，英文：'''{{lang|en|Ninetales}}'''）是[[精靈寶可夢|精靈寶可夢]]系列登場的{{NUMBEROFPOKEMON}}種虛構[[角色|角色]]（[[怪物|怪獸]]）中的一種。

== 特徵 ==
[[六尾|六尾]]的進化型。與六尾時期茶色的毛髮不同，經過歲月的累積散發出如黃金般的光輝。

除了岩石屬性與傳說的寶可夢之外最為長壽的寶可夢，能夠活1000年以上。

原型應為神話中的[[妖怪|妖怪]]－[[九尾狐|九尾狐]]。

英文名稱Ninetales與英文中的nine tails（九隻尾巴）相同。

== [[神奇寶貝遊戲列表|遊戲]]中的九尾 ==
由六尾使用「火之石」後進化而成，之後就無法利用等級的提升學習新招式。特性為「引火」或「日照」，「引火」的效果是受到火屬性招式攻擊時無效，同時自身火屬性招式的傷害上升為原來的1.5倍。

是「[[精靈寶可夢_紅·綠|皮卡丘版]]』中的勁敵所持有的寶可夢之一。

== 動畫中的九尾 ==
紅蓮鎮道館館主夏伯所持有的寶可夢之一，在與小智對戰時，打敗了小智的[[傑尼龜|傑尼龜]]。

在城都地區的斯里巴吉山上原本也有一隻九尾居住於一間廢置大宅裡，該九尾已經在世超過200年，其主人與小剛長得很像。在獨自生活期間習得催眠術和念力。
