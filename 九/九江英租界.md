'''九江英租界'''是近代中国七个[[在华英租界|在华英租界]]之一（另外六个是上海英租界（不久并入[[上海公共租界|上海公共租界]]）、[[天津英租界|天津英租界]]、[[汉口英租界|汉口英租界]]、[[镇江英租界|镇江英租界]]、[[广州英租界|广州英租界]]和[[厦门英租界|厦门英租界]]），也是[[九江|九江]]惟一的租界。

== 开辟 == 
由于清朝在[[第二次鸦片战争|第二次鸦片战争]]中战败，而在1858年《[[中英天津条约|中英天津条约]]》第十款中，规定“准将自汉口溯流至海各地，选择不逾三口，准为英船出进货物通商之区”<ref>《[[中英天津条约|中英天津条约]]》</ref>。第二次鸦片战争结束后，1861年初，英国[[参赞|参赞]][[巴夏礼|巴夏礼]]乘坐军舰前往根据条约新开辟的长江口岸勘察形势，选定将要开辟的租界界址。在镇江和汉口两地租界界址划定之后，3月22日，巴夏礼从汉口返回九江。他一度计划改到控扼鄱阳湖口的[[湖口縣|湖口]]开辟商埠，但是经过江西省官员的一再劝说，最终确定在九江开辟商埠。3月25日，江西布政使[[张集馨|张集馨]]与英国[[参赞|参赞]][[巴夏礼|巴夏礼]]订立开辟九江英租界的约章《九江租地约》。租界的位置位于九江府城以西，[[长江|长江]]与[[甘棠湖|甘棠湖]]之间的狭窄地段，西面到[[龙开河|龙开河]]，东西向长150丈（即沿长江岸线长度），南北向进深为60丈。面积150亩。租界南部包括湓浦港的一部分。湓浦港原是长江的江汊，早已淤塞成池塘。
==扩张== 
1891年，英租界工部局越界填筑了湓浦港的界外部分，辟为公园和球场。
== 居民 == 
===各国侨民===
== 管理 ==
九江英租界的行政体制实行侨民自治，通过选举产生工部局董事会，设有2~4名董事。英国领事不干预租界的日常行政。[[工部局|工部局]]为租界的行政机构，下设有警务机构巡捕房。 
== 市政 ==
九江英租界开辟不久，到1867年，界内修筑了长江码头和大致呈田字形交汇的道路（东西向：江岸路、中街；南北向：庐山路、西河岸路），沿江的土地大多已经被[[怡和洋行|怡和洋行]]、[[太古洋行|太古洋行]]等英国商号租借<ref>梅辉立：《中国和日本的通商口岸》，第431页</ref>。

== 经济 ==
九江开埠和开辟英租界不久，作为江西省唯一的通商口岸，九江很快取代了江西省传统的商业中心[[吴城镇_(永修县)|吴城镇]]（位于[[赣江|赣江]]注入[[鄱阳湖|鄱阳湖]]之处）和[[樟树镇|樟树镇]]（位于赣江中游），成为江西省新的贸易中心<ref>傅春官：《江西商务说略》，《江西官报》1906年第27期</ref>。江西的进出口货物不再经由传统的商路南下广东，而是北上经由九江。

== 事件 ==
在九江英租界存在期间，曾经发生过数起中外冲突事件。一次是租界开辟之初，1862年3月27日，英国人在重新勘测租界界址时，与当地居民发生冲突。1909年4月26日，英国巡捕打死中国商贩余发程，引发九江持续数月之久的抵制英货运动。1919年，发生了中国群众抗议英租界当局对过境行李、货物抽税的活动。1925年6月13日，九江的中国群众在全国性的[[五卅运动|五卅运动]]中，发动对英租界的冲击行动。最后一次则是1927年1月6日大规模的冲击行动，这次行动最终导致了九江英租界的结束。
== 收回== 

1927年1月6日，'''九江英租界'''被大批中国民众冲击。2月20日，武汉国民政府外交部长[[陈友仁|陈友仁]]与英方代表欧玛利签订《收回九江英租界之协定》。3月15日，武汉国民政府正式收回'''九江英租界'''。
