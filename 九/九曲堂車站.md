'''九曲堂車站'''位於[[高雄市|高雄市]][[大樹區|大樹區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[屏東線|屏東線]]的[[鐵路車站|鐵路車站]]，亦為過去[[臺灣糖業鐵路|臺灣糖業鐵路]][[旗尾線|旗尾線]]的起點站。車站增闢後站，於2016年2月底發包動工，2018年10月9日正式啟用。<ref>[https://udn.com/news/story/7327/3412937 九曲堂後站落成啟用 台鐵宣布增加班次加碼放送]</ref>

== 車站構造 ==
*[[島式月台|島式月台]]兩座，並以地下通道和無障礙電梯連接車站大廳及1、2月台、後站。
*1A與2B月台主要為待避之用。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]。
*停靠狀態：
**[[自強號列車|自強號]]停靠：
***（逆行）333、115、313、317、123、125、129、135、165/327、141、145次
***（順行）108、112、114、118、304、306、128、134、142、314、150、152、156次
**[[莒光號列車|莒光號]]停靠：
***（逆行）701、751、507、705、513、561次
***（順行）506、510、704、524、706、708次
**[[臺鐵區間車|區間車]]：皆停靠。
**[[臺鐵區間快車|區間快]]：皆停靠。
**[[復興號列車|復興號]]停靠:
***（逆行）783、791次
***（順行）不停靠
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[一卡通_(台灣)|一卡通]]、[[悠遊卡|悠遊卡]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1908年6月25日：設站。
*2012年3月26日：增設無障礙[[電梯|電梯]]。
*2013年9月：無障礙[[電梯|電梯]][[File:Handicapped_mark.svg|20px]]啟用。
*2013年9月30日：啟用多卡通刷卡機。
*2018年10月9日：後站啟用。
