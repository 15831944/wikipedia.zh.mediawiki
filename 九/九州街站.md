'''九州街站'''是[[杭临城际铁路|杭临城际铁路]]的西端起点站，位于[[杭州市|杭州市]][[临安区|临安区]][[玲珑街道|玲珑街道]]锦南新城<ref>{{cite web |title=杭临城际铁路建得怎么样了？临安境内6个站点探访 |url=http://hznews.hangzhou.com.cn/chengshi/content/2018-06/07/content_7015546_2.htm |website=杭州网}}</ref>九州街双拥路口，沿九州街东西向布置。

== 结构 == 
九州街站为地下二层[[岛式站台|岛式站台]]。该站主体长391.6米，宽21.3米；有效站台宽度12.6米、长80米。该站为双柱三跨[[钢筋混凝土|钢筋混凝土]]箱型框架结构<ref>{{cite web |title=杭临城际铁路锦南新城站-中间风井区间盾构左线顺利贯通 |url=http://www.lanews.com.cn/jrla/content/2018-10/31/content_6364181.htm |website=临安新闻网}}</ref>。

该站西侧设有出入库线，连接[[上泉车辆基地|上泉车辆基地]]。
