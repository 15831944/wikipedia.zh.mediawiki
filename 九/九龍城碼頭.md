'''九龍城碼頭'''，全稱'''九龍城渡輪碼頭'''，是位於[[香港|香港]][[九龍|九龍]][[九龍城區|九龍城區]][[馬頭角|馬頭角]]的一個[[渡海小輪|渡海小輪]][[碼頭|碼頭]]，望向[[啟德機場|啟德機場]][[跑道|跑道]]一帶。值得留意的是，雖然叫「九龍城碼頭」，由於[[啟德機場|啟德機場]]擴建，此碼頭與真正的[[九龍城|九龍城]]相距甚遠。

九龍城碼頭旁邊有一個[[:Template:香港公眾碼頭|公眾碼頭]]，名為「馬頭角公眾碼頭」。

== 歷史 ==
九龍城曾在現太子道附近設有碼頭（參見[[龍津石橋|龍津石橋]]），後隨[[啟德機場|啟德機場]]發展填海而湮沒。

九龍城碼頭首班渡輪航線於1956年7月3日下午6時開出往[[灣仔|灣仔]]（1967年6月24日停辦）<ref>九龍城灣仔線小輪 昨晚六時正式通航 [[大公報|大公報]]1956年7月4日</ref>，至今碼頭已經有60年歷史。九龍城碼頭是香港戰後建成的第一個永久渡輪碼頭。九龍城碼頭曾經開辦多條航線，包括九龍城至[[灣仔碼頭|灣仔]]、九龍城至[[北角碼頭|北角]]、九龍城至[[東平洲|東平洲]]、九龍城至[[太古城|太古城]]（後來改往[[西灣河碼頭|西灣河]]），以及九龍城至北角[[汽車渡輪|汽車渡輪]]（於1965年開辧，但於1972年[[紅磡海底隧道|紅磡海底隧道]]開通後一度停辦，其後復辦，直至1998年1月停辦至今）等。

==現況==
目前九龍城碼頭只有一條往來北角的渡輪航線行走，由[[新渡輪|新渡輪]]經營 <ref>[http://www.nwff.com.hk/chinese/schedule/route.asp?Origin=NP&Dest=KC&From=25&line=H 新渡輪網頁 - 北角至九龍城服務]</ref>，每天07:05至19:35提供服務，每隔半小時一班。由於收費設施設於[[北角碼頭|北角碼頭]]，乘客無論是往北角方向還是往九龍城方向都需要在[[北角碼頭|北角碼頭]]以[[港幣|港幣]]或[[八達通|八達通]]繳付船費。

汽車渡輸設施現為停車場。

== 將來發展 ==
將動工的[[中九龍幹線|中九龍幹線]]將會以[[隧道|隧道]]形式在九龍城碼頭旁廣場及附近海床下通過，施工其間須封閉九龍城碼頭巴士總站土地進行大規模挖掘以及在碼頭以北海面進行臨時[[填海|填海]]以便興建海床下的[[沉箱|沉箱]]隧道。馬頭角公眾碼頭須拆卸以配合施工。工程完成後海面將會還原。<ref>[http://www.central-kowloon-route.com.hk/eng/home.htm 中九龍幹線工程] {{webarchive|url=https://web.archive.org/web/20100906114207/http://www.central-kowloon-route.com.hk/eng/home.htm |date=2010-09-06 }}</ref>
