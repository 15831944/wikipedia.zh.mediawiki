'''九讚頭車站'''位於[[台灣|台灣]][[新竹縣|新竹縣]][[橫山鄉_(臺灣)|橫山鄉]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[內灣線|內灣線]]的[[鐵路車站|鐵路車站]]，也是距離橫山市區最近的車站。

== 車站構造 ==
*[[島式月台|島式月台]]一座。

== 營運狀況 ==
*目前是乙種[[臺灣鐵路管理局車站等級#簡易站|簡易站]]，但無派員售票。
*行駛車輛為[[台鐵DR1000型柴油客車|DR1000]]型[[台鐵柴油客車|柴油客車]]，等級為[[臺鐵區間車|區間車]]。
*過去本站的[[水泥|水泥]]貨運業務相當熱絡，最盛時曾為台鐵貨運量第二名的車站<ref>{{cite news|title=運送水泥業務 九讚頭車站盛名一時|url=http://web.pts.org.tw/hakka/news/detail.php?id=102383|publisher=[[客家電視台|客家電視台]]|date=2013-10-02|accessdate=2014-03-30}}</ref>（台鐵貨運量第一名為[[北迴線|北迴線]][[和平車站|和平站]]），因而保有十分寬闊的站場。
*車站附近有一小段由[[亞洲水泥|亞洲水泥]]新竹製造廠連接至臺鐵路線的火車軌道，不過部分路線已被周圍居民用作菜圃。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1950年12月27日：設站。
*2001年9月1日：降等為丙種簡易站，由[[竹東車站|竹東站]]管理。
*2007年4月16日：調整為簡易站，仍由[[竹東車站|竹東站]]管理。
*2011年11月11日：調整為乙種簡易站。
*2013年9月30日：啟用多卡通刷卡機。
