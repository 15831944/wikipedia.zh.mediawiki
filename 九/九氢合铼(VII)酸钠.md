'''九氢合铼(VII)酸钠'''是一种过渡金属氢负离子配合物，化学式为Na<sub>2</sub>[ReH<sub>9</sub>]。<ref>Лидин Р.А. и др. Химические свойства неорганических веществ: Учеб. пособие для вузов. — 3-е изд., испр. — М.: Химия, 2000. — 480 с. — ISBN 5-7245-1163-0.</ref><ref>Рипан Р., Четяну И. Неорганическая химия. Химия металлов. — М.: Мир, 1972. — Т. 2. — 871 с.</ref>
==制备==
金属钠还原[[高铼酸钠|高铼酸钠]]或[[高铼酸铵|高铼酸铵]]的[[乙醇|乙醇]]溶液可以得到。<ref name=Zhu />
::<math>\mathsf{NH_4ReO_4 + 18Na + 13C_2H_5OH \ \xrightarrow{}\ Na_2[ReH_9]\downarrow + 13C_2H_5ONa + 3NaOH + NH_3\cdot H_2O }</math>
==物理性质==
Na<sub>2</sub>[ReH<sub>9</sub>]可溶于[[水|水]]和[[甲醇|甲醇]]，微溶于乙醇。<ref name=Zhu>朱文祥. 无机化合物制备手册. 化学工业出版社. pp 312-314. 【VII-74】九氢合铼酸二钠（sodium nonahydridorhenate）</ref>
==化学性质==
* Na<sub>2</sub>[ReH<sub>9</sub>]受热分解：
::<math>\mathsf{2Na_2[ReH_9] \ \xrightarrow{245^oC}\ 4NaH + 2Re + 7H_2 }</math>

* 和热水反应：
::<math>\mathsf{Na_2[ReH_9] + 5H_2O \ \xrightarrow{100^oC}\ NaReO_4 + NaOH + 9H_2 }</math>

* Na<sub>2</sub>[ReH<sub>9</sub>]在碱性溶液中稳定，和稀酸反应放出[[氢气|氢气]]：<ref name=Zhu />
::<math>\mathsf{2Na_2[ReH_9] + 4HCl \ \xrightarrow{}\ 4NaCl + 2Re\downarrow + 11H_2\uparrow }</math>

* 加热情况下可以被热而浓的氧化剂氧化为[[高铼酸钠|高铼酸钠]]：
::<math>\mathsf{4Na_2[ReH_9] + 22HNO_3 \ \xrightarrow{100^oC}\ 4NaReO_4 + 9NH_4NO_3 + 4NaNO_3 + 11H_2O }</math>
::<math>\mathsf{Na_2[ReH_9] + 9H_2O_2 \ \xrightarrow{100^oC}\ NaReO_4 + NaOH + 13H_2O }</math>

* 可以被空气氧化：
::<math>\mathsf{4Na_2[ReH_9] + 11O_2 \ \xrightarrow{\tau}\ 4Re + 8NaOH + 14H_2O }</math>
