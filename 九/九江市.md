'''九江市'''，简称'''浔'''，古称[[柴桑|柴桑]]、[[潯陽|潯陽]]、[[江州|江州]]，是[[中华人民共和国|中华人民共和国]][[江西省|江西省]]下辖的[[地级市|地级市]]，位于该省北部，有江西“北大门”之称。市区坐落于[[长江|长江]]南岸、[[庐山|庐山]]北麓、[[鄱阳湖|鄱阳湖]]以西。
九江市是江西省級歷史文化名城.

== 历史 ==
* 据《晋太康地记》记载，九江的得名源于“[[刘歆|刘歆]]以为湖汉九水（即赣水、鄱水、余水、修水、淦水、盱水、蜀水、南水、彭水）入[[彭蠡泽|彭蠡泽]]也”。

* 秦设[[九江郡|九江郡]]，治所在[[寿春|寿春]]（今[[安徽|安徽]][[寿县|寿县]]），范围包括了今天的九江市，不过并未置县。这是九江之名第一次出现，但是很快九江郡范围大大缩小，到三国时被改名为[[淮南郡|淮南郡]]，所以从[[西汉|西汉]]到[[明朝|明朝]]之间的一千五百多年间“九江”的地名一般与今天的九江市并无关系。

* 汉高祖六年（公元前201年）在此置[[柴桑县|柴桑县]]，汉文帝设[[寻阳|寻阳]]县（县治在江北，古长江大概为今[[太白湖|太白湖]]、[[龙感湖|龙感湖]]一线），是此地置县之始。

* 三国时属孙吴，柴桑隶[[武昌郡|武昌郡]]，寻阳隶[[庐江郡|庐江郡]]。

* [[西晋|西晋]][[永兴|永兴]]元年（304年），析庐江之寻阳、武昌之柴桑二县立[[寻阳郡|寻阳郡]]，初治寻阳县，咸康中移治柴桑县，隶[[江州|江州]]。[[永嘉|永嘉]]三年（309年），寻阳县治南迁，江北划归蕲春。江北的寻阳老城改称为“浔水城”。

* [[东晋|东晋]][[咸和|咸和]]中[[温峤|温峤]]移江州治寻阳，从此“江南之寻阳著，江北之寻阳益晦，后遂废汉寻阳县入柴桑县，自是以后皆以寻阳郡城为寻阳城矣。”（《[[读史方舆纪要|读史方舆纪要]]》）

* [[南朝|南朝]][[梁|梁]]析柴桑县置[[汝南|汝南]]县，析[[豫章郡|豫章郡]]别置[[豫宁郡|豫宁郡]]，属[[高州|高州]]，寻阳郡改属[[西江州|西江州]]。[[陈|陈]][[天嘉|天嘉]]四年废高州，旋省西江州，寻阳、豫宁2郡仍属江州。

* [[隋|隋]][[开皇|开皇]]九年（589年）废郡置江州，废柴桑、汝南，复立寻阳县。十九年（599年），改浔阳县为彭蠡县。[[大业|大业]]二年（606年），改彭蠡县为彭城县，寻废彭城县为[[湓城|湓城]]县。三年又改江州为[[九江郡|九江郡]]。

* [[唐|唐]][[武德|武德]]四年（621年）复称江州，[[贞观|贞观]]初属[[江南道|江南道]]，[[开元|开元]]二十一年属[[江南西道|江南西道]]，[[天宝|天宝]]元年（742年）江州改称浔阳郡，[[乾元|乾元]]元年（758年）复为江州。析湓城置浔阳、[[楚城|楚城]]县，后俱入浔阳县。

* [[南唐|南唐]]改浔阳为德化，废江州。

* [[宋|宋]]复置江州。[[太平兴国|太平兴国]]七年（982年）析江州[[星子|星子]]、[[都昌|都昌]]和[[洪州|洪州]][[建昌|建昌]]3县置[[南康军|南康军]]，[[天禧|天禧]]四年江州、南康军同属[[江南东路|江南东路]]，[[绍兴|绍兴]]元年江州隶[[江南西路|江南西路]]，南康军隶江南东路。

* [[元|元]]为[[江州路|江州路]]和南康路，属[[江西行中书省|江西行中书省]]。元末[[陈友谅|陈友谅]]称帝，以江州为都。

* [[明|明]]改江州路和南康路为[[九江府|九江府]]和[[南康府|南康府]]，从此九江之名基本固定。[[清|清]]沿明制。

* 1861年，九江根据《[[天津条约|天津条约]]》和《[[北京条约|北京条约]]》开辟为[[商埠|通商口岸]]，并在城西长江和[[甘棠湖|甘棠湖]]之间的狭长地段设立了[[九江英租界|九江英租界]]，面积150亩（1927年中国收回）。开埠不久，作为[[怡和洋行|怡和]]、[[太古洋行|太古]]轮船在江西唯一的停靠港，九江迅速取代[[樟树镇|樟树镇]]、[[吴城镇|吴城镇]]成为江西全省贸易的枢纽。发展为中国最著名的“[[四大米市|四大米市]]”、“[[四大茶市|四大茶市]]”之一。19世纪末，[[庐山|庐山]]山顶的[[牯岭|牯岭]]发展为著名的避暑地，每年夏天吸引众多中外人士上山避暑，[[民国|民国]]时期甚至号称中国的'''夏都'''。

* 1912年废府，各县直隶于省。1914年设浔阳道，德化县因与[[福建|福建]][[德化县|德化县]]同名，改为九江县。1917年设九江市，1936年又改为九江县。同年把江北部分（今黄梅下乡地区）划归[[黄梅县|黄梅县]]。

* 1949年5月7日九江解放，7月19日设九江专员公署，同年析[[九江县|九江县]]城区设县级[[九江市_(县级)|九江市]]；

* 1980年3月28日九江市从九江地区划出，升为省辖市，同年5月20日市设庐山区、浔阳区、郊区三区<ref>[http://www.xzqh.org/html/show/jx/10392.html 1980年江西省行政区划]</ref>，1983年7月27日[[地市合并|地市合并]]<ref>[http://www.xzqh.org/html/show/jx/10395.html 1983年江西省行政区划]</ref>。

* 2010年9月，设立县级[[共青城市|共青城市]],原德安县茶山街道办事处、甘露镇、金湖乡，永修县江益镇和燕坊镇坪塘村、燕坊村，星子县苏家垱乡、泽泉乡划归共青城市管辖<ref>[http://www.jiujiang.gov.cn/zjjj/xqjj/201506/t20150622_1300571.htm 九江市政府门户网站-活力四射共青城市]</ref>。

* 2015年5月，市政府由浔阳区长虹大道66号搬迁到八里湖大道166号<ref>[http://epaper.jjxw.cn/xywb/html/2015-04/09/content_394139.htm 市民服务中心20日开始搬迁]</ref>

* 2016年5月庐山区更名为[[濂溪区|濂溪区]]，撤星子县设立县级[[庐山市|庐山市]]<ref>[http://www.jiujiang.gov.cn/zjjj/jjgk/lsyg/201506/t20150613_1216637.htm 九江市政府门户网站-历史沿革]</ref>。

* 2017年8月撤九江县，设立[[柴桑区|柴桑区]]<ref>[http://www.jiujiang.gov.cn/zjjj/jjgk/lsyg/201506/t20150613_1216637.htm 九江市政府门户网站-历史沿革]</ref>。

== 地理 ==
九江为江西北部门户。

北隔[[长江|长江]]、[[幕阜山|幕阜山]]与[[安徽|安徽]]、[[湖北|湖北]]相邻。西为[[罗霄山|罗霄山]]之余脉，与[[湖南|湖南]]接壤。

西部和东部多山，在永修、武宁境内建有[[柘林水库|柘林水库]]，为江西最大水库。中部为[[鄱阳湖平原|鄱阳湖平原]]和[[鄱阳湖|鄱阳湖]]区，在九江市区和鄱阳湖之间为庐山。

九江市行政区域内有人口为472万人，其中87万为城市居民。人口密度为235人／km²。99.8%的人口为[[汉族|汉族]]。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]总数为4728763人<ref>{{cite book |author=九江市统计局|title=《九江市2010年第六次全国人口普查主要数据公报》}}</ref>，同2000年[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加了217199人，增长4.81%，平均每年增加21720人，年平均增长率为0.47%。其中，男性为2437330人，占总人口的51.54%；女性为2291433人，占总人口的48.46%。人口性别比（以女性为100）为106.37。0—14岁的人口为987185人，占总人口的20.88%；15—64岁的人口为3388207人，占总人口的71.65%；65岁及以上人口为353371人，占总人口的7.47%。

=== 语言 ===
九江市区以及瑞昌大部所讲方言属于官话中的[[江淮官话|江淮官话]][[黄孝片|黄孝片]]，是一种北方方言，其余的县市属于[[赣方言|赣方言]]。

== 经济 ==
航运、炼油、旅游、化肥是九江市的主要经济产业。

== 旅游 ==
[[File:九江琵琶亭白居易石像.JPG|thumb]]
[[File:九江市锁江楼古塔.JPG|thumb]]
[[File:Lushan_White_Lotus_Grotto_Academy.JPG|thumb]]
[[File:东林寺.JPG|thumb]]

* 九江市拥有丰富的旅游资源，集名山（[[庐山|庐山]]）、名川（[[长江|长江]]）、名湖（[[鄱阳湖|鄱阳湖]]）于一身。

* 市内有风姿绰约的[[甘棠湖|甘棠湖]]，有传说中[[三国|三国]][[周瑜|周瑜]]操练水军的点将台——[[烟水亭|烟水亭]]，有[[白居易|白居易]]创作千古之作《[[琵琶行|琵琶行]]》的[[琵琶亭|琵琶亭]]，有[[宋江|宋江]]题反诗的[[浔阳楼|浔阳楼]]。

* 有“[[匡庐奇秀甲天下山|匡庐奇秀甲天下山]]”之称的庐山是世界文化、自然遗产，山下有[[李白|李白]]名句“飞流直下三千尺，疑是银河落九天”所吟咏的秀峰风景区，有被称为“海内第一书院”的[[白鹿洞书院|白鹿洞书院]]，[[朱熹|朱熹]]曾在此讲学。还有[[佛教|佛教]][[净土宗|净土宗]]开源之地的[[东林寺|东林寺]]等。

* [[鄱阳湖|鄱阳湖]]有三分之二在九江境内。湖口有[[石钟山|石钟山]]、大孤山、小孤山胜景，并曾是[[朱元璋|朱元璋]]大败[[陈友谅|陈友谅]]、[[太平天国|太平天国]]军大败[[曾国藩|曾国藩]][[湘军|湘军]]的古战场。

* [[彭泽县|彭泽县]]有号称“可藏百万兵”的[[龙宫洞|龙宫洞]]，曲折奇险；[[庐山市|庐山市]]温泉密布；[[共青城市|共青城]]还有[[胡耀邦|胡耀邦]]陵园。

* 庐山西海风景名胜区，原为云居山-柘林湖风景区，国家级风景名胜区、国家森林公园、省级地质公园、全国三大样板丛林之一，地跨永修、武宁两县，总面积680平方公里，是集游览观光、休闲度假于一体的山岳湖泊型省级重点风景名胜区。
