'''中国九校联盟/C9联盟'''（简称'''C9'''）是[[中国大陆|中国大陆]]首个大学联盟，於2009年10月启动，联盟成员包括[[清華大學|清華大學]]、[[北京大學|北京大學]]、[[浙江大學|浙江大學]]、[[南京大學|南京大學]]、[[復旦大學|復旦大學]]、[[上海交通大學|上海交通大學]]、[[西安交通大學|西安交通大學]]、[[中國科學技術大學|中國科學技術大學]]、[[哈爾濱工業大學|哈爾濱工業大學]]9所高校。

==简介==
清華大學、北京大學、浙江大學、南京大學、上海交通大學、復旦大學、中國科學技術大學、哈爾濱工業大學、西安交通大學皆是985計划首批高校，C9聯盟也可視為是該计划的延伸部分。

C9中不隶属于[[中華人民共和國教育部|中華人民共和國教育部]]的两所大学：哈爾濱工業大學隶属于[[工业和信息化部|工业和信息化部]]（[[大部制改革|大部制改革]]前属[[国防科工委|国防科工委]]）并设立“一校三区”的“[[大哈工大|大哈工大]]”发展格局；中國科學技術大學隶属于[[中國科學院|中國科學院]]。

C9在[[中華人民共和國|中華人民共和國]]政府大力支持下取得了长足的进步。C9联盟基本代表了中国大陆的最顶尖高校。 另外，C9各高校从2010年开始实行本科生可以在联盟内部互相交流访学，和暑期学校交流。其中，清华和北大由于各自特殊原因，不再参加互访。

==历史渊源==
该大学联盟源于[[985工程|985工程]]9校的“[[一流大学建设系列研讨会|一流大学建设系列研讨会]]”。985工程是1998年5月开始的一项系统工程，旨在重点支持中国大陆的一批高校优先发展，成为世界知名的大学。985工程首批选定了9所大学作为重点支持，目标是“创建世界一流大学”；随后，985工程又先后增加了30所大学，总数达到39所，后增加的30所定位为“创建世界知名高水平大学”。

2003年，首批进入985工程的9所大学校长召开了首届“一流大学建设系列研讨会”，此后会议固定下来，每年召开一届。

在2009年，第七届研讨会于西安交通大学举行，会议最重要的议题就是建立“C9联盟”，旨在共享资源，加速发展。9所大学校长共同签署了《一流大学人才培养合作与交流协议书》。<ref>一流大学建设系列研讨会隆重举行：{{cite web |url=http://xjtunews.xjtu.edu.cn/zhxw/2009-10/1255082272d24515.shtml |title=存档副本 |accessdate=2009-10-24 |deadurl=yes |archiveurl=https://web.archive.org/web/20091015052825/http://xjtunews.xjtu.edu.cn/zhxw/2009-10/1255082272d24515.shtml |archivedate=2009-10-15 }}
</ref><ref>九校签订《一流大学人才培养合作与交流协议书》：{{cite web |url=http://xjtunews.xjtu.edu.cn/zhxw/2009-10/1255091987d24517.shtml |title=存档副本 |accessdate=2009-10-09 |deadurl=yes |archiveurl=https://archive.is/20120805225932/http://xjtunews.xjtu.edu.cn/zhxw/2009-10/1255091987d24517.shtml |archivedate=2012-08-05 }}</ref>

2014年11月7日，[[中国科学院大学|中国科学院大学]]（简称[[国科大|国科大]]）正式参与C9联盟活动。<ref>[http://news.nju.edu.cn/show_article_1_36935 中国科学院大学也介绍了各校在学位工作方面的举措]</ref>

==定位==
九校联盟（C9）成员大学的定位是建设“世界一流大学”。
