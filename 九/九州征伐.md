'''九州征伐'''發生於[[天正|天正]]14年（1586年）至15年（1587年），是[[丰臣秀吉|丰臣秀吉]]率领其同盟[[大名|大名]]攻略[[岛津家|岛津家]]，进而征服[[九州_(日本)|九州]]的战争。

== 起因 ==
[[大友宗麟|大友宗麟]]的軍隊因為[[島津氏|島津氏]]的崛起而節節敗退，只得向[[豐臣秀吉|豐臣秀吉]]求援。已经统一[[四国岛|四国岛]]的豐臣秀吉召喚[[黑田孝高|黑田孝高]]詢問對策，黑田認為这是平定九州的好機會，於是秀吉下令備戰，同時回信要求[[大友宗麟|大友宗麟]]繼續堅守等待援軍。

島津軍因對大友的戰爭失利而聯合[[有馬晴信|有馬晴信]]攻打[[龍造寺政家|龍造寺政家]]，[[龍造寺氏|龍造寺]]只剩下[[肥前國|肥前國]]三座城，令[[鍋島直茂|鍋島直茂]]乞求停戰。[[島津義弘|島津義弘]]不許，仍下令攻打龍造寺，[[島津家久|島津家久]]攻下水之江和勢福寺城。龍造寺走向滅亡之日將近，所以鍋島直茂請求龍造寺成為島津臣下，降伏於[[島津氏|島津氏]]，被免去一死。唇亡齒寒，不出十日，助攻龍造寺的有馬晴信也被島津順便滅掉，有馬晴信投降島津氏。

島津氏至此在九州只剩下大友家一个对手，而大友家的滅亡也只是時間上的問題。豐臣秀吉覺得島津氏過度猖狂，影響豐臣氏的政權，所以向島津氏發出交涉書信，命令九州兩勢力停止戰爭，服從豐臣政權領導。島津義久拒絕豐臣秀吉的提案並攻打[[筑前|筑前]]，致使豐臣秀吉以'''救援盟友'''（大友家）為名出兵九州。

== 毛利軍方面 ==
1586年9月，[[毛利輝元|毛利輝元]]以[[小早川隆景|小早川隆景]]和[[吉川元長|吉川元長]]為先鋒經[[長門國|長門國]]出佂[[豐前國|豐前國]]支援[[立花宗茂|立花宗茂]]攻打[[島津忠長|島津忠長]]和[[伊集院忠棟|伊集院忠棟]]的[[筑前|筑前]]方面軍。因為[[鍋島直茂|鍋島直茂]]趁機聯合[[有馬晴信|有馬晴信]]攻下[[肥後|肥後]]，形成夾擊[[筑前|筑前]]的形勢。1586年10月[[黑田孝高|黑田孝高]]由海路到達後形成三方包圍。

== 長宗我部軍方面 ==
1586年9月，以[[仙石秀久|仙石秀久]]為首[[長宗我部元親|長宗我部元親]]和兒子[[長宗我部信親|長宗我部信親]]同[[十河存保|十河存保]]渡海攻打[[豐後|豐後]]。在1586年12月12日早晨，兩軍在戶次川對峙，第一陣的仙石秀久一開始因為急進就先告潰敗，而第二陣的長宗我部信親勢遭到伏兵包圍而壞滅。長宗我部信親、十河存保戰死，而長宗我部元親的第三陣見到情況不妙後撤退，而仙石秀久因為這次戰敗而被豐臣秀吉流放到高野山。

== 龍造寺軍方面 ==
因為[[毛利輝元|毛利輝元]]入侵[[豐前|豐前]]引開了島津軍所以[[鍋島直茂|鍋島直茂]]趁機聯合[[有馬晴信|有馬晴信]]攻下[[肥後|肥後]]，形成夾擊[[筑前|筑前]]的形勢，1586年10月[[黑田孝高|黑田孝高]]由海路到達後形成三方包圍。

== 宗軍方面 ==
[[宗義智|宗義智]]讓出[[對馬島|對馬島]]讓豐臣駐軍，並需要提供[[資金|資金]]與[[糧草|糧草]]。

== 大友軍方面 ==
大友家除了大將[[立花宗茂|立花宗茂]]在各地征戰外，全部武將都籠城堅守，主公[[大友義鎮|大友義鎮]]病危由少主[[大友義統|大友義統]]臨時統領大友家。大友義鎮則堅守臼杵城。

== 豐臣軍方面 ==
=== 豐臣秀吉北九州方面軍 ===
1587年3月豐臣秀吉先鋒[[蒲生氏鄉|蒲生氏鄉]]、[[前田利長|前田利長]]用一日攻下[[秋月種實|秋月種實]]的岩石城，第二日攻下益富城，秋月種實投降豐臣秀吉。[[中川秀政|中川秀政]]、[[福島正則|福島正則]]、[[高山長房|高山長房]]到達後擊敗[[島津忠長|島津忠長]]和[[伊集院忠棟|伊集院忠棟]]，兩人降伏。至此[[筑前|筑前]]、[[筑後|筑後]]、[[肥前|肥前]]、[[豐前|豐前]]、[[豐後|豐後]]成為豐臣支配地。之後全軍入侵[[肥後|肥後]]，肥後領主[[島津忠辰|島津忠辰]]降伏。全軍於肥後會師。準備攻打[[薩摩|薩摩]]。

=== 豐臣秀長南九州方面軍 ===
1587年2月，豐臣秀長率八萬兵力在[[日向|日向]]登陸，將守在岸邊的[[島津家久|島津家久]]軍撃退，直接與[[島津義弘|島津義弘]]在日向交戰。因為島津義弘屢次發動奇襲，於是豐臣秀長下令實行[[堅壁清野|堅壁清野]]，以壓倒性的兵力將島津義弘的五千兵力團團圍住，用[[車輪戰|車輪戰]]牽制住島津義弘。但是卻被島津義弘突圍而出。

== 根白坂之戰 ==
{{main|根白坂之戰}}
1587年4月17日，[[島津義久|島津義久]]和[[島津義弘|島津義弘]]打算和豐臣方決戰於日向國根城坂，但是因為豐臣秀長軍一早挖好[[陷阱|陷阱]]建造板塀和空堀等軍事設施，射箭來阻礙島津方進攻。島津方始終衝不上板塀所在之處。在[[島津忠隣|島津忠隣]]、[[猿渡信光|猿渡信光]]戰死後島津方撤退。

== 九州國分 ==
1587年4月28日，[[島津義久|島津義久]]向豐臣秀吉表示降伏，[[剃髪出家|剃髪出家]]並改名為[[島津龍伯|島津龍伯]]。5月8日島津龍伯在[[肥後|肥後]][[泰平寺|泰平寺]]正式表示完全降伏。並由弟弟[[島津義弘|島津義弘]]繼承家督，而豐臣秀吉也針對九州的領地重新進行分配，曾經一度控制九州大部分地區的島津氏被大幅削減，但仍擁有[[薩摩|薩摩]]和[[大隅|大隅]]兩國，以及日向國的諸縣、真幸院等地。

其中，小早川隆景由平定四國後獲得的伊予移封至[[筑前|筑前]]、筑後兩國的大部分跟[[肥前|肥前]]一郡，共37萬石的領地，其弟小早川秀包也獲得筑後得到7萬5千石的領地。而在大友家方面，因為大友宗麟拒絕秀吉加封日向一國作為封賜，所以其子大友義統僅獲得[[豐後|豐後]]一國的安堵，原為大友家臣的[[立花宗茂|立花宗茂]]因出色武勇得到秀吉賞識，加封[[筑後|筑後]]13萬2千石的領地，獨立為一家大名，[[肥前|肥前]]的龍造寺政家、大村喜前、松浦鎮信等人及[[對馬|對馬]]國的宗義智，也在臣服秀吉後獲得本領的安堵。而豐臣秀吉的家臣中，黑田孝高得到了[[豐前|豐前]]六郡12萬5千石(後增封至17萬石)的領地、毛利勝信則獲得豐前小倉6萬石。

肥後國的大部分則交給了原本支配越中國的[[佐佐成政|佐佐成政]]，當地豪族天草五人眾也獲得安堵，另外在相良氏家臣[[深水長智|深水長智]]的奔走下，相良賴房也重新獲得肥後人吉2萬石的安堵。但因為佐佐成政檢地失力，引發嚴重的[[肥後國人一揆|肥後國人一揆]]，佐佐成政被問罪賜死，在平定當地一揆後，原屬佐佐成政的領地一分為二，切割成加藤清正擁有肥後北半國19萬5千石，小西行長則擁有肥後南半國20萬石，但小西行長在進行檢地時，又遇到天草五人眾的抵制（天草國人一揆），最後與加藤清正平定天草國人一揆，徹底掌握肥後。

曾在北九州積極抵抗豐臣軍的[[秋月種實|秋月種實]]在戰敗後，由筑前、筑後遷調至[[日向|日向]]高鍋3萬石，秋月種實的次子高橋元種因較早投降，移至日向縣5萬石。過去被島津氏打敗的伊東氏，因伊東祐兵早就成為豐臣秀吉的家臣，並曾參與山崎之戰等戰役，順利回歸舊領，擁有日向飫肥3萬6千石的領地。

== 影響 ==
豐臣秀吉將九州平定以後，鞏固了豐臣的威權統治，更獲得了地利與外國通商，更展開大規模的檢地，在[[肥後|肥後]]的[[佐佐成政|佐佐成政]]亦進行檢地，但是遭到部份國人反對，引發肥後國人一揆。佐佐成政被處死後，北肥後歸[[加藤清正|加藤清正]]管轄，南肥後歸[[小西行長|小西行長]]、[[相良賴房|相良賴房]]管轄，但仍爆發了天草國人一揆，最後被小西行長跟加藤清正合力消滅，而豐前國也有城井鎮房跟野中鎮兼等勢力拒絕接受黑田家統治的反抗動作，最後黑田孝高暗殺了城井鎮房後，平定這波動亂。
