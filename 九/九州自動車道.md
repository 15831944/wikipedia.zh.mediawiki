'''九州自動車道'''（{{jpn|j=九州自動車道|hg=きゅうしゅうじどうしゃどう|rm=Kyūshū jidōshadō|f=y}}）是[[日本|日本]][[福岡縣|福岡縣]][[北九州市|北九州市]][[門司區|門司區]][[門司交流道|門司交流道]]至[[鹿兒島縣|鹿兒島縣]][[鹿兒島市|鹿兒島市]][[鹿兒島交流道|鹿兒島交流道]]，總長346.3公里的[[日本高速公路|高速公路]]（[[高速自動車國道|高速自動車國道]]），簡稱'''九州道'''（{{lang|ja|九州道／きゅうしゅうどう}}，{{Lang|en|KYUSHU EXPWAY}}）。[[國土開發幹線自動車道|國土開發幹線自動車道]]的法定路線名為[[九州縱貫自動車道|九州縱貫自動車道鹿兒島線]]（[[門司交流道|門司交流道]] - [[蝦野系統交流道|蝦野系統交流道]]間與九州縱貫自動車道宮崎線重複）。

此外，門司交流道 - [[福岡交流道_(福岡縣)|福岡交流道]]間被指定為 {{AHN-AH|1}} [[亞洲公路1號線|亞洲公路1號線]]。高速公路路線編號被編為 {{Ja Exp Route Sign|E3}}。

== 歷史 ==
* 1971年6月30日 : 植木IC-熊本IC間通車。
* 1972年10月21日 : 南關IC-植木IC間通車。
* 1973年11月16日 : 鳥栖IC-鳥栖JCT-南關IC間通車，與長崎自動車道（當時為九州道鳥栖IC-鳥栖JCT間）連接。
* 1973年12月13日 : 加治木IC-薩摩吉田IC間通車。
* 1975年3月13日 : 古賀IC-鳥栖JCT間通車。
* 1976年3月4日 : 蝦野IC-蝦野JCT間通車（當時為宮崎道蝦野IC-蝦野JCT間），同日與宮崎自動車道連接。
* 1976年11月26日 : 熊本IC-御船IC間通車。
* 1976年11月29日 : 溝邊鹿兒島空港IC-加治木IC間通車。
* 1977年7月21日 : 若宮IC-古賀IC間通車。
* 1977年11月15日 : 薩摩吉田IC-鹿兒島北IC間通車。
* 1978年12月15日 : 御船IC-松橋IC間通車。
* 1979年3月8日 : 八幡IC-若宮IC間通車。與[[山口縣|山口縣]]山口IC起點之[[收費道路|一般收費道路]][[北九州高速4号線|北九州道路、北九州直方道路]]經由熊本縣松橋IC與[[高速道路|高速道路]]連結。
* 1980年3月12日 : 松橋IC-八代IC間通車。
* 1980年3月22日 : 栗野IC-溝邊鹿兒島空港IC間通車。
* 1981年10月1日 : 蝦野JCT-栗野IC間通車。
* 1984年3月27日 : 門司IC-小倉東IC間通車，與關門橋直通。
* 1987年2月5日 : 於鳥栖JCT與大分自動車道連接。
* 1988年3月29日 : 鹿兒島北IC-鹿兒島IC間通車。
* 1988年3月31日 : 小倉東IC-八幡IC間通車。
* 1988年10月19日 : 於鹿兒島IC與南九州自動車道連接。
* 1989年12月7日 : 八代IC-人吉臨時出入口間通車。
* 1992年3月25日 : 於加治木IC與隼人道路連接。
* 1993年3月31日 : 新門司IC通車。
* 1995年7月27日 : 人吉臨時出入口-蝦野IC間通車，'''全線通車'''。由此自[[青森市|青森]]-鹿兒島間（約2170km）的高速道路全部連結完成。人吉主線收費站停用，人吉臨時出入口封閉。
* 1997年7月14日 : 除肥後隧道外，八代IC-人吉IC間拓寬為4車道。
* 1997年11月28日 : 鳥栖JCT-久留米IC間拓寬為6車道。
* 1998年3月23日 : 廣川IC通車。
* 1998年3月31日 : 筑紫野IC通車。
* 1998年4月20日 : 八代JCT開設，與南九州西回自動車道連接。
* 1999年3月24日 : 益城熊本空港IC通車。
* 1999年4月14日 : 肥後隧道前後區間拓寬為4車道。
* 2001年3月24日 : 鳥栖JCT九州道下行方向至長崎道長崎方面直通連絡橋「{{lang|ja|サガンクロス橋}}」通車。
* 2001年4月26日 : 蝦野停车区-蝦野IC間拓寬為4車道。
* 2001年12月19日 : 加治木JCT通車，與加治木IC間連接部分廢除，並與東九州自動車道連接。
* 2004年12月12日 : 人吉IC-蝦野IC間全線4車道通車，'''全線4車道以上通車'''。
* 2004年12月18日 : 須惠停车区智能IC社會實驗開始。
* 2006年2月26日 : 北九州JCT通車，與東九州自動車道連接。
* 2006年10月1日 : 須惠SIC開始啟用。
* 2009年3月29日 : 三山柳川IC通車。
* 2011年2月19日 : 鞍手IC通車。
