'''九龍區專線小巴12線'''是往來[[白田邨|白田邨]]及[[旺角東站|旺角東站]]的一條專線小巴路線，由中運有限公司承辦，主要方便[[白田邨|白田邨]]、[[大坑東邨|大坑東邨]]和[[南山邨|南山邨]]一帶居民往來旺角。<ref name="td12">{{cite web|url=http://hketransport.td.gov.hk/ris_page/get_gmb_detail.php?route_id=2110&lang=TC|language=中文|title=九龍區專線小巴路線第12號|publisher=運輸署|date=|accessdate=|archive-url=https://web.archive.org/web/20180218145514/http://hketransport.td.gov.hk/ris_page/get_gmb_detail.php?route_id=2110&lang=TC|archive-date=2018-02-18|dead-url=no}}</ref>

'''九龍區專線小巴12A線'''是往來[[大同新邨|大同新邨]]及[[旺角東站|旺角東站]]的一條專線小巴路線，由中運有限公司承辦，主要方便[[大同新邨|大同新邨]]一帶居民往來旺角。<ref name="td12A">{{cite web|url=http://hketransport.td.gov.hk/ris_page/get_gmb_detail.php?route_id=2111&lang=TC|language=中文|title=九龍區專線小巴路線第12A號|publisher=運輸署|date=|accessdate=|archive-url=https://web.archive.org/web/20180218145729/http://hketransport.td.gov.hk/ris_page/get_gmb_detail.php?route_id=2111&lang=TC|archive-date=2018-02-18|dead-url=no}}</ref>

'''九龍區專線小巴12B線'''是往來[[港灣豪庭|港灣豪庭]]及[[旺角東站|旺角東站]]的一條專線小巴路線，由中運有限公司承辦，主要方便[[港灣豪庭|港灣豪庭]]和大同新邨一帶居民往來旺角。<ref name="td12B">{{cite web |url=http://hketransport.td.gov.hk/ris_page/get_gmb_detail.php?route_id=2443&lang=TC |language=中文 |title=九龍區專線小巴路線第12B號 |publisher=運輸署 |date= |accessdate= |archive-url=https://web.archive.org/web/20180218145733/http://hketransport.td.gov.hk/ris_page/get_gmb_detail.php?route_id=2443&lang=TC |archive-date=2018-02-18 |dead-url=no }}</ref>

'''九龍區專線小巴12S線'''是往來[[帝柏海灣|帝柏海灣]]（[[奧海城|奧海城]]二期）及[[旺角東站|旺角東站]]的一條循環專線小巴路線，由中運有限公司承辦，主要方便[[帝柏海灣|帝柏海灣]]和大同新邨一帶居民往來旺角和[[油麻地|油麻地]]。<ref name="td12S">{{cite web |url=http://hketransport.td.gov.hk/ris_page/get_gmb_detail.php?route_id=2121&lang=TC |language=中文 |title=九龍區專線小巴路線第12S號 |publisher=運輸署 |date= |accessdate= |archive-url=https://web.archive.org/web/20180218145647/http://hketransport.td.gov.hk/ris_page/get_gmb_detail.php?route_id=2121&lang=TC |archive-date=2018-02-18 |dead-url=no }}</ref>

== 歷史 ==
'''12'''線於1979年7月16日投入服務，當時來往白田及[[大角咀碼頭|大角咀碼頭]]，1980年3月23日起大角咀總站遷往[[大同新邨|大同新邨]]<ref>〈九龍兩專綫小巴調整路綫〉，香港政府新聞公報，1980-03-21</ref>，兩年後因客量上升而增加班次<ref>《白田與大同新邨間專線小巴增加班次》，大公報，1982-02-22</ref>。1985年來回程加經新落成的[[旺角東站公共運輸交匯處|旺角東站公共運輸交匯處]]。<ref>〈旺角站交通交匯處啟用〉，載《鐵路之聲》第十六期（香港：九廣鐵路公司公共關係處），1985年10月。</ref>

1992年2月3日12線增設來往旺角火車站及[[大同新邨|大同新邨]]的短途班次，後特別班次改名為'''12A'''線。

2001年6月1日，12線縮短至旺角火車站，以騰出資源以便開辦12B線（[[港灣豪庭|港灣豪庭]]↔旺角火車站）<ref>{{cite web|url=http://www.681busterminal.com/version2/index.php?fi=a201&id=108|title=681巴士總站：九龍區專線小巴12線|website=|access-date=2018-11-18|archive-url=https://web.archive.org/web/20190221125941/http://www.681busterminal.com/version2/index.php?fi=a201&id=108|archive-date=2019-02-21|dead-url=no}}</ref>，而12B線亦於2003年投入服務。

循環來往大角咀（[[奧海城|奧海城]]）及旺角東站的[[九龍區專線小巴21K線|九龍區專線小巴21K線]]，營辦商國松汽車於2009年決定放棄營運<ref>[http://www.hkitalk.net/HKiTalk2/redirect.php?tid=604962&goto=lastpost#lastpost 九龍綠小21K即將於2011年4月10日換新營辦商]，hkitalk.net，文章#13</ref>，運輸署於2011年4月10日起，將21K線重新招標<ref>[http://www.gld.gov.hk/egazette/pdf/20101428/cgn201014284477.pdf 第4477號公告：〈道路交通 (公共服務車輛) 規例 (第374D章)--九龍及新界的公共小巴 (專線) 路線組別〉] {{Wayback|url=http://www.gld.gov.hk/egazette/pdf/20101428/cgn201014284477.pdf |date=20181101030512 }}，香港特別行政區政府憲報第14卷第28期，2010年7月16日。</ref>，由[[誠利有限公司|誠利有限公司]]投得專營權<ref name="TD2011">[http://www.td.gov.hk/tc/traffic_notices/index_id_25565.html 運輸署交通通告：〈九龍專線小巴第21K號(大角咀(櫻桃街) 至 旺角東站 ( 循環線 )) 新營辦商〉] {{Wayback|url=http://www.td.gov.hk/tc/traffic_notices/index_id_25565.html |date=20140226212843 }}，2011-04-07</ref>。但由於長期經營虧蝕，營辦商通知運輸署擬於2012年9月15日停止營辦21K線。運輸署曾物色其他營辦商以繼續營運21K線，但無人問津<ref>[http://www.districtcouncils.gov.hk/ytm/doc/2012_2015/common/committee_meetings_doc/TTC/2974/YTM_TTC_2012_37_TC.pdf 為九龍專綫小巴路綫第21K號(大角咀(櫻桃街)至旺角東站(循環綫))遴選新營辦商]，油尖旺區議會交通運輸委員會文件第37/2012號</ref>。21K線最終於2012年10月15日取消服務，由走線相若的'''12S'''線代替。<ref>[http://www.td.gov.hk/tc/traffic_notices/index_id_31297.html 運輸署交通通告：〈行走大角咀(櫻桃街)至旺角東站的專綫小巴服務〉] {{Wayback|url=http://www.td.gov.hk/tc/traffic_notices/index_id_31297.html |date=20160304194928 }}，2012-10-09</ref>
== 服務時間及班次 ==
12線於星期一至六的雙方服務時間為早上6時正至晚上9時正，每20-30分鐘一班；假日雙方服務時間為早上11時正至晚上9時正，每30分鐘一班車。<ref name="td12"/>

12A和12B線雙方方向在每日上午6時正開始服務，而去程方向終止服務的時間為晚上11時20分，回程則是晚上零時，12A線兩個方向的班次皆是7-9分鐘一班，而12B線則為6-8分鐘一班。此外，12B在平日早上繁忙時間設有特別班次只由大角咀前往旺角站，每7-8分鐘一班。<ref name="td12A"/><ref name="td12B"/>

12S線之所有班次皆由[[帝柏海灣|帝柏海灣]]（[[奧海城|奧海城]]二期）開出，在每日上午6時半至下午11時20分提供服務，班次為10-15分鐘一班。<ref name="td12S"/>

== 客量 ==
12A和12B線是大角咀居民往返旺角的主要途經，因此該兩路線客量極高，在繁忙時間例必出現滿座的情況，導致中途站的候車的乘客根本無法上車，令大角咀的居民甚為不滿。<ref>[https://www.districtcouncils.gov.hk/ytm/doc/2012_2015/tc/committee_meetings_doc/TTC/2966/YTM_TTC_2013_19_TC.pdf 要求加強專線小巴12A及12B服務 延長下午留座時間，並將相關安排伸延至周六、日]，油尖旺交通運輸委員會第19/2013號文件</ref>

可是，12A系路線曾經遇上一段「黑暗時期」。2014年「[[雨傘運動|雨傘運動]]」爆發，旺角[[亞皆老街|亞皆老街]]被示威者所佔，路線須改道[[太子道西|太子道西]]來往大角咀，導致車程延長之餘，亦失去了[[朗豪坊|朗豪坊]]一帶的乘客支持，客量急降，導致司機的薪金大減，路線嚴重虧蝕。營運者感經營困難，遂向運輸署申請加價<ref>[http://www.mingpaocanada.com/VAN/htm/News/20141112/HK-gca1_r.htm 稱佔旺損生意 小巴促早批加價 涉4路線 建議增幅一成] {{Wayback|url=http://www.mingpaocanada.com/VAN/htm/News/20141112/HK-gca1_r.htm |date=20181118002058 }}，明報，2014年11月12日</ref>。最後運輸署批准12A系路線於2015年1月正式加價。<ref>{{cite web |url=http://www.td.gov.hk/tc/traffic_notices/index_id_40852.html |language=中文 |title=調整九龍專線小巴第12A、12B及12S號線收費 |publisher=香港特別行政區政府運輸署交通通告 |date=2015年1月14日 |accessdate= |archive-url=https://web.archive.org/web/20160101020044/http://www.td.gov.hk/tc/traffic_notices/index_id_40852.html |archive-date=2016年1月1日 |dead-url=no }}</ref>
