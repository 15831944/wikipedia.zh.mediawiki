'''荣耀V10'''是由[[华为公司|华为公司]]设计和销售的[[智能手机|智能手机]]，为荣耀系列手机，也是[[荣耀V9|荣耀V9]]的换代产品，于2017年11月28日发布。

== 配置 ==
荣耀V10搭载了[[海思半导体|麒麟]]970芯片，设有后置双摄像头、前置摄像头、红外识别和前置指纹识别，并引入了长宽比18:9的5.99英寸屏幕，具备NFC功能<ref name="engadget">{{cite web |title=荣耀 V10 的正面既有 18 : 9 屏幕也有指纹识 |url=https://cn.engadget.com/2017/11/28/honor-v10/ |website=Engadget 中国版 |access-date=2018-10-06 |archive-url=https://web.archive.org/web/20171230200037/https://cn.engadget.com/2017/11/28/honor-v10/ |archive-date=2017-12-30 |dead-url=no }}</ref>。继华为[[Mate_10|Mate 10]]后，该款手机是第二款搭载麒麟970芯片的手机<ref>{{cite news |title=拍照游戏看视频更畅快·荣耀V10AI功能抢先玩 |url=http://www.sinchew.com.my/node/1717122 |work=星洲网 Sin Chew Daily |date=2018-01-07 |language=zh-hans}}</ref>。该手机有四种配色：幻夜黑、魅丽红、沙滩金、极光蓝<ref name="fabuhui">{{cite web |title=荣耀V10正式发布：全面屏+AI带来不一样的体验 |url=http://tech.sina.com.cn/mobile/n/n/2017-11-28/doc-ifypathz6977865.shtml |website=tech.sina.com.cn |access-date=2018-10-06 |archive-url=https://web.archive.org/web/20171201032136/http://tech.sina.com.cn/mobile/n/n/2017-11-28/doc-ifypathz6977865.shtml |archive-date=2017-12-01 |dead-url=no }}</ref>。

== 功能 ==
该手机因其摄像功能的软件调教、[[智能个人助理|智能个人助理]]和“随行翻译”功能的引入，被宣传为“AI智能手机”<ref name="fabuhui" />。

2018年6月，华为宣布将为该机型适配[[GPU_Turbo|GPU Turbo]]和AI识图软件优化功能，宣称前者能优化手机上的游戏体验和能耗<ref>{{cite web |title=荣耀V10全面开放升级GPU Turbo技术，手机性能火力全开_中华网 |url=https://tech.china.com/article/20180629/kejiyuan0531160102.html |website=tech.china.com}}</ref>。2018年9月，华为为该机型适配了[[微信支付|微信支付]]的指纹支付功能，并确保将指纹数据存储于本地<ref>{{cite web |title=华为荣耀将支持微信指纹支付 数据存储本地安全系统 |url=https://tech.sina.com.cn/t/2018-09-21/doc-ihkhfqnt4273353.shtml |website=tech.sina.com.cn |access-date=2018-10-06 |archive-url=https://web.archive.org/web/20180921064026/https://tech.sina.com.cn/t/2018-09-21/doc-ihkhfqnt4273353.shtml |archive-date=2018-09-21 |dead-url=no }}</ref>。2018年10月，华为对欧洲市场的V10手机推送了基于[[Android|Android]] 9的[[EMUI|EMUI]] 8.1更新<ref>{{cite web |title=欧洲荣耀V10开推Android 9 Pie：带来GPU Turbo 2.0 - 荣耀,荣耀V10,GPU,Turbo,安卓9 - IT之家 |url=https://www.ithome.com/0/386/555.htm |website=www.ithome.com}}</ref>。

== 销售与推广 ==
该机型于2017年11月28日发布销售，代言人为[[赵丽颖|赵丽颖]]、[[孙杨|孙杨]]和[[胡歌|胡歌]]。在[[中国大陆|中国大陆]]地区，该手机全网通版发布时的售价为：4GB随机内存+64GB内置存储版本（标配版）为2699元[[人民币|人民币]]，6GB随机内存+64GB内置存储版本（高配版）为2999元人民币，6GB随机内存+128GB内置存储版本（尊享版）为3499元人民币<ref name="fabuhui" />。

== 评价 ==
[[Engadget|Engadget]]的作者认为，该手机的前置指纹识别可能会增加解锁、点按操作时重心不稳摔机的风险，但也能吸引到很多厌恶后置指纹识别的消费者<ref name="engadget" />；同时，该手机搭载了长宽比18:9的屏幕，但前置指纹识别所在的手机下部较为细长，故没有破坏手机整体的观感；而后置主相机组突出过于明显，日间成像的风格更偏真实，暗光拍摄因嵌入式神经网络处理器（[[嵌入式神经网络处理器|NPU]]）加持而让成像效果达到平衡<ref>{{cite web |title=荣耀 V10 评测：按部就班，稳扎稳打 |url=https://cn.engadget.com/2018/01/13/honor-v10-review/ |website=Engadget 中国版 |access-date=2018-10-06 |archive-url=https://web.archive.org/web/20180115205359/https://cn.engadget.com/2018/01/13/honor-v10-review/ |archive-date=2018-01-15 |dead-url=no }}</ref>。而[[太平洋电脑网|太平洋电脑网]]认为，该款手机配色抢眼、质感出色、整体性强、单手操作舒适易用，提供Type-C接口和3.5英寸音频接口，有快速充电功能，显得考虑周全，设计优秀<ref>{{cite web |title=荣耀V10上手评测：AI芯片加持，还有全面屏双摄 |website=太平洋电脑网 |url=http://mobile.pconline.com.cn/1039/10394427.html |date=2017-11-28 |access-date=2018-10-06 |archive-url=https://web.archive.org/web/20180312053000/http://mobile.pconline.com.cn/1039/10394427.html |archive-date=2018-03-12 |dead-url=no }}</ref>。

2018年1月，该款手机在[[CES|CES]]上获得包含[[PCMag|PCMag]]最佳手机奖在内的九个奖项<ref>{{cite web |url=http://www.honor.cn/news/sharedetail/id/115/source/news/ |title=荣耀V10斩获9项CES殊荣 |website=荣耀官网| date=2018-01-23}}</ref>。
