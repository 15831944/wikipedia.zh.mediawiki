'''类型电台'''或'''类型化电台'''（{{Lang-en|'''Radio Format'''}}），有时也称为“'''类型节目'''”（{{lang|en|'''Programming Format'''}}），描述的是某个[[广播电台|广播电台]]节目内容的整体性质归属。类型电台淡化了电台节目之间的区别，为特定的受众提供特定的节目内容。<ref name="wangli">{{cite web |author1=王丽 |title=类型化电台与广播改革 |url=http://media.people.com.cn/GB/40628/5604952.html |website=[[人民网|人民网]] |publisher=新闻前哨 |accessdate=2018-08-03}}</ref><ref name="taiwan">{{cite web |title=台湾音乐电台的类型化发展探析 |url=http://gmgcri.china.com/industrial-info/xq/9348/20150506/366257.html |website=国广环球传媒控股有限公司 |publisher=微传媒 |accessdate=2018-08-03}}</ref> 在某些广播频段受到监管的国家，类型电台或因其播送特定内容而更容易获得播出许可。<ref>{{cite web |title=About Radio Formats |url=https://www.ofcom.org.uk/__data/assets/pdf_file/0016/90160/About-radio-formats.pdf |publisher=[[通讯管理局|通讯管理局]] |accessdate=2018-08-03}}</ref>

类型电台最早出现美国，而最早出现的类型电台则是{{Link-en|音乐电台|Music radio}}。<ref name="jiangli">{{cite web |author1=江俐 |title=音乐电台类型化的“拿来”与再造 |url=http://media.people.com.cn/n/2013/0116/c354358-20223073.html |website=[[人民网|人民网]] |publisher=试听界 |accessdate=2018-08-03}}</ref> 时至今日，类型电台的概念常常被电台所有者或运营方当作一种营销策略，同时电台的所属类型也会随市场的变动而变化。<ref>{{cite news |url=http://radiostationworld.com/directory/Radio_Formats/ |title=Radio Formats |work=RadioStationWorld |date=2010-01-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20100102161551/http://radiostationworld.com/directory/Radio_Formats/ |archivedate=2010-01-02 |accessdate=2018-08-03 }}</ref> {{Link-en|音乐电台|Music radio}}、{{Link-en|黄金时代电台|Golden Age of Radio}}、{{Link-en|全新闻电台|All-news radio}}、{{Link-en|体育电台|Sports radio}}、{{Link-en|谈话电台|Talk radio}}和{{Link-en|气象电台|Weather radio}}都是不同的类型电台运营类别，而这些类别还可以再进一步的细分。<ref name="douban">{{cite web |author1=方颂先 |title=纽约市广播电台的类型化和节目构成 |url=https://www.douban.com/note/354802085/ |website=[[豆瓣|豆瓣]] |accessdate=2018-08-03}}</ref>

中国第一家音乐类型电台是2002年开播[[中央人民广播电台音乐之声|中央人民广播电台音乐之声]]；而第一家全新闻类型电台则是2004年开播的[[东广新闻台|东广新闻资讯频率]]。<ref name="wangli" />

== 电台类型 ==
类型电台的类型是不断变化发展的，同时每种类型还可以再细分出诸多子类型。以下电台类型可能仅仅只是地区性电台，或通过卫星广播、[[网络广播|网络广播]]等形式播出。<ref name="guide">{{cite web |title=Radio Station Format Guide |url=http://www.nyradioguide.com/formats.htm |publisher=New York Radio Guide}}</ref>
