'''舊社站'''（正式站名）位於[[臺中市|臺中市]][[北屯區|北屯區]]，為[[臺中捷運|臺中捷運]][[臺中捷運綠線|綠線]]（[[烏日文心北屯線|烏日文心北屯線]]）之[[捷運車站|捷運車站]]<ref>[http://news.ltn.com.tw/news/local/paper/291933 新增3站…臺中捷運綠線確定設18站]，自由時報，2009-04-01</ref>。站名取自當地舊地名'''舊社'''<ref name="市政新聞 站名">{{Cite web |url=https://www.taichung.gov.tw/9962/1030128/post |title=台中捷運綠線18個車站站名出爐 搭乘友善度為命名首要原則 |publisher=[[臺中市政府|臺中市政府]] |work=[[臺中市政府交通局|臺中市政府交通局]] |date=2018-06-30 |archiveurl=http://archive.is/jVboW |archivedate=2018-07-01 }}</ref>。

== 車站概要 ==
車站位於[[松竹路|松竹路]]一段與敦富路口附近（松竹路一段上）；車站代碼為'''G3'''<ref>[http://www.dorts.gov.taipei/ct.asp?xItem=660998&CtNode=20285&mp=115001 臺北市政府捷運工程局 >捷運興建中路網 >臺中捷運烏日文心北屯線 >路線說明]，臺北市政府捷運工程局</ref>。

== 車站構造 ==
高架車站，兩座[[側式月台|側式月台]]，一個出入口。

=== 車站出口 ===
於松竹路與敦富路口附近設有單側出入口，以土地開發方式辦理。另預留未來增設對側出入口之銜接口。
