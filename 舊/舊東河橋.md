'''舊東河橋'''為一座[[臺灣|臺灣]][[臺東縣|臺東縣]][[東河鄉_(台灣)|東河鄉]]境內的一座[[橋梁|橋梁]]，該橋梁橫跨在[[馬武窟溪|馬武窟溪]]之間，由[[臺東縣政府|臺東縣政府]][[東河鄉公所|東河鄉公所]]以及[[東部海岸國家風景區|東部海岸國家風景區]]管理處共同管理與維護，屬於臺東縣政府所列管的[[臺東縣歷史建築|歷史建築]]之一，並被[[文建會|文建會]]獲選為「臺灣歷史建築百景」之一，也曾是名列於「臺東八景」之中。<ref>{{cite web |url=http://tour.taitung.gov.tw/zh-tw/Travel/ScenicSpot/321/%E6%9D%B1%E6%B2%B3%E6%A9%8B%E9%A2%A8%E6%99%AF%E5%8D%80 |title=臺東縣政府 臺東觀光網 東河橋風景區 }}</ref>

== 建設資訊 ==



東河橋建造於1930年，由[[臺灣日治時期|日治時期]][[臺灣總督府|臺灣總督府]]指派[[臺東廳|臺東廳]][[土木課|土木課]]吉田[[技師|技師]]進行規畫並建造，東河橋全長440尺，是可供汽車行駛的鐵索[[吊橋|吊橋]]。<ref>{{cite web |url=http://www.eastcoast-nsa.gov.tw/Portal/Content.aspx?lang=0&p=003030001&area=2&sid=36 |title=東部海岸國家風景區 東河橋 |deadurl=yes |archiveurl=https://web.archive.org/web/20141017055342/http://www.eastcoast-nsa.gov.tw/Portal/Content.aspx?lang=0&p=003030001&area=2&sid=36 |archivedate=2014-10-17 }}</ref>

而在橋建成之前吉田技師因病逝世，為紀念他的貢獻，當時的東河橋特此命名為'''「吉田橋」'''。

後來「吉田橋」因颱風損毀，而於1953年重建改採用[[混凝土|混凝土]]興建，為今日所見的東河橋，現今橋寬約4.7公尺，長約130公尺。原日治時代架設在溪中[[石灰岩|石灰岩]]上的中央橋塔，仍加以保留。<ref>{{cite web |url=http://www.walkingtaiwan.org/content/county/city_POI.asp?ids=492&jid=76&bid=8 |title=走讀臺灣 東河橋—海岸公路的重要橋樑 |deadurl=yes |archiveurl=https://web.archive.org/web/20141109180512/http://www.walkingtaiwan.org/content/county/city_POI.asp?ids=492&jid=76&bid=8 |archivedate=2014-11-09 }}</ref>

1992年(民國81年)，[[臺11線|臺11線]][[東部濱海公路|東部濱海公路]]正在進行拓寬工程，由於當時的東河橋寬度僅能單線通行，因此當地民眾建議在下游處另建新橋，東河鄉公所於下游處重新建了一座鋼架鐵橋，為現今的[[新東河橋|新東河橋]]，並將舊橋規劃為東河橋風景區，禁止任何汽機車輛進入，規劃為行人景觀橋，供遊客在橋上休憩賞景。<ref>{{cite web |url=http://www.xinmedia.com/t/scenery_reply.aspx?tagid=12486 |title=欣景點 東河橋風景區 }}</ref>
