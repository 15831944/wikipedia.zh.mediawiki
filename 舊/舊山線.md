{{舊山線RDT}}
'''舊山線'''是由[[臺灣鐵路管理局|臺灣鐵路管理局]]所經營的傳統鐵路線。廣義而言，「[[臺中線|臺中線]]（山線）雙軌工程」中，遭廢止的原有線路皆可稱為「舊山線」；不過在絕大多數語境係指最-{後}-一段停用、里程也最長的三義＝豐原段。本文援用此定義，詳見後述。

== 定義 ==
=== 廣義 ===
廣義的舊山線泛指1998年9月24日「臺中線雙軌工程」（以下簡稱「新山線」）完工後廢止的[[臺中線|臺中線（山線）]]舊線。自[[竹南車站|竹南]]至[[豐原車站|豐原]]皆為該項工程的改建範圍，不過多為截彎取直工程，改線幅度不大。

=== 狹義 ===
一般人大多引用舊山線的狹義定義：單指[[三義車站|三義]]＝[[后里車站|-豐原]]間遭廢止的路段。該路段為「新山線」完工後，最後一段也是里程最長的廢止區間。台鐵西部幹線海拔最高的車站－[[勝興車站|勝興]]一併停用，此外並將[[泰安車站_(台中線)|泰安車站]]向西遷至現今的高架站址。此段經多年爭論後，於2010年6月5日(週六)起以觀光方式恢復行駛，目前尚無定期客運列車行駛。此舊山線路段古蹟處處、翠嶺環繞、地勢雄偉、山青水碧，風景的確是美不勝收。

== 路線資料 ==
*營運管理：[[臺灣鐵路管理局|臺灣鐵路管理局]]
*路線長度：三義＝豐原間23.6公里
*[[軌距|軌距]]：1067 毫米
*車站數：4（含起訖車站；2010年6月）
*全線均為單線鐵路，其中僅167號誌站及[[勝興車站|勝興車站]]以及[[泰安舊站|泰安舊站]]交會。
*原為全線[[電氣化鐵路|電氣化]]，路線停用後已拆除所有相關設備。

*1903年9月：苗栗至三叉河（今 [[三義鄉_(台灣)|三義]]）間鐵道完工。
*1903年11月：三叉河以南施工期間的替代線—伯公坑運搬線開工，隔年5月完工。
*1905年5月：另一條替代線渦狀輕便線完工，與伯公坑運搬線共同擔負山線工程的運輸需求。
*1908年4月：三叉河至葫蘆墩（今[[豐原區|豐原]]）間鐵道完工，兩條替代用路線拆除。
*1935年4月21日：[[1935年新竹臺中地震|新竹、臺中州大地震]]，山線遭受地震破壞，山線一直到1938年7月15日才修復完成。當時的泰安(大安)站內尚有一列北上22次普通列車停靠該站，因地震破壞山線，該列車只好就地停放，直到三年後山線修復通車才被拖出修理。
*1979年：山線（[[臺中線|臺中線]]）全線完成電氣化(同時間第一代造橋隧道：[[見返坂隧道|見返坂隧道]]停用)(舊銅鑼灣隧道也在此時停用)。
*1987年：豐原＝成功段雙軌化完成。
*1990年4月24日：北上30次[[莒光號列車|莒光號]]列車於[[造橋車站|造橋]]＝[[豐富車站|豐富]]間的[[西坑尾隧道|西坑尾隧道]]（第二代造橋隧道）南口，因山崩土石滑動發生撞山重大出軌意外，造成2名司機員喪生。[[台鐵E200－E400型電力機車|E221]]報廢
*1991年11月15日：造橋＝豐富間[[134號誌站|134號誌站]]發生1006次[[自強號列車|自強號]]與1次莒光號列車邊撞的重大事故，造成30人喪生。相關單位因此決定將造橋＝豐富段改線，並開闢新線及新開鑿[[豐富隧道|豐富隧道]](第三代造橋隧道)，停用造橋隧道。
*1997年10月7日：豐原＝-{后里}-段雙軌化完成。並在1998年新舊山線切換後，舊大甲溪橋及九號隧道停用；舊線改為[[后豐鐵馬道|后豐自行車道]]。
*1998年9月18日：[[苗栗車站|苗栗]]＝[[南勢車站|南勢]]段切換新線，[[南勢北號誌站|南勢北號誌站]]裁撤、[[苗栗隧道|苗栗隧道]]（功維敘）停用。苗栗站南路線高架化並新開鑿苗南隧道。
*1998年9月23日：三義＝-{后里}-間的舊山線在最後列車（29次莒光號）於晚上通過勝興站後切換線路，隔天早上啟用新線。
*1998年10月11日：山線雙軌化工程通車典禮於[[苗栗車站|苗栗站]]舉行，並於10月16日進行列車改點，大幅縮短行車時間。
*2009年2月15日：[[臺灣鐵路管理局|臺灣鐵路管理局]]副局長[[張應輝_(台灣)|張應輝]]表示，預定6月辦理舊山線招商，由民間業者經營舊山線復駛。
*2010年5月25日:舊山線三義至（舊）泰安間試運轉，做為復駛的準備。
*2010年6月5日：舊山線三義至（舊）泰安間復駛，復駛編組為（舊）泰安←[[臺鐵CK124號蒸氣機車|CK124]] + 14CW8944（煤水車） + TPK32229（TP/TPK32200型客車） + TP32220 + TP32264 + TP32267 + TP32215 + [[臺鐵R100型柴電機車|R123]]→三義。
*2016年3月1日，配合[[苗栗縣政府|苗栗縣政府]]辦理「舊山線跨域亮點整備示範計畫」需要，停用勝興站至泰安站間路線。
*2016年7月15日:配合[[臺中市政府|臺中市政府]]之計畫「浪漫台三線」,將舊山線納入重新計畫復駛。
2023年12月:三義-后里(全線)以多元運具常態復駛

== 名景 ==
*[[三義車站|三義車站]]：1903年10月7日開業，原有木造車站則於1916年啟用。舊名「三叉河」，1955年3月1日改名為三義站。1995年9月配合新山線工程，木造站房拆遷至他處復原，目前保存於苗栗（小雪霸渡假村）。
*一號隧道：位於三義＝勝興間，長230公尺。三義＝勝興間路段為坡度千分之二十六的陡坡，是許多鐵道攝影者喜愛的取景地點。一號隧道以紅磚砌成；1970年代進行鐵路電氣化工程時，由於淨空不足，而將軌道面下挖，以增加空間架設電車線。
*[[勝興車站|勝興車站]]：位於[[苗栗縣|苗栗縣]]三義鄉勝興村；1907年開業時設「十六份信號場」，並不辦理客、貨運，只作為列車交會處。1930年4月1日改設「十六份停車場」並開辦客、貨運；1958年2月10日改名為勝興車站。本站仍在營運時，是台鐵路線的最高點，[[海拔|海拔]]402公尺；站房前立有前局長莫衡題字的台灣鐵路西部幹線最高點紀念碑。1981年8月22日10:09，[[遠東航空|遠東航空]]發生[[三義空難|三義空難]]（日本名作家[[向田邦子|向田邦子]]於此次空難中逝世）時，罹難者遺體即從本站運出。本站為木造站房，以依據[[風水|風水]]所作的特殊設計為特色（例如屋簷有[[八卦|八卦]]與尖茅）。
*二號隧道：興建於1905年，全長726公尺；往南為坡度千分之二十四點七的下坡。隧道北口有[[台灣日治時期|日治時代]][[後藤新平|後藤新平]]所題的「開天」兩字。
*[[167公里號誌站|167公里號誌站]]：位於台鐵[[縱貫線_(北段)|縱貫線]][[基隆車站|基隆]]起167公里處，因此得名。全長2.3公里，是1970年代鐵路電氣化時所設置的號誌站；設立原因是為了增加勝興＝泰安間長達8.6公里單線路段的[[路線容量|路線容量]]。號誌站的北邊有條特殊的分叉線，稱為[[折返線|折返線]]，台鐵所有路線中目前僅此一處。折返線為往南上坡千分之二十五，長250公尺。用途在於使無力上坡至勝興的火車可以倒退入折返線重新起步。號誌站的南邊另外設立[[安全側線|安全側線]]，可避免進入副正線交會，但[[制軔|制軔]]失靈的列車往南衝，與正線列車發生對撞而造成事故。安全側線是往南千分之二十五的上坡，長400公尺，利用坡度讓列車停下來以防止意外。安全側線旁有「故線路工夫組頭加藤勝三郎夫妻表彰碑」，以紀念道班工。
*[[魚藤坪斷橋|魚藤坪斷橋]]：又稱「'''龍騰斷橋'''」，位於三義鄉龍騰村，為[[1935年新竹臺中地震|1935年新竹臺中地震]]震毀後的橋墩遺跡；由紅磚所構成，與[[日本|日本]][[本州|本州]][[長野縣|長野縣]][[信越本線|信越本線]][[碓冰峠|碓冰嶺]]的眼鏡橋相似。震災後在旁邊新建一座水泥橋墩、鋼樑橋台的新橋取代。
*連續隧道：為三號、四號、五號、六號四座隧道構成的短隧道群。由於該段為直線，而且隧道接連設置，因此能夠在一端同時看到多個隧道。此段隧道為1935年大地震後改建的水泥隧道。
*鯉魚潭鐵橋：又稱「'''內社川鐵橋'''」，位於[[鯉魚潭水庫|鯉魚潭水庫]]的後池堰，北端連接六號隧道；南端連接七號隧道。橋樑在1935年的震災中一度毀壞；因兩端緊鄰隧道，於是在原地重建新橋墩，並改成上承式花樑。1970年代進行鐵路電氣化工程時，橋的兩端運用降低[[枕木|枕木]]厚度及挖除隧道頂一小塊的方式，騰出空間架設電車線。
*七號隧道：在1935年的震災中損毀，後來重建。北隧道口有[[後藤新平|後藤新平]]所題的「巨靈讓工」四字；南隧道口則有[[兒玉源太郎|兒玉源太郎]]所題的「一氣通」三字。
*[[大安溪|大安溪]]鐵橋：橋的北端連接七號隧道，南端離泰安站不遠。全長637公尺，有10孔KS-15花樑。是苗栗縣三義鄉與[[臺中市|臺中市]][[后里區|后里區]]的分界。
*[[泰安車站_(舊山線)|泰安車站]]：位於臺中市后里區泰安-{里}-。1910年12月1日，最早設「大安溪信號場」。1912年11月1日改設「大安溪停車場」；1954年改稱泰安車站。1998年新山線通車後，泰安站搬遷到距原址一公里左右的新線高架橋下，為台灣第一個高架車站；由於風大，月台還裝設欄杆。泰安站內還有座砲彈形的「震災復興紀念碑」，上面刻有日文碑文，以將1935年新竹臺中地震復建工程的概要敘述給後人。
*九號隧道：舊山線最長的隧道，長1269.5公尺。隧道北口題有「潛行不窒」四字；南口則題有「氣象雄深」四字。隧道南端連接大甲溪橋；北端離-{后里}-站不遠。
*[[大甲溪鐵橋|大甲溪鐵橋]]：長382公尺，有六座花樑。是[[后里區|后里區]]與[[豐原區|豐原區]]的分界。

== 保存情況 ==
*由於舊山線上保有[[勝興車站|勝興站]]與[[龍騰斷橋|龍騰斷橋]]等史蹟，以及縱貫線最陡斜坡「十六份坡」等諸多特色，此段路線停用後反而吸引許多遊客前往遊覽觀光。之前曾出現將原路線改建為[[輕軌|輕軌]]路線的提議，後來經過相關單位評估，於路線停用六年後納入[[行政院|行政院]]「環島觀光鐵路計畫」推動復駛，成為新的觀光客運支線。
*后里站以南的舊山線開闢為「[[后豐鐵馬道|后豐鐵馬道]]」，出租[[腳踏車|腳踏車]]給遊客，可騎到[[大甲溪鐵橋|大甲溪鐵橋]]以及[[東勢線|東勢線]]廢止後開闢的[[東豐自行車綠廊|東豐自行車綠廊]]。

== 復駛 ==
[[File:The_Platform_3_of_TRA_Sanyi_Station.JPG|thumb]]
近年來，[[台鐵|台鐵]]舊山線沿線陸續成為觀光景點，地方政府及民眾要求復駛的聲音愈來愈高。[[台鐵|台鐵]]經過幾年的籌措，終於修復[[台鐵|台鐵]]舊山線軌道。2010年6月5日，舊山線在[[三義車站|三義]]和[[臺中線|新山線]]銜接復駛。

因土地取得問題，舊山線復駛目前將僅於[[三義車站|三義]]至[[泰安車站_(舊山線)|泰安]]站間，[[泰安車站_(舊山線)|泰安]]至[[后里車站|后里]]站暫不復駛（列入第二階段復駛目標）。

2010年10月13日[[台鐵|台鐵]]在網路上公告舊山線的委外經營（ROT）案<ref>[http://web.pts.org.tw/hakka/news/detail.php?id=63362 試營運結束 舊山線ROT本月招商]</ref>。
舊山線於2011年暑假期間，2011年10月15日至2011年12月31日行駛。
