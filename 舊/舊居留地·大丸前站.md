'''舊居留地·大丸前站'''（{{jpn|j=旧居留地・大丸前駅|hg=きゅうきょりゅうち・だいまるまええき|rm=Kyūkyoryūchi-Daimarumae eki|f=y}}），是一個位於[[日本|日本]][[兵庫縣|兵庫縣]][[神戶市|神戶市]][[中央區_(神戶市)|中央區]]{{link-ja|三宮町 (神戶市)|三宮町 (神戸市)|三宮町}}三丁目，屬於[[神戶市營地下鐵|神戶市營地下鐵]][[海岸線_(神戶市營地下鐵)|海岸線]]的[[鐵路車站|鐵路車站]]。車站編號是'''K02'''。
[[File:Platform_of_Kyūkyoryūchi-Daimarumae_Station.JPG|thumb]]

== 歷史 ==
* 2001年（[[平成|平成]]13年）7月7日：與[[神戶市營地下鐵|神戶市營地下鐵]][[海岸線_(神戶市營地下鐵)|海岸線]]的[[三宮·花時計前站|三宮·花時計前站]]－[[新長田站|新長田站]]間開通的同時開業。

== 車站結構 ==
地下2樓是大堂、閘口，地下3樓是月台的[[地底車站|地底車站]]。地下3樓的月台是島式1面2線。地下通道與[[大丸|大丸神戶店（本館）]]、[[舊居留地|舊居留地]]以及[[三宮站|三宮站]]方向連接。

== 使用情況 ==
2016年度的1日平均'''上車'''人次為'''1,906人'''<ref>[http://www.city.kobe.lg.jp/life/access/transport/kotsu/gaiyou/index.html 神戸市：神戸市交通局　事業概要]</ref>。開業至2006年度有上昇傾向，在此以後減少，2009年度以後開業時的上車人次下跌。
* 截至2000年度，2005年度的預測上車人次為11,772人<ref name="jigohyoka">{{PDFlink|[http://www.city.kobe.lg.jp/life/access/transport/subway/kaigannsen/img/houkokusyo.pdf 地下鉄海岸線事後評価]}} - 神戸市、2007年3月</ref>。
