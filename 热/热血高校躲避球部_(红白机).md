{{vgname|热血高校躲避球部|ja=熱血高校ドッジボール部|en=Super Dodge Ball|diff=yes}}是一款由[[Technōs_Japan|Technōs Japan]]制作和发行的躲避球[[体育类游戏|体育类游戏]]。本游戏大致根据[[热血高校躲避球部|同名]]街机游戏制作。<ref name="mobygames">[http://www.mobygames.com/game/super-dodge-ball/release-info 《super-dodge-ball》] game overview at [[MobyGames|MobyGames]]</ref>本游戏于1988年7月26日在日本地区[[FC游戏机|FC游戏机]]发行。<ref name="gamefaqs">{{cite web | url = http://www.gamefaqs.com/nes/587662-super-dodge-ball/data| title = Release information | publisher = [[GameFAQs|GameFAQs]]| date = | accessdate = 2015-12-17}}</ref>于1989年6月在北美地区发行。

FC版的游戏相对于街机版本而言，可以添加队员的名字和状态。此外游戏增加了巡回赛模式和两人对战模式。

与街机版本类似，本游戏的日本地区和北美地区版本交换了日本队和美国队。
