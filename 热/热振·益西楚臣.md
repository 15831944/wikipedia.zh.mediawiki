'''热振·益西楚臣'''（{{bd|||1993年|12月}}），西藏人，藏族学者<ref name="档案馆">{{cite journal |author1=吴妙发 |author2=沈中明 |title=历史的见证——访西藏自治区档案馆 |journal=世界知识 |date=1991 |issue=7 |pages=24-25}}</ref>，第五世热振活佛[[热振·图旦绛白益西丹巴坚赞|热振·图旦绛白益西丹巴坚赞]]的[[古嘉堪布|曲本堪布]]{{efn|“曲本堪布”为藏传佛教僧职名，与森本堪布、索本堪布同属是随侍大活佛、照料其生活起居一应事务的高级僧官“古嘉堪布”}}<ref name="王贵1991">{{cite book |author1=王贵 |title=藏族人名研究 |date=1991-11 |publisher=民族出版社 |isbn=9787105013630}}</ref>{{rp|11-12}}，西藏自治区第六届人民代表大会常务委员会委员。

== 生平 ==
1947年，第五世热振活佛热振·图旦绛白益西丹巴坚赞遇害，益西楚臣到内地为其申冤<ref name="档案馆" />。1949年12月2日，益西楚臣到青海西宁，向人民解放军控诉帝国主义者破坏西藏内部团结的罪行，要求迅速解放西藏<ref>{{cite web |title=一、西藏和平解放的实现 |url=http://www.china.com.cn/ch-book/2011-07/12/content_22972646.htm |website=中国网 |accessdate=2018-12-27 |date=2011-07-12}}</ref>。同年加入[[中国人民解放军|中国人民解放军]]<ref name="档案馆" />。

益西楚臣曾任西藏自治区档案馆顾问、西藏自治区政治协商会议常务委员会委员<ref name="王贵1991" />、西藏自治区第六届人民代表大会常务委员会委员<ref name="人大"/>。

1993年12月，热振·益西楚臣病故<ref name="人大">{{cite web |title=西藏自治区三至九届人大常委会组成人员名单 |url=http://www.npc.gov.cn/npc/zt/qt/dfrd30year/2011-03/29/content_1649589.htm |website=中国人大网 |accessdate=2018-12-27 |date=2011-03-29}}</ref>。
