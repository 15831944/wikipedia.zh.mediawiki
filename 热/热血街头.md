《'''热血街头'''》（英語：Fearless），2012年7月20日上映于中国大陆的动作片。

== 剧情 ==
李斯和李克是[[四川人|四川人]]，李斯从小和哥哥相依为命，父母很早因为一场[[空难|空难]]逝世。由于缺少家庭文化的教育，李斯并非淑女，喜好在街头和人[[街舞|斗舞]]，一次因为斗舞，和别人打了起来，哥哥李克保护妹妹而杀人，并且被关押在[[监狱|监狱]]长达7年。

为了新生活，李斯开始前往[[深圳市|深圳市]]打拼，她碰上了好友樊星，并且去报考[[芭蕾舞|芭蕾舞]]学校，结果因为跳的是[[街舞|街舞]]而被拒，不得不再度踏入街舞，并且为街舞冠军而努力奋斗。最终，在李斯的努力和队伍的合作之下，她们赢得了[[冠军|冠军]]。

== 花絮 ==
电影的主角分别为[[柯受良|柯受良]]之子柯有伦和[[快乐女声|快乐女声]]李斯丹妮。<ref>{{cite web |url= http://news.mtime.com/2012/02/21/1482502.html|title=青春歌舞《热血街头》杀青 柯有纶联手快女李斯 |accessdate=2012-02-21 |publisher= 时光网|work= |language =zh-cn}}</ref>

王传君此前在《[[爱情公寓|爱情公寓]]》中出演“关谷神奇”，而在此片扮演李斯的哥哥李克。<ref>{{cite web |url=http://news.mtime.com/2012/07/20/1493143.html |title= 《热血街头》今日上映 五大看点盘点热血青春|accessdate= 2012-07-20 |publisher= 时光网|work= |language =zh-cn}}</ref>
