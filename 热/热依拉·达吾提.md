'''热依拉·达吾提'''（{{bd|1966年|5月||}}），女，[[维吾尔族|维吾尔族]]，生于[[新疆|新疆]][[乌鲁木齐市|乌鲁木齐市]]，[[中华人民共和国|中华人民共和国]]语言学家、民俗学家，[[中共|中共]]党员。

==生平==
1987年，自[[新疆大学|新疆大学]]中文系维吾尔语言文学专业毕业，获学士学位。1990年7月，在新疆大学中文系获少数民族民间文学（含[[民俗学|民俗学]]）专业硕士学位。1998年7月，在[[北京师范大学|北京师范大学]]中文系获民俗学博士学位。1990年7月起，一直在新疆大学中文系任教。1999年2月晋升副教授，2004年11月晋升教授。2001年被评为少数民族语言文学方向硕士研究生导师。1999年5月到2000年12月，参加了新疆大学教务处举办的“[[波斯语|波斯语]][[察合台语|察合台语]]强化班”。2001年6月20日到7月6日，参加了在[[西北民族大学|西北民族大学]]举办的中国第六届[[社会学|社会学]][[人类学|人类学]]高级研讨班。2003年2月到7月，在[[北京语言文化大学|北京语言文化大学]]接受[[英语|英语]]强化培训。2003年9月到2004年9月，在[[美国|美国]][[宾西法尼亚大学|宾西法尼亚大学]]民俗学系及[[印第安纳大学|印第安纳大学]]当[[访问学者|访问学者]]。2006年12月到2007年6月，在美国[[加利福尼亚大学|加利福尼亚大学]]当访问教授<ref name=re>{{Cite web |url = http://www.chinesefolklore.org.cn/expert/index.php?ExpertID=119 |title = 热依拉·达吾提 |author =  |date =  |publisher =  中国民俗学网|language =  |accessdate = 2018-08-08 |quote =  }}</ref>。2008年12月14日，“第七届[[钟敬文|钟敬文]]民俗学奖”（2008年度）颁奖仪式在北京师范大学敬文讲堂举行，获奖人是新疆大学教授热依拉·达吾提、北京师范大学历史系教授[[赵世瑜|赵世瑜]]<ref>{{Cite web |url = http://www.chinesefolklore.org.cn/web/index.php?NewsID=3535 |title =2008年度“钟敬文民俗学奖”颁奖仪式在北京师范大学举行  |author =  |date =2008-12-15  |publisher =中国民俗学网  |language =  |accessdate =  |quote =  }}</ref>。她是[[全国青联|全国青联]]第十届委员会委员<ref>{{Cite web |url =http://zhuanti.youth.cn/2005/top10youth/gyql/200601/t20060120_45984.htm  |title =  全国青联第十届委员会委员名单|author =  |date =  2006-01-20 |publisher = 中国青少年计算机信息服务网 |language =  |accessdate =  |quote =  }}</ref>。
2017年12月，她告訴親戚要去北京，但從那時候起至今8個月音訊全無，彷彿人間蒸發。達吾提的家人和關係密切的朋友確信，她已在一場嚴厲打擊維吾爾人的行動中被祕密拘捕。<ref>{{cite news|url =https://newtalk.tw/news/view/2018-08-14/135212  |title =  新疆加強壓制維吾爾人 學者達吾提失蹤逾8月應已被捕||author =陳重生|date =  2018.08.14 |publisher = 新頭殼newtalk }}</ref><ref>{{cite news|url =https://newtalk.tw/news/view/2018-08-14/135212  |title =  新疆加強壓制維吾爾人，著名學者失蹤逾八月||author =儲百亮, 王霜舟|date =  2018.08.13 |publisher = 紐約時報中文網 }}</ref>
