'''热乾麵'''是一种[[武汉|武汉]]的特色[[早点|早点]]，吃早餐在武汉被称为“过早”，热干面因为方便快捷美味，被很多武汉人用来过早，在武汉街头巷尾随处可见售卖。热干面现在不仅是武汉的特色美食，如今在湖北很多地方都十分受欢迎。热干面面身为[[碱水麵|碱水麵]]，淋以[[芝麻酱|芝麻酱]]，辅以葱花、辣萝卜等各种佐料，价廉味美。是很多武汉市民早餐的首选。武汉本地最为著名的品牌是蔡林记。热干面与[[山西|山西]][[刀削麵|刀削麵]]、[[两广|两广]][[伊府麵|伊府麵]]、[[四川|四川]][[担担麵|担担麵]]、北方[[炸酱麵|炸酱麵]]并称为[[中国|中国]]五大名[[麵|麵]]<ref>{{cite web|url =http://sb.66wz.com/system/2005/12/16/100040405.shtml|title =全面报告|accessdate =2008-12-01|date =2005-12-16|publisher =温州商报|language =zh-hans}}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。并被评为中国十大面条之首。<ref name="“中国十大面条”出炉热干面居首 你吃过几样">{{cite web|url=http://hi.people.com.cn/n/2013/0705/c231186-19007599.html |title=“中国十大面条”出炉热干面居首 你吃过几样 |publisher=人民网 |accessdate=2013-07-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20140117044720/http://hi.people.com.cn/n/2013/0705/c231186-19007599.html |archivedate=2014-01-17 }}</ref>如今在全国各地都有贩售。

== 典故 ==
[[File:Hot_Dry_Noodles_stirred.jpg|thumb]]
至今，热乾麵如何出现还没有明确的考证，但广为流传的版本是在1920年代，[[汉口|汉口]]长堤街有个名叫李包的[[小贩|小贩]]，在[[关帝庙|关帝庙]]一带靠卖[[凉粉|凉粉]]和汤麵为生。有一天，天气异常炎热，不少剩麵未卖完，他怕麵条发馊变质，便将剩麵煮熟沥乾，晾在案板上。不小心，碰倒案板上的油壶，[[麻油|麻油]]泼在麵条上。李包见状，无可奈何，只好将麵条用油拌匀重新晾放。第二天早上，李包将拌油的熟麵条放在沸水里稍烫，捞起沥乾入[[碗|碗]]，然后加上卖凉粉用的调料，结果热气腾腾，香气四溢。人们争相购买，吃得津津有味。有人问他卖的是什么麵，他脱口而出，说是“热乾麵”。从此他就专卖这种麵，不仅人们竞相品尝，还有不少人向他拜师学艺。<ref>{{cite web|url =http://news.xinhuanet.com/food/2004-05/14/content_1470610.htm|title =湖北小吃-热乾麵|accessdate =2008-12-01|date = 2004-05-14|publisher =[[新华网|新华网]]|language = zh-hans}}</ref>

到1928年汉口满春路口的蔡明伟夫妇打出“[[蔡林记|蔡林记]]”的招牌经营热乾麵。店名的因蔡家门前两棵葱郁的大树，取名“蔡林记”喻意蔡家生意兴隆。<ref>[[蔡林记|蔡林记]]热干面制作工艺简介：{{cite web|url =http://hb.qq.com/a/20101014/000525.htm|title =百年"蔡林记"传人公布热干面配方 可无偿传授|accessdate =2010-11-29|date = 2010-10-14|publisher =[[腾讯网|腾讯网]]|language = zh-hans}}</ref>

1955年夫妇俩将“蔡林记”迁至更加繁华的中山大道726—728号。1974年店堂再次迁移至目前的[[佳丽广场|佳丽广场]]之正门下。

== 制作方法 ==
{{wikibooks}}
#[[碱水麵|碱水麵]]煮或蒸至九成熟捞出，加芝麻油拌匀，将面摊开在案面至凉
#待食用时，用[[漏勺|漏勺]]将麵置於滚水中焯热捞出。
#淋上调好的芝麻酱、酱油、辣萝卜丁，还可根据口味加入咸菜丁、葱花、辣椒油、味精、胡椒粉、醋、雪菜、鸡汤等佐料。

== 饮食习惯 ==
=== 武汉 ===
多年来，热乾麵的做法并未发生很大变化，大街小巷的摊点大多都销售热乾麵。武汉人没有在家自己制作热乾麵的习惯，一是因为热干面的制作过程十分耗时，仅是碱水麵过水淋油这个过程就要花费两、三小时的时间，家里自己做的小批量热干面效益不高；二是因为热乾麵在大街小巷均有销售，十分常见，另外武汉的早餐品种十分丰富，所以武汉人在清晨有外出吃早餐的习惯。热乾麵在武汉较有名的店铺为蔡林记，李记，庞记等。<ref name="热干面的制作">{{cite web| url=http://www.wuxingsnack.com/reganmiandezhizuo.html| title=热干面的制作| accessdate=2012-04-12| deadurl=yes| archiveurl=https://web.archive.org/web/20120326193736/http://www.wuxingsnack.com/reganmiandezhizuo.html| archivedate=2012-03-26}}</ref><ref name="武汉人过早">{{cite web| url=http://whjpz.com/news.asp?act=content&cid=3&id=167&pid=| title=武汉人过早| accessdate=2012-04-12| deadurl=yes| archiveurl=https://web.archive.org/web/20160304130742/http://whjpz.com/news.asp?act=content&cid=3&id=167&pid=| archivedate=2016-03-04}}</ref>

=== 信阳 ===
信阳热干面据信是从武汉传过来的，大致做法和武汉一样，口味略有不同。主料仍然是特制的面条、芝麻酱、麻油等，但最后可能会放[[千张|千张]]、[[豆芽|豆芽]]、[[香菜|香菜]]、[[榨菜|榨菜]]、醋或者鸡汤中的一种或多种，具体视区域以及个人喜好而定。<ref name="信阳热干面">{{cite web|url=http://www.guangshan.net/meishi/meishicaipu/83.html|title=信阳热干面|accessdate=2012-08-30|date=2010-02-24|language=zh-hans|author=佚名|publisher=光山热线|deadurl=yes|archiveurl=https://web.archive.org/web/20121018002238/http://www.guangshan.net/meishi/meishicaipu/83.html|archivedate=2012-10-18}}</ref><ref name="左手牛肉汤，右手热干面">{{cite web |url=http://blog.sina.com.cn/s/blog_49cbae750100ciqr.html|date=2008-12-07 | archiveurl=http://xinyang.cncn.com/article/133619/ | title=左手牛肉汤，右手热干面 | accessdate=2012-08-30|archivedate=2010-01-15 |author=胡则牧|language=zh-hans|publisher=欣欣旅游网}}</ref><ref name="xinyang reganmian">{{cite web | url=http://www.tourunion.com/info/htm/24154.htm | title=信阳热干面 | accessdate=2012-08-30|date=2009-07-31|publisher=旅游联盟网|author=佚名|language=zh-hans}}</ref><ref name="信阳热干面1">{{cite web|url=http://www.100ts.cn/article-6131.html|title=信阳热干面|accessdate=2012-08-30|date=2011-08-07|author=佚名|publisher=中国地方特色网|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20141109141214/http://www.100ts.cn/article-6131.html|archivedate=2014-11-09}}</ref>
