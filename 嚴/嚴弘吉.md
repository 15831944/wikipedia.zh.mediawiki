'''嚴弘吉'''（[[諺文|諺文]]：{{lang|ko|'''엄홍길'''}}，{{bd|1960年|9月14日|||}}），生於[[韓國|韓國]][[慶尚南道|慶尚南道]][[固城郡|固城郡]]，為韓國著名[[登山家|登山家]]，為亞洲首位、世界第八位完登14座[[八千公尺高山|八千公尺以上山峰]]的登山家。<ref>{{cite web|url=http://www.8264.com/viewnews-12267-page-1.html |title=严弘吉：大山是我的老师和上帝 [组图] |accessdate=2013-01-19 |date=2003-03-11 |work= |publisher=戶外資料網}}</ref>

== 登山歷程 ==
嚴弘吉早期曾登頂[[南美洲|南美洲]]最高峰[[阿空加瓜山|阿空加瓜山]]數次。1985年9月，嚴弘吉首次嘗試攀登[[珠穆朗瑪峰|珠穆朗瑪峰]]，但嘗試失敗，隨後於1988年9月再次嘗試，並成功登頂，開啟14座八千公尺以上高山的挑戰。於2001年9月登頂[[希夏邦馬峰|希夏邦馬峰]]，以將近13年的時間完登14座八千公尺以上高山。完登之後，於2002年與2003年兩度攀登[[珠穆朗瑪峰|珠穆朗瑪峰]]，完成南、北兩條攀登路線，共三次登上珠穆朗瑪峰，<ref>[http://www.everestnews.com/history/climbers/Um%20Hong-Gil.htm EverestHistory.com:  Um Hong-Gil]</ref><ref>嚴弘吉個人名片</ref>並於2004年攀登金城章嘉峰西峰（Yalunkang Mountain），2007年攀登洛子峰東副峰（Lhotse Shar）等兩座8000公尺以上山峰。另外，於2007年12月13日登頂[[南極洲|南極洲]]最高峰[[文森山|文森山]]。

== 與臺灣的互動 ==
[[File:Mount_Yu_Shan_-_Taiwan.jpg|thumb]]]]
嚴弘吉曾於2008年1月受[[臺灣|臺灣]][[交通部觀光局|交通部觀光局]]之邀，任臺灣[[玉山|玉山]]的宣傳大使，並邀約共同攀登玉山，同年5月，嚴弘吉與韓國登山聯合會及記者共九人來臺，與臺灣首位登頂珠穆朗瑪峰的女性登山家[[江秀真|江秀真]]、中華民國山-{岳}-協會國際組組長陳劉茂松、中華民國登山健行會理事胡瑞發等人共同攀登玉山。<ref>{{cite web|url=http://www.epochtimes.com/b5/8/5/7/n2108560.htm |title=南韓登山探險家嚴弘吉 將挑戰玉山 |accessdate=2013-01-19 |date=2008-05-07 |work=姜遠珍 |publisher=大紀元新聞網}}</ref><ref>{{cite web|url=http://theme.taiwan.net.tw/epaper/chi/20080523.htm |title=邀請韓國登山專家嚴弘吉與國內專家同登玉山 |accessdate=2013-01-19 |date=2008-05-23 |work=交通部觀光局電子報 |publisher=交通部觀光局}}</ref>
2012年5月，嚴弘吉二度來臺，與中華民國山-{岳}-協會理事長等人攀登[[雪山_(台灣)|雪山]]。<ref>{{cite web|url=https://www.facebook.com/media/set/?set=a.327553757314692.69871.141201395949930&type=3 |title=5/12-13韓國登山家嚴弘吉一行登雪山 |accessdate=2013-01-19 |date=2012-05-15 |work= |publisher=中華山-{岳}- Chinese Taipei Alpine Association}}</ref>

== 展覽館 ==
於嚴弘吉的故鄉，韓國慶尚南道固城郡，設有嚴弘吉展覽館，主要設立目的為呈現嚴弘吉艱辛的攀登過程，以及勉勵年輕人要具有挑戰大自然的勇氣和力量。整個展覽館共分為5區，各展區之展示內容如下：
* 第一區：描述嚴弘吉的成長過程
* 第二區：喜馬拉雅山脈的地質、地形、氣候等概況
* 第三區：攀登16座八千公尺以上高山的過程
* 第四區：喜瑪拉雅山的登山知識，包括危險性、裝備、緊急應變等
* 第五區：介紹固城郡十大名山、販賣紀念品等<ref>{{cite web |url=http://blog.ifeng.com/article/10113351.html |title=固城——严弘吉展览馆 |accessdate=2013-01-19 |date=2011-02-23 |work=韓國慶尙南道 |publisher=鳳凰博報 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304093106/http://blog.ifeng.com/article/10113351.html |archivedate=2016-03-04 }}</ref>
