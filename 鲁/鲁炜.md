'''鲁炜'''（{{bd|1960年|1月|||catIdx=L鲁}}），[[安徽|安徽]][[巢湖市|巢湖]]人，原[[中国共产党|中国共产党]]党员及[[中华人民共和国|中华人民共和国]]官员，[[中共十八大|中共十八大]]代表<ref name=weiji/>。曾任[[中共中央宣传部|中共中央宣传部]]副部长，[[中央网信办|中央网信办]]、[[国家互联网信息办公室|国家网信办]]主任，是[[中共十九大|中共十九大]]后落马的首位[[正部级|正部级]]官员。

== 生平 ==
===早年经历===
1978年10月鲁炜参加工作，1991年2月加入[[中国共产党|中国共产党]]，研究生学历，硕士学位，高级记者。1978年7月至10月，[[安徽省|安徽省]][[巢县|巢县]]庙岗乡鲁集大队回乡[[知青|知青]]。1978年10月至1984年9月，安徽省[[巢县小学|巢县小学]]、中学[[教师|教师]]，[[桂林|桂林]]风动工具厂机械工人、宣传干事。其间，1982年9月至1985年8月在[[广西广播电视大学|广西广播电视大学]]语文专业学习<ref name=weiji>{{Cite web |url =http://news.ifeng.com/a/20171121/53466831_0.shtml  |title =中宣部原副部长鲁炜涉严重违纪被查  |author =  |date = 2017-11-21 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

1984年9月至1991年4月，鲁炜任[[广西壮族自治区|广西壮族自治区]][[桂林市|桂林市]][[中级人民法院|中级人民法院]][[书记|书记员]]、干事，《[[广西法制报|广西法制报]]》记者、总编辑助理。

===从新华社到北京市===
1991年4月至1994年6月，任[[新华社|新华社]]桂林记者站站长、桂林支社社长。1994年6月至1997年11月，任新华社广西分社副社长、[[党组|党组]]成员。1997年11月至2001年10月，任新华社广西分社社长、党组书记。其间，1997年9月至2001年6月在[[广西大学|广西大学]]研究生班学习<ref name=weiji/>。2000年起为享受国务院[[政府特殊津贴|政府特殊津贴]]专家。

2001年10月至11月，鲁炜任新华社副秘书长，兼任总经理室总经理。2001年11月至2004年5月，任新华社党组成员、副秘书长、秘书长，兼总经理室总经理、中国经济信息社社长、新闻信息中心主任。2004年5月至2011年3月，任新华社副社长、党组成员，并先后兼任秘书长、经济信息编辑部主任、中国经济信息社控股董事长兼总经理、金融信息交易所理事长、中国金融信息大厦有限公司董事长、金融信息交易所理事长、国家金融信息中心有限责任公司董事长兼总经理。其间，2006年9月至2009年1月在[[中央党校|中央党校]]研究生班学习，2006年9月至11月在中央党校省部级进修班学习，2008年11月至2011年7月在[[中国人民大学|中国人民大学]]金融学院工商管理专业攻读高级管理人员[[工商管理硕士|工商管理硕士]]学位<ref name=weiji/>。

2004年5月升任新华社副社长后，据称鲁炜组织了一个专人团队，改变了以前对中共领导报道严肃、呆板的做法，在报道中大胆刻画[[中华人民共和国国务院总理|国务院总理]][[温家宝|温家宝]]的个人思想及鲜明个性。该团队的作品包括指出温家宝出身贫贱、与中国网民交流谈患[[脑溢血|脑溢血]]母亲、下矿井与矿工一起啃吃馒头、邀[[农民工|农民工]]到[[中南海|中南海]]会见、探视[[爱滋病|爱滋]]村孤儿、读诗《[[仰望星空|仰望星空]]》、引经据典掉书袋、替无助的农民工讨薪等<ref>{{Cite web |url = http://www.chinese.rfi.fr/%E4%B8%AD%E5%9B%BD/20110326-%E6%B8%A9%E5%AE%B6%E5%AE%9D%E2%80%9C%E5%85%AC%E5%85%B3%E5%B8%88%E7%88%B7%E2%80%9D%E9%B2%81%E7%82%9C%E5%8D%87%E4%BB%BB%E5%8C%97%E4%BA%AC%E5%B8%82%E5%B8%B8%E5%A7%94 |title = 温家宝“公关师爷”鲁炜升任北京市常委 |author =  |date =2011-03-26  |publisher = 法广新闻 |language =  |accessdate =  |quote =  }}</ref>。

2011年3月至2013年4月，鲁炜任[[中共北京市委|中共北京市委]]常委、宣传部部长，[[北京市人民政府|北京市人民政府]]副市长<ref name=weiji/><ref>{{cite web|url=http://politics.people.com.cn/n/2013/0128/c1001-20351105.html|title=北京副市长鲁炜：从新闻记者走来的政府官员|publisher=人民网|date=2013-01-28}}</ref>。

===执掌中央网信办===
2013年4月至2014年5月，任[[国家互联网信息办公室|国家互联网信息办公室]]主任兼[[国务院新闻办公室|中央外宣办]]、[[国务院新闻办公室|国务院新闻办]]副主任<ref>{{cite web |accessdate=2013-05-01 |url = http://news.china.com.cn/txt/2013-05/10/content_28785567.htm |title=鲁炜担任国信办主任、国新办副主任|publisher=[[中国网|中国网]]}}</ref><ref>{{cite web|url=http://politics.people.com.cn/n/2013/0510/c1001-21438171.html|title=国务院任免：蔡名照为国新办主任 鲁炜为国信办主任|publisher=人民网|date=2013-05-10}}</ref><ref name=weiji/>。2014年5月起，任[[中共中央宣传部|中共中央宣传部]]副部长、[[中央网络安全和信息化领导小组|中央网络安全和信息化领导小组]]办公室主任、[[国家互联网信息办公室|国家互联网信息办公室]]主任<ref name=weiji/>。

2013年8月10日，鲁炜召集[[微博|微博]]、网络名人座谈，提出“[[七条底线|七条底线]]”的共识，并敦促名人遵守该七条原则<ref name="fish">{{cite web |url = http://news.163.com/13/0825/09/9745L89300014Q4P.html |title=“秦火火”们玩火自焚网上哪些言论会犯法？ |accessdate=2013年8月25日 |publisher=网易 |work=华商晨报 |language=zh-hans}}</ref>。同时，鲁炜希望名人承担社会责任，作为[[正能量|正能量]]，为实现[[习近平|习近平]]总书记提出的“[[中国梦|中国梦]]”而出力<ref>{{cite web |url=http://news.china.com.cn/live/2013-08/13/content_21659942.htm |title=微博"大V"与官员探讨坚守底线--网管释放由堵到疏信号 |accessdate=2013年8月13日 |publisher= 中国网|work=光明网 |language=zh-hans}}</ref>。当日参加会议的名人有[[纪连海|纪连海]]、[[廖玒|廖玒]]、[[陈里|陈里]]、[[潘石屹|潘石屹]]、[[古永锵|古永锵]]、[[陈彤|陈彤]]、[[周小平|周小平]]、[[薛蛮子|薛蛮子]]、[[徐世平|徐世平]]、[[齐向东|齐向东]]、[[孙健|孙健]]、[[胡延平|胡延平]]、[[张国庆|张国庆]]、[[高龙|高龙]]、[[徐小平|徐小平]]、[[李未柠|李未柠]]等<ref name="bee">{{cite web |url = http://news.xinhuanet.com/2013-08/12/c_132621768.htm |title=国信办主任鲁炜与网络“大V”畅谈社会责任  |accessdate= 2013年8月12日 |publisher=新华网 |work=新华网 |language=zh-hans}}</ref>。

2013年12月10日，鲁炜在第二届中韩互联网圆桌会议上作主旨演讲<ref>{{cite news|url=http://news.xinhuanet.com/world/2013-12/10/c_125832289.htm|title=中韩互联网圆桌会议在首尔举行 鲁炜发表主旨演讲（组图）|date=2013-12-10|publisher=新华网|deadurl=yes|archiveurl=https://web.archive.org/web/20140428015344/http://news.xinhuanet.com/world/2013-12/10/c_125832289.htm|archivedate=2014-04-28}}</ref>。2014年6月23日，鲁炜出席在[[伦敦|伦敦]]举行的[[互联网名称与数字地址分配机构|互联网名称与数字地址分配机构]]（ICANN）第50次大会高级别政府会议，阐述了中国关于ICANN国际化的观点和主张<ref>{{cite news|url=http://news.xinhuanet.com/world/2014-06/24/c_1111281780.htm|title=ICANN国际化应坚持四项原则|publisher=新华网|date=2014-06-24|deadurl=yes|archiveurl=https://web.archive.org/web/20140630004011/http://news.xinhuanet.com/world/2014-06/24/c_1111281780.htm|archivedate=2014-06-30}}</ref>。2014年9月10日，[[夏季达沃斯论坛|夏季达沃斯论坛]]第八届年会在[[天津|天津]]开幕，鲁炜出席了以“网络经济的未来”为主题的分论坛活动并发言<ref>{{cite news|url=http://news.xinhuanet.com/newmedia/2014-09/11/c_126973982.htm|title=鲁炜出席2014夏季达沃斯谈互联网共识、共治、共享、共赢|publisher=新华网|date=2014-09-11|deadurl=yes|archiveurl=https://web.archive.org/web/20141201231855/http://news.xinhuanet.com/newmedia/2014-09/11/c_126973982.htm|archivedate=2014-12-01}}</ref>。2014年9月19日，鲁炜出席首届“中国－[[东盟|东盟]]网络空间论坛”<ref>{{cite news |url=http://world.people.com.cn/n/2014/0919/c157278-25695126.html |title=建设“海上丝绸之路”的信息枢纽|publisher=人民网|date=2014-09-19}}</ref>。

2014年10月30日，鲁炜出席首届[[世界互联网大会|世界互联网大会]]举行的新闻发布会并答记者问。對於[[中华人民共和国网络审查|網絡審查]]制度，他表示：“我們現在不能允許的是，（外國互聯網企業）既佔了中國市場，又掙了中國的錢，還來傷害中國<ref>{{cite news|url=http://news.youth.cn/gn/201410/t20141030_5947141.htm | title=鲁炜回应无法登陆facebook|date=2014-10-30|publisher=中国青年网|deadurl=yes|archiveurl=https://web.archive.org/web/20170729131458/http://news.youth.cn/gn/201410/t20141030_5947141.htm|archivedate=2017-07-29}}</ref><ref name="紐約時報"/>。”

[[紐約時報|紐約時報]]中文网称鲁炜为“中國互聯網的守門人”<ref name="紐約時報">{{Cite news |url = http://cn.nytimes.com/china/20141202/c02internet/zh-hant/ | title=魯煒，中國互聯網的守門人 | date=2014-12-02 | publisher=[[紐約時報|紐約時報]]中文网}}</ref>。2015年4月，因被称为“有权决定亿万网民看什么”，鲁炜入选《[[时代周刊|时代周刊]]》2015年“[[时代百大人物|百位最具影响力人物]]”<ref>{{cite web |url = http://www.bbc.co.uk/zhongwen/simp/china/2015/04/150416_time_100_list | language= | title = 《时代》杂志“百人榜”：习近平及柴静均上榜 |publisher=BBC中文网 |date=2015-04-16 |accessdate= }}</ref><ref>[http://time.com/3823285/lu-wei-2015-time-100/ Lu Wei | TIME]</ref>。2015年9月，鲁炜陪同[[习近平|习近平]][[2015年习近平访美|访问美国]]<ref>{{cite news|title=习近平陪同访问正部以上官员超15人 外交部2位|url=http://news.qq.com/a/20150926/009039.htm|date=2015-09-26|work=腾讯|agency=南方都市网}}</ref>。

2015年12月18日，鲁炜在第二届[[世界互联网大会|世界互联网大会]]闭幕式上致辞，呼吁以[[烏鎮鎮|乌镇]]峰会为起点，携手推动全球互联网共享共治；以四项原则（尊重主权、维护和平安全、促进合作开放、构建良好秩序）为基础，积极推进全球互联网治理体系变革；以五点主张为指引，共同构建网络空间命运共同体<ref>{{Cite web|url =http://it.people.com.cn/n1/2015/1218/c1009-27947975.html|title =鲁炜在第二届世界互联网大会闭幕式上的致辞|author =|date =2015-12-18|publisher =人民网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20151222122041/http://it.people.com.cn/n1/2015/1218/c1009-27947975.html|archivedate =2015-12-22}}</ref><ref>{{Cite web |url = http://gb.cri.cn/42071/2015/12/16/7551s5200311.htm  |title = "构建良好秩序"就是世界互联网治理体系变革的辩证法 |author =  |date =2015-12-16  |publisher =国际在线  |language =  |accessdate =  |quote =  }}</ref>。

2016年6月起，鲁炜不再担任中央网信办主任职务，仅任中共中央宣传部副部长<ref name=weiji/>。

=== 落马 ===
2017年2月，经中央批准，十八届中央第十二轮巡视对中央网络安全和信息化领导小组办公室、[[国务院扶贫开发领导小组办公室|国务院扶贫开发领导小组办公室]]、[[中国铁路总公司|中国铁路总公司]]、[[中国船舶重工集团公司|中国船舶重工集团公司]]等4个单位试点开展“机动式”巡视。机动式巡视是中共十八大以来巡视制度的一项创新，被称为“游动哨”、“突击队”，其问题导向更鲜明，根据实际情况，动态、机动地选定巡视任务，哪里问题反映突出就到哪里去。2017年6月10日，中央第八巡视组向中央网络安全和信息化领导小组办公室领导班子反馈了“机动式”巡视情况，指出“巡视组还收到反映一些领导干部的问题线索，已按有关规定转[[中央纪委|中央纪委]]、[[中共中央组织部|中央组织部]]等有关方面处理。”<ref>{{Cite web |url = http://www.guancha.cn/politics/2017_06_11_412768_s.shtml  |title =  中央第12轮巡视“回头看”4省区 “机动式”巡视4单位，发现哪些问题？|author =  |date = 2017-06-11 |publisher =观察者网  |language =  |accessdate =  |quote =  }}</ref><ref>{{cite web |url = http://www.guancha.cn/politics/2017_11_22_435965.shtml |title = 中纪委刊文：十九大后鲁炜落马，机动式巡视作用彰显 |author =  |date = 2017-11-22 |website = 观察者网  |language =  |accessdate =  |quote =  }}</ref>

2018年10月19日，宁波市中级人民法院一审公开开庭审理了中共中央宣传部原副部长鲁炜受贿一案。宁波市人民检察院指控：2002年7月至2017年下半年，被告人鲁炜利用职务上的便利以及职权、地位形成的便利条件，为有关单位和个人在网络管理、职务晋升等事项上谋取利益，直接或者通过他人收受相关单位和个人给予的财物，共计折合[[人民币|人民币]]3200万余元。<ref>{{cite news |title=中共中央宣传部原副部长鲁炜受贿案一审开庭-新华网 |url=http://www.xinhuanet.com/politics/2018-10/19/c_1123583584.htm |accessdate=2018-10-22 |agency=新华社 |date=2018-10-19}}</ref>

2018年11月14日，鲁炜的忏悔书手稿在[[中国国家博物馆|中国国家博物馆]]举行的“伟大的变革——庆祝改革开放40周年大型展览”中首次公开展示。他在忏悔书中写道：“我在政治上、经济上、工作上、生活上，都犯下了严重的不可饶恕的错误，严重丧失了一名共产党员基本的党性原则和操守底线……我严重违反六大纪律，‘七个有之’条条都犯，自己犯错之多、之深、之恶劣，给党的事业带来巨大伤害，给党的形象抹了黑，辜负了组织30年的教育培养……我诚恳地向组织认错，悔错，改错。”而他在忏悔书中，亦提及了他的妻子和儿子：“我的生活作风问题使她受到了很大伤害，我们经常为此吵架，她对我完全绝望，曾经悲愤地对我说：‘我管不了你，但迟早共产党会管你。’而今，正是一语成谶！我的儿子，刚刚度过30岁生日。在他人生的路上，我没有做好人生的榜样，也没有尽到一个父亲的责任<ref>{{cite news |title=鲁炜忏悔书在这场展览曝光:妻子对我完全绝望|url=http://news.sina.com.cn/c/2018-11-15/doc-ihmutuec0253868.shtml|accessdate=2018-11-16 |agency=北京青年报 |date=2018-11-15}}</ref>。”

2019年3月26日，[[浙江省宁波市中级人民法院|宁波市中级人民法院]]公开宣判中宣部原副部长鲁炜受贿一案，对被告人鲁炜以受贿罪判处有期徒刑十四年，并处罚金人民币三百万元；对鲁炜受贿所得财物及其[[孳息|孳息]]予以追缴，上缴国库，不足部分继续追缴。鲁炜当庭表示服从判决，不上诉<ref>{{cite web |title=中宣部原副部长鲁炜受贿案一审宣判 |url=http://www.xinhuanet.com//2019-03/26/c_1124285790.htm |website=新华网 |accessdate=2019-03-26 |date=2019-03-26}}</ref><ref>{{cite web |title=鲁炜获刑14年 主动交代办案机关未掌握的犯罪事实 |url=http://news.sina.com.cn/c/2019-03-26/doc-ihtxyzsm0672438.shtml |website=新浪网 |accessdate=2019-03-26 |date=2019-03-26}}</ref>。
