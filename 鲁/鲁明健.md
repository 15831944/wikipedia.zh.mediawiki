'''鲁明健'''（{{bd|1917年||1998年|}}），[[湖南省|湖南省]][[长沙县|长沙县]]人，[[中华人民共和国|中华人民共和国]]政治人物<ref name=tianmen>{{Cite web |url = http://www.tmswb.gov.cn/dstd/dsrw/sdjz/201207/t20120713_213229.html |title =中共天门市（县）委历届书记简介（二）   |author =  |date =2012-07-13  |publisher = 中共天门市委办公室 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
1936年，鲁明健在[[北平|北平]]就读[[天主教辅仁大学|天主教辅仁大学]]期间参加[[一二九运动|一二九运动]]，同年加入[[中国共产党|中国共产党]]。1937年8月到1938年9月，在[[武汉|武汉]]从事青年工作<ref name=tianmen/>。曾和[[于光远|于光远]]合办革命刊物《呼声》<ref name=hebing>{{Cite web |url = http://dajia.qq.com/blog/237822053114953.html |title = 最高法院的右派们：朱耀堂和鲁明健—— 红旗下的法官继庭长贾潜之后 |author = 何兵 |date =2013-02-24  |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。1938年9月赴鄂中参与创建抗日根据地。此后，在[[抗日战争|抗日战争]]和[[第二次国共内战|第二次国共内战]]中，历任中共京安县委书记、中共鄂南中心县委书记、中共[[随县|随县]]县委书记、中共鄂西北工委书记、中共天京潜县委书记（1948年7月至1949年5月）等职<ref name=tianmen/>。

[[中华人民共和国|中华人民共和国]]成立后，历任中共天门县委书记（1949年6月至1950年3月），中共[[宜昌市|宜昌市]]委副书记，[[最高人民法院|最高人民法院]]办公室副主任、党组成员、[[最高人民法院审判委员会|审判委员会]]委员（1979年11月至1983年9月）、[[最高人民法院研究室|研究室]]主任等职<ref name=tianmen/>。1957年[[反右运动|反右运动]]中，在最高人民法院和[[贾潜|贾潜]]、[[林亨元|林亨元]]、[[朱耀堂|朱耀堂]]等人都被打成右派<ref name=hebing/>。

1978年9月9日，中共中央发布《关于坚决保证刑法、刑事诉讼法切实实施的指示》，核心内容是取消党委断案。起草者曾到最高人民法院征求意见，当时已复出任最高人民法院研究室主任职务的鲁明健说：“我们十分赞同取消这一制度，但最好由你们学者提出来，不然人们会说我们是向党要权。”据说该指示在最高人民法院党组讨论时，全体鼓掌<ref name=hebing/>。

1998年，鲁明健逝世<ref name=tianmen/>。
