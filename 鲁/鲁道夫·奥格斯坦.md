'''鲁道夫·奥格斯坦'''（'''Rudolf Karl Augstein'''，{{bd|1923年|11月5日|2002年|11月7日|catIdx=A}}），生于[[汉诺威|汉诺威]]，逝于[[汉堡|汉堡]]。他是[[德国|德国]]新闻从业员、出版家，《[[明镜|明镜]]》周刊创始人。

== 早年生涯 ==
奥格斯坦出生在汉诺威一个富裕的[[天主教|天主教]]家庭。其父弗里德里希·奥格斯坦（{{lang|de|Friedrich Augstein}}）曾是照像机厂的[[企业主|企业主]]和[[商人|商人]]。其母格特鲁德·玛丽亚·奥格斯坦（{{lang|de|Gertrude Maria Augstein}}）共生下五女二男。鲁道夫排行最末。奥格斯坦九岁时，[[纳粹党|纳粹党]]在德国掌权。1941年中学毕业后，奥古斯坦在汉诺威一家当地报纸做学徒[[记者|记者]]。1942年起，他被征入伍，作为报话员参加[[二战|二战]]。战争末期他还作过[[炮兵勘察员|炮兵勘察员]]，[[军衔|军衔]]至[[少尉|少尉]]。1945年4月获颁二等[[铁十字勋章|铁十字勋章]]。

== 创办《明镜》 ==
二战结束后，奥格斯坦和摄影师罗曼·施特普卡（{{lang|de|Roman Stempka}}）以及编辑格哈特·R·巴什（{{lang|de|Gerhard R. Barsch}}）一起从英国人手中接管了一份名为《本周》（{{lang|de|Diese Woche}}）的[[周刊|周刊]]。这本杂志是按照英国《新闻评论》（{{lang|en|News Review}}）和美国《[[时代周刊|时代周刊]]》的模式发展出来的。奥格斯坦等人延用了该形式，并将其改名为《明镜》。1947年1月4日，第一期《明镜》出版。此时的奥格斯坦才年满23岁。

== “明镜高悬” ==
[[联邦德国|联邦德国]]建立之后，新确立的民主、自由和法治的社会秩序还不成熟。《明镜》这一针对时政的新闻刊物在揭露[[腐败|腐败]]，监督政府和保障人民知情权上贡献颇丰，成为“独立于政客外的良心”。战后西德的多起政坛地震，震源都在《明镜》。鲁道夫·奥格斯坦也因此几次惹祸上身。1949年奥格斯坦第一次被法庭传唤，因为他在《明镜》的报导中揭露了[[石勒苏益格-荷尔斯泰因|石勒苏益格-荷尔斯泰因]]州前农业部长在家中囤积肉罐头被搜出的丑闻。在接二连三的报导联邦政府涉嫌腐败的行为之后，奥格斯坦和其他七名《明镜》的编辑在1962年10月26日因[[叛国罪|叛国罪]]的嫌疑被逮捕，逮捕令是在当时联邦国防部长的[[弗朗茨·约瑟夫·施特劳斯|弗朗茨·约瑟夫·施特劳斯]]的授意下发出的，该事件被称为“[[明镜事件|明镜事件]]”。奥格斯坦等人的被捕激起了社会各界强烈的反响。在经历103天拘留调查之后，奥格斯坦于1963年2月获释。此风波对当时德国的政坛也有很大影响，为此身陷困境的国防部长施特劳斯退出了联邦政坛。开国[[总理|总理]][[阿登纳|阿登纳]]的统治也走到终点。在奥格斯坦获释的同年，阿登纳交班给[[路德维希·艾哈德|路德维希·艾哈德]]，结束了14年的[[德國总理|德國总理]]生涯。

== 风波之后 ==
从1968年到1980年，奥格斯坦先后出版了几本书。在1972年和1973年代表[[德国自由民主党|德国自由民主党]]（FDP）出任[[联邦参议院|联邦参议院]]议员。1974年奥格斯坦把对《明镜》百分之五十的所有权送给杂志的同事。1988年他与时任[[苏联共产党|苏联共产党]][[总书记|总书记]]的[[戈尔巴乔夫|戈尔巴乔夫]]进行了关于“[[新思维|新思维]]”政策的访谈。1989年11月9日，在[[柏林墙|柏林墙]]即将倒塌之时，《明镜》总编埃里希·波梅（{{lang|de|Erich Böhme}}）写了一篇名为《我为什么不希望统一》的评论。奥格斯坦立刻表态，和波梅的立场保持距离。尽管他一贯都对[[德国基督教民主联盟|德国基督教民主联盟]]的政客有很多攻击，奥格斯坦还是在1990年[[两德统一|两德统一]]之前以题为《祝福总理》的评论向[[赫尔穆特·科尔|赫尔穆特·科尔]]的历史功绩表示认同和敬意。1998年奥格斯坦宣布在2003年之后将不参与《明镜》的任何事务。2002年8月26日他在《明镜》上发表了最后一篇评论，内容是关于[[美国|美国]]的[[伊拉克|伊拉克]]政策。

2002年11月7日，奥格斯坦因[[肺炎|肺炎]]病逝于[[汉堡|汉堡]]。12天后，下葬于[[北海_(大西洋)|北海]]上的[[西尔特岛|西尔特岛]]。

奥格斯坦生前对[[宗教|宗教]]曾持批判态度，1968年在他虔诚的母亲去世后，他退出了天主教会，成为了一名[[人本主义者|人本主义者]]，并于千年之交时写文表示：“教会在21世纪将失去所有影响”。尽管如此，2002年11月25日，汉堡的米歇尔[[教堂|教堂]]还是为他举行了宗教悼念活动。

== 私生活 ==
奥格斯坦先后四次离婚。在五段婚姻里共育有四个子女。女儿弗兰西丝卡（{{lang|de|Franziska}}）是《[[南德意志报|南德意志报]]》的记者，儿子雅可布（{{lang|de|Jakob}}）是出版商和《[[时代周报|时代周报]]》（{{lang|de|Die Zeit}}）的特约撰稿人。长女玛丽亚·萨比娜（{{lang|de|Maria Sabine}}）是[[律师|律师]]，积极参与[[同性恋者|同性恋者]]和[[变性人|变性人]]的维权活动。他最小的儿子尤利安（{{lang|de|Julian}}）是[[画家|画家]]和[[经济学家|经济学家]]。
