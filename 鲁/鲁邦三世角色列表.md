: 性格風流、愛好女人，特別是峰不二子。雖然經常騙人，但也常常被峰不二子欺騙。

: 身高：178公分／體重：70公斤／國籍：日本
: 超級神槍手，拔槍射擊只需0.3秒，劇場版是0.2秒，而且幾乎百發百中。魯邦三世忠實的搭檔。
: 有著絡腮鬍，平常戴著大帽子遮蓋住眼睛，愛槍是[[史密斯威森M19左輪手槍|史密斯威森M19左輪手槍]]。除手槍之外也會使用[[通用機槍|通用機槍]]、[[狙擊槍|狙擊槍]]、[[榴彈發射器|榴彈發射器]]和[[反戰車步槍|反戰車步槍]]。
: 嗜酒、是老菸槍，對強勢的女性，特別是峰不二子很不拿手。

== 石川五右衛門 ==
; {{nihongo|石川五右衛門|石川 五ェ門（いしかわ ごえもん）}}
: 聲優：
:* 日本：[[大塚周夫|大塚周夫]]（1971-1972年）／[[井上真樹夫|井上真樹夫]]（1977-1985年、1989-2010年）／[[鹽澤兼人|鹽澤兼人]]（1987年）／[[浪川大輔|浪川大輔]]（2011年起至今）
:* 香港：[[盧傑|盧傑]]（ATV）/[[李鎮然|李鎮然]]（[[鲁邦三世VS名侦探柯南_THE_MOVIE|VS柯南電影]]之電視版）／[[陈健豪|陈健豪]]（[[鲁邦三世VS名侦探柯南_THE_MOVIE|VS柯南電影]]之影碟版）
:* 台灣：[[官志宏|官志宏]]／[[陳幼文|陳幼文]]（[[鲁邦三世VS名侦探柯南|VS柯南特別篇]]、[[鲁邦三世VS名侦探柯南_THE_MOVIE|VS柯南電影版]]）／[[喬資淯|喬資淯]]（[[魯邦三世_PART5|魯邦三世 PART5]]）
: 演員：[[綾野剛|綾野剛]]（香港配音：[[關令翹|關令翹]]）
: 身高：180公分／體重：63公斤／國籍：日本
: 全名「十三代目石川五右衛門」，是[[安土桃山時代|安土桃山時代]]大盜[[石川五右衛門|石川五右衛門]]的第十三世孫。
: 劍法快如閃電，手持一把以隕石做為材料、削鐵如泥的斬鐵劍。沒有他砍不斷的東西，包括人、[[鑽石|鑽石]]、[[飛機|飛機]]、[[人造衛星|人造衛星]]甚至是沒有實體的[[火|火]]、[[雷|雷]]。在砍完各種亂七八糟的東西之後會說「又斬了無聊的東西」且精通醫療藥草的技術。動態視力優秀、能清楚地見到每一顆射擊而出的子彈。身輕如燕，力氣驚人，可以以肉眼就看到魯邦要以望遠鏡看到的距離。
: 同樣對女人沒轍，性格相當純情。
