'''蔥油餅'''（{{lang-en|Onion cake}}）是一種由[[蔥|蔥花]]作成的[[煎餅|煎餅]]，在[[華人|華人]]社會街頭如[[中华人民共和国|中國大陸]]、[[香港|香港]]、[[臺灣|臺灣]]，以及[[马来西亚|馬來西亞]]、[[新加坡|新加坡]]等國家常見，常作為早餐或小吃食用。

== 食用方式 ==
由「茶聖」[[陸羽|陸羽]]喜歡吃的湖北的小吃:「支公餅」演變而來。
傳統蔥油餅做法簡單，麵粉和成麵糰，揉成劑子、搓成條狀、刷豬油、扁成薄皮，再在皮上灑了蔥、鹽、抹油，再包捲成餅胚，然後再用油煎即成。其作法因地區而有所變動。

在台灣做法為在大型鐵盤上淋點油，將或著蔥花與麵糰壓成的餅皮攤盤上煎成，然後再切成4塊、8塊或對半賣，也有揉麵時不加油，然後近乎用烙烤而成的做法。
而浙江的蔥油餅做法則融合了江蘇小吃「蔥油紅燒」與江浙小吃「吳山酥油餅」兩者的特色，再加以改良而來。

== 材料 ==
製作蔥油餅時，普遍會使用到的材料有[[中筋麵粉|中筋麵粉]]、[[食用油|食用油]]、[[蔥|蔥]]、滾水、[[冷水|冷水]]、[[胡椒|胡椒]]、[[食鹽|食鹽]]。

== 製作方法 ==
# 首先把中筋麵粉和鹽巴放入盆中，並加入滾燙的熱水攪拌，最後再倒入冷水拌勻。
# 將麵團揉倒沒有麵粉後，抹上少許的油，蓋上保鮮膜醒半小時以上。
# 等待時間將蔥洗淨，切末，加入鹽、胡椒、食用油勻成蔥油備用。
# 麵團桿平成長方形，把蔥油鋪勻，捲成長條狀。
# 再向內捲成螺旋圓形鬆弛15分鐘左右。
# 將麵團擀平。
# 最後在鍋內放少許油，用中大火煎至兩面金黃即可。
<ref>{{cite web |author1=Hank Chung |title=超簡單的-蔥油餅 |url=https://icook.tw/recipes/62718 |accessdate=2019-01-29}}</ref>

== 吃法 ==

煎完後用[[刀|刀子]]或[[剪刀|剪刀]]切分，通常是切成8等份，將2等份疊和在一起後即為1份，可加[[醬油膏|醬油膏]]，[[辣椒醬|辣椒醬]]或[[胡椒粉|胡椒粉]]等調味，亦有加入[[煎蛋|煎蛋]]。

[[台灣|台灣]]販賣的方式大都是以4分之1個圓餅，也就是以2小片為一個單位，又區分為加蛋與不加蛋。

[[山东|山东]]式葱油饼则是按斤两販賣。
