'''碧水站'''（{{jpn|j=碧水駅|hg=へきすいえき|rm=Hekisui eki|f=y}}）是位於[[北海道|北海道]][[雨龍郡|雨龍郡]][[北龍町|北龍町]]碧水，隸屬於[[日本國有鐵道|日本國有鐵道]]（國鐵）[[札沼線|札沼線]]，已廢除的[[鐵路車站|鐵路車站]]。

== 歷史 ==
* 1931年（昭和6年）10月10日 - [[札沼線|札沼北線]][[石狩沼田車站|石狩沼田]]至中德富（初代，現在的[[新十津川車站|新十津川]]）開始營運。
* 1944年（昭和19年）7月21日 - 由於[[太平洋戰爭|太平洋戰爭]]擴大，本線作為[[不要不急線|不要不急線]]要停止營運。
* 1956年（昭和31年）11月16日 - 札沼線[[雨龍車站|雨龍]]至石狩沼田路段恢復營運，全線回復正常服務。
* 1964年（昭和39年）4月1日 - 車站業務委託化。
* 1972年（昭和47年）6月19日 - 札沼線新十津川至石狩沼田路段廢除，本車站同時廢站。

== 車站構造 ==
站房位於札幌方向的右方西側，為[[側式月台|側式月台]]1面1線的地面車站。而站房旁往石狩沼田方向設有貨物月台，並有1條引入線到達。

== 車站周邊 ==
用地已賣出，而舊站房亦以倉庫使用。
* [[國道275號|國道275號]]、{{Link-ja|國道233號|国道233号}}
* 北海道中央巴士（高速留萌號）、道北巴士、[[沿岸巴士|沿岸巴士]]「碧水」站牌（沿國道233號）
* 北海道中央巴士（瀧川北龍線）「碧水市街」站牌（沿國道275號）
** [[JR北海道巴士|JR北海道巴士]][[石狩線|石狩線]]廢除後的替代路線。
