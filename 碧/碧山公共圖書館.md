'''碧山公共圖書館''' （{{lang-en|Bishan Public Library}}）是一所位于[[碧山|碧山]]规划区{{link-en|新加坡圖書館管理局|National Library Board}}旗下的[[公共圖書館|公共圖書館]], 并于2006年图书馆正式开幕。

==历史==
此图书馆由LOOK Architects设计，于2016年9月1日由当时担任副总理的[[黄根成|黄根成]]宣布正式落成并向公众开放。<ref>{{Cite web|url=http://www.lookarchitects.com/zh/bishan-public-library.html|title=LOOK Architects - 碧山公共图书馆|accessdate=2018-06-09|author=|date=|last=|first=|work=www.lookarchitects.com|publisher=|language=zh-cn|deadurl=no|archiveurl=https://web.archive.org/web/20171120205624/http://www.lookarchitects.com/zh/bishan-public-library.html|archivedate=2017-11-20|df=}}</ref><ref>{{Cite web|url=https://www.nlb.gov.sg/VisitUs/BranchDetails/tabid/140/bid/312/Default.aspx?branch=Bishan+Public+Library|title=National Library Board > Visit Us > Branch Details|accessdate=2018-06-09|work=www.nlb.gov.sg|language=en-US|deadurl=no|archiveurl=https://web.archive.org/web/20180516072653/https://www.nlb.gov.sg/VisitUs/BranchDetails/tabid/140/bid/312/Default.aspx?branch=Bishan+Public+Library|archivedate=2018-05-16|df=}}</ref>

图书馆采用了「树屋」的设计概念， 并在2007年荣获新加坡总统设计奖 （新加波设计界最高荣誉）。<ref>{{Cite web|url=http://m.sohu.com/n/443994048/?pvid=000115_3w|title=读书全指南——新加坡即将推行全国读书运动-新闻频道-手机搜狐|accessdate=2018-06-09|work=m.sohu.com|deadurl=no|archiveurl=https://web.archive.org/web/20180712043503/http://m.sohu.com/n/443994048/?pvid=000115_3w|archivedate=2018-07-12|df=}}</ref><ref>{{cite web|url=https://www.designsingapore.org/pda/award-recipients/2007/bishan-community-library|title=Bishan Community Library|website=Designsingapore|language=en|deadurl=no|archiveurl=https://web.archive.org/web/20180712043729/https://www.designsingapore.org/pda/award-recipients/2007/bishan-community-library|archivedate=2018-07-12|df=}}</ref>

碧山圖書館在2018年「BuzzFeed最佳图书馆名单」中名列第三。<ref>{{cite web|url=https://www.odt.co.nz/news/dunedin/universitys-library-best|title=University's library up with the best|date=2018-03-23|publisher=Otago Daily Times|language=en|deadurl=no|archiveurl=https://web.archive.org/web/20180712043729/https://www.odt.co.nz/news/dunedin/universitys-library-best|archivedate=2018-07-12|df=}}</ref>
