《'''狗與剪刀必有用'''》（{{lang-ja|犬とハサミは使いよう}}），簡稱「狗剪」（{{lang|ja|犬ハサ}}），是[[日本|日本]]的[[輕小說|輕小說]]作品。作者為[[更伊俊介|更伊俊介]]，{{nihongo|[[鍋島哲弘|鍋島哲弘]]|鍋島テツヒロ}}繪製插畫。為第12屆[[Entame大獎小說部門|Entame大獎小說部門]]優秀賞的得獎作品和第一屆《[[店員最愛輕小說大賞|店員最愛輕小說大賞]]》第十位<ref>{{cite web|archiveurl=https://web.archive.org/web/20131112152545/http://himarin.net/archives/6091992.html |archivedate=2013年11月12日 |url=http://himarin.net/archives/6091992.html |title=「ラノベ好き書店員大賞」結果発表　1位：のうりん、2位：やはり俺の青春ラブコメはまちがっている、3位：雨の日のアイリス |accessdate=2015年11月24日 |language=ja |deadurl=yes }}</ref>。2013年7月起播出電視動畫。

== 故事簡介 ==
故事主角'''春海和人'''被[[強盜|強盜]]殺死了。然而，他對於書的執著讓他再一次的復活。只是這次不是身為人，而是身為一隻狗被IGGY動物店的老闆救了。苦於無法看書的他，有一天出現了一位穿著一身黑的女子。她能聽得懂春海和人的話，也是一個以剪刀當兇器的狠毒超級虐待狂，誰知她就是知名大作家，也是自己超喜歡的作家「秋山忍」，之后就寄宿在'''夏野霧姬'''（秋山忍）的家……

== 登場人物 ==
=== 主要角色 ===
; {{nihongo|春海和人|春海 和人（はるみ かずひと）||配音員：[[櫻井孝宏|櫻井孝宏]]（日本）；[[李景唐|李景唐]]（台灣）；[[張裕東|張裕東]]（香港）}}
: 高中生，是個隨興散漫但愛書成癡的人，並且時常吐槽。原與父親和希、母親理緒、妹妹圓香同住，後父親調職，新居住地是[[岡山縣|岡山縣]]<ref name="春海和人、本田文夫">《狗與剪刀必有用1》第13頁</ref>，但因當地的到書日比[[東京|東京]]遲一天，而選擇獨自一人留在東京，因為不太常出門而被房東委託為住谷莊的管理人。
: 在咖啡店中為了保護沉迷於寫作的夏野霧姬而被強盜殺害，死後對書的執念使他復生成為一隻長毛臘腸狗。原先被犬飼救助並暫住於寵物店「IGGY」，後在無書可讀而無盡輪迴的執念呼喚中將夏野霧姬（秋山忍）引來把他帶回家。
: 堅持自己為了保護霧姬而死是「遵從自己意識的行為」而無法怪罪任何人，因此不認同霧姬的自責。時常因為吐槽夏野霧姬（的胸部大小）而慘遭剪刀凌虐。
; {{nihongo|夏野霧姬|夏野 霧姫（なつの きりひめ）||配音員：[[井上麻里奈|井上麻里奈]]（日本）；[[石采薇|石-{采}-薇]]（台灣）；[[羅杏芝|羅杏芝]]（香港）}}
: 20歲作家，進入狀況就對周遭充耳不聞的寫作癡。[[筆名|筆名]]「秋山忍」，具有多部暢銷著作，被譽為「將文壇一刀兩斷的黑色利刃」。書迷大多是年輕族群<ref name="秋山忍">《狗與剪刀必有用1》第45-46頁</ref>，而每一部作品的厚度都幾達凶器的程度。此外是個毒舌兼[[虐待狂|虐待狂]]，常拿著剪刀威嚇和修理和人。
: 完美主義者。在寫每部作品之前，皆會去了解學習內容當中將會描寫的技巧（如武術<ref>在觀看映見使用大澤執筆流後，也懂得使用了。</ref>），理由是有經驗的人寫得比沒經驗的人好。喜歡創造新詞並記錄在隨身詞語典「維霧百科」，另外也會把有興趣的事物寫下留待寫作之用<ref name="書七罪、霧百科">《狗與剪刀必有用1》第108頁</ref>。就和內心一樣，外表和[[內衣|內在美]]都穿的一身黑。外表漂亮，本人自稱三圍：92，58，85，不過實際上卻是[[貧乳|貧乳]]，並且當其他女性在她面前炫耀胸圍時會非常忌妒並黑化。
: 在家中寫作和看書時會戴上眼鏡。
: 父親巧雪、母親姬乃經營理髮店，而姐姐夢見是警察<ref name="霧野夢見">《狗與剪刀必有用1》第274頁</ref>。隨身攜帶並使用父親送的剪刀「剪刀次郎」。
: 因為和人對她的書的執念和因保護她而死，使她對變成狗的和人產生了有如讀心般的連結，能夠聽見和人心中的想法。初時一直認為是自己害死了和人而愧疚不已，聽到和人的真心話後才釋懷，並因此開始對和人投注情感，並且其他女性（不限年齡）親近和人時會嚴重吃醋。當和人對自己做出意料之外的親近行為（逆推）、被和人發現自己的小秘密或喝醉酒時會出現嬌羞的一面，儘管清醒後便回復虐待狂性格，明顯屬於[[傲嬌#女王型|女王型傲嬌]]。

; {{nihongo|大澤映見|大澤 映見（おおさわ はみ）||配音員：[[加隈亞衣|加隈亞衣]]（日本）；[[雷碧文|雷碧文]]（台灣）；[[魏惠娥|魏惠娥]]（香港）}}<ref name="大澤映見">《狗與剪刀必有用3》第285頁</ref>
: 私立東川高中高二學生，和人生前的同班同學，性格膽怯，思考極度消極負面，口頭禪是「對不起」、「我果然還是去死好了」，但在認真起來時擁有堅強的意識。與和人在生前有過約定而立志成為作家，筆名「藤卷螢」就是春海和人的姓名重組(羅馬拼音)，並因此對和人抱有好感。習得家傳大澤執筆流的真傳<ref>如「雙流曲筆」、「無法抑制的睡眠衝動」、「隱形徘句」、「深刻震盪」、「紙南細語」及「操讀者槍」</ref>，曾利用其洗腦讀者對秋山忍多次發動攻擊，之後與秋山忍進行百張執筆戰，但以一張之差距落敗服輸。目前與秋山忍相稱為前後輩關係。
; {{nihongo|秋月真岸|秋月 マキシ（あきづき マキシ）||配音員：[[芹澤優|芹澤優]]（日本）；[[雷碧文|雷碧文]]（台灣）；[[羅孔柔|羅孔柔]]（香港）}}
: 偶像作家，在寫作出道的同時也在偶像界大紅大紫的年輕女孩，被粉絲稱為「閃亮美少女」。本名{{nihongo|「山田槙子」|山田マキ子}}，喜歡穿得一身白，在許多方面與夏野霧姬形成強烈對比（胸部大小、時尚感和代表顏色等）也處處針鋒相對。最討厭被人無視，怕狗（和人）。口頭禪「閃亮（Shining）」。身邊總是跟著大群的黑衣保鑣（兼場地工作員與後援團）。有一個妹妹—{{nihongo|雛子|ひなこ}}。
; {{nihongo|佛坂大門|仏坂 大門（ほとけざか だいもん）||配音員：[[小山力也|小山力也]]（日本）；[[梁興昌|梁興昌]]（台灣）；[[陳廷軒|陳廷軒]]（香港）}}
: 秋月真岸身邊的護衛隊隊長，身形高大魁梧，並負責每日的偶像行程表。每次總在秋月真岸與其他作家發生正式爭執之前，將她強行帶離現場回去工作。
; {{nihongo|姬萩紅葉|姫萩 紅葉（ひめはぎ もみじ）||配音員：[[茅野愛衣|茅野愛衣]]（日本）；[[劉凱寧|劉凱寧]]（台灣）；[[曾佩儀|曾佩儀]]（香港）}}: 與霧姬和槙子共稱當代文學三巨頭的年輕作家，被譽為「深淵的對岸」的藝術派風格。性格[[無口|無口]]的千金小姐，似乎行動不便而坐在[[輪椅|輪椅]]上。
; {{nihongo|森部佐茅|森部 佐茅（もりべ さち）||配音員：[[齋藤千和|齋藤千和]]（日本）；[[雷碧文|雷碧文]]（台灣）；[[何寶珊|何寶珊]]（香港）}}
: 紅葉的專屬女僕，一頭顯眼的紅髮，隨身手持內藏利刃的掃帚「穢殺刃（{{lang|ja|ケガレザッパー}}）」。原派遣公司陰乍的第十三分隊王牌（培養超級女僕的特務機關）。對紅葉侍奉得非常忠心，能完美背誦紅葉著作裡的內容或呈現所寫的動作如舞蹈（也不排除是因為紅葉的內容是拿她描寫的）。有妄想症，喜歡在無人的地方以紅葉的著作內容唱歌跳舞，可一旦被發現被人或動物看見，就會出動「穢殺刃」將對方滅口，不過若與服侍大小姐的工作相衝突的話會自動放棄。
; {{nihongo|姬萩九郎|姫萩 九郎（ひめはぎ くろう）||}}
: 紅葉的哥哥，亦是紅葉的編輯。和鈴菜是同事.因患病而死,復活後變成[[烏鴉|烏鴉]].能聽見春海和人的話.是個[[妺控|妺控]],喜歡設置[[陷阱|陷阱]].生前連續三年蟬聯「書林社內最適合穿管家服的男社員排行榜」榜首.
; {{nihongo|刃具莊老闆娘|はもの荘女将|配音員：[[井口裕香|井口裕香]]（日本）；[[凌晞|凌晞]]（香港）}}
:她在夏野還小的時候就已經在刃具莊裏工作，身型會令人以為是小女孩。另外與住谷夕樹幾乎長得一模一樣，但兩人似乎沒有任何關係。看見圓香想做[[咖哩|咖哩]]但沒了電鋸「鮪吞」而諸多不便，所以把旅館內的「鮪吞零式」送給圓香。
