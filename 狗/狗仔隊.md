'''狗仔隊'''指一些[[跟蹤|跟蹤]]、監視知名人士（例如[[演藝人員|演藝人員]]、[[政治人物|政治人物]]、[[皇室|皇室]]成員、[[運動員|運動員]]及[[商業人士|商業人士]]等），然後[[偷拍|偷拍]]、[[竊聽|竊聽]]及翻查垃圾等活動的[[記者|記者]]。此外，亦可解作[[警察|警察]]專門負責跟蹤及拍攝疑犯的部門，如[[香港警務處|香港警務處]][[刑事情報科|刑事情報科]]、香港[[入境事務處|入境事務處]][[專責調查組|專責調查組]]及[[美國|美國]][[聯邦調查局|聯邦調查局]]等等。

== 起源 ==
狗仔隊這種職業在1958年首次出現，原為[[意大利文|意大利文]]「Paparazzi」是意大利[[姓氏|姓氏]]Paparazzo所对应的单词“蚊子”的复数形式。1960年及1963年，[[意大利|意大利]][[導演|導演]][[費德里柯·費里尼|費德里柯·費里尼]]在[[電影|電影]]《甜蜜生活》（La Dolce Vita）及《[[8½|8½]]》中製造出一个名叫Paparazzo的角色，是一个专门拍摄名人隐私的记者。從此「Paparazzi」就成為那些專門搜刮名人私人生活的形象。

== 形式 ==
狗仔隊通常會駕駛[[摩托車|摩托車]]或[[私家車|私家車]]，在目標人物出沒地方守候，進行拍攝或偷拍，並貼身跟蹤目標人物的車輛。他們甚至會搜索目標人物住過的[[酒店|酒店]]房間，連丟棄的垃圾也不放過，務求知道目標人物不為人知的生活習慣及喜好。

狗仔隊對目標人物造成很大的困擾，除了跟隨不捨帶來的麻煩外，亦侵犯了目標人物的隱私，令目標人物仿如24小時活在公眾的監察之中。

但狗仔隊一般認為，自己對目標人物帶來幫助多於滋擾。狗仔隊為目標人物，尤其是藝人帶來人氣，令更多人留意。狗仔隊本身也是一門賺錢的行業，一幅震撼性的照片，在[[歐洲|歐洲]]、[[美國|美國]]等地可賣得價錢數以萬[[美元|美元]]計。

除了追蹤的攝影記者被稱為狗仔隊，政府政風單位專門抓不正當行為的公務員（例如：搞婚外情、喝花酒、出入不正當場所、收賄賂等），這種被稱為[[政風狗仔|政風狗仔]]。

另外[[徵信社|徵信社]]通常也有狗仔隊的功能。

== 狗仔隊引起的事件 ==
*1997年8月31日，[[英國|英國]]儲妃[[戴安娜王妃|戴安娜王妃]]的司機保羅在[[法國|法國]][[巴黎|巴黎]]駕車擺脫狗仔隊跟蹤時發生車禍，戴安娜王妃及其男友多迪喪生，令人關注狗仔隊對目標人物的滋擾。
*1998年，[[香港|香港]][[東方報業集團|東方報業集團]]旗下《[[東方日報|東方日報]]》與[[壹傳媒|壹傳媒]]的一單訴訟，被[[香港最高法院|最高法院]]判以敗訴。《東方日報》於是採取連串報復行動，包括在報章發表激烈言論，並且連續三天派狗仔隊跟蹤上訴庭法官[[高奕暉|高奕暉]]；結果被[[律政司|律政司]]控以[[藐視法庭|藐視法庭]]，總編輯[[黃陽午|黃陽午]]被判入獄四個月，東方報業集團被判罰款500萬港元。
*近年來亦有一些香港明星，包括一些巨星，頻繁受到狗仔隊的騷擾而告到法庭，如[[張學友|張學友]]、[[成龍|成龍]]等。2006年，組合[[Twins|Twins]]的[[鍾欣桐|鍾欣桐]]在[[馬來西亞|馬來西亞]][[雲頂高原|雲頂高原]]一次演出中被香港《[[壹本便利|壹本便利]]》[[週刊|週刊]]記者偸拍其更衣畫面。該批照片被刊登在同年8月22日出版的《壹本便利》第761期，引起軒然大波，[[香港演藝人協會|香港演藝人協會]]對此强烈不滿，并召開[[記者會|記者會]]聲討。成龍、張學友等人更是呼籲[[香港特別行政區政府|香港政府]]頒佈[[法律|法律]]以制止有關不良行爲。
*2004年，張學友因為狗仔隊跟蹤偷拍其家人，亦有同部份傳媒發生衝突，而此年《[[壹周刊|壹周刊]]》刊登出他與其妻子[[羅美薇|羅美薇]]不和分居的消息，令張學友於當年通過法律橋段起訴《壹週刊》，幷於2005年獲得勝訴。當時香港著名娛樂記者[[查小欣|查小欣]]亦稱張學友被狗仔隊激怒的事件為「佛都有火」。<ref>[http://ent.sina.com.cn/r/2004-03-23/1914341220.html 查小欣 張學友火了（一）佛都有火].南方都市报.2004-03-23</ref>另外張學友曾于1995年創作歌曲《[[過敏世界|過敏世界]]》借此諷刺狗仔隊唯恐天下不亂的行事作風。
*2005年，[[台灣|台灣]]偶像藝人[[周杰倫|周杰倫]]因不滿狗仔隊跟踪自己的私人情感生活，多次和狗仔隊發生衝突，憤然寫下一首名為《[[四面楚歌|四面楚歌]]》的歌曲，發表在自己的音樂專輯《[[十一月的蕭邦|十一月的蕭邦]]》中。在歌詞中，他譏諷娛樂新聞記者像狗那樣為了幾根肉骨頭而搜刮自己的[[隱私|隱私]]，以示不滿和抗議。
*2008年7月23日下午，參加[[梁朝偉|梁朝偉]]和[[劉嘉玲|劉嘉玲]]在[[不丹|不丹]]舉行的婚禮後，[[李亞鵬|李亞鵬]]和[[王菲|王菲]]及家人準備返回[[北京|北京]]，在[[泰國|泰國]]機場遭遇香港《[[忽然1周|忽然一周]]》周刊派出的狗仔隊強行拍攝。狗仔隊不滿足於拍攝李、王二人，開始將鏡頭對準李患有先天[[裂顎|裂顎]]的女兒[[李嫣|李嫣]]，遭到李的反感，隨即對狗仔隊動粗（包括掐住男狗仔脖子，在女狗仔故意挑釁下扇男狗仔耳光等肢體攻擊行為），以保護家人不遭受無辜挑釁和傷害。事後雙方均遭泰國警方傳訊，以互相道歉化解衝突。
*2017年10月23日，[[文匯報|文匯報]]派員跟蹤[[鍾翰林|鍾翰林]]，被指是侵犯鍾的個人私隱。<ref>{{cite news |title=《文匯》跟蹤鍾翰林　學生動源發聲明譴責 |url=https://thestandnews.com/politics/%E6%96%87%E5%8C%AF-%E8%B7%9F%E8%B9%A4%E9%8D%BE%E7%BF%B0%E6%9E%97-%E5%AD%B8%E7%94%9F%E5%8B%95%E6%BA%90%E7%99%BC%E8%81%B2%E6%98%8E%E8%AD%B4%E8%B2%AC/ |date=2017-10-23 |accessdate=2018-08-11 |agency=立場新聞}}</ref>
*2018年4月24日，[[香港|香港]][[香港島選區|港島區]][[立法會議員|立法會議員]][[許智峯|許智峯]]在立法會會議室外，取走保安局一名女[[行政主任|行政主任]]的政府手機，衝入男洗手間內10分鐘，一度拒絕歸還。查閱期間，許智峯發現電話內記錄了大量立法會議員的個人資料、於近月多次立法會會議出入記錄以及身處位置及時間，質疑此做法侵犯議員私隱，並將會向私隱專員提出投訴。惟個人資料私隱專員公署私隱專員黃繼兒指出，政府執行通傳及應變職務並不牴觸《私隱條例》。<ref>[http://hk.on.cc/hk/bkn/cnt/news/20180425/bkn-20180425211850636-0425_00822_001.html 私隱署證政府狗仔隊冇侵私隱　摑許智峯一巴].on_cc東網.2018-04-25</ref><ref>林景輝.[https://www.hk01.com/%E6%94%BF%E6%83%85/181981/%E8%A8%B1%E6%99%BA%E5%B3%AF%E6%89%8B%E6%A9%9F%E9%A2%A8%E6%B3%A2-%E9%81%93%E6%AD%89%E8%81%B2%E6%98%8E%E5%85%A8%E6%96%87-%E7%88%86%E6%94%BF%E5%BA%9C-%E7%8B%97%E4%BB%94%E9%9A%8A-%E5%B7%A5%E4%BD%9C%E7%B4%B0%E7%AF%80 【許智峯手機風波】道歉聲明全文　爆政府「狗仔隊」工作細節].香港01.2018-04-25</ref>

== 立法「反狗仔隊」的可行性 ==
2005年10月，藝人出身的[[美國|美國]][[加州|加州]]州長[[阿诺·施瓦辛格|阿诺·施瓦辛格]]簽署通過《反狗仔隊法》，2006年生效；該法案對“以非法手段取得名人照片者”處以高額罰金，僱主連帶受罰。

臺灣[[世新大學|世新大學]]廣電系助理教授[[管中祥|管中祥]]反對制訂“反狗仔隊”專法，因為強勢的公眾人物被狗仔隊侵擾時大都有反制能力，反而是弱勢的受訪者無力保護自己，因此修正現行法律即可。[[中國文化大學|中國文化大學]]新聞系講師[[馬西屏|馬西屏]]認為，不應該用法律限制狗仔隊，而應該由[[市場機制|市場機制]]與社會進步的共同認知來淘汰狗仔隊。
