'''揚琴'''是一種[[东亚|東亞]]民族樂器，亦有'''洋琴'''、'''打琴'''、'''敲琴'''、'''扇面琴'''、'''蝙蝠琴'''、'''蝴蝶琴'''等稱呼（現代幾乎統一稱作「揚琴」），其兼具[[打擊樂器|打擊樂器]]及[[弦樂器|弦樂器]]之特色，但在[[中樂團|-{zh-tw:國樂團;zh-cn:民族管弦乐团;zh-hk:中樂團;zh:現代中華管弦樂團}-]]裡還是編制為「彈撥樂器」。
== 歷史 ==
[[File:BuikCymbaal.jpg|200px]]
揚琴源於[[中亚|中亞]]地區的「[[桑图尔|桑图尔]]」，後流傳到[[歐洲|歐洲]]、[[北美洲|北美洲]]及[[大洋洲|大洋洲]]，有「{{link-en|德西马琴|Hammered dulcimer}}」（Dulcimer）、「{{Internal link helper/en|辛巴龍|Cimbalom}}」（Cimbalom）、「Hackbrett」及「[[薩泰里琴|薩泰里琴]]」等多種變體。

=== 中國揚琴 ===
中國的揚琴起源尚未有定論，但普遍認同在[[明朝|明朝]]時經海路傳進中國的說法。而中國最早出現揚琴的記載，為[[清朝|清]][[李鍇|李鍇]]所撰《含中集》第二卷提到洋琴。

=== 朝鮮揚琴 ===
朝鮮半島的揚琴稱為「洋琴」（{{Lang|ko|양금}}），在18世紀由中國人傳入朝鮮半島。<ref>{{Cite web|url=http://www.rutadeseda.org/corea/musica/instrumentos8.html|title=Yanggeum - paleum: metal "oro"|accessdate=2017-05-28|author=|date=|work=Enciclopedia Web de Artes Escénicas de Asia|publisher=|deadurl=yes|archiveurl=https://web.archive.org/web/20160811015550/http://www.rutadeseda.org/corea/musica/instrumentos8.html|archivedate=2016-08-11}}</ref>

== 傳統流派 ==

* [[廣東音樂|廣東音樂]]揚琴：講究加花裝飾與潤飾發展，華麗活潑流暢，有冒頭、疊尾、雙持加倍音、贈音帶輪等特有竹法，多以「左竹法」處理的曲目。[[嚴老烈|嚴老烈]]所編《旱天雷》、《倒垂簾》、《連環扣》、《到春來》、《歸來燕》等廣東揚洋琴曲多用「座音襯打法」。[[易其仁|易其仁]]於1920年編著《粵曲洋琴譜》，其所創用的洋琴竹法符號為後人延用。[[丘鶴儔|丘鶴儔]]並提出"竹法十度"說。

* [[江南絲竹|江南絲竹]]揚琴：輕細小雅，有「揚琴一捧煙」說法，多用附點與短拍帶輪。

* [[四川|四川]]揚琴：講究演奏技法的特殊效果，較歡樂雄壯。

* [[東北|東北]]揚琴
* [[潮州弦詩|潮州弦詩]]：少用輪奏法，奏法最為樸素，但頭板加花講究抑揚頓挫，各種催奏法講究緊密對比。潮州洋琴保存最原始型制，如銅絲絃、七平均律等。

* [[山東琴書|山東琴書]]
* [[維吾爾木卡姆|維吾爾木卡姆]]
* [[二人台|內蒙二人台]]

== 制式 ==

* 小揚琴：兩橋式，長期用於說唱音樂的伴奏。

* 大揚琴：兩橋式[[十二平均律|十二平均律]]律呂式大洋琴，為兩橋間採半音調絃法，有高音、低音兩種，由[[鄭寶恒|鄭寶恒]]、[[張子銳|張子銳]]、[[趙立葉|趙立葉]]、[[天津樂器廠|天津樂器廠]]合作，於1953年、1954年間共同製作。其後鄭寶恆、[[袁靜芳|袁靜芳]]、[[蘆寶林|蘆寶林]]合作研製第2代全律活馬大洋琴。

* 3橋變音揚琴：被號稱為「快速轉調洋琴」，於1959年由[[楊竟明|楊竟明]]製作，可以臨時以滾軸變半音轉調。

* 4橋變音洋琴：楊竟明於1961年在三橋變音揚琴的基礎上製作而成。

* 401：楊竟明以變音揚琴的基礎上製作的五橋變音洋琴。以「五‧二多碼並列式」、「分層滾軸板」、「轉調半音槽」解决音域、微調、轉調等問題。

* 402：21世纪常用款型

* 501：[[桂習禮|桂習禮]]、[[李江|李江]]、[[周靖石|周靖石]]於1984年據401揚琴研製而成，於高低音區加進半音調絃法，絃軸彷鋼琴用細螺紋以防止跑絃。

* 505：於501內加鋼棍牽柱及雙梆結構，以抵銷琴絃張力及結合洋琴架，加裝腳踏制音器後而成。

* 601：2004年由[[沈阳音乐学院|沈阳音乐学院]][[刘寒力|刘寒力]]研制并设计音位，[[唐山贵军乐器有限公司|唐山贵军乐器有限公司]][[潘贵军|潘贵军]]制作，这是基于501型扬琴为基础重新研制。音域为B1（大字一组）至f4（小字四组），横向二‧五式排列为主，音域宽广且演奏方便。代表曲目有《金翎思-满乡随想》、《骊山池影》、《竹潇琴梦》等。2005年在北京第八届[[世界扬琴大会|世界扬琴大会]]正式亮相。
== 構造 ==

[[File:Yangqin2.jpg|200px]]
揚琴以擊弦發聲，音箱以木製成，呈扁平[[梯形|梯形]]，上有100多條弦，雙手持「琴竹」擊奏。

=== 弦 ===

現代揚琴有100多條金屬弦，通常由1至5條弦1組發1音，低音弦最粗，有效弦長最長，1至2弦1組，高音弦最細；有效弦長短，4至5弦1組。有效弦長為橋碼左則一段，弦由左右兩端蓋板下的[[螺絲|螺絲]]栓著，須以工具調音。
=== 橋碼 ===
常用的「五排碼」揚琴由右至左稱為低音碼、左碼、中碼、右碼、半音碼。亦有3排和4排橋碼的揚琴。

=== 琴竹 ===
亦稱琴筧、竹鍵，以具彈性的竹製作，為擊弦工具，常用有单音琴竹、单音双面琴竹、双音琴竹等。竹身細長，手握一端扁平；擊弦一端為鎚狀，一面以[[橡皮|橡皮]]或其他彈性材料包覆。

揚琴最初的擊弦工具是木製小槌，音色較琴竹清脆。<gallery>

=== 滾珠及變音槽 ===
[[File:Yangqin4.jpg|200px]]
位於揚琴左右兩側，將弦抬高，以降低[[狼音|狼音]]，部分新式揚琴有「變音槽」，可以迅速移動滾珠，改變有效弦長以改變音高，可方便調音、轉調、奏臨時變化音。

== 奏法 ==
以雙手持琴竹左右交替擊奏或齊擊一音以加強音量。以裸竹一面（即無橡皮琴皮包覆之面）擊弦為「反竹」，音色尖銳有如金屬互碰；以琴竹尾部「鈎奏」，奏出彈撥樂效果。揚琴餘音悠長，可以手指按弦消音。戴上「滑音指套」後可奏[[滑音|滑音]]。
[[File:Huayinzhitao.JPG|200px]]

在[[民族樂團|民族樂團]]中，位於[[指揮|指揮]]前方第一排，常奏[[和音|和音]]或[[琶音|琶音]]。亦為[[胡琴|胡琴]]、[[笛|笛]]等樂器獨奏曲重要的伴奏樂器。

== 音階 ==
[[File:Yangqin5.jpg|300px]]
原為[[五聲音階|五聲音階]]樂器，經增加橋碼後可奏[[半音音階|半音音階]]。

各排由下而上為全音音階。
