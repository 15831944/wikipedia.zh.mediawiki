'''新西伯利亚国立大学'''（{{lang-ru|Новосибирский государственный университет}}，{{lang|ru|НГУ}}），简称'''新大'''，一所位于[[俄罗斯联邦|俄罗斯联邦]][[新西伯利亚州|新西伯利亚州]][[新西伯利亚|新西伯利亚]]市{{link-en|科学城 (俄罗斯)|Akademgorodok|科学城}}的大学，于1959年成立。<ref name="zh-history" />

[[File:Старый_копус_НГУ.jpg|thumb]]

== 历史 ==
1957年，苏联在新西伯利亚城南建立{{link-en|俄罗斯科学院西伯利亚分院|Siberian Branch of the Russian Academy of Sciences|苏联科学院西伯利亚分院}}。科学院分院所在地建成了一个大型科研基地，被称为{{link-en|科学城 (俄罗斯)|Akademgorodok|科学城}}。<ref>{{cite news |title=森林之中的科学殿堂――新西伯利亚科学城见闻 |url=http://news.sina.com.cn/w/2006-08-08/01389682114s.shtml |accessdate=2018-11-26 |work=国际在线 |date=2006-08-08 |archive-url=https://web.archive.org/web/20181126181037/http://news.sina.com.cn/w/2006-08-08/01389682114s.shtml |archive-date=2018-11-26 |dead-url=no }}</ref>分院成立之后，一些科学家担忧分院将来会后继乏人，提议在新西伯利亚创办一所大学，就地培训未来的科学家。另一方面，当时西伯利亚工业发展迅猛，在50年代末其煤炭产量占全苏联的75%，经济发展亦需要大量的基础科学人才。1958年1月，苏联部长会议决议建设新西伯利亚国立大学，隶属苏联科学院西伯利亚分院。同年，新西伯利亚国立大学开设预科课程。<ref name="zh-history">{{cite web |title=历史 |url=https://chinese.nsu.ru/about_NSU/history/ |website=新西伯利亚国立大学 |accessdate=2018-11-26 |archive-url=https://web.archive.org/web/20181126181125/https://chinese.nsu.ru/about_NSU/history/ |archive-date=2018-11-26 |dead-url=no }}</ref><ref name="jyb">{{cite news |title=俄罗斯新西伯利亚大学 |url=http://www.jyb.cn/world/cglx/201001/t20100114_335633.html |accessdate=2018-11-26 |work=新华网 |date=2010-01-14 |archive-url=https://web.archive.org/web/20181126180932/http://www.jyb.cn/world/cglx/201001/t20100114_335633.html |archive-date=2018-11-26 |dead-url=no }}</ref>

1959年9月1日，新西伯利亚国立大学正式开学。<ref name="jyb"/>建校时，学校共有学生310人，仅设自然科学系，有10个专业。学校借用第25中学的教学楼、图书馆、体育馆，此外还使用地质研究所、流体力学研究所的实验室。新西伯利亚国立大学自己的教学楼于1962年建成，主楼则在1963年建成。1963年，首批60名学生毕业，获得专家学位。经过数年的建设、发展，1965年时学校设6个系、46个专业，30名苏联科学院院士、通讯院士在校任教，学校的教员共有40名博士、教授和200名副博士、副教授，就读的本科生有3652名，研究生有94名。建校20年时，大学已向西伯利亚分院各研究所输送了约两千名毕业生，分院约500名研究人员在学校授课，大学与科学院的融合取得了成功。1982年的一项评比中，新西伯利亚国立大学排名苏联第二。苏联时代，新西伯利亚国立大学的政治氛围较其他地方宽松，学生们曾抗议[[華約入侵捷克斯洛伐克|華約入侵捷克斯洛伐克]]、声援{{le|金兹堡、加兰斯科夫案|Trial of the Four|金兹堡、加兰斯科夫等四位被捕的异议人士}}。<ref name="zh-history" />

1992年，新西伯利亚大学在[[乌兰乌德|乌兰乌德]]建立分校。<ref name="zh-history" />2009年，学校被[[俄罗斯联邦|俄罗斯联邦]]列为国家研究型大学。2013年，新西伯利亚国立大学等21所大学被列入{{le|5-100工程|Project 5-100}}，这项计划的目标是使至少五所俄罗斯大学进入全球大学前一百名。<ref>{{cite web |title=排名 |url=https://chinese.nsu.ru/about_NSU/rankings/ |website=新西伯利亚国立大学 |accessdate=2018-11-26 |archive-url=https://web.archive.org/web/20181126221431/https://chinese.nsu.ru/about_NSU/rankings/ |archive-date=2018-11-26 |dead-url=no }}</ref>

== 学术 ==
新西伯利亚国立大学约有7,000名学生，教师约2,000名<ref name="overview">{{cite web |title=新大概况 |url=https://chinese.nsu.ru/about_NSU/ |website=新西伯利亚国立大学 |accessdate=2018-11-26 |archive-url=https://web.archive.org/web/20181126181056/https://chinese.nsu.ru/about_NSU/ |archive-date=2018-11-26 |dead-url=no }}</ref>，80%的教师来自俄罗斯科学院西伯利亚分院。<ref>{{cite web |title=为什么选择新西伯利亚国立大学？ |url=https://chinese.nsu.ru/about_NSU/why-nsu/ |website=新西伯利亚国立大学 |accessdate=2018-11-26 |archive-url=https://web.archive.org/web/20181126180937/https://chinese.nsu.ru/about_NSU/why-nsu/ |archive-date=2018-11-26 |dead-url=no }}</ref>学校教师包括570位拥有博士学位的教授、880位副教授、73位俄罗斯科学院院士。学校有约1,400名留学生，来自40个国家。<ref name="overview" />

新西伯利亚国立大学有6个系、3个学院，即物理系、自然科学系、经济系、力学和数学系、地质与地球物理系、信息技术系、医学与心理学学院、人文学院、哲学和法律学院，此外还设有专业教育科学中心、信息学高等学校。<ref>{{cite web |title=结构 |url=https://chinese.nsu.ru/about_NSU/structure/ |website=新西伯利亚国立大学 |accessdate=2018-11-29}}</ref>大学设85个专业、项目。<ref name="overview" />新西伯利亚国立大学旨在培养数学、力学、物理学、化学、生物学、地质学、地球物理学、地球化学、信息学、计算机技术、经济控制论、历史学、哲学、社会学的研究人才。<ref>{{cite news |title=俄罗斯联邦概况 |url=http://www.npc.gov.cn/npc/wyzzy/cfhgjhy/2015-06/09/content_1937894.htm |accessdate=2018-11-29 |work=新华网 |date=2015-06-09}}</ref>

2019年[[QS世界大学排名|QS世界大学排名]]中，新西伯利亚国立大学排名全球第244名，金砖国家榜第12位，新兴欧洲与中亚地区第2名。<ref name="qs">{{cite web |title=Novosibirsk State University |url=https://www.topuniversities.com/universities/novosibirsk-state-university |website=TOPUNIVERSITIES |publisher=QS Quacquarelli Symonds Limited |accessdate=2018-11-26 |archive-url=https://web.archive.org/web/20181126181318/https://www.topuniversities.com/universities/novosibirsk-state-university |archive-date=2018-11-26 |dead-url=no }}</ref>2018年[[世界大学学术排名|世界大学学术排名]]中，新西伯利亚国立大学的数学、物理学、化学、地球科学、化学工程、仪器仪表制造业、能源科学与机器工程、采矿科学与矿业工程八个专业被列入世界一流学科。<ref>{{cite news |title=NSU在世界ARWU-2018排行榜中的地位有所提高 |url=https://chinese.nsu.ru/news-events/news/research/nsu-strengthens-position-in-arwu-world-ranking-2018/ |accessdate=2018-11-29 |work=新西伯利亚国立大学 |date=2018-08-15}}</ref>

== 校园 ==
新西伯利亚国立大学图书馆藏书量超过90万。<ref name="zh-dyk" />校园内设有健身中心，宿舍楼内设有免费健身房。学校的体育中心是新西伯利亚市最大的体育中心之一。在鄂毕河畔的树林中，学校拥有一处夏季营地，可供师生休养。<ref>{{cite web |title=设施 |url=https://chinese.nsu.ru/life/facilities/ |website=新西伯利亚国立大学 |accessdate=2018-11-26 |archive-url=https://web.archive.org/web/20181126181116/https://chinese.nsu.ru/life/facilities/ |archive-date=2018-11-26 |dead-url=no }}</ref>

== 校友 ==
新西伯利亚国立大学毕业生包括1位{{link-ru|俄羅斯教育學院|Российская академия образования}}院士、31位[[俄罗斯科学院|俄罗斯科学院]]院士、36位俄罗斯科学院{{le|通讯院士|Corresponding member}}。1994年[[菲爾茲獎|菲爾茲獎]]得主[[叶菲姆·泽尔曼诺夫|叶菲姆·泽尔曼诺夫]]是新西伯利亚国立大学毕业生。{{link-ru|维克托·弗拉基米罗维奇·哈里托宁|Харитонин, Виктор Владимирович}}和叶戈尔·尼古拉耶维奇·库利科夫（{{lang|ru|Егор Николаевич Кульков}}）两名校友被《[[福布斯|福布斯]]》杂志列为2016年俄罗斯200大富豪。<ref name="zh-dyk">{{cite web |title=你知道吗？ |url=https://chinese.nsu.ru/about_NSU/did-you-know/ |website=新西伯利亚国立大学 |accessdate=2018-11-26 |archive-url=https://web.archive.org/web/20181126181049/https://chinese.nsu.ru/about_NSU/did-you-know/ |archive-date=2018-11-26 |dead-url=no }}</ref>
