'''新進黨'''（{{lang|ja|しんしんとう}}）是[[日本|日本]]的一個已不存在的[[政黨|政黨]]，簡稱'''新進'''。英文名為New Frontier Party（NFP）。其理念為「自由、公正、友愛、共生」，政策主張為[[新保守主義|新保守主義]]、[[新自由主義|新自由主義]]及[[小政府|小政府]]。

== 歷史 ==
該黨由多個除[[日本社會黨|日本社會黨]]外的在野政黨組成，包括[[新生黨|新生黨]]、[[公明黨|公明黨]]、[[日本新黨|日本新黨]]、[[民社黨_(日本)|民社黨]]和自由改革聯合，1994年12月10日成立，以抗衡執政的自民黨和社會黨[[大聯合政府|大聯合政府]]。 

黨首為[[小澤一郎|小澤一郎]]，1996年舉行的眾議院選舉，新進黨成為國會眾議院第二大黨，取得156席，惟無力阻止自民黨首相[[橋本龍太郎|橋本龍太郎]]單獨組閣。後因黨員政治立場分歧，特別是前自民黨議員及原在野黨議員的矛盾，新進黨於1997年12月31日解散，公明黨重新成為獨立政黨。

== 黨首 ==
* [[海部俊樹|海部俊樹]] （1994-1995）
* [[小澤一郎|小澤一郎]] （1996-1997）

== 解散後 ==
其他成員之後加入自民黨、民主黨、[[自由黨_(日本1998年)|自由黨]]、改革俱樂部、新黨平和、[[新黨友愛|新黨友愛]]、黎明俱樂部、國民之聲等政黨，部分成員更重新加入自民黨。原新進黨及反自民黨的黨員，日後多加入[[民主黨_(日本)|民主黨]]。
