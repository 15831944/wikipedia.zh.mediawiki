'''新竹關帝廟'''，是一座位於[[臺灣|臺灣]][[新竹市|新竹市]][[東區_(新竹市)|東區]]的[[關帝廟|關帝廟]]，於1985年8月19日公告為市定古蹟<ref>{{cite web |url=https://nchdb.boch.gov.tw/assets/overview/monument/19850819000056 |title=新竹關帝廟 |publisher=文化部文化資產局 |accessdate= }}</ref>；該廟主祀[[關聖帝君|關聖帝君]]，從神[[關平|關平]][[太子|太子]]、[[周倉|周倉]][[將軍|將軍]]，配祀[[純陽祖師|純陽祖師]]、[[岳武穆王|岳武穆王]]、[[文昌帝君|文昌帝君]]、[[倉頡|倉頡]]先師、[[魁星|魁星]][[星君|星君]]、[[關公|關公]]三代[[祖先|祖先]]、[[太歲|太歲]][[星君|星君]]、[[赤兔馬|赤兔馬]]與牽馬[[校尉|校尉]]等神尊{{r|竹}}{{r|走}}。

== 沿革 ==
新竹關帝廟由[[淡水同知|淡水同知]][[王右弼|王右弼]]所倡建，於1776年動工，1777年2月完工。當時位於[[竹塹城|竹塹城]]南門之內，附近有衙署、北路右營、[[明志書院|明志書院]]和[[考棚|考棚]]等設施。1871年時同知[[周武濂|周武濂]]曾勸募重修該廟{{r|竹}}。

[[臺灣日治時期|日治時期]]，該廟因被視為官方資產而遭接收改作[[陸軍|陸軍]][[憲兵|憲兵]][[駐屯所|駐屯所]]，直到1898年才恢復成廟宇；但在日治時期後期，廟產又被徵收轉賣，神像則移到[[竹蓮寺|竹蓮寺]]存放{{r|竹}}{{r|走}}。

[[二次大戰|二次大戰]]後，該廟由[[張式穀|張式穀]]、張克繩等人捐獻廟地，並於1949年進行整修，而後在1967年以及2005年再度整修{{r|竹}}，成為今日的樣貌。現為市定古蹟。

此外二次大戰後附近的中巷形成[[私娼寮|私娼寮]]，後來由該廟管理人江雲水買下該土地，改設圖書館、K書中心等設施{{r|走}}。

== 建築特色 ==
關帝廟為三進三開間建築，由於具官廟性質，所以廟門不繪門神而是用門釘{{r|竹}}。
