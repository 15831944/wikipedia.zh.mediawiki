'''新疆省'''，簡稱'''新'''或'''疆'''，是[[中華民國|中華民國]]法理上及[[中國歷史|中國歷史]]上存在的省份。[[清朝|清朝]][[光緒|光緒]]十年（1884年）置，治[[迪化|迪化]]。長期以來由[[中華民國|中華民國]]地方軍閥實質統治至[[中華人民共和國|中華人民共和國]]初期，在1955年被新成立的'''[[新疆維吾爾自治區|新疆維吾爾自治區]]'''取代；而中華民國的新疆省至民國三十八年（1949年）前後遷至[[台北市|台北市]]四維路52巷31號辦公（新疆省政府辦事處），於民國八十一年（1992年）1月16日裁撤，至此無該省的省政府編制。

新疆古稱'''西域'''，自西漢[[神爵|神爵]]二年（前60年）作為[[西域都護府|西域都護府]]第一次納入中國的版圖，隨著[[中原王朝|中原王朝]]的更替興衰，新疆長期被[[遊牧民族|遊牧民族]]與[[中原王朝|中原王朝]]交替統治。[[乾隆|乾隆]]二十二年（1757年），清朝[[乾隆皇帝|清高宗]]重新完成了對西域地區的征服。光緒十年（1884年），[[左宗棠|左宗棠]]驅逐了入侵的[[阿古柏|阿古柏]]軍隊，奏請設立「新疆省」，取「故土新歸」之意。

== 行政管轄範圍 ==
[[File:Zhonghua_Minguo_Quhua_Fanti.svg|thumb]]

中華民國時期的新疆省繼承了清代的版圖，周圍鄰省分別為[[蒙古地方|蒙古地方]]、[[甘肅省_(中華民國)|甘肅省]]、[[青海省_(中華民國)|青海省]]、[[西藏地方|西藏地方]]，鄰國為前[[蘇聯|蘇聯]]、[[阿富汗|阿富汗]]、[[巴基斯坦|巴基斯坦]]、[[印度|印度]]。全省範圍為現今的[[中華人民共和國|中華人民共和國]][[新疆維吾爾自治區|新疆維吾爾自治區]]全境，及[[蒙古國|蒙古國]][[巴彥烏列蓋省|巴彥烏列蓋省]]全境、[[科布多省|科布多省]]西南部，巴基斯坦[[北部地區_(巴控克什米爾)|北部地區]]北部。及未有實際管轄過的[[塔吉克|塔吉克]][[戈爾諾-巴達赫尚自治州|戈爾諾-巴達赫尚自治州]]大部、[[阿富汗|阿富汗]][[巴達赫尚省|巴達赫尚省]][[瓦罕走廊|瓦罕走廊]]地區(即帕米爾高原)。民國三十六年（1947年）全省土地面積為1,711,931平方公里。

== 歷史 ==
{{See also|新疆歷史}}

清代自[[甘肃新疆省|甘肃新疆省]]建立后便设立了在[[陕甘总督|陕甘总督]]、[[甘肅新疆巡撫|甘肅新疆巡撫]]與[[伊犁將軍|伊犁將軍]]共治的构架。[[宣統|宣統]]三年（1911年）[[辛亥革命|辛亥革命]]後[[清朝|清朝]]被[[中華民國|中華民國]]取代，革命黨人湖南人[[劉先俊|劉先俊]]等在[[伊犁|伊犁]]策動起義，殺死[[伊犁將軍|伊犁將軍]][[志銳|志銳]]，成立[[伊犂臨時軍政府|伊犂臨時軍政府]]。新疆[[巡撫|巡撫]][[袁大化|袁大化]]電奏[[清廷|清廷]]保[[阿克蘇|阿克蘇]][[道尹|道尹]][[楊增新|楊增新]]為新疆都督，自己逃離新疆。[[楊增新|楊增新]]在1912年和軍政府議和，逐漸取得全疆控制，並擁護[[中華民國|中華民國]]大總統[[袁世凱|袁世凱]]。[[革命黨|革命黨]]人或被殺死，或回到內地。同年底在[[沙俄|沙俄]]的支持下[[喀爾喀蒙古|喀爾喀蒙古]]宣佈獨立（詳見[[外蒙古獨立|外蒙古獨立]]）。1912年[[庫倫|庫倫]]當局攻陷[[科布多|科布多]]後，又入侵[[蒙古|蒙古]]與新疆之間的[[阿勒泰地區|阿勒泰]]，被[[楊增新|楊增新]]派軍擊敗。1919年，[[北洋政府|北洋政府]]同意將[[阿勒泰地區|阿勒泰]]併入新疆，設立[[阿山道|阿山道]]，即今之[[阿勒泰地區|阿勒泰地區]]。楊增新任內都督又陸續改称為将军、督军等。

民國十七年（1928年）6月，[[北伐|北伐]]結束，7月1日楊增新易幟自行就任新疆省政府主席兼總司令，7月7日被刺身亡。11月17日省政務廳長[[金樹仁|金樹仁]]成為新疆省主席。由於金樹仁的獨裁統治，各地民變迭起，[[和加尼牙孜|和加尼牙孜]]等人首先在哈密發生暴動，導致[[甘肅|甘肅]]軍閥[[馬仲英|馬仲英]]入疆。民國二十二年（1933年）2月，[[馬仲英|馬仲英]]部[[第一次迪化之战|第一次攻打迪化]]，被新疆督辦公署參謀長[[盛世才|盛世才]]擊敗。同年4月12日在一些不滿金樹仁的軍官利用流亡新疆的[[白俄|白俄]][[歸化軍|歸化軍]]發動政變，推舉盛世才為新疆督軍。金樹仁取道[[蘇聯|蘇聯]]逃往[[天津|天津]]。

同年11月，在英國的支持下，和闐人[[穆罕默德·伊敏|穆罕默德·伊敏]]、[[沙比提大毛拉|沙比提大毛拉]]等人在[[疏附|疏附]]宣佈成立[[東土耳其斯坦伊斯蘭共和國|東土耳其斯坦伊斯蘭共和國]]，推舉和加尼牙孜為[[總統|總統]]。民國二十三年（1934年）2月，東土耳其斯坦被馬仲英部下[[馬福元|馬福元]]所滅。穆罕默德·伊敏逃回和闐，又擁立其兄[[滿素爾|滿素爾]]為[[帕夏|帕夏]]，成立「和闐伊斯蘭王國」，隨即被[[馬虎山|馬虎山]]所部消滅。
[[File:Flag_of_Xinjiang-Shicai.svg|thumb]]
盛世才前期依靠蘇聯支持鞏固政權，進而控制了新疆全境。為表示其親共，盛世才前往蘇聯考察，還邀請[[中共|中共]][[陳潭秋|陳潭秋]]、[[毛澤民|毛澤民]]、[[林基路|林基路]]等人到新疆工作。後期和中共決裂，民國三十二年（1943年）國民政府人員隨同國軍部隊進入新疆。在整個[[抗日战争|抗日战争]]期間，新疆相對獨立，是抗戰的大後方。盛世才還派軍支持了[[綏遠抗戰|綏遠抗戰]]等戰役。

[[File:Flag_of_Xinjiang-Shicai_2.svg|thumb]]1942-1944年所用省旗]]
民國三十三年（1944年）1月，[[伊宁事变|伊宁事变]]爆发。在[[苏联政府|苏联政府]]的支持下，新疆省开始了反對[[國民政府|國民政府]]統治的「[[三區革命|三區革命]]」（「三區」是指當時新疆的[[伊犁|伊犁]]、[[塔城|塔城]]和[[阿勒泰|阿勒泰]]三個地區），再度打出「[[東突厥斯坦|東突厥斯坦]]」的旗號，並殘酷屠殺當地的漢族回族人民。9月，[[蔣中正|蔣中正]]將盛世才調到[[重慶|重慶]]任国民政府林業部長，派[[吳忠信|吳忠信]]任新疆省長，並命令回族军阀[[馬步芳|馬步芳]]派一個軍的騎兵駐紮在新疆各地，對[[東突厥斯坦獨立運動|東突厥斯坦獨立運動]]的活動進行鎮壓。民國三十五年（1946年）6月，「三區革命」領導人[[阿合買提江|阿合買提江]]、[[阿不都克里木·阿巴索夫|阿不都克里木·阿巴索夫]]等撤銷了[[艾力汗·吐烈|艾力汗·吐烈]]的職務，將「東突厥斯坦共和國」改組為伊犁專區參議會。爆发三区革命的地区，始终效忠中央政府的汉族民众被大肆屠杀，其中，[[伊宁事变|伊宁事变]]中，随着国军战败，伊宁的汉族民众被屠戮殆尽。而在三区当地民众中却形成归化苏联国籍的高潮，更加深了[[新疆苏联侨民问题|新疆固有的苏联侨民问题]]。这一社会问题在此后深刻影响新疆十余年。中国方面研究者将苏联政府归化新疆民众举动，视为其在中国新疆构建势力基础，更有形容为“[[鸠占鹊巢|鸠占鹊巢]]”。

民國三十八年（1949年），[[第二次国共内战|第二次国共内战]]末期，[[中國人民解放軍|中國人民解放軍]]盡佔優勢，國軍節節敗退。在[[彭德懷|彭德懷]]领导的[[第一野戰軍|第一野戰軍]]8月26日佔領[[蘭州|蘭州]]繼而佔領[[甘肅省_(中華民國)|甘肃省]]全境之後，新疆的[[國民革命軍|國民革命軍]]軍政長官[[陶峙岳|陶峙岳]]、[[包爾漢|包爾漢]]等在9月25日、26日相继通电，宣佈脫離迁至[[廣州|廣州]]的中华民国政府[[起義|起義]]。期間已經投向共產黨一邊的[[張治中|張治中]]在中斡旋，而[[中共|中共]]方面的談判代表是[[鄧力群|鄧力群]]。同年前蘇聯讓起義軍停止進入迪化市（今[[烏魯木齊|烏魯木齊]]），並讓起義軍投降於中華人民共和國政府。[[新疆和平解放|新疆和平解放]]后的11月，解放军第一野戰軍第一兵團在[[王震|王震]]將軍率領下进入迪化市。其后，收编国军起义部队、接管政务。1950年初，解放军攻占新疆全境，使其成為[[中華人民共和國|中華人民共和國]]的一省。其后，中华人民共和国政府通过[[新疆剿匪|新疆剿匪]]，清剿[[疆独|疆独]]和中华民国政府残余的武装。1951年7月1日[[堯樂博士|堯樂博士]]奉命於[[台北市|台北市]]四維路52巷31號設立新疆省政府主席辦公處，1971年改組為新疆省政府辦事處，於1992年1月16日裁撤<ref>[http://hc.nccu.edu.tw/public/view.php?main=3&sub=24&ssub=43&id=1831 2015.4.11法提合先生、耿慶芝女士演講「漫談新疆省政府辦事處」]</ref>。

== 行政區劃 ==
=== 省、縣之間的行政區 ===
==== 道制 ====
民國2年（1913年）1月發布臨時大總統令，廢除府州廳制度，實施新的道縣制度，保留了清代的[[鎮迪道|鎮迪道]]、[[伊塔道|伊塔道]]、[[阿克蘇道|阿克蘇道]]、[[喀什噶爾道|喀什噶爾道]]4道。民國3年（1914年）5月，改鎮迪道為[[迪化道|迪化道]]、改伊塔道為[[伊犁道|伊犁道]]。民國5年（1916年）6月、置[[塔城道|塔城道]]。民國8年（1919年）6月，[[阿爾泰區域|阿爾泰區域]]併入新疆省，置[[阿山道|阿山道]]。民國9年（1920年），於新疆省南部的阿克蘇道、喀什噶道分置焉耆道、和闐道，至此共有8道。至民國17年（1928年）道制正式廢止。

==== 行政區 ====
民國17年（1928年）廢道後，因新疆幅員遼闊，治理不便，新疆省政府按原8道區域改置8個行政區。民國22年（1933年）四一二政變後，增置哈密區。民國31年（1942年），增置莎車區。民國32年（1943年）7月才廢除行政區，依內政部頒佈的《省行政督察專員暫行條例》，設行政督察區。

; [[迪化行政區|迪化行政區]]
:民國17年（1928年）以原迪化道置「'''迪化行政區'''」，行政長駐迪化縣（今[[烏魯木齊市|烏魯木齊市]]區），轄迪化、哈密、奇臺、孚遠、阜康、昌吉、綏來、鄯善、鎮西、呼圖璧10縣。同年7月增領乾德縣。民國19年（1930年）2月增領木壘河縣，10月增領七角井設治局。民國22年（1933年）哈密、鎮西5縣劃入哈密行政區，轄區減為10縣1設治局。民國32年（1943年）7月廢除迪化行政區，設「'''新疆省第一行政督察區'''」。

==== 行政督察區 ====

民國21年（1932年）內政部頒佈《省行政督察專員暫行條例》後，新疆省未立即行，至民國32年（1943年）7月，新疆省才將10個行政區改為行政督察區，各設督察專員1人。民國36年（1947年）調整各區轄縣。[[中共建政|中共建政]]後由各專區取代。

=== 縣級行政區 ===
清代新疆省在宣統三年（1911年）時，分為6府、8直隸廳及2直隸州，下轄1廳、1州、21縣。辛亥革命後，民國二年（1913年）4月，各府、廳、州改置為縣，計有37縣。因新疆地域廣大，在地域廣闊的縣設置了多個縣佐，這些縣佐實際作用改與設治局相似，成為與縣相近的獨立行政區劃。其後，南京國民政府頒佈《縣組織法》等法律，規定縣不設縣佐。由於新疆各縣佐已經具有行政區劃的意義，不可能裁併入原來的縣，為此新疆省將經濟條件較好、基本達到設縣標準的縣佐改升為縣；未達標準的，則改設治局。此後，新疆省縣、設治局的置廢，往往由省府決定，事後再申報內政部。[[盛世才|盛世才]]時期先後設置了5縣及6設治局，而未報內政部備案。民國33年（1944年）盛世才下台後。國民政府接管新疆後，盛世才統治時期在新疆增設、裁撤、升格及更名的20餘個縣及設治局即產生非法問題。為此，新疆省政府以「僻處邊陲，交通不便，民族複雜，情形特殊」為由上報國民政府內政部，於民國36年（1947年12月17日內政部方字第1226號公函將上述問題一併解決。

== 人口 ==
據民國十七年（1928年）[[中華民國內政部|內政部]]的統計，新疆省的總人口約為255萬人，其中人口構成方面，[[維吾爾人|維吾爾人]]佔約70%，[[漢人|漢人]]約10%以下。
