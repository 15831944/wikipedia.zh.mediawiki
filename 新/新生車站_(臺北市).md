'''新生車站'''位於[[臺灣|臺灣]][[臺北市|台北市]][[中山區_(臺北市)|中山區]]，是一座[[台鐵|台鐵]][[縱貫線_(北段)|縱貫線]]上已經廢除的[[鐵路車站|鐵路車站]]，舊名'''北台北車站'''。後來在[[板南線|台北捷運藍線（南港線）]]於附近設有[[忠孝新生站|忠孝新生站]]。

== 車站構造 ==
*岸式月台兩座。
== 利用狀況 ==
*目前已廢除，廢除前為招呼站。

== 車站週邊 ==
*[[光華商場|光華商場]]
*[[台北工專|台北工專]]（今[[台北科技大學|台北科技大學]]）

== 歷史 ==
*1930年：設北台北臨時停車場，僅停汽油車。
*1942年：因缺乏汽油，汽油車停駛。
*1953年12月1日：設立新生招呼站<ref>台灣鐵路古今站名，楊鵬飛著</ref>。
*{{fact|廢止年代不詳。}}
