'''新華澳報'''，是[[澳門|澳門]]一家私營的親[[建制派|建制派]][[中文|中文]][[報紙|報紙]]，前身為1989年12月20日創刊的[[中文傳媒|中文傳媒]]私營周報《華澳郵報》<ref name ="WA">{{cite web|title=《新華澳報》簡介|url=http://www.waou.com.mo/aboutus/|website=《新華澳報》官方網站}}</ref><ref>{{cite web|title=練乙錚：曹二寶理論探討 中聯辦澳門落實|url=http://hktext.blogspot.com.tr/2009/04/blog-post_7030.html?m=1|website=港文集|accessdate=2009-04-20}}</ref>。

新華澳報起初每周星期三出版。1994年改為日報《華澳日報》，每天出紙一張半，1995年改為一大張並逢周日休息。於2000年再改以今名。該報社長兼總編輯為[[林昶_(澳門)|林昶]]，副社長兼总经理為[[周海燕|周海燕]]<ref name ="WA"/>。

==歷史==
《新華澳報》最初名為《華澳郵報》，後來在1994年12月20日更名為《華澳日報》並與[[保利達集團|保利達集團]]主要股東[[柯為湘|柯為湘]]的弟弟[[柯為雄|柯為雄]]合辦，1999年，《華澳日報》因勞資糾紛而被[[法院|法院]]下令封產，停刊不久後重組為現時的《新華澳報》<ref>{{cite journal |author1=王濱洋 |title=入主九龍建業「澳門李嘉誠」大起底 |journal=《[[壹週刊|壹週刊]]》|date=2002-01-09 |issue=618 |page=102}}</ref>。

==評價==
