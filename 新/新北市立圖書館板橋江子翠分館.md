'''新北市立圖書館板橋江子翠分館'''為[[臺灣|臺灣]][[新北市|新北市]]的一座[[圖書館|圖書館]]，建築物地址為新北市板橋區莊敬路62號。

民國68年，為配合政府推動十二項重大建設及教育部「建立縣市文化中心計畫」成立「臺北縣立文化中心籌備處」，選定板橋市莊敬路62號，由高而潘建築師規劃設計，民國72年正式啟用。由於文化中心的落成啟用，原縣立圖書館業務移由文化中心負責而走入歷史，縣級圖書館的發展也步入轉型期。

臺北縣立圖書館籌備處縣立文化中心於89年1月改設成立文化局，同時以任務編組成立臺北縣立圖書館籌備處。當時，基於圖書館乃為社會教育機構，其有關業務移由教育局主政，並商請臺北縣立泰山國中教師何文慶支援擔任籌備處主任。因著前瞻考量，圖書館業務於89年8月移回文化局主政。

『臺北縣立圖書館組織規程』亦於九十三年八月十八日經本府法規委員會審議通過，93年11月8日經縣長發佈，刊登於臺北縣政府公報冬字第七期，94年1月1日生效，並於1月2日舉行揭牌典禮，由代理縣長林錫耀親自主持。

99年12月25日，新北市立圖書館配合臺北縣改制為新北市，更名為『新北市立圖書館』，為市圖總館。總館於104年5月10日正式搬遷至板橋區貴興路139號，原總館舊址亦更名為新北市立圖書館板橋江子翠分館。<ref>何玉華.[http://news.ltn.com.tw/news/local/paper/854318 〈台北都會〉新北市圖總館將搬新家 現址變分館].自由時報.2015-02-08</ref>

江子翠分館成立後，考量建築物歷經30多年歲月，於106年1月開始進行空間改造，為搭配原有建築物挑高結構及外觀設計，本次設計搭配紅磚外觀及大片的復古圓拱窗，建築本身即是藝術，整體以舊倉庫意象作為主視覺設計空間，3樓以桁架、紅磚、復舊為主要元素，2樓延續3樓舊倉庫意象以貨櫃、棧板為主要元素，展現「視覺藝術」，表現生活美學及現代網路資訊之時代性，歷經近半年的整修，全新面貌於106年7月15日試營運，7月26日全面對外開放。

== 歷史 ==
* 1923年11月（[[大正|大正]]12年）：[[日治時期|日治時期]]始設「[[海山郡|海山郡]]圖書館」，後改為「板橋圖書館」，為臺北縣立圖書館的前身。<ref>[http://www.library.ntpc.gov.tw/htmlcnt/doquerysinglehtmlcnt/866169fe67124e17a83cb00f8e94a062 歷史沿革].新北市立圖書館.2016-01-16</ref>
* 1946年1月16日[[臺北縣政府|臺北縣政府]]成立，1947年1月16日[[縣治|縣治]]遷於[[板橋區_(新北市)|板橋鎮]]並接管板橋圖書館。
* 1948年奉[[臺灣省政府教育廳|臺灣省政府教育廳]]核准成立「臺北縣立圖書館」。
* 《臺北縣立圖書館組織規程》於2004年8月18日經[[新北市政府法規委員會|臺北縣政府法規委員會]]審議通過，2005年1月1日生效。2005年1月2日，臺北縣立圖書館舉行簡單隆重之揭牌典禮，由時任[[新北市市長|臺北縣代理縣長]][[林錫耀|林錫耀]]親自主持。
* 2010年12月25日，為配合臺北縣改制為[[新北市|新北市]]，特於當天更名為「新北市立圖書館」。
* 2015年5月10日，總館遷址至板橋區貴興路139號新館舍。新總館起辦24小時不打烊作業。<ref>[http://www.femh.org.tw/research/news_detail.aspx?NewsNo=4373&Class=1 讀.書.人專訪／廿四小時不打烊，新北市立圖書館的挑戰與克服].聯合報.2015-04-24{{webarchive|url=https://web.archive.org/web/20160304101347/http://www.femh.org.tw/research/news_detail.aspx?NewsNo=4373&Class=1 |date=2016-03-04 }}</ref>

== 樓層 ==
*3樓： 自習區、書庫、書報閱覽、展覽區
*2樓： 幼兒閱覽區、綜合服務台、電腦查閱、影印服務
*1樓： 黃龜理紀念館（藝文展覽廳）、自習室、書報閱覽、24小時還書箱
*B1樓：藝文展覽廳

== 開放時間 ==
星期一，日上午八點半開放至下午五點。星期二至六上午八點半開放至下午九點。

==週遭==
位處[[新北市|新北市]]立[[文化中心|文化中心]]旁，並於後門設置[[蒸汽機車|蒸汽機車]]展示館（Steam Locomotive Exhibition House），存放'''[[台鐵DT650型蒸汽機車|DT675]]'''[[蒸汽火車|蒸汽火車]]一台，與[[淡水區|淡水區]]老街入口廣場存放的'''[[台鐵BK10型蒸汽機車|BK20]]'''相互對應。
