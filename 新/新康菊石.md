'''新康菊石'''（[[學名|學名]]：{{lang|la|'''''Neocomites'''''}}），又名'''紐康菊石'''、'''新克姆菊石'''，是生存在[[白堊紀|白堊紀]]初[[貝里亞期|貝里亞期]]至[[豪特里維期|豪特里維期]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]。牠們是[[侏羅紀|侏羅紀]]的[[旋菊石超科|旋菊石超科]]演化成白堊紀的[[帶菊石超科|帶菊石超科]]過程中的過渡型類群，因此學者對於其分類定位上的看法有些分歧。<ref name="Neocomites">{{Cite web |url = http://liusir.pixnet.net/blog/post/248634863-neocomites-%E6%96%B0%E5%BA%B7%E8%8F%8A%E7%9F%B3-%28%E6%B3%95%E5%9C%8B%29 |title = Neocomites 新康菊石 (法國) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-10 }}</ref>其[[化石|化石]]被發現於[[歐洲|歐洲]]、[[北非|北非]]、[[馬達加斯加|馬達加斯加]]、[[印度|印度]]北部、[[婆羅洲|婆羅洲]]、[[蘇門答臘|蘇門答臘]]和[[美洲|美洲]]等地。<ref name="Fossilworks" />

== 特徵 ==
新康菊石的殼體內捲，殼形扁平，腹側平坦，兩側肋條的末端和龍骨相互連接，形成「魚骨」狀的構造，是這類菊石的重要特徵。<ref name="Neocomites" />
