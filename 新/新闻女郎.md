'''《新闻女郎》'''（{{jpn|j='''ニュースの女'''|kk=ニュースのおんな|rm=Nyūsu no On'na|f=no}}）是[[日本|日本]]一部由[[铃木保奈美|铃木保奈美]]、[[泷泽秀明|泷泽秀明]]及 [[長塚京三|長塚京三]] 主演的[[电视剧|电视剧]]，该剧于1998年1月7日至3月18日在[[富士电视网|富士电视网]]{{Link-ja|水曜剧场 (富士电视台)|水曜劇場 (フジテレビ)|水曜剧场}}（每周三[[日本标准时间|日本标准时间]]21:00至21:54）播出。<ref name="douban">{{cite web |title=《新闻女郎》 |url=https://movie.douban.com/subject/1484469/ |publisher=[[豆瓣|豆瓣]] |accessdate=2018-08-12}}</ref><ref name="youku">{{cite web |title=《新闻女郎》资料 |url=http://list.youku.com/show/id_zcbfc9ce6962411de83b1.html |publisher=[[优酷网|优酷网]] |accessdate=2018-08-12}}</ref><ref name="rating">{{cite web |title=ニュースの女 |url=http://dramascore.com/drama_detail.php?o=RNaIUBQvcpJygHotwsfxDEKM&d=2036&n=165&r=index_all |publisher=ドラマ視聴率集計 |accessdate=2018-08-12}}</ref>

== 剧情介绍 ==
主要电视台“二频道（{{lang|en|Channel 2}}）”新闻部的记者麻生环被提拔为《晚间新闻（{{lang|en|Evening News}}）》的新闻主播；而她与政治学家工藤海渡的婚期也已经确定下来，何谓是工作与生活双丰收的人生赢家。唯一美中不足的是，工藤海渡与他前妻的儿子北原龙将同他们一起生活。龙对他们的态度非常不友好，而环则是抱着“我们并没有住在一起，我和他没有什么关系”的态度。而在环担任新闻主播的第一次播报时，一条突发新闻插入了进来……<ref name="youku" /><ref name="douban" />
