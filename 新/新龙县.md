'''新龙县'''（{{bo|z=ཉག་རོང་རྫོང|w=nyag rong rdzong}}）位于[[四川|四川]][[甘孜藏族自治州|甘孜藏族自治州]]中部、[[雅砻江|雅砻江]]流域。人口约4.27万（2001年），[[海拔|海拔]]2760-5992米。

== 历史 ==
新龙县原名瞻对，地处川藏南北两条大道之间。[[同治|同治]]四年（1865年），[[康区|康区]]发生农奴大起义，篡夺[[瞻对土司|瞻对土司]]之位的[[波日·工布朗结|波日·工布朗结]]侵扰大道，清廷命川藏两地合剿。乱定后，清廷将该地赏给[[达赖喇嘛|达赖喇嘛]]管理，命其派[[堪布丹傑|堪布]]建庙化导当地居民。但当地不奉[[格鲁派|格鲁派]]，西藏地方政府派代本一人，率兵驻扎瞻对。清末，西藏地方开始以瞻对为跳板，试图夺取川边备土司地方，引起川边社会动荡。[[光绪|光绪]]二十年（1894年），[[四川|四川]][[总督|总督]][[鹿傳霖|鹿传霖]]提出将瞻对收归，次第经营各土司的建议。清政府一度将瞻对划在四川辖下。但后因达赖喇嘛的请求和[[驻藏大臣|驻藏大臣]][[文海|文海]]、[[成都将军|成都将军]][[恭寿|恭寿]]的联合反对，清廷只得将瞻对划归西藏。

1912年[[清|清]]设瞻对委员辖地改置怀柔县，1916年改名瞻化县，1952年改新龙县。

== 行政区划 ==
新龙县辖1个镇、18个乡：
*镇：茹龙镇。
*乡：沙堆乡、乐安乡、大盖乡、绕鲁乡、色威乡、甲拉西乡、拉日马乡、博美乡、尤拉西乡、子拖西乡、和平乡、洛古乡、雄龙西乡、麻日乡、通霄乡、友谊乡、皮擦乡、银多乡。

== 地理 ==
新龙矿藏较丰富，以[[金矿|金矿]]最为著名，被特称为“瞻金”，据称[[拉萨三大寺|拉萨三大寺]]的佛像金饰主要采用“瞻金”。
