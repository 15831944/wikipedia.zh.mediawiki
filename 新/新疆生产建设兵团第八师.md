'''新疆生产建设兵团第八师'''，是[[新疆生产建设兵团|新疆生产建设兵团]]的师，与[[新疆维吾尔自治区|新疆维吾尔自治区]][[石河子市|石河子市]][[师市合一|师市合一]]。

==沿革==

新疆生产建设兵团第八师前身是[[中国人民解放军|中国人民解放军]]第一野战军二十二兵团第九军步兵第二十六师，1950年开始在石河子一代垦植拓荒。

1975年4月，新疆军区生产建设兵团及其所属师建制撤销，成立石河子地区，下辖石河子市、沙湾县、玛纳斯县以及原农八师农七师的农牧团场。1981年12月恢复新疆生产建设兵团，农八师建制同时恢复，与石河子市师市合一。<ref>{{cite web |title=走进兵团农八师师部 |url=http://bingtuan.ts.cn/content/2009-05/27/content_4264010.htm |website=天山网 |accessdate=2018-11-25 |date=2009-05-27}}</ref><ref>{{cite web |title=历史沿革 |url=http://www.shz.gov.cn/structure/newSite/New_Channel_2/New_Channel_1/New_Page_2?infid=11 |publisher=石河子市人民政府 |accessdate=2018-11-25}}</ref>

===团场===
2016年11月团场改革完成后，第八师管辖十四个团场，其中4个中心团场（121、133、134、143），10个一般团场（136、141、142、144、石总场、147、148、149、150、152）。<ref>{{cite web |title=行政区划、区域 |url=http://www.shz.gov.cn/structure/newSite/New_Channel_2/New_Channel_1/New_Page_2?infid=14 |publisher=石河子市人民政府 |accessdate=2018-11-25}}</ref>
