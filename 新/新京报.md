《'''新京报'''》是[[中华人民共和国|中国]][[北京|北京]]一份由[[中国共产党北京市委员会|中共北京市委]]宣傳部主辦的综合类大型城市[[日报|日报]]，创刊于2003年11月11日，主要在北京地区发行。


== 命名 ==
《新京报》创办之初，原计划命名为《北京时报》，后来改为《[[京报|京报]]》，在以《京报》的名义作了一段前期宣传之后，由于某种原因，最后出版时定名为《新京报》。

== 报道风格 ==
新京报曾是中国首家获正式批准的跨地区联合办报试点，也是中国首家股份制结构的时政类报纸。主要採編部門負責人以《[[南方日報|南方日報]]》人員為主，骨干采编人员均来自《[[南方都市报|南方都市报]]》，因而以自由派自居。首任總編是[[程益中|程益中]]，社長是[[戴自更|戴自更]]。该报自称“一出生就风华正茂”，在建立之前及初期，曾经引起了北京新闻圈的波动和讨论。

与中国[[华南|南方]]的知名媒体《南方都市报》相似，《新京报》曾率先报道了[[河北定州暴力事件|河北定州暴力事件]]和[[2005年吉林化工厂爆炸|哈尔滨水污染事件]]，是中国大陆媒体中比较善于打[[擦边球|政策擦边球]]的报纸之一，外界也将《新京报》看作是敢言媒体。在2005年撤编事件发生前，中国媒传界普遍认为《新京报》是中国最有公信力的报纸之一。撤编事件发生后，《新京报》由直属中宣部的《[[光明日报|光明日报]]》主管，光明日报报业集团和[[南方报业传媒集团|南方报业传媒集团]]联合主办（光明日报社控股佔51%，南方日报社佔49%），2011年9月1日被北京市委宣傳部接管<ref>{{cite web|url=http://info.bjnews.com.cn/news/2009/0801/35.shtml|title=关于我们|accessdate=2011-09-05|date=2009-08-01|publisher=新京报|archiveurl=https://web.archive.org/web/20110904051711/http://info.bjnews.com.cn/news/2009/0801/35.shtml|archivedate=2011-09-04|deadurl=yes}}</ref>，9月3日新京报版权页更新。

《新京报》主管部门变更后，该报记者仍然可以进行异地监督，经常都会刊登与异地监督有关的深度调查报道，但到各省調查採訪的阻力會大大增加。2017年1月发生在天津[[独流镇|独流镇]]的调料造假案便由本报记者率先调查曝光。该报道也迫使食药监、公安、检察等机关，对该案展开调查，警方也对10余名涉案人员实施拘捕<ref>{{cite news|url=http://news.qq.com/a/20170116/001176.htm|title=天津独流每天数千件假调料发全国 进入北京已四五年|date=2017-01-16|publisher=新京报|accessdate=2017-02-04}}</ref><ref>{{cite news|url=http://news.xinhuanet.com/ttgg/2017-01/24/c_1120371705.htm|title=检方介入天津独流镇调料造假案 一监管部门“先进集体”被撤|date=2017-01-16|publisher=新京报|accessdate=2017-02-04}}</ref>。

== 发行與競爭者 ==
《新京报》每天发行87.1万份<ref>{{cite news|url=http://info.bjnews.com.cn/news/2009/0801/35.shtml|title=新京报：关于我们|date=2009-08-01|publisher=新京报网|accessdate=2010-12-13|deadurl=yes|archiveurl=https://web.archive.org/web/20101221113050/http://info.bjnews.com.cn/news/2009/0801/35.shtml|archivedate=2010-12-21}}</ref>，发行中心分零售和征订两个部门。零售部门负责往报摊送报纸和自己零售。征订部门负责征订和往订户家送报纸。征订的工作时间是早5点开始到晚上9点。

2012年总发行量为87.1万份，其中征订量为54.88万份，零售量为30.74万份，赠阅量为1.48万份。

《新京报》在北京的主要竞争对手是早午报的《[[京华时报|京华时报]]》（现已停刊）、《[[北京青年报|北京青年报]]》，还有下午的都市报包括立场偏保守的《[[北京晚报|北京晚报]]》。

== 主要事件 ==
2004年3月，《新京报》首任总编程益中在[[南方都市报事件|南方都市报事件]]中被当局免职，逮捕。

2006年5月10日，[[王菲|王菲]]前夫[[竇唯|竇唯]]因不滿該報卓姓記者對他與高原之報道，當天上午十點衝進永安路報社內砸毀電視機與電腦各一台，被報社職員冷靜後離開；但至當天下午五時許，竇再前來報社門前，把門前的一輛轎車以汽油點燃，火勢很快被職員以滅火筒撲滅，竇則被帶往派出所；期間報社有職員阻止拍攝。<ref>{{cite web|url=http://news.xinhuanet.com/legal/2006-05/11/content_4532848.htm|title=砸电脑烧汽车 窦唯对报道不满大闹新京报社|accessdate=2010-05-13|date=2006-05-11|publisher=新华网}}</ref> 報社事後堅持其報道正確，並無對竇有任何侵權事宜，而對事件保留追究責任，竇於翌日凌晨三點被押往[[宣武区|宣武]]第五分局看守所並不准保釋。<ref>{{cite web|url=http://news.thebeijingnews.com/0190/2006/0511/015@179486.htm|title=窦唯火烧路边汽车|accessdate=2010-05-13|date=2006-05-11|publisher=新京报|archiveurl=https://web.archive.org/web/20070927010148/http://news.thebeijingnews.com/0190/2006/0511/015@179486.htm|archivedate=2007-09-27|deadurl=yes}}</ref>

=== 撤编事件 === 
{{main|新京报撤编事件}}
2005年12月下旬，总编[[杨斌_(编辑)|杨斌]]和两名副总编孙雪东、李多钰被突然免职，但没有说明解除他们职务的原因。随后报馆约100名员工罢工抗议，最终2位副总编的职位被恢复。由于新京报的知名度和中国国内对媒体的审查环境，故此起事件引发海内外广泛关注。

===刊登六四照片被收回===
2008年7月24日，《新京报》刊登了记者访问美籍华裔记者[[刘香成|刘香成]]的报道。虽然报道中没有提及[[六四事件|六四事件]]，但报道中出现了一张有数名青年骑着三轮车，用木板载着伤者，高速行走的照片，而伤者都是在镇压时中枪受伤的学生。事发后，当日的《新京报》被紧急收回，同时[[中共中央宣传部|中宣部]]及[[中华人民共和国新闻出版总署|国家新闻出版总署]]已对该事件下令追查。此事件也被定性为“严重政治事件”<ref>{{cite web|url=http://news.bbc.co.uk/chinese/simp/hi/newsid_7520000/newsid_7524900/7524975.stm|title=《新京报》登六四照片后据报被调查|accessdate=2016-10-17|date=2008-07-25|publisher=BBC中文网}}</ref>。

=== 被貶為地方性報紙 ===
2011年9月3日《[[北京日報|北京日報]]》報道<ref>{{cite web|url=http://bjrb.bjd.com.cn/html/2011-09/03/content_447365.htm|title=經國家新聞出版總署批准 《京華時報》《新京報》變更為北京市主管主辦|accessdate=2011-09-03|date=2011-09-03|publisher=北京日報|archiveurl=https://web.archive.org/web/20111113223227/http://bjrb.bjd.com.cn/html/2011-09/03/content_447365.htm|archivedate=2011-11-13|deadurl=yes}}</ref>：經國家新聞出版總署批准，《新京報》與[[人民日報|人民日報]]社主管主辦的《[[京华时报|京華時報]]》從即日起被北京市委宣傳部主管主辦；北京市新聞出版局負責人表示，兩報主管主辦方的變更是落實中央關於深化文化體制改革要求的重要舉措，報道最後引述負責人表示：堅持正確輿論導向，圍繞中心，服務大局，推動首都報業健康快速發展。此舉被海外媒體認為是從中央直屬媒體貶為地方性媒體，到各省調查採訪的阻力會大大增加，變相收緊對社會監督的尺度。

===传被京报集团收购===
2016年10月17日，有网络传闻《新京报》将并入北京日报报业集团。后新京报官方微博发布消息称，此为假消息，“绝无此事”<ref>{{cite web|url=http://finance.sina.com.cn/roll/2016-10-17/doc-ifxwvpqh7612775.shtml|title=新京报辟谣并入北京日报集团：绝无此事|accessdate=2016-10-17|date=2016-10-17|publisher=中国经济网}}</ref>。

===媒体改革===
2018年8月13日，经中共北京市委深改组会议审议通过，中共北京市委印发《新京报》、《[[北京晨报|北京晨报]]》和[[千龙网|千龙网]]整合方案，将上述三家媒体实现整合<ref>{{cite web|url=https://www.thepaper.cn/newsDetail_forward_2585001|title=北京市推进媒体融合：新京报、北京晨报、千龙网将整合|accessdate=2018-10-31|date=2018-11-09|publisher=澎湃新闻}}</ref>。
