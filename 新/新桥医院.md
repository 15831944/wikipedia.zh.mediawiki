'''新桥医院'''（'''中国人民解放军陆军军医大学第二附属医院'''），位于[[中国|中国]][[重庆市|重庆市]][[沙坪坝区|沙坪坝区]]新桥正街83号，隶属[[中国人民解放军陆军军医大学|中国人民解放军陆军军医大学]]。

==简介==
新桥医院的前身是'''国民政府军政部陆军医院'''，始建于1944年。1950年，改编为'''中国人民解放军西南军区总医院'''。1955年转隶中国人民解放军第七军医大学，为'''中国人民解放军第七军医大学第二附属医院'''，对外称'''新桥医院'''。1975年，第七军医大学改称第三军医大学，该医院随之改称'''中国人民解放军第三军医大学第二附属医院'''。该医院现址由[[贺龙|贺龙]]、[[邓小平|邓小平]]等领导亲自选定。该医院是国家第一批“[[三级甲等医院|三级甲等医院]]”，也是重庆市第一批“涉外定点医院”<ref>{{Cite web |url =http://www.xqhospital.com.cn/hospital/hospital01_new.asp  |title =  医院简介|author =  |date =  |publisher = 新桥医院 |language =  |accessdate = 2017-06-06 |quote =  }}</ref>。2017年，改称'''中国人民解放军陆军军医大学第二附属医院'''。

新桥医院在心血管内科、心血管外科、呼吸内科、脊柱外科、肾移植、肿瘤治疗等学科的综合实力位居[[中华人民共和国|中华人民共和国]]国内前列<ref name=xunli>{{Cite web |url = http://www.81.cn/jwgz/2017-06/04/content_7627632.htm |title = 2017军校巡礼第六站：第三军医大学（附报考指南） |author =  |date = 2017-06-04  |publisher = 中国军网 |language =  |accessdate =  |quote =  }}</ref>。
