'''新北市立明志國民中學'''，為一間三年制的[[國民中學|國民中學]]，位於[[新北市|新北市]][[三重區|三重區]][[中正路|中正北路]]107號。現有職員兩百二十一人及學生兩千餘名。

== 沿革 ==
*1954年，成立[[國立臺北商業大學|省立台北商業職業學校]]三重分部。
*1961年，改隸為[[新北市立新莊國民中學|台北縣立新莊中學]]三重分部。
*1963年八月，改制為台北縣立明志[[初級中學|初級中學]]。
*1968年九月，為因應[[九年國民義務教育|九年國民義務教育]]，改制為台北縣立明志國民中學。並由五谷王街舊址（[[興穀國小|興穀國小]]）遷至現址。
*2012年，招標整修操場跑道及廁所，也因此八年級停辦暑期輔導，九年級則縮減暑期輔導的時間。校慶中的園遊會也停辦。隔年因施工不當，廁所天花板有漏水，於是重新招標整修。
*2013年，以「飛越半世紀 舞動新明志」為主題舉行敲鑼、SLOGAN設計競賽、LOGO設計競賽等五十周年校慶活動，各個班級將教室布置也融入五十周年校慶的主題。
*2015年，污水接管工程完工
*2016年八月，新北市國民中學資訊輔導團召集人、工藝樓Aˋ和平樓Aˋ仁愛樓B校舍補強工程完工
*2016年十一月，閱覽室閱覽區裝修工程完工
*2014～2017三年共減班15班，面臨成為中型學校的危機

== 特殊班級 ==
*語資班：國文、英語課會抽離原班至語資教室上課。其他課堂與非語資班同學共同上課。
*體育班：與一般班級不同，第七節課安排彈性課程--體育專項。其他時間與普通班無異。
*資源班
