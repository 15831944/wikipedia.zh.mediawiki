'''新川站'''（{{jpn|j=新川駅|hg=しんかわえき|rm=Shinkawa eki|f=y}}）是位於[[北海道|北海道]][[札幌市|札幌市]][[北區_(札幌市)|北區]]新川4條1丁目3，隸屬於[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）[[札沼線|札沼線]]（學園都市線）的[[鐵路車站|鐵路車站]]。車站編號為'''G04'''。電報略號為'''{{lang|ja|ニイ}}'''。

== 歷史 ==
* 1986年（昭和61年）11月1日 - [[日本國有鐵道|日本國有鐵道]]（國鐵）[[札沼線|札沼線]]'''新川臨時乘降場'''開始營運<ref name=school>{{cite web|url=http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |language=ja |title=札沼線（学園都市線）の電化について |publisher=JR北海道 | date=2009-09-09 |accessdate=2017-08-30 |archive-url=https://web.archive.org/web/20091006234745/http://www.jrhokkaido.co.jp/press/2009/090909-2.pdf |archive-date=2009-10-06 }}</ref><ref name=school2>{{cite web |url=http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |language=ja |title=札沼線（学園都市線）の電化開業時期について |publisher=JR北海道 |date=2011-10-03 |accessdate=2017-08-30 |archive-url=https://web.archive.org/web/20111205160350/http://www.jrhokkaido.co.jp/press/2011/111013-3.pdf |archive-date=2011-12-05 |deadurl=yes }}</ref>，處理乘客業務。當時為1面1線車站。
* 1987年（昭和62年）4月1日 - 由於[[國鐵分割民營化|國鐵分割民營化]]，車站由[[北海道旅客鐵道|JR北海道]]管理。同時升格為'''新川站'''。
* 1991年（平成3年）
** 3月16日 - 札沼線的暱稱設定為「'''學園都市線'''」<ref name=school/><ref name=school2/>。
** 12月25日 - 成為業務委託車站，有職員駐守。
* 1996年（平成8年）6月23日 - 札沼線（學園都市線）[[八軒車站|八軒車站]]及八軒站至本車站部份路段[[交通立體化|高架化]]。
* 1999年（平成11年）8月22日 - 札沼線（學園都市線）本車站至[[新琴似車站|新琴似車站]]路段高架化，本車站成為架空車站。當時只有現在的1號月台暫時營運。
* 2000年（平成12年）
** 3月11日 - 新站房落成，同時本車站變成2面2線。札沼線（學園都市線）八軒站至[[太平車站_(日本)|太平車站]]新設下行線，變成[[複線鐵路|複線鐵路]]。
** 日期不詳 - 札沼線[[桑園車站|桑園車站]]至[[石狩月形車站|石狩月形車站]]路段引入自動進路制御裝置（PRC）。
* 2006年（平成18年）1月 - 引入自動列車導向裝置。並設置電子告示板及按方向的導向用液晶顯示屏。
* 2007年（平成19年）10月1日 - 設定車站編號（'''G04'''）<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |language=ja |title=駅番号表示（駅ナンバリング）を実施します |publisher=JR北海道 |date=2007-09-12 |accessdate=2017-08-30 |archive-url=https://web.archive.org/web/20070930015220/http://www.jrhokkaido.co.jp/press/2007/070912-3.pdf |archive-date=2007-09-30 |deadurl=yes }}</ref>。
* 2008年（平成20年）10月25日 - 開始使用[[儲值車票|智能車票]]「[[Kitaca|Kitaca]]」<ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |language=ja |title=Kitacaサービス開始日決定について |publisher=JR北海道 |date=2008-09-10 |accessdate=2017-08-30 |archive-url=https://web.archive.org/web/20080913103748/http://www.jrhokkaido.co.jp/press/2008/080910-1.pdf |archive-date=2008-09-13 |deadurl=yes }}</ref>。
* 2012年（平成24年）6月1日 - 札沼線（學園都市線）桑園站至北海道醫療大學站正式電氣化（[[交流電|交流電]]20,000[[伏特|V]]、50[[赫兹|Hz]]）<ref>{{cite web|url=http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |language=ja |title=学園都市線電化開業に伴う電車の投入（第一次）について |publisher=JR北海道 | date=2012-03-14 |accessdate=2017-08-30 |archive-url=https://web.archive.org/web/20120925203437/http://www.jrhokkaido.co.jp/press/2012/120314-1.pdf |archive-date=2012-09-25 }}</ref><ref>{{cite web |url=http://www.jrhokkaido.co.jp/press/2012/120803-1.pdf |language=ja |title=平成24年10月ダイヤ改正について |publisher=JR北海道 |date=2012-08-03 |accessdate=2017-08-30 |archive-url=https://web.archive.org/web/20120925203427/http://www.jrhokkaido.co.jp/press/2012/120803-1.pdf |archive-date=2012-09-25 |deadurl=yes }}</ref>。

== 車站構造 ==
現時為[[側式月台|側式月台]]2面2線的高架車站。月台除了樓梯及升降機外，亦設置了扶手電梯。

直至2012年5月31日，本車站與[[八軒車站|八軒車站]]及[[新琴似車站|新琴似車站]]是少數非電氣化的高架及複線鐵路車站。高架化工程之前為1面1線的地面車站。在八軒站與本車站路段進行高架化工程時，首先於當時道床的東側興建架空單線（現時上行線），並於本車站的西方側設置臨時路軌及道床（比現時下行線更外側）。現時的站房位於舊車站之上，工程期間的臨時月台則位於高架部份的西側下方。當現時站房與上行線暫時開始使用後，開始舖設下行線。

由[[桑園車站|桑園車站]]管理的業務委託車站（北海道JR Servicenet），設有[[綠窗口|綠窗口]]（營業時間7時00分至20時00分）、[[自動售票機|自動售票機]]及簡易[[驗票閘門|自動閘機]]（只對應入閘部份，出閘時需要將已使用的車票投進箱內，[[智慧卡|IC卡]]則同時對應出入閘）。過去亦設有[[Kiosk|Kiosk]]。
