'''新埔車站'''是一座位於臺灣[[苗栗縣|苗栗縣]][[通霄鎮|通霄鎮]]的[[鐵路車站|鐵路車站]]，其所屬路線為[[臺灣鐵路管理局|臺灣鐵路管理局]][[海岸線_(臺鐵)|海岸線]]，是[[西部幹線|西部幹線]]與海岸線距離最近的車站，現經指定為[[苗栗縣歷史建築|苗栗縣歷史建築]]。

== 車站構造 ==
*木造車站。
*[[島式月台|島式月臺]]、[[岸式月台|岸式月臺]]各一座。
*跨線[[天橋|天橋]]。
*新埔=通霄：雙軌區間北端。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#簡易站|乙種簡易站]]，僅有[[臺鐵區間車|區間車]]停靠和一人站員服務。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1922年10月11日：「新埔驛」開業。
*1986年6月1日：由[[三等站|三等站]]降為[[甲種簡易站|甲種簡易站]]，並指定由[[白沙屯車站|白沙屯站]]管理。
*2015年6月30日：啟用多卡通刷卡機。
*2017年9月16日：由[[甲種簡易站|甲種簡易站]]調整為[[臺灣鐵路管理局車站等級#簡易站|乙種簡易站]]，並改由[[通霄車站|通霄站]]管理。
