'''新國家連線'''為[[中華民國|中華民國]]曾經存在的[[中華民國政黨|政党]]之一，前身為[[立法院|立法院]]的一個次級[[黨團|黨團]]'''建立新國家陣線'''，由離開[[民主進步黨|民主進步黨]]的[[立法委員|立法委員]][[陳永興|陳永興]]、[[蕭裕珍|蕭裕珍]]、[[彭百顯|彭百顯]]與[[許添財|許添財]]成立，立黨理念和[[彭明敏|彭明敏]]接近。召集人為彭百顯。

==概要==
1997年2月24日，原民主進步黨[[正義連線|正義連線]]三位立法委員彭百顯、陳永興、許添財，與已退出民主進步黨並加入[[建國黨|建國黨]]的[[陳光復|陳光復]]、[[陳文輝_(臺灣)|陳文輝]]，宣布脫離民主進步黨立法院黨團，成立'''建立新國家陣線'''獨立運作；他們同時預告，他們只是建立新國家陣線第一階段的基本成員，預計民主進步黨還會有五至六位立法委員陸續加入。建立新國家陣線成立時人事：發言人許添財、召集人彭百顯、執行長陳永興<ref>{{Cite web|url=http://news.cts.com.tw/cts/general/199702/199702240001561.html|author=|language=zh-tw|title=五原民進黨立委籌組新政團|publisher=[[華視新聞|華視新聞]]|date=1997-02-24|accessdate=2015-07-04}}</ref>。

「堅持建國理念、專業清流問政」為建立新國家陣線成員共同追求的目標。建立新國家陣線的立法委員過去在[[台灣獨立運動|台灣獨立建國]]、[[環境保護|環境保護]]、本土文化教育、[[社區營造|社區營造]]等議題上的專業與付出，讓許多人印象深刻。其他成員如[[陳婉真|陳婉真]]、[[陳欽隆|陳欽隆]]、[[蔡明華|蔡明華]]等人也都擁有相當知名度。

1998年9月18日，由於當時[[民進黨主席|民進黨主席]][[許信良|許信良]]的[[大膽西進|大膽西進]]爭議與建國黨建黨的影響，建立新國家陣線改組為政黨'''新國家連線'''，並提名十八位立法委員候選人與四名[[台北市議員|台北市議員]]候選人參與年底三合一選舉（即[[1998年中華民國直轄市市長暨市議員選舉|1998年中華民國直轄市市長暨市議員選舉]]與[[1998年中華民國立法委員選舉|1998年中華民國立法委員選舉]]），並由彭明敏出任榮譽會長、許添財出任秘書長，定位為一個[[柔性政黨|柔性政黨]]，不招收黨員，只以理念、政策和人才號召選民的支持。

新國家連線成立時，打出「優質政治、關鍵力量」的口號，強調：「[[中國國民黨|國民黨]]對外[[一個中國|一個中國]]、對內[[黑道|黑道]][[金權政治|金權]]包袱嚴重；民進黨雖然促進台灣[[民主化|民主化]]有功，但已日趨國民黨化，尤其提名名單良莠不齊，一些才德不足人士假藉黨內民主挾持民意」，新國家連線「因而必須結合有志，給台灣選民多一種選擇；預期在未來『多黨不過半』的[[立法院|立法院]]和市議會裡，我們將扮演關鍵少數的角色」。新國家連線並且要求選民把「各政黨籃子裡的壞蘋果」淘汰出去，使優質政治成為可能；進一步更要堅持[[台灣主體性|台灣主體]]、捍衛台灣利益，尤其攸關台灣定位和生活品質的教育文化政策、攸關台灣安全的國防外交政策，以及攸關台灣發展的經濟、環保政策。

新國家連線具體政見如下：
*反對大膽西進，以台灣主體、平等互惠原則，謹慎與中國發展經貿文化關係。 
*加強與[[東南亞|東南亞]]國家的區域經貿合作，謀求互補共利。 
*推動[[國防工業|國防工業]][[民營化|民營化]]，以發展成長久持續的民間支援體系。 
*教育內容力求台灣化、生活化，藉本土化的人文、史地、藝術教育凝聚台灣人的國民意識。 
*國民有免費進入[[國立大學|國立大學]]的權利；成立「大學經費分配委員會」，合理分配[[高等教育|高等教育]][[預算|預算]]。 
*制定實施[[多語|多語]]政策，建立多元族群文化的社會。 
*修改《國民教育法》，確立學生學習權，保障家長參與學校教育的充分機會。 
*推動社區主義，激發民間力量，落實[[地方自治|地方自治]]。 
*全面改革稅制，建立公平、合理的經濟社會。 
*修改不利[[中小企業|中小企業]]的法令，協助中小企業取得企業轉型必須的技術。 
*改善經營環境，鼓勵低污染、低耗能、高附加價值的產業。 
*開辦[[國民年金保險|國民年金保險]]，以保障老人與失業者的經濟安全。 
*檢討[[全民健保|全民健保]]制度，減少醫療浪費，公平負擔保費，提昇醫療品質。 
*協助落後國家經濟發展，以台灣名義加入[[聯合國|聯合國]]。 
*制定《[[公民投票法|公民投票法]]》，推動[[公投|公投]]入憲，制定[[三權分立|三權分立]]、[[單一國會|單一國會]]的台灣新憲法。 

新國家連線企圖與同質性高的建國黨進行切割，說「不贊成建國黨的[[剛性政黨|剛性]]做法」，還說「目前新國家連線與建國黨雖有以上的不同，但愛台灣的苦心與建國的決心彼此是一樣的，將來仍不排除所有獨派重組一個政黨的可能性」。並且，新國家連線不提出[[全國不分區|全國不分區]]立法委員候選人名單，理由是抗議「國民黨的不分區代表名單，充滿向[[黑金|黑金]]地方勢力妥協及利益交換的色彩；民進黨的名單也不乏形象惡劣，不敢直接接受選民考驗，只賴[[人頭黨員|人頭黨員]]躋身安全名單之列，殊不合理」。

在[[1998年中華民國直轄市市長暨市議員選舉|1998年中華民國直轄市市長暨市議員選舉]]中，新國家連線提名之台北市議員候選人無人當選，立法委員候選人僅有許添財當選；而許添財不久後又加入民進黨立法院黨團運作，隨後加入民進黨；而其餘新國家連線成員，除重新加入民進黨外<ref>{{Cite web |url = http://www.hi-on.org.tw/bulletins.jsp?b_ID=43834 |title = 台灣人大團結　新國家連線重回民進黨 |author =  |date = 2001-10-25 |publisher = 彭明敏文教基金會 |language = zh-hant |accessdate =  |quote = }}</ref>，也有不少人加入[[台灣團結聯盟|台灣團結聯盟]]。

2018年12月12日，[[中華民國內政部|中華民國內政部]]公告，新國家連線自2018年12月5日解散。<ref>{{cite web |title=公告新國家連線107年12月5日解散 |url=https://party.moi.gov.tw/pgms/politics/home!newsDetail.action?id=ff8080816791bf360167a4d587a71621 |website=內政部政黨及全國性政治團體資訊網 |language=zh-Hant-TW}}</ref>
