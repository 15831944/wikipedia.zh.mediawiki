'''新丸之内大厦'''（{{lang-ja|新丸の内ビルディング}}）是位於[[日本|日本]][[东京都|东京都]][[千代田区|千代田区]][[丸之内|丸之内]]的[[摩天大楼|摩天大楼]]，简称“新丸大厦”。<ref name="SMaB SCP">{{cite web |url=http://skyscraperpage.com/cities/?buildingID=30776 |title=Shin-Marunouchi Building |accessdate= |publisher=Skyscraperpage.com}}</ref><ref name="SMaB EMP">{{cite web |url=http://www.emporis.com/en/wm/bu/?id=newshinmarunouchibuilding-tokyo-japan |title=New Shin-Marunouchi Building |accessdate= |publisher=Emporis.com}}</ref>

== 概要 ==
新丸之内大厦由1952年建造的8层高的旧新丸之内大厦（{{lang|ja|新丸ノ内ビルヂング}}）改建而成，位於[[东京站|东京站]]丸之内北口前，與[[丸之內大廈|丸之內大廈]]相隔[[行幸通|行幸通]]對街而鄰。改建後為楼层38层、楼高197.6米的综合大楼，是东京都千代田区最高的大楼，由[[英国|英国]]建筑师{{link-en|迈克尔·霍普金斯|Michael Hopkins (architect)}}爵士设计，于2007年4月19日竣工，2007年4月27日正式对公众开放。从1998年开始的「丸之内曼哈顿计划」以来，新丸之内大厦是丸之内第3大商业设施，仅次于{{link-ja|丸之內OAZO|丸の内オアゾ}}与{{link-ja|東京大廈|東京ビルディング (丸の内)|東京大廈}}。[[商場|商場]]空間主要设在地下1楼～7楼，拥有153家店铺，9楼～37楼为办公区域。
