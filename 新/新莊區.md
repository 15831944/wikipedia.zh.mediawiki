'''新莊區'''是[[臺灣|臺灣]][[新北市|新北市]]下轄的[[區_(中華民國)|區]]，位於全市西側近[[桃園市|桃園市]]處，屬於[[臺北都會區|臺北都會區]]的[[衛星城市|衛星城市]]之一；人口逾41萬，居於新北市各區第2位，擁有不少在[[臺北市區|臺北市區]]工作求學的[[通勤|通勤]]人口。其既有[[新莊廟街|新莊廟街]]等發展逾百年的市街聚落，亦有[[新北產業園區|新北產業園區]]、[[化成工業區|化成工業區]]、[[西盛工業區|西盛工業區]]等製造業及輕工業基地。2000年以後，區內新開發[[頭前重劃區|頭前重劃區]]及[[新莊副都心|新莊副都心]]，[[行政院|行政院]]亦在後者籌建[[行政院新莊聯合辦公大樓|中央合署辦公大樓]]<ref>{{cite web|url = http://www.chinatimes.com/realtimenews/20140128003205-260405|title = 行政院新莊聯合辦公大樓啟用|author = 謝幸恩|date = 2014-01-28|work = 中時電子報}}</ref>。

== 歷史 ==
18世紀以前，[[凱達格蘭族|凱達格蘭族]][[武朥灣社|武朥灣社]]曾在新莊平原活動。[[觀音山|觀音山]]（八里坌山）曾名興直山，新莊平原位於興直山下，故名[[興直堡|興直堡]]。

18世紀漢人入墾後，通稱為新莊，即「新興的街莊」之意。新莊作為[[淡水港|淡水港]]的內港，被譽為[[北臺灣|北臺灣]]開發最早的地區之一。在[[清朝|清朝]][[乾隆|乾隆]][[嘉慶|嘉慶]]年間舟船熙攘、商賈聚集曾盛極一時，擁有「千帆林立新莊港，市肆聚千家燈火」之美譽。為[[臺北盆地|臺北盆地]]中心都市，土興人旺。[[嘉慶|嘉慶]]中葉以後，因淡水河河道淤積失去航運之便，交通輻輳地位讓予[[艋舺|艋舺]]，新莊的發展也就此停滯。沉寂一時的新莊到[[光緒|光緒]]中葉，因[[臺灣鐵路|臺灣鐵路]]興建完成有了重振跡象。但日據初期鐵路改線經由枋橋（今[[板橋區_(台灣)|板橋]]），新莊再次喪失交通地位。

[[台灣日治時期|日治時期]][[大正|大正]]9年（1920年）隸屬[[臺北州|臺北州]][[新莊郡|新莊郡]]，轄[[新莊街|新莊街]]、[[鷺洲庄|鷺洲-{庄}-]]、[[五股庄|五股-{庄}-]]和[[林口庄|林口-{庄}-]]，是當時農業重鎮。又因臨近[[臺北市|臺北市]]，新莊的商業活動也跟著興盛起來。[[二次大戰|二次大戰]]末期，[[臺灣總督府|總督府]]為避免[[盟軍|盟軍]]轟炸，疏散[[臺北市|臺北市]]的重要工廠都移往鄰近鄉鎮，該政策使新莊的工業開始發展{{r|志|page=47}}。

1945年國民政府光復臺灣後，原臺北州新莊郡改為[[新北市|臺北縣]]新莊區，新莊街則改制為新莊鎮，並成立新莊鎮公所及鎮民代表會。1950年，新莊鎮和泰山鄉分治，新莊區則遭到廢止，新莊鎮行政地位確立。頭前工業區、西盛工業區的成立，是新莊從農業市鎮進入工業都市的重要轉捩點{{r|志|page=48}}。新莊的工廠分佈於四個地區：化成路頭前工業區、新樹路西盛工業區、新莊路兩側、中正路丹鳳段，產業以化學、紡織、機械、金屬所佔的比例最大{{r|王天麟}}，這幾項產業吸引了大量的工業人口，外地人相繼遷入人口急增。1973年1月20日，臺北縣政府頒布實施「新莊都市計畫」，定下了新莊鎮的都市紋理與空間格局{{r|都計頒布日}}。與此同時，[[宏國集團|宏國集團]]、[[林榮三|聯邦集團]]、[[宏泰集團|宏泰集團]]等建設公司紛紛在新莊開發大規模的的公寓社區，如宏安、聯邦、宏泰、正邦、正安、宏友、宏全、元興、金鑫等社區。新莊的人口自1970年五萬兩千人至1980年改縣轄市前夕已達十六萬八千人，十年之間社會增加十二萬人，新遷入居民多過世居的新莊人兩倍{{r|新泰豹比}}，1980年7月1日因破15萬人口而改制為[[縣轄市|縣轄市]]{{r|中國時報19800115}}。2009年，設籍人數突破40萬<ref>[https://www.ca.ntpc.gov.tw/home.jsp?id=220&parentpath=0,2,43,219 新北市歷年人口統計 ，請參照98年度09月。]，新北市新莊戶政事務所</ref>。2010年12月25日，因[[2010年中華民國縣市改制直轄市|五都升格]]，台北縣新莊市改制為新北市新莊區。2012年1月5日捷運[[新莊線|新莊線]]開通。2017年3月2日[[桃園國際機場捷運|桃園國際機場捷運]]開通。

== 地理 ==
新莊位於臺灣島北部，新北市西部[[大漢溪|大漢溪]]以北，面積19.7383平方公里。新莊東鄰[[三重區|三重區]]，南邊以[[大漢溪|大漢溪]]與[[板橋區_(台灣)|板橋區]]相望，西南以[[塔寮坑溪|塔寮坑溪]]與[[樹林區|樹林區]]相鄰，西接[[桃園市|桃園市]][[龜山區|龜山區]]，北部分別與[[泰山區_(台灣)|泰山區]]和[[五股區|五股區]]接壤。

== 政區 ==
[[File:新莊區公所.JPG|200px]]
由於目前人口眾多，習慣上於其下劃分[[次分區|次分區]]。有[[興直|興直]]、[[頭前|頭前]]、[[中港|中港]]、[[福營|福營]]與[[西盛|西盛]]等五個次分區。唯此劃分僅是為行政便利所作的權宜措施，無獨立的次分區機關，次分區亦非法律上的公法人。同樣的情況亦出現於台灣各個其他高度[[都市化|都市化]]的[[市轄區|市轄區]]，如[[板橋區_(台灣)|板橋區]]、[[中和區|中和區]]、[[三重區|三重區]]、[[桃園市|桃園市]][[桃園區|桃園區]]。

除此之外，當地傳統以新樹路為界被分為「上新莊」（或「北新莊」，含興直、頭前、中港次分區，此地名與位在[[淡水區|淡水區]]的「北新莊子」無關）和「下新莊」（或「南新莊」，含丹鳳、西盛次分區）兩個區域。

== 經濟 ==
=== 工業 ===
為臺灣工業重鎮，產業結構中二級產業從業比例達57%，境內有經濟部工業局所轄的[[新北產業園區|新北產業園區]]，以及縣級的[[瓊林工業區|瓊林工業區]]、[[西盛工業區|西盛工業區]]、[[頭前工業區|頭前工業區]]等工業區，的都市計畫是在新莊郡的外圍編定大量工業用地，設籍於新莊區的工廠家數一度將近6,000家(例如知名的[[歌林|歌林]])，因產業變遷和都市更新而有減少，截至2012年底已减少至3,875家。今日仍有[[三洋電機|三洋電機]]、[[台鳳|台鳳]]、[[中華電線電纜公司|中華電線電纜公司]]等大型工廠營運中，未來亦朝城市擴大的都市更新計畫來進行。

=== 商業 ===
發展始於[[新莊老街|新莊老街]]，日治時代的商業中心則轉移到[[縱貫線_(北段)|鐵路]]拆除後鋪設的中正路上。二次世界大戰後的商業發展大抵以中正路、新泰路為中心呈輻輳狀擴散；另外在幸福路一帶、輔仁大學四周、南新莊之民安路－富國路與四維路－後港一路亦各成完整的商圈。

=== 主要道路 ===
新莊區為狹長型狀之行政區，原以[[新北大道|新北大道]]、[[中正路|中正路]] 橫貫區內全境，另有新闢之[[頭前路|頭前路]]、[[思源路|思源路]]、[[中環路|中環路]]則為聯絡新北市各區之重要道路。

=== 全新莊地區 ===
*[[中正路|中正路]]：台北捷運新莊線沿此路而建，貫穿新莊現今正市中心全境，是新莊區最主要幹道之一，也是過去往來[[臺北市|台北市]]及[[桃園市|桃園市]]的必經道路，為[[台1線#甲線|台1甲線]]。
*[[環河路|環河路]]：沿[[大漢溪|大漢溪]]堤防興建，為連絡[[大漢溪|大漢溪]]北岸市區環河道路的一部份。
*[[新北大道|新北大道]]：目前是[[台1線|台1線]](原中山路[8])，但由於鋪設較[[中正路|中正路]]為晚，故地方稱之為二省道。是新莊區北邊的主要道路，也是[[副都心重劃區|副都心重劃區]]和[[頭前重劃區|頭前重劃區]]北側主要道路之一，用以紓解[[中正路|中正路]]車流。[[新北大道|新北大道]]分別於[[副都心重劃區|副都心重劃區]]旁與[[丹鳳_(新北市)|丹鳳]]地區屬新莊區管轄，中間部分路段屬鄰近的[[泰山區_(台灣)|泰山區]]管轄，[[臺灣桃園國際機場捷運|桃園機場捷運]]通過本路上方。

[[File:Old-Taipei.jpg|700px]]所繪製的新莊郡示意地圖。]]

===上新莊地區===
*[[新泰路|新泰路]]：新莊區通往泰山區主要道路，縱貫新莊區全境，金融機構及工商行號林立，為早期台北市往返泰山區、林口區的必經之路，日據時代稱為「山腳街道」，因泰山舊名山腳而得名。為市道106號。
*[[新莊路|新莊路]]：新莊廟街和夜市美食所在之百年老街。
*[[頭前路|頭前路]]：新闢之東西向路幅寬敞40米林蔭大道，橫向貫穿頭前重劃區，可分散上新莊兩條省道通過性車流而新闢的道路。與三重興德路相接成為示範之綠園道，可穿過堤防至大台北都會公園
*[[思源路|思源路]]：原來為南北向路幅寬闊50米林蔭大道，南接大漢橋通往[[板橋區_(台灣)|板橋區]]，北接五工路通往[[新北產業園區|新北產業園區]]。原為新北市特一號道路，目前為[[市道106號#甲線|市道106甲線]]，[[環狀線_(台北捷運)|環狀線]]通過本路上方，[[台北醫院|台北醫院]]亦在本路上。
*[[中環路|中環路]]：台65快速道路高架橋的地上平面道路。
*[[化成路|化成路]]：因紀念[[清代|清代]][[將軍|將軍]][[陳化成|陳化成]]而得名，原工廠林立而現在已成商圈，新莊區則以此路與三重區分界，為區道北63號。
*[[幸福路|幸福路]]：已形成一座新的商圈，為副都心重劃區附近之最熱鬧市區，為區道北66號。
*[[中央路|中央路]]：位於[[副都心重劃區|副都心重劃區]]內新闢30米道路，為副都心重劃區內主要道路之一。
*[[中原路|中原路]]：位於[[副都心重劃區|副都心重劃區]]旁，最寬處為25米而部分道路仍待拓寬，尤連接之中原東路僅15米。
*[[中華路|中華路]]：旁邊為[[中港大排|中港大排]]，是近年來新莊新闢的中港綠堤，原位於[[復興路|復興路]]與[[中原路|中原路]]之間，後將安寧街納入成為一段，原路段則編為二段，後新闢往[[中山路|中山路]]方向為三段。
*[[新五路|新五路]]：通往[[五股區|五股]]的道路，可由此道路連接[[國道一號|國道一號]][[中山高速公路|中山高速公路]]，為市道107甲線。
*[[復興路|復興路]]：區道北68號，連結新泰路與化成路的重要道路，途中經新莊體育場。
*中平路：被稱為新莊文教區，[[行政院新莊大樓|行政院新莊大樓]]、[[新北市立新莊高級中學|新莊高中]]、[[新莊文藝中心|新莊文藝中心]]及[[家樂福|家樂福]]都在本路上。
*[[中港路|中港路]]：位於中港厝而得名，為新莊最早的南北向道路。

=== 下新莊地區 ===
*三泰路：區道北72號，連結泰山明志路與新樹路間的重要道路，可通往輔大後門並銜接[[泰山區_(台灣)|泰山區]]貴子路，曾為[[市道107號|市道107號]]舊路線。
*青山路：近年為取代壽山路新闢的道路，同時也是[[臺灣桃園國際機場捷運|桃園機場捷運]]自[[台北盆地|台北盆地]]爬升至[[林口台地|林口台地]]的衢道，可通往[[林口中正體育園區|林口中正體育園區]]內[[國立體育大學|國立體育大學]]、[[長庚大學|長庚大學]]、[[長庚科技大學|長庚科技大學]]的近道。
*富國路：同相通的雙鳳路為新莊區通往[[龜山區|龜山區]]的近道。
*新樹路：新莊區通往[[樹林區|樹林區]]重要道路，[[市道107號|市道107號]]，亦為西盛地區主要道路，沿線為工業區重鎮。
*龍安路：下新莊的主要文教區，裕民國小、光華國小、丹鳳高中位於此路沿線。
*民安路：福營地區主要道路，有民安東路與民安西路兩條分支，也是下新莊最熱鬧的商圈所在，鴻金寶戲院大樓即坐落於此，為區道北70、71號。
*後港路：分為後港一路與後港二路，與相連的[[四維路|四維路]]沿線皆是下新莊地區主要的工商活動地區。
*福營路：早期福營地區的主要道路，在[[臺灣銀行|臺灣銀行]]進駐後成為下新莊金融機構集中的區域。
*壽山路：區道北118號，自[[中正路|中正路]]開始，為連接[[桃園市|桃園市]][[龜山區|龜山區]]的重要道路。
*四維路、八德街:位於民安路與龍安路之間，下新莊的舊住宅區。

== 藝文 ==
;藝文活動
新莊市鄉土藝術季從民國89年開辦，在每年9至11月之週末假日，由[[新莊文化藝術中心|新莊文化藝術中心]]<ref name="新莊文化藝術中心">[http://www.xzcac.ntpc.gov.tw/ 新莊文化藝術中心]，新莊文化藝術中心。</ref>籌畫，邀請全國傑出藝文團隊深入新莊各公園、廣場等公共空間演出。讓民眾可以近距離的觀賞藝文活動，培養市民對藝文活動的興趣。從2014年起，擴大舉辦為新北市鄉土藝術季，不僅在新莊區演出，還擴展至泰山區、林口區、五股區、三重區、蘆洲區、平溪區、石門區、貢寮區及坪林區等區域演出。
