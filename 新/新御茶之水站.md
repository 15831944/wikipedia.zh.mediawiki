'''新御茶之水站'''（{{jpn|j=新御茶ノ水駅|hg=しんおちゃのみずえき|rm=Shin-ochanomizu eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[千代田區|千代田區]][[神田駿河台|神田駿河台]]三丁目，屬於[[東京地下鐵|東京地下鐵]][[千代田線|千代田線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''C 12'''<ref>[http://www.tokyometro.jp/station/shin-ochanomizu/index.html 新御茶ノ水駅/C12 | 路線・駅の情報 | 東京メトロ]、2017年4月30日閲覧</ref>。

== 歷史 ==
* 1962年（[[昭和|昭和]]37年）：都都市交通審議會答申第6號提出東京8號線。
* 1964年（昭和39年）12月6日：[[建設省|建設省]]（現[[國土交通省|國土交通省]]）告示3379號將第6號答申的東京8號線定為東京9號線，並決定設置御茶之水站（暫稱）。
* 1969年（昭和44年）12月20日：[[帝都高速度交通營團|帝都高速度交通營團]]（營團地下鐵）千代田線[[北千住站|北千住]]－[[大手町站_(東京都)|大手町]]間（9.9公里）開通，新御茶之水站開業。
* 2004年（[[平成|平成]]16年）4月1日：營團地下鐵民營化，由東京地下鐵繼承本車站。
* 2013年（平成25年）：聖橋方向剪票口直通地面的電梯完成。
* 2014年（平成26年）8月1日：連通聖橋方向剪票口與月台的電梯完成。

== 轉乘站 ==
本站與以下路線實施{{link-ja|連絡運輸|連絡運輸}}，可步行轉乘。
* 東京地下鐵[[東京地下鐵丸之內線|丸之內線]] - [[淡路町站|淡路町站]]
* [[東京都交通局|東京都交通局]]（[[都營地下鐵|都營地下鐵]]）[[都營地下鐵新宿線|新宿線]] - [[小川町站_(東京都)|小川町站]]
* [[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[中央線快速|中央線（快速）]]、[[中央·總武緩行線|中央·總武線（各站停車）]] - [[御茶之水站|御茶之水站]]

要前往丸之內線的淡路町站需從月台大手町側出口前往並穿越小川町站。若要搭乘往東京、新宿、荻窪方向的A線須先在池袋方向的B線剪票口進站後利用月台下連絡通道前往。基本上在南鄰的[[大手町站_(東京都)|大手町站]]轉乘較為方便。

若是從千代田線綾瀨方向轉乘丸之內線池袋方向，由於步行時間較長，與大手町轉乘時間相差無幾<!--（大手町での千代田線 - 丸ノ内線間の乗り換えは半蔵門線ホームの上の通路を歩く）-->。

另外，湯島側出口往丸之內線御茶之水站也可步行轉乘。過去曾有連絡運輸設定，但在小川町站開業後改為與淡路町站進行連絡運輸，取消御茶之水站的連絡運輸設定<!--（同じ丸ノ内線の御茶ノ水駅と淡路町駅が連絡できてしまうため）-->。因此，車站與車內轉乘資訊使用「'''淡路町接續丸之內線轉乘'''」這種與其他車站不同的描述方式。通常，同公司的轉乘路線會優先顯示，但在千代田線車輛的車內自動廣播、[[LED|LED]]式車內資訊顯示器（[[JR東日本209系電聯車#1000番台|JR東日本209系1000番台]]除外）與[[小田急4000型電力動車組_(2代)|小田急4000形]]、[[JR東日本E233系電聯車#2000番台（常磐緩行線）|JR東日本E233系2000番台]]車內自動廣播與[[LCD|LCD]]式車內資訊顯示器將該線轉乘擺在最後。在2015年更新的千代田線車內自動廣播與[[東京地下鐵16000系電力動車組|16000系]]LCD式車內資訊顯示器以丸之內線、都營新宿線、JR線的順序顯示，且丸之內線轉乘導覽省略「淡路町接續」。

JR御茶之水站只在利用本站與淡路町站時才可轉乘能（但由於受到丸之內線御茶之水站影響，無法使用定期券轉乘）。

== 車站結構 ==
[[File:Shin-Ochanomizu-Platform.JPG|220px]]
[[島式月台|島式月台]]1面2線之地下車站。車站月台設有二十四節氣之壁畫。因位於坡道下方，南北深度不同，與綾瀨側（JR御茶之水站方向）的長電扶梯相比，代代木上原側電扶梯與階梯明顯較短。兩側皆設有輪椅專用升降機。

代代木上原側剪票口外大廳有電梯通往地面。

月台側壁曾在1997年至2001年進行改建工程，1號線側與2號線側壁面採用明亮的黃色以及[[和曆#月份|和曆月份名]]與[[節氣|二十四節氣]]的[[馬賽克|馬賽克]][[壁畫|壁畫]]。1號線側牆壁還加上日期。
