'''新豐洲站'''（{{jpn|j=新豊洲駅|hg=しんとよすえき|rm=Shin-toyosu eki|f=y}}）是位於[[日本|日本]][[東京都|東京都]][[江東區_(東京都)|江東區]][[豐洲|豐洲]]六丁目，屬於[[東京臨海新交通臨海線|東京臨海新交通臨海線]]（[[百合鷗_(鐵路公司)|百合鷗]]）的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''U 15'''。

== 車站構造 ==
1面2線[[島式月台|島式月台]]的[[高架車站|高架車站]]。

== 歷史 ==
