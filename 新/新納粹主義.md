'''新納粹'''（Neo-Nazism）是[[二戰|二戰]]後歐洲一些為了使當時的[[納粹|納粹]]和[[法西斯|法西斯]][[復辟|復辟]]所出現的[[政治運動|政治運動]]及[[社會運動|社會運動]]。新納粹運動基本上以[[納粹德國|納粹德國]][[元首|元首]][[阿道夫·希特勒|阿道夫·希特勒]]的「繼承人」自居，延續其[[国家社会主义|国家社会主义]]使命，以及支持[[纳粹主义|纳粹主义]]、[[反猶太主義|反猶太主義]]、極端[[民族主義|民族主義]]及[[種族主義|種族主義]]等。這些組織因經常以[[报复|报复]]製造事端（例如塗污[[猶太人|猶太人]]的墓碑，近年[[反猶|反猶]]已大大減少，转而攻擊[[穆斯林|穆斯林]]移民），而引起[[警察|警察]]甚至[[军队|军队]]的關注。他們大多是[[極右派|極右派]]，包括[[國家主義|國家主義]]者。

== 特徵 ==
他們大多數的共同特徵，是口號是1488，14代表[[十四字真言|十四字真言]]，88代表[[希特勒|希特勒]]萬歲。

== 德國 ==
[[File:Neonazi_8.5.2006_München.jpg|right]]示威]]

[[File:Autonome_Nationalisten-Schwarzer_Block.jpg|thumb]]的反资本主义，支持国家社会主义分子。]]

[[德國|德國]]對納粹的「[[卐|卐]]」字[[標誌|標誌]]严格限制，仅限于作为[[教育|教育]]或[[历史|历史]]范畴的使用。由於現今德國法律已明文禁止有關納粹的[[崇拜|崇拜]]，因此有關納粹的物品多是來自[[美國|美國]]和[[北歐|北歐]]國家，且多為非法製造。現時有關新納粹組織的[[網站|網站]]多使用美國、[[加拿大|加拿大]]的[[主機託管|主機託管]]服務，同時使用其他[[符號|符號]]來表示其理念，如黑[[太陽|太陽]]、黑星等。

二戰結束後，[[盟軍|盟軍]]和當時的東西德政府聯手發起[[去納粹化|去納粹化]]，以阻止新納粹的出現。隨著納粹政權倒台，[[歐洲|歐洲]]在1960年代以前並沒有明顯的新納粹活動。

1990年[[兩德統一|兩德統一]]后，新納粹組織出現了更多的追隨者，以[[東德|東德]][[青年|青年]]為主。東德人經歷[[經濟|經濟]]倒退，成了「失業大軍」；他們敵視外國移民，認為移民搶走了他們的工作機會，亦认为前西德输入外勞作为「同化」他們融合如前西德的[[多元文化主义|多元文化主义]]，反對多元文化，並曾發動多起暴力事件攻擊外國人，特別是来自[[土耳其|土耳其]]的穆斯林移民。

虽然目前在德国可能存在有新纳粹倾向的[[政党|政党]]（例如[[德國國家民主黨|德國國家民主黨]]、[[德國另類選擇|德國另類選擇]]），但受到严重打压，德国议会曾经于2005年试图通过判定新纳粹政党的特殊法例以方便取缔。通常每年在纳粹德国投降日（5月8日）的夜晚，德国首都[[柏林|柏林]]街头也会发生一些小规模的新纳粹主义分子与警察之间的冲突。但总体来说，德国绝大多数民众都极度反对新纳粹。現時[[歐洲|歐洲]]各國，例如[[瑞士|瑞士]]、[[奧地利|奧地利]]、[[希臘|希臘]]、[[克羅地亞|克羅地亞]]、[[烏克蘭|烏克蘭]]、[[俄羅斯|俄羅斯]]等，都有新納粹的活動及構想，例如重建[[第四帝國|第四帝國]]，第四帝國（德語：Das Vierte Reich，英語：Fourth Reich）是指新納粹分子幻想中復興第三帝國（納粹德國）的未來國家。

這個詞最早出現於1960年代，是用以諷刺與納粹有過聯繫的[[西德總理|西德總理]][[庫爾特·喬治·基辛格|庫爾特·喬治·基辛格]]的政府。

對新納粹分子而言，想像中的「第四帝國」將恢復第三帝國的許多政策：[[亞利安人種|亞利安人種]]至上主義、反猶太主義、「生存空間」、軍國主義、極權主義等，最終目標是成立一個統治所有亞利安人的「西方帝國」（Western Imperium）。

== 俄罗斯 ==
俄罗斯的新纳粹主要源自[[苏联解体|苏联解体]]后的1990年代。他们行[[纳粹礼|纳粹礼]]，并高喊口号。这些新纳粹成立的组织有十多个，其中规模较大的有“[[人民民族黨_(俄羅斯)|人民民族党]]”、“[[俄罗斯民族统一党|俄罗斯民族统一党]]”、“[[俄罗斯目标|俄罗斯目标]]”等。到21世纪近几年，他们的组织发展得十分庞大，不仅仅只有年轻人，许多中年、老年人也加入到新纳粹的行列。在首都[[莫斯科|莫斯科]]等大城市，在此经商的[[华人|华人]]经常受到新纳粹的袭击。他们宣扬[[种族主义|种族主义]]，并强烈反对俄罗斯的外族移民以及[[移民|移民]]政策，许多[[少数民族|少数民族]][[代言人|代言人]]被杀。

新纳粹在俄罗斯俗称为“[[光头党|光头党]]”，因为他们的标志是[[光头|光头]]、纳粹[[紋身|紋身]]、[[纳粹礼|纳粹礼]]和[[迷彩裤|迷彩裤]]等。“光头党”主要活动在莫斯科、[[圣彼得堡|圣彼得堡]]、[[沃罗涅日|沃罗涅日]]和[[海参威|海参威]]等地，统一的偶像是希特勒，主要的集会日是每年4月的最后一周（希特勒生日）。{{Fact|“光头党”在全俄罗斯的总人数大约在10万名以上，其中80%集中在莫斯科和圣彼得堡。|time=2018-06-08T10:50:49+00:00}}这些组织总体上来说比较分散，部分组织的总部设在[[沃罗涅日|沃罗涅日]]，但都有两个共同特点：
#仇视一切非俄罗斯人（尤其是[[亚洲人|亚洲人]]、[[黑人|黑人]]及國內信奉[[伊斯蘭教|伊斯蘭教]]的少數民族）。
#崇尚暴力，甚至[[种族屠杀|种族屠杀]]和[[种族清洗|种族清洗]]。

“光头党”认为外族人“侵占”了俄罗斯的城市，强占[[俄罗斯人|俄罗斯人]]的[[生存空间|生存空间]]。“光头党”提倡以暴力对待外族人。

== 加拿大 ==

[[File:Aryan_Guard_20.jpg|thumb]]街頭上的新納粹支持者]]

加拿大的新納粹主義萌芽於1965年{{le|加拿大納粹黨|Canadian Nazi Party}}的創立，20世紀70年代至80年代，新納粹主義繼續在該國傳播，唐·安德魯斯於1977年成立了[[加拿大國民黨|加拿大國民黨]]。黨的目標是“促進和維護加拿大的歐洲文化”，且以[[反猶太主義|反猶太主義]]和[[種族主義|種族主義]]而聞名，{{le|沃夫甘|Wolfgang Droege}}等知名新納粹領導人皆來自該黨，可是許多人已經加入'''遺產陣線(Heritage Front)'''，遺產陣線是道洛奇在[[多倫多|多倫多]]建立，當時[[白人至上主义|白人至上主義]]運動的領導人“對激進的權利狀態不滿意”，並希望將無組織的白人至上主義者團結起來，成為具有共同目標的有影響力和有效率的群體。<ref>[https://en.wikipedia.org/wiki/Neo-Nazism#Canada Neo-Nazism - Wikipedia]</ref>。詳見{{le|加拿大新納粹主義|Neo-Nazism in Canada}}。

== 美国 ==
二战結束后，[[纳粹主义|纳粹主义]]及三K党运动受到重创。美国不承认纳粹主义者在本国的政治权利，不准其举行[[游行|游行]]或集会，对其运动都采取[[镇压|镇压]]政策。

1970年代，美国对纳粹主义者的政治权利限制有所松动，部分议员更提出“纳粹主义者也应该享有基本政治权利，不该限制其[[人权|人权]]”的观点。

1990年代，[[苏联解体|苏联解体]]后，新纳粹势力和[[共产主义|共产主义]]流派“当代[[马克思主义|马克思主义]]者”都被允许在美国国内举行游行集会，他们的政治权利得到保障。

自[[911事件|911事件]]以來新納粹運動逐漸復甦，而2015年起[[川普|川普]]在總統競選時期的偏頗言論也讓這些份子支持川普，2016年11月19日，[[另类右派|另类右派]]在华盛顿特区的一次集会中行[[纳粹礼|纳粹礼]]祝贺川普当选，{{le|理查德·B·斯宾塞|Richard B. Spencer|斯宾塞}}在会上说出：“川普万岁，我们的民族万岁，胜利万岁”。但川普随后对其提出谴责，表示和这一派别没有关联，不愿鼓励其行为，其团队也声明“谴责种族主义”。<ref>{{cite web|url=http://www.bbc.com/news/world-us-canada-38069469|title=Trump disavows 'alt-right' supporters|work=BBC|date=2016-11-23|accessdate=2016-11-23}}</ref><ref>{{cite web|url=http://edition.cnn.com/2016/11/21/politics/alt-right-gathering-donald-trump/|title=Alt-right leader: 'Hail Trump! == 以色列 ==
2007年9月，[[以色列|以色列]]警方宣布破獲該國第一個新納粹組織，逮捕八名青年，查扣希特勒[[肖像|肖像]]、納粹[[制服|制服]]、[[刀|刀]]、[[槍械|槍械]]、[[炸藥|炸藥]]等，並指控這些人宣揚希特勒思想及任意攻擊猶太人。這八名青年的[[年齡|年齡]]在16歲到21歲之間，都是來自前蘇聯的[[新移民|新移民]]，他們自己並不認同是猶太人，但是因為他們的祖父母為[[猶太人|猶太人]]才獲准移居以色列。這八名青年涉嫌在2006年惡意破壞[[猶太教|猶太教]]會所，在猶太教會所用噴漆塗上納粹「卐」字標誌，並且多次毆打[[猶太教徒|猶太教徒]]、[[亞洲|亞洲]]勞工、[[同性戀|同性戀]]者與吸食[[毒品|毒品]]者。9月10日，這八名青年被起訴。<ref>蔡筱雯 綜合外電報導，〈以色列逮8新納粹份子〉，[[蘋果日報_(台灣)|蘋果日報 (台灣)]]，2007年9月11日</ref>

== 對猶太人大屠殺的否認 ==
新納粹主義者否認對六百萬的[[猶太人大屠殺|猶太人大屠殺]]行為，並且對外宣稱這是個歷史謊言。他們宣稱納粹德國完全沒有任何屠殺行為或種族清洗的[[政策|政策]]，而有知名歷史學家支持他們的說法。這種[[歷史修正主義|歷史修正主義]]，除了被新納粹主義者運用外，美國[[白人|白人]][[沙文主義|沙文主義]]者也利用它來美化美國南方各州在[[南北戰爭|南北戰爭]]之前實行的[[奴隸制|奴隸制]]。
