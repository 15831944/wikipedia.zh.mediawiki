'''新百广场站'''位于[[石家庄市|石家庄市]][[桥西区_(石家庄市)|桥西区]]，是[[石家庄地铁1号线|石家庄地铁1号线]]与[[石家庄地铁3号线|石家庄地铁3号线]]的一座[[换乘站|换乘站]]，亦是[[石家庄地铁|石家庄地铁]]第一座投入使用的换乘站。2017年6月26日随1号线与3号线首通段开通一同投入使用<ref>[http://www.sohu.com/a/152183848_180330 石家庄正式进入“地铁时代”，1、3号线首开工程通车]</ref>。该站是现时石家庄地铁系统中建筑面积最大的车站。<ref>[http://www.sohu.com/a/138039700_398131 先睹为快，石家庄地铁新百广场站是这样的]</ref>本站建设时曾以中山广场站命名，开通前正式改为现名称。<ref>[http://www.sohu.com/a/138118906_398137 石家庄地铁今天迎来大批“乘客”，四个站改名]</ref>

== 位置 ==
新百广场站位于[[石家庄市|石家庄市]][[桥西区_(石家庄市)|桥西区]]，中华大街与中山西路交叉口。1号线站台呈东西向布置，3号线站台呈南北向布置。<ref>[http://www.sjz.gov.cn/col/1274081553614/2011/10/09/1318128026586.html 石家庄市规划城市轨道交通1、3号线一期工程车站站名方案公示 中国石家庄政府网]</ref>

== 结构 ==
新百广场站为地下三层车站，1号线位于3号线上方，两线十字交叉。两站台均为[[岛式站台|岛式站台]]设计。

== 换乘方式 ==
[[石家庄地铁1号线|1号线]]可通过站台中部楼梯向下[[换乘站#垂直换乘|垂直换乘]][[石家庄地铁3号线|3号线]]，而[[石家庄地铁3号线|3号线]]则需上至地下一层站厅与[[石家庄地铁1号线|1号线]]进行进行[[换乘站#大堂换乘|大堂换乘]]。
