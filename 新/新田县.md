'''新田县'''位于中国[[湖南省|湖南省]]南部，是[[永州市|永州市]]下辖的一个县。

== 历史沿革 ==
[[明|明]][[万历|万历]]二年(1574年)，设'''新田营'''于今[[龙泉镇|龙泉镇]](城关)。[[崇祯|崇祯]]十二年(1639年)，由[[宁远|宁远]]、[[桂阳|桂阳]]析置新田县。<ref name="新田概况">[http://www.xt.gov.cn/Html/zjxt/xtgk_488_266.html 新田概况]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 行政区划 ==
全县辖7镇12乡。

== 地理 ==
县境南北长，东西窄，地势北高南低，地形为向南开口的狭长盆地。境内最高点海拔1080米，最低点海拔200米。<ref name="新田概况"/>

主要河流：[[新田河|新田河]]，建有[[金陵水库|金陵水库]]。

== 历代名人 ==
[[張榮組|張榮組]]（1806～1872）字錫圭，號管卿，號潤農。籍大坪塘鄉長富村人。晚清湘軍將領，詩人。

鄭作民（1902—1940），高山鄉高山村人。1937年參加抗日戰爭，先後參加淞滬會戰、徐州突圍戰和田家鎮守衛戰，屢立戰功。1940年1月，於桂南會戰中奉命由貴州馳援南寧。 1940年2月3日，激戰中犧牲。時年38歲。

[[蔣先雲|蔣先雲]]（1902—1927），男，1902年7月14日生於湖南省新田縣大坪塘村。中華民國軍事將領，黃埔軍校一期畢業。
