{{西域歷史}}
[[新疆|新疆]]自古就有人类活动的迹象，为中亚诸多民族活动的场所。新疆最早的居民是属塞种，後受汉人、突厥人及回鹘人壓力，只分布於中亚，其代表性民族是[[斯基泰人|斯基泰人]]。他們發展出發達的游牧文化。公元前1世纪新疆大部被[[汉朝|汉朝]]纳入中国的版图。6世纪后，新疆人种和语言逐渐回鹘化。11世纪后，因為伊斯蘭教的傳播开始出现[[穆斯林|穆斯林]]民族；而11世纪后新疆开始[[伊斯兰化|伊斯兰化]]后，经历过一系列大大小小国家政权统治，18世纪中叶以后分别归属于[[清朝|清朝]]和[[中华民国|中华民国]]，现在则属于[[中华人民共和国|中华人民共和国]]的[[新疆维吾尔自治区|新疆维吾尔自治区]]。

== 史前時代 ==
考古发现新疆的上古文明可能由[[塞种|塞种]]人创造，也是媒体中经常提及的西域印欧人。如《[[史记|史记]] 大宛列传》和《[[汉书|汉书]] 西域传》中记载的[[楼兰|楼兰]]古国，早在公元前2世纪以前，楼兰就是[[西域|西域]]一个著名的“城廓之国”。它东通[[敦煌|敦煌]]，西北到[[焉耆|焉耆]]、[[尉犁|尉犁]]，西南到[[若羌|若羌]]、[[且末|且末]]。古代“[[丝绸之路|丝绸之路]]”的南、北两道从楼兰分道。《漢書·張騫傳》：“月氏已為匈奴所破，西擊塞王。塞王南走遠徙，月氏居其地。”斯基泰即古伊朗碑銘及希臘古文獻中所載Sacae（Sakas）。公元1世紀， 月氏迫使[[斯基泰|斯基泰]]向西南遷徙，跨過[[錫爾河|錫爾河]]，到達[[河中地區|河中地區]]的[[粟特|粟特]]地方。

== 早期西域綠洲國家 ==

===[[龜茲|龜茲]]===

龟兹，前272年印度[[阿育王|阿育王]]势力扩展到龟兹（今新疆[[阿克苏地区|阿克苏地区]]和[[巴音郭楞蒙古自治州|巴音郭楞蒙古自治州]]部分地区），阿育王赐龟兹为太子法益封地。11世紀末，[[回鶻|回鶻]]的[[黑汗|黑汗]]王朝改宗[[伊斯蘭教|伊斯蘭教]]，對西域諸佛國發起了曠日持久的「[[聖戰主義|聖戰]]」。14世紀，改宗伊斯蘭的[[察合台|察合台]]汗[[禿忽魯帖木兒|禿忽魯帖木兒]]對龜茲的[[佛教|佛教]]教徒進行了殘酷的迫害，對佛教文化進行了毀滅性的破壞。佛教寺院廟宇被拆毀，佛像被搗毀，佛教經典文獻被焚燒，佛教教徒被屠殺，具有千餘年歷史的龜茲佛教文化被破壞殆盡。當地佛教僧侶或被迫接受伊斯蘭教，或逃往異國他鄉，或抗拒被殺。

===[[樓蘭|樓蘭]]（[[鄯善|鄯善]]）===

早在前2世紀以前，樓蘭就是西域一個著名的「城廓之國」。它東通[[敦煌|敦煌]]，西北到[[焉耆|焉耆]]、[[尉犁|尉犁]]，西南到[[若羌|若羌]]、[[且末|且末]]。古代「絲綢之路」的南、北兩道從樓蘭分道。前77年，漢朝使者傅介子刺殺樓蘭王常歸，改立其親漢弟弟[[尉屠耆|尉屠耆]]為王，改國號鄯善。

===[[于阗|于阗]]===

于闐地處塔里木盆地南沿，東通[[且末|且末]]、鄯善，西通[[莎車|莎車]]、[[疏勒|疏勒]]，盛時領地包括今[[和田地区|和田]]、[[皮山|皮山]]、[[墨玉|墨玉]]、[[洛浦|洛浦]]、[[策勒|策勒]]、[[于田|于田]]、[[民豐|民豐]]等縣市，都西城（今[[和田約特干遺址|和田約特干遺址]]）。

===[[車師|車師]]（[[高昌|高昌]]）===

车师，古代[[中亚|中亚]]东部[[西域|西域]][[城郭诸国|城郭诸国]]之一。国都[[交河|交河]]（今[[中国|中国]][[新疆|新疆]][[吐鲁番|吐鲁番]]西北）。东南通往[[敦煌|敦煌]]，向南通往[[楼兰|楼兰]]、[[鄯善|鄯善]]，向西通往[[焉耆|焉耆]]，西北通往[[乌孙|乌孙]]，东北通往[[匈奴|匈奴]]，是[[丝绸之路|丝绸之路]]上的重要商站。从考古发掘上来看，国人操突厥语系Oguz（乌古斯）语支。古代车师人的种族还未完全研究清楚，头骨上显示[[高加索人种|高加索人种]]与[[蒙古人种|蒙古人种]]的特征。有人说他们也是[[铁勒|铁勒]]。

高昌，位于今日的[[新疆|新疆]][[吐鲁番|吐鲁番]]地区，是古时[[西域|西域]]交通枢纽。公元5世纪中叶至7世纪中叶，在这个狭窄的[[吐鲁番盆地|吐鲁番盆地]]中，曾先后出现四个独立王国，分别是[[阚氏高昌|阚氏高昌]]、[[张孟明|张氏高昌]]、[[馬儒|马氏高昌]]及-{[[麴氏高昌|麴氏高昌]]}-。[[唐|唐]][[貞觀_(唐朝)|贞观]]二年（628年），著名高僧[[玄奘|玄奘]]曾途经高昌。[[唐|唐]][[貞觀_(唐朝)|贞观]]十三年（640年），设高昌县。到840年漠北[[回鶻|回鶻]]國崩，一部份回鶻人佔據高昌建國，即[[高昌回鶻|高昌回鶻]]。

===[[焉耆|焉耆]]===

焉耆又称乌夷、阿耆尼，新疆[[塔里木盆地|塔里木盆地]]東北部古国，在今[[新疆维吾尔自治区|新疆维吾尔自治区]][[焉耆回族自治县|焉耆回族自治县]]附近。

===[[疏勒|疏勒]]===

疏勒，古代西域绿洲国家-疏勒王国之地。古代居民属[[原始印欧人|印欧种]]，似操[[印欧语|印欧语]]系语言，自9、10世纪，人种和语言逐渐[[回鹘|回鹘]]化。

===其他國家===
'''[[康居|康居]]'''，古代生活在中亚地区（包括新疆西部）的游牧民族，活动范围主要在今哈萨克斯坦南部及锡尔河中下游。公元前2世纪，控弦者八九万人；前1世纪末年，人口达六十万，拥有军队十二万，以卑阗城为中心（今塔什干或奇姆肯特）。和所有游牧民族一样，康居人随季节的变化而迁徙。冬季他们南下于锡尔河一带，夏季北上至“蕃内”，两地相距数千里之遥。他们是伊朗人种。康居本身是突厥与斯基泰人的一联盟，现在愈来愈多人认为他们与蒙古时期的康里人有关，他们也成为哈萨克汗国大玉兹的主体。他们与[[乌孙|乌孙]]构成联盟，是[[哈萨克人|哈萨克人]]的重要族源。

'''[[月氏|月氏]]'''，公元前3世紀至公元1世紀住在北亞，並經常與匈奴發生衝突，其後西遷至中亞。

== 漢與匈奴爭奪 ==
* 前209年，[[匈奴|匈奴]][[冒顿单于|冒顿单于]]即位，尔后统一[[戈壁沙漠|漠北]]，歼灭[[河西走廊|河西]]的[[月氏|月氏]]，宾服[[西域|西域]]三十六国，进军中原。
* 公元前138年，[[汉武帝|汉武帝]]决心联合[[西域|西域]]各国，夹击[[匈奴|匈奴]]，遂使[[张骞|张骞]]出使[[西域|西域]]，前后两次，皆为匈奴所虏，竟得脱。前115年，[[张骞|张骞]]第二次出使[[西域|西域]]归来。此后[[汉朝|汉朝]]与[[西域|西域]]之间连接起了[[丝绸之路|丝绸之路]]。[[汉宣帝|汉宣帝]]神爵二年（前60年），汉廷在[[龟兹|龟兹]]建立[[西域都護府|西域都護府]]，使得今日新疆地区各國首次在名義上处于[[中原王朝|中原王朝]]的附属國的地位，但維持不久。
* [[烏孫|烏孫]]，西漢時由游牧民族烏孫在西域建立了行國，位於[[巴爾喀什湖|巴爾喀什湖]]東南、[[伊犁河|伊犁河]]流域，立國君主是[[獵驕靡|獵驕靡]]。前蘇聯學者認為烏孫文化是[[塞迦|塞人]]（Saka，即薩迦或塞克，斯基泰人）文化的繼承和發展，並稱塞－烏孫文化，烏孫文化時期是前300年－300年。[[中國|中國]][[新疆|新疆]]社會科學院歷史研究所的學者推斷在先秦时期烏孫自號「昆」，是一個古老的[[西戎|西戎]]部落，操[[突厥語|突厥語]]，[[春秋戰國|春秋戰國]]時代與[[月氏|月氏]]遊牧於河西走廊。首領稱為“昆莫”或“昆彌”。
* 東漢曾設置西域都護府，監控西域諸國，直到公元75年，之後曾在91年到107年間短暫復置。

== 諸國爭鬥時代 ==
* 4世纪初，西晋因来自北方的游牧民族入侵而覆灭，历史进入[[五胡十六国|五胡十六国]]时期。此间在中国西北部建立了一个又一个的不同民族的国家，包括[[前凉|前凉]]、[[前秦|前秦]]、[[北凉|北凉]]、[[后凉|后凉]]和[[西凉|西凉]]。这些政权都试图保持对新疆诸藩的控制。最终[[鲜卑|鲜卑]]的[[北魏|北魏]]重新统一中国北方，控制了今天新疆东南的一部分。[[疏勒|疏勒]]、[[于阗|于阗]]、[[龟兹|龟兹]]和[[且末|且末]]等政权控制了西部，而中央[[吐鲁番|吐鲁番]]附近则被北凉的延续者[[高昌|高昌]]所统治。

== [[突厥|突厥]]崛起 ==

[[image:GökturksAD551-572.png|thumb]]
* 5世纪末，[[吐谷浑|吐谷浑]]和[[柔然|柔然]]开始分别侵入新疆南部和北部，中原政权（[[南北朝|北朝]]）逐渐丧失对此地的控制。从6世纪开始，[[突厥|突厥]]在[[阿尔泰山|阿尔泰山]]地区崛起，數年间击败柔然，建立了西至[[里海|里海]]、东抵[[大兴安岭|大兴安岭]]、横亘[[中亚|中亚]]的[[突厥汗国|突厥汗国]]。583年突厥分裂为东西二部，新疆属[[西突厥|西突厥]]。
* 隋炀帝大业四年（608年），由[[杨广|杨广]]亲自率领的隋军在打败了连年入侵的[[吐谷浑|吐谷浑]]之后，控制了今日的新疆东南部。开始之后[[中原王朝|中原王朝]]经营西域。

== 唐帝國統治時代 ==
{{main|安西都護府|北庭都護府}}
* 唐太宗贞观八年（634年），龟兹、吐蕃、高昌、女国、石国遣使朝贡。贞观二十二年（648年）唐太宗派遣昆丘道副大总管郭孝恪讨伐龟兹，破都城，郭孝恪自留守，龟兹国相那利率众遁逃。那利等率众万余，与城内降胡表里为应攻郭孝恪。郭孝恪中流矢死，将军曹继叔收复都城。贞观二十二年648年设安西都护府，抚宁西域，统龟兹、焉耆、于阗、疏勒四国。安西都护府治所，在龟兹国城内，管戍兵二万四千人。
* 唐太宗贞观十四年（640年），唐军佔領[[高昌|高昌]]，于该地置西州，又于可汗浮图城（今[[吉木萨尔|吉木萨尔]]）设[[庭州|庭州]]；同年在高昌设[[安西都护府|安西都护府]]，后迁至[[库车|库车]]，改置为[[安西大都护府|安西大都护府]]。在之后的二十年间，唐军发动了对[[西突厥|西突厥]]的一系列远征，在657年西突厥彻底投降，670年安西四镇被[[吐蕃|吐蕃]]攻佔，693年[[武周|武周]]再次佔領。702年在庭州设置[[北庭都护府|北庭都护府]]，后又升为北庭大都护府，管理天山北麓及新疆东部地区的军政事务，而安西大都护府管理[[天山|天山]]南部和[[葱岭|葱岭]]以西的广大地区。[[唐玄宗|唐玄宗]][[开元|开元]]年间，曾在两大都护府之上设[[碛西节度使|碛西节度使]]，是当时全国八大[[节度使|节度使]]之一。非汉民聚居区，则设置羁縻府州。同时，还在[[龟兹|龟兹]]、[[于阗|-{于}-阗]]、[[疏勒|疏勒]]、[[托克馬克|碎叶]]（一度是[[焉耆|焉耆]]）设军事建制，史称[[安西四镇|安西四镇]]。
* 唐高宗麟德四年（667年）吐蕃陷白州等一十八州，又陷龟兹拨换城。唐朝罢安西四镇。上元中（675年）龟兹王白素稽献银颇罗、名马。

== [[吐蕃|吐蕃]]入侵和[[回紇|回紇]]汗國 ==

* 8世纪中叶[[安史之乱|安史之乱]]发生时，[[吐蕃|吐蕃]]再次在之后的三十年内逐步控制了天山山脉南部的和河西走廊的广大地区，甚至在763年攻占并洗劫了唐朝京城长安。同时，[[回紇|回紇人]]改族称为[[回鹘|回鹘]]，之后回鹘人控制漠北漠南甚至到中亚的广大地区，包括新疆北部。9世纪中叶后，吐蕃和回鹘都衰落下来，这一地区进入混战时期。

== 回鹘时期 ==
{{main|喀喇汗王朝|高昌回鹘}}
[[File:Asia_1025ad.jpg|thumb]]
* 9世纪之后的[[中原王朝|中原王朝]]无暇顾及西域，西域出现了几个国家并列存在的局面。其中主要有[[高昌|高昌]]、[[黑汗|黑汗]]王朝和-{于}-阗等地方政权。于阗是古老的[[塞迦|塞人]]居地。唐亡后，于阗尉迟王族执政，与中原地区往来密切，因曾受过唐朝册封而自称李姓。
* 宋朝咸平四年（1001年）、大中祥符三年（1010年）、六年、天禧元年（1017年）、四年、天圣二年（1024年）、七年、九年、景祐四年（1037年）、熙宁四年（1071年）、五年、绍圣三年（1096年），[[龟兹|龟兹]]前后遣使朝贡十二次。
* [[黑汗|黑汗]]王朝在10世纪和11世纪控制今日的新疆西部，在中文的记载中，黑汗王朝是起源于中亚，有一个首都在[[喀什地区|喀什]]；起源于中亚的黑汗王朝虽然是操[[突厥语|突厥语]]的民族建立的第一个穆斯林王朝，但力图保存东方王朝的特色，特别是强调与中原的传统联系。在诸大汗称号中，在诸汗铸造的钱币上，经常有“桃花石·卜格拉汗”、“秦之王”、“秦与东方之王”等称号。桃花石和秦都是中亚地区对中国的称呼。[[喀什噶尔|喀什噶尔]]人[[马合木|马合木]]的《突厥语辞典》以及中世纪阿拉伯、波斯文献有多处记载，明确地把黑汗王朝东部疏勒所在的喀什噶尔地区与[[宋朝|宋]]（摩秦）、[[契丹|契丹]]并列，认为中国是由此三部组成。黑汗王朝也有与[[高昌回鹘|高昌回鹘]]（维吾尔人的重要源头）作战，麻赫穆德·喀什噶里的突厥语大词典，有诗把[[回鹘人|回鹘人]]说成是最凶恶的异教徒，也因曾支持[[于阗|于阗]]，两国开战。黑汗王朝后期，[[阿拉伯字母|阿拉伯字母]]代替了新疆原住民的[[回鹘文字母|回鹘文字母]]。回鹘的黑汗王朝改宗伊斯兰教，对西域诸佛国发起了旷日持久的“圣战”。
* [[回鹘|回鹘]]人（今日[[维吾尔人|维吾尔人]]之祖先）住在今日新疆中部[[高昌|高昌]]一带（应是汉籍中的[[西州_(唐朝)|西州]]，即[[高昌回鹘|高昌回鹘]]），信仰[[佛教|佛教]]，并未指明与疏勒的黑汗王朝的关系。

== [[西遼|西遼]] ==

* 1123年，[[遼朝|遼朝]]皇族[[耶律大石|耶律大石]]躲避[[金朝|金朝]]军队追擊，从[[华北|华北]]西迁至新疆，建立[[西辽|西辽]]政权，以中国自居，统治今日新疆地区八十多年。后扩张到[[中亚|中亚]]，首都[[虎思斡鲁朵|虎思斡鲁朵]]（[[吉尔吉斯|吉尔吉斯]][[托克玛克|托克玛克]]附近的[[布拉纳城|布拉纳城]]），一时成为中亚强国。西辽上层似乎使用[[汉语|汉语]]。宗教信仰比较自由，各种宗教都有流行，如[[佛教|佛教]]、[[萨满教|萨满教]]、[[伊斯兰教|伊斯兰教]]、[[摩尼教|摩尼教]]和[[景教|景教]]等。景教在锡尔河以北流行，犹太教徒主要活动于撒马尔罕。属国的宗教文化，特别是[[伊斯兰|伊斯兰]]文化，在这一时期得到了比较好的尊重。但已不是唯我独尊，因此出现了[[阿赫马德·阿萨维|阿赫马德·阿萨维]]的改革运动，争取[[突厥人|突厥人]]入教。1992年在吉尔吉斯坦出土的西辽钱币也使用阿拉伯文。

== 窝阔台和察合台系汗國 ==
{{main|窝阔台汗国|察合台汗國|葉爾羌汗國}}
* 13世纪上半叶至14世纪初时，新疆的天山以北地区大部分属于蒙古[[窝阔台汗国|窝阔台汗国]]。今伊犁河流域曾设置阿里麻里（阿力麻里）行省，但不久就并入[[察合台汗国|察合台汗国]]。阿姆河南岸曾设立行省，后并入[[伊儿汗国|伊儿汗国]]。今乌鲁木齐一带曾设置别失八里行省，后一度被察合台汗国占据，后期重新成为蒙古人建立的中原王朝[[元朝|元朝]]的辖地。14世纪，改宗伊斯兰的東察合台汗[[秃忽鲁帖木儿|秃忽鲁帖木儿]]对龟兹的佛教教徒进行了残酷的迫害，对佛教文化进行了毁灭性的破坏。佛教寺院庙宇被拆毁，佛像被捣毁，佛教经典文献被焚烧，佛教教徒被屠杀，具有千余年历史的龟兹佛教文化被破坏殆尽。当地佛教僧侣或被迫接受伊斯兰教，或逃往异国他乡，或抗拒被杀。
* 新疆中世纪的歷史被一些历史学家认为是“突厥人和蒙古人的侵略史”<ref>[http://www.xjly.net/chuanshuo/info.asp?id=126 突厥人和蒙古人的侵略史]</ref>。
* [[葉爾羌汗國|葉爾羌汗國]]是由[[东察合台汗国|东察合台汗国]]汗[[速檀阿黑麻|速檀阿黑麻]]之子[[蘇丹|蘇丹]][[賽德|賽德]]在1514年於原[[察合台汗國|察合台汗國]]的舊地上創立的一個國家，可认为是东察合台汗国的延续。至1680年成為[[準噶爾汗国|準噶爾汗国]]的附屬國，在17世紀末策妄阿拉布坦時期消亡。

== 準噶爾汗國 ==
[[File:Receiving_the_surrender_of_the_Yili.jpg|200px]]
* [[明朝|明朝]]初年为防范漠北的[[鞑靼_(蒙古)|鞑靼]]和[[瓦剌|瓦剌]]势力於[[哈密卫|哈密]]等地區設置[[衛所|衛所]]；16世紀中葉，[[吐鲁番汗国|吐魯番]]部強大，1472年，哈密等衛所一度被吐魯番攻破，諸衛內遷，後復，1514年再度被併。15世紀後半期後，西北諸衛全部喪失，明朝最後放棄新疆，退守[[嘉峪關|嘉峪關]]。
* 17世纪，[[准噶尔部|准噶尔]]（蒙古的一支）以[[伊犁|伊犁]]为基地建立庞大的游牧帝国，即[[准噶尔汗国|准噶尔汗国]]。
* 1697年，经过激烈的战争后，清军联合[[策妄阿拉布坦|策妄阿拉布坦]]击败了准噶尔汗国大汗[[噶尔丹|噶尔丹]]，立其为新准噶尔大汗，控制了新疆东部。隨後將天山南路（[[回部|回部]]）置於直接統治之下。1727年策妄阿拉布坦之子[[噶尔丹策零|噶尔丹策零]]即位，1745年因瘟疫爆发染病而终。其次子[[策妄多尔济那木扎尔|策妄多尔济那木扎尔]]遂即位，但因不善执政使得准噶尔国内动乱。噶尔丹策零长子[[喇嘛达尔扎|喇嘛达尔扎]]1750年杀弟夺权，众人亦不服。[[达瓦齐|达瓦齐]]遂于1753年夺权并任至1755年，期间与其手下策妄阿拉布坦外孙[[阿睦尔撒纳|阿睦尔撒纳]]反目。1755年，[[阿睦尔撒纳|阿睦尔撒纳]]降于清廷，攻占[[伊宁市|固勒扎]]，大破达瓦齐。阿睦尔撒纳自命准噶尔大汗，并求助于俄国，圖謀恢復獨立。

* [[阿帕克和卓|阿帕克和卓]]掌握[[葉爾羌汗國|葉爾羌汗國]]的實際權力，此家族影響力一直延續到1759年。

== 大清帝國統治時代 ==
{{main|清朝治疆历史|新疆省 (清朝)}}
参见：[[兆惠|兆惠]]、[[阿桂|阿桂]]、[[锡伯族|锡伯族]]、[[左宗棠|左宗棠]]、[[刘锦棠|刘锦棠]]
* 1755年，[[清朝|清朝]]軍队[[準噶爾之役|滅準噶爾]]，三年後又消滅[[阿睦爾撒納|阿睦爾撒納]]勢力，西域底定。[[乾隆帝|乾隆帝]]把這片土地命名為「新疆」，取名為新闢疆土，出自1760年乾隆帝给[[陕甘总督|陕甘总督]][[杨应琚|杨应琚]]的谕令:“新辟疆土如伊犁一带，距内地远，一切事宜难以遥制<ref>[http://www.lsqn.cn/ChinaHistory/qing/200703/110049_9.html 从“斌静案”看清代驻疆官员与新疆的稳定] {{webarchive|url=https://web.archive.org/web/20160420074720/http://www.lsqn.cn/ChinaHistory/qing/200703/110049_9.html |date=2016-04-20 }}</ref>。”
[[File:The_Lifting_of_the_Siege_of_the_Black_River_(Khara-Usu).jpg|200px]]
* 1759年，清朝平定[[回部|回部]]的[[大小和卓之亂|大小和卓之亂]]，采用[[伯克制|伯克制]]和军府制，開始統治南疆。其后大和卓之孙[[和卓玉素普|和卓玉素普]]、[[张格尔|张格尔]]等数次回国叛乱。回部之前在准噶尔暴力统治之下。而准噶尔部的灭亡也导致新疆及中亚部分（吉尔吉斯斯坦全部，塔吉克斯坦、乌兹别克斯坦部分，哈萨克斯坦东部）完全伊斯兰化。之前此区域统治民族准噶尔部蒙古全民信奉藏传佛教。1762年，清朝在[[伊犁|伊犁]]设立[[伊犁将军|伊犁将军]]，统一行使对[[天山|天山]]南北各地的军政管辖。
* 1826年大和卓之孙[[张格尔|张格尔]]依附英国叛乱，后被擒至北京处死。
* 1862年至1873年，'''[[同治陕甘回变|同治回乱]]'''（'''陕甘回变'''），在中国[[陕西|陕西]]、[[甘肃|甘肃]]爆发的一次境内的[[回族|回族]]对汉族之间的仇杀。它维持了十年多，波及陕西、甘肃、[[宁夏|宁夏]]、[[青海|青海]]和新疆等地区，最后被以汉人[[湘军|湘军]]为主的清朝军队镇压。而同治回乱的很多穆斯林首领也退到新疆，或借道新疆逃往[[俄罗斯帝国|俄罗斯帝国]]。
* 与此同时，19世纪中叶，俄罗斯帝国威胁清朝的整个北部边境。1864年的[[塔城条约|塔城条约]]将新疆西北部[[巴尔喀什湖|巴尔喀什湖]]以南大片土地割让给俄国，这些土地现在分别属于[[哈萨克斯坦|哈萨克斯坦]]、[[吉尔吉斯斯坦|吉尔吉斯斯坦]]和[[塔吉克斯坦|塔吉克斯坦]]。
* 1864年，新疆庫車发生叛乱，隨後蔓延到[[喀什噶爾|喀什噶爾]]、[[伊犁|伊犁]]、[[塔城地区|塔城]]。1865年春，邻国[[浩罕汗国|浩罕汗国]]的塔吉克人[[阿古柏|阿古柏]]从喀什进入新疆，扶持布素鲁克建立[[哲德沙尔汗国|哲德沙尔汗国]]。1867年，[[阿古柏|阿古柏]]将“[[哲德沙尔汗国|哲德沙尔汗国]]”改稱“洪福汗国”，自命为汗，攻占库车、库尔勒，佔領天山以南的南疆，介入英俄的[[大博弈|大博弈]]，得到英俄两国支持。随着阿古柏的扩张，大批[[乌孜别克族|乌孜别克族]]人进入[[新疆|新疆]]，成為今天新疆一個重要的少數民族之一。而南疆的傳統民族[[維吾爾族|維吾爾族]]也進入北疆，成為全新疆的主要民族之一，在此前蒙古族在北疆自元朝來一直佔據主要民族的地位。
* 1871年，俄国侵占包括[[伊犁|伊犁]]城（[[伊宁市|伊宁市]]）在内的[[伊犁河|伊犁河]]谷，当时清朝在新疆只剩下[[塔城地区|塔城]]、[[哈密廳|哈密]]等少数据点。
* 1875年，清朝[[陕甘总督|陕甘总督]][[左宗棠|左宗棠]]就任钦差大臣，督办新疆事务。到1877年底，清军陆续收复了中亚[[浩罕汗国|浩罕汗国]][[阿古柏|阿古柏]]侵占了多年的天山南北，史稱「[[清军收复新疆之战|清军收复新疆之战]]」。1881年，清政府收复被俄国占领长达11年之久的[[伊犁|伊犁]]地区。1884年（光绪十年）11月19日，清政府颁发上谕，任命刘锦棠为新疆巡抚，魏光焘为新疆布政使，标志着新疆省的正式建立。'''新疆省'''，实行与中国本部18省一样的行政制度，由巡抚统管全疆各项军政事务，新疆政治中心由[[伊犁|伊犁]]移至[[迪化府|迪化]]（今[[乌鲁木齐|乌鲁木齐]]）。<ref>{{cite web|url=http://wen.org.cn/modules/article/view.article.php/1219/c18 |title=清末新疆建省研究综述 |first=新疆 |language=zh-hans }}</ref>

== 中華民國 ==
{{main|新疆省 (中華民國)|新疆省 (中华人民共和国)|新疆維吾爾自治區|中华人民共和国治疆历史}}
参见：[[袁大化|袁大化]]、[[杨增新|杨增新]]、[[东突厥斯坦伊斯兰共和国|东突厥斯坦伊斯兰共和国]]、[[盛世才|盛世才]]、[[三区革命|三区革命]]
* 1911年[[辛亥革命|辛亥革命]]后[[清朝|清朝]]被[[中华民国|中华民国]]取代，革命党人湖南人[[劉先俊|劉先俊]]等在[[伊犁|伊犁]]策动起义，殺了滿人將軍[[志銳|志銳]]，成立伊犂臨時軍政府。新疆[[巡抚|巡抚]][[袁大化|袁大化]]電奏清廷保[[阿克蘇|阿克蘇]][[道尹|道尹]][[杨增新|杨增新]]为新疆[[都督|都督]]，自己逃离新疆。杨增新在1912年和軍政府议和，逐渐取得全疆控制，臣服[[中华民国大总统|中华民国大总统]][[袁世凯|袁世凯]]（後為[[中華帝國|中華帝國]]皇帝）。革命党人或被杀死，或回到内地。同年在[[俄罗斯帝国|俄罗斯帝国]]的支持下外蒙古宣布獨立（参见[[外蒙古独立|外蒙古独立]]），[[乌兰巴托|库伦]]派兵攻打新疆[[阿爾泰區域|阿爾泰]]被杨增新派军击败。在杨控制新疆的十六年中，发展生产，新疆人民生活得以改善。

* 1928年6月，[[黃埔陸軍軍官學校|黃埔陸軍軍官學校]]校長兼[[國民革命軍|國民革命軍]]總司令[[蔣中正|蔣中正]]領導的[[國民革命軍北伐|國民革命軍北伐]]勝利，[[中華民國國民政府|中華民國國民政府]]設置「中華民國新疆省」，7月1日杨增新自行就任新疆省政府主席兼总司令，7月7日被刺身亡。11月17日省政务厅长[[金树仁|金树仁]]被[[南京|南京]][[国民政府|国民政府]]任命为新疆省政府主席。1930年代初，[[和加尼牙孜|和加尼牙孜]]等人在首先哈密發動暴動，新疆各处起義迭起。1933年西北軍閥[[馬仲英|馬仲英]]進入新疆，攻打[[迪化市|迪化]]，被新疆督辦公署參謀長[[盛世才|盛世才]]擊敗。同年4月12日在一些不滿金樹仁的軍官利用[[白軍|白俄]][[歸化軍|歸化軍]]士兵政變，盛世才入主省政府，掌握了全省軍政大權。金樹仁取道[[蘇聯|蘇聯]]逃往[[天津|天津]]。

[[File:Khotanlik_ulama_in_1933,_muhammad_amin_bughra_wearing_black_in_foreground.jpg|thumb]]
*民國二十二年（1933年）末，在英國的支持下，和闐人[[沙比提大毛拉|沙比提大毛拉]]、[[穆罕默德·伊敏|穆罕默德·伊敏]]等在[[疏附|疏附]]成立「東土耳其斯坦伊斯蘭共和國」，次年初被馬仲英所部擊潰。

* [[盛世才|盛世才]]前期依靠苏联支持巩固政权，还邀请[[中共|中共]][[陈潭秋|陈潭秋]]、[[毛泽民|毛泽民]]、[[林基路|林基路]]等人到新疆工作。后期和中共决裂，1943年在新疆杀死了共产党人。在整个[[抗战|抗战]]期间，新疆相对独立，是抗战的大后方。盛世才还派军支持了[[绥远抗战|绥远抗战]]等战役。

* 1944年，在[[苏联政府|苏联政府]]的支持下，[[新疆|新疆]]再次爆发反对[[中國国民党|中國国民党]]统治新疆的“[[伊宁事变|三区革命]]”（“三区”是指当时新疆的[[伊犁|伊犁]]、[[塔城地区|塔城]]和[[阿勒泰地区|阿勒泰]]三个地区），并再度打出“[[东突厥斯坦|东突厥斯坦]]”的旗号。1944年9月，[[國民政府軍事委員會|國民政府軍事委員會]][[委員長|委員長]]兼[[國民政府主席|國民政府主席]][[蒋中正|蒋中正]]将盛世才调到[[南京|南京]]，派[[吴忠信|吴忠信]]任新疆省长，并命令[[马步芳|马步芳]]派一个军的骑兵驻扎在新疆各地，对东突厥斯坦独立运动的活动进行镇压。

* 1949年，[[第二次国共内战|第二次国共内战]]末期，在[[彭德怀|彭德怀]]的[[中國人民解放军|中國人民解放军]][[第一野战军|第一野战军]]8月26日占领[[兰州|兰州]]继而占领[[甘肃|甘肃]]全境之后，[[中華民國國軍|中華民國國軍]]在新疆的军政长官[[陶峙岳|陶峙岳]]、[[包尔汉|包尔汉]]等宣布脱离[[广州|广州]]的[[中華民國政府|中華民國政府]]。期间已经投向[[中國共产党|中國共产党]]一边的[[张治中|张治中]]在中斡旋，而[[中共|中共]]方面的谈判代表是[[邓力群|邓力群]]。9月25日第一野战军第一兵团在[[王震|王震]]将军率领下攻佔乌鲁木齐，成为[[中华人民共和国|中华人民共和国]]的省，1955年改为自治区。
