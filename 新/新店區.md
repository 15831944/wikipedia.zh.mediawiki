'''新店區'''是臺灣[[新北市|新北市]]所轄管的一個[[市轄區|市轄區]]，位處[[臺北市|臺北市]]的南側，屬於[[臺北都會區|臺北都會區]][[內圈|內圈]]的[[衛星城市|衛星城市]]之一，其境內大多屬[[丘陵|丘陵]]地形，北邊大致以[[新店溪|新店溪]]與其支流[[景美溪|景美溪]]與[[臺北市|臺北市]]南緣的[[文山區|文山區]]為界。新店境內有[[碧潭|碧潭]]風景區，為[[北臺灣|北臺灣]]著名觀光景點之一，而由於新店近年有數座大型造鎮社區進行開發，當地已成臺北南側郊區主要的新興住宅供給地區。

== 歷史 ==
新店的名稱起源有多種說法。其中一種說法提及於[[清朝|清朝]][[乾隆|乾隆]]年間，相傳當時來自[[福建省|福建省]][[泉州|泉州]]地區的林章存等人，在通往[[烏來區|烏來]]山區、出山口河階地的道路旁架設小屋，開設店面銷售雜貨或與山區[[臺灣原住民族|原住民]]交換貨品，並逐漸發展成店鋪街。當時因店鋪沒有正式名稱，往來路人習以「新店」稱呼之，以便與較下游的店仔街（店鋪街，指今日的大坪林地區）區別，而成地名{{R|Bitan}}{{R|Land}}{{R|Culture}}。但又有一說指出林章存的店因在大洪水中遭沖毀，後經重建，為了與原本的舊店做出區別而將其稱為「新店」{{R|Bitan}}。

另一說則是在[[嘉慶|嘉慶]]、[[道光|道光]]年間，由閩南[[墾號|墾首]]陳合發、李詳記等人合資招募泉洲佃戶至台灣開墾。新到的移民將當地原住民泰雅族驅逐至[[灣潭|灣潭]]（新店溪在進入平原地帶後所形成的一個曲流地形，位在碧潭南側），並在碧潭東岸相當於今日新東、新西二里的位置處興建店鋪逐漸發展成商店街。為了與大坪林的店仔街作出區隔，而稱呼為「新店」{{R|Bitan}}。

最後一說則認為新店的發展是由官府起頭，在[[光緒|光緒]]十一年（1885年）時，在新店溪上游的屈尺地區（今屈尺-{里}-）設置撫懇支局，做為與泰雅族原住民交易的場所，並將其稱為「新店」{{R|Bitan}}。

至於新店一名首度被用作正式的行政單位命名，則是到1909年的[[台灣日治時期|日治時期]]，日本政府在此地區設置'''新店支廳'''，後於1920年改設'''新店-{庄}-'''，1943年升格為'''[[新店街|新店街]]'''。[[二次大戰|二次大戰]]結束後中華民國政府接收台灣，將此地改名為'''新店鎮'''，直到1980年7月1日配合人口的增加改制為'''新店市'''。2010年12月25日伴隨臺北縣升格為[[新北市|新北市]]，新店市也一同改制為'''新店區'''{{R|Culture}}{{R|Bitan}}。

對比於新店東半部被稱為「大坪林」的地區，新店西半部俗稱「安坑」；安坑位於新店溪西岸的山谷地區。此地區早期稱作「暗坑仔」；起因於此處林木蒼鬱，後來因為覺得不雅，才被改名為「安坑」。
新店曾為臺灣地區面積第二大的[[縣轄市|縣轄市]]{{efn|原臺灣面積最大的[[縣轄市|縣轄市]]為[[臺中縣|-{台}-中縣]][[太平區_(臺中市)|太平市]]。}}，也是原臺北縣面積最大的縣轄市。
== 地理與氣候 ==
新店位於[[台北盆地|臺北盆地]]南緣，境內多山，平地主要集中在新店溪與其支流沿岸的狹長平原帶，南高北低。境內山區屬於[[雪山山脈|雪山山脈]]分支的丘陵，境南的[[大桶山|大桶山]]是地理高點，海拔超過1000公尺。源自[[烏來區|烏來區]]境內[[拳頭山|拳頭山]]的[[南勢溪|南勢溪]]與源自[[新北市|新北市]][[坪林區|坪林]]地區的[[北勢溪_(新北市)|北勢溪]]在新店境內匯集，因此得到[[新店溪|新店溪]]之名，是臺北主要河川[[淡水河|淡水河]]最重要的支流之一。

新店溪大致是以由南往北的方向流經新店，將境內土地分為河西與河東兩半，雖然被稱為「安坑」的西岸土地面積較大，但該區的商業、行政機能卻集中在新店溪東岸。安坑地區與南部東部的山坡地帶，主要是以較獨立的山坡地[[住宅社區|住宅社區]]為主要人口分佈型態。新店區曾是臺北縣下轄的縣轄市中面積最大者，但轄區內大部分都屬於不適合居住的山區地形，發展腹地有限。以行政區位置而言，新店區位於新北市南側，北與[[臺北市|臺北市]][[文山區|文山區]]隔[[景美溪|景美溪]]相望，東與[[石碇區|石碇區]]相鄰，南與[[烏來區|烏來區]]相鄰，西與[[三峽區|三峽區]]相鄰，西北與[[土城區|土城區]]、[[中和區|中和區]]相鄰。

新店地區之氣候與臺灣北部地區氣候略同，屬[[亞熱帶季風氣候|亞熱帶季風氣候]]，高溫多雨，年平均溫度約在攝氏23度左右，濕度在80％間，因其靠近盆地邊緣山區雨量較多，年平均雨量均在2,000公厘以上。每年十月下旬至翌年三月盛行[[東北季風|東北季風]]，細雨綿綿，六月至九月，則多雷雨及颱風雨，為全年降雨量集中時節。風向以東南風居多，氣候尚稱適宜，依[[交通部中央氣象局|中央氣象局]]調查資料顯示，新店因靠近山區，年雨天數較多約為160天，年日照數1,463.6小時。

== 發展概況 ==
[[File:Taiwan_2009_Taipei_Xindian_FRD_7083.jpg|250px]]
新店原屬於台北[[衛星城市|衛星城市]]，1980年代以前並未有太多開發，繁榮者僅市區與鄰近台北市的主要幹道。諸如現有雙向四至六線道的中興路（自景美溪到碧潭，途經[[家樂福|家樂福]]新店店、[[五峰國中|五峰國中]]等地），還是1990年代初才開通的。

新店區的工業發展在早期最出名者，莫過於1953年10月時創立於新店的[[裕隆汽車|裕隆汽車]]（當時原名「裕隆機械製造股份有限公司」），是一家以代工製造[[日本|日本]][[日產汽車|日產汽車]]起家的台灣車廠{{R|Yulon}}。除此之外，1964年時由美國[[通用儀器|通用儀器]]（General Instrument，後來由[[摩托羅拉|摩托羅拉]]所收購）在新店所設立的「台灣通用電子」則是台灣第一家外商投資成立的公司<ref>《新店市志》</ref><ref>{{cite web|url=http://www.coolloud.org.tw/node/2123 | language=zh-tw | title=通用電子年底前結束台灣廠 訂單轉向鴻海  |publisher=苦勞網 |date=2007年4月7日 |accessdate=2012年4月1日}}</ref>。延續早年以電子產業為主的工業特色，今日在北新路與民族路交叉口，也有知名行動電話製造商[[宏達國際電子|宏達國際電子]]及電腦主機板大廠[[技嘉科技|技嘉科技]]的進駐。

1981年裕隆汽車將生產線移至[[苗栗縣|苗栗縣]][[三義鄉_(台灣)|三義鄉]]之後騰出了大面積的市區用地，開始拆除廠房改建為商場與住宅區，其中包括2004年時在中興路與寶橋路交叉口開設的[[家樂福|家樂福]]與[[特力屋|特力屋]]，一旁的帶狀汽車主題商場（命名為「裕隆汽車城」），與預計在2020年時啟用、由已故英國女建築師[[札哈·哈蒂|札哈·哈蒂]]的事務所所設計，整合[[誠品書店|誠品生活]]、[[威秀影城|威秀影城]]與一般住宅的大型綜合開發案「新店裕隆城」。

由於地處[[淡水河|淡水河水系]]的上游，新店是[[大坪林圳|大坪林圳]]、[[瑠公圳|瑠公圳]]、[[永豐圳|永豐圳]]、安坑圳等台北盆地重要灌溉渠道的水源頭，全台第一座[[水力發電|水力發電]]——[[龜山發電廠|龜山發電廠]]——也在新店境內。

新店境內設有數個重要的中央級政府機關，例如中華民國最高調查與情報機構[[法務部調查局|法務部調查局]]，與唯一可合法印製[[新台幣|新台幣]]紙鈔的[[中央印製廠|中央印製廠]]。另外，位在中山-{里}-、過去稱為「十四張」的地區有一個名為[[中央新村|中央新村]]、佔地約18[[公頃|公頃]]的社區，是1960至1970年代之間由中華民國政府向私人地主購地之後設置，以便安頓自中國大陸隨[[國民政府|國民政府]]遷居台灣的[[立法委員|立法委員]]、[[監察委員|監察委員]]與[[國大代表|國大代表]]等台灣早期的民意代表，而成為新店境內外省族群主要的聚居地{{R|中央新村}}。

1999年11月11日，台北捷運新店線全線通車，促進了捷運沿線各站周邊的發展。小碧潭站下方開設之前，也曾是稻田一片。小碧潭沿新店溪一帶，原本是休閒場地，北二高開通與公園開設後，不見舊日痕跡。

新店區有幾座傳統市場，且都擁有數十年以上的歷史。諸如同位於建國路上便有朝市（包括建國市場與周遭建國路、中正路、三民路等區域的攤商）與[[黃昏市場|黃昏市場]]，在[[大坪林_(新北市)|大坪林]]站2號出口附近，也有一個朝市，綿延到民權路。

=== 道路 ===
[[File:Taiwan_2009_Taipei_Xindian_FRD_7087.jpg|thumb]]
[[File:New_Taipei_City_Library_Xindian_Branch_3rd_Floor_Circulation_Desk_20111023.jpg|thumb]]
在通過新店區境的道路交通路線中，[[福爾摩沙高速公路|北二高]]（福爾摩沙高速公路北部路段）是最重要的一條，設置有新店與安坑兩個交流道。

在一般道路運輸方面，早期新店只有兩條主要交通線通往[[臺北市|臺北市]]中心，其中最重要者是沿著已經拆除的鐵路原址所修築的北新路，與臺北市區南邊的主幹道之一羅斯福路相連，是省道九號（由臺北起經[[宜蘭縣|宜蘭]]、[[花蓮縣|花蓮]]與[[臺東縣|臺東]]，終點為[[屏東縣|屏東]]，是臺灣東部的主要幹線）的一部份。另一條路線則是由新店經木柵（今[[臺北市|臺北市]][[文山區|文山區]]）地區通往臺北市東南側的[[辛亥路|辛亥路]]。

由於這兩條道路都是早期闢設的道路，路寬有限兩側腹地也不多，無法進一步拓寬而易壅塞，再加上北新路只有景美橋與景美舊橋兩條橋樑與[[臺北市|臺北市]]相連，辛亥路則需要穿越[[懷恩隧道|懷恩隧道]]與[[辛亥隧道|辛亥隧道]]，都是關鍵的交通瓶頸。為了解決此問題，臺北縣市政府合作在新店溪沿岸修築了[[環河快速道路|環河快速道路]]及[[水源快速道路|水源快速道路]]，是一條部分平面、部分高架的快速公路，連接臺北市[[公館_(臺北市)|公館]]地區並繼續往北與[[環河南北快速道路|環河南北快速道路]]相連。

在新店溪另一邊的安坑，與新店市區原本只有碧潭橋一條交通動線。早期此地對外主要交通路線有三條，除了往新店方向的碧潭橋外，還有一條通往[[中和區|中和]]（[[安和路|安和路]]，或稱為[[市道111號|市道111號]]），另一條則是通往[[三峽區|三峽]]（[[安康路|安康路]]，或稱為[[縣道110號|縣道110號]]）。由於這些都是早年興建的交通路線，在安坑地區陸續成立了許多大型的住宅社區（如臺北小城、[[玫瑰中國城|玫瑰中國城]]、大臺北華城與綠野香坡等社區）湧進大批居民之後，早已不敷使用而使聯外交通嚴重惡化。為了解決此問題，興建了北二高與連接安坑及環河—水源快速道路中央新村段的中安便橋，長期以來的交通問題才稍稍得到抒解。近幾年通車的[[新北環河快速道路|新北環河快速道路]]、中安大橋以及[[新店安坑一號道路|新店安坑一號道路]]<ref>李光儀，《新店「安坑一號道路」 經建會通過》，聯合報，2007年1月19日</ref>與[[台北捷運安坑線|捷運安坑線]]預計將能解決安坑的交通問題。

新店區是[[北宜公路|北宜公路]]（[[省道|省道]][[台九線|臺九線]]的一部份）的入山口，也是臺北欲前往[[宜蘭縣|宜蘭]]等東岸縣市最主要的必經地，而[[新烏公路|新烏公路]]（新店-烏來，也稱為省道九甲，是省道臺九線的支線）則是進出烏來區唯一的一條交通動線。雖然省道九甲是一條原本計畫要一路連往宜蘭市的交通線，但烏來區孝義至員山鄉雙連埤段未通車且已暫時放棄後續興建計畫，因此變成烏來、宜蘭兩頭各有一段省九甲卻無法連貫的現象。

2011年5月12日下午1點左右，在民生路75巷西側與中央路、民生路158巷北側之十四張空曠區域，發生臺灣北部氣象史上首見的[[龍捲風|龍捲風]]，長達五分鐘，並把一臺汽車捲翻。由於此次罕見氣候現象發生於陸地人口稠密區之白天上班日，鄰近之花開富貴社區、新店高中、附近中央新村、湯泉社區、大豐國小等不同方位有大量民眾及學生近距離或遠方同時目擊。多位民眾即時以具有錄影功能的手持設備拍攝實況影片紀錄後並將數位影片上傳於影音網站。甚至遠在4公里外的福和橋上及臺北市區高樓也能看見南方有大片厚重黑色之詭異雲層聚集。本次也是臺灣氣象史首次有影像紀錄之陸地龍捲風報告（臺灣近海海上龍捲風則有多次影像紀錄）。<ref>{{cite web |url=http://www.nownews.com/2011/05/12/327-2711905.htm |language=zh-tw |title=新店龍捲風翻車驚人　氣象局：鋒面+熱對流旺盛 |publisher=NowNews今日新聞網 |date=2011年5月12日 |accessdate=2012年4月1日 |archive-url=https://web.archive.org/web/20120623234437/http://www.nownews.com/2011/05/12/327-2711905.htm |archive-date=2012年6月23日 |dead-url=yes }}</ref><ref>{{cite web |url=http://www.appledaily.com.tw/realtimenews/article/new/20110512/30788 |language=zh-tw |title=快訊》新店出現龍捲風 |publisher=蘋果日報即時新聞 |date=2011年5月12日 |accessdate=2012年4月1日 |archive-url=https://web.archive.org/web/20160304210558/http://www.appledaily.com.tw/realtimenews/article/new/20110512/30788 |archive-date=2016年3月4日 |dead-url=yes }}</ref><ref>{{cite web|url=http://www.appledaily.com.tw/realtimenews/article/local/20110512/30800 | language=zh-tw | title=龍捲風》新店2百萬休旅車被吹翻 |publisher=蘋果日報即時新聞 |date=2011年5月12日 |accessdate=2012年4月1日}}</ref>

== 知名人物 ==
*[[羅福助|羅福助]]：前立委、[[天道盟|天道盟]]首任盟主。最有名事跡為創設天道盟及因被指稱黑道而掌摑時任立委李慶安。現因涉及刑事案件而被起訴，目前逃亡中國大陸，其子為現任國民黨籍立委羅明才。
