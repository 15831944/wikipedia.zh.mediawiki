'''新澳門學社'''（{{lang-pt|Associação de Novo Macau}}）成立於1992年7月10日，是[[澳門|澳門]]主要的[[民主派_(澳門)|民主派]]政黨，也是澳門少數有組織性的、獨立於政府的政治傳播團體<ref>《新聞學研究》（第102-105期），國立政治大學新聞研究所，2010年，第272頁</ref>。首屆理事長為[[吳國昌|吳國昌]]，現任理事長為[[甘雪玲|甘雪玲]]。

== 成立之初 ==

1980年代，[[澳門|澳門]]政治發展获得契機。時[[澳門總督|澳門總督]][[高斯達|高斯達]]於1984年解散意見不合的立法會，隨後赋予[[華人|華人]]選舉權，[[何思謙|何思謙]]領導的民生派在市政选举中獲得议席。而後[[中葡聯合聲明|中葡聯合聲明]]的簽署和[[中華人民共和國澳門特別行政區基本法|基本法]]的起草，以及1989年北京爆發的[[六四事件|民主運動]]，为新澳門學社的产生提供了時代背景。曾在1990年加盟以[[何思謙|何思謙]]為首的「友誼促進會」名義參與立法會增額補選但最終落敗的[[吳國昌|吳國昌]]，與[[區錦新|區錦新]]、湯家耀和陸偉聰等人，研製系統化政綱，於1992年7月10日宣佈成立新澳門學社，以及其包括政制人權、社會保障、經濟發展、城市建設、消費權益、公共行政、衛生環保、婦女、教育文化、公共財政和勞工11個專責小組，並於當年以「民主新澳門」名義，後來獲得3000多票首次晉身議會<ref name = "A1">[http://aamacau.com/2014/07/25/%E5%AD%B8%E7%A4%BE%E5%89%8D%E8%B7%AF%EF%BC%8C%E5%A6%82%E4%BD%95%E9%82%81%E6%AD%A5%EF%BC%9F%E2%94%80%E2%94%80%E5%B0%88%E8%A8%AA%E6%96%B0%E6%BE%B3%E9%96%80%E5%AD%B8%E7%A4%BE%E7%90%86%E4%BA%8B%E9%95%B7/ 學社前路，如何邁步？──專訪新澳門學社理事長蘇嘉豪], [[論盡媒體|論盡媒體]], 2014-07-25</ref>。

1993年5月，新澳門學社再提名[[區錦新|區錦新]]參加市政議會選舉，成功取得一個議席。1996和1997年，[[吳國昌|吳國昌]]和[[區錦新|區錦新]]分別連任立法會議員和市政議員。

== 澳門回歸後 ==

新澳門學社在1999年[[澳門回歸|澳門回歸]]後主張「反貪腐，爭民主，保民生」，透過議會工作監督特區政府，先後推動限制輸入外勞、十五年免費教育、增建[[公共房屋|公共房屋]]。

2001年，[[吳國昌|吳國昌]]第三度參選立法會，以「票王」身份成功連任，也帶領名單中排行第二的[[區錦新|區錦新]]初登議事廳，兩人於2005年繼續連任，排名第三的[[陳偉智|陳偉智]]以數百票之差未能當選，同年就讀[[澳門大學|澳門大學]]的[[周庭希|周庭希]]、梁博文和[[鄭明軒|鄭明軒]]等人創立仿照[[澳門日報|澳門日報]]的[[愛瞞傳媒|愛瞞日報]]，以「出紙一大張，瞞遍全澳市民」為口號，以[[惡搞|惡搞]]形式諷刺[[賄選|賄選]]、「賤價批地」、[[歐文龍貪污案|歐文龍貪污案]]、樓價高企、[[交通擠塞|交通擠塞]]等議題。

2007年，新澳門學社發表「民主政制發展方案」，主張2019年普選行政長官、逐步減少立法間選和官委議席並最終達致全面普選、設立分區直選的市政議會。同年又首次發起[[2007年澳門民主回歸大遊行|回歸日民主大遊行]]，有超過1,500人參與。[[2009年澳門立法會選舉|2009年立法會選舉]]，[[吳國昌|吳國昌]]和[[區錦新|區錦新]]分拆兩組出戰，以「1+1=4」為口號，呼籲選民平均配票，結果[[陳偉智|陳偉智]]成功當選。

2010年，[[周庭希|周庭希]]接任理事長一職，[[鄭明軒|鄭明軒]]任副理事長，主張關注[[環境保育|環境保育]]、[[新聞自由|新聞自由]]、[[人權|人權]]保障等議題。2012年，[[澳門|澳門]]啟動史上首次政制改革，新澳門學社組織包括全民投票、民意調查、斷食等行動，爭取立法會直選議席過半，結果政改未能通過。有評論員在[[正報|正報]]作出以下評論：「澳門民主運動的尷尬就是缺了社會運動，最常見的就只有遊行遞信，僅此而已，本地的『民主派』算是有少許論述主張，但卻少了指引爭民主的明確方向，更沒有行動綱領。明白他們都是斯文人，縱明『沒有抗爭，那有改變』的道理，也不致輕舉莽動，或許此間『民主派』只滿足做一隻『報喪鳥』 ，告訴人們那裡失火，那裡莊稼被盜（連監察者的角色也算不上、做不好，當然這是制度問題，非戰之罪。）」<ref>[http://www.chengpou.com.mo/news/think/2012/2/20/22131.html 民主運動的尷尬], [[正報|正報]], 2012-02-20</ref>，後來學社於[[聯合國|聯合國]]編製[[澳門|澳門]]首份民間人權報告，指責政府提交的報告中存在失實和虛假資訊，也向檢察院檢舉前特首涉嫌濫用權力批給賭牌等。

2013年，周庭希和陳偉智於[[2013年澳門立法會選舉|立法會選舉]]落敗。2014年，在[[2014年澳門反離補大遊行|反離保運動]]中有所行動的學社成員[[蘇嘉豪|蘇嘉豪]]被選為新任理事長。

2015年，蘇嘉豪以赴[[台灣|台灣]]升讀[[碩士|碩士]]為由請辭理事長一職，鄭明軒獲選為新任理事長，接替蘇嘉豪餘下一年任期。

2016年7月，新澳門學社舉行會員大會，選出新一屆理事會，其中鄭明軒連任理事長一職，兩名前理事長蘇嘉豪及周庭希分別擔任副理事長，而立法會議員吳國昌則選擇不參選理事的選舉，於學社首次不擔任理事。

2017年9月，蘇嘉豪在學社唯一推薦之下以「學社前進」名義成功以9212票當選，成為澳門歷來最年輕議員。吳國昌續以「民主昌澳門」名義參選，亦成功連任。2018年，學社亦遷出巴掌圍斜巷向吳國昌租用之物業，連同蘇之議員辦事處搬到沙欄仔街新社址。

== 分裂 ==
2009年的立法會選舉，[[區錦新|區錦新]]讓仍就讀[[澳門大學|澳門大學]]的[[周庭希|周庭希]]以第二候選人身份跟隨出選立法會，幾乎讓周成為最年輕的[[立法會議員|立法會議員]]。

2010年6月，22歲的周庭希在立法會選舉中敗選後不足一年於新澳門學社選舉中當選為理事長。

2013年6月，[[蘇嘉豪|蘇嘉豪]]於[[國立臺灣大學|國立臺灣大學]]剛畢業，同年7月，區錦新報名參選立法會時，亦讓年僅22歲的蘇嘉豪以第二候選人出選。與此同時，在[[吳國昌|吳國昌]]、區錦新等資深社員的主導下，新澳門學社亦開始換代。時任理事長周庭希邀請早已退出學社的[[張樹堅|張樹堅]]來領導學社的選舉工程，並通過理事會授予張樹堅全權負責擬訂學社的參選名單及選舉策略。

2014年，23歲的蘇嘉豪亦在立法會敗選後當選為學社新一屆的理事長。

2016年1月29日，區錦新退會，在退會聲明中說：由新一代主導的理事會在沒有任何合理原因下，刻意以批准入會作為控制關卡，非確認為「自己人」的就拒絕接納入會，企圖以排斥異己的方式來確保其持續控制理事會，從而牢固地掌握着新澳門學社。批評這做法無疑是把[[澳門民主派|澳門民主派]]旗艦變成了張樹堅的私人資產，完全背離了學社的創社宗旨及基本的民主原則<ref>[http://aamacau.com/2016/01/30/%e5%8d%80%e9%8c%a6%e6%96%b0%e9%80%80%e5%87%ba%e5%ad%b8%e7%a4%be-%e6%8f%ad%e5%85%a7%e7%88%86%e6%96%b0%e5%8e%9f%e5%9b%a0/]</ref><ref>[http://aamacau.com/2016/01/30/%E5%8D%80%E9%8C%A6%E6%96%B0%E9%80%80%E5%87%BA%E5%AD%B8%E7%A4%BE-%E6%8F%AD%E5%85%A7%E7%88%86%E6%96%B0%E5%8E%9F%E5%9B%A0/ 區錦新退出學社 揭內爆新原因]</ref><ref>{{Cite news|url=https://www.thestandnews.com/%E6%BE%B3%E9%96%80/%E6%BE%B3%E9%96%80%E6%B0%91%E4%B8%BB%E6%97%97%E8%89%A6%E5%88%86%E8%A3%82%E8%AA%B0%E4%B9%8B%E9%81%8E/?fref=gc&dti=1472136323095774|title=澳門民主旗艦分裂誰之過 {{!}} 讀者投稿 {{!}} 立場新聞|newspaper=立場新聞 Stand News|accessdate=2017-09-03|language=en}}</ref>。

截至[[2018年|2018年]]底曾代表學社參選立法會的[[周庭希|周庭希]]、[[鄭明軒|鄭明軒]]（以上均曾任理事長）、崔子釗、[[楊晚亭|楊晚亭]]、江洵美、[[鄭妙珊|鄭妙珊]]、李國強、李國富等人均退出了學社。

== 刊物 ==

== 政制發展 ==
2007年7月1日，發表《澳門特別行政區民主政制發展方案》，提出以下要點<ref>[http://newmacau.org/mainbody.php?inner=2007/development 新澳門學社：澳門特別行政區民主政制發展方案] {{webarchive|url=https://web.archive.org/web/20080414164336/http://newmacau.org/mainbody.php?inner=2007%2Fdevelopment |date=2008-04-14 }}</ref>：
# 以提名委員會代替目前的選舉委員會並逐步增加民主成份；
# 逐步減少立法會間接選舉和委任議席並最終透過修改《[[澳門特別行政區基本法|基本法]]》達致立法會全面普選；
# 加強立法會監察權力；
# 設立分區直選的市政機構；以及在
# 諮詢組織引入界別直選。
