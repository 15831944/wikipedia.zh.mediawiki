'''新疆生产建设兵团第二师'''，是[[新疆生产建设兵团|新疆生产建设兵团]]的师，与[[新疆维吾尔自治区|新疆维吾尔自治区]][[铁门关市|铁门关市]][[师市合一|师市合一]]。

==沿革==
1946年8月底，[[八路军|八路军]][[120师|一二〇师]][[359旅|三五九旅]]返回[[陕甘宁边区|陕甘宁边区]]，经历南下北返、[[中原突围|中原突围]]的连续作战，兵力消耗很大。[[王震|王震]]请求[[中共中央|中共中央]]从三五九旅派出一批干部到人口众多的[[山东解放区|山东解放区]]组建一个旅。中共中央很快批准，[[任弼时|任弼时]]还亲自给[[华东野战军|华东野战军]]司令员兼政委[[陈毅|陈毅]]写信。1946年冬，[[张仲翰|张仲翰]]、[[曾涤|曾涤]]、[[贺盛桂|贺盛桂]]、[[熊晃|熊晃]]等人率领三五九旅和[[晋绥军区|晋绥军区]]的321名干部来到山东省[[振华县|振华县]]、[[商河县|商河县]]、[[匡吾县|匡吾县]]、[[临邑县|临邑县]]等县组建新部队。1947年2月25日，新部队举行建军仪式，部队番号为'''渤海军区教导旅'''。同年 11月，该部队归入[[西北野战军|西北野战军]]建制，改称'''中国人民解放军西北野战军第二纵队独立第六旅'''<ref name=lsyg>{{Cite web|url =http://www.nes.gov.cn/zjes/lsyg/108.htm|title =历史沿革|author =|date =2014-06-26|publisher =第二师铁门关市|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170629173522/http://www.nes.gov.cn/zjes/lsyg/108.htm|archivedate =2017年6月29日|df =}}</ref>。

1949年2月，中央军委把独立第六旅统编为'''[[中国人民解放军第一野战军|中国人民解放军第一野战军]]二军步兵第六师'''。在[[彭德怀|彭德怀]]、王震指挥下，该部队在[[第二次国共内战|第二次国共内战]]西北战场先后参加运安、[[宜瓦战役|宜瓦]]等十大战役，随后进军陇东、河湟，相继攻占[[安邑县|安邑县]]、[[咸阳县|咸阳县]]、[[天水县|天水县]]、[[临夏县|临夏县]]等16座城市。1949年9月，陈兵[[嘉峪关|嘉峪关]]，促使[[新疆和平解放|新疆和平解放]]。1949年10月10日，步兵第六师自[[甘肃省|甘肃省]][[张掖|张掖]]出发，进至[[酒泉|酒泉]]、[[玉门|玉门]]、[[敦煌|敦煌]]一线待命。不久，王震命步兵第六师师长张仲翰迅速组建一支先遣队，在11月4日到达新疆南北疆分界处的[[焉耆|焉耆]]、[[库尔勒|库尔勒]]地区踏勘布点，受到当地各族人民欢迎<ref name=lsyg/>。

1953年2月，步兵第六师就地集体转业，执行农垦任务。1953年6月5日，[[新疆军区|新疆军区]]转发中央军委命令，步兵第六师改称'''中国人民解放军新疆军区农业建设第二师'''，下辖的十六团、十七团、十八团依次改名为农业建设第四团、第五团、第六团。师部驻焉耆。1954年10月，新疆军区生产建设兵团成立后，第二师转隶兵团建制，改称'''中国人民解放军新疆军区生产建设兵团农业建设第二师'''（简称“'''农二师'''”）。1960年9月，师部移驻库尔勒。1975年3月，兵团建制撤销。1975年5月，第二师并入[[巴音郭楞蒙古族自治州|巴音郭楞蒙古族自治州]]，组建农垦局，归巴音郭楞蒙古族自治州领导<ref name=lsyg/>。

1981年12月，中央恢复兵团体制，组建“新疆生产建设兵团”。1982年4月1日，农垦局撤销，恢复番号为'''新疆生产建设兵团农业建设第二师'''（简称“'''农二师'''”）<ref name=lsyg/>。

1984年，[[英国首相|英国首相]][[撒切尔夫人|撒切尔夫人]]成为世界上首位食用农二师二十八团香梨的外国首相。2005年新疆维吾尔自治区成立50周年之际，[[中共中央政治局委员|中共中央政治局委员]]、国务院副总理[[回良玉|回良玉]]代表中央慰问团向农二师赠送慰问品<ref name=lsyg/>。

2012年10月25日，[[中央机构编制委员会办公室|中央机构编制委员会办公室]]批准，同意新疆生产建设兵团各农业建设师由“新疆生产建设兵团农×师”更名为“新疆生产建设兵团第×师”。2012年12月24日，新疆生产建设兵团各农业建设师更名揭牌仪式在[[乌鲁木齐|乌鲁木齐]]举行<ref name=dshelye>{{Cite web|url =http://bt.xinhuanet.com/2015-07/30/c_1116094724.htm|title =《新疆生产建设兵团大事记》2012年|author =|date =2015-07-30|publisher =兵团网|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170511044405/http://bt.xinhuanet.com/2015-07/30/c_1116094724.htm|archivedate =2017-05-11}}</ref>。

2012年12月17日，国务院签署《关于同意新疆维吾尔自治区设立县级铁门关市的批复》，铁门关市作为新疆维吾尔自治区直辖的县级市正式成立，按照“师市合一”模式管理。2012年12月29日，第二师铁门关市正式挂牌成立，中共新疆维吾尔自治区党委副书记、兵团党委书记、政委[[车俊|车俊]]和中共新疆维吾尔自治区党委常委、[[新疆军区|新疆军区]]司令员[[彭勇|彭勇]]为“中国共产党铁门关市委员会”、“铁门关市人民政府”揭牌<ref name=dshelye/><ref>{{Cite web |url =http://www.chinanews.com/df/2012/12-29/4448106.shtml |title = 新疆铁门关市挂牌成立 实行“师市合一”管理模式 |author =  |date = 2012-12-29 |publisher =中国新闻网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.163.com/12/1227/23/8JP5IT4D00014JB6.html |title =国务院关于同意新疆维吾尔自治区设立县级铁门关市的批复  |author =  |date = 2012-12-27 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。

1965年1月初，农二师党委向兵团党委、中共巴音郭楞蒙古自治州党委提出开发[[若羌县|若羌县]]境内米兰地区的报告。1965年3月，[[中共新疆维吾尔自治区党委|中共新疆维吾尔自治区党委]]批准了农二师的报告。1965年4月，张仲翰亲自到米兰地区踏勘。至1966年末，米兰农场（现三十六团）已经开荒造田2.5万亩，并兴建了相应的水利工程和其他设施，还开采[[石棉|石棉]]620吨<ref>{{Cite web |url =  http://www.btzx.com.cn/2014/06/30/ARTI1404125246300987.shtml  |title =第十章社会主义建设高潮的兴起（第十节）  |author =  |date =2014-06-30  |publisher =  兵团在线|language =  |accessdate =  |quote =  }}</ref>。2011年3月，三十六团[[米兰河水利枢纽工程|米兰河水利枢纽工程]]开工建设，设计总库容4128万立方米，农业灌溉年供水量3310万立方米，工业设计年供水量3100万立方米，2015年9月[[米兰河|米兰河]]水库大坝主体工程完成<ref>{{Cite web |url = http://news.hexun.com/2015-09-07/178908277.html |title = 三十六团米兰河水利枢纽工程初步建成 |author =  |date = 2015-09-07 |publisher = 和讯 |language =  |accessdate =  |quote =  }}</ref>。

==编制==
新疆生产建设兵团第二师位于新疆维吾尔自治区[[巴音郭楞蒙古自治州|巴音郭楞蒙古自治州]]境内。第二师自北向南依次是焉耆、库尔勒、塔里木、且若四个垦区，分布在焉耆盆地内和[[塔克拉玛干沙漠|塔克拉玛干沙漠]]边缘。第二师下辖14个农牧团场、1家上市公司（冠农股份）、1个省级工业园、20个科教文卫事业单位、200多家中小企业<ref name=gongye>{{Cite web |url = http://www.nes.gov.cn/zjes/gyfz%20/14488.htm |title = 第二师工业特点与发展情况简介 |author =  |date = 2015-07-03 |publisher = 第二师铁门关市 |language =  |accessdate =  |quote =  |archive-url = https://web.archive.org/web/20170401232731/http://www.nes.gov.cn/zjes/gyfz%20/14488.htm |archive-date = 2017-04-01 |dead-url = yes }}</ref>。
