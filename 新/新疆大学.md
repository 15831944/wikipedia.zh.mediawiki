'''新疆大学'''（{{ug|شىنجاڭ ئۇنىۋېرستېتى|lat=Shinjang Uniwërsitëti}}），简称'''新大'''（Sh.U.），位于[[新疆维吾尔自治区|新疆维吾尔自治区]]首府[[乌鲁木齐市|乌鲁木齐市]]，是目前新疆办学规模最大、办学历史最悠久、学科门类最齐全、科研水平最高的综合性大学，是新疆唯一一所全国重点大学，是[[世界一流大学和一流学科建设|一流大学建设B类高校]]和中国政府面向21世纪重点建设的原“[[211工程|211工程]]”百所大学之一，是教育部与自治区“区部共建”高校。<ref name="简介">{{cite web| url=http://www.xju.edu.cn/xxgk/index.htm| language=| title=学校简介| publisher=新疆大学| date=| accessdate=2013年1月| deadurl=yes| archiveurl=https://web.archive.org/web/20121105171222/http://www.xju.edu.cn/xxgk/index.htm| archivedate=2012-11-05}}</ref>

== 历史年表 ==
新疆大学的前身是创办于1924年的[[新疆俄文法政专门学校|新疆俄文法政专门学校]]。
* 1928年8月，改称为[[新疆省立俄文法政学校|新疆省立俄文法政学校]]，校長是[[樊耀南|樊耀南]]。
* 1930年9月改为[[新疆俄文法政学院|新疆俄文法政学院]]。
* 1935年1月改建为[[新疆学院|新疆学院]]。
* 1941年省建设厅农业学校并入[[新疆学院|新疆学院]]。
* 1938年－1941年间，[[林基路|林基路]]、[[杜重远|杜重远]]、[[茅盾|茅盾]]、[[祁天民|祁天民]]、[[张仲实|张仲实]]、[[萨空了|萨空了]]、[[史枚|史枚]]、[[高滔|高滔]]、[[江豪|江豪]]、[[涂冶|涂冶]]、[[赵丹|赵丹]]、[[郭春则|郭春则]]等一批教育家、艺术家和親共人士，先后来学校担任领导、任教和讲学，学校名副其实地成为新疆传播革命真理的阵地和培养各族进步青年的摇篮。

[[中华人民共和国|中华人民共和国]]成立后：
* 1950年[[新疆语文学校|新疆语文学校]]和[[伊犁阿合买提江专科学校|伊犁阿合买提江专科学校]]并入[[新疆学院|新疆学院]]。
* 1950年10月1日更名为[[新疆民族学院|新疆民族学院]]，1951年6月[[新疆第二师范|新疆第二师范]]并入[[新疆民族学院|新疆民族学院]]。
* 1953年11月[[西北民族学院畜牧系兽医班|西北民族学院畜牧系兽医班]]并入[[新疆民族学院|新疆民族学院]]，1954年复名为[[新疆学院|新疆学院]]。
* 1960年10月1日正式成立新疆大学，1962年新疆大学与[[新疆师范学院|新疆师范学院]]合并。
* 1962年7月新疆大学机械、电机、工电等3个工科专业划入[[新疆矿冶学院|新疆矿冶学院]]。
* 1978年学校被[[中华人民共和国国务院|国务院]]确定为新疆唯一的[[全国重点大学|全国重点大学]]。
* 1979年4月，新疆大学教师培训部、三平分校划出成立新疆师范大学。
* 1997年列入国家“[[211工程|211工程]]”重点建设高校。
* 2000年被确定为国家西部大开发重点建设大学。
* 2000年12月30日，新疆大学与[[新疆工学院|新疆工学院]]合并组建新的新疆大学。
# [[新疆工学院|新疆工学院]]前身是1953年1月成立的[[中苏有色金属公司矿山技术学校|中苏有色金属公司矿山技术学校]]。
# 1955年1月更名为重工业部[[乌鲁木齐矿业学校|乌鲁木齐矿业学校]]。
# 1956年9月易名为冶金工业部[[乌鲁木齐矿业学校|乌鲁木齐矿业学校]]。
# 1958年7月改名为[[新疆矿冶学院|新疆矿冶学院]]。
# 1962年7月新疆大学机械、电机、工电等3个工科专业并入[[新疆矿冶学院|新疆矿冶学院]]。
# 1966年5月更名为[[新疆工学院|新疆工学院]]。
* 2004年，新疆大学成为[[新疆维吾尔自治区人民政府|新疆维吾尔自治区人民政府]]与[[中华人民共和国教育部|国家教育部]]的“省部共建”高校。

== 政府态度 ==
新疆唯一一所全国重点大学，中国首批211工程院校之一，省部共建高校，国家首批双一流建设高校。文化氛围浓厚，这所学校先后走出并培养了2位院士。
