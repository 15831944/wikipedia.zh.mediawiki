'''[[新義安|新義安]]'''為香港最有經濟能力的[[三合會|三合會]]，並在香港境內發展出不同派系，各佔據不同地盤<ref>{{Cite web|url=https://news.tvbs.com.tw/world/1061402|title=港最大黑幫三合會 派系搶地盤爆內訌 │TVBS新聞網|accessdate=2019-01-08|last=TVBS|work=TVBS}}</ref>。由於勢力龐大，並且渴望爭取最大利益，因此出現利益瓜葛，多次發生搶地盤。'''內訌事件'''在2018年變得更加明顯，兩區的新義安成員互斬，衝突原因只為意氣之爭，不涉及金錢利益。

== 傷人過程 ==

=== 2018年9月：尖沙咀夜場 始爆衝突 ===

==== 尖沙咀頭目被得罪 ====
2018年9月，一名女子在[[尖沙咀|尖沙咀]]夜場得罪了其中一方的頭目<ref>{{Cite web|url=https://www.hk01.com/突發/279221/追車劈人-新義安內訌升級-將軍澳少君門生遭尖沙嘴同門伏擊|title=【追車劈人】新義安內訌升級 將軍澳少君門生遭尖沙嘴同門伏擊|accessdate=2019-01-08|date=2019-01-05|last=凌逸德|work=香港01|language=zh-HK}}</ref>，該名吳姓女子24歲，手臂有紋身。據知是[[將軍澳|將軍澳]]新義安一小頭目的契女，亦因此觸發將軍澳和尖沙咀派系的衝突。

==== 將軍澳尚德斬人案 ====
數日後的9月26日凌晨3時21分，該名女子與數名親友於將軍澳唐明街[[尚德邨|尚德邨]]尚真樓對開休憩公園閒聊期間，突然遭到多名持刀蒙面男子追斬。女事主身中七刀重傷，血流一地，其後被送院搶救後留醫深切治療部，情況一度危殆<ref>{{Cite web|url=https://www.hk01.com/突發/279304/追車劈人-逾五百警反黑一夜-拘黑幫骨幹-檢血刀防暴盾拘刀手|title=【追車劈人】逾五百警反黑一夜 拘黑幫骨幹 檢血刀防暴盾拘刀手|accessdate=2019-01-08|date=2019-01-06|last=凌逸德|work=香港01|language=zh-HK}}</ref>。至於刀手則成功逃脫。

==== 土瓜灣焚燒賊車案 ====
尚德傷人案的兩名男子行兇後逃去無蹤。約半小時後的凌晨3時52分，一輛懷疑賊車於[[土瓜灣|土瓜灣]][[承啟道|承啟道]]起火，據報有人匆忙下車逃去。消防到場灌救後發現有可疑，警方亦相信與傷人案有關。

==== 寶琳景林邨縱火案 ====
同日清晨5時17分，[[寶林|寶琳]][[景林邨|景林邨]]的停車場突然起火。消防撲救後發現停車場內超過30架車被焚毀，警方調查後相信起火原因有可疑，重案組接手調查。據知尚德傷人案的女事主被斬後，受襲一方查出尖沙咀派其中一位頭目有一輛車泊在景林邨的停車場，於是派人放火燒車報復，但火勢太大令停車場內多部車輛都被焚燒。

=== 2018年11月：尖沙咀新人 被斬報復 ===

==== 九龍城咪錶斬人案 ====
11月3日凌晨零時許，33歲姓冼男子及34歲姓林男子步往[[九龍城|九龍城]][[南角道|南角道]]63號對開咪錶位準備取車，當拉開車門之際，一輛私家車疾馳而至，跳出2名約1.75米高，俱戴口罩及穿黑色上衣大漢，並舉刀向2人揮斬。2名事主始料不及，其中踢拖的姓林男子右臂及腰中刀披血，冼右手腕及大腿被斬傷，鮮血濺地，房車掩門位置亦遺下血漬，兇徒得手後逃去。

警員接報趕至，將2名傷者送院救治，並在附近一帶兜截，惜無發現，正翻查店舖「天眼」搜證。案件列為傷人案，交由九龍城警區刑事調查隊跟進。

據知姓冼男子暱稱「高腳仔」，乃由新義安灣仔分支跳槽到尖沙咀的新力軍。

=== 2018年12月：將軍澳黑夜 多處傷人 ===
==== 將軍澳夜市被搗亂 ====
高腳仔被斬後一個月尖沙咀人馬做嘢12月3日晚上11時，九名男子在[[尚德廣場|尚德廣場]]對開的熟食檔搗亂，擺賣的小販手推車被掃亂，並嚇走市民，大肆破壞後迅速逃離現場。其中一名42歲女檔主疑因走避不及，不幸遭滾湯灼傷臉和手部<ref>{{Cite web|url=https://www.hk01.com/突發/267132/將軍澳黑夜-廝殺再臨-尚德小販遇襲前-藏刀九男聚街頭警截獲|title=【將軍澳黑夜】廝殺再臨？尚德小販遇襲前 藏刀九男聚街頭警截獲|accessdate=2019-01-08|date=2018-12-04|last=黃景洪|work=香港01|language=zh-HK}}</ref>。警方調查後相信與黑幫內訌有關。

==== 寶琳景林邨傷人案 ====
12月28日晚上10時59分，一名35歲馮姓男子， 綽號阿燦，在景林邨停車場取車期間被三名男子斬傷。事主身中幾刀，兇徒則登車逃去，報警後送院救治。據知其中一位兇徒為新義安將軍澳派成員<ref>{{Cite web|url=https://www.hk01.com/突發/276096/將軍澳黑夜再臨-景林邨停車場男子遭三刀手斬傷-兇徒燒車毀證|title=【將軍澳黑夜再臨】景林邨停車場男子遭三刀手斬傷 兇徒燒車毀證|accessdate=2019-01-08|date=2018-12-28|last=賴南秋|first=呂凝敏|work=香港01|language=zh-HK}}</ref>。

==== 寶琳澳頭村縱火案 ====
同日晚上11時15分，[[衛奕信徑|衛奕信徑]]第三段近將軍澳[[翠琳路|翠琳路]]近澳頭村發生山火，消防到場將火救熄，發現原來有一輛私家車起火，波及山草。初步發現私家車旁有懷疑助燃劑，消防相信起火原因有可疑，已交由警方接手調查並列為縱火案處理，不排除與斬人案件有關<ref>{{Cite web|url=https://news.mingpao.com/ins/%e6%b8%af%e8%81%9e/article/20181228/s00001/1545963428847/%e3%80%90%e7%9f%ad%e7%89%87%e3%80%91%e5%b0%87%e8%bb%8d%e6%be%b3%e6%99%af%e6%9e%97%e9%82%a8%e9%bb%91%e5%b9%ab%e4%bb%87%e6%ae%ba-%e7%94%b7%e5%ad%90%e9%81%ad%e4%bc%8f%e6%93%8a%e6%8d%b1%e6%96%ac|title=【短片】將軍澳景林邨黑幫仇殺 男子遭伏擊捱斬 (10:26) - 20181228 - 港聞|accessdate=2019-01-08|work=明報新聞網 - 即時新聞 instant news|language=zh-hant}}</ref>。

=== 2019年1月：九龍區衝突 觀塘參戰 ===

==== 大角咀斬人燒車案 ====
2019年1月1日晚上8時，一名29歲姓張男子駕駛七人車停泊於[[大角咀|大角咀]][[海富苑|海富苑]]海泰閣對開時，突然遭兩架私家車左右夾擊撞停，兩車約有7名男子跳車持刀向事主襲擊，傷者手臂受傷後開車逃走，倒車期間失控撞向後方一輛白色私家車。刀手亦跳上接應車輛逃去。

遇襲後傷者沿[[海庭道|海庭道]]往[[奧海城|奧海城]]方向駛去。約5分鐘後，一輛套上假車牌的黑色私家車被發現在[[油麻地警署|油麻地警署]]對開[[海泓道|海泓道]]起火焚燒，地上可見疑似易燃物料，警方將案件列傷人及縱火處理。私家車起火的地點與刀手斬人位置相距約700米。

遇襲的車主其後駛至海庭道[[富榮花園|富榮花園]]一期16座對開，報警後送[[廣華醫院|廣華醫院]]治理。遺留現場的七人車司機位車門及車窗可見多道刀痕，車頭損毀車牌脫落，車身右後方有凹痕，警方其後作進一步調查<ref>{{Cite web|url=https://www.hk01.com/突發/277644/斬人燒車-大角咀七人車被夾擊-7刀手斬傷司機燒車毀證|title=【斬人燒車】大角咀七人車被夾擊 7刀手斬傷司機燒車毀證|accessdate=2019-01-08|date=2019-01-01|last=陳浩然|work=香港01|language=zh-HK}}</ref><ref>{{Cite web|url=https://www.hk01.com/突發/277694/斬人燒車-警將軍澳厚德邨拘捕三男子-疑涉黑幫同門內訌|title=【斬人燒車】警將軍澳厚德邨拘捕三男子 疑涉黑幫同門內訌|accessdate=2019-01-08|date=2019-01-02|last=黃偉民|first=鄧栢良|work=香港01|language=zh-HK}}</ref>。

==== 彩虹撞車持刀傷人案 ====
1月5日下午3時多，一輛白色私家車沿[[太子道|太子道東]]行駛至彩虹邨對開馬路時，被兩輛銀色私家色夾攻猛撞。七名刀手衝向白色私家車用利刀狂襲擊白色私家車，車上二人受傷。刀手其後逃去無蹤。據悉被襲擊的是將軍澳新義安頭目「少君」的門生，初步懷疑刀手為尖沙嘴新義安成員。

下午4時許，警方於[[九龍城|九龍城]][[啟晴邨|啟晴邨]]尋獲接載兇徒逃走的寶馬私家車，車內放有鐵通及刀。涉事車輛套有一個為車牌<ref>{{Cite web|url=https://www.hk01.com/突發/279221/追車劈人-新義安內訌升級-將軍澳少君門生遭尖沙嘴同門伏擊|title=【追車劈人】新義安內訌升級 將軍澳少君門生遭尖沙嘴同門伏擊|accessdate=2019-01-08|date=2019-01-05|last=凌逸德|work=香港01|language=zh-HK}}</ref><ref>{{Cite web|url=https://www.hk01.com/突發/279152/追車劈人-七刀手彩虹邨外公路攔車夾擊-持利刀狂劈斬傷平治男|title=【追車劈人】七刀手彩虹邨外公路攔車夾擊 持利刀狂劈斬傷平治男|accessdate=2019-01-08|date=2019-01-05|last=余睿菁|first=彭愷欣, 王譯揚|work=香港01|language=zh-HK}}</ref>。四名在平治房車上遭蒙面兇徒追斬的男子，均是涉及大角咀斬人燒車案被捕的將軍澳新義安黑漢，他們剛獲准保釋就遇襲，相信涉及尖沙嘴新義安報復。

據知觀塘分支亦涉及事件，報復數年前爭地盤失敗的仇恨<ref>{{Cite web|url=https://hk.news.appledaily.com/breaking/realtime/article/20190107/59112259|title=新義安廝殺爭地盤 將軍澳「少君」VS觀塘「阿刁」|accessdate=2019-01-08|work=Apple Daily 蘋果日報}}</ref>。
