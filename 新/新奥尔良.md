'''新奧爾良'''（{{lang-en|'''New Orleans'''}}，{{IPAc-en|n|uː|_|ˈ|ɔr|l|ɪ|n|z}}，<ref>{{cite web|last1=Romer|first1=Megan|title=How to Say 'New Orleans' Correctly|url=http://goneworleans.about.com/od/NewOrleansBasics/fl/How-To-Say-New-Orleans-Correctly.htm|website=About Travel|publisher=about.com|accessdate=31 January 2015}}</ref><ref>{{cite web|title=Merriam-Webster Pronunciation|url=http://www.merriam-webster.com/audio.php?file=ggnewo01&word=New%20Orleans&text=\%CB%88o%CC%87r-l%C4%93-%C9%99nz%2C%20%CB%88o%CC%87r-l%C9%99nz%2C%20%CB%88o%CC%87rl-y%C9%99nz%2C%20%28%CB%8C%29o%CC%87r-%CB%88l%C4%93nz\|website=Merriam-Webster|publisher=Merriam-Webster, Inc.|accessdate=31 January 2015}}</ref>{{IPAc-en|n|u:|_|ˈ|ɔr|l|i|ə|n|z}}，{{IPAc-en|n|uː|_|ɔr|ˈ|l|iː|n|z}}或{{IPAc-en|ˈ|n|ɔr|l|ə|n|z}}；{{lang-fr|'''La Nouvelle-Orléans'''}} {{IPA-fr|la nuvɛlɔʁleɑ̃||La Nouvelle-Orleans.ogg}}）舊譯'''紐柯連'''，是[[美国|美国]]主要的港口城市，位于[[路易斯安那州|路易斯安那州]]的南部，同时也是该州最大的城市。行政上與'''奧爾良堂区'''（{{lang|en|Orleans Parish}}）同域。

新奥尔良因为其数目众多的法式建筑、克里奥尔美食和丰富的音乐形式（[[爵士乐|爵士乐]]的诞生地）而闻名，也被人称为“美国最独特的城市”。

== 历史 ==
新奥尔良由[[密西西比公司|密西西比公司]]在[[让-施洗者·勒·蒙|让-施洗者·勒·蒙]]的指挥下建立于1718年5月7日。当时新奥尔良的土地属于[[奇蒂馬查人|奇蒂馬查人]]。这个新城市的名字是为了纪念[[奥尔良公爵|奥尔良公爵]][[腓力二世|腓力二世]]，当时的法国摄政王。

随着[[七年战争|七年战争]]的失利，法王[[路易十五|路易十五]]签署了[[1763年巴黎条约|1763年巴黎条约]]，将[[新法兰西|新法兰西]]的大部分都割让给了英国，失去了密西西比河以东的土地。新奥尔良则在1762年秘密签署的[[枫丹白露条约_(1762年)|枫丹白露条约]]中被割让给西班牙。

新奥尔良在[[美国独立战争|美国独立战争]]时期是支持[[大陆军|大陆军]]的一个重要港口，法国人通过新奥尔良走私数目庞大的军火和补给，缓解了大陆军在战争初期的不利局势。{{tsl|en|Bernardo de Gálvez|伯纳多·德·加尔维斯}}在1779年成功地抵御了英国军队对新奥尔良的进攻。

2005年[[颶風卡特里娜|颶風卡特里娜]]吹襲後，新奥尔良成為重災區。

== 地理 ==
新奥尔良位于29°57′53″N 90°4′14″W的密西西比河岸边上，大致在[[墨西哥湾|墨西哥湾]]上游的169公里处。根据[[美国人口调查局|美国人口调查局]]给出的数据，新奥尔良的城市面积为907平方公里，其中51.55%，也就是467.6平方公里是陆地。
新奥尔良地处[[密西西比河三角洲|密西西比河三角洲]]，同时也毗邻[[庞恰特雷恩湖|庞恰特雷恩湖]]。新奥尔良接近河岸的部分是由许多起伏的丘陵组成的。

新奥尔良属典型的[[副热带湿润气候|副热带湿润气候]]，大体上终年暖和，雨热同季。冬季温和，微潮，时而偏向温暖，日最高气温低于或等于{{convert|10|C}}的平均日数10.4天，日最低气温低于或等于{{convert|0|C}}的平均日数8.0天；夏季炎热潮湿，日最高气温超过{{convert|30|C}}的日数年均有131天，超过{{convert|35|C}}的有12天；日低温大于或等于{{convert|25|C}}的年均日数为23天。<ref name = "NWS Slidell, LA"/>最冷月（1月）均温{{convert|53.4|F|1|disp=flip}}，国际交换站极端最低气温{{convert|11|F|0|disp=flip}}（1989年12月23日）；全市极端低温为{{convert|6|F|0|disp=flip}}（1899年2月13日在Audubon公园）。<ref name = "NWS Slidell, LA"/>最热月（8月）均温{{convert|83.3|F|1|disp=flip}}，国际交换站极端最高气温{{convert|102|F|0|disp=flip}}（1980年8月22日）；全市极端高温为{{convert|105|F|0|disp=flip}}（2004年7月25日在KNEW）。<ref name = "NWS Slidell, LA"/>年均降水量约{{convert|62.5|in|sigfig=3|disp=flip}}；际交换站年极端最少降水量为{{convert|38.88|in|disp=flip}}（2000年），最多为{{convert|102.37|in|disp=flip}}（1991年）；全市年极端最少降水量为{{convert|28.03|in|disp=flip}}（1938年在KNEW），最多为{{convert|113.74|in|disp=flip}}（1991年在Audubon公园）。<ref name = "NWS Slidell, LA"/>雪为罕见；1852年以来，全市有17场可测量的雪，<ref name="snow climo">{{cite web|title=Snow Climatology for Southeast LA/South MS|url=http://www.srh.noaa.gov/lix/?n=snowcli|publisher=NWS Slidell, LA|accessdate=2014年7月22日|language=en}}</ref>最后一场可测量出现于2009年12月4日，<ref name = "NWS Slidell, LA"/>最大一场为{{convert|8.2|in|cm|disp=flip}}（1895年2月14日与15日）。<ref name="snow climo"/>无霜期平均有315天（2月6日至12月17日）。<ref name = "NWS Slidell, LA"/>

== 人口 ==
新奧爾良人口現時以非裔美國人為主，也是[[靈魂樂|靈魂樂]]發源地，更是路易斯安那州第一大城市。在曾经是一个移民城市，除了欧洲的冒险家外，这城市还是当时还有[[奴隶制度|奴隶制度]]的美国最主要输入黑人[[奴隶|奴隶]]的城市。1960年代後大量黑人移居新奧爾良，並逐漸取代白人成為第一大族群。现时大部分人口皆为黑人奴隶的后裔。新奧爾良一直是[[民主党_(美国)|民主黨]]的大本營。

== 经济 ==
[[File:LPD18USSNewOrleansPassingNewOrleans.jpg|thumb]]河畔的新奥尔良中心商务区天际线]]
新奧爾良主要以[[旅遊业|旅遊业]]、[[运输业|运输业]]和[[赌博业|赌博业]]作为经济支柱。

== 文化 ==
外来移民为此城引入丰富多变的文化，以[[爵士乐|爵士乐]]发源地和[[法国区|法国区]]的[[克里奥|克里奥]]（Creole）文化闻名，居住人口大多是黑人及欧裔人士为主，法国区克里奥人是早期法国殖民者后代，克里奥人保存祖先的语言与文化，新奧爾良也因此是美国最有异国情调的城市，每年嘉年华会庆典以及融合法式和西班牙式烹饪的海鲜佳肴吸引川流不息的观光客。

== 交通 ==
新奥尔良国际机场全称为[[新奥尔良路易斯阿姆斯特朗国际机场|新奥尔良路易斯阿姆斯特朗国际机场]]。

新奥尔良国际机场是一座位于美国路易斯安那州[[杰佛逊堂区_(路易斯安那州)|杰佛逊县]]的民用机场。归属于新奥尔良市并位于市中心商业区西10海里（19公里）处。机场地址为路易斯安那[[肯纳_(路易斯安那州)|肯纳市]]航线路900号。一小部分跑道位于圣查尔斯区的飞地。阿姆斯特朗机场为大新奥尔良区和东南路易斯安那的主要商业机场。过去该机场被称为摩圣特机场，也叫做路易斯阿姆斯特朗国际机场和新奥尔良国际机场。

== 教育 ==
[[File:TulaneU4Nov08JonesHall.jpg|thumb]]
新奥尔良市区拥有数座高等院校，其中最知名的是创建于1834年的[[杜兰大学|杜兰大学]]。

杜兰大学常被誉为“南方常春藤”，与[[艾默理大学|艾默理大学]]、[[范德堡大学|范德堡大学]]等大学齐名。学校获得《美国新闻及世界报道》评为一级国家级大学。1882年，新奥尔良商人保尔·杜兰（PAUL.TULANE）捐献了市价超过一百万美金的房地产作为教育基金，并设立杜兰教育基金委员会（Board of Administrators of the Tulane Educational Fund）监管资金使用。1884年州议会通过决议将大学的监管权移至杜兰教育基金委员会，路易斯安娜大学从此转为私立并更名为杜兰大学。
1885年，杜兰大学设立研究生院。1894年，杜兰大学设立工程学院，即现在的理学院。同年杜兰大学搬迁至目前在uptown的校区。1914年杜兰大学设立商学院，是当时美国南部最早的商学院。随后，杜兰大学陆续成立了其他学院。

2005年，由于卡特里娜飓风杜兰大学取消了秋季课程。2006年在灾后重新开课，有93%的学生重新返回了校园。

此外，[[新奥尔良大学|新奥尔良大学]]是新奥尔良市的一所著名公立大学，1956年由州议员西奥多·哈克尼（Theodore M. Hickey）提出相关法案而创立，2011年加入路易斯安那大学系统。
