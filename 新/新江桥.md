'''新江桥'''是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]]一座跨[[姚江|姚江]]桥梁，是姚江汇入[[甬江|甬江]]之前的最后一座桥梁。新江桥最初为1862年由英国人建起的浮桥，1970年建成三孔钢筋混凝土双曲拱桥。2006年9月11日晚9时，由于被鉴定为危桥，新江桥被紧急封桥，2007年12月9日由于[[杭甬运河|杭甬运河]]通航条件的原因而被拆除。2007年1月31日建成新江桥便桥。2013年7月，新江桥原址重建开工，2015年11月10日建成通车。<ref>{{cite web|url=http://dzb.jinbaonet.com/html/2015-11/10/content_291087.htm?div=1|title=阔别八年多 新江桥今日全线通车|publisher=现代金报|author=|date=2015-11-10|accessdate=2015-10-13}}</ref>

== 浮桥 ==
[[File:Floating_Bridge_of_Xinjiangqiao_Bridge.jpg|20世纪40年代末的新江桥浮桥]]
新江桥最初为1862年由英国商人台弗逊建起的浮桥，原位于旧城墙盐仓门外关帝殿道头（即今天[[解放桥_(宁波)|解放桥]]位置），次年迁移到江北岸桃花渡和宁波旧城墙东渡门之间。<ref name="floating">{{cite news|author=王重光 王尧山|title=新江桥的变迁|work=宁波晚报|page=A17|date=2006-11-18|accessdate=2011-03-09|url=http://www.cnnb.com.cn/gb/node2/newspaper/pdf/nbwb/2006/11/18/17.pdf|deadurl=yes|archiveurl=https://web.archive.org/web/20160305163500/http://www.cnnb.com.cn/gb/node2/newspaper/pdf/nbwb/2006/11/18/17.pdf|archivedate=2016-03-05}}</ref> 在此之前，宁波仅有一座跨江桥梁，即[[灵桥|灵桥]]（东津浮桥），而姚江两岸居民来往只能依靠摆渡。由于灵桥被称为'''江桥'''，因而新桥建成后，灵桥被称为'''老江桥'''，而新建浮桥被称为'''新江桥'''。<ref>{{cite web|url=http://www.gotoningbo.com/zx/yznb/lyjd/200908/t32260.htm|title=新江桥简介|publisher=宁波旅游网|date=2009-08-24|accessdate=2011-03-09}}</ref> 桥梁建成后，英国巡捕在桥上收取每人四文的买路钱，引起宁波民众的不满。1869年5月24日，在依照风俗举行迎“五都元帅”赛神会之时，新江桥上人流拥挤，而英国巡捕仍然索要过桥费，最终造成浮桥绷断，400余人落水丧生。1877年，浮桥由宁波人士提议，利用洋药局的积余金赎回。1935年和1946年，宁波曾经两次试图在新江桥位置建设固定桥梁，均未获得成功。<ref name="floating" />

== 混凝土桥 ==
[[File:Remaining_Part_of_Old_Cement_Xinjiangqiao_Bridge.JPG|thumb]]
1958年，新江桥永久桥梁的建设进行了勘探和初步设计，原计划于1961年建设，后因资金困难作罢。1969年4月，定名为'''反帝桥'''的建桥计划重新获批，并于1969年8月13日动工，1970年9月建成通车。<ref name="archive-anti-imperialism">{{cite web|url=http://www.nbdaj.gov.cn/dandt/yczg/lbxd/200801/t20080124_9665.html|title=昔日的辉煌——记1970年的“反帝桥”|publisher=宁波市档案馆|date=2008-09-19|author=林爱|accessdate=2011-03-09}}</ref> 桥梁形制为三孔钢筋混凝土双曲拱桥，长163米，加上引桥全长304米，宽19.5米，中孔跨径37米，边孔跨径33米。<ref name="stop use">{{cite web|url=http://zjnews.zjol.com.cn/05zjnews/system/2006/09/14/007873230.shtml|title=服役36年不堪重负 宁波新江桥“过劳死”|author=王银定 陈进红|publisher=钱江晚报|date=2006-09-14|accessdate=2011-03-09}}</ref> 原浮桥被用于今解放桥位置，解放桥建成后又被用于今天[[江厦桥|江厦桥]]的位置。[[文化大革命|文化大革命]]结束后，新江桥改回原名。<ref name="floating" /> 当年，新桥建成时，正桥栏杆设有[[毛主席语录|毛主席语录]]牌，上书“全世界人民团结起来，打败美国侵略者及其一切走狗”，正桥、引桥栏杆上装饰有火炬和工农兵图案，并设有10对火炬灯组。<ref name="archive-anti-imperialism" />

由于设计和选址存在缺陷，这座桥梁曾经多次遭到船只撞击，毁损严重。2006年5月，经过上海同济建设工程质量检测站检测，新江桥处于E级危险状态，随时可能坍塌。鉴于此，2006年9月11日晚9时，宁波市城管局对新江桥采取了紧急封桥措施，只允许非机动车通行。这一措施在宁波中心城区产生了严重的交通拥堵。<ref name="stop use" />2007年12月9日，新江桥桥面由于达不到[[杭甬运河|杭甬运河]]的通航要求而被拆除。<ref>{{cite web|url=http://fz.ningbo.gov.cn/detail_16776_46.html|title=新江桥老桥告别甬城|publisher=宁波日报|date=2007-12-09|accessdate=2011-03-09|deadurl=yes|archiveurl=https://web.archive.org/web/20160304130725/http://fz.ningbo.gov.cn/detail_16776_46.html|archivedate=2016-03-04}}</ref>

== 临时便桥 ==
[[File:Temorary_Xinjiangqiao_Bridge_in_Ningbo.JPG|thumb]]
为缓解新江桥拆除带来的交通问题，2007年1月31日，新江桥临时便桥通车。这座便桥为4跨钢桁架梁简支桥梁，位于老桥上游30米左右，经过83天建成。桥长146米，宽28米，单跨36.576米，双向4车道。桥面行车道采用混凝土加沥青，人行道上铺塑胶。<ref>{{cite web|url=http://cmspub.cnnb.com.cn/dfwmzx/system/2007/01/31/010003782.shtml|title=新江桥便桥正式通车|publisher=东方文明在线|author=曹爱方 徐波娜|date=2007-01-31|accessdate=2011-03-09}}</ref>2009年，临时便桥桥头涵洞出现开裂，据宁波桥梁管理所称，这是由路基下沉造成，并无安全隐患。<ref>{{cite web|url=http://nb.focus.cn/news/2009-12-11/814697.html|title=新江桥便桥涵洞出现开裂 官方证实无安全隐患|publisher=东南商报|author=汪林|date=2009-12-11|accessdate=2011-03-09}}</ref>

== 永久桥梁 ==
由于新江桥便桥通车已逾4年，桥梁的使用寿命受到了关注。2011年，新江桥永久桥梁正式立项，计划建设的桥梁长672米，宽36米，双向6车道，南起[[中山路_(宁波)|中山东路]]，北至扬善路，计划于当年开工，2014年通车。<ref>{{cite web|url=http://news.cnnb.com.cn/system/2011/03/09/006865825.shtml|title=新江桥年内开工2014年通车 工程总投资3.8亿元|publisher=东南商报|author=林伟|date=2011-03-09|accessdate=2011-03-09}}</ref>

根据三江口核心区改造提升规划，新江桥西移新建，北接[[人民路_(宁波)|人民路]]，向南连接[[车轿街|车轿街]]和[[日新街|日新街]]一组单向交通道路。<ref>{{cite web|url=http://news.cnnb.com.cn/system/2012/04/27/007303043_01.shtml|title=三江口核心区改造提升规划确定 新江桥将西移重建|publisher=中国宁波网|author=王岚|date=2011-04-27|accessdate=2011-04-28}}</ref>

2015年9月27日，新江桥因西侧引道施工临时开通南往北方向的半幅。<ref>{{cite web|url=http://daily.cnnb.com.cn/nbwb/html/2015-09/28/content_896882.htm?div=-1|title=新江桥半幅通行|publisher=宁波晚报|author=龚国荣|date=2015-09-28|accessdate=2015-10-10}}</ref><ref>{{cite web|url=http://daily.cnnb.com.cn/dnsb/html/2015-09/29/content_897248.htm?div=-1|title=新江桥开通了？ 不是，因西侧引道施工需要临时开通半幅|publisher=东南商报|author=林伟|date=2015-09-29|accessdate=2015-10-10}}</ref>
