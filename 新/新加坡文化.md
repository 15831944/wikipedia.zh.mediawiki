'''新加坡文化''' 是融合了[[亞洲文化|亞洲文化]]與[[歐洲文化|歐洲文化]]的一种文化，它受影響於[[馬來人|馬來人]]、[[南亞|南亞]]族裔、[[東亞|東亞]]族裔、以及[[亞歐混血|亞歐混血]]文化。新加坡這裡被詮釋為 "東方遇見西方"、"簡單亞洲（Easy Asia）"、與"花園城市"<ref>{{cite web|title=Singapore Cooperation Program|url=http://www.scp.gov.sg/content/scp/about_us/about_singapore.html|deadurl=yes|archiveurl=https://archive.is/20150124165252/http://www.scp.gov.sg/content/scp/about_us/about_singapore.html|archivedate=2015-01-24}}</ref>。

== 概要 ==
[[File:Raffles_Place.jpg|thumb]]的中央廣場]]
在過去幾个世紀裏，[[新加坡|新加坡]]是[[英屬馬來亞|英屬馬來亞]]的一部分。它由[[柔佛苏丹王朝|柔佛苏丹王朝]]统治。1819年，英國人來到[[馬來西亞|馬來西亞]]並設立殖民地。英國統治期間，新加坡吸引了很多外來移民。[[第二次世界大戰|第二次世界大戰]]後，新加坡正式獨立，建立共和國。

新加坡大約有500萬人口<ref>{{cite web |url=http://www.singstat.gov.sg/stats/latestdata.html |title=存档副本 |accessdate=2012-08-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20120905233037/http://www.singstat.gov.sg/stats/latestdata.html |archivedate=2012-09-05 }}</ref>，主要由[[華人|華人]]（[[新加坡華人|新加坡華人]]）、[[馬来人|馬来人]]、[[印度人|印度人]]及[[亞歐混血|欧亚裔人]]所組成，使新加坡成為一個多種族國家。另外，新加坡有42%的人口為外國人，為世界上第六個擁有最多外國人的國家<ref>http://www.un.org/esa/population/publications/migration/UN_MigStock_2008.pdf</ref><ref name="singstat.gov.sg">{{cite web |url=http://www.singstat.gov.sg/pubn/popn/population2009.pdf |title=存档副本 |accessdate=2010-10-13 |deadurl=yes |archiveurl=https://www.webcitation.org/63okaGChL?url=http://www.singstat.gov.sg/pubn/popn/population2009.pdf |archivedate=2011-12-10 }}</ref>。新加坡的[[人口密度|人口密度]]在世界排名第三，僅次於[[澳門|澳門]]及[[摩納哥|摩納哥]]。

鑑於新加坡的移民歷史，新加坡的文化可以說是中華、印度、英國及馬來文化的熔爐。

==語言==
{{See|新加坡的語言|新加坡英語|新加坡式英語|新加坡式汉语}}
大部分的新加坡人為雙語使用者，當中多數都懂得[[英語|英語]]，和[[汉語|汉語]]、[[馬來語|馬來語]]、[[淡米爾語|淡米爾語]]其中一種語言。

英語為新加坡的第一語言，標準形式稱為[[新加坡英語|標準新加坡英語]]（{{lang|en|Standard Singapore English}}），它跟從英式英語的發音及文法。除此之外也發展出稱為[[新加坡式英語|新加坡式英語]]（{{lang|en|Singlish}}）的方言，它與[[馬來西亞式英語|馬來西亞式英語]]（{{lang|en|Manglish}}）相似。

新加坡人以不同的語言作為他們的母語。根據2005年的數據，50%人口在家說汉語，32%說英語，12%說馬來語，3%說泰米爾語，在家不說英語的新加坡人通常會學習英語作為第二語言。

新加坡的國語是馬來語，這是因為馬來人是新加坡的「本地人」，而華人、印度人等族群是後來才移民到新加坡的。馬來語亦被用於國歌、國家格言及軍事口令。而由於大部分新加坡的南亞裔人士是來自[[印度|印度]]及[[斯里蘭卡|斯里蘭卡]]的[[淡米爾人|淡米爾人]]，所以泰米爾語是新加坡的其中一個官方語言。雖然當初移民到新加坡的華人說的是中国南方的[[漢語族|漢族]]語言，但是汉語卻成為了新加坡的官方語言。

==制度與信仰==
[[File:SingaporeFlag-atwindow-20060809.jpg|thumb]]
===精英政治===
新加坡政府認為[[精英政治|精英政治]]是社會的基石，人民會因為自己的個人成就而得到獎賞，而不考慮他們的種族或背景<ref>{{cite web |url=http://www.pmo.gov.sg/News/Transcripts/Minister+Mentor/Old+and+new+citizens+get+equal+chance.htm |title=存档副本 |accessdate=2012-08-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20140318010112/http://www.pmo.gov.sg/News/Transcripts/Minister+Mentor/Old+and+new+citizens+get+equal+chance.htm |archivedate=2014-03-18 }}</ref>。

===不同宗教並存===
新加坡是一個無宗教的移民國家，而民間的主要信仰是[[佛教|佛教]]、[[道教|道教]]、[[基督宗教|基督宗教]]及[[伊斯蘭教|伊斯蘭教]]，政府高度重視不同宗教及個人信仰<ref>{{cite web|url=http://www.singaporeunited.sg/cep/index.php/web/Our-News/PM-Lee-on-racial-and-religious-issues-National-Day-Rally-2009 |title=存档副本 |accessdate=2011-02-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20100220195750/http://www.singaporeunited.sg/cep/index.php/web/Our-News/PM-Lee-on-racial-and-religious-issues-National-Day-Rally-2009 |archivedate=2010-02-20 }}</ref>。

===民主、和平、進步、公正和平等===
[[新加坡國旗|新加坡國旗]]上的五顆星寓意民主、和平、進步、公正和平等。在[[Freedom_of_the_World|Freedom of the World]]2006年的調查中，新加坡的政治自由度獲評5分，公民自由度則獲評4分（7分滿分，1分為'''最自由'''），被評為「部分自由」。

==種族聚居地==
新加坡有不同的種族聚居地，例如[[小印度|小印度]]、[[牛車水|牛車水]]及{{link-en|甘榜格南|Kampong Glam}}。

'''牛車水'''是華人居民的聚居地，也就是新加坡的唐人街，有濃厚的中華文化氣息。它位於{{link-en|歐南（新加坡）|Outram, Singapore}}。

==節日==
{{ain|新fftt加坡的公眾假期}}
新加坡的公眾假期反映了不同族群的文化差異，有華人的[[農曆新年|農曆新年]]，也有[[佛教|佛教]]的[[衛塞節|衛塞節]]（{{lang|en|Vesak}}）﹔有[[伊斯蘭教|伊斯蘭教]]的[[開齋節|開齋節]]（Eid-ul-Fitr，本地所知的[[马来语|马来]]名Hari Raya Puasa），也有[[印度教|印度教]]的[[屠妖節|屠妖節]]（Diwali，本地所知的[[泰米尔语|淡米尔]]名Deepavali）。在新加坡，信仰基督教的人數亦在不斷增長，所以[[聖誕節|聖誕節]]、[[耶穌受難節|耶稣受难日]]（{{lang|en|Good Friday}}）及[[元旦|元旦]]亦被訂為公眾假期。另外，8月9日為[[新加坡国庆日|新加坡国庆日]]，該天會有一連串的慶祝活動，主要為[[新加坡國慶慶典|新加坡國慶慶典]]。

==宗教==
[[File:Sri_Mariamman_Temple_2,_Dec_05.JPG|thumb]]，建於1827年，是新加坡最大的印度教廟宇。]]
新加坡是一個多宗教的國家，這是由於它成為港口後，不同地區的人紛紛移居至此。現時，33%的人口信仰佛教，它亦是移民新加坡的華人的主要信仰。其他華人的信仰包括[[道教|道教]]（11%）及基督宗教，而新加坡有18%的人口是基督徒。此外，大部分的新加坡馬來人是[[穆斯林|穆斯林]]（伊斯蘭教徒），佔15%人口﹔而大部分的印度人是印度教徒，佔5%人口，也有為數不少的印度人是[[穆斯林|穆斯林]]或[[錫克教徒|錫克教徒]]。有17%人口無宗教信仰。

由於這種宗教上的差異，新加坡有大量的道教宮廟、佛教寺院、印度教的廟宇、基督宗教的[[教堂|教堂]]和[[伊斯蘭教|伊斯蘭教]]的[[清真寺|清真寺]]，有些更有非常重要的歷史價值。那裏也有錫克教的廟宇和[[猶太教|猶太教]]的[[猶太會堂|會堂]]。這些建築有時會成為著名的地標。

==菜餚==
[[File:LavenderFoodSquare-Singapore-20050617.jpg|thumb]]的小販中心]]
新加坡的食物也可以很好地反映出新加坡的文化差異。例如在新加坡的[[小販中心|小販中心]]，有些攤檔雖然售賣中式食物，但是卻混用了馬來西亞式的配料。這使[[新加坡菜|新加坡菜]]有很獨特的吸引力。
