'''新疆兵团警官高等专科学校'''位于[[新疆维吾尔自治区|新疆维吾尔自治区]][[五家渠市|五家渠市]]前进东街158号，是[[新疆生产建设兵团监狱管理局|新疆生产建设兵团监狱管理局]]直属单位。

==沿革==
1980年代初，[[新疆生产建设兵团|新疆生产建设兵团]]接收了来自内地的数万名重刑犯、顽危犯、反改造尖子等罪犯。为培养劳改工作干部，根据1984年1月21日《关于设置兵团劳改局和成立兵团劳改干部学校的批复》（新编字〔1984〕24号），正式成立'''新疆生产建设兵团劳改工作干部学校'''。临时校址在乌鲁木齐市头屯河。1985年1月2日，根据《对兵团党委政法委员会〈关于兵团劳改工作干部学校校址、机构、编制问题的请示报告〉的批复》（兵党字〔85〕01号），校址设在五家渠新区。当时该校的任务主要是培训在职干警，每年培训3到4期，300到400人次。教职员工50余人，教师18人<ref name=yg>{{Cite web |url = http://www.xjbtjx.com/show.aspx?Guid=F4CFA8C5-0DA2-465E-83B8-6029964AE892 |title =历史沿革  |author =  |date = 2016-12-30 |publisher = 新疆兵团警官高等专科学校 |language =  |accessdate =  |quote =  }}</ref>。

1987年4月，经[[中华人民共和国农牧渔业部|中华人民共和国农牧渔业部]]和[[新疆维吾尔自治区人民政府|新疆维吾尔自治区人民政府]]批准，该校更名为“'''新疆生产建设兵团劳改工作警察学校'''”。1988年，经新疆广播电视大学批准，该校成立电大工作站，开展大中专学历教育<ref name=yg/>。

1993年5月，经[[中华人民共和国农业部|中华人民共和国农业部]]和新疆维吾尔自治区人民政府批准，增挂“'''公安司法学校'''”牌子，列入国家普通招生计划，开展高中及初中起点的普通中专学历教育<ref name=yg/>。

1996年3月，根据[[中华人民共和国司法部|中华人民共和国司法部]]指示，经新疆生产建设兵团编委批准，该校更名为“'''新疆生产建设兵团警官学校、公安司法学校'''”，同年加挂“'''兵团政法干部培训中心'''”牌子<ref name=yg/>。

1997年6月，该校通过中华人民共和国司法部及新疆生产建设兵团教委对办学条件及质量的两级评估验收，获确定为省部级重点中等专业学校，成为司法部在西北五省（区）唯一的部级重点中等专业学校<ref name=yg/>。

2001年3月，学校党委启动了申报兵团警官高等职业学院的工作。2001年9月13日，由新疆生产建设兵团教委、新疆生产建设兵团监狱管理局联合向[[新疆生产建设兵团办公厅|新疆生产建设兵团办公厅]]呈请。2001年10月10日，新疆生产建设兵团司令员第14次办公会议一致同意向[[中华人民共和国教育部|中华人民共和国教育部]]申办兵团警官高等职业学院。2001年10月15日，新疆生产建设兵团办公厅正式向中华人民共和国教育部呈请。2002年11月，教育部专家评估组评估了学校申办工作，2002年11月8日评估工作获得通过。并且主动提出将申报兵团警官高等职业学院的名称确定为“兵团警官高等专科学校”。2003年4月16日，教育部教发函〔2003〕115号文件批复同意建立'''新疆兵团警官高等专科学校'''<ref name=yg/>。该校是全日制的普通高校<ref>{{Cite web |url =http://www.xjbtjx.com/show.aspx?Guid=DFE8B62D-73A9-4BD2-A1E4-577BFB443CCF  |title = 新疆兵团警官高等专科学校简介 |author =  |date =  |publisher = 新疆兵团警官高等专科学校 |language =  |accessdate =2017-09-26  |quote =  }}</ref>。
