'''新耶路撒冷'''（{{lang-en|New Jerusalem}}）是[[美國|美國]][[加利福尼亞州|加利福尼亞州]][[聖華金縣_(加利福尼亞州)|聖華金縣]]的一個[[非建制地區|非建制地區]]，坐落在[[特雷西_(加利福尼亚州)|特雷西]]東南方，兩地相距{{convert|8|mi|km}}。{{le|新耶路撒冷機場|New Jerusalem Airport}}坐落於此。

== 歷史 ==
新耶路撒冷以1874年成立之新耶路撒冷小學（New Jerusalem Elementary School）為名。當地開拓者亨利·埃布（Henry Ebe）捐贈土地以興建學校。由於亨利·埃布的家族皆屬Dunkard弟兄會（Dunkard Brethren）的成員，故依規定將學校命名為「新耶路撒冷」。<ref>{{cite web |url = http://www.njesd.net/cms/page_view?d=x&piid=&vpid=1363169007684|title = History of NJESD|accessdate = 2016-09-17|date = |publisher = New Jerusalem Elementary School District}}</ref>

== 地理 ==
新耶路撒冷的座標為{{coord|37|40|44|N|121|18|02|W|type:city_region:US-AL_source:GNS-enwiki|display=inline}}，而該地最高點為海拔高度18米（即59英尺）。
