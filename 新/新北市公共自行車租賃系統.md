'''新北市公共自行車租賃系統'''，是一套由[[新北市政府交通局|新北市政府交通局]]規劃、[[捷安特|捷安特公司]]管理的[[公共自行車|公共自行車]]系統。起初分為NewBike系統與Youbike兩套系統並行，NewBike系統營運至2015年4月止。前身為2008年10月底[[臺北縣政府|臺北縣政府]]試辦的「臺北縣公共自行車租借系統」，採用人工租借方式經營，僅於[[板橋區_(新北市)|板橋區]]設置租借站。2013年9月2日，NewBike系統開始試辦自動化租借服務，成為[[臺灣|臺灣]]第三座啟用的公共自行車自動化租賃系統。

由於NewBike系統無法與[[臺北市|臺北市]]所採用的[[YouBike微笑單車|YouBike微笑單車]]系統整合，[[新北市政府|新北市政府]]於2014年改採用[[捷安特|捷安特]]設計的YouBike微笑單車系統建構新的市區公共自行車系統，原有NewBike系統自行車則轉移至[[貢寮區|貢寮區]]做為[[低碳旅遊|低碳旅遊]]設施。

== 歷史 ==
NewBike系統前身為臺北縣政府設置的臺北縣公共自行車租借系統，於2008年10月31日在[[板橋區_(台灣)|板橋區]]開始試辦，試辦之初僅有設有5個租借站與100輛自行車<ref>孟祥傑，公共單車租借 板橋起跑 新板特區設5處 百輛供借用 甲地借可乙地還 年底前免費，聯合報，2008年11月1日C2版（北基綜合新聞）</ref>，2009年7月2日，擴增10個租借站。<ref>沈旭凱，免費借單車 增10個點，聯合報，2009年7月3日B1版（北縣基隆．運動）</ref>臺北縣公共自行車租借系統，只在周一到周五、每天上午7時到晚上8時提供租借；民眾需先上網登錄成為會員，憑會員卡租借。非會員者需持雙證件（[[健保卡|健保卡]]及[[國民身分證|國民身分證]]、[[駕照|駕照]]等），二小時內均免費；逾時未歸還者，第一次罰一星期內不得再租借，第二次一個月內不得再租車。

2013年9月2日，NewBike系統開始試營運自動化租借服務，並全面更新280輛自行車，前半小時免費，之後每半小時收費10元，需以悠遊卡付費<ref>祁容玉，下月新制上路 新北New Bike 改悠遊卡收費，聯合報，2013年8月21日B1版（新北市．運動）</ref>。

2013年12月1日，[[新北市市長|新北市市長]][[朱立倫|朱立倫]]宣布，新北市公共自行車系統與[[臺北市|臺北市]]所使用[[臺北市公共自行車租賃系統|YouBike微笑單車系統]]整合，新的公共自行車系統要能與臺北市所使用的系統相容。2014年初，YouBike微笑單車系統於[[新店區|新店區]]大鵬華城社區開始試辦，成為新北市公共自行車租賃系統使用YouBike微笑單車系統的首個租賃站。2014年7月30日，[[捷安特公司|捷安特公司]]標下新的新北市公共自行車系統標案，確定之後的新北市公共自行車系統都採用捷安特公司設計的YouBike微笑單車系統<ref>{{Cite web|url = http://g.udn.com/NEWS/LIFE/LIF1/8839155.shtml|title = 建置雙北公共自行車 捷安特得標|accessdate = 2014-07-31}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2014年，新北市政府辦理「自動化公共自行車租借系統建置規劃計畫」委託研究案，其中包括新北市公共自行車整體規劃、財務分析、推動期程及兩市整合方案等，預計2014年中完成整體規劃，後續將依規劃內容分年期推動，初步規劃2014年於[[新店區|新店]]、[[汐止區|汐止]]、[[三重區|三重]]、[[蘆洲區|蘆洲]]、[[新莊區|新莊]]及[[永和區|永和]]等地設置，目標先設置60站共1600輛自行車，最終計畫全市設置300站8000輛自行車。<ref>{{Cite web |url=http://travel.ettoday.net/article/288976.htm |title=U-Bike熱朝襲全台　新北增新站屏東明年有|accessdate=2013-12-03 |work= |publisher=東森旅遊雲 |date=2013-10-31 }}</ref>

2015年3月，新北市政府宣布，在板橋區原有的NewBike系統13個租賃站於3月底至4月初，分三階段終止服務，並於4月底轉換為YouBike系統。NewBike自2013年9月共計1年半，正式走入歷史。未來在板橋區捷運府中站、板橋站等據點也將陸續增設YouBike站點，四月底前板橋合計啟用十六個YouBike站、共四百台車<ref>{{Cite web|url = http://news.ltn.com.tw/news/local/paper/806474|title = YouBike進駐板橋 最快明年初設站|accessdate = 2014-08-21|work= |publisher=自由時報}}</ref>。

2015年4月，NewBike系統轉為YouBike系統之作業完成，日後板橋區同新北市其他區增設YouBike站點。

[[新店區|新店區]]大鵬華城社區於2014年初試辦YouBike租賃站。<ref>{{Cite web |url=http://www.ettoday.net/news/20131202/303025.htm |title=新北市「NewBike」走入歷史　年底前與Ubike二合一|accessdate=2013-12-03 |work= |publisher=東森新聞雲 |date=2013-12-02 }}</ref><ref>{{Cite web |url=http://www.libertytimes.com.tw/2014/new/jan/2/today-taipei6.htm |title=新北YouBike首站 落腳大鵬華城|accessdate=2014-01-02 |work= |publisher=自由時報 |date=2014-01-02 }}</ref><ref>{{Cite web |url=http://www.ntpc.gov.tw/ch/home.jsp?id=28&parentpath=0,6,27&mcustomize=multimessages_view.jsp&dataserno=294471&mserno=201309100001 |title=新北YouBike首站大鵬華城設置完成|accessdate=2013-12-31 |work= |publisher=新北市政府交通局 |date=2013-01-02 }}</ref>原先設置的New Bike系統在[[板橋區_(台灣)|板橋區]]設置的13個租賃站點，已經全數終止服務，並轉換為YouBike微笑單車系統。新北市的YouBike微笑單車系統，除車身印有新北市政府府徽，以標明來自新北市外，其餘與臺北市YouBike相同。

== 租賃方式 ==
YouBike微笑單車採用[[二十四小時制|24小時]]開放的無人管理[[自助结帐系统|自助式服務]]，每個站點設有自動服務機（Kiosk），使用者可利用服務機申請加入會員、付費和租賃單車。

=== 會員申請 ===
租賃模式分為會員與非會員兩種，會員申請可在YouBike微笑單車網站、自動服務機，或服務中心完成。租賃站點的每個停車柱上都裝有多卡通感應設備，「會員」可直接持已經註冊的[[悠游卡|悠游卡]]或[[一卡通_(台灣)|一卡通]]到各停車柱感應卡片後取車；「非會員」則須透過自動服務機付款後選車（付款可使用[[Visa|Visa]]/[[萬事達卡|Master]]/[[JCB|JCB]]晶片[[信用卡|信用卡]]），並在90秒內到停車柱處取出自行車。<ref name="如何租車">{{cite web |url=http://www.youbike.com.tw/cht/f42.html |title=YouBike.com.tw 如何租車 |accessdate=2013-06-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20140420071616/http://www.youbike.com.tw/cht/f42.html |archivedate=2014-04-20 }}</ref> <ref>{{cite web|url= http://www.cw.com.tw/article/article.action?id=5054467 |title= 一秒借還YouBike　靠30人金頭腦  | date= 2013-12-10 | publisher = [[天下雜誌|天下雜誌]] | work = 537期 | author=  劉光瑩 | coauthors =林昭儀 | quote = 為何改90秒 }}</ref>

=== 費用 ===
YouBike微笑單車為每30分鐘[[新台幣|新台幣]]10元，前30分鐘免費，前4小時每30分鐘10元，時間4至8小時內，調整為每30分鐘20元，使用時間超過8小時，則每30分鐘40元。<ref name="費率" />自2015年4月1日開始，YouBike微笑單車採用「累進費率、前半小時收錢」，於台北市內租借站租借者前半小時收5元，4小時內仍維持30分鐘10元，使用時間4至8小時內，調整為每30分鐘20元，使用時間超過8小時，則每30分鐘40元。<ref name="費率" />還車後15分鐘內，不得於同站（還車站）續借。<ref name="費率">{{cite web|url= http://ntpc.youbike.com.tw/cht/f43.html  |title=YouBike.com.tw 費率說明 |accessdate=2014-08-06}}</ref>
