'''新都区'''是[[中华人民共和国|中国]][[四川省|四川省]][[成都市|成都市]]的一个[[市辖区|市辖区]]，位于成都市北部，距离成都市中心[[天府广场|天府广场]]约20公里，是成都的八大[[卫星城|卫星城]]之一、城北副中心，城区已与成都五城区、[[青白江区|青白江区]]、[[广汉市|广汉市]]连成一片，是成都重要的商贸集聚区及高端装备制造业基地<ref>{{cite web |title=未来五年 成都八个卫星城将“独立成市” |url=http://www.sc.gov.cn/10462/12771/2015/11/8/10358076.shtml |publisher=华西都市报 |accessdate=2018-07-25 |date=2015-11-08}}</ref>。2017年，被纳入成都中心城区<ref>{{cite news |title=成都中心城区扩围：将囊括郫都、新都、龙泉驿、双流、温江等 |url=http://news.163.com/17/0702/17/COBTS37R000187VE.html |accessdate=2018-07-25 |work=澎湃新闻网 |date=2017-07-02}}</ref>。

== 历史沿革 ==
新都为[[古蜀国|古蜀国]]都邑，于公元前七世纪左右由蜀王[[开明_(蜀国)|开明氏]]称帝后所建，为有别于[[杜宇氏|杜宇氏]]的旧都郫邑，得名“新都”。<ref name="建置">{{cite web |title=行政区划 |url=http://www.xindu.gov.cn/xdq/xzjz/2016-01/01/content_9111d2b9ae29481698c818599b763701.shtml |publisher=新都区人民政府 |accessdate=2018-07-25}}</ref>

公元前221年，[[秦朝|秦]]置新都县，隶[[蜀郡|蜀郡]]，后隶[[廣漢郡|广汉郡]]（子同郡）。[[西晋|西晋]][[泰始_(西晋)|泰始]]二年（266年），设[[新都郡_(西晋)|新都郡]]，属[[梁州_(曹魏)|梁州]]，领新都等四县。[[刘宋|宋]]、[[南齐|齐]]时属益州广汉郡，[[梁_(南朝)|梁]]属始康郡。[[西魏|西魏]][[元钦|废帝]]二年（533年）改隶蜀郡，[[隋朝|隋]][[开皇|开皇]]二年（598年）改为兴乐县，随即并入[[成都縣|成都县]]。[[唐朝|唐]][[武德|武德]]二年（619年）复置、复名为新都县，隶[[剑南道|剑南道]][[益州|益州]]（后更名为[[蜀郡|蜀郡]]、[[成都府|成都府]]）。[[宋朝|宋]]属[[四川路|四川路]][[成都府路|成都府路]]成都府，元属[[四川等处行中书省|四川等处行中书省]][[成都路|成都路]]，明属[[四川等處承宣布政使司|四川布政使司]]成都路，清属[[四川省_(清)|四川省]][[成绵龙茂道|成绵龙茂道]]成都府，[[中華民國_(大陸時期)|民国]]初年属[[四川省_(中華民國)|四川省]][[西川道|西川道]]。<ref name="建置"/><ref name=":1">陈习删等. 新都县志. 1928（中华民国17年）.</ref>

1950年，新都县隶属[[川西行政区|川西行署]][[温江专区|温江专区]]。1953年，改隶四川省温江专区。1960年2月，撤销新都县并入[[新繁县|新繁县]]。1962年10月，恢复新都县。1965年7月，撤销新繁县并入新都县。1983年，温江地区撤销，改隶[[成都市|成都市]]。2001年11月，国务院批复同意{{Notetag|国函〔2001〕149号}}撤销新都县，设立成都市新都区，以原新都县的行政区域为新都区的行政区域，区人民政府驻[[新都镇|新都镇]]（今新都街道），2002年1月1日正式挂牌成立。<ref>{{Cite web|url=http://www.gov.cn/gongbao/content/2001/content_61206.htm|title=国务院关于同意四川省撤销新都县设立成都市新都区的批复|accessdate=2018-07-25|author=[[中华人民共和国国务院|中华人民共和国国务院]]|date=|last=|work=www.gov.cn|publisher=[[中国政府网|中国政府网]]}}</ref><ref name="概况" /><ref>{{cite web |title=新都区历史沿革_行政区划网(区划地名网) |url=http://www.xzqh.org/html/show/sc/18216.html |website=www.xzqh.org |accessdate=2018-07-25}}</ref><ref>{{cite book |author1=陈潮 |coauthors=陈洪玲 |title=中华人民共和国行政区划沿革地图集 |date=2003.10 |publisher=中国地图出版社 |location=北京 |isbn=978-7-5031-3184-4 |page=124-126,211 |edition=第一版}}</ref>

== 自然地理 ==
新都区地处[[四川盆地|四川盆地]][[成都平原|川西平原]]腹地、[[都江堰|都江堰]]自流灌区，土地肥沃，气候温和，物产丰富，历史上被誉为“蜀中宝玉”、“天府明珠”，是四川著名的粮库之一，是全国著名的商品粮油生产基地，素有“天府粮仓”之称。区内北有[[清白江|清白江]]，南有毗河，中有锦水河分干渠，水源充足。<ref name=":0">{{Cite web|url=http://www.xindu.gov.cn/xdq/zrdl/2016-01/01/content_710e427f1f414a38910855bdad14a15a.shtml|title=自然地理|accessdate=2018-07-26|author=|date=|publisher=新都区人民政府}}</ref>

新都区属于中亚热带湿润季风气候区，气候温和，雨量充足，温湿同季，水热同步，四季分明，年均气温16.1℃，年均降水量为828.3毫米，年均日照时数为1214.9小时。<ref name=":0" />

== 经济 ==
新都区位优越，是[[成德绵高新技术产业带|成德绵高新技术产业带]]重要节点城市，位于成都城市发展北中轴线，是“蓉欧快铁”和“南丝绸之路”的起点城市<ref name="概况">{{cite web |title=新都区概况 |url=http://www.xindu.gov.cn/xdq/xdgl/2016-06/07/content_c951a26cd3ac48b09c0dfa5f43cd7cab.shtml |publisher=新都区人民政府 |accessdate=2018-07-25}}</ref>。

2017年新都区[[地区生产总值|地区生产总值]]达到722.67亿元，同比增长8.9%，其中[[第一产业|第一产业]]、[[第二产业|第二产业]]、[[第三产业|第三产业]]占比为3.9：59.5：36.6<ref>{{Cite web|url=http://www.xindu.gov.cn/xdq/zwgk/gkxx_cont.shtml?id=1913180&tn=2|title=2017年新都区GDP突破700亿元|accessdate=2018-07-25|author=|date=|publisher=新都区人民政府}}</ref>。实现社会消费品零售总额176.01亿元，同比增长14.6%<ref>{{Cite web|url=http://www.xindu.gov.cn/xdq/zwgk/gkxx_cont.shtml?id=1872006&tn=2|title=新都区2017年社会消费品零售总额增长14.6%|accessdate=2018-07-26|author=|date=|publisher=新都区人民政府}}</ref>。
