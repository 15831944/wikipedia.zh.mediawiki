《'''新快報'''》（国内统一刊号：CN44-0180）是一份[[中华人民共和国|中华人民共和国]][[廣州市|廣州市]][[羊城晚報報業集團|羊城晚報報業集團]]旗下的綜合性[[日報|日報]]，1998年3月30日創刊。现在由[[侨鑫集团|侨鑫集团]]实际运营。由羊城晚报报业集团党委书记、管委会主任刘红兵兼任新快报社社长；由集团管委会委员、羊城晚报社社委孙璇兼任新快报社总编辑總編輯<ref name="bj">[http://news.xinhuanet.com/politics/2013-11/01/c_117973597.htm 《新快报》李宜航不再兼任社长总编辑]</ref>。

《新快報》是中华人民共和国第一份全彩印刷的綜合性日報，在廣東省直至中國大陸境內較具影響力與知名度。《新快報》亦是廣州市三大日報之一。2004年的第57屆世界報業大會公佈的“[[世界百大报纸列表|世界日報發行量前100名名單]]”中《新快報》位居第99位<ref>[http://www.people.com.cn/GB/14677/21966/2546253.html “世界日报发行量前100名”中国20强]</ref>。2005年《新快報》第95位<ref>[http://news.sohu.com/20050601/n225778434.shtml 2005年世界日报发行量前100名排行榜]</ref>。2006年未獲得上榜。

== 发行区域 ==
《新快报》在[[珠江三角洲|珠江三角洲]]九个城市的中心区域报刊摊都有零售，但在乡镇农村则难觅其身影。

== 争议 ==
新快报部分报道存在不符事實與偏頗新聞事件主因的文章，受到讀者質疑。《新快報》在“公民新聞”風潮漸起的中國大陸，較推崇“爆料”。在報紙和其官方網站均有“爆料”聯繫方式提供給普通讀者與民眾。

=== 编造关于“[[韩国起源论|韩国起源论]]”的假新闻 ===
自2007年到2008年《新快报》善于利用中国人的民族情绪制造假新闻，编造过韩国有人主张[[风水|风水]]、[[汉字|汉字]]、[[活字印刷|活字印刷]]等申请世界文化遗产的假新闻<ref>[http://zqb.cyol.com/content/2007-12/22/content_2005911.htm 是谁在臆造中韩文化之争][[中国青年报|中国青年报]]，2007-12-22，2008-08-24造访。</ref>
和“韩国《朝鲜日报》报道[[孙中山|孙中山]]是韩国人”的假新闻，引起不少中国人的愤怒，结果证实这些都是编造的<ref>[http://www.xys.org/xys/netters/Fang-Zhouzi/blog/xinkuaibao2.txt 《新快报》如此澄清“孙中山是韩国人”假新闻][[新语丝|新语丝]]网站，2008-08-24造访。</ref>。新快报并没有对此进行道歉。

== 澳洲版 ==
《澳洲新快報》是由羊城晚報報業集團與僑鑫集團有限公司於2004年6月30日在[[澳大利亞|澳大利亞]][[悉尼|悉尼]]出版發行的一份綜合性華文日報。由僑鑫集團董事長周澤榮創辦。最初由僑鑫集團出資收購澳大利亞《華人日報》，後更名爲《澳洲新快報》。報紙新聞內容以悉尼编辑部采编、翻译为主，部分由《新快報》提供。

《澳洲新快報》每周出版6天，其中週一到週五爲4開56版，週六為76版，出版后一直使用[[繁體中文|繁體中文]]印刷，直至2011年2月28日起轉用[[簡體中文|簡體]]<ref>{{cite web |title=本报赞助新年庆典回馈读者 |publisher=澳洲新快网 |date=2011-01-30 |accessdate=2011-02-11 |url=http://www.xkb.com.au/html/news/shehui/2011/0130/51584.html |archive-url=https://web.archive.org/web/20110204232152/http://www.xkb.com.au/html/news/shehui/2011/0130/51584.html |archive-date=2011-02-04 |dead-url=yes }}</ref><ref>{{cite web |title=2011年2月28日澳洲新快報部分版面 |publisher=澳洲新快网 |date=2011-02-27 |accessdate=2011-02-28 |url=http://www.xkb.com.au/html/shouyenarongguanli/dianzikanwu/xinkuaibao/2011/0227/53144.html |archive-url=https://web.archive.org/web/20110306085059/http://xkb.com.au/html/shouyenarongguanli/dianzikanwu/xinkuaibao/2011/0227/53144.html |archive-date=2011年3月6日 |dead-url=yes }}</ref>。
