'''新加坡印度人'''（也被稱為'''新加坡印度裔'''；{{lang-ta|'''சிங்கப்பூர் இந்தியர்கள்'''}}, ''Ciṅkappūr Intiyarkaḷ''）是對生活在[[新加坡|新加坡]]的[[南亞|南亞裔]]的[[稱呼|稱呼]]。印度裔社群佔新加坡人口的7.4％，使印度裔成為新加坡第三大族群。雖然古[[印度文化|印度文化]]對新加坡本土的[[馬來|馬來]]文化有了深刻影響，但印度裔社群真正開始移居到新加坡的時間在[[現代新加坡的建立|現代新加坡的建立]]時期（1819年），當時移居新加坡的印度人包括契約工人，士兵和罪犯。到二十世紀中期，又有一批印度裔移居新加坡。新加坡的印度人社群貧富差距大，自1990年代以來，受過良好高等教育的印度移民移入，使新的矛盾形成。新加坡印度裔社群在語言和宗教上多樣化，多數為來自南印度的坦米爾人。新加坡當地的印度文化已經深耕了近二百年。至1990年代，隨著印度文化在更廣泛在新加坡文化中擴散，已經發展和當代的印度文化有所不同。來自印度新移民增加了當地印度裔社群的規模和多樣性。新加坡印度裔社群透過往返[[印度|印度]]的[[低成本航空|低成本航空]]及影視傳媒與母國[[印度|印度]]連繫在一起。

== 定義 ==
{{main article|{{link-en|新加坡種族|Race in Singapore}}}}
新加坡政府把“新加坡印度人”定義為“ 種族 ”（或“ 族群 ”），包括來自“ 印度，巴基斯坦，孟加拉和斯里蘭卡的[[南亞|南亞]]族群如人[[淡米爾人|淡米爾人]]，[[馬拉亞利人|馬拉亞利人]]，[[旁遮普人|旁遮普人]]，[[孟加拉人|孟加拉人]]，[[僧伽羅人|僧伽羅人]]等。 ”<ref>{{Cite web|url=http://www.singstat.gov.sg/docs/default-source/default-document-library/publications/publications_and_papers/cop2010/census_2010_release1/cop2010sr1.pdf|title=Census of Population 2010 Statistical Release 1: Demographic Characteristics, Education, Language and Religion|last=|first=|date=2010|website=Department of Statistics, Singapore|publisher=|page=185|access-date=|deadurl=yes|archiveurl=https://web.archive.org/web/20170516214219/http://www.singstat.gov.sg/docs/default-source/default-document-library/publications/publications_and_papers/cop2010/census_2010_release1/cop2010sr1.pdf|archivedate=2017年5月16日|df=}}</ref><ref>Leow, p.16.</ref>

== 語言群 ==
{{main article|{{link-en|新加坡印度語言|Indian languages in Singapore}}}}
新加坡印度人（南亞裔）大多數來自[[南印度|南印度]]和[[斯里蘭卡|斯里蘭卡]]，其餘來自[[北印度|北印度]]和[[西印度|西印度]]。通常是19世紀至20世紀初期來自印度次大陸的契約勞工移民的後裔。2010年，新加坡人口普查將237,473名新加坡公民和110,646名永久居民分為多個“ 語言群 ”。

新加坡的印度人（南亞裔）的大多數為[[坦米爾人|坦米爾人]]（54.18％）和各個不同的語言群體。來自印度南部[[喀拉拉邦|喀拉拉邦]]的[[馬拉雅拉姆人|馬拉雅拉姆人]]為人數第二大多的語言群體，佔新加坡的南亞裔（印度裔）的7.57％。[[坦米爾人|坦米爾人]]和[[馬拉雅拉姆人|馬拉雅拉姆人]]是新加坡裔（南亞裔）的兩個主要的語言群體，佔新加坡印度人（南亞裔）人口的三分之二。新加坡三大北印度語言群體（[[旁遮普邦|旁遮普邦]]、[[古吉拉特邦|古吉拉特邦]]、[[信德省|信德省]]）佔南亞裔（印度裔）的7.67％。其餘的29.68％來自[[南印度|南印度]]（如[[泰盧固人|泰盧固人]]），[[東印度|東印度]]（如[[孟加拉人|孟加拉人]]）和[[北印度|北印度]]（[[印度斯坦人|印度斯坦人]]）。

===語言===
{{main article|新加坡語言|{{link-en|新加坡印度語言|Indian languages in Singapore}}}}
[[File:DangerKeepOut.jpg|thumb]]：[[英語|英語]]、[[華語|華語]]（[[漢語普通話|漢語普通話]]）、[[坦米爾語|坦米爾語]]和[[馬來語|馬來語]]。]]
新加坡的官方語言為[[英語|英語]]、[[馬來語|馬來語]]、[[華語|華語]]（[[漢語普通話|漢語普通話]]）和[[坦米爾語|坦米爾語]]。由於新加坡的原住民為[[新加坡馬來人|馬來人]]，新加坡的國歌《前進吧，新加坡！》（[[馬來語|馬來語]]：Majulah Singapura）是以[[馬來語|馬來語]]來演唱而非其他官方語言。[[坦米爾人|坦米爾人]]約佔了新加坡印度裔（南亞裔）社群的58％。其他印度裔語言社群包括[[馬拉雅拉姆人|馬拉雅拉姆人]]、[[泰盧固人|泰盧固人]]、[[旁遮普人|旁遮普人]]、[[信德人|信德人]]、{{link-en|古吉拉特人|Gujarati}}、[[僧伽羅人|僧伽羅人]]和[[印度斯坦人|印度斯坦人]]等。
新加坡印度裔（南亞裔）社群日常語言使用方面，39％的印度裔（南亞裔）在家裡主要使用英語和[[坦米爾語|坦米爾語]]。其餘使用的語言包括[[馬來語|馬來語]]（11%）或其他語言，包括其他[[印度語言|印度語言]]和克里斯坦語（11％），使用[[馬拉地語|馬拉地語]]的印度裔（南亞裔）家庭約有3000個。新加坡全國公民中，3.1%的人主要在家中使用[[坦米爾語|坦米爾語]]，是新加坡的四種官方語言之一。在大多數公立學校，坦米爾語被作為教授的第二語言。[[坦米爾語|坦米爾語]]影視產品（本地或國外製作）可在免費電視和有線電視以及廣播頻道以及圖書館、電影院、劇院和書店購買。[[坦米爾語|坦米爾語]]也用於印度教寺廟、清真寺和教堂。在一些商業和非營利組織中，特別是[[小印度|小印度區]]的企業和非營利組織都使用[[坦米爾語|坦米爾語]]。

[[File:Thaipusam8.jpg|thumb]]的遊行參與者]]
在2010年統計中，新加坡15歲以上的公民有5.1％為印度教徒。幾乎所有新加坡印度教徒都是印度裔。新加坡印度裔社群有58.9％信奉印度教。新加坡約有35座印度教寺廟，大多數都是南印度的建築風格。雖然印度教寺廟可能在歷史上與印度社群有聯繫，但新加坡的印度教寺廟為開放的，無論是種族或宗教信仰（非印度教徒可以參觀印度教寺廟）。新加坡最受歡迎的印度教節日是[[排燈節|排燈節]]和[[大寶森節|大寶森節]]，[[排燈節|排燈節]]為新加坡的全國公共假期。在[[排燈節|排燈節]]一個月前，小印度區以燈飾來裝飾。市場在小印度區內不同地區舉行，攤販販售宗教用品，節日賀卡，節日食品，節日裝飾品，衣服等。在[[排燈節|排燈節]]當天信徒會沐浴和祈禱，並參加節日活動和拜訪親戚朋友。[[大寶森節|大寶森節]]為[[新加坡|新加坡]]和[[馬來西亞|馬來西亞]]的印度裔淡米爾人社群中的重要節日。作為以懺悔為主的節日，參佳節日遊行的信徒會攜帶 kavadis，有些信徒通過刺穿皮膚的鉤子和矛來攜帶 kavadis附著在身上。在[[大寶森節|大寶森節]]節慶活動期間，新加坡主要道路被封閉以進行宗教遊行。

====其他宗教====
[[File:Central_Sikh_Temple.JPG|thumb]]
新加坡錫克教徒佔新加坡15歲以上公民的0.4％。如他宗教一樣，錫克教徒在新加坡興建了幾個宗教場所，其中最古老的錫克教寺廟是成立於1912年的中央錫克教寺廟。在新加坡穆斯林中，印度裔佔了12.4％，其餘大多數是馬來人；印度裔穆斯林常與馬來人通婚。

印度尼西亞人和馬來人透過印度商人而非阿拉伯傳教士了解伊斯蘭教。在新加坡基督徒中，6％是印度裔，其餘大多數是華人。在印度裔基督徒社群中有10.3％信仰羅馬天主教（包括部分印度裔[[克里斯坦人|克里斯坦人]]）和3.9％非天主教基督徒（主要是[[基督新教|新教徒]]）。

印度裔穆斯林和基督徒社群內，有自己專數的宗教場所，在講道、服務和禱告用印度語言來進行。新加坡印度社群有自己的基督教會，清真寺和佛教寺廟；如：建於1826年的[[詹美回教堂|詹美回教堂]]，是新加坡最古老的淡米爾清真寺和國定古蹟。在1888年，淡米爾天主教信徒建立了[[露德聖母堂_(新加坡)|露德聖母堂]]，為新加坡最古老的淡米爾天主教堂，同時也是新加坡國定古蹟。

[[釋迦牟尼菩提迦耶寺|釋迦牟尼菩提迦耶寺]]是一座以中國，印度，斯里蘭卡和泰國風格混合建立的[[上座部佛教|上座部佛教]]寺廟，與大多數新加坡的中國漢傳佛教寺廟完全不同。新加坡[[小印度|小印度]]佛教社群（主要是來自斯里蘭卡的[[僧伽羅人|僧伽羅人]]）常常到釋迦牟尼菩提迦耶寺禮佛。

== 料理 ==
{{main article|{{link-en|新加坡印度料理|Indian cuisine in Singapore}}}}
[[File:Indian_rojak.JPG|thumb]]]]
新加坡料理種類多樣，包括印度料理，大部分是南印度{{link-en|泰米爾料理|Tamil cuisine}}，特別是當地的[[坦米爾人|淡米爾]][[穆斯林|穆斯林]]美食，而北印度料理也很受歡迎。經過多年與其他新加坡文化的接觸，受其他新加坡料理影響，新加坡印度料理已經不同程度地改變了。有些印度食品不太普遍，但在幾個地區仍然可以找到販售的餐館，特別是在[[小印度|小印度]]及其周邊地區。

== 小印度 ==
[[File:Deepavali_22,_Little_India,_Singapore,_Oct_06.JPG|thumb]]期間的[[小印度|小印度]]]]
新加坡有印度文化的地區以[[小印度|小印度]]區最為顯著。1989年7月7日，在新加坡文化資產建築保護開始時，該地區已成為保護區域，使其建築物受法律保護而免受擅自更改及拆卸。該地區被公認為新加坡印度社群的樞紐。主要包含早期、過渡時期和晚期的裝飾藝術的兩層商店，以及不同信仰的禮拜場所。<ref>{{cite web | publisher = [[Urban_Redevelopment_Authority|Urban Redevelopment Authority]] | url = http://www.ura.gov.sg/conservation/india.htm | title = Little India Conservation Area}}</ref> 除了為社會服務，印度社區的文化和商業需求，該地區被確立為新加坡歷史悠久的區域之一，擁有豐富的移民文化，使其成為國家文化認同的不可分割的一部分。在2004年，小印度區已成為新加坡第三大受歡迎的旅遊景點。<ref>{{cite web | publisher = [[Singapore_Tourism_Board|Singapore Tourism Board]] | url = http://www.stb.com.sg/doc/sarts2004.pdf |format=PDF| title = Annual Report on Tourism Statistics 2004}}</ref>
