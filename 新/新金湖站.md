'''新金湖站'''（{{kor|諺=신금호역|漢=新金湖驛|羅=Singeumho yeok|f=y}}）位於[[韓國|韓國]][[首爾特別市|首爾特別市]][[城東區_(首爾)|城東區]]金湖洞2街，是[[首爾地鐵5號線|首爾地鐵5號線]]的[[電鐵站|電鐵站]]。地形上位於地下8層（約42.12公尺），深度僅次於[[釜山都市鐵道3號線|釜山都市鐵道3號線]][[萬德站_(釜山)|萬德站]]與[[盃山站|盃山站]]。站名來自鄰近的[[首爾地鐵3號線|首爾地鐵3號線]][[金湖站_(首爾)|金湖站]]，故站名確定為「新金湖站」。另外此站起至[[忠正路站|忠正路站]]為止是連續的島式月台。

== 车站结构 ==
開通當時月台位於洞窟型人工岩石內部，但是2005年1月3日[[鐵山站|鐵山站]]放火事件以後全部拆除。

1面2線的曲線[[島式月台|島式月台]]的地下車站，月台邊設有[[月台幕門|月台幕門]]。此站是曲線月台，月台與列車之間的間隙較大，上下車時需注意。

== 车站周边 ==
位于[[鹰风山|鹰风山]]东侧的住宅区，一带街道既窄又起伏不平。

== 历史 ==
*1996年12月30日 - 本站随着5号线通车而落成。
==其他==
本站称为新金湖站是因为要跟[[金湖站_(首爾)|金湖站]]作出区别。
