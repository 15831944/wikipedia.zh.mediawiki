'''新竹長和宮'''位於[[臺灣|臺灣]][[新竹市|新竹市]][[北區_(新竹市)|北區]]，主祀[[海神|海神]][[媽祖|媽祖]]是湄洲媽祖廟三份身上遺物中現碩果僅存下來的一份，更是新竹知名的[[媽祖廟|媽祖廟]]。由於位在[[竹塹城|竹塹城]]北門外，所以又稱'''外媽祖廟'''或'''外天后宮'''，以和城內的'''[[內天后宮|內天后宮]]'''做區分{{r|竹}}。該廟於1985年8月19日公告為古蹟<ref>{{cite web |url=https://nchdb.boch.gov.tw/assets/advanceSearch/monument/19850819000002 |title=新竹長和宮 |publisher=文化部文化資產局 |accessdate=}}</ref>。本廟與[[新竹城隍廟|城隍廟]]、[[新竹竹蓮寺|觀音亭]]並稱'''新竹三大廟'''。

== 沿革 ==
長和宮的興建肇始於1742年，當時竹塹市街發展起來，許多郊商便在長和宮為議事地點，所以取名長和宮是有船頭行長久合作，和平共處的意思。[[淡水同知|淡水同知]][[莊年|莊年]]與[[守備|守備]][[陳士挺|陳士挺]]的提倡，並有賴郊商們的支持，而1819年時的重修，亦是由新竹的[[郊商|郊商]]們贊助[[銀兩|銀兩]]為[[香火錢|香火錢]]。1828年[[郊商|郊商]]組織「[[金長和|金長和]]」成立後，選擇將長和宮作為議事場所以及塹郊的會館{{r|竹}}。

== 建築特色 ==
長和宮由[[三川殿|三川殿]]、[[正殿|正殿]]與後殿所組成，其中三川殿的牆堵石雕仍為清朝原件{{r|竹}}。
三川殿正殿之木結構有[[對場作|對場作]]，不過不明顯。前殿石雕有對場作，不過不明顯。是臺灣現存少數同時具有石雕對場作及木結構場作之建築。可能是台灣現存最早之對場作建築。

== 文物 ==
[[File:新竹長和宮「海邦赫濯」匾.jpg|thumb]]]]
*紅紋香爐、大明年間七寶銅爐、溫潤圓融，脈絡有情，聖母來臺乘座鑾椅，色澤斑駁，檀香暗沈，天后[[鳳冠霞被|鳳冠霞被]]，錦袍繡裙，三寸金蓮履，繡花荷包囊，琳瑯耀目，妙肖珍奇，乃由湄洲祖廟來臺所穿。
*右殿牆外，列排古碑三座，各為[[光緒|光緒]]十三年（1887年）「瀨江祀碑」及[[道光|道光]]十五年（1835年）、[[同治|同治]]五年（1866年）所立者，碑文已湮漫不識。
*[[嘉慶|嘉慶]]年間之「德可配天」、「霖雨蒼生」。
*[[道光|道光]]年間之「海邦赫濯」、「后來其蘇」、「海邦砥柱」。
*[[同治|同治]]年間之「泛舟利濟」。
*「萬世永賴」、「盛德在水」、「續著平成」等多方匾額，昭然明禾，引人懷想。
*[[兵部尚書|兵部尚書]]、[[太子少保|太子少保]]、一等[[輕車都尉|輕車都尉]]、[[雲貴總督|雲貴總督]][[岑毓英|岑毓英]]所贈的匾額「與天同功」。
*[[兵部侍郎|兵部侍郎]]、[[都察院|都察院]]右副都御史、福建巡撫[[徐宗幹|徐宗幹]]所贈的匾額「泛舟利濟」。

== 特色與傳說 ==
[[File:新竹長和宮（左）與水仙宮（右）.jpg|thumb]]（右）]]
新竹市長和宮供奉的[[媽祖|媽祖]]神像，不僅是270多年前從[[湄洲|湄洲]]來臺，頭上所鑲頭髮，相傳即為[[媽祖|媽祖]]遺世的真跡。

大門對聯( 代表建廟時與[[湄洲天后宮|湄洲天后宮]]祖廟的特殊情感 )
*左：長發婆心恩垂竹塹 
*右：和舒母德祥溯湄洲

[[桃竹苗|桃竹苗]]等地區最為悠久的媽祖廟、也是內政部[[三級古蹟|三級古蹟]]長和宮，廟內主祀天上聖母，為清乾隆七年時竹塹水郊會與[[福建|福建]][[莆田|莆田]]萬梅堂禪師「義濤上人」供奉，由[[湄洲天后宮|湄洲天后宮]]祖廟迎請至竹塹長和宮。

此媽祖神像特別之處在於「[[軟身神像|軟身神像]]」，即關節四肢可以活動，與戴有「真人頭髮」，史載[[媽祖|媽祖]][[林默|林默]]於[[宋太宗|宋太宗]][[雍熙|雍熙]]四年（987年）仙遊[[羽化|羽化]]，得年二十八歲。當前由湄洲祖廟一直保存著林默娘升天成仙後所遺留的一束頭髮，肉身則[[火化|火化]]，後來為了信眾分靈需要而分成三份，而分別製作成三尊軟身神像。

據宮中史料記載，3尊神像中1尊「大媽」留在[[湄洲|湄洲]][[湄洲妈祖祖庙|祖庙朝天閣]]正殿，惜已於[[文化大革命|文化大革命]]期間被毀；第2尊「二媽」流向[[南洋|南洋]]，不知去向；第3尊「三媽」則為今日新竹市長和宮的軟身真髮媽祖，所以長和宮所祭祀的媽祖又稱「湄洲祖廟正三媽」。

當時媽祖來臺時所坐的神轎、香條板等文物，現都保存在長和宮中。

== 湄洲媽祖廟拜訪 ==
民國86年（1997年）[[中國大陸|中國大陸]][[湄洲天后宮|湄洲天后宮]]新建的朝天閣內，[[文革|文革]]後新雕的鎮殿大媽來抵臺巡禮，駐駕於長和宮時，廟方曾取下后冠，隨行的中國大陸湄洲長者們一看到，就立即跪伏在湄洲正三媽座前叩首，當晚長和宮讓出正龕湄洲鎮殿新大媽駐駕，湄洲朝天閣廟方堅持不可，於是新竹方面提議兩宮媽祖並坐，湄洲的鎮殿新大媽坐左方（大邊）。湄洲方面卻表示，來臺巡禮的這尊湄洲鎮殿新大媽只能右方（小邊），但是最終湄洲媽在長和宮廟方堅持下於大邊駐駕。
== 新竹絲路 ==

在百年前的兩岸通商，就從福建平潭通過船運到南寮，經過舊港沿著東大路康樂里，進入市區北門街長和宮。

== 廟會 ==
結合新竹附近十八鄉鎮,公廟與角頭廟悉盡出轎（竹東、竹北、竹南、頭份、香山、楊梅、後龍、龍潭、寶山、峨眉、芎林、北埔、新豐、湖口、關西、新埔、五峰、尖石）發起湄洲正三媽繞境活動。
