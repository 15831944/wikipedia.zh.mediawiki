'''新实智光'''（{{jp|j=新実 智光|hg=にいみ ともみつ|rm=Niimi Tomomitsu|lead=yes}}，{{bd|1964年|3月9日|2018年|7月6日}}）是一名[[奥姆真理教|奥姆真理教]]教徒、前教团高级干部（正大师、教团自治省大臣），1964年出生于日本[[爱知县|爱知县]][[冈崎市|冈崎市]]。新实因参与多起与奥姆真理教相关的命案，被认为是奥姆真理教徒中继教主麻原彰晃之后最为残忍的一人<ref name=AsahiNiimi>{{cite web |title=「グル（麻原彰晃）の指示なら、人を殺すことも喜び」＜教団エリートの「罪と罰」（2）＞ |url=https://dot.asahi.com/wa/2018070600058.html?page=2 |website=週刊朝日 |accessdate=2018-12-18}}</ref>，2018年7月6日，新实因参与[[松本沙林毒气事件|松本沙林毒气事件]]、[[坂本堤律師滅門慘案|坂本堤律師滅門慘案]]，以及[[東京地鐵沙林毒氣事件|東京地鐵沙林毒氣事件]]在{{link-ja|大阪拘置所|大阪拘置所}}被执行死刑<ref name="mainichi20180706">{{cite web|url=https://mainichi.jp/articles/20180706/k00/00e/040/198000c|title=「松本死刑囚ら７人死刑執行」法務省が発表|publisher=每日新闻|date=2018-07-06|access-date=2018-11-08|archive-url=https://web.archive.org/web/20180707005810/https://mainichi.jp/articles/20180706/k00/00e/040/198000c#|archive-date=2018-07-07|dead-url=no}}</ref>。

==早年经历==
新实智光1964年出生于日本[[爱知县|爱知县]][[冈崎市|冈崎市]]，两亲均从事废品回收工作<ref name="藤田庄市2008">{{cite book|author=藤田庄市|title=宗教事件の内側: 精神を呪縛される人びと|url=https://books.google.com/books?id=kYnkAAAAMAAJ|year=2008|publisher=岩波書店|page=151}}</ref>。新实高中毕业于{{link-ja|爱知县立冈崎东高等学校|愛知県立岡崎東高等学校}}，大学毕业于{{link-ja|爱知学院大学|愛知学院大学}}法学部<ref>{{cite web|author=碓井真史|url=https://news.yahoo.co.jp/byline/usuimafumi/20180709-00088618/|title=マインドコントロールの罠：オウム事件死刑執行から|website=Yahoo|accessdate=2018-12-18}}</ref><ref>{{cite book |author1=降幡 賢一 |title=オウム法廷―グルのしもべたち〈下〉 |date=1998 |publisher=朝日文庫 |isbn=978-4022612243 |pages=169}}</ref>。新实早年对宗教和精神世界很感兴趣，曾通读过五島勉的占星读物《{{link-ja|诺查丹玛斯的伟大预言|ノストラダムスの大予言}}》<ref name=AsahiNiimi/>。

==入教及罪行==
新实智光是奥姆真理教的早期教徒，和奥姆真理教教主[[麻原彰晃|麻原彰晃]]关系相当密切。新实在大学期间曾读到麻原的文章，并认为麻原的“浮空能力”相当让人惊讶。新实1986年即加入奥姆真理教的前身组织，曾出席奥姆真理教前身[[奥姆神仙会|奥姆神仙会]]成立时的研讨会，和奥姆真理教教主[[麻原彰晃|麻原彰晃]]关系相当密切。大学毕业后，新实曾一度在爱知当地的食品企业工作，但半年后就辞职，全职加入奥姆真理教<ref name=AsahiNiimi/><ref>{{cite web |title=Profiles of top Aum Shinrikyo members, including six still on death row |url=https://www.japantimes.co.jp/news/2018/07/06/national/crime-legal/profiles-top-aum-shinrikyo-members-including-six-still-death-row/#.XAkmU3QzYac |website=Japan Times |accessdate=2018-12-06 |archive-url=https://web.archive.org/web/20181112013234/https://www.japantimes.co.jp/news/2018/07/06/national/crime-legal/profiles-top-aum-shinrikyo-members-including-six-still-death-row/#.XAkmU3QzYac#.XAkmU3QzYac |archive-date=2018-11-12 |dead-url=no }}</ref>。新实在奥姆真理教成立时是九名“設立責任人”（{{lang|ja|設立時責任役員}}）之一。新实对教主麻原有极强的忠诚心，在教团内担任自治省的首长（大臣），负责对企图脱离教团的教徒进行追捕、关押。此外，曾参与[[松本沙林毒气事件|松本沙林毒气事件]]、[[坂本堤律師滅門慘案|坂本堤律師滅門慘案]]等案件，并在[[東京地鐵沙林毒氣事件|東京地鐵沙林毒氣事件]]中担任司机的工作<ref name=FNNNiimi>{{cite web |title=“麻原尊師の直弟子”新実智光死刑囚　検察は「最も血なまぐさい役割」と指摘 |url=https://www.fnn.jp/posts/00333450HDK |website=FNN |accessdate=2018-12-18}}</ref>。新实参与了奥姆真理教策划的所有命案，因此被认为是奥姆真理教罪犯中最为残忍的一人<ref name=AsahiNiimi/>。

==逮捕与审判==
1995年，新实在[[东京都|东京都]][[霞关|霞关]]的高级公寓中被逮捕<ref>{{cite book |author1=宮口 浩之 |editor=東京キララ社編集部 |title=オウム真理教大辞典 |date=2003 |isbn=978-4380032097 |pages=103-104}}</ref>。[[检察官|检方]]认为，在新实身上“完全无法看见作为人类的良心”（{{lang|ja|人間的良心の一片も見出せない}}），因此在审判时求刑死刑<ref name=FNNNiimi/>。2010年，新实死刑定谳<ref name=AsahiNiimi/>。

截至2018年3月，新实與其他十二名奧姆真理教相關死刑犯一直都被關押在{{link-ja|東京拘置所|東京拘置所}}。2018年3月21日，這些死刑犯被分散到到能執行死刑的{{link-ja|東京拘置所|東京拘置所}}等5處設施，其中新实被送到{{link-ja|大阪拘置所|大阪拘置所}}<ref name="オウム死刑囚7人移送 p.1">{{Cite news|url=http://www.sankei.com/affairs/news/180314/afr1803140030-n1.html|title=【オウム死刑囚】中川智正らオウム死刑囚７人を移送　執行施設ある５拘置所に分散か（1／2ページ）|newspaper=産経新聞|publisher=産業経済新聞社|date=2018-03-14|accessdate=2018-03-15|archivedate=2018-03-15|archiveurl=https://web.archive.org/web/20180315132452/http://www.sankei.com/affairs/news/180314/afr1803140030-n1.html}}</ref><ref name="移送7人判明 p.1">{{Cite news|url=http://www.sankei.com/affairs/news/180314/afr1803140037-n1.html|title=【オウム死刑囚】移送７人は新実智光、林泰男、早川紀代秀、井上嘉浩、岡崎一明、横山真人、中川智正の各死刑囚（1／2ページ）|newspaper=産経新聞|publisher=産業経済新聞社|date=2018-03-14|accessdate=2018-03-15|archivedate=2018-03-15|archiveurl=https://web.archive.org/web/20180315132717/http://www.sankei.com/affairs/news/180314/afr1803140037-n1.html}}</ref>。2018年7月6日，经时任日本法务大臣[[上川阳子|上川阳子]]批准，新实与包括奥姆真理教教主麻原彰晃在内的7名奥姆真理教相关死刑犯被执行死刑，年54岁<ref name="mainichi20180706"/>。
