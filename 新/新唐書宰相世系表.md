'''新唐書宰相世系表'''5卷，[[呂夏卿|呂夏卿]]撰，收錄於《[[新唐書|新唐書]]》卷71上～75下<ref>卷71、73、74、75各分上、下，卷72分上、中、下，所以實質是11卷。</ref>，網羅唐代曾任宰相的家族世系。

== 作者 ==
[[吳縝|吳縝]]《[[新唐書糾謬|新唐書糾謬]]》序：「今唐史本一書也，而紀、志、表則歐陽公（修）主之，傳則宋公（祁）主之。」以總裁主其名，而志、表等各有撰人。[[蘇頌|蘇頌]]《蘇魏公文集》卷66·《呂舍人文集》序：「呂公缙叔起遠方邑尉，入為編修唐書……又集天下碑刻，類為《唐文傳信》，考歷代氏譜族志為《古今系表》，兩書獨出缙叔一手，可謂勤且博矣。」作為編撰《新唐書》志、表的資料長編。《[[宋史|宋史]]》卷331·呂夏卿傳：「夏卿長於史，貫穿唐事，博採傳記雜說數百家，折衷整比。又通譜學，創為世系諸表，於《新唐書》最有功-{云}-。」因此宰相世系表實際出於呂夏卿之手。

== 體例 ==
新唐書宰相世系表之體，每姓之先，先述姓源，再分房表列世系。諸姓排列順序，則以拜相先後為次，直到唐末<ref>[[王鳴盛|王鳴盛]]《[[十七史商榷|十七史商榷]]》卷83‧宰相世系表先後之次：「眾姓未便以意為先後也，故仍依除拜之先後。宰相表首列武德元年六月[[裴寂|裴寂]]拜左僕射、知政事，世系表即以裴姓居首，而凡此姓中各房所有宰相，直至唐末俱以類從。敘畢裴姓，即及劉姓者，以[[劉文靜|劉文靜]]即次裴寂為納言也。」</ref>。而宗室宰相世系，則另見於[[新唐書宗室世系表|宗室世系表]]<ref>《新唐書》卷70下 宗室世系表後敘</ref>。

表中之人，除拜相者注其入相時代；無傳之人，可考者則注其字號與歷官。表錄世系官職，原則以唐末為斷<ref>如《新唐書》卷75下 宰相世系表五下 代北李氏[[李克用|李克用]]（後唐太祖）稱河東節度使、守太師、中書令、晉王，克用子存勗（後[[唐莊宗|唐莊宗]]）稱隰州刺史、檢校司空，都是唐時所授官，而不及後唐贈廟諡。自莊宗諸子以下，無涉於唐，則皆從略。</ref>，但也有例外。

唐代中期以降，節度使常加宰相銜，謂之使相、又稱外宰相，然無入朝執政之實<ref>[[錢大昕|錢大昕]]《[[二十二史考異|二十二史考異]]》卷50 宗室世系表下</ref>，所以一概不錄。而將帥以軍功位至三公三師者20人，另加魏博節度使、太尉[[田承嗣|田承嗣]]的田氏、天平節度使、守司空[[烏重胤|烏重胤]]的烏氏，其中除[[僕固懷恩|僕固懷恩]]叛變，[[朱泚|朱泚]]稱帝伏誅，[[王建_(前蜀)|王建]]、[[韓建|韓建]]、[[朱溫|朱全忠]]亡唐之賊不錄，其餘均述其世系<ref>《新唐書》卷75下 宰相世系表五下 後敘子注。</ref>。

== 史源 ==
[[洪邁|洪邁]]《[[容齋隨筆|容齋隨筆]]》卷6‧唐書世系表：「《新唐書》宰相世系表皆承用諸家譜牒，故多有謬誤。……歐陽公略不筆削，為可恨也。」自有此論，後人認為以唐代譜學之盛，宋去唐近，應該不難入手公私譜牒文獻以編纂<ref>[[李慈銘|李慈銘]]《越縵堂讀書記》上册299頁，沈炳震《唐書宰相世系訂譌》：「余嘗疑歐公既作此表，當時必聚譜牒，何以所載寥寥？……宗室世系表亦然，防五季散亂之後，人多假託華冑也，然因噎廢食，何足以存譜學。」〔光緒甲申（1884）九月十九日〕</ref>。但是現代的考證則不以為然，岑仲勉指出，《新唐書》宰相世系表主要根據成書於[[唐憲宗|唐憲宗]]元和七年（812）的林寶《[[元和姓纂|元和姓纂]]》，化文字敘述的世系為表格。從元和到唐末未及部分，則參取石刻與文集碑誌、《舊唐書》列傳及部分家譜以增補修正<ref>岑仲勉《元和姓纂四校記》再序 4～20頁</ref>。

== 評價 ==
=== 史意 ===
[[劉知幾|劉知幾]]《[[史通|史通]]》書志篇：「逮乎晚葉，譜學尤煩，用之於官，可以品藻士庻，施之於國，可以甄別華夷……氏族而往，實為志者所宜先，而諸史竟無其錄。」對唐前紀傳體史缺乏氏族紀錄表示不滿；只有[[魏收|魏收]]《[[魏書|魏書]]》官氏志，兼及族姓<ref>蒲起龍《史通通釋》卷3</ref>，所以修《新唐書》時遙承其議。[[嚴耕望|嚴耕望]]則指出，《新唐書》宗室、宰相世系表，一般人看似無用，但魏晉南北朝門閥社會，至唐猶盛，其世系之傳承，今惟存此兩表，就冶中古社會史而言，實為寶貴<ref>嚴耕望「新舊兩唐書史料價值比論」 《嚴耕望史學論文集》（上海古籍出版社＼2009年） 下冊 1152頁</ref>。

關於收錄範圍，[[章學誠|章學誠]]《[[文史通義|文史通義]]》卷6外篇和州志氏族表序例上批評：「歐陽《唐書》，撰為宰相世系，顧清門鉅族，但不為宰相者，時有所遺。」而岑仲勉認為，《元和姓纂》所詳為顯官，但如以全書內容列表，則篇幅不容且難以命名。而顯官莫著於宰相，以宰相為綱，也就差不多能網羅《姓纂》世系的菁華部分<ref>岑仲勉《元和姓纂四校記》再序 2頁</ref>。

魏收《魏書》，[[李延壽|李延壽]]《[[南史|南史]]》、《[[北史|北史]]》，一人有傳，父子兄弟皆附載其後，如同族譜。雖然反映南北朝世族政治的盛衰，存其譜系，然附傳常只記歷官而無生平事跡，亢文累辭，成書當時已不無非議<ref>[[趙翼|趙翼]]《[[二十二史剳記|二十二史剳記]]》卷10‧南北史子孫附傳之例</ref>。《新唐書》宗室、宰相世系表載子弟無顯過亦無積勞，又不宜沒其爵里字號者於表中，可省列傳之繁文<ref>蒲起龍《史通通釋》卷3 表曆篇按語：「揆之史法，參以時宜，親若宗房，貴如宰相，傳有所不登，名未可竟沒，胥以表括之，亦嚴密得中之一道哉！」</ref><ref>徐浩《二十五史論綱》（世界書局＼1947年） 本論 第十七章 新唐書</ref>。

=== 錯謬 ===
岑仲勉、趙超指出《新唐書》宰相世系表常見的錯誤<ref>趙超《新唐書宰相世系表集校》前言 3～4頁</ref>：
#名字譌誤。
#世次脫漏誤附他系。
#官爵與文獻記載不符<ref>岑仲勉：「唐人神道碑、墓誌銘所記世系、官位，往往有詳於《新唐書》宰相世系表者，宰相世系表所列則多與《元和姓纂》同。」（《元和姓纂四校記》再序 9頁）</ref>。  
#兄弟排列次序不依長幼<ref>岑仲勉：「《姓纂》敘例，常有甲生乙、丙，先列丙支然後及乙，未必依長幼之序；而宰相世系表者次序，每與《姓纂》相同。」（《元和姓纂四校記》再序 24頁）</ref>。

而其致誤之因，又可分為五項：
* 《元和姓纂》多本諸家行狀，行狀則多本於六朝以來譜牒舊說，反映唐人附會、偽冒名門之習，而宰相世系表因之。所以述唐以前姓源、世系，考之諸史，每有牴誤。
* 遠在北宋《元和姓纂》已乏善本，有不少節略、傳寫脫漏、錯訛之處。
* 編纂中利用《元和姓纂》與其他史料記敘整合的矛盾。
* 《新唐書》修纂時紀、志、表、列傳由多人執筆，成書前未盡劃一<ref>吳縝〈進新唐書糾謬表〉</ref>。
* 表式繁重，傳寫、刊刻不易<ref>岑仲勉《唐史餘瀋》卷4 楊氏世系</ref>。

==== 變例 ====
卷73上 世系表三上穎川韓氏，實無人入相<ref>[[全祖望|全祖望]]《經史問答》卷10：「問：韓氏，宰相世系表四人，璦為一族，休、滉父子為一族，宏為一族，祇應三表，而今有四，何也？答：是歐公之誤也。退之一支，其家無作宰相者，而今亦入之，故有四篇。」</ref>，宋人崇敬[[韓愈|韓愈]]，為之立表<ref>章學誠《丙辰札記》</ref>。卷72中 世系表二中太原王氏並無宰相，而敘其支，不嫌贅列<ref>岑仲勉《元和姓纂四校記》再序 15頁</ref>。

此外，歐陽氏下敘及[[歐陽修|歐陽修]]高祖，曲江張氏則記至[[張九齡|張九齡]]九世孫<ref>岑仲勉《元和姓纂四校記》再序 21頁子-{注}-：「據《廣州人物傳》，宋景德三年九世孫元吉獻墨蹟等，天聖六年九世孫錫獻告身等，由此逆推，七、八世孫已是五代及宋人物。」</ref>，則是採宋代私家譜牒以補《元和姓纂》所闕世系，而失於斷限者。

== 研究史 ==
《新唐書》成書不久，吳縝《新唐書糾謬》已指摘宰相世系表脫漏之處，歐陽修《[[集古錄|集古錄]]》、鄧名世《古今姓氏書辯證》等金石、氏族學著作，更進一步利用石刻等史料考訂異同，補其不備。而研究《新唐書》宰相世系表的專著，則始於清乾隆年間的沈炳震《唐書宰相世系訂譌》十二卷，沈氏認為宰相世系表缺漏錯誤太多，不宜列入《兩唐書合鈔》，所以將宰相世系表部分析出別行<ref>沈炳震《兩唐書合鈔》敘例</ref>，而考訂則主要利用兩唐書列傳與表的互參。

清代乾、嘉期，考證、金石之學大盛，考證筆記如王鳴盛《十七史商榷》、錢大昕《二十二史考異》、趙紹祖《新舊唐書互證》等都有言及宰相世系表之處，[[王昶|王昶]]《[[金石萃編|金石萃編]]》、陸增祥《八瓊堂金石補正》則集當時可見唐代金石文與題跋並加論斷，增補宰相世系表的不備之處。

民國以降，唐代碑刻的大量出土，提供豐富的研究資料，[[羅振玉|羅振玉]]《新唐書宰相世系表補正》二卷（1937年），岑仲勉〈貞石證史〉、〈續貞石證史〉<ref>收錄於《金石論叢》</ref>，以出土誌銘補充或糾正宰相世系表的世系；特別是《元和姓纂四校記》，闡明《新唐書》宰相世系表與《元和姓纂》的淵源，並利用《新唐書》宰相世系表校補現行輯本《元和姓纂》的缺漏，對學術界影響深遠。而綜合歷代考訂成編者，則有趙超《新唐書宰相世系表集校》五卷與詹宗佑《點校本兩唐書校勘彙釋》的《新唐書》宰相世系表<ref>下冊680～752頁</ref>部分。

此外，因學術方法進步，《新唐書》宰相世系表也有更多不同的研究觀點。哈佛燕京學社《新唐書宰相世系表引得》（1934年），提供人名索引；或用統計方式宏觀分析由唐入宋世族勢力與南北人文的消長，《新唐書》宰相世系表不只是中古世族史重要的文獻，對其編纂意義也有更多正面的評價。
