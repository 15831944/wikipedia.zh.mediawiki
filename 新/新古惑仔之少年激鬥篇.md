{{engname|《新古惑仔之少年激鬪篇》|''Young and Dangerous: The Prequel''}}即依據香港漫畫家[[牛佬|牛佬]]所繪製的漫畫《[[古惑仔|古惑仔]]》為基礎，再進行改編的電影作品，本片由[[橙天嘉禾|嘉禾]]公司出品。

== 劇情簡介 ==

1989年，[[陳浩南|陳浩南]]、[[山雞_(古惑仔)|山雞]]、包皮和巢皮一群在藍田邨長大的朋友，在藍田邨配水庫遊樂場足球場的[[足球|足球]]場內玩耍，碰到靚坤強行索取[[保護費|保護費]]，引起雙方人馬衝突。此時，幸得大佬Ｂ路過勸阻，勸退靚坤不該欺凌。

某日，陳浩南在校門前為了Kelly，阻擋追求者豬兜，引來他的不滿。陳浩南一群人平日學業不算突出，偶而在山上練習BAND。

不久，陳浩南一群人參加學校舉辦音樂比賽，獲得現場學生一致好評，但搖滾音樂卻惹來校長的誤解，並將一干人等全部驅逐出校。

怎料，豬兜率領一班人馬在圍堵，仗勢欺人將陳浩南痛毆一頓，幸好警察隨即趕到阻止打鬥，將一群人全部帶往警局拘留。

但是，警察總認為陳浩南是古惑仔，對他有偏見，家中長輩也總認為陳浩南遊手好閒與流氓無異，原無意成為古惑仔的陳浩南在各界異樣眼光歧視後把心一橫，踏入黑道這條江湖路....

== 劇情與正傳出入之處 ==
《新古惑仔之少年激鬪篇》此片與《古惑仔》正傳不同的是少了「大天二」這人物，而且在「古惑仔之人在江湖」裡開頭陳浩南、山雞、大天二、包皮、巢皮等人球場被靚坤欺負，追隨大佬B的故事設定於1985年。而《新古惑仔之少年激鬪篇》這部則是改編成1989年，比《古惑仔》正傳裡開頭的[[時空|時空]]背景晚了4年。在《古惑仔》正傳裡陳浩南、山雞等人在球場被靚坤欺負之後，追隨大佬B為小弟。在《新古惑仔之少年激鬪篇》此片裡山雞這角色改編成比陳浩南、包皮、巢皮他們早先一步追隨大佬B。

== 政治隐喻 ==
{{seealso|古惑仔電影系列#政治隱喻}}
故事由 《[[古惑仔之人在江湖|古惑仔之人在江湖]]》裏陳浩南少年時期1985年，改為設定以1989年為背景，劇中新聞報導現實中的「[[六四事件|六四事件]]」，以「洪興」與「東星」大廝殺血流成河，對比「六四事件」電視新聞的衝突畫面，旁邊字幕打出「有人為了爭取民主自由而流血，有人為了個人理想而流血，也有人不知為了什麼而流血」的旁白<ref>https://www.hk01.com/電影/196049/新古惑仔之少年激鬥篇-霆鋒拍完未夠秤入場睇-熒幕初吻獻舒淇</ref>。

== 向經典香港電影致敬 ==
*劇情為1996年電影《古惑仔之人在江湖》的前傳，而陳浩南等人在在藍田配水庫遊樂場足球場被靚坤欺負的經典一幕重新拍攝 
*劇中陳浩南等人於學校音樂比賽中途被校長下令離校後，一位老師感嘆：「唉！點解唔好好地讀書呢？」，此對白呼應1982年經典電影《[[靚妹仔|靚妹仔]]》女主角碧琪被校長下令離校後，一位老師感嘆的經典對白，而本電影的編劇[[文雋|文雋]]均與電影《[[靚妹仔|靚妹仔]]》相同
*劇終陳浩南紋身的一幕，亦呼應1982年經典電影《[[靚妹仔|靚妹仔]]》女主角碧琪劇終紋身的一幕
*劇中飾演反派「豬兜」的 演員[[何沛東|何沛東]]曾參與多套青少年問題和校園寫實電影如《[[童黨_(電影)|童黨]]》、《[[學校風雲|學校風雲]]》<ref>{{cite web|url=http://hkmdb.com/db/people/view.mhtml?id=7145&display_set=big5|title=何沛東}}</ref>

== 軼事 ==
*香港演員[[謝霆鋒|謝霆鋒]]在拍攝以及電影上映時年紀只有17歲，不符合入場觀看[[三級片|三級影片]]的法定年齡，故僅能在首映典禮的會場外活動，並無法進場觀看。
*劇中一幕演巢皮角色的[[袁偉豪|袁偉豪]]被取笑「飲你既[[維他奶|維他奶]]啦!」，現實中他在電影上映前（1998年）曾替維他奶拍攝廣告
