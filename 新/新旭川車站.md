'''新旭川車站'''（{{jpn|j=新旭川駅|hg=しんあさひかわえき|rm=Shin-asahikawa eki|f=y}}）是一由[[北海道旅客鐵道|北海道旅客鐵道]]（JR北海道）與[[日本貨物鐵道|日本貨物鐵道]]（JR貨物）所共同經營的[[鐵路車站|鐵路車站]]，位於[[日本|日本]][[北海道|北海道]][[旭川市|旭川市]]東8條6丁目，是[[宗谷本線|宗谷本線]]與[[石北本線|石北本線]]的交會車站，也是石北本線的名義起站。但所有行駛於石北本線上的列車，實際上都是以[[旭川車站|旭川車站]]作為真正的起訖站。

== 歷史 ==
* 1922年（[[大正|大正]]11年）11月4日 - 為了配合石北線通車啟用，[[日本國有鐵道|國有鐵道]]在宗谷線旭川至永山之間路段新設了本站，並設定為同時兼營客貨運的一般車站。初啟用時，站名中「旭川」兩字的日文發音為「{{lang|ja|あさひがわ}}」（Asahigawa）。
* 1939年（[[昭和|昭和]]14年）10月 - 通往國策紙漿工業（{{lang|ja|国策パルプ工業}}，是今日[[日本製紙|日本製紙]]的前身）旭川工廠的[[專用鐵路線|專用線]]開始營運。
* 1978年（昭和53年）12月1日 - 除專用線列車外，停辦所有其他的貨運業務。
* 1984年（昭和59年）2月1日 - 廢除行李托運服務。
* 1984年（昭和59年）11月10日 - 車站[[簡易委託車站|簡易委託化]]。
* 1987年（昭和62年）4月1日 - [[國鐵分割民營化|國鐵分割民營化]]後，本站的客運站與貨運站部分分別由JR北海道與JR貨物繼承。
* 1988年（昭和63年）3月13日 - 站名中的「旭川」兩字更改發音，其日文發音由原本的「{{lang|ja|あさひがわ}}」改成「{{lang|ja|あさひかわ}}」（Asahikawa）。
* 日期不詳 - 廢除簡易委託。
* 1997年（[[平成|平成]]9年）9月 - 通往[[日本製紙|日本製紙]]旭川工廠的專用線廢線。
* 2003年（平成15年）5月10日 - 站內全面[[電氣化鐵路|電氣化]]。

== 車站構造 ==
新旭川車站站內為一座[[側式月台|側式月台]]一條乘車線與一座[[島式月台|島式月台]]兩條乘車線的配置。站房位於車站的西側，而月台之間由[[人行天橋|跨線人行天橋]]連接。緊鄰站房旁的是側式月台及4號線，島式月台則為2號及3號線，沒有1號線，但站內設有幾條其他用途的側線。

新旭川所屬的宗谷本線本身並沒有電力列車行駛，但因旭川車站周遭的再開發計劃「[[北彩都旭川|北彩都旭川]]」（{{lang|ja|北彩都あさひかわ}}）之影響，JR北海道將旗下所屬的[[旭川運轉所|旭川運轉所]]搬遷到[[北旭川車站|北旭川車站]]附近。因此，為了提供回送至旭川運轉所的[[函館本線|函館本線]]列車所需之電力，新旭川站內除側線外的路線皆已全面[[電氣化鐵路|電氣化]]。

目前為無人車站，但站內設有[[自動售票機|自動售票機]]及[[廁所|廁所]]。

至於貨運站部分，目前新旭川是JR貨物[[車攜貨運|車攜貨運]]（{{lang|ja|車扱貨物}}）業務的臨時收發站，僅靠公路運輸進行貨物的運送，沒有貨運列車或專用線列車停靠。最後在新旭川營運的貨運線，是在1997年時廢線的日本製紙旭川工廠專用線。

== 車站周邊 ==
車站周圍為市區。此外，由於不少居民擁有車輛，故車站使用率比周邊街道低。
