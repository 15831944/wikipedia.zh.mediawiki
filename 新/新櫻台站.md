'''新櫻台站'''（{{jpn|j=新桜台駅|hg=しんさくらだいえき|rm=Shin-sakuradai eki|f=y}}）是一個位於[[東京都|東京都]][[練馬區|練馬區]]{{link-ja|櫻台 (練馬區)|桜台 (練馬区)|櫻台}}一丁目，屬於[[西武鐵道|西武鐵道]][[西武有樂町線|西武有樂町線]]的[[鐵路車站|鐵路車站]]。車站編號是'''SI38'''。

== 年表 ==
* 1983年（[[昭和|昭和]]58年）10月1日 - 作為西武有樂町線（新櫻台－小竹向原）的終點站開業。
* 1994年（[[平成|平成]]6年）12月6日 - 西武有樂町線全線開業（但是此站－練馬暫定以[[單線鐵路|單線]]開業），成為中途站。
* 1998年（平成10年）3月26日 - 西武有樂町線此站－練馬[[複線鐵路|複線]]化，開始興池袋線[[所澤站|所澤]]、[[飯能站|飯能]]方向[[直通運行|直通運行]]。
* 2008年（平成20年）6月14日 - 與同日開業的東京地下鐵副都心線開始互相直通運行。
* 2013年（平成25年）3月16日 - 伴隨途經[[飯能站|飯能站]]與[[東急東橫線|東急東橫線]]、[[港未來線|港未來線]]、東京地下鐵副都心線至[[元町·中華街站|元町·中華街站]]開始直通運行，加入新的快速急行，成為其停車站。

== 車站構造 ==
[[對向式月台|對向式月台]]2面2線的[[地底車站|地底車站]]。位於環七通（[[東京都道318號環狀七號線|東京都道318號環狀七號線]]）的正下方。設有[[升降機|升降機]]3部（地面－大堂之間1部、大堂－月台2部）、[[扶手電梯|扶手電梯]]2部（大堂－月台）、多機能[[廁所|廁所]]。

西武鐵道管理的車站當中唯一的地底車站。
