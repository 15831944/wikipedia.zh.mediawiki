'''新竹李長榮化工廠事件'''，簡稱'''新竹榮化事件'''，是一件發生於1980年代在[[台灣|台灣]][[新竹市|新竹市]]的[[環境污染|環境污染]]與[[公害|公害]]事件，被認為是台灣第一起[[大學|大學]][[教授|教授]]參與抗爭運動成功首例，與[[鹿港居民反杜邦設廠事件|鹿港居民反杜邦設廠事件]]並列為台灣環境運動重要的啟始點與里程碑<ref>{{Cite web |url=http://e-info.org.tw/node/13959| author= 本報訊(倪宏坤報導)|language=zh-tw ||title= 高清波癌症謝世 環保鬥士清影長存|publisher=[[台灣環境資訊協會|台灣環境資訊協會]]}}</ref>。

== 背景 ==
1980年，台灣新竹市發生環境災害事件，疑似大量廢水及帶有刺鼻魚腥臭味的廢氣，造成嚴重的[[空氣污染|空氣污染]]與[[水污染|水污染]]{{r|黃提源}}。臭氣籠罩[[國立清華大學|國立清華大學]]與[[國立交通大學|國立交通大學]]兩校校園<ref>{{Cite web |url= http://ecology.org.tw/publication/21/21-p07.htm | author= 林碧堯|language=zh-tw ||title= 台灣的反公害運動|publisher= 生態台灣}}</ref>當時，[[李長榮化工廠|李長榮化工廠]]（簡稱[[李長榮集團|榮化]]）投資三千萬美元於[[新竹市|新竹市]][[東區_(新竹市)|東區]]水源里設廠，生產[[福馬林|福馬林]]、[[二甲基甲醯胺|二甲基甲醯胺]]及[[有機溶劑|有機溶劑]]<ref>{{cite book|title=李長榮化工將投資三千萬美元 建醋酸乙酯等六座化工廠|publisher=經濟日報09版|date=1980-05-26}}</ref><ref>{{cite book|title=李長榮硫酸二甲酯廠 已正式開工生產|publisher=經濟日報02版|date=1980-10-22}}</ref>。遭質疑為罪魁禍首。1983年10月，[[經濟部|經濟部]]下令李長榮化工公司新竹廠所製造的嚴重水污染問題應限期改善，否則必須停工並將撤銷以往所享受的投資獎勵，但榮化仍不承認該公司新竹廠排放的廢水會影響新竹地區水質<ref>{{cite book|title=李長榮水污染問題 經部限月底前改善|publisher=經濟日報02版|date=1983-10-05}}</ref>。

後來，新竹市衛生局和省環保局經過化驗，發現廢水污染物來自人畜的排泄物「游離氨氮」而非榮化所排放的主要有機物「甲醇」，然民眾卻不信任此化驗結果；另外，李長榮的排氣雖符合既有標準，卻不符合該年8月才實施的新標準，因此環保局要求榮化必須在12月以前改善，但居民也不願意等，故發動示威、「自救」抗議，在居民圍廠450天後，榮化被迫關廠。<ref>{{Cite web|url=http://www.cw.com.tw/article/article.action?id=5039035|title=李長榮停工之後｜環境｜2012-06-28｜天下雜誌第67期|accessdate=2016-06-24|language=zh-TW}}</ref>

== 抗爭運動 ==
榮化的工廠氣味讓新竹市水源里附近的農民與居民難以忍受。由於當時的清華大學教授宿舍北院距離污染源不遠，鄰近的清大教授也受不了工廠排放的難聞氣味，因此向榮化溝通與陳情，但未獲得回應。後來向[[新竹市政府|新竹市政府]]陳情，並且由當時的清華大學校長[[毛高文|毛高文]]出面，300多名教授聯名上書向[[行政院長|行政院長]][[俞國華|俞國華]]反應李長榮化工汙染的狀況，但皆未獲得回應。因此清華大學甚至有一位教授戴著[[防毒面具|防毒面具]]上課，這個畫面當時還上了報紙{{r|thb}}{{r|黃提源}}。

在1986年，無法忍受的水源里居民終於發起抗爭行動，農民與居民以[[混凝土車|混凝土車]]載運砂石與水泥倒在榮化門口，築起一道矮牆封閉廠區大門，強迫榮化封廠。居民並在矮牆後搭起棚子，每天派人輪班堵在工廠門口，不讓工廠開工，並要求榮化遷廠。1987年，新竹市成立了[[新竹市公害防治協會|新竹市公害防治協會]]，是新竹第一個民間的環保團體，成員包括清大、交大與[[新竹教大|新竹師院]]的教授，目標是解決榮化造成的公害問題{{r|thb}}{{r|黃提源}}。抗爭運動持續了450天，終於迫使榮化停工。長達一年多的圍廠[[抗爭運動|抗爭運動]]，被譽為[[台灣史|台灣史]]上第一樁因為人民集體力量而成功的[[社會運動|社會運動]]，被視為台灣指標性的圍廠抗爭事件，尤其是清大教授親自參與自救抗爭運動，更是臺灣[[環境運動|環境運動]]的首例。
