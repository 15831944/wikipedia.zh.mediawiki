'''新吉莊北極殿'''，位於高雄市[[左營區#區塊|左營新庄仔]]的[[玄天上帝|玄天上帝]]廟。

== 簡介 ==
新吉莊北極殿，位於高雄市[[左營區|左營區]]為[[左營_(左營區)|左營]][[左營區#區塊|新庄仔地區]]重要信仰中心，東方比鄰河堤公園，前方為[[裕誠路|裕誠路]]商圈，西方鄰近[[高雄巨蛋|高雄巨蛋]]、[[漢神巨蛋|漢神巨蛋]]商圈。主祀[[北極玄天上帝|北極玄天上帝]]（[[廍後|廍後]]三上帝），為[[鳳邑舊城城隍廟#興隆內外里十三公廟|興隆內外里十三公廟]]其中一廟。俗稱「'''三上帝廟'''」，原本廟中以奉祀來自[[廍後|廍後]]庄真武廟的玄天三上帝，現在廟中的大上帝、二上帝是在來到新庄仔地區建廟之後才雕刻、開光。本廟配祀[[中壇元帥|中壇元帥]]、[[八仙|八仙]]、[[福德正神|福德正神]]、[[太歲星君|太歲星君]]等神明。
[[File:Xinjizhuangbeijidian_outside.JPG|thumb]]

== 歷史 ==
清康熙年間，居民創建真武廟於鳳山縣興隆里[[廍後|廍後庄]]厝尾仔。奉祀[[玄天上帝|玄天上帝]]。甲午戰爭，割讓台澎之後，[[台灣日治時期|日治時期]][[昭和3年|昭和3年]](1928年)曾由薛占募捐整修。[[昭和12年|昭和12年]](1937年)，日本人擴建左營軍港，將廍後庄劃入軍事要塞，軍民被迫遷村，北極殿遭到拆除，所有神像被移往[[大崗山|大崗山]][[超峰寺|超峰寺]]，幸好玄天上帝等五尊神像，由薛占等五人藏匿。[[昭和15年|昭和15年]](1940年)由廍後遷村至新庄子之後，由新吉莊王牛潮等耆老，迎請廍後庄真武廟玄天三上帝、中壇元帥、何仙姑金身於其宅。原廍後庄真武廟玄天大上帝仍留於廍後，供奉於現今[[廍後北極殿|廍後北極殿]]，而玄天二上帝之後被請往[[後勁|後勁]]，現今供奉於後勁北極殿。[[民國38年|民國38年]](1949年)，眾弟子集資建廟。[[民國77年|民國77年]](1988年)，擴建廟宇成今日規模。<ref>玄天三上帝出巡遶境科儀手冊，2010，新吉莊北極殿</ref>
[[File:Xinjizhuangbeijidian_zhibei.JPG|thumb]]，擲出[[筊杯|筊杯]]後，筊杯黏著於福德正神神桌桌圍上。]]

== 進香、出巡紀錄 ==
*[[民國77年|民國77年]](1988年)，擴建完成，安座，[[民國78年|民國78年]](1989年)，舉行謝土、五朝清醮。並於晚間往[[大鵬灣|大鵬灣]]海岸[[進香|進香]]招軍。
*[[民國82年|民國82年]](1993年)，往[[旗津區|旗津]]海墘進香招軍。
*[[民國86年|民國86年]](1997年)，往[[茄萣區|茄萣]]海岸進香招軍。
*[[民國88年|民國88年]](1999年)，廍後庄遷村一甲子，往[[菜公|菜公]]、[[後勁|後勁]]、廍後、[[左營庄|左營]]會香。
*[[民國93年|民國93年]](2004年)，奉玄天上帝諭旨北巡，往[[桃園市|桃園]][[新屋區|新屋鄉]]、[[南投縣|南投]][[名間鄉|名間鄉]][[松柏嶺受天宮|松柏嶺受天宮]]進香，回到左營、新庄子遶境。
*[[民國99年|民國99年]](2010年)，廍後庄遷村70周年，奉玄天上帝諭旨南巡，往[[屏東縣|屏東縣]][[東港鎮|東港鎮]]、[[林邊鄉|林邊鄉]]崎峰村進香，之後返回右昌、左營遶境。

== 例行活動 ==
*[[農曆|農曆]]每月20日，舉行賞軍犒將科儀。
*農曆1月9日（1月8日晚上至1月9日凌晨），舉行[[玉皇上帝|玉皇上帝]]祝壽科儀。
*農曆3月3日，玄天上帝聖誕，舉行祝壽慶典。
*農曆7月10日，舉行[[中元節|中元]]法會。
*農曆8月15日前後，舉行安[[五營神將|五營]]、釘青竹符儀式與遶境。
