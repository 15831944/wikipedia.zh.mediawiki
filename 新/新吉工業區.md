'''新吉工業區'''，位於[[臺南市|台南市]][[安南區|安南區]]與[[安定區|安定區]]交界的工業區，為[[2010年中華民國縣市改制直轄市|台南縣市合併]]後開發的首座工業區<ref>{{Cite web|url=https://www.nownews.com/news/20181211/3114851/|title=年產值約380億　台南新吉工業區全區道路啟用通車|accessdate=|author=陳聖璋|date=2018-12-11|publisher=}}</ref>。因位於[[曾文溪|曾文溪]]附近，為避免工業排水影響下游的濕地生態，新吉工業區限制進駐廠商的類型，並標榜為「'''全國第一個污水零排放工業區'''」<ref>{{Cite web|url=https://www.peopo.org/news/387789|title=優質工業區新里程碑 新吉工業區全區道路啟用通車|accessdate=|author=|date=2018-12-11|publisher=}}</ref>。新吉工業區占地約123.3公頃，其中76.8公頃為產業用地，其餘為管理機構、道路等公共設施。

新吉工業區原為種植甘蔗的[[台灣糖業公司|台糖]]新吉農場，1997年被編定為工業區，但後來因故中止開發。直到2010年台南縣市合併後，為輔導未登記工廠合法經營，重啟新吉工業區開發案。預計2020年完成全區開發。

== 開發沿革 ==

* 1997年，[[經濟部工業局|經濟部工業局]]依[[行政院|行政院]]「振興經濟方案」，核准編定為工業區<ref name=":0">{{Cite web|url=http://ud.tainan.gov.tw/UPBUD_sys/Uploads/Backend/YearEPaper/2013%20year%20paper.pdf|title=2013年台南市都市發展年報|accessdate=|author=|date=|publisher=}}</ref>
* 1999年，因故暫停開發<ref>{{Cite web|url=https://www.sin-ji.com.tw/?pageBox=about-overview|title=新吉工業區 開發緣起|accessdate=|author=|date=|publisher=新吉工業區}}</ref>
* 2013年，台南市都市計畫委員會審議通過'''新吉工業區細部計畫<ref name=":0" />'''
* 2014年1月，新吉工業區細部計畫公告實施<ref name=":0" />
* 2014年8月，通過環境影響評估，並承諾'''廢水零排放'''
* 2015年11月，開發動工，園區分六區動工及招商
* 2018年12月，全區道路完工啟用

== 園區規劃 ==
'''新吉工業區服務中心'''

2017年10月24日動工，設有國際會議廳。建築獲得2018國家卓越建設獎，以及銀級綠建築標章。

'''公園'''

園區內有5座公園，其中一座為4.28公頃的「環頸雉保育公園」。

== 進駐廠商 ==
園區產業以低汙染及低耗水為限，且食品、成衣及服飾、塑膠製品製造業外之製造業，不得產生製程廢水<ref>{{Cite web|url=https://www.sin-ji.com.tw/?pageBox=about-features|title=進駐產業類別與特色|accessdate=|author=|date=|publisher=新吉工業區}}</ref>。
