《'''新高漫畫集團'''》，是台灣第一個同人團體。
== 緣起 ==
1938年成立於[[新竹市_(州轄市)|新竹市]]，因參加「'''[[日本漫画研究会|日本漫画研究会]]'''」（Nihon Manga Kenkyuukai）函授課程所贈送之會員徽章，彼此相認，促成 《'''新高漫畫集團'''》的成立。

== 成員 ==
集團成員以"臺灣在住" 對大眾繪畫有研究者為限, 陸續加入之成員包含王花(新竹市), 洪朝明(新竹市), [[葉宏甲|葉宏甲]](新竹市), 陳家鵬(新竹市), 林河世, 蜂野劍(臺中州東勢郡), 梁梓義(新竹州桃園郡)等人.

== 集團規約 ==
一. 本集團以研究下列項目為目的:<br>
# 漫画.
# 插繪.
# 其他大眾繪画.
二. 本集團事務所位置 : 新竹市表町三丁目<br>
