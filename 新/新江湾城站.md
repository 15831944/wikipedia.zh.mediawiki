'''新江湾城站'''是[[上海轨道交通10号线|上海轨道交通10号线]]的一座地下车站，位于[[上海市|上海市]][[杨浦区|杨浦区]][[淞沪路|淞沪路]]与[[殷行路|殷行路]]交叉口东侧、横跨殷行路，地处杨浦区[[新江湾城|新江湾城]]，车站由此得名。<ref name="stn info">{{Cite web |url=http://service.shmetro.com/czxx/index.htm?id=1068&stationname=%E6%96%B0%E6%B1%9F%E6%B9%BE%E5%9F%8E |title=车站信息新江湾城 |accessdate=2018-07-26 |publisher=上海地铁}}</ref>
本站于2010年4月10日随[[上海轨道交通10号线|10号线]]一起启用，在10号线二期工程启用前，本站将一直是10号线的东起始站。<ref name="phase1 operation">{{Cite web |url=http://society.people.com.cn/GB/41158/11334069.html |title=上海地铁10号线今试运营 申城轨道交通里程约420公里 |accessdate=2018-03-02 |author=徐焱 |date=2010-04-10 |publisher=人民网 |archive-url=https://web.archive.org/web/20160303234136/http://society.people.com.cn/GB/41158/11334069.html |archive-date=2016-03-03 |dead-url=no }}</ref><ref name="phase2 line">{{Cite web |url=http://www.shgtj.gov.cn/hdpt/gzcy/sj/201205/t20120504_546959.html |title=关于轨道交通10号线（新江湾城-外高桥）选线专项规划的公示 |accessdate=2018-07-25 |author=上海市规划与国土资源管理局 |date=2012-05-04 |archive-url=https://web.archive.org/web/20180102073336/http://www.shgtj.gov.cn/hdpt/gzcy/sj/201205/t20120504_546959.html |archive-date=2018-01-02 |dead-url=no }}</ref>

===楼层分布===
新江湾城站共有三层。其中大部分出口设于地面，地下一层设有车站商圈和站厅；地下二层设有10号线使用的岛式站台2座。<ref name="stn info2">{{Cite web |url=http://service.shmetro.com/skin/zct/1068.jpg |title=站层图 |accessdate=2018-07-26 |author=上海地铁 |archive-url=https://web.archive.org/web/20180725214701/http://service.shmetro.com/skin/zct/1068.jpg |archive-date=2018-07-25 |dead-url=no }}</ref>

===站台===
新江湾城站共设有3条股道，目前只使用其中一条股道作为10号线列车[[到发线|到发线]]使用，其余股道作为预留不使用，留待10号线二期工程投入运营后适用。内侧的两个站台共用一条轨道。此外，新江湾城站一共设有两个[[島式月台|岛式站台]]，目前只使用西侧[[島式月台|岛式站台]]的外侧[[車站月台|站台]]作为10号线列车到发站台使用，以供到达和始发列车进行乘客乘降活动。在[[上海轨道交通10号线|10号线]]北段新江湾城站至[[基隆路站|基隆路站]]的路段启用后，本站亦会作为[[航中路站|航中路站]]至新江湾城站的[[上海轨道交通10号线|10号线]]区间车终点使用。<ref name="stn info2"/><ref name="stn info3">{{Cite web |url=http://www.envir.gov.cn/docs/2014/20140317256.pdf |title=上海市轨道交通10号线二期工程环境影响评价报批前公示 |accessdate=2018-07-26 |author=中铁第四勘察设计院集团有限公司 |date=2014-03-17 |publisher=上海市环保局}}</ref>
