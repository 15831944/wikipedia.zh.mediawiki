'''新疆菜'''是[[清真菜|清真菜]]的一个分支，受[[波斯菜|波斯菜]]影响较大。不同民族有不同菜肴，但且食材均以羊肉为主。主要菜式有：
== 主食 ==

*[[馕|馕]]，类似面包的圆形面饼，干旱气候下可长时间[[保藏|保藏]]。是新疆常见的主食。
*[[抓饭|抓饭]]，肉抓饭通常用肉块（可用[[羊肉|羊肉]]、[[牛肉|牛肉]]、[[鸡肉|鸡肉]]或[[马肉|马肉]]）、[[胡萝卜|胡萝卜]]、[[食用油|食用油]]和[[大米|大米]]一次焖制而成。不同地区不同风味，且分有甜咸版本。以前吃抓饭时都是直接用手抓食，而近十几年城市中早已改为常用[[勺子|勺子]]吃。
*烤包子
*蒸包子
*[[拉条子|拉条子]]
*油塔子

== 菜式 ==
*[[大盘鸡|大盘鸡]]，主要原料是鸡肉和辣椒，风味偏辣，份量足。通常配以皮带面。
*馕包肉


== 烧烤 ==
*烤[[羊肉串|羊肉串]]，通常认为是目前内地最流行的新疆食品，加以[[孜然|孜然]]和辣椒粉。
*馕坑肉

== 凉菜 ==
*皮辣红（洋葱、青椒、西红柿沙拉）
*酸奶黄瓜沙拉

== 甜食 ==
*酸奶配[[葡萄干|葡萄干]]
*[[玛仁糖|玛仁糖]]
