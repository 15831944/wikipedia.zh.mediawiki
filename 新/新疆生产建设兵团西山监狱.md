'''新疆生产建设兵团西山监狱'''，位于[[新疆维吾尔自治区|新疆维吾尔自治区]][[乌鲁木齐市|乌鲁木齐市]][[沙依巴克区|沙依巴克区]]西山路59号，是[[新疆生产建设兵团监狱管理局|新疆生产建设兵团监狱管理局]]的直属单位。

==简介==
新疆生产建设兵团西山监狱成立于1985年，是看押、监管、改造罪犯，执行刑罚的单位。2001年，创建为沙依巴克区文明单位，2002年创建为乌鲁木齐市文明单位。2004年，在创建新疆维吾尔自治区文明单位过程中，因非人为因素而未能获得荣誉称号。2005年，由沙依巴克区申请确认为新疆维吾尔自治区文明单位<ref>{{Cite web |url = http://www.sayibak.gov.cn/xxgk2/departmentDetail.asp?article_id=7564 |title = 关于申报2005年自治区级文明单位、市级文明单位、市级文明行业的请示 |author =  |date = 2005-05-12 |publisher = 沙依巴克在线 |language =  |accessdate =  |quote =  }}</ref>。2011年，作为新疆维吾尔自治区文明单位届满<ref>{{Cite web |url = http://roll.sohu.com/20111124/n326673993.shtml |title = 关于2011年新申报和届满自治区文明单位、文明村、文明社区公示 |author =  |date = 2011-11-24 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。2010年，获得“全国监狱劳教工作先进集体”称号<ref>{{Cite web |url = http://www.legalinfo.gov.cn/index/content/2010-04/13/content_2110847.htm |title = 全国监狱劳教工作先进集体先进个人表彰名单 |author =  |date = 2010-04-13 |publisher =中国普法网  |language =  |accessdate =  |quote =  }}</ref>。

2014年1月23日，新疆生产建设兵团纪委再次对5起违反[[中央八项规定|中央八项规定]]和新疆生产建设兵团党委26条规定精神典型问题进行了通报，其中包括西山监狱党委书记、监狱长刘生铭收受购物卡、礼品问题，新疆生产建设兵团监狱管理局党委给予刘生铭党内警告处分，并在全系统通报<ref>{{Cite web |url = http://bt.people.com.cn/n/2014/0124/c354305-20467956.html |title =新疆兵团再次通报5起违纪事件 石河子书记带15人公款旅游  |author =  |date = 2014-01-24 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。
