'''新代田站'''（{{jpn|j=新代田駅|hg=しんだいたえき|rm=Shin-daita eki|f=y}}）是一個位於[[東京都|東京都]][[世田谷區|世田谷區]][[代田_(世田谷區)|代田]]五丁目，屬於[[京王電鐵|京王電鐵]][[井之頭線|井之頭線]]的[[鐵路車站|鐵路車站]]。[[車站編號|車站編號]]是'''IN06'''。

== 历史 ==
* 1933年（[[昭和|昭和]]8年）8月1日 - 隸屬於帝都電鐵，以'''代田二丁目'''之站名開始營運。
* 1940年（昭和15年）5月1日 - 與小田原急行鐵道合併，成為該公司帝都線的車站。
* 1942年（昭和17年）5月1日 - [[小田急電鐵|小田急電鐵]]與[[東京急行電鐵|東京急行電鐵]]（大東急）合併。
* 1948年（昭和23年）6月1日 - 從東急分離，成立京王帝都電鐵，成為該公司井之頭線的車站。
* 1966年（昭和41年）7月21日 - 站名改為為'''新代田站'''。

== 車站構造 ==
[[跨站式站房|跨站式站房]]結構，具有2面2線[[側式月台|相對式月台]]的[[地面車站|地面車站]]。月台設置於低於兩側地面的溝地內，車站上方是[[東京都道318號環狀七號線|環狀七號線]]（環七通），月台的一部分則在環七通的道路正下方。車站閘口設在環狀七號線的路沿。

閘口與月台之間有[[輪椅|輪椅]]適用的[[電扶梯|電扶梯]]以及[[電梯|電梯]]。廁所設置於1號線月台，並基於通用設計的一環，同時也設有無障礙洗手間。

== 車站周邊 ==
與鄰站[[下北澤站|下北澤站]]、[[東松原站|東松原站]]兩站的距離皆僅僅500公尺。從2號線月台的中央位置，可以輕易用肉眼看到鄰站。

因為到鄰站[[下北澤|下北澤]]僅徒步數分鐘即可到達，加上車站出入口即是車流量大的環七通之故，本車站的周邊沒有形成商店街的狀況也屬井之頭線上車站中的特例。

* 世田谷區代田區民中心
* 世田谷區立代田圖書館
* {{link-ja|世田谷區役所|世田谷区役所}}新代田[[社區總體營造|社區總體營造]]出張所
* 新代田站前[[郵便局|郵便局]]

=== 巴士路線 ===
車站北側為巴士折返場，有該站發車（一部份為駒澤陸橋發車）的路線巴士運行（下記2條路線曾經為共同經營的直通路線）。
;[[都營巴士|都營巴士]]
* 宿91（北行）：往新宿站西口、{{link-ja|都營巴士杉並支所|都営バス杉並支所|杉並車庫}}（經堀之內）
* 宿91（南行）：往駒澤陸橋
;[[東急巴士|東急巴士]]
* 森91：往大森操車所（經若林站前、上馬、洗足站入口、長原、馬込站、大森站，部分班次只行駛到若林[[派出所|派出所]]）

== 其他 ==
當站於[[小田急小田原線|小田急小田原線]][[世田谷代田站|世田谷代田站]]之間，於[[東京急行電鐵|東京急行電鐵]]（大東急）所經營時，在二戰中期至戰後初期曾經設有
連絡線（[[代田連絡線|代田連絡線]]）。這是由於戰時遭受[[空襲日本|空襲]]，井之頭線的電車幾乎燒失、全毀，因而設置聯絡線由小田急線緊急調度車輛維持運作。在廢除之後，仍有鐵道分歧的部份被保留了一段時間。但是因為住宅區的發展，聯絡線原址因開發而無任何遺跡留存下，僅有一點線索留存在[[小田急線|小田急線]]上。而在小田急線的雙複線軌道完工後，聯絡線的遺跡將全數消失。

該聯絡線的官方資料已難以考究，但戰時美軍的[[航拍|航拍]]中仍可見到當時的聯絡線路線圖。
* 航拍（日本國土地理院：國土變遷檔案 航拍閱覽系統） 地區：東京西南部，攝影機關：美軍 撮影日:1947-09-08[http://archive.gsi.go.jp/airphoto/ViewPhotoServlet?workname=USA&courseno=M449&photono=120]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}
