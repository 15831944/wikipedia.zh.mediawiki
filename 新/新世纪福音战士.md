《'''新世纪福音战士'''》（{{lang-ja|新世紀エヴァンゲリオン}}；{{lang-en|Neon Genesis Evangelion}}）簡稱《'''EVA'''》，是[[日本动画|日本动画]]公司[[GAINAX|GAINAX]]制作的原创[[電視動畫|電視動畫]]作品，由[[庵野秀明|庵野秀明]]导演，共26话，於1995年在[[東京電視網|東京電視網]]首次播放。本作获得了众多奖项和荣誉，是最受欢迎和最具影响力的日本动画之一。

《EVA》讲述14岁的少年少女操纵巨大人型兵器[[Evangelion|Evangelion]]与神秘敌人“[[使徒_(新世纪福音战士)|使徒]]”战斗，最开始是典型的[[机器人动画|机器人动画]]，故事注重描写战斗场景和人物对话。随着情节推移，故事逐渐变成对人物内心世界的[[精神分析|精神分析]]式的叙述。该系列的特色是大量[[宗教|宗教]]、[[哲学|哲学]]、心理意象的运用，以及革命性的强烈[[意识流|意识流]]手法。播出后在日本掀起被称为“社会现象”程度的巨大回响与冲击，并成为[[日本动画史|日本动画史]]上的一座里程碑。动画业界和许多动漫欣赏者常视其为日本最伟大的动画之一。<ref>{{cite web|title=《EVA》击败《变形金刚2》 观众哭成泪人(图)|url=http://ent.sina.com.cn/m/f/2009-07-03/14212594573.shtml|website=新浪娱乐|date=2009-07-03}}</ref><ref>{{cite web|title=日本票房：动画片《新世纪福音战士》问鼎冠军|url=http://ent.qq.com/a/20090701/000237.htm|website=腾讯娱乐|date=2009-06-30}}</ref><ref>{{cite web|title=《EVA新剧场版Q》11月17日上映决定|url=http://comic.qq.com/a/20120702/000013.htm|website=腾讯动漫|date=2012-07-02}} [http://news.cntv.cn/20120703/105513.shtml 央视网发布]、[http://art.china.cn/cartoon/2012-07/02/content_5127547.htm 中国网发布]</ref><ref group="注">此段2006年就已存在，应该是被引用。</ref>

電視版動畫以跳脫了劇情框架的型態完結，為了回應影迷的支持而推出完結篇'''[[新世紀福音戰士劇場版|劇場版]]'''<ref name=proware>[https://web.archive.org/web/20101026013409/http://www.prowaremedia.com.tw/eva_newmov01_info.htm 作品解說]. 普威爾國際</ref>，即1997年先后公映的《[[新世紀福音戰士劇場版：死與新生|新世紀福音戰士劇場版：死與新生]]》和《[[新世紀福音戰士劇場版：THE_END_OF_EVANGELION|新世紀福音戰士劇場版：THE END OF EVANGELION]]》。动画系列的角色设计师[[贞本义行|贞本义行]]将该系列改编成'''[[新世紀福音戰士_(漫畫)|同名漫画]]'''，系列[[衍生作品|衍生作品]]包括[[电影|电影]]、[[漫画|漫画]]、[[电子游戏|电子游戏]]與[[小說|小說]]。
2006年，庵野秀明导演設立[[khara|khara工作室]]将该系列再構築为'''[[福音戰士新劇場版|福音戰士新劇場版]]'''系列<ref>[https://diamond.jp/articles/-/107910 庵野秀明監督が初めて語る経営者としての10年（上）]{{ja}}</ref>。目前版权方已由「GAINAX/Project Eva.」变更为「khara/Project Eva.」<ref>[http://diamond.jp/articles/-/108195 庵野秀明監督が初めて語る経営者としての10年（下）]{{ja}}</ref>

== 故事简介 ==
=== 世界观 ===
{{main|新世纪福音战士名词列表|Evangelion|使徒 (新世纪福音战士)}}
作品中的2000年9月13日，[[南極大陸|南極大陸]]上发生大爆炸。造成南极的冰川融化、[[地軸|地軸]]扭曲，随之而来的[[海平面|海平面]]上升、全球气候异常，加上后来引发的战争等因素，導致世界人口减半。随着接连不断的大规模战争结束，各国军队编入[[联合国|联合国]]军队。此次侵袭人类的大灾难被称为[[新世纪福音战士名词列表#第二次冲击|第二次冲击]]，联合国发表的大灾难原因是「大质量的陨石撞击南极」。实际上却是为了在其他[[使徒_(新世纪福音战士)|使徒]]觉醒之前将第一使徒[[亞當|亞當]]还原成卵的结果，而秘密组织[[SEELE|SEELE]]凭借其巨大影响力隐瞒了事实真相。SEELE通过[[死海文书|死海文书]]获悉了相关信息，事先采取了一系列措施。使徒拥有强大的[[新世纪福音战士名词列表#A.T.力场|A.T.力场]]（绝对领域），一般的武器无法消灭它，为阻止预测中即将来袭的使徒毁灭人类，SEELE在联合国的名义下设立Gehirn开发泛用人型决战兵器-人造人[[Evangelion|Evangelion]]（简称：EVA），并且建立起要塞都市·[[第3新東京市|第3新東京市]]，也是Gehirn总部所在地。Gehirn之后改组为[[NERV|NERV]]，全力推行[[新世纪福音战士名词列表#人类补完计划|人类补完计划]]。故事发生在2015年，此时第3新东京市接二连三地遭逢使徒的来袭。

=== 剧情 ===
第二次冲击十五年后的2015年，不明巨型生命体“使徒”在日本登陆，并向第3新东京市袭来。联合国军与使徒的作战失败，作战指挥权移至NERV。本作主人公碇真嗣被父亲・NERV总司令碇源堂召唤而来，NERV作战部长葛城美里带其到地下都市NERV总部，最后真嗣半推半就地驾驶着巨大人型兵器EVA初号机与使徒对抗。在没有受过任何训练的情况下，真嗣在战斗中很快不堪重负，导致初号机暴走，并以残酷野蛮的方式重创使徒，随后使徒自爆。之后真嗣在第3新东京市展开新生活，并在美里的提议下搬到美里家居住。

使徒不断地出现，在迎击使徒的过程中，真嗣结识了学校同学铃原冬二、相田剑介；并与同是驾驶员的綾波零、明日香联手消灭使徒。EVA3号机在启动试验中被使徒入侵，真嗣顾虑3号机内的驾驶员而不敢攻击，但在碇源堂的命令下真嗣的操纵被强制切断，搭载替代操纵系统的初号机将3号机残忍撕裂并重伤驾驶员铃原冬二。此次事件使真嗣决定不再驾驶EVA并打算离开第3新东京市。然而新出现的使徒打倒了明日香和綾波零继而攻进NERV总部，之后真嗣再次乘上初号机，最终觉醒的初号机将使徒吞食。另一方面，碇源堂与SEELE对人类补完计划的分歧显现、关系急剧恶化。

使徒开始尝试与人类接触并发动心理攻击。明日香因使徒的精神攻击而崩溃，綾波零为消灭使徒选择了自爆。之后失神的明日香被送入病院，綾波是複製人的事被真嗣得知，同学也已离开，孤独的真嗣遇到了渚薰。渚薰以第五适格者的身份被SEELE送到NERV接替明日香，出现后不久便成功打破真嗣的心之壁。但渚薰却是最后的使徒，真嗣必须杀死他。虽然渚薰的行动会灭亡人类而且他自己也期望死亡，但真嗣仍陷入深深地自责。

'''电视版25&26'''
: 电视版结局中，故事发展的描写较少，而是关注人物的心理问题、直接描写真嗣的内心世界。人类补完计划实行，所有人类的心合而为一，真嗣开始内心的补完。在自由的世界中，真嗣描绘了一个没有EVA存在、充满和平日常生活的世界，最终意识到现实世界拥有众多可能性。这时候，真嗣的世界豁然开朗，大家都向真嗣祝贺「恭喜你」。

'''剧场版25&26'''
: 《[[新世紀福音戰士劇場版|剧场版]]》中，SEELE开始入侵NERV的MAGI系统、挑动日本派出战略自卫队抹杀NERV，最后派出9台EVA量产机攻击觉醒的明日香驾驶的贰号机。真嗣在美里的鼓励下再次乘上初号机，在看到被肢解的贰号机后再次陷入混乱。随后补完开始、引发第三次冲击，所有人合而为一。然而，真嗣仍然期望他人存在的世界；最后，真嗣和明日香两个人躺在沙滩上。

電視版動畫以跳脫了劇情框架的型態完結。播放结束后的次年，重新制作的最终两话以《[[新世紀福音戰士劇場版：THE_END_OF_EVANGELION|新世紀福音戰士劇場版：THE END OF EVANGELION]]》的形式公映。因此《EVA》(1995-1997)有两个结局{{Sfn|Booklet|loc=Vol.08}}。

===企划===
《EVA》企划开始于1993年7月动画电影《苍之乌尔》（《[[王立宇宙軍|王立宇宙軍]]》续篇）制作冻结之后{{Sfn|武田康廣|2002|p=6}}。电影冻结之后，导演[[庵野秀明|庵野秀明]]陷入了深深的抑郁之中。而[[King_Records|King Records]]的制作人[[大月俊伦|大月俊伦]]对庵野讲「无论你有怎样的企划，我都让你通过」<ref>[http://khara.co.jp/hideakianno/ 庵野秀明官方網頁]{{ja}}</ref>，交谈之后决定合作制作一部原创电视动画{{Sfn|武田康廣|2002|pp=164–165}}。大月出于对庵野的信任，给予他全权。但也给出了两个条件：一、至少合作五年，二、不要杀死小孩子{{Sfn|Schizo|loc=第三章}}。庵野接受了大月的两个条件后，决定制作自己的原创作品，把自己的存在留存于影像<ref name=nt9501>''Newtype'' 1995年1月号. {{lang|ja|クリエイター対談「庵野秀明×貞本義行」}}</ref>。主题是「不能逃避」，这正是《苍之乌尔》冻结之后庵野的决心{{Sfn|武田康廣|2002|pp=164–165}}。亦呼应庵野在完成《[[冒險少女娜汀亞|冒險少女娜汀亞]]》后的四年间消沉生活<ref name="Wong">{{cite web|author=Amos Wong|url=http://www.aoianime.hu/evangelion/index.php?page=interanno|title=Interview with Hideaki Anno, director of 'Neon Genesis Evangelion'|date=January 1996|language=en}}</ref>。庵野本人在制作中宣称:
{{quote|「新世纪福音战士」是在我颓废了4年，什么事都不干之后，全心投入的作品。逃避了4年，就是死不了的我，秉承着唯一的信念『不能逃避！』的情况下再度创作的作品。想要让自己的心情表现在影片中而产生的作品。|庵野秀明<ref name=mh01>庵野秀明. ［我们到底想做什么？］　[[新世紀福音戰士_(漫畫)|新世紀福音戰士 (漫畫)]] 第一卷附文</ref>}}

1980s末日本动画开始呈现衰败。动画界充斥着无形的闭塞感，庵野秀明也感受到了、并且设法打破这种闭塞感，因此选择了电视这一更为大众的媒介<ref name=vhs00>录像带(VHS)第0卷（Genesis 0:0 IN THE BEGINNING） 庵野秀明 语</ref>。自创作开始以来，就有意扩大动画迷的数量<ref name="Woznicki" />，创造一个能够改变日本动画的新系列<ref name=EyesofAnno>{{cite web|author=Lawrence Eng|date=November 1996|title=In the Eyes of Hideaki Anno, Writer and Director of Evangelion|url=http://www.cjas.org/~echen/articles/spring97/05_03b.html|language=en}}</ref>。对于负责角色设计的[[贞本义行|贞本义行]]来说，《EVA》不同于以往的机器人动画，主角也呈现出不同的英雄形象<ref name=Sada>贞本义行Interview ［现在所思考的事］，[[新世紀福音戰士_(漫畫)|漫画]]第二卷附文。</ref>。庵野秀明也发表了类似的评论：机器人动画已经落伍了，以往的机器人动画总是使用同一种模式，我想打破这一模式<ref name=nt9504>''Newtype'' 1995年4月号，SKILL UP—新たなる挑戦／庵野秀明篇「幸福を求めて...」</ref>。在另一次采访中，导演声称创作《EVA》意在快乐并让动画爱好者开心，试图将尽可能多的观众聚集在一起。庵野以“神与人之间的战斗”为主题，提出将其命名为Alcion（アルシオン 发音为Arushion）。在贞本义行的建议下最终使用带有辅音的「Evangelion」，虽然被否定过一次<ref name=DerMond>貞本義行Interview，Der Mond―貞本義行画集 (Newtype illustrated collection). 1999, ISBN 4-04-853048-8</ref>。

===动画制作===
该系列以「Project Eva.」企划，由[[GAINAX|GAINAX]]制作，[[東京電視台|東京電視台]]和[[日本AD_SYSTEMS|NAS]]两公司出品，并由[[角川書店|角川書店]]和[[世嘉|SEGA Enterprises]]赞助播出。制作人[[大月俊倫|大月俊倫]]（[[King_Records|King Records]]）未表记，据其称「Project Eva.」只有他和庵野秀明二人{{Sfn|Schizo|loc=Staff座谈会}}。作画方面，GAINAX选择与[[龍之子|龍之子]]共同制作，个别话数外包给[[Production_I.G|Production I.G]]或[[吉卜力工作室|吉卜力工作室]]。庵野秀明担任导演和编剧，指导各方面的设定设计，并与[[山下育人|山下育人]]共同负责机械设计。贞本义行负责角色设计，[[鶴卷和哉|鶴卷和哉]]和[[摩砂雪|摩砂雪]]担任副导演。

在该系列播出两年之前，GAINAX制作了一份企划書{{Sfn|Chronicle|loc=No.01，p.26}}。企划书中的剧情大纲与播出的最终版本之间存在巨大差异。使徒的数量设定为28而不是播出的17。在最后，最强的第12使徒从月球飞来以及联合国随后决定放弃人类补完计划，以NERV的毁灭迎来最终明朗的结局。{{Sfn|Newtype 100% Collection|pp=85–88|loc=企画書}}

1994年9月，第一话、第二话开始作画；在1995年7月22、23日举行的第二届GAINAX祭上，公开上映了本作的前两话{{Sfn|Chronicle|loc=SIDE B，p.126}}。受到会场200余观众的欢迎{{Sfn|武田康廣|2002|pp=162–163}}。庵野导演也对此表示满意：前两话忠实地反映了我的心情，当我意识到这点时，我觉得这真是太好了<ref name=nt9504 />。
《EVA》的制作是一种即兴演出的感觉，无论是故事的构成还是人物的配置，都没能有条理地去做。一边进行着作业，一边听取着各种各样地意见，自己分析着自己地心理。<ref name=nt9606 /> 虽然人类补完计划在第二话就作为故事的主线出现了，但做到一半的时候还没有决定究竟要补完什么<ref name=Animage>庵野秀明、宮村優子Interview-{「あんた、バカぁと、言われてみたい。」}- ''[[Animage|Animage]]'' 1996年7月号</ref>。另外，制作这部作品时庵野还在考虑：
{{quote|比如说，过了二十岁还喜欢机器人动画或美少女动画的人真的能得到幸福吗？如果他一辈子都不知道还有更大的幸福在等着他，那么他也许也是幸福的。但是，我却对这种幸福感到疑惑。<br />对这样的人来说幸福究竟是什么？在制作这部作品的同时我想试着找出这个问题的答案。|庵野秀明<ref name=nt9504 />}}

据亲自采访过导演的[[東浩紀|東浩紀]]（日本文化评论家）称，[[奥姆真理教|奥姆真理教]]于1995年3月20日所做出的[[東京地鐵沙林毒氣事件|東京地鐵沙林毒氣事件]]，使庵野改变了部分剧情<ref name="Woznicki">Woznicki, Krystian. ''[http://www.nettime.org/Lists-Archives/nettime-l-9802/msg00101.html Towards a Cartography of Japanese Anime: Anno Hideaki's "Evangelion"]''. Last Updated 1998-02-20.</ref>。因为《EVA》中有一些奥姆风格的东西，不同于奥姆真理教的自我封闭，庵野合理化并将其升华了<ref name=STUDIOVOICE />。

因为东京地区会在1996年1月3日早上八点这一与平常不同的时间播出，所以有意把第十四话前半部分做成总集篇{{Sfn|Booklet|loc=Vol.04}}，后半部分决定对[[綾波零|綾波零]]进行描写<ref name=june2>庵野秀明 JUNE访谈Part II。最初登载于''别册JUNE'' 1996年11月号，后收录于《残酷な天使のように　新世紀エヴァンゲリオン JUNE読本》1997年8月 ISBN 4-906011-25-X</ref>。当时第十六话的剧本已经完成，鶴卷和哉在制作时提出在使徒与真嗣的接触中、不要让使徒用人的语言，庵野受此启发然后把主题决定为自己与自己的对话，以讨论究竟该如何与他人交流作为结局<ref name=Togetter />。庵野在想不出来的时候，看了精神病相关的诗集后一下子就写出了綾波的独白<ref name=june2 />。此次对内心世界描写的成功对之后的故事发展有很大的影响{{Sfn|Booklet|loc=Vol.04}}。

{{quote|以前没怎么看过精神分析的书，只是在大学基础课程上有过些许接触。是课程中最有趣的了。[...] 我其实对人不是很感兴趣。但开始描写内心独白时，途中想要找一些表达的词语。研究后发现最易于使用的是心理学用语。于是找了一些心理学的书籍来看。[...] 故事从第16话开始描写人物的内心世界。[...] 想不出来的时候，有个朋友借给了我一本『别册宝岛』出的精神病专题书，其中的诗集让我十分震撼，直冲大脑，顿时思如泉涌。|庵野秀明<ref name=nt9611>''Newtype'' 1996年11月号 <庵野秀明×上野俊哉></ref>}}

在正式播出前，只完成了12话的作画{{Sfn|Schizo|loc=第四章}}。
据导演说，最后的结局是在第十八、十九话的时候决定的。那时候他发现进度要赶不上了。他们在第十九话那里全力以赴，第十九话也是作画张数最多的一话。{{Sfn|Schizo|loc=第二章}}
本来计划要在第二十五话再做大规模的战斗场景{{Sfn|Schizo|loc=第四章}}，但也不愿意做出不满意的东西，因此取消了{{Sfn|Schizo|loc=Staff座谈会}}（直接跳过）。于是负责大规模战斗场景的副导演摩砂雪抽身制作本来应该是副导演鹤卷和哉负责的第二十四话，两人负责的话数对调了一下<ref>分镜集第三卷附摩砂雪访谈</ref>。由于制作过程产生了很多问题，副导演鹤卷和哉也感到了这种“紧张感”：

{{quote|我并不讨厌。由于制作日程表出现问题，作画枚数减少、质量不理想的地方也有。[...] 当整个制作体制即将崩溃时，虽然也有「如果无法做出满意的东西，那还有什么意义」这样的意见，但是我却不这么认为。「为什么不展示整个过程，包括我们的崩溃。」|鶴卷和哉<ref name="Tsurumaki">劇場版：The End of EVANGELION　场刊. 鶴卷和哉访谈［コミュニケーションの物語］ ''[https://web.archive.org/web/20060114182945/http://www.evaotaku.com/html/Tsurumaki.html A Story of Communication: The Kazuya Tsurumaki Interview]''</ref>}}

===最终两话与剧场版===
{{see|新世紀福音戰士劇場版}}
由于时间不够了，最后两话没有遵循原始剧本<ref name=RCB>劇場版：The End of EVANGELION　场刊</ref>{{Sfn|Schizo|loc=Staff座谈会}}。庵野改写了剧本，尝试新的表现手法。在第二十五话的后期录音结束时，最终话的剧本还没有完成，而第二周就全部完成了，实际的作画作业的时间只有3天。就表达来说庵野觉得甚至连画都没有必要去画，但这个想法被拒绝了。但庵野导演声称：TV播出的最后两话最直接地反映我当时的心境，对此感到很满足，完全不后悔。<ref name=nt9606 />
在2013年的一次采访中，他也声称最终回的26话原本就是那种结局<ref name=Togetter>{{cite web|url=http://togetter.com/li/580471|script-title=ja:大阪芸術大学「庵野秀明監督×小池一夫先生」特別講義まとめ|date=2013-10-22|language=ja}}</ref>。

自《EVA》播出以来，就不断受到许多影迷的支持。故事没有完结的TV版结局引发爆炸性的话题。影迷们对故事的完结、谜题的解答、新的剧情的呼声非常大。为了满足影迷的要求，于是决定重新制作第二十五话和最终话。这就是作为完结篇公开的'''[[新世紀福音戰士劇場版|新世紀福音戰士劇場版]]'''。<ref name=pamphlet>劇場版：死與新生　场刊</ref><ref name=RCB /> 分為電視版前24話的總集篇DEATH和重制的最终两话REBIRTH兩部分，由于时间上很紧张而先推出的《[[新世紀福音戰士劇場版：死與新生|新世紀福音戰士劇場版：死與新生]]》包括DEATH篇而REBIRTH篇仅有重制版第25话的前半部分；與電視版對比，DEATH篇多了20分鐘的新內容，新內容之后流入录像带版第21-24話<ref group="注" name=video>日语：ビデオ版 (video版)。英文区亦用：Director's Cut (导演剪辑版)</ref>。重制的第25話<Air>和第26話<真心为你>以《[[新世紀福音戰士劇場版：THE_END_OF_EVANGELION|新世紀福音戰士劇場版：THE END OF EVANGELION]]》于1997年3月15日在日本公映。之后又对DEATH篇进行了修正，1998年3月7日公映的《[[新世紀福音戰士劇場版：REVIVAL_OF_EVANGELION|新世紀福音戰士劇場版：REVIVAL OF EVANGELION]]》包括最终修正版<DEATH(TRUE)<sup>2</sup>>和第25話<Air>、第26話<真心为你>。
据后来[[庵野秀明|庵野秀明]]所述，原本25话打算作成剧场版25话那样，26话最终话则为电视版25话那样<ref name=jinken>{{Cite web|url=https://jinken-official.jimdo.com/講演録-会見録/庵野秀明-2000/ |title=早稻田大学人物研究会 庵野秀明访谈 |date=2000-12-05 |language=ja}}</ref>。因此有人认为《EVA》(1995-1997)存在三重结局<ref>{{Cite web |url = http://www.style.fm/as/05_column/animesama58.shtml |title = エヴァ雑記 |author =小黒祐一郎  |date = 2006-06-21 |publisher =AnimeStyle  |language = ja |quote = {{lang|ja|つまり、一つめのエンディングが第弐拾伍話、二つめのエンディングが最終話、三つめのエンディングが第25話と第26話ではないか。}} }}</ref>。

==设定设计==
《EVA》在[[宗教|宗教]]、[[生物学|生物学]]、[[哲学|哲学]]以及[[心理学|心理学]]等方面做了很多的引用<ref name=TheTimes>{{cite web|url=http://www.timeslive.co.za/sundaytimes/article183723.ece|title=DStv Pick of the week - Neon Genesis Evangelion : Monday, 15:45, Animax|author=Matthew Vice|publisher=[[The_Times|The Times]]|date=2009-11-15|archiveurl=https://web.archive.org/web/20100505191413/http://www.timeslive.co.za/sundaytimes/article183723.ece|archivedate=2010年5月5日|language=en|deadurl=yes|df=}}</ref>。本作中的引用还有来自《[[宇宙戰艦大和號|宇宙戰艦大和號]]》<ref name="Napier" />、《[[機動戰士GUNDAM|機動戰士GUNDAM]]》、《[[超人力霸王|奥特曼]]》、《[[不明飞行物_(电视剧)|UFO]]》{{sfn|Schizo|p=96}}、[[永井豪|永井豪]]的《[[惡魔人|惡魔人]]》{{sfn|Schizo|p=97}}等。而且早先打算将《EVA》作为《[[冒險少女娜汀亞|冒險少女娜汀亞]]》的某种[[续集|续集]]{{sfn|Schizo|loc=『エヴァ』の始まり（２）}}。
本作中几乎所有人物的名字都是庵野秀明导演取的，可以反映出导演的喜好与想法。其中碇真嗣的同学的名字取自于[[村上龍|村上龍]]的小说{{Sfn|FILMBOOK|loc=Vol.4，p.8–9}}，NERV工作人员的姓氏或名字来自于[[大日本帝國海軍艦艇列表|旧日本海军的舰船名称]]<ref>{{cite web|url=http://www.gainax.co.jp/hills/anno/essay1.html|title=庵野秀明の雑記|publisher=GAINAX|archiveurl=https://web.archive.org/web/20070220135947/http://www.gainax.co.jp/hills/anno/essay1.html|archivedate=2007年2月20日|language=ja|deadurl=yes|df=}}</ref>。

=== 標題 ===
该系列的[[日語|日語]]標題是'''{{lang|ja|新世紀エヴァンゲリオン}}'''（發音：''Shin Seiki Evangerion''），是日語“新世紀”與日語[[外來語|外來語]]“Evangelion”的結合。英文標題“Neon Genesis Evangelion”是一同推出的。
“Neon”語出[[希臘語|希臘語]]單詞''Neos''，意即“新”或者“新生”。
“Genesis”是從[[拉丁語|拉丁語]]中翻譯過來的（語源是希臘），意思是“起源”或“[[創世紀|創世紀]]”，首見於《[[聖經|聖經]]》，用於描述世界的創生或起源。
“Evangelion”是希臘語單詞“好消息”的英語化版本，在《聖經》中翻譯為福音（Gospel）。

=== 宗教 ===
[[File:Kircher_Tree_of_Life.png|thumb]]。]]
本系列中使用大量宗教符号，特别是[[基督教|基督教]]和[[犹太教|犹太教]]<ref name="TheTimes"/><ref name="Napier" />。各种宗教元素交织重组、转化为新的神话，但仍然与来源的传统宗教有着紧密的联系<ref name="Ortega">{{Cite journal|author=Mariana Ortega|year=2010|title=My Father, He Killed Me; My Mother, She Ate Me: Self, Desire, Engendering, and the Mother in Neon Genesis Evangelion|journal=Mechademia|volume=2|pages=216–232|isbn=978-0-8166-5266-2|doi=10.1353/mec.0.0010}}</ref>。例如作品中[[使徒_(新世纪福音战士)|使徒]]的命名大多源于宗教[[聖經|聖經]]中登场的[[天使|天使]]，名字和性质都有相关的情况也有{{Sfn|FILMBOOK|loc=Vol.5，p.81}}。还包括引用[[亞當|亞當]]、[[莉莉斯|莉莉斯]]、[[朗基努斯之枪|朗基努斯之枪]]、[[使徒|使徒]]、[[死海文書|死海文書]]以及[[卡巴拉|卡巴拉]][[生命樹_(卡巴拉)|生命之树]]等。[[Evangelion|EVA]]的造型设计是基于日本传统的「[[鬼_(日本)|鬼]]」<ref name="Wong" />。剧中一些措辞能让人联想到[[佛教|佛教]]语言<ref name="Napier" />。
副导演鶴卷和哉说基督教符号的使用是为了让这个系列更别具一格和充满神秘感，并没有宗教方面的含义。只是觉得基督教的视觉符号看起来很酷<ref>{{cite web|author=Owen Thomas|url=http://www.akadot.com/article/article-tsurumaki2.html|title=Amusing Himself to Death: Kazuya Tsurumaki speaks about the logic and illogic that went into creating FLCL|archiveurl=https://web.archive.org/web/20020110094207/http://www.akadot.com/article/article-tsurumaki2.html|archivedate=2002-01-10|language=en}} [http://www.evaotaku.com/html/evafaq.html EvaOtaku.com FAQ]</ref>，并且认为科学与宗教的混合会使这个系列更有趣。<ref>{{cite web|title=Interview mit Tsurumaki Kazuya (Studio GAINAX)|url=http://www.tomodachi.de/html/archiv/funime/f27_kurz_1.html|publisher=Anime No Tomodachi|language=de}}</ref>
在一次采访中，庵野秀明声称：只是让动画有一个氛围而已。作品中有很多关键词，但实际上就只是些符号而已，并不是一个一个都有含义，而当这些符号混杂在一起时才会出现相互关系或者说是内涵。<ref name=jinken />

=== 心理学和哲学 ===
《EVA》里所有角色都是庵野自身一部分的投影。在动画制作期间，庵野阅读了有关心理学的书籍，并产生极大兴趣<ref name=EyesofAnno />。依照庵野的设计：綾波零有[[精神分裂症|精神分裂症]]<ref name=june2 />、是[[潛意識|无意识]]的真嗣<ref name=jinken />；渚薰是他的[[荣格|荣格]][[心理陰影|阴影]]<ref name=nt9611/>、是理想的真嗣<ref name=june>庵野秀明 JUNE访谈Part I。最初登载于''小说JUNE'' 1996年8月号，后收录于《残酷な天使のように　新世紀エヴァンゲリオン JUNE読本》1997年8月 ISBN 4-906011-25-X</ref>。真嗣有[[恋母情结|恋母情结]]{{Sfn|Booklet|loc=Vol.08}}（恋母仇父）<ref name=anisama59>{{Cite web |url=http://www.style.fm/as/05_column/animesama59.shtml |script-title=ja:第59回　エヴァ雑記「最終話　世界の中心でアイを叫んだけもの」 |author=小黒祐一郎 |date=2006-06-22 |work=アニメ様の七転八倒 |publisher=AnimeStyle |language=ja }}</ref>。真嗣进入插入栓就好比是回归「母亲胎内」{{Sfn|Booklet|loc=Vol.04}}，从EVA初号机逃离被认为是真嗣的「[[成年禮|成年禮]]」<ref>{{cite web |url=http://www.animenewsnetwork.com/feature/2002-06-11 |title=Understanding Evangelion |author=Mike Crandol |date=2002-06-11|website=Anime News Network |language=en}}</ref>。
还包括许多对哲学和[[精神分析学|精神分析学]]的引用，如[[西格蒙德·弗洛伊德|弗洛伊德]]的[[口欲期|口欲期]]理论<ref name=june />、[[克尔凯郭尔|克尔凯郭尔]]的《[[致死的疾病|致死的疾病]]》、[[叔本华|叔本华]]的[[豪猪两难说|豪猪两难说]]<ref>{{cite web |author=Lisa Rivero |url=http://www.psychologytoday.com/blog/creative-synthesis/201201/social-media-and-the-hedgehogs-dilemma |title=Social Media and the Hedgehog's Dilemma |publisher=Psychology Today |date=2012-01-08 |language=en}}</ref>、[[心力内投|心力内投]]、矛盾心态{{Sfn|Booklet|loc=Vol.05}}、{{tsl|en|Death drive|死亡本能}}...

=== 典故和象征 ===
使徒是一个理想化抽象化、观念性的敌人，是登場人物们的危機感的[[具象化|具象化]]。形态从正八面体到电脑病毒无奇不有。使徒的描写不论是抽象或具象方面都达到了一个很高的水準。<ref name=Azuma>{{Cite web|url=http://www.ntticc.or.jp/pub/ic_mag/ic018/intercity/higashi_J.html |author=東浩紀|title=アニメ的なもの，アニメ的でないもの|language=ja|date=1996年}} ''[http://www.ntticc.or.jp/pub/ic_mag/ic018/intercity/higashi_E.html 英文: Anime or Something Like it]''</ref>
「人类补完计划」这个词源于美國科幻作家[[考德懷納·史密斯|林白樂]]的小说《Instrumentality of Mankind》<ref name="Horn">{{cite web|url=http://www.stanford.edu/~fenn/eva/eva1.html|title=Speaking Once as They Return: Gainax's Neon Genesis Evangelion|website=Stanford.edu|author=Carl G. Horn|language=en|archiveurl=https://web.archive.org/web/20120329122552/http://www.stanford.edu/~fenn/eva/eva1.html|archivedate=2012年3月29日|deadurl=yes|df=}}</ref>（不过二者的人类补完计划实际上大相径庭），诚然是一个科幻术语，但它的真面目正是我们现代人“欠损心灵的补完”<ref name=nt9606 />。
绝对领域（A.T.力场）一开始只是使徒的防护罩的设定，但后面深化为任何人都拥有的“心之壁”。与人类补完一样，增添了人文色彩。手法接近早期科幻作品：把抽象的东西以科幻设定具象化。

== 主题与风格 ==
===主题===
{{quote|主人公被设定为14岁的理由，是因为这是个“孩子以上、大人未满”的年纪。能靠自己一个人活下去，也能依附于他人而活。[...] 包含着这些理由，我认为14岁这一作为在精神上有独立可能的年龄，很适合这部作品的主题。|庵野秀明<ref name=nt9606/> }}

在《EVA》中「人际交流」是一个重要的主题{{Sfn|Booklet|loc=Vol.01}}。据副导演鹤卷和哉介绍，自制作开始就确定为一个关于人际交流的故事<ref name="Tsurumaki" />，导演庵野秀明亦称，《EVA》是一群不善于交流的人们之间的故事<ref name=nt9511>''Newtype'' 1995年11月号 <庵野秀明×三石琴乃×幾原邦彥></ref>。[[碇真嗣|碇真嗣]]来到第3新东京市后，与[[葛城美里|葛城美里]]同居、交流，转学后与同学的交流，与[[綾波零|綾波零]]以及父亲[[碇源堂|碇源堂]]的交流，与[[惣流·明日香·蘭格雷|明日香]]齐心协力打倒使徒。

从系列的中盘开始，「人心」成为主题<ref name=pamphlet />，通过主角们的意识流对人际交流进行思考。例如，在第十四话中，綾波零诗意般的长独白，各种断片的语言和影像。在真嗣的情况下，在第十六话和第二十话中存在类似的安排{{Sfn|Booklet|loc=Vol.05}}。这种叙事手法在该系列的最后两话中达到顶峰，「故事」和「主题」倒置，只对故事做了最低限度的说明，可以说是在叙述主题本身，故事发展完全靠登场人物的对话和独白{{Sfn|Booklet|loc=Vol.08}}。最终话的全篇故事也都在真嗣的心中世界展开，他对自己的价值以及跟他人的关系等等感到苦恼{{Sfn|Booklet|loc=Vol.08}}。

《EVA》这部作品，是由故事层面以及庵野导演自身心灵旅程的直播记录，这两个侧面构成的<ref name=nt9606/>。庵野秀明认为动画爱好者（[[御宅族|御宅族]]）太内向、并且在封闭自己，于是向动画爱好者发出劝诫。当然，这并不是动画爱好者所独有的。<ref name="Tsurumaki" /> 播放结束后，庵野对动画爱好者喊出：回到现实吧<ref name=nikkeibp9704 />。但之后也称：并不是批判，只是在思考。或许也有人觉得那是多管闲事吧。<ref name=Togetter/> 据一些评论，《EVA》的情节和人物反映了导演四年的抑郁生活<ref>{{cite web|author=David Samuels|url=http://www.theatlantic.com/magazine/archive/2007/05/let-s-die-together/305776/3/|title=Let's Die Together|date=2007-05-01|work=[[大西洋_(雜誌)|The Atlantic Monthly]]|language=en}}</ref><ref name=top25>{{cite web|author=Ransey Isler|url=http://www.ign.com/articles/2014/02/04/top-25-greatest-anime-characters?page=5|title=Top 25 greatest anime characters|publisher=[[IGN|IGN]]|date=2014-02-04|language=en}}</ref>，其中多次强调了作品的自传性：《EVA》是把我的人生照搬到影像中。我还活着，所以故事也不会终结。{{Sfn|Schizo|loc=第二章}}

===风格===
《EVA》的特点是频繁使用实验性的表现手法；例如在第十六集中，真嗣的内心世界对话是在黑色背景上用移动的白色的「线」表示角色，在插入各种画面等等{{Sfn|Booklet|loc=Vol.04}}。在第二十集中，在真嗣的内心独白期间，显示了彩色斑点、他所认识的人的照片以及不同手绘草图<ref>{{cite book|author=Dani Cavallaro|title=The Art of Studio Gainax Experimentation, Style and Innovation at the Leading Edge of Anime|pages=69-70|date=2009|language=en|isbn=978-0-7864-3376-6}}</ref>。据Dani Cavallaro说：《EVA》向前迈进了前所未有的一步，涉及[[传统动画|赛璐璐动画]]和[[计算机图形学|CG]]的融合、手绘动画和数字特效的融合<ref>{{Cite book |date=2007 |author=Dani Cavallaro|title=Anime Intersections. Tradition and Innovation in Theme and Technique|pages=18|language=en|isbn=0-7864-3234-9}}</ref>。
最终话中，使用分镜的画来代替赛璐璐画的部分，那是故意这么做的，并不是因为时间不够。而是因为制作者想表达：动画作为表现的媒介，只要有线画就能发挥机能了。<ref name=nt9606 /> 这是庵野导演在业余时代的经验{{Sfn|Parano|loc=第一章}}。常常使用简单但黑暗的图像和照片剪辑、令人不安的画外音和让人心生疑惑的音乐<ref name="Napier">{{cite web |author=Susan J. Napier |title=当机器休止：《新世纪福音战士》和《玲音》中的幻想，现实和终极身份 |others=陈荣钢 (译) |url=http://www.aisixiang.com/data/99392.html }}</ref>。

从1984年到《EVA》发行之时，受到高度赞扬的日本动画总是有一种远离“传统”动画风格的趋势<ref name=Azuma />。而《EVA》是彻底的日本动画<ref name="Woznicki" />，不仅包含典型御宅文化所具有的科幻设定机器人和美少女[[殺必死|殺必死]]，还像GAINAX以往的作品一样进行了许多[[恶搞|恶搞]]与[[致敬|致敬]]，特别是一些科幻作品的影响。<ref name="Horn" />
庵野不想把《EVA》做成像真人电影的动画，于是尝试在有限的帧数内展现动画抽象的极致。与《[[亚基拉_(电影)|阿基拉]]》和《[[攻殼機動隊_(電影)|攻壳机动队]]》中的将巨量细节压缩入一帧的手法不同的是，《EVA》经常有长时间静止的或缓慢移动的画面。<ref name="Woznicki" />
《EVA》大量使用静止的图像，有时完全静默，有时伴有画外音。威廉•劳特分析认为，突然静止的画面能让动画牢牢抓住观众的注意力。<ref name="Napier" />

《EVA》的画面简洁，但是却充满了紧张感。受日本导演[[岡本喜八|岡本喜八]]的影响，庵野在动画中加入了节拍：单个画面的信息量有限，通过镜头的快速切换对传达给观众的信息量进行控制<ref name=STUDIOVOICE />。据导演称，《EVA》的信息量过大，该作品能反映出每个观看该作品的人{{Sfn|Schizo}}。
这部作品允许甚至要求读者做出多重解读<ref name="Ortega" />。庵野亦称：《EVA》本身就像一个拼图游戏。任何人都可以看到它并给出自己的诠释。换言之，我们为观众提供了一个自主思考的空间，让每个人都能想象自己的世界。<ref name=EyesofAnno />

'''中國大陆'''：在《EVA》剛剛問世時，它就通過[[盗版|盗版]]被動漫愛好者所熟知並在中國內地迅速走紅。在2000年，由[[深圳电视台|深圳电视台]]引进，并交由[[遼寧人民藝術劇院|遼寧人民藝術劇院]]譯製配音。然而，辽艺第一次译製的版本并没有通过[[广电总局|广电总局]]的审批，因而被迫再次译配，最终以《新世纪天鹰战士》的面貌于2001年在大陆各电视台播出。为通过审批而进行的大量刪節和台词修改、因大量人才流失而出现的译配不到位等问题受到观众们的廣泛批評，「天鹰战士」這個名稱也沒有在中國大陆的動漫界傳開去<ref>{{cite web |title=再谈国漫声优：他曾为EVA、龙珠配音，执导了《大圣归来》 |url=https://new.qq.com/omn/20171228/20171228A0GIOW.html |website=腾讯网}}</ref>。之后以第一版发行了VCD，即目前网络上流传的《2000天鹰战士》<ref>文像進字（2000）724號，參見 [https://web.archive.org/web/20070929010911/http://www.ccm.gov.cn/gongshi/lyjmlsb/sanji.php?id=9374 进口音像制品数据查询系统]</ref><ref group="注">VCD版片头动画中标识为“2000天鹰战士”，但正式商品名称仍使用“新世纪天鹰战士”</ref>。

== 评价及影响 ==
《新世纪福音战士》（以下简称《EVA》）在日本国内和国际上获得了巨大的人气<ref name=PJ>{{cite web |url=http://news.livedoor.com/article/detail/2231472/ |script-title=ja:永久不滅のエヴァ人気 |author=安田モモコ |publisher=PJ News|date=2006-07-23 |archiveurl=https://web.archive.org/web/20080501050708/http://news.livedoor.com/article/detail/2231472/ |archivedate=2008-05-01 |language=ja}}</ref><ref name="Revolutions">{{cite web|author=Lawrence Eng|url=http://www.cjas.org/~leng/revolution.htm|title=A look at "The Four Revolutions of Anime"|date=2004-10-20}}</ref>。并且超出了单纯的人气作品，发展成为日本的社会现象<ref name=anibu>{{cite web |script-title=ja:『 新世紀エヴァンゲリオン 』エヴァはなぜ社会現象となったのか |url=https://anibu.jp/oshiete-evangelion-368.html |website=あにぶ |language=ja |date=2014-05-06}}</ref>，获得了众多媒体的报道、成为讨论和研究的课题。对日本动画界以及大众文化产生了重大影响。

===反響===

《EVA》复杂而引人入胜的故事<ref name=TheTimes/>、华丽的战斗场面、出色的人物设定、精緻的心理描写、嶄新而熟练的演出技法<ref name=anibu/>，以及制作方有意地制造悬念和谜题，使得粉丝们在观看时非常期待着之后的展开<ref name=TheTimes/>。播出當時就獲得愛好者熱烈支持{{Sfn|Booklet|loc=Vol.01}}，1996年2月3日发售的第一卷录像带引发粉丝抢购<ref name=nikkeibp9704>[https://web.archive.org/web/20010212010121/http://ent.nikkeibp.co.jp/ent/bn/9704/report/evan.html 新世紀エヴァンゲリオン“騒動”小史]. 日经娱乐 1997年創刊号{{ja}}</ref>，而在此之前发售的第0卷特别先行篇获得[[Oricon公信榜|Oricon公信榜]]1996年度[[VHS|VHS]]动画类销量冠军。
播放结束后，实验性的跳脱故事的最终回成为激烈讨论的话题，评价呈现两极分化<ref name=nikkeibp9708>[https://web.archive.org/web/20010210203739/http://www.acsys.com/~tallman/rebirth.html 公開直前『エヴァンゲリオン』＆『もののけ姫』]. 日经娱乐 1997年8月号</ref><ref name=Solomon>{{cite web|author=Charles Solomon|url=http://articles.latimes.com/2002/apr/10/entertainment/et-solomon10|title=Anime Series Draws on a World of Alienation|work=[[Los_Angeles_Times|Los Angeles Times]]|date=2002-04-10}}</ref><ref name="Revolutions"/>。当评论家[[大塚英志|大塚英志]]在[[讀賣新聞|讀賣新聞]]刊文批评结局时，辩论进入全国范围<ref name=Legacy>{{cite web|title=Evangelion Special: From phenomenon to legacy |publisher=[[每日新聞|每日新聞]] |language=en |url=https://forum.gaijinpot.com/forum/community-center/latest-japan-news/19979-evangelion-special-from-phenomenon-to-legacy?21810-Evangelion-Special-From-phenomenon-to-legacy= |date=2006-05-06 }}</ref>。
尽管最终回引发了争议，但《EVA》的[[作者论|作家性]]以及时代同步性吸引了公众和评论界的注意、纷纷发表评论分析<ref name=Anitama>{{cite web |author1=高瀬司 |title=《新世纪福音战士》论 |url=https://zhuanlan.zhihu.com/p/27731058 |website=知乎专栏 |publisher=Anitama}}</ref>，它的巨大市場特性和經濟效益也在經濟雜誌被提及，對日本社會產生了強烈的影響<ref name=proware/>。

日本动画杂志[[Animage|Animage]]在1996年和1997年进行的两次年度调查投票中，《EVA》被选为最佳动画<ref>{{cite web|url=http://animage.jp/old/gp/gp_1996.html|title=第18回アニメグランプリ ［1996年5月号］|website=Animage|language=ja|archiveurl=https://web.archive.org/web/20121130222457/http://animage.jp/old/gp/gp_1996.html|archivedate=2012-11-30}}</ref><ref>{{cite web|url=http://animage.jp/old/gp/gp_1997.html|title=第19回アニメグランプリ ［1997年6月号］|website=Animage|language=ja|archiveurl=https://web.archive.org/web/20121130215431/http://animage.jp/old/gp/gp_1997.html|archivedate=2012-11-30}}</ref>。1998年，《[[新世紀福音戰士劇場版|新世紀福音戰士劇場版]]》亦票选为第一位<ref>{{cite web|url=http://animage.jp/old/gp/gp_1998.html|title=第20回アニメグランプリ ［1998年6月号］|website=Animage|language=ja|archiveurl=https://web.archive.org/web/20121130221727/http://animage.jp/old/gp/gp_1998.html|archivedate=2012-11-30}}</ref>。成为第一个连续三次获得该奖项（Anime GrandPrix）的作品。
在1996年第一回[[神戶動畫獎|神戶動畫獎]]，《新世纪福音战士》和庵野秀明分别被评为最佳电视动画和最佳导演；以及第二年《新世紀福音戰士劇場版》获得会場特別賞、衍生游戏《[[新世紀福音戰士：鋼鐵戀人|新世紀福音戰士：鋼鐵戀人]]》当选年度最佳网路媒体作品。<ref name="kobe">{{cite web |url=http://www.anime-kobe.jp/archive/index.html |script-title=ja:アニメーション神戸-これまでの記録 |deadurl=yes |archiveurl=https://web.archive.org/web/20090419102114/http://www.anime-kobe.jp/archive/index.html |archivedate=2009-04-19 }}</ref> 在1997年《EVA》荣获第18回[[日本SF大獎|日本SF大獎]]<ref name=SFWJ>[http://www.sfwj.or.jp/list.html 日本SF大賞 各賞受賞一览]{{ja}}</ref>、第1回[[文化廳媒體藝術祭|文化廳媒體藝術祭]]动画优秀奖<ref>[http://archive.j-mediaarts.jp/festival/1997/animation/ 第1回文化庁メディア芸術祭アニメーション部門 受賞作品]</ref>。在2006年日本文化廳媒體藝術祭所举办的{{Link-ja|日本媒體藝術100選|日本のメディア芸術100選}}投票活动中，《EVA》获得动画第一位<ref name=hundred>{{Cite web |url=http://plaza.bunka.go.jp/hundred/bumon_anime.html |script-title=ja:日本のメディア芸術100選　結果発表 |work=文化庁メディア芸術祭10周年企画|archiveurl=https://web.archive.org/web/20061030110322/http://plaza.bunka.go.jp/hundred/bumon_anime.html |archivedate=2006-10-30 }}</ref>。

不仅在日本在其他地区，《EVA》也获得认可和欢迎。2002年时就有12种语言的800多个网站专门用于《EVA》<ref name=Solomon/>。日本貿易振興機構于2007年发布的調査報告显示[[意大利|意大利]]人气动画No1是《EVA》<ref>{{cite web |script-title=ja:イタリア人気アニメNo1は 「エヴァンゲリオン」 JETRO調査 |url=https://animeanime.jp/article/2007/07/09/1987.html |website=アニメ！アニメ！ |language=ja}}</ref>。在Madman网站的"日本动画Top20"调查投票中排名第一<ref>{{cite web |title=The Fans Have Spoken: Neon Genesis Evangelion Is Number One With A Bullet |url=http://www.animenewsnetwork.com/press-release/2012-05-10/the-fans-have-spoken/neon-genesis-evangelion-is-number-one-with-a-bullet |website=Anime News Network |date=2012-05-10 |language=en}}</ref><ref>{{cite web |title=#1 Neon Genesis Evangelion {{!}} Madman Top 20 Anime Titles |url=https://archive.is/PyOfV |website=archive.is |publisher=Madman Entertainment |date=2013-11-17}}</ref>。[[IGN|IGN]]网站将《EVA》列为其“电视动画Top100”排行榜中的第十位<ref>{{cite web|title=10. === 评价 ===
《EVA》在动画界掀起了十数年未见的大浪<ref name=nt9606>''Newtype'' 1996年6月号 -{「エヴァ、再擧」}-</ref>，普遍认为这是继《[[宇宙戰艦大和號|宇宙戰艦大和號]]》、《[[機動戰士GUNDAM|機動戰士GUNDAM]]》后的第三次动画革命<ref name=PJ/>。引来了众多评论家和学者对其发表分析评论，包括[[大塚英志|大塚英志]]、[[Susan_J._Napier|Susan J. Napier]]、[[小谷真理|小谷真理]]<ref name="Napier" />、[[宮台真司|宮台真司]]、[[東浩紀|東浩紀]]、[[村上隆|村上隆]]和[[宫崎哲弥|宫崎哲弥]]等等。
日本评论家大塚英志和宫崎哲弥指责最终两话就像是[[LGAT課程|自我启发研讨]]（类似洗脑和心理治疗），不过是对[[御宅族|御宅族]]因逃避现实而自闭的一种自我肯定<ref name=Prison>{{cite web|url=http://www001.upp.so-net.ne.jp/tsuribe/anime/critique/evazh.html |title=自我意识的牢狱：论《新世纪福音战士》|author=Manabu Tsuribe |date=1999年2月}}</ref>。東浩紀则认为最终两话是类似[[後設小說|後設小說]]的二重性所表现出的批判和自省，并且认为《EVA》在某种意义上是1980年代日本动画的完成形态<ref name=subculture>東浩紀. -{「庵野秀明は、いかにして八〇年代日本アニメを終わらせたか」}-. 1996年</ref>。
不少社会学者参与《EVA》讨论，并把其评为反映当代日本的镜子般作品<ref name=Anitama/>。[[贞本义行|贞本义行]]认为《EVA》创纪录的大热主要是因为它偶然发现了这个时代正在寻找的东西<ref>{{cite web|title=[EVA] Sadamoto Interview |url=http://eva.onegeek.org/pipermail/oldeva/1999-January/024322.html |date=1999 |language=en}}</ref>。

该系列被许多学者视为杰作<ref name="Napier" />，被日本以及欧美的评论家誉为90年代具有里程碑意义的日本电视动画系列<ref name="Wong"/>。美国学者Susan J. Napier认为《EVA》解构了机械科幻类作品，是史上最重要和最具开创性的日本动画之一。通过展示十四岁的[[碇真嗣|碇真嗣]]这一[[反英雄|反英雄]]形象，打破了传统的"拯救世界"叙事。但其真正具有开创性的是对人物心理斗争的复现，最后两集摆脱了华丽的战斗场面以及传统的末世剧情，取而代之的是人物内心的挣扎和自我剖析，Napier对此赞美不已。<ref name="Napier" />
《[[纽约时报|纽约时报]]》的Mike Hale将其描述为"一部优秀的动画作品，具有不同寻常的深度、情感和细节的巨大机器人故事"<ref>{{cite web|author=Mike Hale|title=Evangelion 1.0: You Are (Not) Alone url=http://www.animenewsnetwork.com/review/neon-genesis-evangelion/dvd-3| publisher=Anime News Network|date=2005-03-18}}</ref>。Mike Crandol表示：《EVA》无疑是日本动画史上最伟大的作品之一<ref name=Crandol>{{cite === 影响 ===
[[File:Fancy_Frontier_22_(9383523595).jpg|缩略图]]
《EVA》是少有的能够引起全年龄段大规模公开讨论的动画，產生了爆炸性的反響，还重新點燃了一大批成年人對動畫的熱愛<ref name=nt9606 />。对日本动画界产生了相当大的影响，缔造了第三次动画热潮<ref group="注">第一次动画热潮(1963至1960s末〜1972)：电视动画《[[铁臂阿童木|铁臂阿童木]]》开始的电视动画热潮；第二次动画热潮(1977至1984〜1988)：《[[宇宙战舰大和号|宇宙战舰大和号]]》剧场版、《[[機動戰士GUNDAM|機動戰士GUNDAM]]》以及[[好莱坞|好莱坞]]科幻潮影响的科幻・机器人动画热潮。第三次动画热潮从1995年《EVA》开始、延续至今。</ref>，使得即便是非常规的动画企划也变得容易通过<ref>{{cite web |script-title=ja:今だからこそリアルに感じるアニメ「lain」の世界観 |url=https://npn.co.jp/article/detail/20210193/ |website=リアルライブ |language=ja}}</ref>。并且对机器人动画进行了历史性革新，推动动画商业模式从商品化主导向[[制作委员会方式|制作委员会方式]]转换<ref name=Anitama/>。

该系列也对日本大众文化产生了显著的影响。据东京电视台制作人岩田圭介表示由于《EVA》的流行，日本动画在全球急速传播<ref>{{cite web|title=TV Tokyo's Iwata Discusses Anime's 'Road to Survival'|url=http://www.animenewsnetwork.com/news/2009-01-29/tv-tokyo-iwata-discusses-anime-road-to-survival|publisher=Anime News Network|date=2009-01-29}}</ref>。由于这部动画的爆红，在社会广大的注目下，[[御宅族|御宅族]]文化才慢慢开始表面化<ref>{{Cite book|author=東浩紀|authorlink=東浩紀 |title=動物化的後現代：御宅族如何影響日本社會 |origyear=2001|year=2012|publisher=大鴻藝術 |others=褚炫初 (譯) |isbn=978-986-86764-6-6|pages=12 }}</ref>。《EVA》促成了“第三代御宅族”的诞生，小说家[[泷本龙彦|泷本龙彦]]就是其中之一<ref name=Legacy/>。由于此作的流行，[[Cosplay|Cosplay]] 得以传遍全球<ref>{{cite web|author=Inui Tatsumi|url=http://www.nippon.com/en/column/g00255/|title=The Expanding Cosplay Universe|date=2015-03-06|language=en}}</ref>，受到歧视的“御宅族”变得流行<ref>{{cite web|author=Carl Gustav Horn|url=http://archive.wired.com/techbiz/media/magazine/16-04/st_15eva|title=15th Anniversary: The Cultural Legacy of Neon Genesis Evangelion|website=[[Wired|Wired]]|date=2008-03-24|language=en}}</ref>。此外，《EVA》还在其它动漫中造成了一次使用基督教符号的热潮。
该系列的影响力也存在于音乐领域，英国乐队[[Fightstar|Fightstar]]从该系列获得灵感创作了专辑《[[Grand_Unification|Grand Unification]]》<ref>{{cite web|author=Liz Kennedy|date=2006-05-31|title=Chart-Topper Charlie Turns to Hardcore|url=http://www.questia.com/library/1G1-146666565/chart-topper-charlie-turns-to-hardcore|work=The News Letter|language=en}}</ref>，日本乐队 Rey 其名便取自剧中人物綾波零<ref>{{cite web|url=http://www.oricon.co.jp/news/music/60445/|title=イケメンアニソンバンドがメジャーデビュー|publisher=Oricon.co.jp|date=2008-11-25}}</ref>以及[[BUMP_OF_CHICKEN|BUMP OF CHICKEN]]的《アルエ》、[[彩虹乐团|彩虹乐团]]的《あなた》<ref>{{cite web|url=http://anibu.jp/eva-character-song-17986.html|script-title=ja:実はけっこうある!?「エヴァ」キャラへ向けた歌！|date=2015-06-17|language=ja}}</ref>。主题曲《[[残酷天使的行动纲领|残酷天使的行动纲领]]》常年居于日本卡拉OK点唱榜前十。

剧中[[新世紀福音戰士名詞列表#暴走|暴走]]一詞成為「失去控制」的新代名詞，[[绝对领域|绝对领域]]（[[新世紀福音戰士名詞列表#A.T.力场|A.T.力场]]）在新时代也有了新的诠释、在[[香港|香港]]拿来恶搞创作出[[李氏力場|李氏力場]]。
《EVA》也被众多作品[[戲仿|戲仿]]、[[恶搞|恶搞]]和[[致敬|致敬]]，如：《[[FLCL|FLCL]]》、《[[阿倍野桥魔法商店街|阿倍野桥魔法商店街]]》、《[[美少女學園|美少女學園]]》、《[[旋風管家|旋風管家]]》<ref>{{cite web|title=Hayate the Combat Butler|author=Theron Martin|url=http://www.animenewsnetwork.com/review/hayate-the-combat-butler-gn-1|date=2006-12-04|publisher=Anime News Network}}</ref>、《[[笨蛋，測驗，召喚獸|笨蛋，測驗，召喚獸]]》<ref>{{cite web|title=Baka and Test|url=http://www.animenewsnetwork.com/review/baka-and-test/blu-ray+dvd-complete-set|author=Theron Martin|date=2011-09-23|publisher=Anime News Network}}</ref>、《[[Keroro軍曹|Keroro軍曹]]》以及美国动画《[[天兵公園|天兵公園]]》<ref>{{cite web | url=https://www.animenewsnetwork.com/interest/2015-04-10/neon-genesis-evangelion-opening-parodied-on-regular-show/.86964 |author=Lynzee Loveridge |title=Neon Genesis Evangelion Opening Parodied on Regular Show |publisher=Anime Netws Network |date=2015-04-10}}</ref>、《[[史帝芬宇宙|史帝芬宇宙]]》<ref>{{cite web|url=http://www.animenewsnetwork.com/the-list/2017-02-04/.111753|title=7 Anime References in American Cartoons|author=Lynzee Loveridge|publisher=Anime Netws Network |date=2017-02-04}}</ref>；还有日本电视剧《[[逃避雖可恥但有用|逃避雖可恥但有用]]》<ref>{{cite web |script-title=ja:なぜ『逃げ恥』『カルテット』は人気を得た？TBS土井裕泰が語る |url=https://www.cinra.net/interview/gyoukairetsuden/vol10-doinobuhiro |website=CINRA.NET |language=ja}}</ref>、美国电影《[[不速之客_(2002年電影)|不速之客]]》<ref>{{cite web|url=http://www.animenewsnetwork.com/news/2014-08-12/comedian-actor-robin-williams-passes-away-at-63/.77541|title=Comedian/Actor Robin Williams Passes Away at 63|date=2014-08-12|website=Anime News Network|language=en}}</ref>等。

Bisty與《新世紀福音戰士》合作，於2004年推出的[[柏青哥|柏青哥]]機台《CR新世紀福音戰士》，售出了10萬台以上。「[[北斗神拳|北斗神拳]]」與「新世紀福音戰士」機台的成功，讓各家柏青哥機台業者積極將動漫作品改編成柏青哥的機台。<ref>{{cite web |author=李世暉 |title=日本博弈默示錄 |url=https://www.ettoday.net/news/20170130/858110.htm |website=ETtoday新聞雲 |date=2017-01-30}}</ref>柏青哥機的大热也使一些不看动画的人知道了该系列，在之后的10多年里，EVA柏青哥以及[[柏青嫂|柏青嫂]]系列機台的累计销量超过200万台，创造了大约7000亿日圓的市场<ref>{{cite news |title=The Future of Fields |url=http://www.fields.biz/ir/e/investors/library/annual/online2015/coo_message.html |work=ONLINE ANNUAL REPORT 2015 |publisher=Fields Corporation |language=ja,en}}</ref>。此外还以實體的柏青哥機為基礎製作了多款柏青哥遊戲<ref>{{cite web |title=人家才不是在赌博啦——谈谈日本的弹珠机 |url=http://www.appgame.com/archives/247383.html?am_force_theme_layout=desktop&cachever=1538391559843 |website=任玩堂 |date=2014-04-04}}</ref>。

作为《福音戰士新劇場版》宣传合作的一环，RUN'A娱乐公司于2010年成立了「EVA賽車隊」（EVA Racing）<ref>[http://eva-racing.com/ EVA Racing]{{ja}}</ref>，有四輪賽車和二輪賽車，并以EVA做為車身彩繪範本，参战了[[Super_GT|Super GT]]300級、[[鈴鹿8耐|鈴鹿8耐]]等赛事<ref>{{cite web |title=Racing RUNA EVANGELION Racing-3年不見Super GT重返發表 |url=https://carnews.com/article/info/1ea9251a-4b0d-11e8-8ee2-42010af00004 |website=車訊網 |date=2016-05-23}}</ref>。现场还有賽車女郎[[Cosplay|Cosplay]]剧中的主要角色，车队官网也是EVA风格，并且推出了相关商品<ref>{{cite web |title=EVA RACING、来袭 |url=https://chejiahao.autohome.com.cn/info/2239032 |publisher=与车同乐 |date=2018-03-23}}</ref>。

2015年，[[西日本旅客鐵道|JR西日本]]新干线运营公司与《EVA》合作推出「新幹線：新世紀福音戰士計畫」，以纪念[[山陽新幹線|山陽新幹線]]运营40周年及《EVA》电视播出20周年。推出的列车「500 TYPE EVA」以[[新幹線500系電聯車|新幹線500系電聯車]]为蓝本，并由《EVA》機械設計師[[山下育人|山下育人]]负责车身的设计、导演庵野秀明担任監修，车厢内部以新世紀福音戰士的世界观为主题打造<ref>[http://www.500type-eva.jp/ 500 TYPE EVA] {{Wayback|url=http://www.500type-eva.jp/ |date=20150723184356 }}</ref>。于2015年11月正式开始运营，原定于2017年3月结束运行，但因深受好评，最终延长1年至2018年5月13日<ref>{{Cite web |url=http://nlab.itmedia.co.jp/nl/articles/1801/19/news117.html |script-title=ja:エヴァ新幹線「500 TYPE EVA」が5月13日に引退　山陽新幹線こだまの500系特別列車 |work=ねとらぼ |publisher=ITmedia |date=2018-01-19 |archiveurl=https://web.archive.org/web/20180119164347/http://nlab.itmedia.co.jp/nl/articles/1801/19/news117.html |archivedate=2018-01-19}}</ref>。

《EVA》还与[[富士急高原樂園|富士急高原樂園]]、[[日本環球影城|日本環球影城]]合作推出了相关主题的游乐设施<ref>{{cite web |title=初號機爆米花桶重出江湖 福音戰士新設施環球影城亮相 |url=https://www.nownews.com/news/20180521/2757764/ |website=NOWnews 今日新聞 |date=2018-05-21}}</ref>。
2017年與[[宏佳騰|宏佳騰]]合作推出全球限量300台的聯名車款「ES 150：Type 01」，運用了EVA初號機的形象來設計，這是EVA首次授權給台灣車廠<ref>{{cite web |title=宏佳騰攜手新世紀福音戰士推限量車款 擦出怎樣的火花？ |url=https://autos.udn.com/autos/story/10137/2261340 |website=發燒車訊 |date=2017-02-02}}</ref>，也是EVA全球唯一聯名量產車<ref>{{cite web |title=【宏佳騰專訪番外篇】700多封郵件往返 台灣車廠打造這輛全球唯一的聯名款 |url=https://www.mirrormedia.mg/story/20180305bus008/ |website=鏡週刊 Mirror Media |date=2018-03-05}}</ref>。原本只在台灣銷售，後來还帶到[[AnimeJapan|AnimeJapan]]展出，結果造成動漫迷的熱烈回響、並爭相拍照<ref>{{cite web |title=轟動日本動漫展！宏佳騰新世紀福音戰士限量車吸睛 |url=https://autos.udn.com/autos/story/10139/2456604 |website=發燒車訊 |date=2017-05-11}}</ref>。
