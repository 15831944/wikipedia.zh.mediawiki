'''空当接龙'''（{{lang|en|FreeCell}}）是一款使用52张牌进行的单人[[卡片游戏|卡片游戏]]，游戏目标是将整副牌搬到右上角的四个空栏中。尽管Windows游戏帮助文档中声称“所有的牌局最终都可解开”，但实际上有十个牌局是无法破解的<ref name="intro">{{en}}{{cite web |url=http://people.cs.uu.nl/hansb/d.freecell/node1.html#SECTION0001 |title=空当接龙游戏介绍 |accessdate=2009-10-10}}</ref>。

最初开发空当接龙的目的是为了测试32位的数据处理子系统，如果系统没有正确安装，空当接龙将不会运行<ref>{{cite news|title=The True Purpose of Microsoft Solitaire, Minesweeper, and FreeCell|url=http://mentalfloss.com/uk/technology/32106/the-true-purpose-of-solitaire-minesweeper-hearts-and-freecell|accessdate=2015-12-23|publisher=Mental Floss|date=2015-08-14|archive-url=https://web.archive.org/web/20151223192108/http://mentalfloss.com/uk/technology/32106/the-true-purpose-of-solitaire-minesweeper-hearts-and-freecell|archive-date=2015-12-23|dead-url=yes}}</ref>。

== 規則 ==
將一副標準52張[[撲克牌|撲克牌]]洗牌，再分別置於8個欄目，其中4個欄目有7張紙牌，其餘4個欄目有6張紙牌，每張[[撲克牌|撲克牌]]皆要以翻開顯示。

另外還有右上角四個本位欄框、左上角四個空白欄框。紙牌 A、2 可立即放到本位欄框，而其他同花色的牌則可以由小到大依序疊上去。只要將所有的牌都放到本位欄框中，就成功了。空白欄框為四個放牌位置。每個欄框都可任意放一張紙牌。

移動紙牌的規則如下：
*將紙牌移動到某一欄時的順序必須為由大到小，而且是不同的顏色（即黑、紅兩種顏色）。 
*將紙牌移至本位欄框時，必須以相同花色，將牌按照從低（A）到高（K）的順序移動。 
*某欄最底端的紙牌可移到空白欄框、另一欄的最底端或本位欄框。 
*空白欄框中的紙牌可移至某一欄的最底端或本位欄框。

== 無解牌局 ==
Windows 98的新接龍共有32000關，但存在利用程序缺陷而通过的方法。到了XP系統時增加至1000000關，其中又牌局#146692、#186216、#455889、#495505、#512118、#517776、#781948是無法破解的。到了Windows 8的Microsoft Solitaire Collection時增加至9999999關。另外也有兩個隱藏關卡，分別是牌局#-1、#-2，同樣也無法破解。Windows Vista增加兩個隱藏關卡，分別是牌局#-3、#-4，為必勝關卡。

新接龍還保留一種類似[[秘技|作弊程式]]的破解方法，同時按住[[鍵盤|鍵盤]]上的Ctrl-Shift-F10，出現視窗後按下視窗中的“-{zh-hant:中止; zh-hans:终止;}-”鈕再來只要再移動任意一張牌就可過關，按“重試”則移動任何一步都會顯示“本局結束”（输了）；然而，此密技在Windows Vista版中被拿掉了。

但新接龍的#11982及#17863是可以破解，參考[[YouTube|YouTube]]的破解方法，分別是"Impossible FreeCell Game #11982"及"Solution to freecell game #17863 in HD"。
