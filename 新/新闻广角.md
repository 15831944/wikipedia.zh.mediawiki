《'''新闻广角'''》是[[泉州广播电视台|泉州广播电视台]]推出的一档新闻资讯类节目，于1997年11月1日开播，是一个以反映社情民意和舆论监督为主的新闻栏目，每日20:00首播<ref>{{cite web | title=栏目介绍-新闻广角 | url=http://www.qztv.cn/qztv/Introduction/detail/id/18.html | accessdate=2016-05-01 | archive-url=https://web.archive.org/web/20170912161050/http://qztv.cn/qztv/Introduction/detail/id/18.html | archive-date=2017-09-12 | dead-url=yes }}</ref>。自开播至今，历经数次改版，节目长度由10分钟扩展至目前的35分钟。同时该栏目亦是泉州电视台除《[[泉州新闻联播|泉州新闻联播]]》以外，大多数新闻报道的首播节目。

== 历史 ==
*1997年11月1日，《新闻广角》开播。初期时长10分钟/期，首期主持人为王欢、王红印。当年12月28日，迁入广播电视大厦的新演播室播出。<ref>{{cite web | title=《新闻广角》20岁啦 | url=http://www.qztv.cn/qztv/News/detail/id/27870.html | accessdate=2017-11-01}}</ref>
*2002年夏，节目时长增加至15分钟
*2002年11月1日，举办开播5周年登山活动<ref>{{cite web | title=泉州电视台《新闻广角》开播5周年 | url=http://www.qzwb.com/gb/content/2002-11/03/content_653815.htm | accessdate=2002-11-03}}</ref>
*2004年8月1日，节目时长增至20分钟。同年，新闻爆料热线22300000开通
*2006年1月1日，节目时长再次增加至28分钟<ref>{{cite web | title=新年新视觉荧屏更精彩 | url=http://news.sohu.com/20051231/n241224623.shtml | accessdate=2005-12-31}}</ref>
*2006年5月，新闻广角获得福建新闻名专栏称号<ref>{{cite web | title=新闻立台结硕果 栏目评奖三连冠 | url=http://blog.sina.com.cn/s/blog_541e942a0100kbee.html | accessdate=2006-05-10}}</ref>
*2011年5月19日，新闻广角官方[[新浪微博|新浪微博]]账号开通。
*2012年1月4日，节目时长增至31分钟。
*2013年12月，节目包装进行小幅改动，标题栏左侧的“NEWS+新闻广角”字样更改为播放缩小版的新闻广角片头。
*2016年7月8日，新闻广角首次打破常规，对[[台风尼伯特|台风尼伯特]]的登陆情况进行全程直播报道
*2017年6月12日，新闻广角再次进行改版。主持方式由坐播改为站播，增加触屏显示器，并更换节目包装：取消标题栏播放缩小版片头，恢复之前的“NEWS+新闻广角”字样。<ref>{{cite web | title=【注意啦】《新闻广角》明天“玩大了” 两个“新面孔 | url=http://www.qztv.cn/qztv/News/detail/id/16946.html | accessdate=2017-06-11}}</ref>节目时长增至40分钟。
*2018年4月1日，新增“广角热搜”板块，由市水务集团赞助播出。并压缩新闻播报条数，节目播出时长改为35分钟。
*2018年7月10日，因对[[台风玛莉亚|台风玛莉亚]]进行特别报道，当日播出时长延长至63分钟。<ref>{{cite web | title= 新闻广角-防抗台风玛莉亚特别报道 | url=https://www.bilibili.com/video/av26586035/ | accessdate=2018-07-11}}</ref>
*2019年3月29日，因《记住乡愁》特别节目制播需要，新闻广角于当日首次启用改造后的2号演播厅，并采用16:9比例录制。<ref>{{cite web | title= 新闻广角 19.3.29 | url=https://www.bilibili.com/video/av47714983/ | accessdate=2019-03-30}}</ref>

== 播出 ==
*首播时间：新闻综合频道20:00
*重播时间：[[泉州广播电视台都市生活频道|都市生活频道]]23:55、新闻综合频道次日1:00
