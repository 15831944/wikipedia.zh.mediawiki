== 概要 ==
スーパーマリオシリーズの中でも[[横スクロール|横スクロール]]・[[サイドビュー|サイドビュー]]のジャンプ[[アクションゲーム|アクションゲーム]]である『[[スーパーマリオブラザーズ|スーパーマリオブラザーズ]]』の流れを汲む「[[2D|2D]]マリオ」・「[[王道|王道]]マリオ」と呼ばれる作品の一つ。2011年に初めて発表されたときのタイトルは『ニュー・スーパーマリオブラザーズ・Mii』だったが、Wii Uの最初のソフトになることから、「U」というタイトルに変更された。

同じ系列の作品で2009年に[[Wii|Wii]]用として発売された『[[New_スーパーマリオブラザーズ_Wii|New スーパーマリオブラザーズ Wii]]』はWiiリモコン（プラス）を4本使用して、4人までの同時プレイが可能だったが、本作ではWii Uゲームパッドを併用することでシリーズ最多となる5人同時プレイが可能となる。Wii Uゲームパッドのみでの1人プレイ（TVなし）も可能。

[[Wiiのコントローラ|Wiiリモコン]]を使用するプレイヤーは[[マリオ_(ゲームキャラクター)|マリオ]]、[[ルイージ_(ゲームキャラクター)|ルイージ]]、黄[[キノピオ|キノピオ]]、青[[キノピオ|キノピオ]]をそれぞれ操作する。ストーリーモード以外では、[[Mii|Mii]]を使用することも可能。

今作では『[[スーパーマリオワールド|スーパーマリオワールド]]』と同じようにワールドに対して地名がついており、またそれぞれのコースにも名前がつけられている。さらに、『[[New_スーパーマリオブラザーズ_Wii|New スーパーマリオブラザーズ Wii]]』にあったおてほんプレイが今作でも採用されている。

小売店でのパッケージ版（Wii U専用12cm光ディスク）のほかに、[[インターネット|インターネット]]を利用した[[ニンテンドーeショップ|ニンテンドーeショップ]]からの[[ダウンロード|ダウンロード]]販売も行われる。

=== バディプレイ ===
協力プレイにおいて、Wii Uゲームパッドを操作するプレイヤーには、ほかのプレイヤーをサポートする特殊な役割が与えられる。これはキャラクターを操作するのではなく、ゲームパッドのタッチパネル画面をタッチすることで、そこにブロックを設置することができる、というものである。これを足場として利用することで、通常では行けないところに行ったり、落下しそうなプレイヤーを助けることが可能。ただし、このブロックはすぐに消えてしまうので、上手くプレイヤー同士が息を合わせる必要がある。敵キャラクターにタッチすると、一時的に動きを止めることも可能。

ただし、Wii UゲームパッドをWiiリモコンの代わりに使用することはできず、Wii Uプロコントローラーは非対応のため、バディプレイでない通常の協力プレイを行う場合は、別途Wiiリモコンと[[センサーバー|センサーバー]]を人数分用意する必要がある。

なお、開発プログラマーがWiiリモコンとWii Uゲームパッド（右手の中指でタッチ）を同時に使用して、ひとりでバディプレイを行う「ノーバディプレイ」を実現している<ref>{{Cite web |date=2012-11-16 |url=http://www.nintendo.co.jp/wiiu/interview/hardware/vol7/index3.html |title=New スーパーマリオブラザーズ U 篇 3. “ノーバディプレイ” |publisher=任天堂 |work=社長が訊く『Wii U』 |accessdate=2012-11-17}}</ref>。

== ゲームモード ==
今作では、物語本編にあたる「ストーリーモード」のほかに、「with Miiモード」と総称される3つのゲームモードが用意されている。新しいゲームモードとして、[[やり込み#タイムアタック|タイムアタック]]や連続1UP、コイン集めなどのカテゴリに分かれたお題をクリアしていく「おだいモード」、全コースが自動でスクロールし、ミスをせずにコインを取っていくことでスクロールのスピードが上昇していき、クリアするタイムを競う「ブーストモード」がある。また、[[New_スーパーマリオブラザーズ_Wii|Wii版]]にあった「コインバトル」モードも再び収録されており、今作では個人戦とチーム戦を選ぶことができる。また、コインを自分の思うように配置して楽しむことができる「コインエディット」機能もある。

== ストーリー ==
ある日、マリオやピーチ姫達がピーチ城で[[パーティー|パーティー]]をしていると、クッパ達の[[飛行船|飛行船]]が出現し、そこから巨大な手が出現した。いつものようにピーチ姫をさらうのかと思われたが、クッパ達は隙を突いてマリオ達に攻撃し、気を失っている隙にマリオ達をピーチ城から遠く離れた場所ヘ投げ飛ばしてしまい、ピーチ城を乗っ取ってしまう。マリオ達はピーチ姫を助けるため、ピーチ城へ向かう。

== New超級瑪利歐兄弟U豪華版 ==
[[File:New_Super_Mario_Bros_U_DX.jpg|200px]]

相對原版本(U)豪華版(UDX)是將'''新超級瑪利歐兄弟U'''+'''新超級路易吉U'''兩款遊戲關卡內容結合為一，'''[[新超級路易吉U|新超級路易吉U]]'''每一個關卡的限制時間設定為200秒計數。

並且玩家可以選擇追加兩位帶有各自特長的角色︰可無視一定程度傷害的偷天兔，以及能夠透過超級皇冠變身公主模式防止掉出關卡及進行暫定浮空的奇諾比珂。
