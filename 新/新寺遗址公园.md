'''新寺遗址公园'''是在'''新寺遗址'''上建立的[[遗址公园|遗址公园]]，新寺遗址位于[[陕西省|陕西省]][[西安市|西安市]][[灞桥区|灞桥区]][[新筑街道|新筑街道]]新寺村西侧<ref name="a2">{{Cite web |url=http://news.cnwest.com/content/2010-11/22/content_3779594.htm |title=灞桥区新寺遗址被多人抢挖 土坑满地瓦当丢弃(图)|accessdate=2016-04-02 |author=编辑：刘青 |date=2010-11-22 |work= |publisher=西部网，来源：华商网-华商报|language=Zh-hans }}</ref>，为汉代建筑遗址。

1988年，西安市进行全市[[文物普查|文物普查]]，对新寺村西的一块突起高地进行实地调查。多次调查后，发现新寺遗址。遗址进行遗址区约3万平方米，文化层厚约2米，曾采集有“[[长乐未央|长乐未央]]”、“天下无敌”、“满院生辉”字样的[[瓦当|瓦当]]以及陶、盆、瓮等物件残片<ref name="a2"/>。根据《[[水经注|水经注]]》的记载，通过换算，确定西汉时[[汉文帝|汉文帝]]的[[陵縣制|陵县]][[霸陵县|霸陵县]]即在今新寺村附近，故认为此处遗址即霸陵县[[县治|县治]]所在。或认为[[傅太后|傅太后]]陵所在<ref>原文作“博太后”，当是傅太后。</ref>，或是恭王庙<ref>{{Cite web |url=http://sn.ifeng.com/wenhua/detail_2014_12/02/3230914_0.shtml |title=生态文明的辉煌时期：汉唐首都的门户|accessdate=2016-04-02 |author= |date=2014-12-02 |work= |publisher=凤凰网|language=Zh-hans }}</ref>。

1992年4月20日，[[陕西省人民政府|陕西省人民政府]]公布新寺遗址为第三批[[陕西省文物保护单位|陕西省文物保护单位]]<ref name="a1">{{Cite web |url=http://www.shxkgy.cn/contents/18/420.html |title=陕西省重点文物保护单位名单|accessdate=2016-04-02 |author= |date=2013-01-17 |work= |publisher=陕西省考古研究院网站|language=Zh-hans }}</ref>。随着[[西安世界园艺博览会|西安世界园艺博览会]]开幕，遗址公园也在2011年4月28日开园<ref>{{Cite web |url=http://epaper.xiancn.com/xarb/html/2011-04/28/content_29260.htm |title=献礼2011世界园艺博览会 西安国际内陆港新寺遗址公园精彩开园|accessdate=2016-04-02 |author= |date=2011-04-28 |work= |publisher=陕西日报社数字报刊|language=Zh-hans }}</ref>。
