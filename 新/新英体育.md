'''新英体育'''（{{lang|en|Super Sports Media Inc.}}）是一家在[[开曼群岛|开曼群岛]]注册的公司，成立于2010年8月。新英体育主营业务为各平台体育节目的推广、销售、宣传。新英体育拥有2010/11赛季至2018/19赛季[[英超联赛|英超联赛]]中国大陆独家转播权。

== 历史 ==
新英体育由[[IDG|IDG]]联合[[新雅迪传媒|新雅迪传媒]]创办，公司在2010年竞标夺得2010/11赛季至2012/13赛季三年中国大陆[[英超联赛|英超联赛]]的独家转播权。新英体育随即与中国大陆多家地方电视台签约，授权转播英超联赛。2012年，新英体育与英超续约，获得了2013/14赛季至2018/19赛季中国大陆及澳门独家转播权，为此新英体育付了大约折合10亿人民币的价钱。2016年5月9日，[[香港交易所|香港交易所]]上市公司[[文化传信|文化传信]]宣布拟以38.75亿港元（约合32.52亿元人民币）收购新英体育，同时将IDG资本合伙人、新英体育董事长[[李建光|李建光]]引入为主要股东。<ref name=chs>{{cite news|title=文化传信接应传中 一射中网侧|url=http://finance.sina.com.cn/stock/hkstock/2016-05-09/doc-ifxryahs0563876.shtml|accessdate=2016-08-27|agency=财华社|date=2016-05-09}}</ref><ref name=ff/> 但交易最後取消。

== 业务 ==
2015/16赛季，[[新浪|新浪]]、[[腾讯|腾讯]]、[[乐视|乐视]]分别以1800万美元自新英体育购得英超联赛网络转播权，总价折合3.35亿元人民币。此外，新英体育与[[中央电视台|中央电视台]]、[[北京电视台|北京电视台]]、[[上海电视台|上海电视台]]、[[广东电视台|广东电视台]]等20家电视台签约，媒体估计这部分版权收入为1.15亿人民币左右。[[上海文广|上海文广]]所属的[[互动电视|互动电视]]公司（SiTV）与新英体育合作开办高清频道[[新视觉高清|新视觉高清]]，互动电视公司提供演播间，直播英超比赛。新英体育的合作方还有[[IPTV|IPTV]]平台的[[百视通|百视通]]。另外，新英体育还保留了约40%比赛场次的转播权不出售给合作方，而是在新英体育网站的平台上进行付费直播。<ref>{{cite news|title=阿里联手新英探路体育电商|url=http://news.xinhuanet.com/tech/2016-08/09/c_129215242.htm|accessdate=2016-08-27|agency=北京商报|date=2016-08-09}}</ref><ref name=ff>{{cite news|title=英超转播权与付费模式的春天来了？|url=http://industry.sports.cn/news/ss/2015/0809/114280.html|accessdate=2016-08-27|agency=华奥星空|date=2015-08-09}}</ref>

== 争议事件 ==
2016年8月2日上午，[[小米盒子|小米盒子]]与百视通联合举办新闻发布会，宣布小米盒子获得新赛季英超联赛转播权。新英体育当天就发布声明，表示从未授权小米盒子转播英超比赛，中国大陆地区的英超转播权必须从新英体育购买，百视通只有播放权，而无转授权。<ref>{{cite news|title=新英体育副总裁：英超转播权必须从新英购买|url=http://sports.sina.com.cn/g/pl/2016-08-02/doc-ifxunyxy6325517.shtml|accessdate=2016-08-28|agency=新浪体育|date=2016-08-02}}</ref>

2016年8月14日，新英体育网站出现技术故障，很多网站会员无法登录网站观看英超联赛第一轮[[曼联|曼联]]的比赛。事后，新英体育表示将对球迷进行补偿，购买单场比赛的球迷将获得三场观赛券，死忠会员、特权会员的会员权限将免费延期一个月。然而，有球迷认为会员权限延期一个月毫无意义，因为补加的一个月正值休赛期，并无英超比赛。<ref>{{cite news|title=曼联直播出故障 新英体育网补偿条件遭吐槽|url=http://finance.sina.com.cn/roll/2016-08-18/doc-ifxvctcc7938247.shtml?t=1471708716338|accessdate=2016-08-28|agency=中国新闻网|date=2016-08-18}}</ref>
