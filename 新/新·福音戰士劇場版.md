《'''新·福音戰士劇場版:│▌'''》（{{lang-ja|シン・エヴァンゲリオン劇場版:<nowiki>|</nowiki>'''<nowiki>|</nowiki>'''}}　EVANGELION:3.0+1.0）是預定于2020年公映的[[日本|日本]][[動畫電影|動畫電影]]。由[[庵野秀明|庵野秀明]]执导、[[khara|khara工作室]]制作，是以《[[新世纪福音战士|新世纪福音战士]]》再構築为[[福音戰士新劇場版|福音戰士新劇場版]]四部曲的第四部。<ref name=news20180720/>

== 標題 ==
原日文标题为「{{lang|ja|シン・エヴァンゲリオン劇場版:<nowiki>|</nowiki>'''<nowiki>|</nowiki>'''}}」，英文副标题为「EVANGELION:3.0+1.0」。標題中「:│▌」的部分不发音，只是作为符号标记<ref>{{cite web |script-title=ja:Q：次回作となっているタイトル『シン・エヴァンゲリオン劇場版:{{!}}{{!}}』の【:{{!}}{{!}}】←これってなんて読むんですか？話に上がった時にわからないのでややこしいです。A：言葉にして発するときは誰も記号は読んでませんね…  #スタジオカラーに質問 |url=https://twitter.com/khara_inc2/status/940177498712694785 |website=@khara_inc2 |language=ja |date=2017-12-11}}</ref>；「:│▌」為[[音樂記譜法|音樂記譜法]]（[[五線譜|五線譜]]）中的反覆記號<ref>{{cite web |script-title=ja:『シン・エヴァンゲリオン劇場版』がついに２０２０年に完結！全国の映画館で特報解禁 |url=http://news.nicovideo.jp/watch/nw3690135 |website=ニコニコニュース |language=ja |date=2018-07-20}}</ref>，「│▌」則是終止線<ref>{{cite web |script-title=ja:シン・エヴァの『:{{!}}{{!}}』はなんて読む？ネットで憶測広がる |url=https://yukawanet.com/archives/4339009.html |website=秒刊SUNDAY |language=ja |date=2012-11-21}}</ref>；通常省略为「{{lang|ja|シン・エヴァンゲリオン劇場版}}」，简称为「{{lang|ja|シン・エヴァ}}」。「{{lang|ja|エヴァンゲリオン}}」变更回原作的写法，而不再是福音戰士新劇場版系列使用的「{{lang|ja|ヱヴァンゲリヲン}}」。標題的「シン」是日语中「新」的[[日語假名|假名]]，因此一般认为是由「福音戰士新劇場版」变为「新福音戰士劇場版」的[[易位构词游戏|文字游戏]]。暂译为「新·福音戰士劇場版:│▌」。

==制作==
前作《[[福音戰士新劇場版：Q|福音戰士新劇場版：Q]]》上映后，导演[[庵野秀明|庵野秀明]]陷入了抑郁状态，而在此期间接受了哥斯拉新作电影（《[[正宗哥吉拉|新·哥斯拉]]》）的导演工作，致使本作的制作一度推迟<ref name=comment/>。

==发行==
===上映与宣传===
前作《[[福音戰士新劇場版：Q|福音戰士新劇場版：Q]]》上映时公开了本作的标题{{lang|ja|『シン·エヴァンゲリオン劇場版:<nowiki>|</nowiki>'''<nowiki>|</nowiki>'''』}}（直译：新·福音戰士劇場版:|'''|'''），但上映日未定。2014年9月5日，[[日本電視台|日本電視台]]放映《福音戰士新劇場版：Q》时释出次回副标题（英文标题）-{『EVANGELION:3.0+1.0』}-。有传言本作将于2015年公映<ref>{{cite web|title=Magazine: Final Evangelion Film Slated for Fall-Winter 2015 With Utada Song |url=http://www.animenewsnetwork.com/news/2014-12-26/magazine-final-evangelion-film-slated-for-fall-winter-2015-with-utada-song/.82629 |publisher=[[Anime_News_Network|Anime News Network]] |date=2014-12-26}}</ref>，但并无官方消息。
2015年4月1日，官网发表庵野秀明署名文章、对由于制作《[[正宗哥吉拉|新·哥斯拉]]》而本作被推迟的制作状况进行了说明<ref name=comment>{{cite web|script-title=ja:『シン・エヴァンゲリオン劇場版』及びゴジラ新作映画に関する庵野秀明のコメント |url=http://www.evangelion.co.jp/news_20150401.html |publisher=evangelion.co.jp |date=2015-04-01 |language=ja}}</ref>。2016年制作完《新·哥斯拉》后，恢复本作的制作<ref>{{cite web|url=http://www.animenewsnetwork.com/daily-briefs/2016-11-17/evangelion-animator-final-evangelion-remake-film-is-in-progress/.108942|title=Evangelion Animator: Final Evangelion Remake Film is in Progress|publisher=Anime News Network|date=2016-11-17}}</ref><ref>{{cite web|url=http://www.animenewsnetwork.com/daily-briefs/2017-04-05/evangelion-studio-khara-confirms-next-shin-evangelion-is-in-production/.114394|title=Evangelion Studio Khara Confirms Next Shin Evangelion Is in Production|publisher=Anime News Network|date=2017-04-05}}</ref>。
2017年7月29日，官网首页更新、刊载印象板，并带有「鋭意制作中」字样<ref>[https://web.archive.org/web/20170729131448/http://www.evangelion.co.jp/ evangelion.co.jp]</ref>。2018年7月20日，在日本影院公開特報映像，表明将于2020年上映<ref name=news20180720>{{cite web |script-title=ja:『シン・エヴァンゲリオン劇場版』公開は2020年　8月末まで全国劇場で特報上映 |url=http://www.oricon.co.jp/news/2115966/full/ |website=ORICON NEWS |date=2018-07-20 |language=ja}}</ref>，7月26日，在網路上亦公开特報<ref>{{cite web |script-title=ja:「シン・エヴァンゲリオン劇場版」特報、公式サイトでも公開 |url=http://anime.eiga.com/news/106811/ |website=アニメハック |language=ja}}</ref>。

==反响==
2018年7月20日，官方（制片商[[khara|khara]]以及发行商[[東寶|東寶]]）宣布本作将于2020年上映。即使这是前作《[[福音戰士新劇場版：Q|福音戰士新劇場版：Q]]》在2012年11月17日上映后，官方第一次发布上映时间方面的消息，但部分观众表示不可信、肯定还会延期的<ref>{{cite web |script-title=ja:『エヴァ新劇場版』当初の公開予定より12年遅れ！「絶対また延期するだろ」 |url=https://getnews.jp/archives/2064010 |website=ガジェット通信 GetNews |language=ja |date=2018-07-26}}</ref>。
