'''新加坡電信有限公司'''（{{lang-en|'''Singapore Telecommunications Limited'''}}；缩写：{{lang|en|'''Singtel'''}}，{{sgx|Z74}}、{{otcbb|SGAPY}}）简称'''新电信'''，在25个国家拥有4亿位用户，是全世界第二十至三十大的移动电话服务商<ref name="cust base">{{Cite web |url=http://info.singtel.com/node/7609 |title=SingTel Group’s mobile customer base reaches 368 million |access-date=2012-07-03 |archive-url=https://web.archive.org/web/20180524151353/http://info.singtel.com/node/7609 |archive-date=2018-05-24 |dead-url=yes }}</ref><ref>{{cite web |url=http://theindonesiatoday.com/technology-headline/7477-singtel-and-telkomsel-record-mobile-subscribers-growth-in-2010-.html |title=存档副本 |accessdate=2011-03-08 |deadurl=yes |archiveurl=https://web.archive.org/web/20110219013328/http://www.theindonesiatoday.com/technology-headline/7477-singtel-and-telkomsel-record-mobile-subscribers-growth-in-2010-.html |archivedate=2011-02-19 }}</ref>，現在正集中於提供互联网接入、IPTV（[[新电信电视|Singtel TV]]）、手機（新电信流动）和固網電話服務。是國際組織[[Bridge_Alliance|Bridge Alliance]]的創始會員。
新電信強烈增加新加坡以外的市場和擁有局部經營者的股份，包括拥有[[澳大利亚|澳大利亚]]電信公司[[Optus|Optus]]的所有股份。至於新電信的大部份股權則是由[[淡馬錫控股|淡馬錫控股]]持有。2012年3月6日，新电信斥资3.21亿美元收购了以色列Amobee公司，该公司主要提供移动广告解决方案。

2012年9月20日新电信以2650万美元收购了社交照片智能聚合服务提供商Pixable，旨在进一步丰富用户的网络内容交流体验。2012年9月26日淡馬錫以每股3.2-3.25[[新加坡元|新加坡元]]出售4億股新电信，涉及13億新加坡元。報導又指淡馬錫可以出售多1億股新电信股份，令交易達到13.4億新加坡元。當減持5億股新电信後，淡馬錫的持股量將由54.4%降至51.3%。{{来源请求}}

== 全球办事处 ==
新加坡电信在22个国家与地区设有共40个办事处，遍布亚太、中东、欧洲与美国等地区，为全球客户提供覆盖广泛又稳定的网络基础设施。每个[http://info.singtel.com/business/singtel-global-offices?utm_source=wikipedia-chi&utm_medium=sgo_link&utm_campaign=wikipedia 新加坡电信的全球办事处]都能提供专业地、在地化的业务及技术上的支持以满足不同客户的需求。

== 其他資料 ==
[[File:Singtel_tower.jpg|thumb]]校園的信號塔。]]
新電信擁有其他附屬公司和其他組全公司，和海外實體控股。新電信的附屬公司包括SingNet（網絡部门）、SingTel Digital Media、NCS有限公司（通信工程部门）。聯營公司有[[新加坡郵政|新加坡郵政]]、[[台灣|台灣]][[新世紀資通|新世紀資通]]（sparq*速博，新电信持有sparq*速博24.5%股份）、[[天空傳媒|天空傳媒]]（yam天空，新電信持有yam天空少量股份）。

新電信在電話發明後3年（1879年）已經在[[新加坡|新加坡]]运营，當時以50條線做為電話交易。
