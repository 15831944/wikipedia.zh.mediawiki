'''新名神高速公路'''（{{jpn|j=新名神高速道路|hg=しんめいしんこうそくどうろ|rm=Shin Meishin kōsokudōro|f=y}}）是以[[三重縣|三重縣]][[四日市市|四日市市]][[四日市系統交流道|四日市系統交流道]]為起點，經[[滋賀縣|滋賀縣]]、[[京都府|京都府]]、[[大阪府|大阪府]]至[[兵庫縣|兵庫縣]][[神戶市|神戶市]][[神戶系統交流道|神戶系統交流道]]的[[日本高速公路|高速公路]]（[[高速自動車國道|高速自動車國道]]）。簡稱為'''新名神高速'''（{{lang|ja|新名神高速／しんめいしんこうそく}}，{{lang|ja|SHIN-MEISHIN EXPWY}}）或'''新名神'''（{{lang|ja|新名神／しんめいしん}}）。

高速公路路線編號與[[新東名高速公路|新東名高速公路]]、[[伊勢灣岸自動車道|伊勢灣岸自動車道]]共同被編為 {{Ja Exp Route Sign|E1A}} <ref>{{Cite web |url=http://www.mlit.go.jp/road/sign/numbering/list/index.html |title=高速道路ナンバリング一覧 |publisher=国土交通省 |accessdate=2017-02-28}}</ref>。

==概要==
新名神高速公路的法定名稱為[[近畿自動車道|近畿自動車道名古屋神戶線]]。<ref>[http://corp.w-nexco.co.jp/corporate/release/hq/h19/0423/ 近畿自動車道 名古屋神戸線の名称について NEXCO西日本]</ref> 。新名神高速首段[[草津JCT|草津JCT]]至[[草津田上IC|草津田上IC]]於2005年3月19日通車，第二段[[龜山JCT|龜山JCT]]至[[草津田上IC|草津田上IC]]於2008年2月23日通車。
<br/>
新名神高速最早於1987年的第四次全國綜合開發計劃中被提出，曾因削減預算和[[信樂交流道#.E6.AD.B7.E5.8F.B2|發現遺跡]]而更改路段設計。[[滋賀縣|滋賀縣]][[大津市|大津市]]至京都府[[城陽市|城陽市]]和[[京都府|京都府]][[八幡市|八幡市]]至[[大阪府|大阪府]][[高槻市|高槻市]]這兩段共計約35公里的路段更因預測低交通流量而凍結工程近十年。目前計劃2023年全線通車。

; 本線
* 道路構造令 : 第1種第1級（龜山西系統交流道 - 甲賀土山交流道下行線以外暫定施工時為第1種第2級）

** 上述以外的路段 : 行車道3.50公尺 超車道3.75公尺（用地徵收 : 6車道）

** [[橋|橋梁]]部（中小橋） : 2.50公尺 - 3.00公尺×2（左側）
** 橋梁部（長大橋） : 1.75公尺 - 3.125公尺×2（左側）
* 路肩（雙向分離區間<ref>上下線中央分離帶有分離</ref>）
** 土工部 : 2.50公尺 - 3.50公尺（左側）1.25公尺 - 1.75公尺（右側）
** [[隧道|隧道]]部 : 1.00公尺 - 2.50公尺
** 橋梁部（中小橋） : 2.50公尺 - 3.00公尺（左側）1.25公尺 - 1.75公尺（右側）
** 橋梁部（長大橋） : 1.75公尺 - 3.125公尺（左側）1.25公尺（右側）
* 中央分離帶 : 4.50公尺
* 中央分離帶的路肩 : 2.00公尺（隧道部1.25公尺）
* 最小曲線半徑 : 標準値3,000公尺 特例値1,500公尺
* 最急坡度 : 標準値2.0% 特例値3.0%
* 最小縱斷曲線半徑 : 凸型標準値40,000公尺 特例値23,000公尺 凹型標準値9,000公尺 特例値6,000公尺
* 停止視距 : 310公尺

; 聯絡路
* 道路構造令 : 第1種第3級
* 設計速度 : 60<ref>高槻連絡路設計速度為60km/h</ref> - 80km/h
* 車道 : 3.50公尺×2
* 路肩 : 2.50公尺（橋梁部1.75公尺）
* 最小曲線半徑 : 標準値700公尺
* 最急坡道 : 標準値4.0%
