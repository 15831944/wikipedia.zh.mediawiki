'''新浪微博'''（{{lang-en|Weibo/Sina Weibo}}，官方名称为'''微博'''），是一个由[[新浪网|新浪网]]推出，提供[[微博客|微博客]]的服务网站。用户可以通过网页、[[WAP|WAP]]页面、外部程序和[[手机短信|手机短信]]或[[彩信|彩信]]等发布信息，并可上传图片和链接视频，实现即时分享<ref name="手机上的微博人生">{{cite web|title=手机上的微博人生|url=http://www.zte.com.cn/cndata/magazine/mobile_world/2010/2_1/magazine/201010/t20101014_192781.html|publisher=中兴通讯|accessdate=2010-11-07|date=2010-10-14}}</ref>。新浪微博可以直接在一条微博下面附加评论，也可以直接在一条微博里面发送图片、视频，新浪微博最先添加这两点功能。

新浪微博是一个基于用户关系的信息分享、传播以及获取信息的平台<ref>[http://hr.weibo.com/jobs/about.php 新浪微博官网——微博介绍] {{webarchive|url=https://web.archive.org/web/20120513171644/http://hr.weibo.com/jobs/about.php |date=2012-05-13 }}</ref>，它占据[[中国微博|中国微博]]用户总量的57%，以及中国微博活动总量的87%，是中国大陆访问量最大的网站之一<ref>{{cite web|title=调查称第一季新浪微博占57%国内微博用户量|url=http://news.sina.com.cn/m/2011-06-29/121922726985.shtml|publisher=新浪科技}}</ref><ref>{{cite web|title=未来资产报告称新浪微博使用率市场份额达87%|url=http://tech.sina.com.cn/i/2011-01-28/18045144960.shtml|publisher=新浪科技|accessdate=2011-07-15}}</ref><ref>{{cite web|title=日本公布2011年中国畅销商品排行榜 新浪微博居首|url=http://world.people.com.cn/GB/15147732.html|publisher=国际人民网|accessdate=2011-07-15}}</ref> 。它是由新浪公司于2009年8月推出，截至2012年12月底，新浪微博注册用户已超5亿，同比大幅增长74%，日活跃用户数达到4620万，用户每日发博量超过1亿条。2013年4月29日，[[阿里巴巴集团|阿里巴巴集团]]入股新浪微博18%股份。<ref>{{Cite web |url=http://tech.163.com/13/0429/20/8TLG41E3000915BF.html |title=阿里5.86亿美元投资新浪微博 占股18%  |author=网易科技  |date=2013-04-29  |publisher=网易科技报道   |accessdate=2013-10-19  }}</ref>

新浪微博采用了与[[新浪博客|新浪博客]]一样的推广策略，即邀请明星和名人加入开设微型博客，并对他们进行实名认证，认证后的用户在用户名后会加上一个橙色字母“V”，以示与普通用户、微博达人的区别，同时也可避免冒充名人微博的行为，但微博功能和普通用户是相同的。

目前新浪微博的邀请的重点转向了媒体工作者<ref name="手机上的微博人生"/>，同时也有大量政府部门、企业公司和民间组织在新浪微博上开通了账号，将其作为一个发布和交流信息的平台。认证后的群体用户在用户名后会加上一个蓝色字母“V”。

此外，港台演藝圈名人以及在大陸經商的港澳台公司行號，大多也以新浪微博當作面向[[中国内地|中国内地]]的信息交流窗口。

新浪微博在2018《财富》发布未来50强中排名第二。<ref>{{cite web|url=http://finance.sina.com.cn/leadership/2018-10-19/doc-ihmrasqs7182023.shtml|title=2018年《财富》未来50强揭晓 微博排名第二|publisher=新浪网}}</ref>
== 历史 ==
新浪微博于2009年8月14日开始内测。<ref>{{Cite web|url=http://www.sohu.com/a/194200560_104421|title=新浪微博八年兴衰史|accessdate=2018-05-28|date=2017-09-24|work=www.sohu.com}}</ref>9月25日，新浪微博正式添加了@功能以及私信功能，此外还提供“评论”和“转发”功能，供用户交流。

新浪微博用户平均每天发布超过1亿条微博内容。除網頁版外，新浪微博也擁有本地客戶端版，目前主流的有[[微博AIR|微博AIR]]、Wing微博、微波炉等，新浪微博也推出了官方的本地桌面客户端-微博桌面。

2010年12月1日10時20分，新浪微博出現不能訪問的故障<ref name="Sina_2010Dec01">{{Cite web|url=http://tech.sina.com.cn/i/2010-12-01/15324930344.shtml|title=新浪微博恢复访问 发布故障致歉声明|accessdate=2011-03-10|author=新浪微博|work=新浪网|date=2010年12月1日|language=zh-hans}}</ref>，有部分用户出現“用户不存在”的現象，微博系統要求填写个人资料，并强制退出。<ref name="36kr_2010Dec01">{{Cite web|url=http://www.36kr.com/%E6%96%B0%E6%B5%AA%E5%BE%AE%E5%8D%9A%E5%AE%95%E6%9C%BA/|title=新浪微博宕机|date=2010-12-01|accessdate=2011-03-10|author=pestwave|work=36氪|language=zh-hans}}{{Dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>新浪微博的訪問於當日14時30分恢復正常，官方解釋事件是由於新浪微博用户量和每日的微博发布量的激增所致的。<ref name="Sina_2010Dec01"/>搜狗CEO王小川則稱該事件是由於微博伺服器進入死循环過程，導致進入首頁後自動跳轉至系統錯誤頁面，4秒後再次自動跳轉至首頁；這對伺服器造成巨大衝擊，從而使伺服器宕機。<ref name="36kr_2010Dec01"/>

2011年3月2日，新浪[[CEO|CEO]]曹国伟表示，新浪微博注册用户数已超1亿<ref name="sina_2011Mar02">{{Cite web|url=http://tech.sina.com.cn/i/2011-03-02/06005233783.shtml|title=新浪发布2010年四季及全年财报 微博用户数过亿|publisher=新浪科技|date=2011-03-02|accessdate=2011-03-05}}</ref>。同時他還透露，新浪將會將微博部門的600人編制擴大至少一倍，並大幅增加對微博的營銷支出<ref name="ftchinese_2011Mar04">{{Cite web|url=http://www.ftchinese.com/story/001037269|title=新浪拟大力投资微博|accessdate=2011-03-06|author=席佳琳 译者/何黎|work=英国《金融时报》|date=2011年3月4日|language=zh-hans}}</ref>。

2011年4月7日起，新浪微博正式啟用新域名weibo.com和weibo.cn，同時啟用新版logo。原域名使用不受影響，现有的WAP域名t.sina.cn将自动跳转到weibo.cn<ref>{{Cite web|url=http://fj.sina.com.cn/news/z/2011-04-07/152896712.html|title=新浪微博今日启用weibo.com域名 同步更换标识|accessdate=2011-04-07|work=新浪科技|date=2011年4月7日}}</ref>。有分析认为，新浪微博可能独立分拆上市<ref>{{Cite web|url=http://tech.sina.com.cn/i/2011-04-07/02235375115.shtml|title=新浪微博开独立域名 流量超部分门户|accessdate=2011-04-07|work=新京报|date=2011年4月7日}}</ref><ref>{{Cite web|url=http://tech.sina.com.cn/i/2011-04-07/14165378096.shtml|title=新浪微博启用新域名或为分拆上市作准备|accessdate=2011-04-07|work=华西都市报|date=2011年4月7日}}</ref>。

2011年5月25日，新浪微博新增两种图片水印，在原来右下角网址的基础上，又增加了微博图标和昵称。用户只有以weibo.com的域名登录，才可以在个性设置中更改水印的数量和位置，目前只有超过一定像素的[[JPEG|JPEG]]格式的图片会加水印。

2018年10月9日，新浪微博官方宣布，为“进一步营造清朗、健康、文明、有序的平台环境，切实保护[[未成年人|未成年人]]网络空间安全”，决定自2018年11月1日上线的新版本客户端起，暂停对未满14周岁的未成年人开放注册功能。同时，新浪微博正在加紧开发针对未成年人的特别版本，专门提供适宜未成年人浏览和参与的内容及活动<ref>{{Cite web|url=https://www.thepaper.cn/newsDetail_forward_2512501|title=新浪微博：下月起暂停对不满14周岁的未成年人开放注册功能|website=澎湃新闻|date=2018-10-09|accessdate=2018-10-10}}</ref>。

== 认证制度 ==
新浪微博与Twitter一样实行认证制度，对名人、企业、机构等进行认证。当一个用户通过认证后，其微博用户名后将会加上橙色“V”的图示，企业、机构、组织则显示蓝色的“V”，其微博主页上也会出现“新浪认证”的图示和说明<ref name="verified">{{cite web|title=微认证|url=http://weibo.com/pub/verified|accessdate=2011-05-26}}</ref>。获得认证的用户不能随意修改昵称，否则可能会需要重新进行认证。

目前新浪微博的认证已由过去的[[电子邮件|电邮]]材料改为在线提交材料。

2011年，新浪承认有个别新浪微博内部人员违反规定进行付费加V认证服务，此行为后来得到制止<ref>{{cite web|url=http://tech.ifeng.com/internet/detail_2011_06/26/7253887_0.shtml|title=新浪承认微博内部人员参与兜售付费加V服务|date=2011-06-26|publisher=北京晨报|author=周宁,逯寒青|accessdate=2011-06-26}}</ref>。

=== 个人认证 ===
新浪微博邀请[[明星|明星]]和名人加入开设微型博客，并对他们进行实名认证，认证后会在用户名后加上一个橙色字母“V”，以示与普通用户的区别，但认证用户和普通用户没有功能上的差别。

认证的对象主要是各行业的明星、企业高管和重要的[[新闻|新闻]]当事人等，以及各种自媒体爱好者。

目前进行个人认证需先上传头像、绑定手机，关注30名用户并获得100名用户的关注（包含两位橙V认证用户并与他们互相关注），之后才可在线提交身份认证材料。

在获得橙色字母“V”后，若上月微博被阅读数大于100万，或上月微博视频播放量大于100万的前提下，即可升级成“微博签约自媒体”，即认证后加上一个红色字母“V”<ref>{{Cite web|url=http://me.weibo.com/|title=微博自媒体 微博}}</ref>。而在升级红色字母“V”后，若下月微博阅读量不达标，则会降级为橙色字母“V”。

==== 代表委员 ====
新浪微博曾在2011年[[两会|两会]]期间推出的针对[[全国人大代表|全国人大代表]]或全国[[政协委员|政协委员]]的认证。现并入个人认证。

=== 机构认证 ===
目前除网站和应用官方认证外，都需要提交盖有公章的认证公函扫描件。认证后会在用户名后加上一个蓝色字母“V”。

====政府====
政府认证的对象是市政府、公安机关、司法、交通、旅游、医院、卫生、市政、工商等机构的官方帐号。认证后会在用户名后加上一个蓝色字母“V”。

====媒体====
[[媒体|媒体]]认证的对象是电视、广播、平面或网络传媒在新浪微博开设的官方账号。媒体认证不等于媒体从业人员认证，后者属于个人认证。认证后会在用户名后加上一个蓝色字母“V”。

====企业====
为了方便[[企业|企业]]在微博上进行宣传，新浪会对企业进行认证。与名人认证相同，经过企业认证后的用户名后也会加上一个字母“V”<ref>{{cite web|title=企业利用微博营销平台进行扩张|url=http://tech.sina.com.cn/i/2010-08-25/15174587111.shtml|accessdate=2010-11-07|date=2010年8月25日}}</ref>。<br>
有营业执照和公章的盈利型各类企业、公司的官方帐号均可申请企业认证。认证后会在用户名后加上一个蓝色字母“V”。

====校园====
原[[高等院校|高校]]认证，是提供给各大院校或其下属机构以及该校校友会等组织的认证方式。认证后会在用户名后加上一个蓝色字母“V”。

==== 机构/团体认证 ====
原机构认证，是提供给图书馆、博物馆、美术馆、粉丝团体、体育俱乐部、车友俱乐部、影视话剧等官方帐号的认证方式。认证后会在用户名后加上一个蓝色字母“V”。

====网站====
网站认证的对象是商业网站或具备一定知名度的个人网站（需要有独立域名），[[淘宝|淘宝]]店铺暂不能通过认证，无信息产业部备案信息的网站通过认证可能有难度。认证后会在用户名后加上一个蓝色字母“V”。

====应用====
此项仅限为新浪微博开发应用的用户申请。认证后会在用户名后加上一个蓝色字母“V”。
