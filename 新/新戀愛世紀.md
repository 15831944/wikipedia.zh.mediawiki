{{engname|《新戀愛世紀》|'' Love Generation Hong Kong ''}}是1998年出品的一部香港電影<ref>{{Cite web |url= http://hkmdb.com/db/movies/view.mhtml?id=8373&display_set=big5|title =新戀愛世紀（1998）| publisher =香港影庫|language = 中文}}</ref><ref>{{Cite web |url= http://www.hkbo.com.hk/search-details-new.php?filmid=8340&run=1 |title =電影資料庫| publisher = 香港票房有限公司|language = 中文}}</ref>，以愛情故事為題材，由[[王晶|王晶]]執導、監製及編劇，[[黎明|黎明]]、[[雷頌德|雷頌德]]、[[劉嘉玲|劉嘉玲]]、[[舒淇|舒淇]]等演出，在香港及日本進行拍攝<ref> {{Cite book | author =黃寤蘭| title =當代中國電影：一九九八| location =台灣| publisher =時報文化出版企業股份有限公司| date =1999年3月1日|ISBN =9789571328461|url = https://books.google.com.hk/books?id=jZ58AAAAIAAJ&q| language =中文}} </ref><ref> {{Cite book | author =吳偉明| title =日本流行文化與香港| location =香港| publisher =商務印書館(香港)有限公司|date =2015年7月6日|ISBN =9789620765629|url = https://books.google.com.hk/books?id=4jarCgAAQBAJ&hl=zh-TW&source=gbs_navlinks_s| language =中文}} </ref>。

==情節介紹==
電影內容以三段的愛情作為故事題材，Maggie是一個化妝品銷售員，其男友Bill是一個白領職員。Maggie既要面對麻煩的老闆和顧客，又要面對經常向她索取金錢的賭徒父親，生活壓力很大。Bill一表人才，但事業發展停滯不前，兩人相戀多年後最終分手收場。Bill決定離開香港，到外地創業，數年後，Bill事業有成，並且認識了一名影星女友Joey。有一天，Bill在東京遇見Maggie，原來Maggie即將與江湖老大平哥結婚，這時候Bill才發現自己深愛著Maggie，與此同事，一個叫Moon的女子原來一直在暗戀著Bill，他們之間展開了多段的愛情故事<ref>{{Cite web |url= https://tw.linkeddb.com/movie/59ed58171d69eb002947e8f4/|title =《新戀愛世紀》劇情介紹| publisher =星關係|language = 中文}}</ref><ref>{{Cite web |url= http://www.filmcritics.org.hk/film-review/node/2015/06/26/%E6%96%B0%E6%88%80%E6%84%9B%E4%B8%96%E7%B4%80%EF%BC%9A%E5%A6%82%E6%9E%9C%E6%8A%B1%E8%91%97%E3%80%8C%E4%BB%A5%E7%82%BA%E3%80%8D|title =新戀愛世紀：如果抱著「以為」| author =列孚| date =1998年|publisher =香港影評庫|language = 中文}}</ref><ref>{{Cite web |url= https://moviecool.asia/film/gdgxDTudOSF|title =新戀愛世紀| publisher =影史庫|language = 中文}}</ref>。
