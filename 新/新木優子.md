'''新木優子'''（{{lang-ja|'''あらき ゆうこ'''}}，{{lang-en|'''Yuko Araki'''}}，{{bd|1993年|12月15日|||Araki Yūko}}），本名：石井優子（いしい ゆうこ）。[[日本|日本]]女[[演員|演員]]，[[東京都|東京都]]出身。隸屬[[星塵傳播|星塵傳播]]娛樂3部，身高164[[公分|公分]]<ref>[http://ameblo.jp/schoolgirls/entry-11120625440.html こんにちはー！（新木優子）] 3年B組School girl BLOG 2011-12-29</ref>。

== 經歴 ==
2004年，小學5年級時在[[原宿|原宿]]的[[竹下通|竹下通]]被[[星塵傳播|星塵傳播]]的星探發掘<ref>{{Cite web |author=高倉文紀 |url=https://www.talentsearch.jp/talent_master/attention/araki-yuuko.html |title=新CMクイーン候補に浮上したフォトジェニック美女☆新木優子 |work=タレントパワーランキング |publisher=アーキテクト |date= 2015-08-05 |accessdate= 2017-09-03 }}</ref>。

2008年5月，初次主演電影《錨を投げろ》。

2014年，成為《[[non-no|non-no]]》雜誌的專屬模特兒。

2016年，初次主演電視劇，為富士電視臺的《愛愛外星人》<ref name=oricon160624>{{cite news|url=http://www.oricon.co.jp/news/2073960/full/|title=『ゼクシィ』CMガール新木優子、連ドラ初主演 『ラブラブエイリアン』実写化|newspaper=ORICON STYLE|date=2016-06-24|accessdate=2016-06-24}}</ref>。

== 人物 ==
2017年，日本《週刊新潮》登出新木為新興宗教「[[幸福科學|幸福科學]]」信徒，該宗教會公布新木守護靈的影片，而引發關注。原為演員的[[清水富美加|清水富美加]]因信仰此宗教而[[出家|出家]]退出演藝圈的事件引發輿論，在餘波未平之際，新木此信仰招致議論猜測。其後經紀公司社長表示新木為該宗教信徒之事為屬實，但並無出家退出演藝圈的想法<ref>[http://www.tokyo-sports.co.jp/entame/entertainment/751647/ 新木優子「幸福の科学」信者発覚で波紋　さらなる女優信者も].東スポ.2017-08-31.[2017-08-31].</ref>。
