{{新疆历史}}
'''新疆和平解放'''指的是1949年9月25日、26日，[[中華民國國軍|中华民国国军]]驻疆部队和[[新疆省政府|新疆省政府]]相续倒戈中共政权的事件，这也是[[第二次国共内战|第二次国共内战]]的一部分。[[新疆省_(中華民國)|新疆省]]时为[[中华民国领土|中华民国面积最大的省份]]，亦是其名义上控制的最后一个[[中国西北地区|西北]]省份。至此，中共政权已占领[[长江|长江]]以北的，[[中国北方|中国北方]]全部省份。其后，[[中国人民解放军|中国人民解放军]]迅速开进新疆，收编起义部队、接管政务，至1950年初将其全境<ref name="a1"/>纳入[[中华人民共和国政府|中华人民共和国政府]]管辖之下。

== 经过 ==

1944年，[[苏联政府|苏联政府]]支持、插手了[[三区革命|三区革命]]，反对[[国民政府|国民政府]]的统治。后期，[[中国共产党|中国共产党]]势力开始影响新疆。1949年随着中国共产党在第二次国共内战中取得绝对优势，夏季末，中国人民解放军占领了[[甘肅省_(中華民國)|甘肃省]]一带的[[河西走廊|河西走廊]]，夺取了通往新疆的道路。此时的新疆分别由两股势力控制着：一股是在[[迪化|迪化]]（今[[乌鲁木齐|乌鲁木齐]]）的[[中华民国国军|中华民国国军]]，另一股是[[苏联|苏联]]扶持下的、割踞[[伊犁|伊犁]]的三区革命政权。在苏联的要求下，三区政权决定加入中华人民共和国。由[[阿合买提江·哈斯木|阿合买提江·哈斯木]]带领的[[新疆保卫和平民主同盟|新疆保卫和平民主同盟]]代表团前往北京，参加中国人民政治协商会议第一届全体会议，途中坠机，[[扎巴依喀勒山空难|全部身亡]]。大部分原东突领导人统一并入中华人民共和国，后来他们与投降的国军高层将领们，都成为中华人民共和国政府的高官。

时任[[西北军政长官公署|西北军政长官公署]]副长官兼任河西警备总司令、新疆警备总司令的[[陶峙岳|陶峙岳]]是国军驻疆部队的主官，他与[[新疆省政府|新疆省政府]]主席[[包尔汉|包尔汉]]主张投降中共<ref name="a1"/>。9月24日晚12时，陶峙岳正式通电，率领河西警备司令部3.8万名官兵起义，是谓[[酒泉起义|酒泉起义]]。25日，解放军和平进入酒泉，酒泉和平解放。在此情景下，当日凌晨，[[叶成|叶成]]、[[马呈祥|马呈祥]]等忠于中华民国的军官携家眷逃离迪化。陶峙岳代表7万名驻疆官兵，通电起义<ref>{{Cite web |url=http://cpc.people.com.cn/GB/85037/85038/7593662.html |title=陶峙岳与酒泉起义|accessdate=2019-03-26 |author=谢德铭 |date= |work=责任编辑：张慧玲 |publisher=中国共产党新闻网，来源：《人民政协报》|language=Zh-hans }}</ref>。9月26日，新疆省政府主席[[包尔汉|包尔汉]]、迪化市政府市长[[屈武|屈武]]通电起义，标志新疆和平解放<ref name="a1">{{Cite web |url=http://www.sohu.com/a/240028359_372971 |title=新疆和平解放纪实 |accessdate=2019-03-26 |author= |date=2018-07-09 |work=|publisher=搜狐网|language=Zh-hans }}</ref>。

10月1日，中华人民共和国建立。5日，[[中国人民解放军第一野战军|中国人民解放军第一野战军]][[中国人民解放军第一兵团|第一兵团]]在酒泉召开进疆誓师大会，[[王震|王震]]宣布10日开始向新疆进军。11月，第一兵团机关在王震的率领下进入迪化。12月17日，[[新疆省人民政府|新疆省人民政府]]成立。原东突的[[三区民族军|三区民族军]]于12月20日正式被收编为[[中国人民解放军第五军|中国人民解放军第五军]]。30日，国军起义部被改编为[[中国人民解放军第二十二兵团|中国人民解放军第二十二兵团]]<ref name="a1"/>。日后，他们大部分成为了[[新疆生产建设兵团|新疆生产建设兵团]]的一部分。其后，解放军通过[[新疆剿匪|新疆剿匪]]战役，迅速控制了[[新疆省_(中华人民共和国)|新疆省]]的局势。当时只有[[尧乐博斯|尧乐博斯]]和[[乌斯满·巴图尔|乌斯满·巴图尔]]抵制共产党，起兵反抗解放军，但最后都被解放军打败。

1954年10月7日，[[新疆生产建设兵团|新疆生产建设兵团]]成立。1955年，根据[[民族区域自治制度|民族区域自治]]的政策，改新疆省为[[新疆维吾尔自治区|新疆维吾尔自治区]]。
