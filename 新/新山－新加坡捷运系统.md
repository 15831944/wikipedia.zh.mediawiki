'''新山－新加坡捷运系统'''{{notetag|该线路英文名称为Johor Bahru–Singapore RTS，精准翻译应为新山－新加坡捷运系统，也可以简称'''新新捷运'''。}}（{{lang-en|Johor Bahru–Singapore Rapid Transit System}}）是一个规划中跨越[[柔佛海峡|柔佛海峡]]连接[[马来西亚|马来西亚]][[新山|新山]]和[[新加坡|新加坡]]的[[地鐵|地鐵]]线路。本线因此也被称为「'''新柔捷运系统'''」、「'''柔新捷运系统'''」、「'''新柔地铁系统'''」、「'''柔新地铁系统'''」等。一旦建成，这将是马新两国继[[马来亚铁道城际铁路|马来亚铁道城际铁路]]南北线（[[西海岸铁路线|西海岸线]]）之后的第二条铁路线和第一个地铁线。初步研究的第一阶段已于2014年3月完成，<ref>{{cite web|url=http://www.channelnewsasia.com/news/specialreports/parliament/videos/preliminary-engineering/1028738.html|title=Preliminary engineering study on Rapid Transit System link to Johor Bahru completed - Channel NewsAsia<!-- Bot generated title -->|publisher=}}</ref>并计划在2019年完工，然而该计划已延迟至2024年12月31日投入服务，并将在2019年动工<ref name="sinchew.com.my">{{cite news|url=http://www.sinchew.com.my/node/1719853/|title=马新捷运比巴士汽车快半小时．武吉查卡至兀兰30分钟|date=2018-01-16|publisher=[[星洲网|星洲网]]}}</ref>。

2016年12月，新加坡总理[[李显龙|李显龙]]与马来西亚首相[[纳吉·阿都拉萨|纳吉·阿都拉萨]]就[[新柔长堤|新柔长堤]]附近可新建一座高桥达成共识，并在同年12月13日在联合新闻发布会上签署了[[马新高速铁路|马新高速铁路]]双边协议项目<ref>{{cite web|url=http://news.asiaone.com/news/singapore/singapore-and-malaysia-boost-travel-links|title=Singapore and Malaysia to boost travel links|publisher=}}</ref>。他们也同意将会建设一条地铁线，并从[[新加坡地铁汤申－东海岸线|新加坡地铁汤申－东海岸线]]（TEL）的[[兀兰北地铁站|兀兰北地铁站]]通过高架桥连接至新山并和汤东线采用相同的系统和列车。2018年1月16日，两国政府正式签署有关协定<ref>{{cite web | url=https://www.channelnewsasia.com/news/singapore/singapore-malaysia-sign-bilateral-agreement-on-jb-singapore-9864584 | title=Singapore, Malaysia sign bilateral agreement on JB-Singapore Rapid Transit System | date=2018-01-16 | publisher=Channel NewsAsia}}</ref>。

==历史==
2017年7月31日，新加坡基础设施和交通运输部长[[许文远|许文远]]和马来西亚首相署部长[[阿都拉曼达兰|阿都拉曼达兰]]宣布，捷运系统将在2024年12月底正式开通{{R|open}}<ref>[http://www.chinapress.com.my/20170731/特許經營費-馬31-新69-新柔捷運2024年通車/ 特許經營費-馬31-新69-新柔捷運2024年通車]</ref>。

2018年1月16日，马来西亚和新加坡政府在于新加坡总统府举行的新马领导人非正式峰会上正式签署捷运系统双边协定。两国签字的代表分别是新加坡基础建设统筹部长兼交通部长[[许文远|许文远]]和马来西亚首相署部长[[阿都拉曼达兰|阿都拉曼达兰]]，其中马来西亚首相[[纳吉·阿都拉萨|纳吉·阿都拉萨]]和新加坡总理[[李显龙|李显龙]]也共同见证此项签字仪式。这项协定涵盖技术、安全、营运模式、采购和监管框架，及通关安排等等事项，车票费用则由两国合资的运营公司决定。<ref>{{cite news|url=http://www.zaobao.com.sg/realtime/singapore/story20180116-827398|title=新马正式签署新柔地铁系统协定|date=2018-01-16|publisher=[[联合早报|联合早报]]}}</ref><ref>{{cite news|url=http://www.zaobao.com.sg/realtime/singapore/story20180116-827428|title=新柔地铁将助舒缓长堤三分二人流|date=2018-01-16|publisher=[[联合早报|联合早报]]}}</ref>

[[2018年马来西亚大选|2018年马来西亚大选]]后，尽管政府推出了一系列减轻国债的措施，但马来西亚交通部长[[陆兆福|陆兆福]]表示政府将保留柔新捷运系统计划，财政部则会负责重新检讨相关合约、附加条件以降低成本。<ref>{{cite web |title=陆兆福：政府将保留柔新捷运系统（RTS）计划 {{!}} The Malaysian Insight |url=https://www.themalaysianinsight.com/chinese/s/51125 |website=www.themalaysianinsight.com |language=zh}}</ref>但是在2018年7月中旬，新加坡交通部长许文远在国会答复议员提问时表示新加坡还未接到马来西亚政府关於兴建马新捷运系统计划的通知。按照双方签署的马新捷运系统协议，新加坡地铁有限公司（SMRT）和马来西亚国家基础建设公司（Prasarana）应该最迟上个月底成立合资企业来经营马新捷运，不过，合资企业至今还未成立。<ref>{{cite web |title=新加坡未接到大马兴建捷运系统计划通知 |url=http://news.seehua.com/?p=376453 |website=马来西亚诗华日报新闻网 |language=zh-CN}}</ref>

2018年7月底，大马交通部长陆兆福表示大马政府已决定会继续新柔捷运计划，但造价、和新加坡成立联营公司等细节还需要再带到内阁讨论。<ref>{{cite web |title=解决长堤塞车问题·陆兆福：续推行柔新捷运计划 |url=https://www.sinchew.com.my/node/1778963/%E8%A7%A3%E5%86%B3%E9%95%BF%E5%A0%A4%E5%A1%9E%E8%BD%A6%E9%97%AE%E9%A2%98%C2%B7%E9%99%86%E5%85%86%E7%A6%8F%EF%BC%9A%E7%BB%AD%E6%8E%A8%E8%A1%8C%E6%9F%94%E6%96%B0%E6%8D%B7%E8%BF%90%E8%AE%A1%E5%88%92 |website=星洲网 Sin Chew Daily |language=zh-hans}}</ref>
同年10月13日，新加坡陆路交通管理局透露新加坡开始为这项捷运建造工程招標。这项招標是为了兴建马新捷运的隧道，以及从新加坡境內横跨柔佛海峡的部分高架桥。所要建造的隧道和高架桥加起来总长约1.1公里，预计会在2019年中动工兴建。<ref>{{cite web |last1=張 |first1=David |title=新国招標建马新捷运 |url=http://news.seehua.com/?p=398165 |website=马来西亚诗华日报新闻网 |language=zh-CN}}</ref>

2019年1月14日，新加坡交通部长许文远指出，过去数个月来大马政府在考虑更换新柔捷运系统的合资企业伙伴，但截至2018年年9月仍无法确认人选，并截至12月28日仍没有着落，接着大马要求于2019年2月28日确定合资企业伙伴人选。由于这些延误，合资公司迟迟无法成立。另外，大马也因此事于去年8月起，单方面取消新马两国有关聘请营运公司的双边讨论会，导致这项计划进度落后且进展不顺利。<ref>{{cite web |title=许文远：马新未成立合资公司 新柔捷运系统计划进度受阻{{!}}中國報 |url=http://www.chinapress.com.my/20190114/许文远：马新未成立合资公司-新柔捷运系统计划进/ |website=中國報 China Press}}</ref>

== 路线资料 ==

=== 路线 ===
本捷运系统路线总长度预计为4公里左右，并有2个终站。捷运将会在2018年6月30日由两国公交部门组成联营公司管理，在9月30日签特许合约，并在该年第四季进行招标遴选建筑商。这张计划可在2019年动工，并预计在2024年12月31日投入服务。根据陆交会首席执行员阿兹哈鲁丁指出，新新捷运总共有7辆4车厢列车，每天可载送7.2万名乘客，并以时速70公里行驶，乘客将可在30分钟内通关。列车在巅峰时间每4分钟就有一次[[班次|班次]]，离峰时间则为每6分钟一次班次。<ref name="sinchew.com.my"/>

目前计划为只设有两站的地铁线，新加坡末端将位于[[共和理工学院|共和理工学院]]附近，并将和[[新加坡地铁汤申－东海岸线|新加坡地铁汤申－东海岸线]]的[[兀兰北地铁站|兀兰北站]]衔接，马来西亚末端将位于武吉查卡站，而该站的规模也将会比[[吉隆坡中央车站|吉隆坡中央车站]]来的大，并与{{tsl|ms|Gerbang Selatan Bersepadu|南方综合关卡}}（[[新柔长堤|新柔长堤]]）、[[马来亚铁道|马来亚铁道]][[西海岸铁路线|西海岸线]][[新山中央车站|新山中央车站]]和规划中的新山快捷巴士系统（IMBRT）衔接<ref name="iskandar">{{cite web|url=http://www.iskandar.asia/insight/iskandar-malaysia-in-the-news/2-uncategorised/145-final-plan-for-singapore-johor-bahru-rapid-transit-system-rts-link-by-end-of-2014.html|title=Final plan for Singapore-Johor Bahru Rapid Transit System (RTS) link by end of 2014|website=www.iskandar.asia}}</ref>。除此之外，这两个站点都将建有大型多元交通枢纽，内有3个站台，两国的关税、移民和检疫站和1000个停车位，可容纳1万人<ref>{{cite web|url=http://app.lta.gov.sg/apps/news/page.aspx?c%3D2%26id%3D0pt1z213cvn5f2h2t1ti39b4tjkckorbk76uh7r32nykq33d1n |title=Archived copy |accessdate=2014-08-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20140903033605/http://app.lta.gov.sg/apps/news/page.aspx?c=2&id=0pt1z213cvn5f2h2t1ti39b4tjkckorbk76uh7r32nykq33d1n |archivedate=2014-09-03 }}</ref>。<ref name="sinchew.com.my"/>

两端之间的桥梁将是直线而非原本设计的弧形，距离水面约25米高<ref>{{cite news|url=http://www.sinchew.com.my/node/1702384/|title=柔苏丹御准 离水面25公尺‧新柔地铁采直桥设计|date=2017-11-18|publisher=[[星洲网|星洲网]]}}</ref>。轨道进入新加坡后将会转入地下<ref name="sinchew.com.my"/>。
