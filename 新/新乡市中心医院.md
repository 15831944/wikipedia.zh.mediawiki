'''新乡市中心医院'''前身是始建于1949年的平原省人民医院，1953年更名为河南省第三人民医院<ref>{{Cite book|title=中国现代医学科技创新能力国际比较 3|last=中国生物技术发展中心编|first=|publisher=北京：中国医药科技出版社|year=2010|isbn=|location=|pages=3004-3014}}</ref>，1986年改为现名。目前是一家综合性[[三级甲等医院|三级甲等医院]]。它是[[新乡医学院|新乡医学院]]附属医院（直属中心医院和第四临床学院）、[[河南省|河南省]][[博士后|博士后]]研发基地，以及[[卫生部|卫生部]]国际紧急救援中心网络医院。也是[[郑州大学|郑州大学]]医学院、新乡医学院等院校的教学医院和对外开放的'''新乡市红十字医院'''。2009年5月18日，省卫生厅批准在该院设置'''新乡市肿瘤医院'''。<ref name="简介">{{cite web |url=http://www.xxszxyy.com.cn/yyls.php |language= |title=医院简介 |publisher=新乡市中心医院 |date= |accessdate=2013年1月 |deadurl=yes |archiveurl=https://web.archive.org/web/20121130232629/http://www.xxszxyy.com.cn/yyls.php |archivedate=2012-11-30 }}</ref>

地址位于[[新乡市|新乡市]]金穗大道56号。

== 简介 ==
医院现有职工2600余人，开放2116张床位。年门诊量80余万人次、收治病人6万余人次。医院拥有博士、硕士学位者300余人；其中有4人被评为国家有突出贡献专家、享受国务院特殊津贴，2名省管优秀专家，15名新乡市专业技术拔尖人才，7名硕士生导师。拥有74个临床医技科室，曾获1项国家科技奖，20项省科技奖，200余项市科技奖。<ref name="简介"/>

设于该院的机构包括有河南省肿瘤诊断网络新乡分中心、河南省新生儿重症救护网络新乡分中心、新乡市抗癌协会、新乡市肿瘤研究所、新乡市心血管研究所、新乡市泌尿研究所、新乡市法医门诊、新乡德信法医临床司法鉴定所。<ref name="简介"/>

== 重点专科 ==

*新乡市重点医学专科：神经学科、儿科
