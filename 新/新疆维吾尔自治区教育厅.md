'''新疆维吾尔自治区教育厅'''是[[中华人民共和国|中华人民共和国]][[新疆维吾尔自治区人民政府|新疆维吾尔自治区人民政府]]的[[新疆维吾尔自治区人民政府组成部门|组成部门]]。主管自治区教育工作。

== 主要职能 ==
# 全面贯彻执行党和国家有关教育工作的方针、政策和法规，拟定自治区的教育法规、实施办法和管理条例，并监督执行。
# 制定自治区教育事业的发展战略、规划和计划，确立教育事业的发展重点、规模、速度和步骤，并指导协调、贯彻实施；指导、协调全区各级各类学校的德育教育工作。
# 综合管理自治区的基础教育、高等教育、职业技术教育、成人教育、社会力量办学等。
# 指导协调各地、各部门的有关教育工作；承担自治区人民政府教育督导工作，制定自治区教育督导计划和评估方案，并进行指导、监督、检查、评估；参与自治区科技体制改革方案的研究，指导教育体制和办学体制的改革。
# 参与自治区对[[教育经费|教育经费]]预决算的管理，会同有关部门筹措[[教育经费|教育经费]]，制定教育基建投资的方针、政策、法规。归口管理国家和国外对我区的教育援助和教育贷款；会同有关部门制定教育系统劳动工资和人事管理工作的有关政策和规章制度。<ref>{{cite web |url = http://www.xjedu.gov.cn/jytgk/2011/42081.htm |title = 委厅概况 |language = zh-hans |deadurl = yes |archiveurl = https://web.archive.org/web/20140201172805/http://www.xjedu.gov.cn/jytgk/2011/42081.htm |archivedate = 2014-02-01 }}</ref>
