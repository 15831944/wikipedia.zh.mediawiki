'''新-{zh-cn:{{ruby-py|干|gàn}};zh-tw:{{ruby-py|幹|ㄍㄢˋ}};zh-hk:{{ruby-py|幹|gon<sup>3</sup>}}}-县'''，旧名'''新{{ruby-py|-{淦}-|-{zh-cn:gàn;zh-tw:ㄍㄢˋ;zh-hk:gam<sup>3</sup>}-}}'''，位于[[中华人民共和国|中华人民共和国]][[江西省|江西省]]中部，是[[吉安市|吉安市]]北部下辖的一个[[县_(中华人民共和国)|县]]。

== 名称 ==
本称“新淦”，1957年中国大陆推行[[简体中文|简化字]]时更名为“-{新干}-”，繁体写法也由“-{新淦}-”变为“-{新幹}-”。

== 历史 ==
新干是江西十八古县之一，[[秦始皇|秦始皇]]二十六年（公元前221年）置新淦县，治所在淦阳（今[[樟树市|樟树市]]区）；[[隋|隋]]时，县治迁到南市村（今[[金川镇|金川镇]]）；1957年改为新干县。

== 地理 ==
东邻[[乐安|乐安]]，南毗[[永丰|永丰]]、[[峡江|峡江]]，西接[[新余|新余]]，北联[[樟树市|樟树]]、[[丰城|丰城]]；[[赣江|赣江]]纵贯。

== 行政区划 ==
全县辖
*7镇：金川镇、三湖镇、大洋洲镇、七琴镇、麦斜镇、界埠镇、溧江镇
*6乡：桃溪乡、城上乡、潭丘乡、神政桥乡、沂江乡、荷浦乡
