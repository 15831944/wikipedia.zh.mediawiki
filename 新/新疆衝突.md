== 背景 ==
[[东突厥斯坦独立运动|东突厥斯坦独立运动]]支持者认为[[新疆_(消歧义)|新疆]]成为中國的一部分是近代征服战争的结果，并引發[[分離主義|分離主義]]運動。

== 编年 ==
在1949年中華人民共和國成立前，獲得[[蘇聯|蘇聯]]支持的[[東突厥斯坦第二共和國|東突厥斯坦第二共和國]]一直掌握着新疆的实际统治权，但及后被中華人民共和國納入了统治。新疆的主要領導势力为泛突厥主义和伊斯蘭組織{{来源请求}}，最引人关注的是[[东突厥斯坦独立运动|東突厥斯坦獨立運動]]，反對[[中华人民共和国政府|中华人民共和国政府]]对新疆的统治。

1950年代，在[[伊犁|伊犁]]、[[昭苏|昭苏]]、[[和田市|和田]]、[[墨玉县|墨玉]]、[[洛浦县|洛浦]]、[[英吉沙县|英吉沙]]等地发起16次含有宗教成分{{来源请求}}，以[[新疆獨立|新疆獨立]]为目的之[[武装起事|武装起事]]<ref>[http://www.weiweikl.com/MZYJ26.htm 新疆反分裂斗争的历史回顾与经验总结] {{webarchive|url=https://web.archive.org/web/20131007033134/http://www.weiweikl.com/MZYJ26.htm |date=2013-10-07 }}</ref>。

1950年阿不都拉大毛拉在伊犁秘密建立[[大突厥主义伊斯兰党|大突厥主义伊斯兰党]]。伊犁组负责人热合曼诺夫，密谋叛乱。被粉碎。8月18日，[[昭苏县|昭苏县]]暴乱被击溃，1951年10月28日，马力克阿吉暴乱，1952年2月，大突厥主义伊斯兰党被剿灭。

亲[[中国国民党|中国国民党]]的[[烏斯滿·巴圖爾|烏斯滿·巴圖爾]]及其亲信在新疆东部组建[[新疆反共复国军|新疆反共复国军]]搞暴乱，1952年被镇压。

1954年至1957年阿不都依米提大毛拉秉承境外[[穆罕默德·伊敏|穆罕默德·伊敏]]旨意在南疆5次暴乱。1954年12月27日、28日两天，以帕提丁为首用宗教做掩护，带领200多教徒集体抱经宣誓，发动圣战，随后分发武器。12月31日晚间阿卜都.依米提大毛拉、帕提丁领导伊斯兰政府联盟委员会，300多人袭击[[墨玉县|墨玉县]]劳改农场，打死农场指导员1人、班长1人、战士7人，抢夺了部分武器和一辆汽车，砸开监狱牢门，煽动犯人参加暴乱或逃跑，进攻和田县城。他们派人串联，图谋扩大。新疆政府采取政治瓦解和武装围剿相结合的方针，两天平息暴乱。1956年3月9日墨玉暴乱，二百多人袭击[[新疆生产建设兵团第三团|新疆生产建设兵团第三团]]；3月11日五百多人袭击墨玉区委、[[区公所|区公所]]、部队等，杀死1名解放军士兵。4月[[英吉沙县|英吉沙县]]暴乱。阿不都依米提大毛拉策动弟子阿不都•卡德尔哈日与策勒县的艾伯都拉哈日准备暴乱。预备会上打出东突厥斯坦伊斯兰共和国旗号，选举阿不都•卡德尔哈日为主席兼临时总司令，艾伯都拉哈日为副主席和副司令；宣布了21条组织纲领，制定计划。5月4日洛浦暴乱。凌晨，阿不都•卡德尔哈日和艾伯都拉哈日纠集260余人，煽动裹胁1300余人，呼喊着“东突厥斯坦伊斯兰共和国万岁”口号，持凶器冲击洛浦县四区区政府，抢劫粮站和物资仓库，割断了通往县城的电话线，分三路进攻洛浦县城。和田地区政府派公安前往平叛，击毙19人、击伤12人，抓获阿不都•卡德尔哈日，暴乱被平息。逮捕20名首要分子；100多名被裹胁者自首，交出[[冲锋枪|冲锋枪]]、手枪、土枪各1枝及各类凶器10多件。5月24日莎车暴乱。建立东突厥斯坦政府，编唱东突厥斯坦国歌。袭击公安干警，被镇压。1957年4月[[和田暴乱|和田暴乱]]。1959年，几位维吾尔族妇女向政府报告阿不都依米提可疑行踪，公安在地窖中逮捕阿不都依米提。

1950年至1956年有131名阿吉由境外回到和田，其中11人负有间谍任务。有明确的目的和纲领。所有暴乱的目的都是要推翻政府，建立东突厥斯坦伊斯兰共和国，阿不都依米提大毛拉公开宣称伊斯兰共和国政府委员会主席暂时由他代理，将来成功后，应由穆罕默德·伊敏来担任。阿不都依米提大毛拉及其头目号召伊斯兰“[[圣战|圣战]]”，驱逐[[异教徒|异教徒]]；同时散发歪曲历史的地下出版物。

1960年代[[中蘇交惡|中蘇交惡]]决裂后，苏联又开始新疆地区的分裂活动，成功策動原[[伊犁民族軍|伊犁民族軍]]将领[[祖农·太也甫夫|祖农·太也甫夫]]背叛中共，率領民族軍旧部及家眷移往蘇聯[[哈萨克苏维埃社会主义共和国|哈萨克苏维埃社会主义共和国]]作為煽動中国境内分離主義的籌碼<ref>劉學銚：《新疆史論》，知書房，2013年2月，ISBN 978-986-5870-51-5，第253頁</ref>。蘇聯駐伊寧領事館向民眾大派蘇僑證及開放邊境，大量民眾逃亡到蘇聯，不少都是「東突厥伊斯蘭獨立運動」的骨幹{{来源请求}}。蘇聯又在[[哈萨克苏维埃社会主义共和国|哈萨克斯坦]]的[[塔拉斯|塔拉斯]]、[[乌兹别克苏维埃社会主义共和国|乌兹别克斯坦]]的[[撒马尔罕|撒马尔罕]]设立训练中心，从逃亡苏联的中国难民中培训[[维吾尔族|维吾尔族]]、[[哈萨克族|哈萨克族]]特工，派遣他们回中国窃取情报、推動分裂運动<ref name="weiweikl1">[http://www.weiweikl.com/MZYJ26.htm 新疆反分裂斗争的历史回顾与经验总结] {{webarchive|url=https://web.archive.org/web/20131007033134/http://www.weiweikl.com/MZYJ26.htm |date=2013-10-07 }}</ref>。1962年5月29日，「[[維吾爾斯坦|維吾爾斯坦]]」組織領袖[[阿不都卡迪爾|阿不都卡迪爾]]率領群眾攻擊當地政府機關，[[伊寧市|伊寧市]]政府被民眾嚴重破壞。據中國資料顯示，當時參與暴動者約5萬6千人得到蘇聯駐伊寧领事館庇護下進入苏联中亞地区，但學界認為這個數字仍然太保守<ref>劉學銚：《新疆史論》，知書房，2013年2月，ISBN 978-986-5870-51-5，第224頁</ref>。

1960年代末，托乎提库尔班、尼牙孜·乌买尔、司马义·依不拉音·哈斯木帕尔沙等建立全疆范围内分裂组织[[东突厥斯坦人民革命党|东突厥斯坦人民革命党]]。1969年8月20日，[[麦盖提县|麦盖提县]]暴乱分子抢劫县民兵武器库。向边境运动时，被公安围歼，枪支弹药及其他凶器被缴获。阿洪诺夫等10名独立分子被击毙，其余均被俘。在中央政府指示下，自治区革命委员会责成公安保卫部门在全疆进行清理，至1970年3月破获全案。全疆涉案人员达5000余人，东突厥斯坦人民革命党成员1165人，依法惩治230人，做其他处理135人。該黨於[[文化大革命|文化大革命]]後成立，其基層組織更已遍及整個新疆{{来源请求}}。解放军抓获12名向国外求援的人士，以及9名企图潜入国境的外国间谍。後來，由於得到蘇聯支持，其領袖阿洪诺夫曾率領部隊從喀什跑到中蘇邊境，企圖立國，但途中被捕{{来源请求}}。1979年[[苏联入侵阿富汗|苏联入侵阿富汗]]、[[中越战争|中越战争]]爆发后，[[中苏关系|中苏关系]]极度紧张，驻扎在中苏边界西段的苏军曾经在中亚腹地进行过入侵新疆的大规模演习。文化方面，這段時期的獨立思潮主要通过讲课、作报告、著书 、写文章等形式傳播。其中較著名的書籍有《维吾尔人》、《匈奴简史》和《维吾尔古代文学》。<ref>http://dailynews.sina.com/bg/chn/chnnews/ausdaily/20130424/04304479893.html</ref>

1984年，[[胡耀邦|胡耀邦]]採取懷柔政策，重建[[清真寺|清真寺]]。他還制定了[[新疆六條|新疆六條]]，大致內容是：充分自治；修養生息；支援大量經費，促進農牧業生産用於新疆各族人民的迫切需要；恢復新疆文化教育科學事業；進疆幹部分批分期調回內地等。中共新疆自治區黨委在全區範圍內大規模調整各級領導班子，增加少數民族幹部比例，漢族幹部可以提前退休，也可以協助調回內地。该政策既得不到包括維吾爾在內的穆斯林民衆的支持，也得不到占新疆不到40％人口的漢族民衆的擁護{{来源请求}}。

1990年，[[1990年巴仁乡暴乱|巴仁乡暴乱]]。位於土耳其的[[世界伊斯蘭統一者|世界伊斯蘭統一者]]宣佈，要協助東突厥伊斯蘭獨立運動在新疆辦學，並選擇人才到土耳其留學。亦有1000名東突厥伊斯蘭獨立運動年青成員到阿富汗塔里班游擊隊中學習戰爭，亦有不少武器由阿富汗等地偷運入疆。

1993年，由新疆[[和田|和田]]人[[买买提托乎提|买买提托乎提]]和[[阿不都热合曼|阿不都热合曼]]在中国境外发起成立[[东突厥斯坦伊斯兰运动|东突厥斯坦伊斯兰运动]]（簡稱[[东伊运|东伊运]]），其宗旨是在新疆建立一个[[政教合一|政教合一]]的“东突厥斯坦伊斯兰国”。該組織於同年解散。曾在阿富汗受[[基地组织|基地组织]]训练，多次制造恐怖爆炸事件，[[苏联解体|苏联解体]]后，此组织扩散入前苏联[[中亚|中亚]]地区国家，也多次制造爆炸事件，引起这些国家警惕，[[上海合作组织|上海合作组织]]正是为了协作打击恐怖组织而成立的。因为東突厥斯坦伊斯蘭運動与[[奧薩瑪·本·拉登|本拉登]]的[[基地组织|基地组织]]关系密切，很多成员本身就是基地组织的成员<ref>[http://www.gov.cn/gongbao/content/2002/content_61790.htm 关于联合国安理会制裁阿富汗委员会将 “东突厥斯坦伊斯兰运动”列入 受制裁实体名单的通知]</ref>，所以東突厥斯坦伊斯蘭運動被中國和[[美國|美國]]認定为非法的[[恐怖組織|恐怖組織]]。美國對其的這個定性在國内具有争议，有學者評論美國此擧主要是爲博取中國支持它們的[[反恐戰爭|反恐戰爭]]，以及為了改善[[中美關係|中美關係]]。
