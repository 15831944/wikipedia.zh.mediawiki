《'''新喜剧之王'''》是一部2019年中國大陆和香港合拍的[[喜劇片|喜劇片]]。由[[周星馳|周星馳]]和[[邱禮濤|邱禮濤]]導演。不同于1999年的电影《喜剧之王》，本片演員全數換新，改由[[王寶強|王寶強]]、[[鄂靖文|鄂靖文]]主演。該片定於2019年2月5日[[大年初一|大年初一]]在中國大陆和香港上映。周星馳稱本片並非續集，而是女版《[[喜劇之王_(香港電影)|喜劇之王]]》<ref>{{cite web|url=http://media.people.com.cn/n1/2019/0117/c40606-30560079.html |title=周星驰：观众看完电影就会知道了。如果一定要带入《喜剧之王》，那她也应该是“尹天仇”，观众也许会问“尹天仇”怎么变成女的了？我们这次讲的是一个女版“尹天仇”的故事。 |work=人民网|date=2019-01-22}}</ref><ref>{{cite web|url=https://www.zhuaying.com/news/201901/19/181385-1.html |title=周星驰剧透《新喜剧之王》女主：不是柳飘飘，而是女版“尹天仇” |date=2019-01-21}}</ref><ref>{{cite web|url=http://www.mnw.cn/movie/huayu/2116768.html |title=周星驰剧透《新喜剧之王》女主定位：是女版“尹天仇” |date=2019-01-21}}</ref>。

== 前期制作 ==
根据上市公司新文化的财报显示，《新喜剧之王》的成本仅5000万人民币。

周星驰和王宝强均未收取片酬，而是以投资出品的形式参与。《新喜剧之王》拍摄时长两个月，从11月30日宣布定档到2月5日上映也只有两个月零五天的宣传期，而且全片无特效及无3D效果，这些都极大的减少了成本。为了节约成本，周星驰甚至放弃过去一直采用的电视节目宣传，转而使用无成本的网络平台宣传。

《新喜剧之王》是近20年成本最低的周星驰电影，与之形成鲜明对比的是同年同为周星驰导演的《[[美人鱼2|美人鱼2]]》成本高达5亿[[人民币|人民币]]。

事实上，在2018年的前三个季度，周星驰一直在从事《[[美人鱼2|美人鱼2]]》的筹备和拍摄工作。直到10月份才突然加入《新喜剧之王》的剧组。据女主角鄂靖文所说，亲自打电话通知她当选主角的并不是周星驰，而是导演邱禮濤。也正是因为以上所述原因，外界当时普遍猜测周星驰只是《新喜剧之王》的监制，而不是导演。

== 宣传 ==
由周星驰自编自导，王宝强、鄂靖文等主演的电影《新喜剧之王》将于2月5日大年初一全国上映。在20年前的《喜剧之王》中，周星驰讲述了一个跑龙套的演员梦想，对着大海喊出：“努力，奋斗”。20年后，周星驰再次回归到小人物的喜剧创作中，请来了同样是跑龙套出身的王宝强，以及一些新人演员，讲述这些小人物的奋斗史。从剧本阶段算起，这部戏周星驰筹备了三年，两个月拍摄完成。不久前，电影发布了首款预告片，预告片中出现了很多周星驰之前的电影片段，比如“脚被踩扁”、“学各种表情”、“踢盒饭”等，很多网友说周星驰江郎才尽，对此周星驰回应：“好像到我这个年纪之后，每个新戏出来都会要‘江郎才尽’一次。我想我还可以给观众一些不一样的东西，每次都会再努力一下。”在《喜剧之王》结尾，尹天仇失去了演主角的机会，但在《新喜剧之王》中，周星驰却给主角安排了一个比较圆满的结局，“20年前，观众没有和尹天仇一起看到‘天亮’，但20年后，大家可以看到了，天亮以后真的很美。”<ref>{{cite web|url=http://www.xinhuanet.com/ent/2019-01/17/c_1124001324.htm |title=周星驰：《新喜剧之王》结局不再遗憾 |work=新华网|date=2019-01-22}}</ref>

一直處於保密狀態的《新喜剧之王》宣布已於2018年11月26日[[殺青|殺青]]，由[[周星馳|周星馳]]自編自導。其計畫最初始於2010年，預定於該年11月底開拍，並開始對演員進行海選<ref name="Mtime(2010-09-18)">{{cite news|url=http://news.mtime.com/2010/09/18/1441018.html|title=张柏芝复出之作告吹 传因坐地起价惹恼尔冬升|author=beryl|work=Mtime时光网|date=2010-09-18|accessdate=2018-12-03}}</ref>，但最後未能執行，而後一直處於{{link-en|籌拍地獄|Development hell}}之中。

2019年1月11日，主題曲MV《疾风》在官方發表會上首度釋出<ref name="Mtime(2019-01-11)">{{cite news|url=http://news.mtime.com/2019/01/11/1588069.html|title=周星驰《新喜剧之王》首曝演员阵容|author=hello王小羊|work=Mtime时光网|date=2019-01-11|accessdate=2019-01-16}}</ref><ref name="臺灣新浪網(2019-01-11)">{{cite news|url=https://news.sina.com.tw/article/20190111/29645708.html|title=「疾風少女」演唱《新喜劇之王》主題曲《疾風》！|work=臺灣新浪網|date=2019-01-11|accessdate=2019-01-16}}</ref>；此外，還宣布了剩餘的主演陣容，包括[[張全蛋|張全蛋]]、[[張琪_(中國女演員)|張琪]]、[[袁兴哲|袁兴哲]]、[[黄骁鹏|黄骁鹏]]、[[蔡哲睿|蔡哲睿]]與[[苗溢伦|苗溢伦]]<ref name="Mtime(2019-01-11)" />。

==劇情==
如夢（鄂靖文 飾）是一個相貌和身材都欠佳的大齡臨時演員，一直希望當女主角，在片場當了十多年的跑龍套和替身，但在成為明星的之路仍沒有任何起色，每天都面對被嘲笑和拒絕，她的明星夢不為父親（張琪 飾）和親友所認同。口硬心軟的父親不支持她追夢，但仍多次到片場暗地裡為她爭取較好的待遇。支持她追夢的只有男友查理（張全蛋 飾），她把賺到微薄金錢都交給男友存入「結婚基金」。如夢和閏密小米（景如洋 飾）逛街時遇到星探，但被看中的只是外表漂亮的小米。在一次機會中，如夢參與一國際大片《白雪公主》的演出，飾演片中主角白雪公主的是她少年時的偶像馬可（王寶强 飾）。曾經當紅的馬可如今早已過氣，失去了演技和觀眾的他，既驕傲又自卑，懷著滿腔狂躁，給予搭戲的如夢極大的打擊和折磨。另外，在一次偶然機會，如夢發現男友查理原來是一名男妓(自稱為合約男友)和渣男，一直在欺騙她的感情和金錢。受到重重打擊的如夢，在風雨中幾乎遇上交通意外。之後如夢放棄追星，回到父母身邊，任職餐廳服務員。一天，突然鹹魚翻身人氣急升的馬可探訪如夢，通知她入選了「周星馳導演新電影」的選角，並激勵她「只要不投降，便是成功」，她得到父母的支持參與選角，並憑著重演自己發現男友是騙子的一幕而入選。一年後，在電影頒獎禮中，如夢當選「最佳女主角」。

== 發行 ==
《新喜剧之王》定於2019年2月5日大年初一在中國、香港、馬來西亞、新加坡、越南上映<ref>{{cite web|url=https://baijiahao.baidu.com/s?id=1622850493720125650&wfr=spider&for=pc |title=周星驰：观众看完电影就会知道了。如果一定要带入《喜剧之王》，那她也应该是“尹天仇”，观众也许会问“尹天仇”怎么变成女的了？我们这次讲的是一个女版“尹天仇”的故事。 |date=2019-01-21}}</ref>。

=== 評價 ===
影評對《新喜劇之王》的口碑成兩極化<ref>{{cite news |title=《新喜劇之王》反應兩極 |date=2-19-02-23 |author=鄭明仁 |url=https://news.mingpao.com/pns/%E4%BD%9C%E5%AE%B6%E5%B0%88%E6%AC%84/article/20190223/s00018/1550859162392/%E3%80%8A%E6%96%B0%E5%96%9C%E5%8A%87%E4%B9%8B%E7%8E%8B%E3%80%8B%E5%8F%8D%E6%87%89%E5%85%A9%E6%A5%B5 |work=明報 }}</ref>。批評內容「炒冷飯」；另一方覺得電影是不錯，認為一如既往的[[心靈雞湯|心靈雞湯]]。

=== 票房 ===
《新喜剧之王》在中国内地取得票房6.24亿人民币<ref>{{cite web |title=新喜剧之王大陆票房 |url=https://maoyan.com/films/580298 |website=猫眼票房 |accessdate=17 March 2019}}</ref>。就票房而言，直至2019年3月1日，《新喜劇之王》之中國內地票房只有6.2億元，由於已經是上映檔期的尾聲，已無望衝破10億元<ref>{{cite web |title=《新喜剧之王》破十亿无望 |work=新浪 |url=https://k.sina.com.cn/article_6435132734_17f90553e00100g9q1.html }}</ref>。這是周星馳自《西遊降魔篇》12億票房之後，首部導演作品未能突破10億元。《新喜劇之王》6.2億元的票房落後於同在2月5日上映的電影：《流浪地球》（44.5億）、《瘋狂的外星人》（21.8億）、《飞驰人生》（16.8億）及《熊出没·原始时代》（7.0億）等<ref>{{cite web |title=CBO中國票房 |url=http://www.cbooo.cn/ }}</ref>。

《新喜剧之王》是2019年首部中国海外票房破千万[[美元|美元]]的华语电影<ref>{{cite web|url=https://www.sohu.com/a/297612663_120005481?sec=wd |title=别小看周星驰的《新喜剧之王》，在东南亚，票房还是第一！  |date=2019-02-25}}</ref>。

《新喜剧之王》于2019年3月12日登陆中国内地网络视频播放平台，《新喜剧之王》在[[网络视频|网络视频]]播放平台采用了[[付费模式|付费模式]]，会员需要6元，非会员需要12元才可完全观看。根据数据显示，截止至3月15日下午14点，光是在[[腾讯视频|腾讯视频]]平台上，《新喜剧之王》的[[播放量|播放量]]达到1.3亿，按照单次6元计算，网络收益至少7.8亿人民币。按照目前的网络收益计算，已经超过内地影院上映票房。<ref>{{cite web|url=http://ent.ifeng.com/c/7l3wrTxChcm |title=《新喜剧之王》网络播放量破亿 目前收益已超7亿  |date=2019-03-15}}</ref><ref>{{cite web|url=http://www.1905.com/news/20190315/1357181.shtml |title=《新喜剧之王》网络播放量破亿 目前收益已超7.8亿  |date=2019-03-15}}</ref>
