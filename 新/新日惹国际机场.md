'''日惹国际机场'''（[[印尼语|印尼语]]：Bandar Udara Internasional Yogyakarta）也叫'''古冷柏格国际机场'''（[[印尼语|印尼语]]：Bandar Udara Internasional Kulonprogo），是[[印度尼西亚|印度尼西亚]][[日惹特区|日惹特区]]的一座在建机场，位于[[古冷柏格县|古冷柏格县]]境内，[[印尼机场公司|印尼第一机场公司]]投资7兆印尼盾<ref>{{cite web |url=http://www.guojiribao.com/shtml/gjrb/20150207/181762.shtml |title=日惹将建价值7万亿盾新机场 |last= |first= |date=2015-02-07 |website=www.guojiribao.com |publisher=国际日报 |access-date=2017-11-11 |quote=}}</ref>，建设总投资达10.9兆盾<ref>{{cite web |url=http://www.shangbaoindonesia.com/indonesia-finance/pembangunan-perumahan%E5%85%AC%E5%8F%B8%E7%9E%84%E5%87%86%E5%BC%80%E5%8F%91%E6%80%BB%E5%80%BC10-9%E5%85%86%E7%9B%BE%E6%97%A5%E6%83%B9kulon-progo%E6%9C%BA%E5%9C%BA.html |title=embangunan Perumahan公司瞄准 开发总值10.9兆盾日惹Kulon Progo机场 |last= |first= |date=2017-07-04 |website=www.shangbaoindonesia.com |publisher=印度尼西亚商报 |access-date=2017-11-11 |quote= |archive-url=https://web.archive.org/web/20171112021111/http://www.shangbaoindonesia.com/indonesia-finance/pembangunan-perumahan%E5%85%AC%E5%8F%B8%E7%9E%84%E5%87%86%E5%BC%80%E5%8F%91%E6%80%BB%E5%80%BC10-9%E5%85%86%E7%9B%BE%E6%97%A5%E6%83%B9kulon-progo%E6%9C%BA%E5%9C%BA.html |archive-date=2017-11-12 |dead-url=yes }}</ref>，由印度GVK公司承建<ref>{{cite web |url=http://obor.chinca.org/scdt/56118.jhtml |title=印度GVK公司将承建印尼Kulon Progo国际机场 |last= |first= |date=2016-02-26 |website=obor.chinca.org |publisher=中国对外承包工程商会 |access-date=2017-11-11 |quote= |archive-url=https://web.archive.org/web/20171111204821/http://obor.chinca.org/scdt/56118.jhtml |archive-date=2017-11-11 |dead-url=yes }}</ref>。预计于2019年3月开放运营，由印尼第一机场公司经营<ref>{{Cite news|url=http://www.thejakartapost.com/news/2017/01/27/new-airport-to-cater-to-more-travelers-in-yogyakarta.html|title=New airport to cater to more travelers in Yogyakarta|newspaper=The Jakarta Post|accessdate=2017-08-20}}</ref>。

新的日惹国际机场将取代拥挤的[[阿迪苏西普托国际机场|阿迪苏西普托国际机场]]，现在服务[[日惹|日惹]]的阿迪苏西普托国际机场旅客量已经远超过其服务承载能力（阿迪苏西普托国际机场的服务承载能力仅为每年300万人次<ref name="印度">{{cite web |url=http://www.shangbaoindonesia.com/business/%E5%8D%B0%E5%BA%A6%E6%9C%89%E6%84%8F%E5%9C%A8%E6%97%A5%E6%83%B9kulon-progo%E5%BB%BA%E6%9C%BA%E5%9C%BA.html |title=印度有意在日惹Kulon Progo建机场 |last= |first= |date=2011-02-24 |website=www.shangbaoindonesia.com |publisher=印度尼西亚商报 |access-date=2017-11-11 |quote= |archive-url=https://web.archive.org/web/20171112021143/http://www.shangbaoindonesia.com/business/%E5%8D%B0%E5%BA%A6%E6%9C%89%E6%84%8F%E5%9C%A8%E6%97%A5%E6%83%B9kulon-progo%E5%BB%BA%E6%9C%BA%E5%9C%BA.html |archive-date=2017-11-12 |dead-url=yes }}</ref>），预计新机场每年吞吐量为3,000万名旅客<ref>{{cite web|url=http://www.traveldailymedia.com/china/10815/rirejichangjianshegongcheng5yuekaishi/|title=日惹机场建设工程5月开始|last=|first=|date=|website=www.traveldailymedia.com|publisher=今日旅业新闻|access-date=2017-11-11|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20171112022029/http://www.traveldailymedia.com/china/10815/rirejichangjianshegongcheng5yuekaishi/|archivedate=2017年11月12日|df=}}</ref>。第一阶段建设的跑道长宽为3,250×60米，第二阶段将会把跑道再延长350米<ref>{{Cite news|url=https://en.tempo.co/read/news/2017/01/27/056840320/Kulonprogo-Airport-Project-to-be-Sped-Up|title=Kulonprogo Airport Project to be Sped Up|newspaper=Tempo|accessdate=2017-08-20}}</ref>。机场的总占地面积为645公顷<ref>{{Cite news|url=http://www.antaranews.com/en/news/111943/construction-of-toll-road-in-yogyakarta-being-reviewed-minister|title=Construction of toll road in Yogyakarta being reviewed: Minister|newspaper=Antaranews|accessdate=2017-08-20}}</ref>。

未来机场将有高速公路和铁路连接日惹市区<ref>{{Cite news|url=http://www.thejakartapost.com/news/2016/05/27/angkasa-pura-i-to-provide-rp-8t-for-new-yogyakarta-airport.html|title=Angkasa Pura I to provide Rp 8T for new Yogyakarta airport|newspaper=The Jakarta Post|accessdate=2017-08-20}}</ref>。铁路线将从日惹的Kedundang车站通往机场，使用现有的轨道外加4公里新建路线，由于需要新建的铁路线较短，预计在机场就绪开通时铁路也将投入使用<ref>{{cite news |url=http://wartakota.tribunnews.com/2015/11/14/kereta-api-layani-penumpang-bandara-kulo-progo-yogyakarta |title=Kereta Api Layani Penumpang Bandara Kulon Progo Yogyakarta |newspaper=Tribunnews |accessdate=2017-08-20}}</ref>。

2017年8月，由于新机场在没有环评手续和排污许可证的情况下进行施工场地平整工作引发争议，带来的扬尘问题导致周边的民众对第一机场公司提出了严重抗议，而第一机场公司则辩解称平整土地和机场施工是两个独立的单元，平整土地不属于机场建设施工的部分。<ref>{{cite web |url=https://www.sohu.com/a/168582038_99945055 |title=没有环评也要施工—印尼日惹机场引争议 |author=布什米尔 |date=2017-08-31 |website=www.sohu.com |publisher=慧检测 |access-date=2017-11-11 |quote= |archive-url=https://web.archive.org/web/20171112073953/https://www.sohu.com/a/168582038_99945055 |archive-date=2017-11-12 |dead-url=yes }}</ref>
