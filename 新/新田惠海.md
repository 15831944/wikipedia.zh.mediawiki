'''新田惠海'''（{{bd|1985年|12月10日}}），日本女性[[配音員|配音員]]、歌手，出身於[[長野縣|長野縣]][[長野市|長野市]]，前屬『[[株式會社S|株式會社S]]』<ref name="s-prof">{{Cite web|url=http://s-inc.jp/artist.html|title=株式会社S/所属アーティスト|accessdate=2011-02-08|deadurl=yes|archiveurl=https://web.archive.org/web/20130530010036/http://s-inc.jp/artist.html|archivedate=2013-05-30}}</ref>，目前隸屬於『DIFFERENCE』。身高153cm。

== 经历 ==
长野县篠之井高等学校毕业<ref name="post-40fa.html">{{cite news |title=声に夢乗せて 信州発 若手声優3 |newspaper=[[信濃每日新聞|信濃每日新聞]] |date=2011-11-19 |accessdate=2013-05-24}}</ref>。高中时代起与友人共同进行乐队活动，后就讀於县外的[[洗足學園音樂大學|洗足學園音樂大學]]。<ref>{{Cite web |date=2015-12-11 |url=http://www.senzoku.ac.jp/music/school/course/anime/?id=753 |title=「第66回NHK紅白歌合戦」に新田恵海さん（本学出身）の出演が決定！ |publisher=[[洗足學園音樂大學|洗足學園音樂大學]] |access-date=2016-04-13 |archive-url=https://web.archive.org/web/20160422120357/http://www.senzoku.ac.jp/music/school/course/anime/?id=753 |archive-date=2016-04-22 |dead-url=yes }}</ref><br />

2009年3月，在株式会社S 新进艺人颁奖中获金奖。<ref name="s-prof"/>并且当天在颁奖会场获知自己与同事务所的社长・佐藤裕美的生日和血型完全一致。<ref>[[初音島III|風見学園新聞部]] 第41回配信分にて。</ref>

2010年4月，加入株式会社S。<ref name="s-prof"/>作为游戏・OVA《T.P櫻～時空樹防衛戰～》中白河小鸟一角的声优出道。<ref name="s-prof"/><ref>{{Cite web|url=http://nittaemi.exblog.jp/14058546/|title=えみりんごの唄 : 「T.P.さくら」前夜祭|accessdate=2011-02-13}}</ref>尽管原以歌手为目标参加了事务所的甄选，经过佐藤裕美的劝诱开始了声优活动。<ref>{{Cite web |author=新田恵海 |date=2014-01-19 |url=http://nittaemi.exblog.jp/21588290/ |title=おへんじ29 |work=えみりんごの唄 |publisher=エキサイト |accessdate=2014-07-11}}</ref>与声优活动同时承担的还有在Elements Garden提供乐曲的样本，以及合唱任务。

2012年1月，以PSP游戏《小魔女帕妃》主题曲《HEARTFUL WISH》一曲作为歌手出道。自称“小时后就喜欢音乐，长大了唱歌也是理所当然的”。

2013年開始感覺喉嚨不太舒服<ref name="disease of throat">{{Cite web|date=2015-02-04|url=http://emitsun.jp/other/%e6%96%b0%e7%94%b0%e6%81%b5%e6%b5%b7%e3%81%ab%e9%96%a2%e3%81%97%e3%81%a6%e3%80%81%e5%a3%b0%e5%b8%af%e7%b5%90%e7%af%80%e6%b2%bb%e7%99%82%e3%81%ae%e3%81%8a%e7%9f%a5%e3%82%89%e3%81%9b%e3%81%a8%e3%81%8a/|title=新田恵海に関して、声帯結節治療のお知らせとお詫び|work=新田恵海Music web site|publisher=ブシロードミュージック|accessdate=2015-03-11|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20150209232710/http://emitsun.jp/other/%E6%96%B0%E7%94%B0%E6%81%B5%E6%B5%B7%E3%81%AB%E9%96%A2%E3%81%97%E3%81%A6%E3%80%81%E5%A3%B0%E5%B8%AF%E7%B5%90%E7%AF%80%E6%B2%BB%E7%99%82%E3%81%AE%E3%81%8A%E7%9F%A5%E3%82%89%E3%81%9B%E3%81%A8%E3%81%8A/|archivedate=2015-02-09}}</ref><ref>{{Cite web |date=2015-02-03 |url=https://twitter.com/Nitta_Emi/status/562809244341526528 |title=新田恵海です。実は年末から… |work=@Nitta_Emi |publisher=新田恵海本人のツイッターアカウントより |accessdate=2015-02-05}}</ref>。2015年2月診斷發現聲帶結節（長繭），宣布短期期間休養<ref name="disease of throat"/><ref>{{Cite web |date=2015-02-03 |url=https://twitter.com/Nitta_Emi/status/562809244341526528 |title=新田恵海です。実は年末から… |work=@Nitta_Emi |publisher=新田恵海本人のツイッターアカウントより |accessdate=2015-03-11}}</ref>。当年以「μ's」成員身分獲得第九回[[聲優獎|聲優獎]]歌唱賞<ref>{{Cite web|publisher=[[声優アワード|声優アワード]]|url=http://seiyuawards.jp/201503/06170044.php|title=第九回声優アワード授賞者発表|accessdate=2015-03-11|language=ja|deadurl=yes|archiveurl=https://web.archive.org/web/20150325034617/http://seiyuawards.jp/201503/06170044.php|archivedate=2015-03-25}}</ref>。

2014年9月，以《{{lang|ja|笑顔と笑顔で始まるよ!}}》一曲作为独立歌手出道。

2015年11月8日，初次開辦個人演唱會，是次演唱會名為「EMUSIC〜始まりの場所〜」(EMUSIC~起始之地~)， 於TOKYO DOME CITY HALL舉行。

2015年12月31日，以μ's一員身份於第66回NHK紅白歌合戰登場。(首次登上)

2016年4月被八卦杂志爆出疑似在以声优身份出道前曾参与成人色情影片拍摄，有關爆料並無直接證據證明屬實，其本人与事务所予以否认，引发Lovelive爱好者等人激烈的争论和猜测<ref>{{cite web|title=μ‘s成员新田惠海被疑出演成人片 事务所否认|url=http://www.sohu.com/a/67589002_126202|website=搜狐娱乐|accessdate=2017-09-01}}</ref>。

2016年6月、於橫濱國際平和會議場開辦了第二次個人演唱會名為「{{lang|ja|新田恵海 LIVE 2016 EAST EMUSIC～つなぐメロディー～}}」(新田惠海 LIVE 2016 EAST EMUSIC～連接的旋律～)

2017年8月9日，其事務所發出消息，指新田惠海將會由8月末完約後移籍至DIFFERENCE。<ref>{{Cite web|date=2017-08-09|url=http://www.eminitta.jp/info/view/145|title=新田恵海退所に関しまして|accessdate=2017-08-14|deadurl=yes|archiveurl=https://web.archive.org/web/20170809122800/http://www.eminitta.jp/info/view/145|archivedate=2017-08-09}}</ref>

2018年10月，其官網上發出新田惠海被診斷出有[[肌張力障礙|肌張力障礙]]的消息。

== 人物 ==
兴趣和喜好是逛咖啡店、卡拉OK、薄荷、恐龙、献血<ref>{{Cite web|url=http://s-inc.jp/artist.html|title=株式会社S-所属アーティスト|accessdate=2013-05-24|deadurl=yes|archiveurl=https://web.archive.org/web/20130530010036/http://s-inc.jp/artist.html|archivedate=2013-05-30}}</ref><ref>{{Twitter|Nitta_Emi|新田惠海}}</ref>。是個重度恐龍迷，曾在LoveLive Fight Club中模仿恐龍叫聲「噗吼」的聲音，也是全日本恐龍鑑定三級。喜欢的人物是坂本真綾。

另外也是Sound Schedule的粉丝，并且参加了演唱会。2014年9月8日于东京电视台播放的《{{lang|ja|アニメマシテ}}》（与徳井青空共同主持）中，Sound Schedule的主唱大石昌良作为嘉宾出场时，吐露了自己是Sound Schedule的忠实粉丝的事实，展现出了紧张不安的姿态。

很容易哭，常在錄音後哭成淚人兒。

曾在錄音現場配音時，大喊LoveLive高坂穗乃果的臺詞「大雨快停」時，身穿的吊帶肩帶滑落，被其他LoveLive成員目擊到，據本人在Fight Club提到，暫時不會穿那件衣服了。

家裡成員有爺爺、奶奶、父母、兩位妹妹和弟弟。
