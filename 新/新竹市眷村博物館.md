'''新竹市眷村博物館'''是專門介紹新竹市眷村文化的博物館，在[[林政則|林政則]]市長任內規劃建設，成立於2002年，位於[[新竹市|新竹市]][[北區_(新竹市)|北區]]<ref name="TaiwanAcademy">{{Cite web |url=http://zh-tw.taiwanacademy.tw/article/index.php?sn=6589 |title=新竹市眷村博物館介紹 |accessdate=2014-10-26 |work= |publisher=臺灣書院 |date= }}</ref>，為全臺灣第一間以眷村為主題的博物館<ref>{{Cite web |url=http://www.studioss.com/cultural/cultural-05.htm |title=新竹市地方文化館介紹 |accessdate=2014-10-26 |work= |publisher=米力文化空間 |date= }}</ref>。該博物館展示新竹市眷村的相關文物，大多是由新竹市眷村居民捐贈而來<ref name="TaiwanStory">{{Cite web |url=https://storytaiwan.tw/Story_Detail_c.aspx?n=BE0A58B8F5E85B8F&s=1D71A01438244DF8 |title=新竹市眷村博物館的幕後推手 |accessdate=2014-10-26 |work=台灣故事島 |publisher=文化部 |date=2014-03-24 }}</ref>。

==歷史==
1949年[[國共內戰|國共內戰]]後，國民政府撤退來台，各省軍民、政府人員被迫轉往台灣定居。據統計，1945年至1950年，中國大陸各地近200萬軍民遷入台灣<ref>{{cite book |author=《文史參考》2011年第22期 |title=寶島眷村背后的國共恩仇 |publisher=人民日報社 |}}</ref>。國民政府為了解決人口激增帶來的居住問題，開始興建房舍或安排[[宿舍|宿舍]]，並將新住民以軍種、職業、特性等，分別群聚於一定範圍，即為現在所知的「眷村」。新竹市眷村就擁有47個，比例相當高。也是在地文化與[[臺灣歷史|臺灣歷史]]裡不可或缺的重要歷史文化資產。

[[新竹市政府|新竹市政府]]為了要展現新竹市眷村的文化，於1997年[[童勝男|童勝男]]市長任內全國文藝季舉辦「竹籬笆內的春天」活動中發想成立一座博物館來展示新竹市眷村的文化資產，之後由[[蔡仁堅|蔡仁堅]]市長團隊推動成立該博物館，原計畫該博物館計劃成立於新竹市空軍一村第八區，但因為土地產權問題，最終[[林政則|林政則]]市長改變地點至原[[法務部調查局|法務部調查局]]新竹市調查站後的[[新竹市環境保護局|新竹市環境保護局]]舊址<ref>{{cite web|url=http://www.hcccb.gov.tw/chinese/16museum/mus_b02.asp?station=104&museum_id=104|title=眷村博物館源起|publisher=新竹市文化局|accessdate=2014-10-26}}</ref>。

眷村博物館於2002年12月28日開館，原本只使用一樓展區，後於2006年10月28日二至三樓的展覽空間設計並重新開放<ref>{{cite web|url=http://cloud.culture.tw/frontsite/map/assetsMapTableListAction.do?method=showMapDetail&mainType=3&mainTypePK=700|title=新竹市眷村博物館|publisher=文化部|accessdate=2014-10-26}}</ref>。

==展示==
新竹市眷村博物館的展覽空間共三層樓，主要展示的文物，大多來自於住在新竹市眷村的居民捐贈而來<ref name="TaiwanStory"/>，包含了眷村生活使用的器具與用品，其中更是從軍用品改造而來；此外，該博物館更將二樓的空間，改造成新竹市眷村的模擬場景，讓參與者感受新竹市眷村的氛圍<ref name="TaiwanAcademy"/><ref>{{Cite web |url=https://www.youtube.com/watch?v=un6LO0zcW-8#t=21 |title=新竹眷村博物館二至三樓整修暨整體環境改造 |accessdate=2014-10-26 |work= |publisher=墨田事業 Sumida Group |date=2013-04-09 }}</ref>。

此外，該博物館會舉辦特展與活動，介紹新竹市眷村美食與新竹市眷村相關連的軍事、部隊文物等文化資產<ref>{{Cite news |url=http://www.chinatimes.com/realtimenews/20141015004925-260405 |title=新竹市眷村藝術季 19日將登場 |accessdate=2014-10-26 |work=中時電子報 |publisher=中時媒體集團 |date=2014-10-15 }}</ref><ref>{{Cite web |url=http://www.ocac.gov.tw/OCAC/SubSites/Pages/Detail.aspx?site=7e595f8b-054a-4a73-826f-7f48a802c5d8&nodeid=1071&pid=21231 |title=竹市眷村麵特展 回味眷村時光 |accessdate=2014-10-26 |work=宏觀全球訊息 |publisher=南非華僑服務站 |date=2014-09-13 }}</ref>。
