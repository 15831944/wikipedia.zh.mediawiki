'''新疆生产建设兵团第一师幸福农场'''，位于[[新疆维吾尔自治区|新疆维吾尔自治区]][[阿拉尔市|阿拉尔市]][[绿原镇|绿原镇]]，是[[新疆生产建设兵团第一师|新疆生产建设兵团第一师]]阿拉尔垦区农场。

==简介==
在[[新疆生产建设兵团第一师|新疆生产建设兵团第一师]]团场土地资源整合的背景下，幸福农场于2013年6月25日组建成立，场部设在阿拉尔市绿园镇。截至2017年，幸福农场有常住人口3754人，在册职工1067人、干部204人，离退休干部14人。幸福农场规划总面积26.37万亩，其中耕地面积占12.89万亩，主要种植[[红枣|红枣]]、[[棉花|棉花]]。从2013年到2016年，幸福农场[[国内生产总值|国内生产总值]]从3.79亿元人民币增长到4.96亿元人民币，职均收入从4.7万元人民币增加到5.11万元人民币，幸福农场的基础设施和公共服务设施建设不断完善，非公有制经济、餐饮、商贸等服务业正逐步兴起<ref>{{Cite web |url = http://www.xjxnw.gov.cn/c/2017-06-06/1209124.shtml |title =  说说幸福农场|author =  |date = 2017-06-06 |publisher = 新疆兴农网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.xinjiangnet.com.cn/xjbt/stxw/201406/t20140611_3914007.shtml  |title = 一师幸福农场加快基础设施建设 |author =  |date = 2014-06-11 |publisher = 新疆网 |language =  |accessdate =  |quote =  }}</ref>。幸福农场下设幸福农场畜牧公司，2015年幸福农场的饲草地和养殖场投入使用，正式开始饲草种植和肉羊肉驴养殖工作，截至2016年幸福农场已成立2个养殖专业合作社<ref>{{Cite web |url =http://www.xjxnw.gov.cn/c/2016-11-02/1041816.shtml |title =“畜”势勃发：第一师幸福农场大力发展畜牧业  |author =  |date = 2016-11-02 |publisher = 新疆兴农网|language =  |accessdate =  |quote =  }}</ref>。2017年，幸福农场果业公司开展“以劳帮困”，帮助困难职工脱贫致富<ref>{{Cite web |url = http://www.xjxnw.gov.cn/c/2017-05-27/1204335.shtml|title = 第一师幸福农场果业公司开展“以劳帮困”助困难职工脱贫致富 |author =  |date = 2017-05-27 |publisher = 新疆兴农网  |language =  |accessdate =  |quote =  }}</ref>。
