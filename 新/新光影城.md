'''新光影城'''（[[英語|英語]]：'''Shin Kong Cinemas'''），為台灣的連鎖[[電影院|電影院]]系統之一。2004年接替本年退出台灣市場、原由[[派拉蒙影業|派拉蒙影業]]與[[環球影業|環球影業]]經營之[[UCI派拉蒙環球影城|UCI派拉蒙環球影城]]，目前共有3個據點。新光影城於2013年開始陸續改裝影城設備，2015年引進[[杜比全景聲|ATMOS]]，2016年引進[[MX4D|MX4D]]，目前共有3個ATMOS影廳、1個MX4D影廳。

== 影城概要 ==
:'''台北新光影城'''
*台北市萬華區西寧南路36號4樓
*藏身於[[西門町|西門町]][[獅子林商業大樓|獅子林商業大樓]]內，原名UCI獅子林影城，其後影廳比照歐美影城設備標準重新裝潢呈現，於2000年1月底重新開幕，一共有3間放映影廳（金獅廳、銀獅廳及寶獅廳）。 
*2015年全新改裝，放映設備標準新穎，音響採用杜比7.1聲道數位環繞系統，座椅升級為英國頂級品牌FERCO豪華座椅，久坐不累，搭配110公分間距，寬敞舒適。台北新光影城，以美式影城管理經營，且常舉辦影展及觀摩影片，服務廣大熱愛電影的消費者。帶給影迷寬敞的空間與美好的視覺饗宴。<br>共 3 間放映廳。<br>共 939 個座位。

:'''台中新光影城'''
*台中市西屯區台灣大道三段301號
*台中新光影城座落於新光三越百貨台中中港店13、14樓，擁有兩座全球最專業的杜比全景聲DOLBY® ATMOS™系統，以最佳立體音效環境，提供顧客如臨其境之撼。
*2016年6月，建置全球第一座杜比全景聲DOLBY® ATMOS™搭配美國MediaMation MX4D，完美呈現聲歷其境的完美震撼。
*搭配五座DOLBY杜比 7.1 環繞音響系統以及全數位化影廳，除此之外，在視覺方面，影廳並設有4K數位投影機、
*高幀率3D(HFR)系統，搭配特殊規格影片之時，更可感受到放映畫質的亮度、鮮銳度以及3D的景深層次。<br>共 8 間放映廳。<br>共 2006 個座位。

:'''台南新光影城'''
*台南市中西區[[西門路|西門路]]一段658號
*台南新光影城座落於新光三越台南西門店的7~9樓，交通便利，停車方便。台南新光影城是南台灣首屈一指的美式現代化設計影城，不但擁有南台灣最大、座位數最多的影廳，影廳採電腦化自動放映及數位環境音響設備，挑高球場式座椅及超大弧形寬螢幕，帶給消費者身歷其境的影音震撼!
*2016年3月建置台南市第一座杜比全景聲DOLBY®ATMOS™系統，共計53.3聲道，搭配19公尺寬巨型銀幕，影音效果完美震撼。<br>共 8 間放映聽。<br>共 2019 個座位。

== MX4D  體感影廳 ==
新光影城於2016年6月21日正式引進美國MediaMation開發之全新技術體感影廳，讓觀眾不只再只是看電影，更是置身其中感受電影的情節，身歷其境體驗各種特效如，泡泡、颳風、座椅搖晃、閃光、水霧等特效。
:台中新光影城
*2016年6月21日開幕
*全球唯一有搭配ATMOS技術之4D影廳，共有112席，另設4席輪椅席，開幕影片為「ID4：星際重生」
