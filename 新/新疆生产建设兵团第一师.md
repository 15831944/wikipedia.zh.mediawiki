'''新疆生产建设兵团第一师'''，是[[新疆生产建设兵团|新疆生产建设兵团]]的师，与[[新疆维吾尔自治区|新疆维吾尔自治区]][[阿拉尔市|阿拉尔市]][[师市合一|师市合一]]。

==沿革==
新疆生产建设兵团第一师的前身，是[[中国工农红军第二方面军|中国工农红军第二方面军]][[红六军团|第六军团]]。[[抗日战争|抗日战争]]后期，为[[八路军|八路军]][[120师|120师]][[359旅|359旅]]。[[第二次国共内战|第二次国共内战]]后期改编为[[中国人民解放军第一野战军|中国人民解放军第一野战军]]第一兵团第二军步兵第五师。1953年3月，新疆军区下达命令，撤销步兵第五师番号，改编为'''[[新疆军区|新疆军区]]农业建设第一师'''。1953年6月，新疆军区农业建设第一师正式成立。这比[[新疆生产建设兵团|新疆生产建设兵团]]的成立还早一年。新疆生产建设兵团成立后，1954年10月该师划归兵团建制，成为'''新疆生产建设兵团农业建设第一师'''（简称“'''农一师'''”），是新疆生产建设兵团第一个[[正师级|正师级]]建制。自1950年代起，农一师三代人用60多年将[[塔里木河|塔里木河]]沿岸的千年荒滩逐步变成了绿洲<ref>{{Cite web |url = http://politics.people.com.cn/n/2014/0819/c70731-25497657.html |title =新疆兵团359旅的继承者们：延续父辈爱国奉献精神  |author =  |date =  2014-08-19|publisher =  人民网|language =  |accessdate =  |quote =  }}</ref><ref name=yjbl/>。

1960年代初，[[中华人民共和国农垦部|中华人民共和国农垦部]]部长[[王震|王震]]认为新疆生产建设兵团的屯垦事业，特别是塔里木垦区的粮、棉、蚕桑基地建设，需要大量[[上山下乡运动|知识青年]]。根据[[周恩来|周恩来]]总理指示，1962年11月13日，农垦部党组呈报国务院，准备在[[第三个五年计划_(中国)|第三个五年计划]]期间动员8.5万名上海支边知识青年去农一师、农二师参加以蚕桑为主的生产建设。截至1966年底，农一师共接收安置上海市知识青年45402人<ref name=shanghai>{{Cite web|url =http://www.bingtuan.gov.cn/bt/tkfb/zbqn/hgyls/530327.shtml|title =上海支边知青在农一师的情况简介|author =|date =2015-01-13|publisher =新疆生产建设兵团|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170401061102/http://www.bingtuan.gov.cn/bt/tkfb/zbqn/hgyls/530327.shtml|archivedate =2017年4月1日|df =}}</ref>。[[文革|文革]]结束后， 1979年3月3日，农一师“新疆阿克苏垦区十四团上海知识青年赴京[[上访|上访]]团”出发赴北京请愿，要求回城。3月8日，他们与十二团、十三团会合组成了“阿克苏垦区（塔南）上海知识青年赴京上访团”。3月18日，上访团分为两路，一路继续北上赴北京，一路回上海。5月12日，上访团离开北京回新疆。1979年3月22日至5月12日，新疆生产建设兵团40多位上海知青组成了赴京请愿团，盼有关领导解决知青返城问题。此后，上海在新疆的知青先后3次上访，分别为7月19日到8月24日，9月1日到10日，12月12日到26日。1979年7月22日晚，国务院向新疆维吾尔自治区、上海市等十个省、自治区、直辖市电话通知：对已离开新疆的上访人员沿途劝阻、收容、遣散回新疆<ref name=jinlin/>。1980年1月初，阿克苏垦区团场部分上海支边知青约3700多人聚集到阿克苏上访，要求上级给予返城政策回上海。国务院派出国家劳动总局、农垦部、[[国务院知青办|国务院知青办]]组成以农垦部副部长[[张修竹|张修竹]]为组长的工作组赴新疆，会同上海市劳动局副局长[[汪力田|汪力田]]、[[新疆维吾尔自治区人民政府|新疆维吾尔自治区人民政府]]副主席、新疆维吾尔自治区农垦总局副局长[[谢高忠|谢高忠]]，举行三方会谈，形成了《关于做好稳定新疆垦区农场上海支边知识青年工作的座谈纪要》（简称“一月会谈纪要”），做出了《关于解决新疆垦区农场上海支边知识青年问题的具体规定》，随后又赴阿克苏举行上海青年代表座谈会<ref name=shanghai/>。经国务院批准，1980年1月21日，新疆维吾尔自治区人民政府颁发《通知》，要求“上海青年联络总部”、“上海青年联合委员会”等组织立即解散并停止一切活动，要求聚集在阿克苏城区的上访人员立即返回原工作岗位，农场对返回人员要欢迎，不得歧视。上访人员随后返回原单位<ref name=jinlin/><ref name=shanghai/>。1980年2月，上海市、新疆维吾尔自治区、新疆生产建设兵团等领导在阿克苏召集知青座谈会<ref name=jinlin/>。

1980年11月8日至12月20日，[[喀什地区|喀什地区]]、[[巴音郭楞蒙古自治州|巴音郭楞蒙古自治州]]、[[阿克苏地区|阿克苏地区]]等地1.5万名上海知青再度聚集到阿克苏，要求办理回上海的手续<ref name=shanghai/>。1980年11月中旬，在阿克苏的部分上海知青占据了中共阿克苏地委办公大楼<ref name=jinlin/>。12月13日，国务院再度派出工作组接见上海青年代表，宣传中央工作会议精神（主要是将大多数上海支边知识青年稳定在新疆，以利于上海和新疆两地的经济和政治秩序）。在各种措施下，垦区社会秩序恢复正常。1981年1月14日到16日，王震在新疆[[乌鲁木齐|乌鲁木齐]]两次接见阿克苏、巴音郭楞蒙古自治州、石河子垦区上海青年代表，勉励其发扬艰苦奋斗的传统，加强纪律性，为“[[四化|四化]]”建设创造安定团结局面<ref name=shanghai/>。1981年3月12日至30日，在国务院主持之下，国家劳动总局、农垦部会同新疆维吾尔自治区和上海市的党政领导在北京开会，商讨解决在新疆的上海知青问题<ref name=shanghai/><ref name=jinlin/>。1981年5月2日，中共上海市[[南市区_(上海市)|南市区]]委召开扩大会议，传达国务院《关于解决新疆垦区农场上海支边知识青年问题的具体规定》和[[中共上海市委|中共上海市委]]的工作部署<ref name=jinlin/>。1981年5月28日，国务院下发《关于解决新疆垦区农场上海支边知识青年问题的具体规定》，主要内容是本着有利开发边疆、建设边疆、巩固国防的原则，将上海知青基本稳定在新疆，同时为照顾上海支边知青本人或家庭困难，在政策范围内（例如“顶替”、“双知青”等）分期分批将符合条件的上海支边知青调回上海<ref name=shanghai/>。1981年8月31日，中共上海市[[徐汇区|徐汇区]]委、徐汇区人民政府组织280名干部到各街道里委宣传国务院《关于解决新疆垦区农场上海支边知识青年问题的具体规定》，动员徐汇区在沪支边青年（618户、1580人），除符合文件规定可以留在上海者，全部回新疆垦区农场工作<ref name=jinlin> {{Cite book | author = 金大陆、林升宝  | title =  上海知识青年上山下乡运动纪事录（1968-1981）| location =  | publisher = 上海书店出版社 | date =2014年  | pages =  | ISBN =  9787545809466| accessdate =  | url =  | language =  | quote =  }} </ref>。1985年7月24日，[[中共中央总书记|中共中央总书记]][[胡耀邦|胡耀邦]]视察阿克苏时，在农一师大光毛纺厂特地接见38名上海支边知识青年代表，并为上海支边知识青年题词：“历史贡献与[[托木尔峰|托木尔峰]]共存，新的业绩同[[塔里木河|塔里木河]]长流。”<ref name=shanghai/>

1986年12月，农一师党委派出政治部主任倪豪梅等人到上海，会同新疆生产建设兵团驻沪办事处组成工作组，与上海方面一起做好将上海支边知识青年稳定在新疆生产建设兵团的工作。1987年，[[上海市人民政府|上海市人民政府]]制定政策，允许照顾一名上海老支边知青子女落户上海。1987年3月9日，上海市人民政府与新疆生产建设兵团签订《关于坚决贯彻党中央指示，把原上海支青稳定在新疆的商谈纪要》。据此，自同年起，上海市中专、技校、职业高中在新疆生产建设兵团招收符合回沪条件的上海支边知青子女。1988年和1989年，上海市高校、普通高中先后在新疆生产建设兵团面向上海支边知识青年子女招生<ref name=shanghai/>。

1995年4月，经新疆生产建设兵团批准，农一师在阿拉尔成立阿拉尔管理局，统一管理镇区规划、土地、社会治安等项事务。1998年4月，根据[[中共中央|中共中央]]、[[中华人民共和国国务院|国务院]]加强新疆生产建设兵团工作的有关文件精神以及1997年底新疆生产建设兵团建市工作会议精神，农一师成立了阿拉尔建市筹建领导机构和工作机构，正式启动阿拉尔建市工作。经与[[新疆维吾尔自治区民政厅|新疆维吾尔自治区民政厅]]、[[新疆生产建设兵团建设局|新疆生产建设兵团建设局]]、[[中共阿克苏地委|中共阿克苏地委]]、[[阿克苏地区|阿克苏地区]]行政公署等协商，形成了建市初步方案。2001年1月4日，[[新疆维吾尔自治区人民政府|新疆维吾尔自治区人民政府]]第64次主席办公会议决定，阿拉尔市市辖区域范围为七团至十六团、塔里木灌区水利管理处、水利水电工程处、塔里木农垦大学（今[[塔里木大学|塔里木大学]]）、[[托喀依乡|托喀依乡]]（时属[[阿克苏市|阿克苏市]]）。会议还决定，阿拉尔设市有关问题由新疆维吾尔自治区人民政府组织民政、财政及新疆生产建设兵团有关部门与国务院有关部门接洽<ref name=tahe>{{Cite web|url =http://www.bingtuan.gov.cn/bt/jrbt/tcsb/csjs/531502.shtml|title =塔河明珠——阿拉尔市|author =|date =2015-01-16|publisher =新疆生产建设兵团|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20170401055946/http://www.bingtuan.gov.cn/bt/jrbt/tcsb/csjs/531502.shtml|archivedate =2017年4月1日|df =}}</ref>。

2002年7月10日，新疆维吾尔自治区人民政府向国务院呈送《关于设立阿拉尔市的请示》。2002年9月17日，国务院下发《关于同意新疆维吾尔自治区设立县级阿拉尔市的批复》（国函〔2002〕81号），将阿拉尔市明确为新疆维吾尔自治区直辖市，并且规定市域四至界线，行政区划面积4196平方公里，市辖区域为10个团场、2个处、1 个大学、1个乡，及国家、自治区、兵团驻市单位52个。2002年12月5日至7日，阿拉尔市第一届[[人民代表大会|人民代表大会]]第一次会议和[[中国人民政治协商会议|中国人民政治协商会议]]阿拉尔市第一届委员会第一次会议召开，选举产生阿拉尔市第一届人民代表大会、阿拉尔市人民政府、中国人民政治协商会议阿拉尔市第一届委员会领导机构。2003年12月30日，新疆维吾尔自治区人民政府下发《关于设立阿拉尔市（县级）有关问题的通知》（新政发〔2003〕85号），明确阿拉尔市由新疆维吾尔自治区直辖，阿拉尔市人民政府驻阿拉尔市，管理方式按照[[石河子市|石河子市]]模式执行，并且规定了阿拉尔市行政区域界线。2004年1月19日，中国共产党阿拉尔市委员会、阿拉尔市人民代表大会常务委员会、阿拉尔市人民政府、中国人民政治协商会议阿拉尔市委员会、中国共产党阿拉尔市纪律检查委员会正式挂牌成立<ref>崔强，一师阿拉尔市房地产现状及对策建议，经营管理者2014年第28期</ref><ref name=tahe/>。农一师与阿拉尔市实行“师市合一”的管理体制，由[[新疆维吾尔自治区|新疆维吾尔自治区]]、新疆生产建设兵团双重直辖<ref name=tahe/>。2012年，农一师机关及师级所有行政事业单位全部由阿克苏市搬迁至阿拉尔市<ref>{{Cite web |url = http://www.huyangnet.cn/news/2016-01/09/content_201894.htm |title =一师阿拉尔市档案局（馆）获誉“全国档案系统先进集体”  |author =  |date = 2016-01-09 |publisher = 胡杨网 |language =  |accessdate =  |quote =  }}</ref>。

2013年1月16日，农一师阿拉尔市举行新疆生产建设兵团第一师更名揭牌仪式，师市党委书记、政委邹跃斌，师市党委副书记、师长、阿拉尔市市长王新民为“中国共产党新疆生产建设兵团第一师阿拉尔市委员会”、“新疆生产建设兵团第一师”、“中国共产党新疆生产建设兵团第一师阿拉尔市纪律检查委员会”更名揭牌。自此，农一师正式更名为'''新疆生产建设兵团第一师'''<ref>{{Cite web |url =http://www.ale.gov.cn/structure/yw/jrywzw_87522_1.htm  |title = 师市由兵团农业建设第一师更名为兵团第一师 |author =  |date =  2013-01-24|publisher = 第一师阿拉尔市 |language =  |accessdate =  |quote =  }}</ref>。
