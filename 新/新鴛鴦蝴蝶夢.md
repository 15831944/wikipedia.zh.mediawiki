《'''新鴛鴦蝴蝶夢'''》是藝人[[黃安_(藝人)|黃安]]的個人[[音樂專輯|音樂專輯]]<ref>{{cite web|title=新鸳鸯蝴蝶梦 - 黄安|url=http://www.kuwo.cn/album/69419?catalog=yueku2016|website=酷我音乐|language=zh-cn}}</ref>，1993年2月1日由[[上華唱片|華星唱片]]發行。於臺灣的銷售量高達94萬張，全球銷售量超過600萬張<ref>{{cite news |language = 中文 |author = |coauthors = |url = http://www.nownews.com/n/2012/04/19/37252/2 |title = 90年代唱片榮景　百萬銷量專輯回顧 |publisher = NOWmews |pages =2 |date =2012-04-19 |accessdate = 2017-01-08}}</ref>。

== 同名單曲 ==
該專輯的同名單曲〈新鴛鴦蝴蝶夢〉被選作1993年華視連續劇《[[包青天_(1993年電視劇)|包青天]]》的[[片尾曲|片尾曲]]。該劇以中國宋朝名臣[[包拯|包拯]]為主角，透過對其名下各種奇特[[刑案|刑案]]審理過程的細膩描繪，獲得廣泛迴響。《包青天》的成功連帶提高了歌曲〈新鴛鴦蝴蝶夢〉及其演唱者黃安的知名度，也使〈新鴛鴦蝴蝶夢〉成為黃安最為人知的經典演唱作品之一，並一度被多位知名人士傳唱，包括時任中華民國法務部長的[[馬英九|馬英九]]<ref>[https://www.youtube.com/watch?v=dAStNGIwwnc 馬英九(法務部長時期) 演唱新鴛鴦蝴蝶夢]，YouTube影片</ref>，臺灣歌手[[費玉清|費玉清]]<ref>[https://www.youtube.com/watch?v=INz6QE8tAJQ 費玉清 - 新鴛鴦蝴蝶夢]，YouTube影片</ref>、[[蕭敬騰|蕭敬騰]]<ref>[https://www.youtube.com/watch?v=R99Ir2_XjBw 新鴛鴦蝴蝶夢新不了情\2015蕭敬騰Triple Jam上海場]，YouTube影片</ref>、[[卓依婷|卓依婷]]<ref>[https://www.youtube.com/watch?v=_dt2iN43n_A 卓依婷---新鸳鸯蝴蝶梦]，YouTube影片</ref>、[[法拉利姐|法拉利姐]]<ref>{{cite news|url= http://star.ettoday.net/news/539416 |title=法拉利姐清唱《新鴛鴦蝴蝶夢》　高EQ感謝黃安鼓勵|author=許瑞麟|language=zh-hant|publisher=[[ETtoday新聞雲|ETtoday新聞雲]]|date=2015年7月23日|accessdate=2016年1月19日}}</ref>，以及香港藝人[[梅艷芳|梅艷芳]]<ref>[https://www.youtube.com/watch?v=-3MuCF2rXmY 梅艷芳 Anita Mui 新鴛鴦蝴蝶夢]，YouTube影片</ref>等等。此外，香港歌手[[何家勁|何家勁]]也曾以[[粵語|粵語]]翻唱該曲<ref>[https://www.youtube.com/watch?v=_rvWSDJz2n0 何家勁- 新鴛鴦蝴蝶夢 (歌詞MV) (HD)]，YouTube影片</ref>。根據「網路溫度計」在2016年所做的一項調查顯示，這首歌曲在「懷舊八點檔」主題曲排行榜中名列第五<ref>{{cite news|url= http://dailyview.tw/Daily/2016/01/16 |title=琅琅上口聽不膩！經典懷舊八點檔主題曲|language=zh-hant|publisher=網路溫度計|date=2016年1月16日|accessdate=2016年12月19日}}</ref>。

在2008年時，音樂人[[黃國倫|黃國倫]]指出〈新鴛鴦蝴蝶夢〉的旋律抄襲自日本音樂團體[[神思者|神思者]]的作品《阿拉伯樂土》（Happy Arabia）<ref>[https://www.youtube.com/watch?v=_JHdjYJ0mV0 神思者 S.E.N.S. / 阿拉伯樂土 Happy Arabia]，YouTube影片</ref>，該曲收錄於1988年推出的專輯《MASALA TEA WALTZ》中。黃國倫表示「兩者簡直一模一樣」<ref>{{cite news |language = 中文 |author = NOW NEWS |coauthors = |url = http://www.nownews.com/n/2008/12/31/965369 |title = 黃國倫：《新鴛鴦蝴蝶夢》的曲涉嫌抄襲！ |publisher = NOW NEWS |pages =  |date = 2008-12-31 |accessdate = 2016-12-19}}</ref><ref>{{cite news |language = 中文 |author = 自由時報 |coauthors = |url = http://ent.ltn.com.tw/news/breakingnews/165923 |title = 黃國倫：《新鴛鴦蝴蝶夢》是抄襲日本歌曲 |publisher =自由時報 |pages =  |date = 2008-12-31 |accessdate = 2016-12-19}}</ref>。

在2015年底，由於黃安引起[[周子瑜國旗事件|周子瑜國旗事件]]，使得黃安在台灣受到許多對中國大陸反感的人士批評，這也連帶影響到歌曲〈新鴛鴦蝴蝶夢〉。例如《自由時報》以〈新鴛鴦蝴蝶夢〉為題的社論<ref>{{cite news |language = 中文 |author =  |coauthors =  |url = http://talk.ltn.com.tw/article/paper/949783 |title = 新鴛鴦蝴蝶夢 |publisher = 自由時報 |pages =  |date = 2016-01-16 |accessdate = 2016-01-19}}</ref>，以及各種對歌曲的抵制：
* 2016年1月12日，臺北一家咖啡店在外帶紙杯上貼上寫著「最恐怖的夢不是惡夢，是新鴛鴦蝴蝶夢」的紙條以諷刺黃安<ref>{{cite news |language = 中文 |author =  |coauthors =  |url = http://www.ettoday.net/news/20160113/629951.htm |title = 比惡夢還恐怖的夢？　咖啡店KUSO酸黃安：新鴛鴦蝴蝶夢 |publisher = ETtoday東森新聞雲 |pages =  |date = 2016-01-13 |accessdate = 2016-01-19}}</ref>。
* 2016年1月13日，[[寶島聯播網|寶島聯播網]]發出公告，自該日起拒絕播出黃安演唱的〈新鴛鴦蝴蝶夢〉以及黃安演唱的其它歌曲<ref>{{cite news |language = 中文 |author = 曾德蓉 |coauthors =  |url = http://ent.ltn.com.tw/news/breakingnews/1571252 |title = 黃安超狂激怒台灣人 鄭弘儀電台開第一槍封殺 |publisher = 自由時報 |pages =  |date = 2016-01-13 |accessdate = 2016-01-19}}</ref>。
* 2016年1月16日，藝人[[蔡昌憲|蔡昌憲]]折斷《新鴛鴦蝴蝶夢》專輯光碟，並將印有歌詞的內頁撕得粉碎以表示對周子瑜的支持，將損毀的專輯照片放上網路，獲得網友一片好評<ref>{{cite news |language = 中文 |author = 華少甫 |coauthors =  |url = http://star.ettoday.net/news/631856 |title = 蔡昌憲折斷黃安CD、歌詞變碎紙　「周子瑜我們都挺妳」 |publisher = ETtoday新聞雲 |pages =  |date = 2016-01-16 |accessdate = 2016-12-19}}</ref>。
* 2016年1月19日，臺中一家[[KTV|KTV]]發出公告，因「無人點播」、「危害系統穩定性」，將黃安的〈新鴛鴦蝴蝶夢〉及其他歌曲從歌庫永久刪除<ref>{{cite news |language = 中文 |author = 陳建志 |coauthors =  |url = http://news.ltn.com.tw/news/local/paper/950442 |title = KTV刪黃安7歌 網友肯定按讚 |publisher = 自由時報 |pages =  |date = 2016-01-13 |accessdate = 2016-01-19}}</ref>。
* 2016年3月10日，長年旅居中國大陸的黃安返台接受手術，此一行為引發網友改編〈新鴛鴦蝴蝶夢〉歌詞並翻唱來表達對黃安的不滿<ref>{{cite news |language = 中文 |author =  |coauthors =  |url = http://news.ltn.com.tw/news/life/breakingnews/1629808 |title = 〈新鴛鴦蝴蝶夢〉惡搞MV 鄉民驚呼：太神啦 |publisher = 自由時報 |pages =  |date = 2016-03-12 |accessdate = 2016-01-19}}</ref>。
