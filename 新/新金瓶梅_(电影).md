== 花絮 ==
该片由《[[赤壁铜雀台|赤壁铜雀台]]》的原班人马打造。<ref>{{cite web |url= http://yue.ifeng.com/neidiliuxing/detail_2012_12/14/20183353_0.shtml|title= 龚玥菲《新金瓶梅》静态电影解密 颠覆潘金莲经典形象|accessdate=2012年12月14日  |publisher= 凤凰网|work= |language =zh-cn}}</ref>

为了给该片造势，龚玥菲身着[[汉服|汉服]]在雪地行走，大秀事业线吸引关注。<ref>{{cite web |url=http://cq.qq.com/a/20121230/000023.htm |title=《新金瓶梅》女主角雪地秀事业线 姐你不冷吗？ |accessdate=2012年12月30日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

该片概念版海报在2012年12月31日发布。<ref>{{cite web|url=http://www.yangtse.com/system/2012/12/31/015768041.shtml|title=3D《新金瓶梅》概念海报曝光 龚玥菲薄纱蔽体香艳四射|accessdate=2012年12月31日|publisher=扬子晚报|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20140110114318/http://www.yangtse.com/system/2012/12/31/015768041.shtml|archivedate=2014年1月10日}}</ref>

对于该片的尺度问题，龚玥菲对记者说：“我们本着尺度上还是尊重原著的前提，尺度上会开一些，但是还是按照内地现有相关规定下拍摄，不会超越底线的。”<ref>{{cite web |url=http://fj.qq.com/a/20121214/000260.htm |title= 《新金瓶梅》尺度大开颠覆传统潘金莲形象|accessdate=2012年12月14日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>
