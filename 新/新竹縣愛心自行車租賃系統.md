'''新竹縣愛心自行車租賃系統'''是[[新竹縣|新竹縣]]的[[公共自行車|公共自行車]]租賃系統，由[[新竹縣政府|新竹縣政府]]規劃，於2012年4月2日啟用，截至2016年9月4日共設有6個站點。

==歷史==
2012年4月2日由新竹縣政府規劃的愛心自行車開始提供借車服務，共有4個站點、110輛自行車（由善心人士捐助）。<ref>{{cite news |author =  |title = 竹縣HTS快捷公車與愛心自行車　下週一上路 |url = http://www.nownews.com/n/2012/03/30/3505/2 |work = NOWnews |date = 2012-03-30 |accessdate = 2016-09-06 |language = zh-tw}}</ref><ref>{{cite news |author =  |title = 油價漲翻天　新竹縣｢免費快捷」來回車資省41元 |url = http://www.ettoday.net/news/20120403/36329.htm |work = ETtoday東森新聞雲 |date = 2012-04-03 |accessdate = 2016-09-06 |language = zh-tw}}</ref>2015年9月自行車數量由原本的110輛減少到70多輛<ref>{{cite news |author = 廖雪茹 |title = 竹縣愛心自行車 3分之1「失蹤」 |url = http://news.ltn.com.tw/news/life/breakingnews/1461518 |work = 自由時報 |date = 2015-09-30 |accessdate = 2016-09-06 |language = zh-tw}}</ref>，之後一度只剩50多輛，2016年7月新竹縣政府獲捐贈100輛自行車，共計有150多輛。<ref name="station">{{cite news |author = 廖雪茹 |title = 竹市有YouBike 竹縣推愛心鐵馬與觀光路線 |url = http://news.ltn.com.tw/news/life/breakingnews/1815806 |work = 自由時報 |date = 2016-09-04 |accessdate = 2016-09-06 |language = zh-tw}}</ref>

==站點==
截至2016年9月4日，共設有6個站點。<ref name="station" /><ref name="epaper">{{cite web |title = 愛心卡打車免錢乎你騎 |url = http://w3.hsinchu.gov.tw/epaper/default.asp?sdDate=2015/5/8&status=1 |work = 亮麗竹縣周報 |publisher = 新竹縣政府 |date = 2015-05-08 |accessdate = 2016-09-06 |language = zh-tw}}</ref>
*新竹縣政府
*文化中心
*新瓦屋
*高鐵新竹站區
*竹東之星（竹東動漫園區）
*富林加油站

==借用方式==
新竹縣愛心自行車借用時間為每日上午8點至下午5點，可甲站借乙站還，借用時須至各站的服務台登記姓名及手機號碼，認可後即能領取鑰匙，歸還時須將鑰匙投入站旁的鑰匙箱。<ref name="epaper" />
