'''新疆维吾尔自治区龟兹研究院'''，位于[[新疆维吾尔自治区|新疆维吾尔自治区]][[阿克苏地区|阿克苏地区]][[拜城县|拜城县]][[克孜尔乡|克孜尔乡]]偏远山区的[[克孜尔千佛洞|克孜尔千佛洞]]，是[[新疆维吾尔自治区文化厅|新疆维吾尔自治区文化厅]]唯一驻[[南疆|南疆]]的直属文博[[事业单位|事业单位]]<ref name=shydw>{{Cite web |url = http://www.gjsy.gov.cn/cxzl/ |title = 新疆维吾尔自治区龟兹研究院 |author =  |date =  |publisher =事业单位在线  |language =  |accessdate = 2018-08-20 |quote =  }}</ref><ref name=xjwhw>{{Cite web |url =http://www.xjwh.gov.cn/index.php?m=content&c=index&a=show&catid=779&id=21833  |title = 2017年自治区文化厅部门预算和“三公”经费预算公开 |author =  |date =2017-02-06  |publisher = 新疆文化网 |language =  |accessdate =  |quote =  }}</ref>。

==简介==
1985年7月，'''新疆维吾尔自治区龟兹石窟研究所'''成立。2009年4月25日，该研究所更名为'''新疆维吾尔自治区龟兹研究院'''<ref>{{Cite web |url =http://news.163.com/18/0205/19/D9TH7BOS000187VE.html  |title = 千人今天送别考古界泰斗宿白，敦煌与龟兹人眼中的先生 |author =  |date =2018-02-05  |publisher =  网易|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://ex.cssn.cn/djch/djch_djchhg/tanmixiaoshiguguozhiyi/201312/t20131217_910463.shtml  |title =  龟兹研究院 打造龟兹研究的开放平台|author = 张春海 吴文康 |date =  |publisher =  中国社会科学报2013年6月21日第465期|language =  |accessdate =  |quote =  }}</ref>。2010年，正式更名挂牌<ref name=zhgl/>。该研究院是新疆维吾尔自治区文化厅、[[新疆维吾尔自治区文物局|新疆维吾尔自治区文物局]]唯一驻南疆的直属文博事业单位，负责对古[[龟兹|龟兹]]地区9处[[全国重点文物保护单位|全国重点文物保护单位]]，共600余个佛教石窟近2万平米壁画进行科学保护、管理和研究<ref>{{Cite web |url =  https://i.gench.edu.cn/e4/61/c2936a58465/page.htm|title = 新疆龟兹研究院院长徐永明等一行来校参观访问 |author =  |date = 2017-11-03 |publisher =  上海建桥学院|language =  |accessdate =  |quote =  }}</ref>。其管理的9处全国重点文物保护单位是：拜城县3处：[[克孜尔石窟|克孜尔石窟]]、[[台台尔石窟|台台尔石窟]]、[[温巴什石窟|温巴什石窟]]；库车县5处：[[库木吐喇石窟|库木吐喇石窟]]、[[克孜尔尕哈石窟|克孜尔尕哈石窟]]、[[森木塞姆石窟|森木塞姆石窟]]、[[玛扎伯哈石窟|玛扎伯哈石窟]]、[[阿艾石窟|阿艾石窟]]；新和县1处：[[托乎克拉艾肯石窟|托乎克拉艾肯石窟]]。

2011年8月8日，新疆龟兹石窟保护与研究国际学术研讨会在新疆龟兹研究院开幕，新疆维吾尔自治区文化厅党组书记[[韩子勇|韩子勇]]出席并讲话，开幕式上举行了[[中国文化遗产研究院|中国文化遗产研究院]]在新疆龟兹研究院设立的“古代壁画保护工程研究新疆工作站”签订合作协议仪式，新疆维吾尔自治区文化厅党组书记韩子勇与中国文化遗产研究院副院长[[柴晓明|柴晓明]]为中国文化遗产研究院古代壁画保护修复研究中心新疆工作站揭牌<ref>{{Cite web |url =http://politics.people.com.cn/h/2011/0809/c226651-2491112069.html  |title =  新疆龟兹石窟保护与研究国际学术研讨会开幕|author =  |date = 2011-08-09 |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。

2011年，上海印刷集团旗下商务数码图像技术有限公司与新疆维吾尔自治区龟兹研究院合作开展龟兹洞窟数字化与还原保护工程项目<ref>{{Cite web |url =  https://fo.ifeng.com/a/20151217/41525273_0.shtml|title =新疆龟兹石窟数字团队：历史传承不能在这里断掉  |author =  |date =2015-12-17  |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。2015年，新疆龟兹研究院在上海设工作站，揭牌仪式由上海印刷集团党委书记[[陆炎|陆炎]]主持，新疆维吾尔自治区文物局副局长[[申文涛|申文涛]]与[[上海市新闻出版局|上海市新闻出版局]]副局长[[蔡纪万|蔡纪万]]为工作站揭牌，同时新疆龟兹研究院院长徐永明同上海印刷集团总经理[[沈剑毅|沈剑毅]]在揭牌仪式上签订双方新的战略合作协议<ref>{{Cite web |url = http://m.haiwainet.cn/middle/457179/2015/1117/content_29364397_1.html |title = 新疆龟兹研究院在上海设立工作站 |author =  |date =2015-11-17  |publisher = 海外网 |language =  |accessdate =  |quote =  }}</ref>。
