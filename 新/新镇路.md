== 道路改建 ==
2012年12月18日，新镇路改建工程开工，作为[[闵行文化公园|闵行文化公园]]、七宝生态商务区配套项目。改建后的新镇路是闵行文化公园和七宝生态商务区的中心道路，连接漕宝路和吴中路。据市政规划总体要求，新镇路走向向东偏移200米，原有道路废弃，改为“4快2慢”车道<ref>{{cite web|title=“一园一区”中心道路——新镇路改建工程启动-上海政务|url=http://shzw.eastday.com/shzw/G/20121218/u1ai95629.html|website=shzw.eastday.com}}</ref>。改造道路长1033.55米，道路红线宽度35米<ref>{{cite web|author1=上海船舶运输科学研究所|title=新镇路(漕宝路-吴中路)道路改建及人行通道工程环境影响评价第二次公示|url=http://www.envir.gov.cn/info/2012/20124105255.htm|website=www.envir.gov.cn|accessdate=2018-04-04}}</ref>。

== 道路阻塞 ==
新镇路由于学生客流集中，又因与之相交的东西向道路不多，交通压力一向较大。<ref name=hpbg>{{cite web|author1=上海顺茂环境影响评价技术服务有限公司|title=闵行区七宝镇农南路(新镇路-七莘路)道路新建工程环境影响报告书|url=http://www.envir.gov.cn/docs/2014/20140417491.pdf|website=上海环境热线|accessdate=2018-04-01|date=2014-04-17|deadurl=no|archiveurl=https://web.archive.org/web/20180401161250/http://www.envir.gov.cn/docs/2014/20140417491.pdf|archivedate=2018-04-01}}</ref>

== 事故 ==
2009年6月2日下午，因避让一辆由农南路东向北右转的奥迪车（车牌号沪A031C8<ref name="hxpj">{{cite news|author1=顾文剑|author2=杨克元|title=[后续]致5人死亡 奥迪司机获刑4年半|url=http://news.sina.com.cn/o/2009-11-24/021516656107s.shtml|accessdate=2018-04-02|work=[[东方早报|东方早报]]|date=2009-11-24|deadurl=no|archiveurl=https://web.archive.org/web/20180402142008/http://news.sina.com.cn/o/2009-11-24/021516656107s.shtml|archivedate=2018-04-02}}</ref>），一辆车牌为沪EV6696的[[上海大众交通股份有限公司|大众]]出租车由南向北闯入双黄线另一侧，和由北向南过路车牌为沪B60827的[[上海公交173路|上海公交173路]]公交车迎面相撞。被撞的173路空调车车头右侧车窗玻璃粉碎性破裂；而大众出租车右前车身完全扭曲变形，打开的气囊和座位上血迹斑斑<ref>{{cite web|title=出租车与173路公交相撞 韩国女乘客2死2伤(图)_新浪上海_新浪网|url=http://sh.sina.com.cn/news/s/2009-06-03/0820112362.html|website=sh.sina.com.cn|publisher=[[东方早报|东方早报]]|accessdate=2018-04-01|date=2009-06-03|deadurl=no|archiveurl=https://web.archive.org/web/20090608171416/http://sh.sina.com.cn/news/s/2009-06-03/0820112362.html|archivedate=2009-06-08}}</ref>。农南路上一家店的店主高小飞与几名朋友找来撬棍，救出驾驶员，并随即拦下一辆出租车，将驾驶员送往[[上海市闵行区中心医院|上海市闵行区中心医院]]。出租车上2名韩国籍女乘客当场死亡，司机在送医院后抢救无效，也于当天死亡；事故出租车上另外两名重伤韩国籍女乘客在送往闵行区中心医院后，也因伤势过重，于6月3日抢救无效死亡。事发后，[[上海市人民政府|上海市人民政府]]有关领导立即赶往事故现场指挥协调并安抚家属；[[韩国|韩国]]驻沪总领馆也得到通报，并派员处理有关事宜。经确认，死亡的韩国籍乘客均为中年女性，系韩籍在沪从业人员家属。事发后，奥迪车驾驶员张峡（一名张夏）立刻遭到[[刑事拘留|刑事拘留]]<ref>{{cite web|title=出租车公交车相撞5人丧生 奥迪司机被刑拘(视频)_新浪上海_新浪网|url=http://sh.sina.com.cn/news/s/2009-06-04/0745112410.html?from=wap|website=新浪网上海频道|publisher=[[东方早报|东方早报]]|accessdate=2018-04-01|date=2009-06-04|deadurl=no|archiveurl=https://web.archive.org/web/20180401161250/http://sh.sina.com.cn/news/s/2009-06-04/0745112410.html?from=wap|archivedate=2018-04-01}}</ref>。

据闵行区交警部门认定，涉事三辆机动车均涉嫌超速驾驶：张峡驾驶车右转弯时，最低行驶速度约33.5 km/h，超出限速3.5 km/h，妨碍被放行车辆通行。潘某驾驶公交车事发前车速约为71~74 km/h，超出限速41~44 km/h。出租车驾驶员吴某事发前车速约为59 km/h，其右方道路来车实施右转弯的情况下，从限速60公里/小时的路段，通过路口进入前方限速40公里/小时的路段过程中，未及时减速，因而三方的行为共同造成这起道路交通事故。经闵行区检察院审查交通事故责任认定报告：张峡在遇红灯右转弯时，妨碍被放行的车辆通行，而且超速行驶，过错的严重程度较大；公交车司机潘某和出租车司机吴某虽也有过错行为，但严重程度较小。故依据相关规定，检察院确定张峡承担事故主要责任，潘某和吴某承担事故次要责任，4名女乘客不承担事故责任。据此，闵行区人民检察院将奥迪车驾驶员张峡批准[[逮捕|逮捕]]<ref>{{cite web|author1=李淑平|author2=宋国伟|author3=陆勤俭|title=妨碍车辆直行 奥迪车司机负主责|url=http://news.163.com/09/0701/14/5D52SR9I000120GR.html|website=网易新闻|publisher=[[东方早报|东方早报]]|accessdate=2018-04-01|date=2009-07-01|deadurl=no|archiveurl=https://web.archive.org/web/20180401164115/http://news.163.com/09/0701/14/5D52SR9I000120GR.html|archivedate=2018-04-01}}</ref>。

在庭审中，张峡辩称，认定其负事故的主要责任与事实不符。案发后其始终未离开现场，并在民警到达后主动表明自己为肇事者；而其辩护人为其做出无罪辩护，认为其它相关方也是交通肇事者。2009年11月24日，案件做出判决。闵行区人民法院宣判时认为，张峡应负主责，但其无前科、有自首情节，故判[[有期徒刑|有期徒刑]]四年半，驳回无罪辩护。<ref name="hxpj" />
