'''新光人壽保險摩天大樓'''，簡稱'''新光摩天大樓'''，是位於[[中華民國|中華民國]][[台北市|台北市]][[中正區_(臺北市)|中正區]]的[[摩天大樓|摩天大樓]]，坐落於[[忠孝西路|忠孝西路]]與[[館前路|館前路]]交叉口，與[[泰迪熊百貨|五鐵秋葉原]]比鄰，隔著忠孝西路對面是[[台北車站|台北車站]]。新光人壽保險摩天大樓為台北市西區市中心的地標建築之一，也是其業主[[新光集團|新光集團]]旗下數家企業的總部所在地。

== 概要 ==
新光摩天大樓完成於1993年12月21日，由旅[[日本|日]][[建築師|建築師]][[郭茂林|郭茂林]]設計，由日本[[熊谷組|熊谷組]]在台分公司[[華熊營造|華熊營造]]負責建造，地上51層、地下7層、總高度245公尺。<ref name="閱">{{cite book|url=http://www.gov.taipei/ct.asp?xItem=48769&CtNode=5270&mp=100021|author=簡愛立撰, 王能佑攝|title=《閱讀台北·與天爭高，心意最重要 新光摩天大樓》|publisher=臺北市政府觀光傳播局|date=2009-04, 486期}}</ref>曾為[[台灣|台灣]]最高的[[摩天大樓|摩天大樓]]，但在[[高雄85大樓|高雄85大樓]]（1997年）、[[台北101|台北101]]（2004年）和[[臺北南山廣場|臺北南山廣場]]（2017年）相繼落成後，目前為台灣第四高樓。大樓內地下2至地上13層為[[新光三越百貨|新光三越百貨]]台北站前店。

新光摩天大樓的原址是[[台灣日治時期|日治時代]]的[[臺灣鐵道飯店|臺灣鐵道飯店]]，該旅館採歐式建築，曾為台灣最高級的旅館之一，建築物部分毀於[[二戰|二戰]]末期[[美軍|美軍]]的[[台北大空襲|台北大轟炸]]。戰後曾小規模復業，改名為「臺灣鐵路飯店」。[[新光人壽|新光人壽]]1963年在[[館前路|館前路]]創立，故挑選此塊鄰近館前路的基地興建摩天大樓。[[新光金融控股公司|新光金融控股公司]]於2002年創立後，也將總部設置於此。

== 頂樓展望台 ==
新光摩天大樓的46樓展望台曾為台北市最知名的旅遊景點之一，只需要購票便可搭[[電梯|電梯]]直上46樓一覽[[台北盆地|台北盆地]]的景色。

新光展望台全盛時期曾創下一天6,000多名參觀者的紀錄，前[[蘇聯總統|蘇聯總統]][[戈巴契夫|戈巴契夫]]也曾前來參觀，但近年來平均遊客僅剩下100多名，{{r|閱}} 並且多為日本觀光客。由於[[台北101|台北101]]觀景台的開放與虧損連連的緣故，新光展望台原本計畫僅開放到2005年年底，但後來在民眾意見的影響下，延長營運時間到2006年2月28日正式結束。

目前展望台的空間已出租給[[綠加利|綠加利]]公司（一家來自日本的直銷公司）使用，同時也是該大樓登高比賽的終點。

== 登高比賽 ==
{{main|新光摩天大樓登高大賽}}
1994年10月16日，新光人壽首度開辦[[大樓登高比賽|大樓登高比賽]]，並開啟台灣[[登高賽|登高賽]]史，比賽使用了46層樓1,006階，[[台北市立體育學院|台北體院]]的邱次郎以及[[台北市|台北市]][[青年公園_(台北市)|青年公園]]慢跑俱樂部的張美蓮分別以5分45秒及8分16秒拿下首屆男子、女子組的冠軍。現今的最佳成績，則是男子選手陳仲仁在第18屆（2004年7月）時的5分32秒與李筱瑜在第19屆（2004年12月）時創下的6分52秒。

而此項比賽雖然並非台灣最高樓的[[大樓登高比賽|大樓登高比賽]]，卻是台灣登高賽史上最負盛名、也是歷史最悠久的登高賽。
