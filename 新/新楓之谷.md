==遊戲問題及爭議==
; 電磁紀錄
: 遊戲橘子保留電磁紀錄一百天，期限已過則不得追查，每次追查將花費新台幣五百元整。

; 伺服器回溯問題
: 臺灣新楓之谷伺服器，時常在玩家上線人數較多時發生延遲(Lag)情形，嚴重時會發生伺服器大斷線甚至[[系統回溯|回溯]]情形，導致玩家經驗值、金幣、道具被回復到先前某時間點的狀態，又因為遊戲橘子宣稱無遊戲歷程相關電磁紀錄，因此玩家在伺服器發生回溯情形時的非現金道具損失皆無法補償，最早2007年的無預警大回朔，對玩家投下了史無前例的震撼彈，從此開始對遊戲失去信賴。

; 伺服器維護時間
: 臺灣新楓之谷維護時間為每週三8:00-14:00，改版維修則通常為0:00-10:00（官方會公告）。目前最長為2013年5月8日的「超能再起」改版，由5月8日0:00維護到5月9日0:00，歷時24小時維修時間，伺服器短暫開放後，又於5月9號上午11時關閉進行超技能改版臨時維修。

; 外掛問題
: 在新楓之谷中，有些玩家會利用修改記憶體，來改變新楓之谷主程式的運行，藉此取得大量遊戲虛擬幣。但由於這類玩家所創的角色數量龐大，使官方難以查緝，而使許多玩家因而受到影響。
: 例如有名的惡劣外掛超越者玩家，館長阿帝冥，長期霸佔新楓之谷武陵道場第一排名、The Seed綠野仙蹤海底之塔第一排名，世人稱其名為『館長』導致新楓之谷不得不加強玩家遊玩排名外掛偵測，詳情見官方遊戲公告。

===其他BUG===
自遊戲正式營運至今，新楓之谷出現過無數[[BUG|BUG]]，每個月不斷產生各種問題，造成玩家困擾，而部分問題尚未解決而持續至今。<ref>{{cite web|url=https://forum.gamer.com.tw/C.php?bsn=07650&snA=959674&tnum=1/|title=【BUG集中】尚未維修的BUG整理集中串}}</ref>

==外掛防護==
新楓之谷之外掛程式撰寫容易，使用[[Visual_Basic|Visual Basic]]或更高階程式語言便能完成。其一原因在於遊戲一開始的對話方塊（防外掛系統未載入前的視窗畫面），使程序容易被鎖定或注入DLL檔（V046到V054版本時楓之谷都沒有對話方塊，於V054更新後才出現），而不會遭防外掛系統轉向。( 自從台版V184改成使用 [[XIGNCODE|XIGNCODE]] 防掛軟件後 ，使用[[Visual_Basic|Visual Basic]]所寫的外掛皆會遭 [[XIGNCODE|XIGNCODE]] 偵測，因此當前楓之谷外掛皆使用[[C++|C++]]來編寫。

===外掛防護系統===
* nProtect GameGuard
:初期各版本楓之谷使用'''「[[nProtect_GameGuard|nProtect GameGuard]]」'''進行[[遊戲外掛|外掛]]防護措施
:在開始遊戲及遊戲進行時自動偵測玩家行為以及確認[[電腦病毒|電腦病毒]]。

* Ahnlab HackShield
:中期各版本楓之谷使用'''「[[Ahnlab_HackShield|Ahnlab HackShield]]」'''進行[[遊戲外掛|外掛]]防護措施。
:由於NEXON將其旗下多款遊戲防外掛軟體由「nProtect GameGuard」改成「Ahnlab HackShield」，臺灣楓之谷也於2009年4月28日的V092版更新後使用了「Ahnlab HackShield」防外掛軟體。

* XIGNCODE
:現在各版本楓之谷陸續使用'''「[[XIGNCODE3|XIGNCODE3]]」'''進行[[遊戲外掛|外掛]]防護措施。
:韓國楓之谷防外掛軟體從Ahnlab HackShield更改為XIGNCODE3，臺灣楓之谷也於2015年10月21日的V184版更新後使用了「XIGNCODE」防外掛軟體。但現今韓國；臺灣楓之谷已不再使用「XIGNCODE3」防護系統。 

* BlackCipher (Nexon Game Security)
:NEXON旗下遊戲的防外掛軟體，臺灣楓之谷於2013年5月22日的V157.2版更新後新增「BlackCipher」防護系統。

* Game Protect Kit
:中国大陆的冒险岛于2017年1月18日起的V139版更新后使用Game Protect Kit配合BlackCipher反外挂。

2006年11月9日，[[NEXON|NEXON Japan]]發表了與[[MADHOUSE|MADHOUSE]]的合作，將楓之谷製成[[動畫|動畫]]。目的在於可使尚未接觸楓之谷的玩家也能參予楓之谷的世界，達到增加玩家人數效果。動畫在2007年10月7日開始放送，全25話。

=== NDS游戏 ===
{{Main|MapleStory DS}}
2007年1月9日韓國[[NEXON|NEXON]]在發表了預定移植MapleStory至[[Nintendo_DS|Nintendo DS]]的消息。

===网页游戏===
2012年3月12日，雅虎日本开启MapleStory的网页游戏版本《-{zh-hans:冒险;zh-hant:皇家}-骑士团》({{lang-ja|メイプルストーリー騎士団}})，同年腾讯中文版《冒险岛之旅》上线，但二者运营没多久就宣布关服。<ref>{{cite web|url=http://www.4gamer.net/games/150/G015015/20120312041/|title=新作ソーシャルゲーム「メイプルストーリー騎士団」，本日サービス開始。最強の騎士団を作ろう}}</ref>

===手机游戏===
2014年9月，NEXON推出智能手机平台新作《-{zh-hans:冒险岛手游;zh-hant:口袋楓之谷}-》({{lang-ko|포켓 메이플스토리}}，{{lang-en|Pocket Maple Story}})。<ref>{{cite web|url=http://tech.hexun.com/2014-07-17/166731339.html|title=Nexon手游新作《口袋枫之谷》9月在韩国推出}}</ref>2015年8月5日宣布上架日区。<ref>{{cite web|url=http://sy.pcgames.com.cn/540/5408987.html|title=《冒险岛》手游先行上架日区App Store}}</ref>

2015年，遊戲橘子旗下的蜂玩娛樂代理該作，于8月初推出台港澳地區服務器，譯名為口袋楓之谷，台港澳地區服務器於2017/3/31關閉。<ref>{{cite web|url=http://shouyou.yesky.com/366/82357366.shtml|title=冒险岛手游《口袋枫之谷》繁中版8月登场}}</ref>中国则由世纪天成代理，于2015年9月22日正式运营，译作冒险岛手游。<ref>{{cite web|url=http://games.sina.com.cn/ol/n/hgwyxw/2015-07-27/fxfhxmp9579154.shtml|title=世纪天成宣布代理Nexon新作《冒险岛手游》}}</ref>

2018年2月22日，Nexon推出新作即時戰略遊戲《楓之谷 : 突擊》（Maple Blitz X／메이플블리츠X，海外版名稱：MapleStory Blitz）

2016年，Nexon推出新作《MapleStory M》，於2016年10月13日下午3點公測。<ref>{{cite web|url=http://news.17173.com/content/2015-11-12/20151112144733956_all.shtml|title=《冒险岛M》体验:比《冒险岛手游》更注重互动}}</ref>

== 相关事件 ==
=== 抄襲事件 ===
2011年6月，北京奇客星空網路技術有限公司推出了一款名叫《[[冒險王_(遊戲)|冒險王]]》的[[網頁遊戲|網頁遊戲]]，在遊戲系統、畫風乃至怪物形態上大量模仿《MapleStory》，引發MapleStory玩家非議。

2015年，奇客创想公司又推出了《冒险島——口袋里的冒险岛》手游再次引起争议。因正值Nexon发行正版手游之际，盛大网络与世纪天成相继将其告上法院，不久该游戏宣布关闭。目前Play商店仍可搜尋到《冒險島》<ref>{{cite web|url=http://bjhdfy.chinacourt.org/public/detail.php?id=3660|title=网上应用商店销售侵权手游，苹果被索赔1000万|publisher=海淀法院网|date=2015年7月10日}}</ref><ref>{{cite web|url=http://www.mxdshouyou.com/homepage/article/2015/09/09/44325.html|title=部分公司厂商侵权《冒险岛手游》的律师声明|publisher=冒险岛手游官网|date=2015年9月9日}}</ref>
