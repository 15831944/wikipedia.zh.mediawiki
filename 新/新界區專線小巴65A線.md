'''新界區專線小巴65A線'''是由順威實業有限公司營運的一條[[香港小巴|香港小巴]]線，行走[[黃泥頭公共運輸交匯處|黃泥頭]]及[[沙田市中心|沙田市中心]]<ref>{{Cite web |url=http://www.681busterminal.com/version2/index.php?fi=a201&id=306 |title=681巴士總站 - 新界65A線 |access-date=2011-04-30 |archive-url=https://web.archive.org/web/20160305095235/http://www.681busterminal.com/version2/index.php?fi=a201&id=306 |archive-date=2016-03-05 |dead-url=no }}</ref>。

==簡介及歷史==
*1997年：路線投入服務，由鴻基專線小巴經營。
*2009年4月12日：本線調整票價，收費由$3.7調整至$3.9，與2008年6月8日加價的[[九龍巴士83K線|九龍巴士83K線]]打成平手。
*2009年8月2日起：為配合[[新界區專線小巴65K線|65K]]不再經沙田市中心，本路線提早頭車時間，協助分流黃泥頭、小瀝源往來火炭及沙田的乘客。
*由於營辦商鴻基專線小巴的掌控者袁氏家族早年處理合伙人關係不當，加上家族內部對各成員於公司內的權限有所爭拗，因此鴻基專線小巴近年多番捲入訴訟<ref>{{Cite web |url=http://legalref.judiciary.gov.hk/doc/judg/word/vetted/other/ch/2003/HCA004252_2003.doc |title=香港特別行政區 - 高等法院原訟法庭 - 民事司法管轄權 - 案件編號2003年第4252號 |access-date=2018-07-24 |archive-url=https://web.archive.org/web/20160106132124/http://legalref.judiciary.gov.hk/doc/judg/word/vetted/other/ch/2003/HCA004252_2003.doc |archive-date=2016-01-06 |dead-url=no }}</ref><ref>[http://hk.on.cc/hk/bkn/cnt/news/20151218/bkn-20151218141338755-1218_00822_001.html 不滿胞姊攬權　小巴公司少東促賣斷股份] {{Wayback|url=http://hk.on.cc/hk/bkn/cnt/news/20151218/bkn-20151218141338755-1218_00822_001.html |date=20180724062931 }}，東網，2015-12-18</ref><ref>{{Cite web |url=http://202.125.90.12/cnt/news/20120204/00176_072.html |title=東方日報：法庭：小股東興訟 專線巴公司判賠（2012年2月4日） |access-date=2018年7月24日 |archive-url=https://web.archive.org/web/20180724062724/http://202.125.90.12/cnt/news/20120204/00176_072.html |archive-date=2018年7月24日 |dead-url=no }}</ref>。運輸署以該公司經營不善為由，決定收回該公司各條專綫小巴路線之經營權，並於2016年11月25日刊憲重新招標。<ref>第6711號公告：〈[http://www.gld.gov.hk/egazette/pdf/20162047/cgn201620476711.pdf 道路交通（公共服務車輛）規例（第374D章）--九龍及新界的公共小巴（專線）路線組別] {{Wayback|url=http://www.gld.gov.hk/egazette/pdf/20162047/cgn201620476711.pdf |date=20180211185932 }}〉，《香港特別行政區憲報》第20卷第47期，2016年11月25日。</ref>
*2017年4月1日起：本線由棉記汽車轄下順威實業有限公司接辦，並加入「兩元乘車優惠」。<ref>[https://www.hk01.com/%E7%A4%BE%E6%9C%83%E6%96%B0%E8%81%9E/81360/%E5%86%8D%E5%A4%9A%E5%85%AD%E6%A2%9D%E5%B0%88%E7%B7%9A%E5%B0%8F%E5%B7%B4%E8%B7%AF%E7%B7%9A%E5%8A%A0%E5%85%A5-%E5%85%A9%E5%85%83%E4%B9%98%E8%BB%8A%E5%84%AA%E6%83%A0 再多6條專線小巴路線加入「2元乘車優惠」] {{Wayback|url=https://www.hk01.com/%E7%A4%BE%E6%9C%83%E6%96%B0%E8%81%9E/81360/%E5%86%8D%E5%A4%9A%E5%85%AD%E6%A2%9D%E5%B0%88%E7%B7%9A%E5%B0%8F%E5%B7%B4%E8%B7%AF%E7%B7%9A%E5%8A%A0%E5%85%A5-%E5%85%A9%E5%85%83%E4%B9%98%E8%BB%8A%E5%84%AA%E6%83%A0 |date=20180417105622 }}HK01 2017年3月30日</ref>

==服務時間、班次及收費==
此路線的服務時間分別為每日上午5時正至下午11時正（黃泥頭開）和每日上午5時10分至下午11時半（沙田大會堂開），班次均為約5分鐘一班。此路線之全程收費為$5.5，並設有$3.1的雙向分段收費，往返小瀝源路草地滾球場至黃泥頭。
