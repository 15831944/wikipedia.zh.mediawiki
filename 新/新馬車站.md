'''新馬車站'''位於[[台灣|台灣]][[宜蘭縣|宜蘭縣]][[蘇澳鎮_(台灣)|蘇澳鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[宜蘭線|宜蘭線]]的[[鐵路車站|鐵路車站]]。

==站名由來==
1920年設站時名為「新城」，1975年改名為「聖湖」。1983年，由於車站鄰近[[宜蘭縣|宜蘭縣]][[蘇澳鎮|蘇澳鎮]]的[[新城_(宜蘭縣蘇澳鎮)|新城]]、[[馬賽_(臺灣)|馬賽]]兩處而各取頭一字改為今名。

== 車站構造 ==
此站只有上下行共用的[[島式月台|島式月台]]一座，但因接近蘇澳新站，站內有多股軌道。

宜蘭線在此站的路段為急彎，[[曲率半徑|曲率半徑]]僅300公尺<ref>[https://udn.com/news/story/12556/3437661 普悠瑪出軌翻覆！ 台鐵「奪命七大彎道」路段曝光]</ref>。一般列車限速65公里通過，[[傾斜式列車|傾斜式列車]]限速75公里<ref>[https://tw.appledaily.com/headline/daily/20181023/38159924/ 台鐵恐怖彎道 共7處 普悠瑪被逼降速 台鐵不擬截彎取直]</ref>。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]，由[[蘇澳新站|蘇澳新站]]管理。
*僅[[臺鐵區間車|區間車]]停靠。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1920年（[[大正|大正]]9年）11月22日：設「新城」站。
*1970年底，完成如圖所示的混凝土站房。
*1975年6月1日：改為「聖湖」站。因在建設[[北迴線|北迴線]]時，於[[花蓮縣|花蓮縣]]附近亦設置一座新城車站，為避免與其混淆，而本站的位於蘇澳鎮的聖湖-{里}-附近，故改名為「聖湖」；同年，本站因應[[宜蘭線|宜蘭線]]鐵路南聖湖（今[[蘇澳新站|蘇澳新站]]）至冬山間雙軌化工程<ref>{{cite book|url=http://trstour.com/yi-lan%20line.htm|author=|title=〈宜蘭線（蘇澳—八堵）〉，《臺灣驛站之旅》網站|location=|page=|date=2004年}}</ref>，而縮短與[[蘇澳新站|蘇澳新站]]間的長度，兩者距離只有1.1公里。
*1983年4月1日：改為「新馬」站。因本站位於蘇澳鎮新城-{里}-和永榮里（古名「馬賽」）之間，所以將兩地各取一字來命名，而更新站名為「新馬」。
*2000年10月1日：由簡易站降為招呼站，由[[蘇澳新站|蘇澳新站]]代管。
*2003年11月，因[[宜蘭線|宜蘭線]]鐵路電氣化工程，又再次縮短本站與[[蘇澳新站|蘇澳新站]]間的距離為九百公尺，也因此新馬—蘇澳新路段成了台灣環島鐵路主線仍在營運的車站間距最短的一段，直至2006年12月1日被新建的[[新烏日車站|新烏日車站]]與[[烏日車站|烏日車站]]八百公尺站距打破紀錄。
*2015年6月30日：啟用多卡通刷卡機。
*2018年
**10月21日：一列搭載約310人由樹林開往台東的6432次[[普悠瑪號列車|普悠瑪號列車]]在通過此站時發生[[2018年宜蘭列車出軌事故|出軌事故]]，造成18人死亡、171人輕重傷，車站設施毀損。
**10月24日：車站軌道修復後，恢復雙線雙向通車。{{main|2018年宜蘭普悠瑪列車出軌事故}}

== 未來規劃 ==
該站因2018年宜蘭普悠瑪列車出軌事故，原站址預定進行路型改善而停止客運，新站址預定於武荖坑風景區附近的武荖坑橋北端，新站址是否更名武荖坑站尚為未知。
