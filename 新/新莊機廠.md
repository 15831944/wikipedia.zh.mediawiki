'''新莊機廠'''位於[[台灣|台灣]][[新北市|新北市]][[新莊區|新莊區]]和[[桃園市|桃園市]][[龜山區|龜山區]]交界處，[[樂生療養院|樂生療養院]]舊址，為[[台北捷運|台北捷運]][[中和新蘆線|中和新蘆線]]的[[機廠|機廠]]。

== 機廠概要 ==
*興建中，2008年12月3日恢復施工，預計2019年完工。

*土建標：2008年1月2日，CK570F區段標廠商工信工程因樂生案本工程停工達183天，依據合約規定辦理終止契約，解除工程契約。2008年6月，更改標號為CK570J區段標，重新發包，由大陸工程得標。<ref>{{cite web |url=http://www.ndpo.dorts.gov.tw/cgi-bin/SM_theme?page=456a7773 |title=存档副本 |accessdate=2008-03-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20141023055527/http://www.ndpo.dorts.gov.tw/cgi-bin/SM_theme?page=456a7773 |archivedate=2014-10-23 }}</ref><ref>{{cite web |url=http://tw.stock.yahoo.com/news_content/url/d/a/080126/1/sqte.html |title=存档副本 |accessdate=2008-03-16 |deadurl=yes |archiveurl=https://web.archive.org/web/20080131213550/http://tw.stock.yahoo.com/news_content/url/d/a/080126/1/sqte.html |archivedate=2008-01-31 }}</ref>

*軌道標：併信義線東延段CR580C區段標發包，由中華工程／日商大豐聯合承攬。

*機電系統標：原廠商[[阿爾斯通|阿爾斯通]]解約，重新發包中。

*工程進度：預定進度為99.96%，實際進度為95.43%（截至2018年12月底）。

== 機廠位置與結構 ==
*廠房規模：三級機廠，負責[[中和新蘆線|中和新蘆線]]列車三級維修。<ref>{{cite web |url=http://www.taipei.gov.tw/cgi-bin/Message/MM_msg_control?mode=viewnews&ts=4876b27f:1344 |title=存档副本 |accessdate=2009年3月31日 |deadurl=yes |archiveurl=https://web.archive.org/web/20160911125523/http://www.taipei.gov.tw/cgi-bin/Message/MM_msg_control?mode=viewnews&ts=4876b27f:1344 |archivedate=2016年9月11日 |df= }}</ref>

*機廠設備：警衛室、駐車場（上方為轉乘停車場）、維修工廠（含員工辦公室、SSS2設施變電站）、c371 400系列車駐車軌、SSS1機廠設施變電站、主變電站/牽引動力配電室（BSS/TSS）、污水處理廠、廢棄物堆置場、電聯車清洗場、捷運警察分隊以及059車站附設轉乘汽、機車停車場等。

*代辦工程：樂生第二人行陸橋（主辦機關：國發會、衛生福利部）

== 歷史 ==
*2012年1月5日：新莊線[[大橋頭站|大橋頭站]]－[[輔大站|輔大站]]通車，因新莊機廠尚未完工，[[丹鳳站_(新北市)|丹鳳站]]與[[迴龍站|迴龍站]]暫作為臨時儲車區，以舒緩列車儲車及調度需求。
*2013年6月29日：輔大站到迴龍站通車。
*2016年6月：[[樂生療養院舊院區保留運動|樂生青年]]及自救會陳情抗議，變更設計增設入口意象（第二人行陸橋）。
*2016年12月：樂生青年再度抗議，除入口意象外，要求依樂青提出台大城鄉所退休教授劉可強的「[https://www.civilmedia.tw/archives/63579 大平台方案]」。12月22日國發會會議中初步確認縮減車軌，改設大平台。
*2017年10月5日：樂生青年侵入機廠工地，於施工機具張貼抗議標語，要求陸橋案停工。<ref>[http://www.appledaily.com.tw/realtimenews/article/local/20171005/1217243/%E4%B8%8D%E6%BB%BF%E9%9B%BB%E6%A2%AF%E9%99%B8%E6%A9%8B%E6%96%BD%E5%B7%A5%E3%80%80%E6%A8%82%E7%94%9F%E8%87%AA%E6%95%91%E6%9C%83%E5%BC%B7%E9%97%96%E6%8D%B7%E9%81%8B%E5%B7%A5%E5%9C%B0 不滿電梯陸橋施工　樂生自救會強闖捷運工地]</ref>
*2019年：預定完工啟用。<ref>[https://www.dorts.gov.taipei/cp.aspx?n=4567F73AFCB83D5E 新莊線施工現況]</ref>
