'''新竹市立建華國民中學'''，簡稱'''建華國中'''，是一所位於[[臺灣|臺灣]][[新竹市|新竹市]][[東區_(新竹市)|東區]]的[[新竹市政府|市立]][[國民中學|國民中學]]，成立於1939年。

== 沿革 ==
1939年5月5日，[[新竹州|新竹州]]'''新竹高等女子家政學校'''成立於今新竹市東區林森路29號對面一帶，首任校長為仲田雅男，是當時新竹地區女子最高學府。

1946年5月，張棟蘭接任校長並將其改為'''新竹初級中學'''，於初期僅設初中男生部共八班。1950年9月，開辦高中部成為完成中學。又同年十月全台行政區域調整，本市改縣，此校隨而易稱新竹縣立新竹中學，1951年遷校於[[十八尖山|十八尖山]]西峰(虎頭山)下之新校舍(即現址)<ref>http://sowhc.sow.org.tw/html/observation/eighteen/01/01.html</ref><ref>http://nicecasio.pixnet.net/blog/post/354658433-新竹市-十八尖山-歷史沿革</ref>；1956年八月新竹縣又奉令試辦國校免試升學，此校校名又改為新竹縣立第一中學。 經歷任校長 - 羅富生、楊卓然、張正春等先生任內逐年增班充實設備。新竹縣立第一中學是當年新竹,桃園,苗栗地區初中生升學的第一志願{{請求來源}}. 竹一中附有高中部.

1982年7月1日新竹市升格為[[省轄市|省轄市]]，此校奉令改為新竹市立建華國民中學，1973年八月一日奉准設附設國民中學補習學校。<ref>{{Cite web |url=http://www.chjh.hc.edu.tw/front/bin/ptlist.phtml?Category=1 |title=存档副本 |access-date=2015-01-28 |archive-url=https://web.archive.org/web/20141015121800/http://www.chjh.hc.edu.tw/front/bin/ptlist.phtml?Category=1 |archive-date=2014-10-15 |dead-url=yes }}</ref>

==校訓由來==
1939年，[[教育部|教育部]]宣佈全國各級學校的共同校訓，出自[[春秋戰國|春秋戰國]]齊國時的宰相[[管仲|管仲]]就提出治國的方針：「禮、義、廉、恥，國之四維，四維不張，國乃滅亡。」

==特殊事蹟==
1968年8月28日，[[紅葉少棒隊|紅葉少棒隊]]對日本和歌山少棒隊友誼賽下午二時於新竹建華國中舉行。校舍、柔道館、綜合科學館屋頂、陽台都擠滿人，據大會人員守估計觀眾達三萬人左右，其中動員大批警員、童子軍及義警維持秩序，延至下午二時四十分才勉強能比賽。 比數五比二，紅葉再度擊敗日本。<ref>https://www.youtube.com/watch?v=NA-b0GGNmSE&x-yt-cl=85114404&x-yt-ts=1422579428#t=17</ref><ref>{{cite web |url=http://120.126.83.211/root1/artsandcultures/yang/file/100411140108-red%20leave%201968.pdf |title=存档副本 |accessdate=2015-01-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20150202113548/http://120.126.83.211/root1/artsandcultures/yang/file/100411140108-red%20leave%201968.pdf |archivedate=2015-02-02 }}</ref><ref>http://mypaper.pchome.com.tw/robert_tu2895/post/1321827084</ref><ref>http://city.udn.com/63078/5191548</ref>
