'''新字体'''是日本在1923年制定的《[[常用漢字表|常用漢字表]]》{{NoteTag|注意此《[[常用漢字表|常用漢字表]]》（1923年草擬）與《[[當用漢字|當用漢字]]表》廢除後的的《[[常用漢字|常用漢字]]表》（1981年公布，2010年改定）不同。}}、1946年公布《[[當用漢字|當用漢字]]表》至今，[[日本|日本]]社會通行的[[漢字|漢字]]字體，其简化程度总体而言不如[[中国大陆|中国大陆]]的[[简化字|简化字]]，[[言部|言部]]、[[金部_(部首)|金部]]、[[糸部|糸部]]等常用[[部首|部首]]没有简化。日本的「字體」與中文的「[[字形|字形]]」是同一個概念。
在日本，日常生活主要使用新字体，但政府并未宣布废止[[舊字體|舊字體]]，在专有名词等很多场合中，[[舊字體|舊字體]]仍可使用。新字体多为'''略字'''（即民间常用的[[简笔字|简笔字]]），但亦有少数增添笔画的情况，如“{{JIS2004字體|歩（步）}}”“{{JIS2004字體|晩（晚）}}”。

== 歷史 ==
日本是一個長期使用漢字的非漢語系國家，由於漢字學之不易，因此民间底層长期流行一些[[简笔字|简笔字]]。在[[第二次世界大戰|第二次世界大戰]]期間，[[日本|日本]]主張侵略一派認為[[漢字|漢字]]過多不利學習，主張完全[[废除汉字|废除汉字]]。但因为日文[[日語假名|假名]]只能表音，而日文漫长的历史过程中吸收的具有表意功能的汉字一旦彻底废除，会形成大量[[同音词|同音词]]，对阅读和理解没有断词的日文造成困难，因此汉字最终得以留存，僅進行了一些簡化。

1946年，[[日本内阁|日本内阁]]公布了《[[当用汉字表|当用汉字表]]》，收字1850个，其中131个采用新字体，原写法則被稱作'''[[舊字體|舊字體]]'''（{{Jpn|j=旧字体|kyu=舊字體|hg=きゅうじたい|rm=Kyūjitai}}）。

1981年10月1日，《[[常用漢字表|常用漢字表]]》代替《[[当用汉字表|当用汉字表]]》，收錄1945字{{NoteTag|1=2010年，日本追加196個新的常用漢字，並削除5個漢字，合計共2136字。}}，其中新字體355個{{NoteTag|1=由於「{{Font-ja|辨}}」、「{{Font-ja|瓣}}」和「{{Font-ja|辯}}」被合併簡化為「{{Font-ja|弁}}」，355個新字体对应的[[舊字體|舊字體]]有357個。}}。後來發佈的《[[人名用漢字表|人名用漢字表]]》收11新字；《[[表外漢字字體表|表外漢字字體表]]》收22新字，共388字。（《人名用漢字表》數次修訂後，新字體數量增至19個；《改定常用漢字表》亦新收錄了一些新字體，故目前已經不止388個。）

但這三百餘個僅是[[日本政府|日本政府]]公佈的新舊字體對照，並不代表學術界使用的，譬如《[[人名用漢字表|人名用漢字表]]》中的“{{Font-ja|聡}}”，在字典中為“{{Font-ja|聰}}”的新字體，但《[[人名用漢字表|人名用漢字表]]》中，只有新字體“{{Font-ja|聡}}”。而且很多[[舊字體|舊字體]]尚未收錄至[[Unicode|Unicode]]編碼。如果以字典的新舊字體對應來看，新舊字體要多于389字，學術界中《大漢和辭典》等書籍則更爲專業。

== 中日漢字異同 ==
新字体中，字形與[[港澳台|港澳台]]取字相同的有「{{Font-ja|為}}」、「{{Font-ja|併}}」等字。

而与[[中国大陆|中国大陆]]的[[规范汉字|规范汉字]]相同的有「{{Font-ja|万}}」、「{{Font-ja|医}}」、「{{Font-ja|会}}」、「{{Font-ja|学}}」、「{{Font-ja|礼}}」、「{{Font-ja|昼}}」、「{{Font-ja|独}}」、「{{Font-ja|争}}」{{NoteTag|[[中華人民共和國|中華人民共和國]]1965年出版的《[[印刷通用漢字字形表|印刷通用漢字字形表]]》選用之字形，多以手寫俗體為準，有些字筆劃少於傳統舊字形，常被錯認為《[[简化字总表|简化字总表]]》裡的簡化字，例如「{{Font-ja|争（爭）}}」等。}}、「{{Font-ja|国}}」、「{{Font-ja|党}}」、「{{Font-ja|乱}}」等；差不多相同的有「{{Font-ja|恋}}（{{lang|zh-cn|恋}}）」{{NoteTag|日本的「{{Font-ja|恋}}」字上部為「亦」，而[[中国大陆|中国大陆]][[简化字|简化字]]的「{{lang|zh-cn|恋}}」上部中間為兩豎。}}、「{{Font-ja|画}}（{{lang|zh-cn|画}}）」、「{{Font-ja|浅}}（{{lang|zh-cn|浅}}）」、“{{Font-ja|鉄}}（{{lang|zh-cn|铁}}）”、“{{Font-ja|変}}（{{lang|zh-cn|变}}）”、“{{Font-ja| 辺}}（{{lang|zh-cn|边}}）”、“{{Font-ja| 圧}}（-{压}-）”、“{{Font-ja| 庁}}（-{厅}-）”、“{{Font-ja| 継}}（{{lang|zh-cn|继}}）”等，其中「{{Font-ja|恋}}（{{lang|zh-cn|恋}}）」、「{{Font-ja|画}}（{{lang|zh-cn|画}}）」、「{{Font-ja|浅}}（{{lang|zh-cn|浅}}）」、「{{Font-ja|将}}（{{lang|zh-cn|将}}）」等与[[简化字|简化字]]擁有相同的[[Unicode|Unicode]]編碼。

也有與中文漢字完全不像的，如「{{Font-ja|図}}（-{圖·图}-）」、「{{Font-ja|糸}}（-{絲·丝}-）」{{NoteTag|「糸」作「-{絲}-」之異體字在《[[集韻|集韻]]》中已有記載；亦有「-{zh-cn:mì; zh-hk:mì; zh-tw:ㄇㄧˋ;}-」的讀法，但該讀音與日文的「{{Font-ja|糸（し／いと）}}」無關。}}。

== 簡化方法 ==
基本上，採中、日典籍或約定俗成的[[俗字|俗字]]。

;行、草書楷化
例：{{Font-ja|図（圖）、観（觀）、昼（晝）、寿（壽）}}等。

;[[俗字|俗字]]、[[异体字|异体字]]升格為標準字體
例：{{Font-ja|体（體）、関（關）、広（廣）、円（圓）{{NoteTag|《倭楷正訛·附錄省文集》二十八頁即有外「囗」內「丨」之形。}}、区（區）}}等。

;統一字體
一個漢字有多種寫法的，取一個為正字。例如：「{{Font-ja|島}}」「{{Font-ja|嶋}}」「{{Font-ja|嶌}}」統一作「{{Font-ja|島}}」（本字為「{{Font-ja|㠀}}」）等。

;替換聲符
將繁複的[[聲符|聲符]]替換為較簡單的。例如借中文死字「{{Font-ja|囲}}」{{NoteTag|[[金代|金代]]《[[四聲篇海|四聲篇海]]·囗部》：「囲，他琮切，策也。」[[江戶時代|江戶時代]]《倭楷正訛·附錄省文集》二十頁：「囲，-{圍}-。」}}為「{{Font-ja|圍}}」，原來的聲符是「{{Font-ja|韋}}」，發音為「{{Font-ja|い}}」，替換成同音的「{{Font-ja|井}}」（不過，這裏的「{{Font-ja|韋}}」是[[音讀|音讀]]，「{{Font-ja|井}}」是[[訓讀|訓讀]]）。

;[[省文|省文]]刪除繁複部分
例：{{Font-ja|応（應）、芸（藝）、声（聲）、医（醫）、糸（絲）、号（號）、虫（蟲）、涙（淚）、者（者）}}等。

;增筆[[俗字|俗字]]
亦有少數增添筆畫的情況，例：{{JIS2004フォント|歩（步）、頻（頻）}}等。

;變更部首
採[[俗字|俗字]]簡化後，一些漢字的部首發生了變化，例如，原本從「[[鬥部|{{Font-ja]]」的「{{Font-ja|鬪}}」字簡化為「{{Font-ja|闘}}」後改從「[[門部|{{Font-ja]]」。另外，「{{Font-ja|聲}}」（[[耳部|耳部]]）、「{{Font-ja|醫}}」（[[酉部|酉部]]）等字簡化時，原來的部首被刪除，變成「{{Font-ja|声}}」（[[士部|士部]]）、「{{Font-ja|医}}」（[[匚部|匚部]]）。

;类推简化
例如，「{{Font-ja|區}}」已简化「{{Font-ja|区}}」，「{{Font-ja|驅}}」「{{Font-ja|樞}}」同样简化作「{{Font-ja|駆}}」「{{Font-ja|枢}}」。類推簡化一般限於《[[常用漢字表|常用漢字表]]》，類推簡化產生的《常用漢字表》外的新字體稱作「[[擴張新字體|擴張新字體]]」，例如「{{Font-ja|鴎}}」{{NoteTag|2000年發布的《表外漢字字體表》收錄「{{Font-ja|鴎}}」為「{{Font-ja|鷗}}」的「簡易慣用字體」。正字「{{Font-ja|鷗}}」在日文環境下是「環境依存文字（Unicode）」，在日本的某些較舊的手機、電腦上可能無法輸入、顯示。}}。

;其他簡化方法
*[[同音漢字書寫規則|同音漢字書寫規則]]
*日本民間還流傳一些[[簡筆字|略字]]、[[擴張新字體|擴張新字體]]，但它們並非正式的簡化漢字。

==與既存漢字衝突==
有些新字体被简化到被另一個原本無關聯的字，與中文[[簡化字|簡化字]]的合併簡化類似。如：
*新字體：{{Font-ja|缶}}；舊字體：{{Font-ja|罐}}；音讀：{{lang|ja|かん}}（kan）
*:「{{Font-ja|缶}}」原本的讀音是「{{Font-ja|ふ}}（fu）」。
*新字体：{{Font-ja|芸}}；旧字体：{{Font-ja|藝}}；音读：{{lang|ja|げい}}（gei）
*:原本「{{lang|ja|芸}}」音读是「{{lang|ja|うん}}（un）」，训读是「{{lang|ja|くさぎる}}」（kusagiru），是「{{lang|ja|草}}（kusa）」和「{{lang|ja|切る}}（kiru）」的合成词，即除草的意思。
*新字体：{{lang|ja|欠}}；旧字体：{{lang|ja|缺}}；音读：{{lang|ja|けつ}}（ketsu）
*:「{{lang|ja|欠}}」原本讀作「{{lang|ja|けん}}（ken）」。如果使用新字體，「{{lang|ja|欠缺（けんけつ/kenketsu）}}」就不得不改為「{{lang|ja|欠欠}}」，顯得十分不自然，因此法律用語中仍用「{{lang|ja|欠缺}}」。
