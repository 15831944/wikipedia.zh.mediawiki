[[File:Roomba3g.jpg|thumb]]
'''機器人吸塵器'''，又稱自動打掃機、自動吸塵機、[[智能|智能]]吸塵、掃地[[機器人|機器人]]等，用以打掃家裡環境。其原理為一[[自動化技術|自動化技術]]機身，搭配有集塵設施的[[真空|真空]]吸塵裝置，由機身內置的[[微處理器|微處理器]]控制路徑行走，加上毛刷或是抹布、除塵紙打掃，達到更好的居家自動打掃效果。

== 歷史 ==
[[File:Roomba_original.jpg|thumb]]
 
第一款量產的自動打掃機是由瑞典的[[伊萊克斯|伊萊克斯]]於1997年引入的{{tsl|en|Electrolux Trilobite|}}，當時曾在英國[[BBC|BBC]]電視台的科學節目，明日世界(Tomorrow's World)中介紹。<ref>[http://news.bbc.co.uk/2/hi/technology/3031219.stm Robot cleaner hits the shops]</ref>

英國的[[戴森_(公司)|戴森公司]](Dyson)於2001年推出名為DC06的自動打掃機，但因為價格過於高昂，並沒有推出市面。<ref>[http://mashable.com/2014/02/12/dyson-robot-vacuum-vision/ Dyson's Vision: A Robot Vacuum that can see]</ref>

其後，美國的[[iRobot|iRobot]]於2002推出[[Roomba|Roomba]]，原本預計生產10,000至15,000部，發售後大受市場歡迎，單是聖誕期間已出售50,000部。<ref>[http://www.businessweek.com/stories/2003-10-05/how-the-roomba-was-realized How the Roomba was realized]</ref>自此，多家公司推出不同型號的自動打掃機。

== 概述 ==
自動打掃機的機身多以圓形為主，也有方型圓角及三角型圓角的，多盡量設計得細小及矮，以便能進入細小空間及傢俬底部。使用[[充電電池|充電電池]]推動，以[[遙控器|遙控器]]、或是機身上的面板操控。一般能設定時間預約打掃，大部份型號能在電源將用盡時自行返回充電器充電。

運作時以特定的路線模式行走，走動間同時開動內置的吸塵機及掃，將地方的灰塵吸除。行走時能自動偵測及避開障礙物，例如碰到牆壁時會自行轉向，有防止墜落功能，以避免掉落梯間等地方。部分較早期機型可能缺少部分功能，現今已慢慢普及。

== 構造及原理 ==
自動打掃機為一移動裝置，在移動的同時以各種方式清潔地面。

=== 行走 ===
[[File:Roomba_time-lapse.jpg|thumb]][[File:Neato_XV-11.jpg|thumb]]

行走推動的方式絕大部份為二輪推動，也有使用[[履帶|履帶]]的設計。

行走路線的制定方式有很多種，最譜遍的是類似首先為iRobot、Roomba所採用的既定預置模式，[[微處理器|微處理器]]根據預置程式以簡單的法則走行，例如螺旋形行走、沿牆邊行走、隨機行走並在碰到障外物後略為改變角度等，這設計是基於身兼[[麻省理工學院|麻省理工學院]]研究員及iRobot的[[首席技術長|首席技術長]](CTO)，Rodney Brooks提出的哲學：[[機人|機人]]應該像昆蟲一樣，以簡單的處理基制應對問題。

有些設計會有自身的導航功能，例Evolution Robotics(於2012年被iRobot收講)的北極星導航器(NorthStar Navigation)，原理就有如室內的[[衛星導航系統|衛星導航系統]]，<ref>{{Cite web |url=http://www.irobot.com/For-the-Home/Floor-Mopping/Braava.aspx |title=存档副本 |access-date=2015-12-06 |archive-url=https://web.archive.org/web/20151208060135/http://www.irobot.com/For-the-Home/Floor-Mopping/Braava.aspx |archive-date=2015-12-08 |dead-url=yes }}</ref>使自動打掃機能知道現正身處於家中的那一位置。

也有採用[[同步定位與地圖構建|同步定位與地圖構建]]（SLAM/Simultaneous localization and mapping），通過感測器在行走過程中即時得出四週的地圖，然後推算出自身位置，並規劃清潔路線。所使用的感測器有影象傳感器，也有激光測距掃描。
2010年2月，[[Neato_Robotics|Neato Robotics]]推出的XV-1使用同步定位與地圖構建系統，以[[激光测距仪|激光距離感感器]]（LDS）作360°掃描得出二維地圖，然後規劃出高效率的清潔跑線，當中原理與Google[[自動駕駛系統|自動駕駛系統]]中使用的成象系統相同。2016年，[[小米科技|小米]]也推出相近設計產品。

Dyson的Dyson 360 Eye，以全方位視野影象界定自身位置，同時判斷那些位置未清潔過。<ref name=360Eye>{{cite web|title=Dyson 360 Eye™ robot|url=https://www.dyson360eye.com/|website=Dyson|accessdate=2015-03-26|deadurl=yes|archiveurl=https://web.archive.org/web/20150326022104/https://www.dyson360eye.com/|archivedate=2015-03-26}}</ref>
偵測障礙物的感測方訓式有接觸式傳感、[[紅外線|紅外線]]感測、[[超聲波|超聲波]]感測等，接觸式傳感的缺點在於有機會碰花傢俱表面及會發出額外的噪聲，當感測到碰到有障礙物時，微處理器向會轉向改變行走向方，避開障礙物。而為防止在梯間等高處掉下，前方底部多有感測器探測地面情況，從而能避免掉下。

然而，自動打掃機仍然有機會因複雜環境碰上而"被困"，這時，自動打掃機就會以閃燈、聲響，甚至模擬語音作"求救"。

=== 清潔 ===
[[File:Evolution_Robotics_Mint_Cleaner.jpg|thumb]]
清潔地面的方法有[[吸塵機#真空吸塵機|真空吸塵機]]、[[毛刷|毛刷]]、[[抹布|抹布]]。大部設計都包括有真空吸塵機及毛刷，一般會在前方左右或其中一方有側邊刷，邊刷旋轉時會將灰掃至真空吸塵機進氣口前方，然後由真空吸塵機把灰塵吸入收集。部份設計會在真空吸塵機進氣口前方加設中旋轉央刷，增加清潔效果。

=== 電源 ===
[[File:IRobot_Roomba_785_03_(fcm).jpg|thumb]]
早期的設計並沒有自動回停泊處充電的功能，當電量不足時，使用者須手動把其放回充電器充電。隨後出現自動回充設計，充電器上設有充電停泊處及作信標用的紅外線發射器，當需要充電時，自動打掃機會依靠紅外信標作引導，自行回到充電停泊處控並碰接上充電電極，為機身內的電池充電。電池多以[[鎳氫電池|鎳氫電池]]為主，部分用[[鋰離子電池|鋰離子電池]]。

==特殊設計==
機器人科技現今越趨成熟，故每種品牌都有不同的研發方向，擁有特殊的設計，如: 雙吸塵蓋、附手持吸塵器、集塵盒可水洗及拖地功能、可放芳香劑，或是[[光觸媒|光觸媒]]殺菌等功能。
虛擬牆是訊號發射器，當自動打掃機接接到其訊號後就會避開，是給使用者限制自動打掃機的活動範圍用。
