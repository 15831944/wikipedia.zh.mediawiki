'''機場線'''（{{jpn|j=空港線|hg=くうこうせん|rm=Kūkō sen|f=y}}）是自日本[[大阪府|大阪府]][[泉佐野市|泉佐野市]][[泉佐野車站|泉佐野車站]]，至同府[[泉南郡|泉南郡]][[田尻町|田尻町]][[關西機場站|關西機場站]]的[[南海電氣鐵道|南海電氣鐵道]]鐵道路線。是前往位於[[大阪灣|大阪灣]]泉州海域[[關西國際機場|關西國際機場]]的交通路線。

== 路線資料 ==
*管轄、路線距離（{{link-ja|營業距離|営業キロ}}）：全長8.8公里
**南海電氣鐵道（[[鐵路公司|第一種鐵道事業者]]）：
***泉佐野站－臨空城站間 1.9公里
**南海電氣鐵道（[[鐵路公司|第二種鐵道事業者]]）、新關西國際機場（[[鐵路公司|第三種鐵道事業者]]）：
***臨空城站－關西機場站間 6.9公里
*[[軌距|軌距]]：1067毫米
*站數：3個（包括起終點站）
*複線路段：全線
*電氣化路段：全線電氣化（直流1500V）
*[[閉塞_(鐵路)|閉塞方式]]：自動閉塞式
*保安裝置：[[自動列車停止裝置#ATS-PN|ATS-PN]]（臨空城站－關西機場站間併設[[自動列車停止裝置#ATS-P形（數碼傳送頻率型）|ATS-P]]）
*最高速度：
**特急：關西國際機場聯絡橋上120公里/小時，其他路段110公里/小時
**其他列車：全線110公里/小時

橫跨臨空城站－關西機場站間，全長3,750公尺的[[關西國際機場聯絡橋|關西國際機場聯絡橋]]，南海電氣鐵道是第二種鐵道事業者、新關西國際機場股份有限公司是第三種鐵道事業者，與[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）[[關西機場線|關西機場線]]除兩站區以外共用上下行線軌道。

==歷史==
* [[1994年|1994年]]（平成6年）
** 6月15日，泉佐野站至關西機場站間路段啟用，提供工作人員往來機場之用。
** 9月4日，關西機場開始營運。特急列車「Rapi:t」開始運行。
* [[2005年|2005年]]（平成17年）11月27日，泉佐野車站周邊的高架化工程完成。
* [[2013年|2013年]]（平成25年）5月31日，[[自動列車停止裝置|自動列車停止裝置]]從開業時的[[ATS-N|ATS-N]]改為[[ATS-PN|ATS-PN]]<ref>{{PDFlink|[http://www.nankai.co.jp/library/company/csr/kankyou_report/pdf/kankyo2014_all.pdf CSR報告書 2014]}} P.34 - 南海電気鉄道</ref>。
*2018年（平成30年）
**9月4日，[[颱風燕子_(2018年)|燕子颱風]]侵襲日本，在[[西日本|西日本]]登陸時刮起的強風，使得一艘為關西機場補給航空油料的[[液貨船|液貨船]]「宝運丸」走錨漂流，並撞擊到關西國際機場聯絡橋南側車道，造成關西機場自動車道的一段橋面偏移，且波及到鐵路的[[架空電車線|架空電車線]]，鐵路路面也偏移約50公分。事故發生後，所有船員生還，鐵公路當時已關閉而沒有車輛受損及人員傷亡，但使得機場連外的鐵路與公路交通暫時中斷<ref>{{Cite web |url=https://www3.nhk.or.jp/news/html/20180904/k10011607811000.html# |title=関西空港の連絡橋にタンカー衝突 乗組員は全員無事救助 |access-date=2018-09-18 |archive-url=https://web.archive.org/web/20180904192123/https://www3.nhk.or.jp/news/html/20180904/k10011607811000.html# |archive-date=2018-09-04 |dead-url=yes }}</ref>，約3000人一時受困在機場內<ref>{{Cite web |url=https://www3.nhk.or.jp/news/html/20180904/k10011609201000.html# |title=関西空港 タンカー衝突で橋不通 利用客3000人が空港内に宿泊 |access-date=2018-09-18 |archive-url=https://web.archive.org/web/20180904192223/https://www3.nhk.or.jp/news/html/20180904/k10011609201000.html# |archive-date=2018-09-04 |dead-url=yes }}</ref>。
**9月14日，吊離遭撞損的南側公路橋面<ref>{{Cite web |url=https://www3.nhk.or.jp/news/html/20180912/k10011626081000.html# |title=関西空港連絡橋の橋桁撤去始まる 鉄道は今月中の復旧目指す |access-date=2018-09-18 |archive-url=https://web.archive.org/web/20180913134312/https://www3.nhk.or.jp/news/html/20180912/k10011626081000.html# |archive-date=2018-09-13 |dead-url=yes }}</ref>，並開始修復鐵路。
**9月18日，清晨起，鐵路恢復通車<ref>[https://udn.com/news/story/6809/3373335 關西機場聯絡鐵路今通車 通勤員工：謝天謝地]</ref>。
