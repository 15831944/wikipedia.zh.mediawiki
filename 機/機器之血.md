《'''機器之血'''》（{{lang-en|''Bleeding Steel''}}，{{lang-ja|''ポリス・ストーリー REBORN''}}）是一部於2017年上映的中國科幻動作片，由[[張立嘉|張立嘉]]執導，[[成龍|成龍]]、[[羅志祥|羅志祥]]、[[歐陽娜娜|歐陽娜娜]]和[[夏侯雲姍|夏侯雲姍]]領銜主演<ref>[http://www.chinatimes.com/realtimenews/20160729002079-260404 羅志祥和成龍合演電影《機器之血》 歐陽娜娜也參一角] 中時電子報，2016年7月29日</ref><ref>[https://tw.appledaily.com/new/realtime/20171215/1260007/ 小豬《機器之血》嗨搭成龍　晚上被「召見」進房…] 蘋果日報，2017年12月15日</ref>。電影在2017年12月29日上映後收穫普遍負評，主要批評劇情混亂老套、特效滑稽、演員不專業；造成電影的票房不佳，為成龍在賀歲檔期最差的表現。

== 劇情 ==
主要講述了主人公林東為配合一起證人保護計畫而捲入一件原本與己無關的案件中展開……之後，林東終究難逃成為無法掌控的巨大陰謀之下的犧牲品的命運。多年後，當初的種種鬥爭也將因為一本即將出版的科幻巨作而被一一揭露……

　　一個瀕臨死亡的女孩Nancy(歐陽娜娜 飾)通過在血液中輸入神秘物質的方式僥倖存活。而救活Nancy的人，也正是Nancy的父親，香港特警林東(成龍 飾)在一樁證人保護案中救出的關鍵人物。由於這個秘密，無法與女兒相認的林東只好將Nancy送往澳洲的孤兒院，自己也在澳洲過著隱姓埋名的生活，在暗中關注並保護著女兒Nancy。

　　2020年，一本關於血液秘密的小說《機器之血》出版，內容正是Nancy的夢。邪惡勢力開始關注這本小說和關於血液的秘密，隨之而來的是對Nancy的爭奪。年輕人李森(羅志祥 飾)也加入到保護Nancy的行動之中，令林東感到十分意外。經歷了一番你死我活的戰鬥，最終邪不壓正，而李森作為復仇者的真實身份也露出水面。

== 製作 ==
《机器之血》為[[成龍|成龍]]、[[羅志祥|羅志祥]]和[[歐陽娜娜|歐陽娜娜]]首度合作的電影，並由[[張立嘉|張立嘉]]執導。[[張立嘉|張立嘉]]透露，片中用的槍械皆是真槍實彈。電影將至澳洲拍攝。<ref>[https://www.beauty321.com/post/14475 《機器之血》全場用真槍實彈！歐陽娜娜、羅志祥首次與成龍合作] beauty美人圈，編輯 / 春捲，2018年1月2日</ref>在開拍前，羅志祥和歐陽娜娜都先在成家班受訓一週，且兩人都對成龍的態度給予好評。<ref>[https://tw.appledaily.com/new/realtime/20171230/1269294/ 成龍攀雪梨歌劇院險凍死娜娜受訓喊「太難」] [[蘋果日報|蘋果日報]] 2017年12月30日</ref>

== 發行 ==
本片於2017年12月22日在中国大陆上映，2017年12月28日在香港上映，2018年1月12日在台湾上映。

== 票房 ==
《机器之血》上映當天僅獲得了1096萬人民幣，上座率僅4.4%，名列第三，慘輸《[[妖貓傳|妖貓傳]]》和《[[芳華|芳華]]》。同時這也是[[成龍|成龍]]在賀歲檔期跌最慘的一次。<ref>[https://www.xcnnews.com/yl/2573826.html 成龍慘遭打臉，《機器之血》票房口碑撲街，連累歐陽娜娜被嫌棄] 中國新聞網 2017年12月23日</ref>首映當週收获7500万人民幣。<ref>[http://liaoning.nen.com.cn/system/2017/12/29/020281018.shtml 《机器之血》《心理罪之城市之光》热映 口碑不好再多“票补”也无用] 東北新聞網 2017年12月29日</ref>電影上映十四天後僅累積了3億人民幣，口碑悽慘，按製片方估计票房得累计7亿元人民币才能回本。<ref>[http://www.g.com.cn/movie/23007747/ 《机器之血》累计票房3亿难回本 马云懵圈了网友：欧阳娜娜负演技] 引力資訊 2018年1月4日</ref>

== 評價 ==
《機器之血》的口碑相當不佳。早在電影上映前，编剧鹦鹉史航在微博上發文，表示自己在看過試映後嘲諷，[[成龍|成龍]]接片功力退步，並認為年輕演員們不太會演戲。<ref>[https://maoyan.com/films/news/30852 知名编剧批成龙《机器之血》烂片：看这种电影，太费我充电宝了] 猫眼，2017年12月13日</ref>此外，[[歐陽娜娜|歐陽娜娜]]的演技也被外界質疑和批評，即使工作人員誇她在戲中能「3秒落淚」。<ref>[http://gotv.ctitv.com.tw/2016/08/255381.htm 歐陽娜娜被砲轟演技爛！工作人員誇「3秒落淚」打臉] 中天快點TV，2016年8月17日</ref>香港新浪給了電影極差的評價，故事、角色、造型、台詞等都遭到批評，並認為各方面都不及同年上映的《[[英倫對決|英倫對決]]》。<ref>[http://sina.com.hk/news/article/20171226/0/6/2/%E6%A9%9F%E5%99%A8%E4%B9%8B%E8%A1%80-%E6%88%90%E9%BE%8D%E4%B9%9F%E6%95%91%E4%B8%8D%E4%BA%86%E8%8F%AF%E8%AA%9E%E7%A7%91%E5%B9%BB-8287141.html 《機器之血》 成龍也救不了華語科幻] 香港新浪，2017年12月26日 11:10</ref>外界認為[[成龍|成龍]]一成不變的風格已令人乏味。<ref>[http://www.huaxiarewen.com/article/1342981.html?p=3 成龙也难保票房，《机器之血》惨遭网友吐槽] {{Wayback|url=http://www.huaxiarewen.com/article/1342981.html?p=3 |date=20180105233522 }} 华夏热闻，2018年1月4日</ref>評論家分析，除了劇本混亂外，[[歐陽娜娜|歐陽娜娜]]的表現令人不知所措，尤其是哭戲和胸替。<ref>[http://new.qq.com/omn/20171223A0R9MH.html 成龙新片《机器之血》，又成烂片，是罗志祥拖了后腿吗？] 腾讯 2017年12月23日</ref>

《[[南華早報|南華早報]]》的James Marsh給了電影0.5/5星，認為「[[成龍|成龍]]最近的電影質量差異很大，這部電影創下各方面新低。」<ref>[http://www.scmp.com/culture/film-tv/article/2125422/film-review-bleeding-steel-globe-trotting-jackie-chan-sci-fi-action Film review: Bleeding Steel – globe-trotting Jackie Chan sci-fi action caper is utter nonsense] [[南華早報|南華早報]] 2017年12月26日</ref>《[[海峽時報|海峽時報]]》的葉偉怡給了電影2.5/5星，批評這部電影的故事背景毫無說服力，從設定到服裝都很荒謬。<ref>[http://www.straitstimes.com/lifestyle/entertainment/steel-yourself-for-action-and-nonsense Bleeding Steel has plenty of stunts from Jackie Chan] [[海峽時報|海峽時報]] 2017年12月20日</ref>
