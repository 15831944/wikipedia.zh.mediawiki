'''秦皇岛港大小码头'''，位于[[河北省|河北省]][[秦皇岛市|秦皇岛市]][[海港区|海港区]]，属于[[全国重点文物保护单位|全国重点文物保护单位]]“[[秦皇岛港口近代建筑群|秦皇岛港口近代建筑群]]”。

==简介==
1898年3月2日，[[总理衙门|总理衙门]]补奏，秦皇岛自开口岸获得批准，1898年4月3日照会各国使节和张贴告示，宣布秦皇岛自行开埠。4月25日，奏准秦皇岛为[[通商口岸|通商口岸]]。自此，[[开平矿务局|开平矿务局]]取得建港、买地、代理地亩权和优先使用权。1898年，开平矿务局在秦皇岛东盐务村设经理处，办理开埠事宜，代理地亩，借款扩大秦皇岛港口，并用已建的木质固定栈桥码头开始煤炭运输等业务<ref name=bainian>{{Cite web|url = http://hebei.sina.com.cn/ts/focus/2015-05-18/10198670.html?from=wap|title = 唐山市档案馆藏百年前秦皇岛码头建设照片|author = |date = 2015-05-18|publisher = 新浪|language = |accessdate = |quote = }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

1899年4月，开始兴建防波堤和木质栈桥码头。施工中利用装载量2250吨的“西平”号轮船及已有的固定木质栈桥码头运输卸载建筑材料。到1900年上半年，已筑防波堤300米，防波堤内侧建木质栈桥码头，码头宽七八米、总长200米，前沿水深18英尺，低潮时14英尺，可泊两艘较大轮船。在初建的防波堤上铺设长约3英里半的[[轻轨铁路|轻轨铁路]]，经港区至[[津榆铁路|津榆铁路]]与[[汤河站|汤河火车站]]连接。堤上设堆煤场、栈房、灯塔。1900年下半年，[[八国联军|八国联军]]侵入，当地[[义和团|义和团]]阻止八国联军在秦皇岛沿海登陆，拆除了部分铁路专用线，并将木质栈桥码头烧毁<ref name=bainian/>。

1900年6月后，[[英国|英国]]骗占开平矿务局，组成“开平矿务有限公司”。该公司依仗英国势力排除了其他列强国家，独占秦皇岛码头经理权，自同年10月起续建扩建大、小码头工程。1903年10月，完成防波堤1158英尺。1914年，基本形成了大、小码头1号至7号泊位。1925年，码头工程全部完工，成为中国北方深水不冻港<ref name=bainian/>。

秦皇岛港口码头对[[开滦煤矿|开滦煤矿]]的经营发挥了重要作用。据不完全统计，自秦皇岛码头转运的开滦煤炭，自1909/1910年度到1928/1929年度共3019万吨，年均150.9万吨。煤炭经秦皇岛码头转运，分销至中国沿海各埠，并销往[[日本|日本]]、[[东南亚|东南亚]]<ref name=bainian/>。

1948年11月，[[中国人民解放军|中国人民解放军]]攻占秦皇岛，秦皇岛港被接收。1952年，人民政府代管开滦之后，开滦经理处移交[[中央人民政府交通部|中央人民政府交通部]]。此后在[[中国共产党|中国共产党]]和人民政府领导下，秦皇岛港为经济建设发挥了巨大作用<ref name=bainian/>。

2013年6月4日，秦皇岛港西港东迁工程启动。当天，西港区最后一艘运煤船“帆顺999”号货轮驶离9号码头，标志着秦皇岛港老煤炭码头完成使命<ref name=bainian/>。

2008年，“大、小码头”作为“[[秦皇岛港口近代建筑群|秦皇岛港口近代建筑群]]”组成部分被公布为[[河北省文物保护单位|河北省文物保护单位]]。2013年，作为“秦皇岛港口近代建筑群”组成部分被公布为[[第七批全国重点文物保护单位|第七批全国重点文物保护单位]]<ref>{{Cite web |url = http://report.hebei.com.cn/system/2017/09/18/018496555.shtml |title = 秦皇岛港大码头美景如画 |author =  |date = 2017-09-18 |publisher =长城网  |language =  |accessdate =  |quote =  }}</ref>。大、小码头位于秦皇岛港务局南山，2010年代正由秦皇岛港务局使用<ref>{{Cite web |url = http://www.qhdblbwg.com/Hnews.php?P=H0002&N=%D7%CA%C1%CF%BC%EC%CB%F7&id=13&t=1 |title =品味镌刻在砖石上的记忆，寻找一座城市的灵魂  |author =  |date =  |publisher = 秦皇岛市玻璃博物馆 |language =  |accessdate = 2018-02-03 |quote =  }}</ref>。
