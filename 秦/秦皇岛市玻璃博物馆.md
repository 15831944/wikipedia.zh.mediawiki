'''秦皇岛市玻璃博物馆'''，位于[[河北省|河北省]][[秦皇岛市|秦皇岛市]][[海港区|海港区]]文化路44号，是由[[秦皇岛市|秦皇岛市]]人民政府投资建设的以[[玻璃|玻璃]]为主题的公立[[博物馆|博物馆]]，也是[[中华人民共和国|中华人民共和国]]第一家玻璃主题博物馆。是秦皇岛市文化广电新闻出版（版权）局直属单位。

==简介==
秦皇岛市玻璃博物馆于2010年12月成立。整个园区依托始建于1922年的[[耀华玻璃厂旧址|耀华玻璃厂旧址]]建设，是河北省重点文化遗产保护工程。博物馆总占地面积11.25亩，建筑面积2822平方米，由博物馆区、服务区、临展区、辅助设施区组成<ref name=ju>{{Cite web |url = http://www.qhdwhj.gov.cn/front/third.action?id=634&tid=125 |title = 秦皇岛市玻璃博物馆 |author =  |date =2016-06-23  |publisher = 秦皇岛市文化广电新闻出版（版权）局 |language =  |accessdate =  |quote =  }}</ref>。2012年8月6日开馆并免费向公众开放<ref name=gywm>{{cite web|author1=秦皇岛玻璃博物馆|title=关于我们|url=http://www.qhdblbwg.com/Cmain.php?P=A0003&N=关于我们|website=http://www.qhdblbwg.com/|accessdate=2015-05-24}}</ref>。秦皇岛市玻璃博物馆是公益性文化设施<ref name=ju/>。

秦皇岛市玻璃博物馆的展览主题为“天地凝光”。展区分为四个：“古代玻璃及发展”，“中国玻璃工业摇篮”，“中国当代玻璃工业”，“璀璨神奇的玻璃工业”<ref name=ju/><ref>{{Cite web |url =http://www.sohu.com/a/162196140_476428  |title =秦皇岛玻璃博物馆 华彩初凝—中国学生玻璃作品展  |author =  |date = 2017-08-04  |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref><ref name=gywm/>。
