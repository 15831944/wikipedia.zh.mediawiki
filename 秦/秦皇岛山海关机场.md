'''秦皇岛山海关机场'''是一个位于[[中国|中国]][[河北省|河北省]][[秦皇岛市|秦皇岛市]]的军用机场。[[九一三事件|九一三事件]]中[[林彪|林彪]]专机就是从这里起飞的。1985年4月15日起，开通民航航线，成为军民合用机场。2016年3月31日，位于[[昌黎县|昌黎县]]境内的民用[[秦皇岛北戴河机场|秦皇岛北戴河机场]]投入使用，山海关机场的民航班机全部转场至北戴河机场，而山海关机场完全转交给军方，由军民合用机场转型为纯军用机场<ref>{{Cite web |url = http://news.carnoc.com/list/311/311875.html |title = 砥砺30载展翅待高飞——秦皇岛机场通航30周年 |author =  |date =2015-04-17  |publisher = 民航资源网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.hebeiairport.cn/news/8694.html  |title = 风劲扬帆正当时——纪念石家庄机场开航20周年漫谈 |author =  |date =2015-02-25  |publisher =  河北机场管理集团|language =  |accessdate =  |quote =  }}</ref>。
