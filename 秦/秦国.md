'''秦国'''是[[春秋战国|春秋战国]]时期[[诸侯|诸侯]]国，[[嬴姓|嬴姓]]，趙氏<ref>《史記》卷43〈趙世家〉：「趙氏之先，與秦共祖。至中衍，為帝大戊禦。其後世蜚廉有子二人，而命其一子曰惡來，事紂，為周所殺，其後為秦。惡來弟曰季勝，其後為趙。」</ref><ref>《史记·卷五·秦本纪》：蜚廉生恶来...蜚廉复有子曰季胜…造父以善御幸于周缪王，得骥、温骊、骅骝、騄耳之驷，西巡狩，乐而忘归。徐偃王作乱，造父为缪王御，长驱归周，一日千里以救乱。缪王以赵城封造父，造父族由此为赵氏。自蜚廉生季胜已下五世至造父，别居赵。赵衰其后也。恶来革者，蜚廉子也，蚤死。有子曰女防。女防生旁皋，旁皋生太几，太几生大骆，大骆生非子。以造父之宠，皆蒙赵城，姓'''赵氏'''。</ref>。[http://www.zxls.com/book/24history/shiji/b005.htm 史记秦本纪]主要根據爲秦人所自述歷史。据被盜掘後搶救回來的[[清华简|清华简]]《系年》的第三章所載，周初[[三监之乱|三监之乱]]平定后，[[蜚廉|蜚廉]]“东逃于商奄氏。成王伐商奄，杀蜚廉，西迁商奄之民于邾，以御奴之戎，是秦先人”<ref>{{cite web|url=http://epaper.gmw.cn/gmrb/html/2011-09/08/nw.D110000gmrb_20110908_4-11.htm|title=清华简关于秦人始源的重要发现}}</ref><ref>{{cite web|url=http://cul.sohu.com/20151220/n431980812.shtml|title=复旦教授：清华简之“高大上”足以抗衡汲冢竹书}}</ref>。清华简本身为海外黑市购买，相对现场出土文物其真实性先天不足，受部分学术界人士质疑，如姜广辉在《光明日报》的质疑、房德邻在《中国文化报》指出简文使用了先秦不可能产生的双音词“瘇脹”。熊铁基认为秦人伪造家谱，丁山在《中国古代宗教与神话考》一书中则说夏、商、周、秦“四代开国前世系皆宗祝伪托”；金景芳認為「《史記三代世表》明確地譜列堯、舜和夏、殷、周王室的祖先同是以黃帝為初祖。雖然在細節上不能保證沒有缺漏和訛誤，但大體上說是有根據的，可信的。」雍际春、王宏谋说秦部落祖先是伯益，祝军指出秦部落是少昊的后裔，柳明瑞认为嬴秦氏族起源於今山东地區<ref>{{cite web|url=http://www.chinanews.com/qxcz/2011/09-26/3354402.shtml|title=秦皇嬴姓源于山东 莱芜史上称"古嬴"}}</ref>。据[http://www.zxls.com/book/24history/shiji/b005.htm 史记秦本纪]，[[西周|西周]][[周孝王|周孝王]]因秦祖先[[非子|非子]]善养马，因此将其封於秦，作为[[周朝|周朝]]分封国。前770年，[[秦襄公|秦襄公]]护送[[周平王|周平王]][[平王東遷|东迁]]有功，獲封为诸侯，為[[公爵|公爵]]地位，秦正式成為一方諸侯国。周朝给其封地在今甘肃河东地区到[[陕西|陕西]]一带。从前677年起，秦国在雍（今甘肃天水到陇南一带）建都近300年。[[雍城遗址|雍城遗址]]有宫殿区、居住区、[[士大夫|士大夫]]与国人墓葬区和秦公陵园。

秦國與[[西戎|西戎]]、[[義渠|義渠]]之間有通婚、結盟的關係，秦國崛起後，這些勢力皆被併入秦國。[[戰國|战国时期]]，[[秦孝公|秦孝公]]<ref>“车同轨，书同文，行同伦。{{Cite book|title=[[礼记|礼记]][[中庸|中庸]]|last=儒家学者|first=|first2=|authorlink2=|last3=|publisher=|year=战国时期|isbn=|location=|pages=第二十八章|quote=“今天下车同轨，书同文，行同伦”。天下：社会。同：统一。大意：目前社会车辆有统一的道路，书籍有统一的文字，行为有统一的伦理“。}}”</ref>实施[[商鞅变法|商鞅变法]]，为[[秦滅六國|秦滅六國]]奠定基础。秦王[[嬴政|嬴政]]在前221年，統一諸夏，[[秦始皇|秦始皇]]使秦国成为[[中国历史|中国历史]]上第一个大一统[[中央集权|中央集权]][[君主制|君主制]][[帝國|帝國]]，秦帝国较西方的[[罗马帝国|罗马帝国]]还要早兩百年。

== 历史 ==
=== 起源 ===
[http://www.zxls.com/book/24history/shiji/b005.htm 史记秦本纪]認為秦先祖中潏為[[商朝|商朝]]“保西垂”。其子[[蜚廉|蜚廉]]是商朝末年[[紂王|紂王]]的寵臣。據[[清華簡|清華簡]]《系年》的第三章，周初[[三監之亂|三監之亂]]平定後，[[蜚廉|蜚廉]]“東逃於商奄氏。成王伐商奄，殺蜚廉，西遷商奄之民於邾，以御奴之戎，是秦先人。”<ref>{{cite web|url=http://epaper.gmw.cn/gmrb/html/2011-09/08/nw.D110000gmrb_20110908_4-11.htm|title=清華簡關於秦人始源的重要發現}}</ref>。李學勤先生鑑於秦人和東方的[[奄國|奄國]]同姓，同主少皞，又[[蜚廉|蜚廉]]東逃於此，認為秦人可能最初起源於東方，和[[奄國|奄國]]同源。在西周初年因獲罪及曾“保西垂”，被謫戍西方。

[[西周|西周]][[周穆王|周穆王]]之后，秦人地位继续提升，其家族首领大骆与周人也有姻亲关系的申侯结为婚姻。因此据[[司马迁|司马迁]]的说法，大骆之子非子被别封于秦，为周王室的大夫。后来升格为附庸。天子之附庸与诸侯之附庸级别是不同的，其地位相当于一个畿内诸侯。司马迁也补充，秦人成为附庸后，长期对西部戎人作战，实力得到扩充。

根據史記，相傳秦的先祖為[[三皇五帝|三皇五帝]]时期的[[伯益|伯益]]<ref>《史記》卷6〈秦本紀〉：「秦之先，帝顓頊之苗裔孫曰女脩。女脩織，玄鳥隕卵，女脩吞之，生子大業。大業取少典之子，曰女華。女華生大費，與禹平水土。已成，帝錫玄圭。禹受曰：「非予能成，亦大費為輔。」帝舜曰：「咨爾費，贊禹功，其賜爾皁遊。爾後嗣將大出。」乃妻之姚姓之玉女。大費拜受，佐舜調馴鳥獸，鳥獸多馴服，是為柏翳。舜賜姓嬴氏。」</ref>，最早為夏朝諸候，後歸順商湯，滅夏。在殷商[[大戊|大戊]]時，[[中衍|中衍]]成為重要的諸侯<ref>《史記》卷6〈秦本紀〉：「大費生子二人：一曰大廉，實鳥俗氏；二曰若木，實費氏。其玄孫曰費昌，子孫或在中國，或在夷狄。費昌當夏桀之時，去夏歸商，為湯禦，以敗桀於鳴條。大廉玄孫曰孟戲、中衍，鳥身人言。帝太戊聞而卜之使禦，吉，遂致使禦而妻之。自太戊以下，中衍之後，遂世有功，以佐殷國，故嬴姓多顯，遂為諸侯。」</ref>。其子[[蜚廉|蜚廉]]，其孫[[惡來|惡來]]、[[季勝|季勝]]。[[武王克殷|武王克殷]]時，惡來被殺。蜚廉一系在此時分為兩支，惡來之子孫為秦國之祖，其弟季勝為趙氏之祖。趙氏先親附周朝，在周穆王時為周王御者。惡來一系居於犬戎，在周孝王時親附周朝。

秦國部落遷至西戎，有兩種說法，一据[[战国|战国]]末年的《[[孟子|孟子]]》說是在晚商时期，因為[[周武王|周武王]]討伐[[殷商|殷商]]時，[[蜚廉|蜚廉]]支持[[殷商|殷商]]，因此被殺<ref>《孟子·滕文公下》说：“周公相武王诛纣，伐奄三年讨其君，驱飞廉于海隅而戮之。”</ref><ref>《[[史记|史记]]·秦本纪》另载「周武王之伐纣，并杀恶来。是时，蜚廉为纣使北方，还无所报，为坛霍太山，而报，得石棺，铭曰：『帝令处父（飛廉别号），不与殷乱，赐尔石棺以华氏。』死，遂葬于霍太山」”</ref>，其氏族被強迫遷至西戎。另一個传说是，在商朝時，[[戎胥軒|戎胥軒]]時，與郦山之女通婚生下[[中潏|中潏]]，遷至西戎。

[[蜚廉|蜚廉]]生下[[惡來|惡來]]、[[季勝|季勝]]二子。惡來被周武王所殺，其後為[[大駱|大駱]]<ref>《史記》卷6〈秦本紀〉：「惡來革者，蜚廉子也，蚤死。子曰女防。女防生旁皋，旁皋生太幾，太幾生大駱，大駱生非子。」</ref>，居犬丘，以畜牧為生。大駱生成與非子，並與周朝申侯通婚。周孝王封[[非子|非子]]，成為秦國先祖。季勝一系則成為趙國先祖。

{{Fact|現代歷史學家[[蒙文通|蒙文通]]假設秦人可能源自[[羌族|羌族]]，另一派則支持秦人源自[[東夷|東夷]]的假說。秦人先祖相傳為傳說人物[[伯益|伯益]]（一作“柏翳”）。據西漢時期司馬遷記載，秦人早期居住地為費，即今山東[[魚台|魚台]]。目前存在一種假設（主要以司馬遷記載的傳說為主）是直至戰國時期，與秦王族同姓的諸侯國絕大多數大致仍然分佈在秦山以南至江淮地區，而只有奄因為遭到周人的驅逐至江南今江蘇常州一帶，但也很難認為這些就是這些諸侯國本有姓氏，而當時也可能沒有姓氏的概念，也有可能和周王族同姓的諸侯國一樣自周武王克商後遍及在各地。也有假設認為秦人雖位於西方，理論上仍然保有東夷人崇拜鳥的習俗，不過在生活習慣上卻因適應當地自然環境而更像是戎狄。|time=2015-10-26T08:43:13+00:00}}

秦最初领地在当时属于[[周朝|周朝]]的边缘地区。[[周孝王|周孝王]]封[[非子|非子]]于秦邑（今甘肃省[[甘肃省|甘肃省]][[天水市|天水市]][[清水县|清水县]]东北）为附庸，使复嬴氏祀，号曰“秦嬴”。《史记·秦本纪》记载西戎伐[[周幽王|周幽王]]时，[[秦襄公|秦襄公]]曾“将兵救周”，成为关中平原西部地区的一支势力。

===春秋时期===
[[春秋时代|春秋时代]]早期[[秦襄公|秦襄公]]被[[周平王|周平王]]封为秦[[伯|伯]]，又被赐封[[岐山|岐山]]以西之地，秦正式成为[[诸侯国|诸侯国]]。
====秦晋关系====
到了[[秦穆公|秦穆公]]时代，致力于开拓[[西戎|西戎]]地方，亦同时参与中原争霸，成为与[[晋国|晋国]]、[[楚国|楚国]]的霸主级別强国。晋献公时，秦晋两国维持着良好关系，而秦亦致力巩固及开拓西陲。公元前651年，[[晋献公|晋献公]]死，众晋公子争位，晋国陷入动乱，而秦穆公亦开始把战略目标东移。晋公子夷吾流亡国外，并许诺秦穆公若以秦助继晋侯位，则以黄河以西的土地报答秦国。及秦军送夷吾回晋，立为[[晋惠公|晋惠公]]，而秦却没有得河西土地。秦穆公亦没有追究。过了数年晋国旱灾，晋求粮于秦，穆公卖粮食及农具与晋。翌年，秦国大旱，穆公欲购粮于晋，而惠公不许。秦穆公大怒伐晋，战于韩，俘虏了晋惠公。因晋惠公的姐姐是穆公的夫人，才得以被放回晋国。以后数年秦晋之间皆有征战。后来秦穆公听说晋献公有一公子重耳流亡于楚国，遂请重耳到秦，并举兵助其回晋国。时晋惠公已死，其子[[晋怀公|晋怀公]]刚立不足一年，被公子重耳和秦军攻杀。公元前636年，重耳继位为[[晋文公|晋文公]]，在位九年，秦晋修好。

公元前628年，晋文公去世，[[晋襄公|晋襄公]]立。这一年秦穆公秘密攻打[[郑国|郑国]]，秦军于行军途中被贩牛的郑国商人[[弦高|弦高]]识破，失去了突袭的先机，遂收兵回秦。当时晋襄公探得秦军回军的路径，两军大战于[[殽|殽]]，秦军覆没。消息传回秦国，穆公懊悔悲恸，秦人大愤。前625年，秦穆公发兵伐晋复仇，又败于彭衙。过了一年，秦穆公亲自率兵再次讨伐晋国，渡过黄河以后，将渡船全部焚毁，表示誓死克敌的决心。晋军拒不出战。穆公在当年的殽战场埋葬尸骨及树立标记，然后班师回国。

及后，因秦国东进的路被晋国堵塞，秦国的战略转回传统的经营西陲。秦康公，桓公，景公时期，秦晋一再发生战争，几经拉锯。秦国国势亦随着战争被拖至逐渐衰弱。

====弭兵之会====
公元前546年，秦景公在位期间，[[宋国|宋国]]、[[晋国|晋国]]、[[楚国|楚国]]、[[齐国|齐国]]、秦国等14国举行[[弭兵之会|弭兵之会]]，签订的盟约是：“晋、楚之从，交相见也。”（奉晋、楚两国为共同霸主，除齐、秦外，各国须向晋、楚同样纳贡，晋的仆从国要朝贡[[楚国|楚国]]，而楚的仆从国要朝贡[[晋国|晋国]]，[[齐国|齐国]]作为[[晋国|晋国]]的盟国，不朝拜[[楚国|楚国]]，秦国为[[楚国|楚国]]的盟国，亦不朝于[[晋国|晋国]]），[[秦哀公|秦哀公]]在位期间，申包胥曾前往秦庭痛哭求师助[[楚昭王|楚昭王]]复国，留下“[[申包胥|申包胥]]哭秦庭”这一典故。

===战国时期===
[[File:ZH-秦国地图260BCE.jpg|thumb]]

[[战国|战国]]初期[[魏国_(战国)|魏国]]连年进攻秦国，夺取[[河西|河西]]之地，設立[[西河郡_(魏国)|西河郡]](見[[河西之战_(战国)|河西之戰]])，秦国被迫退守[[洛水|洛水]]以西，走入下坡。这形势一直到[[商鞅变法|商鞅变法]]才开始改变。

公元前356年，[[秦孝公|秦孝公]]任命商鞅为左庶长，商鞅开始第一次变法。主要内容为：第一，“令民为什伍”，实行连坐法；第二，重农抑商，奖励耕织；第三奖励军功，按功受爵，贵族无军功不再受爵；第四，“[[焚書坑儒|燔诗书而明法令]]”<ref>《[[韩非子|韩非子]]·和氏篇》：[[商君|商君]]教[[秦孝公|秦孝公]]以连什伍，设告坐之过，燔诗书而明法令。</ref>。前350年商鞅又进行第二次变法，主要内容有：统一秦国的度量衡；废分封，行县制；“为田开阡陌封疆”，废除井田制，以法律形式确立土地私有制度。商鞅变法后，秦国逐步强盛起来，为后来秦统一六国奠定了基础。但是商鞅实行的严刑峻法和文化高压政策，对后来的[[秦朝|秦朝]]有消极影响。[[商鞅变法|商鞅变法]]遭到了旧贵族的强烈反对。秦孝公死后，商鞅被施以[[车裂|车裂]]之刑。但商鞅变法的措施在秦国继续下去，他主张的[[法家|法家]]思想，也成为秦国占统治地位的政治思想。

公元前3世纪以后，东方各国衰落下去，秦国无敌于天下。[[秦昭襄王|秦昭襄王]]時，其母[[宣太后|宣太后]]與義渠通婚，使詐殺義渠戎王於[[甘泉县|甘泉]]，將[[義渠|義渠]]併入秦國<ref>《史記》〈匈奴列傳〉：「秦昭王時，義渠戎王與宣太后亂，有二子。宣太后詐而殺義渠戎王於甘泉，遂起兵伐殘義渠。於是秦有隴西、北地、上郡，筑長城以拒胡。」</ref>。公元前279年的郢之战和前278年的鄢之战，[[白起|白起]]统帅的秦军攻占当时政治核心在[[南阳|南阳]]和[[丹阳|丹阳]]一带的[[楚国|楚国]]，攻陷楚国首都，所得領土改置[[南阳郡|南阳郡]]、[[南郡|南郡]]、[[临江郡|临江郡]]（江夏郡）、[[黔中郡|黔中郡]]，楚遷都[[寿郢|寿郢]]；至此南方大国[[楚国|楚国]]因为丧失大片疆土和人口而走向衰落，公元前260年[[白起|白起]]在[[长平之战|长平之战]]重创[[赵国|赵国]]，秦国坑杀赵国降卒四十餘萬，这是战国时期兼并战争中规模最大、杀伤最多的一次战役，秦国威震天下。公元前256年秦国灭[[西周國|西周國]]，[[周王朝|周王朝]]统治的最后象征被消灭。

前247年[[嬴政|嬴政]]登基，开始征服六国。从前230年秦灭[[韩国_(诸侯)|韓国]]起，到前221年秦灭[[田齐|齐国]]，[[秦灭六国之战|统一中国]]。中國歷史結束諸侯[[封建時代|封建時代]]，進入[[中央集權|中央集權]][[君主專制|君主專制]]時代。

== 社会经济 ==
秦国社会经济仍以[[农业|农业]]为主。
*[[春秋时代|春秋时代]]晚期，出现了[[铸铁|铸铁]][[农具|农具]]。
*[[戰國_(中國)|战国时代]]，修建了[[郑国渠|郑国渠]]、[[都江堰|都江堰]]等水利工程，进一步促使农业生产发展。
*手工业以冶铜和制陶最为发达。发明了铬盐氧化处理兵器的新工艺；建筑材料颇具特色，[[瓦当|瓦当]]更是精美的艺术品。

*風俗：有戎狄之風，周幽王被殺之後，秦地人民長期與戎人交流，使其民風有戎狄之氣。

*重視小兒的健康與教育，有專門的小兒醫生。

== 葬仪 ==

秦国国君陵园在雍城陵区发现13座，芷阳陵区发现4座，[[士大夫|士大夫]]与国人墓葬已发掘至近千座，国君称王前使用“中”字形诸侯级墓制，称王后使用“亚”字形王级墓制，广泛使用[[人祭|人殉]]（秦獻公廢除），[[殉葬|殉葬]]物品丰富，规模宏大。其中秦公一号大墓（及[[秦景公|秦景公]]墓地）殉葬186人，为中国有史以来发掘坟墓中殉葬人数最多的一座。

== 人口 ==
西元前360年，約130萬~160萬

== 军事 ==
秦国军队自[[商鞅变法|商鞅变法]]实行奖励军功政策后愈战愈勇。武器装备不断改进。军队数量多时达到“带甲之士百万”，出现了[[尉缭|尉缭]]、[[白起|白起]]、[[王翦|王翦]]、[[蒙恬|蒙恬]]等著名军事家和将领。<br>
