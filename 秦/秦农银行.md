'''秦农银行'''全称为'''陕西秦农农村商业银行股份有限公司'''，是[[陕西省|陕西省]][[西安市|西安市]]的一家农村商业银行<ref>{{cite web |url=http://finance.sina.com.cn/money/bank/20141119/011920854714.shtml |title=陕西秦农银行筹建细节披露 拟定注册资本80亿 |author=乔加伟 |date=2014-11-19 |website=finance.sina.com.cn |publisher=21世纪经济报道 |access-date=2018-01-12 |quote=}}</ref>，注册资本80亿元，由[[新城区_(西安市)|新城区]]、[[碑林区|碑林区]]、[[莲湖区|莲湖区]]、[[雁塔区|雁塔区]]、[[灞桥区|灞桥区]]、[[未央区|未央区]]农村信用合作联社合并重组建成，是陕西省继[[长安银行|长安银行]]之后成立的又一家商业银行。2015年全行资产总额777亿元，负债总额675亿元，资本规模居西北金融机构之首，排全国农村商业银行第五位，共有235个营业网点，是西安市营业网点最多的银行机构<ref>{{cite web |url=http://news.cnwest.com/content/2015-05/29/content_12630492.htm|title=秦农银行昨日盛大开业 成全国第六家省级农商行 |author1=杨少卿 |author2=张问之 |date=2015-05-29 |website=china.huanqiu.com |publisher=陕西传媒网 |access-date=2018-01-12 |quote=}}</ref>。2017年，秦农银行与西安一起涉嫌非法吸收公众存款的案件有较大关联，与其合作的西安一得贸易有限公司对推出的黄金理财产品隐藏风险并夸大安全，涉案金额达1.2亿元人民币<ref>{{cite web |url=http://www.chinatimes.cc/article/67500.html |title=西安一得黄金理财兑付危机调查：秦农银行的关键角色 |author=刘敏 |date=2017-05-19 |website=www.chinatimes.cc |publisher=华夏时报 |access-date=2018-01-12 |quote=}}</ref>。
