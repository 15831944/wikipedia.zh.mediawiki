'''秦孝文王'''（{{bd|前303年||前250年||Q}}），即位前称'''安国君'''，《[[史记索隐|史记索隐]]》记载名'''柱'''，《[[广弘明集|广弘明集]]》引《陶公年纪》记载名'''式'''，[[战国|战国]]时期[[秦国|秦国]]君主。[[秦昭襄王|秦昭襄王]]之子，在位仅三天。

== 生平 ==
秦昭襄王在位时，秦孝文王被封为安国君。

[[悼太子_(秦国)|秦悼太子]]于前267年死于[[魏国_(战国)|魏国]]后，安国君于前265年被立为太子。<ref>《史记·卷五·秦本纪》：（秦昭襄王）四十年，悼太子死魏…四十二年，安国君为太子。</ref>

安国君与[[正妻|正妻]][[华阳太后|华阳夫人]]一直没有子嗣，[[庶出子女|庶子]][[子傒|子傒]]已形成势力，但在[[吕不韦|吕不韦]]的活动下，华阳夫人成功说服安国君立在[[赵国|赵国]]担任[[質子_(東亞政治)|质子]]的[[秦庄襄王|异人]]{{NoteTag|1=秦庄襄王原名异人，后改名子楚。}}担任继承人，为此安国君为华阳夫人刻下玉符作为证物。二人随后命吕不韦担任异人的老师，又委托他带给异人很多礼物。后在吕不韦的帮助下，异人成功返回秦国担任太子。<ref>《史记·卷八十五·吕不韦列传》：华阳夫人以为然，承太子间，从容言子楚质于赵者绝贤，来往者皆称誉之。乃因涕泣曰：“妾幸得充后宫，不幸无子，原得子楚立以为适嗣，以讬妾身。”安国君许之，乃与夫人刻玉符，约以为适嗣。安国君及夫人因厚馈遗子楚，而请吕不韦傅之，子楚以此名誉益盛于诸侯…子楚与吕不韦谋…遂以得归。</ref>

前251年，秦昭襄王去世，其子53歲的秦孝文王继位。继位后秦孝文王尊生母[[唐太后|唐八子]]为[[唐太后|唐太后]]，将其与秦昭襄王合葬。[[韩桓惠王|韩桓惠王]]亲自穿着孝服前来吊唁，其余诸侯各国也派将相前来慰问。<ref>《史记·卷五·秦本纪》：（秦昭襄王）五十六年秋，昭襄王卒，子孝文王立。尊唐八子为唐太后，而合其葬于先王。韩王衰绖入吊祠，诸侯皆使其将相来吊祠，视丧事。</ref>秦孝文王同时立华阳夫人为王后，子楚为太子，赵国也将[[赵姬|赵姬]]及其子[[秦始皇|嬴政]]送还秦国。<ref>《史记·卷八十五·吕不韦列传》：秦昭王五十六年，薨，太子安国君立为王，华阳夫人为王后，子楚为太子。赵亦奉子楚夫人及子政归秦。</ref>

前250年，秦孝文王下令大赦罪人，按功表彰先王功臣，优待宗族亲属，拆除王家园囿。

秦孝文王于前250年11月12日行改元继位之礼，三日之后暴崩，享年54歲，葬于[[秦东陵|寿陵]]（今[[陕西省|陕西省]][[西安市|西安市]][[临潼区|临潼区]]东北），其子[[秦庄襄王|秦庄襄王]]继位。<ref>《史记·卷五·秦本纪》：孝文王元年，赦罪人，修先王功臣，褒厚亲戚，弛苑囿。孝文王除丧，十月己亥即位，三日辛丑卒，子庄襄王立。</ref><ref>《史记·卷六·秦始皇本纪》：孝文王享国一年。葬寿陵。生庄襄王。</ref>

《[[汉书|汉书]]·五行志》记载秦孝文王出游至[[朐衍|朐衍]]（今[[宁夏回族自治区|宁夏回族自治区]][[盐池县|盐池县]]）期间，有人献上五足之牛。<ref>《汉书·卷二十七下之上·五行志》：秦孝文王五年（应为元年），斿朐衍，有献五足牛者。</ref>

== 逸事 ==
有一次异人趁安国君空闲时进言道：“父亲也曾羁留在赵国，赵国的豪杰之士知道父亲大名的不在少数。如今父亲返回秦国，他们都惦记着您，可是父亲却连一个使臣都未曾派遣去抚慰他们。孩儿担心他们会心生怨恨，希望父亲将边境城门迟开早闭，防患于未然。”安国君认为异人说的话极有道理，为他的奇谋感到惊讶。<ref>《战国策·卷七·秦策五·吕不韦贾于邯郸》：间曰：“陛下尝轫车于赵矣，赵之豪桀，得知名者不少。今大王反国，皆西面而望。大王无一介之使以存之，臣恐其皆有怨心，使边境早闭晚开。”王以为然，奇其计。</ref>

== 文学形象 ==
长篇历史小说《[[东周列国志|东周列国志]]》中，秦孝文王于第九十九回《[[白起|武安君]]含冤死[[杜邮|杜邮]] 吕不韦巧计归异人》中登场，其事迹与《[[史记|史记]]》记载基本相同。在第一百零一回《秦王灭周迁[[九鼎|九鼎]] [[廉颇|廉颇]]败燕斩二将》中，秦孝文王继位后立赵女为王后，在为其父除丧三日后，与群臣宴会散席回宫后去世。国人都怀疑是吕不韦准备让子楚提前登基，买通秦孝文王左右鸩杀秦王，但都畏惧吕不韦的气势，不敢多言。<ref>见《东周列国志·第一百零一回·秦王灭周迁九鼎 廉颇败燕斩二将》。</ref>
