'''秦皇岛海洋食品有限公司'''是一家位于[[中华人民共和国|中华人民共和国]][[河北省|河北省]][[秦皇岛市|秦皇岛市]][[海港区_(秦皇岛市)|海港区]]的食品生产企业，始建于1960年。公司原名'''中国人民解放军第4003工厂'''，原属[[中国人民解放军海军后勤部|中国人民解放军海军后勤部]]。2001年，中国人民解放军第4003工厂移交[[中国蓝星（集团）股份有限公司|中国蓝星（集团）股份有限公司]]，改名'''秦皇岛海洋食品总厂'''。2015年底，秦皇岛海洋食品总厂被转让，成为[[民营企业|民营企业]]。

== 历史 ==
秦皇岛海洋食品有限公司原名中国人民解放军第4003工厂，1960年成立，1964年投入生产，<ref>{{cite book |author1=刘纯华 |title=大海的梦幻 |date=1990-10 |publisher=海洋出版社 |location=北京 |isbn=7-5027-1076-0 |page=115}}</ref>隶属[[中国人民解放军海军后勤部|中国人民解放军海军后勤部]]，曾是[[中国人民解放军|中国人民解放军]]唯一兼具研发、生产、销售的大型综合食品加工企业。[[朱德|朱德]]、[[杨尚昆|杨尚昆]]、[[洪学智|洪学智]]、[[刘华清|刘华清]]等政军领导曾视察第4003工厂。杨尚昆曾为工厂题词“生产优质食品，满足部队需要”。<ref name="qhdhysp" />

2001年，4003工厂移交[[中国蓝星（集团）股份有限公司|中国蓝星（集团）股份有限公司]]，改名秦皇岛海洋食品总厂。2010年，秦皇岛海洋食品总厂在秦皇岛北部工业区投资兴建新厂区。<ref name="qhdhysp" /><ref>{{cite web |title=秦皇岛海洋食品总厂 |url=http://www.qhdlw.com/zpxxs.asp?lsh=3220 |website=秦皇岛劳务派遣网 |accessdate=2018-09-27}}</ref>2015年11月，中国蓝星将秦皇岛海洋食品总厂整体产权及1535.163013万元债权在[[北京产权交易所|北京产权交易所]]挂牌交易，挂牌价2406.893013万元人民币。<ref>{{cite web |title=秦皇岛海洋食品公司转让 |url=http://www.360ask.org/zhao/bjyangqi/2015-12-02/892.html |website=找项目网 |accessdate=2018-09-27 |date=2015-12-02}}</ref><ref>{{cite news |title=北京产权交易所挂牌项目信息 |url=http://finance.sina.com.cn/stock/t/20151113/040423756062.shtml |accessdate=2018-09-27 |work=证券时报 |date=2015-11-13}}</ref>同年12月转让成交。<ref>{{cite news |title=2015年12月国有产权成交项目 |url=http://www.cbex.com.cn/article/ggyts/cjgs/201601/20160100057785.shtml |accessdate=2018-09-27 |work=北交所网站 |date=2016-01-04 |archive-url=https://web.archive.org/web/20170125224413/http://www.cbex.com.cn/article//ggyts/cjgs/201601/20160100057785.shtml |archive-date=2017-01-25 |dead-url=no }}</ref>2016年1月，公司完成工商注册变更，成为[[北京市宇邦天业投资发展有限公司|北京市宇邦天业投资发展有限公司]]全资子公司。<ref>{{cite web |title=秦皇岛海洋食品有限公司 |url=https://www.tianyancha.com/company/2352494240 |website=天眼查 |accessdate=2018-09-27}}</ref>

== 产品 ==
秦皇岛海洋食品有限公司产品包括各种[[罐头|罐头]]、[[压缩干粮|压缩干粮]]、[[自热食品|自热食品]]、[[即食食品|即食食品]]、汤制品等六个大类，品种超过百种，年产能超过两万吨。公司累计为部队生产的压缩干粮、罐头超过36.5万吨，累计出口、内销量达15.8万吨。工厂为[[唐山大地震|唐山大地震]]救灾、[[对越自卫反击战|对越自卫反击战]]、[[1998年中国水灾|1998年中国水灾]]救灾、[[汶川地震|汶川地震]]救灾、驻外维和与工程部队生产了大量食品。<ref name="qhdhysp" />

公司拥有注册商标“[[北戴河|北戴河]]”、“900”、“4003”、“滨城春”等。<ref name="qhdhysp">{{cite web |title=企业简介 |url=http://www.qhdhysp.com/index.php/list/index/g/c/id/50.html |website=秦皇岛海洋食品有限公司 |accessdate=2018-09-27 |archive-url=https://web.archive.org/web/20180826121212/http://qhdhysp.com/index.php/list/index/g/c/id/50.html |archive-date=2018-08-26 |dead-url=no }}</ref>
