'''秦明'''（{{bd|1981年|1月}}），生于[[安徽省|安徽省]][[铜陵市|铜陵市]]，[[中华人民共和国|中国]][[法医|法医]]、作家。担任[[安徽省公安厅|安徽省公安厅]]物证鉴定中心法医病理损伤检验科科长，[[三级警督|专业技术三级警督]][[中华人民共和国人民警察警衔|警衔]]、[[副主任法医师|副主任法医师]][[职称|职称]]。2012年，秦明在网络上发表第一部犯罪推理小说《鬼手佛心：我的那些案子》<ref name="a1">{{Cite web |url=http://tj.ifeng.com/a/20161031/5104476_0.shtml |title=秦明：要做一个能写很久的作者 |accessdate=2017-05-09 |author= |date=2016-10-31 |work=责任编辑：吴兴 |publisher=凤凰网，来源：每日新报|language=Zh-hans }}</ref>。2015年<ref>{{Cite web |url=http://zj.people.com.cn/n/2015/1102/c228592-26996547-4.html |title=首届网络文学双年奖颁奖典礼宁波举行 网络小说《将夜》获金奖 |accessdate=2017-05-09 |author= |date=2015-11-02 |work= |publisher=人民网-浙江频道|language=Zh-hans }}</ref>，小说《第十一根手指》获第一届网络文学双年奖优秀奖<ref name="a1"/>。

作品有《[[法医秦明_(小说)|法医秦明]]》系列小说（《尸语者》、《无声的证词》、《第十一根手指》、《清道夫》、《幸存者》）、《逝者证言——跟着法医去探案》<ref name="a1"/>。
