'''秦英林'''(1965年-)，[[河南|河南]][[南阳市|南阳市]][[内乡县|内乡县]]人，畜牧师、企业家、亿万富豪，大学学历，中共党员，现任[[牧原食品|牧原食品]]股份有限公司董事长兼总经理。在2015年福布斯中国富豪榜单中，河南首富秦英林夫妇以171.5亿元人民币位列67位。在2015胡润百富榜上，秦英林以205亿元位列第98名。<ref>{{cite web|url=http://news.dahe.cn/2015/10-27/105875113.html | language= | title=2015福布斯中国富豪榜： 河南首富还是秦英林 |publisher=大河网-大河报 |date=2015年10月27日 |accessdate= }}</ref>

== 简历 ==

秦英林24岁时毕业于[[河南农业大学|河南农业大学]][[畜牧|畜牧]]专业，他被分配到南阳食品公司工作。1992年刚参加工作3年的秦英林下决心，携妻子钱瑛（毕业于[[郑州牧业工程高等专科学校|郑州牧业工程高等专科学校]][[兽医|兽医]]专业）辞职回老家创业从事养[[猪|猪]]产业。截至2013年，牧原食品股份有限公司有2个全资子公司和1个参股公司，年出栏生猪超100万头，成为国家级原种猪场、国家生猪活体储备基地、出口猪肉备案养殖场和国家重点龙头企业。2014年1月28日，[[牧原股份|牧原股份]]（002714.SZ）在[[深圳证券交易所|深圳证券交易所]]正式挂牌上市。<ref>{{cite web|url=http://finance.ifeng.com/a/20140721/12759795_0.shtml | language= | title=秦英林家族：中原养猪大王 世界银行资金源头 |publisher=理财周报 |date=2014年7月21日 |accessdate= }}</ref>

== 头衔 ==
*中国策划学院客座教授
*中国创业致富促进会常务理事
*2002年当选为河南省[[人大代表|人大代表]]
*2007年聘为河南农业大学兼职教授

== 荣誉 ==
*1997年 第2届中国十大杰出青年农民
*1999年 第3届中国青年五四奖章标兵
*2006年，被授予“全国畜牧业先进工作者”称号
