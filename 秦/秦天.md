'''秦天'''（{{bd|1957年|||catIdx=Q}}），[[湖北|湖北]][[红安|红安]]人，[[武警中将|武警中将]]。

==生平==
1973年入伍，毕业于[[中国人民解放军国防大学联合作战学院|石家庄高级步兵学校]]，历任战士、班长、排长、连长、作训股长、作训科副科长、副团长、团长、副师长、[[中国人民解放军总装备部|中国人民解放军总装备部]]防化研究院副院长等。2009年，任[[中国人民解放军国防大学|国防大学]]科研部副部长，2012年，任[[中国人民解放军国防大学|国防大学]]科研部部长。2015年7月，任[[中国人民解放军军事科学院|中国人民解放军军事科学院]]副院长<ref>{{cite news|title=军科院领导调整 名将之后秦天任副院长|url=http://china.caixin.com/2015-07-08/100826861.html|newspaper=财新网|date=2015-07-08}}</ref>。2015年7月，秦天时任[[中国人民解放军军事科学院|中国人民解放军军事科学院]]副院长，其胞兄[[秦卫江|秦卫江]]时任[[中国人民解放军南京军区|南京军区]]副司令员，两人也是当时军界现役将领中唯一同时担任副大军区级领导职务的“兄弟档”<ref>{{cite news|title=军方将领调整现“兄弟档” 系秦基伟之子|url=http://news.china.com.cn/live/2015-07/27/content_33721700.htm|newspaper=中国网|date=2015-07-27}}</ref>。

2015年12月，任[[中国人民武装警察部队|中国人民武装警察部队]]参谋长<ref>{{cite news|title=武警部队参谋长易人：名将之后秦天接棒刘振立|url=http://www.thepaper.cn/newsDetail_forward_1415707|date=2016-01-01|newspaper=澎湃新闻}}</ref>。

2016年7月，晋升[[中国人民武装警察部队警衔|中国人民武装警察部队中将]]警衔<ref>{{cite news|title=武警部队举行晋衔仪式，秦天晋升武警中将警衔|url=http://www.thepaper.cn/newsDetail_forward_1506316|accessdate=2016-07-29|newspaper=澎湃新闻}}</ref>。

2017年9月，转任副司令员<ref>{{cite news|title=秦天任武警部队副司令员 郑家概升任参谋长| url=http://m.china.caixin.com/m/2017-09-24/101149373.html|date=2017-09-24|newspaper=财新网}}</ref><ref>{{Cite web |url = http://news.ifeng.com/a/20170924/52143680_0.shtml |title = 26个月来，武警部队已三换参谋长 |author =  |date = 2017-09-24 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。
