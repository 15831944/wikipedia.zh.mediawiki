'''秦生祥'''（{{bd|1957年|2月||}}），[[湖北|湖北]][[监利|监利]][[龚场镇|龚场镇]]人，[[中国人民解放军海军中将|中国人民解放军海军中将]]<ref name=zgrd/>。现任[[中国人民解放军海军|中国人民解放军海军]][[政治委员|政治委员]]。中共第十九届中央委员。

==生平==
1977年入伍，后加入[[中国共产党|中国共产党]]。[[中共中央党校|中共中央党校]]函授学院经济管理专业毕业<ref name=zgrd/>。长期在解放军总部机关工作，历任[[中央军委政治工作部组织局|中国人民解放军总政治部组织部]]副部长、部长等职。2012年10月，接替升任[[中国人民解放军副总参谋长|中国人民解放军副总参谋长]]的[[王冠中|王冠中]]任[[中央军委办公厅|中央军委办公厅]]主任。2016年，在[[深化国防和军队改革|深化国防和军队改革]]中，出任[[中央军委办公厅|中央军委办公厅]]主任兼[[中央军委改革和编制办公室|中央军委改革和编制办公室]]主任<ref>{{cite web|accessdate=2016-08-28|url=http://www.thepaper.cn/newsDetail_forward_1520531|title=军委办公厅主任秦生祥兼任军委改革和编制办公室主任|publisher=澎湃新闻}}</ref>。2017年1月，明确为[[正战区级|正战区级]]<ref>{{Cite web|url=http://news.163.com/17/0225/10/CE44GI6A000187VE.html|title=官媒披露:军委办公厅主任秦生祥中将晋升正战区级|accessdate=2017-02-25|author=|date=2017-02-25|publisher=网易}}</ref>。2017年8月，任[[中国人民解放军海军|中国人民解放军海军]][[政治委員|政治委员]]<ref>{{Cite web |url = http://news.ifeng.com/a/20170913/51981288_0.shtml |title = 军委办公厅主任秦生祥“空降”海军任政委 |author =  |date = 2017-09-13 |publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

2007年，晋升[[中国人民解放军少将|少将]]军衔。2015年7月晋升[[中国人民解放军中将|中将]]军衔<ref >{{cite web|url= http://news.163.com/15/0722/08/AV46DVDB0001124J.html |language= |title=中央军委办公厅主任秦生祥由少将晋升为中将 |publisher = 澎湃新闻 |date=2015-07-22 |accessdate= }}</ref>。

2013年，当选[[第十二届全国人民代表大会|第十二届全国人民代表大会]]解放军代表<ref name=zgrd>{{cite news |title=全国人大代表信息-秦生祥 |url = http://www.npc.gov.cn/delegate/viewDelegate.action?dbid=122927 |accessdate=2013-04-05 |publisher =中国人大网}}</ref>。
