'''秦宜智'''（{{bd|1965年|12月||catIdx=Q秦}}），[[河南省|河南]][[新乡市|新乡]]人。1988年7月参加工作，1985年1月加入[[中国共产党|中国共产党]]。[[清华大学|清华大学]]工程物理系核反应堆专业及社会科学系思想政治教育专业毕业，[[重庆大学|重庆大学]]热力工程系热能工程专业硕士研究生学历。曾任[[攀枝花市|攀枝花市]]长、中共[[内江市|内江市]]委书记、中共[[拉萨市|拉萨市]]委书记、[[西藏自治区人民政府|西藏自治区人民政府]]常务副主席、[[中国共产主义青年团中央委员会|共青团中央]]第一书记。2012年11月当选[[中国共产党第十八届中央委员会候补委员列表|中国共产党第十八届中央委员会候补委员]]<ref>{{cite web|url=http://news.xinhuanet.com/18cpcnc/2012-11/14/c_113690803.htm|title=中国共产党第十八届中央委员会候补委员名单|accessdate=2012-11-15|author=|date=2012-11-14|publisher=[[新华网|新华网]]}}</ref>，现任[[国家市场监督管理总局|国家市场监督管理总局]]副局长、党组成员（正部长级）。

== 四川 ==
历任攀钢热电厂车间副主任、副科长、厂长助理、副厂长、生产部副主任，攀钢（集团）公司副总经济师、成都无缝钢管公司总经理，2001年12月转入政界，历任中共[[攀枝花市|攀枝花市]]委副书记、代市长、市长，[[内江市|内江市]]委书记、市人大常委会主任。

== 西藏 ==
2005年5月，40岁的秦宜智调任[[西藏自治区人民政府|西藏自治区人民政府]]主席助理，2006年7月升任西藏自治区人民政府副主席，9月兼任[[拉萨市|拉萨市]]委书记。2008年9月任[[中国共产党西藏自治区委员会|中共西藏自治区党委]]常委，2011年11月任西藏自治区人民政府常务副主席，2012年11月在[[中国共产党第十八次全国代表大会|中共十八大]]上当选[[中国共产党中央委员会|中国共产党中央委员会]][[中国共产党第十八届中央委员会候补委员列表|候补委员]]。

== 团中央第一书记 ==
2013年3月19日，秦宜智任[[中国共产主义青年团|中国共产主义青年团]]中央书记处第一书记<ref>{{Cite web|url=http://news.ifeng.com/mainland/detail_2013_03/19/23264879_0.shtml|title=秦宜智任共青团中央书记处第一书记|accessdate=2017-09-15|author=|date=2013-03-19|work=|publisher=凤凰网}}</ref>，晋升正部级，成为[[改革开放|改革开放]]以来第九任团中央第一书记。

由于中共中央总书记[[习近平|习近平]]批评共青团组织脱离群众，“高位截瘫”，中央巡视组批评共青团存在“机关化、行政化、贵族化、娱乐化”等问题。2016年8月，[[中共中央办公厅|中共中央办公厅]]发布《共青团中央改革方案》，开始对共青团进行全面改革，改革试点首先在[[上海市|上海]]和[[重庆市|重庆]]展开。秦宜智表示：共青团改革最核心的一点就是要解决共青团脱离青年的问题，“比如团代会、委员会、常委会等共青团的领导机构，要有更多基层代表，使我们的领导机构能够更接地气。比如在部门设置上，我们新设立了社会联络部，这个部门的主要职能就是联系一些新兴领域的青年、一些新的社会组织。”<ref>{{cite web|url=http://www.zaobao.com.sg/news/china/story20170518-761300|title=中央第一书记秦宜智：共青团与广大青年脱节|accessdate=2017-05-18|date=2017-05-18|publisher=联合早报}}</ref>

2017年6月27日至6月28日，中共中央直属机关党代表会议选举产生109名中直机关出席[[中国共产党第十九次全国代表大会代表名单|中共十九大代表]]。7月6日十九大代表名单公布后，作为共青团第一书记的秦宜智并不在列，当选的是时任共青团中央书记处常务书记的[[贺军科|贺军科]]，这一反常事件引发外界的关注和议论<ref>{{cite web|url=http://district.ce.cn/newarea/sddy/201707/06/t20170706_24059076.shtml|title=中直机关选举产生109名出席党的十九大代表(名单)|accessdate=2017-07-06|date=2017-07-06|publisher=中国经济网}}</ref><ref>{{cite web|url=http://news.dwnews.com/china/news/2017-09-05/60011007.html|title=“团派”有变？ 秦宜智蹊跷落选十九大代表|accessdate=2017-09-05|date=2017-09-05|publisher=多维新闻网}}</ref>，也预示着秦宜智的仕途即将生变。

== 国家质检总局 ==
2017年9月20日，在团中央工作四年半之后，52岁的秦宜智调任[[中华人民共和国国家质量监督检验检疫总局|国家质量监督检验检疫总局]]副局长（正部长级），虽然保留了正部级，但是却排在三名副部级副局长之后<ref>{{Cite web|url=http://news.xinhuanet.com/politics/2017-09/20/c_1121694584.htm|title=国务院任免国家工作人员：秦宜智任质检总局副局长|accessdate=2017-09-20|author=|date=2017-09-20|work=|publisher=新华网}}</ref>。2018年4月，担任[[国家市场监督管理总局|国家市场监督管理总局]]副局长，排名依然在副部长级副局长之后<ref>{{cite web|author1=岳怀让|title=国家市场监督管理总局领导层出炉，具体工作分工已公布|url=http://www.thepaper.cn/newsDetail_forward_2056635|website=澎湃新闻|accessdate=2018-04-04|date=2018-04-03}}</ref>。

相比于前几任团中央第一书记如[[李克强|李克强]]、[[周强|周强]]、[[胡春华|胡春华]]、[[陆昊|陆昊]]分别转任[[河南省|河南省]]省长、[[湖南省|湖南省]]省长、[[河北省|河北省]]省长、[[黑龙江省|黑龙江省]]省长，秦宜智出任国家质量监督检验检疫总局排名垫底的副局长，贬谪意味明显。政论作家高新认为，在[[中国共产党第十九次全国代表大会|中共十九大]]即将召开之前“学习小组”公布《习近平关于青少年和共青团工作论述摘编》一书出版发行，书中节选内容充满习近平对于共青团干部的批评之声，接着中央组织部再将共青团中央书记处第一书记秦宜智贬官到国家质检总局任副局长，这都是[[习近平|习近平]]在为十九大之后高层人事布局所做的铺垫，最终的目的是要打击[[胡春华|胡春华]]（曾任团中央第一书记的60后中央政治局委员）和[[李源潮|李源潮]]（国家副主席分管共青团事务）等[[团派|团派]]高官，以便让习近平的嫡系下属[[栗战书|栗战书]]、[[陈敏尔|陈敏尔]]、[[李强_(1959年)|李强]]、[[刘鹤|刘鹤]]、[[丁薛祥|丁薛祥]]、[[陈希|陈希]]、[[杨晓渡|杨晓渡]]、[[黄坤明|黄坤明]]、[[蔡奇|蔡奇]]等人上位做准备<ref>{{cite web|url=http://www.rfa.org/mandarin/zhuanlan/yehuazhongnanhai/gx-09222017131514.html|title=对共青团干部全盘否定压制胡春华问责李源潮（高新）|accessdate=2017-09-22|date=2017-09-22|publisher=自由亚洲电台}}</ref>。
