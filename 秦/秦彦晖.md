'''秦彦晖'''（{{bd|9世纪||910年代|catIdx=Q}}），[[五代十国|五代十国]]时期[[楚_(十国)|楚国]]武将。

== 家世背景 ==
秦彦晖是[[上蔡|上蔡]]人。父秦辙，本郡牙校。族兄[[秦宗权|秦宗权]]，唐朝末年军阀[[奉国军节度使|奉国军节度使]]。<ref name=h9k>《九国志》</ref><ref name=SGCQ>《十国春秋》</ref>

秦彦晖年轻时强健勇敢有智略。<ref name=h9k/>

== 楚国建立前 ==
唐朝末年，秦宗权反叛朝廷，派弟弟[[秦宗衡|秦宗衡]]、决胜指挥使[[孙儒|孙儒]]等夺取[[淮南|淮南]]。秦宗权即将败亡，宗族担心遭祸，秦彦晖请求跟随族兄秦宗衡。孙儒杀秦宗衡，夺取军权。行军司马[[张佶_(唐朝)|张佶]]、龙骧指挥使[[刘建锋|刘建锋]]、秦彦晖、[[马殷|马殷]]等随孙儒在淮南作战，秦彦晖随孙儒攻[[宁国军节度使|宁国军节度使]][[杨行密|杨行密]]军部[[宣州_(隋朝)|宣州]]。后孙儒被杨行密败亡，刘建锋率其余部夺取[[武安军|武安军]]，称[[武安军节度使|节度使]]，秦彦晖跟随他，为捉生指挥使。后刘建锋被杀，马殷继任，秦彦晖效力马殷为亲校。<ref name=h9k/><ref name=SGCQ/>

[[光化|光化]]元年（898年）五月，[[李琼|李琼]]和秦彦晖被任为岭北七州游奕使，率副将[[张图英|张图英]]、[[李唐_(楚)|李唐]]等攻取[[连州_(隋朝)|连]]、[[邵州|邵]]、[[郴州_(古代)|郴]]、[[衡州|衡]]、[[道州|道]]、[[永州_(隋朝)|永]]六州。<ref>[[:s:资治通鉴/卷261|《资治通鉴》卷二百六十一]]</ref>

三年（900年）十月，[[静江节度使|静江节度使]][[刘士政|刘士政]]派副使陈可璠屯全义岭防备马殷，马殷遣秦彦晖、李琼等率兵七千攻刘士政。武安军到全义，刘士政又遣指挥使王建武屯秦城。陈可璠掠夺县民耕牛犒军，县民生怨，请为武安军向导，说：“这里西南有小径，距秦城才五十里，仅通单骑。”秦彦晖遣李琼派六十骑、三百步兵袭秦城，夜晚越墙而入，擒王建武，天亮复回，用绳子绑着王建武到陈可璠寨下示之，陈可璠还不信。武安军斩王建武首级投进陈可璠寨中，静江军震恐，李琼趁机歼灭陈可璠军，引兵直扑静江军军部[[桂州|桂州]]，从秦城以南二十余寨都望风奔溃，李琼于是围桂州，刘士政出降，桂、宜、岩、柳、象五州皆降于武安军。<ref name=NHFD66>[[:s:新五代史/卷66|《新五代史》卷六十六]]</ref><ref>[[:s:资治通鉴/卷262|《资治通鉴》卷二百六十二]]</ref>秦彦晖同为大将，统诸军，俘斩功多，被马殷奏迁检校[[太保|太保]]。改在城都指挥使。<ref name=h9k/><ref name=SGCQ/>

后[[宣武军节度使|宣武军节度使]][[朱全忠|朱全忠]]控制朝廷，以朝廷名义下诏要马殷等出兵救被杨行密攻打的[[武昌军节度使|武昌军节度使]][[杜洪|杜洪]]。马殷派秦彦晖和[[岳州|岳州]][[刺史|刺史]][[许德勋|许德勋]]率水师东下救杜洪，不及，杜洪败死。<ref name=NHFD66/><ref>《九国志·许德勋传》</ref>

== 效力楚国 ==
马殷被封为楚王，故政权称[[马楚|楚国]]。[[唐朝|唐朝]]被[[后梁|后梁]]取代后，[[开平_(后梁)|开平]]元年（907年）五月，[[杨吴|弘农]]（杨行密奠基的政权）以[[鄂岳观察使|鄂岳观察使]][[刘存|刘存]]为西南面都招讨使，岳州刺史[[陈知新|陈知新]]为岳州团练使，[[镇南军节度使|镇南军节度使]][[刘威_(吴)|刘威]]为应援使，别将[[许玄应|许玄应]]为监军，率水军三万伐楚，马殷命秦彦晖率水军三万顺江东下。秦彦晖在上流埋伏了大船，派水军副指挥使黄璠率战舰三百屯浏阳口。六月，刘存等遇大雨，引兵回到越堤北，秦彦晖追之。秦彦晖与刘存战于越堤，刘存等屡不胜，写信给马殷求和诈降，马殷想答应，秦彦晖说：“淮人多诈，是要让我军懈怠，不可信。”急攻之，与其隔水列阵，刘存在远处呼道：“我已经投降了，公却不许，古人说‘杀降不祥’，公难道不为子孙打算吗？”秦彦晖怒道：“若是贼入吾境到我城下了我还不攻击，还顾什-{么}-子孙？”鼓噪麾兵大进，刘存等逃跑，黄璠从浏阳引兵切断江上道路，与秦彦晖合击，大破弘农军，擒刘存及陈知新，秦彦晖亲自杀弘农裨将百余人，弘农军数万士兵阵亡，武安军得其战舰八百艘。刘威乘轻舟率余众逃回，秦彦晖于是攻占岳州。<ref name=SGCQ/><ref name=NHFD66/><ref name=ZZTJ266>[[:s:资治通鉴/卷266|《资治通鉴》卷二百六十六]]</ref>以功被奏加检校[[太傅|太傅]]。<ref name=h9k/>

十月，秦彦晖会合[[荆南军|荆南军]]将[[倪可福|倪可福]]攻依附弘农的[[武贞军节度使|武贞军节度使]][[雷彦恭|雷彦恭]]军部[[朗州|朗州]]，雷彦恭引[[沅江|沅江]]环绕州城以自守。二年（908年），秦彦晖顿兵一月有余不与其交战，雷彦恭防御稍松懈。五月，秦彦晖派禆将曹德昌率壮士趁夜从水洞入城，内外举火彼此响应，秦彦晖鸣金鼓大噪破城门而入，杀雷彦恭部将区景思，雷彦恭奔淮南军部[[广陵|广陵]]，秦彦晖于是擒雷彦恭弟雷彦雄等七人而回。<ref>《九国志》作三年（909年）事，且称秦彦晖擒住雷彦恭。</ref>于是原来与雷彦恭勾结的[[澧州|澧州]][[向瓌|向瓌]]、[[辰州_(隋朝)|辰州]][[宋邺|宋邺]]、[[溆州|溆州]][[昌师益|昌师益]]等都率溪洞诸蛮来附。<ref name=NHFD66/><ref name=ZZTJ266/>不数年，湖南平定，秦彦晖功最大。<ref name=SGCQ/>迁[[道州|道州]]刺史。<ref name=h9k/>

[[乾化|乾化]]年间，秦彦晖接受代任，回楚国都城[[潭州|潭州]]，卒。<ref name=h9k/>

== 评价 ==
*《十国春秋》论曰：楚介在蛮方，北临吴会，南偪岭表，中间江陵，征讨捍御，故非诸臣莫为功。许（德勋）、李（琼）、'''秦'''、[[王环_(马楚)|王（环）]]，皆桓桓虎臣，允矣干城之选也。<ref name=SGCQ/>
