'''秦皇岛港口近代建筑群'''，位于[[河北省|河北省]][[秦皇岛市|秦皇岛市]][[海港区|海港区]]，包括多个[[全国重点文物保护单位|全国重点文物保护单位]]、[[河北省文物保护单位|河北省文物保护单位]]等。

==简介==
秦皇岛自古便为水陆交通海防、关防要冲。[[清朝|清朝]][[道光|道光]]以后，秦皇岛半岛港湾形成。[[光绪|光绪]]二十四年三月初五（1898年3月24日）[[总理衙门|总理衙门]]补奏《秦皇岛自开口岸折》，内称秦皇岛“隆冬不冻”，若开为[[通商口岸|通商口岸]]，与[[津榆铁路|津榆铁路]]接近，殊于商务有益。同日，[[光绪帝|光绪帝]]朱批“依议钦此”。光绪二十六年（1900年），[[秦皇岛港|秦皇岛港]]与[[开滦煤矿|开滦煤矿]]同被[[英国|英国]]控制后，加紧港口扩建。此后20余年，港口建设日趋完善，形成老港区与大、小码头的格局。随着城市发展，近代工业建筑不断失去功能，大批遗留至今<ref name=qianshu>郭绘宇，浅述秦皇岛近代港口建筑群的历史沿革及现状，企业文化旬刊2015(7)</ref>。

秦皇岛港口近代建筑群位于西港区，集中在东南山林深处以及海滨路、光明路一带<ref name=guo/>。2002年，秦皇岛市文物管理处初步调查部分城市区近代工业遗产，掌握全市共有近代工业遗产37处，包括厂房、仓库、铁路基址、码头、店铺等。2003年，秦皇岛市文物管理处调查近代建筑并制作简略的记录档案。2004
年，随着14处近代建筑被公布为[[秦皇岛市文物保护单位|秦皇岛市文物保护单位]]，秦皇岛市文物管理处为现存的32处近代建筑制作了记录档案。2008年10月17日经河北省人民政府第17次常务会议通过，17处港口近代建筑作为“秦皇岛港口近代建筑群”被公布为第五批[[河北省文物保护单位|河北省文物保护单位]]。2008年，全国第三次文物普查工作启动，文物部门筹资对河北省文物保护单位开平矿务局秦皇岛经理处办公楼进行调查，绘制了平、立、剖面图纸，拍摄了现状照片，走访当地老人，收集资料。2013年3月5日，上述17处港口近代建筑中的9处作为“秦皇岛港口近代建筑群”被[[中华人民共和国国务院|国务院]]公布为[[第七批全国重点文物保护单位|第七批全国重点文物保护单位]]。但仍有部分近代建筑保护等级不够或未被纳入保护范围。随着2010年代中期秦皇岛市启动城市区企业“退城进郊”与“西港东迁”，近代遗址保护再次受到政府及公众关注<ref name=guo>郭绘宇，秦皇岛近代港口建筑群保护初探，大众文艺2015(15):60-60</ref><ref name=qianshu/>。

2015年，结合西港搬迁改造项目，秦皇岛市编制完成《秦皇岛港口近代建筑群保护规划》，并且全面启动港口近代工业遗产保护利用。秦皇岛港口近代建筑群主要包括开滦矿务局高级员司俱乐部、开平矿务局秦皇岛经理处办公楼、南栈房、老船坞、京奉铁路桥等21处优秀近代工业遗产<ref name=gh>{{Cite web |url = http://qhd.leju.com/news/2015-04-14/14155993627481230604289.shtml |title =秦皇岛近代建筑群保护规划 秦皇岛港口成重点保护对象  |author =  |date = 2015-04-14 |publisher = 秦皇岛乐居网 |language =  |accessdate =  |quote =  }}</ref><ref name=qq>{{Cite web |url = http://hb.jjj.qq.com/a/20170707/034120.htm |title =  秦皇岛近代建筑群将成特色文化旅游基地|author =  |date = 2017-07-07 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。其中全国重点文物保护单位9处，河北省文物保护单位8处，秦皇岛市文物保护单位2处<ref name=gh/>。
