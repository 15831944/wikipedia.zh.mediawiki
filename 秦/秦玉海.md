'''秦玉海'''（{{bd|1953年|3月||catIdx=Q秦}}），[[黑龙江|黑龙江]][[泰来|泰来]]人。1976年3月加入[[中国共产党|中国共产党]]。中共河南省委党校法律专业研究生学历。

== 生平 ==
历任[[共青团|共青团]]黑龙江省委副书记、书记，中共[[鹤岗市|鹤岗市]]委副书记、代市长、市长，[[焦作市|焦作市]]委副书记、代市长、市长、市委书记<ref>{{cite web|accessdate=2012-04-21|url=http://news.cqnews.net/html/2011-11/30/content_10597421.htm|title=(资料)河南省副省长、公安厅厅长秦玉海简历|publisher=[[华龙网|华龙网]]}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> 等职。2004年1月，当选[[河南省人民政府|河南省人民政府]]副省长兼[[河南省公安厅|河南省公安厅]]厅长<ref>{{cite web|accessdate=2012-04-21|url=http://news.sohu.com/2004/01/09/61/news218246193.shtml|title=河南省人大常委会任命秦玉海为河南省副省长|publisher=[[搜狐|搜狐]]}}</ref>。2004年7月30日被授予副总警监警衔。

2011年3月23日，当选为河南省残联第五届主席团主席。2012年9月，兼任河南省社会管理综合治理委员会副主任。2013年1月29日在河南省十二届人大四次会议上当选河南省十二届人大常委会副主任。他是[[中共十六大|中共十六大]]代表，第九届、十一、十二届[[全国人大代表|全国人大代表]]。2014年12月28日，十二届全国人大常委会第十二次会议表决通过罢免代表案<ref>{{cite news|title=荣兰祥、秦玉海等人的全国人大代表资格终止|url=http://www.npc.gov.cn/npc/xinwen/2014-12/29/content_1891925.htm|accessdate=2016-01-09|agency=全国人大网|date=2014年12月28日}}</ref>。

2014年9月21日，因涉嫌严重违纪违法，被免去河南省人大常委会党组书记职务，采取双规措施，并立案审查。<ref >{{cite web|url=http://politics.people.com.cn/n/2014/0923/c1001-25719622.html | language= | title=刘春良任河南省人大常委党组书记 秦玉海被免职 |publisher=央视 |date=2014-09-23 |accessdate= }}</ref>他是[[中共十八大|中共十八大]]后首个被查的河南省部级官员<ref >{{cite web|url=http://news.163.com/14/0921/19/A6MJDM3B0001124J.html | language= | title=河南省人大常委会副主任秦玉海被调查 |publisher=央视 |date=2014-09-21 |accessdate= }}</ref>。媒体称秦玉海的落马标志着发轫于“[[皇家一号|皇家一号]]”[[夜总会|夜总会]]案的河南省公安系统集体贪腐案件进入更深阶段。<ref >{{cite web|url=http://news.163.com/14/0922/01/A6N86CRQ0001124J.html | language= | title=河南人大副主任秦玉海被查:或涉"皇家一号"案 |publisher= 第一财经日报 |date=2014-09-22 |accessdate= }}</ref>2015年2月13日，因涉嫌占用公物、挥霍浪费公共财产；滥用职权、受贿、通奸，被开除党籍、公职；移送司法机关<ref>[http://news.163.com/15/0213/13/AIB9BP0L00014JB5.html 武长顺秦玉海均收受巨额贿赂 与他人通奸]</ref>。

2016年11月28日，[[山东省|山东省]][[淄博市|淄博市]]中级人民法院公开宣判，对秦玉海以受贿罪判处有期徒刑13年6个月，并处罚金人民币200万元；对秦玉海受贿所得财物及其孳息予以追缴，上缴国库<ref>{{cite web |url = http://www.ccdi.gov.cn/yw/201611/t20161128_90260.html |title = 河南省人大常委会原副主任秦玉海一审被判十三年六个月 |accessdate = 2016年11月28日 |publisher = 中央纪委监察部网站}}</ref>。

2004年1月至2013年1月曾任职达九年的河南省副省长，兼任河南省委政法委副书记、公安厅长、党委书记兼武警河南总队第一政委、党委第一书记。任内曾力推“命案必破”办案理念，在其河南公安厅长任职末期及卸任后，一系列冤假错案的相继曝光，使得“命案必破”制度开始饱受质疑，防范冤假错案呼声日渐高涨。他也曾一手推动了“撤销公安分局、做大[[派出所|派出所]]”的河南警务改革。不过，在其卸任公安厅长后，很多改革措施又遭遇执行不力，部分地区恢复了原状。<ref >{{cite web|url=http://news.163.com/14/0921/22/A6MT6UI500014SEH.html | language= | title=秦玉海与周永康同推"命案必破" 引多起冤案 |publisher=澎湃新闻网 |date=2014-09-21 |accessdate= }}</ref>

他也是[[中国摄影家协会|中国摄影家协会]]理事，有不少[[摄影|摄影]]作品参加过各大影赛的评比和展览。

== 相关落马官员 ==
