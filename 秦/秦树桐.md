'''秦树桐'''（{{bd|1963年|11月||}}），[[江苏|江苏]][[姜堰|姜堰]]人，[[中国人民解放军少将|中国人民解放军少将]]。

曾任[[中国人民解放军|中国人民解放军]]旅政委，师政委，[[中国人民解放军陆军第七十一集团军|陆军第12集团军]]政治部副主任，[[中国人民解放军陆军第七十三集团军|第31集团军]]政治部主任等职<ref>{{cite web |title=31军高层频繁换血 王安龙大校履新政治部主任 |url=http://mil.sohu.com/20141223/n407199274_1.shtml |website=搜狐 |accessdate=2018-09-08}}</ref>。2014年，任中国人民解放军第31集团军副政委。2015年11月，升任[[中国人民解放军陆军第七十二集团军|陆军第1集团军]]政委<ref>{{cite web |title=南京军区第1集团军政委易人，秦树桐接棒白吕 |url=https://www.thepaper.cn/newsDetail_forward_1393203 |website=澎湃新闻 |accessdate=2018-09-08}}</ref>。2017年3月，任新调整组建的[[中国人民解放军陆军第七十五集团军|第75集团军]]政委<ref>{{cite web |title=原第1集团军政委秦树桐出任第75集团军政委 |url=https://www.thepaper.cn/newsDetail_forward_1732405 |website=澎湃新闻 |accessdate=2018-09-08}}</ref>。2018年5月，升任[[中国人民解放军陆军|陆军]]政治工作部主任<ref>{{cite web |title=中部战区陆军政委周皖柱、新疆军区司令员刘万龙晋升中将 |url=http://finance.ifeng.com/a/20180726/16408747_0.shtml |website=凤凰网 |accessdate=2018-09-08}}</ref>。

2014年7月，晋升少将军衔。
