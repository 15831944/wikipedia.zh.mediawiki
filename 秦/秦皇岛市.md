'''秦皇岛市'''是[[中华人民共和国|中华人民共和国]][[河北省|河北省]]下辖的[[地级市|地级市]]，位于河北省东端，是[[京哈铁路|京哈铁路]]上重要城市之一。靠在[[渤海湾|渤海湾]]边上，西距首都[[北京市|北京]]280公里，距[[天津市|天津]]220公里，西南距省会[[石家庄市|石家庄]]483公里。以[[秦始皇|秦始皇]]求仙驻跸而得名。

秦皇岛是全国首批14个[[中華人民共和國沿海開放城市|沿海开放城市]]之一，中国北方重要的对外贸易口岸，国务院批准的全国甲级旅游城市。著名的风景区[[山海关|山海关]]、[[北戴河区|北戴河]]坐落于此。

== 历史 ==
[[商朝|商]][[周朝|周]]时期为[[孤竹国|孤竹国]]中心区域，[[春秋时期|春秋时期]][[晋国|晋]]灭[[肥國_(春秋)|肥]]，肥子逃奔[[燕国|燕国]]，燕封肥子在此地建肥子国。[[战国|战国]]时期，此地属[[燕国|燕国]][[辽西郡|辽西郡]]。[[秦朝|秦]][[汉朝|汉]]时期这里是东巡朝拜和兵家必经之地。[[秦始皇|秦始皇]]第四次出巡到[[碣石镇|碣石]]，刻碣石门。并派燕人卢生、韩终、侯公、石生等方士入海求仙人和不死之药，秦皇岛由此得名。[[汉武帝|汉武帝]]东巡观海，到碣石筑汉武台，并在此用兵攻朝鲜[[卫满朝鲜|衛氏王朝]]，把[[北戴河区|北戴河]]金山嘴作为屯粮城。[[曹操|曹操]]率兵北伐[[乌桓|乌桓]]，取道[[渤海|渤海]]之滨，望临碣石，赋《观沧海》诗。

[[File:Qinhuangdao_1908_AMS-WO.jpg|thumb]]
[[隋朝|隋]][[唐朝|唐]]时期，这里是抵御关外[[突厥人|突厥]]、[[契丹人|契丹]]的战略要地，[[唐玄宗|唐玄宗]]派[[史思明|史思明]]统平州太守，兼统卢龙军使。[[元朝|元朝]][[忽必烈|忽必烈]]将中书省平滦路设在此地，后改为[[永平府|永平府]]，[[明朝|明朝]]曾在此设立盐署。1381年，[[朱元璋|明太祖朱元璋]]派开国元勋中山王[[徐达|徐达]]主持修建了[[山海关|山海关]]关城。明[[嘉靖|嘉靖]]十四年（1535年）版《山海关志》最早记有这一地名：“秦皇岛，城西南25里，又入海1里。”1644年，明王朝覆灭，[[李自成|李自成]]带兵4万余众到山海关征讨[[吴三桂|吴三桂]]，山海关守将吴三桂投降[[多尔衮|多尔衮]]，开关引清军15万同[[李自成|李自成]]大战，这就是改变中国历史命运的“石河大战”，经三天苦战，农民军终因兵寡失利，退至境内卢龙县（永平府），重整军队再战，大败，李自成逃回京城。清王朝从此入主[[中原|中原]]。

[[清朝|清王朝]]统治时期，在山海关设立臨榆县。1898年，秦皇岛港启锚开运，[[津榆铁路|津榆铁路]]（天津－临榆）通车。同年，清政府正式将北戴河开辟为“各国人士避暑地”。[[中華民國|民国]]初期，秦皇岛属[[直隶省|直隶省]]渤海道，外国军队、牧师开始进入秦皇岛境内。

1948年，[[中国人民解放军|解放军]]入驻秦皇岛；1949年设市；1978年秦皇岛作为首批全国沿海开放城市。秦皇岛作为[[2008年夏季奥林匹克运动会|北京2008奥运会]]协办城市成功举办奥运赛事。

== 地理 ==
秦皇岛市地处河北省东北部，北纬39°24′~ 40°37′，东经118°33′~ 119°51′。南面频临[[渤海|渤海]]，北面背靠[[燕山|燕山]]，东接[[辽宁省|辽宁省]][[葫芦岛市|葫芦岛市]]，西邻[[唐山市|唐山]]、[[承德市|承德]]两市，距首都[[北京市|北京]]约280公里。市境地势北高南低。

秦皇岛气候类型属于半湿润[[温带季风气候|温带季风气候]]，受海洋影响较大。

== 交通 ==
[[File:Qinhuangdao_Railway_Station_Platforms.jpg|thumb]][[津秦客运专线|津秦客專]]{{站台}}]]
[[秦皇岛站|秦皇岛火车站]]为[[京哈铁路|京哈铁路]]、[[大秦铁路|大秦铁路]]和[[津山铁路|津山铁路]]的沿线车站，中国首条快速客运专线——[[秦沈客运专线|秦沈客运专线]]起点。该线于2003年10月12日正式运营。2013年12月，[[津秦客运专线|津秦客运专线]]也投入运营。

[[秦皇岛北戴河机场|秦皇岛北戴河机场]]位于[[昌黎县|昌黎县]]，于2016年3月31日正式启用。其有来往[[石家庄|石家庄]]、[[成都|成都]]和[[上海|上海]]的航班。

[[秦皇岛港|秦皇岛港]]是中国最大的能源输出港口，是中国山西煤炭外运的主要港口之一，是天然不冻良港。

== 工业 ==
主要工业有[[玻璃|玻璃]]、[[造船|造船]]、[[冶金|冶金]]以及[[汽车|汽车]]配件。

== 旅遊 ==
[[File:Shanhaiguan.jpg|thumb]]]]
著名的旅遊勝地[[北戴河区|北戴河]]、[[南戴河街道|南戴河]]、[[昌黎县|昌黎]][[黃金海岸_(昌黎)|黃金海岸]]和天下第一關[[山海关|山海关]]以及青龙满族自治县的祖山自然风景区。

知名的旅游景点有：[[山海关|天下第一关]]，[[老龙头|老龙头]]，[[孟姜女庙|孟姜女庙]]，[[角山|角山]]，[[鸽子窝|鸽子窝]]，[[老虎石|老虎石]]，[[联峰山|联峰山]]，[[海洋馆|海洋馆]]，[[翡翠岛|翡翠岛]]及2014年12月落成的[[中國夢碑|中國夢碑]]<ref>{{cite news|url=http://www.nanzao.com/tc/china/44438/si-mian-gong-zi-wei-ji-bei-dai-he-li-zhong-guo-meng-di-yi-bei|title=四面“共”字為基 北戴河立中國夢第一碑|date=2014-12-12|publisher=[[南華早報|南華早報]]中文版|accessdate=2014-12-13}}</ref>等。
