'''秦琼'''（{{bd|571年|？|638年|catIdx=Q秦}}），[[表字|字]]'''叔宝'''，齐州历城（今[[山东|山东]][[济南|济南]]）人。[[唐朝|唐朝]]开国将领，[[凌烟阁二十四功臣|凌烟阁二十四功臣]]之一，与[[尉迟恭|尉迟敬德]]为传统[[门神|门神]]。济南[[五龙潭|五龙潭]]有其故居。

== 生平 ==
===早年===
秦琼以勇猛驃悍著称。最初是[[隋朝|隋]]将[[来护儿|来护儿]]部将。秦叔宝的母亲逝世，来护儿派人去吊唁，来护儿手下军吏不解，问来护儿：“士卒死亡及遭丧者多矣，将军未尝降问，独弔叔宝何也？”来护儿回答：“此人勇悍，加有志节，必当自取富贵，岂得以卑贱处之。”

大业十年（614年）叔宝跟随[[张须陀|张须陀]]，攻打驻守[[下邳|下邳]]的[[卢明月|卢明月]]。卢明月军队十余万人，声势浩大，而张须陀的隋军才一万余人，势单力薄。于是离卢明月军六七里安营扎寨，相持十多日，粮草殆尽，准备撤军。张须陀对诸将道：“贼见兵却，必轻来追我。其众既出，营内即虚，若以千人袭营，可有大利。此诚危险，谁能去者？”诸将莫敢应答，唯有叔宝和[[罗士信|罗士信]]愿意尝试。张须陀拔营撤走，叔宝和罗士信各领兵千余人埋伏在芦苇间。卢明月见隋军撤走，于是全军追击，叔宝和罗士信领兵攻击卢明月营寨，栅门紧闭，叔宝和罗士信亲自翻过栅门，手杀数人，卢明月军营内大乱。叔宝等又放隋军入营，纵火焚烧三十栅，“烟焰涨天”，卢明月得知大营被偷袭，匆忙撤军，张须陀领军回身追击，大破之。卢明月惨败，带领数百骑兵逃遁，其余人皆被俘虏。此战后，叔宝声名大噪，以勇气闻名于世。随后在海曲征讨[[孙宣雅|孙宣雅]]，先登破之。以前后功，授建节尉。

===瓦岗寨时期===
616年跟随张须陀在荥阳讨伐[[李密|李密]][[瓦岗军|瓦岗军]]，兵败，张须陀战死，秦琼领部下归附隋将[[裴仁基|裴仁基]]。裴仁基以武牢关投降[[李密|李密]]，秦琼也随之归降，受到重用，被任用为帐内骠骑。李密与王世充决战失败后，秦琼和[[程知节|程知节]]、裴仁基等人投降[[王世充|王世充]]。

===李唐时期===
武德二年（619年），因不满王世充，秦琼同程知节、[[吴黑闼|吴黑闼]]、[[牛进达|牛进达]]等人，乘唐郑两军于九曲对阵之时，一起投奔了李唐，被唐高祖[[李渊|李渊]]分配到秦王[[李世民|李世民]]帐下，从镇长春宫，拜马军总管，不久授秦王府右三统军。参加了李世民的历次征战，每战必先，常于万军之中[[斬首|取敌将首级]]。

武德四年（621年），在与[[王世充|王世充]]、[[窦建德|窦建德]]的[[虎牢之战|大决战]]中，率领骑兵突击队衝入窦军阵营，立下了关键的战功，进封'''翼国公'''，赐黄金百斤、帛七千段。武德九年（626年），《旧唐书》说秦琼参与[[玄武门之变|玄武门之变]]，事后被封为[[左武卫大将军|左武卫大将军]]，食[[益州|益州]]实封七百户。秦琼晚年因历次作战负伤太多而疾病缠身，常对人说：“吾少长戎马，所经二百余阵，屡中重疮。计吾前后出血亦数斛矣，安得不病乎？”[[貞觀_(唐太宗)|贞观]]十二年（638年）去世，谥号：'''壮'''，追赠[[徐州|徐州]][[都督|都督]]，陪葬[[唐昭陵|昭陵]]，唐太宗在他墓地内立石人石马，以旌战阵之功。

==后事==
贞观十三年（639年），秦琼被改封为'''胡国公'''。贞观十七年（643年），秦琼与[[长孙无忌|长孙无忌]]等人被图形[[凌烟阁|凌烟阁]]，封为[[凌烟阁二十四功臣|凌烟阁二十四功臣]]之一。

==家族==
《[[旧唐书|旧唐书]]》、《[[新唐书|新唐书]]》中秦琼传记均没有记载秦琼家族情况。1990年代后，陆续有秦氏家族成员的墓志出土，秦琼家族方才渐渐明朗。

1995年，山东省济南市发现[[秦爱|秦爱]]墓葬，墓志记载有秦琼父、祖、曾祖。墓主'''[[秦爱|秦爱]]'''，字季养，为秦琼之父，[[北齐|北齐]]咸阳王[[斛律武都|斛律武都]]府录事参军，北齐亡于北周后告归乡里，隋大业十年卒于齐州历城县怀智里，终年六十九岁，唐贞观初年追封历城县开国公。秦琼祖父'''[[秦方太|秦方太]]'''，北齐广宁王府记室。秦琼曾祖'''[[秦孝达|秦孝达]]'''，元魏广年县令。<ref>{{cite wikisource |title = 秦愛墓志銘|wslink = 秦愛墓志銘|wslanguage= zh|quote= 君諱'''愛'''，字季養……祖'''孝達'''，魏廣年縣令……父'''方太'''，齊廣寧王府記室……齊鹹陽王斛律武都，朝之上將，初開幕府，妙選賢良，乃召君（秦愛）為錄事參軍，禮接殊重，恩紀之深，群僚莫及。周武平齊，君乃告歸鄉里。……以大業十年十一月廿一日終於齊州曆城縣懷智里宅，春秋六十九。……}}</ref>

[[秦怀道|秦怀道]]及其子[[秦佾|秦佾]]墓志记载有秦氏先祖及秦琼子孙。秦怀道墓志记载，'''[[秦怀道|秦怀道]]'''，字理，为秦琼孙，常州义兴县令，袭封历城县开国公，唐嗣圣元年卒于括州括苍县，终年六十岁，唐开元六年改葬于洛阳县清风乡。秦氏祖籍[[右扶风|右扶风]]，为东汉山阳太守'''[[秦彭|秦彭]]'''之后。秦怀道'''六世祖'''为慕容燕乐陵郡守，自此起秦氏迁居至齐州历城县。秦怀道子'''[[秦景倩|秦景倩]]'''，越州山阴县令。<ref>{{cite wikisource |title = 唐故常州義興縣令上柱國秦府君墓誌銘并序|wslink = 唐故常州義興縣令上柱國秦府君墓誌銘并序|wslanguage= zh|quote= 公諱'''懷道'''，字'''理'''。其先出自右扶風，漢山陽太守'''彭'''之後也。'''六世祖'''起，仕燕爲樂陵郡守。子孫徔因官家於齊，爲齊州歷城縣人焉。……曾祖'''季'''，北齊咸陽斛律武都府録事參軍。貞觀初，追贈瀛州刺史，上柱國，歷城縣開國公，食邑一千戶。……祖'''叔寶'''……又轉常州義興縣令，襲封歷城縣開國公……嗣聖元年二月一日，遘疾終於括州括蒼縣，春秋六十。……嗣子前越州山陰縣令'''景倩'''……以開元六年十二月廿三日，改葬於洛陽縣清風鄉之原……}}</ref>秦佾墓志记载，'''[[秦佾|秦佾]]'''为秦琼孙、秦怀道子，潞州司法参军，武周圣历二年卒于洛阳，圣历三年迁葬于北邙山。秦佾九世祖'''秦秀'''，晋金紫光禄大夫、太常卿。<ref>{{cite wikisource |title = 皇朝故潞州司法秦君墓志銘並序|wslink = 皇朝故潞州司法秦君墓志銘並序|wslanguage= zh|quote= 公諱'''佾'''……九代祖'''秀'''，晉金紫光禄大夫太常卿……曾祖'''季'''……祖'''叔寶'''……父'''懷道'''……轉潞州司法參軍……以聖曆二年六月廿日，卒于洛陽道光之里第。以聖曆三年二月二日遷措於北邙山之舊塋禮也。……}}</ref>

[[秦怀道|秦怀道]]、[[秦佾|秦佾]]与秦琼的关系，二人墓志记载有冲突。秦怀道墓志言其“祖叔宝”，秦佾墓志言其“祖叔宝、父怀道”。二人墓志综合来看，一方面，秦怀道墓志为其子[[秦景倩|秦景倩]]为其父迁葬时所刻，志文有可能是以秦景倩的口吻描述，故言“曾祖季、祖叔宝”。另一方面，秦怀道墓志未记载其父，墓志镌刻于其去世34年之后，故可能误记为秦琼孙。故秦怀道应为秦琼之子，秦怀道有子秦景倩、秦佾。秦怀道墓志中的“六世祖”，应为秦琼四世祖，即秦琼曾祖秦孝达之父。秦佾墓志中的“九世祖”，应为秦琼七世祖。

[[秦利见|秦利见]]墓志记载有秦琼另一子孙，并与[[尉迟敬德|尉迟敬德]]家联姻。秦利见墓志记载，'''[[秦利见|秦利见]]'''，字利见，为秦琼孙，许州鄢陵县丞，唐神龙三年卒于合宫县道光坊宅。秦利见父'''秦□道'''（墓志阙字），左卫左郎将。秦利见母为[[尉迟敬德|尉迟敬德]]孙女、[[尉迟宝琳|尉迟宝琳]]之女。<ref>{{cite wikisource |title = 大唐故胡國公嫡孫許州鄢陵縣丞秦府君墓誌|wslink = 大唐故胡國公嫡孫許州鄢陵縣丞秦府君墓誌|wslanguage= zh|quote= 君字'''利見'''……勑授許州鄢陵縣丞……祖'''叔寶'''……外祖'''尉遲琳'''，衛尉卿……父'''□道'''，左衛左郎將……以神龍三年八月廿五日薨於合宫縣道光坊宅。……}}</ref>

[[秦晙|秦晙]]及其子[[秦洽|秦洽]]墓志记载有秦琼另一子孙。秦晙墓志记载，'''[[秦晙|秦晙]]'''，字景嗣，为秦琼孙，吉州司马，唐开元二十七年卒于官舍，年七十一岁。秦晙父'''[[秦善道|秦善道]]'''，左清道率、检校左金吾卫大将军。秦晙有子'''[[秦凑|秦凑]]'''。<ref>{{cite wikisource |title = 大唐故吉州司馬秦府君墓誌銘並序|wslink = 大唐故吉州司馬秦府君墓誌銘並序|wslanguage= zh|quote= 府君諱'''晙'''，字'''景嗣'''……曾祖'''季'''……祖'''叔寶'''……父'''善道'''，皇朝左清道率、檢校左金武衛大將軍……開元廿七年九月十九日遘疾終於官舍，春秋七十有一。……有子'''湊'''等，孝乎罔極，哀以終身。……}}</ref>秦洽墓志记载，'''[[秦洽|秦洽]]'''，字伯淮，为秦琼曾孙、秦晙子，豫章郡参军，唐开元二十三年卒，年三十六岁。秦洽有子'''[[秦寰|秦寰]]'''。<ref>{{cite wikisource |title = 唐故豫章郡參軍秦府君墓志銘並序|wslink = 唐故豫章郡參軍秦府君墓志銘並序|wslanguage= zh|quote= 公諱'''洽'''，字伯淮……曾祖'''叔寶'''……祖'''善道'''……父'''景嗣'''……以開元廿三載正月十六日寢疾而終，享年卅六。……孤子'''寰'''，欲養而親不待，處喪而哀有餘。父母既歿，兄弟無在。……}}</ref>

== 文艺作品中的秦琼 ==
秦琼在民间故事中以忠义著称。《[[隋唐演義|隋唐演義]]》有大量關於秦琼的故事記載，如：{{〈}}秦瓊當鐧賣馬{{〉}}、{{〈}}叔寶救駕紅泥澗{{〉}}，以義士、良將、忠臣的形象貫穿全書。相傳秦瓊的武器是一對金[[鐧|鐧]]，長二尺二寸（約70～80公分）、重二十餘斤，秦瓊過世後，由[[秦懷玉|秦懷玉]]繼承，[[秦懷玉|秦懷玉]]戰死後，由孫[[秦夢|秦夢]]所繼承。小說中說秦瓊之絕招為｢'''[[撒手鐧|撒手鐧]]'''｣，與[[羅成|羅成]]的｢'''[[回馬槍|回馬槍]]'''｣並名於世。

小说《[[西游记|西游记]]》裏记载了秦琼变成[[门神|门神]]的故事：[[唐太宗|唐太宗]]因無心之過，害死了泾河[[龙王|龙王]]，龍王阴魂不散，闹得太宗六神無主。[[魏徵|魏徵]]建议派秦琼、[[尉迟恭|尉迟恭]]守住前后宫门保驾，果然奏效，皇帝由於兩人太過疲憊，命画家[[吴道子|吴道子]]画了两人之像，贴在宫门口，照样管用。后来此举在民间流传开来，秦琼与[[尉迟恭|尉迟恭]]成了最常见的门神。

[[黃易|黃易]]小說《[[大唐雙龍傳|大唐雙龍傳]]》中，秦琼因打賭輸給沈落雁而投效瓦崗寨，武功、兵法皆頗為出色，善用雙鐧，和寇仲、徐子陵有段交情，曾暗戀呂梁派千金多年。在[[李密_(隋朝)|李密]]失勢後，一度投效王世充，後在寇仲幫助下以保護突利回國為名，轉投[[李世民|李世民]]。

相聲作品《[[關公戰秦瓊|關公戰秦瓊]]》中，利用[[关羽|关公]]与秦琼出生于不同年代而不可能相战的因素，並以[[幽默|幽默]]的形式讽刺某些人盲目指挥，成為一著名的传统[[相声|相声]]段子。同詞意者另有「岳飛戰張飛」，意思一樣。
