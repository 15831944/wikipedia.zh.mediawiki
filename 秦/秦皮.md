'''秦皮'''（[[學名|學名]]：''{{lang|la|Cortex Fraxinis}}''，英文：FRAXINI），為[[木樨科|木樨科]]落葉喬木植物[[白蠟樹|白蠟樹]]或苦櫪白蠟樹（Fraxinus rhynchophylla Hance）的樹皮，於春秋兩季剝皮曬乾可作藥，為一味常用的[[中藥|中藥]]材，載於《[[神農本草經|神農本草經]]》<ref name="《神農本草經》"> {{cite wikisource|title=《神農本草經》|date=|wslanguage=zh|wslink=神農本草經}}</ref>。 

== 生長分布 ==
秦皮生長分布於越南、朝鮮以及中國遼寧、吉林、河北、河南、内蒙古、陝西、山西、四川等，生長於海拔800米到1,600米之間。

== 成份 ==
主要含有：秦皮素、秦皮甙、馬栗樹皮素、馬栗樹皮甙等香豆精類及鞣質、皂甙。

== 性味功能 ==
味苦、澀，性寒。清熱燥濕，明目。

== 歸經 ==
色青，歸屬大腸、肝、膽經。

== 主治功效 ==
收澀而寒，能止熱痢、泄瀉，能除肝熱，止赤帶白下、目赤腫痛、目生翳障。<ref> [http://zhongyibaodian.com/archives/9617.html 中醫寶典－《本草備要》木部]</ref>

== 配伍 ==
《本草備要》<ref>[http://ctext.org/wiki.pl?if=gb&res=771254 中國哲學書電子化計劃－《本草備要》]</ref>記載：「大戟为使。恶吴茱萸。」

== 禁忌 ==
《傷寒論》<ref>[http://ctext.org/shang-han-lun/zh 中國哲學書電子化計劃－《傷寒論》]</ref>：「秦皮，苦寒。」所以脾胃虛寒者，須忌用。

== 參考文獻 ==
#《神農本草經》<ref name="《神農本草經》"/>：「秦皮，主風寒濕痹，洗洗寒氣，除熱，目中青翳白膜。」
#《本草綱目》<ref>{{cite wikisource|title=《本草綱目》|author=李時珍|date=|wslanguage=zh|wslink=本草綱目}}</ref>：「秦皮，色清氣寒，味苦性澀，乃是厥陰肝、少陽膽經藥也。故治目病、驚癇，取其平木也；治下痢、崩帶，取其收澀也；又能治男子少精、益精有子，皆取其澀而有補也。」
#《湯液本草》<ref>{{cite wikisource|title=《湯液本草》|author=王好古|date=|wslanguage=zh|wslink=湯液本草 (四庫全書本) }}</ref>：「秦皮，主熱痢下重，下焦虛。」
