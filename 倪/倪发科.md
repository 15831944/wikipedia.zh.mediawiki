'''倪发科'''（{{bd|1954年|1月22日||catIdx=N倪}}），[[江苏|江苏]][[溧水|溧水]]人。中华人民共和国政治人物。

[[中共中央党校|中共中央党校]]函授学院本科班经济管理专业毕业，中央党校研究生院[[法学|法学]]理论专业在职研究生学历。历任中共[[南陵县|南陵县]]委书记，[[芜湖市|芜湖市]]委常委、副市长、常务副市长、市委副书记，[[六安|六安]]地委副书记、行署专员，六安市委副书记、市长、市委书记、市人大常委会主任<ref>{{cite web|accessdate=2012-04-21|url=http://news.sina.com.cn/c/2005-03-03/07175251420s.shtml|title=倪发科当选六安市人大常委会主任 |publisher=[[新浪|新浪]]}}</ref>等职。2008年2月当选[[安徽省|安徽省]]人民政府副省长<ref>{{cite web|accessdate=2012-04-21 |url=http://unn.people.com.cn/BIG5/21674/21717/6849636.html |title=安徽省第十一屆人民代表大會第一次會議於2008年1月31日選舉產生安徽省人民政府省長、副省長。 |publisher=[[人民网|人民网]] |deadurl=yes |archiveurl=https://web.archive.org/web/20131005005637/http://unn.people.com.cn/BIG5/21674/21717/6849636.html |archivedate=2013年10月5日 }}</ref>。2013年1月，倪發科不再担任安徽副省长。

2013年6月4日倪发科涉嫌严重违纪正接受调查。<ref>{{cite web |url = http://www.ccdi.gov.cn/jlsc/201307/t20130713_41675.html |title = 安徽原副省长倪发科涉嫌严重违纪正接受调查 |accessdate = 2013年6月4日 |publisher = 中央纪委监察部网站}}</ref>2013年9月30日，倪发科被开除党籍和公职，其涉嫌犯罪问题移送司法机关依法处理。<ref>{{cite web |url = http://www.ccdi.gov.cn/jlsc/201309/t20130930_41736.html |title = 倪发科严重违纪违法被开除党籍和公职 |accessdate = 2013年9月30日 |publisher = 中央纪委监察部网站}}</ref>倪发科被调查后即被网民以与其名字谐音的[[Fuck|粗口]]调侃，以讽刺其践踏公共利益的行为。<ref>{{cite web|url=http://nb.zjol.com.cn/system/2013/06/06/019387240.shtml|title=发科老弟，谁的合伙人|publisher=浙江在线|date=2013-06-06}}</ref>

2014年12月15日，倪发科受贿、巨额财产来源不明一案在[[山东省|山东省]][[东营市|东营市]]中级人民法院公开审理。公诉机关指控倪发科先后49次非法收受他人给予的字画、玉石、现金等财物，共计折合人民币1300余万元。同时指控倪发科对折合人民币580余万元的财产不能说明来源，其行为构成巨额财产来源不明罪。庭审结束后，倪发科对指控的犯罪事实全部认可，表示服从法院任何判决。

2015年2月28日，倪发科被山东省[[东营市|东营市]]中级人民法院判犯受贿罪、巨额财产来源不明罪获刑有期徒刑17年，并处没收个人财产一百万元人民币。倪发科当庭表示服判不上诉。<ref>{{cite web |url = http://www.chinacourt.org/article/detail/2015/02/id/1559049.shtml |title = 安徽原副省长倪发科一审被判17年 当庭表示不上诉 |accessdate = 2015年2月28日 |publisher = 中国法院网}}</ref>

2018年3月27日，北京[[秦城监狱|秦城监狱]]以倪发科获得一次监狱表扬奖励、一次监狱嘉奖奖励为由，建议减去有期徒刑六个月<ref>{{cite web |url = http://news.sina.com.cn/c/nd/2018-04-03/doc-ifyteqtq3146979.shtml|title = 秦城监狱2名“老虎”拟减刑 1人6月底或出狱 |accessdate = 2018年4月7日 |publisher = 新京报}}</ref>。
