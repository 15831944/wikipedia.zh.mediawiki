'''倪桂珍'''（1869年6月22日—1931年7月23日），[[宋霭龄|宋霭龄]]、[[宋庆龄|宋庆龄]]、[[宋美龄|宋美龄]]三姐妹的母亲，牧师[[宋嘉澍|宋嘉澍]]的妻子。

== 生平 ==
生于江苏省[[上海县|上海县]]川沙（今[[浦东|浦东]]），[[浙江|浙江]]余姚（今[[宁波市|宁波市]][[余姚|余姚]]）倪家堰人。<ref name="'倪桂珍余姚">{{cite web |url=http://cul.cn.yahoo.com/ypen/20101020/55137.html |title=“宋家王朝”的余姚巾帼倪桂珍 |publisher=[[雅虎|雅虎]]文化 |date=2010年10月20日 |accessdate=2011年12月29日 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20130115120137/http://cul.cn.yahoo.com/ypen/20101020/55137.html |archivedate=2013年1月15日 }}</ref>1931年7月23日在[[青岛|青岛]]去世，享年62岁。

== 家庭 ==
父亲[[倪韫山|倪韫山]]也是牧师，母亲姓徐，是[[徐光启|徐光启]]的后代。倪桂珍17岁毕业于[[上海|上海]][[裨文女中|裨文女中]]，擅长数学，喜爱弹[[钢琴|钢琴]]，曾留校任教员。1887年夏与宋嘉澍结婚。先后生有6个孩子，3个女孩3个男孩，分別為靄齡、慶齡、美齡、子文、子良、子安。
