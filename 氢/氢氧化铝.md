'''氢氧化铝'''（aluminium hydroxide），化學式[[铝|Al]]([[氢氧根|OH]])<sub>3</sub>，是[[铝|铝]]的[[氢氧化物|氢氧化物]]。是一种[[碱|碱]]，由于又显一定的[[酸性|酸性]]，所以又被俗称为“铝酸”（[[氢|H]]<sub>3</sub>[[铝|Al]][[氧|O]]<sub>3</sub>　或　HAlO<sub>2</sub>·Ｈ<sub>2</sub>Ｏ），但实际与碱反应时生成的是四羟基合铝酸盐。

== 溶解性 ==

氢氧化铝不溶于[[水|水]]和[[醇|醇]]，但能溶于强[[酸|酸]]、强[[碱|碱]]，并与之反应。

== 制取 ==

=== 实验室制取 ===

在实验室中常常使用铝盐和碱反应生成氢氧化铝，但由于氢氧化铝会与强碱反应但不易与弱碱反应，因此如果使用强碱制取氢氧化铝，加入量的多少不易控制，所以通常不使用如[[氢氧化钠|氢氧化钠]]、[[氢氧化钾|氢氧化钾]]等强碱制取，而是使用[[氨水|氨水]]与铝盐反应，反应式：

: Al<sup>3+</sup> + 3NH<sub>3</sub>.H<sub>2</sub>O → Al(OH)<sub>3</sub>↓ + 3NH<sub>4</sub><sup>+</sup>

在实验室中也可以使用偏铝酸盐和酸反应制取氢氧化铝，与碱相同，氢氧化铝同样易与强酸反应但不易与弱酸反应，所以通常采用[[碳酸|碳酸]]等弱酸反应制取，反应式：

: [Al(OH)<sub>4</sub>]<sup>−</sup> + H<sup>+</sup> → H<sub>2</sub>O + Al(OH)<sub>3</sub>

== 化学反应 ==

氢氧化铝为两性物质，既可以与酸反应，又可以与碱反应。

* 与酸：氢氧化铝与强酸反应，生成[[铝|铝]]盐，反应通式为：
: Al(OH)<sub>3</sub>(s) + 3H<sup>+</sup>(aq) → Al<sup>3+</sup>(aq) + 3H<sub>2</sub>O(l)
* 与碱：氢氧化铝与强碱反应生成[[四羟基合铝酸根|四羟基合铝酸]]盐——[Al(OH)<sub>4</sub>]<sup>−</sup>。反应通式为：
: Al(OH)<sub>3</sub>(s) + OH<sup>−</sup>(aq) → [Al(OH)<sub>4</sub>]<sup>−</sup>(aq)

== 用途 ==

* 氢氧化铝可以用于生产[[氧化铝|氧化铝]]制品。
* 用于生产[[氟化铝|氟化铝]]、[[冰晶石|冰晶石]]、[[硫酸铝|硫酸铝]]等[[无机盐|无机盐]]。
* 用于制造各种[[阻燃剂|阻燃剂]]、[[防水织物|防水织物]]、[[油墨|油墨]]、[[玻璃器皿|玻璃器皿]]、[[纸张填料|纸张填料]]、[[轮滑剂|轮滑剂]]等用品。
* 氢氧化铝[[膠體溶液|膠體溶液]]具很大的[[吸附能力|吸附能力]]，能吸附[[色質|色質]]或與之化合，使色質固著於棉紗[[纖維|纖維]]，在[[染料工業|染料工業]]上被用作媒染劑。
* 氢氧化铝膠體因能[[膠結|膠結]]水中的濁質並沉澱，故用作淨水劑。為一相當常用於除去水中雜質的膠狀[[凝聚劑|凝聚劑]]，故於自來水中添加[[明礬|明礬]]可促使[[鋁|鋁]][[離子|離子]]Al<sup>3+</sup>水解成[[氫氧化鋁|氫氧化鋁]][[膠體|膠體]]吸附雜質，再利用簡單的過濾設備將氫氧化鋁及其吸附的雜質過濾掉，即可完成自來水的初步過濾。
{{medical}}
* 氢氧化鋁是一种历史悠久的抗酸药，能抗酸、吸着、局部止血和保护溃疡面等作用，作用缓慢、持久，但效果较弱。常用于作[[中和反应|中和]][[胃酸|胃酸]]的药物，是[[胃舒平|胃舒平]]的有效成分。

== [[毒性|毒性]] ==

氢氧化铝的毒性一般表现为[[铝|铝]]的毒性，但氢氧化铝更容易引起[[肺泡|肺泡]]上皮增生。
