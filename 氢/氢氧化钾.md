'''氢氧化钾'''（化学式：[[钾|K]][[氧|O]][[氢|H]]），俗稱苛性鉀，白色固体，溶于[[水|水]]、[[醇|醇]]，但不溶于[[醚|醚]]。在[[空气|空气]]中极易吸湿而[[潮解|潮解]]。可与[[二氧化碳|二氧化碳]]反应生成[[碳酸钾|碳酸钾]]。所以它會被用作吸收二氧化碳之用。

氢氧化钾是典型的[[強鹼|強鹼]]，有許多工業上的應用，大部份的應用都是因為氢氧化钾可以和[[酸|酸]]反應，以及氢氧化钾本身的腐蝕性。2005年生產的氢氧化钾估計有700,000至 800,000[[噸|噸]]，約為[[氢氧化鈉|氢氧化鈉]]的百分之一<ref name=Ullmann>H. Schultz, G. Bauer, E. Schachl, F. Hagedorn, P. Schmittinger “Potassium Compounds” in Ullmann’s Encyclopedia of Industrial Chemistry, 2005, Wiley-VCH, Weinheim. {{DOI|10.1002/14356007.a22_039}}</ref>。氢氧化钾常用來製造軟肥皂和液體肥皂，也用來製備含鉀的化學物質。

== 制备 ==
可從電解飽和的[[氯化鉀|氯化鉀]]水溶液製得。
:<chem>2KCl + 2H2O = 2KOH + Cl2 ^ + H2 ^</chem>

古法會用[[碳酸钙|碳酸钙]]加熱，再和水及草木灰反應，生成氢氧化钾。

先將碳酸钙高温加热碳酸钙生成[[氧化钙|氧化钙]]
:<Math> \rm CaCO_3 {=\!=\!=\!=} CaO + CO_2\uparrow</Math>

再將氧化钙与水反应生成[[氢氧化钙|氢氧化钙]]
:<chem>CaO + H2O = Ca(OH)2</chem>

最後將氢氧化钙与草木灰反应生成氢氧化钾
:<chem>Ca(OH)2 + K2CO3 = CaCO3 v + 2KOH</chem>

事實上，水与鉀反應也會生成氫氧化鉀，但反應十分劇烈，不宜使用：

:<Math>\rm 2K + 2H_2O {=\!=\!=\!=} 2KOH + H_2\uparrow</Math>

== 物理性质 ==
氢氧化钾是白色吸湿性固体，高于熔点极易升华。<ref name=prdc /> 极易溶于水和乙醇。固體氫氧化鉀為白色正交結晶，市售品有塊狀、小颗粒状和片狀。

== 安全性 ==
有强烈腐蚀性，溶於水放出大量熱。吸入后强烈刺激呼吸道或造成灼伤。皮肤和眼直接接触可引起灼伤，一旦眼睛或皮膚接觸到氫氧化鉀，迅速將受傷部位以水不斷沖洗15 分鐘以上；口服會灼伤消化道，可致命。

==用途==
氢氧化钾用作生產鉀鹽，如[[高錳酸鉀|高錳酸鉀]]、[[亞硝酸鉀|亞硝酸鉀]]、[[磷酸氫二鉀|磷酸氫二鉀]]等。日化工業用作製造肥皂。電池工業用於製造[[鹼性電池|鹼性電池]]的[[電解質|電解質]]。
