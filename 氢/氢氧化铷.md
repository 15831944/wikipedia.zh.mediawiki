'''氫氧化銣'''（'''RbOH'''），一種強[[碱|碱]]，較[[氫氧化鉀|氫氧化鉀]]的鹼性強<ref name="ChemYQ">{{zh}}[http://www.chemyq.com/xz/xz5/49803thlkj.htm 化工词典 氢氧化铷]</ref>。它由[[銣離子|銣離子]]和[[氫氧根離子|氫氧根離子]]按1：1的物质的量之比构成。<!-- Rubidium hydroxide does not appear in nature. However it can be obtained by [[Chemical_synthesis|synthesis]] from [[rubidium_oxide|rubidium oxide]]. In addition, rubidium hydroxide is commercially available in form of an [[aqueous_solution|aqueous solution]] from a few suppliers. -->

氫氧化銣具強[[腐蝕性|腐蝕性]]，因此使用時應穿著適當的[[防護服|防護服]]、[[手套|手套]]和[[安全眼鏡|安全眼鏡]]。

== 性質 ==
氫氧化銣是具有強[[潮解|潮解性]]的灰白色固體，由[[銣|銣]]離子和[[氫氧根離子|氫氧根離子]]組成。

氢氧化铷能和空氣中的二氧化碳反應，生成[[碳酸铷|碳酸铷]]和水。<ref>{{zh}}[http://www.dictall.com/indu/397/3965201F33B.htm dictall.com 氢氧化铷<冶>]</ref>

:<math>\rm 2 RbOH + CO_2 \ \longrightarrow \ Rb_2CO_3 + H_2O</math>

== 製備 ==

[[氧化銣|氧化銣]]溶於水可生成氫氧化銣：

: <math>\rm Rb_2O + H_2O \longrightarrow 2 RbOH</math>

[[硫酸銣|硫酸銣]]和[[氫氧化鋇|氫氧化鋇]]反應，生成[[硫酸鋇|硫酸鋇]]沉澱和氢氧化铷。過濾後為含氫氧化銣的母液，經蒸發、濃縮、結晶乾燥，得到氫氧化銣。<ref name="ChemYQ" />

: <math>\rm Rb_2SO_4 + Ba(OH)_2 \longrightarrow 2 RbOH + BaSO_4 \downarrow</math>

== 用途 ==

氫氧化銣很少用於工業生產，因為价格昂贵而且主要用途可以由[[氫氧化鉀|氫氧化鉀]]和[[氫氧化鈉|氫氧化鈉]]來取代，後者既反應溫和且操作安全。

氫氧化銣主要用於科學研究，不過往往是小心使用，以免浪費昂貴的[[銣|銣]]元素。

== 安全性 ==

氫氧化銣具有強[[腐蝕性|腐蝕性]]，接觸會被[[灼傷|灼傷]]，因此必須慎重處理，使用時應穿著適當保護衣物。

實驗室中，防護服、手套和眼－臉部防護都必須是抗鹼材料，從而防止氫氧化銣在實驗中發生[[滲漏|滲漏]]而腐蝕皮膚。

[[稀釋|稀釋]]氫氧化銣必須小心地將少量的氫氧化物加入燒杯內的水中。此外，在化學實驗中操作這種化合物必須謹慎，因為該過程為[[放熱反應|放熱反應]]，需避免因釋放出大量的熱量而引起溶液的爆沸或損壞反應裝置。
