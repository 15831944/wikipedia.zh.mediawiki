'''氫氧化鋇'''是一種[[鋇|鋇]]的[[化合物|化合物]]，[[分子式|分子式]]為Ba(OH)<sub>2</sub>，通常外觀為白色顆粒狀。

== 用途 ==

氫氧化鋇在[[分析化學|分析化學]]中的[[滴定|滴定法]]可以當做[[弱酸|弱酸]]的[[指示劑|指示劑]]，對[[有機|有機]]的弱酸別有效。它透明的[[溶液|溶液]]是不含[[碳酸|碳酸]]鹽的。不像[[氫氧化鈉|氫氧化鈉]]和[[氫氧化鉀|氫氧化鉀]]，它的[[溶解性|溶解度]]很低，所以這讓它可以用來測試像是[[酚酞|酚酞]]或[[百里酚酞|百里酚酞]]之類的[[物質|物質]]，而不會因為有[[鹼性|鹼性]]的[[碳酸|碳酸]]鹽[[離子|離子]]存在，而產生錯誤。

在有机合成之中，氢氧化钡也对[[脂类|脂类]]和[[腈|腈]]类的[[水解|水解]]起催化作用。

它也可以用来处理酸泄漏事故。

== 製造方法 ==

直接把[[氧化鋇|氧化鋇]]（BaO）和[[水|水]]混合來製造氫氧化鋇：

:<chem>BaO + 9H2O =  Ba(OH)2* 8H2O</chem>

== 相關反應 ==
氢氧化钡与[[氯化铵|氯化铵]]研磨可放出[[氨气|氨气]]，此方法可用于检验[[铵盐|铵盐]]。

<chem>Ba(OH)2 * 8H2O + 2NH4Cl = BaCl2 + 2NH3 + 10H2O</chem>

<math>\Delta H=80.3</math>KJ(吸熱反應)
== 安全性 ==
类似于其他的[[强碱|强碱]]，氢氧化钡具有[[腐蚀性|腐蚀性]]。
