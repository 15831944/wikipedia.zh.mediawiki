'''沈冰'''{{BD|1976年|7月23日|||catIdx= Shen沈}}出生于[[江苏省|江苏省]][[张家港市|张家港市]]，毕业于[[浙江大学|浙江大学]]和[[南洋理工大学|南洋理工大学]]，原[[中国中央电视台|中国中央电视台]]主持人<ref name="bee" >{{cite web |url = http://www.zhuchijie.com/show_159_z7_t1.html |title = 沈冰 |publisher = 主持界 |accessdate = 2011年 |work = |language = }}{{Dead link}}</ref>、[[中央政法委|中央政法委]]信息中心副主任。

== 简介 ==

== 其它 ==

香港飆笛出版社出版了一本名为《沈冰自述——我和周永康的故事》的书<ref>[http://cwbbooks.com/products.php?product=%E3%80%8A%E6%B2%88%E5%86%B0%E8%87%AA%E8%BF%B0%E2%80%94%E2%80%94%E6%88%91%E5%92%8C%E5%91%A8%E6%B0%B8%E5%BA%B7%E7%9A%84%E6%95%85%E4%BA%8B%E3%80%8B 《沈冰自述——我和周永康的故事》]</ref>，但此書的真實性從一開始就被人懷疑，尤其是因為此書據稱是沈冰在拘留期間所寫的，書中所提及的事實也有不少錯誤（如重要事件的發生日期）。此外，這本書的香港代理商拒絕透露書中談話內容是如何得來的，也拒絕透露這本書的編輯和印刷過程。但是部份中國政界人士卻認為這本書即使並非全屬真實，但大部份內容都是可信的，因為書中的部份細節只有深入了解中國政治和最高領導人生活的人士才能提供。<ref>[http://www.nanzao.com/tc/opinion/47777/wang-xiang-wei-zhuan-lan-cong-chen-bing-zi-shu-shuo-qi 王向偉專欄：從沈冰自述說起 | 觀點 | 香港南華早報]</ref>
