'''沈阳罢市事件'''，是2012年7月至8月辽宁省[[沈阳市|沈阳市]]及周边城市的部分商铺出现了同一时间集体关门歇业的现象。<ref name="大众网报道">{{cite web | url=http://www.dzwww.com/xinwen/guoneixinwen/201208/t20120807_7314927.htm | title=沈阳部分商铺因工商局“打假”传言纷纷歇业 | publisher=大众网 | accessdate=2012-08-07}}</ref>最早2012年7月13日，沈阳市东北市场、五爱市场等开始出现大量商铺关门歇业的现象。此后愈演愈烈，以至于沈阳全城商铺大多停业。到8月6日，罢市事件已经蔓延到沈阳街头的大部分超市、杂货店、饭店、理发店、澡堂等各种商铺，大量商铺白日关门照片涌上网络引发关注。此外，罢市现象也扩散到辽宁省[[鞍山|鞍山]]、[[抚顺|抚顺]]、[[本溪|本溪]]、[[大连|大连]]等地，而罢市的原因也大致相同。

== 事件过程 ==
2012年1月13日，沈阳市人民政府曾召开2012年打假工作会议部署“[[2012年沈阳市打假专项活动|2012年沈阳市打假专项活动]]”，并要求打假活动到[[中华人民共和国第十二届运动会|2013年全运会]]结束。据[[联合早报|联合早报]]、[[BBC|BBC]]等媒体转述的一则流传度甚广的传闻透露：辽宁省沈阳市承办[[中华人民共和国第十二届运动会|中华人民共和国第十二届运动会]]，花费过大导致因经费不足，遂决定由公安、工商、卫生、消防、城管等多部门组成综合执法队开展“打假”行动以罚款的方式敛财，专门瞄准有一定规模又没有背景的弱势店铺，尤其是外地商铺，只要被他们盯上，他们就会以各种理由连人带货带走，逼迫当事人缴付1万至50万元不等的罚款才放人。报道中还，必须证照齐全才能躲过一劫，诸如：卖牙签的商铺，因为没有[[森林采伐许可证|森林采伐许可证]]而被罚款数万。<ref>{{cite web | url=http://www.zaobao.com/special/china/cnpol/pages6/cnpol120808.shtml | title=商铺躲避大检查 沈阳“打假”引发关门现象 | publisher=联合早报 | accessdate=2012-08-08}}</ref><ref>{{cite web | url=http://www.bbc.co.uk/zhongwen/simp/chinese_news/2012/08/120806_weibo_china_shop_strike.shtml | title=沈阳大量商家不堪重罚罢市引热议 | publisher=BBC中文网 | accessdate=2012-08-10}}</ref>

对于传闻，起初多地政府未予证实，仅[[营口市|营口市]]予以辟谣。<ref name="大众网报道" />因而罢市事件愈演愈烈。与沈阳市区相比，罢市现象在沈阳的一些郊县更严重。在[[辽中县|辽中县]]等地，由于绝大部分的店铺关门，甚至粮油等物资都无从购买。根据[[沈阳电视台|沈阳电视台]]消息，都是店家听传言说会“打假”或大检查，要“五证齐全”、“六证齐全”甚至“七证齐全”才能幸免，大家都不想被罚款因而决定歇业。罢市的商铺，大多贴上“外出旅游、暂停营业”等告示。2012年7月31日《[[鞍山日报|鞍山日报]]》报道称，[[鞍山市|鞍山市]]当地的电子市场、建材市场等因“保护[[知识产权|知识产权]]、逃避暗访”出现大面积商户停业现象。<ref name="大众网报道" />2012年8月2日《[[辽沈晚报|辽沈晚报]]》报道称，[[丹东市|丹东市]]当地规模最大的电子市场——华美科技大厦因“打假”而歇业。<ref name="大众网报道" />

== 官方处理及效果 ==


此次事件的谣言有多种版本，但大部分与2013年沈阳召开的[[中华人民共和国第十二届运动会|中华人民共和国第十二届运动会]]有关。沈阳市内包括[[五爱市场|五爱市场]]、[[三好街|三好街]]、[[大西电子城|大西电子城]]、[[南塔鞋城|南塔鞋城]]、[[南湖五金市场|南湖五金市场]]、[[滑翔五金城|滑翔五金城]]等大型市场中的商铺有很多都关门歇业，城市中的沿街商铺也纷纷关门，靠近郊区的地方关门的现象比市中心严重。沈阳市所辖的[[辽中县|辽中县]]情况更加严重，有95%的商铺歇业。<ref>http://news.qq.com/a/20120807/000352.htm</ref>关门歇业的情况波及到[[大连市|大连市]]的胜利地下商业区等，以及鞍山、丹东等辽宁省的其他城市。

== 官方辟谣 ==
8月3日，营口晚报：切勿偏听偏信小道消息，保持正常市场经营秩序<ref>{{cite web |url=http://news.idoican.com.cn/ykouwb/html/2012-08/03/content_4179589.htm?div=-1 |title=存档副本 |accessdate=2012-08-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20120807113222/http://news.idoican.com.cn/ykouwb/html/2012-08/03/content_4179589.htm?div=-1 |archivedate=2012-08-07 }}</ref>
8月7日，辽宁日报：勿信传言，正常营业<ref>http://epaper.lnd.com.cn/lnrb/20120807/index.htm{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

在沈阳罢市事件引发国内外关注之后，[[辽宁省|辽宁省]]和[[沈阳市|沈阳市]]当局才开始采取措施。当地政府一方面表示引起罢市的大规模“打假”已经告一段落，另一方面对于借机敛财的指责予以否认，并称各种传闻纯属谣言。8月7日早晨，中共沈阳市委宣传部在其[[新浪微博|新浪微博]]官方账号“沈阳发布”发布公告：
{{quote|近期，一些经营业户因受不实传言影响，关门停业。经了解，相关部门除正常管理工作外，并未采取集中整治行动，更未采取高额罚款措施。希望广大业户勿信传言，正常营业。}}

8月7日傍晚，“沈阳发布”再度发表声明称：
{{quote|市政府对此高度重视，本着高度负责的态度重申，各级政府必须严格坚持依法行政，保障经营业户合法权益，维护市场经营秩序。对乱检查、滥罚款等违规执法、粗暴执法行为，市民可通过96123市政府监督投诉电话举报，一经查实将依法依纪严肃处理，并追究相关单位和责任人的责任。同时希望经营业户消除顾虑，放心营业。”<ref>{{cite web| url=http://news.gog.com.cn/system/2012/08/08/011583751.shtml| title=沈阳回应商铺歇业：严肃处理滥罚款行为| publisher=金黔新闻| accessdate=2012-08-10| deadurl=yes| archiveurl=https://web.archive.org/web/20130521112050/http://news.gog.com.cn/system/2012/08/08/011583751.shtml| archivedate=2013-05-21}}</ref>}}

2012年8月7日，当地派出所贴出通知：
{{quote|关于前一阶段为配合沈阳市政府召开2012年打假专项活动现已告一段落。根据市政府昨日8月6日召开紧急会议精神对各大市场停止一切检查，望各业户正常营业。<ref name="沈阳通知商户停止打假活动">{{cite web | url=http://www.bbc.co.uk/zhongwen/simp/chinese_news/2012/08/120809_shenyang_shops.shtml | title=沈阳通知商户停止打假活动 | publisher=BBC中文网 | accessdate=2012-08-09}}</ref>}}

同一时间，当地的官方媒体刊登《勿信传言 正常营业》的公告，呼吁当地商家尽快结束罢市活动、恢复营业。通知发出后，当地商铺虽然仍有部分商铺继续歇业，但亦有部分商铺逐步恢复营业。<ref>{{cite web | url=http://www.bbc.co.uk/zhongwen/simp/chinese_news/2012/08/120807_shenyang_bbc.shtml | title=当局通知打假结束后沈阳商店仍关门 | publisher=BBC中文网 | accessdate=2012-08-10}}</ref><ref name="沈阳通知商户停止打假活动" />
