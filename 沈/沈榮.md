'''沈榮'''（{{Bd|1904年|11月1日|1975年|11月7日}}<ref name=":0" />），[[表字|字]]'''公甫'''，是一位出身臺南[[新營|新營]]的[[律師|律師]]及[[政治人物|政治人物]]。他曾在[[臺灣日治時期|臺灣日治時期]]為農民及地主爭取權益，也曾於二二八事件後為被告辯護。

== 生平 ==

=== 早年生活 ===
[[雍正|雍正]]年間，沈參自福建省[[詔安縣|詔安縣]]前往臺南新營定居，並於其後陸續生下六個[[兒子|兒子]]，六個兒子又各自[[繁衍|繁衍]]後代，並有稱號「六順」。

沈榮出生於1904年11月1日，並為二[[房|房]]後裔沈森其的兒子。他的[[父親|父親]]沈森其以在新營經營[[農業|農業]]及[[商業|商業]]為生，並通曉[[十三經|經書]]及[[文言文|詩文]]寫作，又喜愛彈奏傳統[[弦樂器|弦樂器]]、[[下棋|下棋]]、欣賞[[書畫|書畫]]、[[建築設計|設計建築]]、[[騎馬|騎馬]]等；他的[[母親|母親]]是[[蘇姓|蘇氏]]，以[[三從四德|輔佐丈夫及教育子女]]聞名當地，並曾當選模範母親。他有三位[[兄弟|兄弟]]，又有[[姊妹|姊妹]]各一位，而他自己為第三位出生者；但是，他的兄長與姊姊很早就過世了。他的三弟[[沈乃霖|沈乃霖]]<ref name=":3">{{Cite web|url=http://memory.ncl.edu.tw/tm_cgi/hypage.cgi?HYPAGE=toolbox_figure_detail.hpg&project_id=&dtd_id=&subject_name=&subject_url=&xml_id=0000293639&who=%E6%B2%88%E6%A6%AE|title=沈榮|accessdate=2017-10-21|author=郭啟|date=|work=臺灣記憶 Taiwan Memory|publisher=國家圖書館|language=zh-TW}}</ref>因[[叔父|叔父]]沈擇仁無[[子女|子女]]而於四歲時[[入籍|入籍]]其下為[[過房|過房]]子，後就讀於{{Link-ja|昭和醫學專門學校|昭和大学}}，又於稍後取得[[東京帝國大學|東京帝國大學]][[醫學博士|醫學博士]]學位。他的妹妹沈碧雲[[畢業|畢業]]於[[私立台北女子高等學院|私立臺北女子高等學院]]，後與醫學博士鄧燈[[結婚|結婚]] 。

[[童年|幼年]]時，沈榮入[[臺南市鹽水區鹽水國民小學|鹽水公學校]]就讀，又於[[畢業|畢業]]後考入[[臺中市立臺中第一高級中等學校|臺中州立臺中高等普通學校]]。就讀[[中學|中學]]時，沈榮原計畫於畢業後前往[[日本|日本]]攻讀[[醫學|醫學]]；但是，在1924年[[治警事件|治警事件]]發生後，他心生願望欲從[[法律制度|法制]]層面提升[[臺灣人|臺灣人]]的[[政治|政治]]及[[社會地位|社會地位]]，又欲以其為眾人爭取權益。於是，沈榮在畢業後入[[日本大學|日本大學]]攻讀[[法律系|法律]]<ref name=":3" /><ref name=":4">{{Cite book|url=https://books.google.com.tw/books?id=SHrxBQAAQBAJ&pg=PA216&lpg=PA216&dq=%E6%B2%88%E6%A6%AE%E5%BE%8B%E5%B8%AB&source=bl&ots=esddvKxJdb&sig=knT4CFPCVTwbl6AeeND8Wu4uNqs&hl=zh-TW&sa=X&ved=0ahUKEwiQ2drjl4LXAhUIGpQKHQFSCvA4ChDoAQhNMAg#v=onepage&q=%E6%B2%88%E6%A6%AE%E5%BE%8B%E5%B8%AB&f=false|title=總力戰與臺灣: 日本殖民地的崩潰（上）（下）|last=近藤正己|first=|date=2014-09-01|publisher=國立臺灣大學出版中心|year=|isbn=9789863500322|location=|pages=|language=zh-TW|chapter=}}</ref>。<ref name=":2">{{Cite web|url=http://www.cy.gov.tw/sp.asp?xdUrl=./di/member/gen_detail.asp&membertype=1&ctNode=879&sid=119|title=監察院─第1屆監察委員沈榮先生|accessdate=2017-10-21|date=00:00|work=監察院全球資訊網|language=zh-TW}}</ref>

1930年（[[昭和|昭和]]五年），沈榮參加{{Link-ja|高等文官司法科考試|高等文官試験}}合格，但最後決定放棄擔任判事職務的機會。[[1931年臺灣|1931年]]，沈榮返回[[臺南|臺南]]擔任[[律師|辯護士]]<ref name=":4" />職務，並將[[事務所|事務所]]<ref name=":3" />設於日後忠義路二段14號地址處（近[[原日本勸業銀行臺南支店|原日本勸業銀行臺南支店]]等機構）。<ref name=":0">{{Cite journal|title=一代名律師沈榮|author=謝碧連律師|url=|journal=台南律師通訊|issue=八十九|doi=|others=|year=|volume=|page=|pmid=}}</ref>

1934年及1935年，沈榮當選為[[1935年臺灣市會及街庄協議會員選舉|臺南市協議會員及市會議員]]。<ref>{{Cite book|title=台灣士紳名鑑|last=|first=|publisher=新高新報社|year=1937|isbn=|location=|pages=58|language=ja}}</ref><ref>{{Cite book|url=https://books.google.co.jp/books?id=7HpfmgEACAAJ&dq=%E5%8F%B0%E7%81%A3%E4%BA%BA%E5%A3%AB%E9%91%91&hl=zh-TW&sa=X&ved=0ahUKEwiPmrW_0q7XAhVIWLwKHZbrB_MQ6AEIJjAA|title=台灣人士鑑|last=|first=|date=1943|publisher=[[興南新聞|興南新聞]]社|year=|isbn=|location=|pages=|language=zh|chapter=}}</ref>[[File:Shen_Rong_in_court_dress.jpg|缩略图]]之肖像，攝影於1930年代。|281x281像素]]

=== 職業生涯 ===

==== 日治末期 ====

===== 柳營火犁事件 =====
1935年1月12日9時，[[鹽水港製糖|鹽水港製糖]]會社強行以深耕[[犁|犁]]鋤耕[[柳營庄|柳營庄]]柳營一一五番地等三筆[[田地|土地]]，引發耕作[[農民|農民]]包圍並[[抗議|抗議]]阻撓。隨後，郡[[臺灣總督府警察|警察]]課派員將現場[[群眾|民眾]]32人以[[暴力|暴力]]、[[妨礙公務|妨礙業務]]等事由逮捕[[拘留|拘留]]十一[[日|日]]，[[地主|地主]]劉明電也被以具[[教唆犯|教唆]]嫌疑為由逮捕拘留，並不准其[[家屬|家屬]]送[[食品|食品]]及禦寒[[衣物|衣物]]至[[拘留所|拘留所]]，後又不遵[[檢察官|檢察官]]所頒釋放[[命令|命令]]，另以[[行政處分|行政裁決]]（檢束）延長留置數日。時任[[臺灣總督府|臺灣總督府]]水產課長的[[劉明朝|劉明朝]]因無法忍受[[胞弟|胞弟]]劉明電遭如此[[虐待|虐待]]，於是向[[臺灣辯護士協會|臺灣辯護士協會]]申訴；時任協會[[理事会|理事]]的沈榮因而積極遊說協會中眾多[[日本人|日籍]]會員，組織特別委員會進行調查確認其為嚴重侵害[[人權|人權]]之事實，並分批拜會臺灣總督[[中川健藏|中川健藏]]及其內務局長、[[臺灣高等法院#日治時期|高等法院]]檢察官長及臺灣總督府警務局長。同年2月15日，臺灣辯護士協會更發表書面聲明譴責警方侵害人權；隨後，警方終於承認其過失，並同意會社非經地主、耕作人同意不得任意開發田地。<ref name=":0" /><ref name=":1">{{Cite web|url=http://bbs.nsysu.edu.tw/txtVersion/treasure/kmt/M.1082194688.A/M.1083231760.A/M.1083243659.A/M.1083243674.E.html|title=從沈榮的際遇談日據時期的殖民地法制|accessdate=2017-10-21|author=黃靜嘉|date=|work=bbs.nsysu.edu.tw|publisher=}}</ref>

===== 其他事件 =====
1937年，沈榮受一逃脫[[未成年|未成年]]鄭姓[[酒家女|藝妓]]及其連帶保証人委託，在[[臺南地方法院|臺南地方法院]]與其前[[雇主|雇主]]的[[訴訟代理人|訴訟代理人]]針對[[薪酬|酬金]]應返還一事展開攻防，最終以其未成年而違反善良[[風俗|風俗]]應屬無效為由獲得勝訴[[判決|判決]]。

1940年，沈榮出任經營臺南信用組合（二戰戰後改組為臺南市第一信用合作社）直至1975年，並歷任[[臺灣省合作金庫|臺灣省合作金庫]]理事、全國合作社聯合社理事主席、中國合作學社常務理事、中華民國合作事業協會理事<ref name=":2" />長等職位。

[[皇民化運動|1943年]]，李欽明、黃梱、黃嘉宗、李啟明、李明振、顏老牛、林春泰等[[公學校|公學校]]教師藉訓話時機灌輸學生[[民族意識|民族意識]]，又結合在[[東京|東京]]的[[台灣人|臺籍]]抗日學生[[秘密結社|秘密結社]]且於各地設置分支，俟機[[暴動|暴動]]推翻日本統治；隨後，該案經官警舉發，檢察官提起[[公訴|公訴]]請求判以[[死刑|死刑]]或[[無期徒刑|無期徒刑]]。在庭上，[[被告|被告]]均坦承公訴所指事實，並爭相為其他被告擔負其罪刑；沈榮隨即以辯護人身分向[[法官|法官]]訴諸以情，被告皆終被判以[[有期徒刑|有期徒刑]]十二年。<ref name=":0" /><ref name=":1" />

==== 戰後時期 ====
沈榮在[[國民政府接收臺灣|國民政府接收臺灣]]後隨即加入[[中國國民黨|中國國民黨]]，並擔任該黨臺南市委員會財務委員職務。<ref name=":0" />

===== 黃小林案 =====
[[1947年臺灣|1947年]][[3月2日|3月2日]]，黃小林召集[[地痞|地痞]]衝入[[新町|新町]][[派出所|派出所]]破壞[[電話|電話]]並奪取[[短槍|短槍]]二支、[[子彈|子彈]]三十餘發。3月3日，黃小林又與同夥乘[[汽車|車]]至財政部[[臺南鹽場|灣裡鹽場]]事務所要求駐守鹽警繳出[[短刀|短刀]]，後於[[下午|下午]]包圍[[臺南監獄|臺南監獄]]並衝入以槍械脅請典獄長領其至警衛科槍庫房，又破壞其中槍櫃並劫得[[步槍|步槍]]八支、[[手槍|手槍]]一支、子彈共四百十八發、尖尾刀及[[日本刀|日本刀]]共十二把。但是，黃小林同時又擔任[[二二八事件處理委員會列表|二二八事件處理委員會]]臺南市分會[[治安|治安]]小組小隊長，復於3月7日率領眾人佔據新町派出所。

事後，[[臺南地方法院|臺南地方法院]]以[[共同犯罪|共同]][[連續|連續]]強佔強劫公署為[[罪名|罪名]]將黃小林判處死刑並[[褫奪公權|褫奪公權]]終身；沈榮與黃百祿、柯南獻三位共同辯護人[[上訴|上訴]]至第三審[[最高法院_(中華民國)|最高法院]]，並因[[審判權與管轄權_(刑事訴訟)|管轄權]]歸屬問題獲判決撤銷原判發回臺南地方法院更審。隨後，臺南地方法院經調查根據[[證據|證據]]及證詞，認定派出所槍彈事件非其所為，且於警員避難去時接管派出所乃為維持治安，又入典獄長室之代表後經市參議會派員勸阻而離去，僅考慮其公然聚眾進行暴力脅迫行為而判以有期徒刑二年六月；當時，黃小林因被[[羈押|羈押]]時日已超過宣判刑期而被立即釋放。<ref name=":0" />

===== 教唆幫助自殺案 =====
1951年8月8日，一位任教[[臺灣省立工學院|臺灣省立工學院]]的[[婦女|女]]講師跳入[[日月潭|日月潭]]自溺而死；事後，其遺留之公開信及[[日記|日記]]顯示其曾遭已婚[[校長|校長]]誘騙交往，使其[[名譽|聲名]]嚴重受損並多次嘗試[[自殺|自殺]]，復於7月12日於辦公室以言詞[[羞辱|羞辱]]死者。隨後，該校長被以[[教唆犯|教唆]]、幫助自殺等罪責起訴，兼任於該校的沈榮<ref name=":2" />與汪峻、[[端木愷|端木愷]]為該校長擔任共同辯護人，並善用法理而最終使其得以免於[[刑責|刑責]]。<ref name=":0" />

=== 晚年生活 ===
1969年，沈榮於[[中國國民黨第十次全國代表大會|中國國民黨第十次全國代表大會]]當選代表，並獲任該黨臺灣省委員會委員；當時，總統[[蔣中正|蔣中正]]以其積極效力地方事業兩度召見勉勵，他也一再應邀參加陽明山會談。隨後，[[中國國民黨中央委員會|該黨中央黨部]]更多次欲提名沈榮參選中央民意代表或擔任重要職務，但皆遭其婉拒。

1973年，沈榮在中國國民黨一再促請下被提名為臺灣省增額[[監察委員|監察委員]]，並在[[臺灣省議會|臺灣省議會]]高票當選。1974年，沈榮獲選為監察院司法委員會召集人。隨後，沈榮因血壓過高入[[臺大醫院|臺大醫院]]接受診斷，獲知腎臟舊病復發。

1975年11月7日11時42分，沈榮因[[腎病|腎病]]醫治無效[[逝世|逝世]]。

此外，沈榮也長年擔任臺南律師公會常務理事、中華民國律師公會全國聯合會常務理事<ref name=":2" />等職位，歷任[[中華民國紅十字會|中華民國紅十字會]]臺南市支會長、臺南[[扶輪社|扶輪社]]社長、臺南市文史協會理事長，兼任[[淡江大學|淡江大學]]董事、[[南英高級商工職業學校|南英高級商工職業學校]]董事長、[[光華女中|光華女中]]董事長、[[南榮科技大學|南榮工專]]董事長等職位，更曾加入[[臺灣地方自治聯盟|臺灣地方自治聯盟]]並受提名[[1935年臺灣市會及街庄協議會員選舉|當選為臺南市會議員]]<ref name=":4" /><ref>{{Cite web|url=http://www.tncchistory.com.tw/history/tncc_a-1.html#overview-04|title=台南市歷屆議會文史|accessdate=2017-10-21|work=www.tncchistory.com.tw}}</ref>。<ref name=":0" />

== 家庭 ==
沈榮的[[妻子|妻子]]是蘇諒（1904年 - 1996年），她曾歷任臺南市婦女會理事長、臺灣省婦女會理事、[[臺南市議會|臺南市議會]]第一屆議員等職位，並與沈榮育有五個[[女兒|女兒]]：蕙香、佩香、薰香、吟香及妙香。長女蕙香畢業於[[臺南第一高等女子學校|臺南第一高等女子學校]]並與醫學博士及高雄醫學大學教授林本仁[[結婚|結婚]]；次女佩香自日本大學取得碩士學位，後任律師並任教於國立成功大學及淡江大學，並與法學博士及[[國立臺灣大學|國立臺灣大學]][[教授|教授]]林福順結婚；三女、四女留學[[美國|美國]]，復與[[工程師|工程師]]林兆京、工學博士江牧東結婚；五女畢業於淡江大學，後任[[景美女中|景美女中]]教師，並與[[臺灣氰胺|臺灣氰胺]]經理及[[科見美語|科見美語]]創辦人侯登見結婚。沈榮家族人才輩出，外孫[[江俊輝|江俊輝]]任加州州政府財務長，外孫林克忠任[[國立臺灣大學|國立臺灣大學]]教授並為首位入選美國職能治療研究院院士的亞洲學者。<ref>{{Cite web|url=http://www.taipeitku.org.tw/?pagekey=learn_biographyinfo&seq=10|title=社團法人台北市淡江大學校友會|accessdate=2017-10-21|work=www.taipeitku.org.tw}}</ref>。<ref name=":0" />

== 家族 ==
