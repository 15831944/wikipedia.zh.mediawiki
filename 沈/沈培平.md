'''沈培平'''（{{bd|1962年|2月||catIdx=S沈}}），[[云南|云南]][[施甸|施甸]]人。中华人民共和国政治人物。

[[中国共产党|中国共产党]]党员。[[云南省|云南省]][[保山地区|保山地区]][[保山学院|保山师范专科学校]]（今[[保山学院|保山学院]]）中文系中文专业毕业，[[北京师范大学|北京师范大学]]资源学院自然地理学专业在职研究生学历，理学博士。历任云南省[[腾冲县|腾冲县]]委书记，[[保山市|保山市]]委常委；省政府副秘书长；[[思茅市|思茅市]]委副书记、代市长、市长，[[普洱市|普洱市]]委副书记、市长、市委书记等职。2013年1月当选[[云南省人民政府|云南省人民政府]]副省长<ref>{{cite web|accessdate=2013-05-01|url=http://yn.yunnan.cn/html/2013-01/28/content_2596584_2.htm|title=云南省十二届人大一次会议公告|publisher=云南网|archive-url=https://web.archive.org/web/20130522083044/http://yn.yunnan.cn/html/2013-01/28/content_2596584_2.htm|archive-date=2013-05-22|dead-url=yes}}</ref>。2014年3月，云南省副省长涉嫌严重违纪违法被[[中纪委|中纪委]]调查<ref>{{cite news|title=云南省副省长沈培平涉嫌严重违纪违法被调查 |url=http://news.163.com/14/0309/08/9MSP1LSM0001124J.html|accessdate=2014-03-09|newspaper=网易}}</ref>。同年8月被双开<ref>{{cite web |url = http://www.ccdi.gov.cn/yw/201408/t20140806_37450.html |title = 云南省原副省长沈培平严重违纪违法被开除党籍 |accessdate = 2014年8月6日 |publisher = 中央纪委监察部网站}}</ref>。

==人物事件==
=== 将思茅市更名为普洱市 ===
沈培平主政思茅期间，因在任内大力推广[[普洱茶|普洱茶]]，而被成为“茶市长”。另外，他还将“思茅市”改名为“普洱市”，号称要“让每个喝普洱的人都能想到这座城”<ref name=tkp>{{cite web|title=拆迁市长们的诨号：拆迁大佐、李拆城以及季挖挖|url=http://news.takungpao.com/mainland/focus/2014-03/2388732.html|accessdate=2014-03-30|publisher=大公网}}</ref>。

=== 处理孟连事件 ===
沈培平在2008年任云南省普洱市委副书记、市长时，在调集警力未获批准的情况下，仍然从普洱调集警力到[[孟连|孟连]]，最后处理不当，酿成[[孟连事件|孟连事件]]，造成2人死亡，多人受伤。但事后只是被要求写检讨。此后仍然在2013年出任云南省副省长<ref>{{cite web|accessdate=2016-01-03|url=http://epaper.bjnews.com.cn/html/2014-03/11/content_499202.htm?div=|title=“孟连事件”怎没耽搁沈培平升官|publisher=新京报}}</ref><ref>{{cite web|title=云南副省长沈培平被查 孟连事件瞒着省里调动警力|url=http://www.chinanews.com/gn/2014/03-10/5928561.shtml|accessdate=2016-01-02|publisher=中国新闻网}}</ref>。

=== 主张拆迁引争议 ===
2010年，时任普洱市委书记的沈培平推行旧城改造工程，涉及到1812户居民的拆迁项目。但该工程存在补偿过少、程序非法等问题，不少建成不久的新房也被拆，百余居民状告市政府，有民间流传其为“拆迁大佐”<ref name=tkp/>。

== 学位腐败 ==

公开资料显示，沈培平的学历存在造假行为，从博士到教授仅花5个月。

2007年7月，沈培平从资源学院自然地理学专业在职研究生毕业，获博士学位。同年12月，他被聘为“北京师范大学资源学科建设与发展指导委员会”委员、资源学院兼职教授。从博士毕业到成为教授，仅花了5个月时间。<ref>{{cite web|accessdate=2016-01-03|url=http://politics.people.com.cn/n/2014/0312/c1001-24617840-2.html|title=云南原副省长沈培平的学历之谜：博士到教授仅花5个月|publisher=人民网}}</ref>

== 受审 ==

2015年8月4日上午，沈培平因涉嫌受贿在[[北京市第一中级人民法院|北京市第一中级人民法院]]受审。公诉机关指控，2000年至2012年，沈培平利用其担任中共腾冲县委书记、思茅市市长等职务上的便利，为多家公司提供帮助，非法收受财物计1615万元。<ref >{{cite web|url=http://news.163.com/15/0804/18/B06N47I80001124J.html | language= | title=沈培平获博士学位5个月后成教授 被称"茶市长" |publisher=中国青年网(北京) |date=2015-08-04 |accessdate= }}</ref>12月3日，北京市一中院一审判决，沈培平犯受贿罪，判处有期徒刑12年<ref>{{cite web |url = http://www.chinacourt.org/article/detail/2015/12/id/1761006.shtml |title = 云南原副省长沈培平受贿1615万元 获刑12年 |accessdate = 2015年12月3日 |publisher = 中国法院网}}</ref>。因沈培平能够如实供述其所犯罪行，认罪悔罪，法院已对其从轻处罚，且有重大立功表现<ref>{{cite web |url = http://news.ifeng.com/a/20151204/46517644_0.shtml |title = 云南省原副省长沈培平受贿案一审宣判 |accessdate = 2015年12月4日 |publisher = 法制日报}}</ref>。
