'''沈启原'''（{{Bd|1526年|2月22日|1591年|4月17日|catIdx=S沈}}），字'''道初'''，号'''霓川'''，又号'''存石'''，[[浙江|浙江]][[嘉興府|嘉興府]][[秀水縣|秀水縣]]人，[[明朝|明朝]]官员，藏书家。嘉靖三十八年（1559年）中进士，初授[[南京|南京]]工部屯田司员外郎，累官至[[陕西|陕西]]按察司副使。因仕而富，爱好藏书、刻书，著有《存石草堂书目》，已佚。

== 生平 ==
嘉靖五年正月十二日（1526年2月22日），沈启原出生<ref>《陆学士先生遗稿》卷12《明故中宪大夫陕西按察司副使霓川沈公墓志铭》：“公生嘉靖丙戌正月十二日。”</ref>。嘉靖十九年（1540年）成为生员，嘉靖二十五年（1546年）中举，当时只有二十一岁，但是之后连续三次参加会试落第<ref>《国朝献征录》卷94《陕西按察司副使沈先生启原行状》：“岁庚子，督学蒙溪张公校士，得先生，大为称赏，以儒士应试省闱，归而补邑诸生……丙午举于乡，年才二十一也，丁末、庚戌、癸丑三下第归。”</ref>。嘉靖三十八年（1559年），沈启原终于考中进士，官[[南京|南京]][[工部|工部]]屯田司[[员外郎|员外郎]]，代理真州（今江苏省仪征市）钞关税务四十多天，代理期满后负责管理征收芦田的赋税<ref>《陆学士先生遗稿》卷12《明故中宪大夫陕西按察司副使霓川沈公墓志铭》：“丁未举进士高等，例以选首得授比部，会有轧公者，改南司空郎，摄理真州税四十日，商民至貌公生祠之。已，董芦政。”</ref>。嘉靖四十一年（1562年），升为本司郎中<ref>《国朝献征录》卷94《陕西按察司副使沈先生启原行状》：“癸亥，转本司郎中。”</ref>。嘉靖四十三年（1564年）任应天乡试同考官，是科解元[[沈位|沈位]]出其门下，又取中自己的学生[[焦竑|焦竑]]为举人<ref>《万历野获编·卷十四·科场·师弟相得》：“甲子应天乡试，适世宗新更典制，先大父以仪曹正郎为书一房，得焦卷欲首荐，而主考赏大父次卷沈太史虹台，定为解元，焦稍居后。”</ref>。嘉靖四十四年（1565年），转任南京[[礼部|礼部]]仪制司郎中。

嘉靖四十五年（1566年）升任四川布政司右参议<ref>《国朝献征录》卷94《陕西按察司副使沈先生启原行状》：“乙丑，调南仪制司……丙寅冬，迁四川参议。”</ref>，万历二年（1574年）调任山东布政司右参议<ref>《大明神宗实录》卷25万历二年五月：“庚寅，除原任四川右参议沈启原于山东。”</ref>，万历四年（1576年）升为陕西按察司副使<ref>《大明神宗实录》卷46万历四年正月：“甲辰，升山东右参议沈启原为陕西副使。”</ref>，万历五年（1577年）因被陕西巡抚[[董世彦|董世彦]]参劾而罢官归里<ref>《国朝献征录》卷94《陕西按察司副使沈先生启原行状》：“抚台谓为慢己，因以他语中先生。奏既上，先生翩然登舟。”</ref>。万历十九年三月二十四日（1591年4月17日）去世，享年六十六岁<ref>《陆学士先生遗稿》卷12《明故中宪大夫陕西按察司副使霓川沈公墓志铭》：“殁万历辛卯三月二十四日，享年六十有六。”</ref>。

== 藏书 ==
沈启原父亲[[沈谧|沈谧]]官至湖广参议<ref name=":0" />，沈启原也历署好缺，沈家因此聚敛了巨额财富，有仆人数千，为江南屈指可数的巨室<ref>《快雪堂集》卷15《沈母王孺人墓志铭》：“累世鼎贵，僮奴数千，指江以南名家，屈指长溪沈氏。”</ref>。沈氏父子都爱好藏书，沈谧建万书楼，藏书三间；沈启原继续扩充父亲的藏书规模，建造了新的藏书楼<ref>《西园见闻录·卷之八·好学》：“旧有万卷书楼三楹，为石云公藏书处，至是所积滋多，复为楼贮之。”</ref>，并将藏书整理成《存石草堂书目》10卷<ref>《吴晗史学论著选集》卷1《两浙藏书家史略》：“好聚书，有《存石草堂书目》十卷。”</ref>。又与门生焦竑在明人[[黃省曾|黄省曾]]所辑《谢灵运诗集》的基础上，根据自己的藏书加以增补，刊行了《谢康乐集》4卷<ref>《澹园集》卷22《题谢康乐集后》：“《谢康乐集》世久不传，其见《文选》者，诗四十首止耳。后李献吉增乐府若干首，黄勉之增若干首，吾师沈道初先生冥搜博访，复得赋若干首，诗若干首，杂文若干首，譬之裒虬龙之片甲，集旃檀之寸枝，总为奇香异采，不可弃也。辑成，合刻之，而以校事委余。”</ref>，虽号称善本，但实际上疏于考证，把不少他人作品归于[[谢灵运|谢灵运]]，文献学价值不高<ref>{{Cite journal|title=关于谢灵运诗歌的文献学问题|author=周兴陆|url=|journal=《中国古代文学研究》|issue=第2期|doi=|others=|year=2008年|volume=|page=83页|pmid=}}</ref>。沈启原晚年，沈家遭遇火灾，大量藏书被毁<ref>《中国私家藏书史》第二编第四章第四节：“万历丁亥（1587）春，同里有祝融之厄，殃及沈家，书籍版片，化为乌有，几代积存，毁于一旦。”</ref>。

== 家庭 ==
祖父沈复，父沈谧官至湖广布政司右参议，母盛氏<ref name=":0">《陆学士先生遗稿》卷12《明故中宪大夫陕西按察司副使霓川沈公墓志铭》：“至公大父复，始用子贵，封刑科给事中；父谧，以名谏议迁为湖广参议。配盛孺人，嗣配盛恭人，是生公。”</ref>；弟沈启南，捐官为光禄寺署丞<ref>《陈学士先生初集》卷11《沈澄源先生传》：“沈先生名启南，字道明，别号志棠……家称素封，父参藩石云公，兄宪副霓川公，兄子太史几轩公三世举进士。”</ref><ref>《快雪堂集》卷13《光禄寺署丞志棠沈公洎配王孺人合葬墓志铭》：“是时，公业息，意进取，乃入赀拜大官丞。”</ref>；妻钱氏，生一子沈自邠，万历五年（1577年）中进士，授翰林院修撰，先于沈启原两年去世<ref>《国朝献征录》卷94《陕西按察司副使沈先生启原行状》：“配恭人钱氏，子一自邠，丁丑进士，翰林院修撰，先先生二年卒。”</ref>；孙[[沈德符|沈德符]]为举人，著有笔记《[[萬曆野獲編|万历野获编]]》<ref>《列朝诗集小传·丁集下·沈先辈德符》：“德符，字景倩，嘉兴人，故太史自邠之子也……年四十，始上春官，累举不得第而死。”</ref>。
