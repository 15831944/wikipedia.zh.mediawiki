'''沈抚同城'''是指[[中国|中国]][[辽宁省|辽宁省]][[沈阳市|沈阳市]]和[[抚顺市|抚顺市]]为了打造具有强大辐射力的中心城市，以此来支撑辽宁南部沿海经济，并带动辽宁中部腹地经济发展所实施的一项同城化战略。<ref name="提出与背景">{{cite web | url=http://www.sftc168.com/news.asp?id=332 | title=沈抚同城化的提出及背景 | accessdate=2013-04-24}}</ref> 沈抚同城亦为国家新型工业化综合配套改革试验区[[沈阳经济区|沈阳经济区]]的发展先行区。

== 建设背景 ==
中国在[[改革开放|改革开放]]之后，[[深圳|深圳]]与[[香港|香港]]两市提出“同城化”的发展理念，之后相继有吉林省[[吉林市|吉林市]]与[[长春市|长春市]]、[[山西省|山西省]][[榆次市|榆次市]]与[[太原市|太原市]]、[[河南省|河南省]][[开封市|开封市]]与[[郑州市|郑州市]]等都先后开始同城化建设。其中西安和咸阳实现电信同城通话和郑州和开封市金融同城结算、郑汴城际公交车开通，两市政府将促进广播电视同城化及市民待遇逐步同城化。以同城化促进区域经济一体化，已成为根据实际情况提高城市化水平的一种有效方式。 <ref name="提出与背景"></ref>

沈抚同城概念的提出，源于时任中共辽宁省委书记，省人大常委会主任[[李克强|李克强]]提出的 “中部城市群”发展战略。2005年，[[沈阳经济区|沈阳经济区]]建设被列为辽宁省的发展战略。由于“中部城市群”各个城市间发展差异比较大，选择客观条件比较成熟的地区先行突破，成为当时的路径选择。沈抚同城最终进入了决策者的视野。<ref name="先行">{{cite web | url=http://www.sftc168.com/news.asp?id=786 | title=沈阳经济区获批沈抚同城先行 | accessdate=2013-04-24}}</ref> 李克强在中共辽宁省委全会上提出“推进沈阳与抚顺的同城化发展，壮大辽宁中部城市群，打造东北亚最大的区域中心城市，让沈抚链接带成为中国经济新的增长极。”<ref name="核心">{{cite web | url=http://www.cuew.com/qu/df/2013/0418/918.html | title=沈抚新城城市化改革下的辽宁城市群核心 | accessdate=2013-04-24 | archive-url=https://web.archive.org/web/20130919135503/http://cuew.com/qu/df/2013/0418/918.html | archive-date=2013-09-19 | dead-url=yes }}</ref> 这段发言也正是标志着标志着沈抚同城化已经正式被纳入辽宁省经济和社会发展的大局之中，成为辽宁建设沿海经济强省的重要战略措施。 <ref name="提出与背景"></ref> 按照辽宁省委的要求，沈抚两市建立健全了相应的组织领导机构，形成了高层协商机制，各个方面的合作得到快速推进，沈抚同城化已进入实质性操作阶段。

== 建设过程 ==
2007年 9月11日，沈抚两市市长签署了《加快推进沈抚同城化协议》，沈抚同城化建设正式启动。2008年4月9日，由辽宁省发改委负责组织编制的《沈抚连接带总体发展概念规划》通过专家评审，并于同年7月30日正式发布。<ref name="凤凰网">{{cite web | url=http://finance.ifeng.com/news/special/syjjq/20100326/1969721.shtml | title=凤凰网：沈阳经济区获批 沈抚同城先行 | accessdate=2013-04-25}}</ref> 在建设之初， 沈抚同城项目得到了国家相关部门和领导十分关注与支持。时任[[国务院|国务院]][[副总理|副总理]][[曾培炎|曾培炎]]，[[国家开发银行|国家开发银行]]总行行长[[陈元|陈元]]，[[国务院|国务院]]振兴东北办主任张国宝，副主任宋晓梧均到沈阳、抚顺专题调研沈抚同城化项目，听取沈抚同城化汇报，对沈抚同城化的构想和工作进展给予充分肯定。围绕沈抚同城化，从空间同城化、交通同城化、基础设施及产业同城化、滨水及生态同城化等几个方面规划了沈抚新城建设、城际轻轨、城际公路、浑河景观带建设等8大重点项目，总投资215亿元，[[国家开发银行|国家开发银行]]经过充分的调研、审核和论证，确认了149.2亿元的合作额度。这8大重点项目全面建设后，将会为沈抚同城化的全面实施和推进提供强有力的支撑。  

2009年，以重型装备及配件、石化电力、煤矿安全、汽车配件为特色的64个项目已经落地，总投资153亿元，辽宁装备制造业基地初具规模。沈阳丰远热高乐园、旺力商贸城、银江科技集团等投资超过30亿元的大型商住航母陆续起航，百万平方米的绿化工程已经完成，新城呈现出欣欣向荣的景象。

== 争议 ==
沈阳与抚顺之间的城际铁路，[[沈抚城际铁路|沈抚城际铁路]]的平均上座率不足三成。有市民认为铁路的票价不仅贵，而且两地的融合没有达到足够的密切。<ref name="凤凰网"></ref> 在2009年，抚顺市作出了 “举全市之力建设沈抚新城”的决定，并列为政府的“一号工程”。抚顺市市委、人大、政协三大机关初步决定向沈抚新城搬迁，促使全市各种资源向新城集结。有评论指出，在沈抚一体化融合的过程中，相对于抚顺的积极，规划中的沈抚新城沈阳一侧的行动则显得有些缓慢。<ref name="凤凰网"></ref>
