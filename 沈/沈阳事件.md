'''沈阳事件'''是多名[[北京大学|北京大学]]校友于2018年4月在网络实名举报时任[[北京大学中文系|北京大学中文系]]副教授，后为[[南京大学|南京大学]]汉语言系主任，长江学者沈阳，[[性侵|性侵]]北京大学中文系本科女生高岩，致其于1998年自杀死亡的事件。<ref>{{Cite news|url=http://news.sina.com.cn/o/2018-04-08/doc-ifyuwqez6831947.shtml|title=北大出示“沈阳事件”处分文件：曾说“算是女朋友”，并搂抱亲吻高岩|author=|date=2018-04-08|work=新浪|accessdate=2018-04-08}}</ref>

== 沈阳其人 ==
沈阳于1955年12月生于上海<ref name="jb05">{{Cite web |url=https://news.qq.com/a/20180405/012853.htm|title=北大前教授被指性侵女生致其自杀 回应：均为恶意诽谤 |accessdate=2018-04-07 |author=记者：王煜、贾世煜，实习生：王露晓、周小琪 |date=2018-04-05 |work= |publisher=腾讯网，来源：新京报|language=Zh-hans }}</ref>，[[中国共产党|中国共产党]]党员，曾在[[中国人民解放军|中国人民解放军]][[服役|服役]]十六年，从事文秘工作。

1986年，沈阳转业至上海，于同年考入[[华东师范大学|华东师范大学]]中文系硕士班<ref name="b1">{{Cite web |url=http://www.jfdaily.com/news/detail?id=85384|title=沈阳的“两面”：记者赴南大文学院，一探舆情汹涌的背后 |accessdate=2018-04-15|author=殷梦昊 |date=2018-04-08 |work= |publisher=上观 |language=Zh-hans }}</ref>，在1987年获得华东师范大学学士学位，1990年获华东师范大学硕士学位。<ref>{{cite web|title=沈 阳|url=http://chin.nju.edu.cn/grzy2_224-5.html|website=南京大学文学院|accessdate=2018-04-08}}</ref>1993年，沈阳获[[北京大学|北京大学]]博士学位<ref name="jb05"/>，[[博士導師|导师]]为[[陆俭明|陆俭明]]<ref name="sbsy">{{cite book|author1=沈阳|title=甲子学者治学谈|date=2017-01|publisher=北京语言大学出版社|location=北京|isbn=9787561948057|page=372-384|chapter=一直在路上－－六十年人生风景一瞥}}</ref>。博士毕业后，沈阳留在北京大学任教，于次年升为[[副教授|副教授]]<ref name="b1"/>，并担任[[北京大学|北京大学]]人文特聘教授、[[北京大学中文系|北京大学中文系]]副主任。

2011年，沈阳由北京大学轉任[[南京大学文学院|南京大学文学院]]语言学系主任。2017年底，沈阳兼任[[上海师范大学|上海师范大学]]光启讲席教授，在[[上海师范大学|上海师范大学]]兼职授课，档案仍保留于南京大学。2018年时，沈阳成为[[长江学者|长江学者]]特聘教授，[[南京大学文学院|南京大学文学院]]教授、南京大学文学院语言学系主任（2018年1月已卸任）、[[博士生导师|博士生导师]]；北京大学中文系兼职教授、北京大学中国语言学研究中心兼职研究员<ref name="a2">{{Cite web |url=http://news.sina.com.cn/o/2018-04-07/doc-ifyuwqez6020662.shtml |title=原北大教授被指“性侵女生致其自杀”，校方和当事人回应了 |accessdate=2018-04-07 |author= |date=2018-04-07 |work=编辑：周颖 |publisher=新浪网，来源：新闻晨报|language=Zh-hans }}</ref>。

根据2018年5月9日流传到中国互联网上的一张“南京大学内部人士”提供的显示南京大学内部会议上通报内容的[[微信|微信]]聊天截图，沈阳当兵时就曾因在军营嫖宿卖淫女被处分，并因此被勒令转业，这一处分也记在其档案里。通报还称，教育部因沈阳隐瞒这一处分决定而撤销他的“长江学者”称号并追回奖金。消息称，教育部在“五一”前就下发了处理意见给南京大学，但校方没有公开<ref>{{cite web|url=http://news.dwnews.com/china/news/2018-05-10/60057165.html|title=传北大前教授沈阳被撤销长江学者称号 有嫖娼案底|accessdate=2018-05-10|publisher=多维新闻}}</ref>。

== 高岩之死 ==
高岩生于1976年8月26日，<ref name="ifeng05">{{cite news|title=北大2毕业生实名揭露：20年前，同学因教授性侵自杀|url=http://news.ifeng.com/a/20180405/57335140_0.shtml|accessdate=2018-04-25|work=凤凰网资讯|date=2018-04-05}}</ref>母亲周树铭之前是[[北京市育才学校|北京育才中学]]的语文教师，父亲高石曾退休前曾经任职于北京市教研中心。<ref name="cyol09">{{cite news|title=沈阳课上多次举涉性暗示例子 对女大学生动作轻浮|url=http://news.163.com/18/0409/11/DEUSDAMT0001899N.html|accessdate=2018-04-25|work=中国青年报|date=2018-04-09}}</ref>据母亲周树铭的说法，高岩从小听话用功，小学毕业后保送升入[[北京师范大学附属中学|北京师范大学附属中学]]，1995年考入北京大学中国语言文学系。<ref name="jw08">{{cite news|title=北大“高岩自杀事件”：集体创伤和“迟到”的反击|url=http://www.sohu.com/a/227595140_658437|accessdate=2018-04-25|work=鉴闻|date=2018-04-08}}</ref>1998年3月11日，高岩在家中打开煤气自杀。<!-- 尽量不用自媒体来源，如可替代则替代之。-->

李悠悠指称，沈阳“是直接的肇事者，是她自杀的始作俑者”，他多次性侵高岩，对高岩实施了强迫的性行为。<ref name="jb05" />据李悠悠《南京大学文学院语言学系主任、长江学者沈阳教授，女生高岩的死真的与你无关吗？》，高岩考入北大中文系后担任班级学习委员，当时沈阳在北大中文系教授现代汉语课程，指派高岩负责中文系文学、语言、文献三个班收作业、收钱等事务。那时北大文科院系（外国语专业除外）的大一学生都住在[[北京大学校园#昌平校区|昌平园校区]]。高岩家住北京，通常周六下午回家，周日再赶回昌平园。李悠悠说，某一天高岩告诉她，沈阳向校车部门打了招呼，之后高岩可以搭乘教师班车去昌平园；高岩家住[[长安商场|长安商场]]附近，沈阳家住[[三里河路|三里河]]，每周一早上沈阳都在长安商场附近的人行天桥下等高岩，然后两人一起去坐教师班车。大一第一学期，高岩的成绩位列中文系第一名。据李悠悠说，大一第二学期的某一天，沈阳约她到他家中探讨语言学问题，并把作业送到他家里；高岩按约定时间登门，之后沈阳从背后抱了她，还吻了她的脸。李悠悠说，自此之后，高岩渐渐变得闷闷不乐，但第二学期的成绩仍然是中文系第一名。1996年夏天，高岩和同学们返回[[北京大学校园#燕园主校区|燕园校区]]。根据李悠悠的说法，大二上学期的某一天，高岩满眼泪水、吞吞吐吐地告诉她，沈阳“像饿狼一样向我身上扑过来”；之后，高岩陆陆续续告诉她，“沈阳脱光了她的衣服，对她做了她从未做过的事儿”，高岩非常害怕、痛苦；高岩说，她曾告诉沈阳，她不喜欢这样，要沈阳住手，而沈阳说是因为他喜欢她才这样，而高岩觉得喜欢不应该是这个样子。据李悠悠的文章，在此期间沈阳还频繁同其他女生约会、做爱，还对这位女生说他并不喜欢高岩、是高岩主动勾引他的、高岩是精神病，而沈阳的话又被这位女生传了出去，这使得高岩同时承受着性侵和谣言的折磨。高岩曾想休学，但未能成功。李悠悠文中提到，1997年夏天，高岩再也不想与沈阳见面，只想与他最后再谈谈。她约了沈阳在饭馆见面，向沈阳表达了自己的愤怒和不解，要求沈阳尊重自己、尊重别人；但沈阳对她冷嘲热讽，并不承认他的不端行为。高岩气得一口饭也没有吃就愤然离席。<ref name="YYLi">{{cite web|author1=李悠悠|title=南京大学文学院语言学系主任、长江学者沈阳教授，女生高岩的死真的与你无关吗？|url=https://www.douban.com/note/664259813/|website=豆瓣|accessdate=2018-04-05|archiveurl=https://archive.today/20180405035455/https://www.douban.com/note/664259813/|archivedate=2018年4月5日|date=2018-04-05|deadurl=yes|df=}}</ref>

4月6日，北京大学、南京大学相继对指控做出回应。北京大学的说明称，经查阅相关材料，二十年前，即1998年3月，[[北京市公安局|北京市公安局]]西城分局对这一事件作出事实认定，给出调查结果，1998年7月，北京大学对沈阳做出了[[行政处分|行政处分]]；南京大学回应称正在进行研判、密切关注事件进展<ref name="a3">{{Cite web |url=http://finance.ifeng.com/a/20180406/16060626_0.shtml|title=北大、南大回应“教授沈阳被指性侵女生致其自杀” |accessdate=2018-04-07 |author= |date=2018-04-06 |work= |publisher=凤凰网，来源：中国新闻网|language=Zh-hans }}</ref>。当年参与处分的北京大学中文系主任[[费振刚|费振刚]]，接受采访时表示，“我是一个老人，对这个事情有我自己的判断。当年，他是个年轻人，但是从师德讲，这个年轻老师是有问题的。”<ref>{{Cite web |url=http://news.ifeng.com/a/20180406/57357956_0.shtml|title=媒体访北大处分沈阳时任系主任：这个老师师德有问题 |accessdate=2018-04-07 |author=中国新闻周刊 |date=2018-04-06 |work= |publisher=凤凰网|language=Zh-hans }}</ref>
