'''沈旭华'''（{{bd|1969年|12月20日|2002年|8月20日|catIdx=Shen沈旭华}}），中国知名女[[主持人|主持人]]，[[中国中央电视台|中国中央电视台]]《[[夕阳红|夕阳红]]》多彩生活版主持人。2002年因为摔伤抢救19天无效去世。<ref name="webc " >{{cite web |url=http://news.163.com/2004w05/12551/2004w05_1084427971946.html |title=央视《夕阳红》女主持沈旭华意外身亡 |accessdate=2004-05-13  |work=|publisher=网易 |language=zh-hans}}</ref>

== 简介 ==
=== 去世 ===
2002年8月1日，沈旭华和朋友在安贞桥旁边的张生记餐厅聚餐，订房是紧靠消防通道的12号房，吃饭的时候，沈旭华的手机接连响个不停，而包厢过于嘈杂，沈旭华不得不出去接电话，她边走边聊，走到消防通道推门出去，因为没有灯光，不知道没有护栏，才跨出一步就摔到了一楼。<ref name="webc " /> 1个小时候才被送装修材料的工人发现报警。<ref name="webc " />

沈旭华家属将张生记餐厅告上[[法庭|法庭]]，要求张生记餐厅、以及建筑商京浙宾馆一共赔偿246万元。<ref>{{cite web |url=http://ent.sina.com.cn/s/m/2003-08-08/0956182846.html |title=沈旭华亲属状告张生记、京浙宾馆索赔246万元 |accessdate=2003年8月8日 |work=|publisher= 新浪网|language= zh-hans}}</ref>

2002年9月，警务人员向沈旭华的家属宣读了《[[法医鉴定书|法医鉴定书]]》：“认定沈旭华是在张生记吃饭期间，因打电话从消防通道二楼直接摔下一楼，属于高坠死亡，排除了其他[[刑事|刑事]]案件的可能性。”<ref>{{cite web |url=http://ent.sina.com.cn/s/m/2002-09-05/070699070.html |title=警方认定女主持沈旭华死因 录像带再现事故现场 |accessdate= 2002年9月5日 |work=|publisher=新浪网 |language= zh-hans}}</ref>

== 荣誉 ==
沈旭华在央视举办的“荣事达”主持人大赛中获得前50的名次。
