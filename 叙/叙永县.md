'''叙永县'''是[[四川省|四川省]][[泸州市|泸州市]]下辖县，位于[[四川盆地|四川盆地]]南缘，接壤云贵，鸡鸣三省。古时叙永亦称“永宁”。县政府位于[[叙永镇|叙永镇]]。全县幅员面积2984.41平方公里。

== 城市 ==
叙永[[城市|城市]]建设相对较好，城市面积7平方公里。城市人口近10万。县城新区规划23平方公里。
叙永与[[兴文|兴文]]县、[[古蔺|古蔺]]县、[[合江|合江]]县，[[云南|云南]]省[[威信|威信]]县、[[贵州|贵州]]省[[毕节|毕节]]县接壤。
城市分为东城、西城两个部分。以永宁河为界。
叙永城居于[[四川盆地|四川盆地]][[长江|长江]]南岸最南缘，海拔三百至四百米。
出叙永镇，往南，为[[云贵高原|云贵高原]]北段。最高海拔有一千米以上。

==历史==
叙永，古称永宁。旧[[叙州|叙州]]府管辖。因有永宁河流经县城，故集合两者，称为叙永。
叙永明朝建城，古为泸水之地。[[彝|彝]][[苗族|苗]]聚集。后[[汉人|汉人]]来，占据北部丘陵地区，彝苗退居南面山区。
[[明|明]]末[[清|清]]初，[[湖广填四川|湖广填四川]]。叙永为从南部入川之路。人口增长，现老城区人口多为客籍。非本地土著。湖广[[客家|客家]]混杂。通用[[西南官话|西南官话]][[岷江|岷江]]片。
有清一代，叙永[[知县|知县]]曹，开江门峡航道，叙永成为输盐于[[云贵|云贵]]之要道，船从[[自贡|自贡]]产盐区出，经[[沱江|沱江]]南下，入[[泸州|泸州]]，[[纳溪|纳溪]]南行，逆流而上进永宁河。到叙永上岸，马帮当道，往云贵。
[[辛亥革命|辛亥革命]]，[[民国|民国]]建立。叙永[[哥老会|哥老会]]也赴泸州与会，建立[[军政府|军政府]]。后，[[护国战争|护国战争]]，滇军入川。[[蔡锷|蔡锷]]有指挥所驻叙永，今东城公园巷人民医院住院部内。
民国迁播去台后，叙永不甚发展，航路中断。码头地位丧失。
==袍哥开山==
1848年（[[清|清]][[道光|道光]]二十八年）[[永宁|永宁]][[郭永泰|郭永泰]]开荩忠山，手中有在[[福建|福建]]渔人手内寻获的《[[金台山实录|金台山实录]]》。据说[[郑成功|郑成功]]于1662年（[[清|清]][[康熙|康熙]]元年）病故，子[[郑经|郑经]]立奉遗命以《金台山实录》作要件保存。1680年（清康熙十九年）郑经立战败于[[厦门|厦门]]，次年忧愤而亡，次子[[郑克塽|郑克塽]]继位。1683年清军[[提督|提督]][[施琅|施琅]]攻陷[[台湾|台湾]]，[[郑克塽|郑克塽]]将《金台山实录》用铁盒密封沉于海底，免为清军所获。郭永泰于1835年（清[[道光|道光]]十五年）去厦门看族叔，并到一平民家闲游，见其盖米瓮中有旧书一本，封面上赫然写着《金台山实录》五字，书面盖有小图章，为“[[延平郡王|延平郡王]]招讨大将军印”几字，马上知道是[[郑成功|郑成功]]的遗物，忙问此物是从何处而来的。据那人讲，他父亲是渔民，善于游水，能在水中伏一昼夜，一天在海中得一铁盒，无法打开，费了三天时间才将它砸开，发现里面有金银珠宝数件，小玉印一方，旧书数本，因为他不识字，不知道是什么书。郭永泰就用一百两将书购得，又问小玉印现在何处？那人称已经卖给邻居，郭又用白金十两赎回。这就成了他开山的基本思想武器。因为[[郑成功|郑成功]]是[[藩王|藩王]]，书中所用的都是王者的口气，郭在[[开山|开山]]时不敢完全使用，他于是托人仿原本进行了一定的增删，以资实用，名之曰“海底”，又名“金不换”。从此组织规模大张。

== 人口 ==
相关统计数据截止到2005年，叙永县共有人口66.1631万人，[[少数民族|少数民族]]占叙永县总人口的6%，约为3.8万人。叙永县非农业人口9.8602万人，占叙永县总人口14.1%。

== 经济 ==
2005年，全县实现生产总值(GDP)22.55亿元。全社会固定资产投资实现5.26亿元。社会消费品零售总额实现10.68亿元。财政总收入实现14808万元，地方财政收入实现8166万元。农民人均纯收入2510元。县城居民人均可支配收入6177元。

== 交通 ==
这里有321[[国道|国道]]和[[隆叙铁路|隆叙铁路]]。[[川黔高速|川黔高速]]2012年1月1日，川黔高速纳叙段已经正式通车。到2012年底，川黔高速全线通车。
