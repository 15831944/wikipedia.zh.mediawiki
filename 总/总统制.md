'''总统制'''是民主共和政体的一种，由[[行政首长|行政首长]]领导一个独立于立法部门的行政部门。 在大多数情况下，这位[[政府首脑|政府首脑]]兼任[[国家元首|国家元首]]，称为[[总统|总统]]。

在总统制国家，行政机构是选举产生的，对立法机构不负责任，立法机关在正常情况下不能予以驳回。 但是，在不常见的情况下，通常是通过弹劾，这种解职是可能的。

总统制国家起源于美国，美国是世界上第一个完整共和政体下的总统制国家。

== 特徵 ==
* [[總統|總統]]為[[人民|人民]]直選產生，或由选举获胜党派间接选举产生，向人民負責，[[政府|政府]]由總統領導。
* 總統既是[[國家元首|國家元首]]又是[[政府首腦|政府首腦]]，總攬行政權力，公佈[[法律|法律]]、發佈[[命令|命令]]不需副署。除了个别重要任命需要通过议会外，政府官员由总统任命，向总统负责。
* [[行政機關|行政機關]]和[[立法機關|立法機關]]相互獨立。作為立法機關的[[議會|議會]]，其[[議員|議員]]不能兼任行政職務；而作為行政機關的[[政府|政府]]官員，也不能兼任議員。
* 總統的[[選舉|選舉]]與[[國會|國會]]的選舉分開進行，國會中的多數黨不一定是[[執政黨|執政黨]]。
* 總統與議會的任期相對[[內閣制|內閣制]]固定。總統無權解散國會，[[國會|國會]]不能對總統投不信任票，但是总统[[违宪|违宪]]或有嚴重違法行為的话，國會可以对總統提出[[弹劾|弹劾]]案，并交由[[憲法法院|憲法法院]]审理。

== 優缺點 ==
=== 優點 ===
如果能採用一種保證支持當選總統之[[執政黨|執政黨]]或聯合勢力在立法機關之民選代表中佔多數（不必是絕對多數）之選舉制度，那就大有好處。<ref name="冊"/>{{rp|12}}
# [[總統|總統]]在任職期間，沒有因政見不同，失去議会多数支持而倒臺的風險，可以積極推行政策，在行政與立法相互制約的情況下，行政權力高度集中，運作效率充分發揮，適應形勢的變化。
# 在議會中反對黨佔多數時，反對黨在可有效的制衡總統的同時，減少發生憲政危機的可能。
# 在議會中執政黨佔多數時，反對黨同樣可以有效的制衡總統，並且總統制的反對黨不會像半總統制和議會制的反對黨那樣容易被邊緣化。
# 有限的任期可以有效的防止總統任期過長而走向獨裁的局面。（只適用於有限制連任次數的國家，但不少國家並沒有限制連任次數，進而造成該國總統成為長期執政的獨裁者）
# 行政上可以最大限度的趨向專業化。
# [[技術官僚|技術官僚]]不會因爲[[政黨輪替|政黨輪替]]而受到過大的影響，對政策的連貫性有益。
# 清晰的三權分立制度可有效避免任何一個部門形成獨裁。

=== 缺點 ===
[[拉丁美洲|拉丁美洲]]之經驗證據表明，在採用總統制之國家中，如果政黨發展呈高度分化；總統選舉採用兩輪選舉制、立法機關則採用競選名單制且兩個選舉不同時舉行時，行政機關和立法機關就會分裂，關係緊張，國家穩定很成問題。<ref name="冊">{{cite book|title=《選舉制度設計手冊》|editor=國際民主和選舉協助研究所（International IDEA）|publisher=[[商務印書館（香港）有限公司|商務印書館（香港）有限公司]]|year=2013}}</ref>{{rp|12}}

# 總統擁有相當大的行政權，可凌驾國會執行職权，缺乏議會制衡，總統亦可否決議會通過的法案，容易形成僵局及政治危機。
# 總統在擁有行政权同時，可对立法和司法進行支配，通過控制國會及任命大法官，继而同時掌握立法和司法。
# 在議會中，即使[[執政党|執政党]]掌握國會，執政党不一定完全聽命于總統。一旦[[反對黨|反對黨]]占多數，行政與立法常在個別問題上陷於僵局。
# 如總統執行錯誤政策，選民、議會也不能在其任期屆滿之前使其下臺，除非總統被議會[[彈劾|彈劾]]及罷免，但彈劾的門鑑相當高，且操作上通常涉及政治鬥爭，甚少使用，有別於[[不信任動議|不信任動議]]。
# 在[[任期制|任期制]]下，即使是受欢迎稱職的總統，最多也只能連任一次，必須離任；或不能連選連任，須隔一屆才能再次參選。（只適用於[[美國|美國]]等憲法明文限制總統連任次數的國家。部分总统制國家不容許總統連任，如[[墨西哥|墨西哥]]；[[秘魯|秘魯]]和[[智利|智利]]容許隔屆參選。）
# 對於沒有連任限制的國家，容易形成獨裁統治。許多採取總統制的國家往往都是總統專制的局面，總統皆权力独大，長期執政。
# 總統權力非常大，需要完善的彈劾機制和強而有力國會監督，否則容易形成獨裁統治。如[[非洲|非洲]]、[[中美洲|中美洲]]及[[南美洲|南美洲]]許多採取總統制的國家過去長期出現軍事強人專制的局面。
# 總統依靠技術[[官僚|官僚]]治國，強化[[官僚主義|官僚主義]]。
# [[議員|議員]]不能兼任行政職務，部分不用或無意加入政府，因此會加強監察及批評政府，即使是執政黨的議員亦然，行政及立法容易出现緊張局面。
# 定期举行的總統選舉消耗龐大經費，[[金权政治|金权政治]]现象相对[[議會制|議會制]]国家严重。
# 除了[[美國|美國]]以外，所有一開始實行總統制的國家，它們的首次民主嘗試以失敗告終。相反議會制的成功率反而較高。<ref name="袁世凯暗杀宋教仁前后">{{Pt}}{{cite web |url=http://news.cn.yahoo.com/ypen/20130111/1547197.html |title=存档副本 |accessdate=2013-08-28 |deadurl=yes |archiveurl=https://web.archive.org/web/20130626170741/http://news.cn.yahoo.com/ypen/20130111/1547197.html |archivedate=2013-06-26 }}.</ref>
# 總統不一定代多數民意，例如[[美國選舉人團|美國選舉人團]]選出的[[美國總統|美國總統]]並不一定獲得最多普選票，[[多數制|多數制]]選出的總統並不一定獲得過半數的普選票，缺乏認受性。
# 行政权介入司法的批評，並體現在司法判決中。三權分立最成功的美國，三权经常出现矛盾，常出現强行政，弱立法。因憲法保障總統的行政權，總統可以通過行政命令繞過國會推行政策。1933年[[大萧条|大萧条]]后，[[富兰克林·德拉诺·罗斯福|罗斯福]]颁布一系列法令，并通过国会授权取得[[美国总统|美国总统]]史上前所未有的巨大权力。但保守派佔多數的[[美国最高法院|美国最高法院]]却经常驳回一些法令。结果1935年1月最高法院以8比1的票数，宣布罗斯福的《全国工业复兴法》违宪。同年一名失業工人试图利用《最低工资法》来取得工资补偿时，被控方律师则直接指出该法案违反了[[美国宪法第十四修正案|美国宪法第十四修正案]]。此后罗斯福为加快推行新政，于1936年3月6日进行了“[[炉边谈话|炉边谈话]]”，将矛头直指司法部门，要求國會讓他無限制增加最高法院大法官的數目，企图将司法部门置于[[行政部门|行政部门]]管辖下。这就引起了全国范围的激烈讨论。後來联邦最高法院大法官認為《最低工资法》大部分並無違宪。當時大法官是为了保证三权分立的政治格局而退让。及后因部分大法官退休及死亡，使羅斯福得以提名及任命大法官。<ref>{{Cite book|url=http://www.docin.com/p-504605739.html|last=豆丁网|title=美国三权分立政体浅析|date=2012-10-22}}</ref>此后，历任总統均會在任內通過提名與其[[意識形態|意識形態]]相近的大法官，以發揮政治影響力。
