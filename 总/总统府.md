'''总统府'''（或'''總統宮'''）是指[[总统|总统]]等[[共和制|共和制]][[國家元首|國家元首]]的[[官邸|官邸]]或[[辦公室|辦公室]]所在地，通常被視為一個[[國家|國家]]的政治權力象徵。

== 列表 ==
本表收錄[[世界各國|世界各國]]的總統府、或[[共和制|共和制]][[國家元首|國家元首]]之駐地。

===亚洲===
* [[中華民國總統府|中華民國總統府]]，原址位於[[南京市|南京市]][[南京總統府|國府路]]（今長江路），現位於[[臺北市|臺北市]][[博愛特區|博愛特區]]的[[總統府_(臺灣)|臺灣總督府舊址]]。<ref name = "中華民國總統府_2018"> {{ cite web | title = 中華民國總統府 參觀資訊 | url = https://www.president.gov.tw/Page/124 | language = zh-tw | accessdate = 2018-07-17 }} </ref>
* [[中華人民共和國|中華人民共和國]][[中南海|中南海]]，位於[[北京市|北京市]][[西城區|西城區]]，是[[中國共產黨中央委員會|中共中央]]和[[中華人民共和國國務院|國務院]]的所在地。
* [[大韓民國|大韓民國]][[青瓦臺|青瓦臺]]，位于[[首爾|首爾]][[鍾路區|鍾路區]]。
* [[菲律賓|菲律賓]][[馬拉坎南宮|馬拉坎南宮]]，位于[[馬尼拉市|馬尼拉市]]。
* [[伊拉克|伊拉克]][[伊拉克总统|拉德瓦尼亚總統府]]（Radwaniyah Palace），位于[[巴格达|巴格达]]。
* [[印度总统|印度总统]][[印度总统府|总统府]]（Rashtrapati Bhavan），位于[[德里|德里]]。
* [[巴基斯坦|巴基斯坦]]总统府（Aiwan-e-Sadr），位于[[伊斯兰堡|伊斯兰堡]]。
* [[亞美尼亞|亞美尼亞]]总统府（Presidential Palace, Yerevan），位于[[葉里溫|葉里溫]]。
* [[新加坡總統府|新加坡總統府]]（Istana），位于新加坡[[乌节路|乌节路]]。
* [[老挝国家主席府|老挝国家主席府]]，是[[老挝人民民主共和国主席|老挝人民民主共和国主席]]的官邸、[[寮國人民革命黨中央委員會總書記|寮國人民革命黨中央委員會總書記]]的駐地、国家权力的象征。其坐落于首都[[万象|万象]][[湄公河|湄公河]]畔，靠近[[凯旋门_(老挝)|凯旋门]]，建筑风格为法国式。
* [[河内主席府|河内主席府]]（{{vie|qn=Phủ Chủ tịch|hn=府主席}}）位于[[河内|河内]]巴亭广场[[胡志明纪念堂|胡志明纪念堂]]建筑群附近，是一座拥有百年历史的法国殖民地建筑，原为[[法属印度支那|法属印度支那]]总督府，建于1900年和1906年之间。设计者为[[法属印度支那|法属印度支那]]的官方建筑师奥古斯特·亨利Vildieu。这是一座纯粹欧洲风格的黄色建筑，铁艺大门，古典列柱，壮观的楼梯，唯一让人感觉到身处越南的视觉线索是庭院内的[[芒果|芒果]]树。

===北美===
