'''標緻雪鐵龍集團'''（[[法文|法文]]：{{lang|fr|PSA Peugeot Citroën}}）是[[法国|法国]]私营汽车制造公司，由[[标致汽车|标致汽车]]拥有，目前旗下拥有[[标致|标致]]、[[雪铁龙|雪铁龙]]、[[DS_Automobiles|DS Automobiles]]和[[歐寶|歐寶]]四大汽车品牌。在[[欧洲|欧洲]]，標緻雪鐵龍集团是仅次于[[德国|德国]][[福斯集團|福斯集團]]的[[欧洲|欧洲]]第二大[[汽车|汽车]]制造商，同時亦為世界百大企業。2006年，標緻雪鐵龍集团的全球销量达到3,366,000辆，在全球的市场份额达到5.2%。

== 歷史 ==
标致雪铁龙集团的起源可追溯到1896年，那年阿尔芒·标致（Armand Peugeot）先生创建了[[標緻汽車|標緻汽車]]。自此之後，標緻家族就一直是集團的大股東之一。

1976年[[标致汽車|标致汽車]]收購了[[雪铁龙|雪铁龙汽車]]，并成立了PSA标致雪铁龙集团作為其母公司，随后于1978年收购了[[克莱斯勒|克莱斯勒]]欧洲子公司。此後标致雪铁龙集团不僅是欧洲第二大汽车生产厂商，同时也是欧洲第一大轻型商用车生产厂商。

2014年，[[雪铁龙|雪铁龙]]旗下豪華部門[[雪鐵龍DS|雪鐵龍DS]]獨立，成為PSA集團專屬高級車品牌[[DS_Automobiles|DS Automobiles]]。

== 注資 ==
2014年2月19日[[東風集團|東風集團]]與法國政府決定，各自向標緻雪鐵龍投資8億歐元（約85.1億港元），以各自換取14%股權，作為車廠拯救方案的一部份。

東風、法國政府及標緻家族（通過EPF及FFP）將分別持有PSA相同數量的股份，在PSA股東大會上擁有相等的投票權。

== 現今 ==
2017年3月7日，法國[[標緻汽車|標緻汽車]]母公司PSA，同意以13.2億歐元，向美國[[通用汽車|通用汽車]]收購[[歐寶|歐寶]]（Opel）與英國[[佛賀汽車|佛賀汽車]]和以[[法國巴黎銀行|法國巴黎銀行]]組成合資公司以9億歐元（9.55億美金）收購通用歐洲的汽車金融資業務，作為交易代價的一部分，通用獲得標緻4.2%股本的認購期權，惟在期權發行後5年内不得行使。
