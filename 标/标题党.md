'''标题党'''（{{lang-en|Clickbait}}，又稱'''釣魚式標題'''），指网络中故意用较为夸张、耸动的文章标题以吸引[[网友|网友]]点击观看文章或[[帖子|帖子]]的人，特别是與用實際上与内容完全无关或联系不大的文字當標題者<ref>{{cite web|url=http://emarketing.ebdoor.com/EMarketingLib/Articles/8831.aspx | title=由“标题党”谈开去…… |publisher=一比多(EBDoor)网站 |date=2009-04-14 |accessdate=2009-08-09| language=zh-cn}}</ref>。标题党的前身可追朔自十九世纪美国新闻业中存在的[[黄色新闻|黄色新闻]]现象<ref>{{cite news|title=谁都可能是“标题党”幕后推手|url=http://zqb.cyol.com/html/2012-05/31/nw.D110000zgqnb_20120531_3-01.htm|publisher=中青网|date=2012-05-31}}</ref>。

== 特点 ==
在网络中，一些网友在发帖时为了能在海量的资讯中较快的吸引网友的注意，使用一些较为有新意，同时也比较夸张的文章标题，但其文章的实际内容与文章标题关系不大，甚至是毫无关联，只是字面意思所产生的误差所致。例如，某文章以“闹市中惊现悬挂于车外的裸尸”作为标题，其内容只是一张一只已死且被已拔毛的[[鸡|鸡]]悬挂在一辆[[轿车|轿车]]的后尾的图片。<ref>{{cite web|url=http://it.people.com.cn/GB/8219/58830/58834/4860447.html | title=BBS标题党大全，看看你有没有被骗过！ |publisher= 人民网 |date=2006-09-26 |accessdate=2009-08-09| language=zh-cn}}</ref>或純以某些具吸引力的網絡資源名稱作為標題，看似內文有提供該資源，但其實只是發帖者詢問怎樣/何時能取得該資源。

== 影响 ==
标题党作为一种作为较容易被注意的手段，目前在[[中華人民共和國|中国]]，[[香港|香港]]已经有延伸到报刊杂志<ref>{{cite web |url=http://juetuzhi.cn/2009/04/title-party.html |title=报纸也玩标题党 |publisher=掘图志 |date=2009-04-07 |accessdate=2009-08-09 |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20090410145416/http://juetuzhi.cn/2009/04/title-party.html |archivedate=2009-04-10 }}</ref>（如《父涉姦兒子女友被捕》<ref>{{cite web|url=http://orientaldaily.on.cc/cnt/news/20131205/00176_053.html | title=父涉姦兒子女友被捕 |publisher= [[東方日報_(香港)|東方日報 (香港)]] |date=2013-12-05 |accessdate=2014-06-23| language=zh-hk}}</ref>）、出版界<ref>{{cite web |url=http://news.sxpmg.com/ynxx/gndt/200907/10017.html |title=出版界的“标题党”风：哗众取宠湮没内容 |publisher=山西出版传媒网 |date=2009-07-20 |accessdate=2009-08-09 |language=zh-cn }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>的事例，而一些[[影视剧|影视剧]]、[[舞台剧|舞台剧]]的名称也随之模仿。<ref>{{cite web |url=http://pic.nba.tom.com/vw/129074-1.html |title=“标题党”靠模仿忽悠受众 |publisher=[[文汇报|文汇报]]、和讯读书 |date=2009年7月17日 |accessdate=2009-08-09 |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20090517030613/http://pic.nba.tom.com/vw/129074-1.html |archivedate=2009年5月17日 }}</ref>而在[[NBA|NBA]]上的一些[[标语|标语]]，也被认为是标题党。<ref>{{cite web |url=http://pic.nba.tom.com/vw/129074-1.html |title=终极标题党 NBA赛场各色标语集合 |publisher=NBA中国官方网 |date=2009-05-12 |accessdate=2009-08-09 |language=zh-cn |deadurl=yes |archiveurl=https://web.archive.org/web/20090517030613/http://pic.nba.tom.com/vw/129074-1.html |archivedate=2009-05-17 }}</ref>

2015年10月9日，中国网信办在其网站发表了题为《标题党、图片党该收手了》的文章，抨击部分中国媒体从业者忽视甚至无视标题与内容的贴切关系，骗取点击率，文不对题的乱象。<ref name="网信办">{{cite web|url=http://www.cac.gov.cn/2015-10/09/c_1116764701.htm| title=“标题党”“图片党”该收手了 |publisher= 中国网信办 |date=2015-10-09 |accessdate=2015-10-09| language=zh-cn}}</ref>

2017年1月，中国网信办进一步打击各门户网站出现的“标题党”行为，并设立商业网站“标题党”问题举报专区，要求中国6家主要商业网站在其首页显著位置设置举报入口。<ref>{{cite web|title=国家网信办深入整治“标题党”问题|url=http://news.xinhuanet.com/newmedia/2017-01/13/c_1120303875.htm|website=中国网信网|publisher=新华网|accessdate=2017-04-01|date=2017-01-13}}</ref>
