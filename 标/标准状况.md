'''标准状况'''（{{lang-en|'''standard temperature and pressure''', '''STP'''}}，'''标准温度与标准压力'''），简称「'''标况'''」。由於地表各處的[[溫度|溫度]]、[[壓力|壓力]]皆不同，即使是同一地點的溫度壓强也隨測量[[時間|時間]]不同而相異，因此為研究方便，制定出描述物質特徵的標準狀況。

在化学领域，IUPAC在1982年改变了标准温度和标准压力（STP）的定义：

*在1982年以前，标准状况定义为0 [[攝氏溫標|℃]]（273.15 [[热力学温度单位|K]]）、1.01325 × 10<sup>5</sup> [[帕斯卡_(单位)|Pa]]（1 [[气压|atm]]）。1摩尔的[[理想氣體|理想氣體]]在STP下占有的體積為22.414。

* 自1982年以来，标准状况定义为0 ℃（273.15 [[熱力學溫標|K]]）、100[[帕斯卡|kPa]]，這樣的定義接近[[海平面|海平面]]上[[水|水]]的[[冰點|冰點]]。1摩尔的[[理想氣體|理想氣體]]在STP下占有的體積為22.710953（21）L(CODATA 2010)，為標準摩尔體積（{{lang|en|standard molar volume}}）。這是区别[[一般条件|一般条件]]而制定的标准。

历史上由巴黎制定的通行欧洲的舊制標準如下：

* [[压强|压强]]''p'' = 1 atm = 101325 [[帕斯卡|帕斯卡]] = 1013.25 hPa = 101.325 kPa = 1.01325 bar.
* [[温度|温度]]''T'' = 298.15 K ≙ '''25 °C'''，空气密度''ρ'' = 1.184 kg/m³.
:现在也使用''T'' = 293.15 K ≙ 20 °C  空气密度''ρ'' = 1.204 kg/m³.

{{citation needed|在[[化学|化学]]和[[生物化学|生物化学]]中另外还包括定义的观察反应物浓度，一般情况下为1mol/L（1 M）。}}在生物化学中只有[[水合氢离子|氢离子]]例外（H<sup>+</sup>）。限定[[pH值|pH]]为7，即10<sup>-7</sup> mol/l H<sup>+</sup>离子浓度用于准生理条件描述，肌体条件，细胞内部条件等。

== 气体摩尔体积 ==
{{Main|摩尔体积}}
当表示气体的[[摩尔体积|摩尔体积]]时，要指明适用的温度和压力条件。 在不指明温度和压力条件的情况下，说明气体的摩尔体积几乎没有意义并且可能引起混淆。

通过使用[[理想气体状态方程|理想气体定律]]，可以计算在标准状况下的气体摩尔体积。 任何理想气体的摩尔体积可以在各种标准参考条件下计算，如下所示：

* ''V''<sub>m</sub> = 8.3145 × 273.15 / 101.325 = 22.414 dm<sup>3</sup>/mol （在 0 °C 和 101.325 kPa 下）
* ''V''<sub>m</sub> = 8.3145 × 273.15 / 100.000 = 22.711 dm<sup>3</sup>/mol （在 0 °C 和 100 kPa 下）
* ''V''<sub>m</sub> = 8.3145 × 298.15 / 101.325 = 24.466 dm<sup>3</sup>/mol （在25 °C 和 101.325 kPa 下）
* ''V''<sub>m</sub> = 8.3145 × 298.15 / 100.000 = 24.790 dm<sup>3</sup>/mol （在25 °C 和 100 kPa 下）
* ''V''<sub>m</sub> = 10.7316 × 519.67 / 14.730 = 378.61 ft<sup>3</sup>/lbmol （在60 °F 和 14.73 psi 下）
