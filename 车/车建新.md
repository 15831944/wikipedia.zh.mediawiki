==创业经历==
1986年，车建新开始第一次创业。他借资600元创办了手工作坊，制作新款家具。<ref name="家居大佬登上亚洲富豪榜 车建新等人上榜">{{cite web | url=http://xian.qq.com/a/20120216/000135.htm | title=家居大佬登上亚洲富豪榜 车建新等人上榜 |publisher=腾讯网|}}</ref>。1987年，创办青龙木器厂。“青龙木器厂”成为车建新家居创业之路的第一站，也是红星美凯龙的雏形<ref name="600元到500亿，红星美凯龙30年传奇故事">{{cite web | url=http://www.sohu.com/a/86115292_106176 | title=600元到500亿，红星美凯龙30年传奇故事 |publisher=搜狐网|}}</ref>。

1991年，车建新投资100多万元创办常州市及周边地区的第一家大型家具专营商场——“红星家具城”<ref name="车建新-人物频道-和讯网">{{cite web | url=http://renwu.hexun.com/figure_281.shtml | title=车建新-人物频道-和讯网 }}</ref>。此后不到两年，他在常州又创办了5家红星家具城，走上了连锁化经营的道路<ref name="家居大佬登上亚洲富豪榜 车建新等人上榜">{{cite web | url=http://xian.qq.com/a/20120216/000135.htm | title=家居大佬登上亚洲富豪榜 车建新等人上榜 |publisher=腾讯网|}}</ref>；1994年6月27日，车建新注册资金10000万人民币<ref name="600元到500亿，红星美凯龙30年传奇故事">{{cite web | url=http://www.sohu.com/a/86115292_106176 | title=600元到500亿，红星美凯龙30年传奇故事 |publisher=搜狐网|}}</ref>，创立常州红星家具集团，主营家具制造、加工以及销售；2003年6月19日和7月18日，长沙红星美凯龙和上海红星美凯龙梅陇店相继开业，这是红星集团与沃尔玛、欧倍德(OBI)、海尔家居、大连万达达成品牌战略联盟后的第一次联手，也是中国家居业与世界大商业卖场的首度合作<ref name="车建新-人物频道-和讯网">{{cite web | url=http://renwu.hexun.com/figure_281.shtml | title=车建新-人物频道-和讯网 }}</ref>；2015年6月26日，红星美凯龙正式登陆港交所，在A股排队等候IPO多年未果的红星美凯龙终于上市<ref name="红星美凯龙苦等8年终上市 能否逆市突围仍存疑">{{cite web | url=http://finance.qq.com/a/20150626/043479.htm | title=红星美凯龙苦等8年终上市 能否逆市突围仍存疑 |publisher=腾讯财经|}}</ref>；2016年5月13日，红星美凯龙发布A股首次公开发行股票招股说明书，宣布开启A股市场第二次IPO征程<ref name="红星美凯龙再谋A股IPO 募资62.5亿元">{{cite web | url=http://money.163.com/16/0530/08/BOA598GM00251LKI.html | title=红星美凯龙再谋A股IPO 募资62.5亿元 |publisher=网易财经|}}</ref>。

2017年7月，车建新宣布升级企业的新零售系统，称“从今以后，红星美凯龙不再有线上团队线下团队之分，集团2万多名员工都是互联网员工”<ref name="车建新喊话传统企业家：人工智能时代、新零售时代，谁都别当甩手掌柜！">{{cite web | url=http://news.sina.com.cn/o/2017-07-27/doc-ifyinwmp0275387.shtml | title=车建新喊话传统企业家：人工智能时代、新零售时代，谁都别当甩手掌柜！ |publisher=新浪网|}}</ref>。车建新认为，生产企业不走智能化，就跟不上形势；流通企业不走新零售，一定死路一条<ref name="车建新喊话传统企业家：人工智能时代、新零售时代，谁都别当甩手掌柜！">{{cite web | url=http://news.sina.com.cn/o/2017-07-27/doc-ifyinwmp0275387.shtml | title=车建新喊话传统企业家：人工智能时代、新零售时代，谁都别当甩手掌柜！ |publisher=新浪网|}}</ref>。
