'''内阁官房长官'''（{{lang-ja|'''内閣官房長官'''}}、{{lang-en|Chief Cabinet Secretary}}）（簡稱'''官房長官'''）为[[日本国政府|日本国政府]]根据{{link-ja|内閣法|内阁法}}设置[[內閣官房|内阁官房]]的最高[[首長|首長]]，相當於[[日本內閣|日本內閣]][[秘書長|秘書長]]，在[[第二次世界大戰|第二次世界大戰]]前稱為'''{{link-ja|内閣書記官長|内閣書記官長}}'''。

在不设[[副總理_(日本)|副總理]]的情况下，官房長官是日本内阁中仅次于[[日本內閣總理大臣|內閣總理大臣]]（首相）的职位。

== 职务 ==
内阁官房长官统辖内阁官房事务，负责監督管理部门职员的服务（内阁法第13条）。

但是，内阁官房在内阁法上的[[主任大臣|主任大臣]]是内阁总理大臣（内阁法25条）。

内阁官房的事务涵蓋政府几乎所有的领域，因此总管的官房长官职务极其广泛。今日的官房长官发挥尤为重要的作用，主要内容如下。

#对内阁的各种狀況協調行政各部。
#與[[国会_(日本)|国会]]各{{link-ja|院内会派|会派}}（特别是执政党）的協調角色。
#針對日本政府（内阁）处理的重要事项，以及各种事态發表政府正式立场的“政府发言人”。

办公室在[[总理大臣官邸|总理大臣官邸]]5楼，配有一位{{link-ja|特別職|特別職}}{{link-ja|國家公務員|国家公務員}}{{link-ja|秘書官|秘書官|國務大臣秘書官}}<ref group="注釋">還有多名從各省廳借調的秘書官事務處理。</ref>。另外，也可以根据需求配置一位特别职的[[大臣補佐官|大臣補佐官]]。

内阁官房长官是{{link-ja|閣議 (日本)|閣議 (日本)|閣議}}的主持者。

除此之外，内阁官房长官也負責總管[[內閣府|內閣府]]（{{link-ja|大臣委員會|大臣委員会}}及[[內閣府特命擔當大臣|特命擔當大臣]]所掌部署除外）事務（{{link-ja|內閣府設置法|内閣府設置法}}8條），包含[[大臣官房|大臣官房]]、{{link-ja|賞勲局|賞勲局}}、[[迎賓館|迎賓館]]、{{link-ja|官民人材交流中心|官民人材交流センター}}、{{link-ja|再就職等監視委員會|再就職等監視委員会}}、[[國際平和協力本部|國際平和協力本部]]、[[宮內廳|宮內廳]]、[[公平交易委員會_(日本)|公平交易委員會]]等。

[[總理大臣官邸|總理大臣官邸]]建地內的官房長官公邸在2002年（[[平成|平成]]15年）3月啟用，在緊急事態發生時可供官房長官住宿以隨時因應情況變化，非一般居住用官邸<ref>[http://www.shugiin.go.jp/internet/itdb_shitsumon.nsf/html/shitsumon/b177082.htm 衆議院議員浅野貴博君提出内閣官房長官公邸の必要性に関する質問に対する答弁書]</ref>。

== 概要 ==
在[[大眾媒體|大眾媒體]]的報導中，內閣官房長官與[[日本內閣總理大臣|內閣總理大臣]]並列為國民曝光度最高的職位。在實務上，隨著[[中央省廳再編|中央省廳再編]]與之後逐步加強官邸功能，其權限也不斷擴大。在重大案件中可見官房長官的協調能力，是內閣的關鍵人物。

[[日本國憲法|現有憲法下]]，內閣官房長官最初並不是需要[[天皇|天皇]]認證的認證官。1963年（[[昭和|昭和]]38年），時任首相[[池田勇人|池田勇人]]認為內閣官房長官是接收首相指令與[[大臣_(日本)|大臣]]溝通的橋樑，認為與大臣有同等地位<ref name="星">{{Cite book|author=星浩|title=官房長官 側近の政治学|publisher=朝日新聞出版|series=朝日選書921|pages=16–18頁|year=2014|isbn=9784022630216}}</ref>，因此[[第2次池田內閣_(第2次改造)|第2次池田內閣 (第2次改造)]]時代的同年6月11日，內閣官房長官成為{{link-ja|認證官|認証官}}<ref name="星"/>。從此，內閣官房長官擁有大臣待遇。之後，隨著電視曝光度的增加，加上每日透過電視撥放的記者會，內閣官房長官也被稱作「政權」之顔<ref name="星"/>。近年多由重量級政黨要角就任。現在已是實質上的內閣第二號人物。

做為未來首相候選人的[[登龍門|登龍門]]，內閣官房長官多由年輕的明日之星、或擁有政策調整能力的資深政治家出任，一般都是首相最重要的左右手。[[自由民主黨_(日本)|自由民主黨]]執政時，多由總裁[[自由民主黨派閥|派閥]]（首相出身派閥）出任。

在媒體報導中，若是內閣總理大臣的非正式發言，慣例上會以「政府首腦」取代職稱。

2000年（平成12年）4月起，{{link-ja|內閣總理大臣臨時代理|内閣総理大臣臨時代理}}名單列有五位，其中內閣官房長官通常排名第1或第2位。當內閣官房長官以外的國務大臣名列第1位時，該國務大臣稱為[[副總理|副總理]]。
由於需代理時常出訪海外的首相，負責國內的危機管理，因此內閣官房長官鮮少出訪海外<ref group="注釋">內閣官房長官出外的案例有1995年（平成7年）9月的野坂浩賢、2003年（平成14年）9月的[[福田康夫|福田康夫]]、2015年（平成27年）10月的菅義偉。</ref>。若是內閣官房長官離開[[東京|東京]]，則由作為行政最高責任者的內閣總理大臣留守東京。

== 辅佐职 ==
[[Image:Soridaijinkantei3.jpg|thumb]]]]
补佐内阁官房长官的职位如下：
* [[内閣官房副長官|内閣官房副長官]]3人（内阁法第14条）※{{link-ja|認証官|認証官}}
* [[國家安全保障會議|国家安全保障局長]]（内阁法第17条第3項）
* {{link-ja|内閣危機管理監|内閣危機管理監}}（内阁法第15条）
* {{link-ja|内閣情報通信政策監|内閣情報通信政策監}}（内阁法第15条）
* {{link-ja|内閣官房副長官補内閣官房副長官補}}3人（内阁法第18条）
* {{link-ja|内閣廣報官|内閣広報官}}（内阁法第19条）
* {{link-ja|内閣情報官|内閣情報官}}（内阁法第20条）
* {{link-ja|内閣總務官|内閣総務官}}（内閣官房組織令第2条第2項）
* 内閣府[[大臣補佐官|大臣補佐官]]1人（内阁官房长官可以根据自己的要求来设置）

== 沿革 ==
* 1879年（[[明治|明治]]12年）3月12日 - 設置內閣官房長官的前身'''內閣書記官長'''，下屬有大書記官、少書記官。
* 1885年（明治18年）12月22日 - 創立內閣制度，成為正式常設職位。
* 1890年（明治23年）6月30日 - 公布內閣所屬職員官制，為內閣所屬的勅任官。職則為「依內閣總理大臣命令管理機密文書、監督內閣事務、執行內閣所屬{{link-ja|判任官|判任官|判任}}以下職員人事權）」。
* 1898年（明治31年）10月22日 - 對內閣所屬各局局長的指揮權改為命令權。
* 1924年（[[大正|大正]]13年）12月20日 - 內閣所屬職員官制全面修正，書記官長直屬部局改組為內閣官房。職責加上「輔佐內閣總理大臣」。
* 1947年（[[昭和|昭和]]22年）5月3日 - [[日本國憲法|日本國憲法]]施行，廢除原來的'''內閣書記官長'''，依行政官廳法新設'''內閣官房長官'''。由於非[[國務大臣|國務大臣]]所屬職位，由國務大臣出任內閣官房長官時，人事令為「兼任內閣官房長官」。非國務大臣出任時，人事令為「任命內閣官房長官」。
* 1949年（昭和24年）6月1日 - 行政官廳法失效，內閣法成為基礎法。該法允許國務大臣出任職位，因此由國務大臣出任內閣官房長官時，人事令改為「任命內閣官房長官」。
* 1963年（昭和38年）6月11日 - 時任首相[[池田勇人|池田勇人]]<ref name="星"/>修正內閣法，內閣官房長官成為{{link-ja|認證官|認証官}}<ref name="星"/>。由國務大臣出任內閣官房長官時，以國務大臣身分認證，非國務大臣出任時，以內閣官房長官認證。
* 1966年（昭和41年）6月28日 - 修正內閣法，內閣官房長官為國務大臣所屬職位。
* 1984年（昭和59年）7月1日 - 隨著總務廳設置，負責總理府（大臣廳等除外）綜合管理。
* 2000年（[[平成|平成]]12年）4月5日 - 內閣總理大臣臨時代理預定者改為組閣前5順位，原則上由內閣官房長官擔任第1順位。
* 2001年（平成13年）1月6日 - [[中央省廳再編|中央省廳再編]]，負責[[內閣府|內閣府]]（大臣廳等除外）綜合管理。
