'''内华达州'''（{{lang-en|State of Nevada}}）是[[美国|美国]]西部的一个[[美国州份|州]]，以其发达的[[赌博|赌博]]业而著称。该州的邮政缩写是'''NV'''。

“内华达”来源于[[西班牙语|西班牙语]]，意思是“被雪覆盖”，原本是用来形容该州西部的[[內華達山脈_(美國)|内华达山]]的寒冷。多艘[[美国海军|美国海军]]的军舰也被叫作“内华达号”，名字就来源于内华达州。

在南部林肯郡區域是美國秘密研發和測試飛行器的地點，也曾發生過[[不明飛行物|不明飛行物]]墜毀事件，該地點為[[51區|51區]]。

== 历史 ==

1859年在内华达州发现了{{Link-en|康斯塔克矿|Comstock Lode}}，这里盛产[[金|金]]和[[银|银]]，[[弗吉尼亚城_(内华达州)|弗吉尼亚城]]开始繁荣起来。这次的大发现吸引了无数采矿者和商人，希望能够挖掘到宝藏。

1861年3月2日，内华达州脱离犹他领地，以[[內華達山脈_(美國)|內華達山脈]]為名。因為林肯總統爭取美國總統連任時，希望多一個支持北方的州來增加總統選舉人票，加上當時以礦業為基礎的內華達領地民眾也支持北方工業化的訴求，於是剛成立的內華達州在美國總統大選前八天，不惜用電報將整本州憲法傳送到美國國會，以取-{得}-州權，並創下當時最長電報內容的紀錄。1864年10月31日成为美国第36个州。

1930年代的[[大萧条|大萧条]]严重打击了内华达州经济，1931年3月17日，州政府决定开放[[博彩|博彩]]业，以刺激经济发展。

== 法律和政府 ==

内华达州州府设在[[卡森城_(内华达州)|卡森市]]，现任州长是[[美国民主黨|民主黨]]的{{Link-en|史蒂夫·西苏拉克|Steve Sisolak}}。

== 地理 ==

内华达州与[[俄勒冈州|俄勒冈州]]、[[爱德华州|爱德华州]]、[[加利福尼亚州|加利福尼亚州]]、[[亚利桑纳州|亚利桑纳州]]和[[犹他州|犹他州]]接壤。与亚利桑纳州的交界区域内有[[科罗拉多河|科罗拉多河]]。内华达州是美国最干燥的一个州，全州大部分地区都是白雪覆盖的高[[山|山]]、[[峡谷|峡谷]]和[[沙漠|沙漠]]。

== 经济 ==

1999年内华达州的经济总规模是690亿美元，居全国第32位。2000年人均收入为30,529美元，全国第14。农产品包括了[[牛肉|牛肉]]、干草、[[牛奶|牛奶]]制品和[[马铃薯|马铃薯]]。工业产出主要来自[[旅游|旅游]]、采矿、机械、[[出版|出版]]印刷、[[食品|食品]]加工处理和电子器械。内华达州的博彩业和[[夜生活|夜生活]]举世闻名。[[拉斯维加斯|拉斯维加斯]]豪华、大型的[[赌场|赌场]]每年吸引着全球数千万的游客光顾。著名的多家赌博和酒店公司位于内华达州，如[[美高梅國際酒店集團|美高梅國際酒店集團]]，[[拉斯維加斯金沙集團|拉斯維加斯金沙集團]]、{{link-en|凯撒娱乐集团|Caesars Entertainment Corporation}}。

内华达是全美唯一一个允许[[性交易|性交易]]的州，性交易在全州大部分地方都是合法的，但有趣的是，作为旅游热门目的地的[[拉斯维加斯|拉斯维加斯]]中性产业却是非法的。

== 人口 ==

根据2000年的[[人口普查|人口普查]]，內華達州共有人口1,711,263人，人口自1990年增長了8.4%（132,846）。根据2000年的人口普查： 
* 89.6%（1,533,261）為白人 
* 5.5%（94,425）為拉美裔 
* 4%（68,541）是黑人
* 1.3%（21,931）為亞裔 
* 0.9%（14,896）是印第安或原住民
* 0.05%（836）是[[夏威夷|夏威夷]]人或太平洋民
* 2.8%（47,845）為其他族群
* 1.4%（23,953）屬於多於一個族群
 
美国2012年人口估算显示，该州人口达到275.9万。

内华达州6.8%的[[人口|人口]]年龄低于5岁，26.3%未成年（低于18岁），13.6%为65岁或以上的老龄人口。50.7%的人口为妇女。

内华达州的[[离婚率|离婚率]]在美国所有州中排名第一。
