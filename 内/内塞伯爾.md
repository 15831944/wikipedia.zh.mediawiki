'''內塞伯爾'''（[[保加利亞語|保加利亞語]]:{{lang|bg|Несебър}}、[[拉丁化|拉丁化]]：''Nesebar''）是[[保加利亞|保加利亞]]的一座歷史古城，今天則是[[布爾加斯州|布爾加斯州]]的一個沿海度假城市。在[[色雷斯語|色雷斯語]]中的名稱是Menebria、現代[[希臘語|希臘語]]中的名稱是{{lang|el|Μεσημβρια}}（''Mesimvria''），舊名墨森布里亞（''Mesembria''）。內塞伯爾有著超過3000年的悠久歷史，可說是一座城市博物館。

內塞伯爾也是黑海最知名的觀光地和度假地。內塞伯爾附近有不少度假地，其中最大的是位於市北的[[陽光海灘|陽光海灘]]（Sunny Beach）。

內塞伯爾古代都市建設在一個通過細長的人工地峽與大陸連接的半島（過去曾是島嶼）上，因地形關係而得以逃脫眾多其他文明的征服。內塞伯爾眾多的歷史建築使其在1983年被[[聯合國教科文組織|聯合國教科文組織]]列入[[世界遺產|世界遺產]]。

2009年12月，內塞伯爾的人口有11,626人<ref name="statistika">{{en icon}} [http://www.nsi.bg/otrasalen.php?otr=53&a1=1583&a2=1584&a3=1588#cont Bulgarian National Statistical Institute - towns in 2009]</ref>。

== 歷史 ==
內塞伯爾在公元前1000年時的名稱是Menebria ，當時是色雷斯人的村莊。在公元前6世紀早期，[[墨伽拉|墨伽拉]]的[[多利安人|多利安人]]來到這裡，內塞伯爾成為希臘的殖民地。在公元前425-424年，城市加入了[[提洛同盟|提洛同盟]]<ref>{{cite web|last=Petropoulos|first=Ilias|title=Mesembria (Antiquity)|url=http://www.ehw.gr/blacksea/Forms/fLemmaBodyExtended.aspx?lemmaID=11229|work=Εγκυκλοπαίδεια Μείζονος Ελληνισμού, Εύξεινος Πόντος|accessdate=8 June 2011}}</ref>。之後，內塞伯爾發展為重要的交易據點，也是阿波羅尼亞（[[索佐波爾|索佐波爾]]）的競爭對手。[[希臘化時代的希臘|希臘化時代的希臘]]時代的遺物有[[衛城|衛城]]、[[阿波羅|阿波羅]]神殿和[[阿哥拉|阿哥拉]]。在公元前5世紀開始鑄造[[青銅|青銅]]、[[銀|銀]]的硬幣，公元前3世紀時鑄造了[[金|金]]的硬幣。
城市在公元前71年被[[羅馬帝國|羅馬帝國]]征服，但仍然擁有鑄造自己的硬幣的特權。在5世紀之後，內塞伯爾是[[東羅馬帝國|東羅馬帝國]]時代最重要的要塞之一，東羅馬帝國和[[保加利亞第一帝國|保加利亞第一帝國]]之間曾展開戰爭。812年，保加利亞帝國的[[克盧姆|克盧姆]]將這座城市納入保加利亞的支配之下，但經過了兩週的包圍之後，東羅馬收復了這座城市。864年，保加利亞皇帝[[西美昂一世|西美昂一世]]再次征服了這裡，城市再度成為保加利亞領土。

在[[保加利亞第二帝國|保加利亞第二帝國]]時期，內塞伯爾仍是保加利亞和東羅馬相互爭奪的城市。保加利亞帝國的[[伊萬·亞歷山大|伊萬·亞歷山大]]皇帝（1331年-1371年在位）統治時期，城市十分繁榮。[[薩伏伊伯爵|薩伏伊伯爵]][[阿梅迪奧六世|阿梅迪奧六世]]率領的[[十字軍|十字軍]]後來征服了城市。1366年，城市再度被東羅馬帝國統治。城市的[[斯拉夫語|斯拉夫語]]名稱内塞伯爾（''Nesebar''）或墨塞伯爾（''Mesebar''）這兩個名稱是在11世紀之後開始使用的。

[[中世紀|中世紀]]時期的建築物有建于5世紀至6世紀的「古主教區」（{{lang|bg|Стара Митрополия}} / ''Stara Mitropoliya''，聖索菲亞教堂）、10世紀的教堂（聖母教堂）、11世紀的「新主教區」（{{lang|bg|Нова Митрополия}} / ''Nova Mitropoliya''、聖斯蒂芬教堂）、聖帕拉斯科瓦教堂、聖刁多禄教堂、聖天使邁克爾和聖天使加百利教堂、聖約翰阿利特格托斯教堂等。

1453年，城市的統治者由東羅馬帝國改為[[奧斯曼帝國|奧斯曼帝國]]，城市雖然開始了衰退，但歷史建築物得到了保存。在19世紀，保加利亞黑海沿岸地區典型的[[東魯米利亞|東魯米利亞]]様式的木造建築民宅開始增加。在19世紀早期，許多當地人都參加了希臘愛國主義組織<ref name=ehw/>。1878年保加利亞解放之後，内塞伯爾是奧斯曼帝國自治州「[[東魯米利亞自治省|東魯米利亞自治省]]」的一部份，1886年，東魯米利亞自治省和保加利亞合併，城市也成為保加利亞領土。

在19世紀末期，内塞伯爾是希臘人漁民、葡萄酒生產者居住的小漁村。20世紀初期，城市的人口有1,870人<ref name=ehw/>。進入20世紀后，城市是保加利亞沿海地區的重要度假地，得到了開發。在1925年，進行了民族交換，希臘人離開了城市，建設了新市區，歷史地區也得到了修復。

== 教堂 ==
有很多人都認為内塞伯爾的人均教堂數在世界也是屈指可數<ref>{{cite web |url=http://www.villabeneia.com/Site/okolnostite.htm |title=存档副本 |accessdate=2006-08-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20070222105747/http://www.villabeneia.com/Site/okolnostite.htm |archivedate=2007-02-22 }}</ref><ref>http://bg.zonebulgaria.com/cherno_more/nesebar/istoria/</ref>，雖然這不一定是事實，然而內塞伯爾的極為多樣的教堂風格仍然值得一提<ref name=ehw>{{cite web|last=Doncheva|first=Svetlana|title=Mesimvria (Nesebar)|url=http://www.ehw.gr/blacksea/Forms/fLemmaBodyExtended.aspx?lemmaID=11561|work=Εγκυκλοπαίδεια Μείζονος Ελληνισμού, Εύξεινος Πόντος|accessdate=8 June 2011}}</ref>。内塞伯爾具代表性的教堂有：

* 古主教區／聖索菲亞教堂（5世紀-6世紀）
* 聖母大教堂（6世紀）
* 施洗約翰教堂（11世紀）
* 新主教區／聖斯蒂芬聖堂（11世紀。在16世紀-18世紀修復）
* 聖刁多禄聖堂（13世紀）
* 聖帕拉斯科瓦教堂（13世紀-14世紀）
* 聖天使邁克爾和聖天使加百利教堂（13世紀-14世紀） 
* 全能者教堂（13世紀-14世紀）
* 聖約翰阿利特格托斯教堂（14世紀）
* 聖斯帕斯教堂（17世紀）
* 聖克萊門特聖堂（17世紀）

內塞伯爾的教堂群集合了東羅馬帝國時代、保加利亞帝國時代、奧斯曼帝國時代的建築，是[[正教會|正教會]]重要的建築遺跡，在內塞伯爾也能看到從早期基督教樣式教堂到中世紀十字穹頂式教堂的変遷過程。
