[[File:三角形の内心.png|right]]會相交於內切圓的[[圓心|圓心]]]]
在[[數學|數學]]中，若一個[[二維|二維]]平面上的[[多邊形|多邊形]]的每條邊都能與其內部的一個[[圓形|圓形]]相切，該圓就是多邊形的'''內切圓'''，這時稱這個多邊形為圓外切多邊形。它亦是多邊形內部最大的圓形。内切圓的圓心被稱為該多邊形的'''内心'''。

一個多邊形至多有一個内切圓，也就是說對於一個多邊形，它的内切圓，如果存在的話，是唯一的。並非所有的多邊形都有内切圓。[[三角形|三角形]]和[[正多邊形|正多邊形]]一定有内切圓。擁有内切圓的四邊形被稱為[[圆外切四边形|圆外切四边形]]。

==三角形的內切圓==
任何[[三角形|三角形]]<math>ABC</math>都有內切圓。這個內切圓的[[圓心|圓心]]稱為'''內心'''，一般标记为''I''，是三角形內[[角平分線|角平分線]]的交點<ref>R.A.约翰逊，《近代欧氏几何学》，单墫 译，第158页，上海教育出版社，ISBN 7-5320-6392-5</ref>。在[[三線坐標|三線坐標]]，內心是1:1:1。

===性质===
內切圓的[[半徑|半徑]]為<math>\frac{2\triangle}{a+b+c}</math>，當中<math>\triangle</math>表示三角形的[[面積|面積]]。

以內切圓和三角形的三個切點為頂點的三角形<math>T_A T_B T_C</math>是<math>ABC</math>的[[内接三角形|内接三角形]]之一。<math>ABC</math>的內切圓就是<math>T_A T_B T_C</math>的[[外接圓|外接圓]]。而<math>AT_A</math>、<math>BT_B</math>和<math>CT_C</math>三线交于一点，它们的交點就是[[熱爾崗點|熱爾崗點]]（Gergonne point）。内切圆与[[九点圆|九点圆]]相切，切点称作费尔巴哈点（见[[九点圆|九点圆]]）。

若以三角形的内切圆为反演圆进行[[反演|反演]]，则三角形的三条边和外接圆会分别变为半径相等的四个圆（半径都等于内切圆半径的一半）。<ref>《近代欧氏几何学》，第163页</ref>

三角形的外接圆半径''R''、内切圆半径''r'' 以及内外心间距''OI'' 之间有如下关系：
:<math>R^2 - OI^2 = 2Rr</math><ref>《近代欧氏几何学》，第162页</ref>

直角三角形兩股和等於斜邊長加上該三角形內切圓直徑
:<math>a+b=c+2r</math>

由此性質再加上[[勾股定理|勾股定理]]<math>a^2+b^2=c^2</math>，可推得：
:<math>\triangle =r(r+c)</math>

在[[直角座標系|直角座標系]]中，若頂點的[[座標|座標]]分別為<math>(x_1,y_1)</math>、<math>(x_2,y_2)</math>、<math>(x_3,y_3)</math>，則内心的座標為：
:<math>(\frac{ax_1+bx_2+cx_3}{a+b+c},\frac{ay_1+by_2+cy_3}{a+b+c})</math><ref>{{cite web | title =平面向量教学与三角形内心 | url =http://www.cnki.com.cn/Article/CJFDTotal-SXTB200903009.htm}}</ref>

==四边形的内切圆==
不是所有的四边形都有内切圆，拥有内切圆的四边形称为圆外切四边形。凸四边形ABCD有内切圆[[当且仅当|当且仅当]]两对对边之和相等：<math>AB+CD = AD+BC</math>。圆外切四边形的面积和内切圆半径的关系为：
<math>S_{ABCD} = rs</math>，其中''s'' 为半周长。

同时拥有内切圆和外接圆的四边形称为双心四边形。这样的四边形有无限多个。若一个四边形为双心四边形，那么其内切圆在两对对边的切点的连线相互垂直。而只要在一个圆上选取两条相互垂直的[[弦|弦]]，并过相应的顶点做切线，就能得到一个双心四边形。

==正多边形的内切圆==
正多边形必然有内切圆，而且其内切圆的圆心和外接圆的圆心重合，都在正多边形的中心。边长为''a'' 的正多边形的内切圆半径为：
:<math>r_n = \frac{a}{2} \cot \left(\frac{\pi}{n} \right)</math>
其内切圆的面积为：
:<math>s_n =\pi r_n^2 = \frac{ \pi a^2}{4} \cot^2 \left(\frac{\pi}{n} \right)</math>
内切圓面積<math>s_n</math>與正多邊形的面積<math>S_n</math>之比為：
:<math>\varphi_n = \frac{s_n } { S_n } =  \dfrac{ \frac{ \pi a^2}{4} \cot^2 \left(\frac{\pi}{n} \right) } { \frac{ n a }{2} \left[ \frac{a}{2} \cot\left( \frac{\pi}{n} \right)\right] } = \frac{\pi}{n} \cot \left(\frac{\pi}{n} \right)</math>
故此，當正多邊形的邊數<math>n</math>趨向[[無窮|無窮]]時，
:<math>\lim_{n \to \infty} \varphi_n = \lim_{n\to \infty} \frac{ \pi} {n} \cot \left( \frac{ \pi} { n } \right) = \lim_{n \to \infty} \cos^2 \left( \frac{ \pi } { n } \right) = 1</math>
