==系統==
《緣染此葉化戀紅》是[[日本成人遊戲|成人]][[戀愛冒險遊戲|戀愛冒險遊戲]]。玩家在玩《緣染此葉化戀紅》時主要是觀看和聆聽故事情節的發展，不過在部分的預設段落中玩家須藉由點擊的方式選擇行為選項，而這些選擇將會影響之後的人物的行動或者是反應。遊戲藉由這些相互連接的劇情分歧點而組織成多條故事路線，而遊戲的選項便會影響玩家所挑選的故事劇情內容，包含玩家操控的角色和遊戲人物之間的色情[[计算机图形|場景]]，並且進入不同的結局。

== 故事簡介 ==
幻夷是有著獸耳、角和尾巴的種族，而繼椛實町則是人與幻夷兩個種族和平共處的城鎮。由於雙親工作的因素，笹浦清司與笹浦鈴菜兩兄妹搬到了這個鎮上，並由清司雙親的熟人稜未小乃葉照顧。某一天清司出門散步時，遇見了一名幻夷女孩，而這名女孩自稱是清司的寵物<ref>{{cite web|url=http://www.tgsmart.jp/article.aspx?a=19946|title=【TG SCOOP】Lump of Sugar最新作はケモ耳少女とのイチャラブ純愛ADV♪|website=TG Smart|date=2017-08-21|accessdate=2018-05-11|language=ja}}</ref>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|笹浦清司|笹浦 清司（ささうら せいじ）}}
: 作品男主角，繼椛實學園二年級生。擁有治癒的神通力，但代價是會失去部分的記憶。小時候在繼椛實町居住時曾經用能力拯救還是獸型的紅葉，也因此忘記了當時的事情。
; {{nihongo|木那-{里}-紅葉|木那里 もみじ（きなり もみじ）|聲：{{lang|ja|茅ヶ崎リリィ}}<ref>{{cite web|url=https://www.lumpofsugar.co.jp/product/yorikure/character/character01.html|title=木那里 もみじ|publisher=Lump of Sugar|accessdate=2018年9月16日|language=ja}}</ref>}}
: 清司在繼椛實町散步時遇見的幻夷少女，自稱是清司的寵物。
; {{nihongo|笹浦鈴菜|笹浦 すずな（ささうら すずな）|聲：{{lang|ja|桃山いおん}}<ref>{{cite web|url=https://www.lumpofsugar.co.jp/product/yorikure/character/character02.html|title=笹浦 すずな|publisher=Lump of Sugar|accessdate=2018年9月16日|language=ja}}</ref>}}
: 清司的親妹妹，繼椛實學園一年級生。很黏哥哥清司的兄控。
; {{nihongo|稜未小乃葉|稜未 小乃葉（たるみ このは）|聲：{{lang|ja|春乃いろは}}<ref>{{cite web|url=https://www.lumpofsugar.co.jp/product/yorikure/character/character03.html|title=稜未 小乃葉|publisher=Lump of Sugar|accessdate=2018年9月16日|language=ja}}</ref>}}
: 負責照顧笹浦兄妹的幻夷女性，也是繼椛實町的調整者，外表年幼但實際上已數百歲。總是穿著和服。
; {{nihongo|斑鳩和羽|斑鳩 和羽（いかるが かずは）|聲：奏雨<ref>{{cite web|url=https://www.lumpofsugar.co.jp/product/yorikure/character/character04.html|title=斑鳩 和羽|publisher=Lump of Sugar|accessdate=2018年9月16日|language=ja}}</ref>}}
: 清司的學姐，繼椛實學園三年級生。為了將來著想而在稜未家打工，但非常不擅長料理。實際上有著幻夷血統，當情緒高昂便會長出翅膀，因此平常都故作冷靜。
