'''私塾'''，也叫'''私學'''
、'''私塾'''，或'''學堂'''，是古[[中國|中國]]的私立學校或補習班，流行于受儒家影響的文化圈。私塾大多由讀書人、[[秀才|秀才]]等私人開辦，由教書者在自宅設立，入學者多係六歲至八歲孩童。一般百姓多為合資聘師，富家門第則是家教模式居多，私塾分為「[[蒙館|蒙館]]」、「經館」和兩者兼具三種，蒙館其教學宗旨主要是啟蒙識字，經館教學內容與科考有關，包括[[四書|四書]]，[[詩|詩]]、[[書|書]]、[[易|易]]、和[[左傳|左傳]]；年齡較大者，則讀[[古文|古文]]，習字，作詩文，修業年限視各人需要而定。

== 教學內容 ==
私塾的学习内容，从启蒙到逐步深化，在中國一般顺序是：《[[三字经|三字经]]》、《[[百家姓|百家姓]]》、《[[日用杂字|日用杂字]]》、《[[千字文|千字文]]》、《[[论语|论语]]》、《[[孟子|孟子]]》、《[[大学|大学]]》、《[[中庸|中庸]]》、《[[诗经|诗经]]》、《[[书经|书经]]》、《[[礼记|礼记]]》、《[[左氏春秋|左氏春秋]]》等，有的选读《[[女儿经|女儿经]]》、《[[孝经|孝经]]》、《[[幼学琼林|幼学琼林]]》、《[[千家诗|千家诗]]》、《[[古文观止|古文观止]]》等书，也有的加学《[[世事应酬|世事应酬]]》，即契约、媒柬、輓联之内容，以应实用。其他地區除了中國的經典外也包括當地[[儒學|儒學]]著作及詩文等。

== 祠堂 ==
{{main|祠堂}}
祠堂春秋舉行祭禮，而平常則作為私塾，教育家庭子弟。當中供了一個牌位─「大成至聖先師孔夫子神位」，同學站在兩邊，老師站在上面，新來的學生向孔子神位行[[叩首|叩首]]禮。每一次新學生來的時候，這個禮節舊生就看一遍，一生當中從內心裡面尊敬老師，對於道學看得比什麼都重。<ref name="淨土大經解演義第四十一集">{{Cite news|url=http://new.jingzong.org/Category_450/index.aspx?4728 |title=淨土大經解演義-第41集 |author= |date=2013-06-24 |work= |newspaper=archive.is |accessdate=2017-06-05 |deadurl=bot: unknown |archiveurl=https://archive.is/20130624134654/http://new.jingzong.org/Category_450/index.aspx?4728 |archivedate=2013-06-24 }}</ref>

== 現代私塾 ==
在現代東亞地區也有一些私人開辦，專門提供傳統文化課程如[[東亞書法|書法]]、[[茶儀式|茶儀式]]（[[中國茶藝|中國茶藝]]、[[日本茶道|日本茶道]]、[[韓國茶禮|韓國茶禮]]、[[台灣茶藝|台灣茶藝]]、[[越南茶藝|越南茶藝]]、[[琉球茶道|琉球茶道]]）、[[古典文學|古典文學]]欣賞寫作、傳統[[禮儀|禮儀]]等的[[學校|學校]]，也常稱為「私塾」。
