== 車站歷史 ==
[[File:Railways_at_Kaohsiung.svg|200px]]
; 站體部分
* 1907年10月1日：設置鳳山停車場，木造結構站房<ref>[https://www.facebook.com/FengShanRail.SOS/photos/a.663896560476021/663896537142690/?type=3&theater 鳳山停車場明信片]</ref>。
* 1935年：於原址新建磚造結構站房<ref>[http://newnrch.digital.ntu.edu.tw//archive/DOFiles/jpg/00/00/85/07/cca220001-hp-cdn0001964-0001-i.jpg 歡迎大專生入伍，鳳山車站熱烈情形]</ref>。
* 1964年：因應出入旅客增加，磚造結構站房擴建完工。
* 1990年12月：原址新建之二樓鋼筋水泥站房完工。
* 2014年6月22日：配合地下化工程，拆除舊站房，啟用於[[臺灣糖業鐵路|臺灣糖業鐵路]]鳳山站原址興建的[[鐵路臨時站房|鐵路臨時站房]]
* 2018年10月13日22時02分，發出最後一班從潮州開往永康的北上第3288次區間車，以及10月13日22時35分（實際上誤點3分，為22時38分開車）發出最後一班從七堵開往潮州的南下第141次自強號後 原平面月臺及鐵軌正式走入歷史。

; 其他
* 過去車站前有[[臺灣糖業鐵路|臺糖]][[南北平行預備線|小港線]]通過，臺糖鳳山站設置於省鐵車站東側<ref>[http://ticket.myweb.hinet.net/station/b20.htm 看橋工坊-車站巡禮]</ref>。由於是南北線重要的列車轉運站，所以偶見搭乘省鐵旅客因糖鐵調車而等候的情況。
* 1995年5月3日：[[高雄車站|高雄]]－鳳山間電車線通電<ref name=TraElec>[http://www.railway.gov.tw/tw/CP.aspx?sn=3634&n=6842 交通部臺灣鐵路管理局-臺鐵簡介-台灣鐵路電訊]</ref>。
* 1996年7月10日：鳳山－[[屏東車站|屏東]]間電車線通電<ref name=TraElec />。
* 2013年9月30日：啟用多卡通刷卡機。
* 2018年10月14日：配合[[高雄市區鐵路地下化計畫|高雄市區鐵路地下化計畫]]鳳山車站改建為地下化車站完工，部分公車路線延駛鳳山車站。

== 地下化工程相關紀事 ==
; 站體新建計畫
* 配合[[鳳山計畫|鳳山計畫]]之臨時站房於2014年6月22日啟用，車站內股道配置為三股道，設置一個[[岸式月台|岸壁式月台]]、一個[[島式月台|島式月台]]。
* 於2018年10月14日啟用新的地下車站，目前規劃為樓高12樓的車站，車站內股道配置為四股道、二個[[島式月台|島式月台]]<ref name=FengshanNewBuild>[http://www.rrb.gov.tw/upload/documents/projects/20150526_173207.57150.pdf 鳳山車站站體空間規劃簡報(2015/05/26)]</ref><ref name=FengshanNewTraffic>[http://www.rrb.gov.tw/upload/documents/projects/20150527_181437.70555.pdf 鳳山車站交通動線規劃簡報(2015/05/27)]</ref>。

; 鳳山車站設計意象徵圖徵選
因為[[交通部鐵路改建工程局|鐵工局]]規劃的車站無法讓附近居民滿意，因此開放舉辦徵圖的活動<ref>[http://www.ceci.com.tw/page/news/NewsView.aspx?id=0000002299 鳳山車站設計意象徵圖徵選]</ref>。
* 2014年2月：建立活動網站，公告相關文件
* 2014年3月：開放報名
* 2014年4月25日：開放民眾票選
* 2014年5月21日：公告徵選結果，第一名從缺
