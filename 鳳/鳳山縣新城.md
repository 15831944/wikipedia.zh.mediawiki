[[File:Fsha.jpg|right]]
鳳山縣城自[[林爽文事件|林爽文事件]]後，於乾隆五十三年（1788年）遷至埤頭街（今[[高雄市|高雄市]][[鳳山區|鳳山區]]），為別位於[[左營區|左營]]之[[鳳山縣舊城|鳳山縣舊城]]，遂稱其為'''鳳山縣新城'''。鳳山縣新城原有城門六座、砲台六座、土牆及莿竹二重城牆、護城河。經歷歲月摧殘，只餘城門一座、砲台三座，現均列為鳳山縣城殘蹟，為市定古蹟<ref>[https://nchdb.boch.gov.tw/assets/advanceSearch/monument/19880226000004 鳳山縣城殘蹟].文化部文化資產局</ref>。

== 特色 ==
[[File:FengshanEgate.jpg|thumb]]
*縣級城池本當只有東、西、南、北四門，但鳳山縣新城增建外北門、小東門（或稱東便門），共有六座城門
*鳳山縣新城的砲台為獨立式砲台，有別於[[鳳山縣舊城|鳳山縣舊城]]、[[台北府城|台北府城]]的附屬式砲台
*縣城形狀為靴子（或稱為心形、元寶形），也為台灣中式城池中所罕見。
[[File:Fsnc.jpg|thumb]]
== 歷史 ==
*乾隆五十三年（1788年）遷治於埤頭街，植莿竹為城
*嘉慶九年（1804年）知縣吳兆麟倡建六座城門
*道光十八年（1838年）知縣[[曹謹|曹謹]]增建城樓、興建六座砲台、興修[[曹公圳|曹公圳]]
*咸豐四年（1854年）參將曾元福修築土牆
*光緒十八年（1892年）土城傾圮，知縣李淦重修

== 城門 ==
鳳山縣新城共有城門六座，之所以多出兩座城門，有研究指出是因應外北門街及東門溪（今[[鳳山溪|鳳山溪]]）畔之碼頭而增建；但也有為安全考量之說法。此六座城門均位於交通要道，易因現代都市計劃破壞而消失。

===外北門===
－'''郡南第一關'''
*位置：鳳山區中正路與協和路口北側
*重要性：往外可通往[[台南府城|台南府城]]及[[左營舊城|左營舊城]]，往內則有商業活動興盛的外北門街
*命名由來：清代鳳山縣治附近一帶為府城以南最為繁榮之處，故有此名
*保存現況：已於日治時期拆除，橫額題有「郡南第一關」，落款於道光十七年（1837年），現存於高雄市立歷史博物館
[[File:ngofs.jpg|left]]
[[File:egofs.jpg|left]]

===北門===
－'''平朔門'''
*位置：鳳山區中正路與光復路口南側
*重要性：位於繁榮的外北門街與和安街之間的重要關口
*命名由來：「朔」為一個月的開始，按字面「平朔」應指美好、安寧的開始
*保存現況：已於日治時期拆除

===大東門===
－'''朝陽門'''
*位置：鳳山區中山路與三民路口東側
*重要性：為仁和街通往東門溪碼頭之關口，早年東門溪可行船之時，可通往[[打狗港|打狗港]]
*命名由來：多處東門之命名通常與「日昇」有關，如[[竹塹城|竹塹城]]之[[迎曦門|迎曦門]]
*保存現況：已於日治時期拆除，內面橫額題有「東門」，落款於道光十八年（1838年），現存於[[曹公廟|曹公廟]]外之碑林

===小東門（東便門）===
－'''同儀門'''
*位置：鳳山區三民路44巷內
*重要性：城內即為打鐵街，城外有東福橋，旁有[[東福祠|東福祠]]，過橋即為過溝仔街；再往東可通[[阿猴城|阿猴城]]（今[[屏東市|屏東市]]）
*保存現況：為鳳山縣新城唯一留存的城門，橫額落款於道光十九年（1839年），北側仍留有土牆殘跡；原始形式不明，今日所見樣貌乃於民國九十五年（2006年）參考大東門舊照所修復

===西門===
－'''景華門'''
*位置：位於[[捷運鳳山站|捷運鳳山站]]站體上，鳳山區光遠路與中華路口
*重要性：由陸路通往[[打狗|打狗]]之要道
*命名由來：西門之命名與「夕陽」有關
*保存現況：已於日治時期拆除

===南門===
－'''安化門'''
*位置：鳳山區五甲一路與立信街口東側之南門公園
*重要性：可通往[[前鎮區|前鎮]]、[[草衙|草衙]]，城南有鎮南宮，為一重要信仰中心
*命名由來：多處南門之命名常與「教化」有關，如[[彰化縣城|彰化縣城]]之「宣平門」
*保存現況：已於日治時期拆除

== 砲台 ==
鳳山縣新城原有六座砲台，其形式各異，該是順應地形、水文而建；砲台獨立於城牆，僅相連但不相通；位置大都在城牆轉角之處。

*'''平成砲台'''
位在[[曹公廟|曹公廟]]後方，呈長方形，保存最為完整，橫額落款於道光戊戌年（1838年），於民國88年（1999年）整修完成

*'''澄瀾砲台'''
位在鳳山區復興街與立志街口，呈不規則八角形，橫額落款於道光拾柒年（1837年），兩側留有小段土牆；一度遭住戶占用，於民國88年（1999年）整修完成

*'''訓風砲台'''
位在鳳山區中山東路五巷之內，呈長弧形，一度只餘殘壁，於民國88年（1999年）整修完成

*'''迎恩砲台'''
已拆除，橫額落款於道光十八年（1838年），現存於[[曹公廟|曹公廟]]外之碑林

*'''東南砲台'''
已拆除，因無橫額出土，暫定名為東南砲台

*'''東北砲台'''
已拆除，因無橫額出土，暫定名為東北砲台

== 縣城內其他設施 ==
[[File:Cghofs.jpg|right]]

*'''鳳山縣署'''
鳳山縣署在日治初期，曾作[[鳳山廳|鳳山廳]]署使用。[[高雄州|高雄州]]成立後(1920年)，縣城乃屬其下轄之[[鳳山郡|鳳山郡]][[鳳山街|鳳山街]]，鳳山縣署一部分挪作學校用地(高雄州鳳山小學校，今曹公國小)，另一部分則改建為鳳山郡役所。[[臺灣戰後時期|中華民國接管臺灣後]](1945年)，廢[[高雄州|高雄州]]，縣城改隸[[高雄縣|高雄縣]]鳳山鎮，鳳山郡役所改建為高雄縣政府警察局。2010年底，高雄縣市合併，高雄縣政府警察局外觀稍作整建，改為高雄市政府警察局刑事警察大隊、保安警察大隊大樓

*'''鳳儀書院'''
[[鳳山縣舊城|舊城]]屏山書院毀於戰火之後，為提升地方文風，歲貢生張廷欽於嘉慶十九年（1814年）統籌興建[[鳳儀書院|鳳儀書院]]，道光三年（[[1823|1823]]）落成。為全台現存規模最大的書院，建築工藝頗有可觀之處，列為三級古蹟，2014年11月已修復開放。

*'''龍山寺'''
[[鳳山龍山寺|鳳山龍山寺]]為台灣五座龍山寺中，位置最南邊的一座；今列為二級古蹟，建築工藝精巧，仍保有清代格局

*'''雙慈亭'''
當屬鳳山縣新城的寺廟群中，創建年代最早的一座，惜經多次改建已失原貌，故不列入古蹟；廟中保存許多珍貴文物，亦為當地的信仰中心

*'''鳳邑城隍廟'''
創建於嘉慶五年（1800年），因於日治時期移作他用而破敗，民國五十六年（1967年）重建，為當地的信仰中心

*'''曹公廟'''
位於曹公路、曹公國小正對面，原稱曹公祠；清代曾於鳳儀書院祭祀曹謹祿位，日治時期於現址建祠

*'''玉皇宮'''（天公廟）
