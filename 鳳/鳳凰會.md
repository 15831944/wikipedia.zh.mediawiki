'''鳳凰會'''（''' Order of the Phoenix'''），是奇幻小說《[[哈利·波特|哈利·波特]]》中，[[阿不思·鄧不利多|鄧不利多]]於黑魔王第一次崛起時所創建的組織，用來對付[[佛地魔|佛地魔]]和其黨羽[[食死人|食死人]]。在黑魔王重生後不久，阿不思·鄧不利多立即重組此會。這個組織首次出現在《哈利波特-鳳凰會的密令》裡。

== 總部 ==
* 古里某街12號（布萊克家族祖宅）（第5、6集）
* 洞穴屋（第7集）

== 領袖 ==
* [[阿不思·鄧不利多|阿不思·鄧不利多]] （1970年 - 1981年, 1995年 - 1997年6月30日）
* [[瘋眼穆敵|瘋眼穆敵]] （1997年7月1日 - 24日）
* [[金利·俠鉤帽|金利·俠鉤帽]] （1997年7月24日 - 1998年5月2日）

== 敵人 ==
* [[佛地魔|佛地魔]]及[[食死人|食死人]]
* 催狂魔
* 巨人（由新頭目勾勾瑪帶領的巨人勢力）
* 禁忌森林的一群蜘蛛精
* -{zh-hans:死拿錢; zh-hant:死拿錢;}- （一群以捕捉純種叛徒、麻瓜後代和逃亡學生來賺錢的惡棍）
* [[魔法部|魔法部]]（魔法部長康尼留斯·夫子拒絕承認佛地魔重生期間，和其接班人盧夫·昆爵死後至佛地魔被打敗）

== 同盟 ==
* [[魔法部|魔法部]] （魔法部長盧夫·昆爵試圖爭取合作）
* [[鄧不利多的軍隊|鄧不利多的軍隊]]
* [[霍格華茲|霍格華茲]]的師生
* 活米村居民 （第二次[[霍格華茲|霍格華茲]]之戰時，活米村居民亦加入戰鬥）
* 巨人呱啦 （[[海格|海格]]的弟弟）
* 鷹馬巴嘴及大群騎士墜鬼馬 （牠們曾幫助攻擊佛地魔手下的巨人）
* 霍格華茲廚房的家庭小精靈 （幫助攻擊食死人）
* 人馬 （射傷食死人）
