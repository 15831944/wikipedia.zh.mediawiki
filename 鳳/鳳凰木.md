'''鳳凰木'''（[[学名|学名]]：''{{lang|la|Delonix regia}}''）又名'''影樹'''、'''凤凰树'''、'''凤凰花'''<ref name="中国高等植物数据库全库">{{cite web| url =http://www.bioinfo.cn/db05/KmzwSpecies.php?action=view&id=3125| title =凤皇木| work =《中国高等植物数据库全库》| publisher =中国科学院微生物研究所| author =昆明植物研究所| accessdate =2009-02-23| deadurl =yes| archiveurl =https://web.archive.org/web/20160305003753/http://www.bioinfo.cn/db05/KmzwSpecies.php?action=view&id=3125| archivedate =2016-03-05}}</ref>、'''金鳳'''、'''火鳳凰'''、'''洋楹'''、'''紅花楹'''（广州）<ref name="中国高等植物数据库全库" />、'''火樹'''、'''紅火楹'''、'''火焰木'''、'''森之炎'''等，为[[豆科|豆科]][[凤凰木属|凤凰木属]]的植物。

它以華麗的花朵與[[蕨類|蕨類]]植物般的綠葉出色。它生長在世界上許多熱帶亞熱帶地方作為裝飾和遮蔽太陽的樹木。

== 形态 ==

[[File:Coral_Gables_Royal_Poinciana.jpg|thumb]]
[[File:Delonix_regia_(Taiwan).jpg|thumb]]
[[File:Gulmohar_leaves_closeup.JPG|thumb]]
鳳凰木因鮮紅或橙色的花朵配合鮮綠色的羽狀複葉，被譽為世上最色彩鮮艷的樹木之一。由於樹冠橫展而下垂，濃密闊大而招風，在[[熱帶|熱帶]]地區擔任遮蔭樹的角色。
[[File:鳳凰木種子.JPG|thumb]]
[[File:Royal_Poinciana_Tree_Seeds.JPG|thumb]]
[[File:Delonix_regia_Xiamen.JPG|thumb]]的一株凤凰木]]
鳳凰木植株高大，可達20公尺以上。性喜高溫、多日的環境，必需在陽光充足處方能繁茂生長。花大呈紅色，連[[花萼|花萼]]內側都呈血紅色，四瓣伸展約8[[公分|公分]]長，第五瓣直立稍大及有黃及白的斑點，[[雄蕊|雄蕊]]紅色，花萼腹面深紅色，背面綠色<ref>{{cite web |url=http://www.weekendhk.com/lifestyle/%E9%B3%B3%E5%87%B0%E6%9C%A8-%E9%A6%99%E6%B8%AF%E8%B3%9E%E8%8A%B1-%E5%A4%A7%E5%9F%94%E8%B3%9E%E8%8A%B1/2/ | title=鳳凰木 盛開! 紅遍18區 大埔最密集 |publisher=新假期 |date=2016年5月27日}}</ref>。[[花序|花序]]為總狀花序，呈繖房狀排列，花萼和[[花瓣|花瓣]]皆5片，聚生成簇。開花後結出一條條長形豆莢果，長可達60公分，成熟後呈深褐色，木質化，內藏40-50粒細小的[[種子|種子]]，每顆平均只有0.4[[克|克]]重，種皮有斑紋，有毒，不可誤食。葉片是二回羽狀[[複葉|複葉]]，呈羽片[[對生|對生]]，15-30對，每羽片有小葉20-40對。小葉為長橢圓形，基部歪斜，[[葉緣|葉緣]]是全緣，葉端圓鈍。[[葉片|葉片]]是薄紙質，葉面平滑且薄，為青綠色，[[葉脈|葉脈]]則僅中肋明顯。樹形為廣闊傘形，冬天落葉時，多不勝數的小葉如雪花飄落。和許多豆科植物一樣，鳳凰木的根部也有[[根瘤菌|根瘤菌]]。為了適應多雨的氣候，樹幹基部長有[[板根|板根]]出現。鳳凰木適應熱帶氣候，耐旱及可在有鹽分的環境生長。

未曾見過此樹的人，常與[[蝴蝶木|蝴蝶木]]互相混淆，由於此二種花期一樣(５～７月)<ref>{{cite web|url=https://hk.news.appledaily.com/local/daily/article/20160523/19623721 |title=
提早天熱 鳳凰木一片紅 |publisher=蘋果日報 |date=2016年5月23日}}</ref>，同樣二回羽狀[[複葉|複葉]]，呈羽片[[對生|對生]]、開花後結出一條條長形豆莢果，長可達60公分，成熟後呈深褐色，木質化，內藏40-50粒細小的[[種子|種子]]，彼此不同之處只有在花朵大小。

== 分布 ==
原產地為马达加斯加，世界各地熱帶及亞熱帶地區都有栽種。廣泛種植於[[中國|中國]]南部及西南部、在[[美國|美國]]鳳凰木只生長在[[佛罗里达州|佛罗里达州]]、[[德克萨斯州|德克萨斯州]]南部的瑞欧格兰山谷（Rio Grande Valley）、[[亞利桑那州|亞利桑那州]]及[[加利福尼亚州|加利福尼亚州]]的沙漠地區、[[夏威夷州|夏威夷州]]、[[波多黎各|波多黎各]]、[[美屬維爾京群島|美屬維爾京群島]]和[[關島|關島]]。鳳凰木亦廣泛生長在[[加勒比海国家|加勒比海地區]]。

原生[[非洲|非洲]][[马达加斯加|马达加斯加]]，後來引入中國華南（[[海南|海南]]、[[云南|云南]]、[[广东|广东]]、[[广西|广西]]、[[福建|福建]]等省）<ref name="中国高等植物数据库全库" />。臺灣的鳳凰木則在1897年[[台灣日治時期|日治時期]]時由日本[[孟買|孟買]]領事館採購種子引進，最先種植在臺南<ref>{{cite web |author1=臺南市政府農業局 |title=愛樹一生一世 |url=http://oldtree.tainan.gov.tw/tree.asp?treeid=%7BE94E16E5-CB78-4E37-BA4B-0C8CA57982DB%7D}}</ref>；野外屬[[瀕危物種|瀕危物種]]<ref>[http://www.iucnredlist.org/search/details.php/32947/summ IUCN：Delonix regia]</ref>，目前已由人工引种栽培<ref name="中国高等植物数据库全库" />，被廣泛栽種為觀賞樹。在[[香港|香港]]為常見的外來品種落葉[[喬木|喬木]]。

鳳凰木在很多現時生長地區為視為移植品種，在[[澳大利亞|澳大利亞]]更被當作侵入品種，部分原因是其闊大的樹冠及濃密的樹根阻礙其他品種在其下生長。在[[印度|印度]]鳳凰木被稱為高莫哈樹（Gulmohar）。鳳凰木的豆莢在加勒比海地區被用作敲打樂器，稱為沙沙（shak-shak）或沙球。

在[[马来西亚|馬來西亞]]，還可看見黃花的[[盾柱木|盾柱木]]（{{lang|la|''Peltophorum pterocarpum''}}）与红花的凤凰木一起花朵盛开，煞是好看。

== 文化 ==
* 在[[臺灣|臺灣]]，於1897年引入，鳳凰木因在6月大量開花，常與[[蟬|蟬]]鳴並列為[[畢業|畢業]]的象徵物。歌手[[林志炫|林志炫]]曾演唱一首名為鳳凰花開的國語歌。
* 鳳凰木為[[臺灣|臺灣]][[臺南市|臺南市]]市樹，以與「凰鳳城」之別名相稱；[[國立成功大學|國立成功大學]]以鳳凰為其校花。
* 鳳凰木是[[中国大陸|中国大陸]][[福建省|福建省]][[厦门市|厦门市]]市樹<ref>[http://www.xm315.com/xm/about08.jsp 厦门市树] {{webarchive|url=https://web.archive.org/web/20061230152024/http://www.xm315.com/xm/about08.jsp |date=2006-12-30 }}</ref>。[[厦门大学|厦门大学]]以鳳凰為其校花。
* 鳳凰木在[[汕頭|汕頭]]也稱為“金鳳花”，亦同時是汕頭市的市花，[[汕頭大學|汕頭大學]]以鳳凰為其校花。
* 鳳凰树曾经是[[四川省|四川省]][[攀枝花市|攀枝花市]]的市树，但因凤凰树在1990年代中后期连续多年爆发严重的[[尺蠖|尺蠖]]虫灾而被大量砍伐，其市树的位置后被[[木棉|攀枝花树]]取代。
* 鳳凰木是[[深圳中学|深圳中學]]的校花。
