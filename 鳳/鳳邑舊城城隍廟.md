'''鳳邑舊城城隍廟'''，又稱'''舊城城隍廟'''、俗稱'''左營城隍廟'''，在[[高雄市|高雄市]]名勝[[蓮池潭|蓮池潭]]附近，[[高雄市|高雄市]][[左營區|左營區]]店仔頂路一號。鳳邑舊城城隍廟主奉'''鳳山縣[[城隍|城隍]]勅封顯佑伯'''，配祀有[[註生娘娘|註生娘娘]]、[[福德正神|福德正神]]、[[觀音佛祖|觀音佛祖]]、[[虎爺|虎爺]]將軍、[[謝范將軍|謝范將軍]]、[[日遊神|日遊神]]、[[夜遊神|夜遊神]]、文[[判官|判官]]、武[[判官|判官]]、功德爺、報司爺、十八司爺、[[太歲星君|太歲星君]]。每年的[[農曆|農曆]]五月十二日，為主神城隍爺的聖誕日。

==簡介==
[[清聖祖|清聖祖]][[康熙|康熙]]四十三年（1704年）[[鳳山縣_(台灣)|鳳山縣]][[衙門|衙門]]始建，遂設草寮祭祀'''鳳山縣城隍'''。[[康熙|康熙]]五十七年（1718年）鳳山[[知縣|知縣]][[李丕煜|李丕煜]]有感於「獨城隍缺焉未舉」（鳳山縣志.新建城隍廟記），乃捐俸祿，興建鳳山縣城隍廟，位於興隆庄北郊（今[[高雄市|高雄市]][[左營區|左營區]]埤仔頭）。

[[清高宗|清高宗]][[乾隆|乾隆]]五十一年（1787年），[[天地會|天地會]][[林爽文|林爽文]][[起事|起事]]反清，鳳山縣治受[[林爽文事件|戰火]]之擾，殘破不堪，縣治遷至新城（今[[高雄市|高雄市]][[鳳山區|鳳山區]]），因縣治遷移，鳳山縣城隍廟一度稱為[[舊城城隍廟|興隆城隍廟]]；[[清仁宗|清仁宗]][[嘉慶|嘉慶]]五年（1800年），[[知縣|知縣]]吳兆麟於新城（今鳳山區）鳳儀書院旁另建[[鳳山縣城隍廟|新城隍廟]]，而鳳山縣城隍廟因位於[[鳳山縣舊城|鳳山縣舊城]]，遂更名為'''鳳邑舊城城隍廟'''。

鳳邑舊城城隍廟建造完成後，儒學吳策勳於[[清|清]][[乾隆|乾隆]]五十八年（1793年）重修，職員陳大奎於[[清文宗|清文宗]][[咸豐|咸豐]]六年（1856年）重修。[[台灣日治時期|台灣日治時期]]結束後，[[內惟|內惟]]李天輝、[[左營區|左營]]謝耀麟等又於1946年重修。最近一次大整修於1962年，由地方善信[[謝耀霖|謝耀霖]]等人捐資，終於在1969年竣工。

== 歷史定位 ==
鳳邑舊城城隍廟，所奉祀之城隍尊神為鳳山縣城隍，勅封顯佑伯， 奉祀年代始於1704年<ref>高雄市立左營高級中學編，半屏山下話左營，2006年</ref>，為台灣歷史上'''第二悠久'''之「官封」城隍信仰，僅次於[[台灣府城隍廟|台灣府城隍廟]]。但直到1718年才於左營舊城外建廟。

== 廟中對聯 ==
[[清朝|清朝]]廍後庄[[舉人|舉人]]卓肇昌所寫{{cquote|為善必昌，為善不昌，祖宗必有餘殃，殃盡必昌；作惡必亡，作惡不亡，祖宗必有餘德，德盡必亡。}}

== 自問心大算盤 ==
位於城隍廟正殿上方，為總數147顆算珠的自問心大算盤，是鳳邑舊城城隍廟之特色。普遍城隍廟宇中，皆供有一副算盤，命名為「自問心」，有開示來廟者所做之事捫心自問之意。現今，每當城隍出巡時，廟方也將自問心算盤，隨出巡繞境隊伍出巡，以展城隍信仰之特色。

== 2012年遭縱火事件 ==
2012年3月14日，23:35左右，遭一名男子縱火，廟門火燒焦黑。事後這名縱火犯遭到警方逮捕。嫌犯為住在左營區孔營路的李姓男子，具有過失傷害前科，被捕後表示是因為不滿廟中鬼神向他索命，所以縱火，警方依公共危險罪移送法辦。<ref>[http://www.libertytimes.com.tw/2012/new/mar/14/today-so2.htm 不爽「鬼神索命」 半夜火攻城隍廟]</ref> <ref>[http://www.ttv.com.tw/101/03/1010313/10103134934403L.htm 縱火燒城隍廟 辯受不了鬼神索命]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> <ref>[http://www.youtube.com/watch?=1&v=4_qItc_Ml5Y 報復神明! 男子半夜火燒城隍廟] </ref>遭到毀壞的城隍廟廟門為著名廟宇畫師[[潘麗水|潘麗水]]之作品，為左營重要之文化瑰寶。<ref>[http://www.youtube.com/watch?v=OmyXSiYHW0o 好大膽! 醉漢火燒城隍廟] </ref>
