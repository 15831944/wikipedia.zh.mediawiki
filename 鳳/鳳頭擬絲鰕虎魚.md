'''凤头拟丝鰕虎鱼'''（[[学名|学名]]：'''''Cryptocentroides gobioides'''''），为[[輻鰭魚綱|辐鳍鱼纲]][[鱸形目|鲈形目]][[鰕虎亞目|鰕虎亚目]][[鰕虎科|鰕虎科]]拟丝鰕虎鱼属下的一个种，英文名为Crested Oystergoby。<ref>{{Cite web|title=Cryptocentroides gobioides|url=http://fishesofaustralia.net.au/home/species/94#moreinfo|work=fishesofaustralia.net.au|accessdate=2019-02-20|language=en}}</ref>

== 分布 ==
本鱼为[[亚热带|亚热带]]海水鱼，分布于西南[[太平洋|太平洋]][[澳洲|澳洲]]东海岸[[昆士蘭州|昆士兰州]]北部至[[新南威爾士州|新南威尔士州]]远南地区的海域。<ref>{{Cite web|title=Cryptocentroides gobioides summary page|url=https://www.fishbase.us/summary/Cryptocentroides-gobioides.html|work=FishBase|accessdate=2019-02-20|language=en}}</ref>

== 特征 ==
本鱼身体较为细长，体色以橄榄色为主，其头部长有蓝色斑点，身体侧面也长有许多鲜艳的蓝色和橙色条纹，背鳍上有长两排深粉红色条纹，雄鱼的体色相对于雌鱼较深。<ref>{{Cite web|title=Cryptocentroides gobioides|url=http://fishesofaustralia.net.au/home/species/94#moreinfo|work=fishesofaustralia.net.au|accessdate=2019-02-20|language=en}}</ref>

== 生态 ==
本鱼栖息主要栖息于近海及[[河口|河口]]沙泥底质水域，能适应盐度的剧烈变化，喜欢躲藏在在牡蛎壳和石缝下面，可以长时间离水，因此在潮间带的石头下也能见其踪影，和[[槍蝦總科|枪虾]]共生有共生关系。

== 经济利用 ==
由于体型鲜艳，此鱼可作为观赏鱼。
