'''鳳山紅毛港保安堂'''位於[[臺灣|臺灣]][[高雄市|高雄市]][[鳳山區|鳳山區]]，原位於[[小港區|小港區]][[紅毛港_(高雄市)|紅毛港]]姓蘇仔聚落{{r|故事}}，為紅毛港聚落三間陰廟之一{{NoteTag|另外兩間是海眾廟與西南城{{r|陳}}。}}{{r|陳}}。該廟的一大特色是有供奉[[日本|日本]][[軍艦|軍艦]]，每年會有來自日本的訪問團參拜、交流{{r|方}}{{r|林}}。主神為「海府」大元帥（為[[太平洋戰爭|太平洋戰爭]]中殉難的第38號日軍軍艦[[中尉|中尉]][[艦長|艦長]]）、「郭府」千歲、「宗府」元帥{{r|方}}。廟宇風格結合[[琉球|琉球]]及臺灣本土風格。

而由於廟中供奉日本人神明與好兄弟，所以該廟與其他廟宇不同不放梵音，而是「日本海軍進行曲」，並使用日本清酒來祭拜{{r|紅}}。除此之外的習俗，則大多與其他廟宇相同{{r|紅}}。

== 沿革 ==
保安堂的由來，據說是在日本大正十二年（[[1923年|1923年]]）時，紅毛港漁民在海上作業時撈到腿骨，漁民們將之帶回岸上並建竹寮供奉{{r|故事}}{{r|紅}}。該竹寮即是保安堂前身，當時所供奉的即是「郭府」（又稱郭聖公、郭府元帥）{{r|故事}}{{r|紅}}。而宗府元帥的來歷則是當地一位陳姓村民，因於死後無家人處理後事，遂託夢鄰人，之後村民將之安葬於保安堂{{r|故事}}{{r|朱}}。另外《戀戀紅毛港──寺廟建築與信仰》一書則有另一版本，寫說在大正十二年（1923年）時蘇姓人家為祭拜「地基主」而搭建竹寮奉祀「宗府」，並命名為「保安宮」，為保安堂前身{{r|朱}}。1940年代有紅毛港漁民阿忠撈到腿骨，乃將之洗淨送到保安宮，之後阿忠出海都有好收穫，認為是被那根腿骨的主人保佑，遂捐錢並募款將竹寮加以修建，並奉那腿骨的主人為「郭府」{{r|朱}}。

[[二次大戰|二次大戰]]後，於民國35年（1946年）左右，紅毛港漁民在海上撈獲一顆頭顱，亦帶回保安堂供奉，是為「海府」{{r|紅}}{{r|朱}}{{r|故事}}{{r|陳}}。之後先有託夢，後有原先不會[[日語|日語]]的乩童用日語交代海府的來歷，是太平洋戰爭中陣亡的日本第38號軍艦艦長{{r|紅}}{{r|朱}}{{r|故事}}{{r|陳}}。後來據說海府大元帥曾下指示，欲前往琉球護國神社參拜「日本海軍戰歿者慰靈塔」，信徒乃在民國79年（1990年）8月造訪琉球{{r|紅}}{{r|朱}}{{r|陳}}。並在此行中，於紀念碑上發現記載日本海軍震洋隊第卅八號軍艦在太平洋戰爭中被擊沉一事{{NoteTag|根據前川正名的文章，指出廟方所拍的碑文照片應該是指[[震洋|第三十八震洋隊]]而非指艦艇{{r|前川}}。}}{{r|紅}}{{r|朱}}{{r|陳}}。日後信徒們在民國八十年（1991年）請[[哈瑪星|哈瑪星]]船匠黃秀世，打造縮小版的軍艦以祭祀{{r|紅}}{{r|朱}}{{r|陳}}。據說製作途中，海府屢屢託夢，要求上面的士兵器具等物都必須確實復刻，與「卅八號軍艦」一致{{r|陳}}。

而保安堂自大正十二年（1923年）草創後，曾有村民洪送還願重修，將原本供奉於廟內的骨骸遷葬於廟後新建的墓園{{r|故事}}。後來民國64年農曆十月十五（1974年11月28日）蘇現發起重修{{NoteTag|據說當時老漁夫蘇現在海上捕魚小憩時，海府託夢表示想改建廟宇，並指定沙灘上某處龜穴為廟址{{r|故事}}。之後照指示探查，果然於該處挖出海龜蛋，遂發起興建{{r|故事}}。}}，當時有陳新登、陳新教捐地，洪豬哥為主辦人，洪明恩為副辦人{{r|紅}}。民國76年（1987年）時，又興建了牌樓{{r|紅}}。

[[紅毛港_(高雄市)|紅毛港]]遷村至鳳山後，當地居民將保安堂遷至現址，並於[[民國|民國]]102年（2013年）12月29日進行謝土安座大典{{r|方}}{{r|林}}。

== 傳說 ==
據說海府、郭府、宗府三尊神明的神像，是用同一塊樟木所雕成{{r|紅}}{{r|故事}}{{r|朱}}。而這塊樟木的來歷，相傳是過去紅毛港漁民在捕魚時所撈到的{{r|紅}}{{r|故事}}{{r|朱}}。當時漁民們原以為是撈到了大魚，結果卻是塊木頭，遂將之扔回海中再重新下網{{r|紅}}{{r|故事}}{{r|朱}}。然而之後又撈到同樣的木頭，於是漁民們又再次將之丟棄{{r|紅}}{{r|故事}}{{r|朱}}。最後在「三棄三拾」之後，漁民們認為這塊木頭有靈性，遂將之帶回岸上暫置{{r|紅}}{{r|故事}}{{r|朱}}。日後保安堂要雕神像時，找上了這塊木頭，而巧合的是木材的大小剛好就是雕三尊神像所需的大小{{r|紅}}{{r|故事}}{{r|朱}}。

== 其他 ==
《打狗漫騎──高雄港史單車踏查》一書提到，有人根據相關資料推測，海府大元帥可能不是來自「震洋第卅八號軍艦」，而是1944年11月從[[西貢|西貢]]前往菲律賓[[馬尼拉|馬尼拉]]途中被擊沉的「{{link-ja|丁型海防艦|丁型海防艦|第卅八號海防艦}}」{{r|陳}}。另外也有看法認為該軍艦應該是後來改制成「第38號巡邏艇」（{{lang-ja|第38号哨戒艇}}）的原[[枞级驱逐舰|枞级驱逐舰]]{{link-ja|蓬号驱逐舰|蓬 (駆逐艦)|蓬号}}<ref>{{Cite web|url=http://www.sankei.com/world/news/140110/wor1401100025-n1.html|title=台湾・高雄に日本の「軍艦」祭る新堂完成　日本統治時代の軍港、今も追悼供養|accessdate=2018-03-18|author=吉村剛史|date=2014-01-10|publisher=産経ニュース}}</ref>{{r|前川}}。

==戊戌年護漁佑民高雄港內海上遶境法會==
[[2018年|2018年]]適逢3年1次'''海府大元帥'''返鄉參拜年，但廟方擲筊都無回應，經請示海府希望將其頭骨遺骸[[火化|火化]]，並在[[高雄港|高雄港]]內舉辦法會，將船上145名同袍英靈，引回保安堂奉祀，日後期盼能返回日本，魂歸故鄉。
