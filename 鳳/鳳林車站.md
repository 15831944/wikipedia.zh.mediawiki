'''鳳林車站'''位於[[台灣|台灣]][[花蓮縣|花蓮縣]][[鳳林鎮_(臺灣)|鳳林鎮]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
鳳林是一設置有[[島式月台|島式月台]]、[[側式月台|側式月台]]各一座的地面車站，站房與島式月台間是以[[行人地下道|行人地下道]]連結。

== 營運狀況 ==
鳳林是台東線沿線的主要車站之一，根據台鐵的分級屬[[臺灣鐵路管理局車站等級#三等站|三等站]]等級，包含[[普悠瑪列車|普悠瑪號]]、[[自強號列車|自強號]]、[[莒光號列車|莒光號]]等對號車次與一般的[[臺鐵區間車|區間車]]皆有停靠，但並沒有[[觀光列車|觀光列車]]，[[團體列車|團體列車]]停靠。以台東線而言設施較為完備的鳳林車站站內除了設有一台[[自動售票機|自動售票機]]與三具投幣/卡式兩用的[[公共電話|公共電話]]，也有販售月台[[便當|便當]]。2016年時配合沿線票務設施的升級，開始啟用[[一卡通_(台灣)|一卡通]]、[[悠遊卡|悠遊卡]]及[[icash|icash 2.0]]等[[電子票卡|電子票卡]]付費的服務。另外，站內也設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。

== 歷史 ==
*1910年3月：設立停車場，以[[臺車|臺車]]方式先營運。
*1912年1月25日：設站。
*1956年6月1日:鐵路局第四警務段鳳林派出所成立
*1980年：臺東線拓寬原址重建今日站體-與台東關山站同型，日治時木造站體未能保存。
*2003年:配合行政院人力精簡，鳳林派出所裁撤。
*2004年:11月完成花蓮電務段交換式直流供電設備（ SMR）工程(設備容量：臺東站投落站 1000A；和平、花蓮、鳳林、玉里及池上等 5投落站各為 700A
*2012年：台東線電氣化，改為[[菸樓|菸樓]]式站體。
*2014年12月3日：車站改建工程完成。
*2016年6月28日：啟用多卡通刷卡機。
*2017年9月16日:接管鄰近[[南平車站_(臺灣)|南平車站]]任務。
