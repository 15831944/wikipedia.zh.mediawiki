'''耿世民'''（{{bd|1929年|11月28日|2012年|12月17日|耿世民}}），[[江苏|江苏]][[徐州市|徐州市]][[铜山区|铜山县]]人，[[中国|中国]][[古突厥语|古突厥语]][[文学家|文学家]]。[[中央民族大学|中央民族大学]][[哈萨克语|哈萨克语]]言文学系、[[维吾尔语|维吾尔语]]言文学系教授<ref name="中国">{{cite book |author1=中国社会科学院民族学与人类学研究所 |title=中国民族研究年鉴：2010-2012中国社会科学年鉴 |date=2014/07/01 |publisher=中国社会科学出版社 |isbn=9787516144213}}</ref>。

== 生平 ==
儿时就读于徐州少华街小学，后转入私立光启小学，抗战胜利后，从国立二十一中回到了徐州昕昕中学<ref>{{cite book |author1=耿世民 |editor=李飞 |title=维吾尔古代文献研究 |date=2003年12月 |publisher=中央民族大学出版社 |isbn=7-81056-753-5 |pages=1-2 |edition=第1版}}</ref>，1949年考入[[北京大学|北京大学]][[东语系|东语系]]，就读于维吾尔语科。1952年全国高校院系调整，转入中央民族学院，1953年毕业留校任教。在校期间，对[[俄语|俄语]]、[[德语|德语]]、[[日语|日语]]、[[阿塞拜疆语|阿塞拜疆语]]和[[土耳其语|土耳其语]]进行了研习。并且在学校政治活动如反霸、[[减租|减租]]和[[土改|土改]]等时前往哈萨克人居住区，学习[[哈萨克语|哈萨克语]]<ref>{{cite journal |author1=钟慧笑 |title=宁静致远 淡泊自甘——记新中国第一代民族语文学家、“国际知名学者奖”、“PIAC”金奖获得者耿世民教授 |journal=中国民族教育 |date=2005 |issue=06 |pages=42-45}}</ref>。1953年他创办我国高校的哈萨克语专业，开办了第一个哈萨克语本科专业班。1956年，判断出阿勒泰地区[[布尔津县|布尔津县]]的一小部分蒙古人所操语言是一种属于古代突厥语的[[图瓦语|图瓦语]]。1976年，主持开办了我国第一个古代突厥—回鹘语专业班。1978年后，到1990年代，他又去[[甘肃|甘肃]]、[[青海|青海]]等地调查，研究了裕固语和[[撒拉语|撒拉语]]<ref name="新疆">{{cite web |author1=新疆哲学社会科学网 |title=敬悼耿世民先生 |url=http://www.xjass.cn/ls/content/2013-01/20/content_261984.htm |website=新疆哲学社会科学网 |accessdate=2019-04-15}}</ref>。1992 年，[[德国洪堡基金会|德国洪堡基金会]]授予他洪堡科研奖<ref>{{cite journal |author1=赵震声 |title=德国洪堡基金会与中国洪堡学者 |journal=德国研究 |date=1999 |volume=14 |issue=01 |pages=49-53}}</ref>。2000 年，世界[[阿尔泰学|阿尔泰学]]学术机构[[国际阿尔泰学常设会议|国际阿尔泰学常设会议]]（PIAC）将印第安纳大学阿尔泰研究奖颁给了耿世民先生<ref>{{cite web |author1=Permanent International Altaistic Conference |title=Indiana University Prize for Altaic Studies, 1963–2014 |url=http://www.altaist.org/indiana-university-prize-for-altaic-studies/ |website=Permanent International Altaistic Conference |accessdate=2019-04-15 |archive-url=https://web.archive.org/web/20181030035544/http://www.altaist.org/indiana-university-prize-for-altaic-studies/ |archive-date=2018-10-30 |dead-url=no }}</ref>。他一生中最主要的成就是对古代鄂尔浑——叶尼塞突厥语和回鹘语的考释研究。2012年，耿世民在[[北京|北京]]逝世<ref>{{cite web |title=GENG Shimin vefat etti |url=http://tdk.gov.tr/index.php?option=com_content&view=article&id=300&Itemid=74 |website=Türk Dil Kurumu |accessdate=2019-04-15 |archive-url=https://web.archive.org/web/20160305005441/http://tdk.gov.tr/index.php?option=com_content&view=article&id=300&Itemid=74 |archive-date=2016-03-05 |dead-url=no }}</ref>。

== 出版物 ==
教材包括《古代突厥—回鹘语教材》（共八册，包括《古代突厥文碑铭选读》1册、《回鹘文献选读》2册、《黑汗王朝时期文献选读》3册、《总词汇》1册、《古代突厥—回鹘语文献语法》1册）等共五种，用汉、英、德等种语言文字出版了《维吾尔古代文化和文献概论》、《敦煌突厥回鹘文书导论》等专著24部，翻译出版世界著名学者用法、德、英、日、俄、古代突厥等语言撰写的《高地亚洲》(〔法〕伯希和)、《中亚突厥史十二讲》（〔前苏〕巴尔托里德）等著作14部(篇)；用中文、维吾尔文及英、法、德、日等文字发表《回鹘文亦都护高昌王世勋碑研究》、《回鹘文(八十华严)残经研究》等论文183多篇，另有新书短评和学界动态、通讯报道等二十六篇<ref name="新疆" /><ref name="中国" /><ref>{{cite web |author1=赵志研 |title=矢志不移成非凡 ──耿世民教授的维吾尔学及古代突厥语学研究之路 |url=http://iel.zhongyan.org/news_show.asp?newsid=3209 |website=中国社会科学院民族文化所 |accessdate=2019-04-15}}</ref>。
