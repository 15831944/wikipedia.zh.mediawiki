'''單立文'''（{{lang|en|'''Pal Sinn Lap Man'''}}，{{bd|1959年|3月17日|||catIdx=Shan單立文}}），[[香港|香港]]音樂人、演員和節目主持，現為[[無綫電視藝員列表|無綫電視基本藝人合約]]藝員<ref>[http://artiste.tvb.com/main/artistedetail/sinnlapman 單立文 Pal Sinn - TVB藝人資料 - tvb.com]</ref>及[[伯樂音樂學院|伯樂音樂學院]]流行音樂演奏系導師<ref name="伯樂">{{cite web | title = 伯樂音樂學院導師簡介 - 單立文 Pal Sinn |url = http://www.bsm.com.hk/b5/ipop_08.htm }}</ref>。

== 簡歷 ==
單立文家有二姊一弟，小時候因父親是警署警長（於1970年代離世）關係而在香港不同地區的警署宿舍如[[觀塘警署|觀塘警署]]和[[油麻地警署|油麻地警署]]居住，其中以居於[[九龍醫院|九龍醫院]]附近的高級公務員警察宿舍時間較長。他童年時代經常到[[京士柏|京士柏]]一帶，即鄰近昔日[[亞皆老街|亞皆老街]]軍營和[[英皇佐治五世學校|英皇佐治五世學校]]的山上戰壕裡遊玩。他學習資質不錯，但基於懶惰而沒有考取優良成績，且又自覺課堂教學內容不太有用，所以轉向其他方面發展。{{noteTag|資料來自[[香港電台第二台|香港電台第二台]]廣播節目《[[香港電台第二台|守下留情]]》於2014年11月3日播出的訪問《Hong Kong Rocker 系列七 - 嘉賓 單立文「兩位主持夠癲喪 豹哥焗住大解放」》。}}

他在1970年畢業於[[九龍塘學校（小學部）|九龍塘學校（小學部）]]，後於1975年畢業在[[龍翔官立工業中學|龍翔官立工業中學]]<ref>{{cite web | title = 龍翔官立工業中學 1975 至 1985 年度校友畢業名冊 | url = http://lcg.hkcampus.net/~lcg-osa/Grad75_85.htm }}</ref>。網上一直流傳他出生年份為1961年，而根據2011年 [[Now寬頻電視|NOW TV]] 綜藝節目《[[撳錢|撳錢]]》其中一集內容，提及其正確出生年份應為1959年，這已獲證實。

=== 發展音樂時期 ===
單立文在音樂範疇頗有天分，主要受父親聽古典音樂唱片，母親接觸舞蹈音樂，大姊彈鋼琴和二姊跳芭蕾舞所薰陶。後來他在音樂店舖中認識了搖滾樂友，亦在樓上琴行內聽過「細AL」楊雲驃（[[玉石樂隊|玉石樂隊]]成員）教授的一節樂理課。他讀書時已開始玩主音結他，尚未中學畢業前還已經參與樂隊演出，在灣仔[[分域街|分域街]]及[[駱克道|駱克道]]一帶的酒吧表演{{noteTag|資料來自[[香港電台第二台|香港電台第二台]]廣播節目《[[香港電台第二台|守下留情]]》於2014年11月3日播出的訪問《Hong Kong Rocker 系列七 - 嘉賓 單立文「兩位主持夠癲喪 豹哥焗住大解放」》。}}。

1983年，他跟[[唐納德·阿什利|唐龍]]、[[黃良昇|黃良昇]]及[[蘇德華|蘇德華]]組成香港[[樂隊|樂隊]]「Chyna」，並擔任[[低音吉他手|低音吉他手]]<ref name="伯樂"/>；當中《Within You'll Remain》一曲迄今已有7種語言、15個版本，曾獲[[新加坡|新加坡]]樂隊 [[Tokyo_Square|Tokyo Square]] 翻唱和改編成[[徐小鳳|徐小鳳]]的粵語歌曲《一臉紅霞》。1987年，他便與黃良昇、蘇德華再組成另一樂團「[[藍戰士|藍戰士]]（Blue Jeans）」，擔任低音吉他手<ref name="伯樂"/>。

=== 影視時期 ===
1990年，[[藍戰士|藍戰士]]解散後<ref>{{cite web | title = Blue Jeans 敢死隊 | url = http://www.xpweekly.com/share/xpweekly/0315/ent/20040831ent0054/content.htm | work = [[快周刊|快周刊]]（第315期） | date = 2004-08-31 }}</ref>單立文曾參與演出多齣電影和劇集。1990年在《[[潘金蓮之前世今生|潘金蓮之前世今生]]》中「[[西門慶|西門慶]]」一角也獲提名[[1990年香港電影金像獎|第九屆香港電影金像獎]]「最佳新演員」<ref>{{cite web | title = 第九屆香港電影金像獎得獎名單 - 香港電影金像獎官方網頁 | url = http://www.hkfaa.com/history/list_09.html }}</ref>，而同年演出《[[賭俠|賭俠]]》裡「[[侯賽因|侯賽因]]（台譯：海珊）」一角亦使他開始為人熟悉。1993年，他加入香港[[無綫電視|無綫電視]]，且分別在《[[恨鎖金瓶|恨鎖金瓶]]》及1996年《[[新金瓶梅_(电视剧)|新金瓶梅]]》系列內再度飾演「西門慶」，故被稱爲『西門慶專業戶』<ref>{{cite web | title =『西門慶專業戶』單立文月底與名主持結婚（圖）| url = http://entbig5.jxnews.com.cn/system/2008/03/14/002701496.shtml | work = 江西娛樂網 | date = 2008-03-14 | deadurl = yes| archiveurl = https://web.archive.org/web/20130717002824/http://entbig5.jxnews.com.cn/system/2008/03/14/002701496.shtml| archivedate = 2013-07-17}}</ref>。其後持續演出電影和劇集，直到2004年才淡出銀幕，潛心研究音樂。

2010年，單立文重返無綫電視，擔任該台歌唱節目《[[超級巨聲|超級巨聲]]》評判，但於2011年加盟 NOW TV 和主持綜藝節目《撳錢》。2012年又重返無綫電視，拍攝多部電視劇。2014年，單立文首度獲提名《[[萬千星輝頒獎典禮2014|萬千星輝頒獎典禮2014]]》「最佳男配角」。2016年，他再次獲提名《[[萬千星輝頒獎典禮2016|萬千星輝頒獎典禮2016]]》「最佳男配角」。另外，他是無綫電視監製[[林志華|林志華]]的常用演員。

2018年，單立文與[[夏韶聲|夏韶聲]]、[[黃貫中|黃貫中]]、[[鄧建明|鄧建明]]及[[恭碩良|恭碩良]]以組合「Hong Kong Band」身份在[[香港體育館|香港體育館]]舉行兩場演唱會，受到觀眾歡迎<ref>{{cite web | title = 《搖滾大俠奏》尾場 Rock 爆紅館　恭碩良單立文「因鹹濕結緣」？ | url = https://www.hk01.com/%E9%9F%B3%E6%A8%82/197346/%E6%90%96%E6%BB%BE%E5%A4%A7%E4%BF%A0%E5%A5%8F-%E5%B0%BE%E5%A0%B4rock%E7%88%86%E7%B4%85%E9%A4%A8-%E6%81%AD%E7%A2%A9%E8%89%AF%E5%96%AE%E7%AB%8B%E6%96%87-%E5%9B%A0%E9%B9%B9%E6%BF%95%E7%B5%90%E7%B7%A3 | work = [[香港01|香港01]] | date = 2018-06-08}}</ref>。

== 感情生活 ==
單立文與[[胡蓓蔚|胡蓓蔚]]於1996年年底開始交往，2008年結婚<ref>{{cite web | title = 汪琳下嫁浪漫才子 胡蓓蔚 單立文低調拉埋天窗 | url = http://paper.wenweipo.com/2008/03/29/EN0803290001.htm | work = [[香港文匯報|香港文匯報]] | date = 2008-03-29}}</ref>。
