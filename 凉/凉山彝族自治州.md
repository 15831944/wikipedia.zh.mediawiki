'''凉山彝族自治州'''（{{lang-ii|ꆃꎭꆈꌠꊨꏦꏱꅉꍏ}}，[[拉丁转写|拉丁转写]]：{{transl|ii|niep sha nuo su zyt jie jux dde zho}}，[[国际音标|国际音标]]：{{IPA|/nɛ˨˩ʂa˧ nɔ˧su˧ ʦɿ˥ʨɛ˧ ʨu˦dɪ̈˧ ʈʂʊ˧/}}），简称'''凉山州'''，是[[中华人民共和国|中华人民共和国]][[四川省|四川省]]下辖的[[自治州|自治州]]，位于四川省西南部，州境北邻[[甘孜州|甘孜州]]、[[雅安市|雅安市]]、[[乐山市|乐山市]]，东接[[宜宾市|宜宾市]]、[[云南省|云南省]][[昭通市|昭通市]]，西南界[[攀枝花市|攀枝花市]]，西连云南省[[丽江市|丽江市]]、[[迪庆州|迪庆州]]。地处[[横断山脉|横断山脉]]东北缘川西南山地，[[四川盆地|四川盆地]]与[[云贵高原|云贵高原]]过渡带，地势西北高，东南低。[[雅砻江|雅砻江]]自北部入境，以西为高原区，以东为山地区，岭谷与盆地相间，西部为[[大雪山山脉|大雪山山脉]]、[[锦屏山|锦屏山]]，中部为[[小相岭|小相岭]]、[[螺髻山|螺髻山]]、[[鲁南山|鲁南山]]山系，东部为[[大小凉山|大小凉山]]区。[[金沙江|金沙江]]流经南面与东面边界，为与云南省的界河，西北部有金沙江支流[[无量河|无量河]]，中部[[西昌盆地|西昌盆地]]有[[安宁河|安宁河]]纵贯，往南注入雅砻江。西部[[泸沽湖|泸沽湖]]、中部[[邛海|邛海]]、东部[[马湖|马湖]]为三大淡水湖。全州总面积6.03万平方公里，人口468万，[[彝族|彝族]]人口比例约49%。自治州首府驻[[西昌市|西昌市]]，素有“月城”之雅称，为[[西昌卫星发射中心|西昌卫星发射中心]]所在地，是闻名中外的航天城。

== 历史 ==
汉朝属[[益州刺史部|益州刺史部]][[越巂郡|越巂郡]]。南北朝时期名称有所更改。隋唐属[[巂州|巂州]]。唐末属[[南诏国|南诏国]]建昌府。宋短暂经手后属[[大理国|大理国]]建昌府。元属罗罗斯宣慰司，元中期自四川省划归云南省。明初月鲁帖木儿曾率蒙古降军叛乱，被[[蓝玉|蓝玉]]平。清朝属[[宁远府|宁远府]]。民国初年属[[川边特别行政区|川边特别行政区]]，后属[[西康省|西康省]]。1949年后，在西康省辖区内建立凉山彝族自治区，含11县。后改称自治州。

== 地理 ==
凉山州地貌复杂多样，地势西北高，东南低，海拔最高为5958米的木里恰朗多吉峰，最低为305米的雷波大岩洞金沙江谷底；常年最高积温6979℃，最低积温4086℃，年均气温14℃～17℃，[[立体气候|立体气候]]特征明显；属于[[亚热带季风气候|亚热带季风气候]]区，干湿分明，冬半年日照充足，少雨干暖，夏半年云雨较多，气候凉爽，年均降雨量1000～1100毫米；日照充沛，年均日照时数2000～2400小时，无霜期达230～306天。

凉山彝族自治州有耕地面积472万亩，境内江河纵横。水能资源丰富，可开发量达6,000万千瓦，是未来四川重要的水电基地。

[[安宁河谷平原|安宁河谷平原]]是四川第二大平原，是川西南地区农业综合开发重点区域。

== 经济 ==
凉山是中国[[少数民族|少数民族]]地区经济发展得较好的地方之一。2011年，全州生产总值1000.13亿元，地方公共财政收入80亿元，分居四川省第七、第二位，在全国30个少数民族自治州中均居第一位。2012年上半年，地区生产总值500.3亿元，地方公共财政收入55.3亿元。

== 资源 ==
凉山州出产优质烟叶和苹果、石榴、甘蔗等经济作物，品质和口味非常好。

== 文化 ==
凉山是古代[[南方丝绸之路|南方丝绸之路]]的必经要道。历史上司马迁、诸葛亮、忽必烈、马可·波罗等都曾在凉山游历、活动。凉山是[[彝语北部方言|彝族诺苏人]]文化的故乡。州内的昭觉、布拖、美姑、普格、金阳、喜德、普雄（后并入越西县）、雷波、越西被称为彝族老九县。民主改革前仍为半[[奴隶社会|奴隶社会]]。[[火把节|火把节]]是彝族人民最热闹的节日，现在发展已成为各民族朋友喜庆相聚的一个狂欢节。境内泸沽湖畔[[摩梭人|摩梭人]][[母系氏族|母系氏族]]时期的婚姻形态完整保留至今。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全州[[常住人口|常住人口]]为4532809人<ref>{{cite book |author=凉山彝族自治州统计局|title=《凉山彝族自治州2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共增加451112人，增长11.06%。年平均增长1.05%。其中，男性人口为2344030人，占51.71%；女性人口为2188779人，占48.29%。总人口性别比（以女性为100）为107.09。0－14岁人口为1238659人，占27.33%；15－64岁人口为2972335人，占65.57%；65岁及以上人口为321815人，占7.10%。

=== 民族 ===
[[汉族|汉族]]人口为2155357人，占47.55%；[[少数民族|少数民族]]人口为2377452人，占52.45%；其中：[[彝族|彝族]]人口为2226755人，占49.13%。

== 旅游 ==
[[File:1964-12_1964年_凉山人民医院.jpg|thumb]]
凉山拥有[[泸沽湖|泸沽湖]]、[[邛海|邛海]][[螺髻山|螺髻山]]两个国家级风景名胜区和中国四大[[碑林|碑林]]之一的西昌地震碑林。
