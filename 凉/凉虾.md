'''凉虾'''，又叫“米凉虾”，是一种用大米做成的特色小吃，因头大尾细形似虾，故此得名。广泛分布于[[湖北|湖北]][[宜昌|宜昌]]<ref>{{cite web|url =|title =冰凉爽滑的美味凉虾|accessdate =2009-12-18|date = 2006-06-17|publisher =[[中国三峡传媒网|中国三峡传媒网]]|language = zh-cn}}</ref>、[[重庆|重庆]]<ref>{{cite web|url =http://www.dianping.com/shop/533044|title =回头望冰粉凉虾|accessdate =2009-12-18|date = 05-06-09|publisher =[[大众点评网|大众点评网]]|language = zh-cn}}</ref>、[[四川|四川]]<ref>{{cite web|url =http://dzh.mop.com/topic/readSub_7678542_0_0.html|title =特色小吃：凉虾|accessdate =2009-12-18|date =2007-07-20|publisher =[[猫扑大杂烩|猫扑大杂烩]]|language =zh-cn}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>、[[云南|云南]]<ref>{{cite web|url =|title =玫瑰米凉虾|accessdate =2009-12-18|date = 2005-04-26|publisher =[[云南网|云南网]]|language = zh-cn}}</ref>、[[贵州|贵州]]等地。

==典故==
相传是[[长江|长江]][[三峡|三峡]]地区，一彭氏孝子因其母亲冬季想吃虾而制作出的小吃。

== 制作方法 ==
#将大米洗净，用水浸泡半小时
#把浸泡好的大米加水磨制成米浆
#将米浆徐徐倒入沸腾的水中，为防止糊锅边倒边用木勺搅拌，同时加少许食用碱，用微火煮至糊状
#另外再准备一盆凉开水，将煮好的米糊慢慢倒入漏瓢中，米糊通过漏孔滴落至凉开水中，即成“虾”的模样
#等待片刻，用汤勺将凉开水中的凉虾捞起入碗，加入适量红糖水、冰水，便是一碗冰凉可口、消暑解渴的凉虾了，也可以根据个人口味加入蜂蜜、甜米酒、水果蜜饯等<ref>{{cite web|url =http://www.meishij.net/china-food/xiaochi/hubei/14717.html|title =凉虾|accessdate =2009-12-18|date = 2007-01-23|publisher =[[美食杰网|美食杰网]]|language = zh-cn}}</ref>

==都市文化==
在湖北宜昌，凉虾是街边随处可见的平民小吃，由路边小贩经营，多为家庭自制。一只只白色的“小虾”装在透明玻璃容器中，看起来甚是诱人。夏天一到，人们走路走累了，总来上一碗冰凉虾。<ref>{{cite web|url =|title =市民推荐：独具宜昌特色的冰镇“凉虾”|accessdate =2009-12-18|date = 2004-08-06|publisher =[[三峡·宜昌网|三峡·宜昌网]]|language = zh-cn}}</ref>。

凉虾深入宜昌地区的草根文化之中，宜昌方言歌曲《凉虾FEAT赤花子&酸梅汤》<ref>{{cite web|url =http://www.cnblogs.com/DrkBreeze/archive/2008/07/02/1234303.html|title =宜昌方言歌曲《凉虾FEAT赤花子&酸梅汤》|accessdate =2009-12-18|date = 2008-07-02|publisher =[[博客园|博客园]]|language = zh-cn}}</ref>便是以凉虾等夏季消暑小吃为题材的宜昌方言歌曲。
