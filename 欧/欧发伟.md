'''欧发伟'''是[[中国|中国]][[学者|学者]]，[[作家|作家]]，文学[[评论家|评论家]]，中国（海南）[[改革|改革]]发展[[研究院|研究院]]专栏学者，中华人民共和国[[民政部|民政部]][[中国社会工作协会|中国社会工作协会]]专栏学者，中华[[诗词|诗词]]文化传承人，“[[泰戈尔体诗歌|泰戈尔体诗歌]]”创始人，[[中国管理科学研究院|中国管理科学研究院]]学术委员会特约[[研究员|研究员]]。主要研究领域为中国[[经济改革|经济改革]]理论和政策、[[法学|法学]]、[[政治学|政治学]]、[[文化|文化]]现象、中国现当代[[文学|文学]]等，在中华诗词创作和理论研究领域造诣颇深。对著名[[经济学家|经济学家]][[厉以宁|厉以宁]]的非均衡理论等[[经济学|经济学]]观点和著名学者[[王沪宁|王沪宁]]的《比较政治分析》等政治学观点均有研究和见解。

== 人物概况 ==
欧发伟，[[中国|中国]][[学者|学者]]，[[文人|文人]]，[[作家|作家]]，文学[[评论家|评论家]]，国际[[诗人|诗人]]。现为中国（海南）[[改革|改革]]发展[[研究院|研究院]]专栏学者，中华人民共和国[[民政部|民政部]][[中国社会工作协会|中国社会工作协会]]专栏学者，[[中国管理科学研究院|中国管理科学研究院]]学术委员会特约[[研究员|研究员]]，中华诗词学会会员，世界汉诗协会会员。在中国（海南）改革发展研究院中国改革论坛网、中华人民共和国民政部中国社会工作协会官方网站开设学者专栏。长期从事中共党建和[[干部|干部]]选拔任用理论研究与实务工作，在[[人民日报|人民日报]]等[[党报党刊|党报党刊]]和内刊发表研究性和经验性等文章200余万字。对著名[[经济学家|经济学家]][[厉以宁|厉以宁]]的非均衡理论等[[经济学|经济学]]观点和著名学者[[王沪宁|王沪宁]]的《比较政治分析》等政治学观点均有研究和见解。致力于中华诗词创作和理论研究，专著《中国笔记》被文学[[评论家|评论家]]称为“[[中国文学史|中国文学史]]上第一部诗化体随笔集”。曾多次受邀出席国际性[[学术会议|学术会议]]，首次提出四大学术观点，受到学术界广泛关注。

== 研究领域 ==
中国经济改革理论和政策、法学、政治学、文化现象、中国现当代文学等。

== 主要成就 ==
*创立“泰戈尔体诗歌”。

*首次提出四大学术观点。

*获[[国际|国际]]中华诗词文化传承人称号。

== 主要著作 ==
《中国笔记》、《红楼遗梦》。
