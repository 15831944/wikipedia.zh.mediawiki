'''歐洲中央銀行'''（{{lang-fr|Banque Centrale Européenne}}；{{lang-de|Europäische Zentralbank}}；{{lang-en|European Central Bank}}），簡稱'''歐洲央行'''，總部位於[[德國|德國]][[法蘭克福|法蘭克福]]，成立於1998年6月1日，負責[[歐盟|歐盟]][[欧元区|欧元区]]的金融及貨幣政策，為歐元區的[[中央銀行|中央銀行]]。

首任歐洲中央銀行行長為[[维姆·德伊森贝赫|维姆·德伊森贝赫]]，曾任[[荷蘭中央銀行|荷蘭央行]]行長，曾在[[阿姆斯特丹大學|阿姆斯特丹大學]]教授宏觀經濟學。其繼任人為[[让-克洛德·特里谢|让-克洛德·特里谢]]，法國人，曾任法國央行[[法蘭西銀行|法蘭西銀行]]行長，於2003年11月接任。现任歐洲中央銀行行長為[[马里奥·德拉基|马里奥·德拉基]]，[[意大利|意大利]]人，曾任[[意大利银行|意大利银行]]行長，於2011年11月接任。

== 組織和结构 ==
[[File:Frankfurt_European_Central_Bank.JPG|thumb]]]]
歐洲中央銀行由其[[董事會|董事會]]負責管理，設有董事會主席，並設有[[理事會|理事會]]，包括了董事會的成員及各歐洲國家各自的中央銀行代表。

歐洲中央銀行的組織主要是以[[德國中央銀行|德國中央銀行]]及[[西德意志地方票據交換銀行|西德意志地方票據交換銀行]]為主要模仿對象。

== 执行董事会 ==
执行董事会包括六名成员，为欧洲央行的政策制定策略。其中的四个位置为欧盟四个最大的中央银行，法国、德国、意大利和西班牙保持。

== 歐洲中央銀行體系 ==
[[歐洲中央銀行體系|歐洲中央銀行體系]]（ESCB）包含了歐洲中央銀行及歐盟27個成員國各自的中央銀行，只有各成員國自身負責歐盟事務的央行董事可以參與其中及進行決策。

== 批評 ==
歐洲中央銀行主要面對兩種批評，分別為自主性不足及目標不均衡。

=== 自主性 ===
[[File:La2-euro.jpg|thumb]]
部份批評者認為歐洲中央銀行的自主性不足。歐洲中央銀行是各成員國的中央銀行聯合籌備成立的，以避免受到政治的影響。雖然其成立的目的及權力均來自政治，但其決策權力卻由歐洲中央銀行自身掌握，不受各國的影響。而那些不在歐元區內的歐盟國家的中央銀行（如[[丹麥中央銀行|丹麥中央銀行]]及[[英格蘭銀行|英格蘭銀行]]），亦受這些條文約束。部份批評者認為歐洲中央銀行[[民主|民主]]程度不高，其決策過程黑箱作業，缺乏應有的監管，所以其最終政策有侵犯[[人權|人權]]及破壞[[自然環境|自然環境]]的可能性。歐洲中央銀行不會發佈其決定，亦不會接受意見。當執行決策後，其主頁並不會收集市民的意見。因為要保障理事會成員的權益，所以其內部會議的紀錄不會披露。歐盟公民可以通由選舉[[歐洲議會|歐洲議會]]議員而間接影響歐洲中央銀行的決策，然而若然經濟環境不許可，政客亦可能沒法改變歐洲中央銀行的決策。不過歐洲中央銀行需要向歐洲議會負責，主要官員的委任亦需要獲得其通過。而法律亦規定歐洲中央銀行行長需要每年向歐洲議會提交年度報告，除此之外，歐洲中央銀行董事會成員亦需每年四次與歐洲議會進行會面，以交待中央銀行自身情況，若有需要，可以舉行更頻密的舉行。很多經濟學家均認為歐洲中央銀行的自主性是極其重要的，因為這可以防止市場因政治目的而被操控。

=== 通膨目標 ===
部份批評者認為歐洲中央銀行所定下的目標並不合理，歐洲中央銀行以利率機制來控制[[通脹率|通脹率]]，但卻不為[[失業率|失業率]]及[[貨幣匯率|貨幣匯率]]設定調控政策，這使得部份人感到這樣的經濟政策並不平衡，有可能導致各項經濟及民生的問題。很多英國的經濟學家均指出歐洲中央銀行需要實行平衡的目標組合，就如英格蘭中央銀行現在所採用的。而歐洲中央銀行所設定的不平常的低[[利率|利率]]環境亦受到很多批評者批評，認為並不符合歐盟成員國自身的情況。這個低利率環境正是[[愛爾蘭經濟泡沫|愛爾蘭經濟泡沫]]的成因。雖然這可以防止歐盟主要成員國如[[法國|法國]]、[[德國|德國]]及[[義大利|義大利]]出現經濟不景氣，然而卻沒有顧及其他成員國的利益。需要注意的是[[日本|日本]]及[[美國|美國]]近年亦採用低利率政策。
