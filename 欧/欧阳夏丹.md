'''欧阳夏丹'''（{{bd|1977年|7月28日||catIdx=Ou欧阳夏丹}}），[[广西壮族自治区|广西壮族自治区]][[桂林市|桂林市]]人。1999年毕业于[[北京广播学院|北京广播学院]]（现[[中国传媒大学|中国传媒大学]]）。1999年至2003年在[[上海电视台新闻综合频道|上海电视台新闻综合频道]]工作，主播过《上海早晨》、《新闻夜线》等节目。2003年8月进入[[中国中央电视台经济频道|中国中央电视台经济频道]]，主播经济频道早间资讯栏目《第一时间》。2009年7月加盟[[中国中央电视台新闻频道|中国中央电视台新闻频道]]，主播央视新闻频道《[[共同关注|共同关注]]》节目。

2011年9月25日，歐陽夏丹與[[郎永淳|郎永淳]]开始主持[[中国中央电视台|中国中央电视台]]最主要的官方新聞节目《[[新聞聯播|新聞聯播]]》，並獲得普遍好評。<ref>[http://www.nownews.com/2011/09/26/91-2744792.htm#ixzz1Z8IlTooH NOWNEWS：央視美女主播展笑容]</ref>

== 简历 ==
* 1992年至1995年：广西[[桂林市|桂林市]][[桂林中学|桂林中学]]，以桂林市文科状元{{NoteTag|网上误传为嘉兴市文科状元。见[http://media.163.com/10/0416/10/64CQS8D100762621.html 状元出身的七大美女主播]、[http://www.jiaxing.org/news/hechengxinwen/201005/6275.html 欧阳夏丹是嘉兴高考状元？]}}考入[[北京广播学院|北京广播学院]]。
* 1995年至1999年：北京广播学院播音系。
* 1999年至2003年8月：上海电视台新闻综合频道，主播过《上海早晨》、《新闻夜线》等节目。
* 2003年10月至2009年6月：中央电视台经济频道《第一时间》主持人。
* 2007年：以《第一时间》获[[金话筒奖|金话筒奖]]电视主持作品奖。
* 2009年8月至今：中央电视台新闻频道，主播每天18:00-19:00的《共同关注》。
* 2010年11月：主持金话筒奖颁奖盛典。
* 2011年，欧阳夏丹着装[[透视装|透视装]]引发在中央电视台新闻频道播报新闻，舆论热议。<ref>{{cite web |url = http://news.qq.com/a/20110123/000470.htm |title =《央视主播欧阳夏丹透视装出镜》|accessdate=2011年1月23日 |website = 腾讯网 |language = zh-hans }}</ref>
