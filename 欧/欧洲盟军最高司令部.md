'''欧洲盟军最高司令部'''（{{lang-en|Supreme Headquarters Allied Powers Europe}}，缩写'''SHAPE'''）在2003年前是[[北大西洋公约组织|北大西洋公约组织]]'''欧洲盟军司令部'''（Allied Command Europe，ACE）的总部，2003年后则成为了'''盟军作战司令部'''（Allied Command Operations，ACO）的总部，负责统一指挥北约的军事行动。

==历史==
[[朝鲜战争|朝鲜战争]]爆发后，西欧各国对于能否有效抵御[[苏联|苏联]]进攻产生了怀疑，对统一的军事指挥机构的需求也随之诞生。1951年，隶属北约的欧洲盟军最高司令部正式成立，负责北约在欧洲的所有军事行动。首任司令为[[五星上将|美国陆军五星上将]][[德怀特·艾森豪威尔|德怀特·艾森豪威尔]]，副司令则为[[英国陆军元帅|英国陆军元帅]][[伯纳德·蒙哥马利|伯纳德·蒙哥马利]]。自1967年来，其地址设于[[比利时|比利时]][[蒙斯|蒙斯]]以北的{{link-en|卡斯托（比利时）|Casteau|卡斯托}}小镇，在此之前则坐落于[[法国|法国]][[巴黎|巴黎]][[凡爾賽鎮|凡尔赛镇]]附近。<ref>SHAPE, 7010 Casteau Belgium {{cite web|url=http://www.nato.int/shape/|title=SHAPE on NATO homepage|accessdate=2006年3月12日|language=en}}</ref>从1951年设立到2003年，SHAPE一直是北约欧洲盟军司令部的总部，2003年北约大规模改革后则收编{{link-en|大西洋盟军司令部|Allied Command Transformation#Allied Command Atlantic 1952 to 2003}}（Allied Command Atlantic，ACA）的部分机构成为盟军作战司令部总部 ，统一指挥北约在全球的军事行动。<ref>{{cite web|title=北约为何进行史上最大幅度军事改革|publisher=人民网|url=http://www.people.com.cn/GB/guoji/1030/1931386.html|date=2003年6月23日|accessdate=2016年12月13日|language=中文}}</ref>


2003年北约改革时，除欧洲盟军司令部在蒙斯的行动指挥部外，位于[[葡萄牙|葡萄牙]][[里斯本|里斯本]]原本隶属大西洋盟军司令部的行动指挥部也被划分给了新的盟军作战司令部。（大西洋盟军司令部的剩余部分则在重组后成为了{{link-en|盟军转型司令部|Allied Command Transformation}}。）尽管欧洲盟军最高司令部在改革后作为盟军作战司令部所管辖的地域已不仅限于欧洲，SHAPE的名字出于法律考虑却被保留，并没有做出相应的更改。<ref name=PedlowNATO>{{cite web|url=http://www.aco.nato.int/resources/21/Evolution%20of%20NATO%20Cmd%20Structure%201951-2009.pdf|title=Evolution of NATO's Command Structure 1951-2009|language=en}}</ref> 盟军作战司令部的指挥官也依旧保留了“欧洲盟军最高司令”（Supreme Allied Commander Europe，缩写SACEUR）的头衔，并按照传统依旧由一名美军上将担任（此人同时也兼任[[美国欧洲司令部|美国欧洲司令部]]司令）。
