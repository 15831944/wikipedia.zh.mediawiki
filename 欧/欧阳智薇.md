'''欧阳智薇'''{{BD|1987年|10月2日|||catIdx= Ou欧阳}}出生于[[湖南省|湖南省]][[怀化市|怀化市]]，毕业于[[中国传媒大学|中国传媒大学]]播音与主持艺术学院2005级本科班，为[[中国中央电视台|中国中央电视台]]节目主持人<ref>{{cite web |url=http://news.163.com/09/0831/14/5I271DQ600012QEA.html |title=《第一时间》节目接班主持人欧阳智薇 |publisher= [[网易|网易]]|accessdate=2009-08-31 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://jingji.cntv.cn/program/diyishijian/20110222/109112.shtml |title=《第一时间》主持人欧阳智薇  |publisher= [[中国网络电视台|中国网络电视台]]|accessdate=2011-02-02 |work= |language =zh-cn}}</ref>。

2005年，欧阳智薇獲得第三屆“新苗杯”中學生電視節目主持大賽湖南賽區冠軍、全國總決賽優秀獎。2006年8月27日至11月14日，資生堂瑞麗第四屆封面女孩大賽，欧阳智薇獲選“雜誌封面公主”。2009年4月，大學四年級的欧阳智薇开始在[[中国中央电视台财经频道|中国中央电视台财经频道]]《第一时间》实习；2009年4月22日，欧阳智薇首次在《第一时间》出镜。2009年6月，欧阳智薇大學畢業。2009年8月31日，欧阳智薇正式成为《第一时间》主播。2011年5月28日，欧阳智薇赴[[江西省|江西省]][[宜春市|宜春市]]實地主持中国中央电视台第十一届[[CCTV模特电视大赛|CCTV模特电视大赛]]。

2014年6月，媒體報道稱歐陽智薇因涉[[郭振璽|郭振璽]]案，已被檢察機關帶走“協助調查”<ref>{{cite web|url=http://www.chinese.rfi.fr/中国/20140617-央视主持人欧阳智薇涉郭振玺案协助调查|title=央视主持人欧阳智薇涉郭振玺案协助调查|publisher=[[法廣|法廣]]|date=2014年6月17日}}</ref>，但其本人多年后在个人微博账号回应称传言不实<ref>{{cite web |title=欧阳智薇的微博 |url=https://weibo.com/1667491333/Gzr96pNGz |publisher=微博 |date=2018-09-15 |quote=没被查办，没入狱，没结婚，没生子，没高官，没金钱问题，没私生活混乱。简单生活，开心做人，好好学习，天天向上。感谢人生教我高低与珍惜，我管不住传谣言的嘴，但可以管好自己的脚步。感谢多年关心我的朋友：）～}}</ref>。
