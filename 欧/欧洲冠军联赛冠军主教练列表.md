[[File:2005_European_Champion_Clubs'_Cup_(cropped).jpg|thumb]]冠军奖杯]]

[[欧洲冠军杯|欧洲冠军杯]]是一项自1956年至1992年期间举办、由[[歐洲足球協會|歐洲足球協會]]所屬成員國的本土聯賽冠軍参加的赛事<ref name="History">{{en}}{{Cite web | url = http://www.uefa.com/uefachampionsleague/history/index.html | title = Football's premier club competition| publisher = [[UEFA|UEFA]] | accessdate = 2013-06-19| date = 2007-07-02}}</ref>。[[西班牙|西班牙]]主教练[[何塞·比利亚隆加·略伦特|何塞·比利亚隆加·略伦特]]率领[[皇家马德里|皇家马德里]]在首届[[1956年欧洲冠军杯决赛|1956年欧洲冠军杯决赛]]中获得冠军，并在随后的一个赛季复制了这一壮举<ref>{{zh}}{{Cite web | url = http://sports.qq.com/a/20120308/000792.htm | title = 西媒票选皇马史上最佳主帅 博斯克压穆帅当选| publisher = 腾讯 | accessdate = 2013-06-19| date = 2012-03-08}}</ref>。[[英格兰|英格兰]]主教练及球队在1970年代末至1980年代初在这项赛事中占统治地位，曾在1977年至1982年连续6次夺冠<ref>{{zh}}{{Cite web | url = http://sports.cctv.com/20090313/104362.shtml | title = 欧冠八强英独占四席 世界谁将举“抗英”大旗| publisher = 中国中央电视台 | accessdate = 2013-06-19| date = 2009-03-13}}</ref>。尽管如此，[[意大利|意大利]]主教练仍是最为成功的，他们自1956年以来已经赢得11次冠军。

该项赛事在1993年改制为[[欧洲冠军联赛|欧洲冠军联赛]]<ref name="History"/>，[[比利时|比利时人]][[雷蒙·高達斯|雷蒙·高達斯]]率领的[[法国|法国]]球队[[马赛足球俱乐部|马赛]]在该年度获得冠军<ref>{{zh}}{{Cite web | url = http://sports.sohu.com/20080522/n257003713.shtml?wd= | title = 25年纵横欧陆无人可匹敌 不老爵爷直指最强纪录| publisher = 搜狐 | accessdate = 2013-06-19| date = 2008-05-22}}</ref>。

该项赛事仅[[卡洛·安切洛蒂|卡洛·安切洛蒂]]、[[鲍勃·佩斯利|鲍勃·佩斯利]]以及[[齐内丁·齐达内|齐内丁·齐达内]]三人曾获得过3次冠军，其中安切洛蒂分别率领两支不同的球队三次夺冠，佩斯利则率领[[利物浦足球俱乐部|利物浦]]在1977年、1978年和1981年的欧冠决赛中取得胜利，齐达内在2016年、2017年、2018年率领[[皇家马德里足球俱乐部|皇家马德里]]连续三次摘得桂冠；另有十七位主教练则曾2次摘得桂冠<ref>{{zh}}{{Cite web| url = http://news.sports.cn/global/ucl09/fin/2010-05-23/2020678.html| title = 欧冠历届夺冠主教练：穆帅成带不同队夺冠第3人| publisher = 华奥星空| accessdate = 2013-06-19| date = 2010-05-23| deadurl = yes| archiveurl = https://web.archive.org/web/20100528045032/http://news.sports.cn/global/ucl09/fin/2010-05-23/2020678.html| archivedate = 2010年5月28日| df = }}</ref>。只有五位主教练曾率领两支不同的球队夺冠，他们是：卡洛·安切洛蒂，2003年、2007年率[[AC米兰|AC米兰]]和2014年率皇家马德里；[[恩斯特·哈佩尔|恩斯特·哈佩尔]]，1970年率[[费耶诺德|费耶诺德]]和1983年率[[汉堡足球俱乐部|汉堡]]；[[奥特马·希斯菲尔德|奥特马·希斯菲尔德]]，1997年率[[多特蒙德足球俱乐部|多特蒙德]]和2001年率[[拜仁慕尼黑|拜仁慕尼黑]]；[[何塞·穆里尼奥|何塞·穆里尼奥]]，2004年率[[波尔图足球俱乐部|波尔图]]和2010年率[[国际米兰|国际米兰]]；[[積普·希基斯|積普·希基斯]]，1998年率皇家马德里和2013年率拜仁慕尼黑<ref name="multip">{{zh}}{{Cite web | url = http://sports.163.com/13/0524/10/8VKRSFRE00051C97.html | title = 海帅若再夺欧冠可比肩3传奇 一项荣耀爵爷再难追| publisher = 网易 | accessdate = 2013-06-19| date = 2013-05-24}}</ref>。此外，共有七位冠军主教练在其球员生涯中也曾夺冠，分别是[[米基爾·文奴斯|米基爾·文奴斯]]、[[吉奥瓦尼·特拉帕托尼|吉奥瓦尼·特拉帕托尼]]、[[约翰·克鲁伊夫|约翰·克鲁伊夫]]、卡洛·安切洛蒂、[[弗兰克·里杰卡尔德|弗兰克·里杰卡尔德]]、[[何塞普·瓜迪奥拉|何塞普·瓜迪奥拉]]和[[齐内丁·齐达内|齐内丁·齐达内]]<ref>{{en}}{{Cite web|url=http://www.uefa.com/footballeurope/club=50080/coach.html |title=Frank Rijkaard |publisher=[[UEFA|UEFA]] |accessdate=2009-05-28 |archiveurl=https://web.archive.org/web/20080205090519/http://www.uefa.com/footballeurope/club%3D50080/coach.html |archivedate=2008-02-05 |deadurl=yes }}</ref><ref>{{en}}{{Cite web | url = http://www.uefa.com/footballeurope/club=50080/coach.html | title = Josep Guardiola| publisher = [[UEFA|UEFA]] | accessdate = 2009-05-28}}</ref>。
