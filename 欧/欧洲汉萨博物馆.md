'''欧洲汉萨博物馆''' ({{lang-de|Europäisches Hansemuseum}})是德国[[吕贝克|吕贝克]]的一个关于[[汉萨同盟|汉萨同盟]]历史的博物馆，占地面积{{convert|7405|sqm}}，是世界上关于该主题最大的博物馆<ref name=deutschinfo>{{cite web |url= http://www.luebeck.de/tourismus/kultur/museen/europaeisches-hansemuseum/|title= Europäisches Hansemuseum Lübeck|author=|date= |website= |language=de|publisher= Lübeck Fenster. Offizielle Website der Hansestadt Lübeck|access-date= 2016-12-12|quote=}}</ref><ref name=hamburg>{{cite web |url= http://www.hamburg.de/museum-hamburg/4503942/europaeisches-hansemuseum-luebeck/|title= Hansemuseum Lübeck|language=de|publisher= City of Hamburg|access-date= 2016-12-12|quote=}}</ref>。博物馆于2015年5月27日开幕。5月30日迎来第一位访客<ref name=dw>{{cite web |url= http://www.dw.com/en/l%C3%BCbeck-opened-hanseatic-museum/a-18477263|title= Lübeck opened Hanseatic museum|last= Kirchhoff|first= Andreas|date= |website= |publisher= [[Deutsche_Welle|Deutsche Welle]]|access-date= 2016-12-12|quote=}}</ref>。

== 收藏和展览 ==
博物馆包括昔日[[诺夫哥罗德|诺夫哥罗德]]、[[布鲁日|布鲁日]]、[[卑尔根|卑尔根]]和[[伦敦|伦敦]]以及吕贝克等贸易港口的历史场景。

==位置==
博物馆位于[[世界遗产|世界遗产]]吕贝克老城区的北部，是一座13世纪[[多明我会|多明我会]]城堡修道院的一部分，被称为 "德国北部最重要的中世纪古迹之一"<ref name=frame>{{cite web |url= http://www.frameweb.com/news/andreas-heller-s-brick-facade-makes-a-confident-statement-at-the-hansemuseum|title= Andreas Heller's brick facade makes a confident statement at the Hansemuseum|last= Aghabeigi|first= Melika|date= 2015-08-27|website= |publisher= Frame publishers|access-date= 2016-12-12|quote=}}</ref>。
