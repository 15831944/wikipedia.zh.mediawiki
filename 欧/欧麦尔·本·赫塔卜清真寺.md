'''欧麦尔·本·赫塔卜清真寺'''（{{lang-es|Mezquita de Omar Ibn Al-Jattab}}）是[[哥伦比亚|哥伦比亚]][[瓜希拉省|瓜希拉省]][[迈考|迈考]]的一座[[清真寺|清真寺]]，是[[拉丁美洲|拉丁美洲]]第三大清真寺。它在当地通常直接被称为“La Mezquita”（清真寺），因为它是该地区唯一的清真寺。与达尔卡学校（Dar Alarkan School）一起，是该地区[[伊斯兰|伊斯兰]]信仰和文化的中心。这座清真寺建于1997年9月17日，以第二代[[逊尼派|逊尼派]]哈里发[[欧麦尔·本·赫塔卜|欧麦尔·本·赫塔卜]]的名字命名。清真寺由[[伊朗|伊朗]]建筑师阿里·纳马齐（Ali Namazi）设计，由土木工程师奥斯瓦尔多·比斯凯诺·方塔尔沃（Oswaldo Vizcaino Fontalvo）建造，建筑材料使用意大利大理石。它能轻易容纳1000多人。<ref>

==历史==
欧麦尔·本·赫塔卜清真寺是1960年至1970年期间在迈考定居的中东移民浪潮所留下的一个足迹，移民从[[里奥阿查|里奥阿查]]登陆后向东走了72公里来到迈考。清真寺于1997年启用，由[[伊朗|伊朗]]建筑师阿里·纳马齐用意大利大理石建造。<ref name="1R">{{Cite web |url = https://www.ecured.cu/Mezquita_de_Omar_Ibn_Al-Jattab |title = Mezquita de Omar Ibn Al-Jattab|date =  |language = es |accessdate = 2019-03-01 }}</ref>

2007年，清真寺建成10周年，很多来自[[瓜希拉省|瓜希拉省]]的[[穆斯林|穆斯林]]前来参加活动。<ref name="2R">{{Cite web |url = https://www.webislam.com/noticias/50290-la_mezquita_de_maicao_colombia_cumple_diez_anos.html |title = La mezquita de Maicao (Colombia) cumple diez años|language = es |accessdate = 2019-03-01 }}</ref>

==内部==
在入口处有一个很大的使用带框的[[阿拉伯书法|阿拉伯书法]]装饰的开放大厅。再往里还有一个更大的大厅，房间的天花板上有装饰性的雕刻。大厅供[[穆斯林|穆斯林]]祈祷，这也是他们见面以结束禁食的地方。在楼上有一个供妇女祈祷的地方，它面对着圣城[[麦加|麦加]]，可以俯瞰下面的男士祈祷大厅。[[宣礼塔|宣礼塔]]主宰着建筑的上部。

在清真寺出来的大楼梯下面，有一个房间供死者在遗体被送往当地穆斯林公墓之前进行纪念仪式。<ref name="1R"/>

==外部==
在清真寺的外面，可以看到一个坚固的大型结构，圆形彩色玻璃窗和一座31米高的[[宣礼塔|宣礼塔]]。<ref name="1R"/>
