'''矮子丕平'''（{{lang-fr|Pépin le Bref}}，{{bd|714年||768年|9月24日|catIdx=P}}），又称'''丕平三世'''（{{lang|fr|Pépin III}}），是公元751年至768年在位的法兰克国王，[[查理·马特|查理·马特]]之子。'''矮子丕平'''（{{lang|fr|Pépin le Bref}}）一称出现在后期史书中，应该是因为他身材矮小，因Bref在当时法语实意为court，短小之意。丕平是[[加洛林王朝|加洛林王朝]]的创建者。他的兒子是中世纪使法蘭西成為強權國家的[[查理大帝|查理大帝]]。西元768年9月24日，丕平因[[水肿|水肿]]病死在[[巴黎|巴黎]]，他在過世前將权力转移给儿子[[查理曼|查理]]和[[卡洛曼一世|卡洛曼一世]]。

== 与[[教會|教會]]的关系 ==

丕平問教皇：「國王不理政事，[[宮相|宮相]]應該怎麼辦？」当時教皇也急需強有力的支持，於是表示“誰為法蘭克操勞，誰就是它的主人”。這個回答正合丕平的心意。丕平隨即強迫[[墨洛溫王朝|墨洛溫王朝]]的最后一个国王[[希尔德里克三世|希尔德里克三世]][[剃髮|剃髮]][[出家|出家]]，成為[[修士|修士]]，名正言順地結束[[墨洛溫王朝|墨洛溫王朝]]，開創了[[卡洛林王朝|加洛林王朝]]。

751年，[[羅馬教皇|羅馬教皇]][[聖匝加|聖匝加]]派大主教来到[[巴黎|巴黎]]为丕平加冕。这意味着俗世[[政权|政权]]需要来自[[教權|教權]]的批准。这一事件为后来[[神权|神权]]与[[君权|君权]]的斗争留下了争执的根源。

为了报答教皇，丕平随即站在教皇一边，反对[[伦巴底人|伦巴底人]]。751年，由于伦巴底人攻占[[拉文纳|拉文纳]]，教皇[[斯蒂芬三世|斯蒂芬三世]]亲自到法兰克王国向丕平求救。

754年，丕平进军意大利，强迫伦巴底国王{{link-en|埃斯托夫|Aistulf}}答应将[[拉文纳|拉文纳]]及其新近占领的土地交给罗马[[教皇|教皇]]。756年丕平打败并俘虏了伦巴底国王。自此，从拉文纳到罗马的大片领土便划为教皇辖区，形成了一个“[[教皇国|教皇国]]”。史称“[[丕平献土|丕平献土]]”。

== 征讨 ==

丕平在位期间，开始了大规模的征服。与[[阿基坦|阿基坦]]公爵魏法尔连续打过九年阿基坦战役。
