《'''夷酋列像'''》是江户时代后期[[松前藩|松前藩]]的[[家老|家老]]、画家{{tsl|ja|蠣崎波響}}的一组肖像画，绘制了当时[[阿伊努人|阿伊努人]]的十二名首领。

== 起源 ==
[[宽政|宽政]]元年（[[1789年|1789年]]），[[国后岛|国后岛]]和[[目梨郡|目梨]]因不满大和人商人的残酷而起兵，杀死了70个大和人，是为[[国後目梨之戰|国後目梨之戰]]。[[松前藩|松前藩]]派遣260人的军队迎战，蠣崎波響为指挥官之一。平定后，43个阿伊努人因帮助松前藩镇压起义而受邀前往松前城谒见。次年，阿伊努人再次谒见时，受藩主{{tsl|ja|松前道広|松前道广}}的命令，蠣崎波響绘制了其中12个协助镇压有功的阿伊努首领，是为夷酋列像。画作于本年11月完成，作为国後目梨之戰战胜的纪念作品而进献给[[光格天皇|光格天皇]]。<ref>{{Cite book|author=岡田和彦、他|year=1988|title=夷酋列像　蠣崎波響|publisher=図書裡会（市立函館図書館）|pages=36p}}</ref>

== 收藏场所 ==
《夷酋列像》有六个传世版本。
# {{tsl|en|Musée des Beaux-Arts et d'Archéologie de Besançon|贝桑松美术馆}}所藏版本，有除乙箇律葛亞泥之外的11人的画像和两篇序文。来历不明。（上文表格所用的版本）
# {{tsl|ja|函館市中央図書館|函馆市中央图书馆}}所藏版本<ref>已在网络上公开。[http://archives.c.fun.ac.jp/fronts/detail/scrollframe/50474d761a55723cc0000051 ションコ][http://archives.c.fun.ac.jp/fronts/detail/scrollframe/50474d5c1a55723cc0000050 イコトイ] - 函館市中央図書館デジタル資料館</ref>，只有贖穀和乙箇吐壹的肖像，名为《御味方蝦夷之圖》。
# {{tsl|ja|松浦史料博物館}}所藏版本，有全部12人的肖像，是[[平户藩|平户藩]]藩主{{tsl|ja|松浦静山}}命画工摹写的版本。
# [[滨松市|滨松市]]常乐寺所藏版本，有乙唫葛律、訥窒狐殺、卜羅鵶、乙箇律葛亞泥、泥湿穀末決、窒吉律亞湿葛乙六人的肖像，是画家渡邊廣輝于文化元年（1804年）摹写的版本。
# 北尾家所藏版本，有全部12人的肖像，是天保14年（1843年）小島貞喜摹写的版本。
# 函馆市中央图书馆所藏的粉本<ref>已在网络上公开。[http://archives.c.fun.ac.jp/fronts/detail/reservoir/4f0ab6f3ea8e8a08d2000036 夷酋列像伝粉本] - 函館市中央図書館デジタル資料館</ref>，由蠣崎波響之子蠣崎波鶩流传的版本，少了失莫窒的肖像，但多了3个人名未知者的肖像。被列为北海道指定有形文化财产。<ref>[http://bunka.nii.ac.jp/SearchDetail.do?heritageId=190749 夷酋列像伝粉本] - 文化遺産オンライン</ref>

== 评价和影响 ==
北方史研究家谷澤尚一认为，本作不是现实性的绘画，有很多创作性的成分；和芹泽銈介美术工艺館所蔵的《阿伊努人物屏風》有6点构图、人物姿势相似之处。<ref>{{Cite book|author=根室シンポジウム実行委員会・編|year=1990|title=三十七本のイナウ|publisher=北海道出版企画センター}}</ref>

[[北海道新干线|北海道新干线]][[新函館北斗車站|新函館北斗車站]]站内，有当地扶輪社制作的《夷酋列像》的大型展示陶板壁画。<ref>{{Cite news|url=http://dd.hokkaido-np.co.jp/news/area/donan/1-0240762.html|title=夷酋列像の壁画、新函館北斗駅に　通路に完成、除幕式|newspaper =[[北海道新聞|北海道新聞]]|date=2016-03-01|accessdate=2016-03-04}}</ref>
