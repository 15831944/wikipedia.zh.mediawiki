'''存貨'''（{{Lang-en|inventory}}）的認定與該[[公司|公司]]之營業範圍有關，凡[[企業|企業]]-{zh-hans:为;zh-hant:爲;}-了營業活動所[[購買|購買]]或[[製造|製造]]的[[商品|商品]]，才能計入存貨科目中，否則必須計入其他的[[資產|資產]]科目中。

== 存貨的評價 ==
存貨的評價係指正確且合理的評估存貨價值。唯有適當的存貨評價，才能在[[資產負債表|資產負債表]]中正確反映存貨的資產[[價值|價值]]，並在[[損益表|損益表]]上正確表達[[損益|損益]][[數字|數字]]。存貨的價值，首先需先知道存貨的[[數量|數量]]及[[單位|單位]][[成本|成本]]，將兩者加以配合[[計算|計算]]，即可得到存貨的總[[價值|價值]]。

== 存貨[[成本|成本]]的決定 ==
存貨成本是指存貨到達可供銷售的狀況下，一切合理且必要的支出。有4種處理方式：
# 個別認定法(但若採用此法,較容易有操控損益金額的機會)
# 先進先出法
# 後進先出法(但是現在財務報表已經不適用"後進先出法"，因無法可靠表達存貨的實際流動,資產負債表帳面金額也偏離成本水準,所以IAS2不再允許使後進先出法)
# 平均法

== 存貨的兩個重要指标 ==
'''存貨周轉率'''和'''存貨周轉天數'''是衡量企業存貨管理水平高低的兩個重要指標。這兩個指標直接影響了企業的短期償債能力，是企業管理的非常重要的內容。通常，存貨周轉速度越快，說明企業的存貨管理能力越高，這樣，企業的變現能力和盈利能力也隨之提高。

== 持有存貨的原因 ==
因為持有存貨可能需要花費成本且管理不易，因此存貨控制需考量以下這些因素
# 許多情況存在顯著的不確定性:像是供應的數量與品質、供應商的成本以及運送時間等
# 前置時間:即使沒有發生需求或供應上的不確定性，卻會由於運送的前置時間而必須持有存貨
# 運輸公司提供的規模經濟:鼓勵公司運輸大量的品項，因而持有較多的存貨。
# 顧客需求的非預期改變:顧客需求一直都是難以預測的。近幾年來，顧客需求的不確定性由以下原因而增加
* 愈來愈多產品具有較短的產品生命週期，這也表示顧客需求的歷史資料無法取得或有所限制
* 市場上存在許多競爭產品。由於產品項目的多樣化，要預測特定產品的需求變得更加困難。
