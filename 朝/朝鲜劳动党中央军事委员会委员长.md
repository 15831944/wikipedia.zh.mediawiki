{{朝鲜政治}}

'''朝鲜劳动党中央军事委员会委员长'''（{{kor|k=조선로동당 중앙군사위원 위원장|hanja=朝鮮勞動黨中央軍事委員會 委員長}}））是[[朝鲜劳动党|朝鲜劳动党]]的最高军事机关首长，根据《[[朝鲜劳动党章程|朝鲜劳动党章程]]》的规定，朝鲜劳动党中央军事委员会统帅[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]所有[[朝鮮人民軍|武装力量]]，指导所有国防事业，为常设最高军事机关<ref name=zhangcheng>{{cite web |url=http://www.wyzxsx.com/Article/Class10/200908/99336.html |title=存档副本 |accessdate=2011-01-30 |deadurl=yes |archiveurl=https://web.archive.org/web/20101006134621/http://www.wyzxsx.com/Article/Class10/200908/99336.html |archivedate=2010-10-06 }}</ref>，相当于朝鲜军队的最高统帅。领导人为中央军事委员会委员长，根据现行《朝鲜劳动党章程》第22条，[[委员长|委员长]]由[[朝鲜劳动党委员长|朝鲜劳动党委员长]]担任，现任委员长为[[朝鲜最高领导人|朝鲜最高领导人]][[金正恩|金正恩]]。

== 中央軍事委員會 ==
{{Main|朝鲜劳动党中央军事委员会}}

[[朝鮮勞動黨中央軍事委員會|朝鮮勞動黨中央軍事委員會]]是[[朝鮮勞動黨|朝鮮勞動黨]]的最高軍事機關，根據《[[朝鮮勞動黨章程|朝鮮勞動黨章程]]》的規定統帥著[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]所有武裝力量。<ref>{{Cite web|url=http://www.wyzxsx.com/Article/Class10/200908/99336.html|title=朝鲜劳动党章程|author=[[朝鲜劳动党|朝鲜劳动党]]|date=2009年8月16日|publisher=[[烏有之鄉|烏有之鄉]]|language=|accessdate=2011年1月30日|quote=|deadurl=yes|archiveurl=https://web.archive.org/web/20101006134621/http://www.wyzxsx.com/Article/Class10/200908/99336.html|archivedate=2010年10月6日}}{{zh-cn}}</ref> 在功能和特點上，都近似於[[中國共產黨|中國共產黨]]的[[中央軍事委員會|中央軍事委員會]]；同時，[[朝鲜國防委員會|朝鲜國防委員會]]的委員也有兼任该委员会成员者。

最早，由[[朝鲜劳动党中央委员会总书记|朝鲜劳动党中央委员会总书记]][[金日成|金日成]]當選了第一任的中央軍事委員會委員長。隨著他的逝世，其長子[[金正日|金正日]]接任黨總書記時兼任軍委委員長職務。[[金正恩|金正恩]]曾經被任命為中央軍事委員會副委員長，確認為[[金正日|金正日]]的接班人，金正恩掌權以後中央軍委的權力大幅擴張，甚至成為勞動黨的權力核心。目前該委員會委員長由[[朝鲜劳动党第一书记|朝鲜劳动党第一书记]][[金正恩|金正恩]]兼任，他在父親金正日逝世後接任。而副委員長一職为[[黄炳誓|黄炳誓]]次帥、[[李永吉|李永吉]]大将。
== 现任组成人员 ==
[[朝鲜最高领导人|朝鲜最高领导人]]、[[朝鲜劳动党委员长|朝鲜劳动党委员长]]、朝鲜劳动党中央军事委员会委员长[[金正恩|金正恩]]元帅

2010年9月28日举行的[[朝鲜劳动党中央委员会2010年9月全体会议|朝鲜劳动党中央委员会2010年9月全体会议]]选出了新一届朝鲜劳动党中央军事委员会。此次全会选举[[金正日|金正日]]为朝鲜劳动党中央军事委员会委员长，[[金正恩|金正恩]]、[[李英浩|李英浩]]为副委员长，还选举出[[金永春|金永春]]、[[金正阁|金正阁]]、[[金明国|金明国]]、[[张成泽|张成泽]]等16名朝鲜劳动党中央军事委员会委员。<ref name="xj">"[http://news.xinhuanet.com/world/2010-09/29/c_12619212.htm 朝鲜劳动党代表会议选举产生劳动党最高领导机构]，新华网，2010年9月29日</ref>

2012年4月11日举行的[[朝鲜劳动党第四次代表会议|朝鲜劳动党第四次代表会议]]上又选举[[崔龙海|崔龙海]]为副委员长，[[玄哲海|玄哲海]]、[[李明洙|李明洙]]、[[金乐谦|金乐谦]]为委员。

2012年7月15日[[朝鲜劳动党中央政治局|朝鲜劳动党中央政治局]]会议上，中央军委副委员长[[李英浩|李英浩]]被以健康原因为由罢免一切职务。2013年12月8日朝鲜劳动党中央政治局扩大会议上，中央军委委员、国防委员会副委员长张成泽被以反党反革命为由罢免一切职务。

2014年4月朝鲜劳动党中央军事委员会扩大会议举行，会议后，崔龙海不再担任朝鲜人民军总政治局局长职务，由[[黄炳誓|黄炳誓]]接任。
