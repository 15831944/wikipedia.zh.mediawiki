[[File:Funeral_for_Sunjong.JPG|right]]
'''朝鮮純宗'''（{{kor|k=순종|hanja=純宗|rm=Sunjong}}；{{bd|1874年|2月8日|1926年|4月25日|catIdx=L李坧}}）姓'''李'''，諱'''坧'''（{{kor|k=이척|hanja=李坧|rm=I Cheok}}），字'''君邦'''，號'''正軒'''，是[[朝鮮高宗|朝鮮高宗]]李㷩的兒子、[[大韓帝國|大韓帝國]]第2代[[皇帝|皇帝]]、[[朝鮮王朝|朝鮮王朝]]第27代（末代）君主，於1907年7月20日－1910年8月29日期間在位。年號[[隆熙|隆熙]]，所以又名'''隆熙皇帝'''（{{kor|k=융희제|hanja=隆熙帝|rm=Yungheeje}}），[[谥号|谥号]]'''文溫武寧敦仁誠敬孝皇帝'''<ref>《순종문온무녕돈인성경효황제실록》（《純宗文溫武寧敦仁誠敬孝皇帝實錄》）</ref>。

由於二戰結束、日本投降後朝鮮半島的分裂，純宗也是迄今最後一位實際統治整個韓半島的主權國家元首。

== 脈系 ==
[[File:Crown_Prince_Yoshihito_and_Crown_Prince_Lee_Eun_1907.jpg|thumb]]赴韓国親善訪問 皇太子嘉仁親王(右)。中央是韓国皇太子[[李垠|李垠]]。左端韓国皇帝純宗]]
純宗是朝鮮高宗的唯一活到成年的[[嫡子|嫡子]]。他的母親是[[明成皇后|明成皇后]]（閔妃）。李坧是他们二人之间仅存的儿子。高宗15岁大婚，王妃闵妃。但高宗迷恋尚宫[[李順娥|李順娥]]，李尚宫于[[同治|同治]]七年（1868年）首先生子[[完親王|李墡]]。闵妃先后怀孕三次，[[流产|流产]]一次，夭折两次，因此高宗生父[[兴宣大院君|兴宣大院君]]对闵妃的生育能力产生怀疑，主张立永宝堂李氏（李尚宫）之子完和君李墡为世子。闵妃联合自己家族的[[闵升镐|闵升镐]]、[[閔謙鎬|閔謙鎬]]及其他反大院君势力，于同治十二年（1873年）十一月发动[[政变|政变]]，将[[大院君|大院君]]赶下台。闵妃集团掌握实权。李坧出生后闵妃要求立刻封李坧为世子。[[神貞王后|神貞王后]]不满闵妃单独掌权，主张立高宗庶长子李墡为世子。高宗支持闵妃，[[宗主国|宗主国]][[清朝|清朝]]起先支持神贞王后，闵妃以族人[[闵奎镐|闵奎镐]]为礼曹判书，负责对清外交，又急忙派人赴[[大清|大清]]巴结[[李鸿章|李鸿章]]等，甚至与东莱倭馆接洽，透过倭馆贿赂[[日本|日本]]驻华[[公使|公使]]，使其从中斡旋。<ref>尹孝定《风云韩末秘史》，第52—87页。</ref>用时一年多，糜费百万金。[[光绪|光绪]]元年（1875年）[[正月|正月]]初一，高宗决定册封不满周岁的李坧为世子，李坧成为[[朝鲜王朝|朝鲜王朝]]受封年龄最小的世子。正月初七，派[[李裕元|李裕元]]为世子册封使。二月十九日，在[[昌德宮|昌德宮]]仁政殿举行册封仪式。八月，李裕元启程前往[[清國|清國]]。次年正月，清朝派前[[沈阳市|盛京]][[户部|户部]][[侍郎|侍郎]][[吉和|吉和]]、内阁学士[[烏拉喜崇阿|烏拉喜崇阿]]为正副敕使前往朝鲜，册封李坧为王世子。<ref>《清史稿·德宗本纪》</ref>

== 幼年 ==
李坧出生后不久，高宗就特地下令以“增广庆科”的名义举行[[科举|科举]]考试，以示庆祝。李坧经常生病，闵妃除了召集御医进行医治以外，还遍祭[[金剛山_(朝鮮)|金刚山]]一万二千峰，各峰施以米一石、布一匹、清钱千两，希望世子平安。<ref>朴殷植《韩国痛史》第二编，第27页</ref>又招来大批僧道、巫祝为世子作法祈福。<ref>黄玹．《梅泉野录》：韩国国史编纂委会，1955年：第46页</ref>李坧出生后四年间，闵妃又生下二子，皆夭折。光绪八年（1882年）正月，李坧入学，行[[冠禮|冠禮]]，闵妃命[[外戚|外戚]]闵台镐为博士，担任世子的首席老师。<ref>《高宗实录》卷19，十九年正月十一日</ref>又内定闵台镐之女为世子嫔，并于正月二十六日正式决定。二月二十一日，李坧与闵氏行[[嘉礼|嘉礼]]于安国洞新宫。该年六月，爆发[[壬午兵变|壬午兵变]]，闵妃将李坧转移到[[景祐宫|景祐宫]]，自己化装成宫女逃跑。<ref>《高宗实录》卷19，十九年六月十日</ref>兴宣大院君宣布闵妃已死，世子“至恸无穷”。<ref>《高宗实录》卷19，十九年六月十四日</ref>后来清军干涉，大院君被抓到清國，闵妃回到宫廷。世子亲赴宫门迎接。<ref>《高宗实录》卷19，十九年七月二十六日</ref>高宗对世子非常溺爱，“每饭，拣馔而食之；每衣，张袖而穿之”，闵妃经常在世子犯错时打骂世子，呵曰：“汝虽世子，岂无父母！”世子因此不畏惧高宗而畏惧闵妃。<ref>黄玹．《梅泉野录》：韩国国史编纂委会，1955年：第95页</ref>

== 后代 ==
純宗[[勃起功能障碍|阳痿]]，不能人道，无后。<ref>[[黃玹_(朝鮮)|黃玹]]《梅泉野录》，第95页：“世子患阴萎，或曰天阉，或曰幼时宫女吮其茎，一出，不收也。年稍长，茎垂如苽，无举时。小便自遗，常淋沥溅席，一日一易褥，再易袴。大婚有年，不能有人道。明成懊恨狂躁，尝属宫婢嬲世子，教交媾状，自户外高声问曰：‘谐否？’对曰：‘不谐！’明成噫吁数息，槌胸而起。时人以为，此戕完和君之报。”</ref>

== 性格 ==
[[File:Sunjong_of_the_Korean_Empire_02.jpg|250px]]和[[大勳位菊花章頸飾|大勳位菊花章頸飾]]]]
有记载“世子记性绝人，凡朝章、国故、山川、关防、钱谷、甲兵之簿，以及士夫门阀、谱牒、科宦年月，随事胪列，历历不差”，性格开朗活泼，喜欢游戏。由于他聪明绝伦，东宫侍臣们根本比不上他，所以他经常骑着东宫侍臣作赶牛状，引来宫中大笑。<ref>黄玹．《梅泉野录》：韩国国史编纂委会，1955年：第116页</ref>李坧经历了[[乙未事变|乙未事变]]，朝鲜人禹范善参与刺杀闵妃，在闯入坤宁阁杀害闵妃时，日本人控制了朝鲜高宗和世子。高宗在乾清宫长安堂见日本人入侵，想吸引日本人注意，以便闵妃逃脱，便打开长安堂中所有窗户，自己站在前面，一些日本人便进入长安堂，有个日本人扯住高宗的肩臂，有日本人在高宗身边开枪，殴打宫女。李坧被日本人捉住，“牵捽头发，脱毁其冠，拟刃迫问王后处所，而幸不被伤”。<ref>《开国五百四年八月事变报告书》第6页下 韩国学中央研究院</ref>高宗和李坧吓得面如死灰，战战兢兢。李坧受到刺激，经常大哭，并不停地呼喊着妈妈。每逢此时，身边的宫女侍从无不泪流满面。从此性情大变，由活泼开朗变得沉默寡言。据说八年后在日本暗杀禹范善的高永根就是他委派的。

1919年高宗去世，传说被[[日本|日本]]人毒死，他无力报仇，变得异常迷信，他经常到高宗生前的寝宫和高宗、闵妃死后归葬的洪陵，面对空明长诵不已。后来王宫里有了[[电话|电话]]，他便命人架设了通往高宗昔日居所和陵区的专线电话，每逢他因身体不适不能前往时，他便令侍从接通电话，令对方接听者把话筒对着被认为是高宗魂灵寄居之所，然后着素衣，正襟危坐对着话筒轻声诉说。

== 年表 ==
1874年朝鮮純宗出生。

1875年獲[[清朝|清朝]]冊封為王世子。

1894年改稱王太子。

1896年2月11日，随高宗乘妇女轿子逃出日本人控制的宫廷，[[俄館播遷|俄館播遷]]。高宗下令逮捕亲日派[[金弘集_(朝鮮)|金弘集]]等人，任用[[李范晋|李范晋]]、[[李完用|李完用]]等亲俄派。向俄出卖利益。

1897年2月20日，高宗搬出俄国公使馆，偕李坧到庆运宫（今德寿宫）。8月，改年号为光武，10月12日，李熙称帝，改国号大韩帝国。李坧封皇太子。

1898年6月29日开大元帅府，高宗任大元帅，统辖陆海各军，李坧为陆海军元帅。9月13日，茶毒事件，遭[[金鸿陆|金鸿陆]]、厨师孔洪植和金钟浩毒杀未遂，昏厥，摔断两颗门牙损害了他脑神经和身体。

1907年登基。

1910年被日本廢為'''昌德宮李王'''，陆军大将

1926年純宗病逝。安葬在[[京畿道|京畿道]]近郊[[南楊州市|南楊州市]][[金谷洞|金谷]]的[[洪裕陵|裕陵]]。

== 纯祖大王 ==
{{main|朝鮮純祖}}
在朝鮮歷史上曾有另外一位純宗大王[[朝鮮純祖|李玜]]。後來因為純宗大王的旁支即位，所以被追封為純祖。
