'''朝日啤酒''' （{{lang-ja|アサヒビール株式会社}}，{{lang-en|Asahi Breweries, Ltd.}}）是一間釀製[[啤酒|啤酒]]和生產[[軟飲料|軟飲料]]的公司，和[[麒麟啤酒|麒麟啤酒]]、[[三得利|三得利]]、[[札幌啤酒|札幌啤酒]]並列為日本啤酒界四大企業。2016年，朝日啤酒佔據日本啤酒市場39%的份額，位居各企業之首<ref>{{cite web |url=http://www.nikkei.com/article/DGXLASDZ16IPC_W7A110C1TI5000/ |script-title=ja:ビール系飲料、首位アサヒのシェア上昇　16年 |accessdate=2017-06-04 |publisher={{lang|ja|日本経済新聞}} |language=ja}}</ref>。朝日啤酒是在1949年{{link-ja|大日本麥酒|大日本麦酒}}分割過程中成立的企業，最初的業務範圍主要是[[西日本|西日本]]。朝日啤酒在1954年至1960年期間曾是日本啤酒業界第二大企業，但在1960年代之後持續低迷，1980年代前期更幾乎淪為日本啤酒業界第四大企業。然而在公司內改革和{{link-ja|朝日超爽啤酒|アサヒスーパードライ}}熱銷的幫助下，朝日啤酒在短時間內迅速實現業績回升，在1988年奪回業界第二的位置，並在2001年獲得業界首位（包含[[發泡酒|發泡酒]]）。朝日啤酒的企業標語是「分享感動」（{{lang|ja|その感動を、わかちあう。}}）<ref>{{cite web |url=http://www.asahigroup-holdings.com/company/summary/ |script-title=ja:経営理念・方針 |accessdate=2017-06-04 |publisher={{lang|ja|アサヒグループ}} |language=ja}}</ref>。

自2011年7月開始，朝日啤酒是{{link-ja|朝日集團控股|アサヒグループホールディングス}}的全資子公司。朝日集團控股亦繼承了朝日啤酒{{tyo|2502}}的股票代碼<ref name="company">{{cite web |url=http://www.asahigroup-holdings.com/company/history/ |script-title=ja:歴史・沿革 |accessdate=2017-06-04 |publisher={{lang|ja|アサヒグループ}} |language=ja}}</ref>。

==歷史==
[[File:Asahi_Beer.jpg|thumb]]
朝日啤酒的前身是創業於1889年的大阪麥酒。1891年，大阪麥酒在[[吹田市|吹田]]設廠並開始生產名為「朝日」的啤酒品牌<ref name="Asahi">{{cite book |author={{lang|ja|アサヒビール株式会社社史資料室}}|script-title=ja:Asahi 100 |year=1990 |publisher={{lang|ja|アサヒビール}} |location=東京都 |language=ja}}</ref>{{rp|90}}，選擇吹田作為工廠的理由是因為這裡水源豐富且接近[[京阪神|京阪神]]這一巨大的消費市場<ref name="Asahi"></ref>{{rp|120}}。大阪麥酒成立之後發展迅速。1901年時，朝日啤酒的生產量已是1891年的近25倍<ref name="Asahi"></ref>{{rp|145}}。20世紀初期，日本啤酒市場呈現朝日啤酒、[[札幌啤酒|札幌啤酒]]、[[惠比壽啤酒|惠比壽啤酒]]三大品牌爭霸的局面<ref name="Asahi"></ref>{{rp|147}}。為緩和過激的競爭，大阪麥酒、札幌麥酒（札幌啤酒的生產企業）、日本麥酒（惠比壽啤酒）的生產企業在1906年合併為大日本麥酒<ref name="Asahi"></ref>{{rp|152}}。1930年代中期時，大日本麥酒佔據日本啤酒市場三分之二的市場份額，其中朝日啤酒又是大日本麥酒銷量最好的品牌<ref name="Asahi"></ref>{{rp|224}}。

二戰之後，盟軍佔領當局實施[[財閥解體|財閥解體]]政策，大日本麥酒也受這一政策的影響被拆解為以西日本為銷售地區的朝日啤酒和以東日本為銷售地區的札幌啤酒。大日本麥酒的山本為三郎專務就任朝日啤酒首任社長<ref name="Asahi"></ref>{{rp|200}}。由於戰前大日本麥酒主要設備投資集中在西日本，因此可以說朝日啤酒繼承了大日本麥酒的精華部分。大日本麥酒分割時，日本啤酒（現札幌啤酒）的市佔率是38.7%、朝日啤酒是36.1%、[[麒麟啤酒|麒麟啤酒]]是25.3%，朝日啤酒位居次席<ref name="dai7kai">{{cite web |url=http://www2.umds.ac.jp/sem/SemNShimizu/shimizu1996_asahi.pdf |script-title=ja:アサヒビール 〜ビール市場の創造と適応〜 |accessdate=2017-06-04 |publisher={{lang|ja|現代経営学研究学会 第7回シンポジウム資料}} |language=ja}}</ref>。此後至1960年，除了1953年因原料配給導致三家公司市場份額相同之外，朝日啤酒一直都佔據日本啤酒市場第二把交椅<ref name="dai7kai"></ref>。

然而隨著[[東京一極集中|東京一極集中]]的進行，以西日本為主要市場的朝日啤酒的市場份額也逐漸下降。1961年後，朝日啤酒的市場份額降至第三位，並且直到1987年都穩定在這一位置<ref name="dai7kai"></ref>。1985年時，朝日啤酒的市場份額下降到只有9.6%，同年三得利的市場份額有9.2%。朝日啤酒一度被諷為「夕日啤酒」，這一局面使得公司內充滿危機感<ref name="dai7kai"></ref>。為改變這一局面，朝日啤酒決定邀請在短時間內成功重建[[馬自達|馬自達]]的{{link-ja|村井勉|村井勉}}擔任社長<ref name="dai7kai"></ref>。村井在上任之後實施大規模改革。為刷新品牌形象，朝日啤酒自1986年將商標改為現行商標<ref name="Asahi"></ref>{{rp|66}}，並於同年開始銷售新口味新包裝的生啤酒<ref name="Asahi"></ref>{{rp|68}}，朝日啤酒的市佔率開始回升<ref name="dai7kai"></ref>。1987年3月，朝日啤酒開始發售朝日超爽啤酒（{{lang|ja|アサヒスーパードライ}}）<ref name="Asahi"></ref>{{rp|72}}。雖然朝日超爽啤酒的銷售區域最初只限[[首都圈_(日本)|首都圈]]，但因銷量遠超預計，因此朝日啤酒決定將原本自夏季開始在全國銷售的計劃提前到5月<ref name="dai7kai"></ref>。1987年，朝日超爽啤酒的銷量達1350萬箱<ref name="dai7kai"></ref>。超爽啤酒的成功亦導致其他公司爭相開始類似產品，引發「爽啤戰爭」<ref name="Asahi"></ref>{{rp|74}}。朝日啤酒的市佔率因超爽啤酒的成功而快速回升，1988年時已恢復為日本啤酒業界第二大企業，還開始對外國出口超級爽啤<ref name="Asahi"></ref>{{rp|76}}。1998年，朝日啤酒成為日本啤酒市場最大企業<ref name="dai7kai"></ref>。

2000年之後，朝日啤酒也開始生產銷售低價的[[發泡酒|發泡酒]]產品。2001年，朝日啤酒成為包含[[發泡酒|發泡酒]]在內的啤酒業界最大企業<ref name="history">{{cite web |url=https://www.asahibeer.co.jp/saiyo/company/history/ |script-title=ja:アサヒビールの歴史 |accessdate=2017-06-04 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>。2005年後，朝日啤酒又進一步開始生產更廉價的{{link-ja|第三啤酒|第三のビール}}<ref>{{cite web |url=http://www.asahibeer.co.jp/ir/event/pdf/presentation/2008_in_factory.pdf |script-title=ja:アサヒビールグループの概要と事業方針 |accessdate=2017-06-04 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>。除了啤酒本業之外，朝日啤酒還在2001年將{{link-ja|NIKKA威士忌|ニッカウヰスキー}}收購為全資子公司，開始進出洋酒市場<ref name="company"></ref>。朝日啤酒還積極開拓海外市場。2009年，朝日啤酒收購了澳大利亞第二大飲料企業{{link-en|澳大利亞玉泉|Schweppes Australia}}<ref>{{cite web |url=https://www.asahibeer.co.jp/ir/event/kessan/2009_h/html/16.html |script-title=ja:シュウェップス・オーストラリアの買収 |accessdate=2017-06-04 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>。2016年，朝日啤酒以25.5億歐元收購[[南非米勒|南非米勒]]旗下的{{link-en|佩羅尼啤酒|Peroni Brewery}}、{{link-en|葛蘭斯啤酒|Grolsch Brewery}}、{{link-en|Meantime啤酒|Meantime Brewery}}三個品牌<ref>{{cite web |url=http://jp.reuters.com/article/asahi-mrg-idJPKCN0VJ1PJ |script-title=ja:アサヒが欧州ビールブランド買収で基本合意、海外市場強化狙う |accessdate=2017-06-04 |publisher={{lang|ja|ロイター}} |language=ja}}</ref>。

== 主要產品 ==
朝日啤酒最大的招牌商品是朝日超爽啤酒。超爽啤酒是日本唯一每年銷量超過一億箱的啤酒品牌，佔據日本啤酒市場（僅限酒稅上的「啤酒」，不含發泡酒和第三啤酒）的半數份額，也佔朝日啤酒近六成的收入<ref>{{cite web |url=http://toyokeizai.net/articles/-/77303 |script-title=ja:アサヒを支える｢スーパードライ｣に異変発生 |accessdate=2017-06-05 |publisher={{lang|ja|東洋経済オンライン}} |language=ja}}</ref>。超爽啤酒通過改良酵母和發酵技術以提高發酵度降低、降低糖度、提高麥芽之外的原料比例等方法實現清爽口感<ref>{{cite web |url=http://www.asahibeer.co.jp/superdry/philosophy/#/philosophy_01/ |script-title=ja:うまさの原点 |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>。自1997年開始，朝日超爽啤酒就是日本銷量最多的啤酒品牌<ref>{{cite web |url=https://allabout.co.jp/gm/gc/205886/2/ |script-title=ja:キリンラガーはどうしてYMOを起用？ |accessdate=2017-06-05 |publisher={{lang|ja|All About}} |language=ja}}</ref>。朝日啤酒還銷售有超爽黑啤<ref>{{cite web |url=http://www.asahibeer.co.jp/products/beer/dryblack/ |script-title=ja:アサヒスーパードライ ドライブラック |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>、豐釀超爽啤酒<ref>{{cite web |url=http://www.asahibeer.co.jp/products/beer/drypremiumhojo/ |script-title=ja:アサヒドライプレミアム豊醸  |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>等較普通超爽啤酒更為高檔的清爽口感啤酒。自1935年開始，朝日啤酒就生產銷售[[司陶特啤酒|司陶特啤酒]]，現在和[[麒麟啤酒|麒麟啤酒]]並列為日本僅有的兩個生產司陶特啤酒的企業<ref>{{cite web |url=http://www.asahibeer.co.jp/products/beer/stout/ |script-title=ja:アサヒスタウト  |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>。

朝日啤酒的主要[[發泡酒|發泡酒]]產品是Style Free，產品特徵是低糖低卡<ref>{{cite web |url=http://www.asahibeer.co.jp/products/beer/stylefree/ |script-title=ja:アサヒ スタイルフリー  |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>。在啤酒和發泡酒以外的領域，朝日啤酒生產有Clear Asahi<ref>{{cite web |url=http://www.asahibeer.co.jp/products/beer/clear/ |script-title=ja:クリア アサヒ  |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>
、Asahi Off<ref>{{cite web |url=http://www.asahibeer.co.jp/products/beer/off/ |script-title=ja:アサヒ オフ |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>等啤酒類飲品。朝日DRY ZERO在2016年是日本銷量最高的無酒精啤酒口味飲品<ref>{{cite web |url=http://www.asahibeer.co.jp/products/beer/dryzero/ |script-title=ja:アサヒ ドライゼロ |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>。

[[沖繩縣|沖繩縣]]最大啤酒企業[[獵戶座啤酒|獵戶座啤酒]]和朝日啤酒關係密切，其產品在日本本土的啤酒銷售委託由朝日啤酒進行<ref>{{cite web |url=http://www.asahibeer.co.jp/products/beer/orion/ |script-title=ja:アサヒオリオンドラフト |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>。兩者亦有共同研發啤酒產品<ref>{{cite web |url=http://www.asahibeer.co.jp/products/beer/orion_chura/ |script-title=ja:アサヒ　オリオンちゅらたいむ |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>。

朝日啤酒亦莊日本國內銷售[[豪格登白啤酒|豪格登白啤酒]]<ref>{{cite web |url=http://www.asahibeer.co.jp/products/beer/hoegaarden/ |script-title=ja:ヒューガルデンホワイト |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>、{{link-en|盧雲堡啤酒|Löwenbräu}}<ref>{{cite web |url=http://www.asahibeer.co.jp/products/beer/lowenbrau/ |script-title=ja:レーベンブロイ |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>、{{link-en|巴斯啤酒|Bass Brewery}}<ref>{{cite web |url=http://www.asahibeer.co.jp/worldbeer/bass/ |script-title=ja:バスペールエール |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>等外國品牌啤酒。

除了啤酒之外，朝日啤酒亦有生產銷售[[日本燒酎|日本燒酎]]<ref>{{cite web |url=https://www.asahibeer.co.jp/enjoy/shochu/index.html#item |script-title=ja:焼酎ラインアップ |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>、[[雞尾酒|雞尾酒]]<ref>{{cite web |url=http://www.asahibeer.co.jp/products/sour/cocktailpartner/ |script-title=ja:アサヒカクテルパートナー |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>、水果味酒類<ref>{{cite web |url=http://www.asahibeer.co.jp/products/sour/kajitsunoshunkan/ |script-title=ja:アサヒ チューハイ 果実の瞬間 |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref><ref>{{cite web |url=http://www.asahibeer.co.jp/products/sour/slat/ |script-title=ja:アサヒSlat |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>、[[可爾必思|可爾必思]]口味酒類<ref>{{cite web |url=http://www.asahibeer.co.jp/products/sour/calpissour/ |script-title=ja:「カルピスサワー」 |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>等商品。

==總部大樓及生產工廠==
[[File:Asahi_Breweries_Headquarters_(derivative_image).jpg|thumb]]
朝日啤酒的總部大樓位於東京都墨田区吾妻橋一丁目，竣工於1989年，高22層，原為舊吾妻橋工廠。大樓外觀為琥珀色玻璃幕牆，頂端為白色玻璃幕牆，系以啤酒杯為概念設計<ref>{{cite web |url=https://www.asahibeer.co.jp/aboutus/summary/ |script-title=ja:経営理念・会社概要 |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>。大樓的第21層、第22層設有餐廳和咖啡廳，開放普通民眾使用<ref>{{cite web |url=https://www.asahibeer.co.jp/restaurant/azuma/access.html |script-title=ja:アクセス |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>。

朝日啤酒總部大樓的隔壁建築是{{link-ja|超爽大廳|スーパードライホール}}，由法國設計師[[菲利普·斯塔克|菲利普·斯塔克]]設計。該建築最大的特徵是頂部巨大的金黃色雕塑，象徵朝日啤酒迎接新世紀的躍動之心有如火焰，而底部的建築有如聖火台<ref>{{cite web |url=http://www.asahibeer.co.jp/customer/post-5.html |script-title=ja:浅草のアサヒビール本社ビルのとなりにある金色のオブジェは何を表わしているの？ |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>。然而因其獨特的外形和色彩，超爽大廳也被稱為「糞便大樓」(うんこビル)<ref>{{cite web |url=http://asakusanews.com/erudition/174 |script-title=ja:浅草 アサヒビール うんこビル？ |accessdate=2017-06-05 |publisher={{lang|ja|浅草smartニュース}} |language=ja}}</ref>。

朝日啤酒在日本全國有八個工廠，包括吹田工廠（1891年投產，位於[[大阪府|大阪府]][[吹田市|吹田市]]）、博多工廠（1921年投產。位於[[福岡市|福岡市]][[博多區|博多區]]）、北海道工廠（1966年投產。位於[[札幌市|札幌市]][[白石區|白石區]]）、名古屋工廠（1973年投產，位於[[名古屋市|名古屋市]][[守山區|守山區]]）、福島工廠（1979年投產，位於[[福島縣|福島縣]][[本宮市|本宮市]]）、茨城工廠（1991年投產，位於[[茨城縣|茨城縣]][[守谷市|守谷市]]）、四國工廠（1998年投產，位於[[愛媛縣|愛媛縣]][[西條市|西條市]]）、神奈川工廠（2002年投產，位於[[神奈川縣|神奈川縣]][[南足柄市|南足柄市]]）<ref>{{cite web |url=https://www.asahibeer.co.jp/aboutus/branch/ |script-title=ja:事業場一覧 |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>。

==文化體育事業==
朝日啤酒在[[京都府|京都府]][[乙訓郡|乙訓郡]][[大山崎町|大山崎町]]贊助設有{{link-ja|朝日啤酒大山崎山莊美術館|アサヒビール大山崎山荘美術館}}，開館於1996年，擁有約1,000件藏品，其中包括[[莫奈|莫奈]]的[[睡蓮_(莫奈)|睡蓮]]等作品<ref>{{cite web |url=http://www.asahibeer-oyamazaki.com/collection/#cat1 |script-title=ja:所蔵品 |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール大山崎山荘美術館}} |language=ja}}</ref>。朝日啤酒有自己的美式足球球隊<ref>{{cite web |url=http://www.asahibeer.co.jp/silver/teamprofile/ |script-title=ja:チームプロフィール |accessdate=2017-06-05 |publisher={{lang|ja|アサヒビール}} |language=ja}}</ref>，並且是日本棒球國家隊的官方贊助企業之一<ref>{{cite web |url=http://www.japan-baseball.jp/jp/news/press/20150119_1.html |script-title=ja:アサヒビール株式会社が侍ジャパン「オフィシャルパートナー」に決定 |accessdate=2017-06-05 |publisher={{lang|ja|野球日本代表}} |language=ja}}</ref>。
