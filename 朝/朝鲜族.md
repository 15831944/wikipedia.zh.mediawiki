'''朝鮮民族'''（{{kor|k=조선민족|hanja=朝鮮民族}}），又稱'''韓民族'''（{{kor|k=한민족|hanja=韓民族}}）、'''高麗人'''（{{kor|k=고려인|hanja=高麗人}}）等。主要分布在[[朝鲜半岛|朝鲜半岛]]及临近的[[中国东北|中国东北]]和[[俄罗斯远东地区|俄罗斯远东地区]]等地区，是[[朝鮮|-{zh-hans:朝鲜; zh-hant:朝鮮;}-]]和[[大韩民国|-{zh-hans:韩国; zh-hant:韓國}-]]的主体民族，两国共有八千多万人口的朝鲜民族人。

其它朝鲜民族人口过百万的国家有[[中華人民共和國|中華人民共和國]]和[[美国|美国]]，人口均在两百万以上。在[[中華人民共和國|中華人民共和國]]境內的-{zh-hans:朝鲜族; zh-hant:朝鮮族;}-是中國官方认定的55個[[少數民族|少數民族]]之一，也被称为[[中国朝鲜族|中国朝鲜族]]。在[[美國|美國]]境內的朝鲜族属于美國[[亚裔美国人|亚裔]][[少數民族|少數民族]]，被称为[[韩裔美国人|韩裔美国人]]或[[美籍韓裔|美籍韓裔]]。

==历史==
[[韩民族|韩民族]]（한민족）或者是[[朝鲜民族|朝鲜民族]]（조선민족）是古亞洲人和新亚洲人在[[新石器时代|新石器时代]]到[[青铜器时代|青铜器时代]]之间，陆续从[[欧亚大陆|欧亚大陆]]迁徙到[[朝鲜半岛|朝鲜半岛]]后形成的[[扶餘語系|扶餘語系]]民族。<ref>{{Cite web |url=http://d.wanfangdata.com.cn/Periodical_djxk200303007.aspx |title=朝鲜/韩国语和阿尔泰诸语言 |access-date=2009-09-16 |archive-url=https://web.archive.org/web/20110322221151/http://d.wanfangdata.com.cn/Periodical_djxk200303007.aspx |archive-date=2011-03-22 |dead-url=no }}</ref><ref>{{cite book |last = Nelson Sk |first = Sarah M. |title = The Archaeology of Korea |publisher = Cambridge University Press |year = 1993 |pages = 6 |isbn = 0521407834}}</ref>。朝鲜半岛也是除欧洲大陆以外，[[青铜器时代|青铜器时代]]人类遗迹[[支石墓|支石墓]]被发现最多的地区。

公元668年，新罗结束[[朝鲜三国时代|朝鲜半岛三国时代]]，与[[渤海国|渤海国]][[南北国时代|并立]]，开创[[统一新罗|统一新罗时代]]，被部分人认为认为是朝鲜民族第一个统一国家<ref>[http://www.yidudu.com/read_2711_277405.html 亚洲的黎明—新韩风云五十年]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。统一新罗解体后，[[後三國|三国遗民]]曾经一度恢复各自的国家，分别出现了[[后高句丽|后高句丽]]、[[后百济|后百济]]、[[后新罗|后新罗]]，随后[[高麗_（918年－1392年）|高丽]]统一朝鲜半岛，这是今日之[[朝鲜民主主义人民共和国|朝鲜]]政府认定的“朝鲜族历史上第一个统一国家”<ref>[http://www.kcckp.net/ch/periodic/todaykorea/index.php?contents+4541+2009-07+129+23 高丽，朝鲜第一个统一国家]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref> 。 [[蒙古|蒙古]]一度对高丽用兵，此期间大量女真和契丹人遗留在此。 [[蒙元|蒙元]]出现内乱时，高丽曾派兵协助蒙古平叛。高丽大将[[李成桂|李成桂]]受命征伐[[明朝|明]][[辽东|辽东]]，其间[[威化島回軍|回师国内]]，废黜[[高麗禑王|禑王]]，进一步建立[[朝鲜|朝鲜]]。 

与他们最近的民族是[[尼夫赫人|尼夫赫人]]及[[科里亚克族|科里亚克族]]。经过基因、文化、民俗证實，汉族、扶餘、高句丽、百济、新罗都是朝鲜族的直系祖先。现代朝鲜族在19世纪最终基本形成。在这过程中，半岛北部的扶餘、沃沮、东濊、鲜卑、高句丽人、渤海人、女真、契丹及汉族等东北亚和东亚民族也融合进朝鲜民族中。

== 语言和文字 ==
{{main|韩国语}}
[[File:Koreandialect.png|thumb]]}}{{legend|#C19A6B|6. [[濟州語|济州方言]]}}]]

韩语是[[表音文字|表音文字]]，每个字可以根据其构成拼读出来，不需要另外单独的[[拼音|拼音]]系统。韩语共有40个[[字母|字母]]，包括21个[[元音|元音]]和19个[[辅音|辅音]]。元音又分为单元音（10个）和[[双元音|双元音]]（11个）。元音是韩语[[音节|音节]]的中心，辅音只有和元音结合才能构成[[文字|文字]]。韩语是通过附着在单词后面的[[助词|助词]]和[[格_(语法)|语尾的变化]]来表示语法关系，而不是像汉语那样依靠[[词序|词序]]。因此韩语有比较复杂的助词和语尾。韩语的句子结构是[[主语|主语]]+[[宾语|宾语]]+[[谓语|谓语]]，而汉语的句子结构则是主语+谓语+宾语。<ref name=beiyu> {{cite web|title =韩国语简介|url =http://www.eblcu.cn/demo/zhanyu/course/c000.html|publisher =北京语言大学网络教育学院|accessdate =2014-10-31|deadurl =yes|archiveurl =https://web.archive.org/web/20170709090125/http://www.eblcu.cn/demo/zhanyu/course/c000.html|archivedate =2017年7月9日|df =}}</ref>

韩语的[[词汇|词汇]]类型可分为[[固有词|固有词]]、[[汉字词|汉字词]]和[[外来语|外来语]]三种<ref name=jinlin> {{cite web |title = 韩国语词汇来源初探 |url = http://www.jlrtvu.jl.cn/xb/file.asp?fileid=20111040013 |publisher = 吉林广播电视大学 |accessdate = 2014-10-31 |deadurl = yes |archiveurl = https://web.archive.org/web/20170709090151/http://www.jlrtvu.jl.cn/xb/file.asp?fileid=20111040013 |archivedate = 2017年7月9日 |df =  }}</ref>。固有词是朝鲜族自古流传下来的本民族自己固有的词汇，是韩语词汇的核心部分。韩语日常用语绝大多数用的都是固有词<ref name=jinlin />。绝大部分汉字词来源于[[汉语|汉语]]。汉字词读音遵照韩国汉字的读音规则，词义大部分与汉语相同，但也有些差异<ref name=jinlin />。汉字词在数量上要占到韩语词汇一半以上。外来语是指从[[中国|中国]]以外的其他国家借鉴过来的词语，以[[英语|英语]]居多<ref>{{cite web |title = 浅谈韩国语中的《语标记法》 |url = http://wuxizazhi.cnki.net/Search/ZXLJ201412146.html |publisher = 吾喜杂志 |accessdate = 2014-10-31 |archive-url = https://web.archive.org/web/20141031122104/http://wuxizazhi.cnki.net/Search/ZXLJ201412146.html |archive-date = 2014-10-31 |dead-url = no }}</ref>。

西元15世纪以前，韩语只有语言没有文字，以[[汉字|汉字]]为书写工具<ref>{{cite web |title = 現代韓國文字的起源訓民正音 |url = http://www.newsancai.com/big5/history/120-world/29499-2010-10-01-19-34-19.html |publisher = 新三才 |accessdate = 2014-10-31 |archive-url = https://web.archive.org/web/20141031114642/http://www.newsancai.com/big5/history/120-world/29499-2010-10-01-19-34-19.html |archive-date = 2014-10-31 |dead-url = no }}</ref>。由于[[韩语|韩语]]与[[汉语|汉语]]是完全不同的语系，使用汉字记录韩语是一件很不容易的事，加之一般百姓不懂得[[汉文|汉文]]，非常不利于文化的交流与发展<ref name=xunmin>{{cite book |url=http://item.jd.com/10219081.html |title=《新标准韩国语发音教程》 |publisher=天津大学出版社 |author1=李永敏 |year=2008 |page=2 |chapter=韩文的由来 |isbn=9787561828892 |access-date=2014-12-05 |archive-url=https://web.archive.org/web/20141101201026/http://item.jd.com/10219081.html |archive-date=2014-11-01 |dead-url=no }}</ref>。为了解决朝鲜族书写文字的问题，1443年[[朝鲜王朝|朝鲜王朝]][[世宗大王|世宗大王]]组织一批学者创造了适合标记韩语语音的文字体系--[[諺文|韩字]]。这些文字当时被称作“[[训民正音|训民正音]]”，意为“教老百姓以正确的字音”<ref name=xunmin />。韩文的发明推动了国家政治、经济、文化的发展，世宗大王也得到了后世的爱戴<ref name=xunmin />。世宗大王所创造的韩字被公认为是世界上最实用的表音文字之一，其科学性得到了文字研究专家们的高度评价<ref>{{cite web |title =  韩国文字的起源及其发展历史 |url =  http://cs.edulife.com.cn/school/waiguoyu/fulltext-24721.html |publisher =  长沙教育培训网 |accessdate =  2014-10-31 |archive-url =  https://web.archive.org/web/20141109081028/http://cs.edulife.com.cn/school/waiguoyu/fulltext-24721.html |archive-date =  2014-11-09 |dead-url =  yes }}</ref>。

韩语的[[方言|方言]]按地区分布可以分为西北部方言、东北部方言、中部方言、西南方言、东南方言、济州方言、六镇方言7种<ref name=haotui>{{cite web|url=http://www.haotui.com/baike/%B3%AF%CF%CA%D3%EF/|title=朝鲜语|publisher=好推网|date=2014-11-11}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。除了济州方言以外，邻近的方言大都能够通话。但是跨区之间的方言通话就有些困难<ref>{{cite web  |url = http://www.kui.cc/hy/news/40004.html  |title = 朝鲜语使用地区与方言  |publisher = 魁网  |date = 2010-08-20  |access-date = 2014-12-05  |archive-url = https://web.archive.org/web/20160304085053/http://www.kui.cc/hy/news/40004.html  |archive-date = 2016-03-04  |dead-url = no  }}</ref>。現今南韓的“[[大韓民國標準語|標準語]]”是以中部方言的首尔话为基础的。由于[[首尔|首尔]]是[[朝鲜王朝|朝鲜王朝]]的国都，因此這個“标准语”更接近于朝鲜半岛分裂前的官方语言<ref name=haotui />。

=== 建筑 ===
{{Main|韓屋}}
[[韓國傳統建築|-{zh-hant:韓國;zh-hans:朝鲜}-傳統建築]]讲究对地形的选择，[[阴|阴]][[阳|阳]]协调，一般依山朝南而建。傳統民居稱為[[韓屋|韓屋]]，使用一种叫[[温突|温突]]的地板供热系统供热。韩屋在建筑设计上体现人与自然和谐的理念。韩国传统建筑不作违背壮丽自然环境的尝试，也不与自然景色比较高低。韩屋的大部分建筑材料也来源于自然，主要是石头、木材、泥土，充分体现自然美。富人家的屋顶使用瓦建造，而穷人家的屋顶则使用稻草。{{r|韩文化|page =241}}韩国夏季闷热，冬季寒冷。韩屋在设计上采用了冬季取暖的炕和夏季乘凉的大厅。{{r|概况|page =168}}

根据[[儒家|儒家]]的观念，韩屋在空间上做身份、男女、长幼的区分。根据房型的不同和用一道小围墙，韩屋的居住空间分为上中下三等。里屋和[[客厅|客厅]]是主人居住的上等空间；离大门最近的行廊斋是下人居住的地方；中门间行廊斋是中间阶层的守门人居住的空间。[[贵族|贵族]]的韩屋在建造行廊斋等门间斋后，还会建造一个比这些高一截的屋檐上翘的大门。{{r|概况|page =168}}

=== 服装 ===
{{Main|韓服}}
[[File:Korea-Busan-Beomeosa-04.jpg|right]]]]
-{zh-hant:韓國;zh-hans:朝鲜}-的传统服饰是[[韩服|韩服]]。韩服有着[[直线|直线]]和[[曲线|曲线]]相结合的线条美，在色彩搭配上讲究[[阴|阴]][[阳|阳]][[五行|五行]]，天地人和。一般来说上衣用亮色、下衣用暗色。小孩子的韩服一般采用7种颜色，为的是让小孩憧憬绚丽的生活和防止鬼神的接近，无病长寿。{{r|韩文化|page =233}}

男士传统韩服由冠帽（[[黑笠|黑笠]]）、哲高里（男士上衣）、背子（穿在上衣外边的坎肩）、马褂子（穿在哲高里外面的防寒服）、巴吉（裤子）、都鲁玛吉（外出时穿在短衣和裤子外面的外套）和足衣（袜子）组成。{{r|韩文化|page =229}}女士传统韩服由哲高里（女士上衣）、领子（由动襟和领边组成）、高琳（前衣襟上缝有的长带，垂落在长裙前面用作上衣扣，也有装饰作用）、契玛（裙子）、纹样、和绣花鞋组成。{{r|韩文化|page =230}}

根据生活风俗和用途分类，韩服可以分为婚礼服、节日服、花甲服、周岁服等。在春节的早上，子女要穿节日服给父母拜年，小孩子们穿七色彩缎短衣，父母则穿平常韩服；在花甲宴上，男性穿金冠草服，女性穿小礼服；小孩一周岁的仪式上，小孩要穿周岁服；在传统婚礼上，新郎新娘要穿华丽的婚礼服。传统韩服穿戴复杂、繁琐，因此传统韩服一般在特殊的场合穿着。韩国人平时穿着改良的简易生活韩服。{{r|概况|韩文化|page1=450|page2=231-232}}

=== 饮食 ===
{{Main|韓國料理}}
-{zh-hant:韓國;zh-hans:朝鲜}-的主食是[[大米|大米]]。主要农作物除了大米外还有[[大麦|大麦]]，[[大豆|大豆]]等。朝鲜半岛三面沿海，因此海产品也是朝鲜半岛的主要食品。腌制的鱼类和蔬菜是冬季的主要食品。韩国料理味道以辣、甜、酸、淡为特点。汤在韩国料理中不是饭前或饭后的配菜，而是与主食一起食用的主菜。韩国料理中的汤一般放有肉或海鲜，汤水较少<ref name=kfood>{{cite web  |url =http://chinese.korea.net/AboutKorea/Korean-Life/Food  |title =食物  |publisher =Korea.net  |accessdate =2014-10-26  |archive-url =https://web.archive.org/web/20141023003743/http://chinese.korea.net/AboutKorea/Korean-Life/Food  |archive-date =2014-10-23  |dead-url =no  }}</ref>。常见的汤有[[参鸡汤|参鸡汤]]、[[大酱汤|大酱汤]]、[[先农汤|先农汤]]、[[水饺汤|水饺汤]]、[[泡菜锅|泡菜锅]]、[[纯豆腐汤|纯豆腐汤]]、[[海带汤|海带汤]]等。在韩国料理中与米饭一起食用的伴菜叫做“[[饭馔|饭馔]]”，种类繁多，有[[韩国烧烤|烧烤]]、煎炸、蒸煮、[[脍_(韩国)|脍]]、泡制等多种方法制作。而与酒类一起食用的菜被称为“[[按酒_(朝鮮)|按酒]]”。泡菜是韩国料理中的重要成员。韩式泡菜种类繁多，其类别在韩国各地有所不同。韩国人在不同的季节也食用不同的泡菜<ref>{{cite web|url=http://www.chinajilin.com.cn/zhuanti/content/2009-06/26/content_1624754.htm|title=韩国特色美食介绍|publisher=中国吉林网|accessdate=2012-10-13|deadurl=yes|archiveurl=https://web.archive.org/web/20130102121916/http://www.chinajilin.com.cn/zhuanti/content/2009-06/26/content_1624754.htm|archivedate=2013-01-02}}</ref>。

与[[中国|中国]]、[[日本|日本]]和[[越南|越南]]的餐桌礼仪不同，盛米饭和汤的碗是不应该被拿起的<ref name=canli>{{cite web|url=http://www.for68.com/new/201204/li231032592011421029412.htm|title=韩国餐桌礼仪文化|publisher=外语教育网|accessdate=2014-10-26|deadurl=yes|archiveurl=https://web.archive.org/web/20170709083318/http://www.for68.com/new/201204/li231032592011421029412.htm|archivedate=2017-07-09}}</ref>。韩国人使用金属作的长柄[[匙|匙]]来吃饭喝汤。韩式的[[筷子|筷子]]是扁平的金属筷子，只用来夹菜，不用来吃米饭{{r|canli}}。年轻人不应该在长辈或客人之前拿起筷子吃饭，也不应该在长辈或客人之前吃完饭{{r|canli}}。

=== 茶文化 ===
{{Main|韩国茶文化|韩国传统茶}}
公元4世纪，茶文化随[[佛教|佛教]]从中国传入朝鲜半岛，已快有2000多年的历史。<ref>{{cite web |url=http://www.longquanzs.org/articledetail.php?id=1666 |title=存档副本 |accessdate=2009-05-11 |deadurl=yes |archiveurl=https://web.archive.org/web/20100617000023/http://longquanzs.org/articledetail.php?id=1666 |archivedate=2010-06-17 }}</ref>-{zh-hant:韓國;zh-hans:朝鲜}-的茶分为甜、酸、鹹、苦、辛五味。茶文化从中国的传入，也推动力[[朝鲜茶礼|朝鲜茶礼]]的发展。但是在朝鲜王朝，茶文化被认为与佛教有关被废止。韓國傳統茶是以植物為原料，原料包括植物根莖、水果、穀物、種子、花葉等。常見的有[[人参茶|人参茶]]、[[当归茶|当归茶]]、[[薑茶|薑茶]]、[[水正果|水正果]]、[[柚子茶|柚子茶]]、[[大枣茶|大枣茶]]、[[五味子茶|五味子茶]]、[[玄米茶|玄米茶]]、[[決明子茶|決明子茶]]、[[菊花茶|菊花茶]] 、[[灵芝茶|灵芝茶]]等。“传统茶”是将原料长时间浸泡、发酵或熬制而成，是一种天然与健康的饮品，无需开水冲泡，大多数会加入[[蜂蜜|蜂蜜]]。<ref>[http://www.zgchawang.com/Tech/cjsView-id-27431.aspx 韩国茶道:辣椒也能泡着品] {{Wayback|url=http://www.zgchawang.com/Tech/cjsView-id-27431.aspx |date=20130127205843 }}，中国茶叶网</ref>

=== 竞技运动 ===
{{Main|韩国体育}}
-{zh-hant:韓國;zh-hans:朝鲜}-男性的传统体育运动主要包括射箭、[[韩式摔跤|韩式摔跤]]、[[跆拳道|跆拳道]]等。传统的女性体育运动主要是荡[[秋千|秋千]]和[[板舞戲|板舞戲]]。[[跆拳道|跆拳道]]是源于[[朝鲜半岛|朝鲜半岛]]的一种集强身健体，防身自卫和心灵修养为一身的搏击武术，被称为韩国的国技。[[跆拳道|跆拳道]]既是一项能够强身健体，又能够于防身自卫的传统搏击武术，更是一项新兴的集健身、竞技及娱乐为一体的现代体育，集[[力学|力学]]、[[兵学|兵学]]、[[哲学|哲学]]、[[医学|医学]]及[[伦理学|伦理学]]为一体，以技击格斗为基础，以修心养性为核心，以磨练人的意志、振奋人的内在精神气质、培训练习者良好的礼仪及道德为目标。「跆拳道」一詞，是1955年由[[崔泓熙|崔泓熙]]创造。其中「跆」指踢擊、「拳」指拳擊，「[[道|道]]」則是代表道行、自己對禮儀的修練<ref>{{english}} {{cite web|url= http://www.worldtaekwondo.com/history.htm|title= THE HISTORY OF TAEKWONDO|accessdate= 2008-12-23|work= A Report for Recommendation Black Belt Testing 1994|publisher= Glen R. Morris|date= |archive-url= https://web.archive.org/web/20170709083607/http://www.worldtaekwondo.com/history.htm|archive-date= 2017-07-09|dead-url= no}}</ref>。在1986年第10届亚运会，[[跆拳道|跆拳道]]成为亚运会正式比赛项目。2000年悉尼奥运会，跆拳道成为奥运会正式比赛项目。跆拳道已经在世界各国广为推广{{r|概况|韩国|page1=139|page2=211-212}}。

== 传统艺术 ==
-{zh-hant:韓國;zh-hans:朝鲜}-传统民俗音乐主要包括[[盤索里|盤索里]]、[[农乐|农乐]]和[[散调|散调]]。其中盤索里作为一种-{zh-hant:韓國;zh-hans:朝鲜}-传统曲艺形式列入了[[人类非物质文化遗产代表作名录|人类非物质文化遗产代表作名录]]。[[农乐|农乐]]和[[散调|散调]]则被列入韩国[[重要无形文化遗产|重要无形文化遗产]]。

[[朝鮮宫廷音乐|-{zh-hant:韓國;zh-hans:朝鲜}-宮廷音樂]]是朝鲜半岛古代为宫廷王室和贵族演奏的音乐，包括[[朝鮮雅樂|雅乐]]、[[鄉樂|鄉樂]]和[[唐乐|唐乐]]。其中[[朝鮮雅樂|雅乐]]是从中国引进；[[鄉樂|鄉樂]]为[[朝鲜半岛|朝鲜半岛]]本地音乐；[[唐乐|唐乐]]最初是新罗从[[唐朝|唐朝]]引入，后又融合-{zh-hant:韓國;zh-hans:朝鲜}-当地音乐。[[宗庙祭礼乐|宗庙祭礼乐]]作为-{zh-hant:韓國;zh-hans:朝鲜}-宫廷音乐与[[宗庙大祭|宗庙祭礼]]一同被列入[[人类非物质文化遗产代表作名录|人类非物质文化遗产代表作名录]]。

-{zh-hant:韓國;zh-hans:朝鲜}-的传统舞蹈源于[[朝鮮巫教|巫教]]的祭祀活动。分为宫廷舞蹈和民间舞蹈两种。一般说来，在起舞的过程中，跳舞者的腿和脚是不能露出来的。

-{zh-hant:韓國;zh-hans:朝鲜}-的绘画艺术随着[[佛教|佛教]]的传入而发展，佛教绘画技艺成为当时的主流，不过-{zh-hant:韓國;zh-hans:朝鲜}-的自有绘画技艺也一直得以保留。-{zh-hant:韓國;zh-hans:朝鲜}-的绘画一般画在[[桑树|桑树]]纸或[[丝绸|丝绸]]上。18世纪時，-{zh-hant:韓國;zh-hans:朝鲜}-本地的绘画技艺，尤其是书法和刻章方面得到进一步的发展。

== 宗教信仰 ==
朝鲜半岛最初的原始宗教是[[朝鮮萨满教|朝鮮萨满教]]。随着當地与中国的交往，[[佛教|佛教]]和[[儒教|儒教]]后来由中国传入朝鲜半岛。佛教在[[高丽|高丽]]时期曾盛极一时。但在[[朝鲜王朝|朝鲜王朝]]时期，佛教曾受到限制，佛教寺院和佛教徒只能在乡村存在，不许在城市出现，取而代之的是[[儒教|儒教]]。儒家思想一度成为-{zh-hant:韓國;zh-hans:朝鲜}-的国家理念。直到现在儒家思想对-{zh-hant:韓國;zh-hans:朝鲜}-还有着很深的影响，尊敬长辈的儒家思想依然是-{zh-hant:韓國;zh-hans:朝鲜}-家庭的重要部分。但在当今的-{zh-hant:南韓;zh-hans:韩国}-，[[天主教|天主教]]和[[基督新教|基督新教]]{{fact|（主要是浸信会与长老会）}}等基督宗派有着更大的影响力。而且韩国基督徒宣教士直接影响现在中国新兴城市家庭教会的领袖。

== 海外韩民族 ==
=== 中国的少数民族之一朝鲜族 ===
{{main|中国朝鲜族}}
朝鲜族在中国是[[中国东北地区|中国东北地区]]的[[少数民族|少数民族]]，亦是[[中华人民共和国第五次全国人口普查#民族构成|中国第14大民族]]。中国现代朝鲜族大多是19世纪后迁入中国的朝鲜族人的后裔。[[吉林省|吉林]]、[[黑龙江省|黑龙江]]、[[辽宁省|辽宁]]以及[[内蒙古自治区|内蒙]]等省份和北京、上海等城市是现代朝鲜族聚居地，开放的沿海城市也是朝鲜族活跃地区。根据[[中华人民共和国第六次全国人口普查|中华人民共和国第六次全国人口普查]]，中国大陆有朝鲜族人口200多万人。
=== 美国韩民族 ===
{{main|韩裔美国人}}
自19世纪末起就有大批朝鲜半岛人前往[[美国|美国]]定居，其中不乏有[[韩国独立运动|韩国独立运动]]人士[[安昌浩|安昌浩]]及[[李承晚|李承晚]]。根据2006年美国社区人口调查，韩裔美国人约有152万人。美国没有任何美籍朝鲜裔或美籍朝鲜族的统计，只有[[美籍韓裔|美籍韓裔]]的统计。美国是韩人最多的国家之一。
=== 日本朝鲜民族 ===
{{main|日本朝鲜族}}
名称为'''[[在日|在日]](ざいにち)(Zainichi)''
因为日本有日本国籍韩国人，日本国籍朝鲜人，日本国籍朝鲜族，所以要说“朝鲜民族”。
