'''朝鮮总督府'''是[[大日本帝國|大日本帝國]]统治[[朝鲜半岛|朝鲜]]期間，在[[京城府|京城]]（今[[首爾|首爾]]）[[景福宮_(朝鮮)|景福宫]]内设立的最高政府机构。

朝鮮總督府大樓於1926年完工，位在當時漢城主要的南北向大街的主軸線上，1910年[[日韓合併|日韓合併]]后更將景福宮拆除了一部份讓與這座建築當作建地，佔有當年正宮的位置；建築物的造型是歐洲的學院派，上有圓頂、下有列柱的官府建築，體型與規模比起當時的[[台灣總督府|台灣總督府]]顯得恢宏些。駐有朝鲜总督，直隸于[[日本天皇|日本天皇]]，掌握朝鲜殖民地的军事、立法、行政、司法。全权总督下设置政务总監、总督官房、五部（总务部、内务部、度支部、农商工部、司法部）。另外，设有中枢院、警务总監部、法院和铁路司等机构。

朝鲜总督府统治时期可以分為三期；
# 用[[憲兵|憲兵]][[警察|警察]]武力镇压反对派的武断统治（1910年－1919年）；
# 比较融和的文化统治（1919年－1937年）；
# 为进行战争动员人民的战时体制（1937年－1945年）。

1945年9月，最后一位朝鲜总督[[阿部信行|阿部信行]]投降于[[美國|美國]]占领军。韓國獨立以後，韓國人把這棟朝鮮總督府大樓視為國恥的象徵，並沒有把它當作政府建築使用，而是改為[[國立中央博物館|國立中央博物館]]館舍。然而韓國人還是覺得，這樣一棟建築堂而皇之地矗立在漢城的主軸線上是不可忍受的一件事。經過一番爭論，還是決定拆除，以徹底消除國民心上的陰影。1995年，被[[韓國總統|韓國總統]][[金泳三|金泳三]]正式以「清除日本統治時期象徵」為由拆除。

[[File:朝鲜总督府.jpg|thumb]]

由於二戰結束、日本投降後朝鮮半島的分裂，朝鮮總督府也是迄今最後一個實際統治整個半島的政府機關。

==大樓建築==
1995年之前的舊朝鮮總督府大樓坐落在[[景福宮_(朝鮮)|景福宫]]<ref>此為[[朝鮮王朝|朝鮮王朝]]的王宮、皇宮。根據[[風水|風水思想]]建築</ref>前面，於1926年建成，聘請了在日本開辦事處的德國[[建築師|建築師]][[格奥尔格·迪·拉蘭德|格奥尔格·迪·拉蘭德]]設計，他死後由日本建築師（[[野村一郎|野村一郎]]、[[國枝博|國枝博]]等人）完成，樓高四層。

朝鮮總督府大樓在宮殿正門的[[光化門|光化門]]上建築<ref name="iwai">岩井長三郎「総督府新庁舎の計画及実施について」『朝鮮』朝鮮雜誌社 1926年4月p.10-26。復刻、皓星社 1998年。<!--CiNiiでは「新庁舎」の「新」の欠落。誤記。--></ref>、朝鮮王朝的正宮——景福宮周圍的建築物被拆掉不少（一説八成以上<ref name="iwai"/>），但保留了正殿——勤政殿、慶會樓等大部分具象徵意義的建築物。落成後成為朝鮮行政中樞地帶。大樓完成後從街道方向就看不到宮殿，因此[[朝鮮民族|朝鮮民族]]認為總督府大樓是屈辱歷史的[[象徵|象徵]]。

===建築概要===
* 層數：5層/構造：[[鋼筋水泥|鋼筋水泥]]
* 面積：2,134[[坪|坪]]1合6勺（約7,055m²）/總樓面面積：9,619坪7合5勺（約31,800m²）
* 高度：75[[尺|尺]]（約22.7m）、中央塔高180尺（約54.5m）
* 房數：257室/大房間：210坪（約694m²）/大會議室106坪4合（約351m²）/樣式：復興式
* 外牆：[[花崗岩|花崗岩]]（取自漢城東大門外）、内部牆壁鋪上煉瓦
* 費用：636萬4,482圓（其他外圍倉庫及内部整理費38萬7,500圓）

=== 獨立後的大樓===
[[file:Japanese_General_Government_Building_1995.jpg|thumb]]
1948年8月大韓民國政府成立的同時，舊總督府大樓變成政府辦公大樓，稱為「中央廳」，大韓民國的成立宣言也在此發表。

自此韓國內部對此大樓存廢意見分歧，雖然改為[[韓國國立中央博物館|韓國國立中央博物館]]館址，依然有人認為是屈辱歷史的象徵，爭論不休。最後因為政府決定要修復景福宮，故此決定拆卸大樓。政府曾經考慮整棟建築移走，可是因為費用過於高昂，所以於1995年拆卸整座大樓，只保留尖塔部分，放在[[天安市|天安市]]郊外的「獨立記念館」展覽。原被遷移他處的景福宮正門[[光化門|光化門]]，因此得以在此原地重建。

== 警察機構 ==
韓日併合前的1910年7月，韓國統監[[寺内正毅|寺内正毅]]任命[[明石元二郎|明石元二郎]][[日本憲兵|憲兵隊]]司令官兼任警務總長，從此[[憲兵|憲兵]]與普通[[警察|警察]]一體化，稱為{{Link-ja|朝鮮總督府憲兵警察制度|憲兵警察制度 (朝鮮総督府)|憲兵警察制度}}。韓國併合當年「憲兵警察」與「一般警察」合共7712名（當中-{zh-hans:朝鲜;zh-hant:韓國}-人4440名）。「憲兵警察」有2019名（當中-{zh-hans:朝鲜;zh-hant:韓國}-人1012名）<ref>水田直昌監修『統監府時代の財政』122頁</ref>。

[[日本軍|日本軍]]和警察派駐-{zh-hans:朝鲜;zh-hant:韓國}-各地，憲兵以外的[[軍人|軍人]]也擔當統治的角色與執行警察的工作，因為在同時期開設的[[西大門刑務所|西大門刑務所]]負責囚禁不少獨立運動家並執行處刑而為人所知。{{來源請求}}。他們也嚴重鎮壓朝鮮爭取獨立的運動和抵抗日本支配的運動<ref>馬場</ref>。後來憲兵警察隨著改用[[文化政治|文化政治]]而廢止。雖然[[朝鮮總督府警察|朝鮮總督府警察]]轉變普通警察，但仍然擁有連日本本土的警察都沒有的[[機槍|機槍]]和[[野砲|野砲]]等重型裝備，保留著[[準軍事組織|準軍事組織]]的作風，以對付從大日本帝國統治範圍以外的中國境外獨立派[[武裝勢力|武裝勢力]]{{來源請求}}。

== 職員 ==
朝鮮總督府完全接收[[大韓帝國|大韓帝國]]的政府機構，最初開始就已經有很多-{zh-hans:朝鲜;zh-hant:韓國}-人[[官僚|官僚]]，跟其他外地政廳，例如[[台灣總督府|台灣總督府]]從零開始有所不同。

== 王公族、-{zh-hans:朝鲜;zh-hant:韓國}-貴族 ==
設有具特權的身分制度，-{zh-hans:朝鲜;zh-hant:韓國}-的舊皇族成為[[李王家|李王家]]，併合-{zh-hans:朝鲜;zh-hant:韓國}-有功者成為[[朝鮮貴族_(日治時期)|朝鮮貴族]]。

==歷代朝鲜总督==
相較於[[台灣|台灣]]，大日本帝國政府比較關注朝鮮。出任韓國統監、朝鮮總督的官員都是地位相當高的政治家、軍人，相對於出任[[台灣總督|台灣總督]]有部份文官，朝鮮歷任總督全為軍人出身。總督中除了齋藤实是海軍大將以外全是陸軍大將。
