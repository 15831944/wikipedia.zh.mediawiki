'''-{zh-hans:朝鲜语;zh-hant:韓語}-方言'''（{{朝|'''방언'''|方言}}，或{{lang|ko|'''사투리'''}}）分布於[[朝鲜半岛|朝鲜半岛]]各地，各方言的使用范围和[[朝鲜半岛地方|朝鲜半岛]]的自然划分密切相关，所以它們大多都以[[朝鲜八道|朝鲜八道]]命名。

==标准方言==
*[[韩国|韩国]]的标准-{zh-hans:韩语;zh-hant:韓語}-稱為[[大韓民國標準語|標準語]]（{{lang|ko|표준어}}）。[[韩国国立国语院|韩-{}-国国立国语院]]將其定義为“有教養的人廣泛采用的现代首尔話”（{{lang|ko|교양있는 사람들이 두루 쓰는 현대 서울말／敎養있는 사람들이 두루 쓰는 現代 서울말}}）。实际上，南韓標準語並未採納{{en-link|首尔方言|Seoul dialect}}一些獨有的特徵，因此兩者並不完全相等。
*[[朝鲜|朝鲜]]使用的标准-{zh-hans:朝鲜语;zh-hant:韓語}-稱為[[文化語|文化語]]（문화어）。一般認為文化語是以[[平安道方言|平安道方言]][[平壤|平壤]]話作基础，但事實上它是基於開城話（首爾方言的一支），很多平壤話的特徵並未被採用。
虽然南韓标准语和北韓文化语之間存在着一定差異，不过兩者之間基本完全可以互通。

== 朝鮮半島 ==
朝鲜半岛多山，而-{zh-hans:朝鲜语;zh-hant:韓語}-亦按照山川地形而划分为几种地域方言。-{zh-hans:朝鲜语;zh-hant:韓語}-各方言之间并未有明确的分界，所以语言学家有需要按照地形对劃分其方言。根据文献，-{zh-hans:朝鲜语;zh-hant:韓語}-方言大概可以分为以下幾種：

# {{en-link|咸鏡道方言|Hamgyŏng dialect}}（함경도 방언），又名东北方言（동북 방언），分布於朝鲜[[咸镜道|咸镜道]]和[[两江道|两江道]]，以及[[中华人民共和国|中国]][[吉林省|吉林省]][[延邊朝鮮族自治州|延邊朝鮮族自治州]]。这种-{zh-hans:朝鲜语;zh-hant:韓語}-方言共有9个母音，包括ö。
# [[平安道方言|平安道方言]]（평안도 방언），又名西北方言（서북 방언），分布於朝鲜[[平壤直辖市|平壤直辖市]]、[[平安道|平安道]]、[[慈江道|慈江道]]，以及[[中國|中國]][[遼寧省|遼寧省]]东部的朝鮮族聚居地。
# {{ko-link|黄海道方言|황해도 방언}}（황해도 방언），分布於朝鲜[[黄海道|黄海道]]（除[[開豐郡|開豐郡]]、[[長豐郡|長豐郡]]和[[開城特級市|開城特級市]]）。语言学家一般视之为西北方言或中部方言分支，不过和這些方言相比，黄海道方言显得格格不入<ref name=Lee&Ramsey/>。
# {{ko-link|中部方言|중부 방언}}（중부 방언）<ref name=Lee&Ramsey>Lee & Ramsey：The Korean language，2000年。</ref>：
#*{{en-link|首尔方言|Seoul dialect}}（서울 방언），又名京畿道方言（경기도 방언），分布於韩国[[首尔特別市|首尔特別市]]、[[仁川廣域市|仁川廣域市]]和[[京畿道|京畿道]]，以及朝鲜[[黃海道|黃海道]]的[[開豐郡|開豐郡]]、[[長豐郡|長豐郡]]和[[開城特級市|開城特級市]]。南韓标准语和北韓文化語皆以此方言為基礎。
#*{{ko-link|岭西方言|영서 방언}}（영서 방언），分布於[[太白山|太白山]]以西（屬[[江原道_(朝鮮八道)|江原道]]，現分屬[[江原道_(韓國)|-{zh-hans:韩国、;zh-hant:南}-]][[江原道_(朝鮮)|-{zh-hans:朝鲜;zh-hant:北韓}-]]）。
#*{{ko-link|岭东方言|영동 방언}}（영동 방언），分布於太白山以东（屬江原道，現況同上）。
#*虽然人们經常把嶺東和嶺西方言合称“江原道方言”，不过事實上兩者间差異頗大。
#*[[忠清道方言|忠清道方言]]（충청도 방언），分布於韩国[[大田广域市|大田广域市]]和[[忠清道|忠清道]]。
# [[庆尚道方言|庆尚道方言]]（경상도 방언），分布於韩国[[釜山广域市|釜山广域市]]、[[大邱广域市|大邱广域市]]、[[蔚山广域市|蔚山广域市]]和[[庆尚道|庆尚道]]。此方言是眾多-{zh-hans:朝鲜语;zh-hant:韓語}-方言中唯一仍然保留聲調的一支。这种方言共有6个母音，分别是：i、e、a、u、o和eo。
# {{en-link|全罗道方言|Jeolla dialect}}（전라도 방언），又名西南方言（서남 방언），分布於韩国[[光州廣域市|光州廣域市]]和[[全罗道|全罗道]]。这种方言共有10个母音，分别是：i、e、ae、a、ü、ö、u、o、eu和eo。
# [[济州语|济州岛方言]]（제주도 방언），又名济州语（제주어），分布於韩国[[濟州特別自治道|济州岛]]。它和其他方言的差異之大，使得语言学家有時甚至会将之视为一种独立于-{zh-hans:朝鲜语;zh-hant:韓語}-的语言。濟州方言完整的保留了中古-{zh-hans:朝鲜语;zh-hant:韓語}-的9个母音，其中包括「ㆍ」（/ɒ/）。

== 朝鲜半岛以外 ==
* [[高麗語_(蘇聯)|高丽语]]（自稱：{{lang|ru|Корё мар}}、{{lang|ko|고려말 / 高麗말}}，南韓标准语：{{lang|ko|중앙아시아 한국어 / 中央아시아 韓國語}}），通常被视为咸鏡道方言的分支，使用者为[[高丽人|高丽人]]，即移居[[苏联|前苏联]]的朝-{}-鲜族。詞彙以一些-{zh-hans:朝鲜语;zh-hant:韓語}-的基础词汇為主，不过它从[[俄语|俄语]]和[[突厥語族|突厥語族]]各语言借用了很多外来语和转译词。
* [[在日朝鲜语|在日-{zh-hans:朝鲜语;zh-hant:韓國語}-]]（{{lang|ko|재일어}}、{{lang|ko|재일 조선어}}、{{lang|ko|재일 한국어}}）是由[[在日朝鮮人|在日朝-{}-鮮人]]使用的-{zh-hans:朝鲜语;zh-hant:韓語}-方言，受[[日语|日语]]強烈影响。
* [[中国朝鲜语|中国朝鲜语]]与咸镜道方言幾乎相同，但两者之间仍有差异，例如前者有很多来自[[中文|中文]]的现代术语。
