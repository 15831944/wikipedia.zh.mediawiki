'''朝聖'''是[[宗教|宗教]]或[[靈性|靈性生活]]尋覓靈性意義的過程，通常是到一處聖地或者是對某人信仰有重要意義的地方。各大宗教都有朝聖的活動。從世俗的角度而言，朝聖是一種族群的認同，而非基於信仰。[[流行文化|流行文化]]也正把朝聖重新定義，例如去探訪某[[小說|小說]]、[[電視劇|電視劇]]、[[電影|電影]]或[[ACG|ACG]]等作品取材、拍攝的地點也被稱為朝聖、[[聖地巡禮|聖地巡禮]]。[[旅遊業|旅遊業]]興起之前，朝聖是人們離開家鄉出外遊歷的主要方式之一。

== 巴哈伊信仰 ==
{{main|巴哈伊信仰的朝圣}}
[[巴哈欧拉|巴哈欧拉]]在[[亚格达斯经|亚格达斯经]]中为巴哈伊列明兩處朝圣地點（信徒可任選其一，各有相配的儀式）：[[伊朗|伊朗]][[設拉子|設拉子]]的巴孛故居（己毀）和在[[伊拉克|伊拉克]][[巴格達|巴格達]]的[[巴哈欧拉|巴哈欧拉]]故居（未發還），其后，[[阿博都巴哈|阿博都巴哈]]增加[[巴吉宅邸|巴吉宅邸]]作为朝圣地点。目前，因为前往[[伊朗|伊朗]]和[[伊拉克|伊拉克]]朝圣有较多限制，[[巴哈伊教|巴哈伊教]]的信徒一般前往[[以色列|以色列]]西北的[[海法|海法]]、[[阿卡_(以色列)|阿卡]]和[[巴吉宅邸|巴吉宅邸]]（未有特別儀式）。

在能力許可和前無障礙的條件下，信徒有朝聖的義務（女性未被要求參加，但亦不禁止）。

== 基督宗教 ==
基督宗教原則上並不严格要求信徒朝聖。[[天主教|天主教]]和[[東正教|東正教]]認為信徒朝聖後可以得到特別的祝福；相反的，新教則不認為朝聖能取得祝福，但朝聖可令信徒反思聖經的故事。

早期的朝聖環繞[[耶穌基督|耶穌基督]]的一生（4世紀時已經有[[教父|教父]]鼓勵到聖地朝聖的紀錄），後來[[聖母|聖母]]顯現的地點、與[[聖人|聖人]]及[[殉道者|殉道者]]相關的地點，以及[[羅馬|羅馬]]也成為朝聖的地點。但是，广义而论，在本地的短期的[[徒步朝圣|徒步朝圣]]，亦属於[[朝圣|朝圣]]的一种型态。

[[天主教|天主教]]歷史上最大的朝聖活動是[[教宗|教宗]][[若望保祿二世|若望保祿二世]]在2005年4月2日去世後，四百萬來自其他地區的教友 ，連同三百萬在羅馬居民瞻仰教宗遺容。

[[世界青年日|世界青年日]]亦被視為現代的朝聖活動。最大規模的一次是1995年在[[菲律賓|菲律賓]][[馬尼拉|馬尼拉]]，有四百萬人參加。最近一次是2016年在[[波蘭|波蘭]][[克拉科夫|克拉科夫]]。

== 伊斯蘭教 ==
=== 朝覲 ===
{{main|朝觐 (伊斯兰教)}}
[[朝觐_(伊斯兰教)|朝覲]]是[[伊斯蘭教|伊斯蘭教]][[五功|五功]]之一。所有穆斯林，只要有能力，一生必須去一次[[麥加|麥加]]朝聖。另外多數[[什葉派|什葉派]]穆斯林都會到[[伊朗|伊朗]]的[[馬什哈德|馬什哈德]]朝聖。

[[File:-Western_Wall.jpg|right]]

=== 謁陵 ===
在穆斯林國家，有拜謁聖人或英雄的陵墓習俗。在特定時節謁陵更是國族認同的體現，但某些伊斯蘭教派對謁陵持負面態度。

穆斯林的謁陵地點有包含[[沙烏地阿拉伯|沙烏地阿拉伯]][[麥地那|麥地那]]的{{tsl|en|Al-Masjid al-Nabawi|穆罕默德陵墓}}、[[耶路撒冷|耶路撒冷]]的[[圓頂清真寺|圓頂清真寺]]与[[阿克萨清真寺|阿克萨清真寺]]（远寺）（先知穆罕默德登霄处）

== 猶太教 ==

古代[[猶太教|猶太教]]規定有能力者須每年到[[耶路撒冷|耶路撒冷]]慶祝[[逾越節|逾越節]]、[[五旬節|五旬節]]和[[住棚節|住棚節]]，並在[[聖殿|聖殿]]祭獻。在聖殿第二次被毀後，此習慣無法進行，猶太教徒只能在禱詞中祈求有朝一日重回[[錫安山|錫安山]]。直到1967年，猶太人終於可以在[[哭牆|哭牆]]祈禱。
