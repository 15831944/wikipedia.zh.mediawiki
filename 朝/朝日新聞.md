《'''朝日新聞'''》（{{jpn|j=朝日新聞|hg=あさひしんぶん|rm=Asahi Shimbun|f=y}}）是[[日本|日本]]的全國性[[報紙|報紙]]之一，由[[朝日新聞社|朝日新聞社]]發行，於1879年1月25日在[[大阪|大阪]]創刊。2008年，每日日報發行量超過770萬份，为日本国内日報發行量第二名，僅次於《[[讀賣新聞|讀賣新聞]]》。

== 專欄 ==
在《朝日新聞》中有一個卓越的[[社論|社論]]專欄名為「[[天聲人語|天聲人語]]」，針對日本近期新聞或事件發表論見。此專欄目前也提供了電子報的訂閱。

「朝日俳句詩人網」（Achimhae Haikuist Network）則是一週性的專欄，由作家[[大衛·麥馬瑞|大衛·麥馬瑞]]（David McMurray）寫作。大衛是全世界以[[英文|英文]]寫作日本[[俳句|俳句]]的重要人物。此專欄刊出的俳句都有註解。

== 立場 ==
《朝日新聞》政治立場[[中间偏左|中间偏左]]，属于左派媒体<ref name='a1'>[http://kuchiran.jp/business/newspaper.html 各新聞やテレビ局の右翼・左翼に関して | レビュー図書館]</ref>。帶有強烈的[[左派|左派]][[國際主義|國際主義]]觀點<ref>{{cite web |url=http://cc.shu.edu.tw/~atss1/Seminar/20130927/B-3.pdf |title=存档副本 |accessdate=2014-09-01 |deadurl=yes |archiveurl=https://web.archive.org/web/20140904205556/http://cc.shu.edu.tw/~atss1/Seminar/20130927/B-3.pdf |archivedate=2014-09-04 }}</ref><ref>[http://www.chosakai.gr.jp/notification/pdf/report2.pdf 第2回メディアに関する全国世論調査（2009年）] {{Wayback|url=http://www.chosakai.gr.jp/notification/pdf/report2.pdf |date=20130423230741 }}公益財団法人新聞通信調査会</ref>、標榜[[进步主义|进步]]<ref>かつては進歩的文化人と同義で「朝日文化人」という用語も使用されていた。酒井寅吉著『朝日文化人―この日本をどうしようというのか』など。</ref>、[[自由主義|自由]]<ref>[http://chuohjournal.jp/2005/09/post_410.html リベラル派朝日新聞・岩波書店の自滅] {{webarchive|url=https://web.archive.org/web/20101118162141/http://chuohjournal.jp/2005/09/post_410.html |date=2010-11-18 }}</ref>，偏向左派的社會主義。在1936年的[[二二六事件|二二六事件]]中，因其自由主义观点被日本右翼皇道派兵变士兵攻击，60名士兵冲进东京朝日新闻社进行打砸，损失金额3万日元，又士兵指挥官粟原中尉撤退时高喊：“国贼朝日新闻多年来标榜自由主义，拥护重臣集团，此次行动乃天诛。”<ref>《战前日本国家主义运动史》P274页</ref>

== 读者层 ==
朝日新闻社广告局称他们的读者学历高于其他报纸<ref>[http://adv.asahi.com/modules/ad/index.php/about.html 朝日新聞社広告局ウェブサイト] {{webarchive|url=https://web.archive.org/web/20140202110013/http://adv.asahi.com/modules/ad/index.php/about.html |date=2014-02-02 }}</ref>，但也有[[大阪商业大学|大阪商业大学]]等研究指出在日本五大报纸中读者学历最高应该是《[[日本经济新闻|日本经济新闻]]》，《朝日新闻》排第二<ref>[http://jgss.daishodai.ac.jp/research/monographs/jgssm3/jgssm3_05.pdf#search='%E6%9C%9D%E6%97%A5%E6%96%B0%E8%81%9E+%E8%AA%AD%E8%80%85%E5%B1%A4' 現代 日本の新聞読者層]</ref><ref>[http://www.garbagenews.net/archives/739036.html 【更新】主要新聞社サイトの読者数や読者年収などをグラフ化してみる]</ref>。

===吉田證言事件===
1992年1月23日，朝日新聞刊出了[[吉田清治|吉田清治]]的自白。吉田清治自稱曾奉命到韓國濟州島抓捕島上的女性當作性奴隸，其證言被當作日本軍方強抓慰安婦的決定性證據而被稱作[[吉田證言|吉田證言]]。 不過事後有韓國和日本記者相繼到當地採訪，但發現當地沒有人聽過這事。吉田清治最後承認自己是為了金錢而捏造證言<ref>{{cite web|url=http://www.ianfu.net/history/history.html|title=慰安婦にまつわる年表|author=当サイトは、問題になっている従軍慰安婦問題についての真実を掘り起こす為のサイトです。|publisher=}}</ref>。

朝日新聞在事件後未有作出回應，多年來一直受日本保守派作為「賣國的證據」攻擊，朝日最後於2014年8月5日承認有「誤會」並修正了該則新聞，但否認了捏造的指控<ref>{{cite web|url=http://www.asahi.com/topics/ianfumondaiwokangaeru/|title=慰安婦問題を考える|work=朝日新聞デジタル}}</ref>	。此事在日本引發政治風暴，因多年來日本為了慰安婦問題而飽受各國批評，當中吉田證言成為了重要證據之一，然而由於吉田證言被推翻，日本有政客研究向朝日新聞追究責任<ref>{{cite web|url=http://news.sina.com.hk/news/20140822/-12-3342579/1.html|title=日自民黨要求就慰安婦錯誤報道制裁朝日新聞|work=香港新浪}}</ref><ref>{{cite web|url=http://www.cna.com.tw/news/aopl/201408060463-1.aspx|title=慰安婦報導  朝日32年後撤稿|author=中央通訊社|date=2014-08-06|publisher=}}</ref>。

=== 沖繩集會人數失誤事件 ===
朝日新闻在2007年9月谈及[[安倍晋三|安倍晋三]]问题以及对9月底在[[沖繩縣|沖繩縣]][[宜野湾市|宜野湾市]]进行的教科书審查活动反对集会参与人数的严重报道失误，将不到2万（警方某官员认为的4万人）误发表为“11万”人而引起了轰动<ref>{{cite web| url =http://s03.megalodon.jp/2007-1019-1601-30/www.nikaidou.com/2007/10/post_640.html| title=テイケイが証明した沖縄集会のウソ| publisher=nikaidou.com（ウェブ魚拓）| date =2007-10-17| accessdate=2010-02-27|language=ja}}</ref>。{{lang|ja|アサヒる}}（Asahiru）一字開始被部分厭惡朝日新闻的人士當作动词使用，意思是沒有根據、捏造。其中「{{lang|ja|アサヒ}}」是「朝日」的[[片假名|片假名]]寫法，「{{lang|ja|る}}」是[[日語|日語]]的[[動詞|動詞]][[字尾|字尾]]。「{{lang|ja|アサヒる}}」一詞被“[[2ch|2ch搜尋]]”（{{lang|ja|2ちゃんねる検索}}）评为2007年度[[网络流行语|网络流行语]]首奖。

===黑名單事件===	
由於[[海上自衛隊|海上自衛隊]]認為朝日新聞多次對自衛隊作出批判性的報導，於是將其列入黑名單，拒絕其採訪。<ref>{{cite web|url=http://www.cyzo.com/2013/05/post_13353.html|title=海上自衛隊の取材者ブラックリスト - 日刊サイゾー|work=日刊サイゾー}}</ref>

== 跨國合作 ==
《朝日新聞》與《[[紐約時報|紐約時報]]》集團旗下《[[國際先驅論壇報|國際先驅論壇報]]》有合作的關係，2001年4月2日正式發行一同出版發行了《IHT/Asahi》，以取代朝日新聞出版的另一份英文日報《朝日晚報》<ref>{{cite av media|url=http://www.youtube.com/watch?v=csVrA9IcFBg|title=4.演説後、安倍さん麻生さんと握手＆マスゴミへ怒声in秋葉原 字幕入 2/2|date=2012-09-22|publisher=|via=YouTube}}</ref>。

2013年1月起，朝日新聞中文網一度被中國大陸[[防火長城|防火長城]]封鎖，無法直接打開。目前中国大陆基本可以正常浏览朝日新闻中文网。

2013年7月17日，朝日新闻中文网的在中国大陆的四个官方微博全部被关闭。<ref>[http://www.zaobao.com/realtime/china/story20130718-229737 朝日新闻中国微博帐号无法使用].联合早报</ref>现微博账户以非认证用户形式常态化更新，且名称使用英文名“Asahi Shimbun”，但简介中仍说明为朝日新闻中文网微博账号。<ref>[https://www.weibo.com/u/3173216505 AsahiShimbun中文网的微博_微博]
