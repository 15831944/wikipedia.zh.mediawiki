'''[[朝鮮民主主義人民共和國|朝鮮民主主義人民共和國]]'''（以下簡稱'''朝鮮'''）'''的[[啤酒|啤酒]]'''源於1930年代。如今，全朝鮮有至少十家的大型[[啤酒廠|啤酒廠]]和為數眾多的{{link-en|精釀啤酒廠|Microbrewery}}{{refnest|name="Microbrewery"|group="註釋"|原文「Microbrewery」亦指規模較小的釀酒廠<ref name="Microbrewery"/>。}}，其中最大的啤酒品牌為[[大同江啤酒|大同江啤酒]]。朝鮮在物資分配和電力供給方面的問題促使朝鮮釀酒商轉型：因為啤酒配給的問題，許多餐館和酒店都設有自己的小型釀酒廠；由於電力供給不穩定，啤酒不方便冷藏，於是朝鮮人自創{{link-en|蒸汽啤酒|Steam beer}}（源於美國，是一種在高於常溫的溫度下釀造的啤酒）。雖然朝鮮最受歡迎的酒是[[韓國燒酒|燒酒]]，不過啤酒也佔有一席之地。朝鮮人想要喝到啤酒必須使用糧票。然而，前往[[朝鮮旅遊|朝鮮旅遊]]的遊客不受此限制，他们可以用外幣购买啤酒饮用。

== 歷史==

正值[[朝鮮日治時期|朝鮮日治時期]]的1930年代，日本人將德國[[拉格啤酒|拉格啤酒]]引進朝鮮<ref name="KipleOrnelas2000"/>。截止至[[第二次世界大戰|第二次世界大戰]]結束和朝鮮獨立過後的1960年，北韓所有的啤酒都是國內自行釀造的<ref name="Lee2001"/>。啤酒的配給量在1980年代相當高，普通北韓人也可以喝得到<ref name="Cumings2011"/>。然而，到了1990年代中期，北韓的物資配給制度崩潰，北韓人只有在[[朝鮮民主主義人民共和國節日|重要節日]]才能配給到一瓶[[韓國燒酒|燒酒]]和三瓶啤酒{{sfn|Lankov|2007|p=102}}。所有啤酒都裝在劣質的國產玻璃瓶中<ref name="Tripoto"/>。1994年至1998年，[[苦難的行軍|北韓饑荒]]（又稱苦難的行軍）期間，因醫療設備匱乏，啤酒瓶被拿來做[[靜脈注射|靜脈注射]]的用具<ref name="Demick2010"/>。

2000年，北韓經濟已復甦。雖然北韓早已設有[[啤酒廠|啤酒廠]]，不過北韓時任領導人[[金正日|金正日]]打算繼續發展北韓的釀酒工業<ref name="NYT"/><ref name="Vice"/>。當年夏天，北韓國有的[[大同江啤酒|大同江啤酒]]公司獲准買下英國老牌啤酒廠「{{link-en|特羅布里奇的亞舍斯|Ushers of Trowbridge}}」的設備，運回北韓<ref name="NYT"/><ref name="Vice"/><ref name="The Straits Times"/><ref name="蘋果日報"/>。2001年，位於北韓首都[[平壤|平壤]][[寺洞區域|寺洞區域]]的大同江啤酒廠建設完工，並於隔年4月對外公開<ref name="NYT"/><ref name="朝鮮時報"/>。不久後，樂園精釀啤酒廠與[[羊角島國際飯店|羊角島飯店精釀啤酒廠]]也在平壤開設<ref name="Vice"/>。

2016年8月<ref name="BBC"/>，大同江啤酒廠在全國舉辦北韓有史以來第一次的[[啤酒节|啤酒節]]<ref name="BBC"/><ref name="Asia Society"/>。不過到了隔年，啤酒節就因故取消了，原因可能是[[旱災|旱災]]<ref name="NK News"/><ref name="啤酒節"/><ref name="啤酒節2"/>。

== 釀造 ==

由於亞洲啤酒的主原料[[稻米|稻米]]在北韓被当做食物，所以北韓人大多採用不會拿來做食物的[[大麥|大麥]]來釀啤酒<ref name="Tripoto"/><ref name="Asia Society"/>。因此，北韓的啤酒顏色更深、更濃郁且帶有麥芽香<ref name="Asia Society"/>。一位作家說，外在因素「迫使北韓人釀出比世上其他國家更有趣的啤酒」<ref name="Asia Society"/>。啤酒的原料[[蛇麻|啤酒花]]也是北韓自產的<ref name="中時電子報"/>。由於北韓人的口味偏好，北韓啤酒的啤酒花含量較低<ref name="Asia Society"/>。北韓的工業並不發達，沒有水資源的汙染問題，因此北韓的啤酒廠有著豐富的淡水<ref name="NYT"/>。釀酒的其餘原料需從國外進口<ref name="NYT"/>。

== 啤酒文化 ==

雖然北韓處於孤立狀態，卻有著多采多姿的啤酒文化<ref name="Wired UK"/>。在北韓，人們把啤酒當做[[軟性飲料|軟性飲料]]，成人和兒童都會喝啤酒<ref name="The Guardian"/><ref name="三立新聞網"/><ref name="風傳媒"/>，但啤酒並不是北韓最受歡迎的酒。比起啤酒，北韓人更喜愛[[韓國燒酒|燒酒]]，因此北韓啤酒知名度並不高{{sfn|Lankov|2007|p=101}}。雖然如此，啤酒和燒酒仍是北韓最常見的兩種酒{{sfn|Lankov|2007|p=100}}，啤酒在所有酒中佔5.1%<ref name="WHO2014"/>。不過，有說法稱北韓人「熱愛啤酒的程度不亞於歐美人」<ref name="Wired UK"/>。一位業餘釀酒師喬許·湯瑪士（{{lang|en|Josh Thomas}}）說道：

{{quote|平均來說，平壤男性喝啤酒的頻率相當頻繁——每週至少一次。農村人口喝酒的次數就比較低，有的甚至滴酒不沾。北韓的啤酒文化主要是男人下班後去酒吧喝個幾杯，然後回家。這種文化還在成長<ref name="Asia Society"/>。}}

據報導，下班後的上班族、退休老人、家庭主婦、公務員和工人都會到啤酒屋<ref name="三立新聞網"/>。有人認為，喝啤酒的文化在未來也會進入年輕人圈子{{sfn|Yonhap|2002|p=460}}。其實早在1970年代，北韓的年輕人就已仿效[[大韓民國|大韓民國]]（以下簡稱南韓），掀起了一股喝啤酒和演奏[[民俗音樂|民俗音樂]]的風潮{{sfn|Yonhap|2002|p=461}}。

== 取得方式 ==

[[File:Taedonggang_Microbrewery_No._3_Pyongyang_Beer_(10091203546).jpg|thumb]]3號{{link-en|精釀啤酒廠|Microbrewery}}喝啤酒的遊客。遊客可以自由飲用啤酒，不受限制]]

大量生產的啤酒在北韓很常見{{sfn|Tudor|Pearson|2015|p=65}}。若想取得啤酒，北韓人必須使用[[配給制|糧票]]{{sfn|Hokkanen|2013|p=133}}。只有居住在平壤的北韓男子可以拿到啤酒券，供他們每月在低檔次的酒吧喝1至2公升的啤酒<ref name="Asia Society"/>。這些限制使得啤酒成為只有少數人才能喝的飲料{{sfn|Hokkanen|2013|p=133}}。不過對於平壤的[[暴發戶|暴發戶]]來說，進出啤酒店不過是家常便飯<ref name="Nouveau riche"/><ref name="Lankov2014"/>。有不少人會開{{link-en|精釀啤酒廠|Microbrewery}}自釀啤酒（見[[#精釀啤酒廠|#精釀啤酒廠]]），而有的沒有啤酒喝的窮困鄉下人會用他們找得到的任何材料釀造啤酒：「我們拿[[矢車菊|矢車菊]]和[[蛇麻|啤酒花]]釀出了顏色詭異的乳白色的東西，但至少這玩意像啤酒一樣有泡沫」，一位[[脫北者|脫北者]]說<ref name="NYT"/>。

早期，平壤各地仅有政府經營的啤酒屋，如今因政府經濟管控政策放鬆，在私人餐廳也可以喝到啤酒{{sfn|Lankov|2007|p=101}}。酒吧在平壤也十分常見<ref name="Asia Society"/>。另外，在夏天或節日，平壤會出現許多搭著帳篷的啤酒攤販{{sfn|Willoughby|2014|p=127}}。

外國遊客不受與北韓人相同的限制，可以自由用外幣購買啤酒{{sfn|Hokkanen|2013|p=133}}。國際酒店販賣的啤酒要價2歐元左右{{sfn|Hokkanen|2013|p=34}}。

在[[菜市場|菜市場]]，一瓶啤酒要價約53[[北韓圓|北韓圓]]{{sfn|Yonhap|2002|p=377}}。[[海尼根|海尼根]]、虎牌啤酒、{{link-en|艾丁格啤酒|Erdinger}}和巴伐利亞啤酒等國外啤酒在北韓也日漸普遍{{sfn|Willoughby|2014|p=97}}。一罐中國啤酒（如[[青島啤酒|青島啤酒]]或[[哈爾濱啤酒集團|哈爾濱啤酒]]）在[[朝鮮民主主義人民共和國黑市|北韓黑市]]要價可達4,000北韓圓{{sfn|Tudor|Pearson|2015|p=27}}。

== 品牌與種類 ==

[[File:TAEDONGGANG_BEER_DPR_KOREA_OCT_2012_(8196688556).jpg|thumb]]是北韓最大的啤酒品牌]]

北韓有至少十家的大型[[啤酒廠|啤酒廠]]，比[[大韓民國|南韓]]還多{{sfn|Tudor|Pearson|2015|p=65}}。北韓的啤酒廠有樂園精釀啤酒廠（以[[小麥啤酒|小麥啤酒]]為主），[[羊角島國際飯店|羊角島飯店精釀啤酒廠]]（以美式金色麥啤為主）和[[大同江啤酒|大同江啤酒廠]]（以[[拉格啤酒|拉格啤酒]]為主）等<ref name="Vice"/><ref name="Wired UK"/>。北韓的啤酒十分多樣<ref name="NYT"/>，最主要的有{{link-en|蒸汽啤酒|Steam beer}}、燕麥[[司陶特啤酒|司陶特啤酒]]、巧克力[[波特啤酒|波特啤酒]]和[[愛爾淡啤酒|愛爾淡啤酒]]<ref name="Wired UK"/>，也有比較少見的米啤酒<ref name="聯合新聞網"/>。喬許·湯瑪士說，「對一個常受飢荒之苦的國家而言，北韓的啤酒種類相當豐富」<ref name="Wired UK"/>。

[[大同江啤酒|大同江啤酒]]是北韓最大也是最頂尖的啤酒品牌，《[[紐約時報|紐約時報]]》曾將大同江啤酒評為「[[朝鲜半岛|朝鲜半岛]]品质最好的啤酒之一」<ref name="NYT"/><ref name="Vice"/>，也有人認為大同江啤酒比亞洲其他大眾市場啤酒都要好<ref name="Asia Society"/>。2005年至2007年間，大同江啤酒甚至出口到南韓<ref name="Vice"/>。北韓的其他啤酒品牌包含與大同江啤酒並列「北韓四大啤酒」的鳳鶴啤酒（北韓高層幹部專用的酒）<ref name="Irresponsible Life"/>、龍城啤酒和金剛啤酒<ref name="風傳媒"/><ref name="自由時報"/>，以及平壤啤酒、Pohak啤酒<ref name="FordKwon2008"/>、樂園啤酒<ref name="Demick2010"/>和三角啤酒<ref name="North Korean Economy Watch"/>。

=== 精釀啤酒廠 ===

[[File:Rakwon_Paradise_Microbrewery_(11416315073).jpg|thumb]]的樂園精釀啤酒廠是北韓的{{link-en|精釀啤酒廠|Microbrewery}}之一]]

北韓也有許多{{link-en|精釀啤酒廠|Microbrewery}}<ref name="Asia Society"/>。雖然北韓政府不允許人們自釀啤酒，但他們无法遏制这种情况<ref name="The Guardian"/>。有人認為，為數眾多的精釀啤酒廠是為了应对[[对朝鲜的制裁|北韓遭到國際制裁]]、[[石油|石油]]供給不足導致啤酒難以配給的情況：<ref name="Wired UK"/>

{{quote|北韓是個精釀啤酒廠遍布的國家。酒店自己釀啤酒，保齡球館自己釀啤酒，餐廳也自己釀啤酒。固然[[大同江啤酒|大同江啤酒]]在全國各地都買得到，但是一有足夠的空間，人們似乎就會為了抵制低效率的供應鏈而自己蓋精釀啤酒廠來釀啤酒<ref name="Wired UK"/>。}}

許多酒吧和酒店都會自行釀造啤酒{{sfn|Tudor|Pearson|2015|p=63}}，如釀出來的啤酒被譽為「當地第一」的平壤[[高麗飯店|高麗酒店]]{{sfn|Hokkanen|2013|p=34}}和[[清津市|清津市]]的清津观光饭店{{sfn|Willoughby|2014|p=228}}等。[[羅津先鋒經濟特區|羅津先鋒經濟特區]]有一家採用捷克出產設備的精釀啤酒廠<ref name="VOA"/>。事實上，全北韓只有大同江啤酒廠能算得上大型啤酒廠，其餘的啤酒廠都只算是精釀啤酒廠<ref name="Asia Society"/>。

=== 蒸汽啤酒 ===

北韓有各式各樣的{{link-en|蒸汽啤酒|Steam beer}}。蒸汽啤酒源於美國，是一種在高於常溫的溫度下釀造的啤酒。由於北韓電力供給不穩定，啤酒不方便冷藏，因此蒸汽啤酒對北韓人相當重要<ref name="Wired UK"/>。冷藏不便也讓[[愛爾啤酒|愛爾啤酒]]十分受歡迎<ref name="Wired UK"/>。北韓的蒸汽啤酒是由[[拉格啤酒|拉格啤酒]]釀造而成<ref name="Tripoto"/>，在北韓十分普及<ref name="Wired UK"/>。

== 品質與評價 ==

一般而言，北韓的啤酒品質比南韓較好{{sfn|Lankov|2007|p=101}}，且不像南韓的啤酒那麼淡<ref name="Wired UK"/>。與北韓相反，許多人認為南韓的啤酒品質低劣，負面評價不斷<ref name="Vice"/>。英國商業週刊《[[經濟學人|經濟學人]]》一篇於2012年11月下旬發布的文章中指出，啤酒是[[朝鲜民主主义人民共和国|-{zh-hans:朝鲜民主主义人民共和国;zh-hant:北韓;}-]]唯一領先於[[大韩民国|-{zh-hans:大韩民国;zh-hant:南韓;}-]]的領域<ref name="Vice"/><ref name="The Economist"/>。這篇文章引起了軒然大波<ref name="Vice"/>，並刺激南韓修改啤酒方面的法律，南韓精釀啤酒廠也逐漸興起<ref name="Vice"/>。不過，[[英國廣播公司|英國廣播公司]]（BBC）記者史蒂芬·伊凡斯（{{lang|en|Stephen Evans}}）認為「北韓原本可釀造麥香濃厚的上層發酵啤酒『[[愛爾啤酒|愛爾啤酒]]』，他們卻選擇釀口味清爽的下層發酵啤酒『[[拉格啤酒|拉格啤酒]]』，我覺得這個做法不夠明智」<ref name="風傳媒2"/>。

== 相關 ==

近年，疑似南韓一種將燒酒與啤酒混合饮用的「炸彈酒」（类似华人社会流传的“[[深水炸弹_(鸡尾酒)|深水炸弹]]”）傳入北韓，[[朝鲜中央广播电台|朝鮮中央廣播電台]]提醒北韓民眾，炸彈酒「會對心臟和肝臟等調節體溫的器官造成影響」<ref name="炸彈酒"/>。
