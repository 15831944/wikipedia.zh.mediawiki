'''朝鮮正祖'''（{{kor|k=조선 정조|rm=Joseon Jeongjo|hanja=朝鮮 正祖}}；1752年10月28日（農曆9月22日）－1800年8月18日（農曆6月28日）），名諱'''李-{祘}-'''（{{kor|k=이산|rm=Yi San|hanja=李祘}}），[[朝鲜王朝|朝鲜王朝]]的第22代君主，1776年至1800年在位。他的父親是[[朝鮮英祖|朝鮮英祖]]的次子[[莊獻世子|莊獻世子]]（高宗追尊為[[朝鮮莊祖|莊祖]]），生母是[[惠慶宮洪氏|惠慶宮洪氏]]（高宗追尊為[[獻敬王后|獻敬王后]]、獻敬懿皇后）。

== 早年 ==
[[File:2009-01-24_-_Portrait_of_King_Jeongjo_in_Unhangak.JPG|thumb]]

英祖二十八年九月二十二日，生於[[昌慶宮|昌慶宮]][[景春殿|景春殿]]。英祖三十五年，獲冊封為[[王世孫|王世孫]]。英祖三十八年五月十三日，其父莊獻世子遭英祖餓死於米櫃。英祖四十年二月二十日，英祖命李-{祘}-承[[孝章世子|孝章世子]]之嗣，次年命其代理聽政。

== 即位 ==
[[Image:Joseon-Portrait_of_Cha_Jegong-Black_danryeongpo.jpg|thumb]]]]
1776年（英祖五十二年）三月五日，英祖昇遐於慶熙宮集慶堂。三月十日，即位於西內[[慶熙宮|慶熙宮]]之崇政門。
後世認為正祖是極為仁孝的君王，因為朝鮮的禮法制度，他沒有辦法尊母親惠庆宫洪氏為[[王大妃|王大妃]]，但对其相當的孝順，为其屡上尊号。正祖在十一岁那年目睹了父亲[[莊献世子|莊献世子]]的惨死，即位后除了追尊宗法父亲孝章世子为真宗外，对亲生父亲的追慕活动也无休无止，先是将垂恩墓升格为永祐园，祠堂垂恩庙升格为景慕宫，并为其屡次加上尊号。正祖十三年，决定将永祐园迁至风水吉地——京畿道水原郡华山脚下，仿照世祖的光陵加以扩建，并更名为显隆园。正祖十八年在水原八达山麓开工建设[[水原华城|水原华城]]和水原行宫。正祖计划将华城修建为农、工、商兼备的城市，从南方拱卫汉城。

== 逝世 ==
1800年（正祖二十四年）六月二十八日，正祖去世于昌慶宮[[迎春軒|迎春軒]]。在位二十四年，終年四十九岁。葬於[[杨州|杨州]][[健陵|健陵]]。关于正祖之死，老论派散布的“毒杀”之说在岭南的南人党之间广泛流传，[[丁若镛|丁若镛]]的《与犹堂全书》对此也有相同记载。 在正祖去世前18天，即六月十日，他告诉众大臣，自己身上长出脓疮，并开始涂药去脓。之后病情逐渐恶化。六月十四日医官开了夹纸膏与杏仁膏，并制作加减逍遥散。但正祖将脓包自行诊断为心里的火气病，并饮用了两服[[白虎汤|白虎汤]]，之后全身开始出现滚热的症状。六月十六日正祖命令呈上四顺清凉饮、金莲茶与五颗[[牛黄|牛黄]]，之后又服用了乳粉托里散和三仁田螺膏、[[贝母|贝母]]膏、[[香薷|香薷]]调中汤。六月二十五日服用了龙脑安神丸和[[牛黄清心丸|牛黄清心丸]]，六月二十六日服用了琼玉膏、八物汤，吃了五钱[[人参|人参]]。六月二十八日服用了三钱人参，之后在昌庆宫永春轩接见左承旨金祖淳时病情突然恶化，以致说不出话来。此时贞纯王后命人送去[[藿香|藿香]]正气散，正祖服用后不久就在酉时（晚上6时）停止了呼吸。正祖死后，贞纯王后垂帘听政，解散了正祖创设的壮勇营，大举镇压天主教徒。

正祖的[[廟號|廟號]]原為正宗，[[諡號|諡號]]明道洪德顯謨文成武烈聖仁'''莊孝大王'''（清朝賜諡'''恭宣大王'''）。1899年（光武三年），高宗建立[[大韩帝国|大韩帝国]]后，追尊太祖及繼統的四世先祖（莊献世子／莊宗為莊祖，正宗為正祖，纯祖庙号不变，孝明世子／翼宗为文祖），将正宗追尊為'''正祖'''（정조）宣皇帝。

==朝政==
朝鮮正祖時期，[[天主教|天主教]]開始傳入朝鮮，1777年（正祖元年）秋，[[权哲身|权哲身]]、[[丁若铨|丁若铨]]等在[[京畿道|京畿道]]走鱼寺讲学修习天主教义。1784年正祖八年初，朝鮮冬至使书状官[[李承薰|李承薰]]在北京接受天主教的洗禮，成为朝鲜第一个天主教徒。在回國之後開始從事秘密傳教工作，暗中携带大量天主教书籍及圣画、圣像，并为李檗、权日身洗礼，到1784年秋已有数十人受洗。北京教区1790年派澳门人吴约翰抵达中朝边境凤凰城，未得朝鲜教徒迎接而折返。1793年，权哲身派燕行使译官尹有一、池璜再次恳请派神父，北京教区派中国苏州人[[周文謨_(清朝)|周文謨]]赴朝，他在中朝边境滞留10个月后，得尹有一、池璜等人的迎接化装成朝鲜人，于1795年初潜入朝鲜境内傳教，後由於天主教的勢力在朝鮮不斷增大，加上[[禮儀之爭|禮儀之爭]]的延燒，朝鮮政府不斷進行打压，而1801年更有「[[辛酉邪獄|辛酉邪獄]]」的發生。<ref>[http://218.188.3.99/Archive/periodical/ct/CT064/CT064L.htm 韓國教友與韓國天主教，黃德寬著] {{Wayback|url=http://218.188.3.99/Archive/periodical/ct/CT064/CT064L.htm |date=20140416183654 }}</ref>
<ref>[http://ntur.lib.ntu.edu.tw/bitstream/246246/79192/1/952411H002019.pdf 中華民國行政院國家科學委員會專題研究計畫期中進度報告-台灣天主教史與日本、韓國、越南的比較－以中華文化為參照系(2/3)]</ref>

1785年春发生[[乙巳秋曹摘发事件|乙巳秋曹摘发事件]]。大臣纷纷上疏称天主教为“邪教”政府進行打压。1786年，李承薰复建教会，封权日身为主教，崔昌显等人为神父，建立“假圣职制度”。1787发生丁未泮会事件。1791年发生[[辛亥邪狱|辛亥邪狱]]又称辛亥珍山事件。1795年天主教徒崔仁吉、尹有一、池璜被捕，被打死于捕盗厅。

朝鲜崇尚儒学，朝鲜正祖笃信程朱理学，认定天主教为“邪教”。他反复强调“正学明则邪说息”，“予意则使吾道大明，正学丕阐，则如此邪说可以自起自灭”，天主教是陆王心学、佛教道教一类，无需厉行禁止，只要儒生熟读儒家经典、不看杂书，便可迎刃而解。弘扬性理之学，加大管控中国进口书籍，严禁士人阅读明清文集稗官小说，焚毁相关书籍。不断有大臣上疏请求镇压天主教，正祖只在告发的情况下加以惩治，避免事态扩大化，左议政（后为领议政）南人蔡济恭说“佛道中派别”“禁之之道亦难矣”，<ref>《朝鲜王朝实录·正祖实录》卷26，十二年八月初三日</ref>，“必当自起而自灭，恐不足过费圣意矣”<ref>《朝鲜王朝实录·正祖实录》卷33，十五年十月二十五日</ref>。对南人天主教徒庇护。

== 評論 ==
朝鮮正祖被認為是朝鮮後期有作為的君主，在他在位期間，朝鮮開始出現復興。可惜在他去世後，[[貞純王后|貞純王后]]垂簾聽政，將他的改革幾乎廢除殆盡，朝鮮開始走向衰退。
