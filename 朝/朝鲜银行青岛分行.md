'''朝鲜银行青岛分行'''，时称{{lang|ja|'''朝鮮銀行青島支店'''}}，旧址位于[[中国|中国]][[山东省|山东省]][[青岛市|青岛市]][[市北区|市北区]][[馆陶路|馆陶路]]12号，建于1931-1932年，设计者为日本建筑师[[三井幸次郎|三井幸次郎]]。2003年列入青岛市历史优秀建筑名单，2006年成为[[山东省文物保护单位|山东省文物保护单位]][[青岛馆陶路近代建筑|青岛馆陶路近代建筑]]的一部分。<ref Name=dao>{{cite book |author=袁宾久 |title=《青岛老建筑之旅》|year=2016 |publisher=中国海洋大学出版社 |location=青岛 |isbn=9787567010093}}</ref><ref Name=you>{{cite book |author=青岛市史志办公室、青岛建筑委员会 |title=《青岛优秀建筑志》|year=2006 |publisher=青岛出版社 |location=青岛 |isbn=9787543625044}}</ref><ref Name=wen>{{cite book |author=青岛市史志办公室、青岛市文物局 |title=《青岛文物志》|year=2004 |publisher=中国出版社 |location=北京 |isbn=988-97146-6-1|ignore-isbn-error=true}}</ref><ref Name=cheng>{{cite book |author=宋连威 |title=《青岛城市老建筑》|year=2005 |publisher=青岛出版社 |location=青岛 |isbn=9787543632653}}</ref><ref Name=hua>{{Cite book|title=《画说青岛老建筑》|last=窦世强 李明|first=|publisher=青岛出版社|year=2010|isbn=978-7-5436-3134-2|location=青岛|pages=33}}</ref><ref Name=ren>{{Cite book|title=《人文青岛丛书：老楼故事》|last=鲁海|publisher=青岛出版社|year=2010|isbn=9787543600447|location=青岛}}</ref><ref Name=zhong>{{Cite book|title=《中国近代建筑总览·青岛篇》|last=徐飞鹏 张复合 村松伸 堀内正昭|publisher=中国建筑工业出版社|year=1992|isbn=7-112-01483-2|location=北京}}</ref>

==历史==
1914年11月日军[[青岛战役|占领青岛]]后，[[朝鲜银行|朝鲜银行]]开始筹划在青岛设立分支机构。1917年9月17日，朝鲜银行在青岛开设分行的计划经[[朝鲜总督府|朝鲜总督府]]认可开始着手筹备，并于同年10月22日正式成立青岛支店<ref Name=you/><ref Name=hua/>，10月27日，[[青岛守备军_(日本陆军)|青岛守备军]]司令{{link-ja|本乡房太郎|本郷房太郎}}向日本[[陆军大臣|陆军大臣]]{{link-ja|大岛健一|大島健一}}呈文，宣布允许朝鲜银行青岛分行开始营业。分行首任支店长为来自[[大连中山广场近代建筑群#朝鲜银行大连支店|朝鲜银行大连分行]]的片桐笃太郎，最初的行址位于河南町（今[[河南路_(青岛)|河南路]]北段）一番地的一栋两层商业建筑内<ref Name=jie>{{cite web |url=http://news.iqilu.com/shandong/shandonggedi/20180128/3823711.shtml|title=揭秘朝鲜银行青岛支店的兴与衰|accessdate=2018-02-06|date=2018-01-28|work=王栋|publisher=青岛早报 齐鲁网}}</ref>。

第一次日据时期的朝鲜银行青岛分行对青岛及周边地区的经济、金融有一定影响力，其存储业务仅次于[[横滨正金银行青岛分行|横滨正金银行青岛分行]]，同时在青岛获准发行[[朝鲜银行券|朝鲜圆]]，因纸币上的[[李氏朝鲜|朝鲜]]政治家[[金允植|金允植]]头像而俗称“老头票”<ref Name=ren/>。约1918至1919年间，分行行址迁至所泽町，今[[堂邑路|堂邑路]][[市场二路|市场二路]]路口东南角，紧邻[[青岛邮便局|青岛邮便局]]大楼<ref Name=jie/>。

1930年，朝鲜银行青岛分行计划修建新的银行大楼，并于1931年在[[馆陶路|馆陶路]][[上海路_(青岛)|上海路]]路口西北角（即今馆陶路12号）开始建设，由日本建筑师[[三井幸次郎|三井幸次郎]]设计，施工单位为[[大仓土木|大仓土木]]，1932年5月10日竣工<ref Name=cheng/><ref Name=zhong/>。1937年8月31日，分行因局势紧张关闭，次年1月20日复业。1940年3月13日，分行获[[大藏省|大藏省]]许可在奉天路（今[[辽宁路_(青岛)|辽宁路]]）设立派出所（即分支机构），8月12日开始营业。1943年9月4日经许可开设[[沧口|沧口]]出张所，次年4月10日营业<ref Name=jie/>。

1945年[[日本投降|日本投降]]后，朝鲜银行作为[[日治朝鲜|日治朝鲜]]的金融机构而撤出青岛，其分行馆舍由[[中央信托局|中央信托局]]接管。1947年9月，朝鲜银行青岛分行旧行址成为[[中华民国中央银行|中央银行]]青岛分行行址，黄家骧任经理。1949年，[[南京国民政府|南京国民政府]]发行[[银圆券|银圆券]]，中央银行青岛分行曾于5月14日发行“中央银行青岛分行银圆辅币券”，面额为5分、1角、5角三种，以[[青岛栈桥|青岛栈桥]]为图案，仅流通不足20天后，[[解放军|解放军]]便于6月2日占领青岛<ref Name=ren/>。该楼此后曾长期为金融机构所使用<ref Name=hua/>。

该建筑现为[[中国工商银行|中国工商银行]][[市北区|市北区]]办事处所使用<ref Name=zhong/>。2000年列入[[青岛市历史优秀建筑|青岛市历史优秀建筑名单]]<ref Name=you/>，2006年成为[[山东省文物保护单位|山东省文物保护单位]][[青岛馆陶路近代建筑|青岛馆陶路近代建筑]]的一部分<ref Name=dao/>。

==建筑特色==
[[File:中央银行青岛分行.png|thumb]]]]
朝鲜银行青岛分行旧址建筑为[[钢筋混凝土|钢筋混凝土]]结构，占地面积1653.34平方米，建筑面积1500平方米。平面呈“L”形，分为东、南两部分，两面临街<ref Name=cheng/>。东侧临[[馆陶路|馆陶路]]，地上二层，地下一层，南侧临[[上海路|上海路]]，地上三层。屋顶为平顶<ref Name=you/><ref Name=hua/>。

临馆陶路正立面为对称布局，墙面为深赭色釉面砖，[[花岗岩|花岗岩]]砌石勒脚。一楼为宽大的拱窗，装饰以花岗岩条石。正门位于中央，两侧各有一雕花石制立柱，支撑起一个石制雨棚。二楼则仅为无装饰的矩形并排窗洞。檐口有精细的花纹装饰，其纹样与正门[[柱头_(建筑)|柱头]]的花纹、[[青岛取引所|青岛取引所]]（同为三井幸次郎的设计作品）后门门楣的花纹极为相似<ref Name=you/><ref Name=hua/>。外立面整体装饰偏向简洁，构图受到[[现代主义建筑|现代主义]]影响<ref Name=cheng/>。

建筑内部一层为营业大厅，面积约650平方米，内有八根高7米的雕花石柱，天花板使用花纹图案装饰，过厅地面采用彩色石面，利用侧面的窗及顶部天窗采光<ref Name=you/><ref Name=cheng/>。一层两翼及二层为办公室及其他附属用房<ref Name=jie/>。

==文物破坏事件==
2017年7月10日，朝鲜银行青岛分行旧址正门两侧的雕花石柱及上方的石制雨棚被人为拆除，原因不明（可能是银行工作人员迷信[[风水|风水]]）。雨棚的石制构件被毁，去向不明；两根石柱分别断成三截，收藏家赵宝山从垃圾工人手中将其抢购下来，临时放置在龙华路1号院内，并将此事通知当地学者李明等人。7月13日，部分学者及文史爱好者将此事发布在微博等媒体上，并通知本地公众媒体及相关部门<ref>{{cite web|url=http://3g.163.com/news/article/CPAP1DM2000187UE.html|title=保护意识淡薄,朝鲜银行青岛支行旧址惨遭破坏|date=2017-07-14|accessdate=2017-07-16|work=青岛文物保护建筑|publisher=网易新闻|archive-url=https://web.archive.org/web/20170810090819/http://3g.163.com/news/article/CPAP1DM2000187UE.html|archive-date=2017-08-10|dead-url=yes}}</ref>。

当晚，房管员前去调查此事，工商银行人员谎称两根石柱是该行于1995年安装的。该行行长于7月14日上午至赵宝山处时又称拆除是因为“怕石柱倒了砸着客人”。青岛新闻网、青岛电视台等主流媒体于同日对此事进行了报导<ref>{{cite web |url=http://news.qingdaonews.com/qingdao/2017-07/14/content_12112820.htm|title=馆陶路朝鲜银行旧址门口石柱被拆 官方调查|date=2017-07-14|accessdate=2017-07-16|work=井水|publisher=青岛新闻网}}</ref>。

市文化市场行政执法局、市文物局会同市北区文广新局、综合行政执法局、市房产经营公司、即墨路街道办事处等有关部门于14日上午对朝鲜银行旧址进行了现场调查，确认工商银行市北区第二支行未经批准擅自拆除文物构件，对此事立案调查并启动问责机制，同时计划将受损建筑恢复原貌<ref Name=chao>{{cite web |url=http://news.qingdaonews.com/qingdao/2017-07/16/content_12114157.htm|title=“朝鲜银行青岛支行旧址”遭破坏问题初步核查及处理意见 |date=2017-07-16|accessdate=2017-07-16|work=|publisher=青岛新闻网}}</ref>。两根石柱残骸于7月15日自龙华路1号移至他处保存。

至7月17日，该事件已引起各地媒体广泛关注，相当多的媒体已对此进行报导<ref>{{cite web|url=http://yule.bandao.cn/news_html/201707/20170717/news_20170717_2746564.shtml|title="朝鲜银行青岛支行旧址"遭破坏 将原貌复原|date=2017-07-17|accessdate=2017-08-10|work=|publisher=半岛都市报}}</ref>，[[人民网|人民网]]亦发文谴责此事<ref>{{cite web|url=http://opinion.people.com.cn/n1/2017/0717/c1003-29410608.html|title=人民网评：明目张胆破坏文物，可以入刑么？|date=2017-07-17|accessdate=2017-08-10|work=李泓冰|publisher=人民网}}</ref>。

8月9日，青岛市文化市场行政执法局结束了对此案的调查，并根据《[[中华人民共和国文物保护法|中华人民共和国文物保护法]]》对[[中国工商银行|中国工商银行]][[市北区|市北区]]第二支行罚款[[人民币|人民币]]贰拾万元，对施工单位青岛汇丰装潢有限公司罚款人民币五万元，同时责成工商银行市北二支行按照整改方案对文物建筑进行修复。与此同时，工商银行青岛市分行党委对此事进行全系统通报，对支行负责人给予行政记过处分，并给予经济处罚三万元，同时对直接责任人给予行政撤职处分<ref>{{cite web|url=http://news.sina.com.cn/o/2017-08-10/doc-ifyixcaw3871129.shtml|title=重罚25万！青岛馆陶路“罗马柱”被拆一事结案！城市历史文脉不容任意毁坏！|date=2017-08-10|accessdate=2017-08-10|work=yiyi 金鱼|publisher=新浪网新闻中心}}</ref><ref>{{cite web|url=http://news.dailyqd.com/2017-08/10/content_399712.htm|title=馆陶路“朝鲜银行旧址被拆”案：涉事单位被罚25万|date=2017-08-10|accessdate=2018-01-28|work=王国钰|publisher=青岛日报 青报网}}</ref>。

12月22日，[[国家文物局|国家文物局]]公布“2017年度文物行政执法指导性案例”共15例，“中国工商银行股份有限公司青岛市北第二支行擅自修缮山东省文物保护单位朝鲜银行青岛支行旧址案”名列其中<ref>{{cite web|url=http://www.xinhuanet.com/legal/2017-12/23/c_1122155343.htm|title=国家文物局昨日回应：对长城“微破坏”将细化执法 |date=2017-12-23|accessdate=2018-10-28|work=赵婷婷|publisher=新华网 北京青年报}}</ref>。
