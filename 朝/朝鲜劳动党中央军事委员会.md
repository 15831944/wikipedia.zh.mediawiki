
{{朝鲜政治}}

'''朝鲜劳动党中央军事委员会'''（{{lang-ko|조선로동당 중앙군사위원회}}）是[[朝鲜劳动党|朝鲜劳动党]]的最高军事机关，根据《[[朝鲜劳动党章程|朝鲜劳动党章程]]》的规定，统帅[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]所有武装力量，指导所有国防事业，为常设最高军事机关<ref name=":0">{{Cite web|url=http://www.wyzxsx.com/Article/Class10/200908/99336.html|title=朝鲜劳动党章程|accessdate=2018-03-28|author=|date=|publisher=|archiveurl=https://web.archive.org/web/20101006134621/http://www.wyzxsx.com/Article/Class10/200908/99336.html|archivedate=2010-10-06}}</ref>。领导人为[[朝鲜劳动党中央军事委员会委员长|中央军事委员会委员长]]，根据现行《朝鲜劳动党章程》第22条，[[委员长|委员长]]由[[朝鲜劳动党委员长|朝鲜劳动党委员长]]担任，现任委员长为朝鲜最高领导人[[金正恩|金正恩]]。

== 职责 ==
《[[s:朝鲜劳动党章程|朝鲜劳动党章程]]》第27条规定：“中央军事委员会讨论决定贯彻党的军事政策的措施，组织领导加强人民军等所有武装力量、发展军需生产的有关工作，统率我国的军事力量。”<ref name=":0" />但是在[[金正日|金正日]]时代实际被[[朝鲜民主主义人民共和国国防委员会|国防委员会]]取代了最高统帅部的地位，直至[[金正恩|金正恩]]被任命为中央军事委员会副委员长，确认为[[金正日|金正日]]的接班人。金正恩掌权以后中央军委的权力大幅扩张，甚至成为劳动党的权力核心<ref>{{Cite web|url=http://www.nanzao.com/tc/international/15060632c4e909b/mei-zhuan-jia-jin-zheng-en-zhi-xia-zhong-yang-jun-wei-jian-bian-zui-gao-jue-ce-ji-gou|title=美專家：金正恩治下 中央軍委漸變最高決策機構|accessdate=2018-03-28|date=2015-10-13|publisher=南早中文|archiveurl=https://web.archive.org/web/20160304223504/http://www.nanzao.com/tc/international/15060632c4e909b/mei-zhuan-jia-jin-zheng-en-zhi-xia-zhong-yang-jun-wei-jian-bian-zui-gao-jue-ce-ji-gou|archivedate=2016-03-04}}</ref>。

== 人事变动 ==
*[[金正日|金正日]]于1980年[[朝鲜劳动党第六次全国代表大会|朝鲜劳动党第六次全国代表大会]]时当选朝鲜劳动党中央军事委员会委员。1991年12月[[金正日|金正日]]被[[朝鲜劳动党六届十九中全会|朝鲜劳动党六届十九中全会]]拥戴为[[朝鲜人民军|朝鲜人民军]]最高司令官。但是尚无任何公开的会议记录表明金正日是在哪一年接任朝鲜劳动党中央军事委员会委员长（前任是金正日的父亲[[金日成|金日成]]）。朝鲜在1997年拥戴[[金正日|金正日]]出任[[朝鲜劳动党总书记|朝鲜劳动党总书记]]时候是通过首先召开[[朝鲜人民军|朝鲜人民军]]党代会，然后召开中央机关和地方党代会的形式决定的，最后由[[朝鲜劳动党中央委员会|朝鲜劳动党中央委员会]]和朝鲜劳动党中央军事委员会发布特别报道。此时也并无金正日出任朝鲜劳动党中央军事委员会委员长的记录。金正日第一次使用“朝鲜劳动党中央军事委员会委员长”是在2009年2月11日任命[[金永春|金永春]]为[[朝鲜民主主义人民共和国人民武装力量部|人民武装力量部]]部长时，在任命书上署名为“[[朝鲜民主主义人民共和国国务委员会|国防委员会]]委员长、朝鲜劳动党中央军事委员会委员长[[金正日|金正日]]”。
*2009年2月11日，[[朝鲜民主主义人民共和国|朝鲜]]最高领导人金正日在任命[[金永春|金永春]]（朝鲜劳动党中央军事委员会委员、[[朝鲜人民军|朝鲜人民军]]总参谋长）为[[朝鲜民主主义人民共和国人民武力省|朝鲜民主主义人民共和国人民武装力量部]]部长时，使用的是“国防委员会委员长、劳动党中央军事委员会委员长”的名义。
*2009年2月20日，[[朝鲜民主主义人民共和国|朝鲜]]最高领导人金正日在任命[[吴克烈|吴克烈]]（朝鲜劳动党中央军事委员会委员）为[[朝鲜民主主义人民共和国国防委员会|朝鲜民主主义人民共和国国防委员会]]副委员长时，使用的是“国防委员会委员长、劳动党中央军事委员会委员长”的名义。<ref>{{Cite web|url=http://world.people.com.cn/GB/1029/42354/8842940.html|title=朝鲜再次宣布军方高层人事变动|accessdate=2018-03-28|date=2009-02-20|publisher=人民网}}</ref>

*2010年4月，[[朝鲜劳动党中央委员会组织指导部|朝鲜劳动党中央组织指导部]]第一副部长，中央军事委员会委员[[李勇哲|李勇哲]]去世。<ref>{{Cite web|url=http://cnnews.chosun.com/site/data/html_dir/2010/04/27/20100427000007.html|title=北韩劳动党高官李勇哲因病去世|accessdate=2018-03-28|date=2010-04-27|publisher=朝鲜日报中文网|archiveurl=https://web.archive.org/web/20160305062555/http://cnnews.chosun.com/site/data/html_dir/2010/04/27/20100427000007.html|archivedate=2016-03-05}}</ref>
*2010年5月，根据[[朝鲜民主主义人民共和国国务委员会|朝鲜民主主义人民共和国国防委员会]]的命令，国防委员会委员[[金一哲|金一哲]]被免去了国防委员会委员、人民武装力量部第一副部长职务，但是尚不清楚是否免去其朝鲜劳动党中央军事委员会委员职务。

== 现任组成人员 ==
[[朝鲜最高领导人|朝鲜最高领导人]]、[[朝鲜劳动党委员长|朝鲜劳动党委员长]][[金正恩|金正恩]]元帅是朝鲜劳动党中央军事委员会委员长。

2010年9月28日举行的[[朝鲜劳动党第三次代表会议|朝鲜劳动党中央委员会2010年9月全体会议]]选出了新一届朝鲜劳动党中央军事委员会。此次全会选举[[金正日|金正日]]为朝鲜劳动党中央军事委员会委员长，[[金正恩|金正恩]]、[[李英浩|李英浩]]为副委员长，还选举出[[金永春|金永春]]、[[金正角|金正角]]、[[金明国|金明国]]、[[张成泽|张成泽]]等16名朝鲜劳动党中央军事委员会委员。

2012年4月11日举行的[[朝鲜劳动党第四次代表会议|朝鲜劳动党第四次代表会议]]上又选举[[崔龙海|崔龙海]]为副委员长，[[玄哲海|玄哲海]]、[[李明秀|李明秀]]、[[金乐谦|金乐谦]]为委员。

2012年7月15日[[朝鲜劳动党中央委员会政治局|朝鲜劳动党中央政治局]]会议上，中央军委副委员长[[李英浩|李英浩]]被以健康原因为由罢免一切职务。2013年12月8日朝鲜劳动党中央政治局扩大会议上，中央军委委员、国防委员会副委员长张成泽被以反党反革命为由罢免一切职务。

2014年4月朝鲜劳动党中央军事委员会扩大会议举行，会议后，崔龙海不再担任朝鲜人民军总政治局局长职务，由[[黄炳誓|黄炳誓]]接任。
