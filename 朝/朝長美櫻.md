'''朝長美櫻'''（{{lang-ja|朝長美桜}}，{{bd|1998年|5月17日|||}}），出身於[[福岡|福岡]]，是日本女子偶像組合[[HKT48|HKT48]] Team KIV的成員之一，並且兼任[[AKB48|AKB48]] Team 4，經紀公司為AKS。<ref>{{Cite web |url = http://www.hkt48.jp/profile/tomonaga_mio.html |title=朝長 美桜 < プロフィール | HKT48 OFFICIAL WEB SITE}}</ref>

==經歷==
; 2012年
朝長在2012年6月23日通過HKT48甄選，成為21名第2期生之一，並於同年9月23日在HKT48劇場首次亮相，演唱《[[想見你_(AKB48單曲)|想見你]]》一曲。

2012年12月，AKB48發行第29張單曲《[[永遠的壓力|永遠的壓力]]》，其中收錄了HKT48第一首原創歌曲《[[初戀蝴蝶|初戀蝴蝶]]》（{{lang-ja|初恋バタフライ}}），朝長入選此歌選拔。其後HKT48於2013年3月發行的出道單曲《[[喜歡！喜歡！小跳步！|喜歡！喜歡！小跳步！]]》同樣入選選拔組。此單曲只有3名2期研究生入選選拔組，包括朝長、[[田島芽瑠|田島芽瑠]]及[[淵上舞_(歌手)|淵上舞]]。

AKB48第31張單曲《[[再見自由式|再見自由式]]》於2013年5月發行，朝長與同期的田島芽瑠首次進入AKB48 A面曲的選拔組。

; 2013年

朝長在[[AKB48第32張單曲選拔總選舉|第五屆選拔總選舉]]中取得第59名，成為Future Girls。其後她亦在HKT48猜拳大會預賽中擊敗[[指原莉乃|指原莉乃]]，進入AKB本戰。<ref>[http://enjoy.eastday.com/c8/2013/0711/3045582291.html 指原莉乃參加AKB第4次猜拳大會 HKT預賽失利]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://www.girlsnews.tv/unit/90702 HKT48 田中菜津美・宮脇咲良・朝長美桜の3名がじゃんけん大会予備戦を突破！]</ref>朝長更獲指原的支持：「希望她得到第1，我把自己的運氣全部給她！」<ref>[http://kenshin.hk/2013/07/11/【日藝】akb48第4屆猜拳大會hkt48初選，總選冠軍指原莉乃/ 【日藝】AKB48第4屆猜拳大會HKT48初選，總選冠軍指原莉乃敗退]</ref>

營運部於7月20日的福岡巨蛋演唱會首日中宣布成立新的衍生團體「[[小瓢蟲Chu!|小瓢蟲Chu!]]」（{{lang|ja|てんとうむChu！}}），並推出歌曲《[[真心电流|只給你的Chu! Chu! Chu!]]》（{{lang|ja|君だけにChu! Chu! Chu!}}）<ref>[http://www.oricon.co.jp/news/music/2027153/full/ AKB研究生7人組ユニット・てんとうむChu！お披露目　新曲熱唱]</ref>，成員為AKB48及各姊妹團中所選中的研究生，當中朝長與同屬HKT48的田島，AKB48的[[小嶋真子|小嶋真子]]、[[西野未姫|西野未姫]]、[[岡田奈奈_(AKB48)|岡田奈奈]]，SKE48的[[北川綾巴|北川綾巴]]，與NMB48的[[澀谷凪咲|澀谷凪咲]]一同入選。<ref>[http://natalie.mu/music/news/95446 AKB48、ドーム初日にHKT48新曲＆研究生ユニット発表]</ref>朝長在得悉自己被選後表示驚訝，「為什麼是我？我唱歌和跳舞也不是太好呢，想了很久（也不知道選我的原因）」<ref>[http://woman.infoseek.co.jp/news/entertainment/hwchannel_20130726_2581503 初期のなっち似？朝長美桜がHKT48センターに　研究生の「下剋上」がアツい！] {{webarchive|url=https://web.archive.org/web/20131013182622/http://woman.infoseek.co.jp/news/entertainment/hwchannel_20130726_2581503 |date=2013-10-13 }}</ref>。

朝長在HKT48於9月4日發行的第二首單曲《[[甜瓜汁_(HKT48單曲)|蜜瓜汁]]》（{{lang-ja|メロンジュース}}） 中再次入選單曲選拔組，並與同為2期研究生的田島芽瑠一同擔當中央位置成員（{{lang|ja|センター}}，center）<ref>[http://getnews.jp/archives/383842 HKT48新曲「メロンジュース」はグループ初の田島＆朝長ダブルセンター]</ref><ref>[http://www.billboard-japan.com/d_news/detail/13944 HKT48 新作は田島芽瑠、朝長美桜のダブルセンター]，[[Billboard|Billboard Japan]]</ref>。除了是朝長個人首次站在中央位置外，這也是HKT48第二次以研究生擔任單曲中央位置成員，與48系首次全以研究生擔任單曲雙中央成員的紀錄<ref>[http://topics.jp.msn.com/entertainment/movie/article.aspx?articleid=1979716 初期のなっち似？朝長美桜がHKT48センターに　研究生の「下剋上」がアツい！]</ref>。

; 2014年

1月11日，於大分縣{{link-ja|iichiko剧场|大分県立総合文化センター#グランシアタ}}“[[HKT48九州7县巡演～和可爱的孩子们一起旅行～|HKT48九州7县巡演～和可爱的孩子们一起旅行～]]”的首日夜公演中，朝長升格到新Team KIV<ref>{{cite web|url=http://natalie.mu/music/news/107576 |script-title=ja:HKT48ツアー初日に“新クラス”と研究生17名昇格を発表 |publisher=[[Natalie_(网站)|Natalie]] |language=ja |date=2014-01-12 |accessdate=2014-02-05}}</ref>。

2月24日，於「[[AKB48集團大組閣祭|AKB48集團大組閣祭]]」中宣布留任Team KIV，另外新增兼任AKB48 Team B<ref>{{cite web|url=http://www.akb48.co.jp/page/daisokaku/list.php?after=1&group=akb48 |script-title=ja:AKB48グループ大組閣祭り |publisher=AKB48官方網站 |language=ja |date=2014-02-24 |accessdate=2014-02-26 |deadurl=yes |archiveurl=https://web.archive.org/web/20140227172449/http://www.akb48.co.jp/page/daisokaku/list.php?after=1&group=akb48 |archivedate=2014-02-27 }}</ref>。

第三張單曲《[[櫻花，大家一起來品嚐|櫻花，大家一起來品嚐]]》（{{lang|ja|桜、みんなで食べた}}），朝長再次與田島一同擔任A面曲《櫻花，大家一起來品嚐》的雙center<ref name="Excite_20140208">{{cite web|url=http://www.excite.co.jp/music/news/detail/Dwango_5469.html |language=ja |script-title=ja:HKT48「桜、みんなで食べた」センターは田島、朝長！Mステ単独出演も決定 |publisher=Excite Japan（[[多玩國|多玩國]]）| date=2014-02-08 |accessdate=2014-02-15}}</ref>。

朝長在[[AKB48第37張單曲選拔總選舉|第六屆選拔總選舉]]中获得23,766票取得第27名，成為Under Girls。

; 2015年

於「[[AKB48第41張單曲選拔總選舉|AKB48第41張單曲選拔總選舉]]」中以28,197票獲得第21名，成為Under Girls。
