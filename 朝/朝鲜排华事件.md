'''朝鲜排华事件'''，又称'''朝鲜内排华事件'''{{sfn|日本外务省|p=217}}<ref>{{cite book |author1=竹中宪一 |title=満州における教育の基礎的研究 |date=2000 |publisher=柏书房 |page=70}}</ref>、'''朝鲜华侨惨案'''<ref name=":2">{{cite web |author1=蒋中正 |title=告全國同胞一致安內攘外書 |url=https://zh.wikisource.org/wiki/告全國同胞一致安內攘外書 |date=1931-7-23 |access-date=2018-11-08 |archive-url=https://web.archive.org/web/20170928072519/https://zh.wikisource.org/wiki/%E5%91%8A%E5%85%A8%E5%9C%8B%E5%90%8C%E8%83%9E%E4%B8%80%E8%87%B4%E5%AE%89%E5%85%A7%E6%94%98%E5%A4%96%E6%9B%B8# |archive-date=2017-09-28 |dead-url=no }}</ref>、'''朝鲜事件'''{{sfn|日本外务省|p=219}}，是1931年7月发生在[[朝鲜半岛|朝鲜半岛]]的朝鲜人袭击[[唐人街|唐人街]]事件{{sfn|日本外务省|p=217}}，也被认为是[[万宝山事件|万宝山事件]]的一部分。根据[[李頓報告|李顿报告]]，朝鲜排华事件中朝鲜人共杀害了127名中国人<ref>{{cite book |author1=宮脇淳子 |title=世界史のなかの満洲帝国 |publisher=PHP研究所}}</ref>。

== 背景 ==
[[File:Flume_at_Wangpaoshan,_constructed_by_Koreans'_efforts.jpg|缩略图]]
{{See also|万宝山事件}}
在[[乙巳條約|乙巳条约]]签订后建立起[[統監府|统监府]]的[[大韓帝國|大韩帝国]]，约有3,361名华侨；[[朝鮮日治時期|日治时代]]华侨的数量增加到1930年的67,794人，增加的人数中80%来自山东<ref name=":0">{{Cite journal|title=現代中国研究　第19号 「書評 安井三吉著『帝国日本と華僑－日本・台湾・朝鮮』」|author=陈来幸|url=http://modernchina.rwx.jp/magazine/19/chin.pdf|journal=现代中国研究|publisher=中国现代史研究会|issue=19|doi=|others=|year=|volume=|page=76|pmid=|access-date=2018-11-07|archive-url=https://web.archive.org/web/20160304083029/http://modernchina.rwx.jp/magazine/19/chin.pdf#|archive-date=2016-03-04|dead-url=no}}</ref>。如此数量华侨的存在，为排华事件提供了前提<ref name=":0" />。1927年12月，由于朝鲜人在华遭到迫害，朝鲜各地都发生了大规模的排华事件。而1931年的排华事件，是当时定居在[[满洲国|满洲国]][[新京|新京]]（今[[长春市|长春市]]）的朝鲜人在未得到周边中国居民的同意下擅自修建水渠引发争议、中华民国警察和负责保护朝鲜人的日本警察发生的枪击事件所引起。<ref name=":1">{{Cite book|title=幣原喜重郎とその時代|last=冈崎|first=久彦|publisher=PHP研究所|year=2003|isbn=|location=|pages=}}</ref>

原本根据《[[間島協約|间岛协约]]》，日本人和朝鲜人享有租借土地务农的权利，但中方的地主郝勇德在没有中国政府的允许下就私自把土地租赁给朝鲜人。朝鲜人在租来的土地上修建水渠，破坏了当地中国农民的农业生产，使得约400名中国农民开始破坏该水渠。此事件在政府没有针对各国农民未来利益的保障下便结束，这成为致力于使中国人、日本人和朝鲜人在满洲进行平等农业活动的日本时任外务大臣[[幣原喜重郎|币原喜重郎]]提出的“[[幣原外交|幣原外交]]”方针陷入僵局的原因之一<ref name=":1" />。
== 經過==
[[File:Anti-China_riot_in_Heijo.JPG|缩略图]]

受到对万宝山事件错误的夸大报道刺激的朝鲜人{{sfn|日本外务省|p=229}}，以日本设立的[[京城府|京城府]]（今[[首爾|首尔]]）和[[仁川廣域市|仁川]]为中心，在朝鲜半岛各地发起了袭击华人的事件{{sfn|日本外务省|p=217}}。当1931年7月3日凌晨1时仁川发生轻微的暴力事件时，朝鲜总督府立即向各道发出警告{{sfn|日本外务省|p=237}}。由于局势恶化，总督府前后共发出3次警告，但[[平壤|平壤]]仍爆发了重大事件{{sfn|日本外务省|p=237}}。

7月4日晚9时30分，100余名朝鲜人在京城府外新堂里（今首尔[[新堂洞|新堂洞]]）和约40名华人发生冲突，1名朝鲜人死亡，1名华人重伤{{sfn|日本外务省|p=228}}。其他多地也发生了对华人的房屋放火的事件。自当晚11时起，仁川爆发了数千名群众反抗警察并破坏了外里派出所，切断其电线，使暴动更加激化。为此，总督府加强了警察的武装，严格警戒，最终在次日凌晨3点得以解散人群{{sfn|日本外务省|p=228}}。

7月5日，总督府通知各新闻通讯社不得发布刺激、煽动民心的报道{{sfn|日本外务省|pp=213-214}}。自当日晚9时至6日晨3时，平壤爆发了数千朝鲜人袭击当地唐人街的事件。此事造成华人88名死亡、102名受伤{{sfn|日本外务省|p=217}}。而警方在未设法阻止事件时开枪，导致1名朝鲜人身亡{{sfn|日本外务省|pp=232-233}}。为防止事件进一步恶化，总督府从驻屯当地的[[朝鮮軍_(日本陸軍)|朝鲜军]]中调遣了51名官兵{{sfn|日本外务省|p=232}}。此外，又增援了115名警察和350名武装警察，但有35名警察受伤。在仁川有30名警察和5匹马受伤{{sfn|日本外务省|p=237}}。总督府在此次暴动中收容了一些被害者{{sfn|日本外务省|p=237}}。

7月6日，中华民国驻京城领事要求中华民国政府及驻日大使请求日本警方立即武装镇压暴动。{{Sfn|日本外务省|p=217}}

7月7日，平壤、大同警察署，医学讲习所，道立医院和平壤府以及中国人开办的工厂已收容4,000名华人避难者{{sfn|日本外务省|p=228}}。但是，下午4时30分左右朝鲜人袭击了工厂，警察在执法中意外射杀一名朝鲜人{{sfn|日本外务省|p=228}}。骚乱的几天之内，华人开办的所有商店都遭到朝鲜人袭击，华人居住的房屋也遭到了破坏。<ref>{{Cite news|url=|title=血に狂ふ虐殺 支那商店をことごとく襲撃し 平壌暴行事件の詳報|author=|date=1931-07-08|work=北国新闻|accessdate=}}</ref><ref>{{Cite news|url=|title=平壌の鮮支人衝突 昨夜来大暴動下す 鮮人五千の大集団襲撃し 支那人家屋百数十軒を破壊 死傷者二百二十五名|author=|date=1931-07-07|work=北国新闻|accessdate=}}</ref><ref>{{Cite news|url=|title=支那人の死傷者 三百余名に達す 朝鮮憲兵隊の調査|author=|date=1931-07-07|work=北国新闻|accessdate=}}</ref>

== 影响 ==
[[File:Chinese_anti-Japanese_poster_published_after_the_revenge_by_Koreans.jpg|缩略图]]
排华事件的影响并非仅限于朝鲜半岛内，日本也发生了[[在日朝鲜人和韩国人|在日朝鲜人]]和[[日本華人|在日华人]]的冲突。<ref>{{Cite news|url=|title=金沢の朝鮮人支那人を襲ふ 二軒の理髪業者の家で暴行 万宝山事件の報道に雷同|author=|date=1931-07-09|work=大阪每日新闻|accessdate=}}</ref><ref>{{Cite news|url=|title=富山市でも鮮人 支那行商人を襲ふ 例の萬宝山事件に恨みを抱き 昨夜押入って暴行|author=|date=1931-07-11|work=北国新闻|accessdate=}}</ref><ref>{{Cite news|url=|title=日本内地でも暴行 鮮人支那理髪店を襲ふ 棍棒で鏡電灯破壊 金沢駅前の椿事|author=|date=1931-07-09|work=福冈日日新闻|accessdate=}}</ref>

7月9日，中华民国外交部长[[王正廷|王正廷]]在南京召见日方外交人员，日方解释称正致力于解决事件，中方则要求日方用足够的警力保护在日华人{{Sfn|日本外务省|p=229}}。

7月11日，朝鲜总督府宣告已有100名华人死亡、120名华人受伤{{Sfn|日本外务省|pp=232-233}}，而现在亦有141名华人死亡的说法<ref name=":0" />。至于平壤一地的死亡人数，日本外交家[[岡崎久彥|冈崎久彦]]认为超过100名<ref name=":1" />；太平洋战争研究会则认为，死者有109名，伤者有160余名，还有63人生死不明<ref>{{Cite book|title=石原莞爾と満州事変|last=太平洋战争研究会|first=|publisher=PHP研究所|year=2009|isbn=4569774652|location=|pages=108}}</ref>。

7月15日上午11时，报道万宝山事件的《[[朝鮮日報|朝鲜日报]]》满洲长春支局长金利三（本名金永锡{{Sfn|日本外务省|p=248}}）遭到朝鲜刺客上门以3发手枪弹袭击{{Sfn|日本外务省|p=246}}。得知此事的日本驻吉林领事馆派{{Tsl|ja|領事館警察}}朴昌厦赶到事发地点，被先行赶到的中国警察误认为行凶者并被送到警署，后移送到交涉署{{Sfn|日本外务省|p=246}}。此事件引起中国群众集体抗议，前往维持秩序的中国警察试图鸣枪以解散群众，但演变成了射杀1名中国人的惨剧{{Sfn|日本外务省|p=246}}。事后两名伤者被立即送往[[南滿洲鐵道|南满洲铁道]]经营的东洋医院，金利三于当日下午1时死亡{{Sfn|日本外务省|p=247}}。日本领事馆方面，领事馆书记小森和署长田中赶往交涉署，经调查后发现朴昌厦所持手枪是全新的，枪管内没有灰尘，故当日即获释放{{Sfn|日本外务省|p=247}}。然而7月16日的《吉长日报》仍报道称“鮮東亜報記者過害、行兇日巡査韓人朴昌厦”，对此日方要求中方逮捕犯人并消除文章的影响{{Sfn|日本外务省|p=247}}。

7月23日，蒋介石在对全体国民的声明中，将这一朝鲜排华事件称为朝鲜华侨惨案<ref name=":2" />。
