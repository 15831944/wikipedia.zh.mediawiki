'''朝鮮日報'''是[[韓國|韓國]]的主要[[報紙|報紙]]之一，以[[朝鮮語|朝鮮語]]發行，每日[[發行量|發行量]]高達230萬份。《朝鲜日报》创办于[[三一独立运动|三一独立运动]]之后的1920年，是韩国历史最悠久的报纸之一<ref name=ksh>{{cite web |url = http://chinese.korea.net/AboutKorea/Korea-at-a-Glance/Society |title=社会 |publisher = KOREA.NET |accessdate=2014-11-30}}</ref>。與《[[東亞日報|東亞日報]]》、《[[中央日报_(韩国)|中央日報]]》并称為韓國三大報紙。該報的立場趨向於韓國[[保守派|保守派]]的觀點。

朝鮮日報网络中文版具有中文服務的線上新聞，提供给使用[[華語|華語]]的互联网用户浏览。

== 历史 ==
*1920年　日本[[朝鲜总督府|朝鲜总督府]]在新的文化政策下允许《朝鲜日报》、《[[东亚日报|东亚日报]]》等[[朝鮮文|朝鮮文]]日报发行。
*1940年　朝鲜总督府取消《朝鲜日报》等朝鲜文日报发行。
*1945年　日本戰敗結束在朝殖民統治後，《朝鲜日报》复刊。
*1968年　《朝鲜日报》发行《[[周刊朝鲜|周刊朝鲜]]》杂志。
*1980年　《朝鲜日报》发行《[[月刊朝鲜|月刊朝鲜]]》杂志。
