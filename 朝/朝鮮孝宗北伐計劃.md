'''朝鮮孝宗北伐計劃'''[[朝鲜孝宗|朝鲜孝宗]]李淏北伐[[清朝|清朝]]的计划。该计划最终并未实施。但是却起到增加政府财政收入的结果。
== 背景 ==
[[天启|天启]]六年（仁祖四年，1626年），七岁的李淏被封为'''凤林大君'''。1636年[[丙子战争|丙子战争]]，[[朝鲜|朝鲜]]战败，次年初李淏和[[哥哥|哥哥]][[昭显世子|昭显世子]]李溰、[[弟弟|弟弟]][[麟坪大君|麟坪大君]]李濬等数十万朝鲜人被[[清军|清军]]掳到[[沈阳|沈阳]]，1644年清军入关，11月9日，清朝[[摄政王|摄政王]][[多尔衮|多尔衮]]在[[北京|北京]][[紫禁城|紫禁城]][[武英殿|武英殿]]召见昭显世子和凤林大君，说“未得北京以前，两国不无疑阻。今则大事已定，彼此一以诚信相孚。且世子以东国储君，不可久居于此，今宜永还本国。凤林大君则姑留与麟坪大君相替往来”<ref>《朝鲜王朝实录·仁祖实录》卷45，二十二年十二月初四日</ref>。宣布减少朝鲜的岁贡币物。1645年3月，昭显世子等返回汉城，随行清使勒令朝鲜仁祖李倧出城迎接。父迎子不合儒教礼法，且承认清朝敕使代表天子，所以朝鲜[[两班|两班]][[士大夫|士大夫]]坚决反对。5月21日，昭显世子暴毙[[昌庆宫|昌庆宫]]欢庆殿，疑似被宫人在饵饼中下毒（“举体尽黑，七窍皆出鲜血”<ref>《朝鲜王朝实录·仁祖实录》卷46，二十三年六月二十七日</ref>）。李倧上报清朝“病亡”。6月7日，凤林大君回到汉城。11月14日，清朝册封李淏为朝鲜[[世子|世子]]。1649年己丑五月十三日，仁祖大王去世，李淏即位于昌德宫之仁政门,死后庙号孝宗。

经过1627年和1636年的两次战争，[[朝鲜王朝|朝鲜王朝]]对清朝产生了很深的仇恨。李朝上下皆视清朝为犬羊夷狄，私下称清帝为“胡皇”，称清使为“虏使”。除对清朝的公文贺表之外，一切内部公文，包括王陵、宗庙、文庙祭享祝文，仍用[[崇祯|崇祯]]年号。
== 准备 ==
朝鲜君臣认为，“我朝三百年来，服事大明，其情其义，固不暇言。而神宗皇帝再造之恩，自开辟以来，亦未闻于载籍者。宣祖大王所谓义则君臣，恩犹父子，实是真诚痛切语也”。<ref>孝宗 19卷, 8年(1657 丁酉 / 청 순치(順治) 14年) 10月 25日(甲午) </ref>孝宗则以光复大明天下为己任，倡议北伐。
=== 扩军备战 ===
都城御营厅军由7000人增加到21000人。禁军由600名增加到1000名，全部改编为[[骑兵|骑兵]]。御营厅增加了[[大炮|大炮]]，还计划将守卫[[汉城|汉城]]的训练都监军增加10000名，御营厅军增加20000名。由于财政困难，未能实现。<ref>韩国姜万吉《韩国近代史》</ref>
=== 加大税收 ===
孝宗加快了[[朝鲜宣祖|朝鲜宣祖]]时代出现的大同法。大同法将贡物统一为米穀，该制度加快了朝鲜国内贸易的兴起和商品经济的发展。宰相金育大力主张推行大同法，对土地征收附加税。孝宗命令，所有男子，甚至[[和尚|和尚]]，都要纳税以换取免服兵役。

=== 制造借口 ===
1650年，孝宗向清廷奏报“[[日本|日本]]近以密书示通事，情形可畏，请筑城训练为守御计”，以防御日本为由扩军备战。清朝派遣密使前往朝鲜核实情况，查明朝鲜与日本素和好，奏折不实，[[顺治帝|顺治帝]]下诏斥责朝鲜国王，罢其用事大臣，为六使诘责事件。
== 计划流产 ==
朝鲜财政困难，军备薄弱，未能完成备战，而其防御日本借口又引起顺治帝警惕，1659年五月初四，孝宗死於昌德宮大造殿，北伐計劃未能实施。

== 对與清朝关系影响 ==
清朝当时正处于全面上升时期，孝宗的北伐计划没有可操作性。北伐计划加重了朝鲜的财政负担，延缓了雙方关系改善的步伐。

== 其他北伐论 ==
朝鲜早就有类似思想，他们确信“胡人无百年之运”，将地震、彗星等视为清朝灭亡的征兆，对[[南明|南明]]、[[三藩|三藩]]、[[東寧|東寧]]、[[准噶尔|准噶尔]]等反清势力寄以厚望，准备派使臣渡海联络。[[東寧|東寧]]多次请日本联合出师伐清复明，朝鲜对日郑联合乐观其成，[[朝鲜仁祖|朝鲜仁祖]]建议“假道朝鲜，出送援兵”。<ref>《仁祖大王实录》二十四年十二月甲午</ref>。后来[[朝鲜显宗|朝鲜显宗]]说：“群臣皆欲予勿治兵，而予固不听者，天时人事，不知何日是好机会来时。故欲养精兵十万，爱恤如子，皆为敢死之卒，然后待其有衅，出其不意，直抵关外，则[[中原|中原]]义士豪杰，岂无响应者！ 蓋直抵關外, 有不甚難者。 虜不事武備, 遼瀋千里, 了無操弓騎馬者, 似當如入無人之境矣。 且以天意揣之, 我國[[歲幣|歲幣]], 虜皆置之遼瀋, 天意似欲使還爲我國用矣。 且我國被擄人, 不知其幾萬, 亦豈無內應者耶? 今日事, 惟患其不爲而已, 不患其難成。”。“以大志举大事，岂可保其万全也。大义则明，则覆亡何愧，益有光举于天下万世也。且天意有在，予以为似无覆亡之虞也”。<ref>李朝实录 顯改 1卷, 卽位年(1659 己亥 / 청 순치(順治) 16年) 9月 5日(癸亥) </ref>

朝鲜否定清朝统治下的[[中国|中国]][[文化|文化]]和[[经济|经济]]发展，还几乎全面封锁了对[[中国文化|中国文化]]的引进，卻也使得明朝時代的中原文化特色完整的留在韓國。[[明朝|明朝]]灭亡后，李朝王室一直进行各种追思活动，1704年甲申，明朝灭亡60周年，[[朝鲜肃宗|朝鲜肃宗]]自宜春门诣禁苑坛，以太牢祭祀[[崇祯帝|崇祯帝]]。又命汉城府在后苑春塘台设“[[大报坛|大报坛]]”，祭祀[[明神宗|明神宗]]。“大报”出于《礼记》郊特牲，是郊天之义，且兼有报德之意。1749年（乾隆十四年）又以[[明太祖|明太祖]]、神宗、毅宗并享大报坛，并于三帝即位、忌辰日行望拜礼。这种祭祀活动每年进行，直到李朝末年。

朝鲜对清朝的鄙视仇恨和恪守藩属朝贡制度同时并存，且反清复明计划和北伐准备是在隐蔽状态下进行的，清朝政府知之甚少。<ref>[http://news.ifeng.com/history/zhongguogudaishi/detail_2012_01/13/11953685_0.shtml 朝鲜皇室的“反清复明”计划：明亡60年仍想北伐]</ref>
