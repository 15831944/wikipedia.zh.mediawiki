'''朝鲜民主主义人民共和国国务委员会委员长'''是[[朝鲜民主主义人民共和国国务委员会|朝鲜民主主义人民共和国国务委员会]]的最高负责人，前身为朝鲜民主主义人民共和国国防委员会委员长，也是[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]实行[[先军政治|先军政治]]下的[[朝鲜领导人列表|朝鲜最高领导人]]。

== 国防委员会委员长的由来 ==
1972年12月通过《[[朝鮮民主主義人民共和國社會主義憲法|朝鲜民主主义人民共和国宪法]]》规定设立[[朝鲜民主主义人民共和国政府|中央人民委员会]]为国家权力的最高领导机关，由[[朝鮮民主主義人民共和國主席|国家主席]]领导，[[朝鮮民主主義人民共和國主席|国家主席]]兼任中央人民委员会委员长。

1992年4月对[[朝鮮民主主義人民共和國社會主義憲法|宪法]]进行了增删和修改，删去国家主席担任朝鲜一切武装力量的[[朝鮮人民軍最高司令官|最高司令官]]和国防委员会委员长条款。

1998年9月撤销原国家权力最高领导机关[[朝鲜民主主义人民共和国政府|中央人民委员会]]，规定国防委员会为[[主權|国家主权]]的最高军事领导机关和全面管理国防的机关。

== 修宪后的国防委员长职权和地位 ==
2009年4月9日召开的[[最高人民会议|朝鲜最高人民会议]]修订宪法，在第六章中增加关于国防委员会委员长地位的规定（第六章第二节 国防委员会委员长），将国防委员会委员长明确定为“'''国家的最高领导人'''”（第100条），“国防委员长是朝鲜民主主义人民共和国整个武装力量的最高司令官，指挥和统帅国家的一切武装力量”（第102条），由原来的“领导总体的国防工作”变为“领导国家的总体工作”（第103条第1款）。国防委员会委员长“直接领导国防委员会工作，任免国防部门的重要干部，批准或废除与外国缔结的重要条约，行使特赦权，宣布国家的紧急状态和战时状态，发布动员令”（第103条第2款-第6款）。

== 国务委员会委员长 ==
2016年6月29日，[[最高人民会议|最高人民会议]]决定修改《[[朝鮮民主主義人民共和國社會主義憲法|宪法]]》，将国防委员会改组为“国务委员会”，[[金正恩|金正恩]]当选委员长，[[黄炳誓|黄炳誓]]、[[崔龙海|崔龙海]]、[[朴奉珠|朴奉珠]]当选为副委员长，新的国务委员会不再只侧重军事方面的工作。

== 国家元首身份的争议 ==
包括中文媒体在内的一些外文媒体，至今仍传言[[金永南|金永南]]为朝鲜名义上的国家元首<ref>{{Cite news|url=https://cn.nytimes.com/asia-pacific/20180205/korea-olympics/zh-hant/|title=名義上的國家元首：朝鮮派金永南參加冬奧開幕式|date=2018-02-05|work=|newspaper=紐約時報中文網|accessdate=2018-03-28}}</ref><ref>{{Cite news|url=http://www.bbc.com/zhongwen/trad/world-43007675|title=美國副總統避免與朝鮮「國家元首」共進晚餐|date=2018-02-09|work=|newspaper=BBC 中文网|accessdate=2018-03-28}}</ref>，实为过时信息。根据1998年9月5日实行的朝鲜民主主义人民共和国社会主义宪法（第7号）<ref>{{Cite web|url=https://ko.wikisource.org/wiki/조선민주주의인민공화국_사회주의헌법_(제7호)|title=조선민주주의인민공화국 사회주의헌법 (제7호)|accessdate=2018-03-28|author=|date=|publisher=위키문헌|language=ko}}</ref>，第100条的规定，国防委员会是国家主权的最高军事指导机关，同时也是完全承担国防管理职能的机关。第111条规定，最高人民会议常务委员会委员长负责常任委员会的事业和组织指导。最高人民会议常任委员会委员长是国家的代表，负责接受外国使节的国书和传召。这是金永南被称为名义国家元首，或对外礼节元首的法律依据。

2009年4月9日，朝鲜进行了修宪，即朝鲜民主主义人民共和国社会主义宪法（第8号）<ref>{{Cite web|url=https://ko.wikisource.org/wiki/조선민주주의인민공화국_사회주의헌법_(제8호)|title=조선민주주의인민공화국 사회주의헌법 (제8호)|accessdate=2018-03-28|publisher=위키문헌|language=ko}}</ref>，第117条照搬了原宪法111条的叙述，而第100条有关国防委员会的内容，更改为：朝鲜民主主义人民共和国国防委员会委员长是朝鲜民主主义人民共和国的[[朝鲜最高领导人|最高领导者]]。至此，宪法意义上的国家元首由[[金正日|金正日]]担任。

2010年4月9日，颁布施行朝鲜民主主义人民共和国社会主义宪法（第9号）<ref>{{Cite web|url=https://ko.wikisource.org/wiki/조선민주주의인민공화국_사회주의헌법_(제9호)|title=조선민주주의인민공화국 사회주의헌법 (제9호)|accessdate=2018-03-28|publisher=위키문헌|language=ko}}</ref>，有关内容沿用了第8号宪法的描述。

金正日于2011年12月17日[[金正日之死|去世]]，此后一段时间名义上的国家元首不明确。而后在2012年4月13日颁布施行朝鲜民主主义人民共和国社会主义宪法（第10号）<ref>{{Cite web|url=https://ko.wikisource.org/wiki/조선민주주의인민공화국_사회주의헌법_(제10호)|title=조선민주주의인민공화국 사회주의헌법 (제10호)|accessdate=2018-03-28|publisher=위키문헌|language=ko}}</ref>。新宪法第100条更改为：朝鲜民主主义人民共和国国防委员会第一委员长是朝鲜民主主义人民共和国的最高领导者。至此明确了[[金正恩|金正恩]]的国家元首地位。2016年6月30日，朝鲜官方网络媒体[[我們民族之間|我們民族之間]]，引述了党报[[勞動新聞|勞動新聞]]有关宪法补充和修正的报告。<ref>{{Cite web|url=http://www.uriminzokkiri.com/index.php?ptype=igisa2&no=113290|title=조선민주주의인민공화국 사회주의헌법을 수정보충함에 대한 보고|accessdate=2018-03-28|date=2016-06-30|publisher=우리 민족끼리|language=ko}}</ref>根据修改后的宪法，国防委员会改为国务委员会，第100条规定：朝鲜民主主义人民共和国国务委员会委员长是朝鲜民主主义人民共和国的最高领导者。据此，金正恩依旧担任国家元首。
