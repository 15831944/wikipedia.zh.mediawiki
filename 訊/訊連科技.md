'''訊連科技（CyberLink，{{twse|5203}}）'''是一家供應數位影音多媒體的全球性軟體公司，以其旗艦產品[[PowerDVD|PowerDVD]]稱著，總部位於[[中華民國|中華民國]]（[[臺灣|臺灣]]）[[新北市|新北市]]。訊連科技一系列多媒體應用解決方案，主要可區分為兩大類：

媒體創作（Create）–透過一系列的應用軟體[[威力导演|PowerDirector]]、[[PhotoDirector|PhotoDirector]]、[[PowerProducer|PowerProducer]]、創作吸引人的影片及相片，製成藍光與DVD光碟等。

媒體娛樂（Play）– [[PowerDVD|PowerDVD]]為本類別的旗艦產品，提供家中電腦、區域網路中的各式裝置，以及隨身行動智慧型裝置最新的影音多媒體播放技術。

近年來訊連擴展產品線至[[Android|Android]]及[[iOS|iOS]]平台，提供行動裝置App，如[[玩美相機|玩美相機]]、[[YouSnap|YouSnap]]等功能強大且實用免費App

== 產品 ==
* [[PowerDVD|PowerDVD]]與[[PowerDirector|PowerDirector]]是訊連大型[[多媒體|多媒體]]家族軟體的一部份
* [[Power2Go|Power2Go]]於2006年3月12日開始支援[[藍光光碟|藍光光碟]]
* [[CyberLink_Media_Suite|CyberLink Media Suite]]是一個多媒體應用的完整解決方案，其中包括[[PowerDVD|PowerDVD]]、[[PowerDirector|PowerDirector]]、[[PowerProducer|PowerProducer]]、[[Power2Go|Power2Go]]、[[PowerBackup|PowerBackup]]等12個軟體
* [[PhotoDirector|PhotoDirector]]於2011年全新產品，以專業攝影Workflow所設計的軟體
* [[AudioDirector|AudioDirector]]為其於2012年9月所推出的[[音效編輯軟體|音效編輯軟體]]
* [[ColorDirector|ColorDirector]]為其於2012年9月所推出的[[視訊調色軟體|視訊調色軟體]]
* [[Director_Suite|Director Suite]]為專業等級全功能影像編輯軟體組合包，包括[[威力導演|威力導演]]，[[PhotoDirector|PhotoDirector]]，[[AudioDirector|AudioDirector]]與[[ColorDirector|ColorDirector]]
* iPhone App = [[Power_Media_Player|Power Media Player]]，[[PowerDVD_Remote|PowerDVD Remote]]，[[AudioClipper|AudioClipper]]
* Android App = [[Power_Media_Player|Power Media Player]]，[[PowerDVD_Remote|PowerDVD Remote]]，[[YouSnap|YouSnap]]，[[玩美相機|玩美相機]]，[[PhotoDirector_for_Android|PhotoDirector for Android]]
* Windows 8 Metro App = [[Power_Media_Player|Power Media Player]]，[[PhotoDirector_Mobile|PhotoDirector Mobile]]，[[YouMemo|YouMemo]]，[[YouCam_Mobile|YouCam Mobile]]，[[PowerDirector_Mobile|PowerDirector Mobile]]
