《'''瞬间的第六感'''》（{{jpn|j=瞬間的シックスセンス|hg=しゅんかんてきシックスセンス|rm=Shunkan teki Shikkusu sensu}}）是[[日本|日本]][[创作歌手|创作歌手]][[爱缪|爱缪]]的第2張主流[[音樂專輯|音樂專輯]]，於2019年2月13日發行。

==简介==
爱缪在2018年12月17日举行的巡回演唱会《AIMYON TOUR 2018 -HONEY LADY BABY-》最终场的演出现场发表了本作《瞬间的第六感》发售的消息。距离前作《[[青春的悸动|青春的悸动]]》相隔将近一年半。本作收录包括2018年发布单曲《{{link-ja|假如在满月之夜|満月の夜なら}}》、《[[金盏花_(爱缪单曲)|金盏花]]》、《[[今夜就这样|今夜就这样]]》，以及2019年1月动画电影《[[即便明天世界将迎来终结|即便明天世界将迎来终结]]》的主题曲和插曲在内共12曲<ref name='w'>{{cite web |language=ja|script-title=ja:あいみょん、2月に新アルバム発表＆武道館公演開催 |url=https://natalie.mu/music/news/312616 |website=音楽ナタリー |accessdate=2019-01-31 |date=2018-12-17}}</ref>。

专辑共发行3个版本，除通常盘外，还有附带照片集的“爱缪online store限定盘”和附带原创T恤的“HMV·Loppi限定盘”<ref name='w' />。

==反响==
专辑发售首周，Billboard统计共销售59,668张，在专辑周榜排名第二位，创造爱缪自身最高的排位<ref>{{cite web |language=ja|script-title=ja:【ビルボード】ONE OK ROCK『Eye of the Storm』が初週売上20.8万枚で3作連続となる首位 2位のあいみょんも自己ベスト更新 |url=http://www.billboard-japan.com/d_news/detail/72779/2 |website=Billboard JAPAN |accessdate=2019-02-19 |language=ja}}</ref>。Oricon公信榜统计销售61,654张，在Oricon的专辑周榜、专辑下载周榜和专辑综合周榜中同样排名第二位<ref name='p'>{{cite web |language=ja|script-title=ja:【オリコン】あいみょん、3位以下を大きく引き離す合算アルバム2位 |url=https://www.oricon.co.jp/news/2129893/full/ |website=ORICON NEWS |accessdate=2019-02-20 |language=ja}}</ref>。新专辑的发售也带动爱缪的作品在当周Oricon流媒体周榜前10中独占6席<ref name='p' />，前30更占据14席。
