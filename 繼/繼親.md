'''繼親'''是[[親屬|親屬]]的一種，是指生父與生母離異或[[父母|父母]]其中一方[[逝世|逝世]]後，子女跟隨其中一方。再婚配偶和孩子之間的關係，以及孩子和再婚配偶自身兒女之間的關係。

其再婚配偶對於孩子就是'''繼父'''或'''繼母'''，繼父又稱'''後父'''、'''晚爷'''，'''繼母'''又稱'''後母'''、'''晚娘'''。粵語俗稱繼母為'''後抵乸'''，稱繼父為'''後抵公'''。而繼父母在跟生父或生母婚前與他人（如前任[[配偶|配偶]]或[[戀人|戀人]]）所生的孩子，會依其性別及年齡大小稱為'''繼兄'''、'''繼弟'''、'''繼姊'''或'''繼妹'''，合稱'''繼兄弟姊妹'''。對於再婚配偶而言，這個孩子就是其'''繼子'''或'''繼女'''，合稱為'''繼子女'''。

== 法律关系 ==

由於繼親並沒有[[血緣|血緣]]關係，有些[[國家|國家]]或[[地區|地區]]的[[法例|法例]]容許繼親[[結婚|結婚]]或合法發生[[性行為|性行為]]，但有些國家則視之為[[亂倫|亂倫]]。另外，有些國家或地區規定，包括继子女在内的繼親，沒有[[繼承權|繼承權]]。或是在特定情况下，享有继承权。譬如中华人民共和国方面，继子女与继父母形成抚养关系（一是继父母抚养继子女，二是继子女[[赡养|赡养]]继父母），继子女可继承遗产<ref>{{Cite web |url=http://jxzy.hljcourt.gov.cn/public/detail.php?id=11991 |title=成年继子女有继承权吗|accessdate=2018-03-25 |author=|date=2016-10-17 |work= |publisher=鸡西市中级人民法院网|language=Zh-hans }}</ref>。

再婚對象如無[[收養|收養]]另一方於前婚姻所生子女，則繼父母與繼子女之間不為[[血親|血親]]，僅為[[姻親|姻親]]（配偶的血親、血親的配偶）。

== 繼親關係與相關文化 ==

社会生活中，继父母虐待子女的事件并非罕见。而在文化形象上，相较于继父，不少[[童話故事|童話故事]]、[[民間故事|民間故事]]中的繼母形象都是負面的，常會[[虐待|虐待]]、傷害繼子女，如《[[白雪公主|白雪公主]]》中的壞皇后，《[[灰姑娘|灰姑娘]]》中的繼母等。[[俗語|俗語]]「晚娘嘴臉」也是指不友善的態度。中国亦有“有后妈就有后爸”的俗语<ref name="a1">{{Cite web |url=http://wemedia.ifeng.com/24203566/wemedia.shtml |title=《我的前半生》：有了后妈就有了后爸，终于懂这句话的意思了|accessdate=2018-03-25 |author=|date=2017-07-30 |work= |publisher=凤凰网，大风号|language=Zh-hans }}</ref>，认为男性再婚后，会纵容妻子虐待前任所生子女。有鉴于此，“宁要[[乞丐|乞丐]]妈，不要当官爹。”<ref name="a1"/><ref>{{Cite web |url=http://www.takefoto.cn/viewnews-813900.html |title=后妈虐童案宣判 网友:宁要乞丐妈不要当官的爹就是个理儿|accessdate=2018-03-25 |author=|date=2016-06-18 |work= |publisher=北晚新视觉/北京晚报官方网站|language=Zh-hans }}</ref>

雖然現實中不乏繼母虐待、傷害繼子女的例子，但也有繼父母把繼子女視如己出，甚至有些較極端的例子會因為怕繼子女不接受自己，於是對繼子女加以照顧、愛護，忽略親生子女。如[[晉朝|晉朝]]人[[羊祜|羊祜]]的[[母親|母親]]蔡氏（[[蔡邕|蔡邕]]之女，[[蔡琰|蔡琰]]之妹），就因為照顧繼子而忽略了自己親生的長子，令長子夭折。亦有些怕被人指虐待繼子女，於是對繼子女過度溺愛、縱容，對親生子女則較為嚴厲，結果影響[[親子關係|親子關係]]，也令兄弟姊妹間產生矛盾。

收養親子關係以及繼親關係也是不少電影的主題，例如1998年的《{{le|親親小媽|Stepmom (film)}}》，以及2009年的奧斯卡獲獎電影《[[攻其不備|攻其不備]]》即是描述這類的關係。

此外，繼親之間的戀愛和性關係也常用作[[文學|文學]]、[[戲劇|戲劇]]、[[動漫|動漫]]、[[色情片|色情片]]的題材。
