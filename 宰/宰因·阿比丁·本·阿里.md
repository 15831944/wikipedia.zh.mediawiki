'''宰因·阿比丁·本·阿里'''（[[阿拉伯语|阿拉伯语]]：'''{{lang|ar|زين العابدين بن علي|}}'''，{{bd|1936年|9月3日|catIdx=B}}），[[突尼斯|突尼斯]]政治家，[[獨裁者|獨裁者]]，從1987年11月7日起至2011年1月14日任[[突尼斯总统|总统]]，執政23年，是该国自1956年从[[法国|法国]]独立后的第二任总统。

== 生平 ==
本·阿里生于突尼斯[[苏塞|苏塞]]市郊，早年曾在[[法国|法国]]和[[美国|美国]]的军事学校就读。他在1958年至1974年间曾任军事安全局局长，1977年至1980年间任国防部长和国家安全总局局长。1980年起任突尼斯驻[[波兰|波兰]]大使，四年后返国任负责国家安全的国务秘书，成为内阁阁员。因为成功处理政治对手、[[伊斯蘭主義|伊斯蘭主義]]勢力对当时世俗政权的威胁，他在1986年获晋升为内政部长，并担任该职位直至1987年获委任为总理。

1987年10月1日，本·阿里获[[哈比卜·布尔吉巴|哈比卜·布尔吉巴]]总统委任为总理。其后他宣布布尔吉巴因健康问题不适宜继续担任总统，并于11月7日接任总统。1999年，本·阿里在选举中以99.66%的得票率连任总统；2004年10月24日，他以94.48%的得票率再次当选总统。<ref>{{cite web|url=http://news.xinhuanet.com/world/2004-10/25/content_2137978.htm|title=本·阿里以高票再次当选突尼斯总统|author=张有浩|date=2004年10月25日|publisher=新华网|accessdate=2007-11-10}}</ref>2009年10月25日，他又以89.62％的得票率再次当选总统。<ref>{{cite web|url=http://news.xinhuanet.com/photo/2009-10/26/content_12331847.htm|title=本·阿里在突尼斯总统选举中再度获胜|date=2004年10月25日|publisher=新华网|accessdate=2009-11-07}}</ref>

2011年「[[茉莉花革命|茉莉花革命]]」後，本·阿里被迫下台，流亡至[[沙特阿拉伯|沙特阿拉伯]]。

== 總統任內施政 ==
本·阿里在國內實行「[[开明专制|开明专制]]」的統治，放寬對新闻出版的管制，容許反對黨進行活動和參與選舉，也容許工人有自由組織工會和罷工的權利。他於任內扩大了宪法委员会的权限以监督总统和立法机构选举，2002年舉行修憲公投，通過将立法机构從一院制改为[[两院制|两院制]]。<ref name=qq20110116>{{Cite web|url=http://view.news.qq.com/a/20110116/000001.htm |title=庄礼伟：突尼斯变天，原因何在？ |work=腾讯网 |date=2011-01-16}}</ref>

本·阿里於1993年设立“总统人权奖”鼓励人权事业。<ref name=qq20110116/>

=== 經濟 ===
本·阿里在國內推行經濟改革，吸引外國投資。突尼斯的人均[[國內生產總值|國內生產總值]]（調整至現時價格）從1986年的1201美元升至2008年的3876美元。<ref>[http://data.un.org/Data.aspx?q=tunisia&d=SNAAMA&f=grID:101;currID:USD;pcFlag:1;crID:788 UNdata Record View Per Capita GDP at Current Prices]，United Nations。</ref>從1987年起的大約20年間，國內生產總值每年平均以接近5%的速度增長。

在其任內，突尼斯经济发展成就被誉为“突尼斯奇迹”。2009-2010年度世界经济竞争力论坛年报显示，在经济竞争力、抵御金融危机、通讯和信息技术促进和生活质量改善等方面，突尼斯在133个国家排名中名列非洲第1、世界第40。在茉莉花革命前，无论从政府廉洁度（透明国际2010年排名，突尼斯位列排名59位），还是从民生状况来观察，突尼斯都是发展中国家样板。<ref name=voa20140201>{{Cite web |url=http://www.voachinese.com/content/he-qing-lian-20140131/1842250.html |title=何清涟 : 突尼斯宪政的“路径依赖” - 阿拉伯之春三周年回顾（1） |publisher=美國之音 |date=2014-02-01}}</ref>

=== 外交 ===
本·阿里任总统期间继承了布尔吉巴的亲西方政策。

== 下台 ==
2008年的[[金融海嘯|金融海嘯]]导致突尼斯旅游收入下降，失业率上升。2010年的[[突尼斯|突尼斯]]國內[[失業率|失業率]]達14%。2010年12月，一名因失業而當無牌小販的26歲青年[[穆罕默德·布瓦吉吉|穆罕默德·布瓦吉吉]]在[[攤檔|攤檔]]被充公、被女性執法人員掌摑後向地方政府辦公室未果、憤而自焚，觸發民眾[[2010-2011年突尼斯骚乱|騷亂]]，[[警察|警察]]未能平息騷亂。成為日後[[阿拉伯之春|阿拉伯之春]]導火線。

2011年1月13日晚，本·阿里发表全国电视讲话，表示将会进行国家改革，尊重国家宪法，并保证不会为谋求连任而修宪。1月14日，他宣布解散执政党宪政民主联盟组成的一党制政府，决定提前进行议会选举。当天晚间，总理[[穆罕默德·加努希|穆罕默德·加努希]]在国家电视台宣布，由他代理总统职务，本·阿里流亡国外，被迫逃亡至[[沙特阿拉伯|沙特阿拉伯]]。<ref>[http://news.163.com/11/0115/00/6QDAD2DL00014JB5.html 网易新闻：突尼斯冲突升级 总统流亡总理代行其职]</ref>突尼斯憲法委員會認為總統職位「處於永久性空缺狀態」，因此總理任代總統乃違反憲法，應由眾議院議長擔任。1月15日，眾議院議長[[福阿德·迈巴扎|福阿德·迈巴扎]]宣誓就任代總統。
