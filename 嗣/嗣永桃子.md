'''嗣永桃子'''（{{bd|1992年|3月6日|catIdx=T}}）是[[日本|日本]]前女性[[偶像|偶像]]、[[歌手|歌手]]及[[藝人|藝人]]。[[千葉縣|千葉縣]][[柏市|柏市]]出身，是「[[Hello!_Project_Kids|Hello! Project Kids]]」的前成員之一。[[Berryz工房|Berryz工房]]、Buono!、[[Country_Girls|Country Girls]]前領軍成員。暱稱有'''{{lang|ja|ピーチッチ、ももち}}'''等。

== 特徵 ==
* 可愛的聲音是個人特色
* 非常喜歡漫畫，因为卧室里经常堆满了漫画或杂志，所以有时会被父母拿去扔掉。
* 握著東西（尤其是麥克風）的時候小指總是會翹起來，這點不只經常是Berryz其他成員的話題，也經常成為是[[早安家族|早安家族]]其他前輩成員模仿的題材。
* 經常維持著偶像榜樣般的身段與表情，部份歌迷常以「PRO」稱之。
* 在「Hello! Project SPORTS FESTIVAL」有跳箱11段的紀錄。
* 在廣播節目Berryz工房 起立！ 礼！ 着席！裡經常強調自製的嗣永憲法。
* 因为认为自己是大人，所以讨厌被当作小孩子对待．正是这一点是小孩子气的行为。
* 2005年的秋单独「SWICHI!ON！」演唱会上，演出服的号码是「29」，是因为29的发音是其姓——嗣「Tsugu」的谐音。
* 為團中主唱之一，她與[[菅谷梨沙子|菅谷梨沙子]]、[[熊井友理奈|熊井友理奈]]和[[夏燒雅|夏燒雅]]在8人時期被統稱為TOP4，在7人時期與菅谷梨沙子和夏燒雅被統稱為TOP3。
* 2010年4月28日於放送節目『Berryz工房 嗣永桃子のぷりぷりプリンセス』中，大學入學發表，但仍活躍於Berryz工房。
*2013年大學畢業，為hello project 中第一個完成大學課程的藝人。
*歌舞能力超群，外貌可愛，綜藝感極優，可以說是全能偶像，因此被AKB家族成員選為「最感到畏懼的對手」中的第二名
