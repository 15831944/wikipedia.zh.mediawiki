'''己亥邪狱'''又称'''己亥迫害'''，是發生在[[1839年|1839年]]，[[朝鮮宪宗|朝鮮宪宗]]時期政府迫害[[天主教|天主教]]的事件。
== 背景 ==
1801年[[辛酉邪狱|辛酉邪狱]]开启了朝鲜王朝迫害、肃清天主教徒的先河，一改[[朝鲜正祖|朝鲜正祖]]年间放任天主教“自起自灭”的态度，以镇压天主教为国策。但是，辛酉邪狱有时僻党争因素，镇压天主教只是僻派夺权口实，所以适可而止，并未持续，朝鲜天主教很快恢复与发展。时派的[[安东金氏|安东金氏]]势道政治时代，没有改变镇压天主教的国策，但氛围宽松。朝鲜教会继续在地下发展教徒，[[1831年|1831年]][[教皇|教皇]][[格列高利十六世|格列高利十六世]]宣布[[朝鲜|朝鲜]]教会与[[北京|北京]]教区分离，成立独立的朝鲜教区。[[1836年|1836年]]，[[法国|法国]]传教士罗伯多禄（Pierre Philibert Maubant）在朝鲜教徒的接应下渡过鸭绿江入朝，朝鲜始有[[欧洲|欧洲]][[传教士|传教士]]（此前有两位华人传教士周文谟和刘方济），当时朝鲜的传教士都来自法国[[巴黎外方传教会|巴黎外方传教会]]。

[[道光|道光]]十四年（1834年，纯祖三十四年）十一月，[[朝鲜纯祖|朝鲜纯祖]]薨，八岁的世孙李奂即位，为[[朝鲜宪宗|朝鲜宪宗]]，大王大妃金氏（[[纯元王后|纯元王后]]）垂帘听政。纯祖生前将辅佐世孙（宪宗）重任托付给[[赵寅永|赵寅永]]<ref>尹定铉《云石遗稿》序：“我宪宗大王冲龄嗣服，公（赵寅永）以肺腑之亲，独任内讲，如汉之授帝书禁中者……自古文人之所未有也。贱臣尝侍前席，上每语公内讲时事：‘自朝至日昃，不离左右，勤勤恳恳，竭诚勉戒，予敬而惮之，诵读温习，未或少违典学而有进，严师之力也。’”</ref>。纯祖托孤之事在正史中没有明确记载，但仍可得到印证，比如宪宗在赵寅永上疏中的批示<ref>赵寅永《云石遗稿》卷之七，〈乞休致四疏〉“况我先王简拔卿以遗小子，任之以辅导之责”</ref>和纯元王后金氏的讲话<ref>《承政院日记》，宪宗五年十一月初十日“先大王简拔倚毗之重，予之所依赖为理之意，卿（赵寅永）既知之矣”</ref><ref>《承政院日记》，哲宗元年十二月初六“昔我纯宗大王以辅导宪宗之任，委之于此大臣（赵寅永）”</ref>。道光十五年（1835年，宪宗元年）以后，赵寅永历任吏曹判书、奎章阁提学、水原留守、礼曹判书、户曹判书等要职。他以“卫正斥邪”标榜，大力弘扬“正学”即性理学。早年热衷于金石学等考据学（汉学），又坚持[[程朱理学|程朱理学]]正统（宋学）<ref>赵寅永《云石遗稿》卷之九，〈彝斋实记序〉。“盖自程朱继作，圣人之道复明于世，欲求入道之门，惟程朱书是已。”</ref> ，因此被称“汉宋折衷论”。<ref>金明淑．《19世纪政治论研究》：汉阳大出版部，2004年</ref>宪宗初年，[[丰壤赵氏|丰壤赵氏]]与安东金氏势力均衡，明争暗斗。安东金氏对天主教宽容，丰壤赵氏遂祭起“卫正斥邪”的大旗。

== 过程 ==
道光十九年（1839年，宪宗五年），[[赵万永|赵万永]]、赵寅永为首的丰壤赵氏掀起镇压天主教的己亥邪狱，这年春开始搜捕各地天主教徒并处死，最早进入朝鲜的三名法国传教士范世亨（Laurent-Joseph-Marius Imbert，朝鲜教区主教）、郑牙各伯（Jacques Chastan）、罗伯多禄（Pierre Philibert Maubant）同时殉教。七月实行“五家作统法”，十月十八日以宪宗名义颁布由赵寅永撰写的《斥邪纶音》<ref>《朝鲜王朝实录·宪宗实录》卷6，五年十月十八日</ref><ref>赵寅永《云石遗稿》卷之十一，〈斥邪纶音〉</ref>。三天后，赵寅永拜相，授议政府右议政。领议政和左议政长期空缺，他成为实际上唯一的宰相，丰壤赵氏的声望和势力压倒安东金氏，确立了自己势道政治的地位。他对朝鲜宪宗和纯元王后阐述自己的施政理念<ref>《承政院日记》，宪宗五年十一月初十日 臣于向者《斥邪纶音》撰进时，以明正学为辟异端之本者……而至于表彰先贤，风厉人士之方，实为此时不可已之先务……而臣又有因言端继陈者，前此经行别荐，固已多得其人，而山泽林樊之间，或有守善而见漏，则恐非野无遗贤之道也。且明春为式年应荐之时，另饬各道，益加搜访，就应荐之例，寓别荐之意，必以笃学力行之士从实入荐，而有其人则不必拘于多寡，无其人则又不可苟然充数。</ref>对陶山书院、华阳书院等多家著名书院施以特典，对前代名儒宋稚圭、吴熙常等赐谥号，征召和起用“山林”儒士洪直弼、金迈淳等人，以此达到弘扬“正学”的目的。 

== 事后 ==
法国于1846年因法国传教士被杀为由问责朝鲜，法国舰队司令瑟西耳（Jean-Baptiste Cécille）因不熟悉朝鲜海岸在[[忠清道|忠清道]][[洪州|洪州]][[外烟岛|外烟岛]]留下一封致朝鲜宰相的书信后离开，这是朝法之间最早的外交文书。<ref>《朝鲜王朝实录·宪宗实录》卷13，十二年七月初三日</ref>1847年，拉别耳（Augustin de Lapierre）率领法国舰队以迫害传教士为由炮轰[[越南|越南]]沱㶞（[[岘港|岘港]]），随后北上朝鲜接收回信，在[[古群山岛|古群山岛]]触礁，无功而返。<ref>《韩法关系资料（1846-1856）》，《教会史研究》1，1977年。</ref> 1848年法国相继发生革命和政变，政局动荡，无暇顾及远东。
