'''己部'''，為漢字索引裡為[[部首|部首]]之一，[[康熙字典|康熙字典]]214個部首中的第四十九個（三劃的則為第二十個）。就[[繁體|繁體]]和[[簡體|簡體]]中文中，己部歸於三劃部首。己部通常是從上、下、右方及中間均可為部字，它亦可變形為'''已'''和'''巳'''，且無其他部首可用者將部首歸為己部。

== 部首單字解釋 ==
===己===
jǐ
# [[天干|天干]]的第六位。參見「[[己_(天干)|己]]」。
# 自身的代稱。
===已===
yǐ（ㄧˇ）：
# 停止。
# 副詞：⑴表示過去。⑵表示過了一會。⑶表示過甚。
# 助詞，用在語尾：⑴表肯定，同「矣」。⑵表疑問，同「呢」、「嗎」。⑶表感嘆，同「啊」。
# 通“以”。
===巳===
sì（ㄙˋ）
# 胎兒。見[[說文|說文]]通訓定聲。
# 地支名，為十二地支的第六位。參見「[[巳|巳]]」。
# 時辰名，指上午九時至十一時。
# 生肖名，指蛇。
