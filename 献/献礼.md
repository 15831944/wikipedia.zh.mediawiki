'''献礼'''是[[中华人民共和国政府|中国各级政府]]及其它组织为[[节日|节日]]和历次[[中国共产党全国代表大会|中国共产党全国代表大会]]<ref name="a1">{{Cite web |url=http://news.163.com/12/1010/20/8DFTM46600014JB5.html |title=广电总局推介26部十八大献礼片 |accessdate=2017-07-25 |author= |date=2012-10-10 |work=责任编辑：王晓易_NE0011 |publisher=网易，来源：中华网|language=Zh-hans }}</ref><ref>{{Cite web |url=http://data.chinaxwcb.com/epaper2017/epaper/d6539/d5b/201707/79220.html |title=《南方》：创新党建报道献礼十九大 |accessdate=2017-07-25 |author=作者：杜一娜|date=2017-07-04 |work= |publisher=中国新闻出版广电网，来源：中国新闻出版广电报|language=Zh-hans }}</ref>而进行的庆祝活动，除文艺[[表演|表演]]外，包括建设建筑物、排演戏剧、拍摄电视剧、电影等。建筑物称'''[[献礼工程|献礼工程]]'''，戏剧剧目称'''[[献礼剧目|献礼剧目]]'''<ref>{{Cite web |url=http://news.xinhuanet.com/local/2017-03/18/c_1120652308.htm |title=国家话剧院十九大献礼剧《谷文昌》正式公演 艺术再现“党的好干部” |accessdate=2017-07-25 |author=记者：白瀛 |date=2017-03-18 |work= |publisher=新华网|language=Zh-hans }}</ref>，电视剧称'''[[献礼剧|献礼剧]]'''，电影则称'''[[献礼片|献礼片]]'''<ref name="a1"/>。

最为重要的献礼节日是[[中华人民共和国|中华人民共和国]]、[[中国共产党|中国共产党]]和[[中国人民解放军|中国人民解放军]]的成立纪念日，即[[中华人民共和国国庆节|国庆节]]，[[建党节|建党节]]、[[八一建军节|八一建军节]]。1950年代，[[中华人民共和国政府|中国政府]]为迎接建国十周年修建的[[北京十大建筑|北京十大建筑]]，获得广泛认可。现今，与献礼剧、献礼剧相比，各地献礼工程质量问题频出<ref>{{Cite web |url=http://cpc.people.com.cn/GB/64093/82429/83083/15304695.html |title=“献礼工程”向谁献礼|accessdate=2017-07-25 |author=作者：周丹，责任编辑：刘红艳 |date=2011-08-02 |work= |publisher=人民网-《人民日报》|language=Zh-hans }}</ref>，被广泛质疑。地方政府通过献礼工程，滥用[[财政|财政]]费用，官员借机贪污腐败，使得献礼工程变成“献丑工程”<ref>{{Cite web |url=http://jjckb.xinhuanet.com/opinion/2012-10/09/content_404954.htm |title=别让"献礼工程"成为"献丑工程"|accessdate=2017-07-25 |author=陈志龙 |date=2012-10-09 |work= |publisher=经济参考网，来源：国际金融报 |language=Zh-hans }}</ref>。
