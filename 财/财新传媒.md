'''财新传媒'''（{{lang-en|'''Caixin Media'''}}）是[[中华人民共和国|中华人民共和国]]的一家提供财经新闻及资讯服务的全媒体集团，由前《[[财经_(杂志)|財經]]》杂志總編輯[[胡舒立|胡舒立]]發起創建。

== 历史 ==
2009年11月，[[胡舒立|胡舒立]]从《[[财经_(杂志)|财经]]》杂志辞职<ref name=xieren/>。2009年12月，由胡舒立创办的'''财新传媒有限公司'''在北京成立，浙报传媒控股集团有限公司（浙报控股）出资4000万元人民币持股40%。2012年7月19日，财新传媒公布了新一轮融资情况，[[腾讯|腾讯]]成为其股东之一，但不参与财新传媒的日常运营，浙报控股保持原有持股比例不变<ref name=wy>{{Cite web |url =http://tech.163.com/12/0719/10/86P5FN8A000915BF.html  |title = 财新传媒宣布完成新一轮融资 腾讯成新股东 |author =  |date = 2012-07-19 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。根据2012年年报，浙报控股持有浙报传媒64.62%股权<ref name=lirug/>。

2013年12月19日，在[[北京|北京]]举行的第四届财新峰会闭幕演讲中，华人文化基金董事长[[黎瑞刚|黎瑞刚]]宣布，华人文化产业投资基金已经入股财新传媒，黎瑞刚出任财新传媒有限公司董事长。华人文化基金日前与财新传媒的主要股东浙报传媒控股集团有限公司（浙报控股）签署了股权转让协议。财新网的报道称，股权转让后，财新传媒既有采编方针不变<ref name=lirug>{{Cite web |url =http://money.163.com/13/1220/08/9GH97DTI00253B0H.html  |title = 黎瑞刚出任财新传媒董事长 |author =  |date =2013-12-20  |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。财新传媒原投资者浙报控股由此全部退出，财新传媒旗下《新世纪》周刊刊号将转至[[上海|上海]]<ref>{{Cite web |url = http://tech.163.com/13/1219/21/9GG6PGG3000915BF.html |title = CMC入股财新传媒 黎瑞刚成胡舒立团队第一股东 |author =  |date =2013-12-19  |publisher =网易  |language =  |accessdate =  |quote =  }}</ref><ref name="CMC入股">{{Cite web |url = http://finance.ifeng.com/a/20131219/11309714_0.shtml  |title =  CMC入股财新传媒 黎瑞刚成胡舒立团队第一大股东|author =  |date = 2013-12-19|publisher = 凤凰网 |language =  |accessdate =  |quote =  }}</ref>。

2015年3月，《财新周刊》2015年第12期在《权力猎手郭文贵》报道中，宣称揭开政泉控股老板[[郭文贵|郭文贵]]的政商关系，并指控郭文贵涉入2015年1月16日落马的[[中华人民共和国国家安全部|中华人民共和国国家安全部]]副部长[[马建|马建]]及国安系统<ref>{{Cite web |url =http://weekly.caixin.com/2015-03-27/100795235.html?p4  |title =  【特别报道】权力猎手郭文贵|author =  |date =  |publisher =财新周刊2015年第12期  |language =  |accessdate =  |quote =  }}</ref>，被[[BBC|BBC]]认为是胡舒立团队继《谁的鲁能》之後的重大调查报道<ref name=bbc/>。随后郭文贵公开掀起与财新方面的口水战。3月30日财新传媒发表声明，指郭文贵造谣侮辱胡舒立人格，已向警方报案<ref>{{Cite web |url =  http://www.ce.cn/xwzx/gnsz/gdxw/201503/30/t20150330_4974143.shtml|title =财新传媒：郭文贵造谣侮辱胡舒立人格 已报案  |author =  |date = 2015-03-30 |publisher = 中国经济网 |language =  |accessdate =  |quote =  }}</ref>。同时，财新传媒宣佈将到[[香港|香港]]起诉郭文贵及曾采访郭文贵的《[[香港商报|香港商报]]》与《[[香港苹果日报|香港苹果日报]]》等媒体，称郭文贵透过媒体传播虚构事实，败坏财新传媒总编辑胡舒立的人格，使财新传媒公信力受损<ref name=bbc>{{Cite web |url =http://www.bbc.co.uk/zhongwen/trad/china/2015/04/150401_china_hongkong_caixin_guowengui  |title = 財新傳媒赴港狀告郭文貴與港媒 |author =  |date =2015-04-01  |publisher = BBC中文网 |language =  |accessdate =  |quote =  }}</ref>。

2015年6月30日，财新传媒宣布获得Markit公司编制的PMI（[[采购经理人指数|采购经理人指数]]）在中国的冠名权，自2015年8月起，财新中国PMI将正式发布，更名后PMI统计方法不变<ref>{{Cite web |url =https://finance.qq.com/a/20150630/020740.htm  |title =  中国数据：财新传媒8月起接替汇丰与Markit合作发布中国PMI指标|author =  |date = 2015-06-30 |publisher = 腾讯 |language =  |accessdate =  |quote =  }}</ref>。

2017年10月，财新传媒宣布将自11月6日起正式启动财经新闻全面收费<ref>{{Cite web |url = http://www.sohu.com/a/198718203_619317 |title =财新传媒凭什么启动全面收费？  |author =  |date = 2017-10-18 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。

2018年2月12日中信資本旗下私募股權基金牽頭與財新傳媒旗下財新國際聯合組成財團，以總代價1.805億美元(約14.08億港元)，全數收購歐洲貨幣機構投資者集團(Euromoney Institutional Investor)旗下金融訊息數據庫-全球市場情報中心(Global Markets Intelligence Division)

== 产品与服务 ==
===财新网===
财新网由财新传媒创办。2010年1月11日正式上线。2011年9月19日完成改版。2011年9月22日，新版财新网正式发布，完成域名切换，采用了辨识度更高的新域名caixin.cn<ref name=gaiban/><ref>{{Cite web |url =  https://finance.qq.com/a/20110923/006285.htm|title =  新版财新网9月22日正式发布|author =  |date =2011-09-23  |publisher =腾讯  |language =  |accessdate =  |quote =  }}</ref>。2012年1月6日宣布正式启动新的国际顶级域名caixin.com，与此同时Caixin.cn和Caing.com两域名仍可使用<ref>{{Cite web |url = http://tech.qq.com/a/20120106/000206.htm |title =财新网宣布正式启用国际顶级域名caixin.com  |author =  |date = 2012-01-06 |publisher = 腾讯|language =  |accessdate =  |quote =  }}</ref>。

=== 《财新周刊》 ===
《财新周刊》源自1988年10月创刊的《'''新世纪'''》周刊<ref name=xieren/>。《新世纪》周刊是中国[[改革开放|改革开放]]后最早的新闻杂志之一，由[[中国（海南）改革发展研究院|中国（海南）改革发展研究院]]主管主办，定位为综合类新闻周刊，1988年10月由[[国家新闻出版署|国家新闻出版署]]批准成立<ref>{{Cite web |url = http://money.163.com/10/0104/09/5S6353CI00253B0H.html |title =胡舒立:“借壳”《新世纪周刊》  |author =  |date = 2010-01-04 |publisher = 网易 |language =  |accessdate =  |quote =  }}</ref>。2009年12月30日，中国（海南）改革发展研究院发表公告称：2009年12月29日，该院决定聘任[[胡舒立|胡舒立]]为兼职高级研究员、《新世纪》周刊总编辑；[[杨大明|杨大明]]为兼职研究员、《新世纪》周刊副总编辑；[[苗树彬|苗树彬]]为《新世纪周刊》副总编辑；王烁为兼职研究员<ref name=zhan/>。2010年1月起，《新世纪》周刊改由财新传媒与中国（海南）改革发展研究院共同经营<ref name=xieren/><ref>{{cite web |url=http://www.independent.co.uk/news/world/asia/crusading-editor-fights-new-war-on-censorship-1866131.html |title=Crusading editor fights new war on censorship |author=Coonan, Clifford |date=13 January 2010 |work=The Independent World |accessdate=2 January 2010}}</ref><ref>{{cite web |url=http://china.globaltimes.cn/society/2010-01/496193.html |title=Rivalry surrounds controversial editor |author=Huang Jingjing |date=2010-01-05 |work=Global Times |accessdate=2010-01-02 |deadurl=yes |archiveurl=https://web.archive.org/web/20110707031621/http://china.globaltimes.cn/society/2010-01/496193.html |archivedate=2011-07-07 }}</ref>。2010年1月11日正式推出财新《新世纪》“改刊纪念号”，加强了在新媒体方面的力量<ref name=gaiban>{{Cite web |url = http://news.sina.com.cn/m/2011-10-17/162323316602.shtml|title =  财新网改版尝试全媒体 突出图片和视频板块|author =  |date = 2011-10-17 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。此后《新世纪》周刊主管主办单位先改为[[浙江日报报业集团|浙江日报报业集团]]，2014年又改为[[中国文史出版社|中国文史出版社]]。2015年3月，《新世纪》周刊正式更名为《'''财新周刊'''》<ref name=xieren>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1942511 |title = 胡舒立卸任《财新周刊》总编辑改任社长，王烁接任总编辑职 |author =  |date = 2018-01-09 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref><ref name=zhan>詹蔷，《新世纪》周刊的组织结构状况及评析，今传媒2010年10期</ref>。
