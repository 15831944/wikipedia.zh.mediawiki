'''越南經濟'''，是改變[[單一制|單一制]][[計劃經濟|計劃經濟]]後的一個亞洲新興經濟體，目前處於輕工業和農業體系階段，正在發展進出口加工業。高通貨膨脹率、以及鉅額之貿易逆差為越南經濟發展之二大隱憂，通脹高企、且該國央行持有外匯存底有限，通貨膨脹率9.2%(2012年)。

==部门经济==
越南農林水產品於2012年的出口金額239億美元，較2011年衰退4.4%。但依然是多數國民生計，其中能發展出口價值農漁產品的民眾較有機會在農村成為富農，[[稻米|稻米]]以中國大陸、菲律賓為最大出口對象，咖啡以售往美國及德國為多。其他一般農民貧困率仍高，想致富只有前往外國當[[外勞|外勞]]的途徑。

==能源==
越南南部海域雖有[[石油|石油]]儲量，約8,200萬噸（約6億桶）。但只有少量的煉油與開採能力，越南石油天然氣集團和[[日本|日本]]及[[科威特|科威特]]有聯合投資關係進行石油產業建立。

1960年代前[[蘇聯|蘇聯]]和越南地質學家發現越南石溪鐵礦，估計該[[鐵|鐵]]礦商業可開發蘊藏量約3億公噸，中國大陸寶山鋼鐵公司、印度Tata及Essar等三家鋼鐵公司2010年後積極洽談開採權，日本則有興趣介入鋼鐵加工廠後段產業鏈。另老街省有傳統銅礦和提煉廠年產1萬多噸，越南煤炭總儲藏量也有約2,200億公噸但是經過百年以上開採即將耗盡。

==對外貿易==

2012年[[越南|越南]]出口成長19.6%，達1,146億美元，但其國內企業僅出口了423億美元，佔總體的比例為37%。

2013年，以地区而论，越南前五位的[[贸易伙伴|贸易伙伴]]是[[中华人民共和国|中国]]、[[东盟|东盟]]、[[美国|美国]]、[[日本|日本]]和[[韩国|韩国]]。[[欧盟|欧盟]]同样是越南最主要的出口市场<ref name="a1">{{Cite web |url=http://www.mofcom.gov.cn/article/resume/n/201402/20140200502764.shtml |title=中国蝉联越南最大贸易伙伴|accessdate=2015-11-15 |author= |date=2014-02-28 |work= |publisher=中华人民共和国商务部网站，来源：云南日报|language=Zh-hans }}</ref>。

=== 中国 ===

自2004年起，中国已经连续十年成为越南第一大贸易伙伴。同时，中国是越南最大的进口国，占到进口额的三分之一；中国是越南第四大出口国，占出口额的10%。在2013年，两国贸易额为654.8亿美元，同比增长30%<ref>{{Cite web |url=http://focus.news.163.com/14/0527/13/9T8NR4PV00011SM9.html |title=中国企业越南生存现状调查|accessdate=2015-11-15 |author=作者：张璐晶 |date=2014-05-27 |work= |publisher=网易，来源：中国经济周刊(北京)|language=Zh-hans }}</ref>（另一个数字是500.21亿美元，增长22%<ref name="a1"/>）。2013年，中国在越南的直接投资额增长38.7%，达到205.9亿美元<ref name="a1"/>。但由于两国间[[南海争端|南海争端]]，越南在2014年5月爆发[[2014年越南排华暴动|大规模排华暴动]]。大量中资、外资企业受损，多名中国工人死亡。

=== 美国 ===

1975年，[[越南战争|越南战争]]结束，[[越南民主共和国|越南民主共和国]]最终战胜[[美国|美国]]支持的[[越南共和国|越南共和国]]，统一越南。两国关系降至最低点。1994年，美国解除对越南的[[贸易禁运|贸易禁运]]。1995年，两国建立[[大使级外交关系|大使级外交关系]]，美越关系正常化。2000年，[[比尔·克林顿|比尔·克林顿]]成为越南战争后第一位访问越南的[[美国总统|美国总统]]<ref>{{Cite web |url=http://www.people.com.cn/GB/channel2/18/20001121/320680.html |title=克林顿越南之行|accessdate=2015-11-15 |author=记者：任毓骏|date= |work= |publisher=人民网，来源：《人民日报》(2000年11月21日第六版)|language=Zh-hans }}</ref>。至2012年，美国为越南第一大出口国。
