'''越南巡逻艇太平岛鸣枪事件'''是指在2012年3月[[越南|越南]]艦艇两次向[[中华民国|中华民国]]南海领土[[太平岛|太平岛]]驶进引發的外交事件。

== 事件经过 ==
2012年3月22日、2012年3月26日，[[越南|越南]]艦艇两次向[[中华民国|中华民国]]南海领土[[太平岛|太平岛]]驶进；過程中越南巡逻艇曾以机枪鸣枪挑衅，随后中华民国海巡警艇也以鸣枪回应，而太平岛守军一度进入备战状态。<ref name="lion" >{{cite web |url= http://news.ifeng.com/mainland/special/nanhaizhengduan/content-3/detail_2012_04/20/14031835_0.shtml|title=台媒：越军在南沙太平岛鸣枪挑衅 台湾驻军开枪示警 |publisher=凤凰网 |accessdate= 2012年4月20日|work= |language =zh-cn}}</ref><ref name="wolf " >{{cite web |url=http://news.sina.com.cn/c/2012-04-20/081424302638.shtml |title= 台媒称越南舰艇在南沙开火挑衅 台军开枪示警|publisher= 新浪网|accessdate=2012-04-20 |work= |language =zh-cn}}</ref><ref name="sheep">{{cite web|url=http://news.cn.yahoo.com/ypen/20120420/999723.html|title=台媒：台军与越军在南沙太平岛互相鸣枪|publisher=雅虎网|accessdate=2012-04-20|work=|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20120422062448/http://news.cn.yahoo.com/ypen/20120420/999723.html|archivedate=2012-04-22}}</ref><ref name="bear" >{{cite web |url= http://mil.news.sohu.com/20120420/n341113603.shtml|title=越南武装快艇逼近中国南沙太平岛 公然开枪挑衅 |publisher= 搜狐网|accessdate=2012年4月20日 |work= |language =zh-cn}}</ref><ref name="衝突" >[http://magazine.chinatimes.com/ctweekly/20120426003296-300106 南沙衝突官兵嚇破膽 越艦開火挑釁 海巡鳴槍反嗆]</ref>

== 各方反应 ==
=== 中华民国 ===
中华民国总统府紧急召开跨部会会议，一方面明确指示太平岛守军不得率先动武，另一方面则透过外交途径向[[越南|越南]]表达最严重抗议。<ref name="lion" /><ref name="wolf" /><ref name="sheep" /><ref name="bear" /><ref name="衝突" />
