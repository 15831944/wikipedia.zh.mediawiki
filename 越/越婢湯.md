'''越婢湯'''在[[中醫|中醫]]上的分類屬「辛涼解表」{{notetag|辛、涼係指該方劑的藥性，辛的作用為發散，屬陽，例如薑；<ref name = '黃帝內經'/>涼的作用為降低熱度；而所謂的「表」係指表皮肌肉層，表示病邪侵入人體尚淺，只在表層。故辛涼解表的意思為該藥物的藥性為發散除熱，用以解決皮表上病邪的入侵。}}藥，常用於治療因感冒所引發的[[水腫|水腫]]。此病症好發於身體肥胖的人，或者排尿功能欠佳的人在感冒時也容易發這種病。

== 出處 ==
《[[金匱要略|金匱要略]]．水氣水氣病脈證並治篇》：「風水，急風，一身悉腫，脈浮不渴，續自汗出，無大熱，越婢湯主之。」<ref name='金匱要略'/>

== 功用 ==
越婢湯有如下等功用：發汗、退熱、利尿、消水腫、抗菌、抗炎，抗過敏。

== 方劑組成 ==
越婢湯的組成為：[[麻黃|麻黃]]六兩，[[石膏|石膏]]半斤，[[生薑|生薑]]三兩，[[大棗|大棗]]十五枚，[[甘草|甘草]]二兩。右五味，以水六升，先煮麻黃，去上沫，內諸藥，煮取三升，分溫三服。

== 辨症要點 ==
《金匱要略．水氣水氣病脈證并治篇》中提出了辨症上的幾個要點，分別是：「風水，惡風，一身悉腫，脈浮不渴，續自汗出，無大熱。」

*風水：此指指外致病原「[[風邪|風邪]]」入侵，突發頭面部及四肢水腫為主要表現的水腫病。《[[黃帝內經|黃帝內經]]．素問．水熱穴脈論》中說風水係病人因久勞導致腎臟功能低下，又適逢風邪所致之病症。<ref name = '素問'/>《[[諸病源候論|諸病源候論]]·水腫病諸候》中加入了脾臟功能低下的因素，脾、腎臟功能不佳，當腎臟虛弱，便容易出汗，一但受涼，汗就停留在腎；加上脾虛不能治水，便讓水溢於皮膚表層。然後又受到風邪入侵，在風邪與體表之水相互作用之下，便形成水腫之病症。<ref name = '水腫'/>

*惡風：指的是病患害怕被風吹到。

*一身悉腫：即全身有水腫的現象，水停於皮膚肌肉之間。導致水腫的主要原因即是上面所說的「風水」。<ref name='醫方集解'>{{cite web | url = https://ctext.org/wiki.pl?if=gb&res=244407&searchu=越婢湯 | author = 汪昂 | title = 《醫方集解．利濕之劑》| work = 中國哲學書電子計畫網站 | accessdate = 2018-09-30}}</ref>

*脈浮：即輕輕碰觸脈膊即可感覺到脈膊跳動，因水在皮表，故脈浮。<ref name='醫方集解'/>

*自汗：指非因身體勞動或運動而流汗，通常是身體虛弱所致。<ref name='金匱要略'/>
