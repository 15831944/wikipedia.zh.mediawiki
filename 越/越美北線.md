'''越美北線'''（{{jpn|j=越美北線|hg=えつみほくせん|rm=Etsumihoku sen|f=y}}）是一條連結[[福井縣|福井縣]][[福井市|福井市]][[越前花堂站|越前花堂站]]至同縣[[大野市|大野市]]{{link-ja|九頭龍湖站|九頭竜湖駅}}的[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）鐵路線（[[地方交通線|地方交通線]]），愛稱「九頭龍線（{{lang|ja|九頭竜線}}）用於[[福井站_(福井縣)|福井站]]等旅客資訊上。

== 概要 ==
越美北線在建設時是連接福井站與[[岐阜縣|岐阜縣]][[美濃太田站|美濃太田站]]的[[越美線|越美線]]一部分。之後福井縣側維持為[[JR|JR]]路線，岐阜縣側則轉為[[長良川鐵道|長良川鐵道]][[長良川鐵道越美南線|越美南線]]，兩方無法連通。[[日本國有鐵道|日本國有鐵道]]（國鐵）時代，國鐵巴士開設連結大野市大野三番與越美南線{{link-ja|美濃白鳥站|美濃白鳥駅}}的大野線。[[國鐵分割民營化|國鐵分割民營化]]後，[[JR東海巴士|JR東海巴士]]曾開設九頭龍湖站至美濃白鳥站的季節巴士，但已於2002年（[[平成|平成]]14年）廢止。

1974年（[[昭和|昭和]]49年）以前，京福電氣鐵道越前本線（現在的{{link-ja|越前鐵道勝山永平寺線|えちぜん鉄道勝山永平寺線}}）可至大野三番的京福大野站。該路線廢除以後越美北線成為唯一來往大野市的鐵道路線。

國鐵時代，貨物站{{link-ja|南福井站|南福井駅}}至越前花堂站間是越美北線與北陸本線的重複區間。國鐵分割民營化後，越美北線起點改成越前花堂站，解除重複區間。現在，越美北線的越前花堂站是與北陸本線的分岐點，但實際分岐點是在南福井站內。

1960年（昭和35年）最初區間通車時，由於有許多同名車站存在，因此多冠上福井縣北部的舊國名「[[越前國|越前]]」。

全線由{{link-ja|西日本旅客鐵道金澤支社|西日本旅客鉄道金沢支社|金澤支社}}的{{link-ja|福井地域鐵道部|福井地域鉄道部}}管轄，是福井縣內唯一的{{link-ja|非電氣化|非電化}}旅客線。

{{link-ja|越前大野站|越前大野駅}}至{{link-ja|九頭龍湖站|九頭竜湖駅}}間的[[閉塞_(鐵路)|閉塞方式]]採用[[閉塞_(鐵路)#單票閉塞|單票閉塞式]]。2018年現在還採用單票閉塞式的其他JR路線僅剩[[札沼線|札沼線]][[石狩月形站|石狩月形站]]至[[新十津川站|新十津川站]]間以及[[名松線|名松線]]{{link-ja|家城站|家城駅}}至{{link-ja|伊勢奧津站|伊勢奥津駅}}間。

== 運行形態 ==
除了臨時列車，越美北線只行駛普通列車。本線沒有以越前花堂站為起訖站的班次，除了越前大野站至九頭龍湖站間的一往返班次以外，全列車皆延駛至福井站。每日福井站與越前大野站間有九班往返，越前大野站至九頭龍湖站間有五班往返，後者的車距可達四小時以上。原則上實施[[一人控制|一人控制]]，尖峰時刻與活動期間會增派[[車掌|車掌]]。

每月會進行一次白天時段停駛，2011年度（平成23年度）起實施{{link-ja|振替輸送|振替輸送}}（越前花堂站等部分車站外採用巴士代行）。

2001年（平成13年）3月3日改點以前，福井站至越前大野站間有快速定期班次。當時的快速班次停靠福井站、[[越前東鄉站|越前東鄉站]]、[[市波站|市波站]]、[[美山站|美山站]]、[[越前大宮站|越前大宮站]]、{{link-ja|北大野站|北大野駅}}。

九頭龍新綠緑祭的5月中旬周六日，以及九頭龍紅葉祭的10月最後一個周六日會行駛臨時快速列車，但每年的運行班次與區間、停靠站有所差異。

福井市與大野市之間的公共交通，除了越美北線以外還可利用{{link-ja|京福巴士|京福バス}}<ref>{{Cite web |date= |url=http://bus.keifuku.co.jp/rosen/map/7.pdf?20180401 |title=福井駅～市内南東部・美山・池田・大野・一乗谷方面 |format=PDF |publisher=京福バス |accessdate=2018-04-25}}</ref>。
