“'''越调'''”，流行于[[河南|河南]]大部、[[山东|山东]]、[[陕西|陕西]]南部、[[安徽|安徽]]西部以及[[湖北|湖北]]北部等地区，历史悠久，为河南三大剧种之一。因其伴奏乐器以[[四胡|四胡]]为主，故又被称作“'''四股弦'''”。2007年入选[[中华人民共和国|中华人民共和国]][[国家级非物质文化遗产名录|国家级非物质文化遗产名录]]。

== 历史 ==
越调历史悠久，在[[清代|清代]][[乾隆|乾隆]]年间，已经在河南[[南阳|南阳]]等地区广为流传。随后渐出现了越调剧社，并在清末形成了独特的剧种并逐渐走向兴盛。在剧种成后逐渐了三种独特的演出形式，分别是[[皮影|皮影]]越调戏、[[木偶|木偶]]越调戏以及越调大班戏。这三种演出形式，至今仍在在河南南阳、湖北北部以及安徽西部等地区的农村流行。
自起创立以来越调形成了超过五百个传统剧目，而其中尤以擅演“[[三国|三国]]”剧目而闻名。

== 演出特点 ==
越调起源于曲牌体，在后来的发展中逐渐加入了板腔体，最终成为一种[[曲牌体|曲牌体]]与[[板腔体|板腔体]]合一的独特剧种。越调曲牌多来自民间音乐和其他[[戏曲|戏曲]]剧种，分弦牌和笛牌两种共计两百多个。越调的板式也多种多样，常见的有慢板、导板、[[飞板|飞板]]、[[流水|流水]]、哭腔、吹腔、铜器调、杂调、赞子、紧打慢唱、杂调等。
越调的唱腔高亢明快、唱中有笑，笑中有唱，以字带音，以音传情，既能抒发悲伤、深沉的情感，又能书法壮怀激烈的感情。越调唱腔主要为”越调”并辅以昆腔、唱吹腔、七句半等。
越调具备浓重的乡土气息和[[中原|中原]]地区民间音乐的特色、越调的唱腔丰富，具有本行当特有的套路和板式。演员演出时以本嗓演唱同时辅以[[假声|假声]]，其中流行唱腔的主因较其他剧种高出5个[[音节|音节]]。
伴奏乐器以俗名为“上天梯”的四胡为主，[[月琴|月琴]]、[[卧笛|卧笛]]为辅，随着其发展逐渐增加了诸如[[短杆坠壶|短杆坠壶]]、[[二胡|二胡]]、[[唢呐|唢呐]]、[[闷子|闷子]]、[[三弦|三弦]]以及[[琵琶|琵琶]]等。文场与武场的伴奏有所不同，文场乐器以坠壶、二胡、唢呐、笛子、琵琶、三弦为主；武场演出乐器则主要为[[板鼓|板鼓]]、[[堂鼓|堂鼓]]、大小鼓等。

== 演出分类 ==
越调的传统剧目分为两种，分别为[[正装戏|正装戏]]和[[外装戏|外装戏]]，共计四百多个。
*正装戏为越调的传统剧目，以生、净为主角，构思严整，主要演历史历史袍带戏。正装戏唱词深奥，每段唱词都是固定的，并要依据对顶的[[曲牌|曲牌]]和调门演唱。在[[道白|道白]]时正装戏的伴奏为[[卧笛|卧笛]]。代表剧目有《十五宫》、《报火斗》、《乌江岸》、《[[文王|文王]]吃子》等。
*外装戏则多为民间生活戏以及公案戏，词句通俗，多唱少白，生活气息较为浓重浓重，并多以[[小生|小生]]、[[丑角|丑角]]为主角。代表剧目有《李双喜借粮》、《哭殿》、《火焚绣楼》等。 

== 行当与角色 ==
越调行当角色齐全，包括[[文生|文生]]、[[武生|武生]]、[[正旦|正旦]]、[[花旦|花旦]]、[[闺门旦|闺门旦]]、[[浪旦|浪旦]]、[[武旦|武旦]]、[[老旦|老旦]]、[[大净|大净]]、[[二毛|二毛]]、[[大红脸|大红脸]]、[[二红脸|二红脸]]、[[三花脸|三花脸]]、等十几种。
河南越调则尤以女演员扮[[须生|须生]]而闻名，然而在越调的发展历史中，直到三十年代才有女演员登台演出。著名的须生女演员有[[张秀卿|张秀卿]]、[[杨桂芝|杨桂芝]]、[[申凤梅|申凤梅]]、[[申秀梅|申秀梅]]等，

== 著名演员 ==
艺名分别为大宝贝、小宝贝的[[张秀卿|张秀卿]]和[[杨桂芝|杨桂芝]]是越调当中女扮须生的代表人物，代表作有《抱琵琶》、《收[[姜维|姜维]]》等。
越调女扮须生一直是人才不断，继大小宝贝后崛起的就是被称作”大梅“”小梅“的申凤梅和申秀梅女士。”大梅“申凤梅的代表作品为被人称作”两吊一收“的《[[诸葛亮|诸葛亮]]吊孝》、《[[李天保|李天保]]吊孝》、《收姜维》。其中申凤梅被称为“活诸葛”。
[[毛爱莲|毛爱莲]]被称为“越调皇后”，她工于旦角，她塑造的性格各异的[[旦角|旦角]]，从[[闺门旦|闺门旦]]，到[[花旦|花旦]]，到[[正旦|正旦]]，都别具风格，毛尤以其特殊的甜美的发音为观众喜爱。
其他著名艺术家有：何全志、孙太安、李金枝、牛德山（艺名牛和尚）、李明玉、史道玉、张秀卿、张秀兰、李大勋、申凤梅、金凤楼、杨桂芝、邱秀玲、张琴、张桂兰、何金堂、何全金、田發根、李玉华、陈静、李金英、袁秀莲、刘琳

== 主要剧团 ==
现在住要演出越调的专业剧团为河南省越调剧团。河南省越调剧团现驻于河南省周口市。前身是成立于1949年6月的隶属于河南省淮阳专属三分团的民友越调剧团。

== 荣誉与奖项 ==
在已故越调表演艺术家[[申凤梅|申凤梅]]的带领下，越调得以发扬广大，并在20世纪六十年代三次进京演出，先后受到[[毛泽东|毛泽东]]、[[周恩来|周恩来]]、[[刘少奇|刘少奇]]、[[朱德|朱德]]、[[邓小平|邓小平]]等人的接见。
在1963年的4月2日、9日、5月15日，越调前后三次进入[[中南海|中南海]]演出。周恩来在看过越调《收姜维》后上台与申凤梅握手，并对她说“你把诸葛亮演活了，河南的诸葛亮很会做思想工作！”
越调在文艺界也引起关注，[[京剧|京剧]]表演艺术家[[马连良|马连良]]破格把地方戏演员收为徒弟。作家老舍对此题词“东风骀荡百花开，越调重兴多俊才，香满春城梅不傲，更随桃李拜师来。”
[[老舍|老舍]]、[[曹禺|曹禺]]、[[杜近芳|杜近芳]]、[[袁世海|袁世海]]、[[赵丹|赵丹]]等都对越调做出赞扬。

== 影视作品 ==
[[珠江电影制片厂|珠江电影制片厂]]与[[北京电影制片厂|北京电影制片厂]]在1965年至1981年间拍摄了《诸葛亮吊孝》、《智收姜维》、《扒瓜园》、《李天保娶亲》等四部电影艺术片。

== 学术著作 ==
越调戏曲理论研究的著名专家有赵抱衡、魏天葆等。魏天葆著有《河南越调音乐概论》一书。
