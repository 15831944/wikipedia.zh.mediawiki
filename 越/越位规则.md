'''越位規則'''（'''{{lang|en|Off-side rule}}'''）是指[[程式語言|程式語言]]中，用[[缩进风格|縮排]]來表示{{link-en|程式區塊|block (programming)}}的範圍。這名詞是來自[[Peter_J._Landin|Peter J. Landin]]，是[[足球|足球]]中[[越位_(足球)|越位]]（offside）的雙關語。

== 程式範例 ==
以下是一個[[Python|Python]]語言程式的例子，其中用縮排表示其程式區塊<ref>[http://effbot.org/pyfaq/why-are-colons-required-for-the-if-while-def-class-statements.htm Python FAQ on colons]</ref>：
 <source lang="python">
 def is_even(a):
     if a % 2 == 0:
         print('Even!')
         return True
     else:
         print('Odd!')
         return False
 </source>

if後面縮排的二行是a為偶數時會執行的程式，而else後面縮排的二行是a為奇數時會執行的程式。
