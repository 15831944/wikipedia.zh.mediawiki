'''越南人名'''屬於東亞式命名顺序，为[[姓|姓]]前[[名稱|名]]後，與[[漢名|漢名]]有相似的結構。但越南人除了姓與個人名，還有[[墊字|墊字]]、[[輩字|輩字]]，統稱為「[[墊名|墊名]]」（{{lang-vi|tên đệm}}，即[[中間名|中間名]]），全名的次序為「姓、墊字、輩字、個人名」。除姓與個人名外，並非所有越南人都兼備人名所有部份，有些只有墊字，有些有輩字，也有些完全沒有任何中間名。亦有些越南人有多個不同結構的姓名，如有時把墊字省略。

== 姓 ==
{{main|越南姓氏}}
越南人的[[姓|姓]]與[[漢姓|漢姓]]相似，多數是[[單姓|單姓]]，有少數是[[複姓|複姓]]。其中數量最多的是「阮」（Nguyễn）姓，其次為「陳」（Trần）、「黎」（Lê）、「范」（Phạm）、「黃」（Hoàng）等。

== 墊名 ==
墊名（{{lang-vi|tên đệm}}）為越南人名中置於姓後的字，它往往介於姓和個人名之間，具有區分性別和聯宗續譜的作用。

越南女性最常見用的垫名是「氏」（Thị），例如「阮'''氏'''紅」，其次有「妙」（Diệu）、「女」（Nữ）、「玉」（Ngọc）等；男性則以「文」（Văn）為多數，例如「[[阮文張|阮'''文'''張]]」、「[[阮文誠|阮'''文'''誠]]」，其次有「公」、「伯」、「明」。墊名有時會省略，如[[阮文惠|阮文惠]]又稱[[阮惠|阮惠]]。女性在華人媒體或華人社會也常省略「氏」字，如[[潘氏金福|潘氏金福]]常被稱為[[潘金福|潘金福]]，於[[台灣|台灣]]、[[香港|香港]]等地居留的越南女性於當地[[身份證|身份證]]上登記的中文姓名也常會省略「氏」字。

墊名通常情況下是一個字，但在當代越南，越來越多的人選擇使用兩個字作為墊名，這種情況下，墊名的第二個字往往會和正式名組成一個含義美好的詞語。例如“黎'''青明'''珠”、“杜'''文光'''明”。

越南男性使用垫名還有區別[[宗族|宗族]]的功能。因為同姓的人未必是同一宗族，於是在姓與名之間使用不同的墊名以作區別，[[後黎朝|後黎朝]]後期歷代君主及[[宗室|宗室]]均以「維」作為墊名。亦有以個人名的偏旁作墊名者。如[[鄭主|鄭主]]歷代均之「[[木部|木]]」偏旁的字為名。亦有兩者同時使用的，如後黎朝後期後半段時期君主及[[宗室|宗室]]除用「維」作為墊名外，他們的個人名均以「示（礻）」為偏旁。有時墊名會省略，如[[阮世祖|阮世祖]][[阮福映|阮福映]]又稱[[阮福映|阮映]]。

在阮朝時，阮朝宮廷經常將女性的姓氏和家族墊名連稱，如嗣德帝善妃[[阮氏錦|阮氏錦]]又作“阮廷氏錦”；同慶帝輔天純皇后[[阮氏嫻|阮氏嫻]]又作“阮有氏嫻”，佑天純皇后[[楊氏熟|楊氏熟]]又作“楊光氏熟”；成泰帝皇貴妃[[阮氏媖|阮氏媖]]又作“阮嘉氏媖”。<ref>《阮福族世譜》，順化出版社，1995年。</ref>

== 輩字 ==
{{main|輩字}}
輩字是同一宗族中用作區別輩份的字，如[[阮朝|阮朝]]皇室之輩字自[[阮憲祖|阮憲祖]]起依次為「綿、洪、膺、寶、永、保」。但越南人極少使用辈字。

== 個人名 ==
越南人名中，主人專屬的名（{{vie|qn=tên chính|hn=𠸛正}}），是較具實際意義的名字，為一般人與人之間相互辨識的依據。越南人較習慣以個人名來稱呼一個人，包括正式的場合（如為雙名則用後一個字）。例如「阮文明」，在正式場合人們會稱之「明先生」（{{vie|qn=ông Minh|hn=翁明}}），而不是「阮先生」。又例如像「[[吳廷琰|吳廷琰]]政府」，在越南語中會稱之「琰政府」（{{vie|qn=chính phủ Diệm|hn=政府琰}}），而不是「吳政府」或「吳廷政府」。在[[美國|美國]]等國的外語文獻中（尤其是敘述[[越南戰爭|越南戰爭]]時期歷史者），提及越南人物時也常不帶姓地僅以個人名稱呼。

== 其他名稱 ==
一如其他東亞地區，越南人傳統上除本身的姓名之外還有別名，包括[[字|字]]、[[號|號]]等。現代越南人已經很少有字或號。

== 越南少數民族的人名 ==
=== 埃地族 ===
越南[[西原|西原地區]]的[[埃地族|埃地族]]使用[[南島語系|南島語系]][[馬來-波利尼西亞語族|馬來-波利尼西亞語族]]的語言。埃地人的姓名包括三个部分，性别前缀、名和部族名，部族名又包括主干和分支部族名两部分。如男子名Y Ngong Nie Dam（伊·恩贡·涅丹）中，Y是性别前缀，现代埃地族男子都使用这个前缀；Ngong是他的个人名，Nie Dam是他的部族名，亦即姓，Nie是主干部族，Dam是分支部族。又如女子名H’Linh Mlo Duon Du（赫·灵·姆洛敦杜）中，H’是性别前缀，即女子；Linh是她的个人名，Mlo Duon Du是她的部族名。现在受越南京族文化影响，不少埃地人的个人名使用越语字。Nie 和 Eban 是埃地族人常见的姓<ref name="ten cua dan toc">Nguyễn Khôi (2006). ''Cách dùng họ và tên của các dân tộc Việt Nam''. Nhà Xuất Bản Văn Hoá Dân Tộc, Hà Nội.</ref>。

由于[[基督教|基督教]]和[[犹太教|犹太教]]的影响，部分埃地基督徒会在自己的性别前缀后加上一个宗教圣人的名字，如Y-Abraham（伊-[[亚伯拉罕|亚伯拉罕]]）、Y-Jeremi（伊-[[耶利米|耶利米]]）、H’Rêbeka（赫-[[利百加|利百加]]）等。

女性后缀H’源于梵语的Hubaya sri devi，意为“小姐”，历史上的缩写为“Hơ-Bia”，近代直接简写为“H’”。埃地族男性在历史上有前缀丹（Dam），意为男子。现代埃地族的男性前缀Y来自于近代埃地语改革者和文化奠基者[[伊·朱特·赫温|伊·朱特·赫温]]（Y Jut Hwing）的创造。最初Y带有宗教色彩，源自犹太教中的[[四字神名|四字神名]]YHWH（[[耶和华|耶和华]]），在当时被犹太教徒和基督徒使用。伊·朱特将之解释为Yàng（神灵）的缩写，意指所有埃地族人都是神的后裔，用以打破埃地族人不同宗教人群之间的隔阂。
