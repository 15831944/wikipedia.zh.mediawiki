'''戚美珍'''（Jaime Chik Mei Chun，{{bd|1962年|1月5日|catIdx=Q}}），籍貫[[廣東|廣東]][[開平|開平]]，出生於[[香港|香港]]，本地[[女演員|資深女演員]]，其丈夫[[苗僑偉|苗僑偉]]亦是知名藝人。她息影一段時間後，於2008年復出，開拍與[[呂良偉|呂良偉]]、[[羅嘉良|羅嘉良]]、[[馬德鐘|馬德鐘]]、[[袁詠儀|袁詠儀]]、[[郭可盈|郭可盈]]、[[薛家燕|薛家燕]]的無綫電視劇《[[富貴門|富貴門]]》。

==生平==
1979年，畢業於[[天主教鳴遠中學|天主教鳴遠中學]]，為該校第14屆中五畢業生，與前香港女藝人、1985年首屆亞洲小姐亞軍[[潘先儀|潘先儀]]為同級同學兼同屆畢業生{{noteTag|資料出自《天主教鳴遠中學六十周年校慶特刊》第88頁〈歷屆畢業生名單〉。}}。此前她曾就讀[[香島中學|香島中學]]。

1980年，戚美珍參加[[無綫電視藝員訓練班|無綫電視藝員訓練班]]第10期藝員訓練班，隨後成為[[香港無綫電視|香港無綫電視]]藝員。

1988年，因約滿關係，戚美珍轉往[[亞洲電視|亞洲電視]]。

1992年，退出娛樂圈。

2008年，重返無綫電視。

2010年，與[[曾志偉|曾志偉]]合作投資[[社會企業|社會企業]]素食餐廳[[樂農|樂農]]<ref>{{cite web |language=zh-hk |url=http://www.hkcd.com.hk/content/2010-12/06/content_2647626.htm |title=社企齊鋪樂農傷健共融 |publisher=[[香港商報|香港商報]] |date=2010-12-06 |accessdate=2011-04-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20141108123046/http://www.hkcd.com.hk/content/2010-12/06/content_2647626.htm |archivedate=2014-11-08 }}</ref><ref>{{cite web |language=zh-hk |url=http://paper.wenweipo.com/2010/10/05/EN1010050038.htm |title=戚美珍助失聰人士就業 |publisher=[[文匯報_(香港)|文匯報 (香港)]] |date=2010-10-05 |accessdate=2011-04-06 |archive-url=https://web.archive.org/web/20111228092734/http://paper.wenweipo.com/2010/10/05/EN1010050038.htm |archive-date=2011-12-28 |dead-url=yes }}</ref>，並透過慈善機構「[[龍耳|龍耳]]」聘請聾人<ref>{{cite web |language=zh-hk |url=https://hk.news.appledaily.com/local/daily/article/20101219/14780972 | title=生活存關愛：首間素食餐廳聘用聽障人士 |publisher=[[蘋果日報_(香港)|蘋果日報 (香港)]] |date=2010-12-19 |accessdate=2011-04-06}}</ref>，主要目的是希望能幫助到社會上一班弱勢社群，讓他們有機會就業謀生。

2012年，再度重返無綫電視，主持無綫節目「珍情品味」。

2017年，受無綫節目《最佳拍檔》邀請，與一眾《歡樂今宵》的元祖級人馬，盧海鵬、林建明、和賈思樂一起分享趣事及回憶，當中亦提及到不少感情要好的舊拍檔，有些已經離開人世，而憶述當時受癌魔折磨，在最後階段依然不放棄、保持樂觀的盧大偉時，亦不禁落淚<ref>[http://bka.mpweekly.com/focus/local/20171127-92339 【最佳拍檔】戚美珍流淚憶述 盧大偉臨終前片段]</ref>。

==婚姻==
1990年，與[[苗僑偉|苗僑偉]]結婚。

==子女==
育有一子一女。
