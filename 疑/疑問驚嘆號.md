'''疑問驚歎號'''（{{Lang-en|Interrobang}}）表示既疑又驚的情感，混合了[[感歎號|感歎號]]和[[問號|問號]]，通常以「?!」或「!?」表示，亦有用「‽」等單一的符號來表示。

== 顯示 ==
它有[[Unicode|Unicode]]值8253，在[[HTML|HTML]]文件中可以用<code>&#8253;</code>或<code>&#x203D;</code>顯示，雖然有些[[網頁瀏覽器|網頁瀏覽器]]對其支持度不高。

== 用法 ==
在一般[[排版|排版]]中，由於沒有‽符號，所以會用「?!」或是「!?」。

{{Who|有中國人|time=2019-02-26}}認為它並非正式的[[標點符號|標點符號]]，不應該在漢語中使用。如果該句子既疑又驚，就應視乎其疑問和感歎成份的多少決來採用感歎號和問號。

在[[漫畫|漫畫]]和[[廣告|廣告]]中，疑問感歎號的使用非常普遍。除此之外，在[[國際象棋|國際象棋]]的棋步記述中有使用「!?」表示有趣而又未必最好的一步，「?!」表示難以決定優劣的一著。

== 歷史 ==
1962年，[[美國|美國]]人Martin K. Speckter覺得若廣告設計者令宣傳句子結尾只有一個符號會比使用兩個更好看。他將這個想法寫入[[雜誌|雜誌]]《TYPEtalks》的文章內。Speckter亦邀請人們為新的符號命名，其中的建議包括「rhet」、「exclarotive」和「exclamaquest」等，但他最後選擇了[[拉丁語|拉丁語]]的疑問「interrogatio」和[[印刷商|印刷商]]對感歎的行話「bang」混合而成的「interrobang」。

這個符號的[[平面設計|平面設計]]亦在該文中徵求。

1966年，American Type Founders發表了Americana印制面，其中有疑問驚歎號。1968年，疑問驚歎號在某些Remington[[打字機|打字機]]出現。1970年代，有疑問驚歎號的鍵可供購買，可以作替換Smith-Corona打字機的鍵。1960年代，隨疑問驚歎號的流行，interrobang出現了在某些[[辭典|辭典]]，並受雜誌、報紙的文章論及。

疑問驚歎號失敗於它只成一時風尚，從未加入正式的標點符號；很多字型沒有包含它。可是它沒有消失：[[Microsoft|Microsoft]]將疑問驚歎號納入[[Wingdings|Wingdings]] 2的字元中；其後它又被Unicode接納。

顛倒的問號和感歎號的混合體能用於[[西班牙語|西班牙語]]的開首字母，稱為'''gnaborretni，'''意即interrobang的反轉。
