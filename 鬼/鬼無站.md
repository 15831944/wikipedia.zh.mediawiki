'''鬼無站'''（{{jpn|j=鬼無駅|hg=きなしえき|rm=Kinashi eki|f=y}}）是一位於[[日本|日本]][[香川縣|香川縣]][[高松市|高松市]][[鬼無町|鬼無町]]，隸屬於[[四國旅客鐵道|四國旅客鐵道]]（JR四國）的[[鐵路車站|鐵路車站]]。車站編號為'''Y02'''，副站名稱為「鬼無桃太郎」，原因是地方名和日本童話傳說中的[[桃太郎|桃太郎]]有關連。此外，車站亦設有一塊站名橫匾，稱本站為「桃太郎與[[盆栽|盆栽]]之站」。

本站主要停靠普通列車，另外在清晨及深夜時份，部份快速「Marine Liner」亦會停靠。


== 車站結構 ==
站舍為單層[[木|木]]及[[混凝土|混凝土]]混合而成的建築物，曾經進行翻新工程。現時為簡易委託站，由站內開設的[[中華料理|中華料理]]餐廳派員於指定時間在售票窗口發售車票，站舍內的候車大堂並沒有設置自動售票機，但卻於2號月台的有蓋候車亭內設置了一部自動售票機。<ref>[http://ku-gyou.net/JR-S/kinashi.html] 位於2號月台候車亭內的自動售票機。</ref>站舍內的候車大堂則設置了長凳及洗手間<ref>[http://sta.nuttari.net/photo/1172/003/11720031.jpg] 圖片取自非官方車站介紹</ref>，另外車站外設有單車停泊處。

未進行翻新工程前，車站正門口的車站標示牌是以電玩「[[桃太郎電鐵系列|桃太郎電鐵]]」的主角模樣作為站牌<ref>{{cite web |url=http://www.shikoku.org.uk/JR/LINE/yosan/ST-KINASHI_03b.jpg |title=存档副本 |accessdate=2013-02-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304080816/http://www.shikoku.org.uk/JR/LINE/yosan/ST-KINASHI_03b.jpg |archivedate=2016-03-04 }} 圖片取自另一個非官方車站介紹。</ref>，但翻新後則改用JR四國的標準車站標示牌，原有的站牌則被移除。

設有[[側式月台|側式月台]]2個，能提供2面2線的配置。靠近站舍為1號月台，遠離月台為2號月台，月台間以行人天橋連接，<ref>[http://www.eki88.info/cgi/shopimg/img-0003-6.jpg] 圖片取自非官方車站介紹。</ref>兩邊月台均設有蓋候車區域或候車亭。月台有效長度為8輛。然而為配合「Marine Liner」的停靠，往高松方向的6輛月台長度均有加高。往端岡方向的2輛月台長度，平時皆不會使用。

2號月台上放置了由[[電玩|電玩]]公司[[Hudson_Soft|Hudson Soft]]所送贈的一列「[[桃太郎電鐵系列|桃太郎電鐵系列]]」石製模型火車，<ref>{{cite web |url=http://www.city.takamatsu.kagawa.jp/kankou/library/list/10_jrkinas.html |title=存档副本 |accessdate=2013-02-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20120604235644/http://www.city.takamatsu.kagawa.jp/kankou/library/list/10_jrkinas.html |archivedate=2012-06-04 }} 高松市公式網頁旅遊介紹。</ref>並於2002年3月31日舉行揭幕儀式。<ref>[http://www.famitsu.com/game/event/1134955_1114.html]</ref>另外頭端部份月台為切割型港灣式，本來亦設有側線，但現時側線已經被拆走。

==歷史==
* 1897年2月21日：隨[[讚岐鐵道|讚岐鐵道]][[高松站_(香川縣)|高松]]－[[丸龜站|丸龜]]開通而設站。
* 1904年12月1日：經營權改由[[山陽鐵道|山陽鐵道]]營運。
* 1906年：鐵道國有化，成為[[日本國有鐵道|日本國有鐵道]]車站。
* 1969年10月1日：貨物轉運服務中止。
* 1970年6月1日：取消貨運服務。
* 1985年3月14日：停留所化、行李託運服務中止。
* 1987年4月1日：[[日本國鐵分割民營化|日本國鐵分割民營化]]，車站的營運由JR四國繼承。
