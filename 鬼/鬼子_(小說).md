《'''鬼子'''》是[[倪匡|倪匡]]筆下[[科幻小說|科幻小說]][[衛斯理系列|衛斯理系列]]之一，系列編號33，連載於1971年7月28日至1971年9月10日的《[[明報|明報]]》<ref>[http://yehleehwa.net/nni17.htm 衛斯理故事年表]</ref>，作品與《[[環_(小說)|環]]》以《鬼子》之名共同出版。此故事並非[[科幻小說|科幻小說]]，而是[[幻想小說|幻想小說]]，幻想一個[[日本|日本]]人對自己於[[南京大屠殺|南京大屠殺]]所犯的罪感到痛苦，在[[衛斯理|衛斯理]]的追查下發現他在戰爭時所作的暴行，作者認為日本人悔改是機率極少的事。

== 故事概述 ==
[[衛斯理|衛斯理]]從大[[酒店|酒店]]中走出，忽然一個[[日本|日本]]人鈴木正直向衛斯理撞去，衛斯理立時抓住他並將他扶住。從那日本人的神態動作，看出他對一個[[旅行社|旅行社]]的女職員[[唐婉兒|唐婉兒]]感到害怕，但她卻不知何以鈴木正直會對她感到害怕。衛斯理對此感到十分好奇，決定追查此事<ref name="日本遊客態度怪異">《鬼子》 第一部：日本遊客態度怪異 [[倪匡|倪匡]]著</ref>。

衛斯理冒充酒店[[職員|職員]]致電給鈴木正直尋求線索，但卻失敗了。於是他以[[旅行社|旅行社]]的代表身份進入鈴木正直的套房，卻再一次刺激了他，使他瘋狂起來，甚至想殺死衛斯理，但衛斯理將他打傷並揚言不會就此罷休<ref name="日本遊客態度怪異"/><ref name="上天無門入地無路">《鬼子》 第二部：上天無門入地無路 [[倪匡|倪匡]]著</ref>。

衛斯理托[[小郭|小郭]]追查鈴木正直和唐婉兒的身世，卻發現二人並無任何關係。與[[傑克上校_(衛斯理系列)|傑克上校]]對話得到靈感，只要與鈴木正直乘坐同一架[[飛機|飛機]]，那他就無法逃避。在飛機上，衛斯理再一次把他逼瘋，於是他立時以「[[點穴|點穴]]」[[功夫|功夫]]將他穩定<ref name="上天無門入地無路"/><ref name="黑暗之中奇事發生">《鬼子》 第三部：黑暗之中奇事發生 [[倪匡|倪匡]]著</ref>。

飛機著陸後，鈴木正直立即離去，並聘請了全日本最有名的[[私家偵探|私家偵探]]藤澤雄來對付他。但藤澤雄是衛斯理的朋友，於是二人合力追查鈴木正直。夜間，衛斯理和藤澤雄分別潛入鈴木正直的家。看著他對著一個布包跪拜和痛苦呻吟。當他走了後，衛斯理和藤澤雄取走了布包。二人把布包的物件取出，卻只是一件寫有「菊井太郎」的日本軍服和一件染有血漬的[[旗袍|旗袍]]<ref name="黑暗之中奇事發生"/><ref name="調查鈴木的過去">《鬼子》 第四部：調查鈴木的過去 [[倪匡|倪匡]]著</ref>。

衛斯理於是利用他和國際警方的關係，調查日本軍方的舊檔案。花了三天時間，卻找不到他要找的鈴木正直。衛斯理無意中從菊井太郎的名字入手，卻意外發現到鈴木的過去：原來鈴木正直的本名是菊井太郎。衛斯理得知鈴木正直（菊井太郎）曾參與過[[南京大屠殺|南京大屠殺]]<ref name="調查鈴木的過去"/>。

衛斯理到鈴木的家，並以他所得的資料恐嚇他，鈴木正直於是交出他在[[南京|南京]]入城所寫的[[日記|日記]]。衛斯理看後只感到十分血腥殘忍。日記上記載著日本人如何虐待和殺死[[中國人|中國人]]，並記載著鈴木正直曾經花上三天時間姦殺一個中國女子。衛斯理也從鈴木正直口中得知唐婉兒的容貌與那個被姦殺的中國女子十分相似，以為她是來討命。衛斯理分析唐婉兒是那個女子的近親，甚至可能就是她的女兒。衛斯理走後翌日得知鈴木正直上吊[[自殺|自殺]]<ref name="慘絕人寰的大屠殺">《鬼子》 第五部：慘絕人寰的大屠殺 [[倪匡|倪匡]]著</ref>。

== 故事角色 ==

*'''衛斯理'''
書中主角。好奇心極重，因偶然見到失常的鈴木正直便決定追查他的身世。及後發現鈴木正直的殘忍行徑，對此十分反感。

*'''鈴木正直'''
原名菊井太郎，戰時為[[日本皇軍|日本皇軍]][[上尉|上尉]]，現為鈴木電子組合的[[總裁|總裁]]。曾參與[[南京大屠殺|南京大屠殺]]並姦殺一名女子。旅遊時遇到和該女子容貌相似的唐婉兒而被她嚇倒，在衛斯理的追查下交出戰時的日記，然後上吊自殺。

*'''唐婉兒'''
孤兒，[[旅行社|旅行社]]副經理，曾於[[日本|日本]]、[[美國|美國]]讀書。容貌與戰時被鈴木正直姦殺的中國女子相似，衛斯理推測二人為母女關係。故事中認識了小郭，後來和他結婚。

*'''小郭'''
私家偵探，受衛斯理之托調查鈴木正直和唐婉兒，卻因而認識了唐婉兒並喜歡上她。

*'''傑克上校'''
無意注意到衛斯理毆打日本遊客的案件而與衛斯理電話對話，其間讓衛斯理得到對付鈴木正直的靈感。

*'''藤澤雄'''
全日本徵信社社長，極有名的私家偵探。曾和衛斯理共同經歷一件不愉快的事情。受鈴木正直之托對付衛斯理，但因與衛斯理相識而倒戈與其聯手追查鈴木正直。

*'''白素'''
故事末段和衛斯理討論鈴木正直自殺的原因。

== 對《衛斯理系列》的影響 ==

小郭與唐婉兒於故事中相識，在《[[大廈_(小說)|大廈]]》中已成了夫妻，儘管唐婉兒於《[[衛斯理系列|衛斯理系列]]》出場次數少之又少。

從衛斯理口中得知，傑克上校曾於[[二戰|二戰]]期間在[[集中營|集中營]]受日本人的毆打。

從故事得知，衛斯理曾與藤澤雄在發生過一次不愉快的事件，但衛斯理一直沒有將之記述。[[葉李華|葉李華]]所著的[[衛斯理回憶錄|衛斯理回憶錄]]《蓋世》將之記述。

== 作者所要表達的看法 ==

作者強烈地表示了對[[日軍侵華|日軍侵華]]的反感和不滿，並認為[[軍國主義|軍國主義]]者決不會因為自己在戰爭中的所為而會悔改。

作者對日本人[[武士道|武士道]]的精神很反感，認為這種想法終會造成另一場戰爭。從藤澤雄的一句話可以看出：「他為什麼要自殺呢？真洩氣，竟然不是用傳統的[[切腹|切腹]]自殺，而是[[上吊|上吊]]死的。」
