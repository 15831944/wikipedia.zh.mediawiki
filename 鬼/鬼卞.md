'''鬼卞'''（{{bd|1992年}}），本名'''胡雪松'''，[[中国|中国]][[嘻哈|嘻哈]]说唱男歌手。 2017年，参加[[爱奇艺|爱奇艺]]音乐选秀节目《[[中国有嘻哈|中国有嘻哈]]》的比赛，获得[[潘瑋柏|潘瑋柏]]组四强、全国12强。现任重庆市森林实验小学语文老师兼班主任<ref>{{cite web |author1=王雅惠 |title=《嘻哈》地獄使者鬼卞秒變老師 反差萌粉絲喊想師生戀 |url=https://news.tvbs.com.tw/local/757388 |website=TVBS |accessdate=2018-12-02}}</ref><ref>{{cite web |title=《中国有嘻哈》鬼卞：一个非典型嘻哈歌手的双面人生 |url=http://news.sina.com.cn/c/2017-08-06/doc-ifyiswpt5657918.shtml |website=Sina |accessdate=2018-12-02}}</ref>。
