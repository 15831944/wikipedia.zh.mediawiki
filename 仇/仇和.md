'''仇和'''（{{bd|1957年|1月1日|catIdx=C仇}}），[[江苏省|江苏]][[滨海县|滨海]]人。[[中国共产党第十六次全国代表大会|中共十六大]]代表、[[中国共产党第十八届中央委员会候补委员|第十八届中央委员会候补委员]]，曾任[[中国共产党云南省委员会|中共云南省委]]副书记。

2015年3月15日，[[中国共产党中央纪律检查委员会|中共中央纪委]]发布消息，指仇和涉嫌严重违纪违法，接受组织调查<ref name="双规">{{cite web|url=http://www.ccdi.gov.cn/xwtt/201503/t20150315_53328.html|title=云南省委副书记仇和涉嫌严重违纪违法接受组织调查|publisher=中纪委监察部网站|accessdate=2015-03-15}}</ref>。三天后被免职<ref>{{cite web|title=云南省委副书记仇和涉严重违纪违法被免职|url=http://www.chinanews.com/fz/2015/03-18/7139664.shtml|publisher=中国新闻网|accessdate=2015-03-18}}</ref>。2015年7月31日，中纪委宣布，仇和严重违纪违法被开除党籍和公职<ref>{{cite web |url = http://www.ccdi.gov.cn/jlsc/201507/t20150731_60041.html |title = 云南省委原副书记仇和严重违纪违法被开除党籍和公职 |accessdate = 2015年7月31日 |publisher = 中央纪委监察部网站}}</ref>。

== 生平 ==

1977年加入[[中国共产党|中国共产党]]。1978年2月，入学[[江苏农学院|江苏农学院]]，毕业于[[南京农业大学|南京农学院]]植保系学习。1982年参加工作。1982年2月，任[[江苏省农科院|江苏省农科院]]植保研究所科研人员、院团委书记。1984年11月，任[[江苏省农科院|江苏省农科院]]工会副主席、党办副主任、院办副主任。1985年1月，江苏省委整党办委派为[[淮阴市|淮阴市]]整党联络员。1985年9月，到江苏省委党校首期县处级干部培训班学习；1986年3月起，任江苏省农科院植保研究所副所长、党支部副书记兼院团委书记、院工会副主席。

1988年5月，任[[丰县|丰县]]科技副县长。1990年8月，任江苏省科委计划处和农村科技处副处长。1992年12月，任江苏省科委农村科技处处长、江苏省农业科技发展基金办公室主任、江苏省[[星火计划|星火计划]]办公室主任；1995年4月至1996年1月参加江苏省首期高级管理人才经济研究班赴[[美国|美国]]培训。1996年7月，任[[宿迁市|宿迁市]]筹建领导小组成员。1996年9月，任宿迁市副市长。1996年12月，任中共宿迁市委常委、副市长，兼任沭阳县委书记。开始推行铁腕[[反腐|反腐]]，并探索出组织政府工作人员在网络发表评论控制舆论的模式<ref name="paopao_393">{{cite news |author = 吴多雨 |title = 仇和倒台与五毛兴衰史 |url = https://pao-pao.net/article/393 |publisher = 泡泡网 |date = 2015-03-17 |accessdate = 2015-03-19 }}</ref>，宿迁民间更有“一年求和，两年求饶，三年求逃”之说<ref>{{cite news|url=http://cpc.people.com.cn/GB/47816/4064742.html|title=争议书记仇和宿迁十年之治|publisher=[[新京报|新京报]]|date=2006-01-26|accessdate=2016-03-16|deadurl=yes|archiveurl=https://web.archive.org/web/20160321052311/http://cpc.people.com.cn/GB/47816/4064742.html|archivedate=2016-03-21}}</ref>。1998年宿迁市沭阳县给教师下达“[[招商引资|招商引资]]”任务，被[[中国中央电视台|中国中央电视台]]《[[焦点访谈|焦点访谈]]》报道。1999年[[宿迁市|宿迁市]][[沭阳县|沭阳县]]让有小偷小摸等行为的人，在电视上念检讨书，取名“沉重的忏悔”，被《[[南方周末|南方周末]]》报道。2001年起宿迁推行教育事业改革，将学校推向市场变为民营。2000年12月，任[[宿迁市|宿迁]]市委副书记、副市长。2001年1月，任宿迁市委副书记、代市长。2001年2月，任宿迁市委副书记、市长。2001年8月，任宿迁市委书记。2002年1月，任宿迁市委书记、宿迁市人大常委会主任。任内，他在宿迁市强力推进医疗改革，将所有医疗机构进行股份制改革。2002年宿迁推行1/3干部离岗招商、1/3干部轮岗创业，副处级干部的任务是500万元/年，完不成任务的干部，所在部门领导免职。2003年初起仇和在宿迁推行干部任用公推公选制度改革，成为江苏这一制度的发源地。2004年7月宿迁推出勤廉公示制度，将党政主要干部资料在《宿迁日报》上公布，由人民提出问题，干部公开“述职述廉”。2005年宿迁开始招聘[[网络评论员|网络评论员]]，就时政热点引导舆论<ref>{{cite news|url=http://news.sohu.com/20050429/n225390790.shtml|title=宿迁26名网评员今上岗|publisher=扬子晚报|date=2005-04-29|accessdate=2015-03-15}}</ref><ref>{{cite news|url=http://www.dw.de/中国茉莉花革命后五毛爆发/a-6452228|title=“中国茉莉花革命”后“五毛”爆发|publisher=德国之声|date=2011-02-26|accessdate=2015-03-15}}</ref>。宿迁是较早招聘[[网络评论员|网络评论员]]的城市，而仇和则被部分网民讥讽为“五毛之父”。2005年6月至9月参加中国第四期公共管理高级培训班赴[[清华大学公共管理学院|清华大学公共管理学院]]和美国[[哈佛大学|哈佛大学]][[肯尼迪政府学院|肯尼迪政府学院]]学习。2006年1月20日，当选为[[江苏省人民政府|江苏省人民政府]]副省长。

2007年12月24日，调任[[中共云南省委|中共云南省委]]常委兼[[昆明市|昆明市]]委书记。2011年11月29日，升任[[云南省|云南省]]委副书记。

==争议==
他在宿迁市委书记任內曾發出過《限桌令》，限制市內居民的婚宴不能超過8桌，黨員更只可擺5桌，否則，就會被罰款。<ref>{{cite news|url=http://news.163.com/05/0528/09/1KR2KCO30001124T.html|title=铁腕书记批限桌令 行政干预人情风引争议|publisher=荆楚网|date=2005-05-28|accessdate=2015-03-04}}</ref>而且，他更聲言“歡迎舉報”，而有黨員就因為違規而被罰款。事件引起了“行政-{干}-預私生活”的討論，但亦{{Who|有人}}認為這是“[[保先|保先]]”的做法，可以避免黨員因為舖張排場而變得腐化。不過，有[[恶搞|恶搞]]之徒把他頒布的《限桌令》改寫成為限制[[性生活|性生活]]次数的《限床令》，並在網上四散。結果《限床令》的傳播比《限桌令》還高，並引起香港及廣州的[[網友|網友]]討論，而仇和本人曾被香港網民冠以“[[去死去死團|死死]]書記”外號。

此外，在沭阳任职期间即强力推行种树的仇和调至昆明后，下令在已经有两排行道树的道路补种一行树木，挤占了人行道的空间，这些补种的树木也被昆明市民讽刺为“仇和树”，但大多在仇和继任者[[张田欣|张田欣]]推动的“种花运动”中被拔去，此后昆明更一度流传“拔掉求和树，种上甜心花”的顺口溜。<ref>{{cite news|url=http://www.changjiangtimes.com/2015/03/498652.html|title=苏滇二十年仇和养成记|publisher=[[长江商报|长江商报]]|date=2015-03-22|accessdate=2017-01-04}}</ref>

他是[[中国大陆|中国大陆]]为数不多的个性官员之一，他从政以来，多次施行和发布让人惊讶和意外的[[政治|政治]]举措和[[命令|命令]]。人民起初认为他为了政绩不顾[[人权|人权]]，但施政的結果，有效地促进了社会发展<ref>{{cite news|url=http://qjwb.zjol.com.cn/html/2011-12/19/content_1260173.htm|title=“官场另类”升任中共云南省委副书记：铁腕仇和，为政有道|date=2011-12-19|quote=一位网友在“仇和升迁感言”中写道：“一直很关心仇和现象，对他的认识也是一波三折。最早看到的是其‘霸道’的执政风格，当时认为，如果有太多这样的官员，中国就无从谈民主法制了。但后来发现，他给执政地带来的变化和发展是惊人的，也是令人信服的。关键是老百姓对他拥戴的呼声也很高…”}}</ref>，2004年[[南方周末|南方周末]]曾以《最具争议的党委书记》形容他<ref>{{cite web|publisher=南方周末|accessdate=2004-02-05|url=http://www.infzm.com/content/31464|title=最富争议的市委书记|deadurl=yes|archiveurl=https://archive.is/20150315195018/http://www.infzm.com/content/31464|archivedate=2015-03-15}}</ref>。但在云南省政协原副主席[[杨维骏_(1922年)|杨维骏]]看来，仇和一路走来，就是宣扬“以不民主来推进民主，以人治来推进法治。”<ref name="秒杀">{{cite web|publisher=新浪网|accessdate=2015-03-15|url=http://news.sina.com.cn/c/zg/ztx/2015-03-15/2007778.html|title=仇和被“秒杀”背后：老干部举报称其系白恩培战友}}</ref>

仇和在接受中共中央纪委审查期间写的忏悔书承认自己“光环笼罩，头脑发热，作风独断专行”<ref>{{cite web|author1=桂强|title=这位落马副国级亲笔悔过书被公开展出 厚达28页|url=http://news.sina.com.cn/c/nd/2017-09-27/doc-ifymeswe0379474.shtml|website=news.sina.com.cn|publisher=新浪网|accessdate=2018-02-10}}</ref>。

==落马==
2015年3月15日，[[中央纪委监察部|中央纪委监察部]]网站客户端发消息称，中共云南省委副书记仇和涉嫌严重违纪违法，正接受组织调查<ref name="双规" />。据媒体报道，仇和牵扯到[[昆明|昆明]]土地城建系统的腐败问题，其落马疑与昆明大搞拆迁有关。云南省政协原副主席[[杨维骏|杨维骏]]向[[新浪网|新浪网]]表示，他此前一直在向中纪委和有关部门举报仇和，“[[白恩培|白恩培]]的亲密战友就是仇和”。杨维骏称，在仇和主政内，春城昆明在短短时间内变化为“拆城”，600万昆明市民面临拆迁，“完全不符合昆明和[[云南|云南]]的实际，云南90%以上是山地半山地，怎么能适应这种城镇化？”<ref name="秒杀" />因涉嫌严重违纪违法，2015年3月26日，云南省第十二届人大常委会第十七次会议决定罢免其[[第十二届全国人民代表大会代表|第十二届全国人民代表大会代表]]职务<ref>{{cite news|title=第十二届全国人民代表大会常务委员会代表资格审查委员会关于个别代表的代表资格的报告（2015年4月24日第十二届全国人民代表大会常务委员会第十四次会议通过）|url=http://www.npc.gov.cn/wxzl/gongbao/2015-07/06/content_1942890.htm|accessdate=2016-07-03|newspaper=中国人大网}}</ref>。

2015年7月31日，中共中央纪委发布通告称：仇和严重违反廉洁自律规定，收受礼金、礼品；利用职务上的便利，为亲属的经营活动谋取利益；利用职务上的便利，在干部选拔任用、企业经营等方面为他人谋取利益，收受巨额贿赂。其中，受贿问题涉嫌犯罪。中央纪委决定给予仇和开除党籍、开除公职处分。[[最高人民检察院|最高人民检察院]]于同日对仇和执行逮捕。

2016年12月15日，贵州省[[贵阳市|贵阳市]]中级人民法院公开宣判中共云南省委原常委、副书记仇和受贿案，对被告人仇和以受贿罪判处有期徒刑十四年六个月，并处没收个人财产人民币二百万元；对仇和受贿所得财物及其孳息予以追缴，上缴国库。经审理查明：2008年至2015年，被告人仇和利用其担任中共云南省委常委、昆明市委书记、中共云南省委副书记等职务上的便利及职权和地位形成的便利条件，为他人在项目推进、获取银行贷款、工作调整等事项上提供帮助，直接或通过身边工作人员非法收受、索取他人财物，共计折合人民币2433.98万元。贵阳市中级人民法院认为，被告人仇和的行为构成受贿罪，且具有索贿情节，应依法惩处。鉴于仇和到案后，如实供述自己罪行，主动交代办案机关尚未掌握的部分受贿犯罪事实；认罪悔罪，积极退赃，赃款赃物已全部追缴，具有法定、酌定从轻处罚情节，依法可以从轻处罚。法庭遂作出上述判决<ref>{{cite news|title=云南省委原副书记仇和一审被判十四年六个月|url=http://news.xinhuanet.com/legal/2016-12/15/c_1120126520.htm|accessdate=2016-12-15|newspaper=新华网}}</ref>。

仇和获刑后不久，[[昆明市中级人民法院|昆明市中级人民法院]]亦对曾任仇和秘书的原昆明市委常委、副市长[[谢新松|谢新松]]受贿案作出一审判决。谢新松被判处有期徒刑10年，并处罚金100万元<ref>{{cite news|title=仇和“大内总管”谢新松获刑 曾将房产给私生子|url=http://news.sina.com.cn/c/nd/2016-12-30/doc-ifxzczff3446153.shtml|newspaper=新浪新闻}}</ref>。
