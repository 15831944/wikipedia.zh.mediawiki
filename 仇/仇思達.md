'''仇思達'''（{{Lang-en|Sze-tat Chau}}，{{bd|1988年|12月7日||}}），網名'''無敵神駒'''，香港[[本土派_(香港)|本土派]]人士，[[網絡電台|網絡電台]]「本土台」台長<ref name = 屹屹屹>{{cite web |author1= |title=屹屹屹|url =https://books.mingpao.com/product/%E5%B1%B9%E5%B1%B9%E5%B1%B9/|website=明報網上書店|date=|accessdate=2017-07-25|}}</ref>，前[[香港城市大學|香港城市大學]]校董（研究生代表）<ref name = "新成員加入城大校董會">{{cite web |author1=陳龍根 |title=新成員加入城大校董會|url =http://wikisites.cityu.edu.hk/sites/newscentre/ch/Pages/201302211040.aspx|website=城大新聞網|date=2013-02-21|accessdate=2017-07-25|}}</ref>。

仇思達任城大校董時，曾舉辦活動批評城大研究生會。

==擔任城大校董時期==
仇思達於2012年11月15日成爲[[香港城市大學|香港城市大學]]校董會成員研究生代表，任期一年<ref name = "新成員加入城大校董會"/>。任內，仇思達舉辦了活動批評城大研究生會。其於2013年3月28日於香港城市大學舉辦「光復城大」直播論壇，質疑城大研究生會多年來財政開支欠缺監管，違憲拖選舉。仇思達稱，當年其發現大選委員會由時任幹事會擔任，覺得有明顯利益衝突，所以於[[民主牆|民主牆]]貼大字報表達意見。而於選舉日之前，大選委員會宣布將選舉無限期押後，原因則有學生表達大選委員會之組成辦法對選舉會造成不公。仇思達又稱，城大研究生會多次用公費舉辦蛇宴、遊船河等，其覺得乃小圈子活動之活動。其再稱，研究生部分須要返工，好少參加此等活動，而由城大研究生會成立以來無評議會，財務開支欠缺監管<ref>{{cite_news|url=https://hk.news.yahoo.com/%E5%9F%8E%E5%A4%A7%E7%94%9F%E6%8A%A8%E7%A0%94%E7%A9%B6%E7%94%9F%E6%9C%83%E6%8B%96%E5%BB%B6%E9%81%B8%E8%88%89-220412785.html|title=城大生抨研究生會拖延選舉|work=《[[星島日報|星島日報]]》|date=2013-03-29|accessdate=2017-07-26}}</ref>。
