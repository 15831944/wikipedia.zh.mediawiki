'''抓姦'''，亦稱'''捉姦，'''民間則稱之為「'''抓猴'''」，針對「有[[婚姻|婚姻]]或戀愛關係之人與[[配偶|配偶]]或情侣以外之人發生性關係（意即[[通姦|通姦]]），婚姻之受害一方進行證據收集」的行動。一般新聞或民眾通常會把焦點放在「抓姦在床」的那一刻。然而，不是每個國家都還留有「通姦罪」，所以依照各國文化與民情不同，面對「通姦」也會有不同的處理方式。

= 「抓姦」的法源依據與容易觸犯之法條 =
台灣社會之所以能「抓姦」，跟刑法中的「通姦罪」有密切關係。

依照台灣《刑法》第239條:「有配偶而與人通姦者，處一年以下有期徒刑。其相姦者亦同。」<ref>{{Cite web|url=https://law.moj.gov.tw/LawClass/LawSingle.aspx?Pcode=C0000001&FLNO=239|title=《刑法》第239條-通姦罪|accessdate=20190418|author=中華民國全國法規資料庫|date=|publisher=}}</ref>

這也是台灣民眾在「面對配偶外遇」時，據理伸張的法源依據。

然而，在實務上要確定外遇雙方通姦並不容易，所以大部分的做法還是「抓姦在床」。

但細究個案，就會發現抓姦的過程中，非常容易觸犯其他法條，

包含《刑法》第315條之一-「妨害秘密罪」<ref>{{Cite web|url=https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode=C0000001&flno=315-1|title=《刑法》第315條-妨害秘密罪|accessdate=20190418|author=中華民國全國法規資料庫|date=|publisher=}}</ref>

「有下列行為之一者，處三年以下有期徒刑、拘役或三十萬元以下罰金：

　一、無故利用工具或設備窺視、竊聽他人非公開之活動、言論、談話或身體隱私部位者。

　二、無故以錄音、照相、錄影或電磁紀錄竊錄他人非公開之活動、言論、談話或身體隱私部位者。」

或是第306條-「侵入住居罪」<ref>{{Cite web|url=https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode=C0000001&flno=306|title=《刑法》第306條-侵入住宅罪|accessdate=20190418|author=中華民國全國法規資料庫|date=|publisher=}}</ref>

「無故侵入他人住宅、建築物或附連圍繞之土地或船艦者，處一年以下有期徒刑、拘役或三百元以下罰金。

　無故隱匿其內，或受退去之要求而仍留滯者，亦同。」

抓姦過程中的拉扯，把外遇者身上的棉被拉下，以及高漲情緒下與外遇者的肢體接觸亦可能觸法。

像是《刑法》第277條-「普通傷害罪」<ref>{{Cite web|url=https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode=C0000001&flno=277|title=《刑法》第277條-普通傷害罪|accessdate=20190418|author=中華民國全國法規資料庫|date=|publisher=}}</ref>

「傷害人之身體或健康者，處三年以下有期徒刑，拘役或一千元以下罰金。

　犯前項之罪，因而致人於死者，處無期徒刑或七年以上有期徒刑；

　致重傷者，處三年以上、十年以下有期徒刑。」

《刑法》第304條-「強制罪」<ref>{{Cite web|url=https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode=C0000001&flno=304|title=《刑法》第304條-強制罪|accessdate=20190418|author=中華民國全國法規資料庫|date=|publisher=}}</ref>

「以強暴、脅迫使人行無義務之事或妨害人行使權利者，處三年以下有期徒刑、拘役或三百元以下罰金。

　前項之未遂犯罰之。」

有鑑於此，不熟法條的民眾如自行抓姦，很容易在過程中觸犯法條。

值得一提的是，台灣司法界並無接納「毒樹果理論」，而是採用「權衡理論」。<ref>{{Cite web|url=http://jirs.judicial.gov.tw/FJUD/index_1_S.aspx?p=WXGjrfHNfnbNrrKP9wHF9Y9Luxi87SAyqmYqnrgw6FE%3D|title=最高法院刑事判決九十六年度台上字第四一七七號|accessdate=20190418|author=司法院法學資料條整理系統|date=|publisher=}}</ref>

也正因如此，抓姦過程中，蒐證手段是否正當、法庭是否採用，都視個案而定。

※

除了《刑事》外，《民事》的婚姻訴訟也常常牽扯到「通姦」。

依據《民法》第1052條之規定：

「夫妻之一方，有下列情形之一者，他方得向法院請求離婚：

　一、重婚。

　'''二、與配偶以外之人合意性交。'''

　三、夫妻之一方對他方為不堪同居之虐待。

　四、夫妻之一方對他方之直系親屬為虐待，或夫妻一方之直系親屬對他方為虐待，致不堪為共同生活。

　五、夫妻之一方以惡意遺棄他方在繼續狀態中。

　六、夫妻之一方意圖殺害他方。

　七、有不治之惡疾。

　八、有重大不治之精神病。

　九、生死不明已逾三年。

　十、因故意犯罪，經判處有期徒刑逾六個月確定。

　有前項以外之重大事由，難以維持婚姻者，夫妻之一方得請求離婚。但其

　事由應由夫妻之一方負責者，僅他方得請求離婚。」<ref>{{Cite web|url=https://law.moj.gov.tw/LawClass/LawSingle.aspx?pcode=B0000001&flno=1052|title=《民法》第1052條|accessdate=20190418|author=中華民國全國法規資料庫|date=|publisher=}}</ref>

所以，對於早想離婚的怨偶而言，逮到配偶「外遇」並要求離婚，

在台灣是有法律保障的。

= 台灣民眾對「抓姦」的主流民意 =
依照「台灣民意基金會」在2017年5月24日舉辦的民調:「通姦除罪化最近又成為台灣新聞話題。有人主張：全世界只剩少數國家，包括台灣在內，有刑法通姦罪，應予廢除才對，請問您同不同意？」

結果是「10%非常同意，16%還算同意，25.1%不太同意，44.3%一點也不同意，4.6%沒意見、不知道或拒答」。<ref>{{Cite news|url=https://www.storm.mg/article/288950?srcid=gAAAAABcuDfrpq6a-FQyIMqD9ta0EOGDc3TsFhmQTwby8vH4XUFw7Rf0GgGXIqb6L75PCsrBrSXfKE_kROs-j51M56bz9KG5FPaIqOkel2AHGiKtrXBnbK4%253D|title=台灣民意基金會民調》通姦除罪化，6成9民眾反對|author=|date=|work=風傳媒|accessdate=20190418}}</ref>

以此觀之，台灣社會還有七成上下的民眾是不贊成「通姦除罪化」。

華人世界重視家庭、貞節的特質，是台灣民意贊成「保留通姦罪」的主因之一。

=「抓姦」的考量點=

每對家庭碰到的問題各不相同，會需要「抓姦」的理由也各不相同。

儘管民眾的目的各有不同，但真的走到「抓姦」步驟前，通常會需要考慮四大要素：

# 做好最壞的打算－離婚
# 配偶是否有機會回頭
# 孩子權益
# 經濟因素

如果甚麼都沒考慮就抓姦，常常會面臨兩種問題－

# 抓姦失敗
# 談合約不知道要談甚麼，被對方牽著走。

抓姦前，一定要考慮再考慮。<ref>{{Cite web|url=https://putavirgo1.pixnet.net/blog/post/34204563-%E5%BE%B5%E4%BF%A1%E9%98%BF%E5%AE%85%E5%BF%83%E5%BE%97---%E6%8A%93%E5%A7%A6%E5%89%8D%E5%BF%85%E9%A0%88%E6%83%B3%E9%81%8E%E7%9A%84%E5%B9%BE%E5%80%8B%E5%95%8F%E9%A1%8C|title=抓姦前必須想過的幾個問題|accessdate=20190418|author=徵信社阿宅|date=|publisher=}}</ref>

= 民眾可以「自行抓姦」嗎? =
理論上「可以」，但大部分民眾不諳相關法條規定，也無法掌握蒐證的適當尺度與時間，<ref>{{Cite web|url=https://putavirgo1.pixnet.net/blog/post/34160889-%E5%8F%AF%E4%BB%A5%E8%87%AA%E8%A1%8C%E6%8A%93%E5%A7%A6%E5%97%8E---%E7%95%A5%E8%AB%87%E6%89%80%E8%AC%82%E7%9A%84%22%E6%8D%89%E7%8C%B4%22|title=可以自行抓姦嗎-略談所謂的"捉猴"|accessdate=20190418|author=徵信社阿宅|date=|publisher=}}</ref>

所以自行抓姦常常落得「證據不足」或「被對方反告」的下場。

此外，在台灣，合法律師通常不會幫忙委託人外遇蒐證，

這樣的情況，就是民眾儘管對徵信社觀感不好，但碰到婚姻問題還是得求助「徵信社」的原因。

=台灣民眾想「抓姦」會求助哪些單位呢?=
台灣民眾想「抓姦」，通常會經由幾個管道求助。

# 律師事務所－打官司離婚
# 徵信社－蒐證
# 警察局－偕同民眾前往現場蒐證

當然，也有民眾氣不過直接告上法院，這類案例在台灣其實也不少見。

= 「抓姦」的費用 =
抓姦在台灣雖有「通姦罪」法條保障，但不可諱言還是一件具備風險的事情。

也因此，台灣徵信業界的「抓姦費用」都居高不下，動輒20~30萬以上。

甚至達到上百萬也不是奇怪的事。<ref>{{Cite web|url=https://putavirgo1.pixnet.net/blog/post/34192931-%E5%BE%B5%E4%BF%A1%E7%A4%BE%E9%98%BF%E5%AE%85%E5%BF%83%E5%BE%97-%E8%AB%87%E5%BE%B5%E4%BF%A1%E7%A4%BE%E5%83%B9%E6%A0%BC|title=談徵信社價格|accessdate=20190418|author=|date=|publisher=}}</ref>
