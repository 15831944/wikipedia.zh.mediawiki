'''參孫'''（天主教譯名為'''Samson、三松'''）是[[聖經|聖經]][[士師記|士師記]]（天主教譯為民長紀）中的一位[[猶太人|猶太人]]士師（民長），生於前11世紀的以色列，[[瑪挪亞|瑪挪亞]]（或譯瑪諾亞）的兒子。參孫以藉著[[上帝|上帝]]所賜極大的力氣，徒手擊殺雄[[獅子|獅]]並隻身與[[以色列|以色列]]的外敵[[非利士人|非利士人]]（或譯培肋-{舍}-特人）爭戰周旋而著名。

== 出生 ==
参孙出自[[但支派|但支派]]，父亲是瑣拉人瑪挪亞，他的妻子本不孕<ref>[[士師記|士師記]]十三章：2</ref>。後來上帝的恩典臨到他們，上帝的使者指示他們要生一個兒子，且這個兒子一出生就要歸給上帝作[[拿細耳人|拿細耳人]]（Nazirite），上帝要藉著他拯救以色列百姓脫離敵人的壓迫，這個孩子就是参孫<ref>士師記十三章：3-5</ref>。

== 事蹟 ==
参孫在上帝帶有能力的應許中出生，並在上帝的眷顧下成長，他擁有上帝所賜超人的力氣，他就用這力量去攻擊非利士人，利用各種機會制伏他們，曾有一次，他用一塊未乾的驢腮骨，擊殺一千個敵人<ref>士師記十五章：15-16</ref>。参孫作以色列人的士師二十年。

== 失敗 ==
[[File:Samson_and_Delilah_by_Rubens.jpg|thumb]]
参孫蒙上帝賜的力氣，本當可以有更大的作為，可惜他個性頑強<ref>士師記十四章：3</ref>，不能接受旁人的提醒及勸勉；尤其是在婚姻的事上，不尊重以色列人的律法和父母的勸戒，隨自己的喜好，娶[[非利士|非利士]]女子為妻。他更隨意放縱肉體的情慾，與[[妓女|妓女]]和壞女子[[大利拉|大利拉]]（Delilah，或譯德-{里}-拉）交往<ref>士師記十六章：1，4</ref>，不知儆醒，以為自己可以一面享受肉欲，一面保有力量，最後無法抵擋女色的誘惑和纒累，洩露了超人力氣的來源和秘密（也就是头发不能被剪掉）<ref>士師記十六章：15-17</ref>，給敵人有可乘之機，被非利士人挖其雙眼並被囚於監獄中推磨，受盡羞辱。

== 悔改 ==
参孫因頭髮被剪，力量全失，被敵人關在監裡推磨；雖然心中懊悔，卻也無計可施。後來，他的頭髮又漸漸長了起來。一次，非利士人要向他們的神祇[[大袞|大袞]]（Dagon，或譯「達貢」）獻大祭，想利用此時再次羞辱参孫，於是就把参孫從牢裡帶出來，在他們的廟裏戲耍他<ref>士師記十六章：25</ref>。此時参孫向上帝懺悔，求上帝再賜給他力量；他就抱住廟中兩根主要支柱，身體盡力往前傾，結果神廟倒塌，壓死了3,000多人，參孫自己也犧牲了。
