'''贡姓'''是中文姓氏之一，在《[[百家姓|百家姓]]》中排第294位，是现代极罕见的姓氏。
== 起源 ==
出自[[端木姓|端木姓]]，以祖先名字为姓。据《[[元和姓纂|元和姓纂]]》载，[[春秋时期|春秋时期]][[孔子|孔子]]的著名弟子[[端木赐|端木赐]]，字[[子贡|子贡]]，他的后代就有以其字“贡”为姓的。

《[[汉书|汉书]]·[[朱博|朱博]]传》说，贡也作赣。

== 郡望 ==
*[[广平郡|广平郡]]：[[汉景帝|汉景帝]]时置，今[[河北|河北]][[永年县|永年县]]一带。

==歷史名人==
*[[贡禹|贡禹]]：前汉琅琊人。[[汉朝|汉朝]]官员，西汉大臣、博士，官至[[御史大夫|御史大夫]]、[[河南令|河南令]]。
*[[贡祖文|贡祖文]]：宋长垣（今北京市）人。[[南宋|南宋]]官员，曾任武德大夫都总军将。被任命为“秣陵总镇”，并嘉封为“武德大夫”。
*[[贡奎|贡奎]]：元朝宣城人。仕元为斋山书院山长。
*[[贡师道|贡师道]]：元朝时文士。翰林学士和国史编修，朝廷的编史
*[[贡师泰|贡师泰]]：元文学家，宣城（今属安徽）人。官至礼部、户部尚书。
*[[贡汝成|贡汝成]]：[[明朝|明朝]]官员，曾任[[翰林院|翰林院]][[侍诏|侍诏]]。
*[[贡沛诚|贡沛诚]]：江苏省武进人。先后任中央政治学校训导、中央政治学校附设边疆学校副主任兼研究部边务组组长、甘肃省第六区行政督察专员、甘肃省第一区行政督察专员、重庆市政府地政局局长、浙江省政府委员、浙江省建设厅厅长。新国成立后，任芜湖市第四中学地理教员。为民革中央团结委员会委员，江苏省民革顾问，镇江市政协委员。

==現代人物==
*[[贡华章|贡华章]]：男，中華人民共和國籍，中國共產黨党员，教授级高级会计师，中国石油天然气集团公司原总会计师。
*[[贡米_(演员)|贡米]]：女，中華人民共和國籍，演員、歌手。
