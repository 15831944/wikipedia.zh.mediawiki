'''骰（-{zh-hans:tóu;zh-hk:ㄊㄡˊ;zh-tw:ㄊㄡˊ;}-）子'''，亦作'''色（-{zh-hans:shǎi;zh-hk:ㄕㄞˇ;zh-tw:ㄕㄞˇ;}-）子'''，通常作為桌上遊戲的小道具，是古老的賭具之一。骰子也是容易製作和取得的[[亂數|亂數]]產生器。

==歷史==
骰子在五千年前西亞地區就有使用。最早期的骰子並非現在常見的正多面體，而是[[角錐|角錐]]或棒狀的，正多面體的骰子是由[[牛|牛]]或[[羊|羊]]的[[距骨|距骨]]刻成，[[古埃及|古埃及]]、[[古希臘|古希臘]]人與[[古羅馬|古羅馬]]人就有用距骨玩拋擲遊戲。玩法通常是將[[跖骨|跖骨]]拋上，用手接下，同抓布包遊戲一樣考驗小孩的[[神經反應|神經反應]]。跖骨因能擲出四面，可作為骰子遊戲，也被認為是六面骰的前身。

[[中國|中國]]的骰子習慣在一點和四點漆上紅色，据[[清代|清代]][[赵翼|赵翼]]考证红四点最早为[[唐玄宗|唐玄宗]]所使用。<ref>{{cite book|title=《好赌的中国人》|chapter=宫廷内的赌博游戏|author=江泛|url=http://vip.book.sina.com.cn/book/chapter_127587_86916.html}}</ref>

== 面數 ==
最常見的骰子是六面骰，它是一顆[[正立方體|正立方體]]([[正六面體|正六面體]])，上面分別有一到六個孔（或數字），其相對兩面之數字和必為七。在桌上遊戲中常見的正多面體骰子有四面、八面、十面、十二面和二十面骰。

此外還有一些稀有的多面骰子，例如十四面、三十面、六十面、一百二十面骰。以及不太具實用性質的一面骰([[莫比烏斯帶|莫比烏斯帶]])、五面骰、一百面骰和球型骰子等各種不同的種類。

== 重心 ==
理論上，完全公正的骰子的重心應該在正中央，使得擲出每一面的[[機率|機率]]完全相同。但這在製作上並不容易，以六面骰子為例，由於點數六的凹洞較點數一的凹洞多，因此在沒有特別調整的前提下，點數一的那一面勢必會比較重。

此外，灌[[鉛|鉛]]、使用[[磁鐵|磁鐵]]是控制骰子重心與擲骰結果的方式，可以做為[[詐賭|詐賭]]之用。

== 骰子與數學 ==
=== 賽克文的骰子 ===
兩顆不同的六面骰子，擲出來的結果可能跟兩顆相同的一般骰子一樣。其中只有一種組合的不同六面骰子每面都是正整數，稱為賽克文的骰子（{{lang|en|Sicherman's dice}}）。它的表面分別為1, 2, 2, 3, 3, 4及1, 3, 4, 5, 6, 8。對於若骰子表面為[[正四面體|正四面體]]，這些表面則為1,2,2,3及1,3,3,5。這類骰子可以用[[母函数|生成]][[函數|函數]]去求得。

=== 非遞移骰子（Non-Transitive Dice） ===
[[File:intransitive_dice.png|frame]]

考慮以下的遊戲：給定一組骰子。甲先選一顆，乙選另一顆不相同的骰子。乙有可能永遠都有辦法使得自己取勝的[[機率|機率]]較高嗎？

由於勝出的相對機率並非[[遞移關係|遞移關係]]，這樣的骰子是有可能的。

[[File:Efron_dice.png|right]]

Efron骰子便是四個符合以上條件的骰子。將它們分別稱為A,B, C,D，其中A勝B、B勝C、C勝D和D勝A的機率均為2/3。這些骰子的表面分別是：
* A: 4, 4, 4, 4, 0, 0
* B: 3, 3, 3, 3, 3, 3
* C: 6, 6, 2, 2, 2, 2
* D: 5, 5, 5, 1, 1, 1
