{{作品名稱|name=''Pet Sematary''|hk=詭墓|tw=禁入墳場|cn=宠物坟场}}是一部2019年美國[[懸疑片|懸疑]][[驚悚片|驚悚片]]，這是第二次改編自[[史蒂芬·金|史蒂芬·金]]的1983年[[禁入墳場|同名小說]]，第一次改編是[[夜半仔敲門|1989年電影]]。該片由凱文·科爾施（{{lang|de|Kevin Kölsch}}）和丹尼斯·維德米耶（{{lang|en|Dennis Widmyer}}）共同執導，傑夫·布希勒（{{lang|en|Jeff Buhler}}）編劇，[[傑森·克拉克|傑森·克拉克]]、{{link-en|艾美·塞梅茲|Amy Seimetz}}及[[約翰·李斯高|約翰·李斯高]]主演。其劇情圍繞著路易斯一家人在搬到鄉間的一處動物墳場旁後，卻碰上了一連串的怪事。該片定於2019年4月5日在美國上映。

== 劇情 ==
路易斯·克里德博士和他的太太瑞秋，以及他的一對子女從波士頓搬到緬因州的鄉間之後，發現了一處隱藏在這一家人的新家附近的樹林深處的神秘墓場。當一場悲劇發生之後，路易斯就向他異於常人的鄰居賈德·柯蘭鐸求助，觸發了一連串的連鎖反應，並且釋放了一種令人無法想像的邪惡力量，造成了可怕的後果。

== 製作 ==
2010年3月，宣布[[史蒂芬·金|史蒂芬·金]]小說《[[禁入墳場|禁入墳場]]》將推出重拍版電影（該書曾於1989年推出過[[夜半仔敲門|改編電影]]，以及1992年的續集《{{link-en|禁入墳場2|Pet Sematary Two}}》），{{link-en|麥特·格林伯格|Matt Greenberg}}正在撰寫劇本<ref>{{cite web |last1=Outlaw |first1=Kofi |title='1408' Writer Headed for the 'Pet Sematary' {{!}} ScreenRant |url=https://screenrant.com/pet-sematary-remake-writer/ |website=Screen Rant |accessdate=2018-06-18 |date=2010-03-05}}</ref>。2013年10月，製片人[[洛倫佐·迪·波納文圖拉|洛倫佐·迪·波納文圖拉]]和史蒂文·施奈德（{{lang|en|Steven Schneider}}）將參與製作，且由{{link-en|璜·卡洛斯·佛瑞斯納迪歐|Juan Carlos Fresnadillo}}執導<ref>{{cite web|url=https://variety.com/2013/film/news/juan-carlos-fresnadillo-pet-sematary-paramount-1200782223/|title=‘Pet Sematary’ Back From the Dead With Director Juan Carlos Fresnadillo (EXCLUSIVE)|last=Kroll|first=Justin|website=[[Variety_(magazine)|Variety]]|date=2013-10-31|accessdate=2018-06-18}}</ref>。

2017年12月7日，[[派拉蒙影業|派拉蒙影業]]宣布，新版《禁入墳場》電影將由凱文·科爾施（{{lang|de|Kevin Kölsch}}）和丹尼斯·維德米耶（{{lang|en|Dennis Widmyer}}）將根據傑夫·布希勒（{{lang|en|Jeff Buhler}}）的劇本來執導，而迪·波納文圖拉、施奈德及馬克·瓦拉迪安（{{lang|en|Mark Vahradian}}）則會擔任監製<ref>{{cite web|last1=D'Alessandro|first1=Anthony|title=Paramount Sets 2019 Release For ‘Pet Sematary’ & ‘Instant Family’|url=http://deadline.com/2017/12/paramount-sets-2019-dates-for-pet-sematary-instant-family-1202222522/|website=Deadline|accessdate=2018-05-04|date=2017-12-08}}</ref>。

2018年4月16日，宣布[[傑森·克拉克|傑森·克拉克]]加入擔任主演路易斯·克里德（{{lang|en|Louis Creed}}）<ref name="Clarke">{{cite web|last1=Kit|first1=Borys|title=Jason Clarke in Talks to Star in 'Pet Sematary' Remake (Exclusive)|url=https://www.hollywoodreporter.com/heat-vision/jason-clarke-talks-star-pet-sematary-remake-1102774|website=The Hollywood Reporter|accessdate=2018-05-04|language=en|date=2018-04-16}}</ref>。2018年5月4日，據報導，[[約翰·李斯高|約翰·李斯高]]已加入飾演賈德·柯蘭鐸（{{lang|en|Jud Crandall}}）一角<ref name="Lithgow">{{cite web|last1=Breznican|first1=Anthony|title=John Lithgow joins the remake of Stephen King's 'Pet Sematary'|url=http://ew.com/movies/2018/05/04/john-lithgow-jud-crandall-stephen-king-pet-sematary-remake/|website=Entertainment Weekly|accessdate=2018-05-04|language=en|date=2018-05-04}}</ref>。2018年6月1日，宣布{{link-en|艾美·塞梅茲|Amy Seimetz}}將飾演女主角瑞秋（{{lang|en|Rachel}}）<ref name="Seimetz">{{cite web |last1=Fleming Jr |first1=Mike |title=Amy Seimetz Female Lead In ‘Pet Sematary’ Remake |url=https://deadline.com/2018/06/amy-seimetz-pet-sematary-remake-stephen-king-1202402295/ |website=Deadline |accessdate=2018-06-01 |date=2018-06-01}}</ref>。

[[主要拍攝|主要拍攝]]於2018年6月18日開始進行<ref>{{cite web |last1=Trumbore |first1=Dave |title='Pet Sematary' Adaptation Starts Filming; First Set Photo Revealed by Co-Directors |url=http://collider.com/pet-sematary-filming/ |website=Collider |accessdate=2018-06-18 |date=2018-06-18}}</ref>。
