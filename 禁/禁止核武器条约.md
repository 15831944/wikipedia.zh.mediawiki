《'''禁止核武器条约'''》（{{lang-en|Treaty on the Prohibition of Nuclear Weapons}}），是2017年开始谈判的旨在使[[核武器|核武器]][[关于核武器的争论|非法化]]的国际条约。

==内容==
《禁止核武器条约》最早是在2007年由[[哥斯达黎加|哥斯达黎加]]、[[马来西亚|马来西亚]]共同向[[联合国|联合国]]提交的提案。其类似于《[[禁止生物武器公约|禁止生物武器公约]]》（BWC），不设核查机制，意即仅希望“禁止核武器的拥有、研发、储存、转移、试验，或是威胁使用”，让核武器“完全非法化”，直接规定任何核武器均违反[[国际人道主义法|国际人道主义法]]。该条约将参照《[[地雷#《全面禁止杀伤人员地雷公约》（渥太华公约）|渥太华公约]]》和《[[国际禁止集束炸弹公约|国际禁止集束炸弹公约]]》，要求各国销毁所持有的核武器。所以该提案与此前的各种核裁军条约以及《[[不扩散核武器条约|核不扩散条约]]》、《[[全面禁止核试验条约|全面禁止核试验条约]]》完全不同<ref name=guancha>{{cite news|title=堂下何人狀告本官？沒有核國家參加的禁核條約會議在聯合國舉辦|publisher=觀察者網|date=2017-03-27|author=荀越|url=http://www.guancha.cn/military-affairs/2017_03_27_400784.shtml|archiveurl=https://web.archive.org/web/20170616210707/http://www.guancha.cn/military-affairs/2017_03_27_400784.shtml|archivedate=2017-06-16|deadurl=yes}}</ref>。

《禁止核武器条约》草案中将写入禁止使用核武器等内容，这会影响以核威慑力为核心的国际安全体系。所以主要[[核武器擁有國列表|拥核国家]]强烈反对并坚决抵制《条约》的制定，其他拥核国家随之也不参与《条约》的制定。

==历史==
[[File:UN_working_group_on_nuclear_disarmament,_May_2016.jpg|thumb]]
2016年10月27日，[[联合国大会|联合国大会]][[联合国大会#附属委员会|第一委员会（裁军与国际安全委员会）]]以126票赞成，38票反对，16票弃权的结果通过了规定2017年启动《禁止核武器条约》谈判的决议案。该决议案由[[奥地利|奥地利]]、[[墨西哥|墨西哥]]、[[巴西|巴西]]、[[爱尔兰|爱尔兰]]、[[南非|南非]]等非拥核国主导提出。决议案表达了对因为使用核武器而导致人道毁灭的担忧，规定各国为制定旨在禁止核武器且具有法律约束力的文件而谈判。该决议案规定，《条约》的谈判于2017年3月27日至31日、6月15日至7月7日在[[纽约|纽约]]采用少数服从多数等联合国大会规则展开，[[非政府组织|非政府组织]]（NGO）及[[公民社会|公民社会]]也将参加<ref name=lhzb>{{cite news|title=聯合國通過決議案 禁止核武條約談判明年啟動|publisher=聯合早報|date=2016-10-29|url=http://www.zaobao.com.sg/news/world/story20161029-683617|archiveurl=https://web.archive.org/web/20170616211218/http://www.zaobao.com.sg/news/world/story20161029-683617|archivedate=2017-06-16|deadurl=yes}}</ref><ref name=quexi>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1649395 |title = 禁核条约谈判启动：核大国全部缺席，谈判意义何在 |author =  |date = 2017-03-28 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

在此次表决中，以[[发展中国家|发展中国家]]为主的123个国家表示赞成。表示反对的38个国家中包括了[[美国|美国]]、[[英国|英国]]、[[法国|法国]]、[[俄罗斯|俄罗斯]]等主要[[核武器擁有國列表|拥核国家]]。[[北约|北约]]28个成员国中仅[[荷兰|荷兰]]弃权，其他27国全部表示反对。歷史上唯一曾遭核武器攻擊的國家[[日本|日本]]也表示反对。非拥核国家[[澳大利亚|澳大利亚]]和[[加拿大|加拿大]]等也表示反对。表示弃权的16个国家中包括[[中国|中国]]、[[印度|印度]]、[[巴基斯坦|巴基斯坦]]等拥核国家<ref name=guancha/><ref name=lhzb/><ref>{{Cite web |url =http://www.icanw.org/campaign-news/results/  |title = Full voting result on UN resolution L.41 |author =  |date = 2016-10-27 |publisher = ican |language =  |accessdate =  |quote =  }}</ref>。

*'''{{US}}'''就《条约》指出，该条约生效前便会对美国的核威慑力“造成直接影响”，并且条约的非缔约国也会受到影响，条约将“妨碍实际的核裁军”<ref name=sina/><ref name=lhzb/><ref name=quexi/>。
*'''{{flag|NATO}}'''及'''{{JPN}}'''、'''{{AUS}}'''等美国的同盟国，在美国的呼吁下共同反对该条约的制定<ref name=lhzb/><ref name=quexi/>。2016年时，美国致信北约各国，强烈要求对规定启动谈判的联合国大会决议案投反对票和不参加谈判<ref name=kuai>{{Cite web |url = http://www.guancha.cn/internation/2017_07_07_417094.shtml |title = 快讯！联合国通过全球禁止核武器条约 会议无核国家参加 |author =  |date = 2017-07-07 |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref>。日本原先推动该条约颇为积极，但在美国的压力下最终连相关谈判都未参加<ref name=ruidian/>。
*'''{{RUS}}'''由于美国反对该条约，也绝不希望因为该条约而减损自身核实力<ref name=sina/>。
*'''{{UK}}'''、'''{{FRA}}'''作为核武器数量较小的国家，更不会在美国、俄罗斯这两个拥有最多核武器的国家都未让步的前提下表态<ref name=sina>{{Cite web |url = http://news.sina.com.cn/w/sy/2017-03-21/doc-ifycnikk1390150.shtml |title =专家:禁止核武条约谈判 为啥五大国都不参加  |author =  |date = 2017-03-21 |publisher = 新浪 |language =  |accessdate =  |quote =  }}</ref>。
*'''{{DPRK}}'''在2016年10月27日的表决中投了赞成票。2017年3月24日，[[朝鮮民主主義人民共和國外務省|朝鲜外务省]]发言人表示，鉴于美国2017年将战略武器大规模引入[[朝鲜半岛|朝鲜半岛]]，并实施联合军演，[[朝鲜民主主义人民共和国|朝鲜]]必须强化以核武力为中枢的自卫性国防力量，因此朝鲜决定不参加《条约》谈判<ref>{{Cite web |url = http://news.ifeng.com/a/20170326/50837454_0.shtml |title =  不参加联合国禁止核武器条约谈判|author =  |date = 2017-03-26 |publisher =  凤凰网|language =  |accessdate =  |quote =  }}</ref>。
*'''{{CHN}}'''[[中华人民共和国外交部发言人|外交部发言人]][[华春莹|华春莹]]在2017年3月20日回答记者提问时表示，中国将不参加《条约》谈判，但中方坚定支持最终全面禁止和彻底销毁核武器的立场没有改变，愿与各方保持沟通，继续为建立无核武器世界而努力。中国也是最后一个表态的[[联合国安理会|联合国安理会]][[常任理事国|常任理事国]]。至此五个常任理事国均表示不参加该谈判<ref name=sina/>。

2017年2月中旬，为启动该条约的谈判而举行了筹备会议，中国等国家参加了筹备会议<ref name=sina/><ref name=guancha/>。

2017年3月27日至31日，《禁止核武器条约》谈判第一阶段会议如期在[[纽约|纽约]][[联合国大厦|联合国大厦]]举行。没有任何一个拥核国家参加谈判<ref name=guancha/>。

2017年7月6日，反对《禁止核武器条约》的美国政府高官以回覆哈萨克斯坦驻联合国代表邀请函的形式，向日本、澳大利亚、[[以色列|以色列]]以及参加谈判的[[新加坡|新加坡]]、[[荷兰|荷兰]]等友好国家和同盟国政府相关人士发出邮件，要求上述各国缺席该相关会议，邮件中明确称“美国对有关《条约》的一切活动均不参加。希望友好国家和同盟国也不要参加该会议”，“美国的立场不会改变”<ref name=kuai/>。

2017年7月7日，《禁止核武器条约》谈判第二阶段会议以122票赞成、1票反对（[[荷兰|荷兰]]）、1票弃权（[[新加坡|新加坡]]）通过了该条约，这是历史上第一次通过在全球范围内禁止核武器的条约，包括联合国安理会五个常任理事国在内的所有拥核国家全部缺席<ref>{{Cite web|url=https://www.un.org/disarmament/ptnw/index.html|title=United Nations Conference to Negotiate a Legally Binding Instrument to Prohibit Nuclear Weapons, Leading Towards their Total Elimination, 27 April to 22 May 2015|accessdate=2017-07-07|work=www.un.org|language=EN}}</ref><ref name=kuai/>。

2017年7月11日，[[哈萨克斯坦|哈萨克斯坦]]驻联合国代表处主办讨论谈判结果的相关会议，联合国裁军办公室官员和推进《禁止核武器条约》制定的南非、巴西、爱尔兰大使等出席，主要议题是制定《条约》的谈判<ref name=kuai/>。

2017年8月，[[瑞典|瑞典]]外交大臣[[玛戈特·瓦尔斯特伦|瓦尔斯特伦]]表示将签署该条约。2017年8月29日，[[美国国防部长|美国国防部长]][[詹姆斯·马蒂斯|马蒂斯]]致信瑞典国防大臣[[彼得·胡尔特奎斯特|胡尔特奎斯特]]，威胁称瑞典一旦签署该条约，则无论在和平或危机时期，瑞典均可能无法自美国获取足够的军事支持<ref name=ruidian>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1784395  |title =美防长致信瑞典防长公然施压：不得签署《禁止核武器公约》  |author =  |date =2017-09-04  |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。

2018年10月29日，[[联合国安理会|联合国安理会]]五个[[常任理事国|常任理事国]]（中国、俄罗斯、法国、英国、美国）发表联合声明，明确反对《禁止核武器条约》。声明表示五大国希望能建设一个无核武器的世界，但现有的《禁止核武器条约》激进而不切实际，五大国不会遵守这一条约<ref>{{cite news |title=中俄美法英突然“团结”一致，发表共同声明！ |url=http://news.ifeng.com/a/20181029/60134339_0.shtml |accessdate=2018-10-29 |agency=环球时报 |publisher=凤凰网 |date=2018-10-29 |language=zh-hans}}</ref><ref>{{cite web|url=https://www.gov.uk/government/news/p5-joint-statement-on-the-treaty-on-the-non-proliferation-of-nuclear-weapons |title=P5 Joint Statement on the Treaty on the Non-Proliferation of Nuclear Weapons |publisher={{le|GOV.UK}} |date=2018-10-24 |accessdate=2019-02-06 }} </ref>。
