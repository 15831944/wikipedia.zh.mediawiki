
'''禁慾'''是一種[[生活型態|生活型態]]，其特徵通常是對若干俗世歡愉的禁絕，以達至宗教上及靈性上的目標。[[基督宗教|基督宗教]]（特別是[[修道運動|修道運動]]）及[[印度宗教|印度宗教]]（包括[[瑜伽|瑜伽]]）的教導，涉及到「意志—身體」轉化的過程，此過程乃透過對身、言、意方面的控制。

在某些宗教裡，[[神職人員|神職人員]]或其他的虔誠信徒，為了更接近[[神祇|神祇]]或悟得[[宇宙|宇宙]][[真理|真理]]，而採取此類措施，尤其是[[性禁慾|性禁慾]]。

== 基督宗教的禁慾 ==
[[基督宗教|基督宗教]]禁慾主義以[[聖奧古斯丁|聖奧古斯丁]]的學說最為出名，他尊崇[[理性|理性]]，貶低情慾；又認為[[性慾|性慾]]會使人的理性失去自主權；更認為[[性衝動|性衝動]]反應代表著人類犯罪，當有慾望時應該感到羞愧。所以有性慾是可恥的，是犯罪的。

== 佛教的禁慾 ==
[[佛教|佛教]]认为，世界有[[五欲|五欲]]，而[[淫欲|淫欲]]是其首要，正如《[[楞严经|楞严经]]》说“淫心不除，尘不可出。”如果要脱离[[六道轮回|六道轮回]]趋入[[涅槃|涅槃]]，必须断掉淫欲<ref>《[[杂阿含经|杂阿含经]]》说“若诸众生所有苦生,一切皆以爱欲为本。”《楞严经》说“淫心不除，尘不可出。”</ref>。

故对於出家[[僧侣|僧侣]]要求全断淫欲，称爲“[[净行|净行]]”，是一种善法，帮助从[[轮回|轮回]]中脱离而灭苦；不断五欲爲“不净行”，是不善法，会让人深陷在轮回中。

为了遮止五欲贪，除了节制淫欲，出家僧侣还有非常多的相关戒律，比如新鞋必须由他人穿几遍才能自己使用，新得布料不能用整塊必须切割连缀製成[[袈裟|袈裟]]方能使用，以此破除对衣鞋的[[取_(佛教)|执着心]]。甚至一日一餐或两餐，按律过[[中午|午]]則不應食，[[化緣|化緣]]飲食不得挑剔，俱應接受，连对饮食的貪愛也要節制。在[[大乘佛教|大乘佛教]]中，[[斋食|戒食肉荤]]同样很重要。[[沙彌|沙彌]][[十戒_(佛教)|十戒]]和[[八關齋戒|八關齋戒]]中還包括不欣賞[[情歌|情歌]][[豔舞|豔舞]]和不塗香抹粉等。可以說，佛教的修行就是始於對五欲的[[厭離|厭離]]、禁斷。

而对在家居士而言，虽然不及出家人严格禁欲，但也需要恪守最基本的五条原则，即“[[五戒|五戒]]”：不殺生、不偷盜、不邪淫、不妄語、不飲酒。其中的“不[[邪淫|邪淫]]”即不[[欲邪行|欲邪行]]，即允许在家居士进行常规夫妻生活，但不得以不正常、过度追求取乐甚至违背[[人伦|人伦]]、与非夫妻之人行婬。

== [[伊斯蘭教|伊斯蘭教]]的禁慾 ==
[[穆斯林|穆斯林]]不許接触有关[[猪|猪]]的产品，不可以[[飲酒|飲酒]]、[[赌博|赌博]]、[[嫖妓|嫖妓]]，甚至與非[[配偶|配偶]]的人[[性交|性交]]。

== 其他禁慾行為 ==
德國哲學家[[叔本華|叔本華]]主張禁欲，他說：“欲望越强烈，越贪求我欲之满足的人，他所感到的痛苦也就更多更深。因为欲望经常附在他身上不断地啃噬他，使他的心灵充满苦恼。如此积久成习之后，一旦欲望的对象全部消失，他几乎便以看别人的痛苦为乐了。反之，一个否定求生意志的人，从外表看起来，他的确是贫穷、一无所有、既无欢乐亦无生趣的人，则心灵则是一片清澄，充满宁静和喜悦。他们不会被不安的生存冲动或欢天喜地的事情所驱策，因为这些都是强烈痛苦的先导。他们不贪图生之快乐，因为喜悦过后往往是接续苦恼的状态。”<ref>《爱与生的苦恼》.叔本華</ref>
