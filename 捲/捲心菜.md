'''甘藍'''（[[學名|學名]]：''Brassica oleracea'' var. ''capitata''），植物學上稱'''结球甘蓝'''，又稱'''洋白菜'''、'''圓白菜'''、'''莲花菜'''、'''疙瘩白'''，是一種常見[[蔬菜|蔬菜]]。[[山西|山西]]、[[陕西|陕西]]等地区多称'''莲花白'''、'''花白'''或'''茴子白'''，廣東部分地區亦稱“'''椰菜'''”'''或'''"'''包菜'''"。在閩南地區及[[臺灣|臺灣]]習稱-{'''高麗菜'''}-或'''甘藍菜'''也有人稱'''玻璃菜'''<ref>[https://icook.tw/recipes/61378 香菇玻璃菜牛肉粥]</ref>。

== 形态 ==
二年生[[草本|草本]]，卵圆形厚叶片，有黄绿、深绿或蓝绿色；叶柄短、叶心抱合成球状，呈黄白色。按卷心形状分尖头形、平头形、圆头形三种。

== 历史 ==
甘藍起源於[[地中海|地中海]]沿岸。其實古[[希臘|希臘]]人和古[[羅馬|羅馬]]人已廣泛種植。至中世紀以後廣泛傳遍世界。甘藍由[[荷蘭|荷蘭]]帶到荷屬台灣，荷蘭語為「Kool」，故高麗二字源自荷蘭語，[[清朝|清朝]]《台灣方志》名稱為番芥藍（hoan-kè-nâ）。

== 形态 ==
甘藍為卵圆形，外層有藍綠色厚葉片，葉柄短；心葉呈黄白色，抱合成球；依葉球形狀可分為尖球形／圓錐形、圓球形和扁圓形三種。

甘藍有公、母之分，公的甘藍外觀比較尖，筋軟味甜。母的外表較圓，纖維較多。<ref name="性別">{{cite news|title=蔬菜性別大揭密！青椒、冬瓜、高麗菜分「公母」 |url=https://tube.chinatimes.com/20131116002320-261402 |language=zh-tw |author=梁嘉文、黃琲茹 |publisher=中時電子報 |date=2013-11-16 |accessdate=2018-12-10 }}</ref>

甘藍营养相当丰富，含有大量[[维生素C|维生素C]]、[[纤维素|纤维素]]、[[碳水化合物|碳水化合物]]和各种[[矿物质|矿物质]]。除此以外甘藍还含有維生素U<ref>https://tw.news.yahoo.com/%E4%B8%8D%E6%98%AF%E7%B6%AD%E7%94%9F%E7%B4%A0%E7%9A%84%E7%B6%AD%E7%94%9F%E7%B4%A0u-%E8%AE%93%E9%AB%98%E9%BA%97%E8%8F%9C%E5%85%B7%E6%9C%89%E9%A1%A7%E8%85%B8%E8%83%83%E7%9A%84%E5%8A%9F%E6%95%88-070000687.html</ref>。维生素U是抗溃疡因子，并具有分解[[亚硝酸胺|亚硝酸胺]]的作用。甘藍裡的[[吲哚|吲哚]]（indole）能改變雌激素的代謝，降低乳癌風險，其所含有的[[異硫氰酸鹽|異硫氰酸鹽]]，可降低致癌物的毒性，有效預防肺癌和食道癌。甘藍中含有的{{le|蘿蔔硫素|Sulforaphane}}，則是功能強大的抗氧化物，可以增強體內酵素的解毒能力，也是維生素C和植物纖維的良好來源。

==流行文化==
在日本电视动画[[夜明前的琉璃色|夜明前的琉璃色]]中，其中一个下厨情节，一个本应该是卷心菜的东西被绘画成“青绿色的圆形球体”，被不少观众称为“同心圆椰菜”，成为[[作画崩坏|作画崩坏]]的一个经典反面教材。
