'''多明尼加復國戰爭'''（英語：Dominican Restoration War）是1863年[[多明尼加|多明尼加]]為二度獨立並脫離[[西班牙|西班牙]]統治，而展開的復國戰爭，1865年西班牙軍隊撤退，多明尼加成功復國。

== 背景 ==
{{See also|多明尼加共和國歷史}}
[[多明尼加|多明尼加]]在[[佩德羅·桑塔納|佩德羅·桑塔納]]的率領反抗[[海地|海地]]的統治，並在脫離海地的[[多明尼加獨立戰爭|獨立戰爭]]中獲勝，獲得短暫的獨立（1844-1861年），[[佩德羅·桑塔納|佩德羅·桑塔納]]成為第一任[[多明尼加總統|多明尼加總統]]。但獨立後的多明尼加一片混亂，內有嚴重的經濟崩潰問題，外有海地的威脅。在內憂外患的情況下，[[佩德羅·桑塔納|佩德羅·桑塔納]]和西班牙女王[[伊莎貝爾二世|伊莎貝爾二世]]和[[古巴總督|古巴總督]]談判，準備讓多明尼加重新回到西班牙的統治。此時[[美國|美國]]因為[[內戰|內戰]]（[[南北戰爭|南北戰爭]]）如火如荼的進行，所以無暇顧及[[拉丁美洲|拉丁美洲]]的事務（1823年美國發表宣言，歐洲國家如果對美洲進行殖民，或對其干預，都將視為侵略，美國都將介入，此宣言稱為[[門羅主義|門羅主義]]），西班牙認為有機可乘，因為控制多明尼加是重新控制拉丁美洲的第一步。1861年3月18日，西班牙正式兼併多明尼加，[[佩德羅·桑塔納|佩德羅·桑塔納]]成為[[西屬|西屬]][[多明尼加|多明尼加]]的[[總督|總督]]。
但許多多明尼加人都反對回歸西班牙統治，於是1863年，由羅里蓋茲、孟席歐、[[格雷戈里奧·盧柏龍|盧柏龍]]三人領導的多明尼加復國戰爭正式展開。<ref>{{Cite web |url = http://www.spanishwars.net/19th-century-dominican-restoration.html   |title = Dominican Restoration War |accessdate = 2018/01/29 }}</ref><ref>{{Cite web |url = https://books.google.com.tw/books?id=8BfRF9B02kgC&redir_esc=y |title = The Dominican Republic: A National History |accessdate = 2018/01/29 }}</ref>
