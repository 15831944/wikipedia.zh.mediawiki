'''多哈'''（{{Lang-ar|'''الدوحة'''}} {{Lang|la|''ad-Dūḥa''}}）位于[[波斯湾|波斯湾]]畔的著名港口，是[[卡達|卡達]]的首都，[[卡塔尔行政区划|卡塔尔8个区]]之一。多哈是卡塔尔的第一大城市以及全国經濟、交通和文化中心，2009年时，人口为998,651人，约占全国人口的60%。

[[第十五届亚洲运动会|第十五届亚洲运动会]]于2006年12月1日至15日在卡塔尔首都多哈进行，这可能是亚运会有史以来规模最大的一届。2011年12月，多哈举办[[第20届世界石油理事会|第20届世界石油理事会]]，同年的12月，[[2011年泛阿拉伯运动会|2011年泛阿拉伯运动会]]也于多哈举行。在2012年，多哈举办了[[第十八次联合国气候变化大会|第十八次联合国气候变化大会]]。在2022年，多哈将会举办[[2022年世界杯足球赛|2022年世界杯足球赛]]。

== 名称 ==
1825年，該地區名為「'''Al-Bidda '''」，这个名称是在1765年的卡斯滕（Carsten Niebhur）一张不准确地图上首次以英语拼法出现。现在通称的「'''Doha'''」一名来自於[[阿拉伯语|阿拉伯语]]的'''Ad-Dawḥa'''（意为「大树」），这棵树可能是在卡達半島东部海岸的一个原始的渔村；但也有來源指稱「多哈」是来自阿拉伯語“dohat”（「海湾」之意），指多哈湾周围的海滨大道（Doha Corniche）。

== 历史 ==
[[File:Doha_Towers_Through_Arabian_Arcades.jpg|thumb]]远看多哈天际线]]
[[File:Doha_corniche_view_(2).jpg|thumb]]
[[File:Qatar_Doha_west_bay.jpg|thumb]]
多哈这个城市成立于1825年。同年，卡塔尔与[[巴林|巴林]]爆发了战争。战争中，卡塔尔首都多哈已遭到严重破坏，并且邻国[[沙特阿拉伯|沙烏地阿拉伯]]援助巴林。<ref name="autogenerated2">{{cite web|author=Encyclopædia Britannica |url=http://www.britannica.com/EBchecked/topic/168027/Doha |title=Doha – Britannica Online Encyclopedia |publisher=Britannica.com |date= |accessdate=2010-06-27}}</ref>在1882年，政府在赖扬（卡特尔的[[卡塔尔行政区划|一个行政区]])西南部修建了一个堡垒。（距离多哈10英里）1883年，[[谢赫|谢赫]]{{Link-en|卡塞姆・本・穆罕默德・阿勒萨尼|Jassim bin Mohammed Al Thani}}。<ref name="diwan">{{cite web|title=Shaikh Jassim Bin Mohammed Al Thani :: Amiri Diwan|url=http://www.diwan.gov.qa/english/qatar/QassimBinMohammed.htm|publisher=Diwan.gov.qa|accessdate=17 November 2012|deadurl=yes|archiveurl=https://web.archive.org/web/20121105095124/http://www.diwan.gov.qa/english/qatar/QassimBinMohammed.htm|archivedate=2012年11月5日}}</ref>率领卡塔尔军队反抗[[奥斯曼土耳其帝国|奥斯曼帝国]]。在1893年，卡塔尔军队在赖扬修建的堡垒附近发生的战斗中取得了胜利。而且这次独立战争迫使了奥斯曼帝国签订了条约，在卡塔尔原有的基础上，以一个独立自治的国家保留在帝国内。<ref>{{cite web |url=http://www.qatarvisitor.com/index.php?cID=439&pID=1701 |title=Battle of Al Wajbah |publisher=Qatar Visitor |date=2 June 2007 |accessdate=22 January 2013 |deadurl=yes |archiveurl=https://web.archive.org/web/20130117024415/http://www.qatarvisitor.com/index.php?cID=439&pID=1701 |archivedate=2013年1月17日 }}</ref>在这时，多哈被确立为卡塔尔的首都。

在1917年，谢赫{{Link-en|阿卜杜拉·本·卡塞姆·阿勒萨尼|Abdullah bin Mohammed Al Thani}}在靠近大海的沙漠中建立了Al Koot Fort城堡。在20世纪初期，多哈有12,000人口。这时卡塔尔的经济依靠捕鱼和采珠业，多哈就有大概350多辆采珠船。然而在1930年，引进日本的养殖珍珠之后，整个国家多哈都遭陷入严重的经济危机之间。这种贫困的状况一直持续到[[石油|石油]]的发现。而这时却又爆发了[[第二次世界大战|第二次世界大战]]，因此卡塔尔的石油勘探和出口被迫的暂停。在二战后卡塔尔对石油储量的开发拯救了多哈这个城市，至今整个国家每天都还能生产出800,000桶石油。（除了丰富的石油之外，还储存了大量的天然气燃料可使用)在1969年，卡塔尔的政府机关成立。现在它被认为是卡塔尔最著名的地标。在1973年，[[卡塔尔大学|卡塔尔大学]]建立。1975年，在上一任埃米尔的宫殿建成了[[卡塔尔国家博物馆|卡塔尔国家博物馆]]。在1996年，阿拉伯语[[半岛电视台|半岛电视台]][[卫星电视|卫星电视]]新闻频道开始播放。

在2010年，卡塔尔人口已有1,696,563人，是世界上发展最快的国家之一。

== 經濟 ==
多哈以盛產[[石油|石油]]和[[天然氣|天然氣]]聞名，成為卡達的經濟命脈。許多石油氣公司的總部設立在多哈，包括[[卡塔爾石油公司|卡塔爾石油公司]]和[[RasGas|RasGas]]。由於卡達的經濟命脈是建立在石油和天然氣之上，卡達人民政府希望能多樣化他們的經濟。
== 氣候 ==
多哈处于热带[[沙漠气候|沙漠气候]]（按照[[柯本气候分类法|柯本气候分类法]]属于BWh)。多哈的夏天很长，从5月开始一直到9月份结束，这段时间其平均高温超过38°C（100°F），经常会接近47°C（117°F）。在5月和6月，湿度通常会降的很低。在夏天，[[露点温度|露点温度]]会超过25°C（77°F）。多哈每年的平均降雨量为75毫米，在整个夏天，整个城市的降雨量几乎为0，即使是在其他月份，降水量也小于20毫米。<ref>{{cite web|url=http://www.wunderground.com/NORMS/DisplayIntlNORMS.asp?CityCode=41170&Units=both |title=Doha weather information |publisher=Wunderground.com |date=2010-06-10 |accessdate=2010-06-27}}</ref>

== 人口特征 ==
多哈是个典型的移民城市，在城市定居的大多数都是[[侨民|侨民]]。他们中一部分主要来自东南亚和南亚的[[印度|印度]]，[[巴基斯坦|巴基斯坦]]，[[斯里兰卡|斯里兰卡]]、[[尼泊尔|尼泊尔]]、[[菲律宾|菲律宾]]、[[孟加拉|孟加拉]]和[[印度尼西亚|印度尼西亚]]。另外还有大量来自[[累范特|累范特]]的阿拉伯国家、[[北非|北非]]和[[东亚|东亚]]。多哈也同时也是很多国家移民所青睐的城市。在过去，卡塔尔的公民不被允许拥有自己的土地。然而非卡塔尔的公民却可以一定区域买卖土地。而且当他们在卡塔尔有土地所有权时，便可允许他们在卡塔尔工作和居住。

每个月，都有数千名[[外来移民|外来移民]]涌入卡塔尔。由于此，卡塔尔见证了城市人口的快速膨胀。也因此，卡塔尔的房地产市场发现供应的短缺导致房价上涨，也增加了物价上升。然而房地产市场供求之间的差距却渐渐缩小。
