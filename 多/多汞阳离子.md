'''多汞阳离子'''或'''多聚汞阳离子'''是仅含[[汞|汞]]原子的[[原子团|原子团]]，最常见的多汞阳离子是{{chem|Hg|2|2+}}，在亚汞化合物中存在。亚汞化合物中的Hg-Hg键通过1927年的X射线分析<ref name = "Wells3d">Wells A.F. (1962) ''Structural Inorganic Chemistry'' 3d edition Oxford Science Publications</ref>和1934年的拉曼光谱<ref name = "Greenwood">{{Greenwood&Earnshaw}}</ref>发现，并成为了得到表征的共价金属-金属键的最早的例子之一。

其它多汞阳离子还有线性的{{chem|Hg|3|2+}}和{{chem|Hg|4|2+}}离子，<ref name = "Greenwood"/>三角形的{{chem|Hg|3|4+}}离子<ref name ="Broderson">Terlinguait, Hg<sub>4</sub>O<sub>2</sub>Cl<sub>2</sub> - ein Mineral mit ungewöhnlichen Hg<sub>3</sub>-Baueinheiten, K. ==亚汞化合物==
[[File:Calomel-unit-cell-3D-balls.png|thumb]]（Hg<sub>2</sub>Cl<sub>2</sub>）的晶胞结构。]]
最著名的多汞阳离子{{chem|Hg|2|2+}}出现在亚汞化合物中，其中汞的形式氧化态为+1。{{chem|Hg|2|2+}}离子可能是被确认的第一个含金属-金属键的物种。在1898年，Ogg证实了{{chem|Hg|2|2+}}在溶液中存在。<ref>A. Ogg; Zeitschrift Physische Chemie 27, 285 (1898)</ref>在1900年，Baker发现了HgCl二聚体在气态存在。<ref>Vapour density of dried mercurous chloride, H. Brereton Baker M.A.,  J. Chem. Soc., Trans., 1900, 77, 646, {{doi|10.1039/CT9007700646}}</ref>固态中的{{chem|Hg|2|2+}}单元最初于1926年通过[[X射线衍射|X射线衍射]]得到表征；<ref name = "Wells3d"/>而溶液中的Hg-Hg键在1934年通过[[拉曼光谱|拉曼光谱]]确认。<ref name = "Greenwood"/>

{{chem|Hg|2|2+}}在水溶液中稳定，并存在着{{chem|Hg|2+}}和元素汞的[[化学平衡|化学平衡]]，其中{{chem|Hg|2+}}含量在0.6%左右。<ref name = "Greenwood"/>在加入阴离子形成难溶的二价汞盐时，平衡会被打破，如[[硫化物|S<sup>2−</sup>]]，这时一价汞会完全地歧化。如果加入的阴离子可以形成难溶的一价汞盐时，元素汞和{{chem|Hg|2+}}会重新结合为{{chem|Hg|2|2+}}，如[[氯化亚汞|Hg<sub>2</sub>Cl<sub>2</sub>]]。<ref name = "Greenwood"/>

含{{chem|Hg|2|2+}}阳离子的矿物如氯汞矿（eglestonite）。<ref>Eglestonite, [Hg<sub>2</sub>]<sub>3</sub>Cl<sub>3</sub>O<sub>2</sub>H: Confirmation of the chemical formula by neutron powder diffraction, Mereiter K., Zemann J., Hewatt A.W. American Mineralogist, 77, (1992), 839-842</ref>

==线形三汞或四汞阳离子==

含{{chem|Hg|3|2+}}（Hg({{frac|2|3}})）和{{chem|Hg|4|2+}}（Hg({{frac|1|2}})）阳离子的化合物已经被合成出来了。这些离子仅已知在固态的化合物中存在，如{{chem|Hg|3|(AlCl|4|)|2}}和{{chem|Hg|4|(AsF|6|)|2}}。<ref name = "Greenwood"/>其中Hg–Hg键键长为255 pm（{{chem|Hg|3|2+}}）和255–262 pm（{{chem|Hg|4|2+}}）。<ref name = "Greenwood"/>这些成键包含由6s轨道形成的2中心2电子键。<ref name = "Greenwood"/>

==环状汞阳离子==

三角形{{chem|Hg|3|4+}}离子通过1989年的矿物黄氯汞矿（terlinguaite）的探查发现，<ref name ="Broderson"/>并自此合成了一系列的化合物。<ref>[Hg<sub>3</sub>]<sup>4+ </sup>Cation in Inorganic Crystal Structures, S. V. Borisov, S. A. Magarill and N. V. Pervukhina, Journal of Structural Chemistry, 44, 3, 2003, 441-447, {{doi|10.1023/B:JORY.0000009672.71752.68}}</ref>该化合物中的成键为[[三中心二电子键|三中心二电子键]]，由汞原子（D<sub>3h</sub>对称）的键合“a<sub>1</sub>轨道”与6s轨道重叠形成。<ref>Synthesis and crystal structure of the subvalent mercury cluster [triangulo-Hg<sub>3</sub>(-dmpm)4][O<sub>3</sub>SCF<sub>3</sub>]<sub>4</sub> (dmpm = Me<sub>2</sub>PCH<sub>2</sub>PMe<sub>2</sub>), Anna Mühlecker-Knoepfler, Ernst Ellmerer-Müller, Robert Konrat, Karl-Hans Ongania, Klaus Wurst and Paul Peringer, J. Chem. Soc., Dalton Trans., 1997, 1607 - 1610, {{doi|10.1039/a700483d}}</ref>

==链状和层状阳离子==
金黄色的化合物Hg<sub>2.86</sub>(AsF<sub>6</sub>)由其发现者命名为“alchemists' gold”（意思是炼金术士之金）<ref name = "Brown"/>，包含垂直的汞原子链。

具有“金属性”的化合物{{chem|Hg|3|NbF|6}}和{{chem|Hg|3|TaF|6}}含有被{{chem|MF|6|−}}阴离子分离的六边形汞原子层。<ref name = "Brown2"/>它们在7 [[热力学温度|K]]下是[[超导体|超导体]]。<ref>Superconductivity of Hg<sub>3</sub>NbF<sub>6</sub> and Hg<sub>3</sub>TaF<sub>6</sub>,(1983),W. R. Datars, K. R. Morgan and R. J. Gillespie, Phys. Rev. B 28, 5049 - 5052, {{doi|10.1103/PhysRevB.28.5049}}</ref>
