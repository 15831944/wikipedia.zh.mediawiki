'''多羅羅'''（{{lang|ja|どろろ}}）是[[日本|日本]][[漫畫家|漫畫家]][[手塚治虫|手塚治虫]]所畫的漫畫，屬於少年類漫畫的冒險故事。1967年在日本《[[週刊少年Sunday|週刊少年Sunday]]》（[[小學館|小學館]]）開始連載，但後來中斷。於1969年《{{link-ja|冒險王|冒険王}}》（[[秋田書店|秋田書店]]）再次推出，直至結局。真人[[電影|電影]]版由[[妻夫木聰|妻夫木聰]]和[[柴崎幸|柴崎幸]]主演，日本2007年1月27日上映。

== 故事簡介 ==

故事述說一位名叫百鬼丸的男孩，因為父親夢想稱霸亂世，於是與四十八隻魔神做出約定，將百鬼丸的身體分為48個部份，讓魔神拿走。所以百鬼丸為了拿回自己的身體器官便出發討伐魔神，途中遇上小偷多羅羅，展開與妖怪戰鬥的冒險。

故事中帶有強烈的諷刺意味，例如戰爭不斷，但只有平民百姓受苦；當百鬼丸替村民殺死妖怪後，反而被趕出村，表現出人情冷漠；而其中板門之卷帶有諷刺[[柏林圍牆|柏林圍牆]]及[[板門店|板門店]]。

== 登場人物 ==
* {{jp|c='''百鬼丸'''|j=百鬼丸|hg=ひゃっきまる|extra1=Hyakkimaru|help=no}}
:本作主角
:因父親'''醍醐景光'''向四十八魔神祈求能統一天下而被獻上，出世時已被四十八魔神分走[[48|48]]個身體部位，沒有四肢，也沒有眼耳口鼻等。被送到河川，放在竹籃裡送走。後被[[醫生|醫生]]壽海收養，發現他擁有強大的生命力，而且能用心眼視物，用傳念與人溝通，壽海便為其用[[木頭|木頭]]製造假手腳、假眼鼻等器官，而且訓練他[[知識|知識]]和身手，令他能長大成人。2019年動畫版則是無法視物也無法聽見聲音和說話，但能看見靈魂的顏色，並透過靈魂判斷哪些東西有害。
:2019年動畫版縮減十二魔神，出世時被雷劈中，被分走12個身體部位，分別是皮膚、神經、左耳、右耳、口舌（聲帶）、鼻子、右眼、左眼、右手、左手、右腳、左腳。
:不過，由於百鬼丸被魔神選中，吸引大量的[[鬼怪|鬼怪]]來要殺死他，所以踏上找尋身體、討伐四十八魔神旅途，壽海便為其裝上武器，雙手可以拆下來，並且裝有一位將軍賜給壽海的刀；腳拆下來後，只要收縮腰部肌肉，可發出強烈的燒水；鼻子裝有炸藥，是逼不得已才用的武器，遊戲中則改為手背火炮（機關槍）與忍必怒雷（火箭炮），2019動畫版則改為雙手各藏有一把刀（再次被鬼神奪去右腳時在右腳也裝了一把刀，將其斬殺後又再長回右腳），擁有厲害的劍術及敏捷身手，戰勝過多隻[[妖怪|妖怪]]。百鬼丸这个名字来源于[[百鬼夜行|百鬼夜行]]。

* {{jp|c='''多羅羅'''|j=どろろ|hg=どろろ|extra1= Dororo|help=no}}
:本作女主角
:父母為土匪的首領，但只殺害[[武家|武家]]人（士兵的稱呼），但他們的手下地鼠卻想投降武家，將多羅羅的父親火袋打跛。一家過著流浪生活，非常貧困。後來父親火袋被殺，而母親自夜也凍死，多羅羅只有靠偷竊為生，是反映戰爭中的小孩遭遇。

:後來，自稱天下第一神偷的小孩，在被人欺負之時得百鬼丸相救。作男孩打扮，所以難以被人發現其女兒身。因誓要把百鬼丸的手中刀偷過來，所以跟隨百鬼丸冒險。擁有敏捷身手，大叫時能令敵人嚇住，掉石的命中率也很高。在一次百鬼丸捉她洗澡時，發現多羅羅[[母親|母親]]畫在她背上的藏寶圖（所以有數年沒有洗澡）。

* {{jp|c='''醍醐景光'''|j=醍醐 景光|hg=だいご かげみつ|extra1= Daigo Kagemitsu|help=no}}
:百鬼丸、多寶丸的親生父親。[[室町時代|室町時代]]的武士，向四十八魔神許願統一天下而獻上兒子百鬼丸的[[身體|身體]]。用盡各種手段達到目的，但被百鬼丸破壞，更喪失了多寶丸的性命。後來發現，醍醐景光被數隻怪物結合體依附著，百鬼丸替他消滅結合體，但野心不改，最後被農民趕走。

* {{jp|c='''多寶丸'''|j=多宝丸|hg=たほうまる|extra1= Tahōmaru|help=no}}
:是醍醐景光的兒子，'''百鬼丸'''的弟弟。被醍醐景光視為繼承人，與父親一樣不擇手段。不知百鬼丸是自己的哥哥，兩人在一次決戰中，百鬼丸殺死了多寶丸。最後百鬼丸才知道多寶丸是自己的弟弟，並向多寶丸說出真相，多寶丸也含笑而終。

* {{jp|c='''壽海'''|j=寿海|hg=じゅかい|extra1= Jukai|help=no}}
:是一名醫生。壽海去採藥時發現在河上被遺棄的百鬼丸，因為不忍心，所以就將他帶回家，當成自己孩子一樣的養育。幫助百鬼丸做義肢及義眼、木製鼻子之類的東西，希望百鬼丸可以跟普通孩子一樣成長。不過因百鬼丸的關係，家裡吸引了許多妖怪。開始訓練百鬼丸戰鬥。最後在百鬼丸木製身體與義肢加上武器，並讓他開始為自己而冒險。

* {{jp|c='''琵琶法師/琵琶丸'''|j=琵琶法師|hg=びわほうし}}
:是劍術高超的盲目僧侶，其攜帶的琵琶有內藏刀。
:多次與百鬼丸&多羅羅碰面，有時會出手解決百鬼丸無法應付的妖怪。

== 電子遊戲 ==
《多羅羅》曾被[[SEGA|SEGA]]改編成電子遊戲，於2004年9月9日在[[PlayStation_2|PlayStation 2]]推出。

遊戲中對部份情節有所改變，如：
* 百鬼丸被描寫成天之子，魔神因為懼怕對自己不利，所以用詭計騙百鬼丸的父親，讓他把百鬼丸交出來。
* 加上漫畫版中沒有的魔神。
* 以多羅羅為48魔神之首的寄居體。更描寫多羅羅長大後，百鬼丸為她消滅魔神之首的一個完整結局。

===美術人員===
* 魔神、妖怪設計：[[前田真宏|前田真宏]]
* 角色設計：[[沙村廣明|沙村廣明]]
* 標題題字/美術設定：[[雨宮慶太|雨宮慶太]]
