'''多紀理毘賣命'''（{{lang|ja|たきりひめ の みこと}}）為《[[古事記|古事記]]》裡的記述，《[[日本書紀|日本書紀]]》記作'''田心姬'''（{{lang|ja|たこりひめ}}）、'''田霧姬'''，祂是[[日本神話|日本神話]]裡的[[天照大神|天照大神]]和[[素戔嗚尊|素戔嗚尊]]舉行誓約儀式所產生的三位[[女神|女神]]（俗稱[[宗像三女神|宗像三女神]]）之一，別名'''奧津島姬命'''。

== 概要 ==
=== 日本書紀之記載 ===
依照《日本書紀》卷第一神代上之記述，天照大神以素戔嗚尊之[[十握劍|十握劍]]打折為三段，濯於天眞名井，咀嚼吹出狹霧生出'''田心姬'''、次[[多岐都比賣命|湍津姬]]、次[[市寸島比賣命|市杵島姬]]等三[[女神|女神]]。

然後素戔嗚尊取掛在天照大神身上的[[八尺瓊勾玉|八尺瓊勾玉]]，用天真名井之水洗滌後放入口中咀嚼，吹氣陸續生出：[[天忍穗耳尊|正哉吾勝勝速日天忍穗耳尊]]、[[天菩比神|天穗日命]]、[[天津日子根命|天津彥根命]]、[[活津日子根命|活津彥根命]]、[[熊野久須毘命|熊野櫲樟日命]]等五男神。

但是接下來的其他說法裡，卻出現了不同之處：
# 第一種說法：名為'''田心姬'''，第三個出生，奉祀於邊津宮。
# 第二種說法：名為'''田心姬'''，第二個出生，居於中瀛。
# 第三種說法：名為'''田霧姬'''，第三個出生，奉祀於邊津宮。

=== 古事記之記載 ===
據《古事記》上卷之紀錄，天照大神及[[素戔嗚尊|建速須佐之男命]]至天安河而為誓約，前者取後者之[[十握劍|十拳劍]]打折成三段，振響自己配戴的[[八尺瓊勾玉|八尺瓊勾玉]]玉串，並取天之真名井之水清滌斷劍，嚙碎其劍、口吹狹霧陸續生出'''多紀理毘賣命'''（又稱名奧津島姬命）、[[市寸島比賣命|市寸島比賣命]]（又名狹依姬命）、[[多岐都比賣命|多岐都比賣命]]等三[[女神|女神]]。

接著速須佐之男命取天照大神所纏左髻之[[八尺瓊勾玉|八尺瓊勾玉]]玉串，滌勾玉於天之真名井而於口碎之，吹氣生出[[天忍穗耳尊|正勝吾勝勝速日天之忍穗耳命]]、右髻之勾玉則化作[[天菩比神|天之菩卑能命]]、脖子之勾玉生成[[天津日子根命|天津日子根命]]、左手之勾玉變成[[活津日子根命|活津日子根命]]、右手之勾玉生成[[熊野久須毘命|熊野久須毘命]]等五男神。

此外，該書記載[[大國主神|大國主神]]與多紀理毘賣命生下了[[味耜高彥根神|阿遲鉏高日子根神]]、[[下照姬|高比賣命]]。

== 解說 ==
神名裡的「多紀理（{{lang|ja|たきり}}）」可解作海上之[[霧|霧]]，亦可解作「{{lang|ja|}滾（たぎ）り}」，也就是水花激濺的樣子。《日本書紀》寫成「田心（{{lang|ja|たこり}}）」，乃是轉音導致。一般日本民間將此[[神祇|神祇]]視為[[海神|海神]]，保佑日本至[[朝鮮半島|朝鮮半島]]之間的海上[[交通|交通]]安全。
