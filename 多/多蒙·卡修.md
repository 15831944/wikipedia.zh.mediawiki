'''多蒙·卡修'''是日本[[電視動畫|電視動畫]]「[[機動武鬥傳G_GUNDAM|機動武鬥傳G GUNDAM]]」的男主角。他是代表新[[日本|日本]]的[[GUNDAM武鬥#GUNDAM鬥士|GUNDAM鬥士]]，特徵是紅色頭巾和右臉頰的十字傷疤。活躍于「[[超級機器人大戰|超級機器人大戰]]」和「SD高達G世紀」等遊戲中，是可以信賴的超強力角色。

== 角色設定 ==
新日本殖民地出身的機動戰士，右手有著紅心King紋章，紅心King代表着查理曼大帝，是殖民地格鬥技霸主的最強之證。

* 生日：7月24日
* 身高：180 cm
* 體重：78 kg
* 血型：O型
* 星座：[[狮子座|獅子座]]
* 配音声优：[[關智一|關智一]]（日語）、[[夏治世|夏治世]]（國語）、[[馮錦堂|馮錦堂]]（粵語）
* 駕駛機体：[[閃光GUNDAM|閃光鋼彈]]（GF13-017NJ）、神鋼彈（GF13-017NJII）
* 常用招數：閃光掌（Shining Finger）、爆熱神威掌（God Finger）、超級霸王電影彈、石破天驚拳

作品初期是個不擅長表達情感的笨拙男人，經過與許多人的邂逅和生死決鬥後而成長。

== 角色背景 ==
反抗優秀的父親與哥哥，離家出走，在外流浪之時，被東方不敗所救。杜門便就此成為他的弟子，習得流派東方不敗。成為武鬥家的多門不斷修行而成長，東方不敗將紅心King纹章传授給他之後，從此消失了蹤影。接著，杜門終於回到了10年不見的故鄉新日本，從新日本的軍人宇留部·石川那裡聽到了衝擊性的消息。<br />
父親雷蔵·卡修為了地球環境的再生而製作了終極高達（アルティメットガンダム），亦即惡魔高達。但是，哥哥京二·卡修卻計畫將完成的高達當作兵器，征服殖民地聯合與地球。宇留部少校率領新日本軍前往阻止，但是京治以剛完成的惡魔高達強行突破，逃出新日本殖民地並降落到地球。母親美紀乃·卡修捲入戰鬥之中而被京二所殺。父親雷蔵也被視作共犯，在共同研究開發的友人三加村博士的說情之下，免了死刑，因叛國罪嫌處以永久冷凍刑。另一方面，降落到地球的惡魔高達雖受到嚴重損傷，但預期它會再次復原。可是，地球的統治權屬於新香港，新日本不好出手。因此要利用4年一次能夠在地球上自由移動的高達武鬥期間，計畫將惡魔高達從京二手中奪回。<br />
宇留部少校提議，只要多門獲得高達武鬥的優勝並且奪回惡魔高達，作為交換，就可以拯救永久冷凍刑之中的父親。毫無拒絕餘地的多門，成為代表新日本的高達鬥士，一半是被逼，一半是為了尋找成了仇人的哥哥，參加了高達武鬥。但是，就在杜門於高達武鬥中不斷獲勝之時，圍繞著惡魔高達的一連串事件的真相漸漸明朗。杜門的命運正迎向巨大的轉變……



== 戰鬥地點 ==
; 浪跡世界～新宿篇
: 多門的目的是打倒駕駛Devil GUNDAM的哥哥恭志。為此，多門邊與世界各國的GUNDAM Fighter、新撲克牌同盟（奇波迪、賽采之、喬治、阿爾戈）等人戰鬥，邊追著恭志。抵達新宿市之時，多蒙遭遇到了衝擊性的事件。師父東方不敗成了魔鬼GUNDAM的爪牙，告別前代撲克同盟，再會新生撲克同盟，以及謎般的新德國GUNDAM Fighter舒伯茲.布魯達。多門捨不去對哥哥的懷念，沒能給予復活的Devil GUNDAM最後一擊，讓它給逃了。
; 基亞那高地篇
: 在那之後，多門與舒伯茲對戰而發現到自己的不成熟，回到曾與師父東方不敗一起生活過的修行地點基亞那高地，專心於修行。但是，Devil GUNDAM再次現身。Shining GUNDAM「憤怒」的超級模式贏不了Devil GUNDAM，由於舒伯茲的提示，多門領悟了'''「明鏡止水」'''。接著多蒙發動了「真正」的超級模式，打倒了東方不敗與Devil GUNDAM。Shining GUNDAM因累積的傷害而嚴重損壞。多門就在該地換乘至God GUNDAM，前往決賽大會地點：新香港。
; 決賽大會
: 開會式中，多門對著將在[[大嶼山|大嶼山]]舉行的淘汰賽（バトルロイヤル）中等待的東方不敗下挑戰書。並且，在以頂尖成績脫穎至此的各國GUNDAM鬥士面前，明白說出要在「全勝宣言」下勝出。多門打倒了各國GUNDAM鬥士及新生撲克同盟，最後在與舒伯茲的死亡對決（デスマッチ）中，以東方不敗傳授的'''「石破天驚拳」'''作為終結技，達成全勝。
; 最終淘汰賽
: Devil GUNDAM突然闖入淘汰賽中，而Devil GUNDAM四天王（米凱羅、查普曼、阿莉比）阻擋在前。藉著新生撲克同盟、蕾茵與負傷的舒伯茲的幫助，多門終於抵達了Devil GUNDAM面前。在舒伯茲與恭志的犧牲之下，再次打倒了Devil GUNDAM。接著，與因為Devil GUNDAM被破壞而憤怒的東方不敗，進行了最初也是最後的（正式）機動武鬥。戰鬥之中，多門感受東方不敗的悲傷與無私，最後終於超越了師父東方不敗，將其打倒。
; 與Devil GUNDAM間的了結，以及對蕾茵的思念
: 因為哥哥、舒伯茲與師父的死，並且蕾茵為此感到責任而消失，在大會中獲勝的多門，絲毫感受不到優勝的欣喜。此時，Devil GUNDAM事件的主要犯人宇留部少校要將蕾茵作為魔鬼GUNDAM的生體單元（生体ユニット），三加村博士賭上性命將此事告知多門。為了蕾茵對他的心意，也為了至今夥伴們的鼓勵，多蒙乘著風雲再起前往蕾茵的所在地。面對吸收了新日本殖民地魔鬼GUNDAM，在新生撲克同盟與各國的機動戰士的協助下，打倒了宇留部的Grand Master GUNDAM。接著，阻擋在前的最後的敵人Devil GUNDAM正是蕾茵，以被吸收的蕾茵作為對手，多門放棄了……此時阿蓮比鼓勵了多門，並教給他魔法的咒語「我喜歡妳」。多門說出了對蕾茵的感謝，並做出告白。對此回應的蕾茵便從Devil GUNDAM的枷鎖中獲得了解放。蕾茵緊抱多門，兩人發誓相愛。接著兩人以'''「石破LOVELOVE天驚拳」'''擊破Devil GUNDAM。成為「GUNDAM中的GUNDAM」（ガンダム・ザ･ガンダム），多門與蕾茵朝向地球，凱旋而歸。

== 身邊的人們 ==
=== 女主角 ===
* [[蕾茵·三加村|蕾茵·三加村]] - 多蒙的青梅竹馬，也是搭檔。擔任GUNDAM的機體整備等工作。

=== 新日本的人們 ===
* [[機動武鬥傳G_GUNDAM登場人物列表#三加村博士|三加村博士]] - 蕾茵的父親，也是閃光GUNDAM與神GUNDAM的設計者。本行似乎是醫生。
* [[機動武鬥傳G_GUNDAM登場人物列表#宇留部·石川|宇留部·石川]] - 新日本軍的少校，支援多蒙他們。
* [[機動武鬥傳G_GUNDAM登場人物列表#唐渡|唐渡委員長]] - 新日本GUNDAM武鬥的負責人。因為多蒙在第13屆GUNDAM武鬥中優勝的功績而成為新日本的首相。

=== 勁敵們 ===
* [[奇波迪·克洛凱特|奇波迪·克洛凱特]] - 新美利堅的GUNDAM鬥士，多蒙的勁敵。後來成為「黑桃Q」（クイーン・ザ・スペード）。
* [[賽采之|賽采之]] - 新中國的GUNDAM鬥士，多蒙的勁敵。後來成為「梅花A」（クラブ・エース）。
* [[喬裘·德·山特|喬裘·德·山特]] - 新法蘭西的GUNDAM鬥士，多蒙的勁敵。後來成為「方塊J」（ジャック・イン・ダイヤ）。
* [[阿爾戈·高爾斯基|阿爾戈·高爾斯基]] - 新俄羅斯的GUNDAM鬥士，多蒙的勁敵。後來成為「黑色鬼牌」（ブラック・ジョーカー）。
* [[亞蓮比·碧兒絲莉|亞蓮比·碧兒絲莉]] - 新瑞典的GUNDAM鬥士，多蒙的勁敵。對多蒙有好感。
* [[修巴魯茲·布魯達|修巴魯茲·布魯達]] - 新德意志的GUNDAM鬥士。告訴多蒙許多事情，真實身分為恭志的複製人，所以特別照顧多門。

=== 師父 ===
* [[東方不敗_(動畫角色)|東方不敗亞洲天王]] - 多蒙的拳法師父。創立流派東方不敗。是洗牌同盟的一員，堪稱活傳說的武術家，可以只靠自己肉身外加一條裹腰布就破壞一台MS。
