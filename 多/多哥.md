'''多哥共和国'''（{{lang-fr|'''République Togolaise'''}}） 通稱'''多哥'''，[[西非|西非]]国家之一，东面与[[贝南|贝南]]、北面与[[布基纳法索|布基纳法索]]、西面与[[加纳|加纳]]相邻。南面有一小段海岸线面向几内亚湾。首都及最大城市为[[洛美|洛美]]。

== 历史 ==
{{main|多哥历史}}

从15世纪起，[[葡萄牙|葡萄牙]]、[[英国|英国]]、[[法国|法国]]等殖民者侵入沿岸贩卖奴隶，称为“[[奴隶海岸|奴隶海岸]]”。1856年[[德国|德国]]势力侵入，1884年成为[[德国|德国]]保护国（殖民地），称为“多哥兰”，直到1918年。[[一战|一战]]后，东、西两部分成为英国、法国的委任统治地；[[二战|二战]]后成为托管地，[[法国|法国]]根据[[联合国|联合国]]授权对包括[[加纳|加纳]]等地区的[[法属多哥兰|法属多哥兰]]进行托管，1957年西部并入[[加纳|加纳]]，1960年4月27日东部托管到期，独立并成立多哥共和国。

1990年代确立多党制，但是军队仍未失去自1967年起几乎持续的执政地位。[[斯尔法纳斯·奥林匹欧|斯尔法纳斯·奥林匹欧]]（1901-1963）在1960年多哥独立时就成为总统。当他拒绝626名大部分参加过在[[印度支那|印度支那]]和[[阿尔及利亚|阿尔及利亚]]的战争的多哥籍法国退伍军人加入多哥军队时，被那些军人在1963年1月13日发动的政变推翻，他在第二天被杀。非军人出身的总统[[尼古拉斯·格鲁尼兹奇|尼古拉斯·格鲁尼兹奇]]（1913-1969）上台。但就在四年以后，另一次武装政变发生，格鲁尼兹奇逃离多哥并在[[科特迪瓦|科特迪瓦]]的一次车祸中身亡。其中一名发动1963年政变的退伍军人[[纳辛贝·埃亚德马|纳辛贝·埃亚德马]]（1937-2005）于1967年成为总统，直到2005年逝世。他几乎在1998年的选举中被斯尔法纳斯·奥林匹欧的儿子[[吉尔克里斯特·奥林匹欧|吉尔克里斯特·奥林匹欧]]击败。2003年，宪法中对总统连任次数的限制被取消后，埃亚德马再次被选为总统。根据埃亚德马的逝世公告，他的儿子[[福雷·纳辛贝|福雷·纳辛贝]]被军队指定为国家元首。[[非洲联盟|非洲联盟]]形容这次接管为一次武装政变。

2005年4月24日，多哥总统选举举行。[http://news.xinhuanet.com/world/2005-04/24/content_2872629.htm]4月26日，独立选举委员会宣布[[多哥人民联盟|多哥人民联盟]]候选人福雷·纳辛贝获胜。[http://news.xinhuanet.com/world/2005-04/27/content_2885930.htm]6月20日，组成由[[埃德姆·科乔|埃德姆·科乔]]为总理的政府。[http://news.xinhuanet.com/world/2005-06/21/content_3113753.htm]2006年9月20日，组成以总理[[阿博伊博|阿博伊博]]为首的政府。[http://news.xinhuanet.com/world/2006-09/21/content_5118867.htm]

== 地理 ==
{{Image|zh-hans=Duoge-ditu-zh.png|zh-hant=Duoge-ditu-zh-tw.png|thumb|220px|-{zh-hans:多哥地图; zh-hant:多哥地圖;}-}}
:{{main|多哥地理}}

多哥座落在[[西非|西非]]，南临[[贝南湾|贝南湾]]，西接[[迦纳|迦纳]]，东靠[[贝南|贝南]]，北邻[[布基纳法索|布基纳法索]]；国境南北狭长。

内陆多丘陵和高地，海拔200-500米；鲍曼峰（Baumann peak）海拔986米，为全国最高点；沿海有沼泽、泻湖和沙洲。在北方有一个轻微起伏的热带草原，中部有山区；南部有一个高原延向布满广阔的礁湖和沼泽地的滨海平原。

多哥为[[热带草原|热带草原]]和[[热带雨林气候|热带雨林气候]]，年降水量800-1800毫米。

== 人口 ==
:{{main|多哥人口}}
多哥1998年全国人口约433万，2007年全国人口达到571万，年人口增长率约为2.7%，为世界上人口增长率最高的国家之一。男性平均寿命60.4岁，女性平均寿命63.9岁。

===宗教===
約51％的人口宗教是[[泛靈論|當地原始信仰]]([[巫毒教|巫毒教]])，有29％是[[基督教|基督教]]，20％的人信奉[[伊斯蘭教|伊斯蘭教]]。另外，約有二千多百人信奉佛教團體 - SGI 創價學會，設有多哥和平會館。

== 经济 ==
:{{main|多哥经济}}
这个小型亚撒哈拉经济体严重依赖于商业及少有余粮的农业，占有全国65％的[[劳动力|劳动力]]。[[可可|可可]]、[[咖啡|咖啡]]和[[棉花|棉花]]共同创造了大约30％的出口收入。当收成正常时，多哥能对基础[[粮食|粮食]]自给自足，伴有偶然的个别地区供应困难。

在[[工业|工业]]部门中，虽然遭受了全球[[磷酸盐|磷酸盐]]价格的下滑和国际竞争的加剧，开采磷酸盐矿到目前为止仍然是最重要的活动。多哥是一个区域性商业和贸易中心。政府在世界银行和[[国际货币基金组织|国际货币基金组织]]的支持下，通过十年的努力，实施经济改革措施，鼓励外国投资，使增加的收入符合放缓的支出。但是政治局势的动荡，包括[[公营|公营]]和私营机构从1992年持续到1993年的罢工，危害了改革项目，缩小了税基，中断了必需的经济活动。发生在1994年1月12日的一次达50％的货币[[贬值|贬值]]为重新开始的结构性调整提供了一个重要的推动作用：这些努力在1994年的斗争结束，政治局势趋于明显稳定的时候得到明显加强。经济增长依靠着[[私有化|私有化]]的坚持到底，政府[[金融|金融]]业务的不断开放（去调节增加的公告服务开支），及尽可能地裁军。援助的缺乏，连同[[可可|可可]]价格的下降，带来[[GDP|GDP]]在1998年减少了1％，和1999年的再次增长。如果政治气氛没有变坏，经济增长会在2000-2001年增加到5％。

== 政治 ==
:{{main|多哥政治}}

多哥的民主进程已经中断。她的民主制度仍然是初生的和脆弱的。在2005年2月5日逝世的埃亚德马总统在他37年的执政生涯中以一党制统治多哥将近25年。根据宪法，国会议长[[范巴尔·奥瓦特拉·纳扎巴|范巴尔·奥瓦特拉·纳扎巴]]应接任为总统，等待新一届[[选举|选举]]。然而军队宣布埃亚德马总统的儿子，通信部部长[[福雷·纳辛贝|福雷·埃亚德马]]将会接任总统，宣称的理由是纳扎巴在国外。政府同时提议解除纳扎巴的议长职务，并由埃亚德马接任。福雷·埃亚德马，又称福雷·纳辛贝，在国际舆论的抨击下，在2005年2月7日宣誓就职。

2012年7月31日福雷任命[[奎西·塞莱亚戈吉·阿胡梅－祖努|奎西·塞莱亚戈吉·阿胡梅－祖努]]为多哥总理。多哥议会选举于2013年7月25日举行，根据多哥宪法第66条规定，议会选举结束后，政府将辞职。祖努8月27日向福雷提交了辞呈。福雷接受了祖努的辞呈，并要求其所有内阁成员继续处理政府的日常事务，直至新总理的任命。2013年9月6日，福雷再次任命奎西·塞莱亚戈吉·阿胡梅－祖努为政府总理。<ref>[http://news.ifeng.com/world/detail_2013_09/07/29408089_0.shtml 多哥总统再次任命祖努为新总理 要求组建新政府]</ref>

== 通信 ==
{{main|多哥通信}}
多哥电信公司是多哥政府指定的管理和协调通讯服务行业的[[国营|国营]]公司。

2008年固定电话总装机量超过7万台。可覆盖全国16个城市和60多个乡镇地区。截至2009年底，固定电话有28.6万台，投入使用的有17.8万台。全国有[[移动电话|移动电话]]用户161万户。移动通信网覆盖总人口的85%。多哥有两家移动电信运营公司，即多哥移动电信公司，用户达120万，Moov移动电信公司，用户60万。

多哥电信和CAFE信息与电信公司经营互联网业务。上网方式分别有[[ADSL|ADSL]]上网或[[无线上网|无线上网]]两种。

== 军事 ==
:{{main|多哥军事}}
多哥军队实行[[义务兵|义务兵]]、[[志愿兵|志愿兵]]两种制度。[[义务兵|义务兵]]役期两年。18-25岁的青年均可报名。

== 教育 ==
政府每年教育经费占政府支出预算25%左右。政府重视[[高等教育|高等教育]]改革<br />
多哥有两所大学——-[[洛美大学|洛美大学]]（以前称为贝宁大学）和[[卡拉大学|卡拉大学]]。<br />
教育制度分一级教育（[[小学|小学]]）、二级教育（[[初中|初中]]）、三级教育（[[高中|高中]]）和四级教育（[[大学|大学]]）。<br />

== 体育 ==
多哥[[足球|足球]]隊在2005年10月8日以3-2擊敗[[刚果民主共和国|刚果民主共和国]]後，在[[世界盃|世界盃]]非洲區外圍賽第一組中以23分力壓[[塞內加爾|塞內加爾]]，歷史性打入2006年在德國舉行的世界盃。經過在2005年12月9日舉行的抽籤儀式後，多哥被編入G組，同組對手有[[法國|法國]]、[[瑞士|瑞士]]和[[韓國|韓國]]。經過三場分組賽，多哥一分也拿不到而出局。隊中头号球星為效力[[英格蘭超級聯賽|英格蘭超級聯賽]]球隊[[托特納姆熱刺|托特納姆熱刺]]队的[[阿德巴约|阿德巴约]]。

2010年1月9日多哥國家足球隊在[[安哥拉|安哥拉]]赴[[非洲国家杯|非洲国家杯]]途中遇恐怖分子袭击导致3死8伤。事后当地[[卡宾达解放组织|卡宾达解放组织]]宣称对袭击负责。非洲杯官员在召开紧急会议以后最终宣布比赛如期举行。尽管多哥政府在事件发生不久后要求召回国家队，队员决定留下来参加比赛“以纪念在这次遇袭中的遇难者”。後來多哥隊是奉其政府的命令打道回國。<ref>[[新浪|新浪新闻中心]]，2010年1月11日，[http://news.sina.com.cn/o/2010-01-11/071116911716s.shtml 未闻哨声先闻枪声　多哥队遭恐怖袭击非洲杯！非洲悲！]</ref>
