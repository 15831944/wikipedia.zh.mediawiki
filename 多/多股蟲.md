'''多股蟲'''（[[學名|學名]]：{{lang|la|'''''Pliomera'''''}}）是生存在[[奧陶紀|奧陶紀]][[海洋|海洋]]中的一[[屬|屬]][[三葉蟲|三葉蟲]]。牠們的胸部分為12~18節，尾部則分為4-5節，且其尾部肋區（Pleura）並未密合，因此學者推測當牠們身體捲曲成防衛型態時，海水仍可順利由其尾部的縫隙流入下腹，使牠們能以防禦之姿順利呼吸。<ref name="Pliomera">{{Cite web |url = http://tomleetaiwan.pixnet.net/blog/post/32085652-%E4%B8%89%E8%91%89%E8%9F%B2%E5%8C%96%E7%9F%B3%E6%A8%99%E6%9C%AC---pliomera-fischeri-%E8%B2%BB%E6%B0%8F%E5%A4%9A%E8%82%A1%E4%B8%89%E8%91%89 |title = 三葉蟲化石標本 - Pliomera fischeri 費氏多股三葉蟲 @ Tomm Happy Days :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-24 }}</ref>其化石分布於[[阿根廷|阿根廷]]、[[中國|中國]]、[[俄羅斯|俄羅斯]]和[[挪威|挪威]]等地。<ref name="Fossilworks" />
