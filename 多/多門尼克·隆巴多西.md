'''多明尼哥·「多門尼克」·隆巴多西'''（{{lang-en|Domenico "Domenick" Lombardozzi}}，{{IPAc-en|ˌ|l|Q|m|b|Qr|ˈ|d|ou|z|i}}，{{bd|1976年|3月25日||}}）是一名[[意大利裔美國人|意大利裔美國]]演員，在紐約的[[布朗克斯|布朗克斯]]出生。隆巴多西的第一個電影角色是在他17歲時，被[[羅拔·迪尼路|羅拔·迪尼路]]揀選在《[[不一樣的童年|不一樣的童年]]》中飾演低級槍械商人Nicky Zero。他較著名的演出是在{{link-en|A&E (TV network)|A&E (TV network)|A&E}}電視劇《[[脫獄之王|脫獄之王]]》中的Ray Zancanelli，還有[[HBO|HBO]]電視劇《[[監獄風雲_(美國電視劇)|監獄風雲]]》中的Ralph Galino、《[[火線重案組|火線重案組]]》中的{{link-en|Herc|Herc|Thomas "Herc" Hauk}}、《[[色慾荷里活|色慾荷里活]]》中的Dominick，以及《[[酒私風雲|酒私風雲]]》中的[[雷夫·卡彭|雷夫·卡彭]]<ref>{{Cite web |url = http://screencrush.com/boardwalk-emipre-season-4-domenick-lombardozzi-ralph-capone/ |title = 'Boardwalk Empire' Season 4 Casts 'The Wire's Domenick Lombardozzi As... |author = Kevin Fitzpatrick |date = 2013-02-15 |publisher = ScreenCrush |language = en}}</ref><ref>{{Cite web |url = https://deadline.com/2013/02/domenick-lombardozzi-joins-boardwalk-empire-james-hiroyuki-liao-added-to-unforgettable-431365/ |title = Domenick Lombardozzi Joins ‘Boardwalk Empire’, ‘Unforgettable’ Adds Detective |author = Nellie Andreeva |date = 2013-02-15 |publisher = Deadline Hollywood |language = en}}</ref>。
