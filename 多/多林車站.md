'''多林車站'''位於[[台灣|台灣]][[嘉義縣|嘉義縣]][[阿里山鄉|阿里山鄉]]，為[[行政院農業委員會林務局|林務局]][[阿里山森林鐵路|阿里山森林鐵路]][[阿里山線|阿里山線]]之[[鐵路車站|鐵路車站]]，位於海拔1516公尺的多林站，它是沿用[[台灣日治時期|日治時代]]的名字「多囉嘕」，或簡稱「多囉」。因「多囉嘕」語音艱澀，而改為「多林」，在民國53年3月才改名多林站。

== 車站構造 == 
*車站左邊是辦公室，右邊是候車室，站房旁設有供蒸汽機車加水用的水鶴，不過現已停用。

== 利用狀況 ==
*阿里山線(主線)的停靠站，僅為一般旅客或附近住戶搭乘的簡易招呼站。
*因受[[八八風災|八八風災]]影響，曾於2008.8.8~2017.9.1期間停駛。
*2017年7月5日起，本線復駛至十字路,每週三停靠郵輪式列車。
*2017年9月1日起,正班車恢復停靠本站。

== 歷史 ==
*1912年10月1日：設立「トロエン驛」。
*時間不詳，改為「多囉嘕站」。
*1964年3月：改為「多林站」。
