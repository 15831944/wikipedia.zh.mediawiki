'''多爾{{zy|袞|-{zh-cn:gǔn;zh-tw:ㄍㄨㄣˇ;zh-hk:gwan2;zh-sg:gǔn}-}}'''（{{lang-mnc|ᡩᠣᡵᡤᠣᠨ|v=Dorgon|a=Dorgon}}{{NoteTag|[[满语|满语]]意思是“[[獾|獾]]”<ref>《[[清朝通志|清朝通志]]·氏族略·满洲八旗姓》</ref>。}}；{{bd|1612年|11月17日|1650年|12月31日|Dorgon}}），[[满洲|满洲]][[爱新觉罗|爱新觉罗]]氏，曾一度追尊為皇帝，庙号“'''成宗'''”，谥号「'''懋德修道广业定功安民立政诚敬义皇帝'''」。[[努尔哈赤|努尔哈赤]]第十四子，[[皇太极|皇太极]]之弟，母为努尔哈赤大妃[[阿巴亥|阿巴亥]]（[[乌拉那拉氏|烏拉納喇氏]]），同母兄英亲王[[阿济格|阿济格]]，同母弟豫亲王[[多铎|多铎]]。初稱[[台吉|台吉]]。1626年封[[多罗贝勒|贝勒]]，后因战功封“'''和碩睿亲王'''”。

[[順治|順治]]朝多爾袞[[攝政|攝政]]時期，[[入關戰爭|清軍入關]]，對[[清朝|清朝]]入主中原起了關鍵作用。今北京故宫东侧南池子大街东侧的[[普度寺|普度寺]]，即原為多尔衮在京城的宅邸。

== 生平 ==
=== 少年 ===
明[[万历|万历]]四十年[[农历|农历]][[十月廿五|十月二十五日]]，多尔衮出生于[[赫图阿拉城|赫图阿拉城]]（今辽宁省[[新宾满族自治县|新宾满族自治县]]永陵镇老城村）。多爾袞15岁时，[[皇太极|皇太极]]矯旨，逼迫多尔衮的生母[[阿巴亥|阿巴亥]]为努尔哈赤[[殉葬|殉葬]]，多爾袞因此丧失继承大统的可能，之後他以军功替自己爭取到權力。

=== 战功 ===
# 后金[[天聪|天聪]]二年（1628年，明[[崇祯|崇祯]]元年）二月，17岁的多尔衮和15岁的弟弟多铎随皇太极出征，征讨蒙古[[察哈尔部|察哈尔部]]，初八日，多尔衮和多铎奉皇太极之令，以偏师出击，大获全胜；杀[[古鲁台吉|古鲁台吉]]，获人畜1200余。因为军功，三月初七多尔衮被赐号“[[墨尔根戴青|墨尔根戴青]]”（“聪明机警”之意）。<ref name="degz">{{cite book |language = zh-cn |author = 滕绍箴 |title = 多尔衮之谜 |publisher = 中国社会科学出版社 |date = 2008-12 |location = 北京 |pages = 69-72 |url = |doi = |id = |ISBN = 978-7-5004-7271-1 }}</ref>
# 天聪九年（1635年，崇祯八年）二月至九月，多尔衮等率军前往收降蒙古[[林丹汗|林丹汗]]之子[[额哲|额哲]]<ref name="degz" />。多尔衮在[[西拉朱尔格|西拉朱尔格]]、[[托里图|托里图]]，巧妙招降林丹汗之妻[[囊囊太后|囊囊太后]]、[[苏泰太后|苏泰太后]]及其子额哲；并获得元朝[[传国玉玺|传国玉玺]]<ref name="degz" />。该玉玺成为皇太极称帝依据之一，加速了清朝的建立<ref name="degz" />。
# 清崇德元年（1636年，明崇祯九年）十二月，[[丙子戰爭|清军攻陷]][[李氏朝鲜|李氏朝鲜]][[南汉山城|南汉山城]]<ref name="degz" />。次年正月，皇太极令多尔衮追击朝鲜王室，并限“戢其军兵，无得杀戮”<ref name="degz" />。[[朝鮮仁祖|朝鮮仁祖]]于正月三十[[丁丑下城|投降]]。四月初五，多尔衮押送国王家属等182口报捷。<ref name="degz"/>
# 崇德三年（1638年，崇祯十一年），八月二十三日，皇太极命睿亲王多尔衮为“奉命大将军”南征明朝，这次南征，兵分两路，贝勒[[豪格|豪格]]在多尔衮亲率的左翼军中<ref name="degz" />。左翼军从[[墙子岭|墙子岭]]、[[董家口|董家口]]入关，掠山西，破[[济南|济南]]，杀明朝总兵[[卢象升|卢象升]]；然后北掠[[天津|天津]]、[[迁安|迁安]]，出[[青山关|青山关]]返还，往返扫荡数千里，于崇德四年（1639年，崇祯十二年）三月回到辽东；共攻陷城池36座，招降6座，克敌17阵，俘获人畜26万<ref name="degz" />。
# 崇德六年（1641年，崇祯十四年）至崇德七年（1642年，崇祯十五年）[[松锦之战|松锦之战]]<ref name="degz" />。双方各投入十多万大军会战，清军起先以多尔衮、[[济尔哈朗|济尔哈朗]]等为首，后皇太极亲自赶来增援<ref name="degz" />。明军经两年激战最终惨败，[[洪承畴|洪承畴]]松山城破被俘投降，[[祖大寿|祖大寿]]举锦州城投降<ref name="degz" />。松锦之战后，[[辽东|辽东]]全属大清，大明势力退入[[山海关|山海关]]。

=== 争夺皇位 ===
及至[[皇太极|皇太极]]去世时，多爾袞兄弟掌有[[正白旗|正白旗]]与[[镶白旗|镶白旗]]，共有65个[[牛彔|牛彔]]（一牛彔爲300人），占八旗全部210个牛彔的31%，足夠和皇太极长子[[豪格|豪格]]争夺皇位。豪格继承了皇太极的[[正黄旗|正黄旗]]、[[镶黄旗|镶黄旗]]（共40个牛彔）并自掌有[[正蓝旗|正蓝旗]]（21个牛彔）。多爾袞利用豪格的软弱使其不能继位，又畏于两紅旗的实力（[[代善|代善]]一人独掌[[八旗|两红旗]]，共计51个牛彔；济尔哈朗领导的镶蓝旗33个牛彔），不敢自己继位，转而扶持皇太极九子[[福临|福临]]入承大统。当时福临年僅六歲，多爾袞和郑亲王济尔哈朗共同辅政，并实际掌权。

=== 摄政 ===
多爾袞摄政时期，[[清军入关|清军入关]]，[[满清|满清]]入主[[中原|中原]]，对[[清朝|清朝]]开始近300年的统治起了關鍵作用。[[顺治帝|顺治帝]]对他的称呼从“'''叔父摄政王'''”到“'''皇叔父攝政王'''”（1645年）最後演變成“'''皇父摄政王'''”（1649年）{{NoteTag|汉语“叔父”这一称呼，译成[[满语|满语]]为“Ecike/Eqike”。多尔衮曾谕示[[范文程|范文程]]、[[宁完我|宁完我]]等大臣，说自己患[[风疾|风疾]]，不胜跪拜。到了1648年1月24日，拜伊图、锡翰等亲信上疏，谓皇叔父贵体不和，不当向[[顺治帝|皇帝]]行跪拜之礼<ref>鸳渊一：《清朝顺治初世的派阀抗争》，第6-7页。</ref>}}。多爾袞愛好打獵，出狩場面壯大，光是[[獵鷹|獵鷹]]就有一千多隻。府第“翚飞鸟革，虎踞龙蟠，不惟凌空挂斗，与帝座相同，而金碧辉煌，雕镂奇异，尤有过之者”，顺治七年七月下令加派白银二百五十万两，在喀喇河屯（今[[河北省|河北省]][[承德市|承德市]][[双滦区|双滦区]][[滦河镇|滦河镇]]）修建[[避暑山莊|避暑山莊]][[喀喇河屯行宫|喀喇河屯行宫]]（为历史上最早的避暑山庄，位置在今承德市双滦区滦河镇的滦河北岸，承钢医院一带）。種種奢華措施使年幼的順治皇帝心生不滿。

===治國之策===

# [[顺治|顺治]]元年七月初八日，多尔衮宣布取消“三饷”，即辽饷、剿饷和练饷，從而减轻了民众的负担<ref>《摄政王谕官吏军民人等令旨》山西博物馆藏：自顺治元年为始，凡正额之外一切加派，如辽饷、剿饷、练饷及召买米豆尽行蠲免。……如有官吏通同朦胧混征暗派者，查实纠参，必杀无赦！”</ref>。

# [[顺治|顺治]]元年十月二十一日，规定了第一個告發官吏贪污之人的奖勵，赃私將分作三分，一分赏给首告之人，其餘二分入库<ref>《摄政叔父王令旨》历史语言研究所藏：摄政叔父王令旨:“凡贪赃受贿官吏或傍人或所属下人将事情首告审问的实，所犯官吏处以重法。其赃私分作三分，一分赏给首告之人，二分入库。所告如系虚捏，将原告人以盗贼律论罪。</ref>。

# [[顺治|顺治]]元年八月，多尔衮谕令法司官会同廷臣详绎明律。顺治四年三月[[大学士|大学士]]刚林等人以明律为蓝本修订，满汉双语颁行《大清律集解附例》，有吏律、户律、刑律、礼律等，共三十卷，乃為[[清朝|清朝]]第一部成文法典，《大清律例》的制定亦打下基础。

=== 逝世 ===
多尔衮自稱在[[松锦之战|松锦之战]]中劳累过度，[[元气|元气]]大伤<ref>《多尔衮摄政日记》，第10页</ref>，得了三种病症：一是[[怔忡|怔忡]]之症，二是[[中風|中風]]，還患有[[咯血症|咯血症]]。入關後更是“机务日繁，疲于裁应，头昏目胀，体中时复不快”<ref>《多尔衮摄政日记》</ref>。顺治四年以后，風疾加重，跪拜不便<ref>《清世祖实录》卷三十五，第20页</ref>，隨著健康走下坡，因而身心烦燥，容易动怒<ref>《多尔衮摄政日记》第6页。</ref>，“上上下下都怕他，据说就是达官显贵往往也不能直接同他说话，要趁他外出过路时借便谒见。”<ref>《鞑靼漂流记》，《清史研究集》第一辑，第378页。</ref>順治七年（1650年）十一月十三日，多尔衮有疾，居家烦闷，率诸王、贝勒、贝子、公等，及八旗固山额真、官兵等猎于边外。<ref>《内国史院满文档案》记载：顺治七年十一月“十三日，皇父摄政王身体欠安，居家烦闷，欲出口外野游。”</ref>[[十二月初九|十二月九日]]，多爾袞因狩猎坠马，膝蓋受傷，塗以涼膏，竟不治死於[[喀喇河屯行宮|喀喇河屯行宮]]（今河北省[[滦平县|滦平县]]）。<ref>谈迁《北游录·纪闻下·摄政王》：“庚寅冬，摄政王殂于□。初，摄政王膝创，涂以凉膏，太医傅胤祖谓其非宜。”</ref>顺治帝追尊多尔衮为[[中国皇帝|皇帝]]，[[庙号|庙号]]“'''[[成宗|成宗]]'''”，全諡'''懋德修远廣業定功安民立政誠敬義皇帝'''，简称“義皇帝”。但两个月後，顺治帝因多爾袞当年独断专行且迫害其兄[[豪格|豪格]]，于順治八年[[二月|二月]]剥夺多爾袞的封号，并掘其墓，斬其頭骨，又接连处罚其黨羽[[刚林|刚林]]、[[巴哈纳|巴哈纳]]、[[冷僧机|冷僧机]]、[[谭泰|谭泰]]、[[拜尹图|拜尹图]]等。直到[[乾隆|-{乾}-隆]]時才恢复多爾袞的睿亲王封号并“追谥曰忠，补入[[玉牒|玉牒]]”，下令为他修复坟茔。

== 个人生活 ==
多尔衮身体素质并非如[[电视剧|电视剧]]和[[电影|电影]]中的粗大健壮，他是一个身材细弱的患病之人，他患有[[怔忡之症|怔忡之症]]，经常头晕目眩。

== 评价 ==
多爾袞是满人征服漢人的主要功臣之一，但一般認為他有“[[清初六大弊政|六大弊政]]”<ref>[[阎崇年|阎崇年]]：《[[清十二帝疑案|清十二帝疑案]]》</ref>，[[剃髮易服|薙发]]、[[剃髮易服|易服]]、[[圈地|圈地]]、占房（侵占房舍）、[[投充|投充]]（抢掠汉人为奴隶）、逋逃（[[逃人法|逃人法]]），延续时间最长的，是逃人法。顺治七年六月，广西巡抚[[郭肇基|郭肇基]]等人因为“擅带逃人五十三名”，被处死<ref>《清世祖实录》卷四十九</ref>。顺治十年豪雨成灾，“直隶被水诸处，万民流离，扶老携幼，就食山东。但逃人法严，不敢收留，流民啼号转徙”<ref>《清世祖实录》卷七十七</ref>。多尔衮[[满清大屠杀|血洗江南]]，也落下屠夫的恶名<ref>[http://news.sctv.com/rw/lsjm/201008/t20100805_453783_4.shtml 中国史上十五位屠城“高手”排行：忽必烈创纪录] {{webarchive|url=https://web.archive.org/web/20140315081838/http://news.sctv.com/rw/lsjm/201008/t20100805_453783_4.shtml |date=2014-03-15 }} 四川广播电视台</ref>。

多爾袞善戰不善治國且好享樂，其早死對清朝內政有正面作用。

== 与孝莊文皇后的关系 ==
《清朝野史大觀》、古稀老人的《[[多爾袞軼事|多爾袞軼事]]》、[[蔣良騏|蔣良騏]]《東華錄》、《張蒼水全集》中[[張煌言|張煌言]]的《建夷宮詞》都有“[[太后下嫁|太后下嫁]]”一說，即多爾袞按照[[清兵入关|清兵入关]]（[[建州女真|建州女真]]）前的[[夫兄弟婚|妻寡嫂]]的风俗娶了[[皇太极|皇太极]]永福宫[[孝庄文皇后|莊妃]]，即[[顺治帝|顺治帝]]生母[[孝莊文皇后|孝莊文皇后]]，同时孝莊文皇后也为了稳住多爾袞，避免其权势野心膨胀危及[[顺治帝|顺治帝]]的皇位。此一觀點在近代史學界裡雖有人正面認同，也有學者如[[阎崇年|阎崇年]]等人認為此乃當時[[漢人|漢人]]為抹黑[[滿清|滿清]][[皇室|皇室]]而造的謠言，至今她是否下嫁於多爾袞仍是個謎。

多爾袞一生則有妻妾十人。多部[[文藝|文藝]]作品如小說或電視劇，都把多爾袞[[正妻|正妻]][[義皇后|博爾濟吉特氏]]定位为孝莊文皇后之妹，但事實上两人僅是同族並非親姊妹。

=== [[妻子|妻]][[妾|妾]] ===
多爾袞一生所娶[[妻子|妻]][[妾|妾]]中有明确记录的共有十一人。另外，改嫁给[[皇太极|皇太极]]的[[林丹汗|林丹汗]][[遗孀|遗孀]][[衍庆宫淑妃|窦土门福晋]]所抚养的蒙古女子[[淑侪|淑侪]]{{NoteTag|王艳春认为[[淑侪|淑侪]]是[[林丹汗|林丹汗]]女。}}，被皇太极许给多尔衮<ref>《[[清史稿|清史稿]]·卷二百十四·列传一》：“康惠淑妃，博尔济吉特氏，阿霸垓塔布囊博第塞楚祜尔女。崇德元年，封衍庆宫淑妃。抚蒙古女，上命睿亲王多尔衮娶焉。”</ref>。两人在崇德五年（1640年）成婚。其后关于淑侪的记载不详，是否为其中某位妻妾已无法考证<ref name="b1">{{cite web |url = http://cathay.ce.cn/person/201004/21/t20100421_21307358.shtml |title =清太宗的两位蒙古养女(图) |accessdate = 2015-02-25 |author = 王艳春 |editor = 责任编辑 杜佳 |date = |publisher =中国经济网，来源：北国网－时代商报 |language = zh-cn |deadurl = yes |archiveurl = https://web.archive.org/web/20160304195017/http://cathay.ce.cn/person/201004/21/t20100421_21307358.shtml |archivedate = 2016-03-04 }}</ref>。

== 後代 ==
據[[正史|正史]]記載，多尔衮只在松锦之战前生得一女，没有男系后代。2005年，廣州一名男子曾自稱是多爾袞的第十世孫，並聲稱多爾袞被摘[[廟號|廟號]]後，家族一直不敢张扬，但滿族專家及愛新覺羅族後人均公開質疑其真確性<ref>[http://news.xinhuanet.com/newscenter/2005-03/17/content_2710210.htm 新華網 - 一男子自稱是多爾袞10世孫專家指出多處疑點]</ref>。
