'''多輻翼甲鯰'''，俗稱'''琵琶鼠'''、'''垃圾魚'''、'''清道夫'''，是[[輻鰭魚綱|輻鰭魚綱]][[鯰形目|鯰形目]][[骨甲鯰科|骨甲鯰科]]的其中一[[種|種]]。

== 分布 ==
本魚原分布於[[南美洲|南美洲]][[奧里諾科河|奧里諾科河]]流域。現已引進[[美國|美國]]、[[台灣|台灣]]、[[夏威夷|夏威夷]]等地。

== 特徵 ==
本魚魚體呈棕綠色，身上布滿清晰的豹狀斑點，這些斑點延伸至鰭上。表皮上有彩虹色，在光亮下逐漸褪色，有唇形成的吸盤狀圓盤，體長可達50公分。

== 生態 ==
本魚棲息於溪流與湖泊雜草叢生且泥質的水底，夜間活動，屬雜食性，吃[[藻類|藻類]],但是有時也會吃[[蠕蟲|蠕蟲]]，[[昆蟲|昆蟲]]幼生與其他的底棲水生的動物。

== 經濟利用 ==
為觀賞性魚類，不具食用性。
