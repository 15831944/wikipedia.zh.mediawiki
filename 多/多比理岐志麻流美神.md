'''多比理岐志麻流美神'''（{{lang|ja|たひりきしまるみ の かみ}}）為《[[古事記|古事記]]》之記載，祂是[[日本神話|日本神話]]中的[[神祇|神祇]]。

== 概要 ==
=== 古事記之記載 ===
據《古事記》上卷之記錄，[[大國主神|大國主神]]娶八島牟遲能神（{{lang|ja|やしまむち の かみ}}）之女鳥耳神（{{lang|ja|とりみみ の かみ}}），生下鳥鳴海神（{{lang|ja|とりなるみ の かみ}}）。後者娶日名照額田毘道男伊許知邇神（{{lang|ja|ひなてりぬかたひちをいこちに の かみ}}），產下國忍富神（{{lang|ja|くにおしとみ の かみ}}）。後者娶葦那陀迦神（{{lang|ja|あしなたか の かみ}}，又名八河江比賣〔やかはえひめ〕），生下速甕之多氣佐波夜遲奴美神（{{lang|ja|はやみかのたけさはやちぬみ の かみ}}）。後者娶天之甕主神（{{lang|ja|あまのみかぬし の かみ}}）之女前玉比賣（{{lang|ja|さきたまひめ}}），生下甕主日子神（{{lang|ja|みかぬしひこ の かみ}}）。後者娶[[闇淤加美神|淤迦美神]]之女比那良志毘賣（{{lang|ja|ひならしひめ}}），生下'''多比理岐志麻流美神'''。此神娶比比羅木之其花麻豆美神（{{lang|ja|ひひらきのそのはなまつみ の かみ}}）之女活玉前玉比賣神（{{lang|ja|いくたまさきたまひめ の かみ}}），生子美呂浪神（{{lang|ja|みろなみ の かみ}}）。後者娶敷山主神（{{lang|ja|しきやまぬし の かみ}}）之女青沼馬沼押比賣（{{lang|ja|あをぬうまぬおしひめ}}），產下布忍富鳥鳴海神（{{lang|ja|ぬのおしとみとりなるみ の かみ}}）。後者娶若盡女神（{{lang|ja|わかつきめ の かみ}}），生下天日腹大科度美神（{{lang|ja|あまのひはらおほしなとみ の かみ}}）。後者娶天狹霧神（{{lang|ja|あまのさきり の かみ}}）之女遠津待根神（{{lang|ja|とほつまちね の かみ}}），生下遠津山岬多良斯神（{{lang|ja|とほつやまさきたらし の かみ}}）。

== 解說 ==
神名裡的「多比理岐（{{lang|ja|たひりき}}）」為平坦之意，「志麻流美（{{lang|ja|しまるみ}}）」則可另寫成「島」、「海」，故含有平坦的島嶼及海洋之意。

== 信仰 ==
日本祭祀多比理岐志麻流美神的[[神社|神社]]主要有下列：
