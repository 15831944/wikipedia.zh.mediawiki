'''多明妮克·蒂珀'''（'''Dominique Tipper'''）是一名英國女演員、[[創作歌手|創作歌手]]和來自{{link-en|東倫敦 (倫敦地區)|East London|東倫敦}}[[灰岩樓|灰岩樓]]的舞蹈員，她以'''蒂珀小姐'''（'''Miss Tipper'''）的名字作為歌手表演。<ref>{{Cite web |url = http://www.femalefirst.co.uk/movies/Dominique+Tipper-243691.html |title = Dominique Tipper Chats About Fast Girls |author = Helen Earnshaw |date = 2012-06-18 |publisher = Female First |language = en}}</ref>她在2015年[[Syfy|Syfy]]科幻電視劇《[[太空無垠|太空無垠]]》中飾演Naomi Nagata。她是英國和[[多米尼克|多米尼克]]裔。
