'''多倫多大學士嘉堡校區'''（{{lang-en|University of Toronto Scarborough}}，又作{{lang|en|UTSC}}）是[[多倫多大學|多倫多大學]]的一座衛星校園，坐落於[[加拿大|加拿大]][[安大略省|安大略省]][[多倫多|多倫多]]的[[士嘉堡|士嘉堡]]區。此院校的前身為多倫多大學士嘉堡學院（{{lang|en|Scarborough College, University of Toronto}}），於1964年成立，當時是多倫多大學文科和理科學院的組成學院，到1972年則改組為多大的一所獨立校區<ref>{{cite web |url=http://www.utsc.utoronto.ca/~advancement/about/timeline.html |title=University of Toronto Scarborough - Timeline |publisher = University of Toronto Scarborough |accessdate=2013-02-08}}</ref>。此校於2014年約有12,000名學生，提供185種學位<ref>{{cite web |url=http://www.utsc.utoronto.ca/~advancement/about/archive/facts_figures11.pdf |title=Facts and Figures |date=2011-03-15 | publisher = University of Toronto Scarborough |accessdate=2013-02-08}}</ref>。雖然多倫多大學士嘉堡校區是一所獨立於多倫多大學的校區，但其畢業證書只是顯示多倫多大學，而非士嘉堡分校；另外，在多倫多大學士嘉堡校區所修讀的學分亦可與多倫多大學主校區及多倫多大學密西沙加校區互換<ref>{{cite web |url=http://utsc.utoronto.ca/admissions/u-t-internal |title=U of T Internal Transfer |date=2014-02-24 | publisher = University of Toronto Scarborough Admissions & Student Recruitment |accessdate=2014-02-24}}</ref>。

多大士嘉堡校區主要提供[[本科教育|本科教育]]，學系包括生物學系、物理學系(含行星部)、化學系、電腦及數學科學系、英文系、人文學系、管理學系、經濟管理學系、法文與語言學系、國際發展學系、哲學系、物理及環境科學系、心理學系及社會科學系<ref>{{cite web |url=http://webapps.utsc.utoronto.ca/ose/story.php?id=2090 |title=Two new academic departments strengthen teaching and learning at UTSC |date=2010-03-31 |accessdate=2010-07-14}}</ref>。此外，該校亦聯同[[百年理工學院|百年理工學院]]提供新聞系、新興媒體和急救工業雙聯學位<ref>{{cite web | url = http://www.utsc.utoronto.ca/~jtprogs/index.html | title = Joint Programs with Centennial College | publisher = University of Toronto Scarborough | year = 2006 | accessdate = 2007-12-15}}</ref>。

== 本科及研究生課程 ==
===本科課程===
多倫多大學士嘉堡校區現提供180 個本科課程,當中包含三種本學位:榮譽文學士(包含社會科學)，榮譽理學士(生命及健康科學、環境科學、電腦及數學科學、計量金融)和工商管理學士(經濟學、環球商業學、會計學、財務、市場策劃等等)，並於每年開辦超過1200個科目。另外，又與[[百年理工学院|百年理工学院]]合辦三項本科課程。

現時，多倫多大學士嘉堡校區的榮譽文學士包含以下課程，可供學生选讀，例如: 創意寫作、傳理、文學、心理語言學、哲學、城市規劃、政治學和公共法律等等。榮譽理學士方面就包含: 生命及健康科學、國際發展學、太空學、環境科學、物理、原子生物、疾病學、神經科學、救護、生物、軟件工程、電腦及數學科學、計量金融等等。工商管理學士則提供經濟學、環球商業學、會計學、財務、市場策劃、人力資源、策略管理、管理及資訊科技;其中，多倫多大學士嘉堡校區是整所多倫多大學中，提供環球商業學以及帶薪實習(Co-op)課程的唯一校區。

===研究生課程===
多倫多大學士嘉堡校區現分別提供兩個碩士及博士課程，包括環境科學碩士和博士、文學碩士-諮詢和臨床心理學及諮詢和臨床心理學博士。

== 其他地區學生入學辦法 ==
=== 普通入學辦法 ===
陆港澳台或其他地區學生通過投考當地的高級程度考試,例如[[香港中學文憑考試|香港中學文憑考試]],[[英國普通教育高級程度證書|英國普通教育高級程度證書]]等等，以獲取入學資格。<ref>{{cite web |url=http://www.adm.utoronto.ca/admissions/ |title=University of Toronto Admissions |date=2014-03-09 |accessdate=2014-03-09 |deadurl=yes |archiveurl=https://web.archive.org/web/20140312202857/http://www.adm.utoronto.ca/admissions/ |archivedate=2014年3月12日 |df= }}</ref>基本流程是通過{{link-en|加拿大安大略省大學聯招中心|Ontario Universities' Application Centre}}的網頁報名，並把中學校內成績及英語能力評估或其他相關文件郵寄至多倫多大學總入學部，以讓多倫多大學士嘉堡校區入學收生部評估學生的入學資格，並於三月至七月期間發出入學通知書<ref>{{cite web |url=http://utsc.utoronto.ca/admissions/how-apply |title=University of Toronto Scarborough How to Apply |date=2014-03-09 |accessdate=2014-03-09}}</ref>。

=== 中國大陆及台灣自行收生計劃===
多倫多大學士嘉堡校區提供自行收生計劃予此兩地區,讓不欲通過正常入學辦法入讀的學生提供第二選擇<ref>{{cite web |url=http://www.utsc.utoronto.ca/greenpath/?pg=future |title=Green Path Program |date=2014-05-23 |accessdate=2014-03-09}}</ref>。 中國大陆的自行收生計劃稱為绿色通道 (Green Path)<ref>{{cite web |url=http://www.utsc.utoronto.ca/greenpath/ |title=Green Path Program |date=2014-05-23 |accessdate=2014-03-09}}</ref>與本計劃簽訂協議的高中現時有二十一所，就讀該二十一所高中的學生只需通過校內手續，並可申請，再使用多倫多大學士嘉堡校區入學部網站完成申請。學生毌須報考托福或雅思，只需通過多倫多大學士嘉堡校區提供予合作高中的英語能力評估，便可申請。申請時只需提供校內高中成績及高中会考(高中畢業考試)即可，毌須報考全國高考。令本計劃在過去十年錄取人數不斷上升。為保持水準，本計劃由多倫多大學士嘉堡校區國際部直接管理。

大學之道(台灣)自行收生計劃 (Fair Taiwan)<ref>{{cite web |url=http://www.utsc.utoronto.ca/~fair/utsc/english/ |title=FAIR-Taiwan|date=2014-10-18 |accessdate=2014-03-09}}</ref>是項與绿色通道類似的計劃。

== 校園生活 ==
多倫多大學士嘉堡校區除了學生會外，還有近300個大小型的系會，協會和學會。種類繁多,例如:管理及經濟系學生會、生物系學生會、國際發展系學生會、士嘉堡校區康體協會、翻譯及就業發展協會<ref>{{cite web |url=http://www.utsc.utoronto.ca/~tscpa |title=Language Translation & Social Science Career Path Association|date=2015-02-13 |accessdate=2015-01-01}}</ref>、中國學生學者協會、投資社、社交學會等等。
