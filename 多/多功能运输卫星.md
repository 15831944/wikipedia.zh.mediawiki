'''多功能运输卫星'''（{{lang-ja|運輸多目的衛星}}，{{lang-en|'''M'''ulti-functional '''T'''ransport '''Sat'''ellite}}，缩写：'''MTSAT'''）是由日本[[国土交通省|国土交通省]]{{link-ja|航空局|航空局|航空局}}（CAB/MLIT）与[[气象厅_(日本)|日本气象厅]]（JMA）联合研发，由[[宇宙航空研究开发机构|宇宙航空研究开发机构]]发射的大型静止轨道卫星，运行在东经约140度、145度赤道上空约35800千米的静止轨道<ref>{{ja}}{{Cite web |url=http://www.jma-net.go.jp/sat/himawari/mtsat_info.html#info |title=気象庁 ｜ 運輸多目的衛星の概要 |accessdate=2018-02-21}}。</ref>。

== MTSAT-1 ==
多功能运输卫星一号（MTSAT-1）于1999年11月15日由[[H-II运载火箭|H-II运载火箭]]八号机发射，由于火箭一级故障导致发射失败，坠入海中。

多功能运输卫星新一号是为了取代发射失败的多功能运输卫星一号（MTSAT-1）而由美国{{link-en|Space Systems/Loral|Space Systems/Loral|Space Systems/Loral}}制造的。原定于2003年夏天由[[H-IIA运载火箭|H-IIA运载火箭]]六号机发射，但是由于制造上遇到的问题<ref name="delay">{{ja}}{{Cite web |url=http://www.jma.go.jp/jma/press/0304/22a/eisei2.pdf |title=運輸多目的衛星新１号機(MTSAT-1R)の打ち上げについて |accessdate=2018-02-24}}</ref>以及Space Systems/Loral母公司于2003年申请破产<ref>{{en}}{{Cite web |url=http://www.militaryaerospace.com/articles/print/volume-13/issue-4/departments/cots-watch/loral-arms-technology-exports-lead-to-20-million-government-fine.html |title=Loral arms technology exports lead to $20 million government fine |accessdate=2018-02-24}}</ref>（适用[[美国法典第11章破产法|美国法典第11章破产法]]），卫星制造延期。后改由H-IIA运载火箭七号机发射。

由于MTSAT-1R发射延期，美国的[[GOES-9|GOES-9]]卫星从2003年5月22日起负责气象观测任务直至MTSAT-1R投入使用<ref name="delay" />。

2004年3月19日，MTSAT-1R由[[安-124运输机|安-124运输机]]运抵[[种子岛宇宙中心|种子岛宇宙中心]]，但是由于之前发射[[情报收集卫星|情报收集卫星]]二号的H-IIA运载火箭六号机发射失败，发射再次延期。

MTSAT-1R于2005年2月26日18时25分由[[H-IIA运载火箭|H-IIA运载火箭]]七号机发射，并顺利分离。2005年3月8日10时42分进入预定轨道，并赋予昵称'''向日葵六号'''（{{lang-ja|ひまわり６号}}）<ref>{{ja}}{{cite press release |url=http://www.jma.go.jp/jma/press/0503/08a/nickname.pdf |format=PDF |accessdate=2018-02-21 |title=運輸多目的衛星新１号(ＭＴＳＡＴ－１Ｒ)の静止化完了及び愛称について |date=2005-03-08 |publisher=気象庁}}</ref>。

2005年6月28日，MTSAT-1R正式投入气象观测。

2006年4月17日由于姿态控制计算机异常导致姿态异常、无法通讯等问题，同日恢复正常<ref>{{ja}}{{Cite web |url=http://www.jma.go.jp/jma/press/0604/17a/20060417a.html |title=運輸多目的衛星新１号(ひまわり６号)の姿勢の異常について |accessdate=2018-02-24}}</ref>。

2006年7月6日开始航空管制相关的通信任务。由于地面设备故障，从2006年7月10日起出现通讯故障，为查明原因及对策，停止通讯业务至同年11月28日<ref>{{ja}}{{Cite web |url=http://www.mlit.go.jp/kisha/kisha06/12/121128_.html |title=運輸多目的衛星新１号による航空衛星通信サービスの再開について |accessdate=2018-02-24}}</ref>。

2009年11月11日21時26分，姿态控制装置发生异常，导致当日22时起无法进行气象观测。日本气象厅从11月12日凌晨1时起将气象观测任务切换到了备用星MTSAT-2，恢复观测。2009年11月12日2时17分，姿态回复正常，重新投入观测。

2009年11月16日由于地面的处理系统异常，早上八时三十分至十时的一个半小时中，有四次观测图像未能发布<ref>{{ja}}{{Cite web |url=http://www.jma.go.jp/jma/press/0911/16a/091116eisei.html |title=運輸多目的衛星新1号（ひまわり６号）の気象観測の地上処理システムの異常について |accessdate=2018-02-24}}</ref>。11月27日恢复之前，气象观测任务由MTSAT-2负责。

2010年7月1日，气象观测任务交由MTSAT-2负责，MTSAT-1R的气象观测功能进入待机<ref name="himawari6to7">{{ja}}{{Cite web |url=http://www.jma.go.jp/jma/press/1004/23b/Himawari6to7.html |title=運輸多目的衛星「ひまわり６号」から「ひまわり７号」への気象観測運用切り替えについて |accessdate=2018-02-24}}</ref>。2016末，结束航空管制通讯功能，结束生涯<ref name="mlit_report">{{ja}}{{Cite web |url=http://www.mlit.go.jp/common/001125073.pdf |title=将来の航空交通システムに関する推進協議会 ＰＢＮ検討ＷＧ 平成２７年度 活動報告書 |accessdate=2018-02-24}}</ref>。

多功能运输卫星新二号（MTSAT-2），昵称'''向日葵七号'''（{{lang-ja|ひまわり7号}}）于2006年2月18日15时55分由[[H-IIA运载火箭|H-IIA运载火箭]]九号机发射，2006年2月24日进入预定轨道。

2006年9月4日起在东经145度的静止轨道上开始气象观测任务<ref>{{ja}}{{Cite web |url=http://www.jma.go.jp/jma/press/0609/04a/0904MTSAT-2.html |title=ひまわり７号の気象ミッション待機運用の開始について |accessdate=2018-02-21}}</ref>。2007年7月19日开始航空管制通讯任务，与在轨运行的MTSAT-1R形成两机体制。

2010年7月1日，开始气象观测任务<ref name="himawari6to7" />。

2007年11月5日14时40分，由于姿态推进器异常，导致无法进行气象观测和航空管制任务<ref>{{ja}}{{Cite web |url=http://www.jma.go.jp/jma/press/0711/05a/1105MTSAT2.html |title=運輸多目的衛星新２号(ひまわり７号)の姿勢の異常について |accessdate=2018-02-26}}</ref>。两日后，切换到备用推进器，恢复正常运行<ref>{{ja}}{{Cite web |url=http://www.jma.go.jp/jma/press/0711/07a/1107MTSAT2.html |title=運輸多目的衛星新２号(ひまわり７号)の運用再開について |accessdate=2018-02-26}}</ref>。

2015年7月7日，由于向日葵八号（{{lang-ja|ひまわり8号}}）卫星开始气象观测任务，MTSAT-2的气象观测机能转入备用待机。2017年3月10日，向日葵八号的备份星向日葵九号进入轨道待机，因此MTSAT-2终止了气象观测任务<ref>{{ja}}{{Cite web |url=http://www.jma.go.jp/jma/press/1703/09b/20170309_himawari9_start_operation.html |title=「ひまわり９号」の待機運用の開始について |accessdate=2018-02-26}}</ref><ref>{{ja}}{{Cite web |url=http://www.data.jma.go.jp/mscweb/ja/oper/mtsat.html |title=MTSATの運用情報 |accessdate=2018-02-26}}</ref>。航空管制通讯任务将会继续到2019年末<ref name="mlit_report" />。
