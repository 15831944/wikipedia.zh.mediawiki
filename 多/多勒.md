'''多勒'''（{{lang|en|Dollar}}）在[[苏格兰|苏格兰]][[克拉克曼南郡|克拉克曼南郡]][[奥克尔山|奥克尔山]]南麓。这里19世纪是一个村，现在是镇，多勒镇中有德雯河流过。现有A91号马路过镇，多勒镇东8公里有行雷桥（{{lang|en|Rumbling Bridge}}）。多勒镇西行6公里有[[蒂利库特里|蒂利库特-{里}-]]镇（{{lang|en|Tillicoultry}}），再往西5公里有[[阿罗威|阿罗威]]镇（{{lang|en|Alva}}）。多勒镇的产业在18世纪以[[铜|铜]]矿、[[铅|铅]]矿、[[铁矿|铁矿]]为主，19世纪以[[纺织业|纺织业]]为主。

== 多勒镇特色 ==
* 奥克尔山。在多勒镇北，分东西二峰，最高的本克鲁峰(Ben Cleuch)有721米高，两山联合处架有长桥，瀑布奔腾而下成山涧。
* 坎伯古堡（{{lang|en|Castle Campbell}}）。位于多勒镇北的奥克尔山多勒幽谷，为14、15世纪阿该尔诸侯所建，16世纪时苏格兰的[[玛丽一世_(苏格兰)|玛丽女王]]曾在此住过。
[[File:RumblingPark.jpg|thumb]] 绘）]]
* 行雷桥（Rumbling Bridge）在镇东。“桥下泉水之喧若雷”——王韬
* 多勒学院。

== 名人 ==
# [[苏格兰|苏格兰]][[汉学家|汉学家]][[理雅各|理雅各]]的老家在多勒。
[[File:Dolllartown.jpg|thumb]]

# [[中国|中国]]早期改良派作家[[王韬|王韬]]曾在1870年至1872年侨居此地，协助理雅各翻译中国古典书籍。王韬著有《畅游灵囿》、《多勒游山》两篇游记，描写多勒风景。<ref>《漫游随录图记》，ISBN 7-80603-956-2</ref>
# [[英国|英国]][[浪漫主义|浪漫主义]]画家[[约瑟夫·玛罗德·威廉·透纳|约瑟夫·玛罗德·威廉·透纳]]曾来多勒描绘风景。
# [[世界语|世界语]]创建人[[威廉·奥德|威廉·奥德]]曾在多勒居住。
# [[杜瓦瓶|杜瓦瓶]]发明人[[杜瓦|杜瓦]]出自多勒学院。
