'''多维尔'''（{{lang-fr|Deauville}}）位處瀕臨大西洋[[英吉利海峽|英吉利海峽]]的法國[[諾曼底大區|諾曼底大區]]卡爾瓦多斯省，開兩小時車可以到達法國首都巴黎，是享譽[[法國|法國]]的一個靠海休閑度假小城，有各式各樣豪華精致的設施，還有非常多高雅展示活動。

==地名==
{{FR-维勒||多维勒|多城}}

== 歷史 ==
[[File:Bathing_Time_at_Deauville-1865-Eugène_Boudin.jpg|130px]]
多维尔由莫爾尼公爵於1858年用了4年的時間規劃建造的海濱[[度假|度假]]城，當時此地只不過是人口過百的小村子，特別的是'''多维尔'''為[[拿破侖|拿破侖]]三世的同母異父兄弟，建成後此地便成為[[巴黎|巴黎]]貴族與[[藝術家|藝術家]]眼中[[時尚的|時尚的]]度假好去處<ref>[http://www.chinesetoday.com/news/show/id/497146 背景資料：八國集團峰會舉辦地——多維爾]</ref> 。

==地理與人口 ==
多维尔有7200多人，其中5000人只是把它當成臨時的家，所以工作日時此地十分安靜，而到了周末與[[旅游|旅游]]時節時又熱鬧起來，面積為357[[公頃|公頃]]。

== 旅游 ==
[[File:Plage_de_Deauville_-_07.05.05.JPG|130px]]
旅游業是多维尔最重要的產業，城市內有非常多的諾曼底風格的[[別墅|別墅]]與名品店，法國城市、景觀與建築遺產保護區中擁有555棟的建築受到了保護，市內有三座賽馬場所，其中兩所為普通賽馬場，另一座為國際賽馬中心。度假與休閑的地方則有豪華的海濱酒店、[[賭場|賭場]]、沙灘甬道。有5萬觀摩者的美國電影節和2萬人朝暮的亞洲電影節每年於此舉辦。此外還有許多的節日、活動、比賽、[[賽馬|賽馬]]在這裡進行。
