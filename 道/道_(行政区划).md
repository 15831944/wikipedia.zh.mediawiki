'''道'''，是源於[[中國|中國]]的[[行政區劃|行政區劃]]单位，後成為[[儒家文化圈|儒家文化圈]]國家普遍采用的行政區單位。

==中國 ==

===起源===
「道」這種行政區劃始創於[[中國|中國]]。「道」在[[漢朝|漢朝]]開始出现，起初跟[[县|县]]同级别，专门使用于西南少数民族聚居的偏远地区的[[益州刺史部|益州刺史部]]，《[[汉书|汉书]]》地理志解释为“有蛮夷曰'''道'''”或者“县主蛮夷曰'''道'''”。

到了[[隋朝|隋]][[唐朝|唐]]时代，出兵征战经常以方位路向加以命名，为“某某道”，该方面军主将称“某某道行军大总管”。例如攻打[[高句丽|高句丽]]时的'''平壤道'''行军大总管、'''辽东道'''行军大总管、'''浿江道'''行军大总管等等。后来在内务民政上也使用此名词，在撤并郡县（或州县）之初，一级行政区较少时仅作为监察和地理单位，在后来州县增多之后逐渐成为实质性的一级行政区，由州－县二级制演变为道－州－县三级制。这种形式和演变过程，与20世纪的[[法国|法国]]相似，法国起初把80多个省份按历史地理渊源划分为10几个“大区”，后来大区逐渐演变成为[[省份|省份]]的上级单位。

===中世===

唐初，分天下为十道，为[[州|州]]、[[县|县]]之上的[[一级行政区划|一级行政区划]]，之后迭有增加，至二十三道之多（[[唐睿宗|唐睿宗]][[景云|景云]]年间）。但[[唐玄宗|唐玄宗]]皇帝的[[盛唐|盛唐]]時期到[[安史之亂|安史之亂]]以後的[[唐朝|中晚唐]]，[[节度使|节度使]]逐漸掌握地方实权后，「道」也日渐演变为对一个节度使辖区的称呼，所到[[唐朝|中晚唐]]後期時，甚至連[[觀察使|觀察使]]轄區也同節度使轄區一樣被稱為道，已和[[初唐|初唐]]、[[盛唐|盛唐]]时的意义有所不同。如中晚唐時期，[[李吉甫|李吉甫]]所編著的《[[元和郡縣圖志|元和郡县志]]》一書，已將唐朝天下分為四十七道，即是包含了節度使與觀察使的轄區在內。<br />

===近世===
{{See also|明朝行政區劃|清朝行政区划}}

[[元朝|元朝]]建立後，[[某某等处行中书省|某某等处行中书省]]成為一级行政区划，行省下設有道，道下有「[[路_(元朝行政區劃)|路]]」的行政区划单位。至[[明朝|明]][[清朝|清]]時期，「道」成为[[省|省]]之下军区的通称，如「[[福建省|福建省]][[台廈道|台廈道]]」或「[[台湾道|台湾道]]」，但明朝及清朝意义上稍有不同。也有人（例如《内政年鉴》）把清朝的道与省、府、县相提并论，称之为四级地方行政机构。 

[[中華民國_(大陸時期)|民國]][[北洋政府|北洋政府]]時期，曾经在省下设「'''道'''」为次级行政区划，并曾計畫「廢省置道」，将全国划分为九十多个道。分為繁要缺、邊要缺、繁缺、邊缺、要缺、簡缺共6類三個等級。其中「繁要缺」是駐紮在省會的首道，地方形勢緊要，政務繁雜的道；「邊要缺」是地處邊陲，形勢緊要的的道；「繁缺」是轄縣較多、財政情況良好的道；「邊缺」是邊境地區或重要行政據點；「要缺」是境內轄有重要商埠的道；「簡缺」是轄縣較少、事務較簡、財政情況不佳的道。[[京兆|京兆]]及稍後設置的[[東省特別區|東省特別區]]，因政區特殊，均未設道。又[[熱河|熱河]]、[[察哈爾|察哈爾]]、[[綏遠|綏遠]]、[[川邊|川邊]]因縣級政區較少，均只各設1道。各省一般平均在3至4道，僅[[甘肅|甘肅]]及[[黑龍江省|黑龍江省]]最多，皆設置了7道。各道管轄縣數，一般在10至30縣之間，不過也存在了如黑龍江省[[黑河道|黑河道]]轄3縣，[[陝西省|陝西省]][[關中道|關中道]]轄40縣以上的道的特殊例子。

因為道制的等級與行政經費[[預算|預算]]的多少有關，因此各省紛紛提出重評等級的要求，從民國三年（1914年）至四年（1915年）5月之間，[[中華民國內務部|內務部]]先後批准[[河南省_(中華民國)|河南省]][[河洛道|河洛道]]、[[江蘇省_(中華民國)|江蘇省]][[徐海道|徐海道]]、[[廣西省_(中華民國)|廣西省]][[蒼梧道|蒼梧道]]及[[鎮南道|鎮南道]]、[[吉林省_(中華民國)|吉林省]][[依蘭道|依蘭道]]的升等。同一時期內，[[江蘇省_(中華民國)|江蘇省]][[淮揚道|淮揚道]]、[[浙江省_(中華民國)|浙江省]][[甌海道|甌海道]]、[[黑龍江省_(中華民國)|黑龍江省]][[綏蘭道|綏蘭道]]、[[廣東省_(中華民國)|廣東省]][[潮循道|潮循道]]、[[山東省_(中華民國)|山東省]][[東臨道|東臨道]]的升等要求，因行政管轄區域不大，行政事務也不複雜，均被內務、財政兩部駁回。在1年半內，有如此多的道因經費問題而請求升等，引起了北洋政府的注意。內務部及[[中華民國財政部|財政部]]於民國四年（1915年）6月大總統[[袁世凱|袁世凱]]下令，今後若無特別的情況，不再重新評估核辦。但[[民国军阀|地方军阀]]自行废置，譬如山东军阀[[张宗昌|张宗昌]]将山东四道改为十一道<ref name="a1">{{Cite web |url=http://lib.sdsqw.cn/bin/mse.exe?seachword=&K=a&A=3&rec=27&run=13 |title=省情资料库》建置库》第二卷 民国时期行政区划 第一类 北洋政府时期(1912年1月-1928年4月) |accessdate=2018-03-25 |author=山东省地方史志编纂委员会 |date=2003 |work= |publisher=山东省情网，来源于《山东省志·建置志》|language=Zh-hans }}</ref>，

1927年，与北洋政府对峙的[[國民政府|國民政府]]宣布废除道制。随着1928年[[國民政府北伐|國民政府北伐]]胜利，[[蔣介石|蔣介石]]通令由[[省政府|省政府]]直接管辖[[县|县]]，最终彻底废除道制<ref name="a2">{{Cite web |url=http://lib.sdsqw.cn/bin/mse.exe?seachword=&K=a&A=3&rec=29&run=13 |title=省情资料库》建置库》第二卷 民国时期行政区划 第二类 南京政府时期第一辑 南京政府时期(上)(1928年4月-1937年7月) |accessdate=2018-03-25 |author=山东省地方史志编纂委员会，来源于《山东省志·建置志》 |date=2003 |work= |publisher=山东省情网|language=Zh-hans }}</ref>，但事實上，各省依然設置[[行政督察區|行政督察區]]制度，亦相當於「道」，[[蔣介石|蔣介石]]就曾經派他的長子[[蔣經國|蔣經國]]擔任[[江西省|江西省]][[贛南|贛南]][[行政督察區|行政督察區]]專員。

== 朝鮮与韩国 ==
約10世紀開始，當時的[[王氏高麗|高麗]]把全國分為八道。這些道的名稱隨着道府所在地的變化而跟着變遷。直到[[朝鮮王朝|朝鮮王朝]]末年。這八道的名稱是：
*[[咸镜道|咸镜道]]，后分为南、北两道
*[[黄海道|黄海道]]
*[[平安道|平安道]]，后分为南、北两道
*[[江原道|江原道]]，[[韓戰|韓戰]]後被[[三八線|停火线]]切分属于南北韓
*[[京畿道|京畿道]]
*[[忠清道|忠清道]]，后分为南、北两道
*[[庆尚道|庆尚道]]，后分为南、北两道
*[[全罗道|全罗道]]，后分为南、北两道

以上8个道称为'''[[朝鲜八道|朝鲜八道]]'''，之後又把這八道再劃分為[[十三道制|13道]]。这些名字沿用至今。

==越南 ==
[[越南|越南]]在唐朝时属于岭南道，设[[安南都督府|安南都督府]]，後改'''[[安南都護府|安南都護府]]'''、[[鎮南都護府|鎮南都護府]]，後又改回安南都護府。府下管辖12个州，[[交州刺史|交州刺史]]充任[[都护|都护]]，为长官。[[五代十国|五代十国]]时期，[[南汉|南汉]]设置静海节度使。

安南独立之初多沿袭唐朝、南汉旧制，[[丁朝|丁朝]]和[[前黎朝|前黎朝]]将辖区划分为十道，设将军，道下置州、府、县、社，而名称多不可考。[[越南李朝|李朝]]时置24路，陈朝置12路或15路，[[胡朝|胡朝]]改路为镇。

之後，[[黎朝|黎朝]]初期设[[东道|东]]、[[西道|西]]、[[南道|南]]、[[北道|北]]、[[海西道|海西]]等5道，后改为12承宣，分设3司，分别为[[贊治承政使司|贊治承政使司]]（简称处）、[[清刑宪察使司|清刑宪察使司]]（简称道）和[[都总兵使司|都总兵使司]]（简称镇），道、镇也可用于指代行政区划。

[[阮朝|阮朝]]改为27镇。[[明命|明命]]年间分设为30省。[[嗣德|嗣德]]年间，降[[河静省|河静]]、[[富安省|富安]]、[[广治省|广治]]3省为道。嗣德后期又升为省，并增设[[美德道|美德道]]、[[新化道|新化道]]，主官为[[管道_(越南)|管道]]，增设[[端雄道|端雄道]]、[[谅江道|谅江道]]，主官为靖边副使。[[同庆_(阮朝)|同庆]]年间，增设[[海宁道|海宁道]]。法属初期，殖民政府为镇压北圻的勤王运动，在北圻设置多处军区，越南人称作“官兵道”，兼管民政，主要有[[荻林道|荻林道]]、[[东潮道|东潮道]]、[[澫慕道|澫慕道]]、[[老街道|老街道]]、[[安沛道|安沛道]]、[[安边道|安边道]]、[[永安道|永安道]]、[[雅南道|雅南道]]等。1910年代，北圻局势稳定，官兵道固定为4处，分别是第一官兵道[[海宁道|海宁道]]、第二官兵道[[高平道|高平道]]、第三官兵道[[河杨道|河杨道]]和第四官兵道[[莱州道|莱州道]]。此外，殖民政府在今老挝设置第五官兵道（第五军区）。1920年代以后，官兵道也被称为省。

1900年代，殖民政府开始开发[[西原|西原]]，在西原陆续设置5个省份，越南人称为道，分别是[[崑嵩道|崑嵩道]]、[[嘉莱道|嘉莱道]]、[[得勒道|得勒道]]、[[林园道|林园道]]和[[同狔上道|同狔上道]]。此外，[[平顺省|平顺省]][[宁顺府|宁顺府]]单独设立[[宁顺道|宁顺道]]，法文称[[潘郎省|潘郎省]]。

1945年越南独立后，北圻的官兵道和中圻的道均被统一为省。
