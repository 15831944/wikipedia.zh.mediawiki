'''道東自動車道'''（{{jpn|j=道東自動車道|hg=どうとうじどうしゃどう|rm=Dōtō jidōshadō|f=y}}）是起於[[北海道|北海道]][[千歲市|千歲市]]{{link-ja|千歲惠庭系統交流道|千歳恵庭ジャンクション}}，分別至北海道[[釧路市|釧路市]][[阿寒交流道|阿寒交流道]]以及[[足寄町|足寄町]][[足寄交流道|足寄交流道]]的[[日本高速公路|高速公路]]（[[高速自動車國道|高速自動車國道]]）。

略稱為'''道東道'''（{{lang|ja|道東道／どうとうどう}}，DOTO EXPWY），通過十勝地方的路段被暱稱為'''十勝SKYROAD'''（{{lang|ja|十勝スカイロード}}）<ref name="hokkaido-np-1995-10-21">“＜特報 土曜フラッシュ＞「十勝スカイロード」30日開通 道東に夢もたらす初の高速50.3キロ”. [[北海道新聞|北海道新聞]] (北海道新聞社). (1995年10月21日)</ref>。[[國土開發幹線自動車道|國土開發幹線自動車道]]的路線名稱為[[北海道橫斷自動車道|北海道橫斷自動車道]]黑松內北見線與北海道橫斷自動車道網走線、黑松內釧路線和根室線部份區間。

高速公路路線編號分開編號，在千歳惠庭系統交流道 - 本別系統交流道 - 阿寒交流道間編為{{Ja Exp Route Sign|E38}}，本別系統交流道 - 足寄交流道間則編為{{Ja Exp Route Sign|E61}}<ref>[http://www.mlit.go.jp/road/sign/numbering/ 高速道路ナンバリング路線図(国土交通省)]</ref>。

== 設施 ==
* 交流道（IC）編號欄背景色<span style="color:#BFB">■</span>區間是已通車區間，設施欄背景色<span style="color:#CCC">■</span>區間是未通車區間與未啟用設施，未通車區間名稱全部是暫稱。
* (數字)為其他路線的編號，<數字>為預定編號。
* 路線名稱無特別註記者為[[市町村道|市町道]]。
* 停车区：休息區，SA：服務區，IC：交流道，JCT：系統交流道，TB：設於主線之[[公路收費站|收費站]]（料金所）。

== 歷史 ==
* 1995年10月30日 : 十勝清水IC-池田IC間通車。
* 1999年10月7日 : 千歲惠庭JCT-夕張IC間通車。
* 2003年3月15日 : 帶廣JCT通車，與帶廣廣尾自動車道連接。
* 2003年6月8日 : 池田IC-本別IC間、足寄IC通車。
* 2005年10月1日 : [[日本道路公團|日本道路公團]]民營化，成為東日本高速道路管理路線。
* 2007年10月21日 : 苫鵡（{{lang|ja|トマム}}）IC-十勝清水IC間通車。
* 2009年10月24日 : 占冠IC-苫鵡IC間通車。
* 2009年11月21日 : 本別IC-浦幌IC間通車，該區間為北海道第一個實施[[新直轄方式|新直轄方式]]管理（由[[國土交通省|國土交通省]]直接管理）的高速道路。
* 2010年2月2日 : 全線區間被指定為免收通行費[[社會實驗|社會實驗]]之高速道路。
* 2011年9月7日 : 占冠停车区投入服務。
* 2011年10月29日 : 夕張IC-占冠IC間通車。由仁停车区投入服務。
