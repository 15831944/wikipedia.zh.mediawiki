《'''變腦'''》（{{lang-en|'''''Being John Malkovich'''''}}，又譯《'''傀儡人生'''》）是一部1999年的電影。作者是[[查理·考夫曼|查理·考夫曼]]，導演是[[史碧·莊斯|史碧·莊斯]]。此片由[[尊·古錫|尊·古錫]]、[[金美倫·戴雅絲|金美倫·戴雅絲]]和[[約翰·馬克維奇|約翰·馬克維奇]]主演。[[約翰·馬克維奇|約翰·馬克維奇]]在劇中扮演虛構化的自己。

因為在導演和劇本的原創性，作品獲得廣泛好評。提名三项[[奥斯卡金像奖|奥斯卡金像奖]]，包括最佳导演、最佳原创剧本与最佳女配角。

==故事梗概==
Craig Schwartz是一個擁有悽涼婚姻的失敗男人。他得到了Lester博士的僱用，在Mertin Flemmer大廈7½層低屋頂的Lester社團。當史瓦茲擔任文書工作時，他發現了一個儲藏柜後的一扇豪華的大門。當他進入後，他發現他自己在馬克維奇的腦袋裡。在他被彈出去落到[[新澤西|新澤西]]某個收費站鄰近的溝內之前，他能在十五分鐘內看見和感覺馬克維奇做的任何事。當他向他的同事，馬馨揭露此事時，他們打算開始以「體驗馬克維奇」為商業，向體驗者每人賺取200美圓。

==创作历程==
查理·考夫曼关于《變腦》的最初来自于一个很简单的想法“一个关于男人爱上不是自己妻子的女人的故事”。之后考夫曼
逐渐加入了更多他认为有趣的元素，比如Mertin Flemmer大楼的七又二分之一层；在考夫曼最初的想法里，马克维奇是“看不见的”。1994年他创作了这个待售剧本，尽管在制作公司与制片厂被广泛传阅，但全部否决了这个项目。寄望找一个制片人，考夫曼将剧本发给[[弗朗西斯·福特·科波拉|弗朗西斯·福特·科波拉]]，科波拉将之给了他的继子琼斯。<ref>{{cite web| title = ReView by Phil Villarreal : 'Being John Malkovich' a quirky wonder | work = [[Arizona_Daily_Star|Arizona Daily Star]]| url = http://azstarnet.com/entertainment/movies/article_ee681184-48ef-558a-956f-8f431ab4dc0c.html| first = Phil|last = Villarreal| date = January 7, 2007| accessdate = November 4, 2010}}</ref> 

1996年琼斯读到了剧本并于1997年同意执导。琼斯将具备带去[[派拉蒙影业公司|派拉蒙影业公司]]，派拉蒙同意与单细胞影业公司（Single Cell Pictures）联合出品本片。单细胞影业制片人迈克尔·斯塔普与山迪·斯特恩带本项目去了许多制片厂，包括[[新线影业|新线影业]]，在新线首席执行官罗伯特·沙耶说“为什么这鬼东西不能是《成为汤姆·克鲁斯》？”之后放弃了这个项目。

本片预算1000万美元，<ref>{{cite web| title = The Write Stuff: Sea change for scribe's future| work = [[Variety_(magazine)|Variety]]| url = http://www.variety.com/article/VR1117801689.html| first = Jonathan|last = Bing| date = June 19, 2001| accessdate = November 4, 2010}}</ref>自1998年7月20日开机至整个八月。<ref name=vfilming>{{cite web| title = Bean, Place in 'Malkovich'| work = [[Variety_(magazine)|Variety]]| url = http://www.variety.com/article/VR1117479351.html| first = Benedict|last = Carver| date = August 10, 1998| accessdate = November 4, 2010}}</ref>主要拍摄地在[[洛杉矶|洛杉矶]]，<ref name=vfilming/> 包括[[南加大|南加大]]以及玛丽皇后号的露天酒吧。<ref>{{cite web| title = Campuses a favorite locale for filming| work = [[洛杉矶时报|洛杉矶时报]]| url = http://articles.latimes.com/2006/sep/12/local/me-125locations12| first = Steve|last = Padilla| date = September 12, 2006| accessdate = November 4, 2010}}</ref><ref>{{cite web| title = The Queen Mary: Location Filming| publisher = QueenMary.com| url = http://www.queenmary.com/Filming-Locations.aspx| accessdate = November 4, 2010| archiveurl = https://web.archive.org/web/20101126200422/http://queenmary.com/Filming-Locations.aspx| archivedate = 2010年11月26日| deadurl = no}}</ref>

==评价==
影片在全世界范围内受到了极佳评价，著名影评网站[[烂番茄|烂番茄]]评价高达93%。<ref>{{cite web|title = Being John Malkovich (1999) |url = http://www.rottentomatoes.com/m/being_john_malkovich/ |publisher = 烂番茄 |accessdate = September 7, 2009 }}</ref> 在《[[帝国|帝国]]》杂志评选的世界最伟大的500部电影中排第441名。马克维奇的表演被《首映》杂志最伟大的100个电影角色评为第90名。<ref>[http://www.filmsite.org/100characters.html 100 Greatest Movie Characters of All Time]. Filmsite.org. Retrieved on 2010-12-17.</ref>

==发行==
===院线===
《變腦》1999年10月22日在美国小范围的25块银幕上映，首周票房637,731美元，单块银幕票房25,495美元。<ref name=bo1>{{cite web| url = http://www.variety.com/article/VR1117757571.html?categoryid=13&cs=1| work = [[综艺_(杂志)|综艺]]| title = B.O. full 'House'| first = Dade| last = Hayes| date = October 31, 1999| accessdate = November 7, 2010}}</ref>次周放映规模扩大到150块银幕，该周票房超过190万美元。<ref>{{cite web| url = http://www.variety.com/article/VR1117757810.html?categoryid=13&cs=1| work = [[Variety_(magazine)|Variety]]| title = Auds 'Bone' up| first = Dade| last = Hayes| date = November 7, 1999| accessdate = November 7, 2010}}</ref> 第三周银幕增加到468块。在26周下片后，美国总票房2286万美元。<ref>{{cite web| url = http://www.boxofficemojo.com/movies/?id=beingjohnmalkovich.htm| publisher = Box Office Mojo| title = Being John Malkovich (Summary)| accessdate = November 7, 2010}}</ref>

2000年3月在英国放映，首周票房为英镑29,6282元，<ref>{{cite news| title = Film: Box Office| work = [[The_Independent|The Independent]]| date = March 26, 2000| author = Thompson, Jonathan; Redvers, Louise}}</ref>上映十五周之后总票房为超过100万英镑。<ref>{{cite news| title = Film: Box Office| work = [[The_Independent|The Independent]]| date = April 9, 2000| author = Thompson, Jonathan; Redvers, Louise}}</ref>
