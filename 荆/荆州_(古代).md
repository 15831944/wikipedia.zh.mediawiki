'''荊州'''，是中國古代的ㄧ個行政區劃，最早由漢武帝依據九州分區而設置。

== 兩漢時期 ==
{{see also|荊州刺史部}}
[[西漢|西漢]][[漢武帝|武帝]][[元封_(汉武帝)|元封]]五年（公元前106年），全國除中央之外的地域分為13個州[[刺史|刺史]]部，其中荊州的轄域相當廣，大體來說以北為現今[[河南省|河南省]][[南陽市|南陽市]]一帶往南延伸至兩湖（[[湖北|湖北]]、[[湖南|湖南]]）全境內。但是此时的州刺史部只是監察區，無治所，非真正意義上的行政區。

[[東漢|東漢]][[漢靈帝|漢靈帝]][[中平|中平]]五年（188年）后，州成為正式的一級行政區域，依據[[後漢書|後漢書]]記載，荊州轄有[[南阳郡|南陽]]（河南西南）、[[南郡|南郡]]（[[湖北|湖北]]西部）、[[江夏郡|江夏]]（湖北東部）、[[长沙郡|長沙]]（湖南東北）、[[桂阳郡|桂陽]]（[[湖南|湖南]]東南）、[[武陵郡|武陵]]（湖南西北）、[[零陵郡|零陵]]（湖南西南）七郡。州治所一直到[[東漢|東漢]][[漢獻帝|漢獻帝]]時[[劉表|劉表]]領荊州牧以前都在[[漢壽县|漢壽县]]（[[武陵郡|武陵郡]]轄，今[[常德市|常德市]][[汉寿县|汉寿县]]），後因荊南地區[[黃巾賊|黃巾賊]]餘孽尚未退去才將治所移到當時已經相當繁榮的南郡[[襄陽|襄陽]]城（今湖北[[襄陽市|襄陽市]]）。

== 三国兩晉時期 ==
208年[[赤壁之战|赤壁之战]]后，原先短暂领有[[荆州|荆州]]的[[曹操|曹操]]只保住了荆州北部[[南阳郡|南阳郡]]、[[江夏郡|江夏郡]]与[[南郡|南郡]]的一部分，而中南部被孙刘联軍佔据。曹操之后从南郡、南阳郡中分出襄阳郡、[[南乡郡|南乡郡]]，荆州成为三分割据的局面：南郡、零陵、武陵归[[刘备|刘备]]，江夏、桂阳、长沙归[[孙权|孙权]]，南阳、襄阳、南乡归[[曹操|曹操]]<ref>《[[晋书|晋书]]·地理志下》：“于是南郡、零陵、武陵以西为蜀，江夏、桂阳、长沙三郡为吴，南阳、襄阳、南乡三郡为魏。”</ref>。一般认为这阶段三家各占三郡的局面是“荆襄九郡”一词的来源。

219年，荆州牧刘备的守将[[關羽|關羽]]被[[孙吴|东吴]]、[[曹魏|曹魏]]联军击败，在随后的[[夷陵之戰|夷陵之戰]]中劉備又敗於孫權勢力後，[[蜀汉|蜀汉]]再無力奪回原有荊州轄權，荊州成为由曹魏與[[孫吳|孫吳]]兩家分領的局面。

[[三國|三國]]的曹魏荊州，治所在南阳郡[[新野|新野]]，下轄南陽郡、江夏郡（長江以北）、襄陽郡、[[南鄉郡|南鄉郡]]、[[新城郡|新城郡]]、[[上庸郡|上庸郡]]、[[魏興郡|魏興郡]]等共七郡。三國的孫吳荊州，治所在南郡的[[江陵|江陵]]，下轄南郡、江夏郡（長江以南）、長沙郡、[[湘東郡|湘東郡]]、桂陽郡、[[臨賀郡|臨賀郡]]、零陵郡、[[衡陽郡|衡陽郡]]、武陵郡、[[建平郡|建平郡]]、[[宜都郡|宜都郡]]等共十一郡。

[[西晉|西晉]]時期，荊州治所在江陵，下轄23個郡國，[[北魏|北魏]]太和二十一年（497年）于[[穰县|穰县]]（今河南[[邓州市|邓州市]]）置荆州，辖8郡。

== [[南北朝|南北朝]] ==
南北朝時期，州的數量增多，地域縮小，[[劉宋|劉宋]]設有荊州，治所在襄陽（今湖北[[襄陽|襄陽]]），[[蕭齊|蕭齊]]改治所為[[南郡|南郡]]，另設[[雍州|雍州]]，治所在[[襄陽|襄陽]][[寧蠻府|寧蠻府]]（湖北襄陽）。而[[北魏|北魏]]設荊州和[[東荊州|東荊州]]，治所分別在[[山北|山北]]（河南[[魯山|魯山]]）和[[泌陽|泌陽]][[東陵鎮|東陵鎮]]（河南泌陽）。

== [[隋朝|隋朝]] ==
到了[[隋文帝|隋文帝]][[開皇|開皇]]三年（583年），[[楊堅|楊堅]]廢除郡一級行政區劃，改為州縣兩級體制，而[[隋煬帝|隋煬帝]]時，又改州為郡，恢復[[秦朝|秦朝]]的[[郡县制|郡縣兩級體制]]，荊州的名稱只存在地域為現今[[荊州市|荊州市]]附近。一度称为[[南郡|南郡]]、[[江陵郡|江陵郡]]，后来为[[江陵府|江陵府]]、[[荆州府|荆州府]]。
