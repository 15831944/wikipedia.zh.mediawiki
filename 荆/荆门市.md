'''荆门市'''是[[中华人民共和国|中华人民共和国]][[湖北省|湖北省]]下辖的[[地级市|地级市]]，位于[[江汉平原|江汉平原]]以北。东眺[[武汉|武汉]]，西临[[三峡|三峡]]，南望[[潇湘|潇湘]]，北通[[川|川]][[陕|陕]]，素有“荆楚门户”之称。

== 历史 ==
[[商周|商周]]起在此设州置县。[[西周|西周]]分属[[权国|权国]]、[[冉国|冉国]]，[[春秋战国|春秋战国]]归属[[楚|楚]]，[[汉朝|汉朝]]置[[当阳县|当阳县]]，[[唐|唐]][[贞元_(唐朝)|贞元]]二十一年（公元805年），设立荆门县。[[宋朝|宋朝]]建荆门军，[[明朝|明朝]]复荆门县，[[清朝|清朝]]为荆门直隶州，[[民国|民国]]降州为县，新中国成立后续为荆门县。1979年分设荆门县、荆门市，1983年荆门县并入荆门市，直属湖北省人民政府。1996年，划原隶属[[荆州市|荆州市]]的京山县、钟祥市为荆门市直管和代管。

== 地理 ==
接邻市县：[[荆州市|荆州市]]、[[襄阳市|襄阳市]]、[[当阳市|当阳市]]、[[南漳县|南漳县]]、[[宜城市|宜城市]]、[[应城市|应城市]]、[[天门市|天门市]]、[[潜江市|潜江市]]。

坐标范围：东经112°11'~112°14'，北纬31°01'~31°04'。地处荆州以北100公里，为荆楚之门户，所以称之为荆门。

荆门地域东、西、北三面高，中、南部低，呈向南敞开形，形成低山坳谷、丘岗冲沟和平原湖区兼具地势。区域内地势平坦，土壤肥沃，湖泊密布，河网交织，堤垸纵横，是全市粮棉集中产区。

== 经济 ==
以前主要以农业为主，[[改革开放|改革开放]]后重工业、轻工业和军工得到了很大的发展，使荆门市人均[[GDP|GDP]]达到全国中型城市前列。
比较大型的企业有[[中国石化|中国石化]]荆门分公司、荆门热电厂、洋丰集团、中天集团、金龙泉啤酒集团等。
其中，金龙泉啤酒为中国啤酒业十强。
航空航天部605研究所负责研制水上飞行器及飞艇,宏图飞机制造厂已破产,荆襄磷化公司已破产。
中天街是全市的商业中心，商贸发达。
2011年，荆门实现地区生产总值942.59亿，位居全省第一，经济发展处于湖北的中游水平。
