'''傅元國'''[[教授|教授]]（{{lang|en|Prof. Philip Yuen-Ko FU}}；{{bd|1933年||2015年|5月4日|catIdx=F}}），[[山東省|山東省]][[威海市|威海市]]人，香港教育工作者。曾任[[香港中文大學|香港中文大學]][[崇基學院|崇基學院]]院長（1981-1988）及[[香港中文大學|香港中文大學]]會計學院院長（1990-1994）。

== 生平 ==
1933年，傅元國在[[中國|中國]][[山東省|山東省]][[威海市|威海市]]出生。

1957年，傅元國於香港[[崇基學院|崇基學院]]經濟系畢業。之後前往美國[[楊百翰大學|楊百翰大學]]學習[[會計學|會計學]]，於1962年獲得美國[[楊百翰大學|楊百翰大學]]會計學碩士學位。1968年，獲得美國[[伊利諾大學|伊利諾大學]]會計學哲學博士學位。博士畢業後，傅氏返回香港，在崇基學院工商管理學系任教27年，至1995年榮休<ref name="cc65">{{cite book |url=https://www.ccc.cuhk.edu.hk/images/ChungChiPublication/ChungChi_65Book.pdf |title=善彌雅集-崇基學院六十五週年校慶特刊 |page=22-23 |publisher=崇基學院}}</ref><ref>{{cite news |title=會計專業的開路先鋒 |author=李漢祥 |date=2015-10-17 |publisher=信報 |url=https://www1.hkej.com/dailynews/articlePrint/id/1163660 }}</ref><ref>{{cite news |title=頂尖大學 |author=左丁山 |date=2015-10-23 |publisher=蘋果日報 |url=https://hk.lifestyle.appledaily.com/lifestyle/columnist/daily/article/20151023/19343334 }}</ref><ref name="obituary">{{cite web |url=http://www.alumni.cuhk.edu.hk/aausmc/ |title=訃聞 |author=崇基學院 }}</ref><ref name="fu151018">{{cite web |url=http://alumni.bschool.cuhk.edu.hk/zh-hk/%E6%9C%80%E6%96%B0%E6%B6%88%E6%81%AF.aspx?udt_496_param_detail=15230 |title=傅元國教授生命禮讚追思會}}</ref>。

在崇基學院任教期間，曾兼任多項行政要職，包括：
*崇基學院工商管理學系系主任（1970-1972, 1973-1974）
*崇基學院會計及財務學部主任（1974-1977）
*崇基學院工商管理學院院長（1976-1977）
*[[香港中文大學|香港中文大學]]會計及財務學系系主任（1980-1984）
*[[香港中文大學|香港中文大學]][[崇基學院|崇基學院]]院長（1981-1988）
*[[香港中文大學|香港中文大學]]會計學院院長（1990-1994）

1977年，傅氏任崇基宿舍委員會主席期間，捐款創立「[[傅元國盃|傅元國盃]]」，鼓勵宿生參加運動比賽，活動迄今仍每年舉行，成為崇基傳統之一。傅氏於1982年創立「黃林秀蓮訪問學人計劃」，每年邀請國際知名學人到訪崇基學院演講，並一直舉辦至今。傅氏亦致力增加學院獎學金，例如「胡格非紀念獎學金」。<ref name="cc65" /><ref name="obituary" />

傅氏退休後定居美國，於2015年5月4日在美國[[伊利諾州|伊利諾州]][[內珀維爾市|內珀維爾市]]辭世，享年82歲。10月18日，[[香港中文大學|香港中文大學]]崇基學院禮拜堂舉行「傅元國教授生命禮讚追思會」以作追思。<ref name="fu151018" />

== 個人生活 ==
傅元國的夫人謝斐儀（Gail S. Fu），於六十年代中曾任崇基學院的 Wellesley-Yenching Fellow，1968年起於崇基及中大英文系任教多年，直至九十年代末退休。
