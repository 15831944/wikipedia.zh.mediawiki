'''傅铁山'''（{{bd|1931年|11月3日|2007年|4月20日|catIdx=F傅}}）[[主教|主教]]，圣名'''[[弥额尔|弥额尔]]'''，生於中国[[河北省|河北省]][[清苑县|清苑县]]谢庄，生前是[[中国天主教|中国天主教]][[三自爱国教会|三自爱国教会]][[自选自圣|自选自圣]]的[[天主教北京教区|天主教北京教区]][[正权主教|正权主教]]，曾任[[中国天主教爱国会|中国天主教爱国会]]主席、[[第十届全国人民代表大会常务委员会|第十届全国人大常委会]][[全国人民代表大会常务委员会副委员长|副委员长]]。

== 早年 ==
1939年家乡洪水，全家逃荒到[[密云县|密云县]][[檀营天主教堂|檀营天主教堂]]。次年进入[[白檀书院|白檀书院]]读书，1941年被保送到[[北平天主教备修院|北平天主教备修院]]就读。1944年春天，备修院因学生罢课而遣散，同年秋天，又进入北平[[西什库天主堂|西什库小修道院]]。[[二战|二战]]结束后，[[田耕莘|田耕莘]]枢机将这所小修院改为“[[耕莘中学|耕莘中学]]”。1950年傅铁山由耕莘中学高中毕业，又被保送[[文声学院|文声学院]]哲学系。1956年，傅铁山从文声学院神学系毕业，7月1日，晋升[[神父|神父]]。1958年，傅铁山调到[[北京神学院|北京神学院]]担任[[拉丁语|拉丁语]]教师。

== 教难时期 ==
1959年至1962年，下放[[南口农场|南口农场]]劳动。1966年至1973年，傅铁山在[[西北旺地区|西北旺]]劳动。<ref>地之盐  世之光――访中国天主教爱国会主席、北京教区主教傅铁山（转载《中国宗教》1999年第2期）</ref>

==擔任北京主教==

[[File:Coat_of_arms_of_Michael_Fu_Tie_Shan.svg|left]]

1979年，[[中华人民共和国宗教|中华人民共和国宗教]]政策松动，[[宣武门天主堂|宣武门天主堂]]在全国率先开放，傅铁山被推选为[[北京教区|北京教区]]主教（[[自选自圣|自选自圣]]），这时主教职务已空缺15年。12月21日（圣多默宗徒瞻礼）接受祝圣，主礼者是湖南[[天主教常德教区|常德教区]]主教[[杨高坚|杨高坚]]，辅礼者是[[天主教上海教区|上海教区]]主教[[张家树|张家树]]和内蒙古[[天主教呼和浩特教区|呼和浩特教区]]主教[[王学明|王学明]]。也就是从1979年起，傅铁山作为爱国宗教领袖出任或兼任了很多官方、半官方的领导职务。然而這個主教任命，並未受到教廷的同意。

1979年至1992年，出任天主教北京教区[[主教|主教]]，[[全國政協|全國政協]]委員、常委，[[北京市|北京市]]政协常务委员，[[中国国际交流协会|中国国际交流协会]]副会长，[[中国残疾人福利基金会|中国残疾人福利基金会]]理事，中国天主教爱国会副主席。1992年至1998年，出任[[全國人大常委會|全國人大常委會]]委員、教育科學文化衛生委員會委員，[[中国天主教爱国会|中国天主教爱国会]]副主席，中国天主教主教团副主席兼秘书长，北京市天主教爱国会主席、教务委员会主任、北京教区主教，中华海外联谊会常务理事，中国国际交流协会副会长，北京市人民对外友好协会副会长。

==任愛國會主席==
自1998年起，傅铁山出任[[全国人大常委会委员|全国人大常委会委员]]、教育科學文化衛生委員會委員，[[中国天主教爱国会|中国天主教爱国会]]主席，[[中国天主教主教团|中国天主教主教团]]副主席，北京市天主教爱国会主席、教务委员会主席、北京教区主教，[[中国国际交流协会|中国国际交流协会]]副会长，北京市政协副主席。2003年3月，在[[第十届全国人民代表大会第一次会议|第十届全国人民代表大会第一次会议]]上，当选第十届[[全国人大常委会副委员长|全国人大常委会副委员长]]。 

与此同时，傅铁山也多次对[[梵蒂冈|梵蒂冈]]进行了批评、谴责。

== 去世 ==
2007年4月20日，傅铁山因[[癌症|癌症]]医治无效，于当晚在[[卫生部北京医院|卫生部北京医院]]去世，终年76岁。[[新华社|新华社]]发布的讣闻和《傅铁山副委员长生平》中对傅铁山的中共官方评价是：“'''杰出的爱国宗教领袖，著名的社会活动家，中国共产党的亲密朋友'''”。

== 爭議 ==
* 1989年[[六四事件|六四事件]]仅几天后，他是唯一在电视上支持[[邓小平|邓小平]]对街头示威抵抗者动武、强制清场的宗教人士。
* 2000年，他在[[美国|美国]][[纽约|纽约]]出席世界宗教领导人千年峰会时，严正谴责[[丹增嘉措|达赖喇嘛]]、指责一些国家“以人权为借口颠覆其他国家的“主权”。
* 2000年谴责[[罗马教廷|罗马教廷]]封列一百二十位中国和外籍殉道者为[[中华殉道圣人|圣人]]，指外籍殉道圣人是“殖民主义的工具”，指时任教宗[[若望·保禄二世|若望·保禄二世]]在梵蒂冈[[圣伯多禄广场|圣伯多禄广场]]上主持的仪式“是无法容忍的”。<ref>[http://www.asianews.it/index.php?l=en&art=9060 Fu Tieshan, "tragic" figure of the Chinese Patriotic Church, dies AsiaNews]</ref>
