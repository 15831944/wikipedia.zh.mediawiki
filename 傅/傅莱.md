'''傅莱'''({{lang|de|Richard Frey}}，理查德·傅莱) ，生于1920年2月11日，原名理查德·施泰因（{{lang|de|Richard Stein}}），原籍[[奥地利|奥地利]][[维也纳|维也纳]]，原[[奥地利共产党|奥地利共产党]]党员，1939年1月为支援中国人民[[抗日战争|抗日战争]]来到中国，后加入[[中華人民共和国国籍|中華人民共和国国籍]]，在华工作生活65年，于2004年11月16日在北京逝世。傅莱是中国首次成功研制[[青霉素|青霉素]]第一人、是中国中西医结合治疗先行者、是中国医学信息现代化管理开创者。

== 生平 ==
傅莱出生于维也纳一个犹太中产家庭，独子，曾是维也纳第九区文理中学一名优等生，并先后在维也纳赫尔兹科勒西特放射专科院和维也纳医大皇家附属医院等医疗机构，学习放射治疗专科技术及其它专科医学。1934年傅莱加入奥地利[[童军|童军]]组织，并参加[[奥地利内战|奥地利内战]]；战后他先后加入奥地利共青团和奥地利共产党。1938年德国法西斯政权吞并奥地利后，傅莱上了[[纳粹|纳粹]]黑名单；为逃避[[盖世太保|盖世太保]]追捕、二十四小时内他撤离出维也纳。支援中国人民反抗日本法西斯的战争，是他离开欧洲后唯一的志愿。1938年底，傅莱漂洋过海前往中国。

傅莱在上海虹口难民传染病隔离医院工作两个月后，继续前往中国北方寻找抗日队伍。他先后在天津德美医院、北平道济医院、顺德府（邢台）福音医院和天津[[马大夫纪念医院|马大夫纪念医院]]行医，并参加了天津抗日工作。

1941年，傅莱接受[[八路军|八路军]]晋察冀军区司令员[[聂荣臻|聂荣臻]]邀请，从天津冒着生命危险，秘密穿越日军封锁线，来到抗日前线[[晋察冀边区|晋察冀边区]]工作。在抗日根据地，他参加过多次残酷艰巨的反扫荡战斗，抢救过无数抗日将士的生命，并培养出千余名八路军军医和卫生员。1943年晋察冀边区流行[[疟疾|疟疾]]，由于日军封锁，边区缺少治疗药品[[奎宁|奎宁]]；傅莱向当地老中医学习，在八路军和边区采用[[针灸|针灸]]治疗疟疾，取得成果，最终战胜了疾病，避免了许多军民的伤亡，获[[毛泽东|毛泽东]]、[[朱德|朱德]]褒奖，通报全军。1944年傅莱经聂荣臻介绍、[[中共中央组织部|中共中央组织部]]部长[[彭真|彭真]]批准，加入了中国的共产党，并旁听了之后在延安召开的[[中国共产党第七次全国代表大会|中国共产党第七次全国代表大会]]。1945年傅莱利用美国援华委员会捐赠的菌种，在延安艰苦环境和简陋条件下，经过50多次试验，首次在中国成功研制生产出初制青霉素，并应用于战场，医治了许多负伤将士。抗战胜利后，傅莱任全国解放区救济总会晋察冀医药总代表，积极配合[[宋庆龄|宋庆龄]]领导的[[中国福利会|中国福利会]]，将世界多国及[[联合国|联合国]]组织的援华物资，日以继夜分发给边区军民。[[第二次国共内战|第二次国共内战]]期间，傅莱任[[华北野战军|华北野战军]]卫生部顾问，参加了[[大同集宁战役|大同集宁战役]]、[[太原战役|太原战役]]、[[平津战役|平津战役]]。1949年底他随南下大军到达重庆。

[[中华人民共和国|中华人民共和国]]成立后，傅莱选择继续留在饱经战争创伤和贫穷落后的新中国，并加入了中華人民共和国国籍。解放初期，他一直在中国落后的西南边远地区工作，经常深入到农村和少数民族地区研究传染性疾病和地方病，并为中国预防疾病工作编写了许多重要医学文献。1962年，傅莱担任[[中国医学科学院|中国医学科学院]]顾问工作，他领导建立了具有世界先进水平的中国第一个大型医学文献计算机检索系统 MEDLARS中心，开创了中国医疗机构快速高质量检索全世界生物医学文献和医学信息的服务，并同时在全国各地培养出大批医学信息现代管理技术人才。

傅莱在他中国的长期政治生涯里，曾多次遭到各种无端的猜测怀疑，多年失去了应有的信任和任用。[[文化大革命|文化大革命]]结束后、他于1983年当选为[[全国政协|全国政协]]委员，并在20年联任期间，经常到全国各地进行调查研究和到世界多个国家洽谈合作交流，为中国医疗卫生事业贡献出了自己最后的力量。2004年11月16日凌晨，傅莱在[[北京协和医院|北京协和医院]]逝世。按傅莱遗愿，死后将他遗体捐献给了中国医学科学院。中国国家主席[[胡锦涛|胡锦涛]]和奥地利总统[[海因茨·菲舍尔|海因茨·菲舍尔]]两国首脑为他献了花圈。

== 历任职务 ==
傅莱曾先后担任八路军晋察冀军区医药指导委员会委员、[[陕甘宁边区|陕甘宁边区]]中西药研究会顾问、八路军联防卫生部医药卫生委员会委员、中国解放区救济总会晋察冀总代表、华北野战军卫生部顾问、华北医科大学顾问、[[西南军政委员会|西南军政委员会]]卫生部公共卫生处负责人，重庆市卫生局顾问，[[重庆医科大学|重庆医科大学]]顾问，中国医学科学院顾问，中国医学科学院情报研究所副所长，科学院图书馆馆长，中国医学科学院信息研究所名誉所长，科学院图书馆名誉馆长, [[中国人民政治协商会议全国委员会外事委员会|中国人民政治协商会议全国委员会外事委员会]]六、七、八、九届委员等职务。傅莱先后任教于晋察冀军区白求恩卫生学校、延安中国医科大学、[[白求恩医科大学|白求恩医科大学]]、华北医科大学、[[重庆医科大学|重庆医科大学]]、[[北京协和医学院|北京协和医学院]]；他著有《人民保健组织学》等。

== 婚姻家庭 ==
傅莱从天津奔赴晋察冀抗日前线后，与家乡女友—汉娜（Hanna，中学同学、反法西斯战士）失去联系。抗战胜利前夕，傅莱在延安与'''[[李滨珠|李滨珠]]'''结婚。李滨珠于1921年11月在河南[[偃师市|偃师市]]出生，1936年在[[开封|开封]]加入中华民族解放先锋队，1938年2月加入中国共产党，是年8月赴延安参加抗日。1959年傅莱李滨珠夫妇因受害于党内极左路线而被迫结束婚姻。之后傅莱再婚，李滨珠抚养子女未曾再婚。2007年7月李滨珠在傅莱家乡维也纳国家老人院逝世，葬于[[维也纳中央公墓|维也纳中央公墓]]。同日、傅莱骨灰也葬于河北[[晋察冀烈士陵园|晋察冀烈士陵园]]；这两个来自不同国度的战友加伴侣，竟无约地同时埋葬在对方之故乡。

== 永远的纪念 ==
2006年，[[中国人民政治协商会议全国委员会|中国人民政治协商会议全国委员会]]在奥地利维也纳市第九区傅莱就读的文理中学——{{link-de|Döblinger中学|Döblinger Gymnasium}}竖立了傅莱纪念牌，其碑文由奥地利国家总统撰写。
2007年，傅莱纪念墓碑在河北省唐县晋察冀烈士陵园落成。
2015年，纪念世界人民反法西斯战争胜利70周年之际，奥地利总统在国家《记忆》文集中撰文纪念傅莱。
