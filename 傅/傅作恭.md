'''傅作恭'''（{{bd|1920年|5月3日|1960年|}}），生于[[山西省|山西省]][[榮河縣|荣河县]]安昌村，中国著名政治军事人物[[傅作义|傅作义]]的堂弟。

== 早年经历 ==
1920年5月3日（[[农历|农历]]三月十五日），傅作恭生于山西省荣河县安昌村。他祖父傅文鼎育有两子，傅庆泰和傅庆雨，两兄弟一共有十个儿子，依次按仁、义、礼、智、信、温、良、恭、谦、让为他们取名，其中傅作仁、傅作义和傅作良为傅庆泰之子，包括傅作恭在内的其余七个为傅庆雨之子<ref name="kangyi">{{cite web |author1=刘巍 |title=长篇纪实小说《傅作义》作者崔正来再次发声：严正驳斥有关傅作义与八弟傅作恭的网络谣言 |url=http://epaper.sxrb.com/shtml/sxwb/20170323/538841.shtml |website=山西晚报 |accessdate=2018-07-21 |archiveurl=https://web.archive.org/web/20180721143816/http://epaper.sxrb.com/shtml/sxwb/20170323/538841.shtml |archivedate=2018-07-21 |deadurl=no }}</ref>。十兄弟中老二[[傅作义|傅作义]]后来成为中国近现代史上一名重要的政治军事人物。

傅作恭成年后考入[[南京金陵大学|南京金陵大学]]园艺专业就读。[[抗战时期|抗战时期]]，金陵大学因[[南京|南京]]而迁往四川成都办学，傅作恭随之到成都就读。毕业后便到时任[[抗日戰爭第十二戰區|抗日战争第十二战区]]司令長官的二哥傅作义下辖的[[綏遠省_(中華民國)|绥远省]]工作，先后担任绥远陕坝农场场长和奋斗小学校长，其中奋斗小学为傅作义创办。<ref name="kangyi"/>

1950年8月，傅作恭从绥远到北京探望傅作义时，恰好遇见时任甘肃省政府主席的[[邓宝珊|邓宝珊]]。受邓宝珊邀请到甘肃工作，后任[[甘肃省林业厅|甘肃省林业厅]]造林科科长，曾主持开展了多项甘肃省的植树造林工作和[[兰州市|兰州市]]的绿化工作。<ref name="kangyi"/>

== 命丧夹边沟 ==
1957年的[[反右运动|反右运动]]和1959年的[[反右倾运动|反右倾运动]]相继爆发，傅作恭被甘肃省林业部门划为[[右派分子|右派分子]]，随后被送到[[酒泉市|酒泉市]]的[[夹边沟农场|夹边沟农场]]进行[[劳动教养|劳动教养]]<ref name="kangyi"/>。由于恰逢[[三年大饥荒|三年大饥荒]]，夹边沟农场关押劳教人员远超农场可自给自足的数量，当时农场管教和上级部门又按照“右派就是反动派”的逻辑未给农场提供充足的粮食，致使在夹边沟农场的部分劳动教养人员因饥饿等原因死亡<ref name=qius>{{Citeweb |url=http://minzhuzhongguo.org/MainArtShow.aspx?AID=83427 |title=正确评价夹边沟事件 |author=龚云 |website=求是网-世界社会主义研究 |date=2017-04-25 |accessdate=2017-08-20 |archiveurl=https://web.archive.org/web/20170820115934/http://minzhuzhongguo.org/MainArtShow.aspx?AID=83427 |archivedate=2017-08-20 |deadurl=no }}</ref>。

傅作恭写信向傅作义反映了夹边沟农场的问题和个人的生活困难。收到信后，傅作义让警卫秘书给傅作恭寄钱，并向中国共产党党中央反映了甘肃夹边沟农场的问题。中共党中央派出中纪委委员、中央监察委员会副主任[[钱瑛|钱瑛]]为组长、公安部副部长王昭为副组长的中央检察团，赴甘肃夹边沟等地进行实地调查，并及时将调查情况上报中央、通报给中共中央西北局。但当中央检察团来到夹边沟农场实地调查时，傅作恭已经在饥饿和疾病中不幸离世。<ref name="kangyi"/>

== 传言 ==
2000年出版的纪实性小说《[[夹边沟纪事|夹边沟纪事]]》中曾提到傅作恭，但称其为美国[[哥伦比亚大学|哥伦比亚大学]]毕业的水利工程学博士<ref>{{cite web |title=且慢笑刘峙 |url=http://ent.163.com/11/0525/14/74THHMBJ00032DGD.html |website=网易 |accessdate=2018-07-21 |archiveurl=https://web.archive.org/web/20180721143815/http://ent.163.com/11/0525/14/74THHMBJ00032DGD.html |archivedate=2018-07-21 |deadurl=no }}</ref>，1949年后被堂哥傅作义写信从美国劝回国内<ref>{{cite web |title=杨显惠：夹边沟事件是2400多名右派的苦难史 |url=http://history.sohu.com/20131119/n390332190.shtml |website=搜狐 |accessdate=2018-07-21 |archiveurl=https://web.archive.org/web/20140705205318/http://history.sohu.com/20131119/n390332190.shtml |archivedate=2014-07-05 |deadurl=no }}</ref>，与事实并不相符。<ref name="kangyi" />
