'''傅連暲'''{{BD|1894年|9月14日|1968年|3月29日|CatIdx=F傅}}，[[表字|字]]日新，出生於福建[[長汀縣|長汀縣]]河田鄉伯公嶺。曾任[[中央人民政府卫生部|中央人民政府卫生部]]副部长、[[中共中央军委|中共中央军委]]总后卫生部第一副部长<ref name="a1">{{Cite web |url=http://www.mxrb.cn/sqmx/2008-11/19/content_439188.htm |title=傅连暲中将 |accessdate=2015-05-13 |author= |date=2008-11-19 |work= |publisher=闽西新闻网 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20150518085255/http://www.mxrb.cn/sqmx/2008-11/19/content_439188.htm |archivedate=2015-05-18 }}</ref><ref>[http://cpc.people.com.cn/GB/64162/64172/85037/85038/6043584.html 保健医生回忆在刘少奇同志身边工作的日子]，中国共产党新闻网 2014-07-20查阅。{{zh-cn}}</ref>、[[中央保健委员会|中央保健委员会]]首任领导<ref>[http://www.gmw.cn/01wzb/2013-03/02/content_6981112.htm 中南海保健医生的规矩]，文摘报 2013-03-02（2014-07-20查阅）。{{zh-cn}}</ref>。[[中国人民政治协商会议第二届全国委员会委员名单|第二]]、[[中国人民政治协商会议第三届全国委员会委员名单|三届全国政协常务委员]]，[[中共八大|中共八大]]代表。

== 生平 ==
[[File:Changting_Zhongyang_Hongse_Yiyuan_Jiuzhi_2013.10.06_09-38-26.jpg|缩略图]]亚盛顿医馆（傅连暲纪念馆） ]]
父親傅貴成、母親官福娣受土豪欺壓，流浪至長汀縣。自幼加入[[基督教|基督教]]、後來學醫從醫，1918年，成為[[汀州|汀州]]紅十字會的主任醫師，1920年在醫校任教，並開辦診所。1925年，成為[[中央红色医院旧址|福音醫院]]院長。1927年，[[南昌起义|南昌起义]]部队途经长汀时，傅发动当地医务人员为起义部队服务。1929年，[[红四军|红四军]]攻占长汀县。傅为[[中国工农红军|红军]]和中共政权培训了大批医务人员。1933年，参加[[中国工农红军|中国工农红军]]，原福音医院改为中央红色医院，任院长。1934年，随部参加[[长征|长征]]。期间，[[毛澤東|毛澤東]]、[[周恩來|周恩來]]等领导人身患重病時，都是由他专门醫治康复的。

1937年1月，傅連暲在[[延安|延安]]组建中央苏维埃医院，任院长。1938年9月7日，加入[[中国共产党|中国共产党]]。同年冬，任[[中共中央|中共中央]]总卫生处处长。1945年，参加[[中共七大|中共七大]]，后任中共中央军委总卫生部副部长。

[[中华人民共和国|中华人民共和国]]建立后，任[[中央人民政府卫生部|中央人民政府卫生部]]副部长<ref name="a1"/>。1955年被授予[[中国人民解放军中将|中将]]军衔，获二级[[八一勋章_(1955年)|八一勋章]]、一级[[解放勋章|解放勋章]]（未获[[独立自由勋章|独立自由勋章]]）。

== 文革逝世 ==

[[抗美援朝|抗美援朝]]前夕，[[林彪|林彪]]称病拒绝带兵入朝，毛泽东安排傅连璋组织专家为林彪检查身体。会诊检查的结果是：林彪没有什么大病，但染上扎吗啡毒品的毛病。傅连暲只答应[[叶群|叶群]]对别人保密，但对毛泽东无论如何不能隐瞒。林彪因此对傅怀恨，埋下祸根 <ref>屈平 《傅连暲从基督徒到共和国中将》，《党史纵横》2005年第1期</ref><ref name="凤凰">{{cite web|author1=黎勤 郑淑芸 （来源：百年潮）|title=文革中傅连暲为何被迫害死：1950年查出林彪吸毒|url=http://news.ifeng.com/history/zhongguoxiandaishi/detail_2011_03/20/5257111_2.shtml|website=http://www.ifeng.com/|publisher=凤凰新媒体}}</ref>。

文化大革命期间，傅連暲受到迫害，身前他曾寫信請求毛澤東，希望并相信毛泽东一定会救他 <ref name="凤凰"/>。1966年9月3日，毛澤東做了批示：對傅連暲“應予保護”，傅舉家搬至[[香山_(北京)|香山]]附近“安心靜養”，1968年3月，再次被抄家，被[[紅衛兵|紅衛兵]]冠以“衛生部頭號走資派”、“三反分子”、“天主教會間諜”等罪名，死於[[秦城监狱|秦城监狱]]中。1973年11月，经毛泽东批示，[[中国人民解放军总政治部|解放军总政治部]]批准，[[追认|追认]]为[[革命烈士|革命烈士]]<ref name="a1"/>。1975年5月17日，毛澤東批示：“傅已入土，嗚呼哀哉”，“应予以昭雪”。7月，经中共中央军委批准[[恢复名誉|恢复名誉]]。9月，[[中国人民解放军总后勤部|解放军总后勤部]]举行安灵[[仪文|仪文]]<ref name="a1"/>。
