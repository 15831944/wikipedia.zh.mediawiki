== 歷史 ==
[[傅振嵩|傅振嵩]]-{zh-hans:16歲開; zh-tw:16歲開}-始跟-{zh-hans:從; zh-tw:從}-[[陳延禧|-{zh-hans:陳; zh-tw:陳}-延禧]]-{zh-hans:學習; zh-tw:學習}-[[陳式太極拳|-{zh-hans:陳; zh-tw:陳}-式-{zh-hans:太極拳; zh-tw:太極拳}-]]-{zh-hans:。三年後，傅又拜; zh-tw:。三年後，傅又拜}-[[賈岐山|-{zh-hans:賈; zh-tw:賈}-岐-{zh-hans:山; zh-tw:山}-]]-{zh-hans:為師學習; zh-tw:為師學習}-[[八卦掌|八卦-{zh-hans:掌; zh-tw:掌}-]]-{zh-hans:。傅是; zh-tw:。傅是}-最早-{zh-hans:學習這; zh-tw:學習這}-些-{zh-hans:武術的人之一，因為陳; zh-tw:武術的人之一，因為陳}-式-{zh-hans:大極拳一; zh-tw:大極拳一}-直秘-{zh-hans:不外傳，直至; zh-tw:不外傳，直至}-傅氏-{zh-hans:的時候才開; zh-tw:的時候才開}-始招-{zh-hans:收外人; zh-tw:收外人}-不久；而-{zh-hans:董; zh-tw:董}-海川-{zh-hans:才創了; zh-tw:才創了}-八卦-{zh-hans:掌數十年，也只收了幾; zh-tw:掌數十年，也只收了幾}-個徒弟-{zh-hans:，其中一個就是賈; zh-tw:，其中一個就是賈}-琦-{zh-hans:山。; zh-tw:山。}-

傅在26歲時，已經因為單槍匹馬擊敗大批匪徒而聞名，這個故事有多個版本<ref>{{Cite book|title=Fu Zhen Song's Dragon Bagua Zhang|last=Lin|first=Chao Zhen|publisher=Blue Snake Books|year=2010|isbn=978-1-58394-238-3|location=USA|pages=24-25}}</ref>。

傅振嵩後來去了北京，在那裡遇到了當時其他偉大的八卦掌大師。他向他們學習並交流了有關的武術心得。1928年，三位將軍在南京舉辦了首屆全國武術大賽，傅振松是那次比賽的五位評委之一。比賽的冠軍是一位姓王的年輕人，將軍想要測試他的真正本領，當時所有的評委都「提名」傅做「測試者」。他別無選擇，與比賽勝出者作一比武。根據在場的人士說，這場較量持續了很長時間，最後，傅以一記重拳將王打出台外並完結了這場賽事<ref>{{Cite book|title=Fu Zhen Song's Dragon Bagua Zhang|last=Lin|first=Chao Zhen|publisher=Blue Snake Books|year=2010|isbn=978-1-58394-238-3|location=USA|pages=38-39}}</ref>。這是傅著名的三場備受矚目的決鬥之一，另外兩場是與年齡較大的[[八極拳|八極拳]]大師[[李書文|李書文]]的對決，還有一場傅在家鄉與一大群習[[查拳|查拳]]的[[穆斯林|穆斯林]]的搏鬥。

在此期間，傅與[[孫祿堂|孫祿堂]]成為了好友，並教他八卦掌的「趟泥步」。傅又從[[楊澄甫|楊澄甫]]學習[[楊氏太極拳|楊式太極拳]]，有一天在「推手」的比賽中擊敗了他。楊說，「你之所以贏，是因為你用了八卦掌。」傅還與[[李景林|李景林]]一起學習劍術。<ref>Lin(2010年)，第31，36至37頁</ref>

1929年，兩廣國術館在廣州成立，傅振嵩與另外四位武術名家：[[萬籟聲|萬籟聲]]，[[顧汝章|顧汝章]]，[[耿得海|耿得海]]和[[王少舟|王少舟]]應邀赴廣州任教。這就是武林掌故中所說的-{zh-hans:“; zh-tw:「}-五虎下江南-{zh-hans:”; zh-tw:」}-。他們五個不斷受到南方武術家的挑戰，因為南方武術家們對自家的武功非常自豪，對北方武術有所抗拒。但傅未曾失過一場挑戰。<ref>Lin(2010年)，第42至43頁</ref>

傅振嵩搬到廣州居住後，在那裡主持一所武術學校。那時，傅通過學習各家太極拳、不同風格的八卦掌、來自宋唯一的武當劍（可能是從[[李景林|李景林]]那裡學到的，雖然傅在宋下學習了一段時間）、形意拳和八極拳，強調每種功夫最重要的原則和技巧，並消除他認為沒有價值或沒有實質內容的部分，從而揉合出自成一格的武術系統。傅式八卦掌包括陰陽手掌變化、著名的[[龍形八卦掌|龍形八卦掌]]、八卦掌和太極拳的兩儀四象。所使用的許多名字都可能受到易經的啟發，招式則受到孫祿堂的武功和哲學所影響。<ref>Lin(2010年)，pp.37-38,69</ref>

== 學習過程 ==
傅振嵩了解到太極拳和八卦掌之間的巨大差距，他為這個差距創造了一個優雅的解決方案。傅創造了一套招式，他稱之為「兩儀拳」。這套招式是傅式系統學習八卦掌的重要關鍵，因為它是從太極拳到八卦掌所需的一系列先決動作和技巧。

換句話說，如果想學習傅氏八卦掌，他必須要先學好傅氏太極拳，然後學習傅氏兩儀拳至非常熟練的程度，以便進入最高水平，才可練習傅氏八卦掌。

傅氏八卦掌的特-{zh-hans:点; zh-tw:點}-是大量的旋轉動作和點攻擊。 這種風格配以準繩的攻打招式可將對手的-{zh-hans:内; zh-tw:內}-部器官造成破壞。

== 今天的傅氏八卦掌 ==
傅氏八卦掌後由傅振嵩的兒子[[傅永輝|傅永輝]]承傳。後者在原來基礎上增加「四象」、進階太極及其它。傅永輝的徒弟包括[[麥寶嬋|麥寶嬋]]，即甄子丹之母<ref>{{Cite web|url=https://www.itsfun.com.tw/%E9%BA%A5%E5%AF%B6%E5%AC%8B/wiki-2888646-8645526|title=麥寶嬋：麥寶嬋（Bow Sim Mark），祖籍廣東，現居美國，中國武術研究院， -華人百科|accessdate=2018-12-21|work=www.itsfun.com.tw}}</ref>。現在，這個門派由傅永輝居於加拿大溫哥華的兒子傅勝龍(Victor Fu Sheng Long)來繼承。由於他認為現代人沒有足夠的時間來學習傅氏八卦掌的整體，因此他簡化了拳法，認為強身健體更重要，而不是「可以用手把桌子分成兩半」。然而，通過結合調節練習，兩人套路和八卦推手形式的練習，武術精粹得以保持不變。

傅氏八卦掌的另一個分支是由傅振嵩的弟子林朝珍建立的<ref>{{Cite web|url=https://sites.google.com/site/southeninn/about-us/linchaozhen|title=林朝珍 - Southen Inn Kung Fu 南方愛武小站|accessdate=2018-12-21|work=sites.google.com}}</ref>，他同樣修改了教學方法。<br />
