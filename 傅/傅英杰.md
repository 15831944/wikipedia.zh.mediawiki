'''傅英杰'''（{{bd|1925年|1月|2017年|3月22日}}），[[河北省|河北省]][[青县|青县]]人，[[中国人民解放军少将|中国人民解放军少将]]<ref name=pp>{{Cite web |url = http://www.thepaper.cn/newsDetail_forward_1692808 |title = 原总参谋部某部政治委员傅英杰逝世，享年92岁 |author =  |date = 2017-05-24 |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>。

==生平==
傅英杰是河北省青县人，生于1925年1月。1938年3月，参加[[中国共产党|中国共产党]]领导的革命工作。1939年9月入伍。1942年1月加入中国共产党。历任科长、处长、副局长、局长等职。后来担任[[中国人民解放军总参谋部技术侦察部|中国人民解放军总参谋部技术侦察部]]政治委员<ref>{{Cite web|url=http://cpc.people.com.cn/n1/2017/0524/c87393-29297305.html|title=傅英杰同志逝世--新闻报道-人民网|accessdate=2017-05-29|last=103797|work=cpc.people.com.cn}}</ref>。离休后为正军职离休干部<ref name=pp/>。他是[[第七届全国人大代表|第七届全国人大代表]]<ref>{{Cite web |url = http://www.npc.gov.cn/wxzl/gongbao/1988-03/20/content_1481062.htm |title = 中华人民共和国第七届全国人民代表大会代表名单 |author =  |date = 1988-03-20 |publisher =中国人大网  |language =  |accessdate =  |quote =  }}</ref>。

2017年3月22日，傅英杰在[[北京|北京]]病逝，享年92岁<ref name=pp/>。
