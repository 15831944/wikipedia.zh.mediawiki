'''傅筱庵'''（{{bd|1872年|12月30日|1940年|10月10日|catIdx=F傅}}），名'''宗耀'''，字'''筱庵'''，[[以字行|以字行]]，[[浙江省|浙江省]][[寧波府|寧波府]][[鎮海縣|鎮海縣]]人，[[清朝|清末]]及[[中华民国|中华民国]]实业家、银行家、政治家，[[抗日战争|抗日战争]]时期任[[中華民国維新政府|中華民国維新政府]]、[[汪精衛政权|汪精衛政权]][[上海特别市政府_(日占时期)|上海特别市]]市长。<ref name=xu>{{Cite book|author = 徐友春主編|title = 民国人物大辞典 増訂版|year = 2007|publisher = 河北人民出版社|isbn = 978-7-202-03014-1}}</ref><ref name=liu>{{Cite book|author = 劉寿林等編|title = 民国職官年表|year = 1995|publisher = [[中華書局|中華書局]]|isbn = 7-101-01320-1}}</ref><ref name=zx>{{Cite book|author = |title = 最新支那要人伝|year = 1941|publisher = [[朝日新聞社|朝日新聞社]]|isbn = }}</ref>

==生平==
[[File:Fu_Xiaoan.jpg|thumb]]》中的傅筱庵照片]]
[[File:傅宗耀.jpg|thumb]]
1892年（[[光緒|光緒]]18年），傅筱庵任职于[[上海市|上海]]浦東的船工場，并在夜校学习[[英語|英語]]。1909年（[[宣統|宣統]]元年），转入[[招商局|招商局]]所属的華興保险公司，历任副经理、总经理。1911年（宣統3年）10月，[[辛亥革命|辛亥革命]]爆发。[[辛亥上海光复|辛亥上海光复]]时，傅筱庵任[[滬軍都督府|滬軍都督府]]財政部总参議、滬关清理处处長。此后，先后任[[招商局|招商局]]、[[汉冶萍公司|汉冶萍公司]]、[[中国通商銀行|中国通商銀行]]的股权代表。<ref name=xu/><ref name=liu/><ref name=zx/>

1914年（[[民国紀元|民国]]3年），傅筱庵被[[盛宣怀|盛宣怀]]任命为[[招商局|招商局]]经理各地栈租。他还担任中国通商銀行董事、[[上海总商会|上海总商会]]第一届議董。1916年（民国5年）4月，[[盛宣怀|盛宣怀]]病逝，傅筱庵任招商局董事、招商局積余产业经理、内河輪船公司经理兼主船課長。此後，先后在上海总商会、[[中国銀行|中国銀行]]、汉冶萍公司等中国国内重要企业担任要職。1926年（民国15年），任上海总商会会長。但是，[[中国国民党|中国国民党]]的[[北伐|北伐]]軍进入上海后，傅筱庵因为支援[[北洋政府|北京政府]]方面的[[孫传芳|孫传芳]]而遭到通缉，被迫逃到[[大連市|大連]]。<ref name=xu/><ref name=liu/><ref name=zx/>

1931年（民国20年），对傅筱庵的通缉被解除，傅筱庵回到上海，担任中国通商銀行总经理、董事長。1938年（民国27年）10月，被任命为[[中華民国維新政府|中華民国維新政府]]之下的[[上海特别市|上海特别市]]市長（[[汪精卫政权|南京国民政府]]（汪精卫政权）成立後留任）。同年，[[蒋介石|蒋介石]]的[[重慶国民政府|重慶国民政府]]的[[軍統局|軍統]]派出特務，企图暗杀正在上海逗留的[[汪兆銘|汪兆銘]]（汪精衛）。傅筱庵起初与軍統協作，但在暗杀实行前向汪兆铭告密，导致参与暗殺计划的军统特務们遭到逮捕。蒋介石十分愤怒，軍統领导人[[戴笠|戴笠]]严令暗殺傅筱庵。<ref name=xu/><ref name=liu/><ref name=zx/>

1940年（民国29年）10月10日，傅筱庵在就寝中被軍統收买的廚子阿朱<ref>{{Cite book|title=抗戰時代生活史|last=陳|first=存仁|publisher=廣西師範大學出版社|year=|isbn=9787563365388|location=|pages=106}}</ref>暗杀。终年69岁（满67歳）。<ref name=xu/><ref name=zx/>
