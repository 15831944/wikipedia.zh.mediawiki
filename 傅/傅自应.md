'''傅自应'''（{{bd|1957年|9月|||catIdx=F傅}}），祖籍[[中國|中國]][[湖北|湖北]][[监利|监利]]，生于[[湖南|湖南]][[岳阳|岳阳]]<ref name=jianlizf>{{Cite web |url =http://lvyou.jianli.gov.cn/Item/2866.aspx  |title = 傅自应 |author =  |date =2013-11-07  |publisher = 监利县人民政府门户网 |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =  http://finance.ifeng.com/people/official/fuziying.shtml|title =傅自应  |author =  |date =  |publisher = 凤凰网 |language =  |accessdate = 2017-03-08 |quote =  }}</ref>。1974年10月参加工作，1992年7月加入[[中国共产党|中国共产党]]。中国共产党第十九届中央委员会候补委员。

==生平==
1974年10月，在[[湖南省|湖南省]][[岳阳县|岳阳县]]广兴公社立新大队插队。1976年11月，任[[中华人民共和国化学工业部|中华人民共和国化学工业部]]第四化建公司机械加工厂工人。1978年9月，进入[[湖南财经学院|湖南财经学院]]（现并入[[湖南大学|湖南大学]]）工业经济系工业会计专业学习。1981年8月，任[[中华人民共和国对外贸易部|中华人民共和国对外贸易部]]财会局制度处干部。1982年4月，任[[中华人民共和国对外经济贸易部|中华人民共和国对外经济贸易部]]财会局外贸财务处干部。1987年7月，任中华人民共和国对外经济贸易部财会司奖励处副处长、制度处副处长。1993年1月，任[[中华人民共和国对外贸易经济合作部|中华人民共和国对外贸易经济合作部]]经济调节司副处长。1993年10月，任中华人民共和国对外贸易经济合作部利安达会计师事务所主任会计师（副处级）。1994年8月，任中华人民共和国对外贸易经济合作部利安达会计师事务所主任会计师（正处级）。1995年3月，任中华人民共和国对外贸易经济合作部利安达会计师事务所主任会计师（副司级）。1997年9月，任中华人民共和国对外贸易经济合作部计划财务司司长（其间，2001年3月到2002年1月在中央党校一年制中青年干部培训班学习，2001年9月到2004年7月为[[中国社会科学院研究生院|中国社会科学院研究生院]]财贸经济系财政学专业在职博士研究生）<ref name=jianli>{{cite web|date=2011-11-28|url=http://leaders.people.com.cn/GB/16406713.html|title=傅自应同志简历|publisher=人民网}}</ref>。

2003年6月，任[[中华人民共和国商务部|中华人民共和国商务部]]规划财务司司长。2003年7月，任中华人民共和国商务部部长助理、党组成员兼规划财务司司长。2004年2月，任中华人民共和国商务部部长助理、党组成员。2008年3月，任中华人民共和国商务部副部长、党组成员<ref name=jianli/>。

2011年11月，任[[江苏省人民政府|江苏省人民政府]]副省长、党组成员<ref name=jianli/><ref>{{cite web|date=2011-11-27|url=http://finance.sina.com.cn/roll/20111127/070810887852.shtml|title=商务部原副部长傅自应任江苏副省长|publisher=新浪}}</ref><ref name=jianlizf/>。

2015年3月，任[[中共中央纪委|中共中央纪委]]驻[[中共中央宣传部|中共中央宣传部]]纪检组组长<ref>{{cite web|publisher=新浪|date=2015-05-29|title=傅自应辞去江苏副省长职务 已赴中宣部任职|url=http://news.sina.com.cn/c/2015-05-29/105831888963.shtml}}</ref><ref>{{Cite web |url =  http://news.xinhuanet.com/legal/2015-04/01/c_127647004.htm|title =盘点中央纪委新任7派驻组长简历 最年轻者55岁  |author =  |date = 2015-04-01 |publisher =  新华网|language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://news.sina.com.cn/c/2015-08-20/doc-ifxhcvsc4223452.shtml |title =王岐山的28位“特种兵”：7人曾是省委常委  |author =  |date = 2015-08-20 |publisher =  新浪|language =  |accessdate =  |quote =  }}</ref>。

2017年2月，任[[中华人民共和国商务部|中华人民共和国商务部]]国际贸易谈判代表、副部长、党组副书记（正部长级）<ref>{{Cite web |url = http://www.guancha.cn/economy/2017_03_08_397796.shtml |title =新任商务部国际贸易谈判代表傅自应：从中纪委走出来的谈判高手  |author =  |date = 2017-03-08 |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref><ref>{{cite web|title=商务部副部长、党组副书记傅自应被明确为正部长级|url=http://www.thepaper.cn/newsDetail_forward_1635487|website=澎湃新闻|accessdate=2017-03-08}}</ref>。2018年12月，转任[[澳门中联办|澳门中联办]]主任<ref>{{cite web|title=国务院任免国家工作人员|url=http://www.mohrss.gov.cn/gkml/bnrsjy/rsrm/201812/t20181228_307839.html|website=中华人民共和国人力资源和社会保障部|accessdate=2018-12-28}}</ref>。
