'''亮島'''，古稱'''橫山'''，舊稱'''浪島'''，位於[[馬祖列島|馬祖列島]][[北竿鄉|北竿]]與[[東引鄉|東引]]之間，行政上隸屬[[中華民國|中華民國]][[連江縣_(中華民國)|連江縣]][[北竿鄉|北竿鄉]][[橋仔村|橋仔村]]。島嶼長約1400公尺、寬250公尺，面積約0.4平方公里，四周峭壁陡峻，地形顯要，距離中國大陸[[蜘蛛島|蜘蛛島]]24公里，[[雙峰島|雙峰島]]26公里。後因呼應「島立天中，亮照大陸」的期許而更名為''亮島''。<ref>[http://www.chinatimes.com/newspapers/20141004000687-260116 台／灣／趴／趴／走－登陸亮島 一探史前貝塚]</ref>。因限制民間人士登岸，至今仍是軍事重地。

== 名稱 ==
古稱'''橫山'''，現時漁民仍沿續此稱呼。清代周凱纂輯的《廈門志 ‧卷四》附〈北洋海道考〉即載有此名。民國16年（1927）曹剛修、邱景雍纂《福建省連江縣志》仍稱橫山，更名浪島之時間不詳<ref>[http://www.matsu.gov.tw/2008web/news_cnt.php?id=5091&room=news 馬祖亮島島尾遺址群發掘及「亮島人」修復計畫] {{webarchive|url=https://web.archive.org/web/20151225085957/http://www.matsu.gov.tw/2008web/news_cnt.php?id=5091&room=news |date=2015-12-25 }}，連江縣政府</ref>。

== 歷史 ==
1949年後，浪島斷斷續續由[[中国人民解放军|中國人民解放軍]]佔領。

1951年7月，[[中華民國國軍|中華民國國軍]]為維護馬祖[[南竿|南竿]]、[[北竿|北竿]]與[[東引|東引]]之間的航道安全，遂派遣[[反共救國軍|反共救國軍]]海上特種突擊中隊長[[李承山|李承山]]率六名隊員偵搜上岸，順利在浪島上豎立[[青天白日滿地紅旗|青天白日滿地紅旗]]。這是唯一解放軍失守、並由國軍取回的土地。爾後，[[中央研究院|中央研究院]]歷史語言研究所兼任研究員[[陳仲玉|陳仲玉]]等研究團隊，在島上陸續發現多處貝塚遺址，尤其在亮島島尾的遺址群中除了挖掘出新石器時代的陶片、石器、骨器和貝殼外，還有三片人頭蓋骨；1965年，被發現有貝塚的百勝港遺址被興建起來<ref>[http://www.to-go.com.tw/article_info.php?id=335 《TO'GO泛遊情報》雜誌第208期（2014年11月號）] </ref>。

1966年，[[蔣經國|蔣經國]]視察此島，宣布更名'''亮島'''，期許以「島立天中，亮照大陸」之名來照亮[[中國大陸|中國大陸]]。

2004年，島上新建樺恩碼頭，讓現在能夠得以順利進行一周兩次的物資運補。

== 交通 ==
1965年於亮島西南隅興建港口，名為[[百勝港|百勝港]]，而此名乃得名自當時蓋港爆破時，不幸因公殉難的士兵錢百勝先生。<ref>http://www.to-go.com.tw/article_info.php?id=335</ref>而該港附近的邊坡斷面疑似有貝塚，因而通知中央研究院歷史語言研究所兼任研究員陳仲玉，並進一步邀集在地文史工作者組成「馬祖亮島考古隊」展開田野調查。然而由於百勝港易受強烈東北季風和潮汐影響，使得船隻進入[[百勝港|百勝港]]運補不易，於2004年新建[[樺恩碼頭|樺恩碼頭]]，由樺棋營造公司斥資千萬協助興建，改善後，可避開東北季風颳起後的潮浪及影響，有利每周兩次的運補作業。<ref>http://www.chinatimes.com/newspapers/20141004000687-260116</ref>

==亮島之歌==
中華民國國軍著名[[軍歌|軍歌]]《'''[[亮島之歌|亮島之歌]]'''》，是駐守亮島的官兵，由[[中華職棒|中華職棒]]主題曲改編而成。

==亮島人==
連江縣政府與中研院合作，於亮島挖掘出新石器時代人骨（距今約8300-7900多年前），命名為「亮島人」。<ref>[http://www.ettoday.net/news/20120402/36026.htm 7900年！馬祖「亮島人」完整骨骸出土] </ref>
亮島人出於「亮島島尾一號遺址」，而研究團隊亦於「亮島島尾二號遺址」發現史前貝塚。<ref> http://www.chinatimes.com/newspapers/20141004000687-260116</ref>
