'''犰狳'''（{{標音|拼音=Qiú yú|注音=ㄑㄧㄡˊ　ㄩˊ|粵拼=kau4 jyu4|同音字=求餘}}，[[學名|學名]]：{{lang|la|''Cingulata''}}）是有壳的[[哺乳动物|哺乳动物]]，生活在[[中南美洲|中南美洲]]和[[美国|美国]]南部地区。它们的壳分三部分，前后两部分有整块不能伸缩的骨质[[鳞甲|鳞甲]]覆盖，中段的[[鳞甲|鳞甲]]成带状，与肌肉连在一起，可以自由伸缩。尾巴和腿上也有鳞片，鳞片之间还长着毛，腹部无鳞片只有毛。虽然犰狳科动物口中牙齿数量很多，但都很细小。

犰狳科是[[異關節總目|異關節總目]]（又稱貧齒總目）[[有甲目|有甲目]]下的一科。有甲目內包括三科，但現存僅餘下犰狳科一科。犰狳共有9屬21种（有1屬1種已[[滅絕|滅絕]]），根据其中段鳞甲带的多少来区分它们的种类，如有三条鳞甲带的叫[[三带犰狳|三带犰狳]]，有九条鳞甲带的叫[[九带犰狳|九带犰狳]]等等<ref>[http://net.1nongjing.com/a/201506/100264.html 九带犰狳揭秘]</ref>。

== 分布 ==
犰狳是[[美洲|美洲]]的特产。犰狳科是现存異關節總目中种类最多，分布最广的一类。犰狳科栖息于不同的生活环境如[[森林|森林]]、[[草原|草原]]和[[半荒漠|半荒漠]]等。

== 习性 ==
一年中的任何时候都可繁殖，并且每胎通常都有4只。[[九带犰狳|九带犰狳]]是掘洞好手，能在两分钟内钻入坚硬的土地裡，大部分犰狳居于洞中，夜间出来活动，犰狳长得虽威武，但它们生性胆小，一遇[[天敌|天敌]]便钻进洞内，如果来不及，便缩成一团，以坚硬的[[鳞甲|鳞甲]]保护身体。

== 食物 ==
它们是杂食动物，几乎没有不吃的小动物，主要吃[[昆虫|昆虫]]。一只九带犰狳一年要吃掉100公斤昆虫，其中害虫要占四分之三，同时还吃腐肉、毒[[蜘蛛|蜘蛛]]、[[蝎子|蝎子]]和[[蛇|蛇]]等，它们对农作物有很好的保护作用。

==分類==
[[file:Piche_(Zaedyus_pichiy)_in_laguna_Las_Coloradas_Chubut.JPG|thumb]] ''Zaedyus pichiy'']]
犰狳科是[[有甲目|有甲目]]的物種，而有甲目亦作貧齒目。然而「貧齒目」這個名稱帶歧義，所以現時一般都叫做「有甲目」。

犰狳科過往分為三個亞科。後來[[倭犰狳亞科|倭犰狳亞科]]分為兩個亞科：當中主要成員不包括倭犰狳屬，是為[[六帶犰狳亞科|六帶犰狳亞科]]；餘下[[圆头倭犰狳屬|圆头倭犰狳屬]]及[[倭犰狳屬|倭犰狳屬]]物種沿用「倭犰狳亞科」這名稱，但學名改為跟從倭犰狳屬的學名（[[Chlamyphorinae|Chlamyphorinae]]）。
