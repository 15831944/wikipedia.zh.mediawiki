'''互动百科诉百度百科侵权事件'''是指从2010年开始，[[互动百科|互动百科]]向[[北京市海淀区人民法院|北京市海淀区人民法院]]提起[[民事诉讼|民事诉讼]]，状告[[百度|百度]]旗下[[百度百科|百度百科]]大量侵犯互动百科[[著作权|著作权]]词条内容。此前，互动百科曾经多次发通知给百度要求其移除百度百科[[侵权|侵权]]词条内容，但是百度对此置之不理；互动百科为保护[[知识产权|知识产权]]，最终将百度告上法庭<ref name="chzz " >{{cite web |url=http://www.chinaz.com/news/2010/1027/139945.shtml |title=互动百科诉百度百科侵权 双方将进行调解 |publisher= 中国站长网|accessdate= 2010年|work= |language =zh-cn}}</ref><ref name=" kyw" >{{cite web |url= http://www.oschina.net/news/12734/hudong-suse-baidu?from=rss|title=互动百科诉百度百科侵权正式立案 |publisher=开源中国社区 |accessdate= 2010年|work= |language =zh-cn}}</ref>。

== 事件经过 ==
2010年10月27日，[[互动百科|互动百科]]向北京市海淀区人民法院提起[[民事诉讼|民事诉讼]]，状告百度百科大量侵犯互动百科著作权词条内容；海淀区法院随即受理案件，并且在法院的主持下，双方律师展开谈判。互动百科提出的调解协议内容为<ref name="chzz " /><ref name=" kyw" />：
{{quote|
第一，立即移除侵犯互动百科著作权的词条内容及快照。<br />第二，冻结长期抄袭互动百科词条内容于百度百科的用户帐号，同时要求百度提供这些帐号的用户注册信息，互动百科保留进一步起诉该用户的权利。<br />第三，以管理员身份向涉及侵权的用户发布站内“警示函”，提醒百度百科用户在不注明出处前提下不得再抄袭互动百科内容，“警示函”内容需由互动百科确认。<br />第四，要求百度，对互动百科多次发送通知函并已签收情况下却不予回复行为做出解释，并保证日后类似事件的不再发生。 
}}

根据《IT时报》记者调查了解到，百度百科原原本本抄袭互动百科词条至少数千条，在百度百科的“情感词条”随即一搜，对比之后便发现“豆腐渣婚姻”、“两地婚姻”、“凑合婚姻”和“婚姻年检”原原本本复制互动百科词条内容，而创建时间稍微晚于互动百科词条创建时间；同时，百度百科几十个活跃科友每天盯着互动百科的新词条，原封不动地复制[[粘贴|粘贴]]词条内容。<ref name="tjw ">{{cite web|url=http://www.tianjinwe.com/rollnews/201011/t20101108_2406798.html|title=盛大文学互动百科相继控诉百度侵权|publisher=天津网|accessdate=2010年|work=|language=zh-cn}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="fhwa " >{{cite web |url=http://tech.ifeng.com/internet/detail_2010_11/08/3039717_0.shtml?_from_ralated |title=盛大互动百科轮番炮轰百度侵权 |publisher= 凤凰网|accessdate= 2010年|work= |language =zh-cn}}</ref>

互动百科向媒体透露，他们曾经数百次用电话或通知函的形式与百度进行沟通，但百度对此置之不理。<ref name="tjw " /><ref name="fhwa " />

2011年年初，互动百科向[[国家工商行政管理总局|国家工商行政管理总局]]提交针对百度的[[反垄断|反垄断]]调查申请。<ref name="shws " >{{cite web |url=http://roll.sohu.com/20110720/n314017211.shtml |title=互动百科反遭百度起诉 侵权纠纷未曾停息 |publisher=搜狐网 |accessdate= 2011年|work= |language =zh-cn}}</ref>

2011年3月1日，互动百科总裁[[潘海东|潘海东]]曾对外界表示，已与部分[[全国人大代表|全国人大代表]]和[[全国政协委员|全国政协委员]]接触，希望他们在[[两会|两会]]上就“拆分百度”提案；但此事最终未果。<ref name="shws " />

== 开庭 ==
2011年11月1日，互动百科状告百度垄断案在[[北京市第一中级人民法院|北京市第一中级人民法院]]开庭。其后，双方进行了调解，但由于该案件的文书在“中国裁判文书网”上并未公布，相关媒体也未持续跟进，此案进展不明。<ref>{{Cite web|url = http://tech.163.com/11/1101/18/7HPV2546000915BF.html|title = 互动百科诉百度垄断案开庭|accessdate = |author = |date = 2011-11-01|publisher = 网易}}</ref>

2011年8月底，[[百度|百度]]诉[[互动百科|互动百科]] CEO [[潘海东|潘海东]]名誉侵权案在北京市海淀区人民法院作出一审判决。判决书中表示，潘海东对百度的指责无事实依据，构成侮辱及诽谤的侵权行为，判处潘海东等赔偿[[百度|百度]][[人民币|人民币]]12万元并公开道歉。<ref>{{Cite web|url = http://tech.sina.com.cn/i/2011-11-01/18516260517.shtml|title = 互动百科CEO潘海东一审被判赔偿百度12万元|accessdate = |author = |date = 2011年11月1日|publisher = 新浪}}</ref>。

== 社会反响 ==
*《赛迪网》评论：“百度百科与互动百科的纠纷由来已久。前者依靠自身强大的[[搜索引擎|搜索引擎]]，长期以来霸占百科市场；而后者尽管自身不断创新努力，仍难以与其抗衡，实为无奈。”<ref name="shws " />
