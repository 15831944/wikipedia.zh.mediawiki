'''互動式資訊服務站'''（{{lang-en|Interactive kiosk}}，簡稱：Kiosk）是一部[[終端|電腦終端機]]，以特定規格的硬體與軟體來製作，用來提供通訊、商業、娛樂或是教育的資訊與應用服務。<p>早期的互動式資訊服務站外觀很像[[电话亭|电话亭]]，包含了零售與提供食物的服務，並且更具有親和力來改善客戶服務。通常它是被放置在人流與車流較高的區域例如商場、旅館大廳或是[[機場|機場]]。資訊的技術整合使它可以提供更多元的功能，演化成自助服務，例如使用者（消費者）遇到無現有商品庫存時可以用它來訂購、查閱圖書館藏書、搜尋商品資訊、提供{{translink|en|Keycard lock|旅館門卡}}、[[公用事业|公用事业]]的帳單交易、以及收取現金讓使用者可以換取商品或是貨品。客製化零組件可以強化互動式資訊服務站的服務功能，更可以滿足貼近擁有者（商家）的需求，例如[[硬幣|投幣裝置]]、[[驗鈔機|紙鈔接收器]]、[[读卡器|读卡器]]以及[[感熱紙|感熱紙印表機]]等零組件可以給互動式資訊服務站提供交易與資料交換的功能。

== 簡介 ==
通常是採用[[觸控式螢幕|觸控式螢幕]]，再搭配[[多媒體|多媒體]]動態介面以及語音或音效，提供人性化且互動式的使用者介面。日本便利商店[[LAWSON|LAWSON]]從1997年開始，就陸續在門市導入8,000多台的「LOPPY」多媒體機台，目前已經是日本便利商店極為重要的一項服務項目。目前多媒體機台在台灣主要的應用也都是在於[[便利商店|便利商店]]應用，在店面提供[[消費者|消費者]]用於訂購票券、繳費或小額付款、兌換商品等便利服務上面。台灣目前便利商店的多媒體機台應用十分蓬勃發展，一年使用人次已經超過兩億之數<ref >[http://www.ectimes.org.tw/Shownews.aspx?id=130208234228 Kiosk機台上看二億使用人次　「蛇」麼服務攏A通]</ref><ref>[http://www.bnext.com.tw/article/view/cid/0/id/26482 2億人次的新商機 - 四大超商大戰Kiosk服務] {{webarchive|url=https://web.archive.org/web/20130706085431/http://www.bnext.com.tw/article/view/cid/0/id/26482 |date=2013-07-06 }}</ref>。

== 功能 ==
訂購票券、繳費或小額付款、兌換商品、下載列印等便利服務

== 應用 ==
[[File:FM001.JPG|thumb]]
在台灣，最早啟動多媒體機台服務的[[便利商店|便利商店]]為萊爾富在 2003 年推出的「Life-ET」；2005 年，OK 便利商店也推出類似的多媒體機台服務，而在 2006 年統一超商推出「ibon」，全家便利商店也推出「FamiPort」。<ref>[http://www.libertytimes.com.tw/2007/new/aug/7/today-e3.htm 超商紛設多媒體機台 拓商機] {{webarchive|url=https://web.archive.org/web/20090407053648/http://www.libertytimes.com.tw/2007/new/aug/7/today-e3.htm |date=2009-04-07 }}</ref>，目前實際運用的多媒體機台包括了：
*[[全家便利商店|全家便利商店]]的[[FamiPort|FamiPort]]
*[[統一超商|統一超商]]的[[ibon|ibon]]
*[[萊爾富便利商店|萊爾富便利商店]]的[[Life-ET|Life-ET]]
