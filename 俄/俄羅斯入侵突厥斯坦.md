'''俄羅斯入侵突厥斯坦'''（Russian conquest of Turkestan）又稱'''俄羅斯入侵中亞'''（Russian conquest of Central Asia），是19世紀俄羅斯帝國對[[突厥斯坦|突厥斯坦]]（[[中亞|中亞]]、[[新疆|新疆]]的統稱）發動的一系列入侵戰爭，是俄羅斯擴張戰爭（泛指沙俄對高加索、中亞、西伯利亞的入侵戰爭）的一部分。

==背景==
{{see also|俄羅斯帝國#歷史}}
俄國早在[[彼得大帝|彼得大帝]]時期，已有統治中亞的野心。1717年，俄國入侵[[希瓦汗國|希瓦汗國]]，遭到擊退。
==概述==
作為[[新帝國主義|新帝國主義]]主要侵略國之一，俄國自19世紀以來，對突厥斯坦的野心有增無減。
===入侵中亞===
1839年，俄國再度入侵希瓦汗國，由於天氣寒冷，俄軍及其所乘駱駝凍死無數，俄軍不得不撤退。

1842年起，俄國入侵[[布哈拉酋長國|布哈拉酋長國]]，1868年，布哈拉酋長國投降，成為俄國的[[保護國|保護國]]。

1873年，俄國又再度入侵希瓦汗國，希瓦汗國投降，成為俄國的保護國。

1879年起，俄國入侵[[土庫曼斯坦|土庫曼斯坦]]，1884年，[[土庫曼人|土庫曼人]]投降，土庫曼斯坦併入俄國。1885年，俄國將向南推至[[阿富汗|阿富汗]]邊界附近後，便停止前進，俄國入侵中亞的戰爭大致結束。

===入侵新疆===
{{see also|伊犁危機|外西北}}
1862年5月26日，俄軍首次越界進入新疆西北，遭清軍擊退<ref>[http://catalog.digitalarchives.tw/item/00/05/eb/07.html 奏為俄人逼近卡倫及派兵防堵預備援兵由]</ref>，但未能阻止俄國對新疆的野心。6月4日，俄軍以協助鎮壓[[陝甘回變|陝甘回變]]為由，再度進入新疆西北<ref>[http://catalog.digitalarchives.tw/item/00/05/ef/b0.html 奏報哈薩克地方往年夏日應設卡倫地方已被俄國兵隊佔踞]</ref>，並與清軍交火。[[奕訢|恭親王奕訢]]與俄方進行多次談判。1863年3月1日，雙方談判破裂，26日，俄軍大舉入侵，清軍奮勇抵抗，雙方互有勝負。

1864年4月，[[新疆回變|新疆回變]]（1864至1877年間新疆各地叛亂的統稱）爆發。6月，俄軍煽動[[伊犁|伊犁]][[回民|回民]]叛亂，7月9日，俄軍、回民叛軍佔領[[伊犁河|伊犁河谷]]，回民叛軍包圍伊犁城。7月26日，雙方停戰。清廷為避免伊犁陷入回民叛軍手中，使新疆回變加劇，於是同意俄國的割土要求，但不賠款。10月7日，雙方簽訂《[[中俄勘分西北界約記|中俄勘分西北界約記]]》<ref>{{Cite web |url = https://www.npm.gov.tw/exh98/frontier/ch2.html |title = 失落的疆域—清季西北邊界變遷條約輿圖特展＿疆域變遷 - 國立故宮博物院 |accessdate = 2018/10/31  }}</ref>，割讓[[帕米爾高原|帕米爾高原]]以北、[[唐努烏梁海|唐努烏梁海]]以西，共44萬平方公里的土地予俄國。

1864年10月，伊犁[[維吾爾族|維吾爾族]][[塔蘭奇|塔蘭奇]]屯民叛亂，建立“[[伊犁蘇丹國|伊犁蘇丹國]]”。1871年6月，俄國消滅伊犁蘇丹國並佔領伊犁。

1877年，[[湘軍|湘軍]]在[[劉錦棠|劉錦棠]]的率領下，收復除了俄國占領地伊犁之外的新疆全境。<ref>[https://global.udn.com/global_vision/story/8663/1337329 誰的新疆？疆獨是大博弈下的遺緒還是遺毒？]，udn Global，2018/10/27</ref>1881年2月24日，清俄簽訂《[[伊犁條約|伊犁條約]]》，清廷收回伊犁，割讓[[霍爾果斯|霍爾果斯]]河以西，6萬平方公里的土地予俄國。<ref>{{Cite web |url = https://www.npm.gov.tw/exh100/diplomatic/page_ch02.html |title = 百年傳承走出活路 中華民國外交史料特展_歷史之鑑 - 國立故宮博物院 |accessdate = 2018/10/31  }}</ref>
