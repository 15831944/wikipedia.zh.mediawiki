'''俄国二月革命'''（[[俄语|俄语]]：{{lang|ru|Февральская революция}}，按照[[公曆|公曆]]便称为'''三月革命'''）在1917年3月8日（[[儒略历|儒略历]]2月23日；除了特別註明，下列日期一律用格里曆，即[[西历|西历]]显示）于[[俄罗斯|俄罗斯]]发生的[[民主|民主]][[革命|革命]]，是[[1917年俄国革命|俄国革命]]的序幕。其即时结果就是[[沙皇|沙皇]][[尼古拉二世|尼古拉二世]]被迫[[逊位|逊位]]，[[俄罗斯帝国|俄罗斯帝国]]灭亡。革命中沙皇失去权力，而且有大约一千五百至二千人在混乱中被杀或受重伤。继而统治俄国的新政府是[[自由主义|自由主义]]者与[[资本主义|资本主义]]者的政治联盟[[俄国临时政府|俄国临时政府]]，用以谋求政治改革及创造一个以民主选出的[[行政部门|行政部门]]与[[俄国立宪会议|立宪会议]]。

== 革命爆发的原因 ==

革命爆发的原因其中包括[[第一次世界大战|第一次世界大战]]对俄国经济的破坏，以及人民对沙俄政府专制统治的不满。 

当时[[尼古拉二世|尼古拉二世]]自任最高统帅，率领俄军在[[东线|东线]]征战；他就把政府交由其妻[[亚历山德拉·费奥多萝芙娜|亚历山德拉皇后]]与其他官员管理。而皇后寵信[[癲僧|癲僧]][[拉斯普京|拉斯普京]]，拉斯普京令政府充满贪污，使内政混乱。国家局势渐趋紧张，因为俄国之参战在后来造成很多尤其是经济上的问题。 

在[[第一次世界大战|第一次世界大戰]]中，由于俄国加入了[[协约国|协约国]]一方。并且，应[[協約國|协约国]]的多次要求，为了配合西线战事，俄军多次主动发起攻击，迎击[[德意志帝国|德国]]、[[奥匈帝国|奥匈帝国]]以及南方的[[奥斯曼帝国|奥斯曼帝国]]。俄军最初略有小胜，但面对着多线作战的窘迫局面，遭受严重挫败，例如在[[东普鲁士|东普鲁士]]大败于德军。工厂产量下跌、铁路系统欠缺效率以及后勤保障极其不力是俄国数次失利的原因。同时，俄军战术战法陈旧落后，军队领导人将私人恩怨带入战场，导致开战后俄军即遭惨败。后来应时任[[英國|英國]]海军大臣的[[丘吉尔|丘吉尔]]要求，为了配合英军的登陆作战和缓解[[西线|西线]]压力，俄军发动了夏季攻势，遭到重大损失，人员俘虏达100多万，国内统治岌岌可危。

到了战争后期，俄军有超过一百七十万士兵[[陣亡|陣亡]]，五百九十万人受伤。[[逃兵|逃兵]]者众多，将领与军官能力平庸，以致整体士气低落。某些部队出战时，竟然没有充足的弹药与其武器相配。俄国后方则有严重[[饥荒|饥荒]]，物资短缺。基于德军的猛攻与边境封锁，俄国经济最终被拖垮。拥有一些自由派成员的[[杜马|杜马]](指具諮議及立法功能的俄國議會)向沙皇建议组成类似[[1905年俄国革命|1905年俄国革命]]后建立、不久后被他取消的[[君主立宪|君主立宪]]政府，但被他拒绝。

== 圣彼得堡暴動 ==
{{see also|1917年俄國革命}}

1917年2月寒冬、食物短缺、民不聊生，加上经济崩溃，大众起义的诱因业已齐备。圣彼得堡25万驻军，主体是后备役部队。当时后备役管理混乱，一个连高達1350多人，卻只有一个低级军官管理。后来又调进来好几只哥薩克部队。而忠于沙皇的近卫军在前线。[[尼古拉二世|尼古拉二世]]下令从前线调回来25个团的近卫骑兵军到圣彼得堡，把圣彼得堡的乱军拉出去到附近其他城市进行进一步整编。彼得堡市长和驻军司令一听骑兵要来，还得准备场地养马喂马，嫌花钱费事；加上附近城市听说要把乱军拉来自己这里，肯定是花钱还要扰民，所以就各种不配合，政府公文互相写了好几个月推卸责任，调近卫军入首都一事始终无法启动。

3月3日，圣彼得堡的大型工厂[[基洛夫工厂|普提洛夫工厂]]工人宣布罢工。罢工工人被解雇，引致其他工厂的罢工。由于有些示威者要求增加面包供应，部分该厂的工人找到继续罢工的理由。虽然示威者偶尔与政府军队有冲突，但首天没有人死伤。在往后几天，罢工工人雲集在圣彼得堡，局势渐趋紧张。3月8日，示威者为庆祝[[国际妇女节|国际妇女节]]而举办一连串聚会与集会，并渐渐把活动政治化。参与者首次使用在俄国较为令人敏感的[[口号|口号]]，例如“反对战争！”与“结束[[专政|专政]]！”等。二月革命终于爆发。这次，警民冲突，引致双方互有死伤。示威者准备好武器，抢掠警方的总部。

3月10日，沙皇政府派出一大队士兵掃蕩该城。士兵最初拥护政府，杀害了不少示威者。由于俄罗斯工厂主的大部，工人阶层的相当部分都是旧礼仪派教徒出身，所以同样信奉旧教的哥薩克军队除了1905年那次之外，几乎没有参与过镇压工人游行的事情，当时有几个哥薩克大首领特意请求尼古拉二世不要把哥薩克部队投入镇压工人游行。但后来哥薩克军队开始袭击警察导致秩序大乱，加上有九万后备役驻军哗变攻击监狱和政府机构。这个时候，沙皇深感不安，于是解散了杜马（是指具諮議及立法功能的俄國議會)，并下令选出一个临时委员会。

== 沙皇退位 ==
沙皇得悉彼德格勒的示威，并决定乘火车到彼德格勒维持大局，以期改善情况。然而，他返国的真正原因不只於此，主要是因為他的子女們染上[[麻疹|麻疹]]，包括他的唯一兒子[[阿列克谢·尼古拉耶维奇·罗曼诺夫|阿列克谢]][[皇太子|皇太子]]，故此沙皇返回该城市附近的[[普希金市|普希金市]][[沙皇村|沙皇村]]陪伴[[太子|太子]]與[[公主|公主]]們。

不过，当他到达目的地之时，迎接他的[[将领|将领]]都异口同声劝他[[退位|退位]]。

3月15日 （[[儒略历|儒略历]]3月2日） ，沙皇宣布与他的儿子退位，并提名其弟[[米哈伊尔·亚历山德罗维奇_(亚历山大三世之子)|米哈伊尔大公]]继承[[帝位|帝位]]。但是，由于彼德格勒内的所有军队站在示威者的一方，大公在3月17日拒绝[[登基|登基]]，[[俄罗斯帝国|俄罗斯帝国]]就此灭亡。

從1917年2月28日的二月革命推翻沙皇後，俄羅斯的政體成為[[二元制|二元制]]（dyarchy,dvoevlastic） 。
