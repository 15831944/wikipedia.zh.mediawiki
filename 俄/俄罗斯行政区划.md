'''俄罗斯行政区划'''可以作以下幾種分類：

== 联邦主体 ==
{{main|俄罗斯联邦主体}}

俄羅斯本身是一個[[聯邦|聯邦]]國家，共由85個不同大小的'''联邦主体'''（[[俄語|俄語]]：{{lang|ru|субъекты}}）組成。這些联邦主体雖然大小不一，但都有著相同的聯邦權利，同樣有著兩位代表參與[[俄羅斯聯邦委员会|俄羅斯聯邦委员会]]（俗稱“上議院”）不過，雖然他們享有的聯邦權利一樣，每個联邦主体享有的自治程度仍然不一樣。

== 聯邦管區 ==
{{Main|俄羅斯聯邦管區}}

俄羅斯全國被劃分為8個[[聯邦管區|聯邦管區]]（[[複數_(語言學)|复数]]：{{lang|ru|федеральные округа}}，[[單數_(語言學)|單數]]：{{lang|ru|федеральный округ}}），其中[[南部联邦管区|南部联邦管区]]中的[[克里米亞共和國|克里米亞共和國]]和[[塞瓦斯托波尔市|塞瓦斯托波尔市]]未获得国际公认為俄羅斯聯邦領土。联邦管区实际并不是正式的[[一级行政区|一级行政区]]，类似于[[监察|行政监察区]]。每一個聯邦管區都會由一位[[俄羅斯總統|總統]]委派的全權代表管治。

== 联邦直辖市==
{{main|俄罗斯联邦直辖市}}
'''联邦直辖市'''是[[俄罗斯|俄罗斯]]的[[俄罗斯联邦主体|联邦主体]]的一种。俄罗斯联邦到2014年为止共分为85个联邦主体，其中三个是联邦[[直辖市|直辖市]]。但三个直辖市中的[[塞瓦斯托波爾|塞瓦斯托波爾]]市未被国际普遍承认为俄罗斯领土。
== 經濟地區 ==
{{main|俄罗斯经济地区}}

为了监管和统计方便，联邦成员被分成12个“经济地区”（{{lang|ru|экономические районы}}）。

== 下級行政區 ==
每一個聯邦成員可以分為各個行政區。這些二級行政區可以屬於以下其中一種：
* 县级（{{lang|ru|районы}}）。
* 市镇（{{lang|ru|города}}，单数：{{lang|ru|город}}）和“市镇式”乡（{{lang|ru|посёлки городского типа}}，单数：{{lang|ru|посёлок городского типа}}）：在联邦成员的管辖下。
* 自治區（{{lang|ru|автономные округа}}，單數：{{lang|ru|автономный округ}}）：在联邦成员的管辖下（虽然这些地区是下级行政区，但也有联邦成员的地位）。

第三級行政區包括：
* [[乡村委员会|乡村委员会]]（{{lang|ru|сельсоветы}}）。
* [[乡镇|乡镇]]和「城/乡居住区」在区的管辖下。
* 城/镇区。
* 城市居住区。
