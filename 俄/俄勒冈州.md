'''俄勒冈州'''（{{lang-en|State of Oregon}}）是[[美国|美国]]的一个[[美国州份|州]]，位于美国[[美國西北部|西北]]的[[美國太平洋沿岸地區|太平洋沿岸]]，西临[[太平洋|太平洋]]、北接[[华盛顿州|华盛顿州]]、东面是[[爱德荷州|爱德荷州]]、南面是[[加利福尼亚州|加利福尼亚州]]和[[内华达州|内华达州]]。2012年人口3,899,353。州别称：'''海狸州'''，邮编代号'''OR'''。本州的[[俄勒冈州行政区划|行政區劃]]，共管轄36個[[郡|郡]]。

== 歷史 ==
俄勒岡州早在一萬五千年前已有人類居住的紀錄，人口主要集中在西面近海的[[河口|河口]][[平原|平原]]。十六世紀時有大量[[印第安人|印第安人]]居住在俄勒岡州。

十八世紀後期[[歐洲|歐洲]]人開始前來俄勒岡州探險，自此俄勒岡州就湧現了不少公司前來開發。1850年後，俄勒岡州人口不斷上升，並有鐵路興建把俄勒岡州的糧食運到東岸。到二十世紀，隨著水利工程的完成，[[伐木|伐木]]業的興起，使工商業開始在這個州份興起。

== 法律與政府 ==
俄勒岡州跟其他美國的州份一樣，都是行政、立法和司法三權分立，州政府沒有國防和外交等層級實權。

[[美國共和黨|共和黨]]和[[美國民主黨|民主黨]]在這個州份的政府並沒有絕對優勢。但自[[1988年美國總統選舉|1988年美國總統選舉]]起，每屆總統大選，這個州份都是由民主黨人勝出。

== 地理 ==
州名[[印第安语|印第安语]]意为“美丽之水”（指[[哥伦比亚河|哥伦比亚河]]）。地形起伏较大，西部有[[喀斯喀特山脉|喀斯喀特山脉]]和海岸山脉，东半部全为[[高原|高原]]。西部沿岸多[[雨|雨]]，东部高原少雨。[[森林|森林]]占全州面积一半，与[[华盛顿州|华盛顿州]]同为林业最盛的州。木材、纸浆、家具制造业很发达。[[威拉米特河|威拉米特河]]流域人口密集，州人口70％集中在这里，农业发达，栽培[[小麦|小麦]]、[[燕麦|燕麦]]、[[马铃薯|马铃薯]]及各种[[水果|水果]]。东部高原主要为[[小麦|小麦]]、肉牛的农牧业区。沿海以[[鲑鱼|鲑鱼]]为主的渔业很盛。工业仅以果品罐头业为重要。重要自然风景区有[[火山口湖国家公园|火山口湖国家公园]]和[[富德火山|富德火山]]等。

== 經濟 ==
俄勒冈是美國的一個免稅州，居民年收入约8萬美元，体育用品公司[[Nike|Nike]]、户外运动品牌[[美國哥倫比亞運動服裝公司|哥倫比亞]]和科技公司Tektronix的全球总部位于该州。

== 人口 ==
2012年人口普查，俄勒岡州有3,899,353人。

目前俄勒岡州的族群比例可分為：
* 83.5%－白人
* 1.6%－非裔人 
* 8.0%－[[拉美裔人|拉美裔人]]
* 2.9%－亞裔人
* 1.3%－[[印第安人|印地安原住民]]
* 3.2%－混合的族群

俄勒岡州的前五大居民祖先為：[[德國|德國]]人（20.5%）、[[英國|英國]]人（13.2%）、[[愛爾蘭|愛爾蘭]]人（11.9%）、[[墨西哥|墨西哥]]人（6.3%）。

該州有6.5%的人口在5歲以下，24.7%的人口在18歲以下，以及12.8%的人口在65歲以上。女性約占全人口比例50.4%。
