'''俄亥俄州立大学'''（{{lang-en|The Ohio State University}}，縮寫：Ohio State 或 OSU）创建于1870年，是一所坐落于美国[[俄亥俄州|俄亥俄州]]首府[[哥伦布_(俄亥俄州)|哥伦布市]]北部的公立研究型大学。它是北美顶尖的学术联盟[[美國大學協會|美國大學協會]]成员之一以及[[十大联盟|十大联盟]]成员。它也是美国五大湖地区顶尖的公立大学之一，被誉为“公立常春藤”（Public Ivy） 。

以占地面积计算，大学是北美面积最大的大学之一。开设的专业几乎涵盖了所有的学术领域，尤以经济学、商学，工程学，物理学等学科于世界名列前茅。根据[[国家科学基金会|国家科学基金会]]的2007年报告指出，2006年的研究支出为6亿5千2百万美元，在全美国公立大学中，排名第七<ref>[http://www.nsf.gov/statistics/nsf08300/pdf/tab32.pdf National Science Foundation 2007 report - 公立大学排名] {{webarchive|url=https://web.archive.org/web/20150629102457/http://www.nsf.gov/statistics/nsf08300/pdf/tab32.pdf |date=2015-06-29 }}</ref>；在所有公私立大学中，排名第十一<ref>[http://www.nsf.gov/statistics/nsf08300/pdf/tab27.pdf National Science Foundation 2007 report - 公私立大学排名] {{webarchive|url=https://web.archive.org/web/20080910053420/http://www.nsf.gov/statistics/nsf08300/pdf/tab27.pdf |date=2008-09-10 }}</ref>，私人企业赞助研究，全美排名第二<ref>[http://www.nsf.gov/statistics/nsf08300/pdf/tab30.pdf National Science Foundation 2007 report - 私人企业赞助研究] {{webarchive|url=https://web.archive.org/web/20150629102449/http://www.nsf.gov/statistics/nsf08300/pdf/tab30.pdf |date=2015-06-29 }}</ref>。商学院是[[國際商管學院促進協會|国际商学院促进协会]]在1916年的创始者之一。

大学百余年来培养了众多杰出校友，至少包括4位[[诺贝尔奖|诺贝尔奖]]得主，8位[[普利策奖|普利策奖]]得主和1位[[沃尔夫奖|沃尔夫奖]]得主。许多对人类科技进步作出重大贡献的人物，包括大发明家查尔斯·凯特林，发现[[聚四氟乙烯|聚四氟乙烯]]（“特氟龙”）的罗伊·普朗克特，发明霍夫曼代码的[[大衛·霍夫曼|大卫·霍夫曼]]，维基百科的联合发起者和创始人[[拉里·桑格|拉里·桑格]]；以及许多改写历史的人物，从柏林奥运会上的“黑色闪电”[[杰西·欧文斯|杰西·欧文斯]]，到火攻东京的[[柯蒂斯·李梅|柯蒂斯·李梅]]将军，再到冷战后改写政治格局的“民主导师”[[吉恩·夏普|吉恩·夏普]]等，全都毕业于此校。

学术排名方面，在2016年世界四大主流大学排名机构公布的排名中，QS世界排名88，泰晤士高等教育世界排名72，ARWU世界排名79，US News世界排名46。大学声誉方面，在2016年泰晤士高等教育世界大学声誉排行榜中，被列于世界第51至60位。但值得注意的是，由于是公立学校，该校存在严重的官僚主义，导致很多日常维护和信息传达十分缓慢。比如，在2018年，该校主图书馆Thompson图书馆的打印机无法使用整整一学期之久。此外,为了减少经费，该校的商学院依靠部分大三大四的学生在recitation教课。在该校的费舍尔商学院，很多学生反应大三大四的学生在教课方面的能力很有限。曾有学生反映有一门BM2320的统计课存在教授每节课都讲不完的经历。此外，该校经常出现信息不对称的情况。比如有学生表示曾亲身经历从学校得到不准确的信息，导致自己不得不额外多修学分的经历。

== 校园 ==
目前大学在哥伦布的主校园是美国最大的校园之一，在校学生人数50,995名。俄亥俄州立大学主校园占地1764英亩（7.14平方公里），位於哥伦布市中心以北4公里。

威廉汤逊图书馆座落於校园中央，超过二百幢各式教学大楼座落於北面及南面。校园建築物拥有不同风格，包括传统的、新式的、及後现代式的设计。其中，威廉汤逊主图书馆建於1913年采用意大利文艺复兴的设计。在2006年，威廉汤逊主图书馆开展了价值一亿美元的重修计划，目标是打造成一个世界级图书馆揉合21世纪风格并保留意大利文艺复兴的建築。

俄亥俄州立大学图书馆由主馆及21间学科图书馆组成，拥有藏书超过六百万册，北美排名17。 <ref>[http://library.osu.edu/about/departments/assessment/data-portal/arl-ranking/ ARL libraries ranking] {{webarchive|url=https://web.archive.org/web/20140504153538/http://library.osu.edu/about/departments/assessment/data-portal/arl-ranking/ |date=2014-05-04 }}</ref>。
18街图书馆坐落于校园中央偏北，原先名字为科学与工程图书馆。此图书馆除了节日以外全年24小时无休开放，并且一层有咖啡店提供必要的餐饮服务<ref>[https://library.osu.edu/about/locations/18th-avenue-library/ {{Wayback|url=https://library.osu.edu/about/locations/18th-avenue-library/ |date=20170203141907 }}</ref>。
俄亥俄体育馆（Ohio Stadium）是一座露天体育场馆，别名"马蹄铁"（The Horseshoe），因外型酷似而得名。其观众容量可超过十万，位列全美第三，也是大学美式足球队竞赛的主场馆，位於Olentangy River旁边<ref>{{cite web|url=https://en.wikipedia.org/w/index.php?title=Ohio_Stadium&oldid=865604543|title=Ohio Stadium|date=2018-10-24|publisher=|accessdate=2018-11-14|via=Wikipedia}}</ref>。其他的主要体育设施，包括综合体育大楼、溜冰场、田径训练习等，都在校园的西北面。

大学医学院、护理学院、药剂学院等都位於主校园南端。大学医学院更是詹姆斯癌症医院的一个组成部份。

传统的教学大楼都座落在主校园区，即在Olentangy River的东面。农务或相関科目及新建的运动训练设施，则在Olentangy River西面的新校区。主校区与西面校区，有大学巴士穿梭来回连络。

== 历史 ==
1870年俄亥俄农业和机械学院成立，1873年开始接受学生。这所学校最初位于哥伦布市外，其主要专业是农业和机械。俄亥俄州后来决定扩大教育范围，从1878年开始定名为俄亥俄州立大学。运动方面，俄亥俄州立大学於1912年加入[[十大联盟|十大联盟]]。学术方面，俄亥俄州立大学於1916年加入[[美国大学协会|美国大学协会]]。

直至1998年，俄亥俄州立大学一直运行{{Link-en|硕耳射电望远镜|Ohio State University Radio Observatory}}，世界上最大和使用时间最长的用于搜索外星智慧的[[射电望远镜|射电望远镜]]，该望远镜于1977年8月16日发现著名的[[Wow!訊號|WOW!信号]]。

俄亥俄州立大学参与建设[[大双筒望远镜|大双筒望远镜]]，并获得其12.5%的观测时间。

== 体育、俱乐部和传统 ==
俄亥俄州立大学参加与其它大学比赛的运动队称为[[七叶树队|七叶树队]]，俄亥俄州立大学的运动队参加[[全美大学生体育联合会|全美大学生体育联合会]]和[[十大联盟|十大联盟]]的各类赛事。学校的颜色是红色和银灰色。

俄亥俄州立大学有极强的体育传统，也是美國中西部名校聯盟[[十大联盟|十大联盟]]<ref>[[:十大联盟|The Big Ten Conference]]</ref>的会员。十大联盟是由十四间中西部的大型公立大学所组成（除西北大学乃私立大学外）。俄亥俄州立大学有三十七支运动队伍在十大联盟竞赛，其中美式足球队尤其著名。俄亥俄州立大学美式足球队于2015年获得全美大学联赛冠军，这是他们继1942年、1954年、1957年1961年、1968年、1970年和2002年后第八次获冠军。2009年ESPN选举历史上最佳大学美式足球队伍，俄亥俄州立大学位列第三名<ref>{{cite web|url=http://www.espn.com/college-football/news/story?id=3849468|title=Prestige Rankings: Nos. 1-5|date=2009-01-23|website=ESPN.com|accessdate=2018-11-14}}</ref>，仅次於[[俄克拉荷马大学|俄克拉荷马大学]]及[[南加州大学|南加州大学]]。

俄亥俄州立大学与密歇根大学的美式足球对决，是大学运动界的一项大事。对决每一年在常规球季的最後周末举行，梅花间竹地在哥伦布市或安娜堡举行，已经有超过一百一十八年历史。到2010年为止，这对决曾经22次决定这两所大学，谁为十大联盟冠军；另外27次影响十大联盟冠军赛果。

俄亥俄州立大学也是全美曾赢下三项主要男子球类运动（棒球、篮球、美式足球）的四间大学之一（其馀三间为[[密歇根大学|密歇根大学]]、[[史丹福大学|史丹福大学]]、[[柏克萊加州大學|柏克萊加州大學]]）。卓越运动员包括1936年奥运田径金牌得主[[杰西·欧文斯|杰西·欧文斯]]。美式足球队员曾七次拿下[[海斯曼杯|海斯曼杯]]，包括[http://en.wikipedia.org/wiki/Archie_Griffin Archie Griffin]（1974 & 1975）, [http://en.wikipedia.org/wiki/Eddie_George Eddie George]（1995）及[http://en.wikipedia.org/wiki/Troy_Smith Troy Smith]（2006）。

俄亥俄州立大学的铜管乐队也是全美知名，常在美式足球比赛休场期间，上演多幕超难度的表演，疯魔观众，其拥有255名乐手，是世界上最大的铜管乐队。

俄亥俄州立大学的电动汽车“[http://www.buckeyebullet.com/index.html 橡树子弹（Venturi Buckeye Bullet）]”于2010年在[[犹他州|犹他州]]以307.7英里每小时的速度创电动汽车世界纪录。

俄亥俄州立大学有一个电视台和两个广播电台，从2003年开始其电视台开始播送[[高清晰度电视|高清晰度电视]]，此外它还有一个学生组织的互联网电台和一个网络电视台。学校的校报名为《明灯》。
