'''俄罗斯总理'''，正式名称为'''俄罗斯联邦政府主席'''（{{lang-ru|'''Председатель Правительства'''}}，中文也称'''俄罗斯联邦政府总理'''<ref>{{cite web |title=俄驻华大使接受解放日报专访 |url=http://www.russia.org.cn/cn/news/2606/ |website=俄罗斯联邦驻华大使馆}}</ref>或'''俄罗斯联邦总理'''<ref>{{cite web |title=“外交工作者日”：俄罗斯总统普京、总理梅德韦杰夫、外长拉夫罗夫分别向俄外交部官员和退休老前辈们致贺词 |url=http://www.russia.org.cn/cn/news/7019/ |website=俄罗斯联邦驻华大使馆}}</ref>）是[[俄罗斯联邦政府|俄罗斯联邦政府]]的[[政府首脑|首脑]]，[[俄罗斯|俄罗斯]]国家的行政权力被[[俄罗斯总统|俄罗斯总统]]和[[總理|總理]]所共同分享，但职权上较总统小。

在[[俄罗斯帝国|帝俄时代]]，[[俄罗斯帝国|俄罗斯帝国]]大臣会议主席，也可称为[[首相|首相]]，由[[沙皇|沙皇]]任命，不能分享权力。

在[[苏联|苏联]]时代，政府首脑是人民委员会主席（1946年前）和[[苏联部长会议主席|部长会议主席]]（1946年后）。

今日的俄罗斯总理由俄罗斯总统提名，[[俄罗斯国家杜马|国家杜马]]通过后任命。如果议会否决总理提名，总统必须提名其他人选。如果议会连续否决三名人选，总统就必须解散议会，重新大选。总统有权解除总理职务。总统换届后，总理必须率领[[内阁总辞|内阁总辞]]，新任总统重新提名总理人选。在总统死亡或者辞职、解职时，总理作为第一顺位继任人担任[[临时总统|临时总统]]职务。
