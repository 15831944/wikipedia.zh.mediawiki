《'''俄羅斯娃娃：派對迴旋'''》（{{lang-en|'''''Russian Doll'''''}}）是一部[[美國|美國]][[喜劇片|喜劇]][[網路劇集|網路劇集]]，由[[娜塔莎·雷昂|娜塔莎·雷昂]]、[[艾米·波勒|艾米·波勒]]和{{le|萊絲利·海德蘭德|Leslye Headland}}開創。第一季於2019年2月1日在線上[[串流媒體|串流媒體]]平台[[網飛|網飛]]首播，共8集。

== 概要 ==
紐約一名叫娜迪亞的女子，在36歲的生日當晚死於非命......接著發現自己又回到了朋友幫她舉辦的派對現場。這場似乎永遠無法逃離的派對，重複死去的她試圖解開這個謎團<ref name="Premiere">{{cite web |last1=Pedersen |first1=Erik |title=‘Russian Doll’: Premiere Date & First-Look Photos For Netflix’s Natasha Lyonne Comedy |url=https://deadline.com/2018/12/russian-doll-premiere-date-first-look-photos-netflix-natasha-lyonne-amy-poehler-leslye-headlund-1202520011/ |website=[[Deadline_Hollywood|Deadline Hollywood]] |date={{Start date|2018|12|14}} |accessdate={{Start date|2019|1|6}}|language=en-US}}</ref><ref name="Trailer">{{cite web |last=Ramos |first=Dino-Ray |title=‘Russian Doll’ Trailer: Natasha Lyonne Relives Death On An Endless Loop In Netflix Comedy |url=https://deadline.com/2019/01/russian-doll-trailer-netflix-natasha-lyonne-amy-poehler-leslye-headland-1202531787/ |website=[[Deadline_Hollywood|Deadline Hollywood]] |accessdate={{df|January 9, 2019}} |date={{df|January 9, 2019}}|language=en-US}}</ref>。

== 演員與角色 ==
=== 主要演員 ===
*; [[娜塔莎·雷昂|娜塔莎·雷昂]] 飾演 娜迪亞（Nadia Vulvokov）
: 軟體工程師，發現她陷入無限輪迴的時間循環，每當她意外死去，時間會回朔到她36歲的生日派對上，一切重新開始。
*; {{le|格里塔·李|Greta Lee}} 飾演 麥克辛（Maxine）
: 娜迪亞的朋友，為娜迪亞舉辦36歲生日派對。
*; {{le|尤爾·瓦茲奎茲|Yul Vazquez}}　飾演　約翰（John Reyes）
: 房產仲介，娜迪亞的前男友，正在辦理離婚手續，與準前妻育有一女。
*; {{le|伊麗莎白·阿什利|Elizabeth Ashley}} 飾演 露絲（Ruth Brenner）
: 娜迪亞和她母親的治療師，和娜迪亞親如家人。
*; {{le|查理·巴奈特|Charlie Barnett (actor)}} 飾演 艾倫（Alan Zaveri）
: 同樣陷入時間輪迴的男子。

== 製作 ==
[[File:Russian_Doll_poster.jpg|150px]]領銜出演女主角娜迪亞]]
=== 開發 ===
2017年9月20日，[[網飛|網飛]]宣佈預訂本劇集第一季，共8集，由[[娜塔莎·雷昂|娜塔莎·雷昂]]、[[艾米·波勒|艾米·波勒]]和{{le|萊絲利·海德蘭德|Leslye Headland}}開創並擔任執行製片人。萊絲利·海德蘭德執筆第1集劇本，第一季的餘下7集劇本由她與主演娜塔莎·雷昂共同完成。製作公司包括[[環球電視公司|環球電視公司]]、{{le|Paper Kite Productions}}、{{le|Jax Media}}和{{le|3 Arts Entertainment}}<ref name="SeriesOrder">{{cite web |last1=Andreeva |first1=Nellie |title=Netflix Greenlights Comedy From Natasha Lyonne, Leslye Headland & Amy Poehler |url=http://deadline.com/2017/09/netflix-comedy-natasha-lyonne-leslye-headland-amy-poehler-1202173835/ |website=Deadline Hollywood |date={{Start date|2017|9|20}} |accessdate={{Start date|2019|1|6}}|language=en-US}}</ref><ref name="HR1">{{cite web |last1=Goldberg |first1=Lesley |title=Natasha Lyonne to Star in Netflix Comedy From Amy Poehler, Leslye Headland |url=https://www.hollywoodreporter.com/live-feed/natasha-lyonne-star-netflix-comedy-amy-poehler-leslye-headland-1041520 |website=[[The_Hollywood_Reporter|The Hollywood Reporter]] |date={{Start date|2017|9|20}} |accessdate={{Start date|2019|1|6}}|language=en-US}}</ref><ref>{{cite web |last1=MacDonald |first1=Lindsay |title=Amy Poehler Heads to Netflix With a Natasha Lyonne Comedy |url=http://www.tvguide.com/news/netflix-snags-amy-poehler-comedy-series-starring-natasha-lyonne/|website=[[TV_Guide|TV Guide]] |date={{Start date|2017|9|20}} |accessdate={{Start date|2019|1|6}}|language=en-US}}</ref><ref>{{cite web |last1=Wright |first1=Megh |title=Netflix Orders a Comedy Series from Amy Poehler, Natasha Lyonne, and Leslye Headland |url=http://splitsider.com/2017/09/netflix-orders-a-comedy-series-from-amy-poehler-natasha-lyonne-and-leslye-headland/ |website=Splitsider |date={{Start date|2017|9|20}} |accessdate={{Start date|2019|1|6}}|language=en-US}}</ref>。2018年12月14日，網飛公佈劇集第一季上線日期為2019年2月1日<ref name="Premiere"/>。

=== 選角 ===
隨著劇集被預訂，[[娜塔莎·雷昂|娜塔莎·雷昂]]確定飾演劇集女主角<ref name="SeriesOrder"/>。2018年12月，{{le|格里塔·李|Greta Lee}}、{{le|尤爾·瓦茲奎茲|Yul Vazquez}}、{{le|伊麗莎白·阿什利|Elizabeth Ashley}}和{{le|查理·巴奈特|Charlie Barnett (actor)}}確定飾演主要角色，[[克蘿伊·塞凡妮|克蘿伊·塞凡妮]]、{{le|達絲莎·坡蘭科|Dascha Polanco}}、[[布蘭登·薩克斯頓三世|布蘭登·薩克斯頓三世]]、麗貝卡·亨德森、{{le|傑里米·洛威爾·鮑伯|Jeremy Bobb}}、{{le|里特什·拉詹|Ritesh Rajan}}和{{le|喬斯林·畢希|Jocelyn Bioh}}確定客串出演。
