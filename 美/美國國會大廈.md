'''美國國會大廈'''（{{lang-en|United States Capitol}}），別稱'''國會山莊'''，是[[美國國會|美國國會]]所在地，位於[[美國|美國]][[首都|首都]][[华盛顿哥伦比亚特区|華盛頓特區]]的[[國會山|國會山]]，坐落在華盛頓特區[[國家廣場|國家廣場]]東端。在地理上，美國國會大廈位于華盛頓特區偏東處，但常被看作華盛頓特區及美國的政治中心。華盛頓特區建筑物地址的[[方位|東西南北]]，都是以國會大廈作為基準。

== 建築 ==
[[File:US_Capitol_east_side.JPG|250px]]
大厦最初由{{Link-en|威廉·桑顿|William Thornton}}博士设计，该建筑以一个圆形大厅上的圆顶以及它的两翼作为标记。每一翼作为一个议会的所在地，北翼是[[美國参议院|美國参议院]]，而南翼則是[[美國众议院|美國众议院]]。它是一个典型的新古典式建筑。国会大厦是一幢全长233米的3层建筑，以白色大理石为主料，中央顶楼上建有出镜率极高的3层大圆顶，圆顶之上立有一尊6米高的自由女神青铜雕像。

由于[[电视新闻|电视新闻]]报道华府作出决定时多出现的是国会大厦，加之经常配合有“[[白宫|白宫]]方面宣布”的言辞，且其风格巍峨挺拔，导致不了解的人在很长一段时间内将国会大厦认为是白宫——[[美國總統|美國總統]]官邸暨辦公室。

公众可免费参观国会大厦，但每天的名额有限；也可旁听参议院或众议院议事，但須經申請。

== 沿革 ==
[[File:Capitol1800.jpg|right]]
現在的國會大廈實際上是美國的第四個國會大廈。在1783年到1784年間，美國國會的開會地點是位於[[馬里蘭州|馬里蘭州]][[安那波利斯|安那波利斯]]的[[馬里蘭州議會大廈|馬里蘭州議會大廈]]。1789年到1790年期間，使用[[紐約|紐約]]的[[聯邦國家紀念堂|聯邦國家紀念堂]]作為開會地點。1790年到1800年期間，開會地點是[[費城|費城]]的[[美國獨立紀念館|美國獨立紀念館]][[國會廳|國會廳]]。

現在的國會大廈始建於1793年9月18日。首任總統[[喬治·華盛頓|喬治·華盛頓]]為它安放基石，但是確切的安放地點現在卻已經無法考證。其早期的建造由[[本杰明·亨利·拉特孟彼|本杰明·亨利·拉特孟彼]]和[[查爾斯·布爾芬奇|查爾斯·布爾芬奇]]等人監制。當政府于1800年將其临时首部从[[费城|费城]]迁到[[华盛顿特区|华盛顿特区]]时，等待他们的国会大厦仍是尚未完成的砖石结构。上院一側在1800年完工，下院一側在1811年完工。在1812年戰爭中，建築曾被破壞，後修復。後來隨著美國州數與議員數的增加，國會大廈也大幅擴建，并且建設位於中央的巨型穹頂。

== 政治意義 ==
[[File:Apotheosis_of_George_Washington.jpg|250px]]》]]
国会大厦是[[美国政治|美国政治]]的象征，1800年以来就是国会会议的召开地。国会议员聚集在此制定法律，[[美國總統|美國總統]]亦在此宣誓就职、并且宣讲每年的[[国情咨文|国情咨文]]。
