'''美國國務卿'''（{{lang|en|United States Secretary of State}}），直译为'''国务部长'''，是[[美國國務院|美國國務院]]的[[首長|首長]]，同時為[[美國內閣|美國內閣]]成员，主管[[美国外交|美国外交]]事务，代表[[美国总统|美国总统]]執行對外政策，相当于[[美国外交部长|美国外交部长]]，其地位要比其他内阁部长高，是所有內閣成員中的首席。

== 政治地位 ==
美国国务卿是[[美国聯邦政府|美国聯邦政府]]中繼[[美国副总统|美国副总统]]後的第二号内阁成员，如果总统无法有效行使总统权力能力或者意外死亡，按照美国总统的[[美国总统继任顺序|继承顺序]]，国务卿排在第四位，即排在[[美国副总统|美国副总统]]（兼任[[美国参议院|美国参议院]]议长）、[[美国众议院议长|美国众议院议长]]和[[参议院临时议长|参议院临时议长]]之后。

由于副總統根據憲法只是充當總統的備位，因而位高而權不重，即使其影響力視乎總統與副總統的關係，例如近年美國副總統的政治影響力已大大增強。在[[美國聯邦政府|美國聯邦政府]]權力當中，國務卿的政治權力和影響力，比副總統還要大，故國務卿的權力及影響力在[[美國聯邦政府|聯邦政府]]是僅次於總統。正因如此，建國以來，總統任命的國務卿一如副總統一樣，被視作與總統的政治勢力平衡，国务卿一般是國內具重要影響力的[[政治人物|政治人物]]，可與總統，例如[[美國開國元勳|美國開國元勳]][[湯瑪斯·傑佛遜|湯瑪斯·傑佛遜]]便曾任[[喬治·華盛頓|喬治·華盛頓]]的第一任國務卿，[[亨利·基辛格|亨利·基辛格]]曾任[[理查德·尼克松|理查德·尼克松]]的國務卿，[[希拉里·克林頓|希拉里·克林頓]]曾任[[巴拉克·奧巴馬|巴拉克·奧巴馬]]的第一任國務卿。

== 国务卿的職責 ==
# 保存及使用[[美國國璽|美國國璽]]
# 與外國代表處理及協調國際事務
# 作為美國總統對國際事務的首席顧問
# 協調、指揮及監督美國政府整體海外事務（部分軍事行動除外）

== 代理国务卿 ==
如果国务卿在任职期间辞职，那么按美国法律规定，[[美国副国务卿|常务副国务卿]]就将暂行代理国务卿职务，直到总统提名新的國務卿，並經参议院行使同意權，通过新的国务卿任命为止。
