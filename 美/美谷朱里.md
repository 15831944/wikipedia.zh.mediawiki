== 人物 ==
*興趣是[[跳舞|跳舞]]<ref name="スカパー"/>，截至2018年有長達13年的跳舞資歷。身體的背部轉動甚至側面轉動都可以，腰部也很柔軟。在AV表現[[女上位|騎乘位]]的排名之中，這樣的律動常常被稱讚為「小刻み、高速、ダンサブル」<ref>{{Cite news |url=http://dxbeppin-r.com/archives/56612 |title=【ボディラインと騎乗位のエロさが武器！　美谷朱里インタビュー】「相手が喜ぶことは絶対してあげてましたごっくんを求められたら喜んで飲みます」後編 |work={{link-ja|デラべっぴんR|デラべっぴん}} |date=2018-03-05 |accessdate=2018-11-30 |language=ja}}</ref>。

*她也演出許多妹系作品，當中本人[[第一人稱|第一人稱]]的名字是「自己」，這是讓人令人耳目一新的個性<ref>{{Cite Web |url=http://dxbeppin-r.com/archives/56612 |title=録画ライブ あかりんキャス♡ #510066502 |publisher=デラべっぴんR |date=2018-11-30 |accessdate=2018-12-04 |language=ja}}</ref>。

*她從[[中學|中學]]時期開始上網看[[色情片|色情片]]。最初為知名女優[[上原亞衣|上原亞衣]]的粉絲，也因此認識了更多「[[惠比壽麝香葡萄|惠比壽★麝香葡萄]]」的成員與其他作品。

== 簡歷 ==
2017年7月，在日本[[AV片商|AV片商]]「CANDY」（キャンディ）以{{link-ja|女大學生|女子大生}}身分[[AV|AV]][[出道|出道]]<ref>{{cite web|url=http://www.playno1.com/article-23575-1.html#.WwaSb0iFOUk |title=性感帶在舌頭！美谷朱里下海大戰吉村卓 |publisher=AVNo.1-PLAYNO.1 |date=2017-06-09}}</ref>。
