'''美国陆军预备军官训练团第3旅'''（{{lang-en|'''3rd Reserve Officers' Training Corps Brigade'''}}），昵称'''黑鹰旅'''，总部位于[[伊利诺伊州|伊利诺伊州]]的{{Link-en|大湖海军基地|Naval Station Great Lakes}}，负责管辖[[北达科他|北达科他]]、[[南达科他|南达科他]]、[[内布拉斯加|内布拉斯加]]、[[堪萨斯|堪萨斯]]、[[明尼苏达|明尼苏达]]、[[艾奥瓦|艾奥瓦]]、[[威斯康星|威斯康星]]、[[伊利诺伊|伊利诺伊]]、[[密苏里州|密苏里州]]和[[密歇根上半岛|密歇根上半岛]]各大学的陆军预备军官训练团<ref>{{cite web |url=http://www.cadetcommand.army.mil/brigades.aspx |title=Brigades |accessdate=2017-08-01 |date= |publisher=U.S. Army Cadet Command |work=U.S. Army Cadet Command|deadurl=no|archivedate=|archiveurl=}}</ref>。
