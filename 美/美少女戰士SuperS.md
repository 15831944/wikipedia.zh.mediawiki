《'''美少女戰士SuperS'''》（{{lang|ja|美少女戦士 セーラームーンSuperS}}）是1995年3月4日到1996年3月2日播放的[[日本動畫|日本動畫]]作品。

== 概要 ==
本作品是接續第三作《[[美少女戰士S|美少女戰士S]]》的續作，是[[美少女戰士|美少女戰士]]系列的第四部作品。特色是以[[豆釘兔|豆釘兔]]為主角，描述了她與守護夢想世界的獨角馬之間的一段羅曼史。

== 故事背景 ==
生活在馬戲團世界的妮赫蕾妮亞女王从小就很想拥有自己的朋友和爱人，但一直都没有人关心她，只是因為她的美貌，所以才使宮內所有人寵愛她。直到有一日，她想在魔鏡中知道自己的將來是如何，卻看見自己一副可怕的樣子，在魔鏡的唆使下，她為了得到永恆的美貌而不斷吃掉他人的美麗夢想，後來被希蕾妮蒂女王封印在鏡世界之中。

妮赫蕾妮亞女王發現有一顆以美麗夢想作原動力而發放出强大能量的聖石「黃金水晶」，故此她要求看守黄金水晶的守衛艾利歐斯成為她的手下，使她得到黃金水晶後能夠從鏡中解放出來，但遭到拒絕，於是她派出死亡月亮马戏团入侵艾利歐斯所守護的夢想世界「艾利西恩」，並將他囚禁。他的靈魂卻變成一匹獨角馬並帶同黃金水晶一起逃走，躲進某人的美麗夢想之中，死亡月亮马戏团就派出亡靈，去尋找匿藏在美麗夢想中的獨角馬。

== 故事簡介 ==
在上次的戰事結束之後，戰士們再度回歸到了各自安詳寧靜的世界；小小兔融入了[[月野兔|小兔]]的家居生活，作為一名普通的小女孩與[[月野兔|小兔]]、[[地場衛|阿衛]]、以及其他四位內部太陽系戰士們一起生活著。

有一天，小小兔在睡夢中見到一匹獨角馬正在呼喚她，並要求她保守他們二人見面的秘密，之後被小兔喚醒，起床後與其他內部太陽系戰士相約去觀賞難得一見的日全蝕，但在準備歸途的時候，小兔突然乍見一個奇妙的馬戲團帳篷佇立著，以為是看錯了的小兔自然沒放在心上；但就是這個馬戲團，再度為戰士們好不容易平靜的生活，帶來了新的戰鬥。

果然有新的敵人出現，并且襲擊戰士們的好友古幡宇奈月，原来敵人的目標是夢之鏡，小兔與小小兔變身成戰士，之後她們看見一匹獨角馬，還帶來一股新的力量……
