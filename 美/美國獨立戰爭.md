'''美國獨立戰爭'''（{{lang-en|American War of Independence}} 或{{lang|en|American Revolution}}；1775年—1783年），或稱'''美國革命戰爭'''（{{lang|en|American Revolutionary War}}），是[[大英帝國|英國]]與準備獨立的[[美利堅|美利堅]]及[[法國|法國]]這個[[欧洲|欧洲]]强国之間的一場戰爭<ref name="美国的战争" />。

這場戰爭主要是始於是[[北美十三殖民地|北美十三殖民地]]為了對抗英國壓迫性的[[重商主義|經濟政策]]，但後來卻因為法國加入戰爭對抗英國，而使戰爭的範圍遠遠超過英屬[[北美洲|北美洲]]之外。因此，英國決定與當地的[[印地安人|印地安人]]結盟，以增加勢力。在獨立戰爭初期，英國能夠充分利用[[皇家海軍|皇家海軍]]船堅炮利的優勢控制十三殖民地的沿海城市，但對於如何控制內陸地區、及應對盤據內陸、熟悉地形、驍勇善戰的[[一分鐘人|殖民地民兵]]卻束手無策。隨著法國海軍在[[切薩皮克海戰|切薩皮克海戰]]大捷奪得[[制海權|制海權]]，英軍於1781年的[[約克鎮圍城戰役|約克鎮之圍城戰役]]中慘敗投降，並於1783年被逼簽訂《[[1783年巴黎條約|巴黎條約]]》承認[[美國|美國]]獨立。

但後來因為大量{{tsl|en|Loyalist (American Revolution)|殖民地保皇黨}}於戰後由十三殖民地逃至北方的[[英屬北美|英屬北美]]，反而導致新建國的美國立國後經濟困頓蕭條、甚至於冬季出現疫潮及飢荒，以及作為英國殖民地的[[加拿大|加拿大]]出現短暫的興旺。

== 背景 ==

自[[哥伦布|哥伦布]]发现[[美洲|美洲]]以来，[[北美洲|北美洲]]逐渐成为[[欧洲|欧洲]]的[[殖民地|殖民地]]。1763年，随着[[法國印第安人战争|法國印第安人战争]]的结束，英国成为[[北美洲|北美洲]]的霸主。

没有战争威胁的北美洲殖民者们与英国皇室的宗主关系开始异化，他们似乎已经不再需要英国军队的保护。然而英国为了维护本土的[[垄断|垄断]]利益，颁布一系列高额[[税收|税收]]的法令。1764年颁布《食糖条例》，要求美洲殖民地的欧裔殖民者们必须继续大量购买英国的[[食糖|食糖]]、[[咖啡|咖啡]]、[[酒|酒]]等商品；1765年颁布《印花税法》，对所有[[印刷品|印刷品]]直接[[征税|征税]]。这些沉重的经济负担使北美的欧裔殖民者们强烈反抗。因为他们所在的殖民地区在[[英国议会|英国议会]]没有代表席位，喊出“无代表，不纳税”的口号。同时，北美欧裔殖民们抵制并中断与英国商人的贸易往来。

1766年，[[英国议会|英国议会]]被迫废除印花税法，但是第二年又开始对殖民地急需商品（[[玻璃|玻璃]]制品、[[纸张|纸张]]、[[铅|铅]]、[[颜料|颜料]]、[[茶|茶]]、[[糖|糖]]、[[朗姆酒|朗姆酒]]、[[铁|铁]]、[[棉花|棉花]]等）开征高额[[关税|关税]]。殖民地重新断绝与英国的商贸往来，并通过[[加勒比海|加勒比海]]将这些商品走私进来。

在此背景下，北美各地欧裔殖民们对英国本土与北美殖民地之间的关系展开激烈的讨论，主导观点认为[[宗主国|宗主国]]不務正事，故无权向海外殖民地区征税。激进者甚至认为英国议会遠在歐洲，对海外殖民地区没有[[立法权|立法权]]。而对英国政府來說，为维护自身在北美洲殖民地的主导统治地位而颁布的四个法案《唐森德条例》受到的批评更是引起英国本土政府的不安，导致英国政府向[[波士顿|波士顿]]调兵，在人口只有16,000人的[[波士顿|波士顿]]驻扎接近7,000名英国士兵。这更让殖民地的欧裔殖民者认为[[宗主国|宗主国]]不懷好意，想剥夺他们在北美殖民地的权利和自由。

==冲突加剧==
[[File:Boston_tea_party.jpg|350px]]]]
1770年3月5日，一小批[[波士顿|波士顿]]居民向[[普雷斯頓|普雷斯頓]]上尉所率領的英軍士兵投掷包着石块的雪球，随着越来越多人的加入，英国士兵惊慌失措地开枪，最终导致历史上著名的[[波士顿屠杀|波士顿屠杀]]，有5名居民死亡和1名士兵受傷。事件发生后，殖民地开始抵制英国的进口，最终造成1770年4月除茶税以外，所有赋税和关税都被取消。

1773年12月16日發生[[波士頓茶葉黨事件|波士頓茶葉黨事件]]，[[革命|革命]]組織茶葉黨化装成[[印第安人|印第安人]]，将停泊在[[波士顿|波士顿]]港口的英国[[货轮|货轮]]上運载的342箱茶叶抛入海中，以此表示对[[英国国会|英国国会]]的抗议。1774年英國政府针对[[马萨诸塞|马萨诸塞]]通过一系列《[[強制法案|強制法案]]》，旨在加强對其的控制，卻激化了雙方的矛盾。1774年9月5日，除[[佐治亞|佐治亞]]外，12個地区的殖民會議選派56名代表在[[費城|費城]]召開[[大陸會議|大陸會議]]，通過宣言，並建立大陸議會，决定必须彻底改变宗主国与殖民地的关系。<ref name="美国的战争" />[[新英格蘭|新英格蘭]]的殖民势力組織自主的[[民兵|民兵]]武装。馬薩諸塞的代表會議則任命[[約翰·漢考克|約翰·漢考克]]為安全委員會主席。[[美國革命|美國革命]]開始。

==戰事開始—列星頓的槍聲==
{{主|列星頓和康科德战役}}
[[File:Military_Costume_of_the_Revolution_(1855_Chromolithograph).jpg|thumb]]]]
[[File:American_Revolution_Campaigns_1775_to_1781.jpg|250px]]
1775年4月18日，英國駐[[麻省|麻省]]的總督[[托馬斯·蓋奇|托馬斯·蓋奇]]將軍得悉當地民兵在距[[波士頓|波士頓]]21英里的[[康科德|康科德]]設有武器庫，遂出動800名英軍奔襲[[康科德_(新罕布什尔州)|康科德]]，銷毀殖民地民兵的軍械，並意圖一併拘捕該地的「通訊委員會」成員，但卻被殖民地居民得悉，並通知民兵組織「[[义勇军|义勇军]]」。义勇军得知消息後，當晚派人馳赴[[列克星敦_(马萨诸塞州)|列星頓]]和康科德報信，並立即作出防衛。

4月19日清晨5時左右，英軍遭到列星頓民兵阻攔，英軍突然開火，民兵猝不及防，死傷近十多人。隨後民兵與英軍在正式交火，美國獨立戰爭第一槍正式打響，結果英軍損失73人，而殖民地民兵則損失293人。8月23日英王宣佈殖民地居民的反抗運動為非法，並聲言「寧可不要頭上的王冠，也決不會放棄戰爭」。12月22日，英國當局正式調派近50,000英軍鎮壓殖民地。

1775年6月15日，[[大陸會議|大陸會議]]舉行，並決定把殖民地居民組建成正規的大陸軍，即為[[美軍|美軍]]。而[[喬治·華盛頓|喬治·華盛頓]]則為大陸軍[[總司令|總司令]]，发行[[纸币|纸币]]，接管英國对殖民地行使的国家行政管理等[[主权|主权]]。同日，[[美軍|美軍]]成功夺取[[英军|英军]]的[[提康德羅加堡|提康德罗加要塞]]。北美獨立戰爭全面展開。華盛頓率領[[美軍|美軍]]採取持久作戰以消耗[[英軍|英軍]]的策略，與英軍展開長期的作戰。

== 武裝反英 ==
{{主|加拿大戰役}}
於莱克星敦和康科德戰鬥後，美軍為了解除英軍可以由[[加拿大|加拿大]]直下[[紐約|紐約]]、控制[[哈德遜河|哈德遜河]]流域的威脅，於是遠征加拿大。1775年11月12日[[理查德·蒙哥馬利|理查德·蒙哥馬利]]率領軍隊佔領[[蒙特利爾|蒙特利爾]]。1775年12月，從蒙特利爾出征的蒙哥馬利與逆流而上的[[貝內迪克特·阿諾德|貝內迪克特·阿諾德]]的兵分兩路聯合進攻[[魁北克城|魁北克城]]失利；蒙哥馬利本人戰死。冬季過去的1776年初，英國沿[[聖勞倫斯河|聖勞倫斯河]]派出增援軍，趕走圍攻魁北克城的大陸軍。最終，戰敗的大陸軍退出加拿大。由於大陸軍出擊加拿大，英國被迫將半數軍隊留駐加拿大，戰鬥力被削弱。在波士頓，英軍被圍困達11個月之久，並於1776年3月被迫撤出。6月，大陸軍在南方擊退英軍對南卡羅來納的查爾斯頓的進攻，粉碎英軍在南方建立基地的企圖。

[[邦克山战役|邦克山战役]]后，大陆会议呼吁[[乔治三世|乔治三世]]和平解决殖民地问题，但是英國政府仍堅持以軍事鎮壓殖民地。1776年初，[[湯瑪斯·潘恩|潘恩]]出版[[常識_(小冊子)|《常識》]]一書譴責英國政府的高壓政策，並支持北美十三州獨立，大力鼓舞十三州人民，于是大陆会议在1776年7月2日宣佈13個殖民地脱离英国獨立，7月4日通過《[[美国独立宣言|独立宣言]]》。

== 第一階段—戰略防禦階段 ==
[[File:Washington_Crossing_the_Delaware_by_Emanuel_Leutze,_MMA-NYC,_1851.jpg|thumb]]
從1775年4月至1777年10月，是北美殖民地的戰略防禦階段。在這一階段，北部地區為主要戰場，英軍採取主動攻勢。1775年6月17日，英軍與波士頓民兵在[[碉堡山戰役|碉堡山戰役]]進行首次戰鬥。1776年8月，[[威廉·豪|威廉·豪]]接替蓋奇任英軍統帥。他以3.2萬兵力進攻紐約。英軍首先在[[長島|長島]]登陸，經過激戰，美方死傷1500人，英方傷亡不到400人。為避免全軍覆沒，美軍以退為進，主動放棄[[紐約|紐約]]。1776年9月26日英軍佔領[[費城|費城]]。華盛頓被迫率軍退守費城附近的福奇谷過冬。

在戰爭第2階段，美軍開始以攻擊孤立的敵軍支隊的戰術來擺脫所處的被動境地。1776年12月25日夜晚，華盛頓率領美軍橫渡[[特拉華河|特拉華河]]，突襲[[特倫頓_(紐澤西州)|特倫頓]]的[[黑森傭兵|黑森傭兵]]的兵營，並在1777年1月3日擊潰在[[普林斯頓|普林斯頓]]的3團英軍。

英軍在控制重要城市和海岸線後，力圖速戰速決。1777年7月英軍兵分三路，攻往[[奧爾巴尼|奧爾巴尼]]，以切斷殖民地的對外聯繫。但[[巴利·聖萊傑|巴利·聖萊傑]]部在[[莫華克河|莫華克河]]谷遭民兵突襲，被迫退回加拿大休整。[[約翰·伯戈因|約翰·伯戈因]]迅速拿下[[提康德羅加堡|提康德罗加要塞]]，使起义军失掉北方的主要阵地。7月23日，威廉·豪率领约1500名士兵进入[[纽约|纽约]]。

伯戈因統率的北路英軍約7200餘人沿[[尚普蘭湖|尚普蘭湖]]北上，到達[[哈得孫河|哈得孫河]]上游。他派往[[佛蒙特|佛蒙特]]的一支1000人的黑森傭兵分遣隊，在8月16日被[[佛蒙特州|佛蒙特州]]的綠山兄弟會诱入本宁顿战场全殲。之后，威廉·豪从纽约[[切萨皮克湾|切萨皮克湾]]撤出，并在布兰迪万河战役中反攻华盛顿部队，重新占领[[费城|费城]]。

伯戈因行動遲緩，從[[蒙特利爾|蒙特利爾]]南下時，也遇到[[新英格蘭|新英格蘭]]民兵的強烈抵抗，並在9月19日弗裡曼農莊及10月7日貝米斯高地的兩次戰鬥中慘敗，結果被迫退往[[薩拉托加|薩拉托加]]。[[霍雷肖·盖茨|霍雷肖·盖茨]]和[[本尼迪克特·阿诺德|本尼迪克特·阿诺德]]的军队立即以3倍兵力包圍該地的英軍，伯戈因無法再戰，被逼於10月17日率領5700名英軍投降，从波士顿被遣返回国。[[薩拉托加戰役|薩拉托加戰役]]大大改變美國獨立戰爭的形勢，使大陸軍開始掌握戰略主動權。

== 第二階段—戰略相持階段 ==
[[File:Us_unabhaengigkeitskrieg.jpg|thumb]]

從1777年10月至1781年3月，雙方進入戰略相持階段。美國開始在國際外交上獲得優勢，[[法國|法國]]加入戰爭，而[[西班牙|西班牙]]、[[荷蘭|荷蘭]]亦相繼支持美軍對抗英軍。

1778年2月，法國正式承認美國，並與其互訂軍事同盟。英国为了阻止他们同盟，於是废除强制法令和茶税法，并派遣和平使者前往费城。但美国已经批准与法国的条约。1778年6月，法國對英國宣戰，1779年6月西班牙與法國締結聯盟，以法國[[同盟|同盟]]者的身份出資金援助反英戰爭。1780年12月荷蘭亦加入美國陣營，支持法對英宣戰。英國在國際社會被眾[[列強|列強]]孤立。英、美雙方力量趨於平衡。

[[File:The_Siege_and_Relief_of_Gibraltar_(2).jpg|thumb]]中，法軍慘敗給英軍]]

[[File:Combat_de_la_Dominique_17_Avril_1780_Rossel_de_Cercy_1736_1804.jpg|thumb]][[海戰|海戰]]中，法國艦隊協助美國對抗英國艦隊時的情景]]

英軍擔心法國艦隊封鎖[[特拉華河|特拉華河]]，於是撤出[[費城|費城]]，集中兵力退守[[紐約|紐約]]城。1778年，[[亨利·克林頓|亨利·克林頓]]接替豪任英軍統帥，主戰場移向南部地區，兵力集中于南卡罗莱纳和佐治亚<ref name="美国的战争" />。英軍佔領[[佐治亞州|佐治亞州]]首府[[薩凡納|薩凡納]]，1779年佐治亞其他地方也被英军占领。但在1779年9月卻反被美法聯軍包圍。1780年春天，英軍進圍[[查尔斯顿_(南卡罗来纳州)|查爾斯頓]]，該年5月英軍成功逼降被圍的南方美軍，约3000名美军被俘，是独立战争中美国损失最惨重的一次。克林頓以為南方勝局已定，就率軍返回紐約，派[[查爾斯·康瓦利斯|查爾斯·康瓦利斯]]固守查爾斯頓。随后美国盖茨将军的军队与康瓦利斯在南卡罗莱纳的[[坎登|坎登]]遭遇，美军损失1000名士兵。

然而1781年1月南方美軍卻成功引誘英軍至[[南卡羅來納州|南卡羅來納州]]的[[考彭斯_(南卡羅來納州)|考彭斯]]，並[[考彭斯戰役|將其圍殲]]。該年3月美軍在[[北卡羅來納州|北卡羅來納州]]的[[吉爾福德|吉爾福德]]大敗英軍。該年4月[[查爾斯·康瓦利斯|康瓦利斯]]統率英軍向北撤至[[維珍尼亞|維珍尼亞]]。而美軍則由格林接替盖茨統率，揮師南下，一舉收復除[[薩凡納|薩凡納]]和[[吉爾斯頓|吉爾斯頓]]之外的全部南方國土。戰至此時，英軍已漸露敗跡。

== 第三階段—戰略反攻階段 ==
[[File:Yorktown80.JPG|thumb]]
從1781年4月至1783年9月，是美軍的戰略反攻階段。1781年4月大陸軍開始戰略反攻，[[彌敦內爾·格連|納撒尼爾·格林]]率領大陸軍南下轉戰南卡羅來納，迫使英軍退守海岸線。康瓦利斯於4月25日北上維吉尼亞，追擊[[拉法葉侯爵|拉法葉侯爵]]率領的一支大陸軍。但拉法葉擺脫英軍的追擊。

1781年8月，康瓦利斯奉克林顿将军命令統率7000名英軍死守維吉尼亞的[[約克鎮|約克鎮]]。法美聯軍由華盛頓統率，南下維吉尼亞，而增援的法國艦隊也在法國海軍司令[[德格拉斯伯爵|德格拉斯伯爵]]率領下由[[西印度群島|西印度群島]]調來，進入[[切薩皮克灣|切薩皮克灣]]，進抵約克鎮城外的海面，並且擊退增援的英國艦隊，掌握制海權。9月28日，華盛頓部大陸軍和[[羅尚博伯爵|羅尚博伯爵]]部法軍在維吉尼亞與[[拉法葉侯爵|拉法葉侯爵]]部大陸軍會合，法美聯軍共1.7萬人完成對約克鎮的合圍。康瓦利斯無路可退，只得於10月17日與美法聯軍進行投降談判。10月19日，駐守約克鎮的英軍共8000人正式投降。此後，英国放弃对美国革命的压制，雙方只有數次海上和陸上的零星戰鬥，其餘戰事已大致停止。

==结局==
[[File:Treaty_of_Paris_by_Benjamin_West_1783.jpg|thumb]]著名畫家[[本傑明·韋斯特|本傑明·韋斯特]]筆下的[[巴黎和約|巴黎和約]]。]]

[[約克鎮圍城戰役|約克鎮圍城戰役]]之後，[[英國議會|英國議會]]被迫贊成議和，1782年11月30日，英美兩國簽署《[[1783年巴黎條約|巴黎和約]]》的草案，1783年9月3日，美國成為[[美洲|美洲]]首個[[獨立國家|獨立國家]]。共包括13个州，边界北至[[五大湖|五大湖]]，西到[[密西西比河|密西西比河]]，南至[[北纬|北纬]]31度。

==影响==
战争结束后美国便脱离大英帝国和英国皇室的统治，成为了一个独立的国家，但由于美国境内生活着大量的保皇派，在美国独立后大量的保皇派进入了美国北部的英属北美地区，而保皇派的进入直接导致了当地原本以法语人口为主的人口分布被彻底改变，形成了后来的上加拿大和下加拿大，且由于英裔和法裔的文化差异，导致上下加拿大之间的分歧一直存在，后来发展成为[[魁北克独立|魁北克独立]]。

而对于英国来说，它失去了人口最多、资产最丰富的北美十三州殖民地，间接的导致了大英帝国的衰落，而后由于美国的野心，英美之间多次爆发战争，儘管英军有多次戰役击败美军，但英軍整體而言並未佔上風。同样由于大英帝国统治时期对殖民地的掠夺政策，导致美国人对大英帝国很反感，这种反感一直持续到1940年代末大英帝国开始瓦解，而美国在大英帝国瓦解的过程中起到了关键的作用，其帮助许多英国在加勒比地区的殖民地独立。
