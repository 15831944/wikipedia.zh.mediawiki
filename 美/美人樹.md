'''美人樹'''，[[英文名|英文名]]：'''''Silk floss tree''''' or  '''''Floss silk tree'''''，[[學名|學名]]：{{lang|la|'''''Ceiba speciosa'''''}}（舊學名 Chorisia speciosa），[[別名|別名]]：'''美人櫻'''、'''酒瓶木棉'''、'''美丽异木棉'''，为[[木棉科|木棉科]][[爪哇木棉屬|爪哇木棉屬]]下的大[[喬木|喬木]]。

'''開花的數量'''使用[[估算法|估算法]]大概有5000~6000朵[[花|花]]。

[[File:北斗河濱公園_美人樹.jpg|thumb]]沿岸的美人樹]]

== 形态 ==
'''[[落葉喬木|落葉喬木]]'''
=== [[根|根]] ===

=== [[莖|莖]] ===
==== 形狀 ====
[[樹幹|樹幹]]直立，幹基擴大，具有少數分枝，側枝呈斜水平狀向上開展，[[雨林#树冠层|樹冠層]]呈傘。

==== 顏色 ====
綠色。

==== 大小 ====
株高8～15公尺。

==== 質地 ====
[[樹皮|樹皮]]平滑，著生[[尖銳瘤狀刺|尖銳瘤狀刺]]。

=== [[葉|葉]] ===
[[掌狀複葉|掌狀複葉]]，叢生小枝先端，[[小葉|小葉]]5～7枚。
==== 形狀 ====
橢圓形或倒長卵形，先端或銳尖，基部狹，葉緣成鋸齒狀

==== 顏色 ====
表面為有光澤的綠色，背面淡綠色。

==== 大小 ====
長10～15公分，寬2～4公分，[[葉柄|葉柄]]長6～15公分；小葉長不及 1 公分。

==== 質地 ====
紙質。

==== [[葉脈|葉脈]] ====
[[羽狀側脈|羽狀側脈]]。

==== [[葉柄|葉柄]] ====
葉柄成圓柱形，光滑無毛，小葉甚短。

=== [[花|花]] ===
花多先葉或與葉同時長出，大而豔麗，富觀賞價值。
==== 形狀 ====
略粗壯，花緣皺摺，略反捲。

==== 顏色 ====
紫紅色或淡粉紅色，近中心處乳白色有褐色斑點。

==== 大小 ====
長7～19公分，開放時徑10～15公分。

==== [[花序|花序]] ====
腋生，總狀花序

==== [[花瓣|花瓣]] ====
花瓣5瓣，長橢圓形狀披針形，長8～14公分，寬3～4.5公分，先端鈍，有柔毛；花瓣中間的5根雄蕊合生在一起，將更裡面的雌蕊包起來，而靠近基部則有一圈特別的白纖絲狀構造。

==== [[花柄|花柄]] ====
花柄短，長1～2公分，粗壯

==== [[花萼|花萼]] ====
花萼鐘形，帶紫紅色或淡粉紅色，先端有淺裂；每花具2～3[[小苞|小苞]]；萼杯狀，2～5齒。

==== [[子房|子房]] ====
子房上位。

==== [[雄蕊|雄蕊]] ====
雄蕊合生成筒狀，5柱，每柱再裂分為2。

==== [[雌蕊|雌蕊]] ====

==== [[花期|花期]] ====
9～11 月

=== [[果實|果實]] ===
[[蒴果|蒴果]]
==== 形狀 ====
橢圓形，成熟時三瓣裂。

==== 大小 ====
長15～22公分，徑5～7公分。

=== [[種子|種子]] ===
多數，具綿毛。
==== 顏色 ====
黑色

==== 傳播 ====
[[風力|風力]]

== 用途 ==
* [[行道樹|行道樹]]、庭園樹：是高級行道樹，高級庭園造景，美化樹種。
*[[蝴蝶|蝴蝶]]的蜜源

== 美人樹的別名 ==
美人樹與[[木棉|木棉]]同[[科|科]]，由於樹幹造形奇特有如[[日式|日式]][[陶瓷|陶瓷]][[酒瓶|酒瓶]]因而有「'''酒瓶木棉'''」的稱號。開花的時候景觀壯麗，粉紅色大型花掛滿[[樹冠|樹冠]]，映襯著秋天的青空白雲，就像矗立在大地的美人一般。由於木棉的葉子比較大、樹型較陽剛，所以有「'''英雄樹'''」稱號，相較之下，葉子較小、樹型較柔美的就是「'''美人樹'''」了。美人樹的花期在[[秋|秋]]至初[[冬|冬]]或[[夏|夏]]至[[秋|秋]]季，在[[台灣|台灣]]越往北部越早開花。
