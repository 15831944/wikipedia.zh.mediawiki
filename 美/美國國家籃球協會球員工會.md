'''美國國家籃球協會球員工會'''（{{lang|en|National Basketball Players Association}}，'''NBAP'''）是由[[波士顿凯尔特人|波士顿凯尔特人]]的全明星得分后衛[[鮑勃·庫錫|鮑勃·庫錫]]於1954年成立的球員工會，旨在保障、改善[[美國國家籃球協會|美國國家籃球協會]]（NBA）各球隊球員的勞動權利。<ref name=bradley>{{cite web | last = Bradley | first = Robert | title=Labor Pains Nothing New to the NBA|url=http://www.apbr.org/labor.html| accessdate=February 26, 2011}}</ref>。工會成立之初并不被NBA球队的董事所认可，经过十年的努力才逐漸获得各方承认。

== 宗旨 ==
该组织致力于保障并改善NBA现役球员的福利與权利，通过各种措施帮助球员获取最好的机会，提升球员在各項纠纷和官司中的权益，以最大化球員權益为目标。<br />

== 歷史 ==
参与之重要事件與成果：

* 1983年劳资双方达成工资帽协定
* 1995年的劳资纠纷
* 1998～1999赛季的停摆
* 2011年的赛季停摆
