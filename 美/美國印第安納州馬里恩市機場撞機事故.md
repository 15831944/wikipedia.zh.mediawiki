'''馬里恩市機場撞機事故'''在2018年4月2日於美國印第安納州馬里恩市機場發生，造成2人死亡。

==事件原因==
[[美國聯邦航空總署|美國聯邦航空總署]]表示該機場沒有設置塔台，因此飛機起降全靠機長用無線電溝通。當局暫時認為可能因此而導致意外發生。<ref name="name2">{{cite news|language = zh-tw |url = http://www.setn.com/News.aspx?NewsID=364964|title = 美機場沒塔台…班機起降「用說的」　2機互撞釀2死 |publisher = 三立新聞 |date = 2018/04/03}}</ref><ref name="name1">{{cite news |language = zh-tw |url = https://www.ettoday.net/news/20180403/1143514.htm|title = 班機起降「用說的」！美機場沒塔台　2飛機相撞「削尾」2死|publisher = 東森新聞 |date = 2018-04-03}}</ref>目前[[美國國家運輸安全委員會|美國國家運輸安全委員會]]正調查事件詳細起因。<ref name="name1">{{cite news |language = zh-tw |url = https://www.ettoday.net/news/20180403/1143514.htm|title = 班機起降「用說的」！美機場沒塔台　2飛機相撞「削尾」2死|publisher = 東森新聞 |date = 2018-04-03}}</ref>

==事件經過==
小型飛機於當日下午5點9分起飛，卻撞上一架正要降落的噴射機。小型飛機被撞至幾乎報銷，墜毀起火<ref>{{cite news|language = zh-hk |url =https://www.singtao.com.au/172-%E5%8D%B3%E6%99%82/342254-%E6%A9%9F%E5%A0%B4%E7%84%A1%E6%8C%87%E6%8F%AE%E5%A1%94%E3%80%80%E7%BE%8E%E5%9C%8B%E5%85%A9%E6%A9%9F%E7%9B%B8%E6%92%9E%E6%A9%9F%E5%B0%BE%E6%96%B7%E9%96%8B%E6%9C%80%E5%B0%91%E5%85%A9%E6%AD%BB/|title = 機場無指揮塔　美國兩機相撞機尾斷開最少兩死|publisher = 星島日報 |date = 2018-04-03}}</ref>，機上的兩名乘客全數罹難。而噴射機的機尾被撞斷，機上的5名乘客目前受傷狀況不明。<ref name="name2">{{cite news|language = zh-tw |url = http://www.setn.com/News.aspx?NewsID=364964|title = 美機場沒塔台…班機起降「用說的」　2機互撞釀2死 |publisher = 三立新聞 |date = 2018/04/03}}</ref>而當局獲報後於5點30分趕抵現場。<ref name="name1">{{cite news |language = zh-tw |url = https://www.ettoday.net/news/20180403/1143514.htm|title = 班機起降「用說的」！美機場沒塔台　2飛機相撞「削尾」2死|publisher = 東森新聞 |date = 2018-04-03}}</ref>
