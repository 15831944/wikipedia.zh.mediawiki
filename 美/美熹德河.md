'''美熹德河'''（{{lang-en|Merced River}}）是[[美国|美国]][[加利福尼亚州|加利福尼亚州]]内的一条河流，其源头位于[[優勝美地國家公園|優勝美地國家公園]]的南部，此后它注入[[约塞米特峡谷|约塞米特峡谷]]。此后它被[[新埃克斯切凯尔坝|新埃克斯切凯尔坝]]堵住形成[[麦克卢尔湖|麦克卢尔湖]]。在克罗克·哈夫曼分水坝它的水被分开注入美熹德河灌溉区。剩下的水向西南流过山地和[[聖華金谷|聖華金谷]]注入[[圣华金河|圣华金河]]。

美熹德河受美国国家野生和风景法的保护。到麦克卢尔湖为止它可以自由流畅，在冬季和春季容易形成大水，而深夏和秋季则水流极小。{{le|140号加利福尼亚州州道|California State Route 140}}部分沿美熹德河行走，是进入優勝美地國家公園的主要途径。美熹德河畔有许多休闲机会，包括钓鱼、划皮艇、白浪木筏、野营、徒步旅行、游泳和野餐。

== 概述 ==
美熹德河的主支长180千米<ref name=usgs1> [http://geonames.usgs.gov/pls/gnispublic/f?p=171:3:3062783640428646857::NO::P3_FID:272412 Merced River Detail Report], ''United States Geologic Survey'', Geographic Names Information System (GNIS). 2007年4月22日</ref>。此外它还有一些分支（比如北支、南支等）<ref name=usgs2> [http://geonames.usgs.gov/pls/gnispublic/f?p=171:2:3062783640428646857::NO::: Merced River Query], ''United States Geological Survey'', Geographic Names Information System (GNIS). 2007年4月22日</ref>。

== 灌溉和渔业 ==
河流的大多数段落被“改善”来促进鱼产籽。

美熹德河灌溉区由众多运河、小运河、输水管、水库组成<ref>[http://www.mercedid.org/_images/watershed_map.pdf http://www.mercedid.org/_images/watershed_map.pdf] {{webarchive|url=https://web.archive.org/web/20091122164830/http://www.mercedid.org/_images/watershed_map.pdf |date=2009-11-22 }}</ref>。

== 历史 ==
美熹德河今天的河床通过约塞米特峡谷，这个谷是在[[冰河時期|冰河時期]]里由冰川导致的。

[[派尤特人|派尤特人]]曾经在这里居住。

1806年欧洲人首次到达这条河边。
