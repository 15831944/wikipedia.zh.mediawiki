'''美國空軍'''（Jane's USAF─United States Air Force）是一款遊戲的基本背景是由冷戰時期最大的一場區域衝突，歷經後冷戰的沙漠風暴到近未來的空中戰場。玩家可以控制的飛機也橫跨不同的作戰理念和設計方向。從冷戰高峰的戰術核子轟炸機（F─105）和高速攔截為概念的戰機（F─4）、以越戰經驗為借鏡的純空優重武裝戰鬥機（F─15）與戰機黑手黨的代表作（F─16）、面對華約絕對裝甲武力優勢的空中阻絕手段（A─10）、以匿蹤為設計主要手段的銀子彈（F─117）以及極具爭議的下一代匿蹤戰鬥機（F─22），八種涵蓋60年代到下一世紀的戰機，提供玩者多樣化的任務和作戰環境，如同[[詹氏資訊集團|詹氏]]過去的產品一般極具吸引力<ref name="遊戲評析"/>。

== 遊戲任務 ==
遊戲的片頭開場影片以全國空軍的各型飛機的影片配合類似（[[浴火赤子情|浴火赤子情]]）片中[[漢斯·季默|漢斯·季默]]的配樂，遊戲任務分成：「立即任務」，「單一任務」與「多人連線」<ref name="遊戲評析"/>。

「立即任務」中玩者可以調整許多參數改變任務的狀態。「單一任務」中有部分任務是玩者可以立即進行。「紅旗演習」則需要玩者至少往上升一級才能夠進行。當玩家順利執行過某些「戰役」中的任務可以回到這個部分，重新進行任務。

「戰役」模式下有兩類型的作戰：以歷史為背景的越戰與「沙漠風暴」作戰，以及命名為「紅箭行動」與「沈睡巨人」行動的兩場假想戰役。至於「多人連線」模式，則是與網路上其他玩家連線對戰<ref name="遊戲評析"/>。

玩者在選擇任務之後，便會進入簡報畫面，除了傳統的文字簡報和任務地圖外，簡報中還有來自衛星或其他載具的偵察照片供玩者確認目標的外觀。接著玩者就可以為各組出動的飛機改掛和原訂不同的武器設備<ref name="遊戲評析"/>。

在不同的任務下，出動的機組數量、配置與任務開始的地點都會不同。當任務由多機組進行的時候，玩者可以在不同的機組中來回控制以達到較佳的任務執行效率和準確度<ref name="遊戲評析"/>。

== 遊戲畫面 ==
遊戲的成像引擎來自於詹氏前一套的《[[以色列空军(游戏)|以色列空军]]》的改進版，地面景觀是以衛星地圖為來源，在足夠的電腦和影像卡的性能支援下，效果相當不錯，玩家還可以在高低起伏的丘陵或山溝間高速飛行。玩者所能控制的八種飛機的座艙與外觀都算正確，任何掛載都清晰可見，另外自座艙向後方看的景觀都可以見到相關的外掛武器，繪圖的準確度和效果相當不錯。另外在改變飛行姿態的時候也能夠清晰的見到各控制面的相對變化，激烈的動作下在翼端或翼根所帶起的激烈氣流也每每能在飛行中見到<ref name="遊戲評析"/>。

== 遊戲環境 ==
遊戲對各型飛機的飛行模式的模擬，由於受限於當時電腦配備，無法一一將各型飛機的飛行模擬的鉅細靡遺，不過該遊戲的圖形品質和聲音效果，還是有一定程度的表現。遊戲任務都是預先設計好的，雖然變化不多，遊戲任務的難度和變化還算平衡，讓玩家能用較短的時間，可以對遊戲迅速適應和掌握<ref name="遊戲評析"/>。

== 評價 ==
=== 正面 ===
可以體驗多款美國近代最優秀的空中武力，並且遊戲難度適中，娛樂效果充足<ref name="遊戲評析"/>。

=== 負面 ===
真實性稍低，並不是每台飛機皆符合真實情況<ref name="遊戲評析"/>。
