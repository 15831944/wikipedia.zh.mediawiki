'''美国国家侦察局'''（National Reconnaissance Office，'''NRO'''），位于[[美国|美国]][[维吉尼亚州|维吉尼亚州]]，是美国的16个情报机构之一；负责为[[美国政府|美国政府]]设计、组装并发射侦察[[卫星|卫星]]。<ref>{{cite web
 |url          = http://www.nro.gov/index.html
 |title        = Welcome to the NRO
 |accessdate   = 2007-01-15
 |language     = en
 |archive-url  = https://web.archive.org/web/20070117082618/http://www.nro.gov/index.html
 |archive-date = 2007-01-17
 |dead-url     = yes
}}</ref>  并协调、收集和分析从[[中央情报局|中央情报局]]以及军事机构的[[航天飞机|航天飞机]]、[[卫星|卫星]]收集到的情报。<ref>"[http://www.nro.gov/PressReleases/prs_rel18.html NRO Provides Support to the Warfighters] {{webarchive|url=https://web.archive.org/web/20010618144016/http://www.nro.gov/PressReleases/prs_rel18.html |date=2001-06-18 }}"</ref> 该机构得到国家侦察计划（外国情报收集计划的一部分）的拨款。该机构是[[美国国防部|国防部]]的组成部分之一。

国侦局和相关[[太空|太空]]、情报机构密切合作，包括[[美国国家安全局|美国国家安全局]]（NSA）、[[美国国家地理空间情报局|美国国家地理空间情报局]]（NGA）、[[中央情报局|中央情报局]]（CIA）、[[國防高等研究計劃署|美国国防高级研究计划局]]（DARPA）、[[美国战略司令部|美国战略司令部]]、[[美国海军研究实验室|海军研究实验室]]等高级机构。负责运作在世界各地的地面站，用于收集并发布来自侦察卫星的情报。

== 历史 ==
国侦局成立于1961年9月6日，旨在提高国家卫星侦察系统。1958年2月[[苏联|苏联]]发射[[人造地球卫星1号|人造地球卫星1号]]后，在[[艾森豪威尔|艾森豪威尔]]支持下成立，当飞行员[[弗朗西斯·加里·鲍尔斯|弗朗西斯·加里·鲍尔斯]]驾驶的[[U-2侦察机|U-2侦察机]]在1960年5月1日被击落后，该机构显得更加重要。

国侦局的第一项照片侦察卫星计划被成为[[日冕_(卫星)|“日冕”计划]]，根据1995年2月的解密资料，“日冕”计划存在于1960年8月到1972年5月，尽管首次试飞是在1959年2月。“日冕计划”的卫星所释放的摄影太空舱（有时用多个）由军方的飞船来校正。最成功的一次，对发现者13号的校正于1960年8月12日进行。第一张来自太空的图片在六天之后问世，分辨度达8米，后来被改善至2米。一张照片平均能显示大约16到190千米的范围。日冕的最后一次任务（第145次）在1972年5月25日进行。此项任务的最后一张照片拍摄于1972年5月31日。

从1962年5月到1964年8月，国侦局主持了Argon计划中的12次地图绘制任务，但只有7次成功。{{Fact|time=2008-01}} 1963年，国侦局开始用更高的分辨率进行Lanyard计划中的地图绘制部分。{{Fact|time=2008-01}}

国侦局1972年之后的任务仍然是机密，之前计划的剩余部分也不再为公众所知晓。国防部副部长在[[中央情报局|中央情报局]]局长的建议下于1992年9月18日证实了国侦局的存在。{{Fact|time=2008-01}}

1995年，一篇《[[华盛顿邮报|华盛顿邮报]]》文章指出，在[[中央情报局|中情局]]、[[五角大楼|五角大楼]]和[[美国国会|国会]]不知情的情况下，国侦局秘密保留了10亿至17亿[[美元|美元]]的未尽专款。中情局接到投诉，国侦局在一年前从秘密预算中拿出私留3亿美元在维吉尼亚州的尚蒂伊建造一个新的总部，因此介入国侦局的预算调查。[[美国科学家联合会|美国科学家联合会]]获得的一份建筑许可中的[[蓝图|蓝图]]则泄露了国侦局秘密新总部的情况。在[[九一一袭击事件|九一一]]后该蓝图便被列为机密。国侦局挪用资金被证实。发起这项调查的前中情局首席[[律师|律师]]傑佛瑞·史密斯（Jeffrey Smith）表示：“我们的调查显示，国侦局数量惊人的‘私房钱’已经积蓄多年了。”{{Fact|time=2008-01}}

1999年，国侦局和[[波音公司|波音公司]]开始着手名为“Future Imagery Architecture”的项目，即开发新一代的成像卫星。[[纽约时报|纽约时报]]在2007年11月11日发表的一篇调查报告指出：根据国侦局的记录，自2002年开始该项目进度就远远落后于计划，并可能超出预算20至30亿美元。美国政府开始施压，提出应努力完成该项目。但两年之后，因新增的评审小组和数十亿美元的额外开销，该计划被迫中止。纽约时报报道：“也许50年来最引人瞩目和最昂贵的失败就是美国的间谍卫星项目”

== 组织 ==
国侦局是[[美国国防部|国防部]]的一部分，国侦局局长由[[美国国防部长|国防部长]]在[[美国国家情报总监|美国国家情报总监]]的同意下任命，不需要国会的许可。按照惯例，该职位由[[美国空军|空军]]副部长或部长助理来担任， 但Donald Kerr在2005年7月被任命为国侦局局长是个例外。国侦局的员工来自中情局、[[美国国家安全局|国安局]]、军队和民间国防承包商。

国侦局由下列机构组成：信号情报处、通讯处、图像情报处和技术处。<ref>{{cite web |url=http://www.nro.gov/organization.html |title=存档副本 |accessdate=2008-03-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20080503011020/http://www.nro.gov/organization.html |archivedate=2008-05-03 }}</ref>
