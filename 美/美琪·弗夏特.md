'''美琪·弗夏特'''（{{lang-en|Meggie Folchart}}）是[[德國|德國]]作家[[柯奈莉亞·馮克|柯奈莉亞·馮克]]的系列[[奇幻小說|奇幻小說]]《{{le|墨水世界三部曲|Inkheart trilogy}}》中的主要角色，為{{le|莫提瑪·弗夏特|Mortimer Folchart|莫}}與其妻特蕾莎的獨生女兒。

在《[[墨水心_(電影)|墨水心]]》電影版裡，美琪由{{le|伊麗莎·班內特|Eliza Bennett}}飾演<ref>{{Cite news|url=https://www.getreading.co.uk/whats-on/tv-film/new-dvd-inkheart-pg-4239965|title=New DVD: Inkheart (PG)|last=getreading|date=2009-04-22|newspaper=getreading|accessdate=2018-03-20}}</ref><ref>{{Cite news|url=https://www.telegraph.co.uk/culture/film/filmreviews/3709153/Inkheart-review.html|title=Inkheart - review|last=Gritten|first=David|date=2008-12-11|work=|accessdate=2018-03-20|language=en-GB}}</ref><ref>{{Cite web|url=http://www.bbc.co.uk/cambridgeshire/content/articles/2008/12/11/inkheart_iain_softley_feature.shtml|title=Inkheart director's Cambridge days|accessdate=2018-03-20|date=2008-11-12|publisher=BBC|language=en-gb}}</ref>。

== 外觀 ==
美琪被形容為一個身材高大苗條的女孩，她頭髮光滑，有著金色的頭髮和藍色的眼睛，很多人都說，她看起來很像她的母親。她平時常穿[[牛仔褲|牛仔褲]]之類的休閒服裝，偶爾會穿[[毛衣|毛衣]]。

在故事中美琪往往是作為一個聰明、勇敢的女孩，但有時相當頑固<ref>{{Cite web|url=https://www.shmoop.com/inkheart/meggie.html|title=Meggie in Inkheart|accessdate=2018-03-20|work=www.shmoop.com|language=en}}</ref>。在她3歲時母親神秘[[失蹤|失蹤]]，往後她對母親留下了錯誤的負面印象<ref name=":0">{{Cite web|url=http://madeinatlantis.com/movies_central/2009/inkheart_production_details.htm|title=Inkheart Movie Production Notes|accessdate=2018-03-20|date=2014-09-12|publisher=2009 Movie Releases|language=en-US}}</ref><ref name=":1">{{Cite news|url=https://www.ctvnews.ca/close-the-book-on-inkheart-movie-feels-like-an-onslaught-1.363310|title=Close the book on 'Inkheart'; movie feels like an onslaught|last=Lemire|first=Christy|date=2009-01-23|newspaper=CTVNews|accessdate=2018-03-20|language=en-CA}}</ref>。

美琪不知不覺中[[遺傳|遺傳]]了其父莫提瑪的[[魔法|魔法]]天賦，即能夠將書中的人物唸到現實中來，但在《墨水心》故事展開前，她對此渾然不覺<ref name=":1" />。美琪也非常熱愛於[[讀書|讀書]]，但她父親却不敢朗讀，這讓她感到挫折<ref name=":0" />。

== 角色生平 ==
在《墨水世界三部曲》首部《[[墨水心|墨水心]]》中，美琪初登場時12歲，與她的父親[[書籍裝幀師|書籍裝幀師]]莫提瑪·弗夏特（被美琪暱稱為「莫」）兩人相依為命，直到某日一個名叫「髒手指」的男人上門拜訪，而後莫、美琪和親戚愛麗諾相繼被壞蛋山羊的手下抓走。美琪才初次得知父親具有透過閱讀而將書中人物唸到現實世界中的魔法能力，如果某件事或某人進入了書內，那麼就會有人從書中被唸出來；在美琪3歲時，莫無意間將妻子特蕾莎和兩隻[[貓|貓]]唸進了《墨水心》書中，於是「山羊」、髒手指、巴斯塔和一隻[[貂|貂]]（葛文）就跟著出來了。

父女倆經歷了不少冒險，美琪這時發現自己遺傳了父親的魔法能力，為此她被迫要為山羊朗讀，以招喚出怪物「影子」；但最終美琪與父親聯手，一起成功擊敗了山羊的惡勢力，將他們送回書中。而美琪一家也終於重聚。<ref>{{Cite book|title=墨水心|last=柯奈莉亞·馮克|publisher=大田出版|year=2009年1月|isbn=978-957-455-772-1|location=臺北}}</ref>

在續集《[[墨水血|墨水血]]》和《{{le|墨水死|Inkdeath}}》中，美琪作為主角亦有出場。

== 創作 ==
作者柯奈莉亞·馮克在一次受訪時表示，在她的上一部作品《{{Le|神偷|The Thief Lord}}》出版後，一些女性讀者希望書裡能有更多女性角色，於是馮克便向她們承諾下一部作品將會以女生作為主角英雄<ref>{{Cite web|url=http://www.morningstar.com.tw/epaperbox/tm080409.html|title=柯奈莉亞馮克專訪特報NO.3|accessdate=2018-06-28|publisher=大田出版}}</ref>。

== 電影改編 ==
[[英國|英國]]女演員伊麗莎·班內特於《墨水心》的電影改編中飾演12歲時的美琪·弗夏特，在電影選角時她是在數百名候選者當中贏得了這個角色<ref name=":0" />；而童年時的美琪則由[[米拉貝爾·奧克費|米拉貝爾·奧克費]]飾演<ref>{{Cite web|url=http://www.imdb.com/character/ch0037262/|title=Meggie Folchart|accessdate=2018-03-21|publisher=IMDB|archiveurl=https://web.archive.org/web/20160310210652/http://www.imdb.com/character/ch0037262/|archivedate=2016-03-10}}</ref>。對於飾演美琪角色的伊麗莎·班內特，柯奈莉亞·馮克表示：「她（班內特）是一个很可愛的小女孩，虽然她的年齡稍稍的比我原著里的人物大了一點，但是這並不是什麼太大的問題」<ref>{{Cite web|url=http://movie.mtime.com/51245/behind_the_scene.html|title=墨水心 幕后揭秘|accessdate=2018-03-20|publisher=Mtime时光网}}</ref>。

伊麗莎·班內特本人形容電影裡的美琪·弗夏特角色「她一直非常依賴她的父親，因為他是她唯一健在的雙親，但我認為在電影歷程中她被迫變得更加獨立。在故事結束的時候，她成長很多。」<ref name=":0" />。
