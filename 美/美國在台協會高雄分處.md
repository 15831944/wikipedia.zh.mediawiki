'''美國在台協會高雄分處'''（{{lang-en|'''American Institute in Taiwan Kaohsiung Branch Office}}'''，簡稱'''{{lang|en| AIT Kaohsiung Branch Office'''}}）是[[美國國務院|美國國務院]]根據《[[台灣關係法|台灣關係法]]》（Taiwan Relation Act）所設置的民間[[非營利組織|非營利組織]]——[[美國在台協會|美國在台協會]]駐[[高雄市|高雄市]]的辦事處，業務內容相當於[[總領事館|總領事館]]，並由制定《台灣關係法》的[[美國國會|美國國會]]擔任監督的角色。分處首長職稱為'''分處處長'''（{{lang|en|Branch Chief}}）。

== 領事辖區範圍 ==
美國在台協會高雄分處[[領事轄區|領事轄區]]範圍包括[[南台灣|南台灣]]的[[高雄市|高雄市]]、[[台南市|台南市]]、[[嘉義市|嘉義市]]、[[嘉義縣|嘉義縣]]、[[屏東縣|屏東縣]]、[[澎湖縣|澎湖縣]]和[[台東縣|台東縣]]七個縣市，這個區域面積為台灣的三分之一，人口超過六百萬，其中包括六千個[[美國|美國]][[僑民|僑民]]。位於美國在台協會高雄分處轄區的美國機構包括[[高雄市美國商會|高雄市美國商會]]、[[高雄美國學校|高雄美國學校]]（從幼稚園到十二年級）和[[馬禮遜美國學校高雄分校|高雄馬禮遜學校]]；此外，高雄市的[[國立中山大學|國立中山大學]]設有[[中山美國中心|中山美國中心]]，是國立中山大學與美國在台協會高雄分處合作設立<ref>[http://ait.org.tw/zh/ait-kaohsiung.html 高雄分處簡介]</ref>。2009年11月，[[高雄市立圖書館|高雄市立圖書館]]寶珠分館設立[[高雄美國資料中心|高雄美國資料中心]]，由美國在台協會高雄分處與高雄市立圖書館共同合作設立 <ref>[http://www.ait.org.tw/zh/information-resources.html 美國在台協會關於我們]</ref>，現高雄美國資料中心已喬遷至高市圖河堤分館，原寶珠分館之美國資料中心降為美國資料專區。

== 設立和發展 ==
美國在台協會高雄分處原位於高雄市[[新興區_(高雄市)|新興區]][[中正三路|中正三路]]2號國泰中正大樓。2014年8月8日喬遷至位在高雄市[[前鎮區|前鎮區]][[成功路_(高雄市)|成功路]][[中鋼集團總部大樓|中鋼集團總部大樓]]2樓全層和3樓全層，鄰近[[高雄捷運|高雄捷運]]<font color={{高雄捷運色彩|環}}>█</font>[[環狀輕軌|環狀輕軌]][[軟體園區站|軟體園區站]]<ref>[http://www.epochtimes.com/b5/13/10/22/n3992063.htm 中鋼大樓啟用 高雄灣區新地標].大紀元台灣.2013-10-22</ref>。
