'''美國空軍保安部隊'''（{{lang-en|United States Air Force Security Forces}}）是[[美國空軍|美國空軍]]的[[憲兵|憲兵]]部隊，由[[德克薩斯州|德克薩斯州]][[拉克蘭空軍基地|拉克蘭空軍基地]]的{{Link-en|美國空軍保安部隊中心|Air Force Security Forces Center|空軍保安部隊中心}}負責保安部隊全球駐軍的組織、訓練及裝備業務<ref name="中心">{{Cite web |script-title=en:WELCOME TO THE AFSFC |publisher={{lang|en|{{Link-en|美國空軍保安部隊中心|Air Force Security Forces Center|Air Force Security Forces Center}}}} |url=http://www.afsfc.af.mil |accessdate=2017-10-27 |archiveurl=http://archive.is/HYF3K |archivedate=2017-10-27 }}</ref>。

[[File:USAF_photo_110620-F-MA715-172.JPG|thumb]][[烏山空軍基地|烏山空軍基地]]值勤，其[[臂章|臂章]]上寫有「憲兵」（{{lang|kor|헌병}}）的韓語字樣。]]
空軍保安部隊中心的前身最早為1979年9月1日創設的空軍保安憲兵室（{{lang|en|Air Force Office of Security Police}}），1991年8月1日起重編為空軍保安憲兵署（{{lang|en|Air Force Security Police Agency}}），1997年3月17日後則改稱保安部隊中心，地位亦由野戰機構轉變為直屬機構，但由1998年10月1日後又復歸為美國空軍的野戰機構，直至2014年10月1日才取消<ref name="沿革">{{Cite web |script-title=en:Air Force Security Forces Center |publisher={{lang|en|[[美國空軍歷史研究局|Air Force Historical Research Agency]]}} |url=http://www.afhra.af.mil/About-Us/Fact-Sheets/Display/Article/433379/air-force-security-forces-center/ |date=2009-03-10 |accessdate=2017-10-27 |archiveurl=https://archive.is/20171027090301/http://www.afhra.af.mil/About-Us/Fact-Sheets/Display/Article/433379/air-force-security-forces-center/ |archivedate=2017-10-27 |deadurl=yes }}</ref>。
