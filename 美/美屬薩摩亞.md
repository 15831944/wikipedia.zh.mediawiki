'''美屬薩摩亞'''（[[薩摩亞語|薩摩亞語]]：'''Teritori o Amerika Sāmoa'''，{{lang-en|'''American Samoa'''}}）又称'''东萨摩亚'''，是在[[南太平洋|南太平洋]]的[[美國|美國]]非合併非建制領土。最早自公元前1000年就有人定居，[[歐洲|歐洲]]探險家在18世紀到達薩摩亞。首府[[帕果帕果|帕果帕果]]，是太平洋上天然良港之一。面积199平方公里，人口55519人（2010年）。主要的島是[[图图伊拉岛|图图伊拉岛]]（最大和人口最高），其餘島嶼為[[马努阿群岛|马努阿群岛]]、[[罗斯环礁|罗斯环礁]]和[[斯温斯岛|斯温斯岛]]。

19世紀後半叶，[[德国|德国]]、[[英国|英国]]和[[美国|美国]]为争夺[[萨摩亚群岛|萨摩亚群岛]]发生了严重的國際對抗。最后，根据1899年條約規定，[[德國|德國]]和美國分割薩摩亞[[群島|群島]]。次年美國正式佔領群島東半部，由五個火山島與兩個珊瑚礁組成。西薩摩亞現在是[[薩摩亞|薩摩亞]]獨立國。

美国通过1929年2月20日一项国会法案，正式接受将这些岛屿割让给美国的契约。该法案规定，当地居民享有美国国民地位。该法案规定设立一个美属萨摩亚政府，其一切民政、司法和军事方面的权力属于美国总统指定的人。由于美国在该地区的利益主要是军事利益，该领土由美国海军管辖。1951年一项行政命令把对该领土的权力移交给内政部。

2005年5月30日，美屬薩摩亞再次要求[[聯合國|聯合國]]把當地從[[殖民地|殖民地]]名單中剔除。<ref>{{Cite web|url=http://www.ppseawa.org.tw/mambershop_01AmericanSamoa.php|title=會員國介紹Membership--美屬薩摩亞 American Samoa|accessdate=2017-07-11|work=www.ppseawa.org.tw}}</ref>

== 地理 ==
{{main|美属萨摩亚地理}}
美属萨摩亚位于[[大洋洲|大洋洲]]，是美国在[[南半球|南半球]]唯一的领地，土地面积199平方公里（76.8平方英里），比华盛顿市稍大，包含5座崎岖不平的[[火山岛|火山岛]]与2座珊瑚环礁。另外，[[罗斯环礁|罗斯环礁]]，是美国领地的最南端。

在12月到3月间，美属萨摩亚可能受[[颶風|颶風]]袭击。

== 人口 ==
大约有91.6%的人口是[[萨摩亚人|萨摩亚人]]，2.8%是亚洲人，1%白人，4.2%混血，0.3%其他。大部分人具双语言能力，能讲[[英语|英语]]与[[萨摩亚语|萨摩亚语]]。美属萨摩亚基督教徒众多，50%的人口是[[公理會|公理會]]，20%属于[[天主教|天主教]]，30%属于[[基督新教|基督新教]]的其他支派或非基督徒。

美属萨摩亚的州級郵政縮寫是'''AS'''，有一个[[美國郵區編號|美國郵區編號]]：96799。

== 行政区划 ==
{{main|美属萨摩亚行政区划}}
美属萨摩亚分为3个区与2个无建制环礁。3区2环礁又划分为74个村。[[帕果帕果|帕果帕果]]是美属萨摩亚的[[首府|首府]]。它是最大的村庄之一，位于[[图图伊拉岛|图图伊拉岛]]Ma'oputasi县的东部。

== 政治 ==
美属萨摩亚的行政首長為[[美属萨摩亚总督|总督]]，每4年由選舉產生。[[美屬薩摩亞議會|美屬薩摩亞議會]]為兩院制，由參議院和眾議院組成。美屬薩摩亞的議會為一個沒有黨派的議院。21席的眾議院每2年自20個選舉區及[[斯溫斯島|斯溫斯島]]選出，參議院的18席則每4年由美屬薩摩亞的頭目中產生。

出生于美属萨摩亚（含[[斯温斯岛|斯温斯岛]]）的人，是美国国民（national），但不是美国公民（citizen），除非從父母繼承公民權或歸化。

（注：[[斯温斯岛|斯温斯岛]]〔Swains Island〕在1925年3月4日通过合并成为美属萨摩亚的一部分，[[托克劳|托克劳]]独立运动的支持者宣称该岛为托克劳的一部分。）

美属萨摩亚选出一名无投票权的[[美国众议院|美国众议院]]代表（delegate）。

== 经济 ==
岛上的约5000名雇员可分为大体相等的三类：公共部门，两个[[黃鰭鮪|黃鰭鮪]]罐头工厂，其余为私营部分。

公共部门的大多数雇员为美属萨摩亚领地政府工作。两个金枪鱼罐头工厂（StarKist与Samoa Packing）每年輸出美国本土数亿美元的金枪鱼(鲔鱼)罐头。

== 文化 ==
美属萨摩亚的文化与[[萨摩亚|西萨摩亚]]几乎相同。

== 教育 ==
岛上有23个[[小学|小学]]，6个[[中学|中学]]，由[[美属萨摩亚教育部|美属萨摩亚教育部]]经营。

[[美属萨摩亚社区大學|美属萨摩亚社区大學]]，成立于1970年，在岛上提供高等教育。

== 逸事 ==
* 美屬薩摩亞是美國最南的領土，也是美國有人居住領土中唯一屬於南半球的。
* 不論是美國本土還是屬地的所有產品，都可以名正言順的標明“Made in USA”（美國製造），包括美屬薩摩亞的所有產品。
* 現時[[國家美式足球聯盟|國家美式足球聯盟]]有三十位成員來自薩摩亞群島，當中大部份都來自美屬薩摩亞，其餘的散居於美國本土。聯盟球員若按人口比例來看，薩摩亞人比非薩摩亞人高出40倍。
* [[美屬薩摩亞國家足球隊|美屬薩摩亞國家足球隊]]（英式足球）現時保有一個他們並不希望出現的世界紀錄：他們在2001年4月11日以[[澳大利亚31:0美属萨摩亚|0比31]]的懸殊賽果敗於[[澳洲國家足球隊|澳洲國家足球隊]]。
* 所有在美屬薩摩亞出生的人都可享有[[美國國民|美國國民]]（national）的待遇，但不獲[[美國公民|美國公民]]（citizen）的資格。
