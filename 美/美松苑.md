'''美松苑'''（'''Mei Chung Court'''）是[[香港房屋委員會|香港房屋委員會]]的[[居屋|居屋]]屋苑之一，位於[[新界|新界]][[沙田區|沙田區]]的[[大圍|大圍]]美田路33號，與[[美林邨|美林邨]]毗鄰，於1996年落成，屬居屋出售分期的第十六期乙屋苑。

== 設施 ==
屋苑設施包括6個兒童遊樂場、園藝花園、棋藝閣、水池和涼亭。屋苑內設有有蓋行人通道連接各座樓宇。停車場樓高3層，1樓設7-11便利店及自動櫃員機，天台設休憩花園，唯處於荒廢狀態。

== 停車場被水浸事件 ==
1997年7月3日，香港持續暴雨，天文台發出紅色暴雨警告，美松苑停車場嚴重水浸後，汽車積聚大量污泥。

== 停車場加價 ==
2006年12月1日，[[領匯|領匯]]調整轄下43個停車場月租車位的租金，加幅最高是美松苑車位，加幅接近兩成四。美松苑停車場是獨市生意，平均泊車率高達八成，出租率非常高，[[沙田區議會|沙田區議員]]鄧永昌批評領展加幅遠超通脹及薪酬加幅. 美松苑現有380個車位 <small><ref>[http://the-sun.on.cc/channels/news/20061119/20061119023147_0000.html 領匯車場大幅加價]</ref></small>。

== 馬騮為患 ==
美松苑靠近山邊，距離[[猴子|猴子]]聚居的[[金山郊野公園|金山郊野公園]]不遠，加上有居民喜歡餵飼猴子，所以經常有猴子出現。其中[[農曆|農曆]]七月[[盂蘭節|盂蘭節]]前後，有人在地上遺下水果等祭品無人清理，惹來猴群，更曾發生傷人事件<small><ref>[http://www.singpao.com/20070828/local/932536.html 數十馬騮撲倒老翁]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref></small>，管理處每年平均會接到5至6宗居民受猴子滋擾報告，所以保安員需接受[[漁護署|漁護署]]特別培訓，並配備發出刺耳響聲的「蜂鳴器」和木棍驅趕猴子。
