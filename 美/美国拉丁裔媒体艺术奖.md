'''美国拉丁裔媒体艺术奖'''（{{lang-en|'''American Latino Media Arts Award'''}}；简称'''ALMA Award'''）是[[美国|美国]][[拉丁美洲|拉丁美洲]]人的音乐、电视、电影奖项。

== 沿革 ==
1995年，该奖建立，1997年，更名为“美国拉美裔媒体艺术奖”。<ref>{{cite web |url= http://www.mysanantonio.com/sacultura/conexion/article/ALMA-Awards-postponed-831626.php |title=ALMA Awards postponed|work=mysanantonio.com |date= 2010-11-24|accessdate=30 September 2012}}</ref>
