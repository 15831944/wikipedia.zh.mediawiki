《'''美少女戰士'''》（{{lang|ja|美少女戦士セーラームーン}}）是1992年3月7日到1993年2月27日播放的[[日本動畫|日本動畫]]作品。

== 概要 ==
本作品的原作是[[武內直子|武內直子]]所執筆的漫畫《[[美少女戰士|美少女戰士Sailor Moon]]》。由於動畫與漫畫是同時間進行製作的緣故，在設定上有些許的不同。

播出後，[[富士電視系|富士電視系]]節目《[[平成教育委員會|平成教育委員會]]》曾經描述本部作品平均11.6%的[[收視率|收視率]]概況；並且由於動畫版的極高人氣，連帶影響了漫畫版以及續作《美少女戰士R》的第二部故事產生，最後成為了一共五個系列的大作。

本作品的彩色截圖漫畫[[單行本|單行本]]由[[講談社|講談社]]發行，授權[[大然出版社|大然出版社]]發行台灣中文版。

== 故事背景 ==
很久以前，月球上有一個有人居住的地方，稱為月亮王國，因為月球人比地球人長壽，所以招致地球人妒忌。後來因為太陽黑子活動異常，使強大的邪惡能量美達莉亞女王出現，為了征服地球和月球，美達莉亞女王就依附在貝莉爾女王身上，並且利用地球人妒忌月球人長壽的心，向所有地球人洗腦。美達莉亞女王率領貝莉爾女王以及地球人襲擊月球，還突破了月亮王國的最終防衛線，月球上的四位戰士都戰死，貝爾女王妒忌月球人希蕾妮蒂公主與地球人安迪米歐王子一起，決定將他們消滅，安迪米歐王子不敵貝莉爾女王，最後與希蕾妮蒂公主一同死於貝莉爾女王手上。

希蕾妮蒂公主的母親希蕾妮蒂女王見女兒死去，就決定解放銀水晶力量封印美達莉亞女王，同時使月亮王國中死去的人能夠於地球轉生。可是因為希蕾妮蒂女王傷心導致力量不足，使美達莉亞女王得以捲土重來，並成立黑暗王國，目標就是找尋銀水晶，只要銀水晶落入黑暗王國手上，就能夠征服整個地球。

== 故事簡介 ==
主角[[月野兔|月野兔]]是個個性迷糊又愛哭的中學二年級女生，有一天在如同家常便飯的上學遲到趕路途中，救了一隻額頭上有著月印的黑貓'''露娜'''，露娜看著救了她的小兔，似乎起了什麼想法……

一如往常，小兔在學校又接到了不及格的考卷，在與同學奈留以及海野訴苦的時候聊到了最近時常出現打擊邪惡的正義使者'''Sailor V'''，奈留還提到她的媽媽所開的珠寶店裡正展開珠寶特賣會，並且邀請小兔前去看看散心。

小兔與奈留來到奈留媽媽的店中觀賞珠寶，有點喜歡珠寶的小兔本想買下珠寶，但卻礙於零用錢不夠而作罷，奈留建議小兔不妨跟媽媽說說看，小兔聽了之後，想到自己的成績，也只好大嘆三聲無奈。

走出店門口想回家的小兔再次把她那張不及格的考卷拿出來看，有點惱羞成怒的小兔居然把考試卷揉一揉給丟了出去，卻丟到了一個戴著黑框墨鏡的年輕人頭上；只見那年輕人撿起並且翻開被揉成一團的考卷，看到了小兔不及格的分數，調侃了小兔一番；被調侃的小兔一把抓過考卷很生氣走開了，但這時地場衛摘下黑框墨鏡，看著珠寶店，似乎在想些什麼……

離開珠寶店的小兔在大街上看到了有關Sailor V的電動即將推出，心裡憧憬著她，相比之下的自己不禁自慚形穢，於是嘆口氣的繼續著回家的路。這時黑貓露娜再度出現，並且偷偷跟著她回家……

得知小兔考試成績不好的小兔媽媽育子自然是訓了小兔一頓，被罵之後心情不好的小兔回到了房間想休息一下，這時露娜悄悄的溜進了小兔的房間，發現露娜的小兔有點驚訝，此時露娜居然開口說話！以為這是夢境的小兔蒙著頭想回到現實，無奈的露娜只好變出了一個胸針送給了小兔。單純的小兔接過了胸針之後相當高興，並且還照著露娜說的話喊了「咒語」，因此變身的小兔居然變成了……？！
