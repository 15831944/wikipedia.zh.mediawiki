{{Vgname|美國陸軍：真實士兵|en=America’s Army: True Soldiers}}是由[[育碧紅色風暴|育碧紅色風暴]][[游戏开发者|開發]]的[[第一人称射击游戏|第一人称射击游戏]][[电子游戏|游戏]]，[[育碧软件|育碧软件]]在2007年11月17日[[游戏发行商|发行]]。它是[[美国陆军|美国陆军]]遊戲[[美國陸軍_(電子遊戲)|《美國陸軍》]]的[[衍生作品|衍生作品]]。

== 遊戲內容 ==

根據[[IGN|IGN]]的遊戲預告<ref name="announcement">{{cite web |url = http://xbox360.ign.com/articles/820/820752p1.html |title=America's Army: True Soldiers preview |date = 2007-09-18}}</ref>，只要玩家與其他隊員一同合作，即可提升火力的精準度以及反擊力。玩家只能與附近的隊友[[IP电话|通話]]，但是領隊不論遠近皆可用無線電聯絡所有的隊友，而且還可以呼叫火砲支援。一個隊伍有五個作戰單位：步槍兵小隊{{notetag|{{Lang-en|Rifleman}}}}、[[擲彈兵|擲彈兵]]小隊、標定射擊小隊{{notetag|{{Lang-en|Designated marksman}}}}、[[自动步枪|機槍]]小隊、領隊{{notetag|{{Lang-en|Squad Leader}}}}。

在多人連線遊戲，與《美國陸軍》角色是相同的，玩家是扮演[[美国陆军|美国陆军]]。當玩家被敵軍火力擊中而導致受傷，玩家不能執行任何攻擊活動，只能一直倒在地上，直到隊友給予治療才可以活動，遊戲主要是呈現真實的戰場，所以玩家一定要遵守交戰守則{{notetag|{{Lang-en|rules of engagement}}}}。多人遊戲包含的任務有：撤離任務{{notetag|{{Lang-en|reaching extraction points}}}}、防守物件{{notetag|{{Lang-en|defending objectives}}}}、獵殺[[重要人物|VIP]]以及同時防守並持續守住任務點，大多的任務型態都與《美國陸軍三》相同。
