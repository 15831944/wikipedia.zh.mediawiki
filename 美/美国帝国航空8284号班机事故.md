== 飞机历史 ==
执飞8284航班的飞机是一架[[Atr|ATR-42-320]]，1990年由[[ATR|ATR]]公司生产。其注册号是N902FX。这架飞机曾于以下三个航空公司服务：[[Bar_Harbor_Airlines|Bar Harbor Airlines]]，[[美国大陆快运|美国大陆快运]]，[[ExpressJet_Airlines|ExpressJet Airlines]]。该机于2003年交付[[帝国航空|帝国航空]]，同年租借给[[联邦快运|联邦快运]]使用。

== 飞行员记录 ==
操作飞机的机长52岁，男性，总飞行时间13935小时，其中有12742小时此人担任操作飞机的飞行员。他有2052小时的ATR-42飞行经历，其中有1896小时操作飞机。副机长26岁，女性，总飞行时间2109小时。根据帝国航空的记录，她有130小时的ATR-42的副驾驶飞行经历。

== 事故 ==

早晨4:30，8284班机在冻雾中进近[[拉伯克普雷斯顿·史密斯国际机场|拉伯克普雷斯顿·史密斯国际机场]]。<ref name="kcbd">http://www.kcbd.com/global/Story.asp?S=9737826</ref>

在进近时，发生了飞控系统的故障，使得[[襟翼|襟翼]]不能放下。副机长继续进近，同时机长尝试解决[[襟翼|襟翼]]不能放下的问题。<ref name="flightglobal">{{Cite web|url=https://www.flightglobal.com/news/articles/ntsb-pilot-error-in-fedex-atr-42-crash-in-lubbock-356002/|title=NTSB: Pilot error in FedEx ATR 42 crash in Lubbock|accessdate=2018-02-22|date=2011-04-27|last=Croft|first=John}}</ref> 两名机组成员都未注意空速，使得飞机开始以2000英尺（610米）每分钟的速度下降，造成[[地面迫近警告系统|地面迫近警告]]响起。警报响起后17秒，机组开始拉起飞机，造成飞机[[失速|失速]]并撞地。<ref>{{Cite web|url=https://aviation-safety.net/database/record.php?id=20090127-0|title=ASN Aircraft accident ATR 42-320 N902FX Lubbock Preston Smith International Airport, TX (LBB)|accessdate=2018-02-10|last=Ranter|first=Harro}}</ref> 飞机于跑道起点前接地，滑行3300英尺（1000米）并滑出17R跑道。停下后飞机引发了一场小型火灾。<ref>{{Cite web|url=http://www.cnn.com/2009/US/01/27/fedex.crash/index.html?iref=topnews|title=FedEx plane crashes in Texas - CNN.com|accessdate=2018-02-10|language=en}}</ref>

气象条件并不是造成事故的原因。

== 调查 ==
[[NTSB|NTSB]]调查了事故发生的原因。[[飞行数据记录器|飞行数据记录器]]和[[驾驶舱语音记录器|驾驶舱语音记录器]]显示机组成员在[[襟翼|襟翼]]不能放下的情况下选择了继续进近而非[[复飞|复飞]]。机组也没有在失速后立刻将引擎动力加到最大，而只是在近地警报响起17秒后才增大推力。在事故后的采访中，机长称他因工作压力过大而困倦并感到疲劳，影响了他的表现。2011年，在调查完成后，[[NTSB|NTSB]]发布了调查报告。调查人员在报告中称：“NTSB认为，在结冰条件下使用仪表进近时，机组成员未能监视并保持最低安全空速是造成这起事故的可能原因。过低的[[空速|空速]]造成飞机在低高度下发生[[失速|失速]]。”

造成事故的几个原因是：

1) 机组成员未能按照给定的操作程序应对襟翼不能放下的故障。

2) 机长做出的继续进行不稳定进近的决定。 

3) 机组缺少机员资源管理的能力。 

4) 在事故发生时机组成员的疲劳和长期积累的睡眠缺失问题，可能影响了机长的飞行表现。 

== 事故结果 ==
在事故中，编号为N902FX的飞机严重受损并被注销。机组成员受轻伤，故被送往医院，并在之后出院。两位机组成员在1个月后开始继续为[[联邦快运|联邦快运]]飞行。

作为对事故的总结，[[NTSB|NTSB]]提出了9条安全建议，包括防止飞行中结冰的建议。坠机事故使得[[EASA|EASA]]复查飞机的[[震杆器|震杆器]]以防止[[失速|失速]]。<ref>https://www.ntsb.gov/investigations/AccidentReports/_layouts/ntsb.recsearch/Recommendation.aspx?Rec=A-12-027</ref>同时，[[EASA|EASA]]也提出了一条关于在[[飞行模拟器|飞行模拟器]]里模拟结冰条件的规则。<ref>https://www.ntsb.gov/investigations/AccidentReports/_layouts/ntsb.recsearch/Recommendation.aspx?Rec=A-11-046</ref>
