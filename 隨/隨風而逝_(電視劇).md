《'''隨風而逝'''》是 1988 年[[台灣電視公司|台灣電視公司]]製作的一部[[八點檔|八點檔]]國語[[連續劇|連續劇]]，由[[陳冲|陳冲]]、[[魯振順|魯振順]]主演，是[[中國大陸|中國大陸]]開放以來，親赴大陸取景拍攝的劇集之一，劇中空間背景有大陸、臺灣兩地。

本劇是國際巨星陳沖參演的首部臺灣電視連續劇，當時這部電視劇轟動全台，陳沖當時也被譽為[[英國|英國]]國寶級影星[[費雯麗|費雯麗]] ( Vivien Leigh ) 的分身。此後，陳沖先後在台拿下了兩個[[金馬獎|金馬獎]]影后、一個金馬獎最佳導演。

==劇情大綱==
1930 年代 ( 民國二十幾年 )，中國大陸是個山雨欲來風滿樓的時代，對日抗戰前夕，東北到處飄揚著日本國旗。

方思敏 ( 陳沖飾 ) 是留日學醫的學生，義父石原健夫 ( [[曹健|曹健]]飾 ) 為日本滿鐵醫院的院長，義兄石原紀夫 ( [[沈孟生|沈孟生]]飾 ) 對思敏愛慕已久。

思敏回國 ( 中國東北 ) 後，在醫院工作認識了從臺灣來的工程師董其偉 ( 魯振順飾 )。由於其偉的正直厚道，思敏漸與其偉產生了情愫，引起義兄紀夫的嫉妒。某夜，紀夫向思敏告白，不料在爭執中手槍走火，誤傷了紀夫，思敏因此入獄。

董其偉在東北巧遇同事小周的妹妹周家秀 ( [[坣娜|坣娜]]飾 )，家秀對其偉頗有好感。家秀得知其偉的朋友思敏因誤傷人而被關起來，於是暗中斡旋，終於協助思敏出獄。

其偉回臺灣奔父喪，思敏也打算追隨其偉，不料一行人同到臺灣後，思敏赫然發現其偉在臺灣已奉父母之命媒妁訂親，且家人逼迫其偉在百日內完婚，思敏悲傷萬分。適時，思敏的義父調到臺灣醫院工作，母親亦隨行遷到臺灣。其偉訂親的對象翁君蘭 ( [[柯素雲|柯素雲]]飾 )，個性善良，知道其偉心中已有所屬，不想強求，但又不敢違抗父親，終以死來成全他們。

其偉迫於長輩承諾的壓力下，仍娶了君蘭的牌位。思敏哀傷至極，紀夫也正好在此時來臺，給予思敏適時的慰藉。其偉從此陷入憂鬱，性情大變，終日藉酒澆愁，不顧家產事業，不料在一次酒醉中，竟與家秀發生了關係。思敏自此對其偉徹底絕望......
