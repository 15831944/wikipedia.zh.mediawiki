'''勞動部勞動力發展署桃竹苗分署'''是[[勞動部勞動力發展署|勞動部勞動力發展署]]的所屬機構，位於[[桃園市|桃園市]][[楊梅區|楊梅區]]。服務縣市包含：[[桃園市|桃園市]]、[[臺灣省|臺灣省]][[新竹縣|新竹縣]]、[[臺灣省|臺灣省]][[新竹市|新竹市]]以及[[臺灣省|臺灣省]][[苗栗縣|苗栗縣]]，主要提供待業者就業諮詢、職前訓練、在職者的在職訓練、現場考取技術執照的[[全國技術士技能檢定|全國技術士技能檢定]]，以及補助並輔導企業團體辦理職業訓練等服務。

== 歷史 ==
;桃竹苗區就業服務中心<ref>[http://thmr.wda.gov.tw/home.jsp?pageno=201301090013 願景與沿革]{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
*1956年7月，於[[台北市|台北市]]設置「'''台灣省國民就業輔導中心'''」
*1963年7月，建制為「'''台灣省北區國民就業輔導中心'''」 
*1966年7月，改名為「'''台灣省台北區國民就業輔導中心'''」
*1968年4月，遷址[[桃園市|桃園縣]][[中壢區|中壢市]]延平路  
*1986年9月，遷入[[桃園市|桃園縣]][[桃園區|桃園市]]南華街92號現址
*1988年1月，由原隸屬社會處改隸勞工處 
*1994年7月，因修改組織編制及組織規程，更名為「'''台灣省政府勞工處台北區就業服務中心'''」 
*1999年7月，因[[臺灣省政府功能業務與組織調整|臺灣省政府功能業務與組織調整]]，改隸行政院勞工委員會職業訓練局（今[[勞動部勞動力發展署|勞動部勞動力發展署]]），更名為「'''行政院勞工委員會職業訓練局台北區就業服務中心'''」 
*2005年7月，更名為「'''行政院勞工委員會職業訓練局桃竹苗區就業服務中心'''」 

;桃園職業訓練中心 
*1973年7月，於[[桃園市|桃園縣]][[桃園區|桃園市]]設立「'''台灣省政府社會處台北區國民就業輔導中心附設職業訓練中心'''」
*1988年1月，升格為「'''台灣省政府勞工處北區職業訓練中心'''」
*1999年7月，隨精省改隸行政院勞工委員會職業訓練局（今[[勞動部勞動力發展署|勞動部勞動力發展署]]），更名為「'''行政院勞工委員會職業訓練局北區職業訓練中心'''」
*1999年9月，因配合工業發展需要，擴大訓練職類及容量，遷入[[桃園市|桃園縣]][[楊梅區|楊梅市]]秀才路851號現址

;青年職業訓練中心
*1970年1月，於[[裕隆汽車|裕隆汽車]]公司大坪林工廠內設立「'''第一青年職業訓練中心'''」，後設立於[[桃園市|桃園縣]][[楊梅區|楊梅市]][[幼獅工業區|幼獅工業區]]，佔地4,600餘坪。
*1986年1月，更名為「'''行政院青年輔導委員會青年職業訓練中心'''」 
*2013年1月，改隸行政院勞工委員會職業訓練局（今[[勞動部勞動力發展署|勞動部勞動力發展署]]），更名為「'''行政院勞工委員會職業訓練局青年職業訓練中心'''」

== 交通 ==
===自行開/騎車===
自楊梅往新埔方向前往本分署。

中山高速公路楊梅交流道下前往。 

===火車=== 
請至新竹客運楊梅站（火車站前圓環左轉前行100公尺左側）搭乘往新埔（經清水）方向至【職訓中心站】下車 (右方見到中華汽車後即可拉鈴於下一站下車)。 

新竹客運楊梅站發車時間（去）：
 
07：10 / 07：40 / 15：20 / 18：30

新竹客運新埔站發車時間（回）：

06：30 / 08：20 / 16：20 / 17：50 

===計程車===
由楊梅火車站至本分署，車資約130元左右。 

有關專車往返【桃竹苗分署】 

1. 發車時間： 

上午8時20分由楊梅火車站開往本分晵行政大樓；下午17時40分由本分署行政大樓開往楊梅火車站。 

2. 搭車地點：

新竹客運楊梅站（楊梅火車站圓環前20公尺）

本分署行政大樓前 

3. 車資費用：

投現 26元
 
悠遊卡 23元
