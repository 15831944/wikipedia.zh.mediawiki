'''勞加德體育場'''（{{lang-is|'''Laugardalsvöllur'''}}、{{IPA-is|ˈløiːɣar.talsˌvœtl̥ʏr̥}}）是位於[[冰島|冰島]][[雷克雅維克|雷克雅維克]]{{le|勞加達魯爾|Laugardalur}}的多功能體育場，同時是冰島的[[國家體育場|國家體育場]]。目前該體育場是[[冰島國家足球隊|冰島國家足球隊]]的主場，場內共有9,800個席位，最多可容納15,000人。

== 歷史 ==
在{{le|勞加達魯爾|Laugardalur}}建造一座包含一些娛樂設施與運動場地的提議可追溯至1871年，然而，當時[[冰島|冰島]]首都[[雷克雅維克|雷克雅維克]]的人口僅2,000人左右，加上勞加達魯爾距當時首都圈的住宅區至少{{convert|3|km}}，因此在接下來的60年內，這個提議幾乎沒有再出現過。1943年，市議會決定成立勞加達魯爾委員會，其任務主要為該地區發展提出建議和想法。委員會隨後交付一份報告，其中包括建造一座新足球場和{{le|勞加德游泳池|Laugardalslaug|游泳池}}。<ref>{{Cite web |url=http://heyevent.com/venue/jpmzo7zxrijvoa |title=Laugardalsvöllur |website=heyevent.com |language=en |access-date=2018-07-11 |deadurl=no |archiveurl=https://web.archive.org/web/20180712091719/http://heyevent.com/venue/jpmzo7zxrijvoa |archivedate=2018-07-12 }}</ref>

體育場的建造提案後來於1949年開始執行，1952年時已初步完成。1953年起開始建造看臺設施，預計完工後可容納4,000人。勞加德體育場最後於1959年6月17日正式開幕，但第一場在勞加德體育場舉行的比賽是在1957年，當時由[[冰島國家足球隊|冰島]]對[[挪威國家足球隊|挪威]]。自1965年到1970年間，球場看台重新進行翻修和擴建。<ref>{{Cite web|url=http://2012.skjaladagur.is/?p=488|title=Íþróttaleikvangur Reykjavíkur í LaugardalSaga framkvæmdanna|publisher=[[冰島足球協會|冰島足球協會]]|date=2012-11-06|language=is|access-date=2016-09-26|deadurl=no|archiveurl=https://web.archive.org/web/20160927073849/http://2012.skjaladagur.is/?p=488|archivedate=2016-09-27}}</ref>新的球場看台下方另開設一座[[田賽和徑賽|田徑設施]]，但這些設施直到2007年才全面營運，不過球場周圍的[[跑道_(體育)|全天候跑道]]則是一開始就設立，並在1992年時升級。1992年後期，球場周圍加設泛光燈，並在冰島與[[希臘國家足球隊|希臘]]的比賽中投入使用。<ref>{{Cite web |url=https://www.inspirock.com/iceland/reykjavik/laugardalsvollur-stadium-a2322726135 |title=Laugardalsvollur Stadium, Reykjavik |website=inspirock.com |language=en |access-date=2018-07-11 |deadurl=no |archiveurl=https://web.archive.org/web/20180712091719/https://www.inspirock.com/iceland/reykjavik/laugardalsvollur-stadium-a2322726135 |archivedate=2018-07-12 }}</ref>

1997年，原先的舊看台對面新建另一座新看台。在新看台建造前一年，體育場的容量為3,500人。與此同時，透過新看台的投入使用，原先舊看台的容納人數也可稍微緩解。<ref>{{Cite web|language=is|url=http://www.mbl.is/greinasafn/grein/343832/|title=Ártöl í sögu knattspyrnunnar á Íslandi 1892 "Knattleikur er t.d. flestra unglinga yndi og mætti tíðka hann miklu meira en gert|website=www.mbl.is|access-date=2016-09-26|deadurl=no|archiveurl=https://web.archive.org/web/20170127224151/http://www.mbl.is/greinasafn/grein/343832/|archivedate=2017-01-27}}</ref>

2005年，舊看台開始進行大規模翻新及擴建，在2007年建成後，體育場的容納人數提升至9,800人。比賽時，可另外架設2座臨時看台，每座臨時看台可容納1,500人，可讓體育場容納人數加大至15,000，但在[[國際足總|國際足總]]禁止舉辦國際賽事的足球場架設臨時看台後，這種作法被禁止。自2007年以來，勞加德體育場沒有過重大改建或擴建。<ref>{{Cite journal|language=is|last=Rúnarsson|first=Kristján Jens|date=2016-01-01|title=Stækkun Laugardalsvallar : Er skynsamlegt fyrir Reykjavíkurborg að taka þátt í hugsanlegum breytingum?|url=https://skemman.is/handle/1946/25150|deadurl=no|archiveurl=https://web.archive.org/web/20180712091720/https://skemman.is/handle/1946/25150|archivedate=2018-07-12}}</ref>在2004年的冰島對[[義大利國家足球隊|義大利]][[熱身賽|友誼賽]]中，進場人數達20,204人。2007年的一場音樂會上，進場人數更是達到25,000人。<ref name="KSI">{{Cite web|language=is|url=http://gamli.ksi.is/mannvirki/laugardagsvollur/|title=Þjóðarleikvangur Íslands {{!}} Knattspyrnusamband Íslands|website=www.ksi.is|access-date=2016-09-26|deadurl=no|archiveurl=https://web.archive.org/web/20180712091720/http://gamli.ksi.is/mannvirki/laugardagsvollur/|archivedate=2018-07-12}}</ref>

== 設施 ==
勞加德體育場包括兩座大看台。西側看台是主要看台，也是較大的看台。體育場內設有有4間主更衣室供比賽選手使用，2間則是供裁判員使用，另外也有供醫生來進行藥物檢驗提等設備。建築內有足球比賽及田徑錦標賽的管理室。<ref name="KSI"/>

== 未來 ==
自從[[冰島國家足球隊|冰島國家足球隊]]在2013年打進[[2014年國際足總世界盃外圍賽_–_歐洲區第二圈|2014年國際足總世界盃外圍賽歐洲區第二圈]]賽事時，對於勞加德體育場進行翻新和擴建的計畫已被提出。冰島國家足球隊後來甚至打到[[2016年歐洲國家盃|2016年歐洲國家盃]]半準決賽，這迫使冰島足球迷及[[冰島足球協會|冰島足球協會]]迫切尋求新的體育場，因為自2013年以來幾乎所有官方比賽或是友誼賽的座位票數都已售罄。擴建計畫中有數種版本，但至少有移除跑道並加裝{{le|可伸縮上蓋|Retractable roof}}這2個共同點。冰島足球協會表示，它們需要一座擁有可伸縮上蓋的體育場，讓國家擁有第一座{{le|多用途體育場|Multi-purpose stadium}}體育場能夠增加用途，這也將提高體育場的座位量。<ref>{{Cite web |url=http://stadiumdb.com/news/2014/10/new_design_mostly_new_national_stadium_of_iceland |title=New design: (Mostly) new national stadium of Iceland |website=StadiumDB.com |language=en |date=2014-10-16 |access-date=2018-07-11 |deadurl=no |archiveurl=https://web.archive.org/web/20180712091720/http://stadiumdb.com/news/2014/10/new_design_mostly_new_national_stadium_of_iceland |archivedate=2018-07-12 }}</ref>歐洲有數座體育場可以為冰島作為借鏡，這些體育場都很大，但在建造成本上也非常昂貴。例如，新的勞加德體育場預計耗資80億[[冰島克朗|冰島克朗]]，相當於6200萬[[歐元|歐元]]。如果建成，這座新的足球場容納人數預計達20,000人以上。<ref name="news1">{{Cite news|language=is|url=https://island.fotbolti.net/news/19-10-2017/riki-og-borg-vilja-vinna-med-ksi-ad-nyjum-laugardalsvelli|title=Ríki og borg vilja vinna með KSÍ að nýjum Laugardalsvelli|date=2017-10-19|publisher=Fotbolti|access-date=2018-07-10|archive-url=https://web.archive.org/web/20180712120225/https://island.fotbolti.net/news/19-10-2017/riki-og-borg-vilja-vinna-med-ksi-ad-nyjum-laugardalsvelli|archive-date=2018-07-12|dead-url=yes}}</ref>

在新的建造案下，原有的跑道將被移除，體育場的底部邊緣將更靠近中間的足球場，球場本身的地面也須將挖深數公尺，才能擴張容納人數。新的體育場尚未決定草皮種類，若是選擇天然草皮，必須加裝加熱器來抵禦冰島的寒冬氣候，以便來符合[[歐足聯|歐足聯]]的新需求；這表示一部分在冬季舉行的[[歐洲國家盃|歐洲國家盃]]{{le|歐洲國家盃資格賽|UEFA European Championship qualifying|資格賽}}也可如期舉行。<ref>{{Cite news|language=is|url=http://www.ruv.is/frett/hotel-og-haskoli-a-nyjum-laugardalsvelli|title=Hótel og háskóli á nýjum Laugardalsvelli?|date=2016-09-26|newspaper=RÚV|access-date=2016-10-07|deadurl=no|archiveurl=https://web.archive.org/web/20161010110045/http://www.ruv.is/frett/hotel-og-haskoli-a-nyjum-laugardalsvelli|archivedate=2016-10-10}}</ref>為了加裝可伸縮上蓋，現有的體育場建築將會被拆除，新建築外型為橢圓體，周圍另設有支架支撐可伸縮上蓋的軌道，確保體育場頂蓋能完全打開及關閉。<ref name="news1"/>
