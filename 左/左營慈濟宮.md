'''城邑左營慈濟宮'''，簡稱城邑慈濟宮、左營慈濟宮，當地人俗稱'''老祖廟'''，是一間位於[[高雄市|高雄市]][[左營區|左營區]]的[[廟宇|廟宇]]，主祀[[保生大帝|保生大帝]]。

==歷史沿革==
城邑慈濟宮奉祀主神乃是保生大帝，相傳乃明鄭元帥劉國軒，祀奉在戰艦上之保護神。鄭氏登臺後，奉祀於左營萬丹港南岸（現今內惟桃子園，西元1680年鄭經入臺，天行疫癘，將士們損失過半，遂飛船回閩，自青礁祖廟迎回十尊保生大帝聖像，原本的保生大帝化後入神其中，分祀於安平鎮和萬年縣的十角落，翌年(西元1681年)，大疫竟息。

據[[康熙|康熙]]五十八年（1719年）《鳳山縣志》記載，慈濟宮建廟位於興隆莊北。乾隆二十九年（1764年）《重修鳳山縣志》載，建慈濟宮在縣署前大街。道光五年（1825年）鳳山舊城完工，當時的總兵陳瓊號召了地方士紳重新整修慈濟宮。

日治昭和十二年（1937年）建左營軍港，舊城畫為要塞，慈濟宮的保生大帝神像被迫遷出，暫安奉於在內惟鎮安宮。後遷新莊黃宅，再遷埤子頭陳宅。民國三十九年轉購於埤子頭西巷九號，迎回大帝金身奉祀。民國四十八年（1959年）於現在位址蓮潭路九號，左營地方仕紳集資重建城邑慈濟宮，並且成立龍虎社修善堂扶鸞濟世，至今香火鼎盛，成為當地居民守護神。民國六十三年（1974年）奉保生大帝意旨，興建慈濟宮龍虎塔於蓮潭畔，名聞於世，遂成為觀光、宗教勝地。民國八十九年（西元2000年）再奉保生大帝旨，重建慈濟宮並整修龍虎塔，於民國九十三年（西元2004年）四月十六日破土重建，至民國九十七年（西元2008年）四月初一日入火安座大吉。甲午年(西元2014年~2015年)十一月初八日起欽奉玉旨舉行一朝慶成謝土暨三朝祈安清醮；同月16日前往安平北岸恭請聖火回駕遶境。

1963年，[[陳玉峰_(彩繪匠師)|陳玉峰]]曾到此廟繪製廟畫<ref>{{cite book |author=南部寺廟調查小組|title=《臺灣南部寺廟調查曁研究報告》 |date=1997|publisher=國立成功大學歷史學系|location=台灣|language=zh-tw|isbn=}}</ref>。

==奉祀神明==
主祀[[保生大帝|保生大帝]]，同祀朱府千歲（[[鄭成功|鄭成功]]）、曹府千歲（[[曹謹|曹謹]]），陪祀[[註生娘娘|註生娘娘]]、[[福德正神|福德正神]]，配祀[[虎爺|黑虎將軍]]、馬使爺；三樓太歲殿主祀[[太歲星君|太歲星君]]，文昌殿主祀[[文昌帝君|文昌帝君]]；附設龍虎塔奉祀[[孔夫子|孔夫子]]和[[地藏王菩薩|地藏王菩薩]]。

==大道公廟十角落==
據康熙五十八年（西元1719年）《鳳山縣志》記載，大道公廟十角落
# 在興隆莊北:城邑慈濟宮
# 在鳳山上莊:林園興濟宮
# 在硫磺水莊:威靈宮
# 在半屏山後:後勁聖雲宮
# 在安平鎮:安平妙壽宮、安平廣濟宮、安平靈濟殿
# 在維新里竹子港:永安保寧宮
# 在觀音山大社:大社保元宮
# 在土鑿埕:府城四安境下大道良皇宮
