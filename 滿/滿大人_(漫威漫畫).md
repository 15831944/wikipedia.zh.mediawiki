'''滿大人'''（{{lang-en|Mandarin}}），本名汗（Khan）是一位在驚奇漫畫世界中的虛構漫畫超級反派，由[[史丹·李|史丹·李]]、[[拉里·李伯|賴瑞·理柏]]、{{tsl|en|Don Heck|唐·赫克}}與[[傑克·科比|傑克·科比]]共同創造。

滿大人首次出現於《懸疑故事》（Tales of Suspense）第50期（1964年二月），他是[[鋼鐵人|鋼鐵人]]裡主要的敵人跟最大的敵手。

2009年，IGN將滿大人列為漫畫史上百大反派中的第81名<ref>[http://comics.ign.com/top-100-villains/81.html "Mandarin is number 81"] {{webarchive|url=https://web.archive.org/web/20131103090304/http://comics.ign.com/top-100-villains/81.html |date=2013-11-03 }}. [[IGN|IGN]]. Retrieved 10–05–09.</ref>。

==故事背景==
滿大人出生於[[中華民國大陸時期|民國初年]]，約1920年左右，他擁有[[成吉思汗|成吉思汗]]的血統，而母親則是一名[[英國|英國]][[貴族|貴族]]。

滿大人在出生不久後父母便雙亡，後來在他的叔父養育下長大，而滿大人自幼便接受[[科學|科學]]與[[武術|武術]]的訓練，因之後的[[中共建政|時局變動跟戰亂]]，使得滿大人失去了擁有的一切，因此也讓他開始立志要報復社會和摧毀[[人類文明|人類文明]]。

到了現代，滿大人决定捲土重來再創輝煌，不過時代卻早已不屬於他，但這卻沒能讓他放棄自己的野心。

他手上持有共十枚，擁有不同魔力的戒指，而十枚戒指則分別戴在十根手指上，滿大人也是個名符其實富可敵國的人，且他更利用了自己的戒指收編了好幾個[[越南|越南]][[傭兵|傭兵]]團。

之後，滿大人便開始盯上了[[東尼·史塔克|東尼·史塔克]]，且還利用東尼·史塔克前往越南視察自己公司的晶片時，在旅程中绑架他以做為日後统一世界的最大資金來源。

但東尼·史塔克被绑架後，卻也因禍得福的成為了[[鋼鐵人|鋼鐵人]]，滿大人則因此而感到極度的憤怒，並誓言要殺了東尼·史塔克，於是滿大人便開始秘密策劃一項足以顛覆世界的大陰謀，而得知滿大人邪惡野心的鋼鐵人，也在這之後開始了與滿大人一場又一場驚心動魄的死鬥。

==能力==
滿大人是一個武術大師，窮究武術奧秘的他更已掌握了氣，配合他強悍的武術技巧甚至可以徒手拆解掉鋼鐵人的鋼鐵裝。

除此之外他也是漫威漫畫世界觀中，地球上的天才科學家之一。

滿大人除了本身強悍外，他更有蒐集到的十枚具有神秘力量的魔法戒指。

; 心靈增幅器
: 這個戒指可以增幅持有者的心理能量，讓單一或多數的人聽從他的指揮或把他的想法視作現實。滿大人一般用它創造幻覺。雕刻其上的寶石是顆藍色寶石。
|
; 切割光束
: 這個戒指能發射破壞一切物質、分子、元素間鏈結的能量束，每次使用之間必須有20分鐘的冷卻時間。雕刻其上的寶石是金黃色的方塊。

; 閃電風暴
: 這個戒指能夠催動電流，電流的能量上限尚不明。雕刻其上的寶石是綠色的折線狀。
|
; 渦流
: 這個戒指可以催動空氣形成高速旋轉的旋風。創造出來的旋風可以作為武器使用、可以托起物體、也可以用作運輸戒指持有者的移動手段。雕刻其上的寶石是藍色的圓圈狀.

; 火焰爆炸
: 该指环发射的红外辐射（也就是热量）由穿用者精神上确定强度。一般来说，这种热量通过灼烧爆炸范围内的空气分子来释放火焰。可以使用的热束触发化学爆炸。它可以生成的最大热量仍是未知。宝石与“干冰喷射环”有着相似的形状，但具有四个胶囊、而且是红色的。
|
; 打激光束
: 这个环可以投影各种形式的能量，最常见的是快中子与巨大的震荡力。该环也被用来投射强烈的声波振动和创建吸引或排斥的对象的电磁波。该环也许也能发射其他形式的能量，而且它已被用于发送钢铁侠通过一座山丘。宝石的形状像一颗星星，呈紫色（或红色）。
|-
| [[拇指|拇指]]
|
; 白光
: 此环可以沿电磁波谱发出各种形式的能量。它被用来创建强大的引力，足以让钢铁侠自己把自己埋进土里。宝石的形状像两个白色的圆圈，镶在金子上（就像是一对眼睛）。
|
; 物质重组器
: 这个环可以重新排列的物质的原子和分子，或加快或减慢他们的运动，从而产生不同的效果。环已被用于水蒸汽凝结空气中的液态水，凝华气体，创造含有致命毒气的空气，将一群男性石化，把一个人变成甲虫，把山变成一头岩石怪物，并从地上制作出一只石手来握住钢铁侠。环不能改变元素或令铁人的磁束增强装甲的原子和分子重新排列。宝石的形状像一个小的紫色圆圈。
|}
