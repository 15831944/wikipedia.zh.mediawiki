{{日本作品|滿溢的水果塔|おちこぼれフルーツタルト}}是日本漫畫家[[濱弓場雙|濱弓場雙]]的[[四格漫畫|四格漫畫]]作品，於[[芳文社|芳文社]]的雜誌《[[Manga_Time_Kirara_Carat|Manga Time Kirara Carat]]》2015年1月號至3月號進行短期連載<ref>{{cite web|url=http://www.dokidokivisual.com/magazine/carat/book/index.php?mid=417|script-title=ja:まんがタイムきららキャラット 2015年1月号|publisher=[[芳文社|芳文社]]|accessdate=2018-01-30|language=ja}}</ref><ref>{{cite web|url=http://www.dokidokivisual.com/magazine/carat/book/index.php?mid=427|script-title=ja:まんがタイムきららキャラット 2015年3月号|publisher=[[芳文社|芳文社]]|accessdate=2018-01-30|language=ja}}</ref>，並於2015年7月號正式開始連載<ref>{{cite web|url=http://www.dokidokivisual.com/magazine/carat/book/index.php?mid=447|script-title=ja:まんがタイムきららキャラット 2015年7月号|publisher=[[芳文社|芳文社]]|accessdate=2018-01-30|language=ja}}</ref>。已發行3冊[[單行本|單行本]]，中文版由台灣[[東立出版社|東立出版社]]發行，已發行1冊<ref>{{cite web|url=http://www.tongli.com.tw/BooksList.aspx?Page=1&B=HA0130|script-title=zh:滿溢的水果塔|publisher=[[東立出版社|東立出版社]]|accessdate=2018-01-30|language=zh}}</ref>。

於《Manga Time Kirara Carat》2019年5月號發表了製作電視動畫的消息，由本社位於本作品舞台[[小金井市|小金井市]]的[[feel.|feel.]]製作<ref>{{cite news|url=https://natalie.mu/comic/news/325626|script-title=ja:「おちこぼれフルーツタルト」TVアニメ化、川口敬一郎監督で制作はfeel.|publisher=[[Natalie_(網站)|Comic Natalie]]|accessdate=2019-03-28|language=ja}}</ref>。

==故事簡介==
憧憬著成為偶像而來到東京的高中一年級女孩櫻衣乃，搬進了小白鼠製作公司第4宿舍，通稱「老鼠莊」，卻在這裡得知了宿舍即將被拆除的消息。為了阻止老鼠莊被拆除，衣乃與老鼠莊的其他成員決定挑戰經紀人的企劃「滿溢的水果塔」而一同組成了偶像團體「水果塔」。

==登場角色==
登場人物名字的首字來自於[[伊呂波歌|伊呂波歌]]。

===水果塔成員===
; {{nihongo|櫻衣乃|桜 衣乃（さくら いの）}}
: 高中一年級生，在選拔中候補合格成為偶像。
: 擅長帶著笑容與人聊天交際。在意自己有點粗的大腿。

; {{nihongo|關野露子|関野 ロコ（せきの ロコ）}}
: 高中二年級生，前童星。
: 7年前以「花椰子」的身分演出廣告大受歡迎，但本人對於過了7年仍擺脫不掉花椰子的形象一事感到不滿，同時也很在意自己的身高過矮。

; {{nihongo|貫井羽優|貫井 はゆ（ぬくい はゆ）}}
: 高中一年級生，前音樂家。
: 個性開朗，喜歡搖滾樂與英雄類的作品，但亦有著充滿少女情懷的一面。

; {{nihongo|前原仁菜|前原 仁菜（まえはら にな）}}
: 高中二年級生，前時裝模特兒。
: 個性內向，有著適合當模特兒的身高與良好的身材，卻缺乏自信。

; {{nihongo|綠黑萌|緑 へも（みどり へも）}}
: 國中三年級生。
: 在貓貓製作的徵選獲得第一名的新人，事後因本人的願望加入了小白鼠製作公司成為水果塔的成員。

; {{nihongo|梶野穗步|梶野 穂歩（かじの ほほ）}}
: 25歲，水果塔的經紀人。

===冰淇淋蜜糖紅豆成員===
所屬於大製作公司「貓貓製作」的人氣第一名年輕偶像團體。

; {{nihongo|關野知子|関野 チコ（せきの チコ）}}
: 國中三年級生，露子的妹妹。很喜歡姐姐。跟黑萌是同班同學。

; {{nihongo|中町縫愛|中町 ぬあ（なかまち ぬあ）}}
: 高中二年級生。[[抖M|抖M]]。與瑠愛是雙胞胎。

; {{nihongo|中町瑠愛|中町 るあ（なかまち るあ）}}
: 高中二年級生。[[抖S|抖S]]。與縫愛是雙胞胎。

; {{nihongo|東莉莉|東 リリ（ひがし リリ）}}
: 25歲，冰淇淋蜜糖紅豆的製作人。

===其他人物===
; {{nihongo|本町 利音|本町 利音（ほんまち とね）}}
: 高中一年級生，衣乃與羽優的同班同學。在看過了水果塔的出道演唱會後便成為水果塔的粉絲，並以提出企劃為契機成為助理。

; {{nihongo|小金乙|小金 乙（こがね おと）}}
: 19歲，貓貓製作的代理社長。
