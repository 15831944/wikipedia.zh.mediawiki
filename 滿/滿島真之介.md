'''滿島真之介'''（{{Lang-ja|満島 真之介}}，{{bd|1989年|5月30日|||Mitsushima Hikari}}），[[日本|日本]]男[[演員|演員]]，[[沖繩縣|沖繩縣]][[沖繩市|沖繩市]]出身。滿島真之介有四分之一外國血統，姊姊為演員[[滿島光|滿島光]]。

2012年，滿島真之介以電影《{{link-ja|11·25自決之日|11・25自決の日 三島由紀夫と若者たち}}》獲[[報知電影獎|報知電影獎]]和日本放送電影藝術大獎最佳新人獎。

== 經歷 ==
滿島真之介高中畢業後，從家鄉[[沖繩|沖繩]]來到[[東京|東京]]，在東京的[[幼稚園|幼稚園]]工作<ref>{{Cite web |date=2013-01-04 |url=http://trendy.nikkeibp.co.jp/article/column/20121227/1046637/ |title=【NEXT BREAKERS】満島真之介～『梅ちゃん先生』で注目された個性派 |publisher=日経トレンディネット |accessdate=2013-11-09}}</ref><ref>{{Cite web |date=2013-11-08 |url=http://www.cinematoday.jp/page/N0057914 |title=『風俗行ったら人生変わったwww』で主演の満島真之介、前職は保育士！子どもにも観てもらいたい!? |publisher=シネマトゥデイ |accessdate=2013-11-09}}</ref>。2009年，在電影《{{link-ja|悄悄告訴你|ちゃんと伝える}}》劇組擔任導演的助手。

在東京工作約2年後，2009年5月至11月，自[[北海道|北海道]]到[[鹿兒島縣|鹿兒島縣]]以[[單車|單車]]環繞日本<ref>[http://www.sankei.com/west/news/150207/wst1502070009-n3.html “愛され男子”満島真之介　姉・ひかりと若松・蜷川の２巨匠を語る]</ref>。原本對演藝界興趣不大的滿島真之介，在沿途中，因為多次看見姊姊演出的電影[[海報|海報]]、[[雜誌|雜誌]]及聽見他人口中提及的姊姊而受到影響，決定從事演員工作<ref name="sportshouchi">{{cite news|url=http://hochi.yomiuri.co.jp/entertainment/news/20101013-OHT1T00057.htm |title=満島ひかりの弟・真之介が俳優デビュー…姉の活躍に刺激 |publisher=スポーツ報知 |date=2010-10-13 |archiveurl=https://archive.is/20101013013435/http://hochi.yomiuri.co.jp/entertainment/news/20101013-OHT1T00057.htm |archivedate=2010-10-13 |accessdate=2014-02-12 |deadurl=bot: unknown }}</ref>。同年12月，進入經紀公司{{link-ja|Humanité|ユマニテ (企業)}}<ref name="sportshouchi"/>。

2010年，在[[東京藝術劇場|東京藝術劇場]]演出[[舞台劇|舞台劇]]《おそるべき親たち》，以劇中[[中嶋朋子|中嶋朋子]]的男友一角正式出道<ref name="sportshouchi"/><ref>{{cite news|url= http://www.oricon.co.jp/news/80986/full/|title= 女優・満島ひかりの弟・真之介が舞台で“俳優”デビュー|publisher= ORICON STYLE|date= 2010-10-13|accessdate= 2014-02-12}}</ref>。翌年，以人氣電視劇《[[花樣少年少女_(2011年電視劇)|花樣少年少女]]》中的「天王寺」一角開始受到年輕觀眾的注目。2012年，演出NHK[[連續電視小說|晨間連續劇]]《[[小梅醫生|小梅醫生]]》，知名度開始上升。

2013年11月9日，初次主演電影《{{link-ja|风俗店之行改变了我的人生|風俗行ったら人生変わったwww}}》<ref>[http://gb.cri.cn/27564/2013/07/12/3465s4180197.htm 滿島真之介牽手佐佐木希 新片扮處男「買春」]</ref><ref>{{Cite news|title=満島真之介、『風俗行ったら人生変わったwww』で映画初主演！ヒロインの風俗嬢に佐々木希。|publisher=cinematoday|date=2013-07-11|url=http://www.cinematoday.jp/page/N0054648|accessdate=2013-08-01}}</ref>。

== 人物 ==
*[[左撇子|左撇子]]。<ref>《act+》雜誌2015年1月號[[滿島光|滿島光]]專訪、[[NHK|NHK]]「仕事ハッケン伝」滿島真之介專訪</ref>
*祖父為[[法裔美國人|法裔美國人]]，雙親皆為國中體育教師，受父母影響，運動萬能。興趣是[[攝影|攝影]]、[[烹飪|烹飪]]和騎[[單車|單車]]。滿島真之介在四個兄弟姊妹之中排行第二（姊姊是演員[[滿島光|滿島光]]，妹妹是平面[[模特兒|模特兒]][[滿島南|滿島南]]）。
*2014年年底（24歲），和姐姐[[滿島光|滿島光]]的經紀人結婚。<ref name="sina"/>
*騎單車環日本期間，擔任最北端的北海道[[稚內市|稚內市]]「稚內市故鄉大使」<ref>{{Cite web|date=2009-08-29|url=http://www.welcome.wakkanai.hokkaido.jp/topics/2009/08/post-17.html|title=「稚内市ふるさと大使」200人達成！|publisher=稚内観光協会|accessdate=2013-08-01|deadurl=yes|archiveurl=https://web.archive.org/web/20131109153108/http://www.welcome.wakkanai.hokkaido.jp/topics/2009/08/post-17.html|archivedate=2013年11月9日|df=}}</ref>。途中亦結識不同地方的當地人，其後仍以[[筆友|筆友]]的方式保持聯繫<ref name="inlifeweb.com">[http://www.inlifeweb.com/reports/report_4213.html 男の履歴書　満島真之介編｜インライフ]</ref>。2012年，拍攝[[NHK|NHK]]晨間劇《[[小梅醫生|小梅醫生]]》時，收到許多從日本各地寄來的信件<ref>{{cite web|date= 2013-11-07|url= http://www.tbs.co.jp/radio/cycle-r/ps/20131107.html|title= 自転車日本一周したら人生変わったwww|work=ミラクル・サイクル・ライフ|publisher= TBS RADIO|accessdate= 2014-02-12}}</ref>。
