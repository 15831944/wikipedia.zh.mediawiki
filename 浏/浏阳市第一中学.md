'''浏阳市第一中学'''（简称'''浏阳一中'''）位于[[湖南省|湖南省]][[浏阳市|浏阳市]][[圭斋路|圭斋路]]81号，为浏阳市公立学校，湖南省示范性中学之一。<ref name="fish " >{{cite web |url=http://www.lyyzedu.com/Article/Index.asp |title=三湘名校——浏阳市第一中学 |accessdate=2012 |publisher= 浏阳一中官方网站|work= |language =zh-cn}}</ref>

== 历史 ==
1929年，浏阳创建了“浏阳公学”，它是浏阳一中的前身，创建人是县长[[魏明道|魏明道]]，分为男校区和女校区，男校区在浏阳城关磨石街蜈蚣岭红十字会旧址，而女校区在柴家巷原迎佛寺甲种女子师范旧址。9月，学校改名为“浏阳县立初级中学”。<ref name="fish " /><ref name=" fox" >{{cite web |url=http://www.lyyzedu.com/Article/ShowArticle.asp?ArticleID=627 |title=校名沿革 |accessdate= 2009|publisher=浏阳一中官方网 |work= |language =zh-cn}}</ref>

1936年，学校改名为“浏阳县立简易乡村师范”。<ref name=" fox" />

1942年，学校改名为“简师附设初中”。<ref name=" fox" />

1945年，学校改名为“浏阳县立初级中学”。<ref name=" fox" />

[[抗日战争|抗日战争]]和[[国共内战|国共内战]]时期，学校9次更名，5次迁址，2次停办。<ref name="fish " />

1949年，学校迁址到浏阳文庙，改名为“浏阳县立中学”。<ref name="fish " /><ref name=" fox" />

1952年，学校改名为“浏阳县第一初级中学”。<ref name=" fox" />

1955年，学校正式改名为“湖南省浏阳县第一中学”。<ref name="fish " /><ref name=" fox" />

1968年，学校改名为“浏阳县八二五学校”。<ref name=" fox" />

1969年，学校改名为“湖南省浏阳县第一中学”。<ref name=" fox" />

1992年，学校改名为“湖南省浏阳市第一中学”，并且一直沿用至今。<ref name=" fox" />

== 建筑 ==
=== 文庙 ===
文庙创建于1208年（[[南宋|南宋]][[嘉定|嘉定]]元年），原址在浏阳县东门外红狮桥。1818年（[[清朝|清朝]][[嘉庆|嘉庆]]二十三年），迁址。1843年（[[道光|道光]]二十三年），增加建筑崇圣祠、大成殿、奎文阁。1950年，文庙被用作浏阳县立中学校的校址。1986年，文庙转变为浏阳市博物馆。文庙占地面积6000多平方米。为木结构清朝建筑。<ref>{{cite web |url=http://www.lyyzedu.com/Article/ShowArticle.asp?ArticleID=606 |title=江南第一文庙——浏阳文庙 |accessdate=2009-03-21 |publisher=浏阳一中官方网 |work= |language =zh-cn}}</ref>

=== 谭嗣同新算学社 ===
嗣同新算学社原本叫“奎文阁”，始建于1897年，创立者是[[谭嗣同|谭嗣同]]、[[唐才常|唐才常]]。1996年，“嗣同新算学社”被公布为“省级文物保护单位”，2002年，“嗣同新算学社”评选为“长沙市爱国主义教育基地”。嗣同新算学社坐北朝南，是砖木结构建筑，房屋有7间，占地面积400平方米，建筑面积200余平方米。<ref>{{cite web |url=http://www.lyyzedu.com/Article/ShowArticle.asp?ArticleID=621 |title=湖湘新学第一家——谭嗣同新算学社 |accessdate=2009-03-24 |publisher= 浏阳一中官方网|work= |language =zh-cn}}</ref>

== 对外交流 ==
浏阳一中和[[英联邦|英联邦]][[马恩岛|马恩岛]]巴拉克明中学、[[美国|美国]][[北卡罗莱纳州|北卡罗莱纳州]]哈内特郡特莱顿高中建立了友好合作关系，和[[苏格兰|苏格兰]]伯斯和金罗斯理事会艺术发展部建立了教学交流机制。<ref name="fish " />

== 奖项 ==
2006年，[[搜狐网|搜狐网]]、美国伯克利中国学者联谊会网等媒体联合评选浏阳一中为“中国大陆最好的学校”。<ref name="fish " />

2007年，全国中学名校长论坛组委会、中国百强中学网等多家新闻媒体联合评选浏阳一中为“第二届中国百强中学”。<ref name="fish " />

2009年，全国中学名校长论坛组委会、中国百强中学网等多家新闻媒体联合评选浏阳一中为“第三届中国百强中学”。<ref name="fish " />
