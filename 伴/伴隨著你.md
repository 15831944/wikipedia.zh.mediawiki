{{nihongo|《'''伴隨著你'''》|'''君をのせて'''}}為日本女歌手[[井上杏美|井上杏美]]個人生涯首張發表的單曲CD，並同為1986年上映[[吉卜力工作室|吉卜力工作室]]製作的[[動畫電影|動畫電影]]《[[天空之城|天空之城]]》片尾曲。由於此曲旋律簡單而優美並且深入民心，此曲亦成為《[[天空之城|天空之城]]》的代表。

== 概要 ==
歌曲內容為改編自[[久石讓|久石讓]]所創作的《天空之城》印象專輯裡的曲目{{nihongo|《希達與巴魯》|シータとパズー}}，是當時《天空之城》動畫製作人[[高畑勳|高畑勳]]向久石讓的提議。<ref>{{Cite book |script-title=ja:天空の城ラピュタ ロマンアルバム |location=日本 |publisher=[[德間書店|德間書店]] |date=2007-12-20 |pages=141頁 |ISBN=4-19-720156-7 |accessdate=2012-04-26 |language=ja}}</ref>

此作品雖為在1986年時上映的動畫片尾曲，但遲至後年（1988年）才發行單曲版本CD。

2002年歌手[[石井龍也|石井龍也]]以《天空之城》男主角巴魯的內心思緒為靈感寫了一篇歌詞，並將歌詞的內容套在《伴隨著你》的曲調中；以新歌名{{nihongo|《相伴著你》|君をつれて}}的名義翻唱。<ref>{{cite web |url=http://www.sonymusic.co.jp/Music/Arch/SR/TatsuyaIshii/SRCL-5442/ |script-title=ja:君をつれて |publisher=[[新力音樂|新力音樂]] |accessdate=2012-04-26 |language=ja}}</ref>
