'''华能玉环电厂'''位于中国[[浙江省|浙江省]][[玉环市|玉环市]][[大麦屿街道|大麦屿]]，是中国首座自行建设的百万千瓦[[超臨界鍋爐|超超临界燃煤机组]]电厂。<ref name=":3">{{Cite news|url=http://www.sasac.gov.cn/n2588025/n2588124/c3921398/content.html|title=华能集团年内建成3－4个节约环保示范电厂|author=中国华能集团公司|date=2006-04-27|work=中国国务院国资委网站|accessdate=}}</ref><ref name=":0">{{Cite web|url=http://www.chng.com.cn/n31533/n746626/n746630/c824343/content.html|title=[示范单位]玉环电厂：“三力”“三和” 耀“三色”|accessdate=2018-10-13|author=中国华能集团公司|date=2012-03-30|authorlink=中国华能集团|publisher=中国华能|archive-url=https://web.archive.org/web/20130617064929/http://www.chng.com.cn/n31533/n746626/n746630/c824343/content.html|archive-date=2013-06-17|dead-url=no}}</ref>华能玉环电厂隶属于[[中国华能集团|中国华能集团]]浙江分公司，<ref name=":1">{{Cite news|url=http://www.chng.com.cn/n31531/n752608/c39650991/content.html|title=玉环电厂荣获“浙江省电力科普教育基地”称号|author=中国华能集团公司|date=2018-05-17|work=中国华能|accessdate=}}</ref>开工于2003年，投产于2006年11月28日。<ref name=":4" /><ref name=":5" />

== 历史 ==
2003年4月21日，华能玉环电厂开工建设。<ref name=":4">{{Cite web|url=http://news.bjx.com.cn/html/20170117/804117-4.shtml|title=【回顾】中国发电人自己的 “百万超超”——华能玉环电厂十年发展历程(4)|accessdate=2018-10-13|author=|date=2017-01-17|publisher=北极星电力网}}</ref>为了加快建设，[[中国华能集团|华能集团公司]]于2005年10月21日与中国[[浙江省人民政府|浙江省人民政府]]签署了能源项目合作协议，浙江省政府将“华能玉环电厂二期工程2台100万千瓦机组的连续建设项目”列入浙江省“十一五”电力发展规划。<ref>{{Cite web|url=http://www.chng.com.cn/n31529/n31559/c32194/content.html|title=2005年大事记|accessdate=2018-10-13|author=中国华能|date=2006-01-01|publisher=中国华能集团公司|archive-url=https://web.archive.org/web/20171219073651/http://www.chng.com.cn/n31529/n31559/c32194/content.html|archive-date=2017-12-19|dead-url=no}}</ref>2006年11月28日，首台百万千瓦级机组第一次冲击百万千瓦负荷成功，标志着机组正式投产。<ref name=":5">{{Cite news|url=http://news.bjx.com.cn/html/20170117/804117.shtml|title=【回顾】中国发电人自己的 “百万超超”——华能玉环电厂十年发展历程|author=|date=2017-01-17|work=北极星电力网|accessdate=|archive-url=https://web.archive.org/web/20170120132302/http://news.bjx.com.cn/html/20170117/804117.shtml|archive-date=2017-01-20|dead-url=no}}</ref>直至2007年11月25日，电厂相继投产四台百万千瓦[[超臨界鍋爐|超超临界燃煤机组]]，创造了十二个月内建成投产四台此类机组的世界纪录。<ref name="bjx3">{{Cite web|url=http://news.bjx.com.cn/html/20170117/804117-3.shtml|title=【回顾】中国发电人自己的 “百万超超”——华能玉环电厂十年发展历程(3)|accessdate=2018-10-13|author=|date=2017-01-17|publisher=北极星电力网}}</ref>

2008年1月初，玉环电厂因冰雪灾害而紧急停机，仅留有4号机组接带15万千瓦运行4小时，接带50万千瓦单机运行40天。<ref name=":2">{{Cite news|url=http://news.bjx.com.cn/html/20170117/804117-2.shtml|title=【回顾】中国发电人自己的 “百万超超”——华能玉环电厂十年发展历程(2)|date=2017-01-17|work=北极星电力网|accessdate=}}</ref>2009年开始，华能玉环电厂举办职工文艺汇演，此后每年都有举行。 2016年12月13日，华能玉环电厂4号机组超低排放改造完成，实现四台百万千瓦级超超临界机组超低排放投产。<ref name=":2" />2018年9月7日，2号机组通过深度调峰验证性试验。<ref>{{Cite web|url=http://www.chng.com.cn/n31531/n752608/c39666872/content.html|title=玉环电厂2号机组通过深度调峰验证性试验|accessdate=2018-10-13|author=中国华能|date=2018-09-07|publisher=中国华能集团公司}}</ref>

== 社会责任 ==
自2013年以来，玉环电厂通过安全培训、电力知识讲解、现场参观与电力科普下乡村进校园、科普知识手册发放等多种途径，向社会公众普及电力与环保知识。截至目前，仅厂内参观人数就累计超过4500人。<ref name=":1" />

== 荣誉 ==
2009年10月，玉环电厂工程获“[[新舊中國|新中国]]成立60周年百项经典暨精品工程”。2010年12月，玉环电厂二期工程获“2010年度国家优质工程”银质奖。技术方面，华能玉环电厂是中国首座自行建设的百万千瓦[[超臨界鍋爐|超超临界燃煤机组]]电厂<ref name=":3" />。2007年12月，玉环电厂“超超临界燃煤发电技术的研发与应用”课题获中国[[国家科学技术进步奖|国家科学技术进步一等奖]]。2008年9月，1号机组获“全国火电机组竞赛百万超超临界机组”特别贡献奖。环境上，2008年7月，华能玉环电厂获评“国家环境友好工程”。同年11月，获“国家优质工程金奖”。2013年12月，玉环电厂获评“中国美丽电厂”<ref name=":4" /> 企业文化和社会责任方面，2010年，华能玉环电厂获“中国企业文化建设优秀单位”称号；<ref name=":0" />2018年5月16日，浙江省电力学会授予华能玉环电厂“电力科普教育基地”牌匾。<ref name=":1" />

== 评价 ==
中国电机工程学会理事长陆延昌在接受[[新华社|新华社]]记者专访时说：“目前，我国的发电用煤占国内煤产量的一半以上，煤炭资源紧缺已经成为制约燃煤电厂发展的关键因素。华能玉环电厂率先使用国际先进的超超临界燃煤机组，代表了中国燃煤发电厂今后的发展方向，为燃煤电厂节煤发电起到了示范带头作用。”<ref>{{Cite news|url=https://news.qq.com/a/20050919/001427.htm|title=浙江华能玉环电厂：三个不等号昭示电力可持续发展|author=新华网|date=2005-09-19|work=腾讯新闻|accessdate=}}</ref>
