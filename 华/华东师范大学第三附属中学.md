'''华东师范大学第三附属中学'''简称'''华师大三附中'''，是[[上海市|上海市]]的一所实验性示范性高中，由[[华东师范大学|华东师范大学]]、[[金山区_(上海市)|金山区]]政府和[[上海石油化工股份有限公司|上海石油化工股份有限公司]]三方联办<ref name="上海教育新闻网">{{cite web |url=http://www.shedunews.com/zhuanti/xinwenzhuanti/2015expo12th/2015expo12th_syxsfxgzzq/2015/04/07/1559007.html |title=上海市华东师范大学第三附属中学 |author= |date=2015-04-07 |website=www.shedunews.com |publisher=上海教育新闻网 |access-date=2018-04-04 |quote=}}</ref>，校址位于[[杭州湾|杭州湾]]边的石化新城。学校下设有金山光启创新学院华三分院、金山区生物学科基地、华东师大三附中人文书院等附属机构<ref name="上海教育新闻网"></ref>。

==校园==

华东师范大学第三附属中学总建筑面积26,800平方米<ref>{{cite journal |date=2017 |title=华东师范大学第三附属中学简介 |journal=《上海中学数学》 |volume= |issue=Z2 |pages= |doi= }}</ref>。2014年，[[上海市发展和改革委员会|上海市发展和改革委员会]]批复同意华东师范大学第三附属中学迁建项目<ref>{{cite web |url=http://www.shdrc.gov.cn/gk/xxgkml/gcjslyxxgkzl/xmxx/xmjyssp/17732.htm |title=华东师范大学第三附属中学迁建项目 |author= |date=2014-12-02 |website=www.shdrc.gov.cn |publisher=[[上海市发展和改革委员会|上海市发展和改革委员会]] |access-date=2018-04-04 |quote=}}</ref>，学校将迁建至金山区[[张堰镇|张堰镇]]，未来用地面积106,639.1平方米，建筑总面积75,550平方米，其中地下室面积6,400平方米，工程总投资概算49,329.82万元。办学规模将达到72个教学班，其中初中24班，高中48班。<ref>{{cite web |url=http://www.jsjjw.gov.cn/html/zfxxgk/611558407843.html |title=关于华东师范大学第三附属中学迁建工程初步设计的批复 |author= |date=2015-10-15 |website=www.jsjjw.gov.cn |publisher=上海市金山区建设和管理委员会 |access-date=2018-04-04 |quote=}}</ref>
