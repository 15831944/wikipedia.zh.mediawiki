'''华北大演习'''，或称'''八〇二军事演习'''，指[[中国人民解放军|中国人民解放军]][[中国人民解放军北京军区|北京军区]]根据[[中央军事委员会|中国共产党中央军事委员会]]的指示於1981年9月14日至18日在[[中华人民共和国|中华人民共和国]][[河北省|河北省]][[张家口市|张家口市]]举行的一次大型[[軍事演習|軍事演習]]。

[[胡耀邦|胡耀邦]]、[[邓小平|邓小平]]、[[赵紫阳|赵紫阳]]、[[李先念|李先念]]和[[华国锋|华国锋]]等[[党和国家领导人|党和国家领导人]]临场视察，中国人民解放军高级干部战役集训班全体军官、[[中国共产党中央委员会|中共中央]]、[[中华人民共和国国务院|国务院]]各部委和各省、直辖市、自治区主要领导人，解放军师以上和北京军区团以上军官及当地地方[[干部|干部]]和[[民兵|民兵]]共3万2千余人参观演习。

演习结束后，於9月19日上午在[[张家口宁远机场|张家口机场]]进行[[阅兵|阅兵]]，[[中国共产党中央军事委员会主席|中国共产党中央军事委员会主席]]邓小平检阅部队，并且发表讲话<ref>{{Cite web |url = http://zjk.mzfz.gov.cn/ReadNews.asp?NewsID=86&BigClassName=%CE%C4%CF%D7%D7%CA%C1%CF&SmallClassName=%CE%C4%CF%D7%D7%CA%C1%CF&SpecialID=0 |title = 张家口市人大大事记（1980-1984） |accessdate = 2012-12-19 |author =  |date = 2006-08-18 |work =  |publisher = 民主法制网 |language = zh-hans }}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[“八〇二”军事演习观礼台|“八〇二”军事演习观礼台]]现为[[河北省文物保护单位|河北省文物保护单位]]。
