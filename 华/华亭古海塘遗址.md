'''奉贤华亭海塘'''，或称'''华亭东石塘'''<ref name="ldht">{{cite book |author1=上海市奉贤县县志修编委员会 |title=奉贤县志 |date=1987 |publisher=上海人民出版社 |location=上海 |isbn=7-208-00027-1 |url=http://www.shtong.gov.cn/Newsite/node2/node4/node2250/node4422/node5491/node6678/node61362/userobject1ai13367.html |accessdate=2018-09-21 |chapter=历代海塘 |archive-url=https://web.archive.org/web/20160327022912/http://www.shtong.gov.cn/Newsite/node2/node4/node2250/node4422/node5491/node6678/node61362/userobject1ai13367.html |archive-date=2016-03-27 |dead-url=no }}</ref>（原奉贤县登录名称）、'''华亭古海塘'''、'''华亭石海塘'''，是[[上海市|上海市]][[奉贤区|奉贤区]][[柘林镇_(奉贤区)|柘林镇]]境内的一处古代[[海塘|海塘]]遗址，建于[[雍正|雍正]]年间（[[1725年|1725年]]至[[1728年|1728年]]）<ref name="ylb">{{cite web |author1=上海市文物局 |title=上海市市级文物保护单位一览表 |url=http://wgj.sh.gov.cn/node2/n1716/n1718/n1755/n1776/n1785/u1a81911.html |website=上海市文化广播影视管理局（上海市文物局） |accessdate=2018-09-21 |archiveurl=https://web.archive.org/web/20140714210255/http://wgj.sh.gov.cn/node2/n1716/n1718/n1755/n1776/n1785/u1a81911.html |archivedate=2014-07-14}}</ref>，位于原[[华亭县_(唐朝)|华亭县]]南部的古代[[杭州湾|杭州湾]]北侧岸线、今[[沪杭公路|沪杭公路]]和[[奉柘公路|奉柘公路]]南侧路基沿线，发掘区段长4.5千米，被誉为“上海小长城”<ref name="cjgb">{{cite news |author1=裘雯涵 |author2=李宝花 |title=奉贤柘林这座人称“上海小长城”的华亭石海塘，能否冲击国家重点文物保护单位？ |url=https://www.jfdaily.com/news/detail?id=45091 |accessdate=2018-09-21 |work=上观新闻 |publisher=解放日报社 |date=2017-02-20}}</ref>。2002年4月，这段海塘被登录为第七批[[上海市文物保护单位|上海市文物保护单位]]<ref name="ylb" />。

== 建筑特征 ==
[[File:Fengxian_Huating_Seawall_-_25_Sept_2018_22.jpg|thumb]]
奉贤华亭海塘是历史上华亭县捍海石塘的东段，西到龙珠庵（今[[奉贤区|奉贤区]][[柘林镇_(奉贤区)|柘林镇]]{{ruby|漴|-{zh-hans:shuāng;zh-hant:ㄕㄨㄤ}-<ref>{{cite book|title=现代汉语词典|edition=第7版|author=中国社会科学院语言研究所词典编辑室|year=2016|page=1223|location=北京|publisher=商务印书馆|ISBN=978-7-100-12450-8}}</ref>}}缺村附近），东至柘林镇华家角（堰墩湾）与内海塘相连，全长约8.6公里。该段石塘地下基础深1米，地面部分高4米，宽约1.5米，底宽约3米有余；贴土包裹后形成了外土内石结构，塘面宽约15米，底宽约30米，高仍为4米<ref name="ldht" />。此外，在已经发掘的段落中，发现了三十多处铭刻，其中包括了建设者的姓名、监督者的姓名和“长庆安澜”“屹若金汤”等祝福语，可见当时的建设者重视工程质量，并可以看出主持修建的士绅阶层和建设者们对海疆永固的美好愿望<ref name="shll">{{cite news |title=“海上长城”华亭古海塘伤痕累累 与奉柘公路并肩蜿蜒，被车辆碰撞时有发生 |url=http://newspaper.jfdaily.com/xwcb/html/2018-02/23/content_68488.htm |accessdate=2018-09-21 |work=新闻晨报 |publisher=新闻报社 |date=2018-02-23 |page=A05 |archive-url=https://web.archive.org/web/20180222213332/http://newspaper.jfdaily.com/xwcb/html/2018-02/23/content_68488.htm |archive-date=2018-02-22 |dead-url=no }}</ref>。

== 历史 ==
雍正四年（[[1726年|1726年]]），时任[[吏部尚书|吏部尚书]][[朱轼|朱轼]]主持筑华亭县捍海石塘，至雍正十三年（[[1735年|1735年]]）竣工。值得注意的是，为了保证堤防稳固，该海塘规定采用了特殊的工法，即“鱼鳞条块”法，并在石塘筑法上对于石块的长宽高、堆叠层数、堆叠方法和粘合方法等都做了要求。后来，因为下属未能遵照事先定下的石塘筑法来建设，浙江巡抚[[李馥|李馥]]被罢官。朝廷于雍正五年派[[太仆寺|太仆寺]]丞[[俞兆岳|俞兆岳]]监督[[松江府|松江府]]塘工，他发明了铁笋铁箫法，并建设了示范石塘（“样塘”），在建的华亭石塘也按照这个工法进行加固，增强了石塘的整体稳定性。<ref name="gzff">{{cite book |author1=上海市奉贤县县志修编委员会 |title=奉贤县志 |date=1987 |publisher=上海人民出版社 |location=上海 |isbn=7-208-00027-1 |url=http://www.shtong.gov.cn/Newsite/node2/node4/node2250/node4422/node5491/node6678/node61362/userobject1ai13711.html |accessdate=2018-09-21 |chapter=海塘构筑方法 |archive-url=https://web.archive.org/web/20160327022915/http://www.shtong.gov.cn/Newsite/node2/node4/node2250/node4422/node5491/node6678/node61362/userobject1ai13711.html |archive-date=2016-03-27 |dead-url=no }}</ref><ref>{{cite book |author1=《上海水利志》编纂委员会 |title=上海水利志 |date=1997 |publisher=上海社会科学院出版社 |location=上海 |isbn=7-80618-380-9 |url=http://www.shtong.gov.cn/newsite/node2/node2245/node68538/node68544/index.html |chapter=大事记 |access-date=2018-09-21 |archive-url=https://web.archive.org/web/20180118175200/http://www.shtong.gov.cn/newsite/node2/node2245/node68538/node68544/index.html |archive-date=2018-01-18 |dead-url=no }}</ref>

[[File:Fengxian_Huating_Seawall_-_25_Sept_2018_26.jpg|thumb]]
雍正六年、雍正十二年、乾隆十二年，华亭海塘先后进行了几次加固工程，将石塘的三个面都包裹于厚厚的土层之中，构成了包石土塘<ref name="lyxy">{{cite book |author1=《上海水利志》编纂委员会 |title=上海水利志 |date=1997 |publisher=上海社会科学院出版社 |location=上海 |isbn=7-80618-380-9 |accessdate=2018-09-22 |chapter=陆域现有海塘}}</ref>。

道光十五年（[[1835年|1835年]]）六月，因为[[台风|台风]]来袭、潮水暴涨，[[戚家墩|戚家墩]]以东至胡家厂段的外海塘坍入海中。[[道光十七年|道光十七年]]时，戚家墩以西至青龙港故址段的华亭海塘也陷入了海水中。光绪十九年（[[1893年|1893年]]），胡家厂以东至杨公庙段的海塘被海水冲毁。然而，由于海潮顶冲点西移，漴阙一带海岸涨出了很多土地，因而杨公庙以东这段海塘得以保存。后来，[[林则徐|林则徐]]将海塘以杨公庙为界，划分为东、西两塘，因为西塘海浪较险，故较多加固，留存段被称作东二塘；而东塘则只以日常养护为主。<ref>{{cite book |author1=《上海水利志》编纂委员会 |title=上海水利志 |date=1997 |publisher=上海社会科学院出版社 |location=上海 |isbn=7-80618-380-9 |accessdate=2018-09-22 |chapter=陆域古塘演变}}</ref>

同治十三年<!--（[[1873年|1873年]]）-->，华亭海塘置驻塘委员，负责修缮和维护，并设置了华亭县塘工岁修公所负责该事务<ref name="gtcs">{{cite book |author1=上海市奉贤县县志修编委员会 |title=奉贤县志 |date=1987 |publisher=上海人民出版社 |location=上海 |isbn=7-208-00027-1 |url=http://www.shtong.gov.cn/Newsite/node2/node4/node2250/node4422/node5491/node6679/node61363/userobject1ai13715.html |accessdate=2018-09-21 |chapter=固塘措施 |archive-url=https://web.archive.org/web/20160327022917/http://www.shtong.gov.cn/Newsite/node2/node4/node2250/node4422/node5491/node6679/node61363/userobject1ai13715.html |archive-date=2016-03-27 |dead-url=no }}</ref>。

贴土包裹工艺，加之在海塘上[[沪杭公路|沪杭公路]]、[[奉柘公路|奉柘公路]]的建设<ref name="lyxy" />，将石塘隐藏在了地下。[[1996年|1996年]]3月开始，建设于海塘之上的奉柘公路柘林段实施降坡拓宽工程；是年五月，在奉柘公路路基的土被挖去后，部分石塘才被发掘出来<ref>{{cite book |author1=上海市奉贤区《柘林志》编纂委员会 |author2=上海市奉贤区地方志委员会 |title=柘林志 |date=2003 |publisher=[[方志出版社|方志出版社]] |location=北京 |isbn=978-7-5144-0185-1 |url=http://www.shtong.gov.cn/newsite/node2/node4/n107748/n104114/n104118/index.html |chapter=大事记}}</ref>。

[[File:Fengxian_Huating_Seawall_-_25_Sept_2018_05.jpg|thumb]]
被发掘的路段东起柘林镇奉海村，西至柘林镇城南村，长约4.5千米。在石塘被发掘之后，因为露出的堤坝有一定高度，影响了交通，故经过了论证和“备案”后，海塘遗址被挖开了11个缺口<ref name="cjgb" />；因石塘坚固，拆除工程很困难，而拆走的石料已经专门集中堆放以备后续修复；柘林镇安排了[[村民委员会|村民委员会]]、[[居民委员会|居民委员会]]组织了四支志愿者巡查队进行巡查，阻止破坏行为、拔除杂草<ref name="shll" />。

== 现实威胁 ==
[[File:Fengxian_Huating_Seawall_-_25_Sept_2018_02.jpg|thumb]]
[[File:Fengxian_Huating_Seawall_-_25_Sept_2018_08.jpg|thumb]]
{{中国县道名|426|奉柘公路}}沿古海塘而建，是[[奉贤区|奉贤区]]通往[[上海石化|金山石化]]的最短[[中华人民共和国公路分类|县级公路]]，也是金山与奉贤之间物流的一条主要通道，有大量[[集装箱|集装箱]]卡车通行。2017年4月29日14时许，一辆集装箱卡车沿奉柘公路行驶，驶过[[海湾路|海湾路]]西侧80米时，撞上了此处遗迹，导致部分石条移位和碎裂，并引发了集装箱卡车投保的保险公司和政府之间的纠纷，截至2018年2月尚未修复。此外，其上生长的野草会导致石块间粘合力减弱，往海塘上张贴、喷涂的广告会导致石块遭到腐蚀，且清除时会对石块造成损伤。此外，据《[[s:中华人民共和国文物保护法|中华人民共和国文物保护法]]》，海塘两侧的建设控制地带不允许有任何建筑，然而这一点并未能做到。<ref name="shll" />

== 保护与开发 ==
1996年8月14日，这段海塘以'''华亭东石塘'''的名称被奉贤县人民政府公布为[[奉贤区文物保护单位|奉贤县文物保护单位]]。该段海塘以'''奉贤华亭海塘'''的名称于2002年4月27日被列为上海市文物保护单位<ref name="ylb" />，并被列为奉贤区[[爱国主义教育基地|爱国主义教育基地]]；奉贤区人民政府、柘林镇人民政府相关部门正准备相关材料，拟将该海塘申报第八批[[全国重点文物保护单位|全国重点文物保护单位]]；此外，位于海塘正南侧的[[上海应用技术大学|上海应用技术大学]]也围绕该海塘展开了一系列学术研究<ref name="shll" /><ref>{{cite news |title=学校召开“华亭古海塘申报全国重点文物保护单位”专家讨论会 |url=http://www.sit.edu.cn/s/97/t/296/05/67/info66919.htm |accessdate=2018-09-21 |work=上海市应用技术大学 |archive-url=https://web.archive.org/web/20170511073221/http://www.sit.edu.cn/s/97/t/296/05/67/info66919.htm |archive-date=2017-05-11 |dead-url=no }}</ref>。2012年，应用技术大学的师生曾建议相关部门将塘体收入奉贤校区内整体保护，或是建设公园进行保护，但都遇到了瓶颈，无果而终<ref name="why">{{cite news |author1=吴嘉川 |title=为建公路 市级文物被“腰斩” 附近居民大多不知其身世 |url=http://app.why.com.cn/epaper/qnb/html/2012-12/25/content_114363.htm |accessdate=2018-09-21 |work=[[青年报|青年报]] |publisher=青年报社 |date=2012-12-25 |pages=A04-A05}}</ref>。

2018年8月，奉贤区人民政府在答复奉贤区人大常委会主任、党组书记、第十五届[[上海市人民代表大会|上海市人民代表大会]]代表袁晓林等人的建议时提到，拟建设一座“华夏钱滩遗址公园”来介绍古代冈身、古代海塘的历史。此外，奉贤区人民政府提及，2014年时奉贤区文广局已经向[[上海市文物局|上海市文物局]]申请资金，与柘林镇人民政府联合对海塘东段进行了绿化整治；此外，在2017年时，奉贤区人民政府委托的专业团队又新发现了海塘上的碑刻十余块，已经对海塘碑刻进行了拓印并进行研究整理。奉贤区人民政府还明确表示，被拆除的塘段石块已经被编号保存（存放点位于奉贤区文广广场和柘林镇华亭村船浜某地<ref name="why" />），可以在条件允许时，用于整体修复相关段落。<ref>{{cite web |author1=上海市奉贤区人民政府 |title=奉贤区政府对市十五届人大一次会议第0614号代表建议的答复 |url=http://www.shanghai.gov.cn/nw2/nw2314/nw38605/nw42569/nw42584/u21aw1336237.html |website=上海市人民政府 |accessdate=2018-09-21 |language=zh-Hans |date=2018-08-28}}</ref>

== 反思 ==
《[[青年报|青年报]]》记者吴嘉川直指当年拆卸海塘时相关部门和民众的文物保护意识存在缺失，文物保护的法定义务被[[上海市文物局|文物管理委员会]]、规划局的“备案”所替代，令人感到痛心，并指出文物保护不该站现代化建设身后<ref name="why" />。
