'''深圳华强集团'''有限公司是一家以高科技产业为主导的大型投资控股公司，创建于1979年，以文化科技产业、云产业、电子信息高端服务业、商业地产、金融服务业等为主要业务。<ref>{{Cite web |url = http://www.szhq.com/hqintroduce.html |title = 深圳华强集团有限公司介绍 |language = zh-hans |accessdate = 2014-12-12 }}</ref>

目前，华强集团拥有90多家投资企业，拥有多家中国国家级高新技术企业及国家级技术开发中心。

== 概况 ==
=== 产业布局 ===
华强集团旗下有'''深圳华强文化科技集团股份有限公司'''、'''深圳华强实业股份有限公司'''（{{szse|000062}}）、'''深圳华强新城市发展有限公司'''、'''华强云投资控股有限公司'''、'''深圳华强供应链管理有限公司'''、'''深圳华强集团财务有限公司'''等子公司分别从事文化科技产业{{noteTag|1=主要从事主题公园、影视娱乐、媒体网络等文化科技产品的创意、研究、生产与销售}}、云产业{{noteTag|1=以电子教室、电子书包等为载体，以云服务、大数据分析为手段，着力于教育资源公共服务平台的建立}}、电子信息高端服务业{{noteTag|1=包括电子专业市场连锁经营、电子商务平台、物流供应链等}}、商业地产{{noteTag|1=以商业地产开发运行为核心，发展商业运营、酒店经营、物业管理等相关业务}}、金融服务业{{noteTag|1=着力于产业链上下游中小企业为对象的金融服务体系}}等不同产业。

=== 企业现况 ===
该企业在电子精密制造领域处于国际领先地位，生产全世界25%的微型马达、激光头及其应用产品。（旗下激光电子有限公司）

同时在芜湖、泰安、青岛、沈阳、株洲、郑州、厦门建有“[[方特欢乐世界|方特欢乐世界]]”和“[[方特梦幻王国|方特梦幻王国]]”十个大型文化科技主题公园。<ref>{{Cite web |url = http://www.fantawild.com/view.html?url=qygl |title = 华强文化科技集团企业概览 |publisher = 华强文化科技集团 |language = zh-hans |accessdate = 2014-12-14 }}</ref>（旗下方特投资有限公司）

2007年，在[[中国国家信息产业部|中国国家信息产业部]]授权和指导下创建，编制、运行中国电子行业唯一的电子产品市场价格指数——'''华强北·中国电子市场价格指数'''（华强北指数）。<ref name="nfrb071016">{{cite news |language = zh-hans |author = 陶然 |url = http://news.sina.com.cn/o/2007-10-16/094912735320s.shtml |title = 华强北 中国电子市场“晴雨表”“风向标” |publisher = 南方日报 |date = 2007-10-16 |accessdate = 2014-12-12}}</ref>该指数于2010年开始被[[CCTV-2|CCTV-2]]《经济信息联播》采录。目前每周一期报送中国[[中华人民共和国国务院办公厅|中华人民共和国国务院办公厅]]等中国中央政府部委，成为国家制订[[宏观调控|宏观调控]]政策的重要参考数据。<ref>{{Cite web |url = http://www.hcsindex.org/about/ |title = 华强北指数介绍 |publisher = 深圳市华强北电子市场价格指数有限公司 |language = zh-hans |accessdate = 2014-12-12 |quote = 2010年11月8日，正式列为CCTV-2《经济信息联播》数据采录单位；2011年 7月11日，华强北指数正式登陆中华人民共和国国务院办公厅，成为国家进行宏观政策调控的重要参考数据 |archive-url = https://web.archive.org/web/20141213014437/http://www.hcsindex.org/about/ |archive-date = 2014-12-13 |dead-url = yes }}</ref>市场日益认可其权威性，被认为是中国电子市场的“风向标”、“晴雨表”。<ref name="nfrb071016" />（旗下华强北电子市场价格指数有限公司）

另外拥有12家大型电子专业连锁卖场——[[华强电子世界|华强电子世界]]<ref>{{Cite web |url = http://www.hq-mart.com/article/QiYeGaiKuang/2103.html |title = 华强电子世界公司简介 |language = zh-hans |accessdate = 2014-12-14 }}</ref>，和中国电子元器件领域排名前列的垂直[[B2B|B2B]]网站——[[华强电子网|华强电子网]]<ref>{{Cite web |url = http://www.hqew.com/about/us_01.html |title = 华强电子网简介 |language = zh-hans |accessdate = 2014-12-14 }}</ref>。（旗下华强电子世界管理有限公司、深圳华强电子交易网络有限公司）
