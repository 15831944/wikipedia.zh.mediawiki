'''华山路站'''为[[延安路中运量公交系统|延安路中运量公交系统工程]]的一个车站，位于[[上海市|上海市]][[静安区|静安区]][[延安西路_(上海)|延安西路]][[华山路_(上海)|华山路]]路口东侧中央隔离带。站点临近[[静安寺|静安寺]]，为[[上海公交71路|上海公交71路]]及其区间车共用的地面[[岛式月台|岛式车站]]。

== 车站構造 ==
[[File:在建的延安路中运量系统工程华山路站.jpg|thumb]]

华山路站为[[岛式月台|岛式月台]]，为2面2线地面车站组成，站台设有半高屏蔽门，设置一个出入口，通往华山路天桥。

该站点开工于2016年6月25日<ref name="kg">{{cite web | title = 延安路中运量公交系统工程开工 | publisher = 上海市交通委员会 | date = 2016-06-27 | url = http://www.jt.sh.cn/jtw/xwzx/n218/u1ai51190.html | accessdate = 2016-11-30 | language = zh | archive-url = https://web.archive.org/web/20161010223253/http://www.jt.sh.cn/jtw/xwzx/n218/u1ai51190.html | archive-date = 2016-10-10 | dead-url = yes }}</ref>，于2017年2月1日清晨正式投入使用<ref name="syy">{{cite news|title=上海延安路中运量公交开通试运营|url=http://www.thepaper.cn/newsDetail_forward_1610252|publisher=澎湃新闻|date=2017-02-01}}</ref>。

== 周边设施 ==
* 站点南面：[[华山医院|华山医院]]、[[上海市人民政府侨务办公室|上海华侨大厦]]、{{上海地铁线路名|14}}[[静安寺站|静安寺站]]（在建）
* 站点北面：[[静安寺|静安寺]]、中国福利会少年宫
* 站点东面：[[静安公园|静安公园]]、{{上海地铁线路名|2}}{{上海地铁线路名|7}}[[静安寺站|静安寺站]]
* 站点西面：[[华东医院|华东医院]]

== 公交换乘 ==
附近有3个常规公交站点：
* 静安寺（南京西路华山路）：[[上海公交20路|20路]]、37路、[[上海公交57路|57路]]、76路、323路夜宵、330路夜宵
* 静安寺（华山路）：[[上海公交15路|15路]]、45路、94路、315路夜宵、327路夜宵、830路、838路、927路
* 延安西路华山路：01线、48路、311路夜宵、548路、虹桥机场守航夜宵线

== 车站出口 ==
这个车站有一个出入口，位于站点东侧，连接天桥并设有扶梯，乘客需从天桥进站<ref>{{cite web|title=延安路中运量公交 新建天桥吊装完成 A09-上海新闻-新闻晨报|url=http://newspaper.jfdaily.com/xwcb/html/2016-10/27/content_224374.htm|website=新闻晨报电子版|publisher=新闻报社}}</ref>。

天桥分别通往延安西路华山路口的东北、西北、西南方向，在东北角、西南角还设置了电梯。惟东南方向正在建设[[上海轨道交通14号线|上海轨道交通14号线]]，楼梯曾被拆除<ref>{{cite web|title=上海延安路华山路天桥要拆了：配合地铁施工，今后行人走地面_浦江头条_澎湃新闻-The Paper|url=http://www.thepaper.cn/newsDetail_forward_1441960|website=澎湃新闻}}</ref><ref>{{cite web|title=就在6月中旬前！行人又可从华山路天桥过街了--上观|url=http://www.shobserver.com/news/detail?id=17544|website=上观}}</ref>。
