'''華姓'''（陰平音或去聲韻，作姓氏使用時不作陽平音。ㄏㄨㄚˋ，不唸作ㄏㄨㄚˊ、ㄏㄨㄚ，粵語讀waa6(話)）為[[漢字姓氏|漢字姓氏]]之一，在《[[百家姓|百家姓]]》中排名第28位。在當代中國姓氏中排名第196位，人數約占全國人口的百分之零點五（千分之五）。

根據臺灣户政司民國107年（2018年）6月的數據，臺灣華姓人口共有4,288人，為全國姓氏人數排名第154名。<ref>內政部戶政司編，《全國姓名統計分析》，臺北：內政部出版。2018年10月初版一刷</ref>
== 起源 ==
;子姓
*出自[[子姓|子姓]]：[[西周|西周]]初，商王之族[[微子啟|微子啟]]被封於商丘，建立宋國。東周初，宋戴公之子[[好父说|好父说]][[食釆|食釆]]於華，其子孫遂以華為氏。

;羋姓
*源於[[羋姓|羋姓]]：[[秦宣太后|秦宣太后]]同母弟[[羋戎|羋戎]]入秦為官，封華陽君，子孫後人遂以華為氏。

;姒姓
*源於[[姒姓|姒姓]]：相傳夏王仲康於[[西嶽|西嶽]][[華山|華山]][[封禪|封禪]]，其支系後裔由此以華為[[氏|氏]]。[[大禹|大禹]]之孙[[仲康|仲康]]曾去[[华山|华山]][[封禅|封禅]]，居住当地。这支[[姒姓|姒姓]]子孙，以居住地[[华山|华山]]为氏，奉[[仲康|仲康]]为华氏始祖。<ref>百家姓渊源  张世国编著</ref>

;滿蒙回
*清滿洲八旗姓[[愛新覺羅氏|愛新覺羅氏]]之後裔有改華姓者；清嶺夷十二地夷人頭目，嘉慶十三年歸附，十六年改流更姓住牧，芭蕉溝夷人為華姓；錫伯族華西哈爾氏，漢姓為華；今滿、蒙古、回等民族有此姓。華姓可能源於满族，出自明、清时期女真诸部，属于以部落名称汉化为氏。据史籍《清朝通志·氏族略·满洲八旗姓》记载：
**⑴ [[满族|满族]]阿赉氏：Alai Hala，满族姓氏，满语的意思“桦皮”，世居翁郭罗城(今黑龙江阿城金国上京故城)，后多冠汉姓为华氏、阿氏等。
**⑵ [[满族|满族]]瑚锡哈哩氏，Husihari Hala，亦称祜什哈礼、瑚克沙哈里，以地为氏，世居瑚锡哈理(今黑龙江黑河对岸俄罗斯地区)、依兰(今黑龙江依兰)、费尔塔哈(今吉林西南二百五十公里处)、三姓(今黑龙江依兰)。后有[[赫哲族|赫哲族]]、锡伯族引为姓氏者。满族、赫哲族、锡伯族瑚锡哈哩氏，在清朝中叶以后多冠汉姓为华氏、严氏、关氏、胡氏、扈氏等。

*源于蒙古族，出自元、明、清时期蒙古诸部，属于汉化改姓为氏。据史籍《清朝通志·氏族略·蒙古[[八旗|八旗]]姓》记载：
**⑴ [[蒙古族|蒙古族]]谟锡哷氏，亦称木华犁氏，世居克什克腾(今内蒙古克什克腾)。后有满族引为姓氏者，满语为MosireHala。蒙古族、满族谟锡哷氏，在清朝中叶以后多冠汉姓为李氏、严氏等。
**⑵ [[蒙古族|蒙古族]]扎拉尔氏，亦称扎喇亦尔氏，源出元朝时期蒙古扎剌赤儿部，是元朝著名太师，成吉思汗的[[四駒|四駒]][[木華黎|木華黎]]之后裔，以部为氏，世居乌鲁特(今内蒙古[[兴安盟|兴安盟]])、[[察哈尔|察哈尔]](今[[黑龙江|黑龙江]]、[[俄罗斯|俄罗斯]][[哈巴羅夫斯克|哈巴羅夫斯克]]（俄語：Хабаровск）或稱作[[伯力|伯力]]。后有达斡尔族、满族引为姓氏者，满语为Jalar Hala，为达斡尔族著姓之一。所冠汉姓为赵氏、严氏、季氏等。
**⑶ [[蒙古族|蒙古族]]华努特，在清朝末期至民国初期，多冠汉姓为华氏。

== 郡望 ==
*一、[[武陵郡|武陵郡]]，漢代設置，治所在[[義陵|義陵]]，故城在今[[湖南|湖南]][[漵浦|漵浦]]南。
*二、[[平原郡|平原郡]]，漢代設置，治所在[[平原|平原]]，故城在今山東平原縣西南。
*三、沛國，西漢為[[沛郡|沛郡]]，東漢為[[沛國|沛國]]，治所在相縣（故城在今安徽濉溪縣西北）。
*[[堂号|堂号]]：[[汉|汉]]置[[武陵郡|武陵郡]][[武陵|武陵]]堂，现在[[湖南省|湖南省]][[常德市|常德市]]。据《[[广韵|广韵]]》记载：宋正考父食邑于华，后以为氏。望出武陵、平原、沛国。
*華姓分堂號：[[本仁堂|本仁堂]]：蕭山漁臨華氏宗譜·序：華氏「原姓宋，實微子（商·紂王同父異母兄長）之裔，[[食采|食采]]於華，今河南省商丘以東至江蘇省銅山以西地區，西周時，宣王姬靜遂因地賜華姓。華、宋二姓，萬世難婚。說明華氏源流和華、宋不通婚的原因。
*存裕堂、詒谷堂、享敘堂、培元堂、壽和堂、永思堂、享德堂、思訓堂、慶余堂、敦本堂、禮耕堂、佑啟堂等。

== 歷史人物 ==
*[[华元|华元]]，华督曾孙，春秋时宋国大夫。历事昭公、文公、共公、平公四君，执政四十年
*[[华毋害|华毋害]]，西汉将领。以越将从高祖起于留，定三秦，击臧荼，有功，封为终陵侯，谥齐
*[[华佗|华佗]]，漢末名醫，發明了麻沸散這種麻醉藥
*[[华歆|华歆]]（157－231）
*[[華雄|華雄]]，在三國演義中，虎牢關之戰，敗亡於關羽刀下
*[[华胥|华胥]]
*[[华表|华表]]（204－275）
*[[華覈|華覈]]
*[[华峤|华峤]]（？－293）
*[[华温琪|华温琪]]，宋州下邑（今安徽砀山）人，五代梁唐间将领。少从黄巢起义，巢败后，仕后梁，官至齐、晋二州节度使。入后唐后，官至镇国军节度使，后以太子太保致仕，以固守称胜  
*[[华云龙|华云龙]]安徽定远人，明初将领。从朱元璋起兵，南征北战，立功无数。洪武三年，论功封淮安侯
*[[华察|华察]]（1497－1574）
*[[华秋苹|华秋苹]]（1785－1858）
*[[华蘅芳|华蘅芳]]（1833－1902），清末數學家
*[[华彦钧|华彦钧]](1893－1950)，中國民间音乐家、二胡演奏家
*[[华岗|华岗]]（1903－1972），中共早期职业革命家、编辑家、哲学家、史学家、教育家，曾任山东大学校长
*[[华罗庚|华罗庚]]（1910－1985），中國数学家
*[[华君武|华君武]]（1915－2010），中國漫画家

== 現代人物 ==
*[[华春瑩|华春瑩]]，女，汉族，江蘇人，中華人民共和國外交部發言人
*[[华国锋|华国锋]]，中国政治家。
*[[华建敏|华建敏]]，男，汉族，上海人，祖籍江苏无锡，清华大学动力系燃气轮机专业毕业。1961年加入中国共产党，曾任中共第十六、十七屆中央委員、国务委员兼国务院秘书长、国家行政学院院长、第十一届全国人大常委会副委员长等职
*[[连阔如|连阔如]]，男，汉族，北京人，原名华连寿，著名说书人，笔名云游客
*[[华士飞|华士飞]]，男，汉族，上海人，新疆生产建设兵团原党委副书记、司令员，中国新建集团公司原总经理。中共新疆维吾尔自治区第七届党委委员
*[[华晨宇|华晨宇]]，男，汉族，湖北人，着名创作歌手，2013年快乐男声冠军，2016年获得MAMA亚洲最佳艺人奖
*[[華錫鈞|華錫鈞]]，生於江蘇省無錫市，[[中華民國空軍|中華民國空軍]][[二級上將|二級上將]]、臺灣[[中山科學研究院|中山科學研究院]]副院長。任內參與[[經國號戰機|經國號戰機]]（IDF）研發有功，被譽為「IDF之父」。
