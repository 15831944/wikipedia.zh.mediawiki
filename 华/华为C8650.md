'''华为C8650'''是[[中国|中国]][[华为|华为]]公司为[[中国电信|中国电信]]公司定制的一款以Android為操作系統的[[智慧型手機|智慧型手機]]（定制机），于2011年8月推出。

此机型为华为C8500和华为C8600的后续版本，继承了之前版本的优点，并搭载Android 2.3.3操作系統和 [[骁龙|骁龙]]MSM7627 600MHz處理器。该手机使用多点触控电容屏，同时还支持EVDO网络和WLAN无线网络，这种配置在中低端手机中算比较均衡的。<ref>{{Cite web |url= http://info.3g.qq.com/g/s?g_f=18449&g_ut=2&aid=digi_ss&id=digi_20110926001632 |title= 2011国际通信展：灵敏滑屏华为C8650亮相|author=  |date=2011-09-26|publisher= 手机腾讯网 |language= zh-hans|accessdate= 2013-12-11 |quote=  }} </ref><ref>{{Cite web |url= http://m.sohu.com/n/318621541/?v=3&_trans_=000014_baidu_ss |title= 天翼千元3G智能手机典范 华为C8650详细评测|author=  |date=2011-09-07 |publisher= 搜狐科技 |language= zh-hans|accessdate= 2013-12-11 |quote= }} </ref>

== 相似机型 ==
[[File:Hua8650.JPG|thumb]][[华为U8650|华为U8650]]与华为C8650外型极为相似，且系统皆为[[Android_2.3|Android 2.3]]系统，只不过前者为[[中国联通|中国联通]]公司的定制机，所以网络采用的是[[WCDMA|WCDMA]]，而后者为中国电信公司的定制机，所以网络为[[CDMA|CDMA]]。此外，手机的按键和后盖也有差异。华为公司曾推出台湾版IDEOS U8650在台湾发售<ref>[http://www.pcpop.com/doc/0/746/746783.shtml 流畅Android2.3 华为IDEOS U8650评测]</ref>。
