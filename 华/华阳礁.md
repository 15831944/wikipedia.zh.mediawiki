'''華陽礁'''（Cuarteron Reef），是[[南沙群岛|南沙群岛]]中一[[暗礁|暗礁]]。中國漁民俗稱「銅銃仔」<ref>{{Cite web|url=http://old.ccmt.org.cn/shownews.php?id=16248|title=中国涉外商事海事审判网|accessdate=2018-07-09|work=old.ccmt.org.cn}}</ref>。目前由[[中華人民共和國|中華人民共和國]]實際控制，行政上隶属于[[海南省|海南省]][[三沙市|三沙市]]。目前是有中国军队驻守的最南端的岛礁，同时也是南沙群岛中，中国军队驻守的最西端的岛礁。从地理上，华阳礁属于[[尹庆群礁|尹庆群礁]]，位于该群礁东面，该群礁其余三个礁体（[[东礁|东礁]]、[[西礁|西礁]]、[[中礁|中礁]]）都由越南实际控制。华阳礁与越南在南沙群岛的军事中心[[南威岛|南威岛]]距离较近。

== 歷史 ==
* 中國漁民向稱之為「銅銃仔」。
* 1935年，[[中華民國|中華民國]]水陸地圖審查委員會公佈名稱為克德朗礁。 
* 1947年，中華民國內政部方域司公佈名稱為華陽礁。
* 1988年，中華人民共和國[[中國地名委員會|中國地名委員會]]公佈名稱為華陽礁。
* 2013年底，中華人民共和國開始[[填海|填海]]造陸工程。
* 2014年11月，吹填面积约为0.2311平方公里，在中華人民共和國实际控制的南沙岛礁中居第七位<ref>{{cite web|url=http://www.sohu.com/a/245931956_100200965|title=华阳礁最新卫星图曝光，网友：这是在南海建森林公园吗？|date=2018-08-08|publisher=搜狐网}}</ref>。
* 2015年10月，华阳礁已建成一座50米高灯塔，为国际海运提供导航等公共服务。岛上海水淡化处理设备及冷藏设施也已建成并投入使用<ref>{{cite web|url=http://mil.news.sina.com.cn/2015-10-10/0719840700.html|title=中国南海华阳礁与赤瓜礁大型灯塔建成发光|date=2015-10-10|publisher=新浪网}}</ref>。

== 名稱來源 ==
[[中國|中國]]漁民稱此礁為銅銃仔。越南方面稱之為“Đá Châu Viên”（-{zh-hans:𥒥洲圓; zh-hant:𥒥洲圓}-）。 

== 地理位置 ==

* 位於[[尹慶群礁|尹慶群礁]]的東端，與[[東礁|東礁]]相距約9.5浬，為一東西長3.5浬（5.5公里），南北寬1.5浬之新月形礁台。

== 地質地形 ==
* 退潮露出，滿佈[[珊瑚礁|珊瑚礁]]石的[[沼澤|沼澤]]地、無礁湖，礁上海水沖刷，有幾個1.2-1.5米高之岩石立於北邊，四週海水陡深，東端外側急陡變深至深海。

== 島上狀況 ==
* [[水源|水源]]：無地下水源，須靠船艦運補。
* [[人口|人口]]：目前由[[中國人民解放軍|中國人民解放軍]]駐守，無一般平民居住。
* [[建築|建築]]：人工水泥墊高之樓建築物。2015年10月，岛上灯塔已开始发光。<ref name="lighthouse" />
* [[運輸|運輸]]：有簡易運輸補給[[碼頭|碼頭]]。
