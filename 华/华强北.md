'''华强北'''是位於[[中國|中國]][[廣東省|廣東省]][[深圳市|深圳市]]的重要電子產品商業地帶，有“'''中国电子第一街'''”之称。<ref name = "shb081013">{{cite news |language = zh-hans |url = http://www.mofcom.gov.cn/article/zt_gaoxin/lanmuerf/200810/20081005827234.shtml |title = 华强北荣膺“中国电子第一街” |publisher = 中华人民共和国商务部 |date = 2008-10-13 |accessdate = 2014-12-12}}</ref>

狭义的华强北指从[[深南路|深南路]]起，到红荔路止的这一段华强北路及两侧。广义的华强北指东起燕南路，西至华富路；南接深南路，北抵红荔路的一片区域，有时还包括深南路路南的临街建筑，如统建楼和世贸广场等。这片区域大约南北长930米，东西宽1560米，面积约1.45[[平方公里|平方公里]]。从行政区划上，华强北属于[[福田区|福田区]]。华强北是[[中国大陆|中国大陆]]交易额最大的商业街，是大陆最有影响力的[[电脑硬件|电脑硬件]]、[[电子元件|电子元件]]市场之一，也是一个[[业态|业态]]十分丰富的商业街区。

从2013年2月27日起，因地铁七号线施工，华强北路（红荔路-[[深南路|深南路]]）开始封闭。华强北路的华强北商圈段已改造成为步行街，并于2017年1月14日重新开放。<ref name="xkb130227">{{cite news |language = zh-hans |author = 张国锋 |url = http://news.ifeng.com/gundong/detail_2013_02/27/22519636_0.shtml |title = “中国电子第一街”封路3年 |publisher = 新快报 |date = 2013-02-27 |accessdate = 2014-12-12}}</ref><ref>{{cite news |language = zh-hant | |url = http://hk.on.cc/cn/bkn/cnt/news/20170114/bkncn-20170114194905314-0114_05011_001.html |title = 圍封4年升級改造　深圳華強北步行街開街 |publisher = 東方日報 |date = 2017-01-14 |accessdate = 2017-03-05}}</ref>

==详细位置信息==
华强北在地理位置上，靠近深圳特区的中心位置，东面是深圳市政协，西面是[[深圳中心公园|深圳中心公园]]，[[中心商务区|深圳中心商务区]]，以及市政府，亦在老深圳人认可的深圳城区内。华强北拥有深圳的地标之一[[赛格廣場|赛格廣場]]，为世界最高钢管混凝土结构大厦，也是深圳第三高楼。

==成就==
华强北商业区作为全国首批购物放心一条街于2000年获得中国国家工商局授牌。

[[华强北·中国电子市场价格指数|华强北·中国电子市场价格指数]]（华强北指数）于2007年10月在华强北诞生并向全球发布，使深圳成为中国电子市场的“风向标”、“晴雨表”。<ref>{{cite news |language = zh-hans |author = 陶然 |url = http://news.sina.com.cn/o/2007-10-16/094912735320s.shtml |title = 华强北 中国电子市场“晴雨表”“风向标” |publisher = 南方日报 |date = 2007-10-16 |accessdate = 2014-12-12}}</ref> 2010年11月8日，正式列为[[CCTV-2|CCTV-2]]《经济信息联播》数据采录单位；2011年 7月11日，华强北指数被[[中华人民共和国国务院办公厅|中华人民共和国国务院办公厅]]采用，成为国家进行[[宏观调控|宏观政策调控]]的重要参考数据。<ref>{{Cite web |url = http://www.hcsindex.org/about/ |title = 华强北指数介绍 |publisher = 深圳市华强北电子市场价格指数有限公司 |language = zh-hans |accessdate = 2014-12-12 |quote = 2010年11月8日，正式列为CCTV-2《经济信息联播》数据采录单位；2011年 7月11日，华强北指数正式登陆中华人民共和国国务院办公厅，成为国家进行宏观政策调控的重要参考数据 |archive-url = https://web.archive.org/web/20141213014437/http://www.hcsindex.org/about/ |archive-date = 2014-12-13 |dead-url = yes }}</ref>

在2008年第十届[[中国国际高新技术成果交易会|中国国际高新技术成果交易会]]（高交会）华强北分会场开幕仪式上，华强北获得中国[[国家电子商会|国家电子商会]]授予的“'''中国电子第一街'''”称号<ref name ="shb081013"></ref>，标志着行内确认了华强北商业街在全国电子商业界的龙头地位。

据分析机构预测深圳厂商的[[平板电脑|平板电脑]]出货量大约占到了全球平板电脑总出货量的45%，这些产品大部分由华强北销往中国和其他[[发展中国家|发展中国家]]。<ref>{{cite news |language = en |author = Eva Dou |url = http://blogs.wsj.com/digits/2014/07/14/inside-the-chinese-market-that-produces-45-of-the-worlds-tablets/ |title = Inside the Chinese Market That Produces 45% of the World’s Tablets  |publisher = The Wall Street Journal |date = 2014-01-14 |accessdate = 2014-12-12}}</ref>

==建设==
[[File:Entrance_of_Huaqiangbei,_2017.jpg|thumb]]
[[File:Huaqiangbei_walking_street.jpg|thumb]]
华强北是由始建于1982年的[[上步工业区|上步工业区]]改造而来。当时由于深圳特区成立不久，因此城区面积很小，上步工业区相当于市郊。当时原[[电子工业部|电子工业部]]、[[兵器工业部|兵器部]]、[[中华人民共和国航空工业部|航空局]]、广东省电子局等单位在这里成立了华强、赛格、爱华、京华、[[中国电子信息产业集团|中电]]等一系列电子工业企业。（现时部分地块已不属于人们通常认为的华强北）。这时，还没有华强北的概念，此处以电子工业生产为主。

1986年深圳市人民政府、广东省和电子工业部以省市所属的电子企业和部属电子企业为基础，吸收其他部门所属电子企业参加，组建深圳电子集团公司，1988年1月深圳电子集团公司正式更名为[[赛格集团|赛格电子集团]]，1988年3月28日成立了中国首家电子配套市场——赛格电子配套市场，当时由来自深圳本地和内地的160多家厂商以及10家港商，以自营自销、联营代销的方式经营。此时形成了华强北的雏形，上步工业区也从一个工厂区变成了一个国内举足轻重的电子元器件交易市场。

此后，电子产品一直较为畅销。1995年4月深圳国际电子城专业市场正式开业。1997年[[万商电器城|万商电器城]]、大百汇商业城开业。1998年与赛格大厦隔华强北路相望的[[华强电子市场|华强电子市场]]开业。华强北逐步成为[[中国|中国]]南方最大的电子产品的生产及批发零售中心。与此同时，深圳开始调整产业结构，并且上步工业区由于深圳的迅速发展，这片地区慢慢成了市中心而地价大涨，使得不少工业企业纷纷外迁。

由于电子市场规模不断扩大及[[第二产业|第二产业]]的退出，电子市场对周边相关配套行业需求越来越强烈。华强北的[[第三产业|第三产业]]开始迅速发展，以[[酒店|酒店]]及[[商场|商场]]为代表的[[服务业|服务业]]入驻，兼有向商贸、办公、[[银行|银行]]、[[证券|证券]]、[[房地产|房地产]]、[[餐饮|餐饮]]及其他行业，华强北变成以[[商业|商业]]、办公为主导、兼有部分居住的综合区。1994年，[[万佳百货|万佳百货]]入驻华强北，标志着华强北真正衍变成了城市商圈。

1998年开始，深圳市和福田区政府共同投资4500万元人民币对华强北商业区进行了首期改造，12个改造项目于1999年10月全面竣工。此外，由商家进行改造、装修等产生的投资总额达4亿元人民币。2004年底，[[深圳地铁1号线|深圳地铁1号线]]开通，设有[[华强路站|华强路站]]，出口就在华强北，更加速了租金上涨速度，而且人流越来越密集，因此产生了诸如区内交通拥堵和停车位等一系列问题，人们对第二期改造的呼声也越来越高。2006年，深圳市政府又进行“华强北换装”工程，将整个华强北商业街的外观进行升级改造。”<ref>{{cite news |language = zh-hans |author =  陈虹&乐粼 |url = http://www.chinadaily.com.cn/dfpd/2010-07/19/content_11020642.htm |title = 深圳华强北电子市场：电子行业的“风向标” |publisher = CHINADAILY中国日报 |date = 2010-07-19 |accessdate = 2014-12-12}}</ref>

为改变困扰中国电子第一街多年的“一街多治”格局，2009年7月1日，华强北正式成立了[[华强北街道|街道办事处]]。

因地铁7号线施工，2013年2月27日至2017年1月13日，原來雙程四車道的华强北路将进行围挡封路。<ref name="xkb130227"></ref>地鐵7号綫建成後，華強北路（红荔路—[[深南中路|深南中路]]）被改造為步行街，並於2017年1月14日開放。華強北路下方則建有[[地下街|地下街]]，已于2018年7月18日正式开业。

据统计，目前华强北共有各类专业市场23个，专业市场面积超过深圳全市总量的1/5，其中经营面积1万平方米以上的大型商场有14家，是电子配套、[[家电|家电]]、[[珠宝|珠宝]]、[[钟表|钟表]]、[[服装|服装]]等专业市场聚集的商业旺区。每日的客流量在50万人次以上，日资金流量达10亿元人民币。目前华强北的铺位租金是深圳最贵的地方之一，据估计，华强北的租金10年内涨了10倍。此外，華強北的[[茂業百貨|茂業百貨]]（已改造為[[茂業天地|茂業天地]]）是深圳市營業額最高的百貨公司之一，2015年營業額達二十多億元人民幣。

==深圳严打违法经营运动==
2012年，经过高压集中整治，华强商业圈出现“关、停、并、转”潮，一些依靠违法手段经营的商户失去生存空間，华强北手机专业市场3575个商户主动退场。国内外许多知名手机商纷纷进驻华强北，华强北正在严打契机下走上转型。<ref>{{cite news |language = zh-hans |url = http://www.s1979.com/shenzhen/201209/0651698706_2.shtml |title = 深圳华强北3575山寨商户退市 危机中谋变 |publisher = 南方都市报 |date = 2012-09-06 |accessdate = 2014-12-12 |deadurl = yes |archiveurl = https://web.archive.org/web/20141215022329/http://www.s1979.com/shenzhen/201209/0651698706_2.shtml |archivedate = 2014-12-15 }}</ref>

==交通==
=== 道路 ===
毗邻深圳主要干道——深南中路人民南板块，交通可谓四通八达。东西向直线贯穿盐田区、罗湖区、南山区和宝安区。
由于地处中心地区，华强北片区道路在上下班高峰时段会十分繁忙。

=== 公共交通 ===
华强北设有四个地铁站，分别是[[深圳地铁1号线|深圳地铁1号线]]的[[华强路站|华强路站]]，[[深圳地铁2号线|2号线]]、[[深圳地铁7号线|7号线]]的[[华强北站|华强北站]]、[[深圳地铁2号线|2号线]]的[[燕南站|燕南站]]，和[[深圳地铁3号线|3号线]]、[[深圳地铁7号线|7号线]]的[[华新站|华新站]]，四个地铁站使用率十分高，並經常出現拥挤情況。未来[[深圳地铁11号线|深圳地铁11号线]]将在华强北片区设置[[上海宾馆站|上海宾馆站]]。

华强北有多条公交线路，整个华强北交通网络已非常成熟
