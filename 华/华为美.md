{{单双书号转换|'''华为美'''}}是一首由[[李幼容|李幼容]]和[[臧思佳|臧思佳]]作词，[[郑冷横|郑冷横]]作曲的一首[[儿童歌曲|儿童歌曲]]，以[[华为|华为]]手机和中国[[爱国主义|爱国主义]]为主题。歌曲于2019年2月25日以[[音乐视频|音乐视频]]的形式在中国[[珠海|珠海]]儿童合唱团周丹少儿声乐教室官方[[微信公众号|微信公众号]]上首次发布。<ref name="联合早报">{{cite web |title=《华为美》据报有浓厚红色背景 华为：不知情未参与 |url=https://www.zaobao.com.sg/znews/greater-china/story20190301-935911 |website=联合早报 |date=2019-03-01 |access-date=2019-04-07 |archive-url=https://web.archive.org/web/20190307154809/https://www.zaobao.com.sg/znews/greater-china/story20190301-935911 |archive-date=2019-03-07 |dead-url=no }}</ref>

== 作品内容 ==
{{单双书号转换|华为美}}在歌词中称赞了华为手机电池续航时间长、[[手机芯片|手机芯片]]为中国自主研发、造型优美、名字寓意「中华有为」等优点，并表示儿童应该从小树立远大的志向，长大后像华为一样为祖国作贡献。该音乐视频的主要拍摄地点为[[珠海大剧院|珠海大剧院]]门口的广场<ref>{{cite web |author1=RAYMOND ZHONG |title=中国孩童给华为唱赞歌：“天下最美”手机 |url=https://cn.nytimes.com/technology/20190301/huawei-children-music-video/ |website=纽约时报中文网 |date=2019-03-01 |access-date=2019-04-07 |archive-url=https://web.archive.org/web/20190307040004/https://cn.nytimes.com/technology/20190301/huawei-children-music-video/ |archive-date=2019-03-07 |dead-url=no }}</ref>，参与表演与演唱的儿童有29名，分别来自中国内地、[[香港|香港]]和[[澳门|澳门]]，并身着印有「中国」、「China」字样的[[T恤|T恤]]。<ref name="联合早报" />

== 创作背景 ==
2018年7月，[[美國|美國]]、[[澳大利亞|澳大利亞]]、[[加拿大|加拿大]]、[[紐西蘭|紐西蘭]]、[[英國|英國]]等国以網絡安全風險为由对华为的产品进行限制<ref>{{Cite news|url=https://www.bbc.com/zhongwen/simp/chinese-news-46390757|title=华为被“五眼”情报联盟视作威胁 面临抵制|date=2018-12-03|newspaper=BBC中文網|publisher=[[BBC|BBC]]|accessdate=2018-12-06|language=zh-hans|archive-url=https://web.archive.org/web/20181206114013/https://www.bbc.com/zhongwen/simp/chinese-news-46390757|archive-date=2018-12-06|dead-url=no}}</ref>。2018年12月1日，華為公司現任[[副董事長|副董事長]]兼[[財務長|財務長]][[孟晚舟|孟晚舟]][[孟晚舟事件|被加拿大警方逮捕]]<ref name="背后阴谋">{{cite web |title=外媒曝光孟晚舟事件背后阴谋:有组织有预谋的打压 |url=http://money.163.com/18/1218/08/E39VO7F300259CAL.html |website=网易-环球网 |accessdate=2018-12-23 |date=2018-12-18 |archive-url=https://web.archive.org/web/20181218050345/http://money.163.com/18/1218/08/E39VO7F300259CAL.html |archive-date=2018-12-18 |dead-url=no }}</ref>。这一系列事件引起中国政府和不少中国民众的不满，不少中国民众表示支持华为并谴责加拿大政府的行为。<ref name="联合早报" />

== 评价 ==
=== 华为官方评价 ===
该歌曲的音乐视频发布之后，华为官方[[新浪微博|新浪微博]]帐号花粉俱乐部表示该歌曲并非由华为官方制作，而是由喜欢华为的爱好者所制作的，并对支持华为的用户表示感谢，并承诺会继续把华为的产品做得更好，以回馈消费者的信任与支持。<ref>{{cite web |title=华为官微回应《华为美》MV：不知情 非官方出品 |url=https://tech.sina.com.cn/mobile/n/n/2019-02-27/doc-ihsxncvf8333021.shtml |website=新浪科技 |date=2019-02-27 |access-date=2019-04-07 |archive-url=https://web.archive.org/web/20190227141918/https://tech.sina.com.cn/mobile/n/n/2019-02-27/doc-ihsxncvf8333021.shtml |archive-date=2019-02-27 |dead-url=no }}</ref>

=== 正面评价 ===
有网友表示，{{单双书号转换|华为美}}旋律动听，且参与作词作曲的是年龄较高的老艺术家，能够对中国出品的作品如此热爱，值得钦佩。有网友表示，歌曲体现了中国的强大，对此感到自豪。<ref name="联合早报" /><ref name="苹果">{{cite web |title=小朋友拍愛國MV挺華為　中國網友怒批：這就是高級黑 |url=https://tw.appledaily.com/new/realtime/20190228/1525069/ |website=苹果即时 |date=2019-02-28}}</ref>

=== 负面评价 ===
有网友表示，歌词用语不自然，听起来很尴尬。有网友表示，歌词中提到的中国自主研发的芯片真实性存疑。有网友表示该歌曲可能会起到适得其反的负面效果。<ref name="苹果" />
