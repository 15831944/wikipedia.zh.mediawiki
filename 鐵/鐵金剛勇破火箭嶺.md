《'''鐵金剛勇破火箭嶺'''》（{{lang-en|''You Only Live Twice''}}），是1967年拍的第5部[[詹姆士·龐德|詹姆士·龐德]]系列影片，這是首齣有[[華裔|華裔]][[邦女郎|邦女郎]]（[[周采芹|周采芹]]）演出的詹姆士·龐德電影。片中加入了大量亞洲元素，片頭以中國、日本式的[[紙傘|紙傘]]作主題。

== 劇情簡介 ==
美國[[人造衛星|人造衛星]]「木星16號」，在進行[[太空漫步|太空漫步]]時，被不明飛行物體攻擊，下落不明。在美英蘇高峰會上，美國指控蘇聯攻擊美國衛星，並表示二十天後將再度發射[[太空梭|太空梭]]，要求蘇聯不得干擾，否則開戰，而蘇聯則極力否認。英國則表示夏威夷雷達站偵測到不明飛行物體的微弱訊號來自日本，因此認為是[[蘇聯|蘇聯]]所為，並表示已有情報員在香港進行調查。與此同時，[[英國|英國]][[軍情六處|軍情六處]][[特工|特工]][[詹姆士·龐德|詹姆士·龐德]]正與中國美女共度春宵，但沒想到是[[美人計|美人計]]，龐德遭到埋伏槍手射殺。不久，英國為龐德舉行海葬，龐德的棺木被投入水中，但沒想到龐德卻復活了，原來這是龐德為掩飾身分的計謀，上司M將指派龐德調查「木星16號」失蹤的事件，查明原因，以避免爆發[[世界大戰|世界大戰]]，但背後的層層陰謀卻指向了日本......

== 製作花絮 ==
《[[雷霆谷|雷霆谷]]》曾於1970年與第四部[[占士·邦|占士·邦]]系列影片《[[霹靂彈|霹靂彈]]》再次上映。
