'''鐵路線'''是由出發地（起點）至目的地（終點）之間鋪設的[[鐵路|鐵路]]軌道路段。廣義上亦可以指列車運行的[[運行系統|運行系統]]與設有{{link-ja|愛稱|愛称}}的路段。

鐵路線互相聯絡，三重以上的{{link-ja|迂回路|迂回路}}而形成網狀物體稱為「鐵路網」。

== 概要 ==
狹義來說，只有後述基於法令定義才是「路線」。

但是實際上的列車並不限於遵從狹義的路線運行，一條路線可以用於多個運行系統，一個運行系統也可以跨越多條路線。例如[[京濱東北線|京濱東北線]]就是跨越多條路線的運行系統，此處則為廣義的「路線」。

另外，[[東海道本線|東海道本線]]的一部分路段如[[JR京都線|JR京都線]]，狹義的路線也越來越多把並不一致的路段愛稱（路線愛稱）也稱作「路線」。

像[[山手線|山手線]]則有路線與運行系統，雙向使用但雙者並不一致的名稱（參見該條目開頭）。

== 日本法令上的鐵路線 ==
日本在法令上雖然沒有列車的運行系統，但官方規定起點與終點之間為一條路線。

根據{{link-ja|鐵道事業法|鉄道事業法}}（昭和61年法律第92號）第4條，凡是收到鐵道事業許可者，將作為「預定路線」（{{lang|ja|予定する路線}}）向國土交通大臣提出申請書記載於該項目內。

鐵道事業法施行規則（昭和62年運輸省令第6號）第3條，上述的「預定路線」應當記載「起點及終點」、「主要的經過地」。<!-- 定義等あまりよく分かりませんのでとりあえず法令があげておきます。加筆お願いします。-->

== 日本鐵路線名稱 ==
{{see also|鐵道路線的名稱}}
[[日本國有鐵道|日本國有鐵道]]（國鐵）時代的路線名稱起源於「{{link-ja|國鐵·JR線路名稱列表|国鉄・JR線路名称一覧|國有鐵道線路名稱}}」。[[國鐵分割民營化|國鐵分割民營化]]後，各間公司規定了「線路名稱」。私鐵也作出了與此同樣的規定。
