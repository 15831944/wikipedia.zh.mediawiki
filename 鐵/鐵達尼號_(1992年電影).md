'''泰坦尼克号（{{lang-en|Titanica}}）'''是一部1992年发行的有关[[泰坦尼克号|泰坦尼克号]]的[[IMAX|IMAX]]纪录片。该影片导演为{{tsl|en|Stephen Low|史蒂芬·洛}}，旁白由{{tsl|en|Cedric Smith (actor)|塞德里克·史密斯(演员)|塞德里克·史密斯}}，{{tsl|en|Anatoly Sagalevich|阿那托利·萨加洛维奇}}以及[[拉尔夫·韦特(探险家)|拉尔夫·韦特]]录制。该影片主要记录了[[泰坦尼克号残骸|泰坦尼克号残骸]]的镜头，探险队员寻找残骸的镜头，<ref name=ebert/>以及对幸存者{{tsl|en|Frank John William Goldsmith|法蘭克·約翰·威廉·戈德史密斯}}和{{tsl|en|Eva Hart|伊娃·哈特}}的采访。<ref>{{cite web |url=http://www.allmovie.com/movie/titanica-v174282 |title=Titanica (1992) |work=[[AllMovie|Allmovie]] |accessdate=2014-11-03 |archiveurl=https://web.archive.org/web/20160414183901/http://www.allmovie.com/movie/titanica-v174282 |archivedate=2016-04-14 |deadurl=no }}</ref>洛利用该纪录片，借伊娃和船组成员，在展现泰坦尼克号残迹之余，传达了这样一个声音，即泰坦尼克号的残骸应当受到重视并妥善对待。该纪录片是继1991年{{tsl|en|Stones at the Max||Stones at the Max}}之后的第二部IMAX影片。<ref name=ebert>{{cite web |url=http://www.rogerebert.com/reviews/titanica-1995 |title=Titanica |last1=Ebert |first1=Roger |authorlink=Roger Ebert |date=1995-04-14 |work=RogerEbert.com |accessdate=2014-11-03 |archiveurl=https://web.archive.org/web/20141103065220/http://www.rogerebert.com/reviews/titanica-1995 |archivedate=2014-11-03 |deadurl=no }}</ref>随后，1995年发行了一部40分钟的剪辑版IMAX剧场影片。<ref name=marg>{{cite web |url=http://www2.cincinnati.com/freetime/movies/mcgurk/112500_titanica.html |title=Big-screen 'Titanica' |last1=McGurk |first1=Margaret |date=2000-11-25 |work={{tsl|en|The Cincinnati Enquirer||The Cincinnati Enquirer}} |accessdate=2014-11-03 |archiveurl=https://web.archive.org/web/20141103114630/http://www2.cincinnati.com/freetime/movies/mcgurk/112500_titanica.html# |archivedate=2014-11-03 |deadurl=yes }}</ref>该版本由[[伦纳德·尼莫伊|伦纳德·尼莫伊]]配音，尽管其中大部分配音工作皆由怀特完成。<ref name=forty>{{cite web |url=https://www.imax.com/movies/m/titanica/ |title=Titanica |work=[[IMAX|IMAX]] |accessdate=2014-11-03 |archiveurl=https://web.archive.org/web/20141103061103/https://www.imax.com/movies/m/titanica/ |archivedate=2014-11-03 |deadurl=no }}</ref>该版本随后也成为1997年版本的基础，在后者版本中，有超过27分钟都是由拉尔夫·韦特，{{tsl|en|Emory Kristof|埃默里·克里斯托夫}}以及其他专家进行的采访。

==拍摄==
探险队员由美国，法国以及俄罗斯人组成，<ref name=ebert/>他们搭乘{{tsl|en|Akademik Mstislav Keldysh|克尔德什院士号}}研究船前往。<ref name=marg/>残骸镜头皆由两艘[[和平號_(潛水艇)|和平号潜水艇]]拍摄，<ref name=marg/>它们有时候会合力拍摄。其上配有IMAX摄像机和150,000瓦特照明灯，可清楚地照亮海底。<ref name=ebert/>残骸镜头常与历史照片一同比较，以展示这一悲剧的影响。

影片中，伊娃·哈特提到，她的母亲在泰坦尼克号沉没事故之前曾经说到，泰坦尼克号永不沉没这一称谓就如同“在全能者的面前飞翔”一样。<ref name=ebert/><ref name=marg/><ref name=ott/>

==评价==
根据[[爛番茄|爛番茄]]上的5个评测，该影片的点赞率为60％。<ref>{{cite web |url=http://www.rottentomatoes.com/m/titanica/ |title=Titanica (1995) |work=[[爛番茄|Rotten Tomatoes]] |accessdate=2014-11-03 |archiveurl=https://web.archive.org/web/20140812053615/http://www.rottentomatoes.com/m/titanica/ |archivedate=2014-08-12 |deadurl=no }}</ref>[[罗杰·埃伯特|罗杰·埃伯特]]给予该影片3½星的分数(满分为4星)，并称其镜头与泰坦尼克号残骸“建立起了紧密的联系”，但是他也称自己愿意看到更多有关泰坦尼克号的镜头，而非探险队员。<ref name=ebert/>2000年，{{tsl|en|The Cincinnati Enquirer|辛辛那提调查报}}的玛格丽特·麦克戈克给出了正面的评价，并且特别称赞了影片的细节与镜头尺寸，称“这是一个超大IMAX电影真正展现其潜力的例子”。<ref name=marg/>{{tsl|en|NUVO (newspaper)||NUVO}}的爱德华·约翰逊-奥特对残骸与探险队员的镜头都赞赏有加，并称队员们添入了“一些无伤大雅的幽默，与此同时，仍保持了这种旅行中所需要的尊严”。<ref name=ott>{{cite web |url=http://www.imdb.com/reviews/133/13311.html |title=Titanica (1995) |last1=Johnson-Ott |first1=Edward |date=1998 |work=[[互联网电影数据库|IMDb]] |publisher={{tsl|en|NUVO (newspaper)||NUVO}} |accessdate=2014-11-03 |archiveurl=https://web.archive.org/web/20150321180942/http://www.imdb.com/reviews/133/13311.html |archivedate=2015-03-21 |deadurl=no }}</ref>

伊娃·哈特曾对泰坦尼克号残骸遭到掠夺表示担忧。而对于该影片，她评论道：“我不认为在这场IMAX影片中探索泰坦尼克号是在掠夺，相反，我认为它相当精彩。<ref name=ebert/>
