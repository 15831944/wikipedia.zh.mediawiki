'''鐵路車輛'''，即'''車卡'''，俗稱'''火車車廂'''，是指除了專門提供動力的[[鐵路機車|機車]]以外，其他各種在鐵路行走的車輛。這些車輛少部份有[[發動機|發動機]]，但是大部分都是沒有動力的。[[鐵路|鐵路]]車輛連接一起成為[[列車|列車]]或[[火車|火車]]，由機車牽引或靠自帶的發動機行走。自帶發動機行走的列車被稱為[[動車組|動車組]]。

鐵路車輛的主要用途是載客或載貨。有些車輛是為鐵路本身的[[維修|維修]]等原因而設。按用途，鐵路車輛一般分為客車及貨車。

== 客車 ==
{{main|鐵路客車}}
客車或客車車廂可以有多種內部設計：
[[File:InterCity2_-_passenger_car_interior.jpg|thumb]]
[[File:JRC_Shinkansen_Series_N700_Z13_sets_785-13.jpg|thumb]]客車廂]]
[[File:SYZ25K_345648@MYN_(20150106163734).JPG|thumb]]双层客车]]
[[File:JRC_Shinkansen_Series_N700_Interior_of_Green_car.jpg|thumb]]的綠色車廂座椅]]
[[File:Sleeping_bunk_on_the_overnight_to_Paris.jpg|thumb]]]]
座位通常是橫排三、四或五個；中間有行人走廊。組合可能是 2+1, 2+2 或 3+2。如果座位是面對面的，中間可能有小桌子。面向相同方向的座位，有時使用前面座位背後可摺疊的小[[桌子|桌子]]。

*如果座位中間有走廊，座位可能面向相同方向；或每兩行座位面對面坐，組成一組，甚至結合上述兩種座位排定方式，稱為「非」字型。在[[通勤鐵路|通勤鐵路]]或[[城市軌道交通系統|地下鐵路]]，座位有時是橫向面對中央的走廊。

*如果走廊是在旁邊的話，車廂多數是間格成小空間，每個有兩行六或八個座位面對面座。

客車一般有[[空氣調節|空氣調節]]，或可打開的車窗。（部份有空氣調節的客車為安全考慮，亦有必要時可打開的車窗，稱為「緊急通風窗」）。廁所是很多客車都有的配備，雖然[[客車廁所|客車廁所]]所使用的技術很不一樣。

其它種類的客車車廂包括有：[[餐車|餐車]]、電影院車、迪斯可車等等；通常這些車輛只會掛在長途客車上。可供觀看車外風光的[[觀光車|觀光車]]近年很受歡迎，有些國家特意製造玻璃圓頂的車廂，讓乘客可以仔細觀看車外景色。

[[卧車|卧車]]一般會有小小的眈室間格讓乘客在晚間睡覺；坐卧两用车(Couchette)則只有最基本，勉強可供躺下睡覺的座位。

亦有客車是雙層的，用以增加載客量。

一個'''列車組'''，是指並非臨時集合，而是長時期一起使用的車廂組合。它們只會有在維修廠才會被分開。

車廂之間通常會有有[[彈性|彈性]]的通道連接，讓乘客或車務人員可以在中間走過。有些車廂的連接通道是全寬的，而且是半永久連接。這樣的列車就好像是一個很長，但是可扭曲的車廂。在市區行駛的[[城市軌道交通系統|地下鐵路]]經常有這種設計，用來增加載客量。

== 貨車 ==
[[File:High_fructose_corn_syrup_tanker.jpg|thumb]]的液態運輸火車]]
[[File:Railroad_car_with_container_loads.jpg|thumb]]
[[File:DTTX_724681_20050529_IL_Rochelle.jpg|thumb]]
貨車車廂有很多不同的種類，以適應在不同的場合使用。最初出現的貨車是普通的棚車，車的四面好像是箱子一樣，旁邊有可以打開的敞門。

其它的貨車有：
*[[汽車多層平車|汽車多層平車]]：特別用來運載汽車的車廂，通常有多層。
*[[棚車|棚車]]：棚车是有车顶、四面有侧板的货车车厢，用于运送贵重的、或者怕日晒雨林的货物。必要时也可以运送人员和动物。
*[[保温車|保温車]]：帶有冷凍設備，或[[保温|保温]]設備的有篷貨車。
*[[平車|平車]]：用來運載不能放入棚車的大型物件。當中[[低邊平車|低邊平車]]用來運載特大貨物。隨著[[集裝箱|集裝箱]]（貨櫃）運輸的增加，有的平車設計用來專門運載集裝箱，或帶拖車架的集裝箱拖車。有的集裝箱專用平車可以運載雙層集裝箱。以平車載運帶拖車架的集裝箱（貨櫃）的運輸方式，在交通運輸學上被稱為Piggy Back。另有大物車，大物車為特殊型式平車，但其淨空較大，無一般平車普遍，英文稱為Depressed Center Flat Car。
*[[敞車|敞車]]：只有在四面有圍板的車箱，用來運載可能滑下車廂的散裝貨物。有時亦用來運載集裝箱。
*[[漏斗車|漏斗車]]：一種特殊的敞車，特點是可以在車廂的底部把貨物卸下。用來運載煤炭、壙砂、穀物、水泥等貨物。有有蓋及無蓋之分。
*[[罐車|罐車]]：用來運載各種液體。
*[[家畜車|家畜車]]：用來運載牲口。
*[[长大货物車|长大货物車]]
*[[毒品車|毒品車]]
*[[粮食車|粮食車]]
*[[集装箱車|集装箱車]]
*[[矿石車|矿石車]]

== 其他車輛 ==
這些車輛並不運載貨物或乘客：
*[[守車|守車]]：一般是掛在貨車的尾部，用來守望車輛及協助剎車；台灣的守車，也出現在客車上。但中国大陆的守车只用于货运列车。
*[[維修車|維修車]]：用來維修[[鐵軌|鐵軌]]或設備；台灣稱工程車；日本稱事業用車。
*[[台车|手動車]]：車上乘客推動的車輛。
*[[特种車|特种車]]

== 軍事車輛 ==
軍事上使用的[[裝甲列車|裝甲列車]]可能乘載以下各種裝備：
*火炮：載有火炮及機槍，甚至可能有[[飛彈|飛彈]]。
*步兵：有機槍及[[步兵|步兵]]。
*機槍：專載有[[機槍|機槍]]。
*防空：有[[高射炮|高射炮]]或機槍。
*指挥：用來作指揮中心。
*反坦克：有[[反坦克炮|反坦克炮]]，通常有[[坦克|坦克]]炮塔。
*平車：無[[裝甲|裝甲]]，用來運輸或維修。
