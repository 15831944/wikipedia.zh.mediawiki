'''罢免选举'''是指为免去一位当选官员职务而进行的特别投票[[选举|选举]]。同[[創制權|創制權]]和[[复决权|复决权]]一样，罢免选举是19世纪末20世纪初，[[美国|美国]][[进步运动|进步运动]]提倡的一种主要的选举改革。罷免選舉與[[彈劾|彈劾]]不同的是，被彈劾的官員是涉及干犯嚴重的罪行，並由[[議會|議會]]投票罢免，而罢免投票則是選民通過投票，提前結束民选官员的任期。

== 加拿大 ==
在[[加拿大|加拿大]]，[[不列顛哥倫比亞省省議會|不列顛哥倫比亞省省議會]]在1995年容許罢免选举。

== 美國 ==
光是2011年一年當中，[[美國|美國]]就舉行了150次罷免選舉，有75位官員被罷免，9位官員在罷免的聲浪下辭職<ref>{{cite web|url=http://recallelections.blogspot.tw/2011/12/year-in-recalls-150-recalls-in-2011.html |title=The Recall Elections Blog: The Year in Recalls - 151 Recalls in 2011 (edited to add another recall) |publisher=Recallelections.blogspot.tw |date=2011-12-27 |accessdate=2014-05-03}}</ref>

[[美國|美國]]州長罷免選舉當中，2位州長被成功罷免，分別是1921年被罷免的[[北达科他州|北达科他州]]州長Lynn Frazier，以及2003年加州州長Gray Davis。2012年，美國[[威斯康辛州|威斯康辛州]]州長[[斯考特·沃克|斯考特·沃克]]因廢除[[工會|工會]]的[[集體談判權|集體談判權]]被公民連署發動罷免，但在罷免選舉中勝出，保住州長一職，是首位在罷免選舉中勝出的州長。

== 中華人民共和國 ==
[[中华人民共和国宪法|中华人民共和国宪法]]规定[[全国人民代表大会|全国人民代表大会]]作为最高国家权力机关，有权罢免[[中华人民共和国主席|国家主席]]、[[中华人民共和国副主席|副主席]]、[[中华人民共和国国务院总理|国务院总理]]、[[中华人民共和国国务院副总理|副总理]]、[[中华人民共和国中央军事委员会主席|国家军委主席]]、[[中华人民共和国中央军事委员会副主席|军委副主席]]等[[党和国家领导人|国家领导人]]。1968年[[文化大革命|文化大革命]]期间，时任国家主席[[刘少奇|刘少奇]]被[[中国共产党第八届中央委员会第十二次全体会议|中共八届十二中全会]]撤销职务，该罢免案违反了当时宪法的规定。1980年的[[中国共产党第十一届中央委员会|中共十一届五中全会]]宣布撤销该违宪决议。

== 委內瑞拉 ==
在[[委內瑞拉|委內瑞拉]]，2000年後通過的委內瑞拉憲法，需以罢免选举作為罢免[[委內瑞拉總統|委內瑞拉總統]]的方式，取代過去由[[委內瑞拉國會|委內瑞拉國會]]彈劾總統的權力。2000年委內瑞拉憲法設立了罷免總統的全國公民投票制度—也就是委內瑞拉的人民能夠以公民投票的方式提前撤換總統。這樣的公民投票必須有一定數量的連署才能發起。憲法的條款要求必須收集20%選民的簽名才能發起罷免公投。在2004年實行過一次，但並未成功。
