'''鄢烈山'''{{BD|1952年|1月|||catIdx= Y鄢}}出生于[[湖北省|湖北省]][[仙桃市|仙桃市]]，毕业于[[北京师范大学|北京师范大学]]中文系，为《南方周末》编辑，杂文家、时事评论家、作家、[[公共知识分子|公共知识分子]]。<ref name=" fish" >{{cite web |url= http://finance.sina.com.cn/roll/20040930/16321059977.shtml|title=鄢烈山简介 |accessdate=2004年9月30日 |publisher=新浪网 |archiveurl=http://finance.sina.com.cn/roll/20040930/16321059977.shtml |archivedate=2012年11月5日 |work= |language =zh-cn}}</ref>

== 生平 ==
1952年，鄢烈山出生在[[湖北省|湖北省]][[仙桃市|仙桃市]]，[[文革|文革]]期间，鄢烈山当过社员、工人、农民、民办教师，恢复高考之后，考入[[北京师范大学|北京师范大学]]中文系，大学毕业后，曾在《[[武汉晚报|武汉晚报]]》、《[[长江日报|长江日报]]》任职，1995年，鄢烈山加入《[[南方周末|南方周末]]》。<ref name=" man">{{cite web |url=http://www.daynews.com.cn/news/375596.html |title=鄢烈山 |accessdate=2007-10-18 |publisher=山西新闻网 |archiveurl=https://web.archive.org/web/20071024235036/http://www.daynews.com.cn/news/375596.html |archivedate=2007年10月24日 |work= |language=zh-cn |deadurl=yes }}</ref>

== 风格 ==
鄢烈山崇尚“公民写作”，捍卫[[公民|公民]]权力，主张“我手写我心”。<ref name=" fish" /><ref name=" man" /> 其作品主要针对社会事件针砭时弊、纵横谈论。<ref name=" fish" /><ref name=" man" />
