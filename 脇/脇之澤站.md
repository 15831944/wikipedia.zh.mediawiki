'''脇之澤車站'''（{{jpn|j=脇ノ沢駅|hg=わきのさわえき|rm=Wakinosawa eki|f=y}}），位於[[日本|日本]][[岩手縣|岩手縣]][[陸前高田市|陸前高田市]]米崎町字脇之澤，為[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[大船渡線|大船渡線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
BRT開始運行時設置了BRT用的站舍。

東日本大震災前為[[側式月台|側式月台]]1面1線的[[地面車站|地面車站]]，設有[[候車室|候車室]]的[[無人站|無人站]]，由[[氣仙沼站|氣仙沼站]]管理。

== 車站週邊 ==
* 米崎郵局
* [[普門寺_(陸前高田市)|普門寺]]
== 歷史 ==
* 1933年12月15日 - 設站。
* 1983年3月10日 - 改為招呼站。
* 1987年4月1日 - [[日本國鐵分割民營化|日本國鐵分割民營化]]後成為JR東日本的車站。
* 2011年3月11日- 被[[東北地方太平洋近海地震|東北地方太平洋近海地震]]引發的海嘯摧毀。<ref>{{cite web|url=http://kenplatz.nikkeibp.co.jp/article/building/news/20110316/546453/?P=3 |title= 被災地発・陸前高田6：壊滅した高校、図書館、病院、駅……|publisher=ケンプラッツ |date=2011-03-22 |accessdate=2011-03-26 }}</ref>
* 2013年3月2日 - 變為[[BRT|BRT]]車站，不再以鐵路行走。
