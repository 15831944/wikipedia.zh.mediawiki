'''潼关县'''，位于[[陕西省|陕西省]][[渭南市|渭南市]]东部，与[[河南|河南]]省[[灵宝|灵宝]]市交界，隔[[黄河|黄河]]与[[山西|山西]]省[[芮城|芮城]]县相望，西接[[华阴|华阴]]市，南依[[洛南|洛南]]县。因境内[[潼关|潼关]]而得名。总面积526平方公里，人口15万。<ref>[http://www.sn.xinhuanet.com/2005-05/30/content_4333841.htm 潼关] {{webarchive|url=https://web.archive.org/web/20080114014952/http://www.sn.xinhuanet.com/2005-05/30/content_4333841.htm |date=2008-01-14 }}</ref>

== 历史沿革 ==
*[[武周|武周]][[天授|天授]]二年(691年)分[[仙掌|仙掌]]县东部设潼津县，长安二年(公元702年)撤潼津县，并入仙掌县。 

*[[明太祖|明太祖]][[洪武|洪武]]七年(1374年)设潼关守御千户所，九年(1376年)，设潼关卫。

*[[清世宗|清世宗]][[雍正|雍正]]二年(1724年)撤潼关卫，五年(1727年)，设潼关县。

*[[清高宗|清高宗]][[乾隆|乾隆]]十三年(1748年) 改设潼关厅。

*[[中华民国|中华民国]]二年(公元1913年)废厅，设潼关县。

*1958年，与渭南县合并，设潼关人民公社。1961年恢复潼关县至今。<ref>[http://www.tongguan.gov.cn/tongguan/lshyg.asp 历史沿革] {{webarchive|url=https://web.archive.org/web/20070829055805/http://www.tongguan.gov.cn/tongguan/lshyg.asp |date=2007-08-29 }}</ref>

== 地理 ==
潼关县地处[[陝|陝]]、[[晋|晋]]、[[豫|豫]]三省交界处，是[[陕西|陕西]]的东大门，总面积526平方公里。

潼关位于[[鄂尔多斯高原|鄂尔多斯高原]]东南角，[[黄河|黄河]]、[[渭河|渭河]]在此以北交汇，而潼关的南面是[[秦岭|秦岭]]。在此南下的黄河由于秦岭的阻挡而急转向东，形成“几”字型的最后一个急转。因为它是[[关中平原|关中平原]]和[[中原|中原]]之间的交通要塞，在中国古代长期拥有重要的战略地位。

== 风景名胜 ==

[[潼关|潼关]]位于'''潼关县'''北部，为[[山西|山西]]、[[陕西|陕西]]、[[河南|河南]]三省要冲，地理位置十分重要、自古是兵家必争之地。唐代東都洛陽到長安必需經過此處黄河関隘。其後歴朝巡按陝西，復由此入秦。

== 自然资源 ==
盛产[[黄金|黄金]]，有大小金矿200余个，年产黄金近50万两，是全国第三产金大县。
