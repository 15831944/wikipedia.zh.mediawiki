{{selfref|關於維基百科對條目的定義，詳見'''[[Wikipedia:什么是条目|Wikipedia:什么是条目]]'''。}}
'''條目'''是[[百科全書|百科全書]]所含內容的基礎分割[[單位|單位]]（在词典中则称为'''词条'''），有一個[[單一|單一]]的[[主題|主題]]，用於[[闡述|闡述]]一件[[事物|事物]]、一個[[角色|人物]]、或他們具備特定主題的組合，並且對該對[[條目|條目]]所描述之事物作出一個客觀的介定。

一般而言，百科条目的标题（称为'''条头'''）通常为[[名词|名词]]或名词性词组<ref>{{cite journal|author1=全如瑊|title=术语辞书编纂中的元数据设计|journal=中国科技术语|date=2015|volume=17|issue=3|pages=5-13|url=http://www.term.org.cn/CN/abstract/abstract9445.shtml|accessdate=2015-12-17}}</ref>。

==大小条目主义==
在百科全书编纂中，一直有大小条目主义之争。小条目主义以《[[布罗克豪斯社交词典|布罗克豪斯社交词典]]》为代表，突出了百科全书的[[工具书|工具书]]性质，方便查阅；但是与大条目主义的百科全书相比，则往往丧失了[[知识|知识]]的系统性<ref name="金常政" />{{rp|53-54}}。

==条目的组成与常用原则==
百科全书的条目包括标题、正文内容和图表等。在百科全书的编纂中，条目的命名非常重要。好的命名应该便于检索，有学者认为条目应该都是独立的主题，应以方便读者的检索为准则。另外，条目的主题应该是客观形成的，而不是人为拟定的。例如[[海湾战争|海湾战争]]是客观存在的事实，而“狗的神话”则是人为拟定的主题<ref name="胡人瑞">[[胡人瑞|胡人瑞]]，百科全书选条的几个问题，编辑之友.2001(4).-36-39</ref>

条目的命名应该清晰明确，不能含糊不清。例如「新型武器」，很难界定“新型”的划分<ref name="胡人瑞" />。

百科全书条目的选择有一些共通性的原则<ref name="胡人瑞" />：
# 独立主体原则：主体应该是相对独立的，就像我们去一个[[公司|公司]]办事，我们先想到的是公司的名称、地址，而不是其中的某个部门。
# 客观形成原则：这个主题应该是人们在了解，改造世界的过程中客观形成，为人所熟知，而不是人为拟定的。如“[[创世神话|创世神话]]”是一个客观存在的神话题材类型，但是“狗的神话”就是一个人为概括的主题。因此缺少公认的'''规范性'''和'''确定性'''。
# 单一主题原则：如“时间和空间”是两个主题，应该分设。
# 准确性原则：条目的名称应该准确的表明条目的主题。
# 通用性原则：应该使用规范的或约定俗成的名称。
# 名词性原则：条目名称应该是名词性的，静止的。如“解放海南岛”应该改成“[[海南岛战役|海南岛战役]]”
# 简要性原则：如使用“[[唐诗|唐诗]]”，而不用唐代诗文。
# 非研究原则：百科全书不是研究论文。
# 非应用原则：百科全书不是为了指导具体的应用。

此外，条目之间还具有组群关系。例如[[太阳系|太阳系]]中的各大行星条目就属于一个同类型的条目组群。条目组群还包括所谓的“母子条”、“世系条”等具有从属关系的条目<ref name="金常政" />{{rp|90-91}}。例如著名的网络百科全书[[维基百科|维基百科]]就有“特色主题”的评选，來挑选出撰写非常优秀的一组相互有关联的条目<ref>{{cite web|author=Wikipedia contributors|title=Wikipedia:Featured topics|url=http://en.wikipedia.org/w/index.php?title=Wikipedia:Featured_topics&oldid=438457010|publisher=Wikipedia, The Free Encyclopedia.|accessdate=2011-07-11|date=8 July 2011 19:02 UTC}}</ref>

== “参见”系统 ==
[[百科全书|百科全书]]中常常会使用“参见”（See [also]）系统，将与主题相关的知识串联起来。最早使用参见方法的的意大利人[[班迪尼|班迪尼]]编写的《[[宇宙大事源|宇宙大事源]]》<ref name="金常政" />{{rp|124-125}}。

在《[[大英百科全书|大英百科全书]]》中，参见用q.v.（拉丁文quod vide的缩写）或see来表示，《[[布罗克豪斯百科全书|布罗克豪斯百科全书]]》用→，《[[迈耶百科词典|迈耶百科词典]]》用↑，《[[剑桥百科全书|剑桥百科全书]]》用>>，中文百科全书则一般用[[楷体|楷体]]字表示参见。<ref name="金常政" />{{rp|124-125}}
