'''条顿骑士团'''（{{lang-la|Ordo Domus Sanctae Mariae Teutonicorum}}；{{lang-de|Deutscher Orden}}{{RefTag|1={{cite web
 |url        = http://www.press.ntu.edu.tw/ejournal/Files/%E4%B8%AD%E5%A4%96%E6%96%87%E5%AD%B8/pdf/9501/8.pdf
 |title      = 中古慈善收容制度之宗教神髓
 |author     = 蘇其康
 |date       = 2005-05
 |accessdate = 2010-10-12
}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}}}），又译'''德意志騎士團'''{{NoteTag|name="teutonic"|1 = 条顿之音按英语“Teutonic”，本为拉丁语“德意志”之英语音译。}}，正式名称为'''耶路撒冷的德意志弟兄圣母骑士团'''（{{lang-la|Ordo domus Sanctae Mariae Theutonicorum Ierosolimitanorum}}；{{lang-de|Orden der Brüder vom Deutschen Haus Sankt Mariens in Jerusalem}}），与[[圣殿骑士团|圣殿骑士团]]、[[医院骑士团|医院骑士团]]一起并称为三大[[骑士团|骑士团]]。现时，条顿骑士团的口号是“帮助、守卫、救治（Helfen, Wehren, Heilen）”。

==崛起==
[[第三次十字军东征|第三次十字军东征]]（1189年－1192年）期间，来自[[德意志|德意志]]地区的[[骑士|骑士]]多次获赠土地和城堡。

1195年4月，香槟区的亨利伯爵赠[[苏尔|提尔]]（今[[黎巴嫩|黎巴嫩]]境内）作为据点，1196年3月再赠其在[[雅法|雅法]]（今[[特拉维夫|特拉维夫]]附近）的封邑；另有[[神聖羅馬帝國|神聖羅馬帝國]]皇帝[[亨利六世|亨利六世]]在1197年赠送[[意大利|意大利]]和[[西西里|西西里]]的教堂、修道院和医院，逐渐在[[耶路撒冷|耶路撒冷]]地区形成势力。

1198年3月5日，条顿骑士团成立于[[阿卡_(以色列)|阿卡]]，今[[巴勒斯坦|巴勒斯坦]]境内，其后以阿卡作为总部至1291年。

1199年9月19日，[[教皇|教皇]][[英诺森三世|英诺森三世]]颁布训令，规定条顿骑士披和[[圣殿骑士|圣殿骑士]]一样的披风（白色披风，刺繡的紋章由聖殿騎士團的紅十字改為條頓騎士團的象徵黑十字），並执行和[[医院骑士团|医院骑士团]]一样的团规。

13世纪初，随着[[医院骑士团|医院骑士团]]和[[圣殿骑士团|圣殿骑士团]]对立加剧，医院骑士团拉拢条顿骑士团以对抗控制着[[耶路撒冷|耶路撒冷]]地区主要军事据点的圣殿骑士团，其间条顿骑士团从[[医院骑士团|医院骑士团]]手中获贈[[马加特堡|马加特堡]]。

1210年，[[赫尔曼·冯·舒尔茨|赫尔曼·冯·舒尔茨]]（Hermann von Salza, c.1165-1239）担任[[条顿骑士团|条顿骑士团]]团长，条顿骑士团在其指挥下在耶路撒冷地区获得一定的胜利，并参与[[第五次十字军东征|第五次十字军东征]]，进入[[埃及|埃及]]，但最后在[[曼苏拉战役|曼苏拉战役]]（Battle of al-Mansura, 1221年8月30日）中惨败，赫尔曼·冯·舒尔茨与圣殿骑士团团长一同被俘。在赫尔曼·冯·舒尔茨担任团长期间，条顿骑士团获得[[教廷|教廷]]颁发的[[赎罪证|赎罪证]]（1216年2月18日），以及教皇[[霍諾留斯三世|霍諾留斯三世]]授予的113项特权（1221年1月9日）。

1226年，条顿骑士团与[[神聖羅馬帝國|神聖羅馬帝國]]皇帝[[腓特烈二世|腓特烈二世]]达成协议，获得[[普鲁士|普鲁士]]境内的所有贵族特权。1234年，条顿骑士团赢得[[瑟格纳战役|瑟格纳战役]]的胜利，教廷控制[[普鲁士|普鲁士]]全境，将之租借给条顿骑士团，但直到1285年，[[条顿骑士团|条顿骑士团]]才最终征服[[普鲁士|普鲁士]]，迫使普鲁士人改宗[[天主教|天主教]]。其间，1237年4月合并[[利沃尼亚|利沃尼亚]]的[[寶劍騎士團|寶劍騎士團]]，并于1241年初开始进军[[诺夫哥罗德公国|诺夫哥罗德公国]]。

==衰落==
[[File:Deutscher_Orden_in_Europa_1300.png|thumb]]

1242年4月5日，1万2千名条顿骑士团部队（包括[[丹麦|丹麦]]和[[波罗的海|波罗的海]]沿岸各地的骑士和民兵）与[[亚历山大·涅夫斯基|亚历山大·涅夫斯基]]指挥的1万5千至1万7千人的[[诺夫哥罗德公國|诺夫哥罗德公國]]軍队展开[[楚德湖战役|楚德湖战役]]，条顿骑士团以重骑兵作为前锋和作战主力，其后是步兵，两翼和后方有重裝騎兵，俄军则以[[輕骑兵|輕骑兵]]和装备弓箭、標槍的轻步兵配置在中央，诺夫哥罗德公國的精锐重裝步兵在两翼，亚历山大·涅夫斯基的亲卫队和贵族骑兵队作为预备队埋伏在左翼后侧。最终条顿骑士团的重裝骑兵由于行动不便以及冰层开裂等原因遭到惨败，损失超过1万人，再无能力向俄国腹地进军。楚德湖战役也被称作[[冰湖大战|冰湖大战]]。

1291年5月18日，[[马木留克|马木留克]]骑兵[[阿卡圍城戰_(1291年)|攻陷阿卡]]，条顿骑士团总部迁往[[威尼斯|威尼斯]]。1320年，条顿骑士团团长在[[希腊|希腊]]南部[[摩里亚半岛|摩里亚半岛]]被当地人杀死。1391年，条顿骑士团在[[摩里亚战役|摩里亚战役]]中失败，被逐出摩里亚半岛。{{Fact}}500年，条顿骑士團被[[土耳其人|土耳其人]]彻底逐出[[希腊|希腊]]。{{Fact}}

1346年，条顿骑士团从丹麦人手中夺取[[爱沙尼亚|爱沙尼亚]]，控制了[[波罗的海|波罗的海]]东岸的全部出海口。1410年7月15日，条顿骑士团与[[波兰|波兰]]、[[立陶宛|立陶宛]]、[[俄罗斯|俄罗斯]]联军进行[[格倫瓦德之戰|格林瓦尔德战役]]，条顿骑士团再次败北，包括总团长在内的全部指挥官阵亡，从此一蹶不振。

1466年結束戰爭的《[[第二次托倫和約|第二次托倫和約]]》（Thorn）規定，騎士團除赔款600万{{link-en|格罗申|Groschen}}外，還要將普魯士一分為二，[[西普魯士|西普魯士]]併入[[波兰王国_(1385年-1569年)|波蘭王國]]，[[東普魯士|東普魯士]]仍由騎士團統治，要對波蘭國王稱臣，成为波兰的[[附庸国|附庸国]]。

1512年，来自[[勃兰登堡|勃兰登堡]][[霍亨索倫家族|霍亨索倫家族]]的[[阿尔布雷希特_(普鲁士)|阿尔布雷希特]]被选为条顿骑士团总团长，他是[[勃兰登堡选帝侯|勃兰登堡选帝侯]]的近親。在[[馬丁·路德|馬丁·路德]]的影響下，1525年他宣布改信[[路德宗|路德宗]]，从而切断了与骑士团名义宗主[[罗马教廷|罗马教廷]]的联系，随后宣布解散騎士團，騎士團領地改为[[普鲁士公国|普鲁士公国]]，阿尔布雷希特自任[[普鲁士公爵|普鲁士公爵]]，成为臣服于波兰最高权力之下的世俗君主。

1809年，[[拿破崙|拿破崙]]解散了作为军事组织的條頓騎士團。

== 改制 ==
1929年，教宗[[庇護十一世|庇護十一世]]訓令，條頓騎士團成為[[純宗教修士會|純宗教修士會]]，以协助與公益性质現存至今。
