'''望古車站'''位於[[台灣|台灣]][[新北市|新北市]][[平溪區|平溪區]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[平溪線|平溪線]]的[[鐵路車站|鐵路車站]]。

== 車站構造 ==
*[[岸式月台|岸式月台]]一座。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#招呼站|招呼站]]。
*行駛車輛為DR1000型[[柴油|柴油]]客車，等級為[[臺鐵區間車|區間車]]。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1972年7月30日：因應「慶和煤礦」的設立，以配合慶和煤礦之輸送；以及服務在慶和煤礦上下班的員工而設'''慶和招呼站'''。
*1989年1月23日：因本站所在地行政區屬[[新北市|台北縣]][[平溪區|平溪鄉]]望古村，故改為「望古」站。
*2013年9月30日：啟用多卡通刷卡機。
