'''釧路綜合振興局'''（{{jpn|j=釧路総合振興局|hg=くしろそうごうしんこうきょく|rm=Kushiro sōgō shinkōkyoku|f=y}}），為[[日本|日本]][[北海道|北海道]][[道東|道東]]地方的[[振興局|振興局]]，行政中心設於[[釧路市|釧路市]]。

== 地理 ==
*位於北海道東部，東臨[[太平洋|太平洋]]，北部為山岳地帶，南部為[[釧路濕原|釧路濕原]]
*氣候：夏季多[[霧|霧]]；冬季降雪，晴天較多。
*山：雌阿寒岳、[[硫磺山_(弟子屈町)|硫磺山]]、摩周岳
*河川：[[釧路川|釧路川]]、[[阿寒川|阿寒川]]、庶路川
*湖沼：[[阿寒湖|阿寒湖]]、[[摩周湖|摩周湖]]、[[屈斜路湖|屈斜路湖]]、塘路湖、シラルトロ湖、春採湖、厚岸湖

== 历史 ==
*1897年11月：設置'''釧路支廳'''，下轄原[[釧路國|釧路國]][[白糠郡|白糠郡]]、[[釧路郡|釧路郡]]、[[厚岸郡|厚岸郡]]、[[阿寒郡|阿寒郡]]、[[足寄郡|足寄郡]]。<ref>{{Cite web| url=http://homepage1.nifty.com/ishato/tiri/sityoson/01hokkai/01_hokkai21.htm#kusiro| title=市町村的變遷 - 北海道［道東・道北］| publisher=ISSIE| accessdate=2008年7月20日| language=ja| deadurl=yes| archiveurl=https://web.archive.org/web/20091001194212/http://homepage1.nifty.com/ishato/tiri/sityoson/01hokkai/01_hokkai21.htm#kusiro| archivedate=2009年10月1日}}</ref>
*1922年8月：改稱為釧路國支廳。
*1948年10月20日：[[足寄郡|足寄郡]]足寄村（現在的[[足寄町|足寄町]]）、陸別村（現在的陸別町）改隸屬[[十勝支廳|十勝支廳]]管轄。
*1957年4月1日：改稱為釧路支廳。
*2010年4月1日：北海道廢除支廳，改設為'''釧路綜合振興局'''。

== 產業 ==
以[[釧路市|釧路市]]為中心，有數個北海道規模較大的港口，作為東北海道的國際物流據點，並開設有定期國際航線（釧路～[[釜山|釜山]]）。港口裡的臨港工業園區包括造紙紙漿業（日本造紙、王子製紙）、機械製造業、飼料肥料聯合企業，與[[苫小牧市|苫小牧]]、[[室蘭市|室蘭]]同為北海道主要的工業集散地。

釧路市是[[道東|道東]]地區的中心城市，也設有許多國家的分支機構、金融機關，同時也是國際會議旅遊城市，有許多場地及旅社可舉辦國際型會議，是北海道東部的主要工商業城市。

[[白糠町|白糠町]]被指定為[[次世代能源|次世代能源]]特區，成為開發新能源的據點。

因為土壤含有[[火山灰|火山灰]]及[[泥炭|泥炭]]，使得農作難以生長，因此主要產業為[[奶酪畜牧業|奶酪畜牧業]]，乳製品的品質較日本其他地區高，也成為[[喜見達|喜見達]]（Häagen-Dazs）的原料來源。

沿岸地區[[漁業|漁業]]盛行，釧路市的除[[釧路漁港|釧路漁港]]的漁獲量為日本前幾名，[[厚岸町|厚岸町]]、[[釧路町|釧路町]]以牡蠣有名，[[濱中町|濱中町]]則是以海帶著名。

主要觀光景點包括[[阿寒國立公園|阿寒國立公園]]、[[釧路濕原國立公園|釧路濕原國立公園]]、[[厚岸道立自然公園|厚岸道立自然公園]]，藉由東北海道的主要機場釧路機場，每年約有1000萬名遊客來到此地。目前[[摩周湖|摩周湖]]正致力於登錄世界自然遺產，厚岸道立自然公園以致力於半耕維國定公園。

[[弟子屈町|弟子屈町]]有較多來自本州的移民。
