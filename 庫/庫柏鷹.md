'''庫柏鷹'''（{{lang|la|''Accipiter cooperii''}}），外號'''[[雞鷹|雞鷹]]'''<ref>{{cite web |url=https://www.facebook.com/photo.php?fbid=427479380626376&set=a.188951251145858.45277.188946924479624 |title=動物粉絲：庫柏鷹 |language=zh-hant}}</ref>，是一種原生於[[北美洲|北美洲]]的中型[[鷹|鷹]]，從[[加拿大|加拿大]]南部到[[墨西哥|墨西哥]]北部均去找到牠們的身影。與一般的鳥類一樣，雄鳥的體型較雌鳥小；而在[[密西西比河|密西西比河]]以東的體型普遍較在河以西的為大。

== 分類 ==

庫柏鷹最早由[[法国|法国]][[博物学|博物学]]家和鸟类学家[[夏尔·吕西安·波拿巴|夏尔·吕西安·波拿巴]]於1828年描述。牠是[[鷹屬|鷹屬]]的一個品種。庫柏鷹命名自[[美國|美國]][[紐約|紐約]][[紐約自然史學苑|自然史學苑]]（即後來的[[紐約科學院|紐約科學院]]的前身）的創辦人、[[貝殼學家|貝殼學家]][[威廉·庫柏|威廉·庫柏]]。庫柏鷹其他常見的名稱有：雞鷹、Big Blue Darter、墨西哥鷹、Quail Hawk、Striker及Swift Hawk等<ref>{{cite web |url=http://zipcodezoo.com/Animals/A/Accipiter_cooperii/ |title=Accipiter cooperii (Big Blue Darter) |publisher=Zipcodezoo.com. |accessdate=2011-12-18 |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20120531193502/http://zipcodezoo.com/Animals/A/Accipiter_cooperii/ |archivedate=2012-05-31 }}</ref>。
