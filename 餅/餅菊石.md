'''餅菊石'''（[[學名|學名]]：{{lang|la|'''''Popanoceras'''''}}）是生存於[[二疊紀|二疊紀]]的一[[屬|屬]][[菊石|菊石]]。其外殼呈扁平的餅狀，因而得名。殼的臍部狹小，側面有細密的波浪狀肋條。<ref name="Popanoceras">{{Cite web |url = http://liusir.pixnet.net/blog/post/248635817-popanoceras-%E9%A4%85%E8%8F%8A%E7%9F%B3-%28%E5%B8%9D%E6%B1%B6%29 |title = Popanoceras 餅菊石 (帝汶) @ 菊海神遊 Swimming with Ammonites :: 痞客邦 :: |language = zh-hant |accessdate = 2018-08-02 }}</ref>其[[化石|化石]]被發現於[[俄羅斯|俄羅斯]]、[[哈薩克斯坦|哈薩克]]、[[帝汶島|帝汶島]]、[[中國|中國]]及[[塔吉克斯坦|塔吉克]]等地。<ref name="Fossilworks" />
