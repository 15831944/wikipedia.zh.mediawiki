'''帕劳共和国'''（{{lang-en|Republic of Palau}}；[[帕勞語|帕勞語]]：Beluu er a Belau；{{lang-ja|パラオ共和国}}），通稱'''帕劳'''（{{lang-en|Palau}}，旧称{{lang|en|Belau}}或{{lang|en|Pelew}}），也称'''-{zh-cn:帛琉;zh-hk:帕勞;zh-tw:帕勞;zh-mo:帕勞}-'''，是位于西[[太平洋|太平洋]]的[[島嶼國家|岛嶼国家]]。全国有约340座岛屿，属于[[密克罗尼西亚群岛|密克罗尼西亚群岛]]中[[加罗林群岛|加罗林群岛]]的西链，总面积为466平方公里<ref>{{cite web |url = http://palaugov.pw/rop-statistical-yearbooks/ |title=ROP Statistical Yearbooks {{!}} Palau National Government |accessdate=2017-05-09 |website = palaugov.pw |language = en-US }}</ref>。人口最多的岛屿是[[科羅爾|科罗尔]]。首都[[恩吉鲁穆德|恩吉鲁穆德]]位于临近岛屿[[巴伯尔道布岛|巴伯尔道布岛]]，隶属于[[梅莱凯奥克州|梅莱凯奥克州]]。帕劳的海上邻国包括[[印度尼西亚|印度尼西亚]]、[[菲律宾|菲律宾]]和[[密克罗尼西亚联邦|密克罗尼西亚联邦]]。

约3,000年以前，来自菲律宾的移民最先在这里定居；直到约900年以前，当地人种都属于[[尼格利陀人|尼格利陀人]]。群岛在16世纪初被欧洲人发现，1574年成为[[西屬東印度群島|西属东印度群岛]]的一部分。1898年，西班牙在[[美西战争|美西战争]]中被击败；1899年，根据德国－西班牙条约，群岛被出售予[[德意志帝國|德意志帝国]]，隶属于[[德屬新幾內亞|德属新几内亚]]。[[第一次世界大战|第一次世界大战]]期间，帕劳被[[大日本帝國海軍|日本帝国海军]]占领，随后被[[國際聯盟|国际联盟]]归入日治[[南洋群岛|南洋群岛]]。[[第二次世界大战|第二次世界大战]]期间，作为[[馬里亞納群島及帛琉戰事|马里亚纳群岛及帕劳战事]]的一部分，美军与日军在此进行过多次小规模战斗，包括主要的[[佩莱利乌战役|佩莱利乌战役]]。1947年，帕劳同其它几个[[太平洋三大島群|太平洋群岛]]一起成为美属[[太平洋群島託管地|太平洋群岛托管地]]的一部分。1979年，帕劳公投反对加入密克罗尼西亚联邦；根据与美国之间的[[自由联合协定|自由联合协定]]，帕劳在1994年完全独立。

1783年，[[葡萄牙人|葡萄牙人]]發現帛琉，但不久後被[[西班牙人|西班牙人]]統治。1899年由[[西班牙|西班牙]]售予[[德国|德国]]。[[第一次世界大戰|第一次世界大戰]]後，被[[日本|日本]]以[[國際聯盟託管地#第三等託管地|C類託管地]]進行託管。[[第二次世界大戰|第二次世界大戰]]中，被盟軍攻佔。戰後於1947年由[[聯合國|聯合國]]授權[[美國|美國]]以[[太平洋群島託管地|太平洋群島託管地]]進行託管，1981年獲准成立自治政府，稱為「帕劳共和國」，並制定禁止貯藏及攜入[[核武|核武]]之「非核憲法」。

1986年美國以15年內分期提供4億5千萬美元經濟援助及准許其獨立為條件與帛琉簽訂為期50年之「[[自由聯合|自由聯合]]協定」（Compact of Free Association），依據該協定帛琉享有內政自治權並得與他國政府、區域暨國際組織簽訂條約及協定，涉外事務需與美方進行協商，國防亦由[[美軍|美軍]]協防，美方還有權派遣[[核子動力潛艇|核子軍艦]]及[[軍機|軍機]]至帕劳領域，且無義務承認或否認在帕劳部署核武；此一條件有違帛琉憲法中「非核條款」，故要求帛琉暫時凍結此一憲法條款。此協定先後7次遭到帕劳公民投票否決，直至1993年11月第8次公民投票始獲通過，帕劳遂於1994年10月1日正式獨立。

== 史前遺址 ==
最早期的帛琉人可能來自[[波里尼西亞|波里尼西亞]]和[[亞洲|亞洲]]。從該地各家族的起源傳說可以看出，帛琉人可能混雜了[[美拉尼西亞|美拉尼西亞]]、[[密克羅尼西亞群島|密克羅尼西亞]]和波里尼西亞的人種。因此帛琉人在嚴謹的定義上，並非歸屬於密克羅尼西亞人（[[南島民族|南島民族]]）。

這個島群早期被認為是「黑色群島」，在[http://www.plosone.org/article/info%3Adoi%2F10.1371%2Fjournal.pone.0001780 澳大利亚在線圖書館]中可以找到早期的地圖和村落的描繪，也可以找到早期[[科羅州|科羅州]]大酋長Ibedul有著[[紋身|紋身]]和耳洞裝飾的照片。

近年的研究中，從[[碳-14|<sup>14</sup>C]]的定年結果和出土的考古證據，把這個島群的歷史帶到了新的面向。在島上發現的墓葬遺址顯示，帛琉應該是整個[[大洋洲|大洋洲]]最早具有喪葬儀式的地方。但是對於帛琉人何時定居在此有兩種說法，一個說法是西元前2500年，另一個則是西元前1000年。

在爪哇，這個遠古時期跟帛琉有貿易往來關係的地區，曾經發現了[[佛羅勒斯人|佛羅勒斯人]]。2007年，有考古學者在帛琉的洛克群島發現了疑似佛羅勒斯人的矮小體型的人骨遺留<ref>http://www.plosone.org/article/info%3Adoi%2F10.1371%2Fjournal.pone.0001780</ref>。這個發現可能會推翻以往的假設，但是目前這個研究發現仍有爭議。

幾千年來，帛琉有著發達的[[母系社會|母系社會]]，一般認為是從爪哇遷移來的族群所帶來的制度。傳統上，土地、貨幣、跟頭銜都由母系傳承。氏族所有地，由女性的族長掌管並傳給第一個女兒。但是當代也有實行[[父系社會|父系傳承]]制度，這是在日本殖民帛琉時所傳入該社會。在二次大戰期間，[[日本政府|日本政府]]試圖把土地重新分配，轉為個人所有權，並且也意圖革除舊有的制度。現在這些法律的衝突仍在許多家族中上演。

== 政治 ==

===行政===
帛琉總統為國家及內閣組織的最高元首，每四年由帛琉人民選舉產生。

現任總統為[[湯米·雷蒙傑索|-{zh-cn:小汤米·雷门格绍;zh-tw:湯米·雷蒙傑索}-]]，於2017年1月起擔任第4任期帛琉總統，任期4年。

內閣（2017-2021）設8部，分別為國務部、財政部、基建工商部、社區暨文化部、教育部、衛生部、司法部及自然資源環境暨觀光部，部長由總統任命並須經參議院同意。

===立法===
[[帕劳国会|帕劳国会]]实行[[兩院制|兩院制]]，稱為''Olbiil Era Kelulau''，分參（Senate）、眾（House of Delegates）兩院。

參議員共13名，按人口比例由全國選區（Senatorial District）選出，眾議員共16名，由16個行政區各選出1名，兩院議員任期均為4年。

現為第十屆（2017-2021），於2016年11月1日改選，於2017年1月19日就職。

== 外交 ==
{{Main|帛琉外交}}
{{Details|帛琉共和國駐外機構列表}}

帛琉共和国现已與79個國家及政治體建立[[外交关系|外交关系]]，详见[[帕劳建交列表|帕劳建交列表]]。並在[[美國|美國]]華盛頓與紐約、[[中華民國|中華民國]]、[[日本|日本]]、[[菲律賓|菲律賓]]等地設立[[外交代表機構|外交代表機構]]；[[中華民國|中華民國]]、[[日本|日本]]及[[美國|美國]]也在帛琉設館，並派駐大使。

===與日本關係===
帛琉共和国與日本關係密切，第一次世界大戰期間，帛琉曾被[[日本帝國海軍|日本帝國海軍]]占領，隨後被國際聯盟歸入日治[[南洋群島|南洋群島]]。[[日本戰敗|日本戰敗]]，帛琉獨立後，曾選出日裔的[[中村國雄|中村國雄]]任該國總統。

===與中華民國關係===
{{Main|中華民國－帛琉關係}}

帛琉共和國與中華民國政府的建交，引發中華人民共和國施壓，發出旅遊禁令。2017年12月30日帛琉總統表示，目前並不打算改變和台灣的外交關係，在外交上將會有自己的決定。

===與中華人民共和國關係===
{{Main|中華人民共和國與帛琉關係}}
兩國現時無邦交。中國公民前往帛琉可以通過[[香港|香港]]、[[澳門|澳門]]、[[首爾|首爾]]、[[台北|台北]]等中轉站去帛琉。目前中國政府已經發布旅遊禁令禁止所有中國旅行團前往帛琉旅遊，但[[自由行|自由行]]旅客則不受影響<ref>http://www.mafengwo.cn/gonglve/ziyouxing/172114.html</ref>。

== 地理 ==
[[File:Orthographic_projection_centred_over_Palau.png|thumb]]
[[File:Palau-map.png|thumb]]
{{Main|帛琉地理}}

帛琉由一個大[[堡礁|堡礁]]和無數小島和較少堡礁構成。主島本身其實也由多個小堡礁構成，較大的有[[安加爾|安加爾]]、Babeldaob（Babelthuap）、Koror、Peleliu。亦因為當地有大量珊瑚，故其有大量是由海洋生物的屍體沉降累積而成的石灰岩，甚至成為其附近[[雅浦島|雅浦島]]，[[雅浦島石幣|雅浦島石幣]]的制作原料。全國有三分之二人口在舊都[[科羅爾|科羅爾]]居住。主島群以北是Ngeruangel和Kayangel。主島以西是Rock Islands，無人居住。西南部距離主島群約600公里的西南群島，也是帛琉的一部份。

=== 氣候 ===
帛琉的氣候屬[[熱帶型雨林氣候|熱帶雨林氣候]]，全年平均溫度約28.9℃。全年皆為雨季，平均全年降雨量為380厘米，平均濕度為82%。雖然降雨主要集中在7月到10月，但這段期間也經常陽光普照。帛琉偏離颱風主要路徑範圍，因此鮮少受到[[熱帶氣旋|熱帶氣旋]]影響，僅有2013年[[颱風海燕_(2013年)|颱風海燕]]以極高強度直襲帛琉，所幸未在當地造成嚴重傷亡。

== 生态 ==
{{Main|帛琉生态}}

在帛琉，热带雨林覆盖大多数的岛屿，有黑檀木、孟加拉榕树、面包树、椰子树、露兜树等，这些热带雨林形成绿意盎然的树海。同时帛琉还有野生红树林与热带大草原生态可以探访。

帛琉海底生物形形色色，场面壮观包括超过1,500种的多样鱼类、700种的珊瑚和海葵。其他值得一看的景观尚包括巨型蛤蜊、海龟、蝠魟鱼、灰礁鲨、海蛇、海牛等。靠近陆地之处，有入海口附近的鳄鱼、大蜥蝪。此外，尚可看到数十种鸟类、果蝠在洛克群岛的聚居地、在Angaur岛的猿猴、一些无毒的蛇类、小型蜥蝪及其他昆虫，而且在帛琉的陆地上，没有任何有毒的陆生动物。

帛琉政府指定了洛克群岛中70个无人居住的岛屿作为海洋生态保护区，禁止民众进入，以充分保护海龟与海鸟的生态环境。

== 經濟 ==
{{Main|帛琉經濟}}

[[旅遊業|旅遊業]]是經濟的支柱，除此之外還有農業和漁業。在2000-2001財政年度，約有共50,000外國人來帛琉經商或觀光。政府是本國最主要的雇主，但國家經濟很大程度上依賴美國的經濟援助。儘管如此，人均國民收入是[[菲律宾|菲律賓]]和大多數[[密克罗尼西亚群岛|密克羅尼西亞]]國家的兩倍。帛琉的旅遊市場正在傾向於吸引更多來自[[東亞|東亞]]的遊客，並希望外國投資可以幫助他們興建更完備的基礎設施。

== 人口 ==
{{Main|帛琉人口}}

當地居民約20,000人，其中大部分為帛琉土著，屬[[密克罗尼西亚|密克羅尼西亞人]]。少數族裔有來自菲律賓為主的亞洲人和少量歐洲人。三分之二的居民信奉[[基督教|基督教]]，當中[[天主教|天主教]]徒佔大多數，其餘的信奉其他[[新教|新教]]派系。除了基督教外，當地一種叫''Modekngei''的地方宗教也十分流行。

當地官方語言為[[英語|英語]]和[[帛琉語|帛琉語]]。其中三個州有地方語言作為自己的官方語言。而在安加爾島上，[[日語|日語]]曾经是官方語言。目前安加爾全岛人口80人，所有人讲英语。

== 交通 ==
[[File:Japan_Palau_Friendship_Bridge.jpg|thumb]]
{{See also|羅曼·莫圖國際機場|貝里琉空軍基地|安加爾起降場}}

帛琉全國共有3個機場，分別位於主島、貝里琉和安加爾島上。其中[[帛琉國際機場|帛琉國際機場]]為唯一的國際機場，位於主島南部，埃拉伊州境内。
[[帛琉太平洋航空|帛琉太平洋航空]]、[[大韓航空|大韓航空]]、[[達美航空|達美航空]]和[[聯合航空|聯合航空]]等提供連接香港、東京、首爾、台北、高雄及關島等航班。
