《'''這！就是原創'''》（{{lang-en|''Chuang''}}）是一檔由[[優酷|優酷]]、[[燦星製作|燦星製作]]和[[騰訊音樂|騰訊音樂]]聯合出品的華語原創音樂選秀節目，原本是同為燦星出品的原創音樂節目《[[中國好歌曲|中國好歌曲]]》的第四季<ref name=yzwb>{{cite web | url = http://epaper.yzwb.net/html_t/2019-01/31/content_503642.htm?div=-1 | title = 停播近三年的《中国好歌曲》有了接班人 《这！就是原创》高调官宣三月归来 | work = 揚子晚報 | date = 2019-01-31 | accessdate = 2019-02-26 | author = 张艳}}</ref>。節目於2019年3月9日起每週六在優酷首播。

== 前身與籌備 ==

《[[中國好歌曲|中國好歌曲]]》是一檔由燦星製作出品的華語原創音樂選秀節目，[[中國好歌曲_(第一季)|首季]]於2014年在[[中國中央電視台綜藝頻道|中國中央電視台綜藝頻道]]播出，之後兩年又分別播出了[[中國好歌曲_(第二季)|第二季]]和[[中國好歌曲_(第三季)|第三季]]；2016年底，在節目第四季原定檔期前兩個月，擔任前三季導師的[[劉歡|劉歡]]在社群媒體上發表聲明，稱央視綜藝頻道退出了該節目的製作發行，第四季節目擱淺<ref>{{cite web | url = http://www.sina.com.cn/mid/hot/2016-11-18/doc-ifxxwrwh4609034.shtml | title = 刘欢承认《好歌曲》停播：容不下原创音乐 | work = 新浪娛樂 | date = 2016-11-18 | accessdate = 2017-09-11}}</ref>。製作方燦星方面隨後否認了媒體在報道中使用的「停播」說法，表示節目製作團隊正在對《中國好歌曲》的模式和賽制進行改良升級，因此第四季節目需要推遲播出<ref>{{cite web | url = http://bjcb.morningpost.com.cn/html/2016-11/19/content_422033.htm | title=《中国好歌曲》停播？灿星：消息不确切 | work = 北京晨報 | date = 2016-11-19 | accessdate = 2017-09-11}}</ref><ref>{{cite web | url = https://weibo.com/3577435414/Ei6POkAvb | title = 中国好歌曲的微博 | work = 新浪微博 | date = 2016-11-18 | accessdate = 2017-09-11}}</ref>。

2017年8月，任職於燦星的《[[中國好聲音|中國好聲音]]》、《中國好歌曲》導演韓劼發布[[新浪微博|微博]]，宣布第四季《中國好歌曲》正式啟動，開始徵集報名<ref>{{cite web | url = https://weibo.com/1838721191/Fhcym5XtZ | title=韩小草_Jessie的微博 | work = 新浪微博 | date = 2017-08-15 | accessdate = 2017-09-11}}</ref><ref>{{cite web | url = http://money.163.com/17/0824/08/CSJDNV40002580S6.html | title = 《中国好歌曲》第四季回归，“原创”标签能否让灿星挣脱疲劳期？ | work = 網易財經 | date = 2017-08-24 | accessdate = 2017-09-11}}</ref>。同年12月，網傳消息稱，有燦星導演在私人社群媒體上表示本季節目再次停擺<ref>{{cite web | url = http://cnews.chinadaily.com.cn/2017-12/13/content_35294426.htm | title = 失望！因综合因素《中国好歌曲》又将停录 | work = 北京青網 | date = 2017-12-13 | accessdate = 2017-03-22}}</ref>。2018年3月，韓劼在微博上確認，節目計劃延期至2018年底<ref>{{cite web | url = https://weibo.com/1838721191/G5vHg7Ksg | title = 韩小草_Jessie的微博 | work = 新浪微博 | date = 2018-03-02 | accessdate = 2018-03-22}}</ref>。

2018年9月，《中國好歌曲》第四季正式更名為《這！就是原創》，由原班團隊製作，並改為在網路平台[[優酷|優酷]]播出<ref name=yzwb/><ref>{{cite web | url = http://www.sohu.com/a/251418464_100063290 | title = 好歌曲推出新综艺《原创是猛兽》，网友：刘欢回归再加上周杰伦？ | work = 搜狐娛樂 | date = 2018-09-02 | accessdate = 2018-09-05}}</ref>，成為繼《[[這！就是街舞|這！就是街舞]]》、《[[這！就是鐵甲|這！就是鐵甲]]》、《[[這！就是歌唱·對唱季|這！就是歌唱·對唱季]]》、《[[這！就是灌籃|這！就是灌籃]]》後的第五檔「這就是」系列節目<ref>{{cite web | url = http://www.chinanews.com/yl/2018/10-16/8651196.shtml | title = 福克斯传媒买下优酷《这就是灌篮》模式版权 | work = 中國新聞網 | date = 2018-10-16 | accessdate = 2019-01-20}}</ref>。節目由優酷、燦星和[[騰訊音樂|騰訊音樂]]聯合出品，用「原創是猛獸」作為口號，以「全新原創音樂節目」的身份亮相<ref name=release>{{cite web | url = https://v.youku.com/v_show/id_XMzgzMDgwNTc0MA==.html?spm=a2h1n.8261147.around_3.5~5~5~5~A&s=72211ecece9948529d80 | title = 吴群达：原创是猛兽 林宥嘉作为原创捕手 邀请每一个原创的灵魂 | work = 優酷 | date = 2018-09-20 | accessdate = 2018-10-08}}</ref>。同年11月30日，《這！就是原創》正式發布招募令，開通優酷和騰訊音樂兩條報名渠道，邀請[[創作歌手|唱作人]]遞交原創音樂作品報名參賽<ref>{{cite web | url = http://ent.people.com.cn/n1/2018/1203/c1012-30439380.html | title = 《这就是原创》发布招募令 原创音乐人集结来袭 | work = 人民網 | date = 2018-12-03 | accessdate = 2019-01-19}}</ref><ref>{{cite web | url = https://www.weibo.com/6709675679/H559IevHv | title = 這就是原創的微博 | work = 新浪微博 | date = 2018-11-30 | accessdate = 2019-01-19}}</ref>。2019年1月17日，節目公佈了在海選中入圍的109名唱作人，並分別在[[QQ音樂|QQ音樂]]和優酷上發布了「猛獸原創榜」和「原創盤獸榜」，由用戶在九天的「打榜期」裡通過歌曲試聽、视频播放、社群網路分享等方式支持自己喜愛的唱作人；打榜期結束時，「猛獸原創榜」上的前十名唱作人可以獲得節目「直通搶位賽」資格，「原創盤獸榜」的前三名則可在節目的第一輪競爭中獲得「加唱權」，演唱第二首原創歌曲<ref name=list>{{cite web | url = https://www.weibo.com/6709675679/HcpDmgmbF | title = 這就是原創的微博 | work = 新浪微博 | date = 2019-01-17 | accessdate = 2019-01-19}}</ref><ref>{{cite web | url = https://www.weibo.com/6709675679/HcFEnulRE | title = 這就是原創的微博 | work = 新浪微博 | date = 2019-01-19 | accessdate = 2019-01-19}}</ref>。

2019年1月，《這！就是原創》正式宣布[[陳粒|陳粒]]、[[蕭敬騰|蕭敬騰]]、[[王嘉爾|王嘉爾]]加盟節目，擔任「原創捕手」<ref name=yzwb/>。此前，在2018年9月的優酷節目發佈會上，節目組曾宣布[[林宥嘉|林宥嘉]]是三位原創捕手之一，但他最終並沒有出現在這一陣容中<ref name=release/>。

2019年2月28日，《這！就是原創》發佈了由中國樂團[[南征北戰_(組合)|南征北戰]]創作和演唱的節目主題曲《創兒》<ref>{{cite web | url = https://v.youku.com/v_show/id_XNDA3MjYzNzkwMA==.html | title = 南征北战献唱，《创儿》唱出原创者心声 | work = 優酷 | date = 2019-02-28 | accessdate = 2019-02-28}}</ref>。
