《'''終極追殺令'''》（原題：Léon，美國上映片名：The Professional）是一部1994年的電影，由[[法國|法國]]導演[[盧·貝松|盧·貝松]]編劇及執導，也是他首部往好萊塢發展拍攝的電影。由[[尚·雷諾|尚·雷諾]]、[[蓋瑞·歐德曼|蓋瑞·歐德曼]]及[[娜塔莉·波曼|娜塔莉·波曼]]主演。

本片的背景和主要拍攝地是[[紐約市|紐約市]]。

== 故事簡介 ==
里昂·蒙泰拿（[[尚·雷諾|尚·雷諾]]飾）是[[義大利裔美国人|義大利裔美国人]]的頂尖職業殺手，一直孤獨的住在紐約[[小意大利|小意大利]]，为一名叫托尼的意大利餐厅老板（也是意大利裔美国人）服务，只有一盆[[粗肋草屬|萬年青]]是他最好的朋友。他形容：「它永遠都很開心，從來都不問問題，而且也跟我一樣—沒有根。」他雖然身手不凡，但內心其實非常缺乏安全感。他甚至從來都不敢在床上睡覺，而是坐在椅子上睡覺，並把手槍放在手邊以便隨時應付任何入侵的敵人。

一天，12歲的鄰居女孩玛蒂达·蘭度（[[娜塔莉·波特曼|娜塔莉·波特曼]]飾）罕有地敲響了里昂的家門。原来，玛蒂达當毒販的父親因私吞了一包[[美国缉毒局|緝毒局]]黑警的毒品，全家慘遭黑警警官諾曼·史丹菲爾（[[蓋瑞·歐德曼|蓋瑞·歐德曼]]飾）冷血滅門，只有瑪蒂达因為出門購物逃過一劫，當下只好哭著向里昂求救收留。在得到里昂的收留後，開始幫里昂管家並教他識字，里昂則教她殺手的技能，兩人漸漸生起既似父女又似戀人的複雜情愫。

之後玛蒂达跟踪史丹菲爾，贸然去报仇反而被抓。里昂及时赶到，将她救回。他们再次搬家，但由于里昂的雇主托尼向史丹菲尔供出了里昂的下落，最终导致玛蒂达被警方跟蹤。這次史丹菲爾調動了超過200名配備[[裝甲車|裝甲車]]、[[突擊步槍|突擊步槍]]及[[榴弹发射器|榴弹发射器]]的重裝備[[特種武器和戰術部隊|特警]]包圍公寓，決心殺死里昂。里昂擊倒大量特警队员，再次救出玛蒂达。由於管道太小，只有體型較小的玛蒂达能通過，里昂於是讓她帶著那株盆栽通过通风管道逃生，并將畢生積蓄交托予她。其後，里昂試圖戴著面罩装成受傷的特警队员混出重圍，但最後仍被埋伏在背後的史丹菲爾識破，中槍倒地。最後，垂死的里昂引爆裝在身上的所有的手榴彈，和史丹菲爾一起同歸於盡。
里昂死後，托尼拒绝让玛蒂达继承里昂的钱。玛蒂达乘坐缆车来到了位于[[罗斯福岛|罗斯福岛]]上的孤儿院，將里昂生前唯一的朋友—那盆萬年青移出了窄小的花盆，並栽種在大地，里昂終於得以落葉歸根。

== 加長版 ==
本片上映當時的版本片長為110分鐘，但另有一133分鐘的版本。該加長版本正式名稱為「國際版」（International Version），不過只在日本上映，之後並以影像軟體方式發售。但導演盧貝松表示，他自己比較喜歡稱之為「加長版」（Long Version）。

該23分鐘片段主要在中間增加了里昂和瑪蒂爾達相處，尤其是偏向男女感情描寫的片段（這些在正式版本裡著墨非常之少）。包括她向他明白表示想成為他的戀人而里昂拒絕的對話、兩人同床共眠的鏡頭、里昂提到因自己失去舊情人而成為殺手的談話、與一些她跟著他觀摩殺人技巧的鏡頭。

=== 关于两个版本情节的差异 ===
以下资料出自台湾马路秀影评（2001年1月19日）「《這個殺手不太冷》被剪掉的那二十四分钟」中对相关情节的记录。透过该篇影评，可以了解到，加长版的思想性和可观赏性都强过在美国上映的110分钟版本。目前加长版陸續已發行[[LD|LD]]、1区、2区[[DVD|DVD]]及[[BD|BD]]版本。

* 曾經有谣传指出片中有出現床戲，但實際上所谓的“床戏”不过是瑪蒂達邀请里昂躺到床上休息（因為里昂从不在床上睡觉）。后来瑪蒂達枕着里昂的胳膊一起睡了一夜。这一段场景在里昂的心情转变过程中也很重要。
* 在旅馆登记时，瑪蒂達告诉里昂自己十八岁了，里昂半信半疑。
* 瑪蒂達查到凶手的姓名和地址，要雇里昂杀他。里昂表示他厌倦了瑪蒂達的游戏，不想再跟她有牵扯。瑪蒂達於是在左轮手槍中装进三發子弹，和里昂玩俄罗斯轮盘賭。
* 里昂带著瑪蒂達去见東尼，宣布她是他的搭档。
* 里昂和瑪蒂達初次一起行动。瑪蒂達看到桌上的毒品，想起自己最疼愛的小弟正是因毒品而慘遭橫禍，將之一把火烧光。
* 他们在一家餐厅庆祝第一次的合作行动成功。瑪蒂達想吻里昂，害羞的里昂坚持不要。瑪蒂達喝掉一整杯酒，歇斯底里地一直笑，搞得里昂很尴尬。
* 某次的袭击行动失败，里昂把手榴弹丢进房间去炸人（導演盧貝松並親自客串其中一名受害人）。他指著手上的插鞘，说这是「拉環遊戲」（the ring trick）。片末，他用同一招解決了史丹菲爾。
* 某次里昂要去杀人，不让瑪蒂達跟。这里有一段重要对话。对於里昂和瑪蒂達的实际年龄和心理年龄逆差的理解相當重要
(里昂:『妳也需要時間長大。』
瑪蒂達:『我已經長夠了，里昂、我正在變老。』
里昂:『我正好相反，我夠老了，但我需要時間長大。』)。

== 片尾曲 ==
片尾曲為[[史汀|史汀]]的“Shape of My Heart”，該曲背景旋律之後被[[Monica|Monica]]、[[宇多田光|宇多田光]]等人引用。

== 迴響 ==
本片在觀眾評價非常之高,但專業評論沒有那麼熱情,爛番茄新鮮指數72%,平均分6.7。票房方面，本片以1600萬美金預算得到4500萬美金票房。本片是[[娜塔莉波曼|娜塔莉波曼]]出道作，也是[[尚雷諾|尚雷諾]]第一部取得巨大成功的英語片，打開了兩人在英語圈的知名度。於2014年9月1日的資料，其在[[imdb|imdb]]的歷史250佳片（Top 250）中排名第27。

== 影響 ==
* [[周星馳|周星馳]]的電影《[[回魂夜|回魂夜]]》中，[[周星馳|周星馳]]飾演的捉鬼專家和[[莫文蔚|莫文蔚]]飾演的阿群，是模仿了本片中里昂和瑪蒂達的造型。
* 在由[[孙红雷|孙红雷]]主演的中國电视剧《[[征服_(电视剧)|征服]]》中，三个亡命杀手在躲藏的地点每天都看的同一部电影即為本片。
*在韓國女團[[TWICE|TWICE]]的一次回歸《[[What_is_Love?|What is Love?]]》MV裡，團員[[彩瑛|彩瑛]]的橋段中與團員[[多賢|多賢]]分別飾演瑪蒂達與里昂，一同致敬本片。
