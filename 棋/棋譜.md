'''棋譜'''是一盤棋局對弈發展的過程紀錄，有人將古今中外的對局，或者是某人排擬的棋局，刊輯成書供人閱讀。不同的棋戲會有不同的棋譜，例如[[圍棋譜|圍棋譜]]、[[中國象棋棋譜|中國象棋棋譜]]、[[國際象棋棋譜|國際象棋棋譜]]和[[五子棋譜|五子棋譜]]等等。

在現今，更有將這些棋譜轉存為電子檔案格式（如：[[:en:Smart_Game_Format|'''SGF''']]<ref>'''SGF'''，Smart Go Format / Smart Game Format</ref>；[[:en:Portable_Game_Notation|'''PGN''']]<ref>'''PGN'''，Portable Game Notation</ref>），以供[[軟件|軟件]]閱讀和方便打譜。
