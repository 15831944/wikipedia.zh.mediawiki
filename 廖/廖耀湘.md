'''廖耀湘'''（{{bd|1906年|5月16日|1968年|12月2日|catIdx=Liao廖}}），别号建楚，[[中国|中国]][[湖南|湖南]]邵阳县（今[[新邵县|新邵县]]）北乡酿溪镇人，[[中华民国|中华民国]]高级将领。

== 早年 ==

廖耀湘1918年考入邵阳县立高等小学，1920年毕业。次年，考入[[长沙|长沙]][[岳云中学|岳云中学]]。1925年，廖耀湘决心[[投笔从戎|投笔从戎]]，但因缺少盘缠，无法南下投考[[黄埔军校|黄埔军校]]，转而考入湖南陆军第3师第3旅教导队，毕业后担任班长。1926年，廖耀湘离开湘军，考入黄埔军校第六期騎兵科，1928年毕业后，又考入留学军官预备班。1930年，廖耀湘赴[[法国|法国]]公费留学，1936年以第一名毕业于[[法国圣西尔军校|法国圣西尔军校]][[机械化骑兵|机械化骑兵]]专业。同年回国，任[[国民革命军|国民革命军]]中央军校教导总队骑兵队少校连长。1937年，升任教导总队第2旅中校参谋主任。


== 抗日战争期间 ==

1937年，[[抗日战争_(中国)|抗日战争]]爆发，廖耀湘参加了[[南京保卫战|南京保卫战]]，战败之际化装突围。国民政府撤退至[[武汉|武汉]]后，廖耀湘升任军官训练总队上校大队长，并上书[[蒋介石|蒋介石]]，提出建军改进方案，得到蒋介石赏识。

1938年3月，廖耀湘因属全军唯一一个具备留学外国机械化专业学历的军官，被提升为装甲兵团扩充成为的摩托化部队第200师少将参谋长。当年9月，200师在湖南[[湘乡|湘乡]]扩充为[[国民革命军第五军|国民革命军新十一军]]，旋改为第五军（军长杜聿明），廖任该军所辖新22师副师长兼军干训班主任，因师长[[邱清泉|邱清泉]]未到职，廖耀湘代理师长之职。

1939年9月，第5军自广西全州驻地出发参加[[桂南会战|桂南会战]]。廖耀湘配合邱清泉指挥新22师，参加了对[[昆仑关|昆仑关]]的争夺战。之后，随第5军进入云南滇西驻防，廖耀湘升任新22师师长。

1942年2月，应[[盟军|盟军]]要求，中国政府组成[[中国远征军|远征军]]，进入[[缅甸|缅甸]]作战，新22师作为第5军的一部分一参战。3月，新22师驰援固守[[同古|同古]]的第200师，接应其从日军的包围中突围。4月底，因日军攻占[[腊戍|腊戍]]，切断滇缅公路，中国远征军决定撤回国内。廖耀湘随远征军第1路军副司令长官兼第5军军长杜聿明率领所部在缅甸印道集中，准备经密支那归国时，因卡萨失守，被迫向西北退至曼西。之后，[[杜聿明|杜聿明]]决心放弃重装备，取道俗称[[野人山|野人山]]的敏金山脉地区回国。途中因遭遇雨季，山洪暴发，新22师主力因迷失方向，最终撤往[[印度|印度]][[列多|列多]]。新22师入缅的9000人中，战斗阵亡2000人，撤退中损失4000人，仅约3000人抵达印度<ref name="参考资料">杜聿明《中国远征军入缅对日作战述略》，载《远征印缅抗战——原国民党将领抗日亲历记》，P34</ref>。

在雷多，补充、重建的新22师、新38师在盟军印缅战区副司令史迪威将军的安排下，受到系统的美式训练和整编，并全部换装美械和美式装备，一跃成为中国军队中装备最好的部队之一，战斗力也得到了质的飞跃。之后，廖耀湘率新22师在缅北战场屡建功勋，先后歼灭日军12000余人，其中包括日军精锐之一的[[第18師團|第18師團]]。1944年，新22师扩编为[[国民革命军新六军|新六军]]，成为[[国军五大主力|国军五大主力]]之一。

== 国共内战期间 ==

[[第二次国共内战|第二次国共内战]]爆发后，新六军被调往[[中国东北地区|东北]]战场。廖率领新六军于1946年1月抵达东北，到5月，他已经攻下[[长春|长春]]等地，但是之后被[[林彪|林彪]]等人的[[游击战|游击战]]法拖住手脚，未能北渡[[松花江|松花江]]。1947年8月，新六军与[[国民革命军新三军|新三军]]组成第九兵团，廖出任兵团中将司令，到了1948年，东北战局发生逆转，第九兵团逐步被压缩回[[沈阳|沈阳]]一带。

1948年9月，[[辽沈战役|辽沈战役]]开始，[[中国人民解放军|中国人民解放军]]包围[[锦州|锦州]]，希望切断东北战场国军部队的退路。廖奉命前去解锦州之围未果，被解放军反包围于[[黑山|黑山]]、[[大虎山|大虎山]]一带。但是期间廖耀湘一直在支援锦州，撤回沈阳，营口突围中犹豫，失去了最宝贵的开战初的五天。[[锦州战役|锦州战役]]后，廖耀湘兵团以“[[滚轮战术|滚轮战术]]”撤退，廖耀湘由于兵团部及多个军部遭到打击，在一片混乱中10月26日，廖耀湘部被全歼，本人被俘；僅[[劉玉章|劉玉章]]部自[[營口|營口]]搭艦撤退至[[葫蘆島|葫蘆島]]。

== 结局 ==
廖被俘之后，被长期关押于[[北京|北京]][[功德林战犯管理所|功德林战犯管理所]]，1961年12月25日出狱。之后，他担任[[全国政协|全国政协]]文史资料委员会专员。

“[[文化大革命|文化大革命]]”中遭到[[红卫兵|红卫兵]]羞辱，1968年12月2日廖在批鬥大會上不堪受辱，情緒激動下心臟病發當場猝死。
