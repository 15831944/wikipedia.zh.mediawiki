'''廖國棟'''（[[阿美語|阿美語]]：{{lang|ami|'''Sufin Siluko'''}}，{{bd|1955年|1月8日|catIdx=L廖}}），[[臺灣|臺灣]][[阿美族|阿美族]][[平地原住民選舉區|平地原住民選舉區]]立法委員。

== 生平 ==
=== 個人背景 ===
廖國棟([[阿美族|阿美族]]名稱:Sufin Siluko)，台灣台東縣東河鄉都蘭村人([[都蘭部落|都蘭部落]])，1955年1月8日出生於農村家庭，共有12位兄弟姊妹，家中排行第九位，自幼失去雙親，七歲時母親因難產早逝，九歲時父親因病過世。[[阿美族|阿美族]]是屬於母系社會，因而由姐姐們養育成長。幼年時期因家境貧寒，經常靠[[基督教長老教會|基督教長老教會]]救濟度日，因而與教會結下不解之緣，國小時成績優異，畢業時因為貧窮，亦無法報考初中；那一年，是考初中的最後一屆。

國小畢業後原計畫與哥哥們一樣，當遠洋船員，有幸經由花蓮北埔｢[[基督教芥菜種會|基督教芥菜種會]]｣習藝所名為｢墊腳石計畫｣全額資助補習與打工一年。1968年，台灣開始實施[[九年國民義務教育|九年國民義務教育]]開辦，順利就讀[[都蘭國中|都蘭國中]]。國中時期成績優秀，靠領獎學金填補家用，高中除了考上台東中學外，同時以考上淡江中學(現已改名為：[[私立淡江高級中學|私立淡江高級中學]])以公費獎學金資格就讀，高中畢業後考上台北師範大學，隔年甄試考上「[[台灣省山地暨離島地區醫護人員養成計畫|台灣省山地暨離島地區醫護人員養成計畫]]公費生」，[[高雄醫學大學|高雄醫學大學]]醫學系畢業後，歷任台東縣延平鄉、海端鄉、蘭嶼鄉及長濱鄉、成功鎮衛生所的主任兼醫師、[[台東衛生局|台東衛生局]]醫政課長、[[台東基督教醫院|台東基督教醫院]]主治醫師、東甯診所院長。<ref>[http://www.hwe.org.tw/award_winners_1_12.asp 廖國棟] 1991年，財團法人厚生基金會/立法院厚生會官網</ref><ref>
[http://enews2.kmu.edu.tw/index.php/Enews286_%E3%80%90%E6%9C%88%E5%8D%B0%E8%90%AC%E5%B7%9D%EF%BC%8D%E9%AB%98%E9%86%AB%E7%B6%93%E5%85%B8%E4%BA%BA%E7%89%A9%E3%80%91_%E5%BB%BA%E7%AB%8B%E9%AB%98%E9%86%AB%E4%BA%BA%E7%9A%84%E6%A6%AE%E8%80%80%EF%BC%8D%E5%BB%96%E5%9C%8B%E6%A3%9F]月印萬川－高醫經典人物</ref>

==政策==
=== 競選政見 ===
守護原鄉 • 都會關懷

# 打造智慧原鄉：全面鋪設原鄉寬頻網路，健全原鄉遠距醫療、遠距教育及產銷平台。
# 土地及海域權：制定「原住民族土地及海域法」，維護原民使用土地及海域權益。
# 原鄉照顧關懷：依據長期照顧服務法建置符合原鄉需求的長期照顧體系。
# 保障都會權益：繼續推動都會原民保障條例，完善都會原住民住宅、母語文化、職業訓練、社會福利等政策。
# 花東交通建設：繼續推動花東快速道路，便捷聯絡道，提高花東產業競爭力，增加花東就業機會。
# 花東鐵路改善：推動鐵路雙軌化，提升花東鐵路運載量。
# 母語文化復育：推動海岸山脈兩側阿美族傳統文化生活復育區，復振母語文化。
# 原住民自治權：繼續推動「原住民族自治法」，讓原住民實現自治理想。<ref>[https://chan1688.blogspot.tw/2016/01/blog-post.html 中華民國第九屆立法委員政見總覽] [[公民時報|公民時報]], 2016年1月30日</ref>

=== 其他政策 ===

==== 原住民族教育法修正通過 ====
2014年1月29日，廖國棟委員修法要求中央政府應寬列預算，專辦理原住民族教育，其比率不得少於中央政府教育預算從1.2%調高至2%，另修法保障原住民幼兒有就讀公立幼兒園、非營利幼兒園、社區或部落互助教保服務中心優先權益，並要求直轄縣市社政主管機關應鼓勵、輔導、委託民間或自行辦理未滿二歲原住民幼兒之托育服務，以及鼓勵、輔導或補助法人、團體辦理社區或部落互助式幼兒教保服務，中央社政主管機關得視實際需要予以協助。<ref>[http://law.apc.gov.tw/LawContentDetails.aspx?id=FL008443&KeyWordHL=%20 原住民族教育法], [[原住民族委員會|原住民族委員會]],2014-01-29</ref>

==== 提高原住民社會住宅分配額度 ====
2014年3月17日，廖國棟委員為解決都會區原住民住宅問題提出質詢，[[毛治國|毛治國]]院長同意三個月內提出原住民住宅政策，目前內政部為避免都會區原住民無法申請到社會住宅，已行文要求各縣市政府必須提高原住民社會住宅分配額度，另外各縣市政府的住宅審議委員會至少得設一名原住民族委員，藉此保障原住民居住權益。<ref>[https://lis.ly.gov.tw/lylegisc/lylegiskmout?.5d1f000B201500000000010400000000000^20000100010000330BBC4232 原住民族地區土地合法化面臨問題], [[立法院國會圖書館|立法院國會圖書館]],2014-04-16</ref>

==== 通過石油管理法補助範圍擴及所有原住民地區 ====
2014年4月16日，過去石油管理法僅補助山地原住民及離島地區，廖國棟委員於2013年3月27日率先提案修法讓偏遠地區、原住民及離島地區，其石油設施、運輸費用都能獲得補助，經多次召委身分協商，於103年5月20日通過，並於104年5月公告施行，最高每戶可補助19桶桶裝瓦斯。<ref>[http://misq.ly.gov.tw/MISQ/IQuery/misq5000Action.action 103年4月16日立法院經濟委員會議事錄], [[立法院議事錄|立法院議事錄]],2014-04-16</ref>

==== 學童營養早餐擴大辦理 ====
2014年4月24日，偏鄉經濟弱勢的小朋友，有高達4成以上沒有辦法固定吃早餐，而原住民則更高達5成以上，不只影響兒童的身心健康，更讓學童沒有體力上課，廖國棟委員於總質詢時，向行政院長[[毛治國|毛治國]]建議補編預算，毛院長同意辦理，已在台東縣辦理共有1,115位原住民學童受惠。<ref>[https://lis.ly.gov.tw/lylegisc/lylegiskmout?.a6fe00B2CBB00330000000001000000010000^20000000402000080041c7 偏鄉及原住民地區營養早餐], [[立法院國會圖書館|立法院國會圖書館]],2014-04-24</ref>

==== 土地及自然資源必須依法補償及補助原鄉醫療交通費 ====
2014年5月21日，原基法是民國94年初廖國棟委員擔任立法院原住民問政會會長時通過，為強化對原住民的保障，於2014年5月21日率先提案修正原基法21條及24條三讀通過，未來政府限制原住民使用土地及自然資源必須依法補償，同時明訂原住民遠距醫療交通費補助法源，105年度衛生福利委員部已編列3000萬元補助，較104年度1500萬元增加一倍。<ref>[https://lci.ly.gov.tw/LyLCEW/lcivCommQry.action#pageName_searchResult=15 立法院第104卷第12期], [[立法院議事公報|立法院議事公報]],2014-05-21</ref>

== 軼事 ==
廖國棟非常喜歡唱歌，對音樂充滿熱情，年輕時音質很特別。從國中到高中，除了有四重唱及合唱團的歷練；大一時又特別去學聲樂，就讀高醫時曾經參加過個人獨唱比賽拿到冠軍，也參加過校園民歌比賽獲得第二名。大三時候，還參加過第一屆金韻獎比賽，總決賽當天，因為吉他一條弦斷了沒有得名。事後主辦單位「新格唱片」的董事長還說：「拜託大家，樂器要準備好。」當醫生之後，只能利用空暇之餘唱歌，就連參加喜宴，都會想高歌一曲，大家經常笑他是卡拉OK王子。<ref>[http://enews2.kmu.edu.tw/index.php/Enews286_%E3%80%90%E6%9C%88%E5%8D%B0%E8%90%AC%E5%B7%9D%EF%BC%8D%E9%AB%98%E9%86%AB%E7%B6%93%E5%85%B8%E4%BA%BA%E7%89%A9%E3%80%91_%E5%BB%BA%E7%AB%8B%E9%AB%98%E9%86%AB%E4%BA%BA%E7%9A%84%E6%A6%AE%E8%80%80%EF%BC%8D%E5%BB%96%E5%9C%8B%E6%A3%9F]</ref>
