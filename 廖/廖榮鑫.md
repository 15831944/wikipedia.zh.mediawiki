'''廖榮鑫'''（{{bd|1955年5月1日||||}}），[[中華民國空軍|中華民國空軍]][[二級上將|二級上將]]退役，曾任[[中華民國經濟部|經濟部]][[漢翔航空|漢翔航空]]公司董事長，[[空軍官校|空軍官校]]67年班（1978年班）畢業，同期包括[[本省人|本省人]][[中將|中將]][[陳添勝|陳添勝]]，曾任[[總統府戰略顧問|總統府戰略顧問]]、副參謀總長執行官、國防部常務次長、[[國防部參謀本部防空飛彈指揮部|飛指部]]指揮官、[[空軍防空砲兵指揮部|空軍防砲部]]指揮官、國防部整評室主任、[[空軍官校|空軍官校]]校長、空軍499聯隊長。

==介紹==
戰鬥機飛行員，熱衷賞鳥和閱讀，透過鳥類演化的歷史悟出許多經營及人生哲學<ref>{{cite web | url =http://www.new7.com.tw/coverStory/CoverView.aspx?NUM=1547&i=TXT201610281045275TA | title=【封面故事】廖榮鑫使盡洪荒之力拉拔台灣航太業 |publisher= 新新聞 | accessdate=2016-11-03 }}</ref>。

擔任[[漢翔公司|漢翔公司]]董事長、[[航太公會|航太公會]]理事長及「A-Team 4.0聯盟」會長，並肩負國機國造的重責大任，以畢生經歷透過對各級學校及企業的演講中，給年輕人在人生及職場生涯的兩則信條：人生若要精彩充實，請設下「夢想、選擇和堅持」三個目標；進了職場，請努力培養「視野、胸襟和決心」<ref>{{cite web | url =http://www.chinatimes.com/newspapers/20171011000063-260202 | title= 給年輕人的3句話 夢想、選擇、堅持 |publisher= 工商時報 | accessdate=2017-10-11 }}</ref>。以自身為例：「夢想」是脫離貧窮、「選擇」 是接受教育、「堅持」是不輕言放棄；進入職場的「視野」是飛行的高度與視角、「胸襟」為利他是最好的利己、「決心」則是達成任務的強烈意願（2018年6月9日應邀國立中正大學畢業典禮演講）。

2019年3月17日，在LINE貼文：「一日漢翔人，終身漢翔人」，參與並守護漢翔公司1,476個日子，總能在所有同仁的共同目標、披荊斬棘與創新務實的努力下，劃下完美的句點。

==漢翔公司期間==

* 2015年3月2日就任[[漢翔航空工業股份有限公司|漢翔航空工業股份有限公司]]董事長。須辦理退役後人事令才能生效。
* 2015年8月13日臺北航展期間，有感於全球航太產業的激烈競爭環境，臺灣處於先進國家與新興國家上下夾擊的情況下，遂倡議籌組「[[臺灣航太產業A-Team_4.0聯盟|臺灣航太產業A-Team 4.0聯盟]]」，期以「精實體質、提升競爭」的理念，運用「異業結盟、同業整合」的模式，結合國內原物料、機械設備、生產製造與物流運輸，同時亦獲得銀行業者與學研單位的支持，聯盟於2016年6月2日誓師成立<ref>{{cite web | url =http://www.chinatimes.com/realtimenews/20160602003797-260410 | title=航太ATEAM 4.0爭商機 擺脫同業殺價 |publisher= 中時、中央社 | accessdate=2016-06-02 }}</ref>，目標是成為國際航太大廠的Tier 1供應商。除藉著新式高教機「自研自製」的機會帶著航太供應鏈一起練兵外，更不遺餘力地力邀各個領域的專家，免費為臺灣航太產業的新進企業做教育訓練、奠定基礎，分享產業經驗、國際趨勢與科技新知，希望將人才及技術根留臺灣，向世界級的航太供應鏈聚落邁進，以因應未來可能遭遇的衝擊。
* 2015年8月要求[[漢翔公司|漢翔公司]]於臺中、沙鹿及岡山廠區成立[[工業4.0|工業4.0]]廠區，並開放業界觀摩研討。其後指示投入「iAIDC智慧製造平台」、「決策戰情中心」、「iAI人工智慧」之開發，達成智慧製造三部曲。
* 2015年11月27日當選[[台灣區航太工業同業公會|台灣區航太工業同業公會]]理事長，訂定公會三大方向：資訊分享、擴大產業參與、國際化，同年12月開始在每年舉辦國內航太產業論壇凝聚產業共識，並逐漸醞釀且促成了今日「[[A-Team4.0聯盟|A-Team4.0聯盟]]」2百餘家廠家異業聯盟、優勢整合分工，以及「國機國造」大計畫的實現。「[[A-Team4.0聯盟|A-Team4.0聯盟]]」最高成員數達330家廠家，歷經5次推動委員會審查精實KPI成績，2017年7月10日縮減至256家廠家。
* 推動「國防自主、國機國造」，2017年4月25日代表與[[中科院|中科院]][[張冠群|張冠群]]院長簽訂「空軍新式高教機」合約<ref>{{cite web | url =http://news.ltn.com.tw/index.php/news/business/breakingnews/2047629 | title=漢翔接到高教機委製訂單 今年就可望有入帳 |publisher= 自由時報 | accessdate=2017-04-25 }}</ref>。
* 2018年4月18日指出航太產業未來發展的命脈，必須依靠「三個關鍵要素和三個重點工作」<ref>{{cite web | url =http://www.cna.com.tw/news/afe/201804190024-1.aspx | title=漢翔航空未來發展 就靠3關鍵3重點 |publisher= 中央通訊社 | accessdate=2018-04-19 }}</ref>，「三個關鍵要素」是在親身走訪全球指標性航展、供應商大會、國際航太論壇後，所領悟到的收穫，包括人才、爐灶（練兵場域）、柴火（民機市場與國機國造之業務機會）；「三個重點工作」則是要求公司經營團隊最須聚焦的項目，包括國機國造、智慧製造、供應鏈整合（新式高教機、民機及發動機、機艙內裝等三大供應鏈體系）。
* 2018年6月1日 總統[[蔡英文|蔡英文]]主持漢翔新式高教機組裝開架儀式，宣示國機國造之新式高教機專案已達成2020年6月首飛前的重要里程碑<ref>{{cite web | url =http://www.cna.com.tw/news/firstnews/201806010224-1.aspx | title=高教機產線啟動 總統：走向國防自主 |publisher= 中央通訊社 | accessdate=2018-06-01 }}</ref>。
* 2018年6月26日 續任漢翔董事長。
* 2019年3月4日 鑑於「盱衡當前政經情勢、建立領導傳承制度、紓緩工會緊張關係、穩定同仁工作心緒」等綜合考量，遞陳辭呈「自請退休」<ref>{{cite web | url =https://money.udn.com/money/story/12607/3680021 | title=離新式高教機首飛不到一年卸任 漢翔董座廖榮鑫說心情 |publisher= 經濟日報 | accessdate=2019-03-06 }}</ref>。
*2019年3月6日「自請退休」，將由國防部副參謀總長胡開宏退伍接任漢翔董座。<ref>[https://www.ettoday.net/news/20190306/1392866.htm 從戰鬥機飛行員到漢翔董座　廖榮鑫：自請退休、了無遺憾 ]</ref>
