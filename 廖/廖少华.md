'''廖少华'''（{{bd|1960年|11月||catIdx=L廖}}），[[四川|四川]][[遂宁|遂宁]]人。[[中国共产党|中国共产党]]党员。[[西南交通大学|西南交通大学]]铁道工程系铁道工程建筑专业毕业，[[中共中央党校|中共中央党校]]经济管理专业[[在职研究生|在职研究生]]学历。历任中铁五局副局长兼[[南昆铁路|南昆铁路]]指挥长，共青团贵州省委书记、省青联主席，贵州省计划委员会副主任，中共贵州省[[六盘水市|六盘水市]]委副书记、常务副市长、市长等职。2005年7月调任中共[[黔东南苗族侗族自治州|黔东南苗族侗族自治州]]州委书记<ref>{{cite web|accessdate=2012-05-01 |url=http://gz.people.com.cn/n/2011/1226/c194834-16632014.html |title=黔东南：廖少华当选黔东南州委书记 |publisher=[[人民网|人民网]] |deadurl=yes |archiveurl=https://web.archive.org/web/20120202014148/http://gz.people.com.cn/n/2011/1226/c194834-16632014.html |archivedate=2012-02-02 }}</ref>，2012年7月改任[[遵义市|遵义市]]委书记。

2013年10月28日，贵州省委常委、遵义市委书记廖少华涉嫌严重违纪违法，正接受组织调查。<ref>{{cite web |url = http://www.ccdi.gov.cn/yw/201310/t20131028_34326.html |title = 贵州省委常委廖少华涉嫌严重违纪违法正接受调查 |accessdate = 2013年10月28日 |publisher = 中央纪委监察部网站}}</ref>2014年5月8日被开除党籍和公职，[[最高人民检察院|最高人民检察院]]经审查认为廖少华涉嫌受贿和滥用职权犯罪，依法对其立案侦查并采取强制措施，案件侦查工作正进行中。<ref>{{cite web |url = http://www.ccdi.gov.cn/yw/201405/t20140508_36508.html |title = 贵州省委原常委廖少华被开除党籍和公职 |accessdate = 2014年5月8日 |publisher = 中央纪委监察部网站}}</ref>

2015年1月22日，[[陕西省|陕西省]][[西安市|西安市]]中级人民法院一审公开开庭审理了廖少华受贿、滥用职权案，并于4月9日宣判获刑16年，并处没收财产人民币130万元，他当庭表示认罪悔罪。<ref>{{cite web |url = http://www.chinacourt.org/article/detail/2015/04/id/1580716.shtml |title = 原贵州遵义书记廖少华两罪并处一审获刑16年 |accessdate = 2015年4月9日 |publisher = 中国法院网}}</ref>
