'''廖啟智'''（'''Dick Liu Kai Chi'''，{{bd|1954年|4月7日|catIdx=L廖}}），綽號婚前稱'''智仔'''，婚後改稱'''智叔'''，[[香港|香港]]知名演員<ref>[http://ent.sinchew.com.my/node/37838  廖啟智‧說故事的人 ]2013-01-25 星洲日報</ref>，現為[[香港浸會大學|香港浸會大學]]電影學院全職講師<ref>{{Cite web|url = http://buenews.hkbu.edu.hk/tch/update/news.do;jsessionid=0414259E2C2D118087E4E9E500121820?newsId=731|title = 廖啟智：三年教學路|accessdate = 2015-11-17|author = |date = |publisher = |archive-url = https://web.archive.org/web/20151118033911/http://buenews.hkbu.edu.hk/tch/update/news.do;jsessionid=0414259E2C2D118087E4E9E500121820?newsId=731|archive-date = 2015-11-18|dead-url = yes}}</ref>，其妻是前[[無綫電視|無綫電視]]（TVB）女藝員[[陳敏兒|陳敏兒]]。

== 簡歷 ==
廖啟智在基層家庭長大，自小從[[粵語長片|粵語長片]]看到童星[[馮寶寶|馮寶寶]]，啟蒙他做演員。他曾就讀[[九龍|九龍]][[旺角|旺角]]某一所小學，後來在當時位於[[觀塘|觀塘]]雞寮的天主敎聖神第二校（即現時的[[天主教佑華小學|天主教佑華小學]]）畢業。廖啟智升上中學以後經常轉校，先後入讀鐵路特別中學、摩托工會屬下中學、中業中學以及新僑中學（[[筲箕灣|筲箕灣]]）。{{noteTag|有關廖啟智曾就讀的學校與學習情況的資料出自1994年由明報出版社出版、明報校園版記者所著的《名人說母校》第155頁。}}在1972年中學畢業後，他兩次投考[[無線藝員訓練班|無線藝員訓練班]]才能入圍，成為1979年第8屆無綫藝員訓練班學員<ref name="專訪">{{cite news|title=譚蕙芸：死水翻不了波濤──專訪廖啟智|url=http://www.pentoy.hk/%E6%99%82%E4%BA%8B/v39/2013/10/21/%E8%AD%9A%E8%95%99%E8%8A%B8%EF%BC%9A%E6%AD%BB%E6%B0%B4%E7%BF%BB%E4%B8%8D%E4%BA%86%E6%B3%A2%E6%BF%A4-%E5%B0%88%E8%A8%AA%E5%BB%96%E5%95%9F%E6%99%BA/|newspaper=Pentoy|date=2013-10-21|deadurl=yes|archiveurl=https://web.archive.org/web/20141220081427/http://www.pentoy.hk/%E6%99%82%E4%BA%8B/v39/2013/10/21/%E8%AD%9A%E8%95%99%E8%8A%B8%EF%BC%9A%E6%AD%BB%E6%B0%B4%E7%BF%BB%E4%B8%8D%E4%BA%86%E6%B3%A2%E6%BF%A4-%E5%B0%88%E8%A8%AA%E5%BB%96%E5%95%9F%E6%99%BA/|archivedate=2014-12-20}}</ref>。他的同期同學有[[湯鎮業|湯鎮業]]、[[景黛音|景黛音]]、[[周秀蘭|周秀蘭]]、[[黃造時|黃造時]]、[[陳安瑩|陳安瑩]]、[[艾威|艾威]]、[[陳敏兒|陳敏兒]]、[[李成昌|李成昌]]、[[梁潔華|梁潔華]]、[[黃敏儀|黃敏儀]]、[[傅玉蘭|傅玉蘭]]等。

廖啟智畢業後簽約[[無綫電視|無綫電視]]（TVB），更旋即獲得《[[上海灘|上海灘]]》飾演丁力助手「陳祥貴」一角的機會。他在效力無線期間演過60多部電視劇<ref name="專訪"/>，走實力派路線，擅長演出[[警察|警察]]等角色。1982年離開無綫電視，1984年重返無綫電視至2004年廖啟智不再與無綫電視續約而離開。2005年起在[[香港有線電視|有線電視]]與[[何秀蘭|何秀蘭]]合作主持親子節目《[[我們都是父母|我們都是父母]]》。2005年2月在[[亞洲電視|亞洲電視]]與主持靈異節目《[[驚天大事件|驚天大事件]]》。2006年重返無綫電視，2009年再離開無綫電視，2011年再重返[[無綫電視|無綫電視]]，2012年簽約[[城市電訊|城市電訊]]（後易名爲[[香港電視網絡|香港電視網絡]]）成爲旗下藝員，首部在香港電視網絡的劇集是《[[警界線|警界線]]》，飾演「杜一飛」一角。2013年因香港電視網絡未能如期取得免費電視牌照而暫時離開，同年底再度迴歸無綫電視以自由身並接拍廉政單元劇集《[[廉政行動2014|廉政行動2014]]》，飾演「孖田Sir」一角。2014年拍畢劇集後五度離開無綫電視，同年7月自薦回巢香港電視網絡，接拍《[[選戰|選戰]]》，至10月拍畢。廖啟智在2015年底再以自由身回巢無綫，與[[方中信|方中信]]拍攝劇集《[[律政強人|律政強人]]》。

廖啟智亦有演出電影，在1993年憑《[[籠民|籠民]]》獲得[[第12屆香港電影金像獎|第12屆香港電影金像獎]]最佳男配角獎，2004年則憑《[[無間道II|無間道II]]》再獲提名[[第23屆香港電影金像獎|香港電影金像獎]]最佳男配角，2006年以《[[殺破狼|殺破狼]]》第三度獲提名最佳男配角獎項，2009年憑《[[證人|證人]]》再度榮獲[[第28屆香港電影金像獎|香港電影金像獎]]最佳男配角獎，2016年憑《[[愛存在|愛存在]]》榮獲第一屆[[馬來西亞金蝶獎|馬來西亞金蝶獎]]最佳男演員獎，首膺影帝。

== 家庭 ==
廖啟智在藝員訓練班受訓時，認識了同學[[陳敏兒|陳敏兒]]，二人於1987年11月11日註冊[[結婚|結婚]]，婚後育有3名兒子。幼子廖文諾在2003年證實患上[[血癌|血癌]]，至2006年4月5日<ref>[http://bbs.ent.163.com/bbs/tvb/198504.html [明报周刊]剖白陪伴兒子抗癌感受 陳敏兒抑鬱症曾想死 廖啟智兼數職險崩潰]</ref>病逝。

廖啟智一家皆是[[基督徒|基督徒]]。

== 經典演出 ==
廖啟智經過[[無綫電視|無綫電視]][[長壽節目|長壽節目]]《[[歡樂今宵|歡樂今宵]]》長期訓練，成為節目中其中一名表演高手，他於1990年的《[[歡樂滿東華|歡樂滿東華]]》表演了「穿高跟鞋行鋼線」籌款，成為其最經典的現場表演項目，當時在[[清水灣電視城|清水灣電視城]]外的停車場搭了兩個高台作表演，由於該環節為戶外表演，故曾傳出因風大會增加危險而可能要取消的消息，最後廖啟智穿著約兩至三吋的高跟鞋，雙手拿著油紙傘作平衡，從一個高台走到另一個高台，成功為[[東華三院|東華三院]]籌得善款。另外其妻[[陳敏兒|陳敏兒]]雖然同為該節目的大會司儀之一，但並非該環節的主持，她於丈夫表演期間亦於台下觀看，而無綫則多次拍攝她在鏡頭背後擔心得淚流滿面的表情，場面十分感人。
