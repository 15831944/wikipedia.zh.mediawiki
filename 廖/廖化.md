'''廖化'''{{BD|？||264年||L}}，本名'''廖淳'''，字'''元儉'''，[[荊州_(古代)|荊州]][[襄阳郡|襄阳郡]]中盧縣<ref>《[[s:三國志補註_(四庫全書本)/卷5|s:三國志補註 (四庫全書本)/卷5]]·宗預》：廖化襄陽人也 【襄陽耆舊傳作中盧人】</ref>（今[[湖北省|湖北省]][[襄阳市|襄阳市]]）人，是[[三国|三国]]时期[[蜀汉|蜀汉]]後期的重要将领。

== 生平 ==
[[東漢|東漢]]末年，廖化擔任[[前将军|前将军]][[关羽|关羽]]的主簿，但因[[孫吳|孫吳]]於[[建安|建安]]二十四年（219年）[[樊城之戰|襲取荊州]]，不久关羽[[麥城之戰|战败身亡]]，廖化一度從屬孫吳<ref>《三国志》蜀书十五“为前将军关羽主簿，羽败，属吴。”</ref>。

之后廖化詐死騙過人們，帶著母親昼夜赶路西行，回到[[劉备|劉备]]陣營；此時劉備率軍東征（見[[夷陵之戰|夷陵之戰]]），與廖化在[[秭歸|秭歸]]相遇，劉備見到廖化時大悦，任命为宜都太守<ref>宜都郡治夷道（今[[宜都市|宜都市]]）在孫吳的秭归和夷陵东南，刘备并未实际占领，太守只是虚封。</ref>，其后又任命廖化為[[別督|別督]]，隨大軍駐紮於[[夷陵|夷陵]]界<ref>黃武元年，劉備率大眾來向西界，權命遜為大都督、假節，督朱然、潘璋、宋謙、韓當、徐盛、鮮于丹、孫桓等五萬人拒之。備從巫峽、建平連圍至夷陵界，立數十屯，以金錦爵賞誘動諸夷，使將軍馮習為大督，張南為前部，輔匡、趙融、'''廖淳(廖化)'''、傅肜等各為別督。</ref>，最終劉備被吳將[[陸遜|陸遜]]打敗而結束此役。

[[建興_(蜀漢)|建兴]]元年（223年），丞相[[诸葛亮|诸葛亮]]开府，举[[蒋琬|蒋琬]]为[[茂才|茂才]]，蒋琬坚持推让给[[劉邕_(蜀漢)|劉邕]]、[[陰化|陰化]]、[[庞延|庞延]]、廖化。 <ref>《三國志·蜀書·蒋琬傳》：建兴元年，丞相亮开府，辟琬为东曹掾。举茂才，琬固让刘邕、阴化、庞延、'''廖淳(廖化)'''。</ref>廖化出任丞相参军，随[[诸葛亮北伐|诸葛亮北伐]]。

建兴七年（229年），诸葛亮[[建威之戰|平定武都、阴平二郡]]。后令廖化督[[广武|广武]]<ref>蜀汉时所置，屬陰平郡，治所在今[[四川省|四川省]][[绵阳县|绵阳县]]内</ref>，領[[陰平|陰平]]太守。<ref>郡治阴平县（今[[甘肃省|甘肃省]][[文县|文县]]），原属魏地，建兴七年（229年）春为蜀汉占领。</ref>

[[延熙_(蜀漢)|延熙]]元年（238年）九月，廖化率兵攻打魏国守善羌侯[[宕蕈|宕蕈]]驻守的营寨，魏雍州刺史[[郭淮|郭淮]]派遣广魏太守[[王赟_(三国)|王赟]]、南安太守[[游奕|游奕]]率兵救援，两军沿东西两面分兵合进，欲夹击廖化军。廖化抓住机会各个击破，游奕军被击退，王赟在交战中箭身亡。<ref>《三國志·魏書·明帝纪》裴松之注引魏书曰：九月，'''蜀阴平太守廖惇反'''，攻守善羌侯宕蕈营。雍州刺史郭淮遣广魏太守王赟、南安太守游奕将兵讨惇。淮上书："赟、奕等分兵夹山东西，围落贼表，破在旦夕。"帝曰："兵势恶离。"促诏淮敕奕诸别营非要处者，还令据便地。诏敕未到，奕军为惇所破；赟为流矢所中死。</ref>

延熙十年（247年），[[凉州|凉州]]胡王[[白虎文|白虎文]]、[[治无戴|治无戴]]等率众叛魏降蜀，为魏将郭淮所破，[[姜维|姜维]]率军迎接。次年，姜维令廖化于[[成重山|成重山]]筑城守备，后被魏将郭淮所败。但成功将胡部迁入[[繁县|繁县]]。<ref>《三國志·魏書·郭淮傳》：姜维出石营，从彊川，乃西迎治无戴，留'''阴平太守廖化'''于成重山筑城，敛破羌保质。淮欲分兵取之。诸将以维众西接强胡，化以据险，分军两持，兵势转弱，进不制维，退不拔化，非计也，不如合而俱西，及胡、蜀未集，绝其内外，此伐交之兵也。淮曰：“今往取化，出贼不意，维必狼顾。比维自致，足以定化，且使维疲于奔命。兵不远西，而胡交自离，此一举而两全之策也。”乃别遣夏侯霸等追维于沓中，淮自率诸军就攻化等。维果驰还救化，皆如淮计。</ref><ref>《三國志·蜀書·后主傳》：凉州胡王白虎文、治无戴等率众降，卫将军姜维迎逆安抚，居之于繁县。</ref>

延熙十二年（249年），廖化随姜维进攻[[雍州|雍州]]，姜維遣廖化在[[白水|白水]]南岸扎营与魏将[[邓艾|邓艾]]对峙，欲牵制邓艾军，借以出兵袭取[[洮城|洮城]]。但被邓艾识破，抢先据守洮城，姜维不能攻克，于是随撤兵退走。<ref>《三國志·魏書·鄧艾傳》：三日，维遣'''廖化'''自白水南向艾结营。艾谓诸将曰：“维今卒还，吾军人少，法当来渡而不作桥。此维使化持吾，令不得还。维必自东袭取洮城。”洮城在水北，去艾屯六十里。艾即夜潜军径到，维果来渡，而艾先至据城，得以不败。</ref>

[[景耀|景耀]]二年（259年），廖化再升为右车骑将军<ref>《华阳国志》：以张翼为左车骑将军，廖化为右车骑将军，阎宇为右卫大将军。</ref>、[[节钺|假节]]，领[[并州|并州]][[刺史|刺史]]<ref>并州实际位于[[曹魏|曹魏]]腹地，故刺史之职只是虚封。</ref>，封'''中乡侯'''。其官位與左车骑将军[[张翼_(三国)|張翼]]相當，在鎮軍大將軍[[宗预|宗预]]之上。

景耀五年（262年），廖化於[[姜维|姜维]]出兵[[狄道|狄道]]時表示：「『兵不戢，必自焚』，伯約之謂也。智不出敵，而力少於寇，用之無厭，何以能立？詩云『不自我先，不自我後』，今日之事也。」<ref>《[[漢晉春秋|漢晉春秋]]》</ref>點出了廖化對姜維[[姜維北伐|連年用兵]]、耗損蜀漢國力的擔憂。 

景耀六年（263年）夏，魏军大举伐蜀，廖化与張翼、[[董厥|董厥]]等率軍抵抗<ref>《三国志·蜀书·后主传》：六年夏，魏大兴徒众，命征西将军邓艾、镇西将军钟会、雍州刺史诸葛绪数道并攻。于是遣左右车骑将军张翼、'''廖化'''、辅国大将军董厥等拒之。</ref>，廖化受命往[[沓中|沓中]]支援姜维，后隨姜維棄陰平，與張翼、董厥合軍退守[[劍閣|劍閣]]，魏將[[鍾會|鍾會]]始終未能攻破。<ref>《三国志·蜀书·姜維传》：翼、厥甫至汉寿，维、化亦舍阴平而退，適与翼、厥合，皆退保剑阁以拒会。会与维书曰：“公侯以文武之德，怀迈世之略，功济巴、汉，声畅华夏，远近莫不归名。每惟畴昔，尝同大化，吴札、郑侨，能喻斯好。”维不答书，列营守险。会不能克，粮运县远，将议还归。</ref>是年冬，蜀主[[刘禅|刘禅]]降魏，廖化等人得到刘禅投降的敕令，隨姜维在涪县向钟会投降，蜀漢亡。

咸熙元年（264年）春，廖化與宗預内徙[[洛阳|洛阳]]，途中病死，年逾七十<ref name="七十">《三國志·宗預傳》：時都護諸葛瞻初統朝事，廖化過預，欲與預共詣瞻許。預曰：「吾等年逾七十，所竊已過，但少一死耳，何求於年少輩而屑屑造門邪？」</ref>。

== 典故 ==
*「[[蜀中|蜀中]]無大將，廖化作先鋒」
此諺語並非出自正史記載，而是出自清末反迷信文學作品《[[掃迷帚|掃迷帚]]》第二十四回中老儒所言：「好，好，『蜀中無大將，廖化作先鋒。』，你這少年，公然充起著述名家來，怪極，怪極。」

*「時人語曰：『前有王句，後有張廖。』」
此句出自正史《三國志·蜀志第十三·之王平傳》裴松之注引《[[華陽國志|華陽國志]]》記載，指前期有[[王平_(三国)|王平]]﹑[[句扶|句扶]]，後期有[[張翼_(三国)|張翼]]﹑廖化。意指廖化與另外三位蜀漢將領齊名。(「後有張廖」指張翼﹑廖化在時間上比較後。)

*王平（？－248年），鎮北大將軍，安漢侯。
*句扶（？－249年），左將軍，宕渠侯。
*張翼（？－264年），左車騎將軍，領冀州刺史，都亭侯。
*廖化（？－264年），右車騎將軍，假節，領并州刺史，中鄉侯。

== 生年 ==
廖化生存年代比劉備時代的大將後數十年，根據《三国志》記載初次登場已是大約219年關羽敗亡之際，至於關於早年參加黃巾黨，壽命達一百歲之說，純屬小說三國演義虛構，根據《三國志·宗預傳》記載:「時[[都護|都護]][[諸葛瞻|諸葛瞻]]初統朝事，廖化過預，欲與預共詣瞻許。預曰：『吾等年逾七十，所竊已過，但少一死耳，何求於年少輩而屑屑造門邪？』」，即廖化於264年死時年愈70歲，以71歲～79歲之間計算，即生年在185年～193年之間，219年關羽敗亡之際，廖化應是26歲～34歲之間。<ref name="七十"/>

== 艺术形象 == 
=== 小说形象 ===   
《[[三國演義|三國演義]]》中的廖化從184年的[[黃巾之亂|黃巾之亂]]開始，直到264年蜀漢滅亡，經歷八十年的興衰。原是黃巾賊的餘黨，黃巾之亂結束後，搶劫掠奪生存。在關羽千里尋兄之時，廖化山寨的另一個[[伙伴|-{伙}-伴]][[杜遠|杜遠]]，把關羽的兩位嫂嫂搶到山上，打算將二人分作各人的妻子。廖化誓要歸還，但杜遠不聽從，廖化便將其斬殺。歸還劉備嫂嫂給關羽，打算跟隨關羽，但關羽考慮到他之前是黃巾賊，所以就此拜別。直到劉備奪得荊州後，廖化前來投靠，獲任命為關羽的[[主簿|主簿]]。在樊城之戰，東吳呂蒙襲擊荊州的時候，廖化脫出城池回蜀國找救援。劉備攻打吳國，廖化跟隨。在諸葛亮北伐的時候，曾追擊[[司馬懿|司馬懿]]，得其頭盔。諸葛亮死後，仍是北伐的先鋒。後姜維復國詐降鍾會，廖化知道計劃後陪同，但最後反亂失敗而死于亂軍。

《三国人物事典》認為廖化可能是同名同姓的两个人。但亦可能是廖化沒有直接參與黃巾之亂，而是在黃巾之亂平定後加入黃巾殘黨。《三國志·宗預傳》中記載諸葛瞻在景耀年間剛任都護時，宗預提到自己和廖化都年過七十<ref name="七十"/>。

《三國演義》中廖化升為「飞卫将军」是误用的官名，東漢、三國時代並無此名。<ref name="三国演义大辞典">{{cite book |author=沈伯俊、谭良啸|title=《三国演义大辞典》 |date=2007-07-01 |publisher=中华书局出版社 |location=中國 |language=zh-hans |isbn=9787101044317 }}</ref>
