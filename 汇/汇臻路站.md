==事故==
2019年4月9日下午2时，受强对流天气影响，浦江线汇臻路站站外区间设备遭到雷击，造成该站道岔故障，并伴有异响、烟雾及火花现象。事发时现场无列车通行，未造成人员受伤。浦江线的列车服务亦因此受到影响，事发后列车限速运行，发车班次间隔延长至15分钟。事发后，运营单位迅速启动应急预案，通过调整运营交路确保列车的正常运营。下午2时37分，上海地铁官方微博发布消息称此次故障已经排除，全线运营已恢复正常<ref>{{Cite web |url=http://www.shxwcb.com/253656.html |title=浦江线汇臻路站站外轨道遭雷击，造成道岔故障，无人员伤亡 |accessdate=2019-04-11 |author=钟晖 |date=2019-04-09 |publisher=周到上海（新闻晨报社）}}</ref><ref>{{Cite web |url=https://www.thepaper.cn/newsDetail_forward_3276439 |title=烟雾、异响还有火花！上海轨交浦江线汇臻路站遭雷击无人受伤 |accessdate=2019-04-11 |author=陈逸欣 |date=2019-04-09 |publisher=澎湃新闻}}</ref><ref>{{Cite web |url=https://mini.eastday.com/a/n190409225459101.html?qid=360sousuo&vqid=5LiK5rW35Zyw6ZOB56uZ6YGt6Zu35Ye7MjAxOTA0MDk |title=上海地铁站遭雷击 闪电火花闪耀无人员受伤_社会频道_东方新闻 |accessdate=2019-04-11 |date=2019-04-11 |work=mini.eastday.com}}</ref>。
