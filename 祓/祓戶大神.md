'''祓戶大神'''（{{lang-ja|ハラエドノオオカミ}}）為[[日本|日本]][[神道|神道]]中掌管[[祓|祓]]的[[神祇|神祇]]，《[[記紀|記紀]]》二書並無記載，但《[[延喜式|延喜式]]》大祓詞則提到此四神乃'''[[瀨織津姬神|瀨織津姬神]]'''（{{ja}}セオリツヒメノカミ）、'''[[速秋津日子神與速秋津比賣神|速開都比賣]]'''（{{ja}}ハヤアキツヒメ）、'''氣吹戶主'''（{{ja}}イブキドヌシ）、'''速佐須良比賣'''（{{ja}}ハヤサスラヒメ）等。

== 概要 ==
=== 延喜式之記載 ===
根據《延喜式》卷八神祇八祝詞六月晦大祓<ref>請見{{ja}}[http://www004.upp.so-net.ne.jp/dassai1/engishiki/gen10.htm 古代史獺祭延喜式卷八神祇八祝詞より廣瀬大忌祭原文]。</ref>之記載，祓戶四神為：
# '''[[瀨織津姬神|瀨織津姬神]]'''（{{ja}}セオリツヒメノカミ）：將所有一切禍事、罪愆汙穢沖流入海。
# '''[[速秋津日子神與速秋津比賣神|速開都比賣]]'''（{{ja}}ハヤアキツヒメ）：在海底將一切禍事、罪愆汙穢一飲而盡。
# '''氣吹戶主'''（{{ja}}イブキドヌシ）：將前者吞飲之一切禍事、罪愆汙穢吹往[[根之國|根之國]]、底之國。
# '''速佐須良比賣'''（{{ja}}ハヤサスラヒメ）：將這些帶往根底之國的一切禍事、罪愆汙穢消滅殆盡。

== 解說 ==
[[江戶時代|江戶時代]][[日本國學|國學]]專家[[本居宣長|本居宣長]]在《{{link-ja|古事記傳|古事記伝}}》中提到，瀨織津姬神與[[禍津日神|八十禍津日神]]、速開都比賣與[[伊豆能賣神|伊豆能賣神]]、氣吹戶主與[[直毘神|神直日神]]可以相互對應，而氣吹戶主與[[家宅六神#日本的家宅六神|家宅六神]]的天之吹男神為同一神。《倭姫命世記》<ref>《倭姬命世記》記載：「荒祭宮，一座。[[伊勢神宮|皇太神宮]]荒魂，[[伊奘諾尊|伊弉那伎大神]]所生神，名[[禍津日神|八十枉津日神]]也」。</ref>、《天照坐伊勢二所皇太神宮御鎮座次第記》、《伊勢二所皇太神宮御鎮座傳記》、《中臣祓訓解》等書均記載[[伊勢神宮|伊勢神宮]]内宮別宮{{link-ja|荒祭宮|荒祭宮}}之祭神別名為「瀨織津姬」<ref>{{link-ja|垂加神道|垂加神道}}之看法亦同。</ref>。

== 信仰 ==
在日本，祭祀祓戶大神的[[神社|神社]]主要有下列：
* 佐久奈度神社（[[滋賀縣|滋賀縣]][[大津市|大津市]]） 
* 日比谷神社（[[東京都|東京都]][[港區_(東京都)|港區]]） 
* 鹽竈神社（[[和歌山縣|和歌山縣]][[和歌山市|和歌山市]]）
* 宇良神社（[[京都府|京都府]][[與謝郡|與謝郡]]伊根町）
* 綱越神社（[[奈良縣|奈良縣]][[櫻井市|櫻井市]]）
* 祓戶神社（[[鹿兒島縣|鹿兒島縣]][[霧島市|霧島市]]國分府中町）
