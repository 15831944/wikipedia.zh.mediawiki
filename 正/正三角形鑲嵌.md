在[[幾何學|幾何學]]中，'''正三角形鑲嵌'''、又稱為'''正三角方格'''<ref name="圖解數學辭典">《圖解數學辭典》天下遠見出版 P.50  ISBN 986-417-614-5</ref>是一種[[正多邊形|正多邊形]]在[[平面|平面]]上的[[密鋪|密鋪]]，又稱[[正鑲嵌圖|正鑲嵌圖]]。

== 命名 ==
[[康威|康威]]稱正三角形鑲嵌為deltille。
deltille一詞來自於外形為三角形的[[希臘字母|希臘字母]] [[Delta|Delta]] （[[Δ|Δ]]），有時也稱作[[Kleetope|六角化]][[正六邊形鑲嵌|正六邊形鑲嵌]]。

== 性質 ==
由於正三角形鑲嵌是由[[正三角形|正三角形]]組成，又因正三角形[[內角_(數學)|內角]]為60[[度_(角)|度]]，因此每個[[頂點|頂點]]周圍都有6個[[三角形|三角形]]，且剛好占滿[[360|360]]度。

正三角形鑲嵌在[[施萊夫利符號|施萊夫利符號]]中，用{3,6}表示。

正三角形鑲嵌是三個的平面正鑲嵌圖之一。另外兩個是[[正方形鑲嵌|正方形鑲嵌]]和正六邊形鑲嵌。

一般將畫在紙上的正三角方格稱作正三角格紙<ref name="圖解數學辭典"/>，正三角格紙是用來畫三維立體圖或三維透視圖用的。使用正三角格紙作圖會比較容易做出三維[[立體圖|立體圖]]或三維[[透視圖|透視圖]]，而且圖形看起來比較接近[[三維|三維]]<ref name="圖解數學辭典"/>。

== A<sub>2</sub>晶格和圆堆砌 ==
正三角形镶嵌的顶点排布被称作A<sub>2</sub>晶格<ref>http://www.math.rwth-aachen.de/~Gabriele.Nebe/LATTICES/A2.html</ref>。正三角形镶嵌是{{link-en|单纯形堆砌|Simplectic honeycomb}}家族的二维成员。

A<sub>2</sub><sup>*</sup>晶格（又称A<sub>2</sub><sup>3</sup>），可由所有3种A<sub>2</sub>晶格组合得来，就等价于A<sub>2</sub>晶格。
:{{CDD|node_1|split1|branch}} + {{CDD|node|split1|branch_10lu}} + {{CDD|node|split1|branch_01ld}} = {{CDD|node_1|split1|branch_11}} 的对偶 = {{CDD|node_1|split1|branch}}

以正三角形镶嵌的顶点为圆心，我们可以得到二维的最密{{link-en|圆堆砌|Circle Packing}}，每个圆都与6个相邻圆接触（{{link-en|接触数|kissing number}}），堆砌密度为<math>\frac{\pi}{\sqrt{12}}</math>或90.69%。由于3个A<sub>2</sub>晶格组合还是A<sub>2</sub>晶格，这种圆堆砌种的圆可被涂成三种颜色。

A<sub>2</sub>晶格的[[沃罗诺伊图|沃罗诺伊图]]是[[正六边形镶嵌|正六边形镶嵌]]，它也是正三角形镶嵌的对偶。因此，正六边形镶嵌也与最密圆堆砌有直接的对应关系。
