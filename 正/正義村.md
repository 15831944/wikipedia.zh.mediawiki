'''正義村'''是[[嘉義縣|嘉義縣]][[六腳鄉|六腳鄉]]轄下的行政區，面積5.730034平方公里，有18鄰，人口約1,516人，建有宮廟，和風車的特色景觀。

== 地理 ==
正義村位於六腳鄉南界，東林溪厝、潭墘村，南以朴子溪朴子市相鄰，北街六南村<ref>{{Cite web|url=http://lioujiao.cyhg.gov.tw/cp.aspx?n=F0866217F17BDF5F&s=577B920B4675B832|title=嘉義縣六腳鄉公所|accessdate=2017-07-04|author=|date=|publisher=}}</ref>

== 聚落歷史 ==
村中有蒜頭支線流經，於永曆25年建成，劉傳招佃開墾，面積5.730034平方公里，村莊由兩溪合併日治時期'''[[嘉義潭子墘|潭子墘]]'''中西南部中溝，占厝厝，將村名訂為正義<ref>{{Cite web|url=http://lioujiao.cyhg.gov.tw/cp.aspx?n=F0866217F17BDF5F&s=577B920B4675B832|title=嘉義縣六腳鄉公所|accessdate=2017-07-04|author=|date=|publisher=}}</ref>

== 教育 ==
本村全村均在更寮村[[嘉義縣六腳鄉更寮國民小學|嘉義縣六腳鄉更寮國民小學]]<ref>{{Cite web|url=http://www.gles.cyc.edu.tw/|title=嘉義縣六腳鄉更寮國民小學|accessdate=2017-07-04|author=|date=|publisher=}}</ref>學區。該校創立1946年，南緊鄰朴子市，北接雲林北港鎮，學習園地校區面積1.6公頃，約1.03甲 <ref>{{Cite web|url=http://puzih-hro.cyhg.gov.tw/cp.aspx?n=C4B185FF957E611E|title=朴子戶政事務所|accessdate=2017-07-04|author=|date=|publisher=}}</ref>

== 文化資源 ==
[[六腳鄉威靈廟|威靈廟]]：創立於清乾隆年間，主奉祖師爺由來位於嘉義縣六腳鄉正義村占富厝44號，昭和八年(1933)嘉南大圳開築，是耆老蔡東(蔡文載之父)及保正陸志隆兩人發起鳩金，遷建廟宇，據傳，原為占姓者所開基隨時建變化，占性已稀少，現有居民多雜姓，原初神像俊奉祀於民宅，，迨至戰後經久失修，西年1947年是村長蔡文載發起清杞清水祖師廟基金，重建廟宇。<ref>{{Cite web|url=http://crgis.rchss.sinica.edu.tw/temples/ChiayiCounty/lioujiau/1008019-WLG|title=文化資源地理資訊系統|accessdate=2017-07-04|author=|date=|publisher=}}</ref>

== 景點 ==
