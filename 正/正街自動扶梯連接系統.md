'''正街自動扶梯連接系統'''（{{lang-en|Centre Street Escalator Link}}）為位於[[香港|香港]][[香港島|香港島]][[中西區_(香港)|中西區]][[西營盤|西營盤]][[正街|正街]]的[[自動扶梯|自動扶梯]][[系統|系統]]，系統主要包括了在介乎在[[第三街|第三街]]及[[般咸道|般咸道]]的一段正街、一道長約85米，以及設有4台[[輪椅升降台|輪椅升降台]]的有蓋自動扶梯連接系統等。正街自動扶梯連接系統興建工程於2009年9月展開，至2012年11月15日，[[第三街|第三街]]至[[高街|高街]]一段的扶手電梯啟用。2013年4月25日，[[英華臺|英華臺]]至[[般咸道|般咸道]]一段的雙向扶手電梯啟用。高街至英華臺一段於同年10月28日啟用，表示全個扶梯連接系統於當日起完全啟用。

== 系統內容 ==
正街自動扶梯連接系統內容包括：

*在第三街與般咸道之間的一段正街、全長約85米，並且設有4台輪椅升降台的有蓋自動扶梯連接系統，當中包括：
#在第三街與英華臺之間，3段全長約60米的單向自動扶梯；
#在英華臺與般咸道之間，一段全長約25米長的雙向自動扶梯；
#在第三街與般咸道之間的一段正街的東邊[[行人路|行人路]]，全長約1.6米至2米闊的上蓋
#在第三街與英華臺之間的一段正街進行了[[行人路|行人路]]擴闊及改善工程；
#及，其附屬工程，包括相關的[[道路|道路]]、渠務、環境改善及[[機電工程|機電工程]]等。

== 歷史 ==
===地區背景===
正街連接西營盤海旁的[[德輔道西|德輔道西]]和[[半山區|半山區]]的般咸道，其附近地區是區內的樞紐，設有新鮮食品市場、[[商店|商店]]、[[學校|學校]]、[[社區中心|社區中心]]和[[健康|健康]]護理設施等。路政署預計該地區於2011年將會服務約40,600名居民、約4,500名[[學生|學生]]和約18,400名就業人士。惟該段街道十分陡斜，當中第三街和般咸道之間的路段高度差距高達27米。而街道陡斜對於[[行人|行人]]（尤其是[[老人家|老人家]]及傷殘人士）而言，往來該段街道均會造成困難，在天氣惡劣（例如[[暴雨|暴雨]]）時則尤其顯著。
===發展方向===
加上[[港島綫西延|西港島綫]]工程計劃及西營盤地區的重新發展，預料未來幾年將會為到正街帶來更多人流。擬定興建的正街自動扶梯連接系統將會沿正街為行人提供既[[安全|安全]]又舒適的行人通道。正街自動扶梯連接系統在興建落成後，將會配合西營盤街市大樓樓內現有的自動扶梯，組合成為一個由[[第一街|第一街]]通往半山區般咸道的連續自動扶梯連接系統。該系統亦會連接西港島線[[西營盤站|西營盤站]]的行人通道，以方便往來的行人。此外，預計至2012年，正街自動扶梯連接系統會將正街的雙向行人流量由2009年的每日約12,000人次提高至每日約14,000人次。與此同時，在興建而在興正街自動扶梯連接系統外，亦會進行行人路擴闊和改善工作，以進一步改善環境。
===工程內容===
2008年1月15日，[[路政署|路政署]]就正街自動扶梯連接系統興建計劃的內容外觀[[設計|設計]][[諮詢|諮詢]][[橋樑|橋樑]]及有關[[建築物|建築物]]外觀諮詢委員會，後者對此表示接納。同年2月21日，路政署就正街自動扶梯連接系統興建計劃[[諮詢|諮詢]][[中西區區議會|中西區區議會]]屬下的[[交通及運輸委員會|交通及運輸委員會]]，後者對此表示支持。同年5月20日，路政署向立法會工務小組委員會申請撥款。同年5月31日，路政署根據《道路(工程、使用及補償)條例》(第370章)的規定，將工程刊憲。[[行政長官|行政長官]]會同[[行政會議|行政會議]]在審議未能調解的反對意見後，在2009年3月10日根據該條例在沒有修改的情況下批准工程，公告亦於同年4月3日刊憲。同年6月5日，路政署向立法會財務委員會申請撥款。同月26日，[[路政署|路政署]]招標承接投意正街自動扶梯連接系統工程，至同年7月31日截標<ref>[http://www.info.gov.hk/gia/general/200906/26/P200906260208.htm 正街自動扶梯連接系統工程招標] 《新聞公布》 2009年6月26日</ref><ref>[http://news.hkheadline.com/instantnews/news_content/200906/26/20090626a181235.html?cat=a 正街自動梯工程招標] 《頭條日報》 2009年6月26日</ref>。同年9月，正街自動扶梯連接系統興建工程展開，耗資6,070[[港元|港元]]；是次工程創造了約44個職位，並且提供約800個人工作月的就業機會。

== 負面影響 ==
2014年，西營盤正街有很大一部分的平民化店鋪正在消失，給這片居民區的商業生態系統帶來不利影響的正是正街的自動扶梯系統。由於舖租飆升，很多以往生意興隆的露天大排檔和打印店都已經消失。<ref>{{Cite web |url=http://cn.nytimes.com/china/20140329/c29hongkong/zh-hant/ |title=登山扶梯漸漸改變了老香港的面貌 |publisher=紐約時報中文網 |date=2014-03-29 |accessdate=2014-04-01}}</ref>
