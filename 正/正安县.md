'''正安县'''是[[中华人民共和国|中华人民共和国]][[贵州省|贵州省]][[遵义市|遵义市]]下属的一个[[县级行政区|县]]，位于贵州北端，[[乌江|乌江]]支流[[芙蓉江|芙蓉江]]上游，邻接[[重庆市|重庆市]]。

面积2595平方公里，2017年人口60万；[[邮政编码|邮政编码]]563400，县政府驻[[鳳儀街道|鳳儀街道]]。

==历史沿革==

[[宋朝|宋朝]]初年，属珍州乐源县；[[元朝|元朝]]先属珍州，隶遵义总管府播南路。[[至元_(元惠宗)|至元]]二十三年（1363年），[[明玉珍|明玉珍]]踞蜀称帝，建立[[明夏|明夏]]政权，改珍州为真州，称真州思宁等处长官司。[[明朝|明朝]]时属[[播州杨氏|播州杨氏]]管辖。[[万历|万历]]二十九年（1601年）[[播州之役|播州之役]]后，[[改土归流|改土归流]]，真州长官司改为[[真安州|真安州]]，属[[遵义军民府|遵义军民府]]，隶[[四川布政使司|四川布政使司]]。[[清朝|清朝]][[雍正|雍正]]二年（1724年），[[避讳|避]][[雍正帝|雍正帝]]讳改为[[正安州|正安州]]。雍正六年，正安州随遵义府隶[[贵州布政使司|贵州布政使司]]，自此正安始入黔籍。

[[民国|民国]]二年（1913年），全国[[废府州厅改县|废州]]，改正安县。1949年12月，[[中华人民共和国政府|中华人民共和国政府]]和平接管正安县，设立正安县人民政府，但尚未掌控区、乡、村地区。1950年初，中华民国政府残余势力（即中华人民共和国方面所称的[[土匪|土匪]]）攻占正安县。3月，正安县人民政府撤至[[绥阳县|绥阳县]]旺草区办公<ref name="a1"/>。9月27日<ref>{{Cite web |url=http://www.sohu.com/a/159277490_99932826 |title=时盘棋纪实摄影：解放初期的大西南 |accessdate=2018-03-05 |author= |date=2017-07-23 |work= |publisher=搜狐网|language=Zh-hans }}</ref>，[[中国人民解放军|解放军]]攻占正安县城。与其它地区类似，通过[[全国大剿匪|剿匪]]稳定了当地局势<ref name="a1">{{Cite web |url=http://www.guizhouhong.cn/article-138671.html |title=回忆正安解放初期端恭乡儿童团的故事 |accessdate=2018-03-05 |author=讲述：袁照明，整理：郑雕  |date=2018-02-25 |work= |publisher=贵州红|language=Zh-hans }}</ref>。此后，历属[[遵义专区|遵义专区]]、[[遵义地区|遵义地区]]。
