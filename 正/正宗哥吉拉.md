本作并非某一哥吉拉作品的续作、而是独立的世界观，设定与系列第一作《[[哥吉拉_(1954年)|哥吉拉 (1954年)]]》一样是「巨大不明生物」哥吉拉第一次出现并登陆日本。哥吉拉不仅拥有强大的破坏力还会放出[[核辐射|核辐射]]，令日本陷入前所未有的危机之中。本作不僅呈現出[[福島第一核電廠事故|311核電廠事故]]在日本社會引发的焦虑，也帶有回歸原點的想法。

本片在日本票房大卖82.5亿[[日元|日元]]，在口碑方面也获得非常高的评价。获得第40届[[日本电影金像奖|日本电影金像奖]]最佳影片、最佳导演、最佳摄影、最佳剪辑、最佳美术指导、最佳灯光、最佳音效共计7项大奖。<ref>{{cite web|url=http://www.hollywoodreporter.com/news/godzilla-resurgence-wins-big-at-40th-japan-academy-prize-awards-982922|title='Godzilla Resurgence' Wins Big at 40th Japan Academy Prize Awards|last=Blair|first=Gavin J.|work=Hollywood Reporter|date=March 3, 2017|accessdate=March 4, 2017}}</ref>

=== 哥吉拉 ===
==== 設計 ====
[[前田真宏_(動畫導演)|前田真宏]]繪製[[概念圖|概念圖]]，並起用《[[巨神兵现身东京|巨神兵现身东京]]》擔當[[巨神兵|巨神兵]]造型的[[竹谷隆之|竹谷隆之]]操刀。最初[[樋口真嗣|樋口真嗣]]去電竹谷僅言及設計有[[背鰭|背鰭]]的怪獸，[[庵野秀明|庵野秀明]]則拿著初代哥吉拉塑像找竹谷，想要致敬初代但非照抄而是吸收其優點。庵野提示概念為「完全生物」，由地球生物組成、位居頂點並超出其它存在，因此腳不像[[好萊塢|好萊塢]]版設計成動物作法，無警戒必要就不用耳朵，皮膚形狀參考[[苦瓜|苦瓜]]、呈現潰爛的樣子，身上參考流動熔岩猶如慘痛淌血樣貌，頭部取自[[蕈狀雲|蕈狀雲]]形象，眼睛採[[人眼|人眼]]、理由是生物之中以人眼最可怕。此外，胸骨突出為凸顯頭部屹立模樣，還設定因[[突變|突變]]而急遽進化的五種型態：
*第1型態：有如巨大[[蝌蚪|蝌蚪]]，片中未出現全身造形，僅見到游泳的身影。
*第2型態：頭部像[[皺鰓鯊|皺鰓鯊]]、具備[[兩棲類|兩棲類]]的樣子，片中於[[蒲田|蒲田]]初次登陸。
*第3型態：登陸後長出前肢且能直立，之後回到海中。
*第4型態<ref name=tmc>{{cite video| people     =| date= 2016-08-09 | script-title = ja:シン・ゴジラ』白組によるCGメイキング映像 | url        = https://www.youtube.com/watch?v=66SAVZ4JxY8 | medium     =| publisher  = {{youtube|u=OfficialTohoMovie|{{lang|ja|東宝MOVIEチャンネル}}}}| location   =| accessdate =| accessmonth=| accessyear =| time       =| quote      =| language=}}</ref>：片中再次登陸後的較為完整型態，並因美軍的投彈攻擊發射廣角熱線反擊，造成包括東京政治中心永田町在內的數個行政區幾乎完全毀滅。
*第5型態：片末哥吉拉遭凍結後，最後一幕畫面中顯示轉移至尾巴末端持續進化的型態。
另在構思階段尚有分裂出雙頭，或胸肩腹部長出臉的想法；定案擇取哥吉拉含各生物要素、尾部混同人類特徵再度分裂增生。<ref>{{lang|ja|「シン・ゴジラ」劇場パンフレット：プロダクションノート}}</ref>由於首創日本《[[哥吉拉系列電影|哥吉拉系列電影]]》怪獸描寫使用全CG{{r|tmc}}、儘量去除人味<ref name=toho>{{Cite web | url = https://www.toho.co.jp/movie/news/1607/09godzilla2016_kh.html | script-title = ja:「シン・ゴジラ」完成報告会見| publisher = {{lang|ja|[[東宝|東宝]]}}| date = 2016-07-19| accessdate = }}</ref>，所以在外型上不以「裡頭有人的道具服」為考量來設計。<ref>{{Cite web | url = http://www.cinematoday.jp/page/A0005053 | script-title = ja:庵野秀明、エヴァからゴジラへ創造の裏側～『シン・ゴジラ』を作った男たち - こだわりのビジュアル| publisher = {{lang|ja|シネマトゥデイ}}| date = 2016-07-08| accessdate = }}</ref><ref>{{Cite web | url = http://www.toy-people.com/?p=33239 | title = 《正宗‧哥吉拉》哥吉拉原型與美術設定稿公開！| publisher = 玩具人| date = 2016-07-08| accessdate = }}</ref><ref>{{Cite web | url = http://www.52toys.com/post/detail?id=1006328 | title = 新·哥斯拉诞生记 | publisher = 52toys | date = 2016-07-19| accessdate = }}</ref>

=== 取材 ===
製作人佐藤善宏敘述本作「模擬真實災害電影」、「電影成品裡的幻想產物就只有哥吉拉」。庵野從劇本的執筆階段即請求[[防衛省|防衛省]]、[[自衛隊|自衛隊]]協助合作，假設哥吉拉真的出現時自衛隊的應對方式，以及對哥吉拉武器的使用能否認可，並基於政府[[官僚|官僚]]過去災害事例表現構成劇中對哥吉拉「巨大不明生物」的反應。為了拍攝戰鬥場景，則参加{{link-ja|富士綜合火力演習|富士総合火力演習}}；<ref name="cinema16715">{{Cite web | url = http://www.cinematoday.jp/page/A0005069| script-title = ja:庵野秀明、エヴァからゴジラへ創造の裏側2～巨大生物襲来への妥協なきリアル| publisher ={{lang|ja|シネマトゥデイ}}| date = 2016-07-15| accessdate = }}</ref>其他還有去[[總理大臣官邸|首相官邸]]、赴自衛隊基地出外景，雖經過許可，不過仍遭官邸職員告誡、也不准於作戰本部等處攝影<ref>{{Cite web | url = http://www.cinematoday.jp/page/A0005073 | script-title = ja:庵野秀明、エヴァからゴジラへ創造の裏側3～シン・ゴジラの世界を追求した美術の裏側| publisher = {{lang|ja|シネマトゥデイ}}| date = 2016-07-22| accessdate = }}</ref>。另外，演員並聆聽[[政治人物|政治人物]]、官僚的開會錄音來揣摩說話方式<ref name="cinema16715"/>。

附帶一提，現實中在[[防衛省|防衛省]]改組前的防衛廳時代，曾紙上推演哥吉拉來襲，結論同大量[[老鼠|老鼠]]、[[蟑螂|蟑螂]]成災情形，依《自衛隊法》第83條屬「其他災害」可出動自衛隊，為「有害鳥獸驅除」可使用武器彈藥。<ref>{{Cite web|script-title=ja:ゴジラへの対処、研究していた＝自衛隊出動、武器使用も可能−防衛省 |date=2007-12-21 |url=http://headlines.yahoo.co.jp/hl?a=20071221-00000198-jij-pol |accessdate= |archiveurl=https://web.archive.org/web/20071224132440/http://headlines.yahoo.co.jp/hl?a=20071221-00000198-jij-pol |publisher={{lang|ja|Yahoo!ニュース ([[時事通信社|時事通信社]])}} |archivedate=2007-12-24 |language= |deadurl=yes }} </ref>
