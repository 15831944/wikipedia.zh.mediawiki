'''正广和洋行大楼'''，位于[[上海市|上海市]][[黄浦区|黄浦区]][[福州路_(上海)|福州路]]44号，现为[[中国邮政集团公司|中国邮政集团公司]]上海市机要通信局，建于1937年，由[[公和洋行|公和洋行]]营造。

== 历史 ==
1864年，[[英國|英國]]商人史密斯在[[上海英租界|上海英租界]]创立[[广和洋行|广和洋行]]，主要經營[[洋酒|洋酒]]和[[汽水|汽水]]。同是英國商人考尔伯克和麦克利格兩人在1882年加盟洋行，但史密斯不久後停止合作關係，另外開設老广和洋行；考尔伯克和麦克利格便把广和洋行易名為正广和洋行，以示與史密斯的老广和洋行区分，並取意自「正本清源、广泛流通、和颜悦色」<ref name="東方網1">{{cite news|title=“正广和”的“前世今生” |url=http://history.sina.com.cn/bk/lszh/2014-06-06/192492663.shtml |accessdate=2016-11-25 |agency=东方网 |date=2014-06-06 |archiveurl=https://archive.is/20150723090825/http://history.sina.com.cn/bk/lszh/2014-06-06/192492663.shtml |archivedate=2015-07-23 |deadurl=no }}</ref>。

1937年，正广和洋行在福州路44号建设上海总部大楼，其与[[正广和大班住宅|正广和大班住宅]]同属英国乡村式别墅建筑<ref name="上海市地方志办公室">{{cite web|title=英国商人的上海家园：正广和洋行大班住宅 |url=http://www.shtong.gov.cn/newsite/node2/node71994/node81772/node81775/node81786/userobject1ai109013.html |website=上海市地方志办公室 |accessdate=2016-11-25 |archiveurl=https://archive.is/20161125142936/http://www.shtong.gov.cn/newsite/node2/node71994/node81772/node81775/node81786/userobject1ai109013.html |archivedate=2016-11-25 |deadurl=no }}</ref>。正广和洋行大楼具有红瓦陡坡屋面，具有哥特式锯齿平面状建筑风貌的清水红砖烟囱<ref>{{cite journal |title=150年初心不改好味依旧 |journal=申江服务导报 |date=2014-05-21 |page=A18 |url=http://newspaper.jfdaily.com/sjfwdb/html/2014-05/21/content_1168731.htm}}</ref>。

1994年，该建筑被上海市公布为[[上海市优秀历史建筑|上海市第二批优秀历史建筑]]<ref>{{cite book |author1=上海市地方志办公室 |title=上海名建筑志 |date=2005 |publisher=上海社会科学院出版社 |url=http://shtong.gov.cn/node2/node71994/node81772/node81779/node81794/userobject1ai108895.html |accessdate=2018-09-04 |chapter=上海市人民政府1994、1999年公布的337处上海市优秀历史建筑一览表（第二批、第三批）}}</ref>。2004年2月16日，该建筑被公布为黄浦区不可移动文物<ref>{{cite web |language =  |publisher = 上海市文化广播影视管理局 |title = 黄浦区县级文物保护单位、登记不可移动文物 |url = http://wgj.sh.gov.cn/wgj/node1257/node1264/u1a75610.html |author = 上海市文化广播影视管理局 |date = 2011-10-28 |accessdate = |archiveurl= https://web.archive.org/web/20120512174125/http://wgj.sh.gov.cn/wgj/node1257/node1264/u1a75610.html | archivedate=2012-05-12 }}</ref>。同日，该建筑也被公布为上海市第一批登记不可移动文物<ref>{{cite book |author1=上海市地方志办公室 |title=上海名建筑志 |date=2005 |publisher=上海社会科学院出版社 |isbn=7-80681-736-0 |url=http://shtong.gov.cn/node2/node71994/node81772/node81779/node81794/userobject1ai108900.html |section=上海市第一批登记不可移动文物一览表（4）}}</ref>。在[[第三次全国文物普查|第三次全国文物普查]]中，该建筑也被列为不可移动文物<ref>{{cite web |author1=上海市文化广播影视管理局 |title=上海市第三次全国文物普查不可移动文物名录 |url=http://wgj.sh.gov.cn/node2/n2029/n2031/n2062/u1ai107979.html |website=上海市文化广播影视管理局 |language=zh-CN |quote=310102945190000084 正广和公司旧址 民国 外滩街道}}</ref>。
