在[[幾何學|幾何學]]中，'''正六邊形鑲嵌'''是一種[[平面|平面]][[鑲嵌|鑲嵌]]，由[[正六邊形|正六邊形]]重覆[[組合|組合]][[排列|排列]]而成，且填滿整個[[平面|平面]]，而且沒有任何[[空隙|空隙]]或[[重疊|重疊]]，由於皆由[[正多邊形|正多邊形]]組成，因此稱為[[正鑲嵌圖|正鑲嵌圖]]。正六邊形鑲嵌是三维欧几里得空间中三个正密铺之一。另外两个分别是[[正三角形镶嵌|正三角形镶嵌]]和[[正方形镶嵌|正方形镶嵌]]。

由於'''正六邊形鑲嵌'''是由[[正六邊形|正六邊形]]組成，又因正六邊形[[內角_(數學)|內角]]為[[120|120]][[度|°]]，因此每個[[頂點|頂點]]周圍都有3個[[正六邊形|正六邊形]]，且剛好占滿[[360|360]]°，才能填滿[[平面|平面]]。

在[[施萊夫利符號|施萊夫利符號]]中，'''正六邊形鑲嵌'''可用{6,3}或t{3,6}表示。

== 圆堆砌 ==
正六边形镶嵌可以被用来进行{{link-en|圆堆砌|Circle packing}}，以其每个顶点为圆心放置等[[直径|直径]]的圆。在这个堆砌里，每个圆都与3个相邻圆接触（{{link-en|接触数|Kissing number problem}}）。每个正六边形中间的部分实际上还可以再放入一个圆，这样我们就会得到二维最密圆堆砌——[[正三角形镶嵌|正三角形镶嵌]]式圆堆砌，这时接触数达到最大值6。

== 相关半正镶嵌 ==
正六边形镶嵌可以通过截角操作得到一系列与之相关的半正镶嵌，其与正六边形镶嵌拥有相似的对称性：
{{正三角形镶嵌家族}}
正六边形镶嵌在拓扑上与一系列一直延伸到双曲镶嵌的[[顶点图|顶点图]]为''n''<sup>3</sup>的（广义）多面体相关：
{{三阶正镶嵌}}
（三阶）正六边形镶嵌在拓扑上与一系列面为[[正六边形|正六边形]]的密铺相关联，这些镶嵌都可称之为“正六边形镶嵌”，所以我们以“n 阶”来区分，其[[施莱夫利符号|施莱夫利符号]]为{6,n}，{{link-en|考克斯特符号|Coxeter diagram}}{{CDD|node_1|6|node|n|node}}，一直到n = ∞：
{{正六边形镶嵌}}
这个镶嵌还是一系列有[[考克斯特对称群|考克斯特对称群]][n,3]对称性的（半）截角菱形多面体或镶嵌的一员。[[立方体|立方体]]可以被看作是“菱形六面体”，这里菱形就是正方形。它们的截角形在原顶点处有正的多边形，而原来的菱形面则被截成了非正六边形。这一系列多面体或镶嵌有两种顶点图：(n.6.6)和(6,6,6)。

== 應用 ==
正六邊形鑲嵌是二維空間最密的排列方式。在[[蜂窩猜想|蜂窩猜想]]中，正六邊形鑲嵌是使用最少的總[[周長|周長]]將該表面劃分成[[面積|面積]]相等的區域的最佳方法。<ref>
