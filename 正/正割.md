'''正割'''（Secant，<math>\sec</math>）是[[三角函数|三角函数]]的一种。它的[[定义域|定义域]]不是整个[[实数集|实数集]]，[[值域|值域]]是[[絕對值|絕對值]][[大於等于|大於等于]][[一|一]]的[[实数|实数]]。它是[[周期函数|周期函数]]，其最小正[[周期|周期]]为<math>2 \pi</math>。

'''正割'''是[[三角函数|三角函数]]的[[正函數|正函數]]（[[正弦|正弦]]、[[正切|正切]]、'''正割'''、[[正矢|正矢]]）之一，所以在<math>2k \pi</math>到<math>2 k \pi + \frac{\pi}{2}</math>的區間之間，函數是[[遞增|遞增]]的，另外'''正割'''函数和[[餘弦|餘弦]]函数互為[[倒數|倒數]]。

在[[單位圓|單位圓]]上，正割函数位於[[割線|割線]]上，因此將此函數命名為正割函数。

和其他[[三角函數|三角函數]]一樣，正割函数一樣可以擴展到[[复数_(数学)|複數]]。

==符号史==
正割的数学符号为<math>\sec</math>，出自英文secant。该符号最早由数学家吉拉德在他的著作《三角学》中所用。

==定义==

=== 直角三角形中 ===
[[File:Rtriangle.svg|left]]
在[[直角三角形|直角三角形]]中，一个锐角<math>\angle A</math>的'''正割'''定义为它的斜邊与鄰邊的[[比值|比值]]，也就是：

:<math> \sec \theta = \frac {\mathrm{c}}{\mathrm{b}}\,\!</math>

可以發現其定義和[[餘弦函數|餘弦函數]]互為[[倒數|倒數]]。

===直角坐标系中===
设<math>\alpha</math>是平面直角坐标系xOy中的一个[[象限角|象限角]]，<math>P\left( {x,y} \right)</math>是角的终边上一点，<math>r = \sqrt {x^2 + y^2 }>0</math>是P到原点O的[[距离|距离]]，则<math>\alpha</math>的正割定义为：

:<math>\sec \alpha = \frac{r}{x}\,\!</math>
{{clear}}
===单位圆定义===

[[File:Unit_circle_angles.svg|300px]]]]

图像中给出了用弧度度量的某个公共角。[[逆时针方向|逆时针方向]]的度量是正角而[[顺时针|顺时针]]的度量是负角。设一个过[[原点|原点]]的[[线|线]]，同''x''轴正半部分得到一个角<math>\theta</math>，并与单位圆[[相交|相交]]。这个交点的''y''坐标等于<math>\sin \theta</math>。在这个图形中的[[三角形|三角形]]确保了这个公式；半径等于斜边并有长度1，所以有了<math>\sec \theta =\frac{1}{x}</math>。单位圆可以被认为是通过改变邻边和对边的长度并保持斜边等于1查看无限数目的三角形的一种方式。
{{clear}}
对于大于<math>2\pi</math>或小于<math>-2\pi</math>的[[角度|角度]]，简单的继续绕[[单位圆|单位圆]][[旋转|旋转]]。在这种方式下，正割变成了周期为<math>2\pi</math>的[[周期函数|周期函数]]：

:<math>\sec\theta = \sec\left(\theta + 2\pi k \right)</math>

对于任何角度<math>\theta</math>和任何[[整数|整数]]<math>k</math>。

=== 與其他函數定義 ===
[[正割函數|正割函數]]和[[餘弦函數|餘弦函數]]互為[[倒數|倒數]]

即：
:<math>\sec x = \frac{1}{\cos x}</math>

=== 級數定義 ===
正割也能使用泰勒級數來定義：
:<math>\sec x = 1+\frac{x^2}{2}+\frac{5 x^4}{24}+\frac{61 x^6}{720}+\frac{277 x^8}{8064}+\frac{50521 x^{10}}{3628800}+.....</math>。

=== 微分方程定義 ===
[[sec|sec]]的微分是[[sec|sec]]和[[tan|tan]]的[[乘積|乘積]]
:<math>\sec 'x\ = \sec x\tan x</math>

[[sec|sec]]的[[導數|導數]]如下：
*<math>\sec 'x\ = \sec x\tan x</math>
*<math>\sec ''x\ =\sec^{3} x +\sec x\tan^2 x</math>
*<math>\sec '''x\ =5\sec^{3} x\tan x +\sec x\tan^3 x</math>
*<math>\sec ''''x\ =5\sec^{5} x+18\sec^{3}x\tan^{2}x +\sec x\tan^4 x</math>

另外
:<math>\int \sec \theta  \, d\theta = -\ln \left(\cos \frac{\theta }{2}-\sin \frac{\theta }{2}\right)+\ln \left(\cos \frac{\theta }{2}+\sin \frac{\theta }{2}\right)</math>

所以微分方程定義為：
:<math>\sec x = \left[-\ln \left(\cos \frac{\theta }{2}-\sin \frac{\theta }{2}\right)+\ln \left(\cos \frac{\theta }{2}+\sin \frac{\theta }{2}\right)\right]'</math>

=== 指數定義 ===
<math>\sec \theta = \frac{2}{e^{{\mathrm{i}}\theta} + e^{-{\mathrm{i}}\theta}} \,</math>

== 恆等式 ==
=== 和差角公式 ===
:<math>\sec(\theta\pm\psi)=\frac{\sec\theta\sec\psi} {1\mp\tan\theta\tan\psi}
</math>

== 巴洛的正割積分 ==
巴洛在1670年提出正割的積分
:<math>\int_0^{\phi}\sec t \, dt = \ln\tan\left(\frac{\pi}{4}+\frac{\phi}{2}\right)</math>
== 正割定理 ==
[[File:LabeledTriangle.svg|thumb]]
有一些含有正割的恆等式，滿足任意三角形ABC：
:<math>a = \frac{b}{\sec C}+\frac{c}{\sec B}</math>
:<math>b = \frac{a}{\sec C}+\frac{c}{\sec A}</math>
:<math>c = \frac{a}{\sec B}+\frac{b}{\sec A}</math>

這些實際上是[[射影定理|射影定理]]的[[倒數|倒數]]。
