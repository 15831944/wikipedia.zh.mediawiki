'''條紋林狸'''（''Prionodon linsang''）是西[[馬來西亞|馬來西亞]]、[[蘇門塔臘|蘇門塔臘]]、[[婆羅洲|婆羅洲]]、[[爪哇|爪哇]]、[[泰國|泰國]]及[[印尼|印尼]]的[[麝貓|麝貓]]。

== 特徵 ==
條紋林狸連尾巴約長74厘米。牠們呈淡黃色，有5條深色條紋。牠們[[頸部|頸部]]的條紋較闊，尾巴上有幾條深色斑紋。牠們的爪很鋒利及可以伸縮，[[牙齒|牙齒]]像剃刀般鋒利。

== 食性 ==
條紋林狸是[[雜食性|雜食性]]的，主要吃[[松鼠|松鼠]]、[[大家鼠|大家鼠]]、[[鳥類|鳥類]]及[[蜥蜴|蜥蜴]]。

== 棲息地 ==
條紋林狸分佈在西[[馬來西亞|馬來西亞]]、[[蘇門塔臘|蘇門塔臘]]、[[婆羅洲|婆羅洲]]、[[爪哇|爪哇]]、[[泰國|泰國]]及[[印尼|印尼]]。牠們棲息在[[熱帶|熱帶]][[森林|森林]]，且大部份時間會在樹上活動。
