'''戛纳'''<ref>其简体中文译名，在法国旅游发展署官方网站的网页上同文出现过"戛纳"、"嘎纳"两种，而现在"戛"与"嘎"读音已不同。嘎纳（CANNES），精品生活的目的地http://cn.rendezvousenfrance.com/zh-hans/information/23301，文内；嘎纳博彩中心http://cn.rendezvousenfrance.com/zh-hans/news/23110{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>（{{lang-fr|Cannes}}，{{IPA-fr|kan}}； [[加斯科方言|加斯科]]：''Canas''）是法國南部的一個[[市鎮_(法國)|市鎮]]，位在[[濱海阿爾卑斯省|濱海阿爾卑斯省]]的[[蔚藍海岸|蔚藍海岸]]地區，鄰近[[地中海|地中海]]，人口約70,000人，是[[坎城影展|坎城-{zh-hans:电影节;zh-hant:影展;zh-tw:影展;zh-hk:影展;zh-mo:影展;}-]]的所在地。坎城也以其優美的沙灘為名。

== 歷史背景 ==
[[File:Cannes_SPOT_1160.jpg|thumb]]
在[[中古世紀|中古世紀]]時，康城是[[Lérins修道院|Lérins修道院]]的封建屬地。直到19世紀早期，康城成為一個農漁業的小村落，在Suquet山丘的人口密度相當高。1830年代開始，法國及外來的[[貴族|貴族]]逐漸在此建立他們的度假小屋，因此康城漸漸成了一個[[度假村|度假村]]。

[[亨利·彼得·柏罕|亨利·彼得·柏罕]]（Henry Peter Brougham，1778年-1868年）是負責康城發展的領主，當時他是一位倍受尊崇且頗有才能的英國政治家，1834年在前往義大利的途中，發現了康城。買下[[Suquet|Suquet]]山丘西邊的土地之後（現今為進康城的必經之路），柏罕靠著在法國政治圈的門路發展其餘的土地。

此濱海村落的發展帶動當地企業發展以及[[有軌電車|有軌電車]]的興建。[[康城電車|康城電車]]之後於1899年啟用，而[[康城車站|康城車站]]更早於1863年啟用。

二戰時期1942年[[義大利王國|義大利]]軍队佔领康城，直至1943年。

== 經濟 ==
康城的週邊區域已發展成一個高科技的重地，[[:en:Sophia-Antipolis|Sophia-Antipolis]]的[[科技園區|科技園區]]就位在康城後方的山丘上。 

=== 活動慶典 ===
*[[坎城影展|坎城影展]]於每年5月舉辦，是全球電影業的重要盛會。
*電視節在9月最後一個禮拜舉辦。
*Mipim是大型的商業不動產展覽會，於3月舉辦，展覽攤位可由此網站[[www.palmedazur.com|www.palmedazur.com]]租用。
*[[:en:Cannes_Lions_International_Advertising_Festival|康城獅國際廣告节]]
*其他大主要的活動節慶有：[[:en:Midem|Midem]]、[[:en:MIPIM|MIPIM]]、[[:en:MIPTV_Media_Market|MIPTV Media Market]]、Mipcom、Mapic和TFWA。

== 文化 ==
{{see also|坎城影展}}
由於許多組織的貢獻，康城經年都受到各種藝術的薰陶。康城市立音樂戲劇學校創立了音樂启蒙計畫，對象有3歲以上學習大中小提琴的幼兒以及5歲以上註冊報名參加音樂启蒙計畫的兒童。

== 交通 ==
[[File:Cannes_57.jpg|thumb]]
[[戛纳站|戛纳站]]位在[[義大利|義大利]]到[[馬賽|馬賽]]間的鐵路運輸線上，[[法國高速列車|法國高速列車]]、國內長途火車及區域快車（[[:en:Transport_express_régional|Transport express régional]]）都在此提供運輸服務。[[戛纳站|戛纳站]]的位置靠近市中心，有巴士往返於康城車站和市政廳。

巴士由[[:en:Bus_Azur|Bus Azur]]公司營運，主要分為兩種：一為電汽巴士，駛於市中心；另一為雙層巴士，往返於市政廳與[[:en:Palm_Beach_Casino|棕櫚灘賭場]]。

康城位於[[尼斯|尼斯]]以西25公里，從[[尼斯藍色海岸機場|尼斯藍色海岸機場]]搭乘公車即可到達，接駁巴士服務時間是上午8點至下午8點，每半小時發車（早上及下午6時過後為每小時發車），搭乘時間為30分鐘。若搭乘計程車，一趟花費為60[[歐元|歐元]]。
