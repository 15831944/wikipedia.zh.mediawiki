《'''遊戲王VRAINS'''》（{{lang-ja|遊☆戯☆王VRAINS}}）是一部於2017年開播的[[日本|日本]]電視動畫，為《[[遊戲王|遊戲王]]》系列的第六部作品。該作最初於2017年2月對外宣布，電視動畫定於2017年5月起在[[東京電視網|東京電視網]]開始播放，接續前作《[[遊戲王ARC-V|遊戲王ARC-V]]》<ref>{{Cite web |url = https://twitter.com/yugioh_anime/status/809982779886686208 |title = 【新情報】アニメ「遊☆戯☆王」シリーズ第６作目 |work = アニメ「遊☆戯☆王」公式 |date = 2017-02-16 |language = ja |accessdate = 2017-02-16}}</ref>。該作的標語是「讓我們向前邁進一步，嘗試一下吧！」，並透過[[虛擬實境|虛擬實境]]展現決鬥。作品的主角是{{nihongo|藤木遊作|ふじき ゆうさく}}，當他在VR空間內時，他的髮型和服裝會改變<ref name="ANN wsj">{{Cite web |url=http://www.animenewsnetwork.com/news/2017-02-16/new-yu-gi-oh-tv-anime-reveals-title-plot-details/.112302 |title=New Yu-Gi-Oh! TV Anime Reveals Title, Plot Details |date=2017-02-16 |publisher=[[動畫新聞網|動畫新聞網]] |access-date=2017-03-15}}</ref>。作品首次引入「連結召喚（リンク召喚）」<ref>[http://www.yugioh-card.com/japan/howto/new_master_rule/ リンク召喚]</ref>和「[[虛擬實境|VR]]決鬥（ブイアールデュエル）」等設定。2017年3月，原作者[[高橋和希|高橋和希]]在自己的[[Instagram|Instagram]]透露，該作的[[人物設計|人物設計]]完全由動畫團隊所負責<ref name="kazuki1">{{Cite web |author=高橋和希 |date=2017-03-14 |url=https://www.instagram.com/p/BRnnm0fAGV9/ |title=kazuki takahashiのInstagram |accessdate=2017-03-15}}</ref>。

名稱「VRAINS」為「VR」（虛擬實境）、「AI」（人工智慧）和「NS」（網路系統）所組成。

== 故事大綱 ==
=== 漢諾騎士篇 ===
* '''TURN 1 ～ 46'''

故事設定在科技及網路技術發達的城市「{{nihongo|Den City|デンシティ}}」，由大型企業「{{nihongo|SOL技術社|SOL テクノロジー社}}」設立的[[虛擬實境|VR]]空間「{{nihongo|LINK VRAINS|リンク ヴレインズ|-{LINK VRAINS}-}}」深受民眾喜愛，人們能在該空間內體驗決鬥樂趣，但LINK VRAINS內也存在著駭客組織「{{nihongo|漢諾騎士|ハノイの騎士}}」，他們的目的是將存在於網路深處中由AI「伊格尼斯」所創造的世界——「網域界」給消滅殆盡。為了逃離全面毀滅的命運，其中一個「伊格尼斯」將整個網域界隱藏起來，並帶著藏匿地點的座標逃逸無蹤。

五年後，「LINK VRAINS」裡出現了一位神秘決鬥者「Playmaker」，他的真實身份是藤木遊作，為了追查自己過去遭遇的綁架事件而與該組織對抗，他在過程中捕獲了伊格尼斯作為人質，並結識決鬥者「Go鬼塚」鬼塚豪和「藍天使」財前葵。在Playmaker首次戰勝漢諾騎士成員左輪後，伊格尼斯取回了自己的身體；而在擊敗魅影嬌娃和SOL的課長財前晃後並取得SOL的資料後，他發現綁架事件的策畫者鴻上博士已經身亡。之後不久，LINK VRAINS的決鬥者陸續出現了被強制登入與漢諾騎士成員決鬥的現象，戰敗者將成為陷入昏迷的「往生者」。Go鬼塚為了變成往生者的童年朋友而擊敗了往生者事件的執行者之一基因組博士。之後藍天使也戰勝另一名執行者拜菈後，拜菈自行將電腦病毒的刪除程式公諸於世，使往生者們恢復原狀。剩下的一名執行者浮士德則為了引出Playmaker而綁架了島直樹，最終也在與Playmaker的交戰中戰敗；但在Playmaker戰勝後也對伊格尼斯的存在感到質疑。

當鴻上、左輪和亡靈開始執行完成漢諾塔的行動時，Playmaker察覺這行為可能會使網絡從此消失，便和GO鬼塚與藍天使共同阻止對方；但在此前，魅影嬌娃卻遭左輪擊敗而數據化。在亡靈擊敗藍天使後，和Playmaker展開決鬥；過程中，亡靈透露出自己的過去和真實身份，最後輸給了Playmaker，而晃也在此前遭數據化。不久後，Playmaker看見Go鬼塚正與左輪進行決鬥，並在前者敗北後，左輪接著和Playmaker展開交戰。但在決鬥中途，左輪獲得父親給予的新力量後發現父親已失去了生命跡象，並自行將該決鬥以平局收場。回到現實，遊作從對方提及的「星塵大道」得知了左輪的位置並隨草薙前往該處。左輪與兩人見面後，透露了自己的本名「鴻上了見」及關於Lost 事件的種種細節。遊作和了見決定再次展開決鬥，並在漢諾塔頂進行；最後，Playmaker成功擊敗了對方，使LINK VRAINS恢復和平。待了見乘船離去後，遊作放走了伊格尼斯。

=== 伊格尼斯篇 ===
* '''TURN 47 ～'''
數個月後，先前漢諾騎士摧毀的LINK VRAINS在SOL技術社的重建下成為了「新生LINK VRAINS」，其樣貌的構造與先前完全不同，且也使決鬥者們盡情地進行高速決鬥。某日，草薙的弟弟仁遭到一名神秘人奪走了意識，使遊作再次以Playmaker的身份進入LINK VRAINS，但此時的他已是SOL的通緝目標。Playmaker在透過決鬥擊敗該神秘人物後，得知對方名為「鮑曼」，且還有一名同夥「哈爾」，但最後仍被對方逃走。與此同時，Playmaker與Ai重聚，並表示網域界已遭人毀滅；且還認識了前來幫助的決鬥者「Soulburner」，並在事後得知了對方的真身「穂村尊」及同行的炎之伊格尼斯「不靈夢」，且也知曉對方是Lost 事件的六名受害者之一。達成結盟共識的Playmaker和Soulburner打算潛入LINK VRAINS最底層的禁止進入區域，在Soulburner擊敗受SOL雇傭為賞金獵人的Go鬼塚，而Playmaker擊敗賞金獵人「血色牧羊人」後，兩人遇見了風之伊格尼斯「溫迪」。之後，當兩人繼續朝內部深入時，換Soulburner對上打算前來攔截的「蒼藍嬌娃」（藍天使的另一身份）；蒼藍嬌娃在輸給對方後與魅影嬌娃一同登出。與此同時，Playmaker則再次與似乎被植入了假記憶的鮑曼展開決鬥，最終由Playmaker取勝；決鬥結束後，該空間崩塌而使Playmaker與Soulburner登出。另一方面，漢諾騎士的了見隨亡靈、基因組博士、浮士德一同潛入高密度監獄中救出了拜菈，並打算再次展開行動。

在闖入陷阱的Soulburner與血色牧羊人打成平手後，Playmaker和Ai遇見了地之伊格尼斯「亞斯」；雖然在透過決鬥取勝對方，但亞斯卻完全不知曉關於鮑曼的事。亞斯也在決鬥中透露，水之伊格尼斯「阿葵雅」早已預知到網域界即將遭到消滅的命運，但阿葵雅則下落不明。之後，溫迪和光之伊格尼斯「萊德寧」向Ai透露，他們正打算重建網域界並企圖讓人類滅絕，因為伊格尼斯本身就是作為人類的後繼種而存在。但當Ai和Playmaker拒絕合作時，溫迪與萊德寧卻打算消滅他們；此時，左輪帶領漢諾騎士們現身阻止並打算摧毀此處。為了消滅伊格尼斯，左輪向溫迪提起決鬥，首次展現同步召喚並華麗逆轉勝，刪除溫迪的過程中卻遭到萊德寧介入而未果。接著，鮑曼與哈爾出現，萊德寧揭露手邊的黑影人形正是Playmaker尋找多時的人，也是他的原型草薙仁，承認自己就是摧毀網域界的幕後黑手，並向人類正式宣戰。而後，為了修復受損的溫迪，萊德寧一行人起身離開，Playmaker與Soulburner雖然追了上去，可卻分別受到鮑曼與哈爾、大量彼特與布特的阻撓，加上SOL的賞金獵人隊的到來，不但Soulburner沒能追上萊德寧，Playmaker也未能戰勝鮑曼，無功而返。

追擊期間，Playmaker與Ai從鮑曼那裡得知，原來鮑曼、哈爾彼特與布特都是萊德寧製作出來的新世代AI，目的是為了探尋伊格尼斯的未來，但初步製作出來的鮑曼仍不完全，需要以他人為原型來完善自我，所以才三番兩次與Playmaker決鬥，因為萊德寧認為自己不夠有個性，而他與Ai會是很好的範本，先前清除記憶與捏造記憶也都是為了這個。之後，鮑曼以自身和Ai為賭注與Playmaker進行高速決鬥，利用風暴連線召喚出全新的王牌怪獸，將Playmaker逼入絕境，Playmaker則以Ai製作的新技能「新風暴連線」成功施展同步召喚加以反擊，最後鮑曼使用墓地的卡片效果使雙方平攤傷害，決鬥以平局收場。回到現實世界後，了見來到遊作身邊，交給他一個特殊程式，用以防止遊作遭到鮑曼襲擊而被奪取意識數據。在Go鬼塚接受被SOL植入AI晶片的試驗後，他獲得了全新的力量並擊敗了救出水之伊格尼斯「阿葵雅」的亞斯，而阿葵雅則又被蒼藍嬌娃及魅影嬌娃救走；在魅影嬌娃被血色牧羊人擊敗後，阿葵雅正式與蒼藍嬌娃共同行動，而蒼藍嬌娃則藉此改變形象為「蒼藍少女」並準備與Playmaker、Soulburner聯手。在血色牧羊人為了阿葵雅而擊敗妹妹魅影嬌娃後，他依序敗給左輪和萊德寧；之後，Playmaker與以「亞斯」訊號作為誘餌的Go鬼塚展開決鬥，並在最後取勝。

為了共同對抗萊德寧，左輪召集了Playmaker、Soulburner、蒼藍少女、魅影嬌娃。並在說服Soulburner後，左輪利用改寫的漢諾塔掃描整個網絡找出萊德寧等人隱匿的位置——鏡像LINK VRAINS。在哈爾戰敗給蒼藍少女後，萊德寧主動出擊打敗了亡靈，而溫迪則敗給了Soulburner。蒼藍少女主動找上鮑曼一戰，可惜吞敗，鮑曼獲得水與地兩個伊格尼斯的力量，而後萊德寧使出秘策，以仁為人質威脅翔一要和Playmaker決鬥，期間Playmaker出於感情無法認真，而後被翔一提醒過往的約定才出手擊敗之，但自身也因為心理衝擊過大暈倒。漢諾三騎士隨即解除鏡像LINK VRAINS的封鎖，重獲自由的Soulburner滿懷怒意與鮑曼展開決鬥，最後不敵鮑曼的骰子戰術吞敗，鮑曼取得火與風兩個伊格尼斯的力量，認定自身已完成最後的試煉，即將與Playmaker一戰。萊德寧則收到左輪傳來的簡訊，前往應戰。
