'''許維恩'''（{{bd|1981年|11月11日|||H許}}），[[臺灣|臺灣]][[女藝人|女藝人]]、[[模特兒|模特兒]]、[[演員|演員]]，擁有[[荷蘭人|荷蘭人]]血統<ref>{{cite news|title=圖／許維恩混血姊絕美曝光！許路兒私照「正到被盜用」|url=http://star.ettoday.net/news/146309|work=娛樂星光雲|author=|publisher=東森新聞雲股份有限公司|accessdate=2015-11-15|language=zh-tw|date=2012-12-28 }}</ref>。
== 演藝生涯 ==
許維恩原為婚紗模特兒<ref name="恢單3個月">[https://tw.entertainment.appledaily.com/daily/20180214/37933285/ 許維恩恢單3個月 駁共買愛巢 分手KID「像家人」]，[[蘋果日報_(台灣)|蘋果日報 (台灣)]]，2018-02-14</ref>，2003年，她拍攝[[王菲|王菲]]的MV《旋木》出道，同時持續從事平面廣告拍攝和參與各式活動。2009年，許維恩開始參與戲劇演出，參演[[華視|華視]]的[[偶像劇|偶像劇]]《[[敲敲愛上你|敲敲愛上你]]》。至2012年6月22日，她跨足歌唱領域，發行首張個人EP《維恩的秘密花園》。2014年，許維恩再跨足電影界，演出電影《[[屍城|屍城]]》。2018年，許參演電影《[[角頭2：王者再起|角頭2：王者再起]]》，飾演[[王識賢|王識賢]]的情人暨紅粉知己；該片票房達新台幣1.27億元<ref name="TFI">{{cite web |title=全國電影票房2018年06/18-06/24統計資訊 |url=https://www.tfi.org.tw/Content/TFI/PublicInfo/全國電影票房2018年0618-0624統計資訊.pdf |website=[[國家電影中心|國家電影中心]] |date=2018-06-28 |access-date=2018-07-27 |archive-url=https://web.archive.org/web/20180628125042/https://www.tfi.org.tw/Content/TFI/PublicInfo/%E5%85%A8%E5%9C%8B%E9%9B%BB%E5%BD%B1%E7%A5%A8%E6%88%BF2018%E5%B9%B40618-0624%E7%B5%B1%E8%A8%88%E8%B3%87%E8%A8%8A.pdf |archive-date=2018-06-28 |dead-url=yes }}</ref>。同年10月，她加入經紀公司「鼎盛文化國際媒體股份有限公司」<ref name="復合KID"/>。

== 個人生活 ==
許維恩的前男友包括已故作詞人[[楊明學|楊明學]]。2009年，許維恩與前知名團體[[B.A.D|B.A.D]]成員[[田恩沛|田恩沛]]結婚，2011年傳出兩人關係出現危機；2012年，媒體報導許維恩和[[林柏昇|KID]]兩人一同過夜，許和田恩沛於同年11月離婚<ref name="恢單3個月"/>。自2013年起，許維恩和KID陸續傳出彼此有好感、正在交往的消息，媒體多次報導兩人私下出遊。2016年4月13日，兩人公開戀情<ref>[https://star.ettoday.net/news/679525 許維恩終於認愛KID！ 「我會小心翼翼」超甜告白]</ref>。直至2018年2月13日，兩人遭爆料已於2017年11月分手，四年情暫停。外傳兩人因對於女方在中國大陸的事業發展意見不一而分開，KID受訪表示兩人未分手，許維恩則表示雙方已協議分手，但否認外傳的相關細節<ref name="恢單3個月"/>。

2018年2月22日，許維恩與KID同遊[[泰國|泰國]]被爆料，KID對此回應表示：「我們就互相在努力呀，講過了嘛，就一直在做這件事。」另表示這次旅行還有很多朋友參與，「我們都在努力，我們只是很想低調」。許維恩的經紀公司回應：「請給他們空間，公司不明她行蹤也不便干擾。」 <ref>[https://tw.entertainment.appledaily.com/daily/20180223/37940038/ KID 泰國強抱照外洩咒攝郎 10天回收許維恩「莊孝維」]</ref>。
*同年3月28日，新聞報導爆出許與[[馬來西亞|馬來西亞]]籍師弟呂銳傳出熱戀<ref>[https://tw.entertainment.appledaily.com/realtime/20180328/1323138 一張木桌出賣了她 許維恩被爆祕戀大馬男星]</ref>。因經紀公司將呂打造成彷似知名男模藝人[[高以翔|高以翔]]的形象，號稱「大馬高以翔」。但大部份網友指出和KID神似，亦被稱為「大馬KID」。隔日雙方皆否認戀情，澄清彼此只是生意上的夥伴，呂銳亦強調女方是自己在台灣的「恩人」<ref>[https://news.tvbs.com.tw/entertainment/892185/ 都看見了！遭爆無縫接軌　許維恩無奈澄清遭打臉]，[[TVBS|TVBS]]新聞，2018-03-29</ref><ref>[https://star.ettoday.net/news/1139943/ 呂銳認了！PO文嘆「請給我們空間」　許維恩回應有漏洞]，[[ETtoday新聞雲|ETtoday新聞雲]]，2018-03-29</ref>。

*同年4月11日，媒體報導許維恩與KID被目擊於當月10、11日同遊[[墾丁|墾丁]]，且穿情侶裝在當地夜宿一家別墅，有遊客表示看到兩人互動親密，如同一般情侶，指KID會幫女方提重物並夾菜，許維恩如「女王一樣」。據悉此次出遊為女方主動邀約，爆料者認為兩人感情事件的一連串發展似有[[炒作|炒作]]嫌疑。稍晚許維恩在[[臉書|臉書]]證實兩人確實在墾丁，但是為了品牌工作拍照而前往，並非甜蜜出遊度假，雙方經紀人亦強調兩人純為工作同行<ref>[https://star.ettoday.net/news/1148133/ 獨／許維恩KID同遊墾丁ING　穿情侶裝「親密過夜」]，[[ETtoday新聞雲|ETtoday新聞雲]]，2018-04-11</ref><ref>[https://news.tvbs.com.tw/entertainment/899355/ 度量真大！許維恩偕KID墾丁過夜　談劈腿：一時糊塗]，[[TVBS|TVBS]]新聞，2018-04-11</ref><ref>[http://www.chinatimes.com/realtimenews/20180411002710-260404/ 無視劈腿開房醜聞 許維恩回收KID墾丁蜜遊過夜]，[[中時電子報|中時電子報]]，2018-04-11</ref><ref>[https://tw.appledaily.com/new/realtime/20180411/1332506/ 演哪齣？許維恩KID墾丁曬情侶裝　真相是…]，[[蘋果日報_(台灣)|蘋果日報 (台灣)]]，2018-04-11</ref><ref>[https://stars.udn.com/star/story/10091/3081169/ 被爆料當過KID小三 奶精鬧家庭革命]，[[聯合新聞網|聯合新聞網]]，2018-04-11</ref>。

*至同年10月29日，許維恩受訪表示自己是單身，與KID只是朋友關係，對於兩人未來的發展，她回應：「順其自然。」<ref name="復合KID"/>。

除了演藝事業以外，許維恩亦擁有個人副業投資。許維恩於網路販售品牌面膜，並於2017年初開設美甲店<ref name="恢單3個月"/>。
