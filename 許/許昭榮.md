'''許昭榮''' (Khoo, Chau Ing)（{{bd|1928年|11月13日|2008年|5月20日|catIdx=Xu許}}），[[台灣|台灣]][[屏東縣|屏東縣]]人，前[[台籍日本兵|台籍日本兵]]及[[中華民國海軍|中華民國海軍]]的退伍軍人。[[台灣前國軍退役軍人暨遺族協會|台灣前國軍退役軍人暨遺族協會]]前任會長。[[台灣獨立運動|台灣獨立運動]]參與者，生前不斷為[[第二次世界大戰|第二次世界大戰]]和[[國共內戰|國共內戰]]的[[台籍日本兵|台籍日本兵]]、[[台籍國軍|台籍國軍]]、[[台籍共軍|台籍共軍]]的權益奔走，2008年5月20日因抗議[[中華民國政府|中華民國政府]]對台籍老兵暨遺族不聞不問而[[自焚|自焚]]身亡。

== 早年生平 ==
許昭榮出生於[[昭和|昭和]]3年（1928年）[[日治台灣|日治時期]]的屏東枋寮鄉水底寮，1940年自枋寮公學校畢業。

== 日本及中華民國海軍 ==
在[[第二次世界大戰|第二次世界大戰]]，日本政府實施「志願兵」制度，號召台灣人至海外作戰。許昭榮曾經接受[[日本海軍|日本海軍]]術科訓練，昭和18年（1943年）參加日本海軍特別自願役第二期，成為[[台籍日本兵|台籍日本兵]]。昭和19年自日本海軍飛行機整備預備學生第二期畢業(「海軍第二期飛行機整備生」(即地勤人員練習生)；一般由飛機航空工廠代訓、比如日治時期台灣岡山的「第61空廠」，沒有專門學校；其它類似的專門學校有日本[[舞鶴市|舞鶴市]]的「海軍機關學校」)。1947年[[二二八事件|二二八事件]]，收到不認識的[[吳振武|吳振武]]來信，以其有海軍術科專長，被[[國民政府|國民政府]]徵召投入海軍「台灣技術員兵」<ref name="許昭榮">{{cite book |author=許昭榮|chapter=〈接收台灣的國民黨軍〉|title=《臺籍老兵的血淚恨》|date=1995-01-01|publisher=前衛出版社|location=台灣|language=zh-tw |isbn=9578010052}}</ref>。遣往[[上海|上海]]、[[青島|青島]]等地，從事戰後接收日本卅四艘賠償艦的修復工作，目睹戰爭的殘酷與無情，後隨國民政府敗退返台。1949年被派至[[美國|美國]]接收「太湖號」護航驅逐艦。

== 因[[海軍台灣獨立案|海軍台灣獨立案]]入獄並成為政治難民 ==

1955年許昭榮再度被派赴美接收「咸陽號」[[驅逐艦|驅逐艦]]。期間許昭榮看到《[[紐約時報|紐約時報]]》報導[[台灣共和國臨時政府|台灣共和國臨時議會]]在[[東京|東京]]成立的消息，因此形成了許的台獨意識，決定將[[台灣獨立運動|台灣獨立運動]]的消息傳入台灣島內。許透過其在東京開設餐廳的叔叔許仕寄來的由[[簡文介|簡文介]]（台灣共和國臨時政府秘書長）主編的《台灣獨立運動第十年》一書。許昭榮認為必須將此書帶回來讓台灣人傳閱，因此傳閱給當時的民主人士如[[郭國基|郭國基]]、[[李源棧|李源棧]]，以及軍隊中的知心同袍看。因此遭當局逮捕，在海軍招待所關了一年後，被左營軍法處依叛亂條例二條三判刑，送往[[綠島|綠島]]服刑十年，期間遭逢妻子改嫁。1967年移送泰源監獄。

在獄期間，[[童尚經|童尚經]]寄日文版《世界民間故事》請許昭榮翻譯，並在《新生兒童》每週發表一篇，持續一年，還讓許昭榮把全稿售予[[水牛出版社|水牛出版社]]<ref name="李禎祥">{{cite journal |author=李禎祥|title=〈稿費資助政治犯 童常主編被槍決〉|journal=《[[新台灣新聞週刊|新台灣新聞週刊]]》|issue=第614期|date=2007-12-26|publisher=本土文化事業有限公司|location=台灣|language=zh-tw}}</ref>。

1968年出獄，進入日商「信和」公司任職。1970年以將產品產地印為「MADE IN REPUBLIC OF TAIWAN」（台灣共和國製造）而遭密告涉嫌叛亂，再度遭[[警備總部|警備總部]]逮捕。幸救援得當，而於1972年獲不起訴處分。1971年許昭榮創立「台暉貿易公司」，仍被警總禁足出國，直到1981年才赴美開發台灣[[草蝦|草蝦]]外銷市場，參加南加州台灣人社團聲援[[施明德|施明德]]獄中絕食，訴求政府「釋放政治犯」遊行示威，一夕之間淪為政治難民，流亡海外。1986年蒙各方援助，才獲得[[加拿大政府|加拿大政府]]的政治庇護。

== 組織台籍老兵 ==
1987年許昭榮從[[加拿大|加拿大]]回台，一一將台籍老兵找回來。，在許昭榮的精神感召下，老兵們從各地聚集到[[旗津區|旗津]]海岸，在這裡凝聚起老兵的精神、感受當年被莫名送到外地打仗的悲哀。許昭榮認為有一萬五千名台灣子弟兵被國民政府遣往中國大陸剿匪，其中至少有一萬人被押往東北[[錦州|錦州]]、[[塔山|塔山]]、華北[[魚台|魚台]]、[[濟寧|濟寧]]及[[徐蚌會戰|徐蚌會戰]]等戰役，成了「無名戰士」。1949年隨國民黨政府平安撤退返台的只有四百多人，其中大多數是海軍人員。1992年解除[[黑名單|黑名單]]後，返台投入為原台籍老兵暨遺族討回公道的工作，1994年創設[[全國原國軍台籍老兵暨遺族協會|全國原國軍台籍老兵暨遺族協會]]。

2003年許昭榮等人為[[蘇素霞|蘇素霞]]舉行公祭<ref name="自由時報1">{{Cite news|url=http://old.ltn.com.tw/2003/new/sep/1/today-o6.htm|title=回憶綠島百合蘇素霞（上篇）|language=zh-tw|publisher=《自由時報》|author=[[許昭榮|許昭榮]]|date=2003-08-31|accessdate=2016-01-12}}</ref><ref name="自由時報2">{{Cite news|url=http://old.ltn.com.tw/2003/new/sep/1/today-o6.htm|title=回憶綠島百合蘇素霞（下篇）|language=zh-tw|publisher=《自由時報》|author=[[許昭榮|許昭榮]]|date=2003-09-01|accessdate=2016-01-12}}</ref>。

== 台灣無名戰士紀念碑 ==
2005年11月10日[[高雄市政府|高雄市政府]]於[[旗津風車公園|旗津風車公園]]旁為[[台灣無名戰士紀念碑|台灣無名戰士紀念碑]]揭幕。2005年12月13日[[行政院文化建設委員會|行政院文化建設委員會]]與高雄市政府在旗津興建的「戰爭與和平紀念公園」正式落成啟用。2008年3月13日，[[高雄市議會|高雄市議會]]決議，戰爭與和平紀念公園更名為「[[和平紀念公園|和平紀念公園]]」。

== 自焚死亡 ==
2008年5月20日晚間6點47分，許昭榮於台灣無名戰士紀念碑前，在[[汽車|汽車]]內澆[[汽油|汽油]]引火[[自焚|自焚]][[死亡|死亡]]。現場留下遺書指責，[[國民黨|國民黨]]與[[民進黨|民進黨]]漠視「台灣歷代戰歿英靈」。遺書中寫著：「我依據自己的意-{志}-，以死抗議台灣執政者長期對『歷代軍人軍屬台籍老兵』之精神虐待。國不像國，政府不像政府；議會亂武，司法亂彈；自由民主脫線；愚兵一世人！」對現行退輔制度偏袒「老芋仔」剝削「蕃薯囝」，表示不滿；國民黨、民進黨執政期間，不但未給予「台灣歷代戰歿英靈」歷史定位，且未曾舉辦國家級的追思或弔祭，讓約4萬位台灣先靈在海外流浪60餘年。遺書最後強調，政府[[重文輕武|重文輕武]]、欺辱軍人，他甘願死守台灣唯一的「戰爭與和平紀念公園」，直到催生國立「台灣歷代戰歿英靈紀念碑」為止。

許昭榮生前不斷為二次大戰台籍老兵的權益奔走。他認為：國民黨、民進黨執政期間，均未重視台籍老兵；設立在旗津風車公園旁的「台灣無名戰士紀念碑」落成時，當時的總統[[陳水扁|陳水扁]]婉拒題名落款，當時的高雄市長[[謝長廷|謝長廷]]也未到場參加落成典禮。許昭榮遺書中表示，他一再訴求應該為台籍老兵設碑，如今好不容易在旗津戰爭與和平紀念公園設立台灣無名戰士紀念碑，政府竟然要遷走台灣無名戰士紀念碑，他因此用這種激烈手段抗議。

2013年1月11日，[[黨外運動|黨外運動]]人士[[陳婉真|陳婉真]]說，她到各地旅遊都會看到許多戰爭遺跡，只有台灣沒有；許昭榮希望陳水扁為台灣無名戰士紀念碑寫碑文，陳水扁卻拒絕；諷刺的是，當初戰爭與和平紀念公園的土地是國民黨籍高雄市長[[吳敦義|吳敦義]]任內撥用的，民進黨籍高雄市長都在迴避這個議題<ref>{{cite news|author=林朝億|publisher=[[新頭殼|新頭殼]]|url=https://newtalk.tw/news/view/2013-01-12/32791|title=街頭闖將陳婉真 投入二戰離亂十年記載|date=2013-01-12|accessdate=2018-12-31|quote=1980年代末、90年代最衝，甚至準備汽油彈要跟台中高分院對抗的黨外女將陳婉真，……她昨（11）日受訪……陳婉真說，當她到各地旅遊時都會看到許多戰爭時留下來的遺跡，只有台灣沒有。高雄戰爭與和平紀念公園當時許昭榮希望總統陳水扁寫碑文時，他還拒絕。但令人覺得諷刺的是，當初這塊土地竟然還是吳敦義當市長時撥用的，民進黨的市長都在迴避這個議題。}}</ref>。

== 紀念碑和紀念館舍落成 ==
位於旗津的高雄市戰爭與和平紀念公園內有「台灣無名戰士紀念碑」和「二次大戰戰俘船紀念碑」之碑文，以及一座「飛鄉」紀念碑，該紀念碑柱腳有三座碑體，分別是「臺灣歷代戰歿將士英靈紀念碑」、「國共內戰殞身原日本軍、前國軍台灣無名戰士紀念碑」、「二次大戰戰俘船紀念碑」。2009年5月，公園內的[[高雄市戰爭與和平紀念公園主題館|高雄市戰爭與和平紀念公園主題館]]落成，該館為臺灣唯一同時紀念台籍日本兵、台籍國軍、台籍共軍之館舍，並兼及日本高座少年工、看護婦、慰安婦、戰俘船等，館內展示各種文物，並舉辦各類相關活動。
