'''許-{}-厥恆'''{{BD|1967年|10月3日|catIdx=C}}，業界慣以 '''小恆''' 為稱呼。祖籍 [[浙江|浙江]][[定海|定海]]。出生於[[台灣|台灣]][[台北|台北]]。台灣的[[音樂家|音樂家]]、[[教師|教師]]、[[唱片製作人|唱片製作人]]、[[鼓手|鼓手]]、[[視覺藝術|視覺藝術]] 創作者。

 
曾任：「伍士塔克流行音樂中心( Woodstock Pop Music Center)」負責人。「WOBU 全國創作樂團聯盟」執行長。「[[刺客樂團|刺客樂團]]」鼓手。「Woodstock 全國熱門音樂大賽」主辦人。「SEEDER 樂團」團長兼鼓手。[[爵士鼓|爵士鼓]] 教師、Clinic 講師、音樂比賽評審。 許厥恆 於台灣音樂圈培育為數不少的 職業音樂人、演藝人員、表演團體、以及 職業鼓手。

== 生平 ==

=== 早年經歷 ( 1967 - 1985 ) ===
 
* 1967/10/03 - 生於台灣台北。喜好繪畫。 
* 1982 (15歲) - 接觸西洋音樂，偏好 [[搖滾樂|搖滾樂]]。
* 1983 (16歲) -「爵士鼓」啟蒙於「台北/東南工專」學長 吳明輝。
* 1983 (16歲) - 於台北「特磯音樂」，師從 江天賦、「[[紅十字|紅十字]]」樂團 ([[趙傳|趙傳]]) 羅明立(已故)。
* 1983 (16歲) - 於校園自組樂團，擔任團長兼鼓手。「熱門音樂社」代理社長。
* 1984 (17歲) - 擔任「特磯音樂」爵士鼓教師。師從「憲光義工隊」陳威中。 
* 1984 (17歲) - 與 張文森 (日後為發片歌手、音樂製作人)、楊致綱、張惠民、游正偉 (日後為「中影」導演)、傅希信 ... 等人，組織第一個職業搖滾樂團「Hairy Mary」。 
* 1985 (18歲) - 入伍「中華民國海軍」。獲「軍區指揮部」繪畫比賽「首獎」。

=== 職業生涯 1( 1988 - 1996 ) ===

* 1988 (21歲) - 進入「朱子廣告」擔任美術設計。向鼓界名師 卜致平 進修學習爵士鼓。  
* 1989 (22歲) - 擔任「非凡廣告」([[盧昌明|盧昌明]]，知名廣告人，音樂製作人，已故。代表作品為：司迪麥口香糖、開喜烏龍茶) CF 腳本繪師。
* 1989 (22歲) - 自修 MIDI。任職台北「敦煌樂器」爵士鼓教師。
* 1990 (23歲) - 擔任 「[[中央電影公司|中央電影公司]]」專屬配樂。 
* 1990 (23歲) - 擔任 「[[公共電視|公共電視]]」專屬配樂。「[[華視|華視]]新聞」配樂。
* 1990 (23歲) - 參加「台北市政府」優良詞曲創作比賽，獲「最佳作詞」獎。
* 1990 (23歲) - 擔任「水晶唱片」發片歌手 [[周志華|周志華]] 之巡迴演唱 鼓手，開始 Session Player 生涯。
* 1990 (23歲) - 自組 PARADIDDLE 商業表演樂團。主唱為 曹琳 ( 演藝人員 [[曹蘭|曹蘭]] 之妹 )
* 1991 (24歲) - 與音樂人 高林煒 合組「派拉蒂杜」音樂製作公司，代表作品為「中華民國建國80周年」。
* 1991 (24歲) - 與 葉慶煌 等人，擔任「客家」歌手 蔡孟甫 專輯之 [[編曲|編曲]] 與 [[製作|製作]] (「[[漢興|漢興]] 唱片」發行 )，並進行巡迴演唱。
* 1991 (24歲) - 擔任台北「海國樂器」爵士鼓教師。
* 1992 (25歲) - 與 [[潘學觀|潘學觀]] (吉他教材「瘋狂電吉他」等書作者) 合組搖滾樂團 [[Rufusion|Rufusion]]。
* 1992 (25歲) - 師從「視覺藝術家」 劉勝雄，學習 [[油畫|油畫]]。
* 1993 (26歲) - 油畫作品參與台北「七海藝廊」開幕聯展。
* 1994 (27歲) - 油畫作品參與台北「美屋坊藝術中心」開幕聯展。
* 1994 (27歲) - 許厥恆 /油畫作品個展，於「七海藝廊」(台北阿波羅大廈)。
* 1995 (28歲) - Rufusion 搖滾樂團，改組為「融合爵士」樂團，正式跨入「[[爵士|爵士]]」音樂領域。
  
【註】：Rufusion 歷代團員有：林頌閔 (團長)、潘學觀、江力平、[[呂聖斐|呂聖斐]] ( 第45屆金馬獎最佳電影原創音樂「[[海角七號|海角七號]]」)、陳文祥、陳任祐、李欣怡、游天仁、陳玉( 華納唱片 )、鄒炯如 、翁百璋...等。 
  
* 1995 (28歲) - 擔任商業表演團體「Relax」樂團鼓手，主唱為：[[張惠妹|張惠妹]] ( 未發片前。本時期為「[[寶麗金|寶麗金]]」簽約歌手；後轉簽約「[[豐華唱片|豐華唱片]]」，才發行首張專輯：「[[姊妹|姊妹]]」)。
* 1995 (28歲) - 拜師「台灣鼓王」[[黃瑞豐|黃瑞豐]] 門下。
* 1996 (29歲) - 擔任搖滾團體「[[刺客樂團|刺客樂團]]」鼓手。

== 伍士塔克流行音樂中心 ( 1996 - 2003 ) ==
* 1996 - 於台灣台北板橋市，創立「伍士塔克流行音樂中心」(Woodstock Pop Music Center)，初期為「音樂教學」與「錄音練團室」租賃。

== 種子樂團 ( 1996 - 2003 ) ==
* 1996/11/22 - 自組商業表演團體「SEEDER」( 種子樂團 )，擔任「團長」兼鼓手。
  
 
【註】'''歷任主唱''' ：[[信|信]] ( [[蘇見信|蘇見信]]，「信樂團」主唱 )、張競業 ( 發片歌手 )、[[姚可傑|姚可傑]] ( 發片樂團「[[東方快車|東方快車]]」主唱 )、[[唐林|唐林]] ( 演藝人員 )、[[阿飛|阿飛]] ( 台灣原住民，[[拉卡·巫茂|拉卡·巫茂]]，「金曲獎」最佳原住民流行音樂專輯演唱新人獎、「金馬獎」獲獎電影「賽德克巴萊」電影音樂歌手 )、Joyce ( 台北「EZ-5」音樂餐廳之著名歌手 )、[[陳葳|陳葳]] ( 發片歌手 )、[[汪佩蓉|汪佩蓉]] ( 發片歌手 )、Milly、Kay、[[鐘樺|鐘樺]] ( 「山頂黑狗兄」原唱，發片歌手 )、戎祥 ( 演藝人員。已故 )。

== 演藝經紀與活動公關事業 ( 1997 - 2003 ) ==
* 1997 - 「伍士塔克流行音樂中心」二代店，遷址至台北縣永和市永貞路，增設「樂器事業部」。並於互聯網架設網站 (台灣樂器圈之首例)
* 1997 - 舉辦第零屆 (試辦) 「Woodstock 全國熱門音樂大賽」。
* 1997 - 在互聯網發表「許厥恆 /線上爵士鼓教學」共連載百餘篇，成為「華語地區」最早期之互聯網「爵士鼓教材」，至今刊登於中國大陸各大音樂討論區。
* 1998 - 「伍士塔克」新設「娛樂事業處」，下設「活動公關部」與「演藝經紀部」。
* 1998 - 擔任「無限延伸整合行銷傳播公司」音樂總監。
* 1998 - 舉辦第一屆 (正式) 「Woodstock 全國熱門音樂大賽」。

【註】：第三名得獎樂團 Bee3，由「[[華納唱片|華納唱片]]」( 美國 Warner Bros. Records 台灣分公司 ) 簽約發行，改名「六甲樂團」，後再改名為「[[化學猴子|化學猴子]]」 。 
 
* 1999 - 成立「WOBU 全國創作樂團聯盟」(Woodstock Original Bands Union) 推動台灣翻唱樂團進行原創之風氣。( 截至 2000 年 3 月，共計創作樂團 88 組 )
* 1999 - 於台北「[[京華城|京華城]]」舉辦「FLY-WOODS 全國熱門音樂大賽」初賽，參賽團體 50 餘組。連續演唱 10 小時。
* 1999 - 於國際連鎖餐廳「[[Hard_Rock_Cafe|Hard Rock Cafe]]」(台北) 舉辦「FLY-WOODS 全國熱門音樂大賽」媒體記者會 及 總決賽。
** 首次介紹 [[視覺系|視覺系]]、[[黑死系|黑死系]] ...等新式音樂風格與造型，供台灣電子媒體報導。

【註】：總冠軍「潑猴」樂團 ，由 [[華納唱片|華納唱片]] 簽約與發行。

* 2000 - 籌辦台灣屏東 / 墾丁地區之官方「[[千禧年|千禧年]]」( 公元 2,000 年 ) 跨年活動。並帶領「WOBU 全國創作樂團聯盟」旗下樂團，進行 8 小時跨年表演，現場匯聚約 2 萬民眾。
* 2001 - 帶領「WOBU 全國創作樂團聯盟」旗下樂團 20 組，前往桃園「[[台茂購物中心|台茂購物中心]]」，進行每周兩天之駐唱，共計半年。亦協助旗下樂團於台灣諸多商業表演場地登台獻唱。
* 2001 - 與「屏東縣政府」於[[墾丁|墾丁]]辦理「海嘯之歌」音樂祭，連續演出三天兩夜。樂團 50 餘組。為台灣音樂史上第一個「官方版」的樂團音樂祭。
【註】：本活動造成「[[台視|台視]]」將新聞「主播台」架設到活動現場，成為全程在戶外播報新聞的首例，並持續進行三天。諸多電子媒體亦以 SNG 車全程現場報導。「[[民視|民視]]新聞」則進行長達一週的深度報導。<ref>[http://memory.ncl.edu.tw/tm_cgi/hypage.cgi?HYPAGE=toolbox_vnews_detail.hpg&subject_name=%20&subject_url=%20&project_id=news&dtd_id=8&xml_id=0000044031 台灣記憶‧台灣新聞影音資料 ]</ref>
  
* 2003 - 截至本年，「伍士塔克流行音樂中心」承辦近百場工商活動，成為台灣「演藝事業型-活動公關」的先驅，包括如：[[廣達電腦|廣達電腦]]、[[技嘉科技|技嘉科技]]、[[華碩電腦|華碩電腦]]、[[可口可樂|可口可樂]]、[[家樂福|家樂福]]、[[和平醫院|和平醫院]]、[[京華城|京華城]]、[[中泰賓館|中泰賓館]]、[http://www.switer.com.tw/ 斯威特連鎖美容]、[http://en.wikipedia.org/wiki/T.G.I._Friday%27s T.G.I. Friday's]、[[台茂購物中心|台茂購物中心]]、[[大江國際購物中心|大江國際購物中心]]、[[福特汽車|福特汽車]] 等。 
  
*'''許厥恆''' 合作過的藝人，除 '''SEEDER''' 內部歌星外 ( 參見  '''SEEDER''' 的歷史 <ref>[http://seeder.50megs.com/history96_99.html SEEDER 的歷史 (1996-1999)]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，<ref>[http://seeder.50megs.com/history_2000.html SEEDER 的歷史 (2000-2003)]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> )，另有：[[孫燕姿|孫燕姿]]、[[順子|順子]]、[[張信哲|張信哲]]、[[高凌風|高凌風]]、[[張惠妹|張惠妹]]、[[施孝榮|施孝榮]]、[[泰迪羅賓|泰迪羅賓]](香港)、[[康康|康康]]、[[謝金燕|謝金燕]] ... 等數十位。
