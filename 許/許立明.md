'''許立明'''（{{bd|1969年|1月17日||}}），[[中華民國|中華民國]][[政治|政治]]人物，出生於[[臺灣|臺灣]][[雲林縣|雲林縣]][[麥寮鄉|麥寮鄉]]，曾任[[高雄市市長|高雄市代理市長]]。

==個人背景==

===家庭===
姐姐[[許淑芬|許淑芬]]，現任[[臺南市政府|臺南市政府]]新聞及國際關係處處長，曾任民進黨中國事務部副主任，及前行政院長謝長廷秘書等職務<ref name ="專訪">{{cite news|author=金名|url=https://newtalk.tw/news/view/2018-04-10/120409|title=人物》愛貓男人掌市政 許立明看守高雄恰如其分|publisher=新頭殼|date=2018-04-10|}}</ref>。

==早期經歷==
從大學時期就參與黨政活動，並歷任行政院新聞局長室秘書、中華電信公司董事長賀陳旦的特別助理、高雄市政府新聞處處長、行政院勞委會主委辦公室主任、新聞局局長室秘書以及[[高雄市政府|高雄市政府]]研究發展考核委員會主任委員、副市長等職務。

許立明與[[陳菊|陳菊]]結緣長達20年，曾出任陳菊勞委會主委辦公室主任。也因處事圓融，調和鼎鼐功力一流，經常負起市府跨局處協調溝通的角色，被稱為“市政大管家”。

因長期擔任幕僚的細密心思，及與陳菊多年培養的默契，許立明也深獲陳菊的信任。與前後任高雄副市長[[劉世芳|劉世芳]]、[[史哲|史哲]]等人合作無間<ref name ="專訪"></ref>。

==高雄代理市長任內==
2018年4月14日，由於原市長[[陳菊|陳菊]]將在同年4月23日升任[[總統府秘書長|總統府秘書長]]，行政院核定由許立明接任故接替[[陳菊|陳菊]]擔任高雄市代理市長，任期至[[2018年中華民國直轄市長及縣市長選舉|2018年中華民國直轄市長及縣市長選舉]]選出新一任高雄市長為止。當日受訪時他表示，希望市民支持他，支持他不是指挺他的個人榮辱，而是剩下的7、8個月，還是市長陳菊的任期，他希望市長率領的團隊歷經12年的辛苦打拚後，能畫下完美的句點。而陳菊也力挺許立明，認為他對高雄市政非常熟悉，也了解有多少建設必須在年底如期如質完工<ref >{{cite news|author=蔡孟妤|url=https://video.udn.com/news/862809|title=接下高雄代理市長 許立明：盼能畫下完美句點|publisher=聯合影音網|date=2018-04-17|}}</ref> 。2018年4月23日，許立明正式接任高雄市代理市長，再次強調與陳菊市府無縫對軌<ref name ="市長">{{cite news|author=王榮祥|url=http://news.ltn.com.tw/news/politics/breakingnews/2404708|title=高市代市長許立明率團隊宣誓就任 強調無縫接軌|publisher=自由時報|date=2018-04-24|}}</ref>

===任內政策===
許立明在接任代理市長時，便宣誓任內延續前市長[[陳菊|陳菊]]政策<ref name ="市長"></ref>。就任後進行多項人士調動，他認為，新任首長都在高雄市服務多年，學歷經歷都很好，也熟悉團隊施政價值。並期許新團隊能無縫接軌，穩健推動市政，在任期內完成對市民的承諾<ref>{{cite news|author=鄭仰哲 |url=https://newtalk.tw/news/view/2018-04-21/121688|title=高雄人事大洗牌 許立明任代理市長、原農業局長蔡復進任副市長|publisher=新頭殼|date=2018-04-21|}}</ref>。

在他任內，多項大型公共工程如[[衛武營國家藝術文化中心|衛武營國家藝術文化中心]]、[[高雄鐵路地下化|高雄鐵路地下化]]、[[登山街60巷歷史場域|登山街60巷歷史場域]]等先後落成啟用。

===任內爭議===
====決策爭議====
'''停班停課爭議'''
2018年8月28日，受到低壓帶及西南氣流的影響，高雄市在凌晨暴雨不斷，高雄市政府在早上6點12分宣布停班停課，引起不少民眾抱怨太晚宣布。對此代理市長許立明當天主持災情會議時向市民致歉，表示由於雨量超過預期，雨勢來的極快，雖已立即實施淹水管制，但由於接近上班時間，才臨時宣布停班停課<ref>{{cite news|author=鄭仰哲|url=https://newtalk.tw/news/view/2018-08-28/137088|title=6點停班停課挨轟 許立明道歉：5點多才下雨|publisher=新頭殼|date=2018-08-28|}}</ref>。

====發言爭議====
'''稱「北漂青年」不到1萬人'''
2018年10月4日，國民黨籍高雄市議員[[陳美雅|陳美雅]]在市議會總質詢時就針對北漂青年議題，質詢時任高雄市代理市長許立明並問道「高雄北漂青年究竟有多少人？」對此，代理市長許立明回答說，他估計約在1萬人以內，所謂的30萬或50萬人，嚴重懷疑是造假資料，並進一步指出，根據勞動部資料，高雄市在過去11年來，勞保投保人數增加14萬人，顯示高雄市的常住人口及就業機會是增加的<ref>{{cite news|author=葛祐豪|url=http://news.ltn.com.tw/news/politics/breakingnews/2570321|title=高雄北漂青年30~50萬人？許立明：1萬以內|publisher=自由時報|date=2018-10-04|}}</ref>。
