'''許劭'''（{{bd|150年||195年|catIdx=X許}}），字'''子將'''，[[汝南郡|汝南]][[平輿縣|平輿]]（今屬河南）人。[[東漢|東漢]]末年著名人物評論家。
許劭“少峻名节，好人伦，多所赏识”，和堂兄[[許靖|許靖]]在年輕時，每個月都會舉辦[[月旦評|月旦評]]的人物品評會<ref>《后汉书》本传</ref>，影響力之強，甚至是受其評論就能成名的地步。但許劭與许靖常有爭執<ref>《[[三国志|三国志]]》卷三八《许靖传》载，靖“少与从弟劭俱知名，并有人伦臧否之称，而私情不胁。劭为郡功曹，排摈靖不得齿叙，以马磨自给”。又《[[太平御览|太平御览]]》卷四九六引[[曹丕|曹丕]]《典论》曰：“汝南许劭与族兄靖俱避地江东，保吴郡。争论于太守许贡座，至于手足相及。”</ref>。有人問荀靖与[[荀爽|荀爽]]孰贤？許劭說：“二人皆玉也，慈明外朗，叔慈内润。”<ref>《三国志》卷一0《荀彧传》裴松之注引皇甫谧《逸士传》</ref>[[刘繇|刘繇]]进军讨伐[[笮融|笮融]]，許劭告訴劉繇：“笮融出军，不顾名义者也。朱文明善推诚以信人，宜使密防之。”笮融果然杀死太守[[朱皓|朱皓]]。<ref>《三国志》卷四九《刘繇传》裴松之注引《献帝春秋》</ref>[[袁紹|袁紹]]、[[劉表|劉表]]等人皆接受過他們品評，[[曹操|曹操]]也在[[橋玄|橋玄]]的引薦之下，得到著名的“治世之能臣，亂世之奸雄”的評價<ref>裴松之注《三国志》引[[孙盛|孙盛]]《异同杂语》</ref>。

[[汝南|汝南]][[太守|太守]][[徐璆|徐璆]]曾聘其為[[功曹|功曹]]，之後許劭為了避中原戰亂，移居江南。195年<ref>《[[太平御览|太平御览]]》卷五五六引《豫章记》曰：许子将墓在郡南四里。昔子将以中国大乱，远来渡江，随刘繇而卒，藏于昌门里。于时'''汉兴平二年'''也。</ref>病逝於[[豫章|豫章]]郡，享年46歲。

== 軼事 ==
曹操没有得志顯名的時候，曾經置辦厚礼，很謙遜地求許劭為他品評一番。許劭原本看不起他，於是曹操威脅許劭，許劭唯有退讓。歷史記載，評語有兩個版本。《三國志》注引孫盛《異同雜语》作“子治世之能臣，乱世之奸雄”；《後漢書》則作“君清平之奸賊，乱世之英雄”。當時曹操聽後，大笑而去。後來[[羅貫中|羅貫中]]的《[[三國演義|三國演義]]》據此事改編：曹操去拜見許劭，問道：“我是甚麼人？”許劭不答。曹操再問，許劭才說：“你是治世的能臣，乱世的奸雄啊！”曹操聽到后相當開心。其中許劭形象與正史相似，有鑒察人的品行、才能的名聲。

== 子 ==
许混，许劭子。清醇有鉴识，[[魏明帝|魏明帝]]时为[[尚书|尚书]]，显名魏世。

== 评价 ==
*[[蒋济|蒋济]]：“许子将褒贬不平，以拔樊子昭而抑许文休。”（《三国志·庞统传》注引 蒋济《万机论》）
*[[诸葛亮|诸葛亮]]：“许子将长于明臧否，不可以养人物。”（《诸葛亮文集》）
*[[诸葛恪|诸葛恪]]：“自汉末以来，中国士大夫如许子将辈，所以更相谤讪，或至于祸，原其本起，非为大雠。惟坐克己不能尽如礼，而责人专以正义。”（《三国志·诸葛恪传》注引 诸葛恪《与丞相陆逊书》）
*[[汝南|汝南]]人：“平舆渊有二龙（指许劭与许虔）。”（《后汉书·卷六十八·郭符许列传第五十八》）
*[[山简|山简]]：“是以[[郭泰|郭泰]]、许劭之伦，明清议于草野；[[陈蕃|陈蕃]]、[[李固|李固]]之徒，守忠节于朝廷。”（《晋书·卷四十三·列传第十三》）
*[[葛洪|葛洪]]：“许子将之徒，以口舌取戒，争讼论议，门宗成雠，故汝南人士无复定价，而有月旦之评。魏武帝深亦疾之，欲取其首，尔乃奔波亡走，殆至屠灭。”（《抱朴子外篇．自敍》）
*[[范晔|范晔]]《[[后汉书|后汉书]]》：“符融鉴真，子将人伦。守节好耻，并不逡巡。”（《后汉书·卷六十八·郭符许列传第五十八》）
*[[司马光|司马光]]：“自三代既亡，风化之美，未有若东汉之盛者也。及孝和以降，贵戚擅权，嬖幸用事，赏罚无章，贿赂公行，贤愚浑肴，是非颠倒，可谓乱矣。然犹绵绵不至于亡者，上则有公卿、大夫袁安、杨震、李固、杜乔、陈蕃、李膺之徒面引廷争，用公义以扶其危，下则有布衣之士符融、郭泰、范滂、许邵之流，立私论以救其败。是以政治虽浊而风俗不衰，至有触冒斧钺，僵仆于前，而忠义奋发，继起于后，随踵就戮，视死如归。”（《资治通鉴·卷第六十八·汉纪六十》）
*[[王相|王相]]：“东汉名士许劭好评人，时人称之‘月旦评’。”
*[[郝经|郝经]]：“处士尽矣，汉已亡矣，劭犹硁硁以吻颊为能，揭揭焉为月旦评，奔走夺命，幸而获免，岂尚德之士哉。”（《郝氏续后汉书》）
