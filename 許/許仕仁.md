'''許仕仁'''（{{lang-en|Rafael Hui Si-yan}}，{{bd|1948年|2月8日|catIdx=X}}<ref>[http://www.hkexnews.hk/reports/prelist/CAIA-20101004-16.pdf 董事及高級管理人員]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>）是[[天主教徒|天主教徒]]，前[[全國政協常委|全國政協常委]]，[[香港特別行政區|香港特別行政區]]前[[香港特別行政區行政會議|行政會議]]非官守成員、前[[政務司司長|政務司司長]]，祖籍[[廣東|廣東]][[東莞|東莞]]。香港傳媒以暱稱「肥龍」稱之，在政府內部曾公認其為「橋王」。

他2000年6月離開政府，到[[強制性公積金|強制性公積金]]計劃管理局（積金局）擔任首位[[行政總監|行政總監]]，主司設計、策劃[[強制性公積金|強制性公積金]]。他亦因此獲稱為香港的「強積金之父」。

許仕仁在2005年重返官場、任[[政務司司長|政務司司長]]後，與商界藕斷絲連，最終捲入刑事指控，在2014年於香港高等法院被裁定公職人員行為失當、串謀向公職人員提供利益等5項罪名成立，判監禁7年半。

政府於2018年3月2日刊憲，宣佈撤除授予許仕仁的[[大紫荊勳章|大紫荊勳章]]和[[金紫荊星章|金紫荊星章]]，並同時褫奪[[太平紳士|太平紳士]]身份。

==政界生涯==


許仕仁2005年6月30日獲[[中華人民共和國中央人民政府|中華人民共和國中央人民政府]]委任為[[香港特別行政區|香港特別行政區]][[政務司司長|政務司司長]]。作為[[特區政府|特區政府]]問責班子的首席主要官員，他協助[[行政長官|行政長官]]管治香港，並就政策事務向行政長官提供意見。

===早年官職===
許仕仁1970年9月加入香港政府，任職助理教育主任。1971年8月獲聘任為政務主任，1996年晉升至布政司署司級政務官。在政府工作的早期，他曾任職多個決策科及部門，包括[[民政署|民政署]]、[[保安科|保安科]]、[[經濟科|經濟科]]及[[運輸科|運輸科]]。

許仕仁1977年至1979年間借調往[[廉政公署|廉政公署]]工作；其後在1982年至1983年被派往[[哈佛大學|哈佛大學]]受訓，並獲得公共行政碩士學位。他在1985年至1986年間出任前行政立法兩局[[非官守議員|非官守議員]]辦事處副秘書長；1986年至1990年出任副經濟司；1990年至1991年出任副工務司，繼而獲委任為[[新機場工程統籌署署長|新機場工程統籌署署長]]。

1992年11月，許仕仁出任[[運輸署署長|運輸署署長]]，直至1995年獲委任為[[財經事務司|財經事務司]]（1997年7月1日回歸後稱[[財經事務局|財經事務局]][[局長|局長]]），攀上其[[公務員|公務員]]生涯的事業高峰，掌管香港財經事務長達五年，期間致力鞏固香港作為國際金融中心的地位。在1998年[[亞洲金融風暴|亞洲金融風暴]]期間，[[金融管理局|金融管理局]]總裁[[任志剛|任志剛]]聯同當時的[[財政司司長|財政司司長]][[曾蔭權|曾蔭權]]以及當時的財經事務局局長許仕仁取得共識，動用逾千億港元的[[外匯儲備|外匯儲備]]大戰炒家，入市購入多隻[[港股|港股]]，結果擊退炒家，其後成立[[盈富基金|盈富基金]]將港股出售，成為許仕仁政界生涯最大功績之一。

===官場外之重要職位===
他2000年6月離開政府，到[[強制性公積金|強制性公積金]]計劃管理局（積金局）擔任行政總監；同年12月成功推行強積金計劃，為全港就業人士提供[[退休|退休]]保障。他2003年6月離開積金局。2004年2月，他出任[[九龍巴士|九龍巴士]]控股有限公司及其附屬公司──九龍巴士(一九三三)有限公司及[[龍運|龍運]]巴士有限公司──之董事，至2005年5月辭去職務。

===出任政務司司長===
曾蔭權在2005年接任行政長官後，便向中央人民政府提名許仕仁出任政務司司長。許仕仁原本已經退休，但曾蔭權請他出任政務司司長。[[中華人民共和國國務院|中華人民共和國國務院]]於2005年6月30日宣佈任命，許仕仁即日下午2時在[[香港特別行政區政府總部|香港特別行政區政府總部]]宣誓就職。

在許仕仁上任開始，他對政界中人聲稱只任兩年政務司司長，兩年後（即2007年7月新一屆政府上場時）不會續任。儘管政界傳言他或會續任，但他仍然於2007年7月1日政府換屆時離職，但成為行政會議非官守成員。

==貪污被捕入獄==
{{main|新地風暴}}
2012年3月8日，香港廉政公署表示特區政府前政務司司長許仕仁涉嫌收受多筆款項及無抵押貸款，當日在東區裁判法院應訊時被加控一項公職人員行為失當罪名。<ref>[https://web.archive.org/web/20131004224031/http://www.apdnews.com/news/22722.html 港前政務司長被加控行為失當]，亚太日报，2013年3月11日</ref>同年3月29日，[[香港廉政公署|香港廉政公署]]邀請[[新鴻基地產|新鴻基地產]][[董事長|董事局聯席主席]][[郭炳江|郭炳江]]、[[郭炳聯|郭炳聯]]，與及前[[政務司司長|政務司司長]]許仕仁到廉署總部協助調查一宗貪污案，同日傍晚，廉政公署正式宣佈拘捕三人。<ref name="rthk">[http://rthk.hk/rthk/news/expressnews/news.htm?expressnews&20120329&55&829767 廉署拘捕郭氏兄弟及許仕仁涉嫌貪污行為失當] {{webarchive|url=https://web.archive.org/web/20140508133357/http://rthk.hk/rthk/news/expressnews/news.htm?expressnews&20120329&55&829767 |date=2014-05-08 }}，[[香港電台|香港電台]]，2012年3月29日</ref><ref>[http://rthk.hk/rthk/news/expressnews/news.htm?expressnews&20120329&55&829775 消息稱廉署循官商勾結方向調查郭氏兄弟案件] {{webarchive|url=https://web.archive.org/web/20140508133120/http://rthk.hk/rthk/news/expressnews/news.htm?expressnews&20120329&55&829775 |date=2014-05-08 }}，[[香港電台|香港電台]]，2012年3月29日</ref>他是港澳兩地特區政府成立以來，被拘捕的最高級政府官員，及首位被捕的大紫荊勳賢。

2013年11月27日，[[香港高等法院|香港高等法院]]以聆案官简易程序裁定向许仕仁颁发[[破产|破产]]令，此前受到贪污腐败罪起诉的许仕仁遭[[东亚银行|东亚银行]]讨债，香港媒体报道涉案金额至少达5000万港元的[[信用卡|信用卡]]欠款。除东亚银行外，另外四名债主分别是[[恒生银行|恒生银行]]、[[渣打银行|渣打银行]]、[[创兴银行|创兴银行]]和[[新鸿基地产|新鸿基地产]]旗下的忠诚财务公司<ref >{{cite web|url=http://www.bbc.co.uk/zhongwen/simp/china/2013/11/131127_hongkong_rafael_hui_bankrupt.shtml |language= |title=香港前政务司长许仕仁被法院颁令破产 |publisher=BBC中文网 |date=2013年11月27日 |accessdate= }}</ref>。

2014年5月8日，[[香港高等法院|香港高等法院]]开审许仕仁贪污腐败案，预计该案将审理70天，检方将传召包括多名港府前任和现任高官在内的82名证人。<ref >{{cite web |url=http://www.bbc.co.uk/zhongwen/simp/china/2014/05/140508_hongkong_corruption_trial.shtml | language= | title=香港高等法院开审许仕仁贪污腐败案 |publisher=BBC中文网 |date=2014年5月8日 |accessdate= }}</ref>

2014年9月25日，香港媒体报道，被称作香港史上规模最大的腐败审判之一许仕仁案，发生了出人意料的情节。许仕仁称，自己收取了来自北京的一笔多达1100万港元（约合870万元人民币）的秘密款项。<ref >{{cite web |url=http://cn.nytimes.com/china/20140925/c25hui |language= |title=香港前政务司司长承认接受来自北京的千万港元 |publisher=纽约时报中文网 |date=2014年9月25日 |accessdate= }}</ref>

2014年12月，許仕仁於香港高等法院被裁定公職人員行為失當、串謀向公職人員提供利益等5項罪名成立，判處許仕仁監禁7.5年並需交還1,118.2萬[[港元|港元]]賄款。<ref name=":0">{{Cite news|url=https://theinitium.com/article/20160216-dailynews-Rafael-Hui/|title=香港前政務司司長許仕仁貪污案上訴被駁回|last=|first=|date=2016-02-16|work=端传媒|accessdate=2017-02-17}}</ref>控罪指許仕仁在擔任積金局行政總裁、政務司司長及行政會議成員前後，隱瞞與新地的關係，收取新地提供的免費住所、免費辦公室、貸款及現金等利益。<ref name=":1">{{Cite news|url=http://hk.on.cc/hk/bkn/cnt/news/20160712/bkn-20160712090938002-0712_00822_001.html|title=終院受理上訴　炳江准保　暴瘦許仕仁續服刑|last=|first=|date=2016-07-12|work=東網|newspaper=|accessdate=2017-02-17|language=zh-hk}}</ref>審理此案的法官麥機智（Justice Andrew Macrae）表示，許仕仁涉及的案件非常嚴重，收取金錢會令他傾向於優待地產商，向對方洩露政府的資料和策略。<ref name=":0" />

許仕仁被下令即時收押在[[荔枝角收押所|荔枝角收押所]]。12月20日，許的妻子前往[[荔枝角收押所|荔枝角收押所]]探望他，許向妻子表示在收押所內感到怕冷，其後轉移至[[赤柱監獄|赤柱監獄]]。許妻表示會上訴。12月22日向法院求情，但其後上訴被駁回，繼續服刑。

2016年7月12日，許仕仁向[[香港終審法院|終審法院]]申請終極上訴許可，在終院上訴委員會開庭聆訊後，決定受理其中串謀公職人員行為失當罪的上訴申請，並排期2017年5月9日至10日由5名法官正式審理。<ref name=":1" />

2017年6月14日，許仕仁上訴被終審法院駁回，須繼續服刑。成為香港回歸以來首位入獄的前香港最高級官員。

2018年3月2日，許仕仁分別於於1998年和2007年獲頒授之[[金紫荊星章|金紫荊星章]]和[[大紫荊勳章|大紫荊勳章]]被[[香港特別行政區政府|政府]]刊登[[香港特別行政區政府憲報|憲報]]褫奪，其[[太平紳士|太平紳士]]委任亦被撤銷。<ref>{{cite news|title=許仕仁郭炳江慘被「搣柴」 港府刊憲撤回勳銜|url=https://hk.news.appledaily.com/local/realtime/article/20180302/57894221?_ga=2.106059395.1871829226.1519967822-1014019929.1519967821|accessdate=2018-03-02|publisher=[[蘋果日報_(香港)|蘋果日報]]|date=2018-03-02}}</ref>

== 欠債 ==
在2013年4月，許仕仁已被[[東亞銀行|東亞銀行]]入稟追討6,000萬元；到了8月，先後再有3間公司向許追討債項。先是新地旗下[[忠誠財務|忠誠財務]]入稟向許本人及其妻的公司追討316萬元；[[創興銀行|創興銀行]]其後亦入稟高院追討984.2萬元貸款及卡數；[[渣打銀行|渣打銀行]]亦入稟追討約120萬元個人戶口透支及信用卡欠款。2013年9月24日，許仕仁被[[東亞銀行|東亞銀行]]入稟高等法院申請破產。<ref>{{cite web|url=http://inews.mingpao.com/htm/Inews/20130924/gb21148k.htm|title=許仕仁被東亞入稟申請破產|date=2013年9月24日|publisher=''[[明報|明報]]''}}</ref> 2013年11月27日，許仕仁正式被高院頒令破產，成首名擁大紫荊勳章的破產人士。[[破產管理署|破產管理署]]將接管其財產和每月逾八萬元長俸，用來償還欠債，而他被控貪污的案件將在2014年5月開審，屆時有可能要由[[法律援助署|法律援助署]]聘請律師為他辯護。<ref>{{cite web|url=http://orientaldaily.on.cc/cnt/news/20131128/00176_007.html|title=許仕仁破產 納稅人代還債|date=2013年11月28日|publisher=''[[東方日報_(香港)|東方日報 (香港)]]''}}</ref>

===名下馬匹===
許仕仁是[[香港賽馬會|香港賽馬會]]會員（曾擔任香港賽馬會董事），自1995-1996馬季開始擁有名下馬匹，包括「有計」（與任志剛合夥）、「有聲有色」、「聲色妙韻」、「再接再厲」、「真財實料」（個人名下）、「大師」、「知己知彼」（與唐英年合夥）等。2011年末，許仕仁突然宣布拍賣名下馬匹「各適其適」，最後由紅駿團體（當時是鄧明與譚達明名下）以261萬港元投得。<ref>[http://racing.on.cc/racing/new/20130101/rjnewa0005x0.html 各適其適心急交代]</ref><ref>[http://orientaldaily.on.cc/cnt/news/20130325/00176_087.html 政情：易主有運行 許仕仁前愛駒又贏]</ref>

== 荣誉 ==
許仕仁1986年12月獲委任為[[太平紳士|太平紳士]]，1998年獲頒授[[金紫荊星章|金紫荊星章]]，2007年獲頒授[[大紫荊勳章|大紫荊勳章]]。但3項榮譽均已在2018年3月2日被褫奪。
