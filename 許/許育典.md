'''許育典'''（{{bd|1969年|10月20日|||許育典}}），現任[[中華民國|中華民國]]（[[臺灣|臺灣]]）[[臺南市|臺南市]]副市長，[[德國|德國]][[法學|法學]][[博士|博士]]，曾任[[國立成功大學|國立成功大學]][[法律|法律]]學系特聘[[教授|教授]]兼[[社會科學|社會科學]]院院長，並曾獲第47屆[[中華民國十大傑出青年|中華民國十大傑出青年]]<ref>{{Cite web |url = http://www.president.gov.tw/Default.aspx?tabid=131&itemid=15577|title = 總統接見第47屆十大傑出青年當選人 |author =  |date = 2009-11-24 |publisher = [[中華民國總統府|中華民國總統府]] |language = 中文 |accessdate =}}</ref><ref>{{Cite web |url = http://www.persons.org.tw/front/bin/ptlist.phtml?Category=25|title = 歷屆十大傑出青年當選人 |author =  |date =  |publisher = [[財團法人十大傑出青年基金會|財團法人十大傑出青年基金會]] |language = 中文 |accessdate =}}</ref>。許出身[[工人|工人]]家庭，本身為[[身心障礙者|身心障礙者]]，長期致力教育、文化、行政法研究。

== 早年 ==
「斷了左手掌，但是我比別人認真。」台南市副市長許育典出身於一個挑磚工人的家庭，六歲被火車撞到而失去左手<ref>{{Cite web|url = http://news.secr.ncku.edu.tw/files/13-1054-60070-1.php|title = 斷手少年 因慈悲而偉大 成大教授許育典 24日獲頒十大傑出青年|author = |date = |publisher = [[國立成功大學|國立成功大學]]|language = 中文|accessdate = |archiveurl = https://web.archive.org/web/20170305003733/http://news.secr.ncku.edu.tw/files/13-1054-60070-1.php|archivedate = 2017-03-05|deadurl = yes}}</ref><ref>{{Cite web |url = http://www.epochtimes.com/b5/9/11/24/n2732758.htm|title = 10大傑青許育典 睹母挑磚滲血發奮唸書|author = 陳虹瑾 |date = 2009-11-24 |publisher = [[大紀元|大紀元]] |language = 中文 |accessdate =}}</ref><ref>{{Cite web |url = http://news.ltn.com.tw/news/local/paper/353670|title = 〈南部〉斷手教授 許育典榮獲十傑青年|author = 洪瑞琴 |date = 2009-11-25 |publisher = [[自由時報|自由時報]] |language = 中文 |accessdate =}}</ref>，為了不讓挑磚的父母辛苦工作，他從小即奮力向上，靠著公費取得德國杜賓根大學法學博士，回國後即在成功大學法律學系任教。許副市長在大學任教期間，主要講授憲法、行政法、教育法、宗教法與文化法等課程，主要研究重點為憲法、人權、教育法、宗教法與文化法等議題，並長期深入中央與地方政府機關貢獻所學，擔任各種中央與地方政府機關的委員會，更曾當選中華民國第47屆[[中華民國十大傑出青年|十大傑出青年]]。作為一個身心障礙者且長期成長於弱勢家庭，許副市長深感國家教育制度對一個人的人格開展影響深遠，長期依賴著獎學金與公費在求學，所以擔任教職以來，即以建構一個不受身份與身體影響的國家教育制度為念，並立志以人的自我實現為核心建構教育法制，且因緣這個教育憲法學的思考與建構，實踐了許副市長基於部門憲法的脈絡，建立出教育、宗教與文化不同領域間的法體系，讓教育基本權、宗教自由、文化基本權可以依其個別的基本權功能建構，形成不同的基本權類型化思考，並進一步聯結到相關的行政法規，促使憲法上的教育基本權、宗教自由、文化基本權獲得最大的實踐。

「做事不難，只要重複練習就行」每天總會有不同的事情進來，那你怎麼樣把自己的時間保留給自己一定要去做的一些事情，把它變成是一種習慣一種平凡，許副市長覺得那是需要練習的。……德文有一句俗諺是「練習，會造就大師」未來在擔任台南市副市長期間，許副市長期盼以自身奮鬥的經歷，協助台南市邁向一個正向城市、大學城市、法治城市3大指標城市。更進一步來說，許副市長希望藉由從小在艱困中奮鬥所養成的正向思考特質，讓台南市成為一個正向城市，使公務人員和市民都有同理心；許副市長也會努力將台南市朝向大學城市邁進，並運用成功大學的研究能量，讓成功大學成為台南市的智庫，建構人格自由開展的環境，幫助偏鄉弱勢等；許副市長未來也將運用自身的法律專業，將台南市打造成法治城市，並協調公部門跨局處的橫縱向溝通，任何事都以人民為依歸。其實，許副市長長期以來所關心的對象其實就是人，許副市長的著作及文章處處都在談人的自我實現，許副市長的法學思想就是以人的自我實現為核心，怎麼樣讓一個人不因為他的出生背景，不因為他週遭的環境，不因為他是怎麼樣的一個條件，而能夠實現他自己，這是許副市長期待的一個社會。
