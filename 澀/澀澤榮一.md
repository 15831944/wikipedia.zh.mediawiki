'''澀澤榮一'''（{{jpn|j=渋沢栄一|kyu=澁澤 榮一|hg=しぶさわ えいいち|rm=Shibusawa Eiichi}}，[[天保_(日本)|天保]]11年二月十三日（1840年3月16日） - [[昭和|昭和]]6年（1931年）11月11日）是[[幕末|幕末]]─[[大正|大正]]初期活躍的[[日本|日本]][[武士|武士]]（[[幕臣|幕臣]]）、[[官僚|官僚]]、[[實業家|實業家]]。1867年出訪法國、比利時，參加當年在法國舉辦的萬國博覽會，1868年結束出訪回到日本。設立{{link-jp|第一國立銀行|第一銀行}}、[[東京證券交易所|東京證券交易所]]等多種企業和[[理化学研究所|理化学研究所]]。有'''日本[[資本主義|資本主義]]之父'''之稱。

== 家族 ==
[[File:Eiichi_Shibusawa_in_his_final_days.jpg|thumb]]
涩泽荣一与第一任妻子涩泽千代（1841年－1882年）育有两女一子，与第二任妻子涩泽兼子育有三子一女。
*长女：涩泽歌子（1863年－1932年）：后嫁给法学家{{link-ja|穗积陈重|穂積陳重}}。
*次女：涩泽琴子（1870年－1925年）：后嫁给{{link-ja|阪谷芳郎|阪谷芳郎}}（1906年至1908年任日本[[大藏大臣|大藏大臣]]）。
*长子：涩泽笃二（1872年－1942年）：曾任{{link-ja|涩泽仓库|澁澤倉庫}}会长，其子{{link-ja|涩泽敬三|渋沢敬三}}曾于1945年至1946年任日本大藏大臣。
*次子：涩泽武之助（1886年－1946年）：[[立飛企業|石川岛飞机制作所]]第二任社长。
*三子：涩泽正雄（1888年－1942年）：日本制铁副社长，石川岛飞机制作所初代社长。
*三女：涩泽爱子（1890年－?）：后嫁给银行家{{link-ja|明石照男|明石照男}}。
*四子：涩泽秀雄（1893年－1984年）：曾任[[東京寶塚劇場|東京寶塚劇場]]会长、[[東寶|東寶]]董事长。
*孫子: [[澀澤侑哉|澀澤侑哉]]（1998年－）：日本[[模特兒|模特兒]]，日美混血(爸爸為美國人，媽媽為日本人)，精通英語、日語。

== 諾貝爾獎提名 ==
根據[[諾貝爾獎|諾貝爾獎]]官方資料庫，澀澤榮一曾於1926年、1927年被提名為[[諾貝爾和平獎|諾貝爾和平獎]]候選人<ref>[http://www.nobelprize.org/nomination/peace/nomination.php?action=show&showid=1506 Nomination Database - Peace]</ref>。

==紀念==
