'''澁江政光'''（1574年—1614年12月26日）是安土桃山時代至江戶時代前期的武將。父親是荒川秀景。

== 生平 ==
在天正2年（1574年）出生。父親秀景是[[下野國|下野國]]的小山秀綱的家臣。天正18年（1590年），{{tsl|ja|小山氏}}因為抵抗[[豐臣秀吉|豐臣秀吉]]的[[小田原征伐|小田原征伐]]，於是被{{tsl|ja|改易}}。此後，成為[[浪人|浪人]]。後來因為[[佐竹氏|佐竹氏]]家臣[[人見藤道|人見藤道]]推舉，仕於佐竹義宣。以20歲之齡，繼承曾是佐竹家重臣的澁江氏，改名為'''澁江內膳政光'''（一說指名字中的「'''政'''」字是以前小山秀綱的兒子[[小山政種|政種]]所賜的[[偏諱|偏諱]]）。

在[[關原之戰|關原之戰]]後，義宣在被減封至秋田的佐竹家中進行改革。慶長8年（1603年），被提拔為家老。不過因為原本是其他家的舊臣，於是引起佐竹[[譜代|譜代]]家臣的不滿，家老[[川井忠遠|川井忠遠]]等人企圖暗殺義宣和自身，最後忠遠等人被肅清（{{tsl|ja|川井事件}}）。因此，被昇格為家老一事暫時被擱置。慶長12年（1607年），正式被任命為家老。

慶長19年（1614年），與主君義宣一同參與[[大坂冬之陣|大坂冬之陣]]，在今福與[[後藤基次|後藤基次]]、[[木村重成|木村重成]]戰鬥（[[今福之戰|今福之戰]]），最後被流彈擊中而戰死（受義宣之命而檢視政光屍體的[[須田伯耆|須田伯耆]]則指「有多處槍傷，無[[鐵砲|鐵砲]]傷痕」（{{lang|ja|何れも槍疵であり、鉄砲疵はなかった}}））。享年41歲。

== 子孫 ==
死後，嫡男宣光（受義宣賜予偏諱）成為繼承人，不過宣光在沒有嗣子的情況下死去。庶長子光久（宣光的兄長）成為繼承人。澁江家直到[[幕末|幕末]]為止，合計有8人成為家老，於藩內最多，成為家中第一名家而為人所知。

澁江氏宗家的屋敷在[[久保田城|久保田城]]三之丸下中城的西側高台（現今{{tsl|ja|秋田県民会館|秋田縣民會館}}所在地），於本丸出現火災時，被當作藩主的御殿。在[[明治維新|明治維新]]後，該地亦設有傳習學校和[[秋田縣公會堂|秋田縣公會堂]]等大規模的施設。

== 人物 ==
[[File:Citizens'_Forest_of_Akita_Prefecture_008.jpg|thumb]]「秋田縣縣民之森」）]]
*重視開發，曾言「國的寶物是山，山的衰落即國的衰落」（{{lang|ja|国の宝は山なり、山の衰えはすなわち国の衰えなり}}）。
*與[[梶原政景|梶原政景]]一同參與建築久保田城。另外，改革[[檢地|檢地]]制度等，協助農業生産和安定藩內財政，施行的農業政策被稱為「澁江田法」，甚至成為其他藩和[[江戶幕府|江戶幕府]]的參考。
