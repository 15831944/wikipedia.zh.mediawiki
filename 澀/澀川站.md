'''澀川站'''（{{jpn|j=渋川駅|hg=しぶかわえき|rm=Shibukawa eki|f=y}}）是位於[[日本|日本]][[群馬縣|群馬縣]][[澀川市|澀川市]]澀川辰巳町的[[東日本旅客鐵道|東日本旅客鐵道]]（JR東日本）[[鐵路車站|鐵路車站]]。

澀川市的中心車站。[[伊香保温泉|伊香保温泉]]等觀光地的玄關口，除了[[寢台列車|寢台列車]]之外，全部[[特別急行列車|特急]]皆停車。

**[[吾妻線|吾妻線]]

== 車站構造 ==
地上車站。月台為[[相對式月台|相對式]]1面1線、[[島式月台|島式]]1面2線，共計2面3線，站舎旁（西側）的1號線是往水上方向、長野原草津口方向的列車，島式月台内側的2號線是高崎方向列車發着用，3號線是供澀川站發的特急列車發車與其他列車退避、折返用。

== 車站周邊 ==
位於澀川市中心部的市街地，附近有不少商店和銀行。徒步5分鐘距離的站前兒童公園（通稱SL公園）保存有[[日本國鐵D51形蒸氣機關車|D51形蒸氣機關車]]724號。

[[站前廣場|站前廣場]]有往附近各温泉的巴士路線。

== 历史 ==
*1921年（[[大正|大正]]10年）7月1日 上越南線、新前橋 - 澀川間開業時開設
*1924年（大正13年）3月31日 同、澀川 - 沼田間延伸開業
*1945年（[[昭和|昭和]]20年）1月2日 長野原線、澀川 - 長野原間開通、貨物營業開始
*1945年（昭和20年）8月5日 同、澀川 - 中之条間的旅客營業開始　
*1947年（昭和22年）4月1日 上越線、高崎 - 水上間[[電氣化|電氣化]]
*1963年（昭和38年）8月2日 上越線、八木原 - 澀川間複線化
*1964年（昭和39年）3月18日 同、澀川 - 敷島間複線化
*1967年（昭和42年）6月10日 長野原線、澀川 - 長野原間電氣化
*1987年（昭和62年）4月1日 [[國鐵分割民營化|國鐵分割民營化]]後成為東日本旅客鐵道車站
*2004年（[[平成|平成]]16年）10月16日　[[東京近郊區間|東京近郊區間]]開始可以使用[[Suica|Suica]]
