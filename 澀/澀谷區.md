'''涩谷区'''（{{jpn|j=渋谷区|hg=しぶやく|rm=Shibuya ku|f=y}}）<ref>「{{lang|ja|渋谷}}」常見的譯名有兩種：「澀谷」與「涉谷」。「{{lang|ja|渋谷}}」原本所對應的傳統汉字寫法為「'''-{澁}-'''谷」；按照日本漢字簡化規則，「-{澁}-」的簡化字為「{{lang|ja|渋}}」。「{{lang|ja|渋}}」字下的四個小點即代表上面的「止」字重覆2次（「{{lang|ja|摂}}」亦同此理，為「-{zh-hant:攝;zh-hans:攝(摄)}-」的簡化字）。在中文汉字裡，「-{澁}-」為「-{zh-hant:澀;zh-hans:澀(涩)}-」的[[異體字|異體字]]。「涉谷」此譯名則是因為字形相似而來（但「涉」的對應日本簡化漢字實為「-{渉}-」，而日語平假名拼寫為わたる，羅馬字為Wataru）。而在使用狀況方面，大部分具有官方地位的日本單位與網站在其中文版訊息中，大都使用「澀谷」；而「涉谷」則幾乎都是一般民間用戶以訛傳訛的[[錯別字|錯別字]]用法。</ref>，是[[日本|日本]][[東京都|東京都]]的[[東京23區|特别区]]之一，其名稱源自於位於該行政區中央、以[[澀谷車站|澀谷車站]]為中心的[[澀谷|澀谷]]地區。

與[[新宿|新宿]]、[[池袋|池袋]]等因為位處於交通要衝上而發展出的商業街相同，澀谷也是因為位居[[山手線|山手線]]重要轉運站的地位而發展成今日的盛況。其為[[東急|東急]]集團的大本營，且各種商業活動興旺，這尤其表现在[[澀谷車站|澀谷車站]][[忠犬八公|忠犬八公]]出口處，著名的[[百货店|百货店]]、[[時裝|時裝]]專賣店、[[飲食|飲食]]店、[[咖啡|咖啡]]店、[[休閒|休閒]]設施、[[特種行業|特種行業]]等密集如雲，是與新宿同樣並列為“24小时不眠之街”的地區。另外，以[[西武百貨|西武百貨]]、[[東急百貨|東急百貨]]、[[巴而可|巴而可]]（{{lang|ja|パルコ}}）與[[109百貨|109百貨]]為中心再加上周邊許多服飾店所構成的商店街是最受東京年輕消費族群喜愛的購物勝地，向來擁有「年輕人之街」之美稱，也使得澀谷成為日本國内外各種流行風尚的發源地。而澀谷車站八公口外、具有[[行人保護時相|行人保護時相]]配置、屬[[井之頭通|井之頭通]]（{{lang|ja|井ノ頭通り}}）一部份的大型五叉路口，由於超高的行人穿越量經常被譽稱為「全世界最大的交叉路口」，其龐大的行人流通量成為許多觀光客參訪與攝影、拍照的聖地，也經常出現在電視或電影劇情中，作為東京的象徵。

== 概要 ==
[[File:Omotesando_spring.jpg|thumb]]的[[欅|欅]]樹]]
澀谷區位於東京的城西地區，與[[千代田區|千代田區]]、[[中央區_(東京都)|中央區]]、[[港區_(東京都)|港區]]、[[新宿區|新宿區]]為「[[都心|都心]]5區」。

以終點站[[澀谷站|澀谷站]]為中心的[[澀谷|澀谷地區]]是東京的[[副都心|副都心]]之一，鄰近[[新宿站|新宿站]]的[[代代木|代代木]]與[[千駄谷|千駄谷]]與[[新宿|新宿]]的辦公商區、繁華街連成一氣。此外，鄰近[[青山_(東京都港區)|青山]]（港區）的[[原宿|原宿]]、[[表參道|表參道]]是時尚的中心，[[代官山町|代官山]]周辺與[[惠比壽_(東京都)|惠比壽]]等地商業設施與時尚相關產業聚集。澀谷區有[[明治神宮|明治神宮]]與[[代代木公園|代代木公園]]等都心少見的廣大綠地，周辺的[[松濤|松濤]]、[[代代木上原|代代木上原]]是都內有名的高級住宅地。另外，同樣為高級住宅地的[[廣尾_(澀谷區)|廣尾]]則是因鄰近[[麻布|麻布]]地區而興起。日本演藝圈的經紀公司也都設在澀谷區。

另一方面，區内北部[[甲州街道|甲州街道]]與[[京王線|京王線]]沿線的北側地區與[[中野區|中野區]]、[[新宿區|新宿區]]組成連密的住宅商業地區。

區內的年間平均氣溫約15℃。冬季最低氣溫與[[大手町|大手町]]附近相比較低，冬日日數也較多。

{{Main|澀谷站|澀谷}}

; 神宮前、表參道（包含[[原宿|原宿]]）
: [[表參|表參]]沿線與周邊聚集許多精品店、服裝店、相關事務所與美容室。2006年「[[表參道之丘|表參道之丘]]」開幕。[[明治神宮|明治神宮]]的新年參拜客是日本第一。
; 代官山
: 代官山是澀谷區中僅次於表參道的時尚重鎮，[[舊山手通|舊山手通]]沿線有許多知名餐廳。
; 惠比壽
: 惠比壽原為下町地區，往JR山手線沿線通勤便利，近年車站周邊辦公大樓增加，白天上班族眾多。1994年開幕的[[惠比壽花園廣場|惠比壽花園廣場]]帶動此地發展。
; 神泉町、澀谷區澀谷
: 兩者都在澀谷站往坂上的方向。有「Bit Valley」之稱，聚集許多[[IT|IT]]相關企業。
; 澀谷區南部的住宅地
: [[南平台町|南平台町]]、[[廣尾_(澀谷區)|廣尾]]、[[松濤|松濤]]等地價在澀谷區中特別高昂。

; [[甲州街道|甲州街道]]沿線（京王新線沿線）
: [[笹塚|笹塚]]、[[幡谷|幡谷]]、[[初台|初台]]（本町）等3區。與澀谷相比，前往新宿較為便利。
; [[代代木|代代木地域]]（小田急線沿線）
: 代代木地域指JR代代木站周邊，廣義包含了[[代代木上原|代代木上原]]與[[富谷|富谷]]。最近的鐵道路線是小田急線。宇田川等河川在此地形成谷地，豐富的地形起伏類似鄰近的世田谷區。

== 歷史 ==
1932年（昭和7年），原屬[[東京府|東京府]][[豐多摩郡|豐多摩郡]]的[[澀谷町_(東京府)|澀谷町]]、[[千駄谷町|千駄谷町]]、[[代代幡町|代代幡町]]3町編入[[東京市|東京市]]成立「澀谷區」<ref>{{cite web |url=http://www.city.shibuya.tokyo.jp/shibuya/profile/uraig.html |title=存档副本 |accessdate=2013-10-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20140112044529/http://www.city.shibuya.tokyo.jp/shibuya/profile/uraig.html |archivedate=2014-01-12 }}</ref><ref>澀谷町、千駄谷町、代代幡町各別的町域相當於現在的[[澀谷警察署|澀谷警察署]]、[[原宿警察署|原宿警察署]]、[[代代木警察署|代代木警察署]]管轄地區</ref>。

區名是以合併當時發展最佳的澀谷町命名<ref>[http://www.e-koto.co.jp/useful/chimei_shibuya/p498.html 澀谷區の変遷と由来等] {{webarchive|url=https://web.archive.org/web/20100522123424/http://www.e-koto.co.jp/useful/chimei_shibuya/p498.html |date=2010-05-22 }}</ref>。
