'''加烈·約翰·希隆'''（{{lang-en|'''Garrett John Hedlund'''}}，{{bd|1984年|9月3日}}）是一名美國男演員、模特兒和歌手。

== 早期生活 ==
荷德倫出生在[[明尼蘇達州|明尼蘇達州]]小镇附近的一个牧场。14歲搬家到[[亞利桑那州|亞利桑那州]]、在那里引起了球探的注意。高中參加過[[摔角|摔角]]、[[田徑|田徑]]隊和[[美式足球|美式足球]]隊 ，在成為演員之前當過[[模特兒|模特兒]]。

== 職業生涯 ==
荷德倫高中开始上表演课，畢業後到洛杉磯、仅仅用了一个月，二十岁的他便得到了在史诗大片《[[特洛伊：木馬屠城|特洛伊：木馬屠城]]》中扮演Achilles（[[布拉德·皮特|布拉德·皮特]]）的堂弟、勇敢而年轻的Patroclus的机会。2006年他在《[[俠影魔龍|龙骑士]]》中扮演剑术高超的Murtagh一角，2010年在電影《[[创：战纪|创：战纪]]》飾演男主角山姆·費林。2011年6月7日，他獲得了Hedlund的年度人物魅力獎<ref>{{cite web|title=Garrett Hedlund - Man of the Year 2011|url=http://www.glamourmagazine.co.uk/celebrity/pictures-today/glamour-awards/2011/kristen-stewart-and-garrett-hedlund|publisher=''Glamour''|accessdate=August 9, 2011|date=June 7, 2011|deadurl=yes|archiveurl=https://web.archive.org/web/20110817015430/http://www.glamourmagazine.co.uk/celebrity/pictures-today/glamour-awards/2011/kristen-stewart-and-garrett-hedlund|archivedate=八月 17, 2011}}</ref><ref>{{Cite web|url = http://zap2it.com/2011/06/garrett-hedlund-is-glamours-man-of-the-year/|title = Garrett Hedlund is Glamour's Man of the Year|date = June 8, 2011|accessdate = October 9, 2015|website = zap2it.com|last = Liz Kelly Nelson}}</ref>。

2015年2月25日，據透露，[[李安|李安]]有意讓荷德倫出演《[[比利·林恩的中場戰事|比利·林恩的中場戰事]]》<ref>{{cite news|last1=Fleming Jr|first1=Mike|title=Ang Lee Finds ‘Billy Lynn'; Eyeing Garrett Hedlund For Big Part?|url=http://deadline.com/2015/02/ang-lee-billy-lynn-joe-alwyn-garrett-hedlund-eyed-1201381691/|accessdate=February 26, 2015|publisher=deadline.com|date=February 25, 2015}}</ref>。電影於2015年4月開拍，並於2016年上映。

== 個人生活 ==
2012年，荷德倫開始和共同在《[[浪蕩世代|浪蕩世代]]》中擔任主演的[[克絲汀·鄧斯特|克絲汀·鄧斯特]]約會<ref>{{Cite journal|url = http://www.usmagazine.com/celebrity-news/news/garrett-hedlund-reveals-how-he-began-dating-kirsten-dunst-201381|title = Garrett Hedlund Reveals How He Began Dating Kirsten Dunst|last = Johnson|first = Zach|date = January 8, 2013|journal = [[Us_Weekly|Us Weekly]]|access-date = October 9, 2015}}</ref>。
