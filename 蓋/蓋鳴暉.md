'''蓋鳴暉'''（{{lang-en|'''Joyce Koi Ming Fai'''}}；{{bd|1967年|10月3日}}），本名'''李麗芬'''，香港著名[[粵劇|粵劇]]女演員，師承[[林家聲|林家聲]]，也是[[鳴芝聲劇團|鳴芝聲劇團]]的[[文武生|文武生]]。
鳴芝聲劇團於1990年成立，曾為多個慈善機構如公益金、保良局及東華三院等籌款活動 義演，並經常於港九新界多個表演場地作公開演出。劇團於1991年赴美國拉斯維加斯及 三藩市作海外演出。1993年曾獲新加坡「獅城地方戲曲展」及香港「紅樓夢文化藝術展」邀 請演出，深受歡迎。
1993年林家聲正式收其為義女，言傳身教，繼承聲派演唱神韻及精湛舞台藝術。先後主演《南唐李後主》，《牡丹亭驚夢》、《紫釵記》等五十幾部劇目並發行唱片，成績不俗。
每年公演過百場粵劇，在港澳及東南亞地區影響深遠，被譽為當今港界粵壇第一女文武生。率團出訪美、加等國家，均在當地引起熱烈反響。
除了粵劇，她也曾參加電視劇及電影的演出，並與香港中樂團合作表演﹔並在2006年度的香港藝術節演出兩齣好戲，示範性別反串這個有趣的戲曲傳統: 《龍鳳爭掛帥》和《俏潘安》。 粵劇：主攻文武生，也可應花旦。電視劇 : 常演反串角色，因此經常引人注目。
蓋鳴暉多來為慈善盡心盡力，多次為慈善服務。1992年開始為保良局粵劇籌款，以及1995年開始為東華三院粵劇籌款。蓋鳴暉於1995、1997、1998及1999年度獲選為「香港十大愛心之星」與張學友、劉德華、黎明、鄭秀文等藝人一起獲選香港十大愛心之星。2001年獲選為「香港十大傑出青年」及於同年獲東華三院頒發「東華永恆愛心之星」。
2005年成立了「蓋鳴暉慈善基金」為推廣中國傳統戲曲藝術、培育年幼、關顧長者及幫助社會上有需要人仕。於同年8月為助新界鄉議局重建籌募經費，赴英國義演。10月應加拿大世界宣明會邀請參加「愛心之旅2005」，到外蒙古進行助養兒童計劃拍攝特輯。2004、2005及2008年，三度受邀與香港中樂團合作演出。2008應香港駐紐約經貿辦之邀，在「紐約香港節2008」演出，吸引當地非華裔市民觀賞，對推動粵劇邁向國際化很有幫助。多次遠赴中國、美加、新加坡、澳洲、英國等各地演出。

== 個人經歷 - 一切由喜歡開始 == 

===1986年以前：香港八和粵劇學院 學習生涯===
自小鍾情粵劇
「我喜愛粵劇好像是與生俱來，由小時候開始，我已經十分喜歡看戲，那時候我們一家人住在新界，每逢觀音誕或佛誕，都會有劇團在戲棚演大戲。」蓋鳴暉說，每到那個時候，年紀輕輕的她總會嚷著同樣愛好粵劇的媽媽帶她一同去看戲。除了在戲棚，電視的粵語長片也讓她認識粵劇。眾多名伶中令她最難忘的大老倌，非鳳凰女「女姐」莫屬，因為「女姐」生鬼、搞笑及親民的形象深入民心，蓋鳴暉說「還記得自己當了『女姐』的粵劇Fans後，經常到戲棚看她做戲，散場以後『女姐』雖然急趕，但總會跟戲迷打招呼或是寒暄幾句，有幸與偶像傾偈實在一生難忘。」從此時起，蓋鳴暉便立志要入戲行。由於自己很喜歡花旦的裝扮，所以想讓鳳凰女收她做徒弟，但是鳳凰女說她當時還太小，讓她好好讀書先。
雖然蓋鳴暉熱愛粵劇，但父母並不贊成她學戲，因為他們認為戲行是一個大染缸、大花筒，所以不希望女兒加入演藝界，直到中學階段，蓋鳴暉見到報章上八和會館招生的廣告，她覺得是千載難逢的機會，於是瞞著家人走去面試。得到通知獲面試已欣喜若狂，面試前夕更徹夜難眠。就這樣蓋鳴暉1983年開始了在八和會館為期兩年的訓練，每星期受訓三日，每堂兩小時，每次上堂既要拉筋又要操曲，還要給導師罵個狗血淋頭，受訓途中，很多學員受不了要退出，還有些沒天分的不獲留下，結果原本一班20名學員只剩下7人。而蓋鳴暉，卻愈學愈起勁。不過真正學藝的兩年日子卻是十分辛苦，單是練習壓腿已教她痛不欲生，一度想過放棄，幸好能夠刻苦堅持才造就了今天的成功。八和會館成員首次演出是由當年擔任八和主席的琴姐李香琴帶領眾學員一起踏上舞台，這讓蓋鳴暉深切體會到演出粵劇殊不簡單，正所謂要「話頭醒尾，一眼關七。」
重溫往事，蓋鳴暉仍難掩興奮之情，雖然考官對她的評語是：「花旦唔夠靚，小生唔夠高」，反而激勵了蓋鳴暉要加倍努力的決心。

===1987年-1989年：偶遇生命中改變命運的兩位大貴人===
蓋鳴暉說自己是初生之犢不畏虎，為了有更多演出機會，87年後她與幾位學員一起參加區域市政局粵劇新秀匯演，演出經典戲寶《牡丹亭驚夢》和《龍鳳爭掛帥》，龍鳳爭掛帥也是林家聲的開山名劇，所以蓋鳴暉大膽的邀請鼎鼎大名的大老官[[林家聲|林家聲]]前來觀看演出，聲哥也應約來看演出，還親自到後台給Joyce上大靠，教她上了大靠後應該怎麼呼吸才最舒服。一場演出竟然因此讓她遇上改變命運的兩位大貴人。蓋鳴暉感恩地說：「一位是粵劇界泰斗聲哥林家聲，他看完《龍鳳爭掛帥》還到後台探班，我和其他團員都開心到震，之後Joyce也帶著自己演出的錄像帶到聲哥家請求指導，同年受名伶林家聲賞識，於「頌新聲劇團」《周瑜》一劇客串趙雲一角，並要求一同練功。師徒緣分也是從這個時候開始的。其後更得聲哥收為徒弟及結成誼親。
另一位是班主劉金耀先生，劉先生是「鳴芝聲劇團」的班主，他之前是一個生意人，但劉氏夫婦倆一直愛好粵劇，認為蓋鳴暉有潛質，希望全力支持，於是設立劇團。
「得到劉生劉太的栽培，實在感激，他倆實在是我終生的恩人，猶如再生父母，由90年劇團的成立，開頭的路的確不容易走，最初的幾年一直虧本，他們不斷為劇團找出路，很不容易才走過16年。」認識劉金耀夫婦成了蓋鳴暉事業上的轉捩點。對於一個出道僅三年的新秀能有這般際遇可說是奇迹，更對於蓋鳴暉而言更是人生另一頁光輝歲月。不過她謙遜地表示：「出道不算太崎嶇，也不算平步青雲，中途也曾有少許波折，做人眼光要放遠和知足，多謝曾幫過自己的人。
蓋鳴暉在八和會館的日子大概有5年多，在1986年因學院有匯演，當時蓋鳴暉第一次擔正做主角。她坦言，「可能真是錯有錯著，因為班裡男角較少，沒有花旦僧多粥少的問題。」若非反串當男角，恐怕在會館的匯演也沒有機會當主角。「匯演就是畢業演出，首次開腔當然很緊張，還記得當時有李香琴『琴姐』來看我們，我們看到台下有前輩，不由得更為驚慌，但完成演出後很開心，琴姐還特意走來鼓勵，跟我說覺得我表演不錯，叮囑我多用心在戲曲上。」1989年時蓋鳴暉也追隨楊劍華學習舞台功架。

===1990年-1991年：組建鳴芝聲粵劇團，首次到海外演出===
畢業於香港八和粵劇學院( 第二屆)，專攻文武生行當，1990年Joyce拜粵劇音樂家[[朱慶祥|朱慶祥]]先生為師，也曾追隨[[王粵生|王粵生]]、[[劉兆榮|劉兆榮]]、麥惠文三位音樂老師研習曲藝，研習唱功。隨郭鴻斌、任大勳、李艷霜、陳敏等學藝。1990年蓋鳴暉與楚令欣參與「樂昇平粵劇團」；同年與莊婉仙( 香港八和粵劇學院第一屆畢業生) 合組「鳴芝聲劇團」，擔任文武生，先夥拍尹飛燕，後長期夥拍吳美英至今。

1991年，不到一年的時間蓋鳴暉就受邀到美國拉斯維加斯賭場演出粵劇。

===1992年-1994年：首次落鄉出演神功戲===
1992年受村長的邀請[[鳴芝聲劇團|鳴芝聲劇團]]到[[鴨脷洲|鴨脷洲]]公演神功戲，這也是[[鳴芝聲劇團|鳴芝聲劇團]]首次落鄉做神功戲。1993年，
[[林家聲|林家聲]]收Joyce做為徒弟與義女，言傳身教，繼承聲派演唱神韻及精湛舞台藝術。

===1995年-1997年：首次涉足影視界===
1995年開始蓋鳴暉獲得監製的賞識首次涉足影視，在無綫電視台慶劇《刀馬旦》中客串一名女扮男裝的高一鳴。其端莊清雅的扮相和準確、得體的出色表演深得無線台賞識，1996年再次接拍無綫電視電視劇《[[壹號皇庭V|壹號皇庭V]]》出演女醫生夔彧藠。
1997年蓋鳴暉也開始活躍在電視圈，並多次受邀擔任《歡樂滿東華》等香港演藝界大型活動的司議，也再受邀出演無綫電視收視強檔劇《[[西遊記（貳）|西遊記（貳）]]》中擔綱戲份吃重的觀音菩薩。除了電視圈，在粵劇界，蓋鳴暉也參與了粵劇之家系列的粵劇[[趙氏孤兒|趙氏孤兒]]演出劇目，同年也參與了香港八和會館慶祝香港回歸1997年演出的粵劇劇目《[[七賢眷|七賢眷]]》。

===1998年-2000年：首次登上舞台劇舞台===
1998年主唱了大型電視節目《江山如此多嬌》的主題曲。　
同年的12月至1999年1月蓋鳴暉與香港演技派紅星[[謝君豪|謝君豪]]聯袂演出舞台劇《梁祝》，資深製作，杜國盛任該劇編劇風光藝術總監，曾成功執導《雪狼湖》的著名導演司徒慧焯任該劇導演。終能在舞台上回復女兒身，一樣大受歡迎，蓋鳴暉說：「最珍貴是學會了把舞台劇融會到粵劇演出中，連師母紅豆子也大為讚賞！」《梁祝》在港上演後引起轟動，場場爆滿不得不邊連續加演(一共加演至36場)。1999年Joyce再次接拍無綫電視強檔劇《[[緣份無邊界|緣份無邊界]]》，出演女督察呂秀琴。同年應無線電視台之邀做旅遊節目主持，赴[[越南|越南]]、[[新加坡|新加坡]]及[[海南島|海南島]]等地拍旅遊專輯。在歡樂滿東華慈善節目中上演劇目《喜得銀河抱月歸》及《南唐李後主》。　

===2001年-2004年：獲選為香港十大傑出青年，首次在粵劇中擔任花旦===
2001年再度受邀拍攝電視劇，於《[[尋秦記|尋秦記]]》中反串龍陽君一角。同年9月 受邀於春暉劇團與阮兆輝演《帝女花》（吳美英蓋鳴暉先後分飾長平公主）同年2001年蓋鳴暉獲選為「香港十大傑出青年」，只因她平日常為慈善團體義演和探訪長者而被提名，她說本着平常心去面試但竟緊張得手心冒汗，結果她對社會和公益的貢獻獲得認同而獲選。「做了傑青對自己要求更高，覺得更要腳踏實地多做有意義的事，對得起自己也要對得起別人，粵劇人講求是忠孝仁義，飲水思源。」。2003年蓋鳴暉與[[馬浚偉|馬浚偉]]、[[張可頤|張可頤]]及[[陳豪|陳豪]]等演員合作電視劇《[[金牌冰人|金牌冰人]]》，於劇中飾演令狐喜再次爆紅。2004年蓋鳴暉開始擔任粵韻今宵節目藝術顧問。同年4月，蓋鳴暉首次與香港中樂團合作兩場的演唱會。
同年12月 廈村十年一屆太平清醮，廈村有很多蓋鳴暉的戲迷，但廈村太平清醮一向都請男文武生，而今年請了鳴芝聲，文武生是女兒身的蓋鳴暉，結果大會尊重民意，由總理及長老在祠堂召開大會，一致通過聘請蓋鳴暉，但要求蓋鳴暉在開鑼第一晚的《六國大封相》中反串正印花旦推車，又在《鳳閣恩仇未了情》中擔任正印花旦，這一措施使到大會八千座位戲棚座無虛席。

== 個人經歷 - 靠努力創出一片天 == 
===2005年-2007年：成立慈善基金會，為香港藝術節演出粵劇===
2005年Joyce為推廣中國傳統戲曲藝術、培育年幼、關顧長者及幫助社會上有需要人仕成立了「蓋鳴暉慈善基金」。同年受電視台邀請與[[元華|元華]]等合作出演電視劇《[[蓋世孖寶|蓋世孖寶]]》，飾演女俠董嵐。同年4月，蓋鳴暉與香港中樂團二度合作4場的演唱會。同年7月，蓋鳴暉與梁漢威、吳仟峰、吳美英、陳詠儀、阮兆輝、合演新劇本#[[滿清皇朝|滿清皇朝]]。同年8月 為助新界鄉議局重建籌募經費，赴英國義演。同年10月蓋鳴暉應加拿大世界宣明會邀請參加「愛心之旅2005」，到外蒙古進行助養兒童計劃拍攝特輯。 同年11月，在嶺南大學演講和對談，詳述女文武生藝術和她對香港粵劇文化的見解。 同年11月，Joyce與梁漢威、吳美英及阮兆輝等等合作 漢風粵劇研究院新劇本《銅雀台》。2006年參演電視台處境喜劇《[[高朋滿座|高朋滿座]]》，飾演朱莉亞。同年為2006年度的香港藝術節演出兩齣好戲，示範性別反串這個有趣的戲曲傳統: 《龍鳳爭掛帥》和《俏潘安》。。同年4月 ，Joyce與梁漢威、吳美英等等 合作漢風粵劇研究院新劇本《漢家天下》同年11月，Joyce與梁漢威、吳美英合演 [[搶新娘|搶新娘]]，Joyce在搶新娘中首度飾演醜旦。 2007年4月，馬浚偉和蓋鳴暉於復活節期間受邀到美國Foxwoods Casino演出兩場演唱會，兩日之後兩人到達加拿大多倫多市中HUMMINGBIRD CENTER為《寶濟細胞治療基金會》義演一場，為經濟困難的高危血友病者籌募善款。為此還特別練習了由社交舞老師周志坤編排以《梁祝》及《香夭》為主題的華爾茲以及拉丁舞社交舞段。蓋鳴暉覺得今次慈善演出相當有意義，他們亦是義務演出，為此努力綵排，今次以兩首粵曲來排嶄新舞步亦十分新鮮。

===2008年-2010年：三度與香港中樂團合作===
2008年4月，蓋鳴暉自從和中樂團合作過之後，她說自己的眼光也擴闊了不少，主要是明白到曲藝邊界也是無止境。在04年和中樂團第一次合作，她不諱言為了適應管弦中樂團的演出方式，確是付了相當的努力，而她也很感謝中樂團的指揮兼藝術總監閻惠昌的指導。蓋鳴暉一直都有心願希望能把粵劇的特色帶到普羅的音樂層面，而生長在陝北的閻惠昌也認為中國的地方戲曲的音樂有很動聽的元素，所以他也和樂團的演奏家排練了不少粵曲伴奏，故而在第三次和蓋鳴暉合作時，節目中就有不少的粵曲。同年Joyce應香港駐紐約經貿辦之邀，在「紐約香港節2008」演出，吸引當地非華裔市民觀賞，對推動粵劇邁向國際化很有幫助。2009年在上水河上鄉戲棚演出受傷一事。蓋鳴暉:｢我記得那時我要在大約十呎高的火爐上打空翻跳下來，但當時的佈景搭得不太穩陣，當我準備跳的時候整個佈景一下子便倒塌了。那一刻我很痛、站不起來，我覺得人生好像經歷了地震一樣，我在想我的戲劇生涯是否就此『玩完』。後來到廣華醫院檢查才知道我的尾龍骨兩節因此裂了，幸好不是傷及有神經線的幾節，否則我也不知道有多久不能站起來。｣ 蓋鳴暉難忘東華三院細心治療及各界對她的支持：｢那時候醫生非常細心的治療我，不是因為我是蓋鳴暉，我相信他們治療每一位病患者也如此細心，因此每年我也為東華三院義演。我們這一行不能找替工，受傷了或會取消演出，連累班主賠償，又會令觀眾不開心，因此當時我很怕被傳媒或別人知道我受傷。傷及尾龍骨需要很長的康復時間，我很感謝每次邀請我演戲的單位及支持我的觀眾，他們表示只要聽到我的聲音已經很開心，每次也提醒我慢慢做、不需要急。｣蓋鳴暉就這樣斷斷續續休養了半年，腰傷漸漸有了好轉。同年2010年1月，Joyce拍畢處境喜劇《[[畢打自己人|畢打自己人]]》後繼續往粵劇界發展。

===2011年-2013年：梅蘭芳大劇場聯合匯演新戲《蝶海情僧》===
因得到李居明師傅邀請，演出由他編撰的首部劇本《蝶海情僧》。「這是一次很奇特的機會。記得在2010年，李師傅想找我演《蝶海情僧》，劉先生也考慮了一段時間，後來，他覺得雖然不認識李師傅，但見見面也無妨，不多行一步見面了解，也不知是甚麼一回事？有人想邀請我演新戲，由我開山，也是好事呀！」《蝶海情僧》是李居明師傅第一次寫全劇，「他的詞寫得很美，但美不代表易唱。但我這樣考慮，既然是好詞句，我便應接受挑戰。接劇本後，我花了很多時間去背曲，把詞全都記熟，之後演了李師傅10個劇，我的記憶力竟全面提升，如今我記其他戲易如反掌，一下子就入腦。迎着困難上，不要害怕，不要廻避，總可以把困難克服。我希望自己的行為，能鼓勵到其他同行也熟讀劇本上台，在這方面，我想做一個榜樣。」似乎屬於蓋鳴暉人生和事業的光輝一頁又一頁，2011年她的新劇《蝶海情僧》在香港演出之餘，還在北京梅蘭芳大劇院，聯同京劇、越劇一起演出，開香港粵劇界先河，為廣東粵劇爭光不少，蓋鳴暉表示實在不枉此生。被問到是否滿意一直以來的成績，她堅決地說：「一世不會滿意，但會滿足，與觀眾見面的感覺就如和親人聚會，每次都很享受，對於演戲永不言休，所以一定會做好自己，滿足大家。」
蓋鳴暉的成功引證到需要天時、地利、人和的配合，也證明後天努力的重要性，正如蓋鳴暉說，劉德華去看她和中樂團演出時曾鼓勵她，他可做到的蓋鳴暉也可以，最重要是要努力！因此蓋鳴暉敢於嘗試不同工作，汲取不同經驗，儲備作為日後之用。談到在不同階段的成就時，都不忘多謝出力幫忙過的人。明明是女兒心，確有男兒氣概，難怪舞台上瀟灑俊朗的蓋鳴暉，迷倒萬千「顧曲周郎」了。2013年11月 蓋鳴暉受電視台邀請回巢無綫拍攝《[[2014無綫節目巡禮|2014無綫節目巡禮]]》。

===2014年-2016年：首次到日本演出粵劇，第二次出演舞台劇===
2014年1月，受電視台邀請拍攝電視劇《[[無雙譜_(2015年電視劇)|無雙譜]]》，再次飾演觀音菩薩。同年7月，再次拍攝電視劇《[[水髮胭脂|水髮胭脂]]》。同年月15-17日鳴芝聲劇團在粵劇界發展踏入廿四年，這次被邀在日本演出《蝶海情僧》，也可以說是蓋鳴暉本人在事業上踏出另一個里程碑，連行政人員在內，團隊前往日本的總共有七、 八十人，可算是陣容最龐大的一次。
同年11月8-17日 公演[[杜國威|杜國威]]最新編劇作品《星海留痕》，此次演出，杜國威為蓋鳴暉重新編撰劇本，指小明星的角色非其莫屬。蓋鳴暉曾與謝君豪合作演出舞台劇《梁祝》，今次再演舞台劇，演繹小明星，她坦言是個挑戰。劇中蓋鳴暉及黎耀祥會演唱三十多首歌曲。飾演王心帆的三屆視帝得主黎耀祥首次踏台板演出舞台劇，加上劇團特邀關寶慧演出，憑各人豐富的演藝經驗，相信能於劇中擦出火花。由於反應不俗，並再加演21-23日三場。（一共13場）2015年9月 ，受新加坡[[聖淘沙名勝世界|聖淘沙名勝世界]]邀請公演兩場粵劇《[[紫釵記|紫釵記]]》、《[[帝女花|帝女花]]》完整版。同年11月於[[沙田|沙田]][[沙田大會堂|沙田大會堂]]公演[[黎耀威|黎耀威]]2015新編劇《[[佳偶天成|佳偶天成]]》兩場。
