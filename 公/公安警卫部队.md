'''公安警卫部队'''，是原[[公安现役部队|公安现役部队]]的一个警种，列入[[中國人民武裝警察部隊|中國人民武裝警察部隊]]序列，由[[公安部警卫局|公安部警卫局]]领导，不受[[武警总部|武警总部]]直接领导。

2018年3月21日[[中共中央|中共中央]]印发的《[[深化党和国家机构改革|深化党和国家机构改革]]方案》提出，公安警卫部队不再列武警部队序列，全部退出现役<ref>[https://news.qq.com/a/20180321/028746.htm 公安边防、消防、警卫部队全部退出现役] </ref>。2019年1月1日，公安警卫部队举行集体换装仪式，至此公安警卫部队完成改制<ref>{{Cite web|url=http://www.bjnews.com.cn/news/2018/12/25/533722.html|title=三大公安现役部队全部完成改制|accessdate=2019-01-02|date=2019-01-02|publisher=新京报}} </ref>。改革后，公安部组建'''[[公安部特勤局|公安部特勤局]]'''（八局），撤销公安部警卫局（八局）<ref name="报道0225A">{{cite web |author1=湖南公安 |title=站在新起点、履行新使命 湖南省公安厅特勤局举行集体换装入警宣誓仪式 |url=http://hunan.sina.cn/fazhi/2019-01-02/detail-ihqfskcn3252208.d.html |website=新浪网湖南 |accessdate=2019-02-25 |location=北京 |language=zh-cn}}</ref>。

== 沿革 ==
1981年7月，经中央批准，各省、市、自治区的警卫机构统一归省、市、自治区公安建制领导。此次改革后，各省级党委警卫处划归下属的省（自治区、直辖市）公安厅（局）建制领导，并与公安厅（局）外宾保卫处合并为警卫处，同时挂省（自治区、直辖市）委办公厅警卫处牌子。

1984年3月8日，经[[中共中央书记处|中共中央书记处]]批准，中共中央办公厅转发《全国警卫工作座谈会纪要》，指出：“为了保证警卫人员的政治素质和业务素质，符合革命化、军事化、战斗化的要求，拟将全国现有2000名警卫人员列入武警建制，享受武警待遇。”各省（自治区、直辖市）党委办公厅警卫处改为武警现役编制，副师级，在公安厅内办公。<ref name="guangxi">《广西通志 公安志》第九篇 警卫 第一章 机构 </ref>

公安警卫部队负责[[党和国家领导人|党和国家领导人]]、[[中华人民共和国省级行政区领导人|省级主要领导人]]及重要来访外宾警卫任务。公安部警卫局负责“四副两高”的警卫工作，省厅警卫局和警卫处负责当地[[领导人|领导人]]和重要来宾的警卫工作。警卫部队由[[公安部警卫局|公安部警卫局]]管理，各省、自治区和直辖市公安厅（局）设有警卫局。警卫部队是武警编制，有武警的证件。警卫部队是公安机关的一部分，属兵役制人员，也持有职业警察的[[人民警察证|人民警察证]]。

2018年3月21日[[中共中央|中共中央]]印发的《[[深化党和国家机构改革|深化党和国家机构改革]]方案》提出，公安警卫部队不再列武警部队序列，全部退出现役。公安警卫部队转到地方后，警卫局（处）由同级公安机关管理的体制不变，承担规定的警卫任务，现役编制全部转为人民警察编制<ref>[https://news.qq.com/a/20180321/028746.htm 公安边防、消防、警卫部队全部退出现役] </ref>。

2019年1月1日，公安警卫部队举行集体换装仪式，至此公安警卫部队完成改制，公安警卫部队建制撤销<ref>{{Cite web|url=http://www.bjnews.com.cn/news/2018/12/25/533722.html|title=三大公安现役部队全部完成改制|accessdate=2019-01-02|date=2019-01-02|publisher=新京报}}</ref>。

== 编制 ==
2018年改革前，公安部警卫局指导地方各级警卫任务单位。各省（直辖市、自治区）公安厅（局）设警卫局，省厅警卫局挂省委办公厅警卫局牌子<ref>{{Cite web|url=http://www.henan.gov.cn/jrhn/system/2009/01/24/010116373.shtml|title=徐光春看望慰问省委警卫局和武警河南总队郑州支队一大队三中队官兵|accessdate=2018-04-03|date=2009-01-24|publisher=河南省人民政府门户网站}}</ref>。省厅警卫局为正师级。任务繁重的省级警卫局局长可挂公安部警卫局副局级调研员身份（副军级）、授武警少将警衔<ref>{{Cite web|url=http://news.ifeng.com/mainland/detail_2013_12/12/32065804_0.shtml|title=公安部决定高配西藏警卫局 局长贾利国为副军职|accessdate=2018-04-03|date=2013-12-12|publisher=凤凰网}}</ref>。省厅警卫局的正团级内设机构：办公室、政治部、首长警卫处、外宾警卫处、会议警卫处、现场警卫处、防爆安全检查处、训练处、信息技术处、后勤装备处、审计处、警卫队等<ref name="guangxi" />。地级行政区公安局设警卫处。如辽宁全省共有18个地市级警卫处，其中武警现役6个、非现役职业制民警8个、行业警卫处4个<ref>{{Cite web|url=http://www.mxwz.com/city/view.aspx?CID=11&ID=76541|title=铁岭市公安局警卫处夺取全省警卫系统基本功考核比武团体第一名|accessdate=2018-04-03|date=2009-03-10|publisher=民心网}}</ref>。市级警卫处的人员一般不超过10人。市公安局警卫处也称作市委办公厅警卫处<ref>{{Cite web|url=http://www.changsha.gov.cn/xxgk_1/szfxxgk/ztbd_1/zxzzzt/ldxz/201005/t20100510_298505.htm|title=工程建设领域突出问题专项整治工作专题|accessdate=2018-04-03|date=2010-05-10|publisher=长沙市人民政府|archiveurl=https://web.archive.org/web/20101203222203/http://www.changsha.gov.cn/xxgk_1/szfxxgk/ztbd_1/zxzzzt/ldxz/201005/t20100510_298505.htm|archivedate=2010-12-03}}</ref>（部分地市公安局下设机关是“科”而不是“处”，因此职业民警建制的警卫工作部门称为市公安局警卫科）。警卫处下辖勤务一科、勤务二科等。部分县区公安（分）局根据工作需要设有警卫科，大部分是职业制民警建制，少部分是武警现役建制。部分县级行政区公安局的治安大队兼县局警卫科任务<ref>{{Cite web|url=http://www.yw.gov.cn/ywgaj/008/01/03/200806/t20080602_131599.html|title=治安大队|accessdate=2018-04-03|date=2008-06-02|publisher=义乌市公安局|archiveurl=https://web.archive.org/web/20101204123230/http://www.yw.gov.cn/ywgaj/008/01/03/200806/t20080602_131599.html|archivedate=2010-12-04}}</ref>。部分县级行政区公安局的经济犯罪侦查大队（或设经济文化保卫大队）兼县局警卫科任务。

除公安部警卫局外，[[国务院办公厅警卫处|国务院办公厅警卫处]]负责国务院首长的警卫工作<ref>{{Cite web|url=http://politics.people.com.cn/GB/1026/9768637.html|title=国办警卫处长刘恩耀等3人晋升武警少将 孟建柱颁授|accessdate=2018-04-03|date=2009-07-31|publisher=人民网}}</ref>。国办另设[[国务院办公厅保卫处|国务院办公厅保卫处]]，负责国务院机关保卫工作。[[全国人民代表大会常务委员会办公厅|全国人大常委会办公厅]]、[[中国人民政治协商会议全国委员会办公厅|全国政协办公厅]]也都设有警卫处，负责首长警卫工作。

== 规章制度 ==
=== 人员与训练 ===
警卫人员要求具有良好的业务、技能、体能素质。要求两眼裸视5.0以上、身高1.75米以上。在[[中国人民武装警察部队学院|中国人民武装警察部队学院]]警卫学本科专业开设治安管理学、散打、驾驶技术、防暴技术、警卫基础理论、警卫参谋学、警卫勤务学、警卫战术学、警卫指挥学等业务课程。在职警卫人员需掌握手枪精度射击、游泳、水上救护、搏击、驾驶等技能，达到全面的体能要求。

[[中国人民武装警察部队学院|中国人民武装警察部队学院]]设有警卫系，具有警卫学本专科学历教学资格。

警卫工作是各级公安机关重要职责。在警卫任务繁重的公安机关，由公安部批准设立武警现役的警卫局、警卫处、警卫科；没有现役警卫建制的地市州县公安局设立警卫处或警卫科。武警现役的人员更具有流动性，因此在年龄、业务技能上更具有优势。
