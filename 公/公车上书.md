'''公車上書'''，即'''公車孝廉連署上書'''，指[[康有为|康有为]]于[[清朝|清朝]][[光绪|光绪]]二十一年（1895年）率同[[梁启超|梁启超]]等[[公車_(漢朝)|公車孝廉]]{{noteTag|「公車」指赴京[[会试|会试]]的[[举人|举人]]，典故是[[汉代|汉代]][[孝廉|孝廉]]上京師[[察舉|察舉]]，皆乘「公家馬車」。}}<ref>[[王丹|王丹]]、[[羅文嘉|羅文嘉]]、[[王軍濤|王軍濤]]《公共知識分子 No.1: 茉莉花革命》</ref>联名向[[北京|北京]]的[[光绪皇帝|光绪皇帝]]上书，反对在[[甲午战争|甲午战争]]中败于[[日本|日本]]的清政府签订丧权辱国的《[[马关条约|马关条约]]》。此舉被认为是[[维新派|维新派]]登上历史舞台的标志，也被认为是中国群众的[[政治运动|政治运动]]的开端。

如[[茅海建|茅海建]]等不少学者则考証许多文献表示，康有为所谓的组织「公車[[孝廉|孝廉]]連署上書」，事实上，只是一次自吹自擂的事件，只有八十個舉人參加[[連署|連署]]，而非康有為所稱的一千多人。而當時仅仅一名廣東舉人[[陳景華|陳景華]]的上書都有兩百八十人的連署，遠勝康有為。其實當時也有真正成功的上書，实由当时的[[翁同龢|翁同龢]]、[[李鸿藻|李鸿藻]]、[[汪鸣銮|汪鸣銮]]等京城高官发动组织，目的是阻挠《[[马关条约|马关条约]]》的签订。<ref>[http://www.cnki.com.cn/Article/CJFDTOTAL-JDSY200503000.htm 茅海建《“公车上书”考证补（一）》]、[http://www.cnki.com.cn/Article/CJFDTOTAL-JDSY200504003.htm 茅海建《“公车上书”考证补（二）》]</ref>

== 背景 ==
[[File:Litiefu1904.jpg|缩略图]] 布上油畫 70.7×56cm|右]]
光緒二十年（1894年）[[中日甲午战争|中日甲午战争]]，中國败于日本。光緒二十一年（1895年）春，参加完乙未科會試的各省[[举人|举人]]雲集[[北京|北京]]，等待發榜。[[李鴻章|李鴻章]]與[[伊藤博文|伊藤博文]]簽訂的《[[s:馬關條約|马关条约]]》內割讓[[台灣|台灣]]及[[遼東|遼東]]，賠款[[白銀|白銀]]二萬萬兩的突然消息傳至，群情激愤。4月22日，康有為、梁啟超寫成一萬八千字的《[[s:上清帝第二書|上今上皇帝書]]》，[[內地十八省|內地十八省]]與[[奉天|奉天]]三省（[[中國東北|中國東北]]）舉人接連響應，共一千二百多人[[連署|連署]]。5月2日，由康、梁二人帶領，各省舉人與數千北京官民集於「[[都察院|都察院]]」門前請代奏[[光緒帝|光緒帝]]。

== 结果及影响 ==
虽然，上书被清朝官方拒绝，但在社会上产生了巨大影响。[[康有为|康有为]]等以“变法图强”为号召，组织[[强学会|强学会]]，在[[北京|北京]]、[[上海|上海]]等地发行[[报纸|报纸]]，宣传维新思想。[[严复|严复]]、[[谭嗣同|谭嗣同]]亦在其他地方宣传维新思想。之后，[[光绪帝|光绪帝]]启用康有为、梁啟超等，史称'''[[戊戌变法|戊戌变法]]（或'''百日维新'''）。

公车上书和戊戌变法都先后失败，但是维新思想从此唤醒、激励了愈來愈多的中国人，思考中國如何救亡图存於[[歐洲|歐]][[日本|日]]列強[[帝國主義|帝國主義]]，在中国歷史上有着重要的地位。

== 争议 ==
2005年北京大学历史学教授[[茅海建|茅海建]]在《近代史研究》第3期和第4期上发表《“公车上书”考证补》，根据中国第一历史档案馆所藏档案及其已公开发表过的档案文献，从政治决策高层的角度，重新审视公车上书的背景、运作过程及其影响力。其观点是：“公车上书”是有两个不同的概念，一是由政治高层发动、京官组织的上书，其数量多达31件，签名的举人多达1555人次，且上书已达光緒的[[御座|御座]]；另一个是由康有为组织的「十八行省举人联名上书」，那是一次流产的政治事件。而且康有为组织的「十八行省举人联名上书」，并不是[[都察院|都察院]]不收，而是康有为根本没有去送。由此学术界进行了广泛而激烈的讨论，并且对“公车上书”进行了进一步的审视和研究。根据不少学者的看法，康有为在进行政治宣传以及回忆的时候（最明显如其《康南海自订年谱》）存在着很多不尊重事实的地方，而且又有很多吹牛造假的现象<ref>[http://www.chinanews.com.cn/hb/news/2009/11-03/1944820.shtml  豆棚闲话：康有为这人近乎骗子]</ref>。

如[[茅海建|茅海建]]等不少学者则根据许多文献认为康有为所谓的组织举人联名上书，事实上是一次流产的政治事件。公车上书实则是由当时的[[翁同龢|翁同龢]]、[[李鸿藻|李鸿藻]]、[[汪鸣銮|汪鸣銮]]等京城高官发动组织，目的是阻挠《[[马关条约|马关条约]]》的签订。另有研究者认为，当时清政府内部已经趋于求变，即使是保守派的徐桐和[[荣禄|荣禄]]，也曾对变法做过努力。公车上书的时候，十八行省“公车”绝大多数都没有参加康有為组织的[[簽名運動|簽名運動]]，他只征集到80名广东人的[[聯署|聯署]]。而仅仅是另一人[[陈景华|陈景华]]就鼓动了一场280多人签名的「广东公车上书」。<ref>[http://www.cnki.com.cn/Article/CJFDTOTAL-JDSY200503000.htm 茅海建《“公车上书”考证补（一）》]、[http://www.cnki.com.cn/Article/CJFDTOTAL-JDSY200504003.htm 茅海建《“公车上书”考证补（二）》]</ref>
