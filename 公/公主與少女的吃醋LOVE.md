== 故事簡介 ==
冰澤和希是意之杜學園的學生會長，由於父親欠下鉅款並且失蹤的關係，讓他必須接下學園所有的工作。隔天和希到學校後，史坦哈爾王國的公主亞莉澤突然出現在和希面前，並稱意之杜學園的所有權已讓渡給史坦哈爾王國。因為這突如其來的變化，讓和希的生活有了重大的改變<ref>{{cite web|url=https://www.game-style.jp/soft/201808/08/01yak_ps.php|title= 恋に一途なお姫様＆ヤキモチ焼きなお嬢様が繰り広げる、甘くてエッチな主人公争奪戦！ 『姫と乙女のヤキモチLOVE』9月28日発売！|website=Game-Style|date=2018-08-08|accessdate=2018-08-26|language=ja}}</ref>。

== 登場角色 ==
; {{nihongo|冰澤和希|氷澤 和希（ひさわ かずき）}}
: 作品男主角，意之杜學園二年級生與學生會長。和史坦哈爾王國的公主亞莉澤是青梅竹馬。
; {{nihongo|亞莉澤·維埃爾德|アリーゼ・ヴィルエルド|聲：{{lang|ja|くすはらゆい}}}}
: 史坦哈爾王國的公主，擁有第一王位繼承權，後來到意之杜學園留學。個性單純且不通世事。與和希是青梅竹馬並一直喜歡著他。
; {{nihongo|天都沙良|天都 沙良（あまみや さら）|聲：[[青葉林檎|青葉林檎]]}}
: 意之杜學園二年級女學生，與和希是十年的青梅竹馬。個性不坦率，有著孩子氣的一面。
; {{nihongo|銅襄菜乃花|銅襄 菜乃花（どうじょう なのか）|聲：[[小倉結衣|小倉結衣]]}}
: 意之杜學園三年級女學生，前任的學生會長，對和希來說是無法違抗的存在。喜歡對和希惡作劇。
; {{nihongo|貝兒亞茱·波森|ベルアージュ・ポゾン|聲：{{lang|ja|藤咲ウサ}}}}
: 從外國到日本留學的女孩，就讀意之杜學園一年級，暱稱為「貝兒」。喜歡御飯糰和日本的漫畫，未來想當漫畫家。
; {{nihongo|瑪裘莉努|マジョリーヌ|聲：北見六花}}
: 亞莉澤的侍女。
; {{nihongo|千鳥杏子|千鳥 杏子（ちどり きょうこ）|聲：早瀨彌生}}
: 意之杜學園的新人教師，對和希來說像是姐姐一樣。
