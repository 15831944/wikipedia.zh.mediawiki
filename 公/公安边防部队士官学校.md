'''公安边防部队士官学校'''，简称'''边防士官学校'''，是[[公安边防部队|公安边防部队]]培养士官学校，隶属[[中华人民共和国公安部|中华人民共和国公安部]]，由[[公安部边防管理局|公安部边防管理局]]管理，[[正师级|正师级]]<ref name=xunli>{{Cite web |url = http://www.legaldaily.com.cn/police_and_frontier-defence/content/2011-08/25/content_2901146.htm?node=23301 |title =育警砺剑——公安边防部队士官学校19年发展成就巡礼  |author =  |date =2011-08-25  |publisher = 法制网 |language =  |accessdate =  |quote =  }}</ref>。

== 沿革 ==
* 1992年9月，经[[中华人民共和国公安部|公安部]]批准，'''公安边防部队昆明指挥学校'''成立并招生办学<ref name=fzw/><ref name=xunli/>，隶属[[公安部边防管理局|公安部边防管理局]]，由云南省公安边防总队直管<ref name=xunli/>。
* 2004年2月，根据公安部边防管理局院校工作会议精神，学校担负的任务从培养部队生长干部调整为士官学历教育和入警大学生、直招士官、专业技术学兵及各类短期培训<ref name=fzw/>。
* 2010年11月1日，根据[[中央编办|中央编办]]《关于公安现役部队学校整合并核定编制的批复》（中央编办复字〔2009〕146号）和公安部政治部《关于组建公安边防部队士官学校的批复》（公政治〔2010〕385号）精神，在公安边防部队昆明指挥学校的基础上组建'''公安边防部队士官学校'''，由[[副师级|副师级]]升为[[正师级|正师级]]，直属公安部，由公安部边防管理局管理。2011年8月23日正式揭牌，公安部边防管理局副局长[[周书奎|周书奎]]出席揭牌仪式并与[[云南省公安厅|云南省公安厅]]党委副书记、常务副厅长[[罗石文|罗石文]]一同为学校揭牌。这是公安边防部队唯一一所专门培养士官人才和专业技术研究人才的现役院校<ref>{{Cite web |url =http://news.ifeng.com/gundong/detail_2011_08/23/8618847_0.shtml  |title =公安边防部队士官学校在昆明揭牌成立  |author =  |date =2011-08-23  |publisher =凤凰网  |language =  |accessdate =  |quote =  }}</ref><ref name=fzw>{{Cite web |url = http://www.legaldaily.com.cn/police_and_frontier-defence/content/2011-08/24/content_2908841.htm?node=23290 |title =公安边防部队士官学校在昆明正式成立  |author =  |date = 2011-08-24  |publisher = 法制网 |language =  |accessdate =  |quote =  }}</ref><ref name=xunli/>。
