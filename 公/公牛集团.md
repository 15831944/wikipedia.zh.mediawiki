'''公牛集团'''，简称'''公牛'''，是[[中国大陆|中国大陆]]内生产插座[[开关|开关]]的著名电器公司<ref>{{Cite web |url = http://www.gongniu.cn/about/about.aspx?id=1 |title = 公司简介 |author = 公牛集团 |date =  |publisher =  |language =中文 |accessdate = 2014-02-14 |quote =  }}</ref>，并拥有多家子公司。其总部设立在浙江省慈溪市观海卫镇工业园东区。

== 企业发展 ==

2018年9月28日公牛集團提交的IPO招股書指稱，公司將發行不超過6000萬股的股票，此次發行的股份占發行後公牛集團總股本不低於10%，募資人民幣48.86億元。招股書顯示2017年，公牛集团营收72.4亿元，净利润12.8亿元

== 企业基金 ==

“公牛”手拉手奖教奖学基金会于2013年秋，在[[慈溪市观城中学|慈溪市观城中学]]设立。首届发放仪式共发放首届“公牛”手牵手奖教奖学基金28.35万元，受助师生达129人次<ref>{{Cite web |url = http://www.gongniu.cn/news/newsxy.aspx?id=122 |title = “公牛”手拉手奖教奖学基金会隆重成立 |author = 公牛集团 |date = 2013-09-16 |publisher =  |language = 中文 |accessdate = 2014-02-14 |quote = 首届发放仪式共发放首届“公牛”手牵手奖教奖学基金28.35万元，受助师生达129人次 |deadurl = yes |archiveurl = https://web.archive.org/web/20131029201318/http://www.gongniu.cn/news/newsxy.aspx?id=122 |archivedate = 2013-10-29 }}</ref>。
