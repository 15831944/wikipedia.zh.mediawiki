'''公安机关'''，或称'''公安部门'''，简称'''公安'''，是[[中华人民共和国|中华人民共和国]]中央和地方各级[[人民政府|人民政府]]管理的负责[[公共安全|公共安全]]事务的[[行政|行政]][[执法|执法]]部门，是'''[[公安机关人民警察|公安机关人民警察]]'''的管理机构。但也行使部分非警务的行政许可职能，如户籍管理、特种行业管理等。

== 职责权限 ==
# 预防、制止和侦查违法[[犯罪|犯罪]]活动；
# 防範、打击[[恐怖活动|恐怖活动]]；
# 维护社会治安秩序，制止危害社会治安秩序的行为；
# 管理[[交通|交通]]、[[消防|消防]]、[[危险品|危险品]]；
# 管理[[户籍|户籍]]、[[居民身份证|居民身份证]]、[[国籍|国籍]]、入境事务和外国人在中国境内居留、[[旅行|旅行]]的有关事务；
# 管理[[边境|边境]]治安秩序；
# 警卫国家规定的特定人员、守卫重要场所和设施；
# 管理集会、[[游行|游行]]和[[示威|示威]]活动；
# 监督管理[[公共信息网络安全监察|公共信息网络安全监察]]工作；
# 指导和监督国家机关、社会团体、企业事业组织和重点建设工程的治安保卫工作，指导治安保卫委员会等群众性治安保卫组织的治安防範工作

== 规章制度 ==
=== 编制 ===
* [[中华人民共和国国务院|国务院]]设'''[[中华人民共和国公安部|公安部]]'''
** [[省_(行政區劃)|省]]、[[自治区|自治区]]设置'''公安厅'''；
*** 市（包括[[直辖市|直辖市]]、[[副省级城市|副省级城市]]、[[地级市|地级市]]、[[县级市|县级市]]）、[[县|县]]、[[旗_(內蒙古)|旗]]设'''公安局'''，[[盟|盟]]、[[地级行政区|地区]]设'''公安处'''；
**** [[市辖区|市辖区]]设'''公安分局'''；
***** 县级市、县、旗公安局和市辖区公安分局下设'''[[公安派出所|公安派出所]]'''。

另外，[[交通|交通]]、[[林业|林业]]部门的公安机关和[[海关总署缉私局|海关总署缉私局]]等行业系统人民警察列入公安部序列，接受同级行业行政主管部门和公安部门的双重领导，同时也接受人民政府的监督。
