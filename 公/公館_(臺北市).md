'''公館'''為[[台灣|台灣]][[台北市|台北市]][[中正區_(臺北市)|中正區]]、[[大安區_(臺北市)|大安區]]、[[文山區|文山區]]地名（1990年[[臺北市行政區劃|臺北市行政區劃]]調整前，原屬[[古亭區|古亭區]]、[[景美區|景美區]]轄境）。以[[台北捷運新店線|台北捷運新店線]][[公館站|公館站]]為中心，區內主要地標有[[國立台灣大學|國立台灣大學]]，以及附近的[[國立台灣科技大學|國立台灣科技大學]]、[[國立台灣師範大學|國立台灣師範大學]]公館校區（位於文山區）、[[水源市場|水源市場]]、台北[[自來水園區|自來水園區]]和[[台電大樓|台電大樓]]，商業發達。

==沿革==
[[清朝|清]][[乾隆|乾隆]]時有大批[[泉州|泉州]][[安溪|安溪]]移民入墾此地，墾戶首領乃於此築公館，以徵[[佃農|佃農]]租榖、防備[[原住民|原住民]]侵犯，「公館」地名即因此而來。最初的位址與今日大不相同，位於今捷運公館站東南方約1公里，[[景美|景美]]地區[[景美溪|景美溪]]匯入[[新店溪|新店溪]]處東岸至[[蟾蜍山|蟾蜍山]]山麓的區域<ref>{{cite web |title=公館街 |url=http://gn.moi.gov.tw/GeoNames/GNMap/GeoNamesSearchPrint.aspx?ID=0063008-081 |website=中華民國內政部地名資訊服務網 |accessdate=2019-03-29 |quote=景美溪會入新店溪的東岸至蟾蜍山間的地域。乾隆元年（西元一七六○年）大批泉州府安溪移民團入墾，逐先墾者與土著族。隨著內湖陂、璢公圳、大坪林圳之陸續鑿成，漢人移民深入新店溪及支流，沿岸谷地墾地日擴。墾首乃於此築公館以徵收佃戶租穀，因地當新店溪流域出入臺北盆地之要口，發展成為街肆，故稱公館街。}}</ref>，該處現在仍有一條道路名曰「公館街」。昔日屬[[文山堡|文山堡]][[萬盛庄|萬盛庄]]，[[台灣日治時期|台灣日治時期]]後期改隸[[文山郡|文山郡]][[深坑庄|深坑庄]][[萬盛_(大字)|萬盛]]下轄之公館及頂公館<ref>{{cite web |title=頂公館 |url=http://gn.moi.gov.tw/GeoNames/GNMap/GeoNamesSearchPrint.aspx?ID=0063008-070 |website=中華民國內政部地名資訊服務網 |accessdate=2019-03-29 |quote=今公館街附近。早期由此至蟾蜍山下之公館，均稱為公館，景美居民為了區別起見，就將地稱為頂公館，以區分下公館。}}</ref>（今文山區[[景美|景美]]次分區萬年-{里}-<ref>{{cite web |title=日據時期住所番地與現行行政區對照表 |url=https://www.ris.gov.tw/app/portal/220 |website=中華民國內政部戶政司 |accessdate=2019-03-29 |page=A53}}</ref>）。

日治時期私鐵[[臺北鐵道株式會社|臺北鐵道株式會社]]興建[[萬華車站|萬華車站]]至[[新店站|新店車站]]的鐵道時，即在[[臺北市_(州轄市)|臺北市]]南緣靠近[[文山郡|文山郡]]公館處設有[[公館站#臺鐵公館車站|公館車站]]，而更北邊的[[臺北帝國大學|臺北帝國大學]]（今臺灣大學）亦設有[[水源地車站|水源地車站]]，兩站距離不足1公里。戰後[[臺灣鐵路管理局|臺灣鐵路管理局]]買收此線，成為[[臺鐵新店線|臺鐵新店線]]。

[[冷戰|冷戰]]時期的公館開始有軍事設施進駐，蟾蜍山上設立了[[中華民國空軍|中華民國空軍]][[空軍作戰指揮部|作戰指揮部]]所在地「福興營區」，台大校園南側也劃為[[駐台美軍|駐台美軍]]的台北通訊站（{{lang|en|Taipei Air Station}}），當作[[美國空軍第327航空師|美國空軍第327航空師]]等部隊的指揮部。

1965年鐵路新店線廢止、大台北[[自來水|自來水]]取水口往上游直潭移動，[[水道町|水源地]]一名與公館逐漸混淆。捷運系統興建時，本預定於在原新店線水源地車站附近設立台灣大學站、公館車站附近設立公館站，後考量兩站過近併為公館一站，而水源地之地名亦逐漸併入公館。不過今日在[[捷運公館站|捷運公館站]]旁，仍留有「[[水源市場|水源市場]]」、臺大「[[國立台灣大學水源校區|水源校區]]」、「思源路」與市定古蹟「[[臺北水道水源地|臺北水道水源地]]」作為歷史的見證。

==現況==
[[File:Ting-chou_Road,_Section_3,_Taipei_City_20120428.jpg|缩略图]]
公館地區處於[[台北市|台北市]]數條幹道－[[羅斯福路_(台北市)|羅斯福路]]（省道[[台9線|台9線]]）、[[新生南路|新生南路]]及[[基隆路|基隆路]]的交會處，因此成為台北南區的[[景美|景美]]、[[新店區|新店]]、大坪林等地進入台北的交通要衝，往北通向[[中正區_(臺北市)|台北城區]]、往東北可至[[信義區_(臺北市)|信義區]]、[[松山區_(臺北市)|松山區]]、往西就是[[中和區|中和]]、[[永和區|永和]]，因此成為[[台北公車|台北公車]]及[[台北捷運|捷運]]一個重要的轉運點，交通地位相當重要。

加上擁有[[台灣大學|台灣大學]]學生、[[台灣科技大學|台灣科技大學]]及[[台灣師範大學|台灣師範大學]]公館校區的消費市場，使當地各種商店包括[[小吃|小吃]]、[[餐馆|餐廳]]、服飾、[[書店|書店]]、體育用品等都甚為繁榮發達，在假日或各種不同的節日時，幾乎所有的商家都會把店布置得美輪美奐。

由於本商圈鄰近[[台灣大學|台灣大學]]，因而吸引許多特色書店或獨立書店設立在本商圈的巷弄間或大樓的二樓或地下室。
