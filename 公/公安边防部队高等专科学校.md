'''公安边防部队高等专科学校'''位于[[广东省|广东省]][[广州市|广州市]][[天河区|天河区]]奥体南路399号，隶属[[公安部边防管理局|公安部边防管理局]]，[[正师级|正师级]]。

== 沿革 ==
1991年，经[[中华人民共和国公安部|中华人民共和国公安部]]批准，筹建'''公安边防部队广州指挥学校'''。1994年6月28日组建，1995年9月1日开学。2003年，学校由广东省公安边防总队领导下的副师级编制，提升成[[公安部边防管理局|公安部边防管理局]]直接领导的正师级编制。公安边防部队广州指挥学校是培养大、中专初级指挥警官的学校，是面向全国[[公安边防部队|公安边防部队]]招生，培养基层指挥管理干部、培训中、高级领导的具有公安边防教育特色的大专院校<ref>{{Cite web |url = http://news.sina.com.cn/o/2004-06-28/09092926620s.shtml |title =广州边防指挥学校　建校十年硕果累累  |author =  |date = 2004-06-28 |publisher =新浪  |language =  |accessdate =  |quote =  }}</ref><ref name=guangzhou>{{Cite web|url =http://www.guangzhou.gov.cn/node_559/node_1070/2006/06/13/1150186456110482.shtml|title =公安武警边防部队广州指挥学校|author =|date =2006-06-13|publisher =中国广州|language =|accessdate =|quote =|deadurl =yes|archiveurl =https://web.archive.org/web/20070402143751/http://www.guangzhou.gov.cn/node_559/node_1070/2006/06/13/1150186456110482.shtml|archivedate =2007-04-02}}</ref><ref>{{Cite web |url = http://military.cnr.cn/ztch/gzzhxy/ |title =  专题：公安边防部队广州指挥学校|author =  |date = 2014-12-29 |publisher =   央广网|language =  |accessdate =  |quote =  }}</ref><ref>{{cite news |title=公安部副部长孟宏伟视察海南广东广西边防总队和广州边防指挥学校时强调——把公安边防部队建设成为公安机关的模范部队 |newspaper=边防警察报 |date=2005-05-13 }}</ref><ref>{{cite news |title=广州边防指挥学校派出警力9877人次——为广州亚残运会提供优质安检服务 |newspaper=人民公安报 |date=2010-12-25 }}</ref>。

1992年3月，公安部批准在新疆公安边防总队教导大队的基础上成立'''公安边防部队乌鲁木齐指挥学校'''<ref>{{Cite web |url =http://www.chinanews.com/df/2012/10-14/4246105.shtml  |title = 公安边防部队乌鲁木齐指挥学校20年培养警官6000名 |author =  |date =2012-10-14  |publisher =中国新闻网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url =http://www.legaldaily.com.cn/police_and_frontier-defence/content/2012-10/15/content_3900560.htm?node=23292  |title =公安边防部队乌鲁木齐指挥学校建校二十周年综述  |author =  |date =2012-10-15  |publisher =法制网  |language =  |accessdate =  |quote =  }}</ref>。1993年之前，组建'''公安边防部队呼和浩特指挥学校'''<ref>杨振海、孙俊棋、刘永文、宋蒙生，电针结合红外线灯照射治疗坐骨神经痛61例，针灸临床杂志1999年01期</ref>。

2009年，[[中央机构编制委员会办公室|中央机构编制委员会办公室]]做出《关于公安现役部队学校整合并核定编制的批复》，批准在公安边防部队广州指挥学校的基础上，整合乌鲁木齐边防指挥学校、呼和浩特边防指挥学校，筹建公安边防部队指挥学院，行政编制提升为副军级。2010年，公安部做出《关于组建公安边防部队指挥学院的批复》，决定在公安边防部队广州指挥学校基础上筹建公安边防部队指挥学院。2014年6月，[[中华人民共和国教育部|中华人民共和国教育部]]、中华人民共和国公安部和公安部边防管理局党委决定启动公安边防部队高等专科学校申办工作<ref name=xuchmin>{{Cite web |url = http://www.legaldaily.com.cn/Police/content/2014-12/29/content_5907257.htm?node=43348 |title =  公安边防部队广州指挥学校徐超敏校长访谈录|author =  |date = 2014-12-29 |publisher =法制网  |language =  |accessdate =  |quote =  }}</ref><ref>{{Cite web |url = http://military.cnr.cn/ztch/gzzhxy/zxxx/20141227/t20141227_517232875.shtml |title = 广州指挥学校发展20年：育人铸魂百炼塑精兵 |author =  |date =  2014-12-27|publisher =央广网  |language =  |accessdate =  |quote =  }}</ref>。

2015年10月17日，公安边防部队广州指挥学校、乌鲁木齐指挥学校、呼和浩特指挥学校完成历史使命，'''公安边防部队高等专科学校'''在广州揭牌成立。公安部现役工作办公室主任[[牟玉昌|牟玉昌]]在揭牌仪式上宣读教育部关于同意建立公安边防部队高等专科学校的函和学校领导任职命令，公安部边防管理局政委[[李乐民|李乐民]]、财政部国防司巡视员[[李军_(财政部)|李军]]、[[广东省公安厅|广东省公安厅]]副厅长[[郭少波|郭少波]]出席揭牌仪式并发表讲话<ref>{{Cite web|url =http://www.chinagabf.com/bfjcb/html/2015-10/19/content_77871.htm|title =公安边防部队高等专科学校揭牌成立|author =|date =2015-10-19|publisher =边防警察报|language =|accessdate =|quote =}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>。
