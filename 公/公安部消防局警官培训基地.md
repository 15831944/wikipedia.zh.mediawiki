'''公安部消防局警官培训基地'''，位于[[天津市|天津市]][[北辰区|北辰区]][[小淀镇|小淀镇]]赵东庄，直属[[公安部消防局|公安部消防局]]，为[[中华人民共和国军事级别#正师级|正师级]]单位，是对[[公安消防部队|公安消防部队]]中、高级警官进行培训和对社会各类相关人员开展职业培训的场所。

==简介==
公安部消防局警官培训基地位于天津市北辰区[[京津塘高速公路|京津塘高速公路]]旁边，距离天津市中心大约20公里，距离[[北京市|北京市]]大约120公里，距离[[渤海湾|渤海湾]]大约45公里，紧邻[[205国道|205国道]]<ref name=zaixian>{{Cite web |url = http://119.china.com.cn/xfyx/txt/2007-05/22/content_3681411.htm |title = 公安部消防局警官培训基地简介 |author =  |date = 2007-05-22 |publisher =中国消防在线  |language =  |accessdate =  |quote =  }}</ref>。

公安部消防局警官培训基地的前身是'''公安消防部队天津指挥学校'''。经[[中华人民共和国公安部|中华人民共和国公安部]]批准，于2003年12月26日正式成立'''公安部消防局警官培训基地'''，为正师级单位，隶属[[公安部消防局|公安部消防局]]直接领导。主要承担对公安消防部队中、高级警官的培训和对社会各类相关人员开展职业培训的职能。该培训基地占地面积260亩，建筑面积近5.5万平方米，划分为场地训练区、教学办公区、生活服务区<ref name=zaixian/>。
