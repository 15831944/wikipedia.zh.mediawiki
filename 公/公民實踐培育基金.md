公民實踐培育基金有限公司（{{Lang-en|Project Citizens Foundation Limited}}），於2014年8月成立，是一個[[非營利組織|非牟利組織]]，主張透過建立互動教育平台，傳承香港賴以成功的核心價值。

此基金主席為詹德隆先生，副主席為張玉堂先生，另有前政務司司長[[陳方安生|陳方安生]]女士及多名專業人士合作籌組。<ref>{{Cite web|url=http://news.now.com/home/local/player?newsId=142791|title=陳方安生成立公民實踐培育基金|accessdate=2017-08-31|work=news.now.com|language=zh-hk}}</ref><ref name=":0">{{Cite web|url=http://hk.apple.nextmedia.com/news/art/20150712/19216809|title=陳太成立基金　助年輕人發聲|accessdate=2017-08-31|work=Apple Daily 蘋果日報}}</ref>

== 成立 ==

=== 緣起 ===
基金董事認為「[[一国两制|一國兩制]]」，「[[高度自治|高度自治]]」及「港人治港」被負責香港事務的領導人不斷地重新演繹。同時間，近年多次[[社會運動|社會運動]]，反映香港社會撕裂，發生不少暴力事件，造成內耗。基金因此而成立，倡議包容、多元和溝通，令年輕人心平氣和討論問題，維護香港核心價值。

=== 使命 ===
透過積極參與，追求高質素的公民參與，把[[民主|民主]]、[[自由|自由]]，公平、公正和公義的核心價值傳承下去。

=== 目標 ===
# 深化公民教育 提升社會大眾的公民意識及擴闊其國際視野 貫徹[[民主|民主]]、[[自由|自由]]、[[人权|人權]]、[[平等|平等]]和[[法治|法治]]等概念 推廣可持續的經濟發展模式和環境保護及保育
# 宣揚及推廣公民實踐
# 培養年輕人的獨立思考及領導才能，讓他們發展潛能，成為未來的社會領袖，貢獻香港
# 建立平台，供年輕人與不同意見的人士交流和分享，從而擴闊視野

==== 邀請彭定康來港出席論壇 ====
'''2016'''年'''11'''月'''26'''日，基金於九龍香格里拉大酒店舉辦名為「公民實踐論壇 - 香港管治：禮崩樂壞？」論壇，邀請前港督[[彭定康|彭定康]]、[[余若薇|余若薇]]女士、[[馬世民|馬世民]]先生及[[馬斐森|馬斐森]]校長作論壇嘉賓，從公共、企業及大學層面探討管治策略。論壇有超過五百人出席，當中包括各國大使館領事、立法會議員及當時受宣誓案纏身的[[梁頌恆|梁頌恆]]。<ref>{{Cite web|url=http://hk.apple.nextmedia.com/realtime/news/20161126/55968001|title=【肥彭駕到】梁頌恆：尊重彭定康說法 英有道德責任處理港「麻煩」|accessdate=2017-08-31|work=Apple Daily 蘋果日報}}</ref>

[[彭定康|彭定康]]於席間提到法治是香港的自由、穩定和福祉最重要的保證，對香港至為重要。另外，他亦強調政治穩定對香港繁榮非常重要，政府必須明白到人民是公民，有他們的權利和責任，並要敏銳地回應人民的期望。<ref>{{Cite web|url=http://hk.apple.nextmedia.com/realtime/news/20161126/55967415|title=【肥彭駕到】彭定康：依法管治不等同法治 當權者都要守法|accessdate=2017-08-31|work=Apple Daily 蘋果日報}}</ref>
