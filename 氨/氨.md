'''氨'''<ref>（拼音：ān）{{標音|拼音=ān|注音=ㄢ|粤拼=|中古=|反切=|同音字=安}}</ref>（{{lang-en|Ammonia}}，或称'''氨氣'''、'''無水氨'''，曾音譯作'''𠼞'''、'''氬'''、'''阿摩尼亞'''，分子式为NH<sub>3</sub>）是无色气体，有强烈刺激气味（[[尿|尿]]味），极易溶于水。常温常压下，1單位体积水可溶解700倍体积的氨。<ref name="hellochem.com">{{Zh icon}}{{cite web|url=http://www.hellochem.com/xz/xz1/672hjybs.htm= | title=氨;氨气;ammonia |publisher=化工引擎 |accessdate=2008-05-06}}</ref>氨對地球上的[[生物|生物]]相當重要，是所有食物和[[肥料|肥料]]的重要成分。氨也是很多[[藥物|藥物]]和商業清潔用品直接或间接的組成部分，具有[[腐蝕|腐蝕]]性等危險性质。

由於氨有廣泛的用途，成為世界上產量最多的無機化合物之一，約八成用於製作化肥。2006年，氨的全球產量估計為1.465億吨，主要用於製造商業清潔產品。

氨可以提供[[孤電子對|孤電子對]]，所以也是[[路易斯鹼|路易斯鹼]]。

== 氨水 ==
{{main|氨水}}
'''氨水'''（NH<sub>3</sub>[aq] 或者 NH<sub>4</sub>OH）又稱為'''阿摩尼亞水'''，指氨的[[水溶液|水溶液]]，有強烈刺鼻氣味，具弱[[鹼性|鹼性]]。

氨水中，氨氣分子發生微弱[[水解|水解]]生成[[氢氧根|氫氧根]]離子及[[铵根|銨根]]離子。「氫氧化銨」事實上並不存在，只是對氨水溶液中的[[离子|離子]]的描述，並無法從溶液中分離出來。

氨的在水中的電離可以表示為：

:<chem>NH3*H2O = {NH4}^{+} + OH-</chem>

反應平衡常數<math>K_b=1.8 \times 10^{-5}</math>。

1M氨水的[[pH|pH]]值為11.63，大約有0.42%的NH<sub>3</sub>變為NH<sub>4</sub><sup>+</sup>。

氨水是實驗室中氨的常用來源。它可與含銅(II)離子的溶液作用生成深藍色的[[配合物|配合物]]，也可用於配置[[银氨溶液|銀氨溶液]]等[[分析化学|分析化學]]試劑。

== 用途 ==
氨水可被土中的土壤膠體吸附和被作物吸收，無殘留物質，適用於各種土壤和作物。

== 氨的合成 ==
1774年，化學家[[普利斯特里|普利斯特里]]加熱[[氯化銨|氯化銨]]和[[氫氧化鈉|氫氧化鈉]]的混合物，利用排汞取氣法取得氨。

[[第一次世界大戰|第一次世界大戰]]以前，大部分的氨都是以[[乾餾|乾餾]]<ref>[http://nobelprize.org/chemistry/laureates/1918/press.html Nobel Prize in Chemistry (1918)] - [[Haber_process|Haber process]]. URL last accessed April 24 2006</ref> 含氮的蔬菜及動物的[[糞便|糞便]](如駱駝糞)，並以氫作為還原劑以把亞硝酸及亞硝酸鹽還原而製成。除此以外，氨也可以在[[煤|煤]]的乾餾或用[[銨鹽|銨鹽]]与氫氧化物(如[[氫氧化鈣|氫氧化鈣]]，即熟石灰)<ref>[http://www.bbc.co.uk/dna/h2g2/A1002934 BBC.co.uk] URL last accessed April 24 2006</ref> 共热製得，所使用的銨鹽普遍為[[氯化銨|氯化銨]]。

::<chem>2NH4Cl + 2CaO = CaCl2 + Ca(OH)2 + 2NH3 ^</chem>

現今的工廠大多使用'''[[哈伯法|哈伯法]]'''{{Fact|time=2015-07-24T22:49:05+00:00}}：
在200[[大氣壓力|大氣壓力]]和500℃的条件下，以[[氧化鐵|氧化鐵]]為[[催化劑|催化劑]]，加熱[[氮氣|氮氣]]和[[氫氣|氫氣]]制得。

:<chem>N2 + 3H2 <=> 2NH3</chem>

<math>K_\mathrm{eq} = \mathrm{\frac{[NH_3]^2}{[N_2][H_2]^3}}</math>

這個反應是可逆的。在25℃時[[平衡常數|平衡常數]]為6.4×10<sup>2</sup>，在500℃時為1.5×10<sup>−5</sup>。

合成氨的原料[[氮|氮]]氣來自於[[空氣|空氣]](以液態空氣的分餾取得)，[[氫氣|氫氣]]來自於[[水|水]]和[[燃料|燃料]]。由於化石燃料短缺， 製氨用的[[氫|氫]]理論上可以用水的[[電解|電解]] (現今4%的[[氫|氫]]由電解製備)或熱化裂解(thermal chemical cracking)製得，但現在來說，這些方法都是不實際的。熱裂解所需的熱能可以從核能反應中取得，而[[風力發電|風力發電]]、[[太陽能發電|太陽能發電]]及[[水力發電|水力發電]]產的的過剩電能可以用來電解水製氫。現在為止，以空氣及燃料製氨的方法以外的替代方案是不經濟的，而且這些方法對環保的作用仍未有定论。

== 用途 ==
* 由於氨擁有强烈的刺激性氣味，在医疗方面，會用少量易於揮發的氨作為使人清醒的吸入劑。
* 生產[[硝酸|硝酸]]
* [[玻璃|玻璃]]清潔劑
* 有八成的氨生產[[氮肥|氮肥]]
* [[航空燃料|航空燃料]]（[[X-15|X-15]]）
* 氨是最广泛用的[[制冷剂|制冷剂]]之一，可用于空调、冷藏和低温，能用于各种形式的制冷压缩机，蒸发温度可控制在5度至零下65度，代號R717。

== 反应 ==
=== 络合反应 ===
NH<sub>3</sub>分子中氮原子有一对[[孤对电子|孤对电子]]，可以作为电子对给予体（[[路易斯碱|路易斯碱]]）形成加合物。如氨在氢离子络合生成铵离子：

:<chem>NH3 + H+ = {NH4}^{+}</chem>

NH<sub>3</sub>亦可与金属离子如Ag<sup>+</sup>、Cu<sup>2+</sup>等发生錯合，生成[[錯合物|錯合物]]：

:<chem>Ag+ + 2NH3 = [Ag(NH3)2]+</chem>
:<chem>{Cu}^{2+} + 4NH3 = {[Cu(NH3)4]}^{2+}</chem>

=== 氧化还原 ===
NH<sub>3</sub>分子中氮为-3价，在适当条件下可被氧化为N<sub>2</sub>或更高价氮化合物。

如NH<sub>3</sub>在纯氧中燃烧，生成N<sub>2</sub>：

:<chem>4NH3 + 3O2 = 2N2 + 6H2O</chem>([[標準生成焓|Δ''H''º<sub>r</sub>]] = –1267.20 kJ/mol)

在铂催化下可氧化生成水与[[一氧化氮|一氧化氮]]，是工业制硝酸的重要反应。

:<chem>4NH3 + 5O2 = 4NO + 6H2O</chem>

可还原CuO为Cu：

:<chem>2NH3 + 3CuO = N2 + 3H2O + 3Cu</chem>

常温下NH<sub>3</sub>可与强[[氧化剂|氧化剂]]（如[[氯气|氯气]]、[[过氧化氢|过氧化氢]]、[[高锰酸钾|高锰酸钾]]）直接反应：

:<chem>2NH3 + 3Cl2 = N2 + 6HCl</chem>

== 液氨 ==
{{main|液氨}}
'''液氨'''（NH<sub>3</sub>）指的是液態的氨，為工業上氨氣的主要儲存形式。是一種常用的非水溶劑和[[致冷剂|致冷劑]]，也是除了[[水|水]]以外最常用的無機溶劑。不過由於它的揮發性和腐蝕性，液氨在儲存和運輸時發生事故的幾率也相當高。
