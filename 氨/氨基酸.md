'''胺基酸'''是[[生物|生物]]學上重要的[[有機化合物|有機化合物]]，由[[胺基|胺基]]（-NH<sub>2</sub>）和[[羧基|羧基]]（-COOH）的[[官能團|官能團]]，以及连到每一個胺基[[酸|酸]]的[[取代基|側鏈]]組成。胺基酸是構成[[蛋白質|蛋白質]]的基本單位。賦予蛋白質特定的分子結構形態，使其分子具有[[生化活性|生化活性]]。蛋白質是生物体重要的[[活性分子|活性分子]]，包括催化[[新陳代謝|新陳代謝]]的[[酶|酶]]（又称“酵素”）。

不同的胺基酸[[脱水缩合|脱水缩合]]形成[[肽|肽]]（蛋白質的原始片段），是蛋白質生成的[[前體|前體]]。

== 基本結構 ==
胺基酸为分子结构中含有[[氨基|氨基]]（-NH<sub>2</sub> ）和[[羧基|羧基]]（-COOH），并且氨基和羧基都是直接连接在一个-CH-结构上的有机化合物。通式是H<sub>2</sub>NCHRCOOH(R代表某種有機[[取代基|取代基]])。根据氨基连结在羧酸中碳原子的位置，可分为''α''、''β''、''γ''、''δ''……等等的氨基酸( C……C-C-C-C-COOH)。

蛋白质经[[水解|水解]]後，即生成20种氨基酸，如[[甘氨酸|甘氨酸]]（Glycine）、[[丙氨酸|丙氨酸]]（Alanine）、[[缬氨酸|缬氨酸]]（Valine）、[[亮氨酸|亮氨酸]]（Leucine）、[[异亮氨酸|异亮氨酸]]（Isoleucine）、[[苯丙氨酸|苯丙氨酸]]（Phenylalanine）、[[色氨酸|色氨酸]]（Tryptophan）、[[酪氨酸|酪氨酸]]（Tyrosine）、[[天冬氨酸|天冬氨酸]]（Aspartate）、[[组氨酸|组氨酸]]（Histidine）、[[天冬酰胺|天冬酰胺]]（Asparagine）、[[谷氨酸|谷氨酸]]（Glutamate）、[[離胺酸|赖氨酸]]（Lysine）、[[谷氨酰胺|谷氨酰胺]]（Glutamine）、[[甲硫氨酸|甲硫氨酸]]（Methionine）、[[精氨酸|精氨酸]]（Arginine）、[[丝氨酸|丝氨酸]]（Serine）、[[苏氨酸|苏氨酸]]（Threonine）、[[半胱氨酸|半胱氨酸]]（Cysteine）、[[脯氨酸|脯氨酸]]（Proline）等。

== 分類 ==

根据其结合[[基团|基团]]不同。可分为[[脂肪族|脂肪族]]胺基酸、[[芳香族|芳香族]]胺基酸、[[杂环|杂环]]胺基酸、無環胺基酸(acyclic)、[[鹼|鹼]]性胺基酸含[[硫|硫]]胺基酸、含[[羥基|羥基]](醇)胺基酸、含[[碘|碘]]胺基酸等。

== 理化特性 ==
[[File:Peptidformationball.svg|thumb]]
# 都是无色[[结晶|结晶]]。[[熔点|熔点]]约在230℃以上，大多没有确切的熔点，[[熔融|熔融]]时[[分解反应|分解]]并放出[[二氧化碳|CO<sub>2</sub>]]；都能溶于[[强酸|强酸]]和[[强碱|强碱]][[溶液|溶液]]中，除[[胱胺酸|胱胺酸]]、[[酪胺酸|酪胺酸]]、[[二碘甲状腺素|二碘甲状腺素]]外，均溶於水；除[[脯胺酸|脯胺酸]]和[[羟脯胺酸|羟脯胺酸]]外，均难溶於[[乙醇|乙醇]]和[[乙醚|乙醚]]。
# 具有[[兩性_(化學)|两性]]。有碱性[二元胺基一元羧酸，例如赖胺酸（lysine）]；酸性[一元胺基二元羧酸，例如穀胺酸（Glutamic acid）]；中性[一元胺基一元羧酸，例如丙胺酸（Alanine）]三種類型。大多数胺基酸都呈显不同程度的[[酸性|酸性]]或[[碱性|碱性]]，呈显中性的较少。所以既能与酸结合成[[盐|盐]]，也能与碱结合成盐。
# 由于有不对称的碳原子，呈[[旋光性|旋光性]]（除甘氨酸外）。同时由于空间的排列位置不同，又有两种构型：[[手性|D型和L型]]，组成天然蛋白质的胺基酸，都属L型。 由于以前胺基酸来源于蛋白质水解（现在大多为人工合成），而天然蛋白质水解所得的胺基酸大多为α-氨基酸，所以在生化研究方面胺基酸通常指α-胺基酸。至于β、γ、δ……ω等的胺基酸在生化研究中用途较小，大都用于[[有机合成|有机合成]]、[[石油化工|石油化工]]、[[医疗|医疗]]等方面。胺基酸及其[[衍生物|衍生物]]品种很多，大多性质稳定，要避光、干燥贮存。

== 必需氨基酸 ==
{{main|必需氨基酸}}
[[人體|人體]]能[[消化|消化]]吸收以及利用的胺基酸只有20種。其中有9種氨基酸（嬰兒為10種）是成人体内不能合成或合成速度不能满足机体的需要，必须从[[膳食|膳食]]補充的氨基酸稱為必需氨基酸（EAA），即[[亮氨酸|亮氨酸]]、[[異亮氨酸|異亮氨酸]]、[[纈氨酸|纈氨酸]]、[[甲硫氨酸|甲硫氨酸]]、[[苯丙氨酸|苯丙氨酸]]、[[色氨酸|色氨酸]]、[[苏氨酸|苏氨酸]]、[[賴氨酸|賴氨酸]]、[[組氨酸|組氨酸]]，精氨酸为小儿生长发育期间的必需氨基酸，胱氨酸、酪氨酸、牛磺酸为早产儿所必需。

肉類中的[[蛋白質|蛋白質]]是完全蛋白質，可以提供人體所需的全部[[胺基酸|胺基酸]]種類，瘦豬肉的蛋白質含量約為10%至17%，肥豬肉則只有2.2%；瘦牛肉為20%左右，肥牛肉為15.1%；瘦羊肉17.3%，肥羊肉9.3%；兔肉21.2%；雞肉23.3%；鴨肉16.5%；鵝肉10.8%。肉類的蛋白質經過烹調，有一部分會散在肉湯中，也有一部分水解成[[胺基酸|胺基酸]]，溶於肉湯裡，故烹調好的肉湯味道鮮美而富於營養。其他13種非必需胺基酸可以用[[葡萄糖|葡萄糖]]或是別的[[矿物质|矿物质]]來源製造。
