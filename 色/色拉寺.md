'''色拉寺'''（{{bo|z=སེ་ར་དགོན|w=se ra dgon}}）位于[[西藏|西藏]][[拉萨|拉萨]]市区北郊的乌孜山南麓，是[[藏传佛教|藏传佛教]][[格鲁派|格鲁派]]六大寺院之一，由[[宗喀巴|宗喀巴]]的弟子“大慈法王”[[绛钦却杰|绛钦却杰]]在公元15世纪初创建。1982年，色拉寺被[[中华人民共和国国务院|中华人民共和国国务院]]公布为第二批[[全国重点文物保护单位|全国重点文物保护单位]]之一。

== 布局 ==
1419年，[[格鲁派|格鲁派]]的始祖[[宗喀巴|宗喀巴]]命弟子[[绛钦却杰|绛钦却杰]]在[[拉萨|拉萨]]北郊的乌孜山南麓创建寺院，命名为“秦清林”，[[藏语|藏语]]意为“大乘洲”。因附近多“色拉”（藏语意为酸枣林），故又通称色拉寺。建寺后不久，宗喀巴即去世。因为之前他还兴建了[[甘丹寺|甘丹寺]]和[[哲蚌寺|哲蚌寺]]，故此人们将三寺并称为“拉萨三大寺”。

色拉寺的规模宏大，依山就势而建，寺前有流沙河流淌而过。寺院占地约11.5万平方米，建筑以措钦（集会）大殿、麦扎仓、吉扎仓、阿巴扎仓等主体建筑为中心，穿插布置了数十座康村（僧舍村落，由一群同鄉的僧人所組成）。虽然建造时并没有进行过规划布局，但由于建筑的外观颜色一致，主体建筑形体高大，因而具有统领全局的作用，使整座寺院显得主次有序。寺内所有屋舍均为石木结构，屋顶覆[[阿嘎土|阿嘎土]]，白色外墙的上部装饰紫黑色[[贝玛草|贝玛草]]，具有浓郁的藏式风格。

== 建筑 ==
{{藏传佛教}}

位于寺院东北部的措钦大殿是全寺的管理中心和主要集会场所，也是寺内最大的殿堂。它建于1710年，平面为方形，由殿前广场、经堂和五座拉康（佛殿）组成，占地面积约2000多平方米。经堂高2层，中部用长柱顶起为天窗，可以采光。四周为短柱，构成相对低矮的空间，用以供奉佛像。经堂的后部有三座佛殿，居中者供奉高6米的[[强巴佛|强巴佛]]（[[弥勒佛|弥勒佛]]）鎏金铜像，其余供[[文殊菩萨|文殊菩萨]]、宗喀巴师徒三尊、绛钦却杰、十一面千手[[观世音菩萨|观世音菩萨]]等尊像。大殿的殿顶为汉式风格的歇山式顶，覆以鎏金铜瓦，装饰宝盘、宝珠、神鸟、宝幢等。

色拉寺中最大的扎仓（经学院）是吉扎仓，建于1435年，18世纪初进行过扩建，目前面积约1700平方米。它的经堂由100根柱子支撑，殿内密布着壁画和唐卡。在经堂的西部和北部建有五座佛殿，内有许多[[活佛|活佛]]灵塔和造像。比吉扎仓略小一点的是麦扎仓，建于1419年，后被雷火烧毁，1761年重建，现面积1600多平方米。<ref>{{Cite web | url=http://www.thlib.org/places/monasteries/sera/#!essay=/cabezon/sera/intro/ | title=Introduction to Sera Monastery |author=José Ignacio Cabezón，[[加利福尼亞大學聖塔芭芭拉分校|加州大學聖塔芭芭拉分校]]達賴喇嘛講座教授}}{{en}}</ref>

阿巴扎仓是寺内唯一的[[密宗|密宗]]扎仓，建于1419年，面积约1500平方米。它原为措钦大殿，当现大殿建成后，便改为扎仓。阿巴扎仓的主体建筑高2层，由经堂和四座佛殿组成。底层经堂的西墙为通顶的大经架，其北有石塔，后部有两座佛殿。二层为绛钦却杰的灵塔殿，平面呈长方形，由6根柱子支撑，柱高仅2米。该殿的东、西、北三面设佛台，北面佛台的正中供奉绛钦却杰和色拉寺第二任主持[[绛才桑布|绛才桑布]]的灵塔。

[[1959年西藏騷亂|1959年西藏騷亂]]時，色拉寺遭到嚴重破壞，部分色拉寺僧侶流亡到印度，1970年197位色拉傑僧侶與103位色拉昧僧侶在達賴喇嘛和印度中央和省政府的協助下，遷移到印度南部的[[卡納塔克邦|卡納塔克邦]][[拜拉库比|拜拉库比]]難民區內，在森林地上重建色拉寺，每位僧人分得3/4英畝，所以色拉傑札倉佔地147.75英畝，色拉昧札倉佔地77.25英畝。<ref>{{Cite web | url=http://selajiesi.org/history/183-re-establishment-at-bylakuppe | title=在貝拉闊北（Bylakuppe）重建| website=色拉傑寺院}}</ref>

1959年之前，色拉傑札倉原有21棟康村，在印度重建後有14棟康村。<ref>{{Cite web | url=http://selajiesi.org/khangtsen|title=康村| website=色拉傑寺院}}</ref>色拉昧札倉原有16棟康村，在印度重建後有13棟，2015年共有1,755名僧侶。<ref>{{Cite web | url=http://www.thetibetpost.com/zh/news/exile/4852-his-holiness-the-dalai-lama-inaugurates-new-sera-pobhor-assembly-hall | title=達賴喇嘛尊者為色拉寺繃波經堂開光加持 |date=2015年12月16日}}</ref><ref name="見悲青增">{{cite web | url=http://enlight.lib.ntu.edu.tw/FULLTEXT/JR-BJ013/bj013133096.pdf | format=PDF|title=我所認識的色拉寺及其分院色拉傑| author=拉然巴[[格西|格西]]見悲青增}}</ref>{{rp|2-3}}

== 其他 ==

色拉寺内有佛像近万座，此外壁画、典籍、唐卡（佛教卷轴画）、经幡、挂幢等也都遍布各殿。寺内还藏有[[明朝|明朝]]皇帝赠送的典籍、佛像和法器。其中最重要的是一套用[[朱砂汁|朱砂汁]]书写的藏汉文对照《[[甘珠尔|甘珠尔]]》（[[大藏经|大藏经]]），乃明成祖馈赠绛钦却杰之物。这是世界上第一部用[[藏文|藏文]]书写的佛经版本，在[[藏传佛教|藏传佛教]]界和[[版本学|版本学]]领域内都具有非凡的价值。此外如金泥书写的《[[大般若经|大般若经]]》、[[白檀木|白檀木]]雕刻的十六尊者像、金汁绘就的[[释迦牟尼|释迦牟尼]]转法轮唐卡和大慈法王缂丝像等也都是弥足珍贵之物。

绛钦却杰曾被[[明成祖|明成祖]]和[[明宣宗|明宣宗]]封为“大国师”和“大慈法王”，因此由他创建的色拉寺也享有了崇高的声望。17世纪寺院全盛时僧员定额为5500人，仅次于哲蚌寺。到1959年时，寺内有僧侣9000名。1947年，色拉寺下属[[热振寺|热振寺]]的[[第五世热振呼图克图·图旦绛白益西丹巴坚赞|五世热振活佛]]被[[噶厦|噶厦]]政府逮捕，吉扎仓的僧人为此组织了武装试图营救。他们与政府军在拉萨的北郊发生战斗，因寡不敌众，而被迫返寺。此事件后来被称为“[[热振事变|热振事变]]”。

色拉寺喇嘛的一项为人熟知的功课是[[辩经|辩经]]。实际上这并非色拉寺独有，只是色拉寺的辩经功课每周一至周五下午固定进行，且向游人开放，故最为著名。比平时的辩经正式得多的是定期举行的辩经法会，由高僧们展开关于佛法的讨论乃至辩论。而每日的辩经，则是众喇嘛参悟奥义、提高修为的一种辅助方法。

现在的喇嘛也并未与世俗生活完全隔绝，他们需要参加一定的劳动，色拉寺周围种植的大片果林也需要他们照料。喇嘛们有时也会进行一些世俗娱乐。2002年时色拉寺的喇嘛甚至成立了一支[[足球|足球]]队，并与一些学校的球队打比赛。为了统一服装，队员们大多购买了[[葡萄牙國家足球隊|葡萄牙國家足球隊]]的球衣，因为其颜色与喇嘛们穿的僧袍比较接近。
