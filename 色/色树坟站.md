'''色树坟站'''，位于[[北京市|北京市]][[门头沟区|门头沟区]][[王平鎮|王平鎮]]色树坟村境内，是[[五等站_(中国大陆)|五等站]]。目前该车站已不办理客运业务。


==简介==
色树坟站是京门铁路区段中间的作业站。色树坟村因原来有座王家坟，坟地里长了一棵色树而得名。<ref>{{cite web |title=京门小票车门头沟穿越百年历史-搜狐旅游 |url=http://travel.sohu.com/20100621/n272957683_1.shtml}}</ref>

==历史==
王平镇过去是京西古镇，并且产煤，一度十分繁华，京平铁路在此站岔出厂线专门运煤。<ref>{{cite web |author1=VANGE |title=几乎全程走完的京门铁路（九）色树坟-清水涧 |url=http://blog.sina.com.cn/s/blog_720f853f0101h0t8.html}}</ref>抗日战争时期，曾有日军警备队驻扎在此火车站。<ref>{{cite web |title=【色树坟小站】_纪实_POCO摄影,抗战,火车站,民俗,故事,色树坟 |url=http://www.poco.cn/works/detail?works_id=2952535&stat_request_channel=1445660563}}</ref>
