《'''太空救援'''》<ref>{{cite web|url=http://www.russia.org.cn/cn/news/6296/|title=影片《礼炮七号》正式预告片出炉|date = 2017-05-30 |publisher = 俄罗斯联邦驻华大使馆|language = zh-cn |accessdate = 2018-02-20 }}</ref><ref>{{cite web|url=http://www.russia.org.cn/cn/news/6691/|title=影片《礼炮七号》雄踞俄罗斯电影票房榜首|date = 2017-10-16 |publisher = 俄罗斯联邦驻华大使馆|language = zh-cn |accessdate = 2018-02-20 }}</ref>（{{lang-ru|''Салют-7''}}）是一部2017年上映的俄罗斯电影。由克里姆·斯彭科执导，是一部取材真实事件的剧情片。電影獲得俄羅斯金像獎的「最佳電影」及「最佳剪接」獎項。

==剧情==
1985年6月，苏联的“[[禮炮七號|禮炮七號]]”空间站突然和地面控制中心失去了联系，有可能坠落。恰逢美国的[[挑战者号航天飞机|挑战者号航天飞机]]即将发射（历史上执行[[STS-51-F|太空实验室2号]]任务的 挑战者号1985年7月29日发射升空）。为了防止礼炮七号落入美国手中，两位宇航员临危受命搭乘联盟T-13号飞船前去维修礼炮七号。
