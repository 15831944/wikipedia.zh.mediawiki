'''礼亲王府'''，位于中国[[北京市|北京市]][[西城区|西城区]][[西黄城根南街|西黄城根南街]]9号，是[[清朝|清朝]][[康亲王|康亲王]][[杰书|杰书]]及其后裔的府第。<ref name=qdwf>[http://tszyk.bucea.edu.cn/web/qdwf/sxwtwf/lqwf/ 礼亲王府，北京清代王府建筑与文化，于2013-02-26查阅]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

==历史==
礼亲王府位于[[西黄城根南街|西黄城根南街]]西北，[[大酱坊胡同|大酱坊胡同]]东口路北。该府所在地在[[明朝|明朝]]是[[崇祯帝|崇祯帝]]外戚[[周奎|周奎]]之宅，后为[[代善|代善]]所有。<ref name=qdwf/>

该府始建于清朝[[顺治|顺治]]年间，为[[礼亲王|礼亲王]][[代善|代善]]之孙[[杰书|杰书]]择地兴建。杰书袭爵后，封号为[[康亲王|康亲王]]，故该府最初称为“康亲王府”。[[乾隆|乾隆]]四十三年（1778年），杰书的裔孙[[永恩|永恩]]复号为“[[礼亲王|礼亲王]]”，故该府亦改称“礼亲王府”。<ref name=qdwf/>

在第九代礼亲王[[昭梿|昭梿]]时期，嘉庆十二年（1807年），该府毁于火灾。后来，经[[嘉庆帝|嘉庆帝]]赐银万两，由礼亲王[[昭梿|昭梿]]主持，在原址大体依照原式重建该府。<ref name=qdwf/>

1927年，礼亲王府的后人将该府前半部分出租给华北学院（后改称[[华北大学|华北大学]]）。[[中华人民共和国|中华人民共和国]]成立后，华北大学结束，该府改为[[中央人民政府内务部|中央人民政府内务部]]（后改称[[中华人民共和国民政部|中华人民共和国民政部]]）办公地。如今，该府中路建筑大部分保存，其余建筑多经拆改。1984年，该府被公布为[[北京市文物保护单位|北京市文物保护单位]]。<ref name=qdwf/>

礼亲王府的西黄城根南街9号院曾是中华人民共和国民政部、[[中华人民共和国国家进出口管理委员会|中华人民共和国国家进出口管理委员会]]、[[中华人民共和国国家农业委员会|中华人民共和国国家农业委员会]]、[[中共中央农村政策研究室|中共中央农村政策研究室]]（[[国务院农村发展研究中心|国务院农村发展研究中心]]）等机关单位的办公地点。1980年代，[[华国锋|华国锋]]、[[姚依林|姚依林]]和[[张劲夫|张劲夫]]迁入西黄城根南街9号院北区居住，南区则是各机关单位的办公地点<ref>{{Cite web |url =  http://news.sohu.com/20151010/n422855926.shtml |title = 杜润生病逝享年102岁 曾主持起草5个一号文件 |author =  |date = 2015-10-10 |publisher = 搜狐 |language =  |accessdate =  |quote =  }}</ref>。后来9号院由[[国家发展和改革委员会|国家发展和改革委员会]]基建物业管理中心管理<ref>{{Cite web |url =  http://www.ccgp.gov.cn/cggg/zygg/zbgg/201510/t20151019_5989228.htm |title = 国家发展和改革委员会西城区西黄城根南街9号院综合整治项目中标公告 |author =  |date = 2015-10-19 |publisher =  中国政府采购网|language =  |accessdate =  |quote =  }}</ref>。

== 建筑 ==
礼亲王府位于[[西黄城根南街|西黄城根南街]]西侧，即大酱坊胡同东口路北，南起大酱坊胡同，北到[[颁赏胡同|颁赏胡同]]。该府占地规模较广，主要建筑布局和其他清朝王府相同。在北京的清朝王府中，最大的便是礼亲王府和[[豫亲王府|豫亲王府]]，故老北京有[[谚语|谚语]]“礼王的房，豫王的墙”<ref name="qdwf" />。

该府建筑分为三路，其中中路为主体建筑，东路为住宅，西路为花园<ref name="qdwf" />。

中路的主要建筑分前后两组。前部有正门（宫门），正殿及其两侧翼楼，后殿及其两厢配殿。后部自成庭院，前为内门（二宫门）、前堂、后堂及其两厢配房，最后是后罩楼。主要建筑有<ref name="qdwf" />：
* 大门
* 正门（宫门）
* 银安殿（正殿）
** 东、西配楼
* 后殿：该殿是礼亲王府的特点。一般清朝王府均以后殿作为后寝门，二者合并。但礼亲王府设有专门的后殿，殿前东、西还各有五间配楼。此为北京清朝王府中的孤例。
** 东、西配殿
* 穿堂门（后寝门、内门、二宫门）
* 神殿
** 东、西配房
* 后罩楼
** 东、西配房

东路建筑是礼亲王及家人居住、休息之所<ref name="qdwf" />。

西路为花园，园林设计极巧妙，花园内有山石、池沼、亭台、轩廊等建筑。[[单士元|单士元]]在《清礼王府考》中称：“西院有假山亭榭，中有断碑一幢，为[[乾隆|乾隆]]二十九年兰亭主人绿漪园所撰《老槐行》；据传，[[八国联军之役|八国联军之役]]，礼府为[[法军|法军]]占领，掘银安殿基得银窖。” <ref name="qdwf" />
