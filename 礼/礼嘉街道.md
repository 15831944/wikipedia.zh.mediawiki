'''礼嘉街道'''，是{{PRC admin/fullname|50/01/12/019/000|3}}下辖的一个[[乡镇|乡镇]]级行政单位<ref>{{Cite web|url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2017/50/01/500112.html|title=2017年统计用区划代码：渝北区|accessdate=2018-09-01|author=|date=|work=www.stats.gov.cn|publisher=[[中华人民共和国国家统计局|中华人民共和国国家统计局]]}}</ref>，现由[[重庆两江新区|重庆两江新区]]代管。

== 历史沿革 ==
礼嘉原名李家场，始建于[[清朝|清]][[乾隆|乾隆]]年间，隶[[四川省|四川省]][[重庆府|重庆府]][[江北厅|江北厅]]礼里，距江北城52里，每逢三、六、九赶集。<ref>{{cite book |author1=富珠朗阿等 |title=江北厅志 |date=1844（道光二十四年）}}</ref>民国五年（1916年）由原隶礼里、濒临[[嘉陵江|嘉陵江]]，取“礼”、“嘉”二字，更名为礼嘉场，隶[[江北县|江北县]]义里。 十九年（1933年）设礼嘉乡，之后相继隶江北县第四区、第二区、第四区、第一区、第二区、第一区等。1994年撤销江北县礼嘉乡，设[[重庆市|重庆市]][[渝北区|渝北区]]礼嘉镇。2001年，重庆[[北部新区_(重庆市)|北部新区]]成立，管辖礼嘉等4个镇，其中礼嘉镇、[[鸳鸯街道|鸳鸯镇]]由[[重庆经济技术开发区|重庆经济技术开发区]]代管。2010年8月，北部新区由[[两江新区|两江新区]]管辖，礼嘉不再受重庆经济技术开发区管辖。2011年12月23日，经[[重庆市人民政府|重庆市人民政府]]批准，撤销礼嘉镇，设立礼嘉街道<ref>{{cite web |title=礼嘉简介 |url=http://www.cqlj.gov.cn/home/news_content.asp?NewsID=8207 |publisher=礼嘉党政信息网 |accessdate=2018-09-01}}</ref><ref>{{cite web |title=重庆市人民政府关于撤销渝北区礼嘉镇设立礼嘉街道的批复（渝府 〔2011〕 117号） |url=http://www.cq.gov.cn/publicinfo/web/views/Show!detail.action?sid=1055029 |publisher=重庆市人民政府 |accessdate=2018-09-01}}</ref><ref>{{cite web |title=历史沿革 |url=http://www.cqlj.gov.cn/home/news_list.asp?ClassID=9 |publisher=礼嘉党政信息网 |accessdate=2018-09-01}}</ref>。2013年10月，析礼嘉街道、[[大竹林街道|大竹林街道]]设立[[康美街道|康美街道]]<ref>{{cite news |title=北部新区新成立金山、康美街道 下辖9个社区 |url=http://www.liangjiang.gov.cn/Content/2014-02/19/content_3569.htm |accessdate=2018-09-02 |work=两江新闻 |date=2014-02-19}}</ref><ref name=":0">{{Cite web|url=http://www.cqlj.gov.cn/home/news_content.asp?NewsID=10231|title=重庆北部新区管委会关于部分街道行政区划调整的通知（渝新委发﹝2013﹞163号）|accessdate=2018-09-02|author=|date=|publisher=礼嘉党政信息网}}</ref>。2016年2月，北部新区撤销，礼嘉街道纳入两江新区直管区，行政管理工作由两江新区管委会负责，疾病防控、征兵工作由渝北区政府负责<ref>{{Cite news|url=http://www.liangjiang.gov.cn/content/2016-06/03/content_286421.htm|title=两江新区直管片区包括15个建制镇和街道 《重庆两江新区管理办法》实施|author=|date=2016-06-03|work=重庆晨报|accessdate=2018-09-01}}</ref>。

== 行政区划 ==

礼嘉街道东接[[鸳鸯街道|鸳鸯街道]]（以金山大道为界），南靠[[康美街道|康美街道]]（以金渝大道为界），西临[[北碚区|北碚区]][[童家溪镇|童家溪镇]]（与[[嘉陵江|嘉陵江]]相隔），北连[[翠云街道|翠云街道]]、[[悦来街道|悦来街道]]、北碚区[[蔡家岗镇|蔡家岗镇]]（与[[嘉陵江|嘉陵江]]相隔）<ref name=":0" />。现下辖以下地区<ref>{{Cite web|url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2017/50/01/12/500112019.html|title=2017年统计用区划代码和城乡划分代码：礼嘉街道|accessdate=2018-09-01|author=|date=|work=www.stats.gov.cn|publisher=[[中华人民共和国国家统计局|中华人民共和国国家统计局]]}}</ref>：

富安社区、白马社区、嘉兴社区、嘉和社区。
