'''绥远通志稿'''，是[[绥远通志馆|绥远通志馆]]纂修的[[綏遠省_(中華民國)|绥远省]]通志。历经数十年，数次编撰，最终由[[内蒙古人民出版社|内蒙古人民出版社]]在2007年出版，以为[[内蒙古自治区|内蒙古自治区]]成立六十周年[[献礼|献礼]]<ref name="a1">{{Cite web |url=http://www.nmqq.gov.cn/nianjiankanwu/ShowArticle.asp?ArticleID=19147 |title=【文化大观】《绥远通志稿》出版传奇|accessdate=2018-04-27 |author=王钟 |date=2014-10-25  |work= |publisher=内蒙古区情网|language=Zh-hans }}</ref>。

== 概述 ==
始修于1930年，参预修纂者为[[郭象汲|郭象汲]]、[[荣祥|荣祥]]总其事，[[李泰棻|李泰棻]]、[[傅增湘|傅增湘]]先后任总纂。1936年成第1稿本；1937年成第2稿本，分6志36门70目，116卷，舆地图幅、金石列品，另行影印附于篇末，1965年成第3稿本，凡100卷，74门目<ref>{{Cite book|title=哈刺兀纳丛书 8 荣祥文档 上|last=呼和浩特市塞北文化研究会，土默特左旗人民政府编|first=|publisher=呼和浩特：[[内蒙古大学出版社|内蒙古大学出版社]]|year=2014.10|isbn=978-7-5665-0589-7|location=|pages=525}}</ref>。通志稿对绥远地区的地方史作了提纲挈领的记述，是研究中国北部边疆的历史及北方少数民族史的重要参考资料。上述三稿均藏于[[内蒙古自治区图书馆|内蒙古自治区图书馆]]，内蒙古出版社1985年开始付印<ref>{{Cite book|title=文史工具书词典|last=[[祝鸿熹|祝鸿熹]]，[[洪湛侯|洪湛侯]]主编|first=|publisher=杭州：[[浙江古籍出版社|浙江古籍出版社]]|year=1990.12|isbn=7-80518-133-0|location=|pages=697}}</ref>。

== 2007年出版 ==

2004年时，将其定为[[内蒙古自治区|内蒙古自治区]]成立六十周年[[献礼|献礼]]。2007年出版，第一版第一次印刷两千套。封面书名为[[荣祥|荣祥]]所书“绥远通志稿”五字<ref name="a1"/>。
