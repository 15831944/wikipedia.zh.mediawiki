'''鱗羊齒屬'''（[[學名|學名]]：{{lang|la|'''''Lepidopteris'''''}}），又名'''瘤皮羊齒屬'''、'''鱗皮羊齒屬'''，是一[[屬|屬]]已[[滅絕|滅絕]]的[[種子蕨|種子蕨]]，生存於中[[二疊紀|二疊紀]]至晚[[三疊紀|三疊紀]]，在當時是一類常見且廣泛分布的種子蕨類植物。它們雖然在規模最龐大的[[二疊紀－三疊紀滅絕事件|二疊紀－三疊紀滅絕事件]]中倖存下來，卻於之後的[[三疊紀－侏羅紀滅絕事件|三疊紀－侏羅紀滅絕事件]]中滅絕消失。其[[化石|化石]]分布於全世界。

== 特徵 ==
[[File:Lepidopteris_madagascariensis.tif|thumb]]
鱗羊齒的[[葉片|葉片]]為[[蕨類|蕨]]葉型，表皮很厚，有2至3次[[羽|羽]]狀分裂。中段較寬，上下端則較為狹窄。[[複葉|複葉]]的主軸很粗，軸面具泡腫狀的[[鱗片|鱗片]]突起，故名鱗羊齒。軸側具間小羽片。羽狀複葉的小葉呈披針形、線形至長[[卵|卵]]形；小葉為互生至對生，排列緊密。小羽片呈長[[三角形|三角形]]、[[劍|劍]]形或長[[橢圓形|橢圓形]]，以整個基部和羽軸接觸，前端為尖或鈍圓狀，基部厚實且略為毗連，[[葉脈|葉脈]]常不清楚。<ref name="digimuse">{{Cite web |url = http://digimuse.nmns.edu.tw/Demo_2011/showMetadata.aspx?ObjectId=09000001802387f2&TypeKind=suMeta&Type=plantfossils&Part=2-2&Domain=gg&Field=pf&Language=CHI |title = digimuse.nmns.edu.tw/Demo_2011/showMetadata.aspx?ObjectId=09000001802387f2&TypeKind=suMeta&Type=plantfossils&Part=2-2&Domain=gg&Field=pf&Language=CHI |language = zh |accessdate = 2018-07-12 }}</ref>[[胚珠|胚珠]]通常排列成[[盾|盾]]形的結構。
