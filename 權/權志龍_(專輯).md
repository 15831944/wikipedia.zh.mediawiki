'''權志龍'''（{{lang-ko|'''권지용'''}}；{{lang-en|'''Kwon Ji-Yong'''}}）是[[韓國|韓國]][[男子音樂組合|男子音樂組合]][[BIGBANG|BIGBANG]]隊長[[G-Dragon|G-Dragon]]第二張韓語[[迷你專輯|迷你專輯]]<ref>{{cite web |url=https://itunes.apple.com/us/album/kwon-ji-yong-ep/id1246076503 |title=Kwon Ji Yong - EP |language=en |publisher=iTunes |date=2017-06-08 |accessdate=2017-06-08 }}</ref>，由[[YG娛樂|YG娛樂]]製作，[[Genie音樂|Genie音樂]]發行，於2017年6月8日下午6時通過各大數位音樂網站公開線上音源，同年6月10日搶先在演唱會現場販售限量版實體專輯，6月19日正式全面發售<ref>{{cite web |url=https://idol001.com/news/bigbang/detail/593a39dd7a1173aa1f8b4b5f/ |title=170609 G-Dragon同名專輯《權志龍》今日開售 戳我了解詳情 |language=zh |publisher=愛豆 |date=2017-06-09 |accessdate=2017-06-09 }}</ref><ref>{{cite web |url=http://www.koreastardaily.com/tc/news/95306 |title=GD新專輯居然是USB!果然是引領潮流革命的第一人啊！ |language=zh |publisher=韓星網 |date=2017-06-08 |accessdate=2017-06-09 }}</ref>。此張專輯使用G-Dragon本名做為專輯名稱，這也標誌著繼2013年發行的《[[COUP_D'ETAT|COUP D'ETAT]]》後，時隔4年推出的個人音樂作品，也是個人第一張首次以[[USB隨身碟|USB隨身碟]]形式發售的音樂專輯。<ref>{{cite web |url=http://star.ettoday.net/news/940964 |title=GD：只要能再見到你，我願失去一切！　一開口逼哭粉絲 |language=zh |publisher=ettoday |date=2017-06-08 |accessdate=2017-06-09 }}</ref>

==背景==
[[File:G-Dragon_-_Kwon_Ji_Yong.png|thumb]]
早於2017年1月，所屬經紀公司YG娛樂便批露G-Dragon正在專心準備全新個人音樂作品，專輯將於2017年發行並同時還將舉行個人演唱會「[[Act_III,_M.O.T.T.E|Act III, M.O.T.T.E]]」以支持該專輯<ref>{{cite web|url=http://www.yg-life.com/archives/85198?lang=en|title=[Exclusive] GD&TAEYANG confirmed to come back as solo artists… BIGBANG is coming again|date=2016-12-07|website=YG Life|accessdate=2017-02-08}}</ref>。而在5月24日，根據多位歌壇相關人士透露，G-Dragon預定於6月10日展開世界巡迴演唱會前帶著個人專輯回歸韓國歌壇，並表示新作已進入最後製作階段，本次作品至少會拍攝兩支音樂錄影帶，並在當日投入了拍攝工作<ref>{{cite web|url=http://hk.apple.nextmedia.com/enews/realtime/20170524/56734827|title=【韓流】GD單飛出新專輯 今秘密拍MV下月回歸|date=2017-05-24|website=蘋果日報|accessdate=2017-06-05}}</ref>。5月31日，YG娛樂通過官方網站正式宣佈將於6月8日下午6時攜同名專輯《權志龍》回歸樂壇，並於當日公開兩幅預告海報<ref>{{cite web|url=http://www.setn.com/News.aspx?NewsID=257813|title=粉絲預備！G-Dragon「新專輯」日期確定了|date=2017-05-31|website=三立新聞網|accessdate=2017-06-05}}</ref>。其中一張海報中引出的手寫體是G-Dragon母親在他出生時親手寫在本子上的名字，為專輯賦予了特殊意義，也響應了專輯其中所要傳達的「母胎」的含意<ref>{{cite web|url=https://idol001.com/news/bigbang/detail/59360f2a7a11739c688b4b53/|title=GD新專手寫體背後的意義 你一定要知道|date=2017-05-31|website=三立新聞網|accessdate=2017-06-05}}</ref>。然而，因6月1日同組合成員[[T.O.P.|T.O.P]]的負面新聞引起軒然大波，為回歸帶來了不良影響，至6月1日公開其中一支收錄曲海報後便未再進行相關回歸前的宣傳，而G-Dragon本人也從5月31日起暫停了SNS活動<ref>{{cite web|url=http://www.koreastardaily.com/tc/news/95298|title=加油！T.O.P「大麻門」+昏迷事件下G-Dragon的艱難回歸|date=2017-06-08|website=韓星網|accessdate=2017-06-08}}</ref>，直至6月8日，音源發行前夕才通過官方[[社群網站|社群網站]]更新回歸訊息<ref>{{cite web|url=http://star.ettoday.net/news/940715|title=T.O.P持續昏迷　G-DRAGON新專輯、巡演照常進行|date=2017-06-08|website=ettoday|accessdate=2017-06-08}}</ref>，YG娛樂並對於音樂作品發行的形式作出表示，將打破以往一般專輯均以CD形式販售，此次的實體專輯會在6月10日演唱會當天以USB隨身碟取代CD光碟來發行。<ref>{{cite web|url=https://idol001.com/news/bigbang/detail/5938ef807a1173ae1f8b4898/|title=GD新專輯將以USB形式發售 再次打破大眾形式|date=2017-06-08|website=愛豆|accessdate=2017-06-08}}</ref><ref>{{cite web|url=http://www.yg-life.com/archives/90061?lang=en#qtranslate-2-chooser|title=G-DRAGON to Come Back with a USB Album|date=2017-06-08|website=YG娛樂|accessdate=2017-06-08}}</ref>

==專輯製作==
[[File:G-Dragon_usb.jpg|thumb]]
在BIGBANG於5月30日前往[[日本|日本]][[東京巨蛋|東京巨蛋]]展開「{{link-en|BIGBANG Special Event|Big Bang Special Event|BIGBANG Special Event 2017}}」歌迷見面會的舞台上，成員們率先表示G-Dragon此次的音樂作品將會是歷年個人專輯中最悲傷的專輯<ref>{{cite web|title=[Oh! 뮤직] “역대 가장 슬픈 앨범?”…지드래곤, 변화의 서막|url=http://osen.mt.co.kr/article/G1110656408|website=OSEN|accessdate=2017-06-01}}</ref> ，引發大眾關注，在此前曾發表的《{{link-en|Heartbreaker (G-Dragon歌曲)|Heartbreaker (G-Dragon song)|Heartbreaker}}》代表著與所屬組合截然不同的音樂風格、《[[One_of_a_Kind|One of a Kind]]》則是隨心所欲實驗音樂，隔年再推出講述G-Dragon本身對音樂的革命和看法，以及自己在音樂上的經歷和突破，帶有自我認同和認證的《COUP D'ETAT》，三張專輯均以不同的概念問世，當中收錄的主打曲皆為強烈感極重的歌曲，這也說明G-Dragon將會通過此次專輯在音樂上嘗試變化<ref>{{cite web|title=GD新專輯用繁中寫《權志龍》　透露「史上最悲」的一張|url=http://star.ettoday.net/news/935214|publisher=ettoday|accessdate=2017-06-05}}</ref>，而專輯名稱以他的本名「權志龍」命名，在5月31日當天正式宣佈回歸日期時一同公開。<ref>{{cite web |url=http://www.yg-life.com/archives/89899 |title=[공식입장] 지드래곤, 6월 8일 컴백 확정..’4년만에 솔로 귀환’ |language=ko |publisher=YG娛樂 |date=2017-05-31 |accessdate=2017-06-09 }}</ref>

專輯的概念顧名思義是以「人生第三幕」為出發點來進行構思和創作，脫去BIGBANG隊長和獨立音樂家的身份，通過新碟講述與其他經歷成長痛的大人有著相同的苦惱，以及三十而立的感悟，他將這些煩惱、感觸寫進歌詞，展現了「權志龍」做為普通人的人生一面，本張專輯與其說是為了熱門歌曲而打造的專輯，不如說是以平凡人的角色「權志龍」坦誠的做想做的音樂和想說的故事而誕生的專輯。從整體策劃，包括新碟封面設計、相關概念海報均由G-Dragon參與構思設計，更加凸顯了此張專輯的意義所在<ref>{{cite web |url=http://kpopn.niusnews.com/=K2qeq663 |title=G-Dragon 迷你二輯《權志龍》MV、音源 |language=zh |publisher=kpopn |date=2017-06-08 |accessdate=2017-06-08 }}</ref><ref>{{cite web |url=https://idol001.com/news/bigbang/detail/593914797a1173ad1f8b4954/ |title=用音源訴說三十而立的GD同名新專音源&MV正式公開 |language=zh |publisher=愛豆 |date=2017-06-08 |accessdate=2017-06-08 }}</ref><ref>{{cite web |url=http://www.yg-life.com/archives/90128?lang=en#qtranslate-2-chooser |title=G-DRAGON, the icon of No.1 on music charts, captivating the world |language=en |publisher=YG娛樂 |date=2017-06-09 |accessdate=2017-06-10 }}</ref>，而推出的血紅色USB隨身碟專輯則富含特別意義，象徵著權志龍的個人色彩，與他的個人演唱會名稱「母胎」相符相成，這也就像新生兒剛來到世上身上是沾滿鮮血一樣，意味著權志龍剛出生時的模樣，而USB上刻有「權志龍，A型，1988年8月18日」，就是給新生兒記錄個人信息的形式，紅色也代表出生的血、革命、生命源頭、嶄新起點等涵義。<ref>{{cite web |url=http://www.newsen.com/news_view.php?uid=201706160922200410 |title=YG 측 “지드래곤 USB 붉은색 번짐현상 불량품 NO, 의도한 콘셉트” 해명 |language=ko |publisher=newsen |date=2017-06-16 |accessdate=2017-06-17 }}</ref>

本作第一支公開的收錄曲定名為〈Bullshit〉在6月1日於官方網站上揭幕，翻譯成中文有「狗吠」、「胡扯」、「廢話」的意思，如同歌名一般，它是一首具有強烈衝擊感的嘻哈風格歌曲。這首歌有著「所有為世間磨難奔波的人，被迫屈服於更高的勢力，即便是自己也是如此」的涵義，並將「泰然地出生，我的命卻像狗一樣」寫入歌詞調侃自己，有趣的是，G-Dragon過去的歌詞中，就曾用God和Dog表示「被崇拜的時候被稱作神，被辱罵的時候被稱作狗，就是明星的生活常態」<ref name="Song">{{cite web|title=[Official Announcement] Title of G-DRAGON’s New Album is “KWON JIYONG”, and the Title Song is “BULLSHIT”|url=http://www.yg-life.com/archives/89947?lang=en|website=YG Life|accessdate=2017-06-01}}</ref><ref name="auto2">{{cite web |url=http://www.favorites.news/cat/131/node/124500 |title=「BigBang」「盤點」170610 一位唱著神曲的神童——權志龍 |publisher=娛樂新聞網 |date=2017-06-10 |accessdate=2017-06-12 }}</ref>，而這首歌的音樂錄影帶在8月17日YG娛樂預告將於8月18日G-Dragon生日當天晚間8時18分公開，但僅限已購買USB專輯才可用置入的連結中觀賞。<ref>{{cite web |url=http://star.ettoday.net/news/991462?t=GD音樂革命第一步　29歲生日發新MV限買專輯才看得到！ |title=GD生日暖捐8180萬韓幣助難民童 粉絲隨偶像捐款蓋兒童康復中心 |language=zh |publisher=ettoday |date=2017-08-17 |accessdate=2017-08-19 }}</ref>

主打歌〈無題〉及曲目在音源發行當天全數公開<ref>{{cite web |url=http://www.yg-life.com/archives/90049?lang=en#qtranslate-2-chooser |title=Title track of G-DRAGON’s solo comeback album is “UNTITLED”… “The story of ordinary man Kwon Jiyong” |language=en |publisher=YG娛樂 |date=2017-06-08 |accessdate=2017-06-08 }}</ref>。〈無題〉是一首以鋼琴為伴奏的抒情歌曲，歌詞主要表達關於思念前女友的心情，曲子開頭用簡單的鋼琴搭配G-Dragon清柔的嗓音唱著歌詞，「想要你回到我身邊，我明白那是一件無比困難的事情，也明白你害怕再次受到傷害」，與過去發表的主打歌風格不同，走的是抒情哀傷的氛圍，而音樂錄影帶由導演韓夏民（音譯）執導。音樂錄影帶在製作拍攝時為了表現出悲傷的情境，捨棄使用華麗的布景和舞臺照明，而是以特寫鏡頭來呈現G-Dragon細膩的情感表現，本部原計劃在兩天內準備多個方案進行多次拍攝，再編輯出最好的版本，但G-Dragon僅用了一個小時就完成拍攝，而且是一次完成，創下BIGBANG最短記錄的音樂錄影帶拍攝時間。<ref>{{cite web |url=http://news.kstyle.com/article.ksn?articleNo=2071791 |title=G-DRAGON、タイトル曲「無題」MVはデビュー以来最短時間で撮影 |language=ja |publisher=kstyle |date=2017-06-08 |accessdate=2017-06-08 }}</ref><ref>{{cite web |url=https://idol001.com/news/bigbang/detail/59390b4b7a1173b41f8b4893/ |title=無題 音樂錄影帶史上拍攝時長最短 感情演技爆發 |publisher=愛豆 |date=2017-06-08 |accessdate=2017-06-08 }}</ref>

〈神曲〉是G-Dragon與多位著名音樂人共同打造的作品，包括曾獲頒[[格萊美獎|格萊美獎]]的加拿大[[唱片騎師|DJ]]{{link-en|弗蘭克·達克斯|Frank Dukes}}以及曾為美國著名歌手[[Lady_Gaga|Lady Gaga]]編曲的作曲家{{link-en|貝恩．李 (作曲家)|Brian Lee (songwriter)|貝恩．李}}一同參與。這首歌是以[[意大利|意大利]][[中世纪|中世纪]][[诗人|詩人]][[但丁·阿利吉耶里|但丁]]的史詩鉅作《[[神曲|神曲]]》為題材所創作的歌曲。史詩記述著但丁在[[地獄|地獄]]（Inferno）、[[炼狱|煉獄]] （Purgatorio）及[[天國|天堂]]（Paradiso）的遊歷經過，在但丁迷失時，由古羅馬詩人[[維吉爾|維吉爾]]引導走過了地獄、煉獄，最後由他的心上人[[貝緹麗彩·坡提納里|貝緹麗彩·坡提納里]]陪他前去天堂，這首歌便是G-Dragon將自己化身為但丁，講述了成名的辛苦與愛情的可貴。{{r|auto2}}

《權志龍》破格性使用USB當作載具，而非使用傳統CD唱片發行，只要插入USB即可獲得歌曲下載連結，然而此舉卻在南韓音樂界引起了爭論，唱片統計相關網站針對「應該根據USB隨身碟是否包含音源來確定該USB是否屬於唱片」一事產生了意見分歧。<ref name=韓國日報>{{cite web |url=http://www.hankookilbo.com/v/280aaefc6e84443ea83c357daf67e945 |title=CD 버린 지드래곤… 음반 '4차 혁명' 신호탄 |language=ko |publisher=[[韓國日報|韓國日報]] |date=2017-06-14 |accessdate=2017-06-15 }}</ref>

早在6月初韓國音樂內容產業協會（KMCIA）在專輯銷售形式中增加USB隨身碟這一選項，打破只有[[密紋唱片|密紋唱片]]、[[卡式录音带|卡式錄音帶]]、[[CD|CD]]的實體專輯銷售模式慣例，音樂內容產業協會為此表示「這是為了反映音樂載體的變化趨勢而下的決定，歌手今後將會產生不發行CD而是採用USB形式在線下銷售的趨勢」，唱片統計網站[[Gaon音乐榜|加翁]]及Hanteo也認同其發行形式，但是關於《權志龍》USB，加翁表明立場不認同屬於唱片範疇內，認為音源必須依附在某一種媒介上才能稱之為唱片，但該USB專輯中並沒有裝載歌曲，而是只有連結，必須要點進連結後輸入密碼才可以下載音源，因此將不會把此專輯銷量列入計算。而與之相反，Hanteo方面則認為「Kihno智能卡{{NoteTag|1=智能音樂卡「KIhno」是一種以卡片形式傳遞音樂的媒體，利用智慧型手機的[[近場通訊|NFC]]功能，來欣賞畫報圖片與音樂。而智能音樂卡也擁有原曲KTV以及欣賞音樂錄影帶的功能，更可以即時在MR音樂之上，錄製音樂與拍攝影片，並且上傳到網路上的功能。}}也被承認為唱片並且計入銷量，《權志龍》也應以同樣的理由算入唱片的範疇內」並表示將會計算USB專輯銷量{{r|韓國日報}}<ref>{{Cite news|url=http://www.sbs.com.au/popasia/blog/2017/06/14/g-dragons-new-solo-album-kwon-ji-yong-ineligible-gaon-chart|title=G-Dragon's new solo album 'Kwon Ji Yong' ineligible on the Gaon Chart|publisher=SBS|accessdate=2017-06-15|date=2017-06-14}}</ref>。而對於自己的作品產生爭議一事，G-Dragon本人則在[[Instagram|Instagram]]發表了看法。{{r|Instagram}}
;<big>不良品事件</big>
新專輯在6月10日率先在演唱會現場販售限量版本，然而，有部分購買者表示「碰水就掉漆」、「是次品」等表達不滿意見，為此，YG娛樂對USB隨身碟掉色一事做出回應，稱「絕對不是不良品」，解釋是為了展現G-Dragon所期望的DNA及母胎等元素，故意製作的效果，而在專輯發售前便已對於USB可能會掉色的問題進行了公開標示，並同時表示此次USB在製作中遇到了許多困難，工廠進行了多次實驗並最後確定只有純手工製作才能呈現出G-Dragon想要的母胎概念和復古的感覺。<ref>{{cite web |url=http://www.yg-life.com/archives/90389?lang=en#qtranslate-2-chooser |title=YG says “The red color smudging on the KWON JI YONG USB memory stick is not a defect… It is an intended concept” |language=en |publisher=yg-life |date=2017-06-16 |accessdate=2017-06-17 }}</ref>
