'''權玄彬'''（{{Kor|k='''권현빈'''|rm='''Kwon Hyun Bin'''}}，{{lang-en|'''Kwon Hyun Bin'''}}，{{bd|1997年|3月4日}}）是韓國著名模特經紀公司[[YGKPlus|YGKPlus]]旗下模特。2015年3月以模特兒身份出道。於2017年參與韓國[[Mnet|Mnet]]選秀節目《[[PRODUCE_101_(第二季)|PRODUCE 101第二季]]》，最終成績為第22名。賽後所屬經紀公司經過多次討論後，宣佈權玄彬將加入活動期為7個月的正式團體[[JBJ|JBJ]]。<ref name="auto">{{cite press release |language = zh-tw |url = http://www.koreastardaily.com/tc/news/97630 |title = JBJ正式宣佈出道！ 9月真人秀、10月發專輯，然而不是七人只有六人...... |publisher = KSD韓星網 |date = 2017-08-30  }}</ref>

== 个人背景 ==

=== 早期生活 ===
權玄彬出生於南韓[[首爾特別市|首爾特別市]]。小學四年級至初一時，曾到[[日本|日本]][[東京|東京]]居住。從中京高等學校以體育特長生身份轉學至狎鷗亭中學，其後升讀狎鷗亭高中。中學時期曾經是名擊劍選手，並曾作為青少年國家代表隊出賽。後來因為大腿後肌撕裂傷退出擊劍界，聽從學長提議成為模特兒。<ref>{{cite news|url=http://star.ettoday.net/news/950115?t=%E3%80%8A101%E3%80%8B%E6%AC%8A%E7%8E%84%E5%BD%AC%E6%9B%BE%E6%98%AF%E6%93%8A%E5%8A%8D%E5%9C%8B%E6%89%8B%EF%BC%81%E3%80%80%E3%80%8C%E7%8D%B2%E5%8B%9D%E7%9E%AC%E9%96%93%E3%80%8D%E8%84%AB%E5%B8%BD%E6%AD%A1%E5%91%BC%E5%B8%A5%E7%BF%BB|title=《101》權玄彬曾是擊劍國手！　「獲勝瞬間」脫帽歡呼帥翻}}</ref> 於2015年3月以模特兒身份出道，後成為模特兒籃球隊「코드원 (Code One)」一員。

== 經歷 ==

=== 2017年：参加选秀节目《PRODUCE 101 第二季》、以团体身份JBJ出道 ===

==== 《PRODUCE 101 第二季》時期 ====
2017年，權玄彬參加《[[PRODUCE_101_第二季|PRODUCE 101 第二季]] 》。參賽前已經是有一定知名度的模特兒，開播前的自我介紹影片更受到關注，第一周排名為16位。<ref>{{cite news|title=《101》練習生挑戰不眨眼 撐到放大片掉出來|url=http://ent.ltn.com.tw/news/breakingnews/2016699}}</ref>然而，從未有接受過舞蹈訓練，實力被評為最低的F級。第一次分組對抗中，不熟稔舞蹈的情況下仍經常獨自休息，而受到舞蹈導師權宰勝和隊長[[金鐘炫|金鐘炫]]訓話，散漫形象引起大眾惡評。<ref>{{cite news|title=《101》練習生讓NU'EST隊長說重話　態度惹議IG照全刪|url=http://star.ettoday.net/news/914913?t=%E3%80%8A101%E3%80%8B%E7%B7%B4%E7%BF%92%E7%94%9F%E8%AE%93NU%27EST%E9%9A%8A%E9%95%B7%E8%AA%AA%E9%87%8D%E8%A9%B1%E3%80%80%E6%85%8B%E5%BA%A6%E6%83%B9%E8%AD%B0IG%E7%85%A7%E5%85%A8%E5%88%AA}}</ref>

因網友惡評和每周下降的排名，權玄彬在第二次表演中陷入短暫的低潮。<ref>{{cite news|title=權玄彬粉絲公開存摺　送手袋拉票|url=http://hk.on.cc/hk/bkn/cnt/entertainment/20170518/bkn-20170518134705889-0518_00862_001.html}}</ref><ref>{{cite news|title=《101》權玄彬陷低潮　靠尹智聖「揭瘡疤」開導挺過|url=http://star.ettoday.net/news/928533?t=%E3%80%8A101%E3%80%8B%E6%AC%8A%E7%8E%84%E5%BD%AC%E9%99%B7%E4%BD%8E%E6%BD%AE%E3%80%80%E9%9D%A0%E5%B0%B9%E6%99%BA%E8%81%96%E3%80%8C%E6%8F%AD%E7%98%A1%E7%96%A4%E3%80%8D%E9%96%8B%E5%B0%8E%E6%8C%BA%E9%81%8E}}</ref>在第三次比賽中才重新振作，受到隊長[[金鐘炫|金鐘炫]]和[[尹智聖|尹智聖]]的影響，成為《月下少年》的隊長，表演《I know you know》，可惜因改制未能進入決賽。<ref>{{cite news|title=權玄彬受訪談金鐘炫：珍貴的存在|url=http://m.allthatstar.com/tw/view.php?action=19451&postID=344820|access-date=2017-08-15|archive-url=https://web.archive.org/web/20170816012853/http://m.allthatstar.com/tw/view.php?action=19451&postID=344820|archive-date=2017-08-16|dead-url=yes}}</ref>在決賽時，當宣佈[[金鐘炫|金鐘炫]]是第14位時，無緣進入[[Wanna_One|Wanna One]]，權玄彬失望流淚成為話題。<ref>{{cite web|title=權玄彬：金鐘炫淘汰後笑著說沒事更傷心|url=http://kpopn.niusnews.com/=K38fiis0}}</ref>

賽後得到各方邀約，為多款雜誌拍攝不同的畫報，代言美妝品牌《Lilybyred》的唇釉，並於2017年7月21日舉行首次單獨粉絲見面會。<ref>{{cite news|title=權玄彬《Produce 101》節目終演後首個畫報公開！談及因為有了他們的守護才能堅持下來！|url=http://www.koreastardaily.com/tc/news/95729}}</ref><ref>{{cite news|title=「101」權玄彬舉行首次FM 特別舞臺回報厚愛|url=https://hk.celebrity.yahoo.com/101-%E6%AC%8A%E7%8E%84%E5%BD%AC%E8%88%89%E8%A1%8C%E9%A6%96%E6%AC%A1fm-%E7%89%B9%E5%88%A5%E8%88%9E%E8%87%BA%E5%9B%9E%E5%A0%B1%E5%8E%9A%E6%84%9B-053621998.html}}</ref><ref>{{cite news|title=權玄彬代言美妝品牌 《101》賽後人氣高漲|url=http://m.allthatstar.com/tw/view.php?action=5205&postID=349160|access-date=2017-08-15|archive-url=https://web.archive.org/web/20170816013054/http://m.allthatstar.com/tw/view.php?action=5205&postID=349160|archive-date=2017-08-16|dead-url=yes}}</ref> 演技方面，權玄彬確定出演MBC綜藝電視劇《Vogue Mom》和SBS Mobidic《非正規職偶像》。<ref>{{cite news|title=JBJ权玄彬确定加盟新剧《非正规职偶像》|url=http://www.bntnews.cn/app/news.php?nid=47999 |language = zh-cn |publisher = bnt新聞 |date = 2017-09-07}}</ref>

==== JBJ時期 ====
2017年8月30日，所屬社發佈聲明表示權玄彬將在30日開始與《[[PRODUCE_101_第二季|PRODUCE 101 第二季]] 》參賽者[[盧太鉉|盧太鉉]]、[[高田健太|高田健太]]、[[金相均|金相均]]、[[金龍國|金龍國]]、[[金東漢|金東漢]]及因合約糾紛尚未確定合流的成員[[金泰東|金泰東]]組成限定組合[[JBJ|JBJ]]，將展開活動期為7個月的組合活動。<ref name="auto"/> 於同年10月18日攜首張迷你專輯《Fantasy》出道，並於當晚8時在高麗大學花亭體育館舉行出道Showcase「정말 데뷔하기 좋은날 (真的很適合出道的日子)」。
