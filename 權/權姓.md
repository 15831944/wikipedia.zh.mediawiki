'''權姓'''是[[漢姓|漢姓]]之一，在《百家姓》中排名第{{百家姓次序|权}}位。在中国吉林省及其它地方的朝鲜族中，在[[朝鮮半島|朝鮮半島]]是較常見的姓氏，但是朝鲜族中的权姓大多于公元十世纪由高丽王赐封；中原的权姓则公认为起源于春秋之前。

== 姓氏起源 ==
# 出於[[子姓|子姓]]，以國名為姓。 [[商湯|商湯]]建立[[商朝|商朝]]後，第二十三代大王[[武丁|武丁]]，武丁的後人有受封於權地（今中國[[湖北省|湖北省]][[當陽|當陽]]），為[[權國|權國]]，後為[[楚國|楚國]]所滅，權國只好遷到今[[湖北省|湖北省]][[荊門|荊門]]一帶，後來又被[[巴國|巴國]]所滅，權國的貴族與子孫就以國為姓，改姓為「權」。
# 出自[[羋姓|羋姓]]，以邑名為姓。 [[楚武王|楚武王]]滅[[權國|權國]]後，改[[權國|權國]]為縣，權國遺臣[[鬥緡|鬥緡]]圖謀叛楚，失敗後被殺。鬥緡的後人於是就以權為姓。
# 出自[[葛天氏|葛天氏]]，出自遠古時期古帝葛天氏之後，屬於以國名爲氏。據史籍《路史》的記載：“古帝葛天氏之後有權氏、[[葛姓|葛氏]]。” 
# 出自[[伏羲氏|伏羲氏]]。今中國[[河南省|河南省]]境，有個「太昊伏羲氏之陵」，有一段古文中寫道：「渭河之中得此磅礴，故後世據以權為首頭，後裔不已犧皇，德語三山，位在三十三天外天，功為人皇得權，『以權為氏姓』......」。
#出自一位高丽开国功臣新羅貴族金幸，被賜姓权。
