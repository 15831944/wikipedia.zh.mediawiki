'''曼陀罗花'''（[[学名|学名]]：''{{lang|la|Datura stramonium}}''，{{lang-en|Thorn apple}}）是[[茄科|茄科]]類[[曼陀羅屬|曼陀羅屬]]植物，又名醉心花、狗核桃、醉仙桃、疯茄儿、南洋金花、山茄子、凤茄花、朝鮮牽牛花。幾個世紀以來，曼陀羅被用作中醫草藥，以緩解[[哮喘|哮喘]]症狀和作為[[手術|手術]]或接骨中的[[止痛劑|止痛劑]]。它也是一種強大的[[致幻劑|迷幻劑]]和譫妄劑。

== 形态 ==
曼陀罗花是茄科野生直立木质一年生[[草本植物|草本植物]]，在[[低纬度|低纬度]]地区可长成[[亚灌木|亚灌木]]，株高一般在50厘米到200厘米之间，根部已经木质化。曼陀罗花单叶互生，上部呈对生状，叶片卵形或宽卵形，先端尖，基部两侧不对称，全缘或者破转，波状锯齿。曼陀罗花的花蕾荫发于叶叉间，花呈喇叭型，大多数花色为白色，长度约在4厘米到10厘米之间。曼陀罗花的[[果实|果实]]呈球形，外壳表面有棘刺，果实成熟时裂成4瓣，[[种子|种子]]为宽三角形，扁平，淡褐色。

== 分布 ==
曼陀罗花原产于[[墨西哥|墨西哥]]<ref>[https://npgsweb.ars-grin.gov/gringlobal/taxonomydetail.aspx?13323 ]</ref>。目前广泛分布于世界温带至热带地区。生于荒地、旱地、宅旁、向阳山坡、林缘、草地。

== 生长繁殖 ==
曼陀罗花花期很长，只要[[温度|温度]]合适一年四季都可开花，野外生长的曼陀罗花花期在5月到9月之间，果期成熟期於6月到10月之间。曼陀罗花靠种子繁殖。

== 用途 ==
曼陀罗花整株有毒，种子毒性最大。花的主要成份为[[东莨菪碱|东莨菪碱]]及少量[[阿托品|阿托品]]，而起麻醉作用的主要成份是东莨菪碱。它的作用是使肌肉松驰，抑制[[汗腺|汗腺]]分泌，因此古人将此花所制的麻醉药取名为“[[蒙汗药|蒙汗药]]”相當於迷魂藥的一種。

[[宋朝|宋朝]]《扁鹊心书》中说：“人难忍艾火炙痛，服此（曼陀罗花等）即昏不知痛，亦不伤人。”[[明朝|明朝]][[李时珍|李时珍]]《[[本草纲目|本草纲目]]》中记述“八月采此花，七月采[[大麻|火麻子花]]，阴干，等分为末，热酒调服三钱，少顷昏昏如醉。割疮灸火，宜先服此，则不觉其苦也”都强调了它的麻醉作用。據說[[三国|三国]]时期[[华陀|华陀]]所制的“[[麻沸散|麻沸散]]”也含有曼陀罗花，民间的“蒙汗药”也是用其所制。

曼陀罗花除有麻醉作用外，还有止咳平喘功效，可以治疗寒性[[咳喘|咳喘]]、少痰等病症。但不论做为麻醉还是用来治咳喘，都要谨遵医嘱，否则就会中毒。

== 中毒 ==
曼陀羅花主要毒素成分為東莨菪鹼、莨菪鹼，誤食後會出現口乾舌燥、皮膚潮紅、心跳呼吸加快、頭暈，接著可能出現幻聽、幻覺、意識模糊、妄想等神經系統症狀。

== 花語 ==
* 紫色－恐怖
* 藍色－詐情、騙愛
* 紅色－花香傳說有魔力，喚起生前記憶
* 粉色－適意
