'''曼哈顿悬日'''（{{lang-en|Manhattanhenge}}）是指在[[美国|美国]][[纽约|纽约]][[曼哈顿|曼哈顿]]出现的自然现象。由于曼哈顿街道大多呈棋盘式布局，在每年的5月28日和7月12日（或13日），[[日落|日落]]时阳光将洒满曼哈顿的所有的东西向街道，呈现一幅壮观的景象，时间长达15分钟。<ref name="xh">{{cite web|url=http://news.xinhuanet.com/photo/2009-07/13/content_11700450.htm|title=纽约再现“曼哈顿悬日”美景|publisher=新华网|accessdate=2010-02-08}}</ref>而在每年的12月5日和1月8日，这样的景观将伴随着[[日出|日出]]而出现。<ref name="ssh">{{cite web|url=http://songshuhui.net/archives/23411.html|title=冬季到纽约来看日出|publisher=科学松鼠会|accessdate=2010-02-08}}</ref>

曼哈顿悬日的英文名称Manhattanhenge是[[美国自然历史博物馆|美国自然历史博物馆]]的天体物理学家[[奈爾·德葛拉司·泰森|奈爾·德葛拉司·泰森]]于2002年首次使用的。<ref name="xh" /><ref name="ssh" />他将曼哈顿（Manhattan）与[[英国|英国]]著名的[[巨石阵|巨石阵]]（Stonehenge）的词根henge结合，创造了这一词语。

== 相關的現象 ==
在相似街道格局的其他都市也有相似的現象。通常採取棋盤式道路規劃的都市較易發生。其城市如下列所述：
* [[巴爾的摩|巴爾的摩]]—3月25日與9月18日的日出；以及3月12日與9月29日的日落。<ref>[http://weblogs.marylandweather.com/2007/07/canyon_sunset_day_in_manhattan.html Roylance, Frank. "Sunset on 'Manhattanhenge,'" ''Maryland Weather'' (''The Baltimore Sun'' meteorology blog), Friday, July 13, 2007.]</ref>
* [[芝加哥|芝加哥]]—9月25日，有''Chicagohenge''之稱。<ref>{{cite web |url=http://www.chicagoreader.com/TheBlog/archives/2009/08/20/chicagohenge |title=Chicagohenge! |author= |date= |work=Blog |publisher=Chicago Reader |accessdate=20 September 2010}}</ref>
* [[多倫多|多倫多]]—10月25日與2月16日，有''Torontohenge''之稱。<ref name="Torontopedia">{{cite web |url=http://www.Torontopedia.ca/Torontohenge |title=Torontohenge |author= |date= |work= |publisher=Torontopedia |accessdate= }}</ref><ref>{{cite web |url=http://www.gavan.ca/visual-media/photography/experience-manhattanhenge-in-toronto/ |title=Experience "Manhattanhenge" in Toronto |author= |date= |work= |publisher= Gavan.ca |accessdate= }}</ref>
* [[蒙特婁|蒙特婁]]—7月12日，有''Montrealhenge''之稱。<ref>http://www.urbanphoto.net/blog/2009/07/14/manhattanhenge-and-montrealhenge/</ref>
* [[高雄|高雄]]—1月28日與11月12日。封街白天可拍懸日奇景，晚上可拍掛月奇景。<ref>http://news.ltn.com.tw/news/life/breakingnews/1219206/</ref>
