'''曼諾·關狄多·品托·迪·奧利維拉'''（{{lang-pt|Manoel Cândido Pinto de Oliveira}}，{{bd|1908年|12月11日|2015年|4月2日|catIdx=Oliveira, Manoel De}}）是一位[[葡萄牙|葡萄牙]]電影導演與編劇。他是電影史上唯一一位橫跨[[無聲電影|無聲電影]]至[[信息时代|數位時代]]的電影導演<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/entertainment/20150402/586093/世上最老導演奧利維拉辭世　享壽106歲 |title = 世上最老導演奧利維拉辭世　享壽106歲 |author = 林慧娟 |date = 2015-04-02 |publisher = 蘋果日報 |language = zh-hant |accessdate = 2015-04-04}}</ref>，曾於[[第61屆威尼斯影展|第61屆威尼斯影展]]獲頒[[金獅獎|榮譽金獅獎]]、於[[第61届戛纳电影节|第61屆坎城影展]]獲頒[[金棕櫚獎|榮譽金棕櫚獎]]與被法國政府頒發[[法國榮譽軍團勳章|法國榮譽軍團勳章]]。


== 生平 ==
1931年，曼諾·迪·奧利維拉完成第一部短片《{{link-pt|河岸印象|Douro, Faina Fluvial}}》；他在1942年完成首部劇情長片《{{link-pt|童黨小霸王|Aniki-Bóbó}}》。1988年，他以《{{link-pt|食人族 (電影)|Os Canibais|食人族}}》首度入選坎城影展競賽片；1999年，他以《{{link-fr|情歸何處|La Lettre (film, 1999)}}》獲得[[坎城影展評審團獎|坎城影展評審團獎]]。

2008年3月，邁入100歲的曼諾·迪·奧利維拉成為全世界最老的導演<ref>{{cite news| url=http://www.nytimes.com/2008/03/09/movies/09lim.html | work=The New York Times | first=Dennis | last=Lim | title=Centenarian Director's Very Long View | date=9 March 2008}}</ref>，同年獲得[[坎城影展|坎城影展]][[金棕櫚獎|榮譽金棕櫚獎]]肯定。同年，他以短片《虛與實》（Do Visivelao Invisivel）入選[[威尼斯影展|威尼斯影展]]，他一上台致詞，全場起立向他鼓掌致意，威尼斯甚至頒發「表彰年輕電影人」的獎項給這位百歲人瑞導演，影展主席馬克穆勒（Marc Muller）表示，雖然奧利維拉年過百歲，卻是影展上最年輕的一員。<ref>{{Cite web |url = http://www.ettoday.net/news/20150403/488089.htm |title = 拍片拍到106歲！《情歸何處》坎城名導奧利維拉病逝 |author = 洪文 |date = 2015-04-04 |publisher = 東森新聞 |language = zh-hant |accessdate = 2015-04-02}}</ref>

2015年4月2日，他因[[心臟衰竭|鬱血性心衰竭]]病逝，葡萄牙政府宣布全國降半旗哀悼。<ref>{{Cite web |url = http://www.storm.mg/lifestyle/45538 |title = 百歲傳奇導演辭世 葡萄牙全國降半旗誌哀 |author = 閻紀宇 |date = 2015-04-03 |publisher = 東森新聞 |language = zh-hant |accessdate = 2015-04-04}}</ref>
