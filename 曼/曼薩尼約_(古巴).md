'''曼薩尼略'''是[[古巴|古巴]]的城市，属[[格拉瑪省|格拉瑪省]]，始於1840年1月6日，海拔高度25米，是古巴第14大城，且在古巴的非省会城市中是人口最多的。

== 地理 ==
曼萨尼约是古巴东部省份格拉玛省的一个海港城市，临近[[瓜卡納亞沃灣|瓜卡纳亚沃湾]]，在[[考托河|考托河]]三角洲附近。

== 历史 ==
曼萨尼约的移民历史可以追溯到1784年<ref name=EB1911>{{EB1911|inline=1 |wstitle=Manzanillo (Cuba) |volume=17 |page=626}}</ref><ref name="GreatestCitiesEncarta">{{cite web |url= http://www.greatestcities.com/Caribbean/Cuba/Manzanillo_city.html |title= Manzanillo Info. |author= [[Encarta|Encarta]] |work= GreatestCities.com |year= 2001}}</ref>。它曾在[[美西战争|美西战争]]期间四次成为战场。

1792年，法国人洗劫了当地移民，次年该地建立了一座堡垒保护居民。<ref name=EB1911/>

1827年，港口对国内外的经商者开放。<ref name=EB1911/>

1833年，成立市政厅；1837年，由于其不像圣地亚哥一样宣布遵从西班牙宪法，王室授予其''Fiel''的称号。<ref name=EB1911/>

== 经济 ==
主要農產品有[[咖啡|咖啡]]、[[稻米|稻米]]、[[水果|水果]]、[[煙草|煙草]]<ref name=EB1911/>、[[甘蔗|甘蔗]]<ref name=EB1911/>及其他农作物，也有[[蜂蜜|蜂蜜]]和[[家牛|家牛]]。

工业有[[鋸木廠|锯木厂]]、鱼罐头工厂、[[糖蜜|糖蜜]]厂，还有一些[[雪茄|雪茄]]和[[皮革|皮革制品]]厂。<ref name="GreatestCitiesEncarta"/>

市辖区内有[[锌|锌]]和[[铜|铜]]储备。
