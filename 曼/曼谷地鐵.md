'''曼谷地鐵'''（{{thai|t=รถไฟฟ้ามหานคร}}）<!--歡迎更正-->蓝线在2004年7月啟用，從城市北端連到城市中心，共設有19個車站。蓝线和其19个车站，全部由日本政府透过[[日本国际协力银行|日本国际协力银行]]资助，每个车站都有象征日泰友好的纪念资助牌匾。曼谷地鐵是[[軌道運輸標竿聯盟|軌道運輸標竿聯盟]]（NOVA）的成員之一<ref>[http://www.nova-metros.org/  軌道運輸標竿聯盟（NOVA）]{{en}}</ref>。

== 轉乘規劃 ==
* [[曼谷大众運輸系統|曼谷大众運輸系統]]（BTS）：曼谷地下鐵路的乘客可以在[[乍都节公园车站|乍都节公园车站]]、[[素坤逸车站|素坤逸车站]]或[[是隆車站|是隆車站]]進行換乘。
* [[泰國国家鐵路|泰國国家鐵路]]：曼谷地下鐵路的乘客可以在「[[華喃峰火車站|華喃峰火車站]]」或[[挽賜車站|挽賜車站]]，換乘[[泰國国家鐵路|泰國国家鐵路]]到泰國其他地方，另有国际线路至[[馬來西亞|馬來西亞]]的[[檳城|檳城]]的[[北海_(馬來西亞)|北海市]]。

== 近期规划 ==
主要的主线计划可归纳如下：
* [[曼谷地铁蓝线|蓝线]]延线
# 曼赐-塔帕[Bangsue-Thaphra]（13公里）
# 华南峰火车站-曼凱[Hualamphong Railway Station-Bangkae]（14公里）
:两條延線將於[[塔帕車站|塔帕車站]]匯合，合計高架路段長22公里，地下路段長5公里，共有17座高架車站和4座地下車站。总投資金額823億1010萬泰銖。<ref>http://www.mrta.co.th/newLineProject/thBlueLine.htm</ref>
* [[蘇坤蔚線|蘇坤蔚線]]延線
* [[曼谷地铁橙线|橙线]]
# 达铃灿-民武里[Talingchan-Minburi]（37.5公里)
:此路線連接曼谷東部和西部之間。合计高架路段長11公里，地下路段長26.5公里，共有8座高架車站和21座地下車站。总投資金額1379億6000萬泰銖。<ref>http://www.mrta.co.th/newLineProject/thOrangeLine.htm</ref>
* [[曼谷地鐵紫線|紫線]]，分为两期工程
# 曼艾-岛本[Bangyai-Taopun]（23公里）全為高架路段，共有16座高架車站。总投資金額599億4290萬泰銖。
# 岛本-拉布拉那-库内[Taopun-Ratburana-Krunai]（23.6公里），合计高架路段長11公里，地下路段長12.6公里，共有5座高架車站和11座地底車站。总投資金額739億1800萬泰銖。<ref>http://www.mrta.co.th/newLineProject/thPurpleLine.htm</ref>

其他地鐵擴建計劃包括：
* [[曼谷地鐵黃線|黃線]]
# 拉差达勒抛-帕塔那甘[Ratchada-Latphrao-Patthanakarn]（12.6公里），共有10座车站。
# 帕塔那甘-三龙[Patthanakarn-Samlong]（17.8公里），共有11座车站。<ref>http://www.mrta.co.th/newLineProject/thYellowLine.htm</ref>
* [[曼谷地鐵粉紅線|粉紅線]]
# 凯来-民武里[Kaerai-Minburi]（34.5公里）
:此路線計劃使用[[單軌鐵路|單軌列車]]，全為高架路段，共有30座車站。<ref>http://www.mrta.co.th/newLineProject/thPinkLine.htm</ref>
* [[曼谷地铁棕线|棕线]]：曼甲必-兰甘亨路-盛桑[Bangkapi-Lamkamhaeng-Saensap]（10公里）
:此路線已跟曼谷地鐵橙線合併。
