'''雞翼角'''（[[英語|英語]]：'''{{lang|en|Peaked Hill}} 或 {{lang|en|Kai Yat Kok}}'''），又稱'''雞山'''<ref name="分流">蕭國健《香港離島史跡志》第46頁</ref>、'''雞公頭'''<ref name="饒">饒玖才《香港的地名與地方歷史下-新界》第90頁，ISBN 978-988-219-461-8</ref>。是[[香港|香港]][[大嶼山|大嶼山]]西部小島。雞翼角是[[陸連島|陸連島]]，大退潮時有一[[連島沙洲|連島沙洲]]露出海面，亦被稱為香港最西的島嶼、西極<ref name="雞山">朱維德《香港歷史名勝》第122至123頁，ISBN 978-962-973-093-6</REF>。島上並無常住人口。

== 地理 ==
[[File:San_On_Chi.jpg|thumb]]

連島沙洲連接雞翼角東北面以及大嶼山西部的[[煎魚灣|煎魚灣]]，亦與大嶼山的[[雞公山|雞公山]]相對。地形上，雞翼角北面有凹入處，而南面凸出，形狀像[[雞翅|雞翅]]（雞翼），故得此名<ref>{{Cite web |url=http://www.afcd.gov.hk/tc_chi/country/cou_vis/cou_vis_cam/cou_vis_cam_cam/cou_vis_cam_36_TsinYueWan.html# |title=漁農自然護理署指定露營地點--煎魚灣營地 |access-date=2018-01-29 |archive-url=https://web.archive.org/web/20180614230650/http://www.afcd.gov.hk/tc_chi/country/cou_vis/cou_vis_cam/cou_vis_cam_cam/cou_vis_cam_cam_38.html# |archive-date=2018-06-14 |dead-url=yes }}</ref>；而又有說法是自附近大嶼山高地西望雞翼角時會看到雞翼角如同一隻伏下的母雞，北面的凹位就像母雞伸長脖子並將頭擺向左方，故有別稱雞山<ref name="雞山" />。該島海岸線長1.4公里，沿岸有大石灘、海蝕洞及崖壁，綑邊甚具難度<REF>[http://archive.am730.com.hk/article-261968   西極探索 二澳與雞翼角 am730  16-12-2016]</REF>。

== 雞翼角與分流 ==
在雞翼角南方不遠處有[[分流_(香港)|分流半島]]，一些記載稱因為其形狀之故亦有雞翼角之名，加上嘉慶《新安縣志》中「海防」一部亦有載雞翼角炮台，即為現在仍可見的的[[分流炮台|分流炮台]]遺址，二者同名引起了混淆<ref name="分流"/><ref name="饒"/><ref>黃垤華《香港山嶺志——桂角山志．雞公山志．牛潭山志．麒麟山志》 第53頁</ref>
