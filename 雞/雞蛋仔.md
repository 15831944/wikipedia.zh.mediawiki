'''雞蛋仔'''是一種[[澳門|澳門]]和[[香港|香港]]街頭小吃<ref>[http://hk.apple.nextmedia.com/supplement/food/art/20140827/18844292 百分百香港味 炭香雞蛋仔]，[[蘋果日報|蘋果日報]]，2014年08月27日</ref>。製作雞蛋仔時，以[[雞蛋|雞蛋]]、[[白砂糖|白砂糖]]、[[黄油|牛油]]、[[麵粉|麵粉]]和[[淡奶|淡奶]]製成汁液，然後倒進蜂巢型鐵模烘熟；製成後雞蛋仔呈金黃色。

== 起源 ==
雞蛋仔發源自五十年代的香港，當時雜貨店為免白白浪費破裂的雞蛋，於是把麵粉和牛油等食材加入蛋漿裡，然後倒進模具烘焗；後來有人設計出雞蛋狀的模具，於是被命名為「雞蛋仔」<ref>[http://gd.people.com.cn/BIG5/n/2014/1112/c123932-22882942.html 街頭小吃嚼出香港文化味道]，人民網，2014年11月12日</ref>。

== 現況 ==
雞蛋仔傳統上會放在[[木炭|炭火]]上[[烘焙|烘烤]]而成，但近年為節省成本及安全，大多改用電爐。

在香港，雞蛋仔傳統是由街頭[[小販|小販]]販賣<ref>[http://google.com 尋找香港最美味的雞蛋仔之旅]，[[CNN|CNN]]，2010-9-15</ref>，近年則有小食店製造<ref>{{cite web|author=OpenRice|title=雞蛋仔|url=http://www.openrice.com/recipe/detail.htm?recipeid=569|accessdate=2013-06-15}}</ref>。雞蛋仔除了作為小吃外，更可代替[[威化餅|威化餅]]作為[[冰淇淋|雪糕]]的配料<ref>[http://hk.on.cc/hk/bkn/cnt/news/20150108/bkn-20150108171002809-0108_00822_001.html?eventid=4028810746664a6801468b82118557b0&eventsection=hk_news 雞蛋仔新地嘗冰火 熨斗多士亦好味]，[[東方日報|東方日報]]，    
2015年1月8日</ref>。
  在香港，比較人氣的雞蛋仔食店有 ''北角利強記''、''滿屋雞蛋仔''、''媽咪雞蛋仔''。

==制作方法==
准备材料：低筋面粉140克、鸡蛋2粒、泡打粉7克、木薯粉28克、白砂糖140克、牛奶28克、水140毫升、油适量

制作方法：
# 将所有的粉类过筛，并把油、水、牛奶和鸡蛋混合打匀。
# 讲过筛后的粉与蛋浆混合，搅拌均匀。
# 将面糊过滤以去除杂质。
# 将鸡蛋仔的模预热、烧热，並且刷上一层油。
# 倒入面糊至摸子的半分满。
# 停留几秒钟，盖上另外一半的盖子，并马上翻面。
# 翻面后，每面各烧1至2分钟，打开看看，若颜色差不多均匀上色就可以出炉了。
# 可以使用叉子挑一挑，鸡蛋仔就会从模子中脱出来。

注意：热的时候会软软嫩嫩的，待凉之后就会外脆里嫩了。

== 食法 ==
現有不少地區和國家，包括香港，澳門，台灣，馬來西亞，新加坡等東南亞國家在近年開始，都可見到許多店家製作雞蛋仔，並做出許多不同風味的雞蛋仔。例如：香芋雞蛋仔<ref>[http://www.silencetv.hk/program/s%e5%bf%ab%e7%b6%ab%ef%bc%8d%e9%a6%99%e8%8a%8b%e9%9b%9e%e8%9b%8b%e4%bb%94/ S快綫－香芋雞蛋仔]，[[龍耳電視|龍耳電視]]，    
2016年7月12日</ref>、西式雞蛋仔，配合西式風味的巧克力糖漿等，冰淇淋來搭配。

== 類似小吃 ==
[[臺灣|台灣]]有一種類似雞蛋仔的街頭小吃，名為'''雞蛋糕'''，作法類似[[車輪餅|車輪餅]]和[[章魚燒|章魚燒]]，把麵糊倒進模具裏，使用[[發粉|發粉]]（浹打粉）等膨劑，添加入水泥糊，加熱製成。販賣雞蛋糕的攤檔常見於台灣的夜市、市場、商店街等。
