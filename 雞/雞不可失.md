《'''雞不可失'''》（{{lang-ko|극한직업}}，{{lang-en|''Extreme Job''}}）是一部2019年上映的韓國動作喜劇電影，由[[柳承龍|柳承龍]]、[[李東輝_(演員)|李東輝]]、[[李荷妮|李荷妮]]、{{Link-ko|陳善圭|진선규}}、[[孔明_(藝人)|孔明]]主演。電影講述一支由5位警探組成的緝毒小組，為了逮到毒販，意外開起炸雞店，後找到事業第二春的故事。<ref>{{cite web |title=「極限職業」進行媒體試映會 |url=https://chinese.joins.com/big5/article.aspx?art_id=186706 |website=chinese.joins.com |accessdate=2019-01-23 |date=2019-01-11}}</ref><ref>{{cite web |title=陳善奎自曝「極限職業」中與李哈妮吻戲提前向其男友尹啓相報備 |url=https://chinese.joins.com/big5/article.aspx?art_id=186699 |website=chinese.joins.com |accessdate=2019-01-23 |date=2019-01-11}}</ref><ref>{{cite web |title=電影「極限職業」舉行發布會 李荷妮柳承龍等出席 |url=https://times.hinet.net/times/mobile/news/22184756 |website=HiNet生活誌 |accessdate=2019-01-23}}</ref>

==劇情==
由5位掉漆警探組成的緝毒小組，因辦事不力屢屢出包而被長官釘得滿頭包，落到成天埋伏在毒販巢穴對面的[[炸雞|炸雞]]店中監控，案情也毫無進展。既然每天都要待在炸雞店裡，不如親自動手來賣炸雞，這個看似無厘頭的決定，竟讓這群魯蛇警探在誤打誤撞下意外尋獲事業第二春，業績賣到嚇嚇叫的神級美味炸雞，因此讓這間餐廳在一夕爆紅，成為網紅名店，訂單電話接到手軟！這幫副業做太大的警探還有辦法達成使命嗎？對面的毒販是否會聞香而來呢？
