'''雞肉腸'''（英語：'''Chicken Frank'''）是一種用[[雞肉|雞肉]]製造的[[香腸|香腸]]，為常見的急凍肉類產品<ref>[http://hk.on.cc/hk/bkn/cnt/news/20160520/bkn-20160520174143521-0520_00822_001.html 兩個牌子一樣樣 山寨貨定翻譯錯]，東方日報，2016年05月20日</ref>，也有製成[[罐頭|罐頭]]發售，除可以單獨食用外，也可作為製作其他食品的配料。在[[香港|香港]]一般稱為「腸仔」的香腸，所指的就是雞肉腸<ref>[http://sina.com.hk/news/article/20151027/0/1/4/%E6%97%A5%E9%A3%9F%E6%A2%9D%E5%8D%8A%E9%9B%9E%E8%82%89%E8%85%B8-%E5%A4%A7%E8%85%B8%E7%99%8C%E9%A2%A8%E9%9A%AA%E5%A2%9E18-5213534.html 日食條半雞肉腸 大腸癌風險增18%]，新浪香港，2015年10月27日</ref>。

== 成分 ==
市面售賣的雞肉腸因為要配合運輸及長期保存，所以大多加入[[食品添加劑|食品添加劑]]。雞肉腸的主要成分是雞肉，雞肉會打成肉泥，拌入[[麵粉|麵粉]]、[[澱粉|澱粉]]及[[鹽|鹽]]，並加入[[味精|味精]]。為穩定製成品的品質，還會加入[[乳化劑|乳化劑]]、[[抗氧化劑|抗氧化劑]]，並加入[[防腐劑|防腐劑]]以利保存。雞肉混入澱粉及添加劑製成的[[粉紅肉渣|肉泥]]會擠壓成為長條狀加工成為雞肉腸。一般香腸通常都會擠入薄膜製成的腸衣，並與腸衣一起食用，而雞肉腸的製成品則大多沒有腸衣，所以雞肉腸的腸身一般都較其他種類的香腸柔軟。急凍包裝的雞肉腸常會以10條為一單位用[[塑料袋|塑料袋]]密封出售，這種包裝雞肉腸必須保持低溫冷藏，直至烹調前才進行解凍。

== 食用 ==
雞肉腸可以經烹調後單獨食用，在香港是常見的[[燒烤|燒烤]]食品<ref>[https://www.hk01.com/%E7%86%B1%E8%A9%B1/48300/5%E7%B2%92%E9%AD%9A%E8%82%89%E7%87%92%E8%B3%A3%E7%AD%89%E6%96%BC%E7%A2%97%E5%8D%8A%E9%A3%AF-BBQ%E9%BB%9E%E5%85%88%E9%A3%9F%E5%BE%97%E5%81%A5%E5%BA%B7- 5粒魚肉燒賣等於碗半飯　BBQ點先食得健康？]，香港01，2016-12-31</ref>，也可作為[[即食麵|即食麵]]的佐料。雞肉腸可用於食品製作的材料，例如[[腸仔包|腸仔包]]就常以雞肉腸製作<ref>[http://www.nmplus.hk/lifestyle/%e8%85%b8%e4%bb%94%e5%8c%85%e6%af%94%e4%bd%b5/ 5大鏈鎖店腸仔包比併 幫你搵出最抵食又好味]，新Monday，2016-09-22</ref>。

== 健康 ==
雞肉腸是一種[[加工食品|加工食品]]<ref>[http://www2.hkedcity.net/sch_files/a/lkc/lkc-tyl/public_html/life/life14.htm 加工肉三危]，2001年9月29日</ref>，在生產時也會加入多種食品添加劑，所以不是健康的食品<ref>[http://www.cndzys.com/yinshi/changshi/205808.html 鸡肉肠会胖吗]，大众养生网，2014-06-30</ref>，含[[鈉|鈉]]量及[[熱量|熱量]]皆高，而防腐劑的加入也會產生可引發癌症的[[亞硝酸鹽|亞硝酸鹽]]<ref>[http://www.mingpaocanada.com/Food/van/contentFoodNews.cfm?aid=298&m=0 防腐劑高鹽高脂累事]，明報，2015-11-10</ref>。
