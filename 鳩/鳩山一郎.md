'''鳩山一郎'''（{{bd|1883年|1月1日|1959年|3月7日|catIdx=Hatoyama, Ichirou}}），日本政治家，第52、53、54任內閣總理大臣，被認為是二戰後[[日本|日本]]最重要的[[日本內閣總理大臣|首相]]之一。

== 生平 ==
父親[[鳩山和夫|鳩山和夫]]是文部省第一期的[[耶魯大學|耶魯大學]]留學生，母親春子為東京女子師範学校的英語教師、著名作家及共立女子職業學校（現為共立女子大學）創始人，弟弟為日本著名民法學者[[鳩山秀夫|鳩山秀夫]]。孫子是曾任[[民主黨_(日本)|民主黨]]代表和首相的[[鳩山由紀夫|鳩山由紀夫]]與[[自由民主黨_(日本)|自由民主黨]]籍眾議員的[[鳩山邦夫|鳩山邦夫]]。

1907年（明治40年）[[東京帝国大学|東京帝国大学]]英法科畢業，進入父親的律師事務所工作。1911年父親去世，任眾議院議員兼東京市議。曾擔任[[犬養毅|犬養毅]]與[[齋藤實|齋藤實]]内閣的文部大臣，在[[齋藤實|齋藤實]]內閣擔任文部大臣任期內，發生著名的[[瀧川事件|瀧川事件]]。

戰後，1945年11月9日，自由黨成立為第一大黨，鳩山任首屆總裁。大選之前，5月4日，鳩山一郎被整肅（公職追放），只好引退。1951年，他加入了該[[博愛|博愛]]或[[友愛|友愛]]組織「[[共濟會|共濟會]]」。1954年12月10日第一次組閣，1955年3月19日第一次引退。1955年3月19日第二次任內閣總理大臣。同年11月22日率內閣總辭。1955年11月22日第三次組閣，1956年12月23日率內閣總辭。1959年（昭和34年）去世。

== 个人年表 ==
1883年（[[明治天皇|明治]]16年）出生于[[东京|东京市]]牛込区（现[[東京都|东京都]][[新宿區|新宿区]]）东五轩町。父亲[[鳩山和夫|鸠山和夫]]是文部省第1期留学生、律师、东京府会议员。母亲{{Internal link helper/ja|鸠山春子|鳩山春子}}创立了{{Internal link helper/ja|东京女子师范学校|東京女子高等師範}}（现为{{Internal link helper/ja|茶水女子大学|お茶の水女子大学}}）的英语教师，明治19年共立女子职业学校（[[共立女子大学|共立女子大学]]）。

1895年（明治28年）{{Internal link helper/ja|东京高等师范学校|東京高等師範学校}}附属小学（现・[[筑波大学|筑波大学]]附属小学）毕业。

1900年（明治33年）高中师范学校附属中学（现・[[筑波大学附属中学校·高等学校|筑波大学附属中学・高等学校]]）毕业。

1903年（明治36年）毕业于[[舊制第一高等學校|旧制第一高等学校]]（现[[东京大学|东京大学]]教养学部）。

1907年（明治40年）毕业于[[东京帝国大学|东京帝国大学]]法科大学英法专业，在父亲的律师事务所工作。

1908年（明治41年）与爱国团体{{Internal link helper/ja|玄洋社|玄洋社}}出身的众议院秘书课长{{Internal link helper/ja|寺田荣|寺田栄}}的长女寺田薰结婚。

1911年（明治44年）父亲和夫去世（众议院议员兼东京市议员、东京律师会会长）。

1912年（[[大正天皇|大正]]元年）在父亲的补缺选举中首次当选东京市会议员。

1915年（大正4年）在[[立憲政友會|立宪政友会]]公认中当选众议院议员。

1918年（大正7年）长子{{Internal link helper/ja|鸠山威一郎|鳩山威一郎}}诞生。

1924年（大正13年）随着政友会分裂，他加入了{{Internal link helper/ja|政友本党|政友本党}}。

1925年（大正14年）政友本党退党。

1927年（[[昭和天皇|昭和]]2年）在[[田中义一|田中义一]]内阁担任内阁书记官长（1929年为止）。

1931年（昭和6年）[[犬養內閣|犬养内阁]]文部大臣。[[五一五事件|五一五事件]]后，[[齋藤內閣|斋藤内阁]]继续担任文部大臣（到1934年为止）。

1933年（昭和8年）作为文部大臣参与了[[瀧川事件|泷川事件]]。

1934年（昭和9年）因[[帝銀事件|帝人事件]]被怀疑贪污而辞职（斋藤内阁之后也总辞职）。

1937年（昭和12年）与[[中島知久平|中岛知久平]]、{{Internal link helper/ja|前田米藏|前田米蔵}}、{{Internal link helper/ja|岛田俊雄|島田俊雄}}一起就任政友会总裁代理委员。

1939年（昭和14年）随着政友会的分裂，隶属于正统派。

1942年（昭和17年）非推荐参加翼赞选举，无党派当选。

1943年（昭和18年）批判[[東條內閣|东条内阁]]，隐居[[輕井澤|轻井泽]]。

1945年（昭和20年）二战结束后从轻井泽到东京，组成[[日本自由黨|日本自由党]]（总裁）。

1946年（昭和21年）在总选举中，日本自由党成为第一大党，却被开除公职。

1951年（昭和26年）因[[中風|脑出血]]而倒下，解除。

1954年（昭和29年）11月[[日本民主黨_(1954年)|日本民主党]]组成（总裁），12月成为内阁总理大臣。[[第1次鳩山一郎內閣|第一次鸠山一郎内阁]]成立。

1955年（昭和30年）1月“天声解散”，在总选举中，民主党成为第一大党，但过半数仍未制约。3月[[第2次鳩山一郎內閣|第2次鸠山一郎内阁]]成立。11月根据保守联合自由民主党组成(55年体制也参照)。[[第3次鳩山一郎內閣|第3次鸠山一郎内阁]]成立。

1956年（昭和31年）在莫斯科签署了[[日蘇共同宣言|日苏共同宣言]]（恢复[[日苏关系|日苏邦交]]），内阁总辞职于花道。

1959年（昭和34年）去世。墓地是东京都台东区的[[谷中靈園|谷中陵园]]。[[大勳位菊花大綬章|大勋位菊花大绶章]]死后获奖。

== 友愛 ==
鳩山一郎翻譯了[[歐盟之父|歐盟之父]][[理查德·尼古拉斯·冯·康登霍维-凯勒奇|理查德·尼古拉斯·冯·康登霍维-凯勒奇]]的英文書「The Totalitarian State Against Man」為日語。康登霍维-凯勒奇主張了友愛。康登霍维-凯勒奇的友愛思想對鳩山產生了非常大的影響。1920年代，康登霍维-凯勒奇是該共濟會在[[維也納|維也納]]的成員。
