'''鳩谷站'''（{{jpn|j=鳩ヶ谷駅|hg=はとがやえき|rm=Hatogaya eki|f=y}}）是一個位於[[埼玉縣|埼玉縣]][[川口市|川口市]]大字里，屬於[[埼玉高速鐵道線|埼玉高速鐵道埼玉球場線]]的[[鐵路車站|鐵路車站]]。車站編號是'''SR 22'''。

== 歷史 ==
* 2001年（[[平成|平成]]13年）3月28日 - 開業。
* 2011年（平成23年）10月11日 - [[鳩谷市|鳩谷市]]編入合併到川口市，此站成為川口市的車站。

== 車站結構 ==
島式月台1面2線的[[地底車站|地底車站]]。裝有保安用的[[月台閘門|月台閘門]]系統。地下1樓是閘口層，地下2樓是月台層。月台建於[[國道122號|國道122號]]（{{link-ja|岩槻街道|岩槻街道}}）的地下。車站顏色是水藍色。

== 使用情況 ==
2016年度的1日平均上車人次為'''10,504人'''<ref name="sr">[http://www.s-rail.co.jp/about/corporate/financial.html 経営状況・輸送状況] - 埼玉高速鉄道</ref>。埼玉高速鐵道線的車站次於[[赤羽岩淵站|赤羽岩淵站]]、[[東川口站|東川口站]]排行第3位，是單獨站當中最多。
