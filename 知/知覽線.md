'''知覽線'''（{{jpn|j=知覧線|hg=ちらんせん|rm=Chiran sen|f=y}}）是一條由[[鹿兒島縣|鹿兒島縣]][[日置郡|日置郡]][[金峰町|金峰町]]（現在為[[南薩摩市|南薩摩市]]）[[阿多站|阿多站]]至鹿兒島縣[[川邊郡|川邊郡]][[知覽町|知覽町]]（現在為[[南九州市|南九州市]]）[[知覽站|知覽站]]的鐵路線。由[[鹿兒島交通|鹿兒島交通]]營運。在1965年停運。

== 路線概要 ==
*管轄：鹿兒島交通
*路線（營運公里）：阿多 - 知覽 16.3km
*軌距：1067mm
*站數：11個（包括起終點站，在停運前一刻計算）
*複線區間：沒有（全線單線）
*電化區間：沒有（全線非電氣化）

== 運行情況 ==
在路線繁盛時期（1950年代），加世田至知覽站之間一日共有5班直通列車來回。而阿多至知覽站之間一日共有4班旅客或貨物列車來回。在薩摩川邊站可以列車交會設備。

由於客量下降，在1961年10月1日的修訂時間表後至1965年7月停運前，加世田至知覽站之間一日共有2班來回的旅客列車，以及1班來回的貨運列車。

所有列車會在阿多站會掉頭往加世田站。

== 歷史 ==
*1927年（昭和2年）6月1日：薩南中央鐵道阿多至薩摩川邊站開業。
*1928年（昭和3年）9月1日：新設新田站（後來改名為東阿多站）。
*1930年（昭和5年）11月15日：薩摩川邊至知覽站開業。
*1943年（昭和18年）4月2日：南薩鐵道與薩南中央鐵道合併，名為知覽線。。
*1964年（昭和39年）9月1日：南薩鐵道與三州自動車合併，成為鹿兒島交通，成為鹿兒島交通的鐵路線。
*1965年（昭和40年）7月3日：由於雨災，全線不能通行。
*1965年（昭和40年）11月15日：全線停運。
