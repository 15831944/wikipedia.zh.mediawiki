'''知州'''指的是中国[[宋朝|宋朝]]以及之后、[[越南|越南]][[后黎朝|后黎朝]]和[[阮朝|阮朝]][[州|州]]的[[行政长官|行政长官]]，雅稱為[[太守|太守]]或[[刺史|刺史]]。

== 宋朝 ==
[[宋太祖|宋太祖]]为了削弱[[节度使|节度使]]的权力，防止[[唐朝|唐朝]]、[[五代|五代]]时期[[軍閥|軍閥]][[割据|割据]]的局面重演，规定诸[[州|州]][[刺史|刺史]]得直接向[[朝廷|朝廷]]奏报和接受[[诏令|诏令]]，[[节度使|节度使]]不得干预除所驻州之外（所谓[[支郡|支郡]]）的政务。后来，朝廷逐步派遣[[京朝官|京朝官]]（文臣）接替[[刺史|刺史]]管理州务，全称'''知某州军州事'''，俗称'''知州事'''、'''知州'''、'''某州'''、'''郡守'''、'''太守'''、'''守臣'''、'''使君'''、'''两千石'''、'''刺史'''。“知”就是管理的意思；“州军事”的“州”代表民政，“军”代表军政，。由此，[[刺史|刺史]]非奉特别诏令，不得过问所任州的州务，演化为武臣的迁转之阶，属于正任官之一。

同时，节度使也很快演化成一种地位崇高的官階，成为武臣的最高階官，不再有实际权力与职掌；皇帝会给知州特殊的优待，以表示荣宠。<ref>《宋史》卷431，《[[邢昺|邢昺]]》，页12799。</ref>[[范必大|范必大]]出任知州事，[[宋高宗|宋高宗]]亲自斟酒，均系例外施恩。州也就由隶属[[藩镇|藩镇]]变为隶属以[[转运使|转运使]]为长官的[[路|路]]，以及以[[留守|留守]]、[[知府|知某某府事]]、[[尹|尹]]为长官的[[府|府]]。

== 元朝 ==
[[元朝|元朝]]沿用[[宋朝|宋朝]]制度，州的长官正式称为'''知州'''，但是[[路|路]]、[[府|府]]、[[州|州]]均置[[达鲁花赤|达鲁花赤]]，专由[[蒙古|蒙古]]人、[[色目人|色目人]]充，地位在'''知州'''上。

== 明清 ==
[[明|明]]以'''知州'''为一[[州|州]]之长，辖[[县|县]]；[[清|清]]有[[直隶州|直隶州]]、[[散州|散州]]之别，前者直隶于[[省|省]]，可以辖县，后者隶属于[[府|府]]、[[道_(行政区)|道]]，不辖县，长官均称'''知州'''。

== 民国成立以后 ==
民国废府级政区，直隶州与府大致相同，一并废止。1928年以后，在地位比较重要的[[城镇|城镇]]设市，很多府、直隶州治所及一些散州改设市。1949年中华人民共和国建立以后，事实上恢复了府级政区，地区、专区或[[地级市|地级市]]下辖的[[县级市|县级市]]，大致相当于明清时的[[散州|散州]]。

==越南==
越南后黎朝[[黎圣宗|黎圣宗]]重划版图后，在山地少数民族聚居的地方设州，行政长官为知州，品阶稍逊于知县，多由当地土目担任。1948年，[[越南民主共和国|越南民主共和国]]将州改设为县。
