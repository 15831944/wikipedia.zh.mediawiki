'''知行合一'''，简称'''知行'''，是明朝政治家、哲学家[[王守仁|王守仁]]阐述的一种哲学与世界观的方法，其为[[心学|心学]]的核心理念之一<ref>{{cite news|title=王守仁：始创“知行合一”|url=http://xbyx.cersp.com/hdzt/jdcw/200802/2242.html|accessdate=2012-10-31|newspaper=华东师范大学|deadurl=yes|archiveurl=https://web.archive.org/web/20130625013428/http://xbyx.cersp.com/hdzt/jdcw/200802/2242.html|archivedate=2013-06-25}}</ref>。

== 概述 ==
王守仁繼承[[陸九淵|陸九淵]]強調「心即是理」之思想，反對[[程頤|程頤]][[朱熹|朱熹]]通過事事物物追求「至理」的「[[格物致知|格物致知]]」方法，因為事理無窮無盡，格之則未免煩累，故提倡從自己內心中去尋找「理」，認為「理」全在人「心」，「理」化生宇宙天地萬物，人秉其秀氣，故人心自秉其精要。

在知與行的關係上，王守仁從「天地萬物本吾一體」出發，強調要知，更要行，知中有行，行中有知，所谓“知行合一”，二者互为表里，不可分离。知必然要表现为行，不行则不能算真知。

== 出处 ==
王守仁指出：

{{ Cquote2 |知是行之始，行是知之成。若會得時，只說一箇知，已自有行在。只說一箇行，已自有知在。| 傳習錄 卷上 徐爱录  }}

{{ Cquote2 |古人所以既說一箇知，又說一箇行者，只為世間有一種人，懵懵懂懂的任意去做，全不解思惟省察。也只是箇冥行妄作。所以必說箇知，方纔行得是。| 傳習錄 卷上 徐爱录 }}

{{ Cquote2 |……又有一種人，茫茫蕩蕩，懸空去思一索。全不肯著實躬行。也只是箇揣摸影響。所以必說一箇行， 方纔知得真。此是古人不得已，補偏救弊的說話。若見得這箇意時，即一言而足。| 傳習錄 卷上 徐爱录  }}

{{ Cquote2 |……今人卻就將知行分作兩件去做。以為必先知了，然後能行。我如今且去講習討論做知的工夫。待知得真了，方去做行的工夫。故遂終身不行，亦遂終身不知。| 傳習錄 卷上 徐爱录  }}
