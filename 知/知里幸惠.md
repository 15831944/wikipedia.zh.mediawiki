'''知里幸惠'''（{{lang-ja|ちり ゆきえ|Chiri Yukie}}，{{bd|1903年|6月8日|1922年|9月18日}}）為[[北海道|北海道]][[登別市|登別市]]出身的[[阿伊努族|愛努人]]女性。在短短19年的壽命生涯中，出版其著作[[愛努神謠集|愛努神謠集]]，因而復振、搶救滅絕危機的[[阿伊努族|愛努族]]傳統文化而聞名。

近年媒體及各地研討會對其作品再評價的呼聲高漲，並興起「知里幸惠」記念館的建設運動，對知里幸惠表達感謝。2008年10月日本[[日本廣播協會|NHK]]的節目『{{tsl|ja|その時歴史が動いた||轉動歷史的時刻}}』在第340集播出知里幸惠紀錄片，節目標題為「眾神之歌再度回到大地—愛努少女．知里幸惠的奮鬥」(神々のうた 大地にふたたび〜アイヌ少女・知里幸恵の闘い〜  )詳細介紹知里幸惠生平。

知里幸惠著作[[愛努神謠集|愛努神謠集]]在亞馬遜網路書店的銷售量，入榜為暢銷排名前十名，此外[[愛努神謠集|愛努神謠集]]更有[[法語|法語]]、[[英语|英语]]、[[俄語|俄語]]翻譯本，2006年1月法國作家[[让-马里·古斯塔夫·勒克莱齐奥|让-马里·古斯塔夫·勒克莱齐奥]](Jean-Marie Gustave Le Clézio)到知里幸惠的墓前參拜，報告其法語版出版。

另外，知里幸惠弟弟{{tsl|ja|知里真志保||知里真志保}}是語言學學者，同時也是第一個成為[[北海道大學|北海道大學]]教授的[[阿伊努族|愛努人]]學者。知里幸惠的[[愛努神謠集|愛努神謠集]]出版以來，從大正末期到昭和之間，這對姊弟的表現被新聞、雜誌等評價為「[[阿伊努族|愛努]]的天才姊弟」。另一個弟弟[[知里高央|知里高央]]（真志保的哥哥）同樣擔任教職並同時從事[[阿伊努語|愛努語]] 詞彙的研究工作。

== 生平 ==
[[1903年|1903年]]（明治36年）6月8日、北海道[[幌別郡|幌別郡]]（現為[[登別市|登別市]]、[[札幌市|札幌市]]往南約100公里）出生（父：高吉、母：ナミNami）。6歲時被帶到旭川市阿姨家進入小學就讀。最初和[[大和族|大和族]]小孩同所學校，後來被配置到只有[[阿伊努族|愛努人]]念的小學裡，以優等的學業成績從[[阿伊努族|愛努人]]的小學畢業<ref>『人物でつづる被差別民の歴史　続』94頁</ref>。

後來在旭川繼續升學就讀職業學校({{tsl|ja|旭川市立女子高等学校||旭川區立女子職業學校}})。[[阿伊努語|愛努語]]和[[日本語|日本語]]都非常優秀的她，卻是愛努子弟第一次在北海道廳立女子學校的考試中落敗。『明明那麼優秀怎麼會…』『因為是基督徒所以才沒有考上…』鎮上傳著各種流言蜚語<ref>『北の大地、そこに生きる人々の歴史と文化、漫画「ゴールデンカムイ」今こそ知りたいアイヌ 時空旅人別冊』92頁</ref>。知里幸惠的外婆是[[優卡爾庫魯|優卡爾庫魯]] ([[阿伊努語|愛努語]]：Yukarkuru，日語：ユーカラクル，愛努敘事詩的講述者)，也就是愛努口傳敘事詩[[卡姆伊優卡爾|卡姆伊優卡爾]](カムイユカラ，kamuy yukar，神謠)的傳唱者。愛努神謠[[卡姆伊優卡爾|卡姆伊優卡爾]]對於沒有文字的[[阿伊努族|愛努人]]而言，是子孫傳承其價值觀、道德觀、傳統文化等重要的依據。知里幸惠則是身處在能聽聞到神謠[[卡姆伊優卡爾|卡姆伊優卡爾]]的環境下長大。

知里幸惠出生時，明治政府為防止俄羅斯侵略已在北海道開拓30多年。在知里幸惠15歲時，語言學家[[金田一京助|金田一京助]]為了記錄愛努傳統文化，拜訪了知里幸惠一家。知里幸惠看見[[金田一京助|金田一京助]]熱心訪問、記錄祖母們講述愛努傳統的神謠[[卡姆伊優卡爾|卡姆伊優卡爾]]，感受到[[金田一京助|金田一京助]]對愛努傳統文化的尊敬及對神謠[[卡姆伊優卡爾|卡姆伊優卡爾]]的熱情，開始將神謠[[卡姆伊優卡爾|卡姆伊優卡爾]]從[[阿伊努語|愛努語]]翻譯為[[日语|日语]]。最後，知里幸惠接受[[金田一京助|金田一京助]]的請求，將愛努神謠[[卡姆伊優卡爾|卡姆伊優卡爾]]寫成文字讓後世留存，寄宿於東京文京區[[金田一京助|金田一京助]]家繼續完成翻譯工作。

知里幸惠罹患了重度心臟病(當時被診斷為慢性支氣管炎)，卻仍持續翻譯、編集的工作，在[[1922年|1922年]]（大正11年）9月18日完成『[[愛努神謠集|愛努神謠集]]』，卻在完成當日晚上心臟病發辭世，於19歲英年早逝。

知里幸惠完成『[[愛努神謠集|愛努神謠集]]』的隔年[[1923年|1923年]]（大正12年）[[8月10日|8月10日]]，被收錄在[[柳田國男|柳田國男]]所編集的爐邊叢書系列中，由鄉土研究社出版<ref>{{Cite book|和書|title=アイヌ神謡集|url=http://www.aozora.gr.jp/cards/001529/card44909.html|author=知里 幸恵|publisher=郷土研究社|series=炉辺叢書|date=1923}}</ref>。

== 對後世的影響 ==
知里幸惠的[[愛努神謠集|愛努神謠集]]出版，給予[[阿伊努族|愛努民族]]在進入明治時代瀕臨滅絕危機的愛努文化帶來自信與光芒，成為復振的重要轉機。當時的新聞更大幅報導，讓許多人知道知里幸惠，也認識到愛努的傳統、文化、語言及風俗習慣。如同知里幸惠從[[金田一京助|金田一京助]]身上得到啟發一樣，知里幸惠讓許多[[阿伊努族|愛努人]]意識到自身民族的自信與驕傲。知里幸惠影響其弟知里真志保在語言學、[[阿伊努語|愛努語]]學領域上的成就，並當上北海道大學第一位[[阿伊努族|愛努人]]教授。另外，以詩人身份活躍的[[阿伊努族|愛努人]]{{tsl|ja|森竹竹市||森竹竹市}}、{{tsl|ja|違星北斗||違星北斗}}等人和知里真志保，同樣都對社會公眾呼籲提升[[阿伊努族|愛努人]]的社會地位。知里幸惠即是促成[[阿伊努族|愛努人]]轉變的重要契機。

透過知里幸惠的[[愛努神謠集|愛努神謠集]]，使[[阿伊努族|愛努民族]]對身邊「動物神靈」的寄託、祈求日子幸福的神謠故事(カムイユカラ，kamuy yuka)轉為文字留存於世，對於沒有文字的[[阿伊努族|愛努民族]]來說是劃時代的創舉。知里幸惠將過去外婆的謠唱，以平易近人的文筆寫下13編故事。從[[阿伊努語|愛努語]]翻譯到日語，知里幸惠使用[[阿伊努語|愛努語]]及日語雙語運用自如，將其才能毫無遺憾的發揮，將沒有文字的[[阿伊努語|愛努語]]原文以羅馬字標音，用日語譯意，使日本人閱讀皆能輕鬆的朗朗上口。

== 年表 ==
* [[1923年|1923年]]（大正12年）『愛努神謠集』出版。
* [[1970年|1970年]]（昭和45年）絕版的『愛努神謠集』由札幌弘南堂書店再版<ref name="Fujimoto">藤本英夫『知里幸恵 十七歳のウエペケレ』</ref>。
* [[1973年|1973年]]（昭和48年） 藤本英夫評知里幸恵的『下吧！下吧！降下銀色的甘霖吧！ 』出版<ref name="Fujimoto"/>。
* [[1977年|1977年]]（昭和52年）岩波文庫再版『愛努神謠集』<ref name="Fujimoto"/>。
* [[1990年|1990年]] 知里幸惠文學碑落成在知里幸惠和阿姨、外婆一起生活過的旭川，由荒井和子發起向市民募款，立碑在旭川近文的北門中學校內。每年知里幸惠的生日[[6月8日|6月8日]]由近文的[[阿伊努族|愛努人]]們舉辦知里幸惠誕辰紀念活動。
* [[1997年|1997年]][[7月1日|7月1日]] 施行{{tsl|ja|アイヌ文化振興法||愛努文化振興法}}，廢除{{tsl|ja|北海道旧土人保護法||北海道舊土人保護法}}。
* [[2000年|2000年]]開始，在知里幸惠的出生地[[登別市|登別市]]每年在知里幸會祭日[[9月18日|9月18日]]前的連假舉行，以愛努神謠[[卡姆伊優卡爾|卡姆伊優卡爾]] (カムイユカラ)為核心的愛努文化相關活動，由知里幸惠的姪女知里むつみ(Chiri Mutsumi)及其非營利組織「知里森舍」主辦。
* [[2003年|2003年]] 知里幸恵誕生100週年，在其出生地[[登別市|登別市]]發起籌措興建知里幸恵紀念館的募款活動。
* [[2006年|2006年]]1月 法國[[加利瑪出版社|加利瑪出版社]]出版『愛努神謠集』法文版，參與出版工作的小說家[[津島佑子|津島佑子]]與法國作家[[让-马里·古斯塔夫·勒克莱齐奥|让-马里·古斯塔夫·勒克莱齐奥]](Jean-Marie Gustave Le Clézio)來到知里幸惠的墓前參訪。
* [[2008年|2008年]][[10月15日|10月15日]] 日本[[日本廣播協會|NHK]]節目『{{tsl|ja|その時歴史が動いた||轉動歷史的時刻}}』播出知里幸惠紀錄片。
* 2010年9月 「知里幸恵 銀之露紀念館」在其出生地[[登別市|登別市]]揭幕。
