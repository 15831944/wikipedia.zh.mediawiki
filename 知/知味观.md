'''知味观'''是[[杭州|杭州]]的一个[[餐饮|餐饮]]品牌，最初是[[孙翼斋|孙翼斋]]于1913年创建的餐饮店“知味馆”，“知味停车，闻香下马”为其雅号，至今已有百年历史。它是杭城最具知名度的餐饮企业之一，位于杭州仁和路83号的总店建筑面积为7000多平米，知味观已在杭州、[[上海|上海]]及杭州周边地区开设80多家各类连锁店，其中杭州地区有分店42家，并拥有一个建筑面积4000余平米的连锁配送中心和一个建筑面积达2.8万多平米的大型食品厂，位列中国十大餐饮品牌企业。<ref name="走进">{{cite web |url=http://www.zhiweiguan.com.cn/zj/columnsId=2&&i=28&comContentId=28.html |language= |title=走进知味 |publisher=知味观 |date= |accessdate=2015-07-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20150704213152/http://www.zhiweiguan.com.cn/zj/columnsId%3D2%26%26i%3D28%26comContentId%3D28.html |archivedate=2015-07-04 }}</ref>

== 简介 ==
成立于2007年的杭州知味观食品有限公司，是隶属于“知味观”品牌下的一家从事食品生产、加工和销售的综合性食品企业，位于杭州市[[余杭区|余杭区]][[瓶窑镇|瓶窑镇]]凤都工业园区。公司占地面积近26亩，总建筑面积达2.86万平米。<ref name="走进"/>

知味观味庄位于[[杭州西湖|杭州西湖]][[杨公堤|杨公堤]]10－12号，占地面积70亩，沿湖岸线500余米，是杭州高档商务宴请之地。知味观味宅地处[[河坊街|河坊街]]。上海知味观设于上海长寿路831号智慧广场4楼，建筑面积3600余平米。<ref name="走进"/>

== 荣誉 ==
*1993年 被[[商务部|商务部]]评为“[[中华老字号|中华老字号]]”
*1998年 被评为“浙江省著名商标”
*2001年 被中国烹饪协会评为“中华餐饮名店”
*2002年 被中国商业联合会评为“中国名牌企业”，被世界饭店与餐馆协会、中国饭店协会评为“国际餐饮名店”
*2003年 被中国烹饪协会评为“全国绿色餐饮五星级企业”
*2013年 被[[国家工商行政管理总局|国家工商行政管理总局]]授予“[[中国驰名商标|中国驰名商标]]”，是[[浙江|浙江]]首家荣获“中国驰名商标”的餐饮企业<ref >{{cite web|url=http://finance.china.com.cn/roll/20150225/2970801.shtml | language= | title=杭州知味观 传承百年好味道 |publisher=杭州日报 |date=2015-02-25 |accessdate= }}</ref>
