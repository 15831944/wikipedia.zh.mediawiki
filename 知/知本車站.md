'''知本車站'''位於[[台灣|台灣]][[臺東縣|臺東縣]][[臺東市|臺東市]]，為[[臺灣鐵路管理局|臺灣鐵路管理局]][[南迴線|南迴線]]的[[鐵路車站|鐵路車站]]。鄰近[[知本溫泉|知本溫泉]]，但周邊甚為荒涼，多半仍須靠轉車才能前往溫泉區。

== 車站構造 ==
*[[島式月台|島式月台]]兩座，有地下道連接站房，無電梯。

== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]，所有列車均有停靠。
*隨著[[花東線鐵路瓶頸路段雙軌化暨全線電氣化計畫|花東線鐵路瓶頸路段雙軌化暨全線電氣化計畫]]工程完工，本站已成為臺鐵[[東部幹線|東部幹線]]（實際位於[[南迴線|南迴線]]）的[[電氣化|電氣化]]終點，電氣化列車可行駛至本站。
*417、441次[[太魯閣列車|太魯閣號]]以本站為起點站，402、426次為終點站。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
*1985年7月15日：設站。
*2014年6月28日：以[[南迴線|南迴線]]先行工程名義完成單線電氣化。
*2015年3月24日：首次停靠[[普悠瑪列車|普悠瑪號]]。
*2015年10月15日：因應台鐵大改點，停靠於本站的[[普悠瑪列車|普悠瑪號]]班次改為[[太魯閣列車|太魯閣號]]。
*2016年6月28日：啟用多卡通刷卡機。
