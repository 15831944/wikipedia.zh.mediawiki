'''习近平'''（{{bd|1953年|6月15日|||X}}），[[祖籍|祖籍]][[河南省|河南]][[邓州市|邓州]]{{NoteTag|祖籍[[河南省|河南省]][[邓州市|邓州市]][[十林镇|十林镇]]習营村<ref>{{cite web |date = 2011-03-09 |publisher = 大公报 |title = 習近平对南阳心向往之 答应二月河回老家南阳看看 |url = http://news.sina.com.cn/c/2011-03-09/052322080409.shtml }}</ref>。其先人在[[明朝|明朝]]以前，居住於[[江西省|江西省]][[吉安市|吉安]][[新干县|-{zh-hans:新干县; zh-hant:新淦縣}-]]習家村<ref>{{cite web |date = 2005-08-12 |publisher = 新华网 |title = 红色江西，伟人祖籍地大观 |url = http://news.xinhuanet.com/newmedia/2005-08/12/content_3343991_2.htm }}</ref>。}}，[[籍贯|籍贯]][[陕西省|陝西]][[富平县|富平]]，生于[[北京市|北京]]。2012年起为[[中华人民共和国|中华人民共和国]][[中国共产党集体领导制度|第五代]][[中華人民共和國最高領導人|最高领导人]]，现任[[中国共产党中央委员会总书记|中国共产党中央委员会总书记]]、[[中华人民共和国主席|中华人民共和国主席]]、[[中国共产党中央军事委员会主席|中央军事委员会主席]]。自2007年起开始出任中国共产党第十七、十八、十九届[[中国共产党中央政治局常务委员会|中央政治局常委]]。

习近平1979年毕业于[[清华大学化工系|清华大学化工系]]。1979年至1985年，他曾分别任职于[[国务院办公厅|国务院办公厅]]、[[中央军委办公厅|中央军委办公厅]]和[[河北省|河北省]][[正定县|正定县]]。并于1985年至2002年在[[福建省|福建省]]任职，时间长达17年。在调离福建省后，他先后主政[[浙江省|浙江省]]和[[上海市|上海市]]。2007年，他进入中央，并成为[[中共中央政治局常委|中共中央政治局常委]]和[[中共中央书记处|中共中央书记处]]书记，同时先后兼任[[中共中央党校|中共中央党校]]校长、[[中华人民共和国副主席|中华人民共和国副主席]]和[[中国共产党中央军事委员会副主席|中央军事委员会副主席]]。在2012年11月的[[中国共产党第十八届中央委员会第一次全体会议|中共十八届一中全会]]上，他当选[[中国共产党中央委员会总书记|中国共产党中央委员会总书记]]、[[中国共产党中央军事委员会主席|中国共产党中央军事委员会主席]]；于2013年3月的[[第十二届全国人民代表大会#第一次会议|十二届全国人大一次会议]]当选[[中华人民共和国主席|中华人民共和国主席]]、[[中华人民共和国中央军事委员会主席|中华人民共和国中央军事委员会主席]]；于2017年10月[[中国共产党第十九届中央委员会第一次全体会议|中共十九届一中全会]]全票连任中央委员会总书记和中央军委主席<ref name="全票当选">{{cite web |url = http://m.news.cctv.com/2017/10/26/ARTIgl2URwdNtKPWEwBsyEWO171026.shtml|title=当选党代表、当选中央委员、当选总书记，习近平都是全票 |website = 央视网 |date=2017-10-26 |accessdate=2017-10-28}}</ref>；于2018年3月的[[第十三届全国人民代表大会#第一次会议|十三届全国人大一次会议]]全票连任国家主席和国家军委主席。<ref>{{cite news|title=十三届全国人大一次会议选举产生新一届国家领导人-新华网|url=http://www.xinhuanet.com/politics/2018lh/2018-03/17/c_1122550851.htm|accessdate=2018-03-17|work=www.xinhuanet.com|agency=新华社|publisher=新华网|date=2018-03-17|location=北京|language=zh-cn}}</ref>

===文革時期===
1966年12月初，[[文化大革命|文化大革命]]爆发后不久，首都红卫兵联合行动委员会（简称联动）成立。但习近平被归类为黑帮子弟，並沒有很深地涉及联动的活動。後來被抓到[[中共中央党校|中共中央党校]]集中觀察審查，被威胁“枪毙够一百次了”，最后被送往少管所“黑帮”子弟学习班<ref name="早年经历" /><ref name="xxxx" />。根据其弟[[习远平|习远平]]回忆，1968年，习近平15岁，因父亲问题的牵连，被有关部门多次关押审查，出来时，身体非常虚弱，全身都是[[蝨毛目|虱子]]。习近平到[[渭河平原|关中]]富平老家大姑家里休息很长时间，大姑一天一碗鲜羊奶喂着，他才慢慢调养好<ref>{{cite web|author=2014-09-12 17:57:06 来源：中国青年报 |url = http://history.sohu.com/20140912/n404273987.shtml |title=中国青年报:习近平15岁遭“四人帮”疯狂迫害：多次关押审查 |publisher=History.sohu.com |date=2014-09-12 |accessdate=2017-05-23}}</ref>。

1968年12月，[[毛泽东|毛泽东]]发起[[上山下乡运动|上山下乡运动]]，习近平属于被号召之列。次年，被下放到[[延安市|延安市]][[延川县|延川县]][[文安驛鎮|文安驛鎮]][[人民公社|公社]]梁家河大队[[上山下鄉運動|插队]]。习近平初期很不适应当地生活，“别人下去天天上山干活”，年少的习“却很随意”，老百姓对他印象很不好，認為習近平是個遊手好閒、不愛勞動的浪子。習近平才待几个月后，便因受不了苦，偷回到[[北京市|北京]]，又被关进‘学习班’，半年后被放出来，听从姨夫[[魏震五|魏震五]]的建议又回到延川县安驿公社梁家河<ref name="xxxx"/>。回到了梁家河後，习近平很快成为一名壮劳力，並以苦幹、實幹的精神贏得了[[農民|農民]]的認同<ref name="xxxx"/>。

1974年（一說1973年<ref name="xxxx"/>），加入[[中国共产党|中国共产党]]（因为父亲习仲勋的问题，习近平[[中国共产主义青年团|入团]]申请与入党申请屡次被驳回，在递交十余次后才被批准）。此年冬天<ref name="xxxx"/>，习近平当选为梁家河大队[[村党支部书记|党支部书记]]。在任期间，他带领村民“整社（处理[[姓氏|姓氏]][[宗族|宗族]]之间的[[矛盾|矛盾]]）”、修筑[[水坝|大坝]]、修建[[沼氣|沼氣]]池，这些举措为他在当地赢得良好口碑。<ref>{{cite web |url=http://v.ifeng.com/news/mainland/2014006/01094764-95af-413f-a27c-803e5e553173.shtml |title=04年习近平专访：我是延安人 |publisher=凤凰网|date=2014-06-12 |accessdate=2014-06-19}}</ref><ref name="后来早年经历">{{cite web |url=http://news.ifeng.com/mainland/special/2013lianghui/renshi/xijinping.shtml|title=习近平年谱 |publisher=[[凤凰网|凤凰网]]|date= | accessdate=|quote=1969年，习近平与上万名北京知青一起，......习近平被当地推荐为工农兵学员。}}</ref>

习近平回忆1973年申请大学入学时说，“像我这样家庭背景的人，在当时是不可能被录取的”。1973年，[[清华大学|清华大学]]有两个名额在[[延安|延安]]地区，全分给了[[延川县|延川县]]。习近平表示：他三个志愿都填[[清华大学|清华]]，讓我上就上，不讓我上就拉倒<ref>{{cite web|url=http://money.chinatimes.com/news/news-content.aspx?id=20140607000998&cid=1208 |title=7常委憶高考 習近平只填清華 |publisher=Money.chinatimes.com |date=2016-06-08 |accessdate=2017-05-23}}</ref>。清华来招生的人不敢做主，请示学校。时逢所谓的“右倾翻案风”，清华一二号人物[[迟群|迟群]]、[[谢静宜|谢静宜]]都不在家，[[刘冰|刘冰]]掌权，他批准了习近平的入学资质。当时，习仲勋下放劳动的洛阳耐火材料厂开了个“土证明”，称习仲勋同志属人民内部矛盾，不影响子女升学就业。习近平从而获准进入清华就读<ref>[http://www.ln.gov.cn/qmzx/gmy_99505/cw/xjp/np/201211/t20121127_1005892.html 1975-1979年：以工农兵学员身份入读清华大学] {{webarchive|url=https://web.archive.org/web/20160928074410/http://www.ln.gov.cn/qmzx/gmy_99505/cw/xjp/np/201211/t20121127_1005892.html |date=2016-09-28 }}</ref>。

===大學時期===
1975年，習近平獲得推薦，進入[[清華大學化工系|清華大學化工系]]基本有機合成專業學習<ref name="xxxx"/>。

在大學期間，結識了同為化工系工农兵学员的上下鋪室友[[陈希|陈希]]，兩人同齡同系且同是體育迷，使得他們培養深厚的友誼，習近平在[[清华大学|清华大学]]担任学生党支部书记時是陈希的兩名入党介绍人之一。直到多年之後習近平上台出任總書記，陳希也輾轉仕途成為[[中国共产党中央委员会组织部|中共中央组织部]]部長<ref>{{cite news|title=陈希与习近平既是同窗更是同党（高新）|url=http://www.rfa.org/mandarin/zhuanlan/yehuazhongnanhai/gx-04302013112626.html|accessdate=2013-04-30|newspaper=自由亞洲電台}}</ref>。

1979年4月，习近平从清华大学毕业，当时父亲[[习仲勋|习仲勋]]亦已经复出，出任[[中国共产党广东省委员会|中共广东省委]][[第一书记_(中国)|第一书记]]<ref>{{cite news|title=習近平副主席的个人成长历程|url=http://www.china.com.cn/book/txt/2008-12/16/content_16955718.htm|accessdate=2012-11-21|newspaper=中国网}}</ref><ref>{{cite news|title=两会人事习近平年谱|url=http://news.ifeng.com/mainland/special/2013lianghui/renshi/xijinping.shtml|accessdate=2013-01-28|newspaper=凤凰网}}</ref>。

=== 初入军政两界 ===
1979年4月毕业之后，分配到[[中华人民共和国国务院办公厅|国务院办公厅]]和[[中央军事委员会办公厅|中共中央军委办公厅]]担任其父好友當時的[[中共中央政治局|中共中央政治局]]委员兼[[中华人民共和国国防部|國防部]]部長[[耿飚|耿飚]]的三个秘书之一<ref>{{cite news|title=喬石與習近平均受習仲勳好友提攜|url=http://www.epochtimes.com/b5/15/6/24/n4464289.htm|accessdate=2015-06-23|newspaper=大紀元}}</ref>，而在當秘書期間（1979-1982年），也是現役[[軍人|軍人]]<ref>{{cite news|title=习近平军事生涯揭秘,获军头赞赏渊源深厚|url=http://xinweiyuekan.com/news/2010-12-02/%E4%B9%A0%E8%BF%91%E5%B9%B3%E5%86%9B%E4%BA%8B%E7%94%9F%E6%B6%AF%E6%8F%AD%E7%A7%98%E8%8E%B7%E5%86%9B%E5%A4%B4%E8%B5%9E%E8%B5%8F%E6%B8%8A%E6%BA%90%E6%B7%B1%E5%8E%9A|newspaper=新维资讯网|deadurl=yes|archiveurl=https://web.archive.org/web/20160111221242/http://xinweiyuekan.com/news/2010-12-02/%E4%B9%A0%E8%BF%91%E5%B9%B3%E5%86%9B%E4%BA%8B%E7%94%9F%E6%B6%AF%E6%8F%AD%E7%A7%98%E8%8E%B7%E5%86%9B%E5%A4%B4%E8%B5%9E%E8%B5%8F%E6%B8%8A%E6%BA%90%E6%B7%B1%E5%8E%9A|archivedate=2016-01-11}}</ref>。

在[[百万大裁军|军队大裁员]]的情况下，其转向地方。据美国《[[纽约时报|纽约时报]]》的报道，习近平意识到自己强大的家庭背景和在农村的工作积累的政绩会有助于升遷，于是主动提议放弃在北京的工作，前往农村<ref name="nyt2012"/>。在父亲习仲勋的帮助下，选择了离北京相对较近的[[正定县|正定县]]<ref name="nyt2012"/>。《[[纽约时报|纽约时报]]》专文透露，习近平原本希望担任一个县或镇的党委书记，但是由于时任[[中共河北省委|中共河北省委]]书记[[高扬|高扬]]非常反感像习近平这样没有经验却依靠人脉空降而来的人，只任命其担任了正定县委副书记<ref name="nyt2012"/>。1982年3月，习近平上任<ref name="nyt2012"/>，组织关系属于[[中共中央组织部|中共中央组织部]]直管，不属于[[中国共产党河北省委员会|中共河北省委]]管辖<ref>{{cite news|title=海外挺習黨媒：棲身軍委 習近平卅年前埋伏筆
