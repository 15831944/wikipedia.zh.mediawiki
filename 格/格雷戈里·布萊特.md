'''格雷戈里·布萊特'''（{{lang-ru|Григорий Альфредович Брейт-Шнайдер}}, Grigory Alfredovich Breit-Shneider；{{lang-en|Gregory Breit}}；{{bd|1899年|7月14日|1981年|9月13日}}），美國物理學家，生於[[俄羅斯帝國|俄羅斯帝國]][[赫爾松省|赫爾松省]][[尼古拉耶夫_(烏克蘭)|尼古拉耶夫]]，曾任[[紐約大學|紐約大學]]（1929 - 1934）、[[威斯康辛大學麥迪遜分校|威斯康辛大學麥迪遜分校]]（1934 - 1947）、[[耶魯大學|耶魯大學]]（1947 - 1968）、[[紐約州立大學水牛城分校|紐約州立大學水牛城分校]]（1968 - 1973）教授。<ref name = 'nndb'/> 1921年，布萊特曾是[[保羅·埃倫費斯特|保羅·埃倫費斯特]]的助手。

== 生平 ==
1925年，布萊特和{{link-en|杜武|Merle Tuve}}（{{lang-en|Merle Tuve}}）在{{link-en|卡耐基科學研究所|Carnegie Institution for Science}}合作，用脈衝無線電發射機來測量[[電離層|電離層]]的高度，這種測量技術為[[雷達|雷達]]的發展奠定了基礎。<ref name = 'Tuve'/> 1929年，布萊特和[[尤金·維格納|尤金·維格納]]發明了{{link-en|相對論性布萊特-維格納分布|Relativistic Breit–Wigner distribution}}，以描述粒子共震態。布萊特被認為是第一個描述了質子-質子分散的科學家。布萊特也推導出了{{link-en|布萊特方程|Breit equation}}。<ref name="Quantum" /><ref name="Breit 2001" />

1934年，布萊特和[[約翰·惠勒|約翰·惠勒]]一起描述了{{link-en|布萊特-惠勒過程|Breit–Wheeler process}}。1939年，布萊特當選[[美國國家科學院|美國國家科學院院士]]。1940年4月，布萊特對[[美國國家學院|美國國家學院]]提議美國科學家們應遵守「自我审查」政策，以防止他們的研究成果在二戰中被敵方用於軍事目的。<ref name="Secret" />

在二戰前期，[[阿瑟·康普頓|阿瑟·康普頓]]選中了布萊特，讓他來監督第一顆原子彈的早期設計，這項計劃後成為了著名的[[曼哈頓計劃|曼哈頓計劃]]。但布萊特於1942年請辭，因為他感到工作進展緩慢，而且項目出現過安全漏洞。布萊特的職位由[[羅伯特·歐本海默|羅伯特·歐本海默]]接替，歐本海默後來被認命為整個曼哈頓計劃的總指導員。

2014年，[[倫敦帝國學院|倫敦帝國學院]]的物理學家們提出了一種方法來證實布萊特和惠勒在1934年提出的「將光子轉化為物質」的理論（即「布萊特-惠勒過程」）。<ref name = '光子'/><ref name="Imperial" />

布萊特曾四度擔任[[物理評論|物理評論]]（{{lang-en|''Physical Review''}}）的副主編，分別是：1927至1929年，1939至1941年，1954至1956年，1961至1963年。

布萊特於1964年被授予[[富蘭克林獎章|富蘭克林獎章]]，於1967年被授予[[美國國家科學獎章|美國國家科學獎章]]。<ref name="National" />
