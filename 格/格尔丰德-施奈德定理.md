'''格尔丰德-施奈德定理'''（Gelfond–Schneider theorem）是一个可以用于证明许多数的[[超越数|超越性]]的结果。这个定理由Aleksandr Gelfond在1934年、Theodor Schneider在1935年分别独立证明，它回答了[[希尔伯特第七问题|希尔伯特第七问题]]。

== 表述 ==

如果α和β是[[代数数|代数数]]，其中α≠0且≠1，且β不是[[有理数|有理数]]，那么任何<math>\alpha^{\beta} = \exp \{\beta \log \alpha\}</math>的值一定是[[超越数|超越数]]。

== 评论 ==

* <math>\alpha</math>和<math>\beta</math>不限于[[实数|实数]]，它们可以是[[复数|复数]]。當虛部不為零時，視為無理數，既使實部和虛部皆為有理數。
* 一般地，<math>\alpha^{\beta} = \exp \{\beta \log \alpha\}</math>是[[多值函数|多值]]的，其中“log”表示[[复数对数|复数对数]]。
* 该定理的一个等价的表述是：如果<math>\alpha</math>和<math>\gamma</math>是非零的代数数，那么<math>(\log \gamma)/(\log \alpha)</math>要么是有理数，要么是超越数。
* 如果没有<math>\alpha</math>，<math>\beta</math>是代数数的限制，这个定理就不一定成立。例如，如果<math>\alpha=3</math>，<math>\beta=\log 2/\log 3</math>，那么<math>\alpha^{\beta}=2</math>，它是代数数。

== 定理的应用 ==

利用这个定理，立刻就可以推出以下实数的超越性：

* <math>2^{\sqrt{2}}</math>（[[2的根号2次方|格尔丰德-施奈德常数]]）和<math>\sqrt{2}^{\sqrt{2}}</math>。
* <math>e^{\pi}</math>（[[E的π次方|格尔丰德常数]]），以及<math>e^\frac{-\pi}{2} = i^i</math>（这是因为<math>e^{\pi} = e^{-i \ln (-1)}</math>是<math>(-1)^{-i}</math>的值之一）。
