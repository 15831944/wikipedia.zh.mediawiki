'''格罗顿学校'''（Groton School）是一所位于[[美国|美国]][[马萨诸塞州|马萨诸塞州]][[格罗顿_(马萨诸塞州)|格罗顿镇]]的[[高中|高中]]。在2003年的美国高中[[排名|排名]]，格罗顿学校的[[教学质量|教学质量]]位居美国第二。

格罗顿学校为[[寄宿|寄宿]][[私立学校|私立学校]]，学校共有350左右名[[学生|学生]]，为[[八年级|八年级]]至[[十二年级|十二年级]]。同时，格罗顿学校为美国诞生了两位美国[[总统|总统]]以及很多至今活跃在[[政治舞台|政治舞台]]的[[人物|人物]]。

== 历史 ==
格罗顿学校于1884年由[[恩迪葛·皮博迪|恩迪葛·皮博迪]]（[[:en:Endicott_Peabody_(educator)|Endicott Peabody]]）创建，學校制度則模仿皮博迪自己少年时在[[英国|英国]][[公學|公學]][[切尔滕纳姆学院|切尔滕纳姆学院]]的受學經驗。

格罗顿学校在美国历史上扮演了成功的教学[[角色|角色]]，一年至少有25名学生能进入[[哈佛大学|哈佛大学]]。

== 学生 ==

格罗顿学校大部分来自美国本地，其他学生来自于世界各地。

格罗顿学校拥有175名[[男孩|男孩]]以及177名[[女孩|女孩]]（目前总共为352名学生，按照招生数额而时时更改）。在格罗顿学校，30%的学生能够拿取[[奖学金|奖学金]]。即使这样，进入格罗顿学校的[[学费|学费]]还是非常高的。学校在招收学生时甚至需要考虑到学员的[[家庭情况|家庭情况]]。

格罗顿学校同时拥有国际学生，包括[[韩国|韩国]]、[[香港|香港]]、[[中国大陆|中国大陆]]、[[英国|英国]]、[[瑞士|瑞士]]、[[瑞典|瑞典]]以及[[加拿大|加拿大]]等学生。由于格罗顿学校的[[名气|名气]]以及[[质量|质量]]，进入格罗顿学校的[[竞争|竞争]]是非常强的（15%入取，每年三月份公布），格罗顿学校学生有平均48％左右的学生来自世界各地。每一年，格罗顿学校只招收不到120名学生，其中包括美国本地及国际学生。

在学生中，97%的学生为[[寄宿|寄宿]]学生而3%的学生为非寄宿学生。

== 师资 ==
格罗顿学校拥有[[滑雪场|滑雪场]]、[[图书馆|图书馆]]、[[划船场|划船场]]、[[艺术馆|艺术馆]]、[[剧院|剧院]]等设施。另外，格罗顿学校的老师有82％以上持[[硕士|硕士]]或[[博士|博士]]学位，绝大多数老师毕业于[[常青藤盟校|常青藤盟校]]、[[麻省理工大学|麻省理工大学]]等名校。师生[[比例|比例]]为1比2。

==教义==
格罗顿学校是[[美国圣公会|聖公會]]学校，学生在周一到周五（周三除外）都要到学校的礼堂中[[祷告|祷告]]。
