'''格温侍'''（{{lang-en|'''GwenPool'''}}），原名'''格温多林·格温·普爾'''（Gwendolyn "Gwen" Poole），是一名出現在[[漫威漫畫|漫威漫畫]]的虛構人物。起源於[[女蜘蛛人·關|女蜘蛛人·關]]，從2015年6月開始[[秘密戰爭|秘密戰爭]]的[[關·史黛西|關·史黛西]]變體封面。

[[漫威漫畫|漫威漫畫]]曾出版了兩個以格温侍為主的故事，一個是[[霍華鴨|霍華鴨]]（Howard Duck）系列的備份故事，另一個是Gwenpool Holiday Special。儘管命名為「格温侍」，但角色與[[死侍|死侍]]或關·史黛西沒有任何關係。在霍華鴨的故事中提到格温侍的名字其實是格温普爾（Gwen Poole）。後來2016年4月，製作團隊整合起兩個故事，開始出版名為「The Unbelievable Gwenpool」的漫畫。

==出版歷史==
2015年6月，隨著蜘蛛格温的興起，[[漫威漫畫|漫威漫畫]]開始把關·史黛西打造成其他漫威旗下角色，比如[[奇異博士|奇異博士]]（Doctor Strange），[[格魯特|格魯特]]（Groot）和[[金剛狼|金剛狼]]（Wolverine）<ref>{{cite web|first=|last=|url=http://marvel.com/news/comics/24438/gwen_takes_over|title=Gwen Takes Over|publisher=Marvel.com|date=14 April 2015|access-date=2017-12-01|archive-url=https://web.archive.org/web/20150827021439/http://marvel.com/news/comics/24438/gwen_takes_over|archive-date=2015-08-27|dead-url=yes}}</ref>等，也包括死侍<ref>{{cite web |first=Rich |last= Johnston |url=http://www.bleedingcool.com/2015/08/21/and-finally-has-marvel-noticed-that-gwenpool-is-a-thing-now/|title=And Finally… Has Marvel Noticed That Gwenpool Is A Thing Now?R|publisher=[[BleedingCool.com|BleedingCool.com]] |date= August 21, 2015}}</ref>。直到漫威編輯喬丹·懷特（Jordan White）看到有不少人裝扮成一個甚至沒有自己漫畫的角色後，便和作家克里斯托弗·黑斯廷斯（Christopher Hastings）商討，製作一個格温侍的故事。<ref>{{cite web |first= |last= |url=http://www.comicbookresources.com/article/marvel-announces-gwenpool-one-shot-howard-the-duck-back-up-stories|title=MARVEL ANNOUNCES "GWENPOOL HOLIDAY SPECIAL," "HOWARD THE DUCK" BACKUP STORIES|publisher=Comicbookresources |date=September 11, 2015}}</ref>

本來只打算製作成一次性的故事，但在與霍華鴨的編輯Will Moss見面後，提出創造像死侍般打破[[第四面牆|第四面牆]]的角色，「但是她知道自己在漫畫書中的是因為她來自漫威漫畫書是實際存在的世界」同時加入「不認為自己的行為會造成任何後果」，最後黑斯廷斯總結「像利用漫畫書的知識在漫畫世界玩[[俠盜獵車手|俠盜獵車手]]般」，而和死侍最大不同是格溫侍沒有任何訓練或超能力。<ref>{{Cite web |url=https://massappeal.com/marvel-gwenpool-2-christopher-hastings-interview/ |title=WRITER CHRISTOPHER HASTINGS PROVIDES AN EXCLUSIVE LOOK AT MARVEL’S ‘GWENPOOL #2’ |access-date=2017-12-01 |archive-url=https://web.archive.org/web/20171019164858/https://massappeal.com/marvel-gwenpool-2-christopher-hastings-interview/ |archive-date=2017-10-19 |dead-url=yes }}</ref><ref>[http://heroes.direct/interviews/christopher-hastings-interview/ Interview: Christopher Hastings on creating Gwenpool and writing Vote Loki]</ref>

由於2018年的漫威新書中沒有其名字以有網民懷疑被腰斬。<ref>{https://www.bleedingcool.com/2017/12/19/america-gwenpool-cancel-2018-solicits/ No Gwenpool or America From Marvel in March 2018 – Cancelled?]</ref>

==人物故事==
===背影===
通過某種未知方式，格温普爾由她稱為「真實世界(Earth-TRN565)」的宇宙來到漫威的宇宙616。而她拒絕成為一個多出來的背景人物。結果她來到一間超級英雄服裝店，向名叫Ronnie的裁縫要求了一件獨特的服裝，只是Ronnie錯把格温的本名「Gwen Poole」當成她的英雄名「Gwen Pool」，結果給她弄了一套死侍風格的服裝。
