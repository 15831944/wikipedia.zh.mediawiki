'''格鲁派'''或'''格魯巴派'''（{{bo|z=དགེ་ལུགས་པ་|w=Dge-lugs-pa|p=Gelug 或 Gelug-pa}}），為[[藏传佛教|藏传佛教]]四大派之一，由西藏[[班智達|班智達]]——[[宗喀巴|宗喀巴·羅桑札巴]]大师於十四世紀所创立，为[[阿底峽|阿底峽]]尊者[[噶當派|噶當派]]的後裔之一，故又称作'''噶丹派'''（Ganden）。'''格魯'''，意為“善規”，亦稱'''黃教'''，在[[藏区|藏区]]有很大的影响<ref>{{cite web |url = http://www.chiculture.net/1704/html/1704d15/1704d15.html |title = 吐蕃文化}}</ref>。格鲁派主張[[中觀應成|中觀應成]]。

== 歷史發展 ==
明朝时，[[青海|青海]]藏人[[宗喀巴|宗喀巴]]以[[阿底峽|阿底峽]]尊者所創的[[噶當派|噶當派]]教義為主，融合各宗各派的精華，創立[[格魯派|格魯派]]，又被称为'''新噶当派'''。宗喀巴開宗立派之後，受到熱烈歡迎，格魯派因此代替[[噶舉派|噶舉派]]，接管了全藏的政教事務，成為第一大教派，格鲁派以嚴倡守戒，而得名'''善規派'''，又被称为'''格律派'''，他们嚴守[[佛教戒律|佛教戒律]]，所以不近[[邪淫|女色]]，廢除[[双身法|双身法]]。格鲁派僧人多戴黃色的帽子，故[[漢地|漢地]]俗稱為'''[[黄教|黄教]]'''。学修并重的学风使其成为藏传佛教中影响力最大的派别。

由于格鲁派是四大教派中最晚出现，也是目前影响最大的一派，且[[宗喀巴|宗喀巴]]原至[[薩迦|薩迦]]的寺院學習，格鲁派几乎吸取了各个教派的各种精華教法，如：
* 有名的[[萨迦派|萨迦]]十三金法
* [[噶举|噶举]]的[[大手印|大手印]]
* [[宁玛|宁玛]]的[[马头明王|马头明王]]
* [[噶当|噶当]]的十六明点（又称“十六滴”）
* [[夏鲁|夏鲁]]的[[时轮金刚|时轮金刚]]
* [[觉域|觉域]]的断法教授

宗喀巴兩大弟子[[根敦朱巴|根敦朱巴]]与[[克主傑|克主傑]]的法脈逐漸興盛，[[根敦朱巴|根敦朱巴]]的繼承人號稱[[達賴|達賴]]；[[克主傑|克主傑]]的繼承人則號稱[[班禪|班禪]]，達賴與班禪是西藏的兩大政治與宗教領袖，历代[[甘丹寺|甘丹寺]]主持[[甘丹赤巴|甘丹赤巴]]是「格魯派教法」的領袖，格鲁教法的当然[[法台|法台]]。法台几年一任，不[[世袭|世袭]]。

== 學經制度 ==
格魯派重視僧人學經、[[辯經|辯經]]﹐有系統的學經制度，先顯後密。格魯派擁有眾多寺院，發揮教育功能。

其教育科目總稱「[[五明處|五明處]]」，即廣義的佛學。各學堂有10-20個學寮，如[[噶丹寺|噶丹寺]]有二學堂，各有13學寮。各學堂教授內容多少互異，科目共有：1.[[因明學|因明學]]、2.[[般若|般若]][[中觀學|中觀學]]、3.[[中觀學|中觀學]]、4.[[律學|律學]]、5.[[俱舍|俱舍]]學五個領域。其中[[般若|般若]][[中觀學|中觀學]]主要學習《[[現觀莊嚴論|現觀莊嚴論]]》的[[師子賢|師子賢]]《小注》，[[中觀學|中觀學]]則主要學習《[[入中論|入中論]]》及[[宗喀巴|宗喀巴]]的《入中論釋》。學僧要背誦指定的經論，透過爭論加深理解，成績優異者進而學習[[密教|密教]]，學習目的是要成為一切智者。<ref>原田覺著，許明銀譯：〈西藏佛教的中觀思想〉，載山口瑞鳳等：《西藏學與西藏佛學》（台北：文殊出版社，1986），頁151-152。</ref>

宗喀巴規定，《現觀莊嚴論》、《中觀論》、《俱舍論》、《釋量論》和《律經本論》五部經典為該派學僧必修之課稱為顯宗[[五部大論|五部大論]]。凡考取甘丹﹑哲蚌﹑色拉三大寺系統的[['''拉然巴格西'''|'''拉然巴格西''']]學位考試﹐再入推舉進入上下密院學習[[密宗|密宗]]四部(即[[事續|事續]]﹑[[行續|行續]]﹑[[瑜伽部|瑜伽部]]﹑[[無上瑜伽部|無上瑜伽部]])﹐經過逐級的職位升遷﹐可升任甘丹赤巴 。

=== 學位制度 ===
[[格西|格西]]意為[[善知識|善知識]]，或良師益友，相當於漢傳佛教的和尚、親教師，也有譯為佛學博士。 [[格西|格西]]學位傳統上只授予在顯宗學院學習有一定成就的學經僧人，而不授予修習密宗的僧人。 在格魯派各寺院中對於[[格西|格西]]的等級設置並不統一，學位名稱也不盡相同。

一般來說，拉薩三大寺的[[格西|格西]]學位最權威，它的制度也最完善、最健全，所以下面僅就三大寺的四種[[格西|格西]]學位作概要性的敘述。

*第一等[[格西|格西]]是'''拉然巴'''，意思是拉薩的博學高明之士，是[[格西|格西]]中最高等級的學位；
*第二等[[格西|格西]]是'''措然巴'''，意思是全寺的卓越高明之士，比拉然巴[[格西|格西]]要稍低一等。

這兩種[[格西|格西]]，要經過多次考試後，還要經過當時[[噶廈|噶廈政府]]最後的審定方能取得。

*第三等[[格西|格西]]是'''林賽'''，意思是從寺院裡選拔出來的有才學的人；
*第四等[[格西|格西]]是'''多然巴'''，意思是在佛殿門前石階上經過辯論問難考取的[[格西|格西]]。 這兩種[[格西|格西]]的考試和認定，不必經過[[噶廈|噶廈政府]]的認可，由各寺自己決定。

== 教義特色 ==
=== 转世 ===
随着众生的因缘，每一位[[活佛|活佛]]都示现不一样的情况。但众生因缘尽了，喇嘛则会示现圆寂或没有讲法（当众生没具足福德因缘或没有众生请法的缘起而致）。

== 地位 ==
格鲁派四大[[活佛|活佛]]（祖古）在当今有很高的地位，为所有藏传佛教徒崇拜。

* [[达赖喇嘛|达赖喇嘛]]，主要在[[前藏|前藏]]
* [[班禅额尔德尼|班禅额尔德尼]]，主要在[[后藏|后藏]]
* [[章嘉呼图克图|章嘉呼图克图]]，主要在[[内蒙|内蒙]]
* [[哲布尊丹巴呼圖克圖|哲布尊丹巴呼圖克圖]]，主要在[[外蒙|外蒙]]

教義方面，[[格魯派|格魯派]]教法的領導人是甘丹寺的[[方丈|方丈]]，[[甘丹赤巴|甘丹赤巴]]。

== 寺院 ==
格魯派的三大寺指位於[[拉萨|拉萨]]的三個格魯派寺廟，分別是：
* [[甘丹寺|甘丹寺]]（'''黄教母寺'''）
* [[哲蚌寺|哲蚌寺]]
* [[色拉寺|色拉寺]]
格魯派的四大寺表示拉薩的三大寺與
* [[扎什伦布寺|扎什伦布寺]]，位于西藏[[日喀则|日喀则]]
再加上
* [[塔尔寺|塔尔寺]]，位于今[[青海|青海]][[西宁市|西宁市]]附近
* [[拉卜楞寺|拉卜楞寺]]，位于今[[甘肃|甘肃]][[夏河县|夏河县]]
為格魯派的六大寺。六大寺中都有顯教的佛學院。

其他格魯派管理的有名寺廟有：
* [[大昭寺|大昭寺]]，位于西藏拉萨(尽管大昭寺是各教派共尊的神聖寺院，但现在以格魯派为主)
* [[大藏寺|大藏寺]]，位于今[[四川|四川]][[阿壩州|阿壩州]]
* [[大乘寺|大乘寺]]，位于[[尼泊尔|尼泊尔]]
