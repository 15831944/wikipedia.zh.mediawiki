'''格哈特“格里”·诺伊格鲍尔'''<ref>Neugebauer's first name is pronounced ''Gary'', not ''Jerry''.</ref>（{{lang-en|'''Gerhart "Gerry"  Neugebauer'''}}，{{bd|1932年|9月3日|2014年|9月26日}}），[[美国|美国]][[天文学家|天文学家]]，在[[红外天文学|红外天文学]]领域做出了开创性的工作。

== 生平 ==
诺伊格鲍尔出生于[[德国|德国]][[哥廷根|哥廷根]]，他是奥地利-美国[[数学家|数学家]]和[[科学史学家|科学史学家]][[奧托·紐格伯爾|奥托·诺伊格鲍尔]]和Grete Bruck的儿子。7岁时，他移民到了美国。1954年，他在[[康奈尔大学|康奈尔大学]]获得了物理学学士学位。1960年，他在[[加利福尼亞理工學院|加州理工学院]]获得了物理学博士学位，论文主题是源于[[氘|氘]]的正负[[π介子|π介子]]的光产生。

在为[[美国陆军|美国陆军]]服役期间，他驻扎在[[喷气推进实验室|喷气推进实验室]] 为[[美国陆军军械部队|美国陆军军械部队]]工作到1962年。他于同年进入了[[加州理工学院|加州理工学院]]，担任助理教授。1970年，他成为物理学正式教授。1985年，他被任名为[[霍华德·休斯|霍华德·休斯]]教授。1988年，他被任命为物理学、数学和天文学分会的主。在他去世前，他是[[罗伯特·密立根|罗伯特·安德鲁斯·密立根]]物理学教授、荣誉退休教授和[[亚利桑那大学|亚利桑那大学]][[斯图尔德天文台|斯图尔德天文台]]的附属成员。

1980年至1994年间，诺伊格鲍尔担任[[帕洛马山天文台|帕洛马山天文台]]的主任。

诺伊格鲍尔与[[地球物理学|地球物理学家]]马西娅·诺伊格鲍尔结婚，她是[[喷气推进实验室|喷气推进实验室]][[太阳风|太阳风]]研究的先驱。他们居住在[[亚利桑那州|亚利桑那州]][[图森市|图森市]]。 

2014年9月26日，诺伊格鲍尔由于[[小腦萎縮症|小脑萎缩症]]逝世于图森市。<ref>https://www.nytimes.com/2014/10/03/us/gerry-neugebauer-pioneer-in-space-studies-dies-at-82.html?_r=0</ref>

== 学术研究 ==
诺伊格鲍尔活跃于红外天文学领域，并在[[行星|行星]]的红外研究上发挥了领导作用。 此外，得益于他在[[紅外線天文衛星|红外线天文卫星]]（IRA）和红外处理和分析中心（IPAC）上的活动，他领导了[[恒星|恒星]]，[[银河系|银河系]]和[[星系|河外星系]]的地面和空间的红外线研究。他和同事们在 [[威尔逊山天文台|威尔逊山]] 和帕洛马天文台的观测发现了天空中数以千计的红外源，并得到第一个银河系中心的红外影像。 他和[[羅伯·雷頓|罗伯特·B·雷顿]]一起，完成了第一个红外巡天——2微米巡天，编目了5000多个红外源。 

诺伊格鲍尔与埃里克·贝克林一起，发现了[[BN天体|贝克林–诺伊格鲍尔天体]]——一个位于[[猎户座大星云|猎户座星云]]的强烈的红外辐射源。它是[[波长|波长]]小于10微米[[波段|波段]]里最亮的天体。

他在设计和建造[[夏威夷|夏威夷]]的[[凱克天文台|W.M.凯克天文台]]上发挥了作用。
