'''格里戈里·葉菲莫维奇·拉斯普京'''（{{lang-ru|Григо́рий Ефи́мович Распу́тин}}，{{IPA-ru|ɡrʲɪˈɡorʲɪj jɪˈfʲiməvʲɪtɕ rɐˈsputʲɪn|}}；{{bd|1869年|1月22日|1916年|12月29日|catIdx=Rasputin, Grigori}}），又譯'''拉斯普丁'''、'''拉斯普欽'''或'''拉斯普廷'''，[[俄罗斯|俄罗斯]][[萨拉托夫省|萨拉托夫省]]人，[[帝俄|帝俄]]時代[[尼古拉二世|尼古拉二世]]时的[[神秘主义|神秘主义]]者，被認為是[[東正教|東正教]]中的[[佯狂者|佯狂者]]之流。因醜聞百出，引起公憤，為[[费利克斯·尤苏波夫|尤苏波夫]][[亲王|亲王]]、[[德米特里·巴甫洛维奇|狄密翠]][[大公|大公]]、[[弗拉基米爾·普利希克維奇|普利希克维奇]][[議員|議員]]等人合謀[[暗殺|刺死]]。

== 出身 ==
'''拉斯普京'''于1869年1月22日出生於俄罗斯[[萨拉托夫省|萨拉托夫省]]的農家，父亲叶菲姆·维尔金早年好赌，后携家移居[[西伯利亚|西伯利亚]][[秋明|秋明]]地区的波克罗夫斯科耶村，并改姓“诺维赫”（意为“新人”），成为[[富农|富农]]，后因妻子、长子、独女相继去世而家道中落。格里高利年輕時是個無賴，还做过偷马贼，被同村人称为“拉斯普京”，意为“淫逸放荡”（{{lang|ru|распутин}}）。他在三十多岁的时候以散播[[预言|预言]]和施展[[巫医|巫医]]为業。由于拉斯普京预言出俄罗斯某地的三月干旱，以及醫療尼古拉二世叔父尼古拉大公的狗，因而名声大噪。1905年俄国发生革命后，他被[[黑色百人团|黑色百人团]]成员发现，并带到[[帝都|帝都]][[聖彼得堡|聖彼得堡]]。

当时的[[俄国皇帝|俄國沙皇]]为[[尼古拉二世|尼古拉二世]]，[[皇后|皇后]][[亞歷山德拉·費奧多羅芙娜皇后_(尼古拉二世)|亞歷山德拉]]笃信[[神秘主义|神秘主义]]，喜好招待“神僧”、“圣童”，并举行降灵仪式。由于[[阿列克谢_(俄国皇储)|阿列克谢皇太子]]患有[[血友病|血友病]]，所以一些皇族成员和沙皇宠臣在1906年举荐拉斯普京来为太子治病。拉斯普京擅长[[催眠术|催眠术]]，对皇储的病情不无稳定之效，自此大得皇后信任，可以自由出入宫廷；他平时称与他年龄相若的沙皇尼古拉二世为“爸爸”，称皇后亚历山德拉为“妈妈”，称皇太子[[阿列克谢_(俄国皇储)|阿列克谢]]为“小娃娃”。拉斯普京掌握着对-{皇后}-的巨大影响力，被誉为“圣人”，受到贵族妇女的崇拜，在[[圣彼得堡|圣彼得堡]]纵酒宣淫，无法无天，甚至官员的任命都要先博得他的同意，由其[[遊说|遊说]]-{皇后}-来得到批准。许多[[官吏|官吏]]和[[贵族|贵族]]因为得罪拉斯普京而被[[罢免|罢免]]，如[[外務大臣|外務大臣]]伊兹沃利斯基、俄罗斯[[东正教|东正教]]最高会议检察总长卢基扬诺夫等。

由于早年遊历时曾目睹[[伏尔加德意志人|伏尔加德意志人]]的富裕生活，拉斯普京凭借其[[江湖|江湖]]遊历的直觉，在1914年曾极力反对俄国与[[德意志帝国|德意志帝国]]开战。但是由于与德裔-{皇后}-的密切联系，他在[[第一次世界大战|第一次世界大战]]爆发后被公众指责为德国间谍、以及宫廷反动政治的中心人物。到1916年末，上层统治集团中已经出现了许多反对-{皇后}-和拉斯普京的秘密集团，其中最著名的一个由[[费利克斯·尤苏波夫|费利克斯·尤苏波夫]][[亲王|亲王]]领导。尤苏波夫对於拉斯普京对[[沙皇|沙皇]]施加的恶劣影响深恶痛绝，决心通过[[暗杀|暗杀]]拉斯普京来肃清宫廷中的邪恶势力，恢复沙皇的声誉。

拉斯普京生前以生活淫穢出名，死後其[[性器官|性器官]]遭人割除。俄羅斯聖彼德堡一家[[性器官博物館|性器官博物館]]展示了拉斯普京的性器官，拉斯普京之所以吸引諸多女子青睞，甚至男性也被他迷惑，並不單憑他迷人的眼睛，還靠他驚人性器官大小，長達28.5公分。

== 暗杀 ==
[[File:Yusupovirina.jpg|thumb]]
1916年12月29日（[[俄曆|俄曆]]12月16日），[[駙馬|駙馬]][[費利克斯·尤蘇波夫|費利克斯·尤蘇波夫]][[亲王|亲王]]、[[宗室|宗室]]成员[[德米特里·巴甫洛维奇|狄密翠]][[大公|大公]]、[[國家杜馬|帝國議會]]右翼[[议员|议员]][[弗拉基米爾·普利希克維奇|普利希克维奇]]、尤苏波夫的密友[[蘇霍金|蘇霍金]][[大尉|大尉]]和一名[[医生|医生]]在[[圣彼得堡|彼得格勒]]的[[尤苏波夫宫|尤苏波夫宫]]设下陷阱。尤苏波夫以妻子[[伊琳娜|伊琳娜]]公主的美豔肉體为[[美人計|美人計]]诱饵，將拉斯普京誘來[[尤蘇波夫宮|尤蘇波夫宮]]。

拉斯普京的暗杀過程充满了夸张的情节，有很多細節可能是编造的，如下毒一事，一些仆人表示当时并未下毒。由于[[蘇維埃|蘇維埃]]政权建立后很多调查文档被毁或消失，<ref>M. Nelipa (2010), p. 2.</ref>而当事人尤蘇波夫和狄密翠的回忆又不是很可靠，所以当夜的真实细节很难查考。

按照某些说法，当人們給拉斯普京吃了摻有[[氰化鉀|氰化鉀]]的[[蛋糕|蛋糕]]，喝了一瓶摻有氰化鉀的馬德拉[[葡萄酒|葡萄酒]]，想把他毒死，這些毒物足以毒死五個人，但拉斯普京毫無反應。於是尤蘇波夫心焦難耐，向拉斯普京開了一槍，眾人以為他已經死掉，正要處理屍體時，拉斯普京突然甦醒過來，扼住尤蘇波夫的喉嚨，說「費利克斯，費利克斯！我明日就把你[[絞死|絞死]]！」，隨後掙脫了眾人的壓制，跑到庭院。普利希克維奇追出屋外，向拉斯普京開了三槍，最後一槍正中其頭部。当人們將他拖進屋內，拉斯普京再度甦醒過來，尤蘇波夫用[[啞鈴|啞鈴]]猛擊其[[太陽穴|太陽穴]]，將其打得頭破血流，但拉斯普京仍未死去，只是被再度擊昏。拉斯普京最後被扔入[[莫伊卡河|莫伊卡河]]（[[涅瓦河|涅瓦河]]支流）的一個冰洞中，屍體於次日被發現。[[法醫|法醫]][[驗屍|驗屍]]結果表明他是[[溺水|溺水]]而死的，拉斯普京在冰面下的冰水中存活了8分鐘之久。<ref>[http://info.wenweipo.com/index.php?action-viewnews-itemid-35819-page-4 新華網.史上最「命硬」人物大排行--拉斯普廷 ] {{Wayback|url=http://info.wenweipo.com/index.php?action-viewnews-itemid-35819-page-4 |date=20140303161824 }}</ref>

拉斯普京的尸体经过[[防腐|防腐]]处理后被运到[[皇村|皇村]]，亚历山德拉-{皇后}-在那为他修建了地下墓穴，并计划在墓穴上修建[[修道院|修道院]]。[[二月革命|二月革命]]后拉斯普京的尸体被进驻皇村的[[士兵|士兵]]发现，运到[[圣彼得堡|彼得格勒]]遊街示众，然后被焚尸扬灰。

[[File:Dead_Rasputin.jpg|thumb]]
尤苏波夫亲王和狄密翠大公因[[暗杀|暗杀]]拉斯普京而被[[软禁|软禁]]，随后[[流放|發配]]到外地。两人因此在躲過[[十月革命|十月革命]]時，[[共產黨|共產黨]]對[[宗室|皇親]][[外戚|國戚]]的整肅。尤苏波夫在十月革命后返回[[圣彼得堡|彼得格勒]]时，由于对其暗杀拉斯普京的尊敬，共產黨甚至派[[苏联红军|紅軍]]士兵在其宫殿外站岗守卫。

== 评价 ==
[[俄羅斯歷史|俄羅斯歷史]]学家对拉斯普京的评价以负面结论为多。苏联史学家认为他是“罪恶的沙皇政权”的代表。欧美的历史学家也大多认同这一评价。但是1990年代以后，在俄罗斯史学界出现了为拉斯普京正名的学者，他们试图引用沙皇密探对拉斯普京的一些监视报告来为其洗刷名誉，将其描绘为宗教圣人。

== 有关拉斯普京的軼闻 ==
* 据说拉斯普京生前曾经作出[[预言|预言]]，[[罗曼诺夫王朝|罗曼诺夫王朝]]将在他死后3个月之内崩溃。他于1916年12月底被[[謀杀|謀杀]]，次年3月8日俄国发生“[[二月革命|二月革命]]”，罗曼诺夫王朝灭亡，終結了[[帝俄|帝俄]]時代。

* 拉斯普京生前不喜欢[[洗澡|洗澡]]，据他自称洗澡会令他的[[法術|法術]]减弱乃至丧失。

* 拉斯普京生前以生活淫穢出名，每天都臨幸眾多女子。

* [[彼得格勒蘇維埃|彼得格勒苏维埃]]（彼得格勒軍人與工人委員會）焚毁拉斯普京的尸体时，目击者声称敛放其尸体的铅皮和锌皮棺材烧至熔化时，他的尸体还未被烧毁，并且从[[棺材|棺材]]裏坐了起来，眼睛也睁开了。[[紅軍|紅軍]]不断向尸体上泼洒[[煤油|煤油]]，10个小时后才将其烧为灰烬。许多[[贵族|贵族]]妇女化装来到其焚尸地拾[[骨灰|骨灰]]，作为[[圣物崇拜|圣物崇拜]]。

* 拉斯普京喜好收集与其發生過關係的[[处女|处女]]頭髮。1977年[[圣彼得堡|列宁格勒]]市政府拆除其居住过的楼房时，在花园裏发现了许多装有女人頭髮的箱子。
