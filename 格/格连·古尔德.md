'''格伦· 赫伯特·顧尔德'''（'''Glenn Herbert Gould'''，{{bd|1932年|9月25日|1982年|10月4日|catIdx=G}}）出生、去世於[[加拿大|加拿大]][[多倫多|多倫多]]，被公認為是[[20世紀|20世紀]]最著名的古典[[鋼琴家|鋼琴家]]之一，他同時也是一位[[管风琴|管风琴]]家、[[廣播|廣播]]節目[[主持人|主持人]]與[[作曲家|作曲家]]，以演奏[[约翰·塞巴斯蒂安·巴赫|约翰·塞巴斯蒂安·巴赫]]乐曲而闻名於世。於31歲（[[1964年|1964年]]）以后停止公开演奏，转向[[录音|录音]]和其他項目。

== 生平 ==
[[File:Glenn_Gould_as_a_child.jpg|thumb]]

</ref><ref>{{harvtxt|Bazzana|2003|p=30}}</ref>。童年时代从母亲学[[钢琴|钢琴]]，十岁进入[[多伦多皇家音乐学院|多伦多皇家音乐学院]]学习钢琴、[[管风琴|管风琴]]和[[音乐理论|音乐理论]]，他的钢琴老师是一位[[智利|智利]]的[[作曲家|作曲家]]、[[钢琴家|钢琴家]]，名為[[Alberto_Guerrero|Alberto Guerrero]]。

1945年，顧尔德十三岁，第一次作管风琴演出。1946年，顧尔德在[[多伦多交响乐队|多伦多交响乐队]]伴奏下演奏[[贝多芬|贝多芬]]《第四钢琴协奏曲》。

1955年，他第一次在[[美國|美國]][[華盛頓|華盛頓]]演出，演奏的曲目是[[巴哈|巴哈]]的[[哥德堡變奏曲|哥德堡變奏曲]]，經由這次的演出機會，顧爾德與[[哥倫比亞廣播公司|哥倫比亞廣播公司]]簽約，於同年6月26日起至[[錄音室|錄音室]]錄製同一首演出曲目。

1957年，顧尔德前往[[苏联|苏联]]演出，成为[[第二次世界大战|第二次世界大战]]后第一个到苏联演出的[[北美|北美]]音乐家。

1964年，顧尔德在[[洛杉矶|洛杉矶]]举行最后一次公开演出。此后顧尔德专心从事唱片[[录音|录音]]、[[广播|广播]]和[[纪录片|纪录片]]制作，[[写作|写作]]，偶爾有[[作曲|作曲]]。

1982年顧尔德因[[中风|中风]]在[[多伦多|多伦多]]逝世，葬于多伦多安乐山公墓。[[多伦多CBC公司|多伦多CBC公司]]总部门前有顧尔德坐像。

== 特色 ==
顧尔德演奏[[巴赫|巴赫]]时喜歡把每个[[声部|声部]]弹得清晰、无[[连音|连音]]，他幾乎不用踏板，并用上不同的[[音色|音色]]。他改装了他的[[史坦威公司|史坦威]]CD318号[[钢琴|钢琴]]，它的音色類似[[大鍵琴|大鍵琴]]；顧爾德演奏时坐的位置很低，这种[[坐姿|坐姿]]对顧尔德在演奏快速乐段时保持清晰起到了一定作用。但也有人指出，这种姿势不适合演奏[[19世纪|19世纪]]的作品。

== 怪傑 ==
[[File:Glenn_Gould's_chair.JPG|right]]
顧尔德演奏时总是同时哼[[歌唱|唱]]乐曲。他声称这种哼唱是[[潛意識|潛意識]]的，源于他幼年接受母亲的[[音乐教育|音乐教育]]时养成的习惯。他也因演奏时所作出的各种奇怪的身体动作而闻名。顧尔德坚持要坐在他父亲为他制作的一把折叠椅上才能演奏。即使这把[[椅子|椅子]]已经十分老舊，他仍然坚持要使用它。因此在他留下的录音中，可以听到他的哼唱声和椅子发出的声音。

顧尔德总是避免与他人有任何身体接触。后来，他拒绝与他人面对面地交谈，而是通过[[电话|电话]]及[[信件|信件]]来进行[[沟通|沟通]]。他也会自己采访自己，并发表采访稿。他终生未婚。他对孤独与隔离特别感兴趣，曾为[[加拿大广播公司|加拿大广播公司]]制作有关[[加拿大北部|加拿大北部]]地区的「实验广播剧」《孤独三部曲》（Solitude Trilogy），反映孤独与隔离对人所产生的影响。他说自己是最后一个[[清教徒|清教徒]]（the last puritan）。

顧尔德本人有时并不认为自己是[[钢琴|钢琴]]演奏家，而自认为是[[广播|广播]][[業者|業者]]，因为其在[[加拿大|加拿大]]也是广播节目的[[主持人|主持人]]。此外，他曾声称相对于钢琴演奏而言，自己更喜欢[[作曲|作曲]]与[[写作|写作]]。

顧尔德总是穿着暖和的[[服装|衣服]]（例如：[[手套|手套]]、[[圍巾|圍巾]]、[[帽子|帽子]]）。

顧尔德对某些[[药物|药物]]有依赖性。其中的一些药物有[[副作用|副作用]]，对他的[[健康|健康]]产生了不良影响。他对自己的健康状况极度关心，常常为自己[[测量|测量]][[血压|血压]]，并担心自己的手的安全。

== 录音 ==
顧尔德与J·S·[[巴赫|巴赫]]的《[[哥德堡变奏曲|哥德堡变奏曲]]》（Goldberg Variations）之间建立了一种十分紧密的联系，他常常在独奏会上演奏整部作品或是部份选段。对于这部[[作品|作品]]，顧尔德最早的完整录音是在1954年6月21日[[加拿大廣播公司|加拿大广播公司]]（CBC）的广播录音。1955年由[[哥伦比亚广播公司|哥伦比亚广播公司]]（CBS）重录，当时，顧尔德才22岁，[[哥伦比亚广播公司|哥伦比亚广播公司]]的主管认为：「像《[[哥德堡变奏曲|哥德堡变奏曲]]》这样的曲子，是已成名的[[钢琴家|钢琴家]]的曲目，不宜初出茅庐的新手」但顧爾德十分有[[自信|自信]]，最后还是录了音。1981年，他重录了《[[哥德堡变奏曲|哥德堡变奏曲]]》。而這次的[[錄音|錄音]]不同于他早期的录音。这两个迥然不同的录音都获得了巨大迴響。顧尔德早年（1955年前後）的录音速度较快；晚年的录音速度則慢了許多。

他还录制了《[[十二平均律钢琴曲集|十二平均律钢琴曲集]]》、《[[赋格的艺术|赋格的艺术]]》、《[[创意曲集|创意曲集]]》、等一大批[[巴赫|巴赫]]的作品。其演奏的《[[十二平均律|十二平均律]]》第二册的第一首作品，[[C大調|C大调]][[前奏曲|前奏曲]]与[[赋格|赋格]]被[[美国国家航空航天局|美国太空总署（NASA）]]确定为人类突出贡献，被作为[[人|人类]]文化代表之一应用于数项太空探测计划中。

除此之外，顧尔德还录制了[[莫扎特|莫扎特]]、[[贝多芬|贝多芬]]、[[西贝柳斯|西贝柳斯]]、[[吉彭斯|吉彭斯]]、[[斯克里亚宾|斯克里亚宾]]、[[欣德米特|欣德米特]]......等其他作曲家的作品。他所录制的[[阿诺·勋伯格|阿诺·勋伯格]][[钢琴|钢琴]]作品[[全集|全集]]亦受到高度赞赏。顧尔德对[[肖邦|肖邦]]的[[作品|作品]]评价较低，但也录制过[[肖邦|肖邦]]的《[[第三钢琴奏鸣曲|第三钢琴奏鸣曲]]》。他也曾把[[瓦格纳|瓦格纳]]的管弦乐改编成[[钢琴独奏曲|钢琴独奏曲]]并录制了唱片。

一直到1982年顧尔德逝世前，他的唱片已出售1250000张；到2000年，顧尔德1981年录制的《[[哥德堡变奏曲|哥德堡变奏曲]]》已出售超过两百万张。
