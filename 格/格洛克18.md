'''格洛克18'''（Glock 18）是由[[格洛克|格洛克]]公司設計及生產的[[全自動手槍|全自動手槍]]。

== 設計 ==
格洛克18与半自动的格洛克17外型長度相同，最大的外观差别是[[手槍套筒|套筒]]后部有[[擊發調變槍械|快慢機]]。它有如在滑套上裝有射擊模式選擇鈕的[[格洛克17|格洛克17]]，可選擇全自動或單發射擊[[9mm魯格彈|9毫米鲁格弹]]。

選擇鈕負責釋放第一道[[撞針|撞針]]的保險，當射手扣下扳機時立刻釋放撞針來擊發子彈，而當[[手槍套筒|套筒]]往復運行時，因無第一道保險的限制而能[[自動火器|全自動射擊]]；向下為全自動模式，向上為單發模式。由于设计其射击控制机构极其简单，甚至没有增加减速机构，因此格洛克18的理论射速极高，为每分钟1,300发。

雖然兩者大小和外形相似，甚至有些使用者聲稱「不用更換槍套」，但兩者主要運作部件還是有稍微不同，格洛克公司為控制市場方向（意指不向民間銷售），所以兩者部件亦不可交換的使用。

格洛克18標凖[[彈匣|彈匣]]為17發，並對應格洛克17的17／19<ref><small>31／33發為增大容量弹匣底座</small></ref>發標準彈匣，亦有31／33發的大容量彈匣，但大容量彈匣不太普及。

== 採用 ==
格洛克18最早是應[[奧地利聯邦內政部|奧地利聯邦內政部]][[眼鏡蛇作戰司令部|眼鏡蛇作戰司令部]]要求而研發，主要是供[[特種警察部隊|特種警察部隊]]及[[特種部隊|特種部隊]]與[[要員保護|要員保護]]單位所使用。在遭遇持槍[[恐怖分子|恐怖分子]]襲擊時，使用格洛克18的部隊可用手槍達1300發/分的高射速構成彈幕，壓制暴徒或掩護政要迅速撤離現場。

執法人員為避免引人注意，在行動中使用的格洛克18大多使用標凖[[彈匣|彈匣]]，很少裝上超長型[[彈匣|彈匣]]，如此[[恐怖分子|恐怖分子]]及[[平民|平民]]根本無從察覺人員所配帶的是[[全自動手槍|全自動手槍]]。

== 民間的反應 ==
由于是受限制的全自动武器，生产量和销售量都很小，因此世界各地都很少见。有些國家或地區受全自動武器受法例管制，所以格洛克18不能民間市場銷售，而格洛克17不受此法例管制。

在[[美國|美國]]，格洛克18<nowiki>只</nowiki>提供給執法部門，因此在[[美國|美國]]民間只有小量格洛克18依{{tsl|en|National Firearms Act|國家武器管制法}}私下擁有。但由於格洛克17可改裝成衝鋒槍，且相關運作部件的製作也不難，因此雖然是非法行為，民間仍然有人私下自製及使用相關部件令自己的[[格洛克手槍|格洛克手槍]]可以[[自動火器|全自動射擊]]。

== 改進型 ==
格洛克18經歷了三次修改版本，最新的版本稱為'''第三代格洛克18'''（3rd generation Glock 18）。1999年开始，新推出的格洛克18在[[手槍套筒|套筒]]下前方設有導軌，以安裝各種戰術配件，另外亦在握把上有手指凹槽。

'''格洛克18C''' —'''枪口补偿装置改進型'''（'''C'''ompensated model），和其名字相同，就是在的枪管前裝上枪口补偿（抑制枪口跳动）装置版本，其實是在枪管頂部开兩個橢圓形的孔，利用[[自動火器|全自動射擊]]時排出氣體抑制枪口方向。

== 軼事 ==
由于GLOCK18的射速太高，很快就会打光子弹，因此GLOCK公司生产了一种31发容量的弹匣，在更换了“加号底座”后可增加到34发。不过由于31发弹匣太长，有些射手为了便于携事，是使用更换了“加号底座”的标准17发弹匣（增容后为20发）。
值得一提的是，基於格洛克系列手槍可以改裝成衝鋒槍的特性，大量半自動的格洛克17被改裝過，並在一些動作片出現以模仿全自動的格洛克18。
