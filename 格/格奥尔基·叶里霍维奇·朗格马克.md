'''格奥尔基·叶里霍维奇·朗格马克'''（{{lang-en|Georgy Erikhovich Langemak}}，{{bd|1898年|6月26日|1938年|1月11日}}），是一位[[德国|德国]]/[[瑞士|瑞士]]背景的[[前苏联|前苏联]][[火箭|火箭]][[设计师|设计师]]。

1928年初，他与其他几位著名的火箭科学家一起在苏联气体动力学研究所工作，并开发出了无烟火药火箭弹。该研究所后来与另一所火箭研究机构合并成立了喷气推进研究所(RNII)。朗格马克被任命为该研究所副主任。1936年该所完成了火箭滑翔机的技术规格。

1937年[[大清洗|大清洗]]期间，他与研究所主任"[[伊万·捷连季耶维奇·科雷米诺夫|伊万·捷连季耶维奇·科雷米诺夫]]"以及发动机设计师"瓦朗坦·格卢什科"一道，被以莫须有的"反苏维埃组织"的罪名遭苏联政府解职并逮捕<ref name=hrono>{{cite web

 |title       = Лангемак Георгий Эрихович
 |author      = Александр Глушко

}}</ref>。 

朗格马克和克莱米约诺夫受尽折磨，1938年1月3日来自"莫斯科-中央"，根据内务人民委员部第八部门主管提议，由[[安德烈·亚历山德罗维奇·日丹诺夫|日丹诺夫]]、[[维亚切斯拉夫·米哈伊洛维奇·莫洛托夫|莫洛托夫]]、[[拉扎尔·莫伊谢耶维奇·卡冈诺维奇|卡冈诺维奇]]和[[克利缅特·叶夫列莫维奇·伏罗希洛夫|伏罗希洛夫]]签署的163名"第一类镇压对象"(枪毙)名单中他被列为第73号<ref name=signatures>{{cite web

他是后来成功应用于[[第二次世界大战|二战]]中著名[[卡秋莎火箭炮|卡秋莎火箭炮]]的非制导火箭弹的设计师和负责人之一，[[月球|月球]]上的[[朗格马克环形山|朗格马克环形山]]就是以他的名字命名的。
