'''格尔木市'''（{{Mong|m=ᠭᠣᠣᠯ<br />ᠮᠣᠳᠤ|Ppp=γool modu|k=Голмод}}，{{bo|z=ན་གོར་མོ་|w=na gor mo}}），是中国[[青海省|青海省]][[海西蒙古族藏族自治州|海西蒙古族藏族自治州]]的[[副地级市|副地级市]]。是[[青藏高原|青藏高原]]继[[西宁|西宁]]、[[拉萨|拉萨]]之后的第三大城市。总面积近12万平方公里。2017年，格尔木市户籍总户数50933户，总人口137570人，全年全市常住人口为240494人。常住人口中，汉族人口占69.82%；各少数民族人口占30.18%。其中：蒙古族占1.77%；藏族占4.20%；回族占20.34%；土族占1.53%；撒拉族占1.02%；其他少数民族占1.32%。市名是[[蒙古语|蒙古语]]，意为河流密（聚）集的地方。

== 历史 ==
*古代是[[羌|羌]]民族地区。
*[[西晋|西晋]]末年，[[辽东|辽东]][[鲜卑|鲜卑]][[吐谷浑|吐谷浑]]部迁入甘青地区，建立起以鲜卑人为中心与诸羌领袖联合政权的吐谷浑国。格尔木地区成为吐谷浑国属地。
*663年[[吐蕃|吐蕃]]灭吐谷浑国，占吐谷浑故地，从此格尔木地区的游牧民族逐渐变成吐蕃民族了。
*13世纪中叶，[[元|元]]朝开始直接统辖整个藏区，[[元顺帝|元顺帝]]时封蒙古贵族卜烟[[帖木尔|帖木儿]]为宁王，率蒙古兵士镇守[[柴达木盆地|柴达木盆地]]。
*[[明|明]]末1636年，[[卫拉特|卫拉特]]蒙古四部之一的[[和硕特|和硕特]]部在其首领[[固始汗|固始汗]]的率领下自今[[新疆|新疆]][[伊犁|伊犁]]进入青海，几年后略定[[卫藏|卫藏]]，统治了全藏区。清[[雍正帝|雍正帝]]1725年开始直接统治蒙藏地区，将青海各部划定游牧地界，统一编为二十九旗，分左、右翼两盟。
*[[中华民国|中华民国]]建立后，沿用[[盟旗制度|盟旗制度]]。1946年裁通新设治局。台吉乃等旗仍归[[都兰县|都兰县]]辖。
*[[中华人民共和国|中华人民共和国]]成立前不久，活动在格尔木地区的主要是来自新疆的[[哈萨克族|哈萨克族]]。
*1949年，中国共产党控制了青海省大部分地区。1950年8月，中共青海省委派哈族代表[[哈里木|哈里木]]等5人来到海西联系哈萨克族，争取让该地人民纳入中华人民共和国的管理体制。
*1953年9月11日　成立了阿尔顿曲克哈萨克族自治区筹备委员会。
*1954年7月15日　正式成立了海西阿尔顿曲克哈萨克族自治区人民政府。
*1956年3月1日　 成立了格尔木工作委员会，为县级领导机构。
*1960年11月17日 撤消格尔木工委，设立格尔木市。
*1966年3月27日　改设为格尔木县。
*1980年6月14日  成立格尔木市。

== 地理 ==
地处青藏高原腹地，辖区由[[柴达木盆地|柴达木盆地]]中南部和[[飞地|飞地]][[唐古拉山镇|唐古拉山镇]]等区域组成，市区位于柴达木盆地中南部[[格尔木河|格尔木河]]冲积平原上，市区平均海拔2780米。面积为5856平方公里的[[察尔汗盐湖|察尔汗盐湖]]是中国最大的钾镁盐矿床，因此格尔木也有中国[[盐湖城|盐湖城]]之称。

=== 气候 ===
属于高原大陆性气候，日温差较大，降水稀少。冬季漫长寒冷但严寒天气较少，夏季短暂而凉爽，春秋季较为短暂。日照充足，年日照时数在2859~3358小时之间。年平均降水量42.2mm。最低气温-33.6℃（1月），最高气温35.5℃（8月）。

== 经济 ==

格尔木是新兴的一座资源开發型的工业城市。近年来，第二产业在三个产业中的比例上升到60%以上。百万吨[[炼油|炼油]]、百万吨[[钾肥|钾肥]]和[[石油|石油]][[天然气|天然气]]三项工程项目的投资和兴建，形成了格尔木市的工业框架和基础。特别是2000年[[西部大开发|西部大开发]]战略实施以来，一批资源开发型重点工业项目相继在格尔木市建成投产，为推进格尔木市工业化进程奠定了坚实的基础。

格尔木市2004年[[生产总值|生产总值]]达到44亿元，比上年增长16.1%。2004年地方一般预算收入达到2.17亿元。
2004年城镇居民[[人均可支配收入|人均可支配收入]]达到8158.8元；农牧民人均纯收入达到3057.8元。

已探明[[石油|石油]]地质储量3亿多吨，远景储量可达10亿吨。[[涩北|涩北]][[天然气|天然气]]田已探明储量3000亿立方米，远景储量可达5000亿立方米，是中国四大天然气田之一。

格尔木出产[[黄金|黄金]]，并在1989年爆出一时轰动全中国的“格尔木黄金风波”。<ref name="格尔木黄金风波">[http://hongjian.fyfz.cn/b/362266 青海格尔木黄金风波———原青海省格尔木市副市长杨文山等受贿案]</ref><ref name="青海可可西里雪灾事件">
[http://dangshi.people.com.cn/GB/146570/198300/200214/200217/12453083.html 人民网：1990年3月9日，青海可可西里雪灾事件中有关人员严重违法违纪问题受到严处]</ref>

格尔木出产的[[昆仑玉|昆仑玉]]，作为[[2008年夏季奥林匹克运动会#.E7.8D.8E.E7.89.8C|2008年北京夏季奥林匹克运动会金镶玉奖牌]]的原材料<ref name="金镶玉">[http://search.cctv.com/playVideo.html?detailsid=10dfcf7aa68e48a5bfaee10ee9b3ef41&ad_maima=860010-1119020600&title=%E9%87%91%E7%8E%89%E8%89%AF%E7%BC%98 CCTV视频：专家带你看昆仑玉原石"] {{Wayback|url=http://search.cctv.com/playVideo.html?detailsid=10dfcf7aa68e48a5bfaee10ee9b3ef41&ad_maima=860010-1119020600&title=%E9%87%91%E7%8E%89%E8%89%AF%E7%BC%98 |date=20160304133206 }}</ref>后，获国玉美誉，身价爆增。<ref name="昆仑玉">[http://culture.people.com.cn/n/2013/0613/c22219-21824284.html 人民网：青海昆仑玉变身"国玉" 价格10年上涨万倍成"泡沫"]</ref>

== 交通 ==
===公路===
位于青藏高原腹地，格尔木是一座战略地位十分重要的城市。是连接[[西藏|西藏]]、[[新疆|新疆]]、[[甘肃|甘肃]]的战略要塞和中国西部的重要交通枢纽，[[青藏公路|青（海）（西）藏]]、青（海）新（疆）、敦（煌）格（尔木）三条公路干线在此交汇。

===铁路===

[[青藏铁路|青藏铁路]]一期[[西宁站|西宁站]]至[[格尔木站|格尔木站]]段年运输能力已达到1000万吨，年客流量达到200万人次。青藏铁路二期格尔木至[[拉萨站|拉萨站]]段已经完成施工，并于2006年7月1日全线通车。

[[敦格铁路|敦格铁路]]（[[敦煌|敦煌]]至格尔木）已于2012年10月开工建设；[[格库铁路|格库铁路]]（格尔木至[[库尔勒|库尔勒]]）已于2014年10月获[[国家发改委|国家发改委]]批复，计划于2014年底前开工建设。

===航空===
[[格尔木机场|格尔木机场]]已开通[[西宁|西宁]]、[[西安|西安]]、[[成都|成都]]等地的航班。

现在的格尔木，已初步形成包括公路、铁路、民航和[[管道运输|管道运输]]等多种形式的十字立体交通网络。

== 军事 ==

格尔木素有“兵城”和“汽车城”之称，[[人民解放军|人民解放军]]四总部、[[兰州军区|兰州军区]]、[[中国人民武装警察部队|武警部队]]和空军在这里的团级以上单位近30个，主要承担着西藏方面的公路运输、管道输油、通讯及通信线路维护三大任务。
