== 影视生涯 ==
她第一部电影是1991年的《Shout》，同年在[[斯皮尔伯格|斯皮尔伯格]]导演的《[[虎克船长|虎克船长]]》中扮演角色。 1995年她參與演出《[[七宗罪_(電影)|七宗罪]]》時，并与男主角[[布拉德·皮特|布拉德·皮特]]约会。直到两年后分手，两人的关系一直是媒体竞相报道的热点。1996年她在《[[爱玛|爱玛]]》中的演出获得了广泛好评。

1998年，她在《[[莎翁情史|莎翁情史]]》中的演技为她带来了[[奥斯卡最佳女主角奖|奥斯卡最佳女主角奖]]，在发表感言时异常激动泣不成声。

2008年起，開始加入《[[鋼鐵人|鋼鐵人]]》與《[[復仇者聯盟|復仇者聯盟]]》系列商業電影，演出維吉妮雅「小辣椒」角色，讓更多年輕觀眾熟知。

== 私生活 ==
派特洛与[[酷玩樂團|酷玩樂團]]（Coldplay）的主唱[[克里斯·馬丁|克里斯·馬丁]]（Chris Martin）在2003年12月5日秘密结婚，于2004年5月14日生下女兒，艾波·布莉絲·愛莉森·馬丁（Apple Blythe Alison Martin），06年4月8日生下兒子，摩西·布魯斯·安東尼·馬丁（Moses Bruce Anthony Martin）。在此之前她曾与[[布萊德·彼特|布萊德·彼特]]订婚，在拍摄《[[莎翁情史|莎翁情史]]》时也与[[班·艾佛列克|班·艾佛列克]]约会过。

2014年3月25日，派特洛發布公開聲明，宣布與[[克里斯·馬丁|克里斯·馬丁]]分手。「我們最近一年相處得不愉快，分開會更適合。」終結十年婚姻<ref>[http://www.nownews.com/n/2014/03/26/1165421 葛妮絲發聲明：和Coldplay主唱10年婚姻玩完]</ref>。在2015年4月21日以「無法化解的歧見」為由完成離婚手續，資產與監護權的分配受保密條約保護。2018年1月宣布和電視劇製作人Brad Falchuk結婚。
