'''格拉尼庫斯戰役'''（Battle of the Granicus），發生在（公元前334年）五月，是馬其頓[[亞歷山大大帝|亞歷山大大帝]]與[[阿契美尼德帝國|波斯帝國]]三大決定性戰役中的第一場戰鬥，戰場在小亞細亞西北方的格拉尼庫斯河，靠近[[特洛伊|特洛伊]]。在這裡，亞歷山大擊敗各波斯小亞細亞的總督，包括一支[[門農_(羅德島)|門農]]所率領的希臘僱傭軍。

== 背景 ==
亞歷山大繼承父親腓力二世的遺志，在鞏固希臘和馬其頓穩定後，開始進軍亞洲，此時為公元前334年。亞歷山大自己帶著由馬其頓和其他希臘城邦組成的約30000～40000步兵和4000～5000騎兵越過了希里帕（Ελλήσποντος，今日的[[達達尼爾海峽|達達尼爾海峽]]）。

而[[小亞細亞|小亞細亞]]各波斯總督也紛紛帶領自己的部隊集結，並在[[澤雷亞|澤雷亞]]城外紮營。在軍事會議上，希臘僱傭軍將領[[門農_(羅德島)|門農]]建議施行[[焦土|焦土]]政策，焚燒周邊的田地，迫使亞歷山大得不到補給而撤退（當時亞歷山大國庫透支，軍糧不到一個月）。但波斯[[赫勒斯滂弗里吉亞|赫勒斯滂弗里吉亞]]總督[[阿西提斯|阿西提斯]]說：「絕不允許自己領地上有一間房屋被燒」而拒絕提議，其他將領都支持阿西提斯，也對身為希臘人的門農不信任。並決定在格拉尼庫斯河邊決戰。

== 戰役 ==
[[File:Battle_granicus.gif|250px]]
格拉尼庫斯河寬約20~30米，河床東岸堤岸是相當陡峭的高坡，波斯軍隊及佈署在那，騎兵沿著河岸佈署，步兵在後方，包含希臘僱傭軍。馬其頓聯軍按慣例騎兵放置兩側，老將[[帕曼紐|帕曼紐]]率領左翼，亞歷山大在右翼。

戰役一開始，亞歷山大派遣右翼1000名騎兵和1000名重裝步兵朝波斯左翼進攻，其餘部隊依序呈斜型陣列前進。使波斯軍認為馬其頓聯軍意圖側翼突破，並調派中央部隊去增強左翼。亞歷山大看到波斯中央陣線因調動而鬆動，力量也明顯削弱，立即下令朝波斯中央發動總攻，亞歷山大親率[[夥伴騎兵|夥伴騎兵]]強渡格拉尼庫斯河與波斯軍廝殺，又因亞歷山大身披鎧甲亮眼，使波斯軍和將領都朝亞歷山大猛攻，險象環生，在[[克利圖斯|克利圖斯]]解救下亞歷山大才得以保命，終於擊退波斯騎兵。此時[[帕曼紐|帕曼紐]]率領左翼也與波斯右翼交戰，而潰退的波斯騎兵與後方的步兵糾纏一團，波斯軍大亂，許多將領又在與亞歷山大渡河對決時陣亡，波斯軍全線潰敗。

而波斯軍中的希臘僱傭軍，甚有紀律，井然有序的退到山地。但亞歷山大拒絕他們的求和，為恐嚇未來波斯軍中龐大的希臘雇傭軍，亞歷山大讓部隊朝希臘雇傭兵猛攻，殲滅大部分雇傭兵，其餘被押回馬其頓做奴隸。

== 戰後 ==
[[阿契美尼德帝國|波斯帝國]]在小亞細亞大部分的總督在此戰役陣亡，阿西提斯逃出後自殺，[[愛琴海|愛琴海]]東岸的希臘城邦都被亞歷山大自波斯手中解放，為日後東征建立先頭基地。
