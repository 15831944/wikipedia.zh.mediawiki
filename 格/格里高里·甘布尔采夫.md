'''格里高里·亚历山大罗维奇·甘布尔采夫'''（{{lang-ru|Григо́рий Алекса́ндрович Га́мбурцев}}，{{OldStyleDate|1903年|3月23日|3月10日}}－1955年6月28日）是一名[[苏联|苏联]]地震学家<ref name="RAN">{{cite web | url = http://www.scgis.ru/russian/cp1251/h_dgggms/4-99/gamburtsev.htm | script-title = ru:Институту физики Земли РАН присвоено имя академика Григория Александровича Гамбурцева: Вестник ОГГГГН РАН, № 4(10)'99 | accessdate = 2018-07-09 | date = 1999-12-25| publisher = Russian Academy of Science| language = ru| deadurl = yes| archiveurl = https://web.archive.org/web/20070821074404/http://www.scgis.ru/russian/cp1251/h_dgggms/4-99/gamburtsev.htm| archivedate = 2007-08-21| df = }}</ref>。

== 生平 ==
格里高里·亚历山大罗维奇·甘布尔采夫于1903年3月23日出生于[[俄罗斯帝国|俄罗斯帝国]][[圣彼得堡|圣彼得堡]]。1926年，甘布尔采夫从[[莫斯科国立大学|莫斯科国立大学]]毕业，时年23岁。1938年起，甘布尔采夫开始在今[[俄罗斯科学院|俄罗斯科学院]]下属的[[施密特地球物理研究所|施密特地球物理研究所]]工作，在1949年至1955年期间还出任该研究所所长。1946年，甘布尔采夫成为了[[苏联科学院|苏联科学院]]的准会员，后于1953年成为了正式会员。<ref>{{cite web| url = http://www.gubkin.ru/gallery/portrait/detail.php?ID=968| script-title = ru:ГАМБУРЦЕВ Григорий Александрович| accessdate = 2018-07-09| publisher = Gubkin Russian State University of Oil and Gas| language = ru| deadurl = no| archiveurl = https://www.webcitation.org/66nTvP2eM?url=http://www.gubkin.ru/wps/wcm/connect/gubkin/st-gubkin-site/sa-site-rgu/sa-nav-top-data/sa-university-about/ct-1cref-menu-portret-main-design| archivedate = 2012-04-09| df = }}</ref>。

甘布尔采夫发明了一种新型的[[地震仪|地震仪]]<ref name="GSE">{{Cite web |publisher=[[苏联大百科全书|苏联大百科全书]] |volume=3 |title=Гамбурцев |url=http://slovari.yandex.ru/art.xml?art=bse/00016/56600.htm |language=ru |accessdate=2018-07-09 }}{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。他还提出过两种地质勘探方法，用于矿物勘察的方法被称作“相关折射法”和用于观测[[地壳|地壳]]的“测深法”<ref name="GSE"/><ref>{{cite web| url = http://www.synapse.ru/index.php?id=35| title = Deep Seismic Sounding database: 1956 Caspian Sea DSS experiment| accessdate = 2018-07-09| publisher = Synapse Science Center| language = ru| deadurl = no| archiveurl = https://www.webcitation.org/66nTwYs7O?url=http://www.synapse.ru/index.php?id=35| archivedate = 2012-04-09| df = }}</ref>。1955年6月28日，甘布尔采夫于[[莫斯科|莫斯科]]去世，享年52岁。
