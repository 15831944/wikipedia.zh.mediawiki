'''格罗宁根'''（{{lang-nl|Groningen}}）是位于[[荷兰|荷兰]]北部的一座中等城市，也是[[格罗宁根省|格罗宁根省]]的首府，人口约185,000（2007年），是荷兰北部人口最多的城市。

著名景点有[[马提尼塔|马提尼塔]]（Martini Tower）。塔高97[[米|米]]，于1482年建成，是荷兰第四高的[[教堂|教堂]]尖塔。格罗宁根还是[[格罗宁根大学|格罗宁根大学]]和格罗宁根高等职业学校的所在地。全市大学生共有5万多，占人口很大比重。

== 歷史 ==
目前最早記載格羅寧根的文件約在1040年。但目前發現最古老的考古遺跡約在西元前3950至3720年。直到西元三世紀，格羅寧根才出現第一個主要聚落。

13世紀，格羅寧根成為一個重要的貿易中心，並建造[[城牆|城牆]]。該市對於周圍地區有極大的影響力，讓格羅寧根[[方言|方言]]成為當地的主要語言。15世紀末，當時鄰近的[[菲士蘭|菲士蘭省]]由格羅寧根管理，是格羅寧根最具影響力的時代。同一時期，格羅寧根修築了馬提尼塔，高約127公尺，成為當時歐洲最高的建築物。16世紀，[[西班牙帝國|西班牙帝國]]與[[荷蘭共和國|荷蘭共和國]]爆發[[八十年戰爭|八十年戰爭]]。1594年，格羅寧根選擇支持[[西班牙帝國|西班牙帝國]]，但之後改變立場，加入荷蘭共和國。

1614年，[[格羅寧根大學|格羅寧根大學]]成立，最初的目的是[[神學|宗教教育]]。同一時期，城市發展快速，並興建新城牆。1672年，[[第三次英荷戰爭|第三次英荷戰爭]]爆發，格羅寧根受到[[明斯特|明斯特]]主教{{link-en|Bernhard von Galen|Bernhard von Galen}}的猛烈攻擊，但城牆成功阻擋了攻勢。每年的8月28日，民眾會以音樂和[[煙火|煙火]]慶祝這次勝利。

格羅寧根沒有逃過二戰的蹂躪。1945年4月的[[格羅寧根戰役|格羅寧根戰役]]，城中的主廣場格羅特市場廣場（Grote Markt）遭到嚴重破壞。不過馬提尼塔、馬提尼教堂、{{link-en|Goudkantoor|Goudkantoor}}和市政廳都未受到破壞。這場戰爭持續了數天。

== 藝術與文化 ==
雖然格羅寧根不是個非常大的都市，但它仍是荷蘭該地區重要的都會，尤其在音樂和其他藝術、教育、商業等領域中有著重要地位。格羅寧根大量的學生族群也為城市建立了一個多元化的文化背景。

=== 博物館 ===
格羅寧根最重要且最知名的博物館是[[格羅寧格博物館|格羅寧格博物館]]（Groninger Museum）。由[[亞歷山卓·門蒂尼|亞歷山卓·門蒂尼]]（Alessandro Mendini）所設計的建築讓格羅寧格博物館成為荷蘭最現代且最創新的博物館之一。另外，格羅寧根還有一座[[科學博物館|科學博物館]]、一座[[漫畫|漫畫]]博物館和一座[[菸草|菸草]]博物館。

=== 夜生活 ===
格羅寧根的夜生活大部分是依靠其學生族群。格羅特市場等廣場在夜晚十分擁擠，尤其是星期四與星期五，部分酒吧營業至早上7點。2006年，格羅寧根被票選為荷蘭「最佳市中心」（de beste binnenstad）。

== 格羅寧根大學 ==
{{main|格羅寧根大學}}
1614年成立的[[格羅寧根大學|格羅寧根大學]]有著豐富的學術傳統，是荷蘭第二悠久的大學（僅次於[[萊頓大學|萊頓大學]]）。荷蘭第一位女性學生、荷蘭第一位太空人，[[歐洲中央銀行|歐洲中央銀行]]總裁都來自格羅寧根大學，此外还有多位[[諾貝爾獎|諾貝爾獎]]得主。

== 政治 ==
市議會有39個席次。左派政黨[[工黨_(荷蘭)|工黨]]和[[綠色左派|綠色左派]]是最大黨。2002年選舉結束後，他們與[[基督教民主黨_(荷蘭)|基督教民主黨]]、[[自由民主人民黨|自由民主人民黨]]組成聯盟。2006年選舉結束後，三個左派政黨（工黨、綠色左派和[[社會黨_(荷蘭)|社會黨]]）在4月26日組成新政治聯盟。

== 工業 ==
目前CSM（Centrale Suiker Maatschappij）在格羅寧根有間[[食糖|製糖]][[工廠|工廠]]，每年的產量約300,000[[公噸|公噸]]。

== 運輸 ==
格羅寧根被稱為「世界單車之都」，因為市內57%的旅遊是以單車為工具。<ref>{{cite web|url=http://www.globalideasbank.org/site/bank/idea.php?ideaId=378|title=global ideas bank |publisher=globalideasbank.org |date=2007-10-15 |accessdate=2009-07-14}}</ref>格羅寧根有著廣泛的單車網路、良好的公共運輸和市中心大範圍的步行區，十分適合想遠離汽車的人士。在一個義大利節目的調查中<ref>{{cite web |url=http://www.rai.tv/mpplaymedia/0,,RaiTre-Report%5E23%5E37292,00.html |title=Rai.tv |publisher=Rai.tv |date=2007-10-15 |accessdate=2009-05-05 |deadurl=yes |archiveurl=https://web.archive.org/web/20090206235202/http://www.rai.tv/mpplaymedia/0%2C%2CRaiTre-Report%5E23%5E37292%2C00.html |archivedate=2009-02-06 }}</ref>，認為格羅寧根對於單車的使用是義大利值得的仿效對象。

=== 鐵路 ===
格羅寧根設有三個車站：
* [[格羅寧根火車站|格羅寧根火車站]]（Groningen railway station）
* [[格羅寧根歐羅巴公園火車站|格羅寧根歐羅巴公園火車站]]（Groningen Europapark railway station）
* [[格羅寧根北火車站|格羅寧根北火車站]]（Groningen Noord railway station）

主車站（由[[荷蘭鐵路|荷蘭鐵路]]和[[爱瑞发|爱瑞发]]營運）可通往荷蘭大多數的主要城市。

直達地有：
* [[茲沃勒|茲沃勒]]、[[阿默斯福特|阿默斯福特]]、[[希爾弗瑟姆|希爾弗瑟姆]]（Hilversum）、[[阿姆斯特丹|阿姆斯特丹]]南方與[[史基浦機場|史基浦機場]]：荷蘭鐵路
* [[茲沃勒|茲沃勒]]、[[阿默斯福特|阿默斯福特]]、[[烏特勒支|烏特勒支]]、[[豪達|豪達]]（Gouda）與[[海牙|海牙]]：荷蘭鐵路
* [[呂伐登|呂伐登]]：爱瑞发
* [[霍赫贊德-薩珀梅爾|Hoogezand-Sappemeer]]、[[溫斯霍滕|Winschoten]]、德國[[萊爾|萊爾]]（Leer）：爱瑞发
* {{link-en|Roodeschool|Roodeschool}}和[[代爾夫宰爾|代爾夫宰爾]]：爱瑞发
* 爱瑞发計畫在2009年至2010年開始營運往[[芬丹|Veendam]]的路線。

== 足球場館 ==
[[格羅寧根足球會|格羅寧根足球會]]的主場為[[歐羅堡球場|歐羅堡球場]]（Euroborg）。歐羅堡球場在2006年1月正式啟用，可容納22,600觀眾。目前計劃要將容納人數增加到35,000至40,000人。格羅寧根足球會前主場為[[奧斯特帕克球場|奧斯特帕克球場]]（Oosterpark Stadion），可容納12,500人。
