'''格陵蘭'''（{{lang-kl|Kalaallit Nunaat}}；{{lang-da|Grønland}}），面積2,166,086[[平方公里|平方公里]]，是世界第一大島，大约81%都被冰雪覆盖。“格陵蘭”在[[丹麥语|丹麥语]]中的字面意思為「綠色土地」，是在[[丹麥王國|丹麦王国]]框架内的[[附庸國|自治国]]，在[[格陵蘭自治公投|2008年公投]]後，2009年正式改制，成為一個內政獨立的自治區，但外交、國防與財政相關事務仍由丹麥代理。格陵兰随[[宗主国|宗主国]][[丹麦|丹麦]]于1973年加入过欧洲各共同体，但根据[[1982年格陵兰公投|1982年全民公投]]的结果通过1985年的《[[格陵兰条约|格陵兰条约]]》退出了如今欧盟前身——[[欧洲各共同体|欧洲各共同体]]，故如今格陵蘭並不屬於[[歐盟|歐盟]]，但被视为[[欧洲联盟特别领域|欧洲联盟特别领域]]的一部分。格陵蘭如同[[法羅群島|法羅群島]]，在[[丹麥國會|丹麥國會]]派驻有2名議員。

格陵蘭全境大部分处在[[北极圈|北极圈]]内，气候寒冷。隔海峽與[[冰島|冰島]]和[[加拿大|加拿大]]兩國相望。

== 历史 ==
{{main|格陵兰历史}}
格陵兰的原住民為[[因纽特人|因纽特人]]，其究竟何时来到这裡、以及為何要选擇这個荒凉的島嶼居住，皆未有定論。在[[冰岛|冰岛]]的北歐殖民者于982年到来，冰岛人在格陵兰岛的最南端建立了三个据点，在接下来的几个世纪中，他们一直坚持下来。

格陵蘭这个名字正是这些[[斯堪的纳维亚|斯堪的纳维亚]]人所取。根据[[北欧|北欧]]神话史诗[[萨迦_(文学)|萨迦]]记载，[[红胡子埃里克|红胡子埃里克]]因为犯[[谋杀|谋杀]]罪而从冰岛流亡至此。埃里克一家及其[[奴隶|奴隶]]向西北航行，以探寻传说存在在那裡的陆地。当他在岛上定居下来后，便给该岛取名格陵兰（{{lang|da|Grønland}}，意即“绿色的土地”），以吸引更多的移民。（至少该岛南端的[[峡湾|峡湾]]还是多草的。）他的这一妙计果然成功，最終有約4000名北欧移民移入。在早期的[[地图|地图]]上，格陵蘭也曾被標上{{lang|da|Gruntland}}的名字（意为大地之地），到底{{lang|da|Grunt}}是{{lang|da|Grøn}}的讹音，还是反过来，至今无人知道。

12世纪时，在格陵兰甚至还建立一个[[天主教|天主教]]的[[主教|主教]]辖区。1386年，格陵兰正式被[[挪威|挪威]]吞并，成為北欧三国[[卡尔马联盟|卡尔马联盟]]的領土。在该联盟瓦解后，格陵兰又转属于[[丹麦-挪威|丹麦-挪威]]双重君主国的殖民统治。

坚持了五百多年以后，北欧人的定居点在15世纪突然消失，这很可能是因为[[小冰期|小冰期]]引起普遍的食物匮乏。后来发现的该时期居民的遗骨，都带有[[营养不良|营养不良]]的特征。一封1424年由格陵蘭寄至冰島的信件，是北歐人定居在格陵蘭最後的證據，之後就再沒有任何北歐人定居格陵蘭的史料。另一方面，因纽特人土著居民並沒有受到太大影響。

1814年，根据[[基尔条约|基尔条约]]，[[挪威|挪威]]被丹麥割讓給[[瑞典|瑞典]]，而丹麦重新获得格陵兰的控制权。

1931年，挪威占领当时无人居住的格陵兰岛东部地区，宣称该地是[[无主土地|无主土地]]，声称对其拥有主权。1933年，丹麦和挪威两国同意将此争端提交[[国联|国联]]下属的[[常設國際法院|常設國際法院]]訟裁，根据仲裁结果，丹麦获得格陵兰岛的全部权利。

在[[第二次世界大战|第二次世界大战]]期间，由于1940年4月9日[[纳粹德國|纳粹德國]]占领丹麦，格陵兰和丹麦的联系被切断。格陵兰通过开采[[伊維圖特|伊維圖特]]（{{lang|kl|Ivigtuut}}）的[[冰晶石|冰晶石]]矿，设法同[[美国|美国]]和[[加拿大|加拿大]]进行贸易，获得必需的商品。这种情况一直持续到战争结束。

二战期间在这一地区也发生小规模的争夺战斗。1943年，一名丹麦士兵在该地区阵亡，一年以后，一名德国士兵也战死该地。{{来源请求}}

== 地理 ==
{{main|格陵兰地理}}
[[File:Topographic_map_of_Greenland_bedrock.jpg|left]]

格陵蘭是世界最大的島嶼，属于[[大陆岛|大陆岛]]，位於[[北美洲|北美洲]]東北，[[北冰洋|北冰洋]]和[[大西洋|大西洋]]之間，西面隔[[巴芬灣|巴芬灣]]和[[戴維斯海峽|戴維斯海峽]]與[[加拿大|加拿大]]的[[北極島嶼|北極島嶼]]相望，東邊隔[[丹麥海峽|丹麥海峽]]和[[冰島|冰島]]對望。由於面積龐大，格陵蘭常被稱為格陵蘭[[次大陸|次大陸]]。全島約五分之四地區處於[[北極圈|北極圈]]之內，為[[極地氣候|極地氣候]]。

格陵蘭是除[[南極洲|南極洲]]以外，[[大陆冰川|大陆冰川]]面积最大的地区。幾乎全地區都被[[冰蓋|冰蓋]]覆蓋，只有島的極北部和東西兩邊的狹長地帶例外，由於該等地區的空氣異常乾燥，難以形成積雪，地表才得以暴露。亦因為中部地區長期受冰雪蓋壓，故中部地區，若去除雪帽覆蓋後，會比島緣地區低。全島以中部偏東最高[[海拔|海拔]]3300米，邊緣地區約平約海拔1000至2000米。若然將格陵蘭島全部冰雪溶掉，在[[冰河|冰河]][[侵蝕作用|侵蝕作用]]的影響下將會呈現[[群島|群島]]的狀態。同时海平面会上升7米。
[[File:Geography-of-greenland.svg|thumb]]

在1989年至1993年期間，歐洲和美國的[[氣候|氣候]]研究人員到達格陵蘭島冰蓋的高峰，挖掘了一對2英里（約3.2公里）長的[[冰核|冰核]]，去研究其化學成份。冰核內藏有的空氣是不能去除冰蓋的，愈深層的冰核空氣代表該空氣成份是在一個較早的時期的空氣，研究其內容有利於發現遠至100 000年前的[[遠古|遠古]]時期[[北半球|北半球]]的氣候狀況。而研究結果是地球的氣候，通常是在一種較長期穩定的氣候，經過突然急速的不穩定變化，轉為另一種長期穩定的氣候。
北[[大西洋|大西洋]]的气候受格陵兰东北部沿海的[[洋流|洋流]]情况影响很深。

== 政治 ==
格陵兰名义上的国家元首是[[丹麦君主|丹麦君主]]，现任为[[玛格丽特二世|玛格丽特二世]]，丹麦政府任命一位高级专员（'''Rigsombudsmand'''／High Commissioner）代表丹麦国王和政府。

[[格陵兰议会|格陵兰议会]]经普选产生，任期4年，设31个议席。自治政府首脑[[格陵兰总理|格陵兰總理]]通常就是议会多数党首领。

雖然其宗主国丹麥是[[歐洲聯盟|歐洲聯盟]]成員國，但格陵蘭不屬於[[歐洲聯盟|歐洲聯盟]]的一部分。

格陵蘭於2008年11月25日舉行[[格陵蘭自治公投|自治公投]]，公投在全島80個市鎮和村落舉行，當地午夜左右，公布開票結果，投票率71.96%，75.54%贊成自治，23.57%反對。公投案以超過75%選民支持大比數獲得通過，公投通過後，格陵蘭自治地位從2009年6月21日起生效，成為[[國際法|國際法]]下的獨立政治实体，自行管理自身的內政、司法與資源分配運用，但其國防、外交與財政相關事務仍由丹麥管轄。
格陵兰东部不设城市，而划为世界上最大的[[自然保护区|自然保护区]]。

=== 行政区划 ===
{{main|格陵兰行政区划}}
格陵兰全国从2009年1月1日起划分为四个自治区：[[库雅雷克|库雅雷哥]]（Kujalleq）、卡苏伊忒索普（Qaasuitsup）、[[凯克卡塔|凯库卡塔]]（Qeqqata）及瑟摩苏哥（Sermersooq）。

== 经济 ==
[[File:Social_housing_in_Ilulissat,_Greenland.jpg|thumb]]的公共住房]]
格陵兰在上个世纪九十年代经受了经济紧缩，但1993年以来，经济开始复苏。格陵兰自治政府自1980年来一直采取紧缩财政政策并有效保证了盈余的公共预算和较低的[[通货膨胀|通货膨胀]]。自1990年岛内最后的[[铅|铅]]矿和[[锌|锌]]矿的关闭以来，格陵兰的外贸一直处于赤字状态。尽管格陵兰已经重新开始对烃和矿物的勘探活动，要正式投產仍需要数年时间，估計格陵蘭西北部和東北部的油藏量相當驚人，有將近500億桶的[[石油|石油]]與[[天然氣|天然氣]]。另外，在南部納赫薩克市附近的高原，發現了可能世界蘊藏量最大[[稀土|稀土]]金屬礦床。

现在该岛的经济严重依赖于[[渔业|渔业]]和[[渔产品|渔产品]]出口，佔出口值的80％以上，捕虾业是最大的支柱产业。亦有綿羊的牧業發展，但飼料大多由國外進口。

旅游业是该国唯一有短期收益潜力的部门，但这也受到旅游季节短和消费高的限制。包括国有企业和市政府的公共部门在该国经济中扮演主导角色。格陵兰政府约一半的收入来自于丹麦的补助，这也是格陵兰国民生产总值的重要补充。

== 人口==
格陵蘭的人口密度不仅非常低，而且人口呈下降趨勢。這是因為當地昼夜時間非常極端，影響當地人作息，導致當地人患上心理疾病的風險比世上仼何一個地方都要高，[[自殺|自殺]]率也因此極高所致。

== 交通 ==
[[File:Air_Greenland.jpg|thumb]]

格陵兰之间以及它与外界的联系主要靠水运和[[格陵蘭航空|格陵蘭航空]]维系，與丹麦、加拿大和冰岛有定期的航班与[[客轮|客轮]][[货轮|货轮]]。

因为海湾太多的缘故，各地之间没有[[公路|公路]]相连，只有在小块的沿海无冰区才有一些道路，这些地区交通靠[[雪橇|雪橇]]。

2000年该地共有3729辆[[汽车|汽车]]，其中私人小轿车2093辆。

== 文化 ==
[[File:Upernavik_first_day_in_class_2007-08-14_2.jpg|left]]的小学生在开学第一天穿上传统的服饰上学]]
格陵兰文化以[[因纽特人|因纽特人]]的文化为主，并受到[[维京|维京]]探险文化的影响。部分因纽特人仍然以捕魚为生。

還有年度舉辦的[[狗拖雪橇比賽|狗拖雪橇比賽]]，只要有隊伍就可以參加。

格陵兰开始吸引游人参观，在这里可以进行狗拉雪橇比赛、打渔、健行和跨島滑雪。

在第40届[[世界圣诞老人大会|世界圣诞老人大会]]上，'''格陵兰'''被确认为[[圣诞老人|圣诞老公公]]真正的[[故乡|故乡]]<ref>[http://information.byecity.com/info/7125.shtml 格陵兰击败拉普兰成为圣诞老人故乡] {{webarchive|url=https://web.archive.org/web/20081213190134/http://information.byecity.com/info/7125.shtml |date=2008-12-13 }}</ref>。

[[格陵兰大学|格陵兰大学]]建于1987年。

== 体育 ==
[[File:Uummannaq-football-game.jpg|thumb]]
[[足球|足球]]是格陵兰的国家体育项目，但格陵兰并不是[[国际足球联合会|国际足球联合会]]（FIFA）的成员。[[国际足球联合会|FIFA]]要求会员国必须有满足最低标准的天然草皮球场供国际比赛使用，而格陵兰的气候条件使得它不可能拥有符合标准的天然草皮球场。

== 动植物 ==
格陵兰的植物以[[苔藓植物|苔原植物]]为主，包括[[大麻|大麻]]、[[苔草|苔草]]、[[羊胡子草|羊胡子草]]和[[地衣|地衣]]。一些无冰地区除了一些矮小的[[桦树|桦树]]、[[柳树|柳树]]和[[桤树|桤树]]丛外，几乎无别的树木生存。

格陵兰在漫长的冬季没有太阳（[[极夜|极夜]]现象）。但在夏季由于[[极昼|极昼]]24小时的日照，格陵兰因此迎来了大量来此繁殖的鸟类，许多植物也生长旺盛。尽管许多鸟类在冬季来临时又飞向南方，但也有些鸟全年都驻足于此，其中包括[[雷鸟|雷鸟]]和[[小雪巫鸟|小雪巫鸟]]。

格陵兰岛是[[北极熊|北极熊]]的家园。其他动物还有[[狼|狼]]、[[北极狐|北极狐]]、[[北极兔|北极兔]]、[[驯鹿|驯鹿]]和[[旅鼠|旅鼠]]等。大批[[麝牛|麝牛]]分布于岛北部，其极厚的外皮能保护它们免受冰冷的北极风冻害。

在沿岸水域常见[[鲸|鲸]]和[[海豹|海豹]]。咸水鱼有[[鳕|鳕]]、[[鲑|鲑]]、[[比目鱼|比目鱼]]和大比目鱼。
