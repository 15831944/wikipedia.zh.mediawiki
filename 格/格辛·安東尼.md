'''格辛·安東尼'''（'''Gethin David L. Anthony'''，{{bd|1983年|10月9日}}）是一位[[英國|英國]]電視劇及電影演員。

== 教育程度 ==
他曾就讀於牛津大學<ref>{{cite web|title=Gethin Anthony|url=http://unitedagents.co.uk/gethin-anthony|publisher=United Agents|accessdate=30 April 2012}}</ref>。

== 生涯 ==
2012年11月至2013年3月，他於[[皇家莎士比亞劇團|皇家莎士比亞劇團]]的《[[鮑里斯·戈東諾夫|鮑里斯·戈東諾夫]]》舞台劇中飾演Grigory。<ref>{{Cite web |url = http://www.independent.co.uk/arts-entertainment/theatre-dance/reviews/boris-godunov-swan-theatre-stratford-upon-avon-8368204.html |title = Boris Godunov, Swan Theatre, Stratford-upon-Avon |author = Paul Taylor |date = 2012-11-29 |publisher = The Independent |language = en}}</ref>

他在[[HBO|HBO]]影集《[[权力的游戏_(电视剧)|權力遊戲]]》中飾演{{link-en|Renly Baratheon|Renly Baratheon}}，以及在[[全國廣播公司|NBC]]電視劇《[[水瓶宮_(美國電視劇)|水瓶宮]]》中飾演[[查爾斯·曼森|查爾斯·曼森]]。<ref>{{Cite web |url = http://www.vanityfair.com/hollywood/2015/05/gethin-anthony-aquarius |title = Gethin Anthony on Going from ''Game of Thrones'' to the Mind of Charles Manson |author = Krista Smith |date = 2015-05-28 |publisher = Vanity Fair |language = en}}</ref>
