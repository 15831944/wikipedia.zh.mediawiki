'''萬榮車站'''為[[臺灣鐵路管理局|臺灣鐵路管理局]][[臺東線|臺東線]]的[[鐵路車站|鐵路車站]]。該站雖有[[萬榮|萬榮]]之名，但其位置並不在[[萬榮鄉|萬榮鄉]]，而是位於鄰近的[[鳳林鎮_(臺灣)|鳳林鎮]]。昔日此站有[[行政院農業委員會林務局|林務局]]花蓮林區管理處[[萬森線|萬森線]]通至[[林田山|林田山]]（森榮）。本站站房已配合「花東新車站運動」計畫改建完工。

== 車站構造 ==
*第一月台為[[岸式月台|岸式月台]]，第二月台為[[島式月台|島式月台]]。

== 利用狀況 ==
*本站為[[萬榮鄉|萬榮鄉]]主要門戶，目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]。
*停靠狀態：
**[[莒光號列車|莒光號]]：[[觀光列車|觀光列車]]、團體列車以及週五南下616次及週日北上6615次未停靠，其餘皆停靠。
**[[臺鐵區間車|區間車]]：皆停靠。
*對號列車停靠率：14.0%；所有車種停靠率：30.6%。
*站內設有「鐵路之旅 - 小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*本站開放使用一卡通、悠遊卡及icash 2.0付費。

== 歷史 ==
萬榮原名「馬里勿」，在[[阿美族語|阿美族語]]中為「上坡」之意，[[台灣日治時期|日治時期]]日本政府為運送[[林田山林場|林田山林場]]之木材於[[大正|大正]]3年（1914年）3月8日設立馬里勿乘降場，為本站創建之緣起。大正7年改為萬里橋驛，戰後因行政區域重新劃分，萬里鄉成立，本站因處萬里鄉治所在地，遂於1952年1月15日更名為萬里站。後又因「萬里」與台北縣萬里鄉（現[[新北市|新北市]][[萬里區|萬里區]]）同名，遂於1958年6月1日改為萬榮鄉，本站亦隨之改稱萬榮站延用迄今。1982年東線鐵路拓寬完成，本站日治時期原有之木造房舍因過於老舊，亦同時改建為[[鋼筋混凝土|鋼筋水泥]]站房使用迄今。

*1914年3月8日：設'''馬-{里}-勿乘降場'''（招呼站）。
*1918年：改名為'''萬-{里}-橋驛'''。
*1951年1月15日：因鳳林鎮西側新設「萬里鄉」，故配合地名，改名為'''萬里車站'''。
*1958年6月1日：因鄉名與[[新北市|臺北縣]][[萬里區|萬里鄉]]同名，再度配合鄉名的更改，改名為'''萬榮車站'''。
*2016年6月28日：啟用多卡通刷卡機。

===繁榮的過去===
本站在1960至1970年代林田山[[伐木|伐木]]運轉時最為興盛，每日從林田山小火車轉乘的旅客絡繹不絕，木材貨運也極為龐大。在當時與[[鳳林車站|鳳林]]、[[光復車站|光復]]、[[瑞穗車站|瑞穗]]、[[玉里車站|玉里]]、[[富里車站|富里]]、[[池上車站_(臺灣)|池上]]、[[關山車站_(臺灣)|關山]]、[[鹿野車站|鹿野]]等站同樣，皆為台東線各級列車都會停靠的主要車站。根據《台灣車站革沿》一書提到，當時萬榮站1970年代每日上車人數達700人次（一年23萬），每日客運收入[[新台幣|新台幣]]6,000元，貨運每日150公噸，主要以木材為大宗。

不過隨著林田山在1980年代沒落，萬榮站的客貨運營收快速沒落。根據台鐵官方紀錄，1980年上車人數為4萬人，遠低於伐木業盛行的年代，但當時仍停靠台東線最高級的[[光華號|光華號]]列車。東拓完成改點後，台東線上開始有自強、莒光、復興等高級列車運行，但萬榮站僅有非對號列車停靠。再加上[[花東公路|花東公路]]拓寬與公路運輸的普及，使得萬榮站的營運狀況大幅萎縮。1990年代糧食局的側線拆除，萬榮站貨運幾乎處於停擺的狀況，除了[[道渣|道渣]]之外其他貨運物品幾乎都難得一見。客運方面雖然之後有停靠莒光號等對號列車，但還是不敵林田山人口外流之缺，2004年度時上車人數僅7,718人，收入197萬元左右。但由於電腦連線售票及花蓮客運漲價，車站業績自2004年起每年都成長2成，2005年度營收285萬元，上車人數成長至15,011人。

==萬榮車站的名產==
萬榮車站除了販售知名的飯包便當之外，另一在地特產｢[[剝皮辣椒|剝皮辣椒]]｣也是萬榮車站最佳伴手禮之一，過去只要曾在萬榮陸軍基地服役過的軍兵們每到放假、退伍歸鄉時必買，且一次必是一打半打的買回家。號稱軍中伙食'''｢餐餐必有，夏熱食用可解暑，寒冬食用可禦寒｣'''，雖然現在名聲已被[[鳳林車站|鳳林車站]]的｢[[剝皮辣椒|剝皮辣椒]]｣掩蓋過去，但這是花蓮的著名的伴手禮之一。
