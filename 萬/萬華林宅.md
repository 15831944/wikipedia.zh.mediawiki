'''萬華林宅'''，是一棟位在臺北[[萬華|萬華]]的私人[[洋樓|洋樓]]式[[宅第|宅第]]。該宅第興建於[[1931年臺灣|1931年]]，並曾是當地最高的建築物之一。<ref name=":0">{{Cite web|url=http://www.wanhualinhouse.com/place|title=萬華林宅|accessdate=2017-10-20|work=萬華林宅}}</ref>

== 歷史 ==
[[1932年臺灣|1932年]]，林紅麻、林惡水、林遍金、林金水合資興建宅第落成；該宅第乃依據當時[[市區改正|都市計畫]]道路界線建築而成，總樓層共四樓，占地約201[[坪|坪]]。<ref>{{Cite web|url=http://www.wanhualinhouse.com|title=萬華林宅|accessdate=2017-10-20|work=萬華林宅}}</ref>

[[2000年臺灣|2000年]]，該宅第被[[臺北市政府|臺北市政府]]指定為其[[市定古蹟|市定古蹟]]。

[[2009年臺灣|2009年]]，該宅第頂樓雨遮落下並造成[[車輛|車輛]]毀損，其[[所有權人|所有權人]]因而開始商議維修計畫，並由該家族第四代[[兒媳|媳婦]]毛瑩脗協調眾人意見<ref>{{Cite news|url=http://news.ltn.com.tw/news/local/paper/917880|title=〈台北都會〉艋舺昔日最高樓》 萬華林宅修繕 風華再現|author=|date=|work=自由時報電子報|accessdate=2017-10-20}}</ref>，復由第二代[[長子|長子]]林淇清招集親友向[[臺北市政府文化局|臺北市政府文化局]]申請修復。

在臺北市政府文化局補助後，該宅第於2013年10月修繕完工。<ref>{{Cite news|url=https://news.housefun.com.tw/news/article/21486746431.html|title=萬華林宅　重現1930年豪宅風貌|author=廖珮妤|last=|date=|work=|newspaper=好房網News|accessdate=2017-10-20|language=zh-TW}}</ref>

[[2016年臺灣|2016年]][[4月27日|4月27日]]，[[星巴克|星巴克]]艋舺[[門市|門市]]開幕於萬華林宅，並設有十數位「社區大使」為訪客進行導覽解說。<ref>{{Cite news|url=http://news.ltn.com.tw/news/life/breakingnews/1684449|title=星巴克的老師  「萬華社區小學」活化艋舺古蹟|author=|date=|work=自由時報電子報|accessdate=2017-10-20}}</ref>

== 地理 ==
萬華林宅的方位為座東朝西。早期，該宅第高處可向東遠眺[[總統府_(臺灣)|總統府]]、向南遠眺[[臺北車站|臺北車站]]、向西遠眺[[淡水河|淡水河]]及[[關渡|關渡]]地區，甚至可在視線清晰時望及[[公館_(臺北市)|公館]]地區。<ref name=":0" />

== 建築 ==

=== 外觀 ===
萬華林宅的外牆主要以[[清水磚|清水磚]]砌成，[[屋頂|屋頂]]主要以[[黑瓦|黑瓦]]覆蓋，每層樓[[陽台|陽台]]的[[欄杆|欄杆]]及門窗以[[水泥|水泥]]造成並採欄杆透空工法；二樓欄杆透空圖形為[[長方形|長方形]]且兩側為[[海浪|波浪]]狀，三樓欄杆透空圖形為[[菱形|菱形]]，四樓欄杆透空圖形復為長方型。<ref name=":0" />

=== 結構 ===
該宅第採用[[鋼筋混凝土|鋼筋混凝土]]構造建築而成，共有四層樓。其[[地基|地基]]採用{{Link-en|筏式基礎|Floating raft system}}形式建成；該建物的[[結構支承|結構]]因此在多次[[地震|地震]]後仍保持良好狀況。<ref name=":0" />

== 特色 ==
為配合[[台灣日治時期|日治時期]]臺北市[[市區改正|市區改正]]計畫，該宅第所在[[地皮|土地]]及其平面呈不規則形狀，且三面外牆皆臨[[道路|道路]]；其中，臨南側外牆呈ㄇ字形，並因此形成半個[[天井|天井]]，且具[[通風|通風]]效果。<ref name=":0" />
