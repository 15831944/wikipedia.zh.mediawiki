'''萬世線'''（{{jpn|j=万世線|hg=ばんせいせん|rm=Bansei sen|f=y}}）是一條由[[鹿兒島縣|鹿兒島縣]][[加世田市|加世田市]]（現在為[[南薩摩市|南薩摩市]]）[[加世田站|加世田站]]至[[薩摩萬世站|薩摩萬世站]]的鐵路線。由[[南薩鐵道|南薩鐵道]]（鹿兒島交通前身）營運。在1962年停運。

== 路線概要 ==
*管轄：南薩鐵道
*路線（營運公里）：加世田 - 薩摩萬世 2.5km
*軌距：1067mm
*站數：2個（包括起終點站，在停運前一刻計算）
*複線區間：沒有（全線單線）
*電化區間：沒有（全線非電氣化）

== 運行情況 ==
在1960年，一日共有17班列車來回，平均早上約40分鐘一班，日間與晚上則為1小時一班。所有班次都是普通班次。此外，還有數班貨運列車使用本線。

== 歷史 ==
*1916年（大正5年）10月22日：南薩鐵道加世田至薩摩大崎站開通。
*1941年（昭和16年）2月19日：薩摩大崎站改名為薩摩萬世站。
*1962年（昭和39年）9月1日：全線停運。
