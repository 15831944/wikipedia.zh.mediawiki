'''萬芳'''<ref>{{cite web|url=http://www.biosmonthly.com/interview_topic/9486|title=每個人心中都有一首萬芳，一個忘不了的地方|author=BIOS monthly|}}</ref>（{{bd|1967年|7月6日|catIdx=L林}}）是[[臺灣|臺灣]][[女歌手|女歌手]]、[[主持人|主持人]]，本名'''林萬芳'''，出生於[[桃園市|桃園縣]][[八德區|八德鄉]]（今[[桃園市|桃園市]][[八德區|八德區]]）。以高亢、溫和柔美的歌聲在[[華語流行音樂|華語流行音樂]]佔有一席之地。在整個90年代台灣歌壇連續賣出暢銷專輯。曾主唱過許多[[台灣電視劇|台灣電視劇]]主題曲。她主唱[[香港電影|香港電影]]《[[新不了情_(1993年電影)|新不了情]]》的同名主題曲《新不了情》，至今在台灣
[[KTV|KTV]]仍廣為傳唱。

== 經歷 ==
萬芳於[[桃園市|桃園縣]][[八德區|八德市]]大成國小與[[武陵高中|武陵高中]]時期參加過[[合唱團|合唱團]]，國中參加過[[國樂|國樂]]班，音樂素養自小培養。就讀[[文化大學|文化大學]]期間，與同學蘇仁林搭檔吉他伴奏，報名參加[[木船民歌比賽|木船民歌比賽]]，演唱「鎖上記憶」奪得優勝，從此展開歌唱生涯，之後加盟[[滾石唱片|滾石唱片]]，并於1990年發行首張專輯《時間仍然繼續在走》。

1992年，萬芳在年初與年底分別發行《放心》與《真情》兩張專輯，其中《真情》專輯的兩首主打歌《我記得你眼裡的依戀》與《猜心》都是中視八點檔再世情緣主題曲。配合電視劇熱播，專輯光在台灣就超過50萬張，亞洲更是賣到150萬張左右，萬芳也在此專輯開始邁入暢銷歌手行列。

1993年，發行專輯《貼心》，配合主打《試著了解》，在台灣賣出30萬張，亞洲也破百萬張。1994年的專輯《斷線》，創造出至今還是KTV的熱門歌曲《新不了情》，台灣賣出超過45萬張，亞洲更是超過150萬張，打破自己的紀錄。

1996年的專輯《割愛》，讓萬芳再一次登上事業巔峰，台灣銷售超過40萬張亞洲同意賣出超過百萬張。趁勝追擊，同年底發行專輯《就值得了愛》，專輯台灣銷售認證雙白金，超過30萬張。

1999年發行專輯《不換》，儘管在接近千禧年代唱片開始泡沫化，不換依舊賣出25萬張，與同期天后級歌手許茹芸爭奪銷售量。《不換》也是萬芳在淡出歌壇前，後期比較暢銷的專輯。

千禧年後，新生代歌手的取代，萬芳逐漸降低發片量，轉往廣播界，但偶爾過幾年還是會發行專輯回饋歌迷。

萬芳除了歌唱事業之外，舞台劇也有傑出的表現。曾為[[中廣音樂網|中廣音樂網]]的[[主持人|主持人]]（[[DJ|DJ]]）之一。近年來多從事戲劇方面的演出，也晉升為舞台劇導演。

萬芳在2010年10月30、31日於臺北國際會議中心舉办出道20年来个人第一场大型售票演唱会《你所不知道的那些夜晚》。好友[[伍佰|伍佰]]担任台北第二场特别嘉宾（第一場嘉賓為[[亂彈阿翔|亂彈阿翔]]）。<ref>{{cite web|title=万芳20周年演唱会开唱在即 好友伍佰自荐当嘉宾|url=http://ent.qq.com/a/20101018/000096.htm|publisher=qq.com|accessdate=1 November 2010}}</ref>

2015年3月7日，萬芳再次於臺北國際會議中心舉辦《原來的地方》演唱會，這已是她第三次在此地舉辦個人演唱會。

2018年4月14日，萬芳首次登上[[台北小巨蛋|台北小巨蛋]]舉辦《時間仍然繼續在走》演唱會。

== 軼事 ==
[[梁文音|梁文音]]在發表「同性戀朋友需要被神恢復」言論後，引發同志族群的不滿；隔年，萬芳在自己主持的[[中廣|中廣]]節目《i-散步》中播放梁文音的新歌《愛其實很殘忍》，接著說：「愛有時候真的很殘忍，尤其是以愛之名，行恨之實的時候。有時候覺得很殘忍而且很可惡…最後就來聽這首表達支持同志婚姻的作品《[[平等之愛|Same Love]]》。」被指是藉歌暗諷梁文音「殘忍、可惡」。<ref>{{Cite web |url = http://www.appledaily.com.tw/realtimenews/article/new/20141008/483950/ |title = 梁文音淚崩　消毒「被神恢復」沒惡意 |author = 李志展 |date = 2014-10-08 |publisher = 蘋果日報 |language = zh-hant |accessdate = 2015-04-03}}</ref>
