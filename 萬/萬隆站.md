'''萬隆站'''位於[[台灣|台灣]][[台北市|台北市]][[文山區|文山區]]，為[[台北捷運|台北捷運]][[松山新店線|松山新店線]]（[[新店線|新店線]]）的[[捷運車站|捷運車站]]。過去[[臺灣鐵路管理局|臺灣鐵路管理局]][[新店線_(台鐵)|新店線]]曾於本站南方景隆街口設有[[#臺鐵萬隆車站|萬隆車站]]。

== 車站概要 ==
車站南端位於[[羅斯福路_(台北市)|羅斯福路五段]]下方，羅斯福路五段269巷巷口北側，接近萬隆街、景隆街口，設有四個出口，其中兩個出口有手扶梯裝置；車站代碼為'''G06'''，在尚未使用2016車站代碼前則採用工程代碼'''G6'''。站名取自所在地名稱與台鐵新店線原站名「'''萬隆'''」。

== 車站構造 ==
地下二層車站，一個[[島式月台|島式月台]]，四個出入口。本站設有[[半高式月台門|半高式月台門]]。

=== 車站出口 ===
[[Image:Taipei_MRT_Wanlong_Station.JPG|thumb]]
[[Image:Wanlong_Station_Exit3.jpg|thumb]]
出口1、2位於車站南端，出口3、4位於車站北端，[[無障礙|無障礙]]電梯位於出口4；此外出口2、4有聯合開發大樓。
* 出口1：萬隆街（羅斯福路西側、羅斯福路五段236巷口）
* 出口2：景隆街（羅斯福路東側、羅斯福路五段269巷口）(此出口設有捷運共構住宅)
* 出口3：武功國小（羅斯福路東側、羅斯福路五段211巷口）
* 出口4：萬福國小、萬和公園 [[File:Accessibility-directory.svg|20px]] (羅斯福路西側、羅斯福路五段218巷口）

== 利用狀況 ==
本處除捷運站外，車站附近設有公車站牌，分別是羅斯福路上的「捷運萬隆站」、「武功國小(羅斯福)」以及興隆路上的「武功國小」，有多條公車路線行經。

== 歷史 ==
=== 臺鐵萬隆車站 ===
* 1921年3月25日：隨著新店線鐵路「[[公館站|公館]]－[[新店站|新店]]」段正式通車而啟用（當時稱為「十五分乘降場」）。
* 過去有從車站沿景隆街、興隆路通往今 [[行政院|行政院]][[行政院海岸巡防署|海巡署]]一帶的運[[煤|煤]][[台車|台車]]，車站附近有儲煤場。
* 戰後新店線鐵路由臺鐵接手經營；站名從「十五分」改為「萬隆」。
* 1965年3月24日：隨著臺鐵新店線停駛而廢止。
* 站址位於今羅斯福路景隆街口一帶。<ref>{{cite web |title=新北市百年歷史地圖系統 |url=http://gissrv4.sinica.edu.tw/gis/newtaipei.aspx |website=中央研究院人文社會科學研究中心地理資訊科學研究專題中心 |quote=日治二萬五千分之一地形圖，站名記為「じゆうごふん」}}</ref>

=== 台北捷運萬隆站 ===
*1999年11月11日<ref name="start_date"></ref>：隨著捷運新店線「[[古亭站|古亭]]－新店」段正式通車而啟用。
*2018年3月28日：半高式月台門正式啟用。
