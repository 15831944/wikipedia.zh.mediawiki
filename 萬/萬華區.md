'''萬華區'''，是一個位於臺灣[[臺北市|臺北市]]西南側的行政區劃。設立於1990年，由原[[龍山區_(台北市)|龍山區]]、[[雙園區|雙園區]]大部份區域及[[城中區_(台北市)|城中區]]、[[古亭區|古亭區]]小部份區域合併而成。

萬華區是台北市較早發展的區域，有許多老舊社區及[[古蹟|古蹟]]，也有年輕人喜好聚集的[[西門商圈|西門商圈]]徒步區引領時尚潮流。萬華區同時也是台北市供應新鮮食材的果菜、肉類運銷中心，2010年台北市舉辦花卉博覽會而制定區花，萬華區為白牡丹，民政局也特別製作區花版門牌。

== 地名考究 ==
{{main|艋舺}}
「萬華區」是個行政區名，若「萬華」一詞被指為地名，則是專指[[艋舺|艋舺]]這一地區；艋舺是萬華舊稱，[[台灣日治時期|日治時]]，因艋舺以[[台灣閩南語|台灣閩南語]]發音（Báng-kah），與佛典中的「萬華」（Banka）[[日語|日語]][[音讀|音讀]]相似，「艋舺」因此易名為「萬華」，今日[[台灣人|台灣人]]猶稱其為「艋舺」。[[艋舺|艋舺]]原為[[台灣原住民|台灣原住民]][[平埔族|平埔族]][[凱達格蘭族|凱達格蘭族]]語「獨木舟」之意，因其為[[港口|港口]]，多見[[獨木舟|獨木舟]]。[[艋舺|艋舺]]因港商之利，曾經盛極一時，於[[台灣清治時期|清治時期]]，與[[台南市|台南府城]]、[[彰化縣|彰化]][[鹿港鎮|鹿港]]並列為三大重要城市，有「[[一府二鹿三艋舺|一府二鹿三艋舺]]」之稱，現則有城市結盟。同屬[[南島語系|南島語系]]的[[菲律賓語|菲律賓語]]bangka，亦有船之意。

== 地理位置 ==
萬華區共有36個[[里|里]]，經過幾次行政區重畫後，目前的萬華區南以[[新店溪|新店溪]]與[[新北市|新北市]][[中和區|中和區]]、[[永和區|永和區]]為界，西以[[淡水河|淡水河]]、[[新店溪|新店溪]]與新北市[[三重區|三重區]]、[[板橋區_(台灣)|板橋區]]分隔，北邊與[[大同區_(臺灣)|大同區]]以[[忠孝西路_(臺北市)|忠孝西路]]二段為界，東邊則以[[中華路_(台北市)|中華路]]與[[中正區_(臺北市)|中正區]]接壤。

萬華區南北發展主軸為[[康定路_(台北市)|康定路]]及[[萬大路_(台北市)|萬大路]]、東西主軸為[[和平西路|和平西路]]。在大眾運輸方面，除了有[[台北捷運南港線|台北捷運南港線]]、[[板南線|板南線]]的[[西門站|西門站]]和[[龍山寺站|龍山寺站]]外，[[西門站|西門站]]還有[[台北捷運松山線|台北捷運松山線]]通行，以及正在興建中的[[萬大-中和-樹林線|捷運萬大線]][[加蚋站|加蚋站]]一座捷運站，合計共有3座捷運站。

== 分區發展 ==
[[File:Taipei_Cinema_Park_light_box_20171111.jpg|thumb]]
萬華區內可以分為幾個小區：
*北邊的西門中心商業區（[[西門町|西門町]]）為購物娛樂、商業、電影藝術文化中心，[[臺北市電影主題公園|臺北市電影主題公園]]也設於西門町內。
*西邊的[[艋舺龍山寺|艋舺龍山寺]]傳統街區為歷史、寺廟街區，其中有二級古蹟龍山寺、[[青草巷_(台湾)|青草巷]]、三級古蹟[[艋舺青山宮|艋舺青山宮]]、[[華西街夜市|華西街夜市]]等，外國遊客絡繹不絕。
**東三水街市場：位於剝皮寮旁的的東三水街市場的歷史悠久，更是當地人每天都會造訪的地方，市場內的蔬菜、水果和肉類皆一應俱全且極為新鮮，也包含各式生活用品及服飾類，可以說是非常繽紛的一座市場。
**青草巷：緊鄰龍山寺旁的街道便是著名的青草巷，還沒走到路口，一陣青草香便隨之撲鼻而來，有別於以往中藥店販賣乾燥藥材煎藥，青草巷是以販賣新鮮的青草為特點，店家不多，卻有上百種的草藥供顧客選擇，是台灣非常少見的景點。在開墾艋舺時期，流行疾病相當多，而民眾來到龍山寺拜拜祈求健康後便會到青草巷抓幾帖藥草回家煎，因此漸漸在龍山寺附近進形成藥草鋪的市集。

*東南則為[[青年公園_(臺北市)|青年公園]]住宅休閒區，有大量的[[眷村|眷村]]、[[國宅|國宅]]，以及青年路泡沫紅茶街和[[南機場夜市|南機場夜市]]（事實上，南機場夜市大部分位於比鄰的中正區）。
*[[萬華車站|萬華車站]]在萬華區之中，周遭則為[[艋舺服飾商圈|艋舺服飾商圈]]，而[[中國時報|中國時報]]也位於此處。
*萬華區西南新店溪、淡水河沿岸，有台北農產運銷公司、台北漁產運銷公司、台北畜產運銷公司、環南家禽市場、環南綜合市場等，是台北市民生物資的大本營。此外，華江橋水鳥保護區每年冬天有大量[[小水鴨|小水鴨]]、[[琵嘴鴨|琵嘴鴨]]等候鳥棲息，也是一大特色。

萬華區是[[臺北市|台北市]]最老舊的社區，最近幾年人口逐年緩慢下降，人口老化嚴重。由於1949年
[[中華民國政府|中華民國政府]]遷台後，萬華區南機場附近興建了大量[[眷村|眷村]]，所以中華路二段至萬大路間的[[外省人|外省人]]比率較高，除此之外多以[[本省人|本省人]]為主。而南機場至[[青年公園_(臺北市)|青年公園]]之間，隨後又新建了大量的[[國宅|國宅]]，也發展出特有的國宅文化。

*艋舺大道：位於萬華區的中間地帶，原為縱貫鐵路，鐵路地下化後與中華路、板橋縣民大道連成一線。
*[[中華路_(台北市)|中華路]]一、二段：萬華東邊的主要道路，與[[中正區_(臺北市)|中正區]]的區界，為[[台三線|台三線]]一段，日治時期稱「三線路」。
*[[環河南路|環河南路]]一、二段：萬華西邊的主要道路，瀕臨[[淡水河|淡水河]]、[[新店溪|新店溪]]。
*[[和平西路|和平西路]]三段：通往[[板橋區_(台灣)|板橋]]、[[古亭_(臺北市)|古亭]]的道路，為[[台三線|台三線]]一段，與[[華江橋|華江橋]]聯結。
*[[忠孝西路_(臺北市)|忠孝西路]]二段：通往[[三重區|三重]]、[[台北東區|台北東區]]的道路，與[[大同區_(臺灣)|大同區]]的區界。
*[[西園路|西園路]]：貫穿南北萬華的西邊道路，也是通往[[埔墘|埔墘]]、[[中和區|中和]]的道路，為縣道114線的台北市路段。
*[[萬大路_(台北市)|萬大路]]／[[康定路_(台北市)|康定路]]：是貫穿萬華的主要道路，也是通往[[永和區|永和]]、中和的道路。
*[[西藏路_(台北市)|西藏路]]：貫穿雙園地區的主要道路，原瑠公圳支線渠道加蓋。
*成都路：[[西門町|西門]]通往[[三重區|三重]]、[[新莊區|新莊]]的道路，為縣道104號的台北市路段。
*昆明街：為單向道路（往北），貫穿[[艋舺|艋舺]]，來往舊[[龍山區_(台北市)|龍山區]]與[[西門町|西門町]]之間。
*西寧南路：為單向道路（往南），貫穿[[艋舺|艋舺]]，來往舊[[龍山區_(台北市)|龍山區]]與[[西門町|西門町]]之間。
*貴陽街二段：為單向道路（往東），可通往[[西門町|西門]]、[[博愛特區|博愛特區]]，是[[艋舺|艋舺]]歷史最古老的街道。
*桂林路：路底水門可行至[[華江橋|華江橋]]往[[板橋區_(台灣)|板橋]]的機車道和淡水河堤外機車專用道。
*廣州街：許多商家林立，往東可至[[台北府城小南門|小南門]]。
*東園街：為加蚋仔重要道路，是[[東園町|東園町]]與[[西園町|西園町]]分界。
*青年路：相鄰[[青年公園_(臺北市)|青年公園]]，於中華路二段484巷和南海路相連接。
