'''白蔹'''（学名：''Ampelopsis japonica''）亦称'''鹅抱蛋'''、'''白葡萄秧'''。为[[葡萄科|葡萄科]][[蛇葡萄属|蛇葡萄属]][[藤本|藤本]]植物，[[花期|花期]]为5-6月，[[果熟期|果熟期]]为9-10月。

== 形态 ==
白蔹有纺锤状块根，掌状叶，3-5全裂，裂片形状多样，叶轴有翅；夏季开黄绿色小花，聚伞花序；[[浆果|浆果]]大如豌豆，初为兰色，后变白色。

== 分布 ==
分布于中国东北、华北、华东以及中南各省，亦见于日本。全草以及块根可入药，用于清热解毒、消肿止痛，不过不宜与[[乌头|乌头]]同用。

== 药用 ==
中医上以块根入药，性微寒、味苦辛，功能为泻火、散结、生肌、止痛，主治痈肿疮毒、烫火伤、赤白带下、淋巴结核等症。
