'''白米山古墳'''（{{jpn|j=白米山古墳|hg=しらげやまこふん|rm=Shirageyamakofun}}）是位於日本[[京都府|京都府]][[與謝郡|與謝郡]][[與謝野町|與謝野町]]後野的一座{{Link-ja|前方後圓墳|前方後円墳}}，日本國史跡。

== 概要 ==
古墳位於京都府北部，[[丹後半島|丹後半島]]南部{{Link-ja|加悅谷|加悦谷}}東面的丘陵{{Sfn|白米山古墳（続古墳）|2002}}<ref name="国指定"/>。[[昭和|昭和]]初期時，古墳首次被發現<ref name="国指定"/>。1987年4月15日，古墳獲指定為京都府指定史跡<ref name="京都府">{{cite web |url=http://www.kyoto-be.ne.jp/bunkazai/cms/?page_id=200 |language=ja |script-title=ja:京都府指定・登録等文化財 |publisher={{Link-ja|京都府教育委員會|京都府教育委員会}} |accessdate=2018-07-17}}</ref>。1993年，京都府埋藏文化財調查研究中心對隣接於古墳的白米山北古墳展開考古調查<ref name="白米山北古墳"/>，1996年至1999年，{{Link-ja|加悦町|加悦町}}教育委員會對古墳展開範圍確認考古調查{{Sfn|白米山古墳（続古墳）|2002}}，[[奈良女子大學|奈良女子大學]]則在1998年對隣接於古墳的白米山西古墳群展開考古調查{{Sfn|白米山古墳（続古墳）|2002}}。2002年，古墳獲指定為日本國史跡<ref name="国指定"/>。。

古墳是前方後圓墳，前方部分朝北<ref name="国指定"/>。墳丘有兩層{{Sfn|白米山古墳（続古墳）|2002}}<ref name="国指定"/>，長約90米，在加悅谷地區來說僅次於以北約1.5公里外的{{Link-ja|蛭子山古墳|蛭子山古墳|蛭子山1號墳}}（145米）。墳丘斜面有{{Link-ja|葺石|葺石}}，{{link-ja|盛土}}部分則鋪有圓形[[鵝卵石|鵝卵石]]，未見有[[埴輪|埴輪]]{{Sfn|白米山古墳（続古墳）|2002}}。後圓部分中央的墓室內建有{{Link-ja|豎穴式石室|竪穴式石室}}一座、{{Link-ja|土坑墓|土坑墓|土壙墓}}兩座和{{Link-ja|木棺|木棺|木棺直葬}}一處等等，但是由於尚未展開調查，詳情不明{{Sfn|白米山古墳（続古墳）|2002}}。

從出土土器等推測，古墳建於[[古墳時代|古墳時代]]前期中葉（約4世紀）{{Sfn|白米山古墳（続古墳）|2002}}<ref name="国指定"/>。在[[丹後國|丹後地方]]，被稱為「日本海三大古墳」的蛭子山1號墳、{{Link-ja|神明山古墳|神明山古墳}}（[[京丹後市|京丹後市]]丹後町宮）和{{Link-ja|網野銚子山古墳|網野銚子山古墳}}（京丹後市網野町網野）均建於古墳時代前期後半，均早于本古墳，本古墳建成时乃丹後地方規模最大的古墳，在当时日本全國來說也是數一數二<ref name="国指定"/>。另外，古墳亦被認為是[[日本海|日本海]]方面歷史最悠久的前方後圓墳{{Sfn|白米山古墳（国指定史跡）}}，對於研究丹後地方的前方後圓墳的影響來說非常重要<ref name="国指定"/>。

== 規模 ==
古墳規模如下{{Sfn|白米山古墳（続古墳）|2002}}。
* 墳丘長：90米
* 後圓部分：兩層。
** 直徑：54米
** 高：7.5米
* 前方部分：兩層。
** 寬：32米
** 高：5米
墳丘的第一層天然地基，第二層為人工加上的盛土{{Sfn|白米山古墳（続古墳）|2002}}。

== 墓室 ==
[[File:Shirageyama_Kofun,_funcho.jpg|thumb]]
後圓部分中央的墓室內建有豎穴式石室一座、土壙墓兩座和木棺直葬一處等等，但是由於尚未展開調查，詳情不明{{Sfn|白米山古墳（続古墳）|2002}}。豎穴式石室橫置於墳丘內，長約4米，另有6塊天井石{{Sfn|白米山古墳（国指定史跡）}}<ref name="国指定"/>。在石室周圍亦發現了[[樁|樁]]杭列和{{Link-ja|柱穴|柱穴}}列{{Sfn|白米山古墳（続古墳）|2002}}<ref name="国指定"/>。

另外，在盛土部分發現了豎穴式小石槨兩座和{{Link-ja|石棺 (日本)|石棺|石棺}}直葬一處，並未有發現到[[陪葬品|陪葬品]]{{Sfn|白米山古墳（続古墳）|2002}}，古墳周邊建有兩座{{Link-ja|彌生時代的墓制|弥生時代の墓制|方形周溝墓}}（出土了{{Link-ja|刀子|刀子}}一把），溝內部分則是{{Link-ja|割竹形木棺|割竹形木棺}}直葬一處（出土了[[刨|刨]]一個）、木蓋土壙墓一座和土壙墓三座{{Sfn|白米山古墳（続古墳）|2002}}。

== 周邊古墳 ==
古墳的後圓部分以西的四座古墳被統稱為'''白米山西古墳群'''{{Sfn|白米山古墳（続古墳）|2002}}。其中1號墳和2號墳在1998年時展開了考古調查{{Sfn|白米山古墳（続古墳）|2002}}。1號墳屬於小型[[方墳|方墳]]。主體部分為割竹形木棺直葬。出土了鐵斧一把{{Sfn|白米山古墳（続古墳）|2002}}。而2號墳至4號墳則是半圓形的台狀墓。主體部分為割竹形木棺直葬{{Sfn|白米山古墳（続古墳）|2002}}。

另外，古墳的前方部分以西是'''白米山北古墳'''<ref name="白米山北古墳">{{cite book|url=http://www.kyotofu-maibun.or.jp/data/kankou/kankou-pdf/jyouhou/kyoutofu-J50.pdf |language=ja |script-title=ja:京都府埋蔵文化財情報 第50号 |chapter=白米山北古墳 |publisher=京都府埋藏文化財調查研究中心 |date=1993-12-27 |accessdate=2018-07-10}}</ref>，以東則是'''白米山東古墳'''。

以上的古墳的入葬者被認為與白米山古墳的入葬者屬於同一團體<ref name="国指定"/>。
