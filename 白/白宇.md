'''白宇'''（{{bd|1990年|4月8日|}}），生於[[陕西省|陕西省]][[西安市|西安市]]，[[中國大陸|中國大陸]]影视[[男演員|男演員]]，毕业于[[中央戏剧学院|中央戏剧学院]]2009级表演系。因出演电视剧《[[长大|长大]]》谢南翔一角为人熟悉<ref>[http://ent.163.com/14/0418/11/9Q41SBN400031GVS.html 白宇《长大》付出"第一次" 白百何护短小师弟]</ref>。2016年，主演網路劇《[[美人为馅|美人为馅]]》獲得不錯的回響。2018年，主演網路劇《忽而今夏》豆瓣高分收穫好口碑。同年，其主演的网络剧《[[镇魂_(网络剧)|镇魂]]》为當年现象级作品，獲得不少關注，演藝事業穩步上升。<ref>{{Cite web |url=http://ent.southcn.com/8/2018-07/31/content_182775632.htm |title=内容终是王道 《镇魂》“现象级”火爆不可复制|accessdate=2018-08-20 |author=罗予岐 |date=2018-07-31|work= |publisher=南方网|language=Zh-hans }}</ref>

他總對粉絲說的話—“愛自己、愛家人，人間正道是學習，成為一個更好的人。”

== 早年經歷 ==
白宇自承高中屬於學習偏中等的學生，家裡人也覺得他從小在藝術方面有點天份，於是高二時去學播音主持，學了一個多月，老師覺得他適合學表演，這便是白宇報考[[中央戏剧学院|中央戲劇學院]]的契機。畢業時，三個畢業大戲，有兩個戲白宇都是男一號。

== 演藝經歷 ==

=== 2013年—朋友-機會 ===
2013年，喜歡Vespa的白宇，因為車友的推薦而去參加選拔《屌絲日記》，初期試鏡的角色是高富帥男二，被林妍導演看中最終主演男一屌絲設計師─尤東東。正式進入演藝圈。之後又參演了同樣是林妍導演的作品—《[[長大_(電視劇)|長大]]》、《你好喬安》(2015年9月)。

2014年，1月15日《屌絲日記》樂視全網首播。2月《[[長大_(電視劇)|長大]]》開機。6月9日，從外地工作3個多月後返家發現愛車VESPA—vbc小綠被偷了。10月，備受關注的年代戲《[[少帅|少帥]]》的副導演王瀚儷偶爾在《少年班》的面試會上見到遠處的白宇，覺得他可塑性強，試鏡之後發現他很符合馮庸這角色對年齡跨度的要求，是以白宇獲得了這個演出機會，並且在演戲一途上有很大的收穫與提升。

=== 2015年 ===
2015年，1月29日，其參演的改編自網路作家zhuzhu6p同名網路小說《[[長大_(電視劇)|長大]]》在東方衛視、天津衛視播出，故事以醫院為背景，講述了一群實習醫生在他們的外科實習過程中從幼稚邁向​​成熟的故事。白宇在劇中飾演性格逗趣又有義氣的仁華實習生謝南翔。4月18日，客串演出都市愛情劇《[[云巅之上|雲巔之上]]》，飾演深情隱忍又溫柔的牧歌一角。8月22日到11月24日，拍攝根據顧漫同名小說改編的都市青春偶像劇《[[微微一笑很倾城_(电视剧)|微微一笑很傾城]]》，在劇中飾演憤世嫉俗的外文系才子曹光。9月，客串演出都市情感勵志劇《你好喬安》，飾演女主角喬安的前男友、富二代陳驍。

=== 2016年—磨練與成長 ===
2016年，1月1日到2月26日，拍攝《[[微微一笑很傾城_(電影)|微微一笑很傾城]]》電影版，飾演女主角遊戲裡的前夫真水無香，這是他首部電影作品。4月，由於之前客串了[[于正|于正]]製片《[[云巅之上|雲巔之上]]》的牧歌，于正看中邀請他演出《[[美人为馅|美人為餡]]》的男二，而愛奇藝的製片人戴瑩見了白宇本人之後，覺得這就是自己心目中的韓沉，因此決定啟用他演男主，10月24日播出之後，從開始的不被看好，白宇成功的用實力說服了觀眾。在當年的12月12日，《美人為餡》在愛奇藝圓滿收官，三季總點擊率截止12月28日超過23億。並憑此劇與女主楊蓉獲得《愛奇藝尖叫之夜年度最受關注螢幕CP獎》。4月，費時15天，客串朋友的網路電影《騙愛天團》，飾演性感迷人的情場高手楊修賢。8月12日，《[[微微一笑很傾城_(電影)|微微一笑很傾城]]》全院線首映。8月22日，《[[大漠骠骑—霍去病|大漠驃騎—霍去病]]》開機，在劇中飾演單于之子伯力，這是他第一部古裝劇，這部戲的拍攝時長與冰天雪地的外景環境讓白宇在演戲上有不少磨練、挑戰與成長。

同年8月1日建軍節，試鏡獲得了蔡晴川一角的建國三部曲中的最終部《[[建军大业|建軍大業]]》開機，白宇在片中有時間不長卻相當重要的戲───死守三河壩戰役。白宇很好地演出了軍人的英勇與無畏，'''劉偉強導演也對他的表演表達了肯定：「白宇演的蔡晴川是死守三河壩的一位軍官，他的文戲也演得很好，跟陳毅告別時激情中帶著傷感。有個鏡頭是一個爆破把他爆出來，滾到山頭再回去按引爆器，當時我們大概有200個爆破點，這是我全片最難拍的一個鏡頭，能順利完成他很不錯！」'''

=== 2017年 ===
2017年，1月1日，客串于正編劇製作的古裝神話劇《[[朝歌_(電視劇)|朝歌]]》。1月10日，第一個網路綜藝《奇妙大轟趴》愛奇藝上線。4月15日至7月12日，主演priest 同名小說改編的網路劇《[[镇魂_(网络剧)|鎮魂]]》，飾演痞帥不羈的特調處處長趙雲瀾，並演唱推廣曲《時間飛行》。7月24日出席《[[建军大业|建軍大業]]》首映發布會。7月26日，主演改編自明前雨後同名小說的網路劇《[[忽而今夏|忽而今夏]]》在青島開機，歷時90天拍攝，講述了章遠和何洛跨越十年的青春時光歷經千山萬水不忘初心的純愛故事。7月27日，《[[建军大业|建軍大業]]》院線首映。11月，仗義救場友情主演《緝妖法海傳》，飾演緝妖司首領裴文德，胡國翰導演接受專訪時，透露白宇基本零片酬、即使當時健康狀況不好仍敬業演出，並且電影是收錄同期聲。<ref name="a1">{{Cite web |url=https://mp.weixin.qq.com/s/1DfkFxl9pBm4DShvVB5ihQ |title=丟了“玫瑰花瓣”又面臨“換臉”危機，否極泰來，白宇這次結局不會再是BE了吧？|accessdate=2018-08-20 |author=南都娛樂 |date=2018-07-31|work= |publisher=微信公眾號“南都娛樂”|language=Zh-hans }}</ref>《[[大漠骠骑—霍去病|大漠驃騎—霍去病]]》全劇組殺青。

=== 2018年 ===
2018年，4月19日《忽而今夏》騰訊首播，5月31日，以17億播放完美收官。5月7日，網路綜藝《大片起來嗨》錄製。5月20日，主演民國懸疑探案劇《紳探》開機。

== 公益活動 ==

* 2015年5月29日，世界環境日，響應聯合國環境規劃署及微公益發起的挑戰「兩個地球-爬樓行動」，關注節約能源。
* 2016年12月20日，響應「發作性睡病narcolepsy」發起的「我請你睡覺」公益行動，呼籲關注健康。
* 2017年1月30日，響應2017「壹起公益」，呼籲多使用公共交通工具，關心環保。
* 2017年2月28日，「3月3日世界野生動植物日」，參與聯合國環境署、國際愛護動物基金會（IFAW）和大自然保護協會（TNC）聯合發起的“關愛保護野生動植物”公益倡導活動，以“今日小小善舉，換明日生生不息”。宣傳推廣，通過公益視頻和社交媒體等形式發聲，號召群眾積極參與野生動植物的保護，捍衛自然環境
* 2018年2月16日，響應2018「壹起公益」，呼籲多使用公共交通工具，環保是福，享受城市風景。
* 2018年5月19日，全民責任行「為愛520」號召大家一起為愛發聲，為愛行動，為愛聚集，一起用責任為愛加冕。13位行業及院校專家發起了設立5月20日為社會責任日的倡議，希望以設立傳統節日的方式引領有責任感的大國風尚。
* 2018年9月7日，作為「中國扶貧基金會教育扶貧愛心大使」，參與騰訊公益-99公益日活動，號召關注鄉村教育扶貧事業，關注孩子健康成長，助力農村，讓每一個孩子對自己有信心對未來有希望。
* 2018年10月14日，響應17日國家扶貧日，助力「脫貧攻堅戰星光行動」，傳遞正能量、傳遞愛。
* 2018年10月21日，作為「中國扶貧基金會教育扶貧愛心大使」，號召善行100的「為愛大作戰-做孩子們的蓋世英雄」活動，溫暖貧困的孩子。至12月16日歷時3個月，“宇愛同行”戰隊有10556名志願者。
*2018年11月24日，作為「中國扶貧基金會教育扶貧愛心大使」，CCTV-6電影頻道善行100公益廣告，號招愛心包裹溫暖行動，讓孩子們能在這個冬天穿上羽絨服、戴上圍巾和手套，從內到外都暖暖的。
*2019年1月5日，「中國扶貧基金會教育扶貧愛心大使」身份，以及作為善行100公益項目的愛心大使，在CCTV-6電影頻道節目宣導美好學校聯合公益。
* 2019年2月8日，「壹起公益」關注兒童教育。<br />
