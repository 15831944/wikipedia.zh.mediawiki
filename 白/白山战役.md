'''白山戰役'''或称'''白山之战'''發生於1620年11月8日，是[[三十年戰爭|三十年戰爭]]早期的一場戰役。是役，由[[安哈爾特公爵|安哈爾特公爵]]克里斯蒂安率領的20000名[[波西米亞|波西米亞]]及傭兵部隊在[[布拉格|布拉格]]附近的白山被[[神聖羅馬帝國皇帝|神聖羅馬帝國皇帝]][[斐迪南二世_(神聖羅馬帝國)|費迪南二世]]陣營的25000名聯軍部隊擊敗。聯軍由[[法國|法國]]將軍布[[奎伯爵|奎伯爵]]（Karel Bonaventura Buquoy）指揮的帝國軍和[[約翰·蒂利|約翰·蒂利]]指揮的的[[天主教|天主教]]聯盟軍隊組成。這場戰役結束了三十年戰爭的波西米亞階段。

== 背景 ==

[[神聖羅馬帝國|神聖羅馬帝國]]的[[馬蒂亞斯_(神聖羅馬帝國)|馬提亞皇帝]]一直希望他的繼承人費迪南二世能夠繼承波西米亞和[[匈牙利|匈牙利]]的王位。費迪南二世在1617年成功被選為波西米亞王儲，並在馬提亞皇帝死後繼位。波西米亞境內的[[新教徒|新教徒]]變得人心惶惶，因為信仰天主教的費迪南二世極有可能對新教徒不利。同時，由於費迪南二世又是神聖羅馬帝國皇帝，一直得到[[羅馬天主教會|羅馬天主教會]]的支持，他將新教徒視作心腹之患，特別懼怕新教徒轉化天主教領主們的信仰。而此時他領土內的大部分居民都已經改信新教了。這些新教地區有自己的憲法和統治規則，成了帝國內的國中國。隨著費迪南統治下的人民越來越多的信仰了新教，新教領地內的人民開始爭奪天主教會的土地；同時對費迪南的集權行為進行抵制。波西米亞成了第一個反抗天主教勢力的地區，爆發了著名的“[[布拉格拋窗事件|布拉格拋窗事件]]”：波西米亞貴族要求神聖羅馬帝國保證過去費迪南前任給予波西米亞的宗教自由繼續有效，結果爆發衝突，費迪南德大臣被扔出窗外，幸好落在一堆肥料裡，這才撿回一條命。隨即，波西米亞大起義爆發了。

== 战役经过 ==

波西米亞的領主們組織了20000人保衛他們的自由，而費迪南也隨即組織網羅了25000傭兵，由西屬佛蘭德教士、著名傭兵將領蒂利前去鎮壓。大軍直撲起義軍控制的波西米亞首都[[布拉格|布拉格]]。波西米亞軍企圖在一座高地上設防抵抗帝國軍，而帝國軍並不為所動，繞過高地繼續向布拉格進軍。支持波西米亞的安哈爾特公爵克里斯蒂安則拼命強行軍，以便趕在帝國軍之前到達布拉格支援起義軍。錯過遭遇戰的波西米亞軍在布拉格近郊再次設防，但是士氣下降了不少。帝國軍到達後，蒂利將部隊分為​​帝國軍和天主教聯盟兩部分。先是聯軍一方在沒有炮火掩護的情況下就發起了進攻。而安哈爾特公爵也派遣自己的兒子率領步騎部隊向前推進。騎兵衝入帝國步兵陣中，給帝國軍以很大殺傷。蒂利也放出自己的騎兵將波西米亞騎兵擊退。此時接敵的波西米亞步兵看到騎兵敗退了，不由得軍心動搖，在敵軍剛到射程內就匆匆發射然後就開始後退。天主教中的近400名哥薩克騎兵飛一般的衝上去包圍了波西米亞部隊。由於波西米亞軍已經軍心動搖，各個部隊爭相逃命，有的甚至還沒接敵就跑了。隨著波軍不斷潰散，[[哥薩克|哥薩克]]和[[巴伐利亞|巴伐利亞]](由巴伐利亞公爵[[馬克米利安二世|馬克米利安二世]]指揮)的輕騎兵在後面一路追殺。波西米亞軍試圖在星宮（Star Palace）組織一道最后防線，但仍被蒂利的軍隊突破了。白山戰役的戰役規模充其量只能算一次前哨戰。波西米亞軍完全不是訓練有素的帝國傭兵們的對手，不到一個小時被打得一敗塗地。超過4000名新教士兵被俘或陣亡，而帝國軍的損失不到800人。

潰逃的波西米亞逃到布拉格，引起了一陣恐慌。一些起義領袖試圖組織防禦但根本得不到支援。新教徒自己選出來的國王──帕拉丁[[選帝侯|選帝侯]][[腓特烈五世_(普法爾茨)|腓特烈五世]]倉皇而逃，連王冠都忘了拿，混在難民隊伍裡逃出了布拉格。帝國軍隨即在沒有任何抵抗的情況下進入了布拉格。

由於取得了對內戰爭的勝利，費迪南得以騰出精力處理國際問題。他的大臣們則堅定了要加強集權的信念，要求皇帝取消波西米亞的選舉自由以防止再出現叛亂。

== 后续 ==

蒂利大軍開進布拉格，[[天主教|天主教]]徒們彈冠相慶，歡迎帝國軍的到來。47名新教領袖受到審判，其中27人在廣場上被處決，是日被新教徒稱作“血腥之日”。56名波西米亞貴族被流放，財產被剝奪。之後的三十年戰爭中波西米亞一直戰亂不斷，戰前的151000個莊園到1648年戰爭結束時只剩下50000個了。不過，在[[西里西亞|西里西亞]]和[[摩拉維亞|摩拉維亞]]、[[斯拉夫|斯拉夫]]等地區，仍有新教義軍反抗帝國軍，直到1623年。

1621年，帝國皇帝發布敕令：所有[[喀爾文主義|加爾文教徒]]和非[[路德教|路德教]][[新教徒|新教徒]]必須在三日內離開帝國領土。次年又驅逐了[[路德教|路德教]]徒。 1627年，布拉格大主教[[哈拉希|哈拉希]]為異教徒舉行轉信儀式，大部分波西米亞人都改信了天主教，但是仍有一小部分堅定的新教徒存在。與此同時，[[西班牙|西班牙]]大肆進軍[[荷蘭|荷蘭]]和[[普法茲|普法茲]]，[[德國|德國]]和[[丹麥|丹麥]]也捲入了戰爭，三十年戰爭徹底演變成全歐洲的國際大戰。
