'''白露'''，是[[节气|二十四节气]]之一，每年在9月7-9日之间。[[太阳|太阳]]位于黄经165°。

白露是[[节气|二十四節氣]]中的第十五個節氣。每年八月中（[[西曆|西曆]]9月7日或8日）太陽到達[[黃經|黃經]]165度時為白露。《[[月令七十二候集解|月令七十二候集解]]》中說：「八月節……陰氣漸重，露凝而白也。」 天氣漸轉涼，會在清晨時分發現地面和葉子上有許多[[露水|露珠]]，這是因夜晚水氣凝結在上面，故名。古人以四時配[[五行|五行]]，秋屬金，金色白，故以白形容秋露。進入「白露」之後，在晚上會感到一絲絲的涼意。俗语云：“处暑十八盆，白露勿露身。”这两句话的意思是说，处暑仍热，每天须用一盆水洗澡，过了十八天，到了白露，就不要赤膊裸体了，以免着凉。还有句俗话：“白露白迷迷，秋分稻秀齐。”意思是说，白露前后若有露，则晚稻将有好收成。

== 名稱 ==
該[[節氣|節氣]]在歷史上均稱為「白露」，在西夏曆法中則改稱為「露寒」<ref>[http://www.sinoss.net/qikan/uploadfile/2010/1130/10734.pdf 彭向前、李曉玉：《一件黑水城出土的夏漢合璧曆日考釋》]。</ref>。

== 物候 ==
*鴻雁來：[[鴻|鴻]]大[[雁|雁]]小，自北而來南也，不謂南鄉，非其居耳。詳見[[雨水|雨水]]節下。
*玄鳥歸：[[玄鳥|玄鳥]]解見[[春分|春分]]，此時自南而往北也，燕乃北方之鳥，故曰歸。
*群鳥養羞：三人以上為眾，三獸以上為群，群者，眾也，《[[禮記|禮記]]》注曰：「羞者，所美之食。」養羞者，藏之以備冬月之養也。

== 日期计算 ==
[Y*D+C]-L<br />

公式解读：Y=年数后2位，D=0.2422，C=闰年数，21世纪C=7.646，20世纪=8.44。<br />

举例说明：2088年白露日期=[88×0.2422+7.646]-[88/4]=28-22=6，9月6日是白露。<br />

例外：1927年的计算结果加1日。
