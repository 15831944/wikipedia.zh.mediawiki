白爛貓是[[麻糬爸|麻糬爸]]27歲（1989年生）時為[[即時通訊軟體|即時通訊軟體]][[LINE|LINE]]创作的貼圖人物<ref name="lan1"/>。

== 簡介 ==
作者大學時期主修[[室內設計|室內設計]]，畢業後在[[夜市|夜市]]販賣男裝和寵物用品，後來因故結束營業。遂興起再重回設計行業的念頭，第1組作品為以所飼養的[[法國鬥牛犬|法國鬥牛犬]]「麻糬」發想繪出的「法鬥麻糬肥吱吱」，推出之後反應平平，之後朋友建議他將自己的內心世界融入在作品中，冷言冷語的臭跩貓以及愛嗆聲的白爛貓就此誕生。其中角色對話直白嗆辣，某種程度解放了[[上班族|上班族]]的內心苦悶與大眾對生活的無奈。當中的點子皆為日常生活所見與朋友聊天內容。<br />
「臭跩貓愛嗆人3」此組貼圖奪下2016年上半年度全球百大貼圖排行榜第4名<ref name="lan1">{{Cite news|url=https://tw.appledaily.com/new/realtime/20160709/903027/ |title=【暢銷貼圖】麻糬爸夜市擺攤　血淚逼出白爛貓 |publisher= 蘋果日報 |date=2016/07/09 | language=zh-tw}}</ref>、白爛貓系列也奪下2017年度台灣Line原創貼圖銷售第1名<ref>{{Cite news|url=https://tw.appledaily.com/new/realtime/20180209/1295645/ |title=LINE貼圖年度排行出爐　前五名銷售額破7千萬 |publisher= 壹週刊| date=2018/02/09 |language=zh-tw}}</ref>。白爛貓除了貼圖外也有眾多實體商品上市。2017年12月在[[松山文創園區|松山文創園區]]舉辦「白爛貓超有事特展」，將平面貼圖立體化<ref>{{Cite news|url=https://www.ydn.com.tw/News/254638 |title=「白爛貓超有事」特展12月開跑 走出Line給你摸 |publisher= 青年日報| date=2017/09/17 |language=zh-tw}}</ref>。

== 角色 ==
*白爛貓（暱稱：爛爛），性別：男，生日2015年12月2日（射手座），是一隻橘子[[虎斑貓|虎斑貓]]，个性白烂浮夸，极度自恋，直白不做作，喜欢好吃懒做，讨厌、害怕星期一<ref>{{Cite news|url=https://www.cna.com.tw/culture/article/20180209w004 |title=賤的力量：白爛貓貼圖拯救百萬上班族靈魂 |publisher=中央社 |date=2018-03-14 06:30 |accessdate=2018-10-19}}</ref>。
*臭跩猫（暱稱：跩跩），性別：男，生日2015年10月16日（天秤座），是一隻灰色虎斑[[異國短毛貓|異國短毛貓]]，个性高傲冷酷，不爱搭理人，刀子嘴豆腐心，爱玩手机是個暖男。
*麻糬（暱稱：糬糬），性別：女，是個[[天然呆|天然呆]]的[[法國鬥牛犬|鬥牛犬]]，是一個個性善良體貼的[[小可愛|小可愛]]。
*那條魚（暱稱：魚魚），白爛貓手中的魚，[[寵物|寵物]]兼儲備[[糧食|糧食]]。
*那隻雞，自以為是[[鶴|鶴]]的[[雞|雞]]，所以會飛。
*星期一～星期日
