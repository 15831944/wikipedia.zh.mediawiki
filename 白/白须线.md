'''白须线'''（{{jpn|j=白鬚線|hg=しらひげせん|rm=Shirahige sen|f=y}}）是连接[[東京都|東京都]][[墨田區|墨田区]]的[[向島駅_(東京都)|向岛站]]和白须站之间曾经由京成电气轨道（今[[京成電鐵|京成電鉄]]）运营铁路，现已废弃。

== 路线资料 ==
* 路线距离（运营里程）：1.4km
* [[轨距|轨距]]：1372mm
* 车站总数：4站
* 复线区间：全线
* 电气化区间：全线（直流600V）

== 历史 ==
* 1928年（昭和3年）4月7日启用。
* 1936年（昭和11年）1月27日全线关闭。

== 车站 ==
[[向島駅_(東京都)|向岛站]] - 长浦站 - [[東向島站|玉之井站]]（本线站名为“京成玉之井站”） - 白须站
