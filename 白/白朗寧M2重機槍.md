'''白朗寧M2重機槍'''（{{lang|en|M2 Machine Gun}}），俗稱'''五零'''/'''點五零機槍'''（{{lang|en|Browning Machine Gun，Cal. .50， M2HB，Flexible.}}），是由[[約翰·白朗寧|約翰·白朗寧]]在[[一戰|一戰]]後設計的[[重機槍|重機槍]]，發射[[12.7×99mm_NATO|12.7×99毫米]]（.50 BMG）大口徑彈藥。

== 历史 ==
白朗寧M2其實是[[白朗寧M1917重機槍|白朗寧M1917]]的口徑放大重制版本，它的起源也是出自於[[第一次世界大戰|第一次世界大戰]]的作戰需求；在1917年，德意志帝國研發出[[J.I攻擊機|J.I攻擊機]]，她的結構結實足以抵抗[[.30-06春田步槍彈|.30-06春田步槍彈]]，<ref name="Chinn 1951 333">{{Harvnb|Chinn|1951|p=333}}</ref>為此{{link-en|美國遠征軍|American Expeditionary Forces}}指揮官[[約翰·潘興|潘興將軍]]要求要開發出更大口徑的機關槍解決這個困境。<ref>{{Harvnb|Chinn|1951|pp=181–182}}</ref>潘興向[[美國陸軍軍械部隊|美國陸軍軍械部隊]]提出新機槍規格需要口徑0.5英寸（12.7公厘）、槍口初速2,700英尺（820公尺／秒），美軍上校{{link-en|約翰·亨利·帕克|John Henry Parker (general)}}在法國機槍學校目睹法軍配備的[[霍奇克斯M1914重機槍|11公厘機槍]]有配發燃燒穿甲彈，因此美軍曾向柯特公司訂購8挺換用11公厘子彈的實驗槍，然而帕克上校所知的11公厘子彈是18世紀由英國開發俗稱「氣球破壞者」的11×59mmR Gras子彈，動能表現非常不理想；而潘興希望新槍的子彈要有670[[格令|格林]]（43公克）、槍口初速2,700英尺，所以自法國引進槍彈的構想因此打住。

1917年，白朗寧重新設計[[白朗寧M1917重機槍|白朗寧M1917重機槍]]，以符合美軍需求，[[溫徹斯特連發武器公司|溫徹斯特連發武器公司]]則加入開發機槍需要的0.5英吋子彈，初期設計使用0.30春田子彈等比例放大；0.5英吋機槍原型在1918年10月15日測試，射速每分鐘500發、槍口初速2,300英尺（700公尺／秒），性能不如預期，槍枝太重射速也太慢，火力不夠密集；不過在新機槍開發期間，美軍擄獲數批[[毛瑟13.2毫米反坦克步槍|毛瑟13.2毫米反坦克步槍]]與{{link-en|毛瑟13.2公厘反戰車子彈|13.2mm TuF}}，13.2mm TuF尺寸雖然比美軍研發的要大，但是其設計優秀，52公克的彈頭在有著秒速820公尺的良好表現，溫徹斯特借鑑了德國子彈改進彈藥設計，改良後的子彈初速提升到每秒2.750英尺（840公尺）。<ref>{{harvnb|Chinn|1951|p=186}}</ref>

1921年，新枪完成基本设计，1923年美軍把當時的M2命名為「'''[[白朗寧M1921重機槍|M1921]]'''」，該批機槍的特色為水冷式結構、僅能自左側進彈、槍管較後續服役的型號要輕，主要在1921年至1937年量產，配備在飛機與車輛上，用於1920年代的防空及反裝甲用途。1926年白朗宁去世，重機槍的改良仍然繼續進行，從1927年至1932年间，由美国的塞缪尔·格林博士针对M1921的设计问题以及军方需求做出调整<ref name="zzjz" />。1930年，柯尔特还针对M1921推出了部分改进的版本，如M1921A1与M1921E2。1932年，改進版本正式被美軍命名為「'''M2'''」，當時部份的M2裝有水冷散熱裝置<ref>[http://www.skylighters.org/history/mgs/ AAA Weapons in Focus: The M2 .50-Caliber Machine Gun]</ref>，早期的气冷式M2机枪由于枪管太轻，无法承受多角度全方位射击要求，容易過熱，后又推出改用重枪管的版本命名为「'''M2HB'''」（Heavy Barrel），後來更推出了可快速更換槍管的「'''M2QCB'''」（Quick Change Barrel）及輕量化版本，這些版本一直沿用至今。<ref>{{cite news | url = http://www.nownews.com/n/2012/02/16/415673 | title = 老而彌堅！　美軍將4.5萬挺50機槍提升至M2A1標準 | publisher =  NOWnews 新聞 | date = 2012年2月16日}}</ref>

== 設計 ==
M2的採用大口徑[[12.7×99mm_NATO|12.7×99 NATO]]（.50 BMG）[[彈藥|彈藥]]，有高火力、彈道平穩、極遠射程的優點，每分鐘450至550發（[[第二次世界大战|二戰]]時版本為每分鐘600至1200發）的射速及後坐作用系統令其在全自動發射時十分穩定，命中率亦較高，但低射速也令M2的支援火力降低。

白朗寧M2發射M2普通彈時的最大射程可達7.4公里（4.55英里），裝在M3三腳架亦有1.8公里（1.2英里）的有效射程。白朗寧M2淨重38公斤（84磅），其M3三腳架時全重20公斤（44磅），V字「蝴蝶型」扳機裝在[[機匣|機匣]]尾部並附有兩個握把，射手可通過閉鎖或開放槍機來調節全自動或半自動發射。白朗寧M2用途廣泛，為了對應不同配備，它更可在短時間內改為機匣右方供彈而無須專用工具。

由於發射訓練用途的空包彈膛壓較低，需要在槍管以三條特制金屬管裝上[[空包彈助退器|空包彈助退器]]（blank-firing adapter-BFA）以確保有足夠燃氣來保持自動循環作用。

== 採用 ==
白朗寧M2自1920年代起裝備美軍的飛機，於步兵架設的火力陣地與軍用車輛，如[[坦克|坦克]]、[[裝甲運兵車|裝甲運兵車]]等。

白朗寧M2從1921年就開始使用服役至今，並經歴了[[第二次世界大战|二戰]]、[[朝鮮戰爭|韓戰]]、[[越南战争|越戰]]、[[海湾战争|海湾战争]]、[[阿富汗戰爭_(2001年)|2001年阿富汗戰爭]]、[[伊拉克战争|伊拉克战争]]，可說是極為成功的重機槍設計，亦是美軍輕武器中服役時間最長的一種，直到21世纪在各國服役皆有很好的評價。

除了增加了快拆槍管以外，白朗寧M2的原始設計幾乎沒做太大更動並具有多種衍生型。[[瑞典|瑞典]]、[[英國|英國]]、[[澳洲|澳洲]]、[[韓國|韓國]]及[[日本|日本]]獲FN公司合法授權生產M2。

== 衍生型 ==
[[File:Twin_M2HB_machine_gun.jpg|thumb]]
=== M2衍生型 ===
在美國服役的M2有大量衍生型，美軍以不同命名作識別，其他國家亦有相似分類。

[[白朗寧M1921重機槍|白朗寧M1921]]水冷式機槍亦叫作M2，全稱為{{lang|en|Browning Machine Gun, Cal. .50, M2, Water-Cooled, Flexible}}，但白朗寧M2沒有水冷版本。

重槍管氣冷式M2可分為三個小分類：
* '''步兵型'''—{{lang|en|Browning Machine Gun, Cal. .50, M2, HB, Flexible}}，步兵用的版本
* '''同軸機槍型'''—{{lang|en|Browning Machine Gun, Cal. .50, M2, HB, Fixed}}，安裝在[[M6重型坦克|M6重型坦克]]上
* '''砲塔型'''—{{lang|en|Browning Machine Gun, Cal. .50, M2, HB, Turret}}，安裝在砲塔上的版本

飛機及直升機上亦有採用以上版本，名為AN/M2。

=== AN/M2、M3、XM296/M296及GAU-10/A ===
[[File:P-51_Armament.jpg|thumb]]
[[File:P-38_gun_detail.jpg|thumb]]
M2經常被用作飛機上的遙控式固定武器或空用機槍，又名'''AN/M2'''（.30及 .50口徑名稱相同），遙控式的M2以電子或線傳扳機擊發。飛機上的M2採用較輕型的槍管、以飛行時的吸風達至散熱效果，美軍官方命名為{{lang|en|Browning Machine Gun, Aircraft, Cal. .50, M2}}，在機頭的固定版本名字加上{{lang|en|Fixed}}，機組人員的[[航空机关枪|空用機槍]]版本名叫{{lang|en|Flexible}}。

二次大戰至韓戰時的美國戰鬥機（例如：[[P-51戰鬥機|P-51野馬]]，[[F6F地獄貓戰鬥機|F6F地獄貓]]和[[F-86軍刀戰鬥機|F-86軍刀]]）的空戰武器皆為4挺至8挺（通常是6挺）AN/M2重機槍，韓戰期間F-86使用的M2重機槍面對機身以鋼板蒙皮的[[MiG-15|MiG-15]]已經開始力不從心，遂逐漸將空戰武器的位置讓位給機砲，除了戰鬥機，美國轟炸機（例如：[[B-17轟炸機|B-17空中堡壘]]和[[B-24轟炸機|B-24解放者]]）皆以AN/M2重機槍作為機槍手操作的防衛機槍。

M3是M2的專用遙控擊發版本，射速比M2更高。

XM296/M296是M2/M3遙控機槍的改良型，一般整合於直升機武器系统，如[[OH-58奇奧瓦偵察直升機|OH-58奇歐瓦直升機]]。M296與其他遙控擊發的M2不同，不能單發。

'''GAU-10/A'''（國際庫藏編號/NSN 1005-01-029-3428）亦屬於M2系列的成員，但目前沒有相關資料，只發現美國空軍以此命名<ref>[http://www.designation-systems.net/usmilav/asetds/u-g.html GAU/GAK - Airborne Guns]</ref>。

=== XM213/M213、XM218、GAU-15/A、GAU-16/A及GAU-18/A ===
[[File:XM213-M213_.50_cal_UH-1_mount_Wings_over_Wine_Country_2007.JPG|thumb]]
'''XM213/M213'''是'''AN/M2'''的現代化版本，安裝在直升機的機門上。

'''GAU-16/A'''是GAU-15/A的改進版本，改良了握把及瞄準具。

'''GAU-18/A'''（前稱'''XM218'''），是M2/M3的美國空軍輕量化版本，安裝在[[MH-53直升機|MH-53J Pave low II]]及[[UH-60黑鷹直升機|HH-60鋪路鷹]]直升機上。GAU-18/A沒有採用重槍管，[[機匣|機匣]]左面供彈，右面排出彈殼。GAU-18/A裝有軟性供彈滑槽及{{Link-en|彈箱|Ammunition box}}，[[彈鏈|彈鏈]]通過滑槽進入機匣，彈箱連滑槽最多可搭載1700發子彈。

=== GAU-21/A及M3P ===
[[File:Ma_deuce_in_UH-1N.JPG|thumb]]，[[美国海军陆战队|美国海军陆战队]]，伊拉克，2003年。]]
美軍亦有採用[[Fabrique_Nationale|Fabrique Nationale]]推出的M3系列，M3可分為兩個版本。

一种是遙控擊發版本，名為'''FN M3P'''，用於[[M1097復仇者|復仇者防空系統]]，備彈300發。

另一种是版本是'''M3M'''固定式機槍，[[美國空軍|美國空軍]]及[[美國海軍|海軍]]命名為'''GAU-21/A'''，裝於直升機上。

=== M2 E-50（M2E50） ===
[[File:M1025_HMMWV_firing_M2HB.jpg|thumb]]上的白朗寧M2。]]
M2 E-50是一個實行中的長期升級計劃，主要是把步兵及美國陸軍服役的M2加裝可快速更換的槍管（QCB）、[[皮卡汀尼導軌|戰術導軌]]、{{Link-en|槍口消焰器|Flash Suppressor|消焰器}}、及手動保險制。其後新制造的M2重機槍亦會以M2 E-50為生產制式。
