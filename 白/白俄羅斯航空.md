'''白俄羅斯航空'''（{{lang-be|'''Белавія'''}}；{{lang-ru|'''Белавиа'''}}；{{lang-en|'''Belavia'''}}）是[[白俄羅斯|白俄羅斯]]的[[國家航空公司|國家航空公司]]，總部位於[[明斯克|明斯克]]。<ref>{{Template:Cite web|url = http://en.belavia.by/contacts/|title = Belavia website: Contacts|publisher = En.belavia.by|date = |accessdate = 2013-02-04}}</ref> 公司擁有1,017名員工，<ref name="FI">{{Template:Cite news|title = Directory: World Airlines|work = [[Flight_International|Flight International]]|pages = 84–85|date = 2007-03-27}}</ref>提供連接[[明斯克国际机场|明斯克国际机场]]與歐洲和[[獨立國家聯合體|獨立國家聯合體]]的航班服務，以及部分中東航線。<ref name="FI">{{Template:Cite news|title = Directory: World Airlines|work = [[Flight_International|Flight International]]|pages = 84–85|date = 2007-03-27}}</ref>
