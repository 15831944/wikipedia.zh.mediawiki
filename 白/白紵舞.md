'''白紵舞'''，是創於[[三國|三國]]時代的一種著名舞蹈。<ref>{{web reference|URL=http://www.epochtimes.com/b5/7/5/5/n1700077.htm|title=茹之〈唐代揚眉轉袖若雪飛的《白紵舞》〉|work=大紀元|year=2017年|date=5月8日}}</ref><ref name="樂舞大交流-魏晉南北朝時期">[http://www2.jdps.tyc.edu.tw/~dance/chinadance/chinadance4.htm 桃園市建德國民小學〈樂舞大交流-魏晉南北朝時期〉]</ref>

== 歷史 ==
白紵舞是因舞服用質地輕薄的白紵縫製而得名。由舞者穿著產於江蘇一帶的白紵縫製的舞衣可見白紵舞最初是江南的舞蹈。{{noteTag|見南朝沈约《[[宋書|宋書]]》卷19〈志第九 樂一〉記載：「白紵舞，按舞辭有巾袍之言。紵本是吳地所出，宜是吳舞也。」。}}從[[三國|三國]]時代的[[東吳|東吳]] 到[[晉代|晉代]]再到[[唐代|唐代]]年間，白紵舞一直盛行不衰，且是酒宴表演中的常見節目。在[[晉代|晉代]]，該舞蹈已流行於封建貴族的社會。[[西晉|西晉]][[張華|張華]]曾有以該舞為主題的《白紵舞歌詩》傳世。[[唐代|唐代]]時就將《白紵舞》列入《九部樂》及《十部樂》的「清商」樂部中<ref name="中國傳統舞蹈 長久流傳的白紵舞">[https://www.chiculture.net/0507/html/0507d11/0507d11.html 中國文化研究院 燦爛的中國文明〈中國傳統舞蹈 長久流傳的白紵舞〉]</ref>。一方面舞者在宮廷演出白紵舞，另一方面也常在士族家宴及民間表演。<ref name="中国陝西省歌舞劇院 唐代楽舞詩「長安の月">[http://www.min-on.or.jp/special/2012/china/play.html 民主音楽協会〈中国陝西省歌舞劇院 唐代楽舞詩「長安の月〉]</ref>

[[南朝|南朝]][[惠休|惠休]]《白紵舞辭》中也有「桃花水上春風出，舞袖逶迤鸞照日」之詞。而[[唐朝|唐朝]]詩人[[李白|李白]]《白紵辭》曾寫道：「吳刀翦彩縫舞衣，明妝麗服奪春輝。揚眉轉袖若雪飛，傾城獨立世所希。」兩者都讚美了白紵舞的舞姿。<ref name="《白紵辭》的擬代 * ——兼論樂府詩擬代中的復變規律">[http://ahndskxb.ahau.edu.cn/ch/reader/create_pdf.aspx?file_no=20100222 方孝玲〈《白紵辭》的擬代 * ——兼論樂府詩擬代中的復變規律〉，安徽農業大學學報（社會科學版）第19卷第2期，2010年3月 ]</ref>另外，《白紵舞》分為獨舞和群舞。<ref>{{web reference|URL=http://www.epochtimes.com/b5/7/5/5/n1700077.htm|title=茹之〈唐代揚眉轉袖若雪飛的《白紵舞》〉|work=大紀元|year=2017年|date=5月8日}}</ref>[[南朝|南朝]][[梁朝|梁朝]]的[[沈約|沈約]]曾奉[[梁武帝|梁武帝]]之命寫成《四時白紵歌》，共有《春白紵》、《夏白紵》、《秋白紵》、《冬白紵》、《夜白紵》五章。當表演《四時白紵歌》時，五個舞者通常集體起舞。在表演結束後，她們要向觀賞者進酒。

== 服飾 ==
舞者一般要佩戴珠翠飾品，且身穿白色長袖舞衣跳舞。<ref name="中國傳統舞蹈 長久流傳的白紵舞" />

== 舞蹈 ==
舞者在舞動時高舉雙袖展現天鵝飛翔狀，又會低迴婉轉，輕移舞步，如推若引，似留且行。隨樂聲節奏漸漸加快，雙袖急揮如雪飄狀，敏捷步態亦似流波狀。她們輕盈飄逸的舞姿會在觀眾面前掠過。<ref name="中國傳統舞蹈 長久流傳的白紵舞" />而該舞以舞袖為主，要求手與長袖完美的配合。最初節奏徐緩，後轉為急促。舞者會雙手舉起時，又會傾斜身體緩緩轉身，用雙袖微掩面部以半遮嬌態。當長袖輕輕拂面而過時，節奏會加快。節奏加快以後，舞者揮舞雙袖，上下翻飛。而在節奏舒緩時，雙袖則徐徐揚起。隨著長袖的舞動，舞者的舞步亦輕輕移動，輕盈飄逸。除了手部動作外，白紵舞還很重視眼神與舞姿的配合。
