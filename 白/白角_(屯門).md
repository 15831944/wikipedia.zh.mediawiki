'''白角'''（{{lang-en|'''Pak Kok'''}}）為[[香港|香港]][[屯門|屯門]]一個已消失[[海角|海角]]。

白角原為屯門[[青山灣|青山灣]]西南中央凸出之海角<ref>{{cite web |url=http://hkclweb.hkpl.gov.hk/hkclr2/igateway?svc=bsch&stype=itr&param=title&frm=1&to=1&ss=ContentMap&ctrlid=2060540&lang=cht |title=存档副本 |accessdate=2010-01-19 |deadurl=yes |archiveurl=https://web.archive.org/web/20081015215632/http://hkclweb.hkpl.gov.hk/hkclr2/igateway?svc=bsch&stype=itr&param=title&frm=1&to=1&ss=ContentMap&ctrlid=2060540&lang=cht |archivedate=2008-10-15 }}</ref>，又名'''北角'''<ref>{{cite web |url=http://hkclweb.hkpl.gov.hk/hkclr2/igateway?svc=bsch&stype=itr&param=title&frm=1&to=1&ss=ContentMap&ctrlid=760087&lang=cht |title=存档副本 |accessdate=2013-04-15 |deadurl=yes |archiveurl=https://web.archive.org/web/20110809215515/http://hkclweb.hkpl.gov.hk/hkclr2/igateway?svc=bsch&stype=itr&param=title&frm=1&to=1&ss=ContentMap&ctrlid=760087&lang=cht |archivedate=2011-08-09 }}</ref>。當年屯門對外尚未有陸路（[[青山公路|青山公路]]1910年代始通車），只靠水路運輸，人跡罕至但鄰近[[珠江口|珠江口]]，恰好用於匿藏及聯絡，於是[[孫中山|孫中山]]便於白角附近興建[[青山紅樓|紅樓]]作為基地之一，策動反清革命<ref>{{cite web|url=http://www.681busterminal.com/59a.html|title=KMB Air-conditioned Route 九巴空調路線 - 59A|work=www.681busterminal.com}}</ref>。1970年代香港政府為發展[[屯門新市鎮|屯門新市鎮]]，將海角北面的沙灘填海以興建住宅區，只餘下南面部份，即[[蝴蝶灣|蝴蝶灣泳灘]]。

[[蝴蝶灣公園|蝴蝶灣公園]]中的「白角污水泵房」就是以「白角」命名。香港輕鐵亦曾設「白角站」，今改名[[輕鐵車廠站|輕鐵車廠站]]，而九龍巴士早期亦將來往蝴蝶灣的巴士路線，終點站命名為白角。
