'''白雲仙館'''，是[[廣州市|廣州市]]的一座[[道觀|道觀]]建築，位於廣州[[越秀区|越秀]]登峰[[麓湖公园|麓湖公園]][[麓湖路|麓湖路]]，

== 歷史 ==
其始建于清[[嘉慶|嘉慶]]十七年（1812），仙館初為廣州文人墨客雅集之地。後在內供奉[[八仙|八仙]]之一的[[呂洞賓|呂洞賓]]，並有道士主持管理<ref>天河區志·名勝古蹟</ref>。

1900年代初曾毀，1946年修復，時[[中山大学|中山大學]]校長[[鄒魯|鄒魯]]題有石刻<ref name=":0">{{Cite web|url=http://www.baiyunshan.com.cn/publicfiles/business/htmlfiles/byszz/bygj/201001/2617.html|title=白云古迹 白云仙馆|accessdate=|author=|date=|publisher=广州市白云山风景名胜区管理局}}</ref>。

[[文革|文革]]期間遭到徹底破壞，到1984～1985年間經市政府撥資金修復，作為觀光景點重開。後曾被外包做竹林餐厅。

2014年，广州市道教协会与广州市[[白云山_(广州)|白云山]]麓湖公园管理处签订移交管理协议，建築轉交道教打理。经过重修，於2016年1月18日恢復開放，當日舉行了神像开光法会。广州道教养生公益团白云仙馆养生基地同時在館設立<ref>{{Cite web|url=http://travel.people.com.cn/n1/2016/0119/c41570-28065382.html|title=道教白云仙馆恢复开放|accessdate=|author=广州日报|date=2016-01-19|publisher=}}</ref>。

== 位置 ==
所在地方舊日為上白雲山必經之路。直上是姑嫂墳、百步梯、能仁寺<ref name=":0" />。

== 建築 ==
仙館依山臨湖而建，結構玲瓏有致，極具仙山洞府韻味。山門有對聯'''香火千年祖庭瞻仰，雲山四面仙客棲遲'''<ref>廣州市志·社會·宗教</ref>。
