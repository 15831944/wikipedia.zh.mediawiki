'''白色合金'''或稱'''白色黃金'''、'''白色K金'''，是[[黄金|黄金]]和至少一种白色[[金属|金属]]，譬如[[银|银]]、[[钯|钯]]等，所混合而制成的[[合金|合金]]。和黄金一样，白色合金也是以[[克拉|克拉]]为纯度度量单位的。

== 特性 ==
白色合金的质量取决于其所包含的金属种类和比例。不同白色合金有不同的用途：[[镍|镍]]合金硬度高，适合做[[戒指|戒指]]、装饰扣；钯合金软而富于延展性，适合做宝石首饰的底座，如戒托。最高质量的白色合金是至少17K的金[[钯|钯]]合金，有时甚至还要加少许[[铂|铂]]以增加重量和耐磨损性，但这需要专业的金匠。

少数高质量的白色合金可以长期保持银光闪闪，大部分的白色合金则是通过在表面镀一层薄薄的[[铑|铑]]来做到这一点，因为大部分白色合金容易颜色发雾。通常白色合金制品需要清洗，有时还可以重新镀[[铑|铑]]。

== 和铂金的关系 ==
由於白色合金與貴金屬[[鉑|鉑]]外觀相似，有時會被不法商人用于冒充鉑。[[香港|香港]]政府規定这种黄金的白色合金要用「白色黃金」这个名称，以免與俗称“白金”的[[鉑|鉑]]混淆。

== 过敏 ==
大约有八分之一的人会对镍合金白金过敏，有时这种过敏是在女性[[怀孕|怀孕]]之后开始。对钯合金白金过敏的人很少。
