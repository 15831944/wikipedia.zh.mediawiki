'''白羊座'''（{{lang-la|'''Aries'''}}，[[天文符號|天文符號]]：♈）是[[黄道十二星座|黄道十二星座]]之一，位於[[雙鱼座|雙鱼座]]和[[金牛座|金牛座]]之間。面積441.39平方度，占全天面積的1.07%，在全天88個星座中，[[星座面積列表|面積排行]]第三十九。白羊座亮於5.5等的[[恒星|恒星]]有28顆，其中2等星1顆，3等星1顆。每年10月30日子夜白羊座的中心經過[[中天_(天文學)|上中天]]。白羊座雖然不引人注目，但在古希臘很著名，因为古代[[春分|春分]]點就位于白羊座。現在由於[[歲差|歲差]]的关系，春分典已經移到雙魚座。

白羊座在[[日本|日本]]稱為'''牡羊座'''，即「'''おひつじ座'''」，受其影響，在[[大中華地區|大中華地區]]也有用牡羊座這個占星學的名稱。

== 特征 ==
[[File:AriesCC.jpg|thumb|left|256px|牧羊座，因為它可以用肉眼看到（帶星座線繪製）。
[http://www.allthesky.com/constellations/aries.html AlltheSky.com]]]
白羊座的三颗主星α，β，γ组成钝角三角形结构，在没有[[光污染|光污染]]时非常容易辨认，但它的其他恒星都很黯淡，不易分辨。

== 白羊座神話 ==
國王阿塔瑪斯和妻子涅斐勒有一段不愉快的婚姻，於是另娶底比斯国王卡德摩斯之女伊諾為妻，可是伊諾虐待阿塔瑪斯的兩個孩子弗里克索斯、赫勒，又將城內的禾稈燒清。阿塔瑪斯請求神諭，然而伊諾從中作梗，令阿塔瑪斯以為必須犧牲兒子弗里克索斯方可消弭惡運。

阿塔瑪斯帶弗里克索斯到拉菲斯丹山頂，準備將他獻給宙斯。前妻涅斐勒差遣金羊克律索马罗斯救弗里克索斯，弗里克索斯逃脫並帶著雙胞胎妹妹赫勒騎上羊身飛向東面的科爾基斯，怎料赫勒一時大意脫手直墜海中。到了科爾基斯，弗里克索斯把金羊送給科爾基斯国王埃厄忒斯，埃厄忒斯也禮尚往來把女兒卡爾基奧佩賜予弗里克索斯。

後來，弗里克索斯父亲的兄弟克瑞透斯之子珀利阿斯奪去侄儿伊阿宋的皇位，弗里克索斯死後，鬼魂回到希臘纏著珀利阿斯，珀利阿斯無可奈何推搪若伊阿宋找到金羊毛，就會讓位。這引致後來伊阿宋乘阿格號之歷險。
[[File:Sidney_Hall_-_Urania's_Mirror_-_Aries_and_Musca_Borealis.jpg|thumb]]筆下的白羊座]]
伊阿宋千辛萬苦到達科爾基斯，問埃厄忒斯金羊毛之所在卻被拒諸門外，幸好埃厄忒斯之女美狄亚暗戀伊阿宋。入夜，美狄亚引走看守之巨蛇，伊阿宋則乘機偷走金羊毛，而後來金羊毛就被掛在宙斯之神殿前。

后来宙斯将金羊提升到天上，便成为了白羊星座。

== 恒星 ==
[[File:NGC772_SN2003hl_SN2003iq_6223_Dahl.jpg|thumb]].]]
* [[婁宿二|婁宿二]]（白羊座γ）：1664年[[英国|英国]][[科学家|科学家]][[罗伯特·胡克|罗伯特·胡克]]确认了它是一颗[[双星|双星]]。这是[[人类|人类]]确认的最早的双星之一。两颗子星都是白色的，亮度相同（[[星等|星等]]+4.5），相距7.8[[弧秒|弧秒]]。

* [[白羊座λ|白羊座λ]]：这也是一颗[[双星|双星]]，使用稳定的高倍[[双筒望远镜|双筒望远镜]]可以看见它。两颗子星亮度分别为+5和+7.5。

* [[白羊座SX|白羊座SX]]（白羊座56）：[[变星|白羊座SX型变星]]的代表星，亮度变化于5.67-5.81之间，变光周期为17小时28分9.9秒。

* [[蒂加登星|蒂加登星]]：2003年发现的[[红矮星|红矮星]]，距离地球12光年，是目前已知白羊座离地球最近的恒星。

== 中国星官 ==
中国古代传统将白羊座座天区划分为[[娄宿|娄宿]]、[[胃宿|胃宿]]、左更、天阴、天阿等[[星官|星官]]。
* 娄Bond（娄3）：白羊座β、γ2、α 
* 左更Official in Charge of the Forest（娄5）：白羊座ν、μ、ο、σ、π
* 胃Stomach（胃3）：白羊座35、39、41
* 天阴Celestial Yin Force（昴5）：白羊座63、65、ζ、τ、δ
* 天阿Celestial River（昴1）：白羊座62
