'''白人盎格魯-撒克遜新教徒'''（White Anglo-Saxon Protestant，WASP），或稱'''白人凱爾特與日耳曼新教徒'''（White Celtic and Germanic Protestant），本義是指[[美國|美國]]當權的[[精英|精英]]群體及其文化、習俗和道德行為標準，現在可以泛指信奉[[新教|新教]]的歐裔美國人。此群體擁有龐大的[[經濟|經濟]]、[[政治|政治]]勢力，構成美國上流社會和中上階層的絕大部分。儘管美國社會日益多元化，但他們的文化、道德觀和價值取向仍在很大程度上影響著美國的發展。

== 名詞起源 ==
歷史上，“盎格魯 - 撒克遜”在1066年之前英格蘭原住民[[凱爾特人|凱爾特人]]的語言就已經提及，特別是與之後法國的[[諾曼人|諾曼第人]]對英格蘭的影響完全不同。自十九世紀以來，它在英語世界中被普遍使用，但在英國本身卻不是指主要英國血統的新教徒。在二十世紀50年代增加了“W”和“P”，形成了一個詼諧的名詞，表示“ 輕率 ”(這意味著一個容易被激怒和迅速冒犯的人)。

1957年[[政治學家|政治學家]]安德魯·哈克(Andrew Hacker)提出了第一次公開提及WASP這一句[[術語|術語]]，這說明WASP已經被美國[[社會學家|社會學家]]用作通用術語，儘管“W”代表“Wealthy([[富有|富有]])”而不是“White([[白人|白人]])” -  他們是“WASPs”，使用在社會學家的雞尾酒會中的[[行話|行話]]。也就是說，他們很富有，他們是[[盎格魯撒克遜人|盎格魯撒克遜人]]，他們是新教徒(和不成比例的[[聖公會|聖公會]]教徒)。對於他們的定義取向，應該加上居住在[[美國東海岸|美國東海岸]]或[[舊金山|舊金山]]附近的居民，從小被預期安排讀昂貴的私立高中和[[常春藤聯盟|常春藤聯盟]]私立大學教育，並同時能擁有繼承豐富的財富。

用W代表富有而不是白人可能是正確的原始詞源，因為“白盎格魯 - 撒克遜”是多餘的，這個詞恰當地指的是精英個體，而不是英國人的每一個人。

這個詞在1964年出版的書“新教的建立：貴族和貴族的種姓”中被社會學家和[[賓夕法尼亞大學|賓夕法尼亞大學]]教授E. Digby Baltzell推廣，他本人是WASP 。巴爾澤爾強調了這個集團的封閉性或種姓特徵，他們爭辯說：“二十世紀中葉美國領導層出現危機，部分原因是由於現在企業權力的下降一個越來越種姓的白盎格魯撒克遜新教徒(WASP)上層階級”。

== 當代用法 ==
1964年迪畢·波茨爾（E. Digby Baltzell）在其書《新教當權者：美國的貴族和社會等級》中使用後，這個詞便逐漸在美國社會傳播開來（注意此詞是1962年另一位學者首創的）。WASP的原義是指美國的新教[[上流社會|上流社會]]，即殖民時代來自[[英國|英國]]（尤其是[[英格蘭|英格蘭]]和[[蘇格蘭|蘇格蘭]]）的移民，這些人屬於[[基督新教|基督新教]]中的[[長老教會|長老教會]]、[[公理會|公理會]]、[[信義宗|信義宗]]和[[美國聖公會|美國聖公會]]等教派。現在這個詞也出現在其他一些有過相同殖民歷史的[[英語|英語]]國家中，如[[澳大利亞|澳大利亞]]。

現在的WASP詞義比當初創建時涵蓋的內容要廣泛得多。今日所有講英語的[[歐洲|歐裔]][[新教徒|新教徒]]都能稱作WASP，即使不是[[盎格魯人|盎格魯人]]、[[撒克遜人|撒克遜人]]和相近族群的後裔也可以。'''不過[[猶太人|猶太人]]、[[天主教|天主教徒]]和[[東正教|東正教徒]]則不包括在內'''。

這個用法其實陳腐且不嚴謹，因為美國新教徒擁有錯綜複雜的血統，分散在很多教派中。新教徒的祖先可以來自[[英格蘭|英格蘭]]、[[蘇格蘭|蘇格蘭]]、[[威爾士|威爾士]]、[[康沃爾|康沃爾]]、[[愛爾蘭|愛爾蘭]]、[[德國|德國]]、[[荷蘭|荷蘭]]、[[丹麥|丹麥]]、[[斯堪的纳维亚半岛|斯堪的納維亞半島]]或法國的[[雨格諾派|雨格諾派]]；可以屬於[[美國聖公會|美國聖公會]]、[[長老會|長老會]]、[[信義宗|信義宗]]（路德宗）、[[衛理公會|衛理公會]]、[[公理會|公理會]]、[[荷蘭改革派|荷蘭改革派]]、[[貴格會|貴格會]]、[[浸信會|浸信會]]、[[福音派|福音派]]甚至[[摩門教|摩門教]]等等。不光是精英階層，各個社會階層都有他們的身影，即使被諷刺為「[[白色垃圾|白色垃圾]]」的下層貧困白人也可能是WASP。

美國東海岸WASP這個詞可以經常聽到，一般用來與最初的殖民者比較，和後來的歐洲移民，如[[愛爾蘭|愛爾蘭]]裔天主教徒、[[猶太人|猶太人]]、[[意大利|意大利]]裔和其他「少數白人民族」等。

美國南方則不同，因為[[南北戰爭|南北戰爭]]後遷至此地的移民相對很少，所以WASP這個詞也就不常聽到。[[路易斯安那州|路易斯安那州]]南方和[[佛羅里達州|佛羅里達州]]南部是例外，因為法國和[[拉美|拉美]]移民者佔大多數。除這些地方外，美國南方的白人則大部分都是[[不列顛群島|不列顛群島]]人的後裔。不過由於歷史矛盾等原因，居住在[[維吉尼亞州|維吉尼亞州]]、[[南卡羅萊那州|南]]、[[北卡羅萊那州|北卡羅萊那州]]海岸的英格蘭人後裔和居住在高地的蘇格蘭－愛爾蘭人後裔之間也在語言、文化上有很大不同。

美國西部則經常用「[[盎格魯人|Anglo]]」（盎格魯）這個詞形容非[[拉美裔人|拉美裔]]的歐裔英語使用者，所以這個詞也就比WASP範圍要更廣一些。英裔美國人和其他族群之間的區別並沒有像在東海岸那麼明顯。

== 起初的WASP ==
最早的WASP精英階層自從19世紀早期起便牢牢佔據著美國的上流社會。子女從私立中學直接升入[[常春藤盟校|常春藤盟校]]，學習上流社會的習俗、禮儀、舉止，與其他名門望族建立關係，掌控著美國的財經、文化和政治等重大領域。族內通婚防止遺產外流；[[馬球|馬球]]、[[遊艇|遊艇]]等是有錢、有閒階層的獨有娛樂方式。《社會名流錄》等手冊記錄著上流社會家族之間錯綜複雜的關係網。他們流連於相同的私人俱樂部，去相同的[[教堂|教堂]]，居住在一起（[[費城|費城]]的Main Line和[[波士頓|波士頓]]的Back Bay富人區是最明顯的兩個例子），一切都和暴發戶嚴格區分開來。隨時間推移，這些人也逐漸在美國中西部和西部定居，開始分散至全國。

而後來遷至美國的移民者並不富裕，和上流社會也沒什麼聯繫，所以在宗教、文化等各個方面與最開始的WASP相去甚遠。這些人成立了獨立的政治、經濟、學術機構團體來與上流社會抗衡，削弱了原來精英手中的權勢和財富。不過直到[[二戰|二戰]]後，老上流社會的聲勢才開始顯著下滑。《士兵權利法案》規定貧困移民的孩子也可以接受高等教育；公民權立法宣佈工作場所的各種歧視為非法。美國戰後1945年至1964年的繁榮期帶來大量的財富和機會，規模龐大的中產階級興起。儘管如此，舊有的WASP階層仍然在美國文化、政治和經濟界精英中擁有極大的勢力。

WASP族群在今天的美國社會依舊隨處可見。這些人依然處於中上階級以上，家境富庶，出沒於上流社會，子女仍然在私立中學和常春籐盟校學習；如今主要聚居在[[新英格蘭|新英格蘭]]和東北部地區。不過因為移民的關係，這些地區的人口現在是以天主教為主，再也不是WASP的主宰地帶，而且常春藤盟校也開始避免過分地招收WASP學生。在美國中西部地區，雖然主要居民是來自東北部、[[賓夕法尼亞州|賓夕法尼亞州]]、[[維吉尼亞州|維吉尼亞州]]等先鋒的後裔，而且大多數人信奉新教，但這個詞並不適用。在南方這個詞要常用一些，不過因為南方主要是[[福音派|福音派]]，和北方的WASP在[[教育|教育]]和文化價值觀上有分歧，所以這個詞在南方的含義和在東海岸並不一樣。總而言之，美國新教徒的主要聚居地在今日已經轉移到了中西部和南部，隨著這些地區勢力逐漸擴大，他們所代表的形象和文化風貌與原來東海岸的WASP已經大相逕庭。

按血統來劃分的話，WASP在美國仍然是多數族群（用宗教派別劃分並不一定準確，因為今日東海岸的很多人已經轉而相信「[[不可知論|不可知論]]」）。因此，WASP所代表的這個階級或人群通常地位不俗，有錢有勢，於是也成為美國其他種族反感、厭惡的對象。

== 蘊義和典型化形象 ==
此詞本身含義重復，因為盎格魯撒克遜人就是白人，White（白）是沒有必要的，不過，小寫的“wasp”又是“[[黃蜂|黃蜂]]”的意思，會給人一種討厭蜇人的感覺。曾有一段時間人們用“ASP”來替代，既省略了累贅的部分（W），又保留了和有毒生物的聯繫（“asp”是角蝰，一種[[蛇|毒蛇]]）。此詞經常用作貶義，用來打趣、嘲諷這類白人在歷史上盛行的[[種族主義|種族主義]]、[[排外主義|排外主義]]、[[反猶太主義|反猶太主義]]、反天主教和文化種族優越感心態。

有人說，在今日的美國，WASP群體是唯一能夠讓大家隨便嘲笑又不會激發嚴重抗議的族群。一說到WASP，人們通常就會聯想起一些典型的形象，如古板、貪婪、自私、節儉、勢利、冷淡、工作狂、無表情、高傲等等。另一方，有的美國人倒很樂意稱自己為WASP，不過總是帶有一種負罪感（另一個特點）。因為這個詞中的“白”是根據膚色來評判的，所以一般認為是[[種族歧視|種族歧視]]辭彙，不過迄今為止的用法相對其他一些[[歧視語|歧視語]]來說（如nigger，[[黑鬼|黑鬼]]）還是很溫和的。

從外觀上來講，這類人看起來像典型的“[[英格蘭人|英格蘭人]]”：身材瘦長，膚色蒼白裡透著紅斑，金髮碧眼。總喝酒所以臉色通常是甜菜紅；由於基因等生理原因，皮膚無法曬成古銅色，最多是曬傷紅：這兩點加在一起產生了另一個貶義詞“[[redneck|redneck]]”（直譯：[[紅脖子|紅脖子]]），用來形容在戶外辛苦工作被曬得通紅的白人，而這種白人一般都是下層社會的工人或農民，所以“redneck”就是“鄉巴佬、沒文化的下等白人”等意思。

== 有關WASP的笑話 ==
WASP是美國的主流族群，別的種族要經過他們的文化道德標準來評判，因為經濟政治實力存在巨大差異，很自然會被他們視為劣等，所以有關這類人的笑話一般都有嘲諷的情緒在內，並不一定是因為WASP本身有多逗樂，這一點正好和他們“無趣、冷漠、沒有人情味”的典型形象呼應。

* 什麽情況下屋子裏會聚集4個以上的WASP？ 聚餐時。（諷刺愛獨處、愛安靜、不活潑）
* WASP媽媽晚飯會做什麽？ 湯、沙拉、麵包、主菜、蔬菜和甜點。（諷刺死板、一成不變）
* 換個燈泡需要多少個WASP？ 三個：兩個人調酒，一個人去叫電工。（諷刺愛喝酒，厭惡體力勞動）

== 近年 ==
近年來美國保守和[[宗教|宗教]]勢力一直在没落，很多WASP唯有從[[基督教基要主義|基督教基要主義]]出發，反對[[墮胎|墮胎]]、[[同性婚姻|同性婚姻]]甚至[[同性戀|同性戀]]等为世界普遍支持的事務，希望能夠“恢復美國式的生活”。這些[[美國人|美國人]]是[[共和黨_(美國)|共和黨]]的重要支持者，共和黨也一直將實現他們的價值觀作為自己的目標之一。
