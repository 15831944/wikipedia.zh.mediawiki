'''白鹤美术馆'''（{{lang-ja|白鶴美術館}}、{{lang-en|Hakutsuru Fine Art Museum}}）是一所于1934年建在[[日本|日本]][[兵库县|兵库县]][[神户市|神户市]][[东滩区|东滩区]]的[[美术博物馆|美术博物馆]]。该美术馆展示了嘉纳财阀的白嘉纳家（白鹤酒造创业家）第7代当家嘉纳治兵卫的收藏品，是日本少数的在[[第二次世界大战|第二次世界大战]]之前就已开幕的美术博物馆之一<ref name="Kobe">{{cite web |url=http://www.city.kobe.lg.jp/ward/kuyakusho/higashinada/foreign/guidemap/guidemap02.html |title=Hakutsuru Fine Art Museum |publisher=[[Kobe|Kobe City]] |accessdate=2018-11-13}}</ref>。白鹤博物馆藏品约1450件，其中包括2件国宝和22件重要文化财产<ref>{{cite web |url=http://www.hakutsuru-museum.org/rekishi01.shtml |script-title=ja:歴史 |trans-title=History |language=ja |publisher=Hakutsuru Fine Art Museum |accessdate=2018-11-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20120226195304/http://www.hakutsuru-museum.org/rekishi01.shtml |archivedate=2012-02-26 |df= }}</ref>。

== 藏品 ==
[[File:Sutra_of_the_Wise_and_Foolish_Hakutsuru.jpg|thumb]]
白鹤美术馆现有藏品约1450件。其中包括[[中国青铜器|中国青铜器]]<ref>{{cite web |url=http://www.hakutsuru-museum.org/hon_ameiho01.shtml |script-title=ja:青銅器・金工品 |trans-title=Bronzes - Metalwork |language=ja |publisher=Hakutsuru Fine Art Museum |accessdate=2018-11-13}}</ref>、[[中國陶瓷史|中國陶瓷]]<ref>{{cite web |url=http://www.hakutsuru-museum.org/hon_bmeiho01.shtml |script-title=ja:陶磁器・漆器 |trans-title=Ceramics - Lacquerware |language=ja |publisher=Hakutsuru Fine Art Museum |accessdate=2018-11-13}}</ref>、银器、镜子和珠宝<ref>{{cite web |url=http://www.hakutsuru-museum.org/hon_cmeiho01.shtml |script-title=ja:銀器・鏡・装身具 |trans-title=Silverware - Mirrors - Jewelry |language=ja |publisher=Hakutsuru Fine Art Museum |accessdate=2018-11-13}}</ref>、书法作品<ref>{{cite web |url=http://www.hakutsuru-museum.org/hon_dmeiho01.shtml |script-title=ja:経巻・書・古代裂・荘厳具具 |trans-title=Writings |language=ja |publisher=Hakutsuru Fine Art Museum |accessdate=2018-11-13}}</ref>以及画作<ref>{{cite web |url=http://www.hakutsuru-museum.org/hon_emeiho01.shtml |script-title=ja:絵画 |trans-title=Paintings |language=ja |publisher=Hakutsuru Fine Art Museum |accessdate=2018-11-13}}</ref>。

白鹤美术馆目前藏有2件[[日本国宝|国宝]]。其中一件为[[奈良时代|奈良时代]]的出于[[圣武天皇|圣武天皇]]之手的《贤愚经残卷》，另一件为[[大般涅槃经_(大乘)|大般涅槃经]]相关的《大般涅盘经集解》<ref>{{cite web |url=http://kunishitei.bunka.go.jp/bsys/maindetails.asp?register_id=201&item_id=763  |script-title=ja:賢愚経残巻（大聖武）|trans-title=Sutra of the Wise and Foolish |language=ja  |publisher=[[文化厅|文化厅]] |accessdate=2018-11-13}}</ref><ref>{{cite web |url=http://kunishitei.bunka.go.jp/bsys/maindetails.asp?register_id=201&item_id=764 |script-title=ja:大般涅槃経集解 |trans-title=Instruction manual on the Nirvana Sutra |language=ja  |publisher=[[文化厅|文化厅]] |accessdate=2018-11-13}}</ref>。

== 建筑 ==
白鹤美术馆本馆是一座铜瓦屋檐的[[钢筋混凝土|钢筋混凝土]]建筑<ref>{{cite web |url=http://kunishitei.bunka.go.jp/bsys/maindetails.asp?register_id=101&item_id=00001442 |script-title=ja:白鶴美術館本館 |trans-title=Hakutsuru Fine Art Museum - Honkan |language=ja  |publisher=[[文化厅|文化厅]] |accessdate=2018-11-13}}</ref>。本馆于1931年开始建设，1934年正式对外开放。其附属的饮茶室比本馆建设时间更早，于1929年就已建成<ref>{{cite web |url=http://kunishitei.bunka.go.jp/bsys/maindetails.asp?register_id=101&item_id=00001445 |script-title=ja:白鶴美術館茶室（崧庵） |trans-title=Hakutsuru Fine Art Museum - Tea House |language=ja  |publisher=[[文化厅|文化厅]] |accessdate=2018-11-13}}</ref>。1995年，白鹤美术馆附属的地毯美术馆开幕<ref name="Kobe"/>。
