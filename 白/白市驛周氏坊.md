'''白市驛周氏坊'''，全稱“旌表歲進士董經之妻周氏坊”，位於[[重慶市|重慶市]][[九龍坡區|九龍坡區]][[白市驛鎮|白市驛鎮]]（舊屬[[巴縣|巴縣]]）北500米（106°21'58.41"东、 29°29'37.11"北），修建於乾隆二十年(1755年)，是一座[[貞節牌坊|節孝牌坊]]，1986年登录为[[巴縣文物保護單位|巴縣文物保護單位]]。1995年3月白市驛鎮改归九龍坡區管辖。2011年，周氏坊重新登录为为[[九龙坡区文物保护单位|九龙坡区文物保护单位]]之一。<ref>{{cite web|work=九龙坡府办发〔2011〕206号|title=关于公布第四批区级文物保护单位及撤销原区（县）级文物保护单位的通知|url=http://zfxx.cq.gov.cn/zfxxgk/jsp/view/infoview.jsp?xxbid=58503|accessdate=2013-01-02|publisher=重庆市九龙坡区人民政府办公室|date=2011-08-13|deadurl=yes|archiveurl=https://archive.is/20130423025323/http://zfxx.cq.gov.cn/zfxxgk/jsp/view/infoview.jsp?xxbid=58503|archivedate=2013-04-23}}</ref> 

旌表牌坊是中國社會中特有的表揚性建築，藉以教化人心，多建在舊時的官道。白市驛周氏坊亦為其中之一，位於重慶到成都的古道上。
[[File:重慶白市驛周氏坊2.JPG|thumb]]

== 牌坊介紹<ref>《重慶文物總目》，巴縣，第26頁</ref> ==
石質仿木結均.重檐山頂· 頂中有寶珠，脊端有吻獸，檐下施斗栱五朵，次間各施斗栱兩朵。坊身、斗栱及部分額坊上有淺浮雕圖案·額坊上有『帝隆孝德』、「天賜慈齡」等文字，落款為"大清乾隆二十年(1755年)仲春月吉旦"。坊柱呈八棱形，柱徑0.6米，柱腳有圓雕夾杆石。坊為四柱三間，面闊7.4米，通高7.5米。

== 牌坊文字 ==
* 牌坊向東面額坊：中書“帝隆孝德”，向西面額坊：中書“天賜慈齡”
* 額坊右均為“大清乾隆十六年季春月，欽奉太子太保吏部尚書兼都察院又都御史總督四川等處地方提督軍務兼理糧餉管巡撫事世襲雲騎加一級紀錄拾三次[[黃廷桂|黃廷桂]]”
* 額坊左均為“四川承宣布政使司布政使紀錄二次[[覺羅齊格|覺羅齊格]]、四川提刑按察使司按察使總理銅政軍功加一級紀錄二十四次[[周琬|周琬]]、大清乾隆二十年仲春月吉旦”
* 額坊下為：“旌表歲進士董經之妻周氏坊”
* 次間右為：“四川東道[[宋邦綏|宋邦綏]]，知府[[徐正恩|徐正恩]]、[[傅顯|傅顯]]，知縣[[王爾鑑|王爾鑑]]、[[張兌和|張兌和]]、[[袁錫夔|袁錫夔]]”
* 次間左為：“分縣左堂金藩，教諭劉瑄，訓導楊文言”

== 坊主 ==
白市驛周氏坊的坊主貢生董經之妻周氏，在同治《巴縣志》中有載：「董節婦，周氏，貢生董經妻，歸八載經歿，三子俱幼，家貧姑老，女紅度日，摒擋一門，老幼婚喪，苦節三十六年，旌表入祠」。<ref>同治《巴縣志》卷三之下人物志節烈孝貞，第4頁</ref>
