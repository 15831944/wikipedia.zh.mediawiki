'''白桑'''（{{lang|la|'''''Morus alba'''''}}）是一種壽命短、生長快速的[[桑樹|桑樹]]，屬於小型到中型的[[落葉|落葉]][[喬木|喬木]]，高度約10–20 米。白桑原生於[[中國|中國]]北部，並廣於世界各地種植<ref name=china>{{cite web |publisher=Flora of China |url=http://www.efloras.org/florataxon.aspx?flora_id=2&taxon_id=200006379 |title=''Morus alba'' |language=en}}</ref><ref name=fao>{{cite web |last=Suttie |first=J. M. |date=(undated) |publisher=FAO Report |url=http://www.fao.org/ag/agp/AGPC/doc/Gbase/DATA/Pf000542.htm |title=''Morus alba'' L. |language=en}}</ref>。由於白桑是养[[蚕|蚕]]业的重要饲料，很早就在中国培育，果实称为[[桑椹|桑椹]]，初生[[白色|白色]]，成熟后成为紫黑色。根皮（桑白皮）、桑椹、桑叶都可以作为[[中药|中药]]使用。此外桑椹可以酿[[酒|酒]]，树皮可以造纸，造出的纸称为“桑皮纸”。

== 概述 ==
白桑幼苗的葉可長達30 cm，會變異產生裂紋，裂紋呈圓形，而且深入、錯綜複雜。

== 用途 ==
[[File:Mulberry.jpg|left]]
白桑的嫩葉可用於餵養[[絲蠶|絲蠶]]，以便生產高品質的絲<ref name=eCosFAQ>{{cite web |url=http://www.ecosopp.com/product/zh_CN/3444/faq.html |title=Nn 液状叶绿素：常見問題 |accessdate=2011-08-05 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20090822194943/http://www.ecosopp.com/product/zh_CN/3444/faq.html |archivedate=2009-08-22 }}</ref>；而在氣候較乾燥的地方，當地面的植被未能為牲口提供食糧時，白桑葉亦會用於餵飼牲口。白桑的果實叫做[[桑椹|桑椹]]，可吃；亦有用來製作[[乾果|乾果]]或[[釀酒|釀酒]]<ref name="fao"/><!--<ref name="bean"/>-->。

在傳統中醫藥，桑椹可用於治療過早出現的白髮，亦可淨化血液，解便秘，治[[消渴|消渴]]，去[[水腫|水腫]]。對於其淨化血液的功能，主要在於其嫩葉的葉綠素含量豐富<ref name=eCosMain>{{cite web |url=http://www.ecosopp.com/product/zh_CN/3444/main.html#content |title=Nn 液状叶绿素：介紹 |accessdate=2011-08-05 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20120203220435/http://www.ecosopp.com/product/zh_CN/3444/main.html#content |archivedate=2012-02-03 }}</ref>，而葉綠素可被身體吸收，其與血紅素近似的化學結構可為身體輸送氧氣及帶走血液裡的廢物，加速體內的淨化<ref name=eCosBkgrnd>>{{cite web |url=http://www.ecosopp.com/product/zh_CN/3444/background.html |title=Nn 液状叶绿素：背景 |accessdate=2011-08-05 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20131217142415/http://www.ecosopp.com/product/zh_CN/3444/background.html |archivedate=2013-12-17 }}</ref>，且不會對人體帶來副作用白桑的樹皮可止咳、利尿，亦可退燒、止頭痛、眼紅及眼乾。

白桑椹的生長周期短，而且其釋放[[花粉|花粉]]的速度也非常快，比音速的一半還要快<ref name="Taylor et al">{{Cite journal

== 药理性能 ==

由於白桑是中國最常見的桑樹品種，藥書裡提及的桑樹，一般都專指白桑。以下為白桑各部份的藥理性能：
* 桑葉: 味苦甘，性寒，疏解風熱，清肝明目。
* 桑椹: 味甘酸，性平，滋陰補血。
* 樹枝: 味苦，性平，祛風通絡，解熱，鎮痛。
* 樹根: 味甘，性寒，止咳平喘，利水消腫。
