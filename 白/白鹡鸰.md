'''白鹡鸰'''（[[学名|学名]]：{{lang|la|''Motacilla alba''}}）是一种小型[[鹡鸰|鹡鸰]]，属于[[鹡鸰科|鹡鸰科]]。

== 生态环境 ==
喜滨水活动，多在河溪边、湖沼、水渠等处，在离水较近的耕地附近、草地、荒坡、路边等处也可见到。

== 分布地域 ==
主要分布在[[欧亚大陆|欧亚大陆]]的大部分地区和[[非洲|非洲]]北部的[[阿拉伯|阿拉伯]]地区，在[[中国|中国]]有广泛分布。

== 特征 ==
體長16.5-19cm，並非直線飛行，而是一上一下地飛行，行走時尾巴會不斷上下擺動。

== 食物 ==
以[[昆蟲|昆蟲]]及[[種子|種子]]为食。
