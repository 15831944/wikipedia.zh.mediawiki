'''若泽·罗贝托·伽马·德·奥利维拉'''（'''{{lang-pt|José Roberto Gama de Oliveira}}'''），绰号'''贝贝托'''（'''{{lang-pt|Bebeto}}'''），{{bd|1964年|2月16日|catIdx=Bebeto}}），前巴西足球國家隊成員，已退役，巴西足球傳奇巨星之一。當年身穿國家隊7號球衣，與身穿11號球衣的[[羅馬里奧|羅馬里奧]]合稱「夢幻組合」，為巴西奪取1994世界盃冠軍。他参加過[[1988年夏季奥林匹克运动会|1988年漢城奧運]]、[[1990年世界盃足球賽|1990年義大利世界盃]]、[[1994年世界盃足球賽|1994年美國世界盃]]（冠軍隊成員）、[[1996年夏季奥林匹克运动会|1996年阿特蘭大奧運]]及[[1998年世界盃足球賽|法國世界盃]]。白必圖在他職業球員生涯上從沒有射失過一球十二碼。

== 球會生涯 ==
贝贝托（Bebeto）原名若泽·罗贝托·伽马·德·奥利维拉，1964年2月16日生于巴西东北部小城巴亚。1979年，年仅15岁的贝贝托以当时巴西最高的转会费转到[[弗拉门戈体育俱乐部|弗拉门戈俱乐部足球队]]，与巴西著名球星[[薛高|薛高]]成为队友。1983年，19岁的贝贝托在巴西维多利亚开始足球生涯。1985年由于[[薛高|薛高]]的离去，贝贝托成为弗拉门戈俱乐部足球队主力。在巴西国内，贝贝托先后效力弗拉门戈、華斯高等队。

1992年，贝贝托登陆[[伊比利亚|伊比利亚]]半岛加盟了[[拉科鲁尼亚体育俱乐部|拉科鲁尼亚]]，并在那里效力4个赛季。在拉科鲁尼亚效力期间，贝贝托曾以29球在1992/93赛季夺取西甲最佳射手，不过，他的最大遗憾是1993/94赛季，拉科鲁尼亚在最后一轮只要取胜就可夺取联赛冠军，但久基奇罚丢点球，拉科鲁尼亚无奈战平[[華倫西亞足球會|華伦西亚]]，拉科鲁尼亚也将冠军拱手送给了[[巴萨|巴萨]]。1995年，与队友合作，使拉科鲁尼亚队获得[[西班牙国王杯|西班牙国王杯]]和[[西班牙超级杯|西班牙超级杯]]。

1996年，贝贝托回到巴西加盟了[[弗拉门戈|弗拉门戈]]，从此开始了漂泊的生活。1997年一年时间里，贝贝托先后在西班牙[[塞维利亚|塞维利亚]]、巴西[[克鲁塞罗|克鲁塞罗]]和巴西维多利亚效力过。在其职业生涯后期，贝贝托还效力过巴西[[博塔弗戈|博塔弗戈]]、[[墨西哥|墨西哥]]托罗斯内萨、日本[[鹿岛鹿角|鹿岛鹿角]]等队，最终在奔赴沙特[[伊蒂哈德足球俱樂部|伊蒂哈德]]短暂淘金后，于2002年正式退役。

== 国家队生涯 ==
1983年，贝贝托入选巴西国青队参加了墨西哥世青赛。在该届世青赛上，巴西获得了最后的冠军。尽管贝贝托在决赛中只替补登场了不到10分钟，但由于其在场上表现出的巨大潜质，他还是与[[雲巴士頓|雲巴士頓]]、[[奧利·普羅塔索夫|普罗塔索夫]]等一起被[[国际足联|国际足联]]技术和统计委员会评为当届世青赛“六大希望之星”。

1985年4月28日[[巴西國家足球隊|巴西]]敗給[[秘魯國家足球隊|秘魯]]0-1，白必圖為巴西國家隊第一次上陣。

1988年，入选巴西奥林匹克足球队，与队友合作，为巴西队获得汉城奥运会足球亚军。

1989年，參加了美洲國家盃，5月10日，巴西勝秘魯4-1賽事中，白必圖連中三元。最終巴西队获得當屆美洲杯足球赛冠军。


1994年世界杯是贝贝托职业生涯最为辉煌的经历。在1994年世界杯上，贝贝托打进了3个进球。當年身穿國家隊7號球衣，與身穿11號球衣的[[羅馬里奧|羅馬里奧]]合組了「夢幻組合」，他与罗马里奥之间的默契配合让人称赞。贝贝托的3个进球有2个来自[[罗马里奥|罗马里奥]]的助攻，而贝贝托同样给罗马里奥也回报了2次助攻。他与罗马里奥组成的前锋线，是森巴军团能够夺冠的最大保证。1994世界盃八強戰，巴西對[[荷蘭國家足球隊|荷蘭]]前數晚，白必圖妻子為他生了第三個孩子。白必圖在比賽中射入一球，隨即跑到邊線以抱孩子動作慶祝入球，成為經典慶祝入球動作。最終巴西奪取了1994世界盃冠軍。


1996奥運會白必圖夥拍[[朗拿度|朗拿度]]的巴西，因攻擊潛力太強猛，成為它國家隊的憂患，因而散播兩人不和的傳言（事後二人再拍擋是1998年法國世界盃）。巴西隊在1996奥運會四強戰在2-0領先情況下以戰敗。憤怒的白必圖在季軍戰毫不留情地對葡萄牙大演帽子戲法。巴西隊亦擊敗由鋼門[[拜亞|拜亞]]所率領的葡萄牙5-1。


1998年，贝贝托第3次随巴西参加了世界杯，这一次他再次攻入3球。只可惜，巴西在决赛中输给了东道主[[法国|法国]]，未能卫冕冠军。世界杯结束后，贝贝托退出了国家队。在他为巴西参加的 75场比赛中，贝贝托一共打进了39个球。

== 技術特點 ==
射门巧妙，走位飘忽不定，令人难以捕捉。他的入球大多是用脚踢进球门的，很少用头球破网。

==退休後工作==
白必圖在2011年當選[[里約熱內盧州|里約熱內盧州]]議員。2012年2月16日，[[2014年世界盃足球賽|2014年巴西世界盃足球賽]]組委會宣佈，白必圖應邀加盟，成為組委會董事會一員。<ref>{{cite web|url=http://soccer.sina.com.hk/news/95/2/1/2197818/1.html|title=受邀加盟巴西世界盃組委會 白必圖：全力做好工作|date=2012年2月17日|publisher=新浪香港}}</ref>

==家庭==
[[1994年美國世界盃|1994年美國世界盃]]，白必圖在八強賽中對[[荷蘭國家足球隊|荷蘭]]入球後，做出的抱兒子慶祝動作成為當屆賽事的經典場面，如今那個叫[[Mattheus_Oliveira|Matheus]]的嬰兒已經長大成人，並且曾入選巴西U20國家隊。目前（2015年）Matheus跟父親白必圖一樣，效力於巴甲球隊[[法林明高|法林明高]]。<ref>{{cite web|url=http://hk.sports.yahoo.com/news/%E7%99%BD%E5%BF%85%E5%9C%96%E9%98%BF%E4%BB%94%E5%85%A5%E9%81%B8u20-030821163.html|title=白必圖兒子入選U20|date=2012年11月15日|publisher=now.com 體育}}</ref>
