'''白胸翡翠'''（学名''Halcyon smyrnensis''）又名'''-{zh-cn:苍翡翠;zh-tw:白胸翡翠;}-'''，，也叫'''白喉翡翠'''，是一种分布在[[亚洲|亚洲]]南部和西部的[[翡翠|翡翠]]。

== 生态环境 ==
捕食于旷野、河流、池塘及海边。

== 分布地域 ==
[[中东|中东]]、[[印度|印度]]、[[中国|中国]]南部、[[东南亚|东南亚]]、[[菲律宾|菲律宾]]、[[安达曼群岛|安达曼群岛]]及[[苏门答腊|苏门答腊]]。

== 特征 ==
体长28cm，颏、喉及胸部白色；[[头|头]]、[[颈|颈]]及下体余部褐色；上背、翼及尾蓝色鲜亮如闪光；翼上覆羽上部及翼端黑色。 [[虹膜|虹膜]]－深褐色；[[喙|喙]]－深红；[[脚|脚]]－红色。 叫声－响亮而清脆的笑声。

== 食物 ==
[[蟹|蟹]]、[[昆蟲|昆蟲]]及[[蜥蜴|蜥蜴]]。

== 繁殖 ==
於軟石面掘洞為巢，多離水道甚遠，繁殖期為4-6月。
