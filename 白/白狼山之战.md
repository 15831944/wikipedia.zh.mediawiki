'''白狼山之战'''是[[中国|中国]][[东汉|东汉]]末年207年的一场战役，发生于今天的中国东北地区。交战双方为军阀[[曹操|曹操]]的军队和与曹操敌人[[袁尚|袁尚]]、[[袁熙|袁熙]]结盟的游牧[[乌桓|乌桓]]部落。战役的结果曹操方面取胜，乌桓的领地被摧毁，之后乌桓的势力被逐渐削弱，该部落的人口融入[[汉人|汉人]]或[[鲜卑|鲜卑]]部落，最终淡出历史舞台。曹營隨軍謀士[[郭嘉|郭嘉]]因水土不服身染重疾在烏桓征討結束後客死遼東。

== 背景 ==
190年代，北方军阀[[袁绍|袁绍]]在对他与乌桓的共同的敌人军阀[[公孙瓒|公孙瓒]]作战时与边境[[乌桓|乌桓]]部落交好。公孙瓒在[[易京之战|易京之战]]中覆亡后，袁绍把印信给乌桓盟军首领们，封他们为[[单于|单于]]，并将宗人的女儿作为己女嫁给乌桓头领，巩固了联盟。乌桓[[辽西单于|辽西单于]][[蹋顿|蹋顿]]尤其强大，因此得袁绍厚待。<ref name="TEP,J11J">de Crespigny (1996), Jian'an 11: J</ref>

袁绍在[[官渡之战|官渡之战]]中败于[[曹操|曹操]]后不久死去，乌桓继续支持袁氏。后袁绍长子[[袁谭|袁谭]]和少子[[袁尚|袁尚]]为争夺继承权而互斗，曹操从中取利。不久袁谭在与曹操[[南皮之戰|作战]]时被杀，袁尚投奔[[幽州|幽州]]的二哥[[袁熙|袁熙]]。袁氏兄弟在[[幽州|幽州]]又遭变兵攻打，北逃乌桓求援。此时乌桓在蹋顿领导下联合，蹋顿的强大势力使人传言他要仿效[[匈奴|匈奴]][[冒顿单于|冒顿单于]]和鲜卑[[檀石槐|檀石槐]]在北方游牧部落中建立霸权。袁氏兄弟及大量追随者到来后，蹋顿得以指挥乌桓和汉人联军多达30万人。<ref>de Crespigny (1984), p. 407</ref>

乌桓以为袁尚收回故地为由，数次袭击中国边境，劫走中国百姓十万户。<ref name="TEP,J11J"/>曹操注意到[[华北|华北]]有此危机，盘算着根除乌桓的威胁。

== 计划和备战 ==
曹操准备远征华北，命[[董昭|董昭]]于206年秋至207年春挖掘[[平虏渠|平虏渠]]和[[泉州渠|泉州渠]]以从水路运粮到华北。<ref>两条渠的具体路线不详，造成一些彼此冲突的詮釋，眾說紛紜。详见de Crespigny (1984), pp. 552-555, note 100.</ref>曹操做这些战争预备工作时，一些将领试图劝阻曹操深入北伐以防南方袭击：“袁尚已成流寇，夷狄贪婪无亲，岂能为袁尚所用！一旦我们深入征伐之，[[刘备|刘备]]必定说服[[刘表|刘表]]袭击[[许昌市|许都]]。若此事发生，后悔都没用了。”<ref name="TEP,J12B">de Crespigny (1996), Jian'an 12: B</ref>但曹操的谋士[[郭嘉|郭嘉]]作了如下分析，敦促采取急袭：“北方游牧人仗着中原离他们远，肯定不防备，趁机猝然攻打他们，可以击破和消灭之。一旦北方未定而南征，袁氏兄弟和乌桓会为患，新安定的北方各州也将丢失。荆州牧刘表不会被刘备说服袭击许都的，他除了坐谈以外无所作为，知道自己的才能不足以驾驭刘备，并不信任，若予以重任则担心不能制住刘备，如仅予以轻任，则刘备不愿为之所用。”<ref name="TEP,J12B" />

曹操赞同郭嘉的分析，207年夏，亲率军至[[易京|易京]]并以易京为大本营，进军[[无终|无终]]，当地向导[[田畴|田畴]]来投。

== 间接交锋 ==
乌桓军的大本营在[[柳城_(三国)|柳城]]，由无终发起的攻击线穿过[[渤海|渤海]]沿岸的平原地带。但七月，雨季降临，大雨在低地泛滥，道路泥泞，使得这片地区不能通过了。乌桓也料到了这条进攻线路，扼住河口，曹操军一时不能前进。

曹操就战况求助于熟悉当地且早先和乌桓有过争执的田畴。田畴告诉曹操一条通往[[西汉|西汉]]废弃的前沿领地的废弃道路。从那里军队可以行经无防守的地带，在乌桓最想不到的地方予以攻击，若如此做，“一仗也不用打就能得到蹋顿的首级”。<ref name="TEP,J12D">de Crespigny (1996), Jian'an 12: D</ref>曹操大喜过望，采纳计划，回师无终，命部下在水边的路上树立标志，写上：“中夏时节，道路不通。我们等秋冬了再进军。”敌军哨兵看到标志，相信曹操真的撤军了。如田畴所料，乌桓人大失算，以为任何不能穿越平原地带的军队都会撤回。<ref name="TEP,J12D" />

郭嘉建议“兵贵神速”，要曹操丢下辎重，迫使军队轻装前进，以免拖后腿和警醒敌军。<ref name="TEP,J12B" />曹操谨记在心。在田畴带路下，曹操率军轻装创造了被认为其戎马生涯中最值得纪念的军事冒险之一。<ref name="NC410">de Crespigny (1984), p. 410</ref>他们攀越徐无山，经卢龙塞退出中国边境进入[[滦河|滦河]]上谷，在这片艰难的地区进军五百余里，在[[平冈|平冈]]转而东向，穿越鲜卑的牧地，再次进入东汉边境的山脉。这时曹操军已经包抄了蹋顿的防守营寨，正沿到海的方向直线前进，即将把敌军的领地一分为二。<ref name="NC410" />

== 战役 ==
秋季八月，曹操军到[[大凌河|大凌河]]谷，蹋顿及其盟军意识到状况，仓促从备战地撤退，合兵向北迎战曹操。蹋顿、袁尚、袁熙和另两位乌桓单于楼班、乌延聚集数万人<ref>《三国志》提到右北平单于[[能臣抵之|能臣抵之]]也是此战的统帅之一。但当时的右北平单于是乌延，能臣抵之只是边远的[[代郡|代郡]]的一位乌桓酋长。《资治通鉴》认为乌延和能臣抵之可能为一人，但更可能是三国志误记。见de Crespigny (1996), Jian'an 12: E, note 11.</ref>在白狼山遭遇曹操军。

这场遭遇对双方都是猝然的。敌军数量占优，辎重又被留在后方，曹军害怕了。另一方面，乌桓显然也无任何作战准备，并未成列，更没有试图阻止曹操进军以抢占先机。<ref name="N411">de Crespigny (1984), p. 411</ref>曹操及诸将登上一处斜坡观察敌军队列，发现乌桓军无序，在张辽的请战下，曹军立即挥兵冲锋，兵分三路，徐晃、于禁、许褚各领一路，曹操將自己的麾旗授予张辽作中军主力直冲蹋顿所部；张辽冲锋进攻，轻兵很快击败乌桓骑兵，曹纯部[[虎豹骑|虎豹骑]]俘获蹋顿<ref> 陈寿，《三國志》，卷09，〈诸夏侯曹传·曹仁附曹純傳〉</ref>，蹋顿当阵为张辽斩杀。<ref>陈寿，《三国志》，卷17，〈张乐于张徐传〉</ref>战役结束後20多万中国居民和乌桓部卒投降曹操。<ref>de Crespigny (1996), Jian'an 12: E</ref>

== 后果 ==
此战是决定性的——曹操仅用这一场遭遇战就摧毁了乌桓抵抗的基础。<ref name="N411"/>袁氏兄弟和幸存的乌桓头领[[苏仆延|苏仆延]]、[[楼班|楼班]]、[[乌延|乌延]]等东逃军阀[[公孙康|公孙康]]治下的[[辽东|辽东]]；由於辽东远离京城，加上當時中国混乱，公孙康实质一定程度上独立于汉廷。九月，曹操继续进军柳城，暂歇，不顾下属敦促，作无意攻辽东状。他说：“我只要安排公孙康砍下袁尚和袁熙的首级送给我。不需要劳烦士卒。”后来他解释说，施压辽东可能让袁尚和公孙康联手对抗他，但若放任他们自行处理，他们将会内斗。<ref>de Crespigny (1996), Jian'an 12: F</ref>

袁尚、袁熙到辽东后数日，公孙康去看他们。袁尚计划杀公孙康，夺其地以为袁氏所有，但公孙康先发制人杀了兄弟俩並献首曹操，跟随袁氏的乌桓头领也未能幸免，皆遭斩首。

此后，曹操军因早冬寒冷、食物短缺、干旱而受损，曹操从柳城艰难撤回[[华北平原|华北平原]]。提出了极大程度上帮助曹操的建议的郭嘉和前军阀[[张繡|张繡]]都在此役中过世。归途中，曹操褒赏了先前反对他攻打乌桓的谋士们，承认此举太过冒险，幸蒙天佑。<ref>de Crespigny (1984), p. 413</ref>雖然刘备曾试图说服刘表趁曹操远在北方時予以攻击，但正如郭嘉所料，刘表并未把握此次机遇，后悔不迭。<ref>de Crespigny (1996), Jian'an 12: L</ref>

十一月，在[[易水|易水]]河岸，没有参与蹋顿的抵抗的乌桓头领[[难楼|难楼]]和[[普富卢|普富卢]]前來向曹操致贺並正式归顺。<ref name="N412">de Crespigny (1984), p. 412</ref>投降的乌桓人被集体迁到[[中国本土|中国本土]]，他们的骑兵成为中国突出的部隊。<ref name="N415">de Crespigny (1984), p. 415</ref>在曹操的餘年里，乌桓再未对他的领地造成危险，<ref name="N412" />尽管在216年和218年爆发了小规模叛乱。<ref name="N415" />

蹋顿和袁氏兄弟死后，曹操对华北的控制无可撼动。在华北的成功给曹操树立了巨大的威信，他在208年成为[[丞相|丞相]]。北疆安定，也使他在同一年能够集中军事力量对付南方的刘表。<ref>de Crespigny (2007), p. 37</ref>

另一方面，白狼山之战打垮了乌桓部落，餘部在[[中國東北|東北]]、[[高句丽|高句丽]]、辽东西界避难。数年后，这些乌桓餘部最终被附近的强大势力所吸收，一些被中原漢族同化，大部分被融入鲜卑。乌桓的衰落由白狼山之战开始，最终摧毁了乌桓本身及其维持政治独立的能力，使乌桓在很大程度上不再被视为华北地区的一个独立族群了。<ref>de Crespigny (1984), p. 416</ref>
