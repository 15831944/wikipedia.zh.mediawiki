''' 白连春 '''{{BD|1965年||CatIdx=B}}，笔名'''李当然'''，[[中国大陆|中国大陆]]著名“打工[[诗人|诗人]]”，因其生活困顿，又被称为“苦瓜诗人”和“苦难的代名词”。<ref name="打工诗人”出书 在北京买房每月仅500元生活费 ">{{cite web |url=http://culture.ifeng.com/8/detail_2011_11/28/10945896_0.shtml|title=《打工诗人”出书 在北京买房每月仅500元生活费》|accessdate=2011年11月28日|work=|publisher= 凤凰网|language=zh-hans}}</ref>

== 简介 ==
1965年，白连春出生于[[四川省|四川省]][[泸州市|泸州市]]，刚刚生下来，他就被父亲遗弃，<ref name="白连春新书四川发布">{{cite web |url=http://sichuan.scol.com.cn/cddt/content/2011-11/08/content_3082487.htm|title=《"苦瓜诗人"出书 白连春作品集发布会在川师举行》|accessdate=2011-11-08 |work=|publisher=四川新闻|language=zh-hans}}</ref>一直挨打挨骂度过十五年穷困潦倒的时光。<br>

[[初中|初中]]毕业后，15岁就离开家乡，最初在[[黑龙江|黑龙江]]当兵，继而流浪，而后打工。<ref name="白连春新书四川发布">{{cite web |url=http://sichuan.scol.com.cn/cddt/content/2011-11/08/content_3082487.htm|title=《"苦瓜诗人"出书 白连春作品集发布会在川师举行》|accessdate=2011-11-08 |work=|publisher=四川新闻|language=zh-hans}}</ref><br>

20多岁的时候因为爱情卖血而感染了[[艾滋病|艾滋病]]。<ref name="白连春新书四川发布">{{cite web |url=http://sichuan.scol.com.cn/cddt/content/2011-11/08/content_3082487.htm|title=《"苦瓜诗人"出书 白连春作品集发布会在川师举行》|accessdate=2011-11-08 |work=|publisher=四川新闻|language=zh-hans}}</ref><br>

1985年，白连春开始正式发表作品，2000年加入了[[中国作家协会|中国作家协会]]，在[[北京|北京]]流浪打工的时候，他除了当过校对，他还曾烧过锅炉，在公司食堂当[[厨师|厨师]]。<ref name="白连春新书四川发布">{{cite web |url=http://sichuan.scol.com.cn/cddt/content/2011-11/08/content_3082487.htm|title=《"苦瓜诗人"出书 白连春作品集发布会在川师举行》|accessdate=2011-11-08 |work=|publisher=四川新闻|language=zh-hans}}</ref><br>

2008年因病回到家乡，2010年再次流浪于[[广东|广东]]、[[湖南|湖南]]、[[河北|河北]]和[[海南|海南]]之间，2011年7月再次返回家乡。<ref name="打工诗人”出书 在北京买房每月仅500元生活费 ">{{cite web |url=http://culture.ifeng.com/8/detail_2011_11/28/10945896_0.shtml|title=《打工诗人”出书 在北京买房每月仅500元生活费》|accessdate=2011年11月28日|work=|publisher= 凤凰网|language=zh-hans}}</ref>
