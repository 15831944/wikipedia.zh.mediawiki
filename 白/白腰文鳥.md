'''白腰文鳥'''（[[學名|學名]]：''{{lang|la|Lonchura striata}}''）是[[梅花雀科|梅花雀科]][[文鸟属|文鸟属]]的一种，俗名白背文鸟、尖尾文鳥、十姐妹、白腰算命鸟、禾谷，成群出現於海岸濕地、平地到低海拔的草叢中，多见于农作区及山脚地带、也见于小溪、池塘边、庭院内的灌木丛或竹林间、巢多筑于山旁、村庄附近以及溪沟边或庭园内的竹丛、灌丛或针、阔叶树上，分布于[[日本|日本]]（引进种）、[[老挝|老挝]]、[[孟加拉国|孟加拉国]]、[[不丹|不丹]]、[[中国|中国]]（[[四川|四川]]、[[云南|云南]]以东的华南各省）、[[越南|越南]]、[[印度|印度]]、[[台湾|台湾]]、[[泰国|泰国]]、[[印度尼西亚|印度尼西亚]]、[[缅甸|缅甸]]、[[斯里兰卡|斯里兰卡]]、[[马来西亚|马来西亚]]、[[尼泊尔|尼泊尔]]、[[新加坡|新加坡]]和[[柬埔寨|柬埔寨]]。该物种的模式产地在斯里兰卡<ref name='id4072'/>，保护状况被评为[[无危|无危]]。

== 特徵 ==
白腰文鳥平均体重约为12.3克，體長只有約11公分，身上顏色大致都是褐色，腰為白色。嘴[[喙|喙]]成三角錐狀，又厚又尖，上嘴喙黑色，下嘴喙灰色。腰部有條白色寬橫紋帶，尾部又長而尖，鳴叫時發出「啾--啾--」的聲音。

== 棲息地 ==
通常活動於草叢，稻田中，在[[榕樹|榕樹]]、相思樹等上築巢，以草葉編織成球狀鳥巢。

== 繁殖 ==
繁殖期於每年的四月至七月。
