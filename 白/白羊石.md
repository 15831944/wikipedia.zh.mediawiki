'''白羊石'''（[[學名|學名]]：{{lang|la|'''''Arietites'''''}}），又名'''白羊菊石'''、'''單脊菊石'''，是生存在[[早侏羅世|早侏羅世]][[海洋|海洋]]中的一[[屬|屬]][[菊石|菊石]]。其化石在[[德國|德國]]、[[義大利|義大利]]、[[英國|英國]]、[[加拿大|加拿大]]和[[中國|中國]]等地都有發現。<ref name="Fossilworks" />

== 特徵 ==
白羊石的殼體為平旋捲式，形似大[[羊|羊]]角；外殼的橫截面近似於[[方|方]]形，殼表有明顯的粗橫肋條，殼腹呈鈍[[圓|圓]]狀而突出，其上有三條縱脊與二個凹溝。<ref name="digimuse">{{Cite web |url = http://digimuse.nmns.edu.tw/Demo_2011/showMetadata.aspx?ObjectId=090000018005b4c9&TypeKind=suMeta&Type=invertebrates&Part=2-2&Domain=gg&Field=fi&Language=CHI |title = digimuse.nmns.edu.tw/Demo_2011/showMetadata.aspx?ObjectId=090000018005b4c9&TypeKind=suMeta&Type=invertebrates&Part=2-2&Domain=gg&Field=fi&Language=CHI |language = en |accessdate = 2018-08-13 }}</ref>

和本屬近似的屬為龍骨較少的''[[Megarietites|Megarietites]]''屬和肋條間距更緊密的''[[Epammonites|Epammonites]]''屬。
