'''白神岳'''（{{jpn|j=白神岳|hg=しらかみだけ|rm=Sirakami-dake}}）是日本[[青森縣|青森縣]]西部[[西津輕郡|西津輕郡]][[深浦町|深浦町]]南部的[[山|山]]，海拔高度为1235米，上面有一等[[三角點|三角點]]標高為1231.9米。這座山是[[世界遗产|世界遗产]]記錄的[[白神山地|白神山地]]核心地區的较高峰之一，也是入選[[日本二百名山|日本二百名山]]的山峰。

== 地理 ==
白神山地（{{jpn|j=白神山地|hg=しらかみさんち|rm=Sirakami-santi}}）大部分落在青森縣西南部，小部分在[[秋田县|秋田县]]西北部，于1993年12月列入世界自然遺產，是日本首批世界遺產之一。此前這地區叫做弘西山地（{{jpn|j=弘西山地|hg=こうせいさんち|rm=kōsei-santi}}）。

這地區位於東北日本内弧隆升帶，現在也處於[[造山運動|造山運動]]活躍期。這個地區的地層有[[白垩纪|白垩纪]]的[[花崗岩|花崗岩]]、[[新近纪|新近纪]]的[[沉积岩|沉积岩]]、侵入岩類。沉积岩里分布着[[泥岩|泥岩]]、[[页岩|页岩]]，多發[[山崩|山体滑坡]]等地質災害。白神山地腹部有廣大的{{link-ja|圓齒水青冈|ブナ}}森林，多種多樣的植物群落也共存其中。<ref>{{Cite web |url = http://www.env.go.jp/nature/isan/worldheritage/japanese/shirakami/about.html |title = 森の博物館|language = ja |accessdate = 2017-11-14}}</ref>

白神岳位於白神山地西北邊界上，距海岸約6千米，有多條登山道路，有現在對登山初學者開設的“{{lang|ja|{{ruby|蟶|マテ}}山}}”主路線，具有歷史的難度較高的“二股”路線等。因為鄰近海邊拔地而起，垂直氣候變化劇烈。<ref name="daiichikanko-Sirakami-dake">{{Cite web |url = http://www.daiichikanko.jp/about07.html |title = オススメの白神「白神岳」の楽しみ方|language = ja |accessdate = 2017-11-14}}</ref>其東北方向約4千米是白神山地最高峰“向白神岳”，海拔高度約1250米，沒有登山道，向白神岳又被記載為“{{lang|ja|女嶽}}”。

山頂附近存在着偽高山帶的[[林木線|林木線]]，從山頂眺望能看到白神山地核心地區的壮观景象。因為擔心入選世界遺產後登山客流增加過度，山頂上設置了廁所和避難小屋。

== 歷史 ==
白神岳殘存的最早記錄是1645年[[弘前藩|弘前藩]]向[[江户幕府|幕府]]提交的《{{lang|ja|陸奥国津軽郡之絵図}}》，此地圖中白神岳被標記為“{{lang|ja|しらかみの嶽}}”。[[江户时代|江户时代]]的紀行家{{link-ja|菅江真澄|菅江真澄}}則記載白神岳為“{{lang|ja|白上山}}”“{{lang|ja|白髪山}}”等。春季從秋田縣[[八峰町|八峰町]]能看見白神岳山頂上的雪像一個“上”字，因此菅江真澄將其記作“{{lang|ja|白上山}}”。

白神岳是當地{{lang|ja|{{ruby|大間越|おおまごし}}}}地方的人的信仰之山，[[農曆|舊曆]]八月初一的時候人們登山祈禱。據說古時不允許女性登山，女性開始登山是{{link-en|戰後|Aftermath of World War II}}的事情。<ref name="daiichikanko-Sirakami-dake" />据[[深浦町|深浦町]]{{link-ja|岩崎|岩崎村 (青森県)}}地區的人傳說，白神嶽和[[岩木山|岩木山]]是姐妹關係，岩崎的人们又把岩木山称为“{{lang|ja|下の神の山}}”。
