'''白洁'''（{{bd|1972年|3月28日|catIdx=B}}），中国女足运动员，生于[[河北|河北]][[张家口|张家口]]<ref name="sina1">[http://sports.sina.com.cn/c/2003-10-01/1305604590.shtml “假小子”白洁的铁血柔情：厉害女人 柔情女人]</ref>，为中国队出场139次，帮助中国队于1999年夺取[[女足世界杯|女足世界杯]]亚军。白洁最初踢左后卫，且因踢球风格被誉为“女[[罗伯托·卡洛斯|罗伯托·卡洛斯]]”<ref>{{cite web |url = http://google.com/search?q=cache:SaV4oQQNCYIJ:www.the-afc.com/eng/awards/annual/awards.jsp_2003awards.html+Bai+Jie+football&cd=7&hl=en&ct=clnk&gl=us&client=firefox-a |title = AFC Annual Awards }}</ref>，但后来转型为国家队的进攻球员<ref>{{cite web |title =Bai Jie: 'China will not fall behind' |url = https://www.fifa.com/tournaments/archive/tournament=103/edition=6929/news/newsid=11741.html |date=19 August 2003 |publisher=FIFA }}</ref>。

1996年<ref name="sina1" />，白洁加入[[中国人民解放军|解放军]]，成为一名[[解放军籍运动员|解放军籍运动员]]。2003年被提名为首届[[亚足联|亚足联]]年度女子球员<ref>{{cite web |url = http://webcache.googleusercontent.com/search?q=cache:dwEsnLfU0uMJ:www.the-afc.com/eng/articles/viewArticle.jsp_168331188.html+site:the-afc.com+%22Bai+Jie%22&cd=2&hl=en&ct=clnk&gl=us&client=firefox-a |title = AFC Women's Awards |date = 19 November 2008 }}{{Dead link|date=2018年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，并在12月10日当选。当年6月11日，在对[[印度|印度]]的世界杯资格赛中，白洁打入五球，队友[[孙雯|孙雯]]也打入五球<ref>{{cite web|url=https://www.fifa.com/tournaments/archive/tournament=521/edition=7055/matches/match=33012/report.html |publisher=FIFA |title=China PR-India, 11 June 2003 Match Report |deadurl=yes |archiveurl=https://web.archive.org/web/20100623063225/http://www.fifa.com/tournaments/archive/tournament%3D521/edition%3D7055/matches/match%3D33012/report.html |archivedate=2010-06-23 |df= }}</ref>。

[[2003年女子世界杯足球赛|2003年女子世界杯足球赛]]后，白洁再也没有在联赛中露面，实际上等于退役。2004年白洁担任[[八一女足|八一女足]]的主教练<ref>[http://sports.sohu.com/2004/03/24/40/news219574025.shtml 白洁挂帅回赛场：退役最痛苦 搞好调动就结婚]</ref>，2005年6月辞职<ref>{{Cite web |url=http://www.ycwb.com/gb/content/2005-06/07/content_916679.htm |title=八一女足主帅白洁辞职 |access-date=2011-09-22 |archive-url=https://web.archive.org/web/20110918130915/http://www.ycwb.com/gb/content/2005-06/07/content_916679.htm |archive-date=2011-09-18 |dead-url=yes }}</ref>。之后以[[中华人民共和国军事级别|团级]][[干部|干部]]身份[[转业|转业]]，现在任职于[[深圳市公安局|深圳市公安局]]龙岗分局巡警大队<ref>[http://www.dayoo.com/roll/201005/31/10000307_102182024.htm 老女足的今日生活：白洁当警察 浦玮定居美国] {{webarchive|url=https://web.archive.org/web/20150402155453/http://www.dayoo.com/roll/201005/31/10000307_102182024.htm |date=2015-04-02 }}</ref>。
