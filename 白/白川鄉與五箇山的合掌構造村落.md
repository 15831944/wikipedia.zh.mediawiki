'''白川鄉與五箇山的合掌造聚落'''<ref>[https://twh.boch.gov.tw/world/intro.aspx?id=365 白川鄉與五箇山的合掌造聚落-世界遺產]</ref>（{{lang-ja|白川郷·五箇山の合掌造り集落}}）是[[日本|日本]]飛驒地區[[白川鄉|白川鄉]]（[[岐阜縣|岐阜縣]][[大野郡_(岐阜縣)|大野郡]][[白川村|白川村]]）及[[五箇山|五箇山]]（[[富山縣|富山縣]][[南礪市|南礪市]]）地區的[[合掌造|合掌造]]聚落的總稱。於1995年12月9日被[[聯合國教科文組織|聯合國教科文組織]]登錄為[[世界文化遺產|世界文化遺產]]。因為冬天的景色就像在童話故事中情境一般，被喻為「[[冬日的童話村|冬日的童話村]]」。也是[[日本|日本]][[三大祕境|三大祕境]]之一。

== 合掌造 ==
[[合掌造|合掌造]]（{{lang|ja|合掌造り}}）是指一種日本特有的民宅形式，其特色是以[[茅草|茅草]]覆蓋的屋頂，呈人字型的[[屋頂|屋頂]]如同雙手合十一般，於是就被稱為「合掌」。合掌造為木造建築物，在興建的過程中完全不用釘子，但仍然十分牢固。合掌造的屋頂十分陡峭，這是使積雪容易滑落不會堆積，以避免冬季的大雪將屋頂壓垮（類似[[歐洲|歐洲]]的建築）。合掌造每隔三、四十年就必須更換老朽的屋頂茅草。更換茅草需要大量人力，故每次有哪一家人的屋頂需要翻修，全村的人就會同心協力一起完成，這種合作方式稱為「結」（{{lang|ja|ゆい}}）。

=== 合掌造的歷史 ===
傳說這種建築形式是13世紀初[[平氏|平氏]]家族在[[源平合戰|源平合戰]]戰敗後遁入深山，為了禦寒並躲避追兵而建造的<ref>[http://www.cctvdf.com/c/column/daifu_column03.php?touristid=55 童话般的小屋——合掌造] {{webarchive|url=https://web.archive.org/web/20070928190043/http://www.cctvdf.com/c/column/daifu_column03.php?touristid=55 |date=2007-09-28 }}</ref>。目前保存下來的建築物大約是在[[江戶時代|江戶時代]]中後期興建的。

將合掌造介紹給全世界的，是[[德國|德國]]的建築學者[[:en:Bruno_Taut|Bruno Taut]]，他在1935年左右於日本進行傳統民居樣式的調查，實地走訪合掌造後大感驚艷，在《日本美的再發現》一書中讚譽其為「極端合理，就連日本境內也相當罕見的傳統庶民建築」。

== 保存 ==
白川鄉原被劃為建設大壩的預定地，但是隨著抗爭事件的發生，日本人注意到合掌造有被保存的必要性。於是自1967年開始以現存最早，有400年歷史的「{{link-ja|和田家住宅|和田家住宅}}」（國家指定[[重要文化財|重要文化財]]）為中心，將各地合掌造的房屋被遷移到白川鄉保存，形成現存最大的合掌造聚落。目前白川鄉共有113棟<ref>[http://www.libertytimes.com.tw/2005/new/nov/21/today-art1.htm 世界遺產成為觀光致富之路 白川鄉合掌村的山中傳奇]</ref>合掌造建築。

為了妥善保護這些特殊建築，村落居民發起保護運動，於1971年成立「白川鄉荻町村落自然環境保護會」，制定了保護資產的住民規章。1976年，在文化財保護法的推動下，依法以「重要傳統建築群保護區」進行全面性的保護。1995年，白川鄉及五箇山的合掌造聚落被登錄為世界文化遺產。

== 登錄標準 ==
白川鄉與五箇山的合掌造聚落因為符合以下的標準，而被登錄為世界文化遺產：
* (iv) 突出地代表了某一類建築或技術的，並且展示了人類歷史上的某一段或幾段非常重要的時期。
* (v) 突出地代表了一種或幾種文化中人類傳統的居住方式、利用土地或海洋的方式，或是代表了人類與環境的互動關係，尤其是當這種關係在不可逆的變化下顯得非常脆弱的時候。

== 問題 ==
目前，合掌造所面臨的主要問題如下：
* 火災：合掌造完全由茅草及木材建成，一旦發生火災，後果真的難以設想。對此，村落除了徹底落實「嚴禁煙火」的管制措施，並增添不少防災設備，包括消防栓、噴水槍等。
* 高速公路：東海北陸高速公路帶來廢氣等污染。
* 觀光人潮：自從被登錄為世界遺產後，觀光的人潮非常多，帶來的混亂和破壞也有增加的趨勢。
