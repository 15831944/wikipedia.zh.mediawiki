'''白腰豆'''（{{lang-en|'''cannellini'''}}），又名'''白腎豆'''、'''海軍豆'''<ref>{{cite web |url=http://cn.iherb.com/eden-foods-organic-navy-beans-15-oz-425-g/10643 |title=Eden Foods: 有機海軍豆（白豆）|language=zh-hans |date=2014-11-26 }}</ref>，是[[菜豆屬|菜豆屬]]的[[菜豆|菜豆]]（{{lang|la|''Phaseolus vulgaris''}}）的一種<ref>{{cite web|url=http://www.whfoods.com/genpage.php?pfriendly=1&tname=foodspice&dbid=87|publisher=The world's healthiest foods|title=Kidney Beans|accessdate=2007-11-05 |language=en }}</ref><ref>{{cite web |url=http://www.pulsecanada.com/food-health/what-is-a-pulse/bean |title=Bean (''Phaseolus vulgaris'') |accessdate=2014-08-30|language=en |publisher= Pulse Canada}} </ref><ref>{{cite web|url=http://www.ahdictionary.com/word/search.html?q=pea+bean|title=Pea bean|publisher=The American Heritage Dictionary |accessdate=2013-09-17 |language=en }}</ref>。色白，形狀似[[腎臟|腎臟]]（又叫作「腰」），因以為名。

這種白腰豆在[[英國|英國]]和[[美國|美國]]很受歡迎。相較於其他白色的豆類，長圓形的白腰豆的體積較細小。常用作烹調[[焗豆|焗豆]]和[[豆餡餅|豆餡餅]]及加入到多種西湯內（例如：著名的[[參議院豆湯|參議院豆湯]]<ref>{{cite web |url=http://www.senate.gov/reference/reference_item/bean_soup.htm |title=Senate Bean Soup |language=en }}</ref>）。由於近年研究發現白腰豆含有豐富的[[阿魏酸|阿魏酸]]（Ferulic acid）及[[p-香豆酸|p-香豆酸]]（p-Coumaric acid），各種各樣聲稱含有「白腰豆精華」（{{lang|en|White Kidney Bean Extract}}）的保健品都在全球熱賣，而且聲稱是「減肥減重」的恩物。

==營養價值==
白腰豆含有豐富的植物性[[磷脂絲胺酸|磷脂絲胺酸]]（{{lang|la|phosphatidylserine}}，簡稱PS）<ref>{{cite book
