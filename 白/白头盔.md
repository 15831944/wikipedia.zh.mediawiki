'''敘利亞公民防衛'''（{{lang-ar|الدفاع المدني السوري}}），通稱'''白頭盔'''，是一個在2013年[[叙利亚内战|叙利亚内战]]時期於[[敘利亞|敘利亞]]叛軍控制地區成立的以“[[民事防护|民事防护]]”为名义的組織，由英国军人[[梅苏里尔|梅苏里尔]]组建。<ref name=voc20180425>{{Cite web|url=http://dsj.voc.com.cn/article/201804/201804261158141303.html|title=叙利亚“白头盔”的爹，居然是这个英国军官！ |accessdate=2018-05-05 |author=|date=2018-04-26 |work=dsj.voc.com.cn }}</ref>由於行动時會戴上白色的頭盔，所以有「白頭盔」這個外號。他們分別派駐於各個前線戰區（如[[阿勒颇|阿勒颇]]、[[大马士革|大马士革]]、[[霍姆斯_(敘利亞)|霍姆斯]]等）提供醫療服務。据宣称，该机构自成立以來，已拯救了六萬多個平民的性命，同時也有超過上百名志願者犧牲。<ref>{{Cite web|url=http://news.now.com/home/international/player?newsId=194363|title=敘利亞民間防衛組織|accessdate=2016-12-15|author=|date=|work=news.now.com|publisher=}}</ref>

2013年至2016年年间，这个团体从美英等国的政府、或其他“非政府组织”名号的机构获得了大约1.23亿美元的运作资金。<ref name=voc20180425/>2018年5月4日，据[[哥伦比亚广播公司|哥伦比亚广播公司]]（CBS）报道，在“白头盔”所制作的“叙利亚化武袭击视频”被[[俄罗斯|俄罗斯]]等国揭露为造假之后，[[美国政府|美国政府]]冻结了对“白头盔”机构的资金供给。<ref name=sina20180504>{{Cite web|url=http://mil.news.sina.com.cn/2018-05-04/doc-ifzfkmth8812076.shtml|title=“白头盔”被指责视频造假 今面临美国务院资金断供 |accessdate=2018-05-05|date=2018-05-04 |work=sina.com.cn }}</ref>“白头盔”的组织负责人对此表示，美国断供会导致他们营收额损失达三分之一。<ref name=sina20180504a>{{Cite web|url=http://mil.news.sina.com.cn/2018-05-04/doc-ifyuwqfa6240575.shtml|title=该组织导演叙利亚化武事件 因美国中断援助急眼了 |accessdate=2018-05-05|date=2018-05-04 |work=sina.com.cn }}</ref>

2018年12月20日，俄罗斯向[[联合国|联合国]]呈交了关于“白头盔”在叙利亚境内活动的相关调查报告，指控“白头盔”参与了强行摘除人体器官、勾结恐怖分子、抢劫、盗窃与腐败等活动。<ref>{{Cite web|url=https://www.guancha.cn/internation/2018_12_21_484085_s.shtml|title=俄向联合国提交调查报告：白头盔在叙摘除人体器官、勾结恐怖分子|accessdate=2018-12-28|work=www.guancha.cn}}</ref>

== 簡介 ==
“白头盔”全名为叙利亚民防组织，成立于2013年初，主要活动在叙利亚反对派占领区阿勒颇省。因为成员在救援活动中常常戴着一顶白色头盔，该组织又被称为“白头盔”。

根据其官网介绍，“白头盔”组织大约有3000名志愿者，他们的主要工作是当袭击发生后，“尽可能在最短的时间内挽救更多的生命，并尽量减少对人和财产的损害”。“白头盔”组织负责人雷德 萨利赫宣称，该组织成员是“叙利亚令人难以置信的英雄”，他们在世界上最危险的地方工作，已从袭击中解救了99220人。

“白头盔”的创始人是詹姆斯 勒 梅西耶尔。公开信息显示，梅西耶尔毕业于英国桑赫斯特皇家军事学院，是一名英国前军事情报官员，曾参与了北约一些以“人道主义干预”为名的战争，包括波斯尼亚、科索沃和伊拉克等。

根据联合国的报道，2000年时，梅西耶尔离开部队为联合国工作，因为他意识到较之军队在一场战争中的威慑力，“人道主义援助更有效”。此后，他开始重点在一些“不稳定”的国家，通过安全公司等进行“稳定的”活动。

== 联合国关于叙利亚白头盔组织会议 2018 ==

2018年12月20日，联合国进行了关于叙利亚白头盔组织的讨论会议，各国代表和记者们都在场，这段视频在联合国电视频道UN LIVE上进行了现场直播，并在Youtube上供的观察者观看。<ref>[http://webtv.un.org/www.unwomen.org/en/news/watch/roundtable-discussion-on-the-middle-east-issues-activities-of-the-white-helmets-organization-in-syria/5982221816001/?term=%20&page=18 联合国会议及视频链接]</ref><ref name="guancha">{{Cite web|url=https://www.guancha.cn/internation/2018_12_21_484085_s.shtml|title=俄向联合国提交调查报告：白头盔在叙摘除人体器官、勾结恐怖分子|accessdate=2018-12-28|work=www.guancha.cn}}</ref>

其中联合国讨论了“人道救援组织”所参与犯罪活动的无可辩驳的证据，其中包括器官盗窃，与恐怖分子合作进行假救助，从平民那里偷窃，以及其他非救助行为。

英国记者凡妮莎·贝利（Vanessa Beeley）根据她多年来对“白头盔”的创立、资助来源和罪恶活动的研究，进行了事实性讲座。研究包括多次访问“白头盔中心”，对叙利亚平民的无数次访问，甚至包括对叙利亚一名“白头盔”头目的访谈。

俄羅斯民主研究基金会的主任（联合国全球反恐研究网络的成员之一）主任马克西姆·格里高里耶夫（Maxim Grigoriev）发表了长篇讲话，详细介绍了他的基金会采访过的100多名目击者中的一些人，其中包括40多名白盔成员，15名前恐怖分子和50名来自恐怖分子。另有500人在阿勒颇和达拉接受访谈。马克西姆提供的证词中，有许多关于白头盔参与器官盗窃的案例。

证词引用阿勒颇的护理负责人的说话，他看到他的邻居的尸体，而上前这位邻居被白头盔人员带到土耳其进行“治疗”。“我掀起床单，看到一个从喉咙切到肚子的大伤口......我用手触摸他，明白没有任何器官留下。”

另一位受访者说：“一个人受了轻伤，被救了出来……然后被带了回来，但胃被割开了，内脏也不见了。”

对平民、白头盔士兵和恐怖分子成员的采访，揭穿了西方媒体的谎言：在白头盔中有一些害群之马。但总的来说，这些人都是人道主义救援人员。

例如，叙利亚平民奥玛尔·马斯塔法（Omar al-Mustafa）说：“几乎所有在附近白头盔部队中心工作的人都是努斯拉阵线的战士，或者与他们有关联。我想要加入白头盔部队，但有人告诉我，如果我不是从努斯萨拉来的，他们就不能录用我。”

还有更多的证词详细描述了假营救和假化学攻击。奥马尔·穆斯塔法被引述说：“我看到他们（白盔部队）把活着的孩子带来，把他们放在地板上，就好像他们死于化学武器袭击。”

这些证词不仅对白头盔组织构成了刑事指控，也对那些在2016年受到西方媒体追捧的医生构成了刑事指控。

据一名受访者穆罕默德·巴希尔·比拉姆（Mohamed Bashir Biram）称，他试图把父亲送到巴扬医院（al-Bayan hospital）的白盔部队附属医院，但以失败告终。他说:“因为我父亲不是战士，医院的医生拒绝帮助他，他死了。”

专家指出，“我们理解为什么白盔部队要被西方国家政府保卫？他们并没有隐瞒他们向本组织提供了大量财政支助，并利用它在人道主义掩护下实现政治目标。保护你的资产是合乎逻辑的。

上周，《明镜》周刊的德国记者克拉斯雷·洛蒂乌斯（Claas Relotius）伪造大量文章被曝光，他是2018年德国记者奖的得主。一名知情者指出，克拉斯雷“承认在55篇文章中至少编造了14篇”，其中包括“一篇关于一名叙利亚男孩的故事，他认为自己的造假引发了该国的内战。这篇文章三周前刚刚获得德国记者奖（German Reporter Prize），但纯属虚构。”<ref>{{Cite web |url =

== 爭議 ==
===被指持有武裝，是恐怖分子===
有人指出白頭盔持有武裝，是恐怖分子，且與敍利亞的反抗軍是一丘之貉，與外界認知大為相反。
* [[俄罗斯|俄羅斯]]總統[[弗拉基米尔·弗拉基米罗维奇·普京|普丁]]曾說他們為「恐怖分子」，背後有美英等國控制，敍利亞政府軍的戰機更轟炸他們多棟大樓。
* 曾任英軍官員的梅熱勒（James Le Mesurier）則澄清，組織的資金來自包括[[丹麦|丹麥]]、[[英国|英國]]、[[德国|德國]]等多國政府贊助，亦有來自民間的個人捐款。<ref>{{Cite news|url=http://hk.on.cc/int/bkn/cnt/news/20161006/bknint-20161006110240648-1006_17011_001.html|title=敍國戰火中救援　白頭盔成諾貝爾和平獎大熱|newspaper=on.cc 東網|accessdate=2016-12-15}}</ref>
* 白头盔的视频是通过“伊德利卜省叙利亚民防”渠道上传，由白头盔与“协作sarmin”制作，后者的标志是一个黑色的基地组织旗帜。另有视频显示，白头盔人员与恐怖分子一起鸣枪庆祝，替被处决的恐怖分子收尸。甚至，连短片《白头盔》拍摄者之一、“白头盔”负责人雷德·萨利赫还因为有与叙利亚“基地组织”、“努斯拉阵线”等恐怖组织联系的嫌疑，未被美国获准入境。 <ref>[http://www.guancha.cn/internation/2017_04_17_404058_s.shtml 当联合国都被《白头盔》感动的热泪盈眶时，终于有人站出来了]</ref>

===救援视频疑似摆拍造假===
据[[中华人民共和国|中国]][[央视网|央视网]]报道，“白头盔”组织公布的救援视频有摆拍嫌疑，譬如在某部视频的一开始，伤员和“白头盔”救援人员均保持不动，在“开拍”指令下达后，救援人员才开始将倒在战墟中的伤员救出。 <ref>{{Cite news|url=http://www.guancha.cn/global-new/2016_12_13_384147.shtml|title=叙利亚“白头盔”组织公布的视频引起广泛质疑|accessdate=2016-12-13|newspaper=观察者网}}</ref>

2017年2月13日，瑞典医生人权组织（SWEDHR）发布了一份“回顾性报告”报告，直指白头盔的“宣传”视频存在疑点，“心脏注射器是空的”、“注射方式不正确”，疑似摆拍。此外，视频里还传出一段阿拉伯语对话：“包括照片在内，母亲必须在下面、孩子在上面。必须确保母亲一定要在孩子下方。” <ref>[http://www.guancha.cn/internation/2017_04_17_404058_s.shtml 当联合国都被《白头盔》感动的热泪盈眶时，终于有人站出来了]</ref>

2018年4月11日，伊朗媒体法尔斯通讯社报道，在叙利亚政府军（SAA）占据东古塔杜马镇郊区村庄Saqba以后，叙利亚政府军发现了一处“白头盔”组织的“影视基地”。军方人员在现场发现了相机和电影制作设备。政府军称这个基地是用来拍摄类似于空袭和化学武器袭击等视频。叙利亚驻联合国代表在周一的安理会上讽刺称，这些化学武器从来都不去攻击手里有武器的人，而且已经学会分辨出女人和孩子了。他们像拍好莱坞一样拍摄的视频，就是为了栽赃叙利亚政府。<ref>[http://www.guancha.cn/military-affairs/2018_04_11_453368_s.shtml 叙军声称找到白头盔虚假化武袭击拍摄地点]</ref><ref>[https://www.youtube.com/watch?v=cOk_VbaA5qU 央視官方頻道-白頭盔假視頻再起?]</ref>

===美国政府于2018年5月冻结对“白头盔”机构的资金流===
2018年5月4日，据哥伦比亚广播公司（CBS）报道，在“白头盔”所制作的“叙利亚化武袭击视频”被俄罗斯等国揭露为造假之后，美国政府冻结了对“白头盔”机构的资金供给。<ref name=sina20180504/>

===涉嫌偷窃===
据俄罗斯[[今日俄罗斯|今日俄罗斯电视台]]报道，“白头盔”组织成员有偷窃行为。有[[阿勒颇|阿勒颇]]平民表示，“白头盔”在救援伤者时会顺走伤者的珠宝、黄金等贵重物品，甚至会强占平民的住房。<ref>{{Cite news|url=http://world.huanqiu.com/hot/2016-12/9819684.html|title=叙利亚宣传战：叙民众称“白头盔”是小偷|accessdate=2016-12-16|newspaper=环球网}}</ref>

=== 被指控强行摘除人体器官 ===
2018年12月20日，在联合国有关叙利亚问题会议上，[[俄罗斯民主研究基金会|俄罗斯民主研究基金会]]主任格里高利耶夫（Maxim Grigoriev）向[[联合国|联合国]]呈交了关于“白头盔”在叙利亚境内活动的相关调查报告，指控“白头盔”参与了强行摘除人体器官、勾结恐怖分子、抢劫、盗窃与腐败等活动。<ref name="guancha"/>
