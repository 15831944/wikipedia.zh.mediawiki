《'''白幽灵传奇之绝命逃亡'''》（{{lang-en|'''''Outcast'''''}}）是一部2014年中美合拍的[[史诗片|史诗]]、[[动作片|动作]]和[[武术片|武术片]]，原名「最後的戰爭」和「絕命逃亡」，電影為[[尼古拉斯·鲍威尔|尼古拉斯·鲍威尔]]執導，由[[海登·克里斯滕森|海登·克里斯滕森]]、[[刘亦菲|刘亦菲]]、[[尼可拉斯·凱吉|尼可拉斯·凱吉]]、[[安志杰|安志杰]]、[[吉克隽逸|吉克隽逸]]主演。<ref>{{cite web|url=http://ent.xinmin.cn/2013/09/13/21901120.html|title=《最后的战争》开机现场照 人物造型场景曝光|accessdate=2013|publisher=新民网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20160305021203/http://ent.xinmin.cn/2013/09/13/21901120.html|archivedate=2016-03-05}}</ref><ref>{{cite web |url= http://www.m1905.com/news/20121116/589061.shtml|title=凯奇加盟《最后的战争》 海登留胡子玩混合武术 |accessdate= 2012|publisher=电影网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.qq.com/a/20131231/006587.htm |title=《最后的战争》新年特辑 凯奇刘亦菲致跨年问候 |accessdate=2013 |publisher=腾讯网 |work= |language=zh-hans}}</ref><ref>{{cite web |url= http://news.xinhuanet.com/yzyd/ent/20131021/c_117801776.htm|title= 凯奇《最后的战争》发布会 刘亦菲变花仙子完胜吉克隽逸|accessdate=2013 |publisher=新华网 |work= |language=zh-hans}}</ref>尼可拉斯·凱吉称拍摄该片是為了向华人功夫影星[[李小龙|李小龙]]致敬。<ref>{{cite web |url=http://ent.qq.com/a/20131008/011205.htm |title=凯奇拍戏拿奖两不误 《最后的战争》致敬李小龙 |accessdate=2013 |publisher=腾讯网 |work= |language=zh-hans}}</ref>為[[云南电影集团|云南电影集团]]和[[电弧光影业|电弧光影业]]一同制作。

電影原定於2014年9月26日在中國上映<ref>{{cite web |url= http://news.mtime.com/2014/06/16/1528434.html#p2 |title= 《绝命逃亡》定档9月26日 刘亦菲花裙造型甜美 凯奇将来华宣传 |date=2014-06-16 |accessdate= 2014-07-03 |work= mtime.com |language=zh}}</ref>，後延期至2015年4月3日中国上映。

==剧情==
公主蓮的父皇被長兄無情殺害，不得不帶著年幼的弟弟昭踏上逃亡之路，他們將穿越重重險境與國境另一端的將軍們會合。這一支忠心護主的軍隊已整裝待發，誓死匡扶幼主重回朝廷繼承皇位。謀反的皇兄在國境內懸賞通緝蓮昭二人，使得兩人逃亡的路上無人相助。姐弟二人不得不僱傭看起來蠻橫無理的異族流浪漢——阿肯充當他們的保鏢。然而阿肯的來歷並不簡單：他飽受戰爭摧殘，來到東方是為了逃避十字軍帶給他的血腥記憶...<ref>{{cite web |url=http://news.163.com/13/1216/11/9G7D2EG800014Q4P.html |title=《最后的战争》杀青计划于明年上映 |accessdate=2013 |publisher=网易 |work= |language=zh-hans}}</ref>

==製作==
该片曾在[[北京|北京]]、[[天津|天津]]、[[河北省|河北省]]、[[云南省|云南省]]等地取景。<ref>{{cite web |url=http://ent.163.com/13/1231/11/9HDV0MIP000300B1.html |title=《最后的战争》发新年特辑 凯奇刘亦菲致问候 |accessdate=2013 |publisher=网易 |work= |language=zh-hans}}</ref><ref>{{cite web|url=http://xmwb.news365.com.cn/wy/201310/t20131022_1666681.html|title=天津城变身“古罗马”|accessdate=2013|publisher=新民网|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20131025200610/http://xmwb.news365.com.cn/wy/201310/t20131022_1666681.html|archivedate=2013-10-25}}</ref>

该片在2013年12月杀青。<ref>{{cite web |url=http://ent.163.com/13/1219/15/9GFG1MKJ00032DGD.html |title=《最后的战争》杀青 凯奇吉克隽逸上演老少恋 |accessdate= 2013|publisher=网易 |work= |language=zh-hans}}</ref>

尼可拉斯·凱吉的著名代表作如《[[驚天奪寶|国家宝藏]]》，這也是他首次參與中美合拍的電影演出，海登·克里斯滕森的著名代表作如《[[星球大戰前傳II：複製人侵略|星球大戰前傳II：複製人侵略]]》。<ref>{{cite web |url=http://ent.sina.com.cn/m/c/2013-12-11/14234058609.shtml |title=《最后的战争》杀青 刘亦菲受尽百般苦 |accessdate=2013 |publisher=新浪网 |work= |language=zh-hans}}</ref>

该片是歌手吉克隽逸的电影处女作。<ref>{{cite web |url=http://www.chinanews.com/yl/2013/10-22/5407386.shtml |title=吉克隽逸出演《最后的战争》 献上电影处女秀(图)  |accessdate= 2013|publisher=中新网 |work= |language=zh-hans}}</ref>

==發行==
電影定於2014年9月26日在中國地區首映，但卻在當天上映的前兩小時緊急撤檔，雲南電影集團並未回應原因<ref>{{cite web|url=http://big5.soundofhope.org/node/510511|title=《絕命逃亡》上映前兩小時撤檔|accessdate=2014年9月26日|publisher=希望之聲電台|work=|language=zh-hans|deadurl=yes|archiveurl=https://web.archive.org/web/20141019053410/http://big5.soundofhope.org/node/510511|archivedate=2014年10月19日}}</ref>。后定档2015年1月16日，但为避免与发行方乐视影业的片子相撞，再次撤档，直到4月3日才正式上映。

中国大陆首周三天取得1130万人民币列第九位。

制片方云南电影集团的董事长张伦表示本片亏损一亿多人民币<ref>{{cite news|url=http://ent.sina.com.cn/m/c/2015-04-15/doc-ichmifpy8226235.shtml|title=《绝命逃亡》逃啥？出品方谈撤档：赔上亿|date=2015-04-15|publisher=新浪|accessdate=2015-04-15}}</ref>。
