'''白恩培'''（{{bd|1946年|9月8日|CatIdx=Bai,白}}），乳名珍琦，[[陕西省|陕西省]][[清涧县|清涧县]]人；[[西北工业大学|西北工业大学]]航海设备与自动控制系毕业，工程师。曾于1973年6月加入[[中国共产党|中国共产党]]，是中共第十三、十四届中央候补委员，第十五、十六、十七届中央委员；先后在[[内蒙古自治区|内蒙古]]、[[青海省|青海]]、[[云南省|云南]]等中国边境[[少数民族|少数民族]]聚集地区，担任主要领导职务。

2014年8月29日，[[中国共产党中央纪律检查委员会|中共中央纪律检查委员会]]对外宣布，白恩培涉嫌严重违纪违法正在接受组织调查<ref>{{cite web|url=http://sc.people.com.cn/n/2014/0829/c345454-22156004.html|title=人大环资委副主任委员白恩培涉违纪违法被调查|accessdate=2014-08-29|author=|date=2014-08-29|publisher=中国新闻网|language=}}</ref>；经过中共中央纪委的立案审查，2015年1月13日，中共中央纪委宣布，白恩培因涉嫌利用职务便利，为他人谋取利益、收受巨额受贿，经[[中国共产党中央纪律检查委员会|中共中央纪委]]报[[中国共产党中央委员会|中共中央]]批准，决定开除其中共党籍、公职，并移送司法机关依法处理<ref>{{cite web|url=http://politics.people.com.cn/n/2015/0113/c1001-26377297.html|title=云南省委原书记白恩培被双开 涉收受巨额贿赂|accessdate=2015-01-13|author=|date=2015-01-13|publisher=人民网|language=zh-cn}}</ref>。同日，[[最高人民检察院|最高人民检察院]]经审查决定以涉嫌受贿罪对白恩培立案侦查并采取强制措施。

2016年10月9日，河南省安阳市中级人民法院宣判，白恩培犯受贿罪、巨额财产来源不明罪，被判处[[死刑缓期执行|死刑，缓期二年执行]]，剥夺政治权利终身，并处没收个人全部财产，在其死刑缓期执行二年期满依法减为无期徒刑后，终身监禁，不得减刑、假释。对白恩培受贿所得财物和来源不明财产予以追缴，上缴国库<ref name="审判">{{cite web|url=http://news.sina.com.cn/c/2016-10-09/doc-ifxwrapv1102703.shtml|title=云南省委原书记白恩培被双判死缓，成为首位终身监禁官员|accessdate=2016年10月9日|author=|date=2016年10月9日|publisher=新浪新闻|language=zh-cn}}</ref>。白恩培是[[中华人民共和国刑法|中华人民共和国刑法]]第九修正案实施后，第一个因贪污、受贿数额特别巨大，并使国家和人民利益遭受特别重大损失而被判处死缓，减刑后终身监禁的副部级以上官员<ref>{{cite web|url=http://news.sohu.com/20161010/n469840450.shtml|title=白恩培成终身监禁“第一人”|accessdate=2016-10-10|author=|date=2016-10-10|publisher=搜狐|language=}}</ref>。

== 早年经历 ==
白恩培在出生于[[陕北|陕北]][[清涧县|清涧]]袁家沟村；读完小学后，十几岁时就离开家乡。1965年，考入自[[西北工业大学|西北工业大学]]航海设备与自动控制系学习。1970年毕业后，被安排到[[中国人民解放军陕西省军区|陕西省军区]]独立师农场劳动锻炼；1972年，调入刚刚创办的延安市柴油机厂工作。白恩培曾在这家地方国有企业前后工作11年；历任车间调度员、技术员，厂革委会副主任，副厂长、厂长，厂党委副书记等职。1983年2月，被调到当时[[延安市|延安]]地区最大的企业[[延安卷烟厂|延安卷烟厂]]，任厂长、兼党委副书记。

== 仕途 ==
1980年代初，中共推行“干部年轻化”的政策。既有学历、又懂技术的青年厂长白恩培，被立为培养对象。1983年9月，时年37岁的白恩培出任[[中共|中共]][[延安|延安]]地区委员会副书记；两年后，升任地委书记。1987年，在[[中共十三届一中全会|中共十三届一中全会]]上，作为革命圣地[[延安市|延安]]的“第一把手”，白恩培当选候补中央委员。<ref>[http://news.xinhuanet.com/ziliao/2002-03/04/content_299512.htm 白恩培简历，新华网，2002-03-04]</ref> 

1990年5月，白恩培离开家乡，在青海省以及边境省份[[内蒙古自治区|内蒙古]]、[[云南省|云南]]工作；历任[[中国共产党内蒙古自治区委员会|中共内蒙古自治区党委]]常委、组织部部长，自治区委副书记；[[中国共产党青海省委员会|中共青海省委]]副书记，副省长、代理省长，省长，[[省委书记|省委书记]]、兼[[人民代表大会制度|人大常委会]]主任；2001年10月起，担任[[中共云南省委|中共云南省委]]书记，后又兼任[[云南省|云南省]][[人民代表大会制度|人大常委会]]主任<ref>{{Cite web|url=http://www.chinavalue.net/Blog/29723.aspx|title=中国政坛新星系列——云南书记白恩培（连载四）|accessdate=2017-10-17|author=蔡鹏生|date=|work=|publisher=价值中国网}}</ref>。 2011年8月，因年龄到限，白恩培卸任云南省的领导职务； 8月28日，在[[第十一届全国人民代表大会常务委员会|第十一届全国人大常委会]]第二十二次会议上，被增补为[[全国人民代表大会环境与资源保护委员会|全国人大环境与资源保护委员会]]副主任委员<ref>[http://news.xinhuanet.com/politics/2011-08/28/c_121922386.htm 全国人大常委会表决通过有关任免案，新华网，2011-08-28]</ref>。2013年3月，当选[[第十二届全国人民代表大会常务委员会|第十二届全国人大常委会]]委员，继续担任环境与资源保护委员会副主任委员<ref name="diy">[http://www.court.gov.cn/xwzx/rdzt/2013qglh/ldr/201303/t20130315_182710.html 中华人民共和国全国人民代表大会公告（第一号），中华人民共和国第十二届全国人民代表大会第一次会议主席团，2013年3月14日] {{webarchive|url=https://web.archive.org/web/20130517145057/http://www.court.gov.cn/xwzx/rdzt/2013qglh/ldr/201303/t20130315_182710.html|date=2013年05月17日}}</ref>。

== 涉贪下台 ==
2014年8月29日，[[中国共产党中央纪律检查委员会|中共中央纪律检查委员会]]透过官方网站宣布，白恩培因涉嫌严重违纪违法，目前正接受组织调查。据《财经》发表的《刘汉朋友圈》一文，白恩培与[[刘汉_(四川商人)|刘汉]]关系密切。也有媒体曝出白恩培与2014年7月受处分的原中共[[昆明市|昆明市]]委书记[[张田欣|张田欣]]过从甚密。据媒体披露，2001年时任[[中国共产党四川省委员会|中共四川省委]]书记的[[周永康|周永康]]在调任中央前，曾致电白恩培照顾其子[[周滨|周滨]]公司旗下的矿产生意<ref>{{cite web|url=http://sc.people.com.cn/n/2014/0830/c345454-22159073-3.html|title=中纪委一日查处三名省部级高官 白恩培与刘汉有交际【3】|accessdate=2014-08-30|author=|date=2014-08-30|publisher=人民网|language=}}</ref>。2014年9月26日，云南省第十二届人大常委会第十二次会议决定罢免其第十二届全国人民代表大会代表职务<ref>{{cite news|url=http://www.npc.gov.cn/wxzl/gongbao/2014-12/23/content_1892465.htm|title=第十二届全国人民代表大会常务委员会代表资格审查委员会关于个别代表的代表资格的报告（2014年11月1日第十二届全国人民代表大会常务委员会第十一次会议通过）|author=|date=2014-12-23|work=|newspaper=中国人大网|accessdate=2016-07-03}}</ref>。11月1日，全国人大常委会发布公告，称“依照选举法等有关法律的规定，白恩培的第十二届全国人民代表大会常务委员会委员、全国人民代表大会环境与资源保护委员会副主任委员职务相应撤销”<ref>[http://www.npc.gov.cn/npc/xinwen/2014-11/02/content_1884657.htm 全国人民代表大会常务委员会公告〔十二届〕第八号]</ref>。

2015年1月13日，中纪委发布信息，白恩培严重违纪违法被开除党籍和公职。经查白恩培利用职务上的便利为他人谋取利益，收受巨额贿赂，涉嫌犯罪，移送司法机关<ref>{{cite web|url=http://news.163.com/15/0114/02/AFSSN6370001124J.html|title=白恩培被指"怕媳妇" 周永康曾致电其照顾周滨|accessdate=2015-01-14|author=|date=2015-01-14|publisher=第一财经日报|language=|deadurl=yes|archiveurl=https://web.archive.org/web/20150114100648/http://news.163.com/15/0114/02/AFSSN6370001124J.html|archivedate=2015-01-14}}</ref>。据港媒《[[明報|明报]]》报导，在2015年3月4日的全国政协侨联小组会上，香港[[宣威集团|宣威集团]]董事长浦江在谈到白恩培时说：“（白恩培）家中查到37个亿呀，外面才报一、两个亿！”

2016年6月16日，河南省[[安阳市|安阳市]]中级人民法院公开审理白恩培受贿、巨额财产来源不明一案，他被控受贿2.467亿人民币，另有巨额资产不能说明来源，当庭认罪<ref>{{cite web|url=http://news.sina.com.cn/c/nd/2016-06-16/doc-ifxtfrrc3684767.shtml|title=云南原书记白恩培被控受贿近2.5亿 当庭认罪|accessdate=2016-06-16|author=|date=2016-06-16|publisher=新浪|language=}}</ref>。

2016年10月9日，河南省安阳市中级人民法院宣判，白恩培犯受贿罪、巨额财产来源不明罪，被判处[[死刑缓期执行|死刑，缓期二年执行]]，剥夺政治权利终身，并处没收个人全部财产，在其死刑缓期执行二年期满依法减为无期徒刑后，终身监禁，不得减刑、假释。对白恩培受贿所得财物和来源不明财产予以追缴，上缴国库<ref name=审判/>。

== 家庭 ==

父亲：白炳信，最高官职是西安草滩农场党委副书记。

大伯父：[[白治民|白治民]]，曾任[[中国共产党福建省委员会|中共福建省委]]书记。

妻子：张慧清（第二任？），人称“张姐”，从一名招待所[[服务员|服务员]]升任[[云南电网公司|云南电网公司]]党组书记正厅级干部，但讲话常跑题、口误百出，饱受诟病<ref>{{cite web|url=http://news.163.com/14/0830/00/A4RSJAQH00014SEH.html|title=曝白恩培妻子从服务员变厅官 讲话跑题口误多|accessdate=2014-08-30|author=|date=2014-08-30|publisher=澎湃新闻网|language=|deadurl=yes|archiveurl=https://web.archive.org/web/20140901022451/http://news.163.com/14/0830/00/A4RSJAQH00014SEH.html|archivedate=2014-09-01}}</ref>。在云南，当时流传着一句话：有事找“张姐”，在云南没有“张姐”办不了的事。而张慧清曾向商人索要价值1500万元人民币的手镯<ref>{{cite web|url=http://epaper.jinghua.cn/html/2016-10/18/content_340413.htm|title=白恩培之妻向商人索要天价手镯|accessdate=2016-10-18|author=|date=2016-10-18|publisher=京华时报|deadurl=yes|archiveurl=https://web.archive.org/web/20161019231204/http://epaper.jinghua.cn/html/2016-10/18/content_340413.htm|archivedate=2016-10-19}}</ref>。2014年10月17日被撤销云南省第十一届省政协常委、委员资格<ref>{{cite web|url=http://news.sina.com.cn/c/p/2014-10-19/165731012086.shtml|title=白恩培妻子张慧清被撤销云南政协常委资格|date=2014-10-19|publisher=新浪|accessdate=2014-10-19}}</ref>。
