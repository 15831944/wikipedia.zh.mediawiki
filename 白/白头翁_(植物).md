'''白头翁'''（[[学名|学名]]：''{{lang|la|Anemone chinensis}}''）是一种多年生[[草本|草本植物]]，属[[毛茛科|毛茛科]]<ref>“茛”字汉语拼音为gèn</ref>，其[[根|根]]是常见[[中药|中草药]]之一。

== 形态 ==
全株被白色柔毛。叶基生，有长柄，叶片三全裂，顶片和侧片隔离很远，各片再二回深裂，最终裂片先端屡有一二缺刻；早春开暗紫色花，单生于花茎顶端，外被白毛；[[瘦果|瘦果]]多枚相聚成头状，先端各有被长白毛的花柱，外观像一堆白法。

== 分布 ==
主要分布于[[中国|中国]][[中国东北|东北]]、[[华北地区|华北]]以及[[陕西|陕西]]、[[安徽|安徽]]、[[江苏|江苏]]、[[湖北|湖北]]、[[四川|四川]]等地。

== 药用 ==
{{medical}}
入药的白头翁根有苦味，中医认为其性微寒，可用于清[[火邪|热]]凉血、解毒；主要治疗热毒泻痢、赤痢下血等症。
