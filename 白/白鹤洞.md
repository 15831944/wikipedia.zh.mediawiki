'''白鶴洞'''，俗稱'''山頂'''，是[[廣州市|廣州市]]外郊一個地方，位於[[芳村區|芳村區]]東南，與廣州[[河南_(广州)|河南]]隔江相望。

== 歷史 ==
曾在白鹤洞山顶发现[[汉代|汉代]]木椁墓，出土[[陶器|陶器]]鉴定为[[東漢前少帝|东汉前期]]的器物<ref>[http://www.gzsdfz.org.cn/was5/web/detail?record=121&channelid=47398&templet=gzhj_detail_temp2.jsp 白鹤洞街] 程慧  广州市地方志</ref>。

清代，地域屬番禺縣茭塘司崇文社（鄉）管轄，民國初年屬番禺縣三區管轄。

芳村連帶白鶴洞地區舊時不属于廣州城區，歸郊區管轄，街坊自嘲此地為'''廣州的西伯利亞'''<ref>[http://gd.sina.com.cn/zs/travel/2017-03-10/city-zs-ifychhus0459405.shtml 广州芳村有着一历史浓厚的地方名为白鹤洞]   大洋网-广州日报 2017-03-10</ref>。

1938年10月廣州淪陷前一天，大批市民撤離廣州。由於政府對難民潮處理滯後，教會自發設置收容站、難民所，白鶴洞成為時安置區之一。 時[[培英中學|培英中學]]、[[真光中學|真光女子中學]]均避難香港，留下校舍和留守人員，由[[中華基督教會|中華基督教會]]仁濟堂代管。 時收容難民的場所有<ref>[http://www.cqvip.com/qk/97690x/2015008/87878468504849534856484954.html 抗战时期的广州白鹤洞难民区]  曾巧兰 《文物天地》2015年 第8期 </ref>：
* 培英中學、真光女子中學、協和神學院三個主要場所
* 麥屋（鶴翔路2號，現真光中學北門對面）、譚信屋和另三所尚不具名的洋房
* （後）麥屋和真光中學之間搭建的大片棚區。 
[[太平洋戰爭|太平洋戰爭]]爆發後，日軍拘捕外籍教士，遣散難民。

[[文革|文革]]時期吊劳改犯波及，時郊區白鶴洞人民公社禮堂前的墟鎮街口，也吊了兩具死屍，旁貼紙'''死不悔改的地主仔'''，實際只是在地的無辜村民<ref>文革中广州街头“吊劳改犯事件”调查 谭加洛</ref>。

== 產業 ==
共和國後成為廣州市的重要工業基地之一。自1950年代起，中央、省、市在白鶴洞興建一批大中型工廠，主要工廠有：
* 廣州造船廠、廣州鋼鐵廠、廣州製藥廠、廣州自行車二廠、廣州制漆廠、廣州打火機廠、廣州起重設備廠、廣州光學材料廠、廣州宏達毛棉製品廠、廣州白鶴洞水廠等<ref>芳村區志·街鎮·白鶴洞街</ref>

== 規劃 ==
2013年，[[广钢集团|廣鋼]]開始整體搬遷至[[湛江市|湛江]]的計畫。到2016年4月，大部分廠區清空籌備[[廣鋼新城|廣鋼新城]]的建設，殘餘廣鋼高爐、橋吊、鐵路等工業遺跡。廠區大部分轉予[[廣州市土地開發中心|廣州市土地開發中心]]，按照政府規劃保留下來的工業遺跡，將改造為廣鋼新城中軸公園的一部分<ref>[http://news.ifeng.com/a/20160502/48655007_0.shtml 白鹤洞：钢城已落幕 新城正崛起]  羊城晚报 2016-05-02

</ref>。
