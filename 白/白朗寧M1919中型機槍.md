'''白朗寧M1919'''（{{lang|en|'''M1919 Browning machine gun'''}}），俗稱（{{lang|en|'''Browning Machine Gun'''，'''30 Cal''' }}），是由[[約翰·白朗寧|約翰·白朗寧]]在[[一戰|一戰]]後設計的[[機槍|機槍]]，主要是把水冷式M1917改為風冷式，採用[[.30-06春田步槍彈|.30-06 Springfield]]  步槍彈藥，被美國等多個國家用於[[中型機槍|中型機槍]]、[[同軸武器|同軸武器]]及[[航空機槍|航空機槍]]等用途。此機槍在[[二戰|二戰]]及[[朝鮮戰爭|朝鮮戰爭]]中被廣泛使用，當時已推出至M1919A4，同時美國人也把M1919A4加上兩腳架、提把和[[槍托|槍托]]變成M1919A6輕機槍，但和其他機槍相比仍然顯得笨重，因為槍身不計彈鏈已重達14.7公斤，故在1950年代後期被[[M60通用機槍|M60通用機槍]]取代。在部分[[北約|北約]]成員國及其他國家中則服役至六、七十年代，甚至在某程度上仍在使用。

== 变形 ==
=== AN/M2航空機槍 ===

[[File:Douglas_SBD-3_VB-8_May_1942.jpg|thumb]]
[[File:PBY_Gun_Blister.jpg|thumb]]的後部防衛機槍塔]]

白朗寧生產了一種特別的M1919A4的[[航空機槍|航空機槍]]變形，型號名'''AN/M2'''，這種航空機槍使用'''[[.30卡賓槍彈|.30口徑子彈]]'''，同時槍管和供彈機壁也變得更薄以降低重量。為了不與[[M2重機槍|M2重機槍]]的航空機槍衍生型（也被稱為AN/M2，型號全名Browning Machine Gun, Cal. .50, M2, Aircraft）發生混淆，AN/M2 .30航空機槍的型號全名為'''Browning Machine Gun, Cal. .30, M2, Aircraft'''。在[[第二次世界大戰|第二次世界大戰]]初期的美國戰機上大量採用AN/M2 .30航空機槍。但隨著戰爭的發展，使用.30口徑子彈的武器越來越多地被作為訓練用，因此隨後被12.7毫米口徑的的M2重機槍的輕型槍管型號替代。另外'''[[柯爾特公司|柯爾特公司]]'''還生產了一種AN/M2 .30航空機槍的民用型，以'''MG40'''的名稱投入民間市場。

=== 白朗寧.303 MkII航空機槍 ===

[[File:Mémorial_du_Souvenir_08.JPG|thumb]]
[[File:RAF_Museum_Cosford_-_DSC08412.JPG|thumb]]早期型的火力是機翼8挺白朗寧.303 MkII航空機槍]]

'''[[英國皇家空軍|英國皇家空軍]]'''採用白朗寧M1919機槍後，皇家空軍M1919機槍的生產就交由[[維克斯-阿姆斯特朗公司|維克斯-阿姆斯特朗公司]]和伯明瀚輕兵器公司，并定型為'''白朗寧.303 MkII航空機槍'''。白朗寧.303 MkII航空機槍發射「[[英式.303口徑子彈|英式.303口徑子彈]]（英式7.7毫米口徑子彈）」，使用液壓開火，射速1150發/每分鐘，槍口初速將近811米/秒。該航空機槍被皇家空軍在戰鬥機機翼上，但也用做轟炸機和偵察機的機頭機槍。在戰爭臨近結束的時候，轟炸機上的機頭機槍被M2重機槍替換。<ref>[http://www.historyofwar.org/articles/weapons_browning_mkII_raf.html 白朗寧.303 MkII航空機槍]</ref><ref>[http://www.aviation-history.com/guns/303.htm 白朗寧.303 MkII航空機槍]</ref><ref>[http://www.saairforce.co.za/the-airforce/weapons/23/0-303-inch-browning-machinegun .303白朗寧機槍]</ref>

===Mk 21 Mod 0機槍===
[[File:Browning_M1919_Cal_.30.jpg|thumb]]期間的Mk 21]]
美國人在[[越南戰爭|越南戰爭]]中的涉入加深為輕兵器創造了需求，尤其是新型的[[M60通用機槍|M60通用機槍]]。而美國海軍在[[第二次世界大戰|第二次世界大戰]]和[[朝鮮戰爭|朝鮮戰爭]]後遺留了大量的機槍，但使用的都是更早的[[.30-06春田步槍彈|.30-06 Springfield]]子彈，而不是新型標準彈藥 - [[7.62×51mm_NATO|7.62×51mm NATO]]子彈。'''Mk 21 Mod 0機槍'''由此改進而來，這種機槍經過改進後能過發射[[7.62×51mm_NATO|7.62×51mm NATO]]子彈。主要改進為：機槍槍管、槍機、受彈機蓋被更換，增加了彈鏈退卸機、第二個彈鏈扣勾等，並在供彈裝置的前後增加了墊片，用以引導較短的[[7.62×51mm_NATO|7.62×51mm NATO]]子彈，阻止更長的[[.30-06春田步槍彈|.30-06 Springfield]]子彈的使用。此外，在槍口還增加了一個6英寸長的{{Link-en|槍口消焰器|Flash Suppressor|消焰器}}來降低射擊[[7.62×51mm_NATO|7.62×51mm NATO]]子彈時產生的{{Link-en|槍口焰|Muzzle flash}}。<ref name="warboats.org">[http://www.warboats.org/stonerordnotes/Mk%2021%20Mod%200%20R4.html '''Ordnance Notes: Mk 21 Mod 0 7.62mm Machine Gun''' by Bob Stoner GMCM (SW) Ret.]</ref>
1966～1967年间，改進後的機槍進行了演示。M1919A4機槍改進後被重新命名為'''Mk 21 Mod 0 7.62mm機槍'''，並在供彈機側蓋上用1/4英寸大小的字母刻上型號名“Machine Gun, 7.62mm / Mk 21 Mod 0”，更換後的槍管則用1/8英寸大小的字母刻上“7.62mm NATO-G”，以區別M1919A4機槍或M60機槍槍管，其中字母“G”代表其使用的帶凹槽的{{Link-en|槍管隔熱罩|Barrel shroud|槍管套}}。

Mk 21機槍供彈原理修改後衹能左側供彈。它同[[M60通用機槍|M60通用機槍]]不同的一點是其彈鏈必須背面朝上，這樣機槍才能抽出彈鏈。為了準備彈藥，機槍手不得不从M19A1彈藥箱取出兩條100发的彈鏈，再同時把它們連在一起，最後把連好的200发彈鏈倒置裝回M19A1彈藥箱，以確保其能夠正常供彈。<ref name="warboats.org"/>

Mk 21機槍使用'''[[M13彈鏈|M13彈鏈]]'''和'''M1彈鏈'''。由於美國海軍的[[7.62×51mm_NATO|7.62×51mm NATO]]子彈庫存量減少，因此美國海軍使用Mk 21機槍時通常選擇M80 7,62mm[[穿甲弹|穿甲弹]]或M80 7,62mm[[穿甲弹|穿甲弹]]和M62 7,62mm曳光彈以4:1的比例混合。

==M1919在中國==
[[File:HAFm_Browning_30_cal_7089.JPG|thumb]]
[[中華民國陸軍|中華民國陸軍]]主力武器並非美製機槍，早期M1919主要用戶是[[中華民國空軍|中華民國空軍]]，由於南京國民政府在1933年與美商合資興建[[中央飛機製造廠|中央飛機製造廠]]組裝各式美製飛機，也一併將美系機槍帶入空軍武備清單之列，M1919機槍幾乎是1930年代採購的所有美製戰機標準武器。也因此在7.92公厘子彈大興其道的1930年代中國，只有空軍使用7.62公厘子彈與美國原廠M1919機槍；而後美軍正式向軸心國開戰後，由於戰機上配備的多半是[[白朗寧M2重機槍|白朗寧M2重機槍]]，M1919在空軍規模上也日趨萎縮。

中華民國陸軍開始使用M1919始於1944年美軍協助訓練[[中華民國空降特戰部隊|傘兵]]，因空降部隊不適合攜帶水冷式的[[二四式重機槍|二四式重機槍]]，美軍軍援一批M1919機槍配備給傘兵團，即便是美械部隊也屬罕見的武器；在豫東會戰時，國軍傘兵編成第三快速縱隊隸屬第七兵團，歸黃百韜指揮。在帝邱店戰鬥時，均有傑出表現。

國民政府敗退臺灣重整之際，也從美國獲得M1919機槍撥交給陸軍步兵單位運用，變成空軍淘汰但陸軍接續運用的狀況；[[國防部軍備局生產製造中心第205廠|兵工署60廠]]在1961年生產仿M1919A6的50式輕機槍，但因科技落後沒有大量配發。1961年，在採購美國機具生產的M60機槍（[[57式機槍|57式機槍]]）成軍後，M1919機槍遭到取代，但是金馬外島防禦據點的M1919可考者至少到1990年代仍在編裝列內。

==流行文化==
M1919曾出現在二戰後拍攝的眾多電影、電視劇和遊戲當中，例如：
*《[[太平洋戰爭_(電視劇)|-{zh-hans:太平洋 (电视剧);zh-hk:雷霆戰海;zh-tw:太平洋戰爭;}-]]》

*《[[決勝時刻：戰爭世界|-{zh-hans:使命召唤：战火世界;zh-hk:使命召喚：戰爭世界;zh-mo:使命召喚：戰爭世界;zh-tw:決勝時刻：戰爭世界;}-]]》：命名為“勃朗寧M1919”，單機模式中被[[美国海军陆战队第一师|美国海军陆战队第一师]]所使用。聯機模式中可被所有陣營使用。
《[[魔鬼終結者3|魔鬼終結者3]]》（英語：Terminator 3：Rise of the Machines，終結者T-850([[阿諾德·施瓦辛格|阿諾德·施瓦辛格]]飾演)在墓園扛著棺材攻擊[[警察|警察]][[警車|警車]]時所使用
