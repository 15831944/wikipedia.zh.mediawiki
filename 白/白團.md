'''白團'''為[[臺灣戰後時期|臺灣戰後時期]]的[[日軍|日軍]][[顧問團|顧問團]]，1949年成立於[[日本|日本]][[東京|東京]]，並隨即到[[臺灣|臺灣]]幫助[[中華民國政府|中華民國政府]]抵抗[[中華人民共和國政府|中華人民共和國政府]]，與1951年成立的[[美軍顧問團_(中華民國)|美軍顧問團]]，及1936年成立的[[德國軍事顧問團|德國軍事顧問團]]並列[[國民政府|國民政府]]三大外國軍事顧問團。[[古寧頭戰役|古寧頭戰役]]與[[八二三砲戰|八二三砲戰]]的勝利，與三大外國軍事顧問團密切配合有關。

== 歷史 ==
=== 岡村寧次 ===
1945年[[第二次世界大戰|第二次世界大戰]][[日本投降|日本投降]]後，日本[[參謀本部|參謀本部]]次長[[河邊虎四郎|河邊虎四郎]]要求[[支那派遣軍|在華方面軍]]總司令[[岡村寧次|岡村寧次]]，與在[[山西|山西]]的[[第1軍_(日本陸軍)|第一軍]]司令官[[澄田𧶛四郎|澄田𧶛四郎]]，與[[國民政府|國民政府]]軍最高指揮官[[何應欽|何應欽]]和將領[[閻錫山|閻錫山]]締結「共同打擊共軍」的祕密軍事協定。其中岡村寧次與何應欽之密約在[[芷江|芷江]]簽訂，被稱為「[[芷江協定|芷江協定]]」。而岡村寧次便出任「中國戰區日本官兵善後聯絡部」[[長官|長官]]，並且出任[[國民革命軍|國民革命軍]]參謀。

=== 成立 ===
1949年7月，岡村寧次與澄田睞四郎、[[十川次郎|十川次郎]]（前[[第6軍_(日本陸軍)|第六軍]]司令官）商議，募集舊日軍兵團參謀或連隊長級軍官[[富田直亮|富田直亮]]等十七名，在[[東京|東京]]組團（9月1日，後稱「白團」），其中[[首領|首領]]富田直亮化名為「白鴻亮」，1949年11月1日抵達[[台北|台北]]，其他成員隨後也陸續抵達。也因[[領袖|領袖]]為「白鴻亮」，故稱「白鴻亮軍事顧問團」，簡稱白團，而且「白團」之「白」字，恰好與「[[中国工农红军|紅軍]]」之「紅」字對抗，意為對抗以[[中共|中共]]為首的「赤魔」。

1950年2月「圓山軍官訓練團」成立，由[[彭孟緝|彭孟緝]]擔任教育長，由「白團」成員對[[中華民國國軍|國軍]]校級軍官及將領開班授課，進行軍事教育與訓練，同時協助蔣中正擬定各項軍事計畫<ref name="世界日報">[http://www.worldjournal.com/view/full_news/15322386/article-%E6%97%A5%E8%BB%8D%E5%AE%98%E7%B5%84%E3%80%8C%E7%99%BD%E5%9C%98%E3%80%8D-%E5%8A%A9%E8%94%A3%E4%BB%8B%E7%9F%B3%E4%BD%9C%E6%88%B0?instance=m2b 日軍官組「白團」 助蔣介石作戰]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }} 2011-09-04 世界日報</ref>。到1951年，白團的教官增加至八十三人<ref>[http://news.chinatimes.com/reading/0,5251,110513x112010121300106,00.html 日少將密訓國軍 骨灰一半留台 「白團」團長白鴻亮 在台協訓國軍20年 包括[[胡璉|胡璉]]、[[郝柏村|郝柏村]]、[[蔣緯國|蔣緯國]]等都曾受其指導 為我首位外籍陸軍上將｜藝文新聞｜中時電子報]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，都是過去日軍少將至少佐級中堅核心精英，實力相當於戰前日軍三個師團的腦力。

白團的成立除了[[蔣介石|蔣介石]]曾經留學[[日本|日本]][[東京|東京]][[振武學校|振武軍校]]，對日本有特殊的情感外，蔣也深知[[中國抗日戰爭|中國抗日戰爭]]若無[[美軍|美軍]]的介入根本不可能獲勝，但是在中國抗日戰爭、[[國共內戰|國共內戰]]與[[美國|美國]]的交涉經驗，使他理解美國對中國政策依歸於美國利益，必要時美國隨時會因利益轉向而放棄[[中華民國|中華民國]]的不確定感上，使他寄望與在[[反共|反共]]傾向與國民政府接近的[[大日本帝國|大日本帝國]]軍官，借重他們的力量來進行軍隊的再教育，加上[[岡村寧次|岡村寧次]]等日本[[將軍|將軍]]過去在[[中國大陸|中國大陸]]與[[中國人民解放軍|解放軍]]作战较有经验，所以为了实现[[反攻大陸|反攻大陸]]，必須要借重日本[[軍人|軍人]]的力量，白團於焉成立。

=== 實踐學社 ===
白團另有一微妙考量，蔣介石藉著引入日本的力量來制衡具有強烈[[美國|美國]]背景、並曾一度獲得美國支持取代蔣的[[中華民國陸軍司令|陸軍總司令]][[孫立人|孫立人]]<ref name="世界日報"/>。1951年[[美軍駐台|美軍駐台]]後，首任[[美軍顧問團_(中華民國)|美軍顧問團]]團長——[[美國陸軍|美國陸軍]]少將[[蔡斯_(美國陸軍將領)|蔡斯]]反對日本軍官擔任國軍顧問，1952年白團因此縮小規模、駐在地由[[圓山|圓山]]轉往[[石牌_(臺北市)|石牌]]，趨向地下化，並改以「[[實踐學社|實踐學社]]」的名稱，對高階[[將領|將領]]進行講習、訓練<ref>“1954年美国与国府签署“华美防卫条约”后，美国重新派军事顾问到台湾。美国军事顾问对蒋介石启用旧日本军人做军事教官十分不快，要求蒋介石辞退日本教官，蒋介石说：“在你们抛弃我们的时候，是他们冒着生命危险来帮助我们。在我还活着的时候，不会辞退他们的。””（[[林思云|林思云]]《蒋介石的眼泪》）</ref><ref name = "郝柏村曾受訓  新書揭白團神秘面紗   藝文   新頭殼 Newtalk_2015"> {{ cite web | title = 郝柏村曾受訓  新書揭白團神秘面紗   藝文   新頭殼 Newtalk | url = https://newtalk.tw/news/view/2015-01-14/55845 | publisher = 新頭殼newtalk | author = 莊豐嘉台北報導 | language = zh-tw | date = 2015-01-14 }} </ref>。實踐學社之總教官為白鴻亮（富田直亮），副總教官為帥本源（[[山本親雄|山本親雄]]）、范健（[[本鄉健|本鄉健]]），其他教官皆由過去的日籍[[軍官|軍官]]中選聘。並以當時中華民國陸軍32師為實驗對象，進行訓練計畫。

=== 黨政軍聯合作戰研究班 ===
1952年12月到1963年12月，白團為國民黨訓練軍政高級幹部「黨政軍聯合作戰研究班」共11期，主要將領有[[郝柏村|郝柏村]]、[[宋長志|宋長志]]、[[蔣緯國|蔣緯國]]、[[胡璉|胡璉]]、[[許歷農|許歷農]]、[[宋心濂|宋心濂]]。<ref name = "台前幕後: 1949_2017"> {{ cite book | title = 台前幕後: 1949 | url = https://books.google.com.tw/books?id=nadBDwAAQBAJ&pg=PT217&dq=%E9%83%9D%E6%9F%8F%E6%9D%91+%E7%99%BD%E5%9C%98&hl=zh-TW&sa=X&ved=0ahUKEwj6582AoIjdAhWMdt4KHdt8AAkQ6AEIPzAE#v=onepage&q=%E9%83%9D%E6%9F%8F%E6%9D%91%20%E7%99%BD%E5%9C%98&f=false | publisher = 九州社 崧博出版社, 2017 | author = 作者：馬振犢 主編 | isbn = 9789869587631 | language = zh-tw | date = 2017年12月5日 }} </ref>

==== 肯定郝伯村 ====
1959年2月4日戶梶在視察小金門，時任陸軍第9師師長郝柏村將軍向戶梶進行防衛作戰的報告後，戶梶日記中記錄感想：郝師長的思考方向相當明確，而且值得信賴。擔任孤島的獨立司令官，他可說是再適合不過了。<ref name = "最後的帝國軍人：蔣介石與白團: ラスト・バタリオン 蒋介石と日本軍人たち_2015"> {{ cite book | title = 最後的帝國軍人：蔣介石與白團: ラスト・バタリオン 蒋介石と日本軍人たち | url = https://books.google.com.tw/books?id=c44fBwAAQBAJ&pg=PA275&dq=%E9%83%9D%E6%9F%8F%E6%9D%91+%E7%99%BD%E5%9C%98&hl=zh-TW&sa=X&ved=0ahUKEwj6582AoIjdAhWMdt4KHdt8AAkQ6AEIMzAC#v=onepage&q=%E9%83%9D%E6%9F%8F%E6%9D%91&f=false | publisher = 聯經出版事業公司, 2015 | author = 作者：野島剛（Nojima Tsuyoshi） | isbn = 9789570845075 | page = 275 | language = zh-tw | date = 2015-01-08 }} </ref>

=== 高級兵學研究班 ===
1963年舉辦「高級兵學研究班」。白鴻亮教學嚴格，曾經以「過去日軍在與中國軍隊戰鬥時，一個營的日軍可以對抗一個師的中國軍」對學員訓話。1965年9月1日起，白團因為人數減少被迫縮編為成員江秀坪、喬本、賀公吉、楚立以及團長白鴻亮等5位顧問的「實踐小組」，聯絡人為蔣緯國。白團後於1968年撤出台灣，直到1969年底只剩白鴻亮團長一人，工作項目改名為「實踐專案」；在實踐專案中，蔣緯國從學員學習成長到後期成為負責人。

=== 解散 ===
1969年2月1日於[[東京|東京]]解散，不少成員復入[[日本自衛隊|日本自衛隊]]，將對[[共產黨|共產黨]]作戰的經驗帶回日本。

白鴻亮團長在年老體衰後返回日本，過世前囑咐妻子將一半骨灰安置台灣，目前放置在台灣新北市樹林的海明寺中。到了2017年白團成員還有36人在世，在日本仍會以「岡村寧次同志會」名義進行聯誼聚會。<ref name = "台前幕後: 1949_2017" />

==根本博事件==
眼見[[蔣介石|蔣介石]]在[[國共內戰|國共內戰]]中兵敗如山倒，[[根本博|根本博]]與[[岡村寧次|岡村寧次]]等人有志一同，都願意幫助蔣介石，以合作共同[[反共|反共]]，此即為白團（日軍顧問團）的雛型。

1949年，根本博率領[[吉川源三|吉川源三]]、[[淺田哲|淺田哲]]等七名日軍軍人，至[[臺灣|臺灣]]拜會蔣介石，意欲提供協助，蔣託付根本博等人至[[金門|金門]]參與[[古寧頭戰役|古寧頭戰役]]，為[[國軍|國軍]]制定謀略。根本博不負所託，以[[口袋戰術|口袋戰術]]打敗了來犯的[[中國人民解放軍|解放軍]]部隊<ref name = "青天白日下的軍魂_2016" />，蔣介石特別嘉許根本博，還致贈了自己珍視的花瓶，該花瓶本為[[總統府_(臺灣)|總統府]]中一對花瓶的其中一隻，是[[景德鎮|景德鎮]][[官窯|官窯]]所燒製，同樣形式的花瓶，全世界只有六隻。

當時[[中國國民黨|國民黨]]與岡村寧次有意招募舊日軍人員，來防禦解放軍對臺的進攻；但根本博高調為蔣參戰，導致事跡敗露，日本社會動盪，後以「臺灣募兵問題」此一名稱被國際媒體廣泛報導後，害得岡村寧次遭到[[駐日盟軍總司令|駐日盟軍總司令]][[麥克阿瑟元帥|麥克阿瑟元帥]]的斥責。根本博秘密來臺的事件也遭[[日本國會|日本國會]]查問，當時的[[日本首相|日本首相]][[吉田茂|吉田茂]]，在[[國會|國會]]答詢時言詞閃爍，模擬兩可<ref>第6回国会 衆議院 本会議 第7号 昭和24年（1949年）11月10日（[http://kokkai.ndl.go.jp/SENTAKU/syugiin/006/0512/00611100512007c.html 議事録]）</ref>。加上吉川源三又涉嫌侵吞國民黨給白團的款項，鬧得風風雨雨，[[美軍|美軍]]特別注意白團的動態，而後根本博遭到白團中岡村等人的排斥，未能加入白團。

== 影響軍事思想與制度 ==
1949年曹士澂發動白團以外力協助保衛台灣重整國軍，白團在台灣訓練中華民國國軍長達20年，總受訓人數達到2萬多人，白團解散後1968年12月中華民國國防部編纂「日本軍事顧問團教官在華工作紀要」感謝白團對中華民國國軍的作戰立案、軍事教育與戰術思想等貢獻；白團訓練中華民國國軍內容包括：反瞻陣地、火網交叉、火力配備、宿題作業與預備隊等軍事訓練。<ref name = "青天白日下的軍魂_2016" />

白團的思想影響[[中華民國軍事|中華民國軍事]]甚鉅，臺灣曾長久實施的[[義務役|義務役]][[徵兵制|徵兵制]]、[[預官|預官]]制度即為白團之提議，使每個[[成年|成年]][[男子|男子]]皆入伍服[[徵兵制#中華民國|義務役]]，以具備足夠兵力[[反攻大陸|反攻大陸]]。各種軍事教範、準則，亦多出於白團之手，本為[[日文|日文]]，再譯成[[中文|中文]]，如新兵訓練實施的「[[刺槍術|刺槍術]]」、「單兵戰鬥教練」之準則皆為白團所創。[[中華民國國軍|中華民國國軍]][[軍官|軍官]]，受過白團[[軍訓|軍訓]]者中，以中華民國前[[參謀總長|參謀總長]][[郝柏村|郝柏村]]最為有名，郝柏村為白團「黨政軍聯合作戰研究班軍事組」學員之一。<ref>[http://news.chinatimes.com/showbiz/100103/112012123100076.html 異人的足跡 白團也有份] {{webarchive|url=https://web.archive.org/web/20140201183757/http://news.chinatimes.com/showbiz/100103/112012123100076.html |date=2014-02-01 }}2012-12-31 《[[中國時報|中國時報]]》 李明賢／台北報導 </ref><ref name = "台前幕後: 1949_2017" />
