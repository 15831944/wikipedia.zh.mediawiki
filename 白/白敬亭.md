'''白敬亭'''（{{bd|1993年|10月15日|catIdx=B白}}），[[中國|中國]]新生代[[男演員|男演員]]，[[滿族人|滿族人]]，出生於[[北京|北京]][[懷柔區|懷柔區]]，畢業於[[首都師範大學|首都師範大學]]音樂學院的錄音藝術專業，精通鋼琴。2014年，因參演由小說改編的網路劇《[[匆匆那年_(網路劇)|匆匆那年]]》飾演男二號喬燃一角而出道，引起關注。2016年，主演首部電影《[[谁的青春不迷茫_(電影)|谁的青春不迷茫]]》，同年因錄製《[[明星大偵探|明星大偵探]]》獲好評，於後期更成為固定嘉賓出演。

== 簡歷 ==
=== 背景 ===
白敬亭初中時特別喜歡體育，熱愛籃球，主攻鉛球、跳遠和跳高，不愛學習的他中考只考了300多分，面臨沒學校上的境地，但因從小學鋼琴，基礎良好，首都師範大學附屬紅螺寺中學破格以比較低的分數錄取他<ref>{{Citation|last=优酷全明星|title= 【正片】白敬亭寻求突破挑战新角色|date=2016-04-01|url=http://v-wb.youku.com/v_show/id_XMTUxODY0Mjc2OA==.html}}</ref>
。上了高中他几乎戒掉了一切玩乐，专心学习，最终以文科班第二的成绩考上了首都师范大学。大學並非表演科班出身的白敬亭，起初他想為自己一直以來學習的音樂行業充電，但上了大學後才發現和自己的想像還是有些出入的，想起高中時曾萌生想成為演員的想法，於是積極尋找能成為演員的機會，更曾投簡歷到韓國經紀公司。

=== 名字由来 ===
白敬亭的名字很文艺范，在一次专访中得知这个名字是来自[[李白|李白]]的一首诗—独坐敬亭山，这里的“敬亭”就是白敬亭名字的由来。而根据排辈儿，白敬亭那代是“岩”字辈儿的，爷爷称：男的就叫“白岩龙”女的叫“白岩凤”而白敬亭的父亲还是坚持了自己的决定——白敬亭这个名字就是这么来的，寓意是希望男人像山一样，靠得住<ref>[http://news.ifensi.com/article-14-3816182-1.html 粉丝网专访白敬亭:塑造不一样的角色]</ref>。

=== 演藝生涯 ===
2014年，大學三年級的白敬亭看到[[搜狐視頻|搜狐視頻]]網路劇《[[匆匆那年_(網路劇)|匆匆那年]]》劇組的演員面試的信息，抱著試試看的心態參加面試。<ref>{{Cite web |url=http://yule.sohu.com/20140626/n401430619.shtml |title= 《匆匆那年》主演曝光 暖男白敬亭逗乐全场|date=2014-06-28 |publisher= 搜狐娱乐|language= zh-hans}}</ref>，但因表演生澀幾乎落選，再多次試戲後劇組發現白敬亭學習能力強，導演姚婷婷說：「白敬亭最打動人的不是演技，而是目光的純淨。」白敬亭最終憑自身努力與符合角色的氣質飾演男二號喬燃，成為出道代表作品。同年客串電視劇《極品女士第三季》。

2015年3月，白敬亭主演兩部微電影，分別是與[[關曉彤|關曉彤]]合作出演國內首部魔幻愛情攻略新媒體電影《魔法觸戀》<ref>{{Cite web |url=http://ent.qq.com/a/20150107/051673.htm#p=1 |title= 《魔法触恋》即将开机 关晓彤白敬亭牵手新作|date=2015-01-07 |publisher= 腾讯娱乐|language= zh-hans}}</ref>和《無二之旅》。

2015年7月，搭檔[[楊洋|楊洋]]、[[胡冰卿|胡冰卿]]等出演據[[明曉溪|明曉溪]]小說改編電視劇《[[旋風少女|旋風少女]]》<ref>{{Cite web |url=http://ent.sina.com.cn/v/m/2015-03-24/doc-iawzuney1032261.shtml |title= 《旋风少女》曝阵容 胡冰卿搭杨洋白敬亭|date=2015-03-24 |publisher= 新浪娱乐|language= zh-hans}}</ref>，在劇中飾演喻初原。與《[[匆匆那年_(網路劇)|匆匆那年]]》中相比，白敬亭拍《[[旋風少女|旋風少女]]》並不算是一帆風順的。從最開始的定妝照形象被吐槽等事件，白敬亭承受了很多的壓力，為了符合劇中初原的形像，白敬亭在短時間內減重10公斤，雖在劇中沒有太多動作戲，但也全程跟著其他演員一起練習。<ref>{{Cite web |url=http://ent.sina.com.cn/s/m/2016-08-04/doc-ifxutfpc4420698.shtml |title= 白敬亭赴韩国接受专访：我的青春不迷茫|date=2016-08-04 |publisher= 新浪娛樂|language= zh-hans}}</ref>

2015年9月，白敬亭坦言當時對表演失去信心，需要時間沉澱，於是決心參與[[東方衛視|東方衛視]]大型自然旅遊紀實真人秀《跟著貝爾去冒險》錄製，需要在野外生存15天，不允許帶任何東西，沒有經紀人和助理，一個讓明星在封閉艱苦的條件下成長，節目於同年10月16日播出。因節目中有從直升機跳水環節，意外曝露白敬亭曾耳朵受傷，耳膜穿孔，有中耳炎的病歷，更需配戴耳塞參與該環節。

2016年，主演的青春電影《[[誰的青春不迷茫|誰的青春不迷茫]]》於4月22日上映，這也是其主演的首部電影作品，為了專心拍攝，甚至在一年多的時間內沒有接戲。

2016年4月，參加中國首檔明星[[推理|推理]][[真人秀|综艺秀]]《[[明星大偵探|明星大偵探]]》，原只作為嘉實宣傳電影作品而出演，但播出後在節目中的亮眼表現受到節目組與觀眾的愛戴，小白發揮了自己性格中活潑的一面，為自己的綜藝市場打開了良好的局面，人氣急升接連錄製節目，更於第一至四季成為節目的常駐嘉賓，與[[何炅|何炅]]、[[撒贝宁|撒贝宁]]、[[吳映潔|吳映潔（鬼鬼）]]和[[王鸥|王鸥]]共同主持。

2016年7月，攜手[[王凱|王凱]]、[[井柏然|井柏然]]、[[蕭敬騰|蕭敬騰]]、[[楊爍|楊爍]]和[[王嘉爾|王嘉爾]]一起參加由[[江蘇衛視|江蘇衛視]]綜藝《[[我們戰鬥吧|我們戰鬥吧]]》。

2016年8月，搭檔[[陳學冬|陳學冬]]、[[郑爽|鄭爽]]拍攝改編自作家[[郭敬明|郭敬明]]的同名小說的《[[夏至未至|夏至未至]]》。

2017年5月，白敬亭接拍首部[[古裝電視劇|古裝電視劇]]《[[天盛長歌|天盛長歌]]》，飾演武林高手顧南衣，拍攝長達半年，與[[陳坤|陳坤]]和[[倪妮|倪妮]]合演。白敬亭表示，由於認為自己臉型長相不適合古裝，有點圓潤，跟自己角色相差較大，於是花了很長時間減脂，讓自己臉上棱角分明。

2018年1月，結束古裝劇拍攝後，加入《[[二十四小时_(综艺节目)|二十四小時]]》第三季成為固定成員，與[[林志穎|林志穎]]、[[余文樂|余文樂]]、[[魏大勛|魏大勛]]、[[胡一天|胡一天]]、[[熊梓淇|熊梓淇]]和[[林允|林允]]主持。

2018年3月，与[[孙怡|孙怡]]一同主演的青春励志电视剧《[[初晨，是我故意忘记你|初晨，是我故意忘记你]]》在杭州开机，该剧改编自籽月同名小说。

2018年6月，改编自韩国漫画《[[未生|未生]]》的现实主义职场剧《[[平凡的荣耀|平凡的荣耀]]》在上海开机，由白敬亭与[[赵又廷|赵又廷]]担纲主演。
