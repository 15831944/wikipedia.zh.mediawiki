'''白石'''（[[英語|英語]]：White Rock）是[[加拿大|加拿大]][[卑詩省|卑詩省]][[大溫哥華地區|大溫哥華地區]]的一個城市。它位於[[大溫地區|大溫地區]]南部，南臨[[:en:Semiahmoo_Bay|森美亞姆灣]]，其餘三方則被[[素里|素里]]包圍，而[[美加邊境|美加邊境]]亦近在咫尺。據2011年人口普查，白石約有19,339名居民。<ref name="StatsCan2011"/>

== 名稱 ==
[[File:thewhitreock.jpg|thumb]]

白石市的名稱來自市内海濱一顆白色巨礫。此巨礫是一顆[[:en:glacial_erratic|冰河迷子石]]，於[[末次冰期|末次冰期]]從北方南移至此，約486噸重。它長年累月受[[海鳥糞|海鳥糞]]覆蓋而呈白色，到19世紀更被海員用作地標。巨礫於過去一世紀成為塗鴉人士的目標，現時由白石市公園局不時鬆上白色油漆以保持其外貌。<ref name="RockOrigin">[http://www.whiterock.museum.bc.ca/downloads/Origins%20of%20white%20rock.pdf Theories abound on white rock's origin] {{webarchive|url=https://web.archive.org/web/20111227230757/http://www.whiterock.museum.bc.ca/downloads/Origins%20of%20white%20rock.pdf |date=2011-12-27 }}，《Peace Arch News》，2010年4月9日。於2010年6月17日查閱。</ref>

巨礫於不少當地[[:en:Coast_Salish|薩利希原住民族]]的傳統故事中出現，其中一個傳説指一名居於[[溫哥華島|溫哥華島]]的海神與一個原住民少女墮入愛河，但他們的戀情被雙方家長反對。海神於是把一顆巨礫掉到[[喬治亞海峽|喬治亞海峽]]對岸，再和他的新娘遷至石頭著地的地方（即白石市所在）落地生根，而該傳説則指該帶的森美亞姆族人為他們的後人。<ref name="RockOrigin"/>

== 歷史 ==

現白石市一帶是海峽薩利希原住民族的活躍範圍。歐洲人於1791年首次航行至此時已發現[[:en:Semiahmoo_First_Nation|森美亞姆族人]]的永久性營地。

[[美國|美國]]和[[英國|英國]]於19世紀初葉就[[:en:Oregon_Country|俄勒岡地區]]的邊界劃分出現[[:en:Oregon_boundary_dispute|紛爭]]，擾攘多年後隨著兩國於1846年簽訂[[俄勒岡條約|俄勒岡條約]]而告一段落。兩國同意以北緯49度為界，而經過[[美加邊境|國際邊界勘測委員會]]進行勘測後，現白石市一帶劃分英國管轄。[[素里|素里]]於1879年設立地方行政架構時的管轄範圍亦包括白石，而歐洲裔民眾亦從1890年代起到白石一帶定居。<ref name="WRHistory"/>

到了20世紀初葉，大北方鐵路公司（現屬[[:en:BNSF_Railway|BNSF鐵路]]）開始修築一條連接卑詩省和[[華盛頓州|華盛頓州]]的鐵路。鐵路於1909年抵達白石，該處亦被界定為出入境口岸。<ref name="WRHistory"/>鐵路再於1913年在白石設站，吸引不少[[低陸平原|低陸平原]]居民到此度假。<ref>[http://www.virtualmuseum.ca/pm_v2.php?id=exhibit_home&fl=0&lg=English&ex=00000506&pg=0 Virtual Museum Canada]</ref>數座磨坊陸續在區内開業，而長達1616英尺的碼頭則於1915年落成，為船隻提供停靠處之餘亦成為區内休閒活動的重點。

[[二戰|二戰]]過後素里北部陸續發展之際，白石居民卻有感他們的訴求被素里政府忽略，因此居民決定投票脫離素里自組地方政府。白石於1957年4月15日獲省政府發出特許狀，正式設市。<ref name="WRHistory"/>

[[卑詩省99號公路|卑詩省99號公路]]於1960年代開通後，來往白石和[[溫哥華|溫哥華]]以至大溫其他城鎮的車程大為縮減，加上白石的樓價較溫哥華低，吸引不少退休人士和年輕家庭到此置業，白石亦逐漸從大溫地區的避暑勝地轉型為衛星城鎮。

== 交通 ==
白石市内沒有任何省級公路，但[[卑詩省99號公路|卑詩省99號公路]]的走綫離白石的市界綫不遠。這一段99號公路屬高速公路資格，北通[[溫哥華|溫哥華]]和[[威士拿|威士拿]]等地，南端則於[[美加邊境|美加邊境]]駁上[[5號州際公路|5號州際公路]]。

白石的巴士服務是由[[運輸聯線|運輸聯線]]（TransLink）提供。區内沒有[[捷運|捷運]]車站，但居民可乘搭巴士到[[素里市|素里市]]的[[:en:Surrey_Central_Station|素里中央站]]或[[:en:King_George_Station|喬治王站]]，再換乘[[溫哥華架空列車|架空列車]][[博覽線|博覽線]]前往大溫其它地區。運輸聯線過往亦曾經提供連接白石和溫哥華市中心的巴士線，但自從架空列車[[加拿大線|加拿大線]]於2009年通車後，那些巴士線均改以[[列治文|列治文]]的[[:en:Bridgeport_Station_(TransLink)|橋港站]]為北端總站，前往溫市中心的乘客需於該處轉乘加拿大線。
