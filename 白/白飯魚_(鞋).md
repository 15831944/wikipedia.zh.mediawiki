'''白飯魚'''是源自[[香港|香港]]的詞語，是'''白布鞋'''的俗稱。

==名称由来==
白飯魚是白鞋的綽號來，但因粵語「鞋」與「骸」同音，廣東人傳統上對「骸」這個發音有忌諱，不喜事事都「骸骸」聲，因此有說正月（農曆一月）不能買鞋。

而這對還叫白鞋，「白白咁骸！」有些人覺得白鞋全身白色，看上去就像兩條大大的白飯魚，於是便開始給它一個別稱——白飯魚。避免再用白鞋「白骸」這兩字。

久而久之，大家都習慣了叫白鞋做白飯魚了，反而他的原名白鞋就較少人說。<ref>{{Cite web |url = http://hk.knowledge.yahoo.com/question/question?qid=7006081101749 |title = 點解白飯魚(鞋)叫白飯魚?? |author = Yahoo!知識+ |date = 2006年8月11日 |accessdate = 2014年10月28日 |deadurl = yes |archiveurl = https://web.archive.org/web/20141028130745/https://hk.knowledge.yahoo.com/question/question?qid=7006081101749 |archivedate = 2014年10月28日 }}</ref>

==简介==

[[File:Lancelet_shoes01.jpg|thumb]]
在[[香港|香港]]，'''白飯魚'''（Plimsolls/sneakers）是一種廉價[[布鞋|帆布鞋]]的俗稱，通常為出產自[[中國大陸|中國大陸]]。這種鞋有著[[白色|白色]]的[[帆布|帆布]]鞋身和單薄的[[橡膠|橡膠]]鞋底（一般為[[綠色|綠色]]或[[黑色|黑色]]），固得白飯魚此名。

白飯魚一般的售價只需十數港元，故此成為[[中学|中]][[小学_(教育)|小]][[學生|學生]]上[[體育|體育]]課常用的[[運動鞋|運動鞋]]。另外，由於價錢便宜，白飯魚亦是低下階層、[[晨運|晨運]]客、[[地盤|地盤]]和[[裝修|裝修]][[工人|工人]]常穿的便鞋。

另外一個常常使用白飯魚的場合就是中式[[喪禮|喪禮]]。在中式喪禮中，出席者的全身衣服需要是白色。一些人因為本身沒有全白色的鞋（全白色的鞋較易顯得骯髒），所以便臨時買一雙廉價的白飯魚來參加喪禮，完事後棄掉亦不太可惜。

因為白飯魚的結構單薄，除了不能對[[足部|足部]]提供足夠的保護之外，還可能因此引致受傷，所以現時已經愈來愈少人穿著白飯魚來進行認真的運動。

2006年，法國年青企業家尼古拉斯、帕特斯·巴斯天（Patrice Bastian）及克雷門·法爾斯（Clement Fauth)看準了這種中國簡約品味而買下了法國的銷售權。短短5年, 這古老的「飛躍」不單多了一個浪漫的法國名字 - 更打造成為法國市場最大品牌之一。
