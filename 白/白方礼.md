'''白方礼'''（{{bd|1913年|5月13日|2005年|9月23日|白方礼}}），或误写作'''白芳礼'''<ref>{{cite web |url=http://www.tjhbq.gov.cn/ReadNews.asp?NewsID=2551 |title=是白方礼 还是“白芳礼” |date=2007年3月7日 |accessdate=2009-08-18 |work=天津市河北区政务网 |publisher=天津市河北区人民政府 }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，河北[[沧县|沧县]]人，是一位三轮车夫和慈善家。1944年逃避戰亂來到[[天津|天津]]，此后开始从事人力三轮车工作，曾获“劳动模范”荣誉称号。晚年创办“白方礼支教公司”，专门对贫困学生进行资助。2005年9月23日，白方礼因患肺癌辞世。

== 事跡 ==
他靠自己辛勤的劳动养大了4個孩子，其中3個考上了大學，这在当时是一件很难做到的事。1982年，年迈的白方礼開始通过從事個體人力三輪車客運[[支教|支教]]。他自1986年至1999年省吃俭用，用蹬三轮车积攒的钱热心资助贫困学生。一些媒体报道在他的事迹時曾将名字寫錯為：白芳礼。在白方礼支教的19年中，他共积攒了35万元，先后资助了近300名贫困学生的学费与生活费。<ref>{{cite web | url=http://finance.people.com.cn/GB/8215/69438/71371/4856127.html | title=奉献奖候选——白方礼同志简介 | publisher=人民网 | accessdate=2012-02-04}}</ref>据媒体报道，在白方礼支教的19年中，若以每公里三轮车收5角钱计算，白方礼奉献的是相当于绕地球赤道18周的劳动量。一個冬天，他到天津耀華中學，遞上飯盒裡的500元說：“我幹不動了，以後可能不能再捐了，這是我最後的一筆錢。”老師們全哭了。<ref name="新快报" />白方礼临终前的遗言是“来世再蹬三轮支教”。<ref>{{cite web | url=http://news.sohu.com/20050925/n227055631.shtml | title=白大爷临终前感谢各界关心：来世再蹬三轮支教 | publisher=搜狐网 | accessdate=2012-02-04}}</ref>

== “感动中国”落选争议 ==
白方礼曾两度入围央视“感动中国”候选人名单，但两度落选。2004年度“感动中国”评选，白方礼网络投票仅排在[[刘翔|刘翔]]、[[任长霞|任长霞]]之后，名列第三，但最终经评委评选后落榜。2005年白方礼去世后，逝世的消息经媒体报道后引起舆论关注，再次入选2005年度“感动中国”候选人名单，但却再度落榜。<ref>{{cite web | url=http://news.enorth.com.cn/system/2012/02/04/008585718.shtml | title=央视“感动中国”10周年 向白方礼表示致敬 | publisher=北方网 | work=转自《城市快报》 | accessdate=2012-02-04}}</ref>

他支教的事迹感人，在网络上传播后影响广泛，却两度落选，引发公众的不满。在媒体和网络上流传着诸如“感动了中国人民却感动不了CCTV？”<ref name="新快报">{{cite web | url=http://news.163.com/06/0309/12/2BP9905R0001122B.html | title=蹬三轮助学老人白芳礼落选感动中国引发质疑 | publisher=网易 | work=转自《新快报》 | date=2006-03-09 | accessdate=2012-02-04}}</ref>“感动中国容易，感动评委很难”<ref>{{cite web | url=http://www.tianshannet.com/pn/content/2009-02/05/content_3823371.htm | title=感动中国容易，感动评委很难 | publisher=天山网 | accessdate=2012-02-04}}</ref>等质疑[[中国中央电视台|中国中央电视台]]的声音。尤其《白芳礼，你凭什么感动中国？》一文在网络上转载率很高，获得读者很多共鸣。<ref name="新快报" />

2012年，中央电视台感动中国人物评选，向白方礼特别致敬，并称像白方礼一样感人却未受到表彰的人们为“白方礼们”。
