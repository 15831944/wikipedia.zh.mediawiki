'''白雪岑'''{{BD||5月16日||}}，原名白雪，中国女配音演员。

==生平==
毕业于[[北京电影学院|北京电影学院]]表演系02级高职配音班，与演员[[黄渤|黄渤]]，配音演员[[边江|边江]]、[[邱秋|邱秋]]等是同班同学。代表作有电视剧《[[雪山飞狐_(2007年电视剧)|雪山飞狐]]》的程灵素、《[[老九门|老九门]]》的霍锦惜<ref>{{Cite web |url=http://ent.sina.com.cn/2017-04-14/doc-ifyeimqy1472384.shtml |title=《画心师》海报公开 熊梓淇祝绪丹戏中爱斗嘴|accessdate=2018-04-17 |author= |date=2017-04-14 |work=肉圆 |publisher=新浪娱乐|language=Zh-hans }}</ref>、《[[伪装者|伪装者]]》的汪曼春<ref>{{Cite web |url=http://ent.rednet.cn/c/2016/06/03/4000449.htm |title=《双程》首发“致敬”海报 耽美超级大IP来袭|accessdate=2018-04-17 |author= |date=2016-06-13 |work=潘宣 |publisher=红网|language=Zh-hans }}</ref>，游戏《[[仙剑奇侠传|仙剑奇侠传]]》系列的柳梦璃、凌波<ref>{{Cite web |url=http://v.qq.com/x/star/160759#!view=detail?tabid=2 |title=白雪岑|accessdate=2018-04-17 |author= |date=|work= |publisher=腾讯视频|language=Zh-hans }}</ref>等。
