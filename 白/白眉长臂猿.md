'''白眉长臂猿屬'''（[[学名|学名]]：''Hoolock''），是[[长臂猿科|长臂猿科]]下的一属。

== 特征 ==
白眉长臂猿是体型仅次于[[合趾猿|合趾猿]]的长臂猿，成年白眉长臂猿的体长可达90厘米，体重6-9公斤。雄性和雌性的体型差别不大，但雄性的体毛为黑色，眉毛为白色，而雌性的体毛为灰褐色，胸部和颈部的颜色较深。

== 分布 ==
白眉长臂猿主要分布在[[印度|印度]]东北部到[[缅甸|缅甸]]地区，[[中国|中国]]西南部和[[孟加拉国|孟加拉国]]东部也有少数种群。

== 习性与繁衍 ==
它们栖息在树上，白天活动。以水果、昆虫和树叶为食。

孕期为七个月，幼猿的体毛为奶白色，经过六个月颜色变黑。8-9年完全成熟，野外平均寿命为25年。
