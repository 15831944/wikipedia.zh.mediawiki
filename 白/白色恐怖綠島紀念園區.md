'''白色恐怖綠島紀念園區'''，位於[[台灣|台灣]][[台東縣|台東縣]][[綠島鄉|綠島鄉]][[法務部矯正署綠島監獄|法務部矯正署綠島監獄]]旁，隸屬[[中華民國文化部|文化部]][[國家人權博物館|國家人權博物館]]。園區原址曾經拘禁相當多台灣[[政治犯|政治犯]]<ref>{{cite web |language=zh-tw |url = http://www.nhrm.gov.tw/NHRM/Code/gi_park_about.aspx|title=關於園區 |author=|publisher=國家人權博物館|date= |accessdate=2013-12-01}}</ref>。文化園區包含景觀保護區（海岸、燕子洞）、地質保護區（新生訓導處隘口鬼門關）、保護區（新生訓導處公墓、流麻溝）、公園用地（人權紀念碑）、機關用地（[[中華民國國防部|國防部]]綠島感訓監獄、原[[臺灣警備總司令部|臺灣警備總司令部]]新生訓導處、綠島技能訓練所）。

==歷史==
1951年，[[台灣省警備總司令部|台灣省警備總司令部]]在[[綠島|綠島]]成立「新生訓導處」，監管、改造思想或政治上有問題的犯人 。1970年2月8日[[泰源事件|泰源事件]]後，當局在[[綠島|綠島]]新生訓導處舊址西側興建國防部綠島感訓監獄，稱「綠洲山莊」，復於1972年將泰源監獄和各地軍事監獄的政治犯集中送到綠島監禁。<ref>{{cite web | url = http://2009greenisland.blogspot.tw/2009/02/19721987.html| title = 國防部綠島感訓監獄（綠洲山莊）1972～1987 | publisher =「2009綠島．和平．對話」 | language = zh-tw }}</ref>

1997年，當時的[[立法委員|立法委員]][[施明德|施明德]]等16人向[[行政院|行政院]]提案，要求將綠洲山莊改為史蹟館或紀念館。1999年3月29日，行政院核定將綠洲山莊規劃成立史蹟館或紀念館<ref>{{cite web | url = http://www.libertytimes.com.tw/2001/new/feb/9/today-p9.htm| title =綠島換裝 將闢人權紀念園區 | publisher =[[自由時報|自由時報]] |language = zh-tw |date=2001-02-09 }}</ref>。2000年11月，綠島綠洲山莊由[[中華民國法務部|法務部]]移交[[交通部觀光局|交通部觀光局]]接辦，2001年2月23日[[行政院|行政院]]核定名稱改為「綠島人權紀念園區」，範圍從綠洲山莊擴充至莊敬營區、國防部醫務所、[[行政院海岸巡防署|行政院海岸巡防署]]岸巡及安檢兵力駐用廳舍、人權紀念碑、公館漁港及週邊海岸地區。

區內犯人蠟像姿態，是根據當時政治犯[[歐陽劍華|歐陽劍華]]所繪的人體素描所製作<ref name="自由時報2">{{Cite news|url=https://tw.news.yahoo.com/%E6%94%BF%E6%B2%BB%E7%8A%AF%E6%AD%90%E9%99%BD%E5%8A%8D%E8%8F%AF%E8%BE%AD%E4%B8%96-%E8%80%81%E9%9B%A3%E5%8F%8B%E9%80%81%E8%A1%8C-202848070.html|title=政治犯歐陽劍華辭世 老難友送行|language=zh-tw|publisher=《自由時報》|author=謝文華|date=2011-10-02|accessdate=2015-08-14|deadurl=yes|archiveurl=https://web.archive.org/web/20160305205054/https://tw.news.yahoo.com/%E6%94%BF%E6%B2%BB%E7%8A%AF%E6%AD%90%E9%99%BD%E5%8A%8D%E8%8F%AF%E8%BE%AD%E4%B8%96-%E8%80%81%E9%9B%A3%E5%8F%8B%E9%80%81%E8%A1%8C-202848070.html|archivedate=2016-03-05}}</ref>。

2002年12月10日[[世界人權日|世界人權日]]，綠島人權紀念園區正式啟用。2006年1月1日，綠島人權紀念園區由交通部觀光局[[東部海岸國家風景區|東部海岸國家風景區]]管理處移交[[行政院文化建設委員會|行政院文化建設委員會]]<ref>{{cite web | url =http://tw.epochtimes.com/b5/6/1/1/n1174345.htm |title =台文建會今接管綠島人權園區 展新頁 | publisher = [[大紀元時報|大紀元時報]] | 、language = zh-tw |date=2006-01-01 }}</ref>。2007年並更名為「綠島文化園區」。2009年6月與[[景美人權文化園區|景美文化園區]]同時第三次更名為「綠島人權文化園區」<ref>{{cite web | url  =http://www.cna.com.tw/postwrite/Detail/53909.aspx#.UVIyfRxz5dw| title =文建會「人權文化園區」名稱更迭聲明 | publisher =[[中央社|中央社]]訊息服務 | language = zh-tw |date=2010-03-20 }}</ref>。2014年1月27日，依《[[文化資產保存法|文化資產保存法]]》公告登錄為文化景觀類文化資產。2018年5月17日，國家人權博物館舉行「白色恐怖綠島紀念園區」揭牌儀式。<ref>{{Cite web|url=https://2011greenisland.wordpress.com/2011/05/31/%e7%b5%ae%e8%aa%9e-%e9%87%8d%e8%bf%94%e7%81%ab%e7%87%92%e5%b3%b6%ef%bc%8c%e8%a6%8b%e8%ad%89%e9%82%a3%e4%b8%80%e9%a0%81%e6%82%b2%e6%85%98%e6%ad%b7%e5%8f%b2/|title=[絮語] 重返火燒島，見證那一頁悲慘歷史|accessdate=2016-12-29|author=Uncle Sam|date=2011-05-31|work=2011 綠島 ˙ 和平 ˙ 對話|publisher=}}</ref>

== 新生訓導處 ==
1950年代，臺灣地區約有一千多名政治犯被移送綠島「新生訓導處」。該單位當時有12個由男性囚犯所組成的中隊（每個中隊約有數十位成員）；其中有些人被監禁並強迫接受政治教育，也有些人被外放並在監視下從事「[[勞動改造|勞動改造]]」，但大都居住於數間木房搭建而成的牢舍內。

「新生訓導處」有關當局曾驅使其男性囚犯至附近海岸搬運石塊回單位所在地，建築圍牆與簡易屋舍，其獄友也因此曾自我謔稱「自己蓋自己的監獄」；此後，當局亦曾驅使其囚犯從事農耕及食品加工活動，以達自給自足之目標。<ref name=":0">{{Cite web|url=http://ndweb.iis.sinica.edu.tw/TWM/Public/content/story/seal/lyudao.html|title=【綠島十三中隊】台灣外省人生命記憶與敘事資料庫數位典藏計畫，不凡的年代；我們的故事。|accessdate=2016-12-29|last=魁亨科技|first=iifun,|work=ndweb.iis.sinica.edu.tw}}</ref>

當時，新生訓導處大多犯人持續維持不主動探詢他人罪刑內容的默契。<ref>{{Cite news|url=https://2011greenisland.wordpress.com/2017/12/27/%E7%83%BD%E7%81%AB%E4%B8%AD%E7%9A%84%E5%AE%B6%E2%94%80%E2%94%80%E6%AF%9B%E6%89%B6%E6%AD%A3%E8%A8%AA%E8%AB%87%E7%B4%80%E9%8C%84%EF%BC%88%E4%B8%8B%EF%BC%89|title=烽火中的家──毛扶正訪談紀錄（下）|date=2017-12-27|newspaper=2011 綠島 ˙ 和平 ˙ 對話|accessdate=2018-01-10|language=zh-TW}}</ref>

== 第十三中隊 ==
「新生訓導處」囚犯若因故身亡且家屬未領取遺體者，大都埋葬於「新生訓導處」與「[[燕子湖_(綠島)#海蝕洞|燕子洞]]」之間的面海的山坡下；埋葬或死亡於當地者皆被統稱為「第十三中隊」，而該埋葬地點又被倖存者稱作「十三中隊公墓」。<ref name=":0" /><ref>{{Cite journal|title=重返火燒島|author=陳彥斌|url=http://mocfile.moc.gov.tw/history/NTLAC/App_Themes/images/file/PIC1000715.pdf|journal=曙光|issue=38|doi=|others=|year=2011|volume=|page=|pmid=}}</ref>曾被豎立於流麻溝旁之「[[周文彬|周文彬]]紀念碑」及「孫連崑紀念碑」亦於2011年遷至該地。<ref>{{Cite web|url=https://zh-tw.facebook.com/notes/%E8%94%A1%E5%AE%8F%E6%98%8E/%E8%A7%A3%E9%96%8B13%E4%B8%AD%E9%9A%8A%E7%9A%84%E8%AC%8E%E5%9C%98/802832236402273/|title=解開13中隊的謎團|accessdate=2016-12-29|work=zh-tw.facebook.com}}</ref><ref>{{Cite web|url=http://m.ltn.com.tw/news/politics/paper/318370|title=綠島人權藝術季╱重返十三中隊 悼念受難亡魂 - 政治 - 自由時報電子報|accessdate=2016-12-29|work=m.ltn.com.tw}}</ref>
