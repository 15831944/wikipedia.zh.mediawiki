'''白雪山'''（{{bd|1961年|5月}}），[[陕西|陕西]][[吴起|吴起]]人，1984年2月参加工作，1984年5月加入中国共产党，[[中共中央党校|中共中央党校]]函授学院经济管理专业毕业，中央党校大学学历<ref name="调查"/>。曾任[[宁夏回族自治区|宁夏回族自治区]]人民政府副主席。

== 生平 ==
1984年2月在银川市原郊区第二建筑工程公司参加工作，同年5月加入[[中国共产党|中国共产党]]。1985年4月，任银川原郊区二建经理。1990年8月，宁夏烟草系统窝案爆发，宁夏烟草专卖局原局长杨杰落马，白雪山因涉案被检察机关带走，40多天后重获自由。此事深深触动了白雪山，没有权力有再多的钱也没用，从此，白雪山有了弃商从政的念头<ref>{{cite web|url=http://toutiao.com/i6216504337212899841|title=宁夏“首虎”白雪山落马或与当地房企相关|publisher=财经网|accessdate=2015-11-13}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1992年，白雪山告别原郊区二建，进入银川市人民政府驻上海联络处工作，一年后就升任为该联络处主任。1994年9月，白雪山从上海调回银川，历任银川市郊区党委常委、副区长、银川市郊区党委副书记、代区长、区长、银川市郊区党委书记、银川市委常委、郊区党委书记、[[贺兰县|贺兰县]]委书记、银川市委副书记、银川市副市长、银川市市长、[[吴忠市|吴忠市]]委书记、吴忠市人大常委会党组书记、吴忠市人大常委会主任。2013年1月任[[宁夏回族自治区|宁夏回族自治区]]人民政府副主席、党组成员<ref name="调查"/>。

白雪山在主政银川、吴忠期间，一直对房地产开发受到重视，并被当地网友称作“白乱拆”<ref>{{cite news|title=宁夏首虎白雪山落马，主政吴忠市期间曾被称作“白乱拆”|url=http://www.thepaper.cn/newsDetail_forward_1393948|publisher=澎湃新闻 |date=2015年11月7日  |accessdate=2015年11月7日 }}</ref>。

2015年11月6日因涉嫌严重违纪，被组织调查<ref name="调查">{{cite news|title=宁夏回族自治区政府副主席白雪山涉嫌严重违纪接受组织调查 |url=http://www.ccdi.gov.cn/xwtt/201511/t20151106_64664.html |publisher=中央纪委监察部网站  |date=2015-11-06  |accessdate=2015年11月6日 }}</ref>。他是中共十八大以来首位在宁夏被调查的副部级官员<ref>{{cite news|title=中纪委今年已打落26虎 十八大后仅京沪尚无虎 |url=http://www.chinanews.com/gn/2015/11-07/7610806.shtml|publisher=中国新闻网  |date=2015年11月7日  |accessdate=2015年11月7日 }}</ref>。白雪山被带走后，其多名家人也被带走，包括白雪山的老婆和儿子，及其内弟<ref>{{cite web|url=http://toutiao.com/i6216504337212899841|title=宁夏“首虎”白雪山落马或与当地房企相关|publisher=财经网|accessdate=2015-11-13}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。12月28日，中纪委宣布白雪山被双开，移送司法机关<ref>{{cite web |url = http://www.ccdi.gov.cn/xwtt/201512/t20151228_71708.html |title = 宁夏回族自治区政府原党组成员、副主席白雪山严重违纪被开除党籍和公职 |accessdate = 2015年12月28日 |publisher = 中央纪委监察部网站}}</ref>。

2017年4月28日，西安市中级人民法院一审宣判，白雪山利用职务之便为他人提供便利，收受贿赂共计折合人民币3886.680806万元，以受贿罪判处其有期徒刑十五年，并处罚金人民币三百五十万元；对其受贿所得财物及其孳息予以追缴，上缴国库<ref>{{cite web |url = http://news.sina.com.cn/c/nd/2017-04-28/doc-ifyetwtf8769022.shtml |title = 宁夏原副主席白雪山受贿超3800万 获刑15年 |accessdate = 2017年4月28日 |publisher = 新浪}}</ref>。
