'''白蜡树'''（[[学名|学名]]：{{lang|la|''Fraxinus chinensis''}}）亦称'''中國梣'''，是[[木犀科|木犀科]][[梣属|梣属]]的植物。

== 形态 ==
[[落叶乔木|落叶乔木]]。奇数羽状对生[[复叶|复叶]]，小叶通常5--7枚，椭圆形或卵状披针形；圆锥[[花序|花序]]顶生及侧生，下垂，夏季开花；披针形扁平[[翅果|翅果]]。

== 分布 ==
分布于[[越南|越南]]、[[朝鲜|朝鲜]]以及[[中国大陆|中国大陆]]的南北各省等地，生长于海拔800米至1,600米的地区，喜湿润，生长快，多生于山地杂木林中。

== 用途 ==
白蜡树木材坚韧，可以制家具、农具、车辆、胶合板等用；树皮称“秦皮”，中医学上用为清热药；树可放养白蜡虫，以取白蜡；白蜡树又可以作为固沙树种。
