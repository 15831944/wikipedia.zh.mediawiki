'''丽水市文物保护单位'''是中国[[浙江省|浙江省]][[丽水市|丽水市]]的市级[[文物保护单位|文物保护单位]]，现有24处（截止2017年1月），已升级至浙江省文物保护单位和全国重点文物保护单位的以灰底表示<ref>{{cite web |title=《丽水市新增四个市级文物保护单位 应星楼遗址万象山石马等榜上有名》 |publisher=浙江省文物局 |url=http://www.zjww.gov.cn/culture/2016-07-11/1128073590.shtml}}</ref>。

*第一批、第二批：公布时间和数量不详。

*第三批：2004年3月25日公布，同时对第一批、第二批重新核定公布，数量不详。

*第四批：2009年9月18日公布，共7处<ref>{{cite web |title=《第四批市级文物保护单位公布》 |publisher=浙江在线 |url=http://lstk.zjol.com.cn/06lstk/system/2009/10/10/011477053.shtml}}</ref>。 

*第五批：2013年5月29日公布，共6处。含古遗址1处，古建筑2处，石窟寺及石刻2处，近现代重要史迹及代表性建筑1处<ref>{{cite web |title=《丽水市公布第五批市级文保单位保护范围和建设控制地带》 |publisher=浙江省文物局 |url=http://www.zjww.gov.cn/news/2014-11-10/953145942.shtml}}</ref>。 

*第六批：2016年7月4日公布，共4处。含古遗址1处，石窟寺及石刻1处，古建筑1处，近现代重要史迹及代表性建筑1处<ref>{{cite web |title=《丽水市人民政府关于核定并公布第六批市级文物保护单位的通知》 |publisher=丽水市政府 |url=http://www.lishui.gov.cn/zwgk/zwxxgk/002645662/4/2/201607/t20160708_1228261.html}}</ref>。
