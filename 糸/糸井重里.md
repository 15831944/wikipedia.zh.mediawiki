'''糸井重里'''（{{bd|1948年|11月10日|catIdx=Itoi Shigesato}}）是日本[[群馬縣|群馬縣]][[前橋市|前橋市]]出身的男性散文家、作詞家、廣告文宣創作家、電子遊戲創作者。其妻子是日本女演員[[樋口可南子|樋口可南子]]。

個人代表的文宣創作有為[[吉卜力工作室|吉卜力工作室]]的系列動畫或[[西武百貨|西武百貨]]創作的廣告標語，代表性的電子遊戲創作則是《[[地球冒險系列|地球冒險系列]]》。<ref name="Tabibun">{{Cite web |url=http://www.tabibun.net/review/author.php?id=10 |script-title=ja:糸井重里プロフィール＆ガイド |work=Tabibun.net | accessdate=2013年2月11日 }}</ref>糸井重里亦有在[[東京|東京]]成立了糸井重里事務所。

== 經歷 ==

小時的糸井重里是由祖母所照顧，因他的雙親在他出生後不久即離婚。年少時期的糸井重里曾嚮往過從事漫畫家的行業，並喜好著[[北杜夫|北杜夫]]的文學作品。

在進入[[日本法政大學|法政大學]]就讀文學部後的隔年輟學。<ref name="page59">{{cite book |script-title=ja:もののけ姫 ロマンアルバム |publisher=[[德間書店|德間書店]] |isbn=978-4-19-720026-9 |page=59頁 |language=ja |date=1997年11月1日 |accessdate=2013年2月11日 }}</ref>，之後於株式會社宣傳會議公司開設的課堂裡學習廣告文案的創作，在1968年則進入Something事務所工作，而個人的糸井重里事務所則在1979年設立。<ref name="Itami13" />

在1975年糸井重里起多次獲得TCC（Tokyo Copywriters Club）的關於廣告文宣獎項<ref name="page59" />，代表性的有1982、83年為[[西武百貨|西武百貨]]所想出的{{nihongo|「就是喜歡稀奇的東西」|不思議、大好き。}}、{{nihongo|「美妙的生活」|おいしい生活。}}等標語。<ref>{{Cite web |url=http://www.1101.com/news/2012-11-26.html |script-title=ja:糸井重里がコピーライターの殿堂入り！ 「今日いただいた賞は、 『ふつうの人』というあかしです。」 |work={{lang|ja|ほぼ日刊イトイ新聞}} | accessdate=2013年2月11日 }}</ref>。另在1988年起糸井重里為動畫工作室[[吉卜力工作室|吉卜力工作室]]往後多部作品構思文宣標語，而首次接案的動畫是1988年上映的《[[螢火蟲之墓|螢火蟲之墓]]》及《[[龍貓|龍貓]]》，<ref>{{cite book |script-title=ja:映畫道樂 |publisher={{lang|ja || びあ}} |author=[[鈴木敏夫|鈴木敏夫]] |isbn=4-8356-1540-9 |page=134頁 |language=ja |date=2005年4月11日 |accessdate=2013年2月11日 }}</ref>他同時亦為《龍貓》裡登場的角色{{nihongo|草璧達郎|草璧タツオ}}配音。<ref name="Tabibun" />

1989年糸井重里成立了一所專門開發電子遊戲的公司「APE」，<ref>{{Cite web |url=http://www.nintendo.co.jp/nom/0007/kawaguti/page01.html |script-title=ja:ポケモン誕生秘話大公開!!　川口孝司さん（任天堂）インタビュー１ |work=[[任天堂|任天堂]] | accessdate=2013年2月11日 }}</ref>其代表性的作品有他親自參與設計的[[電子角色扮演遊戲|電子角色扮演遊戲]]《[[地球冒險系列|地球冒險系列]]》。而在1998年糸井重里則開設了專門放送網路新聞、名人訪談、及商品販賣的網站「{{lang|ja|ほぼ日刊イトイ新聞}}」。<ref>{{Cite web |url=http://www.mobygames.com/developer/sheet/view/developerId,99881/ |title=Shigesato Itoi |work=MobyGames | accessdate=2013年2月11日 }}</ref>

除在廣告文宣方面的創作外，糸井重里也為多數歌手創作歌詞、以及編寫多本散文集，並曾在2009年獲得獎勵文學創作的[[伊丹十三獎|伊丹十三獎]]。<ref name="Itami13" />
