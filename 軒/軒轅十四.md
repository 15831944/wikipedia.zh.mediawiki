'''軒轅十四'''（'''α Leo''' /'''獅子座α'''）為星官[[軒轅_(星官)|軒轅]]的第十四星，是[[獅子座|獅子座]]最明亮的[[恆星|恆星]]（主星），英文名'''Regulus'''，也是全天空二十五顆最明亮的恆星之一，為第21亮的恆星。

軒轅十四是一顆[[白色|白色]][[主序星|主序星]]，距離[[地球|地球]]約77.5[[光年|光年]]，是距离地球最近的B型主序星。在地球上看它位于獅子座的「[[心臟|心臟]]」位置。軒轅十四被認為是最黯淡的1等星，因為排在它之後的[[弧矢七|弧矢七]]的視星等為+1.5等，被天文學界視為是一顆2等星。

== 名稱 ==
[[轩辕氏|軒轅]]是中國古代帝王[[黃帝|黃帝]]的名字，他被認為是[[中華民族|中華民族]]的始祖。軒轅共十七星，形狀如黃龍蜿蜒於天際之上，其最亮星十四星因在[[五帝座|五帝座]]旁，多借指皇后。

'''Regulus'''在[[拉丁語|拉丁語]]中是「王子」或「小國王」的意思。[[希臘語|希臘語]]的變體'''Basiliscus'''及'''Qalb Al Asad'''也被使用來稱呼軒轅十四，'''Qalb Al Asad'''是從[[阿拉伯語|阿拉伯語]] قلب الأسد qalb演生出來的，意為「獅子的心臟」。

== 觀測 ==

[[File:Erigone_Regulus_path.png|thumb]]掩蔽軒轅十四的軌跡。]]

軒轅十四是最靠近[[黃道|黃道]]的一顆亮星，因此不僅經常會被[[月球|月球]]掩蔽，偶爾還會很罕見的被[[水星|水星]]、[[金星|金星]]和[[小行星|小行星]]掩蔽。軒轅十四上一次被行星掩蔽是在1959年7月7日，下一次將在2044年10月1日，而且這兩次都是被金星掩蔽。由於軌道交點的位置，在未來的數千年內都不會被其它的行星掩蔽。

雖然在北半球的黃昏最適宜觀測軒轅十四的時段是晚冬和春季，但除了8月22日前後一個月的時間會因為軒轅十四接近太陽而不能觀測，在一年當中其它的夜晚都可以找到軒轅十四<ref name="randydspence">[http://earthsky.org/brightest-stars/best-regulus-the-heart-of-the-lion#] EarthSky › Brightest Stars › Tonight, ''Regulus: Heart of the Lion'', July 5, 2009.</ref>。對地球上多數的觀測者來說，軒轅十四[[偕日升|偕日升]]的時間出現在9月的第一個星期。每隔8年[[金星|金星]]也會在這顆恆星偕日升的時候在附近徘迴；上一次是出現在2006年。

軒轅十四的主星是軒轅十四A，質量大約是[[太陽|太陽]]的3.5倍，並且是一顆年齡只有數億歲的年輕恆星。它的自轉非常快速，週期只有15.9小時，因此使它的形狀高度的[[扁橢球體|扁平]] <ref name="apj628">{{cite journal

== 系統 ==

軒轅十四是包含4顆恆星的[[聚星|多星系統]]。軒轅十四A包含一顆光譜類型為B7V的藍色主序星，和一顆至少有0.3太陽質量，可能是白矮星的伴星環繞著。這兩顆恆星以大約40天的週期環繞著共同的質心。由於主星極其扭曲的形狀，所以不能忽略長期軌道攝動的影響，例如它們的[[軌道週期|軌道週期]]，就不能以純粹的二體運動[[克卜勒|克卜勒]]軌道來處理相對的軌道運動。換言之，[[克卜勒|克卜勒]]第三定律，它只適用於兩個質點，不適用於主星高度變形扭曲的環境。

在距離軒轅十四A4,200[[天文單位|天文單位]] {{Citation needed|date=April 2011}}是有著共同自行運動的雙星，分別被標示為軒轅十四B和軒轅十四C，這一對在[[HD星表|HD星表]]的序號是HD87884。第一顆的光譜類型是K2V，它的伴星可能是M4V<ref name="apj628"/>。這一對聯星的軌道週期是2,000年，相距大約100天文單位{{Citation needed|date=April 2011}}。

== 可見性 ==

軒轅十四是天空中21顆[[恆星亮度列表|最亮的恆星]]之一，它的[[視星等|視星等]]為+1.35等，主要是軒轅十四輸出的光。如果將軒轅十四B獨立來觀察，使用雙筒望遠鏡可以觀察到它的光度是+8.1等，而它的伴星軒轅十四C是+13.5等，是三顆恆星中最暗的，需要大口徑的望遠鏡才能直接看見。軒轅十四A本身是光譜聯星：第二顆恆星因為比主星暗淡而不能直接看見。BC對與A的角距離為177弧秒，使用業餘的小望遠鏡就能看見。

== 占星術 ==
軒轅十四是大約前3000年的[[波斯|波斯]][[占星學家|占星學家]]所認定的4顆[[王者之星|王者之星]]（Royal stars）之一。它也是[[中古時代|中古時代]][[占星學家|占星學家]]的15顆[[伯利恒之星|伯利恒之星]]之一，與[[花崗岩|花崗岩]]、[[艾草|艾草]]有關，[[卡巴拉|卡巴拉]]（Kabbalah）的符號則類似一頭野獸[http://www.renaissanceastrology.com/hermesfixedstars.html ]。

在[[中世紀威卡教|中世紀威卡教]]（Stregheria）的信仰中，軒轅十四是[[墮落天使|墮落天使]]（Fallen angel）的化身，也是南方通道的地區看守者。
