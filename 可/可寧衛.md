'''可寧衛股份有限公司'''('''Cleanaway Company Limited''')，通稱可寧衛或可寧衛集團。是位在[[南台灣|南台灣]][[高雄市|高雄市]]的一綠色環保企業，經營一般廢棄物、一般事業廢棄物以及有害事業廢棄物等一站式清除處理服務業務。集團組成包含國內目前最大處理許可量之有害重金屬中間固化處理廠及最終處置民營[[堆填|掩埋場]]<ref>[https://magazine.businessweekly.com.tw/Article_page.aspx?id=32236&p=0 處理垃圾毛利率超過6成，一年賺超過一個股本]商周雜誌.2017年5月21日[2018-05-17]</ref>，還有清除處理清運車隊等。

== 營運設施與技術 ==
*'''一般事業廢棄物處理場'''：可寧衛擁有台灣最大廢棄物處理及掩埋場，其有害事業廢棄物固化處理量是第二名中聯資源的五倍<ref>[https://magazine.businessweekly.com.tw/Article_page.aspx?id=32236&p=0 處理垃圾毛利率超過6成，一年賺超過一個股本]商周雜誌.2017年5月21日[2018-05-17]</ref>，迄今已陸續完成8座掩埋場的開發營運。
*'''汞熱脫附處理'''：可寧衛於2011年開始進行汞熱脫附廠之營運作業，主要是處理高濃度汞廢棄物<ref>[http://ic.tpex.org.tw/company_vision.php?stk_code=8422]產業價值鏈資訊平台[2018-05-17]</ref>。

== 轉投資事業與子公司 ==
*'''一般事業廢棄物處理場'''：可寧衛企業、大倉實業、吉衛、大寧。
*'''事業廢棄物清除公司'''：岡聯事業。
*'''事業廢棄物綜合處理中心'''：2018年4月2日 可寧衛與[[蘇伊士環境集團|全球第二大環保廠─法商蘇伊士集團台灣昇達公司]]聯手取得國內唯一公營的有害事業廢棄物處理「大發廠」民營化的最優投資人團資格，可寧衛和台灣昇達各持股29%，榮工持股42%，榮工大發廠正式轉為民營化<ref>蔡慧貞，[https://www.upmedia.mg/news_info.php?SerialNo=39815?=fb 廢棄物廠聯手世界第二大環保廠]上報.2018年4月29日[2018-05-17]</ref>。

== 勞資關係 ==
可寧衛致力於維持最高標準之公司治理，其董事會於2006年成立「可寧衛職工福利委員會」，而每年的國內外員工旅遊及各項職員工福利，在南部企業裡名列前茅，不僅是名符其實的「幸福企業」，也反應出廢棄物處理產業的前景可期<ref>劉學聖，[https://udn.com/news/story/7241/2743512 南北呼應 高雄可寧衛集團全面調薪5%]聯合新聞網.2017年10月06日[2018-05-17]</ref>。

== 殊榮 ==
*[[金氏世界紀錄大全|金氏世界紀錄]]-'''世界最大垂直花園'''：2015年6月29日可寧衛以百合竹、羅漢松、波士頓蕨等植栽打造一座高13.8公尺、長180公尺綠化植生牆，實測面積約為2,594平方公尺，比金氏世界紀錄保持者新加坡CDL樹屋大樓多出300平方公尺，成為世界最大的垂直花園<ref>郭韋綺，[http://www.chinatimes.com/realtimenews/20150629004148-260405 世界最大垂直花園! 綠化牆破金氏世界紀錄]中時電子報.2015年06月29日[2018-05-17]</ref>。
*高雄厝綠建築大獎：2017年8月18日可寧衛企業總部榮獲[[高雄市政府|高雄市政府]]2017年度第六屆「高雄厝綠建築大獎(商辦建築組)」及「高雄厝創意綠建築大獎(開放空間立體綠化組)」兩項大獎<ref>[http://build.kcg.gov.tw/index.aspx?au_id=12&sub_id=80&id=173994 2017年度第六屆高雄厝綠建築大獎名單出爐]高雄市政府工務局建築管理處.2017年08月18日[2018-05-17]</ref>。
