'''可汗學院'''（{{lang-en|Khan Academy}}）是由[[孟加拉|孟加拉]]裔[[美國|美國]]人、[[麻省理工學院|麻省理工學院]]及[[哈佛大學商學院|哈佛大學商學院]]畢業生[[薩爾曼·可汗|薩爾曼·可汗]]在2006年創立的一所[[非營利組織|非营利]]教育機構<ref>{{cite web |url=http://khanacademy.desk.com/customer/portal/articles/337790-what-is-khan-academy- |title=Khan Academy - What is Khan Academy? |publisher=可汗學院 |accessdate=2013-08-15}}</ref><ref name="khanacademy">[http://www.khanacademy.org/contribute Contribute | Khan Academy]</ref>。機構通过网络提供一系列免費教材，現於[[Youtube|Youtube]]載有超過5,600段教學影片<ref name="about" />，內容涵蓋[[數學|數學]]、[[歷史|歷史]]、[[醫療衛生|醫療衛生]]及[[醫學|醫學]]、[[金融|金融]]、[[物理|物理]]、[[化學|化學]]、[[生物|生物]]、[[天文學|天文學]]、[[經濟學|經濟學]]、[[宇宙學|宇宙學]]、[[有機化學|有機化學]]、[[美國|美國]][[公民教育|公民教育]]、[[美術史|美術史]]、[[宏觀經濟學|宏觀經濟學]]、[[微觀經濟學|微觀經濟學]]及[[電腦科學|電腦科學]]。

== 沿革 ==
創辦人薩爾曼·可汗擁有[[麻省理工學院|麻省理工學院]]的數學學士與電機碩士學位，以及[[哈佛大學|哈佛大學]]的[[工商管理碩士|工商管理碩士]](MBA)學位，曾從事[[金融業|金融業]]。為了幫助住在遠處的親人，他試著把自己的教學影片放上網路，主要是在[[YouTube|YouTube]]網站。由於受到廣泛好評，相關影片觀看次數急速成長，受到鼓勵的薩爾曼於2009年辭去工作，全職從事相關課程的錄製。目前已有自己的獨立網站。該機構曾獲得2009年[[微軟|微軟]]教育獎，2010年[[谷歌|谷歌]]「十的一百次方計畫」教育項目的兩百萬美元資助。

== 課程特點 ==


可汗學院利用了網路傳送的便捷與錄影重複利用成本低的特性。每段課程影片長度約十分鐘，從最基礎的內容開始，以由易到難的進階方式互相銜接。教學者本人不出現在影片中，用的是一種電子黑板系統。其網站目前也開發了一種練習系統，記錄了學習者對每一個問題的完整練習記錄，教學者參考該記錄，可以很容易得知學習者哪些觀念不懂，目前已有557個練習。傳統的學校課程中，為了配合全班的進度，教師只要求學生跨過一定的門檻（例如及格）就繼續往下教；但若利用類似於可汗學院的系統，則可以試圖讓學生懂得每一個未來還要用到的基礎觀念之後，再繼續往下教學，進度類似的學生可以重編在一班。在美國，某些學校已經採用「回家看可汗學院影片代替家庭作業，上課時則是做練習，再由老師或已懂的同學去教導其他同學」這樣的教學模式。

== 发展 ==

=== 列入正規學分 ===
[[愛達荷州|愛達荷州]]在2011年立法通過中學畢業47學分中的2分必須為[[線上學習|線上學習]]課程，而2013年可汗學院的課程會在二十多所公立學校採用。<ref>[http://www.sfgate.com/news/article/Online-academy-classes-to-be-tested-in-ID-schools-4152085.php Online academy classes to be tested in ID schools]</ref>

=== 官方SAT考试练习网站 ===
可汗学院从2015年起，成为了sat考试的官方授权练习网站。<ref>{{Cite news|url=https://www.collegeboard.org/releases/2016/One-Year-Since-Launch-SAT-Khan-Academy-Leveling-Playing-Field|title=One-Year-Since-Launch-SAT-Khan-Academy-Leveling-Playing-Field|author=collegeboard|date=2017-1-20|work=|accessdate=2018-6-2}}</ref>研究<ref name=":0">{{Cite web|url=https://www.khanacademy.org/about/blog/post/160451329150/studying-for-the-sat-for-20-hours-on-khan-academy|title=studying-for-the-sat-for-20-hours-on-khan-academy|accessdate=2018-6-2|author=Khan Academy|date=2017-5-8|publisher=}}</ref>指出，通过可汗学院官网学习并准备考试仅20小时，考试者的成绩显著提高。2017年，有超过16000考生报告，通过在可汗学院网站上的学习，他们的成绩都提高了200分以上<ref name=":0" />。

可汗学院的sat练习提供个性化，高效的复习方式。

国内多家教辅书和练习册都从Khan Acdemy的sat备考试题中选题，并明确标注了来自可汗学院。

=== 与皮克斯的合作 ===
Khan Academy 与 Pixar合作，在2015年8月推出了 Pixar in a box项目，目标是展示学生在学校学习的学术概念如何用于解决制作皮克斯电影时的创造性挑战。<ref>{{Cite web|url=https://www.khanacademy.org/partner-content/pixar|title=Pixar in a box|accessdate=2018-6-2|author=|date=|publisher=}}</ref> Pixar in a box中，来自皮克斯和迪士尼的工作人员，详细的讲解并展示了一部优秀动画的诞生过程：从故事构思，到3D渲染。

== 中文翻譯計劃 ==
2013年開始，可汗學院開放給[https://crowdin.com/project/khanacademy 世界各國]將其本站翻譯為自身語言，完成後並可以隨時多語切換，提供[https://translate.khanacademy.org/ 全球一致的學習體驗]。「[http://www.criticalinno.org 台灣關鍵創新教育協會]」為華語地區主要推動者，目前「可汗學院中文翻譯計畫」共有超過500名志愿者加入，並[http://www.criticalinno.com/p/blog-page_6.html 持續招募翻譯及校對者]。
