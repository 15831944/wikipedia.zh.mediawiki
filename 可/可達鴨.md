{{vgname2|可達鴨|ja=コダック|en=Psyduck|2=香港舊譯'''傻鴨'''}}是[[任天堂|任天堂]]和[[Game_Freak|Game Freak]]所特許經營的《[[精靈寶可夢系列|精靈寶可夢]]》系列裡的虛構生物「[[寶可夢列表|寶可夢]]」之一，在《[[精靈寶可夢_紅·綠|精靈寶可夢 紅]]》與《[[精靈寶可夢_紅·綠|精靈寶可夢 綠]]》中首次被引入，並出現在該系列的後續作品及改編作品中。可達鴨的外型類似一隻鴨子，長期受到頭痛的困擾。當它具有足夠的經驗值時，會進化成[[哥達鴨|哥達鴨]]。

==設計==
可達鴨是首次出現於《[[精靈寶可夢_紅·綠|精靈寶可夢 紅]]》與《[[精靈寶可夢_紅·綠|精靈寶可夢 綠]]》的第一世代寶可夢，由[[杉森建|杉森建]]設計。<ref>{{cite web |url=https://www.nintendo.co.jp/ds/interview/irbj/vol1/index2.html |title=2. 一新されたポケモンの世界 |page=2 |work=Nintendo.com |publisher=[[任天堂|任天堂]] |accessdate=2010-09-10 |author=Staff |language=ja}}</ref><ref>{{cite web |url=http://www.computerandvideogames.com/article.php?id=91965 |title=Game Freak on Pokémon! |author=Stuart Bishop |publisher=CVG |date=2003-05-30 |accessdate=2008-02-07 |archiveurl=https://www.webcitation.org/5VSJaR6xT?url=http://www.computerandvideogames.com/article.php?id=91965 |archivedate=2008-02-08 |deadurl=yes }}</ref>

==反響==
可達鴨受到[[GameSpy|GameSpy]]的讚揚，且被描述為「愚蠢至極」<ref>{{cite journal |url=http://www.gamespy.com/articles/508/508764p1.html |publisher=[[GameSpy|GameSpy]] |accessdate=2009-10-03 |title=Hall of Fame: Pokémon |author=Leeper, Justin |date=2004-04-24}}</ref>。《[[紐約時報|紐約時報]]》則將之與[[鴨嘴獸|鴨嘴獸]]做對比，稱可達鴨為「看起來超蠢」<ref>{{cite book |title=The New York Times Film Reviews 1999-2000 |author=New York Times Theater Reviews |page=178 |publisher=Taylor & Francis |year=2001 |isbn=0-415-93696-9}}</ref>。{{tsl|en|GameDaily||GameDaily}}則認為可達鴨是很獨特的寶可夢，且即使是將範圍拉大到整個系列來看，這個評價也不為過<ref>{{cite web |url=http://www.gamedaily.com/articles/galleries/retro-rewind-then-and-now--pokemon/?page=16 |archiveurl=https://web.archive.org/web/20090413015150/http://www.gamedaily.com/articles/galleries/retro-rewind-then-and-now--pokemon/?page=16 |archivedate=2009-04-13 |title=Retro Rewind: Then and Now - Pokémon |accessdate=2009-06-09 |publisher=[[美国在线|美国在线]] |work={{tsl|en|GameDaily||GameDaily}} |author=Workman, Robert}}</ref>。[[IGN|IGN]]指出這個角色之所以受歡迎是因為它感到困惑的怪異外表<ref>{{cite web |url=http://stars.ign.com/objects/143/14350017_biography.html |title=Psyduck Biography |publisher=[[IGN|IGN]] |work=[[IGN|IGN]] |accessdate=2009-09-28 |deadurl=yes |archiveurl=https://archive.is/20120723060104/http://stars.ign.com/objects/143/14350017_biography.html |archivedate=2012-07-23 }}</ref>，更表示可達鴨是他們辦公室成員最愛的角色之一<ref>{{cite web |archiveurl=https://web.archive.org/web/20010331212321/http://pocket.ign.com/news/12065.html |title=Pokémon of the Day: Psyduck |author=Staff |publisher=[[IGN|IGN]] |work=[[IGN|IGN]] |url=http://pocket.ign.com/news/12065.html |accessdate=2009-10-05 |archivedate=2001-03-31 |date=1999-11-11 |deadurl=yes }}</ref>。[[GamesRadar|GamesRadar]]也稱可達鴨是第一世代遊戲中最受歡迎的寶可夢之一 <ref>{{cite web |url=http://www.gamesradar.com/ds/f/the-complete-pokemon-rby-pokedex-part-5/a-20070820141819445080/g-2006100415372930075/p-10 |page=10 |title=The complete Pokemon RBY pokedex, part 5 |publisher=[[Future_plc|Future plc]] |work=[[GamesRadar|GamesRadar]] |accessdate=2009-10-01 |author=Elston, Brett}}</ref>。《Coventry Evening Telegraph》將可達鴨與[[傑尼龜|傑尼龜]]相提並論，並讚揚它們的設計<ref>{{cite news |first=Frings |last=Tim |title=CINEMA: PEAK emon! |newspaper={{tsl|en|Coventry Evening Telegraph||Coventry Evening Telegraph}} |date =December 22, 2000}}</ref>。

部份臺灣的網友認為可達鴨的外型與台北市長[[柯文哲|柯文哲]]很相似，[[林昶佐|林昶佐]]、[[郝龍斌|郝龍斌]]等政治人物也曾以此消遣柯文哲<ref>{{cite news |language = 中文 |author = 徐政璿 |coauthors =  |url = http://www.ettoday.net/news/20160910/772822.htm |title = 用餐巧遇柯文哲！　林昶佐：怎麼是可達鴨啊？ |publisher = 東森新聞雲 |pages =  |date = 2016-09-10 |accessdate = 2016-12-27}}</ref><ref>{{cite news |language = 中文 |author = 程平 |coauthors =  |url = http://udn.com/news/story/10014/1948095 |title = 郝酸柯：500天孵1隻CP157可達鴨 |publisher = 聯合新聞網 |pages =  |date = 2016-09-08 |accessdate = 2016-12-27}}</ref>，但柯文哲本人表示自己不像可達鴨<ref>{{cite news |language = 中文 |author =  |coauthors =  |url = http://news.ltn.com.tw/news/politics/breakingnews/1766434 |title = 被說像「可達鴨」 柯文哲大笑：哪裡像？ |publisher = 自由時報 |pages =  |date = 2016-07-18 |accessdate = 2016-12-27}}</ref><ref>{{cite news |language = 中文 |author = 顧荃|coauthors =  |url = http://www.cna.com.tw/news/firstnews/201607180111-1.aspx |title = Pokemon GO太夯 柯文哲:可達鴨哪裡像我 |publisher = 中央社 |pages =  |date = 2016-07-18 |accessdate = 2016-12-27}}</ref>。
