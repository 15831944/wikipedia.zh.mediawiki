{{地區用詞4|cn=科伦坡|tw=可倫坡|sg=哥伦坡|start=[[僧伽罗语|僧伽罗语]]：'''කෙළංඹ'''；[[泰米尔语|泰米尔语]]：'''கொழும்பு'''；拉丁字母转译为：'''Colombo'''|as=又譯作}}，位於[[錫蘭島|錫蘭島]]西南岸、瀕[[印度洋|印度洋]]，是[[斯里蘭卡|斯里蘭卡]]的最大城市與商業中心，同时也是[[西部省_(斯里蘭卡)|西部省]]的行政中心。而斯里蘭卡的政治首都[[斯里賈亞瓦德納普拉科特|斯里賈亞瓦德納普拉科特]]位於其郊區，但因距離過近，近年來許多國際的官方資料仍記載可倫坡為斯里蘭卡的首都，包括[[美國中央情報局|美國中央情報局]]，可倫坡的名稱來自[[僧伽羅語|僧伽羅語]]的Kola-amba-thota，意為「芒果港」，其後[[葡萄牙|葡萄牙]]人將其拼寫成Colombo以紀念[[哥倫布|哥倫布]]。
根据[[布鲁金斯学会|布鲁金斯学会]]科伦坡市区人口为752,993人，[[都会区|都会区]]则多达5,600,000人。科伦坡是一个繁忙且充满生气的城市，有着现代化和殖民时期残留下来的建筑及生活风格<ref name="rweb1">{{cite web |last=Jayewarden+-e |first=Mr |title=How Colombo Derived its Name |url=http://www.rootsweb.ancestry.com/~lkawgw/colombo.html |doi= |accessdate=2007-01-18}}</ref>。科伦坡直到1982年为斯里兰卡的首都。

如同许多其他城市，科伦坡的郊区范围远超过其行政区界限，并包含了许多其他市区或郊区的行政区，例如[[斯里賈亞瓦德納普拉科特|斯里賈亞瓦德納普拉科特]]、[[代希瓦勒-芒特拉維尼|代希瓦勒-芒特拉維尼]]、[[科洛那瓦|科洛那瓦]]（Kolonnawa）、[[卡杜维拉|卡杜维拉]]（Kaduwela）和[[科迪卡瓦他塔穆勒利亚瓦|科迪卡瓦他塔穆勒利亚瓦]]（Kotikawatta Mulleriyawa）。斯里兰卡主要的行政办公室、餐厅和娱乐场所都位于此。

== 歷史 ==
關於科倫坡的記載，據信在兩千年的[[羅馬|羅馬]]、[[阿拉伯|阿拉伯]]中就已經出現。[[八世紀|八世紀]]時，僧伽羅王國與外界的貿易大多由在科倫坡的[[穆斯林|穆斯林]]商人所經手。[[中國|中國]]史料當中最早见于[[元代|元代]][[航海家|航海家]][[汪大渊|汪大渊]]所著的《[[岛夷志略|岛夷志略]]》，称之为'''高朗步'''<ref>元[[汪大渊|汪大渊]]著《[[岛夷志略|岛夷志略]]·高朗步》 中华书局 2000年版 270页 ISBN 7-101-02026-7</ref>。

葡萄牙人於1517年來到科倫坡，建設了保護其[[香料|香料]]貿易的據點。接下來科倫坡先在1656年被[[荷蘭|荷蘭]]佔領、1796年再被[[英國|英國]]佔領；英國並將科倫坡建設成為英屬錫蘭的首都。自1948年斯里蘭卡獨立起，科倫坡一直是斯里蘭卡的首都，直到1985年遷都至科倫坡東南郊區的[[斯里賈亞瓦德納普拉科特|斯里賈亞瓦德納普拉科特]]為止。

== 地理 ==
科伦坡陆地和水域交杂，有许多条运河，市中心还有65公顷的贝拉湖（Lake Beira）。贝拉湖是科伦坡最有特色的地标之一，数百年来被殖民者用来当作屏障抵御外敌，現在則用来举办许多水上娱乐活动，例如赛帆船、在岸边放映电影等等。科伦坡的北边及东北边以[[凱拉尼河|凱拉尼河]]为界，凯拉尼河的出海口位于当地一个被称为莫德拉（mōdara，僧伽罗语中的[[三角洲|三角洲]]）的区域。

=== 气候 ===
根据[[柯本气候分类法|柯本气候分类法]]，科伦坡被归类于[[热带季风气候|热带季风气候]]，接近[[热带雨林气候|热带雨林气候]]。科伦坡的气候在一整年都相当温和。二月到四月是最热的季节，平均高温超过31°C<ref>{{cite web |url=http://www.weather.com/outlook/travel/businesstraveler/wxclimatology/monthly/graph/CEXX0001?from=_bottomnav_business |title=Colombo weather |accessdate=2006-12-02 |work= |archive-url=https://web.archive.org/web/20091005234250/http://www.weather.com/outlook/travel/businesstraveler/wxclimatology/monthly/graph/CEXX0001?from=_bottomnav_business |archive-date=2009-10-05 |dead-url=yes }}</ref>。唯一的气候主要变化发生在从四月到六月及从九月到十二月，这段期间[[季风|季风]]会带来丰沛的雨量。日温差较小，只有在冬季的月份时，平均低温可降达22°C左右。平均一年的降雨量超过2,500毫米。<ref>科伦坡的史上记录最低温为15°C。[http://www.weatherbase.com/weather/weather.php3?s=076434&refer=/ Weatherbase]</ref>

== 人口 ==
科伦坡是一个多元民族、多元文化的城市。主要民族包括[[僧伽罗人|僧伽罗人]]、[[斯里兰卡摩尔人|斯里兰卡摩尔人]]及[[泰米尔人|泰米尔人]]等。其他还有包含华人、葡萄牙人、荷兰人、马来人和印度人等外来民族的小社区，以及来自欧洲的被流放者。科伦坡是斯里兰卡人口最多的城市，其行政区内有752,993人。在1886年时人口则大约有8万人。根据2012年的人口普查，科伦坡市区的人口组成如下表<ref name='cen01'>{{cite web |url= http://www.statistics.gov.lk/PopHouSat/CPH2011/index.php?fileName=pop32&gp=Activities&tpl=3 |title=  "Department of Census and Statistics Sri Lanka – Population by divisional secretariat division, religion and sex 2012"|accessdate=2019-04-11}}</ref>：
