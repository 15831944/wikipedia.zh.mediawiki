== 沿革 ==

* 2015年3月27日：聯發科技的Helio系列晶片最早是在2015年3月27日時在[[中國|中國]][[北京|北京]]正式發表，其首款產品為Helio X10（內部代號為MT6795）。<ref name="聯發科技官網 聯發科技發佈高階智慧型手機晶片品牌Helio"/>
* 2015年4月15日：聯發科技針對Helio系列的中文徵名活動正式開始。<ref>{{cite web |url=http://bj.vdong.cn/helio/helio_pc/p/ |title="一名驚人 聯發科技百萬爭名 |accessdate=2015-04-15 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 介紹 ==
=== 分支系列 ===
Helio系列的分支目前有兩大系列，其中以主打頂級性能為主的是Helio X系列，主打科技時尚的是Helio P系列。Helio X系列具備强悍極致的運算能力和多媒體功能，而Helio P系列提供最佳化的功耗管理，優化[[PCB|PCB]] 尺寸同時兼顧頂級規格，可實現輕薄時尚的智慧型手機設計。<ref name="聯發科技官網 聯發科技發佈高階智慧型手機晶片品牌Helio"/>2018年推岀入門級Helio A系列。

=== 搭載技術 ===
Helio系列搭載了許多由聯發科技自行開發的多項[[專利|專利]]技術。<ref name="聯發科技官網 聯發科技發佈高階智慧型手機晶片品牌Helio"/>

*MiraVision：高性能即時像素處理[[引擎|引擎]]，可高效控制智慧型裝置的[[螢幕|螢幕]]敏銳度、對比度以及顔色來提高畫質。<ref name="聯發科技官網 聯發科技發佈高階智慧型手機晶片品牌Helio"/>
*SmartScreen：能即時壓縮和提高智慧型裝置的螢幕畫質，爲使用者提供他們需要的亮度。此時，手機的背光度其實是沒有改變的。智慧顯示的這一特性能有效節省[[電池|電池]]用量，僅需極低功耗就能體驗媲美[[電視|電視]]般的畫質。<ref name="聯發科技官網 聯發科技發佈高階智慧型手機晶片品牌Helio"/>
*120Hz動態顯示技術：首款能爲智慧型手機提供120Hz[[動態顯示|動態顯示]]技術的智慧型裝置處理器就是Helio X10。當智慧型手機顯示高動態影像或是快速滑動時，120Hz可顯示較60Hz雙倍流暢的捲動及清晰的影像，降低文字及影像的殘影或模糊現象。<ref name="聯發科技官網 聯發科技發佈高階智慧型手機晶片品牌Helio"/>
*Super Slow-Motion：Helio系列提供智慧型手機[[超慢鏡|超慢鏡]]（Super Slow-Motion）技術，讓智慧型手機能夠製作每秒480幅畫面更新率（480 [[fps|fps]]）的1080p高畫質影片，並以1/16的超慢速度播放。<ref name="聯發科技官網 聯發科技發佈高階智慧型手機晶片品牌Helio"/>
*InstantFocus：能使聚焦時間縮减到1.5秒，同時提高影像追蹤能力確保使用者能清晰地看到每一個移動中的影像。<ref name="聯發科技官網 聯發科技發佈高階智慧型手機晶片品牌Helio"/>
*True Bright影像處理器：內建RWWB影像感測器，在同樣曝光時間下可較傳統RGB感測器捕捉更多光影與保留貼近真實的細節，在低光源環境下仍可有卓越表現。與RGBW感測器相比，RWWB可增強顏色解析度。首個搭載該技術的晶片產品為Helio P10（內部代號為MT6755）。
