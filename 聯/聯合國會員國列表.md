}}[[聯合國|聯合國]]一共有193個會員國。所有'''聯合國會員國'''都是[[聯合國大會|聯合國大會]]的正式成員，在聯合國大會中擁有平等的代表權。

原則上，只有[[主權國家|主權國家]]可以成為聯合國會員國，今日聯合國所有193個成員國全部皆是[[主權|主權]][[獨立|獨立]]的[[國家|國家]]。不過，在聯合國初創時，[[菲律賓|菲律賓]]、[[印度|印度]]這2個創始會員國尚未從他們的[[殖民|殖民]]母國獨立，[[白俄罗斯|白俄罗斯]]和[[乌克兰|乌克兰]]为[[苏联|苏联]][[加盟共和国|加盟共和国]]。而[[美国|美国]]在[[联合国|联合国]]成立之初也争取到了与[[苏联|苏联]]对等的增加两个席位的权利，却因无法在当时的48个州（阿拉斯加和夏威夷当时还未升级为州）中选出两个州而搁置此事。如今，[[美国国务院|美国国务院]]网站在介绍联合国的创建过程时，依然声称“美国至今仍保留在适当时候增加两个联合国代表席位的权利”。再者，由於所有欲申請成為會員的國家必需通過[[安全理事會|安全理事會]]和大會的批准，部份依照《[[蒙特維多國家權利義務公約|蒙特維多國家權利義務公約]]》可視為主權國家的政治實體，因為聯合國不承認他們的主權、[[未被国际普遍承认的国家列表|缺乏國際普遍承認]]、或是遭受特定成員國的反對，導致這些國家沒有辦法成為聯合國的會員。

== 現任會員國 ==
[[File:UN_Member_Countries_World.svg|thumb]]

聯合國共有51個創始會員國，在1945年加入成為會員，其中49個國家或其後繼者至今仍然是成員國（例如：[[俄羅斯|俄羅斯]]接替[[蘇聯|蘇聯]]的席位），另外兩個國家是[[捷克斯洛伐克|捷克斯洛伐克]]和[[南斯拉夫|南斯拉夫]]，這兩個國家皆已解體，並且沒有國家接替他們的席位。

以下列出了至最新一個成員國[[南蘇丹共和國|南蘇丹共和國]]在2011年7月14日加入後，全部193個聯合國成員國。並以國名之英文字母首字為序排列，其中創始會員國以<span style="background-color:#CCDDFF;width:50px;border:1px solid #AAAAAA">天蓝背景色</span>衬托、'''粗體'''标示。* {{cite web |url = http://www.un.org/en/member-states/ |title=联合国会员国名单 |accessdate=2016年7月16日 |author=联合国 |authorlink=联合国 |date=2006年10月3日 |language = en}}

== 觀察員和非會員 ==
=== 观察员 ===
{{Main list|聯合國大會觀察員列表}}

联合国还设有[[联合国大会|联合国大会]][[聯合國大會觀察員列表|观察员]]制度，邀请国际组织、非政府组织、政治实体参与联合国事务。观察员在联合国大会上可以发言，但没有参与会议投票的权利。截至2012年，联合国大会共有64个观察员。当中包含2个观察员国、4个观察员实体、和58个观察员组织。

觀察員國被視為擁有主權的政治實體，隨時可以依照自由決斷遞交請願書申請成為聯合國的正式會員國，例如：[[瑞士|瑞士]]在2002年9月10日成為正式會員國之前的1948年至2002年間，亦是聯合國的常任觀察員國；[[聖座|聖座]]自1964年4月6日起成为聯合國觀察員國。

[[巴勒斯坦解放組織|巴勒斯坦解放組織]]在1974年11月22日根据聯合國大會的3237號決議得到觀察員席位。在[[巴勒斯坦解放組織|巴勒斯坦解放組織]]宣稱擁有[[巴勒斯坦國|巴勒斯坦國]]的主權之後，1988年12月15日大會依照A/RES/43/177號決議將其名稱改為「[[巴勒斯坦|巴勒斯坦]]」。2012年11月29日，根据[[联合国大会67/19号决议|联合国大会67/19号决议]]成为观察员国。

許多[[國際組織|國際組織]]、[[非政府組織|非政府組織]]，以及諸如[[歐洲聯盟|歐洲聯盟]]、[[紅十字會與紅新月會國際聯合會|紅十字國際委員會]]、[[馬爾他騎士團|馬爾他騎士團]]等主權地位沒有明確定義的政治實體，皆被聯合國大會邀請成為觀察員。

=== 非会员 ===
[[西撒哈拉|西撒哈拉]]的主權爭議在[[摩洛哥|摩洛哥]]和[[撒拉威阿拉伯民主共和國|撒拉威阿拉伯民主共和國]]之間爭持不下，摩洛哥實際上統治大部分西撒哈拉地區，剩下的少部份地區則由撒拉威阿拉伯民主共和國控制。撒拉威阿拉伯民主共和國是[[非洲聯盟|非洲聯盟]]的[[非洲聯盟成員國列表|會員國]]，但既不是聯合國的會員國，也不具觀察員身分。

[[索馬里蘭|索馬里蘭]]是位於[[索馬里|索馬里]]東北部一個未被普遍承認的政治實體，自[[索馬里內戰|索馬里內戰]]之後，索馬里四分五裂，被索馬里及索馬里蘭、[[邦特蘭|邦特蘭]]、[[西南索馬里|西南索馬里]]等政權割據統治，其中，西南索馬里和邦特蘭已名義上有重新加入索馬里的意願并落實實處，而索馬里蘭事實上獨立于索馬里，并不屬於[[索馬里聯邦政府|索馬里聯邦政府]]管控。目前無任何為承認，政治較為穩定，與[[埃塞俄比亞|埃塞俄比亞]]和[[歐盟|歐盟]]有較多經貿往來。

新西蘭的[[聯繫邦|聯繫邦國]]，[[庫克群島|庫克群島]]和[[紐埃|紐埃]]擁有完整的自治權利，但是其國防、外交事務由新西蘭負責，兩國都不是聯合國會員國。

[[中華民國|中華民國]]以[[台灣|台灣]]、[[澎湖|澎湖]]、[[金門|金門]]以及[[馬祖|馬祖]]以[[台澎金馬個別關稅領域|台澎金馬個別關稅領域]]之名義做為[[世界貿易組織|世界貿易組織]]（WTO）的會員。基於以「中華民國」名義推動「[[2008年台灣返聯公民投票|返聯]]」的效果有限，[[台灣獨立運動|台灣獨立運動]]推動者與支持者根據[[台灣主權未定論|台灣主權未定論]]，推動以「台灣」的名義取代「中華民國」加入聯合國（[[2008年台灣入聯公民投票|入聯]]），與[[聯合國大會2758號決議|聯合國大會2758號決議]]的「[[一個中國|一個中國]]」代表權之爭做出區別，並主張台灣早已是獨立於[[中國|中國]]內政之外的政治實體。主張方式有以下幾種方式<ref name = "防止激化中國再啟斷交攻勢　今年我參與聯合國將採「溫和原則」_2018"> {{ cite web | title = 防止激化中國再啟斷交攻勢　今年我參與聯合國將採「溫和原則」 | url = https://www.upmedia.mg/news_info.php?SerialNo=47159 | publisher = 上報 UP Media | author = 仇佩芬 | language = zh-tw | date = 2018-08-30 }} </ref>：
* 爭取在總務委員會進行台灣加入聯合國之辯論
* 爭取友邦在大會發言幫助台灣發聲
* 爭取友邦聯名致函聯合國秘書長表達友邦支持台灣加入聯合國

== 会员国席位变迁 ==
=== 中國席位 ===
{{main|中國與聯合國}}
由[[中華民國|中華民國]]代表的「'''[[中國|中國]]'''」，是聯合國5個創辦國之一，並於1945年10月24日成為會員，為創始會員國。因[[中國國民黨|中國國民黨]]在[[第二次國共內戰|第二次國共內戰]]中失利，[[中華民國政府|中華民國政府]]被迫在1949年自[[南京市|南京市]]遷至[[台北市|台北市]]，[[中國共產黨|中國共產黨]]於1949年10月1日宣布成立[[中华人民共和国中央人民政府_(1949年－1954年)|中央人民政府]]，但當時以[[美國|美國]]為首的多數國家仍承認中華民國政府為中國的唯一合法政府，因此中華民國政府在聯合國依然維持中國席位，儘管相較於統治中國大部分地區的[[中華人民共和國|中華人民共和國]]政府，中華民國政府的實際管轄範圍已僅限於[[臺灣地區|臺灣地區]]。1971年10月25日，中華民國被迫提前宣布退出聯合國，隨後聯合國大會通過[[聯合國大會2758號決議|2758號決議]]，不再視中華民國政府為中國的合法代表，轉而承認中華人民共和國政府是中國的唯一合法政府，自此中華人民共和國政府取代中華民國在聯合國的中國席位。因此目前中華民國政府所統治的[[臺灣地區|臺灣地區]]，[[實際上|實際上]]被排除在[[聯合國|聯合國]]以及[[聯合國系統|聯合國系統]]機構之外。

自1990年代起，中華民國多次嘗試申請加入聯合國，中華民國政府在申請書中表示在聯合國中將只代表[[臺灣地區|臺灣地區]]的人民，而非全中國，並以「[[中華民國（臺灣）|中華民國（臺灣）]]」、或僅「[[臺灣|臺灣]]」等各種名義提出申請；但因聯合國遵守[[一個中國|一個中國]]政策，所有申請要求皆被拒絕。除了會員外，中華民國同樣也被聯合國拒絕給予「觀察員國」資格(如[[梵蒂岡|梵蒂岡]]、[[巴勒斯坦國|巴勒斯坦]])，或「觀察員實體」資格(如[[馬爾他騎士團|馬爾他騎士團]])。目前，有16個聯合國成員國和[[梵蒂岡|梵蒂岡]]與中華民國之間有大使級外交關係<ref>詳細請參考[[中華民國外交|中華民國外交]]條目</ref>。

=== 前南斯拉夫相关 ===
[[南斯拉夫|南斯拉夫]]是联合国的创始会员国，于1945年6月26日签署，1945年10月19日批准《宪章》，在下列国家成立并随后加入为新会员国后解体：波斯尼亚和黑塞哥维那、克罗地亚共和国、斯洛文尼亚共和国、前南斯拉夫的马其顿共和国、南斯拉夫联盟共和国。南斯拉夫联盟共和国与2003年改名为塞尔维亚和黑山，而黑山则于2006年独立，从此塞尔维亚共和国继承塞尔维亚和黑山在联合国的席位和权利。目前存在领土争议的科索沃地区未被联合国承认，联合国认定其为塞尔维亚共和国的一部分。
*[[波斯尼亚和黑塞哥维那|波斯尼亚和黑塞哥维那共和国]]由大会1992年5月22日A/RES/46/237号决议接纳为联合国会员国。
*[[克罗地亚|克罗地亚共和国]]由大会1992年5月22日A/RES/46/238号决议接纳为联合国会员国。
*[[斯洛文尼亚|斯洛文尼亚共和国]]由大会1992年5月22日A/RES/46/236号决议接纳为联合国会员国。
*[[北马其顿共和国|北马其顿共和国]]由大会1993年4月8日A/RES/47/225号决议决定接纳为联合国会员国，在该国国名引起的争端解决之前，该国在联合国内暂时称为 “前南斯拉夫的马其顿共和国”。
*[[南斯拉夫联盟共和国|南斯拉夫联盟共和国]]由大会2000年11月1日A/RES/55/12号决议接纳为联合国会员国。
**2003年2月4日，南斯拉夫联盟共和国通过并实行了塞尔维亚和黑山共和国宪法，南斯拉夫联盟共和国正式改国名为“[[塞尔维亚和黑山|塞尔维亚和黑山]]”。
**2006年6月3日，[[黑山共和国|黑山共和国]]议会通过了独立宣言。根据塞尔维亚和黑山宪法宪章第60条规定，[[塞尔维亚共和国|塞尔维亚共和国]]承续塞尔维亚和黑山在联合国、包括联合国系统内所有机构和组织的成员资格。
**黑山共和国由大会2006年6月28日第60/264号决议接纳为联合国第192个会员国。

=== 前苏联相关 ===
[[蘇聯|蘇聯]]是聯合國5個創辦國之一，其在1945年10月24日成為創始會員國。1991年12月24日，[[俄羅斯|俄羅斯]]總統[[葉利欽|葉利欽]]於[[蘇聯解體|蘇聯解體]]前夕通知[[聯合國秘書長|聯合國秘書長]]，在[[獨立國家聯合體|獨立國家聯合體]]的11個會員國支持下，「俄羅斯聯邦」將會繼承蘇聯在聯合國大會和常任理事國的席位。目前存在争议的[[卢甘斯克人民共和国|卢甘斯克人民共和国]]和[[顿涅茨克人民共和国|顿涅茨克人民共和国]]被联合国认定为是[[乌克兰|乌克兰]]的一部分，[[阿布哈兹|阿布哈兹]]和[[南奥塞梯|南奥塞梯]]为[[格鲁吉亚|格鲁吉亚]]的一部分，[[德涅斯特河沿岸|德涅斯特河沿岸]]为[[摩尔多瓦|摩尔多瓦]]的一部分。而[[克里米亚自治共和国|克里米亚自治共和国]]和[[塞瓦斯托波尔|塞瓦斯托波尔]]由于事实脱离[[乌克兰|乌克兰]]后直接并入[[俄罗斯联邦|俄罗斯联邦]]并未独立，所以其在联合国没有席位问题。其他前蘇聯成員國目前全部皆是聯合國成員國：
* [[白俄羅斯|白俄羅斯]]和[[烏克蘭|烏克蘭]]都在1945年10月24日加入聯合國，皆是聯合國創始會員國。1991年兩國重新獨立以前在聯合國的席位名稱分別是[[白俄羅斯蘇維埃社會主義共和國|白俄羅斯蘇維埃社會主義共和國]]和[[烏克蘭蘇維埃社會主義共和國|烏克蘭蘇維埃社會主義共和國]]。
* [[愛沙尼亞|愛沙尼亞]]、[[拉脫維亞|拉脫維亞]]、[[立陶宛|立陶宛]]在1991年9月17日加入聯合國。
* [[亞美尼亞|亞美尼亞]]、[[阿塞拜疆|阿塞拜疆]]、[[哈萨克斯坦|哈薩克]]、[[吉尔吉斯斯坦|吉爾吉斯]]、[[摩爾多瓦|摩爾多瓦]]、[[塔吉克斯坦|塔吉克]]、[[土库曼斯坦|土庫曼]]、[[乌兹别克斯坦|烏茲別克]]在1992年3月2日加入聯合國。
* [[格魯吉亞|格魯吉亞]]在1992年7月31日加入聯合國。
