'''联合国安全理事会否决权'''，简称'''一票否决权'''，是指聯合國安全理事會的5個[[聯合國安全理事會常任理事國|常任理事國]]（中国、法国、俄罗斯、英国和美国）各自擁有对[[联合国安全理事会决议|联合国安全理事会决议]]草案的[[否決權|否決權]]，以达至对具有强制性的安理会决议的“[[权力平衡|大国一致]]”。

即使安理會決議獲大部分[[联合国安全理事会成员国列表|成員國]]支持，常任理事國仍可投反對票（即行使否決權），使任何安理會決議無效，但安理會仍可對該議案進行討論；若常任理事國於投票時缺席或投棄權票，則不視為否決。

== 歷史 ==
否決權是為了保障聯合國創始國——即[[第二次世界大戰|第二次世界大戰]]的主要戰勝國——的權益而設立。

於1944年的聯合國成立會議上，通過由美（[[美利堅合眾國|美利堅合眾國]]）、中（[[中華民國|中華民國]]）、苏（[[蘇維埃社會主義共和國聯盟|蘇維埃社會主義共和國聯盟]]）、英（[[大不列顛及北愛爾蘭聯合王國|大不列顛及北愛爾蘭聯合王國]])、以及法（[[法蘭西第四共和國|法蘭西第四共和國]]）五國成為常任理事國。

當時法國已戰敗，被[[納粹德國|納粹德國]]佔領，但其[[國際聯盟|國際聯盟]]常任理事國地位、殖民力量和參與[[盟軍|盟軍]]活動的[[自由法國|自由法國]]使法國與其餘四國享有同等地位；而且當時英國积极为法國爭取法國的常任理事國資格，用以对抗苏联。1958年10月4日後，[[法蘭西第五共和國|法蘭西第五共和國]]建立。于該年取代[[法蘭西第四共和國|法蘭西第四共和國]]，继承联合国法国席位代表权。

1971年10月25日，[[联合国大会2758号决议|联大2758号决议]]通过后，[[中華人民共和國|中華人民共和國]]取代[[中華民國|中華民國]]在退出联合国後的中国代表权，继承其联合国安理会常任理事国的所有權利。

1991年12月27日，[[俄羅斯聯邦|俄羅斯聯邦]]在联合國取代[[蘇聯|蘇聯]][[蘇聯解體|解體]]後的俄國代表權，继承其联合国安理会常任理事国所有权利。

== 授权法源 ==
根據《[[聯合國憲章|聯合國憲章]]》第27條：
{{cquote|安全理事會每一理事國應有一個投票權。<br />
安全理事會關於程式事項之決議，應以九理事國之可決票表決之。<br />
安全理事會對於其他一切事項之決議，應以九理事國之可決票包括全體常任理事國之同意票表決之；但對於第六章及第五十二條第三項內各事項之決議，爭端當事國不得投票。}}

== 否決權的行使 ==
[[File:UNSC_veto.svg|right]]

歷史上大部分的否決權都是由[[蘇聯|蘇聯]]所行使。在蘇聯解體後，[[美國|美國]]是行使否決權最頻繁的國家。<ref>[http://www.globalpolicy.org/security/membship/veto.htm 否決權資料庫]，[[環球政策論壇|環球政策論壇]]</ref>

至2012年，5個常任理事國共行使274次否決權：
# {{RUS}}：130次（{{URS}}為121次、{{RUS}}9次）
# {{USA}}：84次
# {{GBR}}：33次
# {{FRA}}：19次
# {{CHN}}：14次（{{ROC}}為1次、{{PRC}}13次）




=== 蘇联／俄羅斯 ===
在聯合國成立早期，蘇聯外交人民委員及後來的外交部長[[維亞切斯拉夫·米哈伊洛維奇·莫洛托夫|維亞切斯拉夫·米哈伊洛維奇·莫洛托夫]]曾多次投反對票，故此他又被稱為「否決先生」（Mr. Нет）。事實上，歷史上近半數的否決權的是由蘇聯行使的──在首10年就行使了79次。莫洛托夫不斷否決接納新成員國，以報復美國拒絕接納蘇聯的[[衛星國|衛星國]]。蘇聯解體後，繼承其安理會常任理事國席位的俄羅斯僅小量行使其否決權。

=== 美國 ===
美國於1970年首次就[[羅德西亞|羅德西亞]]危機行使否決權，並於1972年起單獨行使否決權，以阻撓制裁[[以色列|以色列]]的議案通過。自那時起，美國成為最常行使否決權的常任理事國，主要用於否決批評及譴責以色列的議案（參見[[內格羅蓬特主義|內格羅蓬特主義]]）。這也是[[聯合國大會|聯合國大會]]和安理會經常起磨擦的原因，從美國於2003年發動未獲聯合國授權的[[美伊戰爭|美伊戰爭]]可見。

=== 中華民國／中華人民共和國 ===
{{See also|中國與聯合國}}
自1945年至1971年，安理會的「中國」席位都是由[[中華民國政府|中華民國政府]]代表。在此期間，[[中華民國|中華民國]]只在1955年12月13日行使了一次否決權，以阻止[[蒙古人民共和國|蒙古人民共和國]]加入聯合國，因中華民國宣稱蒙古地區為中華民國法理領土之一部分。直至1960年，[[蘇聯|蘇聯]]宣稱若蒙古不獲接納成為會員國，它會阻止所有新獨立的非洲國家加入。時值拉攏非洲友邦期間，之後中華民國便没有参加在此議題的投票以表示不滿。

自1971年10月25日[[聯合國大會2758號決議|聯合國大會2758號決議]]，[[中華人民共和國|中華人民共和國]]接手中華民國於安理會「中國」的議席後，它第一次行使否決權是於1972年。截至2019年，中華人民共和國共曾13次行使否決權。

# 1972年8月25日，中华人民共和国否决了印度、南斯拉夫和英国提出的关于接纳新会员国（[[孟加拉国|孟加拉国]]）的决议草案，以阻止孟加拉国加入聯合國。孟加拉国（原东巴基斯坦）申请加入联合国，中華人民共和國鉴于当时印度军队未撤出[[东巴基斯坦|东巴基斯坦]]，且[[巴基斯坦|巴基斯坦]]并未承认孟加拉国的独立，投票反对肢解一个主权国家<ref name="中国否决1">{{cite web|author1=观察者网|title=中国在UN投过的十次否决票|url=https://3g.163.com/war/article/BGOO44OL00014U9B.html|website=网易网军事|accessdate=2018-05-01}}</ref>。
# 1972年9月10日，中華人民共和國否决了英、法等国对索马里、几内亚、南斯拉夫三国反对以色列侵略叙利亚、黎巴嫩的修正案草案<ref name="中国否决1"/>。
# 1997年1月10日，中華人民共和國否决了安理会关于向[[危地马拉|危地马拉]]派遣联合国军事观察员的决议草案<ref name="中国否决1"/>。
# 1999年2月25日，中華人民共和國否决了安理会关于同意联合国驻[[马其顿共和国|马其顿]]预防性部署部队延期半年的决议草案，因为该国近年来频繁于台湾海峡两岸之间建交断交<ref name="中国否决1"/>。
# 2007年1月12日，中华人民共和国否决了针对美国和英国提出的有关缅甸问题的决议草案<ref name="中国否决1"/>。
# 2008年7月11日，中华人民共和國否决了美国、英国提出的关于制裁[[津巴布韦|津巴布韦]]问题的决议草案<ref name="中国否决1"/>。
# 2011年10月4日，中华人民共和国否决了英国、法国提出的关于[[叙利亚內戰|叙利亚问题]]的决议草案<ref name="中国否决1"/>。
# 2012年2月5日，中华人民共和国否决了英国、法国提出的关于[[叙利亚內戰|叙利亚问题]]的决议草案<ref name="中国否决1"/>。
# 2012年7月19日，中华人民共和国否决了英国等国家提出的关于[[叙利亚內戰|叙利亚问题]]的决议草案<ref name="中国否决1"/>。
# 2014年5月22日，中华人民共和国否决了美国等国家提出的关于[[叙利亚內戰|叙利亚问题]]的决议草案<ref>{{cite news|url=http://news.cntv.cn/2014/05/23/ARTI1400806576327631.shtml|title=中俄否决安理会“将叙利亚问题提交国际法庭”议案
|date=2014-05-23}}</ref>。
# 2016年12月5日，中华人民共和国否决了埃及、新西兰和西班牙提出的关于[[叙利亚內戰|叙利亚问题]]的决议草案。
# 2017年2月28日，中华人民共和国否决了美国等国家提出的关于[[叙利亚內戰|叙利亚问题]]的决议草案。
# 2019年2月28日，中华人民共和国否决了美国等国家提出的关于[[2019年委内瑞拉总统地位危机|委内瑞拉局势]]的决议草案<ref name="报道0328A">{{cite web |author1=李杭 |title=美俄分别提交涉委局势决议草案 安理会均未通过 |url=http://news.163.com/19/0301/08/E95VJMVC0001899N.html |website=网易网新闻 |accessdate=2019-03-22 |location=广州 |language=zh-cn}}</ref>。

中國（中華民國及中華人民共和國）為五個安理會常任理事國之中，現時使用否決權最少的國家。

=== 法國 ===
法國不常行使其否決權。它在1976年否決承認[[科摩罗|科摩罗]]對[[馬約特|馬約特]]島的主權，因當時該島居民公投是否對脫離法國管治。2003年法國威脅美國否決有關[[美伊戰爭|美國入侵伊拉克]]的1441號決議，法、美兩國因此起磨擦；而巴黎由於跟華盛頓和倫敦在此議題上交惡，站向北京與莫斯科一邊。除此在外，法國在大部分決議均與英國、美國立場相近。

=== 英國 ===
1956年，英國曾與法國聯手否決美國要求兩國自[[第二次中東戰爭|第二次中東戰爭]]撤兵的議案。但是，當美國依據[[聯合國大會377號決議|聯合國大會377號決議]]召開緊急會議，並通過撤兵決議後，兩國便答應撤軍。英國也曾就[[羅德西亞|羅德西亞]]獨立問題上行使否決權。

== 爭議 ==
=== 重組否決權 ===
聯合國安理會內的否決權是否合適，一直引起爭議。主要的論點包括：常任理事國已經不再代表聯合國內最穩定和負責任的成員國，而且否決權拖慢重要的決策過程。

由於安理會成立至今，全世界的政治和經濟形勢已大有改變，有爭論究竟現任常任理事國是否最合適擁有否決權。另一反對現有保留否決權的論點是，它不利於迅速和平衡的政治決策。有言論質疑常任理事國可用否決權作為其盟邦的政治支援，如美國就曾多次否決譴責其盟邦以色列的議案；同理，北京和莫斯科也曾為朝鲜、伊朗、伊拉克、敘利亞、利比亞、委內瑞拉等而對華府和倫敦祭出否決權。

支持保留否決權者相信這對於現今[[地緣政治|地緣政治]]仍是必須的；而且就算取消否決權，安理會仍會在全球議題上，受到「少數服從多數」的問題影響<ref>{{cite web |url=http://www.mtholyoke.edu/acad/intrel/schles.htm |title=存档副本 |accessdate=2006-09-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20060906102312/http://www.mtholyoke.edu/acad/intrel/schles.htm |archivedate=2006-09-06 }}</ref>。

===擴展安理會常任理事國議席===
應否增設常任理事國議席的議題引起熱烈討論，包括新常任理事國應否擁有否決權的議題。現任常任理事國樂意增設議席，但對擴展否決權抱不樂意甚至反對態度，聲稱此舉會癱瘓安理會，使其失去效用。除此之外，由[[印度|印度]]、[[德國|德國]]、[[巴西|巴西]]和[[日本|日本]]組成的「[[四國聯盟|四國聯盟]]」反駁說現時的安理會只代表1946年的世界，而非2007年的。此四國聲稱，安理會須代表更多人口，否則會失去重要性，使聯合國淪為[[國際聯盟|國際聯盟]]般。四國聯盟已展開外交遊說，使更多國家支持他們。

但由于国家利益或历史因素，“四国联盟”入常被现有的常任理事国或其他成员国所反对。
*中國反對日本成為常任理事國。
*美國只接受日本、印度和另一個非洲國家入常，但因巴基斯坦反對印度入常，因此美国政府不主张赋予新常任理事国否决权。
*俄羅斯僅支持印度入常，对其他國家态度模糊。
*英国、法国反对德国入常。<ref>{{cite news|url=http://www.thehindu.com/news/international/article2233806.ece |location=Chennai, India |work=The Hindu |first=Ananth |last=Krishnan |title=China ready to support Indian bid for UNSC |date=16 July 2011}}</ref>
*意大利等國發起的「[[咖啡俱樂部|咖啡俱樂部]]」一直反對擴充常任理事國的名額。
