'''聯邦三角站'''（{{lang-en|Federal Triangle Station}}）是[[華盛頓地鐵|華盛頓地鐵]][[華盛頓地鐵橘線|橘線]]、[[華盛頓地鐵藍線|藍線]]和[[华盛顿地铁银线|银线]]交會的[[地鐵車站|地鐵車站]]。本站的出口設在Ariel Rios聯邦大廈的下方。
[[File:Federal_Triangle_pylon.jpg|250px]]
[[File:FederalTriangle_WashingtonDC_2009.jpg|300px]]

== 位置 ==
本站所在地是華府一個聯邦辦公樓聚集的地方，站名取自由賓州西北大道、憲法西北大道及15街劃出的三角形區域，即所謂的「[[聯邦三角|聯邦三角]]」。這塊區域包含了[[美國司法部|美國司法部]]、[[美國商務部|美國商務部]]、國稅局及[[美國國家環境保護局|美國國家環境保護局]]等聯邦機構的總部大樓。

== 車站結構 ==
聯邦三角站採用簡單的[[島式月台|島式月台]]布局，有三條藍線银线及橘線共用的軌道：D1為南向軌道，通往華府以東的地區；D2則為北向軌道，通往華府以西的地區，<ref name="tracks">[http://mysite.verizon.net/cambronj/ John R Cambron track maps] {{Wayback|url=http://mysite.verizon.net/cambronj/ |date=20140902033747 }} ''John R Cambron'' Retrieved 2008-09-11</ref>月台邊緣則如同華盛頓地鐵大部分車站，設有閃爍警示燈。2008年，華盛頓都會區交通局在[[畫廊-中國城站|畫廊-中國城站]]新增紅色[[LED|LED]]燈的測試成功後，便將之增設於本站及市中心其他車站內的月台邊緣。<ref name="lights">[http://wmata.com/about/met_news/PressReleaseDetail.cfm?ReleaseID=2029 Metro installing more red platform edge lights]{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }} ''WMATA'' Retrieved 2008-09-11</ref>聯邦三角站是華盛頓地鐵最早採用格狀拱形內牆的車站之一，但後來其他地鐵站均改採更簡潔的混凝土拱牆，而捨棄了聯邦三角站的這種形式。

本站只有一個通往月台層的入口，位在月台中段略偏南之處，<ref name="nycs">[http://world.nycsubway.org/us/washdc/blue.html Blue line] ''NYCSubway'' Retrieved 2008-09-11</ref>並有[[電扶梯|電扶梯]]聯絡月台上層的中間層及街道層。在本站內的天花板上有個扁平的方塊，是第二個入口將可能興建的位置（但仍未開始使用），在[[國家文物館-海軍紀念館-賓州區站|國家文物館-海軍紀念館-賓州區站]]等地鐵站內也有類似情形。

== 出軌事故 ==
1982年1月13日時，一列橘線列車在本站與史密森尼站之間一個失靈的[[轉轍器|轉轍器]]出軌，造成3死25傷，是華盛頓地鐵第一次造成人員死亡的事故<ref name = "January 14, 1982_1">{{cite news | title = Metro Train -Derails; 3 Die | author = Stephen J. Lynton | work = The Washington Post | date = January 14, 1982 | accessdate = 2009-06-25}}</ref>。
