'''御峰'''（'''The Summit'''），是一幢位於香港島[[灣仔區|灣仔區]]東半山[[司徒拔道|司徒拔道]]的高級住宅，鄰近同一建築師設計的[[曉廬|曉廬]]<ref>{{cite web | URL=https://books.google.com.hk/books?id=w7vsDQAAQBAJ&pg=PA347&lpg=PA347&dq=御峰the+summit&source=bl&ots=cdvmblNFKQ&sig=hdC_bM9LiOqQLwmL3MRyHWnZicU&hl=en&sa=X&ved=0ahUKEwjYh43GtMrUAhUQhbwKHfCbDFA4KBDoAQgrMAc#v=onepage&q=御峰the%20summit&f=false |title=城傳立新──香港城市規劃發展史（1841-2015） |author=何佩然}}</ref>。目前，此樓宇全部由[[恆隆地產|恆隆地產]]持有，為同區少有的出租豪宅。<ref>{{cite web |url=http://hk.apple.nextmedia.com/financeestate/art/20030202/3098449 |title=御峰高聳入雲鶴立東半山 |date=2003年2月2日 |publisher=蘋果日報}}</ref>

==戶型==
御峰設有共54伙複式出租住宅，分為標準複式及相連複式單位；前者實用面積為2,390平方呎，而後者則為4,771平方呎。而在實用率方面，二者均為79%至80%。<ref>http://www.thesummit.com.hk/zh-hk/show-suites 御峰-示範單位</ref>
