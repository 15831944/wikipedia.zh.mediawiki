'''御風者'''（{{lang-en|ALPS}}，為{{lang|en|Advanced Large-scale Parallel Supercluster}}的縮寫<ref>{{Cite web |url=http://top500.org/system/details/10815 |title=ALPS |accessdate=2011-06-22 |work=National Center for High-performance Computing, Taiwan |publisher=top500.org |date=2011-06-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20110629080406/http://top500.org/system/details/10815 |archivedate=2011-06-29 }}</ref>）是一台[[超級電腦|超級電腦]]，位於[[中華民國|中華民國]][[財團法人國家實驗研究院|國家實驗研究院]][[國家高速網路與計算中心|國家高速網路與計算中心]]（簡稱「國網中心」）。它是由[[中華民國|中華民國]][[行政院|行政院]][[國科會|國科會]]所支持的國家高速網路與計算中心，和電腦製造商[[宏碁|宏碁]]公司所共同建置<ref>{{Cite web |url=http://www.acer.com.tw/ac/zh/TW/press/2011/19277 |title=Acer completes fastest supercomputer in Taiwan Ranks #42 in Top500® - 177 TFLOPS supercomputer, scalable with flexibleservices |accessdate=2011-06-22 |author= |work= |publisher=Acer |date=2011-06-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20150131033850/http://www.acer.com.tw/ac/zh/TW/press/2011/19277 |archivedate=2015-01-31 }}</ref>，是台灣到2010年為止，所建置出運算能量最大的[[計算機|計算機]]系統<ref name="new1" />。在2011年6月發佈的[[TOP500|TOP500]]世界五百大超級電腦排名中，「御風者」排名'''第42名'''<ref>{{Cite web |url=http://top500.org/list/2011/06/100 |title=TOP500 List - June 2011 (1-100) |accessdate=2011-06-22 |work= |publisher=top500.org |date=2011-06-20 }}</ref>，是台灣繼2007年登上全球'''第35名'''、'''49名'''後，第二台能躋身進入前50強的超級電腦<ref>{{Cite web |url=http://www.nchc.org.tw/tw/news/index.php?NEWS_ID=61 |title=台灣高速計算效能首度登上全球前35強 |accessdate=2011-06-22 |work= |publisher=國網中心 |date=2007-06-28 }}</ref>。「御風者」是台泛用型的超級電腦，設計理念為適合各領域的研究，不是為單一研究領域而設計，並且考量到低耗電的營運模式，因而在2011年6月發佈的[[Green500|Green500]]世界500大綠色超級電腦排名中名列'''第25名'''<ref>{{Cite web |url=http://www.green500.org/lists/2011/06/top/list.php?from=1&to=25 |title=Green500 List - June 2011 (1-25) |accessdate=2011-06-30 |work= |publisher=top500.org |date=2011-06-30 }}{{dead link|date=2018年2月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，是台灣到2010年為止所建置出的超級電腦中最節能的主機。

==命名緣由==
「御風者」之名，是以《[[莊子_(書)|莊子]]·[[逍遙遊|逍遙遊]]》裡描述[[戰國時代|戰國時代]][[列子|列子]]潛心修道九年後「御風而行」為典故來命名。<ref>{{Cite web |url=http://mag.udn.com/mag/digital/storypage.jsp?f_MAIN_ID=320&f_SUB_ID=2942&f_ART_ID=326299 |title=我超級電腦「御風者」 運算全球第42快 |accessdate=2011-06-22 |author=蔡永彬 |work= |publisher=聯合新聞網 |date=2011-06-23 }}</ref>

==歷史==
台灣在2008年「第八次全國科技會議」決議中，揭示強化國內高速計算的政策方向；在[[行政院|行政院]][[國科會|國科會]]支持下，國網中心負責建置，<ref name="new1" />2010年由[[宏碁|宏碁]]得標負責系統整合，「御風者」座落在[[台中科學園區|台中科學園區]]。<ref name="new4" />

國網中心自1993年6月起即以第485名進入到TOP500排行榜裡<ref>{{Cite web |url=http://www.top500.org/site/history/1248 |title=National Center for High Performance Computing - Site history |accessdate=2011-06-22 |work= |publisher=top500.org |date= }}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>，期間數次取得較佳的名次，分別為2002年6月以IBM系統得到TOP500'''第60名'''，2007年6月以當時新完成的「IRIS」拿到'''第35名'''，之後因其他國家積極投資超級電腦，2009年11月被擠出500大行列，一直到2011年6月「御風者」才重返TOP500超級電腦排行榜名列'''第42名'''，但「御風者」的運算速度為「IRIS」的9倍，2011年的「御風者」的運算能力相當於2007年排名中的全球「第二名」。<ref name="new4" />

==詳細配置==
「御風者」擁有25,600顆計算核心（AMD Opteron 12 Core 2200 MHz，每顆8.8 GFlops）、73,728 GB記憶體及1074 TB儲存，計算量每秒高達177兆次浮點運算（[[FLOPS|Tflops]]），<ref name="new1">{{Cite web |url=http://www.nchc.org.tw/tw/news/index.php?NEWS_ID=169 |title=「御風者」－國網中心新建超級電腦進全球前百大 開創台灣大規模高速計算新紀元 |accessdate=2011-06-22 |work= |publisher=國網中心 |date=2011-06-22 }}</ref>相當於2萬台個人電腦，<ref name="new2">{{Cite web |url=http://www.cna.com.tw/ShowNews/Detail.aspx?pNewsID=201106220269&pType0=aIT&pTypeSel=0 |title=國產超級電腦速度全球42強 |accessdate=2011-06-22 |author=吳佳穎 |work= |publisher=中央社 |date=2011-06-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20160304114910/http://www.cna.com.tw/ShowNews/Detail.aspx?pNewsID=201106220269&pType0=aIT&pTypeSel=0 |archivedate=2016-03-04 }}</ref>較國網中心前代超級電腦「IRIS」不到20Tflops的效能提昇8.8倍。

==費用==
「御風者」機房佔地為29坪，若以相同效能來看，佔地比IRIS需使用33.5坪更小；而且系統用電800KW，每年用電成本為新台幣3千萬元，估計較之前可省下新台幣1千萬元。<ref name="new4">{{Cite web|url=http://www.ithome.com.tw/itadm/article.php?c=68297|title=國網中心正式發表「御風者」超級電腦|accessdate=2011-06-22|author=蘇文彬|work=|publisher=電週文化|date=2011-06-22|deadurl=yes|archiveurl=https://web.archive.org/web/20110625144310/http://www.ithome.com.tw/itadm/article.php?c=68297|archivedate=2011-06-25}}</ref>
