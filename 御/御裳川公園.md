'''御裳川公園'''（日文：'''{{lang|ja|みもすそ川公園}}'''）是位於[[日本|日本]][[山口縣|山口縣]][[下關市|下關市]]御裳川町地區的城市[[公園|公園]]，面積約0.48公頃<ref>{{Cite web| url=http://www.oidemase.or.jp/tourism-information/spots/15267| title=みもすそ川公園| work=山口縣的旅行、觀光情報| language=ja| accessdate=2019-03-02}}</ref>，介於[[國道9號_(日本)|國道9號]]和[[關門海峽|關門海峽]][[壇之浦|壇之浦]]之間，因為12世紀的[[壇之浦之戰|壇之浦之戰]]及19世紀的[[下關戰爭|下關戰爭]]發生於公園旁的海域，在公園內設有許多與這兩場戰爭有關的紀念設施<ref>{{Cite news| url=https://mainichi.jp/articles/20181123/ddl/k40/040/483000c| title=みもすそ川公園と赤間神宮（山口県下関市）　散策、源平から維新まで　／福岡| date=2018-11-23| newspaper=每日新聞| language=ja| accessdate=2019-03-02}}</ref>；此外由於距離跨越關門海峽的[[關門橋|關門橋]]僅約200公尺，也成為觀賞關門橋的景點。

公園在隔著國道9號的對面為{{Tsl|ja|関門プラザ|關門廣場}}，設有[[關門人行隧道|關門人行隧道]]下關側的電梯入口。

== 公園內設施 ==

; 壇之浦之戰相關

* 壇之浦古戰場遺址碑
* [[安德天皇|安德帝]]御入水之處碑
* 源義經與平知盛像：描繪源義経連續飛跳八艘船的故事。
* [[NHK|NHK]][[大河劇|大河劇]][[義經_(大河劇)|義經]]演員的銅板手印：留下手印的演員包括飾演[[源義經|源義經]]的[[瀧澤秀明|瀧澤秀明]]、飾演[[平德子|建禮門院德子]]的{{Tsl|ja|中越典子}}、飾演[[平资盛|平资盛]]的[[小泉孝太郎|小泉孝太郎]]、飾演[[平時子|二位尼]]的[[松坂慶子|松坂慶子]]。<ref name="gururich-kitaq">{{Cite web| url=http://www.gururich-kitaq.com/kanmon/cn/detail/index.php?id=13&app=0| title=御裳川公園| work=北九州市觀光網站| language=zh-tw| accessdate=2019-03-02}}</ref>

; 下關戰爭相關

* 長州砲複製品：公園內留有19世紀下關戰爭中屬於[[長州藩|長州藩]]的砲台遺跡，因此園內設立五座複製的大砲，其中一座在投入100日圓硬幣後會發出模擬的炮擊聲。<ref name="gururich-kitaq"/><ref>{{Cite web| url=http://www.city.shimonoseki.lg.jp/www/contents/1387327848698/index.html| title=みもすそ川公園内の長州砲レプリカ　リニューアル！| date=2013-12-18| publisher=下關市政府| language=ja| accessdate=2019-03-02}}</ref>

; 其他

* 御裳川碑：御裳川原本為流經此地區，從公園處游入關門海峽的小河川，目前的河口位於公園地下。
* 松本清張文學碑：作家[[松本清張|松本清張]]在小時候曾住於下關市此公園附近，在其小説『半生之記』中曾敘述在此眺望位於關門海峽對岸的{{Tsl|ja|和布刈神社}}。<ref>{{Cite web| url=http://www.stca-kanko.or.jp/sightseeing/person/| title=ゆかりの人物紹介| publisher=下関観光コンベンション協会| accessdate=2019-03-02| language=ja}}</ref>
