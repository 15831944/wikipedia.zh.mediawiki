'''御蔭杯'''（{{lang-ja|おかげ杯}}），或稱為[[福蔭杯|福蔭杯]]<ref>[http://taiwango.org.tw/ 台灣棋院]</ref>，由[[日本棋院|日本棋院]]主辦，株式会社濱田總業贊助，創辦於2010年。2014年，作為青年國際團體戰和青年國際女流個人戰的[[御蔭杯國際新鋭抗戰|御蔭杯國際新鋭抗戰]]舉行，御蔭杯冠亞軍及最佳成績的女流棋士會出戰，與中台韓棋士作賽。優勝賞金：淘汰賽300萬[[日圓|日圓]]，團體戰270万日圓，女流個人戰20万日圓。

== 名称由来 ==
由於贊助商濱田總業的所在位置於[[三重縣|三重縣]][[伊勢市|伊勢市]]之[[御蔭横丁|御蔭横丁]]。

== 參賽資格 ==
30歲以下,包括次年將入段之棋士

== 比賽規則 ==
* 採用日本圍棋規則
* 一律分先，黑先貼還六目半
* 本戰16人,採單淘汰賽形式
* 時限：沒有規定時間，每1手30秒，1分鐘之考慮時間10回
