'''御谷'''（''Pennisetum glaucum''），又名'''蜡烛稗'''、'''珍珠粟'''。

== 形态 ==
[[一年生草本|一年生草本]]。秆直立单生，[[花序|花序]]以下密生柔毛，叶鞘平滑，叶舌不明显，具长纤毛，叶片扁平；圆锥花序，紧密呈柱状，主轴粗硬，小穗倒卵形，通常孪生呈束，具有纤毛；[[颖果|颖果]]，成熟时膨胀外露。

== 分布 ==
主要分布于[[南亚|南亚]]和[[非洲|非洲]]的[[谷物|谷物]]。原产非洲，史前传到南亚，魏晉時傳入中國。

== 歷史 ==
御穀本身起源於[[非洲|非洲]][[撒哈拉沙漠|撒哈拉沙漠]]以西的地區，後來先傳到非洲南部，再傳到南亞地區。[[印度|印度]]早在二千年前就已有栽種御穀的紀錄。隨著新航路開通帶動了國際貿易，御穀先於1850年代傳入[[美國|美國]]，在1960年代傳入[[巴西|巴西]]。

== 特徵 ==
御穀屬一年生，植株可達2-3米，一年可收割2-3茬。御穀能適應在高[[鹽|鹽]]度的或酸性的[[土壤|土壤]]，因之在一些氣候乾燥的地區，如[[沙漠|沙漠]]，御穀佔有很重要的地位。

其性质和用处大概类似[[粟|粟]]米，种子可供食用，秆、叶可供饲料。御穀耐旱，是很多干燥地区贫穷农民生存离不开的重要作物。
