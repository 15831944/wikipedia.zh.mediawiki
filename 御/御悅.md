'''御悅'''（英文：{{lang|en|'''Baker Residences'''}}），是位於[[香港|香港]][[紅磡|紅磡]]的單幢住宅，樓高共26層，由[[信和置業|信和置業]]及[[市區重建局|市區重建局]]（以聯丰投資有限公司名義）發展，於2011年10月入伙。

== 歷史 ==
御悅 前身為位於「必嘉圍8-12號」2幢3層高的唐樓<ref>[http://www.ura.org.hk/tc/projects/redevelopment/ma-tau-kok/baker-court-project.aspx 市區重建局必嘉圍項目]</ref>，2003年[[市區重建局|市區重建局]]以約七千二百萬[[港元|港元]]進行收購，於2008年邀請合作發展意向書，共收到7份意向書，終只收獲1份標書，由於地盤為[[市建局|市建局]]歷來招標的最細規模項目，故成為當時歷來招標最差的項目<ref>[http://www.singtao.com/archive/fullstory.asp?andor=or&year1=2008&month1=5&day1=6&year2=2008&month2=5&day2=6&category=all&id=20080506a03&keyword1=%20&keyword2=%20 信和無對手 奪紅磡蚊型重建項目]</ref>，最後其決定與[[信和置業|信和置業]]合作<ref>[http://www.ura.org.hk/tc/media/press-release/2008/20080505-3.aspx 市建局與信和置業有限公司合作發展紅磡必嘉圍項目]</ref>。 同時該地段地契改訂為'''九龍內地段第11181號'''，地價一千萬。<ref name=landsd>http://www.landsd.gov.hk/tc/exc_mod/ptg/ptg2008c.pdf</ref>

最終[[信和置業|信和置業]]及[[市區重建局|市區重建局]]把上址發展成單幢商住項目 －御悅，原只計劃興建46個單位，後共建有68個單位，並於2012年2月開售。

== 簡介 ==
項目地盤面積約2,982平方英呎（277平方米<ref name=landsd/><ref>http://ura.org.hk/tc/projects/redevelopment/ma-tau-kok/baker-court-project.aspx</ref>），可建總樓面面積（Permitted GFA）為2,338平方米（25,162平方英呎）<ref name=bd>http://www.bd.gov.hk/english/GFA/GFA_c/KN2011026GFACe.pdf</ref><ref>地盤為乙類地盤，純非住用建築物下地積比率為15；純住用建築物下地積比率為9；綜合樓宇依香港附例123F章 21條計算。發展商選擇261平方米商業及2077平方米住宅的組合用盡地積比</ref>，惟建築樓面面積為33,197平方呎（Construction Floor Area），發水比率達31%。其中機房佔3,316平方呎，提供環保設施以及適意
設施所獲得的額外樓面分別佔2,454平方呎及2,265平方呎。<ref name=bd/><ref>[http://www.etnet.com.hk/www/tc/news/categorized_news_detail.php?category=property&newsid=ETN210831511 信置紅磡「御悅」市建局發展項目，萬呎樓面獲豁免發水率達43%]</ref>。

物業實際樓高26層，惟由於不設4、13、14及24字樓，並設有UG層及天台[[隔火層|隔火層]]，故有28字樓，其中基座佔3層，設4個商鋪及機房，其餘22層為住宅單位，當中2字樓除了2個單位外，並設有[[會所|會所]]，其設施包括遊戲室、閒坐室等，供住戶享用。

而物業共設68個單位，建築面積由約270-370呎不等，全屬開放式單位，其中4伙為連[[平台|平台]]的特色戶，而一梯有二伙至三伙，由於單位面積小，故入場費低於200萬元，被譽為近年新盤罕見的迷你上車盤<ref>[http://orientaldaily.on.cc/cnt/finance/20120222/00204_001.html 迷你新盤 御悅料200萬有找]</ref>。

== 商鋪 ==
基座地下、高層地下及一樓為商鋪，面積合共約2928方呎，發展商於2014年1月透過[[第一太平戴維斯|第一太平戴維斯]]進行[[招標|招標]]出售，意向價為2200萬元<ref>[http://www2.hkej.com/property/article/id/349768/%E7%B4%85%E7%A3%A1%E5%BE%A1%E6%82%85%E8%88%96%E4%BD%8D2200%E8%90%AC%E6%8B%9B%E6%A8%99%E5%87%BA%E5%94%AE 紅磡御悅舖位2200萬招標出售]</ref>，終在3月成功以2000萬售出<ref>[http://orientaldaily.on.cc/cnt/finance/20140313/00204_061.html 地產速報]</ref>。
