'''御器所車站'''（{{jpn|j=御器所駅|hg=ごきそえき|rm=Gokiso eki|f=y}}）是位在[[愛知縣|愛知縣]][[名古屋市|名古屋市]][[昭和區|昭和區]]御器所通三丁目。為[[名古屋市營地下鐵|名古屋市營地下鐵]]的車站之一。
<br/>因本站鄰近昭和區公所，因此在站名板中附有「昭和區役所」作為副站名。

== 可利用之鐵路路線 ==
*名古屋市營地下鐵
**[[鶴舞線|鶴舞線]] - 車站編號為'''T12'''。
**[[櫻通線|櫻通線]] - 車站編號為'''S10'''。

== 車站結構 ==
本站鶴舞線為2面2線[[側式月台|側式月台]]，櫻通線為1面2線[[島式月台|島式月台]]之[[地下車站|地下車站]]。本站所附設的[[電梯|電梯]]僅設於櫻通線月台側。

== 歷史 ==
*1977年（[[昭和|昭和]]52年）3月18日 - 鶴舞線御器所站開始營運。
*1994年（[[平成|平成]]6年）3月30日 - 本站櫻通線開始營運，成為兩線之轉乘站。
