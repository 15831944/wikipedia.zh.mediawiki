'''御桥站'''位于[[上海|上海]][[浦東新区|浦東新区]][[北蔡鎮|北蔡鎮]]，车站位于[[御桥路|御桥路]][[御青路|御青路]]，车站主体位于御桥路地下，横跨路口，呈近东西方向。为[[上海轨道交通11号线|上海轨道交通11号线]]的地下车站，于2013年8月31日启用<ref>{{Cite web |url= http://www.shmetro.com/node49/201308/con113351.htm |title= 11号线二期8月31日起通车试运营 |author= 上海地铁运管中心 |date=2013-08-29 |language= zh-cn |accessdate= 2013-08-29 }}</ref>。<br>规划中的[[上海轨道交通18号线|轨道交通18号线]]也会经过本站，成为11号线与18号线的[[换乘站|换乘站]]。

== 车站设计 ==
