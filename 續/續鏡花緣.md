《'''續鏡花緣'''》是長篇小說《[[镜花缘|鏡花緣]]》的續集，全書四十回，由[[清朝|清代]][[華琴珊|華琴珊]]創作於[[1910年|宣統二年]]。作者認為，《鏡花緣》原書的某些情節似乎未完，因此創作此續作。 

== 主要內容 ==
本書緊接原作最後一回，從女試的第二年，[[唐中宗|唐中宗]]復位開始寫起。對於原作角色，主要描寫了陰若花、顏紫綃、唐閨臣、易紫菱等人，加入的新角色主要有黃蕊珍、梅鳳英、坤蕙芳等人。在結局中，作者安排所有仙女轉世回歸天堂，「不至久溷塵凡」。

== 版本 ==
1987年日本汲古書院出版的《增補中國通俗小說書目》([[大塚秀高|大塚秀高]]編著)根據鄭逸梅的記載，稱此書為「周越然舊藏」，1992年10月，[[书目文献出版社|書目文獻出版社]]出版了此書稿本的影印本，歸於[[北京圖書館|北京圖書館]]稿本鈔本叢刊，說明手稿收藏於北京圖書館。在書的第一頁有周越然的印章，第一回第一頁印有周越然、北京图书馆藏、[[徐鼒|四明敝帚斋]]藏、仰周所鉴五個印章。可見此書完成時沒有刊印，只有手稿留存，[[北京圖書館|北京圖書館]]藏有唯一的孤本<ref>{{Cite book|title=500种明清小说博览 下|last=张兵主编|first=|publisher=上海：上海辞书出版社|year=2005|isbn=7-5326-1388-7|location=|pages=1637}}</ref>。

== 關於作者 ==
作者華琴珊為清末民初人，號醉花生<ref>{{Cite book|title=中国古代小说总目 白话卷|last=石昌渝主编|first=|publisher=太原：山西教育出版社|year=2004.09|isbn=7-5440-2816-X|location=|pages=465}}</ref>。影印本的出版說明認為他「大致是科場中失意之人」<ref>{{Cite book|title=《續鏡花緣》出版說明|last=薛|first=英|publisher=[[書目文獻出版社|書目文獻出版社]]|year=1992年10月|isbn=7-5013-0949-3|location=|pages=2頁}}</ref>，但書中顧學鵬的《序》說他「槐黃十度，有誌未償」，胡宗堉的《序》稱他「秉性豪邁，放懷詩酒，落拓不羈。詩賦、策論、雜著，各擅勝場，尤工製藝。棘闈屢薦，終不獲售。及科舉既廢，遂絕意功名」，可見他的「絕意功名」是由於廢除了[[科举|科舉]]製度，並非主觀上放棄追求功名富貴。從書中內容可以看出他對於封建事物的留戀，比如[[缠足|三寸金蓮]]、[[重农主义|重農抑商]]思想等。
