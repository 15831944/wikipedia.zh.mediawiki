'''兒歌'''，又稱'''童謠'''，是一種以[[兒童|兒童]]為主要對象的[[歌曲|歌曲]]，曲色旋律多半較為簡單，歌詞亦簡單琅琅上口，使兒童容易記憶（但未必各年齡層都能充分理解）。這類的歌曲可能是民謠（如[[丟丟銅仔|丟丟銅仔]]）、模擬[[兒童|兒童]]的思想和內心[[世界|世界]]所做的曲子（如[[妹妹背著洋娃娃|妹妹背著洋娃娃]]），或是大人將現實社會與未來期許簡化放進歌曲（如[[只要我長大_(兒歌)|只要我長大]]），今代也有相當數量為[[動畫|動畫]][[卡通|卡通]]的主題曲（如[[櫻桃小丸子|櫻桃小丸子]]主題曲）。歷史上[[民變|民變]]的領導人，往往利用童謠散布政治立場，甚至讓民眾認為是一種預言。

== 國語/普通話 ==
[[中華民國|中華民國]]大陆时期的一首歌曲《[[国民革命歌|国民革命歌]]》歌词为：

{{Cquote|打倒列强，打倒列强，除军阀，除军阀；

努力国民革命，努力国民革命，齐奋斗，齐奋斗。

打倒列强，打倒列强，除军阀，除军阀；

国民革命成功，国民革命成功，齐欢唱，齐欢唱。}}

后来又流行了一首相同曲调的儿歌《[[两只老虎|两只老虎]]》：

{{Cquote|两只老虎，两只老虎，跑得快，跑得快，一只没有尾巴，一只没有耳朵，真奇怪，真奇怪。}}

[[中华人民共和国|中华人民共和国]]成立後，出現了不少以[[革命|革命]]為題材的兒歌，这些歌曲都曾经流传于中国，但由于中国政府的政策改变，很多儿歌都已经失传了。

下面是两首女生在玩跳[[橡皮筋|橡皮筋]]时，一边跳一边唱的儿歌。

{{Cquote|[[江姐|江姐]]江姐好江姐，你是人民的好江姐，[[东方红|东方红]]，[[太阳|太阳]]升，中国出来一个[[毛泽东|毛泽东]]，很[[革命|革命]]的，跳得头发都散了。}}

{{Cquote|爷爷是个老[[红军|红军]]，爸爸妈妈都是[[解放军|解放军]]，姐姐参加[[红卫兵|红卫兵]]，我也参加[[红小兵|红小兵]]，全家都是[[毛主席|毛主席]]的兵。}}

下面这首儿歌《[[一分钱|一分钱]]》随着时代的改变，演化成很多的版本，这是原始版本：

{{Cquote|我在马路边，捡到一分钱，把他交给[[警察|警察]]叔叔手里边，叔叔拿着钱，对我把头点，我高兴的说一声，叔叔再见！}}

下面这首儿歌是为了纪念[[中国|中国]]的[[革命战士|革命战士]][[董存瑞|董存瑞]]。

{{Cquote|[[董存瑞|董存瑞]]十八歲，參加革命[[游擊隊|游擊隊]]，炸碉堡犧牲了，全國人民流[[眼淚|眼淚]]。}}

有一些则传唱至今：比如[[电影|电影]]《[[祖国的花朵|祖国的花朵]]》插曲《[[让我们荡起双桨|让我们荡起双桨]]》。

== 粵語 ==
{{Main|粵語童謠}}

[[粵語|粵方言]]地區亦有不少獨特的兒歌，貼近人們的生活，琅琅上口。例如下面這首在[[廣州|廣州]]廣為流傳的「落雨大」：

{{Cquote|落雨大，[[水浸街|水浸街]]，阿妈担柴上街卖，阿嫂出街绣花鞋，花鞋花袜花腰带，珍珠蝴蝶两边排。}}

上面的「董存瑞」兒歌亦有粵語版本：
{{Cquote|董存瑞，十八歲，為國犧牲炸堡壘，炸到堡壘ngok7 ngok7脆{{NoteTag|ngok7 ngok7脆：形容脆。}}，全國人民流眼淚。}}

但由於受[[推廣普通話|推廣普通話]]的影響，不少粵語童謠有消失的危險<ref>{{cite news |url = http://informationtimes.dayoo.com/html/2012-08/22/content_1847806.htm|title=久违嘅粤语童谣 |publisher=信息时报|date=2012-08-22|language=yue|deadurl=yes|archiveurl = https://web.archive.org/web/20140904084405/http://informationtimes.dayoo.com/html/2012-08/22/content_1847806.htm |archivedate=2014-09-04}}</ref>。許多[[幼兒園|幼兒園]]教師不會粵語，只教[[普通話|普通話]]兒歌，導致小朋友不會唱粵語兒歌，對此，有專家建議將這些[[廣府文化|廣府文化]]編入教材。<ref>{{cite news|url=http://news.sina.com.cn/c/2009-04-17/020615478349s.shtml|title=幼儿园不闻广味童谣|publisher=羊城晚报|date=2009-04-17}}</ref>

== 历史上的儿歌 ==
历史上一些著名的政治事件和儿歌有关，例如：

*[[秦末|秦末]]：大楚興、[[陳勝|陳勝]]王。（假託為狐狸所唱）

*[[西汉|西汉]]：燕燕，尾涎涎，张公子，时相见。木门仓琅根，燕飞来，啄皇孙。皇孙死，燕啄矢。（[[赵飞燕|赵飞燕]]）

*[[东汉|东汉]]：千里草，何青青，十日卜，不得生（[[董卓|董卓]]）

*[[孫吳|孫吳]]：[[诸葛恪|诸葛恪]]，何弱弱；芦单衣，蔑钩络；何处寻，石子冈。

*[[孫吳|孫吳]]：宁饮[[建业|建业]]水，不食[[武昌鱼|武昌鱼]]。宁还建业死，不止武昌居。（[[孙皓|孙皓]]迁都）

*[[北朝|北朝]]：百升飞上天，明月照[[长安|长安]]（[[北齐|北齐]]名将[[斛律光|斛律光]]）
