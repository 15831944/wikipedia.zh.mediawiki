'''神奇女侠'''自从1941年12月在《{{Link-en|全明星漫画|All Star Comics}}》第8期上首次登场之后，{{Link-en|戴安娜·普林斯|Diana Prince}}{{\}}[[神奇女侠|神奇女侠]]这个形象除了在漫画书之外，还在其他多种媒体上出现。这些类型包括：[[电视动画|动画剧]]、[[录影带首映|直接发行DVD]]式[[剧场版动画|动画电影]]、[[电子游戏|电子游戏]]、70年代的[[电视剧|真人电视剧]]（例如：《{{Link-en|神奇女侠 (电视剧)|Wonder Woman (TV series)|神奇女侠}}》、2014年发行的[[计算机动画#电影及电视剧|计算机动画电影]]（例如：《[[樂高玩電影|乐高大电影]]》）、真人版[[DC擴展宇宙|DC拓展宇宙]]电影（例如：2016年的《[[蝙蝠俠對超人：正義曙光|蝙蝠侠大战超人：正义黎明]]》、2017年的《[[神力女超人_(2017年電影)|神奇女侠]]》和《[[正義聯盟_(電影)|正义联盟]]》）。她还将在2017年发行的《神奇女侠2》上继续出现。

|last = Kroll |first = Justin |date = 2013-11-07 |work = Variety |accessdate = 2013-12-02 }}</ref><ref name=hr2013>{{cite web |last = Kit |first = Borys |title = 'Wonder Woman' Gal Gadot Signs Three-Picture Deal with Warner Bros. |url = http://www.hollywoodreporter.com/heat-vision/warner-bros-finds-wonder-woman-661991 |work = The Hollywood Reporter |date = 2013-12-04 |accessdate = 2017-06-08 }}</ref><ref name="variety.com">{{cite web |last = Kroll |first = Justin |title = 'Wonder Woman' Gal Gadot Signs Three-Picture Deal with Warner Bros. |url = https://variety.com/2014/film/news/wonder-woman-gal-gadot-signs-three-picture-deal-with-warner-bros-1201067961/ |work = Variety |date = 2014-01-23 |accessdate = 2014-01-23 }}</ref><ref name=dt2014>{{cite web |last = Hawksley |first = Rupert |title = Wonder Woman to be first female-led superhero film since 2005 |url = http://www.telegraph.co.uk/culture/film/film-news/11166315/Wonder-Woman-to-be-first-female-led-superhero-film-since-2005.html |work = The Daily Telegraph |date = 2014-10-06 |accessdate = 2017-06-08 }}</ref> 一些漫画迷最初对这个选择做出了过激反应，他们批评了盖尔·加朵的外貌。<ref name=abc1228>{{cite web |url = http://abcnews.go.com/blogs/entertainment/2013/12/gal-gadot-defends-wonder-woman-casting/ |title = Gal Gadot Defends 'Wonder Woman' Casting |accessdate = 2017-06-09 |last = Fisher |first = Luchina |date = 2013-12-28 |publisher = ABC News |deadurl = yes |archiveurl = https://web.archive.org/web/20170807235431/http://abcnews.go.com/blogs/entertainment/2013/12/gal-gadot-defends-wonder-woman-casting/ |archivedate = 2017-08-07 }}</ref> 查克·史奈德在看过评论之后表示他在选择盖尔·加朵之前面试了很多女演员，但盖尔·加朵的优势在于她漂亮、强壮，而且她还是一个友善的人，同时她还很凶猛；而查克·史奈德表示他们寻找的就是这样的人。<ref name=somekind>{{cite web |url = http://filmink.com.au/2016/some-kind-of-wonderful/ |title = Some Kind of Wonderful |accessdate = June 9, 2017 |last = Pringle |first = Gill |date = 2016-01-05 |work = Filmink }}</ref>

==== 神奇女侠（2017年） ====
{{Main|神奇女侠 (2017年电影)}}

[[File:Gal_Gadot_as_Wonder_Woman.jpg|200px]]所饰演的[[神奇女侠|神奇女侠]]宣传照（《[[蝙蝠侠大战超人：正义黎明|蝙蝠侠大战超人：正义黎明]]》）]]

[[盖尔·加朵|盖尔·加朵]]在2017年的[[超级英雄电影|超级英雄电影]]《[[神奇女侠_(2017年电影)|神奇女侠]]》担任主角，这是DC扩展宇宙的第四部电影作品，也是第一部以神奇女侠为主角的院线电影。<ref name=ew16>{{cite news |url = http://ew.com/article/2016/07/15/wonder-woman-gal-gadot-interview/ |title = Wonder Woman: Gal Gadot interview |last = Sperling |first = Nicole |date = 2016-07-15 |access-date = 20170-06-22|work = Entertainment Weekly }}</ref>{{efn|1=1974年版的电影《神奇女侠》最终并没有在院线上映，而是在[[美国广播公司|美国广播公司]]旗下的电视频道播出，因此不能算作“院线电影”。}} 这部电影由[[派蒂·珍金斯|派蒂·珍金斯]]执导，{{Link-en|艾伦·海恩伯格|Allan Heinberg}}担任剧本执笔，而故事编剧则由艾伦·海恩伯格、扎克·施奈德（{{lang|en|Zack Snyder}}）和杰森·福克斯（{{lang|en|Jason Fuchs}}）担任。同时本片的其他出演明星还包括：[[克里斯·潘恩|克里斯·潘恩]]、[[羅蘋·萊特|罗宾·怀特]]、[[丹尼·休斯顿|丹尼·休斯顿]]、[[大卫·休里斯|大卫·休里斯]]、[[康妮·尼尔森|康妮·尼尔森]]和[[埃琳纳·安娜亚|埃琳纳·安娜亚]]。派蒂·珍金斯成为影史上第一位担任制片厂超级英雄题材电影的女导演。<ref>{{cite news |url = http://www.mercurynews.com/2017/05/31/how-wonder-woman-lassoed-the-first-female-director-of-a-studio-superhero-movie-2/ |title = How 'Wonder Woman' lassoed the first female director of a studio superhero movie |work = The Mercury News |first = Bob |last = Strauss |date = 2017-05-31 }}</ref>

该片的故事发生于1918年，讲述了天堂岛亚马逊族的戴安娜公主的成长故事。当美国飞机驾驶员史蒂夫·特雷弗（{{lang|en|Steve Trevor}}）所驾驶的飞机在天堂岛附近海域坠毁时，戴安娜公主将他救了下来。他告诉戴安娜，天堂岛外的人类世界正在发生[[第一次世界大战|第一次世界大战]]。想要结束这场战争的戴安娜公主跟随史蒂夫·特雷弗离开了天堂岛，而在这个过程中，她逐渐成长为神奇女侠。该电影于1996年立项，派蒂·珍金斯在2015年签约成为导演。电影的主要拍摄期从2015年11月21日开始，主要的取景地包括：[[英国|英国]]、[[法国|法国]]和[[意大利|意大利]]。最终于2016年5月9日结束拍摄，而这一天也刚好是《神奇女侠》系列漫画的主要创作者之一的[[威廉·莫尔顿·马斯顿|威廉·莫尔顿·马斯顿]]的123岁冥寿。随后在2016年11月进行了一些补拍工作。

《神奇女侠》于2017年5月15日在[[上海|上海]]首映，随后于2017年6月2日在美国以2D、[[3D電影|3D]]和[[IMAX#IMAX_3D|IMAX 3D]]等形式发行。本片收到了评论界的积极评价，认为在执导、表演、动作设计和音乐编排上有很好的表现。这部电影在美国国内创下了女性导演执导影片的最高开盘记录（1.033亿美元）、女性主演的漫画改编电影的最高开盘纪录、女性执导真人影片的最高票房记录和美国国内最卖座的超级英雄诞生电影。它在全球拿下超过8.12亿美元的票房，成为2017年度全球票房第六高的电影。<ref name=reviews>{{cite news |url = https://www.independent.co.uk/arts-entertainment/films/news/wonder-woman-review-roundup-critics-dc-superman-batman-a7762471.html |title = Wonder Woman review roundup – Critics conclude Patty Jenkins' film better than Batman v Superman |first = Jack |last = Shepherd |date = 2017-05-30 |work = The Independent }}</ref>

=== 真人剧集 ===
{{Main|神奇女俠 (1974年電視電影)}}
{{Main|神力女超人 (電視劇)}}

《神奇女侠》电视剧既指1974年拍摄的电视电影版的《[[神奇女俠_(1974年電視電影)|神奇女侠]]》，也指1975年到1979年播出的电视连续剧版的《[[神力女超人_(電視劇)|神奇女侠]]》。由{{Link-en|文森特·麦克伊维提|Vincent McEveety}}执导并由{{Link-en|卡西·李·克罗西比|Cathy Lee Crosby}}主演的1974年版《神奇女侠》其实是[[美国广播公司|美国广播公司]]筹拍连续剧版《神奇女侠》的试播集。但是这个试播集被评价为“可敬却毫不神奇”，并且美国广播公司最终也没有购买这个试播集。<ref name = Shales>{{cite news |last = Shales |first = Tom |title = Wonder Woman Tries Comeback |work = The Washington Post |date = 1975-11-07 }}</ref> 

相反，华纳兄弟公司为美国广播公司开发了一套完全迥异的神奇女侠的电视概念。这套概念中的神奇女侠形象更符合原创作者[[威廉·莫尔顿·马斯顿|威廉·莫尔顿·马斯顿]]笔下的传统形象，这种1968－1972年代的形象最终影响了试播集的拍摄。《[[神力女超人_(電視劇)|神奇女侠]]》电视连续剧从1975年开始首播，[[琳達·卡特|琳达·卡特]]成为主演。不过卡西·李·克罗西比表示她原本也有机会出演这个版本的神奇女侠。<ref>{{cite news |last = Joby |first = Tom |title = Cathy Crosby turns down 'Wonder Woman' offer |agency = Associated Press |date = 1980-05-12 }}</ref>

=== 废弃项目 ===
==== 谁怕戴安娜·普林斯 ====
《谁怕戴安娜·普林斯（{{lang|en|''Who's Afraid of Diana Prince''}}）》是一部在1967年被提出来的电视剧制作计划，但是最终仅仅拍摄了一个短小的试播集。<ref>[https://www.bleedingcool.com/2017/05/31/lauren-looks-back-are-you-afraid-of-diana-prince/ Lauren Looks Back: Who’s Afraid Of Diana Prince? The Forgotten 1967 Wonder Woman TV Pilot]</ref> 由于《{{Link-en|蝙蝠侠 (电视剧)|Batman (TV series)|蝙蝠侠}}》电视剧版的成功，该剧的制作人{{Link-en|威廉·多齐尓|William Dozier}}委托{{Link-en|斯坦·哈特|Stan Hart}}和{{Link-en|拉里·西格尔|Larry Siegel}}撰写了试播集的剧本。但是随后这个剧本被要求交给《蝙蝠侠》电视剧剧本的编剧{{Link-en|斯坦利·拉尔夫·罗斯|Stanley Ralph Ross}}重写，因为哈特与西格尔的剧本被认为是不合适的。<ref>{{cite web |last =  |first =  |authorlink =  |title = IMDB: Wonder Woman: Who's Afraid of Diana Prince? (writers) |work =  |publisher = imdb. com |url = http://www.imdb.com/title/tt0293979/fullcredits#writers |doi =  |accessdate = 2009-10-01 }}</ref><ref>{{cite web |last =  |first =  |authorlink =  |title = Wonderland: 1967 Presentation, Who's Afraid of Diana Prince? |work =  |publisher = wonderland-site.com |url = http://www.wonderland-site.com/html/series/wonder/60s/ww67-00-00.htm |doi =  |accessdate = 2009-10-01 }}</ref> 试播集被拍摄出来的部分长度不足5分钟，由《蝙蝠侠》电视剧的幕后制作公司格林威制作公司（{{lang|en|Greenway Productions}}）负责拍摄，而拍摄的标题就是《谁怕戴安娜·普林斯（{{lang|en|''Who's Afraid of Diana Prince''}}）》。<ref>{{cite book |last = Daniels |first = Les |authorlink =  |author2 = Chip Kidd |title = Wonder Woman: The Life and Times of the Amazon Princess |publisher = Chronicle Books |year = 2000 |location =  |url =  |doi =  |id =  |isbn = 0-8118-4233-9 |page = 120 }}</ref> 在试播集中，由{{Link-en|小罗伯特·沃克|Robert Walker (actor, born 1940)}}的妻子艾莉·伍德·沃克（{{lang|en|Ellie Wood Walker}}）饰演戴安娜·普林斯；{{Link-en|琳达·哈里逊|Linda Harrison (actress)}}饰演变身后的神奇女侠；{{Link-en|莫迪·普里克特|Maudie Prickett}}饰演戴安娜的母亲（但似乎并非[[希波吕忒_(DC漫画)|希波吕忒女王]]）。在这个电视剧的企划中，戴安娜·普林斯（并非神奇女侠）是剧中的焦点所在。在片中，戴安娜·普林斯是一个笨拙而朴实的年轻女孩，和她的母亲一起居住在离[[美国空军|美国空军]]基地不远的寓所里。在试播集的录像带内容中，绝大部分时间都是戴安娜的母亲指责戴安娜没有男朋友。而当她母亲离开房间后，戴安娜就换上了她的神奇女侠服装，并欣赏镜中的自己。不过戴安娜在镜中看到的并不是自己，而是由琳达·哈里逊饰演的性感的超级女英雄——神奇女侠。神奇女侠伴随着《{{Link-en|美丽娃娃|Oh, You Beautiful Doll}}》的曲子翩翩起舞。在录像的最后，戴安娜爬上窗户飞走了。这表明她虽然有一点[[人格分裂|人格分裂]]的[[妄想症|妄想症]]，但她的确具有[[超能力|超能力]]。<ref name="lesdaniels" /> 这个试播集从来没有公开播映过，同时这个电视剧企划也被取消了。
