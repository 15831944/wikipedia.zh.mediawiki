'''氯化鋇'''（[[化学式|化学式]]：[[钡|Ba]][[氯|Cl]]<sub>2</sub>）是[[钡|钡]]的[[氯化物|氯化物]]，有毒，灼烧時產生黃綠色的光。

== 製备 ==
氯化鋇可以由[[碳酸鋇|碳酸鋇]]（自然界中的[[毒重石|毒重石]]）或[[氫氧化鋇|氫氧化鋇]]和[[鹽酸|鹽酸]]的反應得到。工業上可以從[[硫酸鋇|硫酸鋇]]經過兩個步驟得到：

: [[硫酸鋇|BaSO<sub>4</sub>]] + 4 [[碳|C]] → [[硫化钡|BaS]] + 4 [[一氧化碳|CO]]（加熱）
: BaS + [[氯化鈣|CaCl<sub>2</sub>]] → BaCl<sub>2</sub> + [[硫化鈣|CaS]] （溶掉混合物）

== 化學性質 ==
氯化鋇溶於水解离为Ba<sup>2+</sup>和Cl<sup>-</sup>，解离出的离子可以产生沉淀反应：<ref name=chen>陈寿春. 重要无机化学反应（第三版）. 上海科学技术出版社, 1982. 第一章 阳离子. 四-2. 钡(Ba)</ref>
: BaCl<sub>2</sub> + Na<sub>2</sub>SO<sub>4</sub> → 2 NaCl + BaSO<sub>4</sub>↓
: BaCl<sub>2</sub> + K<sub>2</sub>CrO<sub>4</sub> → 2 KCl + BaCrO<sub>4</sub>↓
: BaCl<sub>2</sub> + Na<sub>2</sub>C<sub>2</sub>O<sub>4</sub> → 2 NaCl + BaC<sub>2</sub>O<sub>4</sub>↓
: BaCl<sub>2</sub> + 2 AgNO<sub>3</sub> → 2 AgCl↓ + Ba(NO<sub>3</sub>)<sub>2</sub>

在1000℃，熔化的氯化钡和[[钠|钠]]有如下平衡：<ref name=chen />
:2 Na + BaCl<sub>2</sub> ↔ 2 NaCl + Ba

== 用途 ==
在[[實驗室|實驗室]]，它可用來測試硫酸鹽。

在[[煙花|煙花]]中，{{citation needed|BaCl<sup>+</sup>用來製造鮮[[綠色|綠色]]光}}。

== 安全 ==
氯化钡与其它的水溶性钡盐一样，因其可在水溶液中[[离解|离解]]出钡离子，所以都是有高毒性的。可溶性钡盐中毒通常使用[[硫酸镁|硫酸镁]]作为解毒剂，它们会反应产生[[硫酸钡|硫酸钡]]，因其不溶性而相对于氯化钡则为无毒的。
