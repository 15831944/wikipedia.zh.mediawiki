'''氯酸银'''（AgClO<sub>3</sub>）是白色四方晶体。像所有氯酸盐一样，它可溶于水并在热水中溶解度增加<ref name=bj />，是强氧化剂。作为一种简单的金属盐，在碱性无机化学实验中，它是一种常见的化学品。对光敏感，必须存储在深色密闭容器中。

== 制备 ==
* 用[[氧化银|氧化银]]或[[碳酸银|碳酸银]]与[[氯酸|氯酸]]反应：
::<math>\mathsf{Ag_2O + 2HClO_3 \ \xrightarrow{}\ 2AgClO_3 + H_2O }</math>
::<math>\mathsf{Ag_2CO_3 + 2HClO_3 \ \xrightarrow{}\ 2AgClO_3 + CO_2\uparrow + H_2O }</math>

* 向[[氧化银|氧化银]]的悬浮液通入[[氯气|氯气]]：
::<math>\mathsf{Ag_2O + 6Cl_2 + 5H_2O \ \xrightarrow{}\ 2AgClO_3 + 10HCl }</math>

* 用[[氯酸钾|氯酸钾]]与[[硝酸银|硝酸银]]反应：
::<math>\mathsf{AgNO_3 + KClO_3 \ \xrightarrow{85^oC}\ AgClO_3 + KNO_3 }</math>

== 性质 ==
*加热分解：
::<math>\mathsf{2AgClO_3 \ \xrightarrow{270^oC}\ 2AgCl + 3O_2\uparrow }</math>

== 用途 ==
作为[[卤化|氯化]][[苯|苯环]]时的催化剂。<ref>[http://books.google.com/books?id=Ld-AGnffxXIC&pg=PA157&lpg=PA157&dq=%22silberchlorat%22&source=bl&ots=zYCBf-tcLV&sig=oh4ejpORxAu1J9TtYncdmbhh2SA&hl=de&ei=ua_wSpKVI46QmwPB3P3tBw&sa=X&oi=book_result&ct=result&resnum=6&ved=0CBcQ6AEwBTgU#v=onepage&q=%20&f=false Organische Chemie]</ref>
