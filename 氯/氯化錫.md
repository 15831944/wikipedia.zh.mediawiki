'''氯化錫'''，化學式[[錫|Sn]][[氯|Cl]]{{sub|4}}，為無色發煙性液體。

== 製備 ==

可於[[氯化亞錫|氯化亞錫]]溶液中，通入過量的[[氯气|氯气]]而製得。反應式：

: <math>\ SnCl_2+Cl_2\to SnCl_4</math>

== 用途 ==

氯化錫的蒸氣與[[氨|氨]]及[[水|水氣]]混合，生成[[氫氧化錫|氫氧化錫]]及[[氯化銨|氯化銨]]之微粒而呈濃煙狀，[[軍事|軍事]]上用以製作[[煙幕彈|煙幕彈]]。反應式：

: <math>\ SnCl_4+4NH_3+4H_2O\to Sn(OH)_4+4NH_4Cl</math>
