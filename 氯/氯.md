'''氯'''是一种[[卤素|卤族]][[化学元素|化学元素]]，[[化学符号|化学符号]]為'''Cl'''，[[原子序数|原子序数]]為17。

== 自然分布 ==
自然界中游离状态的氯存在於[[大氣層|大氣層]]中，是破壞[[臭氧層|臭氧層]]的单质之一。氯氣受紫外線分解成兩個氯原子（自由基）。大多數通常以[[氯化物|氯化物]](Cl<sup>-</sup>)的形式存在，常見的主要是[[氯化鈉|氯化鈉]]（[[食盐|食盐]]，NaCl）。

== 单质：Cl<sub>2</sub>==
氯[[單質|單質]]由两个氯原子构成，化學式為Cl<sub>2</sub>。氣態氯單質俗稱「氯氣」，液態氯單質俗稱「液氯」。
=== 發現 ===
[[瑞典|瑞典]]化学家[[卡尔·威廉·舍勒|卡尔·威廉·舍勒]]（{{lang|sv|Carl Wilhelm Scheele}}）在1774年使用[[盐酸|盐酸]]与[[软锰矿|软锰矿]]通過下述反应制得[[氯氣|氯氣]]：
: <Math>\rm MnO_2 + 4\ HCl \stackrel{\Delta}{\longrightarrow} MnCl_2 + 2\ H_2O + Cl_2 \uparrow </Math>

继舍勒之后，贝托雷对氯气进行了更加深入的研究。发现将氯气通入水中会有盐酸生成同时还伴随着有能使带火星木条复燃的气体（氧气）放出；盐酸通过金属会放出氢气。

所以他认为氯气中含有氧，但尝试用当时已知的还原剂像金属、木炭、磷等还原剂来还原氯却没能得到相应的氧化产物，这又强有力地说明氯气中不含氧元素。后来戴维用白热的木炭仍不能使氯气分解，而且从盐酸和金属的反应中也不能得到氧化物，所以上面提到的氧气应该是由水提供了氧元素，并且他认为应该将这种绿色的气体视为一种新的元素，氯元素就这样被发现了。

=== 名稱由來 ===
[[英文|英文]]名稱chlorine來自於[[希臘文|希臘文]]{{lang|el|khlôros}}（{{lang|el|χλωρóς'}}，淡[[綠|綠]]色）。19世纪70年代，中国化学家[[徐寿|徐寿]]将其译为绿气。1933年，化学家[[郑贞文|郑贞文]]在《化学命名原则》中把这个字改为了“氯”。<ref>{{cite journal|author=刘怀乐|title=化学鉴源与略考|journal=化学教育|year=1994|issue=04|url=http://www.hxzxs.cn/html/4295.html|accessdate=2013-02-25|deadurl=yes|archiveurl=https://web.archive.org/web/20131004213854/http://www.hxzxs.cn/html/4295.html|archivedate=2013-10-04}}</ref>在[[日文|日文]]與[[韓文|韓文]]中，因為氯是[[鹽|鹽]]的主要成分之一而稱為「鹽素」（[[日本漢字|日本漢字]]現在寫作「{{lang|ja|塩素}}」）。

=== 物理性質 ===
在常温下，氯氣是一种黃綠色、刺激性氣味、有毒的[[气体|气体]]。[[压力|压力]]为1.01×10<sup>5</sup>[[帕斯卡|Pa]]时，氯單質的沸點為−34.4℃，熔点为−101.5℃。氯氣可溶於水和碱性溶液，易溶於[[二硫化碳|二硫化碳]]和[[四氯化碳|四氯化碳]]等有機溶劑，难溶于饱和氯化钠溶液，飽和時1體積水溶解2體積氯氣。

=== 毒性 ===
氯气具有强烈的刺激性、窒息气味，可以刺激人体[[呼吸道|呼吸道]]黏膜，輕則引起[[胸部|胸部]]灼熱、疼痛和咳嗽，严重者可导致死亡。<ref>{{citeweb|url= http://www.pcc.vghtpe.gov.tw/old/docms/30201.htm|title= 氯氣（Chlorine Gas）之簡介|language= zh-tw|deadurl= yes|archiveurl= https://web.archive.org/web/20120110125148/http://www.pcc.vghtpe.gov.tw/old/docms/30201.htm|archivedate= 2012-01-10}}</ref>

=== 化学性质 ===
氯气的[[化学性质|化学性质]]很活泼，它是一种活泼的非[[金属|金属]]單質。

氯[[原子|原子]]的最外[[电子层|电子层]]有7个[[电子|电子]]，在[[化学反应|化学反应]]中容易结合一个[[电子|电子]]，使最外[[电子层|电子层]]达到8个[[电子|电子]]的稳定状态，因此'''氯气'''具有强[[氧化性|氧化性]]。氯氣的強氧化性表現為以下幾個方面：

==== 漂白性 ====
湿润的氯气可用于[[纸浆|纸浆]]和棉布的[[漂白|漂白]]，不同於[[二氧化硫|SO<sub>2</sub>]]的漂白性，氯氣的漂白性為不可還原且較為強烈，因此不宜以此作為絲綢之漂白劑。

之所以强调湿润的是因为
:<Math>\rm Cl_2 + H_2O \rightleftharpoons HCl + HClO</Math>
:<Math>\rm HClO \rightleftharpoons HCl + [O]</Math>（生成的<Math>\rm [O]</Math>是游离氧，正是这个游离氧，氧化了有机染料使之褪色）次氯酸的分解反应在光照或受热时速度加快。

因此干燥的氯气并不具有这个性质。

==== 與金屬反應 ====
氯氣可與[[钠|钠]]等活泼[[金属|金属]]直接化合。也能跟[[铜|铜]]等不活泼[[金属|金属]]起反应。
:<Math>\rm 2\ Na + Cl_2 \rightarrow 2\ NaCl</Math>（生成白烟）
:<Math>\rm Cu + Cl_2 \stackrel{\Delta}{\longrightarrow} CuCl_2</Math>（生成棕黄色的烟）
当氯气和[[铁系元素|铁系元素]]反应时，只有铁能被氧化至+3价，其余为+2价：
:<Math>\rm 2\ Fe + 3\ Cl_2 \stackrel{\Delta}{\longrightarrow} 2\ FeCl_3</Math>（生成棕褐色的烟）
:<Math>\rm Co + Cl_2 \stackrel{\Delta}{\longrightarrow} CoCl_2</Math>

:<Math>\rm Ni + Cl_2 \stackrel{\Delta}{\longrightarrow} NiCl_2</Math>

==== 與非金屬反應 ====
[[氢气|氢气]]可在氯气中点燃并产生[[氯化氢|氯化氢]]。反应的化学方程式：
:<Math>\rm Cl_2 + H_2 \rightarrow 2\ HCl</Math>

值得一提的是反应条件对上述反应的现象有很大影响：
# 如果氢气与氯气事先充分混合，在光照条件下发生爆炸；
# 如果氢气在氯气中安静地燃烧，现象为蒼白色火焰，同時伴有白霧（氯化氢溶解于空气中的水形成的盐酸小液滴）生成。

== 用途 ==
氯可以作为一种較不昂貴的[[消毒劑|消毒劑]]，一般的[[自来水|自来水]]及[[游泳池|游泳池]]就常采用它来[[消毒|消毒]]。但由於氯氣的水溶性較差、毒性較大、會放出特殊氣味，且容易产生有[[致癌|致癌]]風險的[[三鹵甲烷|三鹵甲烷]]等[[有机氯化合物|有机氯化合物]]，故中國、美國等國常改用[[二氧化氯|二氧化氯]]（ClO<sub>2</sub>）、[[氯胺|氯胺]]或[[臭氧|臭氧]]等代替氯氣作為水的消毒劑。

除了用於消毒，氯氣是一种重要的化工原料，用於制造[[盐酸|盐酸]]和[[漂白粉|漂白粉]]、制造[[氯代烃|氯代烃]]。也可以用于制造多种[[农药|农药]]、制造[[氯仿|氯仿]]等[[有机溶剂|有机溶剂]]。

氯氣還广泛用于造纸、纺织、有机合成、金属冶炼、化工原料等行业，也有作為[[化學武器|化學武器]]的紀錄。

== 氯离子的检验 ==
检验水中是否含有氯离子可以向其中加入[[硝酸|硝酸]]酸化的银离子（如[[硝酸银|硝酸银]]）（加入酸性硝酸银(即硝酸银和酸的混合物)可以排除其他离子(如碳酸根、亚硫酸根)干扰），银离子和氯离子反应会生成[[氯化银|氯化银]]白色[[沉淀|沉淀]]，反应式：
:<Math>\rm Cl^- + Ag^+ \rightarrow AgCl\downarrow</Math>
