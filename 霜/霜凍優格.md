'''霜凍優格'''是用[[酸奶|酸奶]]，有時候用其他乳製品製成的冷凍[[甜點|甜點]]。它通常比[[冰淇淋|冰淇淋]]酸，脂肪也比較少 （因為使用的是牛奶而不是[[奶油|奶油]]）。它跟牛奶凍 （近幾年被稱之為低脂肪或輕冰淇淋）和常見的[[霜淇淋|霜淇淋]]不一樣。跟優格不同的是，霜凍優格不受[[美国食品药品监督管理局|美國食品和藥物管理局]] （FDA） 監管<ref>http://www.gpo.gov/fdsys/pkg/FR-2009-01-15/pdf/E9-736.pdf</ref><ref name="aboutyogurt.com">{{cite web|url=http://aboutyogurt.com/index.asp?bid=28|title=Live and Active Culture （LAC） Yogurt Facts|accessdate=2015-03-22|date=|publisher=Aboutyogurt.com}}</ref>，而是受到美國一些州的監控<ref>http://www.pacode.com/secure/data/007/chapter39/007_0039.pdf</ref><ref>{{cite web|url=http://law.justia.com/codes/california/2005/fac/36991-36994.html|title=Article 7. Frozen Yogurt - Sections 36991-36994 :: California Food and Agricultural Code :: 2005 California Code :: California Code :: US Codes and Statutes :: US Law :: Justia|accessdate=2015-03-22|date=|publisher=Law.justia.com}}</ref>。霜凍優格可能含有活性的酵母菌。<ref name="aboutyogurt.com" />

== 歷史 ==
人們吃純優格（酸奶）超過四千年以上，尤其是在中東和[[印度|印度]]。優格（酸奶）在二十世紀初，被引進美國，且在後來幾十年，作為營養食品人氣穩步上升。在30年代，[[Dannon|Dannon]]首次在美國販賣事先包裝好的優格（酸奶）。到了70年代，隨著冰淇淋的普及，冷凍和生產技術轉移到生產冷凍優格（酸奶）。然而，許多消費者抱怨酸奶味道。<ref name="madehow2">{{cite web|url=http://www.madehow.com/Volume-2/Frozen-Yogurt.html|title=How frozen yogurt is made - making, history, used, processing, composition, product, industry, Raw Materials, The Manufacturing Process of frozen yogurt, Quality Control, The Future|accessdate=2015-03-22|date=1994-11-22|publisher=Madehow.com}}</ref>

善用消費者對味道像冰淇淋但又更健康的甜品需求，[[TCBY|TCBY]]在1981年開創了第一家店。與早期推出已事先包裝好的不同，TCBY的優格（酸奶）需在現場通過機器售賣。TCBY當時，成為世界上最大的冷凍優格（酸奶）加盟店。其他人看到TCBY的成功時，想到冷凍優格（酸奶）在80年代成為熱門，而在1986年銷售額達2500萬美元。在美國，科倫坡，南奇和卡倫小姐等品牌當時都非常有名。霜凍優格在90年代中期佔有冷凍甜品市場10％份額，銷售額超過3億美元<ref name="madehow3">{{cite web|url=http://www.madehow.com/Volume-2/Frozen-Yogurt.html|title=How frozen yogurt is made - making, history, used, processing, composition, product, industry, Raw Materials, The Manufacturing Process of frozen yogurt, Quality Control, The Future|accessdate=2015-03-22|date=1994-11-22|publisher=Madehow.com}}</ref>。由於美國人將注意力轉向高蛋白，高脂肪飲食，在90年代末期，冷凍優格（酸奶）的需求大量減少。[[低脂|低脂肪]]的食物，如冷凍優格（酸奶），在千禧年間，由於食品趨勢更傾向高脂肪和低售價的冰淇淋，因此失去了大家的喜愛。

在千禧年中期，隨著John Wudel發明[[益生菌|益生菌]]粉混合產品的出現，作為冷凍甜點行業替代甜味劑的先驅，食品趨勢逐漸變回冷凍優格（酸奶）。乾性混合物使冷凍優格（酸奶）首次進口到美國以外的許多國家。2005年，一家在[[加利福尼亚州|加利福尼亞州]]，店名為[[Pinkberry|Pinkberry]]的小零售店，向優格市場推出了一種新的冷凍優格（酸奶）。2006年，Yogurtland在加利福尼亞州富勒頓，開創了第一家[[自助餐|自助]]概念的冷凍優格（酸奶），為客人創造可以自己混合的機會。這種新歐洲酸的口味與美國西部許多自助酸奶店，導致世界各地對冷凍優格（酸奶）的興趣和需求的復甦。自助店的先驅Yogurtland，目前擁有超過320多家加盟店，有無奶，無添加糖和[[无麸质饮食|無麩質]]冷凍優格（酸奶）的選擇。他們的公司還擁有自己的乳製品，使其能夠控制整個過程，以確保高品質的產品<ref>Yogurtland. [https://www.yogurt-land.com/ourstory], ''[[Yogurtland|Yogurtland]]''. Retrieved on 15 December 2016.</ref>。到2013年，美國和許多其他國家的消費者，對冷凍優格（酸奶）的需求達到了前所未有的程度，與70年代對冷凍酸奶的初次接觸，有了明顯的對比<ref name="IFYA2">{{cite web|url=http://internationalfrozenyogurt.com/resources/industry-faqs/|title=The International Frozen Yogurt Association Frozen Yogurt FAQs - Learn About Frozen Yogurt | IFYA|accessdate=2015-03-22|author=IFYA Administrator|date=2013-10-17|publisher=Internationalfrozenyogurt.com}}</ref>。

== 生產 ==
霜凍優格通常包含乳固體，某種甜味劑，[[乳脂肪|乳脂肪]]，優格菌種（[[保加利亞乳桿菌|保加利亞乳桿菌]]和[[嗜熱鏈球菌|嗜熱鏈球菌]]是常見培養物<ref name="madehow4" />），天然或人造調味料，有時是天然或人造色素。

乳脂肪約占優格的0.55-6％。添加量與乳固體的量成反比，而乳脂肪會使優格更加濃稠。乳固體佔優格體積的8-14％，提供[[乳糖|乳糖]]的甜度和蛋白質的光滑性，並增加抗熔性<ref name="madehow4" />。甘蔗或甜菜糖占了優格15-17％的成分。除了加入甜味之外，糖會增加優格中固體成分的體積，提升優格的口感。動物凝膠或植物添加劑（瓜爾豆膠，[[鹿角菜膠|鹿角菜膠]]等）會穩定優格，減少結晶並提高優格融化的溫度。這種穩定性確保了霜凍優格在處理或溫度變化的情況下，都會保持光滑的稠度<ref name="madehow4" />。

大公司經常使用專業的流水線生產霜凍優格。乳製品和穩定劑結合並均質化。在32℃時，加入乳酸菌培養。混合物保持在此溫度，直到它固化並準備冷卻。之後，將混合物在0至4℃的溫度下冷卻。一旦達到理想的溫度和粘度，就可以將優格置於熟化罐中長達四個小時。然後把甜味劑，調味料和色素混合，並將優格混合物在-6至-2℃的溫度下冷卻。為了產生更多的容量和光滑的稠度，當混合物攪拌時，會將空氣併入酸奶中。當產品中已經摻入足夠量的空氣，優格就會被迅速冷凍，以防止形成大塊冰晶，並儲存在一個寒冷的地方待運<ref name="madehow4" />。

霜凍優格可以在[[霜淇淋|霜淇淋]]機中製作，與霜淇淋製作方式大致相同。霜凍優格以粉末形式出售，需要與水或液體混合來準備倒入霜淇淋機器中。可以選擇具有高脂或低脂含量的混合物，而加入軟霜凍優格的空氣量是可變的。脂肪含量越高，優格能吸收的空氣越多。當混合物凝固時，隨著更多空氣的加入，產品的味道會越濃。

== 應用 ==
霜凍優格已經大受歡迎，像冰淇淋一樣，並且有各種各樣的口味和風格。許多公司允許顧客選擇添加各種配料，從草莓、香蕉到迷你m&ms和奧利奧，或者要求裝在紙杯或甜筒中的霜凍優格。也有些賣家提供無糖種類。一些連鎖店生產的霜凍優格更接近原始配方，而其他公司則致力於讓霜凍優格嚐起來像冰淇淋<ref>{{cite news|url=https://www.nytimes.com/2007/02/21/dining/21pink.html|title=Heated Competition. Steaming Neighbors. This Is Frozen Yogurt?|last=Steinhauer|first=Jennifer|date=2007-02-21|publisher=New York Times|accessdate=2007-02-21}}</ref>。
