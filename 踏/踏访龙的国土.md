《'''踏访龙的国土'''》（{{lang-th|ย่ำแดนมังกร}}）是由[[诗琳通公主|诗琳通公主]]写作的一部[[游记|游记]]，讲述的是1981年5月12日至5月20日，诗琳通公主首次访华<ref>{{cite web |url=http://news.163.com/07/0404/18/3B8N8L37000120GU.html |title=泰国公主：龙的国土吸引我 |last= |first= |date=2007-04-04 |website=网易 |publisher= |access-date=2018-12-03 |quote=我首次踏上龙的国土，就被这块神奇的土地深深吸引。我每次到一个新的地方，都要拍摄照片，因为它们是了解中国社会的窗口。}}</ref>；在中国的[[北京市|北京市]]，[[陕西省|陕西省]][[西安市|西安市]]，[[四川省|四川省]][[成都市|成都市]]、[[都江堰市|都江堰市]]，[[云南省|云南省]][[昆明市|昆明市]]以及[[香港|香港]]等地游览了9天<ref>{{cite web |url=http://bclc-tu.com/Dragonland.html |title=ย่ำแดนมังกร |last= |first= |date= |website=bclc-tu |publisher= |access-date=2018-11-07 |quote=}}</ref>。中文版标题由诗琳通公主御笔题写。

2011年8月17日，泰国[[朱拉隆功大学|朱拉隆功大学]]举办“踏访龙的国土”30周年演讲会<ref>{{cite web |url=http://th.china-embassy.org/chn/xwdt/t849809.htm |title=泰国朱拉大学举办诗琳通公主访华三十周年演讲会 |last= |first= |date=2011-08-19 |website=中华人民共和国驻泰王国大使馆 |publisher= |access-date= |quote=}}</ref><ref>{{cite web |url=http://www.cim.chinesecio.com/hbcms/f/article/info?id=eccb8810d04c42bab7d242314c003925 |title=“踏访龙的国土三十周年”泰国诗琳通公主访华30周年专题演讲（二） |last= |first= |date=2016-06-30 |website=孔子学院 |publisher= |access-date= |quote=}}</ref>。

== 轶闻 ==
早在1961年，中国画家[[孙多慈|孙多慈]]画了一幅名为《[[泰国公主|泰国公主]]》的油画作品，与20年后诗琳通公主首次访华形成巧合。
