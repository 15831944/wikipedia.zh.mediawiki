'''厲鼎毅'''（{{bd|1931年|7月7日|2012年|12月27日|catIdx=L}}），出生于[[中華民國|中華民國]][[南京市|南京市]]，毕业于南非金山大学和[[西北大学_(伊利诺州)|西北大学]]，擁有中華民國與美國雙國籍，美籍华人，为著名物理学家、[[美国国家工程学院|美国国家工程学院]]院士、台灣[[中央研究院|中央研究院]]院士和[[中国工程院外籍院士列表|中国工程院外籍院士]]，被誉为“光纤通信波分复用之父”。<ref>{{cite web |url=http://news.163.com/11/0703/07/7817M10O00014AED.html |title=150余名海内外专家来蓉共庆厉鼎毅80寿辰 |accessdate=2011-07-03  |publisher= 网易 |work= |language=zh-cn}}</ref><ref name=" bear">{{cite web|url=http://cjmp.cnhan.com/cjrb/html/2010-09/12/content_4028200.htm|title=岳父吴国祯在汉求婚轶事|accessdate=2010年9月12日|publisher=长江日报|work=|language=zh-cn}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>

== 生平 ==
1931年，厲鼎毅出生在[[中華民國|中華民國]][[南京市|南京市]]，他父亲[[厲斯昭|厲斯昭]]是[[中華民國|中華民國]]的外交官，母亲[[谢纬鹏|谢纬鹏]]是一名[[女权运动|女权运动]]人士，曾在[[宋美龄|宋美龄]]手下工作 (中央委員會婦女工作指導委員會副主任)。厲鼎毅為家中長子，其弟名厲鼎凱（1934-2018年11月18日）。<ref name=" tiger">{{cite web |url=http://cn.nytimes.com/article/science-technology/2013/01/09/c09li-obit/ |title=美籍华裔科学家厉鼎毅逝世 |accessdate=2013年1月9日 |publisher=纽约时报中文网 |work= |language=zh-cn |archive-url=https://web.archive.org/web/20130113020223/http://cn.nytimes.com/article/science-technology/2013/01/09/c09li-obit/ |archive-date=2013年1月13日 |dead-url=yes }}</ref><ref name=" bear" />12岁时，厉鼎毅父亲厉斯昭出任中華民國驻加拿大温哥华总领事，厉鼎毅跟着父亲离开中国去了[[加拿大|加拿大]]，并且在加拿大读高中；<ref name=" bear" />后来，父亲出任中華民國驻南非约翰内斯堡总领事，他又跟着父亲去了[[南非|南非]]，在南非金山大学获得电气工程学士学位。<ref name=" tiger" /><ref name=" bear" />大學畢業後，厉鼎毅至美國求學，1955年及1958年在[[西北大学_(伊利诺州)|西北大学]]获得碩士及博士学位。<ref name=" tiger" /><ref name=" bear" />其父則調任中華民國駐哥倫比亞大使。

1957年，厉鼎毅至美國AT&T公司的[[贝尔实验室|贝尔实验室]]工作，在此工作了41年，直到1998年退休。<ref name=" tiger" />工作期间，他写了许多关于“天线、微波传播、激光和光学通信”的论文、书籍等。<ref name=" tiger" />

1961年，厉鼎毅和同事A·加德纳·福克斯一起开发了“计算机模拟程序”，从而给激光奠定了基础，后来广泛应用于医疗和航空领域。1960年代后期，厉鼎毅开始研究“光波”，后来被应用于电信行业。1992年，他和他的团队成功实验了光波技术。

2012年12月27日，厉鼎毅在美国[[犹他州|犹他州]]雪鸟滑雪度假村因为[[心脏病|心脏病]]逝世。<ref name=" tiger" />

== 个人生活 ==
厉鼎毅的妻子为吴修惠，毕业于西北大学，两人在大学相识后结婚，育有女儿厉琼、厉玳。<ref name=" tiger" />

吴修惠之父[[吳國楨|吳國楨]]，曾出任[[上海市|上海市]]市长与[[台湾省|台湾省]]主席。<ref name=" bear" />

== 荣誉 ==
*1980年，美国国家工程学院院士。
*1981年，西北大学校友绩效奖。
*1983年，美籍华人专业学会成就奖。
*1994年，台灣中央研究院院士。
*1995年，约翰·丁达尔奖。
*1996年，中国工程院外籍院士。
*2004年，电气电子工程师学会光子学奖。
*2009年，电气电子工程师学会爱迪生奖章。
