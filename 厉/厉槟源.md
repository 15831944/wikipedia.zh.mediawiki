'''厉槟源'''（{{bd|1985年}}）出生于中国[[湖南省|湖南省]][[永州市|永州市]]，2001年[[本科|本科]]毕业于[[中央美术学院|中央美术学院]]雕塑系并获得[[学士学位|学士学位]]，现于[[北京市|北京市]]工作，是一名[[行为艺术|行为艺术]]家<ref>{{cite web |title=厉槟源作品欣赏 |url=http://fashion.163.com/14/0612/20/9UIKNM5S00264MK3.html |website=网易艺术 |date=2014-06-12}}</ref>。

== 轶事 ==
=== 裸奔 ===
2013年3月20日前后，居住在[[望京|望京]]城区的厉槟源因为在艺术创作的过程中遇到困难，心情压抑，故开始通过在深夜[[裸奔|裸奔]]的方式来发泄自己的压力。他使用颜料把全身涂成蓝色后进行裸奔，有时扛着木制[[十字架|十字架]]，有时抱着[[充气娃娃|充气娃娃]]，有时开着电动摩托车，在两个月间裸奔达十次。有行人看到此行为，便拍照或录影发布到社交网络上，引起网友的热议。后来他在[[新浪微博|新浪微博]]上公开身份，承认自己便是裸奔的那个人。他表示，但是天气寒冷，气温在零[[摄氏度|摄氏度]]以下，裸奔的时候很有快感，非常刺激。对于厉槟源的裸奔行为，有网友解读说，独自裸奔是为自我而跑，抱充气娃娃裸奔是为情欲而跑，扛十字架裸奔是为信仰而跑<ref>{{cite web |title=厉槟源：用裸奔“治疗”自己 |url=http://lady.163.com/14/0116/11/9IN6F1E600264N2N.html |website=网易女人 |date=2014-01-16}}</ref>。对于自己的行为，厉槟源认为虽然{{tsl|en|Public nudity|公共裸露}}违反法律和传统道德，但因为是在深夜进行，并没有影响到其他人。北京市锋锐律师事务所的刘晓原律师也持同样的观点<ref>
{{cite web |title=北京望京神秘裸奔哥现身：美院毕业 为宣泄困惑 |url=http://news.sohu.com/20130518/n376323488.shtml |website=搜狐新闻 |date=2013-05-18 |access-date=2019-03-02 |archive-url=https://web.archive.org/web/20130609153256/http://news.sohu.com/20130518/n376323488.shtml |archive-date=2013-06-09 |dead-url=no }}
</ref>。

=== 毕业设计 ===
厉槟源的本科毕业设计是一瓶瓶口套着一个橘黄色气球的2.5[[升|升]]瓶装[[可口可乐|可口可乐]]，并用橡皮筋使气球口和可乐瓶口贴合，右侧倒放着可乐瓶盖。可乐因为被摇晃而分解出[[二氧化碳|二氧化碳]]气体，使得气球胀大。他将这个花费7元[[人民币|人民币]]的毕业设计命名为「越晃越大」，这个作品的灵感来源于他打开可乐瓶盖时听到「呲」的一声。《越晃越大》因原料普通，制作简单，而指导老师还有四位等原因引发了网友的热议。有网友称赞此设计简洁有创意，不过因为厉槟源本人并没有解释这个作品想要表达的内在含义，故不同网友对其含义进行了不同方向的猜测，也有人对四位指导老师的分工进行了猜测<ref>
{{cite web |title=“裸奔哥”毕业作品引热议：可乐+气球=越晃越大 |url=http://www.chinanews.com/cul/2013/05-27/4860420.shtml |website=中国新闻网 |publisher=现代快报 |date=2013-05-27 |access-date=2019-03-02 |archive-url=https://web.archive.org/web/20130528152813/http://www.chinanews.com/cul/2013/05-27/4860420.shtml |archive-date=2013-05-28 |dead-url=no }}
</ref>。对于不同层面的解读，厉槟源表示答案并不是唯一的，不同方向不同层面的解读都是可以接受的<ref>{{cite web |author1=徐婉娟 |title=行为艺术家厉槟源：我的创作方式就是不择手段（图） |url=http://news.99ys.com/news/2014/0422/45_174754_5.shtml |website=99艺术新闻 |publisher=雅昌艺术网 |date=2014-04-22}}</ref>。
