'''溥傑'''（{{lang-mnc|ᡦᡠ<br />ᡤᡳᠶᡝ|v=Pu Giye|a=Pu Giye}}；{{bd|1907年|4月16日|1994年|2月28日|catIdx=A爱新觉罗溥杰}}），乳名「'''譽格'''」，[[表字|字]]「'''俊之'''」，[[號|號]]「'''秉藩'''」{{NoteTag|[[溥儀|溥儀]]取[[號|號]]，意思是「秉承[[曾國藩|曾國藩]]之志」。}}，[[清朝|清朝]]末代[[皇帝|皇帝]][[溥儀|溥儀]]的胞弟。

== 家世 ==
溥傑的祖父為[[道光帝|道光帝]]七子、[[咸豐帝|咸豐帝]]之弟醇賢親王[[奕譞|奕譞]]，伯父為[[光緒帝|光緒帝]]，父親[[載灃|載灃]]繼承醇親王爵位，後因輔政為攝政王。母親[[幼兰|幼兰]]是正白旗[[瓜尔佳氏|瓜尔佳氏]][[榮祿|榮祿]]之女。

溥傑有兄弟三人，姐妹七人。依次為[[溥儀|溥儀]]（1906年生）、[[韞瑛|韞瑛]]（1909年生，嫁[[潤良|潤良]]）、[[韞龢|韞龢]]（1911年生，嫁[[鄭廣元|鄭廣元]]）、[[韞穎|韞穎]]（1913年生，嫁[[潤麒|潤麒]]）、[[韞嫻|韞嫻]]（1914年生，嫁[[趙琪璠|趙琪璠]]）、溥倛（1915年生，早夭）、[[韞馨|韞馨]]（1917年生，嫁[[萬嘉熙|萬嘉熙]]）、[[溥任|溥任]]（1918年生）、[[韞娛|韞娛]]（1919年生，嫁[[王愛蘭|王愛蘭]]）、[[金志堅_(皇妹)|韞歡]]（1921年生，嫁[[喬宏治|喬宏治]]）。

== 生平 ==

溥傑幼年在醇王府作溥仪伴读。1924年，与满族女子唐怡莹成婚。1928年，北伐军进入北京前夕，载沣全家由[[张学良|张学良]]安排，赴天津租界避难，溥杰夫妇住在张学良小妾谷瑞玉公馆中。溥杰秘密前往奉天入讲武堂求学，为溥仪阻止，通过日本警察扣留于大连，十余天后回到天津。

欲图复辟的溥仪另外安排溥杰到日本学习军事，并为其起名金秉藩，於1929年3月，与[[润麒|润麒]]一同到[[日本|日本]][[东京|东京]][[学习院|学习院]]、[[日本陆军士官学校|日本陆军士官学校]]学习。1935年，溥杰回到[[满洲国|满洲国]]。在满洲国任军职，任[[滿洲國宮內府|滿洲國宮內府]][[侍从武官|侍从武官]]。

1937年4月，溥杰与日本华族女子[[嵯峨浩|嵯峨浩]]成婚。数日后，哥哥溥仪新娶[[譚玉齡|譚玉齡]]，与此前所娶[[婉容|婉容]]、[[文绣|文绣]]一样皆无生育子女。在溥仪长期无子的状态下，溥杰与嵯峨浩“未来的儿子”被认为是溥仪的接替者。同年，[[满洲国国务院|满洲国国务院]]通过《帝位继承法》，规定：皇帝死后由子继之，如无子则由孙继之，如无子无孙则由弟继之，如无弟则由弟之子继之。从法律上，为溥杰继承帝位铺平道路。另有说法则认为，日方并未承认溥杰“[[皇族|帝族]]”身份，他无权继承满洲国帝位。根据1937年溥仪与[[关东军司令部|日本关东军司令]]签署的备忘录规定，满洲国皇位继承人由[[日本天皇|日本天皇]]选定，溥仪无权干涉<ref>{{Cite web |url=http://history.people.com.cn/GB/205396/17747254.html |title=溥仪签署的密约：把“满洲国”传位给日本人？(3) |accessdate=2017-11-29 |author=李诚、罗娱 |date=2012-04-25 |work= |publisher=人民网，来源：《今晚报》|language=Zh-hans }}</ref>。

1943年，溥杰入[[日本陸軍大學|日本陸軍大學]]學習，畢業後回[[满洲国|满洲国]]。1945年8月[[二次世界大戰|二次世界大戰]]結束，日本投降，滿洲國滅亡。溥傑與哥哥溥儀逃至[[瀋陽|瀋陽]]打算改乘飛機前往日本，在瀋陽機場被[[蘇聯|蘇聯]]軍隊俘獲，在蘇聯被拘留了5年。1950年8月被移交给[[中華人民共和國政府|中華人民共和國政府]]羁押，關押在[[撫順戰犯管理所|撫順戰犯管理所]]，接受[[中國共產黨|中國共產黨]]的思想改造，到1960年11月获特赦释放（11月19日签署特赦令，11月28日执行）。之後與妻子浩重逢，並度過[[文化大革命|文化大革命]]，但在文革期間曾被紅衛兵闖入家中而讓他和妻子浩受到不小驚嚇，妻子浩從日本帶到中國的照相機也被紅衛兵給偷走。

1980年代後任[[全國人民代表大會常務委員會|全國人民代表大會常務委員會]]委員、[[全國人大|全國人大]]民族委員會副主任委員。1991年11月28日獲日本立命館大學頒發名譽法學博士學位。

1994年2月28日7时55分因病于北京逝世，享年87歲。死后骨灰一半葬于日本山口县下关市{{tsl|ja|中山神社 (下関市)|中山神社}}（嵯峨家的神社）的爱新觉罗分社内，另一半葬于北京。

=== 妻子、女儿 ===
[[File:Aisin-Gioro_Pǔjié_and_Lady_Hiro_Saga.jpg|250px]]

溥傑的第一任妻子[[唐怡莹|唐怡莹]]（唐石霞，1904年－1993年），姓[[他他拉氏|他他拉氏]]，字怡莹，系[[珍妃|珍妃]]和[[瑾妃|瑾妃]]的哥哥志錡之女。1924年1月12日溥傑17岁时与唐怡莹结婚，婚后夫妇关系不睦。1926年唐怡莹成为[[张学良|张学良]]的情妇，并曾推动溥杰前往[[奉天讲武堂|奉天讲武堂]]求学，此后与溥杰家人的关系破裂。溥杰在日本读书期间，唐怡莹又成为[[卢永祥|卢永祥]]之子[[卢筱嘉|卢筱嘉]]的情妇，并将[[北平|北平]][[醇亲王|醇亲王]]府的大批财物盗走。随后两人分居，陷入长期离婚争议中。离婚后成为畫家。1949年迁居香港，任教於香港大学东方语言学校。外甥即伊爾根覺羅耆齡之孫惠伊深，其母唐梅是唐石霞姐姐。

1935年溥傑从日本回国后，溥仪曾为其物色了一满族女子王敏彤，但最终未能成婚。

溥傑的第二任妻子[[愛新覺羅浩|浩]]（1914年－1987年），是日军准备为其物色的日本[[貴族|貴族]]妻子，因为溥仪无后，日方希望由一个有日本血统的人即位。嵯峨浩为日本[[华族|华族]]侯爵{{tsl|ja|嵯峨實勝}}的长女，1937年4月3日两人在东京结婚（婚后名'''爱新觉罗·浩'''）。虽被认为是[[權宜婚姻|權宜婚姻]]，但两人婚后幸福圓滿。

兩人誕下兩名女兒，長女[[慧生|慧生]]（1938-1957）、次女[[嫮生|嫮生]]（1940-，其夫福永健治，婚后名福永嫮生）。

1957年12月10日，長女慧生在日本[[伊豆半島|伊豆半島]][[天城山|天城山]]遭槍擊身亡，現場尚有另一名男大學生陳屍，因此有殉情與遭男方情殺兩種說法，為當時喧騰一時的刑事案件([http://ja.wikipedia.org/wiki/%E5%A4%A9%E5%9F%8E%E5%B1%B1%E5%BF%83%E4%B8%AD 天城山心中])。

次女嫮生現居於日本，並育有五名子女。

== 著作 ==
溥杰擅书法，传世有《溥杰诗词选》。中国文史出版社1994年4月出版有叶祖孚执笔的《溥杰自传》。

== 相关影视作品 ==
2003年，日本[[朝日電視台|朝日電視台]]為紀念開播45週年，拍攝特製的[[電視連續劇|電視連續劇]]《[[流轉的王妃·最後的皇弟|-{zh-hans:流转的王妃·最后的皇弟;zh-hk:溥傑與王妃;zh-tw:流轉的王妃·最後的皇弟;}-]]》，把戰事當作背景，描繪溥傑與夫人之間的感情。溥傑、浩分別由[[竹野內豐|竹野內豐]]及[[常盤貴子|常盤貴子]]飾演。
