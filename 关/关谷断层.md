'''关谷断层'''（{{lang-ja|関谷断層}}）是一条位于[[日本|日本]][[栃木县|栃木县]]北部的[[断层|断层]]，也被称为'''关谷构造线'''<ref name=Tadahiro1 />。该断层是栃木县境内最长的活断层，全长约38千米<ref name=shioya /><ref name=honbu1 />。

== 概况 ==
关谷断层位于[[日本|日本]][[栃木县|栃木县]]北部的{{link-ja|那须岳|那須岳}}西麓，南起[[盐谷町|盐谷町]]，北至[[那须町|那须町]]，全长约38千米，[[倾角|倾角]]约为15°至40°<ref name=honbu1 /><ref name=jmaut />。其西侧相对于东侧更偏向于拥有[[逆断层|逆断层]]的构造，平均上下变位速度为1至2米每千年<ref name=honbu0 /><ref name=Yukari2 />。根据{{link-ja|地震调查研究推进本部|地震調査研究推進本部}}地震调查委员会的推测，关谷断层每2600年至4100年左右便会发生一次7.5级程度的[[地震|地震]]<ref name=honbus />。除此之外，关谷断层是23个[[关东地方|关东地方]]中可能会发生[[日本气象厅震度阶级|烈度]]5强地震的[[活断层|活断层]]之一<ref name=JMA />。

== 研究历史 ==
日本当局在[[阪神大地震|1995年阪神淡路大地震]]中吸取经验教训，为了加强防震措施，共挑选出98个可能的活动断层带，其中就包括关谷断层<ref name=tochigiofc />。2000年（平成12年），为阐明关谷断层在新活动时期的活动特征，日本活断层研究中心对该断层进行了实地考察<ref name=Yukari1 />。

== 活动历史 ==
根据{{link-ja|地震调查研究推进本部|地震調査研究推進本部}}地震调查委员会的推定，关谷断层的上一次活跃期处于14世纪至17世纪之间<ref name=honbus /><ref name=ijis />。其中，[[1683年日光地震|1683年日光地震]]即发生在该活跃期内。该地震造成了大规模的滑坡，在[[震中|震中]]附近形成了一个[[堰塞湖|堰塞湖]]，当时名为五十里湖。该堰塞湖在40年后（即1723年）的一场大雨中决堤，造成[[宇都宫|宇都宫]]地区约千人遇难<ref name=jmaut />。
