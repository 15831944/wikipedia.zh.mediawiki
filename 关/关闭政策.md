'''关闭政策'''或称'''闭港政策'''<ref name="a1"/>是自1949年6月起，[[中华民国政府|中华民国政府]]针对[[中国共产党|中国共产党]]政权，由[[中华民国海军|中华民国海军]]在[[渤海|渤海]]至[[南海|南海]]海域实行的军事[[封锁|封锁]][[政策|政策]]。1950年，扩大封锁区域，基本封锁了[[中国大陆|中国大陆]]全部[[海岸线|海岸线]]，意图摧毁中国共产党控制下的中国大陆的经济<ref name="a1">{{Cite web |url=http://taiwan.huanqiu.com/taiwan_history/2009-01/331471.html|title=“闭港政策”封锁大陆 台英军舰险爆海战 |accessdate=2017-07-08 |author= |date=2009-01-01 |work= |publisher=|language=Zh-hans }}</ref>。1950年代后，逐步演变成中华民国方面在[[台湾海峡|台湾海峡]]针对[[中华人民共和国|中华人民共和国]]的军事封锁和对往来民用船只的劫略。

此项政策对中华人民共和国海运，尤其是对外贸易产生不利影响。而随着双方军事实力及国际政治环境变化，中华人民共和国方面逐步突破军事封锁，恢复海运。1979年9月12日，中华民国政府公告中止《[[戡乱时期截断匪区海上交通办法|戡乱时期截断匪区海上交通办法]]》，有研究者将之视为法理上终止关闭政策<ref name="a6"/>。允许[[中華民國海軍|海军]]、[[中華民國空軍|空军]]搜捕、击毁中华人民共和国籍船只的《[[投匪資匪之輪船公司及船隻緊急處置辦法|投匪資匪之輪船公司及船隻緊急處置辦法]]》直到1992年1月方才廢除<ref name=海賊王>{{Cite web |url=https://www.storm.mg/lifestyle/939009|title=蔣介石是國際公認海賊王！派軍艦洗劫貨船，連英國、蘇聯都敢惹…揭台灣「海盜王國」黑歷史 |accessdate=2019-02-24 |author=蔡佳妘 |date=2019-02-14 |work= |publisher=風傳媒|language=Zh-tw }}</ref>。

== 开始 ==
1949年6月[[渡江战役|渡江战役]]结束，中国共产党领导[[中国人民解放军|中国人民解放军]]占领包括中华民国首都[[南京市_(中華民國)|南京市]]、全国经济中心上海在内的[[长江中下游|长江中下游]]流域，这是中国人口密集和最为富庶的地区，同时亦标志着中华民国政府及[[中华民国国军|中华民国国军]]在[[第二次国共内战|第二次国共内战]]中败局已定。解放军攻占[[浙江省_(中華民國)|浙江省]]、[[福建省_(中華民國)|福建省]]、[[廣東省_(中華民國)|广东省]]后，逐步攻占[[中国东南地区|东南]]各省的沿海岛屿。

1949年6月18日，[[行政院|行政院]]发布《卅八穗五字第四八九六号令》，自6月26日上午零时零分起，“关闭”北起[[辽河口|辽河口]]、南至[[闽江口|闽江口]]的[[中国领海|中国领海]]，所有外籍船舶及航空器均不得进入上述区域，而区域内各港口，也自即日起停止对外开放<ref name="a2">{{cite thesis|chapterurl=http://nccur.lib.nccu.edu.tw/bitstream/140.119/49868/4/15300104.pdf |chapter=第一章 緒論|title=《封鎖大陸沿海──中華民國政府的「關閉政策」，1949-1960》 |format=PDF |author=林宏一| date=2009-06-00 |degree=碩士|accessdate=2019-07-08|language=Zh-hant}}</ref><ref>原文引用来源：行政院令（卅八穗五字第四八九六号）（1949年6月18日），总统府第五局编，《总统府公报》第贰贰玖号（广州：总统府第五局，1949），4版。</ref>。时辽河口东部的[[旅顺港|旅顺港]]被[[苏联|苏联]]占领，是否因此不在关闭之列，不详。[[6月23日|6月23日]]起，[[中华民国海军|中华民国海军]]海防第一艦隊开始執行封锁[[长江口|长江口]]。在海軍第一艦隊司令[[刘广凯|刘广凯]]的指挥下，海防第一舰队自舟山出动百余次，拦截美、英籍货轮41艘，捕获汽船25艘、机帆船200余艘<ref name="a3">{{Cite web |url=http://news.sina.com.cn/c/2009-05-20/111217853853.shtml|title=封锁长江口使上海经济陷入困境 |accessdate=2017-07-08 |author= |date=2009-05-20 |work= |publisher=新浪网，来源：新华社-瞭望东方周刊|language=Zh-hans }}</ref>。

10月1日，中华人民共和国成立。10月3日，[[苏联|苏联]]与中华人民共和国建立外交关系，此后中华人民共和国成为[[东方阵营|东方阵营]]一员，与[[美国|美国]]、中华民国所属的[[西方阵营|西方阵营]]在[[冷战|冷战]]期间长期对立。10月至11月，[[金门战役|金门战役]]、[[登步岛战役|登步岛战役]]中，解放军相继战败，国军得以固守[[金门岛|金门岛]]、[[舟山群岛|舟山群岛]]。11月下旬，[[英国海军远东舰队|英国海军远东舰队]]四艘驱逐舰护送多艘商船前往[[长江口|长江口]]，意图前往[[上海港|上海港]]。24日，英国商船在长江口外九段沙灯标水域被国军拦截。英军、国军双方在海上僵持三天，期间美籍商船“富兰克林”号强行闯关，国军开炮阻击，“富兰克林”号几乎被击沉，勉强驶入上海港。27日，英军主动退出长江口<ref name="a1"/><ref name="a3"/>。12月24日，中华民国海军开始在长江口布雷<ref name="a3"/>。

== 1950年代 ==

1950年1月，[[美国总统|美国总统]][[哈里·S·杜鲁门|杜鲁门]]发表“不介入台湾海峡争端”的声明。在此前的1949年8月，《[[中美關係白皮書|中美關係白皮書]]》发表后，[[美国政府|美国政府]]已停止对中华民国的军事援助。2月12日，中华民国政府方面宣布关闭海域扩大至中国大陆的全部沿海地带<ref name="a2"/><ref>原文未提及宣布“关闭港口政策”的部门。原文引用来源：[[许雪姬|许雪姬]]总策划，《[[Wikipedia:傳統百科全書條目專題/台灣歷史辭典|臺灣歷史辭典]]》（台北：行政院文化建设委员，2004），页1338，“关闭港口政策”条；（海军总司令部三十九年度作战关闭概况报告书），国史馆藏，《行政院暨所属档案》，军5—2—2.1，“海军总部卅十九年作战关闭报告书”。</ref>。

随着[[中国人民解放军空军|中国人民解放军空军]]实力增强，[[中华民国空军|中华民国空军]]已无明显优势确保执行封锁长江口的任务。4月，解放军[[华东军区|华东军区]]组建扫雷舰大队，10月前完成长江口扫雷任务<ref>{{Cite web |url=http://mil.sohu.com/20120727/n349181628.shtml|title=长江口扫雷 |accessdate=2017-07-08 |author= |date=2012-07-27 |work= |publisher=搜狐网，来源：新华网|language=Zh-hans }}</ref>。5月，中华民国政府完成[[舟山撤退|舟山撤退]]。6月，[[朝鲜战争|朝鲜战争]]爆发，美国政府改变态度，下令[[美國海軍第七艦隊|美国海军第七舰队]]巡防台湾海峡。

美方出于避免双方擦枪走火的可能性和团结西方国家的目的，要求中华民国政府停止关闭政策。[[中华民国国防部|中华民国国防部]]于6月底和7月30日，两度明令暂停实施闭关政策。但实际执行时却是阳奉阴违<ref name="a4"/>。8月，[[中华民国外交部|中华民国外交部]]提交行政院审核《关闭部分领海及港口暂行办法》。同时，行政院通过[[中华民国交通部|中华民国交通部]]提交的《投匪资匪之轮船公司及船只紧急处置办法》<ref name="a4">{{cite thesis|chapterurl=http://nccur.lib.nccu.edu.tw/bitstream/140.119/49868/6/15300106.pdf |chapter=第三章 第一节|title=《封鎖大陸沿海──中華民國政府的「關閉政策」，1949-1960》 |format=PDF |author=林宏一| date=2009-06-00 |degree=碩士|accessdate=2019-07-09|language=Zh-hant}}</ref>。12月15日，行政院核准《与中华民国无外交关系国家之商船及民用航空器出入境临时规则》。细则于此前3月14日，由中华民国交通部公布实行<ref name="a5"/>。

1952年5月23日，行政院修正公布实行《[[截断匪区海上交通办法|截断匪区海上交通办法]]》<ref name="a5">{{cite thesis|chapterurl=http://www.maritime.org.tw/cmi/WangKung/04.pdf |chapter=第三章 我国航政法规之体系 七 其他有关航政法规|title=航政法规要义 |format=PDF |author=王洸| date= |degree=|accessdate=2017-07-09|pages=14.15|language=Zh-hant}}</ref>。

1954年，中華民國國軍分別攔捕波蘭貨輪高德瓦（Prezydent Gottwald號）、蘇聯籍油輪[[陶普斯號事件|陶普斯號]]；同年9月，[[日本|日本]]籍貨輪[[日洲丸|日洲丸]]在[[基隆港|基隆港]]躲避風浪時被發現是運[[煤礦|煤礦]]到[[秦皇島|秦皇島]]而依相關規定扣押。<ref name=日洲丸>[https://www.archives.gov.tw/ALohas/ALohasColumn.aspx?c=1637 關閉政策－日洲丸扣留事件]，國家發展委員會檔案管理局</ref>

== 1960年代 ==
1963年7月12日，行政院發布《台五十二防字第四七二〇号令》，修正《[[截断匪区海上交通办法|截断匪区海上交通办法]]》为《[[戡乱时期截断匪区海上交通办法|戡乱时期截断匪区海上交通办法]]》<ref>{{cite thesis|chapterurl=http://twinfo.ncl.edu.tw/tiqry/hypage.cgi?HYPAGE=search/merge_pdf.hpg&sysid=E0969610&jid=79002356&type=g&vol=52080700&page=頁2-3%2B8 |chapter=中央法规 戡乱时期截断匪区海上交通办法 |title=台湾省政府公报五十二年秋字第三十三期 |format=PDF |author=| date=1963-07-12|degree=|accessdate=2019-07-09|language=Zh-hant}}</ref>，规定由[[中华民国国防部|国防部]]命令[[国防部海军司令部|海军总司令部]]执行对中华人民共和国的航运封锁，制定详细规则，明确奖赏金额。

1968年，中华人民共和国籍船只以绕行[[太平洋|太平洋]]的方式[[打通南北航线|打通南北航线]]，恢复中国大陆南北港口间的航运。

== 结束 ==

1974年1月，[[西沙海战|西沙海战]]爆发。[[中国人民解放军海军东海舰队|解放军东海舰队]]驰援[[中国人民解放军海军南海舰队|南海舰队]]。21日，[[海军舰队首航台湾海峡|南下舰艇编队在福州军区全力策应下首航台湾海峡]]。解放军海军由此突破封锁。1979年5月19日，中华人民共和国籍货轮眉山轮在[[台湾海峡复航|台湾海峡复航]]。9月12日，中华民国政府公告中止《戡乱时期截断匪区海上交通办法》。由中华民国政府内部秘密颁行的规范外籍船只的《[[管制外籍船舶资匪航运临时办法|管制外籍船舶资匪航运临时办法]]》，何时终止不详<ref name="a6">{{cite thesis|chapterurl=http://nccur.lib.nccu.edu.tw/bitstream/140.119/49868/8/15300108.pdf |chapter=结论|title=《封鎖大陸沿海──中華民國政府的「關閉政策」，1949-1960》 |format=PDF |author=林宏一| date=2009-06-00 |degree=碩士|accessdate=2019-07-09|language=Zh-hant}}</ref>。

== 影響與評價 ==
藉由閉關政策，中華民國政府劫了140多艘船，為政府提供急需物資與金錢。雖然此政策不被國際承認的，各國船隻仍前往中國大陸，卻多次被台灣攔截、搶奪，包含[[德國|德國]]、[[英國|英國]]、[[波蘭|波蘭]]、[[日本|日本]]、[[荷蘭|荷蘭]]、[[蘇聯|蘇聯]]等國的船隻。<ref name=海賊王/>

一些掠奪來的船隻多數都改名編入艦隊，包括原屬蘇聯的[[會稽號|會稽號]]，還有原屬於[[波蘭|波蘭]]商輪和[[油輪|油輪]]的[[賀蘭號|賀蘭號]]及[[太湖號|太湖號]]。閉關政策雖然解決台灣戰後物資不足的困境，但也傷及中華民國的國際名聲。<ref name=海賊王/>

蘇聯曾經在[[聯合國|聯合國]]提案指控中華民國對蘇聯、英國、波蘭等國的船隻進行[[海盜|海盜]]行為，並且違反公海[[航行自由|航行自由]]原則；波蘭也在聯合國國際法委員會中將[[中華民國海軍|中華民國海軍]]的行為斥為海盜行為。但根據《[[公海公約|公海公約]]》及《[[聯合國海洋法公約|聯合國海洋法公約]]》，海盜行只限於「由私船所發動者」，因此中華民國海軍的行為不構成海盜行為。<ref name=海賊王/>
