'''关晓彤'''{{BD|1997年|9月17日|||catIdx=G关}}，[[中國大陸|中國大陸]]女演员，出生于[[北京|北京]]，[[满族|满族]]人，滿姓為[[瓜爾佳氏|瓜爾佳氏]]，父亲是影视演员关少曾，祖父是北京[[琴书|琴书]]大师关学曾。4歲開始拍戲，參演過的影视作品超過100部；現就讀于[[北京电影学院|北京电影学院]]2016級表演系。2017年，凭借在《[[好先生_(电视剧)|好先生]]》中出演叛逆小孩彭佳禾一角，荣获第23届白玉蘭獎最佳女配角奖。

== 生平 ==
1997年，关晓彤出生于[[北京市|北京市]]的一个艺术世家，祖父关学曾是[[北京琴书|北京琴书]]大师，父亲关少曾是影视演员，由于家庭文化的熏陶，她从小也热爱艺术，喜欢[[舞蹈|舞蹈]]和[[钢琴|钢琴]]。<ref name=" girl">{{cite web|url=http://ep.zjgy.gov.cn/hengdian/html/2008-12/17/content_19082.htm|title=关晓彤在横拍戏|accessdate=2008年12月17日|publisher=横店报纸|work=|language=zh-cn}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>4岁半的时候，关晓彤和父亲关少曾一起拍戏，自此出道。身高172公分，2016年4月，關曉彤[[北京电影学院|北京電影學院]]藝考90.41分第一名，六月高考成績公布552分。2016年7月27日，被北京电影学院表演学院表演专业本科录取<ref>{{cite web |url=http://ent.qq.com/a/20160727/029701.htm?qqcom_pgv_from=tips |title=关晓彤父亲晒女儿录取通知书：老爸衷心祝贺 |accessdate= 2016年7月27日|publisher=腾讯娱乐 |work= |language=zh-cn}}</ref>。

=== 争议 ===
2019年2月15日在[[浙江卫视|浙江卫视]]播出的《[[王牌对王牌_(真人秀)|王牌对王牌]]》节目中，关晓彤和心灵之声残疾人艺术团表演的舞蹈《[[千手观音_(舞蹈)|千手观音]]》涉嫌侵权。节目首播一小时后，其著作权人[[中国残疾人艺术团|中国残疾人艺术团]]为此发表声明，表示将保留进一步追究法律责任的权利<ref>{{cite web |url=http://ent.sina.com.cn/tv/zy/2019-02-16/doc-ihrfqzka6199583.shtml |title=中国残疾人艺术团发声明 关晓彤千手观音舞蹈侵权|accessdate= 2019-02-19|publisher=新浪娱乐}}</ref>。此次事件引发了网友争议。2月16日，《王牌对王牌》节目组通过社交媒体账号发表了《关于舞蹈节目的情况说明》，并公开致歉<ref>{{cite web |url=http://ip.people.com.cn/n1/2019/0218/c179663-30760072.html|title=关晓彤版《千手观音》引发侵权争议|accessdate= 2019-02-19|publisher=北京青年报}}</ref>。
