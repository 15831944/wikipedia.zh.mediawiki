'''关庄站'''位于[[北京市|北京市]][[朝阳区_(北京市)|朝阳区]]，是[[北京地铁15号线|北京地铁15号线]]的一座车站。车站于2012年7月29日封顶，为15号线西段各站中首个封顶的车站<ref name="ztzx" />。车站于2014年12月28日随15号线一期西段开通一同投入使用。

== 位置 ==
关庄站位于[[北京四环路|北四环]]外，[[大屯路|大屯路]]（东西向）与一条规划路相交处，呈东西走向布置。<ref>{{cite web|url=http://www.bjghw.gov.cn/web/static/articles/catalog_98000/article_ff808081271d5bde01278f4bd81901e0/ff808081271d5bde01278f4bd81901e0.html|title=北京市规划委员会关于轨道交通15号线（一期）沿线车站命名预案的公示书书|date=2010-04-06|accessdate=2014-12-27|publisher=北京市规划委员会|archive-url=https://web.archive.org/web/20141227235123/http://www.bjghw.gov.cn/web/static/articles/catalog_98000/article_ff808081271d5bde01278f4bd81901e0/ff808081271d5bde01278f4bd81901e0.html|archive-date=2014-12-27|dead-url=yes}}</ref><ref>{{cite web|url=http://www.bjghw.gov.cn/web/static/articles/catalog_14300/article_ff808081262b02de01262c30a5a2000a/ff808081262b02de01262c30a5a2000a.html|title=轨道交通15号线一期工程规划方案公告|date=2010-01-14|accessdate=2014-12-27|publisher=北京市规划委员会|archive-url=https://web.archive.org/web/20100728091626/http://www.bjghw.gov.cn/web/static/articles/catalog_14300/article_ff808081262b02de01262c30a5a2000a/ff808081262b02de01262c30a5a2000a.html|archive-date=2010-07-28|dead-url=yes}}</ref>

== 结构 ==
车站为地下车站，由于拆迁问题，车站明挖用地不足，故而增加设备层，为地下三层车站<ref>{{cite news|url=http://epaper.jinghua.cn/html/2013-03/25/content_1977462.htm|title=地铁15号线西段明年底通车|author=黄海蕾|work=京华时报|date=2013-03-25|accessdate=2014-02-06|deadurl=yes|archiveurl=https://web.archive.org/web/20140221232706/http://epaper.jinghua.cn/html/2013-03/25/content_1977462.htm|archivedate=2014-02-21}}</ref>。车站结构总长144.1米，标准段宽20.9米，基坑深度为25.68米。采用[[岛式站台|岛式站台]]设计<ref name="ztzx">{{cite news|url=http://zt.ztzxcn.cn/bencandy.php?fid=220&id=8180|title=北京地铁15号线09标段关庄车站主体结构顺利封顶|author=张扬|publisher=中铁在线|date=2012-08-06|accessdate=2014-02-06|deadurl=yes|archiveurl=https://web.archive.org/web/20140222164125/http://zt.ztzxcn.cn/bencandy.php?fid=220&id=8180|archivedate=2014-02-22}}</ref>。

== 名称问题 ==
关庄站与[[管庄站|管庄站]]的英文名称同为Guanzhuang，且站名标示中没有声调符号，极易混淆，曾有在京外国人因此误乘。2012年7月曾有网民建议关庄站更名，但这一问题并未得到足够重视。<ref>{{cite news|url=http://bjwb.bjd.com.cn/html/2016-04/15/content_25954.htm|title=站名相似闹笑话 让人“傻傻分不清”|publisher=[[北京晚报|北京晚报]]|date=2016-04-15|accessdate=2016-04-15|deadurl=yes|archiveurl=https://web.archive.org/web/20160508110608/http://bjwb.bjd.com.cn/html/2016-04/15/content_25954.htm|archivedate=2016-05-08}}</ref>类似问题在广州[[蕭崗站|蕭崗站]]与[[晓港站|晓港站]]之间同样存在，但广州地铁为区分两个车站，将萧岗站的英文拼写定为Xiao-gang Station，北京地铁并无类似的区分手段。
