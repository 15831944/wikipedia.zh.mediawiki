《'''关于我被小学女生绑架这件事'''》是[[中国大陆|中国大陆]]的[[同人游戏|同人游戏]]制作组猫薄荷制作组和ZiX Solutions所创作的一部[[视觉小说|视觉小说]]，于2018年6月1日登上[[Steam|Steam]]游戏平台，是一款免费游戏。玩家所扮演的游戏男主角是一名有着[[萝莉控|萝莉控]]倾向的刚入职的小学教师，故事围绕着这名教师暑假到学校整理资料却被一名小学女生[[绑架|绑架]]展开。<ref name="快吧游戏">{{cite web|title=关于我被小学女生绑架这件事|url=http://pc.kuai8.com/game/gywbxxnsbjzjs/|website=快吧游戏|accessdate=2018-06-16|archiveurl=https://archive.today/20180616041325/http://pc.kuai8.com/game/gywbxxnsbjzjs/|archivedate=2018年6月16日|deadurl=no|df=}}</ref><ref>{{cite web |author1=小〇驴 |title=每日新游预告 《关于我被小学女生绑架这件事》同人社团倾力之作 |url=http://www.yxbao.com/xw/141616.html |website=游戏堡 |accessdate=2018-06-16 |archiveurl=https://web.archive.org/web/20180616025304/http://www.yxbao.com/xw/141616.html |archivedate=2018-06-16|deadurl=no |date=2018-05-17}}</ref>

==故事简介==
男主角是一名有着萝莉控倾向的刚入职的小学老师，暑假受教导主任之托到学校档案室整理资料，却被一名小学女生小可用[[电击枪|电击枪]]击晕并用[[手铐|手铐]]铐住。男主角应要求跟小可玩[[脑筋急转弯|脑筋急转弯]]的过程中，发现小可是他小学时期的恋人的孩子，也了解到小可的父亲便是教导主任且实施[[家庭暴力|家庭暴力]]致小可的母亲死亡。最终男主角决定收养小可，但也惊奇地发现其实小可是男生。

在游戏的坏结局中，男主角被以绑架和[[谋杀|谋杀]]的罪名起诉。

在游戏续集中，男主角因为要去[[相亲|相亲]]，便把小可托付给一个女同事照顾。小可知道这个女同事一直暗恋男主角，便带着她跟踪男主角。

==角色==

;小可
:配音：閒踏梧桐
:游戏女主角，一名穿着女装的小学男生，因为家庭的原因而喜欢把电击枪、刀等危险物品当玩具。

==评价==
玩家对[[背景音乐|背景音乐]]的评价褒贬不一，[[17173|17173]]的编辑新游酱认为背景音乐很动听<ref name="17173.com">{{cite web |author1=新游酱 |title=关于我被小学女生绑架这件事 新游酱每日游戏推荐 |url=http://newgame.17173.com/content/06042018/102008636_all.shtml |website=[[17173|17173]] |accessdate=2018-06-16 |archiveurl=https://web.archive.org/web/20180616025242/http://newgame.17173.com/content/06042018/102008636_all.shtml |archivedate=2018-06-16|deadurl=no |date=2018-06-04}}</ref>，但ACG批评的站长windchaos则表示背景音乐没有很好地渲染出氛围<ref name="ACG批评">{{cite web |author1=windchaos |title=国产同人Gal《关于我被小学女生绑架这件事》赏析 |url=https://www.acgpiping.net/2018/8195/国产同人gal《关于我被小学女生绑架这件事》赏析/ |website=ACG批评 |accessdate=2018-06-16 |archiveurl=https://web.archive.org/web/20180616030031/https://www.acgpiping.net/2018/8195/%E5%9B%BD%E4%BA%A7%E5%90%8C%E4%BA%BAgal%E3%80%8A%E5%85%B3%E4%BA%8E%E6%88%91%E8%A2%AB%E5%B0%8F%E5%AD%A6%E5%A5%B3%E7%94%9F%E7%BB%91%E6%9E%B6%E8%BF%99%E4%BB%B6%E4%BA%8B%E3%80%8B%E8%B5%8F%E6%9E%90/ |archivedate=2018-06-16|deadurl=no |date=2018-01-21}}</ref>。

在剧情上，不少玩家表示最后的[[伪娘|伪娘]][[彩蛋_(媒体)|彩蛋]]让人很惊喜<ref name="17173.com" />，windchaos则不以为然，他觉得这个彩蛋对表达作品的中心思想没有帮助。也有玩家指出，这款游戏的剧情跟[[日本|日本]]作家[[西尾维新|西尾维新]]的[[轻小说|轻小说]]作品《[[少女不十分|少女不十分]]》存在雷同之处，均讲述了一名成年人被一名小学生限制人身自由的故事。<ref>{{cite web |author1=St淼 |title=关于我被小学女生绑架这件事 |url=https://up.163.com/appreciation?appreciationId=18325 |website=UP |accessdate=2018-06-16 |archiveurl=https://web.archive.org/web/20180616025315/https://up.163.com/appreciation?appreciationId=18325 |archivedate=2018-06-16|deadurl=no |date=2018-06-12}}</ref>
