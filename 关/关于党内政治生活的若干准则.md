《'''关于党内政治生活的若干准则'''》，是1980年3月4日，[[中国共产党第十一次全国代表大会|中国共产党第十一届]][[中国共产党第十一届中央委员会#五中全会|五中全会]]通过的一项文件，其目的是为了恢复自[[文化大革命|文化大革命]]动乱后的[[中国共产党|中国共产党]]党内纪律与作风问题的一项准则<ref>{{cite news|title=中共十一届五中全会通过了《关于党内政治生活的若干准则》|url=http://fanfu.people.com.cn/GB/143349/165093/165096/165116/9892759.html|accessdate=2012-06-07|newspaper=人民网}}</ref>。

此次第十一届五中全会做出了原国家主席[[刘少奇|刘少奇]]平反的决议；并增选[[胡耀邦|胡耀邦]]、[[赵紫阳|赵紫阳]]为[[中共中央政治局常委|中共中央政治局常委]]；恢复设立[[中共中央书记处|中共中央书记处]]，胡耀邦为[[中国共产党中央书记处总书记|总书记]]<ref>{{cite news|title=《关于党内政治生活的若干准则》|url=http://gaige.rednet.cn/c/2008/02/28/1448953.htm|accessdate=2012-06-07|newspaper=红网}}</ref>。

== 内容 ==
其中包括十二项原则<ref>{{cite news|title=关于党内政治生活的若干准则|url=http://news.xinhuanet.com/ziliao/2005-02/04/content_2548139.htm|accessdate=2012-06-07|newspaper=新华网}}</ref>：
# 坚持党的政治路线和思想路线；
# 坚持集体领导，反对个人专断；
# 维护党的集中统一，严格遵守党的纪律；
# 坚持党性，根绝派性；
# 要讲真话，言行一致；
# 发扬党内民主，正确对待不同意见；
# 保障党员的权利不受侵犯；
# 选举要充分体现选举人的意志；
# 同错误倾向和坏人坏事作斗争；
# 正确对待犯错误的同志；
# 接受党和群众的监督；
# 努力学习，做到[[又红又专|又红又专]]。
