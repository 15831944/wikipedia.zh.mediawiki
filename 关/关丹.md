'''关丹'''（{{lang-ms|Kuantan}}），是[[马来西亚|马来西亚]][[彭亨|彭亨]]州的首府，也是西马东海岸最大的城市，面向[[南中国海|南中国海]]（South China Sea），位于关丹河口附近。人口大约340,000人，其中约58%马来人、32%华人、4%印度人以及6%其它人种。行政区面积达324平方公里。1955年8月27日，彭亨州政府首府从[[瓜拉立卑|瓜拉立卑]]迁至此地。

== 历史 ==
学界一般认为关丹建城于1850年，历史文献中，最早有关丹这个名称出现是在1851年或1852年期间。早期的关丹是一座村庄，村民以打鱼和小型贸易为生。19世纪末到20世纪初，华人矿工和商人建立了关丹地区最早的城镇，附近的锡矿以及橡胶庄园吸引了来自印度的移民。

== 经济 ==
旅游业是关丹的一个重要经济支柱，其在马来西亚国内是一个著名的手工艺产地，比如峇迪印染。同时它也是彭亨州的行政和经济中心，贸易和商业也是关丹经济的重要组成成分，林业，工业和渔业对当地经济发展也起着非常关键的作用。

此外，关丹还有石油工业，这些企业主要位于市北约25公里处的工业区。在这里的企业有巴斯夫国油化学私人有限公司、英国石油化学公司、MTBE、富兰克斯、伊士曼化学公司、原钟渊化学工业、宝理、美固、科恩马、喜悦尔公司等。

关丹港位于该工业区以东仅5公里处，它是一个多功能、全天候海港，拥有复杂的器械来处理各种货物，包括集装箱、液体、-{zh-hans:干; zh-hant:乾;}-散货和杂货。1998年该港被私有化，它是马来半岛东海岸最重要的海港和物流中心。

== 基础建设 ==
关丹与[[吉隆坡|吉隆坡]]通过东海岸大道相连，这条高速公路也将关丹与其它重要城市（哥打巴鲁和[[瓜拉登嘉楼|瓜拉登嘉楼]]）连在一起。

市内有数座医院，其主要医院是Tengku Ampuan Afzan医院。这座医院拥有许多现代设备，包括X射线断层成像和核磁共振成像。它也是[[马来西亚国际伊斯兰大学|马来西亚国际伊斯兰大学]]的教学医院，郊区有小型的社区卫生中心，市内还有两座私人卫生中心。

距离市中心16 km还有一座'''[[苏丹阿末沙机场|苏丹阿末沙机场]]'''作为空中交通的联络站，除了马来西亚国内航线之外，也曾经有来往台湾台北-桃园的旅游包机。

== 旅游业 ==
[[File:TelukCempedak.jpg|thumb]]
关丹的主要观光景点为直落尖不辣海滩（{{lang|ms|Teluk Cempedak}}），其他临近著名海滩包括黑沙滩（{{lang|ms|Batu Hitam}}）, 巴洛（{{lang|ms|Balok}}）以及芝拉汀（{{lang|ms|Cherating}}）。芝拉汀附近有一个海龟保护区，海岸前数千米有一座蛇岛（{{lang|ms|Pulau Ular}}），这里有一个“蛇与渔村”的传说。

关丹的瀑布也很著名，市内还有三个公园，关丹的手工艺品如白眉印染也很著名，关丹的大清真寺也是旅游景点。

关丹有许多购物中心以及数座五星级旅馆和四星级旅馆，外国人一般选择靠海滩的休假地。

== 食品 ==
关丹的炸鱼片和咸鱼很有名，咸鱼的制法是用盐腌鱼后放在日下晒数日。其它著名食品包括[[馕|馕]]和[[烤鸡|烤鸡]]。当地的馕与[[印度煎饼|印度煎饼]]类似，但是比印度煎饼大，油比较少，可以沾各种汁，包括[[拉茶|拉茶]]。

市中心有多种当地的食品。

== 教育 ==
关丹的基础教育由国民学校提供，其学科是统一由马来西亚教育部规定的。关丹有一座伊斯兰宗教学校，它除宗教课程外也提供教育部规定的普及教育课程。市内的两座国际学校不使用马来西亚教育部的课程安排。关丹有数座高等教育机构，如[[馬來西亞國際伊斯蘭大學|马来西亚国际伊斯兰大学]]以及一些私人大学。

== 媒体文化中的关丹 ==
《[[大日子_(电影)|大日子]]》是马来西亚本土的首部中文贺岁电影，于2010年发行，全片大部分是在关丹的米昔拉村拍摄。

== 其它 ==
* 彭亨州的苏丹宫殿位于关丹。
* 在电脑游戏《[[命令与征服|命令与征服]]》中关丹是挪得兄弟会的重要基地。
* 关丹华裔多数以粤语为沟通语言。
