'''关节炎'''是全世界最常见的[[慢性疾病|慢性疾病]]之一，主要是关节的[[软骨|软骨]]退化或者[[结缔组织|结缔组织]][[发炎|发炎]]，導致关节疼痛從而干擾关节的正常運動就叫关节炎，總共有一百多個種類<ref name="zz">{{cite web |url = http://www.jjkkusa.com/modules/news/article.php?storyid=521 |title=关节炎 |publisher=健健康康網 |accessdate=2011-11-14}}</ref>。导致关节炎的原因很多，引起关节损伤也各有不同。全世界约有3.55亿关节炎患者，其中在[[中华人民共和国|中國]]就占了一億以上<ref name="zz"/>。在[[美国|美国]]，每五个人中就有一人出现关节炎的疼痛與不适；而在[[亚洲|亚洲]]，每六個人中就有一個人在人生某個時刻罹患此病。关节炎並不是[[老年|老年]]人獨有的疾病，它可以影響包括[[兒童|兒童]]在内的各个年齡層<ref name="zz"/>。

== 概述 ==
關節炎可以發生在[[背|背]]、[[颈|颈]]、[[膝盖|膝盖]]、[[肩关节|肩关节]]、[[手|手]]、[[髋关节|髋关节]]、[[足踝|足踝]]<ref name="zz" />。很多的關節炎和人體老化有關聯，六十歲以上人群会有某种類型的关节炎但年輕人一樣會患關節炎<ref name="zz" />。有一百多個種類多见的是[[骨性關節炎|骨性關節炎]]、类风湿性、风湿性、化脓性关节炎、外伤性骨性关节炎、自身免疫性關節炎，僵直性關節炎也是關節炎

根據病癥與患病時間可分成急性與慢性這兩種，急性关节炎為突發疾病，病患的关节會發紅變腫且痛热并會發生功能障碍與全身发热；慢性关节炎則是顯示出关节肿痛和[[畸形|畸形]]、不同水平的功能障碍，早期关节炎病变只侵擾关节[[滑膜|滑膜]]，如果到了晚期关节软骨和[[骨质|骨质]]都可能會出現变化甚至被破坏<ref name="rr">{{cite web |url = http://www.haodf.com/jibing/guanjieyan/jieshao.htm |title = 关节炎介绍 | publisher=好大夫在線 | accessdate=2011-11-14}}</ref>。

最常見的骨性關節炎和[[类风湿性關節炎|类风湿性關節炎]]的并發癥不相同，骨性關節炎會并发肢体关节活动障礙，嚴重的時候会出現肢体内翻、屈曲挛缩畸形，最終導致關節殘疾。而类风湿性关节炎如果長時間在床上休養又或者[[激素|激素]]的服用時間過長可導致[[免疫功能|免疫功能]]下降而出现一些并发症，主要會有[[肺炎|肺炎]]、[[泌尿系统|泌尿系统]]感染、柯兴氏综合征、口腔溃疡、[[传染病|传染病]]等<ref>{{cite web |url = http://www.qqyy.com/jibing/changshi/bingfa/132.html |title=关节炎的并发症有哪些？ |publisher=[[同濟醫院|同濟醫院]] | accessdate=2011-11-14}}</ref>。

== 症狀 ==
關節炎的症狀非常多樣，主要出現受累关节發红、肿、热、痛與功能障碍，身體检查可以觀察到病变关节出現異常現象，像是关节局部[[皮肤|皮肤]]发红和温度發生变化、周围[[软组织|软组织]]發腫發漲、关节腔积水、用手指按病人的关节局部可產生[[疼痛|疼痛]]現象，如果严重的會有关节畸形、关节活动範圍變小等<ref name="kk">{{cite web |url = http://www.haodf.com/zhuanjiaguandian/hudawei_46188174.htm |title = 千变万化的关节炎 |publisher = 上海[[仁濟醫院|仁濟醫院]] | accessdate=2011-11-14}}</ref>。

其他臨床現象還有肌肉酸痛、面部红斑、無力、[[发热|发热]]、尿路刺激、咽痛、复发性[[口腔溃疡|口腔溃疡]]、[[腹泻|腹泻]]、眼口干燥，有的甚至會出現[[贫血|贫血]]、肾發炎、浆膜發炎等，這些現象可以單獨出現也可同時出現，這些現象為醫生判斷疾病的重要證據<ref name="kk"/>。

== 病因 ==
目前醫學界中對關節炎的產生原因不是了解的很清楚，有些专家相信[[自由基|自由基]]是導致[[退行性疾病|退行性疾病]]的主謀，關節炎是其中一種。


如果缺少必需[[礦物質|礦物質]]那[[胶原蛋白|胶原蛋白]]基质中容有滑液的囊便無法形成，滑液可以使[[骨骼|骨骼]]末端不至于受到相鄰骨骼與軟骨的沖擊。

=== 类风湿性 ===
类风湿性关节炎為免疫系统疾病，病因目前不清楚還沒有根治方法。医学研究认为类风湿性关节炎和[[遗传|遗传]]有關係，因為自身免疫系统失调，從而錯誤攻擊关节软骨和骨骼组织。病患全身关节滑膜常常会被侵犯而发炎，要長期服藥才能控制<ref name="ybl">{{cite web|url =http://www.hkscf.org/cn/index/headline/?newsid=165|title =認識免疫系統疾病類風濕性關節炎治療新知|publisher =香港專科醫護基金|accessdate =2011-11-14|deadurl =yes|archiveurl =https://web.archive.org/web/20131105135431/http://www.hkscf.org/cn/index/headline/?newsid=165|archivedate =2013-11-05}}</ref>。类风湿性常和受寒、受潮、外伤、精神刺激有關聯，這些因素可能是誘因而不是病因。

一些致病原像是[[细菌|细菌]]、[[病毒|病毒]]、[[衣原体|衣原体]]、[[螺旋体|螺旋体]]等都可導致不同动物出現类风湿性样病癥。临床也發現一些类风湿性发生在某些感染後，像是结核杆菌、奇异变形杆菌、[[链球菌|链球菌]]、EB病毒、衣原体感染等。於病人的[[血清|血清]]或滑膜液里面觀察到相對应抗原之抗体提高，但還沒理清致病的抗原成分<ref name="dd">{{cite web |url = http://www.cnkang.com/nkwk/gjycs/110927.html |title=关节炎的病因与发病机制 |publisher=中華康網 |accessdate=2011-11-14}}</ref>。感染因子可能在类风湿起病中有重要作用<ref name="dd" />。

=== 骨性關節炎 ===
{{Main|骨關節炎}}

骨性關節炎分為繼發性骨性关节炎和原发性骨性关节炎，继发性骨性关节炎局部病因基礎上因软骨退变而发生的骨性关节炎，導致原因有关节内骨折、关节[[发育不良|发育不良]]、关节外畸形、关节不稳定、医源性、[[肥胖|肥胖]]超重、過去有关节感染、关节过度使用、肌肉无力，骨性关节炎通常發病原因不是一種而是多種因素相互交織產生<ref name="shangh">{{cite web|url=http://www.cnback.com/html/gjy/gjyzz/bybl/1619.html|title=继发性骨性关节炎的病因|publisher=脊梁網|accessdate=2011-11-14}}{{dead link|date=2017年11月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

原发性骨性关节炎致病原因不清楚，為生理性骨关节软骨退化，在体力劳动者和妇女上比較常見，通常和年龄、遗传、体质肥胖超重的老年人、免疫异常、软骨代谢和其他因素有關聯。也和骨内[[静脉|静脉]]淤滞、骨内压作用、遗传有關係<ref name="shangh" />。

== 检查 ==
要做一般检查如观察病人走路姿態、面色、皮膚，如懷疑患有炎癥性關節炎則要細心檢查[[指甲|指甲]]和頭部、肘膝伸侧是否有[[银屑病|银屑病]]。還有关节檢查如紅斑活动范围、畸形软组织肿胀、关节渗液、骨性肿胀、骨擦音压痛肌萎缩等等<ref name="yy">{{cite web |url = http://www.qqyy.com/jibing/changshi/huayan/132.html |title=关节炎具体要做的检查化验？ |publisher=全球醫藥網 | accessdate=2011-11-14}}</ref>

[[实验室|实验室]]检查需要和患者的[[临床|临床]]表现相符合，因為整套的筛选实验检查沒有必要，實驗室檢查需要說明一個具體問題如最終结果應支持或推翻一個臨床結果，大多數的人只需要做些簡單的实验检查，而对所有人做急性反应检测是比較必要的，如果不尋常的升高則要考慮是炎症性关节炎。大部分的人需要做[[血常规|血常规]]检查如果贫血說明患有系统性或炎症性疾病，但也可能是為继发抗炎药導致的的上[[消化道|消化道]]出血<ref name="yy" />。

有炎症性关节炎的临床現象應該做[[免疫|免疫]]学检查，不過没有关节炎的人也会有些低度阳性自身抗体，要與临床表现相結合否则“假阳”性结果常常会引起誤診，如症状倾向于骨关节炎那除了血沉和血常规外那就不需要做其他檢查否則會誤導<ref name="yy" />。

影像学检查如[[B超|B超]]、[[X光|X光]]片、[[CT|CT]]、[[MRI|MRI]]等<ref name="yy" />。

== 治療 ==
* 一般療法：經常休息要保护关节，避免劇烈和過度活動，严重損傷时要臥床休息支架固定防止畸形<ref name="rr"/>

* 药物療法：[[非類固醇消炎止痛藥|非類固醇消炎止痛藥]]（NSAID）能缓解痛楚抗發炎，[[中草药|中草药]]内服和外部热敷、熏洗浸泡能緩解病癥延遲病程。关节內部注射[[透明质酸|玻尿酸]]，這是靠它的流变特性用作粘弹性物质补充，作用是润滑关节保护关节软骨。在關節內注射[[皮质激素|皮质激素]]类药物能於短期内缓解症状，但对软骨的损害隨著注射次數增多而增加<ref name="rr" />。治療關節炎的药可分成非特异性治疗药物（对症治疗）、特异性治疗药物（对因治疗）<ref name="gg" />

* 手術療法：骨关节炎晚期有畸形或持续性疼痛現象且生活不能自理的時候可以[[手术|手术]]治疗，如果膝内翻畸形可以于胫骨上實行高位截骨术、髋关节炎晚期可實行截骨术等，按年級和職業與生活作息等可选用关节置换术髋关节置换术等<ref name="rr" />。
* 物理治疗：[[微波|微波]]热疗是近年來國內外比較成功的理療手段，能對一些藥物無法治療的部分起到效果<ref name="gg">{{cite web |url = http://www.qqyy.com/jibing/changshi/zhiliao/132.html | title=关节炎应该如何治疗？ |publisher = 全球醫藥網 |accessdate=2011-11-14}}</ref>
*輔助治療：烤电、[[推拿|推拿]]、[[按摩|按摩]]、[[针灸|针灸]]等等，雖沒有根本療效但起到鎮靜、松弛的作用<ref name="gg" />。

== 膏药穴位贴敷法 ==

穴位贴敷疗法通过药物直接刺激穴位，并通过透皮吸收，使局部药物浓度明显高于其他部位，作用较为直接，其适应证遍及临床各科，“可与内治并行，而能补内治之不及”，对许多沉疴痼疾常能取得意想不到的显著功效。穴位贴敷疗法不经胃肠给药，无损伤脾胃之弊，治上不犯下，治下不犯上，治中不犯上下。即使在临床应用时出现皮肤过敏或水泡，亦可及时中止治疗，给予对症处理，症状很快就可消失，并可继续使用。穴位贴敷有许多较简单的药物配伍及制作，易学易用，不需特殊的医疗设备和仪器。无论是医生还是患者或家属，多可兼学并用，随学随用。穴位贴敷法所用药物除极少数是名贵药材外（如麝香），绝大多数为常见中草药，价格低廉，甚至有一部分来自于生活用品，如葱、姜、蒜、花椒等。且本法用药量很少，既能减轻患者的经济负担，又可节约大量药材。贴敷疗法集针灸和药物治疗之所长，所用药方配伍组成多来自于临床经验，经过了漫长岁月和历史的验证，疗效显著，且无创伤无痛苦，对惧针者，老幼虚弱之体，补泻难施之时，或不肯服药之人，不能服药之症，尤为适宜。 <ref>孙六合, 曹健美. 浅谈穴位贴敷疗法[J]. 河南中医药学刊, 2001, (2).</ref>

== 預防 ==
多吃新鮮蔬果，攝取[[維生素|維生素]]，[[葡萄胺|葡萄胺]]、软骨素、鸡软骨能起到修復和再生的作用。肥胖也會使患關節炎的機會增加，經常運動及減少關節承重壓力（如游泳。關節炎患者避免强烈運動如跑步等，以减少骨头間的摩擦），多運動能保持關節彈性防止軟骨退化。
