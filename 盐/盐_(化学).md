[[File:Lead(II)_sulfate.jpg|thumb]]
'''盐'''在[[化学|化学]]中，是指一类[[金属|金属]][[离子|离子]]或[[銨|銨根離子]]（NH<sub>4</sub><sup>+</sup>）与[[酸根离子|酸根离子]]或非金屬離子结合的[[化合物|化合物]]，如[[硫酸钙|硫酸钙]]，[[氯化铜|氯化铜]]，[[醋酸钠|醋酸钠]]，一般来说盐是[[复分解反应|複分解反应]]的生成物，如[[硫酸|硫酸]]与[[氢氧化钠|氢氧化钠]]生成[[硫酸钠|硫酸钠]]和[[水|水]]，也有其他的反应可生成盐，例如[[置换反应|置换反应]]。

盐分为[[單盐|單盐]]和[[合盐|合盐]]，單盐分為[[正盐|正盐]]、[[酸式盐|酸式盐]]、[[碱式盐|碱式盐]]，合盐分為[[復鹽|複盐]]和[[络盐|錯盐]]。其中酸式盐除含有金属[[离子|离子]]与酸根离子外还含有氢离子，碱式盐除含有金属离子与酸根离子外还含有[[氢氧根|氢氧根]]离子，[[複盐|複盐]]溶於水時，可生成與原盐相同[[离子|离子]]的合盐；[[络盐|络盐]]溶於水時，可生成與原盐不相同的複雜[[离子|离子]]的合盐－[[絡合物|絡合物]]。

通常在[[標準狀況|標準狀況]]下，不可溶的盐會是[[固態|固態]]，但也有例外，例如[[熔盐|熔盐]]及[[离子液体|离子液体]]。可溶盐的[[溶液|溶液]]及[[熔盐|熔盐]]有导电性，因此可作為[[電解質|電解質]]。包括[[細胞|細胞]]的[[細胞質|細胞質]]、[[血液|血液]]、[[尿液|尿液]]及[[礦泉水|礦泉水]]中都含有許多不同的盐類。

强碱弱酸盐是[[强碱|强碱]]和[[弱酸|弱酸]]反应的盐，溶于水显碱性，如[[碳酸钠|碳酸钠]]。而强酸弱碱盐是[[强酸|强酸]]和[[弱碱|弱碱]]反应的盐，溶于水显酸性，如[[氯化铁|氯化铁]]。

== 盐的分类 ==
=== 单盐 ===
*[[正盐|正盐]]：
正盐是一鹽类的一种，既不含能电离的氢离子，又不含氢氧根离子。
正盐是酸和碱完全中和的产物<ref name="魏明通2006">{{cite book|author=魏明通|title=普通化學|url=http://books.google.com/books?id=BGBqsgM1LIsC&pg=PA24-IA17|year=2006|publisher=五南圖書出版股份有限公司|isbn=978-957-11-4349-1|pages=24–}}</ref>，但正盐的水溶液不一定显中性，如<chem>Na2CO3</chem>（碳酸钠）溶液显碱性，<chem>(NH4)2SO4</chem>（硫酸铵）溶液显酸性。酸跟碱完全中和生成的盐中，不会有酸中的氢离子，也不会有碱中的氢氧根离子，只有金属阳离子和酸根离子，这样的盐为正盐。生成正盐的反应。

*[[酸式盐|酸式盐]]：
[[File:SodiumBicarbonate.png|thumb]]
酸式盐是盐类的一种，由[[阳离子|阳离子]]和多元[[酸|酸]]的不完全电离酸根[[阴离子|阴离子]]组成。由于阴离子中含有在水中可电离的[[氢|氢]]原子，因此被称作“酸式”盐<ref name="魏明通2006"/>。但实际上，只有强酸（如[[硫酸|硫酸]]）及少部分中强酸（如[[磷酸|磷酸]]）的酸式盐呈[[酸性|酸性]]，大多数弱酸的酸式盐都因阴离子的[[水解|水解]]而显[[碱性|碱性]]。

要注意的是，酸式盐在以[[离子晶体|离子晶体]]形式存在时，阴离子并不电离出氢离子，氢离子是酸式酸根离子的一部分。在[[熔融|熔融]]状态下，酸根离子也不电离。

*[[碱式盐|碱式盐]]：

碱式盐是盐类的一种，除了含有金属离子和酸根离子外，还含有氢氧根或氧離子的盐类；也可认为是碱的氢氧根没有被酸完全中和所得的产物。
含羥基的鹼式盐又稱為羥基盐，可視為金屬離子、氫氧根及阴離子合成的鹽類，如碱式碳酸銅（<chem>Cu2(OH)2CO3</chem>）、碱式氯化镁（<chem>Mg(OH)Cl</chem>）等為羥基盐。

含氧基的鹼式盐稱為氧化鹽，可視為金屬離子、氧離子（O<sup>2-</sup>）及阴離子合成的鹽類。如鹼式碳酸鉍（<chem>Bi2O2(CO3)</chem>）即為氧化盐。

=== 合盐 ===
*複盐：
{{main|複盐}}
复盐是指含有两种或以上的同种晶型的简单盐类<ref name="魏明通2006"/>，属于化合物，溶于水会离解出所有的离子。复盐通常可由混合这两种盐饱和溶液并结晶而制得。在自然界中廣泛存在，如[[明礬|明礬]]就是天然而有廣泛應用的複鹽。

== 属性 ==
=== 颜色 ===
盐的颜色可以是纯洁透明的（如[[氯化钠|氯化钠]]）、不透明的或者是带有金属光泽的（如[[黄铁矿|黄铁矿]]）。大多数情况下盐表面的透明或不透明只和构成该盐的[[单晶材料|单晶体]]有关。当光线照射到盐上时，就会被晶界（晶体之间的边界）反射回来，大的[[晶体|晶体]]就会呈现出透明状，多晶体聚集在一起则会看起来更像白色粉末一样。

盐有许多颜色，例如：
* 黄色（例如[[铬酸钠|铬酸钠]]）
* 橙色（例如[[重铬酸钾|重铬酸钾]]）
* 红色（例如[[铁氰化钾|铁氰化钾]]）
* 淡紫色（例如[[氯化钴|氯化钴]]）
* 蓝色（例如[[硫酸铜|硫酸铜]]、[[亚铁氰化铁|亚铁氰化铁]]）
* 绿色（例如[[氯化镍|六水合氯化镍]]）
* 无色（例如[[氯化钠|氯化钠]]、[[硫酸镁|硫酸镁]]），粉末状时为白色
大部分矿物质、无机[[色素|色素]]以及很多人工合成的有机[[染料|染料]]都是盐，有一些盐能够呈现出其它颜色是过渡元素的d轨道存在未成对电子导致的。

=== 气味 ===
强酸或强碱盐（强盐）是不可挥发的，且没有气味。而弱酸或弱碱盐（弱盐）则会根据[[共轭酸碱对|共轭酸碱对]]而产生不同的气味（例如[[醋酸盐|醋酸盐]]则会有[[醋酸|醋酸]]的味道，[[氰化氢|氰化氢]]则会有苦杏仁味等），此外弱酸或弱碱盐还会挥发和分解，并且由于水解反应和弱盐合成的反应是可逆反应，因此当有水分存在时会加速弱盐的分解。

===溶解度===
<!--{{see also|Solubility#Solubility of ionic compounds in water}}-->
{{main|溶解度}}
許多離子化合物都可以[[溶解|溶解]]在水或是類似的溶劑中，化合物的溶解度和組成的陰離子和陽離子以及溶劑都有關，因此有一特定的關係。陽離子當中，所有[[鈉|鈉]]離子、[[鉀|鉀]]離子及[[銨|銨]]離子的鹽均可溶於水；而陰離子中，[[硝酸鹽|硝酸鹽]]離子及[[碳酸氫鹽|碳酸氫鹽]]離子的鹽類可在水中溶解。除了[[硫酸鋇|硫酸鋇]]、[[硫酸鈣|硫酸鈣]]及[[硫酸鉛|硫酸鉛]]外，大部份的[[硫酸鹽|硫酸鹽]]離子的鹽類也都可以溶於水。不過若離子之間的鍵結很強，產生高度穩定的固體，在水中也就不易溶解，例如大部份的[[碳酸鹽|碳酸鹽]]就不溶於水，例如[[碳酸鉛|碳酸鉛]]及[[碳酸鋇|碳酸鋇]]等。
盐的溶解性口诀：
钾钠铵盐都易溶，
硝酸盐遇水无影踪，
硫酸盐不溶铅和钡，
盐酸盐不溶银亚汞，
碳酸盐除钾钠铵外都不溶。

== 系统命名法 ==
一般而言，无氧酸盐的名字是非金属元素名称在前，金属元素名称在后，两者名字之间添加“化”，称为“某化某”。如果是含氧酸盐，则盐的名称为酸的名称后面之间添加元素名称，称为“某酸某”。如果某种金属元素有多个化合价，那么低化合价形成的盐的名称是在金属元素名称前添加“亚”，例如铜元素的氯化物中，如果铜的化合价为+2价，那么形成的盐CuCl<sub>2</sub>称为“氯化铜”，如果化合价为+1价，形成的盐CuCl称为“氯化亚铜”。

含有相同阳离子或阴离子的一类盐一般都会统称为“某盐”，例如“钠盐”、“硫酸盐”等。

常见的形成盐类的阳离子包括：
* [[铵根|铵根]]NH<sub>4</sub><sup>+</sup>
* [[钙|钙离子]]Ca<sup>2+</sup>
* [[亞铁離子|亞铁離子]]Fe<sup>2+</sup>
* [[鐵|鐵離子]]Fe<sup>3+</sup>
* [[镁|镁离子]]Mg<sup>2+</sup>
* [[钾|钾离子]]K<sup>+</sup>
* [[吡啶翁|吡啶翁]]C<sub>5</sub>H<sub>5</sub>NH<sup>+</sup>
* [[季铵阳离子|季铵阳离子]]NR<sub>4</sub><sup>+</sup>
* [[钠|钠离子]]Na<sup>+</sup>

常见的形成盐类的阴离子包括：
* [[乙酸盐|乙酸根]]CH<sub>3</sub>COO<sup>−</sup>（[[乙酸|乙酸]]）
* [[碳酸盐|碳酸根]]CO<sub>3</sub><sup>2−</sup>（[[碳酸|碳酸]]）
* [[氯|氯離子]]Cl<sup>−</sup>（[[盐酸|盐酸]]）
* [[氰化物|氰化物]]C≡N<sup>−</sup>（[[氰化氢|氰化氢]]）
* [[硝酸盐|硝酸根]]NO<sub>3</sub><sup>−</sup>（[[硝酸|硝酸]]）
* [[亚硝酸盐|亚硝酸根]]NO<sub>2</sub><sup>−</sup>（[[亚硝酸|亚硝酸]]）
* [[磷酸盐|磷酸根]]PO<sub>4</sub><sup>3−</sup>（[[磷酸|磷酸]]）
* [[硫酸盐|硫酸根]]SO<sub>4</sub><sup>2−</sup>（[[硫酸|硫酸]]）
* [[硼酸鹽|硼酸根]]BO<sub>3</sub><sup>3-</sup> （[[硼酸|硼酸]]）

== 有機離子的鹽類 ==
一般鹽類會歸類為[[無機化合物|無機化合物]]，但也有一些[[有機化合物|有機化合物]]的鹽類，例如[[有機酸|有機酸]]的鹽類就是有機化合物。其中較重要的是[[羧酸|羧酸]]鹽，例如[[乙酸|乙酸]]根CH<sub>3</sub>COO<sup>−</sup>和鈉離子Na<sup>+</sup>會形成[[乙酸鈉|乙酸鈉]]，和銅離子Cu<sup>2+</sup>會形成[[乙酸銅|乙酸銅]]。乙酸只有一個COOH基，只帶-1價，[[柠檬酸|柠檬酸]]有三個COOH基，帶-3價，形成的鹽類有[[柠檬酸鈉|柠檬酸鈉]]和[[柠檬酸鈣|柠檬酸鈣]]根，乙酸鹽和柠檬酸鹽的陰離子和陽離子都以[[離子鍵|離子鍵]]鍵結，不像一般有機化合物以間用[[共價鍵|共價鍵]]鍵結，因此稱為鹽類。

這些鹽類中較常用的是[[羧酸|羧酸]]鹽，其中也包括[[脂肪酸|脂肪酸]]鹽。像[[肥皂|肥皂]]就是由許多不同分子量脂肪酸鈉及脂肪酸鉀的混合物。這類長鏈脂肪酸的鹽類就沒有晶體的結構。

類似硫酸根SO<sub>4</sub><sup>2−</sup>的[[磺酸|磺酸]]根<chem>R-O-{SO3}^{-}</chem>也是有機酸根的一種，磺酸鹽是洗髮精及沐浴乳中常見的[[表面活性剂|表面活性剂]]。醇類的酸性很弱，一般不會稱為酸，不過其羟基氢被金属取代后會形成[[醇盐|醇盐]]。醇盐在水中會水解，產生對應的醇類，以下是醇盐和金屬[[鹼酐|鹼酐]]水解反應的比較。

也有些有機的陽離子會形成鹽類，例如類似[[銨根|銨根]]離子（NH<sub>4</sub><sup>+</sup>）的[[季铵阳离子|季铵阳离子]]，季铵阳离子是由一個氮原子和四個[[烷基|烷基]]（<chem>{R}{-}</chem>）組成，帶一個正電荷。例如[[溴化十六烷基三甲铵|溴化十六烷基三甲铵]]中，十六烷基三甲铵離子是陽離子，和陰離子溴離子形成鹽類。實用的季铵鹽類也有類似的結構，季铵阳离子中只有一個長鏈的烷基，當溶於水中，陽離子有[[表面活性劑|表面活性劑]]的作用。這類物質在生物體新陳代謝中有重要的作用，例如維生素B中的[[胆碱|胆碱]]就是季铵鹽類。

在氨分子（NH<sub>3</sub>）中加入一個氫離子（H<sup>+</sup>）就變成銨離子（NH<sub>4</sub><sup>+</sup>），同樣的在有機[[胺|胺]]中加入一個氫離子就變成陽離子。例如一級胺（<chem>R-NH2H</chem>，R為烷基）加入一個氫離子就變成陽離子（<chem>R-{NH3}^{+}</chem>）這類的化合物比原來的化合物極性要強，因此在水中的溶解度比原來的化合物要高。例如一些含氮的[[藥物|藥物]]會和[[鹽酸|鹽酸]]反應來產生鹽酸胺類化合物，人體對於這類化合物的吸收度比原來的含氮藥物要好。也有些藥物是和[[溴化氫|溴化氫]]或[[氟化氫|氟化氫]]反應成類似的藥物。

[[內鹽|內鹽]]也稱為兩性離子，是一種特殊的化合物，化合物中同時有帶正電及帶負電的原子，但是帶正電和負電的原子不同（在一些定義中限定是未具有相鄰正負電荷的化合物）<ref>{{GoldBookRef | file = Z06752 | title = zwitterionic compounds/zwitterions}}</ref>，像許多[[胺基酸|胺基酸]]都屬於內鹽。若胺基酸中包含羧基（<chem>{-COO}-</chem>）和一個胺基（<chem>-{NH3}^{+}</chem>），就是內鹽，會在不同的反應中呈酸性或是鹼性。不過內鹽和一般電解質不同，溶於水中只有微弱的導電性（[[兩性_(化學)|兩性物質]]）。

== 製備 ==
盐可以通过[[化学反应|化学反应]]而製備，包括有：
* [[盐基|盐基]]与[[酸|酸]]，例如[[氨|NH<sub>3</sub>]] + [[氯化氢|HCl]] → [[氯化铵|NH<sub>4</sub>Cl]]
* [[金属|金属]]与[[酸|酸]]，例如[[镁|Mg]] + [[硫酸|H<sub>2</sub>SO<sub>4</sub>]] → [[硫酸镁|MgSO<sub>4</sub>]] + [[氢气|H<sub>2</sub>]]
* 金属与非金属，例如[[钙|Ca]] + [[氯气|Cl<sub>2</sub>]] → [[氯化钙|CaCl<sub>2</sub>]]
* [[碱|碱]]与[[氧化物|酸性氧化物]]，例如2 [[氢氧化钠|NaOH]] + [[一氧化二氯|Cl<sub>2</sub>O]] → 2 [[次氯酸钠|NaClO]] + [[水|H<sub>2</sub>O]]
* [[酸|酸]]与[[氧化物|碱性氧化物]]，例如2 [[硝酸|HNO<sub>3</sub>]] + [[氧化钠|Na<sub>2</sub>O]] → 2 [[硝酸钠|NaNO<sub>3</sub>]] + [[水|H<sub>2</sub>O]]
* 盐也可以由不同的盐溶液混合，其中的离子发生重组，形成溶解度较低的盐（另见：[[溶解平衡|溶解平衡]]），例如<chem>Pb(NO3)2 + Na2SO4 -> PbSO4 + 2NaNO3</chem>

== 盐的性质 ==
在化學上，鹽是由陽離子（正電荷離子）與陰離子（負電荷離子）所組成之中性（不帶電荷）[[離子化合物|離子化合物]]。

*和[[酸|酸]]发生反应。[[复分解反应|复分解反应]]。
:酸+盐→新盐+新酸（强酸→弱酸）这里的盐可以是不溶性盐。
::<chem>2HCl + Na2CO3 -> H2O + CO2 ^ + 2NaCl</chem>
::碳酸不稳定会继续分解成[[水|水]]和[[二氧化碳|二氧化碳]]。
*和[[碱|碱]]发生反应。[[复分解反应|复分解反应]]。
:碱（可溶）＋盐（可溶）→新碱＋新盐
::<chem>2NaOH + CuSO4 -> Cu(OH)2 v + Na2SO4</chem>

*和盐发生反应。[[复分解反应|复分解反应]]。
:盐（可溶）＋盐（可溶）→两种新盐
::<chem>CuSO4 + BaCl2 -> BaSO4 v + CuCl2</chem>

*和某些金属反应。[[置换反应|置换反应]]
:盐＋金属（某些）→新金属＋新盐 反应中的金属一定要比盐中的金属活泼才可以把它给置换出来。请看[[金属活动性|金属活动性]]。
::<chem>Zn + CuSO4 -> ZnSO4 + Cu</chem>
