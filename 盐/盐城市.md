'''盐城市'''是[[中华人民共和国|中华人民共和国]][[江苏省|江苏省]]下辖的[[地级市|地级市]]，位于江苏省中部，东濒[[黄海|黄海]]，是江苏沿海城市之一。全市总面积1.69万平方公里，是江苏省面积最大的地级市。其中市辖区面积5,131平方公里，市辖区常住人口237.25万。海岸线长度为江苏省各地级市首位。

全市地势平坦，河渠纵横，交通发达，物产富饶。盐城大多数人口是历朝江南移民后裔，因而传统文化和民众的心理状态都属于传统的江南文化范畴。盐阜大地人杰地灵，名人数不胜数，如[[南宋|南宋]]丞相[[陆秀夫|陆秀夫]]，《[[水浒传|水浒传]]》作者[[施耐庵|施耐庵]]等。

== 历史 ==
盐城所在地域，[[周朝|周]]以前为淮夷地；周时属[[青州_(古代)|青州]]，[[春秋|春秋]]时属[[吴国|吴]]，后属[[越国|越]]。[[战国|战国]]时属楚。[[秦代|秦代]]属[[东海郡|东海郡]]；[[西汉|西汉]]初为[[射阳侯|射阳侯]][[刘缠|刘缠]]封地。西汉时期，盐城尚为位于海岸线上的海港，以煎炼海水得盐为主要产业，从而得名。

[[汉武帝|汉武帝]][[元狩|元狩]]四年（前119年），建[[盐渎县|盐渎县]]，这是盐城置县的开端，[[孙坚|孙坚]]曾任盐渎[[县丞|县丞]]。[[三国|三国]]时属[[魏|魏]]，废县制；[[西晋|西晋]]又复县制；[[东晋|东晋]][[义熙_(东晋)|义熙]]七年（411年）改名盐城，此为现名之始；[[南北朝|南北朝]]称[[盐城郡|盐城郡]]；[[隋朝|隋]]初仍为县，属[[江都郡|江都郡]]；隋末，[[韦彻|韦彻]]据盐称王，分为[[新安县|新安]]、[[安乐县|安乐]]两县，[[唐朝|唐]]初复置盐城县；[[宋代|宋代]]属[[楚州_(古代)|楚州]]；[[元朝|元朝]]属[[淮安路|淮安路]]；[[明代|明代]]属[[南直隶|南直隶]][[淮安府|淮安府]]；[[清朝|清]]初属[[江南省|江南省]][[淮安府|淮安府]]，[[康熙|康熙]]六年（1667年），江苏、安徽分省后属于[[江苏省|江苏省]]淮安府。[[民国|民国]]初，盐城县隶属江苏省[[淮扬道|淮扬道]]。

国民政府时期，盐城县为江苏省第六[[行政督察区|行政督察区]]首县，下辖[[东台县|东台县]]，[[兴化县|兴化县]]，[[阜宁县|阜宁县]]。抗日战争后期，[[新四军|新四军]]渡江北上，军部设于盐城，控制附近盐城、阜宁一带，析东台县北部设[[大丰县|台北县]]、析盐城县东部设[[盐东县|盐东县]]、析阜宁县东部设[[阜东县|阜东县]]、析盐城县北部设[[建阳县|建阳县]]4县。1946年盐城县改名为[[叶挺县|叶挺县]]，城区设[[叶挺市|叶挺市]]，1949年撤销叶挺市，叶挺县建制，恢复盐城县原名。另外台北县因与台湾省台北县重名，改名大丰县；建阳县与福建省建阳县重名，改名建湖县；盐东县与阜东县调整为射阳县与滨海县。

盐城地区其它县的设置：[[阜宁县|阜宁县]]系清朝[[雍正|雍正]]九年（1731年）析自淮安府山阳县；[[东台县|东台县]]系[[乾隆|乾隆]]三十三年（1768年）析自[[扬州府|扬州府]]泰州；1966年新建[[响水县|响水县]]。1983年撤销盐城县，设立盐城市，下辖[[城区_(盐城市)|城区]]、[[郊区_(盐城市)|郊区]]2区和[[响水县|响水]]、[[滨海县|滨海]]、[[阜宁县|阜宁]]、[[射阳县|射阳]]、[[建湖县|建湖]]、[[大丰县|大丰]]、[[东台县|东台]]7县。1987年，东台设县级市、1996年，大丰设县级市，1996年，撤销郊区，建盐都县。2004年城区更名为[[亭湖区|亭湖区]]，撤销[[盐都县|盐都县]]，设立[[盐都区|盐都区]]，原盐都县伍佑镇，步凤镇，便仓镇划归亭湖区。2015年7月，大丰撤市设区，设立盐城市大丰区。<ref>{{cite web |url=http://202.108.98.30/description?dcpid=2015|title=中华人民共和国二〇一五年县级以上行政区划变更情况}}</ref>

== 地理 ==
[[File:New_Fourth_Army_Memorial.jpg|thumb]]
盐城地处江苏省中东部的[[滨海平原|滨海平原]]和[[里下河平原|里下河平原]]，东临[[黄海|黄海]]，具体位于北纬32.85°～34.2°、东经119.57°～120.45°，南与南通市接壤，西南与扬州市、泰州市为邻，西与淮安市相连，北隔灌河和连云港市相望，属北[[亚热带季风气候|亚热带季风气候]]。年平均气温13.7℃-14.4℃，年平均无霜期209-218天，年平均降水量 900 毫米 ～ 1066 毫米<ref name="YanChengWeather">[http://www.ycqxw.cn/index-Detail.asp?ChannelID=5&ClassID=1&ArticleID=1 盐城市气象局简介]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}，盐城市气象局</ref>。以“东方[[湿地|湿地]]、水绿盐城”而闻名。

{{盐城市气候数据}}

盐城是江苏省面积最大的[[省辖市|省辖市]]。全市总面积1.69万平方公里，其中耕地面积77.82万公顷，河湖水面1300多平方公里。海洋和滩涂资源十分丰富。沿海滩涂总面积4550平方公里（含辐射沙洲），其中潮上带1677平方公里，潮间带1610平方公里，分别占全省的75%、64.6%、60.8%。隶属于[[东台|东台]]、[[大丰|大丰]]、[[射阳|射阳]]、[[滨海|滨海]]、[[响水|响水]]等县（市）的沿海滩涂，近期可供开发利用的面积达1300平方公里。目前，[[射阳河|射阳河]]口以南沿海地段还以每年10多平方公里的速度向大海延伸<ref name="YanChengGeo">[http://www.yancheng.gov.cn/ycweb/TemplateA/OverView/InfoList.aspx?ParentCategoryID=bf2e35b4-2939-4cf4-86cf-73049e6c991a&CategoryID=bf2e35b4-2939-4cf4-86cf-73049e6c991a 盐城市情概况] {{Wayback|url=http://www.yancheng.gov.cn/ycweb/TemplateA/OverView/InfoList.aspx?ParentCategoryID=bf2e35b4-2939-4cf4-86cf-73049e6c991a&CategoryID=bf2e35b4-2939-4cf4-86cf-73049e6c991a |date=20100302162212 }}，盐城政府网</ref>。

[[石油|石油]][[天然气|天然气]]资源蕴藏较多。已探明石油天然气蕴藏量达800亿立方米，预计总储量达2000亿立方米，为中国东部沿海地区陆上最大的油气田。沿海和近海有约10万平方公里的[[黄海|黄海]]储油沉积盆地，居全国海洋油气沉积盆地第2位。

位于盐城市境内的[[大丰麋鹿国家级自然保护区|大丰麋鹿]][[国家级自然保护区|国家级自然保护区]]是目前世界上占地面积最大、麋鹿种群数量最多、并建立了第一个麋鹿基因库的自然保护区。同样位于盐城境内的盐城国家级珍禽自然保护区为中国最大的海岸带保护区，1992年11月被[[联合国教科文组织|联合国教科文组织]][[人与生物圈协调理事会|人与生物圈协调理事会]]批准为生物圈保护区，并纳入“[[世界生物圈保护区网络|世界生物圈保护区网络]]”。1996年又被纳入“[[东北亚鹤类保护网络|东北亚鹤类保护网络]]”，1999年成为“[[东亚—澳大利亚涉禽迁徙网|东亚—澳大利亚涉禽迁徙网]]络”成员，2002年1月被列入[[湿地公约|Ramsar]]国际重要湿地名录，保护区是挽救一些濒危物种的最关键地区，如[[丹顶鹤|丹顶鹤]]、[[黑嘴鸥|黑嘴鸥]]、[[獐|獐]]、[[震旦鸦雀|震旦鸦雀]]等。每年来区越冬[[丹顶鹤|丹顶鹤]]达到千余只，占世界野生种群50%以上，是全球最大的[[丹顶鹤|丹顶鹤]]越冬地。

根据2010年[[第六次全国人口普查|第六次全国人口普查]]，全市[[常住人口|常住人口]]为7260240人<ref>{{cite book |author=盐城市统计局|title=《盐城市2010年第六次全国人口普查主要数据公报》}}</ref>，同[[第五次全国人口普查|第五次全国人口普查]]相比，十年共减少832001人，减少10.28%。年平均减少率1.08%。其中，男性人口为3650255人，占50.28%；女性人口为3609985人，占49.72%。总人口性别比（以女性为100）为101.12。0－14岁人口为1042107人，占14.35%；15－64岁人口为5349041人，占73.68%；65岁及以上人口为869092人，占11.97%。

== 经济 ==
产稻、麦、棉等。盐城为中国棉花与淡水鱼重要产区，是江苏省新兴桑蚕基地。

工业有轻纺、电子、机械、化工、食品、建材等，其中“燕舞牌”收录机曾因广告有特点而全国闻名。现在有[[悦达集团|悦达集团]]、[[中大集团|中大集团]]、[[江动集团|江动集团]]等一系列企业成为经济新的增长点，江苏省惟一的乘用车企业“东风悦达起亚汽车有限公司”就落户在盐城。在清洁能源方面，盐城拥有国内最大海上风电机组<ref name="YanChengEng">[http://city.cctv.com/html/yancheng/ead1b822f5f823dd6bf1762808170a11.html 国内最大海上风电机组盐城下线]{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}，CCTV城市</ref>。

[[中国百强城市|2003年全国(大陆)综合实力百强城市]]中排名第83名，2004年[[2004年中国大陆城市竞争力排名|上升至第75名]]，2008年[[2008年中国大陆城市竞争力排名|上升至第53名]]。2013年[[2013年中国大陆城市GDP排名|上升至第42名]]2013年盐城市实现地区生产总值达3475.50亿元，人均地区生产总值达48161.80元（按2013年年均汇率折算为7776.56美元），高于全国平均水平。

== 交通 ==
[[File:Yancheng_Railway_Station.jpg|thumb]][[盐城站|盐城站]]]]
盐城的交通事业最近几年发展迅速。盐城为江苏东部南北交通要冲，高速公路通达各县区，盐城也是江苏省重要的水网地区。

* [[高速公路|高速公路]]：全市高速公路总里程359.08公里。[[沈海高速公路|沈海高速公路]]（编号：G15）（原称“[[沿海高速公路|沿海高速公路]]”）、盐靖高速公路（编号：S29）（原称“[[宁靖盐高速公路|宁靖盐高速公路]]”）、盐淮高速公路（编号：S18）（原称“[[宿淮盐高速公路|宿淮盐高速公路]]”或“[[盐徐高速公路|盐徐高速公路]]”)以上三条高速公路已经建成合围盐城城区。另外还有[[江苏临海高等级公路|江苏临海高等级公路]]，建湖—阜宁高速公路。
* [[铁路|铁路]]：[[新长铁路|新长铁路]]、[[青盐铁路|青盐铁路]]已建成通车，可到达北京，南京，徐州，郑州，西安等交通枢纽。另外有[[徐宿淮盐铁路|徐宿淮盐铁路]]、[[盐通高速铁路|盐通高速铁路]]两条高速铁路在建，建成后，1-2小时可到达上海，南京等大城市，4小时可到达北京。另外有盐泰锡常宜城际铁路在规划中。
* [[民航|民航]]：盐城[[南洋机场|南洋机场]]为国家一类航空开放口岸，已开通盐城至[[香港|香港]]、[[韩国|韩国]][[首尔|首尔]]、[[台湾|台湾]]等国际及地區班机，并将陆续开通至[[东南亚|东南亚]]国家和[[日本|日本]]的航班。
* [[港口|港口]]：[[盐城港|盐城港]]下辖四个港区：[[大丰港区|大丰港区]]、[[滨海港区|滨海港区]]、[[射阳港区|射阳港区]]、[[陈家港区|陈家港区]]。其中大丰港区是国家一类开放口岸，盐城港的主港区。
* [[航道|航道]]：全市共有航道4504公里，占全省航道总里程的五分之一，是水运大市。
* [[巴士快速交通系统|巴士快速交通系统]](BRT)：于2010年5月1日正式通车 为全国第17条，江苏第2条快速公交线路。全线目前有9条线路：快速公交一号线(B1)，快速公交二号线(B2)， 快速公交支线1(B支1)， 快速公交支线2(B支2)，快速公交支线3(B支3)，快速公交支线4(B支4)，快速公交支线5(B支5)，快速公交支线6(B支6，快速公交支线7(B支7))。

== 文化 ==
[[File:Lu_Xiufu_Temple.jpg|thumb]]
盐城出现过文雄海内的"[[建安七子|建安七子]]"之一[[陈琳_(三国)|陈琳]]、[[南北朝|南北朝]]名医[[徐道度|徐道度]]、[[清|清]]代治水专家[[冯道立|冯道立]]、明清诗人[[吴嘉纪|吴嘉纪]]、[[孙玉澍|孙玉澍]]、书法家[[宋曹|宋曹]]、画家[[万岚|万岚]]、[[评书|评书]]的开创者[[柳敬亭|柳敬亭]]等名人。

建湖的"[[百戏|百戏]]"在[[唐代|唐代]]即享有盛誉，庆丰十八团是中国三个半[[杂技|杂技]]发源地之一（另两个是[[山东|山东]][[聊城|聊城]]、[[河北|河北]][[吴桥|吴桥]]；半个是[[北京|北京]][[天桥|天桥]]）。[[明|明]]代中叶以[[东台|东台]][[安丰|安丰]]人[[王艮|王艮]]为代表的[[泰州学派|泰州学派]]在全国颇有影响。四大名著之一的《[[水浒传|水浒传]]》的作者[[施耐庵|施耐庵]]为盐城[[大丰|大丰]]白驹人。

盐城是[[淮剧|淮剧]]的主要发源地和流行地区之一。

盐城有始建于[[唐|唐]]代的[[永宁禅寺|永宁禅寺]]、[[南宋|南宋]]丞相[[陆秀夫|陆秀夫]]祠堂、[[明|明]]末著名书法家[[宋曹|宋曹]]故居、《[[水浒传|水浒传]]》作者[[施耐庵|施耐庵]]和现代名人[[胡乔木|胡乔木]]、[[乔冠华|乔冠华]]的故里。

== 名人 ==
* [[陈琳|陈琳]]，[[汉|汉]]朝末年著名文学家、“[[建安七子|建安七子]]”之一，字孔璋，原盐渎县人。
* [[陆秀夫|陆秀夫]]，[[南宋|南宋]]末年著名民族英雄，字君实，祖籍[[建湖|建湖]]建阳。[[祥兴|祥兴]]元年（1278年）为左丞相，驻军[[崖山|崖山]](今[[广东省|广东省]][[新会县|新会县]]境内）抵抗入侵的元军。次年元军攻破[[崖山|崖山]]，陆秀夫坚贞不屈、宁死不降，先驱妻、子入海，尔后背负皇帝[[赵昺|赵昺]]从容投海而死。
* [[王艮|王艮]]，[[明|明]][[成化|成化]]十九年（1483）生于今[[东台|东台]][[安丰|安丰]]镇，是[[明|明]]代著名的思想家和教育家，[[王阳明|王阳明]]最著名的学生，[[泰州学派|泰州学派]]的创始人。
* [[施耐庵|施耐庵]]，元末明初小说家。祖籍[[泰州|泰州]]，随父[[施元德|施元德]]迁居[[兴化|兴化]]，落户[[白驹场|白驹场]]（今[[大丰区|大丰区]]境内），在此写下名著《[[水浒传|水浒传]]》。
* [[戈公振|戈公振]]，新闻学家，1890年生于[[东台|东台]]县。他编写了中国第一部论述新闻史的著作《中国报学史》，考察过欧美国家的新闻事业，编译了《新闻学撮要》。
* [[乔冠华|乔冠华]]，中国当代著名的外交家。1913年3月8日，生于今盐城市建湖县庆丰乡。乔冠华历任[[中华人民共和国|中华人民共和国]][[新闻总署|新闻总署]]国际新闻局局长，[[外交部|外交部]]政策研究室副主任（主任由[[周恩来|周恩来]]兼任），[[外交部|外交部]][[亚洲司|亚洲司]]司长，[[外交部|外交部]]副部长、部长等职。1971年10月，[[联合国|联合国]]大会通过决议恢复中国在[[联合国|联合国]]的合法席位后，乔冠华为出席第二十六届联大中国代表团团长，11月15日，他代表[[中华人民共和国|中华人民共和国]]政府在联合国大会上首次作了发言。
* [[胡乔木|胡乔木]]，出生于[[盐都区|盐都区]][[鞍湖乡|鞍湖乡]]张本庄。从[[1941年2月|1941年2月]]起，任[[毛泽东|毛泽东]]的秘书、[[中共中央政治局|中共中央政治局]]秘书，一直继续到1966年6月"[[文化大革命|文化大革命]]"爆发以前。他亲自或参与起草和编纂了许多[[中共中央|中共中央]]的重要文件。胡乔木是中共第八届、十一届、十二届[[中央委员会|中央委员会]]委员，第二届、第三届、第五届[[全国人大常委会|全国人大常委会]]委员，第一届[[中国人民政治协商会议|中国人民政治协商会议]]全国委员会常务委员，[[中顾委|中顾委]]常务委员，[[中共中央党史工作领导小组|中共中央党史工作领导小组]]副组长，[[中国社会科学院|中国社会科学院]]名誉院长。
