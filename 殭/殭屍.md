-{zh-hans:'''僵尸'''; zh-hant:'''殭屍''';}-，亦称'''跳屍'''，是源於[[中國|中國]][[明代|明]][[清代|清]][[民間傳說|民間傳說]]的一種復活[[死屍|死屍]]，全身僵硬，指甲发黑尖锐，有锐利犬齿，惧[[阳光|阳光]]，日間躲於[[棺材|棺材]]、[[山洞|山洞]]等[[潮湿|潮湿]]阴暗的地方，入夜后出没，以吸食活人或家畜[[血液|血液]]保持行动力，有意识的往人口集中地前进，对活物攻击性强且力大无穷，跳跃前进時雙手向前伸。

據南袁北紀《[[子不語|子不語]]》及《[[閱微草堂筆記|閱微草堂筆記]]》所記載，殭屍還有三個別名：移屍、走影、走屍。與中國殭屍相似存在的外國鬼怪，有[[吸血鬼|吸血鬼]]、[[喪屍|喪屍]]等。

在[[香港電影|香港電影]]中，最初的殭屍都是身穿[[清朝官员服饰|清朝官员服饰]]，因从[[月亮|月亮]]吸收[[陰|陰]][[氣|氣]]，死不瞑目，殭屍因積攢怨氣、人為破壞或[[墓地|墓地]][[風水|風水]]屬性屍變而形成。以爪為武器，被抓傷者會感染屍毒死亡，繼而變成另一隻殭屍；於月圓之夜，陰氣增強的時候，力量亦會隨之而增強。中國殭屍電影的始祖是[[林正英|林正英]]，由他出演的殭屍電影超過25部。

== 起源 ==
=== 湘西趕屍之說 ===
中國傳說[[上古時代|上古時代]][[蚩尤|蚩尤]]與[[黃帝|黃帝]]交戰，廝殺慘烈。蚩尤爲了運回戰死兵士的遺體，便命手下軍師設法運屍。相傳趕屍便由此而來。<ref>{{cite episode | title=《行走的屍體》 | series=《[[走近科學|走近科學]]》 | network=[[中央電視臺|中央電視臺]] | airdate=2006年7月17日}}</ref>

{{main|趕屍}}
殭屍的傳說在[[湘西土家族苗族自治州|湘西]]出現最頻繁，可能源於當地[[湘西趕屍|湘西趕屍]]的傳說。[[湘西趕屍|湘西趕屍]]，又稱[[移靈|移靈]]，屬[[茅山術|茅山術]][[祝由科|祝由科]]，發源於[[湘西土家族苗族自治州|湘西]][[沅陵|沅陵]]、[[泸溪|泸溪]]、[[辰溪|辰溪]]、[[溆浦|溆浦]]四縣，在屍體未腐化時由術士趕回鄉安葬。趕屍的術士大約三五同行，有的用繩繫著屍體，每隔幾尺一個，然後額上貼[[黃紙符|黃紙符]]，另外的便打鑼響鈴開路，晝伏夜行。天光前投棧，揭起[[符紙|符紙]]，屍靠牆而立，到夜間繼續上路。
。

== 殭屍成因 ==
=== 小說描述殭屍成因 ===
《[[閱微草堂筆記|閱微草堂筆記]]》使屍體變成殭屍（或稱屍變）的原因分成兩項：'''新屍突變'''及'''葬久不腐'''，可能的構成原因如下：
# 養屍地，這較為[[科學|科學]]。土壤土質酸鹼度極不平衡，不適合[[有機物|有機物]]生長，因此不會滋生蟻蟲細菌，[[屍體|屍體]]埋入即使過百年，[[肌肉|肌肉]]毛髮也不會腐壞，有些資料顯示屍體的毛髮，[[指甲|指甲]]會繼續生長，實際上是因為死後人體的肌肉收縮，致使原本藏在肉中的毛髮和指甲部份顯露出來。這種屍體稱作[[蔭屍|蔭屍]]，算是屍體和殭屍之間的過渡期形態，如果不趕快處理很快便會化為殭屍。[[風水|風水]]學中亦有此一說。此種情況在外國一部份地方的[[吸血鬼|吸血鬼]]判斷方式相似。
# 坊間流傳[[道教|道教]]有太陰煉形之法，屍體葬數百年，期滿便會復生。
# 新死的屍體被[[邪物|邪物]]／[[邪氣|邪氣]]附身。
# 屍體吸收了[[陽氣|陽氣]]，借人生氣而屍變。
# 人死之際，魂一散而魄滯。根據袁枚《[[子不語|子不語]]》：「人之魂善而魄惡，人之魂靈而魄愚」[[魄|三魂七魄]]主宰人身，當魂離開人體，便會淪為惡鬼殭屍。
# 人死後殯而不葬。1950年代香港人仍有家中殮葬和停屍，所以當時社會屍變之說大行其道，如行雷[[閃電|閃電]]，大肚貓跳過[[棺材|棺材]]，屍體便會出現異變（稱為尸变）。

== 殭屍的種類與特性 ==
傳說殭屍會進食死後未腐的屍體，並吸食其血液

殭屍的毛髮常常作為特徵出現在袁枚的筆記中，《子不語》第九卷的《掘塚奇報》提及了幾種死後未腐的屍體「[[紫殭|紫殭]]、[[白殭|白殭]]、[[綠殭|綠殭]]、[[毛殭|毛殭]]之類」。《秦中墓道》則又提到因毛色黑白，殭屍又可稱為[[黑凶|黑凶]]或[[白凶|白凶]]。《屍奔》一文甚至提到一隻自稱[[黃小二|黃小二]]的生有五色毛的殭屍。

此外《骷髅三种》中也提到類似僵屍但僅存骨骸的三種作祟之屍體。[[不化骨|不化骨]]：人死後身體某些部位因為[[精神|精神]]灌注而使其部位屍骨不化，所謂精神灌注就比如扛米工人因為一直用肩膀出力，他死了之後可能其肩膀附近就都不會腐化，而肩膀不化的部位就叫做不化骨，而不化骨若是之後得了日月[[精氣|精氣]]就會作祟。[[伏屍|伏屍]]：千年不朽的屍體，無法移動，可以說是完整全屍的不化骨，久了之後得日月精華就會逐漸化為游屍。而《岣嶁神書》云：「老蛤能辟伏屍。」[[遊屍|遊屍]]：會隨著月氣因時節移動，居無定所，久了之後更會化為[[飛行夜叉|飛行夜叉]]。<ref>[http://nccur.lib.nccu.edu.tw/bitstream/140.119/44178/1/14.pdf 《子不語》正、續二書中僵屍故事初探]，馮藝超，《東華漢學》第6期</ref>

《[[閱微草堂筆記|閱微草堂筆記]]》曾對殭屍的形貌作出描述：「白毛遍體，目赤如丹砂，指如曲勾，齒露唇外如利刃……接吻噓氣，血腥貫鼻……。」殭屍也能成[[妖|妖]]，變[[魃|魃]]（或稱[[旱魃|旱魃]]，因傳說[[黃帝|黃帝]]之女'''旱魃'''為殭屍始祖而得名）。

《[[神異經|神異經]]》載：「南方有人，長二三尺，袒身，兩目頂上，走行如風，名曰魃，所見之國大旱，赤地千里。」變魃殭屍能飛，殺龍吞雲，做成旱災。所以人們每逢[[旱災|旱災]]出現，便會四出搜索殭屍，把它們燒成灰燼。盗墓笔记中也出现过和旱魃几乎是同一级别的双头煞，需借助天雷将其杀死，原文”那双头煞浑身硬如钢铁，很可能是侵入了铁一类的元素，再加上尸煞自身的生物电，就如同一个大型的倒雷针，引来雷电也不是不可能的事情。”

[[清代|清代]][[袁枚|袁枚]]志怪小說集《[[子不語|子不語]]》《[[續子不語|續子不語]]》有多篇短篇提及各地目擊殭屍的紀錄。除了一般的殭屍作怪的紀錄外，也提到各種殭屍的變體，比如殭屍可能化成[[犼|犼]]：「常州蔣明府言：佛所騎之[[獅|獅]]、[[象|象]]，人所知也；佛所騎之犼，人所不知，犼乃殭屍所變。」殭屍屍變後，便會發出怪聲哀號，三躍三跳，化作獸形而去。故或曰：屍初變旱魃，再變即為[[犼|犼]]。犼有神通，口吐煙火，能與[[龍|龍]]鬥，故佛騎以鎮壓之。當中也提到雲南礦工遇土壓不得出，或數十年，或百年，為土金氣所養，身體不壞，化成[[乾麂子|乾麂子]]。或是提到直隶安州出現過擅长法术的[[飞殭|飞殭]]。

== 對付殭屍的方式 ==
在一些[[古藉|古藉]]和[[小說|小說]]中，都有提及到能對付殭屍的器物：
# [[鏡子|鏡子]]或[[八卦|八卦]]。《[[本草綱目|本草綱目]]》有提：「鏡乃金水之精，內明外暗。」
# [[桃枝|桃枝]]、[[桃木劍|桃木劍]]。殭屍的身體刀槍不入，唯有桃木劍可以刺穿他們。《[[荊楚歲時記|荊楚歲時記]]》：「桃者，五行之精，能厭服邪氣，制御百鬼。」
# 雞鳴。《[[子不語|子不語]]》：「鬼聞雞鳴即縮。」
# [[棗核|棗核]]七枚。《[[子不語|子不語]]》：「棗核七枚，釘入屍脊背穴。」
# [[墨斗線|墨斗線]]、[[硃砂|硃砂]]
# [[符咒|符咒]]
# [[銅鈴鐺|銅鈴鐺]]、[[銅錢|銅錢]]、[[銅錢劍|銅錢劍]]、[[銅錢鏢|銅錢鏢]]
# [[糯米|糯米]]、[[鹽米|鹽米]]、[[米篩|米篩]]、[[赤豆|赤豆]]
# [[火燒|火燒]]。為終極滅屍方法。《[[子不語|子不語]]》：「放火燒之，嘖嘖之聲，血湧骨鳴。」
# [[黑狗血|黑狗血]]、[[雞血|雞血]]
# [[孔明燈|孔明燈]]、[[風箏|風箏]]
# [[毒藥|毒藥]]
# [[人偶|人偶]]
# [[道教|祖師爺]]附身
# [[醋|醋]]（[[闽东|闽东]]一带收尸骨者说）
# [[陽光|陽光]]或強[[光|光]]。殭屍跟[[吸血鬼|吸血鬼]]一樣，被[[陽光|陽光]]或強光照久了也會化成灰。
# 黑驢蹄子。

== 大眾文化 ==
=== 電影中的殭屍 ===
在香港電影中，最初的殭屍都是身穿清朝官員服飾，因從月亮吸收陰氣，死不瞑目，殭屍因積攢怨氣、人為破壞或墓地風水屬性屍變而形成。以爪為武器，被抓傷者會感染屍毒死亡，繼而變成另一隻殭屍；於月圓之夜，陰氣增強的時候，力量亦會隨之而增強。中國殭屍電影的始祖是林正英，由他出演的殭屍電影超過25部。
{{main|殭屍片}}
殭屍在1980年代在[[香港電影|香港電影]]是很受歡迎的主題。代表作為[[林正英|林正英]]的[[殭屍道長|殭屍道長]]系列。在電影中，用[[符籙|符]]貼在殭屍的前額可以讓殭屍睡著不動。也是在[[電影|電影]]中，殭屍總是穿著[[清朝官服|清朝官服]]並總是向前伸直手臂，一般以跳躍前進。不過[[外國|外國]]的[[喪屍|喪屍]]和[[清朝|清朝]][[中國|中國]]殭屍不同，手臂可以随意伸展，也以近似[[人類|人類]]的形式行走。
