'''經濟部中央地質調查所'''（簡稱'''地調所'''，{{lang-en|'''Central Geological Survey'''}}）為[[中華民國|中華民國]][[中華民國經濟部|經濟部]]所屬之行政機關，主管全國[[地質|地質]]與礦藏的調查。

== 沿革 ==
[[File:ROC-MOEA-CGS_booth,_Taipei_IT_Month_20161211.jpg|thumb]]，經濟部中央地質調查所攤位]]
*1916年，[[北洋政府|北洋政府]]於[[北京|北京]]成立'''[[中央地質調查所|地質調查所]]'''，隸屬於[[工商部|工商部]]；首任所長是[[丁文江|丁文江]]。
*1936年，地質調查所遷往[[南京市|南京市]]，屬經濟部。
*[[國民政府|國民政府]]遷往[[臺灣|臺灣]]後並未立即恢復地質調查所建制，而是由1946年成立的'''臺灣省地質調查所'''執行地質調查工作，首任所長為[[畢慶昌|畢慶昌]]。
*1978年11月20日，於[[臺北市|臺北市]][[仁愛路_(台北市)|仁愛路二段]]恢復成立'''經濟部中央地質調查所'''。臺灣省地質調查所併入經濟部中央地質調查所，由[[徐鐵良|徐鐵良]]出任所長。
*1983年5月，經濟部中央地質調查所遷至臺北縣中和市（今[[新北市|新北市]][[中和區|中和區]]）。
*未來，隨行政院環境保護署改組升格為[[中華民國環境資源部|環境資源部]]後，由[[行政院農業委員會水土保持局|行政院農業委員會水土保持局]]整併經濟部[[經濟部礦業司|礦業司]]和[[經濟部礦務局|礦務局]]為「環境資源部水土保持及地質礦產署」。

== 組織 ==
