'''經濟剩盈餘'''是用於[[經濟學|經濟學]]的一種概念，可分為[[#消费者盈余|消费者盈余]]、[[#生产者盈余|生产者盈余]]，及兩者加總形成的總盈餘。

== 消费者剩余 ==
[[File:Economic-surpluses.svg|170px]]

'''消费者剩余'''（consumer surplus）是指购买者的支付意愿减去购买者的实际支付量。[[消费者|消费者]]剩余衡量了购买者自己感觉到所获得的额外利益。例如某场[[电影|电影]]票价为20元，可是消费者認為其應價值50元，那么消费者剩余则是30元。如果想尊重买者的[[偏好|偏好]]，那么消费者剩余不失为经济福利的一种好的衡量标准。

我們也可以從[[無差異曲線|無差異曲線]]分析中看出消费者剩余。
:<math>\Delta CS = \frac{1}{2} \left( {Q_1  + Q_0 } \right)\left( {P_1 - P_0} \right)</math>

== 生产者剩余 ==
'''生产者剩余'''（producer surplus）意思是卖者出售一种物品或服务得到的价格减去卖者的[[成本|成本]]。如电影公司提供一部[[电影|电影]]的成本是5元，可票价是20元，那么生产者剩余是15元。生产者剩余衡量生产者所得到的额外利益。

== 總剩餘 ==
'''總剩餘'''（total surplus）是指[[#消费者剩余|消费者剩余]]和[[#生产者剩余|生产者剩余]]的總和。總剩餘可以衡量社會的經濟福利。消費者剩餘是買者從參與[[市場|市場]]活動中得到的利益，而生產者剩餘是賣者得到的利益。因此，常把總剩餘作為社會經濟福利的衡量指標。

以剛才的例子作為例子：

  -{zh-hans:'''总剩余=消费者剩余+生产者剩余=买者的评价-买者支付的量+卖者得到的量-卖者的成本''';zh-hant:'''總剩餘=消費者剩餘+生產者剩餘=買者的評價-買者支付的量+賣者得到的量-賣者的成本''';}-

由於買者支付的量等於賣者得到的量，因此：

  -{zh-hans:'''总剩余=买者的评价-卖者的成本''';zh-hant:'''總剩餘=買者的評價-賣者的成本''';}-

這場電影的社會總福利是50-5=45元。
