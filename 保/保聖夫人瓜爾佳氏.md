'''奉聖夫人'''（{{bd|？||1699年||catIdx=F}}），碑文记为[[瓜爾佳氏|瓜爾佳氏]]。內務府三旗[[包衣|包衣]]佐領下人。[[康熙帝|康熙帝]]之乳母，因襲[[明朝|明朝]]制度而追封瓜爾佳氏为「保聖夫人」<ref>{{cite book |author1=徐廣源著 |title=《清朝皇陵探奇》 |publisher=新世界出版社 |isbn=9787800054471 |edition=1998年版 |accessdate=2019-01-08}}</ref>。

== 生平 ==

關於瓜爾佳氏的出身背景，在[[清朝|清朝]]正史中語焉不詳。僅知瓜爾佳氏是由[[孝莊文皇后|皇太后]][[博爾濟吉特氏|博爾濟吉特氏]]從眾​​保母中親自揀選出來，以“謹厚”入選，進宮看護襁褓中的玄燁。瓜爾佳氏忠於職守，不分黑夜白天，盡心盡意地把康熙帝照料得無微不至，親如慈母<ref>雍正朝《八旗通志初集》下所錄之《列女傳一》，現藏哈佛燕京圖書館。</ref>。為了照看好康熙帝，瓜爾佳氏把自己的家都不顧了。

康熙帝大多數時間隨保母避痘在外，不能享受尋常人家的天倫之樂。這段經歷既使他抱憾終生，也是他與保母瓜爾佳氏感情深篤的重要原因。因此，康熙帝為了回報瓜爾佳氏的養育之恩，在生活上處處給她以優厚待遇，使她“以樂餘年”<ref>{{Cite web |url=http://jds.cssn.cn/webpic/web/jdsww/UploadFiles/ztsjk/2010/12/201012020920159360.pdf |title=满洲皇帝与保母|accessdate=2019-01-08 |author=作者：刘小萌 |date=2010-11-25|work= |publisher=中国人民大学清史研究所网，來源：《北京社會科學》|language=Zh-hans }}</ref>。

[[康熙|康熙]]三十八年（1699年），瓜爾佳氏得了重病，臥床不起。康熙帝知道這個消息後，立刻派去禦醫為她治病，送去皇宮裡的好藥，並不時派人探問病情。瓜爾佳氏終因年事已高，加之操勞過度，於本年閏七月離開了人世。康熙帝聽到噩耗，悲痛萬分，親自到瓜爾佳氏的靈前祭奠舉哀。不久，康熙帝頒發諭旨，追贈她為保聖夫人，喪事全部由皇家辦理。而且，在皇陵近地、[[奉聖夫人朴氏|奉聖夫人]]園墓之旁選擇了一處佳壤作保聖夫人園墓，遣官諭祭四次<ref>{{Cite web |url=http://www.cssn.cn/zgs/zgs_zggds/201702/t20170222_3425502_2.shtml |title=再论清代皇室之乳保|accessdate=2019-01-08 |author=作者：沈欣 |date=2017-02-22|work= |publisher=中国社会科学网，来源：《北京社会科学》|language=Zh-hans }}</ref>。

瓜爾佳氏的丈夫[[圖克善|圖克善]]原為散員，因其妻撫育幼帝頗有勞績，因此，“屢加恩眷”，康熙六年賞給[[都尉|拜他喇哈番]]加一級的世職。這說明奶公的收入水平，大體接近八旗的四品官員，予給他們的待遇是不低的。圖克善去世後十二年，瓜爾佳氏病歿，夫妻合葬在一座地宮內。玄燁再次御賜乳公圖克善碑文，共立於墳前<ref>{{cite book |author1=徐廣源著 |title=清皇陵地宮親探記 |publisher=遠流出版事業股份有限公司 |isbn=9789573270102 |edition=2013年版 |accessdate=2019-01-08}}</ref>。
