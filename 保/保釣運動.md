'''保釣運動'''始於1970年，是由[[中国内地|中国内地]]、[[香港|香港]]、[[臺灣|臺灣]]地区[[华人|华人]]和[[海外華人|海外華人]]為了回應[[釣魚臺列嶼主權問題|日本宣稱擁有釣魚島主權]]所發起的一系列民間運動。其活动方式包括[[游行|游行]]、[[示威|示威]]、驾船出海至[[釣魚臺列嶼|釣魚臺列嶼]]海域与登陸[[釣魚島|釣魚島]]宣示[[中國|中國]][[主權|主權]]等。

== 緣起 ==
{{main|釣魚台列嶼主權問題}}
釣魚臺列嶼為[[臺灣省|臺灣省]][[基隆市|基隆市]]東北方約186公里（102海浬）的[[釣魚島|釣魚島]]與附近七個小島（共計五個無人島與三個岩礁）的合稱，總面積約6.1636平方公里，是[[大屯山|大屯山]]及[[觀音山_(新北市)|觀音山]]山脈向東北延伸入東海海底的突出部分，實為臺灣北部的地質延伸，在地質上與臺灣東北方[[花瓶嶼|花瓶嶼]]、[[棉花嶼|棉花嶼]]、[[彭佳嶼|彭佳嶼]]一脈相承，同為臺灣的附屬島嶼。<ref>{{Cite web |url=http://www.president.gov.tw/Default.aspx?tabid=131&itemid=28069&rmid=514&word2=And&word3=%E7%B8%BD%E7%B5%B1%E8%A6%96%E5%AF%9F%E5%BD%AD%E4%BD%B3%E5%B6%BC |title=總統視察彭佳嶼 |author= |date=2013年1月31日 |publisher=[[中華民國總統府|中華民國總統府]] |language= |accessdate=2012年11月16日 |quote= }}{{zh-tw}}</ref>。釣魚島上有淡水，沒有居民，附近海域自古是臺灣、福建漁民的傳統[[漁場|漁場]]。

1968年10月，[[聯合國亞洲經濟開發委員會|聯合國亞洲經濟開發委員會]]（ECAFE）指出釣魚台列嶼海底[[大陸棚|大陸棚]]為一大[[油田|油田]]，引起[[台灣|台灣]]、[[日本|日本]]與[[中國大陸|中國大陸]]方面的關注。

1970年9月10日，美日兩國私自達成協議，美國準備在1972年把美軍二戰時所佔領的[[琉球群島|琉球群島]]交予日本，當中包括釣魚台列嶼。過程中，雖然美國沒有明文提及釣魚台的法定地位，但是由於[[美國駐日大使館|美國駐日大使館]]表示「釣魚台為琉球群島一部份」，日本可以對釣魚台進行直接管轄，而日本也開始驅逐來自台灣的漁民，引發全球各地華人抗議。在台灣的[[中華民國|中華民國]]一直主張管有釣魚台，美方的舉動令[[中華民國政府|中華民國政府]]極其尷尬，而中國大陸當局亦從未承認日本管有釣魚島。

== 民间主要活动 ==
=== 20世纪 ===
==== 台灣 ====
1970年9月，[[中國時報|中國時報]]董事長[[余紀忠|余紀忠]]與總編輯[[臧遠侯|臧遠侯]]決定指派記者登上[[釣魚台|釣魚台]]採訪。9月1日，記者[[宇業熒|宇業熒]]、[[姚琢奇|姚琢奇]]、[[劉永寧|劉永寧]]、[[蔡篤勝|蔡篤勝]]等人在[[基隆|基隆]]搭乘於[[水試所|水試所]]租用的[[海憲號|海憲號]][[遠洋漁船|遠洋漁船]]啟航。9月2日上午9點30分，一行人成功登陸釣魚台，並在島上升起[[中華民國國旗|中華民國國旗]]，海憲號船長[[王德泉|王德泉]]則用八厘米錄影機拍下這段畫面。隨後中國時報記者又在峭壁上留下「蔣總統萬歲」五字<ref>中國時報，王銘義，A3版，民國101年8月29日</ref>。

1970年11月17日，[[美國|美國]][[普林斯頓大學|普林斯頓大學]]的台灣留學生組成「保衛釣魚台行動委員會」，表示「反對美日私相受授」、「外抗強權，內爭主權」，一方面抨擊美國與日本，另一方面也要求[[中華民國政府|中華民國政府]]應該力爭主權。

1971年1月29日，二千多位台灣及香港留美學生在[[聯合國總部|聯合國總部]]外面示威，高呼「保衛釣魚台」。以台灣和香港留學美國為主的學生，分別在[[華府|華府]]、[[紐約|紐約]]、[[舊金山|舊金山]]、[[西雅圖|西雅圖]]、[[洛杉磯|洛杉磯]]、[[芝加哥|芝加哥]]等地舉行[[第一次保釣示威|第一次保釣示威]]，光紐約就有近一千三百人參與示威。兩周後，[[香港|香港]]教師、學生組成「香港保衛釣魚台行動委員會」，發動學生上街示威，指控美日勾結；其中，該年7月7日[[香港專上學生聯會|學聯]]在[[維園|維園]]發起的[[七七大示威|七七大示威]]更演變成嚴重衝突：[[香港皇家警察|香港皇家警察]]威利[[警司|警司]]率領千多名警察，以近乎一比一比例武力驅散示威者，用[[警棍|警棍]]隨意毆打手無寸鐵的市民，由此為香港70年代的學運潮揭開了序幕。<ref>[http://www.xinmiao.com.hk/0001/20110707.02T.htm 保衛釣魚台 七七大示威 學生血染公園] {{webarchive|url=https://web.archive.org/web/20141101182500/http://www.xinmiao.com.hk/0001/20110707.02T.htm |date=2014-11-01 }}</ref>

1971年3月，包含[[趙元任|趙元任]]、[[余英時|余英時]]、[[李遠哲|李遠哲]]、[[張系國|張系國]]在內的500多位[[華裔|華裔]]學生上書[[中華民國總統|總統]][[蔣中正|蔣中正]]，呼籲政府抵抗日本新侵略，並拒絕參加「中日韓聯合開發海底資源協定」。在1971年[[布朗大學|布朗大學]]的保釣運動國是討論會以及[[密西根州|密西根州]][[安娜堡|安娜堡]]國是大會，兩場會議辯論中國在聯合國的代表權問題，是保釣運動中首次有[[左派|左派]]台灣留學生公開支持只有[[中華人民共和國|中華人民共和國]]才能代表中國（CHINA）；決議1971年9月21日在聯合國總部前舉行[[九二一聯合國大遊行|九二一聯合國大遊行]]，爭取中華人民共和國在聯合國的合法席位。留美的左派台灣學生以[[五星紅旗|五星紅旗]]開道，後面跟著的是10個人掛著「中華人民共和國萬歲！」的牌子，再後面是10面[[毛澤東|毛澤東]]畫像招牌。<ref>董慶園，〈悼君復吾兄〉，《讀書》2006年3月，107-112</ref>

在臺灣，由[[張俊宏|張俊宏]]和[[陳鼓應|陳鼓應]]等人於1968年所創立的改革派雜誌《[[大學_(雜誌)|大學雜誌]]》，是臺灣最早的保釣運動發聲管道之一。1971年1月開始，大學雜誌開始刊登保釣運動的消息，包括學者[[丘宏達|丘宏達]]的相關文章，同時也鼓吹自由主義，希望政府恢復人民政治自由。

1971年6月，在美日簽署移轉琉球（包括釣魚臺列嶼在內）管轄權文件的同一日，上千大學生發動示威遊行，至美、日大使館遞交抗議書。不過此後隨著美日的簽約已定，台灣學生的保釣運動也就暫停。{{r|"AutoDI-320"}}

1971年10月，[[聯合國大會2758號決議|中華民國退出聯合國]]，引起台灣內部的憂患意識，學生以保釣運動為名，開始組織社團，發起學生運動。1972年4月4至9日，國民黨政府擔心學生運動失控，連續6天在《[[中央日報|中央日報]]》副刊登出孤影（[[敏洪奎|敏洪奎]]）《一個小市民的心聲》一文，希望將學生運動降溫。

因為保釣運動與聯合國等議題，讓[[國立台灣大學|台大]]學生[[王杏慶|王杏慶]]、[[王復蘇|王復蘇]]等[[國立政治大學|政大]]、[[國立師範大學|師大]]學生，經過學生社團間的相互聯絡、交換意見，於1972年12月初在報紙聯名發表《我們的呼籲》，要求青年學生擁抱國家、社會，表達自己的聲音。王復蘇等150多人組織「社會服務團」，開始調查社會議題，學生運動由愛國保釣轉向關心社會議題。1973年，台大學生又發起以服務農村、漁村、山地部落等的「百萬小時奉獻運動」，希望學生主動進行社會關懷。中華民國政府擔心因保釣運動而衍生的學生運動失控，中國國民黨透過[[中國青年反共救國團|救國團]]介入並接手各類社會服務團體，以控制學生運動。同時，[[警備總部|警備總部]]搜索陳鼓應等人的住宅，指控他們立場親共。1974年，警備總部發動[[臺大哲學系事件|臺大哲學系事件]]，解除自由派學者的教職，台大哲學研究所因此停招一年。學生運動遭到壓制，保釣運動的風潮也在台灣降溫。

==== 港臺民間聯合行動 ====
{{wikinews|香港保釣船與日本警方對峙後駛離}}
1996年以後，[[香港|香港]]和[[台灣|台灣]]民間多次組織了保釣的行動。但是日本每次都派出警船攔截，只有極少幾次登島成功。当时尚为[[英属香港|英属香港]]的[[全球華人保釣大聯盟|全球華人保釣大聯盟]]的[[陳毓祥|陳毓祥]]等五人隨[[保釣號|保釣號]]貨輪在9月26日抵達釣魚台海域，多次被日本舰艇与军机围追堵截，難以搶灘，五人跳海示威，向钓鱼岛游去，其中陳毓祥遇溺身亡<ref>[http://news.ifeng.com/history/gaoqing/detail_2012_08/17/16888905_0.shtml#p=1 揭秘中国保钓烈士第一人惨烈牺牲真相]，凤凰网，2012年8月17日</ref>，另一名遇溺的保釣人士[[方裕源|方裕源]]則被日本的直升機救起，送往[[石垣島|石垣島]]的醫院救治。陳毓祥的逝世激發港、台兩地的保釣行動。10月6日，即陳毓祥舉殯當日，由[[新北市|臺北縣]]議員[[金介壽|金介壽]]和香港立法局議員[[曾健成|曾健成]]領導的新一輪保釣行動，於基隆租用了上千艘漁船出發駛往釣魚台列嶼。參與者在10月7日成功登陸釣魚島，並一同在島上同時揮舞[[五星紅旗|五星紅旗]]及[[青天白日滿地紅旗|青天白日滿地紅旗]]的旗幟，以表示釣魚台是所有中國人的領土。<ref>[http://www.youtube.com/watch?v=VZyjL3b6VqU Youtube上的電視新聞片段]</ref> 當日，[[台灣建國黨|台灣建國黨]]的成員在香港保釣人士下塌的酒店外示威，並焚燒了兩-{zh-hans:面;zh-hant:支;}-[[五星紅旗|五星紅旗]]。10月9日，保釣行動委員會成員闖入[[日本駐港領事館|日本駐港領事館]]，在館內高唱[[義勇軍進行曲|義勇軍進行曲]]達半小時，以示抗議。其後[[日本青年社|日本青年社]]向[[新華通訊社香港分社|新華通訊社香港分社]]發出恐嚇信，表示會殺死所有中國人。

==== 中國大陸 ====
在1970年代初保釣運動興起時，[[中華人民共和國政府|中華人民共和國政府]]官方並沒有表示支持，其民間也沒有相關活動。

1978年8月，中华人民共和国与日本簽署《[[中日和平友好條約|和平友好條約]]》。

1978年[[日本右翼團體|日本右翼團體]]提出於釣魚岛修建直升機場，其後200多艘中國漁船到有關海域宣示主權，日本右翼團體放棄建機場計劃。當時中國領導[[鄧小平|鄧小平]]表示，釣魚岛主權問題可在日後慢慢解決。<ref>{{cite web|author=新华网|title=新闻资料：钓鱼岛 钓的什么鱼？ 日本又搞小动作|url=http://news.ifeng.com/mainland/special/zrczdydxz/content-3/detail_2010_09/07/2455064_0.shtml|publisher=凤凰新闻|accessdate=2012-07-15}}</ref>

=== 2000年代 ===
{{see|聯合號海釣船事件}}
2000年以後，[[中國大陸|中國大陸]]逐漸放寬民間保釣行動的限制，民間保釣行動比以往積極。

2003年12月26日由中國918愛國網、愛國者同盟網在[[廈門市|廈門市]]舉行全球華人保釣論壇，彙集了海內外華人代表30多人，並決定成立[[中國民間保釣聯合會|中國民間保釣聯合會]]，推選[[童增|童增]]為會長。

2004年1月，[[武漢大學|武漢大學]]學生[[梁振|梁振]]參與保釣並出海，是湖北地区直接参加保钓行动的第一人。{{r|"AutoDI-346"}}

2004年3月24日中國保釣人士[[馮錦華|馮錦華]]等7人成功登上釣魚島，這是[[中華人民共和國|中華人民共和國]]建政後[[中國大陸|中國大陸]]人民首次登上[[釣魚島|釣魚島]]。<ref>{{cite news|title=回眸周刊：生死保钓行|work=《楚天都市报》|date=2004-02-08|accessdate=2008-08-19|page=19页|author=周洁、潘勤|publisher=湖北日报传媒集团}}</ref>

2006年10月22日[[香港|香港]][[保釣行動委員會|保釣行動委員會]]的「[[保釣二號|保釣二號]]」漁船載著[[中国大陆|中国大陆]]及[[香港特别行政区|香港特别行政区]]、[[澳门特别行政区|澳门特别行政区]]保釣人士起程前往[[台灣|台灣]]，預定於抵台後與台灣及海外保釣人士會合，共同前往釣魚岛宣示主權，與此同時，[[中華人民共和國外交部|中華人民共和國外交部]]發表聲明警告[[日本政府|日本政府]]冷靜對待保釣人士。但因為受到[[中華民國|中華民國]]執政[[民進黨|民進黨]]政府的阻撓，禁止「[[保釣二號|保釣二號]]」停靠基隆港，台灣保釣人士未能出海，最終「[[保釣二號|保釣二號]]」直接前往釣魚岛。但「保釣二號」進入釣魚岛附近海域後，遭到[[日本海上保安廳|日本海上保安廳]]艦隻和偵察機包圍，並遭日方艦隻撞擊驅逐，結果「[[保釣二號|保釣二號]]」在進入距離[[釣魚島|釣魚島]]12海里的水域停下後，船員為十年前於保釣行動中在[[釣魚岛|釣魚岛]]海域溺斃的保釣領袖[[陳毓祥|陳毓祥]]舉行公祭儀式，然後被迫折返回航。[[中國大陸|中國大陸]][[外交部|外交部]]發表嚴正聲明，重申[[中國|中國]]擁有[[釣魚島|釣魚島]]主權。當時出於維護地區穩定的目的，两岸政府都沒有派出軍隊參與保釣運動。

2008年6月10日，發生[[聯合號海釣船事件|聯合號海釣船事件]]，台灣[[聯合號|聯合號]]漁船在[[釣魚島|釣魚島]]海域遭日本巡邏船撞沉，船上16人被日本巡邏船救起，但船長[[何鴻義|何鴻義]]和兩名船員則被日本當局扣留二至三天後才獲釋返抵台灣。此一事件引發台灣社會的不滿。日本當局發表聲明，對日方巡邏船撞沉台灣海釣船聯合號一事，承認日方確實有疏失，於當時臺北縣政府出面打國際官司取得勝利後，日方召開記者會發表了正式道歉聲明，九十度鞠躬對此表示遺憾，並正式賠償船員損失。{{r|"AutoDI-323"}}{{r|"AutoDI-324"}}

十二名台灣保釣人士在6月15日搭乘「全家福號」漁船前往釣魚台宣示主權，並由台灣[[海巡署|海巡署]]派出巡防艦保護，在釣魚台海域順利繞島一周後返航。之後日本發表聲明，對台灣當局侵犯領海表示遺憾。

2008年11月9日[[中華保釣協會|中華保釣協會]]在臺北縣永和市(今[[新北市|新北市]][[永和區|永和區]])成立。

2009年，台湾和中国大陸，以及香港、澳门的保钓团体曾计划进行新一轮联合保钓行动，计划5月上旬乘船前往钓鱼岛<ref>{{cite news |author=叶兵|title=两岸四地保钓人士准备新一轮保钓 |date=2009-03-22 |publisher= |url = http://www.voanews.com/chinese/w2009-03-22-voa26.cfm?rss=long%20story |work =VOA |pages = |accessdate = 2009-03-22 |language = }}</ref>，但因天氣惡劣而取消。

== 参与者倾向發展 ==
參與美國保釣運動的不少右派台灣留學生之後都成為[[反共愛國聯盟|反共愛國聯盟]]等團體的重要成員，例如[[馬英九|馬英九]]；而一些參加保釣運動的左派台灣留學生倒向[[中華人民共和國|中華人民共和國]]；五位左派臺灣留學生李我焱、陳恆次、陳治利、王正方和王春生並於1971年9月訪問[[北京|北京]]，受到[[周恩來|周恩來]]接見。但当时中国大陆处于[[文革|文革]]时期，周恩來以當時環境不佳為由，並不贊成他們留在中國大陸。不少台灣留學生如[[林孝信|林孝信]]、[[郭松棻|郭松棻]]等人也因此被列入[[黑名單|黑名單]]而長期無法返台。[[林盛中|林盛中]]於1971年获美国布朗大学地质系博士学位，1972年赴大陸任中国地质科学院矿床地质研究所副研究员，是少數左派在當時赴大陸的例子。

一些[[統派|統派]]學者如[[陳鼓應|陳鼓應]]、[[王曉波|王曉波]]等人，也將保釣運動視為[[中國民族主義|中國民族主義]]的號召基礎。2008年11月9日，中華保釣協會在[[永和區|永和市]]成立，第一屆理事長劉源俊。
