'''保羅·約瑟夫·布賽克'''（{{lang-en|Paul Joseph Busiek}}；{{bd|1923年|4月1日|2016年|4月18日}}），是美國的[[共和黨_(美國)|共和黨]]政治人物及醫生，前[[密蘇里州眾議院|密蘇里州眾議院]]議員。

==生平==
布賽克出生在[[密苏里州|密苏里州]][[斯普林菲尔德_(密苏里州)|斯普林菲尔德]]，曾入讀{{le|杜利大學|Drury University|杜利學院}}和[[圣路易斯华盛顿大学|圣路易斯华盛顿大学]]，並在後者獲得醫學學位。他在[[紐約州|紐約州]][[羅徹斯特_(紐約州)|羅徹斯特]]實習，又在聖路易斯的兒童醫院任職兒科醫生，後來在[[斯普林菲尔德_(密苏里州)|斯普林菲尔德]]開辦私人診所。

1976年，他當選為密蘇里州眾議院第一百四十選區議員，但在1978年離開<ref>[https://books.google.ca/books?id=-L9sAAAAMAAJ&q=%22Busiek,+Paul%22+1923&dq=%22Busiek,+Paul%22+1923&hl=en&sa=X&ei=mgaGVLCWLMGvogT-g4Aw&ved=0CBwQ6AEwAA 'American Pediatric Directory, Volume 14', American Pediatric Directory Company, 1969]</ref><ref name="mo">{{cite web|url=http://www.sos.mo.gov/archives/history/historicallistings/molegj.asp|title=SOS, Missouri - State Archives: Missouri State Legislators 1820-2000|author=Missouri Secretary of State :: State Archives|publisher=sos.mo.gov|accessdate=2014-01-25}}</ref>。

2016年4月18日，布賽克逝世<ref>[http://www.legacy.com/obituaries/news-leader/obituary.aspx?n=paul-j-busiek&pid=179677903&fhid=12405 Dr. Paul J. Busiek]</ref>。
