'''保山—腾冲高速公路'''简称'''保腾高速'''，是[[云南省|云南省]]的一条[[云南高速公路|省级高速公路]]，编号为S10，连接[[保山市|保山]]和[[腾冲市|腾冲]]两座城市，起点是[[保龙高速|保龙高速]]上的小田坝互通，止于腾冲市[[中和镇_(腾冲市)|中和镇]]毛家营村南侧岔路口，连接已建成通车的[[腾密公路|腾冲至缅甸密支那公路]]<ref name="2012-06-04">{{cite web |url=http://yn.yunnan.cn/html/2012-06/04/content_2234210.htm |title=国际大通道建设提速 保腾高速公路年底有望通车（图） |author=杨之辉 |date=2012-06-04 |website=yn.yunnan.cn |publisher=云南网 |access-date=2018-01-18 |quote= |archive-url=https://web.archive.org/web/20180119120041/http://yn.yunnan.cn/html/2012-06/04/content_2234210.htm |archive-date=2018-01-19 |dead-url=yes }}</ref>，保山至腾冲过路费为82元<ref name="2016-05-04">{{cite web |url=http://www.yn.chinanews.com/news/2016/0504/13854.html |title=保腾高速勐柳收费站5月4日开始拆除 |author=黄朝红 |date=2016-05-04 |website=www.yn.chinanews.com |publisher=[[都市时报|都市时报]] |access-date=2018-01-18 |quote=}}</ref>。保腾高速全长63.86公里，项目概算投资46亿余元，平均每公里造价7,197.21万元，双向4车道，设计时速80公里/小时<ref name="china-highway">{{cite web |url=http://www.china-highway.com/Hw/info/hid/112.html |title=S10保腾高速-简介 |author= |date= |website=www.china-highway.com |publisher=中国高速公路网 |access-date=2018-01-18 |quote=}}</ref>，路基宽度为24.5米，桥涵及构造物设计荷载等级为公路CI级<ref>{{cite web |url=http://www.yncllq.com/cpdetail_72.htm |title=保腾高速 |author= |date= |website=www.yncllq.com |publisher=云南驰龙路桥工程有限公司 |access-date=2018-01-18 |quote=}}</ref>。全线共有特大桥两座、大桥114座、中小桥48座、隧道3座，桥隧总长占全线的29.5%，重点控制性工程是峡谷跨径世界第一、海拔高度亚洲第一的[[龙江大桥|龙江大桥]]<ref name="2007-10-29"></ref>。保腾高速较原有的89.54公里[[317省道_(云南)|保腾二级路]]短22公里<ref>{{cite web |url=http://baoshan.yunnan.cn/html/2013-02/05/content_2607092.htm |title=全长63.8公里 保腾高速路试通车 |author=崔敏 |date=2013-02-05 |website=baoshan.yunnan.cn |publisher=云南网 |access-date=2018-01-18 |quote= |archive-url=https://web.archive.org/web/20160306074625/http://baoshan.yunnan.cn/html/2013-02/05/content_2607092.htm |archive-date=2016-03-06 |dead-url=yes }}</ref>，高速的修通，实现[[昆明市|昆明]]至腾冲全程[[高速公路|高速化]]、昆明到[[缅甸|缅甸]][[密支那|密支那]]全程公路高等级化<ref name="2012-06-04"></ref>，对开发滇西旅游、拉动区域经济增长、建设云南面向南亚的国际大通道有重要意义<ref name="2007-10-29">{{cite web |url=http://news.sina.com.cn/c/2007-10-29/100212805414s.shtml |title=保腾高速公路昨动员建设昆明到腾冲可望7小时到达 |author1=赵学康 |author2=崔敏 |date=2007-10-29 |website=news.sina.com.cn |publisher=[[春城晚报|春城晚报]] |access-date=2018-01-18 |quote=}}</ref>。高速公路的通车，也使得腾冲的房地产迎来提价峰值<ref>{{cite web |url=http://sh.house.163.com/12/0621/06/84GKMPQT00073SDJ.html |title=保腾高速建成后 腾冲房地产或迎来提价峰值 |author= |date=2012-06-21 |website=sh.house.163.com |publisher=[[云南信息报|云南信息报]] |access-date=2018-01-18 |quote=}}</ref>。

==地质环境==

保腾高速穿过[[高黎贡山|高黎贡山脉]]和[[瑞丽江|龙江]]大峡谷，气候上属于[[亚热带气候|亚热带]]干河热谷气候，沿线地形地质复杂，经过高[[地震|地震]]区、高[[风化|风化]]区和高[[地热|地热]]区，路基施工、边坡护理、隧道施工等难度较大<ref name="2007-10-29"></ref><ref>{{cite web |url=http://news.cngaosu.com/wenhua/whdt/2016/07/51638.html |title=“美丽高速行”推荐路线：云南保腾高速公路 |author=张建全 |date=2016-07-09 |website=news.cngaosu.com |publisher=中国高速网 |access-date=2018-01-18 |quote=}}</ref>。保腾高速公路经过的[[腾冲坝|腾冲坝]]为近代[[火山|火山]]群，在60多平方公里的地面上分布着97座新生代火山，保腾高速也是中国第一条修建在火山区的高速公路<ref name="2007-10-29"></ref>。
