'''雖然'''保加利亞的女同性戀、男同性戀、雙性戀與跨性別者權益'''受法律保障合法，但該國仍不允許[[同性婚姻|同性婚姻]]。

== 歷史 ==
在保加利亞歷史上，同性戀非刑事化始於1858年，當時[[保加利亞|保加利亞]]仍為[[鄂圖曼土耳其帝國|鄂圖曼土耳其帝國]]的一部份。1878年，保加利亞脫離鄂圖曼土耳其獨立。然而，1896年保加利亞修改刑法，16歲以上的男同性戀性行為需被處以6個月[[有期徒刑|有期徒刑]]。這項刑罰在1951年3月13日加長，同性性行為在保加利亞需被處以最長3年的有期徒刑，併科[[罰金|罰金]]<ref>{{Cite web|url=http://penallaw.wordpress.com/2009/09/08/наказателен-кодекс-от-1951-г-особена-част-4/|title=Bulgarian Penalty Code of 1951|publisher={{lang|bg|БЛОГ ЗА НАКАЗАТЕЛНО ПРАВО}}|language=bg|accessdate=2011-04-18}}</ref>。

多數[[東歐|東歐]]國家過去在共產黨執政期間，基於共產黨政權[[反教權|反教權]]的官方政策，對涉及教會反對的性議題採取支持的政策。

1968年5月1日，保加利亞通過刑法修正案，將同性戀相關的規定自刑法典中完全廢除。

民主化後，隨著教會的影響力恢復及增加，政府對[[同性戀|同性戀]]議題逐漸持保留態度，但傳播媒體上也偶有同志情節的節目放映，近年公開[[出櫃|出櫃]]的保加利亞人也有增長趨勢。

== 同性伴侶關係 ==
2009年，同性伴侶關係成為保加利亞的重要議題之一，因為保加利亞政府提出新的《家庭法》草案，規定伴侶關係應在政府機構登記。草案將在2009年6月進行表決。但由於條文裡並未提及同性伴侶，因此遭到保加利亞反歧視委員會和同性平權運動者的抗議，爭論持續擴大。

== 歧視與仇恨罪保護 ==
2003年，通過的《反歧視保護法》保障不同[[性傾向|性傾向]]在就業、交易、教育與居住權上不得有差別待遇。
