'''保送生'''或是一种让某方面突出的高中生不必参加[[大專|大專]]或[[大學|大學]]入學考試，直接获得[[高等院校|高等院校]]录取资格的制度。一般根据学校不同，对保送生的资格要求不同。但大都要经过学生所在学校推荐，并通过大学组织的专家评定。

== 中国大陆 ==
具有以下条件的考生具有保送资格<ref>{{cite web |url = http://gaokao.chsi.com.cn/gkxx/bss/200912/20091217/57961318.html |title = 2010年普通高校招收保送生办法 |accessdate = 2011年2月19日}}</ref>：
# 在高中阶段被评为省级[[优秀学生|优秀学生]]的应届高中毕业生；
# 高中阶段在[[全国中学生学科奥林匹克竞赛|全国中学生学科奥林匹克竞赛]]全国决赛中获得一、二、三等奖的应届高中毕业生；
# 高中阶段在全国中学生学科奥林匹克竞赛省赛区竞赛中获得一等奖的应届高中毕业生。
# 高中阶段在全国青少年科技创新大赛或“明天小小科学家”奖励活动或全国中小学电脑制作活动中获得一、二等奖的应届高中毕业生；
# 高中阶段在[[国际科学与工程大奖赛|国际科学与工程大奖赛]]或[[国际环境科研项目奥林匹克竞赛|国际环境科研项目奥林匹克竞赛]]中获奖的应届高中毕业生；
# 根据高校外国语言文学类专业对生源的特殊要求，经教育部批准具有推荐保送生资格的外国语中学仅可向高校的外国语言文学类专业推荐思想品德和学习成绩优秀且高中阶段均在本校就读的应届高中毕业生；
# 曾获得全国体育比赛前三名、亚洲体育比赛前六名、世界体育比赛前八名和获得球类集体项目运动健将、田径项目运动健将、武术项目武英级和其他项目国际级运动健将称号的退役运动员；
# 公安英烈子女。

2010年，为促进升学公平，教育部决定，自2011年秋季进入高中阶段一年级的学生开始，取消了在学科奥赛中，除全国一等奖且被选入国家集训队的学生以外的获奖者的保送资格。2010年(含)以前已进入高中阶段学习的学生，仍可适用调整前的相关政策。<ref>{{Cite web |url=http://www.ce.cn/xwzx/gnsz/gdxw/201011/19/t20101119_21982817.shtml |title=中国经济网：教育部调整高考加分 奥赛国内奖等取消保送 |access-date=2010-11-24 |archive-url=https://web.archive.org/web/20160304200212/http://www.ce.cn/xwzx/gnsz/gdxw/201011/19/t20101119_21982817.shtml |archive-date=2016-03-04 |dead-url=yes }}</ref>

具有保送资格的学生还要经过所在[[中学|中学]]推荐，通过申报学校的评定才能保送。评定一般由笔试和面试组成，也有部分高校规定保送生仅通过面试即可。上述调整政策适用。
2008年[[中国|中国]]高校中，[[北京大学|北京大学]]、[[清华大学|清华大学]]、[[上海交通大学|上海交通大学]]、[[中国科学技术大学|中国科学技术大学]]、[[浙江大学|浙江大学]]、[[复旦大学|复旦大学]]录取保送生人数较多。<ref>{{cite web |url = http://gaokao.chsi.com.cn/zsgs/bssnlqmd--method-groupByYx,year-2008.dhtml |title = 2008年保送生拟录取名单 |accessdate = 2008年5月6日}}</ref>

不仅应届高中毕业生有保送到高校就读[[本科|本科]][[学士学位|学士学位]]的情形，应届本科毕业生{{noteTag|这里指的是大学四年级开学后约60天内。}}也有资格保送到該校或者其他学校（或单位）的研究生院攻读[[硕士学位|硕士学位]]，称为推荐免试攻读硕士（[[博士|博士]]）研究生资格。详情参见[[中国研究生入学考试|中国研究生入学考试]]。此外，应届硕士毕业生也有机会申请免试直接攻读[[博士学位|博士学位]]，依照情形不同，分为[[硕博连读|硕博连读]]和[[直博|直博]]两种。

==香港==
於[[香港中學會考|香港中學會考]]中成績優異者，俗稱「[[尖子|尖子]]」。考生在香港中學會考取得六科A級，以及中英文不低於C級成績，可豁免參加香港高級程度會考，直接在中六時透過優先取讀計劃報讀大學，坊間俗稱此計劃為「拔尖」；此等計劃源自於[[香港中文大學|香港中文大學]]在1980年代推出的暫取生政策，已隨「334學制」推展而廢止。

== 台灣 ==
[[台灣教育史|台灣教育史]]上出現過好幾次以保送為名的入學制度，但不見得是在升大學、大專階段，目前仍存在之制度如「中等以上學校技藝技能優良學生甄審及保送入學辦法」，簡稱「技優保送入學」。
