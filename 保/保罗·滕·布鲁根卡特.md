'''保罗·滕·布鲁根卡特'''（Paul ten Bruggencate）是一位[[德国|德国]][[天文学家|天文学家]]暨[[天体物理学|天体物理学家]]。

==青年及教育==
滕·布鲁根卡特1901年2月24日出生在[[瑞士|瑞士]][[阿罗萨|阿罗萨]]，曾在[[瑞士|瑞士]]、[[荷兰|荷兰]]和[[德国|德国]]等多所学校上过学，后来在[[哥廷根大学|哥廷根大学]]和[[慕尼黑大学|慕尼黑大学]]攻读[[天文学|天文学]]，1924年在[[雨果·汉斯·里特·冯·泽利格|雨果·汉斯·里特·冯·泽利格]]的指导下开始了博士生研究。但不久由于健康原因，泽利格将指导博士生的工作转交给了汉斯·金勒教授（Hans Kienle）<ref name="Jäg">{{cite journal|title = Nachruf auf P. ten Bruggencate|first = F. W.| last = Jäger|journal = Mitteilungen der Astronomischen Gesellschaft|volume = 15|pages = 21–23|url = http://articles.adsabs.harvard.edu/cgi-bin/nph-journal_query?volume=15&plate_select=NO&page=21&plate=&cover=&journal=MitAG}}</ref><ref name="Kienle">{{cite journal|doi = 10.1007/BF00622019|title = Paul ten Bruggencate 24.2.1901 14.9.1961|year = 1962|last1 = Kienle|first1 = H.|journal = Die Naturwissenschaften|volume = 49|issue = 4|pages = 73|bibcode = 1962NW.....49...73K }}</ref>。

1924年滕·布鲁根卡特从慕尼黑大学毕业后，与汉斯·金勒一道到了哥廷根天文台，在那里担任[[哥廷根大学|哥廷根大学]]助教一直到1926年离开远赴[[爪哇岛|爪哇岛]]。到达[[印度尼西亚|印尼]]后，他与[[荷兰|荷兰]][[天文学家|天文学家]]琼·沃特（Joan Voûte）在伦邦附近的博斯查天文台开始了对[[造父变星|造父变星]]的研究。在博斯查天文台的两年中，他曾造访过[[威尔逊山天文台|威尔逊山天文台]]和[[哈佛大学天文台|哈佛大学天文台]]。1929年，他又转至德国北部的[[格赖夫斯瓦尔德大学|格赖夫斯瓦尔德大学]]，1935年完成了有关[[造父变星|造父变星]][[特许任教资格|特许任教资格]]认证，该项研究大部分是根据他在伦邦所作的观察<ref name="Jäg"/><ref name="Kienle"/><ref name="Acade">{{cite book|url = https://books.google.com/books?id=oDam18HnrzwC&pg=PA524| title = Göttinger Gelehrte: die Akademie der Wissenschaften zu Göttingen in Bildnissen und Würdigungen 1751-2001|pages = 524| isbn = 978-3-89244-485-5| year = 2001| publisher =Wallstein Verlag| first1 = Karl|last1 =Arndt| first2 = Gerhard|last2 =Gottschalk| first3 = Rudolf|last3 =Smend| first4 = Ruth|last4 = Slenczka}}</ref>。

==学术生涯==
1935年，滕·布鲁根卡特成为了波茨坦天文台首席天文学家，当时波茨坦天文台最昂贵的设备是[[爱因斯坦塔|爱因斯坦塔]]上的[[太阳望远镜|太阳望远镜]]。从那时起，滕·布鲁根克特的研究就主要集中在太阳上。在[[第二次世界大战|二战]]中的1941年，他出任哥廷根大学天文台台长。为满足对太阳的研究，他渴望能得到一架新的太阳望远镜。而位于哥廷根东南的海恩堡天文台是该地区最佳的太阳观察点，当时军方对太阳对短波无线通讯的干扰也逐惭重视，这就使得在战争期间改建该天文台成为了可能。军方的赞助，加上滕·布鲁根卡特对太阳望远镜的渊博知识很快帮助波茨坦天文台建造起了一架新的望远镜<ref name="Jäg"/><ref name="Kienle"/>。

滕·布鲁根卡特与也曾在哥廷根大学天文台工作过一段时间的卡尔-奥托·基彭豪伊尔（Karl-Otto Kiepenheuer）的私人恩怨，使得德国出现了二组太阳研究团队。基彭豪伊尔组建了弗劳恩霍夫研究所（Fraunhofer Institute，不是现今的弗劳恩霍夫协会）并负责协调欧洲数座太阳观测台，而滕·布鲁根卡特也建立了自己的天文台，这二组研究团队都得到军方的支持，以帮助预报短波通讯环境<ref>{{cite book|url = https://books.google.com/books?id=qRWwcIgvWLYC&pg=PA78|title = Kommandosache "Sonnengott": Geschichte der deutschen Sonnenforschung im Dritten Reich und unter alliierter Besatzung| first = Michael P.|last = Seiler| pages = 78–79| isbn = 978-3-8171-1797-0| year = 2007| publisher =[[Verlag_Harri_Deutsch|Verlag Harri Deutsch]]}}</ref>
。

战后，哥廷根天文台继续运转，而滕·布鲁根卡特也着手在瑞士建立第二太阳观察台-洛迦诺天文台（Locarno Observatory ），那里的气象条件较多云的德国更有利。该天文台于上世纪50年代末开始设计动工<ref>{{cite web|url = http://www.irsol.ch/|title = Istituto Ricerche Solari Locarno|accessdate = 2010-04-25}}</ref>，在他死后不久开始全面投入运行<ref name="Jäg"/><ref name="Kienle"/>，后在1984年关闭，其设备被转送至[[西班牙|西班牙]][[特内里费岛|特内里费岛]]泰德天文台（Teide Observatory）<ref>{{cite web|url = http://www.astro.physik.uni-goettingen.de/~ewiehr/gregory.html|title = Das Gregory Coude Teleskop auf Teneriffa|publisher = University Göttingen|accessdate = 2010-04-25}}</ref>，泰德天文台也拥有基彭霍伊尔太阳物理研究所（即弗劳恩霍夫研究所）的望远镜。

1958年至1961年间滕·布鲁根卡特曾当选哥廷根科学院主席<ref name="Acade"/>，直止1961年9月14日去世前，他一直担任哥廷根大学天文台台长一职

[[月球|月球]]上的[[滕·布鲁根卡特环形山|滕·布鲁根卡特环形山]]就是以他的名字命名的。
