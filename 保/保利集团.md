'''中国保利集团公司'''（简称'''保利集团'''）是[[国务院国有资产监督管理委员会|国务院国有资产监督管理委员会]]监管的大型国有中央企业<ref name="a1">{{Cite web |url=http://www.poly.com.cn/1081.html |title=集团简介|accessdate=2018-12-10 |author= |date= |work= |publisher=保利集团网站|language=Zh-hans }}</ref>。公司是[[中华人民共和国|中国]]最大的[[军火|军火]]进出口集团公司，兼营[[房地产|房地产]]、[[文化|文化]][[艺术|艺术]]经营、和矿产资源投资开发等其他业务。公司网站称，2017年时，公司营业收入2501亿元，利润总额300亿元，资产总额8910亿元，位居[[世界500强|世界500强]]第341位<ref name="a1"/>。总部位于北京市[[东城区_(北京市)|东城区]][[朝阳门北大街|朝阳门北大街]]1号新保利大厦28层<ref>{{Cite web |url=http://www.poly.com.cn/1087.html |title=企业名录》中国保利集团公司|accessdate=2018-12-10 |author= |date= |work= |publisher=保利集团网站|language=Zh-hans }}</ref>。

保利科技主要经营通用商品和特种装备及技术的进出口业务，先后引进了包括[[黑鹰直升机|黑鹰直升机]]，瑞士防空系统等产品，并且出口了大量的防务装备。

== 「保利」的由来 ==
* 1983年，[[人民解放軍|解放军]][[总参谋部|总参谋部]]装备部（归口管理对外军贸）、[[中国国际信托投资公司|中国国际信托投资公司]]（简称'''[[中信公司|中信公司]]'''）联合组建，保利科技有限公司。当时中信公司总经理[[徐兆龙|徐兆龙]]起名“保利”，保卫胜利之意。英文POLY与“保利”发音相近。保利公司的标记变形的英文字母P，是由[[赖维武|赖维武]]、[[王小朝|王小朝]]1987年设计，取自英文“PLA”（中国人民解放军缩写）、“POLY”、“POWER”的头一个字母，标记外观像拳头，代表信心和力量。P中心的空间，喻意把握现在，放眼未来。“中国保利集团公司”这几个字是保利公司十周年时贺平总经理请著名书法家[[启功|启功]]书写的墨宝。

== 发展 ==
* 1993年2月，经[[中华人民共和国国务院|国务院]]、[[中央军委|中央军委]]批准，在保利科技有限公司基础上组建，注册资金15亿人民币。王军、贺平等是公司创始人。<ref>[http://ent.sina.com.cn/2004-08-09/1419468411.html 中国保利集团公司简介]</ref>同年，批准为[[国家计划单列企业集团|国家计划单列企业集团]]。
* 1999年3月，中国保利集团公司由军队划归中央大型企业工作委员会领导管理，成为国有重要骨干企业。
* 2000年，划归[[中央企业工委|中央企业工委]]领导，是61家资产百亿元以上的大型企业之一。
* 2003年，划归[[国务院国有资产监督管理委员会|国务院国资委]]领导。
* 2017年8月22日，經國務院和國務院國資委批准，[[中国轻工集团公司|中国轻工集团公司]]、[[中國工藝美術（集團）公司|中國工藝美術（集團）公司]]整體併入中國[[保利集团|保利集团]]公司，成為其全資子公司。中国轻工集团公司与中國工藝美術（集團）公司不再作為國資委監管企業
