{{日本作品|爆音少女！！|ばくおん!!}}，是日本漫畫家[[織本任那|織本任那]]創作的機車題材[[漫畫|漫畫]]作品，自2011年開始於[[秋田書店|秋田書店]]的[[漫畫雜誌|漫畫雜誌]]《月刊Young Champion烈》連載。衍生[[電視動畫|電視動畫]]全12話已於2016年4月至6月播放。

== 概要 ==
剛進入丘乃上女子高等學校就讀的女學生佐倉羽音，原本過著與騎乘[[摩托車|摩托車]]無緣的生活。開學第一天，羽音騎[[腳踏車|腳踏車]]吃力地爬上坡道時，她遇見了騎著摩托車上學的天野恩紗，為此感到震撼，而開始對摩托車有了興趣。之後，羽音與剛好成為同班同學的恩紗加入了摩托車社，並打算考取摩托車駕照、過著能騎摩托車上學的生活。

本作從作品名稱、單行本封面到角色設計皆影射漫畫《[[K-ON！輕音部|K-ON！輕音部]]》。

== 登場角色 ==
=== 丘乃上女子高等學校 ===
下記角色之年級以初登場時為準。
; {{nihongo|佐倉羽音|佐倉 羽音（さくら はね）|聲：[[上田麗奈|上田麗奈]]}}
: 就讀丘乃上女子高等學校一年A班。個性非常悠然自得。原本毫無摩托車相關知識，甚至不知道[[駕照|駕照]]是什麼。在第一天騎腳踏車上學時遇見騎摩托車的恩紗後，開始被摩托車吸引，並決定取得駕照。
: 名字由来是[[本田技研工業|本田]]的摩托車正式代理店「WING店」和羽翼標誌。
; {{nihongo|天野恩紗|天野 恩紗（あまの おんさ）|聲：[[內山夕實|內山夕實]]}}
: 羽音的同學。有著一頭豎起的自然捲髮，並用緞帶綁起來的女孩子，被凜稱做「蓬蓬頭」。家中經營名為『微笑第一』的中古車行，店名的日文原意是拼裝車。有豐富的摩托車相關知識，亦懂得修繕摩托車。
; {{nihongo|鈴乃木凜|鈴乃木 凜（すずのき りん）|聲：[[東山奈央|東山奈央]]}}
: 羽音的同學。有著一頭金色長髮並綁成雙馬尾，是「鈴菌」（[[鈴木公司|鈴木]]牌機車偏執狂）。屁股上有S的烙印，是因為摔落到火燙的鈴木（SUZUKI）牌車輛廠牌標誌上而印上的。
; {{nihongo|來夢學姊|来夢先輩（らいむせんぱい）|}}
: 包括全名在內的資料皆不詳，學生證上的名字「川崎來夢」是多津子以其所騎摩托車車廠取名的。丘乃上女子高等學校摩托車社的學姊，總是戴著一頂全罩式的SIMPSON白色安全帽，並且[[無口|不發一語]]。是目前登場的角色中駕駛技術最好的角色。然而也有一個弱點，對於川崎從未製造過的[[速克達|速克達]]難以操駕（文化祭車賽用的{{tsl|ja|カワサキ・エプシロン|EPSILON}}雖是川崎旗下車種，實為鈴木代工生產）。早年曾與多津子和明菜等人共組車隊參賽，擔任車手。
: 角色藍本似乎是[[英國廣播公司|BBC]]著名汽車節目《[[Top_Gear|Top Gear]]》的專屬試車手「[[试替哥|The Stig]]」。名字由來是川崎的標準色「萊姆綠」的諧音。
; {{nihongo|三之輪聖|三ノ輪 聖（みのわ ひじり）|聲：[[山口立花子|山口立花子]]}}
: 一年B班的學生，有著一頭黑色長直髮並戴著眼鏡。三之輪集團的千金，沒有駕照，所以平常由早川管家接送，直到首次考取駕照為止。由於從未拿過比筷子重的東西，初期取得駕照的方式是在東南亞國家「買」駕照再回國辦理「{{tsl|ja|日本の運転免許#海外での免許取得とその切替|外免切替}}」，被恩紗指摘後才正式考取[[摩托化自行車|原付]]（50cc）駕照。後來與千雨同時在駕訓班考普通自動二輪車（126～400cc）駕照時曾指導千雨。名字的由來是「[[三輪車|三輪車]]」。
; {{nihongo|佐倉由女|佐倉 由女（さくら ゆめ）|聲：[[田所梓|田所梓]]}}
: 比羽音小一歲的妹妹。比姐姐更具有常識也更成熟。名字由來是本田正式代理店「DREAM店」。
; {{nihongo|中野千雨|中野 千雨（なかの ちさめ）|聲：[[木户衣吹|木户衣吹]]}}
: 由女的同學，是摩托車賽車手，由於身高不夠（144.8公分），加上長期操駕賽車所養成的習慣，沒辦法騎一般的摩托車。認為摩托車不應該騎到馬路上。
