== 簡介 ==
{{Plot_style|time=2018-10-07T17:24:05+00:00}}
在線上遊戲公司工作的29歲的程式設計師鈴木一郎，遊戲角色名「佐藤」，某天在爆肝工作後一覺醒來，發現身處荒野（龍之谷），而且身體變回15歲時的年輕模樣。一大群從未見過的怪物逼近眼前，使用遊戲介面發了3發流星雨，龍之谷的所有怪物全滅，從等級1瞬間等級到達310級<ref>{{cite web|title=【試閱】爽爽開外掛養眼兼養肝《爆肝工程師的異世界狂想曲》|url=http://gnn.gamer.com.tw/9/113529.html|website=巴哈姆特|publisher=GNN新聞|accessdate=2015-04-15|language=zh-tw}}</ref>。

== 用語 ==
; 龍族
: 龍神的眷族，以龍神「阿空加古拉」為頂點的世界最強種族，其中按照階級分為天龍、古龍、成龍、下級龍。龍族生性懶散因此很喜歡睡眠，此外因為強大的關係而喜好戰鬥，無論對手是誰只要發出挑戰就會欣喜的接受，但是因為力量太強所以一開始戰鬥就很容易造成大規模的破壞。
: 龍最強的攻擊莫過於龍的吐息與龍牙的啃咬，比起龍的魔法造成的傷害，龍的吐息甚至能夠焚燒大地，而龍之牙則能夠貫穿萬物，無論是神授的聖具還是魔王都能一擊貫穿最強之矛。

; 魔族
: 魔神的眷族，作亂世界的種族，以魔神為頂點分為上、中、下三個等級，只能以魔法或是魔法道具才能造成傷害。

; 貨幣
: 金幣1枚=銀幣5枚=大銅幣25枚=銅幣100枚=賤幣500枚

; 源泉
: 能夠自然湧出丰沛魔力的地點。源泉的支配者能夠使用仪式魔法，调整领地內的氣候和架設保護都市的結界。
: 根據大小和性質有『精灵堆』、『魔物堆』、『都市核』、『迷宫核』等不同類型。

; 希嘉八劍
: 希嘉王國最強聖騎士首席的八名劍士獲贈的稱號。

; 魔王信徒
: 魔王的信奉者，打著「自由」作為旗號在各地進行活動。
: 有『自由之翼』、『自由之風』、『自由之光』等各個組織在不同國家活動。

;{{nihongo|越後屋|えちごや}}
:佐藤在希嘉王國王都創立的商會，在希嘉王國各地及其他王國均有分店。從事各項物品之買賣及製作，同時也是佐藤個人的情報網路。
