'''寒山'''，生卒年不详，約為[[唐玄宗|唐玄宗]]至[[唐代宗|唐代宗]]間。<ref>[http://sou-yun.com/PoemIndex.aspx?dynasty=Tang&author=%E5%AF%92%E5%B1%B1]《唐诗大辞典 修订本》</ref>[[长安|长安]]人（今[[西安|西安]]人），[[唐朝|唐朝]]著名[[诗僧|诗僧]]。{{noteTag|1= [[宋朝|宋朝]][[李防|李防]]《[[太平广记|太平广记]]》卷五《[[仙传拾遗|仙传拾遗]]》： “寒山子者，不知其名氏，曾于[[贞观|贞观]]（627年-649年）中或[[大历|大历]]（766年-779年）中，隐居[[浙江|浙江]][[天台|天台]][[翠屏山|翠屏山]]。其山深邃，当暑有雪，亦名寒岩，因自号寒山子。无人确知其来历，寒山子和[[拾得|拾得]]是[[国清寺|国清寺]]僧厨的[[苦行僧|苦行僧]]，面貌枯瘁，以桦木为冠，穿一双大木屐，说话疯疯癫癫，但寻思其意，又都合于佛理。他且笑且歌，常独自一人从寒岩隐居地来到七十里外的[[国清寺|国清寺]]。”}} <ref name=" xjsa" >{{cite web |url=http://www.hanshansi.org/download/zx/200903/200903f022.htm |title= 从《寒山诗》看寒山子生平|accessdate= |work=肖太国|publisher=寒山诗 }}</ref>寒山、[[拾得|拾得]]、[[丰干|丰干]]一起隐居于[[天台山_(浙江)|天台山]][[国清寺|国清寺]]，被誉为“[[国清三隐|国清三隐]]”。

== 生平 ==
=== 出家前 ===
寒山的祖籍为[[陕西|陕西]][[咸阳|咸阳]]，出生于[[长安|长安]]，而《全唐诗》则否认了此说法。{{noteTag|1=《[[全唐诗|全唐诗]]》第806卷：“寒山子，不知何许人。居[[天台|天台]][[唐兴县|唐兴县]][[寒岩|寒岩]]，时往还[[国清寺|国清寺]]，以桦皮为冠，布裘敞履，或长廊唱咏，或村野歌啸，人莫识之。”。 }} <ref name=" xjsa"  /> 他乐观豁达，雅好游历，喜爱自由自在的生活。{{noteTag|1=《寒山诗注》：“忆昔遇逢处，人间逐胜游。乐山登万仞，爱水泛千舟。” }} <ref name=" xjsa"  /> 由于多次参加[[科举|科举]]考试不中第，{{noteTag|1=《寒山诗注》：嫌身不得官。 }}加上生性孤僻，喜欢独居，最终导致了他出家{{noteTag|1=《寒山诗注》：自乐平生道，烟萝石洞间。 }}。<ref name=" xjsa"  /> 

=== 出家 ===
寒山出家后，来到[[天台山_(浙江)|天台山]]，隐居于[[寒岩|寒岩]]的[[丹丘山|丹丘山]]，{{noteTag|1=《寒山诗注》：丹丘迥耸与云齐。 }} 其名字“寒山子”也因此而得名。<ref name=" xjsa"  />

== 文学成就 ==
寒山的思想融合了[[儒家|儒家]]、[[道家|道家]]和[[佛家|佛家]]，{{noteTag|1=《寒山诗注》：读书兼咏史，炼药空求仙。 }}知识渊博和广泛。<ref name=" xjsa"  /> 

寒山的诗作分为：劝世诗、玄境诗、咏物诗。<ref>{{cite web |url= http://www.nanputuo.com/nptxy/html/200711/1814325131973.html|title= 唐代的诗僧与僧诗|accessdate= 2007年|work=|publisher= 南普陀网}}</ref> 寒山诗作中的思想一部分是描绘景物、揭露社会黑暗、思怀亲友为主，另一部分则宣传佛教思想，佛法精深，耐人寻味和把玩。<ref name=" xjsa"  />
== 影响 ==
寒山的诗作生前默默无闻，死后却闻名世界。[[日本|日本]]、[[美国|美国]]的学者竞相模仿和解读。<ref name="snxsax" />

=== 日本 ===
==== 文学 ====
日本“俳圣”[[松尾芭蕉|松尾芭蕉]]的文学风格深受寒山的影响，他自云：“李杜尝心酒，寒山啜法粥。故其句见之遥，闻之远。”而后[[坪内逍遥|坪内逍遥]]、[[森鸥外|森鸥外]]、[[夏目漱石|夏目漱石]]、[[芥川龙之介|芥川龙之介]]、[[冈本可能子|冈本可能子]]、[[安西冬卫|安西冬卫]]、[[井伏鳟二|井伏鳟二]]等作家都备受其影响。<ref name=" xmles">{{cite web|url=http://nf.nfdaily.cn/nfdsb/content/2011-12/05/content_34365689.htm|title=深刻影响日本的寒山诗|accessdate=2011-12-05|work=|publisher=南方都市报网|deadurl=yes|archiveurl=https://web.archive.org/web/20111206034132/http://nf.nfdaily.cn/nfdsb/content/2011-12/05/content_34365689.htm|archivedate=2011-12-06}}</ref>
==== 艺术 ====
[[File:Portrait_of_Kanzan.jpg|缩略图]]
从[[镰仓|镰仓]]时期到[[江户|江户]]时期，[[一山一宁|一山一宁]]、[[可翁宗然|可翁宗然]]、[[足利义持|足利义持]]、[[灵彩|灵彩]]、[[周文|周文]]、[[墨溪|墨溪]]、[[雪舟|雪舟]]、[[雪村友梅|雪村友梅]]、[[宗达|宗达]]、[[池大雅|池大雅]]等画过《寒山图》；[[与谢芜村|与谢芜村]]、[[宋紫石|宋紫石]]、[[圆山应举|圆山应举]]、[[长泽芦雪|长泽芦雪]]、[[若冲|若冲]]、[[曾我萧白|曾我萧白]]、[[仙崖义梵|仙崖义梵]]、[[苏山乔玄|苏山乔玄]]等画过《寒山拾得图》。<ref name=" xmles" />

=== 美国 ===
20世纪50年代寒山诗在[[美国|美国]]被“[[垮掉的一代|垮掉的一代]]”奉为鼻祖。<ref name="sxns " >{{cite web |url= http://www.guoxue.com/www/xsxx/txt.asp?id=1124|title=当代美国、日本“寒山热”原因探索 |accessdate= 2001-12-08|work=|publisher=国学网 }}</ref>

== 评价 ==
*寒山自谓：“下愚读我诗，不解却嗤诮。中庸读我诗，思量云甚要。上贤读我诗，把着满面笑。杨修见幼妇，一览便知妙。若能会我诗，真是[[如来|如来]]母。”<ref name="hanshshi " >{{cite book |language =zh-hant|author=寒山|title=《寒山诗注》|year=2000-03-01|publisher=中华书局|location=北京|pages=|isbn=9787101016451 }}</ref>
*宋人将“[[陶渊明|陶渊明]]、[[李白|李白]]、[[杜甫|杜甫]]、[[白居易|白居易]]、寒山、[[苏轼|苏轼]]、[[黄庭坚|黄庭坚]]、[[陈无己|陈无己]]”的诗作并列为“[[八老诗|八老诗]]”。<ref name="snxsax" >{{cite web |url=http://news.eastday.com/epublish/gb/paper10/20001030/class001000021/hwz229754.htm |title=寂寞寒山路 |accessdate= |work=文匪报|publisher=东方新闻网 }}</ref>
*[[宋朝|宋朝]][[王安石|王安石]]《拟寒山拾得十九首》：“失志难作福，得势易造罪。苦即念快乐，乐即生贪爱。无苦亦无乐，无明亦无昧。不属三界中，亦非三界外。”
*[[清朝|清朝]][[王士祯|王士祯]]《[[居易录|居易录]]》： “清淡冲朕，唐人所不好；机趣横溢，韵度自高，在[[皎然|皎然]]上道显下”。
*[[日本|日本]][[汉学家|汉学家]][[闾丘胤|闾丘胤]]：“寒山诗机趣横溢，韵度极高，在皎然上道显下，是木铎者所潜心。”<ref name="sxns " />
