'''綜合格鬥'''（{{lang-en|'''Mixed Martial Arts'''}}，[[縮寫|縮寫]]：'''MMA'''，又稱'''總合格鬥技'''、'''無限制格鬥、混合格鬥'''），是一種允許運用多種不同武術的搏擊運動。此類格鬥運動最初被稱為Vale Tudo（巴西葡萄牙語，意即「無限制」），後來改稱為MMA（Mixed Martial Arts），著重於競技運動這一點來宣傳，避免給人過多暴力與血腥的負面形象。

MMA 它本身不是一種武術，混合武術的意思是把各種武術混合在一起比賽的模式。

混合武術格鬥是一允許雙方選手使用打擊技、摔技與地板技的比賽。因此，任何一種的武術在這一種極為靈活的規則之下，都可以自由發揮，不受限制。

混合武術格鬥在美國的出現，因为美国种种的 搏击流派 和 格斗技法，认为自己搏击厉害，在60年代的，不断的 相互跨界擂台 才产生的，1963年有录像的第一场美国类似MMA擂台赛，一人戴分指拳套，是 gene lebell VS milo savage 的跨界擂台赛，就可以看到其雏形，在1993年所創立的[[UFC|UFC]]，更是走上了正规，原本比賽的目的，是想要找出最有效率可以擊敗敵人的武術。但是最後隨著不斷的比賽經驗，創造出了現在的 MMA FIGHTING。混合武術的名字，是由Jeff Blatnick所命名。他是一位[[古典式摔跤|古典式摔跤]]的[[奧運|奧運]]金牌選手。

在混合武術格鬥興起的早期非常強調比賽儘量接近實戰，因此比賽規則也儘量簡單，力求無規則，這使得比賽非常暴力血腥。比如早期的[[UFC|UFC]]比賽，選手不帶任何護具，比賽也沒有時間限制，勝負以一方認輸或失去反擊能力判定，經常有選手在賽場上受傷甚至失去知覺。隨著混合武術的發展，各類比賽也逐步制定了比較詳盡的規則，以保證比賽的公平和選手的健康。

== MMA格鬥 ==
構成MMA格鬥主要分為
「站立戰」(打擊技 、摔技）與
「地戰」（寝技）兩部份。

打擊技， 指以站立進行打擊的武術，例如：
* [[西洋拳|西洋拳]]
* [[散手|散打]]
* [[泰拳|泰拳]]
* [[跆拳道|跆拳道]]
* [[踢拳|踢拳]]
* [[法國踢腿術|法國踢腿術]]
* [[空手道|空手道]]
* [[以色列搏擊防身術|以色列搏擊防身術]]
* [[南洋防身術|南洋防身術]]

投技， 指破壞對手站立平衡使其倒地甚至撞擊地面受到傷害的武術，例如:
* [[柔道|柔道]]
* [[合氣道|合氣道]]
* [[日耳曼擒拿术|日耳曼擒拿术]]
* [[蒙古式摔跤|蒙古式摔跤]]
* [[摔跤|摔跤]]

寢技，指以二人或一人在地上進行纏鬥，以壓制-關節技，絞頸降服對手的武術，例如:
* [[柔術|柔術]]
* [[巴西柔術|巴西柔術]]
* [[德國柔術|德國柔術]]
* [[桑搏|桑搏]]

== 着装要求 ==
常要求男性比赛者除了半裸之外还要穿短裤，大多数运动委员会都要求男性比赛者在他们的下身穿着腹股沟保护装置，女性比赛者穿着短裤和运动胸罩或其他类似舒适的上衣。

== 打斗区域 ==
根据混合武术统一规则，MMA比赛或表演可以在环形或围栏区域举行。围栏区域可以是圆形的或至少是六边形。笼子各有所不同：有些用普通网取代金属栅栏，有些用八角形。

== 安全 ==
自1993年第一届终极格斗冠军赛以来，综合格斗比赛发生了巨大变化，特别是随着综合格斗统一规则的出台。MMA中发生的伤害数据仍然很少，因此对MMA安全性的担忧仍然存在。
