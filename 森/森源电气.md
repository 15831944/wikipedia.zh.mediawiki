'''河南森源电气股份有限公司'''（{{szse|002358|name=森源电气}}）是一家生产中、低压开关成套设备、[[断路器|断路器]]元件及其配件，中压户内隔离开关和SAPF有源滤波装置等电力电子与智能电器设备的[[上市公司|上市公司]]。位于[[长葛市|长葛市]]人民路北段。<ref>{{cite web |url=http://www.hnsydq.com/invest170-detail185.html |language= |title=公司资料 |publisher=森源电气 |date= |accessdate=2012年11月 |deadurl=yes |archiveurl=https://web.archive.org/web/20120117055530/http://www.hnsydq.com/invest170-detail185.html |archivedate=2012-01-17 }}</ref>

== 简介 ==
公司成立于2000年10月30日，2010年2月10日在[[深圳证券交易所|深圳证券交易所]][[上市|上市]]。<ref>[http://finance.sina.com.cn/realstock/company/sz002358/nc.shtml 森源电气公司简况]</ref>

作为一家国家重点高新技术企业，公司是中国电器工业协会高压开关分会常务理事单位、中国电器工业协会电力电子分会副理事长单位、全国电力电子百强企业。拥有[[博士后|博士后]]科研工作站、河南省中压输配电装置工程技术研究中心、省级企业技术中心。自1998年来先后通过[[ISO9001|ISO9001]]（质量）、[[ISO14001|ISO14001]]（环境）、[[OHSAS18001|OHSAS18001]]（安全）国际标准认证<ref name= "简介"/>。

2000年以来曾承担的重点项目有：国家重大电力装备自主化专项、国家新型电力电子器件产业化专项、国家火炬计划项目、河南省高新技术产业化重点项目、河南省重大科技攻关项目和杰出人才创新基金项目。共获得250多项[[专利|专利]]授权。<ref name="简介">{{cite web |url=http://www.hnsydq.com/about-detail2.html |language= |title=简介 |publisher=森源电气 |date= |accessdate=2012年11月 |deadurl=yes |archiveurl=https://web.archive.org/web/20120827200217/http://www.hnsydq.com/about-detail2.html |archivedate=2012-08-27 }}</ref>
