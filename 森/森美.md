'''森美'''（'''Sammy Leung Chi Kin'''，{{bd|1973年|12月3日|catIdx=L梁}}），本名'''梁志健'''，[[香港|香港]][[無綫電視|無綫電視]]（TVB）基本藝人合約藝員，兼任[[香港商業電台|香港商業電台]][[唱片騎師|唱片騎師]]，組合[[森美旅行團|森美旅行團]]之隊長。

在[[香港|香港]]和[[廣東|廣東]]等[[珠三角|珠三角]]地區，森美與[[小儀|小儀]]早已是街知巷聞的[[金牌司儀|金牌司儀]]和[[名嘴|名嘴]][[主持|主持]]，兩人近年更加進軍[[電視|電視]]業和大螢幕，成功擔任香港多部[[電影|電影]]的主角，是[[香港|香港]]影、視及播的三棲藝人。

==履歷==
[[File:SammyLeung2007.jpg|250px]]
森美於[[葵盛西邨|葵盛西邨]]長大，現居於[[九龍塘|九龍塘]][[廣播道|廣播道]]，曾就讀聖方濟愛德小學（已於1997年遷往[[聖方濟各書院|聖方濟各書院]]舊址），亦曾就讀名校[[聖士提反書院|聖士提反書院]]，他在接受電視節目《[[東张西望|東张西望]]》訪問時，表示曾在[[聖士提反書院|聖士提反書院]]、[[喇沙書院|喇沙書院]]和[[新亞書院|新亞書院]]就讀，以會考25分（1A5B）畢業。森美亦曾在節目中封中大校友，及喇沙書院中六同學[[孫耀威|孫耀威]]跟自己為「中大雙雄」。1994年於[[新亞書院|新亞書院]]畢業前參加商業電台「[[叱吒殺戮新大陸|叱吒殺戮新大陸]]」（DJ選舉比賽）而入行，1996年畢業於[[香港中文大學|香港中文大學]]新聞與傳播學院。

森美剛入行時，[[香港商業電台|香港商業電台]]幕後人員「媽咪」把他的英文名Sammy讀成森美，從此森美就成為他的藝名<ref name="My Name is 邦 2009-08-13">[[J2|J2]]，My Name is 邦 2009-08-13 [http://mytv.tvb.com/variety/nib/8562/48#page-1] {{webarchive|url=https://web.archive.org/web/20090818150552/http://mytv.tvb.com/variety/nib/8562/48|date=2009-08-18}}</ref>。

森美首個節目為與[[潘紹聰|潘紹聰]]主持的《[[青蘋果與紅提子|青蘋果與紅提子]]》，之後主持週日節目《[[恐怖份子_(電台節目)|恐怖份子]]》，及星期六、日節目《末日嬉戲》, 幾年後與[[阮小儀|阮小儀]]（小儀）共同主持《[[森美變態樂園|森美變態樂園]]》<ref name="My Name is 邦 2009-08-12">[[J2|J2]]，My Name is 邦 2009-08-12 {{cite web |url=http://mytv.tvb.com/variety/nib/8525 |title=存档副本 |accessdate=2009-08-21 |deadurl=yes |archiveurl=https://web.archive.org/web/20090822215423/http://mytv.tvb.com/variety/nib/8525 |archivedate=2009-08-22 }}</ref>，其後二人一直合作主持電台節目，如《[[森美小儀_dot_dot_dot|森美小儀 dot dot dot]]》、《[[森美小儀山寨王|森美小儀山寨王]]》、《[[架勢堂|架勢堂]]》及《[[早霸王|早霸王]]》。2001年開始與[[詹志文|Leo]]、[[蘇耀宗|細so]]、[[孔慶慇|Marco]]、以及後期加入[[陳強|陳強]]主持《[[森美移動|森美移動]]》。後期森美移動成員各自在自己的節目中穩定發展，2009年10月正式解散。

由於森美的粉絲眾多，與小儀於2000年成立 [[I_Love_SK|I Love SK]]（森美托小儀所），讓各位fans參加森美小儀各種活動，包括BBQ和一般聚會、探訪慈善機構、賣旗籌款等。最矚目的是一年一度的森美小儀生日會，通常於11月尾至12月初舉行,（Fans Club已於2011年於森美意願下停止接收新會員）。

森美醉心於演藝事業，努力嘗試在各娛樂方面發展。由2001年起，與小儀合作成立[[森美小儀歌劇團|森美小儀歌劇團]]，至少每年舉辦一次舞台演出，並邀請商業電台眾[[DJ|DJ]]以及多位藝人參與。2008年以個人身份首次在舞台上嘗試棟篤笑表演，演出名為《金牌司儀》。及後於2011年舉行了第二次個人演出《情聖》並將於2013年暑假舉行第三次個人演出《森美要滾》。除了每年親自參與舞台劇編劇、導演以及演出等工作外，森美更親自操刀設計各種歌劇團紀念品，包括T-Shirt、帽子、錶、環保袋等。

森美甚少提及妻子[[陳雅嫻|陳雅嫻]]（Veronica），因為她是圈外人，這樣做是尊重她的意見，使外間誤解森美隠瞞已婚事實。但他早於2003年已在[[森小報|森小報]]（森美小儀於商台的網誌）透露自己剛為人父，及在節目談及新生命可為生活帶來喜悅。其後更於[[明報|明報]]再次公開自己有妻有兒，更說：「未結婚又點生仔呢？唔可以咁做。」他與妻子認識於「叱吒殺戮新大陸」。她曾是[[童星|童星]]，與[[許冠傑|許冠傑]]在1984年拍《[[全家福|全家福]]》，並曾於有線兒童台及商台任職。

森美畢業於[[香港中文大學|香港中文大學]]新聞及傳播學院，自2001年起森美獲邀回歸母校任教電台製作，成為兼職講師達5年之久，直至2006年請辭，及後致力於電台及幕前發展。

2004年簽約[[金牌娛樂|金牌娛樂]]成為旗下藝人作多方面發展，2004年首次擔任男主角演出電影《[[甜絲絲|甜絲絲]]》，（其實2001年亦曾擔當《[[4x100_水著份子|4x100 水著份子]]》男主角，但並未正式在電影院公演，只發行VCD及DVD）。森美近年更參與非喜劇電影演出，如《[[性工作者2_我不賣身·我賣子宮|我不賣身 我賣子宮]]》及2007年的亞洲電影節開幕電影《[[七月好風|七月好風]]》等。且亦於2011年參與福音電影《翻生奇兵》擔任男主角。

2006年擔任[[無綫電視|無綫電視]][[國際華裔小姐競選|國際華裔小姐競選]]司儀，同年3月與電台拍檔[[阮小儀|阮小儀]]主持綜藝節目《[[15/16|15/16]]》，憑此節目於2006年度[[無綫電視|無綫電視]][[2006年度萬千星輝頒獎典禮|萬千星輝頒獎典禮]]中勇奪得「最佳節目主持」，完成他其中一個夢想<ref name="My Name is 邦 2009-08-13"/>。 同年森美在《[[架勢堂|架勢堂]]》節目內舉行的遊戲環節《香港十大》選舉「我最想非禮的香港女藝人」，由於標題過於出位，被婦女團體指侮辱女性。事後他於商台及商台網上公開道歉（參見[[架勢堂#架勢堂「我最想非禮的香港女藝人」投票風波|架勢堂「我最想非禮的香港女藝人」投票風波]]）。

2009年2月4日11時44分，他於架勢堂節目中呼籲聽眾於[[維基百科|維基百科]]中加入「森美是個靚仔」以及「森美真人靚過上鏡」，是繼[[卓韻芝|卓韻芝]]公開談論其維基頁面應該註明為「最漂亮的女藝人」後的另一位商台主持呼籲聽眾修改其維基頁面。當時有人響應加入該些內容，最後因為不斷的[[Wikipedia:破壞|破壞]]，該頁面被[[Wikipedia:半保護|半保護]]。

2009年森美替[[尚方文化|尚方文化]]出版社出任社長（現已離職，只在書展期間擔任），並替多位藝人包括[[王貽興|王貽興]]、[[小儀|小儀]]、[[曾志偉|曾志偉]]、[[林以諾|林以諾]]牧師、少年食神Michael和[[吳日言|吳日言]]出版著作。森美更曾為[[成報|成報]]、[[新Monday|新Monday]]、[[Auto_Express|Auto Express]]、[[Milk|Milk]]和澳門出版的[[力報|力報]]等等撰寫專欄<ref name="My903 交流區">[[My903_forum|My903 forum]]，早霸王  [http://forum.my903.com/viewthread.php?tid=127670&extra=page%3D4]{{Dead link|date=2018年9月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。而現時正在撰寫的, 包括有[[星島日報|星島日報]]的「公子日記」專欄（星期一至六刊出）、[[明報|明報]]的「森底話」專欄（逢星期二刊出）、[[Cosmopolitan|Cosmopolitan]]雜誌專欄以及[[Beauty_Exchange|Beauty Exchange]]網站專欄等。

2009年8月至10月期間，森美與[[陳志雲|陳志雲]]分別代表[[香港商業電台|香港商業電台]]及[[無綫電視|無綫電視]][[J2台|J2台]]跨媒體合作節目《[[十人巷|十人巷]]》，希望透過節目培訓一個有多種技能的藝人，令香港娛樂圈更加豐富<ref name="My Name is 邦 2009-08-13"/>。

[[File:曾俊華_VS_森美.jpg|thumb]][[曾俊華|曾俊華]]合照]]
2015年，森美替無綫電視的直播節目《[[超強選擇一分鐘|超強選擇一分鐘]]》當主持，此節目設立全民網上投票機制，但首集直播時於全民網上投票的實行上出現機件故障，結果森美的臨場執生技巧備受讚賞。森美同時擔任[[香港足球總會|香港足球總會]]市務及傳訊委員會成員。<ref>[http://www.hkfa.com/ch/load_page/5/ 香港足總網頁] {{Wayback|url=http://www.hkfa.com/ch/load_page/5/ |date=20160628212248 }} 香港足總網頁</ref>

2016年，森美獲邀擔任韓星[[宋仲基|宋仲基]]、[[BIGBANG|BIGBANG]]、[[朴寶劍|朴寶劍]]及[[李棟旭|李棟旭]]亞洲巡迴見面會香港站的主持人。

2017年，森美獲邀擔任韓星[[孔劉|孔劉]]及[[Wanna_One|Wanna One]]香港站見面會的主持人。

2017年11月，甚少公開談論家人的森美日前訪問時，罕有地談及父母和剛赴英國升學的兒子，而訪問中亦多次落淚。他憶述身為的士司機的父親載他到喇沙書院面試的難忘片段，當日下滂沱大雨，的士竟在途中爆胎，擔心森美趕不及面試的父親情急下冒雨換車胎，頭頂紅膠袋，推車胎往附近的油站充氣，望多送兒子一段路程。而他又指兒子現居的地方位於海邊，而他當年入讀的寄宿學校亦是位處海邊，海邊正正連繫他、父親與兒子這三代人的關係，送兒子到外國升學令他體會到當時父親送他寄宿的心情<ref>[http://bka.mpweekly.com/focus/20171118-91078 森美淚謝雙親栽培]</ref>。

2017年12月，於喇沙書院畢業的森美接受訪問時表示自己在屋邨長大，中三那年和三個差不多背景的同學一起考入喇沙，最初擔心會和同學相處不來，但漸漸發現，不管你背景如何，貧富也好、原校升讀也好，大家都能打成一片<ref>[https://bkb.mpweekly.com/cu0001/20171226-62435 喇沙書院 ― 兄弟情常在]</ref>。
