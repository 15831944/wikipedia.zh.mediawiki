'''森本敏'''（{{lang-ja|'''森本敏'''}}，{{bd|1941年|3月15日|||catIdx= Morimoto Satoshi}}），[[日本|日本]][[政治人物|政治人物]]、第11届日本[[防卫大臣|防卫大臣]]，[[东京府|东京府]][[东京市|东京市]]出生，毕业于[[防卫大学校|防卫大学校]]，並曾任[[航空自衛隊|航空自衛隊]]佐官（校級軍官）。森本是日本第二位自衛隊出身的國防部門首長（第一位為[[陸上自衛隊|陸上自衛隊]]退役的前防衛廳長[[中谷元|中谷元]]）<ref name="空權">{{Citation |title=空權的理論與實踐：以日本航空自衛隊為例 |author=許哲睿 |publisher=[[淡江大學|淡江大學]]國際事務與戰略研究所碩士班 |date=2015 |page=頁176、173}}</ref>，也是防衛大臣之職位設立後，首位非國會議員出身的防相<ref>{{cite web |url= http://gb.cri.cn/27824/2012/06/07/5951s3717938.htm|title=森本敏 —首位来自民间的日本防卫大臣 |accessdate=2012-06-07  |publisher= 国际在线|work= |language =zh-cn}}</ref>。

== 生平 ==
1941年，森本敏出生在[[大日本帝国|大日本帝国]][[东京府|东京府]][[东京市|东京市]]，早年，森本敏在丰中市立第六中学校、大阪府立丰中高等学校和防卫大学校读书。<ref name="fox " >{{cite web |url=http://news.ifeng.com/mil/1/detail_2012_06/07/15124372_0.shtml |title=日新防卫大臣森本敏获多数人认可 安保理念右倾 |accessdate=2012年6月7日  |publisher=凤凰网 |work= |language =zh-cn}}</ref>

1965年（昭和40年），森本敏加入日本航空自卫队。<ref name="fox " />曾任日本驻尼日利亚大使馆[[參贊|參贊]]，后一度担任日本外务省安全保障政策室室长。<ref name="fox " />森本敏最後以三等空佐（空軍少校）階級退役，在外务省工作时，主要负责情报工作、安全保障和日美同盟防卫合作。<ref name="fox " /> 

2009年，森本敏担任防卫大臣辅佐官。<ref name="fox " /> 

2012年，森本敏担任日本防卫大臣。<ref name="fox " />

== 思想 ==
森本敏既是大学教授出身的文职人员，也是日本自卫队出身的武职人员，能文能武。<ref name="fish " /> 他主张美日同盟安保为基础，以[[中华人民共和国|中华人民共和国]]为假想敌，面对日益崛起的中华人民共和国，必须修改[[宪法|宪法]]，恢复交战权，修改自卫队防卫章程，可使用交战权。<ref name="fish " >{{cite web |url= http://news.ifeng.com/opinion/zhuanlan/yutianren/detail_2012_06/05/15062939_0.shtml?_from_ralated|title=森本敏出任日防卫大臣意义重大 |accessdate=2012年6月5日  |publisher=凤凰网 |work= |language =zh-cn}}</ref>
