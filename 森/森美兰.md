'''森美兰'''（[[马来语|马来语]]：{{lang|ms|Negeri Sembilan}}，[[爪夷文|爪夷文]]：{{rtl-lang|ar|نڬري سمبيلن}}，在[[马来语|马来语]]是“第九个州”的意思，通称'''九州'''）是[[马来西亚|马来西亚]]位于[[马来半岛|马来半岛]]西海岸的一个州属。此州实際上沒有中文名，仅因有[[华人|华人]]居住这裡才得到“Sembilan”中的中文譯名。

森美兰州首府是[[芙蓉市|芙蓉]]，州王宫所在地位于神安池（马来语：{{lang|ms|Seri Menanti}}）。有别于[[马来半岛|马来半岛]]其他州属君主的世袭制，森美兰州的君主称为[[最高统治者|最高统治者]]（也称“严端”），而并非苏丹。最高统治者是由州内四大部落即[[双溪芙蓉|双溪芙蓉]]、林茂、[[日叻务|日叻务]]和[[柔河|柔河]]的酋长选出的。

== 歷史 ==
来自[[苏门答腊|苏门答腊]]的[[米南佳保人|米南佳保人]]在15世纪在今森美兰地區建立城邦，受到[[马六甲王朝|马六甲王朝]]和随后的[[柔佛王朝|柔佛王朝]]的保护。18世纪起，来自印尼群岛的[[武吉斯人|武吉斯人]]开始进犯马来半岛。由于柔佛王朝的势力逐渐式微，米南加保人决定回苏门答腊寻求援助。在这期间，各米南加保城邦相互兼并，自立为王，内战连绵。

英国人在19世纪占领[[槟城|槟城]]、[[马六甲|马六甲]]和[[新加坡|新加坡]]后，垂涎马来半岛的矿产及天然资源，便开始插手干预各马来州属的政治。

1874年，[[拿督万达|拿督万达]]（Dato' Bandar）为反对[[拿督克拉那|拿督克拉那]]（Dato' Kelana）征收锡米税而爆发了内战。双方在[[宁宜河|宁宜河]]（{{lang|ms|Sungai Linggi}}）爆发激烈的争斗。宁宜河是芙蓉主要的贸易水路，通往马六甲的贸易码头，许多人都乘船通过，税收颇为可观。拿督万达倚赖华族矿工的支持占上风，拿督克拉那为了巩固本身经济利益，便去寻求英国人协助。拿督万达最后虽然联合其他马来贵族与农民进行抗争，但仍被英国人打败。英国殖民当局于是扶持拿督克拉那为芙蓉的统治者，并委派本身的白人[[助理参政司|助理参政司]]掌管政务。

内战结束后，拿督克拉那立盟约接受英国的保护。拿督克拉那顺利地当上芙蓉区酋长的位子，而英国有权利索取宁宜河一半的税金，并委派参政司协助英国殖民政府管理该区。英国的势力逐渐扩张，最后在英国殖民地[[参政司|参政司]]下统一了全州。[[马来联邦|马来联邦]]随后成立，森美兰州，连同[[雪兰莪|雪兰莪]]、[[彭亨|彭亨]]跟[[霹雳|霹雳]]被迫加入，沦为英国的保护国。森美兰政府被迫割让[[南宁|南宁]]、[[士毛月|士毛月]]、[[昔加末|昔加末]]等地予临近州属。

[[二战|二战]]时期，日军占领马来半岛三年零八个月，并在森美兰州[[知知港|知知港]]屠杀了当地百姓一千四百七十四人，引起国际震惊及关注。

战后，1948年森美兰州加入[[马来亚联合邦|马来亚联合邦]]，并以马来亚身份独立。

==首長==
森美兰州实行君主立宪制度，现任森美兰最高统治者是 Tuanku Muhriz ibni Almarhum Tuanku Munawir，而最高统治者-{后}-是 Tuanku Aishah Rohani binti Almarhum Tengku Besar Mahmud。

== 人口 ==
大马公民中（970,604人）以马来人土著为主（58.3%），其餘依次為华人（24%）、印度人（15.7%）、其他土著（1.57%）及其他（0.43%）。在包括非大马公民方面中，马来人土著为主（55.8%），其餘依次為华人（23%）、印度人（15%）、其他土著（1.5%）、其他（0.43%）及非大马公民（4.27%）。

大概40%的人口都居住在芙蓉县，使得芙蓉县成為州内人口最密集的区域；其余6成人口则分布在其他县。州内的华人以客家人为主，因为地理环境靠近吉隆坡，大部分芙蓉人皆以粤语交谈。而南部靠近马六甲州和柔佛州的市镇则一般都以华语或福建话沟通。

== 交通 ==
[[File:Seremban_railway_station_(Rawang-Seremban_Line)_(front),_Seremban.jpg|thumb]]
[[芙蓉线|芙蓉线]]是[[马来亚铁路|马来亚铁路公司]]旗下[[KTM通勤铁路|KTM通勤铁路]]中马区服务的其中一条路线，也属于[[陆路公共交通委员会|陆路公共交通委员会]]（SPAD）的大吉隆坡/[[巴生谷综合运输系统|巴生谷综合运输系统]]，路线编号为1，而颜色则为蓝色。该路线途径[[森美兰州|森美兰州]]、[[雪兰莪州|雪兰莪州]]及[[吉隆坡|吉隆坡]]，连接首都吉隆坡和[[芙蓉市|芙蓉]]、[[加影|加影]]、[[冼都|冼都]]等周边市郊城镇，于1995年8月开始投入服务。

森美兰的芙蓉－淡边电动火车双轨工程于2012年8月16日竣工，同时也增加两了全新的列车站，分别位于[[新那旺|新那旺]]（Senawang）和[[双溪芽笃|双溪芽笃]]（Sungai Gadut），并于2011年1月开始运作。森美兰目前沒有民用機場，最临近的机场位于[[吉隆坡国际机场|雪邦国际机场]]。

== 旅遊景點與美食 ==
森美兰较其他州属的旅游景点少，其中有[[神安池博物館|神安池博物館]]、[[芙蓉博物馆|芙蓉博物馆]]、[[乌鲁文都|乌鲁文都]]瀑布（{{lang|ms|Ulu Bendol}}）、[[波德申|波德申]]海滩等等。购物中心也鲜少。现今多数政府部门都迁移去芙蓉新城（Seremban 2），警察、消防局、法庭都坐落在该区。

较热闹的地方是靠近芙蓉新城的奥克兰区（Oakland Commerce Square），那里多数是商业区。州内主要商有芙蓉新城[[永旺|永旺]]商场、Era Square / Terminal 2（坐落在明佫酒家的旁边）、Terminal 1、Seremban Parade等。
