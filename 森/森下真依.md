'''森下真依 '''（平假名：{{lang|ja|もりした まい}}，1997年6月30日－），是一名[[日本|日本]][[艺人|女藝人]]，出生於[[東京都|東京都]]。<ref>{{Cite web |url = http://gacharicspin.com/profile.html |title = Gacharic Spin Official Website PROFILE |language = ja |accessdate = 2018/4/19 }}</ref>主要職業為[[寫真偶像|寫真偶像]]與[[舞者|舞者]]，現於日本女子搖滾團體[[Gacharic_Spin|Gacharic Spin]]任表演者。

== 簡歷 ==
2010年12月，在成為中學生後的第一張DVD《{{lang|ja|すまいる}}》正式發行。2011年7月，發行DVD《{{lang|ja|まい☆ペース}}》，並於隔月首次參加談話活動。2012年6月，發行DVD《{{lang|ja|まい☆ウェイ}}》，此作品是在成為Chu→Boh學園學生會長後的首次攝影作品。

2013年，參與日本女子團體[[Gacharic_Spin|Gacharic Spin]]的「{{lang|ja|ガチャガチャダンサーズ}}」以「{{lang|ja|1号 まい}}」的藝名擔任表演者，於2014年6月7日成為正式成員。
