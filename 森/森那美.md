'''森那美有限公司'''（{{lang-en|Sime Darby Berhad}}；{{myx|4197}}）是一家总部位于[[马来西亚|马来西亚]][[吉隆坡|吉隆坡]]的企业，主营工业设备、汽车、物流及医疗保健。森那美工业部是[[卡特彼勒|卡特彼勒]]全球代理商之一<ref>{{Cite web |title=森那美、森那美种植、森那美产业。该卖哪只？ |url=http://www.sinchew.com.my/node/1721449 |date=2018-01-22 |newspaper=[[星洲日报|星洲日报]] }}</ref>。

森那美于1910年由英籍商人创立，初期卷入英属[[马六甲|马六甲]][[橡胶|橡胶]]种植业，之后多元化进入[[棕油|棕油]]种植业、工业设备贸易、汽车代理商、房地产<ref>{{Cite web |title=Company in focus: Sime Darby bets its kingdom is better off in pieces |url=https://asia.nikkei.com/Economy/Malaysias-Sime-Darby-splits-up-to-raise-equity-value2 |date=2017-03-23 |newspaper=[[日本经济新闻|日本经济新闻]] }}</ref>。2007年，马来西亚三大种植集团，即森那美、金希望（Golden Hope）及牙直利集团（Kumpulan Guthrie），合并为世界种植面积最大的上市企业，合并公司保留森那美名称<ref>{{Cite web |title=森那美集团SIME DARBY BERHAD （Sime，4197） 全球种植面积最大的公司 |url=http://www.bigorangemedia.com/3649 |date=2013-01-05 |newspaper=大橙报 }}</ref>。

2017年11月，公司[[业务分拆|分拆]]为三家各别上市公司：森那美（保留贸易与物流业务），森那美种植和森那美产业<ref name=":0">{{Cite web |title=分拆上市3公司升沉不一　森那美集团市值蒸发26亿 |url=http://www.orientaldaily.com.my/s/222345 |date=2017-11-30 |newspaper=[[东方日报_(马来西亚)|东方日报]] }}</ref>。森那美种植与森那美产业即森那美分拆前旗下种植和房地产业臂膀<ref>{{Cite web |title=
森那美拟分拆业务 种植产业臂膀上市 |url=http://www.theedgemarkets.com/article/%E6%A3%AE%E9%82%A3%E7%BE%8E%E6%8B%9F%E5%88%86%E6%8B%86%E4%B8%9A%E5%8A%A1-%E7%A7%8D%E6%A4%8D%E4%BA%A7%E4%B8%9A%E8%87%82%E8%86%80%E4%B8%8A%E5%B8%82 |date=2017-01-27 |newspaper=The Edge }}</ref>。但分拆後，3家公司當日闭市时的股价总和为8.56[[馬來西亞令吉|令吉]]；相比分拆前掛价8.94令吉，下跌4.25%，或38仙。这也意味著森那美集团的市值，在當日就蒸发了26亿令吉<ref name=":0" />。
