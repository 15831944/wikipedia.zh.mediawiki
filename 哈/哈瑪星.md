'''哈瑪星'''是位於[[台灣|台灣]][[高雄市|高雄市]][[鼓山區|鼓山區]]的一個地名，泛指現今[[五福四路|五福四路]]與[[鐵路平交道|鐵路平交道]]的交會處以南，至鼓山漁港、漁市場，東至[[高雄港車站|高雄港車站]]一帶的[[高雄臨港線|臨港線鐵路]]（已停用），西至[[哨船頭|哨船頭]]東側。曾是高雄的政經中心，臺南廳打狗支廳廳舍、高雄郡役所、高雄街役場、高雄市役所都曾設於此。

== 歷史 ==
[[File:Minato-machi,_Takao.jpg|left]]

哈瑪星原本是海域，[[台灣日治時期|日治時期]]時，[[日本|日本]]當局在高雄建立港口，為了疏濬航道，於是利用淤泥[[填海|填海]]造陸而形成。「哈瑪星」此名稱的由來，是因為當地有兩條濱海[[鐵路|鐵路]]通往商港、漁港和漁市場，[[日語|日語]]稱為「濱線」（{{lang-ja|はません}}，Hamasen），當地居民以[[臺灣話|臺灣話]]稱之為「哈瑪星」（Há-má-seng）。

哈瑪星一帶在日治時期分屬[[壽町_(高雄市)|壽町]]、[[新濱町|新濱町]]、[[湊町|湊町]]等行政區，都是新生地。而從新濱町港邊至渡船頭邊的漁市場有一條專為轉運鮮魚的濱海鐵路，且因該地區的各種行業幾乎皆與港區及濱線具有密不可分的關係，因此後來哈瑪星即泛指今南鼓山地區。

哈瑪星設立的「打狗停車場」（今[[高雄港車站|高雄港車站]]）為縱貫鐵道的終點。附近不但是高雄市政經中心，更是[[高雄港|高雄港]]現代化發展的起源地，曾是當地最繁榮的地區。1917年完工的[[臺南廳|臺南廳]][[打狗支廳|打狗支廳]]廳舍即位於哈瑪星；1920年廢廳改州，打狗支廳廳舍續用為高雄州[[高雄郡|高雄郡]]郡役所，1924年廢高雄郡，郡役所改為[[高雄警察署|高雄警察署]]。此外1920年高雄郡下轄[[高雄街|高雄街]]的街役場亦位於哈瑪星，1924年高雄設市，街役場續用為市役所，直到1939年[[高雄市立歷史博物館|新高雄市役所]]完工才遷移，足見哈瑪星曾是高雄的政經中心。但因[[市中心|市中心]]東移而慢慢沒落；近年來則朝著觀光等方向發展，並因捷運的興建而形成新的商圈。

== 事件年表 ==
*1912年，歷時四年、共分六期、面積達60,570坪的湊町埋立工程宣告完成<ref>{{Cite web |url = http://jonyao1978.pixnet.net/blog/post/22569075 |title = 發現老高雄 - 哈瑪星的誕生 |author = Jon |date = 2008年11月27日 |publisher = Pixnet |language = zh-tw |accessdate = 2015年12月30日 |quote = 1912年（大正元年），歷時四年，共分六期，面積達60,570坪的湊町埋立工程宣告完成，今天的哈瑪星也正式誕生。這片領台後的第一號海埔新生地，吸引了當時日本各工程領域的菁英前來大顯身手。此時的湊町，不僅有寬廣的街道、新穎的建築，更有現代化的自來水與電氣設施。此外，緊臨港口與停車場（火車站）的地理優勢，更使得地價飛漲，金融業與商業蓬勃發展，而湊町也成為近代高雄都市化的濫觴。}}</ref>，今天的哈瑪星也正式誕生。
*1923年4月21日，下午一點二十八分，時任皇太子的[[昭和天皇|昭和天皇]]裕仁，從臺南抵達高雄驛，並前往[[高雄州廳|高雄州廳]]、[[高雄第一尋常高等小學校|高雄第一尋常高等小學校]]。

*2015年4月29日，哈瑪星居民與中山大學師生，在西子灣隧道口抗議大量遊覽車爆量湧入，影響居住品質，呼籲市府立即總量管制大型巴士進入<ref>{{Cite web |url = http://www.chinatimes.com/realtimenews/20150429004240-260402 |title = 哈瑪星人扛大砲抗議 拒陸客遊覽車進入 |author = 劉宥廷、謝明祚 |date = 2015年4月29日 |publisher = 中國時報 |language = zh-tw |accessdate = 2015年12月30日 |quote = 高雄市哈瑪星一帶居民與中山大學師生，今日前往中山大學隧道路口抗議大量遊覽車進入，讓居民也飽受塞車造成廢氣污染。居民還抬出道具清朝大砲，要市府立即總量管制大型遊覽車進入。 }}</ref>。
