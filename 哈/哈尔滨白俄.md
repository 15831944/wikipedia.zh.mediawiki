'''哈尔滨白俄'''指[[十九世纪|十九世纪]]末至[[二十世纪|二十世纪]]中叶居住于[[哈尔滨|哈尔滨]]的[[俄羅斯人|俄罗斯裔]]居民。

十九世纪末开始，[[俄罗斯帝国|俄国]]开始在北满修建“[[东清铁路|东清铁路]]”（[[中華民国|中華民国]]建立后改称[[中东铁路|中东铁路]]），开始向铁路沿线，尤其是铁路枢纽[[哈尔滨|哈尔滨]]移民。1918年，俄国爆发[[十月革命|布尔什维克革命]]，大批俄国军人、贵族和平民避難至哈尔滨，使哈尔滨成为俄国境外最大的俄国人聚居地。哈尔滨白俄创办了几十间俄国人学校，并出版俄文[[杂志|杂志]]和[[报纸|报纸]]。

[[满洲国|满洲国]]时代，白俄居民受到[[滿洲國政府|滿洲國政府]]的压制。1935年，苏联将“东清铁路”出售給满洲国。一些俄国人在苏联政府鼓励下返回苏联。后来这些人中的大多数受到[[斯大林|斯大林]]领导的[[苏共|苏共]]清洗。<ref>{{Cite web |url = http://www.guancha.cn/zhangxiaobo/2013_09_02_169817.shtml |title = 苏联大清洗中的“哈尔滨人”悲剧 |author = 张晓波 |date = 2019-09-02 |publisher = 观察者 |language = zh-cn |accessdate = 2017-10-04 }}</ref>

1949年[[中华人民共和国|中华人民共和国]]成立后，哈尔滨白俄大多陆续迁出。赫鲁晓夫上台后，[[1950年代遣返苏联侨民|1954、1955年又有大批哈尔滨白俄被当做侨民遣返回苏联]]。<ref>{{Cite web |url = http://china.huanqiu.com/roll/2009-05/472860.html |title = 50年代苏联“撤侨”11万的来龙去脉 |author =  |date = 2009-05-27 |publisher =  |language = zh-cn |accessdate = 2017-10-04 }}</ref><ref>{{Cite web |url = http://phtv.ifeng.com/program/tfzg/detail_2010_05/25/1551446_0.shtml |title = 外国人在中国：“打扫干净屋子再请客” |author =  |date = 2010-05-25 |publisher = 凤凰网 |language = zh-cn |accessdate = 2017-10-04 }}</ref>目前哈尔滨白俄及后裔分布于[[俄罗斯|俄罗斯]]、[[美国|美国]]、[[以色列|以色列]]及[[南美|南美]]国家。
