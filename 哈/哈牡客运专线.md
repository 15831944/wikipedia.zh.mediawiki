'''哈牡客运专线'''，是[[中国|中国]]一条连接[[黑龙江省|黑龙江省]]内[[哈尔滨市|哈尔滨市]]与[[牡丹江市|牡丹江市]]的客运专线，设计时速250公里的[[客运专线|客运专线]]，正线长度299.340公里<ref name="sdpc650275">{{cite web|url=http://www.sdpc.gov.cn/zcfb/zcfbtz/201411/t20141128_650275.html|title=国家发展改革委关于新建哈尔滨至牡丹江铁路客运专线可行性研究报告的批复|publisher=中华人民共和国国家发展和改革委员会}}</ref>。线路从[[哈尔滨站|哈尔滨站]]引出，向东经[[阿城区|阿城]]、[[帽儿山镇|帽儿山]]至[[尚志市|尚志市]]，再向东南经万山、[[苇河镇|苇河]]、[[亚布力镇|亚布力]]、横道河子、[[海林市|海林]]至终点[[牡丹江站|牡丹江站]]<ref name="sdpc650275" />。

== 概述 ==
哈牡城際鐵路由[[中国铁路总公司|中国铁路总公司]]，[[黑龍江省|黑龍江省]]以及[[哈爾濱鐵路局|哈爾濱鐵路局]]共同修建。项目建议书于2014年2月得到[[中华人民共和国国家发展和改革委员会|中国国家发展和改革委员会]]的批准，预计總投資共計365.2億元人民幣<ref>{{cite web|url=http://www.sdpc.gov.cn/zwfwzx/xmsphz/201403/t20140313_601921.html |title=哈尔滨至牡丹江铁路客运专线项目建议书通过批准 |publisher=中华人民共和国国家发展和改革委员会 |deadurl=yes |archiveurl=https://web.archive.org/web/20141204181037/http://www.sdpc.gov.cn/zwfwzx/xmsphz/201403/t20140313_601921.html |archivedate=2014-12-04 }}</ref>，工程预计于2015年開工建設，于2018年12月25日开通运营<ref>{{cite web |author1=吴利红 |title=哈牡高铁时速250公里载客运行 500人试乘感受冰雪两城近距离 |url=http://sjdb.hljnews.cn/news/91575.html |website=黑龙江日报}}</ref><ref>{{cite web |author1=新华网 |title=哈牡高铁开通运营 |url=http://www.xinhuanet.com/travel/2018-12/26/c_1123904807.htm |publisher=新华网 |accessdate=2018-12-25}}</ref>。
2014年11月5日，哈牡客专项目可研报告获得国家发改委正式批复<ref>{{cite news|url=http://heilongjiang.dbw.cn/system/2014/11/10/056115623.shtml|title=哈牡客专项目获批复或明年开工 建成后从哈到牡90分钟|publisher=东北网}}</ref>，同年12月15日，哈牡客运专线在[[海林市|海林市]][[横道镇|横道镇]]威虎山隧道出口施工现场正式开工<ref>{{cite web|url=http://hlj.hljnews.cn/system/2014/12/15/010181378.shtml|title=龙江铁路项目建设获重大进展：哈牡铁路客运专线开工|publisher=黑龙江新闻网|date=2014-12-15|deadurl=yes|archiveurl=https://web.archive.org/web/20141219162946/http://hlj.hljnews.cn/system/2014/12/15/010181378.shtml|archivedate=2014-12-19}}</ref>。
2018年11月底，哈牡客专开始试跑<ref>{{cite news |author1=哈尔滨市政府网站 |title=哈牡高铁昨试跑年末通车 两地车程将缩至两小时 |url=https://news.sina.com.cn/c/2018-11-30/doc-ihpevhcm4318414.shtml |accessdate=2019-02-01}}</ref>。
2018年12月25日，哈牡客专（也称作哈牡高铁)开通运营。从牡丹江到哈尔滨最快运营时间为1小时28分钟，一般在两小时左右。开通初期每日开行约20对列车，春运期间增加至29对。<ref>{{cite web |author1=哈尔滨市政府网站 |title=哈牡高铁“满月”运客59万人次 |url=https://news.sina.com.cn/c/2019-01-28/doc-ihqfskcp1040728.shtml |accessdate=2019-02-01}}</ref>

== 沿线車站 ==
线路自[[哈尔滨站|哈尔滨站]]引出，全线设[[哈尔滨站|哈尔滨]]、[[新香坊北站|新香坊北]]、[[阿城北站|阿城北]]、[[帽儿山西站|帽儿山西]]、[[尚志南站|尚志南]]、[[一面坡北站|一面坡北]]、[[苇河西站|苇河西]]、[[亚布力西站|亚布力西]]、[[横道河子东站|横道河子东]]、[[海林北站|海林北]]、[[牡丹江站|牡丹江]]等11座车站，预留[[牡丹江西站|牡丹江西站]]<ref name="sdpc650275" />。
