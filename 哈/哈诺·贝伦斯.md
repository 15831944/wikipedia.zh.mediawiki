'''哈诺·贝伦斯'''（{{lang-de|Hanno Behrens}}，{{bd|1990年|3月26日|catIdx=Behrens, Hanno}}）是一位[[德国|德国]][[足球运动员|足球运动员]]，自[[2014年至2015年德国足球乙级联赛|2014-15赛季]]起效力于[[德乙|德乙]]俱乐部[[纽伦堡足球俱乐部|纽伦堡]]，于场上司职[[中场|中场]]。

==生平==
贝伦斯是在家乡俱乐部{{ld|埃尔姆斯霍恩足球俱乐部|FC Elmshorn|埃尔姆斯霍恩}}开启自己的足球生涯，并于2005年转投[[汉堡体育俱乐部|汉堡体育俱乐部]]的青训营。在那里，他于2007年从U17梯队晋升至U19梯队，2008年则进入[[汉堡二队|汉堡二队]]。至[[2009年至2010年德國足球甲級聯賽|2009-10赛季]]，他入选了汉堡的一线队名单，并一直待到[[2011年至2012年德国足球甲级联赛|2011-12赛季]]，但从未获得出场机会。取而代之的是，他代表二队在[[德国足球北部地区联赛|北部地区联赛]]完成了80场比赛并射入8球。

贝伦斯于[[2012年至2013年德国足球丙级联赛|2012-13赛季]]转会至身处[[德丙|德丙]]作赛的[[达姆施塔特98|达姆施塔特98]]<ref>{{cite web |url=https://sv98.de/home/lilien/aktuelles/details/news/sv-98-vermeldet-die-ersten-neuzugaenge/?tx_news_pi1%5Bcontroller%5D=News&tx_news_pi1%5Baction%5D=detail&cHash=867985a1b5145865bdb25e00b8f2760e |title=SV 98 vermeldet die ersten Neuzugänge |work=sv98.de |publisher=SV Darmstadt 1898 e.V. |date=2012-05-24 |accessdate=2017-10-15}}</ref>，并于2012年8月8日（第4轮）以1比3不敌[[开姆尼茨足球俱乐部|开姆尼茨]]的比赛中，于第56分钟替换{{ld|埃尔顿·达·科斯塔|Elton da Costa}}出场，完成了德丙联赛首秀。跟随达姆施塔特一起，他于[[2013年至2014年德国足球丙级联赛|2013-14赛季]]升入[[德乙联赛|德乙联赛]]，并于[[2014年至2015年德国足球丙级联赛|随后的赛季]]进一步升入[[德甲|德甲]]。

[[2015年至2016年德国足球乙级联赛|2015-16赛季]]，贝伦斯转会至德乙俱乐部[[纽伦堡足球俱乐部|纽伦堡]]<ref>{{cite web |url=http://www.fcn.de/news/artikel/club-verpflichtet-hanno-behrens/ |title=Club verpflichtet Hanno Behrens  |work=fcn.de |publisher=1. Fußball-Club Nürnberg e.V. |date=2015-05-28 |accessdate=2017-10-15}}</ref>。在那里，他自[[2017年至2018年德国足球乙级联赛|2017-18赛季]]起被任命为新的[[队长_(足球)|队长]]<ref>{{cite web |url=http://www.fcn.de/news/artikel/hanno-behrens-ist-neuer-club-kapitaen/ |title=Hanno Behrens ist neuer Club-Kapitän |work=fcn.de |publisher=1. Fußball-Club Nürnberg e.V. |date=2017-07-17 |accessdate=2018-02-20}}</ref>，并将原本期至2018年6月30日届满的合同<ref>{{cite web |url=http://www.nordbayern.de/sport/kapitan-behrens-erst-die-binde-dann-ein-neuer-vertrag-1.6403455 |title=Kapitän Behrens: Erst die Binde, dann ein neuer Vertrag |work=nordbayern.de |publisher=Nürnberger Nachrichten |date=2017-07-24 |accessdate=2018-02-20}}</ref>延长至未知期限。对此决定，他表示：“我在纽伦堡感到很舒服，这是一座宜居的城市，俱乐部及其球迷很特别，而且它目前非常适合运动。我们的团队非常有趣，对未来充满信心。基于所有这些原因，我很清楚我想留在俱乐部”。<ref>{{cite web |url=http://www.fcn.de/news/artikel/club-verpflichtet-hanno-behrens/ |title=Behrens bleibt: Der Kapitän verlängert vorzeitig |work=fcn.de |publisher=1. Fußball-Club Nürnberg e.V. |date=2017-07-28 |accessdate=2018-02-20}}</ref>
