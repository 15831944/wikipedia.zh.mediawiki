'''哈萨克族'''（[[哈薩克語|哈薩克語]]：{{rtl-lang|kk|Қазақ}} / {{lang|kk-latin|Qazaq}} / {{lang|kk-arabic|قازاق}}）是[[哈萨克斯坦|哈萨克斯坦]]的主要民族和[[中国|中国]]、[[蒙古国|蒙古国]]、[[乌兹别克斯坦|乌兹别克斯坦]]、[[俄罗斯|俄罗斯]]、[[伊朗|伊朗]]等国家的的少数民族，全球总人口约1860萬。据[[第五次人口普查|2010年人口普查数据]]，中国有146万哈萨克族，主要分布于[[新疆维吾尔自治区|新疆维吾尔自治区]][[伊犁哈萨克自治州|伊犁哈萨克自治州]]、[[木垒哈萨克自治县|木垒哈萨克自治县]]、[[巴里坤哈萨克自治县|巴里坤哈萨克自治县]]和[[甘肃省|甘肃省]][[阿克塞哈萨克族自治县|阿克塞哈萨克族自治县]]。[[蒙古国|蒙古国]]有10万以上的哈萨克族，俄罗斯的[[奥伦堡|奥伦堡]]与[[车里雅宾斯克|车里雅宾斯克]]也有称为[[小玉兹|小玉兹]]的哈萨克族人。在烏兹别克斯坦有40万人生活在[[卡拉卡尔帕克斯坦自治共和国|卡拉卡尔帕克斯坦自治共和国]]，10万人生活在[[塔什干州|塔什干州]]。土耳其和伊朗也有为数不少的哈萨克人。哈萨克族的人种属于蒙古人种北亚类型和[[欧罗巴人种|欧罗巴人种]]印度地中海类型之间的混合类型，属于混血民族。

哈萨克族的传统语言是[[哈萨克语|哈萨克语]]，在哈萨克斯坦，多掌握[[俄语|俄语]]，在中国、蒙古国、乌兹别克斯坦等国则多有兼通当地语言者。传统的哈萨克语以阿拉伯字母书写，主要在[[中国哈萨克族|中国哈萨克族]]中使用，哈萨克斯坦、蒙古等国哈萨克族多使用西里尔字母书写的哈萨克文。中国曾在1959年设计了以[[拉丁字母|拉丁字母]]为基础的新文字方案，1982年废除。哈萨克斯坦自2010年后开始逐步推行文字[[拉丁化|拉丁化]]，计划到2025年正式使用拉丁字母哈萨克文。

== 历史 ==
[[钦察|钦察]]草原的最早居民目前认为是[[斯基泰人|斯基泰人]]。[[匈奴人|匈奴人]]西迁以后，各[[突厥语|突厥语]]民族也开始进入。[[哈萨克汗国|哈萨克汗国]]成立后，周边很多游牧民族部落加入，经过几十年的分化和融合逐步形成了哈萨克族。因此，哈萨克人的族源比较复杂，包括了古代不同时期进入这个地区的印歐民族（[[斯基泰人|斯基泰人]]，[[乌孙|乌孙]]、[[康居|康居]]、[[奄蔡|奄蔡]]人、[[大月氏|大月氏]]）、突厥民族（[[匈奴|匈奴]]、[[铁勒|铁勒]]、[[可薩人|可薩人]]、[[基馬克|基馬克]]、[[欽察|欽察]]、[[康里|康里]]、[[阿兒渾|阿兒渾]]、[[突騎施|突騎施]]、[[葛邏祿|葛邏祿]]、[[蔑儿乞|蔑儿乞]]、[[札剌亦兒|札剌亦兒]]、[[汪古|汪古]]、[[克烈|克烈]]、[[乃蠻|乃蠻]]、[[诺盖人|诺盖人]]）等民族。他们分布于阿尔泰山、哈萨克丘陵至烏拉爾山脈。

哈薩克這个词有各种解釋：

1. '''尼合邁德·蒙加尼'''指出是[[奄蔡|奄蔡]]的異譯。這字也有白天鵝的意思。kaz是天鵝，ak是白。他們把自己說成是天鵝[[化身|化身]]成的美女與一個男人生出的後裔。

2. 俄羅斯人'''拉甫羅夫'''指出在[[八世紀|八世紀]]僧侶'''葉弗卑尼'''著作中的'''哈索格'''即是哈薩克。

3. 哈薩克在[[北高加索|北高加索]]的[[卡巴爾達-巴爾卡爾語|卡巴爾達-巴爾卡爾語]]中是指處境不好，無家可歸的人。在[[晚清|晚清]]，[[丁謙|丁謙]]指出哈薩克是[[可薩人|可薩人]]的異譯。在[[北高加索|北高加索]]民族的語言中'''哈索格'''有高大奴隸的意思，最早的哈薩克是由里海一帶的卡斯比與[[塞種|塞種]]構成。哈萨克也有不服从苏丹，不服从汗自由过日子和自由不羈的人之意。

4. 十三世紀時阿拉伯人將哈薩克解釋為自由人，游牧战士等。哈萨克意为“自由之地”，延伸泛指所有生活在中亚大草原的自由部族，这些部族并不一定有血缘关系，文化多样，但是语言高度统一。波斯[[菲爾多西|菲爾多西]]的[[列王紀|列王紀]]也说[[咸海|咸海]]以北有哈萨克汗国。

5. 另一种观点认为哈萨克是源于古老地名，据记载罗马大帝康士坦丁留给儿子的遗嘱中说：“在那外围有着哈萨克亚大草原，再过哈萨克亚就是阿兰”。康士坦丁大帝是913-959年代之间称帝之。有些部落與突厥并无關係：[[托熱|托熱]]是贵族阶级(出自[[朮赤|朮赤]]家族)；sunak是敍利亞人後人；霍加部有阿拉伯傳教士和士兵意思；alshen是[[阿兰|阿兰]]居民後人。

19世紀時俄羅斯人把哈薩克人稱為吉爾吉斯。

哈薩克斯坦的哈薩克族在[[1932年苏联大饥荒|1932年苏联大饥荒]]損失了37%人口。

== 體型外表 ==
[[File:Kazakhs_people.jpg|thumb]]
由於其複雜的歷史和歐亞血統，哈薩克族外表明顯多樣性，哈薩克族主要表現出[[蒙古人種|蒙古人種]]的特點與[[高加索人種|高加索人種]]特點。他们属于[[图兰人种|图兰人种]]，膚色淺棕色，高颧骨。哈薩克人的頭髮顏色變化從棕色、紅色到烏黑。哈薩克人眼睛颜色有棕色、藍色与綠色。

== 文化 ==
哈萨克的文化属于突厥文化与伊斯兰文化和斯拉夫文化混合体。哈萨克属于伊斯兰民族，但仍有[[萨满教|萨满教]]遗留。哈薩克人也崇拜狼，認為狼是[[騰格里|騰格里]]的寵物與使者。他们是外婚制，有规定同一部落七代之内不可结婚。由於其游牧的生活方式，哈薩克族保持口述歷史的史詩傳統 。合併各種哈薩克起源的游牧部落，保留原建氏族遙遠的記憶。重要的是為哈萨克族保留不低於七代的祖先（被稱為知道他或她的家族樹）。

== 宗教 ==
[[伊斯蘭教|伊斯蘭教]]是在八世纪时成為哈萨克人宗教，之前曾信仰薩滿教、[[祆教|祆教]]、[[佛教|佛教]]和[[景教|景教]]，[[萨曼王朝|萨曼王朝]]向[[塔拉斯|塔拉斯]]进攻引入伊斯蘭教。在[[喀喇汗朝|喀喇汗朝]]有相当多突厥人信教，但到十九世纪才全民信教(城市比较正统)。而在靠近俄羅斯的小玉茲則小部分東正教徒。

他們雖然是穆斯林，但仍保留一些前伊斯蘭信仰如薩滿教的儀式和觀念（如用火辟邪和認為世界有善和邪的精靈）。

== 民族服饰 ==
哈萨克族服饰多用牲畜皮毛加工而成。妇女夏季穿长花布连衣裙，喜用银元或银制品作装饰。冬季外罩对襟大衣，大衣多用冬羊皮缝制。妇女头戴白布盖头，外披白布大头巾，长及脚跟，头巾左上端有饰品装饰。年轻女子头戴圆形花帽，帽顶由[[猫头鹰|猫头鹰]]羽毛作帽缨。冬天男子多穿羊皮、狐皮、狼皮大氅和用驼毛絮里的大衣，腰束皮带，右侧佩有小刀，裤子多用牛皮缝制，名曰“夏里巴尔”。因经常骑马，男女都穿长筒皮靴。
