'''哈卡舞'''（''Haka''）在[[紐西蘭|紐西蘭]]泛指[[毛利人|毛利人]]的傳統舞蹈形式。於各太平洋地區中，諸如[[東加|東加]]、[[大溪地島|大溪地島]]、[[薩摩亞|薩摩亞]]等國都有略異的哈卡舞。源於[[夏威夷州|夏威夷]]和[[玻里尼西亞|玻里尼西亞]]<ref>{{cite web |url=http://www.newzealand.com/travel/about-nz/features/old-features/discovery-vintage-new-zealand/profile-interviews/wira-gardiner.cfm/context/popup/language_uuid/69B3E9FB-2E57-4EDB-9046-4684AA660A92.html |title=Interviews-Wira Gardiner |accessdate=2009-01-13 |author=newzealand.com |date= |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20081119032337/http://www.newzealand.com/travel/about-nz/features/old-features/discovery-vintage-new-zealand/profile-interviews/wira-gardiner.cfm/context/popup/language_uuid/69B3E9FB-2E57-4EDB-9046-4684AA660A92.html |archivedate=2008-11-19 }}</ref>。哈卡舞是動作、拍打配以叫嚷和哼聲的團體舞蹈。

哈卡舞表演的目的，包括消遣、對來賓的熱情歡迎、對特殊成就或場合的承認、部落之間相聚的儀式；現時最廣為人知的，是[[紐西蘭國家欖球隊|紐西蘭國家欖球隊]]全黑隊在比賽前表演的哈卡舞“Ka Mate!”。哈卡舞的詞和動作隨著時代不斷的創新，如今現代的創作反映的是當代和政治問題。

社會上最常見的哈卡舞由男人表演，或在男人後面加上女人助音。傳統而言，女性因精於反白眼（毛利人稱pukana）之技，在哈卡舞主團旁邊被賦予主要位置。多數手持武器，保護側翼；她們的存在也是提升其他表演者的氣勢<ref>{{cite web|url=http://www.newzealand.com/travel/app_templates/haka/en/index_content.html |title=Haka - a New Zealand icon |accessdate=2009-01-13 |author=newzealand.com |date= |language=en }}</ref>。也有小部份哈卡舞是主要由女性表演的，如著名哈卡舞「Ka Panapana」；更有簡單的哈卡是為小孩而設。

== 源起 ==
太平洋地區的祖先從[[夏威夷州|夏威夷]]和[[玻里尼西亞|玻里尼西亞]]帶到各個地區。

=== 毛利人 ===
在毛利傳說中，[[太陽神|太陽神]]Ra有兩位妻子，分別是夏季Hine-raumati和冬季Hine-takurua。舞蹈的始源，歸功於夏季為太陽神生下的兒子旦那羅(Tanerore)。旦那羅為母親表演，從而產生了今天的哈卡舞。<ref>{{cite web |url=http://www.allblacks.com/index.cfm?layout=haka |title=The Haka and the All Blacks |accessdate=2009-01-13 |author=allblacks.com |date= |language=en |deadurl=yes |archiveurl=https://web.archive.org/web/20081029211223/http://www.allblacks.com/index.cfm?layout=haka |archivedate=2008-10-29 }}</ref><ref>{{cite web|url=http://www.newzealand.com/travel/app_templates/haka/en/index_content.html |title=Haka - a New Zealand icon |accessdate=2009-01-13 |author=newzealand.com |date= |language=en }}</ref>

毛利人哈卡舞的形態之一，以抖動的手表示夏季熱天所見的抖動的閃光；另外在舞蹈完結之時，舞者為把[[舌頭|舌頭]]伸長，意思是扮演敵人被殺後，頭颅被[[獵首|獵首]]，掛在長竿上的模樣。

== 毛利人哈卡舞 ==
=== 種類 ===
按照毛利人的分類，哈卡舞可以分成以下幾種：
* Whakatu waewae
* Tutu ngarahu
* Ngeri－不用武器，用作心理上激發戰士
* Peruperu－戰爭式哈卡，在開戰之前所演示，祈求戰爭之神庇護，顯示己方之力量和無畏，恫嚇敵方；兩方相遇，還可以對哈卡。<ref>{{cite web|url=http://www.youtube.com/watch?v=8eGCsEQ15L4&feature=related |title=橄欖球賽前對哈卡舞－紐西蘭對東加　The Haka - New Zealand Vs Tonga
|accessdate=2009-01-13 |author= |date= |language= }}</ref>
* Haka taparahi－儀式性哈卡，不用武器，現時最流行的哈卡形式
* Manawa wera haka－一般與葬禮或有關死亡的場合有關，不用武器，沒有預設動作

== 同源術語 ==
毛利語裏的哈卡，在許多中太平洋地區語言中有同源的術語：
* 東加語haka－「歌唱中做手部動作」
* 薩摩亞語saʻa；托克勞語haka；那勞東加語ʻaka；夏威夷語haʻa；馬克薩斯語haka－「舞蹈」
* 孟加利瓦語ʻaka－「以傳統方式舞蹈；舞蹈配以唱頌，一般帶類似作戰性質」
在一些語言中，其意思有分歧，比如提可比亞語中saka指「在傳統儀規系統中進行儀式」。以原代語解譯原始玻里尼西亞語系是*saka，最早衍生自原始大洋語系*saŋka(g)。
