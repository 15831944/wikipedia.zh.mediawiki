'''《哈利波特－死神的聖物2》'''（{{lang-en|'''Harry Potter and the Deathly Hallows – Part 2'''}}）根據[[J·K·罗琳|J·K·罗琳]]所寫的[[哈利波特|哈利波特小說系列]]的第七本《{{哈利波特7}}》改编。此書是唯一一本被改編拍成兩套電影的小說，這兩套電影為[[哈利波特系列電影|哈利波特系列電影]]中第七及第八部電影，亦是該系列的最後兩部。導演繼續由拍攝第五、六集的導演—[[大衛·葉斯|大衛·葉斯]]執導。這兩套電影由[[華納兄弟|華納兄弟]]發行，第二部以[[3D|3D]]及[[2D|2D]]兩種版本上映，第二部是哈利波特電影系列中首次把整套電影以[[三维电影|3D]]形式播放<ref>[http://www.imdb.com/news/ni4800748/ 3D Version Of Next Harry Potter Pic Abandoned]</ref>，另外亦推出[[IMAX|IMAX]]版本<ref>[http://www.hpana.com/news.21098.html Confirmed: 'Deathly Hallows: Part II' set for 3D; both parts in IMAX]</ref>。

== 概要 ==

在終極篇的第二部，正邪一戰將會全面爆發。危機升級至前所未見的凶險，沒有一個人能置身事外。但是，只有哈利波特一人會需要獨自與佛地魔展開終極一戰。

== 原著小说 ==
《哈利·波特》系列原本是由英国Bloomsbury出版社和美国Scholastic Press出版社出版的读物。后来，它逐渐赢得了广大的青少年和成人的欢迎。

哈利·波特系列的每一本书都代表着哈利在学校的某一年，跟着哈利的成长，每一本书都比前一本更复杂，更成熟。尽管作者说过她在出版第一本书以前就已经把整个系列的情节都想好了，但她也表示过在她写书的同时，她做了一些修改。

哈利·波特系列的第一本书只有223页，而后来到了第五本，已经有766页厚。哈利·波特系列含有很多幻想和魔法的元素，而再跟侦探小说的结构结合。每一本书都有一个任务让主人公去完成，而附加着关于魔法世界的信息和为未来故事所埋下的伏笔。

所以，这本书吸引着很多读者去发掘哈利·波特的冒险将有什么样的结果（这也是本书畅销原因之一）。作者曾说过，第七本书将解开第六本书中的所有问题，也会解开整个系列中的所有谜团。
