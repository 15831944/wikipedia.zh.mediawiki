'''哈崙森林鐵路'''為一座位在[[臺灣|臺灣]][[花蓮縣|花蓮縣]][[木瓜山|木瓜山]]區的森林鐵路系統，是花蓮縣境內包括[[林田山森林鐵路|林田山森林鐵路]]與[[嵐山森林鐵路|嵐山森林鐵路]]在內的三座花蓮山區林業鐵路之一。整座運輸系統由7條[[鐵道|鐵道]]路線與5條[[索道|索道]]所組成，原管理單位為木瓜山林場哈崙工作站，當哈崙工作站停止伐木業務並裁撤後，伴隨的整座森林鐵道系統於1989年（民國78）放棄使用至今，並已呈現荒廢狀態。<ref name="花蓮林區管理處">{{Cite web|url=http://hualien.forest.gov.tw/ct.asp?xItem=29398&CtNode=2207&mp=380|title=林務局花蓮林區管理處-歷史沿革|accessdate=2015-06-23|author=花蓮林區管理處|publisher=花蓮林區管理處|language=zh-hant|date=2008年8月3日|deadurl=yes|archiveurl=https://web.archive.org/web/20090308020925/http://hualien.forest.gov.tw/ct.asp?xItem=29398&CtNode=2207&mp=380|archivedate=2009年3月8日}}</ref><ref name="哈崙伐木場">{{Cite web |url=http://library.taiwanschoolnet.org/cyberfair2002/C0236970055/color-htm/index.htm |title=繁華盡落的山林-哈崙伐木場 |accessdate=2015-06-23 |author=孫承偉、童如妤、官美媛所帶領之本專題製作學生|publisher=花蓮縣鑄強國民小學|language=zh-hant|date=2002年2月8日}}</ref><ref name="繁華落盡 再探哈崙工作站">{{Cite web |url=http://old.ltn.com.tw/2004/new/sep/22/today-life9.htm |title=繁華落盡 再探哈崙工作站 |accessdate=2015-06-23 |author=楊宜中|publisher=自由時報|language=zh-hant|date=2004年9月22日}}</ref><ref>{{Cite web|url=https://www.agriharvest.tw/theme_data.php?theme=theme&id=9&artid=158|title=【攝影集】《無盡藏  哈崙百年林業影像集》｜農傳媒|accessdate=2017-03-20|work=www.agriharvest.tw}}</ref>

== 沿革 ==
=== 日治時期 ===
哈崙森林鐵路最早是在[[臺灣日治時期|臺灣日治時期]]1930年（[[昭和|昭和]]5年）由當時的[[花蓮港木材株式會社|花蓮港木材株式會社]]開始進行林業鐵路的鋪設並執行伐木業務。<ref name="哈崙伐木場"/>

=== 省政府時期 ===
[[臺灣戰後時期|二戰後]]，[[臺灣省政府|臺灣省政府]]接收[[日本|日本]][[臺灣總督府|臺灣總督府]]所遺留下來的伐木機具與既成鐵道繼續進行伐木業務，在1951年（民國40年）將木瓜山的伐木業務及相關鐵道設備改由[[花蓮縣政府|花蓮縣政府]]設立的木瓜山林場來經營，並接續[[日本人|日本人]]已留下之路線繼續興建完成大景、哈崙軌道總共23.5公里及4座索道。<ref name="哈崙伐木場"/>1958年（民國47年）木瓜山林場收歸省政府營運，1960年（民國49年）2月，依照省政府之指示，將木瓜山林場、[[太魯閣|太魯閣]]林場、花蓮山林管理所所轄屬之[[新城|新城]]、[[壽豐|壽豐]]、[[鳳林|鳳林]]三個分駐工作站與林田山工作站併入木瓜林區管理處。<ref name="花蓮林區管理處"/><ref name="哈崙伐木場"/>整併完成後，哈倫林業鐵路繼續延長軌道。1965年（民國五54年）完成5號索道，並且在當時哈崙的伐木業務達到了全盛時期，當時總計索道共5座，全長4707.12公尺，軌道全長49.51公里，而自山區砍伐之木材經鐵道與索道運輸後再運送到現今花蓮[[鯉魚潭|鯉魚潭]]旁的池南工作站地面上進行加工處裡。<ref name="哈崙伐木場"/>

1977年（民國66年）以後，臺灣省政府實施「台灣林業經營改革方案」，將臺灣的森林經營改以多目標的保育與利用為前提，而使得伐木業務逐漸減少，當時的哈崙工作站也開始結束伐木業務，改以造林為主，爾後在1989年（民國78年）木瓜林區管理處和前玉里林區管理處合併成立花蓮林區管理處，而原隸屬於木瓜林區管理處的哈崙工作站正式裁撤，連帶的哈崙林業鐵路也正式廢棄使用至今。<ref name="花蓮林區管理處"/>
=== 現今 ===
哈崙林業鐵路廢棄後，僅將鯉魚潭湖畔旁原哈崙林業鐵路平地伐木集材地－池南工作站，改建為[[池南國家森林遊樂區|池南國家森林遊樂區]]以供觀光，其餘山區中的林業鐵路則無計劃用做觀光用途而進行修復。<ref name="花蓮林區管理處"/><ref name="繁華落盡 再探哈崙工作站"/><ref name="花東縱谷國家風景區-池南國家森林遊樂區">{{Cite web |url=http://old.ltn.com.tw/2004/new/sep/22/today-life9.htm |title=花東縱谷國家風景區-池南國家森林遊樂區 |accessdate=2015-06-23|publisher=花東縱谷國家風景區|language=zh-hant}}</ref>

== 未來展望 ==
花蓮縣政府於2015年5月以花東基金補助[[新台幣|新台幣]]600萬辦理可行性評估計劃在[[六十石山|六十石山]]、[[赤柯山|赤柯山]]與鯉魚潭推動「觀光[[纜車|纜車]]」計畫，其中鯉魚潭的觀光纜車即以原哈崙森林鐵路所留下之索道哈崙纜車站作為連結點。<ref>{{Cite web |url=https://news886.wordpress.com/2015/05/05/%E7%B8%A3%E6%94%BF%E5%BA%9C%E8%A9%95%E4%BC%B0%E6%8E%A8%E5%8B%95%E9%AF%89%E9%AD%9A%E6%BD%AD%E3%80%81%E8%B5%A4%E6%9F%AF%E5%B1%B1%E5%8F%8A%E5%85%AD%E5%8D%81%E7%9F%B3%E5%B1%B1%E7%BA%9C%E8%BB%8A%E8%A8%88/|title=縣政府評估鯉魚潭、赤柯山及六十石山纜車計畫 |accessdate=2015-06-23 |author=花蓮GO新聞/編輯|publisher=花蓮GO新聞|language=zh-hant|date=2015年5月5日}}</ref>
