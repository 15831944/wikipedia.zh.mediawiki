{{nihongo|《'''霍爾的移動城堡'''》|ハウルの動く城}}，是在2004年由[[宮崎駿|宮崎駿]]導演，後繼[[動畫電影|動畫電影]]《[[千與千尋|千與千尋]]》之後，[[吉卜力工作室|吉卜力工作室]]推出的一部動畫電影，是帶有宮崎駿動漫特色的作品。作品總作畫張數達148,786張，片長119分11秒。<ref>The Art Of Howl's Castle：13頁</ref>

故事內容改編自[[英國|英國]][[奇幻文學|奇幻文學]]作家[[戴安娜·韋恩·瓊斯|戴安娜·韋恩·瓊斯]]在1986年的著作《[[魔幻城堡|魔幻城堡]]》（Howl's Moving Castle）。<ref>{{lang|ja|ハウルの動く城 ロマンアルバム}}：22頁</ref>

== 劇情簡介 ==
在19世纪的[[歐洲|歐洲]]世界，史柏麗王國裡出現了一座會移動的[[城堡|城堡]]。

蘇菲前往探望在工作的蕾蒂，被軍官搭訕，蘇菲不知該如何擺脫時冒出了霍爾搭救。

蘇菲遇見荒野[[女巫|女巫]]。當女巫得知霍爾與蘇菲見面時，為了報復蘇菲，對她下了變老的[[詛咒|詛咒]]。

變老的蘇菲決定離家出走。在路上蘇菲救了[[稻草人|稻草人]]，稻草人為蘇菲引來了霍爾的移動城堡。

== 角色介紹 ==
*{{nihongo|'''蘇菲'''|ソフィー・ハッター}}
:本片女主角，年齡18歲，以長女的身分代替繼母哈妮繼承父親遺留下來的哈達帽子店，店面不大且樸素。蘇菲在故事一開始顯得沒有主見，對自己的外貌也缺乏自信，經常因為自卑而陷入低潮的情緒裡，對自己備受桎梏的生活感到無奈。
:因為與霍爾有一面之緣而被荒野女巫盯上，被荒野女巫施以詛咒變成90歲的老婆婆，之後誤打誤撞進入霍爾的移動城堡，並以城堡清潔婦的身分住下。
:住下期間與霍爾互相吸引，彼此相愛。從小就很討厭[[蕪菁|蕪菁]]。最後成功恢復了原本的年紀（但是頭髮依舊為變成老婆婆後的銀髮），同時拯救了差點死掉的霍爾並解開卡西法的契約，在重建完畢的移動飛行城堡中過著幸福快樂的日子。
*{{nihongo|'''霍爾'''|ハウル}}
:本片男主角，法力高強的魔法師，是移動城堡的主人，謠傳他會吃掉年輕漂亮女孩的心臟。為了自由生活而有很多名字（詹金斯、潘德拉肯等）。
:平常是個金髮的俊美青年，嚮往自由自在的生活，但因受國家徵召，而時常變身成巨鳥形的怪物，來回穿梭於戰場。
:非常注重外表，經常改變髮型與穿著，會因髮色不如意和被女孩子拒絕而發脾氣。
:是王宮御用魔法師莎莉曼夫人唯一的門下弟子，曾因主動接近荒野女巫而身陷詛咒與惡魔訂下契約。其實霍爾是一個很膽小的人，是因為害怕莎莉曼強大的魔法而叫蘇菲代替他去，而且派她手下來找霍爾的家，因為不能讓霍爾繼續跟惡魔作交易而被迫停止，不然就會像荒野女巫一樣奪走她的魔力。
:在少年時與火魔卡西法訂下契約，而陷入「成為魔王」的險境，差點失去心性，唯一方法是用卡西法將心臟還給霍爾而存活下來，最後與蘇菲過著幸福快樂的日子。

*{{nihongo|'''荒野女巫'''|荒地の魔女}}
:曾是個才華洋溢的魔女，卻因和惡魔作交易，不敵邪惡誘惑導致身心都被腐蝕而誤入歧途。只會施魔法但不會解魔法。
:五十年前從王宮被趕到荒地，使喚著許多利用魔法塑造出的橡膠人作為手下。
:因霍爾的逃避而遷怒蘇菲，是使蘇菲變成老婆婆的元兇。但後來被莎莉曼夫人用計奪走了法力，因而恢復原本年老的外表。
:失去法力後變成平時無惡意的老人，之後與蘇菲霍爾一家住在一起。也因變老而造成腦筋不靈光，喜歡大口大口抽雪茄，有時會因對於霍爾身上心臟的偏執，而作出不可理喻的行為。

*{{nihongo|'''卡西法'''|カルシファー}}
:是一個火惡魔，前身是流星─星之子。在下著流星雨的溼地高原的夜晚與少年霍爾以「心臟」訂下契約，之後居住在城堡壁爐中，是移動城堡的動力來源。
:主食是木柴；但蛋殼、培根、荷包蛋這類的副食品也很喜歡。不喜愛由火藥產生的火燄，認為它們不懂禮貌。
:平時只接受霍爾的命令，但也曾被蘇菲以半強迫的方式，讓她在它上面煮東西，原因是她要拆穿卡西法和霍爾契約而聽從的。
:之後因蘇菲解開它與霍爾之間的契約而獲得自由，但仍選擇回來與大家在一起。

*{{nihongo|'''馬魯克'''|マルクル }}
:霍爾的徒弟，年紀為8歲而已，他無依無靠住在霍爾的城堡裡，平常一邊打雜一邊學習魔法，腰間總是背著大荷包。
:平時在霍爾不在家時，會以魔法斗篷變身老人接待來客，第一次見到蘇菲不喜歡她，但隨著蘇菲對他們的肯定，馬魯克漸漸變得很懂事，也會幫忙蘇菲作收衣服等家事，相當喜愛蘇菲，最後成為一家人。
:缺點是討厭吃魚和馬鈴薯。因本身喜愛狗而願意收養因因，並常與因因一起行動。

*{{nihongo|'''稻草人-蕪菁'''|カブ}}
:蘇菲在荒地遇到，並無意間救了倒栽蔥的他，頭頂是蕪菁，蘇菲叫他稻草人。
:為了報恩，稻草人一直跟著蘇菲，除了替蘇菲找來柺杖之外，也為她喚來移動的城堡。
:真實身分為被詛咒的鄰國王子。
:哈尔见到稻草人曾说稻草人中了很复杂的魔法。
:在临近结局时，稻草人为了拯救苏菲一行人，用自己的身体阻挡将掉进悬崖的苏菲一行人，苏菲一行人得救了，但自己的身体却被拆散。之后的靠苏菲對稻草人感激的亲吻而解除诅咒。

*{{nihongo|'''因因'''|ヒン}}
:莎莉曼夫人所養的使喚犬可與莎莉曼透過水晶球聯絡。年紀很大，但算是隻活力充沛的老狗。
:在蘇菲前往王宮時帶領她前進，而被蘇菲誤以為是霍爾的變身。
:外型腿短，無法爬樓梯，後來擅自離開莎莉曼和蘇菲一起行動，成為霍爾蘇菲一家的一份子；常與和馬魯克黏在一起。

*{{nihongo|'''莎莉曼夫人'''|サリマン}}
:深得史柏麗國王信賴的宮廷魔法師，握有巨大的實權，曾在魔法學校任教，亦是霍爾的老師，在此次戰爭中急欲獲得最後弟子─霍爾的協助，而這場戰爭就是她下達指令給史柏麗國王，目的就是要對付霍爾。最後因見到鄰國的王子，決定就此休兵停戰。

*{{nihongo|'''哈妮'''|ファニー}}
:蘇菲的繼母，蕾蒂的親生母親，穿著華麗俗艷，個性開朗而多話，在蘇菲離家後賣掉帽店並與一名富豪再婚。
:與蘇菲略有心結，重逢時蘇菲很高興能和好如初。但因哈妮的再婚，丈夫被莎莉曼當作人質而被迫將偷窺蟲帶進蘇菲家，為此對蘇菲感到愧疚。

*{{nihongo|'''蕾蒂'''|レティー}}
:蘇菲的妹妹，也是哈達家的次女，蘇菲同父異母的繼妹，在「伽加俐西點店」大型咖啡店的巧克力專賣部工作。
:遺傳母親開朗的性格與華貴的姿容，相當受異性歡迎，性格獨立有主見。和繼姊蘇菲感情很好。

*{{nihongo|'''史柏麗國王'''|国王}}
:史柏麗王國國王，十分信任莎莉曼，是個有著紅髮紅鬍子的中年男人，穿著空軍制服，對於戰爭相當積極，而霍爾就是去見莎莉曼而假扮成此人。

*{{nihongo|'''莎莉曼的僕役們'''|小姓}}
:一律留著金橙色妹妹頭的少年。

*{{nihongo|'''貝西'''|ベッシー}}
:在蘇菲經營的帽子店裡工作的人員。 

*{{nihongo|'''瑪姬'''|マッジ}}
:住在港口的紅髮女孩，替母親來「詹金斯的店」（移動城堡在港口開業的名稱）找馬魯克拿風咒語，曾詢問蘇菲是否為女巫。

*{{nihongo|'''霍爾的叔父'''|ハウルの叔父}}
:在動畫中的未露面角色，因為他是已過世的人物，僅由霍爾於言談中提及已過世的人物而已，有留予霍爾位於溼地高原的一間水車小屋。

*{{nihongo|'''橡膠人'''|ゴム人間}}
:荒野女巫的手下，無臉，部分身著禮服及禮帽，部分只戴帽子。

*{{nihongo|'''歐塔瑪人'''|オタマ人}}
:戴著面具和帽子，有翅膀，被作為空中戰艦用來對抗敵手的生物兵器。霍爾稱其為「莎莉曼夫人手下的手下」。

*{{nihongo|'''移動城堡'''|動く城}}
:由火魔卡西法驅動，有四支細腳，會噴出蒸氣。
:城堡的大門轉盤上有四種顏色，可經由轉動指針而通往四個不同的地方。

== 製作 ==
=== 過程 ===
[[File:Sortie_de_l'opéra_en_l'an_2000-2.jpg|right]]1882年的作品《{{lang|fr|La Sortie de l'opéra en l'an 2000}}》]]
作品改編成動畫的契機來自於1999年期間。當時宮崎駿每月手上會收到從[[德間書店|德間書店]]所發行的新書，而其中一本則為翻譯自戴安娜·韋恩·瓊斯的著作《[[魔幻城堡|魔幻城堡]]》日文版小說，宮崎駿在閱讀後引起了興趣；進而與吉卜力動畫製作人[[鈴木敏夫|鈴木敏夫]]商量將小說改編為動畫的念頭。<ref>{{lang|ja|ハウルの動く城 ロマンアルバム}}：97頁</ref>

之後吉卜力邀請了[[細田守|細田守]]擔任《霍爾的移動城堡》的導演，並預定將此作品作為在2002年動畫《[[貓的報恩|貓的報恩]]》後發表的新作。<ref>{{Cite web|url=http://www.toho.co.jp/lineup/welcome-j.html |title=TOHO LINE-UP 2002 |trans_title=東寶上映新作2002 |accessdate=2012-03-30 |work=[[東寶|東寶]] |language=ja |deadurl=bot: unknown |archiveurl=https://web.archive.org/web/20020602114340/http://www.toho.co.jp/lineup/welcome-j.html |archivedate=2002-06-02 }}</ref>但之後細田守擔任導演一事被吹斷，而在2003年時改讓宮崎駿勝任此片導演。（後續細田守將當時被撤換的心情；反應在他執導的2005年《[[航海王|航海王]]》劇場版動畫《[[ONE_PIECE_祭典男爵與神祕島|ONE PIECE 祭典男爵與神祕島]]》上。）<ref>{{Cite web |url=http://www.style.fm/as/13_special/mini_050816.shtml|script-title=ja:『ONE PIECE　―オマツリ男爵と秘密の島―』細田守インタビュー（２）| trans_title=《ONE PIECE 祭典男爵與神祕島》導演細田守的訪談之其二 |author=小黑祐一郎 |date=2005-08-08| accessdate=2012-03-30   |work=Web Animation Magazine |language=ja }}</ref> 

在宮崎駿開始執導此片後，改為預定讓《霍爾的移動城堡》在2004年夏季上映。但中間因製作延遲拖至11月底才正式公開，成為少數出現延遲上映狀況的宮崎駿作品。<ref name="Kanou">{{lang|ja|叶精二：《宮崎駿全書》、フィルムアート社、2006年}}</ref>

當時製作期間因正逢美國於2003年對[[伊拉克|伊拉克]]發動[[伊拉克戰爭|二次波斯灣戰爭]]，該時期世界的局勢也間接影響到宮崎駿對於故事情節的編寫。<ref>{{Cite web |url=http://www.nausicaa.net/miyazaki/interviews/newsweek.html|title=Howl's Moving Castle Movie Reviews| trans_title=《霍爾的移動城堡》訪談 |author=Devin Gordon |year=2005| accessdate=2012-03-30   |work=Nauasicaa.Net |publisher=[[新聞週刊|新聞週刊]]|language=en }}</ref>而法國畫家[[亞伯特·羅比達|亞伯特·羅比達]]的插畫也對作品中登場的虛構交通工具有著影響。<ref>{{lang|ja|ハウルの動く城 ロマンアルバム}}：49頁</ref><ref>{{lang|ja|ハウルの動く城 ロマンアルバム}}：22頁</ref>

==== 場景 ====
*{{nihongo|'''史柏麗王國'''|ズベリー}}
:霍爾及蘇菲所在的國家之首都，因鄰國王子的失蹤而與鄰國開戰，國內愛國主義高漲。史柏麗空軍制服為綠色並佩帶防風鏡，兩人一組，一艘飛行艇、陸軍制服為藍底紅相間，配長槍、海軍制服則為[[水手服|水手服]]。
:在史柏麗王國的首都中，霍爾在此以姓氏「潘德拉肯」開業。港口一帶，霍爾則以「詹金斯」(THE GREAT WIZARD JENKINS)的名號開業。
:動畫中的城鎮主要參考法國東部[[亞爾薩斯|亞爾薩斯]]、法國首都[[巴黎|巴黎]]及德國[[海德堡|海德堡]]所繪製。<ref>The Art Of Howl's Castle：12頁</ref>

*{{nihongo|'''哈達帽子店'''|ハッター帽子屋}}
:英文原名HATTER，原著譯名海特，引用自名著[[愛麗絲夢遊仙境|愛麗絲夢遊仙境]]瘋帽商海特。電影設定初期為卡達，後來改為哈達。其為蘇菲父親所傳承下來的帽子店，後來被改販售鮮花，成為哈達花店。

*{{nihongo|'''伽加利西點店'''|カフェ・チェザーリ}}
:位於蘇菲出生的城鎮，是一間大型咖啡店，有巧克力專賣部門，蘇菲妹妹蕾蒂在此工作。<ref name="RomanPage92">{{lang|ja|ハウルの動く城 ロマンアルバム}}：92頁</ref>

*{{nihongo|'''星之湖'''|星の湖}}
:荒地附近，移動城堡曾來到此處並晾曬衣服，蘇菲在此感到心靈的平靜。
:霍爾兒時曾在此地住過一段時間，也是在此與卡西法訂下契約。<ref name="RomanPage94">{{lang|ja|ハウルの動く城 ロマンアルバム}}：94頁</ref>

*{{nihongo|'''黑暗精靈'''|闇の精霊}}
:霍爾曾在被女孩子甩掉時呼喚過一次，後來染髮失敗時又呼喚了一次。<ref name="RomanPage94" />

*{{nihongo|'''偷窺蟲'''|のぞき虫}}

== 反應 ==
=== 票房收視 ===
*《霍爾的移動城堡》在日本上映第二天便突破110萬人觀看；到5月1日止累積有1500萬人次數。之後在2004年度票房收入近200億日元，<ref>{{cite web |url=http://www.eiren.org/toukei/2004.html |title=2004年度日本電影票房收入 |publisher=一般社團法人日本電影製作者連盟 |accessdate=2012.03.30 2011.12.08 |language=ja |deadurl=yes |archiveurl=https://www.webcitation.org/5m2LWZu0T?url=http://www.eiren.org/toukei/2004.html |archivedate=2009-12-15 }}</ref>為至今吉卜力工作室中僅次於《神隱少女》的最賣座作品。後續2006年7月21日於電視頻道上首播時，則曾在[[關東|關東]]一帶地區創下32.9%的收視率。<ref name="Kanou"/>
*[[台灣電影票房|台灣電影票房]]表現獲得9545萬元新台幣的亮眼票房<REF>[http://ent.ltn.com.tw/news/paper/1046445 你的名字「鈔」越霍爾 台灣票房10天破億],2016.10.29 自由時報</ref>。

=== 評價 ===
英國的電影報導雜誌；《Empire》將《霍爾的移動城堡》列為史上前500大電影中的第230名。<ref>{{Cite web |url=http://www.empireonline.com/500/53.asp|title=500 Greatest Movies Of All Time  | trans_title=史上最佳的前500名電影 | publisher=Empire| accessdate=2012-03-30 |language=en }}</ref>[[芝加哥太陽報|芝加哥太陽報]]的評論家理察·羅恩波（Richard Roeper）則肯定片中充滿處處驚奇的創意。

影評家[[羅傑·艾伯特|羅傑·艾伯特]]另認為雖然片中的城堡造型相當驚人，但女主角蘇菲的表現不夠突出，僅給予《霍爾的移動城堡》兩顆半星的評價。<ref>{{Cite web |url=http://rogerebert.suntimes.com/apps/pbcs.dll/article?AID=/20050609/REVIEWS/50601002|title=Howl's Moving Castle | trans_title=羅傑·艾伯特評《霍爾的移動城堡》 |author=[[羅傑·艾伯特|羅傑·艾伯特]] |date=2005-06-09| accessdate=2012-03-30   |work=rogerebert.suntimes.com |language=en }}</ref>
