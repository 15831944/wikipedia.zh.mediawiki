'''哈米斯·马里'''（Khamis Al-Marri，{{bd|1984年|}}），[[卡塔尔|卡塔尔]]足球裁判，2010年起担任[[FIFA|FIFA]]国际足球裁判。<ref>{{cite web |title=Qatar - Referees |url=https://www.fifa.com/development/education-and-technical/referees/association=qat/index.html |website=FIFA |accessdate=2018-11-11}}</ref>

哈米斯·马里曾执法[[2018年國際足協世界盃外圍賽_–_亞洲區第三圈|2018年國際足協世界盃外圍賽亞洲區12强赛]]的比赛。<ref>{{cite web |title=KHAMIS AL MARRI - RECORD OF ALL REFEREED MATCHES 2016/2017 |url=https://www.transfermarkt.com/khamis-al-marri/leistungsdaten/schiedsrichter/8748/saison_id/2016/wettbewerb_id/WMQ1 |website=transfermarkt |accessdate=2018-11-11}}</ref>[[亚洲冠军联赛|亚洲冠军联赛]]中，他曾担任16强赛的主裁判。<ref>{{cite web |title=KHAMIS AL MARRI - RECORD OF ALL REFEREED MATCHES 2016/2017 |url=https://www.transfermarkt.com/khamis-al-marri/leistungsdaten/schiedsrichter/8748/saison_id/2016/wettbewerb_id/AFCL |website=transfermarkt |accessdate=2018-11-11}}</ref>
