==生平==
巴特图勒嘎1963年生于[[乌兰巴托|乌兰巴托]]<ref name=yuanye>{{Cite web |url = http://www.guancha.cn/YuanYe/2017_06_26_415063_3.shtml |title = 袁野：蒙古总统大选前瞻——内亚大陆的茶杯风暴 |author =  |date = 2017-06-26 |publisher = 观察者网 |language =  |accessdate =  |quote =  }}</ref>。早年接受中等职业教育。被培训为绘图员<ref name=san/>。1979年至1990年是蒙古[[桑搏|桑搏]]国家队队员，在世界桑搏锦标赛（World Sambo Championships）中曾获金、银、铜牌。退役后转向商界，通过经营畜牧业白手起家成为百万富翁<ref name=yuanye/>。他是JenCo基金会会长，该基金会促进牲畜放牧、作物栽培、小企业合作。自1999年起，成为Makhimpex公司的所有人，该公司经营[[巴彦洪戈尔省|巴彦洪戈尔省]]等地的牲畜及[[牦牛|牦牛]]出口<ref name=san/>。

在2000年国家大呼拉尔选举中，巴特图勒嘎作为独立候选人在[[巴彦洪戈尔省|巴彦洪戈尔省]]第8选区参选，并加入了“祖国－蒙古民主新社会主义党”（Motherland-Mongolian Democratic New Socialist Party），后来该党改为“[[祖国党_(蒙古国)|祖国党]]”。在2004年国家大呼拉尔选举中，巴特图勒嘎作为“[[祖国－民主联盟|祖国－民主联盟]]”候选人，在[[巴彦洪戈尔省|巴彦洪戈尔省]]第9选区当选[[国家大呼拉尔|国家大呼拉尔]]委员。2006年5月，加入[[民主党_(蒙古)|民主党]]（DP），并当选为蒙古民主协会（Mongolian Democratic Association）主席。在[[2008年国家大呼拉尔选举|2008年国家大呼拉尔选举]]中，巴特图勒嘎作为民主党候选人，在新设的有3个席位的[[巴彦洪戈尔省|巴彦洪戈尔省]]第3选区当选。2008年9月，被任命为[[第二次巴亚尔内阁|巴亚尔内阁]]的蒙古国交通、运输、建筑和城市建设部部长<ref name=san/>。此后在[[巴特包勒德内阁|巴特包勒德内阁]]中留任。任内他曾介入中蒙两国跨境窄轨铁路建设计划，导致该计划一度搁浅<ref name=chushen/>。2012年8月至2014年，任[[阿勒坦呼亚格内阁|阿勒坦呼亚格内阁]]的工业和农业部部长<ref>{{Cite web |url = http://www.dzwww.com/xinwen/guojixinwen/201302/t20130216_8027747.htm |title = 蒙古国将建工业园发展皮革加工业 产品出口中国 |author =  |date =2013-02-16  |publisher = 大众网 |language =  |accessdate =  |quote =  }}</ref>。2006年起，兼任蒙古国桑博协会主席<ref name=dangxuan>{{Cite web |url =http://world.people.com.cn/n1/2017/0709/c1002-29392122.html  |title =蒙古国当选总统巴特图勒嘎  |author =  |date =2017-07-09  |publisher = 人民网 |language =  |accessdate =  |quote =  }}</ref>。

在[[2016年国家大呼拉尔选举|2016年国家大呼拉尔选举]]中，[[蒙古民主党|蒙古民主党]]惨败，巴特图勒嘎也丢掉了国家大呼拉尔委员席位。在[[2017年蒙古国总统选举|2017年蒙古国总统选举]]中，巴特图勒嘎在民主党党内初选中击败了包括前总理[[林钦尼亚木·阿玛尔扎尔嘎勒|林钦尼亚木·阿玛尔扎尔嘎勒]]和民主党主席[[诺罗布·阿勒坦呼亚格|诺罗布·阿勒坦呼亚格]]在内的众多民主党元老，成为民主党的总统参选人<ref name=yuanye/><ref>{{Cite web |url = http://news.xinhuanet.com/world/2017-05/27/c_1121051181.htm |title = 蒙古国正式确认三名总统候选人资格 |author =  |date =2017-05-27  |publisher = 新华网 |language =  |accessdate =  |quote =  }}</ref>。在竞选纲领中，他提出复兴国家经济的“蒙古必胜”口号，赢得多数选民支持，击败[[蒙古人民党|蒙古人民党]]候选人[[米耶贡布·恩赫包勒德|米耶贡布·恩赫包勒德]]，成功当选[[蒙古国总统|蒙古国总统]]<ref name=dangxuan/>。在宣布投票统计结果后的新闻发布会上，巴特图勒嘎称将努力解决经济困难局面，让蒙古国人民过上无债生活，并承诺将促进就业，缩小城乡差距。尽管他在竞选中提出反寡头，但他本人就是蒙古国最富的人之一，其私人产业主要是农业及旅游业。他在竞选中表示，蒙古国经济受到“南部国家”（即中国）的威胁，蒙古国大量地下资源出口到中国，他发誓将平衡与邻国的进出口，并提议向[[俄罗斯|俄罗斯]]修铁路、建厂房处理矿产资源，支持深化同俄罗斯的关系。但他此前也曾力挺修建自蒙古国塔班陶勒盖（Tavan Tolgoi）通向中国的铁路项目，并赞扬中国“[[一带一路|一带一路]]”泛亚洲基础设施建设的倡议<ref name=chushen/>。2017年7月10日，巴特图勒嘎在乌兰巴托的[[国家宫_(蒙古国)|国家宫]]宣誓就任总统，[[国家大呼拉尔|国家大呼拉尔]]主席恩赫包勒德主持了当天的就职仪式<ref>{{Cite web |url =  http://www.thepaper.cn/newsDetail_forward_1729280|title = 蒙古国新总统宣誓就职：让人民尽快脱贫，与中俄保持友好关系 |author =  |date = 2017-07-10 |publisher =澎湃新闻  |language =  |accessdate =  |quote =  }}</ref>。
