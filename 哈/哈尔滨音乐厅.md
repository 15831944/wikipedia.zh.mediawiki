'''哈尔滨音乐厅'''，位于[[哈尔滨市|哈尔滨市]][[道里区|道里区]]群力新区金河公园，原位于道里区买卖街69号，于2014年搬至现址。
== 历史 ==
哈尔滨音乐厅原址位于道里区买卖街69号，2011年5月由日本[[矶崎新|矶崎新]]工作室设计的新音乐厅在群力新区动工<ref name=":0">{{Cite web|title=哈尔滨音乐厅|url=http://www.hrbso.com/about/page/12|work=www.hrbso.com|accessdate=2019-03-08}}</ref>，并于2014年投用。<ref name=":1">{{Cite web|title=哈尔滨“音乐之城”实至名归 城市新地标演出国际范--黑龙江频道--人民网|url=http://hlj.people.com.cn/n2/2017/1015/c220027-30830567.html|work=hlj.people.com.cn|accessdate=2019-03-08}}</ref>

== 建筑 ==
哈尔滨音乐厅总建筑面积36,109平方米，内部设有1200座的大音乐厅，及400座的小音乐厅。音乐厅采用了日本规划建筑师矶崎新工作室的“浮游冰晶”的设计方案，是群力文化产业示范区的一部分<ref name=":0" />。大音乐厅中设有意大利鲁法第公司生产的管风琴。<ref name=":1" />

== 驻场团体 ==
哈尔滨交响乐团2011年成为为哈尔滨音乐厅的驻场乐团。哈尔滨交响乐团的前身是1908年4月俄国外阿穆尔铁道兵旅团管弦乐团在哈尔滨改编成为的为中国第一支交响乐团中东铁路俱乐部交响乐团<ref>{{Cite web|title=哈尔滨音乐厅|url=http://www.hrbso.com/about/page/3|work=www.hrbso.com|accessdate=2019-03-08}}</ref>，现任团长兼总经理曲波，艺术总监汤沐海，常任指挥于雪峰。
