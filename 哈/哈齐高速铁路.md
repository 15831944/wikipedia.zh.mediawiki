'''哈齐高速铁路'''又称'''哈齐客运专线'''，是哈尔滨至齐齐哈尔铁路高速铁路的简称，是[[中国|中国]]一条连接[[黑龙江省|黑龙江省]]省会[[哈尔滨市|哈尔滨市]]与同省[[齐齐哈尔市|齐齐哈尔市]]的高速铁路。线路全长286公里，设计时速300公里，初期运营时速250公里，途径哈尔滨、[[绥化|绥化]]、大庆、[[齐齐哈尔|齐齐哈尔]]四市。于2009年11月30日开工建设，2015年5月23日开始联调联试<ref name="cnr20150524">{{cite news|url=http://china.cnr.cn/ygxw/20150524/t20150524_518622400.shtml|title=中国最北高寒高铁哈齐客运专线开始联调 预计8月通车|publisher=央广网}}</ref>，同年7月13日开始试运行<ref name="xinhuanet20150713">{{cite news|url=http://news.xinhuanet.com/fortune/2015-07/13/c_1115902865.htm|title=中国最北高铁哈齐客专试运行 可适应-40℃严寒|publisher=新华网}}</ref>，并于2015年8月17日通车<ref>{{cite news|url=http://news.163.com/15/0816/01/B13PJ7DD00014AEE.html|title=我国最北高铁哈齐客专17日正式通车 首张车票今日售出|publisher=网易新闻}}</ref>。

== 概述 ==
哈齐高铁自[[哈尔滨站|哈尔滨站]]接轨，全长281.5公里。总投资355亿元。

该铁路等级为高速铁路，正线为双线，速度设计值250千米每小时，目标值300千米每小时，近期规划每天开行97对客车，年运送旅客2172万人，远期规划每天开行129对客车，年运送旅客2904万人<ref>{{cite web |url=http://suihua.dbw.cn/system/2010/11/22/052840725.shtml |title=2012年年底乘火车哈尔滨至大庆只需40分钟 |accessdate=2010-11-22}}</ref>。全线轨道采用I型板式无砟轨道，桥梁、路基工程各约占线路全长的50%。建成后，将采取小编组、高密度的运行方式。今后，哈齐客运专线还将与[[哈大客运专线|哈大客运专线]]等高速铁路衔接。

哈齐高铁贯穿[[哈大齐工业走廊|哈大齐工业走廊]]，建成后，对于[[振兴东北老工业基地|振兴东北老工业基地]]具有重要意义。

== 线路走向 ==
自哈尔滨站始，经[[霁虹桥|霁虹桥]]沿既有线右侧通过[[松花江|松花江]]。新建哈尔滨至齐齐哈尔铁路客运专线[[松花江|松花江]]特大桥，过松花江后线路自[[哈尔滨北站|哈尔滨北站]]沿既有线右侧前行，经[[松北区|松北区]]、[[呼兰区|呼兰区]]，进入[[肇东市|肇东市]]<ref name="chaiqian">[http://harbin.big5.northeast.cn/system/2008/11/21/051611539.shtml 昨日哈齊鐵路客運專線征地拆遷工作啟動] {{webarchive|url=https://archive.is/20141110025236/http://harbin.big5.northeast.cn/system/2008/11/21/051611539.shtml |date=2014-11-10 }}</ref>。

=== 城市 ===
由[[哈尔滨市|哈尔滨市]]开始，向西北方向经过[[肇东市|肇东市]]、[[安达市|安达市]]、[[大庆市|大庆市]]，终到[[齐齐哈尔市|齐齐哈尔市]]。

==== 车站 ====
[[哈尔滨站|哈尔滨站]]、[[哈尔滨北站|哈尔滨北站]]（松北区呼兰区之间，利民大道以西新建站）、[[肇东站|肇东站]]、[[安达站_(黑龙江省)|安达站]]、[[大庆东站|大庆东站]]（[[龙凤站|龙凤站]]扩建）、[[大庆西站|大庆西站]]（[[让湖路站|让湖路站]]扩建）、[[泰康站|泰康站]]、[[红旗营东站|红旗营东站]]（越行站）、[[齐齐哈尔南站|齐齐哈尔南站]]（[[齐齐哈尔市|齐齐哈尔市]][[龙沙区|龙沙区]]嫩江环路）。

== 历史 ==
=== 规划阶段 ===
2006年3月7日，[[铁道部|铁道部]]与[[黑龙江省|黑龙江省]]政府签署《关于加快黑龙江省铁路建设会谈纪要》，确定“十一五”期间将开工建设[[哈尔滨市|哈尔滨市]]至[[齐齐哈尔市|齐齐哈尔市]]之间的客运专线<ref>[http://chinaneast.xinhuanet.com/2006-04/18/content_6778522.htm 黑龙江与铁道部签署加快十一五铁路建设的会谈纪要] {{webarchive|url=https://web.archive.org/web/20120216151438/http://chinaneast.xinhuanet.com/2006-04/18/content_6778522.htm |date=2012-02-16 }}</ref>。
2007年10月，设计招标、筹资等前期启动<ref>[http://harbin.northeast.cn/system/2007/10/18/051016739.shtml 哈大齐工业走廊铁路工程前期工作现已启动] {{webarchive|url=https://web.archive.org/web/20081003031156/http://harbin.northeast.cn/system/2007/10/18/051016739.shtml |date=2008-10-03 }}</ref>。
2008年3月，哈齐客运专线铁路项目专家组到大庆市调研<ref>[http://www.daqing.gov.cn/daqinggov/73470466479620096/20080331/321971.shtml 哈大齐城际铁路项目专家组来我市调研]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }} 大庆市政府</ref>。
截至2008年4月前期工作已经完成三分之一，在黑龙江省与铁道部会谈纪要中升级为哈齐客运专线<ref>[http://www.hlj.gov.cn/zwgk/ztxx/lgyjd/jsdt/200804/t20080416_109286.htm 哈齐客运专线力争下半年开建全长约285公里]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }} 黑龙江省人民政府</ref>。
同年10月，哈齐客运专线项目专家组到[[齐齐哈尔市|齐齐哈尔市]]、[[肇东市|肇东市]]调研。
11月，地标勘测与拆迁工作开始<ref>[http://chinaneast.xinhuanet.com/dbtk/2008-11/03/content_14808540.htm 哈齐铁路专线开始勘测路标] {{webarchive|url=https://web.archive.org/web/20150713220112/http://chinaneast.xinhuanet.com/dbtk/2008-11/03/content_14808540.htm |date=2015-07-13 }}</ref><ref name="chaiqian" />。
2009年7月，哈齐客运专线可行性研究报告获得国家发改委正式批复。

=== 建设阶段 ===
2009年7月5日，哈齐客运专线建设动员大会在哈尔滨举行<ref>{{cite news|url=http://www.gov.cn/gzdt/2009-07/06/content_1358340.htm|title=哈尔滨至齐齐哈尔铁路客运专线暨哈西站建设开工|publisher=中国政府网}}</ref>，同年11月30日正式开始建设。2014年7月1日全线开始进行轨道的铺设，历时两个月，于同年9月22日铺通<ref>{{cite news|url=http://www.chinanews.com/cj/2014/09-22/6617751.shtml|title=世界首条中温带高寒高铁哈齐客运专线铺轨完成|publisher=中国新闻网}}</ref>。2015年3月完成全部主体工程<ref>{{cite news|url=http://www.dqdaily.com/news/2015-03/19/content_2601005.htm|title=哈齐客运专线主体完工 大庆人今夏可在家门口坐高铁|publisher=大庆网|deadurl=yes|archiveurl=https://web.archive.org/web/20160304190440/http://www.dqdaily.com/news/2015-03/19/content_2601005.htm|archivedate=2016-03-04}}</ref>，于同年5月23日开始进行联调联试<ref name="cnr20150524"/>，并于同年7月13日开始进行试运行<ref name="xinhuanet20150713"/>。

由于哈齐、[[沈丹客运专线|沈丹]]客专的开通会大幅调整东北地区的列车运行图，进出东北的各次旅客列车及沈、哈两局管内旅客列车车票均暂售至2015年8月27日。<ref>{{cite news|url=http://enews.xwh.cn/shtml/xwhb/20150709/188693.shtml|title=沈阳铁路局暂停预售8月28日以后的火车票|publisher=[[新文化报|新文化报]]|date=2015-07-09|accessdate=2015-07-24}}{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite news|url=http://henan.china.com.cn/latest/2015/0717/596327.shtml|title=出行动态：部分经停泰康站列车取消 暑期增开多趟列车|publisher=东北网|date=2015-07-17|accessdate=2015-07-24}}</ref>

=== 运营阶段 ===
哈齐客运专线于2015年8月17日开通运营，初期按时速250公里的速度等级开行20对[[普通动车组列车|动车组列车]]<ref>{{cite news|url=http://www.chinanews.com/cj/2015/08-17/7471476.shtml|title=中国最北高铁哈齐高铁正式运营通车|publisher=中国新闻网}}</ref>，其中包含哈尔滨至齐齐哈尔15对，哈尔滨至大庆3对，以及齐齐哈尔至北京的2对跨线列车<ref>{{cite news|url=http://hlj.ifeng.com/news/detail_2015_08/17/4237436_0.shtml|title=哈齐高铁今日开通运营 首趟车票售罄|publisher=凤凰网}}</ref>(北京局担当D25/26次，哈尔滨局担当D29/30次)。哈齐客运专线本线车辆初期使用[[和谐号CRH5型电力动车组|CRH5A型动车组]]，从哈尔滨站至齐齐哈尔南站最快用时85分钟<ref>{{cite news|url=http://m.news.cntv.cn/2015/08/17/ARTI1439765093475454.shtml|title=哈齐高铁今日将开通运营 设计时速250公里|publisher=央视网}}</ref>。此外，哈尔滨西站也将承担一部分哈齐动车组的运营。

截止到2017年1月5日铁路调整运行图，哈齐客专本线每日将开行约30对动车组列车，包括开行2对齐齐哈尔-北京动车组(D25/26次每年11月中旬到第二年三月中旬改为哈尔滨西终到)，1对齐齐哈尔-沈阳高峰线动车组和1对齐齐哈尔-珲春动车组列车。

== 争议 ==
由于工程设计方案恐将拆除或改建近百年历史的[[霁虹桥|霁虹桥]]，引起了哈尔滨文物保护专家和市民的关注，部分市民和专家建议修改设计方案，保护相关文物建筑<ref>[http://special.dbw.cn/system/2009/07/04/051993654.shtml 东北网：霁虹桥是否应该拆？]</ref>。

该工程原设计方案计划拆除并重建百年历史的[[滨洲铁路桥|滨洲铁路桥]]，经过文物保护工作者和志愿者的共同努力，建设方最终更改原设计方案，保留大桥，并新建一座铁路桥供哈齐客运专线使用<ref>[http://www.harbinnews.com/201001/20100114/20100114FBFD4F8C7CE5921BDDCB5911E2543C44.html 哈报新闻网：百岁滨洲桥将卸载变身景观桥]{{dead link|date=2017年12月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>[http://imharbin.com/367 大话哈尔滨：永远的滨洲铁路桥] {{webarchive|url=https://web.archive.org/web/20100310104459/http://imharbin.com/367 |date=2010-03-10 }}</ref>。
