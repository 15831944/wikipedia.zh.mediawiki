== 概览 ==
哈里杰是埃及西部现代化程度最高的绿洲。主城镇的运行高度依赖各种现代化的设施，而看不到任何古建筑。虽然被绿洲包围，但没有任何绿洲的感觉。不像埃及的其他绿洲，哈里杰被一片广袤的[[棕榈|棕榈树]]、[[合欢|合金欢树]]和[[枣|枣树]]林环绕着，还有许多残存的野生物种栖息在此。

== 人口 ==
哈里杰原住民属于贝沙族，尽管阿拉伯语是主要语言，但他们说自己的贝沙语——一种[[亚非语系|亚非语系]]的语言。

== 交通 ==
一套定时公交系统连接着哈里杰绿洲和其他西部绿洲和埃及其他绿洲。1996年起，哈里杰—[[基納|基纳]]（尼罗河谷）—[[塞法杰|塞法杰港]]（红海）的铁路线开始运行。
