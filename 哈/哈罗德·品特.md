'''哈罗德·品特'''<small>，[[名誉勋位|CH]]，[[CBE|CBE]]</small>（{{lang-en|'''Harold Pinter'''}}，{{bd|1930年|10月10日|2008年|12月24日|catIdx=Pinter, Harold}}），[[英國|英國]][[劇作家|劇作家]]及[[劇場導演|劇場導演]]，他的著作包括[[舞台劇|舞台劇]]、[[廣播|廣播]]、[[電視|電視]]及[[電影|電影]]作品。品特的早期作品經常被人們归入[[荒誕派戲劇|荒誕派戲劇]]。他也是2005年[[諾貝爾文學獎|諾貝爾文學獎]]的获得者。

== 早年生平及劇作 ==
品特出生于[[倫敦|倫敦]][[哈克尼區|哈克尼區]]一个犹太工人家庭，曾就讀於當地學校，並短暫於[[皇家戲劇藝術學院|皇家戲劇藝術學院]]就讀。年輕時，品特曾出版其[[詩|詩]]作；並以藝名「大衛·巴倫」 (David Baron) 作登台[[演員|演出]]。他的首部[[劇本|劇本]]《一间房子》 （''The Room''），于1957年由[[布里斯托爾大學|布里斯托爾大學]]學生演出。

1958年品特完成《生日派對》（''The Birthday Party''）一剧，儘管《星期日泰晤士報》的演藝評論家 Harold Hobson給該劇不錯的評價，但公众反应不佳。不過隨著1960年《看門人》（''The Caretaker''）一劇的成功，讓品特聲名大噪後，《生日派對》再次受人關注。這些劇的情節往往把一些無傷大雅的情況，逐漸變壞成荒誕的局面，劇中人物的行為有時令觀眾、甚至劇中其他角色費解。于是這兩部劇作，連同其他如《回鄉》（''The Homecoming''）等早期作品，有時被稱為“威胁喜劇”，归入荒诞派戏剧，品特本人也於早期就深受荒诞派戏剧代表人物[[塞繆爾·貝克特|塞繆爾·貝克特]]的影響，後來兩人更成為深交。品特於1966年獲[[CBE|司令勳章]]。

== 新路線 ==
品特於1970年代起有更多機會導演，於1973年他成為[[國立皇家劇院|國立皇家劇院]]的副導演。他後期的劇作都比較短，題材也比較[[政治|政治]]化，往往在諷諭強權壓迫。品特也於該時期開始對政治直言不諱，明顯地採取[[左翼|左翼]]立場。他繼續把侵犯[[人權|人權]]及強權壓迫等議題放在公眾視線上。品特常寄函於英國《[[衛報|衛報]]》及《[[獨立報|獨立報]]》等報章。

於1985年品特跟[[美國|美國]]劇作家[[亞瑟·米勒|亞瑟·米勒]]造訪[[土耳其|土耳其]]，並與許多政治壓迫的受害者見面。在美國[[大使館|大使館]]一項向米勒致敬的活動中，品特並沒有跟其他客人寒暄，反而談論被壓迫者的生殖器官遭受電擊的場面，結果被趕出館外。米勒也隨同他一起離場，以示支持。品特於土耳其所見到的迫害，及對[[庫爾德語|庫爾德語]]的抑制，成為了1988年劇作 《山脉语言》（''Mountain Language''）的靈感。

1999年品特對[[北約|北約]]參加[[科索沃戰爭|科索沃戰爭]]大表不滿。他也公開反對[[阿富汗戰爭_(2001年)|美國攻擊阿富汗]]及2003年的[[美伊戰爭|美伊戰爭]]。2002年他獲得[[名譽勳位|名譽勳位]]（此前他曾拒絕[[爵士|爵士]]头衔）。他是[[RESPECT|RESPECT]]聯盟的支持者。於2005年他宣佈將退出劇壇，投身政治活動。2005年10月，[[瑞典文學院|瑞典文學院]]宣佈品特為2005年諾貝爾文學獎得主，颁奖理由是：「揭示了日常絮談中的危機、強行打開了壓迫的封閉房間。」12月7日，品特因病住院不能亲自到[[斯德哥尔摩|斯德哥尔摩]]发表获奖演说。但是他在事先录制的讲话中，严厉抨击了英国和美国的外交政策，还说[[乔治·沃克·布什|布什]]和[[布莱尔|布莱尔]]应该就入侵伊拉克接受[[国际法庭|国际法庭]]的审讯。[https://web.archive.org/web/20051210021314/http://news8.thdo.bbc.co.uk/chinese/simp/hi/newsid_4500000/newsid_4508700/4508786.stm]

== 電影作品 ==
品特的首部[[電影劇本|電影劇本]] ''The Servant'' 寫於1962年。其後他也為 ''The Go-Between'' 以及[[法國中尉的女人|法國中尉的女人]]等電影編寫劇本。他也為[[馬塞爾·普魯斯特|馬塞爾·普魯斯特]]的《追憶逝水年華》寫出電影劇本，不過該劇從未拍成電影。他有數部舞台劇作也被搬上大銀幕：如《看門人》（''The Caretaker''，1963年）、《生日晚会》（''The Birthday Party''，1968年）、《回鄉》（''The Homecoming''，1973年）及《背叛》（''Betrayal''，1983年）。

== 私人生活 ==
1977年，品特因為移情別戀，與妻子離異，惹來不少緋聞。他於1978年寫成的劇作《背叛》（''Betrayal''）有時被看成這段新戀情的描繪，但其實是根據他早年與一位電視女主持人的七年愛情而寫。

他也因為劇場導演{{Link-en|彼得·霍爾 (導演)|Peter Hall (theatre director)|彼得·霍爾}}1983年出版的日記裡經常把品特寫成醉漢而鬧翻，不過兩人後來修好。

品特熱愛[[板球|板球]]，是 Gaieties Cricket Club 的主席；他同時也是[[國家世俗協會|國家世俗協會]]的榮譽準會員。

== 逝世 ==
{{Wikinews|英國著名劇作家哈羅德·品特逝世 享年78歲}}
品特在2008年[[平安夜|平安夜]]逝世，他的第二任妻子在12月25日對外宣布，享壽78歲的他已經因癌症過世。
