'''哈米吉多頓'''（{{lang-en|'''Armageddon'''}}）是[[基督教|基督教]]《[[新約聖經|新約聖經]]·[[啟示錄|啟示錄]]》所[[圣经预言|预言]]的[[基督教末世论|末世末期]]善惡對決的最終戰場，只一次出現在《启示录》第16章所记述的[[異象|異象]]中。

== 来源 ==
這個詞來自[[希伯來語|希伯來語]]中一座名為「[[米吉多|米吉多]]」的山（{{lang|en|Megiddo}}，位於現今的[[以色列|以色列]]）。哈米吉多頓的發音來自[[希臘語|希臘語]]<span lang="el">Ἁρμαγεδών</span>（{{IPA|/harmagedōn/}}），由于拉丁语中H发音在共和国晚期已经接近变成完全哑音，[[英語|英語]]由于继承了拉丁语的词汇，在[[英語|英語]]中省略了詞首的氣音而變成Armageddon。

米吉多位于一个交叉路口，往北可以到[[欧洲|欧洲]]，往东可達[[土耳其|土耳其]]和过去的[[波斯|波斯]]地区，往南則可到[[非洲|非洲]]。米吉多地形处于一个隘口，是一个易守难攻的所在，故是古代兵家必争之地。夺得米吉多就等于夺得一千个城镇<ref>詹姆斯·普里查德，《有关旧约全书的古代近东文献》，237页，美国新泽西州普林斯顿，1969年</ref>，圣经说万国的王会聚到这里指的是这个意思。

== 意义 ==
根據《聖經》記載，全能者會在此擊敗[[魔鬼|魔鬼]]和「天下眾王」。第六個天使把瓶倒在幼發拉底大河上，河水就乾了，要給東方的眾王預備道路。
	我又看見三個污穢的靈，好像青蛙，從龍口、獸口並假先知的口中出來。
	他們本是鬼魔的靈，施行奇事，出去到地上並全世界的眾王那裏，聚集他們在　神─　全能者的大日爭戰。
	看哪，我來像賊一樣。那警醒、看守衣服、免得赤身而行、叫人見他羞恥的有福了。
	他便叫眾王聚集在一處，希伯來話叫作哈米吉多頓。
	第七個天使把瓶倒在空中，就有大聲音從天上殿中的寶座上出來，說：「成了。」
	就有聲音、雷轟、閃電，又有大地震，自從人在地上以來，沒有這樣利害、這樣大的地震。
	那大城裂為三段，列國的城也都倒塌了；　神也想起大巴比倫來，要把那盛自己烈怒的酒杯遞給她。

== 艺术创作 ==
哈米吉多頓也是[[科幻小說|科幻小說]]、[[科幻電影|電影]]中常見的題材，例如《[[絕世天劫|-{zh-hans:世界末日;zh-hk:絕世天劫;zh-tw:世界末日;}-]]》。
經常被放在一起的地名包括有：[[瑪各|瑪各]](可能指[[蒙古|蒙古]])、[[歌革|歌革]](可能指具[[KGB|KGB]]背景的[[普京|普京]])、[[羅斯|羅斯]](即[[俄羅斯|俄羅斯]])、[[米設|米設]](即[[莫斯科|莫斯科]])、[[歌蔑|歌蔑]]、[[土巴|土巴]]([[托博爾斯克|托博爾斯克]])。還有[[約沙法谷|約沙法谷]]。([[以西結書|以西結書]]、第卅八章)
