'''哈比村'''（英語：'''Hobbiton Movie Set'''）是[[北島_(紐西蘭)|紐西蘭北島]]的觀光景點，位於{{link-en|馬塔馬塔|Matamata}}小鎮。原是《[[魔戒電影三部曲|魔戒]]》與《[[霍比特人电影系列|哈比人]]》電影系列的取景地<ref>{{Cite web|url=http://travel.china.com/hotspot/13000493/20171221/31849463_all.html|title=中土世界新西兰漫游指南  霍比屯，一个美丽的童话之乡|accessdate=2018-01-21|author=|date=2017-12-21|publisher=中华网|archive-url=https://web.archive.org/web/20180121184634/http://travel.china.com/hotspot/13000493/20171221/31849463_all.html|archive-date=2018-01-21|dead-url=yes}}</ref><ref>{{Cite web|url=http://jingji.cntv.cn/2013/03/04/ARTI1362365151266898.shtml|title=跟随霍比特人穿越中土世界 邂逅意外之旅|accessdate=2018-01-21|date=2013-03-04|publisher=央视网}}</ref>，電影結束拍攝後設立「Hobbiton Movie Set Tours」旅遊專案，由電影製片人[[彼得·傑克森|彼得·傑克森]]及當地的亞歷山大（Russell Alexander）家族共同投資推出<ref>{{Cite news|url=http://www.chinatimes.com/newspapers/20150125000133-260210|title=哈比人炒熱紐西蘭|author=洪紹運|date=2015-01-25|work=|newspaper=中時電子報|accessdate=2017-11-01|language=zh-TW}}</ref>；該地便成為魔戒電影系列中唯一被完整保留下來的電影場景<ref name=":4">{{Cite news|url=https://udn.com/news/story/7176/913177|title=紐西蘭旅遊不能放過的明星景點：哈比村|date=2015-05-20|newspaper=聯合新聞網|accessdate=2018-01-21}}</ref>。

== 歷史 ==
[[File:Hobbiton mill and double-arched bridge.jpg|缩略图|
哈比村裡的磨坊和雙拱橋。
]]
[[File:The_Green_Dragon_(1).jpg|缩略图]]
1978年亞歷山大家族搬至該地，經營著一個占地約1250英畝的[[農場|農場]]。

1998年9月因《[[魔戒首部曲：魔戒現身|魔戒首部曲：魔戒現身]]》電影開拍，彼得·傑克森與勘景團隊發現坐著飛機尋找適用於拍攝[[J·R·R·托爾金|托爾金]]小說中[[哈比人|哈比人]]的故鄉——[[夏爾_(中土大陸)|夏爾]][[哈比屯|哈比屯]]的地點，最後發現了亞歷山大的農場<ref name=":0">{{Cite news|url=http://www.hobbitontours.com/en/our-story/|title=Our Story|author=|date=|work=Hobbiton Movie Set 官網|accessdate=2017-11-01|language=en}}</ref><ref>{{Cite news|url=http://www.theonering.net/torwp/2012/11/21/65567-how-a-sleepy-country-town-became-hobbiton/|title=How a sleepy country town became Hobbiton|date=2012-11-21|newspaper=TheOneRing.net|accessdate=2018-02-24|language=en-US}}</ref>，這裡的巨樹和湖泊都與小說裡對描述的哈比屯十分接近<ref>{{Cite news|url=https://travel.ettoday.net/article/1298003.htm|title=不是魔戒迷也要去！紐西蘭「不存在的村莊」奇幻哈比屯|author=于佳云|date=2018-11-06|newspaper=ETtoday 旅遊雲|accessdate=2018-11-07|language=zh-TW}}</ref>。在與地主伊恩·亞歷山大（Ian Alexander）協商之後，電影劇組獲得許可在該處拍攝電影<ref name=":1">[https://www.youtube.com/watch?v=y2RuxW5eZ6c Peter Jackson《哈比人：意外旅程》幕後花絮第五集]，[[朱學恆|朱學恆]]翻譯</ref>。

布景團隊先依照概念設計師[[阿兰·李_(插画家)|艾倫·李]]和{{link-en|約翰·豪威|John Howe (illustrator)}}的[[素描|素描]]插圖建立一個4~5平方米的模型以預覽<ref name=":6">{{Cite web|url=http://news.mtime.com/2016/12/18/1564537-all.html|title=永不落幕的中土传奇 《魔戒首部曲》上映15周年回望|accessdate=2018-07-08|date=2016-12-19|publisher=Mtime时光网}}</ref>。而後便在[[紐西蘭軍隊|紐西蘭的軍隊]]協助底下，於1999年3月開始在該地用[[聚苯乙烯|保麗龍]]和胶合板材質實地搭建夏爾哈比屯的場景，包括39個臨時的哈比人洞穴<ref name=":0" /><ref>{{Cite web|url=http://archives.theonering.net/perl/newsview/1/923891184|title=Hobbiton Is Being Built|accessdate=2017-11-01|author=|date=1999-04-10|publisher=}}</ref>，一座[[磨坊|磨坊]]和[[雙拱橋|雙拱橋]]。[[美術指導|美術指導]][[丹·漢納|丹·漢納]]指出這個布景應該要看上去很自然，像是已經存在很久的樣子，如[[蔬菜|蔬菜]]和[[果樹|果樹]]上的水果都要是真實的，而且為此拍攝團隊等了一年之後才正式在該處開拍電影裡的相關鏡頭，以保證所有的人工痕跡都被[[大自然|大自然]]給抹去<ref name=":6" /><ref>{{Cite web|url=http://dailynews.sina.com/bg/ent/film/sinacn/m/2011-12-18/10443510568.html|title=《魔戒》十年：你可能不知道的30個《魔戒》秘密|accessdate=2018-07-08|date=2011-12-17|publisher=新浪網}}</ref>。

隨著魔戒電影拍攝結束，哈比屯的相關場景也隨之拆除<ref name=":1" />。但仍留下17個半毀損的塑膠門面，於2002年開放參觀；其門票收入除稅收外，共歸[[新線影業|新線影業]]、亞歷山大家族及原著小說的作者[[托爾金家族|托爾金家族]]共同擁有<ref>{{Cite news|url=http://www.epochtimes.com/b5/5/7/25/n997016.htm|title=尋訪電影「魔戒」場景 哈比人的故鄉|author=尼克|date=2005-07-25|work=|accessdate=2017-11-01|language=zh-TW}}</ref>。光在《[[魔戒三部曲：王者再臨|魔戒三部曲：王者再臨]]》上映的2003年內，就有5萬多名遊客參觀了哈比村<ref name=":3">{{Cite news|url=https://cn.nytimes.com/travel/20121215/c15hobbiton/|title=新西兰“霍比屯”之旅|date=2012-12-15|newspaper=纽约时报中文网|accessdate=2018-01-03|language=zh}}</ref>。

2009年，因應《[[哈比人電影系列|哈比人電影系列]]》的籌備階段，電影美術部門與[[植物|植物]]部門花了近兩年的時間在同一地點重新搭建哈比屯，由[[木材|木材]]、[[石頭|石頭]]和[[磚塊|磚塊]]的堅固材質建造，以便永久保存<ref name=":1" />。2011年於該處拍攝了電影裡的比爾博奔出家門及返鄉等場景。

在《哈比人》電影拍攝結束後，該處即作為一個永久的旅遊景點「哈比村」保留下來<ref>{{Cite news|url=http://travel.cnn.com/explorations/life/hanging-out-hobbiton-539076/|title=Hanging out in Hobbiton: 'The Hobbit''s stunning movie set|author=Adam Bray|date=2012-05-21|work=CNN Travel|accessdate=2017-11-01}}</ref><ref name=":5">{{Cite web|url=http://travel.china.com/hotspot/13000493/20170929/31532169_all.html|title=《霍比特人》中的夏尔镇竟然是这样打造出来的！|accessdate=2018-01-21|date=2017-09-29|publisher=中华网|archive-url=https://web.archive.org/web/20180121184637/http://travel.china.com/hotspot/13000493/20170929/31532169_all.html|archive-date=2018-01-21|dead-url=yes}}</ref>，開放給來自全球的遊客。

2012年11月，哈比村內的「[[綠龍酒店|綠龍酒店]]」開幕<ref name=":0" /><ref name=":5" />，[[紐西蘭總理|紐西蘭總理]][[約翰·凱伊|約翰·凱伊]]與幾位在《哈比人》中飾演[[矮人_(中土大陸)|矮人]]的演員皆參與了[[開幕典禮|剪綵]]儀式<ref>{{Cite news|url=http://www.theonering.net/torwp/2012/11/29/66231-green-dragon-opens-in-hobbiton-for-thirsty-fans-new-zealand/|title=Green Dragon opens in Hobbiton for thirsty fans, New Zealand|date=2012-11-29|newspaper=TheOneRing.net|accessdate=2018-02-24|language=en-US}}</ref><ref>{{Cite news|url=https://www.huffingtonpost.com/2012/12/12/green-dragon-pub_n_2287852.html|title=The Green Dragon Inn From 'The Hobbit' Is Now A Working Pub With Real Booze (PHOTOS)|last=Tepper|first=Rachel|date=2012-12-12|newspaper=Huffington Post|accessdate=2018-02-24|language=en-US}}</ref>。

== 景點 ==
哈比村占地約4.8公頃<ref name=":5" />，共有44個哈比人洞穴<ref name=":2">{{Cite news|url=https://travel.ettoday.net/article/680390.htm?t=%E3%80%8A%E5%93%88%E6%AF%94%E6%9D%91%E3%80%8B%E5%94%AF%E4%B8%80%E4%BA%BA%E9%80%A0%E6%A8%B9%E6%98%AF%E5%8F%B0%E7%81%A3%E8%A3%BD%E3%80%8020%E8%90%AC%E7%89%87%E6%A8%B9%E8%86%A0%E8%91%89%E6%89%8B%E5%B7%A5%E9%BB%8F|title=《哈比村》唯一人造樹是台灣製　20萬片樹膠葉手工黏|author=施羽璇|date=2016-04-19|work=|newspaper=ETtoday 旅遊雲|accessdate=2017-11-01|language=zh-TW}}</ref><ref>{{Cite news|url=https://travel.ettoday.net/article/157615.htm?t=%E3%80%8C%E4%B8%AD%E5%9C%9F%E4%B8%96%E7%95%8C%E3%80%8D%E7%9C%9F%E7%9A%84%E5%AD%98%E5%9C%A8%E3%80%80%E7%B4%90%E8%A5%BF%E8%98%AD%E5%93%88%E6%AF%94%E4%BA%BA%E9%96%8B%E6%9D%91%E9%96%80%EF%BC%9A%E6%AD%A1%E8%BF%8E%E5%8F%83%E8%A7%80|title=「中土世界」真的存在　紐西蘭哈比人開村門：歡迎參觀|author=游琁如|date=2013-01-27|work=|newspaper=ETtoday 旅遊雲|accessdate=2017-11-01|language=zh-TW}}</ref><ref>{{Cite web|url=http://paper.udn.com/udnpaper/POE0024/231480/web/|title=《哈比人：意外旅程》 再掀紐西蘭旅遊熱|accessdate=2017-11-01|author=秦雅如|date=2013-02-12|publisher=行遍天下旅遊電子報}}</ref><ref name=":3" />；其他相關設施尚有：
* [[袋底洞|袋底洞]]：是哈比村中唯一可允許遊客入內拍攝的哈比人洞穴<ref name=":4" />。其上方的[[橡樹|橡樹]]是該園區裡唯一的人造樹，由鐵和塑膠製成，上面20萬片的塑膠葉產自[[台灣|台灣]]<ref name=":2" /><ref>{{Cite news|url=http://news.ltn.com.tw/news/supplement/paper/1012092|title=歡迎光臨魔戒世界─紐西蘭．北島|author=孫梓評|date=2016-07-18|work=自由時報電子報|accessdate=2017-12-04}}</ref>。在小說與電影中是哈比人[[比爾博·巴金斯|比爾博]]和[[佛罗多·巴金斯|佛羅多]]的家。
* [[宴會樹|宴會樹]]：是哈比村中最挺拔的大樹，已有100多年的樹齡。在電影裡比爾博於該樹下演說完便戴上[[至尊魔戒|魔戒]]消失。<ref name=":2" />
* 綠龍酒店
