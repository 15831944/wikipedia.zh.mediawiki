《'''哈瑪斯之子'''》（{{lang-he|הנסיך הירוק}}，{{lang-en|''The Green Prince''}}）是一部2014年以色列、英國、德國合拍的紀錄片，由納達夫·謝爾曼（Nadav Schirman）執導。本片改編自[[摩薩·哈珊·約瑟夫|摩薩·哈珊·約瑟夫]]的自傳《哈瑪斯-{}-之子：恐怖組織頭號叛徒的告白》（''Son of Hamas: A Gripping Account of Terror, Betrayal, Political Intrigue, and Unthinkable Choices''）。<ref>{{Cite web |url = https://www.haaretz.com/1.5051790 |title = Haaretz Exclusive: Hamas Founder's Son Worked for Shin Bet for Years |author = Avi Issacharoff |date = 2010-02-24 |publisher = Haaretz |language = en}}</ref><ref>{{Cite web |url = https://www.haaretz.com/1.5051710 |title = When Palestinians Keep Israelis Safe |author = Amos Harel |date = 2010-02-24 |publisher = Haaretz |language = en}}</ref>

該紀錄片獲得四個獎項。<ref>{{Cite web |url = https://www.imdb.com/title/tt2304915/awards?ref_=tt_awd |title = The Green Prince |publisher = imdb |language = en}}</ref>
