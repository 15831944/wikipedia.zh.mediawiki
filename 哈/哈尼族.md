'''哈尼族'''是[[中国西南地区|中国西南]][[少数民族|少数民族]]，[[中华人民共和国第五次全国人口普查#民族构成|列中国第16大民族]]。[[云南省|云南]]是哈尼族世居地。根据[[中华人民共和国第五次全国人口普查|2000年人口普查]]，中国大陆哈尼族人口有1,440,029人，其中云南有1,424,990人，占哈尼族人口的98.96%。哈尼族主要分布于云南西南部礼社江下游[[红河|红河]]西侧[[哀牢山|哀牢山]]区新平、镇源、墨江、元江、红河、元阳、绿春、金平、江城等县。[[山东省|山东]]、[[江苏省|江苏]]、[[湖南省|湖南]]、[[四川省|四川]]和[[广东省|广东]]五省哈尼族人口均超过千人，其余各个[[中国一级行政区|省市区]]及现役军人中均有哈尼族人口分布。中国哈尼族使用[[哈尼语|哈尼语]]，原来没有文字，1957年创制了以拉丁字母为基础的文字。哈尼族先民为[[羌族|羌族]]，在海拔2500米山區閞墾[[梯田|梯田]]，種植[[水稻|水稻]]。哈尼族（{{vie|v=Người Hà Nhì|hn=𠊛何貳}}）也是[[越南民族|越南54个民族]]之一，人口17,535（1999年普查）。

哈尼人的一支'''阿卡人'''（自稱為雅尼人），在[[缅甸|缅甸]]、[[老挝|老挝]]、[[泰国|泰国]]都有分布，缅甸、老挝称其为高族（{{my|m=ကော်|y=kau}}）。

== 族源 ==
哈尼族可能源于古代氐羌族群。漢文史籍中，對哈尼族有“和夷”、“和蠻”、“和泥”、“離泥”、“倭泥”、“哈尼”、“斡泥”等稱呼。

他们在老挝叫[[老松族|老松族]]，有“山上的民族”之意，在西双版纳自称阿卡，意思是“我是远方来的人”(另一說他們在古代被[[傣族|傣族]]打敗成為他們的奴隸，因此被稱為阿卡)。

== 語言 ==
{{main|哈尼語}}
哈尼語屬于[[漢藏語系|漢藏語系]][[藏緬語族|藏緬語族]]的[[彝語支|彝語支]]，同[[彝語|彝語]]、[[拉祜語|拉祜語]]、[[傈僳語|傈僳語]]比較接近。分為哈雅、碧卡、豪白三個方言，彼此差別較大，相互通話困難。哈尼族本無文字，有些地方曾經刻木、[[結繩記事|結繩記事]]。1957年，中國政府為其創立了一種以[[拉丁字母|拉丁字母]]為基礎的拼音文字。

== 文化 ==
哈尼族的民間文化藝術形式豐富多樣，包括口頭文學、舞蹈等。口頭文學中，有神話傳說《創世》講述萬物起源，《洪水記》講述人類戰勝洪水，《哈尼祖先過江來》講述哈尼族歷史；有詩歌“拉八熱”在慶典、祭祀等莊重場合吟唱，“阿基估”為男女對唱的山歌。

== 宗教 ==
哈尼族相信[[万物有灵|万物有灵]]，包括自然崇拜、祖先崇拜和靈魂崇拜。

== 經濟 ==
哈尼族的分布地區多為海拔800至2500米的山區，氣候溫和，雨量豐沛，物產豐富。哈尼族長久以來在此閞墾[[梯田|梯田]]種植[[水稻|水稻]]，而此梯田謂之[[红河哈尼梯田|红河哈尼梯田]]，主要分佈在[[雲南|雲南]]南部[[红河哈尼族彝族自治州|紅河州]][[元陽縣|元陽縣]]、[[紅河縣|紅河縣]]、[[金平縣|金平縣]]和[[綠春縣|綠春縣]]。
