|archive-url=https://web.archive.org/web/20171222051801/http://www.shangbaoindonesia.com/indonesia-berita/%E9%9B%85%E4%BA%AC%E5%93%88%E6%9E%97%E6%9C%BA%E5%9C%BA%E9%9A%86%E9%87%8D%E4%B8%BE%E8%A1%8C%E6%88%91%E5%9B%BD%E7%A9%BA%E5%86%9B%E6%88%90%E7%AB%8B71%E5%91%A8%E5%B9%B4%E5%BA%86%E5%85%B8%E9%98%85%E5%85%B5.html 哈利姆机场以[[印度尼西亚民族英雄|民族英雄]]、空军副元帅{{le|哈利姆·珀达纳库苏马|Halim Perdanakusuma}}为名<ref name="Sejarah">{{cite web |url=https://www.kaskus.co.id/thread/000000000000000016746161/sejarah-bandara-halim-perdana-kusuma/ |title=Sejarah Bandara Halim Perdana Kusuma |last= |first= |date= |website=kaskus.co.id |publisher= |access-date=2017-07-28 |quote=}}</ref>。现在机场是数量庞大的涡轮螺旋桨飞机、[[包机|包机]]及[[通用航空|通用]]航空公司的基地，也是[[印尼空军|印尼空军]]的一个主要[[空军基地|基地]]，印尼空军有4个中队驻扎在这里<ref>{{cite web |url=https://bandarahalimperdanakusuma.wordpress.com/alamat-bandara-halim-perdana-kusuma-jakarta/ |title=BANDARA UDARA HALIM PERDANA KUSUMA JAKARTA 2017/2018 |last= |first= |date= |website=bandarahalimperdanakusuma.wordpress.com |publisher= |access-date=2017-07-28 |quote=Bandar udara ini juga digunakan sebagai markas Komando Operasi Angkatan Udara I TNI-AU.}}</ref>：{{link-id|印尼空军第2航空中队|Skadron Udara 2|第2中队}}、{{link-id|印尼空军第17航空中队|Skadron Udara 17|第17中队}}、{{link-id|印尼空军第31航空中队|Skadron Udara 31|第31中队}}和{{link-id|印尼空军第45航空中队|Skadron Udara 45|第45中队}}。

最初机场名为“'''吉利利坦机场'''”（Tjililitan）以其所在的镇区为名<ref name="HandoyoSudibyo2011">{{cite book|author1=Singgih Handoyo|author2=Dudi Sudibyo|title=Aviapedia: ensiklopedia umum penerbangan|url=http://books.google.com/books?id=MgG22qh4OK0C&pg=PA151|accessdate=2012-05-22|year=2011|publisher=Penerbit Buku Kompas|isbn=978-979-709-547-5|pages=151–}}</ref>。首班从这里起航的飞机是{{le|荷兰皇家印度航空|KNILM}}1928年11月1日从该机场经停[[万隆|万隆]]飞往[[泗水_(印尼)|泗水]]的航班。1929年，荷兰皇家印度航空创下了当时跨大陆飞行的世界纪录，耗时10天跨越14,500公里，从[[荷兰|荷兰]]飞到[[巴达维亚|巴达维亚]]，经停多个国家后飞机在吉利利坦机场平安降落。5年后的1934年，从荷兰到巴达维亚的飞行时间缩短到5.5天，使用机型为[[道格拉斯DC-2|道格拉斯DC-2]]。<ref name="merdeka"></ref>

1950年6月20日，[[荷属东印度|荷兰]]将机场交给印度尼西亚；1952年更名为“'''哈利姆·珀达纳库苏马空军基地'''”<ref name="Sejarah"></ref><ref name="halimperdanakusuma-airport.co.id">{{cite web |url=http://halimperdanakusuma-airport.co.id/en/general/about-us |title=Halim Perdanakusuma Airport |last= |first= |date= |website=halimperdanakusuma-airport.co.id |publisher= |access-date=2017-07-28 |quote=}}</ref>。1984年8月13日，[[印尼机场公司|印尼机场管理局第二机场经营公司]]开始负责机场的运作<ref name="cnnindonesia"></ref>。

作为一个民用机场，哈利姆机场与{{le|马腰兰机场|Kemayoran Airport}}一同是雅加达的两个主要机场之一，直到1985年在[[唐格朗|唐格朗]]的新机场[[苏加诺-哈达国际机场|苏加诺-哈达国际机场]]建成<ref>{{cite web |url=https://tiketturindo.com/blog/index.php/2016/03/03/sejarah-bandar-udara-internasional-soekarno-hatta/ |title=Sejarah Bandar Udara Internasional Soekarno-Hatta |last= |first= |date= |website=www.tiketturindo.com |publisher= |access-date=2017-07-28 |quote=}}</ref>。在那之前哈利姆机场服务所有飞往[[雅加达|雅加达]]的国际航线，而马腰兰机场则服务国内航班。1985年马腰兰机场被关闭，哈利姆机场成为雅加达的次要机场，主要任务是在之后的29年内为包机航班、通用航空提供服务以及作为飞行学校的基地。1990年代，印尼民航总局规定哈利姆机场将只为非定期航班以及100座以下的小型飞机定期航班提供服务。

为了缓解[[苏加诺-哈达国际机场|苏加诺-哈达国际机场]]的拥挤状态，哈利姆机场管理方宣布每小时为定期航班提供60架次的服务<ref>{{cite web |url=http://jabar.tribunnews.com/2013/12/21/layani-penerbangan-komersial-bandara-halim-kebut-renovasi |title=Layani Penerbangan Komersial, Bandara Halim Kebut Renovasi |date=2013-12-21}}</ref>，2013年前往麦加的朝觐者使用了该机场<ref>{{cite web |url=http://www.thejakartapost.com/news/2013/07/24/halim-undergoes-renovation-ease-air-traffic-soekarno-hatta.html |title=Halim undergoes renovation to ease air traffic at Soekarno–Hatta |date=2013-07-24}}</ref>。从2014年开始，哈利姆机场为国内定期航班提供服务，限制年旅客量为220万人，2013年仅有20余万人（承载能力过剩）<ref>{{cite web |url=http://jabar.tribunnews.com/2013/12/21/layani-penerbangan-komersial-bandara-halim-kebut-renovasi |title=Layani Penerbangan Komersial, Bandara Halim Kebut Renovasi |date=2013-12-21}}</ref>，到2016年，全年旅客量则高达560余万，远超其服务承载能力。

目前印尼政府正在计划修建一条机场快速铁路连接哈利姆机场和[[苏加诺-哈达国际机场|苏加诺-哈达国际机场]]，2013年12月进行了可行性研究。此外，雅加达政府计划拓展到哈利姆机场的道路，以缓解{{le|查旺|Cawang, Kramat Jati, East Jakarta}}附近地区拥堵的交通。<ref name="express train"></ref>

==机场快速铁路==
机场到机场快速铁路的可行性研究已经完成，准备进行资格预审。快速铁路初步计划从[[苏加诺-哈达国际机场|苏加诺-哈达国际机场]]到芒加莱（Manggarai），但是为了满足哈利姆·珀达纳库苏马国际机场交通的需要，铁路线将从芒加莱延伸到哈利姆机场。线路总长为33公里，路线为：哈利姆－查旺－丹拿望（Tanah Abang）－苏迪曼（Sudirman）－珊瑚新村（Pluit）－苏哈机场2号和3号航站楼，线路由地面、地下和高架部分组成，已获得印尼交通部的同意（2013年第1264号部级条例，印尼语：Peraturan Menteri Nomor 1264 Tahun 2013）。快速铁路的单程运行时间为30分钟，该段路程驾车时间长达1至3小时。<ref name="express train">{{cite web |url=http://www.metrotvnews.com/metronews/read/2014/01/13/2/207414/April-Tender-Kereta-Halim-Bandara-Soekarno-Hatta- |title=April, Tender Kereta Halim-Bandara Soekarno-Hatta  |accessdate=2014-01-13}}</ref>
