'''哈菲兹·阿萨德'''（{{lang-ar|حافظ الأسد}}，{{bd|1930年|10月6日|2000年|6月10日|الأسد}}）[[敘利亞|敘利亞]][[政治家|政治家]]，前[[敘利亞总统|敘利亞总统]]、[[獨裁者|獨裁者]]、空軍司令及國防部長，阿塞德1970年發動軍事政變掌權，統治敘利亞長達三十年。相對其子，人稱老阿薩德。

== 生平 ==
阿萨德1930年生于叙利亚西北部乡下[[拉塔基亚省|拉塔基亚省]]的Qardaha镇一个[[阿拉维派|阿拉维派]]的家庭中，出身贫穷。在学校期间成绩突出，是家族中首个考入中学的成员。由于认同[[阿拉伯复兴社会党|阿拉伯复兴社会党]]的意识形态，阿萨德16岁入党。中学毕业后因家庭贫困且军校免学费并可步入仕途，1951年，阿萨德进入军校学习并被分配到新成立的叙利亚空军，他还曾在[[苏联|苏联]]接受飞行员训练。1963年，以軍方為後盾的[[阿拉伯社會主義復興黨|阿拉伯社會主義復興黨]]發動軍事政變，取得政權。

== 總統 ==
1970年，於該黨担任要職的阿塞德則以不流血政變方式奪取政權，出任總理兼國防部長，翌年正式通過全民公投當選總統，之後每七年以民選方式連任四屆，1978年、1985年、1992年和1999年成功連任，他成為20世紀後期[[中東|中東]]地區在位最久之國家元首。
[[File:Mausoleum_of_Hafez_al-Assad_1.jpg|thumb]]
2000年，掌權三十年的哈菲兹·阿萨德逝世，其次子[[巴沙尔·阿萨德|巴沙尔·阿萨德]]继任叙利亚总统。

== 評價 ==
阿塞德執政評價兩極，正面的評價方面他穩定政局，与埃及一样實行[[世俗主义|世俗主义]]的[[独裁統治|独裁統治]]，打擊極端[[伊斯蘭原教旨主義|伊斯蘭原教旨主義]]，但作为少数什葉派中的阿拉维派，與什葉派的[[伊朗革命|伊朗革命]]後政權關係密切，加上与[[伊拉克|伊拉克]]爭奪阿拉伯世界的主導权，在[[兩伊戰爭|兩伊戰爭]]中，敘利亞支持[[伊朗|伊朗]]，長期插手[[黎巴嫩內戰|黎巴嫩內戰]]並實際控制黎巴嫩，與[[美國|美國]]及[[以色列|以色列]]關係緊張，但維持不打不和的狀態，1991年參與[[波斯灣戰爭|波斯灣戰爭]]，其後同意與以色列進行和平談判，他堅持要求以色列歸還[[戈蘭高地|戈蘭高地]]，因此和談沒有成果，但其政权被以色列視為維持中東和平的支柱。

反對的評價方面，阿塞德被反對派批評其實行[[獨裁|獨裁]]統治，打壓政治[[異議分子|異議分子]]，1982年的[[哈馬大屠殺|哈馬大屠殺]]，導致近四萬平民喪命，籍只佔國內少數的[[阿拉維派|阿拉維派]]軍方維持政權，長期實行[[緊急狀態|緊急狀態]]法，與國內人口佔絕大多數的[[遜尼派|遜尼派]]關係緊張。
