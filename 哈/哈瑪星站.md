'''哈瑪星站'''位於[[中華民國|中華民國]][[高雄市|高雄市]][[鼓山區|鼓山區]]，為[[高雄捷運環狀輕軌|高雄捷運環狀輕軌]]以及[[高雄捷運旗津線|高雄捷運旗津線]]的車站，鄰近[[哈瑪星|哈瑪星]]聚落並因此得名。可站外轉乘[[高雄捷運橘線|高雄捷運橘線]][[西子灣站|西子灣站]]。

「哈瑪星」此名稱的由來，是因為當地有兩條[[高雄臨港線|濱海鐵路]]通往商港、漁港和漁市場，日語稱為「濱線」（日語：はません，Hamasen），當地居民以臺語稱之為「哈瑪星」，故本站英譯名稱採原始日語發音譯為'''Hamasen Station'''。本站亦為台灣第一個採用[[日文|日文]][[音譯|音譯]]的捷運站名。

== 車站概要 ==
車站代碼為'''C14'''，站址位於已廢止的[[台鐵|台鐵]][[高雄港車站|高雄港車站]]調車場內（現為[[打狗鐵道故事館|打狗鐵道故事館]]）；型式為兩座[[側式月台|側式月台]]。

此站定位為環狀輕軌「特色站」，可站外轉乘[[高雄捷運橘線|高雄捷運橘線]][[西子灣站|西子灣站]]

== 歷史 ==
===高雄環狀輕軌第一段C1-C14車站全民參與命名活動===
* 命名活動第一階段（2014年10月18日至11月20日）建議站名：打狗驛站、哈瑪星站、高雄港站
* 命名活動第二階段（2015年2月16日至3月6日）票選選項：哈瑪星站、打狗驛站、高雄驛站
* 命名結果（2015年3月29日公告）：哈瑪星站<ref>[http://www.thenewslens.com/post/144007/ 高雄環狀輕軌C1-C14站有名字了！最快8月通車]</ref>
===通車===
* 2017年9月26日下午4點：環狀輕軌第一階段[[駁二大義站|C12站]]-[[駁二蓬萊站|C13站]]-[[哈瑪星站|C14站]]通車啟用<ref>[https://udn.com/news/story/7327/2724644  最後一哩 高雄輕軌一階全線通車]， 聯合新聞網，2017-09-27 04:15。</ref><ref>[http://m.ltn.com.tw/news/local/paper/1138720 〈南部〉輕軌一階14站 全線通車了！]</ref>。
== 預定時程 ==
* 2019年中：環狀輕軌[[哈瑪星站|C14站]]~[[鼓山區公所站|C17站]]預計為二階輕軌優先通車段<ref>[http://cissearch.kcc.gov.tw/Upload/Attachment/BusinessReport/1018/5569e979-41aa-48cb-8a17-0ec982d5c057.ppt 高雄市政府捷運局 業務報告]</ref>。
