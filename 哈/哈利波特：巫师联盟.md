《'''哈利·波特：巫师联盟'''》是一个[[网络游戏免费模式|免费]]的、{{le|基于地理位置的游戏|Location-based game|基于地理位置}}的[[擴增實境|实境增强游戏]]。游戏背景基于[[哈利·波特|哈利·波特系列]]中的{{le|魔法世界 (哈利·波特)|Wizarding World|魔法世界}}。游戏由[[华纳兄弟互动娱乐|WB Games]] San Francisco和[[Niantic|Niantic]]开发，Niantic发行<ref>{{Cite web|url=https://nianticlabs.com/press/2017/wizardsunite/|title=NIANTIC, INC. AND WARNER BROS. INTERACTIVE ENTERTAINMENT ANNOUNCE PARTNERSHIP FOR HARRY POTTER: WIZARDS UNITE, AN AUGMENTED REALITY MOBILE GAME INSPIRED BY J.K. ROWLING’S WIZARDING WORLD |accessdate=2018-06-29 |archiveurl=https://web.archive.org/web/20171110005040/https://nianticlabs.com/press/2017/wizardsunite/# |archivedate=2017-11-10}}</ref>。计划支持[[Android|Android]]与[[iOS|iOS]]设备，[[擴增實境|增强现实]]的部分与Niantic之前的游戏《[[精灵宝可梦GO|精灵宝可梦GO]]》、[[Ingress|Ingress]]类似。

== 概述 ==
《巫师联盟》项目于2017年11月8日宣布，是一款由WB Games San Francisco，{{le|门钥匙游戏|Portkey Games}}与[[Niantic|Niantic]]开发的AR游戏<ref>{{Cite web|url=https://harrypotterwizardsunite.com/faq/ |title=FAQ - Harry Potter: Wizards Unite |accessdate=2017-11-16 |archiveurl=https://web.archive.org/web/20181116174541/https://harrypotterwizardsunite.com/faq/# |archivedate=2018-11-16}}</ref> 。玩家将能够访问的现实世界的位置，同时施放咒语，寻找神秘道具，而且可以遇到[[哈利波特|哈利波特]]系列中的经典人物与魔法生物<ref>{{Cite web|url=https://www.nianticlabs.com/press/2017/wizardsunite/|title=NIANTIC, INC. AND WARNER BROS. INTERACTIVE ENTERTAINMENT ANNOUNCE PARTNERSHIP FOR HARRY POTTER: WIZARDS UNITE, AN AUGMENTED REALITY MOBILE GAME INSPIRED BY J.K. ROWLING’S WIZARDING WORLD |accessdate=2018-03-17 |archiveurl=https://web.archive.org/web/20171110005040/https://nianticlabs.com/press/2017/wizardsunite/ |archivedate=2017-11-10}}</ref>。游戏灵感源自Niantic的两款AR游戏前作《[[精灵宝可梦GO|精灵宝可梦GO]]》与[[Ingress|Ingress]]。<ref>{{Cite web|url=https://www.dailystar.co.uk/tech/gaming/681611/Harry-Potter-Go-Wizards-Unite-release-date-spells-characters-Pokemon-Go |title=Harry Potter Wizards Unite: Movie fans new Pokemon GO style mobile game coming 2018 |accessdate=2018-02-21 |work=[[每日星報|每日星報]] |archiveurl=https://web.archive.org/web/20180220231504/https://www.dailystar.co.uk/tech/gaming/681611/Harry-Potter-Go-Wizards-Unite-release-date-spells-characters-Pokemon-Go |archivedate=2018-02-20}}</ref>
