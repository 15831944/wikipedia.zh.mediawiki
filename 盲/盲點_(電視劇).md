《'''盲點'''/'''盲點追兇'''》（{{lang-en|'''''Blindspot'''''}}）是一部於2015年9月21日在[[國家廣播公司|NBC]]開播、由{{link-en|馬丁·格羅|Martin Gero}}開創的美國[[犯罪片|犯罪]]影集。2015年5月1日，NBC預訂了本劇<ref>{{cite news|url=http://www.latimes.com/entertainment/tv/showtracker/la-et-st-nbc-picks-up-dramas-heartbreaker-chicago-med-blindspot-20150501-story.html|title=NBC picks up dramas 'Heartbreaker,' 'Chicago Med,' 'Blindspot'|last=Collins|first=Scott|date=May 1, 2015|work=[[洛杉磯時報|洛杉磯時報]] |accessdate=May 2, 2015}}</ref>。2015年10月9日，NBC預訂了全季22集集數<ref>{{cite web|last1=Andreeva|first1=Nellie|title=‘Blindspot’ Gets Full-Season Order By NBC|url=http://deadline.com/2015/10/blindspot-full-season-back-9-order-nbc-1201572392/|website=Deadline Hollywood|accessdate=October 10, 2015|date=October 9, 2015}}</ref>。

== 故事 ==
警員在人潮洶湧的紐約時代廣場發現一個可疑的大型手提袋，上面寫著：『通知FBI』，未爆彈處理小組來拆卸疑似炸彈時，發現裡面裝的是一個全身赤裸佈滿刺青的女人，女子背後刺著FBI特別探員科特·偉勒的名字，為查清無名女子身份，上級責令科特·偉勒領導新成立的緊急事件反應小組（FBI Critical Incident Response Group）……

== 海外播出 ==
2015年9月22日，本劇正式於晚間9點50分在[[臺灣|臺灣]]與[[香港|香港]]等東南亞地區的[[華納電視頻道|Warner TV]]首播，僅晚美國首播約12小時左右。

[[香港|香港]] 免費電視 [[ViuTV|ViuTV]] 旗下之英語頻道 96 [[ViuTVsix|ViuTVsix]] 於 2018年1月10日 逢星期三 晚間 9時30分 (翌日下午1時30分重播) 開播第1季，2018年6月20日 緊接 開播第2季。
