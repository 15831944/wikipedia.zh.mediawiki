'''覆盆子'''（[[學名|學名]]：'''''Rubus idaeus'''''）是[[薔薇科|薔薇科]][[懸鉤子屬|懸鉤子屬]]的木本植物，其果实味道酸甜，植株的枝干上长有倒钩刺。

==名稱==
覆盆子有很多別名，例如：
'''紅桑子'''、 '''覆盆'''<ref>《别录》</ref>、'''覆盆莓'''、'''托盘'''（东北）、'''饽饽头'''（东北）、'''公饭'''（广东）、'''野莓'''、'''野草莓'''、'''乌藨子'''<ref>《纲目》</ref>、'''小托盘'''<ref>《中药材手册》</ref>、'''竻藨子'''<ref>《江西中药》</ref>、'''花蜜托盘'''、'''蛇莓'''、'''蛇头莓'''、'''桑莓'''、'''紅莓'''、'''公公帽'''（温州）。
山东“猫眼子”、“拖鞋”
*一般[[樹莓|樹莓]]（Raspberry）指的即是覆盆子，但廣義上的範圍並非如此。
*明代《本草綱目》等記載，覆盆子與懸鉤子是兩種植物。本條目所指植物為懸鉤子，又稱沿鉤子、藨子。而古稱覆盆子的植物相當於Bilberry。

== 型態 ==

覆盆子的果實是一種[[聚合果|聚合果]]，有红色、金色和黑色。

== 分佈 ==

覆盆子在[[日本|日本]]、[[南韓|南韓]]、歐美等地是常見水果；在[[中国|中国]]大量野生分布，但却很少人食用，仅在东北地区有少量栽培，市场上比较少见。

== 營養價值 ==
{{medical}}
覆盆子油屬於[[不飽和脂肪酸|不飽和脂肪酸]]，可促進[[前列腺|前列腺]]分泌[[荷尔蒙|荷尔蒙]]<ref>{{cite book|author=王本祥|title=《現代中藥藥理學》|publisher=天津科學技術出版社|year=1999|page=1248-1250}}</ref>。由於覆盆子也是低升糖指數（GI值）的水果，可以延緩體內[[血糖|血糖]]上升值，加上富含高纖維，讓人不容易感到飢餓，更能幫助維持體重。

== 食用方法 ==

紅色覆盆子葉具有香草的養生功能，也可搭配香草使用。一般來說覆盆子常拿來做為甜點或果醬食用，[[朝鮮半島|朝鮮半島]]則會將覆盆子發酵後作為酒類飲用。

== 轶事 ==
[[金酸莓獎|金酸莓獎]]（{{Lang-en|Golden Raspberry Awards}}）原意为“金覆盆子奖”。
