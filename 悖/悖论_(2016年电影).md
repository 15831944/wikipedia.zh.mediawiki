== 剧情 ==
{{See also|时间旅行|时间悖论|封闭类时曲线|無窮迴圈|命定悖论|诺维科夫自洽性原则}}
这一日白天，兰迪{{noteTag|1=来自未来的兰迪}}从880实验室出来后匆忙跑进保安室给880项目的出资人兰道先生打电话，警告兰道不要前往880实验室并必须取消时间穿越实验，否则参与880项目的所有人都会死。此刻，兰道正在看着2009年少年时的自己和母亲而没有接到兰迪的电话。于是，兰迪便给家里打电话，接电话的母亲不相信是儿子打来的电话，因为儿子{{noteTag|1=现在的兰迪}}正在家中洗澡，但是兰迪还没有打完电话便被神秘人物杀害。

这天晚上，吉姆、李维斯、盖尔、比尔（威廉·韦什曼）、兰迪等人在一座大楼附近会合，准备进行时间穿梭的实验。这些人在更衣间更换衣物时，闻到了一股刺鼻的臭味{{noteTag|1=来自未来的兰迪被神秘人物杀害后，尸体被藏匿于此处}}，但未引起人们的注意。880项目的出资人兰道先生决定派吉姆穿越时间到另一时刻的880实验室，吉姆到达了一个小时后的880实验室{{noteTag|1=此时为午夜12点}}惊恐的发现，除了盖尔其他人皆被神秘人物杀害{{noteTag|1=盖尔则遭神秘人物绑架}}，880实验室也被开启了自爆，吉姆带着午夜12点的摄像机{{noteTag|1=李维斯在死亡前委托吉姆把摄像机带回一个小时前}}急忙穿越回了现在。吉姆慌张的告诉众人一个小时后实验室将发生的一系列事件，力劝他们立即离开实验室。但在此之前，时间机器的启动导致全市大停电，并且实验室的通讯线路也被神秘人物切断，电梯和通讯设备均停止运行，众人无法离开实验室。他们已经困在了命定悖论的[[死循环|怪圈]]之中，无论他们做了何种努力，终究也躲不开死亡的[[命运|命运]]……<ref name="Culture Crypt">{{cite web |title=PARADOX (2016) |url=https://culturecrypt.com/movie-reviews/paradox-2016 |website=Culture Crypt |accessdate=2018-08-19 |language=en |archive-url=https://web.archive.org/web/20180201100801/http://culturecrypt.com/movie-reviews/paradox-2016 |archive-date=2018-02-01 |dead-url=no }}</ref><ref>{{cite web |title=Paradox (2016) Movie Script（电影《悖论》的台词） |url=https://www.springfieldspringfield.co.uk/movie_script.php?movie=paradox |website=Springfield! == 評價 ==
《悖論》整體獲得的評價以負面居多。《[[洛杉磯時報|洛杉磯時報]]》的馬丁·蔡（{{lang|en|Martin Tsai}}）給了該片負面的評價，他稱讚了佐伊·貝爾的表現，但也批評其劇本及導演方面都很糟糕，並在評論中寫道：「其時間之謎足以令人從笨拙的方向、明顯廉價的布景設計和俗氣的特效上分散注意力。」<ref name="latimes" />{{link-en|JoBlo.com|JoBlo.com|JoBlo}}網站的傑克·迪（{{lang|en|Jake Dee}}）從滿分10分中給了該片5分的中庸評價，並在評論中寫道：「這部電影擁有一個非常令人欽佩的論點，不幸的是就像大多數時間旅行故事在微不足道的預算下創造的那樣，缺乏令人信服的執行力，最終只不過是個荒謬、腦袋錯亂的蠢材。」<ref name="JoBlo_Review">{{cite web|url=http://www.joblo.com/horror-movies/news/paradox-movie-review-849|title=Paradox (Movie Review)|last=Dee|first=Jake|work=JoBlo.com|date=2016-04-15|accessdate=2018-08-21|archive-url=https://web.archive.org/web/20171218204913/http://www.joblo.com/horror-movies/news/paradox-movie-review-849|archive-date=2017-12-18|dead-url=no}}</ref>。

{{lang|en|The Other View}}網站的{{lang|en|Valkor}}對於《悖論》表示喜愛，他認為雖然該片不是最好的時間旅行故事，但對打發時間來說還算過得去<ref name="TheOther_View">{{cite web|url=http://www.the-other-view.com/paradox.html|title=Paradox (2016) Movie Review|author=Valkor|work=The Other View|date=2016-05-04|accessdate=2018-08-21|archive-url=https://web.archive.org/web/20171023044250/http://www.the-other-view.com/paradox.html|archive-date=2017-10-23|dead-url=no}}</ref>。
