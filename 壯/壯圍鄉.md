'''壯圍鄉'''位於[[蘭陽平原|蘭陽平原]]東部，東臨[[太平洋|太平洋]]，又居[[蘭陽溪|蘭陽溪]]下游及出海處，地勢平坦，且土壤屬沙質土壤，排水良好加上日照充足。

蘭陽溪將宜蘭平原劃分為二，蘭陽溪以北俗稱「溪北」，以南為「溪南」，壯圍鄉便是位於溪北，臨近宜蘭市區。

早期居民產業以[[農漁業|農、漁業]]為主，出海捕魚、[[牽罟|牽罟]]是舊時沿海地區賴以為生的活動(於民國105年，[[壯圍廍後社區|壯圍廍後社區]]規劃讓民眾體驗牽罟)。現當地產業以[[農業|農業]]為主，特產瓜果類及[[青蔥|青蔥]]，而[[稻米|稻米]]的產量更名列[[台灣|台灣]]三大米倉之一，是一個充滿鄉村氣息的濱海地區，同時位於[[宜蘭縣|蘭陽平原]]的海岸中心，是欣賞「龜山朝日」的最佳地點。

== 歷史 ==
'''壯圍鄉'''舊名「民壯圍」、「民壯圍堡」，[[嘉慶|嘉慶]]七年（1802年）時，由[[吳沙|吳沙]]率眾進入開墾蘭陽平原，後為酬謝有功青壯鄉民，故稱之。1920年，[[臺灣總督府|台灣總督府]]在此設立「-{[[壯圍庄|壯圍庄]]}-」，劃歸[[臺北州|台北州]][[宜蘭郡|宜蘭郡]]管轄。戰後，改街-{庄}-為鄉鎮改稱「[[新北市|臺北縣]]壯圍鄉」，1950年宜蘭縣設立，改稱「宜蘭縣壯圍鄉」，沿用至今。

「壯圍」原為平埔嘉慶七年，吳化（吳沙之姪）部屬九旗首之陳孟理（或說陳明燈）見此地土壤肥沃，率眾劃界管墾，因無天險可守，只有青壯年圍屯聚居，而稱為「民壯圍堡」（所屬民莊十七所），此即「壯圍」名稱由來，亦為壯圍鄉自治之始。嘉慶十五年，宜蘭縣收歸版圖，設置「噶瑪蘭廳」，光緒六年，改為「宜蘭縣」。甲午戰後，日本領台，改為「宜蘭支廳」，民國九年，日人施行街庄役場制度，改為「壯圍庄役場」。[[臺灣戰後時期|二戰後]]，初隸台北縣，稱為「台北縣壯圍鄉」，民國三十九年宜蘭縣設立，改稱「宜蘭縣壯圍鄉」迄今。

== 宗教祭祀 ==
早期壯圍鄉八大庄分別是永鎮村、忠孝村、過嶺村、新社村、後埤村、廍後村、美城村、古亭村，並以壯圍永鎮廟(開漳聖王)為公廟，每年農曆二月十五日壯圍永鎮廟邀請八大庄頭大廟主神前往一同參與過火。於民國67年行政區域重整，後埤村1-4鄰與紅葉村合併「復興村」，後埤5-10鄰併入過嶺村。

早期壯圍八大庄廟宇為：[[壯圍永鎮廟|壯圍永鎮廟]](主神開漳聖王)、[[過嶺保安宮|過嶺保安宮]](主神中壇元帥)、[[壯六鎮安廟|壯六鎮安廟]](主神開漳聖王)、[[社頭鎮安廟|社頭鎮安廟]](主神開漳聖王)、[[後埤振安廟|後埤振安廟]](主神開漳聖王)、[[美城福德廟|美城福德廟]](主神福德正神)、[[古亭永惠廟|古亭永惠廟]](主神開漳聖王)、[[番社同安廟|番社同安廟]](主神玄天上帝)。
