《'''鸟笼山剿匪记'''》，原名《'''[[大规模杀伤性武器|大杀器]]传奇'''》，是[[中国|中国]][[中国大陆|大陆]][[自由职业者|自由职业者]][[胡戈|胡戈]]继《[[一个馒头引发的血案|一个馒头引发的血案]]》、《[[春运帝国|春运帝国]]》之后的第三部[[网络电影|网络电影]]。与前两部不同，此片并非由其它电影素材剪辑而成，而是由真人扮演。《鸟笼山剿匪记》长度为50分钟，用[[DV|DV]]拍摄。据胡戈本人讲，此片系他与他人投资15万人民币拍摄<ref>{{Cite web |url=http://ent.sina.com.cn/m/c/2006-06-07/15501114116.html |title=胡戈新片《鸟笼山剿匪记》首播 网友挤爆宽带|accessdate=2018-09-17 |author=贾中山 |date=2006-06-07 |work= |publisher=新浪网，来源：北京晚报 |language=Zh-hans }}</ref>。该剧名称[[戏仿|戏仿]]了中国1980年代末的知名电视连续剧《[[乌龙山剿匪记|乌龙山剿匪记]]》。

专业视频网站——[[六间房|六间房]]是此片的唯一赞助商。胡戈宣布2006年6月6日10点在六间房独家播出此片。当日，由于访问量过大，致使网站数度出现访问故障<ref>{{Cite web |url=http://news.163.com/06/0607/13/2J13374H0001124J.html |title=弄巧成拙 胡戈新片第一天“累垮”赞助网站|accessdate=2018-09-17 |author= |date=2006-06-07 |work=责任编辑：王晓易_NE0011 |publisher=网易，来源：新闻晨报(上海) |language=Zh-hans }}</ref>。当时，[[网剧|网剧]]在中国大陆尚不成熟。胡戈的行为被认是在“走钢线”，挑战网络短片的监管空白<ref>{{Cite web |url=http://ent.sina.com.cn/m/c/2006-06-10/07311117621.html |title=胡戈新短片引来众多非议 广电总局欲出手监管|accessdate=2018-09-17 |author=唐爱明 |date=2006-06-10 |work= |publisher=新浪网，来源：华商网-华商报 |language=Zh-hans }}</ref>。

同年10月，胡戈在六间房推出另一部相关作品——《[[鸟笼山电视购物|鸟笼山电视购物]]》。

== 剧情介绍 ==

傻大木藏身在鸟笼山，秘密制造大杀器（即“大规模杀伤性武器”简称），企图毁灭地球。世界警察总部“乔不士”司令派特工“有名”渗入侦查“傻大木”盘踞的“鸟笼山”。有名靠演技突出成功潜入，却在深夜秘密侦查时被起夜的喽啰捕获，傻大木通过“hoogle.com”搜到了有名的blog，进而得知其前来的目的。乔司令借口傻大木制造和储藏“大杀器”，派“安鸟部队”攻打鸟笼山。傻大木不战自散，无奈借“老拉”帮忙利用神秘方式传送走。士兵们冲进仓库后发现大杀器其实是馒头。

潜逃后的傻大木开始在街头散发“大杀器”，却不料被一处居民楼的保安擒获。

字幕后，开发商在鸟笼山出售高价房产，声称其交通便利、环境优雅。紧接着一群抗议强制性拆迁的人来示威，结果被乔不士手下打手使用建筑用标准红砖痛打。

==涉及事件==
*片中大量使用谐音和名称变体，如傻大木被认为与[[萨达姆|萨达姆]]谐音，乔不士与[[乔治·沃克·布什|乔治·布什]]谐音。类似的还有靖国神厕（[[靖国神社|靖国神社]]），蚂蝗卫视（[[凤凰卫视|凤凰卫视]]），小全（[[小泉纯一郎|小泉纯一郎]]），旧闻和废纸摘要（[[中央人民广播电台|中央人民广播电台]]每天凌晨的《新闻和报纸摘要》节目），“满城尽戴黄金乙”（《[[满城尽戴黄金甲|满城尽戴黄金甲]]》）等。
*“大杀器”既是[[馒头|馒头]]，这是沿用了胡戈首作《[[一个馒头引发的血案|一个馒头引发的血案]]》的情节。
*傻大木在洞口朝天开枪，把侦查飞机（CG效果是以F-35战斗机做原型）打了下来，讽刺[[朝鲜|朝鲜]]小学语文课本中“《伟大领袖[[金日成|金日成]]将军举起了枪》”打下飞机的情节。[http://www.ccdy.cn/2010-07/06/content_484900.htm]{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}
*乔不士在面对记者的质疑时说：“人可以无耻到我这种地步”，是反讽[[陈凯歌|陈凯歌]]对胡戈说的“人不能无耻到这种地步”。
*片中喽啰们所使用B41武器其实是[[AK-47突击步枪|AK-47]][[突击步枪|突击步枪]]，在著名[[第一人称射击游戏|第一人称射击游戏]]《[[反恐精英|反恐精英]]》（1.5版本）中，恐怖分子一方购买武器时以“B－4－1”的顺序按键所购得的武器就是AK-47。
*傻大木因发帖被骂“[[网络语言#形容词类|火星人]]”而萌发了要毁灭地球的想法。而片中乔不士在有名进来前正在给傻大木的帖子回帖“楼主是火星人，鉴定完毕”。
*傻大木虐待囚犯的手段是给他们播放“第十五代导演大片集锦”，而中国当前的知名导演（如陈凯歌、[[张艺谋|张艺谋]]、冯小刚等）被称为第五代导演。
*乔不士在准备捉拿傻大木时，小全想上去动手，乔不士说：“不行，按规定你是不能参战的……”，小全回答说：“……我是自卫啊……”。影射日本派遣[[自卫队|自卫队]]上伊拉克战场。
*乔不士的军队进洞，准备活捉傻大木，傻大木却通过电话消失，是模仿《[[黑客帝国|黑客帝国]]》。
*乔不士的军队找到大杀器准备开箱检查时，要求小全上前探查，小全不愿意，乔不士说：“你不是要入常么……”。（指日本希望加入[[联合国常任理事国|联合国常任理事国]]一事）
*乔不士命令打手攻击抗议拆迁人群时，从打手乘坐的[[大众波罗|大众Polo车]]内走出了远超过Polo车额定乘员数（5人）的众多打手，类似冯小刚导演的影片《[[手机_(电影)|手机]]》中，夜间归校从轿车内钻出的多名影校学生。
*乔不士发言时所用的讲台前侧面，写着“Yamen House”(Yamen即中文衙门的拼音)和美国总统府代表[[白宫|白宫]](White House)发言所用蓝底白字的讲台类似。
*“有名”在傻大木面前表演的歌舞是以改编自迈克尔·杰克逊的名曲《Smooth criminal》为原曲歌词
