'''娥蘇拉·勒瑰恩'''<ref>另譯：-{zh-hans:娥苏拉·勒瑰恩;zh-hant:厄休拉·勒吉恩}-，爾蘇拉·勒昆。</ref>（{{lang-en|Ursula Kroeber Le Guin}}，{{bd|1929年|10月21日|2018年|1月22日||Le Guin}}），[[美國|美國]][[科幻小說|科幻]]、[[奇幻小說|奇幻]]與[[女性主義|女性主義]]與青少年兒童文學作家。著有小說20餘部，及詩集、散文集、遊記、文學評論與多本童書。並與人合譯[[老子|老子]][[道德經|道德經]]。所獲文學獎與榮譽不計其數。她深受老子與[[人類學|人類學]]影響，作品常蘊含[[道家|道家思想]]，寫作手法流露民族誌風格。

其最廣為人知之作包括奇幻小說《[[地海|地海]]》系列與科幻小說《[[黑暗的左手|黑暗的左手]]》與《[[一無所有_(小說)|一無所有]]》。《地海》系列常與[[J·R·R·托爾金|J·R·R·托爾金]]之《[[魔戒|魔戒]]三部曲》或[[C·S·路易斯|C·S·路易斯]]之《[[納尼亞年代記|納尼亞年代記]]》相提並論。

== 生平 ==
勒瑰恩出身學術家庭，父親是美國[[人類學家|人類學家]][[阿弗烈·克魯伯|阿弗烈·克魯伯]]，母親是心理學家及作家，三位兄長都成為學者，均非常關注美國原住民文化。家中時常高朋滿座，除了知名學者、研究生之外，還有許多印第安人，套句勒瑰恩母親說的話，他們家就是「一整個世界」。在這樣富有學術氣氛的環境成長，勒瑰恩年紀輕輕即開始寫作，在11歲時便已向''Magazine''，''Astounding Science Fiction''投稿，雖然不幸被退稿。1951年於[[哈佛大學|哈佛大學]]的[[雷地克里夫學院|雷地克里夫學院]]取得學士學位，1952年於[[哥倫比亞大學|哥倫比亞大學]]取得碩士學位，並在大學任教。

== 写作生涯 ==
她早期的作品並非科幻或奇幻小說，而是根基於真實世界的[[虛構國度|虛構國度]]。大部分這些小說在當時都沒有被出版，但後來以合集方式收錄於''[[Orsinian_Tales|Orsinian Tales]]''與''[[Malafrena|Malafrena]]''中。因為不受出版商青睞，她轉而嘗試她早期較感興趣的科幻小說，在1960年代漸漸寫出名聲。1969年《黑暗的左手》出版，獲頒[[雨果獎|雨果獎]]與[[星雲獎|星雲獎]]，同時奠定她在文壇的地位。之后《[[一無所有_(小說)|一無所有]]》使她成为第一个两次因同一部作品同时獲頒[[雨果獎|雨果獎]]與[[星雲獎|星雲獎]]的作家。

== 影响 ==
影响勒瑰恩创作的包括奇幻作家[[J·R·R·托爾金|J·R·R·托爾金]]，[[科幻小说|科幻小说]]作家[[菲利普·狄克|菲利普·狄克]]，西方文學大家[[列夫·托爾斯泰|列夫·托爾斯泰]]、[[维吉尔|维吉尔]]和[[勃朗特三姊妹|勃朗特三姊妹]]，女權主義作家[[弗吉尼亚·吴尔夫|弗吉尼亚·吴尔夫]]，[[兒童文學|兒童文學]]如《[[爱丽丝梦游仙境|爱丽丝梦游仙境]]》、《[[柳林風聲|柳林風聲]]》、《[[叢林奇譚|叢林奇譚]]》，[[北欧神话|北欧神话]]，以及[[老子|老子]][[道德經|道德經]]。

== 作品風格 ==

勒瑰恩的小說最有特色的是她對[[社會學|社會學]]與[[人類學|人類學]]的關懷。在勒瑰恩的科幻作品中，她經常架空出一個相當真實但又與地球不盡相似的世界，與相當人性化的族群，並以這些完全迥異於地球的文化流暢地帶出她的主題。

她作品為分析對象的文學評論眾多，甚至出版專書探討。舉凡《[[地海傳說|地海傳說]]》的成長主題與道家思想、《[[黑暗的左手|黑暗的左手]]》的敘事方式與性別議題、《[[一無所有_(小說)|一無所有]]》的[[烏托邦|烏托邦]]與[[反烏托邦|反烏托邦]]等等，皆對主流文學界產生重大影響。

勒瑰恩除了創作，更投入老子《[[道德經|道德經]]》的英譯注解工作，耗時四十年之久，此版本推出之後獲得相當高的評價。她並將老子思想融入創作，在一向以西方文明為骨幹的奇幻、科幻小說中，發揮東方哲學的無為、相生與均衡概念。此外，《[[地海傳說|地海傳說]]》中的島嶼世界（相對於歐美的大陸世界）與骨架纖細、黑髮深膚的民族（相對於西方人種的外貌），以及隱喻西方文明的侵略與破壞性格，這種「去西方中心」的敘述觀點與一般西洋奇幻文學形成強烈對比。
