'''娥眉洲'''形狀呈U形，屬[[印洲塘|印洲塘]]範圍<ref>{{cite web|url=https://www.google.com.hk/maps/@22.5307661,114.3159844,15.31z|title=Google Maps|website=Google Maps}}</ref>，是[[印洲塘|印洲塘]]三個島嶼中面積最小而最荒蕪的，島上沒有居民及任何設施。<ref name="night-eagle.com">{{cite web|url=http://www.night-eagle.com/trip/yantong/ngomei.html|title=印塘三島－娥眉洲|website=www.night-eagle.com}}</ref>政府於1996年將印洲塘劃為海岸公園，而娥眉洲亦屬海岸公園的一部分。<ref name="appledaily.com">{{cite web|url=https://hk.news.appledaily.com/breaking/realtime/article/20190103/59102782|title=娥眉洲現女浮屍 身份不明|website=Apple Daily 蘋果日報}}</ref>
== 環境 ==
位於[[往灣洲|往灣洲]]東北方；[[白沙洲|白沙洲]]西南方。中間與[[往灣洲|往灣洲]]相隔[[直門|直門]]海域。娥眉洲島上遍佈叢林，山中更寸步難行。<ref name="night-eagle.com"/>西部有島上最高的山峰——[[紅花嶺|紅花嶺]]，但要登上紅花嶺則需自行開路，披荊斬棘，才能到達山頂。娥眉洲形似U形，北部構成[[峨眉灣|峨眉灣]]，珊瑚群落存育在U形內灣平靜水域，適合浮潛。灣中亦有一無名小島，遊客游泳時可到該小島休息。
== 交通 ==
由於娥眉洲沒有碼頭，所以一般時間沒有渡輪開往此地。若要登遊此島，應跟隨旅行隊伍出發同行。<ref name="night-eagle.com"/>
== 旅遊 ==
島上沒有淡水供應，出發前應先自行準備食水，海邊有各種海洋生物如：[[水母|水母]]、[[海螺|跳螺]]、[[寄居蟹|寄居蟹]]、[[梭羅魚|梭羅魚]]等。<ref>{{cite web|url=http://lihung1117.blogspot.com/2014/09/14-816.html|title=Mimi Blog: 14-816娥眉洲两日一夜(一)|first=Mimi|last=Li|date=2014-09-19|publisher=}}</ref>受《海岸公園及海岸保護區規例》保護，吸引不少潛水愛好者慕名而至，欣賞海底世界。
== 意外 == 
=== 紅鞋女擱淺事件 ===
2015年3月22日，有釣魚人士發現有穿紅鞋女屍擱淺在一石灘上，屍首已嚴重腐爛，相信已死去多時；由於現場毗鄰內地水域，不排除女死者為內地人，死後隨水流漂至石灘。水警聯同[[消防|消防]]趕往調查，初步證實女死者年約20歲至30歲，1.5米高，瘦身材，短黑髮，被發現時身穿粉紅色外套、黑色長褲、及穿一雙紅色[[皮鞋|皮鞋]]，但卻無任何身份證明文件，屍首已嚴重腐爛並傳出惡臭。<ref>{{cite web|url=http://paper.wenweipo.com/2015/03/23/HK1503230020.htm|title=紅鞋女腐屍擱淺娥眉洲石灘|website=paper.wenweipo.com}}</ref>
=== 浮潛男遇溺事件 ===
2016年8月28日，一名男子乘船觀光，其間於娥眉洲浮潛時遇溺，由友人合力救起，再由[[飛行服務隊|飛行服務隊]]直升機送院，惟搶救後不治。<ref>{{cite web|url=https://news.mingpao.com/pns/%e6%b8%af%e8%81%9e/article/20160829/s00002/1472407525505/%e5%85%a9%e5%91%a8%e4%b8%89%e5%ae%97-%e6%b5%ae%e6%bd%9b%e6%bc%a2%e5%a8%a5%e7%9c%89%e6%b4%b2%e6%ba%ba%e6%96%83|title=兩周三宗 浮潛漢娥眉洲溺斃 - 20160829 - 港聞|website=明報新聞網 - 每日明報 daily news}}</ref>
=== 海面女浮屍事件 ===
