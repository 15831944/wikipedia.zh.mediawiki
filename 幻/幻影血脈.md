《'''JoJo的奇妙冒險 第一部 幻影血脈'''》（{{lang|ja|ジョジョの奇妙な冒険 Part1 ファントムブラッド}}）為日本漫畫家[[荒木飛呂彥|荒木飛呂彥]]所著的[[日本|日本]][[漫畫|漫畫]]《[[JoJo的奇妙冒險|JoJo的奇妙冒險]]》系列的第一部，於《[[週刊少年Jump|週刊少年Jump]]》連載，自1987年1·2號連載至同年46號，共44話，收錄於單行本1～5卷。

「幻影血脈」為之後添加的副標題，連載當時的副標題為「第一部 喬納森·喬斯達 他的青春」。本作於2006年10月26日推出PS2遊戲，於2007年2月17日推出劇場版，並於2012年推出電視動畫。

== 故事大綱 ==
《幻影血脈》的故事舞台是19世紀末的[[英國|英國]]，達利歐‧布蘭度因為碰巧遇到遭逢馬車車禍的貴族喬治‧喬斯達而被喬治當成救命恩人，在達利歐因病過世之後，達利歐的兒子'''[[迪奧·布蘭度|迪奧·布蘭度]]'''（DIO）成為了喬斯達家的養子。迪奧為了奪取喬斯達家的家產，處處迫害喬治的兒子'''喬納森·喬斯達'''（JoJo），不但在喬納森的朋友中散播謠言、強吻喬納森的初戀情人艾琳娜，害死喬納森的愛犬，還企圖毒殺喬納森的父親喬治。

迪奧被收為養子七年後，他企圖下藥害死喬治的計畫被發現，喬納森在追查毒藥的來源時結識了食屍鬼街的流氓老大羅伯特‧E‧O‧史比特瓦根。與此同時，迪奧得知了一-{只}-被喬斯達家當作收藏品的面具「石鬼面」的用途，只要將面具戴上並沾血，就能將佩帶的人變成[[吸血鬼|吸血鬼]]。

喬納森從食屍鬼街歸來後，與變成吸血鬼的迪奧在喬斯達宅邸死鬥，他雖然重創了迪奧，自己也受了重傷。喬納森在艾琳娜的看護之下逐漸康復，並遇到了追查石鬼面的威廉·A·齊貝林男爵，喬納森為了打倒迪奧，拜齊貝林為師學習「波紋氣功」，這項能力可以透過呼吸產生的能量打倒吸血鬼。他們打倒了迪奧派出的手下，齊貝林在過程中喪命，但他在死前將能量傳遞給喬納森，讓喬納森得以擊敗迪奧。

在戰鬥結束後，1889年2月，喬納森和艾琳娜結婚、並乘船到美國度蜜月，但大難不死的迪奧也悄悄地埋伏在船上。在迪奧的突襲下，受了致命傷的喬納森不得不與迪奧同歸於盡，而救了一個嬰兒的艾琳娜則藏身在鐵製的棺材中逃過一劫。雖然喬納森已經逝世，但孕育於艾琳娜體內的新生命，以及艾琳娜在船上所救下的嬰兒都繼承了喬納森的高潔的精神，展開了[[戰鬥潮流|新的故事]]。

== 專有名詞 ==
; 石鬼面
: 一個石頭製成的面具，沾上血液後就會從面具側邊伸出數根骨針，刺進戴面具者的腦內並改變人類的基因，使他成為吸血鬼。變成吸血鬼之後，肉體會變得年輕、並且擁有強大的生命力，性格也會變得邪惡。能吸取他人體內的血液和能量，被吸取的人會變成屍生人([[殭屍|殭屍]])，大多數會喪失理性並按照本能行動。但是身體在照到太陽光([[紫外線|紫外線]])時就會化為灰燼。面具使用在人類以外的生物也會有同樣的效果。
: 在第一部登場的石鬼面是考古學家在阿茲特克遺跡發現，並且由倫敦的藝術品商人賣給喬斯達家的。而它的來源直到第二部才被交代清楚。
; 波紋氣功
: 從東方傳來的一種仙術，透過獨特的呼吸法讓體內的能量活化，並將這種生命能量變成波紋，這種波紋和太陽光的波形是相同的。修練者能延緩老化、以波紋療傷、傳遞能量，強大的波紋能擊潰吸血鬼的身體組織。波紋在水中傳遞的速度很快，但在無法呼吸的環境（例如水中）無法產生出波紋氣功。波紋可以透過生物的體液間接傳遞，但如果以金屬為媒介則需要直接接觸。吸血鬼使用了波紋氣功就會自我毀滅。
