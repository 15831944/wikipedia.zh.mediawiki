{{Engname|《幻乐之城》|''PhantaCity''}}，原名'''《创演之王》'''，是[[中国大陆|中国大陆]][[湖南卫视|湖南卫视]]与酷博特文化联合推出的中国首档音乐创演综艺秀，该节目是湖南卫视的原创节目，由制作过《[[百变大咖秀|百变大咖秀]]》、《[[全员加速中|全员加速中]]》的安德胜团队（原许可团队）打造，由[[香港|香港]]资深音乐人[[梁翘柏|梁翘柏]]担任联合出品人与幻乐发起人，由《[[我是歌手|我是歌手]]》及《[[歌手_(湖南卫视)|歌手]]》系列节目制作人[[洪涛_(制片人)|洪涛]]担任节目监制，由[[何炅|何炅]]、[[刘烨_(女主持)|刘烨]]联合主持，并由[[王菲|王菲]]担任幻乐体验官，主打「唱+演」结合的全新表演形式，以独创的创演表达来重塑音乐作品，并利用电影[[长镜头|长镜头]]的表现形式使观众获得沉浸式观演体验<ref>{{Cite web|url=http://ent.ifeng.com/a/20180709/43066039_0.shtml|title=洪涛梁翘柏创新出发 《幻乐之城》打造电视新物种|website=凤凰网娱乐|date=2018-07-09|accessdate=2018-07-11|deadurl=no|archiveurl=https://web.archive.org/web/20180709083439/http://ent.ifeng.com/a/20180709/43066039_0.shtml|archivedate=2018-07-09|df=}}</ref>。节目于2018年6月21日在[[江苏省|江苏]][[无锡市|无锡]]国家数字电影产业园启动录制<ref>{{Cite web|url=http://ent.sina.com.cn/music/zy/2018-06-20/doc-ihefphqk1763458.shtml|title=网曝王菲易烊千玺加盟《幻乐之城》 首期无锡开录|website=新浪娱乐|date=2018-06-20|accessdate=2018-07-11|deadurl=no|archiveurl=https://web.archive.org/web/20180711153558/http://ent.sina.com.cn/music/zy/2018-06-20/doc-ihefphqk1763458.shtml|archivedate=2018-07-11|df=}}</ref>，2018年7月20日起每週五20:00接档《[[我想和你唱_(第三季)|我想和你唱第三季]]》播出，并与《[[中餐厅|中餐厅]]第二季》叠播。歌曲版权由腾讯音乐娱乐集团独家取得，并通过旗下的[[QQ音乐|QQ音乐]]、[[酷狗音乐|酷狗音乐]]、酷我音乐联合发行。

== 赛制 ==
每期节目将召集四位歌手或演员作为唱演人，并在提出创意后分别与四位导演组成「幻乐拍档」（配对方式由四位导演抽签决定），与电影团队密切配合，共同完成幻乐之作。每部幻乐之作时长为8分钟，从创意提出到舞台公演都需要历经32天的打磨时间。节目充分融合了「[[歌舞片|音乐电影]]」与「[[现场直播|现场直播]]」两种艺术表现形式，现场分为「观影厅」与「幻乐空间」两个区域，观众在观影厅通过屏幕看到的画面均为幻乐空间内的实时唱演画面。四组拍档必须一气呵成，且只有一次唱演机会。唱演过程中，配乐、音效、镜头切换等效果均为一次性同步完成（现场乐队位于观影厅侧台），唱演人在唱演过程中不使用[[舞台监听系统|耳返]]，所有音乐和音效均通过监听音箱传达。同时，为了让观众获得沉浸式的观演体验，节目采取隐藏式收音的收音方式，唱演人在唱演过程中不使用手持式[[麦克风|话筒]]、耳挂式麦克风等设备，而是通过纽扣型隐藏式话筒进行收音<ref>{{Cite web|url=http://zixun.hunantv.com/hntv/20180720/1652198576.html|title=暑期重磅，开播在即！如何看懂《幻乐之城》？|website=金鹰网|date=2018-07-20|accessdate=2018-07-21|archiveurl=https://web.archive.org/web/20180721060809/http://zixun.hunantv.com/hntv/20180720/1652198576.html|archivedate=2018-07-21|deadurl=no}}</ref>。

节目组为四组幻乐拍档提供了三个不同等级的拍摄资源，而每组幻乐拍档获得的资源等级则会在幻乐之作创作初期由四位导演抽签决定。其中，A级资源对应300平方米的表演场地，B级资源对应200平方米的表演场地，C级资源对应100平方米的表演场地，三个资源等级提供的拍摄经费也各有不同。除B级资源为每期两组外，其余两个等级的拍摄资源均各为每期一组。

《幻乐之城》首创「幻乐体验官」概念，以三重视角全程多维度参与每部幻乐之作的诞生过程<ref>{{Cite web|url=http://zixun.hunantv.com/hntv/20180716/1013381554.html|title=王菲综艺首秀任“体验官” 湖南卫视《幻乐之城》即将献映|website=金鹰网|date=2018-07-16|accessdate=2018-07-21|archiveurl=https://web.archive.org/web/20180721060809/http://zixun.hunantv.com/hntv/20180716/1013381554.html|archivedate=2018-07-21|deadurl=no}}</ref>。首季参加节目的幻乐体验官为[[王菲|王菲]]。同时，每期节目还将邀请一位「幻乐好友」，与观众和体验官共同欣赏幻乐之作。此外，现场每位观众与嘉宾在观演过程中都将全程佩戴心动手环，实时监测观演过程中的心情曲线。

== 制作过程 ==
2018年4月27日，《幻乐之城》在2018湖南卫视创研大会上首次亮相<ref>{{Cite web|url=http://zixun.hunantv.com/hntv/20180428/144005237.html|title=湖南卫视创研大会原创飙速升级 爆款综艺、品质晚会、精彩大剧三连发|website=金鹰网|date=2018-04-28|accessdate=2018-07-11|deadurl=no|archiveurl=https://web.archive.org/web/20180711160642/http://zixun.hunantv.com/hntv/20180428/144005237.html|archivedate=2018-07-11|df=}}</ref>，随即引发舆论广泛关注。5月4日，香港媒体爆出王菲将作为体验官参加《幻乐之城》，酬劳高达3亿元人民币，引发网民热议，还有媒体爆出《幻乐之城》是为王菲定制的音乐节目，不仅节目名与王菲的「幻乐一场」演唱会相似，而且王菲在节目中也有极大的主导权<ref>{{Cite news|url=http://www.chinatimes.com/realtimenews/20180504003953-260404|title=王菲傳將參加湖南衛視歌唱節目 酬勞3億人民幣|publisher=中时电子报|date=2018-05-04|accessdate=2018-07-11|deadurl=no|archiveurl=https://web.archive.org/web/20180711160641/http://www.chinatimes.com/realtimenews/20180504003953-260404|archivedate=2018-07-11|df=}}</ref>。针对这些消息，节目组于5月5日发表声明，确认王菲正式加盟，并表示网传节目形式、嘉宾片酬等内容的报道为虚假消息，否认了所谓的「3亿片酬传闻」<ref>{{Cite web|url=http://ent.163.com/18/0505/20/DH2NKGIM00037VVV.html|title=《幻乐之城》官方发声明 网传谣言不攻自破|website=网易娱乐|date=2018-05-05|accessdate=2018-07-11|deadurl=no|archiveurl=https://web.archive.org/web/20180711160642/http://ent.163.com/18/0505/20/DH2NKGIM00037VVV.html|archivedate=2018-07-11|df=}}</ref>。

== 反响与评价 ==
《幻乐之城》以「唱+演」结合的全新表演形式，并利用现场实演无剪辑的表现形式使节目成为一个即兴的音乐电影拍摄舞台，而类似的综艺呈现形式在国外也鲜有节目敢于尝试，《幻乐之城》也因此被业内专家称为「中国电视新物种」<ref>{{Cite news|url=http://epaper.gmw.cn/gmrb/html/2018-10/22/nw.D110000gmrb_20181022_5-03.htm|title=【文化评析】观众需要更多原创电视新物种|publisher=光明日报|author1=汤嫣|date=2018-10-22|accessdate=2018-11-01|deadurl=no|archiveurl=https://web.archive.org/web/20181101180839/http://epaper.gmw.cn/gmrb/html/2018-10/22/nw.D110000gmrb_20181022_5-03.htm|archivedate=2018-11-01}}</ref>。

2018年10月16日，由[[北京大学|北京大学]][[北京大学新闻与传播学院|新闻与传播学院]]、北京大学视听传播研究中心主办的《幻乐之城》品牌研讨会在北京举行，来自全国各地的数十位业界、学界代表和媒体记者对节目的品牌价值进行了积极探讨。北京大学新闻与传播学院院长、教授陆绍阳对节目的制作模式给予充分肯定，并对《幻乐之城》坚持「艺术精湛、思想精深、制作精良」的「三精」创作标准进行了高度赞扬，整个创作过程完全回归到作品本身，是综艺节目制作的一个良好导向。《[[人民日报|人民日报]]》政文部文教週刊副主编刘阳，[[中国艺术研究院|中国艺术研究院]]电影电视艺术研究所所长、研究员丁亚平，[[中国传媒大学|中国传媒大学]]戏剧影视学院副院长、教授关玲则对节目全电影化的呈现形式给予认可。[[中国社会科学院|中国社科院]]新闻所世界传媒研究中心秘书长、副研究员冷凇，中国传媒大学新闻传播学部教授、博士生导师郎劲松，[[中国新闻网|中国新闻网]]副总编辑、中新经纬CEO[[符永康|符永康]]分别从幻乐之作的选题以及表达出的情感等方面作出了认可<ref>{{Cite web|url=http://zixun.hunantv.com/hntv/20181019/140608829.html|title=新时代，新价值！坚定文化自信，坚持自主创新 “中国电视新物种”《幻乐之城》品牌研讨会在京召开|website=金鹰网|date=2018-10-19|accessdate=2018-11-01|deadurl=no|archiveurl=https://web.archive.org/web/20181101180839/http://zixun.hunantv.com/hntv/20181019/140608829.html|archivedate=2018-11-01}}</ref>。同年11月26日，《幻乐之城》被[[国家广播电视总局|国家广播电视总局]]评选为2018年第三季度广播电视创新创优节目<ref>{{Cite web|url=http://www.nrta.gov.cn/art/2018/12/3/art_113_39855.html|title=国家广播电视总局关于公布2018年第三季度广播电视创新创优节目的通知|publisher=国家广播电视总局|date=2018-12-03|accessdate=2018-12-05|deadurl=no|archiveurl=https://web.archive.org/web/20181205181420/http://www.nrta.gov.cn/art/2018/12/3/art_113_39855.html|archivedate=2018-12-05}}</ref>。

《幻乐之城》的热播还获得了包括《人民日报》、《[[光明日报|光明日报]]》等在内的多家权威媒体的关注。节目最后一集播出后，已有豆瓣网友超过2万人参与打分，评分为7.0分，且播出期间曾创下最高评分7.2分的好成绩。截至北京时间11月5日17:50，《幻乐之城》微博主话题#幻乐之城#阅读量超过58亿，粉丝讨论量破2415万，节目首播期间持续位居综艺话题实时榜榜首；12期节目共有50个热搜词上榜，其中有26个跻身前五；节目相关视频仅在微博秒拍播放量便超过36亿；微博互动总数突破607万次；各项-{zh-hans:数据;zh-hant:資料;}-均远超同期热门节目<ref>{{Cite web|url=http://zixun.hunantv.com/hntv/20181022/1459324016.html|title=中国电视新物种《幻乐之城》收官高口碑爆款引爆电视革命|website=金鹰网|date=2018-10-22|accessdate=2018-11-01|deadurl=no|archiveurl=https://web.archive.org/web/20181101180839/http://zixun.hunantv.com/hntv/20181022/1459324016.html|archivedate=2018-11-01}}</ref>。

在获得好评的同时，《幻乐之城》也因疑似[[假唱|假唱]]、缺乏综艺感以及体验官王菲镜头过少等问题遭到观众的质疑<ref>{{Cite web|url=http://ent.163.com/18/0721/05/DN7D4A5O00038FO9.html|title=《幻乐之城》假唱？洪涛：人格担保100%是真唱|website=网易娱乐|author1=宋玉鑫|date=2018-07-21|accessdate=2018-11-01|deadurl=no|archiveurl=https://web.archive.org/web/20181101180839/http://ent.163.com/18/0721/05/DN7D4A5O00038FO9.html|archivedate=2018-11-01}}</ref><ref>{{Cite news|url=http://www.xinhuanet.com/ent/2018-08/05/c_1123223679.htm|title=《幻乐之城》导演回应质疑：未来增加综艺性|website=新华网娱乐|publisher=广州日报|author1=莫斯其格|date=2018-08-05|accessdate=2018-11-01|deadurl=no|archiveurl=https://web.archive.org/web/20181101180840/http://www.xinhuanet.com/ent/2018-08/05/c_1123223679.htm|archivedate=2018-11-01}}</ref>。
