《'''幻影少年'''》（{{lang|ja|モノクローム・ファクター}}）是一部從2004年初夏開始於[[COMIC_BLADE_MASAMUNE|COMIC BLADE MASAMUNE]]連載的漫畫作品，作者是[[空廼海里|空廼海里]]。從2007年9月起轉到[[月刊Comic_Blade_Avarus|月刊Comic Blade Avarus]]繼續連載。同名的[[電視動畫|電視動畫]]於2008年4月開始在日本播放。

== 故事簡介 ==
:中學生二海堂 昶與一個自稱白銀的神秘男子相遇后，被捲入shin的紛爭中，生活從此徹底改變了。

== 登場人物 ==
<small>※[[配音員|配音員]]一项中左邊為動畫版配音員，右邊則為CD版。</small>

; 二海堂 昶（{{lang|ja|にかいどう あきら}}）
:配音員：[[小野大輔|小野大輔]] / 同左
:動漫中為十六歲的高校生，對平常日子的感想是「無聊」。
:包括運動能力在內，各方面都很強，但不主動顯示出來。
:繼承了光之王劉黑的因子，類似劉黑的轉世。漫画最后刘黑将力量全部托付给昶后消失，昶也成为了新的光之王。
:武器為兩把獵[[刀|刀]]，顏色透明為其特色。他要變成他的影子「蜃」(Shin)才能跟蟲崎戰鬥
:喜歡戰鬥[[靴|靴]]和[[項鍊|項鍊]]，生日：12月11日體重：57kg

; 白銀（{{lang|ja|しろがね}}）
:配音員：[[諏訪部順一|諏訪部順一]] / [[子安武人|子安武人]]
:動漫中很少戴上墨鏡，平常總是溫柔地微笑，實際上有點腹黑。
:動畫中設定為「白銀喜歡昶」，漫畫則沒有。
:動畫能力方面也比漫畫顯示得少。
:影世界的王，是「闇」的直系王族。與劉黑是相對應的存在，很在乎他。漫画中曾想要抹杀昶以希望刘黑可以回归。
:動畫在和昶合力打敗焰緋後消失，但被昶等人認為白銀還活在這個世上。

; 淺村賢吾（{{lang|ja|あさむら けんご}}）
:配音員：[[神谷浩史|神谷浩史]] / [[關智一|關智一]]
:動漫中為昶的同班同學，有一個姐姐，常被指「笨蛋」。
:以前的頭髮是黑色的，高中時染為黃色。
:漫画中是“黑暗因子持有者”，因此易被影影响及附身。
武器為拳擊[[手套|手套]]。
生日：8月22日體重：61kg

; 鈴野 綾（{{lang|ja|すずの あや}}）
:配音員：[[淺野真澄|淺野真澄]] / [[長澤美樹|長澤美樹]]
:動漫中為昶的同班同學、校中的風紀委員。個性兇悍在第1集被旭稱為(學校第一兇悍的人),時常喜歡管束旭.
:最怕的東西是[[幽靈|幽靈]]。
:武器為[[日本刀|日本刀]]連同右手裝甲 生日：9月15日體重：45kg

; 洸（{{lang|ja|こう}}）
:配音員：[[小西克幸|小西克幸]] /－
:動畫第十三集才出現的角色。
:喜歡可愛的小姑娘，常常對綾性騷擾。
:把悠當作弟弟一樣來看待，甚至還要悠叫他哥哥。
:其實是劉黑創造出來的生命，本体是一只大型犬，製造方法與技術為禁術。

; 我妻秋一（{{lang|ja|わがつま しゅういち}}）
:配音員：[[羽多野涉|羽多野涉]] / [[石田彰|石田彰]]
:動漫中為白銀的好友，一般都被叫作"Master"。
:是"BAR STILL"酒吧的老闆，同時是個有著「靈力」的盲目男人。
:漫畫中，真實身分為祀翠，是劉黑以外的另外一位光之王。

=== 影世界 ===
; 焰緋（{{lang|ja|ほむらび}}）
:配音員：[[置鮎龍太郎|置鮎龍太郎]] /－
:因為不願祀翠被換掉，要讓兩人成為直系王族，而決定弒王，假借跟劉黑談祀翠的事，偷偷叫澤木從後方刺殺劉黑。
:之後把劉黑的因子拿走，用其放逐白銀。

; 冰瀏（{{lang|ja|ひりゅう}}）
:焰緋用劉黑的因子創造出來的「子」，最後回歸「核心」的昶身上。

; 淺村 麻結（{{lang|ja|あさむら まゆ}}）

:與昶等人為好朋友，實際上卻不斷提供情報給七夜及露露，他們才不斷遭受攻擊。

; 大久保 彦十郎（{{lang|ja|おおくぼ ひこじゅうろう}}）

:配音員：[[森川滋夫|森川滋夫]] /－

== 特殊詞彙 ==
; 即時分身
:製造分身（影子）來使SHIN具現化的道具，灰色的娃娃。
:在動漫中有嘲笑昶的表情，白銀的解釋為「錯覺」。
:在SHIN時踩它進入具現化狀態。
:脫離狀態有兩種說法，一是Master說的－－「心裡想著就行」，二是白銀及事實証明－－喊出「影化解除」(doppelganger release)。
