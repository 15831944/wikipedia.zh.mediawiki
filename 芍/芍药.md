'''芍藥'''（[[学名|学名]]：''{{lang|la|Paeonia lactiflora}}''），又稱作'''白芍''''、'''杭白芍'''、'''金芍藥'''，是一種[[芍藥科|芍藥科]][[芍藥屬|芍藥屬]]的著名草本花卉。

== 形态 ==

[[多年生草本|多年生草本]]，高60－80厘米，地下有圆柱形或纺锤形块根。二回三出[[复叶|复叶]]。初夏开美丽的花，花大型，有白、红、粉红等颜色，[[雌蕊|雌蕊]]常无毛，花型多样，有单瓣、半重瓣和重瓣等多种花型。遍布于中国[[中国东北地区|東北]]及[[華北|華北]]，亦见于[[西伯利亚|西伯利亚]]等地。一般可以成活数年。

=== 与牡丹的区别 ===
芍药是蓄根[[草本植物|草本]]，[[牡丹|牡丹]]是[[灌木|灌木]]木本，这是它们的主要区别，它们花型、叶片非常相似，牡丹于5月初开花，芍药花期要晚一些。在[[英语|英语]]和其他[[欧洲|欧洲]]语言中，牡丹和芍药是同一个词。芍药一般可以成活数年，牡丹可以存活30-60年。

== 历史 ==
芍药在[[夏代|夏]][[商代|商]][[周代|周]]时期，即已被中国人作为观赏[[植物|植物]]培育。

== 药用 ==
芍药的[[根|根部]]可药用。在夏秋两季将根部挖掘后，去除[[根茎|根茎]]及须根，洗净去皮得到'''白芍'''<ref>{{Cite web|url=http://zhongyaocai360.com/b/baishao.html#6255|title=中药大辞典 白芍|accessdate=2018-07-14|author=南京中医药大学|date=2006年|authorlink=南京中医药大学|work=|publisher=上海科学技术出版社|language=zh-CN}}</ref>。野生种'''芍药'''及[[草芍药|草芍药]]、[[川赤芍|川赤芍]]的不去皮根也被称为'''赤芍'''<ref>{{Cite web|url=http://zhongyaocai360.com/zhongyao/chishaoyao.html|title=中药大辞典 赤芍|accessdate=2018-07-14|author=南京中医药大学|date=2006年|authorlink=南京中医药大学|work=|publisher=上海科学技术出版社|language=zh-CN}}</ref>，所以[[汉语|汉语]]中称其为芍“药”{{来源请求}}'''。'''

味苦酸，性凉，入[[肝_(臟腑)|肝]]、[[脾_(臟腑)|脾]]经。'''白芍'''的功效为养血敛银、柔肝止痛、平抑肝阳。'''赤芍'''的功效则为清热凉血、祛瘀止痛<ref>{{Cite book|url=https://www.worldcat.org/oclc/132681238|title=中药学|last=凌一揆|first=|authorlink=凌一揆|last2=颜正华|first2=|last3=|last4=|date=1984|publisher=上海科学技术出版社|year=|isbn=7532304973|edition=第一版|location=上海|pages=|chapter=|oclc=132681238}}</ref>。

== 欧洲引种 ==
芍药於1700年代中期被引种到[[英格兰|英格兰]]，并培育出很多当地普遍种植的园艺栽培种<ref>{{cite book|last=Loewer|first=H. Peter|title=Fragrant Gardens|publisher=Houghton Mifflin Gardening|date=1999|pages=48|isbn=9780395884928|url=http://books.google.com/books?id=bBJdIKPiGuMC&pg=PA48&dq=Peony+common+garden&as_brr=3&ei=t7JXSdakD4TOlQT-0ZmXCA&client=safari#PPA48,M1}}</ref>。芍药在过去很长的一段时间里都被描述为''P. albiflora''，因为芍药是第一个被引种到欧洲的白色芍药属品种。芍药繁殖力很强，是切花贸易所需的芍药属花卉中最主要的来源<ref name="Halda">{{cite book|last=Halda|first=Josef J.|coauthors=James W. Waddick|title=The Genus Paeonia|publisher=Timber Press|date=2004|pages=196-205|isbn=9780881926125|url=http://books.google.com/books?id=mDLASKInbjgC&pg=PA196&dq=Peony+common+garden&lr=&as_brr=3&ei=obNXSaPWNoL4lQSq3q3XAw&client=safari#PPA202,M1}}</ref>。

在中国，芍药的观赏价值稍逊於[[紫斑牡丹|紫斑牡丹]]（''Paeonia rockii''）和[[牡丹|牡丹]]（''Paeonia suffruticosa''）。

== 其他 ==
在中国，大多数车迷将[[韶山1型电力机车|韶山1型电力机车]]昵称为“芍药”。
