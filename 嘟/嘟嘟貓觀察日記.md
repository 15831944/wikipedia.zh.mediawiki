《'''嘟嘟貓觀察日記'''》（{{lang|ja|ポヨポヨ観察日記}}）是日本漫畫家{{lang|ja|樹るう}}的[[四格漫畫|四格漫畫]]作品。

== 作品概要 ==
作品以架空的田園都市千葉縣的竹馬市為舞臺。一天晚上，'''佐藤 萌'''于紅燈區喝醉酒，在街上無意地把一隻流浪貓當作枕頭睡在上面，並在第二天把此流浪貓帶回家中。從此以後，那隻被稱為'''嘟嘟'''的貓被佐藤一家飼養，并與身邊的人展開了溫馨的日常生活。

== 登場人物 ==

: 球形貓，毛色為黃色，頭上有一個類似王字型的粽色斑紋，四肢和尾巴短小，喜歡發出「唏呀（ヒァ）」的聲音。被萌帶回家中飼養，時常和英有爭拗，并抓傷英。

; {{nihongo|佐藤 萌|佐藤 萌（さとう もえ）|聲：[[葉月奈乃|葉月奈乃]] / [[三森鈴子|三森鈴子]]}}（22歲）
: 佐藤家的長女。嘟嘟的飼主。是她在[[紅燈區|紅燈區]]喝醉酒把流浪貓（嘟嘟）當枕頭度過一晚而帶回來的人。

; {{nihongo|佐藤 英|佐藤 英（さとう ひで）|聲：[[堀龍也|堀龍也]] / [[市瀨秀和|市瀨秀和]]}}（17歲）
: 小萌的弟弟。有著很冷酷的個性，時常和嘟嘟爭吵。

; {{nihongo|佐藤 茂|佐藤 茂（さとう しげる）|聲：[[神谷明|神谷明]]}}（47歲）
: 佐藤家的父親。

; {{nihongo|山下 真紀|山下 マキ（やました マキ）|聲：[[千葉芙美香|千葉芙美香]]}}（17歲）
: 小英的同班同學。

; {{nihongo|鈴木 浩太|鈴木 浩太（すずき こうた）|聲：[[室元氣|室元氣]]}}（17歲）
: 小英的同班同學。

; {{nihongo|杉田 花|杉田ハナ（すぎた ハナ）|聲：[[神谷明|神谷明]]}}
: 佐藤家的鄰居，一人獨居的老奶奶。年輕時是個頭髮如柔絲的美少女。

; {{nihongo|黑兵衞|クロベエ（雄：愛原クロ）|聲：[[三宅麻理恵|三宅麻理恵]]}}
: 小花奶奶所養的貓。

; {{nihongo|獸醫／甘木 雄一郎|獣医／甘木雄一郎（あまぎ ゆういちろう）}}
: 嘟嘟們曾光顧的獸醫。診所名是「{{nihongo|套房動物醫院|スイート動物病院}}」。

; {{nihongo|田中 由香|田中 ユカ（たなか ユカ）|聲：[[三宅麻理恵|三宅麻理恵]]}}
: 搬遷到佐藤家附近的年輕太太。與丈夫一起搬遷到有園藝憧憬的鄉下來。

; {{nihongo|由香的丈夫|ユカの夫|}}
: 研究員。為了在筑波學園都市工作，夫婦一起搬遷過來。

; {{nihongo|艾莉|エリ}}（雌）
: 萌所報讀的書道教室的老師家中所養的小貓。

; {{nihongo|阿虎|トラ}}（雌）
: 嘟嘟的母親。與嘟嘟毛色一樣，體型正常。

; {{nihongo|三毛子|ミケコ}}（雌）
: 阿虎的女兒，嘟嘟的妹妹或者姐姐，如名字所見，是一隻三色貓，體型正常。

; {{nihongo|小嘟嘟|小ポヨ}}（全是雄）
: 三毛子的孩子們。與嘟嘟一樣是球型貓。'''分別是{{nihongo|小丸|マル}}'''（鯖魚貓）與'''{{nihongo|小滾|コロ}}'''（三毛貓）兩兄弟。

; {{nihongo|渡邊|渡辺（わたなべ）}}
:阿虎、三毛子、小嘟嘟們的青年飼主。
