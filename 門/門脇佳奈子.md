'''門脇佳奈子'''（{{lang-ja|門脇佳奈子}}，{{bd|1996年|10月24日|catIdx=}}）是[[NMB48|NMB48]] Team BII的成員。大阪府出身。所屬經紀公司為KYORAKU吉本。

== 經歷 ==
; 2010年
: 9月20日、參加『NMB48一期生徵選會』合格（參加總數7,256名、最終合格者26名）。
; 2011年
: 1月1日、[[大阪市|大阪市]]・[[難波|難波]]的[[NMB48劇場|NMB48劇場]]落成，『[[teamA_3rd_Stage「为了谁」#NMB48_team_N_1st_Stage「為了誰」公演|NMB48 研究生 1st Stage「为了谁」]]』公演開始，為選抜成員16名之1人在劇場公演。
: 2011年3月10日、NMB48第1期研究生25名中選出16人組成Team N，門脇亦是其中一員。
; 2014年
: 2月24日 - 於「AKB48集團大組閣祭」中宣布所屬隊伍由Team N轉移至Team BII。
; 2015年
: 10月22日 - 於「NMB48 5th Anniversary LIVE」最終日中宣布畢業。
