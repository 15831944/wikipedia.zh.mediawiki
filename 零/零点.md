对[[全纯函数|全纯函数]]''f''，称满足''f''(''a'') = 0的[[复数|复数]]''a'' 为 ''f'' 的'''零点'''。

== 零点的阶 ==

复数''a''是''f''的'''简单零点'''，或称''f''的'''一阶零点'''，如果''f''可以写成以下的形式：

:<math>f(z)=(z-a)g(z)\,</math>

其中''g''是全纯函数，''g''(''a'')不为零。

一般地，''f''在''a''处的零点的阶，是满足下式的最大正整数''n''，其中''g''是全纯函数：

:<math>f(z)=(z-a)^ng(z)\ </math>且<math>\ g(a)\neq 0.\,</math>

== 零点的存在 ==

[[代数基本定理|代数基本定理]]说明，任何一个不是常数的複系数[[多项式|多项式]]在[[复平面|复平面]]内都至少有一个零点。这与[[实数|实数]]的情况不一样：有些实系数多项式没有实数根。一个例子是''f''(''x'') = ''x''<sup>2</sup> +1。

== 性质 ==

不恒为0的全纯函数的零点有一个重要的性质：零点都是孤立的。也就是说，对于不恒为0的全纯函数的任何一个零点，都存在一个[[邻域|邻域]]，在这个邻域内没有其它零点。
