{{vgsname|零|ja=零|na=Fatal Frame|eu=Project Zero}}為日本厂商[[特庫摩|特庫摩]]开发的恐怖游戏系列，自[[2001年|2001年]]問世以来，已推出了五部正傳作品：《[[零～zero～|零～zero～]]》、《[[零～紅蝶～|零～紅蝶～]]》、《[[零～刺青之聲～|零～刺青之聲～]]》、《[[零～月蝕的假面～|零～月蝕的假面～]]》和《[[零～濡鴉之巫女～|零～濡鴉之巫女～]]》，一部外傳作品《心靈照相機～被附身的筆記本～》，以及一部重製作品《零～真紅之蝶～》；游戏平台則橫跨了[[PS2|PS2]]、[[Xbox|Xbox]]、[[Wii|Wii]]、[[3DS|3DS]]和[[Wii_U|Wii U]]。

[[2012年|2012年]]，零系列的版權商標正式公布由[[任天堂|任天堂]]與[[光榮特庫摩|光榮特庫摩]]共同所持有，其中任天堂佔有相當重要的份額。

=== 零～zero～ ===
{{main|零～zero～}}

[[PS2|PS2]]{{vgname|零～zero～|bold=no|ja=零〜zero〜|na=Fatal Frame|eu=Project Zero}}为系列的第一作，[[2001年|2001年]]首发于日本[[PlayStation_2|PlayStation 2]]<ref>{{cite web|url=http://www.gamespot.com/fatal-frame/related/release/platform/ps2/|title=Fatal Frame Release Summary|publisher=[[GameSpot|GameSpot]]|accessdate=29 July 2012}}</ref>。

[[2002年|2002年]]发行的[[Xbox|Xbox]]版在日本称为“Fatal Frame -零 Special Edition-”，加入了新结局，更多服装，新的游戏模式和更困难的设置{{citation needed|time=2014-09-05T16:48:00+00:00}}。

=== 零～紅蝶～ ===
{{main|零～紅蝶～}}

[[PS2|PS2]]{{vgname|零～紅蝶～|bold=no|ja=零〜紅い蝶〜|na=Fatal Frame II: Crimson Butterfly|eu=Project Zero II: Crimson Butterfly}}是系列的第二作，[[2003年|2003年]]於日本PlayStation 2平台首发。

翌年，在[[Xbox|Xbox]]发行的“导演剪辑版”收录了一些新增要素。游戏被广泛认为是史上最恐怖的电子游戏之一<ref name="gspot">{{cite web|url=https://www.gamespot.com/fatal-frame-ii-crimson-butterfly/|title=Fatal Frame II Review|last=Massimilla|first=Bethany|date=11 December 2003|work=[[GameSpot|GameSpot]]|accessdate=14 January 2012}}</ref><ref name="ign">{{cite web|last=Dunham|first=Jeremy|title=Fatal Frame II: Crimson Butterfly Review|url=https://www.ign.com/articles/2003/11/19/fatal-frame-ii-crimson-butterfly|work=[[IGN|IGN]]|accessdate=17 January 2013|date=19 November 2003}}</ref>。

在[[北美|北美]]，[[2013年|2013年]][[5月7日|5月7日]] 发行了可供[[PlayStation_3|PlayStation 3]]下载的PlayStation 2版。PlayStation 3版在发行后不久因为各种模拟技术问题移除了在线商店<ref>{{cite web |author= |url=http://community.us.playstation.com/t5/PlayStation-Network-Support/Known-Issue-Fatal-Frame-2-on-PSN/td-p/40386809 |title=[Known Issue] Fatal Frame 2 on PSN - PlayStation® Community Forums |publisher=Community.us.playstation.com |date=2013-05-16 |accessdate=2013-07-25 |archive-url=https://web.archive.org/web/20130709000150/http://community.us.playstation.com/t5/PlayStation-Network-Support/Known-Issue-Fatal-Frame-2-on-PSN/td-p/40386809 |archive-date=2013-07-09 |dead-url=yes }}</ref>，直到 6月30日 修复并重新发行<ref>http://www.destructoid.com/sony-takes-fatal-frame-2-in-for-repairs-this-week-on-psn-258841.phtml</ref>。

=== 零～刺青之聲～ ===
{{main|零～刺青之聲～}}

[[PS2|PS2]]{{vgname|零～刺青之聲～|bold=no|ja=零 〜刺青ノ聲〜|na=Fatal Frame III: The Tormented|eu=Project Zero 3: The Tormented}}是系列的第三部作品<ref>{{cite web|url=http://www.gamespot.com/fatal-frame-iii-the-tormented/related/release/platform/ps2/|title=Fatal Frame III: The Tormented Release Summary|publisher=[[GameSpot|GameSpot]]|accessdate=29 July 2012}}</ref>，[[2005年|2005年]]在[[PlayStation_2|PlayStation 2]]平台发行。

游戏同时使用了旧角色和新角色，并使用了新闹鬼地点<ref>{{cite web|title=Fatal Frame 3: The Tormented for PlayStation 2|url=http://www.gamestop.com/ps2/games/fatal-frame-3-the-tormented/39626|work=[[GameStop|GameStop]]|publisher=GameStop|accessdate=19 November 2012|deadurl=yes|archiveurl=https://web.archive.org/web/20121123050624/http://www.gamestop.com/ps2/games/fatal-frame-3-the-tormented/39626|archivedate=23 十一月 2012}}</ref><ref>{{cite web|url=http://us.playstation.com/games/fatal-frame-iii-the-tormented-ps2.html|title=Fatal Frame III: The Tormented for the PS2™ system. === 零～月蝕的假面～ ===
{{main|零～月蝕的假面～}}

[[Wii|Wii]]{{vgname|零～月蝕的假面～|bold=no|ja=零 〜月蝕の仮面〜}}是系列的第四作，[[2008年|2008年]]在日本發售。本作是特库摩和[[任天堂|任天堂]]的合作项目，特库摩负责游戏的开发制作，此外[[Grasshopper_Manufacture|Grasshopper Manufacture]]也有参与开发工作；而任天堂则担当游戏的监修和发行。

本作成为零系列中第一部[[Wii|Wii]]主机独占的游戏；同时据任天堂的消息，本作也有别于前三作而不会在北美地区发行<ref name="ign2">[http://wii.ign.com/articles/970/970875p1.html Fatal Frame Never Coming to America?]</ref>。

=== 心靈照相機～被附身的筆記本～ ===

[[3DS|3DS]]{{vgname|心靈照相機～被附身的筆記本～|bold=no|ja=心霊カメラ 〜憑いてる手帳〜|na=Spirit Camera: The Cursed Memoir}}，是本系列的第一款外傳作品，[[2012年|2012年]]1月在日本發行，遊戲中收錄「幽靈相機（ゴーストカメラ）」模式，將可利用[[3DS|3DS]]攝影鏡頭模擬照相機來尋找和除靈。

而「AR日記（ARノート）」，可以體驗專屬故事{{nihongo|《零～紫之日記～》|零 〜紫の日記〜}}，玩家要與被帶離現實世界的少女「真夜」一起探索，揭開被詛咒的日記背後的真相。此外遊戲中還收錄「驚悚日記（ホラーノート）」模式，讓玩家利用3DS AR功能體驗許多不同的小遊戲。

=== 零～真紅之蝶～ ===


[[Wii|Wii]]{{vgname|零～真紅之蝶～|bold=no|ja=零 〜眞紅の蝶〜|eu=Project Zero 2: Wii Edition}}為《[[零～紅蝶～|零～紅蝶～]]》的重製版本，[[2012年|2012年]]在日本、澳洲和歐洲发行<ref name="pz2">{{cite web|title=Project Zero 2: Wii Edition|url=https://www.gamespot.com/project-zero-2-wii-edition/|last=|first=|date=|work=[[GameSpot|GameSpot]].com|publisher=[[GameSpot|GameSpot]]|accessdate=10 January 2013}}</ref><ref name="Nintendo germany">{{cite web|url=https://twitter.com/#!/NintendoDeutsch/status/172310652319318017|title=Nintendo Germany Twitter|language=de |quote=}}</ref>。

為配合Wii的主機特性，本作翻新人物模組、重製過場動畫和新增體感操作系統，並加入可供雙人遊玩的『鬼屋模式（お化け屋敷）』。

=== 零～濡鴉之巫女～ ===
{{main|零～濡鴉之巫女～}}

[[Wii_U|Wii U]]{{vgname|零～濡鴉之巫女～|bold=no|ja=零 〜濡鴉ノ巫女〜}}由[[光荣特库摩|光荣特库摩]]与[[任天堂|任天堂]]联合制作并由任天堂发行在 Wii U 平台上，為零系列首次[[高畫質|高畫質]]化之作品。

由于 Wii U GamePad 平板控制器的加入，使得系列游戏特色之一的“射影機”使用起来更加得心应手。

游戏[[2014年|2014年]]9月27日在日本发售。

== 设定 ==
{{see also|零系列角色列表}}

; 射影機 : 由日本民俗學家麻生邦彥博士研製，能夠拍攝到“看不見的事物”的特殊照相機。零系列最關鍵物品，在零系列中有數台不同型號的射影機存在，根據遊戲中獲得的麻生博士本人遺留的筆記來看，除了《零～zero～》中使用的以外，其餘均爲“試作品”。
; 靈石收音機 : 以用礦石做為部份迴路來收訊的礦石收音機改良而成，據說可以收到靈界的聲音，靈界的聲音可以藉由收音機而被聽見，與人能夠感受某些氣息產生預感、雙胞胎之間的共鳴有著類似的特性。
; 膠卷（菲林） : 分〇七式、一四式（十四式）、三七式（《零～zero～》特有，相當於六一式）、六一式、七四式（《零～zero～》特有，相當於九〇式）、九〇式和零式。每種膠卷皆有不同的攻擊力和填充時間，是射影機能攻擊怨靈的原因。
; 鏡石 : 數量稀少，主角身上一般只能裝備一塊。當怨靈攻擊主角，导致体力降到0时，鏡石會自动地完全回復體力，但受到的伤害仍会保留。例如，遇到超強怨靈（通常在惡夢难度下）的攻击時，可能會先爆鏡石再掉半條血。
; 万葉丸 : 遊戲中的補血物品，能補充少量體力，可同時持有多個。
; 御神水 : 遊戲中的補血物品，數量稀少，能完全補充體力，和鏡石一樣重要，可同時持有多個。
; Zero Shot : 靈力圈全蓄力狀態與靈力圈呈紅色狀態下拍照。（包含Fatal Frame）
; 靈石 : 《零～zero～》中用于使用照相機的強化功能（如：壓、遲、視、痺、探等）的物品，每使用一次要消耗一顆。在其后作中变为“靈子”，通过对怨灵造成伤害获得，直接填充在射影機左下角。不同的強化鏡頭消耗的靈子數也不同，例如“擊”消耗全靈力（依當時所剩靈子決定攻擊力）、“零”消耗3顆、“連”消耗1顆等。

== 音乐 ==

系列从《零～紅蝶～》开始到《零～真紅之蝶～》的主题曲皆由[[天野月|天野月]]（原天野月子）演唱，包括：《零～紅蝶～》主题曲《蝶》、《零～刺青之聲～》主题曲《聲》、《零～月蝕的假面～》两个难度版本的主题曲《NOiSE》和《零的調律》，以及《零～真紅之蝶～》主题曲《くれなゐ》。而《零～濡鴉之巫女～》的主题曲《HIGANBANA》改由AnJu演唱，另外在達成特殊條件之下（夕莉身穿白無垢），片尾曲將為[[天野月|天野月]]所演唱的《鳥籠 -in this cage-》。
