'''{{Ruby|羧|-{zh-cn:suō;zh-hk:粵音so1 同 "梳" ;zh-tw:ㄙㄨㄛ}-}}酸'''<ref>[http://terms.naer.edu.tw/terms/manager_admin/new_file_download.php?Pact=FileDownLoad&button_num=g5&source_id=159&Pval=67 國立編譯館《化學命名原則》第四版〈附錄一　化學名詞用字之讀音〉]</ref>（{{lang-en|Carboxylic acid}}），[[有機酸|有機酸]]的一種，是帶有[[羧基|羧基]]的[[有機化合物|有機化合物]]<ref>{{GoldBookRef|title=carboxylic acids|file=C00852}}</ref>，通式是R-COOH。羧基的[[化學式|化學式]]為-C(=O)OH，通常寫成-COOH或-CO<sub>2</sub>H，它是由一個{{Ruby|羰|-{zh-cn:tāng;zh-hk:粵音tong1 同"湯";zh-tw:ㄊㄤ}-}}基<ref>[http://terms.naer.edu.tw/terms/manager_admin/new_file_download.php?Pact=FileDownLoad&button_num=g5&source_id=159&Pval=67 國立編譯館《化學命名原則》第四版〈附錄一　化學名詞用字之讀音〉]</ref>（RR'C=O）和一個{{Ruby|羥|-{zh-cn:qiǎng;zh-hk:koeng5;zh-tw:ㄑㄧㄤ}-}}基<ref>[http://terms.naer.edu.tw/terms/manager_admin/new_file_download.php?Pact=FileDownLoad&button_num=g5&source_id=159&Pval=67 國立編譯館《化學命名原則》第四版〈附錄一　化學名詞用字之讀音〉]</ref>（R-O-H）組成的[[官能基|官能基]]。羧酸是[[質子|質子]]（H<sup>+</sup>）授予者，也就是[[酸鹼質子理論|質子酸]]（[[酸鹼質子理論|布-洛酸]]）。可分為[[脂肪酸|脂肪酸]]及[[芳香酸|芳香酸]]二大類。

[[螞蟻|螞蟻]]和[[蜜蜂|蜜蜂]]的分泌液中含的[[甲酸|甲酸]]H-COOH、使[[醋|醋]]有[[味覺|酸味]]的[[乙酸|乙酸]]CH<sub>3</sub>-COOH...等都是常見的羧酸。擁有兩個羧基的分子叫二酸，最簡單的分子是由兩個羧基結合而成的[[草酸|草酸]](COOH)<sub>2</sub>。其他像是可在[[檸檬|檸檬]]中發現的[[檸檬酸|檸檬酸]]和[[酸豆|酸豆]]內含的[[酒石酸|酒石酸]]都是自然界中重要的羧酸。

羧基[[去質子化|去質子化]]以後的共軛鹼便是含有羧酸根[[離子|負離子]]（-COO<sup>−</sup>）的[[羧酸鹽|羧酸鹽]]，與[[醇類|醇類]]混合經[[酯化反應|酯化反應]]後形成[[羧酸酯|羧酸酯]]。羧酸離子因為[[共振_(化學)|共振]]而比較穩定，因此其[[酸|酸性]]會比[[醇類|醇類]]物質高。

== 化學性质 ==
[[File:Carboxylic_acid_dimers.png|thumb]])]]
因為[[羧基|羧基]]由能產生氫鍵的[[羥基|羥基]]和[[羰基|羰基]]組成，所以所有羧酸都具有[[極性|極性]]。 存在於非極性物質中的羧酸常以[[二聚體|二聚體]]的形式存在。碳數較少的羧酸（C<sub>1</sub>～C<sub>5</sub>）較能溶於水，而隨著碳數增加就越難溶於水，但羧基所接的烷基碳數較多的羧酸卻比較能溶於極性較小的[[醚|醚]]類跟[[醇|醇]]類物質。<ref name="M&B">R.T. Morrison, R.N. Boyd. ''Organic Chemistry'', 6th Ed. (1992) ISBN 0-13-643669-2.</ref>

從結構中可以預期羧酸是[[極性|極性]]的，它可以跟同類或其他分子生成[[氫鍵|氫鍵]]，所以羧酸跟醇有相似的[[溶解性|溶解性]]。它們都可溶於非極性有機溶劑如[[苯|苯]]、[[醚|醚]]，也可溶於極性有機溶劑如[[二氯甲烷|二氯甲烷]]、[[乙酸乙酯|乙酸乙酯]]等。[[甲酸|甲酸]]、[[乙酸|乙酸]]、[[丙酸|丙酸]]、[[丁酸|丁酸]]可溶於水。

氣態的羧酸常以[[二聚體|二聚體]]形式存在，所以與相若[[分子量|分子量]]的其他化合物相比，它有異常高的[[沸點|沸點]]。例如[[丁酸|丁酸]]的沸點為164℃，[[丁醇|丁醇]]的沸點則只有118℃。

常温下，[[甲酸|甲酸]]、[[乙酸|乙酸]]、[[丙酸|丙酸]]是具有刺激性气味的液体，[[丁酸|丁酸]]至[[壬酸|壬酸]]是有不舒服气味的油状液体，10个碳以上的一元羧酸是无味的蜡状固体，二元羧酸和[[芳香族|芳香族]]羧酸是结晶态的固体。

羧酸分子可能因分子間氫鍵而產生偶合，導致所測得之分子量為原本的１～２倍。

== 合成方法 ==
* [[醇|醇]]或[[醛|醛]]的氧化作用：在酸性環境中將[[伯醇|伯醇]]或醛跟強氧化劑如[[高錳酸鉀|高錳酸鉀]]或[[亞氯酸鈉|亞氯酸鈉]]加熱：
** RCH<sub>2</sub>OH + [O] → RCOOH
** RCHO + [O] → RCOOH

* [[酯|酯]]或[[腈|腈]]的水解
** RCOOR' + H<sub>2</sub>O + [H<sup>+</sup>] → RCOOH + R'OH
** RCN + H<sub>2</sub>O + [H<sup>+</sup>] → RCOOH + NH<sub>3</sub>

* [[格氏試劑|格氏試劑]]
** R-X + Mg → R-MgX
** R-MgX + CO<sub>2</sub> → R-COOMgX
** R-COOMgX + [H<sup>+</sup>] → RCOOH

== 反應 ==

⑴　酸性氫氧鍵的反應

　　羧酸是酸的一種，它會與鹼反應生成鹽，例如乙酸會與[[氫氧化鈉|氫氧化鈉]]反應生成[[乙酸鈉|乙酸鈉]]：

　　　　CH<sub>3</sub>COOH + NaOH → CH<sub>3</sub>COONa + H<sub>2</sub>O

　　羧酸根不是一個強力的[[親核試劑|親核試劑]]，所以它不常用來作[[親核取代反應|親核取代反應]]。不過，當遇上強力的[[親電試劑|親電試劑]]，羧酸根亦會有反應：

　　　　RCOOH + CH<sub>2</sub>N<sub>2</sub> → RCOOCH<sub>3</sub> + N<sub>2</sub>
