'''康定市'''（{{bo|z=དར་རྩེ་མདོ།|w=dar-rtse-mdo|p=Darzêdo}}），古名'''打箭炉'''（简称{{bo|z=དར་མདོ།|w=dar mdo|p=Tando}}{{可疑 |康定市的藏语名？ |reason=藏语名是不是搞反了? |date=2018年1月}}），位于[[四川|四川]][[甘孜藏族自治州|甘孜藏族自治州]]东部，素以“藏卫通衢”、“川藏要冲”著称，是全州政治、经济、文化、商贸中心和交通枢纽，也是州首府所在地。康定因民歌《[[康定情歌_(歌曲)|康定情歌]]》而闻名。風景區[[贡嘎山|贡嘎山]]位于康定、[[泸定|泸定]]、[[九龙|九龙]]三县市。

== 歷史 ==
康定，古为[[羌族|羌族]]之地，[[三国|三国]][[蜀汉|蜀汉]]称“打箭炉”。历史上，康定是汉藏贸易物资集散中心，内地通往[[西藏|西藏]]的门户，川藏“[[茶马互市|茶马互市]]”的经济交往中心。

[[唐|唐]][[宋|宋]]属吐蕃。[[元代|元代]]设置有以土头为土官的长河西千户、长河西招讨司。[[明|明]][[永乐|永乐]]五年（1407年）正式敕封[[明正土司|“长河西鱼通宁远宣慰”司]]，世袭其职。[[崇祯|崇祯]]十二年（1639年）[[固始汗|固始汗]]部于土司境设置营官，形成营官、土司双重统治。[[清|清]][[康熙|康熙]]五年（1666年）重新认定土司。[[雍正|雍正]]八年（1730年），置[[打箭爐廳|打箭爐廳]]。[[光緒|光緒]]三十年（1904年）升打箭爐廳為[[康定府|康定府]]，此為“康定”定名之始。[[中華民國|中華民國]]成立後廢府為縣，置康定縣。1939年[[西康省|西康省]]成立，康定縣為[[省會|省會]]。2015年3月16日，[[中华人民共和国国务院|中华人民共和国国务院]]批准撤销康定县，设立县级康定市<ref>{{cite book|title=四川省人民政府关于同意撤销康定县设立县级康定市的批复（川府函〔2015〕43号）|publisher=四川省人民政府|date=2015-03-16}}</ref>。

== 地理 ==
康定市属于[[高地气候|高地气候]]，冬季寒冷，夏季凉爽，受[[季风|季风]]影响夏季多雨，冬季干燥。

== 语言 ==
康定是一个藏、汉、羌、回、彝等多民族杂居市。当地的藏族使用[[藏语|藏语]]的[[康方言|康方言]]和[[安多方言|安多方言]]、[[木雅语|木雅语]]以及[[贵琼语|贵琼语]]。
