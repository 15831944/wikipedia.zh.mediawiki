'''康大川'''（{{Bd|1915年|3=2004年}}），[[原名|原名]]'''康天順'''，[[綽號|綽號]]'''老康'''，是一位出身臺灣[[苗栗|苗栗]]的[[中華人民共和國|中華人民共和國]]官員、編輯及[[翻譯|翻譯]]從業人員。他是日文雜誌《[[人民中国|人民中國]]》的共同[[創辦人|創辦人]]和資深成員之一。

== 生平 ==
1915年，康大川出生於苗栗。後來，他前往東京{{Link-ja|錦城學園高等學校|錦城学園高等学校}}就讀。

1938年，他畢業於[[早稻田大学|早稻田大学]]商科，並前往中國大陸參與於[[抗日戰爭|抗日戰爭]]；隨後，他曾在[[郭沫若|郭沫若]]領導下於[[郭沫若故居和國民政府軍事委員會政治部三廳舊址|國民政府軍事委員會政治部第三廳]]從事對敵[[宣傳|宣傳]]和[[戰俘|戰俘]]管理，並與被俘日軍一同起居，又讓其從事於製作[[玩具|玩具]]及[[編輯|編輯]]刊物等活動。

1948年，他加入[[中国共产党|中国共产党]]，並在1949年11月奉指示前往[[北京|北京]]任職。1952年，他在[[外文出版社|外文出版社]]負責籌辦《人民中國》日文版，於是前往[[瀋陽|瀋陽]]尋求已停辦[[報紙|報紙]]《民主新聞》的成員池田亮一、菅沼不二男等及[[排版|檢字]]工人與日文[[字模|字模]]、[[印刷|印刷]]器材等，又到[[大連|大連]]借調[[劉德有|劉德有]]及安淑渠等人，再於北京招募返國留學日本學生。在經過兩次試刊後，該刊正式開始於1953年發行於[[日本#二戰以後|日本]]地區。<ref name=":0">{{Cite web|url=http://big5.china.com.cn/book/zhuanti/qkjc/txt/2009-10/12/content_18688465.htm|title=康大川：《人民中國》的創刊人_中國網|accessdate=2018-01-13|work=big5.china.com.cn}}</ref><ref>{{Cite web|url=http://www.peoplechina.com.cn/wenhua/2009-08/31/content_214271.htm|title=『人民中国』の生みの親、康大川_人民中国|accessdate=2018-01-13|work=www.peoplechina.com.cn}}</ref>

== 代表作 ==
* 《人民中國》
* 《[[毛泽东选集|毛泽东选集]]》（日文翻譯定稿）

== 軼事 ==
* 在擔任《人民中國》編輯工作時，康大川習慣於每日[[下班|下班]]後攜帶大量[[日本報紙|日本報刊]]回家閱讀，也習慣仔細閱讀讀者來信及友人意見，以不斷改進刊物內容，增進[[中國大陸|中國大陸]]與[[日本人|日本地區民眾]]之間的理解與聯繫。<ref name=":0" />

== 評論 ==
