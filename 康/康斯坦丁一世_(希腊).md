希腊人国王'''君士坦丁一世'''（[[希腊语|希腊文]]：'''{{lang|el|Κωνσταντίνος A', Βασιλεύς των Ελλήνων}}'''，拉丁文转写：''{{lang|la|Konstantínos A', Vasileús ton Ellínon}}''，{{bd|1868年|8月2日|1923年|1月11日|catIdx=K}}），1913年至1917年、1920年至1922年在位。

== 家庭出身与早年生活 ==
康斯坦丁一世是希腊[[格呂克斯堡王朝|石勒苏益格-荷尔斯泰因-宗德堡-格吕克斯堡王朝]]的第一位君主[[乔治一世_(希腊)|乔治一世]]与王后俄罗斯女大公奥尔嘉·康斯坦丁诺芙娜的长子，1868年8月2日生于希腊首都[[雅典|雅典]]。之所以取名康斯坦丁，一方面与他的俄国外祖父[[康斯坦丁·尼古拉耶维奇|康斯坦丁·尼古拉耶维奇大公爵]]同名，更重要的是，希腊人民希望有朝一日他能够统治与他同名的故都[[君士坦丁堡|君士坦丁堡]]，实现“伟大理想”（{{lang|el|Μεγάλη Ιδέα}}）。

他一-{出}-生便被确立为希腊王位的继承人，称“[[斯巴达|斯巴达]]公爵”（{{lang|el|Δουκός της Σπάρτης}}）。年轻时就读于[[希腊军事学院|希腊军事学院]]。1886年他离开祖国来到[[德国|德国]]，在[[海德堡大学|海德堡大学]]深造，在普鲁士军队接受军事训练。1889年，康斯坦丁与[[德国君主列表|德意志皇帝]][[威廉二世_(德国)|威廉二世]]的妹妹[[索菲公主_(普鲁士)|索菲公主]]结婚。

康斯坦丁在王储时期曾担任[[1896年夏季奥林匹克运动会|1896年夏季奥林匹克运动会]]组委会主席。[[1897年希土战争|1897年希土战争]]时，康斯坦丁王储带领的希腊军队惨败，使希腊不得不割地赔款。1909年的军事联盟政变时，康斯坦丁王储被解除了军队的所有职务。

== 就任国王 ==


1912年，康斯坦丁王储重新被授予希腊陆军总司令，参加了[[第一次巴尔干战争|第一次巴尔干战争]]。在战争中他带领希腊色萨利军在萨郎塔波罗和吉安尼察的战役中屡战屡胜，并于11月9日占领[[塞萨洛尼基|塞萨洛尼基]]，取得了光辉的胜利。1913年的比赞尼战役中，康斯坦丁王储带领的色萨利军再次获得胜利，歼灭了奥斯曼帝国军队，决定了第一次巴尔干战争中巴尔干联盟的胜利。

然而在1913年3月15日，国王乔治一世在塞萨洛尼基遇刺身亡，康斯坦丁在悲剧的气氛中继承王位，称'''希腊人国王君士坦丁一世'''。许多的人愿意称他君士坦丁十二世，认为他是[[羅馬帝国|羅馬帝国]]末代皇帝[[君士坦丁十一世|君士坦丁十一世]]的继承者。

同年5月，奥斯曼帝国宣布投降。希腊通过战争得到了约阿尼纳、塞萨洛尼基和克里特。因为许多方面的矛盾，特别是[[馬其頓|马其顿]]的归属问题，希腊、塞尔维亚与罗马尼亚联合起来反对保加利亚。1913年6月19日[[第二次巴尔干战争|第二次巴尔干战争]]爆发。7月便以保加利亚的失败告终，马其顿一分为三，希腊确定了对[[愛琴海馬其頓|愛琴海馬其頓]]的所有权。

== 一战期间 ==
1914年，[[第一次世界大战|第一次世界大战]]爆发。根据1913年的希塞条约，发生情况时希腊就得援助塞尔维亚。但这次进攻塞尔维亚的不是保加利亚，而是[[奥匈帝国|奧匈帝國]]。首相[[埃莱夫塞里奥斯·韦尼泽洛斯|埃莱夫塞里奥斯·韦尼泽洛斯]]向塞尔维亚保证如果保加利亚进攻它，希腊将向保加利亚宣战。而康斯坦丁一世作为德国皇帝的妹婿，反对自己的国家加入反对德国的战争，赞成[[中立|中立]]。1914年8月8日，康斯坦丁一世被德国的[[威廉二世_(德国)|威廉二世]]皇帝授予[[德国元帅|德意志帝国元帅]]荣誉。

1914年11月，奥斯曼帝国加入[[同盟國_(第一次世界大戰)|同盟国]]。韦尼泽洛斯答应[[協約國|协约国]]派一支军队进攻[[達達尼爾海峽|达达尼尔海峡]]，遭到了总参谋长[[约安尼斯·梅塔克萨斯|约安尼斯·梅塔克萨斯]]（{{lang|el|Ιωάννης Μεταξάς}}）和国王的反对。国王与韦尼泽洛斯决裂。韦尼泽洛斯辞去首相的职务，国王选择了亲德的[[季米特里奥斯·古纳里斯|季米特里奥斯·古纳里斯]]（{{lang|el|Δημήτριος Γούναρης}}）组建政府。随着1915年大选韦尼泽洛斯党的获胜，韦尼泽洛斯于8月再任首相。

1915年9月，保加利亚宣布动员令，并于10月4日加入同盟国参战。康斯坦丁一世于9月亦签署了动员令，但拒绝希腊军队采取任何行动。10月7日，协约国军队在塞萨洛尼基登陆，国王罢免了韦尼泽洛斯首相。[[亚历山德罗斯·扎伊米斯|亚历山德罗斯·扎伊米斯]]（{{lang|el|Αλέξανδρος Ζαΐμης}}）担任了一个月的首相职务后，国王解散议会并指派[[斯特凡诺斯·斯库鲁迪斯|斯特凡诺斯·斯库鲁迪斯]]（{{lang|el|Στέφανος Σκουλούδης}}）为新任首相。

1916年4月，为了抗议协约国帮助战败的塞尔维亚军队在塞萨洛尼基避难，以及意大利进入[[北伊庇鲁斯|北伊庇鲁斯]]，康斯坦丁一世宣布-{希}-腊合并北伊庇鲁斯。6月，协约国向希腊发出[[最後通牒|最后通牒]]，要求解散希腊军队，斯库鲁迪斯因此下台，扎伊米斯就任首相。大选一再推迟，韦尼泽洛斯与康斯坦丁一世彻底决裂。

== 第一次被废黜与復位 ==
1916年9月19日，埃莱夫塞里奥斯·韦尼泽洛斯与海军总司令[[保罗·孔杜里奥提斯|保罗·孔杜里奥提斯]]（{{lang|el|Παύλος Κουντουριώτης}}）在克里特成立敌对政府。10月，韦尼泽洛斯北上塞萨洛尼基。韦尼泽洛斯的敌对政府控制了希腊北部、克里特和爱琴海诸岛。韦尼泽洛斯的支持者纷纷在塞萨洛尼基汇合。俄罗斯与意大利反对这个政府：俄罗斯反对他是因为康斯坦丁一世的母亲是俄罗斯的公主，意大利人把韦尼泽洛斯当作他们对[[安那托利亞|小亚细亚]]野心的潜在敌人。英法担心引起希腊内战，没有承认[[韦尼泽洛斯政府|韦尼泽洛斯政府]]。

英法趁此机会对康斯坦丁一世及其政府施加压力，但国内的亲德派发动了反协约国宣传。法国上将[[毛瑞斯·萨莱伊|毛瑞斯·萨莱伊]]派出军队登陆，并向雅典逼近。12月，双方军队发生交火。

因为此次事件，雅典爆发了反韦尼泽洛斯的运动。为了报复，协约国于12月19日承认韦尼泽洛斯政府，萨莱伊的海军封锁了希腊。协约国承认了一个希腊、两个政府。

1917年5月，塞萨洛尼基举行了支持韦尼泽洛斯的游行，韦尼泽洛斯的支持者们要求协约国废黜康斯坦丁一世。6月10日，协约国要求康斯坦丁一世逊位。康斯坦丁一世同意逊位，并让长子[[乔治二世_(希腊)|乔治王储]]离开希腊，因为乔治王储曾在德国参军，协约国不同意由他继承王位。次子[[亚历山大一世_(希腊)|亚历山大]]将继承王位。康斯坦丁一世于6月11日正式退位，流亡[[瑞士|瑞士]]。

1920年10月25日，亚历山大国王因被猴子咬伤而引起的[[敗血症|敗血症]]而去世。韦尼泽洛斯劝说康斯坦丁一世的第三子[[保罗一世_(希腊)|保罗]]继承王位但遭拒绝。11月，韦尼泽洛斯的自由党在大选中惨败，韦尼泽洛斯下台。在11月的[[公民投票|公投]]中，支持康斯坦丁一世復位的投票占总票数的98.97%。康斯坦丁一世于12月19日復位。

== 第二次逊位与去世 ==
康斯坦丁一世復位后重蹈韦尼泽洛斯的覆辙，将希腊推向战争的深渊。当时的希腊正与即将灭亡的奥斯曼帝国进行着[[希土战争_(1919年-1922年)|第二次希土战争]]，战争已深入到[[安那托利亞|小亚细亚半岛]]，康斯坦丁一世命令希军继续深入，准备进攻正在建设的新首都[[安卡拉|安卡拉]]。随着1921年8月[[萨卡里亚战役|萨卡-{里}-亚战役]]的失败，希军在小亚细亚已丧失得胜的可能。

1922年3月，列强在[[巴黎|巴黎]]开会提出停火建议，希腊接受了停火。但[[穆斯塔法·凯末尔·阿塔蒂尔克|穆斯塔法·凯末尔·阿塔土克]]要求希军撤出小亚细亚，列强拒绝了他的要求。于是土耳其军队于8月向[[伊兹密尔|士麦那]]发起总攻，并于9月9日占领士麦那。9月13日起，士麦那燃起了持续四天的[[士麦那火灾|大火]]，城内的希腊人或选择逃跑，没逃掉的遭到了惨绝人寰的大屠杀。

希腊政府因此解散军队并辞职。康斯坦丁一世也于9月27日宣布退位，流亡意大利[[西西里岛|西西里岛]]。长子乔治继位，称乔治二世。

四个月后的1923年1月11日，康斯坦丁一世在西西里的[[巴勒莫|巴勒莫]]去世，享年54岁。康斯坦丁一世的葬礼在[[佛罗伦萨|佛罗伦萨]]的东正教堂举行。1936年君主制复辟后，康斯坦丁一世的遗体重新安葬在塔托伊的王室墓园。

== 大眾形象和遺產 ==
在20世紀上半撕裂希臘的兩大派勢力（保王派和自由派）中，康斯坦丁一世是保王派最得民心之領袖（代表保守的右派），如同[[韋尼澤洛斯|韋尼澤洛斯]]是自由派（代表偏共和左派）最得民心之領袖那般，康斯坦丁和韋尼兩人成為20世紀上半最得民心的兩位希臘人；然而如今大多希臘人擁抱的是韋尼澤洛斯的自由價值與共和遺產，卻遺忘（或忽視）了康斯坦丁曾經被推崇的偉大，因為康斯坦丁代表的希臘民族主義的核心──[[東正教|東正教]]之神權和[[拜占庭帝國|拜占庭帝國]]之繼承人，很難經受20世紀下半的[[現代化|現代化]]考驗。

在流行文化中，保王派的口頭禪──「麵包、橄欖和康斯坦丁國王」（'''psomi，elia ke Kotso Vasilià'''）仍是極受愛戴的流行語；但另一方面韋尼澤洛斯派則喜歡用「娘們康絲坦」（'''psorokòstena'''）來諷刺國王，極盡侮辱之能事。

== 婚姻与子女 ==
[[File:Queen_Helen_of_Romania.jpg|thumb]]
1889年10月27日，康斯坦丁一世在雅典表姨与德意志皇帝兼[[普鲁士国王|普鲁士国王]][[腓特烈三世_(德国)|腓特烈三世]]的第三女[[索菲公主_(普鲁士)|索菲·多萝泰娅·乌尔丽克·艾丽丝]]（德文名：{{lang|de|Sophie Dorothea Ulrike Alice}}，希腊文名：{{lang|el|Σοφία της Πρωσίας}}，1870年6月14日—1932年1月13日）结婚，二人婚后有三子三女：
* 长子'''[[乔治二世_(希腊)|乔治]]'''（'''{{lang|el|Γεώργιος}}'''，1890年7月19日—1947年4月1日），1922年康斯坦丁一世第二次逊位后即位，1925年希腊实行共和制被迫退位，1935年复位；
* 次子'''[[亚历山大一世_(希腊)|亚历山大]]'''（'''{{lang|el|Αλέξανδρος}}'''，1893年8月1日—1920年10月25日），1917年康斯坦丁一世第一次逊位后即位，1920年被猴子咬伤致死；
* 长女'''[[艾莱妮公主_(希腊)|艾莱妮]]'''（'''{{lang|el|Ελένη}}'''，1896年5月3日—1982年11月28日），1921年与[[罗马尼亚君主|罗马尼亚国王]][[卡罗尔二世_(罗马尼亚)|卡罗尔二世]]结婚，有一子，即罗马尼亚国王[[米哈伊一世_(罗马尼亚)|米哈伊一世]]；
* 三子'''[[保罗一世_(希腊)|保罗]]'''（'''{{lang|el|Παύλος}}'''，1901年12月14日—1964年3月6日），1947年继承长兄成为国王；
* 次女'''[[艾琳妮|艾琳妮]]'''（'''{{lang|el|Ειρήνη}}'''，1904年2月13日—1974年4月15日），1939年在[[佛罗伦萨|佛罗伦萨]]与[[意大利|意大利]]王子[[斯波莱托公爵|斯波莱托公爵]][[托米斯拉夫二世|艾蒙内]]结婚，后成为挂名的[[克罗地亚|克罗地亚]]王后；
* 三女'''[[爱卡特琳妮|爱卡特琳妮]]'''（'''{{lang|el|Αικατερίνη}}'''，1913年5月4日—2007年10月2日），1947年与英国人理查德·坎贝尔·布兰德拉姆结婚，自称凯瑟琳·布兰德拉姆女爵士，有一子。
