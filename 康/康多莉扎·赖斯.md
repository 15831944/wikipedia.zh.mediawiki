'''康多莉扎·“康迪”·赖斯'''（{{lang-en|'''Condoleezza "Condi" Rice'''}}，{{bd|1954年|11月14日}}），美国[[政治家|政治家]]，前[[美国国务院|美国国务院]][[美国国务卿|國務卿]]。生於[[美國|美國]][[亚拉巴马州|亚拉巴马州]][[伯明翰_(亚拉巴马州)|伯明翰]]，她是美國歷史上就任此職的第一位女性[[非裔美国人|非裔美国人]]，亦是就任此職的第二位非裔美国人（继[[克林·鲍威尔|克林·鲍威尔]]之后）以及第二位女性（继[[马德琳·奥尔布赖特|歐布萊特]]之后）。

== 童年 ==
赖斯出生於[[美国|美国]][[亚拉巴马州|亚拉巴马州]][[伯明翰_(亚拉巴马州)|伯明翰]]，是家中的独生女。父亲[[小约翰·韦斯利·赖斯|小约翰·韦斯利·赖斯]]是西敏教堂长老教会的牧师，母亲[[安吉利娜·赖斯|安吉利娜·赖斯]]是一名音乐教师。曾任[[哥伦比亚大学|哥伦比亚大学]]新闻研究所所长的尼古拉斯·莱曼刊在《纽约人》中的文章写道，“伯明翰最知名的富有黑人家庭是从事保险业的加斯顿家族。排名第二的是[[阿尔玛·鲍威尔|阿尔玛·鲍威尔]]一家；她的父亲和叔叔曾经是城中两所黑人中学的校长。”<ref>[http://www.newyorker.com/fact/content/?021014fa_fact3 Condoleezza Rice - Academic Kids]</ref> （阿尔玛·鲍威尔嫁给了柯林·鲍威尔。）1967年，由於赖斯的父亲接受[[丹佛大学|丹佛大学]]所提供的一项管理职务，一家遷居[[丹佛|丹佛]]。她的名字是由意大利语中表达“甜美的演奏”的单词“con dolcezza”变化而来。<ref>[http://www.wnyc.org/legacy/shows/madaboutmusic/madabout_transcript090701.html The Hair-raising Truth - Condoleezza Rice] {{webarchive|url=https://web.archive.org/web/20050403220428/http://www.wnyc.org/legacy/shows/madaboutmusic/madabout_transcript090701.html |date=2005-04-03 }}</ref>

== 求學 ==
賴斯首先於[[丹佛大學|丹佛大學]]取得政治学学士学位，后再取得[[圣母大学|圣母大学]]政治学硕士学位及丹佛大学国际研究學院博士学位。她在丹佛大學的啟蒙老师是美国前国务卿馬德琳·歐布萊特的父亲──自[[捷克斯洛伐克|捷克斯洛伐克]]流亡至美国的外交官－－科贝尔教授。除了母语[[英语|英语]]，赖斯还会讲[[俄语|俄语]]、[[法语|法语]]和[[西班牙语|西班牙语]]。

== 政治生涯 ==
在[[美国总统|美国总统]][[乔治·沃克·布什|乔治·沃克·布什]]的第一任期间任职[[国家安全顾问|国家安全顾问]]。她是擔任此職的第二位非裔美国人和第一位女性。

2004年11月16日，布什提名赖斯，接替[[克林·鲍威尔|克林·鲍威尔]]成为国务卿。2005年1月26日，[[美国参议院|美国参议院]]以85票对13票确认她的提名，而她在当日尔后宣誓就职。赖斯的主張向來是以軍事手段解決對外關係的主要手段，作風跟[[唐纳德·拉姆斯菲尔德|倫斯斐]]等官員同屬鷹派。

2006年[[中東|中東]]的[[2006年以黎衝突|黎巴嫩戰爭]]引發賴斯與總統布什的意見不同<ref>[http://world.people.com.cn/GB/1029/42355/4663768.html]</ref>。在布什担任总统期间，曾公开表示过对国务卿赖斯的不满。 
她表示美國長久以來的立場未變，根本利益在於一個和平與穩定的兩岸關係，美國反對任何一方片面改變現狀。萊斯表示「中美有許多歧見須面對，要求中方立即停止南海造陸」，同時她也表示「希望中美願意就氣候變遷的問題共同想辦法解決」。

== 教授生涯 ==
退出政坛后，赖斯担任了[[斯坦福大学|斯坦福大学]]教授一职。

== 私人生活 ==
賴斯未婚，無子女。
