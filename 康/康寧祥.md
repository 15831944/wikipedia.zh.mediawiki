'''康寧祥'''（{{bd|1938年|11月16日|||catIdx=K康}}），[[台北市|臺北市]][[萬華區|萬華區]]人，台灣政治人物、民主運動參與者。曾任[[臺北市議員|臺北市議員]]、[[立法委員|立法委員]]、國家安全會議秘書長。為台灣[[黨外運動|黨外運動]]要角。是自1970年代之後[[台灣|臺灣]]新一代崛起的本土[[政治人物|政治人物]]<ref>黃文博總編輯，《[[s:南瀛探索/第九章/第一節|南瀛探索]]，第九章，第一節》：新一代崛起的本土政治人物（黃信介、康寧祥等）</ref>。康寧祥也是少數與[[蔣經國|蔣經國]]有過來往與交情的臺籍政治人物之一。

== 早年經歷 ==
康寧祥祖籍[[桃園縣|桃園縣]][[大園鄉|大園鄉]]，父親康振福在十多歲時到[[臺北|臺北]]討生活，在日本人開的「藤田製果」當學徒，學成後自己開店，取名「福田製餅」，由於生意不錯，隨後大員康家的伯公、祖父、叔叔們也接著過來。{{r|康|page=16}}住在[[堀江町_(台北市)|堀江町]]。戰後康振福曾被國民黨吸收，當選過[[雙園區|雙園區]]一屆區民代表。{{r|康|page=30}}母親辛寶玉在二戰期間曾為艋舺後火車站地區婦女防護團的負責人，在[[日本投降|日本投降]]那年（1945年）過世，留下康寧祥及兩個弟弟、兩個妹妹，和大他兩歲的養姐。{{r|康|page=30}}康寧祥大舅舅在二戰期間曾被抓去[[南洋|南洋]]當[[軍伕|軍伕]]，沒被美國砲彈打死，卻死在「祖國」的槍下，[[二二八事件|二二八事件]]發生後的清鄉，被蔣介石的部隊抓去[[馬場町|馬場町]][[新店溪|新店溪]]旁槍斃。{{r|康|page=31}}

康寧祥在雙園國小畢業後，由於沒考上臺北市的初中，到[[樹林中學|樹林中學]]就讀，曾受教於音樂家[[許石|許石]]，也曾受教於郭衣洞（[[柏楊|柏楊]]）。{{r|康|page=32-33}}初中畢業後，由於沒考上高中，進入[[延平補校|延平補校]]就讀，{{r|康|page=34}}與[[郭國基|郭國基]]之子郭蓋世成為同學，康寧祥與郭蓋世互有往來，也時常前往郭國基家，聽他分析以往奮鬥過程。1959年康寧祥延平補校高二時，適逢郭國基競選省議員連任，在臺北參選。康寧祥亦為其助選發傳單，崛江町父老也動員為郭國基助選。{{r|康|page=38-43}}1957年大學聯考，康寧祥考上臺北法商學院（今[[國立臺北大學|國立臺北大學]]）。康寧祥為大學畢業需繳交畢業論文的最後一屆，康寧祥的畢業論文為《英國政府與政黨政治》，在大學四年間，康寧祥也積極閱讀《[[自由中國|自由中國]]》、《民主潮》、《時與潮》、《[[文星雜誌|文星雜誌]]》等雜誌，以及[[吳濁流|吳濁流]]的《台灣文藝》。{{r|康|page=44}}大學畢業後，康寧祥於屏東空軍機場服[[預官|預官]]役，在[[屏東機場|屏東機場]]管制中心任副管制長。{{r|康|page=157}}期間繼母過世。{{r|康|page=45}}退伍後，康寧祥曾參加電台播音員考試，到面試時未過，就待在家中製餅。期間康寧祥曾考上中油當加油站員工，月薪七百多。{{r|康|page=46}}

== 市議員生涯 ==
民國五十八年(1969年)，[[臺北市|臺北市]]舉行改制[[直轄市|直轄市]]後的第一屆[[台北市議員|臺北市議員]]選舉，康寧祥以無黨籍身分參選第五選區（龍山、雙園、古亭、中正），11月當選市議員。當選後不久，康寧祥即幫[[黃信介|黃信介]]助選[[1969年中華民國立法委員選舉|國民黨政府遷台後首屆增額立法委員選舉]]，並幫忙其演講。{{r|康|page=62}}當時臺北應選四席，國民黨提名兩席、重點支持[[洪炎秋|洪炎秋]]，剩下一席黨外遇上黃信介與[[張詩經|張詩經]]爭取。{{r|康|page=64}}最後黃信介成功當選立法委員。

在市議員時期，康寧祥認識了曾任[[林獻堂|林獻堂]]秘書的[[葉榮鐘|葉榮鐘]]，透過葉蓉鐘介紹認識日治時期的社會運動要角[[陳逢源|陳逢源]]、[[巫永福|巫永福]]、[[黃得時|黃得時]]等人。在康寧祥當上市議員後受特務監視時，[[王詩琅|王詩琅]]的書房也成了社會運動前輩與康寧祥傳授[[台灣史|台灣史]]的地方。{{r|康|page=76}}也曾得到《臺灣總督府警察沿革志第二篇》，由王詩琅譯成中文。{{r|康|page=78-79}}1969年，康寧祥受美國大使館二等秘書傅嵐（Jerry A.Flower）代表[[美國|美國]]邀請訪美，然而申請護照卻遭到擱置半年。直到康寧祥受行政院副院長蔣經國召見，蔣經國問他「聽說你要到美國去？」康寧祥回答申請護照都沒下文，[[蔣經國|蔣經國]]命令國民黨組工會主任[[陳建中|陳建中]]給他護照。三天後康寧祥就收到公務護照，由國民黨中央黨部專程派人送到他家。{{r|康|page=80-81}}1970年6月，康寧祥訪美。在美國新聞處安排下與[[葛超智|柯爾·喬治]]見面，在[[華府|華府]]與[[費正清|費正清]]見面，並在[[休士頓|休士頓]]認識[[蔡斯_(美國陸軍將領)|蔡斯]]將軍。{{r|康|page=88-89}}8月回臺。{{r|康|page=90}}

== 立法委員生涯 ==
=== 首屆立法委員時期 ===
民國六十一年(1972年)，因應[[蔣經國|蔣經國]]內閣的「吹台青」政策，國民黨政府舉行[[1972年中華民國立法委員選舉|增額立法委員選舉]]，臺北市分到五席立委名額。國民黨提名四席，包括軍方支持的[[李志鵬|李志鵬]]、上海幫生意人，也是[[明星花露水|明星花露水]]老闆[[周文璣|周文璣]]、馬偕醫院董事長[[李東輝|李東輝]]、受到新光吳家支持的[[丁守真|丁守真]]，剩下一席則放出風聲要給國泰蔡家代表蔡萬財（後改名[[蔡萬才|蔡萬才]]）。康寧祥於8月宣布參選，並說服原先不支持的黃信介，推出其弟[[黃天福|黃天福]]競選國大代表，成聯合競選。{{r|康|page=109-110}}當時臺北市在五年前剛併入士林、北投、景美、木柵、內湖、南港，選區遼闊。康寧祥以臺灣人意識覺醒為政見主題，{{r|康|page=111}}並抨擊國民黨限播臺語歌曲、臺語連續劇不准用臺語播出的歧視語言政策，{{r|康|page=112-113}}也獲得民眾的支持。選前康寧祥更受到黑函攻擊，稱他因案準備退選。日後康寧祥獲前國民黨立委[[雷渝齊|雷渝齊]]向他證實臺北市黨部有一特殊小組，專門針對康寧祥造謠、抹黑。{{r|康|page=114-115}}最後康寧祥以第二高票當選立委，只輸給軍眷票支持的李志鵬，並擠下丁守真。1973年2月1日報到。由於康寧祥的口才與辯才是頗能吸引民眾的目光，而且康寧祥與資深立委[[梁肅戎|梁肅戎]]、[[陳蒼正|陳蒼正]]等人有都有不錯的交情。

在立法委員任內，康寧祥於1973年4月16日初試啼聲質詢行政院長[[蔣經國|蔣經國]]，針對違憲徵收的電話、電報臨時捐。{{r|康|page=131}}1973年10月，對省縣自治提出法制化，未獲蔣經國回應。{{r|康|page=132-134}}在1975年質詢時，康寧祥首度將[[台灣抗日運動|台灣抗日運動]]史搬上國會殿堂，日後1975年康寧祥連任後，蔣經國曾數度邀請他到行政院辦公室茶敘，表示康寧祥質詢的台灣先民抗日運動史他聽都未聽過，聽得非常感動。{{r|康|page=144-145}}並對國防預算及中央銀行內黑官氾濫提出質詢。{{r|康|page=156-161}}1974年9月，康寧祥應邀二次訪美。{{r|康|page=168}}在1974年的《[[時代雜誌|時代雜誌]]》中，康寧祥被列入150名「全球未來領袖」之一。

1975年，康寧祥與[[黃信介|黃信介]]、[[張俊宏|張俊宏]]等人合作出版《[[台灣政論|臺灣政論]]》[[雜誌|雜誌]]，作為[[黨外運動|黨外運動]]的傳播工具。年底，《[[台灣政論|臺灣政論]]》遭到國民黨當局停刊。

民國六十四年(1975年)[[1975年中華民國立法委員選舉|第二次增額立法委員選舉]]中，康寧祥角逐連任，適逢康寧祥父親在競選活動前一週過世，康寧祥忙著處理其後事，最後以吊車尾連任。{{r|康|page=202}}當時蔣經國在立法院說過「支持現任立委連任」，因此對於無黨籍人士相對客氣，如康寧祥所在的臺北市選區、[[黃順興|黃順興]]所在的臺灣省第三選區、[[許世賢|許世賢]]所在的臺灣省第四選區，國民黨都採取不足額提名，在競選活動上不像過去發動媒體醜化黨外，或者任國民黨籍候選人大肆[[買票|買票]]。{{r|康|page=202}}當時國民黨臺北市黨部主委[[易勁秋|易勁秋]]曾兩度到康寧祥家中拜訪，第一次純粹禮貌性，第二次就問康寧祥「需不需要什麼幫忙？」康寧祥對他說「只要國民黨不要到萬華、龍山亂買票，我就感激不盡了！」。{{r|康|page=202}}有人認為由於康寧祥都主張「議會路線」而非「街頭路線」，所以都獲得連任。這是奇怪的說法，因為街頭路線要一直要到1980年代末期才真正出現，至於1979年美麗島雜誌社集團可以說才可能要開始走上街頭就全部被逮捕了，而1983年後的批康也只是為1980年代末的街頭路線作䁔身而己，街頭運動還沒有真正展開。換句話說說在1983年之前台灣根本沒有什麼街頭和議會路線的區别的。因此康以和街頭對比的議會路線而當選，是不知所云的看法。

民國67年(1978年)12月15日，美國與[[中華人民共和國|中華人民共和國]]簽署《[[中美建交公報|中美建交公報]]》。同年12月16日，美國總統卡特宣自1979年1月1日起，承認中華人民共和國，《中美協防條約》（即《中美共同防禦條約》）亦於1979年12月31日終止。同日，時任總統[[蔣經國|蔣經國]]就美中建交發表聲明，指出美國背信毀約；強調絕不與中國共產黨談判。當時，黨外人士如[[黃信介|黃信介]]、[[許信良|許信良]]等人都提議組黨，但康寧祥認為時候未到，導致康寧祥與其他黨外領袖分道揚鑣。

民國68年(1979年)6月，當時擔任立法委員的康寧祥又出版《[[八十年代_(雜誌)|八十年代]]》雜誌，並擔任《[[八十年代_(雜誌)|八十年代]]》雜誌的發行人兼社長。

民國68年(1979年)12月10日的國際人權日在臺灣高雄市發生的[[美麗島事件|美麗島事件]]，康寧祥並沒有參與。且康寧祥問政溫和，導致黨外新生代的不滿：美麗島事件發生後，[[艾琳達|艾琳達]]撰文〈反對運動總批判〉，以「革新保臺」與「革命保臺」兩種路線嚴厲批評康寧祥。

民國69年(1980年)的[[1980年中華民國立法委員選舉|第三次增額立法委員選舉]]中，康寧祥角逐連任，當時社會氣氛普遍低迷，關心民主政治的學者、記者都很洩氣，然而康寧祥基於多年與國民黨搏鬥的經驗及對台灣社會的觀察，仍鼓舞「晚則五年，快則三年，黨外一定重新站起來。」{{r|康|page=308}}。康寧祥鼓勵受刑人家屬及黨外僅剩的精英代表黨外參選，一方面拜訪當時國民黨省黨部主委[[宋時選|宋時選]]，要求互設溝通管道。{{r|康|page=308}}以「黨外候選人聯誼會」推薦候選人。康寧祥指出除了「延續黨外香火」外，亦有為美麗島事件翻案的意味。{{r|康|page=309}}選舉結果「真黨外」推荐10席立委，當選六人，包括美麗島事件受刑人家屬[[許榮淑|許榮淑]]、[[黃天福|黃天福]]，辯護律師召集者之一的[[張德銘|張德銘]]，以及黃煌雄、黃余秀鸞和康寧祥。{{r|康|page=310}}由於康寧祥還是主張「議會路線」而非「街頭路線」，所以仍獲得連任。

=== 「批康運動」與落選 ===
1982年5月，黨外雜誌《深根》批評康寧祥對黨外立委杯葛事件「放水」，開始了「批康運動」，包括[[邱義仁|邱義仁]]、[[李敖|李敖]]加入戰局。{{r|康|page=361-364}}起因於1982年立委[[黃正安|黃正安]]等人提案，要求警備總司令列席立法院報告業務，共二十五人登記發言，然而在僅二人發言後，大量國民黨立委就硬以表決方式否決提案，事後黨外人事開會決定以杯葛法案，國民黨在派出黨團書記長[[周慕文|周慕文]]「懇談」，使康寧祥「陣前倒戈」。李敖批評康寧祥放水。<ref>李敖，〈放火的，不要變成放水的〉，《戰鬪是檢驗黨外的唯一標準》臺北：獅谷，1983年11月初版。頁123-140。</ref>隨後在1983年康寧祥針對行政院長[[孫運璿|孫運璿]]要求取消《[[臨時條款|臨時條款]]》中，孫答以「充實」臨時條款，獲康寧祥贊同，再被批評「放水」。<ref>李敖，〈別幫國民黨討姨太太！〉，《戰鬪是檢驗黨外的唯一標準》臺北：獅谷，1983年11月初版。頁157-170</ref>1983年9月，[[黨外編輯作家聯誼會|黨外編輯作家聯誼會]]在各政論雜誌發起「批康運動」批判康寧祥<ref>{{cite news|author=林朝億|publisher=[[台灣日報|台灣日報]]|url=http://www.hi-on.org.tw/bulletins.jsp?b_ID=49117|title= 老康與司馬文武三度共事|date=2003-01-26}}</ref>，導致康寧祥於[[1983年中華民國立法委員選舉|1983年立法委員選舉]]落選。

康寧祥認為「批康運動」給了國民黨「順風推倒牆」全力拉下他的機會，在國民黨臺北市黨部主委[[關中_(中華民國)|關中]]主導下，提名大批被視為財團「[[金牛_(選舉)|金牛]]」的候選人參選立委，並動員社會網路及黨團組織，在康寧祥傳統票倉萬華、龍山、雙園、建成大肆[[買票|買票]]。讓[[蔡辰洲|蔡辰洲]]、[[高忠信|高忠信]]、[[洪文棟|洪文棟]]如入無人之境，賄聲賄影。終於讓康寧祥基層票源慘敗。選舉結果，四位黨外候選人的得票比1980年選舉康寧祥和黃天福兩人的總和還少了三萬票。{{r|康|page=371-372}}

1984年6月，康寧祥接受美國[[哥倫比亞大學|哥倫比亞大學]]邀請，赴美一年作訪問學者。並參加台灣同鄉會演講，也拜訪[[雷根|雷根]]在[[休士頓|休士頓]]的競選總部，由於以往德州是民主黨天下，[[1984年美國總統選舉|1984年總統選舉]]時竟由雷根勝過[[孟岱爾|孟岱爾]]，因此康寧祥拜託美國國務院，接觸共和黨當地操盤者，了解當時共和黨對墨西哥裔社區的經營。其間康寧祥曾於1985年1月前往[[巴西|巴西]]，會見台獨運動參與者、政治犯[[鍾謙順|鍾謙順]]。1985年3月，回到台灣。{{r|康|page=373-384}}

1984年，「[[黨外公職人員公共政策研究會|黨外公職人員公共政策研究會]]」成立，被視為針對黨外編聯會的抗衡性組織。康寧祥試圖化解「公政會」和「編聯會」的矛盾。{{r|康|page=395-396}}1985年地方選舉（省市議員及[[1985年中華民國縣市長選舉|縣市長選舉]]），「公政會」和「編聯會」合組「中央委員會」，派出黨外精銳參選，結果省議員推荐18席當選11席、臺北市議員推薦11席全數當選，宜蘭縣[[游錫堃|游錫堃]]、桃園縣[[黃玉嬌|黃玉嬌]]、新竹市[[莊姬美|莊姬美]]、屏東縣[[蘇貞昌|蘇貞昌]]、臺北市第三選區[[謝長廷|謝長廷]]都以最高票當選。選後康寧祥等人在[[費希平|費希平]]家開會，商討黨外組織化。{{r|康|page=398}}1986年4月，康寧祥等人申請成立公政會「首都分會」，進一步組織化，陳水扁也宣布成立公政會「台北分會」。1986年5月17日，黨外公政會「首都分會」於臺北市濟南路成立。{{r|康|page=401-402}}6月13日，公政會「首都分會」提出康寧祥與[[林正杰|林正杰]]草擬「民主時間表」，要求1987年成立新黨、1988年解嚴行憲、1989年國會全面改選、1990年總統直選、1991年台海和平。{{r|康|page=403}}

=== 重回立院 ===
民國七十五年(1986年)，康寧祥也參與民進黨創黨「工作小組」，為建黨十八人之一。1986年9月28日，[[民主進步黨|民主進步黨]]成立。年底[[1986年中華民國立法委員選舉|第五次增額立法委員選舉]]中，民進黨在台北市提名三席（康寧祥、謝長廷、[[吳淑珍|吳淑珍]]），康寧祥重新當選立法委員。唯選前被看好的謝長廷因票源遭「代夫出征」的吳淑珍吸走，意外落選。

民國76年(1987年)，籲請政府調查[[東崗慘案|東崗慘案]]。

== 轉戰媒體 ==
民國78年(1989年)，康寧祥宣佈不再連任立委，並且創立《[[首都早報|首都早報]]》；但是《首都早報》由於資金不足，於1990年8月27日停刊。

== 監察院監察委員生涯 ==
民國82年(1993年)，[[監察院|監察院]]改制之後，康寧祥接受[[中華民國總統|中華民國總統]][[李登輝|李登輝]]推薦而成為第二屆[[監察委員|監察委員]]；並且於1999年連任監委。

民國90年(2001年)，榮膺第一屆[[國立臺北大學|國立臺北大學]]傑出校友。
== 陳水扁政府時期 ==
民國91年(2002年)，出任[[中華民國國防部|國防部]]副部長，2003年出任[[國家安全會議_(中華民國)|國家安全會議]]秘書長，於陳水扁總統時期曾任[[總統府資政|總統府資政]]。
