'''康斯坦丁二世'''（{{lang-el|Κωνσταντῖνος Βʹ}} {{IPA-el|ˌkonstanˈdinos}}，{{bd|1940年|6月2日|catIdx=Konstantin II}}），為希臘的末代國王。他在1964年即位，稱希臘國王康斯坦丁二世陛下（{{lang-el|{{lang|grc|Κωνσταντῖνος Βʹ, Βασιλεὺς τῶν Ἑλλήνων}}}}），1973年6月1日遜位，次年公投後承認[[希臘第三共和國|希臘第三共和國]]的成立。

== 早年 ==
康斯坦丁二世生於[[雅典|雅典]]普希基科宮；他是[[保羅一世_(希臘)|保羅一世]]的長子，[[喬治二世_(希臘)|喬治二世]]的姪子。同時，他亦與英國[[維多利亞女王|維多利亞女王]]及德皇[[威廉一世_(德国)|威廉一世]]等多個歐洲主要皇室有著血緣關係。

1960年，當時尚為王子的康斯坦丁曾在[[羅馬|羅馬]]所舉行的[[夏季奧林匹克運動會|夏季奧林匹克運動會]]當中出賽，並在風帆的項目中奪得金牌，時年僅20歲。同時，他也擅長游泳、[[空手道|空手道]]。1964年與一位丹麥公主結婚。

== 即位 ==
1964年3月，其父保羅就因為癌症而逝世。他便繼承王位，成為康斯坦丁二世。同年9月18日与丹麦國王的幼女[[安妮-玛丽_(希腊王后)|安妮-瑪麗公主]]结婚。

== 軍政府上台 ==
{{main|希臘軍政府時期}}
1967年4月21日清晨，由陸軍上校[[喬治斯·帕帕佐普洛斯|喬治斯·帕帕佐普洛斯]]所率領的軍方發動政變。不久之後，軍方首腦在坦克車的掩護之下與國王會面，國王見狀，起初將帕帕多普洛斯打發走；過了一會兒，才親自前往國防部。抵達時，所有主要將領均已聚集在部；在經過一番討論之後，康斯坦丁二世勉強答應軍方成立名為「上校團」的軍政府，以取代原來Panayiotis Kanellopoulos的右翼政府，並接受其宣誓。隨後，「上校團」政權便開啟7年的獨裁統治。同年12月，[[國王|國王]]與[[王室|王室]]出奔至[[羅馬|羅馬]]。國王的離開，使得君主制度已經是名存實亡。1973年，上校團逕自宣布廢除[[君主制|君主制]]。

== 民主的恢復與公民投票 ==
1974年7月，[[土耳其入侵塞浦路斯|土耳其入侵塞浦路斯]]一事導致軍政府垮台。而原先被迫流亡的[[康斯坦丁·卡拉曼利斯|康斯坦丁·卡拉曼利斯]]順利回國。康斯坦丁二世對此表示歡迎，並重新邀請他出任首相；同年11月，卡拉利曼斯所屬的新民主黨在國會選舉中大獲全勝，他便宣佈將於12月8日舉辦公民投票，決定希臘是要繼續採用[[君主立憲制|君主立憲制]]，還是改建[[共和|共和]]。公投的結果，絕大部分的民眾贊成改建為共和國。

== 退位後的生活 ==
[[File:Former_King_Constantine_&_Queen_Anne_Marie_of_Greece_in_Colour.jpg|thumb]]和[[菲利普_(希腊王子)|菲利普王子]]]]
對此，康斯坦丁二世表示尊重其結果，也承認共和國的成立。後來他在接受美國[[時代雜誌|時代雜誌]]專訪時表示：
{{cquote|如果希臘人決定他們要一個共和國，則他們擁有這個資格並在和平之中享受它。}}
從此退居幕後。
2004年12月24日，他與全體的舊王室成員一同前往總統府，拜訪希臘總統[[科斯蒂斯·斯特凡諾普洛斯|科斯蒂斯·斯特凡諾普洛斯]]。

==家庭==
1964年9月18日，国王在[[雅典|雅典]]与表妹[[丹麦|丹麦]]公主[[安妮-玛丽_(希腊王后)|安妮-玛丽]]结婚，婚后有三子二女：
*[[亞歷克西婭_(希臘公主)|亞歷克西婭公主]]（1965年7月10日生於[[雅典|雅典]]）
*[[帕夫洛斯_(希臘)|帕夫洛斯王儲]]（1967年5月20日生於[[雅典|雅典]]）
*[[尼古拉斯_(希腊王子)|尼古拉斯王子]]（1969年10月1日生於[[羅馬|羅馬]]克勞迪婭診所）
*[[狄奧多拉_(希臘公主)|狄奧多拉公主]]（1983年6月9日生於[[倫敦|倫敦]][[聖瑪麗醫院|聖瑪麗醫院]]）
*[[菲利普(希腊王子)|菲利普王子]]（1986年4月26日生於[[倫敦|倫敦]][[聖瑪麗醫院|聖瑪麗醫院]]）
