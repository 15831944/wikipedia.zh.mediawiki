'''康春元'''（{{bd|1958年|4月||}}）[[河北|河北]][[平山|平山]]人，[[中国人民解放军中将|中国人民解放军中将]]。

==生平==
长期在[[北京军区|北京军区]]从事军队政治工作，曾任[[北京军区|北京军区]]政治部宣传部部长。2005年7月接替[[张瑞林|张瑞林]]任[[中国人民解放军陆军第六十五集团军|中国人民解放军陆军第六十五集团军]]政治部主任。2010年接替到龄退役的[[王秉伦|王秉伦]]任北京军区政治部副主任，跻身正军级。任内曾任[[北京市人大|北京市人大]]代表。2014年12月，首次跨大军区交流，任[[兰州军区|兰州军区]]副政委，成为副大军区级将领<ref>{{cite news|title=七大军区岗位调动超30人 史鲁泽直升北京军区参谋长|url=http://hebei.ifeng.com/news/fengguanyanzhao/detail_2015_01/08/3397557_1.shtml|accessdate=2015-01-08|newspaper=凤凰网}}</ref><ref>{{Cite web |url = http://china.caixin.com/2016-08-29/100983056.html |title =原兰州军区副政委康春元晋升中将  |author =  |date =  2016-08-29|publisher = 财新网 |language =  |accessdate =  |quote =  }}</ref>。2016年任[[中国人民解放军战略支援部队航天系统部|中国人民解放军战略支援部队航天系统部]]政委<ref name=yi/>。

2006年7月，晋升少将军衔。2016年，晋升[[中国人民解放军中将|中将]]军衔<ref name=yi>{{cite web|date=2016-08-29|url=http://www.thepaper.cn/newsDetail_forward_1521187|title=副大军区职军官康春元已晋升中将军衔，曾任原兰州军区副政委|publisher=澎湃新闻}}</ref>。
