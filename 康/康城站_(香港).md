'''康城站'''（{{lang-en|'''LOHAS Park Station'''}}）是[[港鐵|港鐵]][[將軍澳綫|將軍澳綫]]現時在[[香港|香港]][[新界|新界]][[將軍澳|將軍澳]]的其中一個終點站，是將軍澳綫第二期工程的車站，位於[[小赤沙|小赤沙]]將軍澳第86規劃區的康城路。車站服務[[日出康城|日出康城]]一帶，即港鐵[[港鐵車廠#將軍澳車廠|將軍澳車廠]]旁邊，2009年7月26日啟用，承建商是[[前田建設|前田建設]]工業株式會社。

康城站目前是在港鐵系統中最東的車站（此站開通前則為[[坑口站_(香港)|坑口站]]）。另外，康城站是[[兩鐵合併|兩鐵合併]]後全香港第一個開通的車站。而且，本站是關站時間最遲的港鐵車站。

==車站設計==
康城站面積達35,000平方米，採用密封式設計，並以紫色為主要色調，與其他將軍澳綫車站一樣，以搪瓷板及[[紙皮石|紙皮石]]作車站牆壁。

本站上蓋為[[港鐵公司|港鐵公司]]迄今最大規劃的上蓋物業發展項目──[[日出康城|日出康城]]<ref>{{Cite news |url=http://hk.apple.nextmedia.com/financeestate/art/20070919/10175829 |title=日出康城下月應市|work=[[蘋果日報_(香港)|蘋果日報]]|date=2007-07-19|page=B6}}</ref>，一個比[[杏花邨|杏花邨]]發展項目還要大超過一倍（地盤面積亦等同於四個[[駿景園|駿景園]]）的綜合[[住宅|住宅]]發展項目。

規劃初期，本站被設計為地底車站，大堂會與毗鄰的購物商場相通，從當時的地鐵公司為該發展項目製作的宣傳影片可見，本站大堂跟商場無縫對接，設計一如[[九龍站_(港鐵)|九龍站]]跟[[圓方|圓方]]般。然而，最終無論住宅、社區配套和車站設計，均與當時由地鐵公司所規劃的「夢幻之城」（日出康城於規劃初期的暫名）有很大的差別，最終商場更要遲於2020年才落成，日出康城居民至今仍需要到將軍澳站一帶購買日用品。

雖然康城站屬於地面車站，但是為了避免列車對附近居民造成噪音滋擾，整個車站被混凝土圍封。列車由車站開往將軍澳站方向時，會有一段露天路段才進入隧道，但該段露天路段只有幾十米，這段路可從[[康城路|康城路]]的地面看見，而該段路軌亦是將軍澳綫以及整個西貢區的唯一一段露天路軌。

另外，在距離地面較接近的一段路，港鐵亦用鐵絲網圍起，並在有關路段附近的單車徑豎立路牌提醒道路使用者勿進鐵路範圍。

===樓層===
康城站樓高3層，底層為地面車站月台，車站內共設有4條扶手電梯、3部升降機及樓梯，頂層則為日出康城住宅平台，在月台旁邊為港鐵將軍澳車廠。

===大堂===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = LOHAS Park Station 2018 03 part3.jpg | caption1 = 大堂，近A出入口
| image2 = LOHAS Park Station 2018 03 part4.jpg | caption2 = 大堂，近C出入口
| image3 = LOHAS Park Station 2018 03 part5.jpg | caption3 = 商店
}}
康城站的大堂位於月台上層，大堂設有客務中心，商店等設施。由於康城站只是為日出康城發展項目而設，故只設有4部單程售票機及2部八達通增值機<ref name="tvb_630_news_2009_03_16">根據無綫電視於2009年3月16日[[六點半新聞報道|六點半新聞報道]]。</ref>，並平均分佈於大堂兩面。

客務中心位於大堂的中央，方便乘客作出查詢及處理票務問題事宜。此站的客務中心在通車初期由港鐵負責安排站務主任在客務中心當值。直至2010年1月，此站客務中心連同西鐵線（[[美孚站|美孚站]]、[[尖東站|尖東站]]、[[紅磡站|紅磡站]]除外）客務中心外判，由外判商安排客戶服務主任在客務中心當值。

===月台===
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = LOHAS Park Station 2018 05 part1.jpg | caption1 = 1號月台﹙往北角方向﹚
| image2 = LOHAS Park Station 2018 05 part2.jpg | caption2 = 2號月台﹙往北角方向﹚
}}
康城站設有2個月台，並以[[島式月台|島式月台]]排列，月台設有[[月台幕門|月台幕門]]，位於將軍澳車廠地面旁。月台並不設落客專用月台及調頭隧道，列車到站後會在該月台同一時間進行上落客及調頭。
列車通常只使用1號月台，而2號月台為後備月台。現時有部分列車暫時停泊在2號月台。由旁邊的將軍澳車廠開出到本站載客的列車、到達此站清客的回廠列車或部分常規班次列車亦會使用1號月台，以便列車直接進出車廠。由車廠駛出車站也會在2號月台稍作停留。類似車站為[[屯門站_(西鐵綫)|屯門站]]。由於車站只是為日出康城發展項目而設，因此車站比其他車站的月台闊度為窄，月台的闊度只能容納一條單向扶手電梯<ref name="tvb_630_news_2009_03_16" />。由於此站扶手電梯使用量低，它們採用省電模式。當乘客進入扶手電梯範圍，扶手電梯才會加快至普通速度。此站是第一個設有此類扶手電梯的港鐵車站。

康城站啟用初期設有3個出入口，其中2個出入口連接上蓋物業──日出康城的住宅平台，另一個出入口則會連接車站的臨時公共運輸交匯處。此外，車站亦預留B出口作為未來出入口，以通往日出康城商場。工程在2018年8月展開，預計2020年啟用。

==班次詳情==
{{ multiple image | align = right | direction = vertical | width = 220
| image1 = MTR LOHAS Park Station time table.jpg | caption1 = 本站列車開出時間表（2011年1月）
| image2 = LOHAS Park Station 2017 08 part8.jpg | caption2 = 願景顯示屏（班次顯示器）
}}
康城站為將軍澳綫北行的其中一個終點站（另一個為寶琳），將軍澳綫列車於平日非繁忙時間以「2+1」的安排：每兩班由[[北角站|北角]]開往[[寶琳站|寶琳]]的列車，會開出一班由北角開往康城站的列車，班次為5.5至12分鐘一班；而在非繁忙時間及假日，康城站只提供往來[[北角站|北角]]及[[將軍澳站|將軍澳]]的穿梭列車服務，每12分鐘一班。若果乘客需要在該站往來[[坑口站_(香港)|坑口站]]或寶琳站，需在將軍澳站轉乘往寶琳方向的列車；在繁忙時間及假日，若要由本站前往將軍澳綫沿途各站或其他路綫的車站，需在將軍澳站或調景嶺站轉車。

2012年11月12日起，將軍澳綫增加一班由康城站往[[油塘站|油塘站]]列車，平日非繁忙時間於早上7時12分開出，由早上7時05分至7時32分的班次相應加密至5.5至7分鐘一班。此外，列車服務在非繁忙時間完結前，也會略作調整，讓班次更為穩定。而星期六的班次也是保持在10至14分鐘一班。

==利用狀況==

康城站鄰近只有[[日出康城|日出康城]]、[[將軍澳工業邨|將軍澳工業邨]]及一個[[公共運輸交匯處|公共運輸交匯處]]，周邊現時仍無購物商場，亦甚少巴士路線途經。因此，大部份使用車站進出的人流都是日出康城的居民或在將軍澳工業邨裡工作的員工。所以，除了上下班繁忙時間外，整體出入閘的人流會比其他車站較少。因此，康城站是港鐵[[將軍澳綫|將軍澳綫]]8個車站中較少人使用的車站，在通車初期，港鐵因實行「3+1」的被批評導致寶琳站及坑口站的乘客等候列車時間更長。但自其他期數開始動工及開始入伙，人流高速增加，港鐵列車服務班次在上下班繁忙時間內並沒有因此而作出改善。

==接駁交通==
康城站近A出入口的地面設有[[康城站公共運輸交匯處|康城站公共運輸交匯處]]，雖然交匯處內設有巴士、小巴及的士站，但車站啟用初期，只有穿梭巴士往返日出康城第一期－[[首都_(香港)|首都]]，以及市區的士站，並無其他巴士及小巴路線往返康城站。

不過，由於連接港鐵康城站至首都的新建行人通道於2010年10月30日啟用後，令往來日出康城的居民巴士客量大幅下跌，因此營辦商與業主立案法團開會後決定由2010年12月1日起停辦居民巴士服務，現時該處只尚餘[[領都|領都]]免費居民巴士服務。

2011年2月14日起，康城站才有首項該處為總站的公共交通服務，當日起[[新界區專線小巴112M線|新界區專線小巴112M線]]投入服務，為日出康城居民及[[將軍澳工業邨|將軍澳工業邨]]上下班人士提供接駁港鐵的服務。

==歷史==
===命名===
康城站在將軍澳綫規劃時，由於車站只是位於[[將軍澳|將軍澳]][[百勝角|百勝角]]對出的第86規劃區填海用地上，故此車站在方案中只是以'''86區'''作車站命名。直至後期的[[港鐵|地鐵]]中期報告、年報及相關文件中<ref>[http://www.mtr.com.hk/chi/investrelation/Interim2001/mapoperating_c.pdf 地鐵有限公司 - 二零零一年度中期報告]</ref>，以'''將軍澳南'''為站名。而在車站工地外圍板上的模擬圖片，中文站名為'''將軍澳南'''，英文站名則是'''Tseung Kwan O South'''<ref>根據康城站工地外圍板上的模擬圖片。</ref>。亦因之前為垃圾堆填區，故在特大暴雨後一帶會散發出異味。

根據[[香港鐵路有限公司|香港鐵路有限公司]]（前身為香港地下鐵路公司，2000年[[私有化|私有化]]後改名為地鐵有限公司）於2007年12月的[http://www.mtr.com.hk/chi/publications/images/business_overview_c.pdf 業務概覽]中，車站是以'''日出康城'''作為標示，而在2008年2月向傳媒展示的內容，內容並無提及車站名稱，只是以「日出康城新車站」為標題向傳媒簡介，並非以將軍澳南命名。不過，根據香港討論區於2008年7月刊登本站內部裝修照片，此站將以'''康城'''為站名。2008年10月18日港鐵正式將此站命名為'''康城'''<ref name="tks_name">{{Cite news|language=zh-hant|url=http://news.sina.com.hk/cgi-bin/nw/show.cgi/5/1/1/909933/1.html|title=將軍澳港鐵新站命名「康城」|work=[[星島日報|星島日報]]|date=2008-10-19|format=再版}}</ref>，並於2009年7月26日啟用。
