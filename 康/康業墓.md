'''康業墓'''是一座於2004年在[[西安|西安]]北郊上林苑住宅區發現的[[北周|北周]][[粟特人|粟特人]]古墓葬。

== 墓主簡介 ==
根據墓誌記載，墓主康業，字元基，為[[康居|康居]]國王後裔，歷任魏大天主、羅州使君、車騎大將軍等職。北周[[天和_(北周)|天和]]六年（公元571年）逝世，去世後被詔增為甘州刺史。墓葬出土時，康業的遺骨安睡於石榻上，有絲綢包裹的痕跡。由於粟特人大多信仰[[祆教|祆教]]，通常以火燒的方式處理遺體，因此，存留至今的完整粟特人遺骨很少。康業的遺骨是目前中國發現的唯一一具保存完整的北周粟特人骨架。<ref>{{cite web |url=http://news.sina.com.cn/c/2004-11-19/19114289724s.shtml |title=北周粟特人墓葬發掘的新發現 |author=肖芬 |date=2004 |website=news.sina.com.cn |access-date=2018年7月29日}}</ref>

== 墓葬簡介 ==
[[File:Stone_Couch_from_the_Tomb_of_Kʻang_Yeh.jpg|thumb]]
康業墓位於西安北二環地區，由墓道、甬道和墓室三部份組成，墓道已毀壞。墓門上有線雕，門楣正中有一獸面，左側[[靑龍|靑龍]]，右側[[朱雀|朱雀]]，門框頂端各立一隻鳳鳥，下方各有一鎭守墓門的線刻神像。墓內出土一具圍屏石榻，由四塊石板組成，上繪十幅屏風畫像，畫面殘留貼金彩繪的痕跡。畫中人物計有70多位，有高鼻深目的胡人，也有衆多漢人，服飾多已漢化。這些人物大多置身蔥蘢茂密的山林花草間，其中景致山巒起伏、雲氣繚繞、垂柳依依、泉水潺潺，天空翱翔之鳥、地上食草之牛，展現出一派舒適安然的生活寫照。此墓畫像同其他粟特人墓葬，如[[安伽墓|安伽墓]]、[[史君墓|史君墓]]、[[虞弘墓|虞弘墓]]出土的文物不同在於，康業墓的畫像以陰線雕繪，並且沒有很多關於宗教的內容，<ref>{{cite web |url=https://www.xueshu.com/jlysxyxb/201504/2349712.html |title=康業墓石棺床圍屏山水圖像研究 |author=陳思 |date=2015 |website=www.xueshu.com |access-date=2018年7月29日}}</ref>漢化程度也高於前面提到的三座墓葬。<ref>{{cite book |editor=吳天泰 |date=2015 |title=《族群與社會》 |url=https://books.google.com/books?id=bsYgpuud5gMC&pg=PA290&lpg=PA290&dq=%E5%BA%B7%E6%A5%AD%E5%A2%93+%E6%BC%A2%E5%8C%96&source=bl&ots=x6FACl0aD-&sig=243EbRyh7rP4XmmNZVN8EFhd8Yw&hl=en&sa=X&ved=0ahUKEwjIr-KbxbHcAhXOIlAKHQQyBGMQ6AEIJzAA#v=onepage&q=%E5%BA%B7%E6%A5%AD%E5%A2%93%20%E6%BC%A2%E5%8C%96&f=false |location=臺北 |publisher=五南圖書出版 |page=290 |isbn=9789571141756 |accessdate=2018年7月29日}}</ref>

康業的遺骨躺在圍屏石榻上，口中含有{{Link-en|拜占庭貨幣|Byzantine coinage|拜占庭金幣}}一枚，右手還握有北周布泉銅幣一枚。金幣正面有[[拜占庭皇帝|拜占庭皇帝]]胸像浮雕，胸像左右兩邊刻[[拉丁|拉丁]]銘文{{lang|la|DNIVSTINI}}（左）、{{lang|la|ANVSPPAV}}（右）；背面是一位手持十字架和[[十字聖球|十字聖球]]的[[天使|天使]]全身像，天使像周圍環繞三條銘文：{{lang|la|VICTORI}}（左）、{{lang|la|AAVGGGZ}}（右）、{{lang|la|CONOB}}（底部）。<ref>{{cite web |url=http://publications.nichibun.ac.jp/region/d/NSH/series/niso/2008-12-26/s001/s013/pdf/article.pdf |title=ソグド人墓の発見と最新研究 |author=王維坤 |page=第14頁（PDF頁數） |date= |website=publications.nichibun.ac.jp |language=ja |access-date=2018年7月29日}}</ref><ref name=TKY>{{cite web |url=http://www.kaogu.cn/uploads/soft/Chinese%20Archaeology/9/Tomb%20of%20Kang%20Ye%20of%20the%20Northern%20Zhou%20in%20Xi%E2%80%99an%20Shaanxi.pdf |title=Tomb of Kang Ye of the Northern Zhou in Xi'an, Shaanxi |last1=Cheng |first1=Linquan |last2=Zhang |first2=Xiangyu |translator=Ho, Judy Chungwa |page=第3頁（PDF頁數） |date=2008 |website=www.kaogu.cn |language=en |access-date=2018年7月29日}}</ref>除了錢幣、墓誌銘、墓門線雕和圍屏石榻，康業墓還出土有甬道及墓室內的壁畫、銅製腰帶扣及相關飾件。<ref name=TKY />
