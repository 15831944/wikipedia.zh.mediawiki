'''康乃狄克州'''（{{lang-en|State of Connecticut}}），{{各地中文名 |cn=「康涅狄格州」 |hk=「康州」 |tw=「康乃狄克州<ref name="boca-Connecticut-Boston"/>、 康州<ref name="boca-Connecticut-Consumer"/><ref name="boca-Connecticut-NY"/>、康乃迪克州<ref name="boca-Connecticut-NY"/>」|display=inline}}，是[[美國|美國]][[美國東北部|東北部]]的一[[美國州份|州]]，也是[[新英格蘭|新英格蘭]]區域中最南的一州。在[[美國獨立戰爭|美國獨立戰爭]]時期，是[[十三殖民地|13州聯盟]]之一。州花[[山桂|山桂]]，州鸟[[美洲知更鸟|美洲知更鸟]]，州树[[白橡|白橡]]。[[美國海軍|美國海軍]]曾以該州名稱分別命名[[康涅狄格號戰艦|康涅狄格號戰艦]]（BB-18）、{{tsl|en|USS Connecticut (SSN-22)|康乃狄克潛艦 (SSN-22)|康涅狄格号核动力潜水艇}}（SSN-22）。

== 歷史 ==
「康涅狄格」（Connecticut）一名来自[[莫希干|莫希干]]（Mohegan）[[印第安语|印第安语]]的「Quinnehtukqut」，意思是「有长长河流的地方」或是「在长长的[[感潮河|感潮河]]边」。

1638年，哈特福德的居民通过了现代史上第一部成文[[宪法|宪法]]。

== 法律與政府 ==
{{main|康乃狄克州同性婚姻}}
2008年11月12日，康涅狄格州成為正式继[[麻塞诸塞州|麻塞诸塞州]]之後，第二个[[同性婚姻|同性婚姻]]合法化的[[美國州份|美國州份]]。

== 地理 ==
該州可分為五個地理區：塔柯尼克地區、中央河谷、西新英格蘭高地、東新英格蘭高地、與海岸平原。

== 人口 ==
[[File:Hartford_Connecticut_Skyline.JPG|缩略图]]
[[File:New_Haven_from_East_Rock.jpg|缩略图]]]]
2003年人口普查，康涅狄格州有3,483,372人，比1990年增長了6%。其中
* 77.5% 白人
* 9.4% [[西班牙|西班牙]]裔美國人
* 9.1% 非裔美國人
* 2.4% 亞裔美國人
* 0.3% [[印地安人|原住民]]
* 2.2% 混合的種族
在康涅狄格州的居民祖先中，[[義大利|義大利]]裔美國人（18.6%）最多，其次是[[愛爾蘭|愛爾蘭]]人（16.6%）、[[英國|英國]]人（10.3%）、[[德國|德國]]人（9.9%）和[[波蘭|波蘭]]人（8.3%）。

在總人口中，五歲以下的居民占6.6%，18歲以下占24.7%，65歲以上占13.8%。女性則占有全人口的51.6%。

康涅狄格州的人口信仰比例<ref name="pew2014">[http://www.pewforum.org/religious-landscape-study/state/connecticut/ The Pew Forum - America’s Changing Religious Landscape]</ref>：
* 35% [[新教|新教]]徒
* 33% [[羅馬天主教|羅馬天主教]]徒
* 1% 其他基督教派的教徒
* 7% 其他宗教教徒
* 23% 無信仰者

康涅狄格州新教徒中，各教義派所占比例不同，其中以[[浸禮會|浸禮會]]最高（5%）、其次為[[聖公會|聖公會]]（4%）和[[衛理公會派|衛理公會派]]（4%）。

== 行政区划 ==
康乃狄克州下辖8个县，分别是费尔菲尔德、哈特福德、利奇菲尔德、米德尔塞克斯、新港、新伦敦、托尔兰及温德汉姆。
