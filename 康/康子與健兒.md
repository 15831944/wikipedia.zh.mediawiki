《'''暴走兄妹'''》（{{lang-ja|'''ヤスコとケンジ'''}}；另譯'''康子與健兒'''）是[[日本|日本]][[漫畫家|漫畫家]][[或子|或子]]所畫[[日本漫畫|漫畫作品]]。在[[集英社|集英社]]發行的[[女性漫畫|女性漫畫]]雜誌《[[Deluxe_Margaret|Deluxe Margaret]]》的2004年1月號發表後，於《[[別冊Margaret|別冊Margaret]]》2005年4月號至2006年11月號連載。[[單行本|單行本]]全5冊。

2008年由[[日本電視台|日本電視台]]改編拍成[[日本電視劇|電視劇]]，7月到9月於[[日本電視台週六連續劇|每週六]]播出，由[[松岡昌宏|松岡昌宏]]（[[東京小子|TOKIO]]）、[[廣末涼子|廣末涼子]]、[[多部未華子|多部未華子]]、[[大倉忠義|大倉忠義]]等人主演。

另外在《Deluxe Margaret》從2009年1月號開始、發表番外編的「暴走兄妹」。

== 故事 ==
6年前的12月24日，雙親因為交通事故而死，於是沖康子和原[[暴走族|暴走族]]總長的哥哥沖健兒開始住在一起。但是由於哥哥的阻撓，使得她總是無法好好談戀愛。於是兄妹因此每天過著吵架的日子，但康子卻某天遇上了資優生樁純而愛上他，但純的老姐卻是以前常和健兒吵架的另一原暴走族總長惠理香，因此康子除了要讓純喜歡上她外，還要努力改善健兒和惠理香這對歡喜冤家的關係，故事也在此展開了一段非常暴走的生活。

== 登場人物 ==
===主要角色===
; 沖康子（{{lang|ja|沖 康子、おき やすこ}}）
: 主角。有著愛做白日夢的性格。17歲，高校2年級。在補習班上遇見純而一見鍾情。老是和自己的哥哥健兒吵架。對於健兒都以「你」來稱呼他、自己的手機則是把他設定登錄成「那傢伙」，看起來似乎討厭哥哥，但實際上比誰都還要信任他。和純的姐姐惠理香很合得來，現在在她開的花店「樁花」打工中。學校的成績是接近不及格的邊緣。5月17日生。
:雖然會做菜，但廚藝似乎不佳，煎蛋常會做得太鹹。

; 沖健兒（{{lang|ja|沖 健児、おき けんじ}}）
: 康子的哥哥。24歲。是暴走族「Naked Gun」的原總長，因為雙親俱亡而解散掉。現在的職業是成人向漫畫的漫畫家（戲劇版是少女漫畫家），筆名健健（戲劇版是櫻葉麗華），有著愛強迫人和很凶暴的性格。因為對康子管教嚴格而常與康子吵架，無論對手是誰都不會放過。康子有時候會被他丟到洗衣機裡並且就這樣讓她旋轉著。平常戴著一副眼鏡，化身為暴走族時則會拿掉。視守護康子為自己的正義，沒有結婚的願望。

; 椿純（{{lang|ja|つばき じゅん}}）
: 惠理香的弟弟，是個就讀男子高中的二年級生，在校內的考試都是拿到第一名。假日時會幫忙花店的工作。討厭的食物是紅蘿蔔。對於幹夫一直纏著他並且想辦法對付他的行為感到討厭，所以完全不把他當對手。

; 椿惠理香（{{lang|ja|椿 エリカ、つばき えりか}}）

:經常在遇到健兒時，原來想表示好意的行為，都會變成和他吵架。

; 摩斯（{{lang|ja|モス}}）
: 暴走族「Naked Gun」的原親衛隊隊長。現在擔任健兒的助手。有一頭[[光頭|光頭]]和嚇人的長相、但人卻非常好。23歲。本名是正助（{{lang|ja|しょうすけ}}）。在短篇中的右眼上有一個舊傷。

===和原作的相異點===
*康子和純相遇，健兒和惠理香的再見面的方法都不一樣。
*康子和純被設定成高校3年級生（17歳）。另外健兒是二十九歲。
*健兒的筆名從「健健」改成「櫻葉麗華（{{lang|ja|さくらば れいか}}）」。另外原作是個成人向漫畫家，戲劇版中則是在讀者中有著廣大人氣的少女漫畫家。
*純轉學到康子的高中。另外和惠理香住一起。
*惠理香經營花店「樁花」、宮園則在裡面工作。
*愛迪達和佳志是青梅竹馬、愛迪達很喜歡她。之後成為戀人。
*佳織的第一人稱是「{{lang|ja|あっし}}」。容姿也和原作大大不同。
*原作是青田對健兒夜襲、戲劇版中則是青田欺騙黒薔薇的成員並讓她們去夜襲健兒。
*女暴走族時代、惠理香並沒有對健兒做第二次告白。
*康子和健兒的雙親的忌日是10年前的7月26日，也是康子和純的生日。
*山田幹夫設定成純小学時代的同學。和原作同様是很卑鄙的個性、但對付純的方法並不一樣。另外和原作不同，不是打從真心喜歡康子。
*設定惠理香和純的父親是大型物産公司的社長、母親則在惠理香小時候就過世。原作雙親健在、經營花屋。
*戲劇版中有多個原創角色。
