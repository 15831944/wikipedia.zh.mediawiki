'''康辉'''（{{bd|1972年|1月17日||catIdx=K}}），男，[[汉族|汉族]]，籍贯[[河北省|河北省]][[石家庄市|石家庄市]][[无极县|无极县]][[北苏镇|北苏镇]]。[[中国中央电视台|中国中央电视台]]新闻[[播音员|播音员]]、[[主持人|主持人]]、央视新闻中心播音部主任。在[[党和国家领导人|党和国家领导人]]出访时，亦会以[[记者|记者]]身份出访。有一个大他两岁的姐姐。

== 生平 ==
康辉初中时毕业于[[河北省|河北省]][[石家庄市|石家庄市]][[石家庄市第四十中学|第四十中学]]。

高中时毕业于[[河北省|河北省]][[石家庄市|石家庄市]][[河北师范大学附属中学|河北师范大学附属中学]]。

1993年，毕业于[[北京广播学院|北京广播学院]]（现[[中国传媒大学|中国传媒大学]]）。毕业后进入[[中国中央电视台|中国中央电视台]]，成为新闻节目[[主持人|主持人]]。

2006年6月5日，第一次主持《[[新闻联播|新闻联播]]》节目。

2007年12月8日，再次主持《新闻联播》，并主持《新闻联播》至今。

2013年7月，康辉作客[[凤凰卫视|凤凰卫视]]節目《[[魯豫有約|魯豫有約]]》。<ref>[http://www.youtube.com/watch?v=27hlaDL4LIo 鲁豫有约 康辉："国脸"的台前幕后 2013-07-17]</ref>

2015年，康辉正式主持[[2015年中国中央电视台春节联欢晚会|2015年中国中央电视台春节联欢晚会]]，此为他首次担任春晚主持人。<ref>{{cite web |url=http://media.people.com.cn/n/2015/0204/c14677-26508321.html|title= 羊年春晚主打"家和万事兴" 康辉小尼首主持春晚|accessdate=2015年2月4日|publisher= 人民网|work= |language =}}</ref>在[[2017年中国中央电视台春节联欢晚会|2017年]]、[[2018年中国中央电视台春节联欢晚会|2018年]]的春晚上，康辉亦担任主持人。<ref>{{cite web|url=http://news.cctv.com/2018/02/06/ARTIqGCpHCnohSWRNPYbB8Rs180206.shtml|title=2018年春节联欢晚会主持阵容发布 |publisher=央视网 新闻频道 |author= |date=2018-02-06|accessdate=2018-02-06}}</ref>
