'''重庆轨道交通环线'''是[[重庆轨道交通|重庆轨道交通]]运营中的一条环线地铁，是重庆轨道交通线网中唯一的闭合环形线路，是重庆市轨道交通第二轮建设规划中的重要线路<ref>{{Cite web|url=http://www.cqmetro.cn/wwwroot_release/crtweb/jiansheguihua/elgh/336033.shtml|title=重庆市城市轨道交通第二轮建设规划概况|accessdate=|author=|date=|work=|publisher=|archive-url=https://web.archive.org/web/20180418161710/http://www.cqmetro.cn/wwwroot_release/crtweb/jiansheguihua/elgh/336033.shtml|archive-date=2018-04-18|dead-url=no}}</ref>。分为环线一期工程和环线二期工程建设，一期开工采用六编组钢轮钢轨山地地铁As型车制式<ref>{{Cite web|url=http://paper.people.com.cn/rmrb/html/2017-05/31/nw.D110000renmrb_20170531_5-10.htm|title=首款山地型地铁在中车四方下线|accessdate=|author=|date=|last=|work=人民日报|publisher=|archive-url=https://web.archive.org/web/20170806152114/http://paper.people.com.cn/rmrb/html/2017-05/31/nw.D110000renmrb_20170531_5-10.htm|archive-date=2017-08-06|dead-url=no}}</ref>，也是重庆第一条全线一次性开工的地铁线，全长50.93公里，于2013年10月28日开工，2018年12月28日开通[[海峡路站|海峡路站]]至[[重庆图书馆站|重庆图书馆站]]之间的东北段。重庆轨道交通环线共设3座车辆基地，分别为涂山车辆段、四公里停车场和马家岩停车场。其中马家岩停车场与轨道交通一号线共用马家岩综合基地，出场站为天星桥。

环线是集连接铁路、公路、水运、空运多位一体的轨道交通线路，将有效建立便捷、高标准的内部交通和对外交通体系，促进客运交通一体化建设，对重庆市空间结构优化和有序建设起到重要地引导与支持作用<ref name=":0">{{Cite web|url=http://www.cqmetro.cn/wwwroot_release/crtweb/jiansheguihua/zjxl/336045.shtml|title=重庆地铁环线（重庆西～上桥～重庆北～弹子石～重庆西）工程概况|accessdate=|author=|date=|work=cqmetro|publisher=|archive-url=https://web.archive.org/web/20180418161554/http://www.cqmetro.cn/wwwroot_release/crtweb/jiansheguihua/zjxl/336045.shtml|archive-date=2018-04-18|dead-url=no}}</ref>。

==  环线走向  ==
[[File:Path4236-6-3-6-0.png|缩略图]]
地铁环线起於沙坪坝区新建[[重庆西站|重庆西站]]，采用地下敷设方式。后在沙正街北侧转为高架，利用高家花园轨道专用桥上跨[[嘉陵江|嘉陵江]]。过江后进入地下穿行15公里，利用[[朝天门长江大桥|朝天门长江大桥]]下层轨道交通专用通道上跨[[长江|长江]]。過長江後轉為地下，向南經彈子石、上浩站後轉為高架敷設，經海棠溪、羅家壩、四公里後再次轉為地下，沿海峽路向西，經大石壩立交後在重庆[[鵝公岩軌道專用長江大橋|鵝公岩軌道交通專用橋]]<ref>{{Cite web|url=https://www.miaopai.com/show/xzO1~wxqp5OJrrePyNF7pxE6xFsn9cAgNGqfGg__.htm|title=世界最大跨度的自锚式悬索桥——重庆鹅公岩轨道专用桥顺利合龙！长江天堑迎来世界最大跨度“牵手”-人民网|accessdate=|author=|date=|work=|publisher=|archive-url=https://web.archive.org/web/20180508185205/https://www.miaopai.com/show/xzO1~wxqp5OJrrePyNF7pxE6xFsn9cAgNGqfGg__.htm|archive-date=2018-05-08|dead-url=no}}</ref>東橋頭爬出地面上跨長江，過江後以地下方式向西接入線路起點重庆西站。<ref name=":0" />。

== 车辆概况 ==
[[File:CRT_Loop_Line_Train_on_Gaojiahuayuan_Rail_Bridge.jpg|thumb]]
重庆轨道交通环线线路采用双线全封闭独立运行系统，地铁山地城市As型车（也称：山地城市A型车），6辆编组（远期可运用7辆编组），轴重≤15 吨。采用 DC1500V 架空悬挂接触网供电。As型地铁列车<ref>{{Cite web|url=http://www.xinhuanet.com/politics/2017-05/27/c_1121049826.htm|title=我国首款山地型地铁列车在中车四方下线-新华网|accessdate=|author=|date=|work=|publisher=|archive-url=https://web.archive.org/web/20180418162138/http://www.xinhuanet.com/politics/2017-05/27/c_1121049826.htm|archive-date=2018-04-18|dead-url=no}}</ref>是介于Ａ、Ｂ型地铁之间的新车型，适用于山地环境，具有爬坡能力强、转弯半径小、载客量大等特点，最高运行时速为100公里，６编组列车最大载客量为2322人。重庆地铁四号线、五号线、九号线、十号线、环线等线路均将使用山地城市As车型车，并实现重庆地铁互联互通和跨线运营<ref>{{Cite web|url=http://www.cqmetro.cn/wwwroot_release/crtweb/xwzx/mtbb/336572.shtml|title=重庆轨道交通将互联互通 列车可跨线运行到目的地|accessdate=|author=|date=|work=|publisher=|archive-url=https://web.archive.org/web/20180418161631/http://www.cqmetro.cn/wwwroot_release/crtweb/xwzx/mtbb/336572.shtml|archive-date=2018-04-18|dead-url=no}}</ref><ref>{{Cite web|url=http://www.bj-tct.com/view.php?id=202|title=重庆轨道交通互联互通“示范工程”专题汇报会_交控科技|accessdate=|author=|date=|work=|publisher=|language=|archive-url=https://web.archive.org/web/20180418161555/http://www.bj-tct.com/view.php?id=202|archive-date=2018-04-18|dead-url=no}}</ref>。信号系统采用车地无线通信支撑CBTC系统（列车自动控制系统）。通信服务提供商为[[华为|华为]]，其OneAir车地无线解决方案，同时承载了CBTC和车辆状态监控等多种业务，为轨道的安全运行提供保障——采用授权1.8G频谱，99.999%的可靠性，毫秒级的时延，能够充分满足CBTC的需求。基于不同业务属性进行多业务QoS设计，在无线资源不足的情况下优先为CBTC业务提供最高保障。[[华为|华为]]OneAir车地无线解决方案基于3GPP标准，是面向轨道交通行业适配的增强LTE解决方案，有效支持不同LTE网络互联互通。<ref>{{Cite web|url=http://e.huawei.com/cn/case-studies/cn/2016/201612270918|title=华为OneAir行业无线保障重庆地铁安全可靠运营|accessdate=|author=|date=|work=|publisher=|language=|archive-url=https://web.archive.org/web/20180702180109/http://e.huawei.com/cn/case-studies/cn/2016/201612270918|archive-date=2018-07-02|dead-url=no}}</ref>

*2018年12月28日环线东北环正式开通，但包括渝鲁站、重庆北站南广场站在内的多个站点内部装修存在明显赶工现象。上新街站等换乘站换乘标志均使用临时制作导向牌，与其他线路及设计标准不符；由于灯箱未完全准备完毕，许多导向牌及出口标志使用纸板并以透明胶贴合在出口、灯箱架上临时代替；线路开通前一天，许多车站仍在进行基本装修施工。通车后，涂山等站点才开始逐渐更换纸板导向牌为标准灯箱。

*2019年1月8日下午試營運期間，因海峡路-南湖区间隧道内人防门侵入限界<ref>{{cite web |title=突发运营信息续报 |url=https://www.weibo.com/2152519810/Hb1svzGyu?from=page_1001062152519810_profile&wvr=6&mod=weibotime&type=comment |website=重庆轨道交通官方微博}}</ref>，环线00014号列车与人防门发生碰擦事故，导致列车第一节车厢偏移，列车车头严重损毁，但未脫軌，<ref>{{cite web |title=重庆轨道交通环线发生事故 有人员受伤 |url=http://news.ifeng.com/a/20190108/60229049_0.shtml |website=凤凰网}}</ref>并造成1名乘客和2名工作人员受伤，另外1名工作人员抢救无效死亡。<ref>{{cite web|title=轨道环线人防门与列车发生擦碰 致3个站暂停运 其余线路正常运行|url=http://app.cqrb.cn/economic/2019-01-09/65182.html|website=重庆日报}}</ref>事故后至1月10日17:00，海峡路至四公里共三站临时停运。
