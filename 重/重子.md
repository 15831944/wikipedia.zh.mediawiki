'''重子'''（Baryon）是一個現代[[粒子物理學|粒子物理學]]名詞，在[[標準模型理論|標準模型理論]]中，「重子」這一名詞是指由三个[[夸克|夸克]]（或者三个[[反夸克|反夸克]]组成的「反重子」）组成的複合粒子。<ref name=Halliday>{{citation|last1=Halliday|first1=David|last2=Resnick|first2=Robert|last3=Walker|first3=Jerl|title = Fundamental of Physics|publisher = John Wiley and Sons, Inc.|location = USA|edition = 7th|isbn=0-471-23231-9|year=2005|quote=Each baryon is a combination of three quarks}}</ref>{{rp|1232}}在這理論中它是[[強子|強子]]的一類。值得注意的是，因為重子屬於複合粒子，所以「不是」[[基本粒子|基本粒子]]。最常见的重子有組成日常[[物質|物質]][[原子核|原子核]]的[[质子|质子]]和[[中子|中子]]，合称为[[核子|核子]]。其它重子中，有比这两種粒子更重的粒子，所谓的[[超子|超子]]。重子这个称呼是指其质量相对重于[[轻子|轻子]]和介于两者之间的[[介子|介子]]起的。

重子是强相互作用的[[费米子|费米子]]，也就是说它们遵守[[费米-狄拉克统计|费米-狄拉克统计]]和[[泡利不相容原理|泡利不相容原理]]，它们通过组成它们的夸克参加[[强相互作用|强相互作用]]。同时它们也参加[[弱相互作用|弱相互作用]]和[[引力|引力]]。带电荷的重子也参加[[电磁学|电磁力作用]]。

重子与由一个夸克和一个反夸克组成的介子一起被合称为[[强子|强子]]。强子是所有强相互作用的粒子的总称。

质子是唯一独立稳定的重子。中子假如不与其它中子或者质子一起组成[[原子核|原子核]]的话就不會稳定，並產生衰变。

== 重子家族 ==
重子中除了核子外，还有[[Δ|Δ]]、[[Λ|Λ]]、[[Σ|Σ]]、[[Ξ|Ξ]]和[[Ω|Ω]]等重子家族。

;核子（p<sup>+</sup>、n）
:由上夸克和下夸克组成，是組成原子核的粒子。其中中子會衰變為一個[[電子|電子]]、一個[[反微中子|反微中子]]及一個質子，質子則是唯一獨立穩定的重子。

;Δ重子（Δ<sup>++</sup>、Δ<sup>+</sup>、Δ<sup>0</sup>、Δ<sup>−</sup>）
:由上夸克和下夸克组成，它们衰变为[[π介子|π介子]]以及一个质子或者一个中子。

;Λ重子（Λ<sup>0</sup>、Λ<sup>+</sup><sub>c</sub>）
:由一个上夸克、一个下夸克以及一个魅夸克或者一个奇异夸克组成。电中性的Λ重子的发现首次显示了奇异夸克的存在。

;Σ重子（Σ<sup>+</sup>、Σ<sup>0</sup>、Σ<sup>−</sup>）
:由一个奇异夸克和两个上或者下夸克或者一个上夸克和一个下夸克组成。电中性的Σ重子与电中性的Λ重子相同，因此比其它Σ重子衰变得快。

;Ξ重子（Ξ<sup>0</sup>、Ξ<sup>−</sup>）
:由两个奇异夸克和一个上夸克或者一个下夸克组成。电中性的Ξ重子（由一个上夸克和两个奇异夸克组成）衰变为一个电中性的Λ重子和一个电中性的π介子，后者很快衰变为一个[[电子|电子]]和一个[[阳电子|阳电子]]，两者互相湮灭，因此整个过程看上去好像是电中性的Ξ重子衰变为电中性的Λ重子和[[伽马射线|伽马射线]]。

;电负性的Ω重子（Ω<sup>−</sup>）
:由三个奇异夸克组成。理论家首先预言了它的存在、质量和衰变结果。它是通过这些预言通过它的衰变结果被发现的，因此它的发现是夸克理论的一个重要成果。

== 重子物质 ==
'''重子物质'''是指从质量上来看主要由重子组成的物质，比如所有由[[原子|原子]]组成的物质。而'''非重子物质'''则是从质量上来看主要不是由重子组成的物质，比如[[中微子|中微子]]、自由[[电子|电子]]、[[光子|光子]]等等。对于[[暗物质|暗物质]]主要是由重子物质还是由非重子物质组成的问题有很多争议。重的可能组成非重子暗物质的物质可能有[[超对称|超对称粒子]]、[[黑洞|黑洞]]等。对于[[宇宙学|宇宙学]]来说这个问题非常重要，因为早期[[宇宙|宇宙]]中重子物质的量直接影响到[[大爆炸理论|大爆炸理论]]中的[[核合成|核合成]]模型。

重子物质的存在本身在宇宙学中就已经提出了一个难题。大爆炸理论预言宇宙开始时重子与反重子的量应该是一样多的，此后重子超过反重子的过程被称为重子相变。

'''重子的分类与命名'''

根据[[同位旋|同位旋]]（I）和所含夸克的种类将重子分为两类六组：
*[[核子|核子]]（N）：[[质子|质子]]（p）、[[中子|中子]]（n）、[[第二代中子|第二代中子]]n<sub>μ</sub>（1149[[GeV|GeV]]）、[[第三代中子|第三代中子]]n<sub>τ</sub>（49501[[GeV|GeV]]）
*[[超子|超子]]：[[Δ粒子|Δ重子]]、[[Λ粒子|Λ重子]]、[[Σ粒子|Σ重子]]、[[Ξ粒子|Ξ重子]]、[[Ω粒子|Ω重子]]

命名规则依据的是轻夸克（上夸克、下夸克、奇夸克）与重夸克（粲夸克、底夸克、顶夸克）的组合情况，规则涵盖了六种夸克所有可能的三夸克组合的情况，包括包含顶夸克的组合：
*重子包含三个（{{SubatomicParticle|link=yes|Up quark}} 、 {{SubatomicParticle|link=yes|Down quark}}）夸克：{{SubatomicParticle|link=yes|Nucleon}}(''I'' = {{sfrac|1|2}}) 、{{SubatomicParticle|link=yes|Delta}}(''I'' = {{sfrac|3|2}}) 
*重子包含两个（{{SubatomicParticle|link=yes|Up quark}} 、 {{SubatomicParticle|link=yes|Down quark}}）夸克和一个（{{SubatomicParticle|Strange quark}}）夸克：{{SubatomicParticle|link=yes|Lambda}}(''I'' = 0) 、 {{SubatomicParticle|link=yes|Sigma}}(''I'' = 1)，如果{{SubatomicParticle|Strange quark}}夸克是重夸克（{{SubatomicParticle|Charm quark}}、 {{SubatomicParticle|Bottom quark}}、 {{SubatomicParticle|Top quark}} ）将夸克符号标为下标
*重子包含一个（{{SubatomicParticle|link=yes|Up quark}} 、 {{SubatomicParticle|link=yes|Down quark}}）夸克和两个（{{SubatomicParticle|Strange quark}}）夸克：{{SubatomicParticle|link=yes|Xi}}(''I'' = {{sfrac|1|2}})，如果{{SubatomicParticle|Strange quark}}夸克是重夸克（{{SubatomicParticle|Charm quark}}、 {{SubatomicParticle|Bottom quark}}、 {{SubatomicParticle|Top quark}} ）将夸克符号标为下标
*重子没有包含（{{SubatomicParticle|link=yes|Up quark}} 、 {{SubatomicParticle|link=yes|Down quark}}）夸克，包含了三个（{{SubatomicParticle|Strange quark}}）夸克： {{SubatomicParticle|link=yes|Omega}}(''I'' = 0)，如果{{SubatomicParticle|Strange quark}}夸克是重夸克（{{SubatomicParticle|Charm quark}}、 {{SubatomicParticle|Bottom quark}}、 {{SubatomicParticle|Top quark}} ）将夸克符号标为下标
*對於重子強衰變粒子，''J''<sup>''P''</sup>值被視為其名稱的一部份，共振態的质量添加在括号中（单位：[[電子伏特|MeV]]/[[光速|''c'']]<sup>2</sup>）。

实际使用时还有一些额外的规则对重子之间进行区别，会用到一些不同的符号：
*只含有一種夸克的重子（如 uuu 和 ddd）存在 ''J''<sup>''P''</sup> = {{frac|3|2}}<sup>+</sup> 組態，而 ''J''<sup>''P''</sup> = {{frac|1|2}}<sup>+</sup> 組態是[[泡利不相容原理|泡利不相容原理]]所不允許的。
*含有二種夸克的重子（如 uud 和 uus）和三種夸克的重子（如 uds 和 udc）可以存在''J''<sup>''P''</sup> = {{frac|1|2}}<sup>+</sup> 和 ''J''<sup>''P''</sup> = {{frac|3|2}}<sup>+</sup> 两种組態，添加上标（''' ∗ '''）区别。
*含有三種夸克的重子（例如 uds 和 udc）可以存在''J''<sup>''P''</sup> = {{frac|1|2}}<sup>+</sup> 的两种組態。添加上标（''' ′ '''）区别。
*根据重子的电荷数添加上标（'''<sup>0</sup>'''、'''<sup>+</sup>'''、'''<sup>-</sup>'''）号
