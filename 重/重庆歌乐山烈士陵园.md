'''重庆歌乐山烈士陵园'''（'''重庆歌乐山革命纪念馆'''），位于[[重庆市|重庆市]][[沙坪坝区|沙坪坝区]][[歌乐山|歌乐山]]麓，主要安葬[[中华人民共和国|中华人民共和国]]成立前牺牲在重庆的[[烈士_(中华人民共和国)|烈士]]，由重庆歌乐山烈士陵园管理处管理。

==简介==
重庆歌乐山烈士陵园位于沙坪坝区歌乐山麓，是[[中华民国|中华民国]]时期[[军统局|军统局]]总部、电台、监狱所在地。[[第二次世界大战|第二次世界大战]]后期，曾在此建“[[中美特种技术合作所|中美特种技术合作所]]”。

1949年11月底，[[中国人民解放军|中国人民解放军]]逼近重庆，[[中华民国政府|中华民国政府]]在退守[[台湾|台湾]]前將此处的[[白公馆|白公馆]]、[[渣滓洞|渣滓洞]]集中营中關押的囚犯全部處決，酿成惨绝人寰的[[一一·二七大屠杀|一一·二七大屠杀]]，其中仅有极少数脫逃。1949年11月30日，中国人民解放军攻占重庆。新政权追认死难者为“[[烈士_(中华人民共和国)|烈士]]”。

1955年修建烈士公墓（后称重庆11·27烈士墓）和烈士纪念碑。1963年又设立[[红岩魂陈列馆|重慶中美合作所集中營美蔣罪行展覽館]]，并恢复了白公馆、渣滓洞的原貌。1984年7月，整个陵园改称“重庆歌乐山烈士陵园”，由[[邓小平|邓小平]]题写园名。1993年增挂“重庆歌乐山革命纪念馆”馆牌。是[[全国爱国主义教育示范基地|全国爱国主义教育示范基地]]、国家国防教育示范基地、国家4A级旅游景区<ref>{{Cite web |url = http://www.redsa.com.cn/html/2017-05/15/content_41621500.htm |title =重庆歌乐山革命纪念馆  |author =  |date = 2017-05-15  |publisher = 重庆党史网 |language =  |accessdate =  |quote =  }}</ref>。

1956年8月16日，“烈士公墓及中美合作所集中營舊址”公佈為四川省第一批歷史及革命文物保護單位<ref>『四川省人民委員會關於在工農業生產建設中注意保護文物的通知』，「（56）省二文字第0618號」</ref>。1980年7月7日，“重慶市11·27烈士墓暨中美合作所集中營舊址”重新公佈為四川省第一批文物保護單位<ref>『關於重新公布全省文物保護單位名單的通知』，四川省人民政府，一九八〇年七月七日，川府發（1980）154號文件</ref>。1983年12月1日，“重慶11.27烈士墓中美合作所集中營舊址”列為重慶市第一批市級文物保護單位<ref>「重慶市人民政府關於重新公布重慶市文物保護單位名單的通知」，重慶市人民政府文件，重府發〔1983〕208號，1983年12月1日</ref>。

重庆歌乐山烈士陵园占地面积2.14平方公里，主要由集中营旧址、烈士群雕、烈士诗文碑林、大型浮雕、烈士墓、陈列总馆（[[红岩魂陈列馆|红岩魂陈列馆]]）组成<ref>{{Cite web |url = http://cpc.people.com.cn/GB/33843/2524399.html|title = 重庆歌乐山烈士陵园 |author =  |date =  |publisher = 中国共产党新闻网 |language =  |accessdate =  2017-07-17|quote =  }}</ref>。
