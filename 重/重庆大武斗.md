'''重庆大武斗'''是1967年7月至次年9月间，在[[重庆市|重庆]]地区发生的一系列[[武斗|武斗]]。其中1967年7月至9月间的武斗又称为重庆百日大武斗<ref name="a3"/>。1967年夏至1968年夏，重庆地区见于官方记载的武斗有31次，使用军械兵器24次，645人死亡<ref name="a1">{{Cite web |url=http://news.163.com/10/0119/09/5TCO7LGU00011SM9.html |title=重庆红卫兵墓园 唯一保存完好的“文革”墓群 |accessdate=2017-07-23 |author=记者：吴伟 |date=2010-01-19|work= |publisher=网易，来源：新京报(北京)|language=Zh-hans }}</ref>（有疑议，仅[[重庆文革墓群|重庆文革墓群]]一处武斗死难者就有四百余人）。最重要的[[造反派|造反派]]有“[[重庆市无产阶级革命造反联合委员会|八一五派]]”和“[[重庆工人反到底总司令部|反到底派]]”。

== 背景 ==

1960年代，在[[中共中央西南局|中共中央西南局]]第一书记[[李井泉|李井泉]]主持下，将中共宜宾地委书记[[刘结挺|刘结挺]]、市委书记[[张西挺|张西挺]]打成反党分子。1967年5、6月间，刘结庭、张西挺获得[[平反|平反]]，二人准备出任筹建中的[[四川省|四川省]][[革命委员会_(文革)|革命委员会]]副主任。当时重庆的“8·15派”反对刘张二人，“反到底派”则拥护刘张。

== 武斗经过 ==

1967年6月5日至8日，“8·15派”和“反到底派”参加[[西南师范大学|西南师范学院]]武斗，被视为重庆大武斗的序幕。7月8日，两派在红岩柴油机厂武斗，打死9人，伤近200人。因此次武斗首次使用枪弹，故称为“打响重庆斗第一枪”。7月31日至8月6日，[[荣昌县|荣昌县]]武斗，死亡78人。8月8日，[[重庆八八海战|重庆八八海战]]，打死240人，伤129人，沉船3艘，重创12艘<ref name="a2"/>，[[长江航运|长江航运]]因此中断。8月12日，[[中國嘉陵|嘉陵机械厂]]武斗，打死数十人，伤600余人。8月13日，[[抗战胜利纪功碑暨人民解放纪念碑|解放碑]]武斗。8月18日至20日，沙坪坝区[[潘家坪武斗|潘家坪武斗]]，打死128人<ref name="a3"/>（另说近百人<ref name="a2"/>）。8月22日，[[南岸区|南岸区]]上新街武斗，打死22人，伤数十人<ref name="a3"/>。8月23日，[[北碚区|北碚区]]歇马场武斗，打死40人<ref name="a2">{{Cite web |url=http://news.ifeng.com/history/zhiqing/ziliao/detail_2012_03/23/13407104_1.shtml |title=亲历者回忆重庆武斗最惨无人道罪行：枪毙俘虏 |accessdate=2017-07-23 |author=作者：余刘文、韩平藻 |date=2012-03-23 |work= |publisher=凤凰网，来源：中国新闻网|language=Zh-hans }}</ref>。8月下旬的[[杨家坪武斗|杨家坪武斗]]打死200余人，[[杨家坪|杨家坪]]半数房屋被毁。10月27日，[[中国人民解放军陆军工程大学通信士官学校|解放军通讯兵工程学院]]警卫连驻地武斗，打死32名学院成员及群众，受伤53人<ref name="a3">{{Cite web |url=http://www.yxjedu.com/li_shi_shun_jian/wen_ge/wg_lian_jie_wy/chong_qing_wu_dou.htm |title=惨绝人寰的67年重庆百日大武斗|accessdate=2017-07-23 |author= |date=2006-05-16|work= |publisher=星岛环球网|language=Zh-hans }}</ref>。

1968年3月中旬，[[长寿县|长寿县]]云台处的川东北石油处本部武斗，打死40人。4月28日，长安机器厂因武斗大火，重庆市消防大队救火途中遭遇枪击，打死9人，打伤30余人，3辆消防车受损。7月，[[江陵机械厂|江陵机械厂]]武斗，大批人员伤亡<ref name="a3"/>。其后，“8·15派”控制重庆，在“大清洗”的谣言下，数十万“反到底派”成员和家属逃离重庆。9月23日，[[重庆市革命委员会|重庆市革命委员会]]、[[中国人民解放军重庆警备区|重庆警备区]]发布命令，严令一切群众组织、团体和个人上缴武器、车辆，拆除工事、据点，解散武斗队。10月15日，“8·15派”和“反到底派”宣布撤销总部，解散组织<ref name="a2"/>。
