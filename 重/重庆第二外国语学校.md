'''四川外国语大学重庆第二外国语学校'''是一所位于重庆市[[南岸区|南岸区]]，由南岸区人民政府、重庆宏声集团、北京同景集团共同投资管理的重庆市[[重点中学|重点中学]]<ref>{{cite web|url=http://www.cq.gov.cn/zwgk/zfxx/2014/7/25/1326890.shtml |title=重庆市南岸区多举措推进区域教育特色发展 |publisher=重庆市政府网 |date=2014-07-26 |accessdate=2016-08-11 }}</ref>。学校前身为1940年成立的重庆市女子学校，现占地面积168[[亩|亩]]，有学生5000人，教职工422人。

== 历史沿革 ==
*1940年，重庆市女子中学校成立
*1952年，学校从[[南岸区|南岸区]][[玄坛庙|玄坛庙]]迁址至[[黄桷垭|黄桷垭]]，并更名为重庆市第四女子中学
*1969年，重庆市第四女子中学更名为重庆市第四中学校
*1997年，经[[重庆市人民政府|重庆市人民政府]]和重庆市教育委员会批准改制为四川外语学院重庆第二外国语学校
*2002年1月，学校进行了资产重组，分别由南岸区政府、重庆宏声集团、北京同景集团共同投资管理学校。<ref>{{cite web |url=http://cqew.paibid.com/www/site/site_index_2_4.shtml |title=学校简介 |accessdate=2016-08-11 |publisher=四川外国语大学重庆第二外国语学校 }}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>
*2013年4月，因四川外语学院更名[[四川外国语大学|四川外国语大学]]，该校跟随更名四川外国语大学重庆第二外国语学校。<ref>{{cite web |url=http://cqcbepaper.cqnews.net/cqcb/html/2013-04/27/content_1633913.htm |title=四川外语学院更名为四川外国语大学 |date=2013-04-27 |accessdate=2016-08-11 |publisher=重庆晨报 |deadurl=yes |archiveurl=https://web.archive.org/web/20151211074442/http://cqcbepaper.cqnews.net/cqcb/html/2013-04/27/content_1633913.htm |archivedate=2015-12-11 }}</ref>
