'''重男轻女'''是一種認為男女不平等的观念，認為[[男性|男性]]各方面皆較[[女性|女性]]重要，導致女性[[人权|因其为人而应享有的权利]]經常因為要遷就男性而犧牲。這種觀念常見於[[父權社會|父權社會]]，常伴隨著[[男尊女卑|男尊女卑]]和[[性别角色|性別分工]]的觀念。

==成因==
在早期人类还处于[[狩獵採集|狩猎采集]]社会时，性别角色在社会地位中的影响并不大，人们处于一个相对平等的状态。当人类过渡到[[農業社會|农业社会]]时，性别造成的社会分工逐渐分化，而因为农耕使人们能够积累资源，使性别不平等逐渐开始产生。<ref>http://www.worldclass.net/geo/Reilly/menwom1.htm</ref><ref>http://www.theguardian.com/science/2015/may/14/early-men-women-equal-scientists</ref>

==东亚社会生活與習俗上的重男輕女==
[[東亞|東亞]]地區的[[傳統|傳統]]喪葬儀式中的[[五服|五服]]，為[[父系|父系]][[親屬|親屬]]所服的喪會比同輩且同等的[[母系|母系]][[親屬|親屬]]服的喪要重，如為[[叔父|叔父]]服的喪會比為[[姨母|姨母]]服的喪重；即使同樣是父系或母系親屬，為男性服的喪也會比為女性服的喪重，如為叔父服的喪會比為姑母服的喪重。

一些語言也表現出重男輕女的思想（亦有相反的詞語）。例如在[[中文|中文]]裡「婦人之見」指沒見識的人的見解，「婦人之仁」指不顧全大局的仁慈，「三姑六婆」指是非之人，均含有輕視女性的意思。中国上古的卜辞里也视生男为“嘉”，生女为“不嘉”。在[[日語|日語]]裡，稱別人的丈夫時會稱為「御主人樣（{{lang|ja|御主人様}}）」，稱別人[[妻子|妻子]]時卻只會稱為「奧樣（{{lang|ja|奧様}}）」或「奧方樣（{{lang|ja|奧方様}}）」（「奧」是「屋內」的意思），意味著只有丈夫才是一家之主。

在傳統禮儀上的重男輕女：社會地位相同的男女，男方要較女方行較重的禮<ref>段氏釋拜又云：『婦人以肅拜，拜男子之空手，以手拜當男子之稽首。』鄭注《[[少儀|少儀]]》曰：『手拜至地，手至地如男子稽首，然則婦人敬之至，而手拜，正猶男子必敬之至，乃稽首也，其拜如舅姑，亦寸肅拜而已。』</ref>。

历史法律上，女性犯下某些罪行，刑罰會較男性為輕，如《[[左传|左传]]·襄公十九年》记载“ 妇人无刑，虽有刑，不在朝市”。另外部份朝代的法律規定妻子犯罪，即使刑罰非[[連坐|連坐]]或[[族誅|族誅]]，丈夫也同樣要受罰，甚至由丈夫代妻子受罰<ref>《[[唐律|唐律]]》：“若家人共犯，止坐尊长。”《[[律疏|律疏]]》：尊长谓男夫者, 假有妇人尊长，共男夫卑幼同犯，虽妇人造意，仍以男夫独坐。”</ref>。但丈夫犯罪除非被判[[連坐|連坐]]或[[族誅|族誅]]，否則妻子不需受罰。《[[唐律|唐律]]》又規定[[婚姻|婚姻]]的[[妄冒|妄冒]]罪，其处罚女方比男方轻一等《唐律》：“诸为婚而女家妄冒者，徒一年。男家妄冒，加一等。未成者，依本约已成者，离之”。這些看似保護或優待女性的措施其實是把女性視為從屬於男性，代表男性对家中女性有绝对代表权。一般来说，历史上的女性在法律意义上是不作为个人存在的。

===举例===
* 2012年中国共产党内部选举出的205名[[中央委员|中央委员]]中，女性仅有10人；171名[[中央候补委员|中央候补委员]]中，女性仅有23人；390名省级党委常委中，女性仅有35人。
* 2014年，中国大陆的新生儿性别比虽然实现了自2009年以来的连续第六次下降，但仍然高达每100名出生女婴对应115.88名出生男婴，性别选择性[[堕胎|堕胎]]依然较为猖獗。
* 日本虽然拥有女子[[相扑|相扑]]比赛，但男子相扑行业保持着女性“不洁”的观念，身为女性的[[大阪府|大阪府]]知事[[太田房江|太田房江]]曾于2000年和2001年两度欲为男子相扑比赛优胜者颁奖而均遭拒绝。
* 香港地区直到1972年才颁布《婚姻改革条例》正式禁止男性同时拥有多个配偶。

==西方社会生活与风俗上的重男轻女==
西方社会很早就有就有重男轻女的风俗，《[[圣经|圣经]]》裏注明“女人最初是男人的[[肋骨|肋骨]]所生的”、“妻子要听丈夫的话”等。

[[第一次世界大战|第一次世界大战]]以前，歐洲君主获得子嗣时要鸣[[礼炮|礼炮]]表示庆祝，君主生兒子的禮炮會較生女兒多，例如大部份國家有王子出生時鸣101响礼炮，王女則只鸣21响。[[俄罗斯|俄罗斯]]帝国皇子（大公）出生鸣放300响礼炮，皇女（女大公）則只鸣放101响。

而现代西方社会，经过[[女权主义运动|女权主义运动]]，重男轻女已逐渐减少，但女性还是会遭遇[[潜规则|潜规则]]效应或者职场[[性骚扰|性骚扰]]等情况。
===举例===
* [[英法百年战争|英法百年战争]]和[[奥地利王位继承战争|奥地利王位继承战争]]的直接导火索均为《[[萨利克法典|萨利克法典]]》中禁止女性继承家业的相关规定，[[神圣罗马帝国|神圣罗马帝国]]一直不允许女性成为皇帝。
* 法国女性直到1944年才开始拥有[[选举权|选举权]]和被选举权，瑞士直到1971年才允许女性参与[[全民公投|全民公投]]。
* 美国、英国、法国、俄罗斯、巴西等西方主要大国的国会女性议员比例均不足30%，德国和北欧各国则为40%左右。
* 从二战后截至2000年，对女性开放军队战斗岗位的西方国家仅有[[加拿大军事|加拿大]]一国。[[澳大利亚军事|澳大利亚]]于2013年对国防军现役女兵开放了全部前线岗位，但仅有不足20名女兵申请加入前线作战部队。
* 2014年的北美票房前100名中，有21部影片是女性担当第一主演或者和男一号平分秋色；其中，年纪超过45岁或者性取向为[[同性恋|同性恋]]、[[双性恋|双性恋]]的女主角数量为零。<Br/>动画影片中，女性为主角的影片数量低于1/4；动作／冒险类影片中，女性担任一号主角的比例为21.8%。<Br/>从事2014年北美百部影片内容生产的1326位电影工作者中，女性仅占15.8%，其中导演占1.9%，编剧占11.2%，[[制片人|制片人]]占18.9%。<Br/>在2007-2014年的700部北美高票房电影中，导演总数为779名，其中女导演有28名。

==家庭中的重男輕女==
很多有重男輕女思想地方一些重男輕女的家庭<ref>{{cite web |title=中国重男轻女最严重的10个地方，谁救救樊胜美们？ |url=https://cj.sina.com.cn/article/detail/1894964897/253130 |publisher=新浪网}}</ref>，為了生男孩而不顧自己負擔能力不斷生育，直至生出男孩為止。有些則透過[[性別選擇|性別選擇]]去選擇生男孩。而一些法制不健全的地方還會把[[女嬰溺死|女嬰溺死]]。這些現象可能令人口[[性別比|性別失衡]]，在廣東省，出生嬰兒的性別比高達137.76<ref>{{cite web|url=http://news.sina.com.cn/cul/2005-01-07/2931.html|title=今日话题：从13亿人说中国社会重男轻女焦虑症|date=2005-01-07|publisher=新浪网|language=zh-cn}}</ref>。
<ref>{{cite web |title=重男轻女的家庭有多可怕？“原来我存在的意义，是为了供养弟弟啊” |url=http://www.sohu.com/a/252745085_554740 |publisher=搜狐网}}</ref>
有些重男輕女觀念較深的家庭裏女兒出世，父辈乃至祖辈都会极度失望懊恼。即使是程度一般的，也常會認為女孩子長大後要嫁人，不需要讀太多書，於是不顧各子女的能力和意願，要女兒輟學工作賺錢供她們的[[兄弟|兄弟]]讀書。<ref>{{cite web|url=http://edu.hangzhou.com.cn/20050801/ca935338.htm|title=十八岁女孩的心事：爸爸重男轻女 我该不该辍学|date=2005年11月21日|publisher=杭州网|language=zh-cn}}</ref>在某些中下階層或工人階級家庭裡，即使女兒成績很好，兒子無心向學，這些父母也照樣如此。平時也會對兒子和女兒有不同待遇，例如有些有好吃的、好玩的都優先考慮給兒子，卻要女兒做較多的家務。極端的情況甚至是把兒子視為寶貝，在家裡有如小皇帝，女兒則被視為奴僕看待，也常被打罵。

另一方面，男性被重視的同時也常意味着要負較重的責任，一些家庭就會對兒子管教特別嚴厲，對兒子的要求也比對女兒嚴格。
