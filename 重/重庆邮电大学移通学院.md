'''重庆邮电大学移通学院'''是经[[中华人民共和国教育部|中华人民共和国教育部]]批准设立的普通高等本科院校<ref>教发函[2004]41号</ref>，是2010年度十佳优秀独立学院之一<ref>{{cite web |url= http://www.eol.cn/2010eol_10907/20101221/t20101221_556434.shtml |title=2010年度十佳优秀独立学院|date=2010-12-21 |accessdate=2016-01-07 |publisher=中国教育在线 }}</ref>，位于[[重庆市|重庆市]][[合川区|合川区]]。

== 学校简介 ==
重庆邮电学院移通学院成立于2000年，是经教育部批准，由重庆邮电大学举办的独立学院，属全日制普通本科院校，面向全国招生。2006年3月，重庆邮电学院更名[[重庆邮电大学|重庆邮电大学]]，该校随即更名重庆邮电大学移通学院。学校位于[[重庆市|重庆市]]合川大学城。目前占地面积1350余亩，总建筑面积近49万平方米，教学仪器设备总值9999.4万元，校外实习实训基地98个，图书资料150余万册。现有通信与物联网工程学院、智能工程学院、大数据与软件学院、淬炼商学院、远景学院、中德学院、艺术传媒学院、创意写作学院、创业学院等10个二级学院，管理工程系、外国语言文学系、光电信息系等3个系，4个教学部，以及完满教育委员会、双体系卓越人才教育基地等部门。共开设通信工程、电气工程及其自动化、轨道交通与信号控制、财务管理、工程管理、广播电视编导等34个本科专业，通信技术、工商企业管理等4个专科专业，形成了以工为主，经、管、文、艺五大学科协调发展的专业体系。目前在校学生接近19000人。现有专任教师930人，外籍教师10余人，硕士以上学位教师占比60%。

== 教学单位 ==
移通学院下辖13个二级院系，分别是：

*艺术传媒学院
*远景学院
*外国语文学系
*中德应用技术学院
*通信与物联网工程学院
*智能工程学院
*大数据与软件学院
*淬炼商学院
*管理工程系
*光电信息系
*继续教育与培训学院
*创业学院
*创意写作学院
