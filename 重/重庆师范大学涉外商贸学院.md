'''重庆师范大学涉外商贸学院'''是[[重慶師範大學|重慶師範大學]]的一个[[獨立學院|獨立學院]]。总占地面积800余亩。有两个校区，分别位于重庆市[[合川区|合川区]]和[[铜梁区|铜梁区]]。

== 校史更迭 ==
*2001年，成立重庆师范学院海棠学院（重师与民营企业共建）
*2002年，海棠学院正式开始招生
*2013年，重庆师范学院更名为[[重庆师范大学|重庆师范大学]]，该校校名随即更改为重庆师范大学涉外商贸学院

== 教学单位 ==
该校共有7个二级学院，分别是：

#外国语学院
#文学与新闻学院
#经济与贸易学院
#管理学院
#数学与计算机学院
#表演艺术学院
#艺术设计学院

== 教育情况 ==
*[[图书馆|图书馆]]：拥有馆藏纸质图书130万册，电子图书146万册，期刊1100种
*学生：学院有全日制在校学生15000余人
*教师：该校有专任教师885人，专任教师中具有[[研究生|研究生]]以上学历的教师占70%<ref>{{cite web |url=http://swsm.cn/web/guest/13 |title=学校简介 |accessdate=2016-01-05 |publisher=重庆师范大学涉外商贸学院 }}</ref>
