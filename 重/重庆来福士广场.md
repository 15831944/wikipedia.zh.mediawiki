'''重庆来福士广场'''，是位于[[重庆市|重庆市]][[渝中区|渝中区]][[朝天门|朝天门]]地区的一处包括办公楼、酒店、服务式公寓、购物中心于一体的大型商业中心，由[[凯德置地集团|凯德置地集团]]（CapitaLand）和星桥腾飞联合开发，是凯德置地在中国的第八座[[来福士广场|来福士广场]]，也是其中规模最大的项目。

重庆来福士广场由8座塔楼、底层裙楼以及横跨4座塔楼的水晶连廊组成。

== 历史 ==
重庆来福士广场的区块原为重庆港客运大楼和三峡宾馆，朝天门这两栋地标性建筑于2012年8月30日爆破拆除。<ref name=history>{{cite web |url=http://cq.cqnews.net/html/2014-07/16/content_31380720.htm |language= |title=[重庆记忆]建筑立寓意 看"古渝雄关"朝天门"变脸" |publisher=[[华龙网|华龙网]] |date=2014-07-16 |accessdate=2018-08-01 |archive-url=https://web.archive.org/web/20180802002248/http://cq.cqnews.net/html/2014-07/16/content_31380720.htm |archive-date=2018-08-02 |dead-url=no }}</ref>来福士广场于2012年9月28日奠基，<ref>{{cite web|url=https://www.zaobao.com/consumer/property/oversea/story20120929-27005 | language= | title=中国新地标 重庆来福士朝天门动工 |publisher=[[联合早报|联合早报]] |date=2012-09-29 |accessdate=2018-08-01 }}</ref>投资额超过人民币240亿元，是当时新加坡在中国投资最大的单体项目。

2017年6月18日，来福士广场T3S塔楼封顶，成为8座塔楼中最先完成封顶的一座。<ref>{{cite web |url=http://cq.qq.com/a/20170618/022072.htm |language= |title=“朝天扬帆”封顶啦！ 来看看新地标如何炫酷 |publisher=腾讯大渝网 |date=2017-06-18 |accessdate=2018-08-01 |archive-url=https://web.archive.org/web/20180802002247/http://cq.qq.com/a/20170618/022072.htm |archive-date=2018-08-02 |dead-url=no }}</ref>

2019年1月，淡馬錫轄下星橋騰飛被凱德集團收購，此項目亦相應改由凱德獨資發展。

== 建筑架构 ==
=== 建筑设计 ===
[[File:RafflesCity_Chongqing_Skywalk.jpg|220px]]
重庆来福士广场由[[摩西·萨夫迪|摩西·萨夫迪]]担纲设计。萨夫迪将渝中半岛比作一艘航行在河流中的大船，而朝天门作为[[嘉陵江|嘉陵江]]和[[长江|长江]]两江交汇处处在船头的位置，因而选取“朝天扬帆”的立意。<ref>{{cite web|url=http://www.cqcb.com/caifangbuhuodong/fxctm/2016-08-18/185083.html | language= | title=“朝天扬帆”是未来重庆之门 设计师首谈设计灵感 |publisher=[[重庆晨报|重庆晨报]] |date=2016-08-18 |accessdate=2018-08-01 }}</ref>8座塔楼临水北向，外立面均为弧线造型，呈现出风帆的效果；同时萨夫迪采用水晶连廊将4座塔楼相连，组成一扇通透的重庆未来之门，诠释朝天门“古渝雄关”的气势。

横跨塔楼的水晶连廊长约300米、宽约30米、高约22.5米，被称为“横向的摩天大楼”。

== 争议 ==
重庆来福士广场的设计和建设过程都受到不少重庆市民的非议。<ref>{{cite web |url=http://www.chinanews.com/cul/2018/04-23/8497631.shtml |language= |title=重庆朝天门广场改造引质疑 古城文化地标保护之争 |publisher=[[中国新闻网|中国新闻网]] |date=2018-04-23 |accessdate=2018-08-01 |archive-url=https://web.archive.org/web/20180423194513/http://www.chinanews.com/cul/2018/04-23/8497631.shtml |archive-date=2018-04-23 |dead-url=no }}</ref>争议的焦点主要集中在大楼的庞大的体量将破坏天际线，并削弱[[朝天门|朝天门]]作为历史地标的影响力。

2015年6月，在来福士广场的修建过程中，发现了一条长达140米的南宋古城墙遗迹，<ref>{{cite web|url=http://www.xinhuanet.com/2015-06/10/c_1115565064.htm | language= | title=重庆：主城首段宋代古城墙现身朝天门 |publisher=[[新华网|新华网]] |date=2015-06-10 |accessdate=2018-08-01}}</ref>后疑似遭到施工人员的破坏。2018年年初，由重庆市民组成的“重庆市文物保护志愿者服务队”向重庆市委、市政府提交了一封名为《关于保护重庆地标、精神之门——朝天门历史文化》的公开信，质疑政府对古朝天门文化遗产保护不足。<ref>{{cite news |author1=李国 |title=重庆朝天门广场改造引质疑 古城文化地标保护之争 |url=http://cnews.chinadaily.com.cn/2018-04/23/content_36074767.htm |accessdate=2018-08-08 |work=中国日报网 |date=2018-04-23}}</ref>

还有一些观点认为大楼形似[[摩西·萨夫迪|摩西·萨夫迪]]本人之前设计的新加坡[[滨海湾金沙酒店|滨海湾金沙酒店]]，缺少城市特色。政府和开发商对此的回应是大楼的设计融合了重庆特色，包括城市街道直接连接屋顶花园的“重庆式”建筑风格、与城市山水一脉相承的叠瀑设计等。<ref>{{cite web|url=https://www.zaobao.com/news/china/story20180402-847395 | language= | title=旧城区与新发展的拔河 重庆来福士广场引发朝天门文化保护争议 |publisher=[[联合早报|联合早报]] |date=2018-04-02 |accessdate=2018-08-01}}</ref>

也有人称重庆来福士广场酷似弯刀的造型会破坏渝中半岛的[[风水|风水]]。<ref>{{cite web |url=http://www.kanshangjie.com/article/136513-1.html |language= |title=一个把重庆人丑哭了的建筑，正在毁掉朝天门 |publisher=商界 |date=2018-02-01 |accessdate=2018-08-01 |archive-url=https://web.archive.org/web/20180802002248/http://www.kanshangjie.com/article/136513-1.html |archive-date=2018-08-02 |dead-url=no }}</ref>
