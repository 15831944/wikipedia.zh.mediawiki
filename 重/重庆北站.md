'''重庆北站'''位于[[中国|中国]][[重庆市|重庆市]][[渝北区|渝北区]]龙头寺，是线上式布局的特大型客运站，是重庆主城区客运站规划中“三主两辅”中的三个主要客运站之一，也是目前重庆使用中规模最大的铁路客运站。隶属[[成都铁路局|成都铁路局]]重庆车站管辖的[[特等站_(中国大陆)|特等站]]。

==名称==
由于位于[[沙坪坝区|沙坪坝区]]的[[沙坪坝站|沙坪坝站]]在此站启用之前原名为“重庆北站”。为避免市民将这两个车站混淆，在该重庆北站启用之前，位于沙坪坝区的老“重庆北站”更名为建站之初的“沙坪坝站”，然而此后仍有出租车司机因习惯将沙坪坝站称为北站而将欲前往龙头寺的乘客误载至沙坪坝站。<ref name="mmp_spb">{{cite news|author=蒋艳|url=http://cqcbepaper.cqnews.net/cqcb/html/2009-10/15/content_1005924.htm|title=重庆几个火车站取名混乱太误人|publisher=[[重庆晨报|重庆晨报]]|date=2009-10-15|accessdate=2016-05-06|deadurl=yes|archiveurl=https://web.archive.org/web/20110222015013/http://cqcbepaper.cqnews.net/cqcb/html/2009-10/15/content_1005924.htm|archivedate=2011-02-22}}</ref>

重庆北站北广场于2015年启用后，南北广场一带的地名也对乘客造成了困扰（见[[重庆北站#建设规划和设计存在的问题|建设规划和设计存在的问题]]一节）。<ref name="mmp1"/>

== 历史 ==
*2005年12月5日，龙头寺火车站竣工。
*2006年9月25日，龙头寺火车站正式命名为重庆北站。
*2006年10月22日，重庆北站投入客运。
*2008年9月，重庆北站开工新建第二售票厅。
*2009年1月16日，重庆北站开工新建第二售票厅投入使用。
*2009年9月，重庆北站原售票亭改造为和谐号动车组候车室。
*2011年1月，重庆北站增建第二售票处投入使用。
*2011年2月，在原动车组候车室外新建“动二候车室”，解决动车组候车室面积过小的问题。
*2011年2月，在第一候车厅出入口处进行扩建工程，解决第一候车厅面积过小的问题。
*2011年5月8日，重庆北站新建临时站台5站台投入使用。
*2012年8月，重庆北站北场扩建工程土石方工程开工。
*2012年12月，第一候车厅再次进行入口改造，以增加候车区域面积。
*2013年下半年，重庆北站北场扩建工程全面开工，将建设新的候车大楼、站台、站前广场等。
*2013年12月，在原候车大楼东南侧，增建临时第三候车厅，以缓解随着渝利铁路的开通导致的原候车区域面积过小的问题。
*2015年1月1日，重庆北站北场（渝利场部分）正式投用，继续建设渝万场部分。

== 车站规模 ==
重庆北站由渝怀场、渝利场和渝万场三部分组成，目前已建成渝怀场4座标准站台、1座临时站台，共计9条到发线，用于普通列车的接发车作业。2015年1月1日新建成投用渝利场5座站台，共计9条到发线，用作动车组列车和经由渝利铁路运营的普通列车的接发车作业。目前仍在建设中的部分为渝万场部分站台和站房。待渝万场部分建设完毕后，将封闭改造既有南场（渝怀场）站台和站房。预计全部改扩建工程完工后，重庆北站将成为拥有14座站台，26条到发线的特大型线上式布局客运站。
===南场===
重庆北站南场，即渝怀场，已建成4座标准站台、1座临时站台，共计9条到发线，用于普通列车的接发车作业。­

===北场===
重庆北站北场扩建工程于2012年6月开始土石方工程建设，于2013年年底全面开工建设。2015年1月1日新建成投用渝利场5座站台，共计9条到发线，用作动车组列车和经由渝利铁路运营的普通列车的接发车作业。<ref name="mmp_namen"/>目前仍在建设中的部分为渝万场部分站台和站房。<ref>[http://www.cq.gov.cn/publicmail/citizen/ViewReleaseMail.aspx?intReleaseID=166298 重庆火车北站北场的设计按什么规模来打造、是否已完成、何时开建？]</ref><ref>[http://www.cnnz.gov.cn/bbxq_content/2009-08/19/content_329225.htm 龙头寺综合换乘枢纽将开建 去重庆北站可坐轻轨] {{webarchive|url=https://web.archive.org/web/20140413142830/http://www.cnnz.gov.cn/bbxq_content/2009-08/19/content_329225.htm |date=2014-04-13 }}</ref><ref>[http://www.cq.gov.cn/publicmail/citizen/ViewReleaseMail.aspx?intReleaseID=232046 重庆北站北场规划设计问题]</ref>。

== 使用情况 ==
重庆北站目前办理渝利、渝怀、遂渝和襄渝方向的旅客列车到发任务。2015年1月1日开通的兰渝铁路重庆至合川段的列车也在此接发。此外，[[成渝客运专线|成渝客运专线]]因{{站|沙坪坝}}、菜园坝两站尚未完工，列车在[[璧山站|璧山站]]东侧的半边山线路所转入井西联络线（沿途有芭蕉沟线路所、[[歌乐山站|歌乐山站]]），随后经[[井口站|井口站]]向东接入重庆北站渝万场。<ref>{{cite web|url=http://cq.xinhuanet.com/2015-12/15/c_1117459731.htm|title=成渝高铁重庆沿线高铁站基本建成 部分公交已开行|publisher=[[新华网|新华网]]重庆频道|date=2015-12-15|accessdate=2016-11-08|deadurl=yes|archiveurl=https://web.archive.org/web/20170305035006/http://cq.xinhuanet.com/2015-12/15/c_1117459731.htm|archivedate=2017-03-05}}</ref>

== 质疑 ==
===候车厅===
重庆北站南场原有候车厅规模较小，无法满足乘客需求，后经历了多次临时性扩建工程，如增建临时第三候车厅等。车站也不得不采取“2小时内发车车次才能进站候车”的特殊措施，但给乘客造成了不便。

重庆北站北场（渝利场部分）候车楼于2015年1月投入使用，一定程度上缓解了重庆北站候车面积过小的局面。

===售票厅===
重庆北站南场在建成投入使用时的售票厅仅有16个售票窗口，面积较小，无法满足本站购票客流需求，被人们戏称为“售票室”，更有重庆本地媒体使用“售票亭”这一调侃称呼。鉴于此情况，重庆北站南场在2008年9月动工新建了1700平米的第二售票厅，增建了28个售票窗口以减轻售票压力，使重庆北站南场的售票窗口共达到44个。<ref>{{cite news|url=http://news.sina.com.cn/o/2008-07-04/033114113272s.shtml|title=火车北站九月拟建第二售票厅|publisher=[[重庆晚报|重庆晚报]]|date=2008-07-04|accessdate=2015-10-14}}</ref>在2009年9月遂渝铁路开通之时，原售票厅被改造为和谐号动车组候车室，重庆北站南场仅剩新建的第二售票厅，售票窗口28个。为满足旅客订票需求，2011年1月，重庆北站南场再次新建第二售票处，第二售票处位于主候车厅和动车候车厅之间，面积556平方米，共有23个售票窗口。目前，重庆北站南场共计售票厅和第二售票处两处售票地点，售票窗口共计51个，并在售票厅东侧设有若干台自助取售票机。

重庆北站北场（渝利场部分）于2015年1月1日开通运营，北场站房站厅层入口处左右两侧各设有一个售票厅，另一处售票厅设在负一层，每处售票厅设有20多个人工售票窗口、10台自助售票机和8台取票机，基本能满足乘客需求。

===建设规划和设计存在的问题===
[[File:Line_4_Entrance_in_North_Square_of_Chongqingbei_Railway_Station.jpg|thumb]]的交通核，连接[[重庆轨道交通4号线|重庆轨道交通4号线]]、[[重庆轨道交通10号线|10号线]]。[[重庆北站南广场站|重庆北站南广场]]连接[[重庆轨道交通3号线|重庆轨道交通3号线]]、[[重庆轨道交通10号线|10号线]]和[[重庆轨道交通环线|环线]]。]]
由于相关部门在修建重庆北站时对重庆的社会经济发展速度预计严重不足，导致新建成的重庆北站南场在竣工不久便开始超负荷运转。投入使用后，又历经了新建售票大厅、改建第一候车厅出口、增设第二售票处、新建动车组第二候车室、增建临时5站台、新建第三候车厅等等“弥补性工程”。重庆北站在投用后短短几年内便“不堪重负”，引发市民对当初车站规划等的质疑。

重庆北站北场由于开工较晚，使得在渝利铁路开通，新增大量车次的情况下，南场已开始超负荷运转。原本“建成渝利场、渝万场站房后封闭南场改造”的方案不得不修改为“渝利场建成即投用，与南场分工协作”的方案，但这一方案导致了在一段时期内，重庆北站成为“[[De_facto|事实上]]的两个火车站”，渝利场站房开通运营初期也导致大量乘客因不知情或不熟悉交通情况而误车。<ref name="mmp1">{{cite news|author=张珺|url=http://www.cqrb.cn/html/cqrb/2015-01/02/002/content_80591.htm|title=重庆北站北广场昨起投运,部分旅客“找不着北” 3700多位旅客错过列车|publisher=[[重庆日报|重庆日报]]|date=2015-01-02|accessdate=2015-10-14}}{{dead link|date=2018年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref name="mmp2">{{cite news|url=http://cq.sina.com.cn/news/b/2015-01-02/detail-icesifvy2494478.shtml|title=坐动车高铁都是到北站房哦 昨日3700多人误车|publisher=重庆商报|date=2015-01-02|accessdate=2015-10-14}}</ref>

重庆北站周边的接驳交通站点包括服务于南广场的[[重庆北站南广场站|重庆北站南广场站]]和重庆北站南广场汽车站，以及北广场一带的[[龙头寺站|龙头寺站]]和重庆北站北广场汽车站。此外，渝北区红旗河沟还设有与重庆火车北站相隔6公里的重庆汽车北站，这些站点的名称极易被混淆，因此常有乘客搭错车、下错站，并最终误车，这种站名混乱的现象也被重庆市民调侃为“[[哥德巴赫猜想|哥德巴赫猜想]]”。<ref name="mmp_namen">{{cite news|url=http://cq.cqwb.com.cn/sh/2015-12/30/content_345071.htm|title=乘客吐槽火车北站站名像绕口令 体验：搞清楚并不难|publisher=[[重庆晚报|重庆晚报]]|date=2015-12-30|accessdate=2016-01-28|deadurl=yes|archiveurl=https://web.archive.org/web/20160204095646/http://cq.cqwb.com.cn/sh/2015-12/30/content_345071.htm|archivedate=2016-02-04}}</ref>为此，重庆市政协委员李耀于2016年1月建议重庆市相关部门统一对龙头寺地区火车站、汽车站、轨道车站的站名进行规范，并在周边建立更加清晰的导向标识。<ref name="suggest124">{{cite news|url=http://cq.people.com.cn/n2/2016/0124/c365401-27615253.html|title=重庆龙头寺车站站名混乱 委员建议重新改名并规范|publisher=[[人民网|人民网]]|date=2016-01-24|accessdate=2016-01-28}}</ref>2016年4月7日，重庆市交委称重庆北站周边公交车、轨道、长途汽车站、出租车站、停车场的更名工作将于4月30日前完成。其中，轨道交通3号线“重庆北站”更名为[[重庆北站南广场站|重庆北站南广场站]]，在建的轨道4号线和10号线在重庆北站北广场设[[重庆北站北广场站|重庆北站北广场站]]，龙头寺站不更名。位于红旗河沟的“重庆汽车北站”则更名为重庆红旗河沟汽车站。<ref name="mp_update">{{cite news|url=http://cq.cqnews.net/html/2016-04/07/content_36655343.htm|title=快讯：重庆北站铁路综合交通枢纽站场更名 来看有哪些变化|publisher=华龙网|date=2016-04-07|accessdate=2016-04-09}}</ref>

2016年6月，重庆北站新建一条建一条临时地下通道和一座临时天桥，这两个新建设施均位于付费区域，预计于2017年1月竣工，以一定程度上连通南广场和北广场。新建的临时通道位于重庆北站渝万场9站台下方，全长约126.64米、宽约5.7米、高约3.3米；通道成都端连接既有渝怀场出站地道，怀化端连接北广场的城市通廊。临时天桥位于渝万场9站台高架站房至渝怀场11站台既有天桥间，天桥长约11米、宽约8米。<ref>{{cite news|title=从重庆北站北广场步行到南广场 明年仅需10分钟|url=http://weibo.com/ttarticle/p/show?id=2309613998852989494683|publisher=华龙网|date=2016-07-19}}</ref>
