'''重庆轨道交通（Chongqing Rail Transit）'''是[[中华人民共和国|中华人民共和国]][[重庆市|重庆市]]的[[城市轨道交通|城市轨道交通]]系统，由重庆市轨道交通（集团）有限公司运营。目前重庆轨道交通已建成或建设中的线路采用以下两种制式：[[单轨铁路|跨座式单轨]]：[[重庆轨道交通2号线|2号线]]、[[重庆轨道交通3号线|3号线]]。[[地铁|地铁]]（即传统钢轮钢轨铁路）：[[重庆轨道交通1号线|1号线]]、[[重庆轨道交通4号线|4号线]]、[[重庆轨道交通5号线|5号线]]、[[重庆轨道交通6号线|6号线]]和[[重庆轨道交通10号线|10号线]]和[[重庆轨道交通环线|环线]]，以及建设中的[[重慶軌道交通9號線|9號線]]。两种制式在建设成本，适宜环境，噪音控制，速度运力等方面各有不同。

于2004年11月6日開始載客运营的[[重庆轨道交通2号线|重庆轨道交通2号线]]，是中国首条[[单轨铁路|跨座式单轨]]线路，也是西部地区第一条[[城市轨道交通|城市轨道交通线路]]。2号线用40年的271亿日本政府提供的[[日本政府发展援助|政府间发展援助]]（ODA）日元贷款建设。

[[重庆轨道交通1号线|重庆轨道交通1号线]]是山城重庆开通运营的第一条钢轨轨道交通线路，[[重庆轨道交通3号线|重庆轨道交通3号线]]是中国第一条跨越[[长江|长江]]的城市轨道交通线。

目前开通有1号线、2号线、3号线、4号线、5号线、6号线、10号线、环线及相关支线，而9号线则正在建设中。以运营里程计算，重庆轨道交通目前是中国第七大城市轨道交通系统，2018年底数据显示，运营里程已达310公里<ref>{{Cite web |url= http://www.cqmetro.cn/wwwroot_release/crtweb/xwzx/zhxw/337181.shtml |title=轨道交通5号线一期北段、10号线一期12月28日开通试运营}}</ref>，居西部城市首位，中国第七位。全网日均客运量201万人次，日均开行班次2205次，其中单日最高客运量339.0万人次（2019年4月4日）。<ref>{{Cite web |url= http://weibo.com/2152519810/HobCneSAL |title=4月4日重庆轨道交通线网客运量_@重庆轨道交通}}</ref>

== 历史 ==
=== 早期规划 ===
<ref>《直辖十年重庆城市交通规划与实践》，蒋勇主编，重庆大学出版社, 2007。{{Zh-hans}}</ref>
[[File:Chongqing_Metro_Evolution.gif|thumb]]
*民国三十五年（1946年），[[国民政府|国民政府]]拟订《[[陪都十年建设计划草案|陪都十年建设计划草案]]》，提出建设高速电车。拟建三条高速电车线，建设标准采用每公尺重47.77公斤的钢轨；轨距为1公尺；最大坡度9%；弯度最小半径80米；主城区车速25公里/小时，郊区45公里/小时；车辆长8公尺，宽1.8公尺，双电机，每台分别为35马力；带拖车，每列乘240人；初定10分钟发一班，三线每天总运量约10万人次，部分线路在地下。
**甲线：从龙门浩到磁器口，途经龙门浩、小什字、较场口、七星岗、两路口、李子坝、小龙坎、沙坪坝、磁器口9座车站，全长14.75公里
**乙线：由龙门浩到南温泉，途经龙门浩、烟雨堡(距海棠溪1公里)、大坪溪、二塘、董家湾、刘家湾、南温泉7座车站，全长19.49公里
**丙线：由龙门浩到大田坎。途经龙门浩、石坎、大田坎3座车站，全长6.9公里
* 1960年：《重庆市城市初步规划》（1960一1980年）：提出「直通与环状」线网布局的「地下快速铁道线网」，以市中区为核心，将新牌坊、小龙坎、杨家坪、石桥铺、两路口等地区串联起来，总长约100公里。
* 1983年：《重庆市城市总体规划》（1982—2000年）：「地下铁道」，由朝天门-杨家坪，沿途经过较场口、菜园坝、两路口、鹅岭、大坪、谢家湾，全长约12.2公里。现状的[[重庆轨道交通二号线|重庆轨道交通二号线]]走向与这个线路基本一致。该规划提出，菜园坝火车站改建后，由梨树湾经沙坪坝至菜园坝的铁路联络线开行城市列车，发挥轨道交通功能。
* 1991年：《重庆市综合交通规划》（1990—2005年）：该规划提出了南坪-新牌坊、朝天门-双碑、朝天门-九宫庙3条干线轻轨，以及杨家坪-石桥铺轨道连接线的规划方案，线网总长约55千米。目前的轨道交通一、二、三号线走向与其基本一致。该规划基本确定了目前轨道线网的骨架雏形。并对轨道线路的断面流量进行了预测。
** 朝沙线为朝天门-较场口-七星岗-两路口-大坪-石桥铺-沙坪坝(远期延伸至双碑)；
** 朝新线为朝天门-较场口-七星岗-两路口-大坪-杨家坪-马王场-新山村(远期延伸至九宫庙)；
** 新牌坊-南坪空客线(适时延伸至四公里)；
** 杨家坪-石桥铺连网线。
* 1998年：《重庆市城市总体规划》（1996—2020年）：在1991年综合交通规划基础上，保留了朝沙线，调整了朝新线半岛地区走向，将新牌坊-南坪线北向延伸至江北机场，将杨家坪一石桥铺连网线东向延伸至四公里，增加五号线由童家院子-冉家坝-高家花园大桥-杨公桥-上桥-中梁山。轨道线路总长约119千米，线网密度约0.36公里/平方公里。
* 2003年《重庆市主城区综合交通规划》（2002—2020年）和2007年《重庆市城市总体规划》（2005—2020年），主城区轨道交通线网格局扩展为「九线一环」，此线网骨架承袭了1998年总体规划轨道线网结构，总长约522公里，线网密度约0.66公里/平方公里。

=== 早期建设与试验 ===
*1958年下半年，“重庆市市中区地下铁道工程队”成立，拟对原有[[防空洞|防空洞]]进行拓宽、加固，并新挖从十八梯到临江门的地下隧道，工程在进行了一年多后停滞。
*1960年，重庆市总体规划提出建设“直通与环状”线网布局的“地下快速铁道线网”，以市中区为核心，将新牌坊、小龙坎、杨家坪、石桥铺、两路口等地区连接起来。
*1965年，再次成立“重庆市中区地下铁道工程处”，工程处共分4个工程队，共有1000余人，1966年下半年，工程再一次瘫痪。但工程基本完成了市中区的地下隧道网：千厮门、望龙门、中华路、十八梯、兴隆街、枣子岚桠、燕喜洞部分贯通。至1971年，工程处被迫停工解散，修建的隧道移交人防部门。
*1983年，总体规划提出建设朝天门到杨家坪的地下铁道，途经较场口、菜园坝、两路口、鹅岭、大坪、谢家湾，全长约12.2公里。
*1988年，曾引入外地商人组建地铁集团投资修建地铁，公司设两路口，并曾开工建设，主要仍是在原人防工事基础上扩展施工。
*轨道交通1号线（B型地铁）、轨道交通2号线（跨座式单轨）的城区部分利用了大量60—80年代修建的防空隧道和地铁隧道。

== 分类 ==

'''单轨：'''重庆轨道交通2、3号线为单轨（即[[跨座式单轨|跨座式单轨]]）制式的城市轨道交通，习惯上又被市民称为[[轻轨|轻轨]]，但并非准确意义上的轻轨，从路权标准来看，单轨拥有完全封闭的专有路权，而从小时最大运送能力来看，属中大量运送能力轨道交通，因此不符合国际上通用的轻轨的定义，而属于地铁。

'''地铁：'''重庆轨道交通1、5、6、10号线，以及正在建设中的环线、4号线、9号线均属于传统的钢轮钢轨[[地铁|地铁]]制式的城市轨道交通。

{{参见|重庆市郊铁路}}

'''都市快轨：'''重庆还在主城区至璧山、铜梁、合川、江津等城市间之间规划建设一种新型轨道交通，被称为重庆都市快轨（即[[通勤铁路|通勤铁路]]）<ref>{{Cite web|url=http://news.163.com/17/0706/17/COM7LJD100018AOR.html|title=好消息！以后主城市民到区县更方便了，可选择都市快轨|accessdate=|author=|date=|last=|work=|publisher=}}</ref>。都市快轨将采用钢轮钢轨制式，采用双流制都市快轨列车<ref>{{Cite web|url=http://cupta.org/newsx.php?id=24871|title=中国土木工程学会城市公共交通学会——【重庆】轨道交通实行互通互联 都市快轨接入地铁网络|accessdate=|author=|date=|work=|publisher=|language=}}</ref>，能够兼容地铁和国铁两种受电模式，速度比地铁更快，站距也比市内地铁车站站距更大，运营方为重庆铁路集团。

====第二轮建设项目建成后重庆市轨道交通线网构成表<ref name="ReferenceA">重庆市城市快速轨道交通第二轮建设规划（2012~2020）及线网规划环境影响报告书（简本）{{Zh-hans}}</ref>====

==== 1号线 ====
[[File:CRT1_Logo.png|thumb]]

{{main|重庆轨道交通1号线}}

标识色：<font color={{重庆轨道交通标识色|1}}>■</font>石榴红

1号线是重庆市建成投入使用的首条地铁线路，开启了重庆轨道交通的换乘时代。

1号线线路识别色为石榴红色，采用B型[[地铁|地铁]]制式，一期工程为[[朝天门|朝天门]]至[[沙坪坝|沙坪坝]]，长16.5公里，二期工程为[[沙坪坝|沙坪坝]]至[[大学城|大学城]]，长20.2公里。一期工程于2007年6月开工，于2011年7月28日通车，二期工程于2012年12月20日建成通车。目前规划1号线全长约50公里，车站28座。续建的大学城站至尖顶坡站区间，向西延伸约2公里，已于2014年底建成通车。向西延伸至[[璧山区|璧山]]城区的区间也在建设中，该区间长7.52公里。

==== 2号线 ====
[[File:CRT2_Logo.png|thumb]]
{{main|重庆轨道交通2号线}}

标识色：<font color={{重庆轨道交通标识色|2}}>■</font>森林绿

2号线全长31.2公里，是国内首条采用[[跨座式单轨|跨座式单轨]]制式的轨道交通线路，也是西部第一条[[城市轨道交通|城市轨道交通]]线路。

2号线线路识别色为森林绿，一期工程[[较场口|较场口]]至[[重庆动物园|动物园]]段长14.3公里，2004年11月开始试运行，2005年6月18日（[[重庆市|重庆]][[直辖市|直辖]]8周年纪念日）正式载客运营；二期[[重庆动物园|动物园]]至[[新山村|新山村]]段2006年7月投入运营，一二期贯通线路全长19.1公里。南延伸段延伸至[[鱼洞|鱼洞]]，长12.1公里，设车站7座，已于2014年底建成通车。

== 车票 ==
重庆轨道交通实行里程计价、递远递减的计程票价，单轨与地铁票面通用。乘客从进入付费区开始，须在180分钟内搭乘完地铁，否则车票作超时作废处理，并按最高单程票价（2012年12月更新为7元）付费买票。在180分钟内，按照乘坐里程计算票价。最低票价2元，最高票价10元。

车费可使用单程票和宜居畅通卡付费，其中宜居畅通卡包括普通卡、优惠卡、学生卡、敬老卡、爱心卡和限量发行的重庆轨道交通季节或线路开通纪念票。

=== 重庆宜居畅通卡===
[[宜居畅通卡|宜居畅通卡]]是重庆市区内使用的具有多种付费功能的非接触式IC卡，可以在重庆轨道交通、公交、索道、扶梯等多种公共交通场合使用。持有[[宜居畅通卡|宜居畅通卡]]乘坐交通工具（包含轨道交通、近500条公交线路、索道、两路口大扶梯——连接了两路口和重庆火车南站的手扶电梯）时，享受票价9折优惠以及一小时免费/优惠换乘。

=== 单程票 ===
单程票适用于临时乘坐地铁的乘客，可使用站厅中设置的自动售票机购买。单程票仅限在车票发售当天有效，一次性使用，乘客出站时单程票将通过闸机进行回收。

==== 退票 ====
仅限当日当站未使用车票可去服务中心退票，如遇列车故障15分钟以上也可以办理退票，并可索取致歉信。

=== 一日票 ===
一日票适用于短期内需多次乘坐轨道交通的外地游客，旅游票特性显著，票价为18元/张。乘客使用自首次在进站检票闸机上刷卡开始计算，之后24小时内不限次数乘坐重庆轨道交通的所有线路。<ref>{{Cite web |title=重庆轨道交通“一日票”发行通告 |url=http://www.cqmetro.cn/wwwroot_release/crtweb/xwzx/gsgg/331037.shtml |accessdate=2013-12-20 |work= |publisher=重庆市轨道交通（集团）有限公司 |date=2013-12-17 |language=zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20131220083614/http://www.cqmetro.cn/wwwroot_release/crtweb/xwzx/gsgg/331037.shtml |archivedate=2013-12-20 }}</ref>

* 2011年12月19日，延伸段九公里道岔桥工地发生事故，工地脚手架被一辆客车撞击，使正在浇筑的桥墩垮塌，导致1人死亡，5人受伤。<ref name=事故>{{cite news|title=重庆客车撞垮在建轻轨桥墩已致1死5伤|url=http://news.sina.com.cn/c/2011-12-19/220523658276.shtml|accessdate=2011-12-20|newspaper=新华网}}</ref>
* 2016年2月18日，重庆轨道交通3号线一列车的乘客物品发生爆炸。经轨道分局干警在现场确认，爆炸物为乘客携带的蓄电池。事故没有造成人员伤亡。<ref>[http://news.163.com/16/0218/19/BG4LNOKJ00011229.html 重庆轨道3号线一车厢内发生爆炸 爆炸物为乘客携带的蓄电池]</ref>
