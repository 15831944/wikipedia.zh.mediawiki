'''拉玛尔·帕特森'''（{{lang-en|'''Lamar Patterson'''}}，{{bd|1991年|8月18日|}}），美国职业[[篮球|篮球]]运动员，目前于[[NBA|NBA]]的[[亚特兰大老鹰队|亚特兰大老鹰队]]效力。大学时期他曾是[[匹兹堡大学|匹兹堡大学]]的一名队员。

== 大学生涯 ==
来自[[宾夕法尼亚州|宾夕法尼亚州]][[兰开斯特_(宾夕法尼亚州)|兰开斯特]]的帕特森是一名身高6英尺5英寸（大约196公分）的[[锋卫摇摆人|锋卫摇摆人]]。高中时期，起初帕特森为兰开斯特当地的高中J. P. McCaskey High School打球，之后他转学到了新泽西州[[纽瓦克|纽瓦克]]的Saint Benedict's Preparatory School打球。高中毕业后，他选择了为[[:en:Pittsburgh_Panthers_men's_basketball|匹兹堡大学黑豹队]]打球。大一赛季，在2009年9月对阵[[卫奇塔州立大学|卫奇塔州立大学]]的一场比赛中帕特森脚踝受伤而不得不以[[:en:Medical_redshirt|伤病红衫]]球员的身份错过了那个赛季的大部分比赛<ref>{{cite web |last = Jones|first = Gordie|title = McCaskey product Lamar Patterson redshirting at Pitt|work = [[:en:Lancaster_Online|Lancaster Online]]|date = March 11, 2010|url = http://lancasteronline.com/sports/mccaskey-product-lamar-patterson-redshirting-at-pitt/article_b67eae62-2baa-5d0b-85e0-fc68e60a39c9.html|accessdate = April 13, 2014}}</ref>。帕特森在[[:en:2010–11_NCAA_Division_I_men's_basketball_season|2010-11赛季]]回归球场，并成为了黑豹队轮换阵容中重要的一员。2011-12赛季中，作为一名红衫大二生，帕特森成功地成为了一名球队首发并且把他的场均得分数从2.6提升至了9.6。随着他带领球队赢下了[[:en:2012_College_Basketball_Invitational|2012年大学篮球邀请赛]]冠军，他高调地完成了那个赛季。在邀请赛中他场均得到13.3分，6个篮板和4.5次助攻并且成为了那届邀请赛的[[最有價值球員|最有价值球员]]<ref>{{cite web |last = Fittipaldo|first = Ray|title = Pitt's season ends on high note with CBI win|work = [[:en:Pittsburgh_Post-Gazette|Pittsburgh Post-Gazette]]|date = March 31, 2012|url = http://www.post-gazette.com/home/2012/03/31/Pitt-s-season-ends-on-high-note-with-CBI-win/stories/201203310122|accessdate = April 13, 2014}}</ref>。

大三赛季，帕特森成功帮助了匹兹堡大学重回NCAA锦标赛（英文：[[:en:NCAA_Division_I_Men's_Basketball_Tournament|NCAA Division I Men's Basketball Tournament]]）。然而，在锦标赛中匹兹堡大学被后来打入四强的卫奇塔州立大学淘汰。大四时，帕特森选择了继续为匹兹堡大学效力，而也就是在这个赛季匹兹堡大学黑豹队从大东联盟（英文：[[:en:Big_East_Conference|Big East Conference]]）搬到了[[大西洋沿岸联盟|大西洋沿岸联盟]]（英文：[[:en:Atlantic_Coast_Conference|Atlantic Coast Conference]]）。帕特森在这个赛季中大放异彩，把他的场均得分数从大三时的10.0分提升到了17.1分<ref>{{cite web |last = DeCourcy|first = Mike|title = Pitt's Lamar Patterson: Focus on performance, not Wooden snub|work = [[:en:Sporting_News|Sporting News]]|date = January 26, 2014|url = http://www.sportingnews.com/ncaa-basketball/story/2014-01-26/pitts-lamar-patterson-focus-on-performance-not-wooden-snub|accessdate = April 13, 2014}}</ref>。在赛季接近尾声时，他被选入了联盟All-ACC二队<ref>{{cite web |last = Zeise|first = Paul|title = Pitt standout Lamar Patterson named second-team All-ACC|work = [[:en:Pittsburgh_Post-Gazette|Pittsburgh Post-Gazette]]|date = March 10, 2014|url = http://www.post-gazette.com/sports/Pitt/2014/03/10/Patterson-named-second-team-All-ACC/stories/201403100196|accessdate = April 13, 2014}}</ref>。

== 职业生涯 ==
2014年6月26日，帕特森在[[2014年NBA選秀|2014年NBA选秀]]中被[[密尔沃基雄鹿队|密尔沃基雄鹿队]]用48号签选中。选秀日当晚他被交易至了[[亚特兰大老鹰队|亚特兰大老鹰队]]<ref>[http://www.nba.com/hawks/news/hawks-trade-get-lamar-patterson Hawks Trade To Get Lamar Patterson]</ref><ref>[http://www.sbnation.com/nba/2014/6/26/5848272/lamar-patterson-milwaukee-bucks-nba-draft-results-2014 2014 NBA Draft Results: Atlanta Hawks acquire 48th pick Lamar Patterson]</ref>。2014年7月，他代表老鹰队参加了2014年[[NBA夏季联赛|NBA夏季联赛]]<ref>[http://www.nba.com/hawks/2014-atlanta-hawks-summer-league-roster 2014 Atlanta Hawks Summer League Roster]</ref>。2014年8月5日，帕特森与土耳其的Tofaş Bursa队签约<ref>[http://www.sportando.com/en/europe/turkey/130255/tofas-bursa-lands-rookie-lamar-patterson.html Tofas Bursa lands rookie Lamar Patterson]</ref>。2014-15赛季，在为Tofaş效力的29场比赛中，帕特森场均得到11.2分，3.4篮板，2.2次助攻和1.0次抢断。

2015年7月24日，由于在之前参加的七场[[:en:2015_NBA_Summer_League|2015NBA夏季联赛]]比赛中场均能得到13.1分，5.1篮板，3.4次助攻和1.6次助攻，亚特兰大老鹰队与帕特森签下了一份合同。<ref>[http://www.nba.com/hawks/news/hawks-sign-lamar-patterson-terran-petteway Hawks Sign Lamar Patterson, Terran Petteway]</ref>
