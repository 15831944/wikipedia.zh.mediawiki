[[File:Tensile_testing_on_a_coir_composite.jpg|缩略图]]
'''拉伸试验'''（{{lang-en|Tensile testing}} 或 Tension testing）<ref>{{Cite book|last=Czichos|first=Horst|title=Springer Handbook of Materials Measurement Methods|publisher=Springer|location=Berlin|year=2006|pages=303–304|url=https://books.google.com/books?id=8lANaR-Pqi4C|isbn=978-3-540-20785-6}}</ref>是[[材料科学|材料科学]]和[[工程学|工程]]中一種基本物質[[張力|张力]]測試。藉由拉伸试验測量的性質有：[[强度|極限抗拉强度]]、[[斷裂|断裂强度]]、最大[[形變|伸长量]]和斷面縮率<ref name="davis1">{{Cite book|last=Davis|first=Joseph R.|title=Tensile testing|publisher=ASM International|year=2004|edition=2nd|url=https://books.google.com/books?id=5uRIb3emLY8C|isbn=978-0-87170-806-9}}</ref>。下列特性也可以被檢測：[[杨氏模量|杨氏模量]]、[[泊松比|泊松比]]、[[屈服|屈服强度]]、[[Strain-hardening|加工硬化]]<ref>{{Harvnb|Davis|2004}}.</ref>。单轴拉伸试验是常用於获得[[各向同性|等向性]]材料的机械特性。某些材料是使用[[Plane_biaxial_tensile_test|双軸拉伸试验]]。

== 目的 ==
使用拉伸试验可能具有各种目的，例如：
* 為了應用而选择特定材料或物品
* 预测材料在使用的正常和极端[[力|施力]]的型態。
* 確認是否符合[[Specification|规范]]、[[行政法规|法规]]或[[契约|合同]]的要求。
* 决定[[新產品開發|新产品开发]]是否有望成功：[[概念验证|概念证明]]。
* 证明提出[[专利|专利]]的实用性。
* 为其他科学、工程和[[品質保證|品質保证]]提供[[标准|标准]][[数据|数据]]。
* 提供基础的[[技术传播|技术通信]]。
* 用於比较几种选择的方法。
* 提供诉讼程序的[[Evidence|证据]]。

== 拉伸試件 ==
[[File:Tensile_specimen-round_and_flat.jpg|缩略图]]
如何准备試件取决于测试目的、[[Test_method|试验方法]]或[[Specification|规范]]。試件通常是一个标准化的截面。試片分為肩部及標距（中央部分）：肩部較大，以容易使機器夾持；中央部分具有较小横截面，故变形和破壞可在这个区域發生<ref name="davis1">{{Cite book|last=Davis|first=Joseph R.|title=Tensile testing|publisher=ASM International|year=2004|edition=2nd|url=https://books.google.com/books?id=5uRIb3emLY8C|isbn=978-0-87170-806-9}}</ref><ref name="davis2">{{Harvnb|Davis|2004|p=2}}.</ref>。通过使用非常相似的試件，可以找到试验机的可重复性<ref name="davis8">{{Harvnb|Davis|2004|p=8}}.</ref>。[[File:Tensile_specimen_nomenclature.svg|缩略图]][[File:Tensile_specimen_shoulders.svg|缩略图]]

== 機器设备 ==
[[File:Inspekt_desk_50kN_IMGP8563.jpg|缩略图]]
拉伸试验最常於材料测试实验室進行。ASTM D638是最常见的拉伸试验標準。ASTM D638用於測量塑料的拉伸性質，包括極限抗拉强度，屈服强度，伸長量與泊松比。

最常见的测试机器的使用在拉伸试验是 ''[[Universal_testing_machine|萬能试验机]]''。这种类型的机器有两个''十字头''：一个是隨試件長度调整；另一個是對試件施力。有两种类型：[[液压机械|液压]]和[[電磁鐵|电磁]]驅動<ref name="davis2"/>。

机器必须有適合性能以用于测试試件。有四个主要参数：拉伸力、速度、[[準確與精密|精确度和准确性]]。拉伸力是指机器必须能够产生足够的力以拉断試。机器必须能够迅速或緩慢的施加負荷，以模仿实际应用。最后，机器必须能够准确地和精确地的测量长度和拉伸力：例如一个大型机器的目的是测量大的伸长率，可能無法用於脆性材料，因其破裂之前將只有短伸长率<ref name="davis9">{{Harvnb|Davis|2004|p=9}}.</ref>。

== 試驗过程 ==
测试的过程包括将試件放置於試驗机和缓慢延伸試件直到破裂。在此过程中，[[形變|伸长]]量的將與施力紀錄。数据已被設計好，因此將不隨几何形状而變。伸长量测是用工程应变ε，並使用以下等式<ref name="davis2"/>：

: <math>\varepsilon =\frac{\Delta L}{L_0}=\frac{L-L_0}{L_0}</math>

其中Δ''L'' 的標距变化的长度，''L''<sub>0</sub> 是初始標距的长度，和''L''是最终的標距长度。力的量測是用工程压力σ，使用以下等式<ref name="davis2"/>：

: <math>\sigma = \frac{F_n}{A}</math>

其中F为拉力和A为試件的標稱横截面。隨著施力變化，机器將做这些计算以绘制[[应力-应变曲线|压力–应变曲线]]<ref name="davis2"/>。
