'''拉斯佩尼亚斯'''（{{lang-es|Las Peñas}}）是[[瓜亚基尔|瓜亚基尔]]的标志性社区。它以殖民地建筑风格而闻名，是城市诞生的地方。它位于{{Link-en|圣安娜山|Cerro Santa Ana (Guayaquil)}}的山脚下，<ref name="1R">{{Cite web |url = http://www.telegrafo.com.ec/cultura1/item/las-penas-bajo-la-mirada-patrimonial.html |title = Las Peñas, bajo la mirada patrimonial |date =  |language = es |accessdate = 2019-02-28}}</ref>它的名字来源于十六世纪西班牙人在那里定居时山丘曾经拥有的悬崖。<ref name="2R">{{Cite web |url =http://www.eluniverso.com/2012/07/29/1/1380/las-penas-barrio-reminiscencia.html  |title = Las Peñas, un barrio con reminiscencia|date =  |language = es |accessdate = 2019-02-28 }}</ref>

最初在此居住的家庭主要是渔民和工匠，但因为20年代的可可热潮，该地区逐渐被富有的地主家庭占据，大部分房屋为贵族风格，就是今天所看到的景象。<ref name="1R"/><ref name="2R"/>

1982年，社区被宣布为[[厄瓜多尔|厄瓜多尔]]文化遗产，这是第三个国家文化遗产，<ref name="3R">{{Cite web |url = http://www.ppelverdadero.com.ec/mi-guayaquil/item/las-penas-cuna-de-heroes-y-refugio-de-artistas.html |title = Historia de mi barrio: Las Peñas, cuna de héroes y refugio de artistas|date =  |language = es |accessdate = 2019-02-28 }}</ref>
主要是在这一地区的住宅，有些是已有100多年历史的古迹。<ref name="4R">{{Cite web |url = http://www.andes.info.ec/es/turismo/penas-lugar-ensueno-recordar-guayaquil-antano.html |title = Las Peñas, lugar de ensueño para recordar al Guayaquil de antaño |date =  |language = es |accessdate =  }}</ref>

在2002年至2008年期间，城市市政部门在该地区进行了修缮和再生的过程。<ref name="2R"/>
