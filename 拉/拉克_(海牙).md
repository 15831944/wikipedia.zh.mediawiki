'''拉克'''（{{lang-nl|Laak}} {{IPA-nl|laːk}}）是[[荷蘭|荷蘭]][[海牙|海牙]]的一個行政區，擁有面積4.3平方公里和居民40,222人（2017年<ref name="haguepopulation" />），是海牙八個分區中腹地與人口規模最小的行政區。拉克區又分為三個區域：[[拉克瓦提爾|拉克瓦提爾]]（Laakkwartier）、[[史普爾維克|史普爾維克]]（Spoorwijk）和[[賓霍斯特|賓霍斯特]]（Binckhorst）<ref>{{cite web |url = http://www.denhaag.nl/home/bewoners/stadsdelen-en-wijken/laak.htm |title = Laak |website = Denhaag.nl |language = nl |access-date = 2018-06-21 |archive-url = https://web.archive.org/web/20161026115712/http://www.denhaag.nl/home/bewoners/stadsdelen-en-wijken/laak.htm |archive-date = 2016-10-26 |dead-url = yes }}</ref>。[[海牙大學|海牙大學]]的主校區即位在拉克。
