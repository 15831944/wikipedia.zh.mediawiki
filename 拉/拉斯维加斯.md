'''拉斯維加斯'''（{{lang-en|City of Las Vegas}}）是[[美國|美國]][[內華達州|內華達州]]人口最多的[[城市|城市]]，也是内華達州[[克拉克縣_(內華達州)|克拉克縣]]的[[縣治|縣治]]，有着以[[赌博业|赌博业]]为中心庞大的旅遊、購物、度假产业，是世界知名的度假地之一。2005年人口普查局的官方數字為59萬1536人。2005年，此城的都會區人口為191萬2654人<ref>[http://www.lasvegasnevada.gov/FactsStatistics/demographics.htm Demographics]</ref>。

== 历史 ==
拉斯維加斯建于1854年，由当时在美国西部的[[摩门教|摩门教]]徒建成，後來摩门教徒迁走，美国政府使其变成一个兵站。“拉斯維加斯”的名称是[[西班牙|西班牙]]人起的，意思是“肥沃的山谷”，因为拉斯維加斯是周围荒凉的[[沙漠|沙漠]]和半沙漠地带唯一有泉水的[[绿洲|绿洲]]，由于有泉水，逐渐成为来往公路的驿站和铁路的中转站。它开埠于1905年。[[内华达州|内华达州]]发现金银矿后，大量淘金者涌入，拉斯維加斯开始繁荣，但如同西部各采矿城镇一样，一旦矿被采光就会被抛弃。在1931年[[經濟大蕭條|經濟大蕭條]]时期，为了度过经济难关，[[内华达州|内华达州]]议会通过了赌博合法的议案，拉斯維加斯成为一个[[赌城|赌城]]，从此迅速崛起。同年兴建[[胡佛水坝|胡佛水坝]]，工程于1935年完成。

赌城拉斯维加斯在很长时间被[[黑手党|黑手党]]控制。1960年代末期亿万富翁[[霍华德·休斯|霍华德·休斯]]从[[黑手党|黑手党]]手中收购了很多资产，使拉斯维加斯摆脱了[[黑手党|黑手党]]的控制。

== 经济 ==
拉斯維加斯的主要经济支柱是[[賭博|博彩业]]，由于赌场是个淘金碗，美国各地的大亨纷纷向拉斯維加斯投资興建赌场，甚至[[日本|日本]]的富豪、[[阿拉伯|阿拉伯]]的王子、著名[[演员|演员]]均来投资。博彩业带动了旅游业、娱乐业的发展。为了使经济稳定发展，赌业公会实行严格的自律机制，对投资人进行严格的审查，对各赌场进行严格的监督；一旦发现问题，當事人將永遠從允許在當地經營赌业的名單中除名。为了吸引遊客，社会治安治理良好，对中了头奖的人，如果需要，由两名警察将其全程护送到在美国任何地方的家中。

雖然内华达州是美国唯一[[性交易|性交易]]合法的州，但惟独[[拉斯維加斯|拉斯維加斯]]所在的[[克拉克县|克拉克县]]以方便维持社会治安為由禁止性交易。各个赌场的設計以金碧辉煌、奇形怪状的建筑物來吸引游客，所有赌场24小时开业。机场的班机通往世界各地，任何私人飞机都可以在拉斯維加斯降落。虽然后来内华达州的其他城市如[[雷諾_(內華達州)|雷諾]]也发展赌博业，但拉斯維加斯憑著交通便利和接近大城市[[洛杉磯|洛杉磯]]不遠距離的地理優勢，仍然佔據着美國賭博業的頭把交椅。

拉斯維加斯蓬勃的賭博業帶動起了當地娛樂事業的發展，各國著名的歌舞團體及世界知名影星和歌星都以能登上拉斯維加斯的華麗大舞台而感到自豪，並因此吸引了眾多遊客進場欣賞其演出。1990年甚至[[唐人街|唐人街]]也在拉斯維加斯落户，很快成為[[亞洲裔美國人|亞洲裔美國人]]的聚集地，因為這些關係拉斯維加斯成為美國發展最迅速的城市之一，且當地賭博事業為該地區重要的經濟支柱。

拉斯維加斯的賭場主要集中在兩個區域。[[拉斯維加斯舊城|拉斯維加斯舊城]]（Las Vegas Downtown）是該市初期的發展原點，雖然曾經一度沒落，但近年來仍試圖透過許多街廓再造工作吸引觀光人潮回流；[[拉斯維加斯大道|拉斯維加斯大道]]（Las Vegas Strip）則是最新開幕的賭場與飯店匯聚之處，堪稱是賭城的門面。

== 地理 ==
拉斯維加斯地处内华达州被荒凉的沙漠和半沙漠地带包围的山谷地区，雨量很少，冬季和夏季的溫差相當大。受[[焚風|焚風]][[欽諾克風|欽諾克風]]影響，夏季極之酷热，最高溫度可達攝氏50度；冬季寒冷多风沙，偶然的暴雨会导致洪水。總面積293平方公里。

== 人口 ==
拉斯維加斯平均人口密度为每平方公里8547人，其中[[白人|白人]]34.7%，墨西哥裔33.6%,  [[黑人|黑人]]17.1%，[[亞裔|亞裔]]人10.3%，[[印第安人|印第安人]]和阿拉斯加當地人1.2%，除了在唐人街经商的黄种人外，在赌场中的发牌员有很多亞洲人。

== 教育 ==
小学和中学公共教育主要位于克拉克县学区，该学区的人口位居全国第五。

== 娱乐 ==
拉斯维加斯有68个公园。拉斯维加斯总共有四个高尔夫球场：天使公园高尔夫俱乐部，沙漠之松高尔夫俱乐部，杜兰戈山高尔夫俱乐部和拉斯维加斯市高尔夫球场。它还有123個遊樂場，23个垒球场，10个美式足球场，44个足球场，10个dog parks，6个社区中心，4个老人中心，109个冰鞋公园，6个游泳池等。
