'''拉格朗日力学'''（{{lang-en|Lagrangian mechanics}}）是[[分析力学|分析力学]]中的一种，于1788年由[[約瑟夫·拉格朗日|約瑟夫·拉格朗日]]所创立。拉格朗日力学是对[[经典力学|经典力学]]的一种的新的理论表述，着重于数学解析的方法，並運用[[最小作用量原理|最小作用量原理]]<ref>{{Cite book|last=Goldstein|first= H. |title=Classical Mechanics|edition=3rd| page=35 |publisher=Addison-Wesley|year= 2001}}</ref>，是[[分析力学|分析力学]]的重要组成部分。

经典力学最初的表述形式由[[艾萨克·牛顿|牛顿]]建立，它着重於分析[[位移|位移]]，[[速度|速度]]，[[加速度|加速度]]，[[力|力]]等[[矢量|矢量]]间的关系，又称为[[矢量力学|矢量力学]]。拉格朗日引入了[[广义坐标|广义坐标]]的概念，又运用[[达朗贝尔原理|达朗贝尔原理]]，求得与[[牛顿第二定律|牛顿第二定律]]等价的[[拉格朗日方程|拉格朗日方程]]。不仅如此，拉格朗日方程具有更普遍的意义，适用范围更广泛。还有，选取恰当的广义坐标，可以大大地简化拉格朗日方程的求解过程。

== 自由度 ==
{{main|自由度 (物理学)}}

力学系统可以由一组[[坐标|坐标]]来描述。例如，一个[[质点|质点]]的[[运动|运动]]（在[[笛卡尔坐标系|笛卡尔坐标系]]中）由x、y、z三个坐标来描述。一般而言，<math>N</math>个质点组成的力学系统由<math>3N</math>个坐标来描述。力学系统中常常存在着各种[[约束|约束]]，使得这<math>3N</math>个坐标并不都是独立的。力学系统的独立坐标的个数称之为'''自由度'''。对于<math>N</math>个质点组成的力学系统，若存在<math>m</math>个约束，则系统的自由度为
:<math>S=3N - m</math>。

== 广义坐标 ==
{{main|广义坐标}}

在[[矢量力学|矢量力学]]中，约束的存在体现于作用于系统的[[约束力|约束力]]。约束力引入额外的[[未知量|未知量]]，通常使问题变得更为复杂。但若能选取适当的<math>s</math>个完全满足约束条件的独立坐标，则约束不再出现在问题中，只需要求解关于<math>s</math>个未知变量的[[方程|方程]]，使问题得以大大简化。而如果运用牛顿力学来解约束问题，通常约束越多，需要求解的方程个数就越多，反而增加了一定的难度。这样的<math>s</math>个坐标不再局限于各质点的位置坐标，而可以是任何能描述系统的几何参量，因此称为“广义坐标”。

== 拉格朗日量 ==
{{main|拉格朗日量}}

拉格朗日力学的一个基本假设是：具有<math>n</math>个自由度的系统，其运动状态完全由<math>n</math>个[[广义坐标|广义坐标]]及[[广义速度|广义速度]]决定。或者说，力学系统的运动状态由一个广义坐标和广义速度的[[函数|函数]]描述：
:<math>\mathcal{L}(q_1,\ q_2,\ \dots , \ q_n;\ \dot{q_1},\ \dot{q_2},\ \dots ,\ \dot{q_n},\ t)</math>。

这个函数称为[[拉格朗日函数|拉格朗日函数]]或拉格朗日量。

引入势能函数<math>V \!</math><ref>{{cite book|title=理论力学简明教程|author=陈世民|publisher=高等教育出版社|isbn=978-7-04-023918-8|pages=185-186页}}</ref>。这时拉格朗日函数表示为：
:<math>\mathcal{L} = T - V</math>
其中<math>T \!</math>和<math>V \!</math>分别是这个力学体系的动能和势能。

== 拉格朗日方程 ==
{{main|拉格朗日方程}}

拉格朗日力学中，运动方程由<math>n</math>个[[二阶微分方程|二阶微分方程]]（拉格朗日方程）给出：
:<math>
{\mathrm{d}\over \mathrm{d}t}{\partial{\mathcal{L}}\over \partial{\dot{q_i}}} - {\partial{\mathcal{L}}\over \partial q_i} =Q_i
</math>；

其中<math>Q_i</math>为<math>q_i</math>所对应的非保守的广义力。
 
拉格朗日方程的地位等同于牛顿力学中的[[牛顿第二定律|牛顿第二定律]]。但具有更普遍的意义。

== 拉格朗日力学的扩展 ==
[[哈密顿力学|哈密顿量]]<math>H</math>可以通过对拉格朗日量进行[[勒壤得轉換|勒让德变换]]得到。哈密顿量是经典力学的另一种表述[[哈密顿力学|哈密顿力学]]的基础。拉格朗日量可以视为定义在所有广义坐标可能值组成的组态空间的[[切丛|切丛]]上的函数，而哈密顿量是相对应的[[余切丛|余切丛]]上的函数。哈密顿量在[[量子力学|量子力学]]中到处出现（参看[[哈密頓算符_(量子力學)|哈密頓算符 (量子力學)]]）。

1948年，[[费曼|费曼]]发明了[[路径积分表述|路径积分表述]]，将[[最小作用量原理|最小作用量原理]]扩展到[[量子力学|量子力学]]。在该表述中，粒子穿过所有可能的始态和终态的所有路径；特定终态的概率是所有可能导向它的轨迹的概率之和。在经典力学的范围，路径积分表述简单的退化为[[哈密顿原理|哈密顿原理]]。
