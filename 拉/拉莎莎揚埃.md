'''拉莎莎揚埃''' ( 馬來語、印尼語︰'''Rasa Sayange''' or '''Rasa Sayang''' ( 讀音為{{IPA|rasa sajaŋeː}}) 是[[馬來亞群島|馬來亞群島]]一帶的民歌。雖沒有官方記載，此歌相信是由安汶人創作的。

== 爭議 ==
2007年，[[馬來西亞|馬來西亞]]政府旅遊局播放「拉莎莎揚埃」廣告片，用以宣傳馬來西亞的[[旅遊|旅遊]]業。[[印尼|印尼]]人則指馬來西亞不公平獲得了歌曲拥有權。<ref>{{cite news|publisher=新海峽時報|accessdate=2008-01-21|title=雅加達 (政府) ︰用我們的歌前請通知我們|url=http://www.nst.com.my/Current_News/NST/Saturday/Frontpage/2130429/Article/index_html|date=2008年1月12日|deadurl=yes|archiveurl=https://web.archive.org/web/20080115083340/http://www.nst.com.my/Current_News/NST/Saturday/Frontpage/2130429/Article/index_html|archivedate=2008年1月15日}}</ref>一些印尼人則指責馬來西亞是文化遺產盜賊<ref>{{cite news|publisher=[[:en:Bernama|報南亞]]|accessdate=2008-01-21|title=印尼立法者指責馬來西亞偷曲行為|url=http://www.bernama.com/bernama/v3/printable.php?id=288075|date=2007年10月7日}}</ref>，因根據[[印尼|印尼]]版權法，歌曲係由國家拥有及控制，是國家的知識財產，故指他們侵權。馬來西亞政府反駁指，歌曲係由[[馬來亞群島|馬來亞群島]]人拥有，包括馬、印人民等。<ref>{{cite news|publisher=馬來西亞星報|accessdate=2008-01-21|title=部長︰拉莎莎揚埃屬於每個人|url=http://thestar.com.my/news/story.asp?file=/2008/1/12/nation/19988592&sec=nation|date=2008年1月12日}}</ref>

== 其它版本 ==
*最早粵語版本相信由林鳳主唱，後由[[劉珺兒|劉珺兒]]翻唱，歌名為「任你抱我」。
*另一版本為[[馬來西亞|馬來西亞]]網絡歌手[[黃明志|黃明志]]自編自導自演的電影《[[辣死你媽|辣死你媽]]》（Nasi Lemak 2.0）的電影主題曲，由黃明志和[[龔柯允|龔柯允]]主唱，歌名為「Rasa Sayang 2.0」。
*[[邓丽君|邓丽君]]在1970年翻唱了此歌曲的国语版本，收录在《邓丽君之歌第十五集》专辑中，歌名为“拉萨沙哟”。

== 事實 ==
* 當印尼還是荷蘭殖民地時，荷屬東印度政府已於二戰前採用此歌宣傳旅遊。
* 拉莎莎揚埃 ( Rasa Sayang Eh ) 是1959年馬來西亞影片。<ref>馬來西亞電影庫 [http://www.filemkita.com/filem/r/rasa_sayang_eh_01.html Rasa Sayang Eh (1959)].</ref> <ref>[http://www.youtube.com/watch?v=-3wuNi6qe2k 馬來亞相關電影]</ref>
* 1962年第四届亞運會，歌曲被刻到黑膠唱碟，是碟中印尼歌曲之一。
* 自1960年代開始，幼稚園生都開始有學習本歌，成為印尼流行民歌之一。
* 莎揚埃的“埃”字是安汶人的口音，用作辨別其他馬來亞群島的口音。
* 除馬、印二國外，此歌於[[新加坡|新加坡]]頗為流行。
