'''拉玛逊'''<ref name="hj">{{Cite web |url = https://th.hujiang.com/new/p626835/ |title = 台风“威马逊”取名字自泰国，你造啥意思咩？ |author = 沪江泰语 |date = 2014-07-23 |publisher =  |language = zh-cn |accessdate = 2017-09-05 |quote = }}</ref>（{{lang-th|รามสูร}}）是[[泰国|泰国]][[史诗|史诗]]《[[拉玛坚|拉玛坚]]》中的[[雷神|雷神]]，相当于中国的[[雷公|雷公]]。拉玛逊的原型是{{link-en|持斧罗摩|Parashurama}}，为[[印度教|印度教]]神祇[[毗湿奴|毗湿奴]]的化身之一。

== 传说 ==
[[媚卡拉|媚卡拉]]喜欢拿着一颗[[如意宝珠|如意宝珠]]{{noteTag|有一说是[[红宝石|红宝石]]，但大多数造像手持如意宝珠。}}抛在各个云朵之间上玩，有一个叫拉玛逊的[[夜叉|夜叉]]用[[斧头|斧头]]作为武器，看到宝珠请求媚卡拉索要，但是不愿意。这时拉玛逊很生气，扔着斧头追逐，媚卡拉每次都会及时闪避。

媚卡拉的宝珠发出的光明是闪电，而拉玛逊斧头劈开天空的声音是雷声，因此就有了雷电。<ref>{{cite web |url=http://www.thairath.co.th/column/pol/chuckthong/398279|title=เมขลาล่อแก้ว|author=กิเลน ประลองเชิง|date=24 มกราคม 2557|work= |publisher=ไทยรัฐ|accessdate=26 มกราคม 2557}}</ref>
