'''拉爾富·麥可泰爾'''('''{{lang-en|Ralph McTell}}''')出生時原名拉爾富‧梅({{lang-en|Ralph May}})，出生於[[英格蘭|英格蘭]][[倫敦|倫敦]][[法恩伯勒|法恩伯勒]]，生於{{Birth date and age|1944|12|3|df=no}}是一名[[英國|英國]][[創作歌手|創作歌手]]，[[原聲吉他|原聲吉他]]、[[鋼琴|鋼琴]]、[[口琴|口琴]]等音樂彈奏歌手。他在[[英國|英國]]六十年代的[[鄉村藍調|鄉村藍調]]和[[民間音樂|民間音樂]]中佔有一個舉足輕重及具影響力的地位。

麥克泰爾最著名的曲目是[[倫敦街角|倫敦街角]] ( Streets of London ) ，全球有超過二百名歌手翻唱，影響力可想而知。踏入八十年代，他為二個電視兒童節目創作歌曲及演奏兒歌，在兒童節目{{tsl|en|Alphabet Zoo|字母動物園}}中共有二個系列，由{{tsl|en|Nerys Hughes|奈瑞斯·休斯}}一起主持節目，
節目主題曲為Tickle On The Tum ( 在肚上嘀嘀嗒嗒 ) ，伴唱的還有扎基‧雷丁 ( Jaqui Reddin ) 。大碟在二個系列都有播放。<!-- Albums were also released from both series. --> 他亦有為動畫公司科斯格羅夫堂 ( Cosgrove Hall ) 的柳林風聲<ref>http://movie.iamcool.net/tw/coolmovie_show_movie.php?movieID=5002{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref> ( Wind in the Willows )創作及錄製主題曲，同唱的還有基斯‧霍普伍德 ( Keith Hopwood ) ，在[[ITV|獨立電視]]播放前就在1984年製成了唱片。

麥可泰爾的吉他風格，因在求學時的閱讀，認識到[[藍調|藍調]]、[[爵士樂|爵士樂]]和[[節奏藍調|節奏藍調]]，而受到非裔美國音樂家的影響，買了一把吉他開始學習，練習早期的美國[[藍調|藍調]]吉他手{{tsl|en|Blind Willie McTell|盲人威利·麥克泰爾}}、{{tsl|en|Blind Blake|盲人·布雷克}}、和[[羅伯·強生|羅伯·強生]]。這些學習模仿影響，導致一個朋友建議，在他演藝事業初具規模時，將[[藝名|藝名]]改為麥克泰爾(McTell) 。
