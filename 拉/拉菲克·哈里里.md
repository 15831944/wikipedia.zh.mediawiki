'''拉菲克·哈里里'''（[[阿拉伯语|阿拉伯语]]：'''{{rtl-lang|ar|رفيق بهاءالدين الحريري}}'''，拉丁化：'''{{lang|la|Rafik Bahaa Edine Hariri}}'''；{{bd|1944年|11月1日|2005年|2月14日|catIdx=Hariri, Rafik}}），[[黎巴嫩|黎巴嫩]]前總理。他從政前是黎巴嫩一位白手興家的億萬富豪。他曾兩度擔任[[黎巴嫩總理|黎巴嫩總理]]。2005年2月14日，他在[[贝鲁特|贝鲁特]]一所著名的五星級酒店——聖喬治酒店門外被由一名自殺式狙擊手所引爆的[[汽車炸彈|汽車炸彈]]炸死。

== 调查 ==
2005年2月25日，负责调查哈里里被谋杀事件的[[联合国|联合国]]调查小组在[[贝鲁特|贝鲁特]]宣布开始调查工作。<ref>http://news.xinhuanet.com/world/2005-02/25/content_2620978.htm</ref>该调查组在3月底提交给[[联合国安理会|联合国安理会]]的一份报告中说，黎政府组织的调查没有按照可以接受的国际标准进行。调查组建议对哈里里遇害案展开独立的国际调查。4月7日，安理会一致通过[[联合国安理会1595号决议|第1595号决议]]，决定设立一个独立的国际调查委员会，对哈里里遇害一事展开全面调查。<ref>http://news.xinhuanet.com/world/2005-04/08/content_2801487.htm</ref>5月13日，联合国发言人宣布，[[科菲·安南|科菲·安南]]秘书长已决定任命德国资深检察官[[德特勒夫·梅利斯|德特勒夫·梅利斯]]为哈里里遇害案国际调查委员会负责人。<ref>http://news.xinhuanet.com/world/2005-05/14/content_2955636.htm</ref>

9月3日，负责调查哈里里案的黎巴嫩司法调查法官[[埃利亚斯·伊德|埃利亚斯·伊德]]正式宣布拘捕4名参与策划者：黎巴嫩前安全总局局长赛义德、前国内安全局局长哈吉和黎军前情报局长阿扎尔以及现任黎共和国卫队司令哈姆丹。<ref>http://news.xinhuanet.com/photo/2005-09/04/content_3440995.htm</ref>

10月20日，国际独立调查委员会提交给安南的首份报告说，有证据表明，黎巴嫩和[[叙利亚|叙利亚]]两国的一些高级安全官员与哈里里谋杀案有牵连。<ref>http://news.xinhuanet.com/world/2005-10/21/content_3661596.htm</ref>12月12日，安理会收到国际调查委员会的第二份报告，报告称叙利亚在调查过程中未能履行安理会要求其予以充分配合的决议，有故意设置障碍的现象存在。<ref>http://news.xinhuanet.com/world/2005-12/13/content_3912916.htm</ref>12月底，叙利亚前副总统哈达姆透露，哈里里在遇刺前几个月曾经受到叙利亚总统[[巴沙尔·阿萨德|巴沙尔·阿萨德]]的威胁。<ref>http://news.bbc.co.uk/chinese/simp/hi/newsid_4570000/newsid_4570900/4570904.stm</ref>

2006年1月11日，安南任命来自比利时的[[塞奇·布拉默茨|塞奇·布拉默茨]]接替梅利斯担任国际调查委员会负责人。<ref>http://news.xinhuanet.com/world/2006-01/12/content_4040739.htm</ref>3月29日，安理会通过第1664号决议，要求安南与黎巴嫩政府进行协商，以成立调查哈里里遇害案的国际法庭。<ref>http://news.xinhuanet.com/world/2006-11/17/content_5340593.htm</ref>9月25日，独立调查委员会第三份报告称，哈里里确实死于自杀式汽车炸弹袭击，但尚不清楚这起袭击事件的幕后操纵者及袭击者的动机。<ref>http://news.xinhuanet.com/world/2006-09/26/content_5137208.htm</ref>12月12日，国际独立调查委员会递交的一份报告表示，有证据表明哈里里遇害案与之前发生在黎巴嫩的14起谋杀案有关联。<ref>http://news.xinhuanet.com/world/2006-12/13/content_5477485.htm</ref>2007年5月30日，联合国安理会通过决议，决定在黎巴嫩设立国际法庭以审理哈里里遇害案。<ref>刘历彬，王湘江，[http://news.xinhuanet.com/world/2007-05/31/content_6176498.htm 安理会决定设审哈里里遇害案国际法庭 中俄弃权]，新华网</ref>

== 传言 ==
=== 死因 ===
* [[美國|美國]]認為哈里里是被敘利亞行刺，並要求成立國際法庭來調查敘利亞。
* [[伊朗|伊朗]]的《[[伊斯蘭共和報|伊斯蘭共和報]]》則認為：哈里里謀殺案是由美國與以色列[[情報特務局|情報特務局]]（俗稱[[摩薩德|摩薩德]]）共同策劃的，他們由於侵略中東的計畫失敗而轉而暗殺哈里里，目的是想嫁禍給敘利亞，顛覆敘利亞政權。<ref>{{cite web |url=http://www.irib.com/worldservice/chinese/news/06-03-06/06030602.htm |title=存档副本 |accessdate=2006-11-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20071006063633/http://www.irib.com/worldservice/chinese/news/06-03-06/06030602.htm |archivedate=2007-10-06 }}</ref><ref>{{cite web |url=http://www.irib.com/worldservice/chinese/news/06-05-08/06050804.htm |title=存档副本 |accessdate=2006-11-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20071006063434/http://www.irib.com/worldservice/chinese/news/06-05-08/06050804.htm |archivedate=2007-10-06 }}</ref>

=== 身世 ===
哈里里1944年生于黎巴嫩南部塞达市，在[[贝鲁特|贝鲁特]]以及[[华盛顿|华盛顿]]决策圈有关他最流行的传闻是：哈里里并非是白手起家的富翁，而是[[沙特阿拉伯|沙特阿拉伯]]国王[[法赫德|法赫德]]私生子，他通过与沙特国王的关系得到[[皇室|皇室]]优待及有力的建筑合同而迅速发家。实际上法赫德国王与哈里里的关系的确很亲密，哈里里15年来的政治地位也多得到沙特阿拉伯的帮助和巩固。

== 家族 ==
* [[巴哈·哈里里|巴哈·哈里里]]，长子，[[美国|美国]][[波士顿大学|波士顿大学]]毕业，为哈里里的商业继承人，管理家族大多数公司业务。
* [[萨阿德·哈里里|萨阿德·哈里里]]，次子，[[美国|美国]][[乔治敦大学|乔治敦大学]]毕业，为哈里里的政治继承人，2005年[[议会|议会]]选举中，领导反叙联盟获得议会128席位中的72席位，控制了黎巴嫩历史上最多的议会议席。
* 艾曼·哈里里，三子。
* 法赫德·哈里里，四子。
* [[辛迪·哈里里|辛迪·哈里里]]，独女，家产14亿美元，2006年年仅22岁的她成为[[福布斯|福布斯]]富豪榜上公布的全球最年轻的亿万富翁。
