'''咖哩牛腩'''是一種[[香港|香港]]咖哩菜式，把[[印度|印度]]的[[咖哩|咖哩]]與[[廣東|廣東]]的[[牛腩|牛腩]]結合起來，是一道中印合璧的混合菜，而其起源與[[香港殖民地時期|香港的殖民地歷史]]有關<ref name="metrohk">[http://www.metrohk.com.hk/?cmd=detail&id=150088 印菜中食香港咖喱香]，[[都市日報|都市日報]]，2010-12-05</ref>。

== 起源 ==
[[File:Red_Curry_Beef_Brisket_in_Hong_Kong_Cha_Chaan_Teng.JPG|thumb]]售賣的咖哩牛腩]]
咖哩流行於香港，與香港在[[十九世紀|十九世紀]]中葉起成為[[英國殖民地|英國殖民地]]有關。香港在1841年被割讓予英國後，不久英國人就在香港建立維持治安的警隊，因為[[印度|印度]]早已成為英國的殖民地，所以不少印度人不但在英軍服役，更跟隨英國人來到香港，並在英國人的統籌下加入[[香港警察|香港警察]]，而當時印度人的地位雖然遠低於英國人，但仍比[[華人|華人]]的地位高，而這些印度人來到香港工作，也把咖哩香料及烹調方法帶到香港<ref name="metrohk"/>。

雖然印度人跟隨英國統治者而把咖哩香料帶來香港，但印度人因為信奉[[印度教|印度教]]而把牛隻視為神靈，不會食用牛肉。中國當時仍是傳統農業社會，出於尊敬牛隻對耕作的貢獻，同樣很少食用牛肉，而且咖哩也不是中國的傳統食品。可是英國統治香港後，[[香港人|香港人]]逐漸採納西方的飲食文化而把牛肉入饌。因為香港人認為牛腩最能突顯牛肉的味道，所以香港人特別喜歡吃[[炆|炆]]牛腩。由於製作咖哩的[[香料|香料]]隨着印度人在香港定居而容易取得<ref>[http://news.ebc.net.tw/%E5%86%AC%E5%AD%A3%E9%99%90%E5%AE%9A%EF%BC%81%E8%8C%B6%E9%A4%90%E5%BB%B3%E6%8E%A8%E5%92%96%E5%93%A9%E7%89%9B%E8%85%A9%E3%80%81%E8%97%A5%E8%86%B3%E9%8D%8B-501.html 冬季限定！茶餐廳推咖哩牛腩、藥膳鍋]，[[東森新聞|東森新聞]]，2015-11-15</ref>，於是香港人使用咖哩香料來炆牛腩，咖哩的香辣與牛腩的濃郁配合起來的美味，深得食客喜愛，所以咖哩牛腩便在香港流行起來，更因為其出現反映香港的歷史背景，而成為具有香港特色的菜餚<ref name="singtaousa">[http://www.singtaousa.com/558627/post-%E6%AE%96%E6%B0%91%E5%9C%B0%E6%99%82%E4%BB%A3%E7%9A%84%E6%99%9A%E9%A4%90/?variant=zh-hk&fs=16 殖民地時代的晚餐] {{webarchive|url=https://web.archive.org/web/20160603192222/http://www.singtaousa.com/558627/post-%E6%AE%96%E6%B0%91%E5%9C%B0%E6%99%82%E4%BB%A3%E7%9A%84%E6%99%9A%E9%A4%90/?variant=zh-hk&fs=16 |date=2016-06-03 }}，[[星島日報|星島日報]]，2015-11-01</ref>。

== 普及 ==
現今，咖哩牛腩是香港[[茶餐廳|茶餐廳]]及[[港式快餐|港式快餐]]的常見菜式，在午餐及晚餐時段皆有供應。咖哩牛腩通常與[[白飯|白飯]]一起進食，如果兩者以同一碟上菜，便成為'''咖哩牛腩飯'''。另外，香港還有一種稱為'''咖哩牛腩煲'''或'''咖哩牛腩鍋'''的[[煲仔菜|煲仔菜]]。
