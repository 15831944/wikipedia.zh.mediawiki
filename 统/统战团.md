'''统战团'''，指由[[中华人民共和国政府|中华人民共和国各级政府]]出资，以[[访问|访问]]、交流名义组织的各类至[[中国大陆|中国大陆]]旅游的台湾团体。{{不可靠來源}}

从[[胡锦涛|胡锦涛]]时代起，中华人民共和国政府便耗费巨资邀请、组织各类台湾团体至中国大陆参观、旅游。由青年学生所主导的2014年[[太阳花学运|太阳花学运]]后，中华人民共和国政府在2015年[[对台工作会议|对台工作会议]]中<ref>{{Cite web |url=http://www.huaxia.com/jjtw/dnzq/2015/01/4252337.html |title=2015年大陆对台工作新布局 打“三中一青”牌|accessdate=2017-08-19 |author= |date=2015-01-28|work= |publisher=华夏经纬网 |language=Zh-hans }}</ref>提出[[三中一青|三中一青]]（中小企业、中低阶层收入、中南部民众及台湾青年），首次单独提及台湾青年。有认为，中华人民共和国政府组织的统战团，并未改变台湾青年的[[国族认同|国族认同]]<ref>{{Cite web |url=http://www.chinatimes.com/newspapers/20150728001092-260306 |title=台青有沒有被統戰團影響？（上） |accessdate=2017-08-19 |author=峻嶺 |date=2015-07-28 |work= |publisher=中时电子报|language=Zh-hant }}</ref>。即使是“[[天然独|天然独]]”仍会参加统战团，免费旅游、获取资源<ref>{{Cite web |url=https://crossing.cw.com.tw/blogTopic.action?id=505&nid=8179 |title=為什麼我建議「天然獨」的台灣大學生，都該去參加中國「統戰團」？|accessdate=2017-08-19 |author=金妍冬 |date=2017-06-21 |work=執行編輯：YUKI，核稿編輯：張翔一 |publisher=换日线Crossing|language=Zh-hant }}</ref>。
