'''燃灯会'''（{{朝|名=南|諺=연등회|羅=Yeondeunghoe|漢=燃燈會}}）是[[朝鲜半岛|朝鲜半岛]]地区的一项传统[[灯火节|灯火节]]，最早起源于[[新罗|新罗]]年间，定于每年的农历四月初八前的周末，用以庆祝[[释加牟尼|释加牟尼]]的诞辰。<ref>{{cite web |author1=韩国旅游发展局 |title=燃灯会2018연등회2018 ：韩国旅游官方网站 |url=http://chinese.visitkorea.or.kr/chs/ATR/SI_CHG_2_1_1_1.jsp?cid=699078 |website=chinese.visitkorea.or.kr |accessdate=2018-11-09 |archiveurl=https://web.archive.org/save/http://chinese.visitkorea.or.kr/chs/ATR/SI_CHG_2_1_1_1.jsp?cid=699078 |archivedate=2018-11-09 |language=zh-cn| deadurl=no}}</ref><ref name=nfm>{{cite web |author1=安智源 |title=연등회 ( 燃燈會 , Yeondeunghoe ) |url=http://folkency.nfm.go.kr/cn/topic/%E7%87%83%E7%81%AF%E4%BC%9A/4525 |website=nfm.go.kr |publisher=国立民俗博物馆 |accessdate=2018-11-09 |archiveurl=http://archive.is/AYxHY |archivedate=2018-11-09 |language=zh-cn |date=2016-11-08| deadurl=no}}</ref>

== 历史 ==
有关燃灯会最早的书面记载见于《[[三国史记|三国史记]]·感通篇》中[[新罗|新罗]]年间[[景文王|景文王]]六年（866年），当时的燃灯会因为中国[[元宵节|元宵节]]的缘故，而在正月十五的上元节时举行。后[[燕山君|燕山君]]时期，因推行抑佛政策使得[[首尔|汉阳]]都城里的寺庙全部关闭，僧人也只准在四月初八这天出入都城。因而到了四月初八，民众会前往城外拜佛，并攀山观灯。1975年，韩国政府将四月初八指定为公休日。<ref name=nfm />2018年4月2日，[[韩国无形文化遗产|韩国文化财厅]]宣布对燃灯会申请[[非物质文化遗产|非物质文化遗产]]。<ref>{{cite web |author1=何路曼 |title=韩国燃灯会申请列入非物质文化遗产 结果2020年揭晓 |url=http://www.chinanews.com/gj/2018/04-03/8482619.shtml |website=中国新闻网 |accessdate=2018-11-10 |archiveurl=http://archive.is/c3JcZ |archivedate=2018-11-04 |language=zh-cn |date=2018-04-03| deadurl=no}}</ref>

== 习俗 ==
[[F:KOCIS_Korea_YeonDeungHoe_20130511_13_(8733834627).jpg|thumb]]
朝鲜时期的燃灯会期间，人们会用一捆竹竿做成灯架，顶端饰以五色绸缎做成彩旗。百姓家里还会按照子女的数量来挂灯，一直挂到初九。1996年起，以[[曹溪宗|曹溪宗]]为首的佛教教团在燃灯会期间开始举办游行、燃灯庆典等活动，以庆祝释加牟尼的诞辰，燃灯节也成了韩国最盛大的佛教节日。<ref name=nfm />

到了2017年，燃灯会的相关庆典活动持续了一整月，但其中最为热闹的庆祝集中在4月28日至30日。<ref>{{cite news |title=Seoul gets lit for Buddha’s birthday : Streets will fill with revelers, lanterns for month-long celebration |url=http://koreajoongangdaily.joins.com/news/article/article.aspx?aid=3032701 |accessdate=2018-11-10 |work=Korea JoongAng Daily |archiveurl=http://archive.is/yZAtg |archivedate=2018-11-10 |language=en| deadurl=no}}</ref>截至该年，燃灯会是吸引外国游客最多的韩国节日，有超30万韩国人、3万外国游客参与了庆祝活动，其中5万余人参与了制灯。<ref>{{cite web |author1=Yeon Deung Hoe Preservation Committee |title=Lotus Lantern Festival (Yeon Deung Hoe) will be held on May 11-13, 2018 in Seoul |url=https://www.prnewswire.com/news-releases/lotus-lantern-festival-yeon-deung-hoe-will-be-held-on-may-11-13-2018-in-seoul-300551726.html |website=www.prnewswire.com |accessdate=2018-11-10 |archiveurl=https://web.archive.org/web/20171110003903/https://www.prnewswire.com/news-releases/lotus-lantern-festival-yeon-deung-hoe-will-be-held-on-may-11-13-2018-in-seoul-300551726.html |archivedate=2017-11-10 |language=en| deadurl=no}}</ref>2018年，游行期间展出的灯超过了10万盏。<ref>{{cite web |author1=JANEEN CHRISTOFF |title=Experience a 1,200-Year-Old Lantern Festival in South Korea |url=https://www.travelpulse.com/news/destinations/experience-a-1200-year-old-lantern-festival-in-south-korea.html |website=TravelPulse |accessdate=2018-11-10 |archiveurl=http://archive.is/QyAok |archivedate=2018-11-10 |language=en |date=2018-03-26| deadurl=no}}</ref>
