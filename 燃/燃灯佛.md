'''燃灯佛'''（{{IAST|Dīpaṃkara}}），意为“锭光”，出生時周圍一切光明故得名，[[佛教|佛教]]中纵[[三世佛|三世佛]]之过去佛，为[[释迦牟尼|释迦牟尼]]佛之前的[[佛|佛]]，地位极尊，而来世佛则为“[[弥勒佛|弥勒佛]]”。因燃灯佛出生时身边一切光明如灯，故称为燃灯佛，又名'''定光如来'''、'''锭光佛'''、'''普光佛'''，或加古字作'''燃燈古佛'''等。

==經典記載==
《[[增一阿含經|增一阿含經]]》記載燃燈佛（定光如來、燈光佛）曾在过去世为[[释迦牟尼|释迦牟尼]]授记<ref name="v">參見 [[授記|授記]]</ref>，预言他未来将成佛，是为释迦牟尼佛授记<ref name="v" />之师。曾有一次，燃燈佛經過一污水，一個梵志（[[婆罗门|婆罗门]]）匍匐，用自己頭髮鋪在地上，使燃燈佛不須把腳弄髒，此人被[[預言|預言]]成佛，即釋迦牟尼佛。

《[[:s:修行本起經|修行本起經]]》和《[[:s:太子瑞應本起經|太子瑞應本起經]]》記載传说释迦牟尼过去世曾以五茎[[莲花|莲花]]供养燃灯佛，而被预言将在九十一劫后的[[贤劫|贤劫]]成佛。佛经中的许多佛、[[菩萨|菩萨]]都曾是他的弟子。

《[[贤愚经|贤愚经]]》卷三〈[[:s:賢愚經/賢愚經卷十二|贫女难陀品第五十三]]〉記載，过去久远二[[阿僧祇|阿僧祇]]九十一劫時，有一大國，國王波塞奇，兒子出家成佛爲[[寶髻佛|寶髻佛]]（Ratnaśikhin，勒那識祇，他經作“能作光佛”）<ref name=ran>[https://books.google.com.hk/books?id=l6woDwAAQBAJ&pg=PA41&lpg=PA41&dq=%E8%83%BD%E4%BD%9C%E5%85%89%E4%BD%9B&source=bl&ots=AvEjEmm1Qn&sig=ACfU3U1r6xFDVGxXcL2fcs9nn3xfESYdJA&hl=zh-CN&sa=X&ved=2ahUKEwiy1MGPh__gAhUJebwKHQaVD94Q6AEwAXoECAgQAQ#v=onepage&q=%E8%83%BD%E4%BD%9C%E5%85%89%E4%BD%9B&f=false 般若花開: 佛教諸佛最精彩的故事]，彭友智，{{ISBN|9789869279284}}</ref>，有一比丘名聖友（阿梨蜜羅），承擔了為期三個月的爲寶髻佛供燈，日日操辦，引起王女牟尼的詢問，牟尼得知供燈的詳情後，歡喜承諾承擔操辦，將燈具送至[[佛寺|寺舍]]，供聖友[[供養|供養]]。於是寶髻佛爲他們授記，聖友於一阿僧祇劫後成佛爲定光佛，牟尼於二大阿僧祇劫并九十一劫後成佛爲释迦牟尼佛，牟尼心生大歡喜，立即得轉女成男。

另有經典說，過去有睫如來出世時，一童子以瓦片盛油，手捧著供養如來，因此得到授記未來成佛爲燃燈佛，頸上和肩上會有大光明。<ref name=ran/><ref>[https://www.youtube.com/watch?v=W-gpIiahM14 燃灯古佛的故事：瓦片供灯（庄严劫千佛之一普光如来）]</ref>

《[[无量寿经|无量寿经]]》谓过去久远劫，锭光佛出世。其後依次有五十三尊佛出世。至第五十四尊，号[[世自在王如来|世自在王如来]]时，于其僧团中有一比丘号曰法藏，後成佛爲[[阿弥陀佛|阿弥陀佛]]。

《[[:s:大悲经|大悲经]]》卷三〈殖善根品〉谓燃灯佛以後，有莲华上佛，至过去七佛，总有十四佛出世。《[[大毗婆沙|大毗婆沙]]·卷一七八》说釋迦牟尼佛于第三阿僧祇劫曾遇七万七千佛，其中之首位，即为燃灯佛。汉传佛教認為燃燈佛是[[庄严劫|庄严劫]]千佛第一尊，或许即《[[:s:过去庄严劫千佛名经|过去庄严劫千佛名经]]》中的首尊华光佛。

== 寺院 ==
中国各地有一些燃灯寺，如[[四川|四川]][[成都|成都]]市内有燃灯寺公交车站（寺庙已于[[文革|文革]]期间被毁），而距离成都市中心18公里[[龙泉驿|龙泉驿]]区洛带镇龙泉山脉中段三峨山则另有一始建于[[隋|隋]][[开皇|开皇]]年间的有1700年历史的[[燃灯寺|燃灯寺]]。此外还有河南[[南阳|南阳]]西峡燃灯寺、[[云南|云南]][[红河|红河]]建水城东门外燃灯寺、[[通州|通州]]北端运河西岸的燃灯佛舍利塔等等。

== 佛像 ==
希腊化[[健驮逻国|犍陀罗]]佛像艺术的一种——[[迦毕试样式|迦毕试样式]]（Kapicsa Style）——是指[[贵霜帝国|贵霜帝国]]时代流行于阿富汗[[迦毕试|迦毕试]]（今贝格拉姆）一带的一种佛像风格，表现最多的是为燃灯佛和舍卫城神变中[[焰肩佛|焰肩佛]]两种形象。原喀布尔博物馆收藏多尊3-4世纪的此类佛像。

2001年3月19日被[[阿富汗|阿富汗]][[塔利班|塔利班]]政权毁掉的两尊[[巴米扬大佛|巴米扬大佛]]，其中较大的那尊也被认为是燃灯佛的形象。<ref>{{de}}[http://www.buddha-bamiyan.ag.vu/s19.html Buddha Bamiyan]</ref>

== 其他 ==
[[藏传佛教|藏传佛教]]的[[燃灯节|燃灯节]]与燃灯古佛无关，是纪念[[格鲁派|格鲁派]]（俗称黄教）開山[[祖師|祖師]][[宗喀巴|宗喀巴]]1419年在[[甘丹寺|甘丹寺]][[圆寂|圆寂]]成佛的日子。每年[[藏历|藏历]]10月25日，各家[[寺院|寺院]]和俗家屋顶均要点亮无数盏灯，故称为燃灯节，[[藏语|藏语]]为“葛登阿曲”，“阿曲”意为“五供”。[[中國|中國]]以[[農曆|農曆]][[八月廿二|八月廿二]]日為燃燈古佛誕。

==文學形象==
*在《[[西游记|西游记]]》中有“南无燃灯上古佛”，列为众佛之首。
*在《[[封神演义|封神演义]]》中，燃灯佛的前身為[[闡教_(封神演義)|闡教]]仙人[[燃灯道人|燃灯道人]]，日后加入[[佛教|西方教]]成佛。
*在《[[三寶太監西洋記|三寶太監西洋記]]》中，有「釋迦如來是諸釋之法王，燃燈古佛是如來授記之師父」的記載，又引詩為證：「嘗聞釋迦佛，先授燃燈記。燃燈與釋迦，只論前後智。前後體非殊，異中無一理。一佛一切佛，心是如來地。」並描述其佛相「燃燈古佛，又名定光佛。你看他無我相，無人相，無眾生相，無壽者相，頂上光明直沖千百丈」。小說中的僧人[[金碧峰|金碧峰]]，為[[南朝|南朝]]國師，正是燃燈佛轉世。
