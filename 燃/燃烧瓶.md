'''燃烧瓶'''（{{lang-en|'''Molotov Cocktail'''}}，{{lang-ru|'''Кокте́йль Мо́лотова'''}}），又称燃烧彈，是[[游擊隊|游擊隊]]等非正規部隊，以及街頭[[暴乱|暴乱]]群眾的常用[[武器|武器]]，有些罪犯也會以它來[[縱火|縱火]]。军方亦有采用，不同于制式武器[[凝固汽油彈|凝固汽油彈]]。

== 成份 ==
燃燒瓶的主要構造為：[[玻璃瓶|玻璃瓶]]，瓶內裝有半滿的[[易燃液體|易燃液體]]，例如三分之二的[[汽油|汽油]]與三分之一的[[油|油]]混合而成的液體或酒精（[[甲醇|甲醇]]或[[乙醇|乙醇]]）；
瓶口以軟木塞或[[塑膠|塑膠]]、[[橡膠|橡膠]]、[[電線膠帶|電線膠帶]]、[[牛皮膠帶|牛皮膠帶]]等不透氣塞堵住；瓶口上紮上布塊作引。
使用之前把布塊沾透易燃液體，燃點布塊後把瓶拋出。玻璃瓶撞擊目標破裂，易燃液體傾倒在目標之上，被火點著燃燒，可造成一定的阻絕及殺傷能力。

苏联曾经制造制式燃烧瓶，外面装有两根密封有浓硫酸的玻璃管。投掷碰碎之后靠化学反应产生的热量点燃油料，从而节省了点火步骤。

燃燒瓶的原理跟正規軍隊使用的燃燒彈基本相同；兩者都是以汽油混合使汽油變稠的物料。

另外也有人使用[[塑膠袋|塑膠袋]]裝汽油，因為塑膠袋可以承載著一定容量的液體。

== 战争中的发明与发展 ==
[[File:Molotov_bread_basket.jpg|thumb]]
燃烧瓶实际上在[[西班牙內戰|西班牙內戰]]时期就已出现。当时内战交战双方即使用燃烧瓶获得部分成功。

二次大戰中參戰各方都有使用燃烧瓶作為防守的武器，它對於阻擋輕型坦克特別有效。

坦克車的底部往往是防禦最薄弱的的，當燃烧瓶在坦克的尾部燃燒時，可能令坦克的油缸爆炸。部分蘇製坦克設有外挂油箱，很容易被从外面打开后点燃。當游擊隊在近距離以燃烧瓶快速突擊，坦克車的靈活性不足應付，於是陷入被動。這亦是燃烧瓶在當時能成為成功武器的原因之一。

1948年[[以色列|以色列]]和[[阿拉伯|阿拉伯]]戰爭中的以色列、1956年[[匈牙利事件|匈牙利事件]]中的匈牙利羣眾，都曾成功使用燃烧瓶阻止敵人裝甲兵的前進。

== 常見錯誤 ==
由於汽油彈製作容易，使它幾乎已成為游擊隊及街頭暴民的「標準裝備」。但對全無經驗的人來說，製造及使用汽油彈亦非全無難度。第一種常犯的錯誤是瓶內放進太多的汽油，在玻璃瓶碎開後過多汽油會難以著火，正確的份量約為瓶的一半至三分之二，不可超過瓶頸位置。另一種錯誤是點燃布塊前沒有把玻璃瓶抹乾淨，使瓶外殘餘的汽油燃燒起來，導致投擲者手部燒傷。此外，用作引信的布塊來封住玻璃瓶也是錯誤的作法。其他各種錯誤包括瓶子的塞不密封，選用錯誤（太堅固，瓶頸太長的玻璃瓶，如伏特加、香檳酒瓶）等等。但最常見的錯誤還是點燃以後的投擲過程出錯。

== 投擲方法 ==
主手持瓶，輔手持打火機，使用打火機燃點浸有液體的布塊之後，馬上扔出。
在各式示威暴動裡，燃燒瓶多拋在對峙的空隙之中，用以阻截對方繼續推進。因為其殺傷力有限，所以並非主用以傷人。
而將燃燒瓶拋在地上，比起拋到人群之中，更具阻截成效。

== 關於燃烧瓶的法律 ==
在世界大部分的地區，製造及持有燃烧瓶都是違法的。使用燃烧瓶更可能會被控[[傷人|傷人]]、[[縱火|縱火]]、[[誤殺|誤殺]]、[[謀殺|謀殺]]等罪名。

== 別名 ==
[[File:Talvisota_Molotov_Cocktail.PNG|thumb]]中芬蘭軍人使用的莫洛托夫雞尾酒]]
燃烧瓶的別名「'''莫洛托夫鸡尾酒'''」，得名自[[蘇聯|蘇聯]]的外交人民委員（外交部長）[[维亚切斯拉夫·米哈伊洛维奇·莫洛托夫|维亚切斯拉夫·米哈伊洛维奇·莫洛托夫]]。

[[二次大戰|二次大戰]]時蘇聯入侵[[芬蘭|芬蘭]]爆发[[苏芬战争|苏芬战争]]，芬蘭士兵無論是参战人數或是裝備水平皆處於下風。在冬季遊擊戰期間，面對紅軍的[[坦克|坦克]]，由於芬蘭軍戰防炮數量極度不足，因此大量使用發明於[[西班牙內戰|西班牙內戰]]的武器燃烧瓶。以玻璃瓶等容器盛裝汽油、磷、鋁、機油等助燃劑的燃燒瓶，對蘇軍陣地與車輛造成極大損害。

由於战争中，苏联[[轰炸机|轰炸机]]曾用燃烧弹轰炸芬兰的城市和平民目标，濫殺無辜平民，遭到世界各國的強烈譴責。面对国际社会的指责时，苏联外长莫洛托夫辯詞漏洞百出，甚至宣稱苏联飞机没有向芬兰人民扔下炸弹，而是去[[貧民區|貧民區]]為貧苦的芬蘭人民空投麵包。这一可笑的谎言被当时在轰炸现场的外国使团和新闻记者所揭露，芬蘭軍民便將苏联投下燃烧弹謔称为“莫洛托夫的面包篮”。由於此燃燒瓶多以酒瓶為容器，随后并将他們準備的這種小型的燃烧彈稱為「回敬莫洛托夫的雞尾酒」，用來回敬蘇聯投下的“面包篮”。芬蘭軍方在之後持續的戰事中大量生產燃烧瓶應對蘇聯坦克大軍，以手或掷石器投擲。

而被稱作「[[雞尾酒|雞尾酒]]」的原因是汽油彈內能夠混合不同物料以應對不同狀況，與雞尾酒混合各式酒品的原理相同：加入[[白磷|白磷]]能夠令擊發時火勢更猛烈，加入[[麵粉|麵粉]]、[[鋁粉|鋁粉]]、[[氧化鐵|鐵鏽]]或[[鋁熱法|鋁熱劑]]能夠釋放高熱熔化金屬，加入[[砂糖|砂糖]]、[[花生醬|花生醬]]能夠延長燃燒時間，加入[[洗衣粉|洗衣粉]]、[[潤滑油|潤滑油]]、[[油漆|油漆]]、[[肥皂|肥皂]]或被有機溶劑熔解後的[[保麗龍|保麗龍]]碎片可以增加粘著度，加入[[樟腦丸|樟腦丸]]能產生煙霧，加入[[硫酸|硫酸]]則能產生腐蝕效應。
