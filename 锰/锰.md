'''锰'''（'''manganese'''）是一种化学[[元素|元素]]，它的[[化学符号|化学符号]]是'''Mn'''，它的[[原子序数|原子序数]]是25，是一种[[过渡金属|过渡金属]]。锰一般以[[化合物|化合物]]的形式存在于自然界中，不会以[[单质|单质]]出现。锰主要用作[[合金|合金]]。

== 性质 ==
錳是灰白色金屬，性堅而脆，潮溼處會[[氧化|氧化]]。

=== 化合物 ===
锰的[[氧化数|氧化数]]有-3至+7，其中以氧化数+2、+4和+7的[[化合物|化合物]]最重要。

{{chem|Mn|2+}}最稳定，呈粉红色，不容易被[[氧化|氧化]]，也不容易被[[还原|还原]]。[[硫酸锰|硫酸锰]]（{{chem|MnSO|4}}）、[[氯化锰|氯化锰]]（{{chem|MnCl|2|}}）等属于这氧化数。{{chem|MnO|4|-}}和{{chem|MnO|2}}有强氧化性。

氧化数+7的[[高锰酸盐|高锰酸盐]]（{{chem|MnO|4|-}}）呈紫色，多数是强氧化剂，如[[高锰酸钾|高锰酸钾]]、[[高锰酸钠|高锰酸钠]]、[[高锰酸钡|高锰酸钡]]等。

在酸性溶液中，{{chem|Mn|3+}}和{{chem|MnO|4|2-}}均易发生[[歧化反应|歧化反应]]：
::2 {{chem|Mn|3+}} + 2 {{chem|H|2|O}} → {{chem|Mn|2+}} + {{chem|MnO|2}}↓ + 4 {{chem|H|+}}
::3 {{chem|MnO|4|2-}} + 4 {{chem|H|+}} → 2 {{chem|MnO|4|-}} + {{chem|MnO|2}}↓ + 2 {{chem|H|2|O}}

在碱性溶液中，{{chem|Mn(OH)|2}}不稳定，易被空气中的氧气氧化为{{chem|MnO|2}}；{{chem|MnO|4|2-}}也能发生歧化反应，但反应不如在酸性溶液中进行得完全。

[[锰的氧化物|锰的氧化物]]及其[[水合物|水合物]][[酸碱性|酸碱性]]的递变规律，是[[过渡金属|过渡金属]]中最典型的：随锰的氧化数的升高，酸性逐渐增强。<ref>《无机化学》第四版.高等教育出版社.P<sub>392</sub>. ''13.4.2 锰的重要化合物''</ref>

=== 同位素 ===
{{See also|锰的同位素}}
锰共有32个同位素，[[原子量|原子量]]从44至69。其中最稳定同位素是<sup>55</sup>Mn。其余的大部分有放射性。

== 发现 ==

18世紀後半瑞典化學家Johann Gahn研究[[軟錳礦|軟錳礦]]時，認為它是一種不同於以往金屬的氧化物，但他並沒有成功分離。舍勒嘗試分離也沒有成功，於是他向他的助手，同時也是他好朋友的甘英求助，到了1774年[[约翰·戈特利布·甘恩|甘恩]]才成功的從錳礦中分離出錳。
並將之命名為Manganese（錳）其[[拉丁语|拉丁语]]：magnes，意即「具[[磁|磁性]]的」（但只有经过特殊处理的锰才会具有磁性），及元素符號Mn亦從之而來。人們早在1913年就已知錳是組成動物的重要元素之一，但直到1931年才經由[[動物實驗|動物實驗]]得知和錳有關的症狀。

==来源==
锰在地壳含量约1000[[百万分率|ppm]]（0.1%），居元素分布序列中的第12位。

錳會以[[软锰矿|软锰矿]]（{{chem|MnO|2}}）[[硬锰矿|硬锰矿]]（{{chem|(Ba.H|2|O)2Mn|5|O|10}}）和[[菱锰矿|菱锰矿]]（{{chem|MnCO|3}}）等形式存在于自然界中。

== 制取 ==

将软锰矿用铝盒盛放，下方有导管连接，使用氯酸钾和镁条进行加热，使铝化为三氧化二铝（高温耐火材料），软锰矿分解为单质锰和氧气，单质锰融化顺导管导出。

== 用途 ==
冶金工业中用以制造特种钢，在钢铁生产上用[[锰铁合金|锰铁合金]]作为去硫剂和去氧剂。

此外锰也用作[[合金|合金]]，[[电池|电池]]等。[[二氧化锰|二氧化锰]]（{{chem|MnO|2}}）用作[[催化剂|催化剂]]和[[棕色|棕色]][[颜料|颜料]]，[[高锰酸钾|高锰酸钾]]（{{chem|KMnO|4}}）用作[[氧化剂|氧化剂]]及[[消毒剂|消毒剂]]。

== 對人體的影響 ==
{{see|en:Manganese#Biological role}}

錳是身體所必需的微量元素之一，可構成生物體中具重要生理功能之[[酶|酶]]或[[輔酶|輔酶]]，每日攝取約3-9mg。
具有以下之功能：
# 促進骨骼之發育以及生長
# 維持腦功能之正常運作
# 維持糖以及脂肪之正常代謝
# 維持細胞粒線體之完整
# 構成輔酶

由於[[粒線體|粒線體]]需要錳，所以錳在粒線體多之組織含量較高，常見於[[骨骼|骨骼]]、[[肝臟|肝臟]]、[[腎臟|腎臟]]、[[胰臟|胰臟]]。然而過量錳之攝取依然會對生物有所影響（[[神经退行性疾病|-{zh-hans:神经退行性疾病;zh-hant:神經退化性疾病;}-]]），常見於職業中，其發生原因為吸入含錳濃度高之錳煙及錳塵。
