在[[数学|数学]]中，'''比例'''是兩個非零數量<math>y</math>與<math>x</math>之間的比較關係，記為<math>y:x \; (x, y \in \mathbb{R})</math>，在計算時則更常寫為<math>\frac{y}{x}</math>或<math>y/x</math>。若两个[[變量|變量]]的关系符合其中一个量是另一个量[[乘法|乘以]]一个[[常数|常数]]（<math>y=kx</math>），或等价地表达为两變數之[[比率|比率]]為一個常數（稱為'''比值'''，<math>y/x=k</math>），则称两者是'''成比例的'''。

如果<math>y</math>與<math>x</math>是[[通約性|可通約的]]，亦即它們之間存在一個'''公測量'''（{{lang|en|common measure}}）<math>m \; (m \in \mathbb{R})</math>使得<math>y = mp, x = mq \; (p, q \in \mathbb{Z})</math>，<math>y:x</math>就相等於兩個整數的比：<math>y:x = mp:mq = p:q</math>，那麼<math>y:x</math>就稱為'''可通約比'''（{{lang|en|commensurable ratio}}），<math>\frac{p}{q}</math>稱為一個[[分數|分數]]，其比值稱為[[有理數|有理數]]；否則，如果不存在一個公測量，<math>y:x</math>就稱為'''不可通約比'''（{{lang|en|incommensurable ratio}}），其比值稱為[[無理數|無理數]]，亦即無法表達為分數的數。

兩個比例之間也可以互相比較。如果兩個比例相等，亦即，它們的比值相同，這個相等關係稱為一個[[比例論|等比關係]]，例如，<math>y:x=u:o</math>是一個等比關係，其中<math>xu=yo</math>。特別是，如果第二項等於第三項，例如<math>y:x=x:z</math>，那麼<math>x^2 = yz \rightarrow x = \sqrt{yz}</math>，<math>x</math>稱為<math>y</math>與<math>z</math>的[[幾何平均數|幾何平均數]]（{{lang|en|geometric mean}}）<ref>[https://books.google.com.tw/books?id=BQI7CgAAQBAJ&pg=PT38#v=onepage&q&f=false A Brief History of Numbers, 3.2 Ratios and proprotions]</ref>。

== 定义 ==
若存在一非零常数<math>k</math>使
:<math>y = kx</math>

则称变量<math>y</math>与变量<math>x</math>成比例（有时也称为成'''正比'''）。当<math>x</math>和<math>y</math>成'''正比'''关系，表示当<math>x</math>变为原來<math>k</math>倍时，<math>y</math>也会变为原來的<math>k</math>倍。

* <math>y</math>是因变量
* <math>x</math>是自变量
* <math>k</math>则是变分常数，而<math>k</math>不等于<math>0</math>。如<math>k=0</math>，则不能成立正比关系。也就是说，<math>x, y</math>两个[[变量|变量]]成[[线性函数|线性函数]]关系。

该关系通常用<math>\propto</math>（[http://www.fileformat.info/info/unicode/char/221D/index.htm U+221D]）表示为：
:<math>y \propto x</math>

并称该常数比率
:<math>k = \frac{y}{x}</math>

为'''比例常数'''或比例[[二元关系|关系]]中的'''比例恒量'''。

在日常生活中，正比这个词的使用并不严格局限于线性函数，一般来说，一个变量随着另一个变量的增大／缩小而相应地增大／缩小，近似地满足线性关系的时候，我们可以说这两个变量成正比。

== 用法與歷史 ==
現代數學對於比例的用法並沒有嚴格限制，例如，在一個班級裡面，我們可以說：「男孩與女孩的比例是2比1」。然而，在古希臘數學中，由於比例是用來表示[[倍數|倍數]]關係，所以必須是相同種類的數量才能構成比例，例如，[[歐幾里得|歐幾里得]]在《[[幾何原本|幾何原本]]》第五冊中如此定義比例<ref>[http://www.perseus.tufts.edu/hopper/text?doc=Perseus%3Atext%3A1999.01.0085%3Abook%3D5%3Atype%3DDef%3Anumber%3D3 《[幾何原本》原文第五冊，定義三]，柏修斯數位圖書館</ref>：
{{quote|
{{lang|el|λόγος ἐστὶ δύο μεγεθῶν ὁμογενῶν ἡ κατὰ πηλικότητά ποια σχέσις.}}

{{lang|en|A ratio is a sort of relation in respect of size between two magnitudes of the same kind.}}

比例是兩個同類數量之間的大小關係。
}}

[[阿基米德|阿基米德]]使用這個定義來敘述均勻運動（{{lang|en|uniform motion}}）的[[比例論|等比關係]]<ref>[https://books.google.com.tw/books?id=Vvj_AwAAQBAJ&pg=PA140#v=onepage&q&f=false Archimedes, 10. Elements of Mechanics], by Eduard Jan Dijksterhuis</ref>：
{{quote|在一個均勻運動中，兩段距離的比例相等於它們所需時間的比例。}}

阿基米德所要描述的，就是[[勻速運動|勻速運動]]，但是古希臘數學並不接受[[距離|距離]]與[[時間|時間]]的比例<ref>[http://www2.clarku.edu/~djoyce/ma121/ftc.pdf Origin of the Fundamental Theorem of Calculus]，by D Joyce，克拉克大學</ref>（亦即[[速率|速率]]），因為它們是不一樣的數量，所以他沒有辦法直接說：「均勻運動就是每一點上的速率皆相等」。當採用古希臘的比例論來敘述時，必須取兩段距離<math>L_1</math>與<math>L_2</math>以及所需時間<math>T_1</math>與<math>T_2</math>，均勻運動（勻速運動）就是<math>L_1 : L_2 = T_1 : T_2</math>。

== 例子 ==
* 假设某人以匀[[速度|速]]运动，则其运动的[[距离|距离]]是和运动的[[时间|时间]]成正比的，所以速度就是当中的比例常数。
* [[圆|圆]]的[[圆周|周长]]与其[[直径|直径]]成正比，当中的比例常数就是[[圆周率|π]]。
* 在按[[比例尺|比例尺]]绘制的[[地图|地图]]上，地图上任意两点间的距离是和该两点所代表的实际地点之间的距离成比例的，当中的比例常数即是绘制该地图所使用的比例尺系数。
* 物理学中，[[地球|地球]]的[[重力|重力]]对在海平面上的某物体的[[力|作用力]]的数值与该物体的[[质量|质量]]成正比，当中的比例常数是地球的[[重力加速度|重力加速度]]。{{ref|earth_gravity_constant}}

== 性质 ==
因为
:<math>y = kx</math>

等价于
:<math>x = \frac{1}{k} \cdot y</math>

因此可推出，若 <math>y</math> 与 <math>x</math> 之间存在正比关系，则 <math>x</math> 与 <math>y</math> 之间存在反比关系。

<math>y</math> 与 <math>x</math> 的正比关系也可以被解读为一条在[[笛卡尔坐标系#二維坐標系統|二维直角坐标系]]穿过[[原点|原点]]的[[直线|直线]]，其[[斜率|斜率]]为比例常数。

比例关系中，位于两端的两数之积等于位于中间的两数之积：
:<math>\frac{a}{b} = \frac{c}{d} \iff ad = bc</math>

=== 其他性质 ===
* <math>\frac{a}{b} = \frac{c}{d} \iff \frac{b}{a} = \frac{d}{c}</math>
* <math>\frac{a}{b} = \frac{c}{d} \iff \frac{a}{c} = \frac{b}{d}</math>
* <math>\frac{a}{b} = \frac{c}{d} \iff \frac{a \pm b}{b} = \frac{c \pm d}{d}</math>
* 若 <math>\frac{a}{b} = \frac{x}{y}</math> 且 <math>\frac{c}{d} = \frac{x}{y}</math>，则 <math>\frac{a+c}{b+d} = \frac{x}{y}</math>。

== 反比关系 ==
在上面定义中，我们说有时称两个成比例的变量成'''正比例'''，这是为了和'''反比例'''关系相对应。

如果两变量中，一个变量和另外一个变量的[[倒数|倒数]]成正比，或等价地，若这两变量的[[乘积|乘积]]是一个常数，则称这两个变量是成'''反比例'''（或'''相反地变化'''）的。从而可继续推出，若存在一非零常数<math>k</math>使
:<math>y = \frac{k}{x}</math>

则变量<math>y</math>和变量<math>x</math>成反比。

反比例关系的概念基本上说明的是这样一种关系，即当一个变量的值变大时，另一变量的值相应变小，而两者之积总是保持为一常数（即比例常数）。

举例来说，运动中的车辆走完一段路程所花费的时间是和这辆车运动的速度成反比的；在地上挖个坑所花的时间也（大致地）和雇来挖坑的人数成反比的。

在笛卡尔坐标平面上，两个具有反比例关系的变量的图形是一对双曲线。该图线上的每一点的 X 和 Y 坐标值之积总是等于比例常数<math>k</math>。由于<math>k</math>非零，所以图线不会与坐标轴相交

== 指数比例和对数比例 ==
若变量<math>y</math>与变量<math>x</math>的[[指数函数|指数函数]]成正比，即：若存在非零常数<math>k</math>使
:<math>y = k a^x</math>

则称<math>y</math>与<math>x</math>成[[指数比例|指数比例]]。

类似地，若变量<math>y</math>与变量<math>x</math>的[[对数函数|对数函数]]成正比，即：若存在非零常数<math>k</math>使
:<math>y = k \log_a{x}</math>

则称<math>y</math>与<math>x</math>成[[对数比例|对数比例]]。

== 确定比例关系的实验方法 ==
用实验方法确定两个[[物理|物理]]量是否具有正比关系，可采用这样的办法，即进行多次测量并在[[笛卡尔坐标系|笛卡尔坐标系]]中将这些测量结果用多个点来表示，而绘制出这些点的分布图形；如果所有点完全（或接近）地落在一条穿过原点<math>(0, 0)</math>的直线上，则这两个变量（很有可能）具有比例常数等于该直线[[斜率|斜率]]的正比关系。
