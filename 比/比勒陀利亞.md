'''普利托里亞'''（[[南非語|南非語]]、[[英語|英語]]：Pretoria），是位於[[南非|南非]][[豪登省|豪登省]]（{{lang|af|Gauteng}}）北部的城市，亦是南非的行政[[首都|首都]]。南非總統府位於這個城市內，各國使館亦集中於此城的使館街，所以事實上-{zh-hans:比勒陀利亚;zh-hk:普利托里亞;zh-tw:普勒托利亞;}-是南非的政治決策中心。城市建於1855年，以[[布爾人|布爾人]]領袖[[安德列斯·比勒陀利乌斯|-{zh-hans:比勒陀利乌斯;zh-hk:普利托里奧斯;zh-tw:普利托里奧斯;}-]]的名字命名，不過使用殖民者的名字使得當前城市名稱還有爭議。市內種滿不同的花草樹木，故亦有“'''花園城'''”的美譽。

== 歷史 ==
第一批定居於-{zh-hans:比勒陀利亚;zh-hk:普利托里亞;zh-tw:普勒托利亞;}-的居民是[[恩古尼人|恩古尼人]]（Nguni）的移民，他們居住於現時-{zh-hans:比勒陀利亚;zh-hk:普利托里亞;zh-tw:普勒托利亞;}-市內的河谷一帶。這批操[[恩古尼語|恩古尼語]]的居民慢慢地組成了一個族群，名為[[恩德貝勒人|恩德貝勒人]]（Ndebele），不過此詞從[[索托語|索托語]]的意思則被解釋為流亡者。而在他們聚居地附近的河流則以他們其中一個族長的名字茨瓦內 （Tshwane）來命名，但由於Tshwane一詞在[[恩德貝勒語|恩德貝勒語]]中的意思是小[[猿猴|猿猴]]，故此在南非荷蘭語中就直接被翻譯為猿猴河（Apies River）。

-{zh-hans:比勒陀利亚;zh-hk:普利托里亞;zh-tw:普勒托利亞;}-是由“移民先驅”（Voortrekkers）的領袖[[馬菲洛斯·普利托里奧斯|馬菲洛斯·-{zh-hans:比勒陀利奥斯;zh-hk:普利托里奧斯;zh-tw:普利托里奧斯;}-]]於1855年所創立的，並以其父安第列斯·-{zh-hans:比勒陀利奥斯;zh-hk:普利托里奧斯;zh-tw:普利托里奧斯;}-的名字命名。馬菲洛斯於[[血河之戰|血河之戰]] （Battle of Blood River）中擊退了[[祖魯人|祖魯人]]而成為國家英雄。而他所建立的這個城市之後亦成為了南非共和國 （South African Republic）的首都。

== 城市更名 ==
2005年5月26日，[[南非地理名稱委員會|南非地理名稱委員會]] （South African Geographical Names Council，簡稱SAGNC）通過將-{zh-hans:比勒陀利亚;zh-hk:普利托里亞;zh-tw:普勒托利亞;}-的名稱更改為[[茨瓦內|茨瓦內]] （Tshwane）。而-{zh-hans:比勒陀利亚;zh-hk:普利托里亞;zh-tw:普勒托利亞;}-本身是隸屬於[[茨瓦內大都會自治市|茨瓦內大都會自治市]]的其中一部分，故將屬於大都會自治市中心的-{zh-hans:比勒陀利亚;zh-hk:普利托里亞;zh-tw:普勒托利亞;}-更名成與大都會自治市名稱一樣。但是，更名一事到現時尚未得到[[藝術|藝術]]及[[文化|文化]]部長[[派路·佐敦|派路·佐敦]] （Pallo Jordan）的核准。現時部長仍就更名事宜和程序作研究和考量，並在向[[國會|國會]]提交決議案前將城市更名事宜向公眾咨詢意見。以現時的進程來說，就算文化部長在2006年上半年核准城市更名事宜，城市的名稱應該不會在2006年內被更改。

== 經濟 ==
[[File:Pretoria_streetsign.jpg|thumb]]
-{zh-hans:比勒陀利亚;zh-hk:普利托里亞;zh-tw:普勒托利亞;}-是南非重要的[[工業|工業]]中心，採礦、汽車製造、鐵路和機器製造業等[[重工業|重工業]]都相當發達。城市的經濟產業約佔全南非[[國民生產總值|國民生產總值]]的8.55% （2002年數字），是繼[[约翰内斯堡|约翰内斯堡]]和[[開普敦|開普敦]]之後的南非第三大。

== 地理與氣候 ==
-{zh-hans:比勒陀利亚;zh-hk:普利托里亞;zh-tw:普勒托利亞;}-位於高木草原 （Highveld）和灌木草原 （Bushveld）交界位置，位於[[约翰内斯堡|约翰内斯堡]]北方約50[[公里|公里]]遠。城市被位於[[海平面|海平面]]以上1,370[[米|米]]的[[馬加拉斯堡|馬加拉斯堡]] （Magaliesberg）的[[山脈|山脈]]環繞，被天然的屏障阻隔，形成溫暖的[[氣候|氣候]]以及肥沃的山谷。城市的[[經緯度|經緯度]]約為{{coord|25|43|S|28|17|E|type:city(1,500,000)_region:ZA}}。

== 文化與學術 ==
-{zh-hans:比勒陀利亚;zh-hk:普利托里亞;zh-tw:普勒托利亞;}-是南非其中一個學術中心，全南非最大的寄宿大學[[比勒陀利亞大學|-{zh-hans:比勒陀利亚大學;zh-hk:普利托里亞大學;zh-tw:普勒托利亞大學;}-]]以及最大的遙距教學大學[[南非大學|南非大學]]都位於此地。另外，[[南非科學及工業研究委員會|南非科學及工業研究委員會]] （Council for Scientific and Industrial Research，簡稱CSIR）亦位於此城。

城市中心約有一百萬人口，如以全個茨瓦內大都會自治市的人口計算的話，則約有二百萬人口。-{zh-hans:比勒陀利亚;zh-hk:普利托里亞;zh-tw:普勒托利亞;}-的居民主要日常用語為[[茨瓦納語|茨瓦納語]]、[[恩德貝勒語|恩德貝勒語]]、[[南非荷蘭語|南非荷蘭語]]和[[英語|英語]]。

比勒陀利亞市有撒哈拉以南非洲最大的白人人口。 城市自成立以來，它一直是南非白人的主要的人口中心，目前大約有數百萬南非人生活在城市或城市周圍。 

即使種族隔離結束以來，比勒陀利亞本身也是白人多數，儘管黑人中產階級不斷增加。最大的白人族裔群體是南非白人。 

比勒陀利亞包含以前的白人指定地區，因此白人佔多數，使人口的估計較低。 然而，包括地理上分開的鄉鎮，比勒陀利亞增加的外來人口超過一百萬，使白人變少數。

== 城市趣聞 ==
* -{zh-hans:比勒陀利亚;zh-hk:普利托里亞;zh-tw:普勒托利亞;}-的主要街道教堂街 （Church Street）是全南非最長的市區街道。
* 由於城市內有很多街道都種滿了[[紫葳|紫葳]]，所以城市亦有別稱為“紫葳城”。
* 全球最大的鑽石原石[[卡利南鑽石|卡利南鑽石]]是於1905年1月26日在-{zh-hans:比勒陀利亚;zh-hk:普利托里亞;zh-tw:普勒托利亞;}-附近的[[古利蘭鎮|卡利南鎮]]被發現的。
* -{zh-hans:比勒陀利亚;zh-hk:普利托里亞;zh-tw:普勒托利亞;}-擁有全球第二多的大使館，僅次於[[華盛頓特區|華盛頓特區]]。

* 看中医在比勒陀利亚逐渐流行，南非中医药总公司（SA CHINESE MEDICINE）在总统府联合大厦前面的Sunnyside商业区开办的药店每天都有百人以上光顾。
