'''比通港'''（[[印尼语|印尼语]]：Pelabuhan Bitung）缩写BIT，是[[印度尼西亚|印度尼西亚]][[北苏拉威西省|北苏拉威西省]][[比通市|比通市]]的海港，是印尼[[比通#经济特区|比通经济特区]]的一部分，由{{le|印尼港口公司|Indonesia Port Corporations}}经营。印尼政府通过2012年第26号总统规定，设定比通作为印尼东部地区远洋以及跨太平洋地区转运商业利益的国际枢纽海港<ref>{{cite web |url=http://haobaodaily.co.id/news/read/2016/02/26/104673/#.WcBk2MiGPic |title=经济统筹部长：比通将成为东部　进出口活动门户 |last= |first= |date=2016-02-26 |website=haobaodaily.co.id |publisher=好报 |access-date=2017-09-19 |quote=}}</ref>。比通港属半日潮港，平均潮差0.9米。港区主要码头泊位岸线长682米，最大水深10.6米，最大可靠6万载重吨的船舶，大船锚地最大水深达34米。装卸设备有起重能力为15吨的岸吊及拖船等，但装卸作业主要使用船上设备。<ref>{{cite web |url=http://article.bridgat.com/guide/trans/port/BITUNG.html |title=BITUNG 比通港口介绍 |last= |first= |date= |website=The Bridgat.com |publisher=Bridgat |access-date=2017-09-14 |quote=}}</ref>

比通港是北苏拉威西省最大的港口和主要椰干出口港，印尼的城际客船经停此港。2012年2月16日，印尼海事渔业部长夏利夫（SharifCicipSutardjo）表示，比通港将被规划成为印尼最大渔港<ref>{{cite web |url=http://news-com.cn/asean/indonesia/a/201202/176147.shtml |title=比通港口将规划成为印尼最大渔港 |last= |first= |date=2012-02-17 |website=新文网 |publisher=印尼星洲日报 |access-date=2017-09-14 |quote=}}</ref>。2014年印尼贸易部将比通港列为进口饮食品、成衣和电子产品的目的港<ref>{{cite web |url=http://www.shangbaoindonesia.com/indonesia-berita/%E6%AF%94%E9%80%9A%E6%B5%B7%E6%B8%AF%E5%92%8Ccikarang%E9%99%86%E6%B8%AF-%E6%94%B9%E4%B8%BA%E8%BF%9B%E5%8F%A3%E7%89%B9%E5%AE%9A%E4%BA%A7%E5%93%81%E7%9B%AE%E7%9A%84%E6%B8%AF.html |title=比通海港和Cikarang陆港改为进口特定产品目的港 |last= |first= |date=2017-04-24 |website=www.shangbaoindonesia.com |publisher=印度尼西亚商报 |access-date=2017-09-14 |quote=}}</ref>。印尼因大量进口纺织品和瓷砖，使其国内纺织业和瓷砖业生产商出现困境，2017年印尼工业部表示其计划将进口纺织品和瓷砖限制在比通港和[[杜迈|杜迈港]]<ref>{{cite web |url=http://www.shangbaoindonesia.com/indonesia-finance/%E5%B7%A5%E4%B8%9A%E9%83%A8%EF%BC%9A%E6%9D%9C%E8%BF%88%E5%92%8C%E6%AF%94%E9%80%9A%E6%B8%AF%E5%8F%A3%E6%88%90%E4%B8%BA%E7%BA%BA%E7%BB%87%E5%93%81%E5%92%8C%E7%93%B7%E7%A0%96%E8%BF%9B%E5%8F%A3%E6%B8%A0.html |title= 工业部：杜迈和比通港口成为纺织品和瓷砖进口渠道 |last= |first= |date=2017-04-24 |website=www.shangbaoindonesia.com |publisher=印度尼西亚商报 |access-date=2017-09-14 |quote=}}</ref>。比通经济特区比通港总规划面积为2,000公顷，其中起步区面积为128公顷 ,起步区建设1个5万吨级集装箱泊位和1个5万吨级多用途泊位<ref>{{cite web |url=http://www.sc-road.cn/park/Introduction?id=1 |title=印尼比通港经济特区 |last= |first= |date= |website=www.sc-road.cn |publisher=丝路驿站集团有限公司 |access-date=2017-09-17 |quote=}}</ref>。

==码头==
比通港共有四个码头：
*Pier Ocean：长607米，水深5米
*Pier Nusantara：长652米，水深6米
<small>集装箱码头：</small>
*Pier VIII：长182米，水深20米
*Pier IX：长60米，水深10米
