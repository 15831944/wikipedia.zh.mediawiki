在[[漢語|中文]]裡，'''比率'''這個詞被用來代表兩個數量的'''比值'''，這包括了兩個。。相似卻在用法上有所區分的概念：一個是[[比例|比]]（{{lang|en|ratio}}）的值；另一是'''變化率'''（{{lang|en|rate of change}}，或簡稱{{lang|en|rate}}），是一個數量相對於另一數量的變化量，例如，[[速率|速率]]是物體的移動距離相對於時間的變化量，以每單位時間的移動距離來表示；[[心率|心跳率]]是每分鐘的心跳次數；[[稅#稅率|稅率]]則是每單位收入所應繳的稅金。

==物理學中的比率==
在[[物理學|物理學]]中，當我們提到比率（{{lang|en|rate}}）或變化率（{{lang|en|rate of change}}）時，通常是指某物理量在單位時間內的變化量，以此概念為基礎，可以引申出各種物理變化率。然而，在不同的學術領域中，[[计量单位|單位]]相同的變化率可能有不同的用法，因此有不同的名稱與解釋；相對的，基於[[歷史|歷史]]因素或其它原因，同一個名詞也可能在不同的領域中有相似但是不同的解讀與[[计量单位|單位]]。

=== 流率（{{lang|en|flow rate}}） ===
流率即單位時間內的[[流量|流量]]。在[[流體力學|流體力學]]中，流量的單位可以是[[體積|體積]]，這稱為[[體積流率|體積流率]]；流量的單位也可以是[[質量|質量]]，這稱為[[質量流率|質量流率]]。[[水文學|水文學]]中的「流量」（{{lang-en|discharge}}）也是一種體積流率，但是它有另外的研究內涵。雖然流率在概念上只與流量和時間有關，但是在測量與計算上，一個可用的定義是[[流體動力學|流體速度]]與所通過的[[曲面|表面]]或[[截面_(幾何)|截面]]的[[內積|內積]]，亦即流體在單位時間內垂直通過某表面或截面的流量。

=== 通量（{{lang|en|flux}}） ===
從流率可以導出[[通量|通量]]，但是通量的確切定義要依照學術領域或是討論對象來決定。

在[[流體力學|流體力學]]中，通量是流體在單位時間內垂直通過單位表面或截面的[[流量|流量]]，亦即流率的面積密度，例如，體積通量與[[質量通量|質量通量]]。因此，在已知流體通量的情形下，流率是通量的[[面積分|面積分]]，而在已知流率與[[正交|正交]]面積的情形下，通量即流率與正交面積的[[比例|比例]]。

在[[電磁學|電磁學]]中，通量的定義是[[電場|電場]]或[[磁場|磁場]]通過某正交表面或截面的流量（[[場線|場線]]的總數，參見[[電通量|電通量]]與[[磁通量|磁通量]]），這與流率的定義相似，不同的是這裡沒有[[時間|時間]]的概念，在數學上則是[[向量場|向量場]]與正交表面或截面的面積分。場線的面積密度可以用來比較場的強弱：[[電通量|電通量]]的面積密度稱為「電通密度」，[[磁通量|磁通量]]的面積密度稱為「磁通密度」。

== 比例 ==
[[File:Aspect-ratio-4x3.svg|thumb]]的长宽比]]
{{main|比例}}

在[[数学|数学]]中，'''比'''（{{lang|en|ratio}}）是同类型的两个数之间的关系<ref>Wentworth, p. 55</ref> （例如，[[对象|对象]]，人，学生，或任何相同单位的数值）。通常表示为<math>a</math>比<math>b</math>，或<math>a : b</math>，有时表示为两个数的无量纲[[商数|商数]]，<ref>New International Encyclopedia</ref>以明确表示第一个数是第二个数的几倍（不一定为[[整数|整数]]）。<ref>Penny Cyclopedia, p. 307</ref>

通俗地说，比例表示在一个东西中，有多少是特定的东西。
例如，假设有一碗水果，其中有8个柳橙和6个柠檬，那么柳橙与柠檬之比为<math>4 : 3</math>（相当于<math>8 : 6</math>），而柠檬与柳橙之比为<math>3 : 4</math>。
另外，柳橙与所有水果之比为<math>4 : 7</math>（相当于<math>8 : 1</math>4）。比例<math>4 : 7</math>可以转化为[[分数|分数]]<math>\frac{4}{7}</math>，表示有多少水果是柳橙。

=== 符号和术语 ===
数<math>A</math>和<math>B</math>的比例可以表示为：<ref>New International Encyclopedia</ref>
* <math>A</math>与<math>B</math>之比
* <math>A</math>比<math>B</math>
* <math>A : B</math>
* <math>A</math>除以<math>B</math>商的[[分数|分数]]（[[有理数|有理数]]）
* <math>\frac{A}{B}</math>

表达<math>A : B</math>与<math>C : D</math>相等的形式为<math>A : B = C : D</math>或<math>A : B :: C : D</math>。口头或书面的表达形式也可以为「<math>A</math>比<math>B</math>等于<math>C</math>比<math>D</math>」。

在比例中，<math>A, B, C, D</math>有特定的名称。<math>A, D</math>称为比例的'''外项'''，<math>B, C</math>称为比例的'''内项'''。比例中，若有三个或更多个比例相等，则称为'''连比'''。<ref>New International Encyclopedia</ref>
