'''威廉·亨利·盖茨三世'''{{KBE}}（{{lang-en|William Henry Gates III}}，{{bd|1955年|10月28日|catIdx=Gates, Bill}}<ref>Bill Gates: Chairman, Microsoft Corp. http://www.microsoft.com/presspass/exec/billg/bio.mspx</ref>），通稱'''比爾·蓋茨'''（{{lang|en|Bill Gates}}{{NoteTag|Bill是其名William縮短發音之後形成的通稱。}}），是一名[[美國|美國]]著名[[企業家|企業家]]、[[投资者|投资者]]、[[軟件工程師|軟件工程師]]、[[慈善家|慈善家]]。他與[[保羅·艾倫|保羅·艾倫]]一起創建[[微軟公司|微軟公司]]，曾任微軟[[董事長|董事長]]、[[CEO|CEO]]和首席軟件設計師，並持有公司超過8%的[[普通股|普通股]]，也是公司最大的個人股東。<ref>[http://www.microsoft.com/msft/reports/proxy2006.mspx Microsoft 2006 Proxy Statement]. 2006-10-06.</ref><ref>http://finance.ifeng.com/a/20140619/12572191_0.shtml</ref>

1995年到2007年的《[[福布斯|福布斯]]》[[全球富豪榜|全球富豪榜]]中，比爾·蓋茨曾經連續13年蟬聯世界首富。2008年6月27日離開微軟公司，並把580億[[美元|美元]]個人財產捐到[[比尔及梅琳达·盖茨基金会|比尔及梅琳达·盖茨基金会]]<ref>比尔盖茨:27日正式退休 580亿美元做慈 http://tech.qq.com/zt/2008/bill_gates/</ref>。《[[福布斯|福布斯]]》杂志2014年美国富豪排名，比尔·盖茨以812億美元资产重登榜首。比爾·蓋茨在彭博亿万富翁指数和福布斯2019年億萬富翁排行榜中名列第2位，資產達到1000億美元<ref>{{Cite web |url = https://www.forbes.com/profile/bill-gates/?list=billionaires#278a075c689f | title = #2 Bill Gates | publisher = Forbes |language =en |accessdate = 2019 -03-03 }}</ref>

盖茨在2000年1月辞去微软首席执行官。他仍然担任董事长，并为自己创立一个新职位「首席软体架构师」（Chief Software Architect）。2006年6月，盖茨宣布，他将在微软的全职工作转变为兼职工作，他渐渐的转移他的职责。他于2014年2月辞去微软的董事长，同时仍作为技术顾问，協助新任命的首席执行官[[萨帝亚·纳德拉|萨帝亚·纳德拉]]。

2017年11月当选为[[中国工程院外籍院士|中国工程院外籍院士]]<ref>[https://www.youtube.com/watch?v=Tj-YlwSLMsQ&t=1382s 東森官方頻道-比尔·盖茨加入外籍院士]</ref>。

== 早年生活 ==
[[File:Bill_Gates_mugshot.png|thumb]][[新墨西哥州|新墨西哥州]][[阿布奎基_(新墨西哥州)|阿布奎基]]，盖茨因交通违规，在警局被拍留有案底的照片。该照片的轮廓为[[Microsoft_Outlook|Microsoft Outlook]] 2010的联系人默认头像<ref>{{Cite web|url=https://www.ithome.com/html/it/355793.htm|title=微软Outlook联系人默认头像揭秘：竟是比尔·盖茨入监照片轮廓 - 比尔盖茨,Outlook - IT之家|accessdate=2018-04-17|work=www.ithome.com}}</ref>]]
1955年10月28日，比爾·蓋茨出生於華盛頓州西雅圖的一個家庭，父親[[老威廉·H·盖茨|老威廉·H·盖茨]]（William Henry Gates, Sr.）是當地的著名[[律師|律師]]，他過世的母親[[瑪麗·馬克斯韋爾·蓋茨|瑪麗·馬克斯韋爾·蓋茨]]是银行系统董事，他的外祖父[[J·W·马克斯韦尔|J·W·马克斯韦尔]]（J. W. Maxwell）曾任国家银行行长<ref>[http://www.historylink.org/index.cfm?DisplayPage=output.cfm&file_id=2296 "玛丽盖茨于1994年6月10日去世"](英文)</ref>。比尔和兩個姊妹一塊長大，曾就讀於西雅圖的公立小學和私立的[[湖濱中學|湖濱中學]]，在湖濱中學蓋茨認識了保羅艾倫，比爾·蓋茨是一名出色的學生，在他13歲时候就開始了電腦[[程式設計|程式設計]]<ref name="gcsm">[http://www.microsoft.com/china/billgates/ 微軟公司主席和首席軟體設計師]，Microsoft Corporation</ref>，而且以[[个人自由|个人自由]]和维护自己的[[知识产权|知识产权]]而聞名。

[[File:Poker_Room.jpg|thumb]]宿舍楼的扑克室是比爾·蓋茨與夥伴保羅·艾倫創建微軟公司的地方。]]
在1973年，盖茨进入[[哈佛大学|哈佛大学]]成为了一名新生。他在前任的微软首席执行官[[史蒂夫·巴爾默|史蒂夫·巴爾默]]家的大厅住了下来。在哈佛期间，盖茨开发了一个编程语言版本，并為第一家微型计算机公司 - MITS - 的Altair牛郎星8800 - 设计了BASIC解譯器。

在大三那年，盖茨离开了哈佛，他将全部精力投入到了微软。他于1975年与保羅
一起创立了微软公司。盖茨认为電腦将会是一个宝贵的工具，它们将出现在每个办公室、每一个家庭的桌面上，在这样的理念之下。他们开始为个人電腦开发软件。盖茨本人对于電腦领域的先见之明是微软及其软件产业成功的关键。

在盖茨的领导下，微软不斷追求發展和改善電腦软件的技术，使之更容易操作，更具成本效益和令使用电脑的人更感方便。

== 微软 ==

=== BASIC语言 ===
[[File:Altair_8800_Computer.jpg|left]]
在1970年代，還在[[哈佛大學|哈佛大學]]讀書的蓋茨與夥伴保羅·艾倫一起為[[Altair_8800|Altair 8800]]電腦設計[[Altair_BASIC|Altair BASIC]]解譯器。Altair是第一台商業上獲得成功的個人電腦，而[[BASIC|BASIC]]語言是一種易用易學的[[程式設計語言|電腦程式設計語言]]，蓋茨與艾倫所開發的[[BASIC|BASIC]]版本就是後來的[[Microsoft_BASIC|Microsoft BASIC]]，也是[[MS-DOS|MS-DOS]]作業系統的基礎，而後者又是[[微軟公司|微軟公司]]早期成功的關鍵。Microsoft Basic後來成了[[Microsoft_QuickBASIC|Microsoft QuickBASIC]]，並逐漸演變成為今天依然流行的[[Visual_Basic|Visual Basic]]。

在1970年代早期，比爾·蓋茨寫了一封著名的《[[致愛好者的公開信|致愛好者的公開信]]》，震驚了資訊界。比爾·蓋茨在公開信中說：「有誰會在沒有任何報酬的情況下來做這些專業的工作？什麼樣的愛好者可以為他的產品投入三年的開發時間，並且發現所有的錯誤、編寫文件以及免費發佈這個產品？」比爾·蓋茨宣稱電腦軟體將會是一個龐大的商業市場，資訊愛好者不應該在未獲得原作者同意的情況下隨意複製電腦程式。但是當時的資訊界受到[[黑客|黑客]]文化影響，認為創意與知識應該共享。比爾·蓋茨隨後離開校園，一手創辦了世界上最成功的企業——[[微軟公司|微軟公司]]，並逐漸將軟體產業化。

=== 与IBM合作 ===
但是比爾·蓋茨的商業手法往往招至非議，其中之一就是[[MS-DOS|MS-DOS]]的來源。在1970年代末，[[IBM|IBM]]正在計劃進入[[個人電腦|個人電腦]]市場，並在1981年正式推出了IBM個人電腦。IBM需要為自己的產品尋找基於[[英特爾|英特爾]]x86系列[[處理器|處理器]]的合適[[作業系統|作業系統]]。IBM在與另一家公司簡短談判後找到了[[微軟|微軟]]，據稱微軟在手上沒有任何籌碼即提出DOS一詞。而微軟則又在未告知自己正在與IBM談判的情況下，靠關係找到了[[西雅圖電腦公司|西雅圖電腦公司]]，以據說是5萬[[美元|美元]]的價格向該公司購買他們所開發的作業系統（微軟的支持者稱，當時微軟與IBM有協議，規定微軟不得向外界透露談判事宜）。微軟將該軟體更名為[[PC-DOS|PC-DOS]]，再授權IBM使用該作業系統。微軟還與其他電腦生產商談判，將經過更改後的[[MS-DOS|MS-DOS]]系統安裝到每一台新電腦上。事後西雅圖電腦控告微軟在未告知事實的情況下以極低的價格購買該公司的產品，但是雙方最終達成庭外和解。

1980年代中期比爾·蓋茨對[[光碟|光碟]]作為[[数据储存|数据储存]]媒介的前景感到樂觀，因此積極推廣[[CD-ROM|CD-ROM]]。

=== Windows操作系统 ===
微軟發佈[[Windows_1.0|Windows 1.0]]時，[[蘋果電腦|蘋果電腦]]認為與其產品有相似的[[GUI|GUI]]。結果兩者達成一項許可，給微軟使用和再許可Windows 1.0衍生作品的權利。後來微軟發佈了[[Windows_2.03|Windows 2.03]]和[[Windows_3.0|Windows 3.0]]，蘋果電腦認為這些版本超出了那項許可，侵犯了它的[[版權|版權]]，於是提起訴訟。地區法院裁決因缺乏實際一致性（virtual identity）而不構成侵權。蘋果的上訴被第九巡迴上訴庭駁回。<ref>[http://home.earthlink.net/~mjohnsen/Technology/Lawsuits/appvsms.html Apple Computer, Inc. v. Microsoft Corporation]，美國第九巡迴上訴庭，1994。</ref>

=== 投资 ===
比爾·蓋茨不僅專注於電腦、軟體企業，對生物技術也很有興趣。他是[[ICOS公司|ICOS公司]]的董事長，這是一家專注於[[蛋白質基體|蛋白質基體]]及[[小分子療法|小分子療法]]的公司。比爾·蓋茨還成立了[[Corbis公司|Corbis公司]]，它正在研究開發世界最大的可视讯息資源之一–來自於全球公共收藏和私人收藏的藝術及攝影作品綜合數位檔案。<ref name="gcsm"/>

美國加州漢普頓溪食品公司研發出混合12種植物而成的「人造蛋」，吃起來和雞蛋一樣，希望讓人們吃得更健康；比爾蓋茲、亞洲首富李嘉誠等富豪均看好市場潛力，對該公司共投資2300萬美元。比爾蓋茲除了掏錢投資該公司，還親自擔任該公司顧問，在矽谷大力推廣人造蛋，「我們可在人造蛋裡添加開發中國家飲食缺乏的營養素，解決人民營養不良的問題」<ref>{{Cite news|url=http://www.appledaily.com.tw/appledaily/article/international/20140219/35650676/|title=好吃 李嘉誠投資「人造蛋」不怕膽固醇高 攜手蓋茲砸7億|last=|first=|date=|work=|newspaper=蘋果日報|accessdate=2017-03-02|language=zh-TW}}</ref>。

蓋茲是中國[[一帶一路|一帶一路]]計畫的明確支持者，至2017年止至少七次前往中國大陸。他認為，歐美很多民眾因為經濟競爭落敗而產生「向內看」的心態對世界是危險的，一帶一路戰略讓多國紛紛加入、是中國走出去幫助其他國家，「中國做出很好的表率」；他並佩服[[中共總書記|中共總書記]][[習近平|習近平]]的國內脫貧戰略，而全球的脫貧進展能造成世界更穩定和每個人更大利益<ref>{{cite news|author=陳君碩|publisher=旺報|title=蓋茲讚一帶一路：中國好樣兒的|url=http://www.chinatimes.com/newspapers/20170325000753-260309|date=2017-03-25|accessdate=2017-03-25}}</ref>。他的「蓋茲基金會」正在和[[中華人民共和國商務部|中國商務部]]、[[中華人民共和國農業部|農業部]]合作推進非洲農業的[[可持續發展|可持續發展]]，同時認同中方以[[核能發電|核能發電]]解決[[全球暖化|全球暖化]]的策略是最務實可行、與他長期觀點相同，他旗下[[泰拉能源|泰拉能源]]將與[[中國核工業集團|中國核工業集團]]探討[[第四代反應爐|第四代反應爐]]的[[行波反應爐|行波反應爐]]建設<ref>{{cite news|author=梁世煌|publisher=旺報|title=蓋茲推新核電 6度登陸談合作|url=http://www.chinatimes.com/newspapers/20170325000750-260309|date=2017-03-25|accessdate=2017-03-25}}</ref>。
