《'''比宇宙更遠的地方'''》（{{lang-ja|宇宙よりも遠い場所}}）是由[[MADHOUSE|MADHOUSE]]製作的[[日本|日本]]原創[[電視動畫|電視動畫]]作品，於2017年7月在[[Anime_Expo|Anime Expo]]公開動畫消息<ref>{{Cite web|url=http://news.mynavi.jp/news/2017/07/03/174/|script-title=ja:女子高生、南極へ行く! 完全オリジナルTVアニメ『宇宙よりも遠い場所』|date=2017-07-03|website=Mynavi News|accessdate=2017-11-10|language=ja}}</ref>，於2018年1月播出<ref>{{Cite web|url=http://www.animatetimes.com/news/details.php?id=1510130272|script-title=ja:水瀬いのりさん・花澤香菜さん・井口裕香さん・早見沙織さん出演で、オリジナルTVアニメ『宇宙よりも遠い場所』2018年1月放送開始！|date=2017-11-08|website=animate Times|accessdate=2017-11-10|language=ja}}</ref>。

作品名稱源自於日本首位進入太空的太空人[[毛利衛|毛利衛]]在被邀請至南極[[昭和基地|昭和基地]]時所說的話：「只要數分鐘便能抵達宇宙，但抵達昭和基地要花數日的時間，它比宇宙更遙遠。」

== 登場人物 ==
=== 主角 ===
; {{nihongo|玉木真理|玉木 マリ（たまき まり）||聲：[[水瀨祈|水瀨祈]]<ref name="character">{{cite web|url=http://yorimoi.com/character/|title=CHARACTER|work=電視動畫「比宇宙更遠的地方」官方網站|language=ja}}</ref>}}
: 綽號小理，就讀多多良西高校的高二女生。想在上了高中後做些甚麼，卻總是卻步。因在車站撿到一百萬圓而與報瀨結識，被報瀨邀請同赴南極。平常看起來傻呼呼的，但擁有比任何人還要堅定的精神。
: 頭髮瀏海看起來很奇怪，原因是瀏海是自己剪的。
: 睡相很差，睡覺的時候還會張開眼睛說一些自己都不知道的夢話，因此其他三人總會希望可以避免與她一起睡覺。
; {{nihongo|小淵澤報瀨|小淵沢 報瀬（こぶちざわ しらせ）||聲：[[花澤香菜|花澤香菜]]<ref name="character"/>}}
: 就讀多多良西高校的高二女生，綽號「南極」。看起來很冷漠，但實際上性格害羞淳朴，神經有點大條，对南极有关的事特别认真和严肃。因想尋找母親小淵澤貴子而下定決心前往南極。因遺失打工賺來的一百萬圓而與真理結識，並邀請真理一同前往南極。
: 三年間不斷打工並輾轉各種方式尋求去南極的方法，甚至還拉著真理和日向想去色誘民間南極觀測團的男性帶她們去南極。
: 與其他三人不同的是，在鏡頭前會結巴、緊張不敢看鏡頭。
; {{nihongo|三宅日向|三宅 日向（みやけ ひなた）||聲：[[井口裕香|井口裕香]]<ref name="character"/>}}
: 在真理就讀學校附近的便利商店打工的女生。因為受不了身處團體裡的感覺，國中畢業後就未再升學，但高中畢業認定的模擬考試成績很好，未來想報考大學。暱稱真理為「小決」{{註|玉木真理的名字發音（Tamaki Mari）後面部分和決（kimari）發音相同。}}
: 因為想在認真備考大學前完成一件事情，在聽到店外真理和報瀨的對話後，決定前往南極。
: 在國中時是優秀的短跑選手，但因為某些原因離開了田徑部。起初不太會在報瀨等朋友面前表現出自己動搖的一面，大多數時候都將其藏起來。
; {{nihongo|白石結月|白石 結月（しらいし ゆづき）|聲：[[早見沙織|早見沙織]]<ref name="character"/>}}
: 比真理和报濑小一年的女生，四歲出道的童星，因為忙於工作而交不到朋友。本來想辭退女高中生南極觀測團的企劃，被真理等人感動，答应如果是與她們三人一起的話，就願意去南極。
: 渴望朋友，但因為從小沒交過任何朋友，對朋友這樣的概念沒有頭緒，對此有一點死腦筋。
: 電視上與自己實際上行為不太符合，對此會感到害羞到想死。
: 因為從小就在鏡頭前生活，因此不畏懼在鏡頭前說話。

: 与小渊泽贵子是高中熟人，三年前在贵子的邀请下加入观测队，也是这次探险导致贵子失踪而深感内疚。

; {{nihongo|前川佳苗|前川かなえ（まえかわ かなえ）||聲：[[日笠陽子|日笠陽子]]<ref name="character"/>}}

; {{nihongo|鮫島弓子|鮫島弓子（さめじま ゆみこ）||聲：[[Lynn_(日本女配音員)|Lynn]]<ref name="character"/>}}
: 观测队厨师长。

; {{nihongo|財前敏夫|財前敏夫||聲：[[松岡禎丞|松岡禎丞]]<ref name="character"/>}}
: 观测队通信员。

; {{nihongo|安本保奈美|安本保奈美||聲：[[小松未可子|小松未可子]]<ref name="character"/>}}
: 观测队微生物研究员。很活泼，经常与一行的高中生一起玩。

; {{nihongo|迎千秋|迎 千秋||声：[[寺杣昌纪|寺杣昌纪]]}}
: 观测船船长。

; {{nihongo|冰见大|氷見 大||声：[[福岛润|福岛润]]}}
: 观测队建筑组组长。几乎不说话。白石结月的粉丝。

; {{nihongo|轰信惠|轟 信恵||声：[[阿澄佳奈|阿澄佳奈]]}}
: 同行的地质学家。经常与在日本的恋人阿优联系。

; {{nihongo|佐佐木梦|佐々木 夢||[[远藤绫|远藤绫]]}}
: 同行的天文学家。沉默寡言。

=== 其他人物 ===
; {{nihongo|高橋惠|高橋めぐみ（たかはし めぐみ）||聲：[[金元壽子|金元壽子]]<ref name="character"/>}}
: 與真理於幼稚園就認識，就讀同校同班的好友。真理自小就十分依赖惠，对于真理独自决定去南极感到十分难受而暗中阻止，最终在真理出发前想与其绝交，但被真理拒绝。在真理回来后与其通信聊天时显示她也去了北极圈以内。

; {{nihongo|白石民子|白石民子（しらいし たみこ）||聲：[[大原沙耶香|大原沙耶香]]<ref name="character"/>}}
: 白石结月的母亲、经纪人。强烈要求白石结月去南极。

; {{nihongo|小淵澤貴子|小淵沢貴子（こぶちざわ たかこ）||聲：[[茅野愛衣|茅野愛衣]]<ref name="character"/>}}
: 小淵澤報瀨的母親，曾是南極觀測隊員，是「比宇宙更遠的地方」一書作者。於報瀨就讀中學時失蹤，毫無找尋的線索。报濑去南极后最终找到其遗落的筆記型电脑，可能是为了拿回电脑独自离队而被暴风雪吹走失踪。

; {{nihongo|玉木凛|玉木 リン||配音：[[本渡枫|本渡枫]]}}
: 真理的妹妹，在学校担任学级委员。对姐姐的神经大条十分担心，对于姐姐去南极探险表现依依不舍。
