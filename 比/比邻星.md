'''比鄰星'''或'''毗鄰星'''（Proxima Centauri）位於[[半人馬座|半人馬座]]，是[[半人馬座α|半人馬座α]][[三合星|三合星]]的第三顆星，依[[拜耳命名法|拜耳命名法]]也稱為'''半人馬座α星C'''，是距離[[太陽|太陽]]最近的一顆[[恆星|恆星]]（4.22[[光年|光年]]），恆星分類屬於[[紅矮星|紅矮星]]。

它是由天文學家[[羅伯特·因尼斯|羅伯特·因尼斯]]于1915年在[[南非|南非]]發現的，當時他是擔任[[約翰尼斯堡|約翰尼斯堡]][[聯合天文台|聯合天文台]]的主管。

[[File:Centaurus_constellation_map.png|thumb]]

== 亮度 ==

通常紅矮星的[[亮度|亮度]]都很弱，以肉眼觀測是看不見的，比鄰星也不例外。它的[[視星等|視星等]]是11等，[[絕對星等|絕對星等]]是非常弱的15.5等。如果從半人馬座α[[三合星|三合星]]的其他兩個星觀測，將是4.5等星。 

== 距離、直徑與質量 ==
[[File:Alpha_Centauri_relative_sizes.svg|left]]
由歐洲天文衛星Hipparcos量測到的[[視差|視差]]772.33 ± 2.42 [[角秒|毫角秒]]推算，比鄰星離地球大約是4.22光年遠，或者地球到太陽距離的270,000倍（AU，天文單位）。離它最近的鄰居依序為：半人馬座α三合星的其他兩顆星（0.21光年）、[[太陽|太陽]]（4.22光年）和[[巴納德星|巴納德星]]（6.55光年）。從地球觀測，比鄰星離半人馬座α星A視角約2°，或相當於[[滿月|滿月]]直徑的4倍。由於比鄰星離半人馬座α星AB双星與太陽距離的比率僅僅是20分之1，天文學家猜測它可能是以500,000至2,000,000年或更長的週期在繞半人馬座α星AB双星的一個軌道上運轉。因此，比鄰星也稱為'''半人馬座α星C'''。

用[[歐洲南天天文台|歐洲南天天文台]]（ESO）位在智利的[[甚大望遠鏡|甚大望遠鏡]]在2002年以光學干涉測量得到比鄰星的[[角直徑|角直徑]]為1.02 ± 0.08毫角秒。由已知它的距離，推算實際直徑大約是太陽的7分之1，或者[[木星|木星]]的1.5倍。它的質量約爲太陽的8分之1，或者木星的150倍。

== 形成年代與壽命 ==
[[File:Proxima_Centauri_2MASS_Atlas.jpg|thumb]]
它的形成年代，與半人馬座α星A和B雙星相同，約48.5億年前，比太陽的46億年略早些。但由於大小僅比產生[[氫|氫]]核融合反應所需的[[恆星臨界質量|恆星臨界質量]]稍大，[[融合反應|融合反應]]的速率很慢且不穩定，因此天文學家推算它的壽命可達數千億年以上。

== 表面活動觀測 ==
比鄰星有很活躍的[[色球層|色球層]]活動，在X-光波段可觀測到它色球層的噴發，是屬於典型的[[耀星|耀星]]（[[變星命名|變星命名]]：半人馬座V645）；在[[紫外線|紫外線]]波長觀測它的色球層的變化，得知它的自轉週期大約31天。由90年代[[哈伯太空望遠鏡|哈伯太空望遠鏡]]所量測的資料顯示，比鄰星應存在有一顆尚未觀測到的暗淡伴星。2016年8月24日[[歐洲南天天文臺|歐洲南天天文臺]]正式宣布發現了比鄰星的一顆行星，命名為[[比鄰星b|比鄰星b]]<ref>{{cite web|author1=网易|title=比邻星b或是太阳系外距地最近的宜居星球|url=http://news.163.com/16/0826/07/BVCLFQV400014Q4P.html}}</ref>，它位於比鄰星的[[適居帶|適居帶]]內。即与恒星距离适中、可能有液态水存在的区域。但也有人提出，比邻星是一颗M型矮星，这类恒星比较容易爆发耀斑，破坏行星的大气。

2017年3月24日，位于智利的[[阿塔卡马大型毫米波／亚毫米波天线阵|阿塔卡马]]望远镜观测到比邻星亮度在10秒内上升了1000倍，随后迅速回落。研究分析观测数据后发现，亮度上升集中发生在极短时间内，应该是一次比最强烈的[[太阳耀斑|太阳耀斑]]还强10倍的恒星[[耀斑|耀斑]]。行星“[[比鄰星b|比鄰星b]]”在这次事件中受到的辐射，比通常太阳耀斑爆发时地球受到的辐射高出约4000倍。研究人员认为，比邻星b在它近50亿年的生涯中，可能曾多次遭受强烈耀斑袭击，即使表面曾有过液态水和大气，也早就被摧毁殆尽，环境不适合生命存在。<ref>[http://www.sohu.com/a/225222361_119562　[[新华社|新华社]]北京2018年3月7日电 《天文研究人员发现离我们最近的恒星——比邻星于2017年3月爆发了一次强烈耀斑，导致它的行星被高能射线“烘烤”了一遍》，美国卡内基科学学会研究人员的论文发表在《天体物理学杂志通讯》上]</ref>
