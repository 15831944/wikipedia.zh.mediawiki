{{佛教}}
'''比丘尼'''（{{lang-sa|भिक्षुणी|bhikṣunī}}，{{lang-pi|bhikkhunī}}），又譯為'''苾芻尼'''、'''比呼尼'''、'''除饉女'''、'''乞士女'''、'''[[沙門|沙門]]女'''，中文俗稱'''尼師'''、'''女尼'''、'''尼姑'''（“尼姑”之稱含不敬意）。比丘尼則是指受過[[具足戒|具足戒]]的女性出家僧侶。「尼」在[[梵語|梵語]]中是女性詞尾，[[比丘|比丘]]與比丘尼合稱出家二眾。

受沙彌尼十戒的稱為[[沙彌尼|沙彌尼]]。之后受了式叉摩那戒者，稱為[[式叉摩那|式叉摩那]]，又名學法女。最后受具足戒者为比丘尼。

== 歷史 ==
在佛教僧團中，第一位出家的比丘尼，是養育[[釋迦牟尼|釋迦牟尼]]長大的[[姨母|姨母]][[大愛道|大愛道]]比丘尼<ref>[http://tripitaka.cbeta.org/T24n1478 《大正新脩大藏經》No.1478.《大愛道比丘尼經》]</ref>。在佛陀時代，[[修行|修行]]成就的比丘尼甚多，著名的有[[神通|神通]]第一的[[蓮花色比丘尼|蓮花色比丘尼]]。

印度傳統上對女性有歧視，但佛陀允許建立比丘尼僧團，讓印度女性有多一個選擇。至[[部派佛教|部派佛教]]時期，[[阿育王|阿育王]]派僧伽密多（Sanghamitta）比丘尼僧團遠渡[[斯里蘭卡|斯里蘭卡]]，在[[錫蘭|錫蘭]]建立起[[上座部|上座部]]的比丘尼僧團。後在公元435年，[[錫蘭|錫蘭]]派十一位比丘尼僧團前往當時的[[晉朝|晉朝]]，為[[漢地|漢地]]的比丘尼授二部僧戒，[[漢傳佛教|漢傳佛教]]地區就有比丘尼戒傳承。漢傳佛教地區的比丘尼戒，源自於[[上座部|上座部]]，分別由[[分別說部|分別說部]]以及[[法藏部|法藏部]]傳至中國，直到今日尚有傳承。

1996年，斯里蘭卡引[[韓國|韓國]]比丘尼僧團前往授戒。第一位斯里蘭卡復興比丘尼的是庫蘇馬比丘尼（Bhikkhuni Kusuma）。現時上座部已經復興比丘尼傳承, 比丘尼僧團包括：斯里蘭卡，以及[[泰國|泰國]]第一位比丘尼{{link-en|達曼納達比丘尼|Dhammananda Bhikkhuni}}（俗名Chatsumarn Kabilsingh），西方第一位上座部比丘尼[[艾雅·凱瑪|艾雅·凱瑪]]（Ayya Khema），[[澳洲|澳洲]]第一批上座部四位受[[具足戒|具足戒]]的尼眾，香港第一位上座部比丘尼[[胜悦比丘尼|胜悦比丘尼]]（Bhikkhuni Uttaranandi），在[[美國|美國]]、[[英國|英國]]、[[歐洲|歐洲]]亦恢復了比丘尼僧團。

[[華人|華人]]世界中，[[中國大陸|中國大陸]]因[[文化大革命|文化大革命]]影響，[[比丘尼戒|比丘尼戒]]傳承一度斷絕，只剩[[港澳|港澳]]、[[马来亚联合邦|新马]]及[[台灣|台灣]]，尚有比丘尼戒傳承。但隨著中國[[改革開放|改革開放]]，[[中國大陸|中國大陸]]的比丘尼傳統也已恢復。

== 戒律 ==
巴利律比丘尼有311條戒。8條驅擯戒；17條暫時拘留戒；30條懺悔心墮落戒；166條心懺悔戒；75條學處戒。

== 現代發展 ==
{{main|八敬法}}
但是有一些戒律是專門為了比丘尼（比丘尼戒約漢傳是348條, 南傳是311條）而設定，其中最受爭議的是[[八敬法|八敬法]]。非在[[波羅夷法|波羅夷法]]及[[僧殘法|僧殘法]]中，繼承了分別說部的巴利律166條心懺悔戒裡有八敬法的規定。

現代被部分佛教徒認為是歧視女性的規定。但由於這些規定是從佛陀時代所流傳下來的，認為應該廢除與不應廢除的分成兩種意見，各自引經律來說明。[[分別說部|分別說部]]與[[說一切有部|說一切有部]]都有流傳有八敬法，但是[[大眾部|大眾部]]只有敬法，沒有八敬法的規定，規定上較為寬鬆。[[漢傳佛教|漢傳佛教]]戒律源自於說一切有部，[[上座部佛教|上座部佛教]]源自分別說部，傳統上都遵守八敬法。隨住南傳上座部各國恢復比丘尼僧團, 這八條戒與其他百幾條戒都隨時地而有所變遷。台灣[[釋昭慧|昭慧]]法師為首的比丘尼僧團更公開表示應廢除八敬法。

兩派爭論的焦點是八條戒是否在波闍波提喬達彌要求出家時，佛陀就制定八條戒。根據佛陀隨犯隨制戒的原則，此八條不可能在其姨母出家時就提出。主要是當中的大戒，是佛陀在證悟後十二年才制定。大部份上座部比丘還是讚成恢復比丘尼僧團，而將此八條戒模糊化。
