== 職業生涯 ==
2016年，蒙格加入{{tsl|en|JHR Developments|JHR發展車隊}}，參賽於{{tsl|en|F4 British Championship|英國四級方程式錦標賽}}，<ref>{{cite news|url=http://www.thecheckeredflag.co.uk/2016/02/monger-stays-with-jhr-for-msa-formula-move/|title=Monger stays with JHR for MSA Formula move|last=Paice|first=Simon|date=2016-02-25|publisher=thecheckeredflag.co.uk|accessdate=2016-02-26|deadurl=no|archiveurl=https://web.archive.org/web/20160226104617/http://www.thecheckeredflag.co.uk/2016/02/monger-stays-with-jhr-for-msa-formula-move/|archivedate=2016-02-26}}</ref>共取得3次頒獎台完賽，年度積分名列第12。隔年，他選擇繼續效力該隊。2017年4月16日，蒙格在{{tsl|en|Donington Park|多寧頓公園|多寧頓}}分站發生一起高速撞車事故，致使他的雙腿遭到截肢。<ref>{{cite web |url=https://www.bbc.co.uk/sport/motorsport/39615361 |title=Formula 4: British driver Billy Monger suffers leg injuries in crash |work=[[BBC體育|BBC Sport]] |date=2017-04-16 |accessdate=2018-05-07 |deadurl=no |archiveurl=https://web.archive.org/web/20171014005001/http://www.bbc.co.uk/sport/motorsport/39615361 |archivedate=2017-10-14 }}</ref><ref>{{cite web |url=http://www.bbc.co.uk/news/uk-england-leicestershire-39647622 |title=Formula 4: British driver Billy Monger has legs amputated |work=[[英國廣播公司新聞|BBC News]] |date=2017-04-19 |accessdate=2018-05-07 |deadurl=no |archiveurl=https://web.archive.org/web/20171004145754/http://www.bbc.co.uk/news/uk-england-leicestershire-39647622 |archivedate=2017-10-04 }}</ref>他的車隊在募款平台{{tsl|en|JustGiving|JustGiving}}發起專案，在24小時內便為蒙格籌得超過50萬[[英鎊|英鎊]]。包括[[简森·巴顿|简森·巴顿]]、[[路易斯·咸美頓|路易斯·咸美頓]]、[[馬克斯·維斯塔潘|馬克斯·維斯塔潘]]、[[丹尼尔·里卡多|丹尼尔·里卡多]]及[[尼古拉斯·賀根堡|尼古拉斯·賀根堡]]在內的[[一级方程式赛车|一级方程式赛车]]車手，也透過各種行動幫助蒙格度過難關。<ref>{{cite web |url=https://www.justgiving.com/crowdfunding/billymonger23 |title=Crowdfunding to Help Billy Monger beat life changing injuries, after being involved in one of the most horrific crashes Motorsport 據2017年6月報導，蒙格將在2017年11月重返賽場，與四肢截肢車手{{tsl|fr|Frédéric Sausset|弗雷德里克·索塞}}共同駕駛{{tsl|en|Group CN|CN組}}{{tsl|en|Ligier|利吉爾}}JS53 Evo 2-本田，出賽位於[[葡萄牙|葡萄牙]]{{tsl|en|Autódromo do Estoril|埃斯托里爾賽道}}的{{tsl|en|V de V Sports|V de V Sports|V de V原型挑戰耐力賽}}，以為派出三名殘疾車手組隊出賽2020年[[勒芒24小时耐力赛|勒芒24小时耐力赛]]的目標奠定基礎。<ref>{{cite web |url=http://www.autosport.com/news/report.php/id/130192 |title=Billy Monger to make racing return as Sausset targets Le Mans |last=Watkins |first=Gary |date=2017-06-17 |website=autosport.com |access-date=2017-06-17 |deadurl=no |archiveurl=https://web.archive.org/web/20170620044303/http://www.autosport.com/news/report.php/id/130192 |archivedate=2017-06-20 }}</ref>2017年7月，蒙格在{{tsl|en|Brands Hatch|布蘭茲-哈奇賽道}}駕駛改裝手動控制裝置的[[大众甲壳虫|大众甲壳虫]]進行測試。<ref>{{cite web |url=http://www.bbc.co.uk/news/uk-england-40497230 |title=Billy Monger crash: Amputee teen racer back behind wheel |author= |date=2017-07-04 |website=[[BBC在線|bbc.co.uk]] |access-date=2017-07-04 |deadurl=no |archiveurl=https://web.archive.org/web/20170704162403/http://www.bbc.co.uk/news/uk-england-40497230 |archivedate=2017-07-04 }}</ref>

2018年2月，蒙格自撞車事故後首度駕駛[[開輪式賽車|開輪式賽車]]，他在{{tsl|en|Oulton Park|奧爾頓公園}}為{{tsl|en|Carlin Motorsport|卡林車隊}}測試[[英國車手俱樂部英國三級方程式錦標賽|英國車手俱樂部英國三級方程式錦標賽]]車輛。<ref>{{cite web |url=http://www.espn.co.uk/f1/story/_/id/22354372/double-amputee-billy-monger-completes-first-single-seater-car-test |title=Double amputee Billy Monger completes first single-seater car test |last=Saunders |first=Nate |date=2018-02-07 |website=ESPN |access-date=2018-03-28 |deadurl=no |archiveurl=https://web.archive.org/web/20180329054246/http://www.espn.co.uk/f1/story/_/id/22354372/double-amputee-billy-monger-completes-first-single-seater-car-test |archivedate=2018-03-29 }}</ref>蒙格於隔月證實，他將會在此賽道為卡林車隊出賽2018年英國車手俱樂部英國三級方程式錦標賽開幕站，<ref>{{cite web |url=https://www.autosport.com/national/news/135060/monger-racing-return-gets-green-light |title=Billy Monger's racing return confirmed for BRDC British F3 opener |last=Mackley |first=Stefan |date=2018-03-27 |website=autosport.com |access-date=2018-03-28 |deadurl=no |archiveurl=https://web.archive.org/web/20180328055817/https://www.autosport.com/national/news/135060/monger-racing-return-gets-green-light |archivedate=2018-03-28 }}</ref>最終，他駕駛著將油門改裝至方向盤上的車輛，於奧爾頓公園取得第3名。<ref>{{Cite web | url = https://www.bbc.co.uk/sport/motorsport/43606264 | title = Billy Monger: Double amputee hails 'surreal' podium finish on British F3 debut | publisher = [[BBC體育|BBC Sport]] | accessdate = 2018-03-31 | date = 2018-03-31 | deadurl = no | archiveurl = https://web.archive.org/web/20180331173628/http://www.bbc.co.uk/sport/motorsport/43606264 | archivedate = 2018-03-31 }}</ref>
