'''比約恩·庫伊佩爾斯''' ({{lang|nl|Björn Kuipers}},{{IPA-nl|ˈbjʏrn ˈkœypərs}};1973年3月28日—)，生于荷兰[[上艾瑟尔省|上艾瑟尔省]][[奥尔登扎尔|奥尔登扎尔]]，是一名足球[[裁判_(体育)|裁判]]，主要在国内[[荷兰足球甲级联赛|荷甲联赛]]和各类欧洲赛事中执法，被评为欧足联精英裁判（UEFA Elite referees）。<ref>{{en}}{{cite web
|url   = http://footballrefereeing.blogspot.com/p/uefa-referee-categories.html
|title = 欧足联裁判名录| accessdate  =  2012-06-16 }}</ref>

==欧洲赛事执法经历==
2006年，库伊佩尔斯取得FIFA国际裁判资格。<ref>{{en}}{{cite web
|url   = http://worldreferee.com/site/copy.php?linkID=548&linkType=referee&contextType=bio
|title = worldreferee.com - Björn Kuipers|date = 2012-06-06 }}</ref>同年，他执法了[[欧洲U-17足球锦标赛|欧洲U-17足球锦标赛]]决赛。此后，库伊佩尔斯曾多次在[[欧洲联盟杯|联盟杯]]、[[欧足联欧洲联赛|欧罗巴联赛]]、[[欧洲冠军联赛|冠军联赛]]和[[欧洲足球锦标赛|欧锦赛]]中担任裁判，属于荷兰足协A级裁判之列。
===冠军联赛===
2012年4月，库伊佩尔斯在冠军杯复赛次回合AC米兰对阵巴塞罗那的比赛中担任裁判，比赛进行至第40分钟，巴塞罗那准备开出角球，米兰中卫[[内斯塔|内斯塔]]在禁区内拉扯对方中场[[布斯克茨|布斯克茨]]球衣后将其拉倒，主裁判库伊佩尔斯决定判给巴塞罗那一个点球，这一判罚引起了极大争议，库伊佩尔斯也在赛后迅速成名。<ref>{{zh-hans}}{{cite web

==个人生活==
库伊佩尔斯的父亲也是一名裁判。年轻时代的库伊佩尔斯希望当一名职业足球运动员，他曾在当地业余球队快速’20（Quick '20）效力。后来，随着他对球员生涯失去兴趣，库伊佩尔斯继承父业转作裁判员。<ref name = goal/>
库伊佩尔斯曾在奈梅亨大学（Radboud University Nijmegen）修习经管学科。除裁判员以外，他还兼任数家荷兰连锁超市C1000的共有人，并在奥尔登扎尔开有一间发廊。<ref name = goal/>
