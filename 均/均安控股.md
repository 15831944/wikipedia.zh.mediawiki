'''均安控股有限公司'''，簡稱'''均安控股'''（'''{{lang-en|Kwan On Holdings Limited}}'''，{{hkex|1559}}），在1975年，由黃宜通（主席）和林錦權，於[[香港|香港]]（總部）成立「義年益建築工程有限公司」。於2016年1月,該公司市值大約為4億元。

業務香港在經營土木工程，包括水務工程、道路造橋等<ref>{{Cite web |url=http://www.kwanonconstruction.com/wp/?page_id=137 |title=均安控股有限公司資料 |access-date=2015-03-31 |archive-url=https://web.archive.org/web/20150402090747/http://www.kwanonconstruction.com/wp/?page_id=137 |archive-date=2015-04-02 |dead-url=yes }}</ref>。

股份在2015年3月27日於港交所創業板上市，當時代碼為8305.HK。配售股份價格為0.3港元，配售股份數目為240,000,000股，集資額為2,670萬港元<ref>{{Cite web |url=http://kwanonconstruction.com/wp/wp-content/uploads/2013/04/chinese2.pdf |title=均安控股有限公司以配售方式 在香港聯合交易所有限公司 創業板上市通告 |access-date=2015-03-31 |archive-url=https://web.archive.org/web/20150402133331/http://kwanonconstruction.com/wp/wp-content/uploads/2013/04/chinese2.pdf |archive-date=2015-04-02 |dead-url=yes }}</ref><ref>{{Cite web |url=http://kwanonconstruction.com/wp/wp-content/uploads/2013/04/LISTING-ON-THE-GROWTH-ENTERPRISE-MARKET-OF-THE-STOCK-EXCHANGE-OF-HONG-KONG-LIMITED-BY-WAY-OF-PLACING1.pdf |title=均安控股有限公司招股資料 |access-date=2015-03-31 |archive-url=https://web.archive.org/web/20150402090445/http://kwanonconstruction.com/wp/wp-content/uploads/2013/04/LISTING-ON-THE-GROWTH-ENTERPRISE-MARKET-OF-THE-STOCK-EXCHANGE-OF-HONG-KONG-LIMITED-BY-WAY-OF-PLACING1.pdf |archive-date=2015-04-02 |dead-url=yes }}</ref><REF>[http://hk.apple.nextmedia.com/financeestate/art/20150327/19091243 實戰理論：結算前搵食甚艱難]</REF>。

2016年8月15日（美食博覽最後1天閉幕時期），該公司由創業板轉為主板上市<REF>[http://www.hkexnews.hk/listedco/listconews/sehk/2016/0805/LTN20160805953_C.pdf 由香港聯合交易所有限公司創業板 轉往主板上市通告]</REF>。
