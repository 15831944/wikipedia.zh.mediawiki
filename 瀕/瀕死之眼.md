《'''瀕死之眼'''》是[[日本|日本]]作家[[東野圭吾|東野圭吾]]的[[長篇小說|長篇]][[推理小說|推理小說]]。2007年[[光文社|光文社]]發行[[單行本|單行本]]，2011年發行[[文庫版|文庫版]]。繁體中文版由[[高寶國際有限公司|高寶國際有限公司]]在台發行。

首發於《[[小說寶石|小說寶石]]》1998年2月號，連載一年。8年後才發行單行本，相比起同期發售的作品《[[使命與心的極限|使命與心的極限]]》、《[[流星之絆|流星之絆]]》的間隔要長得多，有「幻之作品」之稱。東野也表示「現在回讀這本書，我想我是寫不出來了」。<ref>取自新星出版社2013年3月月出版書籍《-{zh:東野圭吾的最后致意;zh-hans:东野圭吾的最后致意;zh-hant:東野圭吾的最后致意}-》，ISBN 9787513311052，翻譯者： 潘璐</ref>

== 故事簡介 ==
雨村慎介下班回家的時候被人毆打頭部，受了重傷幾乎喪命。數日後恢復意識的他發現自己有一部分重要的記憶忘記了。那是段關於自己之前駕車引發交通事故致人死亡的一段記憶。雨村想要恢復自己的記憶，但卻回想不起來那時的自己到底要幹什麼。就在這時他的身邊開始發生了奇怪的事：先是同居的女友失踪，然後神秘的女人出現了。

在序章中敘述的死於交通事故的女人，她臨死前的瞬間將所見烙印於瞳孔中的那雙眼睛的魔力，逐漸將一切支配……

: 來東京後大學退學，然後進入夜之世界。現在在酒吧「茗荷」裡做調酒師。發生意外失去了過去的部分記憶。

; {{nihongo|村上 成美|{{ruby|村上|むらかみ}} {{ruby|成美|なるみ}}|Murakami Narumi}}

; {{nihongo|江島 光一|{{ruby|江島|えじま}} {{ruby|光一|こういち}}|Ejima Kouichi}}
: 酒吧「天狼星」的老闆。發現慎介的調酒師才能，曾讓他到店裡幫忙一陣子，慎介發生意外後就讓他回到「茗荷」。性格寬容，也有強勢的一面。

; {{nihongo|木内 春彥|{{ruby|木内|きうち}} {{ruby|春彦|はるひこ}}|Kiuchi Haruhiko}}
: 帝都建設的職員。因對慎介駕駛的車避讓不及，結果將一個女人當場碾死，是事故的當事人。

: 在製造塑膠人體模特的工作室工作。是交通事故意外死亡的女人.美菜繪的丈夫。襲擊慎介之後服毒自殺。
