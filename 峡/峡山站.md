'''峡山站'''，旧称'''昌邑站'''、'''丈岭站'''<ref>{{cite web |title=峡山区胶济客专开办客运改造工程现场踏勘会议召开 |url=http://www.wfxiashan.gov.cn/xsdt/jrxs/201801/t20180111_2721282.htm |website=潍坊市峡山区政务信息网 }}{{Dead link|date=2019年4月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。是一个位于[[胶济线|胶济线]]和[[胶济客运专线|胶济客运专线]]上的铁路车站<ref>{{cite web |title=胶济铁路将成国内首条正式运行的铁路客运专线--时政--人民网 |url=http://politics.people.com.cn/GB/14562/8556903.html |website=人民网}}</ref>，位于[[山东省|山东省]][[潍坊市|潍坊市]][[昌邑市|昌邑市]]丈岭镇，建于1902年，后一度废弃，2002年重新启用<ref>{{cite web |title=百年胶济线上的那些车站_网易新闻 |url=http://news.163.com/14/0915/12/A66EC5P700014AED.html |website=网易新闻}}</ref>，2008年实现客货分线运输<ref>{{cite web |title=胶济客专新年第一个站改工程昌邑站顺利完成_客运专线_中华铁道网 |url=http://www.chnrailway.com/news/20090109/0109260609.html |website=中华铁道网}}</ref>，目前为[[四等站_(中国大陆)|四等站]]，邮政编码为261319。目前客运：不办理旅客乘降，仅办理普速列车待避动车组列车；货运：办理整车货物发到；危险货物仅办理农药、化肥发到。

2018年6月21日，中国铁路总公司关于部分车站命名和更名的通知（铁总客电[2018]74号），自2018年9月1日起，济南局集团公司昌邑站更名为峡山站。
