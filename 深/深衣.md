'''深衣'''是一种記載於[[中國|中國]][[先秦|先秦]]時期、於[[漢代|漢代]]失傳後被[[宋朝|宋朝]]學者[[朱熹|朱熹]]重新設計或復原的中國古代服飾，後來再傳至[[朝鮮_(稱謂)|朝鮮]]，是自宋代以降，士庶在部份禮儀場合如[[冠禮|冠禮]]、[[祭祀|祭祀]]等，以及部份[[文人|文人]]、[[儒者|儒者]]欲向他人表示尊重古禮、不同俗流的人物才於平日穿用的服飾<ref name = lee&chan2012>李申＆陳衛平《哲學與宗教》，上海人民出版社，2012年，第154-155頁</ref><ref name = dong>董進《Q版大明衣冠圖志》，北京邮电大学出版社，2011年，第292頁</ref>。

== 得名 ==
在《[[礼记|礼记正义]]· 深衣》中有提到其他的服饰的上衣和下裳均不相连，而「深衣」上衣和下裳相连，将整个身体均包裹在内，被体深邃，因此而被称作深衣。自古天子诸侯朝会服玄端，燕居服深衣<ref>《礼记·玉藻》“朝玄端，夕深衣”</ref><ref>《[[礼记|礼记正义]]· 深衣》曰：“续衽，钩边。要缝半下。格之高下，可以运肘。袂之长短，反拙之及肘……袂圆以应规，曲夹如矩以应方，负绳及踝以应直……纯袂，缘，纯边，广各寸半。”</ref>。宋末[[金履祥|金履祥]]认为深衣乃古代士大夫闲居之服，合于周礼"规矩准绳" ，并撰《深衣小传》<ref>《與石居深衣之制書》</ref>。

== 復原的漢服深衣及其傳播 ==
===中國===
[[深衣|深衣]]是一種記載於儒家經典《[[禮記|禮記]]》中相傳為[[先秦|先秦]]時代的天子和庶人所共同穿著的服飾、並於[[漢代|漢代]]失傳。[[北魏|北魏]][[魏孝文帝|孝文帝]]推行[[孝文帝改革|漢化]]時曾嘗試按照經典的紀載進行復原，<ref>《資治通鑑》卷137。</ref>歷史上[[宋朝|宋朝]]的[[朱熹|朱熹]]或明末的[[黄宗羲|黃宗羲]]也都曾試圖按照經典重新設計。深衣在《[[禮記|禮記]]》等中國文獻當中作為一種簡便的禮服而被記錄，到了漢代以降，深衣作為一種傳說被人時時想起，並沒有受到太大關注。到了[[北宋|北宋]]中期，[[司馬光|司馬光]]根據古籍記載自製深衣，並且作為[[常服|常服]]穿著。某天司馬光對學者[[邵雍|邵雍]]說：「先生可衣此乎」，邵雍回答：「某為今人，當服今時之衣。」司馬光認為其言合理<ref name = lee&chan2012/>。朱熹以後，深衣只是在一部分儒者之間被使用，且被視為奇裝異服，[[南宋|南宋]]學者[[史繩祖|史繩祖]]就稱深衣冠履為「怪服妖服」，南宋學者[[周密|周密]]也批評道，就在[[慶元黨禁|慶元黨禁]]發生之前，一部分士人身著裒衣博帶（指深衣），裝扮成一副道學家的氣派。《[[宋史|宋史]]》中除了朱熹門人[[黃灝|黃灝]]以及南宋參知政事的[[高定子|高定子]]的事例以外，並沒有其他有關深衣的記載<ref name = lee&chan2012/>。

宋代以後的中國，深衣在特別的場合以外並不被使用。到了明朝，深衣一般只用作特定禮儀場合的服飾，平日並不穿著<ref name = dong/>。《明儒學案》裡也只有很少的有關使用深衣的記載。明代學者[[丘濬|丘濬]]說：「按馬（端臨）氏此言（指馬端臨在《文獻通考》的考究），則深衣次在宋服之者，固已鮮矣。況今又數百年後哉！」表示明代士人通常不穿深衣。1644年，明代學者[[朱舜水|朱舜水]]在[[日本|日本]]被問及深衣是如何製作時，他回答他從未親眼見過深衣<ref name = lee&chan2012/>。現存的明代深衣實物有[[張懋|張懋]]墓出土的深衣<ref name = lee&chan2012/>。

===朝鮮===
[[朝鮮王朝|朝鮮王朝]]從明代傳入深衣，自此成為當地[[士大夫|士大夫]]常穿的服飾，直至[[大韓帝國|大韓帝國]]滅亡後，人們多穿西服，深衣漸漸只在特定場合出現。

== 考究 ==
宋元[[歷史學家|歷史學家]][[马端临|马端临]]在《文献通考》中提到，夏商周三代可考的服制中虽然有所變化，但是除了[[冕服|冕服]]之外，只有[[玄端|玄端]]（[[端衣|端衣]]）和深衣两者最廣為流传。玄端是自天子至士人均可穿着的，而深衣则是自天子至平民百姓均可穿着的。也是古代士人未得功名时所穿衣服<ref>《大明會典》“士人未為官者、則幅巾深衣。”</ref>。深衣裁制时加长了衣襟，穿着符合礼法，因此不论贵贱，不论身份地位均可以穿着，而没有等级之分<ref>“按三代时，衣服之制，其可考见者，虽不一，然除冕服之外，唯玄端(端衣)深衣二者，其用最广。玄端则自天子至士，皆可服之，深衣则自天子至庶人皆可服之…… 至于深衣，则裁制缝衽，动合礼法，故贱者可服，贵者亦可服，朝廷可服，燕私亦可服，天子服之以养老，诸侯服之以祭膳，卿大夫服之以夕视私，庶人服之以宾祭，盖亦未尝有等级也。”</ref>。

然而隨著衣裳形式的失傳，到了後代卻被視為怪異，穿著者往往被認為是腐儒，即使是北宋理學家紹雍也對該服飾有所保留，[[司馬光|司馬光]]、[[呂希哲|呂希哲]]和[[朱熹|朱熹]]等人則只在私底下穿著。<ref>《文獻通考》，卷111，「三代時衣服之制，其可考見者雖不一，然除冕服之外，惟元端、深衣二者其用最廣。元端則自天子至士皆可服之，深衣則自天子至庶人皆可服之，蓋元端者，國家之命服也，深衣者聖賢之法服也。然元端雖曰命服，而本無等級，非若冕弁之服，上下截然者之比。故天子服之而不卑，士服之而不為僭。至於深衣，則裁制縫衽，動合禮法，故賤者可服，貴者亦可服；朝廷可服，燕私亦可服。天子服之以養老，諸侯服之以祭膳，卿、大夫、士服之以夕視私朝，庶人服之以賓祭，蓋亦未嘗有等級也。古人衣服之制不復存，獨深衣則《戴記》言之甚備。然其制雖具存，而後世茍有服之者，非以詭異貽譏，則以儒緩取哂，雖康節大賢，亦有今人不敢服古衣之說。司馬溫公必居獨樂園而後服之，呂滎陽、朱文公必休致而後服之，然則三君子當居官蒞職見用於世之時，亦不敢服此，以取駭於俗觀也。蓋例以物外高人之野服視之矣，可勝唧哉。」</ref><ref name = lee&chan2012/>

== 相關文獻記載 ==
《礼记》的《深衣》、《玉藻》两篇分别记载了对深衣的标准形制。首先，礼经中的深衣由[[苎麻|苎麻]]布制成。<ref>郑玄《毛诗笺》“麻衣,十五升,布深衣也。纯用十五升。”</ref>幅宽二尺二寸。<ref>[[黄宗羲|黄宗羲]]《深衣考》“布幅闊二尺二寸。用二幅，長各四尺四寸，中屈之，亦長二尺二寸。”</ref>每匹大约45-60厘米宽，<ref>《大戴礼记》“布指知寸”言平布十指,即是古一尺,指广一寸。</ref>12米长。在《诗经·蜉蝣》中的“麻衣如雪”中，郑玄注：“麻衣，深衣也。”古时没有棉布，而“布”均指的是[[苎麻|苎麻]]布，別名苧麻、青苧、紵。<ref>《[[日知录|日知录]]》“《诗》：“麻衣如雪，”郑氏曰：“麻衣，深衣也，古时未有棉布，凡布皆麻为之。《记》曰'治其麻丝，以为布帛'是也。”然则深衣亦用白。”</ref>苎麻成布之后，需加灰锻濯漂白，制成白纻、白紵細布，<ref>朱熹《家礼·通礼·深衣制度》“裁用白细布,度用指尺,衣全四幅,其长过胁。”</ref>然后成衣。<ref>礼记深衣注：“深衣者，用十五升布锻濯灰治。”</ref>《乐府白紵歌》曰：“质如轻云色如银，制以为袍余作巾。”中唐时“沤纻为缊袍”、“春衫裁白纻，朝帽挂乌纱。”<ref>常建，《渔浦》</ref><ref>《酬乐天得稹所寄纻丝布白轻庸制成衣服以诗报之》</ref><ref>刘复,唐,《春游曲》</ref>。明代深衣用玉色，素带，素履，白靴。

《[[礼记|礼记正义]]·深衣》中有提到深衣最重要的特征为深衣的衣衽是由一幅布交解裁之，上尖下阔。內衽連于前右之衣，外衽連于前左之衣。<ref>黄宗羲《深衣考》</ref>并且衣襟边缘有勾边。<ref>《[[礼记|礼记正义]]·深衣》:“续衽，钩边。要缝半下。格之高下，可以运肘。袂之长短，反拙之及肘……袂圆以应规，曲夹如矩以应方，负绳及踝以应直……纯袂，缘，纯边，广各寸半。”</ref>也被称为青衿。<ref>毛传：“青衿，青領也。學子之所服。”</ref><ref>《北史》“宣帝時為魯公，與諸皇子等咸服青衿，行束脩之禮，受業於徵。仍並呼徵為夫子，儒者榮之。”</ref>唐宋学生所服。<ref>杜佑《通典》“学生青衿服。”</ref><ref>《新唐書》“請諸生及進士第並謁先師，衣青衿，介幘，以還古制。”“其日，鑾駕將至，先置之官就門外位，學生俱青衿服，入就位。”“其日平明，皇子服學生之服其服青衿。至學門外。博士公服。”</ref><ref>《[[文献通考|文献通考]]》“国子生黑介帻，青衿，单衣”</ref>深衣的领型为斜襟方领。<ref>《後漢書·儒林列傳》“建武五年，乃修起太学，稽式古典，笾豆干戚之容，备之于列，服方领习矩步者，委它乎其中。”</ref><ref>北齐颜之推《颜氏家训·书证》：“按：古者，斜領下連於衿，故謂領爲衿。”</ref><ref>《朱子家礼》“两襟相掩，衽在腋下，则两领之会自方。”</ref>[[司马光|司马光]]指出应頸下別施一衿，映所交領，使之方正尺。但明代后期也出现了方形直襟领。如朝鲜儒学者[[朴世堂|朴世堂]]的深衣。衣领宽二寸。<ref>《礼记·玉藻》“袷二寸。”</ref>袖口广一尺二寸。<ref>《礼记·玉藻》“袪尺二寸”。</ref>衣缘用幅宽三寸黑繒，对折缝之，则广一寸半。<ref>《礼记·玉藻》“緣寸半。”</ref>

上衣用两幅布，对折，两幅布缝拢为后背中缝。<ref>《礼记·深衣》“負繩及踝以應直。”</ref>理學家[[吴澄|吴澄]]说深衣的裳是以六幅布裁为十二片制作的，象征一年十二个月中有六阴六阳。[[司马光|司马光]]依《礼记》作深衣、冠簪、幅巾、缙带。<ref>邵伯温《邵氏闻见录》卷十九</ref>後期的深衣以此為基礎製作，有[[朱子深衣|朱子深衣]]、[[江永深衣|江永深衣]]、[[明深衣|明深衣]]等形制，<ref>朱之瑜《深衣议》“深衣之制有二：一见于玉藻，温公所 并刃， 家礼之所辑是也；其一为明室之制。明室之制有衣而无裳，冠七星巾，系涤纳履，非吉服，非常服，非儒服也。与古相戾，不必言矣。”</ref><ref>[http://nmh.gsnu.ac.kr/jsp/search/search_list_content.jsp?article_id=GAD2_ZZZL_MDJ203_AV003_01_004&flag=mun&searchType=ganada&tableName=MCG_MUN&RECID=5032512&serge_id=GAD2_ZZZL_MDJ203&serge_title=%E6%A2%85%E5%A0%82%E9%9B%86&volume_id=GAD2_ZZZL_MDJ203_AV003&volume_title=%E6%A2%85%E5%A0%82%E9%9B%86%E5%8D%B7%E4%B9%8B%E4%B8%89&style_id=GAD2_ZZZL_MDJ203_AV003_01&style_title=%E9%9B%9C%E8%91%97&searchSetId=190101&pTotalCount=57&second_flag=%20&semyung=%20&author=%20 李隱，《梅堂集·深衣制度》]“”</ref>常配搭[[幅巾|幅巾]]。缁布冠亦用苎麻布，比深衣经纬密度大一倍。<ref>朱熹注《论语》：“缁布冠，以三十升布为之，升八十缕，则其经二千四百缕矣。细密难成，不如用丝之省约。”</ref>大帶广四寸。垂其余為紳，所以服深衣的士大夫又被称为士紳、縉紳、鄉紳。
