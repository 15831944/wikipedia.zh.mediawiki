'''深谷花園站'''（{{jpn|j=ふかや花園駅|hg=ふかやはなぞのえき|rm=Fukaya-hanazono eki|f=y}}），是一個位於[[日本|日本]][[埼玉縣|埼玉縣]][[深谷市|深谷市]]黑田，屬於[[秩父鐵道|秩父鐵道]][[秩父本線|秩父本線]]的[[鐵路車站|鐵路車站]]。2018年10月20日開業於秩父本線{{link-ja|永田站 (埼玉縣)|永田駅 (埼玉県)|永田}}－{{link-ja|小前田站|小前田駅|小前田}}之間。

== 概要 ==
深谷花園站是[[秩父本線|秩父本線]]的一個鐵路車站，深谷花園站距離{{link-ja|永田站|永田駅}}有1.1公里，距離{{link-ja|小前田站|小前田駅}}有2.3公里。

== 車站結構 ==
深谷花園站將設有一個110公尺長的月台，可供「Paleo Express」在此停靠。<ref name="tetsudo20170615">{{cite web|url=https://www.tetsudo.com/news/1756/ |accessdate=2017-06-15 |date=2017-06-15 |work=Tetsudo.com |publisher=Asahi Interactive Inc. |language=ja |archiveurl=https://web.archive.org/web/20170615115606/https://www.tetsudo.com/news/1756/ |archivedate=2017-06-15 |script-title=ja: 秩父鉄道、深谷市に新駅を設置 |trans_title=Chichibu Railway to build new station in Fukaya |location=Japan |dead-url=no }}</ref>

== 歷史 ==
[[秩父鐵道|秩父鐵道]]在2017年6月14日宣佈興建深谷花園站<ref name="saitamashimbun20170615">{{cite web|url=http://www.saitama-np.co.jp/news/2017/06/15/06_.html|accessdate=2017-06-15|date=2017-06-15|work=|publisher=The Saitama Shimbun|language=ja|archiveurl=https://web.archive.org/web/20170615123405/http://www.saitama-np.co.jp/news/2017/06/15/06_.html|archivedate=2017-06-15|script-title=ja: 新駅「ふかや花園駅」秩父鉄道が１８年開業予定　近くにアウトレット|trans_title=Chichibu Railway to open new Fukaya Hanazono Station in 2018 close to outlet mall|location=Japan|dead-url=yes}}</ref>。深谷花園站的建設計劃於2017年6月開始，全部費用約為四億日元。深谷花園站在2018年10月20日開放使用。

== 使用情況 ==
深谷花園站附近的商場開放後預計每年將接待64萬9000名乘客，使其成為繼[[熊谷站|熊谷站]]和[[羽生站|羽生站]]後第三大繁忙的車站，每年的收入將達三億日元。

== 車站周邊 ==
深谷花園站位於[[深谷市|深谷市]]黑田，並與鄰近車站、預計於2020年10月開放的Hanazono Premium Outlet購物中心接駁。
