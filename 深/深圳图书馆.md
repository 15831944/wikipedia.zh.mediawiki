'''深圳圖書館'''（'''Shenzhen Library'''）是[[中國|中國]][[廣東省|廣東省]][[深圳市|深圳市]]的一座[[圖書館|圖書館]]，位於[[福田區|福田區]]福中一路，前身是「寶安縣圖書館」。新館是深圳市政府投資興建的大型現代文化設施，佔地29,612[[平方米|平方米]]，總建築面積49,589平方米，設計藏書容量400萬冊、讀者座席2,000個、網路接點3,000個，每日平均可接待讀者8,000人次。新館於2006年7月12日落成開館<ref>{{cite news|url=http://www.gd.xinhuanet.com/dishi/2006-12/01/content_8675034.htm|title=深圳图书馆新馆今日起全面开放|author=马璇|publisher=深圳特区报|date=2006-12-01|deadurl=yes|archiveurl=https://web.archive.org/web/20070124180027/http://www.gd.xinhuanet.com/dishi/2006-12/01/content_8675034.htm|archivedate=2007-01-24}}</ref>。

== 建築特色 ==
深圳圖書館建築由[[日本|日本]][[建築師|建築師]][[磯崎新|磯崎新]]主持設計，建築造型獨特，富現代感，建築模式從傳統模式變為全開放、大開間、無間隔的「模數式」佈局。與[[深圳音樂廳|深圳音樂廳]]一道構成深圳文化中心。

圖書館的正門寓意文化森林的「銀樹」和音樂廳正門「黃金樹」象徵中心區文化城的「城門」。圖書館南側的三棟黑色放射性建築形狀有如三本翻開的書本，而東面的水幕和[[三維|三維]]玻璃曲面如同「豎琴」的模樣。

== 管理 ==
自1986年开馆起，深圳图书馆即实行全开架式服务，面向所有[[公民|公民]]开放，普通阅览不需要证件。2006年7月新馆开放，图书馆提出“开放、平等、免费”之办馆理念，“拉开了全国公共文化场所全面免费的大幕”<ref name="renmin20130731">{{cite news|url=http://paper.people.com.cn/rmrbhwb/html/2013-07/31/content_1276152.htm|title=深圳图书馆缘何“全球最繁忙”|author=张岩& 张森&林坤城|publisher=[[人民日报|人民日报]]海外版|date=2013年7月31日|accessdate=2015-12-02}}</ref>。

同时深圳图书馆以[[科技|科技]]带动管理与服务，在中国大陆率先实现图书馆的[[信息化|信息化]]，“引领我国图书馆的自动化进程”。自开馆起即使用其自行研制的“实时计算机光笔流通管理系统”，使图书馆的文献流通高效化<ref name="renmin20130731"></ref>。

深圳图书馆是中国大陆首家试行[[法人|法人]]治理结构的公益性文化机构，现由深圳图书馆理事会管理。理事会是深圳图书馆的议事和决策机构，行使大事项议事权和决策权，负责发展战略和规划的确定<ref name="2014report"></ref>。

== 新技術應用 ==
[[File:Shenzhen_LibraryCeiling_20061222.jpg|thumb]]
深圳圖書館將在全中國率先採用DILAS系統和RFID技術。

DILAS（數位圖書館體系結構研究和應用平臺開發項目）是由深圳圖書館自行開發，在全國擁有近3000家用戶，具有自主[[知識產權|知識產權]]並被[[中华人民共和国文化部|文化部]]評為重大科技成果推廣專案及[[国家标准化管理委员会|國家計委]]下達的國家級重點科技項目。

深圳圖書館也將是全中國第一家採用[[RFID|RFID技術]]（無線射頻技術）的圖書館，可為讀者高效利用圖書館和圖書的防盜安全提供技術保障。

深圳图书馆RFID项目由深圳市海恒智能技术有限公司提供方案及实施，于2008年7月21日正式启动[[自助图书馆|24小时街区自助图书馆]]项目，至今为止已有160台自助图书馆设备摆上深圳街头。<ref>{{cite news|url=http://www.infzm.com/content/33126|title=深圳自助图书馆 ATM机：吐的不是钱，是知识|author=万静|publisher=[[南方周末|南方周末]]|date=2009-08-20}}</ref>
