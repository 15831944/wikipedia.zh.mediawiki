'''深圳地铁3號線''' (英語：'''Line 3''') 是[[深圳地鐵|深圳地鐵]]的一條运营中路線，屬於[[城市軌道交通系統|城市軌道交通系統]]，由[[深圳市地鐵三號線投資有限公司|深圳市地鐵三號線投資有限公司]]負責營運。3號線由[[雙龍站_(深圳)|雙龍站]]至[[益田站_(深圳)|益田站]]，總長41.7公里，設30座車站。[[草埔站|草埔站]]以北为高架段（塘坑站除外），草埔站以南为地下段。[[雙龍站_(深圳)|双龙]]至[[草埔站|草埔]]一段於2010年12月28日投入運營。草埔至[[益田站_(深圳)|益田]]段于2011年6月28日开通运营<ref>[http://news.sznews.com/content/2011-06/28/content_5787493.htm 深圳地铁3号线开通首日市民争相「尝鲜」] {{Wayback|url=http://news.sznews.com/content/2011-06/28/content_5787493.htm |date=20120205085342 }}，2011年6月29日，深圳新聞網</ref>。

3號線連接了[[福田|福田]]、[[罗湖区|羅湖]]和[[龙岗区|龍崗]]三個行政區，大大加強了地區間的聯繫。3號線作為市中心至關外龍崗區的唯一主要客運軌道，緩解了布吉、橫崗、龍崗中心等各區多年來的的交通擠塞問題。同時3號線亦是[[深圳市|深圳市]]2011年主辦第26屆[[2011年夏季世界大學生運動會|夏季世界大學生運動會]]的交通樞紐專案。

3號線在路線圖中以<span style="color: {{深圳地鐵色彩|3}};">淺藍色</span>標示。

== 歷史 ==
=== 路線更名 ===
深圳市規劃局在2008年4月23日宣佈，深圳地鐵3號線會易名為「龍崗線」<ref name="szplan-naming-20080423">[http://www.szplan.gov.cn/main/ghdt/tzgg/200804230217672.shtml 關於我市近期建設地鐵線路及網站命名的通告] {{webarchive|url=https://web.archive.org/web/20080430203949/http://www.szplan.gov.cn/main/ghdt/tzgg/200804230217672.shtml |date=2008-04-30 }}，2008年4月23日，深圳市規劃局</ref><ref>[http://news.qq.com/a/20080422/003925.htm 深圳5條地鐵線路有了新名字]，2008年4月22日，深圳新聞網</ref>。

=== 第一期工程 ===
3號線首期段(雙龍站至紅嶺站)主體工程於2007年7月正式動工，並於2010年8月11日實現全線貫通，全面進入系統聯調和運營籌備的階段<ref>[http://www.peoplerail.com/jianshe/2010811/n680623630.html 深圳地鐵3號線(3號線)首期段實現全線貫通]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}，2010年8月12日，人民鐵道網</ref>。全線設高架站15座，半地下站1座，地下站6座和橫崗車輛段1座。正線全長32.859 km（雙線），其中高架線長21.727 km，地面線（即半地下站，含過渡段）長2.599 km，地下線長8.533 km<ref name="ditiezu">[http://www.ditiezu.com/thread-75933-1-1.html 關於深圳地鐵3號線車站及線路圖]，2010年8月8日，地鐵族</ref>。

;雙龍－草埔
3號線[[雙龍站|雙龍站]]至[[草埔站|草埔站]]一段（高架段）於2007年7月動工興建，在2010年12月28日正式開始運營，行車時間約為32分鐘<ref>[http://www.youtube.com/watch?v=TaPCU5m2LZA 深圳地鐵三號線3號線 0308 雙龍站-草埔站]，2011年1月11日，YouTube</ref>。該段以龍崗區雙龍站為起點，主要沿著[[龍崗大道|龍崗大道]]，經過[[橫崗街道|橫崗]]、[[布吉街道|布吉]]等地到達布吉的草埔站，以高架鐵路（除[[塘坑站|塘坑站]]附近一段）形式建造，設有16個車站，全長25.138公里。車輛段設在[[六約站|六約站]]附近。

;草埔－紅嶺
3號線[[草埔站|草埔站]]至[[紅嶺站|紅嶺站]]以草埔站為起點，經[[水貝站|水貝站]]、[[田貝站|田貝站]]、[[翠竹站|翠竹站]]、[[曬布站|曬布站]]、[[老街站|老街站]]到達紅嶺站，全程在地底行走，本段于2011年6月28日建成通車。

=== 第二期工程 ===
;紅嶺－益田
3號線紅嶺站至[[益田站_(深圳)|益田站]]一段（西延段）主體工程於2008年5月開始動工，於2011年6月28日建成通車。西延段由[[紅嶺站|紅嶺站]]折返線末端開始伸延，經過[[通新嶺站|通新嶺站]]、[[華新站|華新站]]、[[蓮花村站|蓮花村站]]、[[少年宮站|少年宮站]]、[[福田站_(地鐵)|福田站]]、[[購物公園站|購物公園站]]、[[石廈站|石廈站]]到達[[益田站_(深圳)|益田站]]，全段為地下鐵路，正線全長8.8公里（雙線）<ref name="ditiezu"/>。西延段亦會在中心公園新建一處地鐵車輛停車場，用於軌道列車的停車檢修任務，停車場全部設於地下，建成後地面還建公共綠地，地下用地面積約51400平方米<ref>[http://house.focus.cn/news/2010-06-05/953357.html 深圳市地鐵龍崗線西延段中心公園將建停車場]，2010年8月10日，深圳商報</ref>。

== 使用車輛 ==
===北车长客B型电动列车===
{{main|深圳地铁北车长客B型电动列车}}
3號線列車為[[北車長客|北車長客]]所制的地鐵車，採用4動2拖6輛編組，2動1拖為一個動力單元，最高運行速度85km/h，初期配屬車輛數為144輛（24列），及後增添另外19列<ref>[http://bbs.hasea.com/viewthread.php?tid=413030 深圳地鐵三號線（龍崗線）靚圖] {{Wayback|url=http://bbs.hasea.com/viewthread.php?tid=413030 |date=20180108174957 }}，2010年8月8日，海子鐵路網</ref><ref>[http://www.chinahighway.com/news/2009/357550.php 深圳市軌道交通二期龍崗線工程首列車下線]，2010年8月10日，中國公路網</ref>。此外，每節列車亦安裝了空氣淨化系統，為全國最先採用，有效降低細菌總數和濃度<ref>[http://www.building.hk/cn/view.asp?id=1595 地鐵龍崗線車廂安裝空氣淨化器]，2011年1月12日，中港澳建造消息</ref>。该型号列车采用了韩国Rotem的牵引系统以及庞巴迪CityFlo650信号系统，匹配性较差，同时上线前并未进行足够的调试，导致该列车在營運初期，經常出现进站时列车不稳，摇晃的状况。<ref>[http://www.chinabaike.com/z/jz/jt/438604.html 深圳地铁3號線CBTC降级模式下的行车组织探讨]。</ref>

===南车浦镇B型电动列车===
{{main|深圳地铁南车浦镇B型电动列车}}
[[File:Car_03333_of_Line_3_Shenzhen_Metro.jpg|缩略图]]
3號線增購列車為[[南车浦镇|南车浦镇]]所制的地鐵車，採用4動2拖6輛編組，2動1拖為一個動力單元。

== 車站特色 ==
3號線大部分車站都繪有花卉圖案，如[[荷坳站|荷坳站]]以荷花為主題。

== 行车間隔 ==
为满足工作日早晚高峰乘客需求，3號線从2013年1月21日开始，在工作日的上下班高峰时段在华新站至塘坑站间双向增开“早晚高峰区间车”，行车间隔将从5分钟缩短至3分30秒，运能提升45%。2014年8月6日，龙岗线实施3/6分钟长短交路运营组织模式。原来益田-华新、双龙-塘坑间需等待7分钟才一班的地铁列车，现在需等待6分钟就有一班，而华新-塘坑间则只需等待3分钟就有一班列车。自实施以来，车厢拥挤程度有所改善，据悉，3/6分钟模式实施后，全线运能增幅约20%，同时高峰时段也扩展至5小时，实现与客流增长趋势的完美匹配。

== 未來發展 ==

;益田－福保
地铁3号线在工程规划设计阶段，考虑过自益田站向南延长至[[福田保税区|福田保税区]]的方案，但由于保税区实行封闭管理等原因，现阶段的政策不能支持在其内设置车站<ref>[http://wb.sznews.com/html/2012-12/04/content_2301256.htm 地铁3号线或将延至保税区]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

2014年8月20日，深圳市轨道建设指挥部第二十次会议决定近期开展地铁3号线南延段工程。3号线南延段工程属于深圳市轨道三期工程（修编）线路，预计将于2015年9月开工建设。根据三期修编规划方案，3号线南延[[福保站|福保站]]设置于保税区3号门前<ref name="nfdsb-20100808">[http://gd.qq.com/zt2014/futianmspx/jiaotong.html 开展福田保税区交通拥堵综合治理]</ref>。

;雙龍－坪地
3号线將由東面終點站[[雙龍站|雙龍站]]延伸至[[坪地|坪地]]一帶。东延段线路由双龙站引出，沿龙岗大道，规划龙坪大道、坪西路、吉祥路、振兴路敷设，到达富坪街站。线路全长约7.3公里，设站7座，平均站间距为1.22公里，全线采用地下敷设方式，将与坪西路改造工程一体化实施<ref>{{Cite web |url=http://ilonggang.sznews.com/lgnews/content/2015-04/23/content_11503213.htm |title=地铁十号线年内或全面开工 连接福田龙华龙岗 |access-date=2015-04-24 |archive-url=https://web.archive.org/web/20150427234350/http://ilonggang.sznews.com/lgnews/content/2015-04/23/content_11503213.htm |archive-date=2015-04-27 |dead-url=yes }}</ref>。

;延伸至惠州
在2013年8月7日的[[深莞惠党政联席会议|深莞惠党政联席会议]]中，深圳市政府与惠州市政府签订《深惠合作备忘录 (2013年)》，提出两市应成立专项工作小组，启动深圳地铁线路延伸至惠州市境内的路由、投资模式等相关问题研究，深惠双方将切实做好深圳市轨道3号线、12号线等线路延伸至惠州市境内的城市规划、土地利用等各项对接工作，尽早落实延伸线路路由。<ref>《[http://wb.sznews.com/html/2013-08/08/content_2580522.htm 深圳地铁3号线12号线延至惠州]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}》(深圳晚报，2013年8月8日)</ref><ref>《[http://e.hznews.com/paper/hzrb/20130808/A4/2/ 尽快建成深莞惠轨道交通网络]》(惠州日报，2013年8月8日)</ref>
