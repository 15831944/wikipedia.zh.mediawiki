'''深圳实验学校'''（英文：'''Shenzhen Experimental School'''，简称：'''深圳实验'''、'''深实'''或'''SZSY'''）是位于[[中国|中国]][[广东省|广东省]][[深圳市|深圳市]]的一所公办学校，创办于1985年5月。2003年9月，深圳实验学校发起组建了深圳市首个公办教育集团——'''深圳实验教育集团'''。该校拥有小学部、初中部、中学部、高中部四个学部和百花、西丽两个校区。其中百花校区位于深圳市[[福田区|福田区]]百花片区，包括小学部、初中部、中学部；西丽校区位于深圳市[[南山区|南山区]][[西丽街道|西丽片区]]，为高中部所在地。全校共有学生约7000人，教职工约700人。

深圳实验学校系深圳市教育局直属学校、广东省首批“一级学校”，所属高中部为广东省首批“国家级示范性高中”<ref name="深圳实验学校简介">{{cite web |url=http://www.szsy.cn/fbrole/main/content.pr.prContentView.do?Contents/contentID=1201KqYAd7CEMc4Ugvm67h0O9601601 |title=深圳实验教育集团简介 |work=深圳实验学校 |language=zh-hans |access-date=2017-07-10 |archive-url=https://web.archive.org/web/20150924112715/http://www.szsy.cn/fbrole/main/content.pr.prContentView.do?Contents%2FcontentID=1201KqYAd7CEMc4Ugvm67h0O9601601 |archive-date=2015-09-24 |dead-url=yes }}</ref>。与[[深圳中学|深圳中学]]、[[深圳外国语学校|深圳外国语学校]]合称深圳三大名校<ref>{{cite news |author = 孙妍 |url = http://www.sznews.com/news/content/2014-04/25/content_9412683.htm |title = 深圳三大名校自主招生即将启动 |publisher = 深圳新闻网 |date = 2014-04-25 |accessdate = 2014-12-05}}</ref>。

==学校历史==
1985年5月3日，深圳市人民政府正式批准成立，当年9月，借用前深圳园西小学校舍举行开学典礼；

1986年，小学部建成；1987年中学部建成；1988年，幼儿部建成，形成三部格局；

1998年，原深圳福田区华强中学并入为深圳实验学校初中部，形成四部格局；

2001年，收购亚太（深圳）国际学校，原址举办深圳实验学校国际部；

2003年9月3日，深圳实验教育集团成立；

2004—2007年，陆续成立深圳实验教育集团幼教中心、培训中心、服务中心、口腔门诊部；

2002年至2007年先后承办新亚洲幼儿园、盛世幼儿园、嘉宝田幼儿园、万科城幼儿园；

2006年，原所属幼儿部划归深圳市国资委；

2007年，市政府一次性偿还原国际部债务，校舍收归国有，成为公办高中部；

2012年，承办深圳实验星河幼儿园、深圳实验星河小学。<ref>{{cite video |people=深圳实验学校 |date=2015年5月19日 |title=风起蔚蓝：深圳实验30年 |url=http://v.qq.com/boke/page/o/0/c/o0154bhe6ec.html |language=zh |format= |medium= |accessdate=2015年5月21日 |time= |quote=学校历史}}</ref>

==学校架构==
深圳实验教育集团经深圳市教育局批准，于2003年9月成立。它是由深圳实验学校发起、组建，利用深圳实验学校品牌，采用机构办学的方式，具有独立法人的教育事业集团。<ref name="深圳实验学校简介" />

深圳实验教育集团目前有'''深圳实验学校高中部'''、'''深圳实验学校中学部'''、'''深圳实验学校初中部'''、'''深圳实验学校小学部'''和'''深圳实验学校光明部'''5个公办学部，以及深圳实验幼教中心、以及深圳实验教育培训中心、深圳实验口腔门诊部、深圳实验师生生活服务中心等四个相关教育机构，幼教中心包括深圳实验嘉宝田幼儿园、深圳实验新亚洲幼儿园、深圳实验万科城幼儿园、深圳实验盛世幼儿园、深圳实验星河时代幼儿园。<ref name="深圳实验学校简介" />

===高中部===
[[File:深圳实验学校高中部综合楼夜景.jpg|缩略图]]
2001年，按上级教育主管部门要求，深圳实验学校收购原亚太国际学校，在其原址兴办国际部。2002年9月，深圳实验学校高中部搬迁至南山区西丽同发路99号，首先在深圳市创办全寄宿制高中。现任部主任程学军。

高中部占地面积73000多平方米，建筑樓面48000多平方米，有教学楼、综合楼、体育馆、文体馆、学生宿舍、教工宿舍等建筑。高中部拥有约36个教学班、1500多名学生、120名教师。高中部课外活动有运动会、社会实践周、科技节、艺术节、游园会等。<ref name="高中部简介">{{cite web |url=http://www.szsy.cn/fbrole/main/content.pr.prContentView.do?Contents/contentID=1202ktTDdogFd3tPmspbnLGaDvU21603 |title=深圳实验学校高中部简介 |work=深圳实验学校 |language=zh-hans |access-date=2017-07-10 |archive-url=https://web.archive.org/web/20160304081535/http://www.szsy.cn/fbrole/main/content.pr.prContentView.do?Contents%2FcontentID=1202ktTDdogFd3tPmspbnLGaDvU21603 |archive-date=2016-03-04 |dead-url=yes }}</ref><ref>{{Cite web|url=https://www.bilibili.com/video/av14305256|title=2017欢迎来到实验_广告_广告_bilibili_哔哩哔哩|accessdate=2017-11-27|last=花季电视台BTS|work=www.bilibili.com}}</ref>

===初中部===
深圳实验学校初中部位于福田区百花七路2号，前身为深圳市华强中学。初中部现有30个教学班，1380多名学生，教学人员约103人。初中部课外活动有科技节、艺术节、运动会、社会实践周等。初中部自主研发开设了校本课程、活动课程、陶艺课、书法课、研理课等一些有特色的课程以培养学生能力。<ref name="初中部简介">{{cite web |url=http://www.szsy.cn/fbrole/main/content.pr.prContentView.do?Contents/contentID=1204gakACPs9Czv4f4DebdJ7S4a71675 |title=深圳实验学校初中部简介 |work=深圳实验学校 |language=zh-hans |access-date=2017-07-10 |archive-url=https://web.archive.org/web/20160304070752/http://www.szsy.cn/fbrole/main/content.pr.prContentView.do?Contents%2FcontentID=1204gakACPs9Czv4f4DebdJ7S4a71675 |archive-date=2016-03-04 |dead-url=yes }}</ref>

===小学部===
[[File:Clock_Tower_of_SZEXPSPD.jpg|缩略图]]
[[File:Teaching_building_of_SZEXPSPD_(5).jpg|缩略图]]
小学部创办于1985年，位于深圳市福田区红荔路，妇儿医院旁。占地面积20604平方米，校舍建筑面积15000平方米。现有教学班36个，学生1800余名。<ref name="小学部简介">{{cite web |url=http://www.szsy.cn/fbrole/main/content.pr.prContentView.do?Contents/contentID=1205NyVstCzgDgZmLcGKET6XAYx1690 |title=深圳实验学校小学部简介 |work=深圳实验学校 |language=zh-hans |access-date=2017-07-10 |archive-url=https://web.archive.org/web/20160304072645/http://www.szsy.cn/fbrole/main/content.pr.prContentView.do?Contents%2FcontentID=1205NyVstCzgDgZmLcGKET6XAYx1690 |archive-date=2016-03-04 |dead-url=yes }}</ref>

===坂田部(又称新城校区)===
深圳实验学校坂田部的前身是坂田新城学校，其于2017年开始由深圳实验教育集团接管，并更名为深圳实验学校坂田部。深圳实验学校坂田部占地面积4.2万平方米，建筑面积2.8万平方米，工程总投资约1亿元。规划54个教学班，可提供2700个学位，是坂田片区规模最大的九年一贯制公立学校。<ref>{{Cite web|url=http://www.szsy.cn/fbrole/main/content.pr.prContentView.do?Contents/contentID=532298WCGJPLSXZAP8123646890wcgj|title=深圳实验学校新城校区诞生了！|accessdate=2018-2-11|author=深圳实验学校|date=2017-9-8|publisher=|archive-url=https://web.archive.org/web/20180211131640/http://www.szsy.cn/fbrole/main/content.pr.prContentView.do?Contents%2FcontentID=532298WCGJPLSXZAP8123646890wcgj|archive-date=2018-02-11|dead-url=yes}}</ref>
现有14个教学班，其中初一10个班，初二4个班。学校有音乐室、舞蹈室、美术室、语音室、电脑室、电子阅览室、足球场、室内外篮球场等功能场馆，设备设施先进。

=== 运动会开幕式 ===
每隔三年，深圳实验学校都会在[[深圳体育场|深圳体育场]]举行所有学部的田径运动会开幕式，是深圳实验学校“体育节”的一部分。开幕式上有各个部表演节目以及体育特长生的颁奖等环节。此开幕式亦会邀请一些深圳市领导及所有的学生家长来观看。但是各部的运动会仍是在各部进行。<ref name="深圳实验学校运动会开幕式">[http://www.sznews.com/news/content/2012-12/16/content_7504888.htm 深圳实验学校举办运动会 逾万名家长来捧场]</ref>此前在金式如任深圳实验学校校长时期，开幕式为每年举办一次，曹衍清继任校长之后，出于各方面考虑，开幕式改为三年一次，这样的设置仍然能让每一位在实验就读的学生在学习期间至少经历一次开幕式。但由于种种原因，2015年和2016年的开幕式都因故改为分部举行。<ref>{{Cite news|url=http://www.szsy.cn/fbrole/main/content.pr.prContentView.do?Contents/contentID=489986QDEDKCPJVVJ6111316388qded|title=壮哉，实验！
