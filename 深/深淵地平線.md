==簡介==
本遊戲的前身為重慶煜顏文化傳播有限公司，於2016年在[[京東_(網站)|京東]][[群眾募資|眾籌]]上的遊戲企劃案《'''艦隊2次元'''》；該遊戲已於2017年2月成功達標五萬元人民幣的製作費用<ref>{{Cite web |url = https://z.jd.com/project/details/67949.html |title = 舰队2次元 操作萌娘战斗吧 |author = 重慶煜顏文化傳播有限公司 |date = 2018-03-22 |publisher = 京東 |language = zh-cn |accessdate = 2018-04-20 }}</ref>。後於2018年3月上海晨之科預告此一遊戲，跟先前版本相比，除了新增或重新繪製登場艦艇外，遊戲架構和操作介面也獲得極大的調整。

玩家可蒐集來自[[美國|美國]]、[[日本|日本]]、[[蘇聯|蘇聯]]、[[英國|英國]]、[[義大利|義大利]]、[[法國|法國]]、[[納粹德國|納粹德國]]等不同國家，從[[第二次世界大戰|二戰]]到[[20世紀|20世紀]]末的艦艇，組織六艘艦艇編隊，於殲滅戰、追擊戰、玩家對戰等模式中，殲滅[[人工智慧|AI]]敵軍首腦以及由AI或真人對手所控制的敵艦，同時獲得各項戰利品與成就，壯大玩家的艦隊實力。

除了遊戲中的戰鬥外，玩家可藉由[[擴增實境|AR]]視角，在真實世界中投射出玩家艦艇各式各樣的英姿<ref>{{Cite web |url = https://gnn.gamer.com.tw/8/160438.html |title = 3D 艦姬手機新作《深淵地平線》事前登錄開跑 上坂堇、佐倉綾音等人獻聲演出 |author = Edward |date = 2018-03-22 |publisher = 巴哈姆特電玩資訊站 |language = zh-tw |accessdate = 2018-04-07 }}</ref>。

==登場艦艇==
玩家可蒐集並操控的艦艇，統稱為「'''艦姬'''（ふなひめ）」。

另外，中國版的日本艦艇名稱，並未像其他中國製艦娘遊戲一樣，進行[[河蟹_(網路用語)|和諧]]更名處理。

==敵方艦艇==
敵方艦艇統稱為「'''鬼姬'''（おにひめ）」，名字出處源自[[所羅門七十二柱魔神|所羅門七十二柱魔神]]。玩家可藉由在單人任務獲得勝利，取得鬼姬碎片，再利用淨化劑（可從任務取得或遊戲內購買）變回可操控之艦艇。

==參見==

*[[蔚藍戰爭|蔚藍戰爭]]─由中國[[騰訊|騰訊]]所製作與發行（''2016年9月起無限期停止改版''）。

==爭議==
*2018年7月11日，日本遊戲公司[[世嘉|世嘉]]於發出聲明，認為'''上海晨之科所發行的智慧型手機遊戲《深淵地平線》當中的影像表現等內容，侵害了世嘉與[[C2機關|C2機關]]旗下大型電玩遊戲[[艦隊Collection|艦隊Collection大型機台版]]的著作權'''，違反了日本《不正競爭防止法》，因此提出[[假處分|假處分]]訴訟<ref>{{cite web |url= https://sega-interactive.co.jp/news/180711_information_1/|title= 「アビス・ホライズン」に対する配信等差止仮処分命令申立てについて|author=<!--Staff writer(s); no by-line.--> |date= 2018-07-11|website= Sega Interactive|publisher= 世嘉|access-date= 2018-07-11|language=ja}}</ref>。
*另一方面，被告上海晨之科則不甘示弱地表示，'''針對世嘉的上述指控為毫無根據的無稽之談，絕無侵權之虞，並保證絕對不會因此結束營運'''；晨之科也同樣藉由法律手段，為本遊戲主張及辯護合法性<ref>{{cite web |url= https://abyss.morningtec.jp/newsdetail/112|title= アビス・ホライズンに関する仮処分申立てについて|author=<!--Staff writer(s); no by-line.--> |date= 2018-07-11|website= 深淵地平線官網|publisher= 上海晨之科|access-date= 2018-07-11|language=ja}}</ref>。
*2019年1月11日，世嘉和煜顏先後在各自官方網站發布聲明。'''世嘉表示晨之科於2018/12/31將本遊戲移轉日本經營權給煜顏之後，撤銷告訴；煜顏則表示世嘉和C2機關向晨之科和煜顏提出和解聲請'''。本次事件經歷半年纏訟暫時平息<ref>{{cite web |url= https://gnn.gamer.com.tw/1/173941.html|title= 《深淵地平線》與 SEGA 官司暫告一段落 原營運公司宣布撤退並已由開發商接手日版經營|author=尊師 K |date= 2019-01-14|publisher= 巴哈姆特電玩資訊站|access-date= 2019-01-16|language=zh-tw}}</ref>。
