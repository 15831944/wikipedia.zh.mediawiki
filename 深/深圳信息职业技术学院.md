'''深圳信息职业技术学院'''是于2002年4月创办的一所全日制高等院校。其位置在[[中华人民共和国|中国]][[广东省|广东省]][[深圳市|深圳市]]，由[[深圳市政府|深圳市政府]]主持创办<ref>{{cite web |title=学校简介-深圳信息职业技术学院 |url=https://sziit.edu.cn/xxgk/SCHOOL_PROFILE.htm}}</ref>。创办此学校目的之一为为经济发展提供了中等层次的技术人才<ref>{{cite web |title=举全深圳之力办一所大学，它有什么不一样_搜狐教育_搜狐网 |url=http://www.sohu.com/a/235236973_220095}}</ref>。

深圳信息职业技术学院同时也是继[[深圳大学|深圳大学]]、[[深圳职业技术学院|深圳职业技术学院]]后的第三所主校区位于深圳的高等院校。

2018年，深圳信息职业技术学院开始招收本科生<ref>{{cite web |title=深圳信息学院今年首招本科生_华禹教育网【原中国高校网】 |url=http://www.huaue.com/gzxx2018/201862893909.htm}}</ref>。
