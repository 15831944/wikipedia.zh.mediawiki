'''深浦站'''（{{jpn|j=深浦駅|hg=ふかうらえき|rm=Fukaura eki|f=y}}）位于[[青森县|青森县]][[西津轻郡|西津轻郡]][[深浦町|深浦町]]大字深浦字苗代泽，是[[东日本旅客铁道|东日本旅客铁道]]（JR東日本）管辖的[[五能线|五能线]]上的[[鐵路車站|鐵路車站]]<ref name="station"/>。

五能线内重要车站 。普通列车运转系统在本站分界。临时快速"[[Resort白神|Resort白神]]"全部列车停车站。2014年3月14日成为快速列车深浦号的始发站。

== 历史 ==
[[File:Fukaura_stn.jpg|thumb]]
* 1934年12月13日 - 车站投入运营。
* 1963年4月1日 - 早晨车站附近失火导致站舍被波及<ref name = "rj">{{Cite journal |  | author = 壇上完爾 | title = みちのくのローカル線に語りかける津軽の海と山 けっぱれ 五能線 | journal = 鉄道ジャーナル別冊37 懐かしの国鉄現場 | year = 1999 | publisher = 鉄道ジャーナル社 | pages = 130 - 138}}</ref>。
* 1964年3月 - 站房重建完成<ref name = "rj" />。
* 1980年9月20日 - 停止办理货运业务，成为客站<ref name = "dic">《停車場変遷大辞典 国鉄・JR編》1998年 JTB</ref>。
* 1984年2月1日 - 停止办理行李托运业务<ref name = "dic" />。
* 1987年4月1日 - [[国铁分割民营化|国铁分割民营化]]后成为JR东日本车站。
* 2010年4月1日 - [[大間越站|大間越站]] - [[風合濑站|風合濑站]]间车站移交五所川原站管理。

== 车站结构 ==
[[岛式站台|岛式站台]]1台2线的[[地上车站|地上车站]]<ref name="station"/>。

直营站，清早与夜间无人值守。2010年3月31日[[大间越站|大间越站]] - [[风合濑站|风合濑站]]间车站转属[[五所川原站|五所川原站]]管理，本站不再管理其他车站。

配备有[[绿色窗口|绿色窗口]]（营业时间：7时30分 - 18时20分）和[[自动售票机|自动售票机]]。站内残存转车台自2005年被沙尘覆盖，无法使用。
