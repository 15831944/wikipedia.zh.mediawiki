'''深圳证券交易所100收益指数'''（简称'''深证100R指数'''、'''深证100R'''，英文'''SZSE 100 Return Index'''），指数代码为{{SZSE|399004}}，是[[深圳证券交易所|深圳证券交易所]]主要的成分指数之一，由100只规模大、市场份额高、交易活跃的股票组成，指数反映了深圳市场核心优质上市公司的股价变动走势<ref name="编制方案">{{Cite web |url=http://www.szse.cn/main/files/2015/01/30/%E6%B7%B1%E8%AF%81100%E6%8C%87%E6%95%B0%E7%BC%96%E5%88%B6%E6%96%B9%E6%A1%88%EF%BC%882014%E5%B9%B41%E6%9C%88%EF%BC%89.pdf |title=深证100指数编制方案（2014年1月） |accessdate=2018-02-12 |author=深圳指数 |date=2015-01-30 |publisher=深圳交易所}}</ref>。

深圳100R指数于2003年1月2日首次发布<ref>{{Cite web |url=http://www.cnindex.com.cn/zstx/szxl/ |title=深证系列规模指数 |accessdate=2018-02-12 |publisher=国证指数}}</ref>，主要包含了深市主板中的[[蓝筹股|蓝筹]]价值型企业和一部分来自中小板和创业板的成长型优质企业<ref>{{Cite web |url=http://www.cnindex.com.cn/docs/jj_399330.pdf |title=指数简介 |accessdate=2018-02-12 |publisher=国证指数}}</ref>。指数发布时简称为“深证100指数”，2006年4月24日随着以价格法计算的[[深证100指数|深证100指数]]的发布，为便于识别、防止混淆，简称变更为“深证100R指数”<ref>{{Cite web |url=http://finance.sina.com.cn/stock/y/20060413/14452498942.shtml |title=深证100更名深证100R |accessdate=2018-02-13 |author=晶报 |date=2006-04-13 |publisher=新浪财经}}</ref>。
== 样本股标准 ==
从在深圳交易所上市的股票中，具备下列条件的股票，按个股平均总市值占市场比重、平均自由流通市值占市场比重和平均成交金额占市场比重来评估选取<ref name="编制方案"/>：
*非ST、*ST的A股；
*有一定上市交易日期（一般为六个月，总市值和自由流通市值综合排名位于深圳市场前 10 名的股票不受此限制）；
*公司最近一年无重大违规、财务报告无重大问题；
*公司最近一年经营无异常、无重大亏损；
*考察期内股价无异常波动。
