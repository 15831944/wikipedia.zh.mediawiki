《'''深海利剑'''》（{{lang-en|'''''Deepwater Forces'''''}}），2017年[[中国|中国]][[军旅片|军旅片]]，2016年11月至2017年4月拍攝。由[[高旻睿|高旻睿]]、[[刘璐_(演员)|刘璐]]、[[王阳_(演员)|王阳]]、王強领衔主演<ref>{{Cite web|url=http://ent.sina.com.cn/v/m/2017-03-28/doc-ifycstww1555313.shtml|title=王阳再合作赵宝刚导演 《深海利剑》演绎严酷教官|accessdate=2017-09-27|author=|date=2017-03-28|work=|publisher=新浪}}</ref><ref>{{Cite web|url=http://www.chinanews.com/yl/2017/04-05/8191538.shtml|title=《深海利剑》杀青 “宝女郎”王佳宇引关注|accessdate=2017-09-27|author=|date=2017-04-05|last=|work=|publisher=中新网}}</ref>，2017年7月27日在[[北京卫视|北京卫视]]、[[浙江卫视|浙江卫视]]首播<ref>{{Cite web|url=http://ent.sina.com.cn/v/m/2017-07-03/doc-ifyhrxsk1586777.shtml|title=赵宝刚《深海利剑》定档7.27 利剑出鞘气势如虹|accessdate=2017-09-27|author=|date=2017-07-03|work=|publisher=新浪}}</ref>。

== 劇情 ==
以海军372潜艇官兵真實先进事迹为素材创作，面对X国高科技不明潜艇，代號“黑鲨”的屡次侵擾，[[中國人民解放軍海军|中國人民解放軍海军]]火速制定“T计划”，展开全国特招，为正在研发的某新型潜艇培育指挥型军官。

一群玩世不恭個人至上的[[八零後|90后]]大学生经过层层筛选，进入神秘的潜艇世界。他们在超常严苛训练里，从懵懂新兵成长为肩负国防重任的铁血军人，驾新型潜艇深海潜航，一次次展開與黑鲨的較量，用生命鑄就了海上長城的重剑精神，捍卫“听党指挥，生死与共，永不沉没”的不朽誓言。

== 原型 ==
本剧的主要人物原型是：
* [[中国人民解放军海军第三十二潜艇支队|中国人民解放军海军第三十二潜艇支队]]（92474部队）
** 原支队长[[王红理|王红理]]
** [[远征72|远征72]]号（舷号：372）[[基洛级潜艇|基洛级潜艇]]全体官兵

[[中国人民解放军南部战区海军|南部战区海军]]某潜艇支队支队长[[王红理|王红理]]率372艇在执行任务途中，突遇重大险情，经过排险，圆满完成后续任务。2014年8月27日，[[中央军委主席|中央军委主席]][[习近平|习近平]]签署通令，为92474部队部队长王红理记一等功<ref>{{Cite web |url =http://cpc.people.com.cn/n/2014/0828/c64094-25556429.html|title=习近平签署通令 给1个单位、24名个人记功|author = |date = 2014年8月28日07:10|publisher = 人民网－人民日报 |language =  |accessdate =  |quote =  }}</ref>。2014年9月2日，中央军委给王红理、海军给372潜艇记一等功庆功大会在南海舰队某潜艇支队礼堂举行，中央军委委员、海军司令员[[吴胜利|吴胜利]]宣读了中央军委和海军的通令，分别给王红理和372潜艇颁发了奖章证书和奖状<ref>{{Cite web |url= http://www.mod.gov.cn/auth/2014-09/03/content_4534864.htm|title =中央军委给王红理、海军给372潜艇记一等功庆功大会举行 |author = 徐叶青、特约记者高毅|date = 2014-09-03 07:32:27|publisher = 中华人民共和国国防部 |language =  |accessdate =  |quote =  }}</ref>。
