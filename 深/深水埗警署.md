'''深水埗警署'''（官方名稱：'''深水埗分區警署'''）是[[香港警務處|香港警務處]][[西九龍總區|西九龍總區]][[深水埗|深水埗]][[警區|警區]]的[[分區|分區]][[警署|警署]]，位於[[香港|香港]][[九龍|九龍]][[深水埗區|深水埗區]][[深水埗|深水埗]][[欽州街|欽州街]]37號A，屬於[[香港二級歷史建築|香港二級歷史建築]]。

== 歷史 ==
為了處理大量[[中國大陸人|中國大陸人]]湧入香港後，在[[深水埗區|深水埗區]]所引起的[[香港治安|治安]]問題，深水埗警署於1924年動工，於1925年竣工，當時鄰近海旁及[[深水埗軍營|深水埗軍營]]；深水埗警署僅比九龍現存最古舊的警署──[[油麻地警署|油麻地警署]]，遲兩年落成。

[[香港保衛戰|香港保衛戰]]時，深水埗警署被[[駐港英軍|駐港英軍]]充作[[軍營|軍營]]用途，至[[香港日治時期|香港日治時期]]時，被[[日本皇軍|日本皇軍]]改為深水埗集中營的指揮部。

1978年，[[長沙灣警署|長沙灣警署]]落成前，深水埗警署為深水埗分區總部<ref>[http://www.lcsd.gov.hk/specials/deltaphoto2013/b5/hk7.php 深水埗警署]</ref>，管轄範圍於[[香港保衛戰|香港保衛戰]]前遠及[[荃灣|荃灣]]一帶。

== 建築 ==
深水埗警署由[[公和洋行|公和洋行]]設計，是[[英國殖民地|英國殖民地]]典型的[[新古典主義|新古典主義]]建築，分為A至E座，其中以面對[[欽州街|欽州街]]及[[荔枝角道|荔枝角道]]的C座歷史最悠久。C座樓高3層，建有傳統圓柱門廊，並且設有裝飾欄杆。傾斜屋頂鋪上中式瓦片，數支煙囪在屋頂突出。面對內庭一面設計則比較現代化，展露[[包豪斯|包浩斯]]風格。深水埗警署樓高3層，為了應付需要，大樓旁於後來分別增建了多座現代化[[建築物|建築物]]。

== 軼聞 ==
深水埗警署[[羈留室|羈留室]]內其中一間羈留倉多年來儘管於疑犯人滿為患時，均不被使用。傳聞早年每逢有疑犯被羈押在該倉內，於睡覺時均被[[鬼魂|鬼魂]]騷擾，例必於翌日要求調換倉<ref>[http://hk.apple.nextmedia.com/news/art/20120113/15979191 一夜槍殺兩警　同袍拒登鬼船] 《蘋果日報》 2012年1月13日</ref>。

2008年[[盂蘭節|盂蘭節]]後次日[[凌晨|凌晨]]，[[香港傳媒|香港傳媒]]報道有人員從[[閉路電視|閉路電視]]中觀察到拘留所內出現著白衫、白褲及披長髮的[[女鬼|女鬼]]飄移，巡遊倉間。當時有約7名[[疑犯|疑犯]]被拘留，當人員巡察時，當中一名疑犯突然大叫，人員因此查問，疑犯回應表示目睹鬼魂飄遊，另外一名疑犯異口同聲認同。[[警察公共關係科|警察公共關係科]]被傳媒問及，表示將會作出跟進。
