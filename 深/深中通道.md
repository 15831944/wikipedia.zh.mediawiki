'''深中通道'''属于[[中国国家高速公路网|中国国家高速公路网]]中[[深岑高速公路|深岑高速公路]]的起点段（国家高速编号：'''G25<sub>18</sub>''' <ref>"[http://www.gdcd.gov.cn/zxgh/69640_1.jhtml 广东省高速编号命名以及调整]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}". 广东省交通运输厅. 于2017年3月20日查阅.</ref>），是建设中的橫跨[[珠江口|珠江口]]、連接[[中國|中國]][[廣東省|廣東省]][[深圳市|深圳市]]與[[中山市|中山市]]的公路通道，由海底（[[珠江|珠江]]底）隧道及跨海（跨珠江）公路橋組成。

==歷史==
1983年，香港[[合和集團|合和集團]]主席[[胡應湘|胡應湘]]提出興建[[伶仃洋大橋|伶仃洋大橋]]，連接香港和珠海。1998年香港的[[青嶼幹線|青嶼幹線]]通車，[[胡應湘|胡應湘]]又提出香港應善用[[青馬大橋|青馬大橋]]等基建設施；於是大橋香港起點改為[[大嶼山|大嶼山]]，原伶仃洋大橋計劃擱置，並再加入[[澳門|澳門]]（單Y方案）和[[深圳市|深圳市]]（雙Y方案）<ref>{{cite web|url=http://paper.wenweipo.com/2003/01/23/NS0301230005.htm|title=粵專家提交「雙Y」大橋方案 - 香港文匯報|author=|date=|website=paper.wenweipo.com}}</ref><ref>{{cite web |url=http://www.waou.com.mo/wa/2011/02/20110214a.htm |title=存档副本 |accessdate=2012-03-25 |deadurl=yes |archiveurl=https://web.archive.org/web/20140814150636/http://www.waou.com.mo/wa/2011/02/20110214a.htm |archivedate=2014-08-14 }}</ref>，其中後者（雙Y方案）在東、西兩端分別以香港、深圳，和珠海、澳門為落腳點，在中間某處匯合共線。惟最後雙Y方案在2005年被否決。<ref>{{cite web|url=http://www.ghdi.com.cn/newsview.asp?id=107|title=交通部部长：港珠澳大桥单Y确定 落脚点未定 新闻阅读|author=|date=2005-07-12|website=羊城晚报 www.ghdi.com.cn}}</ref>

1998年，中山市规划局就提出了建设深中通道的设想。<ref>{{cite web|url=https://xueqiu.com/2829032936/36496465|title=财经洒洒水: 深中通道：深圳市长为什么急了？ “有些问题长期论证下去没有必要，希望省里与我们一起努力推动深中通道尽快开工建设！”昨日下午，在参加广东省十二届人大三次会... - 雪球|author=|date=|website=xueqiu.com}}</ref><ref name="auto">{{cite web|url=http://www.zsnews.cn/News/2014/08/08/2664010.shtml|title=深中通道建设万事俱备“欠东风”_本地新闻_新闻频道_中山网|first=|last=zsnews|date=|website=www.zsnews.cn}}</ref>

2004年，“深中通道”就被提上议程，当年原深圳市交通局与深高速联合委托中国公路工程咨询监理总公司和铁道第四勘察设计院开展“深中通道”研究工作，初步提出“采用公铁合建的隧道过江方案”。<ref name="auto"/>

2006年中旬，[[深圳市政府|深圳市政府]]提出興建'''深中大橋'''<ref name="ref1">{{cite web|url=http://paper.wenweipo.com/2008/03/29/CH0803290016.htm|title=深中通道作可行性研究 - 香港文匯報|author=|date=|website=paper.wenweipo.com}}</ref>，以連接深圳市和中山市，而大橋位於[[港珠澳大橋|港珠澳大橋]]北面。

=== 定案 ===
2012年11月，廣東省交通廳宣布推薦方案，採用“東隧西橋”<ref name="ref3">{{cite web|url=http://news.ifeng.com/gundong/detail_2012_11/07/18904354_0.shtml|title=深中通道地位等同深圳机场_资讯频道_凤凰网|author=|date=|website=news.ifeng.com}}</ref>方式興建，並同時開始進行[[環評|環評]]公示。

== 方案 ==
深中通道推薦方案全長51公里，按照通道的設計速度為100公里/小時。起點位於深圳市寶安區[[西乡街道|西鄉]]鹤洲立交，與[[機荷高速公路|機荷高速公路]]對接，以隧道和大橋穿過[[珠江口|珠江口]]中部；先在中山市最東面登陸，然後沿着[[橫門水道|橫門水道]]一直西進，最後在中山[[港口镇_(中山市)|港口鎮]]新隆立交接上[[中江高速公路|中江高速公路]]{{Update inline}}；此外並建有[[南沙区|南沙]]連接線。工程將在深圳近海興建兩個人工島以貫通長6.72公里的隧道，另外還在西段興建一座通航用特大橋樑。

深中通道主体工程已经于2016年底正式开工，预计2024年建成通车。<ref>{{Cite web |title=深中通道西人工岛首个钢圆筒开始拖运安装|date=2017-04-28|url=http://sz.southcn.com/content/2017-04/29/content_169844256.htm}}</ref>

工程建成後，珠三角东西两岸将会打通，拉近了西岸南沙、中山、珠海、江门、茂名、湛江乃至海南省与东岸深圳、东莞、惠州、汕尾等地的时空距离。由中山至深圳駕車用時將由現在的超過90分鐘大大縮短至30分鐘內，亦可紓緩[[虎門大橋|虎門大橋]]的擠塞情況。

== 爭議 ==

=== 規劃 ===
珠江口沿岸各城市在港口航运中心方面競爭非常激烈，在确定跨江通道登陆地点时，沿线各市的意見不一，[[广州市|广州市]]支持與[[虎门大桥|虎門大橋]]走向相似的“东莞[[虎门镇|虎门]]-广州[[南沙|南沙]]”路线，[[深圳市|深圳]]和[[中山市|中山]]则要求在各自辖区选择线路节点，最後以“深圳机场南-中山新隆”方案中選。而在建設方案的選擇上，工程報告提出全桥、东桥西隧、东隧西桥和全隧等多种方案，全桥及东桥西隧方案因不能满足[[深圳宝安国际机场|深圳机场]]航空限高要求遭到否決，东隧西桥和全隧兩種方案進入最後比選，2013年，廣東省政府常务会议同意深圳和中山两市聯合支持的“东隧西桥”方案为推荐方案，“长隧道”方案作为备选方案<ref>{{Cite web|url=http://news.hexun.com/2014-07-18/166775314.html|title=深中通道线路明确 推荐采用“东隧西桥”|accessdate=2018-12-18|author=|date=|work=news.hexun.com|publisher=}}</ref>。

=== 阻隔航運 ===
由於深中通道经过的伶仃航道是[[南沙港|南沙港]]和[[广州港|广州港]]的主要出海航道，广州方面担心，深中通道的跨海大桥限制了進出船隻的高度，影響廣州建设国际航运中心的地位。廣州市隨即向省政府請示，建议采用全隧方案，并称此方案可实现广深中三市共赢<ref>{{Cite web|url=http://weekly.caixin.com/2018-12-08/101357082.html|title=广深中博弈：一座大桥引发的争议|accessdate=2018-12-18|work=weekly.caixin.com}}</ref>。广州市人大常委會主任[[陈建华|陈建华]]在广东省十三届人大一次会议召开期间提交議案，希望采用“全隧方案”替代“东隧西桥”方案，替代桥梁的隧道段可由广州市代建，广州市愿意承担方案修改后引发的新增建设投资<ref>{{Cite web|url=http://www.sohu.com/a/281266868_100239955|title=一座大桥引发的争议，广州愿意给深中通道出资金全走隧道_陈建华|accessdate=2018-12-18|work=www.sohu.com}}</ref>。廣州市人大常委會更罕見將與大橋相關爭議的報道轉載至官網頭條<ref>{{Cite web|url=http://www.rd.gz.cn/xwdt/content/post_82169.html|title=财新周刊：一座大桥引发的争议_广州市人大常委会|accessdate=2018-12-18|work=www.rd.gz.cn}}</ref>。但最後不獲批准。

廣州社科院城市戰略研究院副院長張強稱，廣州國際航運中心會受大橋影響，亦會削弱廣州的經濟實力及發展，甚至影響在[[大灣區|大灣區]]的地位，城市話語權也會相應下降<ref>[http://news.tvb.com/greaterchina/5c26ce68e6038365490a0c48/ 深中通道動工船隻難進出南沙港 學者料窒礙廣州發展]TVBnews 2018-12-29</ref>。大橋一旦落成，會封鎖[[广州南沙新区|南沙港]]和[[广州港|廣州港]]的大型貨櫃船進出，廣州官員認為，工程不但影響船運，更會搶走空運和陸路運輸的生意，威脅廣州作為廣東省中心地位<ref>[https://hk.on.cc/hk/bkn/cnt/cnnews/20181217/bkn-20181217214413846-1217_00952_001.html 伶仃洋大橋限船舶高度　官員憂影響廣州航運業]

東方即時 2018-12-17</ref>。
