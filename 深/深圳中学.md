'''深圳中学'''（{{lang-en|Shenzhen Middle School}}）是[[中國|中國]][[广东省|广东省]][[深圳市|深圳市]]的一所公办[[完全中学|完全中学]]。

深圳中学创办于1947年，1983年6月被定为深圳市唯一的省重点中学，1993年11月被评为广东省首批一级学校。1998年原洪湖中学并入深圳中学，校园成为其初中部。深圳中学的高中部分为东校区和西校区<ref name="学校概况" />。深圳中学高中部、初中部与其他一些附属学校，包括深圳亚迪学校、深中龙岗初级中学、深中龙岗小学、深中第一附属幼儿园、深中第二附属幼儿园共同组成“深圳中学教育共同体”<ref>{{cite news |url=http://www.shenzhong.net/news/20141130/n52879041.html |title=创造未来——深圳中学教育共同体第四次会议召开 |date=2014-11-30 |accessdate=2014-12-06 |publisher=深圳中学 |quote=参加会议的有深圳中学高中部、初中部、深圳亚迪学校、深中龙岗初级中学、深中龙岗小学、深中附属第一幼儿园、第二幼儿园各校领导 |deadurl=yes |archiveurl=https://web.archive.org/web/20141211035919/http://www.shenzhong.net/news/20141130/n52879041.html |archivedate=2014年12月11日 |df= }}</ref>。深圳中学在2016年中国内地高中美国留学排行榜上位居第八。<ref>人民网 "http://gz.people.com.cn/n2/2016/0224/c358162-27802234.html" 2016-02-24 </ref> <ref> 联校网排名 "{{cite web |url=http://www.collegenode.com/list.php |title=存档副本 |accessdate=2016-04-05 |0=2016-10-14 |deadurl=yes |archiveurl=https://web.archive.org/web/20161014013045/http://www.collegenode.com/list.php |archivedate=2016-10-14 }}" </ref>

== 历史 ==
1947年，广东省[[宝安县|宝安县]]的水贝、向西、湖贝三村张氏族人在雍睦堂集资筹办一所初级中学，每村推举四人组成筹备董事会，向宝安县政府呈交申请书。<ref>{{cite web |url=http://www.shenzhong.net/news/2009410/n985619.html |title=校史-1947年 |work=深圳中学 |deadurl=yes |archiveurl=https://web.archive.org/web/20130926092259/http://www.shenzhong.net/news/2009410/n985619.html |archivedate=2013年9月26日 |df= }} </ref>1948年7月广东省教育厅批准办校的批文下达，准予注册，于是在1948年9月深圳初级中学正式开学，校址在旧镇公所附近的养生街四巷10号。<ref>{{cite web |url=http://www.shenzhong.net/news/2009410/n985619_1.html |title=校史-1948年 |work=深圳中学 |deadurl=yes |archiveurl=https://web.archive.org/web/20130926092148/http://www.shenzhong.net/news/2009410/n985619_1.html |archivedate=2013年9月26日 |df= }}</ref>1949年11月，[[中華人民共和國成立|中華人民共和國成立]]后，宝安县人民政府接管了深圳初级中学，并将其改名为宝安县第二初级中学。<ref>{{cite web |url=http://www.shenzhong.net/news/2009410/n985619_11.html |title=校史-1949年 |work=深圳中学 |deadurl=yes |archiveurl=https://web.archive.org/web/20130926092414/http://www.shenzhong.net/news/2009410/n985619_11.html |archivedate=2013年9月26日 |df= }} </ref>当时学校的校舍为两层楼，楼下有一个小礼堂，中间还有一个天井。<ref>{{cite web|url=http://www.shenzhong.net/news/2009519/n4451230.html|title=校史-1955年|work=深圳中学|deadurl=yes|archiveurl=https://web.archive.org/web/20130926092411/http://www.shenzhong.net/news/2009519/n4451230.html|archivedate=2013年9月26日|df=}} </ref>1983年6月在经过广东省人民政府的批准后，深圳中学成为广东省重点中学。<ref>{{cite web |url=http://www.shenzhong.net/news/200972/n00302073.html |title=校史-1983年 |work=深圳中学 |deadurl=yes 2002年，[[王铮_(校长)|王铮]]由[[北大附中|北大附中]]的副校长兼深圳南山分校校长转任为深圳中学校长。同年，深圳中学开始了对新课程方案改革的探索。<ref>{{cite news |url=http://www.21cbh.com/HTML/2009-12-8/156945.html |title=教育评价标准的改变：这是一件非常难的事情 |publisher=[[21世纪经济报道|21世纪经济报道]] |author=马娟 |date=2009-12-08 |accessdate=2014-01-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20100112075408/http://www.21cbh.com/HTML/2009-12-8/156945.html |archivedate=2010-01-12 }}</ref>2003年9月开始，深圳中学尝试将教学班与行政班脱钩，取消了班主任和班级制并代之以导师制。<ref>{{cite news|url=http://www.chinanews.com/cul/2011/11-16/3464957.shtml |title=深圳中学学生创办《涅槃周刊》 被称“胆子大” |publisher=[[中国新闻网|中国新闻网]] |author=刘炎迅 |date=2011-11-16 |accessdate=2014-01-18}}</ref>另外，学生的权利也得到大大加强。学生会被定义为学生自治机构，鼎盛时期全体学生的三分之一是其会员。<ref name="晶报">{{cite news |url=http://jb.sznews.com/html/2012-07/20/content_2130227.htm |title=深圳10年课改， 终向高考举白旗？ |publisher=[[晶报|晶报]] |author=吴建升 |date=2012-07-20 |accessdate=2014-01-18 |deadurl=yes |archiveurl=https://web.archive.org/web/20150604023909/http://jb.sznews.com/html/2012-07/20/content_2130227.htm |archivedate=2015年6月4日 |df= }}</ref>

王铮所进行的改革产生了很大的争论，一些教师批评其改革纵容了学生，一些家长认为改革影响了学生高考成绩<ref>{{cite news|url=http://zqb.cyol.com/content/2008-04/02/content_2128196.htm |title=谁要是动了高考这根弦，就是找死？ |publisher=[[中国青年报|中国青年报]] |author=亓树新 |date=2008-04-02 |accessdate=2014-01-18}}</ref>。而一些已毕业的学生则认为，改革之后的教育模式使他们明白了作为一个公民的权利和义务，是公民意识启蒙。当时的教育业界和社会也将王铮的改革视为公民教育实验。<ref name="晶报" /><ref name="财经" />

2010年4月，王铮返回北大附中出任校长。同年，[[王占宝|王占宝]]接任深圳中学校长，到任后他也进行了一系列改革，主张“建设学术性高中，培养创新型人才”。学校恢复了班主任和班级制，走课的课程也相应减少。起初，他所进行的改革被部分学生批评为“应试教育复辟”，但深圳中学高考[[重点本科|重点本科]]上线率有所提升。<ref name="财经">{{cite news|url=http://magazine.caijing.com.cn/2011-10-23/110913108.html |title=深圳中学教改争议 |publisher=财经 |author=舒泰峰、莫莉 |date=2011-10-23 |accessdate=2014-01-18}}</ref>

2016年2月19日，王占宝因身体原因辞去校长职务提前退休，[[深圳第二实验学校|深圳市第二实验学校]]校长赵立接任校长职务。<ref>{{Cite web|url=http://szeb.sz.gov.cn/xxgk/flzy/tzgg2/201602/t20160229_3489959.htm|title=深圳市教育局关于赵立等同志职务任免的通知|accessdate=2016-03-09|author=深圳市教育局|date=2016-02-26|publisher=|archive-url=https://web.archive.org/web/20160309040833/http://szeb.sz.gov.cn/xxgk/flzy/tzgg2/201602/t20160229_3489959.htm|archive-date=2016-03-09|dead-url=yes}}</ref>

2017年1月17日，赵立校长于2016年11月提任深圳市教育局副局长，广州市教育研究院院长朱华伟接任校长职务。<ref>{{Cite web|url=http://www.szeb.edu.cn/jyxw/jyxw/201701/t20170117_5939630.htm|title=深圳中学迎来新校长|accessdate=2017-01-28|author=深圳市教育局|date=2017-01-17|publisher=}}</ref>

2017年10月21日，西校区食堂二楼因充值处电路老化而发生火警。校园学生媒体《涅槃新闻》发表火情新闻。事后，部分学生声称，有教师向《涅槃新闻》人员的家长施压，要求撤稿。事件引起在校学生及校友的关注、讨论。<ref>{{Cite web|url=http://sms-alumni.com/1021ourhighschool|title=1021深中校内到底发生了什么|accessdate=2017-10-22|publisher=晒布茶室}}</ref>

==== 东校区 ====
*'''成美楼'''（艺术楼）位于A旧办公楼原址，旧楼已于2012年完成拆除。预计2014年内完工。
*'''旧教学楼'''（B楼） 校舍迁至晒布岭后，经过数轮的补修工作一直维持到现在。目前，1楼设有媒体中心、学生事务中心、教师事务中心及部分国际体系选科教室，2楼为艺术科授课教室，3-4楼为国际体系，5楼为行政人员办公室。按照四期工程计划，B栋将给予拆除。
*'''科学馆'''（C楼） 建设面积达15000平方米，有40多间功能室，另外还作为实验体系教学楼使用。<ref>{{cite web|url=http://guangdong.eol.cn/guangdongnews_5112/20110627/t20110627_641000.shtml |title=深圳中学 简介|date=2011-06-27 |accessdate=2013-08-31 |publisher=中国教育在线}}</ref>
*'''新教学楼'''（D楼）目前为标准体系、荣誉体系和整体课程体系教学楼。
*'''力行楼'''（原文体楼）2014年，原“文体楼”更名为“力行楼”。
*'''学生宿舍''' 目前一共有四栋，1、2栋相连，3栋独立，4栋为原教工宿舍12栋。<ref>深中观览车 深中新闻 正文 我校学生宿舍改造完毕交付使用 深圳中学　2012-10-15　 </ref>深圳中学学生宿舍实行男女分开住宿、管理。
*'''新体育馆''' 已于2012年开工，位置为B楼和C楼的西侧原露天篮球场处。拟定一楼为羽毛球馆，二楼为篮球馆，并开设新的天桥通往足球场。预计2014年内完工。

==== 西校区 ====
西校區原為深圳市財經學校校址，于2002年劃歸深圳中學管轄。自2006年起，深圳中學人民北路書院街校區（簡稱西校區）成為普通高三年級學生及整体体系初一和初二年级的学生學習、生活的場所。

共有教學樓三幢，其中1號樓為理科生專用，2號樓為文科生專用，3號樓為教師辦公室，4號樓属于办公楼兼做教学使用，二层及三层属于深圳市教育局招生考试办公室，四至六层為整體教育實驗體系教室。

另外，4號樓的七層及以上，為隸屬深圳市教育局的深圳招生考試辦公室的辦公場所。另有作為住校生宿舍的5、6號樓。

=== 初中部 ===
深圳中学初中部校舍位于深圳市罗湖区贝丽南路46号，占地26602平方米，原为于1985年成立的洪湖中学，于1998年并入深圳中学，成为其初中部。

初中部亦为深圳中学的第二大生源中学。

==办学成绩==
===学科竞赛===
深圳中学自20世纪90年代开始鼓励和支持学生参与各项学科竞赛，范围涵盖数学、物理、化学、生物、计算机、天文等诸多学科。[[王铮|王铮]]校长上任后大力支持学生参与学科竞赛，配备专职竞赛教练进行教学。此后深圳中学平均每年均有学生代表中国参加国际学科竞赛并获奖项。[[王占宝|王占宝]]校长上任后设立竞赛课程体系（后改称奥赛课程体系，现为荣誉课程体系），将奥林匹克竞赛培训课程纳入校本选修课内进行规范管理，并对参与学生赋予相应荣誉学分和学分积点，出台《深圳中学高端学术活动管理办法》。

深圳中学每年均有数十位学生因为在学科竞赛中的表现，获得[[北京大学|北京大学]]、[[清华大学|清华大学]]、[[中国科学技术大学|中国科学技术大学]]等顶尖大学的[[保送|保送]]录取和降分优惠。其中部分学生前往美国[[哈佛大学|哈佛大学]]、[[普林斯顿大学|普林斯顿大学]]、[[斯坦福大学|斯坦福大学]]、[[麻省理工学院|麻省理工学院]]等知名高校深造学习。

== 文化 ==
深圳中学自王铮校长改革后，以自由、民主、开放的文化和公民精神著称。<ref name="财经" />

=== 单元 ===
王铮校长任职期间，深圳中学高中实行新的管理模式，突破班级的界限，设立了单元。当时体系已有雏形，学校将其分为四个体系下管理的单元，分别为常规单元、自主单元、竞赛单元、出国单元（四大体系的前身）。后来改为数字，如一单元、二单元等。

每个单元内有两至三个班级，单元内设立有单元主席、单元副主席、单元学生代表委员（现已变为独立的单元议事会委员）、单元共青团员代表委员（现已取消）、单元干事，组成本单元的内阁。每个单元都有自己的品质、意象、价值观和单元色。

2010年第一次“校长面对面”时，学校恢复班级制，已经不承认单元作为学校行政单位存在，但单元制仍可作为传统，继续流传在学生群体当中。单元已经成为不属于学校任何部门管理的学生自发传统。但学校方面没有排斥单元的存在，许多活动（如体育赛事）仍旧以单元为单位展开。单元内阁代表单元内全体学生在某些方面进行决策，同时也要对单元内全体学生负责。 

每一年新生入学，将由上一届内阁选举出下一届，选举方式由内阁决定。单元主席拥有任命内阁其他成员的权利。高一的新内阁将全面负责组织单元活动以及管理单元事务。

深圳中学设立了多个学生自主管理的组织和七十余个社团：
* 共青团深圳中学委员会
* 深圳中学学生议事会：深圳中学学生自治决策的非常设机关。议事会所作任何决议享有全体学生共同的建议权、监督权及决策权，任何会议参与人员享有言论自由权，并受到深圳中学全体学生的保护。议事会进行代表学生决策，实现学生自治管理，维护学生合法权益。学生议事会由各班级代表学生的议事委员（简称议委）参会，其日常事务由议事理事会管理。议事会为学生代表大会改革产物。
* 深圳中学学生会：一个自创立便扎根于学生深处，致力为学生服务的学生组织。其职能包括沟通学生与校方，担当连接学生与校方间的桥梁与纽带；维护学生权益，解决学生校园生活中实际遇到的困难与问题；弘扬深中精神，弘扬自由、民主与理性的思想，引领大事件中深中民主的浪潮。曾于09年改革。
* 深圳中学社团联盟理事会：社团联盟理事会作为与学生会平行的独立的学生组织，主要负责新社团的注册成立、旧社团的破产解散、社团日常事务的管理以及文体活动的组织。2013年9月末，社联理事会开始改革，初拟定以助社团独立为宗旨。2014年6月底，对外公布章程；7月召开第一次社团成员代表大会，但被议事会否认其合法性。
* 深圳中学学生活动中心：一个学生自主程度较高的组织团队，全面负责学校大型活动（如十大歌手比赛、游园会等）的策划、宣传、组织以及举办。

== 主要活動 ==

=== 遊園會 ===

[[File:2010_Shenzhen_middle_school_garden_fete.jpg|thumb]]
每年1月1日，深中会举办“深中游园会”，其中有来自各个学校学生摆的各种摊位。学校中央亦搭设舞台，有来自诸多学校的各种表演节目。深圳中學游园会每年都吸引市民前往，其中包括许多是來自深圳其他學校的學生。
