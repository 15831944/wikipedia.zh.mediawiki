'''深圳通'''是通用於[[中華人民共和國|中國]][[廣東省|廣東省]][[深圳市|深圳市]]的[[電子貨幣|非接觸式智能卡系統]]，由[[深圳市|深圳市]][[深圳通有限公司|深圳通有限公司]]發行。最初為僅適用於公共巴士的智能IC卡，自2004年12月28日[[深圳地鐵|深圳地鐵]]開通以來，其應用範圍已逐步拓展至各種公共交通工具及超市、便利店、快餐店等小額支付領域。

於2012年9月，深圳通有限公司與[[八達通|八達通]]卡有限公司宣布推出二合一聯名卡-「'''互通行'''」（漢語拼音：Hu Tong Xing），首批限量版於2012年9月4日發行，香港普通版將於9月11日於香港和深圳兩地同時發售。<ref>[http://www.octopus.com.hk/release/detail/2012/tc/20120903.html 八達通與深圳通宣布推出「互通行」二合一聯名卡 八達通卡有限公司 新聞稿] {{webarchive|url=https://web.archive.org/web/20140106033105/http://www.octopus.com.hk/release/detail/2012/tc/20120903.html |date=2014-01-06 }}(2012年9月3日)</ref>。至於深圳普通版「互通行」卡，原先預計於2012年年底推出，但事實上深圳普通版「互通行」卡是直至2014年1月才開始發售。諷刺的是深圳地區初期是發售香港普通版「互通行」卡，但其後在香港有少部份[[港鐵|港鐵]]車站卻有深圳普通版「互通行」卡發售，至2016年2月，為配合[[前海深港現代服務業合作區|前海深港現代服務業合作區]]之發展，「互通行」卡推出了「互通行·前海」，於[[深圳地鐵1號線|深圳地鐵1號線]][[羅湖站_(深圳)|羅湖站]]、[[世界之窗站|世界之窗站]]及[[大新站_(深圳)|大新站]]之「深圳通便民服務點」發售<ref>[http://www.shenzhentong.com/products/prodetail_101012001_100000126666753.html 新品推荐-深港“互通行”（前海）卡 深圳通有限公司]</ref>。

於2012年12月28日，深圳通與[[嶺南通|嶺南通]]正式達成合作協議，將於2013年年中試推出「嶺南通·深圳通」，並計劃於2013年年底讓所有深圳通自動升級為嶺南通。<ref>[http://www.shenzhentong.com/news/index.do?i=291&m=0301&a=detail 岭南通与深圳通正式联手 深圳通有限公司] {{webarchive|url=https://web.archive.org/web/20130120200751/http://shenzhentong.com/news/index.do?i=291&m=0301&a=detail |date=2013-01-20 }}(2012年12月28日)</ref>但是直到2014年尚未推出销售。由于早期（2007年前）發行的“深圳通”及「互通行」卡，均與“岭南通”使用不同-{芯}-片，故不能直接升级为岭南通，只有後期采用英飛凌-{芯}-片標準的Type A卡版本的深圳通才有望跟岭南通兼容，不過嶺南通用於深圳通設備預計於2014年年底測試完成。預計於2015年年中當嶺南通公司跟八達通卡有限公司推出新一代「一卡一芯雙錢包」運作方式的「嶺南通·八達通」推出後，嶺南通將可於深圳通設備上使用，並同時於深圳发行「嶺南通·深圳通」，另預計於2016年年中，原有後期采用英飛凌-{芯}-片標準的Type A卡版本的深圳通亦會升級為嶺南通。<ref>[http://www.lingnanpass.com/view.aspx?q_aid=1936&q_cid=40 岭南通累计发卡超3960万张 将扩大联名卡应用]</ref>

目前，深圳通累計發卡2600萬張，手機深圳通用戶達到160萬，成為國內發卡量和使用量最大的“交通移動支付”應用。為方便市民辦理“深圳通”卡，深圳通在建設7個自營客服網點的基礎上，還在地鐵沿線站內設立了43個便民服務網點，並結合便利店、郵政支局、郵政代辦所等服務網點，共在全市布建了5500多個銷售點。據介紹，深圳通已在地鐵站等投放壹批自動售卡機，進壹步方便市民自助購卡，拥有官方移动应用“鹏淘”（原“小魚支付”）。<ref>http://news.sina.com.cn/o/2015-04-20/084031737182.shtml</ref>

2018年起，深圳通二維碼業務目前已在深圳公交全線使用（公營線路）。<ref name=":0">{{Cite web|url=http://jt.sz.bendibao.com/news/2018119/800691.htm|title=深圳所有公交线正式支持刷二维码乘车 附开通、乘车流程 - 深圳本地宝|accessdate=2018-03-06|work=jt.sz.bendibao.com}}</ref>

== 經營現狀 ==
截至2012年12月，“深圳通”累计发卡量已经超过1800万张，设备装机超过50000台，充值终端7000多个，日刷卡量超过700万人次；应用范围全面覆盖了地铁、地面公交以及2500多家便利店等小额消费商户；由包括中国邮政在内的2000多个人工服务网点以及近500个24小时自助充值网点。<ref>{{cite web |url=http://www.shenzhentong.com/news/index.do?i=246&m=0301&a=detail |title=存档副本 |accessdate=2012-05-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20120514150555/http://shenzhentong.com/news/index.do?i=246&m=0301&a=detail |archivedate=2012-05-14 }}</ref>

深圳通公司与深圳市邮政局合作，开通学生卡办理业务，同时由深圳邮政参与“深圳通”小额支付平台的市场推广和运营。

== 發展更新 ==
* 2004年12月28日，[[深圳地铁|深圳地铁]]公司发行了一种可以用来乘坐地铁的深圳通。此种卡可以用来乘坐小巴（但经常不能使用），但旧版的公交IC卡不能用来乘坐地铁（由深圳市交通局於2002年至2004年間發行的深圳通卡則可乘坐深圳地鐵）。

* 2005年底，深圳通改由[[深圳通有限公司|深圳通有限公司]]统一发行并启用，新型的深圳通是一张集[[深圳地鐵|地铁]]储值卡、公交[[IC卡|IC卡]]、出租车和商场小额消费功能于一身的IC型储值卡。

* 2006年，[[深圳市|深圳市]]采用深圳通取代原来的公交IC卡。2006年5月开始，可以在相关地点办理换卡手续，而6月15日以后公交IC卡将退出使用。

* 2007年1月1日，深圳通首次推出地铁与地面公交换乘优惠，凡刷深圳通卡在78条公交线路与[[深圳地鐵|地铁]]之间换乘，可享受2角的票价优惠。

* 2007年6月1日至2009年7月23日，深圳通与[[7-Eleven|7-Eleven]]便利店合作推出以下各项业务<ref>{{cite web |url=http://www.shenzhentong.com/news/index.do?i=46&m=0301&a=detail |title=存档副本 |accessdate=2007-10-27 |deadurl=yes |archiveurl=https://web.archive.org/web/20071103224409/http://www.shenzhentong.com/news/index.do?i=46&m=0301&a=detail |archivedate=2007-11-03 }}</ref><ref>{{cite web |url=http://www.shenzhentong.com/news/index.do?i=174&m=0301&a=detail |title=存档副本 |accessdate=2009-10-22 |deadurl=yes |archiveurl=https://web.archive.org/web/20090907111322/http://www.shenzhentong.com/news/index.do?i=174&m=0301&a=detail |archivedate=2009-09-07 }} 关于7-11便利店暂停办理深圳通业务的通知</ref>：
** 消费，市民可持深圳通卡直接刷卡购买店内商品。
** 充值，每间店铺均可为市民提供深圳通卡的现场充值服务。
** 购卡，每间店铺均有深圳通卡陈列销售。

* 2007年12月1日，深圳公交巴士全面调价<ref>{{cite web |url=http://www.shenzhentong.com/news/index.do?i=75&m=0301&a=detail |title=存档副本 |accessdate=2007-12-17 |deadurl=yes |archiveurl=https://web.archive.org/web/20071217201139/http://www.shenzhentong.com/news/index.do?i=75&m=0301&a=detail |archivedate=2007-12-17 }}</ref>，深圳通在特区内的大部分巴士上都可使用，且可以享受到一定的折扣优惠：
** 票价3元以下（含3元）部分8折优惠
** 3元以上6元以下（含6元）部分7.5折优惠
** 6元以上部分6.5折优惠

* 2009年7月，深圳通总发卡量突破600万张。

* 2009年9月，“深圳通”通达范围进一步扩大到出租车。

* 2011年6月，與[[中國移動|中國移動深圳分公司]]合作的手機深圳通正式發行，號碼所屬地在[[深圳|深圳]]的[[中國移動|中國移動]]手機用戶可於各移動服務廳換取手機深圳通卡(無需更換手機號碼)，便可使手機兼具深圳通功能而無需額外攜帶深圳通卡片。

* 2011年12月，[[中國聯通|中國聯通]]手機深圳通正式推出。

* 2012年9月，深圳通有限公司與八達通卡有限公司宣布推出二合一聯名卡-「'''互通行'''」（{{lang-en|-{'''Hu Tong Xing'''}-}}）。

* 2012年12月28日，深圳通有限公司與廣東嶺南通股份有限公司達成合作協議，將於2013年正式加入[[嶺南通|嶺南通]]。<ref>[http://www.lingnanpass.com/view.aspx?q_aid=303&q_cid=40 深圳通即将加入全省公交一卡通大家庭 嶺南通公司(2012年12月28日)]</ref>

* 2016年2月，為配合[[前海深港現代服務業合作區|前海深港現代服務業合作區]]之發展，「互通行」卡推出了「互通行·前海」。<ref>[http://www.shenzhentong.com/products/prodetail_101012001_100000126666753.html 新品推荐-深港“互通行”（前海）卡 深圳通有限公司]</ref>

未来，深圳通應用範圍還將擴展至缴纳停车费、水电燃气费、物业管理费，甚至是房租费、商场购物、公园景点门票费和小区体育设施使用费，以及乘搭[[廣深鐵路|廣深線列車]]。另外，相关部门正在沟通，深圳通可能成为“区域一卡通”（即同'''[[岭南通|岭南通]]'''性质差不多），正考慮跟[[广州市|广州市]]的「[[羊城通|羊城通]]」卡、[[中山市|中山市]]的「中山通」卡、[[珠海市|珠海市]]的「通達IC卡」、[[廣深鐵路|廣深鐵路]]，以及[[韶關市|韶關市]]和[[清遠市|清遠市]]等地合作，推出通用的IC卡。

=== 一般卡種 ===
* 押金卡 （押金20元/張，不记名卡，不可挂失，可按规定退卡{{NoteTag|1=原租用版卡会從押金中每月扣除0.5元作為租金，40個月扣完即止，自2014年1月1日起发行押金卡，不再销售租用版卡，押金卡不收取租金}}）
* 纪念卡 （不记名卡，无押金，不可透支，按市场需求自主定价发行；退卡时购卡费用不退，只能退还卡内余额，如在质保期（第一笔消费交易3个月）内，卡表面完好，卡芯读不到的坏卡，可换新卡。）

=== 特殊卡種 ===
* 特殊优惠卡（学生卡、残疾人卡、优抚卡、儿童乘车卡等，须按规定在客服網點办理）
* 银行联名卡
* 企业定制卡
* 手机深圳通（适用于[[中国移动|中国移动]]、[[中国联通|中国联通]]及[[中国电信|中国电信]]深圳用户）（已絕版）
* 互通行卡（深圳通[[八達通|八達通]]二合一聯名卡，[[人民幣|人民幣]]、[[港幣|港幣]]雙帳戶）<ref name="2 in 1">[http://money.163.com/12/0811/04/88JMIQTA00253B0H.html 八达通与深圳通年内推出联名卡 证券时报网2012年8月11日]</ref>：

在早期由深圳市交通局、[[深圳地鐵|深圳地鐵]]和深圳通有限公司發行的「深圳通」卡均與香港的[[八達通|八達通]]一樣使用了[[Sony|Sony]]公司[[FeliCa|FeliCa]]的[[射频识别|RFID]]晶片卡，這是因為當時深圳地鐵和深圳市交通局均希望其系統可兼容香港的[[八達通|八達通]]，使兩種卡在兩地互通。可是因涉及貨幣的自動兌換，如何將[[人民幣|人民幣]]金額轉化為[[港幣|港幣]]金額的問題未能達成共識而進度緩慢，不過商議已開始有進展<ref>[http://www.smartcard.org.cn/date1/page6366.html 香港“八達通”與“深圳通”有望合二為一 - 中國智能卡網]</ref>。[[八達通|八達通]]跟[[羊城通|嶺南通]]於2011年8月23日簽定合作協議<ref>[http://www.hkcd.com.hk/content/2011-08/24/content_2798447.htm 港深通卡 咫尺天涯 香港商報2011年8月24日] {{webarchive|url=https://web.archive.org/web/20141031212045/http://www.hkcd.com.hk/content/2011-08/24/content_2798447.htm |date=2014年10月31日 }}</ref>，而八達通也於2012年8月宣佈年內將推出深圳通八達通二合一聯名卡<ref name="2 in 1"></ref>。

然而早期的「深圳通」卡是由日本[[Sony|Sony]]公司直接供貨，其高昂的卡片成本，封閉的卡片標準和漫長的供貨周期（最長為四個月），在2006年至2007年間便經常出現供應短缺的情況<ref>[http://www.smartcard.org.cn/date1/page2061.html 批量深圳通IC卡將上市 新型卡改造有望擺脫受制於外企的困境 - 中國智能卡網]</ref>，直至2007年12月，深圳通有限公司決定改用由中國內地[[握奇數據系統有限公司|握奇數據系統有限公司]]生產<ref>[http://www.smartcard.org.cn/date1/page2106.html 握奇數據中標深圳通 非接觸智能卡促出行方式變革 - 中國智能卡網]</ref>，使用[[德國|德國]][[英飛凌|英飛凌]]提供的[[TimeCOS®FLY|TimeCOS®FLY]]芯片<ref>[http://www.smartcard.org.cn/date1/page4285.html 采用英飛凌芯片的深圳通CPU卡成功發行200萬張 - 中國智能卡網]</ref><ref>[http://www.smartcard.org.cn/date1/page3313.html 握奇數據：TimeCOS®FLY暢行深圳 - 中國智能卡網]</ref>，符合[[ISO/IEC_14443|ISO/IEC 14443]]標準的Type A卡之後（2008年1月15日起推出），「深圳通」卡供不應求的情況才有所改善<ref>[http://www.smartcard.org.cn/date1/page6347.html 升級版深圳通IC卡安全無虞 - 中國智能卡網]</ref>。

不過深圳通和八達通的聯名卡 - '''「互通行」'''則仍然使用[[Sony|Sony]]公司[[FeliCa|FeliCa]]的[[射频识别|RFID]]晶片卡，以配合該卡「一卡一芯雙錢包」運作方式，其中深藍色款的深圳版本「互通行」卡已於2014年1月19日開售，但在2016年起已改為發售「互通行·前海」。

由于深圳通尚未完成升级为[[岭南通|岭南通]]，故目前尚未推出「嶺南通·深圳通」，唯於2015年2月1日起[[岭南通|岭南通]]可於部份深圳通設備上使用。當嶺南通公司跟八達通卡有限公司推出新一代「一卡一芯雙錢包」運作方式的「嶺南通·八達通」推出後，預計嶺南通2017年年底將可於深圳通設備上全面使用，並同時於深圳发行「嶺南通·深圳通」。另預計於2019年初，原有後期采用英飛凌-{芯}-片標準的Type A卡版本的深圳通亦會升級為嶺南通。

=== 增值 ===
[[File:Shenzhen_Tong_Add_Value_Machine_(bank_card).jpg|thumb]][[車公廟站_(深圳)|車公廟站]]内的深圳通銀行卡自助增值机，可進行插卡增值及空中增值操作，但僅接受銀行卡支付。]]

==== 增值方式 ====
深圳通的增值方式主要分為三種：
* 插卡增值：將深圳通插入自助增值機完成充值手續，適用於卡片式深圳通。
* 感應增值：將深圳通輕觸感應器完成充值手續，適用於任何形式的深圳通。
* 空中增值：通過專門的充值機、POS機、網上營業廳或官方移动应用“鹏淘”完成充值手續，適用於手機深圳通及部份銀行聯名卡。
