'''深水埗'''（{{lang-en|Sham Shui Po}}）是[[香港十八區|香港十八區]]的[[深水埗區|深水埗區]]之主要部分，位於[[九龍半島|九龍半島]]西北部，有着悠久的歷史。當中「埗」、「埔」及「埠」三字相通<ref>例子有[[深水埔街坊福利會小學|深水埔街坊福利會小學]]</ref>，意思是深水而無淺灘的海灣<ref>{{Cite web|url=http://nextplus.nextmedia.com/news/latest/20140216/20860|title=說南蠻|accessdate=2018-08-17|author=陶傑|date=2014-02-16|publisher=壹週刊}}</ref>，皆適合作碼頭，由此可知深水埗過去是一個深水碼頭。

== 地域 ==
深水埗包括[[深水埗區|深水埗區]]原有臨近海傍的一部分，大致上以[[大埔道|大埔道]]、[[界限街|界限街]]和[[欽州街|欽州街]]為界，1970-80年代以[[通州街|通州街]]為海岸綫，其後因應填海工程而成為現今模樣。欽州街至[[東京街|東京街]]則為'''深水埗'''與[[長沙灣|長沙灣]]的過渡地帶，故此[[深水埗區議會|深水埗區議會]]的[[長沙灣_(選區)|長沙灣選區]]大部份屬於'''深水埗'''範團。深水埗與[[又一村|又一村]]、[[石硤尾|石硤尾]]、[[長沙灣|長沙灣]]、[[荔枝角|荔枝角]]、[[昂船洲|昂船洲]]等地在行政區劃上同屬[[深水埗區|深水埗區]]，而這些地區沒有明顯分界而且互相交接，加上興建[[香港地鐵|地鐵]][[荃灣綫|荃灣綫]]時，[[深水埗區|深水埗區]]的車站沒有按照其所在地的地名進行命名{{efn|例如現時[[荔枝角站|荔枝角站]]其實位於[[長沙灣|長沙灣]]北面，而[[美孚站|美孚站]]所在實際為位處[[荔枝角|荔枝角]]}}，區內設施亦有類似情況{{efn|例如[[長沙灣政府合署|長沙灣政府合署]]位於深水埗[[長沙灣道|長沙灣道]]與[[欽州街|欽州街]]交界，以長沙灣道命名以區別位於[[元州街|元州街]]的[[深水埗政府合署|深水埗政府合署]]；而服務全區的[[深水埗運動場|深水埗運動場]]位處[[長沙灣站|長沙灣站]]和[[荔枝角站|荔枝角站]]中間的[[長沙灣道|長沙灣道]]、[[昌華街|昌華街]]、[[荔枝角道|荔枝角道]]}}，因此市民經常出現混淆，將[[深水埗區|深水埗區]]內非深水埗的地方誤作深水埗的一部分。

==歷史==
過去數十年香港不少地區出土過[[新石器時代|新石器時代]]的文物，其中[[大埔區|大埔區]]更發現[[舊石器時代|舊石器時代]]文物，所以深水埗也可能在新石器时代已經有人類定居和活動。隨着[[李鄭屋東漢墓|李鄭屋漢墓]]的出土，可以確定不晚於[[東漢|東漢]]時代深水埗附近有人類定居。
[[File:Victoria_City_and_Kowloon_1915.jpg|thumb]]
據舒懋官主编的[[新安縣志|新安縣志]]記載，清代[[嘉慶|嘉慶]]年間，[[九龍半島|九龍半島]]的[[深水埗區|深水埗區]]內已有「深水莆」（亦作「深水埔」，亦即現今的深水埗）、「[[長沙灣|長沙灣]]」等村落。1898年[[大英帝國|英國政府]]與[[滿清政府|滿清政府]]在北京簽訂《[[展拓香港界址專條|展拓香港界址專條]]》租借[[新界|新界]]，[[港英政府|港英政府]]在取得[[界限街|界限街]]以北至[[深圳河|深圳河]]的土地，其後1930年代將界限街以北的深水埗、[[長沙灣|長沙灣]]、[[荔枝角|荔枝角]]及[[九龍塘|九龍塘]]一帶歸入九龍範圍，並命名為「[[新九龍|新九龍]]」。

進行市區發展之前，深水埗一帶已經有村落聚居，有位於北部的[[元洲|元洲]]村，東南的[[田寮村|田寮村]]，西面的[[菴由村|菴由村]]，以及西北的[[馬龍坑村|馬龍坑村]]。至於深水莆甚麼時候改名爲「深水埗」則難以稽考。1912年以前，政府並未熱衷於發展深水埗區，區內主要的道路設施僅有1902年建成的[[大埔道|大埔道]]及其後建成的[[青山道|青山道]]，其餘發展則由私人發展商小規模進行，如[[南昌街_(香港)|南昌街]]至[[桂林街|桂林街]]一帶的填海工程。數十年前，深水埗曾稱作「深水埔」。時至今日，深水埔[[街坊福利會|街坊福利會]]仍然以「深水埔」爲名。

現今深水埗已經成為九龍西北部的核心區，人口密度非常高，區內有業主經營[[劏房|劏房]]、棺材房以至[[籠屋|籠屋]]，部分居所生活環境惡劣。商業活動發達，並已經成為電子產品、潮流玩意、服裝配飾、古玩玉石等多方面商品集中地，每天車水馬龍、熙來攘往。而[[西九龍填海區#深水埗西|深水埗西]]填海區[[深旺道|深旺道]]一帶亦陸續發展高密度的住宅區，區內亦有零星的重建項目，改變外界對深水埗集中着傳統[[唐樓|唐樓]]的印象。

1980年代曾為報紙撰寫本地旅遊故事的方志研究專家[[沈思|沈思]]先生於2018年接受有關地名由來的訪問時表示，早於鴉片戰爭前已經有「深水埗」這地名出現在文獻中，而於1898年，於《展拓香港界址專條》後，深水埗就是當時人口密度最多的地區<ref>[https://bkb.mpweekly.com/cu0001/20180301-68027 【真本土】深水埗命名因為水夠深？方志研究專家沈思拆解地名由來]</ref>。

* 電腦商場：深水埗福華街一帶的電腦商場世界知名。最早改建成電腦商場的是[[黃金戲院|黃金戲院]]附近的“[[高登電腦中心與黃金電腦商場|高登電腦中心]]”，主要發售與電腦有關的電子零件及[[Apple_II|蘋果II型]]兼容機。後來因為電腦在香港開始普及，加上高登商場的名聲開始流傳，連地下賣時裝的“[[黃金商場|黃金商場]]”也轉型售賣電腦產品。由於兩家商場英文名字非常相近，不少外國人都以為是同一家商場。隨着人流增加，兩家商場也開始分工：樓上主要賣電腦硬體，樓下主要賣電腦配件、電腦書籍及電腦傢俬等。到了1990年代末至2000年代初，部分店舖被利用作販賣盜版軟件及成人影蝶，經政府大力打擊之下，逼使這些商舖轉型。而社會對電腦的需求愈來愈大，兩個商場的人流未有減少。黃金商場目前主要售賣數碼產品周邊配件及電腦書籍，高登商場則仍以售賣電腦硬件為主，[[福華街|福華街]]，[[欽州街|欽州街]]、[[福榮街|福榮街]]、[[桂林街|桂林街]]所在的街區內曾有四家電腦商場；而鄰近的[[西九龍中心|西九龍中心]]亦一度有多家大型電腦連鎖店開業，但都已經結束營業。2002年區內舉辦首屆[[香港電腦節|香港電腦節]]，每年約10月至翌年2月舉行大型電腦業戶外展覽活動，每次活動吸引超過十萬人人次計算參觀者進場。最初舉辦地點在[[福華街|福華街]]、[[福榮街|福榮街]]及[[桂林街|桂林街]]。2010年起，改於[[長沙灣遊樂場|長沙灣遊樂場]]舉行。2002年至2010年，[[香港電腦商會|香港電腦商會]]和深水埗[[區議會|區議會]]合辦。2010年後，[[香港電腦商會|香港電腦商會]]主辦，惟2011年後則未有舉行。而區內電腦商場針對每年8月的[[香港電腦通訊節|香港電腦通訊節]]，往往於會期相近日子舉行[[腦場電腦節|腦場電腦節]]，以免生意有所影響。

* [[西九龍中心|西九龍中心]]：是區內大型購物商場，亦為區內地標之一，屬區內少有的商業用地。

* [[鴨寮街|鴨寮街]]：鴨寮街本來是專門賣二手貨品的街道，於1970年代變成[[電子零件|電子零件]]集散地，以出售價廉物美的貨品為主，例如二手電器(如電視機、音響設備)、舊書刊、舊[[唱片|唱片]]等。1990年代[[香港有線電視|香港有線電視]]開播，不少不法商人售賣聲稱可收看有線電視的[[解碼器|解碼器]]一類電子器材。經多次打擊後，攤販轉型售賣影音設備、[[手提電話|手提電話]]及mp3配件。有關鴨寮街的風貌曾被[[無綫電視|無綫電視]]拍成劇集[[翻新大少|翻新大少]]。區內後巷亦曾吸引小販擺賣玉石，吸引顧客「尋寶」，更曾釀成無牌小販於法庭上自焚的慘劇<ref>{{cite web|language=zh-hk|url=http://orientaldaily.on.cc/cnt/news/20110824/00176_004.html/|title=玉石小販庭上自焚轟動|publisher=東方日報}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref><ref>{{cite web |language=zh-hk |url=http://christiantimes.org.hk/Common/Reader/News/ShowNews.jsp?Nid=3847&Pid=2&Version=591&Cid=87&Charset=big5_hkscs/|title=螳臂集-留民眾一條生路|publisher=時代論壇}}</ref>，促使政府於[[通州街|通州街]]設立[[深水埗玉石市場|深水埗玉石市場]]，加以管理。

*深水埗區也是多個[[香港歷史建築|香港歷史建築]]的所在地，包括[[深水埗警署|深水埗警署]]、[[北九龍裁判法院|北九龍裁判法院]]、[[美荷樓|美荷樓]]、[[深水埗公立醫局|深水埗公立醫局]]、[[南昌押|南昌押]]、[[嘉頓|深水埗嘉頓中心]]及多棟[[第二次世界大戰|第二次世界大戰]]前的[[唐樓|唐樓]]；以及多座古廟，計有[[深水埗天后廟|深水埗天后廟]]、[[深水埗三太子及北帝廟|深水埗三太子及北帝廟]]及[[深水埗武帝廟|深水埗武帝廟]]等。

*[[福華街|福華街]]、[[福榮街|福榮街]]、[[北河街|北河街]]及[[桂林街|桂林街]]一帶[[食肆|食肆]]林立，當中較有特色的食品包括[[咖央|咖央]][[多士|多士]]、[[豬潤麵|豬潤麵]]、[[牛柏葉|牛柏葉]]麵、[[蝦籽|蝦籽]][[撈麵|撈麵]]、[[削搖麵|削搖麵]]、[[車仔麵|車仔麵]]、[[糖水|糖水]]及[[豆腐花|豆腐花]]等。而區內著名餐廳包括[[公和荳品廠|公和荳品廠]]、由[[陳灼明|陳灼明]]（明哥）經營的[[北河燒臘飯店|北河燒臘飯店]]、強記大牌檔、[[維記咖啡粉麵|維記咖啡粉麵]]、[[新香園|新香園]]、[[添好運點心專門店|添好運點心專門店]]、[[劉林記麪家|劉林記麪家]]及屹立深水埗80年的[[信興酒樓|信興酒樓]]等，部分食肆更獲得[[米芝蓮指南|米芝蓮指南]]所推薦。

*區內紡織製衣業源遠流長，相關行業蓬勃，形成多條特色街街道<ref>{{cite web |language=zh-hk |url=http://hknea.org/download/002.pdf|title=各行各業-疋頭批發業(P.240-242) 及 各行各業-製衣業|publisher=深水埗風誌 }}</ref>，包括銷售布料的[[欽州街小販市場|欽州街小販市場]]、[[南昌街|南昌街]]以南的[[大南街|大南街]]、[[基隆街|基隆街]]、[[汝州街|汝州街]]、[[黃竹街|黃竹街]]、[[楓樹街|楓樹街]]等；[[長沙灣道|長沙灣道]]介乎[[欽州街|欽州街]]至[[界限街|界限街]]的成衣批發零售店舖；位於[[荔枝角道|荔枝角道]]的縫紉機械相關店舖；[[南昌街|南昌街]]上的鈕扣繩索拉鍊花邊等店舖；稱為「珠仔街」的[[汝州街|汝州街]]相關珠飾絲帶等製衣配件店舖；而[[基隆街|基隆街]]、[[大南街|大南街]]亦吸引數間大大小小的手作皮革店設立，如阿里皮藝、兄弟皮藝、AURORA Leatherware、昌興皮革和聯昌皮號等。<ref>{{cite web |language=zh-hk |url=http://blog.ulifestyle.com.hk/blogger/siumeng/2014/08/%E6%89%8B%E4%BD%9C-%E6%B7%B1%E6%B0%B4%E5%9F%97%E8%97%9D%E9%81%8A%E3%83%BB%E7%9A%AE%E9%9D%A9%E5%BA%97%E6%BC%AB%E6%BC%AB%E9%81%8A/|title=深水埗皮革店|publisher=Blog.ULifestyle.com.hk}}</ref>

*[[北河街市政大樓|北河街市政大樓]]所在的「北河四街」一帶（北河街、[[基隆街|基隆街]]、桂林街及[[大南街|大南街]]）附近，近年就出現夜墟、天光墟的地攤，基層市民把不少殘舊衣物及雜貨擺賣，滿足基層街坊所需之餘，同時賺取微薄收入，亦成為區內的[[南亞裔|南亞裔]]聚集的地點。唯2015年起，食環處疑「趕盡殺絕」，曾勒令該處嚴禁擺賣十天，未知否跟當區區議會施壓有關。<ref>[http://www.post852.com/%E9%A3%9F%E7%92%B0%E7%BD%B2%E5%8B%92%E4%BB%A4%E7%A6%81%E5%9C%B0%E6%94%A4%E6%93%BA%E8%B3%A3%E5%8D%81%E5%A4%A9%E3%80%80%E8%B6%95%E7%B5%95%E6%B7%B1%E6%B0%B4%E5%9F%97%E5%9F%BA%E5%B1%A4%E5%A4%9C%E5%A2%9F/ 食環署勒令禁地攤擺賣十天　趕絕深水埗基層夜墟 852郵報 2015-01-21]</ref>
