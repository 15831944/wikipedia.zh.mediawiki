'''深坑黃氏永安居'''，1915年建造完成，位於今[[台灣|台灣]][[新北市|新北市]][[深坑區|深坑區]][[北深路|北深路三段]]8號。[[中華民國政府|中華民國政府]]於1992年5月22日認定它為三級[[古蹟|古蹟]]，具有極大的價值與歷史保存意義。其命名的由來為「永久平安居住」的意思。根據[[中華民國政府|中華民國政府]]文化部文化資產局的介紹，興建永安居所使用的木材是從[[中國大陸|中國大陸]]運過來的福杉，永安居是一個典型的泉州安溪三合院大厝<ref>[https://nchdb.boch.gov.tw/assets/advanceSearch/monument/19920523000001 深坑黃氏永安居].文化部文化資產局</ref> 。

作者李乾朗在其著作台灣古建築圖解事典（ISBN 957-32-4957-X）一書的第245頁提到永安居的牆上有鑿銃眼數十孔，是為了防止外敵入侵而建的槍孔。

== 歷史沿革 ==
[[乾隆|乾隆]]年間，黃氏紫雲一族由[[福建|福建]][[泉州|泉州]][[安溪|安溪]]來到[[台灣|台灣]]，到今[[深坑區|深坑區]]定居。日治時期，黃氏子孫經商有成，先後在深坑建了七座古厝，也因此為深坑最重要的古蹟文化遺產<ref>[http://thesis.lib.cycu.edu.tw/ETD-db/ETD-search-c/getfile?urn=etd-0613101-135011&filename=8681008.pdf 黃氏宗族在深坑地區的墾拓歷程及其古厝之研究-碩士論文]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。[[中華民國政府|中華民國政府]]打算把七座古厝都認定為古蹟，但是由於另外六座古厝的黃氏子孫不願意將其古厝列為古蹟，所以政府將這六座古厝本身和週邊相關列為非古蹟的古建築保存區<ref>[http://www.cpami.gov.tw/chinese//filesys/file/chinese/committee/dpc/649.pdf 內政部都市計畫委員會第649 次會議紀錄]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。
*深坑黃氏七座古厝：
**開基祖厝（北深路二段120號）
**黃連山祖厝（文山路二段25號）
**福安居（文山路二段31號）
**永安居（北深路三段8號）
**興順居（北深路二段133號）<ref>[http://travel.udn.com/mag/travel/storypage.jsp?f_ART_ID=52770 聯合新聞網2012/08/01報導：繼永安居之後，興順居保持完整將被政府指定為古蹟。]</ref>
**潤德居（阿柔洋33號）
**六房厝（阿柔洋35號）

== 觀光產業 ==
深坑黃氏永安居帶動週邊觀光產業，最著名而廣為人知的就是[[深坑區|深坑]][[臭豆腐|臭豆腐]]。不過本來被吸引來看古蹟完順便吃臭豆腐的，反而現在臭豆腐比古蹟有名。週末假日也有許多外國[[觀光客|觀光客]]前來欣賞古蹟<ref>{{Cite web |url=http://yoshiko2.web.fc2.com/taiwan2-2.html |title=日本遊客來台觀光後用日文在其網站上介紹永安居(有附圖)。 |access-date=2012-05-24 |archive-url=https://web.archive.org/web/20150119061827/http://yoshiko2.web.fc2.com/taiwan2-2.html |archive-date=2015-01-19 |dead-url=yes }}</ref>。

== 票價 ==
深坑黃氏永安居入內參觀採收費方式，全票60元，學生票40元<ref>[http://www.hty.idv.tw/feedback.htm 深坑黃氏永安居參觀資訊。]</ref>，所有的門票收入做為[[中華民國政府|中華民國政府]]維護永安居的經費。
