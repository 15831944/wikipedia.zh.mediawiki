'''深圳站'''，當地人稱之為“羅湖站”或“羅湖火車站”，位於[[中國大陸|中國大陸]][[廣東省|廣東省]][[深圳市|深圳市]][[羅湖區|羅湖區]][[南湖街道_(深圳市)|南湖街道]]建設路，是[[廣深鐵路|廣深鐵路]]的南端起點[[铁路车站|车站]]。該站東廣場地下是[[深圳地鐵|深圳地鐵]][[深圳地鐵1號綫|1號綫]]的東端終點[[罗湖站_(深圳)|羅湖站]]，南側為[[罗湖口岸|罗湖口岸]]。旅客可在[[羅湖口岸|羅湖口岸]]的羅湖邊檢大樓辦理出境手續後，步行前往位於邊檢大樓另一側的[[香港入境處|香港入境處]]的[[羅湖管制站|羅湖口岸管制站]]，進入香港境後，於[[香港|香港]]的[[港鐵|港鐵]][[羅湖站_(香港)|羅湖站]]，乘搭[[東鐵綫|東鐵綫]]前往香港各區。

1898年，[[英国|英国]]开始向[[清政府|清政府]]提出[[九龙|九龙]]—[[广州|广州]]的铁路修筑计划，以方便英国将货物和人员快速运往内地市场。1898年，英国強索[[廣九鐵路|廣九鐵路]]的貸款權，並在英国的压力下，清政府铁路大臣[[盛宣怀|盛宣怀]]与英商[[怡和洋行|怡和洋行]]于1899年簽定了《广九铁路借款草合同》。按这份草约，广九铁路计划修筑121英里，中间以深圳（当时实为[[宝安县|新安县]]）[[罗湖桥|罗湖桥]]为界，分为英、华两段。其中英段铁路（今[[港铁|港铁]][[东铁线|东铁线]]）长22英里，由英国政府负责修建，资金由[[英国政府|英国政府]]在[[香港|香港]]发行[[公债|公债]]筹集；华段铁路（今[[广深铁路|广深铁路]]）长度为89英里，由清政府负责修建。 

广九铁路华段在当时的[[深圳墟|深圳墟]]设站，名为'''深圳墟火车站'''，始建于1910年<ref name=1992nj>{{cite book |author=深圳經濟特區年鑒編輯委員會 |title=深圳經濟特區年鑒：深圳鐵路新客站建設|year=1992年|publisher=廣東人民出版社|location=|isbn=|page=280-281}}</ref>，原址位于今天的深圳罗湖区解放路、和平路路口，[[東門老街|東門老街]]附近。1911年10月8日，隨著[[廣九鐵路|廣九鐵路]]華段建成通车同時啟用，为客货兼营车站。由于铁路的贯通，令深圳墟的发展比其他周边地区更快，渐渐形成以“罗湖旧城”为主体的东门区域，也成为深圳最早期的经济活动中心，深圳墟从此逐渐发展成为一个边城小镇<ref>{{Cite web|url=http://www.szdaj.gov.cn/content.aspx?cid=b470520c-a94a-4716-8fca-2f6ed50709ef|title=深圳墟当年的确有道“东门”|accessdate=2010-11-25|publisher=深圳档案|date=2007-04-16|deadurl=yes|archiveurl=https://web.archive.org/web/20110809064852/http://www.szdaj.gov.cn/content.aspx?cid=b470520c-a94a-4716-8fca-2f6ed50709ef|archivedate=2011-08-09}}</ref>。1949年，深圳墟火车站仅有两条股道，车站候车室35平方米，维持基本旅客出入境及货物列车过轨。  

[[中華人民共和國|中華人民共和國]]成立以后，深圳墟火车站于1950年从东门老街遷到較接近[[羅湖口岸|羅湖口岸]]、[[罗湖桥|罗湖桥]]的現址，与香港的罗湖站仅一桥之隔。因当地连接广九铁路，交通便利，人口聚居較多，工商業比較興旺，寶安縣政府在1953年由南头遷至10公里外的深圳墟（今蛟湖路附近），深圳墟改名爲“深圳鎮”，而深圳墟火车站也改称'''深圳站'''。

当时车站非常简陋，由车站职工自己动手，伐木裁板作顶，撑起枕木为柱，利用简单材料，建成了第一代候车室，站场面积仅有35平方米。由1950年代起深圳站经历了多次改扩建。车站地势较低，周围都是泥塘和稻田，一下雨就被淹，车站职工挖塘泥、挑煤渣垫高铁路两侧，建成了第一个站台。车站门前有一个小广场。旅客从广场沿台阶拾级而上进入售票厅和候车室，左边是出入境大厅，由边防检查站和海关把守，从出入境通道，一眼望去就可以看见罗湖桥。

1955年车站进行了首次改建，站房右侧建起了一栋两层楼，楼上是旅客食堂，楼下供旅客候车，面积从原来的400多平方米扩大到977平方米。1959年车站进行第二次改建，站内新设立了贵宾室、[[海关|海关]]和边防检查站，站场面积扩大到1870平方米。1964年车站进行了第三次扩建，站场面积扩大到2800平方米。1972年车站进行了第四次扩建，站场面积扩大到4500平方米。

1977年[[笋岗站|笋岗站]]改造成货运站以前，深圳站是一个客货运三等[[铁路车站|站]]，车站有7条股道和一个[[机车转盘|转盘]]，运送过港物资的“三趟快车”也在这里接受[[海关|海关]]、边防和卫生[[检疫|检疫]]检查。<ref>《人民铁道》报，2006年11月2日，第二版</ref> 1979年，深圳站扩建一栋平房，专门作为内地旅客售票厅，原售票处改为港澳旅客售票处。1979年3月，中共中央和广东省政府决定把宝安县改爲深圳市，受廣東省和惠陽地區雙重領導；同年11月，中共廣東省委决定將深圳市改爲[[地級市|地區一級的省轄市]]，但直到1980年代初，深圳站站房仍然只是一座兩層小樓，設有候车室、贵宾室和港澳候车室。<ref>[http://www.chnrailway.com/news/2009220/200922011493992262481.shtml 百年古榕见证深圳站的变迁]</ref> 

1982年，深圳站引进港资合作经营车站旅客食堂，成为了全国铁路第一家与港商合资经营的企业<ref name=szps>{{Cite web|url=http://sztqb.sznews.com/html/2010-07/13/content_1152016.htm|title=小平题写站名特意少写个“站”字|accessdate=2010-11-25|publisher=深圳特区报|date=2010-07-13|deadurl=yes|archiveurl=https://web.archive.org/web/20100723062251/http://sztqb.sznews.com/html/2010-07/13/content_1152016.htm|archivedate=2010年7月23日|df=}}</ref>。

在改革开放之前，深圳站一天只有3趟旅客列车始发，发送旅客不足2000人<ref>{{Cite web |url=http://www.cnr.cn/home/column/jdzg/huiyi/200408220099.html |title=小平同志曾为深圳火车站题写站名 |accessdate=2010-11-25 |publisher=中广网 |date=2004-08-22 }}</ref>。

随着中国实行[[改革开放|改革开放]]政策，以及1980年深圳设立[[经济特区|经济特区]]，经由深圳站前往[[中国大陆|中国大陆]]各地的旅客日渐增加。为配合经济发展，1983年7月14日，深圳火车站改造工程动工，工程包括车站扩建、一座联检大楼，以及行人天桥等。1983年深圳火车站由三等站升为二等站。1985年2月，深圳站由一个[[二等站_(中国大陆)|二等站]]升级成为[[一等站_(中国大陆)|一等站]]。1985年6月13日，[[羅湖口岸|罗湖联检大楼]]投入使用，更加方便了深港两地人员的往来。1987年广深铁路复线建成，深圳站已扩至2条正线，4条到发线，5条其他用途线，候车室1610平方米，年输送旅客510万人次<ref>《广东省年鉴志书－广东省志－铁路志》</ref>。从1980年至1990年，深圳站始发的旅客列车和旅客发送量逐年上升，到1990年发展至每天开行25趟旅客列车，发送旅客2.2万人次<ref name=szps />。但至1980年代末，深圳站的规模已不能适应粤港间旅客快速增长的需要。

===重建===
[[File:Shenzhen0.JPG|thumb]]
[[File:China_railways_CRH1_high_speed_train_cimg1667bvehk.jpg|thumb]]停靠深圳站站台（2006年12月）]]
1988年初，经铁道部批准，决定在原位置进行扩建改造<ref name=1992nj />，深圳市委书记、市长[[李灏_(1926年)|李灏]]亲自担任新火车站建设领导小组组长。深圳新客站工程自1988年11月10日动土施工，东主楼站屋1990年6月26日动工。这次是把车站几乎全部爆破拆除，工程进行时旅客乘车出站全部暂移到西边的和平路一带。1991年10月12日，投资2.2亿元人民币的深圳新客站建成并举行了落成典礼并正式投入使用，同时升级为[[一等站_(中国大陆)|一等客运站]]，当日上午时任国务院副总理的[[田纪云|田纪云]]为新站落成剪彩，下午时任[[国务院总理|国务院总理]]的[[李鹏|李鹏]]视察深圳站并题词：“加强深圳新车站管理，创建文明窗口”<ref>{{Cite web |url=http://www.cb-h.com/news/hzbh/2010/1110/10111063F370K09KD033492.html |title=深圳火车站“三级跳” |accessdate=2010-11-25 |publisher=中国商报 |date=2010-11-11 }}{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。1990年10月，时任深圳市委书记、市长的李灏到北京请邓小平题词时，邓小平出于对深圳特区发展的期望，决定不单为车站题词，而要给深圳人民题，因此特意少写了个“站”字。深圳火车站由此成为全国继[[重庆站|重庆站]]、[[天津站|天津站]]等之后又一个由邓小平题写站名的车站，同时也是部分没有“站”字的车站名<ref name=szps />。1992年1月19日上午，鄧小平在[[深圳经济特区|深圳经济特区]]建立12年之后，乘坐專列抵達深圳火車站，展开了[[邓小平南巡|邓小平南巡]]的序幕。

2003年，深圳站再次進行了大规模的整治改造工程<ref>{{Cite web |url=http://www.southcn.com/news/gdnews/areapaper/200306190367.htm |title=深圳火车站将整体改造 重新规划提高企业形象 |accessdate=2010-11-25 |publisher=南方网-南方都市报 |date=2003-06-19 }}</ref>。在改造之前，火车站内商铺摊位泛滥，仅签有合同的商铺就近百家，旅客的候车面积竟被占用了5000余平方米，严重影响车站秩序和候车环境，治安不佳，为人诟病<ref>{{Cite web |url=http://www.southcn.com/news/dishi/shenzhen/ttxw/200312040204.htm |title=深火车站乞讨拉客等现象仍较为普遍 |accessdate=2010-11-25 |publisher=南方网-南方都市报 |date=2003-12-04 }}</ref>。为此，是次工程首要是整治站内商铺，尤其将火车站内庞大的桑拿按摩业取消<ref>{{Cite web |url=http://www.gzuda.gov.cn/news/view.asp?id=XW200312291034203841&fdID=CL200405270901056577&tbColor=DBE2FF&trColor=C7CFF3 |title=百年老站改造竣工 深圳市火车站完成“变脸” |accessdate=2010-11-25 |publisher=南方都市报 |date=2003-12-29 }}{{Dead link|date=2018年6月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>，扩大了候车室，几个月里拆除了近100间商铺和200多块广告牌，清理经商人员近3000名<ref>{{Cite web |url=http://news.sina.com.cn/o/2006-01-25/03298072200s.shtml |title=车站变迁 旅客心暖 |accessdate=2010-11-25 |publisher=中国青年报 |date=2006-01-25 }}</ref>。另一方面，为方便旅客候车，车站改造后，将一楼候车室划分为数个候车室，将购票、候车的短途、长途旅客分开，廣深城際列車短途旅客和长途列車旅客都分開在不同候车室，而長短途售票處亦被分開，长途候车室还增设了软席候车室、军人候车室、母婴候车室等服务设施<ref name=gzuda>{{Cite web |url=http://www.gzuda.gov.cn/news/view.asp?id=XW200312291034203841&fdID=CL200405270901056577&tbColor=DBE2FF&trColor=C7CFF3 |title=百年老站改造竣工 深圳市火车站完成“变脸” |accessdate=2010-11-25 |publisher=南方都市报 |date=2003-12-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20131022124057/http://www.gzuda.gov.cn/news/view.asp?id=XW200312291034203841&fdID=CL200405270901056577&tbColor=DBE2FF&trColor=C7CFF3 |archivedate=2013年10月22日 |df= }}</ref>。同時深圳站站前廣場也因修建深圳地鐵羅湖站和換乘體而封閉。至2004年12月28日，深圳地铁1號綫正式开通运营，与之配套的火车站广场地铁交通层投入使用；2005年8月连接火车站和罗湖口岸的走廊开通；2005年10月，历时两年的火车站客运改造工程竣工，形成一个集公交车、出租车、地铁、火车等各种交通为一体的立体交通枢纽。

2009年，为配合深圳市罗湖区的市容环境提升行动方案，广深铁路股份有限公司投资1700余万元，展开广深铁路深圳段沿线环境综合整治，对深圳火车站及周边环境整治，先后完成了深圳站的围墙和电子屏改造，地下二层广深售票厅即地铁交通层增加一个入口，方便旅客进站乘车；车站东侧新增一块长20米、高6米的进站电子显示屏，滚动显示车次、票额、候车区域等实时信息；改造东、西侧外墙，行李包裹托运厅、行李包裹领取处等合计总面积2274平方米的外立面；更换一、三站台约550平方米的地面；对车站一楼动车组的售票场地进行了内部装饰<ref>{{Cite web |url=http://sztqb.sznews.com/html/2010-08/03/content_1179013.htm |title=广深铁路：火车站及周边整治9月底完成 |accessdate=2010-11-25 |publisher=深圳特区报 |date=2010-08-03 |deadurl=yes |archiveurl=https://web.archive.org/web/20101111054519/http://sztqb.sznews.com/html/2010-08/03/content_1179013.htm |archivedate=2010年11月11日 |df= }}</ref><ref>{{Cite web |url=http://jt.sz.bendibao.com/news/2009918/131573.htm |title=深圳火车站新装迎国庆 |accessdate=2010-11-25 |publisher= |date=2009-09-18 }}</ref>。

== 旅客列车目的地 ==
深圳站每天最多共开行130班列车，且全部为始发列车。其中超过80%的车次为[[广深动车组列车|广深动车组列车]]（车次为C7002-C7228（逢双号），终到{{站|广州}}或{{站|广州东}}），其余20班均为普速列车。

===城际列车===
[[File:201901_CRH6A-4133_as_DJ7787_at_Shenzhen_Station.jpg|thumb]]在深圳站站台待发]]
深圳站是[[广州|广州]]与深圳间[[城际列车|城际列车]]的始发站，其中深圳站城际动车组每天最高可开行110对，日常开行91对，视客流情况随时加开的备用动车组19对；官方時刻表報稱高峰時段每5分鐘一班、平时每10至15分鐘一班，是現時[[中國大陸|中國大陸]]列車班次最頻密的城際交通線路。這使得乘搭城際火車如同乘搭公交車或乘搭地鐵一樣方便。

深圳站也有开往[[蘇州|蘇州]]、[[九江|九江]]、[[上海|上海]]、[[北京|北京]]、[[沈阳|沈阳]]、[[西安|西安]]、[[梅州|梅州]]、[[武漢|武漢]]、[[青島|青島]]等方向的长途列车。據報道车站日均客流量9万余人，其中旅客5万余人，春运高峰时发送量超过11万人。
