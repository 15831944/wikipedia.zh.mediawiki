'''深圳技师学院'''是[[深圳市|深圳市]]公立全日制高等技工院校，4个校区分布在[[福田区|福田区]]、[[龙岗区|龙岗区]]、[[大鹏新区|大鹏新区]]。该校由政府全额资助运作，其毕业学生学历相当于[[專科學校|專科學校]]毕业。<ref>{{cite web |url=http://www.sz.gov.cn/zfgb/2009/gb640/200903/t20090303_832781.htm |title=深府办〔2009〕24号深圳市人民政府办公厅关于我市高级技工学校技师学院中等职业学校毕业生大专待遇问题的通知 |publisher=深圳政府在线}}</ref>福田区职教街的教学和培训用址在2014年搬迁<ref>{{cite web|url=http://www.zfcg.sz.gov.cn/sjgg/shdljg/cggg/201406/t20140603_2464895.htm|title=深圳技师学院投影设备搬迁招标公告|publisher=深圳市政府采购网|deadurl=yes|archiveurl=https://web.archive.org/web/20140915120449/http://www.zfcg.sz.gov.cn/sjgg/shdljg/cggg/201406/t20140603_2464895.htm|archivedate=2014-09-15}}</ref>至龙岗校区<ref>{{cite web |url=http://jb.sznews.com/html/2014-04/16/content_2843204.htm |title=技师学院9月启用龙岗新校 |publisher=深圳晶报多媒体数字报刊平台 |deadurl=yes |archiveurl=https://web.archive.org/web/20140915120638/http://jb.sznews.com/html/2014-04/16/content_2843204.htm |archivedate=2014年9月15日 |df= }}</ref>。

==学制==
该校设立三种学制：初中五年制高技班、高中三年制高技班、高中四年制预备技师班。<ref>{{cite web |url=http://www.ssti.net.cn/main/overview/collegeprofile/index.shtml |title=学院简介 |publisher=深圳技师学院}}</ref>

==专业==
该校开设汽车技术等10个系部28个专业。
