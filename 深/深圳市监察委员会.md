'''深圳市监察委员会'''，简称'''深圳市监委'''，是[[中华人民共和国|中华人民共和国]][[广东省|广东省]][[深圳市|深圳市]]的地方[[中华人民共和国国家监察机关|国家监察机关]]，与[[中国共产党深圳市纪律检查委员会|中国共产党深圳市纪律检查委员会]][[合署办公|合署办公]]，成立于2018年1月21日<ref>{{cite web |title = 深圳市监察委员会领导班子产生 |url = http://www.szmj.gov.cn/ttxw/201801/t20180121_10680309.htm |website = 深圳明镜网 |date=2018-01-21 |accessdate = 2018-03-23 }}</ref>，取代[[深圳市监察局|深圳市监察局]]的职能。深圳市监察委主任由[[深圳市人民代表大会|深圳市人民代表大会]]选举产生，并向其和[[深圳市人大常委会|深圳市人大常委会]]以及[[广东省监察委员会|广东省监察委员会]]负责。

现任主任为中共深圳市委常委、市纪委书记[[张子兴|张子兴]]<ref>{{cite web |url = http://www.szmj.gov.cn/xxgk/bzcy/ |title=班子成员 |website = 深圳明镜网 |date= |accessdate = 2018-03-23 }}</ref>。
