'''深圳出租车'''，是指在[[中国|中国]][[广东省|广东省]][[深圳市|深圳市]]行驶的出租车，是深圳主要公共交通工具之一，当地人俗称“-{的士}-”（来自英语音译），乘坐出租车俗称为“打的”。

除在街道上扬手招呼和在固定站点乘坐外，深圳出租车也可通过[[电话|电话]]及网络预约软件召唤，车费根据行驶里程及等候时间而定，并须缴付燃油附加费（电动车除外），大件行李则需另外收费。

深圳出租车营运牌照由政府统一投放，实行无偿使用，设立使用年限，<ref>{{cite web |author1=深圳市人民政府 |title=深圳市人民政府关于深化改革推进出租汽车行业健康发展的实施意见 |url=http://jtys.sz.gov.cn/jtzx/wycx/czcx/zc/201612/t20161228_5798694.htm |website=深圳市交通运输委员会 |publisher=深圳市交通运输委员会 |accessdate=2019-1-14}}</ref>并只能由具有相应资质的企业委托、发包或者出租给出租车驾驶员，个人不能直接承包。为促进珠三角地区一体化，深圳也允许周边地级市的出租车沿指定线路行驶。

深圳的士分为红色及蓝白色两种，两种的士均可在全市范围内行驶，惟前者以燃油为动力，后者为纯电动汽车。此外曾经在深圳服务的黑绿色{{link-en|吉利TX4|TX4}}无障碍的士与红色的士性质相同。截至2018年年底，深圳共有21,789辆的士，其中纯电动的士已达21,478辆。<ref name="sina">{{cite news |title=再见了，红绿的！深圳出租车颜色曾这样变迁…… |url=http://auto.sina.com.cn/j_kandian.d.html?docid=hqfskcn4079430&subch=bauto&hpid=00042 |accessdate=2019-1-14 |work=新浪网 |date=2019-1-4}}</ref>

===分区域经营===
[[File:Shenzhen_taxi_green_taxi_Volkswagen_Santana_Vista.jpg|300px]]绿色的士]]
2019年前，深圳的士曾实施分区域经营，红色的士（俗称“红的”）可行驶全市范围，绿色的士（俗称“绿的”）只能在原特区外（[[宝安区|宝安区]]、[[龙华区_(深圳市)|龙华区]]、[[龙岗区|龙岗区]]、[[坪山区|坪山区]]、[[大鹏新区|大鹏新区]]、[[光明区|光明区]]）经营，不能进入特区内，黄色的士（俗称“黄的”）则只能在原特区内（[[南山区|南山区]]、[[福田区|福田区]]、[[罗湖区|罗湖区]]、[[盐田区|盐田区]]）行驶。深圳首批共200辆绿色的士于2002年11月19日投入服务<ref name="sina"/>，至2016年底深圳共有绿色的士4,368辆。<ref name="yearbook2017">{{cite journal|author=深圳市史志办公室 |date=2017-12 |title=公共交通运输 |journal=《深圳年鉴2017》 |pages=223 |issn=1005-1996 |accessdate=2018-6-13 }}</ref>首批黄色的士则于1987年11月1日投入服务。<ref>{{cite book |author=《深圳市交通运输志》编纂委员会 |title=深圳市交通运输志 |publisher=方志出版社 |isbn=7801226992 |page=197 |accessdate=2019-1-14}}</ref>

的士实行分区域经营旨在通过限制的士经营范围的手段，改善特区内外因发展水平不同导致的的士服务资源不平衡情况，并借助较低运价减轻特区外的士乘客出行负担。但随着特区内外一体化，再加上2010年深圳经济特区范围扩大至全市，分区域经营模式逐渐失去意义，而在该模式下提供服务的绿色的士更因收费偏低、行驶范围有限，导致绿色的士行业亏损严重，营运效益偏低，滋生长途拒载、不打表等违法行为。为推进关内外的士一体化，2015年，深圳开始拟定方案允许绿色的士于晚上10点到次日早上6点之间，载客进入原特区内，但下客之后不得搭载起点和终点都在原特区内的乘客。此外自2017年5月5日起，深圳红绿的士统一运价。<ref name="sina"/><ref name='wenweipo20170504'/><ref>{{cite web |author1=深圳市交通运输委员会 |title=深圳“红绿的”5日起统一收费（图） |url=http://www.sz.gov.cn/cn/xxgk/zfxxgj/bmdt/201705/t20170505_6582266.htm |website=深圳政府在线 |accessdate=2019-1-14}}</ref><ref>{{cite news |title=深圳绿的或可全市运营 |url=http://sz.people.com.cn/n/2015/1213/c202846-27304153.html |accessdate=2019-1-15 |work=深圳晚报 |publisher=人民网 |date=2015-12-13}}</ref>

深圳黄色的士最迟于2009年底终止服务<ref group="注释">虽然《[http://jtys.sz.gov.cn/jtzx/wycx/czcx/zc/200906/t20090630_5319369.htm 深圳经济特区出租小汽车管理条例] {{Wayback|url=http://www.sztb.gov.cn/xxgk/xxgkml/zcfg/dlys_122196/201711/t20171112_9630412.htm |date=20171215111632 }}》有相关条例承认黄色的士的存在（第六章第三十九条第七款及第七章第五十六条第八款），但是自2009年年底起，深圳已没有黄色的士经营。（{{cite journal|author=深圳市史志办公室 |date=2010-10 |title=城市公共交通-出租小汽车 |journal=《深圳年鉴2010》 |pages=237 |issn=1005-1996 |accessdate=2016-11-20 }}）</ref>。2018年12月20日，深圳市福骏通公司的最后一批共141辆绿色的士全部退出市场，自此深圳绿色的士正式终止服务。<ref>{{cite news |title=时间都去哪儿了？让红的绿的都变成了蓝的…… |url=http://www.sznews.com/news/content/2018-12/28/content_21312558.htm |accessdate=2018-1-14 |work=晶报 |publisher=深圳新闻网 |date=2018-12-28}}</ref>

目前深圳已不再实行分区域经营，所有的士均可行驶深圳全境。

*体积超过0.2立方米、重量超过20公斤的大件行李，每件加收0.5元。
*提供出租车电召服务的，每车次加收2元。
*出租车经过合法批准收费的路段或设施（包括过桥、过路、过渡、过隧道等）的费用由乘客支付。
*深圳实施出租车油价运价联动机制，规定出租车服务根据现行汽油油价加收'''燃油附加费'''，具体细则如下：<ref name='sztb20170505'>{{Cite web |url = http://www.sztb.gov.cn/jtzx/wycx/czcx/cxtx/201705/t20170505_6676264.htm |title = 深圳“红绿的”5日起统一收费（图） |author = 深圳市交通运输委员会 |date = 2017-05-05 |accessdate = 2017-05-14 }}</ref>
**'''燃油出租车'''：以油价每变动0.87元/升为一联动区间，基点油价为5.21元/升，即：
***油价进入5.65元/升至6.52元/升区间时，每车次向乘客加收1元燃油附加费；
***油价进入6.53元/升至7.40元/升区间时，每车次向乘客加收2元燃油附加费；
***油价进入7.41元/升至8.82元/升区间时，每车次向乘客加收3元燃油附加费。以此类推。
**'''电动出租车'''：不收取燃油附加费。
深圳市自2017年5月5日起执行统一的巡游出租车运价，系首次实现“同城同价”。<ref name='wenweipo20170504'>{{Cite web |url = http://news.wenweipo.com/2017/05/04/IN1705040014.htm |title = 深圳「紅綠的士」首次同城同價 |author = 郭若溪 | publisher=[[文汇报|文汇报]] |date = 2017-05-05 |accessdate = 2017-05-14 }}</ref> 该次调价预计需要3个月时间调整计价器，期间可能出现“不同车不同价”的情况。此前，深圳市的红色出租车和绿色出租车在收费上有较大区别，相较而言，红色出租车收费更高，主要体现在起步价、候时费和燃油附加费上。统一运价之前，红色出租车起步价格为10元（2公里），绿色出租车为6元（1.5公里）；红色出租车候时费为0.8/分钟而绿色出租车为0.5元/分钟；而燃油附加费则是绿色出租车的调价区间更大。
