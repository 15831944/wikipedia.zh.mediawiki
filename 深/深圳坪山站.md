'''深圳坪山站'''（原称'''深圳东站'''），位於[[中國|中國]][[廣東省|廣東省]][[深圳市|深圳市]][[坪山區|坪山區]]，[[丹梓西路|丹梓西路]]、[[创业路|创业路]]、[[站前路_(深圳)|站前路]]及[[宝龙路|宝龙路]]之間，是[[杭福深铁路|杭福深铁路]]厦深段的一個車站。<ref>{{cite web |url=http://www.szplan.gov.cn/main/zsgg/200707090211049.shtml |title=存档副本 |accessdate=2007-10-23 |0=2007-10-13 |deadurl=yes |archiveurl=https://web.archive.org/web/20071013040445/http://www.szplan.gov.cn/main/zsgg/200707090211049.shtml |archivedate=2007-10-13 }} </ref>。與此同時，深圳坪山站亦為深圳的六個計劃中的大型交通樞紐之一，其餘五個分別是[[深圳東站|深圳東站]]、[[深圳西站|深圳西站]]、[[深圳北站|深圳北站]]、[[福田站|福田站]]和深圳机场站<ref>{{citeweb|url=http://paper.wenweipo.com/2007/08/12/YO0708120007.htm|title=廣深港客運專線 深圳段年內動工|date=2007-08-12|accessdate=2014-04-05}}</ref><ref>[http://jt.sz.bendibao.com/news/20071010/49975.htm 本地宝-深圳：完成市区内六大交通枢纽的规划设计]</ref>。

==車站結構==

'''深圳坪山站'''站台及站綫股道建築在高架站場，站台分佈成兩個島式、兩個側式，共4個站台提供6個站台面，設到發線6條、直通綫2條。車站房依存在高架站場下的地面，只設一層，售票廳、候車大廳及出站口均設在這一層，並設有多道扶梯及電梯連接站台。

==车站命名==
本站原称深圳东站，由于此名称改由[[深圳东站|布吉站]]使用，故深圳市政府有意将此站更名为深圳新城站。最终铁路部门定名为'''深圳坪山站'''。而在坪山區的道路指示牌及公交系統中，稱本車站為'''坪山火車站'''。

==运营==
根据2017年1月5日起执行的列车时刻表，深圳坪山站每天有70班列车（35对列车）停靠。

2015年9月1日起，深圳坪山站开通至[[深圳北站|深圳北站]]的坪山快捷线通勤动车组，车次为C6801-C6824次，共12对，可使用[[中铁银通卡|中铁银通卡]]<ref name="记事">{{cite web |title=公司记事 |url = https://www.zhongtieyintong.com/mainpage_gongsijishi.page |publisher=中铁银通支付有限公司 |date= |accessdate=2016年3月30日 }}</ref>。但自2017年1月5日起，该列车的开行数量减至5对<ref>{{cite news|title=坪山到福田只要不到45分钟|url=http://news.ycwb.com/2017-01/03/content_23931638.htm|agency=深圳新闻网|date=2017-01-03}}</ref>。
