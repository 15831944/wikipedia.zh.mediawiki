'''默恩达尔'''（{{lang-sv|Mölndal}}）是[[瑞典|瑞典]]西海岸[[大哥德堡|大哥德堡]]的一部分，是[[默恩达尔市|默恩达尔市]]的行政中心。该市60,000居民中约有40,000人生活在默恩达尔本地。

== 地理 ==
默恩达尔位于[[哥德堡|哥德堡]]和[[马尔默|马尔默]]之间的西部干线铁路上，欧洲高速公路[[欧洲E6公路|E6]]/[[欧洲E20公路|E20]]也从北到南贯穿该地区。[[哥德堡電車|哥德堡電車]]系统服务于默恩达尔。

== 历史 ==
[[File:Molndal_1870.jpg|thumb]]
这个城市的名字{{lang|sv|Mölndal}}来源于两个单词：{{lang|sv|''Möln''}}是{{lang|sv|Möllor}}的缩写，Möllor是“[[研磨|磨坊]]”的旧词，dal是[[瑞典語|瑞典語]]的“[[谷地|谷]]”字。 Mölndal是“磨坊谷”。Kvarnbyn狭窄而又高又长的[[瀑布|瀑布]]给所有的[[水磨|水磨]]提供了必要的动力，与山上的风车一起诞生了默恩达尔的早期[[工業化|工業化]]。
[[File:Mölndals_Kvarnby,_den_11_sept_2006,_bild_4.JPG|thumb]]

== 工业 ==
[[File:Aerial_photo_of_Gothenburg_2013-10-27_192.jpg|thumb]]位于默恩达尔的全球研究中心]]
默恩达尔以生命科学公司的高度集中而闻名。[[阿斯利康制药|阿斯利康制药]]的全球研究中心有超过3,100名员工。另外还有其他几家公司，如[[藥品|制药]]、[[生物医学|生物医学]]和[[生物技术|生物技术]]等。靠近[[哥德堡大学|哥德堡大学]]和[[查爾摩斯工學院|查爾摩斯工學院]]的技术园区，支持了其他领域的发展，如[[微波|微波]]技术和[[信息技术|信息技术]]。两个国家研究机构，IFP SICOMP AB和IVF工业研究开发公司，也位于默恩达尔。
