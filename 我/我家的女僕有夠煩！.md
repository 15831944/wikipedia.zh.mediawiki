{{日本作品|我家的女僕有夠煩！|うちのメイドがウザすぎる!}}是由日本女性漫画家{{tsl|ja|中村カンコ|中村官户}}创作的一部漫画，连载于[[双叶社|双叶社]]旗下漫画杂志《月刊Action》，现已发行单行本3册。本作的前两话最初以《'''燕熊！'''》（{{lang-ja|つばくま!}}）为标题刊载于[[新书馆|新书馆]]发行的[[百合_(藝術類型)|百合]][[漫画选集|漫画选集]]《{{lang|ja|ひらり、}}》第12期和第13期<ref>{{cite web|url=http://www.shinshokan.com/hirari/hirari12/index.html|script-title=ja:「ひらり、」vol.12|publisher=[[新书馆|新书馆]]|accessdate=2018-04-11|language=ja}}</ref><ref>{{cite web|url=http://www.shinshokan.com/hirari/hirari13/index.html|script-title=ja:「ひらり、」vol.13|publisher=新书馆|accessdate=2018-04-11|language=ja}}</ref><ref>{{cite web|url=https://www.pixiv.net/member_illust.php?mode=medium&illust_id=58643154|author=中村官户|script-title=ja:新連載 / 月刊アクション10月号|publisher=[[pixiv|pixiv]]|date=2016-08-25|accessdate=2018-04-11|language=ja}}</ref>。

2018年4月11日，本作宣布动画化，于2018年10月5日起[[AT-X|AT-X]]、[[東京都會電視台|東京都會電視台]]等電視台播出。<ref>{{Cite web|url=http://uzamaid.com/onair.html|title=ON AIR|work=電視動畫「我家的女僕有夠煩！」官方網站|accessdate=2018年12月9日|language=ja}}</ref><ref name="acgdoge">{{cite web|url=http://www.acgdoge.net/archives/26182|title= 混血幼女主人与萝莉控女仆，《我家女仆超级烦！》TV动画2018年10月播出|date=2018-04-11|accessdate=2018-04-11|publisher=ACGdoge|language=zh-cn}}</ref>

== 故事简介 ==
有着[[俄罗斯|俄罗斯]]血统的小学二年级学生高梨米夏，因母亲早早去世而与父亲共同生活。米夏不喜欢父亲雇佣[[女佣|女佣]]，总是想方设法进行[[恶作剧|恶作剧]]迫使女佣们离开。直到有一天，一个戴着眼罩的女仆鸭居燕来到米夏的家，使得米夏无法施展她的恶作剧，更糟糕的是燕还是个[[萝莉控|萝莉控]]……<ref name="acgdoge"/>

== 登场人物 ==
{{expand-section}}
; {{nihongo|高梨米夏|高梨 ミーシャ（たかなし みーしゃ）|Tashanashi Misha|声：[[白石晴香|白石晴香]]<ref name="character">{{cite web|url=http://uzamaid.com/chara.html|title=CHARACTER|work=電視動畫「我家的女僕有夠煩！」官方網站|accessdate=2018年12月9日|language=ja}}</ref>}}
:小学二年级学生，擁有純正[[俄羅斯|俄羅斯]]血統的美少女。
:母亲（聲：[[井上喜久子|井上喜久子]]）早逝，现在与無血緣的父親康弘一起生活。
:自從母親去世後，便時常把自己關在房間裡面，並將所有來家中工作的女僕都嚇跑，幾乎足不出戶。
:因為金髮碧眼的外表，讓她成為許多同學的焦點，但因為不喜歡這樣的視線，因此拒絕上學。
:燕來了之後，原也盡一切努力想要把她趕跑，卻發現她與以前的女僕完全是不同檔次的人。
:因為燕的一番話以及變態發言，認為與其待在家裡在學校可能還好一點，因此決定繼續上學。
:在解決班上倉鼠過多的事件後，被同學稱讚，自己也了解到燕那番話是對的。
:在與燕長久的相處後，發現自己需要她的照顧，而不再趕她出去。
:在動畫12集當中，與燕坦誠相待，帶燕走進不曾讓康弘以外的人進入的聖域，也就是她母親的工作室，並相互認為對方是自己的光。
; {{nihongo|鸭居燕|鴨居 つばめ（かもい つばめ）|Kamoi Tsubame|声：[[沼仓爱美|沼仓爱美]]<ref name="character"/>}}
:米夏家的新女仆，原[[航空自卫队|航空自卫队]]一等士官，渾身肌肉，戰鬥能力強，對[[肺活量|肺活量]]很有自信，家事萬能，右眼因病失明，总是戴着有[[蔷薇|蔷薇]]图案的[[眼罩|眼罩]]。
:自称对有过[[初潮|初潮]]的女孩子没有兴趣，本人表示：「因為未經初潮的女孩子不具有生育能力，就像是[[妖精|妖精]]一樣」，是个不折不扣的萝莉控。
:很想跟米夏打好關係，但是做出來的行動時常都很出格。
:在與米夏差不多年紀的時候，父親去世。
:在了解米夏不去上學的原因之後，對她說：「人在關係變得親密之前，只能透過外表來判斷。要讓他人瞭解自己是需要時間的。」說服了米夏去上學（但絕大因素是她的變態發言）。
:希望再見到天上父親的身影，而追隨其腳步，當上自衛官，卻因眼病而[[退伍|退伍]]。
; {{nihongo|高梨康弘|高梨 康弘（たかなし やすひろ）|Takanashi Yasuhiro|聲：[[加藤將之|加藤將之]]<ref name="character"/>}}
:米夏的父親。在米夏的母親去世之後，獨自撫養米夏，但找來的女僕都被米夏趕跑。
:是個溺愛女兒的父親。於漫畫15話（動畫第10話）回想中說明自己是米夏的[[繼父|繼父]]，兩人並無血緣。
; {{nihongo|鹫崎美美香|鷲崎 みみか（わしざき みみか）|Washizaki Mimika|聲：[[原田彩楓|原田彩楓]]<ref name="character"/>}}
:米夏的同班同学，戴着眼镜。
:在学校与米夏一同担任[[仓鼠|仓鼠]]饲养员，在米夏不来上学时，会给她带来課程资料。
; {{nihongo|森川结衣|森川 ゆい（もりかわ ゆい）|Morikawa Yui|聲：[[井澤詩織|井澤詩織]]<ref name="character"/>}}
:米夏的同班同学。自認是個懂得時尚的美少女，但每次向同學尋求認同（其實是炫耀）時，總是因為米夏而顯得相形见绌，常因此向米夏發起莫名其妙的比賽。
:自稱為燕的徒弟。
; {{nihongo|鹈饲綠|鵜飼 みどり（うかい みどり）|Ukai Midori|聲：[[市道真央|M·A·O]]}}
:燕的上司，原航空自卫队二等空尉，超級[[受虐狂|受虐狂]]。
:異常喜歡燕，但因她的特殊體質，每次被拒絕反而感到更興奮。
:因她的特殊體質只吃得下難吃的食物，所以也只煮得出難吃的食物。
:常常想出各種方法讓自己感到痛苦，然後興奮。
; {{nihongo|熊五郎|クマゴロー（くまごろー）|Kumagoro|聲：[[長繩麻理亞|長繩麻理亞]]}}
: 米夏所養的[[雪貂|雪貂]]，能通人話。
:是康弘為了撫慰遭受喪母之痛的米夏而買給她的禮物。
