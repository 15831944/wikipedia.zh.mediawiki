《'''我·放電'''》（{{lang|ja|私と放電}}，''Watashi to Houden／Electric Discharge & Me''）是日本音樂家[[椎名林檎|椎名林檎]]的首張[[精選輯|精選輯]]，同時也是唱片公司為紀念[[椎名林檎|椎名林檎]]出道十週年所發行的首張作品，和音樂MV精選輯《[[我·發電|我．發電]]》一同於2008年7月2日由[[EMI音樂_(日本)|EMI Music Japan]]／[[維京唱片|Virgin Music]]發行<ref name="JPN" />。初回限定版為「特殊紙盒包裝」，並包括十周年紀念演唱會門票抽獎卡與紀念貼紙；而在台灣發行的日本版和台壓版內皆附贈活動卡。

在專輯銷售成績方面，本張專輯在日本Oricon銷售榜上最高位居單週第4名<ref name="WatashitoHoudenOriconWeek" />，名列2008年年度銷售榜第73位<ref name="2008sale" />。在銷售認證上，專輯《我·放電》銷售突破10萬張，在2008年7月通過日本唱片協會認定為金唱片<ref name="RIAJ200807" />。

== 專輯概要 ==
《'''我·放電'''》是椎名林檎繼2007年以「椎名林檎×{{link-ja|齋藤毅 (音樂家)|斎藤ネコ|齋藤毅}}」為名義發行專輯《[[平成風俗|平成風俗]]》後，睽違一年的作品。本張專輯不但是[[椎名林檎|椎名林檎]]的首張[[精選輯|精選輯]]，同時也是唱片公司為慶祝[[椎名林檎|椎名林檎]]出道十週年所發行的作品。本張精選輯收錄椎名林檎自出道以來未曾收錄於專輯內的歌曲{{refnest|group=注|即單曲中B面曲。}}，與非專輯版本的歌曲。在專輯包裝上，則是將精選輯拆成兩張CD，然後再把它裝進同一個專輯包裝內形成一個專輯。

雖然這次是發行精選輯，但在專輯製作上卻遇到很大的困難，因為部分歌曲原先僅有8公分CD版本，必須將歌曲轉換錄製在12公分規格的CD上。對此，椎名林檎曾表示假如轉換後音質過差，她願意重新進行錄製。另外CD 2的第12、13及14首歌為隱藏曲目，在專輯的外包裝上並沒有標示，只在歌詞本上註記歌曲名稱與歌詞。最後，唱片公司為了宣傳本張精選輯特別拍攝的[[音樂錄影帶|音樂錄影帶]]「Mellow」也收錄至同日發售的音樂MV精選輯《我·發電》中。

在專輯發售時程上，新專輯發行的消息最早於2008年5月15日就在官網發佈<ref>{{cite web |url=http://www.vibe-net.com/news/?news=0034631 |title=椎名林檎　デビュー10周年記念企画がスタート！！ |publisher=vibe-net |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20160126164257/http://www.vibe-net.com/news/?news=0034631 |archivedate=2016-01-26 }}</ref>，到了06月9日則開始接受歌曲「Mellow」音樂錄影帶視聽服務<ref>{{cite web |url=http://www.vibe-net.com/news/?news=0034849 |title=椎名林檎　新作ミュージック・クリップが完成！！ |publisher=vibe-net |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20160126230657/http://www.vibe-net.com/news/?news=0034849 |archivedate=2016-01-26 }}</ref>。06月20日則發表椎名林檎將舉行十周年紀念演唱會《[[Ringo_EXPO_08#演唱會簡介|(生)林檎博'08 〜10周年記念祭〜]]》的消息<ref name="RINGO10LIVE1" />。日本於07月2日發行專輯<ref name="JPN" />，台灣則於07月4日當天同步發行進口版與台壓版<ref name="TWN" />。日後，唱片公司將本張專輯收錄至2008年11月25日發行的DVD套裝專輯《[[MoRA|MoRA]]》中，藉以慶祝椎名林檎出道十週年。

最後，在專輯銷售成績方面，專輯在發行當週以8.5萬張的銷售量在日本Oricon銷售榜上位居第4名<ref name="WatashitoHoudenOriconWeek" />，總計在Oriocn銷售榜上榜16週<ref name="WatashitoHoudenOriconConsecutiveWeek" />，累積銷售達15.3萬張，名列2008年年銷售榜第73位<ref name="2008sale" />。同時，專輯銷售突破10萬張，在2008年7月通過日本唱片協會認定為金唱片<ref name="RIAJ200807" />。

== 銷售排行 ==
=== Oricon 銷售榜(日本) ===
《我·放電》於07月2日發行專輯。發行首週，在日本公信榜Oricon的專輯榜2008年6月第5週付的榜單中以85,123張銷售量居單週第四名，同禮拜銷售冠軍為銷售突破37.7萬張的樂團[[GReeeeN|GReeeeN]]的第二張專輯《[[啊，你好。好久不見。|啊，你好。好久不見。]]》<ref name="WatashitoHoudenOriconWeek" />。第二週則以銷售量27,379張居單周第七位，同禮拜銷售冠軍為樂團[[橘子新樂園|橘子新樂園]]的第五張專輯《[[PANIC_FANCY|PANIC FANCY]]》<ref>{{Cite web |url=http://www.oricon.co.jp/rank/ja/w/2008-07-21/ |title=2008年07月第1週の邦楽アルバムランキング情報 |publisher=Oricon |language=ja}}</ref>。總計在日本公信榜Oricon的專輯榜上榜16週<ref name="WatashitoHoudenOriconConsecutiveWeek" />，累積銷售達15.3萬張，名列2008年年銷售榜第73位<ref name="2008sale" />。
