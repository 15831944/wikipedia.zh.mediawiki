《'''我的ID是江南美人'''》（{{lang-ko|'''내 아이디는 강남미인'''}}，{{lang-en|'''Gangnam Beauty'''}}），為[[韓國|韓國]][[JTBC|JTBC]]於2018年7月27日起播出的[[JTBC金土連續劇|金土連續劇]]，由崔成範導演執導與《[[心懷叵測的恢單女|心懷叵測的恢單女]]》[[崔秀英_(編劇)|崔秀英]]作家合作打造。此劇改編自[[Naver_webtoon|Naver webtoon]]的同名網路漫畫，講述從小因為長相醜陋而受欺凌的女主角在整容後，以為能夠過上新生活，卻在進入大學後經歷與夢想中完全不同的挫敗，在過程中找到真愛的故事<ref>[https://www.koreastardaily.com/tc/news/106973 這個月底就看的到啦～JTBC新劇《我的ID是江南美人》首版預告影片公開！]，韓星網，2018-07-01</ref><ref>[https://www.kpopn.com/2018/07/08/%E6%9E%97%E7%A7%80%E9%A6%99%E3%80%81%E8%BB%8A%E9%8A%80%E5%84%AA%E3%80%8A%E6%88%91%E7%9A%84-id-%E6%98%AF%E6%B1%9F%E5%8D%97%E7%BE%8E%E4%BA%BA%E3%80%8B%E6%B5%B7%E5%A0%B1%E3%80%81%E7%AC%AC%E4%BA%8C%E6%B3%A2%E9%A0%90%E5%91%8A%E5%85%AC%E9%96%8B/ 林秀香、車銀優《我的 ID 是江南美人》海報、第二波預告公開]，Kpopn，2018-07-08</ref>。

== 劇名由來 ==
*在韓國[[首爾|首爾]][[江南區_(首爾)|江南區]]有一條兩旁都是專門進行整形手術之醫美診所的知名街道，想要整容之女性都會前往該處，也因此整容後的女性被稱為「江南美人」。
==漫畫版由來==
*本劇改編自網路漫畫，而原本漫畫版的[[作者|作者]][[淇萌琪|淇萌琪]]在[[大學|大學]]的某堂課中，必須依[[教授|教授]]給的[[關鍵字|關鍵字]]來創造[[角色|角色]]的課程。而某日教授給的關鍵字為「[[江南美人|江南美人]]」，但後面的副標題居然是「[[大醬女|大醬女]]」<ref group=註>[[韓國|韓國]]最近的流行語，用來[[諷刺|諷刺]]長相不好卻又愛慕虛榮的[[女人|女人]]。</ref>。

*作者不服教授將[[整形|整形]]與愛慕虛榮兩者相提並論而創作漫畫《[[我的ID是江南美人|我的ID是江南美人]]》<ref group=註>在韓文中，原網路漫畫標題中的ID是以「英文」撰寫，但電視劇則改以「韓文」的ID（{{lang-ko|아이디}}）來作為標題。</ref>。
