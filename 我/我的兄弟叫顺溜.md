《'''我的兄弟叫顺溜'''》，2009年[[中国大陆|中国大陆]]的一部[[抗日战争|抗日战争]]题材[[电视剧|电视剧]]，由[[花箐|花箐]]执导，[[张国强_(中国大陆)|张国强]]和[[王宝强|王宝强]]等主演。该剧是花箐执导的第一部电视剧，也是张国强和王宝强继《[[士兵突击|士兵突击]]》后的再度联手。本剧讲述顺溜如何从一位农家子弟成为一名[[新四军|新四军]][[狙击手|狙击手]]。本片创2009年央视收视最高纪录。

== 制作人员 ==
*导演：[[花箐|花箐]]

*[[王宝强|王宝强]]饰陈二雷（原名陳順溜）
*[[孙逊_(演员)|孙　逊]]饰三营长

*[[郝蕾|郝　蕾]]饰顺溜姐姐（在最後順溜狙擊日軍司令的橋段，被日軍坂田強暴，不堪淩辱自殺）（友情客串）

== 剧情简介 ==
顺溜是猎户的孩子,逐渐成长为一个神枪手的故事。

== 批评 ==
本片中的许多军事方面的错误受到批评。如见《兵器》2009年09期相关文章。
