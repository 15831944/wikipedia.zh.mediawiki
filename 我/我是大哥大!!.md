{{日本作品|我是大哥大!!{{Refnest|group="注"|[[長鴻出版社|長鴻出版社]]取得第一手代理的中文譯名，日語原名直譯為「'''今天開始我要!!'''」<ref name="kknewsr3">這部日劇，竟然9.1分. 每日頭條. 2018-11-29 [2018-12-06] {{zh-tw icon}}</ref>。}}|今日から俺は!!}}為[[日本漫畫家|日本漫畫家]][[西森博之|西森博之]]所創作的[[日本漫畫|日本漫畫]]作品。簡稱「'''今日俺'''」<ref>{{cite web|url=http://websunday.net/museum/no04/no04in.html|title=《我是大哥大!!》Sunday名作博物館|work=[[週刊少年Sunday|週刊少年Sunday]]|publisher=[[小學館|小學館]]|accessdate=2018-07-26|language=ja}}</ref>。

本作品是西森博之在[[小學館|小學館]]Sunday雜誌發表的第一部連載。一開始是在《{{link-ja|週刊少年SundayS|週刊少年サンデーS|增刊少年Sunday}}》1988年9號至1990年8號連載，後轉移至《[[週刊少年Sunday|週刊少年Sunday]]》1990年40號繼續，至1997年47號完結，[[單行本|單行本]]共38冊。曾改編成[[OVA|OVA]][[動畫|動畫]]、真人[[錄影帶|錄影帶]]，2018年10月至12月播出真人電視劇。

單行本銷售量累計至今超過4000萬冊<ref>來自「Otona Fami 2012 4月號 『史上最大 西森博之HISTORY』」。</ref>。2018年10月12日，於電視劇播映期間，將於同年11月24日發售的《週刊少年SundayS》2019年1月號連載「我是大哥大!! 特別篇」<ref name="natalie303405">{{Cite news|script-title=ja:「今日から俺は!!」特別編の連載が決定！11月発売の少年サンデーSにて|url=https://natalie.mu/comic/news/303405|newspaper=[[Natalie_(網站)|Comic Natalie]]|publisher=株式會社Natasha|date=2018-10-12|accessdate=2018-10-12|language=ja}}</ref>，同年11月宣布開始連載續篇<ref name="natalie309319">{{Cite news|script-title=ja:「今日から俺は!!」続編が開幕、高校卒業から25年経ったサガワが主役に|url=https://natalie.mu/comic/news/309319|newspaper=Comic Natalie|publisher=株式會社Natasha|date=2018-11-24|accessdate=2018-11-24|language=ja}}</ref>。

== 登場人物 ==
=== 私立軟葉高中 ===
; {{nihongo|三橋貴志|三橋 貴志（みつはし たかし）}}
: {{配音2|JP|[[松本保典|松本保典]]}}
: 本作男主角。身高（自稱）181公分。高一時以轉學為契機染了金髮並成為不良少年。
: 人稱「金髮惡魔」，為人奸滑狡詐，主張只要能贏則可不擇手段。打架時多使用陰險的手段，吃了虧必將加倍奉還。
: 醉酒後會變得十分溫順，一旦被攻擊則會比平時還兇暴。輕狂不羈的性格下有著怕鬼等膽小的一面。

; {{nihongo|伊藤真司|伊藤 真司（いとう しんじ）}}
: {{配音2|JP|[[堀秀行|堀秀行]]}}
: 本作另一男主角。身高180公分（計入高聳的頭髮約有205公分）左右。正直而剛毅，髮型帶刺的不良少年。與三橋同以轉學為契機成為不良；為人老實、富有正義感，深受校內同儕的青睞。
: 光明磊落的個性使其打架時不免吃虧，卻也因此頗受他人尊崇與賞識。
: 家境富裕，興趣是培養水族箱，喜歡說冷笑話。

; {{nihongo|赤坂理子|赤坂 理子（あかさか りこ）}}
: {{配音2|JP|{{link-ja|草地章江|草地章江}}}}
: 本作女主角。赤坂道場的獨生女，擅長合氣道的嬌小少女。三橋為其解圍後轉至軟高，對三橋抱有好感，對方卻屢屢不能察覺。
: 稱三橋為「小三三（{{lang|ja|さんちゃん}}）」，校內少數能壓制三橋並與之作對的存在，被不少人視作軟高真正的老大。

; {{nihongo|田中良|田中 良（たなか りょう）}}
: {{配音2|JP|[[菊池正美|菊池正美]]}}
: 赤坂道場的門徒。對理子抱有好感。

; {{nihongo|高崎秀一|高崎 秀一（たかさき しゅういち）}}
: 剛正不阿的熱血少年。在校內坐擁不亞於伊藤的人氣。
: 經常撞見伊藤和京子的放閃場面。

; {{nihongo|榊原雄一|榊原 雄一（さかきばら ゆういち）}}
: 三橋等人的學弟，號稱「國中版三橋」。以卑鄙而言有著足以媲美三橋的才能，被三橋等人教訓後折服的幼苗。畢業後進入軟高，被三橋視作自己的小弟。

=== 青蘭女子高中 ===
; {{nihongo|早川京子|早川 京子（はやかわ きょうこ）}}
: {{配音2|JP|[[折笠愛|折笠愛]]}}
: 本作第二女主角。原不良少女兼青蘭老大，為伊藤所救後與之相戀，長相清秀的氣質美女。平時溫柔婉約，生起氣來十分有魄力，連伊藤見了都為之所懼。

=== 紅羽高中 ===
; {{nihongo|今井勝俊|今井 勝俊（いまい かつとし）}}
: {{配音2|JP|[[屋良有作|屋良有作]]}}
: 紅高的老大。外貌挺拔俊俏，頭腦簡單四肢發達的不良少年。

; {{nihongo|谷川安夫|谷川 安夫（たにがわ やすお）}}
: {{配音2|JP|[[矢尾一樹|矢尾一樹]]}}
: 今井的友人兼跟班。身形矮小，同情今井悲慘待遇的同時十分欣賞他耿直的處事作風。

; {{nihongo|中野誠|中野 誠（なかの まこと）}}
: {{配音2|JP|[[檜山修之|檜山修之]]}}
: 在京都與三橋等人結下樑子後轉至紅高的不良少年。性格冷淡，意外地有熱血的一面。

=== 開久高中 ===
; {{nihongo|片桐智司|片桐 智司（かたぎり さとし）}}
: {{配音2|JP|[[大塚明夫|大塚明夫]]}}
: 開久的老大。力量與恐怖兼備的強者。
: 畢業後經營[[章魚燒|章魚燒]]攤販。

; {{nihongo|相良猛|相良 猛（さがら たけし）}}
: {{配音2|JP|[[辻谷耕史|辻谷耕史]]}}
: 人稱「狂犬相良」，開久的No.2。

=== 其他 ===
; {{nihongo|森川涼子|森川 涼子（もりかわ りょうこ）}}
: {{配音2|JP|[[市原由美|黑田由美]]}}
: 經常被誤認為大姐大的劍道少女。對今井抱有好感，卻因不坦率的性格和笨拙的表達方式難以傳至今井的心底。
