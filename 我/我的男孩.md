《'''我的男孩'''》（{{lang-en|''My Dear Boy ''}}），2017年[[台灣|台灣]][[偶像劇|偶像劇]]。由[[林心如|林心如]]、[[張軒睿|張軒睿]]、[[李李仁|李李仁]]、[[高聖遠|高聖遠]]領銜主演，[[林心如|林心如]]同時擔當本劇的製作人。<ref>王麗娟.[http://www.nownews.com/n/2017/03/09/2434495 林心如產後處女秀　搭檔張軒睿談情].NOWnews今日新聞.2017-03-09</ref><ref>趙大智.[http://www.appledaily.com.tw/realtimenews/article/new/20170309/1072630/ 林心如復出首接戲　配對狼王子].蘋果日報.2017-03-09</ref>於2017年5月31日開鏡記者會，10月20日正式殺青<ref>趙大智.[http://www.appledaily.com.tw/realtimenews/article/entertainment/20171020/1225952/papersec_right 林心如《我的男孩》殺青　最爽睡到自然醒].蘋果日報.2017-10-20 </ref>。[[臺灣電視台|台視]]於12月22日晚間10點首播，[[八大戲劇台|八大戲劇台]]則於隔日晚間8點播出，[[臺灣電視台|台視]]於2018年5月11日晚間10點播出最終回，劇末播出幕後花絮《我的男孩3分熟》。而本劇獲得[[中華民國文化部|中華民國文化部]]105年度電視節目製作補助案連續劇類補助2,000萬元新臺幣、5.1聲道補助金額120萬元新臺幣。<ref>[http://mocfile.moc.gov.tw/files/201607/67c83cb4-2ff8-4b67-ad09-2dcb1b1fe079.pdf 中華民國文化部105年度電視節目製作補助案獲補助名單]</ref><ref>林奕如.[http://star.ettoday.net/news/739321 林心如新戲補助金2千萬惹議！引爆討論潮八大回應].ETtoday.2016-07-20</ref>全劇平均收視率0.79。

[[中國大陸|中國大陸]]於2018年11月宣佈將《我的男孩》翻拍，定名為《[[陽光先生_(中國電視劇)|陽光先生]]》，預計於2019年6月開拍，拍攝45集<ref>[https://stars.udn.com/star/story/10091/3471883 林心如「我的男孩」陸翻拍 新戲名讓網友傻眼]</ref>。

== 劇情大綱 ==
講述在愛情裡受傷的羅小菲（[[林心如|林心如]]飾），在遇見相差10歲的男孩安慶輝（[[張軒睿|張軒睿]]飾）後，共同思索生命中的寬恕與感恩，並重拾愛情的勇氣，以及做自己的真諦。

== 角色介紹 ==
=== 主要角色 ===
*; 羅小菲（[[林心如|林心如]]飾演）
: 30歲，為一名[[導演|導演]]。從小與姐姐相依為命，因此姐妹倆的感情至深。曾是個廣告公司的助理，並在那時遇見身為導演的蕭也時，兩人隨即墜入熱戀，但這段戀情卻是見不得人的地下情。當她終於試著放下這段難以割捨的舊情時，遇見了小她10歲的男孩－－安慶輝。
*; 安慶輝（[[張軒睿|張軒睿]]飾演）
: 20歲，出身自[[五年專科學校|五專]]視覺傳達科的平凡男孩。暗戀學妹潘妍婷多年，卻苦無一親芳澤的機會，在他終於鼓起勇氣追愛時，遇見了舉止怪異的阿姨姐姐－－羅小菲。
*; 賴建國（[[李李仁|李李仁]]飾演）
: 綽號「小賴」，個性老實的公務員，喜歡照顧弱勢族群，同時也是名[[國際標準舞|國標舞]]舞者。透過同事小珊的介紹，與其妹妹小菲相親，此後便被小菲的個人特質吸引。
*; 蕭也時（[[高聖遠|高聖遠]]飾演、[[吳文民|吳文民]]配音）
: 廣告公司老闆，是個極具魅力的男性。擁有著對藝術的抱負，並透過女友美雙的幫助得到今日的成就，但他渴望由如知己般的小菲填補其內心的空虛。

=== 其他演員 ===
*; 何美玥（[[嚴藝文|嚴藝文]]飾演）
: 安家媽媽，安親班園長，個性強勢又愛嘮叨。迷失身為女性之自我價值的她，雖然對婚姻有諸多不滿，但仍為家庭犧牲奉獻多年。
*; 安立民（[[屈中恆|屈中恆]]飾演）
: 安家爸爸，安親班校車司機。個性溫順、處處忍讓，雖然不擅於表達，但他相當珍視妻子。
*; 潘妍婷（[[温貞菱|温貞菱]]飾演）
: 綽號「洗髮精」，尼古嚕的直屬學妹。在亮麗的外表下，其實常因外界的評論而質疑自我。
*; 黃厚喻（[[許光漢|許光漢]]飾演）
: 綽號「尼古嚕」，對表演藝術極具熱忱與抱負，是慶輝與搞屁的摯友。
*; 許輔軍（[[章廣辰|章廣辰]]飾演）
: 綽號「搞屁」，家境富裕，是慶輝與尼古嚕充滿義氣的摯友。
*; 李大寶（[[龐庸之|龐庸之]]飾演）
: [[攝影師|攝影師]]出身，小菲的後輩、慶輝的前輩。曾經同小菲與也時打拼，雖然片場工作艱辛，但仍願貢獻身軀。
*; 羅小珊（[[張本渝|張本渝]]飾演）
: 小菲的姊姊，為一名公務員。不論外在條件，選擇嫁給身為聽損者的丈夫，因此，也對所愛之人極有保護慾。
*; 政誠（[[東明相|東明相]]飾演）
: 小珊的丈夫，為一名[[聽覺障礙|聽損者]]。先天的障礙並未影響他的樂觀，並努力證明自我的能力。
*; Toro（[[陳秉立|阿翔]]飾演）
: 美玥的弟弟，自我感覺良好而無定性，總是讓美玥操心。
*; 安柏輝（[[羅明順|羅明順]]飾演）
: 安家的小兒子，在父母期盼生女下出世，其擁有的個人特質令家人有些憂慮。
*; 皮蛋（[[周洺甫|周洺甫]]飾演）
: 個性衝動，是小賴於輔育院認輔的少年。透過小賴的引介，成為小珊與政誠的水餃事業員工。
*; Amy（[[宇宙_(A'N'D)|A'N'D宇宙]]飾演）
: 個性潑辣的飲料店員，與慶輝相識，喜歡皮蛋而努力追求他。
*; 張偉捷（[[李銓|李銓]]飾演）
: 五專的風雲人物，籃球隊隊長，喜歡妍婷。
*; 喵喵（[[安哲|安哲]]飾演）
: 個性鮮明的美術總監，安慶輝的師傅，相當崇拜視為女神的小菲。
*; 蕭念蒔（[[葛璦|葛璦]]飾演）
: 綽號「圈圈」，小菲與也時的女兒，幼時由小珊和政誠扶養。

=== 特別演出 ===
*; 柯美雙（[[楊謹華|楊謹華]]飾演）
: 知名女演員，也時的伴侶。長期扶持也時追夢，因此對他懷有執著的愛，並選擇對他的風流事蹟睜一隻眼、閉一隻眼。

=== 客串演員 ===
*; [[殷悅|Melody]]
: 拍攝吸塵器廣告的女明星（第1集）
*; [[謝瓊煖|謝瓊煖]]
: 與美玥不相識的公車乘客，卻成為美玥傾訴的對象。（第2集）
*; [[廖怡裬|廖怡裬]]
: 安慶輝在學校的導師。（第3集）
*; 小郭（[[陳家逵|陳家逵]]飾演）
: 安親班學生家長，有目的性地接近美玥。（第6集）
*; [[謝佳見|謝佳見]]
: 耍大牌的知名演員，而後與小菲化解嫌隙，相互尊重。（第8－9集）
*; 瑋瑋（[[海裕芬|海裕芬]]飾演）
: Toro的女朋友（第11集）
*; [[禾語辰|禾語辰]]
: 大寶的妻子，個性強勢、剽悍。（第10－11集）
*; [[許瑋甯|許瑋甯]]
: 小菲接案之客戶的秘書，對小菲的態度相當不善。（第19集）
*; [[梁舒涵|梁舒涵]]
: 安慶輝在喵喵美術工作室的同事。（第1集、第12－13、15－17集）

== 播出 ==
本劇在2017年12月22日至2018年5月11日間，於每周五晚間10點在[[臺灣電視台|台視主頻]][[台視周五優質戲劇|週五優質戲劇]]時段首播90分鐘（含廣告），映後均會播放3分鐘幕後特輯《我的男孩3分熟》，[[八大戲劇台|八大戲劇台]]則於隔日（周六）晚間8點聯播。購買播出版權的[[新媒體|新媒體]]有[[LINE_TV|LINE TV]]、[[愛奇藝|愛奇藝台灣站]]、[[CHOCO_TV|CHOCO TV]]、[[KKTV|KKTV]]與[[Litv|LiTV]]等。<ref>{{cite web|url=https://www.gtv.com.tw/entertainmentNews.php?seqId=C0D14D6A-8290-305C-D101-9E716222BB86|title=片頭大首播 林心如情纏三男|publisher=GTV八大電視|date=2017-11-11|accessdate=2017-12-21}}</ref>

全球海外播映版權由{{le|Viki (網站)|Viki (website)|Viki}}取得，僅[[英國|英國]]、[[中東|中東]]、[[新西蘭|紐西蘭]]、[[澳門|澳門]]與[[印尼|印尼]]等地除外。[[香港|香港]]地區由[[Viu|Viu]]線上播出，[[now_TV|now TV]]則於自選服務中推出。[[新加坡|新加坡]]透過[[星和視界|星和視界]]旗下的[[HUB娛家戲劇台|HUB娛家戲劇台]]與[[流媒體|串流媒體]]StarHub Go播出。[[日本|日本]]譯名為《{{lang|ja|年下のオトコ～My Dear Boy～}}》（年下的男子～My Dear Boy～），由Asia Dramatic TV與C-POP TV播映。<ref>{{cite web|url=https://www.facebook.com/mydearboyGTV/posts/398001940679819|title=超緊急~超重要的通知!!!!!!|work=[[Facebook|Facebook]]|author=我的男孩|date=2018-05-17|accessdate=2018-06-01}}</ref>
