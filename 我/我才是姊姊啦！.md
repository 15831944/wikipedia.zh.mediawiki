'''我才是姊姊啦！'''（{{lang|ja|わたしがおねえちゃんなんだからね}}）是日本漫畫家[[鰻丸|鰻丸]]的作品。於[[ウェブコミック誌|ウェブコミック誌]]「[[FlexComixブラッド|FlexComixブラッド]]」（[[FlexComix|FlexComix]]）上進行連載，單行本全2卷。

== 登場人物 ==
; 高槻 美空（{{lang|ja|たかつき みそら}}）
: 高中一年級。個子矮小，不喜歡別人說他是幼兒體型。
; 高槻 小鳥（{{lang|ja|たかつき ことり}}）
: 小學五年級。美空的妹妹，身高比姊姊高出許多，擁有[[巨乳|巨乳]]，看不出是小學生。
; 藤岡 君子（{{lang|ja|ふじおか きみこ}}）
: 美空的同班同學，喜歡美空嬌小可愛的樣子。
; 日高 睦月（{{lang|ja|ひだか むつき}}）
: 美空的同班同學，帶著眼鏡，喜歡美空嬌小可愛的樣子。
; 那須野 舞（{{lang|ja|なすの まい}}）
: 美空的同班同學，巨乳，喜歡美空嬌小可愛的樣子。
