'''我不配'''（'''距离'''）是[[周杰伦|周杰伦]]作曲，[[方文山|方文山]]作词的中文流行歌曲，收录于专辑《[[我很忙|我很忙]]》<ref>{{Cite web|url = http://ent.qq.com/a/20141030/052097.htm|title = 大耳机：岁月不仅是把杀猪刀，岁月也是周杰伦|accessdate = |author = |date = 2014-10-30|publisher = 腾讯娱乐}}</ref>，这首歌被认为是周杰伦为前女友[[侯佩岑|侯佩岑]]而写，而周杰伦否认<ref>{{Cite web|url =http://ent.news.cn/2011-10/31/c_122217125.htm|title =周杰伦新歌“MineMine”歌词曝光 疑为蔡依林所写|accessdate =|author =中新网|date =2011-10-31|publisher =新华网}}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>；而方文山回应：歌曲并非是以周杰伦为主角而写，只是因为歌曲适合[[押韵|压“ei”韵]]<ref>{{Cite web|url = http://www.chinanews.com/yl/2014/04-18/6080734.shtml|title = 《我不配》被疑描述周杰伦分手心情 方文山澄清|accessdate = |author = |date = 2014-04-18|publisher = 中国新闻网}}</ref>，侯佩岑公布婚讯后，周杰伦的演唱会曲目将《我不配》取消，替换为《[[说好的幸福呢|说好的幸福呢]]》<ref>{{Cite web|url = http://star.pclady.com.cn/data/1506/1323121.html|title = 侯佩岑周杰伦分手原因竟是这样|accessdate = |author = |date = 2015-07-01|publisher = 太平洋女性网}}</ref>。
