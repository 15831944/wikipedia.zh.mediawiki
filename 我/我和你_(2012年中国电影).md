《'''我和你'''》（{{lang-en|'''You and Me'''}}）导演[[扈耀之|扈耀之]]，主演[[张信哲|张信哲]]、[[苏慧伦|苏慧伦]]、[[英达|英达]]、[[王劲松|王劲松]]、[[刘金山|刘金山]]。

== 剧情 ==
该片讲述“鸣先生”为了完成母亲的心愿，找来一群乌七八糟的人组建了一个“合唱团”的故事。

== 花絮 ==
该片在2009年就杀青了，并且参加了2010年的[[第二届澳门国际电影节|第二届澳门国际电影节]]，但是不知道什么原因一直推迟到2013年才上映。<ref>{{cite web |url=http://ent.qq.com/a/20090827/000361.htm |title= 张信哲搭档内地笑星 电影《我和你》瞄准贺岁档 |accessdate=2009 |publisher=腾讯网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://gb.cri.cn/27564/2009/07/02/2445s2551858.htm |title=电影《我和你》即将杀青 张信哲出演跑调歌手 |accessdate=2009 |publisher=国际在线 |work= |language=zh-hans}}</ref>

该片2013年在[[北京市|北京市]]首映。<ref>{{cite web |url=http://tv.people.com.cn/GB/79889/17401176.html |title=影片《我和你》在京首映 张信哲苏慧伦颠覆形象 |accessdate=2012 |publisher=人民网 |work= |language=zh-hans}}</ref>

该片是[[喜剧片|喜剧片]]。<ref>{{cite web |url=http://www.m1905.com/news/20120316/503618.shtml |title=张信哲首演喜剧《我和你》 情歌王子曝唱歌跑调 |accessdate=2012 |publisher= |work= |language=zh-hans}}</ref>
