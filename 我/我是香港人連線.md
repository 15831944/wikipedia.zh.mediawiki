'''我是香港人連線'''（{{lang-en|'''Hongkonger Front'''}}），是一個內容為主張[[香港獨立|香港獨立]]、倡議獨立意識的網頁，成立於2004年11月，認為「全民投票、獨立建國」是香港最終的理想。該網頁亦提倡香港應仿傚全球各地的[[反殖民|反殖民化]]運動，舉行[[全民公決|全民公決]]決定香港的未來<ref>{{cite web|url=http://www.hkfront.org/request-ch.htm|title=我們的未來}}</ref>。我是香港人連線的關係伙伴是香港[[李登輝|李登輝]]之友會，倡導學習前[[中华民国總統|中华民国總統]][[李登輝|李登輝]]的思想<ref name="HKFront">[http://www.appledaily.com.tw/appledaily/article/property/20050207/1566995/ 「港獨」網頁遭中國砲轟] [[蘋果日報_(台灣)|蘋果日報 (台灣)]], 2005年02月07日</ref>。

== 事件 ==
「我是香港人連線」網頁於2004年11月1日註冊，其創立之際適逢[[香港立法會|香港立法會]]議員[[張超雄|張超雄]]提案香港就[[普選|普選]]爭議舉行全民[[公投|公投]]，因此[[北京|北京]]抨擊香港有人想搞「港獨」，北京並聲明香港若舉行公投，便是違反[[香港基本法|香港基本法]]。面對北京的指控，香港[[泛民主派|泛民主派]]一再聲稱香港沒有人在搞港獨。2004年11月中旬，香港報章以及[[美聯社|美聯社]]報導「我是香港人連線」網站的出現。之後，2005年2月6日[[中新社|中新社]]發表評論員文章，指控我是香港人連線公然策動「香港獨立運動」，並煽動網民加入。中新社指控「我是香港人連線」網站公然貼出港獨的「理據」，稱「1997年是被[[中共|中共]]吞併」；中國政府視香港為「[[殖民地|殖民地]]」，指[[香港特區政府|香港特區政府]]為「[[傀儡政府|傀儡政府]]」；指責《[[內地與香港關於建立更緊密經貿關係的安排|內地與香港關於建立更緊密經貿關係的安排]]》是「加強吮吸[[香港經濟|香港經濟]]成果」；提出「香港國」將實行的政體以及國歌、國旗等代表獨立主權的符號；並聲稱要透過網絡運動發動輿論攻勢，讓「港獨」發芽，在時機成熟之時成立「香港國」<ref name="HKFront"/>。

有報章指出「港獨」在香港不成氣候，但有一些媒體卻大張旗鼓點名批判<ref name="HKFront"/>。

== 評論 ==
有言論指「我是香港人連線」有關香港獨立建國的政治主張有可能因觸犯中國2005年3月14日第十屆[[全國人民代表大會|全國人民代表大會]]第三次會議通過的《反分裂國家法》而須負上刑責。支持這一論據的人更認為：這種鼓吹香港「獨立」，「違法亂紀之組織，吾人必須以刑法觀點論斷，而不應訴諸言論自由或學術自由考量而縱容此等歪理之蔓延。」

另外有言論指：由於香港人普遍沒有「獨立意識」，所以即使北京方面和香港官方沒有對《我是香港人連線》網站發表任何帶批評性質之言論，其影響力亦非常有限。反而在他們大聲疾呼之下，間接為該網站作宣傳。亦有指本網只是[[中國政府|中國政府]]官員發表「[[公投|公投]]等同[[港獨|港獨]]」論後，網民為回應中國政府而設立之「[[惡搞|惡搞]]」網站。網站內部份內容，有激烈如仿效[[孫中山|孫中山]]之招兵買馬或甘地之不合作運動等不單從未發生過，而內裡部份內容，如「香港人口是由多元種族組成，香港永久居民除[[華人|華人]]佔大多數，尚包括四個[[少數族裔|少數族裔]]，即[[孟加拉人|孟加拉人]]，[[高加索人|高加索人]]，[[印度人|印度人]]及[[巴基斯坦人|巴基斯坦人]]」、其地圖中將中國大陸稱為「[[支那|支那]]豬（無主地）」等。

== 網頁相關內容 ==
除倡導香港獨立運動外，該網頁亦鼓吹「[[東土耳其斯坦|東土耳其斯坦]]」（[[新疆|新疆]]）、「[[滿洲|滿洲]]」（即[[中國|中國]][[東北三省|東北三省]]的合稱，大致版圖近似[[二次大戰|二次大戰]]前的[[滿洲國|滿洲國]]）、[[內蒙古|內蒙古]]、[[台灣|台灣]]、[[西藏|西藏]]、[[福建|福建]]、[[南越|南越]]（或寫[[南粵|南粵]]，即包括[[廣東|廣東]]、[[廣西|廣西]]和[[海南|海南]]三省在地理上稱為[[兩廣|兩廣]]的地方）以及[[吳越|吳越]]（大致包括[[上海市|上海市]]和[[江蘇|江蘇]]、[[浙江|浙江]]兩省，有時還會把[[安徽|安徽]]和[[山東|山東]]兩省包括在內）必須獨立建國，成為主權獨立的國家<ref>[http://www.hkfront.org/sepch.htm 我是香港人連線的網頁]</ref>，並在其“未來[[支那|支那]]走勢圖”中宣稱“滅共不滅支，事成也枉然。滅共不滅豬，還是一場空”<ref>[http://www.hkfront.org/mapnewprovince.jpg 「未來支那走勢圖」]</ref>。
