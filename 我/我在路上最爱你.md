《'''我在路上最爱你'''》（{{lang-en|'''On the Way'''}}；{{lang-ko|'''길 위에서'''}}）中国大陆和韩国联合制作的[[3D|3D]]浪漫爱情片，制作方是巨力影视传媒有限公司、北京麦特文化发展有限公司、太特拉公司，导演[[金丰起|金丰起]]，主演黄圣依、池珍熙、文章、车秀妍。<ref>{{cite web |url=http://ent.qq.com/a/20140304/018400.htm |title= 中韩合拍《我在路上最爱你》 文章黄圣依首谈情|accessdate=2014 |publisher=腾讯网 |work= |language=zh-hans}}</ref><ref>{{cite web |url=http://ent.163.com/14/0327/08/9OB4L74B000300B1.html |title=《我在路上最爱你》点映 黄圣依激情戏喝酒暖场 |accessdate=2014 |publisher=网易 |work= |language=zh-hans}}</ref>该片原本有大尺度的[[床戏|床戏]]，但是由于中国大陆没有[[电影分级制度|电影分级制度]]，导致大部分章节被删除。<ref>{{cite web |url=http://ent.qq.com/a/20140307/007522.htm |title=《我在路上最爱你》黄圣依尺度大 杨子不回避 |accessdate=2014 |publisher=腾讯网 |work= |language=zh-hans}}</ref>

==剧情==
该片讲述“水晶”和韩国男子“周焕”的爱情故事。

==花絮==
该片曾在[[济州岛|济州岛]]、[[首尔|首尔]]、[[云南|云南]]、[[北京|北京]]等取景。
