《'''我不是药神'''》是2018年中国大陆的一部[[现实主义|现实主义]]题材[[剧情片|剧情片]]<ref name="hk01">{{cite web |author1=孟祥蓮 |title=【我不是藥神】高藥價背後的黑幕 令人遺憾的電影結局 |url=https://www.hk01.com/%E5%91%A8%E5%A0%B1/213883/%E6%88%91%E4%B8%8D%E6%98%AF%E8%97%A5%E7%A5%9E-%E9%AB%98%E8%97%A5%E5%83%B9%E8%83%8C%E5%BE%8C%E7%9A%84%E9%BB%91%E5%B9%95-%E4%BB%A4%E4%BA%BA%E9%81%BA%E6%86%BE%E7%9A%84%E9%9B%BB%E5%BD%B1%E7%B5%90%E5%B1%80 |website=香港01 |accessdate=2018-11-14 |language=zh-HK |date=2018-07-23}}</ref>，由[[文牧野|文牧野]]执导，[[宁浩|宁浩]]、[[徐峥|徐峥]]担任监制，徐峥、[[周一围|周一围]]、[[王传君|王传君]]、[[谭卓|谭卓]]、[[章宇|章宇]]、[[杨新鸣|杨新鸣]]、[[王砚辉|王砚辉]]主演。影片改编自于2015年发生的[[陆勇案|陆勇案]]，讲述一名[[印度神油|印度神油]]店的老板从印度[[走私|走私]][[代购|代购]]一种用于治疗[[慢性粒细胞性白血病|慢性粒细胞性白血病]]之药物“[[伊马替尼|格列宁]]”的[[學名藥|仿制药]]的经历。电影于2018年7月6日在中国大陸正式上映。

影片上映后，进一步引发了人们对中国大陸的进口药价格过高的关注。<ref name="hk01"/><ref name="theinitium">{{cite web |author1=張雅馨 |title=電影《我不是藥神》的背後：全球「高價藥」行業的盈利模式與行政刺激 |url=https://theinitium.com/article/20180727-international-drug-price/ |website=端传媒 |accessdate=2018-11-14 |language=zh-Hant |date=2018-07-27}}</ref><ref>{{cite web |author1=韩洁 |title=《我不是药神》揭高价药现状  百姓咋保命？ |url=https://www.rfa.org/mandarin/yataibaodao/huanjing/hj-07092018105317.html |website=Radio Free Asia |accessdate=2018-11-14 |language=zh-cn |date=2018-07-09}}</ref><ref name="thepaper">{{cite web |author1=杨松林 |title=谁催生了药神？高药价背后的体制逻辑和破题对策 |url=https://www.thepaper.cn/newsDetail_forward_2256477 |website=澎湃新闻 |accessdate=2018-11-14 |date=2018-07-13}}</ref>全球知名互联网公司[[Netflix|Netflix]]购买了《我不是药神》的全球播放权。<ref>{{cite web |author1=杜晋轩 |title=热门陆片登NETFLIX 台配额机制挡不住的网络巨浪 |url=http://culture.dwnews.com/renwen/news/2018-07-25/60073002.html |website=多维新闻网 |accessdate=2018-11-14 |date=2018-07-25}}</ref><ref>{{cite web |author1=李森 |title=《我不是藥神》Netflix購播映權 |url=https://hk.entertainment.appledaily.com/entertainment/daily/article/20180711/20446017 |website=香港蘋果日報 |accessdate=2018-11-14 |date=2018-07-11}}</ref><ref>{{cite web |title=不用再等陸片配額！票房83億《我不是藥神》將登Netflix |url=https://www.mirrormedia.mg/story/20180712ent010/ |website=鏡週刊 |accessdate=2018-11-14 |language=zh-TW |date=2018-07-12}}</ref>获得[[第55届金马奖|第55届金马奖]]最佳男主角、新导演和最佳原著剧本奖，以及[[第38屆香港電影金像獎|第38屆香港電影金像獎]]最佳两岸华语电影。

== 剧情 ==
程勇（[[徐峥|徐峥]]饰）是上海一间[[印度神油|印度神油]]店的老板。2002年某日在邻居的引荐下，白血病患者吕受益（[[王传君|王传君]]饰）登门拜访，请他从印度带回一批治疗慢性粒细胞性白血病的特效药“格列宁”。程勇深知走私贩售药物是非法的，但此时的他已经陷入了经济困难，前妻曹玲（[[龚蓓苾|龚蓓苾]]饰）正在争取儿子的全部抚养权，要带儿子移民国外，老父亲做手术急需钱。于是，程勇毅然决定踏上走私的道路。<ref>{{cite web |author1=魯皓平 |title=《我不是藥神》：很多時候，壞制度其實來自於「好人」 |url=https://www.gvm.com.tw/article.html?id=54776 |website=遠見雜誌 |accessdate=2018-11-14 |language=zh-TW |date=2018-11-08}}</ref><ref name="theinitium"/><ref>{{cite web |author1=張詠晴 |title=《我不是藥神》 想活命，有什麼罪 |url=https://www.cw.com.tw/article/article.action?id=5092857 |website=天下雜誌 |accessdate=2018-11-14 |language=zh-TW |date=2018-11-12}}</ref><ref>{{cite web |author1=张彰 |title=《我不是药神》：药贩子怎么就成了“药神” |url=https://www.thepaper.cn/newsDetail_forward_2230911 |website=澎湃新闻 |accessdate=2018-11-14 |date=2018-07-02}}</ref>

瑞士诺瓦公司（影射现实中的瑞士[[诺华公司|诺华公司]]<ref>{{cite web |title=《我不是药神》制药公司原型诺华要裁员了 |url=https://finance.sina.com.cn/chanjing/gsnews/2018-09-26/doc-ifxeuwwr8471864.shtml |website=新浪财经 |publisher=商业周刊中文版 |accessdate=2018-11-14 |date=2018-09-26}}</ref>）生产的治疗[[慢性粒细胞性白血病|慢性粒细胞性白血病]]的[[靶向药物|分子靶向药物]]“格列宁”（影射现实中的药物[[伊马替尼|伊马替尼]]，中文商品名为“格列卫”<ref name="thepaper"/>）药价为人民币四万左右一瓶，而程勇售卖的印度仿制药只需人民币五千元（影片后期变为五百元并向外省售賣）且药效相同，两者价格相差八倍至八十倍。最初，在瑞士诺瓦中国分公司的要求之下，中国大陸警方开始严查市场上流通的仿制药。程勇担心坐牢以及出于家庭因素的考虑而不再售卖仿制药，但一年后受朋友吕受益自杀的触动和病友恳求而再次售卖。之后，诺瓦公司发现了仿制药又在市场上流传，且通过法律程序关停了制造仿制药的印度工厂。影片末尾，程勇被警方逮捕，经审理被判处[[有期徒刑|有期徒刑]]五年。<ref>{{cite web |author1=鸡汤叔叔没有毒 |title=《我不是药神》："那天，走投无路的我在医院的厕所里自杀了" |url=https://wemedia.ifeng.com/68181123/wemedia.shtml |website=凤凰网大风号 |accessdate=2018-11-14 |date=2018-07-07}}{{需要更好来源}}</ref>

=== 药价高 ===

中国大陸的进口药药价居高不下，由以下几大因素共同作用<ref>{{cite web |author1=伍丽青 |title=《我不是药神》错了，进口天价药不能怪药企 |url=http://news.163.com/18/0707/01/DM2ULRVH00018M4D.html |website=网易新闻  |accessdate=2018-11-13|date=2018-07-07}}</ref>：
*进口药多来自最早拥有[[专利|专利]]、原创研发的全球知名巨头医药公司，他们投入了巨大的资金研发药物<ref>{{cite news |author1=高云翔 |title=“天价药”：错不在专利保护 |url=http://www.iprchn.com/cipnews/news_content.aspx?newsId=109819 |accessdate=2018-11-13 |work=中国知识产权报 |date=2018-08-03}}</ref>，需要从市场上找回成本。2014年，根据美国{{le|塔夫茨大学药物发展研究中心|Tufts Center for the Study of Drug Development}}的报告，现今开发一种新[[处方药|处方药]]的总成本会达到29亿美元以上。以伊马替尼为例，瑞士[[诺华公司|诺华公司]]从20世纪80年代末开始，经过十多年的努力花费数十亿美元研发出伊马替尼。2001年5月，伊马替尼通过了[[美国食品药品监督管理局|美国食品药品监督管理局]]的审核并获得了专利，诺华必须在专利保护期之内{{noteTag|1=专利保护期一般为20年左右。}}加紧将这些成本分摊给每一位患者来赚回巨额成本，每一瓶伊马替尼的价格自然会很高昂。
*依照中国大陸《[[:wikisource:zh:药品注册管理办法|药品注册管理办法]]》的相关规定，进入中国大陸的进口药必须重新进行在地临床试验，需要再度收集大量临床数据，儘管此舉目的在於保證外國廠家生產藥品的安全性與可靠性是否符合國內法律規範，但耗费大量人力、金钱与时间。
*中国大陸當局对进口药收取包括关税等各种税款。<ref>{{cite web |author1=向昌明 |title=内地外企原创药八成贵过港台 发改委称税费高所致 |url=http://china.cnr.cn/xwwgf/201309/t20130909_513550305.shtml |website=中国广播网 |accessdate=2018-11-13 |date=2013-09-09}}</ref><ref>{{cite web |title=内地进口药为何贵过香港？税+医院加价+流转费用 |url=https://www.yicai.com/news/2997636.html |website=第一财经日报 |accessdate=2018-11-13 |date=2013-09-09}}</ref>
*2017年7月1日，中国大陸开始实行[[药品加成|药物零加成政策]]，取消了数十年来医院内部的15%药物加价，但进口药不含在内。于是进口药成为了医疗机构的盈利产品，也不被纳入[[药占比考核体系|药占比考核体系]]{{noteTag|1=药占比是药品费用在医疗机构收入中的比重。}}之内，医生更偏好为患者开具进口药的处方。<ref>{{cite web |author1=阿托品 |title=全国药品零加成7月1日实施  国产药受冲击 |url=http://med.sina.com/article_detail_103_1_29213.html |website=新浪医药新闻 |publisher=赛柏蓝 |accessdate=2018-11-13 |date=2017-07-09}}</ref>
*平均[[灰色成本|灰色成本]]，是因医药公司向政府官员、医生等人员行贿而产生的其他外在成本。<ref name="21财经">{{cite web |author1=王卓铭 |title=GSK贿案背面：药品单独定价权被认为特殊福利 |url=http://finance.sina.com.cn/chanjing/cyxw/20130731/022016292094.shtml |website=21世纪经济报道 |accessdate=2018-11-13 |date=2013-07-31}}</ref><ref name="IBTimes">{{cite web |author1=付珊 |title=葛兰素史克行贿与暴利：单独定价权抢夺中国患者的救命钱 |url=http://www.ibtimes.com.cn/articles/31373/20130718/927588.htm |website=IBTimes中文网 |accessdate=2018-11-13 |date=2013-07-18}}</ref>
*2000年11月21日，[[中华人民共和国国家发展和改革委员会|国家发展和改革委员会]]颁布《[[药品政府定价办法|药品政府定价办法]]》，规定“[[原研药|原研药]]”{{noteTag|1=原创性的新药，在中国大陸主要是指过了专利保护期的进口药。}}可以给予[[单独定价权|单独定价权]]。但在中国大陸，即使进口药的专利保护期已经过期，医药公司依然有单独定价的政策，无需遵循政府的指导价。<ref>{{cite news |author1=李松涛 |title=外企原研药的价格特权加剧看病贵 |url=http://zqb.cyol.com/html/2012-08/13/nw.D110000zgqnb_20120813_1-05.htm |accessdate=2018-11-13 |agency=中国青年报 |publisher=中国青年报社 |date=2012-08-13 |page=05版}}</ref><ref name="21财经" /><ref name="IBTimes" />在中国大陸，原研药得到优待或者有优势，并且缺乏竞争对手下，无需降价便可占据市场份额，药物就没有降价的理由。

以上这些因素产生的巨大成本终究转嫁给了每一位患者。
