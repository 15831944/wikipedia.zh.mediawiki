《'''我讓最想被擁抱的男人給威脅了。'''》（{{lang-ja|抱かれたい男1位に脅されています。}}）是[[日本漫畫家|日本漫畫家]][[櫻日梯子|櫻日梯子]]所繪的[[日本漫畫|日本漫畫]]作品。該作以演藝界為主要舞台背景。2013年7月於[[Libre|Libre]]的漫畫雜誌《b-boy》上首次刊載後<ref>{{Cite web|url=https://www.b-boy.jp/magazine/32432|title=b-BOY 超キチク エリート陵辱特集 / 雑誌・アンソロジー|website=B-boy WEB|publisher=Libre|accessdate=2018-09-30}}</ref>，曾經在同出版社的季刊雜誌《JUNK!BOY》上刊載，目前於同出版社的月刊誌《MAGAZINE BE×BOY》上進行連載<ref name="natalie">{{Cite news|publisher=ナターシャ|newspaper=[[Natalie|Natalie]]|url=https://natalie.mu/comic/news/275095|title=「抱かれたい男1位に脅されています。」TVアニメ化！ドラマCDキャスト続投|date=2018-03-25|accessdate=2018-09-30}}</ref>。截至2019年3月時，漫畫在日本的累計發行數已經突破280萬本<ref name=":0">{{Cite news|url=https://prtimes.jp/main/html/rd/p/000000226.000022435.html|title=シリーズ280万部突破！大ヒットBL「抱かれたい男1位に脅されています。」最新コミック6巻「西條さんの日=3月13日」発売!!　”愛のチュン太号”こと「だかいち痛トラ」も走行！|author=|date=2019-3-13|work=|publisher=PR TIMES|accessdate=2019-3-17}}</ref>。該作在同年3月25日舉辦的「[[AnimeJapan|AnimeJapan]] 2018」上，發表了電視動畫化的消息，並於10月份播出。主角西條高人也獲選日本網站Charapedia舉辦「2018年最具魅力動畫角色」萬人投票排行榜第二名，僅次於[[工作細胞|工作細胞]]的[[血小板_(工作细胞)|血小板]]<ref>{{Cite web|url=http://www.charapedia.jp/research/0200/|title=2018年もっとも魅力的だったアニメキャラ TOP20!（総合）|accessdate=2018-12-23|author=|date=2018-12-20|publisher=}}</ref>。

== 劇情簡介 ==
西條高人是演藝經驗長達20年的資深演員，連續榮獲「最想被他擁抱的男人」第1名整整5年，卻被才入行3年的新進演員東谷准太奪去寶座，極度介意此排名的高人之後還遇上必須配合東谷合作出演的電影作品。西條對排名耿耿於懷，然而兩人卻在連串曲折後同行飲酒。

隔天高人在東谷家的床鋪上醒來，發現昨夜喝醉後醜態百出，而東谷則展示昨夜攝下的各種錄影。誤以為東谷要將這些影像公諸於眾的高人，表示願意答應東谷任何要求，但是東谷卻說出令高人感到錯愕的一句話……

== 主要登場人物 ==
<div class="notice metadata" id="spoiler"><small>'''[[聲優|聲優]]：廣播劇CD／電視動畫'''</small></div>

; {{nihongo|東谷准太|東谷准太（あずまや じゅんた）|JUNTA AZUMAYA|聲：[[小野友樹|小野友樹]]{{R|natalie}}<ref name="staff">{{Cite web|url=https://dakaretai-1st.com/staffcast/|title=STAFF・CAST|website=電視動畫官方網站|accessdate=2018-09-30}}</ref>}} 
:生日11月11日，天蠍座，血型O型，身高190cm，體重73公斤，隸屬MITSUYA製作公司。演藝資歷3年，人氣NO.1的新人演員。忠犬系男子，容易親近的閃亮亮系天使。祖父是西班牙人的四分之一混血兒，由於學生時期打過各式各樣的工，社交很廣，給人有「你連這個都會？」的印象，在周遭眾人為之驚訝過後，又會抬高他的身價。進入演藝圈的契機是外送披薩到演藝經紀公司時，當場被星探挖掘，雖然對演藝圈不怎麼有興趣，但參與了西條主演的連續劇後，受到西條的刺激開始認真朝演員之路邁進。平常真的是天使一隻，但在捕食最喜歡的西條高人時，其兇殘模樣簡直可與流冰天使裸海蝶相提並論。私底下高人會稱呼他叫「蠢太」。覺得嘴巴寂寞（高人不在）時會吃薄荷糖解饞，似乎在禁慾的時候也會大量攝取來抑制想推倒高人的念頭。

; {{nihongo|西條高人|西條高人（さいじょう たかと）|TAKATO SAIJYO|聲：[[高橋廣樹|高橋廣樹]]{{R|natalie}}{{R|staff}}}}
: 生日4月20日，白羊座，血型A型，身高176公分，體重55公斤，隸屬日東演藝（日藝）。8歲進入演藝圈，演藝資歷20年的老手，從童星時期就被稱為神童，是個多才多藝的人。專業意識很高，其工作表現在業界內也頗受信賴。由於人生的大部分時間都在演藝圈裡打拚，個性上有些過於坦率且馬虎的地方，但本人自己完全沒有自覺，深信自己是個很能幹的人。心情不好時會吃掉一整個蛋糕來發洩。喜歡喝酒，喝醉酒時就會發酒瘋，變得煩人又好笑，因此獲得好評（如喝醉酒時會一直剝毛豆）。曾在電視劇中因角色的需要而抽菸，結果引發身體不適，因此發誓在工作以外絕對不抽菸。非常喜歡小狗小貓和小孩子，覺得經紀人佐佐木的女兒（現在3歲）和老家的貓（挪威森林貓）黑松都可愛到不行。

; {{nihongo|綾木千廣|綾木千広（あやぎ ちひろ）|CHIHIRO AYAGI|聲：[[佐藤拓也|佐藤拓也]]{{R|natalie}}{{R|staff}}}}
: 生日12月30日，魔羯座，血型B型，身高180公分，體重63公斤，隸屬Japan Art Enterprise。入行3年，不過從入行前開始就常常在朋友的委託下擔任模特兒，藉機賺點零用錢。大學時期在酒吧當酒保，釣上了一位剛好來到酒吧的女性製作人顧客，之後請她介紹了一間在圈內也很有影響力的經紀公司。待人接物基本上是採取玩世不恭的態度，抱著「人生實在是太好混了」的態度而活。與高人一起演出舞台劇卻在第一天沒有出現排演。聽到路人討論高人在「擁抱排行榜」的想法，萌生起「如果把一本正經的高人上了，高人會有什麼表情」的念頭。覺得高人身形纖細，肩膀單薄感覺很容易抓住，並稱讚高人的背很漂亮。

; {{nihongo|成宮涼|成宮涼（なるみや りょう）|RYO NARUMIYA|聲：[[内田雄馬|内田雄馬]]{{R|natalie}}{{R|staff}}}}
: 生日6月12日，雙子座，血型B型，身高163公分，體重51公斤，隸屬Galaxy。入行1年，目前就讀高二，充滿活力笑容的偶像派演員。上了高人的母校，能夠兼顧學業和工作。在家裡專心打電動的時候被人拍下了照片，年長他4歲的姐姐擅自把照片寄給經紀公司，於是他在國中二年級時進入演藝圈。雖然事情似乎正中姐姐的下懷，不過他並沒有特別不滿，是能同時兼顧高水準的高中學業與工作的孩子，朋友也很多。現在正朝著天使支持者之路絕贊邁進中。

; {{nihongo|卯坂和臣|卯坂和臣（うさか かずおみ）|KAZUOMI USAKA|聲：[[鳥海浩輔|鳥海浩輔]]{{R|natalie}}{{R|staff}}}}
: 日視的幹練總製作人，負責電視劇「{{lang|ja|真昼の星}}」。平時都講標準語，不過自言自語之類不經意的時候會脫口講出家鄉方言京都腔。在AD時期就認識高人了。是個老菸槍，而且非常愛吃辣。喜歡的食物是超辣麻婆豆腐。不喜歡甜食，如果吃到就會皺眉。給人的壓迫感很重，難以親近，不過日視的女性員工私底下都很欣賞他。單身。AB型。明明個性冷淡卻很受動物喜愛。

; {{nihongo|在須清崇|在須清崇（ありす きよたか）|KIYOTAKA ARISU|聲：[[羽多野涉|羽多野涉]]{{R|natalie}}{{R|staff}}}}
: 神出鬼沒的神祕男子。
