== 故事簡介 ==
宮原紅葉在升上高中後便離開了家裡一個人獨居。某天放學後，因弄丟車票而走路回家的紅葉在路上發現了一間[[居酒屋|居酒屋]]「季節菜色酒場 小春屋」，卻在這時被剛好從店裡走出來的千百合誤認為是新來的員工而被拉進店裡幫忙。在這次突然的經驗後，紅葉決定繼續待在這間居酒屋打工，也為她的生活帶來了不同的變化。

== 登場角色 ==
; {{nihongo|宮原紅葉|宮原 もみじ（みやはら もみじ）}}
: 高中生，16歲。獨居。擅長做菜，負責料理的工作，有時也會負責點菜。有個在世時曾經營過居酒屋的奶奶。

; {{nihongo|山吹千百合|山吹 ちゆり（やまぶき ちゆり）}}
: 17歲。身形嬌小。負責接待客人與跑腿。對自己的要求標準很高，但也有冒失的一面。

; {{nihongo|南部皐月|南部 皐月（なんぶ さつき）}}
: 小春屋的店長。負責料理。很照顧員工們，也會傾聽紅葉等人的心聲並給予建議。

; {{nihongo|如月真澄|如月 真澄（きさらぎ ますみ）}}
: 負責接客與採買店內食材的從業員。個性開朗。喜歡喝酒，時常帶著宿醉上班。

; {{nihongo|藍川凪|藍川 凪（あいかわ なぎさ）}}
: 紅葉的同班同學，個性較膽小。在千百合感冒請假時臨時幫了紅葉及小春屋的忙，也以此為契機決定留下來打工。
