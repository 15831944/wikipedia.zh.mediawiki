《'''我山之石'''》，著名[[历史学家|历史学家]][[易中天|易中天]]的作品，易中天从书中剖析了[[诸子百家|诸子百家]]治国理政的优点和缺点，同时引用现代生活中的实力来做佐证，最后得出结论，[[儒家|儒]][[墨家|墨]][[道家|道]][[法家|法]]均可借鉴，但是适应不了现代社会，[[民主|民主]]、[[科学|科学]]和[[法治|法治]]才是当今社会的救世救国之道。<ref>{{cite web |url=http://culture.ifeng.com/whrd/detail_2009_08/02/305487_0.shtml |title= 易中天推新书《我山之石》 称儒家思想应付不了现代社会|publisher=凤凰网 |date=2009年8月2日|accessdate=2012-04-24 |work= |language =}}</ref>

== 内容 ==
《我山之石》通过易中天对诸子百家中的儒墨道法四家的治国理政经验和政策的剖析与对比，借鉴[[中华文化|中华文化]]古代的经验，也指明了中国未来的出路：民主、科学和法治。<ref>{{cite web |url=http://news.163.com/09/0915/17/5J95BFAM000120GR.html |title=我山之石　也可攻玉 |publisher=网易 |date=2009-09-15|accessdate=2012-04-24 |work= |language =}}</ref>

== 风格 ==
书籍对先秦诸子百家中的儒墨道法四家的思想用[[现代汉语|现代汉语]]进行了提炼和简要说明，主要集中在“儒墨之争”、“儒道之争”、“儒法之争”的焦点上，通过古人思想的争锋来阐述其思想和智慧。<ref>{{cite web |url=http://www.china.com.cn/book/txt/2009-09/14/content_18521105.htm |title=易中天:推销先秦思想《我山之石》比《品三国》好 |publisher=中国网 |date=2009-09-14|accessdate=2012-04-24 |work= |language =}}</ref>

== 评价 ==
