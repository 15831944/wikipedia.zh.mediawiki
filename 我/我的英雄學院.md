《'''我的英雄學院'''》（{{jp|j=僕のヒーローアカデミア|rm=Boku no Hero Academia|lead=yes}}）是日本漫畫家[[堀越耕平|堀越耕平]]繪製的[[少年漫畫|少年漫畫]]，自《[[週刊少年Jump|週刊少年Jump]]》2014年32號起連載；[[台灣|台灣]]中文版由[[東立出版社|東立出版社]]代理並於《[[寶島少年|寶島少年]]》2014年第43期開始連載；香港中文版由[[玉皇朝|玉皇朝]]代理。

2015年11月宣佈改編電視動畫，第一季於2016年4月至6月播放；隨後第二季於2017年4月至9月播放；緊接著第三季於2018年4月至9月播放；此外另有電影版於2018年8月上映。在第三季播畢後宣布製作第四季。

== 故事大綱 ==
{{more plot}}
這個世界從某一天開始，新生的人們突然開始展現出名為「個性」的特殊能力，本該只出現在架空故事中的超能力者成為現實的存在，讓整個人類社會陷入變化與動盪。在一片混亂之際，心懷邪念的人開始以「個性」為非作歹，而就在政府對這些突如其來的災禍束手無策之際，身負力量與正義感的人開始如同超人漫畫般的揭竿而起，展現正義的「英雄」也相繼而生。現今，有五分之四的人都擁有「個性」的超人社會儼然成形，為了對抗濫用「個性」於犯罪用途的罪犯，政府將英雄定位為特殊的公務職員，依其功績給予報酬。豐厚的報酬與體現正義的榮耀，加上享有民眾的聲援與支持，英雄成為所有人嚮往的最棒職業。

主角綠谷出久是個沒有個性的少年，但他仍憧憬並渴望成為一位英雄，也期望自己能進入培育英雄的菁英學校雄英高校就讀。但周圍的人都不看好沒有個性的他能成為英雄，讓他總是在他人的嘲笑與輕視中度過。直到他遇上了自己最仰慕的英雄，被人稱為「和平的象徵」的歐爾麥特，他的夢想將因此獲得會化為現實的可能性。

== 用語解說 ==
; {{nihongo|個性|個性}}
: 先天性的'''超常能力'''的稱呼，作品當中有80％的人類持有的能力。通常4歳前就會被發現，小學一年・中學一年有個性的一齊診斷，出久所在的個性世代為第五世代。因為也有著輕易取人性命的個性，所以有著嚴格的規範管制著個性的使用。
: 另外沒有個性的人在社會上工作或求學時會受到某種程度的歧視，同時也不能擔任英雄，如果有想擔任英雄的志向卻無個性的人，最多只能從事警察這一方面的職務。
:; 個性特異點
:: 一種末世論，'''個性'''會隨世代變遷而融合深化，可能有沒有人能駕馭的複雜個性出現。

; {{nihongo|英雄|ヒーロー|}}
: 隨著濫用個性而為非作歹的人數增加，開始有人為打擊犯罪而挺身而出，是在輿論之下而產生的新型態職業。由英雄事務所為中心營業，由政府發給薪資。目前就職英雄的人數處於過度飽和狀態，因此也有人爭取讓英雄兼職副業謀生。英雄並沒有懲罰及拘捕惡人的權利，通常會跟警察一同行動，而英雄事務所的實習生並未給予合法使用'''個性'''的權利，故造成傷害的話會對算是監護人的該事務所給予懲處。
:; {{nihongo|戰鬥服|コスチューム}}
:: 英雄活動時所穿的服裝。是作為一個英雄的象徵。而在雄英高校內的學生則可以取得依照個人需求而設計的戰鬥服。

; {{nihongo|敵人|ヴィラン|villain}}
: 使用個性犯罪之人，敵人（villain）的英語意思是'''怪人'''或'''惡人'''，是美式漫畫常見的基本要素之一。

; {{nihongo|雄英高校|国立雄英高等学校（こくりつゆうえいこうとうがっこう）}}
: 國立的名門高校，校訓為“Plus Ultra（不懈進取，永無止境）”。學生平均偏差值高達79。在此的教師全都是職業級的英雄。校風自由，而老師同樣也擁有自由（相澤語）。
: 每個學年的學生分成英雄科A、B班、普通科C、D、E班、輔助科F、G、H班、經營科I、J、K班，總計11班。英雄科每年通常只收40人。若想成為偉大英雄，從雄英畢業是公認的必需條件。
; {{nihongo|敵人聯軍|敵連合（ヴィラン れんごう）}}
: 為了殺死「和平的象徵」歐爾麥特而聚集人員的謎之集團，有著數位強勁的個性使用者；原為聚集地痞流氓的集團，在USJ襲擊失敗後就捨棄不良份子改吸收精英罪犯。發生英雄殺手事件後開始聚集受英雄殺手理念影響的人跟重型犯。

; {{nihongo|折寺中學|折寺中学校}}
: 出久和爆豪的母校，是平凡的市立中學。名稱的由來是電影[[星際大戰|星際大戰]]中出現的行星奧德蘭（オルデラン）」。

; {{nihongo|多古場海濱公園|多古場海浜公園}}
: 原本是積滿漂流物的海灘，在出久以清掃方式進行特訓後，已經煥然一新，成為絕佳的約會地點。名稱的由來是電影星際大戰中出現的行星，路克進行特訓的場所Dagobah（ダコバ）。
