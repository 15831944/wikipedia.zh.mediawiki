《'''我在等你'''》（{{lang-ko|너를 기다린다}}）是[[韓國|韓國]]演唱團體[[Super_Junior|Super Junior]]成員[[圭賢|曺圭賢]]的第三張個人迷你專輯，2016年11月10日發行，並以《Blah Blah》、《Still》為此專輯的主打曲目。此張專輯也是圭賢入伍前最後一張個人專輯。<br />
2017年2月16日，公開《Blah Blah》泰文版音源及製作花絮。<ref>{{Cite web|url=https://www.kpopn.com/2017/02/16/%E5%9C%AD%E8%B3%A2%E5%85%AC%E9%96%8B%E6%B3%B0%E6%96%87%E7%89%88%E3%80%8Ablah-blah%E3%80%8B%E8%A3%BD%E4%BD%9C%E8%8A%B1%E7%B5%AE/|title=圭賢公開泰文版《Blah Blah》製作花絮|accessdate=|author=|date=|publisher=kpopn}}</ref><ref>{{Cite web|url=https://www.youtube.com/watch?v=9HYTSXyGFJo|title=KYUHYUN 규현 'Blah Blah (Thai Ver.)' MV|accessdate=|author=|date=|publisher=SMTOWN}}</ref>

== 發行與宣傳 ==
2016年10月19日，[[SM娛樂|SM娛樂]]表示圭賢將於11月發行第三張迷你專輯<ref>{{cite web |title=圭賢將於11月回歸，成詩京參與作曲 |url=https://www.kpopn.com/2016/10/20/%E5%9C%AD%E8%B3%A2%E5%B0%87%E6%96%BC11%E6%9C%88%E5%9B%9E%E6%AD%B8%EF%BC%8C%E6%88%90%E8%A9%A9%E4%BA%AC%E5%8F%83%E8%88%87%E4%BD%9C%E6%9B%B2/ |publisher=kpopn}}</ref>。此張專輯是繼《[[在光化門|在光化門]]》及《[[再次，秋來|再次，秋來]]》後，再次以秋季抒情曲傳達秋日情感的一張專輯，也是秋日情歌三部曲的最終章<ref>{{cite web |title=圭賢《Waiting, Still》首波概念照公開 |url=https://www.kpopn.com/2016/11/03/%E5%9C%AD%E8%B3%A2%E3%80%8Awaiting-still%E3%80%8B%E9%A6%96%E6%B3%A2%E6%A6%82%E5%BF%B5%E7%85%A7%E5%85%AC%E9%96%8B/ |publisher=kpopn}}</ref>。11月2日至8日，陸續公開預告照；11月7日及8日，分別公開雙主打歌《'''Blah Blah'''》與《'''Still'''》的MV預告。

== 錄製與音樂 ==
雙主打歌《'''Blah Blah'''》與《'''Still'''》分別由著名抒情歌手[[尹鍾信|尹鍾信]]及[[成始璄|成始璄]]參與歌曲製作；《Blah Blah》由{{link-ko|曺圭贊|조규찬}}參與和聲，是一首溫暖吉他聲與復古電子鋼琴旋律完美融合，表現秋日感性的抒情歌曲。《Still（忘不了的溫柔）》是一首突出電子鋼琴、電吉他聲音魅力的90年代流行抒情歌曲。<ref>{{cite web |title=圭賢推solo迷你3輯 攜尹鐘信成始璄打造新曲 |url=http://korea.yule.sohu.com/20161104/n472265173.shtml |publisher=搜狐}}</ref> <br />
《Last Good-bye（獨自說再見）》是一首以圭賢的歌聲為中心，鋼琴和絃樂器演奏華麗的流行敘事情歌，無法實現愛情的痛苦的編曲和富有詩意的歌詞添加了悲切感；《Fall in You》是一首關於兩個戀人的故事的明朗氛圍中等節奏歌曲，圭賢柔和的聲線和甜美歌詞完美和諧；表達對淡忘的過去戀人的懷念和遺憾的抒情曲《Love in Time（時節姻緣）》； 開始戀愛的戀人的心意比喻為‘心靈淨化站’的情歌《When with me（心靈淨化站）》；表現對戀人的告白的溫暖求婚曲《And We（執手偕老）》，共收錄了7首歌曲。<ref>{{cite web |title=“秋季抒情曲歌手”圭賢今晚回歸 期待感UP！ |url=http://korea.yule.sohu.com/20161109/n472680908.shtml |publisher=搜狐}}</ref><ref>{{cite web |title=圭賢第三張迷你專輯『我在等你』台壓版 |url=https://shopping.avex.com.tw/avex/Default.aspx?m=80E3CEC5ACBCD193F672C3247B4575D4BE1D743F00B33C6A&ItemId=9887 |publisher=愛貝克思}}</ref>
