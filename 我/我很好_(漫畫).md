== 故事簡介 ==
{{expand section}}
女主角若草春奈因為幫助經常被她的前男友霸凌、在學校總是獨來獨往的男主角山田一郎之下，兩人來到學校旁邊的河畔發現早已腐爛的遊民屍骨。然後若草透過山田認識當紅名模學妹吉川，看似毫無交集三個人在因緣際會下竟然建立起某種革命情感，一起共享著河畔腐爛屍骨的秘密…

== 登場人物 ==
; {{nihongo|若草春名|若草 ハルナ（わかぐさ はるな）}}
: 作品主角。高中生。從小雙親離異之後跟母親一起居住。與男友觀音崎感情不合，經常目睹他霸凌山田。另外與一起跟山田發現屍體的吉川交情不錯。

; {{nihongo|山田一郎|山田 一郎（やまだ いちろう）}}
: 春名的同班同學。[[男同性戀者|同性戀]]者。在學校相當受到女生的歡迎，因此常被觀音崎霸凌。有時會和女生遊玩。跟田島交往是為了隱瞞自己是同性戀的事實。

; {{nihongo|吉川梢|吉川 こずえ（よしかわ こずえ）}}
: 春名的學妹，當紅模特兒。5歲的時候被母親拉攏進入演藝事業。正因為如此擁有熟女外表和剛強性格，但有厭世的一面、經常[[進食障礙|暴飲暴食然後把食物吐出來]]。[[女同性戀|女同性戀]]者，因此也很喜歡春名。

; {{nihongo|觀音崎-{峠}-|観音崎 峠（かんのんざき とうげ）}}
: 春名的男友。很喜歡春名，對山田充滿嫉妒。與條件優渥的哥哥相反，因為經常進行藥物交易而被趕出家門。其本性相當寂寞。

; {{nihongo|田島神奈|田島カンナ（たじま かんな）}}
: 山田的女友。很喜歡山田，但是山田始終對她感情鬱結，而經常被他拒絕。後來得知山田跟春名兩人的交情不錯，從此展開對春名進行騷擾的行動。

; {{nihongo|小山留美|小山ルミ（こやま るみ）}}
: 春名的朋友。常勾引觀音崎和其他男生搞性關係，以填補自己內心的空虛，並利用[[援助交際|援交]]賺來的金錢購買名牌衣物和化妝品，把自己打扮得花枝招展。然而她自己注意到已經懷孕，因此自身的生活變得很瘋狂。

; {{nihongo|小佳|よっちゃん}}
: 春名的朋友。喜歡和男友高橋兩人一起聽信謠言。對追捧名牌的留美有一點厭惡。

; {{nihongo|小山真子|小山マコ（こやま まこ）}}
: 留美的姊姊。是位從事同人[[BL_(和製英語)|BL]]漫畫執筆的[[御宅族|肥宅]]。暗地裡相當忌妒妹妹擁有美麗外貌和身材。

2017年3月22日宣布拍攝真人電影，[[行定勳|行定勳]]執導，[[二階堂富美|二階堂富美]]、[[吉澤亮|吉澤亮]]領銜主演<ref name="rivmovie" />。2018年2月16日在日本首映<ref name="homepage">{{Cite web|date=|url=http://movie-riversedge.jp/|title=電影『我很好』官方網站|work=|publisher=|accessdate=2017-10-28|language=ja}}</ref>。[[電影分級制度|R15+]]指定。

該片上映一星期之後，2018年2月24日獲頒第68屆[[柏林國際影展|柏林國際影展]]Panorama部門的國際影評家聯盟獎<ref name="Pan2">{{cite web|url=https://www.berlinale.de/en/presse/pressemitteilungen/alle/Alle-Detail_41492.html|title=Programme for Berlinale Panorama 2018 Complete: What’s Your Poison？|work=Berlinale|date=2018-01-25|accessdate=2018-01-29|language=en|deadurl=yes|archiveurl=https://web.archive.org/web/20180129195656/https://www.berlinale.de/en/presse/pressemitteilungen/alle/Alle-Detail_41492.html|archivedate=2018年1月29日|df=}}</ref><ref>{{Cite news|url= http://eiga.com/news/20180224/1/ |script-title=ja:「リバーズ·エッジ」が快挙！ベルリン映画祭パノラマ部門で国際批評家連盟賞受賞|newspaper={{link-ja|電影.com|映画.com}}|publisher=株式會社電影.com|date=2018-02-24|accessdate=2018-02-24|language=ja}}</ref>。

此外，畫面[[長寬比_(影像)|播放比例]]使用的不是現在普及的16:9，而是使用4:3進行拍攝<ref name="bunshun">{{Cite web|date=2018-02-17|url=http://bunshun.jp/articles/-/6232|script-title=ja:岡崎京子の傑作「リバーズ·エッジ」 映画が横長でなく4:3で撮影された理由|work=《週刊文春》2018年2月22日號|publisher=文春Online|accessdate=2018-02-20|language=ja}}</ref>；還有，電影中角色的服裝與造型忠實呈現出漫畫連載當時相同的90年代風格。不過CHRISTIAN DADA和FACETASM等品牌仍將此風格當作「具有街頭感的現在日本品牌」使用<ref>《{{link-ja|裝苑|装苑}}》2018年2·3月合併號，[[文化服裝學院|文化出版局]]。 </ref>。

2019年3月1日起由[[Netflix|Netflix]]獨家在海外地區發佈。
