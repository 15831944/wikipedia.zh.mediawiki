《'''我的祖國'''》 （[[馬來語|馬來語]]：'''{{lang|ms|Negaraku}}'''、[[爪夷文|爪夷文]]：'''{{lang|jv|لاڬو نڬاراکو}}'''、[[坦米爾語|坦米爾語]]：'''{{lang|ta|நெகாராகூ}}'''）是[[馬來西亞|馬來西亞]]的[[國歌|國歌]]。此歌曲採用了[[霹靂州|霹靂州]]州歌的[[旋律|旋律]]。

正式演奏此曲前，有一陣短鼓聲（多數是在[[獨立日|獨立日]]/[[國慶日|國慶日]]時），此實為吸引聽眾注意之舉，並預示將會開始一個莊嚴的步伐及演奏國歌，並暗示已開始唱起了承諾效忠國王和國家的安全的歌。

== 歷史 ==
在[[馬來亞联合邦|馬來亞联合邦]]快要獨立時，11個州（當時為11個州，[[沙巴|沙巴]]和[[砂拉越|砂拉越]]還沒加入）個個都有自己的州歌，但当时联合邦的国歌为英国国歌《[[天佑女王|天佑女王]]》。爲此，馬來亞的第一任[[首相|首相]][[東姑阿都拉曼|東姑阿都拉曼]]與當時的政府部門的高級官員組織了一個委員會，以選擇一首適當的國歌<ref name="auto">馬來西亞報章《星洲日報》副刊“星洲互動”版，第14頁</ref>。他的建議公佈后，便展開了一個世界性的競爭。此賽一共有514人參賽，而且是來自世界各地，但皆被認為不合適 <ref>馬來西亞報章《星洲日報》“國慶專輯”，第12頁（2007年8月31日）</ref>，其中一位參賽者為[[英國|英國]][[作曲家|作曲家]][[本杰明·布里顿|本杰明·布里顿]]。

1957年8月5日，該委員會決定採用改編自印尼民謠《Terang Boelan》的霹靂州州歌的旋律，因爲它具有傳統南洋風味。最後，此歌曲于1957年在吉隆坡Depoh警察禮堂被正式宣佈成爲馬來亞的國歌。

此歌曲的[[節拍|節拍]]曾經被馬來西亞第四任首相[[馬哈迪|馬哈迪]]改編過。新的歌曲于2003年8月31日[[半夜|半夜]]12時于默迪卡廣場作了第一次的演奏。事實上，新的歌曲仍然和舊的歌曲一樣，只不過是樂隊的演奏速度加快而已。

改編自印尼民謠《Terang Boelan》的馬來西亞國歌，在2010年被指為法國人{{link-en|皮耶-讓·德·貝朗瑞|Pierre-Jean de Béranger}}的作品。2016年6月7日《[[星洲日報|星洲日報]]》有報導如下：

{{Cquote|很多人以為我們的國歌《Negaraku》源自於印尼，但是，根據大馬歷史學者丹斯里邱家金透露，這首曲其實是出自法國作曲家皮耶爾讓（Pierre Jean）之手，曲名是《La Rosalie》。<ref>[http://www.sinchew.com.my/node/1535198 《“La Rosalie”傳唱百年‧Negaraku是法國曲》，星洲網新聞，2016-06-06]</ref> }}

== 版本 ==
我的祖国拥有4个不同节拍速度的版本：<ref>[https://www.youtube.com/watch?v=08kCXADSEB4/ Negaraku [Multi-Version] - Youtube], 30 July 2010</ref>
* 1957年版本（第一版本）
* 1992年版本（由第四任首相[[马哈迪|马哈迪]]改编）
* 2003年版本（由第五任首相[[巴达维|巴达维]]改编）
* 2007年版本（由第五任首相[[巴达维|巴达维]]改编）
马来西亚国内普遍上都使用2003年版本与2007年版本，这两个版本都可在[[马来西亚|马来西亚]]境内合法使用。马来西亚大部分公立学校都使用2003年版本；公共场所或大小型活动等都会使用2007年版本。在马来西亚境外或国际上都一律使用2007年版本。

== 曲名 ==
《我的祖國》一曲的曲名曾經于2003年被公佈轉換為《我的马来西亚》（{{lang|ms|Malaysiaku}}）。此事公佈后，有許多人反對。<ref name="auto"/>因此，《我的祖國》一曲的馬來語曲名至今仍是{{lang|ms|Negaraku}}。<ref name="auto"/>

== 改編事件 ==
{{seealso|黃明志}}
《我的祖國》一曲曾經于2007年7月被留學臺灣的馬來西亞學生[[黃明志|黃明志]]改編成[[音樂錄影帶|音樂錄影帶]]，並以[[饒舌|饒舌]]的方式唱該歌曲。<ref>[http://www.sinchew.com.my/content.phtml?sec=15&sdate=%20&artid=200707241397 馬來西亞報章《星洲日報》報道]</ref>他將改編的歌曲取名為《我愛我的國家》（馬來語：{{lang|ms|Negarakuku}}），上載至[[YouTube|YouTube]]<ref>{{cite web|url=http://www.youtube.com/watch?v=3atvcI0oS-w&NR=1|title=YouTube|website=www.youtube.com}}</ref>。上載過後，黃明志馬上遇到各界的批評及評論，馬來西亞的其中一家報館報道他的歌曲有接觸到種族和宗教信仰和政治等敏感問題，可能會導致各族人民分裂。<ref>馬來西亞報章《星洲日報》（2007年7月，國内版）</ref>[[馬來西亞政府|馬來西亞政府]]也表示當局可能會索引《1965年國歌法令》第8（2）條款控告他。<ref name="auto1">馬來西亞報章《星洲日報》（2007年8月14日，國内版，第10頁）</ref><ref>{{cite web|url=http://www.sinchew.com.my/content.phtml?sec=1&sdate=2007-07-19&artid=200707191557|title=404|publisher=}}</ref>有關法令註明，任何人在任何公共場所蓄意表示對國歌不尊重的人，可被判罰款不超過100令吉或監禁不超過1個月。<ref name="auto1"/>

事後，黃明志及他的父親黃安熙通過馬來西亞衛生部部長和[[馬華|馬華]]副總會長拿督斯理[[蔡細歷|蔡細歷]]對這件事公開道歉，並希望國人及政府能夠接受他的道歉，而此事也可以就此結束。<ref> 馬來西亞報章《星洲日報》（2007年8月15日，國内版，第03頁）</ref>另外，馬來西亞行動黨秘書長[[林冠英|林冠英]]則表示，如果黃明志被控上法庭，行動黨將會組織一個由非政府律師和[[馬來西亞律師公會|馬來西亞律師公會]]的律師團，以免費提供他法律協助。<ref> 馬來西亞報章《星洲日報》（2007年8月18日，國内版，第07頁）</ref>

行動黨領袖林吉祥對於此事發表意見時說，內閣在馬來西亞獨立50周年國慶時，不負責任地處理臺灣留學生黃明志的改編國歌事件，激起马来人更大異議，华人却很支持。內閣花了很長時間討論是否接受黃明志的道歉，可是卻沒有認真面對比此事還更重要的事件，已令內閣本身被國人淪為笑柄。<ref>{{cite web|url=http://www.sinchew.com.my/content.phtml?sec=1&sdate=&artid=200708172440|title=404|publisher=}}</ref>

另外，[[馬來西亞民政運動|民政黨]]全國署理主席丹斯理[[許子根|許子根]]認爲，政府若要控告黃明志，就應該一視同仁地將其他所有發表破壞國民團結的言論的人也依法處理，不管該人物是多麽的位高權重。<ref> 馬來西亞報章《星洲日報》（2007年8月18日，國内版，第06頁）</ref>
