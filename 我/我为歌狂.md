'''我為歌狂'''，是一部2000年至2002年制作的[[中國大陸|中國大陸]]的[[中國動畫|國產動畫]]，由[[上海美术电影制片厂|上海美术电影制片厂]]制作、[[上海电视台|上海电视台]]推出，总共52集，前26集于2001年8月6日在上海电视台首播<ref>{{cite news|url=http://news.eastday.com/epublish/gb/paper148/20010731/class014800007/hwz450533.htm|title=《我为歌狂》健康登场|publisher=[[青年报|青年报]]|date=2001-07-31|accessdate=2016-01-31}}</ref><ref>{{cite news|url=http://news.xinhuanet.com/news/20010808/769728.htm|title=《我为歌狂》欲“盗”《灌篮高手》|publisher=[[北京青年报|北京青年报]]|date=2001-08-08|accessdate=2016-01-31}}</ref><ref name="xwcb182">{{cite news|url=http://ent.sina.com.cn/v/2001-08-02/52325.html|title=电视动画片《我为歌狂》起用“童声”班底|publisher=[[新闻晨报|新闻晨报]]|date=2001-08-02|accessdate=2016-01-31}}</ref>，52集完整版于2002年播出。<ref>{{cite news|url=http://news.eastday.com/epublish/gb/paper148/20020320/class014800007/hwz624507.htm|title=千余集国产动画片年内走上申城荧屏|publisher=[[文汇报_(上海)|文汇报]]|date=2002-03-20|accessdate=2016-01-31}}</ref>这部动画是上海美术电影制片厂继《宝莲灯》之后的又一部动画大制作，也是中国第一部校园音乐题材的动画片，影片以青春、节奏和新世纪崭新的视听形象开辟了中国动画片历史的新篇章。後並改編成[[電視劇|電視劇]]。<ref>{{Cite web|url = http://cq.people.com.cn/GB/365409/c26040653.html|title = 国产动画也有精品 盘点10部精良中国动漫|accessdate = |author = 光明网|date = 2015-08-19|publisher = 人民网|deadurl = yes|archiveurl = https://web.archive.org/web/20160304071115/http://cq.people.com.cn/GB/365409/c26040653.html|archivedate = 2016-03-04}}</ref>

== 改编作品 ==
*《[[我为歌狂|我为歌狂]]》，是2002年播出由[[刘芸|刘芸]]、[[姚笛|姚笛]]、[[季晨|季晨]]和[[李立_(演员)|李立]]等主演的一部22集电视剧，后更名为《'''歌让我狂'''》，是中国内地第一部根据动漫改编而成的连续剧。
*《[[我为歌狂：夏日彩虹|我为歌狂：夏日彩虹]]》，[[上海育碧|上海育碧]]制作，2002年发售的恋爱冒险+回合制策略游戏。
