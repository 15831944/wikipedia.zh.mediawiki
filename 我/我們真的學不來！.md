{{日本作品|我們真的學不來！|ぼくたちは勉強ができない}}是由[[日本漫畫家|日本漫畫家]][[筒井大志|筒井大志]]於《[[週刊少年Jump|週刊少年Jump]]》2017年10號開始連載的戀愛漫畫，此部作品是作者在《[[偽戀角色列表#魔法甜點師小咲|魔法甜點師小咲!!]]》（與[[古味直志|古味直志]]作品《[[偽戀|偽戀]]》合作的外傳漫畫）連載完畢後獨自推出的漫畫。每一话以「问○」为题，其副标题总带有「[X]」。<ref>第一话就叫做「问1：天才与[X]表里如一」</ref>在漫畫第74話後宣佈動畫化。

== 故事大綱 ==
家境清寒的優秀高中生唯我成幸，為了獲得免繳學費的大學「特別VIP推薦資格」，必須要讓同學緒方理珠、古橋文乃和武元潤香考上與她們不擅長的科目領域的志願學校，展開了這充滿熱血與浪漫氣氛校園生活。

== 登場人物 ==
=== 主要人物 ===
; {{nihongo|唯我成幸|唯我 成幸（ゆいが なりゆき）|聲：[[逢坂良太|逢坂良太]]<ref name="character">{{cite web|url=https://boku-ben.com/character/|title=CHARACTER|work=電視動畫「我們真的學不來！」官方網站 |language=ja}}</ref>}}
: 本作的男主角，被称作「影之帝王」。自小成績並不出眾，經過父親鼓勵後努力尋求學習的方法，成為秀才學生。在父親過世後，為了家中經濟著想，企圖透過就讀高中內的「特別VIP推薦資格」制度來免除大學學費，学园长對他開出交換條件：讓理珠與文乃考上與他們所擅長領域不同的志願大学，成為她們的「教育員」，此事最初讓成幸傷透腦筋，但在指導兩人的過程中，成幸慢慢了解兩人的性格與她們的理想，也想起曾經那成績不起眼的自己，於是他開始想盡辦法以找到指導她們學習的辦法為目標。
: 家庭成員有母親、妹妹水希、以及雙胞胎葉月與和樹。由於家境清寒，所以他非常重視金錢觀念，能利用家裡的資源自己自製一些克難物品。十分不擅長使用數位機械，對體育也不拿手且不會游泳。
: 在第1回角色人气投票順位第7名(437票)。
: 已放棄成為「特別VIP推薦資格」，將目標轉向師範學校，目標擔當教育員。

; {{nihongo|古橋文乃|古橋 文乃（ふるはし ふみの）|聲：[[白石晴香|白石晴香]]<ref name="character"/>}}
: 面容姣好的長髮女孩，擅長科目為[[文科|文科]]，现代文、文言文等无所不能，被稱作「文學之森的[[睡美人|睡美人]]」，目標是考上理科大學，卻不擅長理科。
: 家境富裕，由於母親自小便撒手人寰，相信母親死後成為星星的文乃便打算成為天文學家，尋找母親變成的星星。她也因而與父親亡故的成幸有共鳴。由於相依為命的父親個性嚴格且很有威勢的緣故，因此父女關係說不上很好，也因此對這類型的男性不太擅長應付。
: 心思細膩，只看-{}-表情就能大致猜到對方的想法。知道潤香與理珠對成幸有好感，為此煩惱要支持哪一位朋友的戀情。因為知道三人都不習慣處理情感，為了不妨礙學習而時常從旁疏通尷尬的場面，也會傾聽成幸感情方面的煩惱並給出建議，不過在對方低情商的表現下會感覺胃痛。實際上本身也對成幸抱有好感，但認為不能喜歡上朋友喜歡的對象而壓抑。
: 似乎很在意自己的貧乳，常常會因為潤香與理珠的姣好身材而陷入消沉。在作品中時常更換髮型。
: 因同与成幸赶不上{{link-ja|末班电车|終電}}，两人以姐弟之名在旅馆中同寝。從此開始對成幸的好感漸漸增加，但本身還是沒有自覺。在一次混乱的舞台剧中<ref>第67及68话</ref>，文乃误认为头套里的人是来救场的小美浪愛澄，隔着头套与成幸在舞台上当众亲吻。
: 在第1回角色人气投票順位第2名(4194票)。

; {{nihongo|緒方理珠|緒方 理珠（おがた りず）|聲：[[富田美憂|富田美憂]]<ref name="character"/>}}
: 胸部大、身材嬌小的[[眼鏡角色|眼鏡女孩]]，擅長科目為[[理科|理科]]，逻辑思维强悍，被稱作「機關精巧的[[拇指姑娘|拇指姑娘]]」，目標是考上文科大學，卻不擅長文科。
: 家裡開[[烏龍麵|烏龍麵]]店。因此放學後會直接回家幫忙，有時也會負責外送。喜歡玩各種益智遊戲，但一旦與他人玩多人對戰類的遊戲，都會因為無法猜透對手的想法而敗北。為了了解人的感情而想學習心理學。
: 個性好勝，所以面對不拿手的文科、妖怪、黑暗、體育等情況都會強撐着否定自己不會並表示可以應付。喜欢唯我成幸，一旦吃醋或生闷气时就会鼓起包子脸。會因為其言行而感到喜悅或憂愁，在成幸與其他女生互動良好時也會因此動搖。与成幸亲吻过。
: 在第1回角色人气投票順位第4名(2421票)。

; {{nihongo|武元潤香|武元 うるか（たけもと うるか）|聲：[[鈴代紗弓|鈴代紗弓]]<ref name="character"/>}}
: 膚色黝黑的活潑女孩，擅長科目為體育，特別是游泳，被稱作「漆黑的白銀[[美人魚_(童話)|人魚公主]]」，因為志願的體育大學開始將英文成績列入入學標準，讓潤香也成為了成幸的學生。
: 是體育特招生，運動萬能。因此除了體育以外的成績都慘不忍睹。她跟成幸是中學時期的舊識，那时就暗恋對方，但是因為提到戀愛話題就會特別害羞，所以不敢主動進攻。在成幸問起喜歡的人是否是他時表示否定，原因是不想影響到他。
: 水希的崇拜對象。
: 在第1回角色人气投票順位第3名(3552票)。

; {{nihongo|桐須真冬|桐須 真冬（きりす まふゆ）|聲：[[Lynn_(配音員)|Lynn]]<ref name="character"/>}}
: 一之瀨學園的教師，理珠與文乃的首任教育員。
: 年輕時是花式溜冰選手，引退後開始對自己的人生產生懷疑，認為當初自己選擇花式溜冰的道路是浪費時間，因此十分反對理珠跟文乃的志願，認為他們應該要依照自己的才能選擇志願，所以理珠跟他很不對盤。但身為教育者還是有溫柔的一面，在林間合宿理珠迷路時有偷偷的去找過她，教人的功力連成幸都非常佩服。
:與擔任教師時的完美姿態相反，私底下生活態度邋遢，住的公寓房間內四處都是雜物與垃圾且毫無整理能力，強烈要求偶然間發現此事的成幸必須保守秘密，成幸也因為看不慣這樣的髒亂環境而會在來訪時幫忙打掃。意外地有著怕蟑螂的天然面，可以因為一隻蟑螂將房子弄成的像是被風災肆虐過後一樣，還要成幸幫忙善後。
:成幸父親的學生之一，但自認沒有達成與恩師同樣的成就。已決定明年擔任學園裡面的女子花式溜冰部教練。
:在第1回角色人气投票順位第1名(5233票)。

; {{nihongo|小美浪愛澄|小美浪 あすみ（こみなみ あすみ）|聲：[[朝日奈丸佳|朝日奈丸佳]]<ref name="character"/>}}
: 成幸在暑期補習班認識的重考生，為了要考上國立大學的醫科而選擇重考，但理科成績慘不忍睹。身體嬌小，也因此常被誤認為是中學生，被誤認的時候會很生氣。
: 想成為醫生的動機是想要繼承家裡的診所。為了存大學學費及補習班的費用在女僕咖啡廳打工，並且會在空閒時間接受成幸的指導，某次不小心被父親發現穿女僕裝的樣子，情急之下說是成幸的興趣並謊稱兩人是男女朋友。害怕齧齒類動物。
: 在第1回角色人气投票順位第5名(2323票)。

=== 親人 ===
; {{nihongo|唯我水希|唯我 水希（ゆいが みずき）|聲：[[高野麻里佳|高野麻里佳]]}}
: 成幸的大妹，負責處理家中大部分家事。是兄控，在兄長帶兩個女孩子回家進行教學時情緒一度十分激動。

; {{nihongo|唯我葉月・唯我和樹|唯我 葉月・和樹（ゆいが はづき・かずき）|聲：[[峯田茉優|峯田茉優]]、[[藤原夏海|藤原夏海]]}}
: 成幸的小妹及弟弟，雙胞胎，葉月為姐姐，和樹為弟弟。常詢問跟成幸一起回家的女性會不會和成幸結婚，也對於水希有戀兄情節感到無奈。

; {{nihongo|唯我花枝|唯我 花枝（ゆいが はなえ）|聲：[[川澄綾子|川澄綾子]]}}
: 成幸的母親，個性坦率的好人，為負擔家計而四處打工。

; {{nihongo|成幸的父親|成幸の父|聲：[[濱田賢二|濱田賢二]]}}
: 故人，本作故事開始的五年前因病過世。生前對兒子的教導深深影響了成幸的人格。
: 生前是教導人員，培養出許多在各行各業皆能有用處的學生，桐須真冬也是其學生之一。

; {{nihongo|桐須美春|桐須 美春（きりす みはる）}}
: 桐須真冬的妹妹，就讀東景女子大學，是一位完美主義者和花樣滑冰界的參賽資格者，一直希望姐姐能重回花樣滑冰界。由於一直就讀女子學校，因而不擅長應付男性，被成幸稱呼名字時就因此激烈動搖過。教育風格十分嚴格，家裡的狗因此忍受不了美春的斯巴達教育而逃跑了。
: 同時是一位姐控，初次與成幸相遇便把他當是變態，更誤會他們是戀人關係，打算誘惑成幸讓姐姐忘記成幸，一直失敗但未放棄。

; {{nihongo|理珠的父親|理珠の父}}
: 經營著烏龍麵店「緒方烏龍麵」，是個笨蛋父親，異常的疼愛女兒，因此在女兒對他很冷淡或做出鄙視的態度時會變得十分難過。因為過度擔心女兒而把負責教學的成幸當成了必須要警戒的對象。在店內負責烹飪，不擅長數字，因此把店中金錢管理全交給女兒負責。

; {{nihongo|古橋零侍|古橋 零侍（ふるはし れいじ）}}
: 文乃的父親，弓弦羽大學的數學教授。熱愛數學，把一切事物包括個人情緒都用數學來解釋，對來見習的理珠表現出的數學天分十分讚賞，但對女兒文乃則表現得漠不關心，並強烈反對她放棄文科轉讀天文學的想法。其強勢的態度是文乃因此對男性保持一定距離的主因。
: 認為亡妻的才能是自己無法相較的而對自己評價很低，對女兒的文學才能十分肯定，因此以「放棄自己的天賦是對世界的損失」的理論強制女兒選擇文學之路。其實因為在妻子亡故後不久情緒難以平復而因故打了女兒一巴掌，自此便和文乃有些疏遠，雖然仍很關心對方卻不知如何彌補關係，只能故作冷酷的在一旁默默守護。

; {{nihongo|古橋靜流|古橋 静流（ふるはし しずる）}}
: 文乃的母親，故人。生前是大學的數學教授，被認為是能解決[[千禧年大獎難題|千禧年大獎難題]]的天才數學家。
