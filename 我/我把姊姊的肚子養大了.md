《'''我把姊姊的肚子養大了'''》（{{lang-ja|姉のおなかをふくらませるのは僕}}）是[[坂井音太|坂井音太]]原作，[[恩田千路|恩田千路]]繪畫的[[日本|日本]][[漫畫|漫畫]]作品<ref>{{cite web|script-title=ja:日常料理におねショタ風味を添えて「姉のおなかをふくらませるのは僕」1巻|url=http://natalie.mu/comic/news/151199|website=[[Natalie_(網站)|Natalie]]|accessdate=2015年10月19日|language=2015年6月19日}}</ref>，[[秋田書店|秋田書店]]出版，最初在《別冊Young Champion》的2014年11號上刊登<ref>{{cite web|script-title=ja:「姉のおなかをふくらませるのは僕」YCに出張掲載、「センセ。」は連載再開|url=http://natalie.mu/comic/news/151563|website=[[Natalie_(網站)|Natalie]]|accessdate=2015年10月19日|language=2015年6月23日}}</ref>。

== 故事簡介 ==
描述一對沒有血緣關係的姊弟，高中生姊姊.京子以及小學生弟弟.忍，相互扶持的餐桌日常。
== 登場角色 ==
; {{nihongo|市川忍|市川 忍（いちかわ しのぶ）}}
:小學男生，擅長料理。
; {{nihongo|市川京子|市川 京子（いちかわ きょうこ）}}
:忍的姊姊，忍的母親再婚對象的女兒，擅長料理的高中生。
; {{nihongo|市川梨由子|市川 梨由子（いちかわ りゆこ）}}
:京子的親姑姑，是市川姊弟的監護人。
; {{nihongo|高木陽子|高木 陽子（たかぎ ようこ）}}
:京子的同學。
