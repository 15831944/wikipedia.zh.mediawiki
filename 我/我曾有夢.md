《'''我曾有夢'''》（{{lang-en|'''I Dreamed a Dream'''}}）是一首來自音樂劇《[[悲慘世界_(音樂劇)|悲慘世界]]》的歌曲<ref name="ibdb">{{cite web |url = http://www.ibdb.com/production.php?id=4443 |title = ''Les Misérables'' at IBDB.com |accessdate = 2009-04-29}}</ref>，為劇中角色法天妮（Fantine）於第一幕演唱的獨唱歌曲。這首歌由[[克勞德-米歇爾·勳伯格|克勞德-米歇爾·勳伯格]]作曲，由{{tsl|en|John Cameron||John Cameron}}負責[[管弦樂|管弦樂]]編曲。歌曲出自[[阿兰·鲍伯利|阿兰·鲍伯利]]所寫的[[法語|法語]][[歌劇劇本|歌劇劇本]]，後由{{tsl|en|Herbert Kretzmer||Herbert Kretzmer}}翻譯至[[英語|英語]]。

這首歌曲是一首哀歌，演唱角色法天妮當時十分痛苦、垂死和很貧窮，並回想起以往的美好時光。

== 歷史 ==
《我曾有夢》原出於1980年於[[巴黎|巴黎]]公開的作品，當時名為《{{lang|fr|J'avais rêvé d'une autre vie}}》（法語，意譯「我曾夢過另一人生」），由{{tsl|en|Rose Laurens||Rose Laurens}}演唱。首個[[英語|英語]]的《孤星淚》於1985年10月在[[倫敦|倫敦]][[西區劇院|西區劇院]]公演，法天妮一角由{{tsl|en|Patti LuPone||Patti LuPone}}飾演。<ref>{{cite web
  |url = http://www.tcm.com/tcmdb/participant.jsp?spid=117747&apid=0
  |title = Patti LuPone: Biography|accessdate=2009-04-29}}</ref>
她在1993年發行的專輯《Patti LuPone Live!》收錄了這首歌<ref>{{cite web
  |url = http://www.billboard.com/bbcom/discography/index.jsp?pid=32625&aid=144802
  |title = Patti LuPone Live! Billboard.com |accessdate=2009-05-06}} {{Dead link|date=2010-10|bot=H3llBot}}</ref>。

這套音樂劇於1987年3月在[[紐約|紐約]][[百老匯劇院|百老匯劇院]]上演，法天妮一角由{{tsl|en|Randy Graff||Randy Graff}}飾演。<ref name="ibdb" />其後在1988年的美國巡迴演出以及1990年於百老匯的公演，該角色由{{tsl|en|Laurie Beechman||Laurie Beechman}}飾演。她的專輯《Listen to My Heart》也收錄了這首歌<ref>{{cite web
  |url = http://www.billboard.com/bbcom/discography/index.jsp?pid=22620&aid=57892
  |title = Laurie Beechman Billboard.com|accessdate=2009-05-06}} {{Dead link|date=2010-10|bot=H3llBot}}</ref>。在《完整交響樂錄音》（Complete Symphonic Recording），這首歌由{{tsl|en|Debra Byrne||Debra Byrne}}演唱。在1995發行的《十周年音樂會錄音》（Tenth Anniversary Concert Recording），這首歌由{{tsl|en|Ruthie Henshall||Ruthie Henshall}}演唱。在百老匯的重演，法天妮曾分別由{{tsl|en|Daphne Rubin-Vega||Daphne Rubin-Vega}}（2006年至2007年）、[[莉亞·莎隆嘉|莉亞·莎隆嘉]]（2007年）和{{tsl|en|Judy Kuhn||Judy Kuhn}}（2007年至2008年）飾演。

這套音樂劇（以及這首歌曲）曾被翻譯為21種語言，包括[[日語|日語]]、[[希伯來語|希伯來語]]、[[冰島語|冰島語]]、[[挪威語|挪威語]]、[[捷克語|捷克語]]、[[波蘭語|波蘭語]]、[[愛沙尼亞語|愛沙尼亞語]]等等；此外也有31個不同劇組的錄音版本<ref>{{cite web|url=http://www.lesmis.com/pages/about/facts_figures_1.htm |title=Translations and Cast Recordings. LesMis.com |accessdate=2009-05-06 |deadurl=yes |archiveurl=https://web.archive.org/web/20090309101814/http://lesmis.com/pages/about/facts_figures_1.htm |archivedate=2009-03-09 }}</ref>。倫敦劇組的版本在英國被認證為三白金，銷量超過90萬張；於美國被認證為白金，銷量更超過百萬。百老匯劇組的版本在美國獲得四白金，銷量超過400萬。其餘四個版本都被認證為金<ref>{{cite web |url=http://www.riaa.com/goldandplatinumdata.php?table=SEARCH_RESULTS |title=Gold and Platinum RIAA.com |accessdate=2009-11-19 |deadurl=yes |archiveurl=https://www.webcitation.org/5aWxoHiRP?url=http://www.riaa.com/goldandplatinumdata.php?table=SEARCH_RESULTS |archivedate=2008-09-02 }}</ref>。

2009年4月，[[蘇格蘭|蘇格蘭]]歌手[[蘇珊·波伊爾|蘇珊·波伊爾]]在英國選秀節目《[[英國達人|英國達人]]》第三季中演唱了《我曾有夢》，令這首歌再受到注目。波伊爾的表現使節目的三位评审[[皮爾斯·摩根|皮爾斯·摩根]]、{{le|Amanda Holden}}和[[西蒙·高維爾|西蒙·高維爾]]一致同意她進入下一回合，摩根說這是他在節目的三年中裏給的最大的「Yes」（節目中以Yes表示支持參賽者晉級）。波伊爾的偶像[[伊蓮·佩姬|伊蓮·佩姬]]其後也表示有興趣与她合唱<ref name="paige">{{cite news |url = http://www.guardian.co.uk/media/2009/apr/19/susan-boyle-duet-elaine-paige |title=Reality TV star Susan Boyle set for duet with idol Elaine Paige |work=The Guardian |author=Caroline Davies |coauthors = Paul Kelbie |date=2009-04-19 |accessdate=2009-04-19 | location=London}}</ref>。這集节目的收視率十分高，波伊爾的演唱片段在[[YouTube|YouTube]]上的觀看次數在上載首月就已達幾百萬次<ref>{{cite news|url=http://www.guardian.co.uk/media/2009/apr/14/britains-got-talent-tv-ratings |title="TV ratings: Britain's Got Talent hits high note" Guardian.co.uk|accessdate=2009-04-29 | work=The Guardian | location=London | first=Leigh | last=Holmwood | date=2009-04-14}}</ref><ref>{{cite news|url=http://blogs.reuters.com/fanfare/2009/04/20/susan-boyle-breaks-past-100-million-online-views/ |title="Susan Boyle breaks past 100 million online views" Reuters.com |accessdate=2009-04-26 |date=2009-04-20 |deadurl=yes |archiveurl=https://web.archive.org/web/20090426034430/http://blogs.reuters.com/fanfare/2009/04/20/susan-boyle-breaks-past-100-million-online-views/ |archivedate=2009-04-26 }}</ref>。在同年5月30日的決賽中，她再次演唱《我曾有夢》，最終取得第2位，敗給跳舞組合{{link-en|Diversity|Diversity (dance troupe)}}<ref>[http://news.bbc.co.uk/2/hi/entertainment/8075510.stm "Dancers beat Boyle in talent show"] BBC.co.uk. [2009-05-30].</ref>。現今一般認為，這首歌是她走上明星之路的起始點，亦有助提升其專輯《[[我曾有夢_(專輯)|我曾有夢]]》的銷量。该專輯是英國售卖最快的出道專輯之一，该專輯在美國的首周銷量也高達70.1萬張，打破個人女藝人的出道銷售紀錄。

波伊爾的演出在[[英國獨立電視台|英國獨立電視台]]於4月播映後，倫敦劇組中由LuPone演唱的版本下載次數同时增加，成功進入英國和美國多個音樂排行榜。該歌曲分別於美國《[[告示牌_(雜誌)|告示牌]]》的熱門下載歌曲榜（Hot Digital Songs）和[[告示牌百強單曲榜|告示牌百強單曲榜]]上取得第61位和第20位。<ref>{{cite web |url = http://www.billboard.com/bbcom/esearch/searchResult.jsp?exp=y&Ntt=%22i+dreamed+a+dream%22&Ntk=Keyword&Ntx=mode+matchallpartial&nor=10&an=bbcom&N=37&Ns=FORMATTED_DATE |title = LuPone 1985 version (US) Billboard.com |accessdate = 2009-04-26}} {{Dead link|date=2010-10|bot=H3llBot}}</ref>此外，同年4月25日，該歌曲亦於[[英國單曲排行榜|英國單曲排行榜]]取得第45位<ref>{{cite web|url=http://www.chartstats.com/songinfo.php?id=34325 |title=LuPone 1985 version (UK) Chartstats.com |accessdate=2009-04-26 |archiveurl=https://web.archive.org/web/20090501174256/http://www.chartstats.com/songinfo.php?id=34325 |archivedate=2009-05-01 |deadurl=yes }}</ref>。
