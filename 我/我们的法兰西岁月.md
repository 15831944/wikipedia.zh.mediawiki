《'''我们的法兰西岁月'''》，2012年6月26日上映于[[中国中央电视台综合频道|中国中央电视台综合频道]]的史诗剧，讲述了1920年开始，[[周恩来|周恩来]]、[[邓小平|邓小平]]、[[陈毅|陈毅]]、[[聂荣臻|聂荣臻]]、[[李富春|李富春]]、[[李维汉|李维汉]]、[[李立三|李立三]]、[[蔡畅|蔡畅]]、[[萧三|萧三]]、[[蔡和森|蔡和森]]、[[王若飞|王若飞]]、[[向警予|向警予]]等青年学子远赴法国留学的故事。

== 剧情 ==
1912年，[[中华民国|中华民国]]建立，开启了[[中国历史|中国历史]]的新纪元，此时，[[五四运动|五四运动]]爆发，青年学子目睹[[帝国主义|帝国主义]]在中华大地肆意侵略，饱览兴亡，决计寻求救国救民，[[李石曾|李石曾]]、[[吴玉章|吴玉章]]、[[吴稚晖|吴稚晖]]、[[张继|张继]]等在[[北京|北京]]发起组织“[[留法俭学会|留法俭学会]]”，[[中华民国教育总长|中华民国教育总长]][[蔡元培|蔡元培]]先生对此极力赞成，后来[[袁世凯|袁世凯]]篡夺民国政权，极力配合帝国主义开始压制国内的留学活动，蔡元培亲自远赴法国，支持留法学生的学习，[[周恩来|周恩来]]、[[邓小平|邓小平]]、[[陈毅|陈毅]]、[[聂荣臻|聂荣臻]]、[[李富春|李富春]]、[[李维汉|李维汉]]、[[李立三|李立三]]、[[蔡畅|蔡畅]]、[[傅钟|傅钟]]、[[何长工|何长工]]、[[萧三|萧三]]、[[赵世炎|赵世炎]]、[[蔡和森|蔡和森]]、[[王若飞|王若飞]]、[[陈延年|陈延年]]、[[陈乔年|陈乔年]]、[[向警予|向警予]]等人正是此时借此机会得到了在法国学习的机会，他们在法国见到了[[欧洲|欧洲]][[工业革命|工业革命]]和[[第二次工业革命|第二次工业革命]]的成就，看到了世界格局和形式的变化，认识到只有[[科学|科学]]和[[民主|民主]]才能救中国，为他们以后的革命事业指明了方向。

== 花絮 ==
因该片涉及到[[中国共产党|中国共产党]]领导人的形象问题，审核颇为艰辛，经过[[中央文献研究室|中央文献研究室]]、[[共青团中央|共青团中央]]、[[江苏省委宣传部|江苏省委宣传部]]、剧中中国共产党家属后人的层层审核才予以通过。<ref name="fish " >{{cite web |url=http://yule.sohu.com/20120626/n346534302.shtml |title=《我们的法兰西岁月》开播 揭秘伟人史诗传奇 |accessdate=2012年6月26日 |publisher= 搜狐网|work= |language =zh-cn}}</ref>

剧中，周恩来被誉为“最具绅士气质”，邓小平被誉为“最热血的少年”，赵世炎被誉为“最具抱负情怀”，蔡和森被誉为“最刚烈的革命先驱”。<ref name="fish " />

该剧最初为40集，后删节大量内容到33集，后又删除部分内容到31集。<ref name="bull " />

邓小平的小女儿[[邓榕|邓榕]]说，“1993年1月3日，父亲邓小平在[[浙江|浙江]][[杭州|杭州]]给他的几位孙辈写了一封信。信中道，‘对中国的责任，我已经交卷了，就看你们的了。我16岁时还没有你们的文化水平，没有你们那么多的现代知识，是靠自己学，在实际工作中学，自己锻炼出来的。十六七岁在法国一呆就是五年，那时连话都不懂，还不是靠锻炼。你们要学点本事为国家做贡献。大本事没有，小本事、中本事总要靠自己去锻炼。’”这封信的话正是该片的教育意义。<ref name="bull " >{{cite web |url=http://ent.qq.com/a/20120625/000563.htm |title=《我们的法兰西岁月》将播 追述90年前中国偶像 |accessdate=2012年6月25日 |publisher=腾讯网 |work= |language =zh-cn}}</ref>

为拍摄此片，摄制组远赴[[法国|法国]]拍摄了一个月。<ref>{{cite web |url=http://ent.163.com/12/0625/02/84QF7DBF00032DGD.html |title= 《我们的法兰西岁月》明登央一 聚焦留法先驱|accessdate= 2012-06-25 |publisher=网易 |work= |language =zh-cn}}</ref> 

该剧的原则是“大事不虚，小事不拘”，大部分真实，合理想象。<ref>{{cite web |url=http://ent.qq.com/a/20120625/000039.htm |title=《我们的法兰西岁月》将播 四年磨一剑还原历史 |accessdate= 2012年6月25日|publisher=腾讯网 |work= |language =zh-cn}}</ref>

该剧是[[中国共产党第十八次全国代表大会|中国共产党第十八次全国代表大会]]的献礼剧。<ref>{{cite web |url=http://ent.ifeng.com/zz/detail_2012_06/26/15568975_0.shtml |title= 《我们的法兰西岁月》|accessdate=2012年6月26日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>

2012年11月1日起在[[央视八套|央视八套]]开始二轮播出。
