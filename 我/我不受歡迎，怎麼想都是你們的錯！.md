《'''我不受歡迎，怎麼想都是你們的錯！'''》（{{lang-ja|私がモテないのはどう考えてもお前らが悪い!}}），簡稱《我不受歡迎》（{{lang|ja|ワタモテ、私モテ}}），為日本漫畫創作雙人組合[[谷川尼可|谷川尼可]]（{{lang|ja|谷川ニコ}}）於2011年8月4日在網路漫畫期刊網站《[[GANGAN_ONLINE|GANGAN ONLINE]]》發表的漫畫作品，並在2012年1月21日首次推出[[單行本|單行本]]<ref>{{Cite web|url=http://www.square-enix.com/jp/magazine/ganganonline/comic/watashiga/#fair|script-title=ja:『私がモテないのはどう考えてもお前らが悪い!』1巻発売記念フェア開催!! |publisher=[[史克威爾艾尼克斯|史克威爾艾尼克斯]]|date=|accessdate=2012-01-29|language=ja}}</ref>。

[[電視動畫|電視動畫]]全12話，2013年7月至9月在[[東京電視網|東京電視網]]首播。

== 內容概要 ==
作品為谷川尼可以用來形容不受歡迎、沒戀愛經驗女性的日本[[網路用語|網路用語]]「[[喪女|喪女]]」一詞為題材，所創作的[[單元劇|單元劇]]形式漫畫。其契機來自於谷川尼可的編輯在瀏覽一些[[網路論壇|網路論壇]]上的喪女討論板後，所作出的提議<ref name="itmedia2">{{Cite web|url=http://nlab.itmedia.co.jp/nl/articles/1206/27/news019_2.html |script-title=ja:喪女が国境を越えた　4chan民が愛する漫画「私がモテないのはどう考えてもお前らが悪い！」(2/3) |publisher=Itmedia.com |accessdate=2013-02-08|language=ja}}</ref>。而漫畫標題則是作者基於該時期多數的日本漫畫、[[輕小說|輕小說]]都採用相當長的名稱傾向，所想出的決定<ref name="itmedia2" />。

劇情描述著缺乏社交技巧的女高中生──黑木智子，在每一日上學時面對周遭人物忽略、漠視時，所作出的各種事蹟<ref>{{Cite web|url=http://www.square-enix.com/jp/magazine/ganganonline/comic/watashiga/index.html|script-title=ja:私がモテないのはどう考えてもお前らが悪い！|publisher=[[史克威爾艾尼克斯|史克威爾艾尼克斯]]|date=|accessdate=2012-01-29|language=ja}}</ref>。在上高中之前，智子便已在多款少女傾向的[[戀愛遊戲|戀愛遊戲]]裏揣摩各種情况；并且幻想著當就讀高中時就能如同遊戲般地会受欢迎。可惜上高中後事与愿违，過了两个月智子仍未交到任何朋友。而在面臨现实情況之下，智子開始試著做出能让自己变得受欢迎的各種無厘頭行动。

== 登場角色 ==
因谷川尼可為[[日本職棒|日本職棒]]球隊[[千葉羅德海洋|千葉羅德海洋]]的支持者，在設定包括主角[[黑木知宏|黑木]]智子等其他角色的姓氏時，皆引用了曾效力於該隊的球員，<ref>{{Cite web |url=http://www.otanew.com/news_aiXEk7gLzu.html |script-title=ja:新刊レビュー！「私がモテないのはどう考えてもお前らが悪い！」第2巻-痛々しすぎる…… |work=OTA New |date=2012-05-27 |accessdate=2015-08-15 |language=ja |deadurl=yes |archiveurl=https://web.archive.org/web/20150924062326/http://www.otanew.com/news_aiXEk7gLzu.html |archivedate=2015年9月24日 |df= }}</ref>像是[[成瀨善久|成瀨]]、[[今江敏晃|今江]]等。

=== 主人公 ===
; {{nihongo|黑木智子|黒木 智子（くろき ともこ）|聲：[[橘田泉|橘田泉]]}}
: 本作主角，為就讀{{nihongo|「原宿教育學園幕張秀英高中」|原宿教育学園幕張秀英高等学校}}的女學生。外貌常以帶著撩亂長髮、疲倦[[黑眼圈|黑眼袋]]的模樣見人。
: 第1回登场。平時的兴趣為觀賞動漫，或遊玩成人題材取向的情色電子遊戲，對於流行方面的事物則較不敏感。
: 不擅長與人交談，和其他人進行對談時常會沒來由地緊張不知所措。
: 本人雖渴望能獲得班上周遭同儕的接納及認同，但一直被冷漠、忽視，因而每每皆獨自一人行動。
: 內心為了安慰時常不被他人注意的自己，而常常對於他人事物有著較反面、憤世嫉俗的批評想法。
: 黑木智子的初期外型設定上為{{nihongo|「不可愛、[[駝背|駝背]]、[[三白眼|三白眼]]、[[平胸|平胸]]」|かわいくなく、背が高くて、三白眼で、胸もない}}模樣的容貌，之後受到漫畫編輯部方面{{nihongo|「希望外型能較可愛」|もっとかわいくしてほしい}}的意見而做了修正<ref name="Weekly Playboy">摘自《[[週刊Playboy|週刊Playboy]]》2013年1月14日號的訪談</ref>。

=== 智子的親人 ===
; {{nihongo|黑木智貴|黒木 智貴（くろき ともき）|聲：[[中村悠一|中村悠一]]、幼少期：{{link-ja|芹亞希子|芹亜希子}}}}
: 黑木智子的弟弟，第1回初登场时是初中三年级生，在學校是足球社的社員。
: 與姊姊性格全然不同，並擁有許多朋友。時常對於姊姊突來的不知所云行為沒轍。
: 在升上高中後，與姊姊就讀同一所高中。小時候很喜歡姐姐。
; 
; {{nihongo|智子的母親|智子の母親|聲：[[早水理沙|早水理沙]]}}
: 家庭主婦，不時被智子一些搞不清情況的行為給惹惱。

; {{nihongo|智子的父親|智子の父親|聲：[[岩崎了|岩崎了]]}}
: 登場次數不多，在此作品裡是刻意未將臉孔給顯露出來的角色。

; {{nihongo|希心/小希|きこ/きーちゃん|聲：[[釘宮理惠|釘宮理惠]]}}
: 智子的表妹。因聽信智子常在她面前吹噓的事蹟而崇拜智子。但自從發現智子跟小孩玩牌時出老千後開始另眼看待。

=== 初中同学 ===
; {{nihongo|成瀨優|成瀬 優（なるせ ゆう）|聲：[[花澤香菜|花澤香菜]]}}
: 漫画第3回登场。黑木智子就讀中學時的同學，畢業後就讀與智子不同的高校，是少數仍與智子聯繫的友人。上高校時成為較懂得打扮的少女，並有一名男友。
: 喜歡親切地稱呼智子為「{{lang|ja|もこっち}}」，是改自於{{nihongo|「小智子」|ともこちゃん}}的叫法。智子平時稱呼她為{{nihongo|「小優」|ゆうちゃん}}，但有時會在心裡直接稱呼為「[[Bitch|Bitch]]」。
: 成瀨優的出現是基於作者受到漫畫編輯部「黑木智子在故事裡一個朋友都沒有太過可憐」的意見，而創造的角色<ref name="Weekly Playboy" />。
; {{nihongo|小宮山琴美|小宮山 琴美（こみやま ことみ）|聲：[[水橋香織|水橋香織]]}}
: 登场于漫画46回。由衍生作品《我的朋友不受歡迎，怎麼想都是你們的錯。》登場的角色，是智子在國中透過成瀨優認識的同學、在高中時也恰巧與智子就讀同間學校。
: 雖在國中期間與成瀨及智子是常一同行動的三人組，但與智子彼此的關係為互相敵視對方；倆人心中都是只想與成瀨優交往作朋友。
: 私下為日職球隊[[千葉羅德海洋|千葉羅德海洋]]的死忠球迷，且對便服的打扮風格有著較異於常人的眼光。另外暗戀著智子的弟弟智貴。
: 高三成为智子的同班同学。

; {{Nihongo|川本|川本（かわもと）}}
: 智子中学時的同班同学，戴眼镜的胖女生。高中时智子再次遇到小宫山的时候，错把小宫山当作川本。

=== 高中同级生 ===
; {{nihongo|根元陽菜|根元 陽菜（ねもと ひな）|聲：[[黑瀨優子|黑瀨優子]]}}
: 漫画第4回登场。高一、高二、高三期間皆與智子就讀同一班級的女學生，有著活潑的個性。
: 是少數會主動向智子交談的角色，不過智子對她的粗略印象是把她姓氏「根元」的拼音「{{lang|ja|ねもと}}」中的「{{lang|ja|ねも}}」、與《[[機動戰士Z_鋼彈|機動戰士Z 鋼彈]]》裡登場的量產型MS{{nihongo|「尼摩」|ネモ}}聯想在一起。
: 自从升上高中二年级之后，就常常会主动找小丧说话，是小丧早期在学校少数的说话对象。虽然一直跟着现充团体活动，不过其实是以成为声优为目标的动画宅，为了维持在学校的生活而隐藏了本性。在小丧发现了她的声优志愿之后，两人的关系也有了些变化。虽然阳菜说小丧可以私下来找自己，两个人讨论动画。但是因为阳菜喜欢日常类的轻松作品，而小丧较倾向于有打斗和杀伤的作品，所以目前都聊不太起来。另外虽然小丧没有注意到，不过在高中的入学考试中，小丧曾向阳菜打过招呼。

; {{nihongo|田村百合|田村 ゆり（たむら ゆり）}}
: 漫画69回登场。智子班上的女同學。平常十分的沉默，话不多、反应也很淡，下课的时间大多都在一个人用随身听听音乐。
: 本来和朋友田中真子约好要一起参加修学旅行，真子却因为无法拒绝南的邀请而背违了和她的约定，百合在分组时为此和真子吵了一架，最终因此而和小丧同组。
: 有著較成熟的言行舉止，心中雖認定智子是個會作傻事的人仍願意主動試著與她相處。虽然是不太主动与人交流的类型，但是在修学旅行中感受到这样与小丧和吉田同行也是一种缘分，所以结束旅行之后，她也会主动去找小丧打招呼、并邀请小丧与自己一同吃饭，现在已经成为了小丧在学校重要的朋友之一。
:高三与智子、真子同班。

; {{nihongo|吉田茉咲|吉田 茉咲}}
: 登场于漫画72回。一名有時會蹺課、在智子班上的不良女學生，於校園旅行被分配與智子同一組。
: 在旅行期間雖常被智子的一些言行給惹惱，但在智子遭遇麻煩時會主動出面替她解圍。
: 個性直接喜形於色，特別喜歡可愛的東西，喜歡的東西被批評時會發怒直接動手，即使是朋友也是照扁無誤。

; {{nihongo|內笑美莉|内 笑美莉（うち えみり）}}
: 登场于漫画59回。表情像是[[表情符號_(圖片)|笑臉符號]]「☺」、有著眾多朋友的智子班上女同學。智子对她的外号是颜文字。
: 在校園旅行前純粹因人數不好分配而先將就與智子同一組，於旅行期間都是離開智子、與其她親友一同行動。高三与智子不再同班。
: 误认为智子喜欢女生，一开始感到厌恶，后来渐渐突变成了喜欢的感情。

; {{nihongo|田中真子|田中 真子}}
: 登场于漫画69回。田村的摯友，外觀爲短髮帶著雀斑的少女。
: 修学旅行原本要和百合同组，但是因为受到南的邀请而忽然变卦，因此和百合吵架，并造成百合在修学旅行落单。事后对于背叛百合的事非常后悔，修学旅行期间也尝试向百合道歉，最终也得到了百合的原谅。旅行结束之后，因为百合开始找小丧一起吃饭，真子也与小丧有了接触，最终在一场误会之中和小丧成为了朋友。

; {{nihongo|岡田茜|岡田 茜（おかだ あかね）|聲：[[朝井彩加|朝井彩加]]}}
: 登场于漫画第4回。在頭頂上紮髮作為個人打扮的智子班上女學生，常與根元及清田等人行動且喜好籃球。正义感强。与智子从高一到高三都是同班。

; {{nihongo|清田良典|清田 良典（きよた よしのり）|聲：[[高梨謙吾|高梨謙吾]]}}
: 登场于漫画第4回。戴著眼鏡、在智子班上頗有人緣的男性學生。高一到高三与智子同班。

; {{Nihongo|加藤明日香|加藤 明日香（かとう あすか）}}
: 登场于漫画第71回。容貌端丽的长发女生。高二高三与智子同班。喜欢化妆和收集小饰品，对待智子友善，帮智子化过妆。

; 南
: 登场于漫画第69回。虎牙女生。高二高三与智子同班。是田中真子的朋友，与田村百合关系不好。不时会说智子的坏话，被智子起了“牙子”的外号。升入高三时除了田中真子以外的朋友都转去别班。

; 伊藤
: 登场于漫画第48回。小宫山琴美的朋友，吹奏乐部的部员。高一高二与小宫山同班，高三与小宫山、智子同班。曾多次见过小宫山暴走的言行，不过仍然愿意继续和小宫山当朋友。和阳菜一样在高中的入学考试中和小丧见过面，并被小丧打了招呼。

; {{nihongo|初芝|初芝（はつしば）|聲：[[杉田智和|杉田智和]]}}
: 別班的男同學，因為上次沒上美術課而被老師指示放學後要留下來補畫，不過智子事後看見初芝把自己畫得很可愛不禁心動。
: 是與小宮山琴美同班的學生。高三分班后与智子同班。

; {{Nihongo|二木|二木（ふたき）}}
: 眼睛是两个大黑点的女生。高三与智子同班。体育很好，在体育祭夺得骑马战的冠军。

; {{Nihongo|柿沼|柿沼（かきぬま）}}
: 智子二年级时，班上的宅男团体的一人。在体育祭时曾与阳菜组队参加两人三脚的比赛，当时的柿沼因为害羞而抬不起脸来。

; {{Nihongo|铃木|鈴木（すずき）|聲：[[岩濑修平|岩濑修平]]}}
: 高中一年级时，时常和阳菜、冈田以及清田四个人一起行动。二年级时与其他三人分到了不同班。

; 班长
: 女生，智子二年级时的班长。在体育祭决定比赛项目时，曾严厉的要求小丧认真参加，不过并没有去管同样趴在桌上睡觉的吉田。

; 班长的朋友

; 宫崎

== 迴響 ==
作品的單行本在2012年6月銷售量突破40萬部<ref name="itmedia">{{Cite web|url=http://nlab.itmedia.co.jp/nl/articles/1206/27/news019.html |script-title=ja:喪女が国境を越えた　4chan民が愛する漫画「私がモテないのはどう考えてもお前らが悪い！」(1/3) |publisher=Itmedia.com |accessdate=2013-02-08|language=ja}}</ref>，之後在同年12月達到累積突破100萬本的成績<ref name="Natalie" />。

而漫畫本身的題材在日本海外的[[ACG|ACG]]論壇[[4chan|4chan]]上，也得到眾多迴響<ref name="itmedia" /><ref>{{Cite web|url=http://kotaku.com/5880409/4chan-used-to-promote-a-japanese-manga-well-sorta |title=4Chan Used to Promote a Japanese Manga |publisher=Kotaku.com |accessdate=2013-02-08|language=en}}</ref>。主角黑木智子的悲慘遭遇，引起許多高中女生及有相同經驗者的共鳴；谷川尼可的男性成員谷川Ikko（{{lang|ja|谷川イッコ}}）感同身受，說「這些內容都沒有我的高中生活悲慘」<ref name="ChinaTimes">{{Cite web|url=http://www.chinatimes.com/newspapers/20130422000605-260115|author=江家華|language=zh-tw|title=喪女沒人愛 漫畫狂銷百萬冊|publisher=[[中國時報|中國時報]]|date=2013-04-22|accessdate=2014-05-11}}</ref>。

另在2012年由[[娜塔莉_(網站)|娜塔莉]]網站舉辦、由漫畫編輯者來票選的{{nihongo|「Manga秋100」|マンガ秋100}}作品競賽中，此漫畫獲得第2名的成績<ref>{{Cite web|url=http://comic-award.natalie.mu/result/#main|script-title=ja:現役マンガ編集者176人が年度折り返しの秋に選ぶ、この1年間に刺激を受けたマンガBest100。略して 第1回「マンガ秋100」|publisher=Comic-award|accessdate=2013-02-08|deadurl=yes|archiveurl=https://web.archive.org/web/20130118033505/http://comic-award.natalie.mu/result/#main|archivedate=2013-01-18}}</ref>。

=== 網路廣播 ===
由此漫畫作品改編的廣播劇{{nihongo|《我不受歡迎RADIO》|ワタモテRADIO}}，在網路廣播電台[[HiBiKi_Radio_Station|HiBiKi Radio Station]]上於2013年7月16日起自2014年3月18日每週二播放<ref name="2IS">{{Cite web |url=http://www.2is.jp/news/2013/06/28/14641/ |script-title=ja:『私がモテないのはどう考えてもお前らが悪い!』動画＆ラジオ配信決定 |work=2IS |date=2013-06-28 |accessdate=2013-07-08 |language=ja }}{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。

節目中朗讀主角黑木智子的劇本是由[[橘田泉|橘田泉]]擔任，為與在電視動畫版替智子配音的同一人選<ref name="2IS"/>。
