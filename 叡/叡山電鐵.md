'''叡山電鐵股份有限公司'''（{{jpn|j=叡山電鉄|hg=えいざんでんてつ|rm=Eizan Dentetsu|f=y}}），簡稱'''叡山電鐵'''，是經營從[[日本|日本]][[京都市|京都市]][[左京區|左京區]]的[[出町柳車站|出町柳車站]]到[[八瀨|八瀨]]、[[鞍馬|鞍馬]]兩地的鐵路線之業者。通稱「'''叡山電車'''」，簡稱「'''叡電'''」<ref name="ryakushou"/>。營業總部於[[京都府|京都府]]京都市左京區田中上柳町25番地3號（緊鄰出町柳車站），而登記上的總店則位於京都市左京區山端壹町田町14番地1號。

原為[[京福電氣鐵道|京福電氣鐵道]]（以下簡稱京福）在1985年設立的完全子公司，但在1991年11月[[京阪電氣鐵道|京阪電氣鐵道]]（以下簡稱京阪）成為[[大股東|大股東]]之後，從2002年3月開始，成了該公司的完全子公司（後述）。

叡山電鐵是日本[[全國登山鐵道‰會|全國登山鐵道‰會]]的成員之一<ref>[http://www.nankai.co.jp/company/news/pdf/090903_1.pdf 南海電氣鐵道・PERMIL會官方發表] {{webarchive|url=https://web.archive.org/web/20090920093724/http://www.nankai.co.jp/company/news/pdf/090903_1.pdf |date=2009-09-20 }}{{ja}}</ref>。

== 概要 ==
叡山電鐵經營[[叡山電鐵叡山本線|叡山本線]]、[[叡山電鐵鞍馬線|鞍馬線]]兩條路線，身任輸送往[[比叡山|比叡山]]、[[鞍馬|鞍馬]]的觀光客和周邊居民的角色。

這些路線曾屬[[京都電燈|京都電燈]]和[[鞍馬電氣鐵道|鞍馬電氣鐵道]]的經營範圍，在二戰時因限制企業用電而重新編排後由京福接手，但由於沿路直達市中心的公車相當充實，[[京都市電|京都市電]]遭到廢止，路線和其他鐵軌間的連結被截斷而成了孤島的狀態，使用者因此減少。雖然被預期在當時尚未開通的[[京阪鴨東線|京阪鴨東線]]完成後乘客會大幅增加，但經營狀況已經陷入危機，於1985年京福設立了叡山電鐵（分公司），將兩條路線分離。

在分離之前電鐵方面努力蒐集客源，舉辦各種活動，一次次改裝當時殘存的京福的[[八瀨遊園|八瀨遊園]]<ref>委託子公司「比叡産業」經營。</ref>。另外值得一提的是，叡山電鐵在這個時期將出町柳 - [[寶池車站|寶池]]的列車增加到每小時8班，建立「不用等待就能搭車」的印象。儘管如此，乘車人數還是直落到只有至1985年為止的尖峰時期1964年的四成而已，而且還有繼續減少的傾向。

1989年10月5日京阪鴨東線開張，在出町柳車站連結上了上述的兩條路線，睽違十一年終於和其他鐵道路線有了連結運輸的結果，從[[大阪|大阪]]前往此地的方便性向上提升，使用者也一口氣增加，再次呈現了以往活絡的情景。但是由於在1997年[[京都市營地下鐵烏丸線|京都市營地下鐵烏丸線]]延伸到了[[國際會館車站|國際會館車站]]，之後以該站為中心的公司線路網又變得更加完善，左京區岩倉地區的居民往市中心移動的交通手段開始轉移至那邊，使用者數又開始逐漸下滑。

另一方面在1997年全景電車（採用較大的窗戶讓乘客充分享受沿途的風景）「900型電車KIRARA」登場，隨著因舉辦各種活動等等積極地確保使用者，叡山電鐵於2004年開始實行在原則上將所有列車改為[[一人控制|單人電車]]等等的經費縮減策略。2005年，[[NHK|NHK]]播出了[[大河劇|大河劇]][[義經_(大河劇)|義經]]，鞍馬地區的觀光蔚為風潮，叡山電鐵展現了近年來未見過的榮景。

但是現在[[京都市|京都市]]建設局計畫了和叡山電鐵路線並走的[[鞍馬街道|鞍馬街道]]（[[京都府道38號京都廣河原美山線|京都府道38號京都廣河原美山線]]）<!--http://www.city.kyoto.lg.jp/kensetu/page/0000012793.html-->的繞行建設與拓寬整備，一旦完成的話通往鞍馬地區的大型觀光巴士將變得更容易通行，因此將來叡山電鐵未必能保有現在的興盛。

簡稱的「叡電」是來自現在叡山本線的前身[[京都電燈|京都電燈]]的叡山電鐵部，成為京福旗下路線之後也被當作叡山本線・鞍馬線的總稱而廣泛使用。而在和京福分家之後則剛好成了其公司名字面上的縮寫。<ref name="ryakushou">家電量販店的[[EDION|EDION]]的前身之一、總公司位在[[愛知縣|愛知縣]][[名古屋市|名古屋市]]的「[[EIDEN|株式会社EIDEN]]」是來自其舊名的「榮電社」，和叡山電鐵沒有任何關聯。</ref>。

分家當時，雖然京福是京阪的子公司<ref>京阪保有約40%的京福發行的股票。</ref>，但因為在車輛技術層命導入DENA500型（原阪神電鐵831型）和在阪神關連公司「[[武庫川車輛工業|武庫川車輛]]」（現在已倒閉）製造車輛等事由，[[阪神電氣鐵道|阪神電氣鐵道]]色彩濃厚，也因此在京阪下保有了京福（叡電）的獨特色彩。自1989年京阪鴨東線開業後，京福在福務層面上也慢慢展現了京阪的色彩<ref>自動售票機的地圖變成京阪樣式、制服帽子也成了所謂的「京阪戴高樂帽（法國軍用平頂帽）」。</ref>，但由於2002年京福發生了[[京福電氣鐵道越前本線列車衝突事故|越前本線列車衝突事故]]，京福的營運惡化，而將全部的持有股都賣給了京阪而成了京阪的完全子公司，自此之後，京福色彩中也開始於各種方面產生了濃厚的京阪色調<ref>除了車站、車内的廣告或告示、廣播之外，實行了車輛元件的再利用<!-- 並不只有車輛身 -->。</ref>。

在[[KANSAI_THRU_PASS|KANSAI THRU PASS]]的乘車卡上印刷的符號為'''EZ'''。

== 歷史 ==
* 1985年7月6日 - 京福全額出資設立叡山電鐵。設立時的資本金為一億日圓。
* 1986年4月1日 - 京福讓渡叡山本線・鞍馬線，開始營運。
* 1991年11月29日 - 資本金增資至二億五千萬日圓。此外，京阪從京福取得叡山電鐵60%的股份（有償）。<!--事先採取形式上的增資-->
* 2002年3月29日 - 京福擁有的股份全盤讓渡給京阪（有償），叡山電鐵成為京阪的完全子公司。
* 2004年3月1日 - 全線適用[[KANSAI_THRU_PASS|KANSAI THRU PASS]]乘車卡。
* 2008年10月19日 - 全車站導入車站編號。

叡山本線的路線顏色為象徵[[比叡山|比叡山]]森林的綠色，從[[八瀨比叡山口車站|八瀨比叡山口車站]]可經由京福的{{link-ja|鋼索線 (京福電氣鐵道)|京福電気鉄道鋼索線|索道}}、[[空中纜車|空中纜車]]到比叡山的山頂。

鞍馬線的路線顏色為象徵[[貴船|貴船]]、[[鞍馬山|鞍馬]]的[[楓葉|楓葉]]紅色。{{link-ja|鞍馬車站|鞍馬駅}}附近的[[鞍馬寺|鞍馬寺]]有非常小規模的纜車{{link-ja|鞍馬山鋼索鐵道|鞍馬山鋼索鉄道|鞍馬索道}}（經營主體為鞍馬寺）運行。

在有大批乘客時的有時會增加臨時班次，特別是在每年10月22日舉行的[[鞍馬火祭|鞍馬火祭]]是實質上唯一的大眾運輸工具，就算多接近讓每一班列車都全力運作，尖峰時刻時還是會因為鞍馬線[[二軒茶屋車站_(京都府)|二軒茶屋車站]]以北只有單線所造成的流量限制等原因，而造成攏長隊伍的情形。

另外，從2004年1月開始全路線除了一部分列車之外都是實施[[一人控制|一人控制]]。

2007年秋天開始在一部分的車站設置配有路線顏色的大型站名看板。看板上還配置了該顏色由來的象徵標記。此外，除了[[羅馬字|羅馬字]]之外也有[[韓文|韓文]]和[[中文|中文]]的標識。新型看板在開始設置時，出町柳、三宅八幡、岩倉・京都精華大前・市原、二之瀨、鞍馬等各站只有設置舊型看板，但現在每一個車站都設置了至少一面的新型看板。2008年10月19日時導入了[[車站編號|車站編號]]，新型看板及一部分的舊型看板都被標記上了車站編號。同時也導入了[[通用設計|通用設計]]的[[象形圖|象形圖]]，以[[京阪本線|京阪線]]的引導標記為基礎。

2011年春天開始追加了英語的車內自動廣播（之前只有出町柳車站）。

== 票價 ==
通過叡山本線、鞍馬線，普通票價為區間制，定期票價為以里程記費。在普通票價的情況下，以[[修學院車站|修學院車站]]、[[岩倉車站_(京都府)|岩倉車站]]、[[二軒茶屋車站_(京都府)|二軒茶屋車站]]、[[二之瀨車站|二之瀨車站]]為界線，只要搭乘路線跨過以上車站，區間數就會加一，票價也跟著改變。此外，從[[鞍馬車站|鞍馬車站]]到[[寶池車站|寶池車站]]轉乘至開往[[八瀨比叡山口車站|八瀨比叡山口車站]]，或相反的情況下區間數也會加一。也因此從[[出町柳車站|出町柳車站]]搭乘到修學院車站共2.9公里只算一區的票價，但從[[八幡前車站_(京都府)|八幡前車站]]搭乘到[[三宅八幡駅|三宅八幡駅]]的話，雖然只有1.5公里，但會計算成兩區。回數票則除了成人、兒童用（價錢普通票價的10倍，可搭乘11回）之外，也有學生折扣回數票（限成人，價錢同一般回數票，但可搭乘15次）、敬老折扣回數票（20回）。

叡山電鐵和京阪相連的雙方第一個乘車區間內（元田中 - 修學院和京阪・神宮丸太町 - 祇園四條）之間的互通有成人20日元、兒童10日元的折扣（必需事先在自動售票機購買聯絡車票。在叡山電鐵這一方自動售票機的營業時間外，可以在出町柳車站的閘門處購買。另外，如果使用KANSAI THRU PASS的話會自動打折，但必需要在30分鐘完成在出町柳車站的轉乘）。

在有人車站搭車時需在自動售票機購買車票，將車票刷過自動閘門。無人車站則幾乎都沒有閘門，可在上車時於車門旁設置的整理券發行機取得整理券，或將[[KANSAI_THRU_PASS|KANSAI THRU PASS]]共通卡片刷過讀卡機，以紀錄乘車車站。在部份無人車站也有限時的自動售票機，若在這類車站預先購入車票的話，直接上車即可。一部份的車站自動售票機在自動閘門的內側，若要使用現金則從閘門的旁邊通過進入月台再購買車票即可（無需返回自動閘門驗票）。出町柳車站、修學院車站的自動札門為全天運作。鞍馬車站、貴船口、八瀨比叡山口車站則為有站員進駐時才運作。此外京都精華大前車站、二軒屋車站，則僅在開往鞍馬方向的月台出入口設有自動閘門，於限定日期和時間才運作。

在有人車站下車時需在閘門口支付現金，或是將車票或KANSAI THRU PASS共通卡片刷過自動閘門。無人車站則要在電車駕駛座旁邊的[[車費箱|車費箱]]投入整理券及現金，或是投入車票。整理券上印有乘車車站，在下車車站時可以讓車費箱自動顯示車費。KANSAI THRU PASS共通卡片則要刷過讀卡機。定期券要向駕駛員出示。

無人車站時的搭車口為後門，下車口為前門。三門車箱的中間車門和兩節車箱編製時後方車箱的車門在有人車站以外都不會開啟。車箱費或讀卡機、整理券發行機只設置於行進方向左側的車門旁邊。

在出町柳車站、鞍馬車站的窗口及修學院車站的定期券販賣處，有出售1000日元（兒童500日元）一日無限搭乘的「1日乘車券 えぇきっぷ」。只要出示就可以於沿線的寺廟或觀光設施取得參拜費、入場費的折扣、拿到小禮物、或於沿線的餐飲店、土產店受到優待。在自動閘門或車內的讀卡機刷過之後，就會決定使用日期並印刷在該票的背面。在無人車站搭車時除了首次搭車之外不需要刷過讀卡機，但在無人車站下車時則一定要將票刷過讀卡機。

母公司的京阪一方有出售和叡電合作的企畫乘車券<ref>[http://www.keihan.co.jp/traffic/otoku/ 特惠票的介紹] - 京阪電氣鐵道{{ja}}</ref>，但在叡電一方則沒有出售，必需在京阪出町柳車站另外購買<ref>[http://eizandensha.co.jp/densya/jyousyaken/otoku/otoku-menu.html 特惠車票一覽] {{webarchive|url=https://web.archive.org/web/20121213153941/http://eizandensha.co.jp/densya/jyousyaken/otoku/otoku-menu.html |date=2012-12-13 }} - 叡山電鐵{{ja}}</ref>。

== 網路宣传和商業合作 ==
從2010年的期間開始，活用[[網路|網路]]的搭配措施開始變得醒目。

從同一年9月底左右，叡電的[[推特|推特]]帳號開始在推特上發送信息。另外，在當年第六回的「叡電祭」上，除了用[[Ustream|Ustream]]即時放送從活動用列車的駕駛席上拍到的影像之外，也逐次在推特發佈活動的情形。此外，叡電也在[[Youtube|Youtube]]開設帳號並經營頻道。

由于漫画杂志《[[Manga_Time_Kirara|Manga Time Kirara]]》的略称和叡山电铁900系电车的爱称均为“Kirara”（{{lang|ja|きらら}}），加之前者连载作品《[[K-ON!|K-ON!]]》的动画舞台包括叡山电铁沿线，2011年起，叡山电铁与出版商[[芳文社|芳文社]]合作启动“Kirara×Kirara Project”（{{lang|ja|きらら×きららプロジェクト}}），通常在Kirara关联作品的电视动画（如《[[若叶女孩|若叶女孩]]》、《[[Manga_Time_Kirara|Manga Time Kirara]]》的《[[三者三叶|三者三叶]]》、《[[Manga_Time_Kirara_Carat|Manga Time Kirara Carat]]》的《[[NEW_GAME!|NEW GAME!]]》、《[[Manga_Time_Kirara_Max|Manga Time Kirara Max]]》的《[[请问您今天要来点兔子吗？|请问您今天要来点兔子吗？]]》、《[[Manga_Time_Kirara_Forward|Manga Time Kirara Forward]]》的《[[花舞少女|花舞少女]]》、《[[Manga_Time_Kirara_Miracle!|Manga Time Kirara Miracle!]]》的《[[幸腹涂鸦|幸腹涂鸦]]》等）或[[剧场版|剧场版]]（如《[[剧场版_魔法少女小圆|剧场版 魔法少女小圆]]》）推出后展开，内容包括出售绘有作品角色的纪念车票、开行彩绘电车、或是在部分列车前挂上特殊车头牌等等<ref name="toyokeizai/167714">{{Cite web|author=森口 誠之|work=东洋经济Online|date=2017年4月16日|url=http://toyokeizai.net/articles/-/167714|script-title=ja:アニメ「聖地」で魅力PR、京阪と叡電の挑戦 きっかけは「けいおん！」巡礼者だった|publisher=[[东洋经济新报社|东洋经济新报社]]|accessdate=2017年4月16日|archiveurl=https://web.archive.org/web/20170416062801/http://toyokeizai.net/articles/-/167714|archivedate=2017年4月16日|dead-url=yes|language=ja}}</ref>。

和動畫『[[K-ON!|K-ON!]]』合作的「EiDEN！×K-ON！TRAIN」於2011年9月1日至2012年2月19日間行駛，叡電使用[[Google定位|Google定位]]持續更新該車輛的位置情報。

2017年，因[[明天，我要和昨天的妳約會|明天，我要和昨天的妳約會]]電影上映，其中一列曾用作電影拍攝用途的Deo800（802）取名「ぼく明日号」。
