'''勝尾寺'''（日語：かつおうじ）是位於日本[[大阪府|大阪府]][[箕面市|箕面市]]的[[高野山真言宗|高野山真言宗]][[佛寺|佛寺]]，山號是應頂山。屬於[[西國三十三箇所|西國三十三所]]的第二十三號禮所。開山者為開成（日語：かいじょう），所祭祀[[本尊|本尊]]是[[千手千眼觀世音菩薩|十一面千手觀世音菩薩]]。寺號日語可讀作「katsuoji（かつおじ）」或「kachioji（かちおじ）」<ref>在20世紀末出版的辭典、事典類中，有許多以「かつおじ」或「かちおじ」收錄的詞條。
</ref>。

== 歷史 ==
根據傳承，勝尾寺的草創經緯如下所述。在[[神龜_(聖武天皇)|神龜]]4年（[[727年|727年]]），藤原致房之子善仲，善算的兄弟，在此地修築草庵，勵行佛道修行。此後約40年，[[天平神護|天平神護]]元年（[[765年|765年]]），[[光仁天皇|光仁天皇]]的[[皇子|皇子]]（[[桓武天皇|桓武天皇]]的異母之兄）開成在2人的指導下進入佛門。[[寶龜|寶龜]]8年（[[777年|777年]]），開成結束了為祈願而抄寫大般若經600卷一事，創建了勝尾寺的前身--彌勒寺<ref name=市史>{{cite book |title=箕面市史: 本編 |date=1964年 |publisher=箕面市役所 |page=87}}</ref>。之後，數年後的寶龜11年（[[780年|780年]]），相傳妙觀製作了作為本尊的十一面千手觀世音菩薩立像<ref>{{cite book |author1=神谷勝広 |title=近世文学と和製類書 |date=1999年 |publisher=若草書房 |page=305}}</ref>。

關於開成成為僧侶的事績，在正史中有很多不明確的記載，在北攝地區的山間一帶，除了該寺以外，也包括了[[高槻市|高槻市]]的神峯山寺等等，散落著開成所開基或中興的寺院<ref>{{cite book |author1=平野邦雄 |author2=瀬野精一郎 |title=日本古代中世人名辞典 |date=2006年 |publisher=吉川弘文館 |page=197}}</ref>。

勝尾寺在平安時代以後，因是山岳信仰的據點而興榮，有許多天皇和貴族的朝聖者。[[元慶|元慶]]4年（[[880年|880年]]），當時的住持行巡因為替[[清和天皇|清和天皇]]祈求病癒有成，而被賜予了「勝王寺」的寺號，有「替王帶來勝利」的意思，誠惶誠恐地選擇使用有相同讀音的勝尾寺替代<ref>{{cite web |last1=Company |first1=The Asahi Shimbun |title=asahi.com（朝日新聞社）：西国三十三所　第２３番　勝尾寺 - 関西 |url=http://www.asahi.com/kansai/kokoro/hibutsu/OSK200804250037.html |website=www.asahi.com |accessdate=2018-09-26 |language=ja}}</ref>。在《[[日本三代實錄|日本三代實錄]]》中，元慶4年清和天皇去世的記錄中，紀載了天皇曾朝聖「勝尾山」的事情，這是勝尾寺初次出現在文獻上<ref>{{cite book |title=勝尾寺文書 第1 |date=1931年 |publisher=大阪府史蹟名勝天然紀念物調査会}}</ref>。

[[元曆|元曆]]元年（[[1184年|1184年]]），在[[源平合戰|源平合戰]]中因[[一之谷之戰|一之谷之戰]]燒毀。在[[文治_(日本)|文治]]4年（[[1188年|1188年]]），在[[源赖朝|源赖朝]]命令之下，由[[熊谷直實|熊谷直實]]、[[梶原景時|梶原景時]]重建<ref name=市史/>。

[[承元|承元]]4年（[[1210年|1210年]]），從[[讚岐國|讚岐國]]流放歸來的老年[[法然|法然]]住在該寺<ref>{{cite book |author1=慈怡 |title=佛光大辭典 |date=2004年 |publisher=北京圖書館出版社 |page=4861}}</ref>。

== 御詠歌 ==

* 日語原文：重くとも　罪には法（のり）の　勝尾寺（かちおでら）　ほとけを頼む　身こそやすけれ。
意思是無論多麼沉重的罪惡，仍比不上佛教的救贖，所以透過奉獻佛陀來獲得安慰。
