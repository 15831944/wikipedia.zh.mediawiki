'''勝興車站'''位於[[台灣|台灣]][[苗栗縣|苗栗縣]][[三義鄉_(台灣)|三義鄉]]，原為[[臺灣鐵路管理局|臺灣鐵路管理局]][[舊山線|舊山線]]（臺中線舊線）的[[鐵路車站|鐵路車站]]，原已因鐵路改線而裁撤，後於2010年6月5日舊山線復駛而重新設站，現為苗栗縣著名觀光景點。

相對於臺中線（山線）之新線，原廢止線習稱舊山線，參見[[舊山線|舊山線]]條目。

== 車站構造 ==
*台鐵[[海拔|海拔]]最高的車站，標高402.326公尺，並在車站內軌道側豎有一石造紀念碑標示出車站高度。
*木造站房。
*岸式月台、島式月台各一座。

== 歷史 ==
*1907年4月1日：設置'''十六份信號場'''（[[號誌站|號誌站]]）。<ref name=":0">{{cite book |author = 戴震宇 |authorlink =  |title = 《台灣的老火車站》 |publisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>
*1930年4月1日：升為驛（[[車站|車站]]）。<ref name=":0" />
*1935年4月21日：[[1935年新竹台中地震|新竹台中地震]]，站房無礙。
*1958年2月10日：與[[南州車站|南州]]、[[港嘴車站|港嘴]]（廢止）、[[內壢車站|內壢]]等車站同時改稱今名。<ref name=":0" />
*1998年9月23日：改線廢止。<ref name=":0" />
*1999年4月16日：登錄為苗栗縣縣定古蹟。
*2010年6月5日：[[舊山線|舊山線]]復駛，僅行駛少數列次，並以[[CK124|CK124]]蒸氣火車行駛睽違12年的[[舊山線|舊山線]]，同時慶祝6月9日123週年[[鐵路節|鐵路節]]暨[[舊山線|舊山線]]ROT暖身，以帶動鐵道觀光旅遊新風潮，開行「CK124舊山線風情半日遊」懷舊郵輪式列車。預計於同年9月公開招商交由民間營運。
