'''勝家衣車'''（Singer Sewing Co.）是一家世界知名的[[美國|美國]][[缝纫机|缝纫机]]品牌，最初創立于1851年，制品銷售至世界各地。

== 历史 ==
[[File:Singer-logo_old.png|left]]
[[File:Singer_Sewing_Machine_1851.jpg|thumb]]]]

{{see also|胜家公司}}
* 1851年，[[艾萨克·辛格|以列察克·梅里瑟 ·胜家]]发明了缝纫机，并于同年8月12日創立'''胜家缝纫机公司'''。<ref name=":0">{{Cite web|url = http://singer.sh.cn/?page-%25E5%2585%25B3%25E4%25BA%258E%25E8%2583%259C%25E5%25AE%25B6.html|title = 关于胜家 上海胜家缝纫机销售有限公司|accessdate = 2015-01-26|language = }}</ref>
* 1867年，胜家公司成为美国第一家跨国工业公司，在世界其他地方进行投产。<ref name=":0" />
* 1889年，胜家公司制造了世界上第一台电动缝纫机。<ref name=":0" />
* 1908年，[[胜家公司|胜家公司]]于该公司美国纽约总部成立，其总部所在大楼——[[胜家大楼|胜家大楼]]是当时世界上最高的建筑，亦是世界上第一幢[[摩天大楼|摩天大楼]]。<ref name=":0" />
* 1957年，胜家公司发明由[[电脑|电脑]]控制的多功能家庭缝纫机，此后逐步用于工业缝纫机。<ref name=":0" />
* 1987年，缝纫机和家具生产业务被分离，成立了一新公司：'''SSMC公司'''（即现今的胜家缝纫机公司）。<ref>{{Cite book|title = 简明版大不列颠百科全书|last = |first = |publisher = |year = |isbn = |location = |pages = 胜家公司}}</ref>
* 1994年11月2日，投资2千万[[美元|美元]]，占地6万平方千米的'''上海胜家缝纫机公司'''在[[上海|上海]][[闵行经济开发区|闵行经济开发区]]成立。<ref name=":0" />
* 2001年，胜家公司迎来150周年纪念。同时推出当时世界上最先进的家用缝纫刺绣系统“QUANTUM XL-5000型”。<ref name=":0" />
* 2004年，胜家公司在上海设立研发中心，并加大投产规模。<ref name=":0" />
* 2005年，胜家公司推出世界上第一款由[[桌面电脑|桌面电脑]]或[[笔记本电脑|笔记本电脑]]控制的FUTARA CE-200型缝纫机。<ref name=":0" />
* 2006年，'''Affiliates of Kohlberg & Company,LLC'''（Kohlberg），即胜家的控股公司，完成了对'''VSM'''（VSM Group Holding AB）的收购。该公司是领先的高端缝纫机和配件生产商，拥有[[百福|百福]]（Pfaff）、Husqvarna Viking两个品牌。合并后公司易名为'''SVP Holdings'''。<ref name=":0" />
