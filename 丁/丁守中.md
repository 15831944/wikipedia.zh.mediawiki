'''丁守中'''（{{bd|1954年|9月1日|CatIdx=T丁}}），[[中華民國|中華民國]]政治人物，出生並成長於[[臺灣|臺灣]][[高雄市|高雄市]]，後搬至[[臺北市|臺北市]]，籍貫[[浙江省|浙江省]][[義烏縣|義烏縣]]。[[中國國民黨|中國國民黨]]籍，[[國立臺灣大學|國立臺灣大學]]政治學系兼任副教授<ref>{{Cite web|url=http://politics.ntu.edu.tw/?cat=9|title=兼任教師|author=國立臺灣大學政治系|publisher=國立臺灣大學}}</ref>，曾任國民黨青工會主委、台北市黨部主委、[[中國國民黨中央委員會組織發展委員會|中國國民黨中央委員會組織發展委員會主委]]、[[立法委員|立法委員]]。任立委期間問政表現多次獲得[[公民監督國會聯盟|公民監督國會聯盟]]優秀立委之評鑑<ref>{{Cite web |url =http://www.ccw.org.tw/legislator/content/62 |title =第七屆立委評鑑  |author =公民監督國會聯盟}}</ref><ref>{{Cite web |url =http://www.ccw.org.tw/legislator/content/14517 |title =第八屆立委評鑑  |author =公民監督國會聯盟}}</ref>。2018年參與[[2018年中華民國直轄市長及縣市長選舉|台北市長選舉]]僅以約3000票之差落後[[台北市長|台北市長]]當選人[[柯文哲|柯文哲]]。

==個人背景==
===家庭背景===
丁守中的父親[[丁堅|丁堅]]是一名軍醫，1949年隨著[[中華民國政府|中華民國政府]]撤退來到臺灣，丁守中的母親則是[[客家人|客家人]]。丁守中小時在國軍高雄總醫院宿舍長大，在上學經過醫院時，曾目睹[[東引海戰|東引海戰]]的傷兵，見到當時急診室人滿為患，連大操場上都放滿了行軍床，上面都是斷手斷腳傷兵的情況。<ref>{{Cite web |url =http://news.ltn.com.tw/news/politics/breakingnews/2539581 |title =邀823老兵座談 丁守中：我是少數經歷戰爭的政治人物  |author =自由時報}}</ref>

1976年，台大畢業。在禮賓司任職一年，認識蔣經國。1979年，丁守中向父親借了三萬美金並赴美留學。

岳父[[溫哈熊|溫哈熊]]，曾任[[聯勤司令部|聯勤司令部]]上將總司令。妻子[[溫子苓|溫子苓]]，畢業於美國哈特福音樂學院，琵琶蒂音樂學院碩士，曾任教於[[實踐大學|實踐大學]]、[[政治作戰學校|政治作戰學校]]及[[東吳大學_(臺灣)|東吳大學]]<ref>[http://www.scu.edu.tw/music/teachers/new_page_6.htm 溫子苓老師簡歷]</ref><ref>{{Cite web |url=http://www.setn.com/News.aspx?NewsID=19825|title=獨家／愛妻溫子苓溫情守護　陪丁守中打最難的一仗|publisher=三立新聞網|date=2014-04-14}}</ref>。兩人生有兩子，長子[[丁世傑|丁世傑]]小時侯為了拯救受困游泳池的同伴，腳反遭排水孔吸入而受傷，見義勇為的義舉聲名大躁一時<ref>陳志平.丁守中／用理性用鼓勵 為兒子療傷.2009-06-14</ref>，已於2013年服憲兵役退伍，次子從美國[[印第安那大學|印第安那大學]]畢業。

丁守中立法院問政之餘，其嗜好為[[園藝|園藝]]。<ref>[http://www.epochtimes.com/b5/11/9/12/n3370572.htm 愛種花 丁守中林濁水各有所好].大紀元台灣.2011-09-12</ref>

===師生關係===
丁守中的老師是前[[中國國民黨|中國國民黨]]主席[[連戰|連戰]]。丁守中的學生包括民進黨的[[段宜康|段宜康]]、政治評論員[[唐湘龍|唐湘龍]]，唐湘龍曾於1989年丁守中參選立法委員期間為其助選<ref>[http://news.ifeng.com/a/20140417/40003238_0.shtml 台北市长关乎蓝营世代更替 连胜文若赢马英九或会提前跛脚].凤凰资讯.2014-04-17</ref>

==選舉經歷==

[[2000年中華民國總統選舉|2000年總統選舉]]時，丁守中擔任[[連戰|連]][[蕭萬長|蕭]]競選總部企劃部召集人。

[[2001年中華民國立法委員選舉|2001年立委選舉]]失利後，連戰延攬丁擔任[[中國國民黨|中國國民黨]]台北市黨部主委，後又出任[[中國國民黨中央委員會組織發展委員會|中國國民黨中央組發會主委]]。

[[2004年中華民國總統選舉|2004年總統選舉]]時，丁擔任連宋競選總部動員部主任<ref>[http://www.epochtimes.com/b5/3/12/1/n421588.htm 連宋競選主軸：拼經濟救台灣　競選架構底定].2003-12-01</ref>。

然而[[2014年直轄市長及縣市長選舉中國國民黨提名作業#臺北市|2014年中國國民黨台北市長黨內初選]]丁守中和[[連勝文|連勝文]]競爭激烈，初選過程中雙方產生不愉快，[[連戰|連戰]]表示十分痛心<ref>盧怡秀.[http://www.chinatimes.com/realtimenews/20140327002760-260401 連戰批丁 連勝文：心痛大於愛子心切].中時電子報.2014-03-27</ref>，丁守中表示「愛子心切，人之常情」，不管初選結果如何，他仍視連戰為一輩子的老師<ref>楊毅.[http://www.chinatimes.com/realtimenews/20140326005481-260401 丁守中：視連戰為一輩子老師].中時電子報.2014-03-26</ref>。最終由[[連勝文|連勝文]]勝出<ref>晏明強.[https://tw.appledaily.com/new/realtime/20140419/382188/ 國民黨北市長初選　連勝文擊敗丁守中勝出].蘋果日報.2014-04-19</ref>，丁守中表示「已盡全力、雖敗猶榮，全力支持連勝文！」<ref>王沛文.[http://www.setn.com/News.aspx?NewsID=20484 丁守中落選感言：已盡全力、雖敗猶榮　全力支持連勝文！].三立新聞網.2014-04-19</ref>。連勝文獲得國民黨提名後，親自拜會丁守中請益，並為初選時期的不愉快道歉向丁守中道歉<ref>[http://hk.crntt.com/crn-webapp/touch/detail.jsp?coluid=46&kindid=0&docid=103273293 修補關係連勝文向丁守中致歉].中評社.2014-07-05</ref>。2018年丁守中獲得提名後拜會連戰<ref>劉宛琳.[https://money.udn.com/money/story/7307/3127845 丁守中將訪黨內大老、連戰連勝文].經濟日報.2018-05-07</ref>，獲得連戰鼓勵和支持<ref>黃麗芸.[http://www.cna.com.tw/news/aipl/201805150326-1.aspx 化解心結尋求團結 丁守中向連戰請益].中央社.2018-05-15</ref><ref>王寓中.[https://udn.com/news/story/11311/3144197 解心結 丁守中訪連戰].聯合報.2018-05-16</ref>。

2018年終於成功獲得國民黨提名參選台北市，一遂24年之大願，但在11月25日凌晨因票數落後而提出驗票與選舉無效之訴。

;推動透明國會
*2008年三讀通過「[[:s:立法院職權行使法|立法院職權行使法]]修正案」。立委丁守中主張密室協商必須公開，杜絕利益交換，回歸委員會專業化，提「立法院職權行使法修正案」，未來議案協商將全程錄影、錄音、記錄，併同協商結論刊登公報；議案協商期也由四個月縮減為一個月。該案經委員會初審通過後，藍綠陣營內部都有保留意見，私下曾試圖尋求轉圜機會，但此案屬陽光法案，社會期待甚深，各方不願公開反對，曾表達保留意見的無盟昨也未要求交付朝野協商，最後於2008年4月25日順利三讀通過<ref>彭顯鈞、施曉光、黃維助.[http://news.ltn.com.tw/news/focus/paper/206847 走出密室 議案協商全都錄].2 自由時報電子報.2008-04-26</ref>。2009年公民監督國會聯盟為彰表推動透明國會委員之努力，頒予「透明國會貢獻獎」。<ref>editor.[http://www.ccw.org.tw/p/1606 【優秀立委】透明國會貢獻獎].公民監督國會聯盟.2009-07-16</ref>

;設限董監表決權
2011年10月立法院三讀通過「公司法第197條之1條文修正草案」，未來公開發行以上公司董事一旦設定股票質權比例，高於選任時持有股份數一半時，超過部分不得行使表決權。此修正有助遏止現行上市櫃董監事質押比例過高弊端，也能保障投資人權益。提案立委丁守中認為， 股東既然質押股票，相當於實際所有權已轉移，不應再藉個人的高持股，參與公司重要議案表決。經濟部商業司昨天也回應，新法限制大股東拿股票質押比例，將有助大股東更專心在公司經營上，投資人權益也可因此獲得保障<ref>徐筱嵐、林安妮.[http://ai100.chihlee.edu.tw/files/16-1018-8330.php?Lang=zh-tw 董監質押比過半 表決權設限].經濟日報.2011-10-26</ref>。
