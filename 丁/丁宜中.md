'''丁宜中'''（{{bd|1890年||1973年|}}）字'''照普'''，[[贵州|贵州]][[贵阳|贵阳]]人。[[中华民国|中华民国]]政治人物。<ref name=gz>贵州省文史研究馆志，贵州人民出版社，2003年</ref>

== 早年經歷 ==
丁宜中毕业于[[贵阳优级师范学校|贵阳优级师范学校]]（一说为贵州优级师范学校选科）。[[辛亥革命|辛亥革命]]后，任贵州军政府财政司预决算科科长，不久历任贵州民政长秘书、贵州巡按使署秘书。1920年11月10日，贵州发生[[民九事变|民九事变]]，丁宜中经[[谷正伦|谷正伦]]掩护出走，随谷正伦到[[甘肃|甘肃]]任其幕僚，此后留在甘肃。

== 中年經歷 ==
1920年代，丁宜中回到贵州，任贵州省财政厅预决算处总编核、贵州省行政公署总务科长、[[贵州省立模范中学|贵州省立模范中学]]校长。1925年，贵州成立[[贵州省行政委员会|贵州省行政委员会]]代行省政，丁宜中曾以政务厅长之职代行委员长职权。<ref name=gz/>

1930年代初，丁宜中任川黔边防督办公署秘书长，[[贵州省政府|贵州省政府]]秘书长。1938年8月，任[[甘肃省政府|甘肃省政府]]秘书长。1944年，任甘肃省政府委员，1945年1月再度出任甘肃省政府秘书长。<ref name=gz/>1949年9月29日，甘肃省政府代主席丁宜中等500餘人到[[酒泉|酒泉]]向軍管會報到。<ref name="民國史大事記">{{cite book|editors=[[李新_(1918年)|李新]]總主編，[[中國社會科學院|中國社會科學院]]近代史研究所[[中華民國史|中華民國史]]研究室，韓信夫、姜克夫主編|title=《中華民國大事記》|location=北京|publisher=[[中華書局|中華書局]]|isbn=9787101079982|date=2011年7月}}</ref>{{rp|9020}}

== 晚年經歷 ==
1950年代后，回到贵阳居住。<ref name=gz/>

1962年，丁宜中获聘为[[贵州省文史研究馆|贵州省文史研究馆]]馆员。1973年，丁宜中逝世。<ref name=gz/>
