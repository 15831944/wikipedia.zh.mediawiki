'''丁聖祐'''（{{bd|1994年|12月13日|}}），出生於[[美國|美國]][[加利福尼亞州|加利福尼亞州]]，是[[臺灣|臺灣]]女子游泳運動員<ref name = L404637/>。

== 生平 ==
丁聖祐在9歲與家人回到臺灣後，曾讀[[臺北市大安區新生國民小學|臺北市大安區新生國民小學]]，後來轉去[[臺北市信義區吳興國民小學|臺北市信義區吳興國民小學]]。國中就讀[[臺北市立南港高級中學|臺北市立南港高級中學]]國中部，並選擇在母校直升高中部<ref name = L404637/>。大學就讀[[國立臺灣大學|國立臺灣大學]]工商管理學系<ref name = E358888/>。

== 爭議 ==
丁聖祐在[[2017年夏季世界大學運動會中華台北代表團|2017年夏季世界大學運動會中華台北代表團]]游泳代表隊名單公佈後，於個人的[[YouTube|YouTube]]頻道上傳影片，及《體育改革聯會》[[Facebook|Facebook]]上指控[[中華民國游泳協會|中華民國游泳協會]]遴選過程黑箱作業<ref>{{cite news|title=未列世大運選手名單 台大泳后控體壇「黑箱作業」|publisher=中國時報|url=http://hottopic.chinatimes.com/20170715003065-260807|date=2017-07-15|author=陳慶餘|accessdate=2017-07-15|language=zh-hant}}</ref>。對於參賽標準的質疑，中華民國游泳協會理事長[[許東雄|許東雄]]人在[[2017年世界游泳錦標賽|2017年世界游泳錦標賽]]現場，而由游泳代表隊總教練[[李澄峯|李澄峯]]代為回應<ref>{{cite news|title=游泳選手丁聖祐質疑 泳協3點回應|publisher=蘋果日報|url=http://www.appledaily.com.tw/realtimenews/article/new/20170715/1161716/|date=2017-07-15|author=詹健全|accessdate=2017-07-15|language=zh-hant}}</ref>。[[國立臺灣大學|國立臺灣大學]]教師[[葉丙成|葉丙成]]在Facebook表示，質疑[[2017年夏季世界大學運動會|2017年夏季世界大學運動會]]派[[全國大專校院運動會|全國大專校院運動會]]100公尺蝶式金牌的選手[[黃渼茜|黃渼茜]]去參加50公尺蝶式的比賽項目，而非全國大專校院運動會50公尺蝶式金牌的丁聖祐，明顯有誤導臺灣社會大眾的說詞<ref>{{cite news|title=丁聖祐控泳協黑箱作業 體育署積極瞭解[影]|publisher=中央社|url=http://www.cna.com.tw/news/firstnews/201707160120-1.aspx|date=2017-07-16|author=李晉緯|accessdate=2017-07-17|language=zh-hant}}</ref>。[[教育部體育署|教育部體育署]]副署長[[林哲宏|林哲宏]]、[[中華民國大專院校體育總會|中華民國大專院校體育總會]]秘書長[[曾慶裕|曾慶裕]]和[[李澄峯|李澄峯]]召開記者會表示，尚未報名選手的個人項目，也沒有確認黃渼茜會報名50公尺蝶式；但為了平息外界爭議，決定女子游泳隊6名徵召入選的夏季世界大學運動會國家選手中，除[[廖曼汶|廖曼汶]]參加個人賽事外，其他選手皆不參加個人比賽項目<ref>{{cite news|title=游泳／丁聖祐有疑慮 教練：將不參加游泳個人賽|publisher=聯合報|url=https://udn.com/news/story/7005/2586795|date=2017-07-16|author=吳思儀|accessdate=2017-07-17|language=zh-hant}}</ref>。
