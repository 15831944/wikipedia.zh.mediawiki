'''丁玲'''（{{bd|1904年|10月12日|1986年|3月4日|catIdx=D丁}}），原名'''蒋伟'''，字'''冰之'''，笔名'''彬芷'''、'''从喧'''等。[[湖南|湖南]][[临澧|临澧]]人。中国当代著名的作家、社会活动家。{{fact|自称为[[明朝|明朝]]末年农民军起义领袖[[李自成|李自成]]后人}}。

== 生平 ==
[[File:Ding_Ling.jpg|thumb]]
*1904年10月12日，生于[[湖南省|湖南省]]安福县西乡黑胡子冲村（今[[临澧县|临澧县]]佘市镇高丰村）；父亲蒋保黔为清末秀才，在丁玲4岁时去世，母亲余曼贞带她去了外祖父家，后任教师供她上学。
*1918年，丁玲考入桃源县的湖南省第二女子师范学校预科，与[[王剑虹|王剑虹]]同学，
*1919年，转入长沙[[周南女中|周南女中]]；
*1922年，夏，好友[[王剑虹|王剑虹]]从上海回乡，带丁玲前往上海，入上海平民女校，改名丁玲，
*1923年，与[[王剑虹|王剑虹]]共入[[上海大学_(民国)|上海大学]]中文系旁听，
*1924年，王剑虹与老师[[瞿秋白|瞿秋白]]相爱并同居，丁于夏季只身赴北京，欲进北京大学学习，无果，通过同室好友曹孟君的男友左恭认识《京报》副刊编辑[[胡也频|胡也频]]，
*1925年，与[[胡也频|胡也频]]在北京同居，
*1927年，12月，在北平写成并发表第一篇小说《梦珂》，
*1928年，2月，小说《[[莎菲女士的日記|莎菲女士的日記]]》发表，轰动文坛；年底丁玲追情人[[冯雪峰|冯雪峰]]至上海，胡也频随后也追去，之后三人到杭州谈判，[[冯雪峰|冯雪峰]]退出，
*1929年，胡也频因欠债到去济南教书，不久丁玲也到济南，
*1930年，5月，因胡受到通缉，双双逃回上海，
*1930年，11月，与胡也频的儿子胡小频出生
*1931年，2月胡也频被枪毙，担任“[[左联|左联]]”刊物《[[北斗_(雜誌)|北斗]]》主编，11月与翻译[[冯达|冯达]]同居，
*1932年，加入[[中国共产党|中国共产党]]；
*1933年，5月，与冯达一起遭[[国民党|国民党]]当局绑架；
*1936年，逃离[[南京|南京]]，抵达陕北，先后担任苏区“中国文艺协会”主任、[[中央警卫团|中央警卫团]]政治部副主任、[[西北战地服务团|西北战地服务团]]团长、《[[解放日报|解放日报]]》文艺副刊主编、陕甘宁边区文协副主任等职；
*1948年，发表其代表作——长篇小说《[[太阳照在桑干河上|太阳照在桑干河上]]》；
*1949年后，先后任《[[文艺报|文艺报]]》主编，中央文学研究所所长、[[中共中央宣传部|中共中央宣传部]]文艺处长、[[中国作家协会|中国作家协会]]党组书记、副主席、《[[人民文学|人民文学]]》主编、中国文联党组副书记等职；
*1951年，因《太阳照在桑干河上》获[[斯大林文学奖金|斯大林文学奖金]]；
*1955年，被定为“[[丁玲、陈企霞反党小集团|丁玲、陈企霞反党小集团]]”主要成员；
*1957年，被定为“[[丁玲、冯雪峰右派反党集团|丁玲、冯雪峰右派反党集团]]”主要成员；被划入右派<ref>1958年2月，第一届全国人民代表大会第五次会议（1～11日））在北京召开。大会开幕前一天，即1月31日，全国人民代表大会常务委员会向大会提出了《关于罢免全国人民代表大会常务委员会民族委员会法案委员会和国防委员会中的右派分子黄绍竑等十人的职务的建议》，请大会予以审议。建议如下：“……（二）关于右派分子[[章乃器|章乃器]]、[[潘大逵|潘大逵]]、[[曾庶凡|曾庶凡]]、黄绍竑、[[陈铭枢|陈铭枢]]、[[黄现璠|黄现璠]]、[[费振东|费振东]]、[[乔传珏|乔传珏]]、[[马哲民|马哲民]]、[[章伯钧|章伯钧]]、[[叶笃义|叶笃义]]、[[程士范|程士范]]、[[潘锷鏱|潘锷鏱]]、[[罗隆基|罗隆基]]、[[费孝通|费孝通]]、[[储安平|储安平]]、[[钱伟长|钱伟长]]、[[钱孙卿|钱孙卿]]、[[欧百川|欧百川]]、[[王天锡|王天锡]]、[[韩兆鹗|韩兆鹗]]、丁玲、[[张东木|张东木]]、[[谢雪红|谢雪红]]、[[杨子恒|杨子恒]]、[[郑立齐|郑立齐]]、[[黄琪翔|黄琪翔]]、[[李伯球|李伯球]]、[[徐铸成|徐铸成]]、[[黄药眠|黄药眠]]、[[王毅齐|王毅齐]]、张轸、[[張雲川_(農工黨)|张云川]]、[[朱君允|朱君允]]、[[毕鸣岐|毕鸣岐]]、[[谭志清|谭志清]]、[[龙云|龙云]]、[[李琢菴|李琢菴]]38人的代表资格问题，代表资格审查委员会已经向大会提出建议，认为他们已经丧失继续执行全国人民代表大会代表职务的合法根据，不应当出席第一届全国人民代表大会第五次会议。</ref>后，受到各种迫害，包括下放北大荒，投入监狱等；
*1979年，经党中央批准平反，并重返文坛；
*1986年3月4日，逝世。

== 争议 ==
=== 与毛泽东一次单独谈话的真伪 ===
[[杨桂欣|杨桂欣]]曾在一篇文章中转述丁玲的回忆说“让我最后悔的，还是在[[中宣部|中宣部]]当文艺处长的时候……跟她（[[江青|江青]]）一起去[[中南海|中南海]]，和[[毛主席|毛主席]]一起吃饭……主席先问我对[[周扬_(学者)|周扬]]的看法……我一个劲儿说着，全都是说周扬的缺点……毛主席突然问我：‘周扬总还是有些优点吧？！’我真后悔自己不会做人……”<ref>杨桂欣，“我丁玲就是丁玲”，炎黄春秋1993年第七期</ref>。2001年2月13日，丁玲的丈夫[[陈明|陈明]]首次对这一记述公开予以否认。<ref>徐庆全，丁玲与毛泽东一次谈话的真伪，炎黄春秋2008年第三期</ref>

==评价==
[[夏志清|夏志清]]的《中國現代小說史》對她的評價並不高，“三十年代丁玲的声望，仅次于[[茅盾|茅盾]]、[[老舍|老舍]]、[[巴金|巴金]]诸人，我审读她那时的作品，实在一篇也说不上是佳作。”

[[沈从文|沈从文]]：“能给朋友的是亲切洒脱。她既不习惯使用脂粉，也缺少女性那分做作。她待人只是那么不可形容的爽直，故朋友相熟略久，就似乎极容易忘掉了她是个女人。然而从另外一方面说来，则凡属于一个女子某种美德，她却毫无缺处。她亲切却不狎亵。她爽直并不粗暴”。<ref>{{cite web |url= http://news.ifeng.com/history/shixueyuan/detail_2012_03/19/13298514_0.shtml|title=沈从文评丁玲：亲切却不狎亵 爽直并不粗暴 |accessdate= 2012年|work= |publisher= 凤凰网|language =zh-cn}}</ref>

[[毛泽东|毛泽东]]高兴地作诗赞其为:“昨天文小姐，今日武将军”。<ref>{{cite web |url=http://www.chinawriter.com.cn/n1/2018/0328/c404064-29894454.html|title=
丁玲：爱折腾的前半生|accessdate= 2018年|work= |publisher= 中国作家网|language =zh-cn}}</ref>
