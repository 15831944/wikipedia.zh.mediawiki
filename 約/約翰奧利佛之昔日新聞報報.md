《'''上周今夜秀'''》（''{{lang-en|Last Week Tonight with John Oliver}}''），美國的晚間娛樂節目。主持人為[[約翰·奧利佛|約翰·奧利佛]]，自2014年4月27日起，在美國[[HBO|HBO]]頻道播出。每集長度為30分鐘，探討各項新聞主題，並加以嘲諷。因2018年6月17日播出的內容以[[中華人民共和國最高領導人|中國領導人]][[習近平|習近平]]為主題，諷刺包括[[2018年中華人民共和國憲法修正案|廢除國家主席任期制]]、[[諾貝爾和平獎|諾貝爾和平獎]]得主[[劉曉波|劉曉波]]被監禁至死、[[新疆|新疆]][[維穩|維穩]]中國政壇的多起事件，以及[[中華人民共和國網絡審查|網路審查制度]]，主持人及節目內容遭到中國互聯網的全網封殺<ref>{{cite web |title=約翰·奧利佛碰紅線 讓中國生氣的政治脫口秀 |url=http://www.bbc.com/zhongwen/trad/chinese-news-44599708 |website=BBC中文網 |accessdate=2018-06-25 |date=2018-06-25}}</ref><ref>{{cite web |title=脱口秀“吐槽”习近平 HBO官网遭封杀 |url=http://www.dw.com/zh/%E8%84%B1%E5%8F%A3%E7%A7%80%E5%90%90%E6%A7%BD%E4%B9%A0%E8%BF%91%E5%B9%B3-hbo%E5%AE%98%E7%BD%91%E9%81%AD%E5%B0%81%E6%9D%80/a-44331920 |website=德國之聲中文網 |accessdate=2018-06-25 |date=2018-06-25}}</ref><ref>{{cite web |title=敢嘴習近平！《上周今夜秀》主持約翰奧利佛在中國網路「被消失」 |url=http://www.storm.mg/article/451346 |website=風傳媒 |accessdate=2018-06-25 |date=2018-06-20}}</ref>。
