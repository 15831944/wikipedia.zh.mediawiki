'''約書亞·庫許納'''（{{lang-en|Joshua Kushner}}，{{bd|1985年|6月12日|catIdx=Kushner, Joshua}}）是一名[[美國|美國]][[實業家|實業家]]及投資者。他是投資公司{{le|興盛資本|Thrive Capital}}的創辦人兼執行合夥人、{{le|奧斯卡健康|Oscar Health}}的共同創辦人，地產大亨{{le|查爾斯·庫許納|Charles Kushner}}的次子。

== 職業生涯 ==

=== 《Scene》、Vostu及Unithrive ===
大二那年，庫許納在[[布克兄弟|布克兄弟]]贊助的校園刊物《Scene》擔任發想執行編輯，希望讓這份雜誌成為哈佛版的《[[時尚_(雜誌)|時尚]]》或《[[浮華世界_(雜誌)|浮華世界]]》。<ref>{{Cite web |title=DOORDROPPED: Which Scene? |publisher=The Harvard Crimson |url=http://www.thecrimson.com/article/2005/12/7/doordropped-which-scene-the-stairwell-at/ |website=thecrimson.com |first=Leon |last=Neyfakh |date=2005-12-07 |accessdate=2016-01-02 |language=en |archiveurl=https://web.archive.org/web/20170131211844/http://www.thecrimson.com/article/2005/12/7/doordropped-which-scene-the-stairwell-at/ |archivedate=2017-01-31 |deadurl=no }}</ref>據《{{le|哈佛大學校報|The Harvard Crimson}}》报道，《Scene》在發行後遭到猛烈抨擊，學生們更在[[Facebook|Facebook]]創立「《Scene》是個狗屁」（{{lang|en|''Scene'' Magazine is Bullshit}}）社團，批評它「不僅欠缺少數民族作為模特兒，更扭曲了哈佛人的形象」。<ref name=":0">{{Cite web |title=Glossies Gear Up For Second Run |publisher=The Harvard Crimson |url=http://www.thecrimson.com/article/2006/3/16/glossies-gear-up-for-second-run/ |date=2006-03-16 |first=Patrick S. |last=Lahue |website=thecrimson.com |accessdate=2016-01-02 |language=en |archiveurl=https://web.archive.org/web/20161111140001/http://www.thecrimson.com/article/2006/3/16/glossies-gear-up-for-second-run/ |archivedate=2016-11-11 |deadurl=no }}</ref>

在他大三那年的春季学期，庫許納與兩位研究生合資1萬美元，創立-{zh-hans:社交; zh-tw:社群;}-媒體{{le|Vostu|Vostu}}，<ref name="forbes.com"/>希望與[[Facebook|Facebook]]做出區隔，創造一個並非以英文溝通的線上社群。根據庫許納表示，拉丁美洲的網際網路使用量逐年增長，科技技術也隨之蓬勃發展，是開創新社群網站的潛力市場。<ref>{{Cite web |title=Students Start Spanish Social Site |publisher=The Harvard Crimson |url=http://www.thecrimson.com/article/2007/3/7/students-start-spanish-social-site-facebook/ |website=thecrimson.com |first=Andrew M. |last=Benitez |date=2007-03-07 |accessdate=2016-01-02 |language=en |archiveurl=https://web.archive.org/web/20161111141505/http://www.thecrimson.com/article/2007/3/7/students-start-spanish-social-site-facebook/ |archivedate=2016-11-11 |deadurl=no }}</ref>

畢業一年後，庫許納與非營利組織{{le|Kiva (組織)|Kiva (organization)|Kiva}}總裁{{le|普立默·夏爾|Premal Shah}}的表弟，同時也是哈佛學弟的塔努吉·帕里克（{{lang|en|Tanuj Parikh}}）共同創立了一家公司Unithrive。Unithrive受到了Kiva的點對點貸款模式啟發，以「緩解大學學費危機」作為主要發想，面臨金錢困難的學生可以上傳個人照片及簡歷，透過網站與校友資助人聯繫，申請最高2000美元的貸款，並享有畢業五年內免息還款。<ref>{{Cite news|title = I'm Going to Harvard. Will You Sponsor Me?|url = https://www.nytimes.com/2009/06/14/fashion/14unithrive.html|newspaper = [[The_New_York_Times|The New York Times]]|date = 2009-06-12|accessdate = 2016-01-02|issn = 0362-4331|first = Allen|last = Salkin|language = en|archiveurl = https://web.archive.org/web/20170116093051/http://www.nytimes.com/2009/06/14/fashion/14unithrive.html|archivedate = 2017-01-16|deadurl = no}}</ref>這家公司已不再運作。
