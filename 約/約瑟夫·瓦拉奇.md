'''約瑟夫·瓦拉奇'''（'''Joseph Valachi'''，本名'''Joseph Michael "Joe Cargo" Valachi'''；{{bd|1904年|9月22日|1971年|4月3日}}），[[意大利裔美國人|意大利裔美國人]]，也被稱為"Charles Charbano"和"Anthony Sorge"，是第一個公開承認[[美國黑手黨|黑手黨]]存在的黑手黨成員。<ref>{{Cite web |url = http://americanmafiahistory.com/joe-valachi-the-first-rat/ |title = Joe Valachi – The First Rat |publisher = americanmafiahistory.com |language = en}}</ref>

他也是令[[西西里黑手黨|Cosa Nostra]]（意思是「我們自己的事」）成為一個家喻戶曉名字的人。
