'''約翰·納皮爾'''（{{lang|en|'''John Napier'''或'''Neper'''}}，{{bd|1550年|2月1日|1617年|4月4日|catIdx=N}}），也译作'''耐普尔'''，是[[蘇格蘭|蘇格蘭]][[數學家|數學家]]、[[物理學家|物理學家]]兼[[天文學家|天文學家]]。他最為人所熟知的是發明[[對數|對數]]，以及[[算尺|滑尺]]的前身──[[納皮爾的骨頭|納皮爾的骨頭]]計算噐。而且他對[[小數點|小數點]]的推廣也有貢獻。納皮爾出生地在蘇格蘭爱丁堡的{{link-en|默奇斯顿塔|Merchiston Tower}}，現在是[[愛丁堡納皮爾大學|愛丁堡納皮爾大學]]的一部分。

== 數學貢獻 ==
納皮爾在數學與工程界之外並不很出名，但他在數學應用上的貢獻則無庸置疑。[[對數|對數]]使得手算變得簡單而且快多了，也因此為後來許多科學進步開啟了大門。他的著作《極好用對數表的一個描述》（或译作《奇妙的对数规律的描述》，{{lang-la|''Mirifici Logarithmorum Canonis Descriptio''}}）總共有三十七頁的解釋和九十頁的對數表，對於後來的[[天文學|天文學]]、[[力學|力學]]、[[物理學|物理學]]、[[占星學|占星學]]的發展都非常重要。

== 其他發明 ==
納皮爾的發明並不侷限於對數。''Rabdologiae''是他發表的另一本論文，裡面討論一種計算[[乘法|乘法]]的簡便方法，並且介紹了後來被稱為「納皮爾棒」或「[[納皮爾的骨頭|納皮爾的骨頭]]」的[[計算器|計算器]]。在附錄裡面他解釋了另外一種用金屬片計算乘法和[[除法|除法]]的方法，這是機械計算器[[算尺|滑尺]]的直接祖先。

另外一個很有用的概念是後來被稱為[[奈培球|奈培球]]或[[奈培五角形|奈培五角形]]的數學記憶法，它被用來幫助記憶[[球面三角|球面三角]]裡面的公式。

與他同一時代的人還提到許多「秘密發明」，包括一種圓形的駕駛戰車，可算是現代[[坦克車|坦克車]]的古早版本。還有能夠把敵方戰船焚毀的大型鏡子，[[潛水艇|潛水艇]]，和可以把所有敵兵全部殺光的[[大砲|大砲]]。

== 神學 ==
納皮爾把他的數學天份也應用到[[神學|神學]]上面去，他根據[[啟示錄|啟示錄]]預言[[基督教世界末日|世界末日]]。納皮爾相信世界末日將在1688年或者1700年到來。有時候他也被說是個[[巫師|巫師]]，不過在那個時代，有科學方面才能的人常常被如此指控而沒有任何根據。

他逝世後埋葬在[[愛丁堡|愛丁堡]]的St Cuthbert教堂。[[羅伯特·納皮爾|羅伯特·納皮爾]]（Robert Napier）是他的兒子。

== 榮譽 ==
電力工程學裡面有個很少用到的單位名稱，[[奈培|奈培]]（neper），與[[愛丁堡納皮爾大學|愛丁堡納皮爾大學]]，都是為了紀念約翰·納皮爾而命名的。[[月球|月球]]上也有個紀念他的[[納皮爾火山口|納皮爾火山口]]。
