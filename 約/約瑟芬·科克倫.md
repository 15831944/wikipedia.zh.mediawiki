'''約瑟芬·加里斯·科克倫'''（{{lang-en|Josephine Garis Cochran}}，科克倫是夫姓），（{{bd|1839年|3月8日|1913年|8月14日}}），[[美國|美國]]女[[發明家|發明家]]，和技工[[喬治·巴特斯|喬治·巴特斯]]（{{lang-en|George Butters}}）共同製作了[[商业|商用]][[洗碗機|自動洗碗機]]<ref name="ColeBrowning2003">{{cite book|url=https://books.google.com/books?id=rVQfBSlAZvAC&pg=PA100|title=Encyclopedia of Modern Everyday Inventions|author2=Eve Browning|author3=Fred E. H. Schroeder|publisher=Greenwood Publishing Group|year=2003|isbn=978-0-313-31345-5|pages=100–|author1=David John Cole}}</ref>，2006年，約瑟芬獲選進入{{link-en|全美發明家名人堂|National Inventors Hall of Fame}}（{{lang-en|National Inventors Hall of Fame}}）。<ref>{{cite web|url=http://invent.org/inductee-detail/?IID=256 |title=Spotlight | National Inventors Hall of Fame |publisher=Invent.org |date=2013-11-21 |accessdate=2016-05-29}}</ref>

== 生平 ==
約瑟芬生於[[俄亥俄州|俄亥俄州]][[阿什塔比拉县|阿士塔布拉縣]] ，生長於[[印第安纳州|印第安那州]][[瓦尔帕莱索_(印第安纳州)|瓦爾帕萊索]]。

父親是土木工程師約翰·加里斯（{{lang-en|John Garis}}），母親是艾琳·菲奇·加里斯（{{lang-en|Irene Fitch Garis}}），家裡還有一個姊妹艾琳·加里斯·蘭森（{{lang-en|Irene Garis Ransom}}）<ref>{{cite web|url = http://www.nndb.com/people/166/000165668/|title = NNBD tracking the entire world|date = 2014|accessdate = 2015-04-05|website = Josephine Cochrane|publisher = Soylent Communications|last = |first = }}</ref>。外祖父{{link-en|約翰·菲奇 (發明家)|John Fitch (inventor)|約翰·菲奇}}是一位發明家，曾獲得[[蒸汽船|蒸汽船]]專利。

1858年10月13日，約瑟芬和威廉·科克倫（{{lang-en|William Cochran}}）結婚。前一年，威廉雖然參與了[[加利福尼亞淘金潮|加州淘金熱]]卻失望而返，轉而成為一個富裕的乾貨商人暨[[民主党_(美国)|民主黨]]的政治家<ref>{{ cite web | url = http://forgottennewsmakers.com/2010/04/20/josephine-cochrane-1839-1913-invented-the-dishwasher/ | title = Forgotten Newsmakers: JOSEPHINE COCHRANE (1839-1913) Invented the Dishwasher}}</ref><ref>{{cite web|url = http://www.uh.edu/engines/epi1476.htm|title = Engines of our Enginuity|date = 1999|accessdate = 2015-04-04|website = Inventing the Dishwasher|publisher = |last = John|first = Lienhard}}</ref>。兩人育有一子一女，兒子名為赫理·科克倫（{{lang-en|Hallie Cochran}}），於2歲時去世；女兒名為凱瑟琳·科克倫（{{lang-en|Katharine Cochran}}）。<ref>{{cite web|url = http://www.nndb.com/people/166/000165668/|title = NNBD tracking the entire world|date = |accessdate = |website = |publisher = |last = |first = }}</ref>

1913年8月14日，約瑟芬逝世於[[伊利诺伊州|伊利諾州]][[芝加哥|芝加哥]]，據傳死因為[[中風|中風]]或[[疲勞|疲勞]]，葬於[[伊利诺伊州|伊利諾州]]{{tsl|en|Shelbyville, Illinois|謝爾比維爾 (伊利諾州)|謝爾比維爾}}格倫伍德公墓（{{lang|en|Glenwood Cemetery}}）。<ref>{{cite web|url = http://www.cooksinfo.com/josephine-garis-cochrane|title = Cook's Info|date = 1998|accessdate = 2015-04-04|website = Cook's Info|publisher = |last = |first = }}</ref>

== 科克倫發明的自動洗碗機 ==

1870年，科克倫一家遷入一座豪宅，在宅邸中舉辦晚宴時使用的都是17世紀的傳家寶瓷器，有一次僕人們不小心打破盤子後，約瑟芬開始尋找更保險的替代方法來清洗盤子<ref name="Sobey2010">{{cite book|author=Ed Sobey|title=The Way Kitchens Work: The Science Behind the Microwave, Teflon Pan, Garbage Disposal, and More|url=https://books.google.com/books?id=RDfCMTQPMucC&pg=PA41|date=2010-04-01|publisher=Chicago Review Press|isbn=978-1-61374-307-2|pages=41–}}</ref>。她也打算為疲憊的家庭主婦減輕飯後洗碗的負擔<ref>{{cite web|url = http://www.partselect.com/JustForFun/Inventing-the-Dishwasher.aspx|title = Inventing the Dishwasher|date = 2015|accessdate = 2015-04-10|website = Parts Select|publisher = Eldis Group Partnership|last = |first = }}</ref>，上述兩者就成了約瑟芬研發自動洗碗機的動機。

雖然在約瑟芬之前，已有發明家發明並洗碗機並嘗試上市，
1850年，喬爾·霍頓（{{lang-en|Joel Houghton}}）設計了手搖試泡碗機<ref name="Snodgrass2004">{{cite book|author=Mary Ellen Snodgrass|title=Encyclopedia of Kitchen History|url=https://books.google.com/books?id=SJGNAgAAQBAJ&pg=PA320|date=2004-12-29|publisher=Routledge|isbn=978-1-135-45572-9|pages=320–}}</ref>，
1860年代，L·A·亞歷山大（{{lang-en|L. A. Alexander}}）將結構改為齒輪轉動，以期讓盤子放進一桶水的桶內架子上後可以旋轉，但是這些設備都缺乏實效。<ref>{{cite news|url = https://www.independent.co.uk/life-style/gadgets-and-tech/features/the-secret-history-of-the-dishwasher-2119320.html|title = The Secret History of: The Dishwasher|date = 2010-10-29|accessdate = 2014-05-06|website = The Independent|publisher = The Independent|last = Hilpirn|first = Kate}}</ref>

約瑟芬設計了她最初的洗碗機<ref>{{cite | last = Blattman | first = Elissa | title = "Three Every-day Items Invented by Women" | publisher = National Women's History Museum | year = 2013 | url = https://www.womenshistory.org/articles/three-every-day-items-invented-women}}</ref>，由首間洗碗機工廠的技工[[喬治·巴特斯|喬治·巴特斯]]幫忙製作出來。為了製造這台機器，她先測量了餐具尺寸，建造內置電線隔間，每個隔間都經過各別設計以容納杯盤。隔間位於一個平放在銅鍋內的輪子內，馬達會驅動輪子，熱的肥皂水會從鍋爐底部噴到碗盤上。該款洗碗機首創利用水壓清洗碗盤，有別於用洗刷設計的洗碗機。<ref>{{cite web|url = http://woh.pdx.edu/heroine/2725|title = Portland's Walk of the Heroines|date = |accessdate = 2015-04-04|website = Portland's Walk of the Heroines|publisher = |last = Johanna|first = Brenner|deadurl = yes|archiveurl = https://web.archive.org/web/20150508232050/http://woh.pdx.edu/heroine/2725|archivedate = 2015-05-08}}</ref>

1886年12月28日，約瑟芬取得了專利<ref>{{cite web | url = https://www.google.com/patents/US355139 | title = US Patent No. 355139A: Dish-washing machine }}</ref>，1893年，她在[[芝加哥|芝加哥]][[芝加哥哥伦布纪念博览会|哥倫布紀念博覽會]]展出此一發明，不久就湧入了訂單，1897年，洗碗機工廠加里斯—科克倫（{{lang-en|Garis-Cochran}}）開始生產製造洗碗機。
