'''約會'''(又稱為'''拍拖'''，俗稱為'''泡妞'''<ref>[http://www.zdic.net/c/1/164/363117.htm 词语“泡妞”的解释]. 汉典 zdic.net</ref>、'''溝[[女性|女]]'''<ref>[https://zh.glosbe.com/zh/en/%E6%BA%9D%E5%A5%B3 溝女 - 翻译 - 中文-英文]. 字典 - Glosbe. </ref>、'''溝[[男性|仔]]'''<ref>[http://paper.wenweipo.com/2018/07/27/YO1807270001.htm 游蕙禎扮孝順博輕判 溝仔樂不思家].  </ref>或[[wikt:把妹|把妹]]，{{lang-en|dating}})是指預先約定時間地點會面的活動。約會原指一般人士的預約會面，現在通常指[[戀愛|戀愛]]的行為。

== 目的和功用 ==

戀愛約會是人類[[求偶|求偶]]活動其中一環，藉相處交談而了解對方，找出[[配合度_(人際關係)|配合度]]，用作考驗及選擇出一位作為[[配偶|配偶]]，過程中亦同時培養並鞏固[[愛情|愛情]]。

[[單身|單身]]者一般要經歷多次約會，方會成功選擇或被選擇成為配偶。婚後伴侶間的約會用作以促進雙方感情、促進了解，因為人是不斷成長和改變的。婚后若與其他人约会，則屬於精神[[外遇|外遇]]，常會受到[[道德|道德]]譴責。但要注意的是普通朋友或有業務往來的人單獨會面一般不會稱為約會。部份人亦會參加快速配對增加與異性約會機會。

== 活動內容 ==

通常是在工餘（下班後、週末及假期）進行雙方均有興趣的活動，例如[[逛街|逛街]]、看[[電影|電影]]、唱[[卡拉OK|卡拉OK]]、吃喝玩樂、[[郊遊|郊遊]]等，藉相處及交談而瞭解對方。

有一部份人認為 有些情侶未瞭解清楚對方就開始有[[亲密关系|親密行為]]，如[[接吻|接吻]]，以至[[人类性行为|性行為]]，有可能對雙方關係有反面的影響。也有人在親密過後促進了雙方的了解。

== 終止/疏遠 ==

情侶之间不再約會，通常是因为發覺對方並非適合對象，並可能进而終止戀愛，最终成为相互疏远的关系。例如和平分手並維持[[朋友|朋友]]關係，或是分手後不再來往與聯絡，也有人会不歡而散。

== 約會成功後的下一步 ==
如果兩人互相喜歡配合度又高，經過一段時間的相處及了解後，就視乎情况而選擇[[訂婚|訂婚]]或[[結婚|結婚]]。也有人選擇[[同居|同居]]不結婚，原因可能是婚姻制度及註冊法律會反而阻礙他們關係的發展，或認為沒有必要結婚，擔心即使結婚後，將來亦會有分居及離婚的可能性。
