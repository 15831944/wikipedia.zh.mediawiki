'''約翰·亞德·謝菲爾德-巴隆'''，[[英帝國官佐勳章|OBE]]（{{lang-en|John Adrian Shepherd-Barron}}，{{bd|1925年|6月23日|2010年|5月15日|catIdx=John Shepherd-Barron}}）是用於可自助辦理銀行服務設備「[[自動提款機|自動提款機]]」的發明人。

== 生平 ==
=== 早年 ===
出生於[[英屬印度|英屬印度]]時期的[[西隆|西隆]]，返回英國後前往了[[愛丁堡大學|愛丁堡大學]]與[[劍橋大學三一學院|劍橋大學三一學院]]等機構接受教育。在發生[[二次大戰|二次大戰]]時約翰擔任了第159輕砲兵團的[[空降部隊|空降部隊]]。<ref name="Telegraph">{{cite web |url=http://www.telegraph.co.uk/news/obituaries/culture-obituaries/7746651/John-Shepherd-Barron.html |title=John Shepherd-Barron |work=電訊媒體集團 |language=en |date=2010年5月10日 |accessdate=2017年7月16日}}</ref>二戰結束後，約翰於1950年進入印鈔廠[[德拉魯公司|德拉魯公司]]就職。<ref name="BBC6230194">{{cite web |url=http://news.bbc.co.uk/2/hi/business/6230194.stm |title=The man who invented the cash machine |work=[[BBC|BBC]] |author=Brian Milligan  |language=en |date=2007年6月5日 |accessdate=2017年7月16日}}</ref>

=== 自動提款機 ===
1965年某日約翰在銀行結束營運時間後而無法領錢情況下，以路邊的[[巧克力|巧克力]][[販賣機|販賣機]]為靈感，開始構想一種像是販賣機模樣；可讓人自助使用銀行服務的機器。<ref name="BBC6230194"/><ref>{{cite web |url=http://www.telegraph.co.uk/finance/newsbysector/banksandfinance/7743635/John-Shepherd-Barron-cash-machine-inventor-dies.html |title=John Shepherd-Barron, cash machine inventor, dies |work=電訊媒體集團 |author=Auslan Cramb、Scottish Correspondent |language=en |date=2010年5月20日 |accessdate=2017年7月16日}}</ref>

在約翰把他的點子推銷給[[巴克萊銀行|巴克萊銀行]]後，於1967年6月世界上第一台[[自動提款機|自動提款機]]；以「巴克萊現金」（Barclaycash）為名稱在[[倫敦|倫敦]]北方巴克萊銀行[[恩菲爾德鎮|恩菲爾德鎮]]分行旁出現，當時第一位試用此機器的人則是英國演員[[瑞格·瓦尼|瑞格·瓦尼]]。<ref name="BBC6230194"/>

起初約翰想以輸入6個數字密碼來作為自動提款機認證方式，並讓他的妻子卡洛琳來測試使用，但因在過程中卡洛琳個人最多只記得住4個號碼，便更改成僅輸入4個數字即可，也間接成為最初全世界提款機的密碼標準。而約翰一開始訂下的取款限制為不超出10[[英鎊|英鎊]]，這出自於約翰認為10英鎊已足以應付週末時刻在外頭花費的個人評估。<ref name="BBC6230194"/>

2005年約翰因發明自動提款機的關係，得到了[[英帝國官佐勳章|英帝國官佐勳章]]的榮譽。<ref>{{cite web |url=http://news.bbc.co.uk/2/hi/uk_news/scotland/4135269.stm |title=ATM inventor honoured 40 years on |work=BBC |language=en |date=2004年12月31日 |accessdate=2017年7月16日}}</ref>

=== 離世 ===
2010年5月15日約翰在[[蘇格蘭|蘇格蘭]][[印威內斯|印威內斯]]的雷格摩爾醫院（Raigmore Hospital）裡，因疾病以84歲年齡離世。<ref>{{cite web |url=http://news.bbc.co.uk/2/hi/uk_news/scotland/highlands_and_islands/8691747.stm |title=Inventor of cash machine, John Shepherd-Barron, dies |work=BBC |language=en |date=2010年5月19日 |accessdate=2017年7月16日}}</ref>

== 家族 ==
約翰的父親[[威爾弗·謝菲爾德-巴隆|威爾弗·謝菲爾德-巴隆]]曾在[[英屬印度|英屬印度]]的[[孟加拉|孟加拉]][[吉大港|吉大港]]、和倫敦港務局裡擔任工程師，並且是[[英國土木工程師學會|英國土木工程師學會]]的成員。<ref name="Telegraph"/>母親[[朵洛西·謝菲爾德-巴隆|朵洛西·謝菲爾德-巴隆]]則是一名[[網球|網球]]選手，曾拿下[[1924年夏季奧林匹克運動會|1924年夏季奧林匹克運動會]]女子網球雙打銅牌、以及1931年[[溫布頓網球錦標賽|溫布頓網球錦標賽]]女子雙打冠軍。<ref name="Telegraph"/>他的兒子[[尼可拉斯·謝菲爾德-巴隆|尼可拉斯·謝菲爾德-巴隆]]有在[[倫敦國王學院|倫敦國王學院]]裡擔任[[代數幾何|代數幾何]]教授。<ref>{{cite web |url=https://www.kcl.ac.uk/nms/depts/mathematics/people/atoz/Shepherd-Barron.aspx |title=Professor Nicholas Shepherd-Barron |work=[[倫敦國王學院|倫敦國王學院]] |language=en |accessdate=2017年7月16日}}</ref>
