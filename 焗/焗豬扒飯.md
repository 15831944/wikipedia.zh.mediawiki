'''焗豬扒飯'''，'''烤豬扒飯'''（[[英語|英語]]：'''Baked Pork Chop Rice'''，中文簡稱：'''焗豬'''）是[[香港|香港]]的一種[[米飯|米飯]]食品<ref name="headline-20130503">[http://news.stheadline.com/dailynews/headline_news_detail_columnist.asp?id=236412&section_name=wtt&kw=256 焗豬扒飯]，頭條日報，2013-5-3</ref>，常見於香港的[[茶餐廳|茶餐廳]]和[[快餐店|港式快餐店]]。

焗豬扒飯是把已[[煎|煎]]香的豬扒（即[[豬排|豬排]]）蓋在[[蛋|蛋]][[炒飯|炒飯]]上，並加入[[番茄|番茄]]或[[芝士|芝士]]等食材[[焗|焗]]製而成。焗豬扒飯以配上番茄和[[番茄糊|茄膏]]烘烤而成的「鮮茄焗豬扒飯」最為常見，也有焗豬扒飯還會加入菠蘿和芝士等食材。如果把茄醬改為[[白汁|白汁]]並配上芝士烘烤，便會成為「芝士焗豬扒飯」<ref>[http://www.seattlechinesetimes.com/2009-09-02-04-33-29/e/16-e/4102-2010-11-20-00-50-36 港式經典 起司焗豬扒飯] {{webarchive|url=https://web.archive.org/web/20141111080129/http://www.seattlechinesetimes.com/2009-09-02-04-33-29/e/16-e/4102-2010-11-20-00-50-36 |date=2014-11-11 }}，西城時報，2010-11-19</ref>。焗豬扒飯是香港具代表性的西式米飯，是[[港式西餐|港式西餐廳]]、[[茶餐廳|茶餐廳]]及[[港式快餐|港式快餐店]]在[[午飯|午飯]]和[[晚飯|晚飯]]時段常見的[[主餐|主餐]]<ref>[http://www.christinesrecipes.com/2008/07/baked-pork-chops-with-rice.html 焗豬扒飯 - 港式經典焗飯 Baked Pork Chops with Fried Rice]，簡易食譜，2008年7月15日</ref>。

焗豬扒飯源於香港，當香港的食肆在1980年代進入中國大陸市場後，也把焗豬扒飯引入內地，然而當時中國大陸的食物烹調方法，並沒有「焗」的概念，為了令中國大陸的顧客能夠理解這種食品，因此在中國大陸的餐牌常以「[[烤|烤]]」字代替原本的「焗」字，所以又名「烤豬排飯」<ref name="headline-20130503" />。

== 成分 ==
製作鮮茄焗豬扒飯的基本材料，通常包括[[雞蛋|雞蛋]]、[[白飯|白飯]]、[[番茄|番茄]]、[[洋蔥|洋蔥]]、[[油|油]]、[[蒜頭|蒜頭]]、[[鹽|鹽]]、[[糖|糖]]、[[醬油|醬油]]、[[生粉|生粉]]、[[上湯|上湯]]和[[番茄糊|茄膏]]。為增加焗豬扒飯的風味，[[芝士|芝士]]和[[迷迭香|迷迭香]]常被加入焗豬扒飯的醬料。除了番茄外，[[西蘭花|西蘭花]]、[[菠蘿|菠蘿]]和[[南瓜|南瓜]]也常被用於製作焗豬扒飯的蔬菜。

== 基本做法 ==
# 預熱[[焗爐|焗爐]]。
# 豬扒調味。
# 番茄、洋蔥等配料洗淨切件。
# 打[[雞蛋|蛋]]變成蛋汁。
# 蒜頭變[[蒜茸|蒜茸]]，用作「起[[鑊|鑊]]」。
# 豬扒[[煎|煎]]至金黃色，上碟備用。
# [[炒飯|炒飯]]，加蛋汁，[[炒|炒]]勻，上碗。
# 豬扒放於炒飯面。
# 澆上芡汁及上湯於豬扒飯之上。
# 放入已經預熱的焗爐，焗至表面呈金黃色後可以享用。

== 健康問題 ==
焗豬扒飯是香港[[港式快餐店|快餐店]]和茶餐廳常見的[[碟頭飯|碟頭飯]]（[[蓋飯|蓋飯]]），因為材料豐富，所以甚受歡迎，但當中的豬扒和炒飯含有高[[脂肪|脂肪]]和[[納|納]]，常吃並不健康<ref>[http://www.cfs.gov.hk/tc_chi/programme/programme_rafs/files/congee_rice_and_noodles_c.pdf 本地粥粉麪飯的營養素含量]，食物環境衞生署，二零零六年三月</ref><ref>[http://hk.apple.nextmedia.com/news/art/20070525/7138655 焗豬扒飯熱量最高]，蘋果日報，2007-5-25</ref><ref>[http://www.jiujik.com/resources/article/7%E5%A4%A7%E6%9C%80hit%E7%A2%9F%E9%A0%AD%E9%A3%AF-%E7%87%9F%E9%A4%8A%E5%83%B9%E5%80%BC%E5%A4%A7%E8%A7%A3%E7%A2%BC 7大最Hit碟頭飯 營養價值大解碼]{{dead link|date=2018年3月 |bot=InternetArchiveBot |fix-attempted=yes }}，招職，2014-8-15</ref>。
