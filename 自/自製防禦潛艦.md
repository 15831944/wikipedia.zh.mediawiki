'''自製防禦潛艦'''（簡稱 '''潛艦國造'''；{{lang-en| '''Indigenous Defence Submarine '''}}，簡稱{{lang|en|'''IDS'''}}），為[[中華民國海軍|中華民國海軍]]自行研發設計建造[[潛艦|潛艦]]的國防政策，這項政策在2014年[[中華民國海軍司令|海軍司令]][[陳永康_(臺灣軍事將領)|陳永康]][[上將|上將]]任內開始規劃。並於2016年的[[中華民國海軍司令|海軍司令]][[黃曙光|黃曙光]][[上將|上將]]任內首度正項編列預算啟動。

==背景==
2001年[[美國|美國]][[共和黨_(美國)|共和黨]]籍[[美國總統|總統]][[喬治·華克·布希|布希]]宣布同意出售8艘柴油[[潛艦|潛艦]]給臺灣方面，後者因為在野的[[國民黨|國民黨]]和親民黨杯葛[[三項軍購案|三項軍購案]]，無法編列預算購買；2007年通過軍購案，但大幅刪減金額，潛艦預算也遭刪除。在[[民主黨_(美國)|民主黨]]總統[[歐巴馬|歐巴馬]]任期中，臺灣重新向美國爭取採購柴電動力潛艦，美方未正面回應，在時任[[中華民國海軍司令|中華民國海軍司令]][[陳永康|陳永康]][[上將|上將]]任內就對外強烈表達「臺灣無法長期等待」，已經啟動「[[自製防禦潛艦|潛艦國造]]」程序，建造小而精、小而巧、小而強的制海武力<ref>{{Cite web |url=http://www.storm.mg/article/31861 |title=潛艦國造啟動 美方私邀廠商踩剎車 |access-date=2015-05-26 |archive-url=https://web.archive.org/web/20150526155713/http://www.storm.mg/article/31861 |archive-date=2015-05-26 |dead-url=no }}</ref>。

國造潛艦由[[國家中山科學研究院|國家中山科學研究院]]（中科院）負責規劃、研製與整合大約10到12類「戰鬥系統」裝備，而「[[潛望鏡|光學潛望鏡]]」、「通信系統」和「[[聲納|聲納]]系統」等透過技術引進或合作的方式來解決，國造潛艦大約1200噸到3000噸之間。[[臺灣國際造船公司|臺灣國際造船公司]]總經理陳豊霖表示，十二年前，德國專家就證明臺船有建造潛艦能力<ref>{{Cite web |url=http://www.appledaily.com.tw/realtimenews/article/new/20141229/532704/ |title=邱國正：潛艦國造合約設計已經正式啟動 |access-date=2015-05-26 |archive-url=https://web.archive.org/web/20150526155839/http://www.appledaily.com.tw/realtimenews/article/new/20141229/532704/ |archive-date=2015-05-26 |dead-url=no }}</ref><ref>{{Cite web |url=http://news.ltn.com.tw/news/local/paper/792922 |title=台船潛艦國造 先通過修艦資格考 |access-date=2015-05-26 |archive-url=https://web.archive.org/web/20150526154115/http://news.ltn.com.tw/news/local/paper/792922 |archive-date=2015-05-26 |dead-url=no }}</ref>。

[[中華民國國防部|中華民國國防部]]公告「潛艦國造第一階段合約設計」，在2015年底前進行公開招標選商，預計最快在2024年建成臺灣第一艘國造潛艦。[[自製防禦潛艦|潛艦國造]]以3年時間在2016年到2019年進行，來完成潛艦構型設計，而首艘原型潛艦估計要用5年建造，因此從構型設計到首艦建成約10年時間，最快2024年首艘原型潛艦下水，再進行1年[[戰術|作戰]]測試後2025年正式成軍。

2016年度中華民國海軍首次編列5億餘元自製防禦潛艦預算案，為進行IDS的[[自製防禦潛艦|潛艦國造]]合約設計第一步，[[國防部海軍司令部|國防部海軍司令部]][[自製防禦潛艦|潛艦國造]]第一階段合約設計，總經費約新台幣30億元<ref>{{Cite web |url=http://www.appledaily.com.tw/realtimenews/article/politics/20150831/681538/%E5%9C%8B%E9%98%B2%E9%83%A8%E6%AD%A3%E5%BC%8F%E5%95%9F%E5%8B%95%E3%80%80%E8%87%AA%E8%A3%BD%E9%98%B2%E7%A6%A6%E6%BD%9B%E8%89%A6%E5%B0%88%E6%A1%88 |title=國防部正式啟動　自製防禦潛艦專案 |access-date=2015-08-31 |archive-url=https://web.archive.org/web/20150903012838/http://www.appledaily.com.tw/realtimenews/article/politics/20150831/681538/%E5%9C%8B%E9%98%B2%E9%83%A8%E6%AD%A3%E5%BC%8F%E5%95%9F%E5%8B%95%E3%80%80%E8%87%AA%E8%A3%BD%E9%98%B2%E7%A6%A6%E6%BD%9B%E8%89%A6%E5%B0%88%E6%A1%88 |archive-date=2015-09-03 |dead-url=no }}</ref><ref>{{Cite news|title=潛艦國造啟動 明年度編5億預算|url=http://news.ltn.com.tw/news/focus/paper/911652|author=羅添斌|publisher=自由時報|location=台北|date=2015-09-01|accessdate=2016-06-19|language=zh-tw|archive-url=https://web.archive.org/web/20160811004923/http://news.ltn.com.tw/news/focus/paper/911652|archive-date=2016-08-11|dead-url=no}}</ref>。

2016年5月27日，海軍司令[[李喜明|李喜明]]上將出任國防部軍政副部長，海軍副司令[[黃曙光|黃曙光]]直升上將海軍司令，[[蔡英文|蔡英文]]總統打破慣例拔擢現役上將擔任軍政副部長，代表這兩位海軍上將是未來推動「[[自製防禦潛艦|潛艦國造]]」政策的真正的執行者<ref>{{Cite web |url=http://news.ltn.com.tw/news/politics/paper/994410 |title=蔡核定首位上將 黃曙光升海軍司令 |access-date=2016-05-28 |archive-url=https://web.archive.org/web/20160528123552/http://news.ltn.com.tw/news/politics/paper/994410 |archive-date=2016-05-28 |dead-url=no }}</ref>。

2016年6月20日，海軍受邀參與高雄國際海事船舶暨國防工業展的台北邀商說明會，並在會上公布規劃的12項造艦案規劃，當中包括「[[自製防禦潛艦|潛艦國造案]]」<ref>{{Cite news|url=http://www.appledaily.com.tw/realtimenews/article/new/20160619/889088/|title=展現國艦國造決心　海軍公布12項造艦案|author=王烱華|publisher=[[蘋果日報_(台灣)|蘋果日報]]|location=台北|date=2015-06-19|accessdate=2016-06-19|language=zh-tw|archive-url=https://web.archive.org/web/20160619035658/http://www.appledaily.com.tw/realtimenews/article/new/20160619/889088/|archive-date=2016-06-19|dead-url=no}}</ref>，另外國防部表示可能會與[[以色列|以色列]]合作<ref>{{Cite news|url=http://www.setn.com/News.aspx?NewsID=157307|title=不排除找以色列助我國艦國造　海軍：22年斥資4700億|author=|publisher=[[三立電視|三立電視]]|location=台北|date=2016-06-20|accessdate=2016-06-20|language=zh-tw|archive-url=https://web.archive.org/web/20160621125246/http://www.setn.com/News.aspx?NewsID=157307|archive-date=2016-06-21|dead-url=no}}</ref>。

2016年8月1日，臺船將於2011年就成立的任務編組「潛艦小組」納入正式編制，提昇成立「臺船潛艦發展中心」（Submarine Development Center of CSBC,SDCC），臺船董事長擔任中心召集人、臺船總經理擔任中心副召集人，臺船副總經理擔任中心主任，以配合[[蔡英文政府|蔡英文政府]]力推「國艦國造、[[自製防禦潛艦|潛艦國造]]」政策<ref>[http://udn.com/news/story/1/1865995 力推「國艦國造、[[自製防禦潛艦|潛艦國造]]」] {{webarchive|url=https://web.archive.org/web/20160927222514/http://udn.com/news/story/1/1865995 |date=2016-09-27 }}陳維鈞 2016-8-1 聯合報</ref><ref>{{Cite web |url=http://www.csbcnet.com.tw/monthly_pub/data/480/01%E5%85%AC%E5%8F%B8%E5%A0%B1%E5%B0%8E/01%E5%8F%B0%E8%88%B9%E5%85%AC%E5%8F%B8%E3%80%8C%E6%BD%9B%E8%89%A6%E7%99%BC%E5%B1%95%E4%B8%AD%E5%BF%83%E3%80%8D%E6%88%90%E7%AB%8B%20%E5%85%A8%E5%8A%9B%E6%8E%A8%E5%8B%95%E3%80%8C%E6%BD%9B%E8%89%A6%E5%9C%8B%E9%80%A0%E3%80%8D.pdf |title=台船公司「潛艦發展中心」成立 全力推動「[[自製防禦潛艦|潛艦國造 |access-date=2017-08-18 |archive-url=https://web.archive.org/web/20170818215259/http://www.csbcnet.com.tw/monthly_pub/data/480/01%E5%85%AC%E5%8F%B8%E5%A0%B1%E5%B0%8E/01%E5%8F%B0%E8%88%B9%E5%85%AC%E5%8F%B8%E3%80%8C%E6%BD%9B%E8%89%A6%E7%99%BC%E5%B1%95%E4%B8%AD%E5%BF%83%E3%80%8D%E6%88%90%E7%AB%8B%20%E5%85%A8%E5%8A%9B%E6%8E%A8%E5%8B%95%E3%80%8C%E6%BD%9B%E8%89%A6%E5%9C%8B%E9%80%A0%E3%80%8D.pdf |archive-date=2017-08-18 |dead-url=no }}</ref>。

2017年3月21日，[[蔡英文|蔡英文]]總統南下高雄左營區，出席「海軍106年敦睦支隊啟航歡送暨潛艦國造設計啟動及和合作備忘簽署儀式」，見證國防部、中科院、台船等單位簽署[[自製防禦潛艦|潛艦國造]]備忘錄，開始「[[自製防禦潛艦|潛艦國造]]」計畫<ref>顏振凱，[http://www.storm.mg/article/236402 國際政治要「先自助而人助」蔡英文：潛艦國造正式啟動] {{Wayback|url=http://www.storm.mg/article/236402 |date=20170321074744 }}，風傳媒，2017-03-21</ref>。[[臺船|臺船]]公司董事長鄭文隆表示，[[原型_(工程)|原型]][[潛艦|艦]]合約設計須四年時間，再花三、四年建造，如自2016年12月起算，預計八年內潛艦原型艦下水，十年內潛艦成軍<ref>{{Cite web |url=http://news.ltn.com.tw/news/focus/paper/1087844 |title=潛艦國造啟動 8年下水 |access-date=2017-08-17 |archive-url=https://web.archive.org/web/20170818004807/http://news.ltn.com.tw/news/focus/paper/1087844 |archive-date=2017-08-18 |dead-url=no }}</ref>。

8月23日臺灣國造潛艦最難自製的「紅區裝備」（如[[潛望鏡|光學潛望鏡]]、[[通信|通信]][[系統|系統]]、[[動力系統|動力系統]]、[[絕氣推進系統|絕氣推進系統]]、[[聲納|聲納]]系統和[[主機|船舶主機]]等）已經取得重大突破，將採用[[法國|法國]]{{link-en|船舶建造局|DCNS (company)}}（DCNS）與[[西班牙|西班牙]]Navantia公司（IZAR）聯合研發製造的[[鮋魚級潛艇|鮋魚級潛艦]]衍生型{{link-en|S-80型潛艦|S-80-class submarine}}紅區裝備，而「戰鬥系統裝備」採用[[美國|美國]][[洛克希德馬丁|洛克希德馬丁]]。自製防禦潛艦打造八艘，每艘四百億元[[臺幣|臺幣]]（約13億[[美金|美金]]）<ref>{{Cite web |url=https://tw.news.yahoo.com/%E6%BD%9B%E8%89%A6%E5%9C%8B%E9%80%A0%E9%87%8D%E5%A4%A7%E7%AA%81%E7%A0%B4-%E7%B4%85%E5%8D%80%E8%A3%9D%E5%82%99%E5%95%86%E6%BA%90%E5%82%99%E4%BE%BF-091823335.html |title=潛艦國造重大突破 紅區裝備商源備便 |access-date=2017-08-23 |archive-url=https://web.archive.org/web/20170824012531/https://tw.news.yahoo.com/%E6%BD%9B%E8%89%A6%E5%9C%8B%E9%80%A0%E9%87%8D%E5%A4%A7%E7%AA%81%E7%A0%B4-%E7%B4%85%E5%8D%80%E8%A3%9D%E5%82%99%E5%95%86%E6%BA%90%E5%82%99%E4%BE%BF-091823335.html |archive-date=2017-08-24 |dead-url=no }}</ref>。

2018年4月7日美國國務院公告「行銷核准證」（marketing license)，協助台灣潛艦國造計畫「紅區裝備」。<ref>{{Cite web |url=https://udn.com/news/story/10930/3073682 |title=美將核我潛艦國造「行銷核准證」 專家：技協輸出許可 |access-date=2018-04-10 |archive-url=https://web.archive.org/web/20180410201916/https://udn.com/news/story/10930/3073682 |archive-date=2018-04-10 |dead-url=no }}</ref>
