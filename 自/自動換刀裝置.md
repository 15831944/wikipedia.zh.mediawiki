'''自動換刀裝置'''（Automatic tool changer）簡稱'''ATC'''，是[[数控机床|数控机床]]上的設備，可以自動更換刀具，提昇設備產能以及增加支援刀具數量。自動換刀裝置可以快速更換[[機床|機床]]上的刀具，減少非運行時間。自動換刀裝置一般是為了提昇機床能力，可以配合多種刀具所設計。自動換刀裝置也可以用來更換磨損或是損壞的刀具。可以讓機床朝[[自動化|自動化]]的目標更進一步<ref>{{cite journal|last1=OBREA|first1=C[laudiu] F[lorin]|last2=PASCU|first2=M[arius]|last3=MIHAILA|first3=L[ucian]|last4=FUNARU|first4=Marian|title=DESIGN OF AN AUTOMATIC TOOL CHANGER SYSTEM FOR MILLING MACHINING CENTERS|journal=DAAAM International|date=2012|volume=23|issue=1|pages=1139–1142|url=http://www.daaam.info/Downloads/Pdfs/proceedings/proceedings_2012/1139_Obreaatal.pdf|accessdate=2018-08-29}}</ref>。

== 說明 ==
簡易的工具機只能配合單一刀具作業。刀塔（Turret）式工具機可以配合較多刀具，但若需要的刀具更多，就需要
配合自動換刀裝置了。自動換刀裝置會配合存放未使用刀具的刀庫一起設計，可以讓機床在沒有人員介入的情形下，自動用多種刀具進行加工。自動換刀裝置的主要部份包括基座、夾取臂、刀具固定架、支承臂及刀庫<ref>{{cite web|last1 = scribd|title = Automatic tool changer|url = https://www.scribd.com/doc/51698181/Automatic-Tool-Changer#scribd|website = scribd|publisher = Rahul Singh|accessdate =2018-08-29}}</ref>。自動換刀裝置雖提昇了速度及精準度，但相較於人工的刀具更換，也有一些額外產生的問題，例如刀具必須非常容易對正、換刀裝置要容易夾取刀具，也要有簡單的方式讓刀具脫離夾具。因此，ATC專用的刀具會放在特別設計的刀夾內<ref>{{cite web|last1=Laboratory for Manufacturing Systems and Automation |title=COMPUTER NUMERICAL CONTROL OF MACHINE TOOLS |url=http://www.lms.mech.upatras.gr/LMS/files-1/students-area/arithmetikos-elegkhos-ergaleiomekhanon/subject-files/ekpaideutiko-uliko/CNCChapter4_11122013_LOCKED.pdf |website=lms.mech.upatras.gr |publisher=Laboratory for Manufacturing Systems and Automation |accessdate=2018-08-29 |deadurl=yes |archiveurl=https://web.archive.org/web/20150402170424/http://www.lms.mech.upatras.gr/LMS/files-1/students-area/arithmetikos-elegkhos-ergaleiomekhanon/subject-files/ekpaideutiko-uliko/CNCChapter4_11122013_LOCKED.pdf |archivedate=2015-04-02}}</ref>。

==自動換刀裝置的分類==
自動換刀裝置可以依刀庫的外形不同而分為二種：鼓式的換刀裝置適用於刀具數量小於三十把的情形，刀具會在鼓式刀庫上平行排放。鏈式的換刀裝置適用於刀具數量大於三十把的情形（實際數字依設計及製造商而不同，不過鼓式換刀裝置的刀具數量一定會小於鏈式換刀裝置），但鏈式換刀裝置的刀具搜尋速度會比較慢<ref>{{cite web|last1 = IGNOU|title = Tooling for CNC machines|url = http://www.ignou.ac.in/upload/Unit-5-57.pdf|website = ignou.ac.in|publisher = IGNOU|accessdate = 2018-08-29}}</ref>。

==自動換刀裝置機構==
ATC在收到換刀指令後，刀庫會將要使用的刀具移動到一固定位置（預備換刀位置）。ATC的換刀臂到此位置將刀具取出，換刀臂會在加工主軸以及刀庫之間旋轉。換刀臂的兩邊都各有一個夾子，<!--Each gripper can rotate 90°, to deliver tools to the front face of the turret.-->一個夾子將不用的刀具從加工刀塔上取下，另一個夾子從刀庫中取下需要使用的刀具，然後旋轉180度，將不用的刀具放回刀具，並且將要使用的刀具更換到加工主軸上。

==ATC的機能==
自動換刀裝置可以提高生產時間，減少非生產時間。ATC可以儲存刀具，而且是在加工後會自動放回刀庫中，因此機床的靈活程式較高。比較可以使用大型且較重的刀具<!--，而且可以進行刀具切削邊的自動renewal--><ref>{{cite web|last1 = NITC|title = Tool Management in Advance Manufacturing Systems|url = http://www.nitc.ac.in/dept/me/jagadeesha/Tool_Engineering_and_Design/CHAPTER12.pdf|website = www.nitc.ac.in|publisher = Dept. of M.E.|accessdate = 2018-08-29}}</ref>。
