'''《自言帖》'''為[[唐朝|唐朝]]中期的知名書法家「草聖」<ref>{{Cite web|url=http://classweb.loxa.edu.tw/jsn9100/data1/story/067.htm|title=草聖張旭|accessdate=2018-06-14|work=classweb.loxa.edu.tw}}</ref>[[張旭|張旭]]之作品，張旭傳世的作品有《古詩四帖》、《肚痛帖》、《自言帖》及楷書《郎官石記》等，自言帖為其代表作品之一。

== 自言帖原文 ==
醉顛嘗自言意。始見公主擔夫爭道。又聞鼓吹而得筆法。及觀公孫大娘舞劍而得其神。自此見漢張芝草書入聖。甚復發顛興耳。

印記款識：有唐開元二年八月望。顛旭醉書。

== 作品由來 ==
張旭性嗜酒，常於醉中以頭髮濡墨而書，故世稱「張顛」。<ref>{{Cite web|url=http://m.yac8.com/news/7634.html|title=颠旭醉书:张旭草书《自言帖》特大图片书法碑帖书法欣赏|accessdate=2018-06-14|work=m.yac8.com}}</ref>

狂草作品《自言帖》中提到其書法靈感來自觀看[[公孫大娘|公孫大娘]]及其弟子練劍時的姿態。<ref>張彥遠《歷代名畫記·吳道子》：「時又有公孫大娘，亦善舞劒器，張旭見之，因為之草書。」</ref><ref>杜甫《觀公孫大娘弟子舞劍器行並序》：「吳人張旭善草書帖，數嚐於鄴縣見雲公孫大娘舞西河劍器，自此草書長進，豪蕩感激。」</ref>張旭的《自言帖》云：「醉顛嘗自言，始見公主擔夫爭道，又聞鼓吹而得筆法，及觀公孫大娘舞劍而得其神。」<ref>{{Cite web|url=http://www.uruur.com/clementi/m4b.php|title=自言帖|accessdate=2018-06-14|work=www.uruur.com}}</ref>（白話文：開始時，我聽說公主與挑夫爭著走路而悟得草書筆法的意境。後來觀公孫大娘舞劍而悟得草書筆法的神韻。）

== 評價 ==
《自言帖》的筆勢雄偉奔放，意氣縱橫自如，能傳達張旭那種寄託懷抱，激發感情。<ref>{{Cite web|url=http://www.cityu.edu.hk/lib/about/event/ch_calligraphy/cursive_chi.htm|title=書體源流與書法賞析 - 草書|accessdate=2018-06-14|work=www.cityu.edu.hk|deadurl=no|archiveurl=https://web.archive.org/web/20180427120421/http://www.cityu.edu.hk/lib/about/event/ch_calligraphy/cursive_chi.htm|archivedate=2018-04-27|df=}}</ref>

* [[唐朝|唐]][[杜甫|杜甫]]《飲中八仙歌》評價張旭與其作品：「張旭飲酒三杯後即揮毫作書，稱爲草聖。他常不拘小節，在王公貴戚面前脫帽露頂，揮筆疾書，若得神助，其書如雲煙之瀉於紙張。」<ref>{{Cite web|url=https://fanti.dugushici.com/ancient_proses/10177|title=飲中八仙歌_原文、翻譯及賞析_杜甫詩詞_讀古詩詞網|accessdate=2018-06-14|work=fanti.dugushici.com}}</ref><ref>杜甫《飲中八仙歌》：張旭三杯草聖傳，脫帽露頂王公前，揮毫落紙如雲煙。</ref>
* [[唐朝|唐]][[韩愈|韓愈]]《送高閑上人序》敘述張旭草書作品，是以情感为核心的表現過程。韓愈站在積極入世的儒家主義上肯定了張旭的書法：「張旭善寫草書不涉及其他技藝。喜怒、窘困、憂悲、愉悅、怨恨、思慕、酣醉、無聊、不平，每有心領神會，必以草書抒發出來。看到山水崖谷、鳥獸蟲魚、草木花果、日月星辰、風雨雷電、歌舞戰鬥等天地間的萬千變化，不管喜或驚，皆以草書來表現其情感。所以張旭的書法神出鬼沒而變幻莫測。」<ref>{{Cite web|url=https://zh.wikisource.org/zh-hant/%E9%80%81%E9%AB%98%E9%96%92%E4%B8%8A%E4%BA%BA%E5%BA%8F|title=送高閒上人序 - 維基文庫，自由的圖書館|accessdate=2018-06-14|work=zh.wikisource.org|language=zh-Hant|quote=張旭善草書，不治他技。喜怒、窘窮、憂悲、愉佚、怨恨、思慕、酣醉、無聊、不平、有動於心、必於草書焉發之。觀於物，見山水崖谷、鳥獸蟲魚、草木之花實、日月列星、風雨水火、雷霆霹靂、歌舞戰鬥，天地事物之變，可喜可愕，一寓於書。故旭之書，變運猶鬼神，不可端倪。|deadurl=no|archiveurl=https://web.archive.org/web/20151122210639/https://zh.wikisource.org/zh-hant/%E9%80%81%E9%AB%98%E9%96%92%E4%B8%8A%E4%BA%BA%E5%BA%8F|archivedate=2015-11-22|df=}}</ref>
