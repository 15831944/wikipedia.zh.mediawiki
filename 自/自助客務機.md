'''自助客務機'''為[[港鐵|港鐵]]於2007年7月推出的機器，設於[[香港站|香港站]]及[[青衣站|青衣站]]的[[東涌綫|東涌綫]]大堂、[[黃埔站_(香港)|黃埔站]]西大堂，以及[[南港島綫|南港島綫]]沿途各站，以方便乘客在無需前往[[港鐵客務中心|客務中心]]的情況下自行處理票務問題。

== 歷史 ==
[[港鐵|港鐵]]在2007年7月19日宣佈，將會推出首部自助客務機，並於7月22日在[[香港站|香港站]]率先投入服務，每部機成本為70萬[[港元|港元]]，屬於試驗性質<ref>{{cite news|title=地鐵新設自助客務機|url=http://hk.apple.nextmedia.com/news/art/20070720/7345363|accessdate=2012-08-21|newspaper=蘋果日報|date=2007-07-20}}</ref>。其後，港鐵亦於2009年於青衣站增設自助客務機。<ref>{{cite news|title=西鐵日增三萬客逼爆紅磡|url=http://orientaldaily.on.cc/cnt/news/20091224/00176_029.html|accessdate=2012-08-21|newspaper=東方日報|date=2009-12-24}}</ref>

== 服務範圍 ==
自助客務機提供下列服務：
* 查詢[[八達通卡|八達通卡]]交易記錄
**乘客亦可付款$3以列印最近十次的交易記錄（該付款可以另外索取收據）
* 處理出入閘問題
**沒有有效入閘記錄：乘客可持八達通卡前往此機坦白輸入入閘車站，加註入閘記錄，便會自動計算車費，其處理方式與客務中心相同；使用此功能多於特定次數八達通卡會自動鎖卡，以防濫用<ref>{{cite web|title=[新聞](2007-07-19)地鐵香港站添置自助客務機(第2段)|url=http://www.youtube.com/watch?v=cA39u9llpTg|accessdate=2012-08-21}}</ref>
**補票：持單程票乘客如車程超出原有購票路段，可以補付車費差額
**遺失車票：乘客需繳付最高車費，之後系統會發出出閘票供乘客離開車站付費區
* 購買單程車票（可以索取收據及只發售普通等車票）（金鐘、香港及青衣站只接受現金（硬幣及紙幣，紙幣面額為$100或以下），而黃埔、海洋公園、黃竹坑、利東及海怡半島站接受則以現金（硬幣及紙幣，紙幣面額為$100或以下）或八達通）
* 購買[[港鐵都會票|港鐵都會票]]（可以索取收據）（不適用於金鐘、香港及青衣站）
自助客務機亦設有視像系統，乘客可以點選「聯絡職員」，視像系統即會啟動，乘客可與職員直接對話，在顯示屏中會出現職員的面貌，但港鐵在機上亦寫著如果需要進一步查詢建議乘客前往客務中心。雖然設有視像系統，但由於部份功能無法於此機執行（如購買全月通、購買八達通卡等），故並非所有乘客都會使用此機。值得一提的是，此機設置的位置並不靠近客務中心，甚至該層並不設有客務中心（如青衣站的自助客務機設置於U2層，最近的客務中心位於U1大堂層或U4往東涌方向月台）。

不過，由於此機在購買[[港鐵單程票|單程車票]]時可以列印收據，而單程票售票機無法做到此功能，故亦有乘客會使用本機購票。

如欲開始使用本機，乘客只需點選「車票」或「八達通卡」，之後根據指示選擇即可。

== 設置位置 ==
此機設置於下列車站：
* [[香港站|香港站]]：L3 東涌綫大堂（近E及F出口）
* [[青衣站|青衣站]]：U2 東涌綫大堂及4號月台（往[[香港站|香港]]方向，近G出口）
* [[黃埔站_(香港)|黃埔站]]：C 西大堂（近A及B出口）
* [[南港島綫|南港島綫]]沿途各站： [[金鐘站|金鐘站]]（近E出口）、[[海洋公園站|海洋公園站]]（近C出口）、[[黃竹坑站|黃竹坑站]]（A及B出口之間）、[[利東站|利東站]]（近A出口）及[[海怡半島站|海怡半島站]]（近A及B出口）
