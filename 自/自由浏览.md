'''自由浏览'''（{{lang-en|FreeBrowser}}）是一款由[[GreatFire|GreatFire]]<ref>{{cite web |author1=Alex Hern |title=GreatFire activist urges western firms to help end Chinese censorship |url=https://www.theguardian.com/world/2016/apr/14/greatfire-activist-urges-western-firms-to-help-end-chinese-censorship |website=[[卫报|卫报]] |accessdate=2018-06-11 |language=en |date=2016-04-14}}</ref>基于[[Chromium|Chromium]]项目开发的[[Android|安卓]][[翻墙|翻墙]][[浏览器|浏览器]]，于2015年2月发布。<ref>{{cite web |author1=Patrick Howell O'Neill |title=How Chinese Internet users are climbing a bigger, badder Great Firewall |url=https://www.dailydot.com/layer8/latern-china-freebrowser-great-firewall/ |website=The Daily Dot |accessdate=2018-06-11 |language=en |date=2015-11-06}}</ref>官方宣称其为免费的安卓应用，可以访问[[中华人民共和国被封锁网站列表|被中国大陆封锁的网站]]。该应用亦使用[[AutoProxy|AutoProxy]]技术，因此大陆网站可以不经[[代理服务器|代理]]直接连接。

2017年9月，官网改版，该应用也随之增加大量敏感权限，有网友怀疑该[[软件|软件]]可能已被中共控制。<ref>{{cite web |author1=guo392 |title=Issues#198 政府已经控制了GreatFire自由浏览和相关人员，使用自由浏览进行刑侦 

现在（2019.1.11），自由浏览连接已经非常不稳定，多数情况无法访问。首页大量仇视大陆政府的言论。网友猜测可能自由浏览已经被反对大陆政府的组合或者个人控制，也可能是单纯的是软件团体对大陆政府的不满情绪。
