《'''自由之翼'''》（{{lang-en|''Bird Set Free''}}）是歌手[[希雅|希雅]]的一首歌曲，出自其第7张专辑《[[超有戏|超有戏]]》<ref name=Gordon>{{cite web|last1=Gordon|first1=Jeremy|title=Sia Shares "Bird Set Free"|url=http://pitchfork.com/news/61928-sia-shares-bird-set-free/|publisher=[[Pitchfork_Media|Pitchfork Media]]|accessdate=2015-11-03|date=2015-11-03}}</ref>。

==背景==
希雅在接受《[[滚石杂志|滚石杂志]]》的访问时说这首歌原是创作给[[完美音调2#音乐|《完美音调2》唱片]]，但是拒绝这首歌而使用了《Flashlight》作为插曲。之后她给了[[蕾哈娜|蕾哈娜]]，但是遭到拒绝。在之后给了[[愛黛儿|愛黛儿]]，她一开始选择录制但最后还是给回了富勒<ref>[http://www.rollingstone.com/music/features/sias-reject-opus-songwriter-on-reclaiming-adele-rihannas-unwanted-hits-20151203 Sia talks reclaiming unwanted hits]</ref>。

==词曲==
《自由之翼》以[[F小調|F小調]]写成<ref>{{cite web|title=Product MN0159570 "Bird Set Free"|url=http://www.musicnotes.com/sheetmusic/mtd.asp?ppn=MN0159570|website=Music Notes|publisher=EMI Music Publishing|accessdate=9 February 2016}}</ref>。

==推广==
2015年11月7日，希雅在《[[週六夜现场|週六夜现场]]》表演《自由之翼》和《我活著》<ref>{{cite web|title=Sia Playing SNL with Host Donald Trump|url=http://www.stereogum.com/1837238/sia-playing-snl-with-host-donald-trump/news/|publisher=[[Stereogum|Stereogum]]|accessdate=October 14, 2015|date=October 13, 2015|first=Gabriela Tully|last=Claymore}}</ref><ref>{{cite web|title=Donald Trump Hosts 'SNL', Sia Performs|url=http://popcrush.com/donald-trump-hosts-snl-sia-performs-watch-the-clips-here/|publisher=PopCrush|accessdate=November 8, 2015|date=November 7, 2015|first=Erica|last=Russell}}</ref>。
