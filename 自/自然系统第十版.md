《'''[[自然系统|自然系统]]'''》'''第十版'''是由生物学家[[卡尔·林奈|卡尔·林奈]]所撰写的书籍，于1758年和1759年分别出版上下两卷。这一版本的《自然系统》标志了[[国际动物命名法规|动物命名法规]]走向系统化的开始。在这本书中，林奈为动物定义了[[二名法|二名法]]命名规则（此前他曾在1753年出版的《[[植物种志|植物种志]]》中介绍了类似的植物命名法则）。

== 开端 ==
1758年以前，包括《[[自然系统|自然系统]]》较早的版本在内，大部分生物学分类都会用多名法为物种命名。但对[[动物界|动物界]]首次进行[[二名法|二名法]]命名的则是《自然系统》的第十版。对此，[[国际动物命名法委员会|国际动物命名法委员会]]宣布1758年1月1日为“命名法”的起点，并要求《自然系统》第十版被视作在那天发行。<ref name="Art3">{{cite book |url=http://www.nhm.ac.uk/hosted-sites/iczn/code/index.jsp?article=3&nfv=true |chapter=Article 3 |title=International Code of Zoological Nomenclature |edition=4th |isbn=0-85301-006-4 |year=1999}}</ref>在此前命名的物种，哪怕符合命名要求，也被宣布无效。

对此铁律唯一的例外是昆虫学家{{tsl|en|Carl Alexander Clerck||卡尔·亚历山大·克拉克}}的《{{lang|la|Aranei Suecici}}》。这本书实际于1757年出版，但人们也当它作1758年1月1日出版的书籍来对待。<ref name="Art3"/>
