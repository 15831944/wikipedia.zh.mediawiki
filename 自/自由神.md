《'''自由神'''》，是1935年中华民国时期的一部黑白剧情类电影，发行公司为[[电通影片公司|电通影片公司]]。导演[[司徒慧敏|司徒慧敏]]、编剧[[夏衍|夏衍]]、摄影[[杨霁明|杨霁明]]，[[王莹_(电影演员)|王莹]]、施超、[[周伯勋|周伯勋]]担任主演，[[吴湄|吴湄]]、[[江青|藍蘋]]（江青）参演。电影主题歌《自由神》由[[孙师毅|孙师毅]]填词，[[吕骥|吕骥]]谱曲<ref>{{Cite book|title=吕骥纪念选集 歌曲卷|last=中国音乐家协会编|first=|publisher=北京：人民音乐出版社|year=2010.04|isbn=978-7-103-03793-5|location=|pages=4-5}}</ref>。

== 背景 ==
1933年3月，[[中共上海中央局|中共上海中央局]]文化工作委员会（文委书记[[阳翰笙|阳翰笙]]）成立“中国共产党电影小组”，组长[[夏衍|夏衍]]，组员[[钱杏邨|钱杏邨]]、[[王尘无|王尘无]]、[[石凌鹤|石凌鹤]]、[[司徒慧敏|司徒慧敏]]。小组通过司徒慧敏的堂兄[[司徒逸民|司徒逸民]]1933年开设了“电通电影器材制造公司”，该公司在1934年春改组为[[电通影业公司|电通影业公司]]，拍摄电影，左翼影界人士纷纷转入该公司。先后拍摄了《[[桃李劫|桃李劫]]》、《[[风云儿女|风云儿女]]》<ref>{{Cite book|title=上海革命史资料与研究|last=中共“一大”会址纪念馆，上海革命历史博物馆筹备编|first=|publisher=上海：上海古籍出版社|year=2012.12|isbn=978-7-5325-6733-1|location=|pages=510}}</ref>。1934年，夏衍在被迫退出[[上海明星公司|明星公司]]后，转移到联华公司，担任联华二厂的特约编剧，协助剧本创作，还为电通影片公司编写了电影剧本《自由神》<ref>{{Cite book|title=广东通史 现代下|last=方志钦，蒋祖缘主编|first=|publisher=广州：[[广东高等教育出版社|广东高等教育出版社]]|year=2014.12|isbn=978-7-5361-4723-2|location=|pages=1048}}</ref>（1935年2月[[中共江苏省委|中共江苏省委]]及上海文委被破坏，电影署司徒慧敏编剧）<ref>{{Cite book|title=“电影小组”与左翼电影运动|last=吴海勇著|first=|publisher=上海：上海人民出版社|year=2014.06|isbn=978-7-208-12342-7|location=|pages=127}}</ref>。1935年春，司徒慧敏导演此剧，这也是他的导演处女作<ref>{{Cite book|title=中国电影年鉴 2011 总第31卷|last=中国电影年鉴社编；俞小一主编|first=|publisher=中国电影年鉴社|year=2011|isbn=|location=|pages=608}}</ref>。由于司徒慧敏亲身经历了[[沙基惨案|沙基惨案]]，所以他在影片中穿插了不少新闻纪录片的镜头，以增强影片的时代感<ref>{{Cite book|title=广东文史资料 第60辑 群星璀璨：广东文化名人录|last=中国人民政治协商会议广东省委员会文史资料研究委员会|first=|publisher=广州：[[广东人民出版社|广东人民出版社]]|year=1989.11|isbn=7-218-00385-0|location=|pages=5-6}}</ref>。

== 剧情 ==
[[File:“自由神”剧照_《电通》_1935_第6期.png|thumb]]
[[File:“自由神”導演司徒慧敏居中，曁主演者：王螢、周伯勳_《电通》_1935_第1期.png|thumb]]
1919年，五四运动时期，青年学生陈行素（[[王莹_(电影演员)|王莹]] 饰）爱上了同学林之彬（施超 饰），与其离家出走；不久，两人带着儿子到了广州，林之彬在[[广州起义_(1927年)|广州起义]]时牺牲，陈行素带着孩子投靠旧友杨棣华（[[周伯勋|周伯勋]] 饰），杨棣华细心照顾陈行素母子并对陈行素表示了爱慕，却冷淡了一直喜欢他的表妹汤秀云（[[吴湄|吴湄]] 饰）。[[國民革命軍北伐|北伐]]开始后，陈行素随军北上，途中她认识青年军官周范（[[顾梦鹤|顾梦鹤]] 饰），两人很快相爱。北伐结束后，陈行素和周范一道来到上海。这时，已到上海行医的杨棣华和汤秀云抱着陈行素的儿子，到黄埔码头迎接陈行素。当杨棣华发现陈行素和周范在一起的情形时，他将孩子交给陈行素，怅然离去。之后杨棣华和汤秀云结婚了。不久，陈行素发现自己所爱的周范是有妇之夫，而且已有了孩子，她的心灵受到伤害，断然拒绝周范的苦苦哀求。[[一·二八事变|一·二八事变]]中，陈行素与儿子失散，但看到更多无家可归的孤儿们后，她忍苦投身教养院，照顾战争中的遗孤<ref name=1s>{{Cite book|title=中国电影名片鉴赏辞典|last=程树安主编|first=|publisher=北京：长征出版社|year=1997.07|isbn=7-80015-360-6|location=|pages=80-81}}</ref>。电影最后，陈行素教孩子们唱《自由神之歌》（电影主题曲），讲述道：人要靠自己的力量，做一个“堂堂的人”<ref>{{Cite book|title=二十世纪中国实录 3|last=《二十世纪中国实录》编委会主编|first=|publisher=北京：[[光明日报出版社|光明日报出版社]]|year=1997|isbn=7-80091-647-2|location=|pages=2301}}</ref><ref>{{Cite book|title=中国影片大典 故事片·戏曲片 1931-1949.9|last=王珍珍主编；中国电影艺术研究中心，中国电影资料馆编|first=|publisher=北京：中国电影出版社|year=2005.12|isbn=7-106-02414-7|location=|pages=1935}}</ref>。

== 评价 ==
电影放映后，上海各大日报对《自由神》进行了评价，其中平野在《[[申报|申报]]》中称，“《自由神》是从‘五四’一直到最近的革命运动的记录。在国产片中，这是一张仅有的有意义的历史片。”并赞扬摄影师[[杨霁明|杨霁明]]、演员施超、蓝蘋、[[任克|任克]]、[[蔡维宁|蔡维宁]]等人的演技。承达在《[[新闻报_(上海)|新闻报]]》中称赞电影的处理技术，并称“导演人司徒慧敏是以最新的姿态出现在这片中的。他对镜头，很能活用。”并称赞蓝蘋“以抽烟而饮弹，虽是袭用了《[[西线无战事|西线无战事]]》的一节，但并没有画蛇添足之感。”<ref>{{Cite book|title=电通半月画报 老上海电影期刊经典 1935年5月-1935年11月|last=冯沛龄著|first=|publisher=上海：上海社会科学院出版社|year=2010.08|isbn=978-7-80745-731-2|location=|pages=122-127}}</ref>旅冈在《上海民报》中称“《自由神》的题材是采取得很有意义的，它反映了中国近代史中最光辉灿烂的一页。”李一在《时事新闻》中赞扬此部电影“更给予了观众一种确信，恋爱的结果不是所谓男主于外女主于内，而是在工作上的相互合作，相互勉励。”<ref>{{Cite book|title=中国电影文献史料选编 电影评论卷 1921-1949|last=陈多绯编|first=|publisher=北京：[[中国电影出版社|中国电影出版社]]|year=2014.06|isbn=978-7-106-03945-5|location=|pages=510-517}}</ref>

在之后，[[姚苏凤|姚苏凤]]主持了一起《[[每日电影|每日电影]]》座评会，邀请[[葉靈鳳|叶灵凤]]、[[劉吶鷗|刘呐鸥]]、[[江兼霞|江兼霞]]、高明、[[穆时英|穆时英]]参加（后载于1935年8月25日上海《晨报》），众位戏剧家与电影家对此电影各方面进行了批评和分析。江兼霞、姚苏凤、穆时英认为王莹的演技无法支撑整一个大时代的女性。高明、穆时英批评编剧用悲剧色彩描述国民革命是不合适的。叶灵凤、刘呐鸥认为电影剧本历史框架太长，致使牺牲细节描述；高明、刘呐鸥特别批评了顾梦鹤妻子的扮演者完全没有情绪表演<ref name=gqq2 />。高明则称电影中表现最好的是蓝蘋，但对其结局不解，称“导演硬要把蓝蘋加入演员中，但是一经加入之后，却拿她没有办法了。这样，当然最简便的办法便是处死她了。”<ref name=gqq>{{Cite book|title=三十年代中国电影评论文选 纪念左翼电影运动六十周年|last=陈播主编；广播电影电视部电影局党史资料征集工作领导小组，中国电影艺术研究中心编|first=|publisher=北京：中国电影出版社|year=1993.12|isbn=7-106-00980-6|location=|pages=543-546}}</ref>穆时英则表示电影不佳，“也许是因为司徒慧敏是新人的缘故吧”<ref name=gqq />。关于摄影方面，刘呐鸥认为电影最不好的是“光调失却基准”<ref name=gqq />。但座谈会最后，姚苏凤总结“批评应该是比较严格的”，对剧中结论的“做一个堂堂的人”表示赞许<ref name=gqq2 />。穆时英则称该部电影团队“在艺术上是失败了，但他们是有希望的，因为他们的态度很严肃，而且又都是很虚心、很肯学习的人。”<ref name=gqq2>{{Cite book|title=穆时英全集 第3卷 散文 理论与评论 译文卷|last=严家炎，李今编|first=|publisher=北京：北京十月文艺出版社；北京：北京出版社|year=2008.01|isbn=978-7-5302-0789-5|location=|pages=240-250}}</ref>

== 影响 ==
《自由神》是蓝蘋（江青）从话剧转型演员后的处女作<ref>{{Cite book|title=影坛钩沉|last=赵士荟著|first=|publisher=郑州：[[大象出版社|大象出版社]]|year=1998.08|isbn=7-5347-2160-1|location=|pages=83-84}}</ref>。这部电影中虽然她的剧情和台词很少，但其演技也得到舆论关注<ref>{{Cite book|title=她还没叫江青的时候|last=王素萍著|first=|publisher=北京：北京十月文艺出版社|year=1993.07|isbn=7-5302-0290-1|location=|pages=183-184}}</ref>。1936年4月，[[夏衍|夏衍]]在《文学》杂志六卷四期推出了自己最新创作的剧本《赛金花》，并交[[上海业余剧人协会|上海业余剧人协会]]在报纸上公开发出招聘《赛金花》女主人公[[赛金花|赛金花]]的扮演者。王莹、蓝蘋均应聘，但蓝蘋在争演的纠纷中败北，并导致两人结怨交恶。1966年[[文化大革命|文化大革命]]爆发后，当时三十年前的当事人王莹、[[金山_(演员)|金山]]、[[郑君里|郑君里]]、夏衍、[[于伶|于伶]]、[[赵丹|赵丹]]等均受到制裁<ref>{{Cite book|title=电影造星渊源考|last=李姝林编著|first=|publisher=北京：[[京华出版社|京华出版社]]|year=2008.01|isbn=7-80724-313-9|location=|pages=187-188}}</ref>。
