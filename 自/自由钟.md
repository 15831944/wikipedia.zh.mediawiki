'''自由鐘'''，又称'''独立鐘'''，是一口位于[[美国|美国]][[宾夕法尼亚州|宾夕法尼亚州]][[费城|费城]]的鐘，它是[[美国独立战争|美国独立战争]]最主要的标志，也象征着[[自由|自由]]和[[公正|公正]]。<ref name="INHP1">{{cite web | first= | last= | url=http://www.cr.nps.gov/nR/twhp/wwwlps/lessons/36liberty/36liberty.htm | title= The Liberty Bell: From Obscurity to Icon | publisher= Independence National Historical Park | date=October 16, 2006 }}</ref>

传统上，人们认为此鐘因在1776年7月8日，大陆会议的代表[[约翰·尼克松|约翰·尼克松]]上校宣读[[独立宣言|独立宣言]]前敲响此鐘来召集费城的市民而出名，但历史学家对此存有疑问，因为当时悬挂该鐘的议会大厦尖塔存在严重的问题。<ref name="USHistory Ringing">[http://www.ushistory.org/LibertyBell/ Liberty Bell<!-- Bot generated title -->] {{webarchive|url=https://web.archive.org/web/20150704163228/http://www.ushistory.org/libertybell/ |date=2015-07-04 }}</ref>1774年的[[大陆会议|大陆会议]]和1775年的[[列克星敦和康科德战役|列克星敦和康科德战役]]都敲响过此鐘。

在1837年之前，该鐘曾被称为“独立鐘”或者“老扬基的鐘”。<ref name="LBMuseum">[http://www.libertybellmuseum.com/resources/faqs.htm FAQs about the Liberty Bell<!-- Bot generated title -->] {{webarchive|url=https://web.archive.org/web/20100505040141/http://www.libertybellmuseum.com/resources/faqs.htm |date=2010-05-05 }}</ref>直至19世纪中叶美国反对[[蓄奴|蓄奴]]运动兴起，废奴主义者把鐘上的铭文大肆宣传，这钟才被世人称呼为“自由鐘”。<ref>{{cite web|url=http://www.bocaiit.com/Article/wenwu/200811/609.html|title=费城自由鐘|deadurl=yes|archiveurl=https://web.archive.org/web/20150923192954/http://www.bocaiit.com/Article/wenwu/200811/609.html|archivedate=2015-09-23}}</ref>

== 铭文 ==
自由鐘上刻有如下文字：
{{cquote|直到各方土地上所有的居民均宣告自由.  XXV X<br />
由宾夕法尼亚州议会为位于费城的州府订购<br />
Pass and Stow（帕思和斯朵）<br />
Philad<sup>a</sup>（费城）<br />
MDCCLIII（1753年）}}
这段文字出自《[[圣经|圣经]]》的《旧约·[[利未记|利未记]] 》25:10： {{cquote|第五十年，你们要当作圣年，在遍地给一切的居民宣告自由。<br />
这年必为你们的禧年，各人要归自己的产业，各归本家。}}

== 历史 ==
[[File:Liberty_Bell_150th_Anniversary_1926_Issue-2c.jpg|thumb]]

=== 订购和重铸 ===
1751年，宾州州议会从英国订购了这口鐘，由[[伦敦|伦敦]]的[[怀特·查珀尔铸钟厂|怀特·查珀尔铸钟厂]]铸造，于1752年9月上旬运至费城，该鐘花费100英镑，高3英尺，重2080磅。顶部周长7英尺6英寸，鐘壁厚1.25英寸；底部周长12英尺，厚为3英寸。令人失望的是，来年3月在试鐘的时候出现了裂纹，当时的宾夕法尼亚州议会的发言人这样写道：
{{quote|在它被挂起来测试声音的时候被击了一下就开裂了，并没有任何暴力行为，听到这个消息，我觉得这是一种羞辱。|<ref name="LBMuseum"/>}}

州府要求怀特·查珀尔铸鐘厂替换一个新鐘，由费城的John Dock Pass和John Stow的重新铸造，这两个人的姓氏（帕恩和斯朵）被刻在钟的铭文上。 他们在合金中添加了铜来铸造该鐘，但是声音并不理想。然后他们两人又重新铸造了一个鐘，第三个钟于1753年6月就被挂在议会大厦的尖塔里。<ref name="Whitechapel">{{cite web |url=http://www.whitechapelbellfoundry.co.uk/liberty.htm |title=THE LIBERTY BELL |id=怀特·查珀尔铸鐘}}</ref>

=== 停用和巡展 ===
1835年在为[[约翰·马歇尔|约翰·马歇尔]][[美國首席大法官|首席大法官]]举行殡葬仪礼时敲鐘而再次开裂。1846年为纪念[[华盛顿|华盛顿]]总统的诞辰鸣鐘庆祝时，遭到严重破裂，发声嘶哑。虽几度企图修复裂缝，都无结果。此后大鐘便停止使用。但前来参观的人很多，于是便送到各地展览。1885年首先送到[[新奥尔良|新奥尔良]]及南方各州首府陈列。1893年运到[[芝加哥|芝加哥]]，1895年运到[[亚特兰大|亚特兰大]]，1902年到[[查尔斯顿|查尔斯顿]]，1903年到[[波士顿|波士顿]]展出。

1915年运到西海岸的[[旧金山|旧金山]]，但每次搬运，都使钟的裂缝增大，独立宫管理委员会决定不再运至外州巡回展览。1976年1月1日起移到上述玻璃房内。

== 复制钟 ==
1915年，[[德克萨斯州|德克萨斯州]][[利柏提縣_(德克薩斯州)|利柏提]]的妇女活动家凯瑟林·卢森伯格复制了一口自由鐘，以此推动妇女获取投票权的运动。这个复制品被称为“妇女自由鐘”或者“正义鐘”。她把鐘的铃舌绑住，使其无法发音。活动家们宣布，要到有关投票结束妇女获得选举权后，才让“妇女自由鐘”发出声音。到1920年，美国宪法第十九条修正案终于赋予妇女投票权。<ref>{{cite web|url=http://news.sina.com.cn/w/pl/2006-10-27/085111347061.shtml|title=达巍：费城自由鐘，被不断诠释的美国历史}}</ref>

在美国之外的[[比利时|比利时]]、[[以色列|以色列]]、[[日本|日本]]都有自由鐘的复制品。<ref name="LBMuseum2">{{cite web|url=http://www.libertybellmuseum.com/exhibits/replicas/replicas.htm|title=自由鐘博物馆|deadurl=yes|archiveurl=https://web.archive.org/web/20100722155746/http://www.libertybellmuseum.com/exhibits/replicas/replicas.htm|archivedate=2010-07-22}}</ref>

德国柏林的自由鐘虽然不是美国自由鐘的复制品，1950年西德战胜欧洲[[共产主义|共产主义]]，美国人做了一个自由鐘做为礼物送给西德。

美国人（主要是从捷克斯洛伐克成立前的[[奥匈帝国|奥匈帝国]]移民到美国的美国人）在1919年的时候将一个自由鐘的复制品送给当时成立的一个新国家[[捷克斯洛伐克|捷克斯洛伐克]]，该鐘在1980年的时候被放置在[[布拉格|布拉格]]的一个塔中。
