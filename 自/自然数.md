[[数学|数学]]中，'''自然数'''指一般指'''非负整数''' <math>( 0, 1, 2, 3, 4, \ldots)</math>。是 [[ISO_80000-2|ISO 80000-2]] 标准中所采用的定义<ref>{{cite book|title=ISO 80000-2:2009|url=http://www.iso.org/iso/catalogue_detail?csnumber=31887|publisher=[[International_Organization_for_Standardization|International Organization for Standardization]]| chapter = Standard number sets and intervals | page = 6 }}</ref>。

用于[[计数|计数]]（如「桌子上有'''三'''个苹果」）和[[全序关系|定序]]（如「国内第'''三'''大城市」）的数字。用于计数时称之为[[基数_(数学)|基数]]，用于定序时称之为[[序数|序数]]。

（在数论中，有时指'''正整数''' <math>( 1, 2, 3, 4, \ldots)</math>）

数学家一般以<math>\mathbb{N}</math>代表以自然数组成的[[集合|集合]]。自然数集是一個[[可數集|可數]]的，無[[上界|上界]]的[[无限集合|無窮集合]]。

== 历史与0的争议 ==

自然数由数数而起。古希臘人最早研究其抽象特性，当中[[毕达哥拉斯主义|毕达哥拉斯主义]]更视之为宇宙之基本。其它古文明也对其研究作出极大贡献，尤其以[[印度|印度]]对0的接受，为人称道。

零早于公元前400年被巴比伦人用作[[数码|数码]]使用。玛雅人于公元200年将零视为数字，但未与其它文明有所交流。现代的观念由印度学者[[婆羅摩笈多|婆罗摩笈多]]于公元628年提出，经阿拉伯人传至欧洲。欧洲人一开始仍对零作为数字感到抗拒，认为零不是一个“自然”数。

19世纪末，[[集合论|集合论]]者给自然数一个较严谨的定义。据此定义，把零（对应于[[空集|空集]]）包括于自然数内更为方便。[[逻辑|逻辑]]论者及[[计算机科学|计算机科学]]家，接受集合论者的定义。而其他一些数学家，主要是数论学家，则依从传统把零拒之于自然数之外。

在全球范围内，目前针对0是否属于自然数的争论依旧存在。

在中国大陆，2000年左右之前的中小学教材一般'''不'''将0列入自然数之内，或称其属于“扩大的自然数列”<ref>王好民，《谈谈中学数学中的“0”》。曲阜师院学报(自然科学版)，1979年03期。</ref>。在2000年左右之后的新版中小学教材中，普遍将0列入自然数。<ref>（沧州市第一中学）李元星，潘峰，《关于0是自然数的探讨》。教育实践与研究，2004年01期。</ref><ref>（江苏省连云港市墟沟实验小学）傅海洋，《“0是自然数”引发的教学问题》。现代中小学教育，2007年08期。</ref>

認為自然數不包含零的其中一個理由是自然數所指為自然界中存在的數，例如一棵大樹、兩條魚、十億個細胞等等，而鮮少有人說零個物品。

国际标准{{link-en|ISO 31-11:1992|ISO 31-11}}《量和单位 第十一部分：物理科学和技术中使用的数学标志与符号》（已被{{link-en|ISO/IEC 80000-2|ISO/IEC 80000}}取代）中，从集合论角度规定：符号 <math>\mathbb{N}</math> 所表示的自然数集是包括正整数和0。

中国于1993年制定的强制性国家标准《物理科学和技术中使用的数学符号》(GB 3102.11-93)参照国际标准ISO 31-11規定：<math>\mathbb{N}</math>表示“非负整数集；自然数集”，<math>\mathbb{N}=\{0,1,2,3,\ldots\}</math>。

== 定义 ==
{{主条目|皮亚诺公理|自然數的集合論定義}}

爲了给出自然数的严格定义，[[朱塞佩·皮亞諾|朱塞佩·皮亚诺]]采用[[序数|序数]]理论提出自然数的5条[[公理|公理]]，被稱爲皮亚诺公理。这五条公理用非形式化的方法叙述如下：

# 1是自然数；
# 每一个确定的自然数<math>n</math>都有一个确定的后继數，记作<math>n^+</math>或<math>n+1</math>。''<math>n+1</math>''也是自然数；
# 1不是任何自然数的后继；
# 不同自然數有不同的后继数
# 如果一些自然数的[[集合|集合]]''<math>S</math>''具有性质：
#*1在<math>S</math>中；
#*設''<math>n</math>''在''<math>S</math>''中，则''<math>n+1</math>''也在''<math>S</math>''中。
那么<math>S=\mathbb{N}</math>。（公理5保证了[[数学归纳法|数学归纳法]]的正确性，从而被称为归纳法原理）

若将[[0|0]]也视作自然数，则公理中的1要换成0。

在[[基数|基数]]理论中，[[集合论|集合论]]的一般做法是首先把0定義為[[空集|空集]]，然後把一非零自然数看作是所有比該數小的自然数组成的集合，即
:<math>0 = \{\}, 1 = \{0\}, 2 = \{0,1\}, 3 = \{0,1,2\}, \ldots</math>

透過[[無窮公理|無窮公理]]，可以得到只包含全體自然數的自然數集'''<math>\mathbb{N}</math>'''。

== 性质 ==

自然数[[加法|加法]]可经<math>a+0=a</math>及<math>a+(b+1)=(a+b)+1</math>递归定义而成。因而得出[[交换律|交换]][[幺半群|幺半群]]<math>(N,+)</math>，是由<math>1</math>生出的自由幺半群，其中[[幺元|幺元]]为<math>0</math>。此幺半群服从[[消去律|消去律]]，可嵌入一[[群|群]]内：最小的是[[整数|整数]]群。 

同理，自然数乘法<math>\times</math>可经<math>a \times 0=0</math>及<math>a \times (b+1)=ab+a</math> 得出。而<math>(N, \times)</math>亦是交换幺半群；<math>\times</math>和<math>+</math>符合[[分配律|分配律]]：
:<math>a \times (b+c)=ab+ac</math>。

我们说<math>a \le b</math>当且仅当有自然数<math>c</math>使得<math>a+c=b</math>。<math>(N, \le)</math>是一个[[良序集|良序集]]，即每个非空子集都有一个最小的自然数。此序也和加法及乘法兼容，即若<math>a</math>，<math>b</math>和<math>c</math>都是自然数且<math>a \le b</math>，则<math>a+c \le b+c</math>及<math>ac \le bc</math>。

给定两个自然数<math>a</math>和<math>b</math>，其中<math>b \ne 0</math>，可找到唯一的两个自然数<math>q</math>及<math>r</math>使得
:<math>a=bq+r,\ 0\le r < q</math>

<math>q</math>称为“[[商數|商数]]”而<math>r</math>称为“[[余数|余数]]”。 若<math>r=0</math>，则稱<math>a</math>可被<math>b</math>[[整除|整除]]，记为<math>b|a</math>。

相关概念有[[关系_(数学)#可除性|可除性]]、[[輾轉相除法|辗转相除]]、[[质数|质数]]及其它[[数论|数论]]概念。

== 推广 ==

自然数有两种推广：[[序数|序数]]用作排列，而[[基数|基数]]用于判定集合的大小。

对于[[有限|有限]]序列或有限集合，序数及基数皆与自然数同。
