'''費沁源'''（{{bd|2001年|3月20日|catIdx=Fei, Qinyuan}}），中國偶像團體[[SNH48|SNH48]]的成員，所屬隊伍是Team HII。她於2015年7月25日的[[SNH48第二屆總選舉|SNH48第二屆總選舉]]中披露，成為SNH48的5期生。2015年12月4日，於《[[Team_B_5th_Stage「劇場的女神」#SNH48_Team_XII「劇場女神」公演|劇場女神]]》公演出正式出道<ref name="sina1">{{Cite web |url = http://ent.sina.com.cn/y/yneidi/2015-12-07/doc-ifxmhqac0105704.shtml |title = SNH48五期生宣布出道 4日上海献公演首秀 |date = 2015-12-07 |accessdate = 2016-01-15 |publisher = 新浪娛樂 }}</ref>。她同時也是组合内小分队[[Color_Girls|Color Girls]]的成員。

== 經歷 ==
2015年7月25日，在[[SNH48第二屆總選舉|SNH48第二屆總選舉]]中披露，成為SNH48的5期生。12月4日，於《[[Team_B_5th_Stage「劇場的女神」#SNH48_Team_XII「劇場女神」公演|劇場女神]]》公演出道<ref name="sina1" /><ref name="wmxa">{{Cite web |url = http://www.wmxa.cn/a/201512/291903_2.html |title = SNH48五期生上海公演正式出道 完整成员名单流出颜值爆表 |date = 2015-12-08 |accessdate = 2016-01-15 |publisher = 西安文明網 |deadurl = yes |archiveurl = https://web.archive.org/web/20160512193236/http://www.wmxa.cn/a/201512/291903_2.html |archivedate =2016-05-12  }}</ref>。12月28日，參與了「[[SNH48第二屆年度金曲大賞BEST_30|SNH48第二屆年度金曲大賞BEST 30]]」<ref>{{cite web|title=SNH48第二届年度金曲大赏 《夜蝶》摘取桂冠|url=http://ent.163.com/photoview/00B60003/583039.html#p=BBQ3UM7C00B60003|website=网易|accessdate=2017-02-26|date=2015-12-26}}</ref>。12月31日，參加東方衛視跨年晚會。

2016年1月10日起，參加SNH48專屬綜藝節目《[[國民美少女|國民美少女]]》，並突破重圍成為首位入隊美少女<ref name="sina2">{{Cite web |url = http://ent.sina.com.cn/zz/2016-01-10/doc-ifxnkkuy7806008.shtml |title = 14岁费沁源夺状元秀 |date = 2016-01-10 |accessdate = 2016-01-15 |publisher = 京華時報 }}</ref><ref name="xkb">{{Cite web |url = http://epaper.xkb.com.cn/view/1021852 |title = 网友投票选出的首位“国民美少女” 是这位14岁的妹妹! |date = 2016-01-08 |accessdate = 2016-01-15 |publisher = 新快報 |deadurl = yes |archiveurl = https://web.archive.org/web/20160202212626/http://epaper.xkb.com.cn/view/1021852 |archivedate =2016-02-02  }}</ref><ref name="youku">{{Cite web |url = http://v.youku.com/v_show/id_XMTQ0MDg4NTA1Ng.html |title = 国民美少女 2016：萌妹跪地拔河 费玉清猛讲段子戏美少女 01 |date = 2016-01-10 |accessdate = 2016-01-15 |publisher = [[優酷網|優酷網]] }}</ref>。1月上旬，費沁源被日媒稱為「四千年一遇的美少女」，在日本颳起旋風，走红日本。这也是继鞠婧祎之后，再度走红日本的SNH48成员<ref>[http://music.yule.sohu.com/20160121/n435314445.shtml SNH48成员费沁源走红日本 被盛赞翻版渡边麻友]</ref>。6月27日，出演網絡劇《[[貼身校花|貼身校花]]》<ref>[http://yule.sohu.com/20160309/n439801388.shtml 《贴身校花》曝光最强“校花团” SNH48首次大合体]</ref>。7月23日，參與湖南衛視節目《夏日甜心》。7月30日，於SNH48第三屆總選舉，獲得第22名，最終獲得20137.9票，入選第三屆總選舉Under Girls<ref>[http://www.snh48.com/html/allnews/zixun/2016/0730/1873.html “比翼齐飞”SNH48第三届偶像年度人气总决选最终结果公布]</ref>。12月20日，入選《[[SNH48原創發行作品列表#新年這一刻|新年這一刻]]》選拔組，也是費沁源首次參與選拔。12月31日，參加湖南衛視跨年晚會。

2017年1月7日，參與了「SNH48第三屆年度金曲大賞BEST 50」<ref>[http://www.snh48.com/html/allnews/zixun/2017/0108/2101.html SNH48第三届金曲大赏完满落幕 50首金曲上演年度巅峰对决]</ref>。7月29日，於SNH48第四屆總選舉，最終獲得20614票，排名第40名，入選第四屆總選舉Next Girls<ref>[http://www.snh48.com/html/allnews/zixun/2017/0730/2456.html SNH48 GROUP第四届总决选落幕 鞠婧祎问鼎年度人气王]</ref><ref>[http://www.snh48.com/html/allnews/zixun/2017/0729/2453.html “我心翱翔”SNH48GROUP第四届总决选最终结果公布 鞠婧祎连霸]</ref>。10月14日，費沁源與口袋48的自制直播節目《[[源來如此|源來如此]]》開播。

2018年1月12日，获得2017年SNH48劇場MVP。2018年2月3日，於「[[SNH48_Group第四屆年度金曲大賞BEST_50|SNH48 Group第四屆年度金曲大賞BEST 50]]」上宣布所屬隊伍改為Team HII<ref>[http://www.snh48.com/html/allnews/zixun/2018/0203/2883.html SNH48 首次大重组各队成员名单公布 分层级打造彰显新格局]</ref>。

== 人物 ==
* 精通射箭和射击，與可愛的外表造成了一種「反差萌」<ref name="snh48official" />。
* 喜歡交朋友、听轻音乐<ref name="snh48official" />。
* 喜歡玩[[摩爾莊園|摩爾莊園]]。
* 喜歡《[[淘氣包馬小跳|淘氣包馬小跳]]》。
* 學過[[烏克麗麗|烏克麗麗]]。
* 被日本网友称酷似[[渡边麻友|渡边麻友]]<ref>[http://ent.163.com/16/0128/14/BEE2H5CI00034VDC.html?f=jsearch 新"四千年"美女亮相 费沁源风头直盖鞠婧祎]</ref>。
* 與[[洪珮雲|洪珮雲]]、姜杉、严佼君、[[張怡_(SNH48)|张怡]]被粉絲稱為五期生「五帝」。
* 關係相近的成員有[[孫珍妮|孫珍妮]]、謝妮、楊冰怡、洪珮雲、姜杉、胡曉慧、陳珂等
