'''莆仙戏'''（[[莆仙語|莆仙語]]：{{lang|cpx|Pó-sing-hi̍}}），又名“'''兴化戏'''”（{{lang|cpx|Hing-hua̍-hi̍}}），是[[福建|福建]]戏曲剧种之一，被誉为[[宋元南戏|宋元南戏]]的“活化石”<ref name=新浪网>{{cite news|title=抢救“莆仙戏”|url=http://news.sina.com.cn/c/2003-07-30/11001442218.html|newspaper=新浪网}}</ref> 。源于[[唐朝|唐]]，成于[[宋朝|宋]]，盛于[[明朝|明]][[清朝|清]]。流行于古称[[兴化|兴化]]的[[莆田|莆田]]、[[仙游|仙游]]二县及[[闽中|闽中]]、[[闽南|闽南]]的[[兴化语|兴化语]]地区。{{Fact|据史料记载和调查考证|time=2012-04-29T09:35:51+00:00}}，它是在古代“百戏”的基础上发展形成的。

== 表演 ==

表演动作受[[杂剧|杂剧]]和民间[[傀儡戏|傀儡戏]]影响。行当脚色承袭南戏规制，有[[生|生]]、[[旦|旦]]、[[贴生|贴生]]、[[则旦|则旦]]、[[靓妆|靓妆]]（[[净|净]]）、[[末|末]]、[[丑|丑]]等七个角色，俗称“七子班”。

== 唱 ==

[[音乐|音乐]][[曲牌|曲牌]]有一千多个。[[唱腔|唱腔]]迄今仍保留不少宋元南戏音乐遗响。莆仙戏的[[声腔|声腔]]主要是“兴化腔”，它综合溶化莆仙民间歌谣俚曲、十音八乐、佛曲法曲、宋元词曲和大曲歌舞而形成，用兴化语演唱。

== 剧目 ==

传统剧目有五千多个，其中保留宋元南戏原貌或故事情节基本类似的剧目有八十多个。莆仙戏比较著名的剧目有《琴挑》、《[[团圆之后|团圆之后]]》、《[[春草闯堂|春草闯堂]]》、《[[状元与乞丐|状元与乞丐]]》等。

莆仙戏的著名剧作家有[[陈仁鉴|陈仁鉴]]、[[柯如宽|柯如宽]]、[[江幼宋|江幼宋]]等。
