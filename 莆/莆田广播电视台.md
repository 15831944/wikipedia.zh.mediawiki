'''莆田广播电视台'''是[[福建省|福建省]][[莆田市|莆田市]]的市级[[电视台|电视台]]，于1983年3月建台，1989年9月25日正式开播。现有新闻综合及经济公共两个频道。

== 历史 ==
*1983年3月，当时的[[莆田县|莆田县]]县委在壶公山电视差转台的基础上，建立莆田电视台，初期隶属于县委宣传部。
*1983年9月，莆田电视台管理权移交至市广播电视事业局。<ref>{{cite web | title=莆田县志-电　视 | url=http://www.fjsq.gov.cn/frmBokkList.aspx?key=2254A60F7B6641F0B3042823FC0161B6 | accessdate=1994-04}}</ref>
*1986年6月，第一次试播《兴化新闻》<ref>{{cite web | title=莆田市志-电　视 | url=http://www.fjsq.gov.cn/frmBokkList.aspx?key=598FF82EB2AF4A7784A8245692179F87 | accessdate=2000-03}}</ref>。后于1987年6月29日，调整为每周播出一次。
*1987年8月7日，[[国家广电总局|国家广电总局]]批准莆田电视台建台。
*1989年9月25日，莆田电视台正式开播。《兴化新闻》调整为每周播出两次。<ref>{{cite web | title=福建广播电视志-四、莆田电视台 | url=http://www.fjsq.gov.cn/frmBokkList.aspx?key=14F77A06AE1A4E36BD629D1564C20AC3 | accessdate=2002-01}}</ref>
*1992年5月9日，莆田有线电视台开播。初期仅以转播内容为主。1997年，开办自制节目<ref>{{cite web | title=莆田之窗-社会事业：新闻 | url=http://www.66163.com/fujian_w/city/putian/gb/shsy/1_6.htm | accessdate=2002-01}}</ref>
*2001年，莆田电视台与莆田有线电视台合并，合并后，分别更名为新闻综合（莆田台-1）、经济生活频道（莆田台-2）
*2002年9月，莆田电视台与莆田人民广播电台合并，成立莆田广播电视台。
*2013年4月22日，互动类节目《你说 我说 大家说》第一期录制，并于随后开播<ref>{{cite web | title=莆田广播电视台民生互动栏目开机录制 | url=http://www.ptxw.com/news/xw/pt/201304/t20130426_18518.htm | accessdate=2013-04-26}}</ref>
*2014年5月5日，莆田广播电视台进行全台改版。新闻综合频道开播《壶兰监督》、《莆田晚新闻》。经济公共频道《来自县区的报道》停播，并拆分为《仙游新闻》、《荔城新闻》、《城厢新闻》、《涵江新闻》、《秀屿新闻》、《北岸新闻》等节目。
*2014年9月18日，方言[[评书|评书]]类节目《秋生讲古》开播。当年10月2日，电视剧《妈祖》[[莆仙话|莆仙话]]版开播<ref>{{cite web | title=《妈祖》电视剧莆仙方言版成功译制 10月2日开播 | url=http://www.ptxw.com/news/xw/pt/201409/t20140930_49828.htm | accessdate=2014-09-30}}</ref>
*2016年2月1日，首次录制《莆田市春节联欢晚会》，随后于2月11日播出<ref>{{cite web | title=我市首届春节联欢晚会昨晚录制 | url=http://www.ptxw.com/news/xw/pt/201602/t20160202_118911.htm | accessdate=2016-02-02}}</ref>
*2017年10月，莆田广播电视台购入高清[[转播车|转播车]]1部，采用[[奔驰|奔驰]]底盘总成。<ref>{{cite web | title=莆田电视台真有钱 | url=http://bbs.0594.com/thread-503981-1-1.html | accessdate=2017-10-17 }}</ref>
