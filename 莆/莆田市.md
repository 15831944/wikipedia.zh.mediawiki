'''莆田市'''（[[興化語|興化語]][[興化平話字|平話字]]：{{sans-serif|{{lang|cpx|Pó-chéng chī}}}}）是[[中国|中国]][[福建省|福建省]]下辖的[[地级市|地级市]]。莆田市史称'''兴安'''、'''兴化'''，又称'''莆阳'''，'''莆仙'''。自古是福建中部的政治、经济、文化中心。历史悠久，文化源远流长，素有“[[海滨邹鲁|海滨邹鲁]]”、“文献名邦”之誉。

莆田市现辖[[仙游县|仙游县]]、[[荔城区|荔城区]]、[[城厢区|城厢区]]、[[涵江区|涵江区]]、[[秀屿区|秀屿区]]一縣四区和[[湄洲岛国家旅游度假区|湄洲岛国家旅游度假区]]、[[湄洲湾北岸经济开发区|湄洲湾北岸经济开发区]]两个管委会。湾口的[[湄洲岛|湄洲岛]]是海内外闻名的和平女神[[湄洲妈祖祖庙|妈祖祖庙]]所在地、妈祖文化的发祥地。

== 历史 ==
莆田歷史上曾是人文薈萃之地。西晉末年，中原戰亂，中原先進的生產技術和文化隨著衣冠南渡，大批南下的漢人進入閩地，莆田亦是閩中一縣，亦也算在八姓入閩之中<ref>衣冠指典章制度、禮樂規儀等文化。《文選˙任昉˙天監三年策秀才》：「何者？百王之弊，齊季斯甚，衣冠禮樂，掃地無餘。」《隋書˙卷四十九˙牛弘傳》：「素歎曰：『衣冠禮樂盡在此矣，非吾所及也。』」</ref>。到了南朝陈光大二年（568年），莆田始立[[縣|縣]]。唐武則天聖歷二年（699年），析莆田西部设立清源县（後改名[[仙游县|仙游县]]），莆仙两县均属当时的清源郡（郡治在今泉州）。宋太平兴国四年（979年），又析莆田、仙游等部分区域设立兴化县，置太平军（旋改兴化军），太平兴国八年（983），兴化军军治由游洋迁到莆田，辖莆田、仙游、兴化三个县。莆田逐步成为闽中的政治、经济、文化中心。宋景炎二年（1277），兴化军升为兴安州。元代改称兴化路。明代改为兴化府，裁撤兴化县，辖地分别归并莆仙两县。清代仍为兴化府。民国时期，废府，隸屬南路道、厦门道、福建省、兴泉省和福建第四行政督察区。中华人民共和国成立后，莆仙两县历属福建第五行政督察区、晋江专区、闽侯专区、莆田地区。

1983年9月，经国务院批准，莆田撤区设市，成为[[省辖市|省辖市]]，辖城厢、涵江两区和莆田、仙游两县。2003年2月1日，经国务院批准，进行部分行政区划调整，将原莆田县的部分乡镇分别划入城厢区、涵江区，并新设立了荔城区、秀屿区。莆田市现辖有城厢、荔城、涵江、秀屿四区和仙游县。

== 地理 ==
莆田市地处福建沿海中部，东面[[台湾海峡|台湾海峡]]與[[台湾|台湾]]隔海相望，北依省會福州市，南接泉州並与[[廈门|廈门]]相近，位于东经118°27`-119°39`，北纬24°59`-25°46`，陆域面积约4200平方公里。地势西北高东南低，背山面海，西北部山峦叠嶂，中部丘陵起伏，东南部[[兴化平原|平原]]广阔与逶迤的[[木兰溪|木兰溪]]、延寿溪、秋芦溪构成了江南水乡。

莆田境内海岸线长达343.6公里，约占福建省海岸线总长的10.3%，拥有[[湄洲湾|湄洲湾]]、[[兴化湾|兴化湾]]、[[平海湾|平海湾]]，其中湄洲湾的[[秀屿|秀屿]]、东吴水深港阔、不冻不淤，被誉为“世界不多，中国少有”的天然良港。

=== 民族 ===
人口302万，有[[汉族|汉]]、[[回族|回]]、[[满|满]]、[[畲|畲]]、[[壮|壮]]、[[苗族|苗]]、[[瑶|瑶]]、[[土家|土家]]等33个[[民族|民族]]。旅居在外的侨胞70万人左右，港澳台同胞近20万人，归侨、侨眷30万人，是全国重点侨乡之一。

== 经济 ==
莆田市为沿海经济开放区，享受国家、省赋予的一系列优惠政策。近年来，这里已成为外商和港澳台商投资的热点，现己批准设立外商投资企业1456家，合同利用外资38．32亿美元，实际到资23.2亿美元，已建成投产“三资”企业668家。
莆田市自改革开放以来，国民经济持续、快速、健康发展，人民生活水平日益提高。[[南日镇|南日镇]]的南日鲍、莆田[[枇杷|枇杷]]、莆田[[桂圆|桂圆、]]莆田鞋业为[[中国地理标志产品|中国地理标志产品]]<ref>[http://kjs.aqsiq.gov.cn/dlbzcpbhwz/bhml/200906/t20090622_118997.htm 《地理标志产品保护规定》实施以来批准的地理标志名录（2005年7月至2009年6月）] {{webarchive|url=https://web.archive.org/web/20110311120900/http://kjs.aqsiq.gov.cn/dlbzcpbhwz/bhml/200906/t20090622_118997.htm |date=2011-03-11 }}</ref>。

=== 第一产业 ===
[[File:1964-06_1964年_福建莆田县麦田.jpg|thumb]]
==== 渔业 ====
莆田市渔业兴旺，水产富饶。主要有鳗鱼、对虾、大黄鱼、马鲛鱼、[[梭魚|梭子鱼]]、鲟、蛏、蛤、海蛎、紫菜等，可供利用的浅海滩涂27.5万亩，淡水面积6万亩，水产养殖具有极大潜力。沿海区域盛产原盐，产品远销东南亚和美国。
==== 种植业 ====
莆田市物产丰富，品类繁多，是福建著名的粮食、甘蔗和水果产区之一。全市土地面积39.73万公顷，果园2.5万公顷，林地24.4万公顷。盛产稻谷、甘蔗、花生、黄麻、茶叶，龙眼、荔枝、枇杷、无籽蜜柚四大名果驰名中外。
==== 采掘业 ====
莆田市矿产丰富，已探明矿13藏种，主要矿产蛇蚊石储量占全省70％，高岭土、花岗石品位高，储量大，具有较高的开采价值。还有铝、锌、铁、铜、锰、钨、铁等可供开发。

== 语言 ==
境内通行[[莆仙话|莆仙话]]。

莆仙话與閩南語[[泉州話|泉州話]]相同的佔約62%，與閩東語[[福州話|福州話]]相同的約佔39%，並據此推斷莆仙語和閩南語同源。

莆田话是最古老的汉语方言之一，保存着很多唐朝之前的中原古汉语发音。莆田话的特点是八音（包括白阳入）分明。清代学者钱大昕提出的“[[古无轻唇|古无轻唇]]”、“[[古无舌上|古无舌上]]”等古音方面的著名论断，可以在兴化方言中找到大量例证。在今普通话中，只有阴平、阳平、上声、去声四个声调，没有入声。这是因为在古汉语的演变中，入声字已于元代消失了。但是，莆仙话中却仍然保留着大量的中古汉语的入声，构成完整的八音。有些人写旧体诗词时，分不清哪些字是仄声，用莆仙话一读便知：音节短促不能拉长尾声的便是入声，入声属于仄声。

莆田市曾先后荣获“福建省首批历史文化名城”、“全国双拥模范城”、“中国木雕之城”、“中国古典工艺家具之都”、“中国珠宝玉石首饰特色产业基地”、“田径之乡”、“篮球城市”、“戏曲之乡”、“绘画之乡”、“摄影之乡”等荣誉称号，而[[莆仙戏|莆仙戏]]、[[妈祖|妈祖祭典]]、莆田九鲤灯舞、黄石惠洋十音及仙游县的枫亭元宵游灯习俗均被列入[[国家级非物质文化遗产名录|国家级非物质文化遗产名录]]。

== 教育 ==
[[File:Putian_University.jpg|thumb]]

南朝陈永定二年（558年）前后，郑露兄弟三人在莆田南山（今凤凰山）设立湖山书堂，世称“开莆来学”，这是莆田最早的学校教育。

武德三年（620年），莆田人金鲤首中[[进士|进士]]。唐朝，林披有9个儿子均当过[[刺史|刺史]]，时称“一门九刺史”、“九牧林家”。大历年间（766～779年），林披之子林蕴创办澄渚书堂并开启莆田私学的先河。而林披次子林藻为莆田立县后第一位中[[进士|进士]]的人。

清光绪二十四年（1898年），培元西学堂作为一家全日制新式学堂成立。清光绪十五年（1889年），天主教会创设涵江育婴堂和平海育婴堂，开始莆田地区的幼儿教育。

莆田市现有大中专学校19所，普通[[中学|中学]]和职业[[中学|中学]]150所，小学近千所。莆田市现有高等院校2所，其中公办本科院校和公办专科院校各1所：[[莆田学院|莆田学院]]和[[湄洲湾职业技术学院|湄洲湾职业技术学院]]。

* 鄭為國:霞皋村人，鄭仁泰之子，是台北市國中退休校長，歷任台北市重慶國中、建成國中校長，(2011年建成國中退休)，於2010年率領中華台北青少棒隊，在美國密西根州，榮獲世界冠軍。
