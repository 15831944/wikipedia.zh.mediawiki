'''淀號劫機事件'''（[[日文|日文]]：'''{{lang|ja|よど号ハイジャック事件}}'''），是指1970年3月31日，[[日本共產主義者同盟|日本共產主義者同盟]][[日本赤軍|赤軍派]]策劃的日本航空351號班機劫機事件，劫匪於劫機後4天成功流亡[[朝鲜|朝鲜]]後投降。

== 事件經過 ==
1970年3月31日，九名[[日本赤軍|日本赤軍]]成員登上由[[東京|東京]]飛往[[福岡|福岡]]的JL351號班機，途中將飛機劫持，聲稱要流亡至[[朝鲜民主主义人民共和国|朝鲜民主主义人民共和国]]。被劫的波音727先後在福岡機場和[[韓國|韓國]][[金浦國際機場|金浦國際機場]]停靠，其後於4月3日抵達朝鮮{{link-en|美林機場|Mirim Airport}}。機上乘客在福岡和漢城被分別釋放，但機組人員只得隨飛機一同從朝鮮返回日本。

== 涉事航機 ==
[[File:Boeing_727-89,_Hapag-Lloyd_AN1138807.jpg|thumb]]]]
遭劫持的JA8315號飛機於1966年製造，最初交付給日本國內航空（後來的[[日本佳速航空|日本佳速航空]]）。這架飛機的名稱來自[[近畿地方|近畿地方]]河流[[淀川|淀川]]，而日航其餘的波音727亦是以河流命名。1976年，JA8315轉售赫伯羅特航空（即今[[TUIfly|TUIfly]]），編號改為D-AHLS。

== 事后 ==
1972年11月6日，与此次遭劫航班同号的JL351号班机（同样是波音727）{{link-ja|日本航空351號班機劫機事件 (1972年)|日本航空351便ハイジャック事件|再次遭到劫持}}，劫机犯企图将飞机劫至[[古巴|古巴]]。<ref>{{cite web|url=http://aviation-safety.net/database/record.php?id=19721106-0|title=ASN Aircraft accident Boeing 727 registration unknown Tokyo|accessdate=2014-05-11}}</ref>
