'''尉健行'''（{{bd|1931年|1月2日|2015年|8月7日|Wei, Jianxing}}），[[浙江|浙江]][[新昌|新昌]]人，前[[中国共产党|中国共产党]]和[[中华人民共和国|中华人民共和国]][[党和国家领导人|领导人]]之一。曾任[[中共中央政治局常委|中共中央政治局常委]]、[[中央纪律检查委员会书记|中央纪律检查委员会书记]]、[[中央书记处|中央书记处]]书记、[[中华全国总工会|中华全国总工会]]主席。<ref>{{cite web |url = http://news.sohu.com/20120625/n346411398.shtml |title = 朱镕基尉健行吴仪张德江:“空降”的政治局委员 |accessdate=2012 |publisher=搜狐网 |language = zh-hans}}</ref><ref>{{cite web |url = http://news.sohu.com/72/55/news148135572.shtml |title = 尉健行警告领导干部：要教育管好自己的配偶子女 |accessdate=2002 |publisher=搜狐网 |language = zh-hans}}</ref>

== 生平 ==
=== 早年经历 ===
尉健行出生在浙江省新昌县的一个普通教师家庭。早年，曾先后读于浙江新昌南明小学、新昌中学、天台育青中学、杭州安定中学；1947年至1949年，在上海光华大学附中学习，并加入中共的秘密外围组织，多次参与组织学生运动，于1949年3月，正式加入中国共产党。

1949年9月，尉健行进入[[大连工学院|大连工学院]]机械系机械制造专业学习；1952年毕业后，在[[东北人民政府#工作机构|东北人民政府工业部]]有色金属管理局[[抚顺|抚顺]][[俄语|俄语]]训练班进修；于1953年至1955年间，赴[[苏联|苏联]]乌拉尔卡明斯克铝加工厂学习企业管理。留学期间，尉健行完成工厂学习、实习任务的同时，还自学了斯维特洛夫工学院经济工程系的主要课程。

1955年回国后，尉健行被分配到任国营东北轻合金加工厂工作。历任厂生产计划科副科长、科长，车间主任，厂长办公室主任、厂生产总指挥、厂党委常委等职务；参与组织编写了46万字的工厂管理方法等材料。[[文化大革命|文化大革命]]期间，尉健行受冲击，下放车间劳动。1970年恢复工作后，历任国营东北轻合金加工厂生产部部长，厂革委会副主任、党委常委、党委副书记等职务；1978年6月起，担任东北轻合金加工厂厂长、党委书记；前后在这家[[国营企业|国营企业]]工作二十五年。

=== 步入政坛 ===
1980年，49岁的尉健行被选拔到首期[[中共中央党校一年制中青年干部培训班|中共中央党校一年制中青年干部培训班]]学习。学习结束后，出任[[中共|中共]][[哈尔滨|哈尔滨]][[市委书记|市委副书记]]、兼市长；后在[[中共十二大|中共十二大]]上，当选候补中央委员。1983年10月，尉健行调升[[中华全国总工会|中华全国总工会]]副主席、书记处书记、党组副书记。在中央工作期间，他受到时任[[中共中央书记处|中共中央书记处]]书记、[[中共中央组织部|中央组织部]]部长的[[乔石|乔石]]欣赏；不久，提拔尉健行作自己的副手，任中组部副部长。1985年，更进一步，接任乔石担任[[中共中央组织部|中组部]]部长，并在当年9月召开的[[1985年中国共产党全国代表会议|中共全国代表会议]]上，被增选为中央委员会委员。

在任中组部部长期间，尉健行亲自考察了大量青年干部，落实中央关于[[平反冤假错案|平反冤假错案]]的要求，坚决清理“文革”中的“[[清理三种人|三种人]]”，受到时任[[中共中央总书记|中共中央总书记]][[胡耀邦|胡耀邦]]的赏识。1987年，胡耀邦下台后，尉健行受到波及，被调任到新恢复设立的[[中华人民共和国监察部|中华人民共和国监察部]]担任部长、党组书记，并连任第十三届中共中央委员。在任监察部部长期间，尉健行主持了数项监察系统的法律法规，并在全国各地县级以上重新架构了行政监察机关，完善了监察系统<ref>{{cite book |author=高屹、缪德修 |title=《邓小平时代的中国》 |year=1996年 |publisher=光明日报出版社 |ISBN = 9787800916694 |page = 1287页}}</ref>。

=== 执掌中纪委 ===
在1992年10月召开的[[中共十四届一中全会|中共十四届一中全会]]上，尉健行晋升中央政治局委员、并当选[[中央书记处书记|中央书记处书记]]，还接替乔石出任[[中央纪律检查委员会|中央纪律检查委员会]]书记。1993年10月，尉健行当选为[[中华全国总工会|中华全国总工会]]第十二届执行委员会主席。在[[陈希同|陈希同]]因涉嫌贪腐下台后，尉健行于1995年4月至1997年8月还兼任[[中共北京市委|中共北京市委]]书记。1997年[[中共十五届一中全会|中共十五届一中全会]]上，当选[[中共中央政治局常委|中共中央政治局常委]]、[[中央书记处|中央书记处]]书记，并连任[[中央纪律检查委员会|中央纪律检查委员会]]书记。翌年连任[[中华全国总工会|中华全国总工会]]主席<ref>{{cite book |author = 中华名人协会、中国文学艺术界联合会、中国中外名人文化硏究会 |title=《中国人物年鉴》|year=2001年 |publisher=华艺出版社 |pages=114-115页}}</ref>。

尉健行在任中纪委书记后，具体落实了“中央纪委、监察部合署办公”的方略，参与领导了全国范围的[[严打|严打]]活动。据资料显示，由1993年至1997年，各级纪律、检察机关共立案六十八万馀件，结案六十三万馀件，开除党籍处分有六十三万多人，其中包括前[[中共北京市委|中共北京市委]]书记陈希同案<ref>{{cite book |author=北京市地方志编纂委员会 |title=《北京年鉴》|year=1997年 |publisher=中国城市出版社 |page = 69页}}</ref>。从1993年到2002年的十年里，共有41名省部级以上官员受到查处；除陈希同外，还包括了，前[[全国人大常委会副委员长|全国人大常委会副委员长]][[成克杰|成克杰]]，辽宁省副省长[[慕绥新|慕绥新]]，公安部副部长[[李纪周|李纪周]]等人。据《瞭望东方周刊》引述原中央纪委副部级官员吴振钧的话，称成克杰案是在尉健行的直接领导下突破的，半年内查清了其贿4100多万元的问题，并使成克杰成为[[中华人民共和国|中华人民共和国]]历史上首个因腐败被判死刑的国家领导人<ref>{{cite news |title = 尉健行同志逝世 曾领导查办成克杰等案|url = http://politics.people.com.cn/n/2015/0807/c1001-27428758.html |publisher = 人民网|date=2015-08-07}}</ref>。

2002年[[中共十六大|中共十六大]]后，尉健行逐渐卸任所有职务，退出政坛。

== 逝世 ==
2015年8月7日8时，尉健行因病医治无效在北京逝世，享年84歲（[[新華社|新華社]]新聞稿稱享年85歲）。中共官方對其的評價是「中国共产党的优秀党员，久经考验的忠诚的共产主义战士，无产阶级革命家、政治家，党和国家的卓越领导人」<ref>{{cite news |title = 尉健行同志逝世 |url = http://news.xinhuanet.com/politics/2015-08/07/c_1116185557.htm |publisher = 新華網 |date=2015-08-07}}</ref>；并特别在其官方生平中提到，在[[六四事件|六四事件]]中，尉健行“经受住了严峻考验”<ref>{{cite news |title = 尉健行同志生平|url = http://news.xinhuanet.com/politics/2015-08/16/c_1116268610.htm |publisher = 新華網 |date=2015-08-16}}</ref>。
