'''事件視界望遠鏡'''（{{lang-en|'''E'''vent '''H'''orizon '''T'''elescope}}, {{lang|en|'''EHT'''}}）是一個以觀測[[星系|星系]]中央[[超大質量黑洞|超大質量黑洞]]為主要目標的計畫。該計劃以[[甚長基線干涉|甚長基線干涉]]技術（{{lang|en|VLBI}}）結合世界各地的[[電波望遠鏡|電波望遠鏡]]，使許多相隔數十萬[[公里|公里]]的獨立[[天線|天線]]能互相協調、同時觀測同一目標並記錄下數據，形成一[[物鏡口徑|口徑]]等效於[[地球|地球]]直徑的虛擬望遠鏡，將望遠鏡的[[角解析力|角解析力]]提升至足以觀測[[事件視界|事件視界]]尺度結構的程度。{{lang|en|EHT}}期望藉此檢驗[[愛因斯坦|愛因斯坦]][[廣義相對論|廣義相對論]]在[[黑洞|黑洞]]附近的強[[重力場|重力場]]下是否會產生偏差、研究黑洞的[[吸積盤|吸積盤]]及[[超相對論性噴流|噴流]]、探討事件視界存在與否，並發展基本[[黑洞物理學|黑洞物理學]]。

{{lang|en|EHT}}的觀測目標主要為位於[[南半天球|南半天球]]、[[銀河系|銀河系]]中央的超大質量黑洞[[人馬座A*|人馬座A*]]以及位於[[北天球|北天球]]的[[橢圓星系|橢圓星系]][[M87|M87]]星系中央的超大質量黑洞。其中人馬座A*在地球天空中佔的盤面較大，而Ｍ87的黑洞則以擁有一道長達5,000[[光年|光年]]的[[超相對論性噴流|噴流]]為著名特色。為了看透銀河盤面及[[吸積盤|圍繞在黑洞周圍的物質]]，{{lang|en|EHT}}將觀測波長設定於1.33[[毫米|毫米]]，並預計於未來提升至能更精細觀測的0.87毫米。由於連線觀測產生的數據量將大到無法使用[[網際網路|網際網路]]傳輸，各觀測台會於觀測後將儲滿數據的[[硬碟|硬碟]]郵寄至[[美國|美國]][[馬薩諸塞州|馬薩諸塞州]]的[[海斯塔克天文台|海斯塔克天文台]]，交由[[超級電腦|超級電腦]]運算，並合成單一影像。根據電腦模擬，環繞黑洞的物質發出的光將被黑洞自身質量產生的[[重力透鏡效應|重力透鏡效應]]彎曲，在黑洞周圍形成一[[愛因斯坦環|光環]]，而光環中央襯托出的圓形剪影便是黑洞的輪廓，也就是事件視界。

2012年，[[天文學家|天文學家]]於美國[[亞利桑那州|亞利桑那州]]首次正式舉辦{{lang|en|EHT}}會議，確立計畫的科學目標、技術計畫和組織架構等。觀測則始於更早的2006年，當時已有三座望遠鏡使用{{lang|en|VLBI}}技術進行連線觀測。多年下來，{{lang|en|EHT}}逐漸從一個鬆散、資金不足的團隊，成長為30多所來自12個國家的大學、[[天文觀測站|天文觀測站]]等研究單位與政府機構參與的國際合作組織。2017年EHT合作協議正式簽署。{{lang|en|EHT}}，2017年4月首次進行為期十天的全球連線觀測，觀測目標為人馬座A*及"M87*"（M87星系中心黑洞的簡稱）。此次觀測也第一次納入位於[[智利|智利]]的[[阿塔卡瑪大型毫米及次毫米波陣列|阿塔卡瑪大型毫米及次毫米波陣列]]（{{lang|en|ALMA}}）、[[南極點|南極點]]的[[南極望遠鏡|南極望遠鏡]]等成員。其中{{lang|en|ALMA}}為一關鍵成員，它的加入將{{lang|en|EHT}}的靈敏度提高了十倍。天文學家希望於此次觀測中攝得第一張黑洞剪影的影像。

2019年4月10日，事件视界望远镜合作组织在全球六地（布鲁塞尔、圣地亚哥、上海、台北、东京和华盛顿）以英语、西班牙语、汉语和日语四种语言，通过协调召开全球新闻发布会，<ref>{{Cite web|title=本世紀重大發現！全球首張黑洞照曝光 見證人類重要一刻 {{!}} 生活 {{!}} 三立新聞網 SETN.COM|url=https://www.setn.com/News.aspx?NewsID=525078|work=www.setn.com|date=2019-04-10|accessdate=2019-04-13|language=zh-Hant-TW|last=三立新聞網}}</ref>发布了于2017年4月11日拍摄的位于[[室女A星系|室女A星系]]的黑洞的照片<ref name="nyt">{{cite news |author1=Dennis Overbye |title=Black Hole Picture Revealed for the First Time |url=https://www.nytimes.com/2019/04/10/science/black-hole-picture.html?action=click&module=Top%20Stories&pgtype=Homepage |accessdate=2019-04-10 |work=纽约时报 |date=2019-04-10}}</ref>。

== 緣起 ==
[[廣義相對論|廣義相對論]]預測了[[黑洞|黑洞]]的存在，天文學家也已觀測到[[宇宙|宇宙]]中存在質量極大的緻密天體，然而目前尚無法確定其即為黑洞。另外，雖然黑洞已是個被科學界與人們廣為接受的概念，其性質及引發諸多現象的機制也仍待釐清。事件視界望遠鏡即是在此一背景下開始的計畫。有別於其他研究黑洞的方法，如透過[[重力波_(相對論)|重力波]]觀測、分析其周圍繞行[[恆星|恆星]]的行為等，{{lang|en|EHT}}的目標為直接取得[[事件視界|事件視界]]尺度的影像進行分析<ref name = "aft obs-170503"/>。

=== 檢驗廣義相對論 ===
{{main|廣義相對論}}
廣義相對論發表至今成功預測過許多[[牛頓力學|牛頓力學]]範疇外的現象，也已在[[地球|地球]]上及[[太陽系|太陽系]]內等弱[[重力場|重力場]]下通過嚴謹的檢驗。然欲判斷廣義相對論是否為正確的[[重力理論史|重力理論]]，仍待強重力場下的檢驗。即使目前已藉一些繞行軌道非常靠近黑洞的恆星精確計算出黑洞的質量，那裡仍屬於弱重力場。高頻率{{lang|en|VLBI}}技術能將觀測推展到重力場極強的超大質量黑洞邊緣、事件視界尺度的結構，因此可直接測試廣義相對論涉及到強重力場方面的論述<ref name = "黑洞剪影"/>{{rp|35}}<ref name = "Sheperd 090621"/>。

=== 事件視界 ===
{{main|事件視界}}
[[File:Black_Holes_-_Monsters_in_Space.jpg|thumb]]對影像的影響。]]

事件視界的存在與否是{{lang|en|EHT}}團隊欲探討的議題之一<ref name = "Sheperd 090621"/>。理論上，事件視界為黑洞周圍一個球形的時空界線，也是[[量子力學|量子力學]]與[[廣義相對論|廣義相對論]]矛盾的展現：根據量子力學，[[时间演化算子|时间演化算子]][[逆映射|可逆]]，每个过程都存在着逆过程，通过系统当前时刻的状态可以唯一地确定其在先前时刻的状态。然而，廣義相對論以空間曲率詮釋重力并預言了黑洞的存在，根据广义相对论，落入黑洞的粒子携带的信息将永远丢失。这表明不同的初态可以演化为相同的末态，演化算子的逆不存在。此即[[黑洞資訊悖論|黑洞資訊悖論]]<ref name = "黑洞剪影"/>{{rp|35}}。以人馬座A*來說，如果它沒有事件視界，則周圍環繞它的物質將會加熱天體表面，使其[[輻射|輻射]]出的[[黑體光譜|黑體光譜]]在[[近紅外光|近紅外光]]出現峰值。然而實際觀測到並無此現象。直接觀測輻射區域的結構將能解決這個問題<ref name = "Sheperd 090621"/>。

=== 黑洞吸積盤 ===

黑洞強大的重力會將附近的物質拉向自己，那些物質將繞著黑洞打轉、最終落入其中。這個過程被稱為[[吸積|吸積]]，物質在繞行時形成的盤狀結構則為[[吸積盤|吸積盤]]。天文學家尚不清楚吸積的詳細過程，如氣體如何從外側向內遷入、掉進黑洞，以及[[帶電粒子|帶電粒子]]在[[吸積流|吸積流]]中產生的[[磁場|磁場]]如何影響氣流的行為。吸積盤上也不時會出現如[[太陽閃焰|太陽閃焰]]般的亮點，使觀測到的亮度產生變化。追蹤亮點的變化可以描繪出黑洞周圍時空的情況<ref name = "黑洞剪影"/>{{rp|35-41}}。

=== 黑洞噴流 ===
{{See also|超相對論性噴流}}
黑洞噴流為[[超大質量黑洞|超大質量黑洞]]產生擁有超高[[準直性|準直性]]的[[超相對論性噴流|超相對論性噴流]]。噴流的構成物質以接近平行的束狀氣流型態強力射出，並以近乎[[光速|光速]]的速度跨越了數個[[星系|星系]]的距離。其成因推測和黑洞強力的[[磁場|磁場]]有關，然欲判斷切確原因與其構成物質仍待更多觀測數據出爐<ref name = "黑洞剪影"/>{{rp|36}}<ref name="Asiaa M87 2012"/>。

== 實驗設計 ==
=== 觀測目標 ===
[[File:X-RayFlare-BlackHole-MilkyWay-20140105.jpg|thumb]]波段觀測的[[人馬座A*|人馬座A*]]]]
[[File:M87_jet.jpg|thumb]]中央黑洞影像]]

*;人馬座A*
:離[[地球|地球]]最近的大質量黑洞為距離地球約26,000[[光年|光年]]、位於銀河系中央的[[超大質量黑洞|超大質量黑洞]][[人馬座A*|人馬座A*]]，其具有約四百萬[[太陽質量|太陽質量]]、30倍太陽體積，為已知黑洞中佔天空[[視面積|視面積]]最大的一個。然而，即使其觀測大小已因自身[[重力|重力]]產生的[[重力透鏡效應|重力透鏡效應]]而放大了兩倍以上，它的事件視界看起來仍僅有55[[微角秒|微角秒]]，相當於從地球看一顆放在[[月球|月球]]上的[[小籠包|小籠包]]<ref name ="EHT-Science"/><ref name = "SPT++"/><ref name = "黑洞大考驗"/>{{rp|82}}<ref name = "黑洞剪影"/>{{rp|37}}。位於南半天球<ref name="GLT170125"/><ref name="GLT170125"/>。

*;M87
:[[室女A星系|室女A星系]]（{{lang|en|M87}}）中心的黑洞為另一個理想的觀測目標。{{lang|en|M87}}的黑洞距離地球5,500萬光年，估計擁有64億倍[[太陽質量|太陽質量]]，因此在天空中看起來只比人馬座A*小一些。它擁有一道橫跨5,000光年的噴流，若能辨識出噴發的起點，可讓理論學家更了解超相對論性噴發物的性質。觀測{{lang|en|M87}}相對人馬座A*有許多優點，如：位在[[天球|北半球的天空]]，使它有更多現成的天文台可進行觀測；質量為人馬座A*的2,000倍，因此動態變化時間較長，較易取得進行中事件的影像；並且，觀測{{lang|en|M87}}的黑洞較不會受其星系盤面的物質影響<ref name = "黑洞剪影"/>{{rp|41}}。

*;活躍星系核
:[[活躍星系核|活躍星系核]]（{{lang|en|AGN}}）為具有高[[紅移|紅移]]、在部分至全部波段輻射出強烈[[電磁波|電磁波]]的遙遠[[天體|天體]]。{{lang|en|EHT}}的高角解析度適合研究AGN細部結構，且觀測波長可以[[光深度|看得更深入]]噴流內部、研究其準直性，高[[頻寬|頻寬]]則適合測量噴射源的高旋轉量。這方面的觀測能力目前只有[[俄羅斯|俄羅斯]]的[[太空望遠鏡|太空望遠鏡]][[Spektr-R|Spektr-R]]可能超過{{lang|en|EHT}}，且是在前者以最高頻率（22[[吉赫|吉赫]]）、單一方向觀測的情況下才會發生<ref name = "obs AGN"/>。

=== 觀測波長===
{{see also|角解析力|光深度}}
因地球與人馬座A*皆位於銀河盤面上，為避免橫亙之間的[[星際氣體|星際氣體]]及黑洞周遭墜落物質影響觀測，必須使用不同於傳統[[VLBI|VLBI]]{{noteTag|1=傳統的{{lang|en|VLBI}}頻率為86吉赫、波長3.5毫米<ref name="higher frequencies 150724"/>}}的1[[毫米|毫米]]以下波長。然地球大氣中的水氣會吸收短波長的電磁波，解決方法便是將觀測站擺在乾燥高地，如位於[[智利|智利]][[阿塔卡瑪沙漠|阿塔卡瑪沙漠]]的[[阿塔卡瑪大型毫米及次毫米波陣列|阿塔卡瑪大型毫米及次毫米波陣列]]（{{lang|en|ALMA}}）與位於[[南極洲|南極洲]]的[[南極望遠鏡|南極望遠鏡]]（{{lang|en|SPT}}）等。但天氣糟時仍會連大氣層都看不出去<ref name = "aft obs-170503"/>。最後觀測波長設定在1.33與0.87[[毫米|毫米]]。在該波長下，銀河幾乎是透明的<ref name = "黑洞大考驗"/>{{rp|82}}<ref name = "黑洞剪影"/>{{rp|37}}。而在該波長下欲達到足以觀測事件視界的[[角解析力|角解析力]]，所對應的望遠鏡[[物鏡口徑|口徑]]為幾千公里，因此必須倚靠VLBI技術達成目標<ref name = "黑洞剪影"/>{{rp|38}}。

=== VLBI與數據處理 ===
{{see also|甚長基線干涉儀|球鞋網路|網格計算}}
天文學家利用[[甚長基線干涉儀|甚長基線干涉儀]]（{{lang|en|VLBI}}）技術整合世界各地的[[電波望遠鏡|電波望遠鏡]]，使它們能同時觀測同一目標，產生能整合為單一影像的數據。其角解析度由<math>\textstyle \frac{\lambda}{B}</math>決定，其中<math>\lambda</math>為觀測波長、<math>B</math>為天線之間的投影距離。兩者比值越小，角解析力越高<ref name="higher frequencies 150724"/><ref name = "radio telescopes-170523"/>。{{lang|en|VLBI}}技術在[[電波天文學|電波天文學]]領域很普遍，然而一般沒有事件視界望遠鏡規模大，不但望遠鏡數量多、分布也廣。

由於每個天文台得到的資料量都非常龐大，無法使用網路傳輸，而需使用[[硬碟|硬碟]]儲存，於觀測完畢之後郵寄出去<ref name = "aft obs-170517"/>。有些天文台一開始使用常規硬碟儲存數據，然而觀測台多位於高地，常規硬碟常因氣壓過低而無法運作。如位於[[墨西哥|墨西哥]]{{tsl|en|Sierra Negra|內格拉火山 (墨西哥)|內格拉火山}}頂（[[海拔|海拔]]4,580[[公尺|公尺]]）的[[大型毫米波望遠鏡|大型毫米波望遠鏡]]，原本裝備的32個常規硬碟中就有28個無法運作。望遠鏡改裝備氦氣封裝硬碟（{{lang|en|Helium-Filled Hard Drives}}）後，不但因其密封環境不會受[[氣壓|氣壓]]影響，能儲存的資料量也從4[[太字节|TB]]提升至8TB。硬碟的效能對於{{lang|en|EHT}}能否順利運作至關重要，因為當望遠鏡觀測時，其每秒產生的數據量是以{{tsl|en|Gbps|每秒十億位元|百億位元}}為單位計算的。一次普通的五天觀測期間，每座望遠鏡會搜集約500TB的數據，整個陣列產生的數據約7[[PB|PB]]，將裝滿1000至2000個硬碟。收集到硬碟上的數據必須從各個望遠鏡透過飛機運送到位於美國[[馬薩諸塞州|馬薩諸塞州]]的[[麻省理工學院|MIT]][[海斯塔克天文台|海斯塔克天文台]]以及位於[[德國|德國]][[波恩|波恩]]的[[馬克斯普朗克電波天文研究所|馬克斯普朗克電波天文研究所]]<ref name="硬碟、計畫06 2015"/><ref name = "aft obs-170503"/><ref name = "Science 170302"/>。

在那裡，資料將會用約800個由{{nowrap|40 Gbit/s}}網路連結的[[CPU|CPU]]組成的相關器（{{lang|en|Correlator}}）——具有專門用途的超級電腦——交叉比對並分析，將各天文台以時間標記的電波信號整合起來<ref name = "aft obs-170517"/><ref name = "radio telescopes-170523"/>。這800個CPU結合其專用的特殊成像軟體後，將數據轉化為單一影像，因此也被稱為「矽透鏡」（{{lang|en|Silicon Lens}}）——用數據而非光去生成影像的「[[透鏡|透鏡]]」。這些數據分開看將只會是來自黑洞的雜訊，只有結合各地的數據才能使它們產生科學用途。其產生的影像解析度將相當於使用地球口徑的望遠鏡所拍攝的<ref name="硬碟、計畫06 2015"/><ref name = "aft obs-170503"/><ref name = "Science 170302"/>。

成像技術對使用VLBI技術的陣列而言非常重要。通常{{lang|en|VLBI}}陣列收集到能用來進行[[傅立葉變換|傅立葉變換]]{{noteTag|1=一種將信號在時域（或空域）和頻域之間轉換的線性積分變換}}的數據相對少，因此僅能使用有限數據和已知限制重建圖像，並從多個符合數據的圖像中篩選出失真程度最低的。{{tsl|en|Astronomical interferometer|天文干涉|電波干涉儀}}數據的標準成像演算法為{{tsl|en|CLEAN (algorithm)|CLEAN}}，該演算法已成功在低頻率的情況下產生足夠質量的[[干涉_(物理学)|干涉]]圖像。但{{lang|en|EHT}}的觀測站分佈地很散，資料覆蓋率低，並不適合使用{{lang|en|CLEAN}}，而是使用了進行可見光干涉觀測的社群發展的新演算法<ref name = "obs AGN"/>。

=== 建立模型與電腦模擬 ===
{{main|數學模型|電腦模擬}}
黑洞本身是很單純的物體，根據[[無毛定理|無毛定理]]，只要[[質量|質量]]、[[電荷|電荷]]、[[自轉|自轉]]三個[[參數|參數]]便能完整描述其特性<ref name = "黑洞剪影"/>{{rp|83}}。複雜的是黑洞周遭的環境，有許多數據上顯示的特徵細節是由環繞黑洞的物質所決定<ref name = "EHT-Science"/>。理論天文物理學家根據廣義相對論建立了[[數學模型|數學模型]]與[[電腦模擬|電腦模擬]]。依[[愛因斯坦方程式|愛因斯坦方程式]]預測，望遠鏡將觀測到近乎圓形的黑洞剪影{{noteTag|1=或稱「陰影」（{{lang-en|Shadow}}）}}，也就是事件視界旁明亮外緣與幽暗內緣所形成的對比。看上去將如黑色的背景繞上了一圈流光。若人馬座A*擁有事件視界，但剪影形狀或大小偏離預測，就代表廣義相對論需要修正<ref name = "黑洞大考驗"/>{{rp|82-85}}<ref name = "aft obs-170503"/>。

== 建置過程 ==
=== 籌備會議 ===
2012年初，來自世界各地的天文學家在[[美國|美國]][[亞利桑那州|亞利桑那州]][[土桑市|土桑市]]第一次舉行{{lang|en|EHT}}會議，確定組成一個團隊，籌劃聯繫全球的[[電波望遠鏡|電波望遠鏡]]以觀測黑洞的事件視界，並將該計畫定名為「事件視界望遠鏡」。大多數相關望遠鏡設施都有代表參與會議，不過尚未確定組織分工和資金來源等細節。當時已有三架分別位於美國[[加州|加州]]、亞利桑那州和[[夏威夷|夏威夷]]的望遠鏡完成連線，以1.3毫米的波長觀測了幾年，並取得[[銀河系|銀河系]]中心模糊的影像，判定其具有黑洞的特徵。天文學家們希望能將波長降至0.83毫米、並與更多望遠鏡連線以擴展基線，提高角解析度。當時全球具有以差不多波長觀測的能力、或調整起來相對容易的望遠鏡數量約有一[[打|打]]。科學家估計需要花幾百萬[[美元|美元]]進行升級、調整一些望遠鏡的儀器。[[阿塔卡瑪大型毫米及次毫米波陣列|ALMA]]的66架電波碟形天線將會是計畫的關鍵之一，它的加入將大幅提升基線。{{lang|en|EHT}}在海斯塔克天文台的研究人員已從[[美國國家科學基金會|美國國家科學基金會]]獲得400萬美元的資助，用以幫{{lang|en|ALMA}}裝備{{lang|en|VLBI}}設備，預計將於2015年加入全球陣列。科學家也希望能將[[南極望遠鏡|南極望遠鏡]]升級後納入陣列之中。{{lang|en|EHT}}團隊預計在2012年夏天建立[[諒解備忘錄|諒解備忘錄]]（{{lang|en|MOU}}），但在此之前的測試等工作仍會在相對非正式的安排下進行<ref name="Science 120127"/><ref name="華盛頓郵報 120123"/><ref name = "EHT 2012meeting"/><ref name = "NSF statement 170405"/>。

=== 前置作業 ===
[[File:Nineteen_ALMA_antennas_on_the_Chajnantor_plateau.jpg|thumb]]（{{lang|en|ALMA}}）]]

觀測的前置作業包含發展及部署{{tsl|en|Submillimetre astronomy|次毫米天文學|次毫米}}雙極化接受器、使{{lang|en|VLBI}}能夠在230 – 450[[吉赫|吉赫]]（3.5毫米{{noteTag|1=<math>\textstyle f = \frac{c}{\lambda}</math>，頻率＝光速 / 波長}}）運行的高穩定頻率標準、更高頻寬的VLBI後端和記錄器，以及運行新的次毫米{{lang|en|VLBI}}觀測站<ref name="建置作業 2016-10"/>。對參與計畫的成員，{{lang|en|EHT}}必須派研究人員前往觀測設施，改裝它們硬體、安裝新的數位訊號處理器和數據記錄器<ref name = "Science 170302"/>。2006年第一次接收數據以來，逐漸有許多來自世界各地的天文觀測單位加入<ref name = "aft obs-170503"/>。經過多年，{{lang|en|EHT}}已從一個鬆散、資金不足的團隊，成長為30多所來自12個國家的大學、天文觀測站等研究單位與政府機構參與的國際合作組織<ref name="NYT-150608"/><ref name = "Science 170302"/>。{{lang|en|EHT}}使用[[Wiki|Wiki]]系統架設內部網站，作為研究人員的聯絡平台<ref name = "EHT WIKI"/>。

{{lang|en|ALMA}}的加入對{{lang|en|EHT}}很關鍵。2014年，科學家在{{lang|en|ALMA}}安裝了以[[邁射|氫邁射]]提供動力的特製[[原子鐘|原子鐘]]，取代以往使用[[銣|銣]]氣的時鐘以更精準計時。一旦完成，不但{{lang|en|ALMA}}將達到與{{lang|en|EHT}}陣列連線的標準，也成為其中儀器最靈敏的一員。它的加入使整體陣列的靈敏度提高10倍（比[[哈伯望遠鏡|哈伯望遠鏡]]還高2000倍<ref name = "aft obs-170503"/><ref name="ALMA Up 140604"/>），使{{lang|en|EHT}}能觀測對象大幅增加，並填補北半球和南極望遠鏡之間的空缺、提高南北向的訊號覆蓋率<ref name = "obs AGN"/>。2015年，{{lang|en|ALMA}}進行{{lang|en|EHT}}陣列中第一次{{lang|en|VLBI}}技術的測試。測試中，{{lang|en|ALMA}}與[[阿塔卡馬探路者實驗|阿塔卡馬探路者實驗]]（{{lang|en|APEX}}）連線形成2公里長的基線，並觀測經常被拿來當望遠鏡測試指標的0522-364[[類星體|類星體]]。兩望遠鏡透過網際網路向海斯塔克天文台傳送了5秒的觀測數據，以供研究人員確認系統是否正常運作。由於完整的數據量過於龐大，存在硬碟中郵寄去資料處理中心是最快的傳輸方式。研究團隊表示該次測試很成功<ref name = "ALMA test 150305"/><ref name = "ALMA & EHT 2017"/>。

位於[[南極點|南極點]]的[[南極望遠鏡|南極望遠鏡]]於2015年加入{{lang|en|EHT}}計畫。由於其位於地球軸點上和高海拔，擁有很乾燥的空氣，適合進行長期觀測<ref name="SPT++"/>。事件視界望遠鏡於2015年時已擁有9年的觀測數據，但當時的資料僅來自三座參與計畫的天線，尚不足以計算出黑洞事件視界的影像。之後陸續將有更多觀測臺加入{{lang|en|EHT}}陣列的全球電波望遠鏡網路，計畫希望能在法國、[[格陵蘭望遠鏡|格陵蘭]]、美國等地增加站點<ref name="硬碟、計畫06 2015"/>。該計畫預計於2017年攝得銀河的[[超大質量黑洞|超大質量黑洞]]、[[人馬座A*|人馬座A*]]的第一張影像<ref name="BBC 2016"/>。

== 觀測計畫==
=== 連線觀測 ===
{{Quote_box

事件視界望遠鏡計畫每年只會進行一次連線觀測，時間依各天文台觀測排程、天候、以及那個時期人馬座A*和{{lang|en|M87}}是否同時在天上而定<ref name = "Science 170302"/>。2006年為第一次觀測，有三座望遠鏡參與連線<ref name = "SA blog"/>。{{lang|en|EHT}}於2007年開始觀測人馬座A*、2009年開始觀測{{lang|en|M87}}，雖然早期角解析度相對低，數據也產生了許多重要的科學成果<ref name = "obs AGN"/><ref name="EHT Sgr A* structure"/>。經過多年籌備，近幾年事件視界望遠鏡的技術逐漸達到目標所要求的標準<ref name = "ESOblog-1"/>。計畫開始以來天文學家持續協調多座望遠鏡的觀測時間，也為各觀測站配置觀測所需儀器。因觀測的波長處於會被水氣吸收的波段，觀測受天氣影響很大<ref name = "bef obs-170405"/>，而大多數{{lang|en|EHT}}站點一般在冬季時會有較穩定的天氣<ref name="higher frequencies 150724"/>。

2017年4月，{{lang|en|EHT}}第一次有夠多的參與成員，使陣列達到足以觀測事件視界的角解析度。由[[哈佛大學|哈佛大學]]「黑洞倡議計畫」的辦公室改裝而成的觀測指揮中心會在4月5日至14日、共十天期間，每天開會決定是否進行觀測，一共會觀測五天。觀測的頭三個晚上所有觀測站的天氣和儀器情況都非常好。最後天文學家不得不中斷觀測兩天，讓工作人員休息、維護儀器，再完成剩下兩天的觀測<ref name = "SA blog"/><ref name = "Science 170302"/>。全球毫米波特長基線陣列（{{lang|en|GMVA}}）更早也在4月1至4日觀測[[人馬座A*|人馬座A*]]。其中{{lang|en|GMVA}}將觀測重點放在銀河系中心區域的[[吸積|吸積]]和[[超相對論性噴流|噴流]]，而{{lang|en|EHT}}則嘗試取得[[黑洞|黑洞]]剪影的影像<ref name = "bef obs-170405"/>。在此之前，{{lang|en|EHT}}也曾觀測過兩黑洞周圍的環境，然這是第一次加入{{lang|en|ALMA}}和[[南極望遠鏡|南極望遠鏡]]<ref name = "ESOblog-1"/><ref name = "aft obs-170517"/>。{{lang|en|ALMA}}為兩陣營重要的成員，除了因其為最大、最靈敏的望遠鏡外，也因其位於[[南半球|南半球]]<ref name = "bef obs-170405"/>。天文學家已能用第一批影像驗證某些[[廣義相對論|愛因斯坦重力理論]]的基本預測，電波天文學家海諾·法爾克（Heino Falcke）並表示這些影像能讓黑洞從某種神祕物體，變成可以研究的實體<ref name = "aft obs-170517"/>。然而因南極的人員必須在那度過南半球的冬天，硬碟要等到同年十月後才能運出來<ref name = "SA blog"/><ref name = "aft obs-170517"/>，整個陣列的數據至少等到該年底或2018年初才會處理完畢並公佈觀測影像<ref name = "aft obs-170503"/>。

未來，天文學家將繼續發展能提高解析度的技術、新的運算方法等，如將觀測頻率提升至345吉赫，也就是以0.87毫米的波長觀測。現在的230吉赫足夠觀察到黑洞的吸積流，而345吉赫將能[[光深度|看得更深]]，對光子軌道進行觀測<ref name="higher frequencies 150724"/>。天文學家也希望將來能做出即將消失於黑洞之中的物質繞行事件視界的動態影像<ref name = "aft obs-170503"/>。[[格陵蘭望遠鏡|格陵蘭望遠鏡]]預計於2018年加入EHT<ref name = "bef obs-170405"/><ref name="GLT170125"/>，且也希望能加入位於非洲以及太空中的站點<ref name = "aft obs-170517"/>。

=== 全球毫米波特長基線陣列 ===
全球毫米波特長基線陣列（{{lang-en|The Global mm-VLBI Array, GMVA}}）是個類似事件視界望遠鏡的電波望遠鏡觀測網路，其成員有位於夏威夷的[[次毫米波陣列望遠鏡|次毫米波陣列望遠鏡]]、智利的{{lang|en|ALMA}}（同時也為{{lang|en|EHT}}成員）、美國的{{tsl|en|Green Bank Telescope|綠堤望遠鏡}}，以及[[法國|法國]]、[[德國|德國]]、[[西班牙|西班牙]]、[[芬蘭|芬蘭]]、[[瑞典|瑞典]]等世界各處的電波望遠鏡<ref name = "bef obs-170405"/>。兩個陣營的差別在於GMVA觀測波長為3毫米，而{{lang|en|EHT}}在1.3毫米。兩者將合作觀測人馬座A*與M87的黑洞<ref name = "ESOblog-1"/>。

== 成果 ==
{{Current|小節|date=2019-04-09T15:26:56+00:00}}
以下列出2006年至2016年的重要觀測成果<ref name = "EHT-Science"/><ref name = "obs AGN"/>：
*若[[人馬座A*|人馬座A*]]和[[M87|M87]]沒有[[事件視界|事件視界]]，則繞行於周圍的[[吸積流|吸積流]]將加熱其表面時，使表面輻射的光譜在近紅外光波段產生峰值。然而，{{lang|en|EHT}}並無觀測到此現象，這表示它們應該擁有事件視界，而擁有事件視界即為黑洞的定義<ref name = "Sheperd 090621"/>。

*觀測數據嚴格限制了人馬座A*和M87的模型類別。從兩者模型看來，它們的旋轉向量相對於觀測方向顯著傾斜，[[吸積盤|吸積盤]]的邊緣比盤面離地球更近。

*[[廣義相對論|廣義相對論]]預測強大的重力將導致黑洞附近的物體從遠處看[[重力透鏡|比實際還大]]。在考慮星際介質造成的[[散射|散射]]之後，天文學家觀測到的毫米波輻射影像大小比理論值還要小30%。對觀測值的其中一個解釋為毫米波輻射源自吸積盤，其旋轉造成的都卜勒效應使輻射在吸積盤旋近與旋離的兩端呈不對稱的分佈<ref name = "results&plans 2011"/><ref name = "Shadow 2016"/><ref name = "Nature 2008"/>。

*天文學家觀測到來自人馬座A*發出輻射的亮度會隨時間變化，但發射區域的大小並沒有隨之而變動。雖然產生這種改變的機制尚不清楚，但{{lang|en|EHT}}的數據顯示這種改變出現在非常靠近黑洞的吸積流上。

*來自{{lang|en|M87}}中央的輻射顯示黑洞正在快速旋轉；同時，天文學家觀測到其巨大的[[超相對論性噴流|噴流]]的來源，推論該高速噴流為[[磁場|磁場]]所造成<ref name="Asiaa M87 2012"/><ref name = "Asiaa-jet-2015"/>。
