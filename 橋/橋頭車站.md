'''橋頭車站'''位於[[高雄市|高雄市]][[橋頭區|橋頭區]]，其臺語名稱以舊稱「橋仔頭」（{{lang-nan-tw|Kiô-á-thâu}}）稱之。為[[臺灣鐵路管理局|臺灣鐵路管理局]][[縱貫線_(南段)|縱貫線]]、[[高雄捷運|高雄捷運]][[高雄捷運紅線_(主線)|紅線]]的[[鐵路車站|鐵路車站]]，其中高雄捷運所使用的站名為'''橋頭火車站'''，代號'''R23'''。在[[中華民國|中華民國]][[中華民國文化部|行政院文化建設委員會]]在2001年舉辦的「[[臺灣歷史建築百景列表|歷史建築百景徵選活動]]」裡，舊臺鐵車站站體被選為第22名。<ref>{{Cite web|url=http://www.railway.gov.tw/Ciaotou/CP.aspx?SN=11248|title=不可不知|accessdate=2015-04-18|publisher=台灣鐵路管理局}}</ref>

== 車站構造 ==
{{rint|tw|kmrt}}捷運紅線車站的兩個出入口目前已經正式通車。並配合捷運轉乘，{{rint|tw|tra}}臺鐵停用舊站房並與捷運使用[[共構與共站|共構]]新站。
:{{rint|tw|tra}}台鐵縱貫線：平面車站（舊站已停用），[[側式月台|側式月台]]、[[島式月台|島式月台]]各一座。
:{{rint|tw|kmrt}}<font color={{高雄捷運色彩|紅}}>█</font>[[高雄捷運紅線_(主線)|紅線]]：高架車站，島式月台一座。
*地上三層，共有兩個出口。

橋頭捷運車站最引人注目的便是二號出入口的巨幅陶壁作品《天工開物》，由台灣知名陶藝家朱邦雄製作。天工開物寬九公尺、高十二公尺，共由1311塊陶板組成，製作過程中耗費的陶土超過70公噸。<ref>{{Cite web|url=http://kcginfo.kcg.gov.tw/Publish_Content.aspx?n=A22859B204186560&s=161DEBC9EACEA333&chapt=7807&sort=1|title=美濃窯 陶壁公共藝術的推手|accessdate=2015-04-18|publisher=高雄市政府新聞局|archive-url=https://web.archive.org/web/20150418115741/http://kcginfo.kcg.gov.tw/Publish_Content.aspx?n=A22859B204186560&s=161DEBC9EACEA333&chapt=7807&sort=1|archive-date=2015-04-18|dead-url=yes}}</ref>

== 營運狀況 ==
*{{rint|tw|tra}}臺鐵車站部分目前為[[臺灣鐵路管理局車站等級#三等站|三等站]]，僅[[臺鐵區間車|區間車]]停靠。為往[[岡山車站|岡山]]、[[路竹車站|路竹]]、[[臺南車站|臺南]]方向旅客兩鐵轉乘站之一，捷運通車後業績增長2倍、假日5.7倍。由於高捷免費搭乘結束，使臺鐵轉搭高捷人潮銳減，臺鐵決定取消[[對號列車|對號列車]]停靠，改停區間快車<ref>[http://www.trc.club.tw/news.php?uid=12&id=7185&stype=2&q=%8B%B4%93%AA%E2%8B 方便轉高捷 橋頭站停靠區間快]{{dead link|date=2018年1月 |bot=InternetArchiveBot |fix-attempted=yes }}TRC!臺灣鐵道網</ref>。
*{{rint|tw|tra}}臺鐵車站一樓站房已經停用，現在由與高捷共構的[[跨站式站房_(台鐵)|跨站式站房]]出入車站。
*{{rint|tw|tra}}臺鐵舊式[[閩南語|閩南語]]站名廣播依橋頭舊名「橋仔頭」發音，新式站名廣播一度更改為「橋頭」直接發音，2016年底將新式的站名廣播由「橋頭」改回「橋仔頭」發音。
*{{rint|tw|tra}}臺鐵站內設有「鐵路之旅－小站巡禮紀念章」之戳章，可供旅客蓋戳留念。
*{{rint|tw|kmrt}}捷運橋頭火車站由於月台前後沒有[[橫渡線|橫渡線]]，列車需經由[[南岡山站|南岡山]]－橋頭間[[北機廠_(高雄)|北機廠]]的[[道岔|道岔]]調頭，但在2012年12月23日南岡山站通車後，本站回復為一般的停靠站，原先的落客月台亦回復載客功能。
*{{rint|tw|kmrt}}高雄捷運所屬的「橋頭火車站」在[[南岡山站|南岡山站]]通車前為紅線的終點站，代號R23。{{KMRT ridership-date}}日均進出人次為{{KMRT ridership|station=橋頭車站}}，在高雄捷運系統中排名第{{KMRT ridership-rank|station=橋頭車站}}名；目前最高紀錄為2012年11月的10,733人次，最低紀錄為2015年8月的4,153人次。在南岡山站通車前，由於是端點站，大岡山地區的接駁車均由橋頭車站發車，因而使用率較高，為高雄捷運高架車站中之第一名；南岡山站通車後，大部分公車客運改至南岡山站發車，且南岡山站有廣大腹地以做停車場等，橋頭車站運量大幅下跌至四千餘人次。2018年底，中央決定在橋頭車站、橋頭糖廠站東側高速公路一帶設立[[南部科學園區|南科]]橋頭園區，完工後運量應能得到挹注。
*本站開放使用[[台灣電子票證|電子票證]]付費：
**{{rint|tw|tra}}台鐵：[[一卡通_(台灣)|一卡通]]、[[悠遊卡|悠遊卡]]及[[icash|icash 2.0]]。
**{{rint|tw|kmrt}}高雄捷運：一卡通、悠遊卡、icash 2.0及[[HappyCash|HappyCash]]。

== 歷史 ==
===臺鐵橋頭車站===
*1901年5月19日：設立「橋仔頭乘降場」。<ref name=":0">{{cite book |author = 戴震宇 |authorlink =  |title = 《台灣的老火車站》 |publisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>
*1920年10月1日：改稱「橋仔頭驛」。<ref name=":0" />
*1955年3月1日：改稱「橋頭」。<ref name=":0" />
*2008年3月18日：停用日治時期舊站房，遷入捷運共構新站。
*2013年9月30日：啟用多卡通刷卡機。
*2018年8月23日：由於熱帶性低氣壓影響，造成岡山=橋頭區間溪水超過警戒線雙向停駛。首次有[[普悠瑪號|普悠瑪號]](北上136次)停靠，隨後原班列車改南下133次普悠瑪號返回潮州。

===高雄捷運橋頭火車站===
*2007年：由美濃窯朱邦雄博士創作橋頭站陶壁公共藝術「天工開物」，尺寸12公尺高 × 9公尺寬 × 6公分至45公分厚。
*2008年3月9日：隨著捷運紅線「'''橋頭火車站'''—[[小港站|小港]]」段正式通車而啟用。
*2012年12月23日：紅線自本站往北延伸通車至[[南岡山站|南岡山站]]。
