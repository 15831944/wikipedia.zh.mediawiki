'''橋口隆志'''（{{lang-ja|はしぐち たかし}}、{{bd|1967年|6月2日|catIdx=Hashiguchi Takashi}}），是日本漫畫家。[[東京都|東京都]]出身。以[[日式麵包王|日式麵包王]]獲得第49回[[小学馆漫画赏|小学馆漫画赏]]。

== 人物 ==

== 作風 ==

== 經歷 ==
* 1987年 - [[週刊Young_Magazine|週刊Young Magazine]]得獎。[[千葉徹彌賞|千葉徹彌賞]]獲得佳作。
* 1988年 - 「コンバットティーチャー」首次亮相。
* 1991年 - [[快樂快樂月刊|快樂快樂月刊]]上開始連載[[チエばーちゃんの知恵ブクロ|チエばーちゃんの知恵ブクロ]]
* 1993年 - [[快樂快樂月刊|快樂快樂月刊]]上開始連載[[ストII爆笑!!4コマギャグ外伝|ストII爆笑!!4コマギャグ外伝]]
* 1995年 - [[週刊少年Sunday|週刊少年Sunday]]雜誌上開始連載[[キャスター参るゾ!|キャスター参るゾ!]]
* 1997年 - [[快樂快樂月刊|快樂快樂月刊]]上開始連載[[超速YOYO|超速YOYO]]。[[週刊少年サンデー超|週刊少年サンデー超]]上開始連載[[ウインドミル_(漫画)|ウインドミル (漫画)]]
* 1998年 - [[超速YOYO|超速YOYO]]在[[東京電視台|東京電視台]]準備開始動畫化。
* 2000年 - [[コミックGOTTA|コミックGOTTA]]雜誌上開始連載[[シザーズ|シザーズ]]
* 2001年 - [[週刊少年Sunday|週刊少年Sunday]]雜誌上開始連載[[烘焙王|烘焙王]]。
* 2003年 - [[烘焙王|烘焙王]]在第49回[[小学馆漫画赏|小学馆漫画赏]]少年部門受賞。
* 2004年 - [[烘焙王|烘焙王]]在[[東京電視台|東京電視台]]準備開始動畫化。
* 2007年 - [[頂尖神醫|頂尖神醫]]連載開始。
* 2010年 - [[頂尖神醫|頂尖神醫]]第二部連載開始。
