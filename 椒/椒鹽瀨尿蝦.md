'''椒鹽瀨尿蝦'''是[[香港|香港]]一道[[海鮮|海鮮]]菜式<ref>[http://hk.apple.nextmedia.com/supplement/food/art/20050823/5160332 飲食人誌：韓國Boyz來港學師 標住眼水食避風塘辣蟹]，蘋果日報，2005-08-23</ref>，[[瀨尿蝦|瀨尿蝦]]經過[[油炸|油炸]]後，再與已炒香的[[辣椒|辣椒]]、[[蒜茸|蒜茸]]及[[食鹽|食鹽]]一起在[[鑊|鑊]]中爆[[炒|炒]]而成，瀨尿蝦的外殼炸得香脆沾滿[[椒鹽|椒鹽]]，而蝦殼內的蝦肉保持鮮甜多汁，成為香港[[避風塘|避風塘]]一道著名的海鮮菜式<ref>[http://www.hkcd.com/content/2014-07/25/content_861241.html 香港吃海鮮好去處 大快朵頤南大門]，香港商報，2014-07-25</ref>。

== 特色 ==
香港常見的海鮮菜式大多保留傳統[[粵菜|粵菜]]的特質，盡量保持原汁原味，較少使用濃味的調味料，並且以清蒸為主，而椒鹽瀨尿蝦卻特顯香港避風塘菜式的特點<ref>[http://bkb.mpweekly.com/%E9%A3%B2%E9%A3%B2%E9%A3%9F%E9%A3%9F/%E6%88%91%E5%BE%88%E9%86%9C%EF%BC%8C%E4%BD%86%E6%88%91%E5%BE%88%E7%BE%8E%E5%91%B3%E3%80%80%E7%80%A8%E5%B0%BF%E8%9D%A6 我很醜，但我很美味 瀨尿蝦]，明周，2017-05-27</ref>，以爆炒為主，令食材香口惹味<ref>[http://hk.apple.nextmedia.com/supplement/food/art/20110317/15082366 避風塘惹味四料 三缺一辣唔成 ]，蘋果日報，2011-03-08</ref>，並保留海鮮的鮮甜味道，更可作為下酒菜<ref>[http://news.cts.com.tw/cts/life/201303/201303081205223.html#.WXS36-lLeUk 蝦螃蟹當主角 港式海鮮攻台]，中華電視，2013-03-08</ref>。椒鹽瀨尿蝦的基本材料是瀨尿蝦，調味料主要是切碎的辣椒、蒜茸及食鹽，為使油炸後的瀨尿蝦外殼能夠香脆可口，通常瀨尿蝦會在油炸前沾上[[玉米粉|玉米粉]]。瀨尿蝦經過第一次油炸後，會盛起備用，然後用熱油把辣椒、蒜茸及食鹽炒香，再把經過油炸的瀨尿蝦放回鑊中，以猛火快炒使辣椒鹽沾滿蝦殼後即可盛起食用<ref>[https://www.e123.hk/ElderlyPro/details/88936/55/ 椒鹽瀨尿蝦]，長青網，2011-08-11</ref>。

== 譯名 ==
椒鹽瀨尿蝦是香港避風塘的著名美食，是位於[[香港仔|香港仔]]、[[鯉魚門|鯉魚門]]、[[西貢|西貢]]、[[南丫島|南丫島]]等旅遊區的海鮮酒家必備的菜式，在[[佐敦|佐敦]]的[[大排檔|大排檔]]，以及一些位於市政大廈的食肆，都有供應這道海鮮菜式。香港有食肆為招待海外遊客於是把椒鹽瀨尿蝦翻譯為英文<ref>[http://orientaldaily.on.cc/cnt/news/20151029/00176_150.html 「優質」餐館亂譯菜名 旅發局丟架]，東方日報，2015-10-29</ref>，但部分食肆的翻譯卻被指令人難以看懂<ref>[https://www.hk01.com/%E7%A4%BE%E5%8D%80/65457/%E8%8C%B6%E9%A4%90%E5%BB%B3%E8%8F%9C%E5%96%AE%E7%A5%9E%E7%B4%9A%E8%8B%B1%E8%AD%AF-%E6%83%B3%E7%9F%A5%E7%80%A8%E5%B0%BF%E7%89%9B%E4%B8%B8%E9%BB%9E%E6%A8%A3%E8%AD%AF- 茶餐廳菜單神級英譯 想知瀨尿牛丸點樣譯？]，香港01，2017-01-14</ref>。

== 過敏 ==
椒鹽瀨尿蝦雖然美味，但部分人會對瀨尿蝦等海鮮食品出現過敏，輕微的皮膚出現紅疹，部分嚴重個案或會有生命危險<ref>[http://ny.uschinapress.com/spotlight/2016/10-14/105023.html 闽男吃濑尿虾过敏致脑死亡]，纽约侨报，2016-10-14日</ref>。此外，處理瀨尿蝦時如果不慎被刺傷，傷口又處理不當，也會有[[弧菌|弧菌]]感染導致截肢的危險<ref>[http://www.appledaily.com.tw/realtimenews/article/new/20150818/672528/ 被瀨尿蝦刺手指 男子高燒險截肢]，蘋果日報，2015-08-18</ref>。
