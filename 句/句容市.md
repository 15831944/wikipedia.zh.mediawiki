'''句容市'''是[[中国|中国]][[江苏省|江苏省]][[镇江市|镇江市]]代管的一个[[县级市|县级市]]。位于镇江市西南部，北临[[长江|长江]]，西接南京，是[[南京|南京]]的东南门户，素有“南京新东郊、[[金陵|金陵]]御花园”之美誉。

== 历史 ==

句容在[[唐代|唐代]]属于[[江南东道|江南东道]]之[[润州|润州]]管辖。[[北宋|北宋]]时期属于江南东路之[[江宁府|江宁府]]（南宋时期改称建康府）管辖。明代属于[[南直隶|南直隶]][[应天府_(明朝)|应天府]]。清代属于江苏省江宁府。

==行政区划==
句容市辖3个街道、8个镇：华阳街道、崇明街道、[[江苏省句容经济开发区|句容开发区（黄梅街道）]]、下蜀镇、白兔镇、边城镇、茅山镇、后白镇、郭庄镇、天王镇、宝华镇。

== 地理 ==
句容位于江苏省西南部，西部毗邻[[南京市|南京市]][[江宁区|江宁区]]，东部毗邻[[镇江市|镇江市]][[丹徒区|丹徒区]]，东南方毗邻[[常州市|常州市]][[金坛区|金坛区]]及[[溧阳市|溧阳市]]，西南方毗邻[[溧水区|溧水区]]。

句容地处[[宁镇丘陵|宁镇丘陵]]地区。

==名胜==
*[[隆昌寺|隆昌寺]]，位于宝华山，是[[汉族地区佛教全国重点寺院|汉族地区佛教全国重点寺院]]，其无梁殿和铜殿是[[江苏省文物保护单位|江苏省文物保护单位]]。
*[[茅山|茅山]]，位于句容东南与[[金坛|金坛]]交界处，中国著名道教名山，[[上清派|上清派]]发祥地，道教称之为“第八洞天，第一福地”。茅山道教，历史悠久，渊源流长。
*[[赤山_(江苏)|赤山]]，位于句容与[[南京|南京]]交界，是江南最大的死火山，赤山映雪是湖熟古八景之一，风景秀丽。
*[[瓦屋山|瓦屋山]]，句容市的南端，北与道教圣地茅山相呼应，西邻东屏湖国际度假村，南靠碧波荡漾的天目湖，距镇江市区50多公里。植被茂盛，峰峦奇秀。置身山中，耳听林涛合着鸟鸣，眼望青山伴着绿水，信步游览，风光无限，真可谓山山显秀，处处见景。经专家评定，瓦屋山的生态环境在全省旅游景点中排名首位。

== 监狱 ==
