'''{{FULLPAGENAME}}'''是位于[[江苏省|江苏省]][[句容市|句容市]]的一个[[国营林场|国营林场]]，是[[镇江市|镇江市]]的五个国营林场之一<ref>{{Cite web |url=http://www.jsforestry.gov.cn/art/2016/5/26/art_6_88321.html |title=仲志勤委员督查镇江市国有林场改革推进工作 |accessdate=2018-11-06 |author=|date=2016-05-26 |work= |publisher=江苏省林业局网站|language=Zh-hans }}</ref>。

2012年及之前，在[[国家统计局|国家统计局]]网站统计中<ref>{{Cite web |url=http://www.stats.gov.cn/tjsj/tjbz/tjyqhdmhcxhfdm/2012/32/11/83/321183407.html |title=句容市林场 |accessdate=2018-11-06 |author=|date= |work= |publisher=国家统计局网站|language=Zh-hans }}</ref>，所在区域以'''句容市林场'''之名列为句容市的一个[[类似乡级单位|类似乡级单位]]<ref>{{PRC admin/ref sgc|{{PRC admin/parent|32/11/83/404/000}}}}</ref>。2017年4月，句容市林场被[[中国林场协会|中国林场协会]]推选为“2016年度全国十佳林场”之一<ref>{{Cite web |url=http://www.jurong.gov.cn/jurong/bmdt/201704/c46403cbbadd402188732eec78975925.shtml |title=句容市林场获评2016年度全国十佳林场 |accessdate=2018-11-06 |author=|date=2017-04-24 |work= |publisher=句容市人民政府网站|language=Zh-hans }}</ref>。
