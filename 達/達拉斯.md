'''达拉斯'''（{{lang-en|'''Dallas'''}}），是[[美国|美国]][[德克萨斯州|德克萨斯州]]第三大城市，美国第九大城市，[[达拉斯县_(德克萨斯州)|达拉斯县]]首府，城市面积共{{convert|385|sqmi|sqkm|0|lk=on}}。美国人口普查显示，至2006年7月1日，达拉斯的人口为1,248,810人。达拉斯与[[沃斯堡|沃斯堡]]、[[阿灵顿_(德克萨斯州)|阿灵顿]]组成达拉斯-沃斯堡-阿灵顿[[城市群|城市群]]，覆盖12个县，总人口615万人（2007年数据），是美国第四大城市群。1999年，达拉斯被[[拉夫堡大学|拉夫堡大学]]的全球化与世界级城市研究小组与网络列为第三类[[世界级城市|世界级城市]]，即小型世界级城市。

== 历史 ==
达拉斯创立于1814年，1856年正式建市，主要经济支柱为石油工业、电信业、计算机技术、银行业和航空运输业。达拉斯位于美国内陆，缺少水上交通线，其蓬勃的经济发展来自于强大的石油和棉花工业。此外，达拉斯位于数条铁路线和高速公路线上，陆上交通非常发达。

1963年11月22日，時任[[美國總統|美國總統]][[約翰·肯尼迪|約翰·肯尼迪]]在達拉斯進行競選活動時被暗殺身亡。

2016年7月7日，達拉斯發生了美國自911事件以來最嚴重的[[2016年达拉斯警察枪击案|槍擊案]]。

== 地理 ==
達拉斯是[[達拉斯縣_(德克薩斯州)|達拉斯縣]]首府。城市的一部分延伸到了周圍的[[科林縣_(德克薩斯州)|科林縣]]、[[登頓縣_(德克薩斯州)|登頓縣]]、[[考夫曼縣_(德克薩斯州)|考夫曼縣]]及[[羅克沃爾縣_(德克薩斯州)|羅克沃爾縣]]。

根據[[美國人口調查局|美國人口調查局]]的資料，達拉斯面積為385平方英里（997.1平方公里），其中水域面積約佔11.03%。目前德州約有四分之一的人住在大達拉斯-[[沃斯堡|沃斯堡]]都會區裡，而達拉斯佔整個大都會區的五分之一人口。

== 气候 ==
达拉斯属[[副热带湿润气候|副热带湿润气候]]，四季分明，降水充足而集中在5、6、和10月。冬季温和，时而偏向温暖，日最高气温低于或等于{{convert|0|C}}的平均日数为2.1天，日最低气温低于或等于{{convert|0|C}}的平均日数为26天，低于或等于{{convert|-5|C}}的平均日数为4.6天；夏季酷暑漫长（西南部沙漠地区之外，达拉斯夏季平均气温为全国最高），日高温达{{convert|30|C}}的日数年均有133天，达{{convert|100|F|disp=flip}}的有19天；日低温达{{convert|25|C}}的年均日数为46天。<ref name = "NWS Fort Worth, TX"/>最冷月（1月）均温{{convert|47.0|F|1|disp=flip}}，极端最低气温{{convert|−3|F|0|disp=flip}}（1930年1月18日）。<ref name = "NWS Fort Worth, TX"/>最热月（8月）均温{{convert|86.6|F|1|disp=flip}}，极端最高气温{{convert|112|F|0|disp=flip}}（1980年6月26日与27日）。<ref name = "NWS Fort Worth, TX"/>无霜期平均为270天（3月2日至11月26日）。<ref name = "NWS Fort Worth, TX"/>年均降水量约{{convert|37.6|in|sigfig=3|disp=flip}}；年极端最少降水量为{{convert|17.52|in|disp=flip}}（1963年），最多为{{convert|55.31|in|disp=flip}}（1981年）。<ref name = "NWS Fort Worth, TX"/>年均降雪量为{{convert|1.5|in|cm|disp=flip}}；1944–45年无雪，并且许多冬季的积累降雪量微少以致难以测量，1977–78年的降雪量最多，积累降雪量为{{convert|14.1|in|cm|disp=flip}}。<ref name = "NWS Fort Worth, TX"/>

== 交通 ==
[[File:Dfw_airport.jpg|thumb]]
[[File:DART_Blue_Line_in_Pearl_Station.JPG|thumb]]
达拉斯市内的主要交通工具是私人汽车。此外，市内拥有德克萨斯州最早建成的[[轻轨运输|轻轨列车]]系统。达拉斯是数条重要[[州际公路系统|州际高速公路]]的交汇处，[[20号州际公路|20号]]、[[30号州际公路|30号]]、[[35E号州际公路_(德克萨斯州)|35E号]]及[[45号州际公路|45号]]州际公路均经过达拉斯。

达拉斯是全美重要的航空中心，拥有2座商用机场，分别为[[达拉斯-沃斯堡国际机场|达拉斯-沃斯堡国际机场]]及[[达拉斯爱田机场|达拉斯爱田机场]]，此外还有1座通用航空机场。另1座通用航空机场位于[[爱迪森_(德克萨斯州)|爱迪森]]境内，邻近的[[麦金尼_(德克萨斯州)|麦金尼]]和[[沃斯堡|沃斯堡]]还各有2座通用航空机场。其中，达拉斯-沃斯堡机场位于达拉斯北部郊区同沃斯堡的交界上，是美国南部的航空中心、德克萨斯州最繁忙的机场，也是全美第3繁忙的机场。[[美国航空|美国航空]]总部即位于距机场1英里范围内。爱田机场则位于市区西北6英里处，是[[西南航空|西南航空]]的总部所在地。
