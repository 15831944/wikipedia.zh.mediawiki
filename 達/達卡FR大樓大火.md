'''達卡FR大樓大火'''，是2019年3月28日發生於[[孟加拉國|孟加拉國]]首都[[達卡|達卡]]的一場火災。當天[[孟加拉國標準時間|當地時間]]下午1:00左右，達卡市中心{{link-en|巴納尼區|Banani Model Town}}一棟高22層的大樓FR大樓（FR Tower）發生大火<ref>{{cite web |title=Banani building fire death toll rises to 19 |url=https://www.thedailystar.net/city/fire-in-dhaka-it-broke-out-at-banani-fr-tower-1721548 |website=The Daily Star |accessdate=28 March 2019 |language=en |date=28 March 2019}}</ref><ref>{{cite web |last1=CNN |first1=Sugam Pokharel and Helen Regan |title=Rescue underway to free people trapped in deadly Dhaka tower fire |url=https://edition.cnn.com/2019/03/28/asia/dhaka-building-fire-intl/index.html |website=CNN |accessdate=28 March 2019}}</ref>，共造成25人罹難<ref>{{cite web |title=People ‘jumping out of 19-storey skyscraper’ as blaze rips through building in Bangladesh capital |url=https://www.independent.co.uk/news/world/asia/dhaka-fire-today-fr-tower-banani-bangladesh-latest-a8843341.html |website=The Independent |accessdate=28 March 2019 |language=en |date=28 March 2019}}</ref>、超過70人受傷<ref>{{cite web |title=Deadly blaze hits Dhaka skyscraper |url=https://www.bbc.com/news/world-asia-47733063 |website=bbc.com |accessdate=28 March 2019 |date=28 March 2019}}</ref>，罹難者中有一人為[[斯里蘭卡|斯里蘭卡]]籍<ref>{{cite web |title=Banani fire: Sri Lankan man falls to death |url=https://www.thedailystar.net/city/dhaka-banani-fr-tower-fire-sri-lankan-man-falls-death-1721584 |website=The Daily Star |accessdate=28 March 2019 |language=en |date=28 March 2019}}</ref>。事發後，附近的{{link-en|Duronto電視台|Duronto TV}}與{{link-en|今日電台|Radio Today}}因辦公室與火災現場太近而停止播送<ref>{{cite web |title=Banani fire: Transmission of Duronto TV, Radio Today shut down |url=https://www.thedailystar.net/city/news/banani-fire-transmission-duronto-tv-radio-today-shut-down-1721617 |website=The Daily Star |accessdate=28 March 2019 |language=en |date=28 March 2019}}</ref>。
