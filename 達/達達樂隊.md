'''達達樂隊'''是1996年成立於[[中華人民共和國|中華人民共和國]][[湖北省|湖北省]][[武漢市|武漢市]]的一個男子[[搖滾樂隊|搖滾樂隊]]。<ref name="全紀錄">{{cite web | url=http://ent.sina.com.cn/2000-09-21/17329.html| title=達達樂隊音樂全紀錄| author=新浪影音娛樂| accessdate=2014-02-09}}</ref>
2006年因為公司和樂團內部問題而解散。<ref>{{cite web | url=http://ent.163.com/07/0814/17/3LSGALLM0003rt.html | title=达达乐队解散主唱单飞 彭坦新专辑依然... (图)| author=[[網易新聞|網易新聞]]| accessdate=2014-02-09}}</ref>

達達樂隊的成員於1999年固定，<ref name="全紀錄"/> 其中主唱為[[彭坦|彭坦]]、吉他手為吳濤和彭坦、貝斯手為魏飛、鼓手為張明。2000年6月達達樂隊與[[華納唱片|華納唱片]]簽約。<ref name="全紀錄"/> 2001年推出首張專輯《天使》，2003年12月發佈第二張專輯《黃金時代》，<ref>{{cite web | url=http://www.chinanews.com/n/2003-12-25/26/384989.html| title=全新专辑全面上市 达达乐队进入“黄金时代”| author=[[中國新聞網|中國新聞網]]| accessdate=2014-02-09}}</ref> 此專輯銷量迅速突破40萬。<ref>{{cite web | url=http://ent.sina.com.cn/p/i/2004-01-11/1901275142.html| title=达达乐队“来势汹汹” 新专辑销售冲40万(图)| author=新浪影音娛樂| accessdate=2014-02-09}}</ref>
