'''達觀號海洋測量艦（AGS-1601）'''為[[中華民國交通部|中華民國交通部]]代[[中華民國海軍|海軍]]於1994年向[[義大利|義大利]]購買的海洋測量艦。平時以[[台灣海峽|台灣海峽]]週邊海域專屬[[經濟海域|經濟海域]]與[[海南諸島|海南諸島]]之航道測量與水文資料蒐集任務為主，必要時支援國內、國際海洋研究計畫及臨時賦予之任務。<ref>{{Cite web |url= http://navy.mnd.gov.tw/Publish.aspx?cnid=866&p=10166&Level=2 |title= 達觀軍艦-海洋測量艦 |date=2013-04-29  |publisher= 中華民國海軍 |language= zh-tw |accessdate= 2013-06-10 |quote= }}</ref><ref>{{Cite web |url= http://60.250.180.26/taiwan/4518.html |title= 1601 達觀 |date= |publisher= 中國軍艦博物館 |language= zh-tw |accessdate= 2013-06-10 |quote= }}</ref><ref>{{Cite web |url= http://www.mdc.idv.tw/mdc/navy/rocnavy/AGS1601.htm |title= 台灣 達觀號海洋測量艦 AGS 1601 |date= |publisher= MDC軍武狂人夢  |language= zh-tw |accessdate= 2013-06-10 |quote= }}</ref>現屬[[海軍192艦隊|海軍192艦隊]]。
