'''達米阿那'''
英文學名：Turnera diffusa、Turnera aphrodisiaca ，被稱為達米阿那 Damiana<ref>{{cite web |title=Damiana |url=https://plants.usda.gov/core/profile?symbol=TUDI |website=Natural Resources Conservation Service PLANTS Database. USDA.}}</ref>，是一種灌木，它屬於西番蓮科，原產於[[美國|美國]]的[[德克薩斯州|德克薩斯州]]南部、中美洲的[[墨西哥|墨西哥]]；南美洲和[[加勒比地區|加勒比海地區]]<ref>{{cite book |author1=J. H. Everitt |author2=Dale Lynn Drawe |author3=Robert I. Lonard |title=Trees, shrubs & cacti of south Texas |date=2002 |publisher=Texas Tech University Press |isbn=978-0-89672-473-0 |page=208 |url=https://books.google.com.tw/books?id=8cEq7weUPYYC&redir_esc=y}}</ref>。

Damiana 是一種相對較小的灌木狀植物，可產生小而芳香的花朵。它在初夏至夏季開花，所結之果實味道類似無花果。由於植物中存在揮發精油，damiana 具有類似洋甘菊的強而芬芳的香料味。<ref>{{cite book |author1=Friedrich Hoffmann |editor=Gildemeister Eduard |title=The Volatile Oils. Volume 3 |date=1922 |edition=2 ed |url=https://archive.org/details/in.ernet.dli.2015.42679/page/n9}}</ref>
<br>
== 傳統使用 ==
* 達米阿那是運用在傳統墨西哥利口酒的成份，有時在瑪格麗塔雞尾酒中代替橙皮甜酒<ref>{{cite web |title=Triple sec |url=https://en.wikipedia.org/wiki/Triple_sec |website=Wikipedia}}</ref>。<br>

* 墨西哥民間聲稱它是瑪格麗塔雞尾酒最原始的原料。

* 達米阿那-瑪格麗塔雞尾酒在墨西哥的[[洛斯卡沃斯|洛斯卡沃斯]]地區很受歡迎。<ref>{{cite web |title=Wine & Spirits; The unexpected thrill |url=http://articles.latimes.com/2007/jun/20/food/fo-drink20 |website=Los Angeles Times}}</ref>

* 達米阿那被列入19世紀的幾種專利藥物中，例如：Pemberton 的 French Wine Coca。<ref>{{cite book |author1=Mark Pendergrast |title=For God, Country, and Coca Cola: The Definitive History of the Great American Soft Drink and the Company That Makes It |publisher=Basic Books |isbn=978-0-46505-468-8 |edition=2nd., rev. and expanded}}</ref>

== 膳食補充的運用 ==

這種植物可以茶的方式使用：每杯使用一茶匙 ( 約 5 克 ) 粉末。將沸水倒入裝有草本粉末的杯中，讓其浸泡 10分鐘。（泡製完成，飲用前可適度添加甜味調味品）。傳統上每天使用 2-3次。<ref>{{cite web |title=Damiana Powder |url=http://www.rain-tree.com/damiana-powder.htm#.XBS4F9sza70 |website=rain-tree}}</ref>

乾葉運用：傳統上使用 2-4 克乾葉，在 150 毫升沸水中 5-10 分鐘，每天使用 2-3 次。<ref>{{cite web |title=Herbs & Supplements: Damiana (Turnera diffusa) |url=http://www.betterlifewholefoods.com/ns/DisplayMonograph.asp?storeID=3d69a9440ed7476b99aaa9ddc54fd883&DocID=bottomline-damiana |website=better life whole foods}}</ref>

「粉狀」或「膠囊」的形式因食用方便，在歐美流行充當食品及膳食補充品，每日兩次，每次 3至4克。<ref>{{cite web |title=Damiana |url=https://www.herbal-supplement-resource.com/damiana-herb.html |website=herbal-supplement-resource}}</ref>佐以酸甜的飲品，可以消除其苦味，或撒於料理之上，其苦味又可襯托及增添料理的風味。

== 屬性 ==
● 它具有提升性慾和催情的特性，由它的學名之一 Turnera aphrodisiaca ( aphrodisiaca 英文一詞是催情劑之義 ) 便可得知其催情的作用。<br>
<br>● 是一種溫和的助排便劑，可用於治療由於胃腸肌肉所引起的便秘。<ref>{{cite web |title=Damiana – A Medicinal Herb |url=https://www.herbal-supplement-resource.com/damiana-herb.html |website=herbal-supplement-resource}}</ref><br>
<br>● 神經系統的刺激物，類似於士的寧，但沒有毒性潛力。<br>
<br>● 它作用於脊柱中樞、刺激排尿、勃起和射精。<ref>{{cite journal |author1=Sabna Kotta, Shahid H. Ansari, Javed Ali|title=Exploring scientifically proven herbal aphrodisiacs. |journal=Pharmacognosy reviews |date=2013-01 |volume=7 |issue=13 |pages=1-10 |doi=10.4103/0973-7847.112832 |pmid=23922450 |url=https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3731873/}}</ref><br>
<br>● 它用於男性陽痿和女性性冷感，特別是如果是由於精神原因所導致的性冷感。<br>
<br>● 根據食用的劑量不同，它也是一種強效的[[抗抑鬱藥|抗抑鬱藥]]，有助於提升整體情緒，帶來一種幸福感。<ref>{{cite web |title=Turnera diffusa |url=https://es.wikipedia.org/wiki/Turnera_diffusa |website=wikipedia}}</ref><br>
<br>● 具有激素平衡特性。它用於幫助女性[[更年期|更年期]]常見的[[荷爾蒙|荷爾蒙]]失調  。<ref>{{cite web |title=Damiana – A Medicinal Herb |url=https://www.herbal-supplement-resource.com/damian |website=herbal-supplement-resource}}</ref><br>
<br>● 中樞神經系統抑製劑，以幫助中樞神經系統疾病，並改變和控制神經反應。<ref>{{cite web |title=Damiana – A Medicinal Herb |url=https://www.herbal-supplement-resource.com/damiana-herb.html |website=herbal-supplement-resource}}</ref><br>
<br>● 具有利尿和防腐性能，並已被用作尿道感染（如膀胱和輸尿管炎症）的草藥治療。這種效果部分是由於植物中發現的[[熊果苷|熊果苷]]物質，它在輸尿管中轉化為對苯二酚（一種強效的防腐劑）。<br>
在熊果（Arctostaphylos uva-ursi）等許多其他植物中也發現了熊果苷。<ref>{{cite web |title=Damiana – A Medicinal Herb |url=https://www.herbal-supplement-resource.com/damiana-herb.html |website=herbal-supplement-resource}}</ref><br>

== 「可能」的副作用和相互作用 ==

*確實有傳統作為流產的使用背景，因此建議 "孕婦應該避免使用"。

*對兒童的安全性尚未經過測試，因此兒童應該避免使用。

*確實具有 "輕微的降血糖" 作用，因此，"糖尿病藥物治療" 或 "低血糖患者" 必須謹慎使用這種植物。

*它可能會 "干擾人體對鐵的吸收" ，因此應該在這種使用時注意鐵的水平。<ref>{{cite web |title=Damiana |url=https://www.herbal-supplement-resource.com/damiana-herb.html |website=herbal-supplement-resource}}</ref>
