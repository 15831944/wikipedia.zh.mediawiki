{{vgname|骑马与砍杀II：霸主|en=Mount & Blade II: Bannerlord}}是一款由[[TaleWorlds|TaleWorlds]]開發的中世紀背景[[動作角色扮演遊戲|動作角色扮演遊戲]]，是2007年[[TaleWorlds|TaleWorlds]]開發游戏《[[骑马与砍杀|骑马与砍杀]]》的续作。

==背景==
===設定===
本作背景設定在《[[騎馬與砍殺：戰團|騎馬與砍殺：戰團]]》的200年前，當時卡拉迪亞帝國正日漸衰敗，在前作中出現過的諸多王國雛形也逐漸崛起。卡拉迪亞帝國的衰亡類似於現實歷史上[[西羅馬帝國|西羅馬帝國]]的衰落。本作的武器、裝備與建築會參考西元600年到1100年這段歷史來設計。<ref>{{cite web|url = https://www.taleworlds.com/en/Games/Bannerlord/Blog/8|title = Bannerlord Development Blog Episode VII: Return of the Blog|date =2014-08-29|accessdate =2014-10-19 |website = Bannerlord Development Blog|publisher = TaleWorlds}}</ref>

===陣營===
本作將出現至少八個主要陣營，每個陣營由數個部族所組成，此外也會出現諸如雇傭兵、不法之徒等小型派系，這些陣營都擁有不同的競爭目標。<ref>{{cite web|url = https://www.taleworlds.com/en/Games/Bannerlord/Blog/8|title = Bannerlord Development Blog Episode VII: Return of the Blog|date =2014-08-29|accessdate =2014-10-19 |website = Bannerlord Development Blog|publisher = TaleWorlds}}</ref>

==開發==
2012年9月，[[TaleWorlds|TaleWorlds]]宣布遊戲正在開發中，同時釋出一部預告影片。<ref>{{cite news|author1=Tom Senior|title=Mount & Blade 2: Bannerlord announced with tiny teaser trailer|url=http://www.pcgamer.com/mount-blade-2-bannerlord-announced/|accessdate=2017-06-25|publisher=[[PC_Gamer|PC Gamer]]|date=2012-09-28}}</ref>

遊戲畫面相比前作《[[騎馬與砍殺：戰團|騎馬與砍殺：戰團]]》來說進步相當顯著，擁有更好的陰影效果與更精緻的模組。<ref>{{Cite web|url = https://www.taleworlds.com/en/Games/Bannerlord/Blog/2|title = Mount&Blade II: Bannerlord Developer Blog 1 - Hairy Artists|date =2013-11-22|accessdate =2014-10-19|website =Bannerlord Development Blog. |publisher =TaleWorlds}}</ref>在人物動作方面使用[[動作捕捉|動作捕捉]]技術來製作，<ref>{{cite web|url = https://www.taleworlds.com/en/Games/Bannerlord/Blog/6|title = Mount&Blade II: Bannerlord Developer Blog 5 – Virtual Skeletons|date =2014-03-14|accessdate =2014-10-19 |website = Bannerlord Development Blog|publisher = TaleWorlds}}</ref>人物臉部表情也會進行升級，讓人物情緒表現更加生動活潑。<ref>{{cite web|url = https://www.taleworlds.com/en/Games/Bannerlord/Blog/7|title = Mount&Blade II: Bannerlord Developer Blog 6 – Astounding Squirms|date =2014-05-07|accessdate =2014-10-19 |website = Bannerlord Development Blog|publisher = TaleWorlds}}</ref>

遊戲系統方面，將會有全新的物品介面以及更好的AI。<ref>{{cite web|url = https://www.taleworlds.com/en/Games/Bannerlord/Blog/4|title = Mount&Blade II: Bannerlord Developer Blog 3 – Unexpected Parties|date =2013-12-30|accessdate =2014-10-19 |website = Bannerlord Development Blog|publisher = TaleWorlds}}</ref>基於玩家們的回饋，攻城系統也會進行改善，在攻城時可以採取更多種策略。<ref>{{cite web|url = https://www.taleworlds.com/en/Games/Bannerlord/Blog/5|title = Mount&Blade II: Bannerlord Developer Blog 4 – Flexible Entries|date =2014-01-24|accessdate =2014-10-19 |website = Bannerlord Development Blog|publisher = TaleWorlds}}</ref>
