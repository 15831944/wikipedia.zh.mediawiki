{{日本作品|窈窕淑女|はいからさんが通る}}是日本[[漫畫家|漫畫家]][[大和和紀|大和和紀]]創作的[[少女漫畫|少女漫畫]]。1975年至1977年間，於[[講談社|講談社]]的週刊《{{link-ja|少女朋友|少女フレンド}}》上連載，並於1977年獲得第1屆[[講談社漫畫獎|講談社漫畫獎]]少女漫畫部門獎<ref>{{Cite web |url = https://gnn.gamer.com.tw/9/132589.html |title = 大和和紀經典少女漫畫作品《窈窕淑女》宣布推出劇場版動畫 |author = 巴哈姆特電玩資訊站 |date = 2016-06-01 |language = zh-tw |accessdate = 2018-04-12 |deadurl = no |archiveurl = https://web.archive.org/web/20180413233109/https://gnn.gamer.com.tw/9/132589.html |archivedate = 2018-04-13 }}</ref><ref>{{Cite web |url = http://www.hahnlibrary.net/comics/awards/kodansha.shtml |title = Kodansha Manga Awards |author = Joel Hahn |publisher = Comic Book Awards Almanac |language = en |accessdate = 2018-04-12 |archive-url = https://web.archive.org/web/20070816031310/http://www.hahnlibrary.net/comics/awards/kodansha.shtml |archivedate = 2007-08-16 |archiveurl = https://web.archive.org/web/20070816031310/http://www.hahnlibrary.net/comics/awards/kodansha.shtml |deadurl = yes }}</ref>。其後推出全八冊[[單行本|單行本]]以及全四冊文庫版。台灣的[[長鴻出版社|長鴻出版社]]和遠大文化公司出版中文版本。

在1978年6月3日至1979年3月31日期間，[[日本動畫公司|日本動畫公司]]改篇製作的電視動畫於朝日放送頻道播出（全42話）<ref>{{Cite web |url = https://natalie.mu/comic/news/194168 |title = TVアニメ「はいからさんが通る」、37年の時を経てBD/DVD BOX化 |author = natalie |date = 2016-07-11 |language = ja |accessdate = 2018-04-12 |deadurl = no |archiveurl = https://web.archive.org/web/20180413233101/https://natalie.mu/comic/news/194168 |archivedate = 2018-04-13 }}</ref>。原本製作方計劃動畫會播放大約1年，但其後因節目遭到[[腰斬_(術語)|腰斬]]而中止，因此在動畫完結時並沒有描述原作中的結局<ref>{{Cite web |url = http://bunshun.jp/articles/-/4866 |title = 原作者が語る、テレビ版『はいからさんが通る』が尻切れトンボで終わった理由 |author = 岸川真 |date = 2017-11-09 |publisher = 週刊文春 |language = ja |accessdate = 2018-04-12 |deadurl = no |archiveurl = https://web.archive.org/web/20180413233136/http://bunshun.jp/articles/-/4866 |archivedate = 2018-04-13 }}</ref>。

本作亦先後被改編為多部真人電視劇，例如在1979至1980年間，由[[寶塚歌劇團|寶塚歌劇團]]團員演出並且於[[關西電視台|關西電視台]]播出的真人版電視劇<ref name="宝塚ジャーナル">{{Cite web |url = http://takarazuka-j.blog.jp/archives/1887907.html |title = 少女漫画の金字塔の見事な舞台化で輝く、柚香光東上初主演作品！宝塚花組公演『はいからさんが通る』 |author = 宝塚ジャーナル |date = 2017-10-29 |language = ja |accessdate = 2018-04-12 |deadurl = no |archiveurl = https://web.archive.org/web/20180413233210/http://takarazuka-j.blog.jp/archives/1887907.html |archivedate = 2018-04-13 }}</ref>；1985年真人電視劇[[富士電視台|富士電視台]]版本；以及於2002年[[TBS電視台|TBS電視台]]播映之節目《{{link-ja|早安少女組。新春！Love故事|モーニング娘。新春! LOVEストーリーズ}}》內的真人電視劇。此外，在1987年12月12日上映了由[[東映|東映]]拍攝的真人電影版本。

2005年，朝日電視台進行了一項全國性調查，訪問不同年齡組別的日本人從而選出100部最受歡迎的動畫作品，結果本作排名95位<ref>{{Cite web |url = https://www.animenewsnetwork.com/news/2005-09-23/tv-asahi-top-100-anime-part-2 |title = TV Asahi Top 100 Anime Part 2 |author = Christopher Macdonald |date = 2005-09-24 |publisher = ANIME NEWS NETWORK |language = en |accessdate = 2018-04-13 |deadurl = no |archiveurl = https://web.archive.org/web/20171201131559/https://www.animenewsnetwork.com/news/2005-09-23/tv-asahi-top-100-anime-part-2 |archivedate = 2017-12-01 }}</ref>。

2017年，為記念「大和和紀畫業50週年」，講談社宣佈改編本作為劇場版，並分前篇和後篇上映。前篇於同年11月11日上映，而後篇將於2018年內上映<ref name="natalie@20171107">{{Cite web |url = https://natalie.mu/comic/news/255744 |title = 映画「はいからさん」前編、紅緒と少尉が偶然出会う本編映像が公開 |author = natalie |date = 2017-11-07 |language = ja |accessdate = 2018-04-12 |deadurl = no |archiveurl = https://web.archive.org/web/20180412212100/https://natalie.mu/comic/news/255744 |archivedate = 2018-04-12 }}</ref>。此外，寶塚歌劇團亦改編本作為音樂劇，在同年10月分別於[[梅田藝術劇場|梅田藝術劇場]]及[[日本青年館|日本青年館]]進行公演<ref name="宝塚">{{Cite web |url = http://kageki.hankyu.co.jp/revue/2017/haikarasan/index.html |title = ミュージカル浪漫『はいからさんが通る』 |author = 宝塚歌劇団公式ホームページ |language = ja |accessdate = 2018-04-12 |deadurl = no |archiveurl = https://web.archive.org/web/20171119001609/http://kageki.hankyu.co.jp/revue/2017/haikarasan/index.html |archivedate = 2017-11-19 }}</ref>。

== 故事簡介 ==
本作舞台是[[大正|大正]]時期，以[[大正民主|大正民主]]、[[西伯利亞干涉|西伯利亞干涉]]及[[關東大地震|關東大地震]]等事件為背景，描述男女主角兩人之間的戀愛故事。

花村紅緒是一位活潑好動少女，不擅於料理裁縫等家務，但擅長劍道。某天，紅緒被父親告知伊集院家的少爺伊集院忍是自己指腹為婚的未婚夫，雖然她極為反感，但也不得不住在伊集院家。

然而命運終究把紅緒與忍牽在了一起。溫柔善良的少尉伊集院忍，對紅緒一見鍾情，包容她的任性，亦不斷鼓勵她。可是，忍前往西伯利亞戰場抗擊俄國軍隊，其後更傳出死訊，令紅緒傷心不已。即使如此，紅緒還是相信著忍仍然活著。

為了守護無後的伊集院家及期望著忍的歸來，紅緒選擇成為一名職場女性，開拓起自己的人生之路。<ref name="natalie@20171107"/><ref name="宝塚"/>

== 電視劇 ==
=== 1979年版 ===
1979年4月7日至1980年8月30日期間，在[[寶塚歌劇團|寶塚歌劇團]]的連續音樂劇節目內以《寶塚浪漫電視·窈窕淑女》的名義於關西電視台播放，並由劇團團員演出<ref name="宝塚ジャーナル"/>。但由於肖像權及其他問題，電視台無法重播本節目及將其製成DVD。

; 製作人員
* 原作：大和和紀
* 演出、劇本：植田紳爾

; 演員
* 花村紅緒：{{link-ja|花鳥いつき}}
* 伊集院忍：{{link-ja|平みち}}
* 青江冬星：{{tsl|ja|剣幸|劍幸}}
* 北小路環：{{link-ja|美雪花代}}
* 鬼島軍曹：{{link-ja|日向薫}}
* 藤枝蘭丸：{{tsl|ja|瀬川佳英|瀬川佳英}}
* 拉里薩：{{lang|ja|遥くらら}}
* 故事旁白：{{lang|ja|キャシー}}

* 音樂：[[城之內美莎|城之內美莎]]
* 制作：富士電視台、泉放送制作

* 伊集院忍：{{link-ja|野口五郎}}

* 其他：久我美子、小鹿番、水野晴郎、笑福亭笑瓶、南州太郎、岡本麗、若村麻由美等

=== 2002年版 ===
2002年於[[TBS電視台|TBS電視台]]播映之節目《{{link-ja|早安少女組。新春！Love故事|モーニング娘。新春! LOVEストーリーズ}}》內的一個環節，演員幾乎為組合[[早安少女組|早安少女組]]的成員。

* 劇本：遠藤彩見
* 演出：星田良子
