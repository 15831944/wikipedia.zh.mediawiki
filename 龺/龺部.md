'''龺部'''，是為簡體字部首之一，歸於八劃部首，由於龺部是[[漢字簡化|漢字簡化]]後[[中國文字改革委員會|中國文字改革委員會]]和[[國家出版局|國家出版局]]於1983年發布的《[[汉字部首表|漢字統一部首表（草案）]]》中添加的新[[部首|部首]]，故此在《[[康熙字典|康熙字典]]》查無此[[部首|部首]]<ref>{{cite web|url=http://www.china-language.gov.cn/wenziguifan/scanning/gfsc/gfbz10.htm|title=汉字统一部首表（草案）|publisher=China-language.gov.cn|date=|accessdate=2013-03-26|deadurl=yes|archiveurl=https://web.archive.org/web/20130524144704/http://www.china-language.gov.cn/wenziguifan/scanning/gfsc/gfbz10.htm|archivedate=2013-05-24}}</ref><ref>[http://www.zdic.net/z/29/js/9FBA.htm]</ref>，但仍有提及「𠦝」此字<ref>[http://ctext.org/dictionary.pl?if=gb&char=%E5%80%9D 中國哲學書電子化計劃：倝]</ref>。因此在康熙字典中，含有「龺」這個部件的字都是歸類在其他部首。

== 部首單字解釋 ==

漢字部件。-{「幹」、「乾」}-字左旁。同「卓」字。

== 詳細說明 ==
“龺部”是不在康熙字典裡的部首，也不存在於正體中文的簡體中文部首。

根據《康熙字典》，這些字的大篆原形都是從「倝」的，例如：
* -{乾}-（倝 + 乙）<ref name="乾">[http://ctext.org/library.pl?if=gb&file=77415&page=84#25 康熙字典：乾]</ref> 、-{幹}-（倝 + 干）、-{榦}-（倝 + 木）、斡（倝 + 斗）、翰（倝 + 羽）、戟（倝 + 戈）
* 另有“朝”字，实际上并非从「倝」，在[[甲骨文|甲骨文]]字典<ref>《新編甲骨文字典》，刘兴隆，三九一页。</ref>中，“朝”的形象就是“日在艹(艸)中，月未落下”是[[会意字|会意字]]，《說文解字》<ref>[http://ctext.org/dictionary.pl?if=gb&id=30653 《說文解字》：𦩻：旦也。从倝，舟聲，「朝」本字。 ]</ref>和《康熙字典》<ref>[http://ctext.org/library.pl?if=gb&file=77415&page=1012#07 御定康熙字典 第1012頁 第7行]</ref><ref>[http://ctext.org/library.pl?if=gb&file=77415&page=506#03 御定康熙字典 第506頁 第3行]</ref>說“倝+舟𦩻”是“朝”的本字是錯誤的，實際上“倝+舟𦩻”是後世經過大篆訛變產生的。

不過由於有些字後來變成楷書時簡化了，失去了「倝」的特徵，所以只好另造「龺部」。
而在漢代的《[[說文解字|說文解字]]》也有說明。<ref>[http://ctext.org/shuo-wen-jie-zi/gan-bu2/zh 說文解字：倝部]</ref>

另外，「𠦝（卓）」字的起源與「倝」字事實上毫無關聯。「𠦝（卓）」為从匕从早，本作「㔬」；而「倝」最初由「旦」和「㫃」構成。

== 部首字元 ==
目前的繁體中文依循康熙字典將龺歸類在[[十部|十部]]，而簡體中文歸類在龺部<ref>{{cite web|url=http://202.115.72.58/refbook/R200708019.html?p=17|title=汉字标准字典|publisher=中国工具书网络出版总库|date=|accessdate=2013-03-26}}{{dead link|date=2018年5月 |bot=InternetArchiveBot |fix-attempted=yes }}</ref>。龺的編碼是U+9FBA，在繁體中文中為[[卓|卓]]之[[異體字|異體字]]<ref>{{cite web|url=http://140.111.1.40/yitia/fra/fra00439.htm |title=A00439 【卓】 |publisher=中華民國教育部 |date= |accessdate=2013-03-26}}</ref>歸十部[[六|六]]劃。

== 簡繁差異 ==
而在簡體中文中歸於其他“龺部”的字，在繁體中文中則以「龺」字旁地另外一邊的偏旁或其一部分為部首，例如：-{[[乾|乾]]}-，簡體中文為“龺部”，而繁體中文歸為「[[乙部|乙部]]」<ref name="乾" />；另一例是韓，簡體中文歸為“龺部”，但在繁體中文歸為「[[韋部|韋部]]」。

但有些時候簡體中文為了回應[[古文|古文]]需求，因此有些簡體中文的資料或辭典會取消“龺部”改為原來的部首<ref>{{cite web|url=http://www.zdic.net/zd/zi/ZdicE6Zdic88Zdic9F.htm |title=漢典(簡體中文)將戟歸於戈部 |publisher=Zdic.net |date= |accessdate=2013-03-26}}</ref><ref>{{cite web|url=http://www.zdic.net/zd/zi/ZdicE4ZdicB9ZdicBE.htm |title=漢典(簡體中文)將乾歸於乙部 |publisher=Zdic.net |date= |accessdate=2013-03-26}}</ref>，使其與康熙字典保持一致，不過大部分[[中國大陸|中國大陸]]出版的簡體字[[字典|字典]]都會採用“龺部”，另外，有些字典會將一字歸於多部。
