'''奧托·恩斯特·林德曼'''（[[德語|德語]]：'''{{lang|de|Otto Ernst Lindemann}}'''，{{bd|1894年|3月28日|1941年|5月27日}}），[[納粹德國|納粹德國]][[海軍|海軍]]將領，服役於[[第一次世界大戰|第一次世界大戰]]和[[第二次世界大戰|第二次世界大戰]]。

== 早年生活與加入海軍 ==
1894年3月28日出生於[[萊茵省|萊茵省]][[韦斯特林山|韦斯特林山]]的[[阿爾滕基興|阿爾滕基興]]地区，是[[法官|法官]]格奧爾格·海因里希·恩斯特·林德曼的第三個兒子。1913年3月26日，林德曼帶著他的父母到[[弗倫斯堡|弗倫斯堡]]，以雄厚的財力和家庭背景，加入了德國的帝國海軍。

== 第一次世界大戰 ==
[[第一次世界大戰|第一次世界大戰]]時，林德曼服役於[[公海艦隊|公海艦隊]]，職位為[[少尉|少尉]]，林德曼在1915年6月1日進入弗倫斯堡的無線電報學校學習其中的課程，並早在1915年7月返回，1915年9月18日晉升[[中尉|中尉]]，進行阿爾比恩行動。
一次大戰後又參加[[西班牙內戰|西班牙內戰]]，回國後又晉升中校。

== 第二次世界大戰與俾斯麥號 ==
林德曼在服役期間並沒有和敵人接戰的機會，直到1941年被認命為[[俾斯麥號戰艦|俾斯麥號]]的艦長，和[[司令|司令]][[君特·吕特晏斯|君特·吕特晏斯]]同行，5月19日凌晨，和[[歐根親王號重巡洋艦|歐根親王號]]橫渡[[波羅的海|波羅的海]]進入[[北大西洋|北大西洋]]，5月24日，成功在[[丹麥海峽海戰|丹麥海峽海戰]]中擊沉英國軍艦[[胡德號戰鬥巡洋艦|胡德號]]，重創[[威爾斯親王號戰艦|威爾斯親王號]]，震驚英國海軍。5月27日，遭到英國艦隊[[喬治五世號|喬治五世號]]與羅德尼號的圍攻而戰死，在當天10:36和俾斯麥號一起沉入北大西洋之下。

1941年12月27日，林德曼死後得到了[[騎士鐵十字勛章|騎士鐵十字勛章]]，因為他的成熟的領導能力，使得英國海軍最強大的胡德號都遭到破壞，頒發勛章以此作為英勇奮戰的代表人物。

== 俾斯麥號和林德曼死亡之謎 ==
在1989年根據[[羅伯·巴拉德|羅伯·巴拉德]]調查俾斯麥號的沉沒原因時，發現大多數炮火在上層建築炸開。 有50多個殼孔在指揮塔的周圍地區。很有可能推測身負重傷林德曼緊靠在一旗桅杆，從此和俾斯麥號消失在北大西洋。
