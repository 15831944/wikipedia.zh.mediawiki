'''恩慕寺'''，位于[[北京市|北京市]][[海淀区|海淀区]][[颐和园路|颐和园路]]西侧，是[[清朝|清朝]][[畅春园|畅春园]]内的皇家佛寺，与[[恩佑寺|恩佑寺]]并列。

==简介==
恩慕寺位于[[北京大学|北京大学]]西门外，颐和园路西侧。[[清朝|清朝]][[乾隆|乾隆]]四十二年（1777年），[[乾隆帝|乾隆帝]]之母[[孝聖憲皇后|孝圣皇太后]]病逝。乾隆帝为圣母皇太后“广资慈福”，在恩佑寺南侧创建恩慕寺。“恩慕寺”一名取自恩佑寺和[[永慕寺|永慕寺]]（位于[[南苑|南苑]]，是[[康熙帝|康熙帝]]为母亲礼佛而建）。恩慕寺坐西朝东，院落两进，面临大道（即今颐和园路）。《[[日下旧闻考|日下旧闻考]]》记载，山门额题“敬建恩慕寺”，二层山门额题“慈云广荫”，均为乾隆帝御书。山门内正殿面阔五间，供奉[[药师佛|药师佛]]一尊，左右供奉药师佛108尊。正殿额题“福应天人”，殿内额题“慧雨仁风”，两侧对联是：“慈福遍人天，祥开佛日；圣思留法宝，妙现心灯。”均为乾隆帝御书。南配殿面阔三间，供奉[[弥勒佛|弥勒佛]]。北配殿面阔三间，供奉[[观音|观音]]，左右分立石幢，石幢中一座刻有全部《[[药师经|药师经]]》，另一座刻有乾隆帝《御制恩慕寺瞻礼》诗：“尊养畅春历廿冬，欲求温清更何从？天惟高矣地惟厚，慕述祖兮思述宗。”<ref name=beihou>{{Cite web |url =http://www.thepaper.cn/newsDetail_forward_1699993  |title = 北大西门的两座古建筑——恩佑寺与恩慕寺残迹背后的故事 |author =  |date = 2017-06-07  |publisher = 澎湃新闻 |language =  |accessdate =  |quote =  }}</ref>

乾隆四十二年五月（1777年）至乾隆四十四年二月（1779年）[[清明节|清明节]]，乾隆帝仅在恩慕寺行礼。其后直至[[嘉庆|嘉庆]]三年（1798）十月太上皇乾隆帝病重期间，乾隆帝大多是恩慕寺和恩佑寺一并行礼。[[嘉庆帝|嘉庆帝]]曾34次到恩慕寺和恩佑寺行礼拈香。[[道光|道光]]三年（1823年）至道光十九年（1839年）每年正月的[[孝圣宪皇后|孝圣宪皇后]]忌辰，[[道光帝|道光帝]]都到恩慕寺和恩佑寺行礼，共计17次。[[咸丰|咸丰]]七年（1857）之前，[[咸丰帝|咸丰帝]]共4次到恩慕寺和恩佑寺行礼，咸丰朝最末三年咸丰帝未行礼<ref name=beihou/>。

咸丰十年（1860），[[英法联军|英法联军]]火烧[[三山五园|三山五园]]，恩佑寺、恩慕寺也被英法联军焚毁。[[内务府|内务府]]大臣[[明善|明善]]在咸丰十年九月二十九日的奏折中说：“（[[圆明园|圆明园]]）大宫门、大东门，以及大宫门外东西朝房、六部朝房……恩慕寺、恩佑寺、清溪书屋……等处均被焚烧。”<ref name=beihou/>

此后，恩佑寺和恩慕寺各仅存一座山门，两座山门南北距离50米，北为恩佑寺山门，南为恩慕寺山门。1981年，“恩慕寺山门”被定为[[北京市海淀区文物保护单位|海淀区重点文物保护单位]]，并于1985年修缮<ref name=beihou/><ref name=meili>{{Cite web |url = http://www.bjhd.gov.cn/mlhd2016/hdww/201611/t20161110_1317328.html |title =  恩慕寺山门|author =  |date =2016-10-30  |publisher = 美丽海淀 |language =  |accessdate =  |quote =  }}</ref>。
