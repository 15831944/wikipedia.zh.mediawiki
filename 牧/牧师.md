'''牧師'''是指[[基督教|基督教]]的[[教會|團體]]中，專職負責帶領及照顧其他[[基督徒|基督徒]]的[[神职人员|神职人员]]。[[聖經|聖經]]原文的用字就是[[牧羊人|牧羊人]]之意。雖然[[天主教會|天主教會]]也有類似的職位，但「牧師」一词在中文语境中特指[[新教|新教]]的基層神職人員。

== 詞源 ==
牧師的一詞源於[[拉丁語|拉丁語]]的{{lang|la|''pastor''}}，而[[希臘文|希臘文]]為{{Lang|el|ποιμην}}，與《[[新約聖經|新約聖經]]·[[以弗所書|以弗所書]]》4:11有關。《[[聖經和合本|聖經和合本]]》譯「他所賜的有使徒、有先知、有傳福音的、有'''牧師'''和[[教師|教師]]」，但此經文在其他[[聖經譯本|聖經譯本]]卻有不同譯法，如牧者（《[[聖經和合本修訂版|聖經和合本修訂版]]》）、牧人（《[[聖經_(呂振中譯本)|呂振中譯本]]》和《[[聖經恢復本|聖經恢復本]]》）、司牧（《[[聖經_(思高譯本)|思高譯本]]》）、牧養（《[[聖經新譯本|聖經新譯本]]》）。

== 歷史演化 ==
許多學者認為，基督宗教在第一世紀後受到周圍不同地方的文化和習俗的影響，逐漸地將[[長老|長老]]和[[監督|監督]]的職務自一般信徒分離出來，然後將'''監督'''提升到高於長老的地位，稱之為[[主教|主教]](Bishop)，而將長老置在主教之下，在英文稱為祭司（priest），在中文稱為“[[司鐸|司鐸]]”（神父）。在這個時候開始，一位主教可以監督很多城或[[教區|教區]]的諸多[[祭司|司鐸]]、[[教會|教會]]、信徒。[[宗教改革|宗教改革]]後，[[加爾文|加爾文]]、[[慈運理|慈運理]]和其他改革者以牧師（pastor）取代[[天主教|天主教]]的[[神父|神父]]（priest）。不同的是：新教教會由牧師和獨立的長老管理，牧師並可以結婚，其妻子在某些教會被稱為「'''牧師娘'''」、「'''先生媽'''」是因為受到[[臺灣閩南語|臺灣閩南語]]的影響，亦有「師母」等源自中國傳統文化師父師母的稱呼。

== 職位概況 ==
===新教===
在新教中，特别是在[[浸信会|浸信会]]中，认为“牧师”（pastor，以弗所书4:11）、“'''长老'''”（elder，使徒行传20:17）、“'''监督'''”（bishop或overseer，提摩太前书3:1）指的是同一个职位，牧师强调的是此职位的牧养工作，长老强调的是这人在属灵上是成熟的（不一定是年龄很老的），监督强调的是他的治理教会的工作。保罗在[[提摩太前书|提摩太前书]]第三章规定了监督的条件，在[[提多书|提多书]]第一章制定了长老的条件，比较一下发现两者是基本一致的；而且保罗在提多书第一章里交替使用长老和监督（1:5,7）。此外，保罗在使徒行传20章见到了以弗所的长老，嘱咐他们牧养神的教会，这也是牧师的工作。<ref>Jeff Brown: Form and Freedom, What the New Testament Teaches about Church Government and Church Leadership, 2004, ISBN 3-937965-06-8.</ref>

由二十世紀開始，[[女性|女性]]亦可以成為牧師。然而，在某些教會中（如[[浸信會|浸信會]]），女性擔任牧師還不普遍；在另一些教會（如[[信義會|信義會]]），女性出任牧師是很普遍的。在圣公会的[[三級聖品制|三級聖品制]]裡，牧師上一級是[[主教|主教]]，低一級是[[會吏|會吏]]。而在[[会众制|会众制]]的新教教會中，[[主任牧師|主任牧師]]則成了一間教會裡最高的決策者。

[[File:Priest_of_Woman.jpg|thumb]]
[[英國|英國]]或其他國家的[[國教|國教]]教會的牧師，尤是[[聖公會|聖公會]]和[[衛理會|衛理會]]的牧師，英文稱之為Minister，中文譯「會長」；如果是指非國教教會的牧師，尤是[[美國|美國]]的教會，英文稱之為Pastor；頭銜為Reverend。

一些[[新教|新教]]教徒（例如[[召會|召會]]）認為，牧師的稱謂違背了「[[信徒皆祭司|信徒皆祭司]]」的教義，因此拒絕使用這個詞和牧師聘用制度，其中包括[[門諾會|門諾會]]和一些受[[弟兄運動|弟兄運動]]影響的教會。

===天主教會===
天主教會中的基层司牧一职，稱為「'''本堂神父'''」或「'''主任司鐸'''」，由具有神品的[[神父|神父]]擔任，主要工作為管理[[堂區|堂區]]并牧养堂区内的羊群。在英语中稱為「parish priest」。

==圣经对牧师之资格的要求==
*[[提摩太前书|提摩太前书]]第三章说：“作监督的，必须无可指责，只作一个妇人的丈夫，有节制，自守，端正，乐意接待远人，善於教导；不因酒滋事，不打人，只要温和，不争竞，不贪财；好好管理自己的家，使儿女凡事端庄顺服（或作：端端庄庄的使儿女顺服）。人若不知道管理自己的家，焉能照管神的教会呢？初入教的不可作监督，恐怕他自高自大，就落在魔鬼所受的刑罚里。监督也必须在教外有好名声，恐怕被人毁谤，落在魔鬼的网罗里。 
*[[提多书|提多书]]第一章说：”若有无可指责的人，只作一个妇人的丈夫，儿女也是信主的，没有人告他们是放荡不服约束的，就可以设立。 监督既是神的管家，必须无可指责，不任性，不暴躁，不因酒滋事，不打人，不贪无义之财。乐意接待远人，好善，庄重，公平，圣洁自持。坚守所教真实的道理，就能将纯正的教训劝化人，又能把争辩的人驳倒了。 ”
== 某些教会对牧师之资格的要求 ==
{{Expert|time=2015-12-14T03:45:57+00:00}}
牧師的資格一般最少要有一個[[神學|神學]]學士學位。{{fact|但亦有部份教會會按立具資深牧養經驗但沒有神學學位的長執或領袖為牧師，而將被按立的候選人一般須由資深牧師或教會的長執會推薦，再經由幾名資深牧師組成的評審委員會評審後，才會正式被按立為牧師。。|time=2009-05-05}}

一般來說，每一間新教教會最少有一個牧師。在中国大陆，情况比较特殊，有些教会没有牧师，而只有一个或几个经验丰富的信徒组织其他信徒活动，被称为“带领人”。而另有些缺少牧师的教会就會由[[傳道人|傳道人]]帶領教會；如果連傳道人也沒有的話，那就由不受薪的幾個執事/[[長老|長老]]共同管理。這些情況常見於一些信徒不多的小型教會，在有些基督徒看来是教会发展不成熟的表现，也不太符合圣经的要求，因为将长老、传道人、执事的定义与职责搞得混乱不清。

== 薪資 ==

基督教萌芽初期，傳道人的生活費用來源主要有：
# 會友奉獻
# 母會或其他堂會的奉獻（羅馬書15：25-27）
# 傳道人兼職（例如保羅以織帳棚為業<ref>{{Cite web |url = http://www.pastor.org.tw/ArticleInfo/Detail.aspx?u=178 |title = 織個帳篷好下網 |author = 夏忠監、夏昊霝 }}</ref>）

在台灣，牧師薪資通常是交由教會資深會友組成的長執會決定；小規模的教會（例如50人以內）的牧師，教會日常運作及薪水即可由宣教機構或母會承擔，但如果是不屬於任何堂會或宣教機構的獨立教會，薪水通常是由牧師/傳道人自籌（如仰賴會友稀少的奉獻度日、仰賴配偶供應或在外另有兼職謀生）。在教會，牧師薪資通常是個敏感的議題，因為部分人士誤以為可以透過薪資來控制牧師，或以為牧師工時很少，不需要撥付相當的工資。<ref>{{cite news |author = 湯姆·雷納（Dr.Thom Rainer） |url = http://chinese.christianpost.com/article/%E7%89%A7%E5%B8%AB%E8%96%AA%E6%B0%B4%E6%88%90%E7%82%BA%E6%95%8F%E6%84%9F%E8%A9%B1%E9%A1%8C%E7%9A%847%E5%80%8B%E5%8E%9F%E5%9B%A0-17622/ |title = 牧師薪水成為敏感話題的7個原因 |publisher = 基督郵報  |date = 2014年7月2日}}</ref>

== 其他解釋 ==
現今許多[[線上遊戲|線上遊戲]]及文學作品中，常出現「牧師」一職業，如[[魔獸世界|魔獸世界]]。通常線上遊戲中的牧師是擔任治療者的角色，藉由神聖的咒語和[[魔法|魔法]]達到補血、驅魔、復活等功能。相較於其他職業多半是為敵方帶來損傷，牧師則是為友方提供增益、治療，因此在一個團隊裡是不可或缺的角色。
