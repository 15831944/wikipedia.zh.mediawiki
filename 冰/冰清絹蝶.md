'''冰清絹蝶'''（學名：''Parnassius glacialis''）是屬於[[絹蝶亞科|絹蝶亞科]][[絹蝶族|絹蝶族]]的一種[[蝴蝶|蝴蝶]]，是[[絹蝶屬|絹蝶屬]]的一種。共有5個亞種。分佈於中國東部、朝鮮半島及日本一帶。被列入中國國家林業局令[[國家保護的有益的或者有重要經濟、科學研究價值的陸生野生動物名錄|國家保護的有益的或者有重要經濟、科學研究價值的陸生野生動物名錄]]。幼蟲[[寄主|寄主]]為馬兜鈴科[[馬兜鈴屬|馬兜鈴屬]]及罌粟科[[紫堇屬|紫堇屬]]植物<ref name=mrs>Chou, 1994. Monografia Rhopalocerorum Sinensium 1: 199</ref>。

== 語源 ==
學名[[種名|種名]]源自[[拉丁語|拉丁語]] glacialis，意為冰凍，中文蝶名也以此命名<ref name=mrs/>。

== 分佈 ==
中國（陝西、黑龍江、吉林、遼寧、河南、山東 、山西、甘肅、貴州、雲南、浙江、安徽）、日本、朝鮮<ref name=chun>許家珠、魏煥志、賴平芳. 秦嶺巴山蝴蝶圖記：76. 中國: 陝西科學技術出版社. 2010年6月.</ref>。

== 形態 ==
體黑色。前後翅白色，半透明，翅脈黑色。前翅外緣、亞外緣可見淺灰色橫帶，中室端、中室內各有1個灰色橫斑。後翅後緣從基角至臀角有1條較寬的黑帶，並附黑色絨毛。翅反面似正面，但後翅基部下腹部和前胸板有黃褐色絨毛，腹部背面黑褐色，附灰色絨毛<ref name=chun/>。

== 習性 ==
成蟲5-7月出現，多分佈在的林緣、山地和溝坎，飛翔較緩慢。幼蟲的[[寄主|寄主]]為馬兜鈴科[[馬兜鈴屬|馬兜鈴屬]]及罌粟科[[紫堇屬|紫堇屬]]植物<ref name=mrs/>，物種包括：
* [[馬兜鈴|馬兜鈴]] ''Aristolochia debilis''
* [[東北延胡索|東北延胡索]] ''Corydalis ambigua''
* [[刻葉紫堇|刻葉紫堇]] ''Corydalis incisa''
* [[夏天無|夏天無]] ''Corydalis decumbens''
* ''Corydalis remota''
