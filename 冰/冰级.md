'''冰级'''是指[[船级社|船级社]]或国家相关机构对船舶在各种[[海冰|海冰]]条件下航行能力的分类。 

长期以来存在多种冰级标准。<ref>{{cite web|url=http://www.engr.mun.ca/~cdaley/8074/Ice%20Class%20Rules_CD.pdf|title=Ice Class Rules: Description and Comparison|last=Daley|first=Claude|date=2014-04-01|format=PDF}}</ref><ref>{{cite web | url = http://www.codanmarine.com/repository/com/Files/Seminar/Ice%20Navigation/Ice%20classes%20in%20brief.pdf | format = PDF | title = Ice Classes in Brief | first1 = Håvard | last1 = Nyseth | first2 = Karsten | last2 = Bertelsen | publisher = DNV GL | date = 2014 }}</ref><ref>* Eyres, David J. Ship Construction. 1st ed. Elsevier, 2007. 36-39. </ref>
