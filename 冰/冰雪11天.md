《'''冰雪11天'''》（{{lang-en|'''The Next 11 Days'''}}）中国大陆灾难片，导演[[陈国星|陈国星]]、王小列，主演冯远征、刘桦。该片融合诸多影视明星，如：冯远征、刘桦、黄奕、郭家铭、梁丹妮、奚美娟、柳岩、巩汉林、侯勇它也是向[[中国共产党|中国共产党]]“[[中共十八大|中共十八大]]”献礼的电影，曾在[[人民大会堂|人民大会堂]]首映。<ref name=" fish" >{{cite web |url=http://v.ifeng.com/ent/movie/201211/e0401141-d9a5-4547-8e23-bbca673d2ae9.shtml |title= 纪实电影《冰雪11天》：众志成城化解危机 |accessdate=2012 |publisher=凤凰网 |work= |language=zh-hans}}</ref><ref name="tiger" >{{cite web |url=http://ent.sina.com.cn/v/c/2012-11-02/12593778495.shtml |title=冯远征领衔《冰雪11天》 献礼十八大 |accessdate=2012 |publisher=新浪网 |work= |language=zh-hans}}</ref>

==剧情==
该片根据[[2008年中国雪灾|2008年中国雪灾]][[广州火车站|广州火车站]]疏散数十万返乡群众的真实事件改编，热情歌颂了中国共产党领导的[[中华人民共和国中央人民政府|中华人民共和国各级政府]]、各级政府部门、[[中国人民解放军|中国人民解放军]]、[[公安民警|公安民警]]、[[武警官兵|武警官兵]]、[[民兵预备役人员|民兵预备役人员]]在抵抗雪灾、保卫人民群众生产生活安全方面付出的努力和牺牲。<ref name="cat" >{{cite web |url=http://henan.sina.com.cn/ent/mov/2012-11-06/235-22361.html |title=《冰雪11天》简介 |accessdate=2012 |publisher=新浪网 |work= |language=zh-hans}}</ref><ref name="tiger" />
