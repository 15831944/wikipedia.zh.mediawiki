'''屈原墓'''分布于[[汨罗江|汨罗江]]北岸的[[汨罗山|汨罗山]]，山上原有十二座大的封土堆，修[[京广铁路|京广复线]]时掩埋一座，是为'''屈原十二疑冢'''。现存十一座墓中以烈女岭11号墓和冲里赵4 号墓为最大，4号墓前立有[[清|清]][[同治|同治]]六年（1867年）刊刻的墓碑，文“故楚三闾大夫之墓”；其余墓碑较小，立于清[[光绪|光绪]]二十八年（1902年），文“三闾大夫之墓”。1956年公布为[[湖南省重点文物保护单位|湖南省重点文物保护单位]]。<ref>{{cite web |title=屈原墓 |url=http://www.miluo.gov.cn/25214/25218/28529/content_650438.html |website=汨罗市人民政府 |accessdate=2018-07-07 |date=2016-12-01}}</ref>1983年汨罗山考古发掘大小古墓101座，年代从[[东周|东周]]、[[秦朝|秦]]、[[西汉|西汉]]至[[南朝_(中国朝代)|南朝]]<ref>{{cite journal |title=汨罗县东周、秦、西汉、南朝墓发掘报告 |journal=《湖南考古辑刊》 |date=1986年6月 |volume=第3辑 }}</ref>，为湖湘楚墓的典型。

== 十二疑冢传说 ==
因年代久远，屈原墓确切位置已经失传<ref name="n1">{{cite journal |author1=刘石林 |title=屈原墓位置何在 |journal=《求索》 |year=1987 |issue=3 |pages=121-123}}</ref>，是否归葬在这十二座墓中亦有争议<ref>{{cite journal |author1=张中一 |title=汨罗山屈原墓辨析 |journal=《求索》 |year=1982 |issue=6 |pages=124-126 }}</ref><ref>{{cite journal |author1=张伦笃，黄靖中 |title=对屈原墓争论的思考 |journal=《云梦学刊》 |year=1990 |issue=1 |pages=10-12 }}</ref><ref>{{cite journal |author1=高正 |title=论屈原与郭店楚墓竹书的关系 |journal=《新华文摘》 |year=2000 |issue=3 |pages=68-69 }}</ref>，因墓葬位置不可考和屈原形象不断升华而产生了屈原十二疑冢的传说<ref name="n1" />。

传[[屈原|屈原]]在汨罗江投江殉国，遗体打捞上来后，一边脸已被恶鱼吃掉一半，其女[[女嬃|女嬃]]配上半边金脸下葬。为防止盗墓和贵族破坏，经协助一晚筑成十二座疑冢<ref>{{cite web |title=屈原墓 |url=http://www.yysqw.gov.cn/43343/43344/44580/content_1269516.html |accessdate=2018-07-07}}</ref>。
