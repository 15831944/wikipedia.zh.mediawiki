《'''回憶中的瑪妮'''》（{{lang-en|When Marnie Was There}}）是由[[英國|英國]]女性作家[[瓊安·G·羅賓森|瓊安·G·羅賓森]]在1967年發表的[[兒童文學|兒童文學]]類型小說。<ref>{{Cite web |url=https://www.goodreads.com/book/show/22844030-when-marnie-was-there |title=When Marnie Was There by Joan G. Robinson |work=Good Reads |accessdate=2014年10月4日 }}</ref>

== 劇情概要 ==
性格不開放，沒有親密朋友的女孩安娜（Anna）小時在父母死去後，便交由普瑞斯頓夫人（Mrs Preston）扶養著。<ref name="kadokawa">{{Cite web |url=http://www.tsubasabunko.jp/special/sp1407-b.php |script-title=ja:新訳 思い出のマーニー |publisher=[[角川書店|角川書店]] |accessdate=2014年10月4日 }}</ref>

某日為了讓健康欠佳的安娜調養身體，普瑞斯頓便委推好友佩格（Pegg）夫婦讓安娜前往他們那靠近海外的住所暫居。來到新環境的安娜有一天在一棟舊房子内見到一名自稱是瑪妮（Marnie）的神秘少女，隨後安娜每日私下常偷去見瑪妮，成為了親密朋友，而安娜與瑪妮之間的來往也成為安娜周遭的人所不知的祕密。

之後在刮強風大雨的某一天，安娜與瑪妮在一棟廢棄的風車時，瑪妮突然不知為何離開了安娜不見蹤影。安娜事後雖對瑪妮在風車裡自行不告而別一事感到憤怒，但仍與瑪妮達成和解，但在和解時瑪妮卻沒來由地消失在安娜的眼前裡。<ref name="kadokawa"/>

== 作品背景 ==
此故事來源是瓊安·G·羅賓森在[[諾福克郡|諾福克郡]]北邊沿海的[[民政教區|民政教區]][[伯納姆·奥弗里|伯納姆·奥弗里]]裡，在一棟穀倉樓上看見窗內有人正對一名金髮女孩梳頭的場景而引發靈感。<ref name="Page226">{{cite book |title=回憶中的瑪妮：後記 |date=2014年11月1日 |publisher=[[台灣東販|台灣東販]] |isbn=978-986-331-549-0 |page=226頁 }}</ref>而瓊安·G·羅賓森自身的小時經歷也反應改編在故事主角安娜身上。<ref name="Page226"/>

起初作品是計畫命名為《瑪妮》（Marnie），但完稿時恰巧與電影導演[[亞佛烈德·希區考克|亞佛烈德·希區考克]]同期間上映的影片《豔賊》（原文同為「Marnie」）撞名而做了更改。<ref name="Page226"/>

== 反應．列名項目 ==
作品於英國首次出版時，有達到25萬本以上的銷售成績，<ref>{{cite video |date= 2014年8月10日 |script-title= ja:ZIP！スピンオフ ～ジブリ最新作「思い出のマーニー」秘密を探るイギリス絶景旅～ |publisher=[[日本電視台|日本電視台]]}}</ref>之後亦被[[卡內基獎章|卡內基獎章]]列入為1967年期間的提名優秀作品之一。<ref>{{Cite web |url=http://www.carolinesheldon.co.uk/pdf/dbpdf.php?dbID=96 |title=joan G. Robinson Agent:Caroline Sheldon |work=Caroline Sheldon |format=[[pdf|pdf]] |page=第1頁 |accessdate=2014年10月4日 |deadurl=yes |archiveurl=https://web.archive.org/web/20141006100943/http://www.carolinesheldon.co.uk/pdf/dbpdf.php?dbID=96 |archivedate=2014年10月6日 }}</ref>多年後日本動畫導演[[宮崎駿|宮崎駿]]在2011年的個人著作{{nihongo|《走向閱讀》|本へのとびら}}裡將《回憶中的瑪妮》納為值得向大眾推薦的50份文學作品名單中。<ref>{{Cite web |url=http://www.iwanami.co.jp/hensyu/sin/sin_kkn/kkn1110/sin_k614.html |script-title=ja:本へのとびら―岩波少年文庫を語る |publisher=[[岩波書店|岩波書店]] |accessdate=2014年10月4日 }}</ref><ref>{{cite book |script-title=ja:思い出のマーニー ビジュアルガイド  |date=2014年7月19日 |publisher=[[角川書店|角川書店]] |isbn=978-4-04-101978-8 |page=79頁 |accessdate=2014年10月4日 }}</ref>
