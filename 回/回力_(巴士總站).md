'''回力'''（{{lang-pt|'''JAI ALAI'''}}、{{lang-en|'''Jai Alai'''}}），前稱'''出入境事務廳總站'''，是位於[[澳門|澳門]][[澳門半島|澳門半島]]海港街的一個終點站。[[澳巴|澳巴]]'''[[澳門巴士8路線|8]]'''路線和[[澳門新福利公共汽車有限公司|新福利]]'''[[澳門巴士28C路線|28C]]'''路線以該站為終點站，[[澳巴|澳巴]]'''[[澳門巴士10B路線|10B]]'''路線途經該站。

== 沿革 ==

== 歷史 ==
[[File:Jai_Alai_Terminal_02.jpg|thumb]]
*1993年前，此站名為港澳碼頭總站，直至1993年位於海港前地的新[[外港碼頭|外港碼頭]]落成，本站附近所在的[[外港碼頭|外港碼頭]]改建為[[治安警察局|治安警察局]]出入境事務廳大樓。
*1993年港澳碼頭搬走後站名分別改為出入境事務廳及友誼巷，而巴士車頭顯示為回力場或回力塲；
*2001年4月25日，[[澳門巴士8路線|8路線]]改以本站為總站。
*2011年2月27日，因應[[民政總署_(澳門)|民政總署]]開展[[高美士街|高美士街]]至[[友誼大馬路|友誼大馬路]]渠網改善工程，所有路線不再停靠友誼巷站，改為停靠設於[[澳門旅遊活動中心|旅遊活動中心]]門前的臨時站（即今「[[高美士街|高美士]]/馬六甲街停車場」站前身），該站遂成為8路線的總站。
*2011年8月起，隨着出入境事務廳遷往北安新大樓辦公，站名改為'''回力'''。
*2016年12月10日起，[[澳門巴士10B路線|10B]]路線停靠本站。

== 設施 ==
