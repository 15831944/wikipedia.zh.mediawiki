{{日本作品|回吻|リプキス}}是[[戲畫_(遊戲品牌)|戲畫]]於2016年3月25日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]<ref>{{cite web|url=http://moedigi.com/game/2016/03/giga-repkiss-master-up/|title=戯画『リプキス』がマスターアップ。等身大の恋愛模様を描くキスシリーズ最新作！|website=moedigi|date=2016-03-04|accessdate=2019-04-13|language=ja}}</ref>。2017年4月13日由Entergram發售[[PlayStation_4|PlayStation 4]]和[[PlayStation_Vita|PlayStation Vita]]遊戲版本<ref>{{cite web|url=https://www.gamer.ne.jp/news/201612150103/|title=イチャラブコミュニケーションADV「リプキス」がPS4/PS Vitaで2017年4月13日に発売！妹・まゆりのオリジナルエピソードが追加 |website=Gamer|date=2016-12-15|accessdate=2019-04-13|language=ja}}</ref>。2018年12月20日發售[[任天堂Switch|任天堂Switch]]遊戲版本<ref>{{cite web|url=https://www.4gamer.net/games/441/G044183/20181122203/|title=恋愛ADV「リプキス」，Switch版が12月20日に発売決定。体験版の配信を本日開始|website=4Gamer|date=2018-11-22|accessdate=2019-04-13|language=ja}}</ref>。

== 故事簡介 ==
立石總司因為父母工作的關係和妹妹搬回了以前曾居住的光葉台，當總司整理行李告一段落去便利商店買東西時，和青梅竹馬五之谷奏撫再次相遇。隔天上學時，除了和奏撫同班外，也有了新的相遇<ref>{{cite web|url=https://www.tgsmart.jp/article.aspx?a=10838|title=【TG SCOOP】戯画の『キス』シリーズ最新作『リプキス』はヒロインたちと淡い恋を成熟させるイチャラブストーリー！|website=TG Smart|date=2015-11-21|accessdate=2019-04-13|language=ja}}</ref>。

== 登場角色 ==
; {{nihongo|立石總司|立石 総司（たていし そうじ）}}
: 作品男主角，光葉台學園二年級生。以前因為父母用接吻代替打招呼，所以對青梅竹馬五之谷奏撫和羽耶音姐妹也採取同樣的做法。
; {{nihongo|五之谷奏撫|五ヶ谷 奏撫（いつがや かなで）|聲：東條美和子}}
: 總司的青梅竹馬與同班同學，很擅於照顧人。小時候便喜歡總司，但知道總司喜歡姐姐羽耶音而不敢透露自己的心事。
; {{nihongo|五之谷羽耶音|五ヶ谷 羽耶音（いつがや はやね）|聲：{{lang|ja|あじ秋刀魚}}}}
: 總司的青梅竹馬與學姐，網球社社員，總司的初戀對象。不擅於作家事，但製作可麗餅的手腕相當高明。
; {{nihongo|雙葉咲希|双葉 咲希（ふたば さき）|聲：{{lang|ja|てんぐまい}}}}
: 總司的同班同學與班長，網球社的社長，擅於與人交流。成績優秀且運動萬能的優等生。跟唯梨是沒有血緣關係的姐妹，起初和唯梨相見時不知道怎麼和她相處，之後唯梨因為看到自己優秀的一面後感到開心，便一直努力回應唯梨的期待。實際上有缺乏幹勁的一面。
; {{nihongo|雙葉唯梨|双葉 唯梨（ふたば ゆいり）|聲：{{lang|ja|萌花ちょこ}}}}
: 總司的學妹，跟咲希是沒有血緣關係的姐妹。總司搬離前便和他認識，將總司當作哥哥一般仰慕。
; {{nihongo|立石真由里|立石 まゆり（たていし まゆり）|聲：空氣無子}}
: PS4、PSV、Switch版攻略對象。總司的妹妹，和唯梨初次見面後便成為好友。很依賴總司，
