{{日本作品|回家之前的棉花糖|お家に帰るまでがましまろです}}是[[MARMALADE|MARMALADE]]在2017年7月28日發售的[[戀愛冒險|戀愛冒險]]類型[[日本成人遊戲|成人遊戲]]<ref>{{cite web|url=http://moedigi.com/game/2017/07/marmalade-mashimaro-master-up/|title=ましゅまろより甘い恋めしあがれ！ま～まれぇど『お家に帰るまでがましまろです』がマスターアップ|website=moedigi|date=2017-07-14|accessdate=2019-04-12|language=ja}}</ref>。

== 故事簡介 ==
宮原亮在爺爺過世後被家裡趕了出來，由於無處可去因此在神扇町的公園餐風露宿。某一天女孩春日部花音給了亮食物並把他帶回自家經營的店「棉花糖樹」。為了回報花音，亮決定要讓生意慘淡的棉花糖樹變得盛況空前<ref>{{cite web|url=https://www.tgsmart.jp/article.aspx?a=18210|title=ま〜まれぇどの新作は『プライマルハーツ』のスタッフが手がけるハートフルラブコメディー！|website=TG Smart|date=2017-02-01|accessdate=2019-04-12|language=ja}}</ref>。

== 登場角色 ==
=== 主要角色 ===
; {{nihongo|宮原亮|宮原 亮（みやはら りょう）}}
: 作品男主角，因為被家裡趕出來而在公園露宿，後來受花音幫助在蛋糕店「棉花糖樹」有了住的地方，為了報答花音立誓要讓店裡生意變好。原本是神扇學園的二年級生，但在沒有金錢資助的情況下暫時休學。
; {{nihongo|春日部花音|春日部 花音（かすかべ かのん）|聲：[[鈴谷真矢|鈴谷真矢]]}}
: 蛋糕店「棉花糖樹」的店主，性格開朗且無憂無慮的女孩，就讀神扇學園二年級。雙親都已過世，因為不想看到店面就這樣荒廢，決定和青梅竹馬朝霞汐一起把店經營下去。
; {{nihongo|朝霞汐|朝霞 汐（あさか うしお）|聲：{{lang|ja|月野きいろ}}}}
: 蛋糕店「棉花糖樹」的糕點師，神扇學園的三年級生，雖然體格嬌小但喜歡以姐姐的身分自居，
; {{nihongo|皇鈴紗紗|皇鈴 紗々（みすず ささ）|聲：[[結衣菜|結衣菜]]}}
: 神扇學園的一年級生，繪本作家。繪本的出道作大受好評，但因為感到壓力而無法再畫出好的作品。性格消極，情感動搖時經常說出負面的言詞。
; {{nihongo|禮羽雷克南|礼羽 ライコネン（らいは らいこねん）|聲：瀨良木若菜}}
: 在有名蛋糕店「Souvenir」工作的糕點師，芬蘭與日本的混血兒，就讀神扇學園二年級。由於對製作甜點感到疲憊而辭掉Souvenir的工作。

=== 其他角色 ===
; {{nihongo|JC|JC（ジェイシー）|聲：{{lang|ja|花園めい}}}}
: 在「棉花糖樹」負責接待客人與收銀的大學二年級女生。
; {{nihongo|所澤麻凜|所沢 まりん（ところざわ まりん）|聲：{{lang|ja|君鳥ふう}}}}
: 在「棉花糖樹」隔壁經營洋食屋的中年女性。性格大而化之，由於擔心花音而給棉花糖樹非常多協助。
; {{nihongo|川越太一|川越 太一（かわごえ たいち）|聲：明羽杏子}}
: 小學男生，總是充滿活力。父親因為腰痛住院代替父親幫忙配送工作。
; {{nihongo|川越靜香|川越 しずか（かわごえ しずか）|聲：一之宮葵}}
: 太一的妹妹，小學低年級生。個性成熟穩重，父親因為腰痛住院而代父親管理酒店的帳務。對亮一見鍾情，稱亮是自己的丈夫。
