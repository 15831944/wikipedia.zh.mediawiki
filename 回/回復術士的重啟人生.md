《'''回復術士的重啟人生'''》（{{lang-ja|回復術士のやり直し}}），是日本小說家{{link-ja|月夜涙}}在[[成為小說家吧|成為小說家吧]]上連載的網路小說，後由[[KADOKAWA|KADOKAWA]]的[[角川Sneaker文庫|角川Sneaker文庫]]出版的輕小說。副標題為「～即死魔法與複製技能的極致回復術～」（{{lang-ja|〜即死魔法とスキルコピーの超越ヒール〜}}）。{{lang|ja|しおこんぶ}}則負責出版後的插畫。

由羽賀{{lang|ja|ソウケン}}負責的漫畫版則在[[Young_Ace|Young Ace]]的網路版免費漫畫連載雜誌Young Ace UP上連載中。

==簡介==
在成為小說家吧連載時期就有不少殘酷、異常性癖的描述，並曾獲得該網站的單日、一週、每月排行的第一名<ref>{{Cite web|url=https://comic-walker.com/contents/detail/KDCW_KS11000027010000_68/|title=回復術士のやり直し ～即死魔法とスキルコピーの超越ヒール～|publisher=ComicWalker|language = ja|accessdate=2018-05-27}}</ref>。

在作者的作品中是唯一一個沒有在故事初期就讓狐耳角色登場的作品，作者表示原因是「不想讓狐耳少女遭受殘酷的待遇」<ref>{{Cite web|url=https://mypage.syosetu.com/mypageblog/view/userid/509642/blogkey/1617184/|title=【謝罪】新作、回復術士のやり直しについて【土下座】|author=月夜涙|publisher=小説家になろう|language = ja|accessdate=2018-05-24|date=2017-01-16}}</ref>。

==故事==

==人物==
以下名稱以台灣角川所出版的繁體中文版小說為準。
===勇者===
; {{nihongo|凱亞爾 / 凱亞爾葛|ケヤル/ケヤルガ}}
:【癒】之勇者。故事開始時唯一一個手無寸鐵、只負責治療，在隊伍中地位有如奴隸的人。
:因為恢復了自我而成功奪取魔王的心臟回溯了時間，並在恢復記憶後發誓要對過去自己所受的悲慘遭遇進行復仇。
:為了躲避王國的追緝而在平時都使用凱亞爾葛的名字行動。
:能力是【恢復（Heal）】，與透過此能力所衍生出的：可複製目標技能的【模仿】、能將肉體改造成自己所希望的模樣的【改良】，與利用治療效果進行惡性改造使肉體壞滅的【改惡】。

; {{nihongo|芙列雅·艾爾格蘭帝·吉歐拉爾|フレア・アールグランデ・ジオラル}}
:【術】之勇者。吉歐拉爾的第一公主。有著桃色長髮的少女。
:有著表裡兩種性格。
:使用著神裝武具·神杖瓦納爾甘德。

; {{nihongo|布蕾德|ブレイド}}
:【劍】之勇者。
:金髮碧眼、肉體結實的男裝美女。其實是本性嗜虐的蕾絲邊。
:使用著神裝武具·神劍拉格納洛克。

; {{nihongo|布列特|ブレット}}
:【砲】之勇者。
:體格高壯的巨漢。其實是喜歡玩弄美少年的男同性戀。
:使用著神裝武具·神砲塔斯拉姆。

===其他角色===
; {{nihongo|安娜|アンナ}}
:將失去父母的凱亞爾扶養長大的女性。

; {{nihongo|克蕾赫·葛萊列特|クレハ・クライレット}}
:單論劍術還在【劍】之勇者之上的女劍聖，凱亞爾初次使用【恢復】治療的對象。

; {{nihongo|剎那|セツナ}}
:亞人。冰狼族的女戰士，因為天生等級上限過低從備受期望的天才淪為族中的吊車尾。只能負責巡邏，被人類抓住後賣為奴隸。
:於此時罹患怪病，被需要前衛擔當肉盾的凱亞爾看上其不屈的精神而買下，為了向叛徒與人類復仇而自願成為凱亞爾的人。
