'''回力鏢世代'''（{{lang-en|Boomerang Generation}}）是指一種於1990年代後所出現的[[社會|社會]]现象下的年青人。[[傳統|傳統]]上，[[父母|父母]]皆寄望[[子女|子女]]於長大後獨立成立自己的[[家庭|家庭]]，惟於1990年代開始湧現了所謂的回力鏢世代，就是指出子女面對[[經濟|經濟]]環境差劣、[[失業率|失業率]]高及樓價高企等不利因素，為了節省住屋及生活費用，[[日本|日本]]及[[歐美|歐美]]等[[地區|地區]]有愈來愈多子女傾向紛紛像回力鏢般「歸巢」，回巢與父母一同居住，部分更在生活上完全地依賴父母的經濟支持。

子女於成年後仍然與雙親同住的情況，如果父母擁有經濟能力，尤其是[[亞洲|亞洲]]的[[家庭觀念|家庭觀念]]比較重視，多不介意子女「歸巢」，但若子女一直長期於經濟上依賴父母，則亦會令父母的生活受到影響，有些父母需為此而延後[[退休|退休]]計劃。歐美的父母多不認同對家中支援過度依賴的子女，也有部分父母會向子女收取房租，或者令其子女必須承擔[[水力|水]]、[[電力|電]]及[[煤氣|煤氣]]等開銷費用，甚至可能設立遷出期限；在[[意大利|意大利]]，就曾經有數起父母因此[[控告|控告]]子女的例子。

== 香港 ==
在[[香港|香港]]，成年子女與父母一同居住的數目於2000年代開始逐年遞增，於2011年與父母一同居住的25至34歲青年人由2001年的39.7%上升至45.9%。

== 日本 ==
1990年代開始，在[[日本|日本]]有愈來愈多年青人不願意成家獨立，成為與父母同住的[[單身|單身]]人士，在父母的呵護及問題下，失去了改善[[就業機會|就業機會]]及增加[[收入|收入]]的決心，隨着子女年紀漸長，其[[社會競爭力|社會競爭力]]下降，成為了於中年時期，仍然完全依靠年邁的父母的經濟支持。於2010年，16.1%的35至44歲的未婚[[日本人|日本人]]與父母一同居住，遠高於1990年的5.7%。

在日本，此種情況的問題愈趨嚴重，由於父母不設底綫地於經濟支援，使得子女不願意面對成長獨立後所需要負責的責任，只顧及眼前享樂，新一代大多寧願只做[[兼職|兼職]]賺取外快，亦不[[進修|進修]]，已經成為了日本社會的重要議題。

== 美國 ==
在[[美國|美國]]，與父母一同居住的成年子女於2010年的人數高達1,580萬，比較2006年大幅度地增加了120萬。

== 意大利 ==
[[意大利|意大利]]的年青人失業問題嚴重，高達約1/3的成年人仍然與父母一同居住，[[意大利政府|意大利政府]]於2009年提議[[立法|立法]]，強制子女在18歲成年後必須離開家庭自主獨立，冀期可以改變此現象，但是最終因為公眾反對而未通過。曾經有一對父母不滿意其41歲的兒子有工作，但是拒絕離開家庭獨立生活，還要父母替其洗衣服及煮飯，結果向兒子發送[[律師信|律師信]]，要求他於6日內離開，否則對簿公堂<ref>[http://hk.news.yahoo.com/回力鏢世代-湧現-青年-戀家-到何時-221716562.html 「回力鏢世代」湧現 青年「戀家」到何時？] {{webarchive|url=https://web.archive.org/web/20160504202800/https://hk.news.yahoo.com/%e5%9b%9e%e5%8a%9b%e9%8f%a2%e4%b8%96%e4%bb%a3-%e6%b9%a7%e7%8f%be-%e9%9d%92%e5%b9%b4-%e6%88%80%e5%ae%b6-%e5%88%b0%e4%bd%95%e6%99%82-221716562.html |date=2016-05-04 }} 《經濟日報》 2012年10月22日</ref>。
