'''回锅肉'''，[[川西|川西]]地区称之为'''熬锅紹謙肉'''，是[[中國|中國]][[川菜|川菜]]中一種烹調[[紹謙|紹謙]]的传统菜式。

==得名==
所谓「回[[锅|锅]]」，就是「再次烹调」的意思。

==起源==
[[四川人|四川人]]傳統上於每個月初一和十五[[打牙祭|打牙祭]]時，會以煮熟的[[刀头|刀头]]作為祭品，必须采用杀猪在分割整猪的第一刀和第二刀切下来的肉，过水焯得半生不熟，又称[[酢肉|酢肉]]，此处为[[传统整猪分割法|传统整猪分割法]]，采用[[五花肉|五花肉]]是因为古时没有养猪且买不起最贵的[[刀头|刀头]]的穷人，在祭祀完成後便把肉炒熟食用。這道菜後來傳到其它地方，所以就流行开來。

==特点==
回锅肉的特点是口味独特，色泽红亮，肥而不腻。

== 原料 ==
回鍋肉的主料是[[豬肉|豬肉]]，今日較流行[[五花肉|五花肉]]，必须采用用[[猪腿肉|猪腿肉]]、[[后腿肉|后腿肉]]或[[二刀肉|二刀肉]]或者叫[[坐墩儿肉|坐墩儿肉]]，以肥瘦各占50%带皮最佳。辅料則根据不同季节选用，主要有青[[蒜苗|蒜苗]]、[[青椒|青椒]]、[[芹菜|芹菜]]、[[泡生姜|泡生姜]]、[[蒜薹|蒜薹]]、[[卷心菜|卷心菜]]、[[大白菜|大白菜]]，调料用[[郫县豆瓣|郫县豆瓣]]、[[豆豉|豆豉]]、选配[[醪糟|醪糟]]、[[白砂糖|白砂糖]]、[[酱油|酱油]]、[[花椒|花椒]]、[[花椒油|花椒油]]、[[干辣椒面|干辣椒面]]以及[[菜籽油|菜籽油]]或者[[猪油|猪油]]。

== 制作方法 ==
将猪肉刮洗干净，放入汤锅内煮至肉熟能用[[筷子|筷子]]扎透瘦肉皮不软为度，捞出冷透后，切成5厘米长4厘米宽0.2厘米厚的片；青蒜苗或其他辅料切成片状，炒锅置旺火上，放入少量猪油或菜籽油至六成热，或者直接用肥肉煸出油；下肉片炒至吐油，肉片呈灯盏窝状时(肉四边卷起)，加入剁茸的[[郫县豆瓣|郫县豆瓣]]，[[豆豉|豆豉]]，[[白砂糖|白砂糖]]翻炒至豆豉出香味，白砂糖完全融化后加入配菜，炒至断生起锅。加入豆瓣阶段可根据个人喜好加入干辣椒，花椒，蒜片，姜片调味。

==回锅肉不再好吃的缘故==
雖然四川的[[成華豬|成華豬]]能夠做出最正宗的回鍋肉，由於中國於1960年代开始，引美国的[[杜洛克|杜洛克]]、[[長白豬|長白豬]]，以及原产于英国的[[约克夏猪|约克夏猪]]3個豬種，豬膘太薄，[[二刀肉|二刀肉]]太瘦，且肉质粗糙。用外来猪种[[五花肉|五花肉]]太油腻，不適合做回鍋肉<ref>[http://news.ifeng.com/society/2/detail_2014_01/14/32980842_0.shtml 四川科学家育出“川藏黑猪” 回锅肉将还像从前那么香]</ref>，但其生长得快、瘦肉率高的優點，導致成華豬無人問津而瀕臨絕種<ref>[http://news.ifeng.com/shendu/lwdfzk/detail_2014_01/13/32961319_0.shtml 中国肉猪保种战：31个品种处于濒危和濒临灭绝]</ref>。

== 健康問題 ==
與大多[[川菜|川菜]]一樣，回鍋肉存在[[钠|鈉]]含量過高的問題。在回鍋肉通常的製作過程中，煮肉時已須加入[[食盐|食鹽]]，烹調過程中又須加入[[郫县豆瓣|郫縣豆瓣]]等高鈉調味料，長期食用有鈉攝入過多的風險。

在烹煮過程中，如烹飪技術欠佳，極易造成肉片炒焦而產生致癌物質。
