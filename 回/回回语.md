'''回回语'''是中亚现代波斯语的初始形式，即早期新[[波斯语|波斯语]]（early new persian），也为元朝及明朝初期回回人所使用最为广泛的语言，曾为[[元朝|元朝]]三种官方语言之一（另外两种为汉语，[[蒙古语|蒙古语]]），并设有回回国子监等机构进行回回语的翻译教学工作，后因明王朝颁布同化政策，它的使用范围从最初的口头语变为清真寺[[回族|回族]]经堂教育的书面语言，当代回族人口语中的波斯语单词即为回回语遗留，具有显著的回回语语音特征及用词习惯。

近现代有不少学者对回回语进行了大量研究和整理，较为有影响力的是南京大学刘迎胜教授、中央民族大学胡振华教授等等，而民间做此项研究的学者也不计其数，根据古籍记载中的回回语口音看，回回语接近于现在的平原[[塔吉克语|塔吉克语]]，而与伊朗的德黑兰方言略有差别，但差别仅在[[元音|元音]]方面。<ref>{{cite web |url=http://www.huizucn.org/forum-42-1.html |title=存档副本 |accessdate=2014-12-04 |deadurl=yes |archiveurl=https://web.archive.org/web/20141210025112/http://www.huizucn.org/forum-42-1.html |archivedate=2014-12-10 }}</ref><ref>{{Cite web|url = http://www.2muslim.com/portal.php?mod=view&aid=62045|title = 丁世仁：中国“回文”——“回回字”的历史演进|accessdate = 2012-05-07|deadurl = yes|archiveurl = https://web.archive.org/web/20120519055301/http://www.2muslim.com/portal.php?mod=view&aid=62045|archivedate = 2012-05-19}}</ref><ref>{{Cite web|url = http://wenku.baidu.com/view/31432a690912a2161479297b.html|title = 回族语言与波斯语的关系探析|accessdate = 2014-12-13}}</ref>
