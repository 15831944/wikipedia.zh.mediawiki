'''回龙山'''，位于[[湖南省|湖南省]][[长沙市|长沙市]][[宁乡县|宁乡县]]境内，古时候称为“南楚灵山”，因其山势如一条[[龙|长龙]]盘旋曲折，仿佛“迴龙望祖”而得名。<ref name="fjsw ">{{cite web|url=http://www.fjdh.com/bnznews/2008/06/08265225145.html|title=南楚灵山佛教圣地——迴龙山|accessdate=2008年|work=|publisher=佛教导航网|language=zh-cn}}{{Dead link|date=2018年7月 |bot=InternetArchiveBot |fix-attempted=no }}</ref>

== 地理 ==
回龙山海拔365米，漫山遍野绿树植被覆盖，天空中白云笼罩山头，又叫“白云窝”。<ref name="fjsw " />

回龙山与[[长沙|长沙]]相距68公里，与[[花明楼镇|花明楼镇]]相距50公里，与[[灰汤镇|灰汤镇]]相距15公里。<ref name="fjsw " />

== 景观 ==
=== 白云寺 ===
回龙山因其[[汉传佛教|汉传佛教]][[沩仰宗|沩仰宗]]而闻名[[湘中|湘中]]，被誉为“南岳山的香，迴龙山的烛”，山上有古寺[[白云寺|白云寺]]。<ref name=" nxw">{{cite web|url=http://www.hn.xinhuanet.com/yxsh/2004-02/05/content_1576779.htm|title=回龙山|accessdate=2004年|work=|publisher=海内外|language=zh-cn|deadurl=yes|archiveurl=https://web.archive.org/web/20160304124616/http://www.hn.xinhuanet.com/yxsh/2004-02/05/content_1576779.htm|archivedate=2016-03-04}}</ref>
[[唐朝|唐朝]][[大中|大中]]十二年（858年），当地人民建立了清林寺，就是白云寺的前身。[[清朝|清朝]][[乾隆|乾隆]]二十八年（1763年），正式更名为白云寺。<ref name=" nxw" />

白云寺有八大景观：鱼山夜月、菱塘晚风、枫林吐雾、石磴穿云、云关锁翠、远岫横琴、仙岩瀑布、蟾谷嘘虹。<ref name="nxwl " >{{cite web |url= http://www.nx123.ccoo.cn/blog/article.asp?id=12368|title=迴龙山白云寺八景 |accessdate= 2005年|work= |publisher=宁乡在线 |language =zh-cn}}</ref>

== 文化 ==
[[乾隆皇帝|乾隆皇帝]]曾来回龙山白云寺游玩。<ref name="fjsw " /><ref name="sww " >{{cite web |url=http://www.sanwen.net/subject/249323/ |title=游回龙山白云寺 |accessdate= 2011年|work= |publisher=散文网 |language =zh-cn}}</ref>

清朝晚年[[左宗棠|左宗棠]]曾题词“南楚灵山”匾额。<ref name="fjsw " /><ref name="sww " />

清朝晚期诗人[[周亮|周亮]]曾作诗吟咏《回龙山》：“西边远岫似横琴，玉轴徽弦隐翠岑。千古不闻弹一曲，高山流水待知音。”<ref name="nxwl " />

1917年，[[毛泽东|毛泽东]]曾借宿于白云寺，并且和方丈大师交流。<ref name=" nxw" /><ref name="fjsw " /><ref name="sww " />
