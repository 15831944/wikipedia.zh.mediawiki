'''宿菲菲'''（{{bd|1979年|12月9日|}}），网名'''不死鸟'''，出生于[[河北省|河北省]][[秦皇岛市|秦皇岛市]]，毕业于[[外交学院|外交学院]]，[[单机游戏|单机游戏]]网站[[3DMGAME|3DMGAME]]创始人及站长<ref>{{Cite web|url = http://www.guangyuanol.cn/news/roll/2014/0103/139072.html|title = 站长鸟姐宿菲菲个人详细资料介绍|accessdate = 2015-08-03|author = |date = 2014-01-03|publisher = 川北在线|language = zh-cn}}</ref>。

== 生平 ==
{{Expand section|time=2015-07-16T03:25:10+00:00}}
* 2003年创立3DM论坛。
* 2007年4月，因涉嫌传播淫秽物品牟利罪被捕，在看守所关押6个月，同年10月获释。其涉嫌的案件被认为是中国大陆首例網路贩卖境外淫秽游戏案<ref>{{Cite web|url = http://www.xiancn.com/gb/news/2007-12/03/content_1391921.htm|title = 网络不死鸟之死|accessdate = 2015-08-03|author = |date = 2007-12-03|publisher = 西安晚报|language = zh-cn}}</ref>。<ref>[https://www.bilibili.com/video/av5923400/ 宿菲菲被公安抓獲]</ref>
* 2015年1月，宿菲菲入驻[[虎牙直播|虎牙直播]]，开始不定期直播单机游戏。
* 2017年1月，宿菲菲入驻[[斗鱼直播|斗鱼直播]]，开始与一些员工定期直播单机游戏。
