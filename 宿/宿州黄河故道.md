'''宿州黄河故道'''或称'''皖北黄河故道'''<ref name="a1"/>、'''萧砀黄河故道'''<ref>{{Cite web |url=http://www.zgfxnews.com/mp/wbck/html/2016-12/16/content_283776.htm |title=故道之水天上来|accessdate=2018-10-03 |author=俞鑫、刘晓琼 |date=2016-12-16 |work= |publisher=拂晓新闻网，皖北晨报数字版|language=Zh-hans }}</ref>、'''萧砀百里黄河故道'''<ref>{{Cite web |url=http://www.ahsz.gov.cn/11708319.html |title=自然地理|accessdate=2018-10-03 |author= |date= |work= |publisher=宿州市人民政府网站|language=Zh-hans }}</ref>，是指位于[[安徽省|安徽省]][[宿州市|宿州市]]下辖的[[萧县|萧县]]和[[砀山县|砀山县]]境内的[[黄河故道|黄河故道]]。清[[咸丰黄河大改道|咸丰五年（1855年）黄河大改道]]之前，黄河由[[砀山县|砀山县]]入境，再从萧县流入[[江苏省|江苏省]][[徐州市|徐州市]]境<ref name="b2"/>。故道全长95.4公里，流域面积470平方公里<ref>{{Cite web |url=http://news.dzwww.com/shehuixinwen/201809/t20180920_17863220.htm |title=让“故道明珠”更璀璨夺目 |accessdate=2018-10-03 |author=李静 |date=2018-09-20 |work= |publisher=大众网，来源：人民政协报|language=Zh-hans }}</ref>。

[[安徽省人民政府|安徽省人民政府]]在两县分别设立[[萧县黄河故道自然保护区|萧县黄河故道自然保护区]]、[[砀山黄河故道自然保护区|砀山黄河故道自然保护区]]和[[古黄河省级森林公园|古黄河省级森林公园]]。[[十二五规划|十二五]]期间，萧县和砀山县准备将两个省级自然保护区整合晋升为[[国家级自然保护区|国家级自然保护区]]。2016年6月7日[[安徽省林业厅|安徽省林业厅]]发文，以对[[中国民主同盟|中国民主同盟]]安徽省委员会在[[安徽省政协|省政协]]提出的建设“皖北黄河故道湿地与森林公园”的相关提案进行回复。林业厅在回复函中表示，将与安徽省和宿州市的相关部门，“进一步指导砀山和萧县编制自然保护区总体规划，申报黄河故道国家级自然保护区工作。”<ref name="a1">{{Cite web |url=http://www.ahly.gov.cn/zwgk2/zwxxgkml/rddbjyzxw/zxwydabl/135085.shtml |title=关于省政协十一届四次会议第297号提案答复的函|accessdate=2018-10-03 |author= |date= |work= |publisher=安徽省林业厅网站|language=Zh-hans }}</ref>。

萧县的黄河故道长48公里<ref name="b1">{{Cite web |url=http://www.zgfxnews.com/mp/wbck/html/2016-04/11/content_260263.htm |title=春光明媚 来萧县黄河故道看花海|accessdate=2018-10-03 |author=王恒 |date=2016-04-11 |work= |publisher=拂晓新闻网，皖北晨报数字版|language=Zh-hans }}</ref>，今分属[[刘套镇|刘套镇]]、[[杨楼镇_(萧县)|杨楼镇]]、[[新庄镇_(萧县)|新庄镇]]，属河流型湿地<ref name="b2">{{Cite web |url=http://www.sohu.com/a/216064342_100001695 |title=绿会代表团一行考察安徽萧县黄河故道|accessdate=2018-10-03 |author= |date=2018-01-10 |work= |publisher=搜狐网|language=Zh-hans }}</ref>。集水面积181.2平方公里，富水期曾近400平方公里<ref name="b1"/>。其中，刘套段长15.6千米，流域面积约22.8平方公里。现存河道的水源来源于自然降水<ref name="b2"/>。

2009年，宿州市人民政府批准设立市级自然保护区。2012年，[[安徽省人民政府|安徽省人民政府]]批准设立省级湿地自然保护区，总面积3200公顷<ref name="b1"/>。

黄河故道贯穿砀山县，西起[[官庄坝镇|官庄坝镇]]下河陈庄桩，东至[[葛集镇|葛集镇]]高寨村，途经七个镇（官庄坝镇、[[玄庙镇|玄庙镇]]、[[周寨镇|周寨镇]]、[[砀城镇|砀城镇]]、[[良梨镇|良梨镇]]、葛集镇、[[唐寨镇|唐寨镇]]）以及官庄林场、果园场、县林场和园艺场。东西长46.8公里，南北平均宽度2.2公里，流域面积277.8平方公里，水面面积36.5平方公里。水面宽度在15米和32米之间，平均宽度21.6米，水深3.6米。故道两侧，林地、湿地间杂分布。土壤以沙质和沙壤质为主<ref>{{Cite web |url=http://www.dangshan.gov.cn/index.php?m=content&c=index&a=lists&catid=512 |title=黄河故道|accessdate=2018-10-03 |author= |date= |work= |publisher=砀山县人民政府网站|language=Zh-hans }}</ref>。

2003年12月，[[安徽省林业厅|安徽省林业厅]]批准设立古黄河省级森林公园，总面积1400公顷。2016年6月7日安徽省林业厅发文，在这份与黄河故道相关的回复函中表示，古黄河省级森林公园自2003年“批建以来，该森林公园未编制总体规划和成立相应管理机构，未开展森林旅游活动，省级森林公园主体功能无法正常发挥。”林业厅已于2016年1月发文责令限期整改<ref name="a1"/>。同年10月，安徽省国土资源厅同意砀山黄河故道申报安徽省省级地质公园。2017年1月，省国土资源厅批准设立安徽砀山古黄河地质公园，位于官庄坝镇和玄庙镇境内。9月26日，在[[岳庄坝水库|岳庄坝水库]]举行公园开工仪式。<ref>{{Cite web |url=http://www.sohu.com/a/194862976_681146 |title=重磅！占地11万多亩砀山最大的公园开工，三年后将建成！！|accessdate=2018-10-03 |author= |date=2017-09-27 |work= |publisher=搜狐网|language=Zh-hans }}</ref>。
