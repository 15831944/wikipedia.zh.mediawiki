'''宿城区'''是[[中国|中国]][[江苏省|江苏省]][[宿迁市|宿迁市]]所辖的[[市辖区|市辖区]]，是宿迁市主城区，地处[[淮海经济区|淮海经济区]]中部，属陇海经济带、沿海经济带、沿江经济带交叉辐射区。[[京杭大运河|京杭大运河]]、[[古黄河|古黄河]]穿越城区，境内有[[骆马湖|骆马湖]]、[[洪泽湖|洪泽湖]]。目前擔任宿城区委书记的是裴承前<ref>{{Cite web|url = http://sqdj.gov.cn/index.php?m=content&c=index&a=show&catid=21&id=3952|title = 宿城区委主要领导调整 裴承前任宿城区委书记|deadurl = yes|archiveurl = https://web.archive.org/web/20160131011606/http://sqdj.gov.cn/index.php?m=content&c=index&a=show&catid=21&id=3952|archivedate = 2016-01-31}}</ref>。

== 历史 ==

[[东晋|东晋]]时置[[宿预县|宿预县]]，[[宿预城遗址|县城遗址]]在今宿城区[[郑楼镇|郑楼镇]]古城山南麓<ref>{{Cite web |url=http://www.zhenglou.gov.cn/Article.aspx?id=40&item=001001002 |title=宿预城遗址碑记 |accessdate=2014-11-10 |author= |date=2012-02-07 |work= |publisher=宿迁市郑楼镇政府网，来源：宿迁市宿城区郑楼镇人民政府所立宿预城遗址碑记 |language=Zh-hans |deadurl=yes |archiveurl=https://web.archive.org/web/20141110153544/http://www.zhenglou.gov.cn/Article.aspx?id=40&item=001001002 |archivedate=2014-11-10 }}</ref>。

原为地级宿迁市市委市政府所在地宿城镇，1996年宿迁单列为地级市后，更名为宿城区并下设有井头乡、双庄镇；2004年3月在新一波的行政区划中，宿城区将原属于[[泗阳|泗阳]]的[[郑楼|郑楼]]、[[屠园|屠园]]、[[洋河镇|洋河镇]]、[[仓集|仓集]]、[[中扬|中扬]]五乡镇及原属泗洪县的[[陈集镇|陈集镇]]以及其周边原属宿豫县的部分乡镇划归旗下，使之一跃成为一个拥有854平方公里土地，89万人口的大区。

==行政区划==
2004年3月区划调整，辖14个乡镇、4个街道和1个省级经济开发区。
* 镇：[[双庄镇|双庄镇]]、[[耿车镇|耿车镇]]、[[埠子镇|埠子镇]]、[[龙河镇|龙河镇]]、[[洋北镇|洋北镇]]、[[仓集镇|仓集镇]]、[[中扬镇|中扬镇]]、[[郑楼镇|郑楼镇]]、[[陈集镇|陈集镇]]。
* 乡：[[罗圩乡|罗圩乡]]、[[南蔡乡|南蔡乡]]、[[屠园乡|屠园乡]]、[[三棵树乡|三棵树乡]]。
* 街道：[[幸福街道|幸福街道]]、[[项里街道|项里街道]]、[[河滨街道|河滨街道]]、[[古城街道|古城街道]]、[[黄河街道|黄河街道]]、[[古楚街道|古楚街道]]。
