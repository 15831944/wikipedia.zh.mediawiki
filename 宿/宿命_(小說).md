== 故事簡介 ==
和倉勇作過去曾以升學醫科大學為目標，但因某事而放棄了，而且和初戀女友美佐子也分了手，最後加入警界，成為一名警官。這時，和倉在學生時代的競爭對手瓜生晃彥作為某件殺人案件的嫌疑人再次出現了。這時的晃彥已經是一名優秀的[[腦外科|腦外科]]醫生，而且竟然已經和美佐子結婚。隨著宿命的再會，幾個秘密也陸續明朗化。最終在等待著他們的會是……

== 登場人物 ==
; {{nihongo|和倉 勇作|{{ruby|和倉|わくら}} {{ruby|勇作|ゆうさく}}|Wakura Yuusaku}}
: 島津警局巡查部長。小時候比較會照顧同學，總是班上的領導人物，課業或體能各項表現皆很優越。
: 從小學到高中一直與瓜生晃彦唸同一所學校，數度曾與晃彥同班級，遇到晃彥只能拿第二。

; {{nihongo|和倉 興司|{{ruby|和倉|わくら}} {{ruby|興司|こうじ}}|Wakura Kouzi}}
: 和倉勇作的父親，是名警察。對日野早苗的凶案緊追不棄，後來莫名的放棄了。

; {{nihongo|瓜生 和晃|{{ruby|瓜生|うりゅう}} {{ruby|和晃|かずあき}}|Uryuu Kazuaki}}
: 瓜生工業股份有限公司社長。瓜生直明的父親；瓜生晃彦的爺爺。

; {{nihongo|瓜生 直明|{{ruby|瓜生|うりゅう}} {{ruby|直明|ちょくあけ}}|Uryuu Chokuake}}
: UR電產的社長，罹患食道癌過世。UR電產是日本電機廠商，是縣內規模最大的企業。

; {{nihongo|瓜生 亞耶子|{{ruby|瓜生|うりゅう}} {{ruby|亜耶子|あやこ}}|Uryuu Ayako}}
: 瓜生直明的第二任妻子，年輕貌美。與直明生下瓜生弘昌與瓜生園子。

; {{nihongo|瓜生 晃彦|{{ruby|瓜生|うりゅう}} {{ruby|晃彦|あきひこ}}|Uryuu Akihiko}}
: 統和醫科大學助理教授；瓜生直明與第一任妻子的兒子。資質相當優修，各科成績總是輕輕鬆鬆拿第一。
: 很難與人相處，同儕並不喜歡他。沒原由的討厭勇作，甚至會做出輕蔑、鄙視、瞧不起勇作的行為。

; {{nihongo|瓜生 美佐子|{{ruby|瓜生|うりゅう}} {{ruby|美佐子|みさこ}}|Uryuu Misako}}
: 瓜生晃彥的妻子，娘家姓江島。就讀高中時與和倉勇作在紅磚醫院認識進而交往，大學主修英文，想當英文老師。
: 報考四家企業只有UR電產錄取她，隸屬人事部，被分派到「董事室特別秘書」職位。

; {{nihongo|瓜生 弘昌|{{ruby|瓜生|うりゅう}} {{ruby|弘昌|ひろまさ}}|Uryuu Hiromasa}}
: 大學生。非常依賴母親亞耶子，有點戀母情結。

; {{nihongo|瓜生 園子|{{ruby|瓜生|うりゅう}} {{ruby|園子|そのこ}}|Uryuu Sonoko}}
: 高中生。瓜生直明晚年得女，非常溺愛園子。園子對於直明的遺書對她隻字未提，相當耿耿於懷。

; {{nihongo|江島 壯介|{{ruby|江島|えじま}} {{ruby|壮介|そうすけ}}|Ezima Sousuke}}
; {{nihongo|江島 波江|{{ruby|江島|えじま}} {{ruby|波江|なみえ}}|Ezima Namie}}
: 瓜生美佐子的父母親。江島壯介在紅磚醫院住院過後進入UR電產工作。

; {{nihongo|松村 顯治|{{ruby|松村|まつむら}} {{ruby|顕治|けんじ}}|Matsumura Kenzi}}
: UR電產常務董事。年輕時一直擔任瓜生直明的心腹，在UR電產裡屬瓜生派系。

; {{nihongo|須貝 忠清|{{ruby|須貝|すがい}} {{ruby|忠清|ただきよ}}|Sugai Tadakiyo}}
: 瓜生和晃的妹夫。瓜生和晃逝世後，須貝派的勢力更加茁壯。

; {{nihongo|須貝 正清|{{ruby|須貝|すがい}} {{ruby|正清|まさきよ}}|Sugai Masakiyo}}
: 須貝忠清的兒子。瓜生直明逝世後接任UR電產社長。在墓園裡被人以毒箭射死。

; {{nihongo|尾藤 高久|{{ruby|尾藤|びとう}} {{ruby|高久|たかひさ}}|Bitou Takahisa}}
: 須貝正清的秘書。瓜生亞耶子的情夫。

; {{nihongo|上原 雅成|{{ruby|上原|うえはら}} {{ruby|雅成|まさなり}}|Uehara Masanari}}
: 上原腦神經外科醫院的院長。醫院的外觀是紅磚砌成，小孩們都稱之「紅磚醫院」。

; {{nihongo|上原 伸一|{{ruby|上原|うえはら}} {{ruby|伸一|しんいち}}|Uehara Shinichi}}
: 上原醫院的院長。上原雅成的入贅女婿；上原晴美的丈夫。

; {{nihongo|日野 早苗|{{ruby|日野|ひの}} {{ruby|早苗|さなえ}}|Hino Sanae}}
: 紅磚醫院的病人，口袋裝有很多糖果、牛奶糖會分給小孩，和倉勇作喊她「姐姐」。離奇的從醫院窗戶摔下死亡。

; {{nihongo|西方|{{ruby|西方|せいほう}}|Seihou}}
: 島津警局警部。

; {{nihongo|織田|{{ruby|織田|むらはし}}|Murahashi}}
: 島津警局警部補。
