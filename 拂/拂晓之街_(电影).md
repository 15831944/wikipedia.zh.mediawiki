《'''拂晓之街'''》（{{lang-ja|'''夜明けの街で'''}}）2011年10月8日上映于[[日本国|日本国]]的电影。

== 剧情 ==
该片根据日本著名作家[[东野圭吾|东野圭吾]]的同名作品《[[拂晓之街_(小说)|拂晓之街]]》改编而成，讲述了会社员渡部和下属仲西秋叶的不伦之恋。<ref>{{cite web |url=http://ent.ifeng.com/zz/detail_2011_10/09/9705482_0.shtml |title=《拂晓之街》首映 主演深田恭子直言不愿当小三 |publisher=凤凰网 |accessdate= 2011-10-09|work= |language =zh-cn}}</ref><ref>{{cite web |url= http://ent.ifeng.com/zz/detail_2011_10/09/9705825_0.shtml|title= 深田恭子新片《拂晓之街》大玩不论恋 坦言无法认同|publisher=凤凰网 |accessdate=2011-10-19 |work= |language =zh-cn}}</ref><ref name=" xlw" >{{cite web |url= http://ent.sina.com.cn/m/f/2011-08-12/14463385226.shtml|title= 深田恭子出席《拂晓之街》见面会 展现不伦恋情|publisher=新浪网 |accessdate=2011-08-12 |work= |language =zh-cn}}</ref>

== 社会反映 ==
《拂晓之街》在[[日本雅虎|日本雅虎]]的评分只有2.52分，在播出的周六周日的票房收入为为3315万4800日元，排名第八。<ref>{{cite web |url=http://ent.qq.com/a/20111012/000382.htm |title=日本：《猩球崛起》夺冠 《拂晓之街》黯淡开局 |publisher=腾讯网 |accessdate=2011-10-12 |work= |language =zh-cn}}</ref>
