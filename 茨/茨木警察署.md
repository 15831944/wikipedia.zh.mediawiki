'''茨木警察署'''（{{lang-ja|茨木警察署（いばらきけいさつしょ）}}）為日本[[大阪府警察|大阪府警察]]所管轄的[[警察局|警察署]]之一。

== 所在地 ==
* [[大阪府|大阪府]][[茨木市|茨木市]]中穗積一丁目6番38號

== 管轄區域 ==
* [[茨木市|茨木市]]

== 交番 ==

* 鮎川交番 - 茨木市鮎川5丁目1-2
* 茨木驛前交番 - 茨木市驛前1丁目1-14
* 永代町交番 - 茨木市永代町2-20
* 春日丘交番 - 茨木市南春日丘1丁目1-22
* 北大阪流通中心ー前交番 - 茨木市宮島2丁目1-5
* 郡交番 - 茨木市上郡1丁目2-2 
* 下穗積交番 - 茨木市下穂積1丁目1-4
* 總持寺交番 - 茨木市庄2丁目25-11
* 玉櫛交番 - 茨木市水尾3丁目1-54
* 玉島交番 - 茨木市玉島2丁目12-28
* 豐川交番 - 茨木市南清水町1-1
* 福井交番 - 茨木市東福井2丁目6-7 
* 双葉町交番 - 茨木市雙葉町11-23 
* 南茨木交番 - 茨木市天王2丁目1-19 
* 耳原交番 - 茨木市南耳原2丁目5-10

* 石河駐在所 - 茨木市大岩312-4
* 清溪駐在所 - 茨木市泉原1-3
* 見山駐在所 - 茨木市下音羽480
