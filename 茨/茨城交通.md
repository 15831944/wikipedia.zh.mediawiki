'''茨城交通株式會社'''（{{jpn|j=茨城交通|hg=いばらきこうつう|rm=Ibaraki Kōtsū|f=y}}）設置於[[日本|日本]][[茨城縣|茨城縣]][[水戶市|水戶市]]，主要從事[[公共巴士|公共巴士]]、[[遊覽車|旅遊巴士]]服務，以及經營[[旅遊業|旅遊業]]和[[不動產|不動產]]的[[公司|公司]]。簡稱'''茨交'''（{{lang|ja|いばこう}}）。雖然茨城交通於2009年根據{{tsl|ja|民事再生法|民事再生法|民事再生手續}}進行了{{tsl|ja|新旧分離|新舊分離}}，但本條目將同名的新舊公司視為有連續性的整體來介紹。「茨城交通」同為[[註冊商標|註冊商標]]<ref>日本 第5354689號，權利者為茨城交通</ref>。

== 歷史 ==
=== 合併前的鐵路公司 ===
茨城交通在成立時，參與合併的鐵路公司在為下列的三間。

; 水濱電車（{{lang|ja|すいひんでんしゃ}}）
: 於1921年設立。以經營{{tsl|ja|電気鉄道|電氣軌道}}及供電事業作為目標。後於[[1922年|1922年]]開業，正式開始了供電及電氣軌道（濱田至磯濱）事業。軌道在不斷地慢慢延長，到合併時袴塚至湊（後來大洗至湊這段在[[1938年|1938年]]6月後停運）這段經已開業。電力事業在[[1942年|1942年]]因{{tsl|ja|日本発送電#配電統制令|配電統制令}}而轉讓給[[東京電力|關東配電]]。茨城交通成立後改為{{tsl|ja|茨城交通水浜線|茨城交通水濱線|水濱線}}，不過在[[1966年|1966年]]卻被廢線。
; 茨城鐵道（{{lang|ja|いばらきてつどう}}）
: 於[[1923年|1923年]]設立。在[[1926年|1926年]]至[[1927年|1927年]]期間，已開業的鐵路線由[[赤塚車站|赤塚]]至御前山。茨城交通成立後改為{{tsl|ja|茨城交通茨城線|茨城交通茨城線|茨城線}}，於[[1971年|1971年]]廢線。通稱為「茨鐵（{{lang|ja|いばてつ}}）」。
; 湊鐵道（{{lang|ja|みなとてつどう}}）
: 於[[1907年|1907年]]設立。在[[1913年|1913年]]至[[1928年|1928年]]期間，已開業的鐵路線由[[勝田車站|勝田]]至{{tsl|ja|阿字ヶ浦駅|阿字浦車站|阿字浦}}。茨城交通成立後改為{{tsl|ja|ひたちなか海浜鉄道湊線|常陸那珂海濱鐵道湊線|湊線}}，後於[[2008年|2008年]]子公司化，成為{{tsl|ja|ひたちなか海浜鉄道|常陸那珂海濱鐵道}}（佔49%股權，餘下為[[常陸那珂市|常陸那珂市]]所有）。到了現在常陸那珂海濱鐵道湊線仍然被俗稱為「湊鐵道」「湊鐵道線」。

===舊公司成立===
戰時體制下因陸運統制令而進行交通整合，於[[第二次世界大戰|第二次世界大戰]]期間的[[1944年|1944年]][[8月1日|8月1日]]，在完成整合茨城縣內交通事業者後正式成立。

整合是以使用固有設備作出資的'''水濱電車'''及其系列下的'''茨城鐵道'''、'''湊鐵道'''及'''袋田温泉自動車'''作為主體，同時也收購了包括個人經營在內的中小巴士事業者，更接收原屬{{tsl|ja|日立電鉄|日立電鐵}}及[[東武鐵道|東武鐵道]]的巴士路線。

水濱電車是由當地的豪商竹內權兵衛創設。竹內亦有參與常北地區各間鐵路公司的經營，除了上述以外，竹內也跟日立電鐵的前身常北電氣鐵道有很深的關係。在整合實施初期，茨城縣下分為常總地區、鹿行（{{lang|ja|ろっこう}}）地區及常北地區3個部分，常北地區雖然只準備將竹內系的企業作整合，但因常北電氣鉄道於[[1941年|1941年]]編入了[[日立製作所|日立製作所]]的系列下而離開竹內的控制，常北地區再被分為水戸地區及日立地區，結果實際上整合後的體制為分為4個部分。

===民事再生手續===
之後，竹内家代代繼任社長，除了公共交通以外也開始涉足多項事業，不過在1990年代後，本業的鐵路及巴士的使用者減少，以及住宅地開發等不動產有關事業失敗的各種問題，令到經營狀態惡化
<ref>與位於東海村的新興住宅地「弗洛倫斯須和間」因[[東海村JCO臨界意外|東海村JCO臨界意外]]而賣不出去有很大的關係。</ref>。為此，在一直往來金融機構的支援下對無利可圖的部門進行重組，而金融機構方面也放棄了對貸款的債權嘗試讓其重整，最後在沒有好轉的情況下只好放棄自力重建，於2008年11月11日，在{{tsl|ja|水戸地方裁判所|水戶地方裁判所}}申請進行{{tsl|ja|民事再生法}}<ref>[http://www.tdb.co.jp/tosan/syosai/2827.html 帝国データバンク 速報] {{webarchive|url=https://web.archive.org/web/20090705053523/http://www.tdb.co.jp/tosan/syosai/2827.html |date=2009-07-05 }} - 2008年12月20日 閲覧</ref>，於同日水戶地裁下達了監督命令<ref>事件番号：水戸地方裁判所 平成20年（再）第6号 民事再生手続開始申立事件</ref>。

於2008年11月19日，決定了再生手續開始後，在司法的監督下開始重組。在{{tsl|ja|経営共創基盤|經營共創基盤}}的支援下在翌年2009年[[3月27日|3月27日]]設立了新公司，並在[[7月1日|7月1日]]繼承了舊公司的事業（新舊分離）。此外連集團內其他公司的經營團隊亦為之一新，由此從創業開始關係就很深竹內家退出了經營的第一線。分割出來的舊公司改名為'''Ai管理'''（{{lang|ja|アイ管理}}）並負責債權的管理及整理。

新公司成立後，新設了前往仙台、名古屋的高速巴士路線，社員從約600增加至約800人，平均年収亦增加了24%左右。而在近郊地域與水戶市內連結路線的車費最多下調近4成，為針對高中生的需要以開拓客源<ref name="茨城新聞10/02/24">[[#茨城新聞10/02/24|2010年2月24日茨城新聞]]</ref>，成功地取得不俗的效果<ref>[http://www.nikkei.com/article/DGKKZO12269300Y7A120C1TZG000/ （９）規模より知恵で勝負　茨城交通、戦略的値下げで需要開拓　：日本経済新聞]</ref>。當初在2010年只不過是試験性質實行<ref>[http://www.nikkei.com/article/DGXNASFB0203X_S0A101C1L51000/ 茨城交通　運賃下げや買い物バス、来春以降も継続　　：日本経済新聞]</ref>。

== 巴士事業 ==
[[file:Transit_bus_for_Fukuroda_Falls.jpg|thumb]]的任務。]]
除了以水戶市為中心，為縣央（[[笠間市|笠間市]]、[[東茨城郡|東茨城郡]][[大洗町|大洗町]]、-{[[城里町|城里町]]}-、[[茨城町|茨城町]]）、縣北（[[那珂市|那珂市]]、[[常陸那珂市|常陸那珂市]]、[[常陸太田市|常陸太田市]]、[[常陸大宮市|常陸大宮市]]、[[久慈郡|久慈郡]][[大子町|大子町]]、[[那珂郡|那珂郡]][[東海村|東海村]]、[[日立市|日立市]]）及[[栃木縣|栃木縣]][[芳賀郡|芳賀郡]][[茂木町|茂木町]]提供普通路線巴士外，也有7條從縣央、縣北地域出發，來往縣內外的高速巴士、機場巴士路線在運行。

巴士線路的沿線內，除了包括縣都水戶、工業都市勝田外，更有[[偕樂園|偕樂園]]、{{tsl|ja|大洗海岸}}、{{tsl|ja|アクアワールド・大洗|大洗水族館}}、{{tsl|ja|国営ひたち海浜公園|日本國營常陸海濱公園|國營常陸海濱公園}}、[[笠間市|笠間]]、[[袋田瀑布|袋田瀑布]]等受歡迎的觀光地。另外，在營業範為內舉行如[[偕樂園|水戶梅祭}}、{{tsl]]及[[水戶蜀葵|水戶蜀葵]]主場球賽等著名的大型活動，均肩負著運送大量乘客的任務。

在茨城縣內的路線規模僅次於[[關東鐵道|關東鐵道]]，而在{{tsl|ja|水戸都市圏|水戶都市圈}}中為最大。就算是日間，在水戶市中心內很多巴士班次的間隔十分頻密僅為1分鐘左右，不過在周邊地方與其他市町村亦有不少班次相當疏落的路線。在{{tsl|ja|鹿行|鹿行地域}}，因水濱電車旗下{{tsl|ja|鹿島軌道}}的關係仍擁有普通巴士路線，但在2009年暫停了鉾田線（系統51）的營運，不過後於[[2015年|2015年]][[12月14日|12月14日]]重開了大洗車站至新鉾田車站的路線，而重返了鹿行地域<ref>[http://www.ibako.co.jp/contents/newsrelease/2015/12/10714.html 12月14日に大洗～鉾田線開業に伴うダイヤ改正を実施します – ニュースリリース - 茨城交通] {{webarchive|url=https://web.archive.org/web/20170711154336/http://www.ibako.co.jp/contents/newsrelease/2015/12/10714.html |date=2017-07-11 }}</ref>。

於[[1992年|1992年]][[10月|10月]]，由於希望將經營合理化，所以將{{tsl|ja|茨城交通大子営業所|茨城交通大子營業所|大子營業所}}子公司化，成為{{tsl|ja|茨城交通大子営業所|茨交縣北巴士}}，但在[[2010年|2010年]][[2月|2月]]，整個集團進行重編及再分割的關係，巴士及旅行事業部門整合到茨城交通內，的士事業則交給了旗下的{{tsl|ja|茨城オート|茨城Auto}}，而茨交縣北巴士則執行清盤<ref>[http://www.ibako.co.jp/info/20100129soshiki.htm グループの組織再編に関するお知らせ] {{webarchive|url=https://web.archive.org/web/20110624024243/http://ibako.co.jp/info/20100129soshiki.htm |date=2011-06-24 }} - 茨城交通グループ（2010年1月29日付）</ref>。另外，於2010年[[6月1日|6月1日]]，茨城Auto的巴士事業合併到茨城交通<ref>{{PDFlink|[http://www.ibako.co.jp/info-bus/20100302auto.pdf グループのバス事業の統合に関するお知らせ]}} - 茨城交通グループ（2010年3月2日付）</ref>。

=== 車輛 ===
與同縣行走的[[關東鐵道|關東鐵道]]不同，沒有使用車輛編號（使用登錄編號）<ref>不過舊茨城Auto是使用引入年＋流水號而成的車輛編號。</ref>。公共巴士的用車基本上是自行購買，設置有前後電閘門。而在{{tsl|ja|日産ディーゼル・スペースランナーRM|日產柴油Spacerunner RM}}、{{tsl|ja|日野・レインボー|日野Rainbow|日野Rainbow II}}、{{tsl|ja|いすゞ・エルガ|五十鈴ERGA}}及[[三菱扶桑MK|三菱扶桑MK]]等[[超低地台巴士|超低地台巴士]]新車引入之前，經常購買二手車使用，多為主要來自[[西武巴士|西武巴士]]及[[京王電鐵巴士|京王巴士]]等，從首都圏各地的營辦者或關西方面各地域購入。在2009年後購入的二手車亦配合低地台化，當中就有來自[[橫濱市營巴士|横浜市交通局]]的{{tsl|ja|日産ディーゼル・スペースランナーJP|日產柴油Spacerunner JP}}及[[京成巴士|京成巴士]]的[[三菱扶桑Aero_Star|三菱扶桑Aero Star]][[低地台巴士|低地台巴士]]。而在2012年，更從{{tsl|ja|尼崎市交通局}}購入了[[三菱扶桑Aero_Star|三菱扶桑Aero Star]]中摺門超低地台巴士<!--中グライドノンステ-->。另外，對比公共巴士，高速巴士所購入的新車更多，包括了很少用在高速巴士，{{tsl|ja|日産ディーゼル・フィリピン|日產柴油菲律賓}}所製造的車輛（[[1999年|1999年]]引入）。而於[[2007年|2007年]]濱田營業所新引入3輛[[三菱扶桑Aero_Bus|三菱扶桑Aero Bus]]高速巴士開始，改用全新的高速、觀光巴士専用塗裝（茨城交通的標誌及IBARAKIKOUTSU HIGHWAY EXPRESS字樣）。

=== 引入IC卡乘車券===
[[file:ibappi.jpg|thumb]]
在2011年12月的報章報導中，展望於2013年準備引入IC卡乘車券<ref>[http://www.shopbiz.jp/nf/news/98192.html 路線バスにIC乗車券、茨城交通、13年にも導入。] - 日経メッセ 2011年12月06日付け。</ref>。

其後，於2015年12月1日正式引入ICカード乗車券「[[Ibappi|Ibappi]]（{{lang|ja|いばっピ}}）」<ref>[http://www.ibako.co.jp/contents/newsrelease/2015/07/9720.html ICカード「いばっピ」のサービス開始時期とその内容のお知らせ] 茨城交通 ニュースリリース 2015年7月21日</ref>。而Ibappi定期券則在2016年2月27日開始。但不能與[[Suica|Suica]]及[[PASMO|PASMO]]等IC卡互通使用<ref>[http://www.ibako.co.jp/contents/newsrelease/2014/11/7399.html ICカード乗車券の導入について] {{webarchive|url=https://web.archive.org/web/20160930222239/http://www.ibako.co.jp/contents/newsrelease/2014/11/7399.html |date=2016-09-30 }} 茨城交通 ニュースリリース 2014年11月12日</ref>。到2016年6月10日為止，已經售出達3萬張<ref name="茨城新聞16/06/23">[[#茨城新聞16/06/23|2016年6月23日茨城新聞]]</ref>。

與此同時除了高速巴士及與其他公司的共通回数券外<ref>而在普通公共巴士，只有來往水戶車站至 大工町之間，以及水戶車站至縣廳的直通巴士的共通回数券仍然有售。</ref>，茨城交通的回数券在2016年6月30日廢止（退票服務也在同年8月31日停止<ref name="茨城新聞16/06/23">[[#茨城新聞16/06/23|2016年6月23日茨城新聞]]</ref>）。

水戶近郊的普通公共巴士，如JR巴士關東早已導入了Suica，此外關東鐵道（除了關東鐵綠巴士外<ref>[http://kantetsu.co.jp/green-bus/ 関鉄グリーンバス・関鉄パープルバス] （トップページ内に注意書き）</ref>）也在2015年10月開始導入PASMO<ref>[http://www.kantetsu.co.jp/bus/pasmo_suica.html PASMO・Suicaご利用案内] 関東鉄道公式サイト</ref>。而高速巴士方面，由JR巴士關東及關東鐵道巴士負責的「水戶號」亦能夠使用Suica和PASMO等交通系IC卡<ref>{{PDFlink|[http://kantetsu.co.jp/img/news/2017/17040701_bus/mito_ic.pdf 4月20日(木)から高速バス「水戸～東京駅線」に交通系ICカード導入のお知らせ]}} - 2017年4月7日 関東鉄道公式サイト</ref><ref>[http://www.ibako.co.jp/contents/newsrelease/2017/04/14543.html 高速バス「水戸～東京線」交通系ICカードへの対応について] - 2017年4月17日 茨城交通</ref>，因此在服務互通的層面上只有茨城交通處於孤立狀態。另外，在[[常陸太田市|常陸太田市]]與來往[[常陸太田車站|常陸太田車站]]至馬場八幡之間所使用的{{tsl|ja|でんてつハイカード|電鐵Hi Card}}（{{tsl|ja|日立電鉄交通サービス|日立電鐵交通服務}}）有出現競爭情況。
