{{engname|《窃听风云3》|''Overheard 3''}}是一部2014年上映的[[中華人民共和國|中]][[香港|港]]合拍電影，為《[[竊聽風雲系列|竊聽風雲系列]]》電影的第三部，亦是最後一部<ref>[http://news.mtime.com/pix/2013/06/03/379180.html "窃听风云3"揭地产黑幕 麦庄称不想再拍第四部] 2013年6月3日 Mtime时光网</ref>，由[[麥兆輝|麥兆輝]]和[[莊文強|莊文強]]联合編導及由[[爾冬陞|爾冬陞]]和[[黄斌|黄斌]]共同監製，[[刘青云|刘青云]]、[[古天乐|古天乐]]、[[吴彦祖|吴彦祖]]和[[周迅|周迅]]等多位知名演員主演。此片仍由[[百姓電影有限公司|百姓電影有限公司]]製作，[[博納影業|博納影業]]與[[發行工作室|發行工作室]]聯合發行。

本片共计获得2015年[[第34届香港电影金像奖|第34届香港电影金像奖]]的最佳编剧、最佳男主角和最佳男配角三项大奖。

== 劇情 ==
在香港，[[新界|新界]]每個男性原居民可以向政府申請建一棟三層獨立屋免地税。這計劃長期引起社會爭議。[[特區政府|特區政府]]決定終止接受[[丁屋|丁屋]]申请。地产公司准备將剩餘的丁權進行統一，興建多層式的「丁屋大廈」。新界鄉紳話事人陸瀚濤（[[曾江|曾江]] 飾）打算獨吞整個地產項目，但各地產巨頭定要分一杯羹。新界大地主陸永遠（[[錢嘉樂|錢嘉樂]] 飾）向誰靠攏，將影響到「丁屋大廈」的歸屬。陸瀚濤的司機羅永就（[[古天樂|古天樂]] 飾）駕車撞死陸永遠。陸永遠之死，令陸瀚濤成功拿下大地產項目。

五年後，羅永就出獄。同為陸瀚濤效力的同村兄弟陸金強（[[劉青雲|劉青雲]] 飾）等人卻不想與他分享既得利益。意想不到的是，羅永就在陸瀚濤的女儿等人指挥下，在酒中下藥迷倒眾人，更找來獄中結識的電腦奇才阿祖（[[吳彥祖|吳彥祖]] 飾），將陸金強等人的手機植入[[竊聽|竊聽]]程式，暗中監視眾人的一舉一動。

阿祖在竊聽行動當中，意外結識了為錢拼搏的[[圍村|圍村]]寡妇、陸永遠的遗孀阮月華（[[周迅|周迅]] 飾）。透過連番竊聽，羅永就發現了陸金強等人的巨大陰謀，几方势力经过缠斗，最后大部分人死掉。

== 製作 ==
=== 發展與選角 ===
2011年8月22日，有報導指《[[窃听风云2|窃听风云2]]》上映後因票房賣座，可能於2013年暑假推出《窃听风云3》，但導演[[莊文強|莊文強]]稱第三集未有任何概念，亦無意中被問到「還會繼續[[劉青雲|劉青雲]]、[[古天樂|古天樂]]、[[吳彥祖|吳彥祖]]的組合嗎？」莊文強回應：「在《[[竊聽風雲|竊聽風雲]]》裡，吳彥祖死了，古天樂斷手斷腳；《[[竊聽風雲2|竊聽風雲2]]》裡我們只讓吳彥祖死了。在第三部中，吳彥祖『可能』繼續死。」其後吳彥祖接受採訪時說：「我和古天樂合作太多，觀眾已經看膩了，會『離婚』一段時間。」<ref>[http://cq.qq.com/a/20110822/000253.htm 《窃听风云2》首周末票房或破亿 导演筹拍续集] 2011年8月22日 华商网</ref>後來於同年9月7日，吴彦祖在電影《窃听风云2》的慶功宴上表示已落實《窃听风云3》此製作。2013年3月15日，莊文強表示原本還考慮邀請[[江毅|江毅]]回來繼續演出<ref>[http://culture.ifeng.com/whrd/detail_2013_03/15/23134832_0.shtml “柯镇恶”江毅因病去世 《窃听风云2》成绝唱] 2013年3月15日 鳳鳯網</ref>，並依舊飾演劇中的反派，但江毅因[[肺癌|肺癌]]病逝使劇情中的反派角色落空。然而於同年6月3日，《竊聽風雲3》開拍<ref>[http://ent.sina.com.cn/m/c/2013-06-03/21583935290.shtml 《窃听风云3》开机 吴彦祖笑称不想死] 2013年06月03日 新浪娱乐</ref>，確定仍由刘青云、古天乐及吴彦祖繼續主演，而[[方中信|方中信]]、[[黄奕|黄奕]]及[[叶璇|叶璇]]等前兩集的相關演員將擔任配角。同年8月3日，出品方博納影業集團總裁-{于}-冬宣布，由原班人馬打造的《竊聽風雲3》定下女主角，[[周迅|周迅]]是此片的正牌女一號，並說明此次是罕見的開機後才決定<ref>[http://ent.southcn.com/8/2013-08/06/content_75620556.htm 周迅出演《竊聽風雲3》女一號開機後才確定] {{Wayback|url=http://ent.southcn.com/8/2013-08/06/content_75620556.htm |date=20130922041338 }} 2013年8月6日 南方網</ref> 。關於周迅的加盟，古天樂表示：「周迅跟吳彥祖在螢幕上談過兩次戀愛、當過劉青雲的七太太，我只跟她合拍過廣告，期待和她在大銀幕上合作好久了」。周迅並在此片全篇以粵語演出<ref>[http://dailynews.sina.com/bg/ent/film/sinacn/20130804/20494825169.html 周迅演《竊聽風雲3》說粵語演香港人] 2013年8月4日 新浪娛樂</ref>，飾演育有一子的寡婦。

=== 拍攝===
報章《[[佛山日報|佛山日報]]》指《窃听风云3》計劃於2012年拍攝<ref>[http://news.cntv.cn/20120502/110243.shtml 《竊聽風雲3》今年開機] 2012年05月02日 中國网絡電視台</ref>，惟過去從來沒有證實。2013年3月7日，電影的製作正進入初步階段，並計劃於4月拍攝<ref>[http://ent.takungpao.com.hk/movie/q/2013/0307/1477619.html 馮小剛2013新作開拍 王朔葛優「鐵山角」再出馬] 2013年3月7日 廣州日報</ref>，但仍然未實行。直到於同年6月3日，該片舉行開鏡記者會，宣布在香港正式開機<ref>[http://news.mtime.com/2013/05/28/1512366.html 《窃听风云3》首曝海报 三男主回归6月3日开机] 2013年5月28日 Mtime時光網</ref><ref>[http://news.sina.com.tw/article/20130603/9824371.html 《竊聽風雲3》開機 投資超前兩部總和] 2013年6月3日 北京新浪網</ref>。於同年8月14日，《竊聽風雲3》召開記者發布會，全片正式殺青<ref>[http://news.mtime.com/pix/2013/08/15/396485.html   "窃听风云3"杀青会遇台风 刘青云周迅曝"恋情"] 2013年8月14日 Mtime時光網</ref>。

== 宣傳 ==
《窃听风云3》的首款海報於2013年3月21日釋出，在香港国际影视展裡的[[发行工作室|发行工作室]]展厅外出現<ref>[http://news.mtime.com/pix/2013/03/21/356159.html 香港国际影视展将落幕 吴宇森新片等曝光概念海报] 2013年3月21日 时光网</ref>。2013年10月8日，製片方公布第一款「正式版」宣傳海報，但主演的[[吳彥祖|吳彥祖]]、[[古天樂|古天樂]]、[[周迅|周迅]]未出現在其中，只有[[劉青雲|劉青雲]]和[[方中信|方中信]]等人。《竊聽風雲3》陣容集結前兩部的演員，一共有20多個角色。在此前的開機發布會上，導演[[麥兆輝|麥兆輝]]與[[莊文強|莊文強]]表示，第三部創作會做出很大程度的改變，「我們以前拍的東西都比較安靜。不過兩年前有次和[[林嶺東|林嶺東]]喝酒，他說你看我電影裡每個人都很衝動，你倆的就不會，然後忽然覺得，我們也有可能去拍一個很野蠻、很衝動的電影，於是就有了現在的故事。」<ref>[http://news.mtime.com/pix/2013/10/08/405325.html 《窃听风云3》曝土豪版海报 刘青云吴孟达等亮相] 2013年10月8日 时光网</ref>

2013年12月12日，片方釋出首波前導預告。監制黃斌表示，服裝設計都是比較符合角色的習慣，穿著不太搭調的奢侈服裝，手裡拿著巨額的賭場籌碼和運用更尖端的監視設備。而且圖像竊聽，可令角色互相制約的程度更深入，也令關係更為難以捉摸<ref>[http://news.mtime.com/pix/2013/12/12/414178.html 《竊聽風雲3》曝先導預告＆新劇照 12人全陣容亮相]2013年12月12日 時光網</ref>。

2014年5月13日，片方在報章、電視及網上以「陸國集團」(01465)招股為名推出一系列廣告，並加上「管理層」劉青雲、林家棟、方中信等人物簡介，成功吸引不少人關注。<ref>[http://lifestyle.etnet.com.hk/column/voting.php?id=411 《竊聽風雲3》以IPO形式賣廣告，你認為這宣傳方式有創意嗎？ etnet 2014-05-30]</ref>

== 评价 ==
《[[新京报|新京报]]》认为：“人与土地的关系是电影戏核。从格局上来说，《窃3》的野心要远远超越两部前作。由“[[丁权|丁权]]”所引发的利益纠纷是电影的主线，有的人被贪婪和欲望所驱使疯狂囤地；有些人因无知、堕落靠卖地过活；有些人则一直在坚守祖先留下的遗产，任凭岁月的变迁我自岿然不动。这三种人构成了电影中矛盾的三方。蔓延开来的是，兄弟情、邻里情、青梅竹马情、贪婪和欲望、背叛和复仇，再加上一点点怀旧和乡土气息。就其故事内核而言，《窃3》完全可以写成一出80集左右的港剧，其精彩程度预计能比肩当年红遍大江南北的《[[创世纪|创世纪]]》。令人感到遗憾的是，电影的容量有限，编剧对故土和乡亲、邻里、手足的浓浓深情不得不一次又一次的点到为止。兄弟之情不慍不火，男女之情不咸不淡，贪婪只剩下盲目，复仇也找不到动机，反面角色一二到底，正面角色好到没有人性。故事是个好故事，只是想说的太多，结果就是什么都没说明白。”<ref >{{cite web|url=http://ent.sina.com.cn/r/m/2014-06-04/11384152842.shtml | language= | title=评《窃听风云3》：不如拍成80集的港剧 |publisher= 新京报 |date=2014年6月4日 |accessdate= }}</ref>

不少观众认为讲述房地产勾心斗角现象的本片有大的视野和比较强的现实意义，剧情有些虎头蛇尾，质量不如前两部。<ref >{{cite web|url=http://ent.sina.com.cn/r/m/2014-06-04/02304152558.shtml | language= | title=评《窃听风云3》：地产风云下的人性拷问 |publisher= 新京报 |date=2014年6月4日 |accessdate= }}</ref>
