== 利用狀況 ==
*目前為[[臺灣鐵路管理局車站等級#二等站|二等站]]，主要停靠[[臺鐵區間車|區間車]](含[[集集線|集集線]])、[[台鐵區間快車|區間快車]]、莒光號為主，少數[[自強號列車|自強號]]也有停靠。
**[[自強號列車|自強號]]停靠有：南下(逆行)177(6177)次，北上(順行)6126次(週日行駛)、170次。
**[[莒光號列車|莒光號]]：除了[[觀光列車|環島觀光列車]]外均有停靠。
**區間快車、區間車皆停靠
*[[集集線|集集線]]於本站南下後，與縱貫線(南段)分歧。
*本站有販售至集集線各站之[[埃德蒙森式鐵路車票|名片式車票]]，以套票形式販售。
*本站開放使用[[悠遊卡|悠遊卡]]、[[一卡通_(台灣)|一卡通]]及[[icash|icash 2.0]]付費。

== 歷史 ==
*1905年1月15日：設「二八水驛」。<ref name=":0">{{cite book |author = 戴震宇 |authorlink =  |title = 《台灣的老火車站》 |publisher = 遠足文化 |series = 台灣地理百科 |year = 2005 |isbn = 957304935X }}</ref>
*1921年：改為「二水」。<ref name=":0" />
*1935年：拆除原木造站房，改建混凝土站房，沿用至今。
*2015年6月30日：啟用多卡通刷卡機。
