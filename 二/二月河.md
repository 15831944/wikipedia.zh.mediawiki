'''凌解放'''（{{bd|1945年|11月3日|2018年|12月15日|catIdx=Ling凌}}），[[笔名|笔名]]'''二月河'''<ref name=" wolf" />，取意“奔腾不息的二月[[黄河|黄河]]”{{notetag|农历二月份黄河冰面开裂，冰凌消融，故笔名二月河和真名凌解放暗合。}}，[[河南|河南]][[南阳|南阳]]人，出生于[[山西|山西]][[昔阳|昔阳]]，[[中國大陸|中國大陸]]作家、教授、中共[[中国共产党第十五次全国代表大会|十五大]]、[[中国共产党第十六次全国代表大会|十六大]]、[[中国共产党第十八次全国代表大会|十八大]]、[[中国共产党第十九次全国代表大会|十九大]]代表。因创作大量帝王系列小说、历史小说而知名。诸如[[康熙帝|康熙帝]]、[[雍正帝|雍正帝]]、[[乾隆帝|乾隆帝]]等等。曾担任南阳市文联副主席、河南省作協副主席、[[郑州大学|郑州大学]]文学院院长等职务。是河南省优秀专家，享受[[中国国务院|中国国务院]][[国务院政府特殊津贴|政府特殊津贴]]。

== 人生经历 ==
=== 早年生涯 ===
1945年11月3日二月河出生于山西昔阳，父母是昔阳县武委会主任凌尔文和妇联主席马翠兰。随着国共内战形式变化，二月河跟随父母辗转河南各地。[[中国人民解放军|中国人民解放军]]占领南阳后，其父母分别留任邓县武装部政委和法院副院长。就此全家在南阳定居。<ref name="eyhyl">{{cite web|url=http://news.ifeng.com/a/20140630/40950468_1.shtml|title=二月河：反腐很累 王岐山在我眼里是英雄|publisher=凤凰网|date=2014年6月30日 | language = zh-hans }}</ref>小学至初中阶段开始接触《[[三国演义|三国演义]]》、《[[水浒传|水浒传]]》、《[[西游记|西游记]]》等中国古典文学作品，以及《[[钢铁是怎样炼成的|钢铁是怎样炼成的]]》、《[[汤姆·索亚历险记|汤姆·索亚历险记]]》等外国名著，高中时期，他开始阅读《[[红楼梦|红楼梦]]》。<ref name=" wolf" />

1967年二月河[[高中|高中]]毕业时中国处于[[文化大革命|文化大革命]]时期，他家亦未能幸免被波及，被红卫兵连续抄家3次。经此一劫，二月河选择去山西太原参军，成为一名工程兵。1969年12月加入[[中国共产党|中国共产党]]。1978年部队转业至[[河南|河南]][[南阳|南阳]]，在南阳市委宣传部任职，利用业余时间研究一直热爱的《红楼梦》，并将研究成果寄到相关刊物，却石沉大海。后在中国红学专家冯其庸的支持下，成为中国红学会会员。<ref name="eyhyl"/>

=== 小说创作 ===
1982年，凌解放到上海参加研讨会，会上学者感慨没有有关康熙皇帝的优秀作品时，二月河便开始了“帝王小说”的写作之路。<ref name="eyhyl"/>1988年，4卷160余万字的《康熙大帝》完稿，[[冯其庸|冯其庸]]看后说，专心致志完成你未来的路。<ref>https://new.qq.com/cmsn/20181215/20181215006900.html</ref>第一卷《夺宫》出版后引起轰动，香港、台湾相继推出繁体竖排版。1990年至1992年，又陆续创作《雍正皇帝》3卷140余万字。1998年，根据此书改编的电视剧在央视播出。1994年至1996年，二月河完成《乾隆皇帝》前3卷130余万字。康熙、雍正、乾隆三朝的历史在他的笔下构成了一幅壮阔的政治长卷。至2006年版稅收入至少1200萬元人民幣，為該年度中國作家的第二名。<ref>{{cite news|title=大陸富豪作家排行 余秋雨占鰲頭|author=彭志平|url=http://news.chinatimes.com/Chinatimes/newslist/newslist-content/0,3546,110505+112006121700060,00.html|date=2006年12月16日|accessdate=2006年12月17日|publisher=中國時報|deadurl=yes|archiveurl=https://web.archive.org/web/20070103224334/http://news.chinatimes.com/Chinatimes/newslist/newslist-content/0,3546,110505+112006121700060,00.html|archivedate=2007年1月3日}}</ref> 2011年6月27日，出任[[郑州大学|郑州大学]]文学院院长<ref>[http://www4.zzu.edu.cn/arts/show.aspx?ID=1728&cid=45 二月河先生赴文学院任职] {{webarchive|url=https://web.archive.org/web/20110902165238/http://www4.zzu.edu.cn/arts/show.aspx?cid=45&id=1728 |date=2011-09-02 }}</ref>。

== 言论观点 ==
2014年5月22日，二月河在接受《人民日报社》旗下的《环球人物》杂志专访时，阐述了自己对中国存在的腐败现象和反腐的看法<ref name="eyhyl"/>：

{{cquote|
（反腐）除严刑峻法外，还需要别的措施。从长远看，整个社会需要反特权意识的觉醒和身体力行。现在[[小学生|小学生]]都知道担任班干部有好处，可以管理别人，可以从老师那里享受到不同待遇，于是他们小小年纪就知道[[贿选|贿选]]，买东西送给同学。孩子从小就知道利益交换。这种现象非常危险。中国的社会教育、学校教育、家庭教育都严重缺失，传统文化中对[[权力|权力]]的迷信、对权威的崇拜根深蒂固，做事首先想到的是行贿受贿、旁门左道。| }}

{{cquote|
我跟[[王岐山|王岐山]]同志说过一个例子：[[满洲人|满洲人]]入关时8.5万人，加上[[吴三桂|吴三桂]]在[[山海关|山海关]]的3.5万驻军，一共不过12万。而[[汉族|汉族]]方面，仅[[李自成|李自成]]的铁骑就有100余万人，加上[[朱聿键|南明唐王]]逃到[[福建|福建]]称帝时手中的200余万人马，以及散落全国的汉族武装力量，总数能超过400万。可最后13万人打败了400万人。与满人相比，汉人的制度不先进吗？当然先进。只能说，如果你腐败，先进制度下的400万人也是一堆臭肉。不腐败，落后制度下的13万人也能变成一把剁肉的刀。| }}

{{cquote|
不能迷信经济实力。政治腐败不能拿经济繁荣去抵消。一个政权如果不能治理腐败问题，其他方面再强大，都不能成为一个强大的国家。因为腐败就像是社会的[[糖尿病|糖尿病]]，它是一个富贵病，隐蔽性很强，不会直接导致社会死亡，但是在不知不觉中会成为社会的一大顽疾，使国家变得极其脆弱，最后很容易引起并发症，不堪一击，无从抢救。如果不把腐败的血糖降下来，不管你是什么制度，不管你有多高的GDP，都会轰然倒塌，彻底完蛋。| }}

{{cquote|
现在，普通[[公务员|公务员]]工作稳定，有较高的社会地位，收入足以养家糊口。如果国家还用高薪去迎合那些蝇营狗苟、追名逐利的人，人们对这个职业的追逐将更疯狂，会花更大代价去走后门、贿赂官员。何况工资再高，也无法与贪污受贿的数额相提并论。对于贪欲无穷的人来说，给多高的薪水都是不够的。| }}

== 主要著作 ==
二月河主要从事历史小说的创作，著名的代表作有《[[康熙大帝_(小说)|康熙大帝]]》、《[[雍正皇帝_(小说)|雍正皇帝]]》、《[[乾隆皇帝_(小说)|乾隆皇帝]]》三部曲，其中《雍正皇帝》被评为[[20世纪中文小说100强|20世纪中文小说100强]]中的第100位。2003年《乾隆皇帝》荣获[[姚雪垠长篇历史小说奖|姚雪垠长篇历史小说奖]]。<ref name="lion"/>这三部小说先后被改编成电视剧，取得较高收视率。另外二月河还出版了随笔、散文集《二月河语》，短篇小说集《匣剑帷灯》。
  
《康熙大帝》改编成电视剧《[[康熙大帝_(电视剧)|康熙大帝]]》、《[[康熙王朝|康熙王朝]]》；《雍正皇帝》改编成电视剧《[[雍正王朝|雍正王朝]]》；《乾隆皇帝》改编成电视剧《[[乾隆王朝|乾隆王朝]]》。　

== 逝世 ==
二月河在2018年上半年曾因脑栓塞到[[北京301医院|北京301医院]]住院治疗，后转到普通病房。在12月15日凌晨2点左右突发心衰，并进行了抢救，<ref>{{Cite web |url = https://baijiahao.baidu.com/s?id=1619880119938608282&wfr=spider&for=pc |title = 著名作家二月河今日凌晨突发心衰，家属：情况不容乐观 |publisher = 澎湃新闻（百家号） |date = 2018年12月15日 |accessdate = 2018年12月15日 }}</ref>但最终于当日凌晨逝世，享壽73岁。<ref name="multiple" />追悼会定于19日在南阳市殡仪馆举行<ref>{{Cite web |url = https://ent.sina.cn/tv/tv/2018-12-15/detail-ihqhqcir4667432.d.html|title = 二月河追悼会19日举行 曾留心愿不希望打扰太多人 |publisher = 新浪网 |date = 2018年12月15日 |accessdate = 2018年12月15日 }}</ref>。
