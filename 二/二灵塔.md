'''二灵塔'''是[[中国|中国]][[浙江省|浙江省]][[宁波市|宁波市]][[鄞州区|鄞州区]]境内的一处古代佛塔，位于[[东钱湖|东钱湖]]东南二灵山上，现为[[宁波东钱湖旅游度假区|宁波东钱湖旅游度假区]]（东钱湖区）实际管辖。二灵塔建于北宋[[政和_(年号)|政和]]年间，为浙江省范围内罕见的北宋石塔，也是北宋石构建筑的代表。1997年8月，二灵塔被列入[[浙江省文物保护单位|浙江省文物保护单位]]，2013年3月成为[[全国重点文物保护单位|全国重点文物保护单位]]<ref name="nbwb-site">{{cite web|url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=24748&type=2|title=二灵塔|accessdate=2013-09-09|author=|date=2013-05-06|publisher=宁波市文物保护管理所}}</ref>。

== 形制 ==
二灵塔位于东钱湖东南蛇形半岛二灵山西北的山脊上，塔下湖边有二灵寺，寺与塔构成“钱湖十景”中的“二灵夕照”。二灵塔为仿木[[楼阁式塔|楼阁式塔]]，由石块堆叠而成。塔总高9.98米，内部中空无法攀登。塔由平台、基座、塔身、塔刹组成。平台长7.5米，宽6.5米，高0.45米，由条石砌成，左右有石凳，有两级台阶可上下。基座为[[须弥座|须弥座]]，高1.78米，最宽处2.47米，北侧有方门可进入。基座圭角四周刻对称素线云卷，束腰上刻莲花，底层东壁有“[[政和_(年号)|政和]]□年”字样。塔身高7.4米，共七层，层高从底部向上依次为1.22米、1.14米、1.1米、1.06米、1.00米、0.98米、0.9米。塔每层有腰檐，檐边长2.35米，檐角略上翘，有云纹装饰，尖端有悬挂风铃的圆孔。塔身每层四周刻有仿窗浮雕，内有浮雕像，其中共有坐佛39尊，金刚3尊。一层还刻有“福延圣寿”和“保国爱民”字样。塔刹为方柱形，上小下大，造型较为简单，下有石板砌成的平台座<ref name="杨晓维">{{Cite journal|title=宁波二灵塔的建造艺术及内涵价值|author=杨晓维|journal=浙江建筑|issue=1|year=2016|volume=33|page=4-9}}</ref>。塔底曾被盗掘，根据被盗情况判断，塔下并无地宫<ref name="段菁菁">{{Cite web|url=http://news.ifeng.com/gundong/detail_2013_11/05/30978626_0.shtml|title=宁波一北宋年间古塔出现盗洞 警方已介入调查|accessdate=2018-04-28|author=段菁菁|date=2013-11-05|publisher=新华网|archive-url=https://web.archive.org/web/20180501005111/http://news.ifeng.com/gundong/detail_2013_11/05/30978626_0.shtml|archive-date=2018-05-01|dead-url=no}}</ref>。

== 传说与沿革 ==
相传，二灵山曾有巨蟒乘雾出没，伤害农人和船只，后有高僧筑宝塔镇蛇，此塔即为二灵塔。根据《天童寺志》载，[[钱元瓘|钱元瓘]]于[[后唐|后唐]][[长兴_(年号)|长兴]]至[[后晋|后晋]][[天福_(后晋)|天福]]年间修建此塔。民国《东钱湖志》记载，北宋时，[[陈禾|陈禾]]曾在此筑庵读书，建寺和塔，后[[雪窦山|雪窦山]]知和禅师在此居住，有二虎相伴，因而寺院称为二灵寺，塔以寺名，禅师后葬于塔旁。而民国《鄞县通志》载，塔由韶国师建于北宋初年。寺院曾有多位高僧停留，明代改为[[天童寺|天童寺]]辖寺，后寺毁，当下寺院为后世兴建，仅剩塔为北宋原物。清[[康熙|康熙]]二十年（1681年），天童寺德介禅师主持重修二灵塔<ref>{{Cite web|url=http://daily.cnnb.com.cn/dnsb/html/2008-12/28/content_51023.htm|title=山灵水灵二灵塔|accessdate=2018-05-01|author=张落雁|author2=杨古城|date=2008-12-28|publisher=东南商报|archive-url=https://web.archive.org/web/20180501161209/http://daily.cnnb.com.cn/dnsb/html/2008-12/28/content_51023.htm|archive-date=2018-05-01|dead-url=no}}</ref>。至20世纪80年代，二灵塔除西北二角塌落外，其余保存基本完好，1986年落架大修后，二灵塔恢复原貌<ref name="杨晓维" />。

== 保护 ==
{{external media|float=|width=250px|image1=[http://www.nbwb.net/UploadFile/images/2018012618350982.jpg 二灵塔保护区划图]，原始内容[https://web.archive.org/web/20180424141614/http://www.nbwb.net/UploadFile/images/2018012618350982.jpg 存档]于2018年4月24日}}
二灵塔于1982年6月被列入鄞县文物保护单位，县政府曾于1986年对二灵塔实施落架大修。1997年8月，二灵塔被列入第四批[[浙江省文物保护单位|浙江省文物保护单位]]<ref name="杨晓维" />。2013年3月，二灵塔被列入[[第七批全国重点文物保护单位|第七批全国重点文物保护单位]]。保护范围为塔身及塔基，建设控制地带确定为塔基向西21米，向北、东、南各20米<ref>{{Cite web|url=http://www.nbwb.net/pd_wwbh/info.aspx?Id=24748&type=2|title=保护区划 - 二灵塔|accessdate=2018-04-28|author=|date=2013-05-06|publisher=宁波市文物保护管理所|archive-url=https://web.archive.org/web/20160304110955/http://www.nbwb.net/pd_wwbh/info.aspx?Id=24748&type=2|archive-date=2016-03-04|dead-url=no}}</ref>。

2013年10月，二灵塔附近发现一盗洞，为盗贼试图偷窃地宫宝物所挖<ref name="段菁菁" />。2014年，二灵塔附近由东钱湖区安装24小时监控摄像头以避免文物遭到破坏<ref>{{Cite web|url=http://www.dqlake.gov.cn/ldzc/info.aspx?Id=17484|title=史济权调研国家级文物保护单位——二灵塔|accessdate=2018-04-28|date=2014-05-09|publisher=中国宁波东钱湖旅游度假区管委会|archive-url=https://web.archive.org/web/20180501093047/http://www.dqlake.gov.cn/ldzc/info.aspx?Id=17484|archive-date=2018-05-01|dead-url=no}}</ref><ref>{{Cite web|url=http://nb8185.cnnb.com.cn/html/1854/2017/0401/824396.html|title=东钱湖余有丁墓道石刻任其破败，原因是……|accessdate=2018-03-27|author=邱韵|date=2017-04-01|work=|publisher=中国宁波网|archive-url=https://web.archive.org/web/20180329184404/http://nb8185.cnnb.com.cn/html/1854/2017/0401/824396.html|archive-date=2018-03-29|dead-url=no}}</ref>。
