'''二里沟站'''位于[[北京市|北京市]][[海淀区|海淀区]]与[[西城区|西城区]]交界处，是[[北京地铁|北京地铁]][[北京地铁6号线|6号线]]一期和在建的[[北京地铁16号线|16号线]]的一座[[换乘站|换乘站]]。6号线一期开通时二里沟站没有投入使用。待16号线南段（西苑站以南）2019年年底建成通车时，6号线二里沟站才会同时投入使用。

== 历史 ==
最初6号线并没有二里沟站。二里沟站是为了解决6号线和16号线换乘问题，在16号线规划定案后添加的。<ref>{{cite news|url=http://zhengwu.beijing.gov.cn/bmfu/bmts/t1165407.htm|title=16号线设10个地下换乘站|date=2011-05-06|author=耿诺|accessdate=2012-02-24|deadurl=yes|archiveurl=https://web.archive.org/web/20110817082130/http://zhengwu.beijing.gov.cn/bmfu/bmts/t1165407.htm|archivedate=2011-08-17}}</ref>

== 位置 ==
二里沟站位于[[三里河路|三里河路]]（南北向）和[[车公庄西路|车公庄西路]]－[[车公庄大街|车公庄大街]]（东西向）交口，6号线车站呈东西向布置，16号线车站呈南北向布置。<ref>{{cite web|url=http://news.163.com/12/0209/02/7PPOGBJ700014AED.html|title=6号线16号线增二里沟换乘站|date=2012-02-09|accessdate=2012-02-24}}</ref>

== 结构 ==
两线均为地下车站。
