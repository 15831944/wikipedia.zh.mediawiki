'''二條站'''（{{jpn|j=二条駅|hg=にじょうえき|rm=Nijō eki|f=y}}）是位於[[日本|日本]][[京都府|京都府]][[京都市|京都市]][[中京區|中京區]]西之京栂尾町，由[[西日本旅客鐵道|西日本旅客鐵道]]（JR西日本）及[[京都市交通局|京都市交通局]]所經營的[[鐵路車站|鐵路車站]]。

==簡介==
二條是JR西日本的[[嵯峨野線|嵯峨野線]]（[[山陰本線|山陰本線]]部分路段在營運時所使用的暱稱）與[[京都市營地下鐵|京都市營地下鐵]][[東西線_(京都市營地下鐵)|東西線]]兩條路線的交會車站，其在JR系統內所使用的車站編號為'''JR-E04'''，而在市營地下鐵系統中則稱為'''T15'''。

在票務上，二條是[[特定都區市內|特定都區市內]]制度中「京都市內」區域的一部份，屬{{link-ja|西日本旅客鐵道京都支社|西日本旅客鉄道京都支社|JR西日本京都支社}}管轄範圍，是由{{ST|龜岡|1}}管理的[[直營車站|直營車站]]，包含[[特急列車|特急列車]]在內的全車等都有在本站停車。

兩家鐵路公司所經營的站區皆可通用[[ICOCA|ICOCA]]、[[PiTaPa|PiTaPa]]以及與這兩種[[智慧卡|智慧卡]]互通的其他電子票卡，至於京都市交通局自行發行的[[磁性票卡|磁性票卡]]Traffica京卡（{{lang|ja|トラフィカ京カード}}）則只能於市營地下鐵的站區內使用。

現時車站是擁有一個[[島式月台|島式月台]]的[[高架車站|高架車站]]，車站大堂的屋頂是以木造構架構成。跟上一代的車站大堂比較，現時的設計雖然有著正反兩論，但設計曾獲得1997年度的[[良好設計獎|良好設計獎]]，在車站月臺上更可以遙望遠方的[[比叡山|比叡山]]等等的山脈。而在高架化前,一般只能由千本通路的入口進入車站。

由剪票口往月臺設有兩個[[樓梯|樓梯]]、一個[[升降機|升降機]]及一個往上的[[電動扶梯|電動扶梯]]。另外，在剪票口前有Kiosk {{tsl|ja|ハートイン|Heart·in}}，而月臺上則有等待室和自動販賣機，但並沒有小賣店之類的店舖。

在從前的地上車站時代，以建造得非常有氣派的車站大堂為全日本所知曉，大堂除了以莊嚴社寺為藍本外，也設有貴賓室等等設施。這個大堂也同時是當時{{tsl|ja|京都鉄道|京都鐵道}}的總部所在，由建築家[[伊東忠太|伊東忠太]]所設計。當初雖然預定以磚瓦建設車站，但為了配合二條城周邊的景觀而變更成日式風格，亦有言論指是模仿當時[[東北本線|東北本線]]中[[宇都宮站|宇都宮站]]大堂的優美。<ref>明治37年2月11日大阪朝日新聞附錄。從梅小路蒸氣火車博物館的展示資料二次引用。</ref>

隨著二條 - 花園之間的高架化工程於1990年10月開始，為了不阻礙建設臨時路軌的工程，舊車站大堂進行了為期兩天，向其東方移動15米的曳屋工程（把建築物整個從原所在地移動到別地），此後大堂仍然作為臨時大堂使用。當高架化工程及新車站大堂於1996年完成的同時，舊大堂的使命也正式完結。同年4月1日舊大堂以象徵京都的近代化建築，成為京都市指定的[[日本文化遺產|文化遺產]]，是[[明治時期|明治時期]]中以日式風格所建造的車站大堂中的唯一殘存例子，並將其移往[[梅小路蒸氣火車博物館|梅小路蒸氣火車博物館]]復原。<ref>高架化工程與舊大堂的曳屋工程、轉移等等的資料來自梅小路蒸氣火車博物館。</ref>。

2002年獲選為第3回[[近畿車站百選|近畿車站百選]]入選車站。

==== 歷史 ====
* 1897年2月15日：京都鐵道本站～[[嵯峨嵐山站|嵯峨站]]之間開通，同時開業。
* 1897年4月27日：京都鐵道大宮站～本站開通。
* 1904年6月：兼備京都鐵道總部的車站大堂完成。
* 1907年8月1日：京都鐵道[[國有化|國有化]]。
* 1912年6月11日：{{link-ja|京都市電千本線|京都市電千本線}}壬生車庫前-千本丸太町間開業，設置二條車站前停留所。
* 1972年1月23日：京都市電千本線廢止、二條站前停留所也同時廢除。
* 1987年4月1日：[[國鐵分割民營化|國鐵分割民營化]]，成為[[西日本旅客鐵道|西日本旅客鐵道]]、[[日本貨物鐵道|日本貨物鐵道]]的車站。
* 1990年10月：因為曳屋工程，車站大堂往東則移動15m。
* 1996年3月16日：車站高架化。同時廢除[[貨物列車|貨物列車]]的設定。從前車站西側有一個小規模的貨物[[調車場|調車場]]存在、線道往南方，越過三條通路往[[日本食糧倉庫|日本食糧倉庫]]與油庫的{{link-ja|専用線|専用鉄道}}伸延。舊大堂遷移往梅小路蒸氣火車博物館。
* 2006年4月1日：JR貨物取消於此站運送貨物。
* 2010年1月31日：[[丹波口站|丹波口]]～本站複線化，交換設備廢除。

== 使用情況 ==
雖然臨近[[二條城|二條城]]，但因為更多觀光客從[[京都市營地下鐵|地下鐵]][[二條城前站|二條城前站]]或直接搭乘公車出入二條城，使用車站的觀光客反而並不多，相比下最常用車站的是鄰近地區的居民及學生。本站也是往京都府北部（丹後）的特急車輛的停車車站，從京都市中心搭乘公車或地鐵到本站轉乘往北部地方的人也不少。

== 車站周邊 ==
[[File:Ritsumeikan_Suzaku_Campus.JPG|thumb]]
以往JR車站只有東邊千本通路的出入口，但由於地下鐵[[京都市營地下鐵東西線|東西線]]的開通，以及[[山陰本線|山陰本線]]（[[嵯峨野線|嵯峨野線]]）的高架化，西側也得以重新整頓，近年更有公車和計程車途經。

車站東側的千本通路附近，2000年9月開設了京都府齒科醫師會口腔保健中心，而JR二條車站NK大樓亦於2006年3月1日開始營業。西側方面，包含{{tsl|ja|TOHOシネマズ|TOHO電影院}}等等複合娯樂設施的{{tsl|ja|BiVi二条|BiVi二條}}於2005年6月25日開業。車站東南所在的千本[[三条通|三條]]交差點西北角的[[立命館大學|立命館]]朱雀學園（[[中川會館|中川會館]]）於2006年9月開設，{{tsl|ja|学校法人立命館|學校法人立命館|學園}}本部與[[法科大學院|法科大學院]]等等搬遷至此。車站東南仍然擁有廣闊用地，現正就此地的使用作出檢討。2008年8月，有報道指此區域的南半部有將會興建佛教大學新校園的計劃。另外，北半部亦計劃興建京都府醫師會的新府醫師會館。

千本通路連結車站東側的南北道路。而連結東西方向的[[御池通|御池通路]]則由於被本車站分斷，行經車輛需向車站北方，高架下的{{tsl|ja|押小路通|押小路通|押小路通路}}迂迴走行再回到御池通路。車站東側以千本通路、御池通路為中心，沿途擁有為數不少的商店。西側則以[[西大路通|西大路通路]]連結住宅地與中小工場的密集地帶，而車站西北也有二條駕駛學校。直至鄰近的[[圓町站|圓町站]]，隨著山陰本線的高架化，道路也重新舖設過，可以使用側道步行來往兩個車站。車站周邊正以成為京都市的副都心努力。

[[神泉苑|神泉苑]]、[[二條城|二條城]]、中京臨總合廳舍（區役所）位於車站東方的300～500m。乘搭地下鐵的話則是[[二條城前站|二條城前站]]距離比較近。

其他公司的路線則以以下車站為最接近車站。
* [[京福電氣鐵道|京福]][[京福電氣鐵道嵐山本線|嵐山本線]] {{tsl|ja|西大路三条駅|西大路三條站}} 徒步15分左右
* [[阪急電鐵|阪急]][[阪急京都本線|京都本線]] {{tsl|ja|大宮駅 (京都府)|大宮站 (京都府)|大宮站}} 徒步15分左右
