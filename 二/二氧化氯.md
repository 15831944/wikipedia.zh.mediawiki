'''二氧化氯'''是黃绿色的[[氣體|氣體]]，是[[氯|氯]]的最稳定的[[氧化物|氧化物]]，也是唯一大量生产的卤氧化物<ref>《无机化学》（第五版）. 大连理工大学无机化学教研室 编. 高等教育出版社. 第三篇 元素化学. P<sub>524</sub>. 1.卤素的氧化物</ref>。二氧化氯在其液態時穩定，但若和一些特定物質接觸，也有爆炸的可能。 它在约−59°C 时凝结成亮橙色的晶体。作为[[氯的氧化物|氯的氧化物]]之一，它是一种极其强而有效的[[氧化剂|氧化剂]]，被用于水处理和漂白。<ref>{{Greenwood&Earnshaw2nd|pages=844–849}}</ref>

==制备==

[[氯酸钠|氯酸钠]]在強酸的溶液和適合的還原劑（例如[[过氧化氢|过氧化氢]]、[[二氧化硫|二氧化硫]]，或[[盐酸|盐酸]]）下進行還原反應，產生二氧化氯：

:2ClO<sub>3</sub><sup>−</sup> + 2Cl<sup>−</sup> + 4H<sup>+</sup> → 2ClO<sub>2</sub> + Cl<sub>2</sub> + 2H<sub>2</sub>O

然而，這個化學反應對於消毒上的應用價值不大。生產消毒用的二氧化氯，大多數是用[[亚氯酸盐|亚氯酸盐]]—氯法：

:2NaClO<sub>2</sub> + Cl<sub>2</sub> → 2NaCl + 2ClO<sub>2</sub>

或是亚氯酸盐-盐酸法：

:5NaClO<sub>2</sub> + 4HCl → 5NaCl + 4ClO<sub>2</sub> + 2H<sub>2</sub>O

後者的特點是它的化學製成品無氯氣以及產量高（用現代的生產機器，一般含量高于95%）。

最近，科学家又研究出了一种新的制备方法，在酸性溶液中用草酸（H<sub>2</sub>C<sub>2</sub>O<sub>4</sub>）还原氯酸盐，化学反应方程式为：

:H<sub>2</sub>C<sub>2</sub>O<sub>4</sub> + 2NaClO<sub>3</sub> + H<sub>2</sub>SO<sub>4</sub> → Na<sub>2</sub>SO<sub>4</sub> + 2CO<sub>2</sub>↑ + 2ClO<sub>2</sub>↑ + 2H<sub>2</sub>O

:2KClO<sub>3</sub> + H<sub>2</sub>C<sub>2</sub>O<sub>4</sub> → K<sub>2</sub>CO<sub>3</sub> + CO<sub>2</sub>↑ + 2ClO<sub>2</sub>↑ + H<sub>2</sub>O

此法的最大特点是由于反应过程中生成的二氧化碳的稀释作用，大大提高了生产及储存、运输的安全性。当二氧化氯的分压超过6.6kPa时会发生爆炸。

==化学性质==
二氧化氯和[[氢氧化钠|氢氧化钠]]反应，得到[[亚氯酸钠|亚氯酸钠]]和[[氯酸钠|氯酸钠]]。
==用途==

二氧化氯的主要用途在自來水的[[消毒劑|消毒]]，和[[麵粉|麵粉]]與[[木質紙漿|木質紙漿]]的[[漂白劑|漂白]]。

它能很有效地對抗[[病毒|病毒]]、[[細菌|細菌]]和包括[[梨形鞭毛蟲|梨形鞭毛蟲]]（學名：{{lang|la|''Giardia lamblia''}}）與[[隱孢子蟲|隱孢子蟲]]（Cryptosporidium）等[[原生動物|原生動物]]所引起的囊腫與卵囊。

從1956年[[比利時|比利時]]的[[布魯塞爾|布魯塞爾]]把自來水消毒劑由氯氣改成二氧化氯後，二氧化氯開始被廣泛地使用。

2005年8月，在襲擊[[紐奧良|紐奧良]]的-{zh-tw:[[颶風卡崔娜|颶風卡崔娜]]; zh-cn:[[飓风卡特里娜|飓风卡特里娜]]}-過後，二氧化氯就被用來清除被洪水淹沒後房屋上的危險[[霉菌|霉菌]]。而中華民國政府也於2006年3月6日預告二氧化氯為自來水消毒劑。[http://gazette.nat.gov.tw/EG_FileManager/eguploadpub/eg012041/ch07/type3/gov60/num1/Eg.htm]
