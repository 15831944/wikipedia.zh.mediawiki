'''《二十點協議》'''（{{lang-ms|Perjanjian 20 Perkara Sabah}}；{{lang-en|20-point agreement}}）是[[北婆羅洲|北婆羅洲]]（今[[沙巴|沙巴]]）领袖在英国伦敦谈判结束之后，对于成立[[马来西亚|马来西亚]]事宜所提呈的基本诉求文件。这份诉求文件并无透过法律程序在立法议会中通过成为一个法律条文，是一份不具有法律地位的备忘录。<ref>{{Cite news|url=http://www.sinchew.com.my/node/1587041|title=阿莎丽娜：“联邦”概念组成大马．东西马地位同等|author=2016-11-15|date=2016-11-15|work=星洲日报|accessdate=}}</ref><ref>{{Cite news|url=http://news.seehua.com/?p=201167|title=保护砂沙权益 马来西亚契约受承认|author=诗华日报|date=2016-08-26|work=诗华日报|accessdate=}}</ref><ref>{{Cite news|url=http://news.seehua.com/?p=92567|title=黄招发：砂沙签署 20条款非法律文件|author=诗华日报|date=2015-09-16|work=诗华日报|accessdate=}}</ref><ref>{{Cite news|url=http://news.seehua.com/?p=107481|title=政府委会报告 大马立国契约保护砂沙|author=诗华日报|date=2015-10-29|work=诗华日报|accessdate=}}</ref>

尽管如此，《二十点协议》中的部份条文在《[[马来西亚协定|马来西亚协定]]》附录A文件，即《[[1963年马来西亚法令|马来西亚法案]]》中获得采纳，并後來成為《[[馬來西亞憲法|馬來西亞憲法]]》的一部份。其他條文則獲口頭接納。{{cn}}

《二十點協議》被一些關注沙巴權益人士所質疑，他們認為该協議需要根據社會、經濟和政治的轉變而作出修訂<ref name="Daily Express 2004(2)">{{cite news|url=http://www.dailyexpress.com.my/news.cfm?NewsID=31501|title=Sabah, S'wak urged to review agreements|date=2004-12-21|publisher=Daily Express|accessdate=2010-09-15|deadurl=yes|archiveurl=https://web.archive.org/web/20110603232955/http://www.dailyexpress.com.my/news.cfm?NewsID=31501|archivedate=2011-06-03}}
</ref>。馬來西亞成立後，一些關注沙巴權益人士批評[[马来西亚政府|馬來西亞聯邦政府]]違反了《二十點協議》的精神<ref name="Human Rights Watch">{{Citation|title=Abdication of Responsibility: The Commonwealth and Human Rights|date=1991-10|url=https://books.google.com/books?id=_QTz5PCDvjEC&printsec=frontcover&dq=abdication+of+responsibility+and&hl=en&ei=a2eQTNLiDYimvQOfjuDVCw&sa=X&oi=book_result&ct=result&resnum=1&ved=0CCUQ6AEwAA#v=onepage&q&f=false|quote=|pages=33–34|location=United States of America|publisher=Human Rights Watch|isbn=1-56432-047-2|accessdate=2010-09-15}}。
</ref>，尽管该协议并不具有的法律地位。

== 背景 ==
[[File:The_Report_of_the_Inter-Governmental_Committee._On_August_1st,_1962.PNG|thumb]]
根據[[英國|英國]]與[[馬來亞聯合邦|馬來亞聯合邦]]所簽的協議，尊重[[北婆羅洲|北婆羅洲]]和[[砂拉越|砂拉越]]人民的[[自決權|自決權]]對成立馬來西亞構成挑戰。

1961年11月23日，英國和馬來亞联合邦政府发布聯合聲明，指作出最後決定前会探悉北婆羅洲和砂拉越人民的意見，成立一個委員會去負責這個任務，並提出建議。

1962年1月，英國政府和馬來亞联合邦政府任命[[科博德委員會|科博德委員會]]去諮詢北婆羅洲和砂拉越的意見，以得悉當地居民會否支持建立馬來西亞聯邦的建議。該五人委員會由兩名馬來亞人和三名英國人組成，並由[[卡梅伦·科博德|卡梅伦·科博德]]擔任主席。此外，為了制訂《馬來西亞協定》的詳情，一個跨政府委員會亦成立，由英國的[[乔治·配第-菲茨莫里斯|乔治·配第-菲茨莫里斯]]和馬來亞的[[阿都拉萨|阿都拉萨]]擔任委員<ref name="Lansdowne Committee">{{cite book |title=Malaysia; Selected Historical Readings |last=Bastin |first=John Sturgus |authorlink= |coauthors= |year=1979 |publisher=University of Michigan Press |location= Ann Arbor |isbn=9783262012165 |pages= |chapter= |url=}}</ref>。

北婆羅洲為了確保在馬來西亞成立時仍能保住自己的利益、權利和自治權，於是制訂《二十點協議》，而砂拉越亦制訂类似文件即《[[十八點協議|十八點協議]]》。

== 內容 ==

=== 第一點：宗教 ===
雖然[[伊斯蘭教|伊斯蘭教]]被列為馬來西亞官方宗教而沒有做出任何反對，然而伊斯蘭教不應該在北婆羅洲實行。且所有在馬來亞联合邦憲法中有關伊斯蘭教的法規應當不許在北婆羅洲實行<ref name = "A J Stockwell 624"/><ref name = "Herman James Luping">{{Cite thesis |type=Ph.D. |chapter= |page=61|title=The Kadazans and Sabah Politics |url=http://researcharchive.vuw.ac.nz/bitstream/handle/10063/768/thesis.pdf?sequence=1 |author= |last=Luping|first=Herman James |year=1985 |publisher=Political Science Department at the Victoria University of Wellington |accessdate=2012-10-28 |docket= |oclc= }}</ref>。

=== 第二點：語文 ===
*馬來文應該被列為马来西亚国语。
*英文必須持續使用至到十年後的[[馬來西亞日|馬來西亞日]]。
*無論在何時、何地點或對洲與直轄區執行的目的，英文必須列为北婆羅洲的官方語言<ref name = "A J Stockwell 624"/><ref name = "Herman James Luping">{{Cite thesis |type=Ph.D. |chapter= |page=61|title=The Kadazans and Sabah Politics |url=http://researcharchive.vuw.ac.nz/bitstream/handle/10063/768/thesis.pdf?sequence=1 |author= |last=Luping|first=Herman James |year=1985 |publisher=Political Science Department at the Victoria University of Wellington |accessdate=2012-10-28 |docket= |oclc= }}</ref>。

=== 第二點：憲法 ===
雖然北婆羅洲接受馬來亞憲法作為馬來西亞憲法的基礎，馬來西亞憲法需要是一份新的草擬文件，並需得到各州的同意。北婆羅洲的新憲法亦應受到重視<ref name = "A J Stockwell 624">

=== 第四點：聯邦最高元首 ===
北婆羅洲元首無權角逐[[马来西亚最高元首|马来西亚最高元首]]<ref name = "Herman James Luping" />。

===第五點：聯邦名稱===
是“馬來西亞”，而不是“大馬來亞”<ref name = "Herman James Luping" />。

=== 第六點：入境事務 ===
應與中央（联邦政府）聯合管理，但中央政府對北婆羅洲的一切移民事務應先得到北婆羅洲政府的首肯。除安全理由外，中央政府對人員入境北婆羅洲並沒有否決權。北婆羅洲在入境與出境事務中有保留權<ref name = "A J Stockwell 624" /><ref name = "Herman James Luping" />。

=== 第七點：退出權力 ===
北婆羅洲不應從聯邦脱离<ref name = "Herman James Luping" /><ref name = "R B Mandal">

=== 第八點：婆羅洲化 ===
指定當地人成為北婆羅洲公務員的過程應盡快實行<ref name = "Herman James Luping" /><ref name = "R B Mandal"/>。

===第九點：英國官員===
尽量鼓励英籍官员留任直到有适当北婆羅洲人选可以取代时为止<ref name = "Herman James Luping" /><ref name = "R B Mandal"/>。

=== 第十點：國籍 ===
根據科博德委員會報告的第148(k)段，北婆羅洲下列居民持有馬來西亞國籍：
*在北婆羅洲居住滿五年
*十年內有七年在北婆羅洲居住
*出生於北婆羅洲，父母為北婆羅洲居民<ref name = "Herman James Luping" /><ref name = "R B Mandal"/>

=== 第十一點：稅收和金融 ===
北婆羅洲負責自己的財務、發展及稅收，並有權自行徵稅及借貸<ref name = "Herman James Luping" /><ref name = "R B Mandal"/>。

=== 第十二點：原住民或土著人的特殊地位 ===
北婆羅洲土著應享有類似於馬來亞联合邦馬來人的特權，但現有馬來亞模式對北婆羅洲未必完全適用<ref name = "Herman James Luping" /><ref name = "R B Mandal"/>。

=== 第十三點：政府 ===
*北婆羅洲總理由立法會非官守議員選出。
*北婆羅洲實行部長制<ref name = "Herman James Luping" /><ref name = "R B Mandal"/>。

=== 第十四點：轉型時期 ===
在加入馬來西亞的七年過渡期內，北婆羅洲之立法權力將由北婆羅洲憲法賦予，聯邦政府暫無權力插手<ref name = "Herman James Luping" /><ref name = "R B Mandal"/>。

=== 第十五點：教育 ===
北婆羅洲的教育系統會被保留，並由北婆羅洲政府控制<ref name = "Herman James Luping" /><ref name = "R B Mandal"/>。

=== 第十六點：保護憲法 ===
在未獲得北婆羅洲政府的同意之下，聯邦政府沒有權利修訂或廢除保護北婆羅洲權益的條文。

北婆羅洲的修憲權歸北婆羅洲人民所有<ref name = "Herman James Luping" /><ref name = "R B Mandal"/>。

===第十七點：聯邦議會的代表===
聯邦國會的代表不僅僅應取決於北婆羅洲人口但也必須取決於州的面積，國會代表的人數亦不可少過[[新加坡|新加坡]]代表的數量<ref name = "Herman James Luping" /><ref name = "R B Mandal"/>。

===第十八點：首長名稱===
國家元首<ref name = "Herman James Luping" /><ref name = "R B Mandal"/>。

===第十九點：国家名稱===
沙巴<ref name = "Herman James Luping" /><ref name = "R B Mandal"/>。

===第二十點：土地、森林、當地政府和其他===
聯邦憲法規定之國家土地局權力並不適用於北婆羅洲。同樣地，北婆羅洲亦不屬地方政府委員會的權力範圍內<ref name = "Herman James Luping" /><ref name = "R B Mandal"/>。

==爭議==

===第七點：退出權力===
{{tsl|en|Universiti Malaysia Sarawak|砂拉越大學}}的仄尼里博士認為，根據第七點，砂拉越和沙巴沒有權利從聯邦分裂出去<ref>{{cite news|url=http://www.malaysia-chronicle.com/index.php?option=com_k2&view=item&id=362451:sabah-swak-secession-really-just-a-vocal-minority-or-is-putrajaya-lying-to-itself?&Itemid=2#axzz3ERgAcdFa|title=SABAH, S'WAK SECESSION: Really just a 'vocal minority' or is Putrajaya lying to itself?|last=|first=|date=2014-02-17|publisher=Malaysia Chronicle|accessdate=2014-09-28}}</ref>。但是，沙巴分離主義組織領袖多丽丝琼斯則認為，條文中使用了「{{lang|en|should}}」一詞，而非使用帶有命令含意的「{{lang|en|shall}}」一詞，顯示這只是一個建議<ref>{{cite news|url=http://sabahsarawakmerdeka.blogspot.com/2014/09/should-vs-shall-consideration-for-point.html|title=SHOULD vs SHALL - A Consideration for Point 7:Right of Secession|last=Jones|first=Doris|date=2014-02-17|publisher=Sabah Sarawak Keluar Malaysia (SSKM)|accessdate=2014-09-28}}</ref>。

根据《[[马来西亚宪法|马来西亚宪法]]》第二條規定，[[马来西亚国会|马来西亚国会]]有權變更州的邊界和允許新州加入聯邦，並沒有規定州可以脫離聯邦<ref>{{cite news |title=Uncommon Sense with Wong Chin Huat:Can Sabah or Sarawak Secede?|url=http://www.thenutgraph.com/uncommon-sense-with-wong-chin-huat-can-sabah-and-sarawak-secede/|date=2014-01-15 |last=Ann Surin|first=Jacqueline|publisher=The Nut Graph |accessdate=2014-09-28}}</ref>。但是曾任职于[[馬來西亞國際伊斯蘭大學|馬來西亞國際伊斯蘭大學]]的學者[[阿都阿兹巴里|阿都阿茲巴里]]表示，该條文亦規定馬來西亞國會具有將州脫離聯邦的最終決定權，如1965年馬來西亞國會曾通過新加坡退出馬來西亞的動議<ref>{{cite news|title=Not easy for Sabah, Sarawak to leave Malaysia, says constitutional expert|url=http://www.themalaysianinsider.com/malaysia/article/not-easy-for-sabah-sarawak-to-leave-malaysia-says-constitutional-expert|date=2014-02-17|last=Anbalagan|first=V|publisher=The Malaysian Insider|accessdate=2014-09-28|deadurl=yes|archiveurl=https://web.archive.org/web/20140302064301/http://www.themalaysianinsider.com/malaysia/article/not-easy-for-sabah-sarawak-to-leave-malaysia-says-constitutional-expert|archivedate=2014-03-02}}</ref>。

根據《{{tsl|en|Sedition Act 1948|1948年煽動法令}}》，提出任何關於沙巴和砂拉越從馬來西亞獨立的建議均是違法行為<ref>{{cite news |title=Rahman Dahlan: Seditious to suggest that Sabah, Sarawak secede|url=http://www.thestar.com.my/News/Nation/2014/02/17/Sabah-Sarawak-suggestion-to-secede-seditious/|date=2014-02-17 |last=Surach |first=G|publisher=The Star (Malaysia)|accessdate=2014-09-28}}</ref>。

===第十八點：首長名稱===
根据《[[马来西亚协定|马来西亚协定]]》第一条，作为英殖民地的北婆罗洲和砂拉越将依据协定及各项附录条文成为沙巴州、砂拉越州，而该协定的附录A文件中定义了“''Governor''”一词含义为无统治者州属的首长，其中包括了时任'''沙巴国家元首'''（''Yang di-Pertua Negara in Sabah''）。<ref>{{Cite wikisource|Agreement relating to Malaysia between United Kingdom of Great Britain and Northern Ireland, Federation of Malaya, North Borneo, Sarawak and Singapore/Annex A|wslanguage=en}}</ref>

尽管《马来西亚协定》定义了沙巴为联邦州属，但“沙巴国家元首”一词随着“''Governor''”这一术语于1963年至1976年依然被写入《[[马来西亚宪法|马来西亚宪法]]》内。直到1976年8月27日，《[[马来西亚宪法|马来西亚宪法]]》第160条规定废除了“''Governor''”一词，并以“''Yang di-Pertua Negari''”即“州元首”取代之。<ref>{{cite book|title=Constitutions of the Countries of the World]|last=Blaustein|first=A.P.|author2=Flanz, G.H|publisher=Oceana Publications|year=1971|page=298}}</ref><ref>{{cite web|url=http://cpps.org.my/resource_centre/Perlembagaan_Persekutuan.pdf|title=Perlembagaan Persekutuan Akta 1260/2006 (The Federal Constitution revision as of 2006) (Malay language), Page 181|accessdate=2014-09-24|author=|date=2006|publisher=|archiveurl=https://web.archive.org/web/20140824221043/http://cpps.org.my/resource_centre/Perlembagaan_Persekutuan.pdf|archivedate=2014-08-24|website=Centre for Public Policies Studies|deadurl=yes}}</ref>结束了“沙巴国家元首”这一称呼，以致沙巴地位和《马来西亚协定》条文中定义为州保持一致。

然而另一边厢，自馬來西亞成立後，砂拉越元首的職銜名稱就是“州元首”<ref>{{cite news|url=http://www.therocket.com.my/en/federal-system-of-malaysia-then-and-now/|title=Federal system of Malaysia-Then and Now|last=Choong Wah|first=Lang|date=2014-01-15|publisher=The Rocket|accessdate=2014-09-24}}</ref>。

但是，部份團體認為沙巴和砂拉越應該是國家，兩地的元首亦應被稱為國家元首。他们认为砂拉越和沙巴分別在1963年7月22日和1963年8月31日獨立，而非自治，然後才在1963年9月16日與馬來亞联合邦及新加坡組成馬來西亞。他們認為1963年至1976年間沙巴的元首稱作國家元首便可證明這一點<ref>{{cite news|title=Negara, not negeri, status for Sabah|url=http://www.freemalaysiatoday.com/category/frontpage/2011/01/24/negara-not-negeri-status-for-sabah/|date=2011-01-24|last=Rintod|first=Luke|publisher=Free Malaysia Today|accessdate=2014-09-24|archive-url=https://web.archive.org/web/20160303231005/http://www.freemalaysiatoday.com/category/frontpage/2011/01/24/negara-not-negeri-status-for-sabah/|archive-date=2016-03-03|dead-url=yes}}</ref><ref>{{cite news |title=Doris explains Sabah ‘secessionist’ Facebook drive|url=http://www.freemalaysiatoday.com/category/nation/2014/09/17/jones-explains-sabah-secessionist-facebook-drive/|date=2014-09-17 |last=Goliu|first=Percy|publisher=Free Malaysia Today |accessdate=2014-09-24}}</ref><ref>{{cite news |title=Sabah&Sarawak Kehilangan Kuasa Kemerdekaan?Kehilangan Taraf Negara? (Malay language)|url=http://sabahsarawakmerdeka.blogspot.com/2012/05/sabah-sarawak-kehilangan-kuasa.html|date=2012-05-25 |last=Jones|first=Doris|publisher=Sabah Sarawak Keluar Malaysia (SSKM)|accessdate=2014-09-24}}</ref><ref>{{cite news |title=Governor should be ‘Yang DiPertuan Negara’|url=http://www.freemalaysiatoday.com/category/nation/2014/02/26/swak-governor-should-be-yang-dipertuan-negara/|date=2014-01-15 |last=Way |first=Winston|publisher=Free Malaysia Today |accessdate=2014-09-24}}</ref>。

但上述说法遭到学者质疑，因为在1963年9月16日（马来西亚日）之前，英女王依然是北婆罗洲、砂拉越和新加坡的合法主权掌握人，英女王是在英女王才放弃对上述三自治区的主权和司法管辖权的。尽管在同年7月22日和1963年8月31日，由于砂拉越和沙巴当地人被英女王透过透过驻地总督委任而获得自治，产生情感上的独立，但无论在事实上或法律上都不曾取得独立。<ref>{{Cite news|url=https://www.malaysiakini.com/columns/245224|title=沙砂真的独立过？|author=蓝中华|date=2013-10-29|work=[[当今大马|当今大马]]|accessdate=}}</ref>
